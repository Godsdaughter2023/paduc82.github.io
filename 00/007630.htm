yangzheng_msft6300 say:

Hi TressaSantillo,

I have got the information from relevant team.
This error message will appear when you remove the ‘To’ field. Since creating an appointment, it is by design that all the attendees will be informed that there is a change happened in the Group calendar. At the same time, when any change happens in the appointment, all the attendees need to be informed. So if you remove the attendees in the ‘To’ field, when canceling the appointment, there will be no attendees found to deliver the cancelation emails. Then the error message appears.

Based on this, it is required to keep the attendees in the ‘To’ field. If you don’t want to send the invitation emails, you can just click ‘save’ to save the appointment in the calendar. Then when canceling the appointment, the attendees will receive cancelation emails and there will be no error message.

Yang

Need a macro to move cell contents

I need a macro to find all the cells in Column A that contain the string "** CURRENCY TOTAL **", and then cut & paste each of these cells to the same row, column B.

jackpot. say:

Hi

 

Try the below macro

 

Sub Macro()

Dim strAddress As String, varFound As Variant, varSearch As Variant

varSearch = "** CURRENCY TOTAL **"

With Range("A1:A" & Cells(Rows.Count, "A").End(xlUp).Row)
Set varFound = .Find(varSearch, LookIn:=xlValues, LookAt:=xlPart)
If Not varFound Is Nothing Then
    strAddress = varFound.Address
    Do
    Range("B" & varFound.Row) = Range("A" & varFound.Row).Value
    Range("A" & varFound.Row).ClearContents
    Set varFound = .FindNext(varFound)
    Loop While Not varFound Is Nothing
End If
End With

End Sub
