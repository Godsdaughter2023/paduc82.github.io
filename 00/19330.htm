steve rindsberg say:

If you choose Paste Special and pick HTML as the paste type, data copied from an Excel worksheet will paste in as a PowerPoint table rather than as an Excel object.  

Perhaps that's what you were doing originally.

john korchok say:

I've never heard of any way to paste an Excel table and have it turn into a PowerPoint table. They're 2 different things. The available table styles change depending on the data selected. You can see this if you create a table in PowerPoint versus pasting in an Excel selection: the table style options are much simpler for the Excel data.

You can replace a range of cells by using Paste Special>HTML Format.

For cross-program use, I usually create Word, PowerPoint and Excel templates for clients that have matching branded themes. This makes it simple to paste between programs as the look is exactly the same in all of them.
