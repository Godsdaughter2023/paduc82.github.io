Try doing a Chkdsk Scan. The scan will look for bad system files and will attempt to repair them, if possible.
a) Click the Start Button.
b) Type cmd in the search results.
c) Right-click the cmd file and select 'Run as Administrator'.
d) In the CMD window, type chkdsk /f /r and hit the Enter key.

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.
Note: Chkdsk /f /r enables chkdsk to check volumes for file system errors and bad sectors on the physical disk. Chkdsk /f /r schedules itself to be run on the next system reboot, by adding an entry into the BootExecute value.
