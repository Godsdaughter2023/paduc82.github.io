You have two options:  GPRS or Wi-Fi

Before you start designing reports you need to tell us - or establish! - the structure of the Tables involved. Do you already have tables set up? or are you working backwards, designing the report first and then fitting the tables to the reports? If so don't: it's vital to come up with a properly normalized table design first. The report is an end product, not a good design tool!

Application crashes when trying to send email in Outlook 2016 version 16.0.7127.1021 (Office version 16.0.7167.2040)

Latest office update 16.0.7167.2040 has some kind of glitch that's crashing any application that's trying to use SimpleMAPI interface to send out an email using Outlook 2016 (32 bit). Anyone else having this problem? We've already got 10+ PCs having the same issue.

Hi GMRMI,

 

Based on your description, after you update the Outlook to version 16.0.7167.2040, office application crashes when you send emails on Outlook 2016.

 

And thanks for sharing of the work around that convert the Outlook client to an earlier version of Office 16.0.7070.2033.

 

When the issue occurs, please try the below steps and check if it can be resolved:

1. If you are using Outlook client, please make sure your Outlook client is the latest version and follow the article below and try an Online repair.

https://support.office.com/en-us/article/Repair-an-Office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b

2. Please run the Support and Recovery Assistant (SaRA) tool.
To download and install SaRA, please refer to: http://diagnostics.office.com/#/?env=Prod  .
If the issue persists after running SaRA, please provide us with the output of SaRA for further troubleshooting.

 

If the issue still exists after following the steps above, I’d like to collect the information below:

  1. Can you send emails out via Outlook Web App (OWA) in the problematic network?
  2. Regarding “that's crashing any application that's trying to use SimpleMAPI interface to send out an email using Outlook 2016 (32 bit)”, do you mean all the Office application (such as word, excel, PowerPoint) or only Outlook client crash?
  3. Did you make any change of SimpleMAPI registry? If yes, what kind of change?

 

To protect the privacy, I have sent you a private message to collect the SaRA result. Click the following link to access it:

https://answers.microsoft.com/en-us/privatemessages/list
