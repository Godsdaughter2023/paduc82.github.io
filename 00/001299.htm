gord dibben say:

How did the cells become colored yellow?

If by conditional formatting, use the same condition for you "IF" formula.

If manually colored then you will need VBA as others have pointed out.

Here a UDF from Chip's site you can use.  Copy/paste it to a General Module in your workbook

Function CellColorIndex(InRange As Range, Optional _
        OfText As Boolean = False) As Integer

' This function returns the ColorIndex value of a the Interior
' (background) of a cell, or, if OfText is true, of the Font in the cell.
  

  Application.Volatile True
    If OfText = True Then
        CellColorIndex = InRange.Font.ColorIndex
    Else
        CellColorIndex = InRange.Interior.ColorIndex
    End If
End Function

Then use a formula in any blank cell.

=IF(cellcolorindex(B10)=6,B8*I10)

The "=6" assumes standard Yellow fill color.

    We suggest that you perform a soft reset. Press and hold the volume down and Power button until your phone vibrates and restarts. Note that you should not touch the device screen during this process (not even when the 'slide down to power off' prompt appears). After performing a soft reset try to check if the issue persists.

    If a soft reset didn't work, we suggest that you perform a hard reset instead. You can reset your phone under Settings: Settings  > about > reset your phone. Make sure to back up all of your files before performing a hard reset. 

    Windows 10

    Go to Settings  > Update and security > Backup >More options > Back up now.

    Windows 8.1

    Go to Settings > Backup > Apps + settings > Back up now.

    If performing a soft reset and hard reset didn't work, we suggest that you use the Windows Device Recovery Tool.

    Start by downloading and installing Windows Device Recovery Tool (for Windows 10, 8.x and 7).

    When the installation has finished, start the application and the main screen will appear:

    The procedure to restore your phone using Windows Device Recovery Tool is as follows:

    1. Connect the phone to your PC (using the USB cable that came with the phone).
    2. After a few seconds the PC should find the phone and you'll see the Phone Selection screen. Click your phone in this screen to select it.
    3. After a few seconds more the latest available firmware version for your device will be shown. Click the 'Install software'/'Reinstall software' button to flash your phone
    Windows Device Recovery Tool will then download and install the latest firmware + OS available for your phone

    Note! Your phone will be reset to factory status when using this tool.
