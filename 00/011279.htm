d r h say:

You are misreading the options.  You do not have 2 entirely separate monitors to do work on. For multiple displays the choices are Show only on 1, Show only on 2, Duplicate these displays or Extend these displays. Whatever background picture you choose will be applied to these displays.

The most commonly used of these is 'Extend these displays' which means that the two monitors are essentially combined into one and anything on the main display can be extended, moved or dragged to the second monitor.

When choosing a background, the same picture will be on both monitors unless using 'Span' as a fit where the picture is stretched to fit across both monitors.  In this case it is better to use a panoramic picture.  The only other way I can think of would be to pre-combine 2 different pictures into one before choosing as a background.

kgriba say:

I want to print a Word document that contains hyperlinks

I want to print a Word document that contains hyperlinks (to the internet as well as to other Word files). How can I print this Word document such that it prints the actual complete hyperlinks as well? Say, on a second page or something, as properties of the Word file?

Any suggestions?

Thanks!
jay freedman say:
This macro should do it (http://www.gmayor.com/installing_macro.htm if needed)

Sub PrintWithHyperlinks()
    Dim PrtDoc As Document
    Dim HL As Hyperlink
    Dim rg As Range
    Dim addr As String
    Const bkmk = "DoNotUse314159"
    ' bkmk used by undo section of macro
   
    Set PrtDoc = ActiveDocument
    Set rg = PrtDoc.Range
    rg.Collapse wdCollapseStart
    PrtDoc.Bookmarks.Add Name:=bkmk, Range:=rg
   
    Set rg = PrtDoc.Range
    rg.Collapse wdCollapseEnd
    rg.InsertBreak Type:=wdPageBreak
   
    For Each HL In PrtDoc.Hyperlinks
        addr = HL.Address
        If Len(HL.SubAddress) Then
            addr = addr & "#" & HL.SubAddress
        End If
        rg.Text = addr & vbCr
        Set rg = PrtDoc.Range
        rg.Collapse wdCollapseEnd
    Next
   
    On Error Resume Next
    Dialogs(wdDialogFilePrint).Show
   
    While (PrtDoc.Bookmarks.Exists(bkmk))
        PrtDoc.Undo
    Wend
End Sub

Follow the steps below to create a new PST file.

1.       Launch Outlook

2.       Click File > New > Outlook Data file.

3.       Select Office Outlook Personal Folder (.pst)

4.       Name the data file and Click OK.

5.       Copy the content of the Calendar from the OLD data file.

Change the view, Click View > Current View > All appointments > Select all appointments (Ctrl + A)

6.       Copy it into the calendar in the new data file.

7.       Change the new Data file as default and then try to sync and check if the issue occurs.

Click Tools > Account Settings > Select the Data file tab > Select the new data file and click set as default.

8.       Copy the contents (Ctrl + A) of the contacts folder from the old Data file and paste (Ctrl+ v) in the contact folder in the new data file.
