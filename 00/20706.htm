Coverting a tab delimited number

I have a tab delimited file that I have opened with Excel.  Numbers appear in a format like this:  9.40551E+21.  How do I convert this string to a simple number like 9405510200830450209236?
j.e. mcgimpsey say: That's not such a simple number to XL.

XL, as most every other spreadsheet, stores numeric values as Double Precision Floating Point numbers. This means that they can contain, at most, about 15 decimal digits of precision, so at least the last 6 digits of that 9.40551E+21 will be truncated to zeros.

Are you really using that value as a number? Performing calculations (I can't think of any applications that require - or even have a real-world analog of - that level of precision)?

If so, to use XL you'll have to break it up into smaller chunks and do math by parts. Exactly how depends on what calculations you intend to do.

If not, then in the 3rd pane of the Text Import dialog, select the relevant columns in the data preview display and click the Text radio button. XL will then import the "numbers" as text, and won't parse them into numeric equivalents.
