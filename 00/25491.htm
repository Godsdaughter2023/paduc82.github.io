hansv mvp say:

You can use a pivot table for this purpose:

Click anywhere in your data.

On the Insert tab of the ribbon, click Pivot Table.

Select a location, then click OK.

Drag Item and Description to the Row Labels area.

Drag Amount to the Values area.

Click the dropdown arrow on the right of 'Sum of Amount' in the Values area.

Select Max as function to summarize the data by, instead of Sum.

Click OK.

On the Design tab of the ribbon (under PivotTable tools), select Subtotals > Do not show subtotals.

Also select Grand Totals > Off for rows and columns.

Finally, select Report Layout > Tabular form.

dougt9 say:

Word will track changes differently depending on how you make them.

If you select a character in a word and over-type, that will result in the change being tracked at the word level.  The whole word will appear as changed.

If instead you backspace over the character and then type a new character in two steps, that will result in the change being tracked at a character level.
