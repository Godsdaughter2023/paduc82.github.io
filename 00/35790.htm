if I would like to change the time for window update at next time, do I need to edit using Gpedit.msc instead of window update under control panel?


You can do it solely with Group Policies or you can manually add entries to the registry and use the Automatic Updates applet in the Control Panel, or you can use a combination of any of those methods. 

You can selectively use Group Policies along with the Control Panel applet.

If you want to use the Control Panel applet but still want to force the machine to reboot after installing updates you can set the "Configure Automatic Updates" policy to "Not configured" and set the "No auto-restart with logged on users for scheduled automatic updates installations" to Enabled.  You can also set the duration of the countdown timer in Group Policies.

If you wish you can configure things as you want them with Group Policies then:
1- Export the KEY_LOCAL_MACHINE \Software\Policies \Microsoft\Windows \WindowsUpdate\AU key.
2- Set all the group policies again to "Not configured", (this will remove the registry entries).
3- Merge the previously exported key back to the registry, this will apply the settings back to the machine.  This can be useful when you want to apply the same settings to other machines that you manage, for example you could merge this to other Windows XP Pro machines in a workgroup, or to Windows XP Home machines, XP Home does not have the Gpedit tool so merging registry entries created on another machine is easier and more reliable than manually entering the entries.  Just keep in mind that you probably wouldn't want all your workgroup machines to do their updates at the same time as this would choke your internet connection.

I've found a solution to this annoying pop up keyboard.  Follow these instructions

1.     Type in ‘services’ into the Cortina search box

2.     Tap the app to open it

3.     Scroll down, search for ‘Touch screen keyboard and handwriting

4.     Double click on this to open up an options window

5.     Beside ‘Startup type’ use the dropdown and select ‘manual’

6.     Then select ‘Service Status’ and click ‘Stop’

7.     Finally, click ‘apply’ and then ‘OK’
