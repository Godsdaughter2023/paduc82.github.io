mark bu say:

Hello,

Sorry for the delay in response.

The Mail app in Windows 10, does not currently support local email storage. It only keeps a local cache of what was available on the server during it's last sync.

When syncing with the email server the Mail app will download any new mail, as well as remove any emails that have been removed from the server.  This behavior is by design.

If you require offline/local storage of email, I would suggest using Windows Live Mail (it is free and is a part of Windows Essentials) or Outlook.
