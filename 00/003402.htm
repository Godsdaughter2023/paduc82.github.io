gary daniels say:

Hello Asbjørn,

Thank you for your feedback. If you plug in a wired headset on Windows Phone 8.0 or 8.1, you'll see that notification sounds, ringtones, etc. come out both the wired headset and the loud speaker simultaneously. We have heard feedback that the lack of these sounds out of the loud speaker when using Bluetooth headphones is an issue and are actively looking into adding the same feature to Bluetooth, so the sounds will come out both Bluetooth and the loud speaker. I hope that will address your issue with missed notifications.

Regarding your preference for music to come out of speaker instead of Bluetooth. The Bluetooth specification allows for a Bluetooth hands free device which can only be used for phone calls (HFP profile), a Bluetooth stereo device (a2dp profile), and a combination device supporting both profiles. When you connect a Bluetooth combination device to windows desktop you will see 3 endpoints added, headset render, headset capture, and headphones. You can see these three in the sound control panel under "playback devices" and "recording devices." These same 3 endpoints are created on windows phone.

  • The Bluetooth "headset" endpoints are used exclusively for phone calls and speech, they are the ones marked as the default communications endpoint on Windows desktop. On both Windows desktop and Windows phone, Bluetooth headset (HFP profile) endpoints are never used for music.
  • The Bluetooth "headphones" endpoint is prioritized similarly to wired headphones and will be automatically selected as the default endpoint for all non-communications audio on both desktop and phone. The difference here is that on desktop you have the ability to manually set a different endpoint as the default, but on phone the default endpoint cannot be manually set. Keep in mind that many devices that report themselves as Bluetooth A2dp profile do have good audio quality for music and would be preferred by most people over the integrated speaker. These are things like cars with integrated Bluetooth, home stereo systems, and stereo headphones.

I mention all of this because it appears you're looking for some way to turn off the a2dp profile on your Bluetooth combo device and use it for phone calls exclusively. Disabling the a2dp profile could certainly be done through the Bluetooth UI on the phone, and we will communicate this request to that team. In the interim, to work around your problem, some Bluetooth devices also have the ability to turn off (or separately pair and control) their Bluetooth profiles. I'd recommend reading the manual for your peripheral to see if it has the ability to separately control the profiles. Turning just a2dp on and off on your peripheral may give you a better experience than turning your device on and off all the time. For example, with the Bluetooth device I use you can hold down the "play" button for a couple seconds and change or disconnect a2dp audio separately from HFP.

    When you use a PST file, the causes and solutions are limited - if the messages are deleted and the deleted folder emptied, there is no recovery option. You need to have a backup of the PST file to recover the messages.

    It's less likely that they are lost due to a corrupt PST file. You'd have more problems than just some missing folders, but you can try scan PST to see if this finds corruption and brings them back. (ScanPST will find some corruption and repair it as there always is minor corruption, but I would be surprised if it restores the missing mail.)

    There are PST recovery utilities you can try but they aren't free - they have trial versions which will show you the messages they can restore before you buy. If you can't find the messages it means the messages were are gone for good unless you have a backup of your PST files.

    A list of utilities is here https://www.slipstick.com/problems/pst-repair/r... - if one of the utilities cant find the missing mail, none will be able to.

    You should make a backup of the PST file weekly if not more often even monthly backups would be better than none.

    karl timmermans (outlook mvp 2012-2018) say:
    The article below describes how to relocate .ost files associated with an Imap account. The one important item to take note of is that if you have any data in folders labeled with the words "This Computer Only" (i.e. contact/calendar folders) - that data must first be exported to a PST file or you will lost that information. If this is the case, once you've moved the .OST file, would then
    1) move the PST file to your "E" drive,
    2) open it in Outlook via File --> Open & Export --> Outlook Data File --> browse to the exported PST file
    3) then File --> Account Settings --> Data Files tab --> select the PST file and set it as the "default data file" which is where your default contact/calendar folders will be located for the profile thereby preventing the risk of losing any contact/calendar data should the Imap account be deleted. Also, in terms of OST files, if Outlook cannot find the file for any reason, it will recreate the file so anything contained in "This Computer Only" folders will be empty.
