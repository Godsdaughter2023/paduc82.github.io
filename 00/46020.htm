cmdrkeene say:
They might look like it, but Libraries are not folders (in the same way that ZIP files are not folders).  You can set their view, but the view will be applied to everything you access via that library.  It won't destroy the other folder's views, but it will ignore them and use the library's view during that one session.

To set a view for a library to be list view, open the library and set the view, then close the library.  Next time you open it, it will still be in list view.
