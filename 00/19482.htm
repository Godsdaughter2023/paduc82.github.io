Obvious, perhaps, but because you can't select individual records on Mac, your best bet is probably to add a column called "Include" into your .csv or Excel sheet, then put (say) a 1 in that column for each row you want to merge, and leave the column blank for the rows you do not want. bob jones aka: cybertaz say:

In Mac Word, the Edit Recipient List button is active only if a Word table document is being used as the record source. However, even when available it only opens the document for editing records... It has nothing to do with selecting recipients to be included in the merge.

What you're probably looking for is the Mail Merge Recipients dialog box which is produced when you click the Edit Recipient List button in Word for Windows. That dialog is available only in the PC version of Word. It's the one that provides a columnar list of the data including check boxes for selecting specific records in addition to the Sort/Filter features.

Mail Merge in Mac Word provides the Sort/Filter functions [Filter Recipients button] but not the option to hand pick recipients.

the-watchmen say:

This is what you should try...

I had the same problem and it worked for me...

First make sure the following is done from an administrator account.

You will be typing the following into the CMD console, so make sure it is copied exactly as I have typed it here

(ONLY what is within the quotes).

1. While holding down the Windows key,

press the R key

The Run dialogue should now be open.

2. In the Run dialogue type "cmd" and press Enter.

The command prompt should now be open.

3. Type "ipconfig /flushdns" and press Enter.

4. Type "ipconfig /registerdns" and press Enter.

5. Type "ipconfig /release" and press Enter.

6. Type "ipconfig /renew" and press Enter.

7. Type "netsh winsock reset" and press Enter.

8. Restart your system.

That's it...

If you can't see them with those fonts, you might need a newer version of Office, or maybe of Windows too.
