a. user say:

You can't "install" Windows onto a flash drive.

What you can do is to make a System image onto your flash drive - having said that a System Image is normally in the region of 30 GB +, depending on how much data and how many programs you have installed when you make the system image, so you would need an enormous flash drive to do that.

Secondly, your Computer vendor has to provide a method for returning your computer to factory specifications. Normally these days that's via a Recovery Partition or a set of Recovery Disks. Read the documentation that came with your computer and find out. If you have a Recovery Partition then it's quite usual for the Vendor to provide a utility for making a set of Recovery disks from that partition. I suggest if that is the case you do so as soon as.

If you have a set of Recovery disks, then you're set.

However, you should also put into place a routine to backup your data (ie your documents, music, videos, favorites, email etc) so that when you have to use the Recovery disks or Partition, you can re-instate your data quickly.

Create A System Image In Windows 7

After you install and activate Windows 7 create a system image as a backup for your current computer in case you ever need to reinstall it. This will restore your computer to it's exact state when the image was made - that will save you the trouble of having to reinstall Windows 7 and everything currently installed again, and it takes much less time to do.


1.  Click the Start Menu and type "Back Up" then click on Back up your Files.  

2.  In the Control Panel window that appears, Click the blue Create a system image link.

3.  The Create a system image window should pop-up.  Select which hard disk, DVD, or network location you are going to save your system image to.  Click Next to continue.

4.  On the next page check both your System and System Reserved drives.  Once you have selected the drives Click Next to continue. Note: You can select as many drives as you want (if available), but the more you choose to include means the larger the image size will be.

5.  The next window is just a review, make sure everything is correct then Click Start backup to start the process.  This can take 10 minutes or more so patience is required.

6.  After the backup completes, you?ll be prompted with the option to create a system repair disc.  This is a good idea just in case your system has any problems in the future.  Click Yes.

7.  Choose which Disc Recording drive you?ll be using to burn the disc.  Insert a blank CD or DVD to the physical drive and then Click Create disc.

syed n say:

Hi,

As you rightly mentioned, My Documents is a junction point to the Documents folder. This is seen as My Documents in Windows Explorer but when you see it from the command prompt it will be shown as Documents.

Now, coming back to the issue, I would want you to check if you have granted permissions as well to the file you are trying to delete and check if that helps.

1.       Right click on the file and select Properties.

2.       Click on the Security tab and click Edit.

3.       Now click Add and type either Everyone or the username that you want to grant permissions to under Enter the object names to select and click Check Names. Click OK.

4.       Now select the username you typed from Group or user names and check Full Control under Permission for.

5.       Click Apply and OK. Check if you can delete the file.

 

As far as default installation path is concerned, check to see if the registry has the right settings.

 

1.       Click Start , type regedit in the Start Search box, and then click regedit in the Programs list.

                If you are prompted for an administrator password or confirmation, type your password, or click Continue.

2.       Locate and click the following registry subkey:

                HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion

3.       On the right pane, make sure the Value data for the Value name ProgramFilesDir is C:\Program Files and for ProgramFilesPath, it is %ProgramFiles%.
