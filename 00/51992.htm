The machine may have a Recovery partition which is used to reinstall the operating system and/or drivers.

If that's indeed the case, not knowing the product key isn't a problem because under the terms of the license agreement the manufacturer has with Microsoft, validation doesn't take place. Therefore, no product key label will be found on the machine.

If that is indeed the case, a third party tool like Tricky suggested will only find the general key which can't be used to validate Windows if the user uses an alternative means to reinstall the operating system like you propose to do for example.

It might be worth your while checking to see whether the machine has a Recovery partition before you proceed. You'll find a list of key combinations for different manufacturers in this article:
http://www.pcadvisor.co.uk/how-to/laptop/3462995/factory-reset-laptop/
