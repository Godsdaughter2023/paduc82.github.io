how do i set my computer to power on when i open the lid - Microsoft Community

larry's wife say:

how do i set my computer to power on when i open the lid

i want my computer to power on when i open the lid  how do i set this up
cmdrkeene say:
Hello and welcome to Microsoft Answers.

Most laptop computers are already built to wake up from sleep if you open the lid, but there's a bit of a catch: it depends on how deep of a sleep the computer is in.  If the computer is fully shut-down and totally powered off, opening the lid won't (usually) turn it on.  Doing so would be up to the manufacturer who built the computer (they would have to build a hardware feature that detects the opening, then triggers the power).  Why can't WIndows do it?  Well, Windows isn't running, so the manufacturer would have to build the feature into the physical case itself.

Windows can wake the computer when it sees the lid open if the computer is merely sleeping, because (without getting too technical here) even in sleep mode there is still some 'awareness'  to the rest of the system (Windows is, in some tiny form, still active).

I am sorry if this doesn't meet your direct desire to power on, but perhaps it sheds light on why that is the case.



PS.  For what it's worth, I only use sleep mode.  Unless there is an update that requires a full reboot, my laptop never actually gets "shut down".  Sleep in Windows 7 is very efficient and reliable, and I recommend giving it a try.

Hello,

Sorry to know that you are facing issue on Windows 10. I will certainly help you.

This issue might occur if the BCD (Boot Configuration Data) files which contains boot configuration parameters and Windows startup files goes corrupted or the Boot\BCD file on the active partition is damaged or missing.

I suggest you to repair your computer by creating a Windows 10 Bootable media (i.e. USB or DVD) using Media Creation Tool. Once you're ready with Bootable Media, follow the below mentioned steps.

1. Insert the Windows installation disc in the disc drive or connect USB media and then start the computer.

2. Press a key when you are prompted.

3. Select a language, a time, a currency, a keyboard or an input method, and then click Next.

4. Click Repair your computer.

5. Click the operating system that you want to repair and then click Next.

6. On the Choose an option screen, select Troubleshoot > Advanced　options > Command Prompt and type

    below commands.

Type bootrec /fixmbr and press Enter.

(This command will write a Windows 10 compatible Master Boot Record to the system partition)

Type bootrec /fixboot and press Enter.

(This command writes a new boot sector to the system partition by using a boot sector that's compatible with Windows 10)

Type bootrec /scanos and press Enter.

Type bootrec /rebuildbcd and press Enter.

(These two commands scans all disks for installations that are compatible with Windows 10)

7. Restart the computer and check if the issue is resolved.

Hope this helps. If the issue remains unresolved, please get back to us and we would be happy to help.

Consumer Windows 32-bit (XP, Vista, 7, 8, 8.1) can only utilize 4GB of memory - and even then, it is usually an amount between 2.75GB and 3.75GB that is visible (more commonly around 3.25GB.)
 
Depending on what you utilize the computer for, you may or may not see any benefit to the extra RAM at all.  It certainly won't hurt to have 4GB and it probably is not all that expensive to go that high.  Your best bet - depending on whether you can or cannot - would be to upgrade to a 64-bit Windows OS (it would be a clean install - but probably worth it if you feel you can utilize 4GB of memory to its fullest - where most computer users will seldom top 2.5GB.)
 
Why do you have a distaste for 64-bit Windows?  Most 32-bit applications work without issue on a 64-bit operating system.  There are few that do not, actually.
