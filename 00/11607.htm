tobytu_msft say:

Hi Manjunatha,

Please follow the steps below to disable the connected account feature for all users.

Sign in to EAC with your Office 365 administrator account < Permission < User roles < Default Role Assignment Policy < Uncheck the MyMailSubscriptions button.

Then all users who have been assigned the “Default Role Assignment Policy” cannot see the Connected Account feature in OWA.

If you have created another policy which is used for your organization, please select this policy. Refer to the following steps to check it.

Sign in to EAC < Recipients < Mailboxes < Double click one mailbox < Mailbox features < Role assignment policy.
