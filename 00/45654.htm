Hi Pedrossa,

 

Thanks for using the Answers Forum.

 

“No contact with DHCP-server can be made” – from your post.

That’s probably the issue. If you can connect to another network with no problems, your settings for the other network might be wrong.

Specifically with the DHCP server, which gives you your IP address. Which in turn, can prevent any access to the network.

 

Is this a business network your trying to connect to and failing?

You may want to contact the person that set up that network or someone who is the administrator for that network for the correct connection settings. I then suggest creating a network profile on the computer.

Copied from http://windowshelp.microsoft.com/Windows/en-us/help/aed591db-d3c6-44fc-9c51-ea51e749a7a61033.mspx#EY

 

Create a network profile

 
 

1.

Open Connect to a Network by clicking the Start button , and then clicking Connect to.

A list of networks currently available is displayed.

2.

Do one of the following:

•

If the network that you want to create a profile for is in the list, click it, and then click Connect.

If you save the network after successfully connecting to it, a profile for the network is stored on your computer. The next time you open Connect to a Network, the network will be in the list of available networks, if it is in range of your computer.

•

If the network that you want to create a profile for is not in the list and you think that it is in range of your computer, make sure that the wireless network adapter on your computer is enabled by doing the following:

   
 

a.

Open Network Connections by clicking the Start button , clicking Control Panel, clicking Network and Internet, clicking Network and Sharing Center, and then clicking Manage network connections.

b.

Right-click the network adapter, and then click Enable.

•

If the network that you want to create a profile for is not in the list, click Set up a connection or network, click Manually connect to a wireless network, click Next, and then follow the instructions. If you save the network, a profile for the network will be created and stored on your computer. To select specific authentication and security settings, click Change connection settings.

a. user say:

Hi,

 

Time Limits should work on Windows Vista when you use Family Safety. It's possible that you are experiencing this issue because you might be using an old version of Family Safety, or setup is not properly done, or the settings are not refreshed at the Family Safety client after making changes at the Family Safety website. The latest version of Family Safety for Windows Vista is Build 15.4.3555.0308.

Apparently, the latest version of Family Safety now uses standard user accounts (Windows login) as a child in the Family Safety. If you need to know about adding a Windows login user with standard privileges, kindly refer to this link.

In case you have already created a standard user and you want to manage it under Family Safety, then follow these steps:

1. Run Family Safety in a Windows account with administrator privileges. If it's your first time running Family Safety, you will need to log in using your Microsoft Account.

2. After logging in, select the standard Windows account you would like to monitor as a child by ticking the checkbox at the right side of the account name and then click Next after.

3. At the right side, you will see Family Safety members with a drop list option underneath. Click the drop list and then select Add name of the standard account.

4. Click Save. Wait until the setup is done.

5. Once done, you should see the account name under Monitored Windows accounts.

 

Setting up Time Limits using Family Safety

After adding the child account(standard user) to your Family Safety, you can then setup the Time Limit restrictions by following these steps:

1. Login to Family Safety client using the Parent Account.

2. Click Child Settings of the Child you want to impose Time Limits.

3. Select the Windows Account you wish to associate in your parent account.   Putting a check mark and clicking next will lead you to the next screen.

4. After associating, you need to configure Time Limits from familysafety.microsoft.com (formerly fss.live.com)

5. Note: For time limits to work, the child Windows account should be a Standard user and NOT an Administrator account.

6. At the left pane, click the name of the computer or child account you want to configure by clicking Edit Settings below it.

7. Click Time Limits.

8. Note: Make sure that the Turn On box is ticked and block out the boxes you want restriction for the said Windows account.  Save to finish.

9. Once done making changes at the Family Safety website, don't forget to refresh the client settings by clicking the 2 arrows at the upper right corner of the client window (Get the latest settings from the website).

 

In case you need further assistance, feel free to post a reply.

 

Thanks and best regards.
