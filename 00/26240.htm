andreas killer say:
You have to use a formula, and the formula in a conditional formatting must return TRUE or FALSE.

So the best way to figure out if a formula works is to try it in the sheet first, e.g.

=COUNTIF($A$1:$A$10,"Complete")<>COUNTA($A$1:$A$10)

The COUNTIF returns the number of "Complete" in a range and the COUNTA returns the number of cells in a range. If they are not equal we get a TRUE and so the conditional formatting is applied.

Macro to delete empty content controls - Microsoft Community

maratae1 say:

Macro to delete empty content controls

I am developing a template with several tables that contain bullets and numbering and an RT content control. Some users won’t address every bullet point. Is there a way to delete the entire table if the user doesn’t enter text into the content control contained in said table? 
jay freedman say:

This macro will remove every table in which any RT content control has not been edited -- that is, it still shows its placeholder ("Click or tap here to enter text" or whatever you may have changed it to). There's no assumption that there is any RT control in every table,  or that there's only one per table.

Sub RemoveUnanswered()

    Dim tbl As Table

    Dim cc As ContentControl

    

    For Each tbl In ActiveDocument.Tables

        For Each cc In tbl.Range.ContentControls

            If cc.Type = wdContentControlRichText Then

                If cc.ShowingPlaceholderText Then

                    tbl.Delete

                End If

            End If

        Next cc

    Next tbl

End Sub
