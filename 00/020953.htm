Sequential dates on different sheets within excel

I am trying to create a series of worksheets for tracking data. I would like the sheets to be dated sequentially. Is this possible? (ie: Sheet 1= Sunday, December 3, 2017... Sheet 2= Monday December 4, 2017... etc)

hansv mvp say:

You could run the following macro:

Sub AddSheets()
    Dim d1 As Date
    Dim d2 As Date
    Dim d As Date
    On Error GoTo ErrHandler
    d1 = InputBox("Enter the first date")
    d2 = InputBox("Enter the last date")
    For d = d1 To d2
        Worksheets.Add(After:=Worksheets(Worksheets.Count)).Name = _
            Format(d, "dddd, mmmm d, yyyy")
    Next d
    Exit Sub
ErrHandler:
    MsgBox Err.Description, vbExclamation
End Sub
