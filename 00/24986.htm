suzanne s. barnhill say:

It is normal to get a new document called Document1. It is not normal for it to contain "garbage data." Somehow a document has been saved in or as the normal template. In the File Explorer (with Word closed), search for Normal.dotm (making sure you have the search set to include hidden and system files and folders). When you find it, rename it Normal.old. When you start Word, it will generate a fresh factory-default copy of Normal.dotm, and you should be back in business.

If you have customizations saved in the Normal template and don't want to lose them by generating a fresh copy, you can open the template for editing and just remove the content you don't want, as follows:

  1. In Word, press Alt+F11; the Visual Basic Editor opens.
  2. Press Ctrl+G to activate the Immediate window.
  3. In that window, type (or paste)

         NormalTemplate.OpenAsDocument

    and press Enter.

Normal opens, but note that it may not be on top of all windows. You can close the Visual Basic Editor. Make the desired changes to the Normal file, and then save and close it. If the changes you have made have not been recognized as changes (the Undo button is not activated), press Spacebar, then Backspace to ‚Äúdirty‚Äù the file, thus ensuring that the template will actually be saved.
