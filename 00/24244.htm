The purpose of an average is to summarize values.   Averaging numbers which are distinctly different is pretty meaningless.   You either have to score your data so that the numbers being averaged are compatible or you don't use an average (weighted or not).     For example, take a weighted average of Height in feet, Weight in ounces, Shoe size in inches, Age in months.   The results would be meaningless.

rick rothstein, mvp - excel say:

Give this macro a try...

Sub RenameWorksheets()
  Dim WS As Worksheet
  On Error Resume Next
  For Each WS In Worksheets
    WS.Name = WS.Range("D1") & Year(Now)
  Next
End Sub

NOTE: You didn't tell us exactly what you meant by "concatonated with a year ex d12011", so I assumed you meant the contents of cell D1 on the particular sheet concatenated with the current year. If this is not correct, then you need to expand your description of what you want.

peter schleif say:

Some questions:

  • Where did you get the year?
  • Is it always 2011?
  • Or the current year?
  • Or it is in another cell e.g. E1?

I've provided all three options and preselected one of them (fixed year 2011). The other two are available as comments in the code. You can activate them by removing the apostrophe.

Peter


Sub NameWorksheets()
   Dim i     As Integer
   Dim sh    As Worksheet
   Dim sh2   As Worksheet
   Dim sName As String

   For Each sh In Worksheets
      i = 1
      Do
         sName = sh.[D1] & " 2011"           'always 2011
        'sName = sh.[D1] & " " & Year(Date)  'current year
        'sName = sh.[D1] & " " & sh.[E1]     'from another cell

         If i > 1 Then sName = sName & " (" & i & ")"
         For Each sh2 In Worksheets
            If sh.Index <> sh2.Index Then
               If Left(sh2.Name, 31) = Left(sName, 31) Then Exit For
            End If
         Next
         i = i + 1
      Loop While Not sh2 Is Nothing

      sh.Name = sName
   Next
End Sub

regile reb say:

Hello,

Temporary Files are created for backup purposes, by programs. For instance, Microsoft Office saves a Temporary File of the open document every few minutes. If you save the document and exit, the Temporary File gets deleted. If the program crashes unexpectedly, the Temporary File is not deleted. They can thus be useful to help recover lost data if the program or the system crashes.

Ideally, the Temporary Files should get deleted once the program exits. But this is not always the case, leading to wasted disk space.

The Temporary Files in Windows are typically found located in two locations:

  • C:\Windows\Temp
  • C:\username\AppData\Local\Temp

Note: C:\username\AppData\Local\Temp is hidden folder, make sure to set Show hidden folders in Folder Options.

To display the Paragraph dialog box you can make use of the right-click (context) menu, which has been around for a while. Or you can click the dialog launcher button (the little arrow on the bottom right) of the Paragraph group on the Page Layout tab.

To change the default format, do the following: Create a new, blank document (for example by pressing Ctrl+N). In the Styles pane (Ctrl+Alt+Shift+S), click the Manage Styles button. On the Set Defaults tab, make the desired changes. In particular, make sure the Line Spacing is set to "Single" and that Spacing Before and After are both set to zero. When you are done, select "New documents based on this template" and click OK.
