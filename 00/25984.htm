bygp say:

My feedback after a 24h testing

Although I'll mainly list negative points here, this do not mean all is negative, positive remarks are useless when you need to improve something.

Main reason I don't feel I will upgrade : this O.S. release don't give enough informations on what it is doing in background, MS wanted to simplify maintenance tasks, and doing so launches a bunch of shadow tasks, some offline, some online, this makes me feel I'm using a Trojan based O.S.

Some other reasons :

GUI Skin : windows buttons look ****, simplicity is good and can be cute but the vector design of these buttons should be improved.

GUI Skin : not enough customization options, bring us back the XP skin configuration pannel, rewritten for W10 so we can change fonts, border size, explorer global background color (many people hate to have a #FFFFFF white background), include a color mixer.

Taskbar : We should be able to change start button, notification zone and other objects position on the bar. This is very usefull when using Dual screen.

Taskbar : Having a root start button is ok, but being able to add other categorized start buttons would be much better (Linux and Mac OS menu bar style)

A pseudo screen saver consisting of reducing screen brightness to X% (user defined) after Y minutes would be great. Animated screen savers were usefull on CRT monitors, I don't think they still are on LED/LCD/TFT and they can be energy waster (GPU ressources).

An almost instant sleep mode with instant wake up, even if using a bit more energy would be great for PC releases.

As said in another post, the install process is simple and nice, but too simple for certain users, there should be an "advanced" option to let choose which Windows components to install or not and asking what main use will be made of the PC in order to optimize the OS from the ground.

Please, get rid of Windows defender, this application never detected anything on my computer when I used it on W7, on another hand, the firewall is good but should be less permissive, blocking all should be the base rule and FW configuration should be more user friendly, using an excel-like grid with dropdown and text fields, and showing activity reports.

A smarter registry engine with changes history, basic errors correction, Dual registry concept (like Dual Bios concept) to allow booting with a working registry when the main has been corrupted. As this file is the most sensible part of the O.S.

An integrated pro backup application should be mandatory on a 2015 OS.

Office could be integrated in some pro releases.

We all know that Windows is mainly an asynchrone OS, but a better task priority management would be very usefull, for exemple, when you use a graphic tablet for artwork production, you want a responsive system focusing only on your task, this would include the painting app but also the USB device priority.

To conclude, I would say that the main improvements IMO should focus on CUSTOMIZATION and TRANSPARENCY, focusing on these two axes could make one of the best Windows release ever.

B.P. Software & System Eng.

Maybe this is already done, but the hibernate system file should be movable to any partition of the system, this is extremly important for systems installed on SSD.

Also, during the install process, the Users folder installation location should allow it to be on another disk or partition, this could also be a post-install process.

gokul t say:

Hi Sunshatoe,

 

DEP is a security feature that can help prevent damage to your computer from viruses and other security threats. DEP monitors programs to make sure they use system memory, safely. If a program tries running (also known as executing) code from memory in an incorrect way, DEP closes the program. DEP automatically monitors essential Windows programs and services, but you can increase your protection by having DEP monitor all programs

 

If you want to turn off DEP for a program you trust, first check to see if the software publisher has made a DEP-compatible version of the program or made an update available before you change any DEP settings. If an update or DEP-compatible version is available, we recommend installing it and leaving DEP turned on so you can benefit from the protection it can provide. But if the publisher has not released an updated, DEP-compatible version of the program, you can turn off DEP for the program. You'll be able to use the program, but it might be vulnerable to an attack that could spread to your other programs and files.

 

For best security, it is advisable to leave the DEP turned on, unless otherwise required to turn off the same.

a. user say:

Hi,

Thank you for your interest in OneNote.

Yes, it is possible to have the Notebooks stored in cloud and have them all saved locally as well. This doesn't require you to have internet connection and doesn't work by downloading them from cloud. Rather, it uses OneNote cache to save it locally and you can access them offline.

The "original" or "master" is where you created it; local on your computer, on a network drive or on OneDrive. a copy of it for offline purpose actually exists on your device, a part of big file called OneNote cache.

You cannot have it only on OneDrive and make them available when you are working offline. OneNote cache file will definitely take space from your mobile device.
