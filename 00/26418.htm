Somehow the document has become enclosed in a form field. In a copy of the document, carefully select all the text in the field and copy it (Ctrl+C). In a new empty document, paste the copied text.

  2. Or, if you have the Developer tab displayed (it is not by default), you can click Restrict Editing in the Protect group and click Stop Protection.

High priority mail

toby ornott say:

Try this:

  • Create a test folder High priority
  • After it has been created, then create the rule: When the message arrives and it matches all of these conditions: it is marked with an importance HIGH.  Do all of the following: move the message to the folder "High priority".  Stop processing more rules.

This then ought to move anything marked "high priority" out from your Inbox to the test folder.  As messages arrive, you can then separate the good from the bad. 

If the task of manually eliminating the good from the bad becomes annoying, you can try modifying the above rule to add an Exception to retain the good in your Inbox but I suspect that that may involve more manipulation than it's worth.
