mike h.. say:

Hi,

Try this macro. Change the bold line to add the other worksheet names ensuring they are separated by a comma.

Sub orange()
Dim ws As Worksheet, s As String
Dim v As Variant
s = "EPI,IDHNR,SBHDR"
v = Split(s, ",")
For Each ws In ThisWorkbook.Worksheets
    If Not IsError(Application.Match(ws.Name, v, 0)) Then
      ws.Tab.Color = 49407
    End If
Next ws
End Sub

Hi,

 

Put this code in a general module and attach a shortcut to it

 

Sub Goto_Sheet()
Dim errmsg As Label, x As Long
On Error GoTo errmsg

Sheets(Format(x, "000")).Select
Exit Sub
errmsg:
MsgBox "Invalid sheet number"
End Sub

brian tillman [outlook mvp 2007-2019] say:

There's an issue at the Outlook.com server that if you try to send from an alias address through the Outlook.com server using a mail client like Outlook 2013, it will always change the sending address to be your primary address.  It is a problem Microsoft has known about for some time now but hasn't addressed yet.

 

One way to solve it is to change the outgoing server for the account in Outlook 2013, using a different service's SMTP server.  Some services (like gmail) allow you to send mail through their servers using a non-gmail account.  Since the recipient will then see the alias address, your Outlook.com account will receive any replies they make.

There are 2 fundamental considerations:

  • Multiple file are being opened because of an OS X setting in Lion & later not specific to Word. Among numerous other threads on the same issue see When I click on a word file, word opens all the files viewed earlier and then opens the clicked file
