bridged network connection questions

Per this article , you can use a bridge to make a computer with two network cards (presumably wired and wireless) act as a wired hub or switch.  That's useful enough for wired connections, since they actually make use of hubs and switches, but what about a wireless connection?  There are repeaters, access points, and so forth.  What component of a wireless network does a bridged connection most resemble?

Also, when I try to bridge two network connections, I get a "The software you are installing for this hardware has not passed Windows Logo testing" message.  The hardware name it gives is MAC Bridge Miniport.  That seems odd since, presumably, a bridged connection isn't going to be making use of any new hardware for which drivers haven't already been installed...

shawn - support engineer say: Hi terrafrost,

Thanks for posting. Below are your questions and answers.

Q. There are repeaters, access points, and so forth. What component of a wireless network does a bridged connection most resemble?
A. A bridge is set up a little different than all of these. All the wireless networking components you mention can handle more than one connection passing through, think of this as an extension cord for a power cable.


Q. I get a "The software you are installing for this hardware has not passed Windows Logo testing" message.
A. This is a generic error based on the driver. You can click "Ok" through it.

Q. The hardware name it gives is MAC Bridge Miniport. That seems odd since, presumably, a bridged connection isn't going to be making use of any new hardware for which drivers haven't already been installed.
A. The reason this happens is because it creates a virtual hardware device on the fly. Even though it looks like it's just part of XP it is actually simulating a piece of hardware.

Hope this helps!

why is explorer.exe running my cpu at 95-99% - Microsoft Community

ijustwanttoaskaquestion say:

azda2 say: To determine that several other details would be needed.

  • What Operating System are you using? (ex. Windows 2000, Windows XP, Windows Vista, Windows 7)
  • What type of processor is in your computer? (i.e Pentium IV, AMD Athlon, Pentium 2 Duo, AMD Phenom, etc.)
  • What kinds of applications are you using?
  • The number of startup programs and applications that are installed/in use on your computer

If you are using a single core processor with Windows Vista or Windows 7 then the amount of processing power in use by your computer when you computer is idling will already be fairly high. I own an older Dell desktop that has a Pentium IV 2.93GHz that idles Windows 7 around 40% (with 31% of it going towards explorer.exe.). Windows Vista requires higher specs than Windows 7 so I imagine that it would use even more processing power. Since this is a Windows XP thread I would assume you aren't using either Vista or Windows 7 so that shouldn't be too big of an issue but your computer will still idle high if you have a lower end processor (Single Core 1.8GHz or less).

The kinds of applications you use and how many you have open will increase the amount of processing power your explorer.exe uses as well. All of your programs work together with explorer.exe, it's the user interface that you click (a.k.a. your start menu, your task bar, etc.). So whenever you open a new program or your program uses more memory so does your explorer.exe. If you have a lot of startup programs on your computer (which most people do without even realizing it) then that too will contribute to having high processing power going towards explorer.exe.

Somethings that you can try to reduce the amount of processing power used by you computer are...
  1. Uninstall programs you don't use.
  2. Defragment your hard drive on a regular basis.
  3. Go to "Start" then "Run" and type 'msconfig' go to the startup tab and uncheck startup programs that you don't need. (Be careful while doing this as there will be some programs you will want to have start up when your computer boots such as your anti-virus).
  4. Reduce the number of applications you have open at a time and the number of tabs you use on web browsers. 
  5. Whenever using applications that require high computer specs (video games, videos, etc.) close other applications.
Sometimes there isn't anything you can really do about the high processing usage on your explorer.exe, for example whenever you open an application your computers processor usage will spike as it devotes more resources towards opening the application. Or programs that you love to have/use on your computer will use a large amount of your processor (which will reflect in your explorer.exe).

Malware and Spyware are a couple of other things that are pesky and can slow down your computer so make sure you have an antivirus installed if you do not already.
I would recommend avast! or AVG if you are just a casual PC user as they are both free to use and work very well.

Hopefully this helps a little! :D
