vikasmalik1 say:

Microsoft Excel

How to open multiple hyperlinks in Excel in a click in Macbook OS-X High Sierra?
alex wang msft say:

Hi Vikas

There is no direct button to open all the hyperlinks in Excel, but you can write a macro with VBA to do this.

I tested the macro from this thread and it can open the hyperlinks when you run the macro: Does anyone know how to open multiple hytper links in excel at once?

Here is how to do this on a Mac:

1. Save your workbook as .xlsm file

2. Click Tools on the top menu > Macro > Visual Basic Editor

3. In the VB editor, right click the sheet that has the hyperlinks > View Code

4. Paste the code and save:

Sub OpenHyperLinks()

    Dim xHyperlink As Hyperlink
    Dim WorkRng As Range
    On Error Resume Next
    xTitleId = "OpenHyperlinksInExcel"
    Set WorkRng = Application.Selection
    Set WorkRng = Application.InputBox("Range", xTitleId, WorkRng.Address, Type:=8)
    For Each xHyperlink In WorkRng.Hyperlinks
        xHyperlink.Follow
    Next

End Sub

5. Clcik the Run  button to run the function names OpenHyperLinks

6. select the range of your hyperlinks

then all the links will open in your default browser.

You could also save the function as an Excel add-in for futuer use: Create custom functions in Excel.

mike h.. say:

Hi,

I have assumed you want to continue with holidays even if weekends are worked. Ty this ARRAY formula. For speed I would shorten that holiday range from a full column.

A letter y in d5 makes it perform the alternative calculation

=IF(D5="y",G5+(SMALL(IF(ISNA(MATCH(G5+ROW(INDIRECT("1:"&F5*10)), Sheet2!A:A,0)),ROW(INDIRECT("1:"&F5*10))),F5)-1)/E5,WORKDAY(G5,(F5/E5)-1,Sheet2!A:A))

This is an array formula which must be entered by pressing CTRL+Shift+Enter
