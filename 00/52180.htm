lemp say:

What does "wireless would not configure" mean?

Why did you disable DHCP and why did you select 10.0.0.8 as an IP address?  It would be pretty unusual for a home network to use a Class A private network.  Most home routers use 192.168.x.0 - 192.168.x.254, where x is 0, 1, or 2.  Are you attempting to connect to a school or corporate wireless network?  If so, don't bother with the 11-step process described below and get help from the organization's Help Desk.

Why did you enable TCP/IP filtering and then select "Permit all"?  I suggest not enabling TCP/IP filtering at all.

You never actually connected to your wireless access point.  The lights, icons, and popups in the System Notification area are quite misleading in this regard.  Nine times out of ten, the symptoms you describe are caused by a failure to authenticate -- i.e., your wireless encryption credentials are not being accepted or you are being blocked by some filter configured on the wireless access point. 

I suggest the following:

  1. First, connect the laptop to the router using an Ethernet cable and confirm that you can connect to the Internet.  This will establish that the basic networking software (e.g., TCP/IP stack, Winsock, etc.) is working correctly (important given your attempts to edit the registry).
  2. Re-enable the Wireless Zero Configuration service that you disabled if you had not already done so.
  3. Open the Properties dialog for your wireless network adapter, select "Internet Protocol (TCP/IP)" and click the Properties button.
  4. On the "General" tab of TCP/IP properties, select "Obtain an IP address automatically" and "Obtain DNS server address automatically."  Click OK
  5. On the main Properties page for your wireless network connection (where you go when you click the OK button in step 2), confirm that the list of items includes only
    • Client for Microsoft networks
    • QoS Packet Scheduler
    • File and Printer Sharing for Microsoft Networks
    • Internet Protocol (TCP/IP)
  6. Click the "Wireless Networks" tab
  7. Confirm that the box is checked to "Use Windows to configure my wireless network settings."  Note:  if you have been using a Dell or Intel utility to configure the wireless adapter, determine how to disable that utility in favor of Windows before you begin this exercise.
  8. Remove all entries in the list of "Preferred networks" (if there actually were entries in this list, please make a note of that fact in your Reply).
  9. OK your way out of the wireless network adapter properties dialog.
  10. Log in to the router's configuration utility and disable all wireless encryption and any other wireless security methods (e.g., filters based on IP address, MAC address, or otherwise).
  11. Attempt to connect to your wireless access point.

If you can successfully connect in step 11, then you can go back into the router's configuration utility and reconfigure your encryption password.  You'll have to enter it when you next attempt to connect.

First you will need to force your machine to get into the recovery screen.
To do this boot up and wait for the blue windows screen with the spinning circle, when you see that press and hold power button to shut down.
Boot up again and do the same thing by waiting for the blue windows screen with the spinning circle then shut down, power up for the 3rd time and you should get repair screen.
Let it run.
You will then get a screen with the Automatic Repair, select the Advanced Options.

Go to the command prompt.
Type the following.

bootrec /rebuildbcd

Meanwhile, you may try this work around and check if you are able to import all the Photos from your phone to computer.

  1. Connect your Phone to your Windows 10 PC using the compatible USB cable.
  2. Open Windows File Explorer. Press the Windows + E keys on the keyboard to open File explorer.
  3. Under “This PC”, you should see your Phone as a drive. Double-click the icon for your Phone here. Now you may have the access to your photos/folders in your Phone.
  4. Photos and videos taken on your Phone should now be visible. Then you can select the photos to transfer/download them to Windows 10. (Copy and paste the photos which you need)
