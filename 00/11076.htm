Visio 2007 toolbars

sekar r - microsoft support say: Hello,

Visio saves customization that you make to the toolbars in a file called "custom.vsu" and it is located at C:\Documents and Setttings\<username>\Application Data\Microsoft\Visio. Copy custom.vsu file to same location onto your target machine. This will make your custom toolbars available on target machine as well. Some other toolbar settings like location are stored in following registry key: HKEY_CURRENT_USER\Software\Microsoft\Office\12.0\Visio\Toolbars. You can export this registry key and import it on target machine if you want to see similar location settings. Please let us know if you need steps to export the registry key.

Hope this helps.

Thanks,

INDEX, HLookup and VLookup formulas - Microsoft Community

alicecastrorv say:

INDEX, HLookup and VLookup formulas

Confused about how to create formulas in INDEX, HLOOPUP and VLOOKUP.  If I want to create a formula from the following list, I need to understand what the zero after A2:A5 represents.  Here is a copy of the list:

               

                            A                                                                   B

               1    Product                                                             Count

               2    Bananas                                                               38

               3    Oranges                                                                25

               4    Apples                                                                   41

               5    Pears                                                                     40

               6    Formula                                                           Description (Result)

               7     =INDEX(A2:B5,MATCH                                  Looks up Pears in Column A and returns the value for Pears in Column B

                       ("Pears",A2:A5,0)2)                                        (40).

 

             The arguments are as follows: 

              1  A2:B5  The entire range in which you are looking up values

              2  MATCH("Pears",A2:A5,0):  The MATCH function determines the row number.

              3  "Pears": The value to find in the lookup column.

              4  A2:A5.  The column for the match function to search.

              5  2:  The column from which to return the value.  The leftmost column is 1.

 

       I just need to understand why there is a zero (0) after the A2:A5 in the formula.  Any help with this will be greatly appreciated.

 

 

hansv mvp say:

The 0 as 3rd argument in the MATCH function specifies that you're looking for an exact match of "Pears".

If you omit 0, MATCH will return the index of the (alphabetically or numerically) largest value that is less than or equal to the search value. The lookup range needs to be sorted in ascending order for this to work.

You could also have used

=VLOOKUP("Pears",A2:B5,2,FALSE)

Here, 2 indicates the column to return a value from, and FALSE that you're looking for an exact match. If you omit FALSE, it acts the same as with MATCH without the 0.

jeeped say:

...   I just need to understand why there is a zero (0) after the A2:A5 in the formula.  Any help with this will be greatly appreciated ... 


Actually, you have mixed and matched the bits and pieces of a VLOOKUP() function and an INDEX(MATCH()) pair together. An INDEX() function looks up a single column (first parameter) and returns the value provided by the row referenced by the second paramter, typically provided by a MATCH() function. A VLOOKUP() combines these two by using a range that references two or more columns, matches the first column and returns the value referenced in the same row from the nth column. Your formula could be expressed as either,
=VLOOKUP("Pears",$A$2:$B$5,2,FALSE)
... or,
=INDEX($B$2:$B$5,MATCH("Pears", $A$2:$A$5,FALSE))
... for an exact match (e.g. FALSE). A boolean FALSE value has a numerical value of 0 (e.g. zero). This is the parameter you are referring to in your question. A value of TRUE or 1 (e.g. one) indicates less than and a -1 (e.g. minus one) will be greater than. for the MATCH() function. For VLOOKUP() a TRUE value is noted as an approximate match. If you use a value other than FALSE, the column must be sorted.
