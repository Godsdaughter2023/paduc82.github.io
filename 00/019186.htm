Determining that all headers are empty (Word VBA) - Microsoft Community

la lawyer say:

Determining that all headers are empty (Word VBA)

What is the VBA code to determine if all headers are empty? (Word 2010)
rich michaels - mvp (greatcirclelearning.com) say:

Here is VBA code that checks.  I just have a debug.print command in the test sample and you'd have to replace that with what specifically you wanted to do when a non-empty header is found.

Sub CheckHeaderParagraphs()
    Dim doc As Word.Document
    Dim sec As Word.Section
    Dim hdrRng As Word.HeaderFooter
    Dim para As Word.Paragraph
   
    Set doc = Word.ActiveDocument
   
    For Each sec In doc.Sections
        For Each hdrRng In sec.Headers
            If hdrRng.LinkToPrevious = False Then
                For Each para In hdrRng.Range.Paragraphs
                    If Len(para.Range.Text) > 1 Then
                        Debug.Print para.Range.Text
                    End If
                Next para
            End If
        Next hdrRng
    Next sec
End Sub

With Range("H2").End(xlDown)
