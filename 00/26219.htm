john mcghie say:
Arshad:

Please read the question before answering.  Neither of your suggestions apply to the question.

Era:

This question is actually "interesting".  You need to convert the dynamic "generated" numbers in your table column into "ordinary text" fixed numbers.

The simplest way to do this is to do the following:
  1. Select all of the cells in the numbered column
  2. Copy
  3. Move to somewhere outside the table
  4. On the Edit Menu, choose Paste Special and then select "Unformatted Text".
  5. The content of the column will be pasted into the document with the numbers converted into normal text.
  6. However, between the number and the text will be a tab character, which we have to remove before you can paste the text back into the table.
  7. Select all of the pasted text.
  8. On the Edit menu, choose Find>Advanced Find and Replace
  9. In the Find field type "^t" without the quotes.  The t must be lower-case.
  10. In the Replace field, type a space (or two).
  11. Click Replace All
  12. Say "No" when asked if you want to do the rest of the document.
  13. Make sure the text is still selected and Copy
  14. Select all of the cells in the table and Paste
  15. Check it is all OK, then remove the working text that you pasted elsewhere.
There you do: it's actually quicker to do it than to read it :-)

Hope this helps

It's not neat but you could do this. Put this code in a 'general module' and attach a keybpard shortcut.

Sub MyQuartile()
Dim MyRange As Range
Set MyRange = Selection.Offset(, 1)

    Range("D2") = WorksheetFunction.Quartile(MyRange, 1)
    Range("D3") = WorksheetFunction.Quartile(MyRange, 2)
    Range("D4") = WorksheetFunction.Quartile(MyRange, 3)
    Range("D5") = WorksheetFunction.Max(MyRange)
End Sub

graham mayor say:

I have checked in Word 2007 (fully updated) running under Windows 10 and both the standard shortcuts appear to work correctly here.

You can test the current status of a selected field with the following macro which reads the status, reports it (If locked it will show True) then toggles the status and reports the status again. Run the macro twice to set it back as it was originally.

Sub Macro1()
Dim orng As Range
Dim oFld As Field
    Set orng = Selection.Paragraphs(1).Range
    For Each oFld In orng.Fields
        oFld.Update
        MsgBox oFld.Locked
        oFld.Locked = Not oFld.Locked
        MsgBox oFld.Locked
        oFld.Update
    Next oFld
End Sub
