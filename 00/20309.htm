Review: Windows 10 build 10049/10061 - Microsoft Community

andre da costa say:

Review: Windows 10 build 10049/10061

Technical Level : Basic

Summary

With Microsoft’s annual developer conference Build fast approaching; I thought I should take some time out and talk about my experiences with the latest Windows 10 builds. It’s been an interesting start to 2015, before the new year arrived Microsoft took a break issuing builds while they tightened up things in anticipation for a more polished build presented at a Preview Event in January at the company’s Redmond campus. This build turned out to be 9926 and it stayed with us for quite some time. So much so it got a lot of Windows Insiders disgruntled wondering what was happening and why they were not seeing a consistent flow of newer builds. Microsoft finally relented and issued its first build since January, build 10041 at the end of March, almost two months after that long drought. Microsoft followed up with an even more refined build, 10049 which showed further UI work and inclusion of universal apps and the new Spartan web browser.


Details

Since then, the drought looked like it was making a comeback, not for long though. The company soon followed up with another build, 10061. With BUILD less than couple days away (as of this writing); we are likely to see an even more polished feature set. Since then, news has leaked that Microsoft will likely launch the milestone RTM (release to manufacturing) build at the end of July 2015. This is both exciting and worrying to many since Windows 10 even though it has come a long way since October 2014 still feels extremely rough around the edges and numerous complaints and reports I and fellow Windows Insiders continue experience leave me with a feeling of, I hope this major release is not being rushed. Honestly, I don’t think it should be, even if Windows 10 had to be delayed until summer 2016, I personally would prefer that. Windows 8.1 has matured nicely and I am sure Microsoft could work on some extras to deliver between now and fall that could rejuvenate some interest in Windows devices expected later this year.

Click here for a larger size.

SETUP

Windows 10 build 10049/10061 so far have only been distributed through Windows Update. Personally, I find the Windows Update mechanism for newer builds disastrous. I think the last build I had successfully updated to through Windows Update was 9860. Microsoft had provided .ISOs for every build up to 10041, but it looks like they are skipping it for build 10049/10061. I personally attempted to update from 10041 through Windows Update and it fell flat. It successfully downloaded, but after a restart and booting into a new setup experience, it remained stuck at 30% and wouldn’t move. Because bandwidth is not a luxury where I live, I decided that I am not going to waste anymore and just downloaded a .ISO file for builds 10049/10061 someone created from a ESD file and uploaded to a torrent. It successfully downloaded in 20 minutes. I honestly did not want to do it, but I had no choice. In less than 15 minutes I had build 10061 up and running from a build to build upgrade using the ISO file compared to the more than 3 hours it would take using Windows Update. I am not the only one who has experienced this fiasco and seriously it concerns me. I personally believe Windows Update was just never designed to deliver software this way.

Windows 10 shows its new slicker setup experience.

The fact that the download through Windows Update takes a considerable amount of time and this is a compressed file that is expanded after it is downloaded, then it has to reboot and actually install the build which can take anywhere between 2 to 3 hours; It’s just unforgiveable knowing this is not a guarantee to work well. This is definitely reminding me of the update experience many faced when updating from Windows 8.0 to Windows 8.1 and even Update 1. Microsoft needs to have an option built into the new Windows Update mechanism to pause the download, suspend, resume and let users download it at their own pace. So they don’t have to spend every waking moment to check if it downloaded or not. The advantages of the Windows Update method are lost between downloading and installing. Microsoft should just push this release as a .ISO and give users instructions how to create a bootable copy on a DVD or USB thumb drive.

A participant on the Windows 10 Insider Forums discovered the best way to work around this bug is to disconnect from the Internet (wired/wireless) after the download reaches 100% then let the installation proceed. Some users are facing issues with finding the update even though they are on the Fast branch. A work around for this is to reset the software distribution folder. This is not something I think will be considered a desirable experience when migrating to Windows 10. Considering that many Windows users are contented with Windows 7 and those who went through Windows 8.0, 8.1, Update 1 are not that enthused for another round of frustration and troubleshooting. So that’s setup in a nutshell, still a work in progress, still unreliable, still not recommended.

OOBE

Other aspects of Windows 10 still remains disappointing. There are a lot of users who do not want to sign in using a Microsoft Account. Microsoft needs to relent on this and let users have the clear choice of signing using a Microsoft Account or a Local Account. The Local Account option is too hidden resulting in complaints from users they are being coerced into a Microsoft Account. Don’t get me wrong, I like using my Microsoft Account with Windows and I see the clear benefits, at the same time, there are a lot of users who simply don’t want this functionality no matter how great it is. Another issue users are reporting is the inability to create a user account. The only work around for this is to initiate a clean install (for some, this means starting from scratch with build 10041 since there is no ISO for 10061). I personally have not experienced the problem, but it’s a common occurrence for many on the Windows Insider forums.

Yep, you still have to go through with this. The Windows Team could at least do a 1 minute tour.

Why can’t the Windows Team move installing modern apps portion from OOBE to the setup phase? I find it absolutely annoying that I have to go through this waiting period every time I install a new build of Windows 10. Does Microsoft really expect users to always go through this provisioning every time a new build of Windows 10 is released even when the product has launched? At least install them in the background and let the user reach the desktop so they can start using their computer. Microsoft Office 365 has a clever capability as part of its Click to Run Technology where you can use parts of the suite while the rest downloads and installs in the background, I recommend the Windows Team look into this for Windows 10.

Since build 9926 Windows 10 includes a Rollback feature as part of its Update and Recovery tools; which allows installations of Windows 8/8.1, Windows 7 and Windows 10 builds to be rolled back to a previous version or build. I tried it out with Windows 7 and it works as intended. Any apps installed after upgrading to Windows 10 though will be removed. It’s a handy feature and will be an important layer to troubleshooting installations of Windows 10 that might fail. I still recommend you create a thorough backup before upgrading though (System Image). These are some of the niceties of the OS I like which I honestly believe Microsoft should have delivered as far back as Windows Vista.

DAILY USAGE

Windows 10 build 10061 features minor improvements and if you have been diligently updating to each build, there is not much to see on the surface. Coming from Windows 7 or even Windows 8, the OS feels like a familiar but dramatic departure. Build 10049 reintroduces the Network Fly out (for Windows 7 users, View Available Networks). This function is definitely a convenience, since past builds required opening Settings and connecting to Wireless Networks from there. Network Fly out is quite buggy though and does not reveal available wireless networks on initial click. Usually there is a delay or you have to toggle the Wi-Fi tab on and off until your network connection appears. It also lacks functions like the ability to define a connection as metered. You can of course access this function from the familiar Show Available Connections Pane in Settings. I hope by the next build these limitations are fixed.

Significant improvements, Network Flyout makes a return, with a vastly updated Notification Center.

Notifications sees some refinements and basically works as intended. I am able to quickly view and read incoming email, view app notifications and information about system updates, although I notice those are not working when I click on them or if it is the intended behaviour. Notification Centre is a useful addition to Windows and was long overdue, its centralized and definitely makes it easy to remember where to check for common task users will often skip or forget about. Remaining in the Notification Area, there are still some issues that exist, the Battery Notification itself is buggy and I notice there seems to be a bug in this build where my computer is not automatically hibernating and an issue with a battery not available for use. Users will notice that Windows Defender now makes a return to the Notification Area for quick access. There is still no context menu for scanning files or folders. Either way, it’s not a big deal since I will be replacing Windows Defender with a more effective third party Antivirus utility.

START

Start is slightly improved over previous builds, featuring improved visuals, apart from that, the only noticeable changes is the addition of default locations for File Explorer, Settings and Documents. There is still no option to move the Cortana Search within the menu itself which would be a better option for persons might not want to have the Search box always present on the Taskbar. The All Apps menu has not changed either, featuring the alphabet layout of programs and groups, I personally preferred the layout in build 9841. A Customize option remains curiously greyed out from the Taskbar and Start menu properties which probably suggest there are more options to come. Since build 10041 I have started using the Start in full screen mode more often, with build 10061, I have moved back to a semi Start menu like experience where I get the best of both worlds. I never had a problem with Windows 8’s Start Screen, I just wish it had embraced the parts of Windows that made it a success, such as the Taskbar which is always on screen, quicker access to your application library and the ability to easily manage and customize modern desktop apps. The Start menu improvements in build 10061 are welcome, it’s now resizable, features quicker access to Power Options in the lower bottom of the Start screen and a slicker design with an option for transparency and a variety of colour choices available in Settings > Personalization.

I miss Jump list on the Start menu, a feature introduced in Windows 7, the only place you will find it now is on the Taskbar and its limited to classic desktop apps only. I really would have love to see this feature make a return. The new modern Settings app includes the option to pin frequently accessed Settings on the Start screen itself. I do not like the fact that when you do this though it just pins a generic cog on the Start screen. Some uniqueness would be nice, but I guess with time not being a luxury, the designers can’t bother to create a unique pictorial representation for each setting.

You can easily rearrange and create groups on the Start Screen. I like how it’s much easier to rename them too, just hover at the top of a group and a field box will be displayed indicating the ability to name or rename a group. I also like the vertical layout instead of Windows 8/8.1 horizontal menu or the semantic zoom if you are still running Windows 8.0, makes it feel more familiar and approachable. Focusing a bit more on Tiles, it looks like you are limited with what you can do in Windows 10. The option to turn off live tiles for instance is not available. This is an option I think is necessary since it’s a bandwidth leaker and since I am on a metered connection, I would rather not have these apps downloading data in the background.

SETTINGS

In Windows 8/8.1 Settings went through a schizophrenic period in regards to configuration Tools. First it was a hidden PC Settings modern app that featured limited options for customizing your PC. Later it evolved in Windows 8.1 to add more capabilities lessening the need to access the desktop Control Panel. In Windows 10, you can pretty much find all you need without turning to Control Panel. First of all, its windowed, but it’s very stoic and focused. Cleverly broken down into nine categories consolidating the plethora of items into groups of logical areas of the system. Users will of course miss the flexibility of Desktop Control Panel (which is still there by the way) in this new Settings app. You can’t for instance view all settings which I think would lead to more confusion considering the wealth of tools the OS includes for adjusting your Windows configuration. The great thing is, there is almost a modern equivalent of what you would find in the desktop Control Panel. Of course some areas such as Power Options, Personalization still lead to areas of the classic Control Panel, but its minor. You can stay in settings for most task. Privacy is a lot richer in functions with the ability to limit what apps can access in a variety of areas which include Location, Camera, Microphone, Contacts, Calendar, Messaging, Radios, Account Info, Other devices, Feedback and Speech/Inking and Typing.

SEARCH

Search is an integral part of Windows, the company first demonstrated this commitment with Windows Vista which included Instant Search. Windows 10 includes a comprehensive solution that is very impressive an example is the new Microsoft’s Digital Assistant Cortana first featured on Windows Phone 8.1. Although Cortana is a big part of using Windows 10, I am limited by my bandwidth, so my experience with Search is specifically around local activities. Local Search breaks down search results into several categories: Documents, Apps, Settings, Photos, Videos, Music and Email. Search also splits your results between those on your local drive and OneDrive.

Search in Windows 10 litterally puts it all at your finger tips.

APPs

Windows 10 introduces a wider selection of modern apps, many of which end up replacing traditional desktop applications. The first one I discovered was Calculator, now a modern app, the experience comes across disconcerting at first, but eventually you accept it. It provides the middle ground between a mouse/keyboard and touch optimized experience. I am also pleased it launches just as fast as the old Calculator, I was starting to worry when I encountered it in previous builds after seeing a splash screen each time I launched the app. The benefits of the modern app model with an app such as calculator its more flexible, you can actually maximize it into a full screen app for instance. The app also shows off some of the changes to modern apps in Windows 10, a hamburger menu provides quick access to settings of the app so you can on the fly switch to Scientific, Programmer and Converter modes. Not only is it touch friendly but works just as great with a mouse.

Music and Video were updated and pushed outside of build 10049 through the Store beta. The apps feature significant improvements and I can safely say, it’s time to retire Windows Media Player and forget about using third party apps like VLC. The great thing about Music in particular it actually looks and works like a modern juke box. Of course, its touch optimized, but I feel right at home in the app. Music is of course part of Microsoft’s Cloud mantra and the app integrates with Microsoft’s Cloud Storage OneDrive, you can sign in and access your synced music library from OneDrive itself. The app features standard juke features such as music categories for your library: album, artiste, songs, now playing and options for creating play list or importing from third party apps such as iTunes. At the bottom, you will find common player controls for scrubbing through a track, play, forward, volume, shuffle and repeat. Of course, Music app reveals some of the minor flaws of Modern apps in Windows 10, the lack some of the truly small but unique features of Windows 7 such as Jump List support and Live window controls such as (fast forward, play) you would find in an app such as Windows Media Player.

Windows 10 features two come backs now bundled with the OS, Mail and Calendar. Versions of Windows dating back to Windows 95 included a Mail client. Windows Vista was the first to include a Calendar app. Windows 7 removed these apps and bundled them as part of the Windows Live Essentials package you had to download. The new mail and calendar apps are clean in design and are easy to use. Account setup is quick and easy especially if you are already using a Microsoft Account with your Windows 10 setup, just connect and that’s it! Mail allows you to add accounts for Microsoft’s popular enterprise mail server Exchange or other popular third party services such as Google Mail or even Apples iCloud. While reading this, you will notice that both Mail and Calendar are separate modern apps. Calendar is not an integrated part of the Mail app for instance, unlike Outlook which includes an integrated calendar. If that is something you like, then Outlook will remain the best choice. Although I don’t see it as a disadvantage, sometimes all you need access to is just your Calendar. Both apps are touch optimized and features large icons that will be more friendly for Tablet users. They work just as well with keyboard and mouse.

In celebration of its 25th anniversary, Windows 10 also sees the return of the popular time killer, Solitaire. Windows 8 did not come with built in Games like previous versions of Windows and I believe it riled up a lot of Windows users who loved having their favourite card games. Although you could always download these from the Windows Store, I believe it was lost on persons they could do so. To be honest, I launched it but it quit instantly, so there seems to be some work to be done before its fully ready.

Photos app now includes the album feature, but crashes immediately after opening. So that’s my time with Photos.

OFFICE for TOUCH

Windows finally sees a modern touch optimized version of Office and its only for Windows 10. If you are using Windows 10 on a desktop/laptop form factor with no touch support, you are not missing anything. The apps can be downloaded from the Store beta. Only Word, Excel, PowerPoint and OneNote (which comes bundled with Windows 10) are supported. Makes sense, I can’t picture Project, Visio and Access having a pressing need to be touch optimized. The apps are extremely basic and features just the essential features necessary for a touch environment and that’s the way it should be. The key here is document fidelity that is optimized to work with touch. If you have content to prepare that requires the precision of a mouse and keyboard, desktop Office remains the best choice. If you own a device like Microsoft’s Surface 3, I am sure these new revisions of Office apps will be a welcome.

Project Spartan

Project Spartan is Microsoft’s next generation web browser that will succeed the 20-year-old Internet Explorer. Internet Explorer 11 will be the last version of the web browser, but Microsoft will continue to maintain it for legacy purposes, knowing that Project Spartan will not feature the full compatibility of IE that has been specifically targeted by many web sites and also thousands enterprise Intranet websites. Project Spartan is based on a new rendering engine called edge.html. The promise of this new web browser is no compromise, allowing web developers to easily build sites just as they easily do today for competing web browsers such as Mozilla Firefox and Google Chrome, both of which have nearly halved Internet Explorer’s market share over the past 10 years.

I am not going to go into the technical refinements, but discuss more about my experience using the web browser on a daily basis like I would IE, Firefox or Chrome. First impressions, it has a long way to go. In fact, when I was using it in build 10049, both it and Internet Explorer had a serious virtual memory bug that caused Windows 10 to hang up requiring a cold boot. That issue seem to have been resolved in build 10061, but the browser is severely lacking functionality in comparison to its predecessor and other third party web browsers. I notice the app stays resident in memory just to give the illusion it loads fast. I also notice it displays a splash screen, something I have not seen since Internet Explorer 5.5. Another thing I notice it lacks some customization which I hope to see fix before the OS launches. For instance, the Favourites bar can’t be customized, so links I commonly access are pushed off to an overflowing menu, but I can’t drag them around. The browser does feature some improved capabilities, for instance, you can on the fly adjust settings which displays a quick sidebar where you can do things like turn off flash, quickly set your home page and adjust cookie settings. The browser though failed when I noticed I had left my PC for a while. When I returned to use it, all the tabs I had open were missing and the app simply reloaded the home page.

Microsoft's successor to Internet Explorer; Project Spartan - still a work in progress.

Some keyboard commands like F6 no longer work. Even when click in the Address bar, I can’t select all the text using keyboard or mouse. Organizing areas of the web browser such as Favourites is quite limited, although I do like the new consolidated view where you can access not only Favourites but your reading list, downloads and Internet History.

How does it do in web browsing? The Spartan browser functions quite fine in this regard, I am able to access and view many popular web sites such as Google News and all the many tech and news web sites I visit daily. I do notice some web site controls such as comment sections do not load properly or not all, in particular Disqus comment systems will fail or not work.

Modern apps after have come a long way from their immature days on Windows 8.0. They are usable and you can go through a full day without experiencing any problems in ‘some’ of these apps. Understanding that Windows 10 is still in development, I am sure a lot of work is being done to make them more robust and ready to truly succeed desktop apps. Of course, with crashing I am seeing in apps like Photos and limited capabilities of Project Spartan, two apps I would describe as being essentials to any computer use, there Microsoft should put significant human resources to ensure these two are as full featured and functional as any classic desktop app. Modern apps need to also support some of the missing classic features of Windows 7 like Jump List and Live Preview controls. For instance, Project Spartan does not support Tab previews from the Taskbar like Internet Explorer which is strange and there is no Private Mode.

FILE EXPLORER

Not much has changed in File Explorer. The Favourites link Navigation Pane is now called Quick Access. When you launch File Explorer, it defaults to Quick Access view which displays your recently created or accessed folders, along with a collection of your most recently accessed files. You can of course change this from the Folder Options. This PC introduced in Windows 8.1 sees some light changes. Storage properties displays a revised visual donut chart instead of the old 3D pie chart we have been seeing since Windows 95. Users will also notice a new icon set with some describing the look reminiscent of defunct 90s OS BeOS and a Recycle Bin that curiously looks like it came off the Windows 95 desktop.

File Explorer in Windows 10

Conclusion

This is just the tip of the iceberg. There is certainly more areas I could have covered, but six pages of content seems like a lot of cover for a still evolving OS. Windows 10 has a lot riding on it. Summer seems like a very ambitious schedule to launch what some are describing as a make or break it release. Since October, I have been through more downs than ups and I know I should be understanding that this OS is still under heavy development. With rumours speculating a possible launch at the end of July, I can’t picture Windows 10 possibly going out with everything in tip top shape, considering there is still a numerous amount of unfinished UX work, along with a still high rate of unstable code throughout the OS. The distribution experience in particular is a concern; I hope the Windows Team surprises me. Windows Update experience is one word: terrible, not just distributing the code, but the lack of flexibility. Many are assuming maybe Windows 10 will launch in a barebones state with features and apps delivered through Windows Update and the Store over time, likely repeating the same ‘it gets better with time’ philosophy of Windows 8/8.1. Something users surely won’t tolerate.

The most important audience to sell this OS to are developers and that is where BUILD comes in. Windows 10 needs to be an enticing story, the idea of Universal applications that span desktop, tablet and phone is an important strategy. The company has been toting this since Windows 8 launched but it has not come to fruition until now. The app experience is still woefully lacking on mobile for Windows Phone. Although I have not looked at that variant of Windows 10, it is critical to its success, since Microsoft is hoping both desktop and mobile can feed off each other’s success.

Windows 10 still remains between a rock and hard place. It needs to embrace its mobile future fully, but it can’t isolate a healthy market of nearly a billion Windows 7 users who the company is enticing with a free upgrade. A needed market to give this OS the clout it needs along with the dominance that so far has escaped its flagship Windows 8.1 release. Windows 10 though is in the right direction. I believe the Company is listening to the feedback, making it the OS that a Windows 7 user will feel right at home in while offering unique experiences on new form factors, there is a chance for Microsoft to reclaim some glory it has lost over the past 3 years, something it can afford to further loose.

Open the saved Excel spreadsheet.

Go back to PowerPoint

el_loco---01 say: I don't know what you mean by "best video players" . Each video player offers something different.

DIVX player offers playback of divx and xVid encoded files (AVI, XVID)

CorePlayer supports a lot more formats as do Smartmovie.

No player plays WMV or MPEG files because the WMV codec is not included in the device and for MPEG well a video file of good quality in MPEG-1/2 format would be just too big. A full length movie at QVGA in MPEG-1 is about 3 GB!

Since WMV is a Microsoft codec, Microsoft would need to licence that to Nokia the way they licencesed the WMA codec.

3GP quality. What do you mean?. 3GP is a container and as such it has no intrinsic quality attached to it. The quality of the video has to do with the codec, the bitrate, the resolution, the framerate and the encoder used. You can get pretty decent video quality while using a 3GP container by using MPEG-4 codec and a high bitrate and framerate.

About uploading, it depends on where you want to upload. Videos created by the phone can be 3GP or MP4 so as far as the service that you are uploading to supports these formats you should be able to upload videos. If by "uploading" videos you mean getting videos into your N95 then the answer is:, -it depends. If the videos you have are compatible with N95 then you don't need to transcode or convert.
