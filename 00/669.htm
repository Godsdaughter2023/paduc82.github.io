Hi Hyon,

 

You can setup only one account's password policy to never expire by using PowerShell. Please follow the steps below:

1. To get started, you need to download and install the Azure AD PowerShell module.

2. Connect to Windows PowerShell by using your company administrator credentials.

   Set-MsolUser -UserPrincipalName <user ID> -PasswordNeverExpires $false

Assuming you're not using the built-in heading styles for the headings in the Preface, there should be no problem. Just define a new multilevel list linked to the styles you're using. If it works as designed, it's not a bug. A mismatch in capabilities between versions is not a bug. Your link is about old versions of Word. The Standard toolbar no longer has an Insert Excel Spreadsheet command, as mentioned in the link. 

When I test copying and pasting a spreadsheet, I don't see any limitation to the number of cells pasted, nothing like the 5x10 limit mentioned in your link.

If you use Edit>Paste Special and Paste link to the Microsoft Excel Worksheet Object format instead of embedding it, you can change the displayed range anytime by following these steps:

  1. In Word, click on Edit Links.
  2. Select the spreadsheet link to revise.
  3. Click on Change Source.
  4. Select the original Excel file, then edit the Range/Bookmark field to your new values.
  5. Click on Open. The revised range displays in Word.

Thanks for the clarification. Using Insert>Object uses OLE, but OLE is not native to OS X and has limited support. There are many OLE objects in Windows for which there is no equivalent in OS X. As an example, in Windows you can insert a PDF using OLE, but you can't on a Mac. For this reason, I advise Mac users to steer clear of OLE when there is an alternative approach.

The Microsoft page does not refer to this as a bug and I did not refer to it as a feature.

It's not necessary to paste in external worksheets every time, I included that technique because you expressed an interest in revising the size of the sheet and that is one way to do it. You can use Paste Special and do an ordinary Paste on a Microsoft Excel Worksheet Object format. This is a true embedded worksheet that can be copied and pasted wherever you need it without leaving Word.

You could even create a variety of of preformatted spreadsheets and add them to AutoText. These can be retrieved with a keyboard shortcut. A workflow like this could significantly increase your productivity over using Insert>Object and having to format from scratch every time.
