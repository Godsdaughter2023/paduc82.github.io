I found the solution on another site, yesterday afternoon.  I am posting for others with the same issue.

1.  Go to "My Computer," then right click on "Manage."

2.  You may get a pop up that asks you if you are sure you want to make changes here, so click "Yes."

3.  The Windows Management console will open. Expand the "Services and Applications" at the bottom.

4.  Select "Services." The middle window pane will expand. Scroll down to find the "WebClient" service. Double click on it.

5.  Another window will open, "WebClient Properties." Change the Startup type to "Automatic." Go ahead and Start it too (click the Start button) if that does not automatically happen. Then save all your changes. 

6.  Close everything out, and then restart your computer. Now when you start Windows this will start this process as well.
