charles kenyon say:

It is possible, but you will have to either use mapped content controls or bookmarks and ref fields.

If you are using documents protected for filling in forms and legacy formfields, each formfield normally has a bookmark assigned, and you can change those in the field's properties. You would also want to check the box to "calculate on exit." Then, you use a cross-reference or REF field to display the formfield contents elsewhere.

Repeating Data (Populating Multiple Like Fields)

You can have bookmarks and REF fields without locking the document, but it is more complicated.

Mapped content controls allow you to have much more flexibility in your documents. However, you can only insert them, currently, using vba or by designing the form in a Windows version of Word.

Mapped Content Controls

I expect that this capability will be added to Word for the Mac at some point, but the ability to enter Contact Controls has been a part of Word for Windows since Word 2007.
