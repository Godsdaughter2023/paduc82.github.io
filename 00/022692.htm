suzy_g say:

You can disable AutoSave for each file that you want to only save changes manually using the AutoSave toggle at the top left of the application. Your preference will be remembered next time you open the file again. Visit this thread for instructions. 

If you accidentally make changes to a document, you can easily revert to an earlier version of the document using Version History, which you can find via the Document Titlebar.

Additionally, if you want to turn AutoSave off by default for all cloud files, you can do so through File > Options > Save in each app where you want it disabled. For example, in Word, simply uncheck “AutoSave OneDrive and SharePoint Online files by default on Word” and then restart for the changes to take effect.

When it’s turned off, you can manually turn on AutoSave by using the AutoSave toggle at the top left of the application window or the Alt + 1 accelerator key.

Please note that AutoSave refers to the new feature for Office 365 subscribers that enables a more seamless experience, including simultaneous real-time editing when working with files stored on the cloud. For files that don’t have AutoSave, such as files on your local hard drive, or for non-subscribers, there is still AutoRecover, which can help recover changes if you forget to save manually.

To learn more, please see https://support.office.com/en-us/article/What-is-AutoSave-6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5

mandyshaw say:
Firstly, your Nokia Lumia 800 is running the Windows Phone 7 operating system (not Windows Phone 8).

To update it, if there is an update available to it (which is dependent on the current software level and on your carrier). Your 'phone will notify you of the existence of an update. Install Zune on your PC, as a free download from zune.net (if not already done), connect the PC and 'phone via USB, and Zune should automatically kick off the update process. Note that any update may take some time (up to a couple of hours) to install & that you should not power off the PC or 'phone or disconnect the USB while it is happening.
If you have had no update notification, what version are you on now? 'Phone Settings>About>More information. Also, what is your carrier/mobile operator?
The 'phone will report 7.10.something as the 'OS version', which will actually mean either 7.5 or 7.8 (confusing, but that's how it is - see http://www.windowsphone.com/en-us/how-to/wp7/basics/update-history to decode). 7.8 is the highest current version of WP7, and given that WP8 has now been shipping for six months I would not personally expect any more significant updates for WP7.

Re contacts, these will automatically have been sync'd to your 'phone's primary Windows Live account. Go to contacts.live.com in your PC browser, sign on using that same Windows Live id, and they should be there.

Re messages, SMS messages cannot be backed up from Windows Phone 7, unfortunately. (Windows Phone 8 provides a limited backup capability for them.)
