david j parker say:

Visio Shape Data properties do not have a built-in setting for limiting the length of a text row. In fact, you can enter 32k character if you want. So, your only option is to use a trigger to shorten the text.

The following formula, if put inot a User-defined cell, will ensure that no more than 20 chars can be entered into a Shape Data row, named SomeText:

 

=IF(LEN(Prop.SomeText)>20,SETF(GetRef(Prop.SomeText),"="""&LEFT(Prop.SomeText,20)&""""),0)
