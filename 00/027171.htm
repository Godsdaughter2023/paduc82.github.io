=OFFSET(INDEX(E3:J3,MATCH(F9,E3:J3,1)),1,0)

I would do this slightly differently. Since you are clicking on button on a form, then I would add an unbound control on the form to return the month. I generally use an Option group for this, but if space on the form is an issue, use a combobos set to a Value list. The list should be:

  • Go back and uninstall the Nokia Connectivity Cable Driver.
  • Visit your countries Nokia website. Click 'Support' and then 'Download software'. Now (probably in a drop-down menu) find your way to the 'Cable drivers' section. For me this was at: http://www.nokia.com.au/support/download-software/cable-drivers
  • Download the driver relevant to you, for me it was 'Driver for Nokia DKE-2, DKU-2, CA-42, CA-53, CA-70, CA-101.'
  • Install this driver.
  • Restart computer.
  • Connect the phone, select PC Suite on the phone. If windows starts auto-installing drivers, it may bring up (several) windows asking where to locate drivers. Ignore this for now.
  • Load Ovi Suite. It will load the 'Add device' screen but it will be greyed out and should say "Please wait while USB drivers are installed on your computer". Afterwards, 'Add device' will now actually detect the device properly.


You may need to restart Ovi Suite and/or disconnect the phone a few times to get the last steps to work, it seems that Ovi Suite 'fights' with Windows over which should install the drivers.

  10. Now, click in each cell you want to become that colour.
  11. Click the "Shading" button just once.

If anyone could help me it will save me many hours of manual work!

 

Many thanks,

Alison

hansv mvp say:

Let's say that row 1 contains headers, and that the data begin in row 2.

Insert a blank column in column B. The original column B is now column C.

In B2, enter the formula =IF(ISERROR(MATCH(A2, $C:$C, 0)), "X", "")

Fill down to the end of the data in column A.

Insert a blank column (if necessary) in column D.

In D2, enter the formula =IF(ISERROR(MATCH(C2, $A:$A, 0)), "X", "")

Fill down to the end of the data in column C.

Entries that do not occur in the other column are now marked with an "X".

You can use AutoFilter to select rows with an "X", or you can use Advanced Filter to copy rows with an "X" to another location.
