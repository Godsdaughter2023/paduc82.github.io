To delete a message or thread

  1. In the App list, tap Messaging .

  2. Under Threads, do one of the following:

    • Tap to the left of one or more messages or threads to select each one, and then tapDelete .

Windows Defender (on Windows 7, Vista, and XP) provides spyware protection only, meaning you need to install an Anti-Virus product.

Installing an Anti-Virus product on Win 7 will in general automatically disable Windows Defender.

In case that there is any other Anti-Virus product installed or came pre-installed, you have to decide on one and uninstall the other, using the respective removal tool (List of anti-malware product removal tools - Microsoft Community ).

In case you want to use Microsoft Security Essentials:

Microsoft Security Essentials Consolidated FAQ
