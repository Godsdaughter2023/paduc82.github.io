a. user say:

Errors that end with 0002 are "file not found" kinds of errors and do not mean that a "current program on your computer may not be compatible with the new update".

It is generally unwise to accept hardware driver (video, audio, network, printer, etc.) updates from Windows Update and you should only get those from the manufacturers WWW site - never from Microsoft or Windows Update.

The best advice when you spot those updates is to right click the update, choose to Hide it and hope you never see it again or install it accidentally.

Windows 7 has built in ways to disable driver updates from Windows Update that you can read about here:
