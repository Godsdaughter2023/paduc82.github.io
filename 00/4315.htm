    john w. vinson say:

    This is a classic symptom of table corruption in the backend. Doublecheck though - if you look at one of these records using two or three different frontends, do they all show the same symptom? Are these Text or Memo fields?

    To resolve, I'd suggest first making sure that your database is securely backed up (you might need to restore a backup from before this started happening if possible). Create a BRAND NEW backend database - the one you're using may be irrevokably damaged. Import all the tables (and anything else, though a backend will typically only contain tables) from the damaged database. Check everything (scroll through a datasheet of the affected tables, the corrupted text will be pretty conspicuous); delete any affected records. They'll need to be reentered (from scratch or from a working backup). Compact and repair the database after you're done, then relink the frontends to this repaired backend. I'd open the master frontend, DELETE all links to the backend, and create new links using External Data... Access... Link; make new .mde's and distribute to your users.

    Good luck! Note that 2003 is over a decade old now; at some point you'll need to bite the bullet and convert to a newer version. 2010 is what I'd recommend - 2013 has a lot of complaints and its new features are almost entirely web-related.

    Opera Mini on S40-based phones (e.g. Nokia 3720 Classic). - Microsoft Community

    fjuk say:

    Opera Mini on S40-based phones (e.g. Nokia 3720 Classic).

     

    I bought a Nokia 3720 Classic which from the factory had the standard browser installed as well as Opera Mini.

     

    I easily recieved the GPRS setup from my provider, but could not make it work in Opera Mini. It would only work in the integrated Nokia-browser.  

     

    Opera Mini would claim that there was no internet connection and trying to use the 'activate default in all applications' (Settings, configuration) on the phone did not resolve the problem.

     When trying to access the internet through Opera Mini (4.2) the 'G' for GPRS-connection would flash but the square around it would never appear and an error message claiming inability to connect to the internet would appear.

     

    The problem showed up to be one of technical definitions. My provider's configuration was basede merely on the integrated browser and was a so-called 'web' configuration.

    What I needed to do was to go to Settings => Configuration => Personal config. sett. => Options => Add new => Access Point (in stead of 'Web')

    Then I had to enter the informations provided from my supplier in this new profile, activate the AP - and voila!

     

    Now I can even use the Opera Mini 5 beta nqa...

     

    I hope this can save some of you outthere's time!
