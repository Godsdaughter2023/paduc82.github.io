It's actually perfectly normal for a phone (or any electronic device) to accumulate some heat when it's being charged or used. That is normal.

What's not normal is, if the heating gets extreme to the point of burning. (but we're talking REALLY hot, before I'd worry)

Bug in "Center Across Selection" - Microsoft Community

misvin say:

Bug in "Center Across Selection"

Feature: Format Cells => Alignment => Text Alignment => Horizontal => Center Across Selection

Example: Put any short word in the cell A1. Select range A1:C1. Execute "Center Across Selection". All works OK. After that select range A1:E1. Execute "Center Across Selection". All works OK.

Bug: Impossible execute "Center Across Selection" in the range A1:C1 (small cell range) after you executed "Center Across Selection" in the range A1:E1 (big cell range) even if firstly you select another horizontal text alingment like "General" or "Left (indent)".

Summary: Excel remembers previous range when you use "Center Across Selection". And you can increase the range only,but not decrease it.

geoffbarraclough say:

Actually you don't need to reset the formatting for ALL the cells in the larger selection

You have the scenario where you set A1:E1 as centre across selection

You then set A1:C1 as "Left"

This leaves D1:E1 as centre across selection

When you then apply A1:C1 as centre across selection, excel sees that a1:c1 are formatted as that AS WELL AS D1:E1 and so centres across all the cells in the contiguous range that have that formtting

If you perform the same actions but set A1:F1 as centre across selection

then set A1:D1 as Left

then set A1:C1 as centre across selection, you should see that it works for A1:C1 as there is a break in the range so excel does not determine the A1:E1 have the same formatting

Sorry but that is not a bug - This is only an "issue" as excel needs to look for a set of cells in one range that all have that formatting applied for it to know which cells to centre across - In your example you are removing the formatting but then putting it back exactly the same as it was so it is not surprising that you get the same results. When you actually change the formatting (i.e. by having D1 with differnet formatting to A1:C1 and E1:F1 then you get the desired results

Rgds Geoff
