I'm fairly certain that this was a rhetorical question... and you aren't really interested in a answer.  But, I'm gonna give you one anyway :)

Storing favorites in Internet Explorer was kinda simple... it just used files.  However, Edge has more stuff to store than Internet Explorer.  So, they took a bold step to move favorites (and a lot of other things) into a database.   Using a database is more efficient when you consider all of the other things that are being stored:  settings, favorites, reading lists, books, etc.

Remember, Windows 10 gave you TWO browsers... the new Edge and the old Internet Explorer.  If you don't like one, then just use the other!

What file systems do Windows Phone 7 devices - Microsoft Community

sledwards say:

What file systems do Windows Phone 7 devices support/implement?

I've been searching for a definitive source that states the file systems used for these devices, but have not come up with anything reliable or official.
jack cook - aka help_line say:

"Two file systems are supported by Microsoft in Windows Phone 7: IMGFS and TexFAT. IMGFS was already in use by previous versions, and holds the actual system and operating system files. TexFAT is an extended version of the FAT file system capable of addressing files larger than 4GB, and is used to store user files. Microsoft has opted for a unified storage approach, which means that applications and/or users can not distinguish between files in local storage or on a memory card. While this increases simplicity, it also means that crucial files may be stored on a memory card - removing said card will cause the phone to be restricted to making emergency calls only."

Uninstall Display Adapter drivers and reinstall drivers in compatibility mode

I suggest you to uninstall the display drivers and reinstall them in compatibility mode.

To uninstall the driver, follow the steps below:

  1. Press Windows key + R
  2. Type devmgmt.msc
  3. Right click on the driver you want to uninstall
  4. Click on Uninstall
  5. Restart the system

After the computer restarts, install the driver in compatibility mode

  1. Select the setup file
  2. Right click on it and click on properties
  3. Click on Compatibility tab
  4. Select the operating system
  5. Click Ok.

A clean boot allows the Windows operating system to launch without any additional programs or services running when you turn the computer on. This allows you to determine if computer issues are caused by a program or service that starts up when you boot the machine.
