Due to the migration of Skype and Microsoft, creating an account under a Skype name is no longer available. You can only now create an account via email address (Microsoft account) or mobile number. To answer your side note, if the account was created under an email address (Microsoft account), the system will create a default username for you. yoga yao msft say:

Hi Dick,

Volume Licensing is different from the normal account activation. When we subscribe to related Office 365 subscriptions with the latest desktop version of Office license, we can use our accounts to activate Office 365. In this case, when Office 365 is activated with User A's account, if User B signs in to Office 365 with account for connecting related services, such as SharePoint libraries or OneDrive for Business, he may encounter credential issues. So as I stated above, if there are multiple users sharing one same computer, we suggest you deploy Office 365 shared computer activation on the computer for avoiding possible credential issues.
