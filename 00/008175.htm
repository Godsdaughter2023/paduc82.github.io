Please use Notepad to make a batch file that reads as follows:
takeown /F "%SystemDrive%\Windows.old" /R /A /D Y
icacls "%SystemDrive%\Windows.old" /grant Administrators:F /T /C
rd /s /q "%SystemDrive%\Windows.old"


Then please name the batch file as "DelWinOld.bat" and save it to your "%UserProfile%\Documents" folder.


Then please open a "Command Prompt (Admin)" window by the following way:
Keyboard: Win+X, A, Alt+Y
Mouse: rightclick Start button, click "Command Prompt (Admin)", click "Yes" button
Touch: press-and-hold Start button, tap "Command Prompt (Admin)", tap "Yes" button

cd "%UserProfile%\Documents"
delwinold

dianeoforegon say: You cannot restore a single item like contacts. You can restore your Identity and not delete current by renaming.

TO RESTORE YOUR IDENTITY FROM A TIME MACHINE BACKUP OR CLONE:

1) Click the Time Machine icon in the Dock or open it from the Applications folder. (For Clone backup just drag the Identity to the Desktop and rename. Skip to step 4)

2) Use the arrows and the timeline along the right edge of your screen to browse through the Time Machine backups. When you find the Main Identity file, select it, and then click Restore. Select to save on the Desktop.

3) Rename Main Identity on Desktop to "Restored Main Identity".

4) Drag "Restored Main Identity" to the Office 2011 Identities folder.

 ~/Documents/Microsoft User Data/Office 2011 Identities/

5) "Switch Identity" How to create a new Identity or switch to a new Identity in Outlook 2011

6) Export your group contacts under File > Export > Category > 'category of group'

*  Each group member will have to be a contact in the Address Book (blue dots indicate it's only in group)
*  Create a category with name of group.
*  Assign each contact in the group the category.

7) "Switch Identity" and select Main Identity.

8) Under File > Import select the .olm file on the Desktop. The imported file will show up in the folder window as Archived Contacts or something similar.

9) Drag items into local contacts.

10) Sort by category > select name of group category. Select all and drag into a group window. Save.

Hope this helps!

Please be informed that the Schedule Cleanup feature that you see on top of the message is not yet enabled. You are given the option to manage an email message or a number of emails. The following are the things that you can do with emails using this function:
