terryfromjhc say:

When I insert a vector png image into PowerPoint, it is blurry.

steve rindsberg say:

When you save as PNG, it's no longer a vector image. PNG is strictly a bitmap/raster format.

How large was it in Illustrator (in inches/cm/whatever) and how large is it in PPT?  300dpi tells us how many dots per inch are in the image.  If the image was originally 1" wide, that's only 300 dots, which might be fine in PPT if you're also displaying it very small there.  If you're displaying it very much more than say 2", it *will* be blurry.

PPT does do two things that may make the image blurry even if it is closer to the needed size:

1) It compresses image when you save the presentation unless you do something about it:

PowerPoint 2007 and 2010 make pictures blurry, loses GIF animation
http://www.pptfaq.com/FAQ00862.htm

2) It "antialiases" images when you insert them.  Read: "blurs".  Not much you can do about that, other than not importing images; consider saving from Illo as EMF instead and insert that into PPT. 

Steve Rindsberg PowerPoint MVP
PowerPoint FAQ
PPTools Add-ins for PowerPoint

    keep TOC hyperlinks when converting to PDF - Microsoft Community

    mproulx@golder.com say:

    keep TOC hyperlinks when converting to PDF

    I just discovered how to keep the hyperlinks in the TOC of our company templates when converting them to PDF. 

    Highlight TOC;   right click, edit field;   click "table of content";   click " use hyperlink instead of page number".

    is there a way of making this function a default?

    MMP

    suzanne s. barnhill say:

    This actually is the default in all versions of Word beginning with Word 2000. The default TOC field is:

         { TOC \o "1-3" \h \z \u }

    The \h switch makes the entire TOC entry (as opposed to just the page number) a hyperlink; the \z switch hides tab leader and page numbers in Web layout view. All this functionality was introduced in Word 2000 to make Word better adapted to creating Web pages (which it still sucks at).

    If you have older documents (older than Word 2000, that is) or documents in which the TOC entries are not hyperlinked (I tend to turn off this feature for print documents because it makes it more difficult to modify TOC styles), you can change them by editing the TOC field manually. Just press Alt+F9 to display the field code, add the \h switch, and then Alt+F9 to toggle back and F9 to update the field.

    Follow the steps to register W32Time:

    a. Press Windows  key + X keys and Click Command Prompt (Admin)

    b. In Command Prompt type w32tm /debug /disable and press Enter.

    c. Then type w32tm /unregister and press Enter.

    You should get the response W32Time successfully unregistered.

    d. Type w32tm /register and press Enter.

    You should get the response W32Time successfully registered.

    e. Type net start w32time and press Enter.

    f. Exit the Command Prompt window and restart the computer.

    1. Click the Gear box at the top right corner in Internet Explorer.

    2. Click the Tools button, and then click Internet Options.

    3. Click the Connections tab, and then click LAN settings.
