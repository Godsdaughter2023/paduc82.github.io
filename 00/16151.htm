olivermcclellan say:

I followed these steps and fixed my update issue:

Softwaredistribution folder contains files related to Windows Updates, it is present on all versions of Windows and it should have a couple of hundreds megabytes of size. But if this folder is larger, you have some corrupted updates in, and it will take more disk space. So, in order to get rid of these corrupted updates and free up some disk space, you’ll have to delete corrupted updates from Softwaredistribution folder.

1. Go to This PC and open the partition you have your Windows installed on (it’s usually C:)
2. Go to Windows folder
3. In Windows folder, find the folder named Softwaredistribution and open it

4. Select all the files and folders and delete them (you will need administrative rights to delete these files)

5. If the files fail to delete, boot into safe mode and try again.

That’s all, your corrupted updates are now deleted and you should be able to check for good ones normally.

Ok found it, you need to uninstall everything on your mac before you start. I mean everything related to OneDrive. if not the Add Account won't appear...

1. Remove all versions of OneDrive from the machine, including OneDrive consumer from the Mac Store as well as any other OneDrive for Business previews.

2. Open a Terminal window and type the command:

 defaults write com.microsoft.onedrivedf-mac EnableAddAccounts 1

3. Refresh your defaults by typing this command in the Terminal window

 killall cfprefsd

4. Download the OneDrive preview package from here: oneclient.sfx.ms/.../OneDriveDF.zip

5. In Finder, move the OneDriveDF package from “Downloads” to “Applications”

6. Run OneDriveDF and add a consumer account (NOTE: A consumer OneDrive will not be required in the final version)

7. From the OneDrive cloud in the Menu Bar, choose “Preferences…”

8. In the Accounts tab, select “Add a Business Account” and go through the account config flow:

9. Choose your home folder. DO NOT select your OneDrive Consumer location. A OneDrive for Business folder will be created under your home folder based on your tenant name:
