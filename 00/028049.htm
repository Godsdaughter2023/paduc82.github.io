Hi, try

1) Do the workbooks have sharing enabled?  This may cause a problem.
2) Do the workbooks have any visual basic macros in them that might be stopping the automatic update?  Some macros run when a workbook opens and one of them might tell Excel not to update external links.

4) When you select Edit Links and it says the status is OK...if you update links from this point does it work (without having the other spreadsheet open)?  Just curious.

6) Try recreating the VLOOKUP function with the other workbook OPEN - this will rebuild a proper link in case Excel thinks it is hosed when it really isnt.  Save both files after rebuilding the VLOOKUP, close both files, and then open the first again...does it work now?
