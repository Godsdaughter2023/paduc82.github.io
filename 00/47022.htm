Hi,

 

We appreciate your earnest effort for posting in Microsoft Community.

 

You can connect all kinds of Bluetooth devices to your PC including keyboards, mice, phones, speakers, and a whole lot more. Possible reason why you cannot see the Bluetooth Stack is because the service is not started or it is not installed on your computer.

 

To start troubleshooting this, we recommend that you perform the following:

 

Check if Bluetooth hardware is installed.

 

  1. Select Windows Start button then click Control Panel.
  2. Select Device Manager.
  3. Check for Bluetooth Radio in the list.

 

If the item is there, your computer has Bluetooth hardware installed and you can safely assume that the software has been set up as well.

 

If your PC didn’t come with the Bluetooth hardware installed, you can easily add it by purchasing a Bluetooth USB dongle.

 

Check for Bluetooth Service

 

1. Press Windows logo key + R at the same time. A Run dialog box will open.

2. Type services.msc in the run box and click OK button.

3. From the Name list, locate Bluetooth Support Service. Right-click on it and select Properties.

4. Change the Startup type to Automatic and click Start button to start the service.

5. Click Apply then OK button.

6. Restart your PC for the changes to take effect.

 

Uninstall then Reinstall the Bluetooth Device Drivers

 

  1. Press Windows logo key at the same time. A Run dialog box will open.
  2. Type devmgmt.msc in the run box and click OK button.
  3. Expand category Bluetooth Radios, right click on one Bluetooth device and select Uninstall. (If you see more than one Bluetooth device under this category, uninstall them one by one.)
  4. After uninstalling the drivers, update the driver.

 

You can go to your PC manufacturer’s website to download the latest Bluetooth driver. Before you get started, ensure that you know the PC model and the system version that you are using.

My prejudice against Microsoft Security Essentials is simply the number of times I have had to cleanup systems that were using just it for protection (and the number of times I was cleaning up systems with just it for protection and found items it let through that weren't necessarily causing issues... yet.)  Not saying the others do not have issues as well - there is no perfect solution.  I have just seen LESS issues with the other solutions over the years.  My actual recommended combination: eSet NOD32 Antivirus (not the suite - just the AV) combined with MalwareBytes Antimalware Professional - a ~60 USD + ~25 USD solution for two years and ~60 USD every two years thereafter.  I have people I swear TRY to get infected/infested stay clean for years so far on that combination.

 

If you are not going to use any of the extras that got installed when you refreshed the machine - get rid of them.  Open Office or Libre Office are fine free office suites and will work for most home users just fine.

 

Just to leave you with one more thing to think about: having free space on your hard drive means very little.  It's good to have if you need it - but it sitting unused rarely does you any good.  You keep ~15GB free on that drive at all times and it won't affect your performance for daily normal use in any way.  Everything above ~15GB is just there *to be used*.    Same with your system memory (which I would bet is the real performance killer on your specific machine) - unused memory does nothing for you *until you need it* and most home users (especially non-gamers) will not be using more than about 2GB at any given time.
