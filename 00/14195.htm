daniel g. grau say:
It can be done, but there are two steps involved:
  1. Add the Character Viewer to the main menu (to insert the Unicode character by clicking)
  2. Define it as an AutoReplace sequence

Add the Character Viewer to the main menu
  • Go to System Preferences
  • In the first row, select Language & Text
  • Click on the Input Sources tab at the top
  • Enable Show input menu in menu bar, at the very bottom

This will add a new menu choice next to the clock, at the top right of the menu bar. Click on it and select Show Character Viewer. You will see a list of Unicode characters. You can search for them using the Loupe tool at the bottom (type "male"). Once you've found your character (and there is an ample selection of fonts for ♂ and ♀), click on the large icon, or drag the samples to your Word document.

Define it as an AutoReplace sequence
Although you cannot assign characters a key combination, you can define them as an AutoCorrect entry. (There are already some such entries, predefined by Word, and Math symbols have their own preference section.)

Assuming you've already added a ♂ to your document (the one on this line is the "Arial Regular" variation):

  • Select the ♂ symbol (highlight it) in your document
  • Go to Word > Preferences > AutoCorrect in the main menu bar
  • Make sure the leftmost tab at the top (AutoCorrect) is selected
  • Check Automatically correct spelling and formatting... at the very top to enable auto-replacements
  • Enable Replace text as you type
  • In the Replace field, type a sequence, such as \male (see more details below)
  • Select Plain text if you want no formatting to be used, or Formatted text if you want to include size an color (test what works best for you)
  • Click OK

How to use
Type \male and Word will "autocorrect" this to ♂. The slash just ensures that the combination will be unique and that Word will consider the sequence a separate word from whatever precedes it. You could use \m or \male or \male\, i.e., something that you would not normally type.
Note that the AutoCorrect feature is language-dependent—what you define for English texts will no longer work if you switch the language to Spanish.
