john mcghie say:

Dictionary-style headers and footers are done with the StyleRef field.

  1. Format the heading of the Entry with a style.  I would use something like Heading 2 style, but it doesn't matter what it is, and you can format it any way you like.
  2. Make sure that only the words you want in the header are in that style, for each entry.
  3. Add two StyleRef fields to your headers or footers: one with the "\l" switch included.
  4. In Word 2011, the fields are on the Insert menu under Insert>Field.  You can't get to them from the ribbon. In Word 2016, it's on the Insert tab in the Text chunk.

The first StyleRef field will give you the first text on the page that contains an item formatted in the entry heading style.  It looks forward from the top of the page until it finds one.

The other StyleRef field looks backwards from the end of the page and gives you the LAST text on the page in that style.

The process is explained very well here:

https://support.microsoft.com/en-us/kb/141994

Don't get lead astray by exhortations that this is "retired" content.  The StyleRef field has worked the same way in every version of Word since 1989 :-)

Hope this helps

amanda zhang msft say:

Hi Shameem,

First I would like to confirm does the user still licensed Office 365 Exchange Online. If yes, you can use an Office 365 admin account to add the Full access and Send As permission of the user mailbox to that admin account.

Then you can configure this admin account in Outlook client and the user's mailbox will automatically show under the admin mailbox. Just go to the user mailbox's calendar and find the meeting to cancel it, about how to cancel the meeting, please refer to this article: https://support.office.com/en-us/article/Cancel-a-meeting-9e1eac0d-0462-4a01-ad0b-7d0543819562.

If the user's license has been removed, in Office 365,  the user's old email, contacts, and calendar are retained for 30 days. So in the 30 days, I suggest you temporarily re-assign the Office 365 license to the user, then convert it to a shared mailbox and remove the license. 

After that you can add the Full access and Send As permission of the shared mailbox to the admin account based on same methods I mentioned above and access the shared mailbox to cancel the meeting.
