Reset Firefox to its default state

  1. Click the Firefox menu button (orange coloured).

  4. To continue, click Reset Firefox in the confirmation window that opens.
  5. Firefox will close and be reset. When it's done, a window will list the information that was imported. Click Finish and Firefox will open.

What does the reset feature do?

All of your Firefox settings and personal information are stored in a profile folder. The reset feature works by creating a new profile folder for you while saving your important data.

Firefox will save these things:

  • Bookmarks
  • Browsing history
  • Passwords
  • Open windows, tabs and tab groups
  • Cookies
  • Web form auto-fill information
  • Personal dictionary

These items and settings will be removed:

  • Extensions and themes, website-specific preferences, search engines, download history, DOM storage, security settings, download actions, plugin settings, toolbar customizations, user styles and social features will be removed.

    Hi,

    Problematic drivers can bring about Windows 10 airplane mode errors. So it is necessary to fix the network adapter drivers on Windows 10 first. To do so, you may follow suggested steps below:

    Method 1: Modify the Network Adapter Properties

    1. Use Windows shortcut keys Win + X (or right-click the Start menu) and select Device Manager from the menu.
    2. Expand the Network adapters, right click on the network adapter of your device and select Properties.
    3. Select Power Management tab from the pop-up dialog box and uncheck the item Allow the computer to turn off this device to save power.
    4. Click OK to save the changes.

    Method 2: Disable Non-Microsoft Services

    Some programs or services can affect the Windows 10 airplane mode. You could try to disable some non-Microsoft services to fix the issue. If you happen to know the program, disable it directly. If not, it may take your time to figure it out.

    1. Use Windows shortcut keys Win + R to launch Run.
    2. Type msconfig into the box and press Enter.
    3. Select the Service tab, check Hide all Microsoft services and click the button Disable. Then click Apply.
    4. Select the Startup tab, click Open Task Manager.
    5. Select Startup tab from the new pop-up dialog and disable all the startup items.

    You need to reboot the computer and re-enable the services you’ve disabled one by one to find out the problematic services or programs that result in the Windows 10 airplane mode error. Once you figure them out, disable them again.

    Hi,

    Thank you for contacting Microsoft Community.

    I suggest you to follow the below methods and check if it helps.

    Method 1: I suggest you to perform an automatic repair using the Windows 10 installation media. You will have an ISO of Windows 10. You may refer to the following link on download and create Windows 10 installation media:

    http://www.microsoft.com/en-us/software-download/windows10

    Once you create the installation media, perform Automatic repair on Windows 10 computer, follow these steps:

    1.       Insert the media such as (USB or DVD) and restart your computer.

    2.       Press F12 key (typically this is F12, but it can differ between computer manufacturers) and choose the drive that you inserted the installation media into.

    3.       Once the Windows Setup window appears, follow these steps:

    4.       Click next and select Repair your computer.

    5.       You will then see a blue screen and an option to choose. Choose the option Troubleshoot and select advanced options.

    6.       You may choose Automatic Repair from Advanced boot option.

    Method 2: If the issue persist, I suggest you to rebuilt BCD and check. Follow the steps:

    1.       Insert the media such as (USB or DVD) and restart your computer.

    2.       Press F12 key (typically this is F12, but it can differ between computer   manufacturers) and choose the drive that you inserted the installation media into.

    3.       Once the Windows Setup window appears, follow these steps:

    4.       Click next and select Repair your computer.

    5.       You will then see a blue screen and an option to choose. Choose the option Troubleshoot and select advanced options.

    6.      Click on command Prompt.

    7.       Type these following commands and hit enter after each line of command:

    ·         Bootrec /fixmbr

    ·         Bootrec /fixboot

    ·         Bootrec /scanos

    ·         Bootrec /rebuildbcd

    8.       Restart the computer and check the issue.

    Hope this information is helpful and do let us know if you need further assistance. We will be glad to assist.

    su airodump-ng -c say: Hello RD46, welcome.
     
    If you have Office 2003 installed, the process should not take long at all. All your emails, contacts, and calendar information is stored in a compressed file, which is called a PST file.
     
    All we need to do is copy the PST file and import it into Outlook 2003. Follow these steps and you should be up and running in no time:
     
    1. Click START
    2. Click 'Control Panel'
    3. At the top left, click 'Classic View'
    4. Open up 'Folder Options'
    5. Click on the 'View' tab at the top
    6. Scroll down and click the 'Show Hidden Files & Folders'
    7. Uncheck 'Hide Protected Operating System Files (Recommended)'
    8. Click the 'Apply' button at the bottom and close the window
     
    9. Click START
    10. Click 'Computer'
    11. Open up the secondary harddrive shown that contains your old system information
    12. Double click on 'Documents & Settings'
    13. Double click on the user that the email was last accessed on
    14. Double click on 'Local Settings'
    15. Double click on 'Application Data'
    16. Double click on 'Microsoft'
    17. Double click on 'Outlook'
     
    Now look for a PST file, its usually 30mb+ in size
     
    18. Right click the file and select 'Copy'
    19. Close all open windows
    20. Click START
    21. Click 'Documents'
    22. Right click on an empty space and create a new folder
    23. Right click and paste the file inside the new folder.
     
    Once the PST file has copied, you then can open up Outlook 2003 and follow these steps:
    Open Outlook.
    On the File menu, click Import And Export. If the command is not available, rest the pointer over the arrows at the bottom of the menu, and then click Import and Export.
    Click Import from another program or file, and then click Next.
    Click Personal Folder File (.pst), and then click Next.
    Browse to your new folder in your 'Documents' and open up the PST file.
    Click Finish and it should import everything.
    The process of importing can take anywhere from a few minutes to an hour or so depending on the size of the file and of course the speed of the system.
