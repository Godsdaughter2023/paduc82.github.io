suzanne s. barnhill say:

You cannot change the color of just part of any font character (which is what this symbol is). You would have to convert the symbol to a picture and then edit it with image-editing software.

Alternatively, you could create the symbol you desire by inserting a circular shape, coloring it as desired, and then using Add Text to add the M to the center (you'll need to set the text box margins to zero and center the text horizontally and vertically). Once you've created the object, you can set the text wrapping to In Line With Text and save it as an AutoText entry.

You could create a union query AllAccountNames that returns all account names:

 

SELECT [AccountName] FROM [QueryFor2011]
UNION

Then create a new query like this:

 

SELECT [AllAccountNames].[AccountName], [QueryFor2011].[TotalAmount] AS Total2011, [QueryFor2012].[TotalAmount] AS Total2012
