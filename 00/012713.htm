scottgem say:
There are two main ways to populate a combobox. Either using a SQL statement or a Value List.

If your combobox's RowSource Type is a Value List, then you use:

comb1.RowSource = "item1;, item2; etc."

But a Value List should only be used for short, static lists. Instead I would use a table and then change the Rowsource to a SQL statement that pulls from the table.
