andre da costa say:
The computer freezing could be caused by hardware factors or software issues. On the hardware side, if the fan is ailing or the air vents are blocked, the machine may be overheating. If this seems to be the case, replacing the fan, cleaning the vents and clearing space around the computer may help. (A utility program that displays the processor’s temperature may be helpful; Core Temp and SpeedFan are two such programs and shareware archives have others.)
 
A bad memory chip can also bring a machine down. A free memory test program like HCI Design’s MemTest or the Windows Memory Diagnostic Tool can help confirm or rule out faulty RAM as the culprit. Problems with the computer’s motherboard, power supply or hard drive could also be the cause but may require a computer technician to properly diagnose and repair.
 
On the software side, malicious programs like viruses and spyware can cause erratic behavior and random reboots, so make sure you have an up-to-date security program installed on the computer. Basic versions of Avast and AVG Anti-Virus Free are free. Microsoft offers its own suite of free security programs as well.
 
If you have installed new hardware devices recently, the machine may be suffering from a driver conflict. Drivers are small files that let Windows communicate with the hardware that’s attached to it, but sometimes these driver files  clash with each other. Microsoft has tips for using the Device Manager utility to troubleshoot driver problems.

If you just upgraded to Windows 10 the problem might actually be fixed automatically in a day or so.

When Windows 10 first installs, it installs a generic video driver.  That is what is installed on your machine right now, and that is why you are missing the 1920x1080 resolution.  Note you are asking the wrong people (AOC support) for help.  What you need is a video driver.  This will usually be supplied by the manufacture of your computer, not the manufacture of the monitor you have.

From what happened on my system.  At first it had the generic video driver installed, but shortly after I upgraded Windows checked what was available to download, and it found and installed the proper driver.  You can try to speed up this process by selecting:

Start -> Settings -> Update & security -> Check for updates

If that doesn't work what you need is to get the proper driver for your machine.  Usually manufactures of machines have these drivers for downloading under a support section on their website.  You need to first try downloading and installing this driver.

Erichsoft,

I've seen a corrupt registry entry cause this in the past. Try the following:

  1. Click Start, click Run, type regedit in the Open box, and then click OK
  2. Navigate to the following registry key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\
  3. Once you've selected CurrentVersion, you should see DevicePath in the right pane. Double click it.
  4. The value should be %SystemRoot%\inf
Once you've made the change, reboot and try to install IPv6 again.
