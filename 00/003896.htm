Using Word for Mac 16.23 (190212)

Edit: changed "header" to "headings"

john korchok say:

This problem is caused by using an Exact line spacing setting in the Paragraph format, a common mistake. In Word, it's a better practice to set line spacing using Multiple and setting the number to give a visual equivalent of the line spacing in points. To illustrate why, insert an inline picture in your text. If your line spacing is set to Exactly 14pt, only the bottom 14 points of the picture display. Set to Multiple 1.05, the entire picture shows and ascenders and descenders are not cut off when you raise or lower a selection.

Exact line spacing should only be used in locked fillable forms and places where a photo will never be inserted, like a table of contents, header or footer.

    rccryan say:

    I came across this thread while searching for an answer to the same question and found it to contain misleading information:

    while technically, it is true that the functionality to enter a superscript-formatted font within a data field is not possible in MS Access, you can, however, enter a Unicode character that has the same effect. Just use Character Map (in Windows - sorry, I'm not sure if Mac has an equivalent). See details in the article linked and excepted below:

    How to Use Superscripts in MS Access

    http://www.pcmag.com/article2/0,2817,2130158,00.asp


    "Data fields in a Microsoft Access database aren't intrinsically associated with any particular font. You can configure a form or report to display the data using any font and font style of your choice, but the format affects the entire data field. You can't display part of the data in italics and part of it not in italics, nor can you display part of it superscripted and part not.

    "What you can do is use characters that are themselves superscript versions of the desired numbers. Almost all fonts include characters for super-scripted 0, 1, 2, and 3. For superscripted numbers 4 or greater, you will have to use a font such as Lucida Sans Unicode that includes those characters.

    "In addition, you'll have to use the Character Map to insert the special characters. Select Start | All Programs | Accessories | System Tools | Character Map from the menu to launch it. Check the box titled Advanced view. In the newly visible drop-down list labeled Group by, select Unicode Subrange. And in the secondary window that appears, select Super/Subscript. Now you can easily choose the characters you want."
