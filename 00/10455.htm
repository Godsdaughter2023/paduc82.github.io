teylyn say:

The conditional formula checks if the value in column J (column 10) to column Z (column 26)  of the current row is equal to the maximum value of  columns K to AA. Both Marianne and Anthony have a 11.5 in column J, so both get the highlight. 

=countif(indirect(address(row(),10)&":"&address(row(),26)),max(($K$5:$AA$47)))

The use of Indirect, Address and column numbers is clunky and error prone (as can be seen). Somebody probably inserted a column before column J, and the reference to the absolute columns 10 and 26 were not adjusted.

To apply a more robust formula that does what you describe, select A5 to A24 and use this formula

=COUNTIF($K5:$AA5,MAX($K$5:$AA$24))

Note the position of the $ signs. It makes the column reference in $K5:$AA5 absolute, but the row reference has no $ sign and will be applied to the current row. The result is a highlight for Marianne only.

The most easy way to fix is to copy the app's private data from another known-good device. Assume you have installed the YouTube app in memory card (not phone memory), then you need a computer to access the files stored on the phone's memory card.

This YouTube app is for S60 device, its private data is stored at E:\Private\2000FBBA, where E is the memory card. Beside the subfolder "res", there are some other files in this folder, see below the example.

backup_registration.xml 225.00b April 8 2012
didk 143.00b June 26 2012
fv1 1.00Kb June 26 2012
pstlgs 44.00b June 26 2012
res 1.50Kb January 18 2013 at 14:57
sehi 27.00b June 26 2012
stgs 148.00b June 26 2012

Simply copy these files from the other S60 device to your phone, then your YouTube should be able to work as expected. Be aware you are using the other phone's private settings including country code and search history. Do no try to "clear private data" under Help option, this will reinitialise the app and bring the problem back.

 

If you just can't borrow a known-good S60 device, try get these files by searching for "2000FBBA" on Google.

This solution worked on my E72.

NOTE: Don't touch the other files as it may brick your phone. Take your risks.

The Windows 10 Mobile Upgrade Advisor app helps you find out if your Windows Phone 8.1 phone is eligible to install the upgrade to Windows 10 Mobile. It can also help free up space on your phone so you're ready to install the upgrade.
