Not all Excel functions work across multiple sheets as 3D formulas. I'm not sure which part of the formula you would like to use as 3D.  

If 3D formulas cannot be used, you will need to create the same formula in the same cell in each sheet, then use a Sum() with a 3D reference to sum all these cells.
