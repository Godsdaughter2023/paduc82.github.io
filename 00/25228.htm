suzanne s. barnhill say:

Word has a built-in feature for this. If you have used the Caption feature to create your figure captions, then you can just insert a Table of Figures that will pick up all the figure captions. Both Insert Caption and Insert Table of Figures are in the Captions group of the References tab.

If you have not used the Insert Caption feature, you can still create a Table of Figures based on a particular style if you have used a uniform style for figure captions (and not for anything else); by default, captions used the Caption style.

The Table of Figures Options dialog (accessed via the Options... button in the Table of Figures dialog) allows you to select a style or, if you choose to base the TOF on table entry fields, a table identifier. This latter case would assume that you used TC fields in your captions to provide the content for entries in the TOF, which would be a lot more trouble than just using the built-in caption feature or the Caption style.

graham mayor say:
This will be quite complicated to achieve, but not impossible. The path is found in Paul Edstein's date calculation document at http://www.gmayor.com/downloads.htm.

Copy the following field construction into your document where you want the Target_Date field to appear. Then use the macro at http://www.gmayor.com/export_field.htm#TextToField to convert the text from this message into working fields.

Having done that you then need to add shading to the text of two of the internal fields. As it seems impossible to show colours in this forum. Colour the first bold field with red shading and the second bold field with orange shading. Do not shade anything else!  You can change the date formatting switches on the two bold fields and the last merge field to suit local conditions. Do not change any of the other switches!

{ IF { Mailmerge Target_Date  \@ "yyyyMMdd" } <= { DATE \@ "yyyyMMdd" } "{ Mailmerge Target_Date \@ "d MMMM yyyy" }" "{ IF { Mailmerge Target_Date  \@ "yyyyMMdd" } < {QUOTE{SET Delay 3}{SET od{DATE \@ d}}{SET om{DATE \@ M}}{SET oy{DATE \@ yyyy}}{SET ld{=(om=2)*(od=28+((MOD(oy,4)=0)+(MOD(oy,400)=0)-(MOD(oy,100)=0)))+((om=4)+(om=6)+(om=9)+(om=11))*(od=30)+(od=31)}}{SET mm{=MOD(ABS(om+Delay+11),12)+1}}{SET yy{=INT(oy+(Delay+om-1)/12)}}{SET dd{=ld*(31-(mm=2)*(3-((MOD(yy,4)=0)+(MOD(yy,400)=0)-(MOD(yy,100)=0)))-((mm=4)+(mm=6)+(mm=9)+(mm=11)))+(1-ld)*(od*(mm=2)*(od<29)+od*(mm<>2)+(od>28)*(mm=2)*(28+(MOD(yy,4)=0)+(MOD(yy,400)=0)-(MOD(yy,100)=0)))}}"{dd}-{mm}-{yy}" \@ "yyyyMMdd"} "{ Mailmerge Target_Date \@ "d MMMM yyyy" }" "{ Mailmerge Target_Date \@ "d MMMM yyyy" }" }"}
