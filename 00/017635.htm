diane poremsky mvp (slipstick.com) say:

In Outlook 2007, press Shift+Enter if it comes at the end of a word, if you keep typing when it comes up after 3 or 4 letters, it should go away before you press enter.

You can use Application.DisplayAutoCompleteTips = False in the immediate window in Word's VBE to disable it in word but that doesn't seem to work in Outlook. 

Note Outlook 2003 and 2010 both have an option to disable it. 2007 does not.

 

if you want to try the macro - paste the following in Outlook's VB editor, set a reference to word application model and run it.

Sub TurnOffAutoComplete()
 Word.Application.DisplayAutoCompleteTips = False
 End Sub
