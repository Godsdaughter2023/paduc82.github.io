graham mayor say:
In Word 2007 you will find Section breaks on the Page Layout Tab > Page Setup Group > Breaks > 'Next Page'. If you want different information in the header/footer then uncheck 'Link to Previous' in the second section header and put the required information in each section.

stefan blom say:
With the insertion point in section 2, select the "Start at" option in the Page Number Format dialog box (Insert tab | Page Number | Format Page Numbers) and specify the number 1. Note that this is also discussed in the cited article.

craig schamel say: Thank you.   Ok, slowly I'm getting there.  Now the pagination is correct, but in the process of getting it there Word has inserted another extra blank page into the document, between the sections that have different pagination.  How do I remove this extra blank page without disturbing the document's now correct pagination?  (When I tried to fix this problem  the pagination went back to the standard pagination).

rohn007 say:
In Print Layout view. If you don't have it on, turn on the show/hide feature
In the Home tab > Paragraph group. Click on the Show/Hide (¶) button to reveal the end of paragraph marks (¶) and nonprinting and normally not visible formatting markers

 

Now go down to the spot where the extra page is showing up. There is probably a blank para mark with a section break on the blank page. Place the insertion point between the para mark and the section break indicator. Then one at a time, backspace to delete the "extra" paragraph marks until the blank page disappears.

stefan blom say:

Did you enable "Different odd and even" or "Mirror margins" (or both of these options)? In that case, Word assumes when you restart page numbering that you want the new page 1 to be a right-hand page and treats the section break as an Odd Page break, that is, it adds a "filler" page if necessary.

craig schamel say: Hello and Thanks Again For the Help From Everyone -

As it turns out, in order to execute a prior step in the pagination process, I had to type some text into the area of the section break.  Once I deleted this text, the extra page disappeared, while the desired pagination was retained, so now all seems well.
