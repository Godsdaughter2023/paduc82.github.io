Follow next solutions:

Solution 1 – Update Windows

The first place to look for Windows fixes is Windows Update. This is how Microsoft delivers fixes to vulnerabilities, problems, and bugs discovered after the release of a Windows distribution. You can check for new updates by opening Settings from the Start menu. Now open the Update & security section, select Windows Update from the left side menu and click on the Check for updates button.


If there are any updates available Windows will download and install them. When it finishes reboot your computer to make sure that the new settings are applied.

Solution 2 – Update drivers

Device drivers are also prone to bugs and keeping them up to date is as important as doing it for your Windows installation. Driver updates can also be found by using Windows Update, but this doesn’t work for every piece of hardware available and your manufacturer might use a custom hardware component that requires a custom driver.

To find the latest hardware drivers visit the manufacturers’ website and head over to the Download or Support section. Type your device’s model name, download and install the latest driver available.

Solution 3 – Check screen saver settings

If your screen saver isn’t working it might be because it isn’t enabled or configured properly. To check the screen saver settings right click on the Start menu button and select Control Panel. Click on Appearance and Personalization and then on Change screen saver under Personalization.

This will bring up the Screen Saver Settings window. Here you can select which screen saver you’d like to use, customize it and set the time before it appears. Double check that everything is configured correctly.



Solution 4 – Disconnect unnecessary devices

Online forums are littered with threads from Windows 10 users who have had problems with the screen saver and most of them have found that this was caused by certain devices, mostly game controllers (Logitech, Razer, PlayStation, Xbox etc.).

To test if this is the reason for your screen saver not working disconnect everything that’s not necessary for your computer to function and check if the screen saver works. If it does re-connect your devices one at a time and check if the screen saver works after each one of them. If it stops working after connecting a certain device then you found your culprit.

Solution 5 – Clean the mouse sensor and mouse pad

Optical mice use an optical sensor to detect movement and direction. Sometimes if the optical sensor or lens have dust on them the mouse might detect false inputs and prevent your screen saver from showing up.

To clean it just flip your mouse over and use a paper towel or a cotton swab to gently clean the optical sensor. You can also use a screen cleaning solution to make sure the sensor is spotless.

The screensaver is not enabled in Windows 10 by default (we’ll talk more about it at the end of the article). So, if the screensaver doesn’t appear even after a couple of hours on idle, there’s a chance it hasn’t been activated at all. Right-click your Desktop, and go to Personalize.

The same can be checked in msinfo32 --> Software environment --> loaded modules. Check if you find any suspicious file. Once you get the suspicious entry, note down the file name and location.

Then go to the mentioned location and check if you can find that file. I can guarantee you that you won’t find it there as it is super hidden. Open the command prompt and then try to delete the file. To be on the safer side, I usually remove its entire file attribute using commands and then delete it. Usually this will fix the issue.

Sometimes I go a step further. I check registry with the suspicious file name and then try to find, if it is located somewhere else. Registry values helps a lot in finding it if there are in multiple locations. Once you find the file in more locations, remove it using command prompt.

      rcwhite say: Hi, dh.

      Since Win7 Search works only in the background, it takes a while for it to
      catch up to file changes, especially on a computer that is used continuously
      for foreground tasks. So when you delete a file, if you continue to
      actively use the computer and then promptly shut it down, the Search utility
      may not get a chance to update its index until you start it again. If you
      leave the computer running while you have lunch - or overnight - it should
      have plenty of time to catch up to you.

      RC

      2.       Then, right click on the drive you wish to scan for errors and select Properties.

      3.       Now click the Tools menu, then Check Now under the error-checking section.

      4.       You have several options within the check disk tool. It is always recommended you leave the "automatically fix file system errors" box checked, as this repairs and problems found. If you want to perform a deeper scan, tick "scan for and attempt recovery of bad sectors". This second option takes longer, but is worth doing if you suspect a drive problem. Once you are configured, click Start.

      5.       If you try to check a disk that is currently in use, you will receive a message asking if you wish to schedule a scan. Accepting this will perform the scan next time you restart your PC.

      4.       Type the name you want to give the user account, click an account type, and then click Create Account.

      5.       Switch to the new user account and check if the issue persists.
