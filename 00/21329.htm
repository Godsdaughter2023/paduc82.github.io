When indentation changes expectedly in a numbered list, that is a sign that numbering (including indents) wasn't set up properly from the start.

Click the first level 1 item of your list and then click Home tab | Multilevel List | Define New Multilevel List. In the Multilevel dialog box, set up all aspects of numbering the way you want them.

Note that for numbering to be truly stable, you will have to set it up with paragraph styles. See http://www.shaunakelly.com/word/numbering/numbering20072010.html.

Boot your computer in Clean boot mode and then start the application. If it works successfully, then some program is conflicting with this.
