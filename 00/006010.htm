mike middleton say:

I agree that it would be tedious to create an XY Scatter chart with a data series having one data point and then add nine more such data series. And I don't know a faster way to implement that approach (except by taking time to write VBA code to automate the process).

As an alternative, I suggest you consider a different chart design: Create an XY Scatter chart with a single data series having ten data points. Then use Rob Bovey's XY Chart Labeler for Mac Office 2011 to place the name adjacent to each data point. Do not use a legend. The add-in is available at

     http://www.appspro.com/Utilities/ChartLabeler.htm

From the perspective of chart design, this design is preferable because the viewer does not have to look back-and-forth between the data point markers and the legend.

- Mike Middleton, www.MikeMiddleton.com
