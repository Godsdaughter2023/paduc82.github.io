  • The PC boots into Windows RE (Recovery Environment).

  • Windows RE installs a fresh copy of Windows.

  • The PC restarts into the newly installed copy of Windows.

Changing Wireless Mode - Microsoft Community

xb1r apollo say:

Changing Wireless Mode

Login Information

  • Gateway - 192.168.1.1 or 10.0.0.1
  • Username - admin
  • Password - password or admin or password1

Verizon Firmware (Red/Black)

Changing Wireless Mode

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless Settings from the menu at the top.
  6. Select Advanced Security Settings from the menu on the left.
  7. Select the "802.11b/g Mode" link.
  8. Select "802.11g only" from the dropdown menu.
  9. Select Apply.

Westell Firmware (Blue/White)

Changing Wireless Mode

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless from the tabs at the top.
  6. Select Basic Setup from the submenu at the top.
  7. Select "11g only" from the dropdown menu labeled Mode.
  8. Select Apply.
