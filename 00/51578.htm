irfan h say:

Hello Gary83,

You can try to set the Wireless network as the top priority from the Advance settings, this will automatically set the Wireless Network with maximum priority then other networks including the dial up network. The default interface metric for all network adapters is set to Automatic, which causes the adapter to use the fastest available connection. But for some reason, the wireless card installation program had set the wired interface metric to 1 and left the wireless interface set to Automatic.

Change the wireless interface to value to 0, this will set the Wireless Network Priority to highest.

To configure which of several possible connections to use to access the Internet:

  1. Open the Control Panel Network and Dial-up Connections applet. Right-click Local Area Connection and choose Properties.
  2. Select Internet Protocol and click Properties.
  3. Click Advanced.
  4. Make sure that the Automatic metric check box is selected. (If you want to force a particular connection to be primary, clear the Automatic metric box and enter interface metric numbers, from lowest to highest.)

Driver verifier

Using Driver Verifier is an iffy proposition. Most times it'll crash and it'll tell you what the driver is.

*But sometimes it'll crash and won't tell you the driver.

*Other times it'll crash before you can log in to Windows. If you can't get to Safe Mode, then you'll have to resort to offline editing of the registry to disable Driver Verifier.

*I'd suggest that you first backup your data and then make sure you've got access to another computer so you can contact us if problems arise.

*Then make a System Restore point (so you can restore the system using the Vista/Win7 Startup Repair feature).

*In Windows 7 you can make a Startup Repair disk by going to Start....All Programs...Maintenance...Create a System Repair Disc - with Windows Vista you'll have to use your installation disk or the "Repair your computer" option at the top of the Safe Mode menu .

Then, here's the procedure:
- Go to Start and type in "verifier" (without the quotes) and press Enter
- Select "Create custom settings (for code developers)" and click "Next"
- Select "Select individual settings from a full list" and click "Next"
- Select everything EXCEPT FOR "Low Resource Simulation" and click "Next"
- Select "Select driver names from a list" and click "Next"
*Then select all drivers NOT provided by Microsoft and click "Next"
- Select "Finish" on the next page.

*Reboot the system and wait for it to crash to the Blue Screen.

*Continue to use your system normally, and if you know what causes the crash, do that repeatedly. The objective here is to get the system to crash because Driver Verifier is stressing the drivers out. If it doesn't crash for you, then let it run for at least 36 hours of continuous operation.

*If you can't get into Windows because it crashes too soon, try it in Safe Mode.
*If you can't get into Safe Mode, try using System Restore from your installation DVD to set the system back to the previous restore point that you created.

fink2105 say:

cant connect to live through wall ethernet jack

I live in an apartment complex and when i plug my computer into the wall jack the internet works fine. When i plug my xbox into the wall jack it says i am not connected to the internet. What can I do?

arminatorx say:

[quote user="FINK2105"]

I live in an apartment complex and when i plug my computer into the wall jack the internet works fine. When i plug my xbox into the wall jack it says i am not connected to the internet. What can I do?

[/quote]

Are/Were you reqired to sign in on a special website of your apartment complex before you could use the internet on the PC?

If so, you will need to "clone" the PC's network card hardware address (called MAC address). If you look at the properties of your network card on your operating system, you should see something like "physical address", which is six pairs of Hexadecimal numbers that look like this: 02:3F:EA:45:8E:B2 for example.

If you found the correct hardware address of your network card (do NOT confuse that address with the IP address!!), write it down, and unplug your PC from the wall network.

On the console go to the network settings and configure the console network. In the network configuration go right once more to the additional network settings.

There you can set up the consoles MAC address. Simply copy the MAC address there. Then plug the Xbox console to the wall network, and cross your fingers.

If that doesn't work, you will have to contact the network provider for your apartment.
