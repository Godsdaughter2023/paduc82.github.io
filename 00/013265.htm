Create a search folder that covers all the emails, then customize the current view, and add a filter to it. On the SQL tab enter this:

Hi Steve,

MSIE browsers have security settings for navigating into domains of different integrity.....

basically.... if you try to navigate to a domain that maps to the IE Internet Zone, from a domain that maps to say your Trusted Sites zone.... then IE will display a "Page cannot be displayed message"....

sites like Google, Youtube, and Microsoft 'Live' sites use a different domain to verify your user credentials when you visit their sites.....

eg... if you go to http://www.google.com your requests first goes to google.com where a new request to account.google.com is made to verify your credentials....

If you have google.com in your Trusted Sites list, but not account.google.com then in IE browsers, a loop is made between google.com and account.google.com

To fix the problem, go

Tools>Internet Options>Security tab, Trusted Sites icon, 'Sites' button....

Remove any google, youtube, Microsoft domains......

for general public web surfing you should no have any domains in your trusted sites list.... the zone is mostly used by large companies that do business with other companies using the internet....

The Trusted Sites zone, actually has lower security restrictions than the Internet Zone...

ALL public access web sites are designed to work in IE's internet zone....

Web sites can still use the https (secure) protocol and your information will be encrypted, even if it is not in the IE Trusted Sites list.

You should always accept the default security zone settings...

Internet Options>Security tab, click "reset all zones to default"....

You should install anti-virus software.... you should turn on IE's Phishing filter..... this stops you from going to sites that are known to be untrustworthy.

try this

Sub SaveAs_02()

'Nov 18, 2014
On Error GoTo errh
Dim cel
cel = Range("E1").Value   '<< file name e.g. file1, file001, book1,
ChDrive "e"
ChDir "e:\tmp\"

Dim fName

fName = Application.GetSaveAsFilename(cel, filefilter:="Excel Files <.xlsm> (*.xlsm), *.xlsm")
If fName <> False Then
If MsgBox("save as " & vbCr & fName, vbOKCancel) = vbCancel Then Exit Sub

Application.DisplayAlerts = False

ActiveWorkbook.SaveAs Filename:=fName, FileFormat:=52 'xlsm

Application.DisplayAlerts = True

Else
MsgBox "cancel"
End If
Exit Sub
errh:
MsgBox "wrong, try again"
End Sub
