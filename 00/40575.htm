janeane cru say:

Hi Christine,

Additional information will help us to identify the cause of this issue. Kindly answer the following questions:

  • How many computers are involved to this issue?
  • What is the exact error message that you have encountered?
  • What is the build number and version of your Windows?
  • The 4 user accounts on your PC, did you set all of it as an administrator?

In the meantime, we suggest that you enable the hidden administrator account in Windows 10 by following these steps using one of your user account:

  1. Click on Start.
  2. Type cmd, and right-click on it.
  3. Select Run as administrator.
  4. Type this command line, and then press Enter:
    net user administrator /active:yes
  5. You should see a message that the command completed successfully. Log out, and you’ll now see the Administrator account as a choice.
    Note: There's no password for the hidden administrator account.

After enabling the hidden administrator account, change the account type of your user accounts to gain administrative privileges:

  1. Press Windows key + I to open Settings.
  2. Click on Accounts.
  3. Click on Family & other people.
  4. Under Other people, select one of the user account, and then select Change account type.
  5. Under Account type, select Administrator from the drop down menu.
  6. Click on OK to complete.

After performing the steps above, we will need to disable the hidden administrator because we are using hidden administrator just for troubleshooting. Kindly follow these steps:

  1. Click on Start.
  2. Type cmd, and right-click on it.
  3. Select Run as administrator.
  4. Type this command line, and then press Enter:
    net user administrator /active:no
  5. The administrator account will now be disabled, and shouldn’t show up on the login screen anymore.

Let us know if you need further assistance.

Stopping Windows Defender from scanning your Computer automatically is not recommended. I'd recommend you to change the time when the scan starts. If you would like to do this anyways, please try the following

Method [1]: Disabling Windows Defender scheduled tasks from the Task Scheduler
  • From the Charms bar, click on Settings then select Control Panel
  • Click on System and Security and then click on Administrative Tools
  • Double-click on Task Scheduler
  • Under Task Scheduler Library, navigate to Microsoft > Windows > Windows Defender
  • From the right pane, right-click on Windows Defender Scheduled Scan and select Disable
    -     To see a screenshot, please click here

Method [2]: Adjusting automatic maintenance settings

  • From the Charms bar, click on Settings then select Control Panel
  • Click on System and Security and then click on Action Center

    -     To see a screenshot, please click here
