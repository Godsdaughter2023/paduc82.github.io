  1. Press Windows key + X on the keyboard.

  2. Click on Command Prompt (Admin).

  3. At the command prompt, type “takeown /f <foldername> /r /d y” without quotes and then press ENTER. (Replace <foldername> with the path of the folder).

don cuthbert say:

If you are pointing to the 16193 Sources, you need to specify the install.wim or install.esd file and index number.

Look in your 16193 ISO and find if it is install.wim or install.esd. At a Command Prompt, check what indexes are in that file using DISM. It may have both Pro and Home Editions, so you need to point to the right index.

DISM /Get-WimInfo /WimFile:H:\Sources\install.esd

Once you have found your Edition index in install.wim or install.esd you have to specify it:

DISM /Online /Cleanup-Image /RestoreHealth /Source:ESD:H:\Sources\install.esd:1 <-- (use the correct index number at the end)

However, as already indicated, if sfc /scannow find no errors, or repairs what it finds, you are probably OK already.
