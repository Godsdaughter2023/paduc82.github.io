How I got Windows 10 Preview to install on my Toshiba computer - Microsoft Community

cookarlen say:

How I got Windows 10 Preview to install on my Toshiba computer

  1. After giving up on trying to install the Windows 10 preview on my computer, I decided to reinstall Windows 7 Pro using the Toshiba provided disks.  After the install completed, I checked for and then installed all the updates reported. The updates included KB2990214 and excluded KB3035583 & KB2952664. Finally, I created a system image.
  2. Then I disabled my wireless and executed the setup program on the Windows 10 ISO. Windows 10 installed on my computer. I found that
    1.  I could not enable the wireless from either the display or the keyboard … I had to enable it by executing TBSWireless.exe in directory C:\PROGRAM FILES\TOSHIBA\TBS. 
    2. When I brought up the new tiled display, I worked fine for about five minutes. After that, it snapped to the lower left corner of the display and could not be moved
  3. I then shut down the computer, restored it using the above image, enabled the wireless and then installed more windows updates. The updates included KB2990214, KB3035583 & KB2952664. Finally, I then created a new system image.
  4. Same as step 2
  5. Same as step 3 EXCEPT The updates did not include KB2990214, KB3035583 & KB2952664.
  6. Same as step 2
  7. Same as step 5 EXCEPT Kaspersky Internet Security installed.
  8. Same as step 2

 

 

BOTTOM LINE:  After Windows 7 Pro was  reinstalled,  I disabled the computer wireless before executing the Windows 10 setup

Hope this helps someone
