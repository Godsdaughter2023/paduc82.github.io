jeeped say:
Sometimes it is easier to make something conspicuous by its absence. This formula counts the difference between the length of a alphanumeric string and the length of the same string with a certain character removed.

         

The character to look for is in column C. D2 has the formula,

=SUMPRODUCT(LEN($A$1:$A$10)-LEN(SUBSTITUTE($A$1:$A$10,C2,"")))

... which can be filled down as necessary. Change the $A$1:$A$10 range to suit the range containing your own 2095 labels. Expect a bit of processing lag when used on the 2900+ rows but it should be an acceptable amount on a modern computer.

jeeped say:
On a related note, you can quickly generate a list of numbers 0-9 and characters A-Z by putting this into a cell like C2 (as above),

=CHAR(ROWS($1:1)+47+((ROWS($1:1)>10)*7))

.. and filling down for 36 cells.

hansv mvp say:

If you mean the last used row in F:

 

Dim lngLastRow As Long
Dim rng As Range
lngLastRow = Range("F" & Rows.Count).End(xlUp).Row
Set rng = Range("A1:F" & lngLastRow)
' Optional: select this range
rng.Select

 

If you mean the last used row in the entire sheet:

 

Dim lngLastRow As Long
Dim rng As Range
lngLastRow = Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
Set rng = Range("A1:F" & lngLastRow)
' Optional: select this range
rng.Select

Word offers Sections, which among other things let you specify different headers/footers and number of columns within the same document. By default, a document has only one section.

Sections are inserted thorough the main menu bar's Insert > Break option and they can cause a page break or be continuous (no page break). Probably the latter is what you need.

Once you have inserted your continuous section break, you can choose Format > Columns to define the parameters. The change will be applied to the current section only.

To go back to one column, insert another section break.

jim gordon mvp say:

My suspicion based on what you have said is that there is a font problem of some sort.

You can do a test by booting to Safe Mode. Restart your Mac and hold the SHIFT key down as your Mac starts up. It will take your Mac a long long time to start up because Disk Utility runs in the background and other things are done to fix your files and file system.

Once you get to the desktop in Safe Mode, try running Word. Does it crash?

Starting up in Safe Mode starts you up with a limited font set. It also disables add-ins such as WebEx, which if not fully up to date can cause word crashes. While in Safe Mode, use Font Book application to verify all your fonts. Delete any duplicate and bad fonts you come across.
