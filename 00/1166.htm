ken sheridan say: I think you'll find that will interpret the return value of the DateAdd function as an arithmetical expression, which will evaluate to a very small number, and consequently set the default value to a date/time value around the end of the 19th century.  The DefaultValue property is always a string expression, regardless of the data type of the column in question, so should be wrapped in literal quotes characters:

If Weekday(Me.Dateordered,vbMonday) <5 Then
     Me.DateOrdered.DefaultValue = """" & DateAdd("d",1,Me.DateOrdered) & """"
Else
    Me.DateOrdered.DefaultValue = """" & DateAdd("d".3.Me.DateOrdered) & """"
End If
