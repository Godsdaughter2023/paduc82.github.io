Step 2:

Create an administrative user account using Command Prompt.

  1. Press “Windows Logo” + “X” keys on the keyboard.
  2. Select “Command Prompt (Admin)” from the menu to open the elevated Command Prompt.
  3. In the “Command Prompt”, type the following command and press “Enter” key on the keyboard:

net user username password /add

(In the above command, replace “username” with any name that you would like to keep for the new account and “password” with a strong password to secure the new account)

Step 3:

Copying old files to newly created User Profile

  1. Press “Windows Logo” + “X” keys on the keyboard and select “Windows Explorer” from the menu.
  2. Click on “View” tab and Click on “Options” at the top right corner.
  3. Click the “View” tab, and then select “Show hidden files, folders, and drives”.

  5. Click “Yes” to confirm, and then click on “OK”.
  6. Now, locate “C:\Users\Old_Username” folder, where C is the drive that Windows is installed on, and “Old_Username” is the name of the corrupted user profile you want to copy files from.

    • Ntuser.dat.log
    • Ntuser.ini

    bruce hagen ~ mvp say:

    That is just an e-mail that was forwarded as an attachment. Windows Live Mail or other e-mail clients will open it. Create a file association.

      1.     Download and save the attachment on the desktop.
      2.     Right click on the saved file.
      3.     Select: Open With from the pop-up menu.
      4.     Click: Choose [default] Program.
      5.     Select: (The e-mail client you use)
      6.     Check: Always use the selected program to open this kind of file.
      7.     Click OK.

    This should enable you to open these attachments directly from the e-mail now.
