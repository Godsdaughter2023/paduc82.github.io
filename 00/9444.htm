boblarson say:

Perhaps this will work:

 

Right([iName],Len([iName])-InStrRev([iName]," ")+1) AS LastName

 

The thing about InStrRev is that it gives you the position of the first item that matches the string that you want from the right but it uses the FORWARD count position.  It doesn't count from the end on back.

Use the previous record info from that field as its default value?

How can I setup a field in a form in access 2010 to use the previous record info from that field as its default value?
john p spencer say:

You can press control + " (quote) to duplicate the value in the prior record.

You can use VBA code to set the default value (temporary) of the control to it's current value.  Use the control's after update event.

 

Private Sub Somecontrol_AfterUpdate

   Me.SomeControl.Default = "=""" & Me.SomeControl & """"

End Sub

 

You can also do this in the after update event of the form if you want to do this for several controls.

Sample code:
Private Sub Form_AfterUpdate()
   Me.fText.DefaultValue = "=" & Chr(34) & Me.fText & Chr(34)
   Me.fLink.DefaultValue = "=" & Chr(34) & Me.fLink & Chr(34)
   Me.fSubject.DefaultValue = "=" & Chr(34) & Me.fSubject & Chr(34)
End Sub
