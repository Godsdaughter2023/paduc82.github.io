Does Contacts from Windows Mail Ver 6 mean Windows Contacts on Vista? If so, the easiest way to proceed is simply to copy the folder C:\Users\{user}\Contacts to the W7 machine, then copy its contents into the same folder on that machine. This should retain all your group data.

Once the Contacts folder is completely populated under W7, in Windows Live Mail's contacts window, select Import > Address book for current Windows user. This should import all of your contacts and translate your Windows Contacts Groups into Windows Live Contacts Categories.

Two points to note:

  1. Importing contacts will import all of the contacts in the source. If some of them are already present at the destination, you'll end up with duplicates. If your current list only consists of the previously imported ones, you might want to delete them all before re-importing from Windows Contacts.
  2. If you're signed in in Windows Live Mail when you conduct the import, the contacts you import will eventually synchronize to http://contacts.live.com, whence they will be available to you from any computer with an Internet connection. If you're not signed in, they will reside only on your computer.

Method 1: Run the SFC.

  1. Open Command Prompt.

  2. Select “Run as Administrator”.

  3. Type “sfc /scannow” without quotes and hit Enter.

Method 2: Run the DISM Tool.

If the issue persists, I would have you run the DISM tool to check the system health and will try to restore the files. 

  1. Click on 'Start'.

  2. Enter 'Command prompt' in the Search box.  

  3. In the list of results, swipe down on or right-click Command Prompt, and then tap or click 'Run as administrator'. 

  4. In the Administrator: Command Prompt window, type the following commands. Press the Enter key after each command:

DISM.exe /Online /Cleanup-image /Scanhealth 

DISM.exe /Online /Cleanup-image /Restorehealth

To close the Administrator: Command Prompt window, type Exit, and then press Enter.

Note: The tool might take 15-20 minutes to finish running, so please do not cancel it.

Method 3: Run the Microsoft Safety Scanner.

The Microsoft Safety Scanner is a free downloadable security tool that provides on-demand scanning and helps remove viruses, spyware, and other malicious software. It works with your existing antivirus software.

ionut lazar say: Hello,

Windows XP have a tool named Magnifier . Magnifier is intended to provide a minimum level of functionality for users with slight visual impairments. You could change the zoom from Magnification level (1=large - 9=very large). You find this tool on Start - All Programs - Accessories - Accessibility - Magnifier .
If you don't have a shortcut to this tool here, then search magnify.exe on Windows\system32 folder .

Good luck!

This feature is not available in Microsoft Skype for Windows 10 (the 12.X version), but is available in the Skype for Desktop version (the 8.X version).

dennisdivito say:

True.

The split screen functionality is built into the 8.x versions of Skype. Just click and drag the small screen with your video feed to the center of the screen. A check mark appears in the center of the screen, release the mouse button and voila, the screen shows both views in each half of the screen. To return; click and drag your video feed to the corner it started out in and it will diminish in size as it was before.
