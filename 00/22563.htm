karl timmermans [outlook mvp 2012-2018] say:

Thanks. So delete the accounts and redo as POP?  argh!  Besides the obvious advantage, do I a lose any 'functionality' using POP?

#1 - The only advantage is that you would be able to share your email folders across multiple devices where the accounts are also configured using IMAP

#2 - The one caveat with using IMAP is that your contact/calendar folders would be stored in folders with the added words "This Computer Only" in the folder description. #3 - If you reconfigure the accounts using POP and you have a copy of the PST file that was used by these accounts originally, you should be sure to set each POP account to use the existing PST file instead of letting Outlook create a new PST file for each account.
