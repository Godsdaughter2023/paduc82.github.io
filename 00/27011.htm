----------------FIXED!!!--------------------

Found this process in another thread which fixed the problem;

Jaysmi23nz replied on

In the end I uninstalled the apps using PowerShell and then installed them via the Store again. They now work perfectly.

1. Open PowerShell as Administrator

2. Delete the app from the computer using command:

Get-appxprovisionedpackage –online | where-object {$_.packagename –like "*windowscommunicationsapps*"} | remove-appxprovisionedpackage –online

3. Delete the app from your local account using command:

Get-AppxPackage *windowscommunicationsapps* | Remove-AppxPackage

4. Open Start and type wsreset.exe and hit enter to refresh the Windows Store

5. Open Store, search for and reinstall Mail & Calendar.

Note: If you only use one of the above codes, you will have an error in installing again. Also, the reinstall might take several hours where Store reports the app is 'Getting user data' or some such. Just wait it out and it does install successfully.

If you have other apps that don't work, simply replace 'windowscommunicationsapps' with the name of that app and repeat the process. All my apps are now working. A list of the Microsoft app names installed can be found in PowerShell by typing 

Get-appxprovisionedpackage –online | Out-GridView

Hope this helps!

a. user say:

Hello,

Welcome to Microsoft Community and thank you for posting your query.

I would like to inform you that you cannot create shortcut to Outlook new email or new task but you may create new email or new task template file in Desktop and use it as shortcut.

To do so, please refer to the following steps.

  • Open Outlook application.

  • Click on Home tab> New email.

  • In the message window, click the File tab.

  • Click Save As.

  • In the Save As dialog box, in the Save as type list, click Outlook Template.

  • In the File name box, type a name for your template, browse to Desktop location and then click Save.

Repeat the same step for Outlook new template.

 

Hope the information helps. Do reply if you need further assistance, we will be happy to help you.
