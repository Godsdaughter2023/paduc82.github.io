ken blake - microsoft mvp say:

On Tue, 2 Apr 2013 15:16:46 +0000, samdar23 wrote:

I have 6 GB of RAM, and Windows 7 32-bits. When i checked the system infomation, the RAM show 6 GB but only useable 2.94 GB. I checked the Resource Monitor and the Hardware Reserved use 3130 MB so i only have 3014 MB left.


My Question:


1. Why Hardware Reserved use so many memory?
2. Is it possible to decrease Hardware Reserved memory?

All 32-bit client versions of Windows (not just XP/Vista/7/8) have a
4GB address space (64-bit versions can use much more). That's the
theoretical upper limit beyond which you can not go.

But you can't use the entire  address space. Even though you have a
4GB address space, you can only use around 3.1GB of RAM. That's
because some of that space is used by hardware and is not available to
the operating system and applications. The amount you can
use varies, depending on what hardware
