Two items

#1 - For Outlook versions '2010 & earlier on Win10 - the dialog you're referring to will always appear regardless of whether or not you check "always use this app". You'll need to apply the following in order to correct that issue

Copy everything below between the lines (but exluding those lines) to a text file with a .reg extension:

-----------------------------------------------------------------------------
Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\htmlfile\shell\opennew\command]
 @="\"C:\\Program Files\\Internet Explorer\\IEXPLORE.EXE\" %1"
----------------------------------------------------------------------------

Once the file is saved - click on it - allow it to make the requisite change and problem will be solved

#2 - On a general note - the way to change a "default app" in Win10 is via

Start --> Settings --> System --> Default Apps
