d. Close all browsers and all applications running on your computer.
 
e. Reopen Fiddler 4, make sure File -> Capture Traffic is ticked.
 
f. Now Fiddler is ready. On the Fiddler 4, if there are some items in the main board, right click on the main board -> Remove -> All Sessions. This will remove the unnecessary logs so we can focus on the required logs.
 
g. Reproduce the issue by navigating to Office 365 admin portal via browser, when it exhibits the error message, go to the Fiddler 4 -> click File -> Save-> All Sessions -> type a name to store the capture as a .saz file.
