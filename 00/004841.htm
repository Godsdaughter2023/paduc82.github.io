doug robbins - mvp office apps & services (word) say:

Access the footer of the first Section of the document (the output of a mail merge will be a document with Section Breaks between the text for each record in the data source) and insert the following fields, using CTRL+F9 to insert each pair of field delimiters { }

{ Page } of { Sectionpages }

Then, run a macro containing the following code:

Dim i As Long
With ActiveDocument
    For i = 2 To .Sections.count
        .Sections(i).Footers(wdHeaderFooterPrimary).LinkToPrevious = True
    Next i
End With

Of course, if you have access to the Mail Merge Main Document and the data source, you could insert the above field codes into the Mail Merge Main Document and then re-execute the merge.

          Dim DataIn, DataOut
  Dim i As Long, j As Long, n As Long

  'Readin all input cells
  DataIn = Range("A1:B2").Value

  'Sum up
  For i = 1 To UBound(DataIn)
    n = n + DataIn(i, 2)
  Next

  'Create space and fill the output
  ReDim DataOut(1 To n, 1 To 1)
  n = 0
  For i = 1 To UBound(DataIn)
    For j = 1 To DataIn(i, 2)
      n = n + 1
      DataOut(n, 1) = DataIn(i, 1)
    Next
  Next

  'Write the result
  Range("D1").Resize(UBound(DataOut)).Value = DataOut
