How to fix Enhanced storage filter driver in Windows 8.1 - Microsoft Community

ohliz say:

How to fix Enhanced storage filter driver in Windows 8.1

nick dich say:

Hi Liz,

It might be that the main reason why your Enhances Storage Filter Driver status shows degraded is because of the fact that you are having an issue with updating Windows as what you have stated on your post. The Enhanced Storage Filter Driver is a kernel service of Windows and gets updated along with the the Operating System. First, we need to troubleshoot your issue regarding the Windows Update. Kindly visit the link provided here for the update troubleshooter: Windows Update Troubleshooter.

After a successful Windows update restarts your computer, the system services should have been restarted as well freshly updated. Should in any case the Enhanced Storage Filter Driver still shows degraded, you can follow the steps provided below to restore the default startup configuration for this kernel service:

  1. Press Windows button + R.
  2. Launch the Command Prompt by typing CMD and press Enter.
  3. Type the following commands:
    • sc config EhStorClass start= boot
    • sc start EhStorClass
  4. Close the Command Prompt and Restart the computer.

To unblock a software publisher:
 1. In Internet Explorer, click the Tools button, and then click Internet Options.
