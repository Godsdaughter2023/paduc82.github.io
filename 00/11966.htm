davidf2 say:

I have found that every once in a while copy and paste just doesn't work as I want it to and I use the suggestion Mary made above. In general if you are having problems copy and pasting or you don't want to paste the most recently copied object/text then open the Clipboard and paste from there. Remember to open the Clipboard before you start copying.

In Pub 2007 go to Edit > Clipboard and the Clipboard task pane will open on the left side of your page. As I said above I have found it best to open the clipboard before I try to copy multiple items. Now go to your source document and copy the text, picture or other object you want to paste and immediately check the clipboard to see if the copy was successful. Now you can go to another Pub file, a web page, a Word doc or any other source and copy up to 24 additional items. When you have copied all then you can select which of the copied objects/text in the clipboard task pane you want to paste. If you first click within an existing text box on the Pub page you can paste within that text box. If you don't then the text will be pasted as a new text box or new picture box on the page.

If this process of using the Clipboard does not work for you then please post back with the specific steps you take and details. Don't just say 'it doesn't work' as that will not help me help you.

Over the years I have frequently had problems pasting HTML formatted text from a web page directly into Word, Publisher and other docs. If you have problems copy and pasting from a web page you might have to go to paste special and choose an alternative to pasting HTML formatted text. Or you may even have to do a 'two step' and copy and paste into NotePad first to strip the HTML formatting, and then again copy that plain text in NotePad and paste into Publisher.

You may try following to import an Excel table into an existing list in SharePoint Online:

Note: the cells you want to copy must match the data type in the default view of the list.

1.Use Internet Explorer to sign in your SharePoint site and open the list.

2.Open the Excel worksheet which contains the data you want to import.

4.In the list, click Quick edit in the ribbon.

5.This is an important tip:

6.Press Ctrl + V to paste the data.

If you get the following window, click Allow access. The items will be added to the bottom of the list.

7.click Exist quick edit to complete the process.

tim liu msft say:

Hi Tanjia,

In Microsoft Forms, there are two types of forms, one is Form, the other is Quiz. Form is used to collect some feedback from users (more like a survey). Quiz is used to make some tests with users.

If we create a new Form, we can Add questions, select Text type, but we cannot add answers (like your first screenshot). This is because for the question, different users can give different answers. It likes we ask user "Which country you think will win the World Cup 2018?"

If we create a new Quiz, add a text type question, we can add answer to that question. We want users give the correct answer for a question like "Which country hosts FIFA World Cup 2018?"

So, if you want to make some tests for users, you could select to create a new quiz. It has the option as your second screenshot.
