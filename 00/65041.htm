Basically you want to use the profile name you assigned to the temporary account, for your original account you're using now.

 

Ok. Yes, this can be done, however it requires 1600msp split between the two accounts (so 800 each). The only way to aquire that name for your original account is to change it, this in turn "frees" it and will allow it to be used again on a new or existing account.
