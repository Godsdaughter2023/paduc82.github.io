zeppa01 say:

WiFi USB Dongle SSID

I have purchased a Ralink Dual Band USB WiFi Adaptor in order to receive 5ghz on my Asus Laptop.  I have installed the program but can't connect because the system keeps requesting the SSID.  I can't find this on the packaging or anywhere else. Can anyone suggest where I can find the SSID?  Thanks 
ralf_g say:

I have purchased a Ralink Dual Band USB WiFi Adaptor in order to receive 5ghz on my Asus Laptop.  I have installed the program but can't connect because the system keeps requesting the SSID.  I can't find this on the packaging or anywhere else. Can anyone suggest where I can find the SSID?  Thanks 

The SSID is the network name/number identification given to the router/access point you are trying to connect to.

The network name can be almost anything, depending on what the user or person who set up the router/access point decided to call it. There would be settings in the router to change the SSID and for whether or not to broadcast that name. If it is set to broadcast the name it should be visible to your WiFi dongle and show up when you click on the Windows network icon in the notification area on the desktop.

If you are using a RAlink utility to manage your WiFi connections you'll have to check in the user manual/help for that product.
