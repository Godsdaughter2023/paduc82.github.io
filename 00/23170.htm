You can try these troubleshooting steps:

    • Press Start, then swipe left to the App list.
    • Select Settings > About > More info. 
    • If there are many WLAN access points nearby, try to use a different wireless channel in your WLAN access point.

3. If you are using a public WLAN access point, it is possible that all IP addresses are in use. Try to create a connection again later.

4. On your Lumia phone, delete non functioning WLAN access point (internet connection) and connect it again:

  • Disconnect the WLAN connection if connected.
  • Press Start, then swipe to left to the App list.
  • Select Settings > Wifi.
  • Press Advanced at the bottom of the screen.
  • Tap and hold on the access point name and select Delete.
  • Now, add the access point again.
  • On the Advanced screen, tap Add, type the network name in the Network name box, and then tap Add.
  • On the Sign in screen, type the password in the Password box, and then tap Done.
  • Tap the Back key once finished.
