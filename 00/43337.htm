    Hi memrymaker, 


    Welcome to Microsoft Answers Forum.

    Microsoft 6to4 #2 Adapter is used for communication in mixed environment including both IPv4 and IPv6.

    My suggestion is to uninstall the component first and then reinstall it.

    Step 1: Uninstall Microsoft 6 to 4 adapter:  

    1.    Click on Start and type device manager in the field provided.

    2.    Click on Continue when UAC prompts.

    3.    Click on the + sign next to the device you want to uninstall to expand it.

    4.    Select Uninstall.

    5.    Click on Yes to confirm.

     Step 2: Reinstall Microsoft 6 to 4 adapter:

    1.     In Device Manager, Click on Network Adapter.

    2.     Go to Action tab and select Add Legacy Hardware. This opens Add Hardware wizard.

    3.     Click on Next.

    4.     Select “Install the hardware that I manually select from a list”.

    5.     Under Network Adapters from the list choose Microsoft – Microsoft 6to4 Adapter.

    6.     Click Next.

    7.     Select Finish the Wizard.


    Hope you would be able to reinstall the adapter successfully after following the steps mentioned above.

    burningorange say:

    I solved the issue and now have full surround sound (but only with the NetGear PTV-3000 only). Note that your suggestion (reply number 1 above) was not the solution. The latest Surface audio drivers are not at fault.

    I had to install the latest Intel Graphics driver (10.18.10.3960) manually. Surround now works with the NetGear PTV-3000 (firmware 2.4.53).

    The steps to install the graphics driver are described at the Intel help page here. Use the Zip file, and do not click on Setup.exe, but perform the manual steps described in that page.

    Caveat 1: surround does not work with the Microsoft Wireless Display Adapter (still plays from FL & FR speakers, not from the others, even though IWDAUD_Device_06 installs fine and Sound>Playback Devices>Intel Speakers shows all 6 speakers). The adapter needs a firmware update.

    Caveat 2: these latest drivers are not validated for the Surface Pro 3 (which is why running setup.exe in the intel driver package will not proceed further than the information box). A windows update once re-installed the earlier 10.18.10.3946 drivers (the latest validated drivers for the Surface Pro 3) which stops Miracast-surround-sound from working, and I had to re-install the 3960 drivers to get surround sound to work again.
