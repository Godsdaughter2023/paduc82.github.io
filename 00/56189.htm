Hi ByteBowser,

Welcome to Windows Vista Answers Forums!

Starting with Windows XP SP2, AutoPlay is enabled for removable drives also. This includes ZIP drives and USB mass storage devices. If you enable the settings to disable AutoPlay, you can disable AutoPlay on a CD drive, on removable media drives, on all drives.

When you set the option as Disable Autoplay options, it is set as default and no changes can be done.

However, To selectively disable specific Autorun features, you must change the NoDriveTypeAutoRun entry in one of the following registry key subkeys:

HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer\
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\policies\Explorer\

The following table shows the settings for the NoDriveTypeAutoRun registry entry.

 Value

Meaning

0x1 or 0x80

Disables AutoRun on drives of unknown type

0x4

Disables AutoRun on removable drives

0x8

Disables AutoRun on fixed drives

0x10

Disables AutoRun on network drives

0x20

Disables AutoRun on CD-ROM drives

0x40

Disables AutoRun on RAM disks

0xFF

Disables AutoRun on all kinds of drives
