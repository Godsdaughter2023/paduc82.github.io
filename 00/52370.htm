I will go through each of these steps so that you can see exactly what needs to be done in order to properly get remote desktop working. First, make sure you set a password for your administrator account. If your version of Remote Desktop Connections supports it, it will say Network Level Authentication supported.

Now open remote desktop connection, type in the computer name and click Connect.
