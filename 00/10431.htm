ossiemac say:

Hello again Moi_NZ,

So what about a VBA solution to set the Conditional format on each row. You don't need to leave the VBA in the workbook after the conditional format is set. Try the code below. Not sure of your expertise with VBA so some guidelines to install.

  • Alt and F11 to open the VBA editor.
  • Select menu item Insert -> Module.
  • Copy the code below and paste into the editor.
  • See my comments in the code re editing the worksheet name, range to format and also the RGB color codes.
  • Alt and F11 toggles between the VBA editor and the worksheet.

To find your required RGB color code.

Go to Manage the conditional format and then edit the condition and where you select the color, click the drop down and select More colors. Select the Custom tab and note the RGB color codes and edit them in the VBA code as required.

To run the code just position the cursor anywhere within the code in the VBA editor and then press F5.

If you want to keep the code in the workbook then save the workbook as Macro enabled and make sure you set macro security options to allow macros ("Disable all macros with notification" options should be OK.)

If you don't want to keep the code in the workbook then Right the Module 1 name in the left column of the Editor and select Remove module.

Sub Macro1()
    Dim rngCondFormat As Range
    Dim rngRow As Range
   
    With Worksheets("Sheet1")   'Edit "Sheet1" to your sheet name
   
        'Following line sets cond format range from cell B2 to last cell in column E
        Set rngCondFormat = .Range(.Cells(2, "B"), .Cells(.Rows.Count, "E").End(xlUp))
       
        rngCondFormat.FormatConditions.Delete   'Removes any existing conditional formats
    End With
   
    For Each rngRow In rngCondFormat.Rows   'Loop through each row in the range
        With rngRow
            .FormatConditions.AddDatabar
            .FormatConditions(rngRow.FormatConditions.Count).ShowValue = True
            .FormatConditions(rngRow.FormatConditions.Count).SetLastPriority
       
            With .FormatConditions(1)
                .MinPoint.Modify newtype:=xlConditionValueAutomaticMin
                .MaxPoint.Modify newtype:=xlConditionValueAutomaticMax
                With .BarColor
                    .Color = RGB(255, 192, 0)  'See notes for finding RGB color values
                    .TintAndShade = 0
                End With
           
                .BarFillType = xlDataBarFillGradient    'Sets Gradient fill
                'Comment out gradient fill line and uncomment following line for solid fill
                '.BarFillType = xlDataBarFillSolid      'Sets solid fill
                .Direction = xlContext
                .NegativeBarFormat.ColorType = xlDataBarColor
                .BarBorder.Type = xlDataBarBorderNone
                .AxisPosition = xlDataBarAxisAutomatic
               
                With .AxisColor
                    .Color = 0
                    .TintAndShade = 0
                End With
               
                With .NegativeBarFormat.Color
                    .Color = 255
                    .TintAndShade = 0
                End With
           
            End With
           
        End With
       
    Next rngRow
End Sub
