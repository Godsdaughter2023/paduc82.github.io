HLOOKUP - Microsoft Community

ashleytebbs say:

HLOOKUP

Does anyone understand this? the green box in the bottom right is a hint for this task but I guess I'm confused what I'm supposed to put in the formula instead since the answer in H is definitely wrong. Thanks in advance to anyone who can point me in the right direction!
jasles say:

Here's the HLOOKUP() function:

HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])

For your problem, the table_array is the product pricing table. The function always looks in the first row of the table to find the lookup_value.

So you could search for "# Shipped", "Blue", or "Red". You want to search for "Blue" or "Red", which is in column C.

Once it finds the color (lookup_value), it then goes down the table_array to find the row number of the table that matches row_index_num.
For your problem, they purposely made the # Shipped in column D equal to the row_index_num in your table, thus allowing you to use the value in that column as your row_index_num.

The function then returns the value where those two intersect.

Looking at the HLOOKUP function again:

HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])

Here it is for how you need it:

HLOOKUP("Red","Product Pricing Table","# Shipped",FALSE)

Or the actual formula in cell H9 would be:

=HLOOKUP(C9,$O$9:$Q$18,D9,FALSE)

To explain it a bit more, the B2 in the function could be replaced with the word "Red", or "Blue" instead of the cell reference that has the color as a string. Same with the C2, it could be replace with a literal number instead of a reference to the cell containing the number.

You want the table_array ( the range of cells from O9 to Q18) to be absolute as you don't want that to change as you copy the formula down the column.

I hope that helps explain it better for you.
