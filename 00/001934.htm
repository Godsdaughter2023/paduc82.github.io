greginmich say:

In the Windows Defender Security Center, the Quarantined Threats list on the Scan History page corresponds to the Quarantined Items page in the classic UI, and that list will display any items that have actually been quarantined, as well as provide the options to remove or restore any of those items – as we can see in this screenshot:

 

You can either click on the Remove all button or click on a down arrow in order to display the option buttons for an individual threat. But remember that the recommended action can sometimes be wrong when a harmless file is quarantined by mistake (a false positive), and that's the reason why threats are quarantined rather than removed automatically. The items in the Quarantine could be safely left there indefinitely; but they'll be automatically removed after 90 days, by default.

The number of days that the item is kept in the Quarantine can be reconfigured with this preference setting:

-QuarantinePurgeItemsAfterDelay

Specifies the number of days to keep items in the Quarantine folder. If you specify a value of zero for this parameter, items stay in the Quarantine folder indefinitely.

https://docs.microsoft.com/en-us/powershell/module/defender/set-mppreference?view=win10-ps

For example, to have quarantined items automatically removed after just a week, run this command at the Administrator PowerShell prompt:

Set-MpPreference -QuarantinePurgeItemsAfterDelay 7

Asif_A  -

Since you mention "2000 numbers," maybe you want a histogram (of your data) instead of a normal density curve (based on the NormDist function).

(1) A histogram is usually a vertical-bar chart of a frequency distribution. It's an important way to show the shape of a data set.

A frequency distribution is a table with buckets or bins or upper-limit values in one column and corresponding frequencies, i.e., counts, of the data in an adjacent column.

Excel's Histogram tool of the Analysis ToolPak allows you to specify the bins, i.e., upper limit of each bucket, and it prepares the frequency distribution and corresponding Column chart type. Since a Column chart has equally-spaced columns, the bins you specify should be equally spaced.

Instead of the Histogram tool, you could use the array-entered FREQUENCY function or COUNTIF functions to prepare the frequency distribution, and you could create a Column chart. Or, you could use a pivot table or pivot chart.

Another alternative is my free Better Histogram add-in, available from the Histogram page atwww.treeplan.com.

(2) The NormDist worksheet function can be used to construct a chart of the probability density of the normal (Gaussian) distribution. A histogram of a data set may or may not have the smooth symmetric shape of the Gaussian bell curve.

How do i format cells with a duration of time in hours and - Microsoft Community

htkporg say:

How do i format cells with a duration of time in hours and minutes?

Example 1: two cells,  entered as 1:30 (1 hour 30 minute), and another as 0:45 with a sum coming out to 2:15?

 

Example 2: Two cells with clock time (3:00 PM , 10:00 AM) to calculate the elapsed time (5 hours).

 

 

 

hansv mvp say:

1. With 1:30 in A1 and 0:45 in B1, enter the formula =A1+B1 in C1 to get the sum.

If the total hours can be more than 24, format C1 with the custom format [h]:mm

 

2. With 10:00 AM in A2 and 3:00 PM in B2, enter the formula =B2-A2 in C2 to get the elapsed time. Format C2 as h:mm.

If the end time can be after midnight, use =B2-A2+(B2<A2)

About the first part: without being able to see the data it's hard to know what's going on.

About the second part: when you enter data in the form 1:23, Excel assumes that the first number is hours and the second number is minutes. If you want to enter minutes and seconds, you *must* enter the hours too: 0:1:23.

In the format mm:ss;@, the @ part is used for text values entered into the cell. In most situations you can simply omit the ;@. You only need it if you want to align text values in a specific way. Try changing the format to mm:ss. You probably won't see any difference.
