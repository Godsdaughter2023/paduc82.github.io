      alankz7 say:

      I had a similar problem after cloning using the Samsung software, and found that it was due to the fact that the location of the recovery partition WINRETOOLS had changed due to the reordering.  To check if this is the problem run the command:

      reagentc /info

      at a command prompt (Admin) window.  Running the original HDD you will likely see something like:

      C:\Windows\system32>reagentc /info
      Windows Recovery Environment (Windows RE) and system reset configuration
      Information:

          Windows RE status:         Enabled
          Windows RE location:       \\?\GLOBALROOT\device\harddisk0\partition1\Recovery\WindowsRE
          Boot Configuration Data (BCD) identifier:  <identifier, etc....>

      If you have the same problem as me, then you will see this when you run the same command on the SSD:

      C:\Windows\system32>reagentc /info
      Windows Recovery Environment (Windows RE) and system reset configuration
      Information:

          Windows RE status:         Disabled
          Windows RE location:
          Boot Configuration Data (BCD) identifier: <identifier, etc....>

      If so, then I believe that windows disabled the recovery options because it couldn't find the recovery partition.  The resolution is to point to the new windows RE location using:

      reagentc /setreimage /path \\?\GLOBALROOT\device\harddisk0\partition1\Recovery\WindowsRE

      where the path is the same as your working HDD, but partition1 is replaced with partition2, partition3 as necessary (in your case probably was partition3 and now needs to be partition2).  Then attempt:

      reagentc /enable

      If all is well you will see REAGENTC.EXE: Operation Successful after each command.

        Change Wireless Channel

        Login Information

        • Gateway - 192.168.0.1
        • Username - cusadmin/chtruser/chtrtech
        • Password - password/charter

        Change Wireless Channel

        1. Open a web browser.
        2. In the address bar at the top type in 192.168.0.1
        3. If asked for a user name and password use (cusadmin/password). If your ISP is Charter you can use chtruser/charter or chtrtech/charter.
        4. In the menu at the top, select Setup.
        5. Under the Wireless section on the left, select Basic.
        6. For New Channel select (1,3,9,11).
        7. Select Apply.
