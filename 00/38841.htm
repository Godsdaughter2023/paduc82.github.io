- Hold down the shift key on your keyboard while clicking the Power button on the screen (while on the login screen)
- Continue to hold down the shift key while clicking Restart
- Continue to hold down the shift key until the Advanced Recovery Options menu appears
- Click Troubleshoot
- Click Advanced options
- Click Command Prompt
- At the command prompt, enter: net user administrator /active:yes
- Close command prompt, and restart
- You should now see an Administrator account you can select (this has no password)

If the above steps do not work, you may need to use a Windows 10 recovery/installation media boot-able USB. This can be created by selecting 'Download tool now' on the following website: http://www.microsoft.com/software-download/wind...

Insert the USB, boot into the USB, and follow these steps:
- Select Next
- Select Repair (bottom left-hand corner)
- Select Troubleshoot
- Select Command Prompt
- At the command prompt, enter: net user administrator /active:yes
(or)
- Enter the following to get your username to use below: net user
- Enter: net localgroup administrators <Your username here> /add

What is the make and model number of the computer?

The fans purpose is to cool down the interior of the PC or, in the case of the CPU fan, to keep the processor cool. Most motherboards have temperature sensors which regulate when and at what speed the fan works.

Initially the fan spins at full speed until the desired reduction is temperature is achieved and then the fan either slows down or, in some cases, stops completely until the temperature builds up again. Once the temperature hits its top limit the fan is initiated again and the cooling process starts all over again. If the fan is on continuously then there is a possibility that the inside temperature of your PC is hotter than it should be.

rajithr say:

Hi,

 

Follow these methods:

Method 1: I would suggest you to run the Power troubleshooter to adjust your computer's power settings, and check if it works properly.
Open the Power troubleshooter
http://windows.microsoft.com/en-US/windows7/Open-the-Power-troubleshooter

 

Method 2: You may also try creating or changing power plan.

a. Open Power Options by clicking the Start button, clicking Control Panel, clicking System and Security, and then clicking Power Options.

b. Under the current power plan, click Change plan settings.

c. On the Change settings for the plan page, click on Change advanced power settings.

d. Adjust the sleep options in advanced power settings and check.

 

Method 3: You can restore power setting to default by executing the command from an elevated command prompt.

Open the Command Prompt window by clicking the Start button, clicking All Programs, clicking Accessories, right click on Command Prompt, and then select Run as administrator. Type the following command in command prompt and press enter.

powercfg.exe -restoredefaultschemes

 

Method 4: I would also suggest you to check if Windows is configured to hibernate after an idle time.

Check the section: How can I wake my computer from Sleep or hibernation from the link:

Sleep and hibernation: frequently asked questions
http://windows.microsoft.com/en-US/windows7/Sleep-and-hibernation-frequently-asked-questions

 

Refer this article: Troubleshoot problems waking computer from sleep mode
http://support.microsoft.com/kb/266283
