With 8GB of physical memory it is going to depend on what you are doing as whether Windows is using Virtual Memory or not. Windows will only employ Virtual Memory when physical memory is nearly full.

You don't want it to use Virtual Memory by choice as this is hard disk based and much slower than RAM however never disable VM altogether as this can cause odd problems.

Reset the system back to Automatically manage paging file size for all drives and let Windows use it as needed, there is little benefit in changing these settings.

The Task Manager > Performance tab along with the Resource Manager button shows you how memory is being used.

hairyfool say:

With XP there was some benefit in terms of paging file usage by manipulating the settings (fixed swap file size, dedicated drive etc.) With each iteration of Windows the integrated algorithms the manager uses to control Swap file has got better.

Unless you go to the extreme of a dedicated hard disk or even SSD the performance gains by changing from the default are minimal.

In your example Windows does not see the sessions as independent tasks so cannot identify what can be rotated out so you are quite right to see it as not using the swap file but that has nothing to do with the manager or its configuration.

How To Find Wireless Passphrase (WPA/WEP)

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.123.254) into the address bar of the internet browser.
  4. When prompted, enter in “admin” for the user name and by default the password should be blank.
  5. From the top menu select Wireless.
  6. From the sub menu select Security.
  7. Your passphrase will be listed as Key or Passphrase.

 

Changing Wireless Security  Type/ Disabling Wireless Security

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.123.254) into the address bar of the internet browser.
  4. When prompted, enter in “admin” for the user name and by default the password should be blank.
  5. From the top menu select Wireless.
  6. From the sub menu select Security.
  7. Select the encryption type from the line marked authentication type
  8. Enter The passphrase you want into the Key or Passphrase line.
  9. Select apply.

techfreak_ say:

Forwarding is controlled at the account level from your account settings on Skype.com or in the client. You can only forward to one number at any given time. No support currently exists for custom voicemail greetings. They are generic. Any incoming calls forwarded to Skype's voicemail are temporarily stored as audio messages in your Skype chat. Voicemail greeting and storage on Skype only occurs if you are forwarding your unanswered calls to Skype's voicemail, not to another number.

In the scenario your described, the voicemail greeting and storage for each number would be configured at the provider's end of each of your mobile numbers. This is due to the destination being those providers and not Skype.  If those mobile numbers do not have their own individual voicemail service enabled, no voicemail could be left.
