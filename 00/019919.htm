ipellant say:

Insider Build 10525 is currently downloading (19 August).

If you are on the Insider Program and have Build 10240 installed and Activated then you will receive the upgrade via Windows Update.

Do not attempt a clean install unless you have a valid license key written down (on paper).

Clean installs will require a key to complete.

Upgrades over an installed and activated Build 10240 do not require a license key.

There are some known problems in Build 10525. Do not upgrade it over your Windows 10240 if it is your primary / work computer.

If you have 10240 installed and activated, you may join the Insider by using the Settings > Windows Update > Advanced Options > Get Insider builds.   Do not do this if you are not prepared to help test and report problems.

Do not use insider builds on your main computer.  If you have not already upgraded to build 10525, go to Settings/System/Upgrade and click on the Advanced button.  You will find the option to leave the Insider Program.
