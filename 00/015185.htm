jay freedman say:
The "Show white space" option in the Options dialog also gets turned on and off if you point to the space between two pages and double-click there. That's probably how it got turned off in the first place.

That setting applies to all documents you open in that copy of Word. If you take the same document to another computer where the option is turned on, you will see the headers and page numbers in them. Take it back to the computer where the option is turned off, and the headers are invisible.

I don't know what you have the Ctrl+F3 keystroke assigned to. By default, it invokes the Spike, a somewhat obscure feature similar to the Cut command but that collects all the items into a special AutoText entry.

Setting Office 365 apps, like Excel, as the default program to launch - Microsoft Community

dr charlesr meyer say:

Setting Office 365 apps, like Excel, as the default program to launch non-typical file types, such as .iif files

When you right click on a file name in File Explorer in Windows 10, you have the option to "open with" a specific launch application.  Working with QuickBooks, I have Tab delimited files with an .iif file extension that I want to be able to launch in Excel by default so I can view and manipulate their contents.  But I'm using Office 365, so Excel isn't actually installed on my workstation, and it isn't offered as an "open with" program option.  Apparently, from the posts I've encountered, this is a recurring issue with Windows 10 and Office 365.  I think I've found a workaround:

Create a text file with Notepad and save it as EXCEL.EXE somewhere on your PC.  You can rename an existing file as an alternative.

Right click on the file type you want to have defaulted to Excel, and pick "choose another app."  When the window pops up, choose "More Apps," scroll to the bottom of the list and search for an app on your PC.

Point the search to the EXCEL.EXE file you created.  It makes no different that it isn't an actual executable program file.  When you "Open" it, it will cause Office3 365 to launch its Web based copy of Excel and you're good to go

You should now be able to go into Settings and set Excel as the default launch program for your unconventional file type.
