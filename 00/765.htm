b. (brord) say:

(mail)merge field does not show all the info from the excel sheet cell

I'm trying to put a note from excel in a word document through a mailmerge. This is not working correctly since the note is cut in the middle. What can I do to get the whole note in my mailing. 

To make it simple, excel gives me info in word, but the info that it takes is being cut. (In excel it's al there)

Thanks in advance

a. user say:

This may be a "mixed data types in the same column" problem. I would expect to get it if

 a. the first 8 cells in the text column had "short texts" (fewer than around 256 characters), and

 b. your note is in the same column lower down.

If that is the situation, you may be able to fix it by putting a long note in one of the first 8 cells in the column. Some people do that by inserting a dummy data row under the title row, with a "long note" in the appropriate column, then excluding it or discarding it from the merge.

[If it is that. it's because the default method that Word uses to get data from Excel (OLE DB) treats Excel data as a database, and wants to assign a specific data type to each column. It applies various rules if the types are mixed. Unfortunately, it regards TEXT (short string, up to 255 or 256 long) and MEMO (longer string) as different types.]

linley meslier say:
You can use Windows Explorer to move and copy files to and from your mobile device:
1. With ActiveSync running, click Explore. The Explorer folder appears.
2. Click the file that you want to move or copy.
3. On the Edit menu, click Copy or Cut.
4. Right-click the location where you want to place the file, and then click Paste.
You can also use Windows Explorer on your desktop PC to do this:
1. Start Windows Explorer.
2. In the folder list in the left pane, click Mobile Device. Your mobile device appears in the right pane.
3. Double-click the icon for the mobile device.
4. Click the file that you want to move or copy.
5. On the Edit menu, click Copy or Cut.
6. Right-click the location where you want to place the file, and then click Paste.

 

Hope this helps.
