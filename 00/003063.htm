bob jones aka: cybertaz say:

Object Linking & Embedding (OLE) is a Windows technology which is not as richly implemented on the Mac platform. It depends on which object type was pasted into the Word document.

Long story short, if pasted as a Microsoft Excel Worksheet Object you can edit & revise the data in Excel & update the link, but Word cannot modify the linked object itself other than scaling as you would any other image. If you want to change the number of rows/columns, alter their height/width, color, etc. you need to do so in Excel, then replace the object in Word with a new version. (The same is true for the PDF, PNG & TIFF object types.)

If the object type is RTF, HTML, or Unformatted Text Word's Table features can be used.
