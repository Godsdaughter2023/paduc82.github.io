saltgrass say:

The Mini Partition tool has a problem with the MSR (128 GB) partition.  One option is to delete the partition, extend or resize the other partitions and leave 128 GB of unallocated space in the same relative position as it is now.  Then you can use Diskpart to recreate the partition in that 128 GB of unallocated space.

When you recreate the partition, make sure the 128 GB is the only unallocated space on the drive.  Windows will run without the MSR partition, so you can wait until you boot back into Windows to recreate it or boot to recovery media which has a Command Prompt Window.

Please understand, my response is for your question about resizing a partition.  I have not gotten into the 9355 situation but I was not aware there were any such problems related to updates.

With Diskpart, you would select the correct disk and then use the command below to recreate the MSR partition.

create par msr

It does not have to be formatted, just created.
