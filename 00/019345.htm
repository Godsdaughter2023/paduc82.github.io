sheeloo say:

There's no built-in function for that in Excel, but you can create your own, with a bit of VBA.

 

See http://blog.contextures.com/archives/2010/12/13/get-the-url-from-an-excel-hyperlink/

 

Press ALT-F11 to open VB Editor

Choose Insert->Module

 

Paste the code below into the right pane

 

Function HLink(rng As Range) As String
'extract URL from hyperlink
'posted by Rick Rothstein
  If rng(1).Hyperlinks.Count Then HLink = rng.Hyperlinks(1).Address
End Function

 

Close the Editor and save your workbook as .xlsm (macro need to be enabled for this to work)

then use

=HLink(A1) in any cell other than A1, with your hyperlink in cell A1
