Here are two possible explanations.

(A) If you are using an Excel VBA add-in which correctly uses the pre-2007 methods for adding a custom tool bar or menu item, and if it is correctly installed, then there will be an "Add-Ins" tab added to the ribbon with a "Menu Commands" group.

(B) Microsoft added some extra security features in July 2016.

Depending on your system and settings and file location, you may have to unblock downloaded files before use.

In your Windows file manager, right-click the XLA or XLAM file, choose Properties > General, check the Unblock box, and click OK.

Then start Excel and use File > Open to load the add-in file.

a. user say:

Hi Jake,

 

Let me help you in finding your emails or get them back.

 

Please be informed that there is no guarantee that the messages that you are looking for will all be recovered when you use the self-recovery feature in Outlook.com. Emails that were recently deleted from your account have a big chance to be retrieved. The system can recover messages but not the folder itself. Recovered messages will be presented in your Deleted folder. Also, emails will not be recovered via recovery link if they were deleted from email clients like Windows Live Mail, Microsoft Office Outlook, etc.

 

If the self-recovery feature is unable to retrieve your emails, it is possible that those emails have already been purged from our system. Emails are beyond retrieval in such case.

 

Meanwhile, here are other possible reasons why you are unable to find your emails:

 

1. Emails got deleted due to Inactivity.

When users don't sign in for more than 365 days or within the first 10 days after signing up for an account, the system will flag it as Inactive. After an account becomes inactive, all messages, folders, and contacts are deleted. Incoming messages will be sent back to the sender as undeliverable.

 

Also, if you were accessing your emails on a mail client such as Windows Live Mail or through your mobile device, but didn’t log into your web account for over 365 days, you may lose emails due to inactivity in web account.

 

2. Account has been hacked or compromised.

You may lose emails if your account was hacked and the hacker had already deleted emails in your account.

 

3. Conversation threading is enabled.

If the feature Conversation threading is turned on wherein all emails with the same subject line are grouped together. You can click here to know more about this feature.

 

4. Schedule Cleanup is set.

This feature can be set to automatically delete emails. You may refer to this link for more information about it.

 

If you have other questions or clarifications, please let us know.

 

Thank you.

Neil

dale howard [mvp] say:

Hugosthere --

Create a custom Base Calendar named Weekend Work Only.  In this custom calendar, set the working schedule to be Saturday and Sunday only from 8:00 AM to 12:00 PM and 1:00 PM to 5:00 PM.  Set Monday through Friday as nonworking time.

Next, you will need to double-click the task name to display the Task Information dialog and then click the Advanced tab.  Click the Calendar pick list at the bottom of the page and select the Weekend Work Only custom calendar.  Select the Scheduling Ignores Resource Calendars checkbox and then click the OK button.

By applying the Weekend Work Only task calendar, you will override the schedule of the task and any resources assigned to the task.  Hope this helps.
