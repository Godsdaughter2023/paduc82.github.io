I'm trying to change the data source from another workbook for all - Microsoft Community

poonamsaini say:

I'm trying to change the data source from another workbook for all pivot tables in a workbook (each worksheet have 4 pivot table)? Can this be done in one shot?

Currently I am doing manually? Is there way to do in one shot? Data is from another work book. New to VBA
bill manville say:

Each PivotTable has a PivotCache (and many pivot tables may share a PivotCache if they are built from the same source data).  Each PivotCache built from data in an Excel workbook has a SourceData property that defines where the data comes from.
It might be something like "[book2.xls]Sheet1!R1C1:R234:C10"

Oddly it is expressed in R1C1 notation rather than A1 notation.

 

So, for example, if all your pivottable data comes from Sheet1 in workbook "C:\temp\MyBook.xlsx" in the region A1:J345 you might use

 

Sub ReConnectPivots()
  Dim PC As PivotCache
  For Each PC In ActiveWorkbook.PivotCaches
    PC.SourceData = "'C:\Temp\[MyBook.xlsx]Sheet1'!R1C1:R345C10"
  Next
End Sub

 

If MyBook.xlsx was open then you could get the address automatically, substitute this for the line above:

 

    PC.SourceData = Workbooks("MyBook.xlsx").Sheets("Sheet1").Range("A1").CurrentRegion.Address(external:=True, referenceStyle:=xlR1C1)

Try:

 

Sub Demo()
Application.ScreenUpdating = False
Dim i As Long, Rng As Range
With ActiveDocument
  For i = .Fields.Count To 1 Step -1
    With .Fields(i)
      If .Type = wdFieldCitation Then
        Set Rng = .Result
        .Cut
        With Rng
          .Endnotes.Add Rng
          .MoveEnd wdCharacter, 1
          .Endnotes(1).Range.Characters.Last.Previous.Paste
        End With
      End If
    End With
  Next
End With
Application.ScreenUpdating = True
End Sub

Hi,

You're correct. If you want to assign the result of DMax() to the box next to the button, then you'll need to use its name. Also, the DMax() arguments are expected to be Text, which you need to delimit with quotes. For example:

Me.TextboxName = Nz(DMax("CatalogNumber, "Catalog_Number"),0)+1

However, if the form is not bound to the Catalog_Number table, then you'll also need a way to assign the number to the Catalog_Number table or you'll always get the same result from DMax().
