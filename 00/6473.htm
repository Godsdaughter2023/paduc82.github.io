It's a really weird f******g tablet, it only has ONE micro-usb port, a microSD port, an HDMI port and that's it. The micro-USB port is used for charging. There are no regular USB ports, which I guess is standard for tablets.

Your first post seems to say that you CAN boot off USB and that the install freezes at some later step?  If so, disregard the following ;) 

If this is not a UEFI boot issue, it may be that your USB port is not in USB mode.  Tablets with only one USB port have to choose between using the port for input power (charging) or for USB (where the tablet  supplies power to the connected USB device).  They normally can't do both at once.  If you are powering the tablet through the USB port, it may not recognize a USB device plugged into that same port.  I forget exactly how it works, but I believe the AC adapter shorts the two center USB pins together and this switches the tablet into charging mode. It may also be that your charging cord shorts the pins together. 

If you can't get it working any other way, look for something called an "OTG" cable.  These are intended to do what you are trying to do.... connect a thumb drive or other USB device to the micro USB port on a one port tablet.  You can get OTG cables on Ebay or maybe at your local computer store.  Google for "OTG CABLE" and you will get lots of hits. 

If you can boot using the OTG cable, make sure the battery is fully charged because you can't run off AC power while using the port for USB. 

ALL tablets should include TWO USB ports.  That would solve this issue.
