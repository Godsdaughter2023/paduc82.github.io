sorry for not making myself clear in the last post. the manage permission is used to control site or library sharing with other users. it isn’t used to control item sharing.

there are three permission levels in sharepoint online: site level, list/library level and item level. to set the individual permission for an item, we need to click the ellipsis next to the item and ellipsis> share> shared with> advanced and click stop inheriting permissions create the unique permission for this document.

if you want to control the sharing of the items in a library, you can create a permission level without the edit items permission in the list permission section as the following shows.

for example, if we don’t want user a to share the document, we can create a new permission without the edit items, and grant this user a unique permission in the item with this new permission. then if he shares this document, a request will be sent to the site collection administrator. only if the administrator approves this, can the document be shared with the other user.

however, this will also disable the edit online feature for this user, so this may not fit your requirement. the best practice is to create two libraries to store different documents for different purposes and set unique permission for them. meanwhile, you can submit feedback via the link in my previous post. many features came directly from your feedback. thanks for your understanding.

for “allow members to share the site and individual files and folders”, it will disable the sharing feature in the site level. everyone will request the approval from the administrator to share documents except the user who has the full control permission.
