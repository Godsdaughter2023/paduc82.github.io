You want a program that doesn't rely on any operating system to restore your backed up operating system and installed programs. Macrium Reflect is excellent, and free for personal use. You use the installed program to make the rescue media, which you boot with to restore your disc-image of the OS partition from the external drive. Note: The free version will only restore to approximately the same size partition as the backup was made from. 

I strongly recommend making your partition for OS and installed programs 100 to 125 GB, (MiniTool Partition Wizard, free for personal use, if needed) and then using all, or most of the rest of the drive to make a "Data" partition to hold everything else. I also use an online backup program (such as Dropbox)  to automatically backup selected data folders/files, to protect against hard drive failure, computer theft, and storm or fire loss.

Example: My accounting program is installed along with the other installed programs in the operating system partition. The data files it generates are in the data partition and backed up as described above. When I upgrade to a different operating system, the accounting program simply begins to be operated by the new OS, but the data files aren't moved or changed in any way, they continue being operated upon by the accounting program, which is operated by the new operating system.

When upgrading to a new build/OS, as soon as I get it cleaned up, (crapware removed) I make a new disc-image of the OS partition, and save it, along with the previous one, on the external drive.

Restoring in the event of hard drive failure usually takes about 10 minutes, once the new drive is ready.
