tygers37 say:

Creating opening screen/menu screen on the movie

I've made several DVD's from Movie maker and it always had a way to set up the opening page of the DVD where you can select play or scenes, etc. Now I can't find where to set that up. I'm using Movie Maker 2012.

 

Original title: how do you set up the opening screen of the movie (e.g. where you can select play or scenes, etc.)?

considerate_guy say:

This isn't a feature of Movie Maker.  The ability to create a video DVD is a feature of Windows DVD Maker which is available in Windows Vista and Windows 7.  If you are on those versions of Windows, when you save your movie out of Movie Maker, there is a choice to "Burn a DVD" and that will launch Windows DVD Maker. You can also launch Windows DVD Maker directly and give it one or more video files rather than launching it from Movie Maker.  That's especially helpful if you have made a separate movie for each "scene" in your movie.  I generally do this where I will make a separate project in Movie Maker for each logical "scene" and save each movie separately on my computer.  In the end I will launch Windows DVD Maker and give it all the movie "scenes" I created previously.

 

If you are on Windows 8, Windows DVD Maker is no longer present and thus the "Burn a DVD" option won't be listed.  In this case, you'll have to search for another program for making video DVDs.  A few freeware options are Freemake Video Converter, DVD Flick and DVD Styler.  None of these are supported in these forums, so you'll have to check the web for tutorials. There are also paid options and you may want to see if your PC came loaded with any such software.

a. user say:

On Vista, the following procedure seems to work:

1. Go to control panel and open Administrative tools.
2. Click on services.
3. Stop these services:
    Windows Security Center
    Windows Managament Instrumentation
    SBSD Security Center Service (if you use Spybot Search & Destroy).
4. Let all them to Automatically start.
5. Remove the directory C:\Windows\System32\wbem\Repository
6. Go to Services and Continue all the Services above or go to next step.
7. Reboot

Or, create a .bat file with the following content:

Code:
echo off
cls
echo.
echo Stopping Windows Management Service - please wait ...
net stop winmgmt
echo.
echo.
echo Removing Repository folder - please wait ...
rd /S /Q %systemroot%\system32\wbem\Repository
echo.
echo.
echo Starting Windows Management Service - please wait ...
net start winmgmt
cls
echo.
echo Done!


Run it in Vista as Administrator (right click the file and chose to run as Administrator).

I hope this helps.

Good luck!
