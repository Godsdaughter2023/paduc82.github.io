The security certificate issue with Microsoft Edge - Microsoft Community

afuso392 say:

The security certificate issue with Microsoft Edge

I keep getting this message when using Microsoft Edge.  "The security certificate presented by this website is not secure.

Security certificate problems may indicate an attempt to fool you or intercept any data you send to the server.

We recommend that you close this webpage and do not continue to this website."   It's not confined to any one website and happens intermittently.  Would appreciate any help. 

a. user say:

That message results when you attempt to reach a secured website (https)  and the internet's security mechanism for verifying the identity of a website's owner (known as the security certificate) is unable to vouch for the owner's identity.

That could mean several things: Scariest of all, it could mean that the website is now controlled by a hacker who will try to attack you if you go to that website. It could also mean that the website's legitimate owner forgot the renew their website's certificate. (It's typically valid for a year.) It could also mean that the browser needs to receive an updated list of root certificate authorities (that provide information on the validity of security certificates.)

There's not much you can do about this, aside from running Windows Update (to check if there is an updated list of root CA's) or possibly visiting the website with a different web browser.

This article explains the issue in more depth. It's written in the context of Internet Explorer but applies equally to Microsoft Edge (and all web browsers.)
