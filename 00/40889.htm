Thank you for posting here on the Xbox forums. Try this out :

1. Open run window (Windowskey + R)

2. Type %appdata%

jesse kaplan [msft] say:

Hi ThorpGx,

We're sorry you're hitting this problem, we're still in our beta period and are working on addressing the bugs found by the community. We're investigating a fix for the 0x80070780 error and hope to have it available soon. 

In the meantime, there are a few options available for you to fix this. Depending on your machine state, some of these may not work, but if they work they may be easier and so I wanted to give you the options. 

1.  If the game is still showing in the Xbox (Beta) app's install queue, you may be able to delete the bits by clicking on the "x" button under the item. 

2. If the item is gone from your install queue, you can try installing the game from scratch. If it succeeds it will clean up the bits from previous attempts. 

3. If you just want to get rid of the bits, you'll need to take over ownership of the folder from "system".

Here are the steps for this from an admin account:

To take ownership of a file or folder

  1. Open Windows Explorer, and then locate the file or folder you want to take ownership of.

  2. Right-click the file or folder, click Properties, and then click the Security tab.

  3. Click Advanced at the bottom

  4. The second line of text will be the "Owner" line and is probably showing "System". Click the "Change" link next to it 

  5. Type in your user name or "administrators" into the "Enter the object name to select" text field and then click "OK"

Once you do this you should have access to this folder and be able to delete its contents. You may have to accept an admin elevation prompt and then click "try again" when it tells you which user needs to be logged in. 

Please let us know if you have any questions. 

Thanks,

Jesse
