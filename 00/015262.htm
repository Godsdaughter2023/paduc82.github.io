Get hold of the free PrimoPDF converter and use it to create a ..pdf file of your document.  Or if you want to create a .jpg file directly, get hold of Snagit fromwww.techsmith.com with which, after you print the document using the Snagit Printer, can save it as a jpg file.
 
However, as a jpg file can only contain a single page, and your document does not contain headers of footers, you could select the text and copy it to the clipboard and then use Paste Special to paste it into a document as a Picture and then save that document in HTML format and the picture that is in it will be saved in a separate folder in the directory in which you save the document with the name of the folder being the same as that of the document.

phillip m jones say:

Microsoft use an Apple Supplied Converter to Transform .xlsx to .pdf. It never has worked correctly even though it was supplied to them by Adobe. 

If you own Acrobat or Subscribe to most recent version a better way is as Follows:

  1. Create your Document and save in xlsx format as Backup.
  2. Next Go to File Menu.
  3. Then click on PDF button and hold down.
  4. A Drop down Menu will appear.
  5. Choose from list: Adobe PDF.
  6. The first item appears looks similar to Job Options in Adobe Distiller Just choose the default setting and choose not to open Acrobat or Reader.
  7. Next either accept title Given I usually Trim so it shows only Filename.pdf
    instead of (Microsoft Word: File Name.xlsx.pdf)
  8. Then Browser to desired location Save and click Save. If setup correctly should look like original file.
