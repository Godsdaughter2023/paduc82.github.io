brennabray say:

Pagination

I'm working on a document that starts on pg 194. In the middle of the document, the pagination suddenly switches from pg 256 to pg 195. Also, the "formatting" goes from saying "section 1" in the footer to "section 4" in the footer. I've tried "removing all headers/footers;" that doesn't do anything. Any suggestions?

guessed say:

You must have section breaks in the document. If there isn't another reason for having them there, delete the section breaks.

Note that if you are tracking revisions, deleted section breaks are 'still in the document' and you will need to accept those deletions.

suzanne s. barnhill say:
If you haven't already done so, start by adding "Formatted Page Number" and "Section" to the status bar (right-click on the status bar and check those items). Those will help you keep track of where you are in the document. As you scroll through, when you see that the Formatted Page Number diverges from the Page X of Y number, click in the document and use Insert | Header & Footer | Page Numbers | Format Page Number to open the Page Number Format dialog, where you can change the "Start at" setting to "Continue from previous section."

stefan blom say:

Section breaks will be added by Word if you change section-level formatting to a selected range in the document, for example, if you apply newspaper columns to part of the text.

The quick way to make all of your numbers continuous would be to run a macro such as the following:

Sub SetContinuousNumberingInDoc()
'Macro created by Stefan Blom, MVP, February 2019
On Error GoTo errhandler
Dim s As Section

For Each s In ActiveDocument.Sections
s.Headers(wdHeaderFooterPrimary) _
.PageNumbers.RestartNumberingAtSection = False
Next s
errhandler:
Exit Sub
End Sub

For installation instructions, see https://www.gmayor.com/installing_macro.htm.
