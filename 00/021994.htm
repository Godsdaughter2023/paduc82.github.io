hansv mvp say:

Use the Value property of the text box, or just the name of the text box:

 

deployTasking = Me.Tasking.Value

 

or

 

deployTasking = Me.Tasking

 

Value is the "stored" content of the text box, and Text is the content of the text box as it is being edited by the user. The latter is only available when the focus is on the text box, Value is always available. Since Value is the default property of a text box, it is assumed if you omit it.
