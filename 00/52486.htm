This is normal behavior for the Bubbles screensaver. What you see in the background when the Bubbles screensaver activates is just the background that was visible when the screensaver activated.

The 3D Text SS will always display a black background.

Back to your problem with the SS not activating at the set time. There can be several causes for this behavior. When you set the time for the SS to activate, this means that the system must be completelty idle for that length of time, before the SS will activate.

One of the most common causes is many people subscribe to a VOIP service. This allows them to make telephone calls directly using their internet connection, instead of a Cell phone or standard, Wired telephone line.

The software or hardware device that enables this will constantly monitor the system for a telephone call event. This generates enough activity so that the system never reaches a true 'idle' state, and subsequently, the screensaver will never activate.

andre da costa say:

When I upgrade a preinstalled (OEM) or retail version of Windows 7 or Windows 8/8.1 license to Windows 10, does that license remain OEM or become a retail license?

If you upgrade from a OEM or retail version of Windows 7 or Windows 8/8.1 to the free Windows 10 upgrade this summer, the license is consumed into it. Because the free upgrade is derived from the base qualifying license, Windows 10 will carry that licensing too.

If you upgrade from a retail version, it carries the rights of a retail version.

If you upgrade from a OEM version, it carries the rights of a OEM version.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM :

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

What happens if I change my motherboard?

This will invalidate the Windows 10 upgrade license because it will no longer have a previous based qualifying license which is required for the free upgrade. You will then have to purchase a full retail Windows 10 license.

How much will Windows 10 cost if I need to perform a new installation on a PC without a qualifying license?

Windows 10 will be free for the first year to all Windows 7, Windows 8.1 and Phone 8.1 users.

a. user say:

I find the the Black Viper site helpful and you can find it here:

http://www.blackviper.com/service-configurations/black-vipers-windows-xp-x86-32-bit-service-pack-3-service-configurations/

Each listed Service also has a clickable link that will give you more information about the Service so you can decide if you really need it or not.  Not all the XP Services are listed there (a few are missing), but it is a good start.

In the Services applet, click the Status and Startup Type column headings to sort the Services by that column and that will help you keep things straight visually.

It is important to be careful with Services and I would not disable or change the Startup Type for too many at a time or you could end up with a no boot scenario.  I would do one or two, reboot and be sure things are okay, then look at some others. 

I would also keep a list of the ones you tinker with as you go so if something goes wrong, you will know how to undo it.  All systems are a little different, so just because I have a Service disabled on my system doesn't necessarily mean you need to disable it on your system.

If you are not sure about a particular Service(s), you should post them back in your next reply and somebody should be able to help you decide if you need it/them or not.

For example, a popular misconception (especially from the Support Engineer "experts" engaged by Microsoft) is that the Help and Support Service needs to be set to Automatic start and that is not true.  I set mine to Manual start and then if (heaven forbid) I ever use Help and Support, the Service will start when it needs to start and not be sitting there running all day and night with nothing to do.
