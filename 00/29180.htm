rajesh govind say:

Hi G.Alvaro,

File History is a backup application that continuously protects your personal files stored in the Libraries, Desktop, Favorites, and Contacts folders on the computer. So, to take the periodic backup of the shared folder on your computer, you should first make sure that the location of this shared folder is added to the library.

The File History periodically (by default every hour) scans the file system for changes and copies the changed files to another location. The frequency of the scan for the changed files on the computer can be decreased from 1 hour (by default) to 10 minutes.

In your network setup, whenever the server is turned ON and logged into and when the other users on the network changes a file (which happens only when the server is turned ON), the file history feature will be able to detect this change in the scan which happens within the next 10 minutes and takes the backup of the changed file.

This approach eliminates the need to set up a backup schedule and leave the computer idle and turned ON for an extended period of time.

cbarnhorst say:

No.  In order to format the system drive you must start the computer with the dvd.  The dvd then loads a special version of Windows called WinPE (Windows Pre-Installation Environment) which runs in memory off of files written to a kind of ramdisk.  That means that the system disk is not in use at that point and can be formatted.  A system disk cannot be formatted while Windows is using it (kinda like committing digital suicide).

The problem is that you cannot run Windows 7 Setup.exe that way with an upgrade product key.  Only with a "full" product key.  This ensures that a customer cannot use the lower priced Upgrade sku to install Windows on a computer without an operating system.  "Upgrade" means....well, upgrade.  If you start the computer with the dvd in order to get access to the drive tools, once you enter the upgrade product key Setup will disable the Custom Installation option and tell you to restart the computer and install Windows from the desktop of the old OS.

Further, if you do format the system drive and then start the computer with the dvd as soon as you enter the upgrade product key Setup will tell you that you cannot use that product key to install Windows on a computer without an operating system.

Of course a Window 7 dvd will allow you to defer entry of the product key so you could still start the computer with the dvd, skip the product key step and then the Drive Tools would be available.  The problem with that is that Windows 7 will not later accept the upgrade product key for activation.  It will require a "full" product key.

What you can do is complete the upgrade to Windows 10 and once activated you can use the Windows 10 media to perform a clean install and reconfigure your drive then.  Windows 10 clean installations are allowed without needing a product key (the page will appear for your convenience only and you simply skip it and move on).  Once you have done that first upgrade the computer is permanently registered with Microsoft through something called  digital entitlement determined during the first upgrade for the edition of Windows 10 installed.  The first time the computer connects to the internet the new installation is activated.
