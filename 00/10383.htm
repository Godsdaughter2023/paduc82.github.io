Hi Mick:

Most of what you need is in this article:
http://wordfaqs.mvps.org/headerFooter.htm

1)  You need a section break at the end of each section where you want page number formatting to change.

2)  To have different headers and footers in a section, you must click in the headers and footers of that section and disable the "Same as Previous" setting.  Remember that there are three independent headers and three independent footers in each section.

3)  You can format a book with only two section breaks, provided you use the "StyleRef" field to call in the text of your headings.  See http://support.microsoft.com/kb/141994

4)  To use the StyleRef field, you must use the same style to format each chapter heading.

5)  For non-TOC headings, Use Heading 5 and below for their formatting.

6)  You can edit the Outline Numbering format for the Heading 5 style to remove the word "Chapter" and the references to previous level numbers so you get the appearance you want.

abi99 (2) say:

Your network operator does not care and accepts all APNs, therefore you cannot create a fake access point nowadays anymore. With a ‘standard’ Nokia C2-01 you are able to delete all configurations:

Nokia » Menu » Settings » Configuration » Delete or Default » Options » Delete

and you are offline.

 

If you are not able to delete all configurations, please, give use the code of your firmware variant which is determined by the Nokia Product Code – that seven character code printed on the label of the phone (below the battery).

 

If you change the SIM, the configuration is reloaded from the internal memory. Therefore, you have to check Nokia » Menu » Settings » Configuration » Delete or Default every time you swap the SIM.

 

If your phone goes online without touching it, you should not disable Internet because an app is still going to try to go online. If you disable Internet in that case, your battery time decreases dramatically because that app tries to go online all the time. You have to find the culprit app. For example, Nokia » Menu » Mail » Options » Settings » your account » Update my Inbox » Manually.

 

Another option is to call your network operator and ask him to disable Internet connectivity (which kills MMS as well).

bruce hagen ~ mvp say:

Whichever shortcut you are using to launch the window, right click it and select properties. Under the 'Run' drop-down, choose "Maximized".

If the window still opens small, try this next:

Grab the corners of the window and drag it to full screen size. Do not use Maximize. Close all other windows first and this one last. Windows will remember the *last* screen size used the next time you open the program.
