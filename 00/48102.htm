    Checking for and Installing Updates - Microsoft Community

    hosch250 say:

    Checking for and Installing Updates

    Technical Level : Basic

    Summary
    Describes searching for and installing updates in Windows 8. Describes some of the problems people have when updating Windows. Basic skills required.
    Details

    Computers that are not updated frequently are not protected against the latest malware.  Microsoft releases major security updates on the second Tuesday of every month, so that is as often as most Windows users need to check for updates, but updates are often released daily for Microsoft's own anti-virus program, Windows Defender, so users of this anti-virus should check for updates daily.  This article will explain how to check for and install updates in Windows 8.

    There are two ways to check for updates - through the Control Panel and through the PC Settings app, which is opened through the Settings Charm.  To check for updates through the Control Panel, open the Control Panel and navigate through the System and Security heading into the Windows Update section.  Click the "Check for Updates" link in the upper left corner of the window and wait for it to check if any updates are available:

    If any updates are available, they should be installed immediately, unless they are Windows features that you do not want, such as Bing Desktop.  You can view the available updates and choose which ones to install by clicking "See available updates":

    I will not explain how to use the PC Settings to check for updates here because it does not have the full features of the Control Panel, such as being able to hide or not install an update.  However, you can read about this feature on my blog:  Updating Windows 8.

    I recommend that you set Windows Update to automatically check for updates, notify you to install them, and not let it install updates automatically.  Many people have problems with the automatic update feature because the updates are corrupted do to the computer shutting down.  Other problems include the computer rebooting during the installation and not saving the current work.

    When an update is not installed, it is often because a third-party anti-virus is blocking some of the files that need to be modified - Norton, McAfee, AVG, and Kaspersky are notorious for this.  The anti-virus product usually needs to be completely uninstalled to fix this problem.  I would recommend switching to Windows Defender while you fix the problem so your computer is fully protected.

    As always, feel free to contact me through the comments section of this article if you need any help.

    I would suggest you to perform Startup Repair and check if the issue is resolved.

    Startup Repair is a Windows recovery tool that can fix certain system problems that might prevent Windows from starting. Startup Repair scans your computer for the problem and then tries to fix it so your computer can start correctly.

    Startup Repair is one of the recovery tools in the System Recovery Options menu. This set of tools is located on your computer's hard disk and on the Windows installation disc.

    1.      Remove all floppy disks, CDs, and DVDs from your computer, and then restart your computer using the computer's power button.

    2.      Do one of the following:

    o    If your computer has a single operating system installed, press and hold the F8 key as your computer restarts. You need to press F8 before the Windows logo appears. If the Windows logo appears, you need to try again by waiting until the Windows logon prompt appears, and then shutting down and restarting your computer.

    o    If your computer has more than one operating system, use the arrow keys to highlight the operating system you want to repair, and then press and hold F8.

    3.      On the Advanced Boot Options screen, use the arrow keys to highlight Repair your computer, and then press Enter. (If Repair your computer isn't listed as an option, then your computer doesn't include preinstalled recovery options, or your network administrator has turned them off.)

    4.      Select a keyboard layout, and then click Next.

    5.      On the System Recovery Options menu, click a tool to open it.
