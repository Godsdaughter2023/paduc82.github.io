Hi Bartosz,

Welcome to the Microsoft Communities and thanks for posting the question.

Outlook for Windows has profiles that you can set up in such a way as to prompt you to open a particular one.

In Outlook for Mac the same is called an identity.

You cannot select switch identities in Outlook 2011, as the feature is not included in Outlook 2011.

As a workaround you could hold down the Option key when launching Outlook 2011. This brings up the Database Utility. Select + "Create a new identity", then "Set as Default".

You would have to do it every time you want to switch identities in Outlook 2011.
