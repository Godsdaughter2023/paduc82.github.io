marrett say:

prevent Excel from automatically converting text to date

Moreover, even the search field in the "AutoFilter - custom" section automatically converts "DEC1" to "dec-1" and I can't even filter the list according to the desired criteria (this is when I try to filter a different column where the abbreviation appears together with other text and is not conveted to a date).

Is there a way to stop this nonsensical auto-conversion?

pete_uk say:

There are a few things you could do to try to get the data back as it should be. If your column is now only a mixture of the text that you want and dates (that have been converted), then you can use a formula like this in a helper column:

=IF(ISNUMBER(A1),TEXT(A1,"mmmd"),A1)

assuming the offending column is A, and then you can copy this down as far as required. This will only give you a 3-letter abbreviation, so you will get SEP4 and MAR8. If you are happy with what is in the helper column, then you can fix the values and copy them over to replace what is in column A, and then you can delete the helper column.

An alternative is to use the other column that you mention where you have other text, and you can use LEFT or RIGHT or MID in conjunction with SEARCH to extract the text that you require - you will need to post some examples of values in the column if you would like a formula that you can use.

Others have mentioned methods of trying to prevent this from happening, but from what you say you already have the problem, so these 2 approaches should enable you to retrieve the situation.

Hope this helps.
