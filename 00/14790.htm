There is a better way than the method suggested by Arshad.

from the original workbook as files, and then import the code into the other workbook.

  1. Open the workbook that contains the macro module that you want to move (source workbook).
  2. Open the workbook to which you want to move the code (destination workbook), or open a new workbook (destination workbook).
  3. In the VB Editor of the source workbook, in the Project window, select the module or userform you wish to move.
  4. In the VB Editor of the source workbook, from the File menu choose Export File... and then name and save the .bas file.
  5. Switch to the destination workbook. 
  6. In the VB Editor of the destination workbook, on the File menu, choose Import File... and the open the saved .bas file you exported from the source workbook.
  7. Your module or userform will now be in the destination workbook.

Microsoft Word 2010 default settings - Microsoft Community

gabriel zamfirescu say:

Microsoft Word 2010 default settings

Hello,

I would like to know if there is any way to pin down all the current settings as default settings (font, size, page layout settings etc.).

Thank you.

suzanne s. barnhill say:

Ideally, you would want to save most of those settings to the Normal template (see http://word.mvps.org/FAQs/Customization/CustomizeNormalTemplate.htm). In Word 2010, you can set font and font size and paragraph formatting in the document defaults (click the Manage Styles button in the Styles pane and select the Set Defaults tab of the Manage Styles dialog) and choose the radio button for "New documents based on this template."

You can also select font and paragraph settings in the Font and Paragraph dialogs and click Set As Default, then select "New documents based on the Normal.dotm template." Similarly, in the Page Setup dialog, you can set paper size, margins, and other page layout settings and click Set As Default and answer Yes to the question about setting defaults for all new documents.

Alternatively, if you don't want these settings for all documents, and in particular if your setup includes a header or footer or other content, you can set up a document the way you want it and save it as a template. Save it in the default templates folder, and it will be available as a basis for new documents.
