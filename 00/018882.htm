dakialvm say:
 I have read others complaint for Offcie / Powerpoint 2007. No solution published, adding charts shoilod be the most basic working functionality. Please advise
This is basically a "default message" from Office for when it can't generate the graph for some reason (usually Excel is blocking the opening of a spreadsheet for that graph). It might be what ZachGardner said but it also might be wrong (too restrictive) configuration in Excel Trust Settings. To check it out go to: Excel > File > Options > Trust Center > Trust Center Settings > File Block Options.
At this part you will see a lot of Excel Filetypes and every box with a tick (either on open or save) means that Excel will block the opening or saving of that filetype. You might have too many ticks in the wrong filetypes (the ones that powerpoint tries to open to generate a new graph) so my tip is to uncheck one by one (or all of them if you don't care about these file blocks) and check again in Powerpoint. You might have to close and open both Excel and Powerpoint (maybe a reboot also).
