"You may have attempted to use an undelared parameter or a field that is not bound to a control on the form or report"

 

Please help

albert d. kallal say:

You cannot use forms! expressions in a saved query.

 

The reason of course is that the query is saved server side, but the form is running as a web browser. So a query cannot "reach out" and grab information running on your web browser.

 

On the other hand if a server could "reach out" and grab form values from your web browser, I suspect that many people would be writing a query to look for and grab expressions from your web  browser that is doing your on line banking!

 

So a saved query simply cannot "reach out" and grab values from a form.

 

However, you can embed the sql in the form and in this case then you use a forms reference in the query.

 

Perhaps it best to outline what you trying to do in place of having a saved query "reach out" and grab values from a form which is not possible.

 

Best regards,

I missed this, but yes – you are correct.

slance310 say:

Hi,

I had a driver problem with PerfectDisk 14... After many different attempts by Raxco to resolve the problem, they found a fix... Something concerning a RegEdit and Legacy IO..

I have since read a number of threads that this same thing is causing problems in other areas...

Here is what I was instructed to do, and the modifications to the instructions I had to make...

Please launch regedit and browse to:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\SessionManager\I/O System

and change the IoBlockLegacyFsFilters value to 0. Reboot the system and then try using PerfectDisk.

NOTE... IoBlockLegacyFsFilters did not exist... Created manually and restarted... PD14 now working...

To add the registry entry manually, right click on the I/O System and select "New"..."Dword (32-bit) Value"...

Enter or Copy\Paste... IoBlockLegacyFsFilters as if simply renaming a file...

It should already have a value of "0", but if not, right click on it and select "Modify"...

If this doesn't help, simply delete the IoBlockLegacyFsFilters and restart...

You can try the following procedures below for the PC didn't show the updates in OneDrive app :

Log into OneDrive on your computer and run the following commands to update OneDrive:

Press the Windows key + R

In the Run window, enter:
%localappdata%\Microsoft\OneDrive\onedrive.exe /reset

Wait 2 minutes and then execute the following command:

%localappdata%\Microsoft\OneDrive\update

Click OK.

Make sure the OneDrive icon in the notification area of the taskbar disappears and then re-appears after a minute or two.

If the OneDrive icon doesn’t re-appear after a few minutes, open the Run window again and enter:

%localappdata%\Microsoft\OneDrive\onedrive.exe
