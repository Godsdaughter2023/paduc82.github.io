no buddy it´s not about hardware bug or software bug u have to understand the difference between GPS & A-GPS. A-GPS, is a carrier network dependent system which can, under certain conditions, improve the startup performance of a GPS satellite-based positioning system. It is used extensively with GPS-capable celluler phones.

 

Conventional or "Standalone GPS" operation uses radio signals from satellites. In very poor signal conditions, for example in a city, these signals may suffer multipath where signals bounce confusingly off buildings, or be weakened by passing through walls or tree cover. When first turned on in these conditions, some non-assisted GPS navigation device may not be able to work out a position due to the fragmentary signal, rendering them unable to function until a clear signal can be received continuously for up to 40 seconds (the time needed to download the GPS ephemeris.) Some newer receivers are better at handling these situations.

An A-GPS system can address these problems in several ways, using an assistance server or other data from a network.

IMAP and POP are not profile types. They're account types. You create new profiles using the Mail applet in Control Panel by clicking Show Profiles, then Add.

Your old data is very likely still on your PC. It's in a PST located either in Documents\Outlook Files or in %LocalAppData%\Microsoft\Outlook. Sub RunMacroOnAllFilesInFolderAndSubFolders()
    Dim flpath As String
    Dim macroname As String
    flpath = GetFolder
    If flpath = "" Then Exit Sub
    macroname = InputBox("Enter the name of the macro to run")
    If macroname = "" Then Exit Sub
    Application.ScreenUpdating = False
    Call ProcessFolder(flpath, macroname)
    Application.ScreenUpdating = True
End Sub
 

    Dim fso As Object
    Dim fld As Object
    Dim fl As String
    Dim sfl As Object
    Set fso = CreateObject(Class:="Scripting.FileSystemObject")
    Set fld = fso.GetFolder(p)
    If Right(p, 1) <> "\" Then p = p & "\"
    ' Process files
    fl = Dir(p & "*.docx")
    Do While fl <> ""
        Call ProcessFile(p & fl, m)
        fl = Dir
    Loop
    ' Process subfolders
    For Each sfl In fld.SubFolders
        Call ProcessFolder(sfl.Path, m)
    Next sfl
    Set sfl = Nothing
    Set fld = Nothing
    Set fso = Nothing
End Sub
 

    Dim doc As Document
    Set doc = Documents.Open(f)
    Application.Run m
    doc.Close SaveChanges:=True
    Set doc = Nothing
End Sub
 

    With Application.FileDialog(4) ' msoFileDialogFolderPicker
        If .Show Then
            GetFolder = .SelectedItems(1)
        End If
    End With
End Function

woodg say:

In the To box you should just need to type "Favorites" and you should get a contact for your favourites in your contact list. Click on this contact and you should end up with an entry called "Favorites" in the To box. Clicking on send will send this email to everyone in your favourites group. If you look at the Favorites entry in the To box you will see a small plus sign on the left hand side. Clicking on this will replace the Favorites entry with an individual entry for each of your contacts in Favorites. This allows you to remove some contacts just for this email.

 

All this works for CC and BCC, too.

 

If you want to see who is in Favorites then you can go to the People page by clicking on the down arrow next to Outlook at the top left of the email page and selecting People. On the People page you will get a list of your contacts. At the top of the list is a star icon. Your favorites are shown under this star.
