john korchok say:

Pasted text will keep the theme color position that it was set to in the original file. So, as an example, if the text was set to Background 1 in the original file's color theme, and Background 1 was set to black, the text will appear as black in the original. But most presentations have Background 1 set to white, so when text set to Background 1 is pasted, it assumes the color in Background 1 in your deck and becomes white.

You don't really want to change your theme to goofy values just to accommodate pasted text. The simplest alternative is to use Home>Paste>Paste Special and paste as Unformatted Text. This is made simpler if you add this command to the Quick Access Toolbar:

  1. Choose File>Options>Quick Access Toolbar.
  2. Change the Choose command from dropdown to All Commands.
  3. Scroll down to Paste and Keep Text Only and select it.
  4. Click on the Add>> button. The command is added to the right-hand list.
  5. OK out. The command appears on the Quick Access Tool bar at the right end.

This button has a keyboard shortcut, determined by its position in the QAT row. If it's the 5th icon in the row, its keyboard shortcut will be Alt + 5.

  • Do you want to continue as a Windows Insider and keep getting preview builds after 7/29?
