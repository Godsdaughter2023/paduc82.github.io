b_a_m say:

After lots of research I was able to piece together a solution on my own.

Solution: 

To fix audio CD media:

open regedit and navigate to HKEY_CLASSES_ROOT\AudioCD\shell\play\command and modify the Default value to C:\Program Files\Windows Media Player\wmplayer.exe /device:AudioCD "%L"

To fix DVD media:

open regedit and navigate to HKEY_CLASSES_ROOT\DVD\shell\play\command and modify the Default value to C:\Windows\ehome\ehshell.exe /directmedia:discplayback

These two registry entries will modify the default double-click and "Play" context menu actions for DVD and Audio CDs when being viewed from File Explorer.  The command line switches are specific to my selected default programs (windows media player for audio discs and windows media center for DVDs)

frederik long say: It would help if you could use snippingtool.exe to take a snapshot of the error message, then include the snapshot with your reply.

davem121 say:

This is caused by an entry in your Startup processes that needs to be removed:

  1. Press Ctrl + Shift + Esc keys together to open Task Manager
  2. On the Startup Tab look for anything relating to this file
  3. If you find something related, select it and click 'Disable'

If you did not find it there:

  1. Press Windows + R keys together to open the Run Dialog
  2. Type msconfig in the run dialog and hit enter
  3. Click on the Services Tab of the resulting dialog
  4. Check the box marked 'Hide all Microsoft Services'
  5. Look for anything relating to this file
  6. If you find it, uncheck it
  7. Click Apply and Reboot
