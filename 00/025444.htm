Windows mobile 7 and activesync - Microsoft Community

rkling say:

Windows mobile 7 and activesync

I am a lawyer with excell spreadsheet files on my Windows mobile 6.5 phone which automatically sync (along with calander, contacts) thru activesync. Will the windows 7 phone similarly sync not only calander and contacts but files as well?
julie fka profjulie say:

The short answer to your question is "No, nothing will sync through ActiveSync in a similar fashion as they do on your windows mobile 6.5 phone."

First of all, there is no such thing as windows mobile 7.  The newest phone is a windows phone 7 device, this is a completely new operating system for mobile devices and it does not share the same feature set and functionality as the windows mobile OS. 

  • Syncing data to the phone is completely different than that of a windows mobile device.  Calendar and Contacts are synced to cloud-based services, such as Windows Live, Google, or Exchange server.  There is no way to sync this stuff via ActiveSync/WMDC to a local computer.
  • Syncing Office files, such as Word, Excel, and PowerPoint is handled exclusively through a Sharepoint 2010 Server (again, a cloud-based service).  If you do not have access to a Sharepoint 2010 Server, you will not be able to sync these files to your phone.  I don't have access to a Sharepoint 2010 Server, so I've figured out a workaround to getting these files on my phone that works for me, but many people are put off with this inability to sync to a local computer.
  • OneNote notebooks can be synced to a windows phone directly from from Windows Live SkyDrive (again, a cloud-based service), which is freely available to anyone who has a windows live account.  If you also have OneNote 2010 installed on your computer, you can sync your Notebooks from your desktop computer to SkyDrive and then to your Windows Phone.  This is actually a very nice way to work with files and syncing is really easy and fast.  You can drag Word, Excel and PowerPoint files into a notebook and sync that to SkyDrive and then to a windows phone and the documents are available for use on the phone
  • Pictures, music and videos sync via wired connection to your computer through the Zune software
  • ActiveSync/WMDC is not used at all by windows phone 7

Probably the most disorienting thing with windows phone for me was the inability to move files easily between my computer and my phone.  There is also no File Manager on the phone, so there's no way to move files or organize files that are stored on the phone.  All Office-related files are stored in the Office Hub, which you could think of as a single file box.  They are stored in the box in the order in which they are placed in the box, so there's no way to sort by file type, no way to sort files alphabetically, etc.  If you are accustomed to working with a lot of files on your mobile device, the storage system will make it extremely difficult to find your files. 

To overcome these issue, I've figured out my own workarounds which involve storing the files I plan to use on my phone on a secure cloud-based website (which is NOT Windows Live SkyDrive), which offers a directory and folders file system, and that also syncs to my local computer.  When I need a file, I download it from the cloud-based website by browsing to it from my phone.

I think the way files get moved to a windows phone and the limitations in the way files are managed on the phone once they are there is is the greatest deficiency of windows phone 7 and I hope to see some major improvements in future OS updates.
