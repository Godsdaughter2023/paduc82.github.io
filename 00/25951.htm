suzanne s. barnhill say:

Word gets the list of "built-in" paper sizes from the printer driver (sizes natively supported by the printer) and also allows you to define "Custom" paper sizes. Some printers don't handle these custom sizes well, but some printers do allow you to define new sizes in the printer driver, so that they appear in the size list in Word. If your printer doesn't allow you to define new custom sizes and doesn't handle Word's custom size specifications as expected, then your only recourse is to set the document up as if it were Letter and adjust margins accordingly. Yes, it's a pain, but it's sometimes the only way.

Rereading your question, I'm wondering if you've actually selected the 4x6 size (if offered by the printer) or set it as a custom size in Word (on the Paper tab of the Page Setup dialog). Just changing the printer's physical arrangements has no effect on Word.

Hi, GaryWearne. If this is for a Windows Phone, can you check if any of the workarounds given below will help?

Restart your phone with SIM card inserted, and retry sending a multimedia message (your phone loads the settings from the SIM card).

  • Tip! Send a multimedia message to the own mobile phone number. This will test sending and receiving capabilities in one go.
  • Important! MMS requires data connectivity, specifically a mobile (cellular) data.
  • Note! If you have Multi-SIM (two SIM cards with the same mobile phone number), note that MMS can only be received on your assigned “primary” SIM card.

If you can connect to the internet but you cannot send the multimedia messages, try entering the new settings manually  

  1. Try searching online to find the multimedia (MMS) settings for your mobile operator. To add your settings manually, select one of the options depending on your phone model:
  • Option 1 - Go to Settings > access point, and tap +.
  • Option 2 - Go to Settings > mobile+SIM > SIM settings > SIM, and tap add MMS APN.

If you cannot connect to the internet using a mobile data connection

  1. Enable the mobile data connection on your phone. Go to Settings > mobile+SIM (cellular) and switch Data connection to On.
  2. Check that your data roaming is activated when you are not in your home network. Go to Settings > mobile+SIM (cellular) and select data roaming options.
  3. Make sure you are not reached your data limit. From the Start screen, swipe left to the App list and start Data Sense app. Switch Restrict background data usage and/or Restrict background data usage when roaming to Off.
  4. Confirm with your mobile operator that you have an active mobile data subscription (access to internet) on your SIM card.
