Can I set all my Outlook Calendar entries to private? - Microsoft Community

nickrm say:

Can I set all my Outlook Calendar entries to private?

Is there a way to automatically set *all* my Outlook Calendar entries to "private" rather than go into each entry one-by-one and select "private"?
talking moose say:
This is possible with an AppleScript.

Launch the AppleScript Editor application found in /Applications/Utilities and paste the following into a new window:

tell application "Microsoft Outlook"

set selectedItems to selection

repeat with anItem in selectedItems

if class of anItem is calendar event then

set is private of anItem to true

end if

end repeat

end tell


Save this as a script called something like "Mark as Private.scpt" to your Desktop or a location where you can find it.


In Outlook, select About This Menu... from the Script menu just to the right of the Help menu. Click the Open Folder button. Drag your newly saved script into this folder. This will add the script to your script menu where you can select it whenever you need it.


To use the script, change to your Calendar and select Edit menu --> Find --> Advanced Find. Change the drop down menu from "Item Contains" to "Folder". Set the other drop down menus that appear to "Is Calendar". This should display all your calendar events as a list.


Select Edit menu --> Select All. Then select the script from the Script menu. Depending on how many events you have this may take several seconds to a couple of minutes.


Unfortunately, you cannot automatically mark new events as Private. You must mark each as you receive or create events.


Hope this helps!

Suppose, for example, that a user password-protects a critical document and then leaves the company.
