gaurav prakash say:

Hi peacebigjonny,

Welcome to Microsoft Answers community Forum!

There is Windows Calender that can be used with Windows Mail.

To access Windows Calender from Windows Mail follow these steps.

1. Open Windows Mail.

2. Click Tools.

3. Click on Windows Calendar.

To create a task follow these steps:

1. On the toolbar, click New Task.

2. In the New Task box, type a description of the task.

3. In the Calendar list on the Details pane, click the calendar where you want the task to appear.

4. In the Priority list, click the priority you want: low, medium, or high.

5. Enter the start and due dates.

6. To set a reminder, in the Reminder list, click On date, and enter the date and time you want to be reminded.

For further information, visit the below mentioned link.

http://windows.microsoft.com/en-US/windows-vista/Customize-Windows-Calendar

Hi JoshuFlood,

When Skype for Business and Outlook connect to server, they will access your network settings, such as required ports, protocols and certificates sometimes. If the server connects only with other clients and servers on a private network, the CA can be an enterprise CA.
