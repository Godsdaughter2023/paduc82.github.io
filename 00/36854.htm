¡firedog say:

This curious display of sent messages sometimes occurs when the messages concerned have taken a trip through another folder, which can happen if there is a database glitch that is eventually resolved. It's also common in messages imported or migrated from another program like Outlook Express.

You can restore the proper view of such messages (the ones that show "Name" <address>) as follows:

  1. Reduce the size of the WLMail window so you can see an empty part of the desktop.
  2. Right-click on this empty part and select New > Folder. Ensure that you can see the folder when the WLMail window is visible.
  3. Select the wrongly-displayed messages, drag them across and drop them on to the new folder.
  4. Open the folder, select all its contents (Ctrl-A) and drag them back where they came from.
  5. If they now display properly, delete the ones that don't. You can also delete the new folder on the desktop.

Noel

Click once on the file name in the new folder to edit the file name and remove the underscore ('_') character, then press Enter. Then try dragging again.

deepspacemillar say:

USB audio device stops playing audio

Hi all,

I have an external amplifier for my speakers/headphones ( a NAD D3020 ) and after upgrading my Win 7 system to Win 10 recently, I have started seeing an issue where after a certain amount of time, the audio from the amplifier simply stops working over USB.

Essentially, from a clean start. if I'm playing a game, after about 10-15 minutes the audio will suddenly mute.
If it's video, it takes about an hour before this happens (sometimes much sooner)
If it's music, it's rare.

Once the audio has stopped, there are some interesting notes; 
* muting/unmuting or changing volume does nothing
* absolutely no app in Windows or even Windows itself is able to generate sounds
* opening a video in Films & TV app for instance, the video plays at 1FPS, no audio. Other video apps act strangely too
* Windows Media Player when asked to play audio will attempt to load the file/start playing, and never actually start
* Windows still sees the amp as connected and working, despite the issues mentioned
* the NAD amp remains responsive to other input sources (only USB is muted)

Cycling the power on the amp itself, or disconnecting and reconnecting the USB cable will temporarily solve the issue, but the issue will re-occur again within 2-3 minutes after the initial occurrence.

After speaking with the amp manufacturer, I have cleaned all associated drivers, performed a firmware upgrade on the amp, tried different USB ports/hubs (USB 2 and 3), tried the proprietary USB driver and the generic Windows USB audio driver, updated Windows 10 to the latest build, performed a factory reset, disabled enhancements for the USB device in Windows, and many more things. These have had no effect on the issue.

I have also sourced a different USB 1 audio device (a headphone amp) which does not exhibit these issues at all.

Has anyone any suggestions on what else I can try at this point?

vijay a. verma say:
Hi Arin
I am Vijay, an Independent Advisor. I am here to work with you on this problem.

Please try these options one by one and check after each option.

1. Run Hardware & Devices / Audio Troubleshooters
Type Troubleshoot in Windows Start Search box > Click Troubleshoot >
A. Click Hardware & Devices > Run the Troubleshooter
B. Click Playing Audio > Run the Troubleshooter
C. Recording Audio > Run the troubleshooter
D. Speech > Run the troubleshooter

2. Microphone is not disabled and is set as default
Type Sound in Windows Start Seach box > Click Sound > Under Recording tab, right click on an empty space and select, Show disconnected devices and Show disabled devices > Select Microphone and click on Properties and make sure that the microphone is enabled > You may also check if the microphone that you are using is set as default.

3. Set up microphone
Type microphone in Windows Start Search box > Click Set up a Microphone > Select required type of microphone (for internal mic, select Others) > Follow the onscreen directions to set it up

4. Take actions related to Audio Driver
Windows Key+X > Device Manager > Sound, Video & game controllers > Right Click on the entry below > Properties > Driver Tab >

A. Roll Back - If Roll back driver is enabled, roll it back

B. Uninstall - Uninstall Device and restart to reinstall it automatically

C. Download and reinstall the driver from computer manufacturer's website - If you face issue, let me know your computer make and model?

(Source - https://support.microsoft.com/en-us/help/402699...)

5. Check out other steps in HP Guide for Microphone Not Working

https://support.hp.com/in-en/document/c03421813

Do let me know if you require any further help on this. Will be glad to help you.
