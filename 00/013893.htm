File opening in read-only mode caused by hung winword.exe process - Microsoft Community

kbrb say:

File opening in read-only mode caused by hung winword.exe process

My file opened in read-only so I saved it to another file name and attempted to delete the original.  It would not allow me to delete it, telling me "The action can't be completed because the file is open in Microsoft Word".  I closed all Word documents and attempted again to delete the file, with the same result.  Aha! - a hung process.  I went into Process Explorer, which I downloaded long ago from Microsoft and like much better than Task Manager, selected the "Winword.exe" process that was still running even though all Word documents were closed, and selected "Kill Process Tree".  Mission accomplished - my document now behaves normally.  Hope that helps someone.
