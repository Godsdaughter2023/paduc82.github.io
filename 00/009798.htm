    peter.mac say:

    A little more explanation of what I believe is the problem. Whenever an entry is made in the worksheet then calculation will take place. Your line of code FindBold = "" is actually changing the previously calculated value of the Function and it will either change it to a blank result or likely generate a #value error and then the remaining code recalculates to the correct value and this is what is making it appear that it is recalculation because of the changing result.

    You do not need the line FindBold = "" so delete it because the line FindBold = Results resets its value. Without the line, the UDF still recalculates but because it is not changing the previously calculated result, it is not visible to the user.
