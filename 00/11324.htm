Part Description

Though not very clear on your requirement ... you may try below:

=OFFSET(Lookups!$A$1,MATCH($B4,INDEX(Lookups!$A$1:$Z$100,,MATCH("Store",Lookups!$A$3:$Z$3,0)),0)-1,MATCH($A$3,Lookups!$A$3:$Z$3,0)-1)

The formula: "=INDEX(Lookups!$A$1:$Z$100,,MATCH("Store",Lookups!$A$3:$Z$3,0))" will search for "Store" in row no 3 and return the relevant column (could be column N or Q or any other). You may then match B4 value in that column. The rest of your formula remains same.

The Start Menu folder you are trying to access is not a normal folder. It is called a Junction (also called a Soft Link).


A Junction is described as "A shortcut link that redirects programs to a different location".

These Junctions first appeared in Windows Vista and have been in every newer version of Windows.

The reason they are used is because starting with Vista some of the legacy User and System folder names that appeared in previous versions of Windows were changed. Older software installation programs stopped working because they could not locate the folders that were not in their normal locations.

Windows placed these Junctions in the locations and using the names the older folders used. When an older program finds a Junction, the program is automatically redirected to the new folder.

All Junctions have their Permissions set to "Access Denied" for "Everyone". This prevents these folders from being accessed, deleted, or moved, which could result in some software programs not being able to work properly.

In PowerPoint, with the presentation open:
