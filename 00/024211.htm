    stefan blom say:

    It seems that style inheritance is the problem with your Normal template, then. The general idea is that if style A is based on style B, changing properties of B will also change the corresponding properties of A. However, if you modify (say) the font of style A, subsequent changes of the font for style B will have no effect on style A. For more, see http://shaunakelly.com/word/styles/howstylescascade.html.

    To quickly change the font of all of your styles, you can use the following macro:

    Sub ChangeFontOfAllParaStyles()
    Dim s As Style
    For Each s In ActiveDocument.Styles
    If s.Type = wdStyleTypeParagraph _
    Or s.Type = wdStyleTypeLinked _
    Or s.Type = wdStyleTypeParagraphOnly Then
    s.Font.Name = "Arial"  'change this if you want some other font
    End If
    Next s
    End Sub

    For assistance, see http://www.gmayor.com/installing_macro.htm.

    You may also be interested in the article at http://wordmvp.com/faqs/macrosvba/ResetStyles.htm, which discusses how to "reset" a particular property in a style so that it will be inherited from the parent style again.
