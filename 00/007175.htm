As long as you've selected a PostScript printer driver for the output, the file it writes will contain PostScript. If you selected a printer driver that does PCL or some proprietary binary format, that's what the .prn file will contain, but a PostScript printer driver just writes PostScript.

Autofill option in Excel - Microsoft Community

gautam padukone say:

Autofill option in Excel

In Excel, when I try to use Autofill to complete a series A, B, C.... by dragging in the column of cells with the mouse right button, the "Fill Series" option (and some others) are greyed out.   What is the Excel setting that will give me this option?

hansv mvp say:

A, B, C, ... is not a series that Excel recognizes. You'll get the Fill Series option if you start with

  • Two cells with numbers
  • A cell with a text string that ends in a number
  • A date
  • The name of a day or month, in full or abbreviated (depends on language)
  • A quarter such as Q1 (depends on language)

To create a series A, B, C, ..., you can use a formula such as =CHAR(64+ROW()) and fill or copy it down (Fill Series won't work for this either).
