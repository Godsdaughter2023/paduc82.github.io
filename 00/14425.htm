"Edit Story in Microsoft Word" command drops text - Microsoft Community

tom warfield say:

"Edit Story in Microsoft Word" command drops text

Problem:  Using the "Edit Story in Microsoft Word" command sometimes causes words to disappear.  For an example, see

http://www.warfieldconsulting.com/Using_Edit_Story_In_Microsoft_Word_Causes_Words_To_Disappear_Example.pub

Cause:  This happens when the text contains non-breaking space characters. These are not visible unless you have "Special Characters" turned on (the paragraph symbol button).  The non-breaking space character can get added to a text box when you cut-and-paste from a web page. 

Solution:  To remove the offending characters, use Search and Replace. In the “Find What” box, put ^s (the code for non-breaking space). In the “Replace With” box, put a space.  Choose Replace All.  Then try “Edit Story in Microsoft Word” again, and the words remain.

 Problem also exists in the Office 2010 version of Publisher.
