Is there a quick way to delete only the even numbered rows from my - Microsoft Community

dlarocca say:

Is there a quick way to delete only the even numbered rows from my spreadsheet? There are over 5300 lines so I'm trying to find a better way than deleting each one-by-one.

Trying to find a faster way than doing one-by-one to delete only the even numbered rows from a 5300+ row spreadsheet.

jllatham say:

One way:

Practice this on a copy of your workbook first!  It is a 'destructive' process and I'd hate to be responsible for helping you to throw away a lot of data you really need to keep.

Pick an empty column and put a label in the first row of it, such as Filters.

In the cell of that column in row 2 enter this formula:

=MOD(ROW(),2)

Select that cell again - for argument's sake we will say the column is F, so select cell F2.  Then in the 'Name Box' (that text looking area just above the column A indicator that shows you the address of the cell you have selected, Type in

F2:F5300
or whatever the ending row is for your sheet, maybe F2:F5339

Now go to the Home tab over in the Editing group, pull down the Fill option and choose "Down".  That will put that same formula in the F2:F5300 range.

For even numbered rows the value displayed will be zero, for odd numbered rows it will be 1.

Click in F1, where you typed your label for the column and go to the Data tab and choose "Filter" from the Sort & Filter group and select 0 to be filtered.  This will leave only the title row (row 1) and all even number rows displayed.  Select them all from row 2 on down and DELETE them!  Quick way to select them all would be to click in A2 and then press [Ctrl]+[Shift]+[End].

Unfilter and delete the contents of your filter column for final cleanup.

I repeat:  Practice this on a copy of your workbook first!
