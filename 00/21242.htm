Yep, the AutoFormat As You Type monster bit you. It has to do with the "I have a lot of one-line paragraphs scattered throughout the document that I have bold'ed, underlined and increased the font size" activity.

First, to prevent this from ever happening again:

  • Go to File > Options > Proofing > AutoCorrect Options. Click the AutoFormat As You Type tab of the dialog. (Not the AutoFormat tab!)
  • Clear the check box for "Built-in Heading styles". While you're there, it will also save a lot of angst to clear all the other check boxes in the "Apply as you type" section, as well as the "Define styles based on your formatting" box in the "Automatically as you type" section. See http://www.word.mvps.org/FAQs/Formatting/TameAutoFormat.htm for details.

To fix the existing problem documents in bulk (by using the Replace dialog), you would have to be able to identify something that distinguishes the "bad" Level 1 paragraphs from the "good" Level 1 paragraphs. That might be the name of the style applied to them, but I can't tell without seeing the document.

Excel workbook (.xlsx)

OpenDocument Spreadsheet file (.ods)
