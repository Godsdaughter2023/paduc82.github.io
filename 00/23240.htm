wyman w say:

Alt+F11 > Insert Module > Copy and Paste the below code > Run Macro highlightPrime

Public Function isPrimeNumber(sinput) As Boolean
    isPrimeNumber = True
    If sinput = 1 Then
        isPrimeNumber = False
    ElseIf sinput > 2 Then
        For i = 2 To sinput - 1
            If sinput Mod i = 0 Then
                isPrimeNumber = False
                Exit Function
            End If
        Next i
    End If
End Function

Public Sub highlightPrime()
    Set sourceRng = Range("B6:G13")
    For Each Rng In sourceRng
        If isPrimeNumber(Rng.Value) Then
            Rng.Interior.Color = vbYellow
            Rng.Copy
            Rng.Offset(0, 13).PasteSpecial xlValue
            Rng.Offset(0, 13).PasteSpecial xlPasteFormats
        Else
            Rng.Copy
            Rng.Offset(0, 26).PasteSpecial xlValue
            Rng.Offset(0, 26).PasteSpecial xlPasteFormats
        End If
    Next Rng
End Sub

mike h.. say:

Hi,

Select B6 then drag to select your block of cells then

Home tab | conditional formatting | New rule | use a formula to decide which cells to format and enter this formula

=IF(OR(B6=2,B6=3),TRUE,IF(AND((MOD(B6,ROW(INDIRECT("2:"&B6-1)))<>0)),TRUE,FALSE))

Click the format button | fill tab | choose a colour and OK out.

If you use this on other blocks then you must change the bold B6 to the top left cell of that block.

Note that if used on the worksheet this is an array formula. Formula used in conditional formatting are always evaluated as array formula so no special entering is required.

    rick rothstein, mvp - excel say:
    Thank you very much. That worked perfect! However, is there a different way of doing this for Excel 2003 without having to install the Analysis Toolpak?


    This should work in all versions of Excel. Put this formula in the cell...

    =A1-DAY(A1)

    and Custom Format the cell using m/yyyy as the format pattern. That, of course, will be a real date with a day number being the last day of the month. If you don't want it to be a real date, but would rather it just be text, then use this formula instead (leave the Cell Format as General)...

    =TEXT(A1-DAY(A1),"m/yyyy")
