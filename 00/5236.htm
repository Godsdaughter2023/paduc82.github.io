Will the 820/920 need to have the same requirements to be met for bluetooth to work? (bluetooth v2.1 with EDR, HSP, HFP & PBAP)

kosh---01 say:

 

While a higher Bluetooth version may be backwards compatible with earlier versions it should be noted that 'new' features will not work with older Bluetooth devices. As an example, while Bluetooth v2.1+EDR is backwards compatible with Bluetooth 1.2 features like streaming stereo audio will not work as these are not supported in Bluetooth v1.2

 

For the best experience it is advised to verify the Bluetooth version of both devices and ensure both support the featureset you are looking for.
