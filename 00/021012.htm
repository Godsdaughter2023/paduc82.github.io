stefan blom say:
Which alignment option did you try to change? Based on your description, you need to change the one found in the Page Setup dialog box. Do the following: On the Page Layout tab, click the dialog launcher button () in the Page Setup group; Word displays the Page Setup dialog box. On the Layout tab, verify that "Vertical alignment" is set to "Top." Make sure that "Apply to" reads "Whole document." Click Set As Default to transfer the setting to the document template.
