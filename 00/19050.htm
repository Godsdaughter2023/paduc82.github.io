If either is 0, wouldn't the variance be 100%?

=IFERROR((A2-B2)/A2, 1)

robert aldwinckle on forums say:
I clicked on "Diagnose Connection Problems" and got no help at all.


Next time before doing that, switch to an elevated cmd window and enter:

     ipconfig  /flushdns

Then, after using the Diagnose...  but before clicking Close enter:

     ipconfig  /displaydns

Then after clicking Close and waiting for the automatic refresh which results from that, enter that last command again.

In fact, to preserve the output you might want to amend the displaydns steps to:

     ipconfig  /displaydns  >>displaydns.txt

and

     notepad  displaydns.txt

It might also help to do

     netstat  /abnop  TCP

after the automatic refresh is done.


FWIW  I just did that with  youtube.com  as the test host name and the  netstat  step  could be revealing.   Mine didn't fail but it did do a redirect so it would seem that avoiding the redirect and going directly to

http://www.youtube.com/

might be best.

I don't have a YouTube account, so I don't know what all else might be involved in trying to access your account.

 

HTH

  4. Click on Uninstall
  5. Restart the system

 

After the computer restarts, install the driver in compatibility mode

 

  1. Select the setup file
  2. Right click on it and click on properties
  3. Click on Compatibility tab
  4. Select the operating system
  5. Click Ok.
