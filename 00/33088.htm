The most common reason for this behavior is that System Restore has been turned on for that recovery drive. It's not necessary to use System Restore for a recovery drive.

 

Open Control Panel and select System.

On the left side menu, select System Protection.

 

In the Protection Settings section, select the Recovery Drive.

At the bottom of the window, select the Delete Button.
