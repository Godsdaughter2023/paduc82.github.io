lyv2lyv say:

If someone uses my old hotmail address....will I still receive the emails in outlook?

Please help. If someone were to email using my Hotmail address will I be able to see it in outlook?? If not, what do I do to sync?! Please help!
ron sommer mvp 2009-2017 say:

You have one email address.  It is a Hotmail address.

When you use a browser to access your Hotmail account, you are accessing the emails on the server.  Hotmail emails are now received on an outlook.com server. 

Your phone connects to the outlook.com server to access your emails.  There are different protocols that can be used to access the emails.

Your phone is using pop3 protocol to download the new emails to the phone.  After the emails are downloaded, the emails are deleted from the server.  So the emails are no longer on the server when you access the server using a browser.

Only emails in the server Inbox will download to the phone.

To access emails in the Junk folder, you will have to login using a browser or change the way that your phone accesses your account.

bob jones aka: cybertaz say:
The maximum file size for Word documents is 32MB, but that pertains to text only. That translates to roughly 3 million words. I'm betting that the document doesn't even approximate that volume of textual content.

If graphics are included  (additional to the 32MB text), Word can open a 512MB file.

Without more detail about the nature, content, origin & format of the file there's no way of knowing why the file cannot be opened. It's quite possible that the message may be misleading due to extensive corruption in the document. Factors other than file size could be involved as well.

Phillip's suggestion to try with LibreOffice may work. There are a few other options as well;

  • If you can take a copy to a Windows PC running Office 2010 it may be able to open the file. The Windows version of Word has more robust file repair capability.
  • You can try using File> Open -- Enable: Recover Text from Any File. If it succeeds, however, that's exactly what you'll get -- text only without formatting & no graphic content.
  • See if you can open the file with TextEdit. If successful Save As using Word 2007 (.dotx) or Rich Text Format

It's possible, though, that available memory (RAM) and/or disk space may also be factors. Be sure your HD has plenty of free space & that you aren't running any other programs when trying to open the file.

clinton fitch say:

Hello,

Your GSM connection will always show as being connected even when you are on WiFi.  This is to let you know that you have signal & connectivity should it be needed.

That said, Windows Phone will always use the "least cost routing" method when it comes to data connectivity.  WiFi is less expensive than GSM so by default if you are connected to WiFi it will use that first over your GSM.

To get a true sense for how much data you are using and what connectivity path you are using, review it on your phone in Data Sense.  See my screenshot below where I have just slightly pulled down Action Center at the top.  You see that I'm showing H for my GSM data connectivity as well as WiFi.  Looking at Data Sense below however, you see I've used a miniscule amount of GSM data as I'm almost always connected to WiFi.
