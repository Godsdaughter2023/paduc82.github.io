Try

=LEN(A1)-LEN(SUBSTITUTE(A1,"\","")) =FIND("!",SUBSTITUTE(A1,"\","!",B1-1)) =FIND("!",SUBSTITUTE(A1,"\","!",B1)) =FIND(" ",A1,C1) =TRIM(MID(A1,E1,D1-E1))

=TRIM(MID(A1,FIND(" ",A1,FIND("!",SUBSTITUTE(A1,"\","!",LEN(A1)-LEN(SUBSTITUTE(A1,"\",""))-1))),FIND("!",SUBSTITUTE(A1,"\","!",LEN(A1)-LEN(SUBSTITUTE(A1,"\",""))))-FIND(" ",A1,FIND("!",SUBSTITUTE(A1,"\","!",LEN(A1)-LEN(SUBSTITUTE(A1,"\",""))-1)))))

Entering lat / long coordinates: a how to (for - Microsoft Community

henkg---01 say:

Entering lat / long coordinates: a how to (for instance: geocaching use)

How to enter coordinates:
Entering coordinates in the maps app does not work for me, due to the nautical coordinate format I use. You also need to enter the coordinates in the correct format, without any clue how.

 

I find this a better way:
Use the Location app (in Applications)
Be sure to set the coordinate format first (Positioning | Notation preferences)
Start Locations | Landmarks | Options | New landmark | Enter manually.

Enter the latitude and longitude field.
(You will notice that entering is easy because the format is defined. Use any digit to toggle between E/W; N/S)
Then Options | Show on map.

 

If you just want to use your Nokia as a simple GPS device, that guides you to a landmark using a compass display:

Location app | GPS data | Navigation | Options | Set destination | Coordinates
Enter Latitude / Longitude. Tap OK.
(And then walk in the direction of the arrow ...)

Good luck, and don't get lost.
