First create a query that selects the records you want to see in the directory.  You could use the employee table, but it might have entries for emplyees that are no longer active or are allowed to keep that info provate so a query would be used to exclude to records.

 

When the query return the correct set of records with the desired fields, then use it as the record source for a report.  Play around with the report until it looks the way you want.  Use the Sort/Group feature to specify group by school and sort by last name.

 

Repeat with another report just sorted by last name.

 

If you need to combine them in a single printout, use another blank report and drag/drop the above two reports as subreports onto the blank report.

andreas killer say:

I recommend not to use the RowSource property, for such kind of tasks it is easier if you use the AddItem method (or the List property if you have arrays).

IMHO you'll find the sub below interesting.

Andreas.

Sub ForceVisibleRows(Box As Control, Optional RowSource)
  'Add only visible cells from RowSource to List-/Combobox
  Dim Source As Range, C As Range, R As Range
  Dim i As Long, j As Long
  'Force exit if something is wrong with RowSource
  On Error GoTo ExitPoint
  If IsMissing(RowSource) Then
    'Get the range from the control
    Set Source = Range(Box.RowSource)
  Else
    If VarType(RowSource) = vbString Then
      'Get the range by a given name
      Set Source = Range(RowSource)
    Else
      'It should be a range in this case
      Set Source = RowSource
    End If
  End If
  'If we get here, we don't need RowSource anymore
  Box.RowSource = ""
  'Exclude hidden cells
  Set Source = Source.SpecialCells(xlCellTypeVisible)
  'If we get here, we have some cells, make sure our process continues
  On Error Resume Next
  'Visit first column of the given range
  For Each C In Intersect(Source, Source.Columns(1).EntireColumn)
    'Add a row
    Box.AddItem C
    j = 0
    'Add each element of that row
    For Each R In Intersect(Source, C.EntireRow)
      Box.List(i, j) = R
      j = j + 1
    Next
    i = i + 1
  Next
  'Clear errors if any
  Err.Clear
ExitPoint:
End Sub
