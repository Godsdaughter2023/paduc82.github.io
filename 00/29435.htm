I understand that the Taskbar pops up when the mouse hovers at the bottom of the screen, while the game is running in Full screen mode. Ideally, some games allow you to hide the Taskbar while playing games and some will show the Taskbar by default. We would like to know, if this issue occurs with all the games or only with one particular game.

 

If you are facing this issue with one particular game, then there must be some in game setting that you might have to change. If you are facing this issue with all the games, then you can try one workaround and see if that helps.

 

The workaround is to close Windows Explorer process while you are playing game. This will allow you to play the game uninterruptedly. However, when you End the Windows Explorer process, all the windows that are running in the background will be closed, which allow you to play the game without any interruption of the Taskbar. When you want all the windows back, you can restart the Windows Explorer process.

 

Please follow the steps mentioned below to do the same:

 

  1. Press Ctrl + Shift + Esc key while playing game, this will open Task Manager.

  2. Look for Windows Explorer process, right click on it and select End Task.

  3. When you want the windows back, open Task Manager by following Step 1, and click on File.

  4. Click on Run new task, type explorer.exe and click on OK. Windows Explorer will be restarted and you will have all the windows back again.

To install Windows 7 on a computer without an operating system

To do this, you'll need to start, or boot, your computer using the Windows 7 installation disc or USB flash drive.

  4. On the Install Windows page, enter your language and other preferences, and then click Next .

  6. On the Which type of installation do you want? page, click Custom .

  7. On the Where do you want to install Windows ? page, choose the partition that you want to install Windows 7 on, or click Unallocated Space if no partitions are listed, and then click Next .

    If a dialog box appears saying that Windows might create additional partitions for system files, or that the partition you chose might contain recovery files or other types of files from your computer manufacturer, click OK .

  8. Follow the instructions to finish installing Windows 7 , which include naming your computer and setting up an initial user account.

Hi,

Base on the information that you provided, probably your computer is set to 16-bit resolution.

To change 16-bit to 32-bit, kindly follow the steps below:

1. Right-click on an empty space on your desktop and select Screen Resolution option to open the screen resolution settings for Windows.
2. If you have more than one monitor connected (like a projector or an HD TV) select it from the drop-down list and click on the Advance Settings link.
3. When you click on the Advance Settings link Windows will open the property window of your default Graphics Card. Navigate to the Adapter tab in the window and click on the button List All Modes.
4. In the popup window select the display mode you would like to apply and click on the OK button. The screen may blackout for few seconds before your settings are applied. Confirm your actions to make the changes permanent.

Note: If you don’t want to change the resolution but just want to toggle the 16 and 32 bit color, click on the Monitor tab in the Driver properties window and select the specific color depth from the drop-down control and click OK .
