macropod say:

You could use a macro like:

Sub Demo1()
Application.ScreenUpdating = False
Dim StrFnd As String, i As Long
StrFnd = "One,Two,Three"
With ActiveDocument.Range.Find
    .ClearFormatting
    .Replacement.ClearFormatting
    .Replacement.Text = "Replacement Text"
    .Forward = True
    .Wrap = wdFindContinue
    .MatchCase = True
    .MatchWholeWord = True
    .MatchWildcards = False
    .MatchSoundsLike = False
    .MatchAllWordForms = False
    For i = 0 To UBound(Split(StrFnd, ","))
        .Text = Split(StrFnd, ",")(i)
        .Execute Replace:=wdReplaceAll
    Next
End With
Application.ScreenUpdating = True
End Sub

It's debatable whether that's more efficient than:

In Word 2010, page numbering [page * of *] and - Microsoft Community

jwilsoodn say:

In Word 2010, page numbering [page * of *] and [file name] in a footer that appears on every page

In Word 2010, How do I put page numbering [page * of *] and [file name]  in a footer that appears on every page of a document?
suzanne s. barnhill say:

To insert Page X of Y:

  1. Double-click in the header area to access the header and footer.
  2. Scroll down to the footer.
  3. Place the insertion point (cursor) where you want the page number (using the built-in tab stops, if necessary, for placement).
  4. Type "Page ".
  5. Press Alt+Shift+P to insert a PAGE (page number) field.
  6. Type " of ".
  7. Press Ctrl+F9 to insert a field. You'll see field delimiters that look like braces but can't be typed from the keyboard.
  8. Between the braces, type NUMPAGES (or numpages or NumPages--Word doesn't care about the case) so that you have { NUMPAGES  }.
  9. Click in the field and press F9 to update.

To insert the filename:

  1. Place the insertion point where you want the filename.
  2. Press Ctrl+F9 to insert a field.
  3. Between the field braces, type FILENAME and add the \p switch if you want the entire field path: { FILENAME \p}
  4. Click in the field and press F9 to update.

You can insert either or both of these fields from the field dialog as Rohn007 describes, but it saves time if you learn to do it manually. To save even more time, you can save these items as AutoText. Select either of the fields you've created and press Alt+F3, which will open this dialog:

Choose a name for your AutoText entry, and you will be able to insert it in one of three ways:

  1. From the Insert | Text | Quick Parts | AutoText menu.
  2. By typing the entry name and pressing F3.
  3. By typing the first few letters of the name and then pressing Enter or Tab when you get the AutoComplete suggestion (popup).

There's actually already a defined Quick Part (in Insert | Header & Footer | Page Number) for Page X of Y, but it uses bold numbers. If you save your Page X of Y in the Page Numbers gallery instead of AutoText, it will be included along with the others.
