gts-nj say:

Your comment - "That's only if I try to do a sector-by-sector image, which I think I need in order to restore my whole hard drive if it fails. :

This is incorrect.  It is NOT necessary to do a sector-by-sector image image in order to do an Acronis restore.  In fact, that option is, and should be, very rarely used.  It greatly increases the backup size with no benefit.  It also creates certain limitations in selecting a replacement target drive.

It's only real value is for backups intended to be used later for forensic investigation.


re. terminology


Clone a disk - This copies your entire system to another disk.  Acronis will set up the boot sector and adjust for different sizes in the disks such that the replacement drive is bootable.  This is usually done if upgrading to a larger hard drive or replacing a problem drive.  


Imaging a disk - This makes an image of the drive and partition information and stores it in one file.  That image file can be used to restore your system to a new drive by running an Acronis restore operation (usually from a bootable media) which uses the image file to build the replacement drive data. 


   When an image is built Acronis (and similar programs) will save only the content of sectors that are actually being used.  The sector by sector option also copies / saves unused sectors from the source drive.  This is generally undesirable as it's wasted space to save the unused sectors in the image.


In other words, Cloning is used when you are immediately replacing a drive.  Imaging is used for backup.

If the Player is currently open and you‚Äôre in Now Playing mode, click the Switch to Library button in the upper-right corner of the Player.

2.    In the Player Library, click the Play tab, if necessary, to open the list pane.

If you need to clear the list pane before creating your playlist, click Clear list.

3.    Drag items from the details pane to the list pane to add them to the new playlist.

To select multiple adjacent items, press and hold the Shift key while selecting. To select non-adjacent items, press and hold the Ctrl key while selecting.

4.    To rearrange items, drag them up or down in the list pane.

5.    To save the list, click Save list at the top of the list pane, type a name for the playlist, and then press Enter.

By default, playlists are saved with a .wpl extension in the Playlists folder, which is located in the default save folder of your Music library. You can also save playlists with the .m3u or .asx extension. To do so, at the top of the list pane, click the List options button, click Save list as, and then change the extension in the Save as type dialog box. After you save the playlist, it's added to the Playlists category in the navigation pane.

To play the playlist in the list pane, click the Play button.

6.    To access the playlist in the future, double-click the Playlists view in the navigation pane or click the arrow next to Playlists to expand the view.

After installing the February 13, 2018 security update, KB4074588 (OS Build 16299.248), some USB devices and onboard devices, such as a built-in laptop camera, keyboard or mouse, may stop working for some users.

Cause
This issue occurs when the Windows Update servicing stack incorrectly skips installing the newer version of some critical drivers in the cumulative update and uninstalls the currently active drivers during maintenance.

A failed Windows Update is the likely cause of this problem.
