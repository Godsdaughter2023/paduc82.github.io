Avoiding issues with the OneDrive Application, on Mac, when it finds a - Microsoft Community

nathan roberts (sunknath) say:

Avoiding issues with the OneDrive Application, on Mac, when it finds a file path that is too long.

Technical Level : Basic

Summary

Throughout this article, I will explain how  you can avoid issues with the OneDrive application, on your Mac, when it finds a file path that is too long to upload onto OneDrive. The user will also get a message, asking them to restart the application all over again. Meaning that the user will need to set up the OneDrive from scratch, just like they would have needed to when they had first downloaded the application onto their Mac. 


Details

When OneDrive encounters a file path is too long it will bring up an error which will tell you that the OneDrive has stopped synchronizing. On this error, it will have an explanation of why OneDrive has stopped working and, if it is because the file path is too long, it will tell you what the name of the file is at the bottom of the message. 

It will also ask you to restart the Application, meaning you will have to set everything back up when you open it up again. This will also include the files and folders you wish OneDrive to Synchronize onto your device. This can be a lengthy process, and you will still need to shorten the file path of the folder that OneDrive says is too long, otherwise you will get the same message and possibly go round in circles. 

Instead, you want to use the error message and find the file that OneDrive is saying is too  long. You should be able to see the path near the bottom of the message. Once you find the file, that is causing the issue, you can, either, rename the file so that the path is shorter, or you can move it out of that location and into another one. Once you have done this, you will then need to open up your Activity Monitor (For Windows this would be called Task Manager) and find OneDrive. Once you have found it, you will need to force the application to close, by clicking on Force Quite. This will bring up a message asking you to verify your choice, you will then need to verify that you would like the device to close OneDrive. 

Once you have closed down OneDrive, you will then be able to open the application again, and it should start synchronising from where it stopped before. This time, the error message should not show.

denislepoivre say:

2 questions:

1.Do you have a Microsoft Acount on your phone? Check Settings > Email+Accounts > Microsoft Account

2.Do you have the same issue installing other apps?

If the answer to both of the above is "Yes", try these steps:

1) Switch off automatic Date and Time:

Go to Settings > Date+Time and switch off Automatic settings if applicable.

2) Soft reset the device (no data loss):

To do a proper soft reset, press and hold VOLUME DOWN + POWER buttons for 10-15 seconds until the phone vibrates and turns off by itself /!\ Do not Slide down to power off /!\

3) Correct Date and Time:

Go back to Date+Time make sure the date and time are correct

4) Force the Microsoft Account sync:

Go to Settings > Email+Accounts > tap and hold Microsoft Account and tap Sync

Once the Microsoft Account has finished syncing, check that no error message appears.
