For those who are facing the same issue, the problem turned to be in the installer program. It checks for the previously installed component by looking for the existence of their respected registry key and skips over their installed if they are there. This won't work if the previous uninstaller didn't clean up properly after itself and left those keys behind!

Deleting the following 3 keys manually before running the new BCM Setup fixed the issue for me.

HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion\Uninstall\{E4B48349-A165-4097-8D78-AC950BD8638E}

HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion\Uninstall\{050BF7DA-82C4-416A-8294-7AFEB8ED94E1}

HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion\Uninstall\Business Contact Manager

bernie deitrick say:

In part, your code did not work because you did not use Set:

Set EndCell = Target.End(xlUp)

To return a value to a UDF function in a worksheet, use code like this, and use the function like

=MyAverage(C100)

If the blanks are really blank:

Function MyAverage(Target As Range) As Double
MyAverage = WorksheetFunction.Average(Range(Target, Target.End(xlUp)(2)))
End Function

If the blanks are formulas:


Function MyAverage(Target As Range) As Double
Dim LastC As Range
Set LastC = Target
While Not LastC(0).Value Like "*Delay Time"
Set LastC = LastC(0)
Wend
MyAverage = WorksheetFunction.Average(Range(Target, LastC))
End Function

Note that by only passing a single cell, your function will not be updated if the values change in the other cells - you will need to force a re-calc.

If the data is on Sheet1 and you have a Sheet2 in the workbook, if you run a  macro containing the following code

Dim i As Long, j As Long, k As Long
With Sheets(2).Range("A1")
    .Offset(0, 0) = "Location"
    .Offset(0, 1) = "Product"
End With
k = 1
With Sheets(1).Range("A1")
    For i = 1 To .CurrentRegion.Rows.Count - 1
        For j = 0 To .CurrentRegion.Columns.Count - 1
            If .Offset(i, j) = "X" Then
                Sheets(2).Range("A1").Offset(k, 0) = .Offset(i, 0)
                Sheets(2).Range("A1").Offset(k, 1) = .Offset(0, j)
                k = k + 1
            End If
        Next j
    Next i
End With
