ossiemac say:

Try the following code. Note the comments.

 

Sub DeleteSheets()

    Dim ws As Worksheet
   
    Application.DisplayAlerts = False   'Optional
   
    For Each ws In Worksheets
        Select Case ws.Name
            'Include sheet names to keep on next line (with comma between)
            Case "Sheet1", "Sheet2", "Sheet3"
                'Do nothing
            Case Else
                ws.Delete
        End Select
    Next ws

    Application.DisplayAlerts = True
   
End Sub

steve rindsberg say:

PPT sure seems to go out of its way to make this difficult, doesn't it?

Go to the slide where you need to swap comments and run this bit of VBA.

It doesn't actually swap comment order; instead it adds a new (third) comment with the same characteristics as the first, then deletes the first, so the second comment is now the first and the newly added third comment is now number two.

That was your cue.  You're supposed to ask me "Who's on first?"

Sub SwapComments()

    Dim oSl As Slide
    
    Set oSl = ActiveWindow.Selection.SlideRange(1)
    With oSl
        With .Comments(1)
        ' add a new comment with the properties of the first
        oSl.Comments.Add .Left, .Top, .Author, .AuthorInitials, .Text
        ' delete the first, making the second comment the first
        ' and the newly added one the second
        oSl.Comments(1).Delete
        End With
    End With
            
End Sub
