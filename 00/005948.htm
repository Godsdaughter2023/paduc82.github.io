techfreak_ say:
Only the last 30 days of active history is stored in the cloud, though you may see a time stamp placeholder/cached singular message of your last correspondence. Any other history over 30 days would be stored in your local history file (main.db located in your Skype name folder in %appdata%\Skype). In some versions of Skype you can export this data into a csv file (Skype 7.x and 8.x). If you no longer have a copy of your older main.db file due to lost data or a format, there would be no history available to retrieve. The newer versions of Skype have their own history file location and do not import older history data.

Hi loangirl,

I don't think you need to replace Windows Defender with a different AV program (it's perfectly fine) - but the decision is yours.

Should you decide to do so anyway, then check with the vendor to be sure the AV program is compatible with your Operating System.  As a general rule, it doesn't matter which browser you use to download the installation file for the replacement AV program.  Once downloaded, the Operating System will be doing the installation.
