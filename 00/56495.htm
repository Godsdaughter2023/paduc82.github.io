TrentQueen, refer to the Excel VBA code at this page:

http://excelpoweruser.blogspot.com/2012/04/looping-through-folders-and-files-in.html

To explain the code:

1. The Sub GetFolderStructure requests the user to select the main folder.  In your example the main folder is "F:\".  "F:\" is the value that you would put in cell A1.

'===================================================
               'A procedure to call the Function  LoopThroughEachFolder(objFolder)
'===================================================
    Sub GetFolderStructure()
    '
    lngCounter = 0
    Set objFso = CreateObject("Scripting.FileSystemObject")
    With Application.FileDialog(msoFileDialogFolderPicker)
        .Show
        Set ObjFolder = objFso.GetFolder(.SelectedItems(1))
    End With
    Range("A1").Offset(lngCounter).Value = ObjFolder.Path
    LoopThroughEachFolder ObjFolder
    End Sub

2. The Function LoopThroughEachFolder loops through each subfolder within "F:\".  This then populates the cells from A1 on down with each folder name.

'===================================================
               'Function to Loop through each Sub Folders
'===================================================


Function LoopThroughEachFolder(fldFolder As Object)

    For Each objFldLoop In fldFolder.subFolders
        lngCounter = lngCounter + 1
        Range("A1").Offset(lngCounter).Value = objFldLoop.Path
        LoopThroughEachFolder objFldLoop
    Next


End Function

       If Len(NewFolderName.Text) > 0 Then

captain_wd say:

Virtualization software is any application that creates a virtual environment such as virtual folders, partitions or whole drives as well as Operating systems that appear as separate physical parts of the system but are only logical. For example, you may have a Windows10 installed on your system but have a virtual machine on top of this Windows10 that is running a Windows XP. This Windows XP has dedicated portion of the CPU, memory and all other resources including portion of the storage drive that isn't available to the main system but will appear in the Disk Management as a separate partition (at certain settings). 

Another thing may be a program that reserves some space for some operations and creates such partitions.
