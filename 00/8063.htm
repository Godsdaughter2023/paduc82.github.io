john sr wilson say:

If you are  atotal beginner this may be a step too far. I would sugggest manually copy the image in Presentation 1 then go to Presentation 2 and select the slide to place the image.

Then try this macro

Sub picinsert()
Dim osld As Slide
Dim opic As Shape
On Error GoTo err
Set osld = ActiveWindow.View.Slide
Set opic = osld.Shapes.Paste(1)
opic.PictureFormat.ColorType = msoPictureWatermark
opic.ZOrder (msoSendToBack)
osld.TimeLine.MainSequence.AddEffect Shape:=opic, _
                                     effectID:=msoAnimEffectAppear, _
                                     trigger:=msoAnimTriggerAfterPrevious
Exit Sub
err:
MsgBox "Something went wrong"
End Sub

anula d'souza say:

Hi Rick,

Welcome to Microsoft Community and thank you for posting the question. As per the issue description, the option enable content updates from office.com is greyed out.

 

Let me assist you with the issue.

 

Follow the steps mentioned and check if it helps:

·         Open PowerPoint application.

·         Click on File->Options -> Trust Centre.

·         Under Microsoft PowerPoint Trust Center click on Trust Center settings.

·         Under Privacy options, check the option Connect to Office.com for updated content.

·         Click Ok twice. Close and open PowerPoint and check.

 

Refer to the link mentioned which discusses about similar issue and check if it helps:
