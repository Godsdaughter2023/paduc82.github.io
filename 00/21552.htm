steve rindsberg say:

It seems possible that the presentation's default language is set to Turkish. Your name suggests that that might be the case ;-)

Try this:

Open the presentation

Press ALT+F11 (this opens the VBA editor; do not be afraid; it will not harm you)

If you don't see a small window with "Immediate" in the blue title bar, press CTRL+G to make it appear

Click in the window, copy and paste this into it:

ActivePresentation.DefaultLanguageID = msoLanguageIDGerman

Press ENTER.

Now add some new text and verify that PowerPoint thinks it's German and not Turkish. 

This will actually make German the default language for *PowerPoint* (not just the current presentation), if you want to change it to some other default language, you'll need to use the VBA editor again. 

Repeat the same steps, then as soon as you type:

ActivePresentation.DefaultLanguageID =

you'll see a whole list of possible languages; it should be easy to choose the one you want.

If it's still not working, please perform the following:
1. Make sure that Skype is no longer running https://support.skype.com/faq/FA10980/.

2. Press the Windows + R

3. Type %appdata%\skype into the Run window then click OK.

4. Find and delete the shared.xml file. If file extensions are
not displayed on your computer, the file will just be called shared.

5. Open the folder that has your Skype name.

6. Locate and delete the configorconfig.xml file.

7. Press the Windows + R keys on your keyboard at the same
time again.

8. Type %appdata% into the Run window and click OK.

9. Locate and rename the Skype folder to Skype_Old.

10. Relaunch Skype.
