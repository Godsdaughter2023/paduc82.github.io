How do I wrap text in a report label box - Microsoft Community

kimspringer say:

How do I wrap text in a report label box

I am trying to set the size of the label so that the text wraps.  I can't find a format option for this for a label in my report.

tom van stiphout say:
It's the CanGrow property.

ken sheridan say: Are you talking about a Label control in a normal report, or a Text Box control in a labels report?  With the former, as the text is normally entered at design time the control is simply sized to sufficient depth to accommodate the text.  With the latter the text box is normally of fixed depth, governed by the size of the labels to which the report is being printed.  To allow it, and consequently the detail section, to grow would  cause misalignment of all following labels.

If you are talking about a normal report and you are setting the Caption property of a Label control dynamically at runtime, then you should use a Text Box control, not a Label control, and assign the text to its Value property, which, as the default property, does not need to be specified.  The text box can then be allowed to grow.
