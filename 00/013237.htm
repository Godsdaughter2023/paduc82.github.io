 UndoList = Application.CommandBars("Standard").Controls("&Undo").List(1)

 If Intersect(ActiveCell, Range("B5:M12")) Is Nothing Or Intersect(ActiveCell, Range("B15:C15")) Is Nothing Or Intersect(ActiveCell, Range("D13")) Is Nothing Then
   
        GoTo LetsContinue
  
   Else
   
        Application.ScreenUpdating = False
        Application.EnableEvents = False
   
        On Error GoTo Whoa
   
        '~~> Get the undo List to capture the last action performed by user

davidwisniewski say:
I reviewed your attached code and I believe I have a solution to your problem, and also a way to accomplish your goal using less resources. Instead of the sub running everytime there is a sheet change, you can simply tell excel to specifically listen for the key combination CTRL V. This will work because all your sub is doing is checking to see if it was a paste action performed, and if it was then it undoes it and redoes it as a paste special. With the new code I have included it skips the whole undo action and just captures the CTRL V combination, which is always a paste, and changes the paste style to Paste Special - Values. Here is how to do it:

With your spreadsheet open, view the Visual Basic Editor. Delete the "Workbook_SheetChange" sub routine and your "Macro1" sub routine. Create a new Module and name it modPasteValues. Paste this text in that module.

''''''''''''''''''''''''''''''''''''''''''
'When the CTRL V key combination event is caught it calls this sub to
'handle the event, which forces a Paste Special - Values rather than just Paste
''''''''''''''''''''''''''''''''''''''''''
Sub subPasteValues()
    On Error GoTo errHandler
    Application.ScreenUpdating = False
    Application.EnableEvents = False
    Selection.PasteSpecial Paste:=xlPasteValues, Operation:=xlNone, SkipBlanks:=False, Transpose:=False
ActiveWorkbook.RefreshAll
CleanUp:
    Application.ScreenUpdating = True
    Application.EnableEvents = True
    Exit Sub
errHandler:
    MsgBox Err.Description
    Resume CleanUp
End Sub
''''''''''''''''''''''''''''''''''''''''''

Now open the "ThisWorkbook" module and paste the following key listener into it.

''''''''''''''''''''''''''''''''''''''''''
'When the workbook opens, an event listener is added to catch the CTRL V
'key combination and runs the specified sub when triggered
''''''''''''''''''''''''''''''''''''''''''
Private Sub Workbook_Open()
Application.OnKey "^v", "subPasteValues"
End Sub
''''''''''''''''''''''''''''''''''''''''''

If this solutions solves your issue then please mark it as the answer. Thanks.
