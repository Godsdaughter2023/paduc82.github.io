julie fka profjulie say:

At this point, the only way to restore the contacts to your HTC S740 is by copying the PIM.vol file to thye HTC S740.  This file won't work on your new HTC Touch2 and attempting to use PIMbackup after the contacts have already been lost won't do anything for you either.  (BTW, PIMBackup is a windows mobile application and would need to be installed on your HTC S740 - it doesn't work on a computer).

So, here's what I think you should do, soft reset the HTC S740.  Rename the existing PIM.Vol file that is on the HTC S740  to PIM.bak and then copy the PIM.Vol file that is on your computer to the HTC S740.  Then check to see if the contacts  have been "restored" to your device.

Once you restore the contacts to your HTC S740, then you can get the data over to your new HTC Touch2 b siyncing both of the devices to your computer via ActiveSync/WMDC.  To do this, you'll need to have Outlook and ActiveSync or WMDC installed on your computer (ActiveSync works on an XP machine; WMDC works on a Vista or Windows 7 machine).  Then you can set up a partnership between your computer and the HTC S740 and sync the data over to Outlook on the computer.  Afterwards, you should be able to set up a syncing partnership between your computer and HTC Touch2 and sync the data over to your HTC Touch2.

Speakers (Cirrus Logic CS4206B) (AB 29)

Default Device - YES

Default Format 32 bit, 48000 Hz (Studio Quality)

Digital Audio (S/PDIF) (Cirrus Logic CS4206B) (AB 29)

2 channel, 16 bit 48000 Hz (DVD Quality)

Then select 'Stereo', Next, Next, Finish.

This should work now.

Phillip, that's not what the poster asked.

NepalKat, this can be done but you'd need a macro for it. You'll find several versions if you google for "macro word date filename" but you'll need to adapt them, then add them to a toolbar or define a keyboard shortcut. This is my version:

Sub SaveWithDate()
If ActiveDocument.Name = ActiveDocument.FullName Then
    If Not Application.Dialogs(wdDialogFileSaveAs).Show Then Exit Sub
End If
    CurrentFormat = ActiveDocument.SaveFormat
    CurrentPathAndName = ActiveDocument.FullName
    CurrentPath = Replace(ActiveDocument.FullName, ActiveDocument.Name, "")
    NewNameSamePath = CurrentPath & Format(Now, "yyyy-mm-dd") & " " & ActiveDocument.Name
    ActiveDocument.SaveAs Filename:=NewNameSamePath, FileFormat:=CurrentFormat
    kill CurrentPathAndName
End Sub

How it works
  • If the file is unsaved, Word will prompt you for a name and save it, then continue executing the rest of the macro (unless you press Cancel).
  • If the file was already saved, Word will add a leading date as YYYY-MM-DD before the current name, save the document with the new name in the same folder, then delete the original.
Use it only once per document, lest you pile up date upon date.

To add the macro to your Normal.dotm template
  • In your main menu bar, select Tools > Macro > Visual Basic Editor
  • You should see a Project box
  • In there, select Normal (Normal.dotm)
  • In the main menu bar of the Editor, select Insert > Module
  • A new window will open
  • Paste the above code (from "Sub" to "End Sub")
  • You can save your Normal.dotm template now (File > Save Normal.dotm) or Word will prompt you when quitting
  • In the main menu, select Close and return to Microsoft Word (or do Command Q)
To create a new toolbar and add the macro to it
  • In the main menu bar, select Tools > View > Customize Toolbars and Menus.
  • At the top of the dialog box, select the Toolbars and menus tab.
  • Click on the New button at the top right. Give the toolbar a name and press OK. Pay attention, as the new toolbar will be hard to see because it's small. You can move it around by grabbing it from the "colon" below the Close circle.
  • While still in the Customize... dialog box, click on the Commands tab at the top.
  • In the "Categories" list to the left, click on "Macros" to select it.
  • In the list to the right, locate "SaveWithDate."
  • Click on it and drag it to the new toolbar.
  • If you click on the "Keyboard" button at the bottom, you'll be able to add a shortcut (for instance, Control S).
  • As before, you'll need to save Normal.dotm (if Word prompts you, do so).

Hi Hsg1,

 

To achieve your requirement, you may need to use VBA code:

  1. Open the Excel sheet>click Developer ribbon>Visual Basic. To add the Developer ribbon, click File>Options>Customize Ribbon>select Developer in the right panel.
  2. In VBA editor>right-click ThisWorkbook>Insert>Module.
  3. Copy and paste the following VBA code

Sub pagenumber()

'updateby Extendoffice 20160506

    Dim xVPC As Integer

    Dim xHPC As Integer

    Dim xVPB As VPageBreak

    Dim xHPB As HPageBreak

    Dim xNumPage As Integer

    xHPC = 1

    xVPC = 1

    If ActiveSheet.PageSetup.Order = xlDownThenOver Then

        xHPC = ActiveSheet.HPageBreaks.Count + 1

    Else

        xVPC = ActiveSheet.VPageBreaks.Count + 1

    End If

    xNumPage = 1

    For Each xVPB In ActiveSheet.VPageBreaks

        If xVPB.Location.Column > ActiveCell.Column Then Exit For

        xNumPage = xNumPage + xHPC

    Next

    For Each xHPB In ActiveSheet.HPageBreaks

        If xHPB.Location.Row > ActiveCell.Row Then Exit For

        xNumPage = xNumPage + xVPC

    Next

    ActiveCell = "Page " & xNumPage & " of " & Application.ExecuteExcel4Macro("GET.DOCUMENT(50)")

End Sub

  1. Close the editor and return to the sheet.
  2. Select a cell where you want to insert page number. Click Developer>Macros>select that macro>Run. You will see the page number is inserted to that cell.
