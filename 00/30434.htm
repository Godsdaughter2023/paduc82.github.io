davem121 say:

You can change the width in your Registry, open Regedit (click your Start button, type regedit and hit enter) 

Navigate to: HKEY_CURRENT_USER\Control Panel\Desktop\WindowMetrics

Then set the ScrollWidth key using this formula:

-15 * desired width in pixels

For example to set the scrollbar width to 20px:    -15 * 20 = -300

Or there is a free utility that will allow you to set the width of the Windows 10 scrollbar - Winaero Tweaker
