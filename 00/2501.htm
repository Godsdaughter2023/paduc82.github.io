rich007a say:

One option is to press the letter H on the keyboard to advance your slide show.  If the next slide is hidden it will be shown, but stupidly it only advances the show if the next slide is hidden.

 

You could use the following code:

Sub UnHideAll()
    Dim sld As Slide
   
    For Each sld In ActivePresentation.Slides
        sld.SlideShowTransition.Hidden = msoFalse
    Next sld
End Sub

 

And you could hide a list of slides like this:

Sub HideList()
    Dim List As Variant, x As Variant
    
    List = Array(2, 4, 5) 'List of slides to be hidden
    For Each x In List
        ActivePresentation.Slides(x).SlideShowTransition.Hidden = msoTrue
    Next x
End Sub

 

Hope that helps.

Rich

Seeking the formula to enter in conditional formatting rules to do this.  Also wondering if there's any way through conditional formatting to format a thick borderline between rows when the value in Column A changes. Thanks!
rich007a say:

OK, not that hard.

Right-click on your sheet's tab and View Code.  Paste in the following:

Private Sub Worksheet_Calculate()
    Dim c As Range, rng As Range
    Dim strLastCell As String
    Dim b As Boolean
   
    Set rng = Range("A2", Cells(ActiveSheet.Rows.Count, 1).End(xlUp))
    For Each c In rng
        If Not c.EntireRow.Hidden Then
            If c.Value <> strLastCell Then b = Not b
            If b Then
                c.EntireRow.Interior.Color = RGB(200, 200, 200)
            Else
                c.EntireRow.Interior.Pattern = xlNone
            End If
            strLastCell = c.Value
        End If
    Next c
End Sub

 

To make the Worksheet Calculate event fire when the auto-filter is chanaged, you have to have a volatile function on the sheet.  So in a cell somewhere out of the way, enter 

=Now()

That should do it.  (And you can get rid of that column with the IFs, and all the conditional formatting now).

Rich

steve rindsberg say:

If you've used two screens before, it's not unusual for apps to remember where they were when last invoked, even if that means they appear offscreen when you're back to one screen.

The trick in that case is this:

Press Alt + TAB until PowerPoint is highlighted.

Then press Alt + SpaceBar then M 

Press the left arrow key until PowerPoint becomes visible on your main screen, then press Enter.

From there you can grab PPT by the title bar and move it wherever you like.
