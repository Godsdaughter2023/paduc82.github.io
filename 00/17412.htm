How to move Personal Folder Files off of C: drive to another drive - Microsoft Community

richardlieber say:

How to move Personal Folder Files off of C: drive to another drive

My C: drive is practically full.  Is there a way to move my personal folder files which are rather large off of C: drive to another data drive in Outlook 2013?
mrs. richardson say:

Hi Richard,

if you're using a POP3 account, which uses pst-files, you can simply move the file to any location you want. If you start Outlook it then tells you that it cannot find the data file and you need to navigate to it and select it.

If you're using an IMAP account, which uses ost-files, it's more complicated as Outlook will recreate the data file when you move it. In this case you need to modify the registry.

1. Back up your data (export them into a pst-file) and close Outlook.

2. Open the Registry Editor (Windows Key + R > regedit > OK)

3. Navigate to the following path: HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Outlook

4. Right-click on Outlook, select New and then String Value.

5. Name it ForcePSTPath and press enter.

6. Double-click ForcePSTPath and as the value add the path for the new location for your data files (e.g. D:\MyDataFiles)

7. Click ok and close the registry editor.

8. Recreate your email profile (http://support.microsoft.com/kb/829918/en-us), the new data file which is added to that profile will be stored at the path you entered in step 6.
