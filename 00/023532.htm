Using Word, how do I insert WHITE text into a picture ? - Microsoft Community

keith sherren say:

Using Word, how do I insert WHITE text into a picture ?

Using Word, how do I insert WHITE text into a picture ?

jay freedman say:
  • Insert a text box and position it over a dark part of the picture. By default, it will have an opaque white background that blocks part of the picture.
  • Type the text inside the text box.
  • Select the text, right-click it, and use the font color tool in the mini-toolbar or on the Home tab to change its color to white. It will seem to disappear, but it's still there. Also, if the picture background isn't uniformly dark, you can use the Text Effects button on the Home tab to add a shadow to the text, which will make it stand out better.
  • On the Drawing Tools tab of the ribbon, click the Shape Fill button in the Shape Styles group and click No Fill. Now you should see the white text against the dark picture. Also click the Shape Outline button and click No Outline.
  • If the picture and the text box are both floating objects (not "in line with text"), you can hold down Ctrl and click each so they're both selected. Then on the Drawing Tools tab, click the Group button in the Arrange section, and click Group in the dropdown menu. Then the picture and the text will move together if the document is edited or reformatted.
