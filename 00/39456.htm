 1. In Programs and Features, the drivers don't show up as "Atheros", or even "Bluetooth"; they are listed under "Dell WLAN and Bluetooth Installation". Similarly, the "Bluetooth Suite" folder is under "C:\Program Files (x86)\Dell Wireless".

(Better to have a functional computer with Windows 7 than a blind and deaf one with Windows 8.) And they do. So here is how it went:

   1. Go to Dell Web site, then to support section, enter service tag.
   2. Go to Drivers tab, make sure Windows 7 is selected, then find the driver for the Dell 1703 WLAN hardware and download it. (I downloaded the Windows 8 version just for fun, too. Also backed up these files to another PC on my home network because after 20 years as a sysadmin I'm paranoid about these things. Yes, I always typed "sync" 3 times before shutdown. :-) )
   3. In Programs and Features, uninstall the Dell WLAN package.
   4. Bingo! Bluetooth gone, but - oh dear - WiFi connection is also toast.
   5. To get back the WiFi so I could do some stuff on my local network before moving the other adapter, I installed the Dell WLAN driver I had downloaded.
   6. Hey! Look: it offers the WLAN and Bluetooth drivers as separate, selectable options! Unselect Bluetooth and install only WLAN.
   7. Eureka! WiFi is back. But no Bluetooth (don't care).
   8. EUREKA! The Windows 8 upgrade setup program no longer complains about Atheros, and off it goes, chugging steadily through the whole process, with no further glitches.

Then I will make another image backup! :) Oh, and install Media Center Pack with the promo code before that expires tomorrow.
