t. valko say:

The lookup table will only have the first 3 characters, below is a better example

 

Manuf P/N           Manufacturer

293D106X...          Not defined

293D106X... is the lookup value

 

Table Array

Manuf P/N            Manufacturer

293                      vishay

 

This is just one example there is thousands of examples.

 

Mike


Try this...

Lookup table in the range F2:G12

288 88
289 77
290 18
291 35
292 32
293 65
294 37
295 65
296 54
297 93
298 11

Lookup value in A2: 293D106X

=VLOOKUP(--LEFT(A2,3),F2:G12,2,0)

However, if some of the part numbers have leading 0s this will not work!
