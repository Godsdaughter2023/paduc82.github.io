mail merged text appearing in bold when i only need some parts of it - Microsoft Community

alexmgzz say:

mail merged text appearing in bold when i only need some parts of it in bold

I have a text with some parts in bold on a cell from an Excel data source, but when i merge it in Word all the text appears in bold, how can i keep the original text formatting or can someone tell me if i can use a code to help me accomplish it.

Any suggestions are well received.

a. user say:

There is no straightforward way to preserve this kind of formatting when using an Excel data source.

One thing you can try, if you do not have more than the maximum number of columns for a Word table, is as follows:

 a. copy/paste your Excel data source with into a Word document, so you have a Word table. The bold formatting should be preserved. Save the document.

 b. use that document as the data source

 c. If the column containing the formatted text is called "mycolumn", you would normally insert it using a mergefield like this

{ MERGEFIELD mycolumn }

(use alt-F9 to reveal the field codes as necessary).

 d. delete the MERGEFIELD so you have a bookmark field:

{ mycolumn }

 e. If you update the field using F9, you may see a message saying that the bookmark cannot be found. Ignore that and merge to a new document (say).

 f. See if that gives you the result you need. It may not, because this is an old, unsupported merge feature and only some formatting features are preserved.

If your formatted text is formatted in a very consistent way there are a couple of other things you can try, but in essence you are reimposing formatting rather than preserving it

1. Say your text always has 3 words, and the middle one is always bold. Say the first cell has "thecat sat". Then the following may work:

 a. keep Excel as the data source

 b. in your Mail Merge Main Document, ensure that your MERGEFIELD has a \*Mergeformat switch, e.g.

{ MERGEFIELD mycolumn \*Mergeformat }

 c. Preview the merge results so that you see 

the cat sat

 d. in the Word document, format "cat" as bold

 e. Merge to a new document and see if that works. FWIW \*Mergeformat is also a rather old and not well documented feature, and will not do its job once the word count in the cell gets too large, or if the word count changes, and so on.

2. If your text always has the same formatting, e.g. as in the above example, or say you have a code which is always XXXNNNN, say ABC1234 or XYZ2345 where the XXX part is formatted bold, then split the text into separate columns, use separate MERGEFIELD fields to insert them in Word, and format those fields how you need.

If none of those looks likely to work, you'd probably have to have a VBA-based solution that copies/pastes the cell content into Word.

 

Peter Jamieson

Please try these options one by one and check after each option.

1. Try repairing your office installation (Quick repair is good enough). Try repairing your Office installation. First try Quick Repair (takes 5 - 10 mins) , if that doesn't help then try Online Repair (Takes more than 30 mins but less than 1 hour in general)

https://support.office.com/en-us/article/Repair...

2. Sometimes, add-ins can cause problems. To determine, if add-in is a problem start your application in safe mode and see if problem has gone away.

To start in Safe Mode -

Hold CTRL key and click on application icon and don't release the CTRL key till it asks you for Safe mode confirmation.

If yes, then start application normally and disable add-ins one by one and start application again every time you disable an add-in to determine the culprit add-in.
To disable add-in > File > Options > Add-ins > Down below you have Manage: Com Add-in : Press Go > Untick the Add-ins to disable and tick to enable them

3. Many third party applications also cause the problem like Tuneup Utilities, Abby Finereader, Kaspersky & other Anti Viruses etc. If you want to know if some program is conflicting -

Boot your computer in Clean boot mode and then start the application. If it works successfully, then some program is conflicting with this.
Refer to the section - How to determine what is causing the problem after you do a clean boot.
Don't forget to do - Reset the computer to start normally after clean boot troubleshooting.
https://support.microsoft.com/en-us/help/929135...

4. Sometimes, this is caused due to application not being able to access the default printer. Close all office applications and change your default printer through Start > Devices and Printers to another printer. It may be some physical one or One Note or XPS or PDF or anything else. If this is the problem, then you will need to update your printer driver for default printer.

5. Login to Windows with a new user profile and see if problem is resolved.

Do let me know if you have any more question or require further help. If above didn't work, I will supply more troubleshooting steps to solve this problem.
