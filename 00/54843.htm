b) Type "ipconfig /all" (without the quotation marks) and press "Enter". This will display your current IP address and other connection information that allows you to talk to other computers on the network.

c) Type "ipconfig /release" and press "Enter" to release the current IP address. This essentially leaves your computer without an address, but that will be fixed in the next step. The command prompt should then indicate that the IP address is 0.0.0.0.

d) Type "ipconfig /renew" and press "Enter". This will get a new address from the DNS source on this network.

e) Close the Command Prompt.

    Run the same install from an elevated command prompt and check to see if the same custom action was successful.
