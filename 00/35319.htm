Hi Walter,

1.       What is the model number of the expansion driver?

2.       Are you able to use the expansion drives  on other computers?

 

Method 1:
You may check in services if the plug and play service is enabled.

a. Type services.msc in the start search box and press Enter.

b. Double-click on the service named Plug and Play.

c. Check the value in the Startup type the device needs to be Started and automatic.

 

Method 2:

You may also check in device manager if the external hard drive is listed there.

a. Connect the external hard drive to the computer.

b. To open device manager, type devmgmt.msc in start search box and hit Enter.

c. Check if the external hard drive is listed.

d. In device manager, expand the hard drive listed in the device list.

e. If the hard drive is listed, right click on it >click Properties.

f. In General, under device status check for the message. Post back if there is any error message.

 

Method 3:

Additionally, also run the “Hardware and Device troubleshooter” and check if the issue persists.

For more information, refer this link.

Open the Hardware and Devices troubleshooter
http://windows.microsoft.com/en-us/windows7/Open-the-Hardware-and-Devices-troubleshooter
