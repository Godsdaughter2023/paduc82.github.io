batman-oz say:

Here are some things you can try:

Network Diagnostic

  1. Being up the Windows menu (Windows key + X)
  2. Click on "Network connections" and make sure the window is maximised 
  3. At the top of the window, choose "Diagnose this Connection"
  4. That will run the network troubleshooter / diagnostic tool.
  5. If that finds any problems it will fix them, but you may have to restart to get the fixes to work.
  6. If it doesn't fix the problem, or doesn't find a problem try Troubleshooting

Troubleshooting

If the above didn't work, try this

  1. Similar to the above but it checks some other settings: click on start and type in "Troubleshooting". (Or access "Troubleshooting" from the control panel
  2. In the Troubleshooting box, choose "Connect to the Internet" (found under the heading "Network and Internet") 
  3. In the next box, click on "Advanced" and choose "Run as Administrator"
  4. Click "Next" and then "Troubleshoot my connection to the Internet"
  5. If it finds problems it will try to fix them - if you need to restart it will tell you.
  6. If it does not find any problems, or does not fix them, try the configuration suggestion below

Configuration

If the above didn't work try this: What we are trying to do is roll back to a previous driver in case a new driver has been installed during the update, which isn't compatible with your adapter.  It is relatively easy to try and if it doesn't work, it is easy to undo.

  1. Bring up the Windows Quick Access Menu (Windows key + X)
  2. Click on Network connections
  3. Right-click on the network connection you want to fix and choose "Properties"
  4. In the box that comes up, click on "Configure"
  5. In that dialogue box, choose the "Driver" tab at the top
  6. Write down the details of the Driver provider, Date and Version number
  7. If "Roll Back Driver" is available (not greyed out) click that, and go to step 12 below. However, it is likely that this will be greyed out and not available to use - in which case continue with the following steps.
  8. Choose "Update Driver"
  9. Of the two options that will present you, choose "Browse my computer for driver software"
  10. In the next dialogue box click on the link at the bottom "Let me choose from a list of drivers on my computer"
  11. It will provide you with a list of drivers already on your computer, including some duplicates. Choose one that is NOT the same as the one currently installed (I For example, if your driver date is July 2016 look for the most recent one before that date (not a really old one)
  12. let that install. It will take a minute or two to reset, during which you will lose Internet Connectivity. Once it is reset and it reconnects to the Internet, test to see if this has improved the situation.
  13. If it does not fix it, or makes it worse, try restarting your PC.
  14. If it doesn't work after the restart, go through steps 1-6 above, but at step 7 choose "Roll Back Driver" instead of "Update Driver", to reset the current driver.

That's all I can think of. Hope one of these suggestions helps.

Steve

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{4D36E965-E325-11CE-BFC1-08002BE10318}
Caution There can be many instances of the registry subkey that is mentioned in step 2. 5. In the right pane, right-click LowerFilters , and then click Delete .
6. Click Yes to confirm the removal of the LowerFilters registry entry.
7. Exit Registry Editor, and then restart the computer.
