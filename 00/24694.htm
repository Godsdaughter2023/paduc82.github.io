ken sheridan say: You don't say in what way it 'doesn't work'.  I assume you've tried using Chr(13) & Chr(10) in place of the vbCrLf constant, but a simpler solution would be to use the CanShrinkLines function published on-line by Microsoft.  You'll find an example of a modified version of this function, along with other concatenation functions, in the file Concat.zip in my public databases folder at:

https://skydrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

The 'address lines' button opens a report which illustrates the use of the function using data from Northwind.  You'd call it with:

DoCmd.SendObject acSendNoObject, "", "", strEmailAddress, "", "", , _
       "Hi here is a list of names" & Chr(13) & Chr(10) _
        & CanShrinkLines(Me.Name1+":",Me.Name2+":",Me.Name2+":")

Note the use of the + operator to force the propagation of any Nulls.  You can see how it works in the debug window:

From what I can gather, even if you say "no" to decline the system from remembering your password, the system registers the username you used and keeps it stored in its database so if you input that username again, it won't ask you if you want to remember it.

To your basic question, should you uninstall unneeded apps, yes it is a GREAT idea. The less software, especially unused software, you have installed, the less opportunities hackers have to access your machine.

As long as you have the Office installation file and Product Key, yes, I think that uninstalling all Office apps and re-installing only what you need is a reasonable idea too. Make sure you reboot the computer after finishing the uninstalls and BEFORE you start the installs.

Do another backup after you finish all your work so that have another restore point in case of future problems.
