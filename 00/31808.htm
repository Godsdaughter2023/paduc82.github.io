Try these basic troubleshooting steps first and check after each step and please update me. (Please note your data is stored on the cloud, hence you won't be losing data)

Microphone > Camera > Make sure Skype is toggle ON

Uninstall
Windows Key+X > Windows Powershell (Admin) > Copy the below, paste into Powershell > Enter

Get-AppxPackage -allusers Microsoft.SkypeApp | Remove-AppxPackage

Get-AppxPackage -allusers Microsoft.SkypeApp | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

Question About Windows 10 Trial

It will be about a month before I can buy a Windows 10 key but I have a working PC so I want to install the trial version of Windows 10. I found two sources that said that the trial is unlimited time but all personalization features are disabled and there is a watermark on the desktop background. Why doesn't the trail have an expiration period? Does Microsoft really not care if people keep the trial version instead of buying the license? What is up with this whole trial thing?

andre da costa say:

There is no trial version of Windows 10 Pro or Home, but you can perform a keyless install and use it generally with being able to personalize it. The vast majority of Settings options will be disabled until a product key is entered that can activate that particular edition.

Why doesn't it have an expiration? Microsoft is trying to reach a middle ground with customers; the company doesn't want to come across as penalizing customers. Instead, the company wants to help them become customers through education and acquiring a license.

Microsoft is concerned about customers running non-genuine or non-activated copies of Windows 10, but they are not going to be aggressive about it either.

The only official trial of Windows 10 is the 90 day Enterprise trial.

Editions such as Pro, Home or Education that you can download and install without activating are actually in a reduced functional mode.
