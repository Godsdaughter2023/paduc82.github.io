Please try these troubleshooting steps to prevent attachments from senders from being blocked.

  4. Under Preventing junk email, click Safe and blocked senders.
  5. Click Safe senders.

rich michaels - mvp (greatcirclelearning.com) say:
Hi Randy,

I will try to answer your questions the best I can. If I state something in error I am certain one of the regulars on the forum will correct me. :-)

Question 1 - I would delete them.  If there's a duplicate then keeping it, even though disabled, doesn't make sense to me.  With that said, I think you have to be very careful with what you delete.  Let me try to explain it and again, my "friends" :-) will surely correct me where I misspeak.

At a minimum there are three font libraries and more depending on how many user accounts are on the system.
  1. System Font Library - this is located in the <Local Drive Name>:System:Library:Fonts folder.  These are the critical fonts that the system needs. Do not touch or alter these fonts.
  2. Main Font Library - this is located in <Local Drive Name>:Library:Fonts and contains all fonts for all applications and users
  3. User Font Library - this is located in <Local Drive Name>:Users:<User Account>:Library:Fonts and contains the fonts that are loaded just for the logged on user.
Word uses the main font library.  The user font library supplements the main library.  If duplicate fonts exist, it is because the same font file, with extension, exists in both the Main and User font libraries.  If you delete a font from the main library and this font is the default font for your Normal template style and/or your default Theme, I am not so sure that Word will pick it up correctly from your User Font library if the font exists there instead.

I'm speculating that some of these problems with documents opening and nothing is being shown has to do with fonts missing (maybe disabled) in the Main Font library and the critical font is one needed by the Normal template's default style or the default Theme.

Question 2 - FontNuke is a free download and the site says that it was recently updated for Lion.  I've not used it so I can't say more than here is the link.  FontNuke

Question 3 - Probably ... Word only uses True Type fonts and if there on other types in your Main Font library then those won't appear in Word's listing of available fonts.

The system won't pick them up.
