“DRIVER_IRQL_NOT_LESS_OR_EQUAL” error means that the system attempted to access pageable memory that was too high. This has many possible causes, but the most common scenarios are:

• Faulty drivers

• Hardware issue

• Anti-virus software problem

• Overclocking

After analyzing the Dump file we got to know “ndis.sys” was causing the issue which refers to network driver, this issue is mainly caused by the driver error with respect to your NIC (Network card).

Please follow the below mentioned steps to uninstall network drivers from device manager

1: Click on start

2: Type device manager in startup search.

3: Once the device manager is up, navigate to network adapters.

4: Expand the network adapter.

5: Right click on the network driver whichever has been installed.

6: Click uninstall on the drop down list.

Note: Do restart the computer once the device driver is uninstalled and do not update or manually install the driver, let the computer start normally and install the basic driver.

Please let us know what is the result after using the computer with the basic driver and we will happy to assist you further.
