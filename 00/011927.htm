doug robbins - mvp office apps & services (word) say:

When a document is created from a template (by using File>New and selecting the template) the macros in the template are NOT transferred into the document, so it is fine for those documents to be saved in docx format.

 

Further, when a document, that was created from a template on one user's machine, is opened on another user's machine, Word will look for a template of the same name in the second user's User Templates folder or the Work Group Templates folder.  If the template exists in either of those locations, it will be attached to the document and the code and other features in the template will be available to the document.  If the template is not available on the second user's machine, the user's normal.dotm template will be attached to the document.

 

See the article "Distributing macros to other users” at:

http://www.word.mvps.org/FAQs/MacrosVBA/DistributeMacros.htm
