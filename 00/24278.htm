First, let us hope you didn't throw away your "empty" files.

 The invisible fonts might be due to an issue that has been around Mac version Office for ages. Usually, this symptom shows up in Excel but can affect Word.

  1. First, go to the location of one of the "invisible" files.
  2. Now select the file by double Clicking the file. 
  3. Next, invoke the get info Command (⌘) - I.
  4. Notice file size, if Zero (0) then you can Empty the file to trash,
    If, however, the File has a number in size other zero, then you have a viable file.
  5. Close GetInfo.

Now try the Following:

  1. Hold down the Command key (⌘).
  2. Now type A (Select All),
    The Background should appear as if you selected text.
  3. Now change the font to anything other than the font already selected.
  4. The text should now show up. 
  5. Now, quit Word and do not save any changes.

If text shows up Open Font Book and do as follows:

  1. Locate Font Book in Utilities – open.
  2. Next in left window Click All Fonts
  3. In right window click on a Font.
  4. Now  choose Select all by typing ⌘ - A.
  5. Go to File menu, locate Validate fonts
  6. Click to run Validate.
  7. In window that pops up select click item above the list and choose Errors and Warnings Only
  8. Let it run until completion.
  9. Write down Fonts that show up with a yellow Minor issues.
  10. Open from the pane with the font list each Font affected.
  11. On fonts that have notation off beside it – choose to remove file.
    (You have to give system password, same as you would when installing software).  
  12. Repeat will all remain fonts.
  13. Now rerun again validate fonts until There are no errors or warnings. 
  14. Close Font Book, restart Computer.

Now open Word and one of the affected document if you cleaned up all the bad or duplicate fonts. Then you document should appear as you originally typed it.
