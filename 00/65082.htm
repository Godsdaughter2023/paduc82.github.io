      Read the above guide:

      "Optimize Windows Vista for better performance"

      Delete programs you never use

      Many PC manufacturers pack their new computers with programs you didn’t order and might not want. These often include trial editions and limited edition versions of programs that software companies hope you will try, find useful, and then pay to upgrade to full versions or newer versions. If you decide you don’t want them, keeping the software on your computer might slow it down by using precious memory, disk space, and processing power.

      It’s a good idea to uninstall all the programs you don’t plan to use. This should include both manufacturer-installed software and software you installed yourself but don’t want anymore—especially utility programs designed to help manage and tune your computer’s hardware and software. Utility programs such as virus scanners, disk cleaners, and backup tools often run automatically at startup, quietly chugging along in the background where you can’t see them. Many people have no idea they are even running.

      Even if your PC is older, it might contain manufacturer-installed programs that you never noticed or have since forgotten about. It’s never too late to remove these and get rid of the clutter and wasted system resources. Maybe you thought you might use the software someday, but never did. Uninstall it and see if your PC runs faster.

      For instructions, see Uninstall or change a program.

      Note: If it has been a hard drive related issue, chances are that you may lose all your data, which further necessitates you to apply your previous backup.

      The above error can occur in various situations, some of which have been mentioned underneath:

       

      ·         The computer tries to boot from a non-bootable diskette or CD present in the drive.

      ·         Hard drive has not been correctly configured within CMOS.

      ·         Any of drvspace.bin, command.com, io.sys, or msdos.sys files is either missing or damaged.

      ·         Hard drive is corrupted- MBR and/or partition table has get damaged

       

      The solutions to the given problems are as below:

      Step 1: Make sure no diskette or CD is there in your computer when you are attempting to start your computer.

      Step 2: Setup the boot options correctly. The right boot sequence is- 'Floppy', 'Hard Drive', 'CD-ROM', and 'Network

      NOTE: Modifying BIOS/complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

       

      Step 3: Repair the master boot record by using the fixmbr command from the Windows XP Recovery Console.

      a.    Boot your computer with a Windows repair or install disc.

      b.     At command prompt, execute'/fixmbr' command without the quotes.

      lemp say:
      Your Dell should have a built-in capability to "restore to factory conditions" -- unless the previous owner deleted it.

      Turn off the computer.
      Turn it back on and check to see if you get a blue bar with www.dell.com just after the power is turned on.  As soon as you see the blue bar, press CTRL+F11 (that is press the CTRL key and the F11 key simultaneously).  Then follow the on-screen directions.

      Your Dell User's Guide is here:  http://downloads.dell.com/Manuals/all-products/esuprt_desktop/esuprt_dimension_desktops/dimension-5000_Owner%27s%20Manual_en-us.pdf?c=us&l=en&cs=04&s=bsd  See page 51 for details about restoring to factory conditions using "Dell PC Restore."

      If you do not see the blue bar (and if CTRL+F11 doesn't start the restore program), you may be able to persuade Dell to send you back-up CDs. See https://www.dell.com/support/diagnostics/us/en/19/nondiagnostichome?c=us&l=en&s=dhs

      Note that doing this will restore the computer to the way it was when it first left the factory -- including all of the Dell-supplied trial software.  Anything installed on the computer later -- including all personal data and any later-installed software -- will be deleted, so before doing this, you need to back up all your personal files and have the installation media for any software you will want to re-install (such as Microsoft Office).

      More than likely, you will end up with Windows XP with either no service packs or service pack 1 (depending on when the computer was manufactured), so you'll have to download and install later service packs and then go to Windows Update and get all of the post-service pack 3 updates.

      UNINSTALL all trial antivirus programs that get reinstalled by the restoration.  The trial period will have expired, and the presence of these programs may interfere with things later.  If a Norton (Symantec) product was installed, use https://support.norton.com/sp/en/us/home/current/solutions/kb20080710133834EN_EndUserProfile_en_us  If a McAfee product was installed, use http://service.mcafee.com/FAQDocument.aspx?id=TS101331

      If the restoration results in service pack 1 or 2, you can install service pack 3.  If the restoration results in no service packs, you have to install sp 2 before you can install sp 3.

      You may want to download these files and burn them to CDs before you start.

      The sp3 installer is here:  http://www.microsoft.com/en-us/download/details.aspx?id=24
      The sp2 installer is here:  http://www.microsoft.com/en-us/download/details.aspx?id=28

      Read this before installing sp3: http://msmvps.com/blogs/harrywaldron/archive/2008/05/08/windows-xp-sp3-read-all-prerequisites-for-a-successful-installation.aspx

      See this thread:  http://answers.microsoft.com/en-us/windows/forum/windows_xp-windows_install/reinstalled-win-xp-from-recovery-disk-now-i-can/830472da-1643-4c55-8eb5-76a08514fee1?msgId=71972678-fff3-43b0-91cf-4b5068f4ec83 (I suggest using Microsoft Security Essentials as your antivirus, at least initially). 

      The download link for IE 8 (see Step 6 of the linked thread) is here: http://www.microsoft.com/en-us/download/internet-explorer-8-details.aspx  After you install IE 8, manually update it using http://www.microsoft.com/en-us/download/details.aspx?id=41074  You will most likely have to do this manual update before Windows Update will work for you.  See Step 9 of the linked thread.

      If you run into problems, post back with details, including the complete text of any error message.

      Although I agree with ZigZag's advice, I realize that the above may seem a bit daunting.  And the previous owner may have installed software that you want to use but for which you don't have media to reinstall.  Or the Dell PC Restore program may not be available and/or Dell won't send you backup CDs.  If for these or any other reasons you don't want to reinstall Windows (and thus take a chance on The Lurking Horror that may be buried on your hard drive), post back and someone will try to help you with your specific issues.  For Skype, the simplest thing to do may be to completely uninstall Skype using Control Panel > Add/Remove Software and then download and install a fresh copy: http://www.skype.com/en/download-skype/skype-for-computer/.
