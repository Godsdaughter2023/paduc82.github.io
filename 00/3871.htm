petrib say: It is a WAP or web site problem. In the case of content that is not text opening in Notepad on the phone, it means that the server did not send the correct MIME type in the HTTP headers.

That is, when the phone's browser is not sent a MIME type it recognizes, it defaults to assuming the content is text and opens it in Notepad.

So, whomever runs/operates the WAP/web severs should configure the servers to send the correct MIME types for different kinds of content/files.

For .3gp files the correct MIME type is "video/3gpp"

How the server is configured depends on the server system.

Glossary: ;)

WAP = Wireless Application Protocol
MIME = Multipurpose Internet Mail Extension (it is used for more than email nowadays, as you can read above)
HTTP = HyperText Transport Protocol

Hi

Question: Can someone please tell me how can I apply a Skype user name instead of a Microsoft ID (e.g. live:xxx_[number])?

Answer: Customizing Skype username or Skype name is no longer possible since the migration of all Skype accounts to Microsoft account. This will enable to have a single credential to access all Microsoft products including Office, Xbox, and OneDrive.

As a workaround, what you can change or modify is the Display name. It can also be used to search an account as an alternative to Skype name, email address and phone number.

Question: Moreover, is it possible to unlink the Skype account to the current Microsoft ID now?

Answer: Unlinking of Skype account from Microsoft is not yet available. While it's not possible to unlink your Skype and Microsoft accounts, you can move your Skype account to a different Microsoft account. 

Follow these steps to move your Skype account.

  • Sign in to the Microsoft account that your Skype account is currently linked to.
  • Select the Your info tab.
  • Select Manage your sign-in email or phone number.
  • Next to your Skype name, select Move to another Microsoft account. 
  • Double-check that you know the correct Microsoft account you want to use, and then select Next to continue.
  • Enter the email or phone number of the Microsoft account you want to move to, and then select Next. 
  • Sign in to your account. 
  • You'll see a confirmation screen verifying that your Skype account has been moved, select OK.

Note: Skype accounts (e.g. live:xxx_[number]) created using email address or phone number cannot be moved.

Check this links for more information.

  • How do I change my Skype display name?
  • Can I move my Skype account to a different Microsoft account if it's already been linked?
