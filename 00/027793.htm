luc sanders say:

Bradex,

John has given you the picture option, here are some other thoughts. There are some differences between using a PPT table and a Word table. For instance applying a decimal tab in a cell in PPT requires you to use the ctrl+tab combination to align the number. In Word applying a decimal tab automatically aligns the number no need to tab first. So copying the ppt table to Word will have an impact on the formatting as the applied tab will take impact in the cell moving the number.

Second, using a table in PPT means you are in landscape, so copying it in Word will inevitably result in a table not fitting between the margins.

Furthermore, the font size is larger in PPT than in Word, 18 pt I believe. Which will also have an impact on formatting.

You could tackle the problem in Word if you want. If the only problem is the double spacing for instance, select your table and apply the no spacing style from the quick styles group on the Home tab. Or create your own table style in Word, save it in the template and apply it whenever it is needed. Custom table styles will appear on the Design tab, table styles group, click the more arrow, you will find your custom table styles right at the top.

Luc Sanders
MVP - PPT
