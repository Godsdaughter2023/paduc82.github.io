jay freedman say:

Starting from the macro posted by Greg Maxey, the following macro will insert all three fields separated by tab characters. (The default Footer style includes tab stops at the center and right margin.)

          

Sub FooterMacro()
Dim oRng As Word.Range
Dim i As Long

For i = 1 To 3
  Set oRng = ActiveDocument.Sections(1).Footers(i).Range
  With oRng
    .Fields.Add oRng, wdFieldFileName, "\p", False
    .InsertAfter vbTab
  End With

  Set oRng = ActiveDocument.Sections(1).Footers(i).Range
  With oRng
    .Collapse wdCollapseEnd
    .Fields.Add oRng, wdFieldLastSavedBy, , False
  End With

  Set oRng = ActiveDocument.Sections(1).Footers(i).Range
  With oRng
    .InsertAfter vbTab
    .Collapse wdCollapseEnd
    .Fields.Add oRng, wdFieldSaveDate, , False
  End With

  ActiveDocument.Sections(1).Footers(i).Range.Fields.Update
Next i
End Sub

graham mayor say:

You didn't say which two languages, but let's assume English UK and Greek (you can change the languages as required)

Create two character styles 'English Style' and 'Greek Style' with the appropriate attributes for your requirements, eached based on the underlying style. At the very least you will need to apply the proofing language, but you can assign fonts or whatever.

Create the following macro and assign it to an unused keyboard shortcut - I used ALT+G. Then you can switch between languages with a single keyboard shortcut. http://www.gmayor.com/installing_macro.htm  

Note this code as written will not work in the 64 bit version of Word, but you probably won't be using that.

Option Explicit
Private Declare Function ActivateKeyboardLayout Lib "user32.dll" _
                                                (ByVal myLanguage As Long, _
                                                 Flag As Boolean) As Long

'define your desired keyboardlanguage
'Get language codes from https://msdn.microsoft.com/en-us/goglobal/bb895996.aspx?f=255&MSPPError=-2147217396

Private Const GRK As Long = 1032        'Greek
Private Const ENG As Long = 2057        'English(UK)

Public Sub ToggleLanguage()
    If Selection.Style = "English Style" Then
        Call ActivateKeyboardLayout(GRK, 0)
        Selection.Style = "Greek Style"
    Else
        Call ActivateKeyboardLayout(ENG, 0)
        Selection.Style = "English Style"
    End If
lbl_Exit:
    Exit Sub
End Sub

The macro can be adapted to any pair of languages by changing the two language constants and the two character styles. You can format those styles however you want them and then the macro will toggle the keyboard layout and the style as appropriate.

I am not familiar with the use of the Arabic langauge (or other right to left languages) so cannot advise on how this will affect the results, but if Right to left languages are enabled, I think it should work..

Option Explicit
Private Declare Function ActivateKeyboardLayout Lib "user32.dll" _
                                                (ByVal myLanguage As Long, _
                                                 Flag As Boolean) As Long

'define your desired keyboardlanguage
'Get language codes from https://msdn.microsoft.com/en-us/goglobal/bb895996.aspx?f=255&MSPPError=-2147217396

Private Const ARA As Long = 1025        'Arabic Saudi Arabia
Private Const SPA As Long = 1034        'Spanish Traditional

Public Sub ToggleLanguage()
    If Selection.Style = "Spanish Style" Then
        Call ActivateKeyboardLayout(ARA, 0)
        Selection.Style = "Arabic Style"
        Selection.RtlPara
        Selection.RtlRun
    Else
        Call ActivateKeyboardLayout(SPA, 0)
        Selection.Style = "Spanish Style"
        Selection.LtrPara
        Selection.LtrRun
    End If
lbl_Exit:
    Exit Sub
End Sub

The code is old style and not compatible with Word 2016 (I've made some tests and found that it doesn't work on my machines in all cases).

Andreas. 

Option Explicit
Dim WithEvents App As Application
Private Sub Document_Open()
  Set App = Application
End Sub
Private Sub App_DocumentOpen(ByVal Doc As Document)
  ForceDraftView Doc.ActiveWindow
End Sub
Private Sub App_NewDocument(ByVal Doc As Document)
  ForceDraftView Doc.ActiveWindow
End Sub
Private Sub App_WindowActivate(ByVal Doc As Document, ByVal Wn As Window)
  ForceDraftView Wn
End Sub

  W.View.Draft = True
End Sub
