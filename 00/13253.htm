I figured out a rather convoluted way to do this assuming you are adding page numbers to a booklet publication.

 

1. Go to your 2nd and 3rd pages in your booklet, or in other words your first inside pages.

2. Insert tab > Page number > drop down arrow > bottom right. This puts the page number on the outside margin on page 3 as you want it but on the inside margin of page 2.

3. Insert tab > footer > and select the footer on page two. Select the page number > Ctrl + L and the page number will be moved to the left, or outside margin. This seems to set the format for the entire booklet.

dan at it associates say:

Office 2010 has this quirk where it really does not like removing one of the installed component programs after the initial installation of the Suite.

 

See this article for the approved way to uninstall one of the programs:  http://office.microsoft.com/en-us/excel-help/install-or-remove-individual-office-programs-and-components-HA010354261.aspx  Essentially, to do so you should uninstall the whole thing, then reinstall using the custom method to leave out the now-unwanted program or feature.

 

But that does not help you now.

 

So use the MS FixIt  http://support.microsoft.com/kb/290301#Method2  to uninstall all of Office.  Restart twice.  Then go to Windows Updates and ensure you have all critical and important updates, especially for .NET and MSXML.  Restart after the updates are done.

 

Now you are ready to install Office again, this time choosing Custom and leaving out the component programs you no longer want to use.  One complete, activate online, restart, then go back to Windows/Microsoft update to get it all updated.
