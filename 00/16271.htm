 In order to use Outlook.com with IMAP, the following settings must be used during the setup process in the email client (you have to remove the quotes):

For incoming connections:

  • Server address: "imap-mail.outlook.com"
  • Server port: "993"
  • Encryption type: "SSL"

For outgoing connections:

  • Server port: "587"
  • Encryption type: "TLS" ("SSL" can be used if TLS is not an option)

Here's how to create an application-specific password:

1. Open the Outlook.com Account Settings.

2. Open Edit Security info.

3. Under App passwords, open Create a new app password.

Outlook.com will then automatically create a new application-specific password for you to use to log in to your account. Remember to heed the aforementioned advice.

ControlSource:   WeaponID (for a bound control; leave blank if the combo box is an unbound 'navigational' control)

BoundColumn:   1
ColumnCount:    2
ColumnWidths:  0cm

If your units of measurement are imperial rather than metric Access will automatically convert the unit of the last one to inches.  The important thing is that the dimension is zero to hide the first column.

How to remove little blue indictaors next to text boxes ? - Microsoft Community

paulfourcade say:

How to remove little blue indictaors next to text boxes ?

When formatting text boxes in publishing layout view I see blue indicators appearing below. How to remove them ?
bob jones aka: cybertaz say:
Without a more exact description I can only guess what you mean by  "blue indicators". If they look like a small rectangle on the bottom edge of the text box containing A... they are overflow text warnings. You can't simply remove them because they show that there is additional text contained within the text box which is not displayed in the document. You need to make 1 or more of the following adjustments;
  • Increase the size of the text box
  • Use a smaller font size
  • Link the Text Box to another to display the overflow there
  • Eliminate any empty ¶s that may have been used for 'spacing' purposes
  • Reduce the line spacing

Often the cause simply is that the return key has been pressed unnecessarily at the end of the last paragraph of text. If you delete that last, needless empty ¶ it may be enough.

If the indicators are on the lower right side & look like a triangle [▶] it means that the box already is linked to another. Clicking the triangle jumps you to the next box in the chain.
