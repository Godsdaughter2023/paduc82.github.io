carolina calling say:

It sounds like your disk isn't formatted.

From an Administrator account, run Start->Control Panel:Administrative Tools->Computer Management.
Once the Computer Management Windows opens, select Storage->Disk Management in the left pane.
Once you see the Disk Management pane on the right.  Expand the window to full screen so you can see what you're doing.

In the upper panel, right click on F: (or wherever your terabyte disk appears).  Select format, answer "yes" and proceed from there...

Do not do a "quick" format.  It sound like you need to build the basic block structure of the disk.

You can read about the "Disk Management" tool in help.  Click on the Computer Management "Help" menu, select "Help Topics". 
When the windows opens, click on the "Diskmanagement" + symbol, click on "How to ..."+ symbol,  click on "Format a basic volume". 
There's a lot of infomation and ideas to absorb.

john vil say:

Hi,

If you would like to reinstall the Windows Store or any other pre-installed apps, you can do so by executing a command in PowerShell. Kindly follow the steps below:

  1. Type Windows PowerShell on Cortana's search field, right-click on it and select Run as administrator.
  2. In the PowerShell prompt, type the following command and press Enter key.
    Get-Appxpackage –Allusers
  3. Scroll down and locate the Windows Store entry. Copy the package name of Windows Store and execute the following command:
    Add-AppxPackage -register “C:\Program Files\WindowsApps\<PackageFullName>” –DisableDevelopmentMode

Note: In the above command, replace PackageFullName with the package name of the Windows Store and replace C with the drive letter of the driver where Windows 10 is installed.

Let us know if you will need further assistance

====================== Keyless Install ================================

How to perform a 'Keyless Install' of Windows:


Boot from the Windows 7 DVD disc you created and start the installation process.


After you enter user name and password you will be asked to: 'Type your Windows product key', do not enter a key.
Also located just below the box where you would enter the key is a box with a check mark to 'Automatically activate Windows when I am online', remove the check mark and click 'Next'.


You will now have a 30 day grace period in which to activate Windows.


During this 30 day grace period you can try resolving your problems. 
After you are satisfied all problems have been resovled then activate Windows.
