Placing a hard drive with windows operating system installed (or cloning a windows image) from one pc to another with different hardware - motherboard, cpu, etc is usually bound to fail whether you are using windows 7 or 8 or 10.

daniel g. grau say:
This is a follow-up to Phillip's instructions.

You can expect a lot of confusion, because it's very difficult to select various disjoint elements, unless you've planned the layout very carefully. This is also the case with the built-in templates (by the way, they can be accessed through File > New from template), some of which are almost impossible to handle.

What I would do is this (I don't have Word 2008 but Word 2011, so part of the following instructions may differ):

In the main menu bar, select Tools > Labels and choose one of the 3.5 x 2 Avery templates. This will create a fixed size table. (The idea is to build one card within a cell, and then copy it elsewhere.)

The secret to an orderly build is to always begin any operation with your cursor placed within the same cell. In this way, any text, textboxes or pictures you place will be anchored to that first cell.

Work at a higher zoom factor (View > Zoom…). It will help if you enable the various options in the Ribbon's Layout > Grid section.

Pictures can be placed "In line" (in which case they behave just like an oversize letter within the text flow) or "floating" (any of the other options). This placement is controlled by Right-Clicking (or Control-Clicking) on the picture, then choosing Wrap Text or Format Picture > Layout. This contextual menu will also let you send the picture to the back (Arrange), etc.

There is an important setting to allow overlap: Right-Click, choose Format picture (or Format shape for text boxes), then select Layout  > Advanced and enable "Allow overlap."

Once you have all your elements within the first cell, click on the cell, do Table > Select Cell, Copy and Paste into the other cells.
