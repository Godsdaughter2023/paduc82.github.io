You can't "search" without using "Find."  However, there are lots of ways to determine if something is repeated.  Until you provide full details of the issue, all anyone can do for you now is guess.

 

If you have a Word document full of address labels, then most likely those labels consist of cells in a word table.  If the first line is of each lable defines the "names" the you could flag duplicates usig:

Sub ScratchMacro()
Dim oCell As Word.Cell
Dim oCol As New Collection
Dim strNames As String
For Each oCell In ActiveDocument.Tables(1).Range.Cells
  On Error GoTo Err_Dup
  oCol.Add oCell.Range.Paragraphs(1).Range.Text, oCell.Range.Paragraphs(1).Range.Text
Err_ReEntry:
Next oCell
Exit Sub
Err_Dup:
  oCell.Range.Paragraphs(1).Range.HighlightColorIndex = wdBrightGreen
  Resume Err_ReEntry
End Sub
