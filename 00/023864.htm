The problem is two-fold: the individual audit shows #DIV errors for the average when there are no scores and #NUM errors for the MEDIAN for the same reason.  If I try to suppress this by telling excel to read a blank as a zero, I will get a skewed number in the monthly spreadsheet which makes it useless as a monitoring tool for my boss.  Is there anyway to get excel to just ignore the work sheets that have no data.  In the winter months, we sometimes have employees who have a months vacation so the DIV/NUM errors will remain right to the end of the month.  This doesn't stay static and I can disable the link for the month but this is a lot of constant fussing that must be avoidable.

bernie deitrick say:

If you want to average a range and not return an error when the range is blank, use a formula like

=IF(ISERROR(AVERAGE(D2:D14)),"",AVERAGE(D2:D14))

If you want to average a range than may include errors from other calculations, array enter (enter using Ctrl-Shift-Enter) a formula like
=AVERAGE(IF(NOT(ISERROR(D15:K15)),D15:K15))
