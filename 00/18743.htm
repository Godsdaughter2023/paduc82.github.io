If the outlook process (outlook.exe) has several instances in Task Manager processes tab then your outlook installation, or its add-ons are at fault. This is not a win problem, but an Outlook problem.

Uninstall/reinstall of outlook is not likely to rectify an outlook problem, since it uses the origonal outlook profile.

Your shortcut solution, is no solution.

If you have set outlook to run on PC startup in the first instance remove that shortcut, then haveing rebooted run Office diagnostics. If the mutiple instances of outlook continues, reboot, before starting outlook useing the safe switch, and see if the problem is replicated.
