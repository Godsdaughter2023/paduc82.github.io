Windows 10 Technical Preview Install from USB stick.

On my second attempt I used a program called "Rufus" to create a bootable USB stick & had the same scenario as my first attempt.

After a bit of research, someone suggested using "WinRAR" to extract the files from the ISO file to the USB stick to create a bootable USB stick, end result was still not able to get it to boot from the USB stick, no matter what I tried.

By this time I was getting frustrated with it, but with perseverance & a bit more research I found the cure, what the problem was with creating a bootable USB using the windows 7 USB tool or Rufus was that these two programs created the bootable USB stick using NTFS file system & for some unknown reason my PC wouldn't recognise the USB stick.

The cure was to format the USB stick using the FAT32 file system & then using "WinRAR" to extract the file from the ISO to the USB stick, once that was done & upon restarting the PC & selecting change boot order to boot from USB it worked like a charm & bingo I was able to install Windows 10 Technical Preview with no problems at all. 

I hope this is of help to anyone who has trouble getting their PC to boot from a USB to install Windows 10 Technical Preview.

PS. I did a clean install rather than an upgrade.

=VLOOKUP($A18,INDIRECT($B17&"!$A$3:$D$136"),E$5,FALSE)

=VLOOKUP($A18,INDIRECT("'"&$B17&"'!$A$3:$D$136"),E$5,FALSE)
