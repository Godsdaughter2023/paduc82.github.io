When you configure Outlook.com account in Outlook 2013 only the PST/OST file for Outlook.com account will synchronize with the webmail. If you have created the task in any other PST file then it will not be synchronized with the Outlook.com account in web-mail.

If you want the task to be automatically synched with the webmail when you create them, then either you have to go to Outlook.com account configured in Outlook and create the task or set the PST/OST file for Outlook.com account configured in Outlook as default so the tasks will be automatically saved in the Outlook.com account in Outlook application and hence synchronizes with the webmail.

Follow the steps mentioned below to set PST/OST file as default for Outlook.com account:

  • Open Outlook.
  • Click on File< Account Settings< Data Files tab.
  • Highlight outlook.com account and click on Set as Default.
  • Close and open Outlook and check the result.

If you want to move the tasks which are present in the other PST file to be synced with Outlook.com account then you may drag and drop the tasks from the old PST file to Outlook.com account data file.

Dim i As Integer
Dim bProtected As Boolean

Dim oFld As FormFields

sNewRow = InputBox("Insert New Row", "New Row", "No")

In the following macro, the formfield in the last cell of what was previously the last row of the table is set to have no macro run on exit from it so that if the user goes back to that formfield and then exits from it, nothing will happen.

 

Sub Addrow()
'
' Macro created 02/02/03 by Doug Robbins
' To add a new row to a table containing formfields in every column
' automatically on exit from the last cell in the present last row of the table
Dim rownum As Integer, i As Integer
Dim newrow As Row
With ActiveDocument
    .Unprotect
    Set newrow = Selection.Tables(1).Rows.Add
    For i = 1 To Selection.Tables(1).Columns.Count
        .FormFields.Add range:=newrow.Cells(i).range, Type:=wdFieldFormTextInput
    Next i
End With
With Selection.Tables(1)
    newrow.Cells(.Columns.Count).range.FormFields(1).ExitMacro = "Addrow"
    .Cell(.Rows.Count - 1, .Columns.Count).range.FormFields(1).ExitMacro = ""
End With
newrow.Cells(1).range.FormFields(1).Select
ActiveDocument.Protect Type:=wdAllowOnlyFormFields, NoReset:=True

End Sub
