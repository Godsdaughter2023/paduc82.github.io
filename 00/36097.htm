a. user say:

It might seem that way, but all sfc /scan<anything> does is make an effort to populate the dllcache folder with backup copies of critical system files.  If anything is missing, SFC will want to replace it and prompt you to insert your genuine bootable XP installation CD that needs to be the same SP as your installed SP (or it is going to complain - a lot).  Then if one of those 3498 files ever comes up missing on your system, Windows File Protection will make an effort to replace it with the backup copy from the dllcache folder.
