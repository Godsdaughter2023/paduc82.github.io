Yes, you're right about using Value to return multiple rows.

 

I discovered in my searching that the following will allow you to get essentially the same result (in this case, at least).

 


Dim fld As DAO.Field2
Dim rs2 As DAO.Recordset
bodies = ""
Set fld = rs.Fields("Body Responsible")

     Set rs2 = fld.Value

Do While Not rs2.EOF

     rs2.MoveNext
Loop
rs2.Close

calculate the length of an Arc - Microsoft Community

lily az say:

calculate the length of an Arc

We curve tracks to our customer's requirements.  They provide the height and width of the arc and we need to determine the length of the Arc.  There are several tools on the web that will do the math, but I would like to see if anyone is aware of an access Function that would do the same thing.  It would be great to incorporate it into our data base. 

 

Somehow I suspect it might be more complex than I can handle... but I thought I would ask anyhow.

hansv mvp say:

I'd create a custom VBA function for this:

 

Function ArcLength(ByVal Width As Variant, ByVal Height As Variant) As Variant
    Dim Radius As Double
    Dim Angle As Double
    If IsNull(Width) Or IsNull(Height) Then
        ArcLength = Null
    Else
        Width = Val(Width)
        Height = Val(Height)
        If Width <= 0 Or Height < 0 Then
            ArcLength = Null
        ElseIf Height = 0 Then
            ArcLength = Width
        ElseIf Width = 2 * Height Then
            ArcLength = Atn(1) * 2 * Width
        Else
            Radius = (Width ^ 2 / 4 + Height ^ 2) / (2 * Height)
            Angle = Atn(Width / (2 * (Radius - Height))) * 2
            If Angle < 0 Then
                Angle = Atn(1) * 8 + Angle
            End If
            ArcLength = Radius * Angle
        End If
    End If
End Function

 

You can then use

 

Length: ArcLength([WidthField], [HeightField])

 

in a query, or

 

=ArcLength([WidthField], [HeightField])

 

in the Control Source of a text box, where WidthField and HeightField are the names of the fields (or controls) containing the width and height of the arc.
