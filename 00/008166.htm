Repeating Number Series on Labels - Microsoft Community

eileen gilbert say:

Repeating Number Series on Labels

I need to create a series of numbered labels on an Avery 5167 labels with the same number repeating three times.
jeeped say:
I need to create a series of numbered labels on an Avery 5167 labels with the same number repeating three times.

This is a little bit difficult because you want to be repeating three times on the Avery 5167 labels which are ½in × 1¾in. This would mean that there are 4 columns × 20 rows of labels which is not directly divisible by 3 so a little extra work is required.
  1. First set up a page with a  table of 4@ 1¾in columns and 20@ ½in rows or use a template based upon the Avery 5167 label format.
  2. Put the blinking cursor in the top left cell of the table and press Ctrl+F9. This will open a set of field braces. Type the following into the braces,
                       { SEQ nmbrs \R 1 \# "0000" }
          ... which will start a sequence called nmbrs at 1 and format the numbers with leading zeroes. You can change that to start at a different number and/or change the number format.
  3. Tap Tab so the blinking cursor is in the next cell to the right and press Ctrl+F9 for a new set of field braces and type the following into the field braces,
                       { SEQ nmbrs \C \# "0000" }
  4. Tap Tab to get the next cell and repeat the previous step.
  5. Tap Tab so the blinking cursor is in the next cell to the right and press Ctrl+F9 for a new set of field braces and type the following into the field braces,
                       { SEQ nmbrs \N \# "0000" }
  6. You now have sequence start, sequence continue and a sequence next fields. Repeat steps 3, 4 and 5 until you fill the table. Hint: if you use Tab or Shift+Tab to navigate into a cell already containing a field you can copy then use Tab to move forward and Paste.

                      
If you do not need to start at a number other than 1 you can use the field from step 5 in step 2.
When you are done press Ctrl+A (e.g. Select All) then F9 (e.g. Update Fields).

                            

If the field code are still showing, tap Alt+F9 to Show field values instead of Show field codes.

An alternative method would be to set up a Print Merge with an Excel worksheet that uses a formula like,
=INT(ROW()/3)+1
... to populate a large number of rows. In that case the contents of the table's cells would be something like.
{ MERGEFIELD nmbrs }  { MERGEFIELD nmbrs }  { MERGEFIELD nmbrs } { NEXT } { MERGEFIELD nmbrs } ... etc.

More information on the SEQ field at,
Field codes: Seq (Sequence field)
