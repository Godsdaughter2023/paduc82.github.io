gegjr say: Well I finally spoke with a MS Rep who helped me. As it turns out Outlook 365 (2013) requires 2 Data .pst files. One is in the "C:\Users\username\AppData\Local\Microsoft\Outlook\emailaddress.ost" folder and the other is in the "C:\Users\username\Documents\Outlook Files\________ Data File(1).pst". Without the latter .ics files cannot be read by Outlook. Apparently one data file is for Outlook and the other is for Outlook.com. So if you only have the one for Outlook.com then Outlook.com does not recognize the .ics file.
