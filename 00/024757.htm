Word Doc with OCR'd text; how to control line breaks - Microsoft Community

frankbriggs_622 say:

Word Doc with OCR'd text; how to control line breaks

I have a long Word doc created from OCR'd text (scanned from type written pages). Problem is that my line breaks are the same as they were in the original type written document. I want to make the paragraphs fit (width-wise) with the margins I have set up in Word.  I know I can do this by adjusting each line - but I have 130 pages!  

Does someone know a function within Word with which I can "undo" the fixed line breaks that I imported? Or, do you have any other suggestion (that would be quicker than line-by-line adjustment?  Could I have done something in the raw text document (produced when I OCR'd the original) that would have "undone" the line breaks?

I am using Office 2016 version of Word (on a Mac Book).

Frank

*** Email address is removed for privacy ***

john korchok say:

Find and Replace is the preferred tool for this job. The first task is to distinguish the line breaks that you want to keep, like the ones that end paragraphs. If you are lucky enough to have a single return at the end of end line and a double return at the end of a paragraph, the job is simple:

  1. In Word, choose Edit>Find>Advanced Find and Replace.
  2. Select the Replace tab.
  3. In the Find what: field, enter ^p^p (code for a pair of carriage returns). In Replace with:, type a character not normally used in text (I like the pipe character, a straight vertical line: |). Click on Replace All. All double returns are replaced with the placeholder character.
  4. Now change Find what: to a single ^p. In Replace with:, delete the characters and type a space. Click on Replace All. All single returns are replaced with space characters.
  5. Finally, change Find what: to the character you used to replace the double returns. Change Replace with: to ^p. Click on Replace All. The placeholders characters are replaced with single returns and your text fits the new margins.

If you only have single returns between all lines, you'll probably have to go through the document and add extra returns to the paragraph ends, then do the Find and Replace operation.
