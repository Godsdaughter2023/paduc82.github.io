If the contacts are Imported to Outlook.com from Gmail.com directly using Gmail.com API then these contacts will NOT appear in CSV file downloaded from People app. Whereas, if you have stored them on People app after downloading a CSV from Gmail and then uploading it to People app, the contacts will appear on CSV file downloaded from People app.
