This problem may occur if there is some other wireless network management software installed on the computer. For example, if Intel PROSet Wireless is installed.

In this case, to delete a wireless profile, click Start->All Programs->Intel PROSet Wireless->WiFi Connection Utility, Click Profile button and manage the profile here.

set devmgr_show_nonpresent_devices=1
start devmgmt.msc

In this Special Device Manager Window; on the menu, click View then Show Hidden Devices

The unwanted devices will appear dimmed. You can right-click on them from there and uninstall them.

Then go back to  your network center and rename your adapters to what you want without the # sign.
