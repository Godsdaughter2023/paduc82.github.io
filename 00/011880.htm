Word 2010 Tables Header Row Inconsistent - Microsoft Community

john.reilly.1954 say:

Word 2010 Tables Header Row Inconsistent

I was looking at repeating headers at the top of each page and discovered an inconsistency in the way the information is proesented.

 

I have created a number of tables which may split across pages as content is added.  Thus I was setting the headers options in Word.

 

  1. Put cursor in top row of the table, go to "Layout" in the Bar (or whatever it is called) for the table.  In the Data Section, the "Repeat Header Rows" option is highlighted (button depressed)
  2. Right click in the first row of the table, select "Table Properties" and go to the "Row" tab in the properties form.  "Repeat as Header Row at the top of each page" option is ticked.
  3. In the "Design" tab got to the "Table Styles" Option.  From drop down list (at bottom) select the table style option, which brings up the "Modify Style" form.  On the bottom right corner there is a button called "Format", which allows you to choose "Table Properties" form (as in 2 above".  Lo and behold the table format and layout is different.

Nothing in item 3 indicates that you are working on a different table to the one selected, but I suspect that you are.  Has anyone else noticed this and the product group should know that this is the case....

pamcaswell say: The Table Style Options are the six check boxes in leftmost section of the  Table Tools-Design  tab.  These options, which may be set in table styles,  are turned on or off in individual document tables here.  When you entered the Table Styles group and clicked modify, you opened a different table, probably Table Grid, the table style for the table you created. But the changes you make in your table are direct formatting and do not change the table style.

Note that for the header row to take effect in the table, the Table Style Option for header row must be checked.

Click Continue
