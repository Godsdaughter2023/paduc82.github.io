    Hi Philip,

    Errors in saving Office files may be due to system file corruptions. To resolve this, we suggest running the System File Checker (SFC) tool. The SFC tool is a utility that scans the intergrity of the system files and replaces the corrupted or damaged system files.

    To run the System File Checker, you can follow these steps:

    1. Press Windows key + X.
    2. Click on Command Prompt (Admin).
    3. Type sfc /scannow then hit Enter.

    Don't close the Command Prompt window until the verification is 100% complete. The scan results will be shown after the process is finished.
