How to fix corrupted Outlook database

The following dummy steps work for me‚Ä¶

Step 1: 

Backup the whole Main Profile folder or at least these 3 files: Outlook.sqlite, Outlook.sqlite-shm, Outlook.sqlite-wal. If something went wrong, it‚Äôs always better to have backup.

Step 2:

and use following commands to export data from corrupted database to dumb.sql file:

sqlite> .mode insert
sqlite> .output dump.sql
sqlite> .dump
sqlite> .exit

Step 4:
