You need very little running at startup for any computer and what *you* need will differ from what anyone else *needs*.

Go through the list of things you have running at system startup/user logon using "AutoRuns" and then configure (or remove if you wish) the applications you don't want to start up accordingly using their own configuration settings (best) or by seeing where their startup request is located (registry, startup folder, etc) and removing it (last resort) or - if you don't need the startup/application at all - remove it through Programs and Features.
