brian tillman [outlook mvp 2007-2019] say:

Your default folders are controlled by the defaultdata file specified on the Data Files tab of File>Account Settings, not by what account is set as default.  I'd say you have the Exchange data store chosen as default.  If you have a PST listed on that tab, select it as the default, then start Outlook.  Copy the contents of the Exchange folders (which will not be your default any more) to the PST.  Once you've copied everything you can then delete the Exchange account.

Keep in mind that if you have any messages that appear to have been received via the Exchange account, Outlook will attempt to send by that account if you reply.  Once you have removed the account, you may  have to watch for that and make sure you select the default account.

diane poremsky mvp (slipstick.com) say:
Do you use the exchange account for anything - scheduling etc? If not, do as Brian suggested and move everything to a local pst and remove the exchange account. You could use the mailbox as the default delivery location, everything works as expected and it won't matter which account sends mail.
