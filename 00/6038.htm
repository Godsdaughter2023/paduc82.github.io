Summing up numbers in a column based on data in another column

Column A = date (3/16/10), column B = $ amount. I need for every DAY (ex. the 16th of the month) to add up the entries in column B that match that particular day (column A) of the month. Example...

Column A     Column B     Column C

3/16/10        $450

3/16/10        $200

3/16/10        $100            $750

3/17/10        $200           

3/17/10        $300            $500

The answer for the 16th would be $750, for the 17th it would be $500. Please HELP!!!!!!!

ed ferrero say:

Enter this formula in cell C6

=SUMIF(A:A,A6,B:B)

If the dates in column A are sorted, and you donly want to see one sum for each day in column C, try this formula

=IF(A7=A6,"",SUMIF(A:A,A6,B:B))

Ed Ferrero www.edferrero.com

ed ferrero say:

To make things easier, I would put

=DAY(A2) in col B and copy down

=MONTH(A2) in col C and copy down

Put 1 through 7 in F1:L1

Enter this formula in next row

=SUMIFS($D:$D,$B:$B,F1,$C:$C,1)

Where Column D is the amount you wish to sum,

Column B is Day No and is matched to value in F1

Column C is Month No and is matched to 1 (or enter a cell reference here).

Many Windows 10 computers no longer allow turning of graphic acceleration in the operating system visual interface. To tell if you computer has this ability, please follow these steps:

  1. Right-click on the Start button and choose Control Panel.
  2. Choose Hardware and Sound, then click on Display.
  3. In the left hand pane of the dialog, click on Change Display Settings.

If there was no troubleshooting tab, your computer may not support hardware acceleration, so there's nothing to turn off.

jllatham say:

Well, even though HansV's may (most likely will) do the job, since I wrote this, I'll post it just in case.  It may be that it is more understandable to you also which would mean you could more easily alter it in the future.  Just change the values of the CONST entries to match your two worksheets.

Sub TransposeAndExpand()
  'these constants refer to the original data sheet
  'change as required
  Const sourceWSName = "OriginalList" ' sheet's tab name
  Const srcStyleCol = "A"
  Const srcDescCol = "B"
  Const srcColorCol = "C"
  Const srcGenderCol = "D"
  Const srcFirstSizeCol = "E"
  Const srcLastSizeCol = "L"
  Const srcFirstDataRow = 2
  'these constants refer to the expanded data sheet
  Const destWSName = "RevisedList"
  Const destFirstDataRow = 2
  Const destStyleCol = "A"
  Const destDescCol = "B"
  Const destColorCol = "C"
  Const destGenderCol = "D"
  Const destSizeCol = "E"
  Const destQtyCol = "F"
  '
  'working variables
  Dim srcWS As Worksheet
  Dim srcList As Range
  Dim anySrcEntry As Range
  Dim currentStyle As String
  Dim currentDescription As String
  Dim currentColor As String
  Dim currentGender As String
  Dim destWS As Worksheet
  Dim rowPtr As Long
  Dim startCol As Integer
  Dim endCol As Integer
  Dim colPtr As Long
 
  Set srcWS = Worksheets(sourceWSName)
  Set srcList = srcWS.Range(srcStyleCol & srcFirstDataRow & ":" _
   & srcWS.Range(srcStyleCol & Rows.Count).End(xlUp).Address)
  startCol = Range(srcFirstSizeCol & 1).Column
  endCol = Range(srcLastSizeCol & 1).Column
  Set destWS = Worksheets(destWSName)
  For Each anySrcEntry In srcList ' look at each style entry
    currentStyle = anySrcEntry
    currentDescription = srcWS.Range(srcDescCol & anySrcEntry.Row)
    currentColor = srcWS.Range(srcColorCol & anySrcEntry.Row)
    currentGender = srcWS.Range(srcGenderCol & anySrcEntry.Row)
    For colPtr = startCol To endCol
      rowPtr = destWS.Range(destStyleCol & Rows.Count).End(xlUp).Row + 1
      destWS.Range(destStyleCol & rowPtr) = currentStyle
      destWS.Range(destDescCol & rowPtr) = currentDescription
      destWS.Range(destColorCol & rowPtr) = currentColor
      destWS.Range(destGenderCol & rowPtr) = currentGender
      'get the size 'name' from row 1
      destWS.Range(destSizeCol & rowPtr) = srcWS.Cells(1, colPtr)
      destWS.Range(destQtyCol & rowPtr) = srcWS.Cells(anySrcEntry.Row, colPtr)
    Next ' end colPtr loop to loop through sizes
  Next ' end anySrcEntry loop
  'housekeeping cleanup
  Set srcList = Nothing
  Set destWS = Nothing
  Set srcWS = Nothing
  MsgBox "Done"
 
End Sub

jay freedman say:
> Is there a way to make an image a MacroButton?

Yes. Just copy the image (from any graphics program or screenshot tool) and paste it into the MacroButton field in place of the text that would ordinarily be displayed:

   { macrobutton macroname [paste image here] }

You can also make the macrobutton's macro fire with a single click. Just create an AutoExec macro like this one in a module in your Normal.dot template

Sub AutoExec()
    Options.ButtonFieldClicks = 1
End Sub

or add that line to an AutoExec macro if you already have one.

Please don't try to use ActiveX buttons. A document with a few of them isn't too bad, but once you reach a dozen or so the document file will become huge and very slow to load.
