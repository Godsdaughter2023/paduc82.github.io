splitting 2 hotmail accounts - Microsoft Community

jlemond say:

splitting 2 hotmail accounts

we have  2 hotmail  addresses in 1 account, but now we want to split them.

miguelchiappori say: Hello,

I hope I can help you today

There are two solutions for this problem:

- Configure a filter to separate emails.

- Create a Profile.



Configure a filter to separate emails.

Objective.-

In this case, the purpose is to create a new message folder and configure a filter so that mails from one of the mail accounts are stored in it.

Configuration process.-

For this purpose, two tasks must be carried out:

- Create a new folder. It's simple:

We click with the right mouse button on "Personal folders" (in the left margin, where the Inbox, Exit, ...) hang
We select the option "New folder ..."
We give it a name (for example, Inbox 2)
Accept, and the new folder appears in our folder tree of Microsoft Outlook.

- Create filter. The steps are:

Tools> Rules and alerts ...> "New rule ..." button
The Assistant appears to create mail rules. We chose the option "Start from a blank rule".
Select "Check messages when they arrive" and click Next.
In step 1, we mark "through the specified account" and in step 2, we click on the "specified" link. We choose the account that we want to filter to separate the emails. Click on "Next"
In step 1, we mark "move to the specified folder" and in step 2, we click on the "specified" link and select the folder that we have created previously. Click on OK and Next.
In principle, we do not mark any type of "exception" and click on "Next".
In step 1, we can specify a name for this rule or leave the name by default. We make sure that the option "Activate this rule" is active and click "Finish" and "Accept".
We already have it, now just check if it works correctly. If it does not work, go through the steps of how to create the filter.

Advantages and disadvantages of this solution.-

Advantage:

It is easier to configure.
You do not have to change your session to read one and another account.
Disadvantages:

We separate the messages we receive, but the messages that are sent are not separated since they are all stored in the "Sent Items" tray.
When messages are sent, by default, they are always sent with the account configured as "default". If you want to send using the second account, you have to change it (in the button bar, click on "Accounts" and choose the other) before sending the message.


Create a Profile

Objective.-

Independent accesses are created to the Microsoft Outlook program, one for each email account. In each of these accesses we have our independent mail trays and our independent address book.

Configuration process.-

In the first place, if we already had the two accounts configured, we have to remove one of them since we are going to configure it in a new "profile", so:

Tools> Email accounts ...
We choose the option "View or change existing email accounts" and Next.
From the two accounts, we select the email account that we are going to delete and click on the "Remove" button. Confirm with a Yes. And click on the "Finish" button.
Next we will create the new "profile", following the following steps:

We close the Microsoft Outlook, and go to the Windows Start button and look for the "Control Panel"
Within the Control Panel, we choose "User Accounts" and click on the Mail icon (if we use the "classic view" of the Control Panel, then we double-click on the Mail icon).
Click on the "Show profiles ..." button
Click on "Add ..."
We give the profile a name (whichever is wanted) and Accept.
The Wizard to configure the mail account opens. If we do not remember how it is done, follow our help from point 2:
How do I configure my email account in Microsoft Outlook?
We have created the profile and configured the email account. We returned to the window where we had clicked Add. We mark the option "Request a profile" and Accept.
Management of the two email accounts.-

Now when we open the Microsoft Outlook we open an initial window where we have to choose the "profile" with which we want to access. Once selected and accepted, we can read and send mail, as well as perform all the tasks that we want with the email account that we have configured in that profile.
Unfortunately, in order to read the other account, there is no other choice but to close the Outlook, reopen it, and choose the other profile.

Advantages and disadvantages of this solution.-

Advantage:

We have a separate email account from the other, since we have different email folders and different address books.

Disadvantages:

The configuration and management of the accounts may be more complex.
Sometimes it is uncomfortable to have to be constantly closing and opening the Outlook to change from one profile to another

For the error message "There was a problem sending the command to the program", use the File Explorer or Windows Explorer to access the C:\Program Files\Microsoft Office\Office16 folder and right click on WINWORD.EXE then on Properties, go to the Compatibility tab and uncheck all the boxes. 

It has also been reported that the problem can be overcome by editing the registry at HKEY_CLASSES_ROOT\Word.Document.12\shell\Open and deleting the ddeexec key.

The ddeexec can also be deleted from

HKEY_CLASSES_ROOT\Word.DocumentMacroEnabled.12\shell\Open.

That fix is described at ...
