How do you convert a word doc to csv file format? - Microsoft Community

wilsonstick say:

How do you convert a word doc to csv file format?

I have a resume in a word doc. I need to upload in csv file format to potential employer site. How do I convert the file?
herb tyson say:

CSV means "comma separated variables," and typically is used for data files, such as address lists or other lists that are arranged as tables.  It's more common to see this request for Excel files. In a CSV file, if there are text elements that include space, those values typically are enclosed in quotes, such as:

"George Washington",703-555-1212

"Thomas Jefferson",703-555-1313

To convert a Word file (a table, most likely) to CSV (without resorting to Excel), would then require several steps, potentially. First, I would use Table Tools - Layout tab and the Convert to Text tool to convert the table to text, choosing Commas as the option for separating variables.

Next, I would use find/replace to insert quotes as needed.

Finally, I would choose Save As and save the file as a plain text file. Word uses .txt as the extension by default. If you enclose the whole name in quotes (e.g. "MyData.CSV"), then you'll get the .csv extension. (If you type MyData.CSV without the quotes, you'll get MyData.CSV.TXT). Either way, the extension can be tidied up in Windows Explorer, if need be.

An easier way would be to copy the table to Excel and then save directly from Excel in .csv format. Of course, you'd still need to deal with the quote issue if you have any text-containing-spaces issues.

Herb Tyson Microsoft Word MVP Herb Tyson Word Bible Blog

The macro recorder generally doesn't record anything regarding graphics, or mouse movements other than selecting text or running commands. It also can't create loops, or use Range objects instead of the Selection object. (More on this at http://www.word.mvps.org/FAQs/MacrosVBA/ModifyRecordedMacro.htm.)

To get a list of macros and their shortcuts (and all other non-default shortcuts you assign), go to File > Print. Open the first dropdown under "Settings" -- the one that usually says "Print All Pages" -- and click the "Key Assignments" item near the bottom. Then print.

If you open the macro editor (Alt+F8) and run the macro from there, the title bar of the editor will include the notation "[Running]" until the macro stops. Of course, that isn't something you want to do regularly. The better idea is to write the macro to show an indication on the document window. One simple way is to include a statement in the loop to write something on the status bar at the bottom, like StatusBar = "text to show". Usually, though, this causes flickering and is hard to read. A better approach is to program a progress bar: http://www.word.mvps.org/FAQs/Userforms/CreateAProgressBar.htm

You can pair your phone with a variety of Bluetooth accessories, such as headsets, car kits, or speakers. One of the most common types is a Bluetooth headset for making hands-free calls.
  1. On Start, flick left to the App list, and then tap Settings .
  2. On the Settings screen, tap Bluetooth. (By default, Bluetooth is set to Off.
  3. Turn Status  to On. This starts a search for any discoverable Bluetooth accessories.
  4. Make sure that the accessory you want to pair is turned on so that your phone can discover it.
  5. Tap the name of the accessory that you want to pair with your phone.
  6. If the accessory requires that you enter a PIN (often called a pairing code) to pair it with your phone, make sure that both the phone and the accessory display the same PIN before tapping Done.
