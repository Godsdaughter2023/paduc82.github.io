We need a helper cell in sheet August (and each other sheet) to perform a as fast as possible calculation:

foxfifth say: Text size on a web page can change if you accidentally touch the Ctrl key when scrolling with either the mouse or the touchpad.  If that is what happened, you can return it to the normal 100% size by holding down the Ctrl key and pressing the 0 (zero) key at the same time.

jeeped say:
web page text is too small to read - not all sites...

It sounds like you've modified the browser window's dynamic text size. In Internet Explorer, tap the Alt key and select View, Text Size to compensate. Some web sites are coded to over-ride this adjustment.
You may also get some gratification by adjusting the browser's zoom level. Press Ctrl+<plus> (e.g. +) together to zoom in and Ctrl+<minus> (e.g. -) together to zoom out. In Internet Explorer you cab also tap the Alt key and select View, Zoom. or use the mouse to select a zoom level in the lower right of the status bar.

... it is something to do with the pad when moving pointer...

You can also adjust the zoom level by holding down a Ctrl key and rotating the mouse scroll wheel forward and backward. You might be able to accomplish this with the Ctrl key and a touchpad's scroll area but I've never had much success with that method.
