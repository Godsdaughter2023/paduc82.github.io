danimaranelo say:

How to find all the combinations of a number?

Hi!

I would like to know if there's any formula that calculates all the possible combinations of a 4 digit number.

So in B2 I have the number 5164, I would like to find all the possible combinations in column D: 6541, 6514, 6451,6415, and so on. So that would give I think 24 possible combinations.

But if the number has repeated digits like 7522, then the possible combinations would be 12: 7522,7252, 7225

gary's student say: Place your value in A1 and run this macro and the combinations will be listed in A2 thru A25:

Sub MakeCombos()
    Range("A2:A25").Clear
    Dim v(1 To 4) As String
    For i = 1 To 4
        v(i) = Mid(Range("A1").Value, i, 1)
    Next i
    
    Dim col As Collection
    Set col = New Collection
    On Error Resume Next
    For i = 1 To 4
        For j = 1 To 4
            For k = 1 To 4
                For l = 1 To 4
                    If i = j Or i = k Or i = l Or j = k Or j = l Or k = l Then
                    Else
                        Vall = v(i) & v(j) & v(k) & v(l)
                        col.Add Vall, CStr(Vall)
                    End If
                Next l
            Next k
        Next j
    Next i
    
    For i = 1 To col.Count
        Cells(i + 1, "A").Value = col(i)
    Next i
End Sub
