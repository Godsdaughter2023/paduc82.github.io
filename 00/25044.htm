bob jones aka: cybertaz say:

What you've seen pertains to the Windows version of Word. Word for Mac does not implement the option in the same way & it cannot be set as the default.

To paste text only use the ▼ on the right edge of the Paste button on the Home tab of the Ribbon, then select the choice.

An alternative is to paste as usual, then click the Paste Options button that appears at the end of the pasted content & select Keep Text Only from there.

You also have the options of assigning a keyboard shortcut to the PasteTextOnly command as well as using Edit> Paste Special - Unformatted Text.

phillip m jones say:

It's obvious the first person to reply only use the Windows version in which remove and install of the Office package will eliminate any preferences, Registry keys, and other files. The Mac version You almost never remove the install. In fact, doing so is likely to cause issues. So about 98% of the time, such action is a waste of time.

Here is a list of things to check:

  1. Repair Permissions
    For versions up to Lion  X.9.x Open Disk Utility and run Repair Permissions. (run twice if necessary) this action will reveal any false positives.

    For versions, X.10.x and above, Apple took the advice of the "so called" Computer industry experts and removed the ability to manually repair permission. It is quite obvious the experts have little to no experience with Office Mac.

    To repair Permissions on OSX.10.x and above
    Close all any applications running, making sure to save any work beforehand.
    Next, restart Computer with the Shift Key down.
    Continue holding the shift key down when begins restarting for 3-5 seconds, after progress bar starts.
    Release shift key and allow progress bar to continue.
    When Computer asked for Password, supply the same one used to install software.
    Allow the computer to Finish starting up.

    Note: At some point in the very first Safe Mode restart on some computers, The screen will turn completely black as if the monitor has been disconnected. (It will depend on the individual computer. I suppose it has to do with the video card and driver.) Once this happens any other safe mode attempts this will not happen.

  2. Next, check Fonts for Duplicates and defective fonts.
    Follow directions at this website under the heading FontBook Lion and Above.

  3. Finally, Look for this file com.microsoft.Word.plist (without the underline).
    You have the option of either renaming extension to .old instead of .plist, or removing the file to the trash. The one for Word 2016 is in the  User Library within either the Container Folder or Group Container Folder.  The one in the application support folder either in user or main library preference folder is for 2011.

    Open Word and if it works correctly.

Aways Restart computer "normally", after trying each procedure. This way clears out any caches and you start fresh. 

Once you have tried one of the numbered steps and, it fixes the issue you do not have to continue. Do not shuffle order because, if the step didn't work you will have to back-track and most likely will have to redo the steps again.
