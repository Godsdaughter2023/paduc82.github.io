Hello David,

Welcome to Microsoft Community Forum.

I can understand the inconvenience caused to you. I will certainly help you out with this issue.

Error code: 1719, “The Windows Installer service could not be accessed, the Windows Installer is not correctly installed”.

This problem may occur if the Windows Installer files are damaged or missing. Other reason can be incorrect permission issue on the related registry keys that Windows Installer wants to access. I have also checked that "Pinnacle Studio 17.5" is compatible with Windows 7. Let’s work together to get this problem resolved.

However, before we proceed I would require some more information to assist you better.

1. Are you getting the error 1719 while installing any other software as well?

2. Do you remember making any changes to the system prior to this issue?

I would suggest you to try the following methods:

Method 1:

Check whether the Windows Installer service is set to a value of Disable.

  1. Click Start, type services.msc in the start search box and press enter.
  2. In the Services (Local) list, right-click Windows Installer, and then click Properties.
  3. If the Startup type drop-down list is set to a value of Disable, select the Manual option from the Startup type drop-down list.
  4. Click on Start button below the heading service status and click OK.

Method 2:

Re-register the Windows Installer.

  1. Click Start, open the run box (windows key +R)
  2. Type msiexec.exe /unreg and click ok.
  3. If you receive a confirmation message, click OK again.
  4. Click Start, open the run box (windows key +R)
  5. Type msiexec /regserver and click ok.
  6. If you receive a confirmation message, click OK again.
  7. Try to install the application and see if the issue persists.
  8. If you still get the error then try next method.

Method 3:

Download the latest version of Windows Installer and install it on your computer.

http://support.microsoft.com/kb/942288
