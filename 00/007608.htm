tom ogilvy say:

For the VBA Listbox (Forms), here is the equivalent code.   It would be assigned to the listbox

Sub ListBox1_Change()
    Dim intCount As Integer
    Dim ListBox1 As ListBox
    Dim s As String
    s = Application.Caller
    Set ListBox1 = ActiveSheet.ListBoxes(s)
    With ListBox1
        For I = 1 To .ListCount
            If .Selected(I) Then
                intCount = intCount + 1
            End If
        Next I
    End With
    Select Case intCount
    Case 0: Exit Sub
    Case 1:  MsgBox "1"
    Case 2, 3, 4: MsgBox "2"
    Case 5: MsgBox "3"
    End Select
End Sub

The problem in both Bernie and my adaptation of his code is that this fires each time the user makes a selecton.  So the challenge would be on when you want to make the determination that they are through selecting.  Of course you could update a cell each time it fires and eventually that cell would reflect the final score for the number of choices.   (or you could have a button that ran code similar to this [application.caller] would not reveal the listbox in that case.
