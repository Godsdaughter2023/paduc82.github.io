How to use sumif or sumifs to using to exact criteria?

Another issue is if the criteria is "0055" and there is "0055" and "0055A" in the criteria range, the sumif(s) only sums the values of "0055A".

amit tandon say:

SUMIF ignores the leading zeros for numbers formatted as text, so you may use SUMPRODUCT, as below:

=SUMPRODUCT((A2:A10="0055")*B2:130)

However, using "0055" will not include "0055A" and consider these 2 text separately because this is alphanumeric.

vijay a. verma say:
This is the limitation of SUMIF. You can use SUMPRODUCT for this where A2:A10 is the range to summed and B2:B10 has the criteria,

=SUMPRODUCT((B2:B10="0055")*(A2:A10))

You need to use an INCLUDEPICTURE field coded as:

{INCLUDEPICTURE "C:\\Filepath\\{MERGEFIELD PicName}.jpg"}

where

'C:\\Filepath\\' denotes the full path to the folder holding image files 

note the double '\\' for the folder separators, though you can use a single "/" instead.

'PicName' is your mailmerge field name that contains the filename of the picture
and '.jpg' is the filetype, which might also be '.gif', '.tif', etc. but that could also been contained in the field as part of the filename.
