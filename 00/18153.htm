When migrating my contacts from an older e-mail system, I accidentally imported the CSV contacts list to Outlook in Office 365 twice.  Is there a way to delete duplicate entries?

Quickest way would be to add the "Created" date field in the view and then sort by that field. Each imported set will have a near identical field value which you can then delete.

- Click on the <View> ribbon

- change your view setting to a list type (i.e. Phone)

- change <View Settings> to add the Created column using <All Contact Fields>

- sort on the <Created> field  within the view or alternatively just click on that column when showing the list view
