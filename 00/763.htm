Hello, I've had the same problem and finally found the solution. The language file for VBA editor interface is stored in the following folder with different subfolders:

C:\Program Files\Common Files\microsoft shared\VBA\VBA7.1

(you may need to search for VBEUIINTL.dll if you can't find this folder on your computer) 

 in this folder the 1033 subfolder and its contents are for English interface. The other ones are for different languages.  Somehow VBA chooses the wrong subfolder. So you basically need to copy the contents of 1033 folder ( as a workaround ):

"C:\Program Files\Common Files\microsoft shared\VBA\VBA7.1\1033\APC71ITL.DLL"
"C:\Program Files\Common Files\microsoft shared\VBA\VBA7.1\1033\VBE7INTL.DLL"
"C:\Program Files\Common Files\microsoft shared\VBA\VBA7.1\1033\VBEUIINTL.DLL"

to the 2052 folder (for chinese for example, may be you have different), for changing the language.

Before doing this , restart and don't start any office products. Then copy the files; overwrite if asked. I also suggest you to take a backup of VBA7.1 folder in case something goes wrong.

Hello Blake McNeill:

 

Thank you for posting to the forum regarding your questions about family options for different services available across your Microsoft devices.  We can certainly appreciate your desire for a easy way to manage and streamline the use of your services across a family.  I will provide the details currently available regarding your questions as well as some resources to make your voice heard when it comes to suggestions for future features.

 

At this time there are not family options for services like Xbox Live Gold and Xbox Music.  There was once a Xbox Live Gold family pack, that was disbanded starting in October.  As far as I am aware there has never been a family pack type option for the Xbox Music Pass, formerly Zune Pass.

 

In the current state a  User for Xbox Live would need to have their own individual account and Live subscription to enjoy all the features of Xbox Live Gold.  On a Xbox console a family could easily share the same profile if there is no need to have individual members using the subscription at the same time.  When it comes to your phones, tablets and Windows 8 PC's this becomes a tougher scenario.  All of these devices work with a primary Microsoft account that is typically individualized to the user.  It is likely unrealistic that 3 family members would want to use the same primary Microsoft account and share the same SkyDrive account, contacts, calendars, Xbox Live, Xbox  Music, etc....

 

When it comes to using the Music Pass I have a couple suggestions that may allow the family to take advantage of one Music Pass, at least on some devices.  One thing you can consider is simply using the Xbox Music website.  This site will allow users to stream music from our marketplace at no cost.  This site will allow each user to sign into the site via their browser and create a collection and build playlists that are unique to their accounts.  This is going to require a connection as music cannot be downloaded and played offline. 

 

If you want to have some expanded use of the service you might consider buying the Music Pass on a fresh account that is created for the purpose of sharing this service.  You could then create a local profile on your Windows 8 devices, this will not be possible on the Windows phone, http://windows.microsoft.com/en-us/windows/create-user-account#create-user-account=windows-8.  Once you create these local profiles on your tablets/PC's you can then sign into the Xbox Music application with the shared account, do this by opening the app > then opening your charm bar > clicking settings > account > sign in.  This would allow you to download music with the Music Pass to a total of 4 devices.  If you need to use the Pass for downloads on more then 4 devices, it would be suggested to purchase a second Pass.

 

As for sharing a playlist you created with another user.  The best and easiest way to do this would be via the Xbox Music website through Facebook.  There is a new feature of the site that allows you to share this playlist on Facebook and those that click this via your Facebook page will be able to stream it seamlessly for Free.  You can do this by creating the playlist and then clicking said playlist.  The icon farthest to the right will be the share icon that allows you to set this up.  If you go with the option of setting up a shared account to be used on multiple devices, this will allow for access to that playlist by multiple users from different platforms.  Now, the use of the playlist will be limited.  The Xbox Music application does not store playlists locally on devices but instead saves them to the cloud.  This means the playlist would have to be streamed and streaming can only be done from one device at a time.

 

Please let us know if you have any other questions or if any of the above information needs additional clarity added.
