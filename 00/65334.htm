a. user say:

Do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesn’t work.

If the System Restore doesn't work, do a Startup Repair by booting to the genuine Windows Vista Installation Disk (or one you can borrow from ANYONE) or from a Recovery Disk.  Here's the procedure: http://www.bleepingcomputer.com/tutorials/tutorial148.html.  To boot to the CD you may need to change the BIOS to make the CD-drive first in the boot sequence.  To do that, wait for the screen that tells you the F key to push to access the boot menu or boot setup.  Push it quickly. Make the changes, save your work, and exit.  Put the CD in the drive and reboot.  When prompted, push any key to boot from the CD.

If you don't have either disk, you can make a bootable Recovery Disk using http://neosmart.net/blog/2008/windows-vista-recovery-disc-download/ along with burning software like: http://www.snapfiles.com/get/active-isoburner.html and, of course, a blank CD.

=====================================================================================================

If those don't work your user profile is probably corrupt.  To fix this use http://windows.microsoft.com/en-AU/windows-vista/Fix-a-corrupted-user-profile.  If that is the only available Administrator Profile (you need to be an administrator to fix this), enable the Hidden Administrator Account (HAA) using http://www.microsoft.com/communities/newsgroups/en-us/default.aspx?&lang=en&cr=US&guid=&sloc=en-us&dg=microsoft.public.windows.vista.administration_accounts_passwords&p=1&tid=d20f9db4-7b2c-48be-a087-7835dc2a9055&mid=d20f9db4-7b2c-48be-a087-7835dc2a9055.  If you can't remember the password, try nothing as that's probably what you (or the vendor) did during installation.  I suggest you create at least two Administrat accounts (to go with your standard user account) with strong passwords that you will remember just to be on the safe side.  Once done, be sure to deactivate the HAA to save it in case it's needed again and for security purposes (since people often try to hack into systems using this account).  Do NOT use the HAA as your administrator account because if you lose the only administrator account on the system or it becomes corrupt again then you are hosed.

Once you're in go to Start / Control Panel / Administrative Tools / Services and scroll to User Profile Service.  Make sure it is set on Automatic (do so if it is something else).  If it is started, stop it and then restart it.  If it is stopped, just restart it.  This probably will be fine, but it is the service that failed so it doesn't hurt to check to make sure it is setup correctly.

I hope this helps.

Good luck!

Panasonic 2123 dot matrix. paperout light on

Need to shut off paper out light on Panasonic KX P2123. Says I can in manual. Don't know how. Ithink this is why my printer won't work. Hooked up with paralle to USB2 cable. Anything else I need to do?
lemp say:

It's been many years since I've owned a Panasonic dot matrix printer and I don't recall how easy or difficult it is to remove the platen.  If you can determine how to do this without damaging the printer, you may be able to see why the paper out detector is giving you a false indication (the paper out sensor is below the platen).  Perhaps all you need to do is blow a little compressed air at it to clean out any debris that is blocking the sensor.

Assuming that this is, in fact, a problem with the printer's Paper Out detector and not a problem with your parallel to USB cable, the first thing to do is to see if you can print a line at a time by pressing the "On Line" switch.

If you can, then disable the Paper Out detection by using the "Initial Setup mode" as described starting at page 3-22 of your User Manual.  As near as I can tell, the steps would be:

  1. Turn on the power switch while pressing the "FUNCTION" switch.  The "On Line/Function" light should be blinking and all row indicators should be lit.
  2. Press and release the "SUPER QUIET" switch until the row indicator lights look like this:
         OFF
         OFF
         OFF
         ON
  3. Press the "ON LINE" switch to make the light in the 3rd column change from on to off
  4. Press the "FUNCTION" switch to save the setting and exit "Initial Setup mode."
