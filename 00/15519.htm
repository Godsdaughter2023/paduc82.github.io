jay freedman say:
You could use a macro to do the copy/paste for you. Put this macro into Excel and make a Quick Access Toolbar button for it. You'll need to open the Word document and put the cursor in the table you want transferred. Then click the button in Excel.

Sub GetWordTable()
    Dim wd As Word.Application
    Dim doc As Word.Document
    Dim sht As Excel.Worksheet
   
    On Error Resume Next
    Set wd = GetObject(Class:="Word.Application")
    If wd Is Nothing Then
        MsgBox "Open the Word document and click in the desired table.", , "Error"
        Exit Sub
    End If
   
    Set doc = wd.ActiveDocument
    Set sht = ActiveWorkbook.Sheets(1)
   
    wd.Selection.Tables(1).Range.Copy
    sht.Range("A1:A1").PasteSpecial Paste:=xlPasteValues, operation:=xlPasteSpecialOperationNone
   
' Optionally, close the document and Word --
'    doc.Close SaveChanges:=wdDoNotSaveChanges
'    wd.Quit
End Sub

You might want to change the references to ActiveWorkbook, Sheets(1), and/or Range("A1:A1") to place the pasted copy somewhere other than the top left corner of Sheet1 of the open workbook.

What you created is not strictly a virus, it's software that disables the keyboard, which in some cases might be included in the payload that a virus delivers.

What really matters here is what the software does in order to disable the keyboard, since that's what you truly need to reverse.  No commercial antivirus software will help you, since that is designed to detect and deal with well known malware produced and distributed widely, so it has no idea what your software did nor even that it exists.

If neither 1a or 2a is the issue, work through these steps- test after each.

1. Shut down the computer, modem and router for a few minutes.

2. Internet Options> Browsing History> Delete> (next screen) uncheck "Preserve Favorites website data." then delete the History, Temporary Internet Files and Cookies. Restart.

3. From Start> All programs. Accessories, right-click Command Prompt and choose "Run as administrator". At the flashing cursor type in ipconfig /flushdns (note the space between config and / ) hit [Enter] then Restart.

4. If amazon is listed in either Trusted Sites or Restricted Sites, remove it (Internet Options> Security.)

5. Go to Internet Options> Privacy and try a cookie setting of Low or Accept all.

6. Verify that the computer's date, time and time zone are correct. These settings can affect cookie handling.

7. If you have a program running that has a cookie manager feature (CA, CCleaner, etc.) disable that feature.

8. With the warning that a reset will delete AutoComplete data, such as passwords and you will have to reinstall or re-enable browser add-ons, use the Reset Internet Explorer Settings button (Internet Options> Advanced.)

iry chen msft say:

Hi Celeste,

You don't need to import .csv file to your Outlook for Mac directly. You should import the .csv file to your System Contact app in your Mac, and then in Contact app, use the File > Export... > Export vCard... option or simply drag and drop your Contacts from the Group list to your desktop (a .vcf file). 

Finally, open Outlook and go to People (contact), drag and drop the .vcf file from the desktop to the Address Book root category. (Make sure a "+" appears as you hover the file over the Address Book category.)

Hope this can help.

Thanks,
