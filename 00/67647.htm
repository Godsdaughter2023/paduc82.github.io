zoe - zfotography say:

For anyone having problems connecting to their printers with Windows 10, you might want to try connecting via Bluetooth.  I just got a new laptop, which has Windows 10 pre-installed (Windows 7 on my old laptop).  I wasn't getting anywhere with wireless, so decided to try Bluetooth.  It was really easy, and I can now print.  I am connecting to an HP Photosmart printer, but I believe the Canon MP160 has Bluetooth capability as do other printers.  FYI, I did not find any instructions with the HP printer that described how to set this up.  Hopefully, newer models have some instructions.  Also, if your computer or printer is not Bluetooth enabled, I believe there are adapters available.

My Bluetooth was turned off my laptop.  To turn on in Windows 10, go to Settings/Devices/Bluetooth.  After it is turned on, your Bluetooth enabled printer should come up on the list of devices.  When I clicked to pair with the printer, I was asked for a passkey code for the printer.  I had no idea what that should be, but when I went to the HP Photosmart printer and pressed the Setup button, I scrolled down to "Bluetooth Menu".  I presume Canon has something similar.  On the HP printer, I scrolled to Passkey, and 4 numbers were listed (newer printers have 6 numbers, I believe).  I went back to the laptop and entered the 4 numbers.  It took a minute or two to fully connect, then it was all set.  I went to the printer again, though, because one of the Bluetooth scroll settings is for Security.  I made sure it was set for High, which requires a passkey code for anyone to pair/connect to the printer.  There is also a setting for Visibility.  After setting my husband's laptop up via Bluetooth (he had not been able to use the printer since he upgraded to Windows 10), I set the Visibility to Invisible.  So, other people in the vicinity (within 30 ft. or so) using Bluetooth, will not find our printer-- seems fairly secure.  It's so easy to enable/disable Bluetooth in Windows 10, that we can enable it only when we want to print something if we feel we need the extra security.

Hope this is of some help for those of you having trouble connecting to printers in Windows 10.

Good luck,

Zoe
