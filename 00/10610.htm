bernd p. say:

Just ignore these messages.

You can share a notebook in your home network (a LAN) by creating a Windows file share as usual for any other files. What is special for OneNote notebooks is that they are not files, they are folders.

The local notebooks by default are stored in

C:\Users\<user>\Documents\OneNote Notebooks as sub-folders.

To share the notebooks with another computer in your LAN just make the OneNote Notebooks folder a Windows shared folder so you can access this folder from another computer.

Then you can open these notebooks in OneNote on the other computer. That's all.

Or you use a folder on a file server to store your notebooks, using the same construction.

This worked since OneNote 2010 and still with OneNote 2016 (the version from Office, not the free one !).
