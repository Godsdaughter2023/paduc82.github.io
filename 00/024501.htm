open excel say: Hi John,

I
Does it show white screen or just hanged excel window? It may also be because of the macro in Personal Workbook.

What is it: 
This is a hidden workbook that opens when you start Excel.
The code you copy in this workbook is available in all workbooks you have opened in Excel.

Where is it: 
If it exists, you can find the file in the Excel startup folder.
Windows XP
C:\Documents and Settings\USERNAME\Application Data\Microsoft\Excel\XLSTART
In Vista or Windows 7 look here
C:\Users\USERNAME\AppData\Roaming\Microsoft\Excel\XLSTART

Backup all files to protect your past work then delete them from this location.



II.
Another expected reason is Excel's toolbar file which tends to grow very big and this toolbar file is loaded when excel starts up. The extension of this excel toolbar file is XLB, default location is "Application Data\Microsoft\Excel\" and normal size is 10 to 90 KB. Problem arises when this size reaches in megaBytes(MB). 

Solution: Delete all xlb file from these locations:
Windows 98 Users --> "C:\Windows\Application Data\Microsoft\Excel\excel*.XLB"
Windows 98SE/NT/ME Users --> "C:\Windows\Profiles\USERNAME\Application Data\Microsoft\Excel\excel*.XLB"
Windows 2000/XP Users --> "C:\Documents and Settings\USERNAME\Application Data\Microsoft\Excel\excel*.XLB" 
Windows 7 Users --> "C:\Users\USERNAME\AppData\Microsoft\Excel\excel*.XLB" 

Assumptions: USERNAME is your system user name and User Profiles or Operating System is installed on drive C. Please change the values accordingly.

WARNING: However, XLB file is re-created automatically but you'll lose all of your toolbar customizations (or say personal menu settings).



Try any of these methods, you may also try 'Ctrl+Pause/Break' to stop any other macro/process when excel starts.

The Set statement is used to assign an object reference to a variable or property; it cannot be used with a string variable.  

Try the following modification to your code:

'=========>>
Option Explicit

'--------->>
Sub FindCountry()
    Dim dd As DropDown
    Dim ddValue As String
    Dim sStr As String

    Set dd = ActiveSheet.Shapes(Application.Caller).OLEFormat.Object
    ddValue = dd.Value   
    sStr = dd.List(ddValue)
    ActiveSheet.Cells.Find(What:=sStr, After:=ActiveCell, _
                           LookIn:=xlFormulas, _
                           LookAt:=xlPart, _
                           SearchOrder:=xlByRows, _

                           SearchFormat:=False).Activate
End Sub

My answers are in bold, embedded in your original post below, but please read this:

https://www.microsoft.com/en-us/windows/windows-10-faq

Hi Everyone,

I have many questions regarding Windows 10, so your help and answers will be greatly appreciated, and assist me in determining if it is the right time for me to install it. I've done some lengthy research, but haven't been able to find the answers on my own. In fact, it's only led to more questions!

I currently have Windows 7, and aside from the graphics card, it looks like my laptop is compatible. (I will have to research/find the graphics card information.)

1. Is the icon on my task/start bar on Windows 7 the link to the download of Windows 10 or just a preview? The icon on the taskbar is to reserve the final version of Windows 10, but it will not download yet.  July 29 is the date it is supposed to be installed.

2. If this is just the preview, will I be prompted to download the complete program of Windows 10 later? If so is Windows 10 really free?  Yes the final version of Windows 10 Home will be really free if you have a qualifying Windows 7 license.

3. From what I've read, I have to know what an ISO File is. Is this for the preview or the actual Windows 10 program.  Is this a Beta? Demo?  An ISO file is just a disc image, so you take the ISO file and burn it to a DVD, then boot the DVD to install Windows 10.  You don't need to use the ISO, but it's a good idea to have one.  Using the Windows 10 reservation system will download Windows 10 via Windows Update, and you won't need to boot from a DVD to install it.

4. Will I be able to reserve Windows 10 from the icon, but install it later, as my antivirus is not compatible with it at this time. (I contacted my internet security software manufacturer, and they said that it isn't compatible with it yet.)  Yes you can choose when to install it.

The following questions assume that Windows 10 is a program and not just a a preview, here are my additional questions:

1. I have a separate "D" sector/drive that was installed on my computer to store my files. This is not affected when I do a system restore. Will the D sector still be available after installing Windows 10, or will it be overwritten?  The D drive should still be there if you do an upgrade install.

2. Is Office 2010 compatible with Windows 10 and other programs, such as Google Chrome, Mozilla Firefox, etc.?  Windows 10 will tell you what is not compatible when it is installing.  I'm not sure about Office 2010, but Chrome and Firefox work.

3. Will I have to reinstall all my programs after installing Windows 10 free version?  If you do an upgrade install, it will preserve all the compatible programs, and all your data/files.

4. I've read that I cannot play DVDs, and that Windows 10 does not include Media Center. Does this refer to the preview or the program -- or both? Do I have to buy and install additional software to play DVDs or use Media Center functions?  You can play DVD's using free software, like VLC.

5. Do I need to create backup disks of Windows 10, like I did with Windows 7 when I first got my computer, or will I receive a key that will allow me to reinstall the program Yes you should make backup discs for Windows 10.

6. If I have to do a system/complete restore after the first year, will I still be able to reinstall Windows 10, or will I have to buy it? If it's the same computer, you will still be able to install it.

7. Will I be provided with a key to use if I have to uninstall/reinstall Windows 10)  I don't think the traditional Windows key will be used with Windows 10, but you should have no problem reinstalling it if you need to, on the same computer that it was originally installed on.

8 Will Windows 7 files still be available if I am not satisfied with Windows 10? (One site states that it will be under "Windows.old," while another said I will need to use my back up of Windows 7.  I am not sure if you will still be able to have a valid Windows 7 license after upgrading to Windows 10.

Any help will be appreciated!
