brian tillman [outlook mvp 2007-2019] say:
This is not an Outlook setting, it is a Windows setting, controlled by the registry key "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\ShellFolders\My Pictures".  Change that key to the full path of the folder you want to use.  All references to "My Pictures" will now use the folder you specified instead of the default location C:\Documents and Settings\user\My Pictures.   The change should take effect immediately the next time you open an application that references "My Pictures".
