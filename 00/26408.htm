If You have a Laptop Mac such as a MacBook, MacBook Pro and happens to be one of the higher end Units.  This Tip may Help. It has helped me, as I have such a Machine.

They sometimes have Dual Display Processing circuits, one circuit is built on the the actual motherboard. The other is discrete Plugin Video Card made by Radeon or another company. In some of those Machines The energy Saver Preference has check box for a setting called Automatic Graphics Switching. If Your machine has the check Box do the Following:

  1. Quit all Office applications.
  2. Next Go to system Preferences,
  3. Open Energy Saver.
  4. Uncheck Automatic Graphics switching.
  5. Next restart computer after closing up Preferencess
  6. Now open in You case Excel 2016.
    Note: Each office Application May take minutes to open the first time after setting this.
  7. Now try Excel for those issues. Operations should speed up some and these issue may/should disappear.

If you machine has this dual video setup but doesn't have this setting, there is a Utility called gfxCardStatus, that does the same thing, only a lot better.

While Turning off Automatic Graphics Switching in Energy Saver, has graphics card stay on full blast all the time; gfxCardStatus has a setup where it detects the application, and and if it needs some extra horsepower, It turns card on and keeps it on, only as long as you have that application running. Then when you quit that application it reverts back to built in Graphics.

What seems to be the possible issue is, that Office 2016 applications are interfering with the Setting in the Mac Energy Saver, so it can't decide whether to use Built in or Discrete Graphics. So the system is Left hung deciding which system to use.
