There's one trick you can try:

You can create a folder in the Finder, select your contacts in Outlook and drag them to that folder.

The update saves your current Windows installation in a specific folder;

C:\Windows.OLD\

That should be taking up that extra 15 GB

You can go to "Storage" in settings to find the C: drive. Click on it to expand it. Then click on OTHER to expand it

That back up folder is there in case you want to undo the update. It has a defined 10 day expiration date.  After that Windows will eventually delete it to free up the space.  You can speed up freeing that space or extend the time limit.

You can also trigger the delete of that folder using the built in Disk Clean tool (see links at bottom of this post).

dale howard [mvp] say:

WhitAnthony --

Bad news for you, my friend.  There is no way to automatically format a user-entered numeric value as a percentage value.  Assuming you are using a Number field, the user will need to enter the percentage as a decimal, such as .12, which would represent a value of 12%.

On the other hand, once the user has entered a value, you could use a custom Text field with formatting to display the value as a percentage.  For example, following is a formula I entered in a custom Text field that refers to the Number field named Schedule Risk:

Format([Number2],"0%")

I tested the formula and it works just fine.  Many people do not know that you can apply Excel formatting to numeric formulas in Microsoft Project.  The above is an example of that.  Hope this helps.
