Entering an asterisk (*) in the search box immediately returns all files in the current folder and all its subfolders.

You can also change the settings to default for all the folders at a same time when you have many sub folders.

Here's how to search subfolders:

1 - Turn indexing back on

2 - Type the word "folder" into your Start menu, then choose the option called "Change search options for files and folders"

3 - Turn on the "Include sub-folders..." option.

Give your index some time for it's first-time update and you should be good.  Newly added files are indexed instantly, so it will only take time this first time.
