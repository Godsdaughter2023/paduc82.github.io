slance310 say:

Hi,

You mentioned that you only have one M.2 connection... If that is the case, you will not be able to clone your SSD as the two drives both need to be hooked up to do that...

You would need to create an image of your current SSD and save it on either your 1TB HDD or a external USB drive... I don't think the Easeus free will do an image, but maybe...

Another issue to check before you get started is whether any of the partitioning\Disk Management tools you may need will even work with a NVME drive... I just recently added one of these to my desktop, and quickly found most of the tools I normally use will not recognize the drive...

If you do a clean install you will lose everything except for Apps that come with Windows... I'm not sure how easily you can replace the ones you lose... I know little about reinstalling Apps as I don't use any.. Just standard installed type programs...

You should be able to reinstall any app or program you have purchased... Any preinstalled programs that you lose, you should be able to download from your computers manufacturer..

I see one potential problem with Cloning or Imaging, and that is the 900mb Recovery Partition... It will need to be moved in order to expand your OS partition... The good news is that your EFI partition is first, which makes things easier...

Again the problem may be finding a BOOTABLE program to move the Recovery Partition and expand the OS Partition... Windows will not let you move the Recovery Partition with it's Disk Management tool... You will need a 3rd party tool of some kind..

None of the programs I normally use for stuff like that would recognize the NVME drive... I was able to find a BETA update for Acronis Disk Director 12 that would do this, but it was hard to find... Partition Wizard is another free program, that does not recognize the drive...

For some things you can use Win 10 installation media and select REPAIR at the Install screen, then Customize... But that only adds, deletes , and formats partitions. It does not provide a method to move them...

Windows 10 does a really good job of finding drivers during the install, or installing them afterward, but it is possible that you may need to add some yourself, or change the ones it uses...

BTW... Windows 10 will create an image of your current SSD, and should also put it on your new SSD without problems... You would need to save the image to your DATA HDD using the backup and Restore (Win 7) tool that is included in Win 10...

Once you have the image you will either need the System Repair Disk, or installation media to load it onto the new SSD... The tool won't work if moving to a smaller drive, but you are going larger.... The problem with this is that you still need to move the 900MB Recovery Partition...

Just so you know... the 900mb Recovery Partition can be deleted... The only drawback to doing that is tha some of the built-in Recovery operations will not function... You can still do them, but would need to boot with installation media or a System Repair Disk...

imran chand say:
Hi selmore11,

User Account Control (UAC) is used to increase the security of the operating system that can help prevent unauthorized changes to your computer. UAC does this by asking you for permission or an administrator password before performing actions that could potentially affect your computer's operation or that change settings that affect other users.

By verifying these actions before they start, UAC can help prevent malicious software (malware) and spyware from installing or making changes to your computer without permission.

To Turn ON User Account Control:

1.Click on Start, type uac in the search box, and then click Change User Account Control settings.

2. To turn on UAC, move the slider to choose when you want to be notified, and then click OK.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
