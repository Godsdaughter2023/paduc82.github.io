gord dibben say:

You could use a macro.

Select the range then run.

Sub RemoveAlphas()
Dim intI As Integer
Dim rngR As Range, rngRR As Range
Dim strNotNum As String, strTemp As String

    Set rngRR = Selection.SpecialCells(xlCellTypeConstants, _
            xlTextValues)

    For Each rngR In rngRR
        strTemp = ""
        For intI = 1 To Len(rngR.Value)
            If Not Mid(rngR.Value, intI, 1) Like "[A-Z,a-z]" Then
                strNotNum = Mid(rngR.Value, intI, 1)
            Else: strNotNum = ""
            End If
            strTemp = strTemp & strNotNum
        Next intI
        rngR.Value = strTemp
    Next rngR

End Sub

Hi,

If you want to combine all the emails of your accounts in one inbox then you need to have a separate

data file and need to do an email rule to forward/ have a copy of emails to that SHARED INBOX.

The process of this one is quite long. But I have faith in you that you can follow the steps :)

Okay, let me give you the step by step on how to create a new data file first.

1. Go to File> Accounts Settings > Data File

2. Click Add then name your data file (eg. COMBINED/SHARED INBOX)

3. Then click OK and go back to Outlook

You will be able to see the newly created data file on the left side of Outlook. right?

 Okay, next step is to create a FOLDER under that data file. You just need to right click the data file then select NEW FOLDER and name it as Inbox.

There you go! Oh, by the way, only new emails will be stored on the Shared Inbox.

Next step is to create an EMAIL RULE.

1. Go to Rules > Manage Rules & Alerts

2.  On the START FROM A BLANK RULE, Select "Apply rule on message I receive"

3. In the Rules and Alerts dialog box, on the E-mail Rules tab, click New Rule.

4. Check Through the specified account and click the word "specified" so that little box will appear and select an account > click ok > then click next

5. Check Move a copy to the specified folder so that you will still have a copy of that email on the individual inboxes of your accounts > then you need to choose a folder, select the INBOX folder that we created on the NEW DATA FILE

6. Click Finish

7. Click Apply then Ok

If you have multiple email accounts you need to set an email rule on each email accounts. Just  follow the steps.

There you have it! Just reply if you have questions/concerns about this one. Okay?

You can only open a .pub file with Publisher.

 

If you are trying to access the images then hopefully you had copies of them on your hard drive and inserted them into the Publisher page, as there is also no way to extract the images without opening the .pub file.
