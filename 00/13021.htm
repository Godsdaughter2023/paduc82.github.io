Find position of a specific charater in a cell? - Microsoft Community

cthr say:

Find position of a specific charater in a cell?

how can I find the position of the specific charater in a text? example below:

 

/abc/cde/fgh/ijk

/abcdefg/hijkl/mno

/ab/cdefg/hijk/lmno

Such that I want to return the psotion of the third "/"  in the cell

jeeped say:

how can I find the position of the specific charater in a text? example below:

 

/abc/cde/fgh/ijk

/abcdefg/hijkl/mno

/ab/cdefg/hijk/lmno

Such that I want to return the psotion of the third "/"  in the cell


FIND() will return the position of a searched character or string of characters and SUBSTITUTE() can be used to replace an instance of a character (e.g. third). Put the two together by using SUBSTITUTE() to change the third instance of a / character to some (almost) never used character and use FIND() to return the position of the (almost) never used character.

              

The formula in D2 is,

=FIND("×",SUBSTITUTE(A2,"/","×",C2))

... which used a reference to C2 to decide the third instance of the forward slash character. I used × (Alt+0215) as the (almost) never used character. If a third instance of the forward slash cannot be found the formula returns the #VALUE! error which can be trapped with IFERROR() if desired.

cody c. say: Hi rleepac,

If Excel worked correctly in a safe boot, that likely means something in start up is causing the problem.

Let's determine what's causing the issue. To do this, follow these steps: 

1. Click the Apple menu and select System Preferences. 
2.  Select Accounts. 
3.  On the left side, the account will appear selected. On the right side, select Login Items.

Note: All the items that are listed here are set to be started during a normal startup of the operating system. If the item is marked with a check mark, that item will be hidden and will not be started during a Safe Boot.

Note: If you are not able to make changes, click the Lock icon on the left bottom corner. Enter your Mac User Password at the prompt and click OK to place the lock in the unlocked position which will now allow you to make changes. 
4.  To remove an item, select the item, and then click the minus sign (-) that is below the list. 

Note: If you remove the item (application), it does not remove the application. You can add the item later on by clicking the plus sign (+) on this screen. However, you will need to know the path to the item (application) to be able to add it back. 
5.  Remove the items one at a time, and then restart normally to test whether the problem still occurs.

If you have determined which item is causing the problem, you may want to contact the software manufacturer of the startup item for a newer version/update.
