hal hostetler - mvp-outlook say:
An Exchange Server is a business class email service that is usually only available in the corporate world.  It is rarely seen in the consumer world, so, unless you absolutely know you need the Exchange Server service (E.G. this is offered to you by your employer), you probably will want either POP or IMAP. 

hal hostetler - mvp-outlook say:
I suspect you may have added the Exchange Server service to your Outlook profile by mistake.  Unless you know for a fact that you have an email account on an Exchange Server, go to Control Panel > Mail > Email accounts, select the Exchange Server account and Remove it, then add a new account of the correct type. If Outlook complains about a password and won't let you remove the account, Control Panel > Mail > Show Profiles, click the Add button to add a new profile, then select the old profile and Remove it.  Make sure the new profile is shown under "Always use this profile" and click OK.  You can now either restart Outlook, which will ask you to setup an email account, or use Control Panel > Mail > Email Accounts to add one.
