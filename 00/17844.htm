mike h.. say:

Hi,

 

I'd do it like this

Sub test()
Dim OldRng As Range
Dim NewRng As Range
Dim LstRw As Long
With ActiveSheet
    LstRw = .Cells(Rows.Count, "A").End(xlUp).Row
    Set OldRng = .Range("A2:A" & LstRw)
    Set NewRng = .Range("B2:B" & LstRw)
    OldRng.Interior.ColorIndex = xlNone
        For Each c In OldRng
            If Not IsError(Application.Match(c.Value, NewRng, 0)) Then
                c.Interior.ColorIndex = 5
            End If
        Next c
End With
End Sub

gary's student say:

Try:

 

Sub test()
Dim OldRng As Range
Dim NewRng As Range
Dim C As Range, val As Variant
Dim LstRw As Long
With ActiveSheet
    LstRw = .Cells(Rows.Count, "A").End(xlUp).Row
    Set OldRng = ActiveSheet.Range("A2:A" & LstRw)
    Set NewRng = ActiveSheet.Range("B2:B" & LstRw)
    For Each C In OldRng
        val = C.Value
        If Application.WorksheetFunction.CountIf(NewRng, val) > 0 Then
            C.Interior.ColorIndex = 5
        End If
    Next C
End With
End Sub

diane poremsky mvp (slipstick.com) say:

Clean up folder removes older messages in a thread that contain duplicate content. Imagine Healee sends David a message,  David replies with the previous message quoted, Healee replies back, quoting all. This goes back and forth for a while and you end up with a bunch of messages repeating the same content. Clean up folder removes the oldest messages in the thread, leaving only the newest messages with the complete thread in the inbox.
