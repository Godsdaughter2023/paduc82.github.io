adam_w_ say:

This error is commonly caused by a corrupt preferences file or a third party tool interfering with Excel.   Follow the steps below to reset your Excel preferences:

 

  1. Quit all Microsoft Office for Mac applications.
  2. On the Go menu, click Home.
  3. Open the Library folder.
  4. Open the Preferences folder. On the View menu, click Arrange by, and then select Name.
  5. Look for a file that is named com.microsoft.Excel.plist.
  6. If the file exists, drag the file to the desktop. If you can’t locate the file, the application is using the default preferences.
  7. If you were able to move the file to the desktop, start Excel, and check whether the problem still occurs. If the problem still occurs, quit Excel, and restore the file to its original location. Then, go to the next step. If the problem seems to be resolved, you can move the com.microsoft.Excel.plist file to the trash.
  8. Quit all Office for Mac applications.
  9. On the Go menu, click Home.
  10. Open the Library folder.
  11. Open the Preferences folder.
  12. Open the Microsoft Folder.
  13. Look for a file that is named com.microsoft.Excel.prefs.plist.
  14. If the file exists, drag the file to the desktop. If you  can’t locate the file, the application is using the default preferences.
  15. If you were able to move the file to the desktop, start Excel, and check whether the problem still occurs. If the problem still occurs, quit Excel, and restore the file to its original location. Then, go to the next step. If the problem seems to be resolved, you can move the com.microsoft.Excel.prefs.plist file to the trash.

 

If the above steps don’t resolve the issue, visit the KB article linked below for more troubleshooting steps:

http://support.microsoft.com/kb/2026104

jay freedman say:
I can't exactly duplicate the context menu with the 3x3 grid of buttons, because that control was removed in Word 2013. The following macro needs to be run only once (see http://www.gmayor.com/installing_macro.htm if needed). It will insert a Cell Alignment item above the Table Properties item in the context menus that appear when you've selected the whole table, the text in a single cell, or several cells.

Sub AddCellAlignmentToContextMenus()
    Dim ctl As CommandBarPopup
    ' insert 'Cell Alignment' before 'Table Properties'
    Set ctl = CommandBars("Whole Table").Controls.Add(ID:=30411, Before:=17)
    Call AddButtons(ctl)
    Set ctl = CommandBars("Table Text").Controls.Add(ID:=30411, Before:=20)
    Call AddButtons(ctl)
    Set ctl = CommandBars("Table Cells").Controls.Add(ID:=30411, Before:=14)
    Call AddButtons(ctl)
End Sub

Sub AddButtons(ctl As CommandBarPopup)
    ctl.Caption = "Cell Ali&gnment"
    With ctl.Controls
        .Add ID:=3914 ' top left
        .Add ID:=3915 ' top center
        .Add ID:=3916 ' top right
        .Add ID:=3917 ' center left
        .Add ID:=3918 ' center center
        .Add ID:=3919 ' center right
        .Add ID:=3920 ' bottom left
        .Add ID:=3921 ' bottom center
        .Add ID:=3922 ' bottom right
    End With
End Sub
