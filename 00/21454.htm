In your translated chat you will be prompted to choose your preferred languages and whether you want a female or male voice to represent you during voice and video calls. The languages will be pre-populated but you can also change them.


During your conversation, you will see your messages as they are originally typed and the translation in your preferred languages.

Note: Call captioning is available on Skype (version 8) and on Skype for Windows 10. To make a translated audio or video call, click on the audio or video call icon in your conversation. Your voice will be translated and the translation will also be shown as a caption in your call window.


Your translated chat will continue to be available in your recent chats with that person. If you want to start a translated chat with someone else, you'll need to start translator with them in the same way.

Important: A translated chat will only support you and a single contact. If you try to add another participant or bot to your translated chat, you will get a message that the translations will no longer work until the added participant is removed. Once they are removed, you will see your translations continue as normal.

graham mayor say:
In theory the following Outlook macro might work for you. It works on the premise that a message with no response should have a ConversationIndex length of 44. and it examines the last 30 days of messages (the number can be changed) in the sent items folder and writes the date, sender and subject to the worksheet named in bold (which must exist). The worksheet is cleared before the data is added so the worksheet contains only the values from the current check.

How reliable the methodology is I cannot say, but it appears to work.

Sub CheckNoReply()
Dim olFolder As Folder
Dim olItem As MailItem
Dim olItems As Items
Dim olDate As Date
Dim strDate As String
Dim strValues As String
Dim xlApp As Object
Dim xlWB As Object
Dim xlSheet As Object
Dim bXStarted As Boolean
Dim rCount As Long
Const iDays As Integer = 30 'The number of days to check
Const strPath As String = "D:\My Documents\Test\User Files\Test\NoReply.xlsx"
    olDate = Format(DateAdd("d", -iDays, Now()), "Short Date")
    strDate = "[Received] >= """ & olDate & """"
    Set olFolder = Session.GetDefaultFolder(olFolderSentMail)
    Set olItems = olFolder.Items.Restrict(strDate)

    On Error Resume Next
    Set xlApp = GetObject(, "Excel.Application")
    If Err <> 0 Then
        Application.StatusBar = "Please wait while Excel source is opened ... "
        Set xlApp = CreateObject("Excel.Application")
        bXStarted = True
    End If
    On Error GoTo 0
    'Open the workbook to input the data
    Set xlWB = xlApp.Workbooks.Open(strPath)
    Set xlSheet = xlWB.Sheets("Sheet1")
   
    rCount = 1
    xlSheet.UsedRange.Clear
    For Each olItem In olItems
        If Len(olItem.ConversationIndex) = 44 Then
            xlSheet.Range("A" & rCount) = olItem.SentOn
            xlSheet.Range("B" & rCount) = olItem.To
            xlSheet.Range("C" & rCount) = olItem.Subject
            rCount = rCount + 1
        End If
        xlWB.Save
    Next olItem
    xlWB.Close SaveChanges:=True
    If bXStarted Then
        xlApp.Quit
    End If
    Set xlApp = Nothing
    Set xlWB = Nothing
    Set xlSheet = Nothing
    Set olItem = Nothing
End Sub
