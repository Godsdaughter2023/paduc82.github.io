govert van drimmelen say: Hi,

(Well, you're right. . . )

I recommend you try Excel-DNA to get started with F# / Excel integration. It will allow you to create an Excel add-in with F#, including high-performance user-defined worksheet functions. Excel-DNA supports you making full-featured F# add-ins for Excel, including custom Ribbons, asynchronous function (based on the F# async features), and real-time data sources (based on the F# IObservable events). And it's open-source and free for all use.

With Excel-DNA, the following F# code would give you a worksheet function in Excel:

    module myFunctions
    open ExcelDna.Integration

    [<ExcelFunction(Description="Add two numbers")>]
    let myAdd x y = 
        x + y

With Excel-DNA, making an add-in with F# is easy and you can focus on F# and the Excel object model, instead of the details of ancient C-based Excel extensibility model, memory management etc.

If you don't want to use something like Excel-DNA to make your .xll, you essentially have to rewrite a lot of the Excel-DNA implementation, including the (smallish) C/C++ parts which interact with the C API. Inspecting the source code of Excel-DNA would be a good start to doing this. But since F# runs on top of the .NET runtime, you need some kind of unmanaged layer to mediate the C-based Excel interface supported for XLLs, and the managed .NET code used by F#. This part is what Excel-DNA provides. You won't be able to make an XLL purely in F#.

The best place to ask for help in getting started with Excel-DNA is the Google group.

Outlook will not sync any account with Outlook.com except for the explicit account you use to log into Outlook.com and then only if you set it up as an Exchange ActiveSync account (full sync of mail, contacts, and calendar) or IMAP account (mail only).  No other accounts you configure in the Outlook client will sync with Outlook.com.  If that's what you want, then forward those accounts to your Outlook.com address.

keep getting a pop up survey

why do i keep getting a pop up survey when I click on a  link, and more importantly, how do i stop it

a. user say:

Hi,

 

Is this survey appearing when you click on a link specific to one website or links to any website?

Have you recently installed new software?

 

To rule out malware, reboot your computer and begin tapping F8 key at the manufacturer's logo screen.

Select Safe Mode with Networking

Download the following tool and run a full scan

http://www.microsoft.com/security/scanner/en-us/sysreq.aspx

 

When finished reboot computer and let Windows load.

 

Launch IE and begin browsing websites.  Click links to determine if the survey reappears.  If it does reappear, do the following:

 

Tools > Internet Options > Privacy Tab

Click on Popup Settings Button

Under Blocking Level, select High

Click Close

Click Ok

Begin browsing websites as before clicking selected links.  Does the popup reappear now?

 

Remember to press Ctrl+Alt to override popup blocker settings if/when you wish to let a popup appear.
