mitch h. say:

Hi Anil,


We appreciate you sharing with us how you get to resolve the issue you were experiencing with your PC's performance.

Here are the things you want to know about disabling the services you've already disabled:

  • Disabling the Search service will stop the indexing of all files. You'll still have an access to search but this time, it will take longer since it has to search through your files every time. You may trim down the files that are getting indexed instead of disabling the said service.
  • The Diagnostic Policy service enables problem detection, troubleshooting, and resolution for Windows components. Stopping this service, diagnostics will no longer function and any services that explicitly depend on it will fail to start.
  • The Diagnostic Service Host is used by the Diagnostic Policy Service to host diagnostics that need to run in a Local Service context. Disabling this service, any diagnostics that depend on it will no longer function.
  • The Diagnostic System Host is used by the Diagnostic Policy service to host diagnostics that need to run in a Local System context. Disabling this, any diagnostics that depend on it will also no longer function.

Basically, disabling a service means a domino effect for the services that depends on it. It seems that all the services that you've disabled are related to diagnostics.
