Operate AutoFilter without VBA

I have a list in column S.  I have set an AutoFilter on this list.  If I click on the pull-down and clear the (Select All) checkbox, all the checks get cleared, but the OK button at the bottom of the Dialog is grayed-out.  This means I can't hide all the rows in the filter.

If I run:

Sub AbsolutelyNothing()
ActiveSheet.Range("$S$1:$S$85").AutoFilter Field:=1, Criteria1:=""
End Sub

all the rows are hidden.

How can I hide all the rows without a macro?
rick rothstein, mvp - excel say:
Click the "Text Filters" item in the top part of the list, select "Does Not Equal" from the flyout menu that appears and then put a single asterisk in the first blank field next to the "does not equal" field, in the dialog box that appears, then click OK.

rick rothstein, mvp - excel say:
Excellent!
Just a follow-up on my last message. I noticed that if there are blank cells in the filtered area, the "Does Not Equal" filter, by itself, won't hide them. To hide all row, with or without text, use the Custom AutoFilter instead of the Does Not Equal filter and set the first field as before ("does not equal" and *), then click the "And" option button and in the select line, chose "equals" from the first field and put an * in the second field... click OK now hides all rows in the AutoFilter'ed range.
