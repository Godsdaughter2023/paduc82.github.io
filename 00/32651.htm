"Devices and drives" blank file - Microsoft Community

stribor14 say:

"Devices and drives" blank file

I found similar queries around internet but there was no solution. Problem is when on Windows (usualy Windows 10), in Windows Explorer under This PC, appears blank file icon in "Devices and drives" and you can only delete, but it returns when you open WE again.

It happened when bluetooth device started to appear without being connected (non-existent device) and user cannot interact with it.

SOLUTION:

  1. Open registry editor (Win + R, regedit)
  2. (OPTIONAL if you still have existent device) search for that device name and delete whole Key from registry tree
  3. go to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\
  4. Export whole NameSpace Key
  5. Delete NameSpace Key (there will be an error msg and NameSpace Key will remain, but empty)
  6. Import it again

Windows will try to return all Folders, Drives and Devices, but as we deleted non-existent device and/or there is no device Key, it will skip import of "Blank file"
