We'd like to confirm if you're referring to the Windows 10 sign-in screen.

Since you've mentioned you're using your Outlook credentials to sign in to the desktop, this would require you to log in to your PC by entering the password of your account. This is by design.

To entirely disable this feature or the Sign-in screen itself, you'd need to change the type of account you're using on your desktop from a Microsoft account to a Local account. To do this, you'd need to remove your Microsoft account from your Windows 10 PC. Here's how:

1. Click the Start button.
2. Go to Settings.
3. Select Accounts.
4. Scroll down and tap on the Microsoft account you'd like to delete.
5. Click Remove.
6. Then, hit Yes.

For guidelines in creating a local user account in Windows 10, check our support page.

After following the advised steps mentioned above, disabling your computer sign-in screen is now possible. Follow these steps:

1. Press the Windows key + R. Run command netplwiz and hit Enter.
2. This will open the User Accounts window. Choose the one that you would like to skip the whole login and password option for. Uncheck the ‘Users must enter a user name and password to use this computer’ then click Apply to save changes made.
3. Restart your PC then check.
