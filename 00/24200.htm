jay freedman say:

The .docx file format is actually a zip file containing several folders and a number of files. You might be able to recover the additional text beyond what's in your last usable version:

  • Make a copy of the corrupted file on the hard drive, and work only on the copy.
  • In Windows Explorer, rename the copy to change the file's extension to .zip.
  • Double-click the zip file to open it. If Windows Explorer refuses to open the file, give up -- there's no way to un-corrupt a corrupted zip file.
  • If the file opens, you should see several folders. Open the folder named "word".
  • Extract the file named "document.xml" to a convenient folder. You can now close the zip file.
  • Double-click the document.xml file, which will open in whatever program your computer has associated with the .xml file type. That may be Internet Explorer or some text editor.
  • After a dozen or so lines of XML information, you'll see the text of the document interspersed with XML "tags" -- information enclosed in angle brackets. Copy the parts you need and paste it into a clean Word document.
  • Remove the tags by using the Replace function to replace them with nothing. For help with http://www.gmayor.com/replace_using_wildcards.htm.
  • Add formatting as needed, and then copy/paste into the old usable document.

In future, never edit a document directly from any removable media. Use your USB drive only for transporting files, not as permanent storage. For any document of value, always have at least one backup of the most recent version, and preferably two in different locations. Cloud storage (OneDrive, Dropbox, Google, etc.) is a good way to have a backup that you can access and update from any computer that has a connection.

jeeped say:
There is no formatting toggle like the Switch Case command, but within Word's fields there are formatting options for cardtext and dollartext. These can be created on-the-fly without resorting to Insert, Quick Parts, Field.
  1. On a new line press Ctrl+F9. This will insert a set of field braces (e.g. { and }).
  2. Type the following into the braces so it looks like this,
          { =123 \* cardtext }
      ... and then move the cursor outside the braces.
  3. Press Ctrl+A (e.g. Select All) and then tap F9 (e.g. Update fields)
You should have one hundred twenty-three on your document.
