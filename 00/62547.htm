Known Workaround for Win 8 and Server 2012:

Basically, Windows 8 and Server 2012, now has SMB2 Signing turned on by default unlike Windows 7. The BlueArc NAS currently does not support SMB2 signing yet.

Recommended Change:

The workaround is to disable “Secure Negotiate” on the Windows client side by running the following PowerShell command:

Set-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Services\LanmanWorkstation\Parameters" RequireSecureNegotiate -Value 0 -Force

Also make sure SMB Required Security Signature is set to false by running the following in PowerShell:

Get-SmbClientConfiguration –RequireSecuritySignature

RequireSecuritySignature              : False

Note: If SmbClientConfiguration is set to true and “Secure Negotiate” is disabled SMB/CIFS authentication will fail.

cmdrkeene say:

Hello again

Yes, you'll be fine with your license entitlement even if you remove all partitions (indeed even if you got an entirely new/blank hard drive, for example if you upgraded to a larger one, you'd still be okay).

When I clean install, there's a step near the beginning of the process that asks if you want to just install or perform a custom install.  If you choose custom, that's when it lists out existing partitions and asks where to install Windows onto.  When I get there, I usually remove every existing partition completely (I believe that step of setup has an erase icon for that function).  Then I just create one single partition (on that same page of setup, there's a 'new' icon for it).  I set it to the max possible size, then click Next.  The setup wizard at that point will actually create an extra very small partition (boot partition, usually 200 MB or less in size).

Sometimes cleaning it up after install in Disk Management is a bit hard (depending on their layout on the physical drive you may or may not be able to remove or resize the partitions), so if you could spare time to do one more install, I'd go through those steps above so it's pristine from the start. 

Or, if you aren't terribly worried about it, you can just go into Disk Management and remove the drive letter assignment for those partitions.  They won't show in File Explorer anymore, and at that point it's basically "out of sight out of mind".
