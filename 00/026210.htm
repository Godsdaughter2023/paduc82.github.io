shyam pillai say:

Hi,

To do that in PowerPoint you will need to edit the motion path information that is stored the path property for that animation. This information is a VML string consisting of line and bezier curve coordinates which define the motion

If msoAnimEffectPathArcDown is the 1st animation in the timeline then this will output the motion path stored for it. You can edit this string and update  the motion path. It would look like this  - M 0 0 L 0.067 0.04 C 0.081 0.049 0.102 0.054 0.124 0.054 C 0.149 0.054 0.169 0.049 0.183 0.04 L 0.25 0 E

With ActivePresentation.Slides(1).TimeLine.MainSequence(1)
    If .EffectType = msoAnimEffectPathArcDown Then
        Debug.Print .Behaviors(1).MotionEffect.Path
    End If
End With

You can read more about vml information.

bernie deitrick, excel mvp 2000-2010 say:

Select the cells, apply your desired formatting (anything but Text) and then run the macro below:

Sub SATextToFormula()
Dim myCell As Range
Dim myCalc As XlCalculation

With Application
   .ScreenUpdating = False
   myCalc = .Calculation
   .Calculation = xlCalculationManual
   .EnableEvents = False
End With

On Error Resume Next

For Each myCell In Selection
   myCell.Formula = myCell.Text
Next myCell

With Application
   .ScreenUpdating = True
   .Calculation = myCalc
   .EnableEvents = True
End With
End Sub

 

The other technique is to select the cells, apply the format, and then replace = with =.

Do a search for the "MSXML2.XMLHTTP" object.  I've used it to check the status of URLs and to return the content of a web page as a text stream but you're also supposed to be able to return a response as an XML document.

strURL = "http://www.microsoft.com/athome/default.aspx"
