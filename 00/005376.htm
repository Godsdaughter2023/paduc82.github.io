Does outlook import sent messages in addition to inbox messages from - Microsoft Community

garyjacobs say:

Does outlook import sent messages in addition to inbox messages from other email accounts and if so how do I find them?

Outlook is transferring my email from my verizon account. But I'm not sure it's importing my sent messages in addition to the messages in my in box.  Does outlook do this? And if so, how do I access my sent messages imported from verizon?

brian tillman [outlook mvp 2007-2019] say:
Since outlook.com can use only the POP protocol for getting mail from other mailboxes, it can get only what's in the Inbox of those other mailboxes.  POP has no concept of "folders".  If you wish to take the time, you can log into each of those other mailboxes, move the contents of the non-Inbox folders to the Inbox, allow outlook.com to pull the messages over, move those messages to a new outlook.com folder that you create, and repeat for the next folder in the remote mailbox.

brian tillman [outlook mvp 2007-2019] say:
Sure, if you did all the folders at once, but you do one folder at a time, let Outlook.com pull them over, move them to another foldet from the Inbox, then do the next folder on the remote system.  That way, only the folder you're working on is in the Inbox and the messages won't get mixed.

garyjacobs say: I'm not sure how to designate a folder on verizon.net to move over.  In importing email, outlook provides these four fields for me to fill in : email address, POP3 user name, password, and incoming mail server.   I used the default for the identity of the incoming mail server, which was "incoming.verizon.net".  I assume was this setting that was used to import the mail in my inbox.  So, to import the mail in my sent email box, do I set the incoming mail server to be "sent.verizon.net" or "sending.verizon.net"? 


brian tillman [outlook mvp 2007-2019] say:

Again, POP doesn't know anything about folders.  It accesses exactly one folder: your Inbox.  So, in the POP settings on outlook.com, you set the server to your incoming Verizon server.  You allow Outlook.com to pull all of the Verizon messages from that Inbox.  When you create the POP settings on Outlook.com, you can have it put the incoming messages in your normal Outlook.com Inbox or you can allow Outlook.com to put them in a separate folder, which will likely be called something like "verizon - yourverizonemail", where "yourverizonemail" will be your Verizon username. 

After all of the mail has been pulled from your Verizon Inbox to the folder you specified on Outlook.com, move them to another Outlook.com folder that you create so they won't get mixed up with any other mail that gets pulled from Verizon.  Now, log into your Verizon account with a web browser and copy (or move) the messages in one of your other Verizon folders (like Sent) to the Verizon mailbox's Inbox.  Outlook.com will think they're new messages and will pull them over to Outlook.com to the same folder it pulled the previous messages.  Now on Outlook.com, move those messages to some other folder, like "Sent" - either your Outlook.com Sent folder, a new Sent folder underneath the "verizon - yourverizonemail" folder, or some other folder of you choosing.

Do this same thing with each folder in your Verizon mailbox, switching between the web browser window where you're logged into your Verizon mailbox and the web browser where you're logged into Outlook.com.  Eventually, all your Verizon mail will have been moved or copied to Outlook.com.  From that point on, anything sent to your Verizon mail address will wind up in Outlook.com and you can file them where you like on Outlook.com.  You won't send from the actual Verizon mailbox again, but your Outlook.com account can be used to send messages whose sending address is your Verizon address, if you so choose.

If you call the TEXT property of a range you get the contents as they appears on screen.

Andreas.

  With Sheets("Sheet1")
    'Get the contents as they appear on screen
    strNewString = .Range("M7").Text & " " & .Range("N7").Text & " " & .Range("O7").Text
    'Remove double blanks
    Do While InStr(strNewString, "  ") > 0
