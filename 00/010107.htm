    tom van stiphout say:

    There appears to be a more fundamental problem: InvoicesToClear is not a table that should be in a normal invoices application. Rather it belongs in a query. What you are doing would create duplicate data which can get out of sync. Problems lay ahead if you don't conform to the relational model.

     

    Second: it appears you are abusing Group By and you're grabbing all possible fields. Rather such totals query should have 2 columns: InvoiceID (or whatever your PK is) and the DollarsToApply expression (note how I am avoiding spaces and other special characters which may cause notational complications). Then if you want your query to return additional columns, you use another query and inner join with InvoiceDetails and select them.

    john w. vinson say:

    There actually is only one Master Model control on the form, and onlyone Master Cage, and so on - each displayed many times. So changing the control's property changes all of the instances of the control!

    The solution is to not use any VBA code at all; instead, open the form in design view, select the MASTER MODEL, MASTER CAGE and MASTER SERIAL NUMBER controls, and use the Conditional Formatting menu option (in 2003) or ribbon tool (in 2007) to make the control enabled or not based on the value of the PO control.
    John W. Vinson/MVP

    marshall barton say:

    It sounds like your form is in continuous view.  If so, that is the normal behavior when you use VBA code to set properties for an unbound control.  There is only one control with one set of properties so a;; copies of the code appear the same.

    The only way I know of to get the effect you want on a continuous form is to remove that code and use Conditional Formatting (Format menu).  Use the Expression Is  option on the three controls with the expression:   [PO] = False  with the Enabled property set to off.

    As far as settings reverting it could just be Windows 10. I remember earlier builds had certain network settings defaulted and attempts to change them would either fail or be almost immediately reversed.

    Also, if you don't mind losing any installed Windows Programs, you can "refresh" your PC from settings, which resets most of Windows to it's default, clean-install state, but retains personalizations (Wallpaper, Sounds, etc.) and your personal files.
