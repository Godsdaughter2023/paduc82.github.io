Hi Spoilt,

 

As Mike suggested, connect the printer to Windows 7 Computer and try sharing the printer from there.

 

Here’s what you can do.

 

Steps:

 

a.    Connect the printer on Windows 7 computer.

b.    Now go to Start > Devices and Printers.

c.    Click on “Add a Printer”.

d.    In the new Windows, select “Add a local printer”.

e.    Click on “Create a new port”.

f.     Enter a port name:  \\xpmachine\printername (use the actual share names for the XP computer and printer share names).

g.    Click on Next.

h.    In the list of printers click on Windows Update, and then select the HP DeskJet 960C from the list.

 

That should be it.

When I close apps, it goes to the desktop instead of Start

When I close apps, Windows 8.1 used to return to the Start
screen, but now it’s sending me to the Desktop.



How do I change the
behavior so that it returns to the Start screen when closing apps?  

a. user say:

In order to return to the Start screen when you close an app, BOTH of the following settings must be de-selected:

  1. Right-click the Taskbar and select Properties.  In the Taskbar tab, make sure the “Show Windows Store apps on the taskbar” setting is unchecked.
  2. Then select the Navigation tab, and make sure the following setting is unchecked: “When I sign in or close all apps on a screen, go to the desktop instead of Start”.

Please be aware, however, that de-selecting the “Show Windows Store apps on the taskbar” setting will also have additional side-effects:

  • You will lose the minimize button in Windows Store apps
  • You will not be able to view the taskbar in the modern interface.

rohit chettri say:

A fix for this issue is included in the August 2014 update rollup for Windows 8.1 (KB 2975719).  After you install this Windows Update, the “Show Windows Store apps on the taskbar” setting will no longer cause you to return to the desktop when closing apps.

 

Taskbar and select Properties.  Select the Navigation tab and deselect the “When I sign in or close all apps on a screen, go to the desktop instead of Start” check box.

greg carmack - windows mvp 2010-2020 say:

Check the manufacturer's Support web page for your full model number to see if there is a battery issue or recall for it.

Run the battery recalibration here: https://www.howtogeek.com/172271/how-to-calibra...

Then generate a battery report and post it back here: https://www.groovypost.com/howto/generate-batte...

Right click Start to open Device Manager, under Battery choose the battery device, then Driver tab, then if available Roll Back the driver. If not available, choose Uninstall Driver, restart PC to reinstall.

There will be other drivers including chipset, ACPI, BIOS that might need updating that can affect this.

If this was caused by April Version 1809 update can roll back Version 1803 in Settings>Update & Security>Recovery and then hide it until it matures using the Hide Updates Tool downloaded from here: http://download.microsoft.com/download/f/2/2/f2...


If you don't have it yet then I'd also install Media Creation Tool and run Upgrade to Latest version 1809 which solves such problems and you need it anyway 

If this was caused by Windows Updates then you can check which were installed at Settings>Update & Security>Windows Update under Installed Updates, then uninstall them from the link there, and hide with the Hide Updates tool downloaded from here: http://download.microsoft.com/download/f/2/2/f2...

You can also use System Restore to get before the problem began, then check for Updates with the Hide Update tool and hide them: http://home.bt.com/tech-gadgets/computing/how-t...


If nothing else helps then you might need to replace the battery. However I will tell you that most OEM after-market batteries and chargers have problems so I would only buy the official manufacturer's battery from the PC maker to be sure.


I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
