How HoloLens will have to work to replace PCs - Microsoft Community

absorr say:

How HoloLens will have to work to replace PCs

Hello everyone!

I just thought that I would share some of my thoughts in hope of helping to shape the future of HoloLens.

Firstly, HoloLens looks like it is capable of replacing a laptop/tablet completely. Quite frankly, if it does not get this kind of functionality, it may not be taken in by society but rather be rejected like Google Glass or Surface RT because it is not as capable as they would like it to be.

In order to reach this level of capability, it needs to be able to do everything other Windows devices can do, including x32/x64 apps. Windows RT devices have been rejected because they look and feel like the full thing but because of its limitations make users feel restricted with everything they do and thus dislike the device. HoloLens is going to need to run full Windows 10, not an RT version, however it is looking like it's leaning towards RT.

In order to do this, it is going to need to meet the expectations of a full Windows 10 device, including hardware capabilities like Bluetooth and USB, as well as support for the use of traditional input methods (keyboard and possible mouse, but defiantly keyboard). Keyboards are going to absolutely without a doubt be a necessity because of the way it uses apps. People will not take kindly to a virtual keyboard. In fact, a lot of people prefer a physical keyboard over a touch version (myself included).

Until the day comes where this kind of technology is a standard, HoloLens is going to need to be capable of today's standard as well. In this case, that would be the desktop interface. The way I picture that this would work is that modern apps will work as they have been demonstrated thus far, however for the average consumer, as well as businesses, a need exists for a familiar desktop interface. This would have to function in a hologram window and act like Windows 10. Essentially, it would be like you pulled up your computer screen the same way you pulled up your movie (tv screen). From there, you could use a touch-interface using the already built in control system for HoloLens (which appears like the combination of Kinect and touch controls, which I believe is perfect), but also be able to use a keyboard and mouse. One of the reasons some people rejected Windows 8 was because of how much it changed without the rest of the world being ready for that (mainly a touch interface as opposed to keyboard/mouse), and Microsoft has seen this and has adapted in Windows 10 to rectify this.

So in conclusion, an interface for using a traditional computer experience is a necessity for HoloLens in order for the general public to take it in.

Thanks!
