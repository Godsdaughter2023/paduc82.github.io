From: http://www.checkupdown.com/status/E401.html:

The Web server (running the Web site) thinks that the HTTP data stream sent by the client (e.g. your Web browser or our CheckUpDown robot) was correct, but access to the URL resource requires user authentication 1) which has not yet been provided or 2) which has been provided but failed authorization tests. This is commonly known as "HTTP Basic Authentication". The actual authentication request expected from the client is defined in the HTTP protocol as the WWW-Authenticate header field.

Generally this error message means you need to log on (enter a valid user ID and password) somewhere first. If you have just entered these and then immediately see a 401 error, it means that one or both of your user ID and password were invalid for whatever reason (entered incorrectly, user ID suspended etc.).

Please verify with the web site that you have logged in properly (or registered or whatever they require - sometimes you even need to reply to an e-mail they send you before you are given authorization) and this problem should go away.

Method 1: Use the hardware troubleshooter to fix the issue

The Hardware Troubleshooter is an automated tool which will check the hardware connected to the computer for any known issues with it and provides the details and on how to fix them. Follow these steps and run the troubleshooter.

Follow these steps:

  1. Press Windows key to get the Start menu.
  2. Type Troubleshooting in the search box right up corner and hit enter.
  3. Click on Troubleshooting tab.
  4. Click on view all option on the upper left corner.
  5. Select the Hardware and Device options from the list.
  6. Click Next to run the Hardware and Device Troubleshooter.

If the issue persists, follow method 2.

Method 2: Update display adapter driver.

Step 1: Check for the latest driver update through Windows Updates.

  1. At the Start screen, type Windows Update.
  2. Click or tap Settings on the right side of the screen, and then select Install optional updates from the left results.
  3. When the Windows Update control panel opens, click on “Check for Updates” on the top left corner to receive a fresh list of updates.

Incompatible or not properly installed graphics card drivers are the most common cause of this issue.

Step 2: Uninstall and reinstall the audio drivers

I would also suggest you to uninstall and reinstall the latest audio drivers from the Manufacturer website.

Uninstall the driver from Device Manager:

  1. Locate audiodevices and expand the same
  2. and then uninstall the drivers.
  3. Reboot the computer after uninstall process has finished.

Now, please go to the manufacturer website and install

Restart the computer and check.

Step 3: Check in compatibility mode.

Download the drivers from the Manufacture and follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the driver from the manufacturer’s website.

b. Right-click on the driver setup file and click on ‘properties’.

c. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 8 operating system from the drop down menu and proceed with the installation.

d. Once this is done, restart the computer and check if the issue persists.

You may also refer: Make older programs compatible with this version of Windows (Steps applies to Windows 10)

Hope this information is helpful. If the issue still persists, please post back for further assistance, we'll be glad to assist you.

melourn say:

NOTHING ONLINE WILL HELP. PERIOD. DO NOT GO BACK TO PREVIOUS WINDOWS OS. DO NOT CALL SUPPORT. DO NOT TRY CHAT SUPPORT.

The ONLY thing that will work is the following:

1.) Find your local Microsoft Store (usually in a mall). [FREE if you have internet/phone service.]

2.) Find your Product Key. [FREE: usually on a sticker on the computer, on the CD provided, in the package for the Windows OS CD, or you can download a program from the internet that will investugate and "rip" the Product key from your console.]

3.) Back-Up all of your data to an external HD. [FREE if you already have an external HD.]

4.) Schedule an appointment. [FREE.]

5.) Attend your appointment and speak IN-PERSON with a tech agent. [FREE.]

6.) Sign the forms allowing them to complete the upgrade procedures. [FREE.]
7.) Wait until the procedures are complete (could be an hour or two- or it could be a week, depending on the situation). [FREE.]

8.) Retrieve and enjoy your ACTIVATED Windows 10 computer. [FREE.]

My situation changed slightly (resulting in the need to wait for a week to get my computer back) in that I tried the last-ditch resort where we "Go Back to Windows #". Unfortunately, this caused multiple drivers for the OS ad Hardware to be uninstalled and not re-loaded with the OS. When Windows 10 upgrades, it re-writes several key drivers for the new OS. In my case, this caused the keyboard and mouse to stop functioning outside of BIOS, after the Windows OS took control of the framework. Hence my warning: "DO NOT GO BACK TO PREVIOUS WINDOWS OS". Now, there were a couple of options available, but it was easiest to take the computer to the Microsoft Store and let them wipe it and install a fresh copy of Windows 10. As long as you have your Product Key, this is possible. ANOTHER thing that IS ENTIRELY POSSIBLE, is that if you provide your Product Key to the Microsoft Store Answer Tech, he/she can provide you with a NEW Product Key in place of one that is being rejected by the flawed detection system.

Enjoy.
