I want to search my index for all files NAMED "Authorization Letter" which CONTAIN the phrase "specialty store." 


For that particular search, enter
name:"Authorization Letter" content:"specialty store"

In general, you're looking for files with particular attributes. There are 300-odd different attributes in all, not all of them applicable to any particular type of file. To search for a file with a particular attribute, enter the query in the format
attribute:value

For multiple attributes, put them all in with a space between each entry:
attribute1:value1 attribute2:value2

Any entry string not preceded by attribute: will normally be treated as a search for files containing the string anywhere in the file contents or properties.

Another work around for Activation is to extend Activation Grace Period for another 30 days. You can do this up to 4 times (3 times for volume licenses) for a total of 180 days extension. After this grace period, you will need to make sure Windows is activated.
