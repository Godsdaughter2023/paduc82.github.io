connoroo say:

Whether this is to easiest way I do not know, but I have a result.  It seems to work so apologies if anyone if currently trying to assist.

Sub Build_Start_Act()

Dim LastRow As Long
LastRow = Cells(Rows.Count, "A").End(xlUp).Row
With Application
    .Calculation = xlAutomatic
End With
    Range("L2").Select
    Selection.FormatConditions.Add Type:=xlExpression, Formula1:= _
        "=IF(L2="""",K2<=TODAY())"
    Selection.FormatConditions(Selection.FormatConditions.Count).SetFirstPriority
    With Selection.FormatConditions(1).Interior
        .PatternColorIndex = xlAutomatic
        .Color = 255
        .TintAndShade = 0
    End With
    Selection.FormatConditions(1).StopIfTrue = False
    Range("L2").Select
    Selection.Copy
    Range("L3:L" & LastRow).Select
    Selection.PasteSpecial Paste:=xlPasteFormats, Operation:=xlNone, _
        SkipBlanks:=False, Transpose:=False
    Application.CutCopyMode = False
End Sub

john mcghie say:

First things first:  Update it!

Your crash dump shows Word is at version 12.0.0 Build 071130 -- that has never had any updates, it's the version on the CD.

To run with OS 10.6.8, several updates are required.  The current version is 12.3.6, so you are three whole revisions behind.

Go to the Microsoft Download Centre (where all the legacy downloads are hiding) at http://www.microsoft.com/en-us/download/search.aspx?q=microsoft%20office%202008%20for%20mac&p=0&r=10&t=&s=availabledate~Descending

You need to first put in 

Microsoft AutoUpdate for Mac 2.3.6 Update


Then 

Microsoft Office 2008 for Mac Service Pack 1 (12.1.0)

Then 

Microsoft Office 2008 for Mac 12.3.6 Update

Do them in that sequence, or they will probably blow up on you.  Theoretically, when you get the AutoUpdate patch in, it will go get the rest for you, but since you are already on the download page, I would just grab them while you are there.

After those updates, it would be a good idea to Repair Permissions http://word.mvps.org/Mac/DiskIssues.html and Weed your Fonts: http://word.mvps.org/Mac/fontweeding.html

Don't bother with the rest of the Disk Issues page, just repair the permissions.

Then you MUST re-start the system, all the way down to power-off.  Logging out is not enough: you need to force a rebuild of all the caches.

Come back if it still won't go after all of that.
