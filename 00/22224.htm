scottgem say:

Set the Default value of the number control on your form to:

=DLookup("[fieldname]")+1

That will assign the next number to the current record.

Then you will have to run an Update query:

UPDATE tblCount SET fieldname = Forms!formname!controlname;

To update tblCount with the new value. The kicker is when you do this. If you don't do it immediately after assigning the number to the new record, then someone else can wind up with the same number. So you may not want to assign it as the default, but use some other event to assign the new number and then immediately update tblCount.
