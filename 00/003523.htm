jay freedman say:

I haven't been able to make this happen in my copy of Word 2010, but I'll guess that the Remove Hyperlink command may sometimes fail to remove the Hyperlink character style that was applied when the hyperlink was created.

To remove the formatting from one location, select the blue underlined text and press the Ctrl key and the spacebar together. 

If there are a lot of former hyperlinks, you can install and run the following macro. It will remove the Hyperlink formatting only from text that is not a hyperlink, leaving any real hyperlinks still formatted.

Sub ReformatExHyperlinks()
    ' if text is not a hyperlink but
    ' is formatted with Hyperlink style,
    ' remove the blue and underline
    Dim rg As Range
    Set rg = ActiveDocument.Range
    With rg.Find
        .Format = True
        .Style = ActiveDocument.Styles("Hyperlink")
        While .Execute
            If rg.Hyperlinks.Count = 0 Then
                rg.Font.Reset
                rg.Collapse wdCollapseEnd
            End If
        Wend
    End With
End Sub

  • Press the Windows button + X on your keyboard, a mini dialogue box will appear

How do I repeat the header row in a table? - Microsoft Community

frustratedwithmsword say:

How do I repeat the header row in a table?

Split from this thread answered 6 years ago.

Ok, so I can either have a table with no repeating header row, or a table which has the header all on it's lonesome on the first page, followed by correctly repeating header tables on subsequent pages. I've created a new table from scratch in the same document, still the same issue. 

AAAARGH!

john korchok say:

It sounds like the style used for the table body has a paragraph attribute set to Keep With Next. That makes the rows underneath the table header jump to the next page, leaving the heading by itself on the first page. To fix:

  1. Click inside the table body.
  2. On the Home tab, a style in the style gallery should be highlighted with a blue border. Right-click on that style and choose Modify.
  3. In the Modify Style dialog, check Add to template. This will apply the change to the template from which your document was created, to eliminate this issue in the future.
  4. Click on the Format button and choose Paragraph.
  5. Select the Line and Page Breaks tab.
  6. Uncheck Keep with next. If Keep lines together is checked, uncheck it. OK out and your table should pop back to its correct display.
