Try something along these lines:

Before or just after making a sliderange visible, do this:

Call UpdateLinks(ActivePresentation.Slides.Range(Array(  [whatever] ) ))

Sub UpdateLinks(oSlideRange As SlideRange)

    Dim oSl As Slide
    Dim oSh As Shape
    
    For Each oSl In oSlideRange
        For Each oSh In oSl.Shapes
            If oSh.Type = msoLinkedOLEObject Then
                oSh.LinkFormat.Update
            End If
         Next
    Next
    
End Sub

This assumes your links are all linked OLE objects.

As formula:


=IF(A1<90%, 0, IF(A1<92%, 0.5, IF(A1<95%, 1, 2)))


where A1 is the other cell you want to refer to.

Alternatively:


=(A1>=90%)/2+(A1>=92%)/2+(A1>=95%)


For the colors, you can use conditional formatting:

Select the cell with the formula.


On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Select 'Format only cells that contain'.

Leave the first dropdown set to 'Cell Value'.

Select 'equal to' from the second dropdown.

Enter 0 in the box next to it.

Click Format...

Activate the Fill tab.

Select red, then click OK.

Click OK again.


Repeat the above steps but with 0.5 instead of 0, and orange instead of red.

Also with 1 and yellow.

And finally with 2 and green.

locking and unlocking workbook - Microsoft Community

chrisodum27 say:

locking and unlocking workbook

When my workbook is opened, I want a message box to pop up and allow the user to select if they want to edit or just read the workbook. If they select edit it will prompt them for a password then unlock certain ranges on each worksheet. If they select read only, the workbook will stay locked.

Then when the workbook is closed, I would like to lock all the cells again. Can anyone help me with this?

bernie deitrick say:

1) Copy the code below
2) Open the VBE (press Alt-F11)
3) Select your workbook in the project explorer
4) Double-click the ThisWorkbook object
5) Paste the code into the window that appears.
6) Save the file as a macro-enabled .xlsm file.

Change the ranges and sheet names as needed, as well as the passwords. You will want to protect the VBA project with a password too - not great security, but better than nothing.

Private Sub Workbook_Open()
    Dim sht As Worksheet

    For Each sht In ThisWorkbook.Worksheets
        ThisWorkbook.Unprotect "Password"
        sht.Unprotect "Password"
        sht.Cells.Locked = True
        sht.Protect "Password", UserInterfaceOnly:=True
    Next sht

TryAgain:
    If MsgBox("Edit the workbook?", vbYesNo) = vbYes Then
        If InputBox("Enter Editing Password") = "EditPassword" Then
            ThisWorkbook.Worksheets("Sheet1").Range("A1:A10,C12:C20").Locked = False
            ThisWorkbook.Worksheets("Sheet2").Range("G45:47").Locked = False
            ThisWorkbook.Worksheets("Sheet3").Range("M1:P10,R1:Z20").Locked = False
        Else
            MsgBox "Sorry - that is not the correct password!"
            GoTo TryAgain
        End If
    End If
End Sub

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    Dim sht As Worksheet
    For Each sht In ThisWorkbook.Worksheets
        sht.Cells.Locked = True
    Next sht
    ThisWorkbook.Save
End Sub
