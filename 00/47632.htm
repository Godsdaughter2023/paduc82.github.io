To back up e-mail in Windows Live Mail, please follow the steps below:


  1. Start Windows Live Mail.

  2. Click on the File tab (Alt + F), click Export email and then select Email messages.

  3. Select Microsoft Windows Live Mail as the export format, and then click Next.

  4. Click Browse and then select the folder in which you want to store the exported messages. IMPORTANT:

      The export directory must be an empty directory.

  5. Click Next.

  6. Select the message folders that you want to export, and then click Next. You will receive a message that states that the messages were successfully exported.  Click Finish.

If you have a backup of the emails from the Windows Live Mail of your old computer, you can transfer it to your new computer and import it on your new instailled Windows Live Mail.

To import your emails, please follow the steps below:


1. Start Windows Live Mail.

2. Click on the Filetab (Alt + F), click Import email and then select Windows Live Mail then click Next.

3. Click Browse and look for the email that you want to import.

Message status is recorded in the message database, not in the message file itself. Status like flagged or replied will not be retained during export/import.

I have not tested this and don't know if it will work, so proceed at your own risk.

If the version of Windows Live Mail on the old drive is the current one (15.4.3555.0308) on the same operating system, you may simply be able to transfer the store folder en bloc to the new drive. I would do this before installing Windows Live Essentials on the new drive; the installer should look for an existing store folder and take it into use. You will be asked to re-enter any stored account passwords, because these are encrypted using a key based on hardware and software. This may be the least of the problems you face after trying this exercise.

On the old drive, find your store folder. You can see its location in WLMail by pressing Ctrl-Shift-O for Options. On the Advanced tab, click Maintenance and then Store folder. Note the path and folder name. Then copy that entire folder to exactly the same path on the new drive. Install Windows Live Essentials, choosing only the bits you want.

If the program version on the old drive is not the latest, you may still be able to do this if you're able to download and install the latest version on the old drive.

If the operating system on the new drive is different from that on the old one, all bets are off.

Good luck!
