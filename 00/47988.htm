a. user say:

A laptop that you purchase directly from a computer manufacturer does not have a profile already setup.

When you press the 'power 'button for the first time, the factory image is installed on the hard drive and Windows starts for the first time. As Windows is setting itself up, it asks you to create a user profile. Setup won't continue until you create a profile. A profile can't create itself.

The same setup procedure occurs when you buy a used computer from a computer manufacturer.

If your computer already had a user profile when you first powered on, you should be concerned.
