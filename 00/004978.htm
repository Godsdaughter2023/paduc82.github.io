Hi Slawomir,

 

You can right-click a message from the sender you want to block, and then click Junk > Block Sender. The blocked person can still send you mail, but if anything from his or her email address makes it to your email account, it’s immediately moved to the Junk Email folder. Note you cannot block a sender from your organization.

 

If you want to check the sender’s domain, you can try the followings, this can show sender’s domain in Mail list of Outlook client:

1. Open the mail folder in which you will view or display the senders’ email domains;

2. Click the Add Columns button on the View tab;

3. In the coming Show Columns (or Show Fields) dialog box, click the New Column button (or New Field button);

4. Now you get into the New Column (or New Field) dialog box, and:

(1)In the Name box, enter a new name for this creating column.

(2) Click the Type box, and select the Formula from the drop down list.

(3) Click the Edit button, and then enter the following text into the Formula box of dialog box. See screen shot below:

right([SearchFromEmail],len([SearchFromEmail])-InStr(1,[SearchFromEmail],"@"))

(4) Click both OK buttons to close two dialog boxes.

5. Now you get back into the Show Columns (or Show Fields) dialog box, select the new created column in the Show these columns in this order box, and move it below From.;

6. Click OK to finish.

Now you will view each sender’s email domain is added and shown below the sender’s displaying name.

Outlook stores the last downloaded information in the pst file. When you create a new profile on a new computer, outlook redownloads everything. This should only happen once for the default account.

Outlook 2010, due to changes in how the "message manager" is handled, may download messages over and over if you deliver multiple accounts to one pst file. To fix, create a separate pst file for each account. In file, account settings, select the account then Change folder button to change the delivery location to a new pst.  http://www.slipstick.com/outlook/outlook-2010/leave-server-setting-multiple-email-accounts/

I need two separate bibliographies in one document - Microsoft Community

vilden say:

I need two separate bibliographies in one document

Hi,

I need a bibliography for books, papers, etc. and a bibliography for Internet references in a single document. How do I do this?

Thanks! 
yves dhondt say:

Are you using some bibliography tool that didn't came with Word? For example: EndNote. If so, your best shot is to try the forums/support of that product.

Have you typed out your bibliography by hand? Unlikely, but possible. In this case you will have to be creative with cutting and pasting and do things yourself.

Have you used the Word citations and bibliography tools? If so, then there is no direct way to achieve what you want. Two possible solutions are:

Create one global bibliography. Once finished, use the little blue arrow in the surrounding box to convert the bibliography to static text. Now cut the parts you don't want in the first bibliography and paste them in the second. This is the most straightforward solution. It does require some work, but anyone could do it and is basically the same as if you typed the bibliography by hand and needed it sorted.

Change the sort string. This one is trickier and requires some knowledge of xslt. When Word generates a bibliography it sends all the sources in xml format, together with one of the XSL files from the style directory to the transformation engine. The XSL file contains information on how to sort the items in the bibliography. What you could do is write a sorting mechanism that sorts based on the type of source. That way, all internet references could be grouped together.

The XSL file responsible for the formatting of your bibliography is located at

<winword.exe>\Bibliography\Style

on most machines with a default installation that is

C:\Program Files\Microsoft Office\Office14\Bibliography\Style 

where Office14 indicates Office 2010 and Office12 would indicate Office 2007.

Assuming you are using one of the styles that came with Word 2010 and not a custom one, the sorting is done based on ab:SortingString element. Changing the value of that element to fit your needs should be enough. If you use a custom style, then you have to figure it out yourself or ask the creator of the style.

If you have at least basic xml knowledge, post a reply and I will provide more detailed instructions on how to proceed if you want. If you don't have an idea what xml is, then don't bother with this solution. Yes, you would be able to do this faster than with the first solution, but it would be a lot more likely that things get messed up.
