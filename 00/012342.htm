neha singh msft say:

Hi KazuTak,

In Excel, if we click Compress image and select the Use original quality option and then save the file, it does not change the size when checked in Format Pane. It only changes the size when we change the scale height and width and then press save. If we change the scale after compressing, that means we are changing the original size as well. Reducing the size of an image will result in a lower quality picture.

If you want to keep the original quality, then do not resize the image.

There is nothing built in that will capitalize the word after a dash. You're probably thinking of the AutoCorrect option (in Word Options > Proofing > AutoCorrect Options) to "Capitalize first letter of sentences". However, that option recognizes only periods, question marks, exclamation marks, and paragraph marks as the ends of sentences, indicating that the next word starts a new sentence.
