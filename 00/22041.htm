bobgreenblatt say: It sounds like you already know the problem: "change text in the cells e.g. from numbers to ...." If the values in the cells are indeed text, Excel treats them as such and they can not be formatted. To convert the text back to values, enter a 1 in any spare cell. Copy it. then select all the text cells that should be numbers and go to Edit-paste Special and select multiply. Now the values should be numbers instead of text and can be formatted as needed.
