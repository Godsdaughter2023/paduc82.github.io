    Depending on how many bookmarks you have in Microsoft Edge you can if you wanted to spend the time, open up every bookmark in Microsoft Edge one by one and in the options menu (the one with the three dots ...) click that for each webpage and select "Open with Internet Explorer" and once it is opened in Internet Explorer, save the favorite there.

    You may or may not know this but Microsoft Edge and Internet Explorer store their Favorites in different locations on your computer. It wasn't so long ago that you could open both folder locations and just copy favorites from one folder to another to sync both browsers so that they both had identical copies of favorite websites.

    Recently though, I am not sure when, Microsoft changed how Microsoft Edge stored it's Favorites; they are no longer loose icons like Internet Explorer's is, Microsoft Edge's Favorites are now stored in a single database file. This means you cannot simply copy from one folder to another anymore.

    My suggestion at the beginning of my reply is the best way I know of for you to (so called) "transfer" them.

    EDIT: Another possibility is to use the Firefox browser. Download it and install it if you haven't done so already and use Firefox to import your Favorites directly from Microsoft Edge. Then after you have them inside Firefox you can save them (export them) to an HTML file and then import that HTML file into Internet Explorer.

    I hope this helps.

    ashish mathur say:

    Hi,

    Suppose you have data in range A2:B8 of sheet1.  In range B2:B8 you have x.  In cell A2 of sheet2, enter this array formula (Ctrl+Shift+Enter) and copy down

    =IFERROR(INDEX(Sheet1!$A$2:$A$8,SMALL(IF(Sheet1!$B$2:$B$8="X",ROW(Sheet1!$A$2:$A$8)-ROW(Sheet1!$A$1)),ROW(1:1)),1),"")

    Hope this helps.
