a. user say:

The simplest way I've found is to use an Explorer to access the files in the menu tree and move them around.

Click your Start orb, then RIGHT-click on All Programs.  You will find you can choose Open or Open All Users.

Keeping in mind that the Start Menu is created from a combination of the folders and files you see at the two locations identified above, now you can open one or both of them, and easily reorganize your Start Menu by dragging (moving) things around and making it easy for you to find things.

If the same folder name (e.g., Applications) appears in both the All Users menu structure and the User menu structure, only one entry will appear in the Start Menu.

As time passes, and you install things, you'll find they often install their shortcuts and/or folders right in the All Programs menu or somewhere you don't want.  No problem, just open the All Users menu structure and/or the other one as listed above.  Move the program's menu entries to where you like.  Personally I like to keep the base menu pretty clean, and move entries for newly-installed programs to my Applications subfolder.

If you'd like to make it even easier, I've listed how to create two new menu items to help with this in this thread:
