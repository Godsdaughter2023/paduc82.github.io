tommcgtx say:

Netgear WGT624 v3 problems

I recently tried to connect a Netgear WGT624 v3 router to my computer with windows 7. When connected, I can't use the internet, and the router just continuously restarts itself. I tried to access the router's config. through Internet explorer, as suggested by Netgear's website, but that doesn't work either. The disk that came with the router, doesn't take me into a setup like a tutorial video on Netgear's site showed. I know next to nothing about this sort of thing, so any help would be greatly appreciated. The diagnostic that window's performs to correct this doesn't work, and says something about not having a valid IP address.
chris - support engineer say:

Hi tom,

In reading your original post, I was wondering if you could clarify a few things?

" the router just continuously restarts itself" 

How do you know the router is restarting? Is this from something you're seeing in Windows or the device its self?

If you're seeing this behavoir in the device, you may want to try reseting it or replacing it.

"I tried to access the router's config. through Internet explorer, as suggested by Netgear's website, but that doesn't work either."

Are you using a wired connection when attempting this?

You should be able to access this page even if your not getting to the internet, if your router is using DHCP.

Are you using static IPs in your network setup? I imagine not if you are unable to access the routers setup page.

However, the IPconfig is showing what appears to be a custom gateway address, as I believe the default is 192.168.0.1

Are you using a cable modem that has a router built-in? It would look similar to your Netgear, in that you can plug several computer into it.

I ask this because if the modem from your internet provider has DHCP enabled on the modem, it could conflict with the Netgear trying to perform the same function.

Here's some information on what DHCP does if you don't already know:

http://documentation.netgear.com/reference/sve/tcpip/TCPIPBasics-3-14.html

Here's my suggestion:

  1. Remove the Netgear router from all other devices, computer and modem alike.
  2. Reset the router using whatever method Netgear suggests, usually pushing in a small button on the back of the device.
  3. Connect one computer to it using a network cable that is known to work.
  4. Attempt to access the routers setup page by typing the default IP address into your browsers address bar.

The default IP should be 192.168.0.1 based on this thread at the Netgear forums, concerning a similar issue:

http://forum1.netgear.com/showthread.php?t=11658

Accessing that IP in your browser should yield a login window.
The username and password for that should be in the Netgear manual.
