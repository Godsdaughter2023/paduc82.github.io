Sub superMe()
    Dim otr As TextRange
    Dim otemp As TextRange
    Dim osld As Slide
    Dim oshp As Shape
    Dim Cr As Long
    #If Mac Then
    Cr = 168
    #Else
    Cr = 174
    #End If
    For Each osld In ActivePresentation.Slides
        For Each oshp In osld.Shapes
            If oshp.HasTextFrame Then
                If oshp.TextFrame.HasText Then
                    Set otr = oshp.TextFrame.TextRange
                    Set otemp = otr.Find(Chr(Cr))
                    If Not otemp Is Nothing Then
                    otemp.Font.Superscript = True
                    End If
                    Do While Not otemp Is Nothing
                        Set otemp = otr.Find(Chr(Cr), After:=otemp.Start + otemp.Length)
                        If Not otemp Is Nothing Then
                        otemp.Font.Superscript = True
                        End If
                    Loop

charles kenyon say:
There is no simple or automatic way to do this in Word. The closest I can imagine involves inserting a bookmark just before your TOC.

Then you could hyperlink your Chapter headings back to that, retaining their native format rather than hyperlink format. A macro could do this.

Otherwise, you could, following each Chapter Heading insert a Cross-Reference to the bookmark. That could also be done with a macro.

With either method, you would want to only run the macro once, after you have finished your editing. If you insert the cross-reference as a separate line, you would need to update page references in the TOC. Except, I realize, on Kindle, there would be no page references.
