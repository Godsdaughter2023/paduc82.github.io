ricw-nw say:

You can  rip CDs with (not to) WMP the same way as in previous versions of Windows. Ripped music will then appear in your WMP library.

It may be that default settings are not giving you this option at present. (MS are pushing Groove Music in W10 and that has no rip option.)

Go to Control Panel - Programs - Default Programs - Set Default Programs

Wait for the left-side pane to populate and then scroll down to WMP - highlight and select 'set this program as default' then click OK... the rip option should be available when you insert and open a disc.
