Is there some other Access solution I can utilize?  Thanks in advance for any guidance.
hansv mvp say:

You *can* use FileDialog to display a Save As dialog in Access 2010 (but not in earlier versions).

You cannot clear or add filters, however. The "Save as type" dropdown contains "All files (*.*)" as only option.

 

Sample code:

 

Sub TestFileDialog()
    Dim strFilename As String
    With Application.FileDialog(msoFileDialogSaveAs)
        If .Show Then
            strFilename = .SelectedItems(1)
        Else
            MsgBox "No filename specified!", vbExclamation
            Exit Sub
        End If
    End With
    ' Do something with strFilename
    ...
End Sub

rekki---01 say: Rekki's guide for making Gizmo VoIP over WLAN with the Nokia E71

Points to note:

- My E71 is configured to use English, and has firmware 100.07.76 (08-06-2008)
- I do not have an operator variant of the E71, so the menus may be different in your E71 if you have an operator variant of the phone.
- My E71 has the needed settings (including security) to use my WLAN access point.
- I reset my E71 to the factory settings, so I know for the purposes of this "guide" that I don't have any conflicting settings.
- Having WLAN always active will in theory consume more battery power. That said, I've had a Nokia E51 without sim card, configured to use another Gizmo account. It's been active on WLAN for a week now without charge. Ok, I haven't made any calls but test calls ;)
- "Disclaimer": The following steps work for me, and might not work for you..


Steps:


1) So, having reset the device and configured it to use my WLAN access point, I can see from:

Menu: Tools --> Settings --> Connection --> Access points

..that my phone has automatically recognised the required settings for my operator plus the wlan ssid (network name).

I also know that the WLAN is working from my phone because web browsing is working from the phone, not via the operator but via the WLAN network.

2) I can also see from the following menus:

Menu: Tools --> Settings --> Connection --> SIP settings
Menu: Tools --> Settings --> Connection --> Internet tel.

..that there are no SIP settings nor Internet telephone settings as of yet.


3) So let's install Gizmo, the first step, start the phone browser (Menu: Web) and browse to http://www.gizmovoip.com.


4) The first choice on that page is basically if you have an N-series device, or E-series device.. I have an E71, so naturally "Nokia E Series" is my choice.


5) Next, as I already have a Gizmo account, I will select option "1 Install Gizmo VoIP".

If you don't have an account, here's your chance to create one by selecting "2 Create Gizmo Account". I don't know where the pages take you after that, but you should be able to "Install Gizmo VoIP" and continue with the following steps.


6) Next, select "Download now"


7) When the phone asks you "Install Gizmo VoIP?", well you select "Yes" then "Continue" to continue.


8) After installing the Gizmo client, the E71 the very promptly shows "Discovering services" and asks you to confirm "Configure Gizmo internet call service?", select "OK"


9) Now it asks "Do you have a Gizmo account with User ID and Password?" .. Select "Yes"


10) Enter your username and passwords :) When finished, press "Done"..


11) After briefly showing "Configuring services" or "Discovering services", the E71 should display in the upper right corner "Not connected" and proceeds to show you a list of WLAN's in the area. Select your WLAN network, then "Options" and "Connect to service". The E71 should show "Connecting to GIZMO via......" followed by a message "If emerg. no dialled outside cell network......etc etc".. read and understand this message, and press "OK".

12) The device then asks you "Save network [your network] for GIZMO VoIP service?" .. so if you want to use this WLAN network for Gizmo in future, then select "Yes". After pressing "Yes" a brief message "Network saved for Gizmo VoIP service" appears. Now you should see your WLAN network name with a star next to it. This star simply signifies the WLAN networks configured for use by Gizmo VoIP.

Note: At this point, you may receive an automated "Internet call" from the GIZMO services, to confirm the successful setting up of your device.. don't worry if you don't ;)

13) Now from the standby mode, you should see two new icons below the battery indicator, one is to signify you are connected to your WLAN, and the other (looks like a telephone's handset next to the Earth) is to signify you are connected to a SIP VoIP service, in this case Gizmo.

**Gizmo is now configured**

Without going into the tricks and shortcuts of E71, one way to stop and start the Gizmo service running is to follow the menus:

Menu: Communcations --> Internet tel.

..now, you will see the WLAN networks in the area and in use for Gizmo. Selecting "Options" then "Disconnect from service" will stop Gizmo VoIP. Naturally, selecting "Options" then "Connect to service" will reconnect you.



If you want to check the settings manually now, follow the menus:

Menu: Tools --> Settings --> Connection --> SIP settings

Here it should show "GIZMO VoIP" and then "Registered"

note: you cannot edit/view the settings whilst the profile is in use..

Menu: Tools --> Settings --> Connection --> Internet tel.

Here it should show "GIZMO VoIP".

suzanne s. barnhill say:
Word 2013 and above do not honor tab settings beyond the right margin. Initially, Word 2013 also ignored tab settings beyond a right indent, but this was corrected by an update. The hitch is that, although you can drag the indent marker on the ruler, you can't do it without dragging the tab stop with it, and then you can't move that tab stop back or set a tab stop to the right of the indent; you have to do that through the Tabs dialog. And some users have found that a tab stop exactly at the right margin will not work; by default, Word sets the tab stop for the page numbers at .01" left of the margin (7.49" for a 7.5" line), so, to be on the safe side, you should do the same.
