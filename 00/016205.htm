jakenyg25 say:

Messages not loading past a certain date?

None of the messages in any of my chats load before April 20th, id like to see them. Any way to fix this?
techfreak_ say:

Skype only stores ~30 days worth of history in the cloud.  When you use Skype it does this automatically. There is nothing you need to do. Older history is 

Any history beyond 30 days is present in a main.db file located in YourSkypeName folder (whatever that happens to be) found in %appdata%\Skype (i.e. %appdata%\Skype\YourSkypeName) on your comptuer. When history is missing it generally means you are using a different computer, your computer was reloaded (deleting the data), or the main.db file is partially or fully corrupt resulting in history that is no longer retrievable from that corrupt file.

If you made any system-wide or Windows profile-wide backups in the past, you may have a copy of a main.db file you could restore. If you did not maintain any such backups, then the data would not be available from Skype. If you used other computers or devices you may want to check to see if their main.db files contain the old history you are looking for.

You can use 3rd party software like SkypeLogView [nirsoft.net] to browse/extract the contents of any main.db files you find on your computers.

daniel g. grau say:
1) Leave them as separate files, but view them in a combined form from a Master Document, then print as a PDF (as long as there are no page-orientation changes, you'll get a single PDF file). See:
  • Create a master document and subdocuments
  • Master Documents without pain
2) Combine PDFs with Apple's Preview application, which you'll find in your Applications folder. This method will work even if the documents have various page orientations or page sizes. (Word cannot create a single PDF file with the current into-PDF converter as supplied by the OS.)
  • Open your PDF documents in Preview.
  • Display their Sidebars (View > Sidebar).
  • Click the curved arrow to expand the document and access individual pages.
  • Select pages in the source document (Shift-Click for contiguous selection, Command-Click for disjoint selection) or whole documents.
  • Drag the document or page icons from the source Sidebar onto the target's Sidebar. (If you drag whole documents, you'll notice that Preview allows two different target locations for the horizontal bar shown below in purple. The lowermost will just merge views, the uppermost will merge documents.)
  • Drag page icons to alter page order.
