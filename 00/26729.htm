john sr wilson say:

A possible problem is the code would not deal with pictures inside placeholders.

here is a slight variation:

Set one picture as you wish and make sure it is selected then

Sub AddPictureBorders()
' Before running this, apply the style you want to one picture
' and make sure it is the only shape selected
Dim oSh As Shape
Dim oSl As Slide
On Error GoTo err
ActiveWindow.Selection.ShapeRange(1).PickUp
For Each oSl In ActivePresentation.Slides
For Each oSh In oSl.Shapes
With oSh
   If .Type = msoPicture Then .Apply
   If .Type = msoPlaceholder Then
   If .PlaceholderFormat.ContainedType = msoPicture Then .Apply
End If
End With
Next oSh
Next oSl
Exit Sub
err:
MsgBox "Error, did you select a picture?", vbCritical
End Sub
