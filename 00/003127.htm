The Same As Previous option (called Link to Previous in some versions of Word) can be used to link a header (or a footer) to its counterpart in the preceding Word section. hansv mvp say:

1) Select cell A1.

Select Data > Data Validation.

Select Custom from the Allow dropdown.

Enter the following formula in the Formula box:

 

=WEEKDAY(A1)=5

If you wish, you can enter an instruction in the Input message box.

ronrosenfeld say: I don't recall where the items are on Excel 2003, but the following should work:

For the data entry into A1, you can use Data Validation.  The entry should be "Custom" and the formula would be:    =WEEKDAY(A1)=5
Adjust the various messages to suit.

For the conditional formatting, a formula which shows there are three PayDays in the month would be:

=SUMPRODUCT((PayDates>(B1-DAY(B1)))*(PayDates<=(B1-DAY(B1)+32-DAY(B1-DAY(B1)+32)))) =3

If that is listed as the first CF condition, processing should stop when those conditions are met, so it should "overwrite" the alternate line shading CF you have already set up.

PayDates is the range where your formulas are.  e.g. $B$1:$C$n

hansv mvp say:

2) I'm afraid you'll have to redo the conditional formatting - you'll want the conditional formatting for months with 3 Thursdays to have precedence above the alternating colouring, so the new rule should be at the top.

 

Select the cells in column B that you want to format conditionally.

Select Format > Conditional Formatting.

Remove the existing rule.

Select "Formula Is".

Enter the following formula, adjusting $B$1:$B$27 the range as needed:

 

=SUMPRODUCT((YEAR($B$1:$B$27)=YEAR(B1))*(MONTH($B$1:$B$27)=MONTH(B1)))=3

 

Click Format etc.

 

After this, you can add the rule for alternating colouring again.

sandy johnson say:

This is an ongoing problem with newer versions of PowerPoint. After pasting the slide, make sure any invisible (empty) placeholders from the "old" presentation are deleted (select CTRL-A) to find them. Then go into Insert > Header/Footer to check the appropriate boxes. Hit Apply (Apply to All doesn't work all the time and you're likely to have to repeat on each pasted slide, anyway).

 

The alternative is to insert a new slide in the new presentation and to copy and paste only content from the old presentation. A hassle, I know, but you'll avoid the Footer issue.

Excel 2010 VBA in Chart: Set hilolines to different colors - Microsoft Community

jimgdal say:

Excel 2010 VBA in Chart: Set hilolines to different colors

I would like to set the hilolines in a Chart on each x axis (xlcategory) to a different color.

I have created VBA code below with a simple example. As in the Sub HiLo() code below, I can set the color of all lines to red, but, for example, would like to set the 2 hiloline to green, and the 3 hiloline to blue.

Using Excel help, web searches, etc., I can not find info to do this?

Can anyone help with info on how to do this.

Thank you.

Sub HiLo()
ActiveSheet.Range("A1:B4") = [{"High","Low";5,3;7,2;3,1}]
Set ch = ActiveSheet.Shapes.AddChart(xlLine, 100, 10).Chart
ch.SetSourceData Source:=ActiveSheet.Range("A1:B4")
ch.SeriesCollection(1).Format.Line.Visible = msoFalse
ch.SeriesCollection(2).Format.Line.Visible = msoFalse
ch.SetElement (msoElementLineHiLoLine)
With ch.ChartGroups(1).HiLoLines.Format.Line
    .Visible = msoTrue
    .ForeColor.RGB = RGB(255, 0, 0)
End With
End Sub
