techfreak_ say:

You did not state what region the number you are dialing is in. You may be providing an unnecessary digit or the issue may be resolved by specifying International dialing format (no specific county code drop-down used/required). 

For example: 

A US number you would input in this format:

+1 555 555 5555 

A UK number you would input in this format:

+44 113 555 5555

Leeds 0113 has the leading zero dropped (113) when dialing its code from outside of the country.

If you are trying to save an extension into Skype, it will not work. There is no feature to store extensions and autodial them back for use with an automatic phone system.

I do not know the number you are using so I cannot examing or test it out.  I would rather you not post the number to avoid it getting spammed. I tested out a South African support number with 10 digits, the zero was dropped and the 9 digits were allowed to dial. These are the only possiblities.
