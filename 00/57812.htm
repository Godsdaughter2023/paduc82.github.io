If the update gets stuck, it's possible that it's due to software conflicts or the update is being blocked by your antivirus software. As an initial troubleshooting step, I suggest that you run the Windows Update Troubleshooter to scan your computer, identify the problems, and attempt to fix the issues automatically. You may download and run the troubleshooter from this link.

If the troubleshooter can't identify the problem or can't fix the issue automatically, try clearing the Software Distribution folder where all of the updates are stored. Doing so will force Windows Update to download fresh. There’s nothing vital here. Windows Update will recreate what it needs the next time you run it. Please note that doing this will also result in losing your Update history. Moreover, the next time you run Windows Update, it will result in a longer detection time.

First, you'll need to stop the Windows Update related services by following these steps:

  1. Right-click on the Start menu and select Command Prompt (Admin).
  2. Type the following command one after the other and hit Enter:

net stop wuauserv

net stop cryptSvc

net stop bits

net stop msiserver

Then, follow these steps to empty the Software Distribution folder:

  1. Launch File Explorer and navigate to the following location: C:\Windows\SoftwareDistribution\Download
  2. Delete everything within the folder. Do NOT delete the folder itself.
  3. Restart your computer and give Windows Update another try to see if the issue still exists.

Now, let’s restart the Windows Update related services by typing the following commands one after the other and hitting on Enter in Command Prompt (Admin):

net start wuauserv

net start cryptSvc

net start bits

net start msiserver

                 
To close the Command Prompt (Admin), type Exit and hit on Enter. Restart the computer.

The new Skype is 100% unusable for both of my companies, which are focused in both application and software development. Below I will outline the pros of the older Skype and features that made it unique from any new application that has been released in the last two years, and why people continue to use and pay for Skype.

Mind you, a few of these features were removed in previous versions, but still important to many people out there that work remotely, in an office or just for social and family communications.

Below are only a few of the advantages to Skype 7 and older:

--------------------------------

Features and Benefits of the Classic Skype:

1) Compact Mode

The compact mode is most appreciative as functionality over 'hip' style is most wanted and appreciated. The ability to keep a clean and sleek programme application is what is needed in most working, educational and even family settings. This allows for more usable screenspace for single and multiple monitor users.

Large bubbles, squiggles and over sized headers only distracts from the actual application and takes up more valuable screen/desktop space. Do not even ask me about the mobile version on small phones and tablets.

The bad with the Single Mode:
Screen/desktop space is valuable to most users. The current are design ideas to make Skype look new and hip, or just following other current trends in newer chat applications is actually going backwards for Microsoft, not forward.

  • Takes up too much space on screens and monitors
  • Inability to view multiple conversations and groups
  • Difficult to switch to new messages at times (long delay, or wrong contact
  • too easy to miss new messages
  • Delays in responsiveness, more delays the more conversations you have
  • Almost triple the overhead on PC and laptops as previous versions (and keeps getting worse)- eg: resources used.
  • Large headers takes away from actual chat space, and therefor takes up more space on the desktop.
  • Bad sort order for chats when there is an issue with connection- Out of sync messages, delayed messages (sometimes hours, sometimes days)
  • Hip style over clean modern functionality looses the uniqueness of what Skype had and made it attractive to all users
  • ... I can actually go on about the design decisions made in current, but I wont.

2) The ability to keep separate chat windows

On the PC, most have multiple monitors, especially for work/business/school/gaming/etc.

For most, the ability to keep separated chat windows is necessary as most conducting business for work, school or even social functions tend to have multiple chats up at one time. This allows the user to be able to be active and talk in one while watching/reading others and staying updated.

This also allows the users, like myself, who tend to multi task, keep up to date with the different chats while working.

Real World Example:

As COO, I am working on some scripts and assets while watching my teams work and communicate. I have four team chat groups for each team, as well as a primary or "Main Room" group. I tend to keep the following windows open while I work in other programmes like 3ds Max, Microsoft VS, SQL, emails. During this time, I have the following group chat windows opened on my 'extra' monitor:

  • Main Team Chat (everyone)
  • Engineering team
  • Programmers team
  • Content team
  • Community (PR) team

Please note that this does not include any separate and individual chats I may have going at the same time. I tend to get a lot of questions or have to get individual updates from my employees. I also do communicate with my family and friends as well separately from work itself.

The bad with the only split window:
The new version eliminates this primary ability. It instead is more like Discord or other new applications that are geared more towards the casual or social user. Trying to emulate something that is already out there in a few different applications is the wrong direction and loses what makes Skype unique.

Removing the functionality of separate windows is the worse thing possible for anyone conducting business or communicates with others while working.

  • It does not notify correctly when new messages come in from others
  • It is easy to miss important messages from others
  • It auto hides other communications and groups from others when in a call or private chat
  • Any notification that does come in is easily missed on this new split Discord like screen
  • The lack of viewing multiple chat/conversations at once to keep up to date.

3) The separate area for each function

Having to go to a separate tab or panel is not unusual in programmes. However for chat programmes this can be and is bad when this is done for all areas of the program. I am not sure how else to word or explain this, but for any type of business communications (or even with family, friends or gaming guilds), this causes more a distraction and frustration than anything that is "user friendly".

Example:

Each functionality is its own separate area and you now lose the other panels you are on. There is no current way to view a chat, be in call and view other areas. You now are forced to view one at a time:

  • Contact list
  • Search
  • Chat
  • Call
  • Screen share
  • Settings

The bad with the only one panel at a time:
Let us say you are in call with one person and talking to another. They want to connect and you have both contacts. It is now VERY difficult to share that contact. In the past it was simple drag and drop. Now, you have to load a different panel, the call and chats are no longer visible and you have to search for that contact. Share feature does not even work all the time.

4) The ability to put contacts and group chats into categories or groups

This allows us to organize our contacts. For those that have a lot, it is nice to separate them from  the other contacts for easy reference and searching. For those that have been using Skype for a while, it is a very useful function that is unique from other programmes (minus the old Yahoo Messenger). Examples (but not limited to):

  • Family
  • Business Contacts
  • Development Team
  • Engineers
  • Clients/Support
  • Friends

The ability and function to keep this form of organization is missed by some, but very important for many others like myself. I have well over 300 contacts. These range from friends, co-workers, business contacts.

The bad with the lack of contract groups:

Losing the ability to separate and place contacts into group categories means we have more of a clutter of contacts that are not easily accessible or logical in any form or order.

  • Easily lost contacts if you have a large list (applies to business, students, teachers, gamers, etc)
  • Inability to know what group someone might be in, especially if they change their visible user name.

5) Renaming Contactsin contact list

This is great for those that have a lot of contacts as it allows one to give a nick name or add their real name to the person in your contact list.

The bad with the lack of renaming contacts:

Losing this ability means that when a user changes their visible user name, it is easy for the following to not know who is who, or mistake someone for someone else. The cons to losing this ability:

  • Causes confusion for the user when a contact changes their visible username
  • Causes frustrations when a business owner or support staff no longer recognizes a client
  • Allows deceptive people to mimic other users
  • With the removal of #2, causes the above three points to become more valid

6) Chat Group Permissions and individual permission groups/roles

This is one area that has gone downhill over the last two years. Permissions are very important ranging from public to private groups. This allows companies, businesses and even gaming guilds to maintain some control and semblance of security in some cases, over the groups.

This also allows a creator or owner of the group to issue out permissions for things like moderating and even basic to advanced administration permissions to specific users. This is important for all types of groups as listed above.

The bad with the removal of permissions:

Slowly losing these features/control means that the burden is placed on one user, the creator of the group.

  • Not able to ban users from a public group
  • Not able to delegate admin and moderator rights to users that are team mates
  • Many other moderation functionality that has been lost means less control for group owners and less users for Microsoft

In conclusion

I really do hope that Microsoft reconsiders following social and casual chat programmes and realizes that it is the uniqueness that was Skype that kept it so popular among old and new users. The examples above are why there are so many discontent users and posters.

In software development, it is often said that the vocal minority is just that, a minority. However in this case it is more that we are the majority, and these changes, especially with Skype 8, is going to cause a mass migration to Discord or other similar applications, even if it costs companies, teams or even gaming guilds more money in the long run.

Thank you very much for your time. I really do hope that you and others at Microsoft take what I have outlined and what other users have been saying in multiple threads into account and either go back to the more unique features that made us choose Skype, or give us an option between the new 'hip' Skype 8 or a compact mode as in Skype 7.

Respectful Regards,

Sarrene Grant

P.S.

I did not get into the issues of a) chat history (which is important), b) search functionality and other important features!

I realized how long this already was, and I need to get back to work. Thank you.
