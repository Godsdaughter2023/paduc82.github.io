The VBA for this is

 

ActiveSheet.ListObjects(1).Unlist

 

Assuming you only have one table on the current sheet. If you have more than one then

 

ActiveSheet.ListObjects("table name").Unlist

 

where table name is the name assigned to the Table.  By default the names are Table1, Table2, etc

superfranky say:

hi jenny,

it’s not feasible to force all staff to send email out from one account.
as a workaround, you can create a shared mailbox and grant send as permission for each staff, and then all staff can manually select the from address as the shared mailbox.

about “have all there replies that come to them go to another email address”, do you mean the emails which reply to them or all the emails which send to them?

if it is for the replies, it’s not feasible to do it.
if it’s for all the emails which send to them, you can create a transport rule.
here are the steps:
1. sign in to the eac.
2. go to mail flow > rules.
3. create a new rule, under apply this rule if… section, choose the recipient is… and select the recipients, under do the following… section, choose redirect the message to… and select the receiver.
4. click save.

feel free to let me know if there is any problem when doing the steps above or anything is unclear.

thanks,
franky
