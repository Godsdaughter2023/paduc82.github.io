like if the word is 'ABCXYZ' or 'XYZ123' then it should not underline 'XYZ'. It should only underline seperate existence of string 'XYZ'.

greg maxey say:

Sub ScratchMacro()
Dim arrString() As String
Dim oRng As Word.Range
Dim lngIndex As Long

 arrString = Split("ABC,XYZ", ",")
 For lngIndex = 0 To UBound(arrString)
   Set oRng = ActiveDocument.Range
   With oRng.Find
     .ClearFormatting
     .Replacement.ClearFormatting
     .Text = "<" & arrString(lngIndex) & ">"
     .Replacement.Text = "^&"
     .MatchWildcards = True
     .Format = True
     .Replacement.Font.Underline = wdUnderlineSingle
     .Execute Replace:=wdReplaceAll
   End With
 Next lngIndex
 End Sub

avrilsam77 say:

Import contacts from Windows Mail to Outlook

marcosaraujo2 say:
Hello AvrilSam77

My name is Marcos and can I help you with your issue>

If you want to import contacts from Windows Mail to Outlook 2016 I suggest to do below troubleshooting:

Manual Steps to Import Contacts from Windows Live Mail  to Outlook

The procedure requires the installation of WLM and Outlook on the machine. It involves following 2 major steps to accomplish contacts migration task:

a. Extract a CSV file, which comprises of all WLM contacts
b. Import the extracted .csv file to Microsoft Outlook app

Step #1 : Extract CSV File From WLM Program

1. Open Windows Live Mail Program on your Windows System.


2. Click on the Contacts tab, which is located at the bottom left of the scree

3. Move the mouse cursor towards toolbar of WLM program and click on Export >> Comma Separated Values (CSV)

4. CSV export windows will appear in which you have to hit on the Browse button.

5. Locate towards the location where you want to save resultant CSV file, name the file, and click on Save.

6. You will be directed back to the CSV Export window in which you have to click on Next. This is done to continue with the procedure to import contacts from Windows Live Mail to Outlook.

7. A wizard appears in which it is totally your choice that what all fields you want to export. Check the checkbox of all those fields and click on Finish button

8. Now an Export Progress box appears in which you will see live status of WLM contacts exporting

9. Close Windows Live mail screen and navigate towards the path that you have selected in Step (5). Here, you will find the CSV file, which comprises of WLM contacts within it.

Step #2: Import Resultant CSV File in Microsoft Outlook

Import WLM Contacts to Outlook 2016

a. Open Microsoft Outlook on your machine and click on the icon of People
b. Go to Open & Export option and then hit on Import / Export option
c. An Import Wizard will appear in which you have to choose Import from another program or file and hit Next.
d. Two more options will appear where you have to select Comma Separated Values and click on Next
e. Click on Browse button to carry on with the procedure to import contacts from Windows Live Mail to Outlook 2016, 2013, and all prior versions.
f. Locate towards the path where WLM contacts CSV file is stored, select it, and click on the Open button.
g. Select any one of the 3 provided options as per your preference and click on Next button.
h. Choose the Outlook contact folder where you want to transfer WLM contacts and hit on Next button
i. On the current wizard, click Finish to start the actual process. Once the procedure gets finished, all your contacts with their details will be displayed in Microsoft Outlook.

Let me know If you need further assistance.
