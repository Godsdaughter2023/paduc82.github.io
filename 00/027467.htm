dale howard [mvp] say:

Kristy --

If you want to specify multiple cost rates for the resources in your Enterprise Resource Pool, you MUST have Microsoft Project Professional installed.  You cannot do this through the PWA interface.

To do this, first select the names of the resources in the Resource Center page in PWA, then click the Open button in the Resources section of the Resources ribbon.  This will open the checked out Enterprise Resource Pool in Microsoft Project Professional.

Double-click the name of a resource in the Resource Sheet view.  In the Resource Information dialog, click the Costs tab to display the Costs page.  The cost rate shown on Rate Table A is the Std. Rate value you should have already entered initially for the resource.  Enter additional rates on Cost Rate Tables B-E as needed.  I would personally recommend that you click the Notes tab and enter a note on how/when to use each of the alternate cost rates.  Click the OK button when finished.

Enter alternate cost rates for each of the checked out resources, then save and close the Enterprise Resource Pool in Microsoft Project Professional.  Hope this helps.
