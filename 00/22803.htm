Steps to Disable UUP and create one large ESD to be made into an ISO

1. Enable Fast on Windows 10 (tested with 14971, might work for other windows builds)

2. Press Windows + R (run)

3. Type regedit and press enter or click [ok]

4. Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsUpdate
change "SupportsUUP"= 1 to 0  (zero)

5. Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate\Orchestrator
change "EnableUUPScan"= 1 to 0 (zero)

6. Reboot for registry changes to take affect.

7. Login and Click start, Settings, Update & security, Windows Update, click [Check for updates]
until you see Downloading Windows 10 Insider Preview 14986.

8. Navigate to C:\Windows\SoftwareDistribution\Download\
& search for *.ESD & it should find a 3gb ESD file in the top yellow folder (edited to make it clearer)

14986.1000.161202-1928.rs_prerelease_CLIENTPRO_RET_x86fre_en-us.esd
or
14986.1000.161202-1928.rs_prerelease_CLIENTPRO_RET_x64fre_en-us.esd



9. When the download is 100% complete (it will then start the process of installing then at 23%)

copy C:\$WINDOWS.~BT\Sources\install.esd before it asks to reboot. (edited to make it clearer)

10. Follow the usual procedure to create the ISO from the ESD.
