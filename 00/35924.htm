I had this same issue when I upgraded one of my windows 8 workstations to windows 8.1.  I was able to find a fix which worked for me.  Since this the first page that comes up in google for this issue I'll post what I did here.

This is a risky procedure as it involves deleting system files.  Perform this at your own risk.




You will need to be logged on as a local administrator in order for this to work
  1. Enable "Show Hidden Files" and uncheck "Hide Extensions for known file types" in the "View" tab of the "Folder Options" window found in the Control Panel. Click OK to set these settings.
  2. Browse to "C:\ProgramData\Microsoft\Windows".
  3. Right click on "SystemData" and click on "Properties"
  4. Select the "Security" tab and click the "Advanced" button
  5. Click the "Change" link next to where it says "Unable to display current owner."
  6. Enter either your username or enter administrators with the local computer set as the location and click "Check Name". Your username should become underlined if it entered correctly. Click OK.
  7. Check the box for "Replace owner on subcontainers and objects" and click "Apply". Click "OK" to close the window.
  8. You are now able to open the SystemData folder. You may need to click Continue if prompted to gain access to the folder.
  9. The structure of this folder will vary from PC to PC, but there should a few folders with  numbers as their name. Look in each folder for a "ReadOnly" folder and a "LockScreen_Z" folder.
  10. Delete the "LockScreen_Z" folder and any other folder with LockScreen in the name (such as LockScreen_A or LockScreen_B).


    DO NOT DELETE ANY OTHER FOLDERS OTHER THAN THE LOCKSCREEN FOLDERS. You may need to reinstall windows if a mistake is made here.
    Note: You may need to enter the LockScreen folder and manually gain control each image by using steps 3-7 befure being able to delete them

Once all of the LockScreen folders are deleted you should be able to change the lock screen as normal. Restarting your computer will revert the permissions on the folders back to their normal settings.

  3. Open the drive corresponding to the memory card to view the card contents.
  4. Check that a System folder exists on the memory card. If not, create one.
  5. Locate the firmware file on your computer and copy it into the System folder on the memory card.
  6. Remove the memory card from the card reader, and insert the memory card into the camera.
  7. Turn off the camera.
  8. Turn on the camera and follow the on-screen instructions to complete the upgrade. The upgrade takes approximately two minutes.
  9. Turn off the camera, and then turn on the camera.
