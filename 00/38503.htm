  • Restart your PC. This will refresh the drivers.
  • See if the keyboard works with another PC. If it does, then there might be a problem with the PC you were using.

asokap say:

Sihost Unknown Hard Error - I Recovered from a disaster - Hurrah

With the help of Andre, Sumit and others I recovered (without the loss of any data or programs) from Sihost Unknown Hard Error, Explorer Unknown Hard Error.

My corrupted PC uses Trend Micro Maximum Security Package. I learned something from the post regarding third party security package (AVG...). Taking that hint, I looked in Trend Micro web site for an "uninstall" package; and under "home" users, found a download - Remnant Removal Tool. Using my working Laptop, I got this into a USB stick, and successfully applied the "Remnant Removal" - but stopped before installing the "Hot Fix" (Why? that fix was for some other problem).

Using a USB stick loaded with latest "Windows 10 Update Installation Media" (downloaded 9 June), I proceeded to successfully complete the Windows 10 April Update - without losing any data or programs. Hurrah!!!

I, along with many others, suffered hugely for 3 to 4 weeks to recover our valued PC / Laptops, without the loss of user data and use installed programs.

What I learned from this episode: 

(1) If faced with the 'sihost unknown error' or any other similar errors (that virtually disables the PC/ Laptop) after these 'Spring or Fall Windows 10 Feature' updates; do not bother doing the following: 

(a) do not down load the previous version of Windows 10 - Why? trying to install an older version results in a 'clean' installation without the ability to keep the user data or programs;

(b) do not try all manner of problem resolution methods found in various forums (google search, windows community etc). For example, sfc scan now; dism health check, chkdsk; all these did not find a problem; They proved to be just a waste of time.

 (2) What may work for me in the future;

(a) Use the 'Show or Hide Updates' tool (search under How to Stop Windows Update), in order to postpone Windows 10 Feature Updates, for at least a month;

(b) before applying the Windows 10 feature update, check this forum; if all is not clear -uninstall any third party security package (in my case Trend Micro); then apply the Windows 10 Update; and reinstall Trend Micro (I can get the latest product download from Trend Micro).

(3) If I have the opportunity to do so, I must create a Windows Repair Disk and Windows Image just before applying Windows 10 "Fall or Spring" Feature Updates;

(4) After all that, if faced with the sihost or similar problems: in order to recover Windows: (a) only download the latest version of Windows 10 Update installation media into a USB stick using another working PC or Laptop; (b) if that update continues to create the same types of problem; then apply the Repair Disk and System Image created as per the Step (3) above;

(5) I am too old to suffer all this every 6 months. So, I decided to wean myself out of Microsoft Windows and other Microsoft software (after being a continuos user for more than 30 years); - and l now learn how to use Apple Mac Book.

Thanks Andre, Sumit and other who try their best to help the Microsoft users.
