Width and height are actually reported in separate units!  Except where the (## pixels) is also shown.  The width is an estimate of the width of the cell based on the default character set/size in use.  The height is measured in points (1 point = 1/72 of an inch)

New software update for Nokia 301 Dual SIM: v. 9.04

Hey,

 

we have started rolling out a new software update for Nokia 301 Dual SIM (RM-839): v. 9.04. This is a major/global update and should be already available for users in many of the countries/regions via FOTA (Over-The-Air).

 

Find the update instructions from here: http://www.nokia.com/global/support/software-update/asha-software-update

 

Key changes compared to previous 2.33 SW version include:

 

  • Phone software stability & battery life improvements.
  • Share to Social Accounts feature added.
  • New icons for Music, Radio and Videos added.
  • WhatsApp launcher, Youtube icon and Transfer app added.
  • Lunar calendar support added (available in China only). 
  • Updated versions of Email, Facebook, Notifications, Nokia Store, Social Accounts and Twitter apps.
  • General performance & usability improvements.

Cheers! :)

Hi Donald,

The only way to transfer photos from SD card to Windows phone is by using a PC or inserting it in the phone. If you are using a standard SD card, you need to use a PC to transfer photos.
