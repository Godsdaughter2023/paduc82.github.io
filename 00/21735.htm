a. user say:

Something like...click in the table then run this...

Sub colourSelectedTable()
Dim c As Word.Cell
If Selection.Information(wdWithInTable) Then
  For Each c In Selection.Tables(1).Range.Cells
    If IsNumeric(Left(c.Range.Text, Len(c.Range.Text) - 1)) Then
      If Val(c.Range.Text) >= 0 Then
        c.Shading.BackgroundPatternColor = wdColorGreen
      Else
        c.Shading.BackgroundPatternColor = wdColorRed
      End If
    Else ' set non-numeric to White
      c.Shading.BackgroundPatternColor = wdColorWhite
    End If
  Next
End If
End Sub

For example; If you have a mainform for Orders and a subform for Order Details, the subform will only show the Details for the Order displayed in the parent form. So If you have a nested subform the same is true. The nested subform should only show records that match the parent form. If the subform records have a different date then the records in the Parent form, I suspect there is something wrong with your table structure. So we would need to understand your table structure.

dhookom say: You can't use the Link Master/Child with ranges of values. There is no such thing as BETWEEN, >, Like, <>, or " OR ". Link Master/Child use single value matching. You can specify more than one field but this will be exact matches.

ken sheridan say:

I'm only trying to see if there can be any possibility of applying the date range to the Link master and child fields of subform2 to filter the records that exist within the date range.

The answer is no.  You would have to restrict the recordsets of both subform 1 and subform 2 by parameters referencing controls in your Criteria Dialog Form.  What makes us suspicious of the database's logical model, however, is that the restriction of the first subform's recordset does not automatically imply a restriction to the referencing rows in the second subform's recordset.  You've given no information as to what each subform's recordset is modelling, so we cannot say for certain whether the model is valid.  All we can do is raise the possibility of it not being so, which Scott quite rightly did.
