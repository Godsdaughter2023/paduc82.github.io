suzanne s. barnhill say:

Word never displays facing pages. The odd-numbered pages are always on the left in Print Layout view. If you have enabled either "Mirror margins" or "Different Odd and Even" headers and footers, you can see facing pages in Print Preview. This requires that you display at least two pages, which is sometimes difficult in the backstage Print Preview (Print tab). If you add the Print Preview Edit Mode command to the Quick Access Toolbar (find it under All Commands in the Customize the Quick Access Toolbar dialog), you can easily see two facing pages.

To delete a header or footer, select all the text and press Delete. For more, see http://wordfaqs.ssbarnhill.com/HeaderFooterRibbon.htm

Images are most easily managed when they are inserted In Line With Text. When they are wrapped, they are anchored to a text paragraph and are more easily dislodged, even if locked to the paragraph in question.

You could try adding the name of the machine that is sharing the file to Internet Explorer’s Local Intranet Sites list on each machine that is trying to open the file.

 

+>Close Access.

+>Go to the problematic machine and open Internet Explorer.

+>Click on the Tools menu and choose Internet Options.

+>Within the Internet Options window, click on the security tab, select “Local Intranet” and then click the Sites button.

+>In the Local Intranet dialog, click on the Advanced button.

+>In this dialog, first make sure the “Require server verification (httpsJ for all sites in this zone” option is unchecked.

+>Then in the “Add this website to the zone” textbox, type in the name of the machine that the file share is located on.  Don’t precede the name of this machine with “\\”, only type in the machine name (ex. Machine1).  After typing this, click on the Add button.

+>Click Close and Ok to close all the open dialogs and then close Internet Explorer.
