lemp say:

Sorry for not replying sooner ... I've been away for a bit.

I assume that what you mean by "Windows XT 3 pack" is that you have a genuine Microsoft-logo'ed CD with Windows XP service pack 3.  That's fine.

Because you say that you didn't save your data, I assume that you don't care about it.  On the other hand, it appears that what you may have done is to install a second instance of Windows on a different partition on your hard drive. That is, your original installation would have had the operating system located at C:\Windows and the new one is likely at D:\Windows. If this is indeed the case, you probably still have the option of booting up using the newer version of Windows and then getting at any data you care about that will be located on the C:\ partition (most likely at "C:\Documents and Settings\<your old user name>\My Documents" but data also could be located anywhere on C:\).

First, make sure that you have all of the required drivers for the hardware in your computer. Specifically, this means your video card, any modem you may have, and similar devices. You can get these from the website of your computer manufacturer or the website of your video card manufacturer, etc.  If you don't have these drivers, Windows XP probably will be able to create a working installation, but you won't have all of the features of your particular hardware. Download drivers and copy them onto some external media, such as a USB stick or burn them to a CD if you have that capability.

After you have the drivers and have copied any data that you care about, do a "clean install." You either can follow the procedures at the link in my earlier post, or look atthis site , which includes illustrative screen shots. Referring to this second site, take a look at Figure 6. What I expect, based on your description, is that you will see are two entries here, one labeled C:\Windows and the other D:\Windows (or something similar). You want to delete bothof these entries by selecting the line and pressing "D". After you have done this, you should see something very like Fig. 6, showing the "unpartitioned space" available on your hard drive.

After you have deleted both partitions (remember -- this will erase any data or installed application programs on your drive), choose to install "create a partition in the unpartitioned space."  Create the maximum sized partition (Fig. 7).  In Fig. 8, you want to install Windows in the newly-created C: partition.

I suggest that you format the new partition using NTFS (Fig. 9).

The rest of the installation is straightforward. When you are done, the first thing you should do is go to Windows Update and install all of the "critical" or "high priority" updates. Do not install any of the optional hardware or software updates. I suggest that you configure Windows Update to "download updates but let me choose when to install them." If you do this and use the "Custom" button, you'll have a chance to select which updates you want to install.

If the above doesn't correspond to what you see on your system, or if the installation procedure appears too complex, take your computer and your original Windows XP CD to a reputable local repair shop (not one operated by a big chain store).
