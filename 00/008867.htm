adam vero say:

Your conditional format formula is "off by one".

If you apply a condition to a range B10 to B15 then the formula in the condition would need to refer to B10 so that each cell is formatted according to its own value.

You have applied the conditional format to A:A as a whole column, so the first cell is A1. Your condition refers to A2, so each cell is coloured if the cell below it occurs at a weekend.

thundertiger say:

The E71.. by itself with no addon software... does not have the functionality to automatically seamlessly switch between Wi-fi, edge, gprs, 3G connections.  There are various softwares you can download which supposedly will do this for you.  Some are 'free'  some you have to pay for. Look in the Download! section of the phone as there are one or two in there that I know of. But I have never used them.

 

If you define your home wifi as an access point and set the programs connections to default to that then in theory it will connect to that while you are AT home and if you START the program while you are AT home.  But it will NOT automatically switch to that connection when you arrive home and are currently using a different connetion (say 3G)  it will continue to use the connection you started with.

 

It would be nice if it was like some of Tmobiles UMA phones that would switch seamlessly between hotspots and the phone network.  But having the wifi connection constantly searching like that DOES drain the battery faster.
