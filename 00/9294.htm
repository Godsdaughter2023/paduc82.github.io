How to customize the Windows 10 Start Menu or Start Screen

Technical Level : Basic

Summary

A little known secret is the ability to group your most often accessed Tiles and add a title. Starting with Build 10061 the Start Menu is now black in color, can be re-sized vertically and horizontally and the “Power Button” has moved to the lower left corner. The Start Screen is also transparent.

Revised April 9th, 2015

Revised April 29th, 2015


Details

You can test this feature with your default Start Menu or Start Screen layout by clicking on the space located just above the top most set of tiles.

April 29th, 2015

Before we proceed I noticed that after upgrading from build 10041 to build 10049 and again from 10049 to 10061 that the tiles on both the Start Menu and Start Screen were scrambled and out of place from their original location on the Menu or Screen. I did not spot this immediately as I had 4 systems to upgrade. This did not happen when upgrading to build 10074.

Some of the changes in build 10049  / 10061 / 10074 make it easier to move the customized Titles and tiles associated with the titled group you created by simply moving your mouse cursor over the twin lined toaster bar located at the right edge of the title box and dragging the Title box up or down.

Not only does this relocate the location of the title box but also keeps / moves the apps that are part of that grouping pinned to the same title box.

Once the Title box is visible you can add your own unique title (My Apps in this example). Then press the ‘Enter’ key to make the title visible.

Title boxes are displayed left to right then top to bottom. Apps not associated with a title box in the Start Screen are also displayed without a title in Start Menu when you scroll down the list.

The next step is to re-arrange the tiles while in the Start Screen mode.

Click and drag any unwanted tiles away from the area directly underneath the title box you created. You can drag them to any open / free space that is available.

For example I’m going to move the following tiles from the My Apps: Search, Xbox, People, Maps, News and Sports.

(It’s best if you do this from the Start Screen)

Now that I have some free space I’m going to add some apps that I frequently access.

You should now see Tiles listed alphabetically.

I am going to add the following three apps:

  My Notes

  OneDrive

  Paint.net

Find the app by scrolling down the list. Then right click on the app you want to add (My Notes in this example) and select ‘Pin to Start’.

Now drag the My Notes tile into the ‘My Apps’ area of the Start screen.

I'd like to set some kind of conditional formating where if any of the words in column A match column B then it will highlight them for me.

An example would be the following~

 A:1 Contains - Smith P. John
 B:1 Contains - John Smith

Cells  A1 and B1 would be highlighted since they both contain the same word, "Smith".

ronrosenfeld say: You can do this with a VBA Macro.  I assumed the words had been manually entered and were NOT the result of a formula, and present a macro that will highlight the individual words that match.  This might make it easier to differentiate between John Smith and Doe John.

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.

=================================
Option Explicit
Sub MatchWords()
'Bold and Red font for matched words
    Dim rCoWorker As Range
    Dim rMine As Range
    Dim c As Range
    Dim re As Object, mc As Object, m As Object
    Dim colCoWorkerWords As Collection
    Dim colMyWords As Collection
    Dim sPat As String
    Dim sWords As Variant
    Dim sUniqueWords() As String
    Dim i As Long
Set rCoWorker = Range("A1", Cells(Cells.Rows.Count, "A").End(xlUp))
Set rMine = Range("B1", Cells(Cells.Rows.Count, "B").End(xlUp))

'Reset font to normal
With Union(rCoWorker, rMine)
    .Font.Bold = False
    .Font.Color = vbBlack
End With

'Get List of CoWorkers Words
    Set colCoWorkerWords = New Collection
    For Each c In rCoWorker
        sWords = Split(WorksheetFunction.Trim(c.Text))
        On Error Resume Next
            For i = 0 To UBound(sWords)
                colCoWorkerWords.Add Item:=sWords(i), Key:=CStr(sWords(i))
            Next i
        On Error GoTo 0
    Next c
   
'Look for Words
'Could use Find except hard to exclude "word within word"
Set colMyWords = New Collection
ReDim sUniqueWords(0 To colCoWorkerWords.Count - 1)

'bold matched words in My List
For i = 1 To colCoWorkerWords.Count
    sUniqueWords(i - 1) = colCoWorkerWords(i)
Next i
sPat = "\b(" & Join(sUniqueWords, "|") & ")\b"
Set re = CreateObject("vbscript.regexp")
With re
    .ignorecase = True
    .Pattern = sPat
    .Global = True
End With
For Each c In rMine
With c
    If re.test(.Text) = True Then
        Set mc = re.Execute(.Text)
        On Error Resume Next
        For Each m In mc
            colMyWords.Add Item:=m, Key:=CStr(m)
        On Error GoTo 0
        With .Characters(m.firstindex + 1, m.Length).Font
            .Bold = True
            .Color = vbRed
        End With
        Next m
    End If
End With
Next c

'Bold Matched Words in CoWorker List
ReDim sUniqueWords(0 To colMyWords.Count - 1)
For i = 1 To colMyWords.Count
    sUniqueWords(i - 1) = colMyWords(i)
Next i
sPat = "\b(" & Join(sUniqueWords, "|") & ")\b"
re.Pattern = sPat
For Each c In rCoWorker
With c
    If re.test(.Text) = True Then
        Set mc = re.Execute(.Text)
        For Each m In mc
        With .Characters(m.firstindex + 1, m.Length).Font
            .Bold = True
            .Color = vbRed
        End With
        Next m
    End If
End With
Next c

End Sub

F-Secure anti-theft; Password forgotten - Microsoft Community

phonehacker say:

F-Secure anti-theft; Password forgotten

I have seen many posts on the forum regarding this thread. So I decided to post the solution here:

 

If you forget your security code, you can reset it from the trusted number. Follow these instructions:

 

  1. Send SMS with the following message to your device: #UNLOCK#
  2. Your device sends the reset SMS message with instructions to the trusted number.
  3. Send the reset code from the trusted number to your device.
  4. Your device is unlocked and you can create a new security code.

Note; this is taken from F-Secure > Help > Anti-theft settings.

tesla5 say:

Add comment to photo/figure caption

I'm editing a paper on my mac and cannot add comments to figures (using MS Word 2011 for Mac). When I select a figure, "new comment" fades out. Same thing happens when I click or highlight text in the caption.

Please advise.

daniel g. grau say:

You cannot insert comments in floating text boxes. All comments must be in the "text layer."

A solution would be to extract the caption from the text box and manually align it under the figure.

(In fact, if you need to build a Table of Figures, you won't be able to make one from from floating captions—you'd have to extract them.)
