Something that is not "pretty"
 
I assumed that your number of data  does not exceed 999 Items.
 
On sheet1 in A1, B1, C1 one can find the name of the columns.

charles kenyon say:
' http://www.gmayor.com/installing_macro.htm
' update TOC fields in a document upon save
    Dim oField As Field
    For Each oField In ActiveDocument.Fields
        If oField.Type = wdFieldTOC Then oField.Update
        End If
    Next oField
    Set oField = Nothing

This assumes that the TOC is in the body of the document.

Hi Valko is refering to the 2007 or earlier versions.  In 2010 STDEV.S or STDEV.P are your better choices

=STDEV.S(IF(A1:A8="this",B1:B8,""))

still as an array.

2010 supports STDEV but not the best choice.

t. valko say:

Try this array formula** :

=STDEV(IF(A1:A10="this",B1:B10))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.
