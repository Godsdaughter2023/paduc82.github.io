burrwalnut say:

There are a number of ntuser.dat files which contain the registry entries of the individual users of the computer.  The Current User registry information is located in \Documents and Settings\username\ntuser.dat.

The 5 main registry hives are located in \WINDOWS\system32\config. It is this version of the registry that is loaded when the computer is booted.

If you think about it, it had to be switched on. It’s unlikely, but the system date/ time could be wrong, or, depending upon internet access, it could have been intentionally changed then corrected. Additionally, the computer may have a ‘Wake on’ setting but I assume you would know about that.
