Hi, 

Since you no longer have access to your old account, the only option to transfer your contacts is to get the main.db from your old account. 

First, make sure that you are already log in to your new account. Once, secured, kindly follow the steps below: 

  • Make sure that Skype is no longer running. Quit Skype.
  • On your keyboard, press the Windows key + R key. Then, the run windows will appear.
  • On the run window, type in %appdata%\Skype and hit enter. Look for a folder Facebook and get the file named main.db file and replace your new folder created by your new account.

charles kenyon say:
Do NOT delete the normal template. Open it and remove the text. Then save.

Normal Template in Microsoft Word - How to Open

The reason to not delete the normal template is that it may contain other resources you want to save such as macros, custom styles, AutoText, AutoCorrect, or keyboard shortcuts.

If you think you need to have a new normal.dotm, then, with Word closed, find the old one and rename it.
