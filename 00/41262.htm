rung_windows7 say: Hi Ramesh,

That did the trick.  For the record:
1) Used msconfig to switch to safe mode boot
2) Booted to safe mode
3) ran cmd.exe as an administrator
4) typed "cd C:\Windows\System32\LogFiles\WMI"

6) Used msconfig to switch to normal boot mode
7) Reboot and found event viewer working

To remotely control (for maintenance/troubeshooting purposes) a Windows computer it depends on the version of Windows your running on the remote computer, ie. the host/server computer...

  • Windows 7 Ultimate/Pro, Vista Ultimate/Business or XP Pro/MCE you can use Remote Desktop.
  • Windows 7 Home Premium or Vista Home Premium you can use Windows Live Mesh 2011 Remote Connections (FREE, see below) or TeamViewer (FREE) or VNC (FREE) or Logmein ($$$) as examples.
  • Windows XP Home you can use TeamViewer or VNC or Logmein as examples.

cager say: ACPI stands for Advanced Configuration and Power Interface - which is implemented by Windows XP.

It allows the system to automatically turn peripherals (CD drives, network cards, etc) on & off, as well as allowing peripherals to turn on or access the pc. This is all part of the installation's power management and during the boot process Windows checks & tests the system's hardware to see if it and the system's BIOS support ACPI.

The error message you received was one that you would normally receive if you were trying to install Windows - Windows does an ACPI compliance test at the start of the installation.

In some older pc's the BIOS needed to be upgraded to get full compliance with the latest version of ACPI as implemented by XP. The error message is telling you that your system's BIOS needs to be upgraded.

However, this may not be the whole truth.

Please confirm how old your pc is and whether or not it came with XP pre-installed - also when you first started getting this problem.

If the pc is relatively new and XP was pre-installed, unless you have been having this probem since you got the pc, something has changed to cause this problem - and that something could well have come from the internet sites you have visited.
