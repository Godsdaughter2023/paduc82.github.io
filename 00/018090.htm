hansv mvp say:

UPDATE [tblmain] SET [fieldC] = DLookup("fieldA", "tblmain", "[fieldB] = " & [fieldC]) WHERE [fieldC] Is Not Null

 

is the SQL for a query, it is not a VBA statement. If you want to execute it from a command button, use code like this:

 

Private Sub cmdSomething_Click()
    CurrentDb.Execute "UPDATE [tblmain] SET [fieldC] = DLookup('fieldA', 'tblmain', " & _
        "'[fieldB] = ' & [fieldC]) WHERE [fieldC] Is Not Null", dbFailOnError
End Sub

 

Note that I have replaced the double quotes within the SQL string with single quotes.
