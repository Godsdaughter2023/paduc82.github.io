jay freedman say:
The solution does involve a macro, so I'll include the steps below. You can see an illustrated version at http://www.gmayor.com/installing_macro.htm.

Here's the macro you need:

Sub ResizeSlides()
    Dim Slide As InlineShape
   
    For Each Slide In ActiveDocument.InlineShapes
        With Slide
            If .Type = wdInlineShapeEmbeddedOLEObject Then
                .ScaleWidth = 100
                .ScaleHeight = 100
            End If
        End With
    Next
End Sub

To install it, use these steps:
  1. Open Word to any document, or the blank document you get when you start the program without another document.
  2. Press Alt+F11. This opens the macro editor.
  3. In the left side of the macro editor is a small window labeled Project. (If you don't see it, click the View menu and click Project Explorer.) It contains a tree diagram, and the first item in the tree says Normal. Click the Normal item.
  4. Click the Insert menu and click Module. On the right side of the macro editor you should now see a big blank area with dropdowns at the top that say (General) and (Declarations).
  5. In this post, select the code (from the word Sub to the End Sub) and press Ctrl+C to copy it to the clipboard.
  6. Click in the big blank area mentioned in step 4, and press Ctrl+V to paste from the clipboard.
  7. Click the File menu and click Save Normal.
  8. Close the macro editor by clicking its red X in the top right corner or by pressing Alt+F4.
To use the macro, first open the Word document containing the PowerPoint slides. Then press Alt+F8 to open the Macros dialog. Click the ResizeSlides item in the list, and click the Run button.
