j.e. mcgimpsey say: All right - one more time...  :-)

I think you're confused about the order of subtraction and which row and column you want to subtract from which. But it's pretty obvious what your goal is.

Starting simply:

In your worksheet that you posted the link to, enter the formula on the next line into cell E4:

     =D4 - D3

and hit Return. You should see the value 23 appear in the cell. 

Now, select cell E4 by clicking once. Hover the cursor over the lower right corner of the cell - the cursor will turn into a thin black cross. When it does, click the left mouse button (or the trackpad button area) and hold it, then drag down to highlight cells E5:E9. Release the button.

You should see the number -97423 appear in E5, and 0 in E6:E9. If you single click on cell E5, you'll see in the formula bar that XL has adjusted the formula to read 

    =D5 - D4

The number -97423 is obviously incorrect, but only because you haven't entered any value in cell D5, so XL interprets the blank cell as a zero.

That's why I suggested using this formula instead (put it into E4):

    =IF(D4<>"", D4 - D3, "")

which only calculates D4 - D3 if cell D4 is not blank. If cell D4 is blank, then the formula returns a "null string" so the cell will appear blank. If you copy THAT formula down (as above), then instead of -97423 in cell E5, cell E5 will appear to be blank. You can then enter the next day's Electric Upper value in D5, and cell E5 will display the difference between D5 and D4.

jose est say:

Hi Susan,

For us to be able to send an email to a cell phone number, we need to know the number and the carrier, that the number is registered to.

Each carrier has a special domain, or email address, that you can use to send email messages to mobile phones on their network.

NOTE: Some carriers have two different domains: One for text-only messages (SMS) and one for messages that contain pictures or other attachments (MMS).

These are the domains for some of major carriers:

Carrier Domain
AT&T @txt.att.net (SMS)
@mms.att.net (MMS)
T-Mobile @tmomail.net
Verizon @vtext.com (SMS)
@vzwpix.com (MMS)
Sprint  @messaging.sprintpcs.com (SMS) 
@pm.sprint.com (MMS)
US Cellular @email.uscc.net (SMS)
@mms.uscc.net (MMS)
Boost Mobile @myboostmobile.com
Virgin USA @vmobl.com
 Rogers (Canada)  @pcs.rogers.com
Orange (UK) @orange.net
Vodafone (UK) @vodafone.net
Orange (India) @orangemail.co.in

To send an email messages with photos using Outlook.com follow the steps below:

  1. Compose a new email message.
  2. Address the recipient in the "To:" field. To do so, type their mobile phone number without the country code or any punctuation followed by the carrier's mobile email domain.
    • For example, to email the U.S. number (123)456-7890 using "PhoneCompany"  you would address the message to 1234567890@mms.PhoneCompany.net.
  3. Send the message. Your recipient should receive your email message in their phone messaging app.

Let us know if you need anything else.
