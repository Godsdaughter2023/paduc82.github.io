vanessa ouyang msft say:

Hi BPPCPA111,

Are you using OWA(https://outlook.office.com/) or Outlook client with office 365 account? If the former, this issue can occur if the following conditions are true: the email message contains deeply nested elements or the Use Conversations view in OWA is turned on. To work around this issue, turn off the Use Conversations view in OWA. To do this, click View and then clear Use Conversations. 

When you search the missing emails in Outlook client, hover your mouse over it. The sub folder these emails belong to will show up, navigate to the sub folder, check emails.

Then, check your OWA inbox rule to see if it affects: Click on settings button on the top-right corner and select Options>  Go to the organize email section and make sure you are on the inbox rules tab.

Regards,

Vanessa
