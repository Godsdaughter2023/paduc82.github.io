The default location for AutoRecovery files is Users/YourUserName/Library/Containers/com.microsoft.Word/Data/Library/Preferences/AutoRecovery/. By default, the user Library folder is hidden. Hold down the Alt key while clicking on the Go menu and choosing Library to show it.
