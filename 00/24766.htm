I cannot get two rectangles to overlap in Visio 2013! For various reasons I need overlapping rectangles in my diagram. How can I do that?
itgovguru say:

Your issue has to do with the automatic insertion of a shape within a connector, i.e. if you drop a shape (even part of it) on or over a connector, the connector will split and reattach to the shape on each side.


To turn this behaviour off, go to the Tools menu and select Options.  When the Options window pops up, select the General tab.  Deselect the "Enable connector splitting" which is on by default.


This will solve your problem.

Authenticate using: clear text.
Under Logon user name, type your Parent email address.

Use 25 or 587 for the port number (If port 25 has been blocked in your network or by your ISP, you can set SMTP port to 587)

There are a number of reasons why you could be facing this issue. 

a. It could be that the files that you are trying to open are corrupt
b. The installed copy of MS Office has been corrupted
c. Recent windows updates may have effected file association for word documents

It seems that a recent windows update somehow "broke" some part of Word so that it's unable to open documents anymore. In order to fix this issue, you’ll need to fix the file association of the word files.  Here’s how you can do that:

1. Open the "Start" button and click the "Default Programs" shortcut. Click the "Set Associations" option.
2. Scroll down the list to find the file extension you want to change (In this case, it would be “.doc” or “.docx”). Click it to select it.
3. Click the "Change program" button. Click the program you want to use to open the file. If the program doesn't appear, click the "Browse" button to find it. Click "OK" to set the file association.

In case the above steps doesn’t work, try restoring the system to a point in time when you weren’t facing this issue.

If that doesn’t work, it might be that MS word has indeed been corrupted and needs to be reinstalled. we would recommend that you check your word files on another system as well, as it could be that those files are corrupt.
