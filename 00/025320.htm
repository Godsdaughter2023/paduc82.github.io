shane devenshire say:

It is not clear from you question if you want two formats - one for values greater than the value in column C and another for the ones less than the value in column C   or are you asking to highlight the ones that are either greater or less than the values in column C (in other words not equal to column C).  So do you want one format or two?

For the situation where you want 1:  (G<>C) one format:


In 2007:
1. Highlight all the cells on the rows you want formatted, say A1:Z100
2. Choose Home, Conditional Formatting, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following formula:

=$G1<>$C1

5. Click the Format button and choose a format.
6. Click OK twice

For the situation where you want two formats - one for C>G and one for G<C:

1. Highlight all the cells on the rows you want formatted
2. Choose Home, Conditional Formatting, Manage Rules, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following
formula:

=$G1>$C1

5. Click the Format button and choose a format.
6. Click OK twice
7. New Rule
8. Choose Use a formula to determine which cell to format
9. In the Format values where this formula is true enter the following
formula:

=$G1<$C1

10. Click the Format button and choose a format.
11. Click OK tree times.
