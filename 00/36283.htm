Hi Dushyant,

We appreciate you for being part of Windows 10.

Background Intelligent Transfer Service (BITS) is a component of 2000 and later operating systems that facilitates prioritized, throttled, and asynchronous transfer of files between machines using idle network bandwidth. It is most commonly used by recent versions of Windows Update, Systems Management Server to deliver software updates to clients, Microsoft's anti-virus scanner Microsoft Security Essentials.

BITS uses idle resuming transfers in case of disruptions.

It is not recommended to disable the Background Intelligent Transfer Service (BITS), as it is required to download the latest updates related to Microsoft/Windows/Server through the Windows updates which will help the Windows to run effectively.
