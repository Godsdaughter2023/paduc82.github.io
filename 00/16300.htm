kergga say:

Right click on the data labels, select "format data labels", click the numbers group, and change to percentage.

 

This assumes that your source numbers are appropriate for a percentage, usually between 0 and 1 (although >100% is always possible). Just saying that if your source number is 75, you won't get 75% when you convert, you'll get some honking big number.

 

HTH

The easy options are:

If Word has crashed, it may not be obvious if it is still open. Press CTRL+SHIFT+ESC and from the 'Processes' tab, see if you have Winword.exe as a running process. If it is there, right click and select End Process Tree. Close the Task Manager.

jay freedman say:

All existing documents that were created in an earlier version will open in Compatibility Mode unless and until you "convert" them one at a time: open the document, click File > Info, click the Convert button, and OK the prompt.

Compatibility Mode is an "edit" mode -- exactly like the editing in the document's original version. That is, Word will disable any features that were introduced after the version that created the document. All unchanged features work as they did in the older version. If you need or want to use the newer features, convert the document at that time. Otherwise, you can leave all existing documents as is. This is not a "problem".

Since you deleted the Normal template, which causes Word to create a new one, all new documents that you create now should not be in Compatibility Mode -- is that what happens?
