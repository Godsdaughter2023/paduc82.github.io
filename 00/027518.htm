peterheide say:

In the Word version in 365 how to Repeat Header Rows

In the Word version in Office 365 Home, how can I retain or re-reinsert Repeat Header Rows when I stop Word from page breaking mid-Cell Row. I can Page Break within a cell in the row and preserve the entire row on the page, but when I do this, it stops repeating the header rows on each page. If I reverse the process and page break within a cell and then try to initiate Repeat Header Rows, I will not execute the command. Is this possible? And if so how do I do it?

doug robbins - mvp office apps & services (word) say:

If you want part of the text in a cell to appear on a new page and have the header row(s) repeat, instead of using a hard page break (CTRL+ENTER), format the paragraph so that it has a page break before it.

However, if the row is formatted so that it is not allowed to break across a page, it is only if the first paragraph in a cell in the row is formatted to have a page break before it that the row will move to the next page.  If that formatting is applied to other than a first paragraph in a cell, it will be ignored.
