christopher miller say:

Hi Walter,

There is a bug that would display the phone as having the Denim update, when it was still on the Cyan firmware.  If you go to Settings, then tap about, then tap the more info button, you can see the OS and firmware version numbers.

For the 1020, the numbers will vary depending on which country you are in and if you have a carrier locked or an unlocked phone.   The build number is the 3rd part of the number and is what you need to look at.

For Cyan on the 1020, the OS version number will be (or very close to) 8.10.12393.890

For Denim on the 1020, you should see OS versions of 8.10.14226.359 or higher.

FWIW, if your phone was enrolled in the Windows Phone Developer Preview Program but was reset or otherwise removed from the program, the Lumia Denim update will not be available, or reports an error 8018830f.
