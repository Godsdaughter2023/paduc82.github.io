When you delete a file or folder from Favorites where does it go? It's not in my trash bin!

When I was in junior college there was a location in the System file that held all that was deleted on your PC computer. Since I was on Macs for a decade forgot the location. Any help with the PC format?

So, let's assume that you're talking about Microsoft Edge... and that you're trying to recover a deleted item

Edge no longer stores its favorites as files like Internet Explorer does.   Instead, they are stored in a database.  So, that means that when you delete a favorite, nothing get's placed in the Recycle Bin.  Further, since they aren't stored as files, there is no file-based unerase utility that will be able to help.

However, things that get deleted from the database, stay there for about 30 days before they are finally "purged".  There is a free 3rd-party application called EdgeManage that will allow you to view the deleted database entries and resurrect them if you deleted them by mistake.
