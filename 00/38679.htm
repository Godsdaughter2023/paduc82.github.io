ajay k say:

Hi M Ham,

Welcome to Microsoft answers!

This issue may occur if the signal from the video adapter exceeds the scan range of the monitor. The video adapter setting for updating the screen (the refresh rate) is incompatible with the monitor.

I would suggest you to try the following steps:

Step 1: Boot the computer in “enable low resolution video” (Windows uses your current video driver and using low resolution and refresh rate settings)

Enable low resolution video is available under advance boot option menu. You can access the advanced boot option menu by turning on your computer and pressing the F8 key before Windows starts.

Step 2: Reduce display settings

1. Open Display Settings by clicking the Start button, clicking Control Panel, clicking Appearance and Personalization, clicking Personalization, and then clicking Display Settings.

2. Under Resolution, move the slider to a lower value and then click Apply.
http://windows.microsoft.com/en-us/windows-vista/Change-screen-resolution

Step 3: Correct monitor flicker (refresh rate)

1. Open Display Settings by clicking the Start button, clicking Control Panel, clicking Appearance and Personalization, clicking Personalization, and then clicking Display Settings.

2. Click Advanced Settings.

3. Click the Monitor tab, and then select a new refresh rate. The monitor will take a moment to adjust. If you want to keep the changes, click Apply. If you don't apply the changes within fifteen seconds, the refresh rate will revert back to your original setting.

Now, restart the computer in normal mode and check if the issue is resolved

You may try to put the monitor settings to default (factory settings) using the settings available on the monitor itself.
Please check the monitor manual to restore the monitor settings to default or contact manufacturer.

To start... There is a Linux based "File Explorer" included in a free 3rd party program called Parted Magic... It will enable you to copy any needed files to another drive of some type..

It downloads as a ISO that burns as a Image File to create a bootable CD... It can also be used to create a bootable Flash Drive if needed...

The newer versions are no longer free, and the older free versions are very difficult to find a download link for... This is the most reliable I could find...

http://www.majorgeeks.com/files/details/parted_magic.html

The computer may be fixable... The problem is most likely just the boot files, but it could be worse than that..

To try to fix this, you would have a better chance of success if you boot the machine with a current version of Windows 10 Installation Media...

You can get this from the Microsoft "Create Media" website,

http://www.microsoft.com/en-us/software-download/windows10

Or a 3rd party source that redirects to a Microsoft server...

https://tb.rg-adguard.net/public.php

The Create Media website is a rather long and drawn out process...

You can create a bootable Flash Drive with the ISO using a tool called RUFUS...

https://rufus.akeo.ie/

Once you have a bootable DVD or Flash Drive, try this... I'm guessing this is a newer machine with UEFI Bios...

UEFI Boot Repair...

1. Insert the Windows installation disc\Flash Drive and then start the computer.

2. Press F9, F11, F12, ESC, or whatever your machine uses to access Boot Options at the Manufcturer's Logo (POST) Screen..

Select the CD\DVD Drive or Flash Drive that starts with "UEFI"... Press any key when "Press any key to boot from CD or DVD …". appears. You may see this for a Flash Drive as well...

If it is by chance NOT a UEFI machine, select the drive without the UEFI prefix..

3. Select a language, a time, a currency, and a keyboard or another input method, and then click Next.

4. Click "Repair your computer" on the lower left of the "Install" screen....

5. Click the "Troubleshoot" or "Advanced" depending upon which appears...

6. Select "Command Prompt". (Capital letters not necessary)

7. Type BOOTREC /FIXMBR, and then press ENTER.

8. Type BOOTREC /FIXBOOT, and then press ENTER.

9. Type BOOTREC /REBUILDBCD, and then press ENTER.

Close Command Prompt and restart to Windows...

Any more questions, feel free to ask...
