ASMedia USB 3.0 extensible Host Controller - 1.0   (Driver Version   1.16.38.1)

ASMedia USB 3.1 extensible Host Controller - 1.10 (Driver Version 10.0.16299.15)

as soon as I updated device drivers for the USB controllers using device manager windows updates the drivers and breaks the Camera app.

I'm not sure if this will be the same for other systems and webcams but it is something worth checking if you have the same problem
