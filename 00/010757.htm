The message "You haven't signed in from this location recently." will occur if you are signing in from a location more than 300 miles from your normal location.

Hi Anthony,

Generally, if DirSync is set up by default, the shared mailbox created on-premsies Exchange synced to Office 365 via DirSync will be mail user. Hi Anthony,

It is the expected behavior. Since you have hybrid, only one side (could or on-premise) has mailboxes. You cannot convert it to a shared mailbox. There are two methods:

1. Directly create a shared mailbox in cloud.
2. Create a on-premise shared mailbox and move it to cloud.

suzanne s. barnhill say:
Unfortunately, Word 2007 doesn't offer the same recovery provisions found in later versions (and even those probably wouldn't have helped). If you had "Always create backup copy" enabled (Office Button | Word Options | Advanced: Save), and if you had saved the document fairly recently before the last, disastrous Save, then you would be able to recover some of the content: display "All Files" in the Open dialog and look for a file named "Backup of <filename>.wbk, which will be the previously saved version. Otherwise I'm afraid you'll have to start retyping.
