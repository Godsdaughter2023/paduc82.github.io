john korchok say:

Microsoft Word uses only RGB and HSL color models. macOS users get confused on this point because Word uses the macOS color picker, which includes a CMYK option. All colors set as CMYK are automatically converted to RGB by Word.

Type in Word should be specced as R0 G0 B0 to translate as 100% black. If that comes into InDesign as any other value, it's macOS/Adobe color management at work, not Word.

The usual solution for this is to use a macro. Before we get into the details, please make a short test.

Have you ever recorded a macro? If not, please have a look here:
https://support.office.com/en-us/article/automa...

Start as usual in your file with no charts.
Select cell A1
Start the macro recorder
Select the data and create your chart as you did before
Stop the macro recorder
Delete the chart
Press Alt-F8 and execute your macro
