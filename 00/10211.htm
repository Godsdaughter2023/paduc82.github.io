Hi ETP2,

If you want to remove your Skype for Business only, you can use Office Deployment Tool.

1. Download the Office 2016 Deployment Tool, run the downloaded .exe, select a local folder(for example C:\ODT2016) and extract the files (you will get two files, setup.exe and configuration.xml).
2. Replace the content of the configuration.xml with the following lines:

<Configuration>

<Add SourcePath="C:\ODT2016" OfficeClientEdition="32">
 <Product ID="O365ProPlusRetail">
 <Language ID="en-us" />
<ExcludeApp ID="Lync" />
 </Product>
 </Add>

<Display Level="Full" AcceptEULA="TRUE" />

</Configuration>


3. Open the Command Prompt window. (Win + R > cmd)

4. Run cd c:\ODT2016. (the folder path where you have saved the installation tool)

5. Download the installation files by running the following command:                                        

setup.exe /download configuration.xml

6. Install the application by running the command below:   

setup.exe /configure configuration.xml

It may take a while to download and install the application.
