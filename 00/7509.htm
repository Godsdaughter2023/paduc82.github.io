If Heading 4 is not displayed in the Style dropdown, it is because it has not been used in the current document (and various other reasons I won't go into), but you can force the Style dropdown to display a complete style list by pressing Shift while clicking the down arrow. Once you have used Heading 4 in the document, it will probably be displayed (if not, there are ways to change that). In the Customize Outline Numbered List dialog, you should be able to select Heading 4 for Level 4.

If that pane does not display that numbering style, select it and click Reset so that it will. When you apply that list template, the numbering is automatically linked to the heading styles and has the format you want.
