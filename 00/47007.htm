vibroplex say:

Lag is slowness in the upload, explained well at www.cringely.com/.../lagbuster-makes-online-games-play-faster-even-call-of-duty (and this does NOT mean to endorse any of the products or services mentioned therein):

The difference between bufferbloat and lag is that bufferbloat is mainly downstream (video server to you) while lag is mainly upstream (you to the game server). Bufferbloat is caused by large memory buffers in devices like routers and in applications like media players messing with the native flow control in TCP/IP. We add buffers thinking it helps but instead it hurts. Something similar happens with lag but it tends to happen at the point where your 100 or 1000 megabit-per-second local area network meets your 3-25 megabit-per-second DSL or cable Internet connection. Lag is caused by congestion at that intersection. You can tell you have lag when you can’t seem to be able to aim or shoot fast enough in your shooter game.

It’s not you, soldier, it’s the lag.

So, lag is caused by the internet infrastructure between you and Xbox.

However, adding Kenya to nations with Xbox service is a different kettle of fish. Copyright issues can limit the ability of Xbox to enter a new country, although I don't know what the exact reason is. Therefore, I don't know when Kenya will appear as a supported nation.

Thank you for writing to Microsoft Community Forums.

When you search in Windows explorer, by default the search results will be displayed using the Content view, however this behavior can be changed by switching to the Detail view, but once your reboot the computer, the settings will be reverted to default.

To make the explorer display the results with Detail view, you may have to make some changes in the registry.

Disclaimer: The registry is a database in Windows that contains important information about system hardware, installed programs and settings, and profiles of each of the user accounts on the computer. Windows often reads and updates the information in the registry.

Normally, software programs make registry changes automatically. You should not make unnecessary changes to the registry. Changing registry files incorrectly can cause Windows to stop working or make Windows report the wrong information.

Please take a backup of the registry. Follow the steps given in the link below:

How to back up and restore the registry in Windows

Copy and paste the following script in a notepad file and save it with any name with extension .Reg.

Script:

Windows Registry Editor Version 5.00

;Fixes search results defaults to Content view.

;Sets Details view as the default search results view for Generic folders.

[-HKEY_CURRENT_USER\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\BagMRU]

[-HKEY_CURRENT_USER\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\Bags]

[HKEY_CURRENT_USER\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\Bags\AllFolders\Shell\{7fde1a1e-8b31-49a5-93b8-6be14cfa4943}]

"LogicalViewMode"=dword:00000001

"Mode"=dword:00000004

Change the Save as type from *.txt to all files.

Click Save to save the file.

Close the file.

Go to the file location, right click on it and select Merge.

Reboot the computer and see if the settings are retained

Hope it helps.
