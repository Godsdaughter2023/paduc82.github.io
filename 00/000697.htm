Search For Partial Text and Return A Variable - Microsoft Community

bbpt say:

Search For Partial Text and Return A Variable

I need help with the following.  A cell in a row contains text and the last 4 digits of an employee’s expense card.  I want to insert a formula in an adjacent cell that will look for the 4 digits within the text and return the name of the employee who was assigned the card.  For example, cell B2 contains the text “CHECK CARD PURCHASE   XXXXX1234 ABC Business Inc   AnyCity  AnyState”  I would like to enter a formula in cell C2 that will return the name Bob if 1234 exists in B2, Bill if 2345 exists in B2, Isabelle if 3456 exists in B2, Lisa if 4567 exists in B2 or leaves a blank if none of 1234, 2345, 3456 or 4567 exist in B2.  Thanks
rich007a say:

If your data always starts with "CHECK CARD PURCHASE   XXXXX", (i.e. 27 characters before the 4 digits) you can use:

=VALUE(MID(B2,28,4))

to return the last 4 digits of the number.

I would set up a table with your names and numbers, e.g. in the range G1:H4 enter:

1234    Bob
2345    Bill
3456    Isabelle
4567    Lisa

You can then use VLOOKUP to return the name.  If the last 4 numbers are now in cell C2, in D2 you could use:

=IFERROR(VLOOKUP(C2,$G$1:$H$4,2,FALSE),"")

or you could just put this in cell C2:

=IFERROR(VLOOKUP(VALUE(MID(B2,28,4)),$G$1:$H$4,2,FALSE),"")

Hope that helps.

Cheers
Rich
