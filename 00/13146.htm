Unfortunately, the cell format ####-##-###-#### is a number format. So if you add text to the cell, the format will not apply.

You could try a simple formula, i.e. if you enter 12341212E1234 in cell A1, then the formula
=LEFT(A1,4) & "-" & MID(A1,5,2) & "-" & MID(A1,7,3) & "-" & RIGHT(A1,4)
will show the serial no as you want it.

This lookup returns the exact or closest match. it's an ARRAY formula so see below on how to enter it. The lookup value is in C1

 

=INDEX(B1:B20,MATCH(MIN(ABS(A1:A20-C1)),ABS(A1:A20-C1),0))
