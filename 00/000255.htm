pambarden say:

templates

how to get templates for  a certain style of labels, tabs?

suzanne s. barnhill say:
  1. On the Mailings tab of the Ribbon, in the Create group, click Labels.
  2. In the Envelopes and Labels dialog (Labels tab), click Options...
  3. In the Label Options dialog, select the type of label (Continuous or Page) and the vendor.
  4. In the Product number list, scroll to find the stock number of the labels, tabs, cards, etc., you are using. Note that the list is sorted alphabetically rather than numerically, so five-digit numbers are mixed in with four-digit; i.e., Avery 18695 precedes 2160.
  5. After selecting the appropriate label, click OK to return to the Envelopes and Labels dialog.
  6. If you want a full page of the same label, you can type the label content into the box and click either Print or New Document. If you want to type different information into each label, click New Document.
  7. You'll get a sheet of the labels you chose. They're set up as a table, so be sure you have table gridlines displayed so you can see the label boundaries. General rules for working with tables apply.
  8. If you need more than one sheet of labels, and the labels are not complex (that is, they don't include separator rows), you can tab out of the last label on the page to start a new page with a new row; alternatively, you can select and copy the entire table and paste it on a new page. If you need very many labels, however, it is usually more efficient to set up a mail merge.
