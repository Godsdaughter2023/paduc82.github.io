Can I export a list of quarantined items from MSE? - Microsoft Community

bert onstott say:

Can I export a list of quarantined items from MSE?

MSE has found and quarantined a bunch of threats in files from a website that I downloaded.  I would like to be able to send the list of infected files to another person via email.  Can I get that list from MSE?
greginmich say:
MSE has found and quarantined a bunch of threats in files from a website that I downloaded.  I would like to be able to send the list of infected files to another person via email.  Can I get that list from MSE?

It’s fairly easy to create a custom log for MSE’s detection/remediation events in Event Viewer. Detections are recorded as Microsoft Antimalware “1116 “ events in the System Event Log, and remediation events have an ID of “1117”:

1.) Open Event Viewer by typing “eventvwr” (or just “event”) in the search box and hitting the “Enter” key.

2.) Expand “Windows Logs” and click on the “System” folder.

3.) Click on the “Filter Current Log…” option in the “Actions” pane or “Action” menu.

4.) Optionally, define a time range in the “Logged” setting at the top of the “Filter Current Log” window.

5.) Select “Microsoft Antimalware” as the Event source and enter "1116, 1117" (or just "1116" for detection events only) for the Event ID.

6.) You can then save the filtered log as a text file (“Save Filtered Log File As…” or “Save All Events in Cust…”).

7.) If you then click on “Save Filter to Custom View” and name the Custom View “MSE Detections”, you'll be able to generate your next detection/remediation log by simply expanding “Custom Views” and clicking on “MSE Detections”.
