waseemulla shariff say: Hi,
 
What is Win VNC?
One of the popular desktop sharing systems is VNC (Virtual Network Computing). The Win VNC is required if you would like to connect to a remote computer in a different location. If you don't use it, then you can remove it.
 
Remote Desktop Connection is a technology that allows you to sit at a computer (sometimes called the client computer) and connect to a remote computer (sometimes called the host computer) in a different location. For example, you can connect to your work computer from your home computer and have access to all of your programs, files, and network resources as though you were in front of your computer at work. You can leave programs running at work and then, when you get home, you can see your work computer's desktop displayed on your home computer, with the same programs running.
 
For more information on "Remote Desktop Connection", visit the following link:
http://windows.microsoft.com/en-us/windows7/Remote-Desktop-Connection-frequently-asked-questions
 
What is Remote Media Server?
A media server is a device that simply stores and shares media. If any media is shared via a Remote Media Server, then it is called as Remote Media Server. If you don't want any media to be shared, then you can stop it on the computer.
 
Hope this is helpful.

saltgrass say:

Well, you are getting a wide variety of answers.  Sort of shows how too many terms are being applied to too many specific processes.

If you have a System Image Backup, you can restore it by using the Recovery options from the Charms bar and drilling down the Advanced options until your get to the Restore System Image.

When you select that, it will reboot and load that utility and then look for an image to use.  When it finds a WindowsImageBackup folder with a valid image, it will offer to use that one.  If you have another one in a different location you want to use, tell it to find that one.  If you forgot to plug in an external drive, you can do that now and have it search for that image.

When you re-image a system, it will overwrite the Hard drive with the same configuration and data contained in the image.  So if there is any data on a partition you did not include in the image, you need to move it first.

Occasionally, there may be a situation where the process asks you to insert boot media.  Probably because the Recovery options are not set up correctly.

I normally try to use the same version of a recovery drive as the version the image was made with.  If you have install media, you may have a better chance of using different OS versions since you won't have to sign in to the install prior to re-imaging.

A license key is not needed when restoring a System Backup Image.

srfreeman say:

The 'SYSYTEM RESERVED' partition has been around since Windows 7 or before. It is simply a utility (stub) partition that is created when the OS is installed on a drive that is not partitioned/formatted. It contains things like boot information and encryption information.

This will not be seen on systems where the OS was installed on a partitioned and formatted drive.

A couple of requirements for the 'SYSYTEM RESERVED' partition.

BIOS/MBR: May be formatted as either FAT32 or NTFS.

UEFI/GPT: Must be formatted as FAT32.

Very simply put, a utility partition is for system use only. The Defrag utility in Windows will not act upon a utility partition. Your system is working as designed. The label is there for a reason.
