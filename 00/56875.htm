In a word, No.

We have the same windows login credentials for that PC but I want Windows Live Mail to ask for the correct user info. 

Windows Live Mail is a strictly per-Windows-user application. Each Windows user has his own store folder for messages and calendars and his own contacts list, in just the same way as he has his own personal folders for documents and music and pictures.

To keep personal stuff private, each user should have his own Windows user account with a password or other authentication token. If all users need to access the same documents etc., keep them in the Public folders - that's what they're for.

If all users share the same company contacts list and/or calendar, have them sign in in Windows Live Mail with a company Microsoft Account.
