One of the reasons why Outlook 2010 search function may not be working is that the search function could be turned off completely. There is a possibility that people who have updated to Outlook 2010 from older version during the updating process, the search becomes disabled and all you have to do is enable it again to get it in working order. Here’s how:

  1. Close Outlook completely.
  2. Go to Control Panel.
  3. Click on Programs.
  4. Select Programs and Features.
  5. Turn Windows Features On or Off.
  6. Make sure there’s a check mark on the Indexing Service check box and hit OK.
  7. Open Outlook again & check if the search bar is working.

If you find the Outlook 2010 search function still isn’t working, then you’re facing a different kind of indexing issue which will require an index rebuild.
