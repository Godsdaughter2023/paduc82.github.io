SELECT Counties.County, Districts.District, Parishes.Parish, Locations.Location

ken sheridan say:
Is it possible to use of Applyfilter command for the above example and use of it for each separately and we take the same result ?

No, the value for the filter property can only be applied at one time.  You can build up a string expression over several lines and then apply the resulting string expression to the filter property, however. If doing this you would not normally test for a Null parameter as part of the expression, but only include a string in the final expression if the parameter is not Null.  The following is code from my MultiSelect demo which does this.  In my case it opens a report, but the same could be done to assign the SQL statement to the current form's RecordSource property, or to assign the part of the string which constitutes the WHERE clause (the value of the strWhere variable less the leading ' And ') to the form's Filter property:

    Dim strSQL As String
    Dim strWhere As String
   
    strSQL = "SELECT Employees.*, FirstName & "" "" & Lastname AS FullName, " & _
        "Projects.Project FROM Projects " & _
        "RIGHT JOIN (Employees LEFT JOIN ProjectEmployees " & _
        "ON Employees.EmployeeID=ProjectEmployees.EmployeeID) ON " & _
        "Projects.ProjectID=ProjectEmployees.ProjectID WHERE True"
   
    If Not IsNull(Me.cboDepartment) Then
        strWhere = strWhere & " And Department = """ & Me.cboDepartment & """"
    End If
   
    If Not IsNull(Me.cboCity) Then
        strWhere = strWhere & " And City = """ & Me.cboCity & """"
    End If
       
    If Not IsNull(Me.txtDateFrom) Then
        strWhere = strWhere & " And DateAppointed >= #" & _
            Format(Me.txtDateFrom, "yyyy-mm-dd") & "#"
    End If
  
     If Not IsNull(Me.txDateTo) Then
        strWhere = strWhere & " And DateAppointed < #" & _
            Format(Me.txDateTo, "yyyy-mm-dd") & "#+1"
    End If
   
    strSQL = strSQL & strWhere & ";"
   
    DoCmd.OpenReport "rptEmployees", view:=acViewPreview, OpenArgs:=strSQL

1. Go over to an application where the duplex or two-sided checkbox is available.  Safari works.

5. Go back to Word and print.  When you want "My Duplex", just select it from the presets menu.
