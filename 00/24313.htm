You could use On Error Resume Next to suppress error messages:

Sub cc()
    Dim fs As Object
    Dim i As Long
    Application.DisplayAlerts = False
    Set fs = CreateObject("Scripting.FileSystemObject")
    On Error Resume Next ' Suppress error messages
    For i = 1 To 9
        fs.CopyFile "C:\Users\Lawrence\Desktop\2011 Jan-Jun\AUDCAD\AUDCAD 2011.01.0" & i & ".xlsx", _
            "C:\Users\Lawrence\Desktop\DB\2011\01 Jan\" & i & "\"
    Next i
    Application.DisplayAlerts = True
End Sub

I also indented the code, declared the variables, placed Set fs = ... before the loop (no need to create fs 9 times), and reset DisplayAlerts to True at the end.
