Stop Contacts from reversing names - Microsoft Community

anteaus. say:

Stop Contacts from reversing names

Windows Mobile 5. Any idea how to stop the Contacts list from reversing every name that's entered? For example, 'John Smith' becomes 'Smith, John.'

Obviously the guy who hardcoded this 'cool trick' thought it was clever. Most users think it's a damn-blasted nuisance.

Been looking for a fix to this for a long time, just thought I'd ask again as the subject came up.

 

julie fka profjulie say:

The default in Outlook is to display names to Smith, John (lastname, firstname) and the default in the Contacts application on windows mobile devices is also to display store names like this.  On Outlook on your computer, you can change this by opening up Outlook, then selecting Tools > Options > Contact Options (found under the Preferences tab) > change the Default "Full Name Order to "First (Middle) Last"."  You'll also change the default setting for how to file new contacts similarly (both these options are found on the same screen).

There's no way to change the default setting in the Contacts application on your windows mobile device, but the contacts that you sync over from your computer, should display the way you've saved them on your computer (First Middle Last).  If you input a new contact on your mobile phone, you'll need to be sure to specify how you want the name to be filed in the "File as" field in the contact record.
