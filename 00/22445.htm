graham mayor say:
Sounds like a retail version to me! If you check the wording of the licence, which you can access from Outlook File > Help, you will probably find that you can install it and activate it on both PCs.

If the licence does not indicate that you may install it on two devices, then you may still transfer it to the new PC.

Install it using the same product key from the packaging, and if you are unable to activate it on line, use the telephone activation. Hold on without pressing any buttons on your phone until an operator answers and tell the operator that you are transferring the activation to a replacement PC.

It doesn't really matter at what point you uninstall from the old PC. It will continue to work to allow you to make the transfer of data.

Assuming that you are not connecting to a company Exchange server, your data will be in a PST file, stored by default in the folder %appdata%\..\Local\Microsoft\Outlook. (%appdata%\..\Local Settings\Application Data\Microsoft\Outlook in Windows XP)

Copy that file to a flash drive, then to the new PC (or use whatever other transfer method you prefer). Open the PST file from Outlook and then set the newly opened file as the default data source - File > Info > Account Settings > Account Settings > Data Files
