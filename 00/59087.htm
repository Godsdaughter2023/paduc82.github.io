bossdweeb say:

"in time of playing games or running a large program like Photoshop or HDD working at 100% sound is higher than usual."

Some HDD's make more noise than others, and some perfectly healthy drives do sound a bit 'grindy' when reading/writing.

However, what you may be hearing in these scenarios is what is called 'thrashing'.  Win10 is supposed to do a better job of maintaining

HDD's than previous Window versions, but if the HDD sounds like it's working extra hard, and it sounds like the head is moving

rapidly back and forth more than usual when a game is loading levels, or you are loading large files the HDD may need optimising.

Traditionally, thrashing was more prominent when a HDD has not been defragmented and optimised in a long while which causes

games or large files to spread the data unevenly over a large area on the platter (particularly if you have installed several software/games,

and a lot of large files  in short order), which can cause the head to have to move back and forth more than is desirable when the drive

is being read.

The optimising process, which is done by the defragmenting tool moves associated files closer together which reduces head movement

during reading.

You can run Windows Defragmenting tool manually if you suspect it needs to be better optimised.

(Start Menu > Windows Administrative Tools.)

I generally defrag/optimise after installing a large game, and after doing a Win10 version update (if using an HDD rather than SSD for Windows).

- Disabling the Page File, which has been popular since more people started having large quantities of RAM, can also

contribute to thrashing.
