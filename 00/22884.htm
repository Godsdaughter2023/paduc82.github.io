How to set Windows Defender to run on a specific time - Microsoft Community

lenora wilke say:

How to set Windows Defender to run on a specific time

Windows Defender runs a complete scan only when I manually set it to.  How can I make it do a complete scan on a fixed time

Original title : Windows Defender

greginmich say:

Windows Defender runs a complete scan only when I manually set it to.  How can I make it do a complete scan on a fixed time

Setup for a Scheduled Full Scan:

Open Task Scheduler (Start button > All apps > Windows Administrative Tools > Task scheduler)

1.) In the right-hand (Actions) panel, click on Create Basic Task… and name the task “WD Full Scan” and click Next

2.) Click on the Weekly option and click Next, and then pick a time and click Next

3.) We’re using the default action (Start a program) so just click Next

4.) Browse to (or just copy and paste) "C:\Program Files\Windows Defender\MpCmdRun.exe" in the Program/script box (with quotation marks)

5.) Enter -Scan -ScanType 2 in the Add arguments box (no quotation marks) and click Next

6.) Check the Open the Properties dialog for this task… box and click Finish

7.) Click the Change User or Group… button and type "system" and click OK. Then click on the Conditions tab, and check the Wake the computer to run this task box and click OK

If you want a quicker way to schedule your Full Scan, then you can use SchTasks.exe: Right-click on the Start button and select Command Prompt (Admin). Then copy, paste, and run (enter) this command line:  

SchTasks /create /sc WEEKLY /tn “Defender Full Scan” /st 19:00  /ru SYSTEM /rl HIGHEST /tr "'C:\Program Files\Windows Defender\MpCmdRun.exe' -Scan -ScanType 2"

You can modify the scheduled start time (/st 19:00) either before running the command, or in the Task Scheduler Library after the task is created. This scan should be set to start at a time when the PC will normally be in use, since it won’t run when the PC is sleeping, and it won’t run on startup after a scheduled start is missed. If you want a more reliable scan, then you’ll have to add these features. You would check the Wake the computer to run this task box (Conditions tab), and the Run task as soon as possible after a scheduled start is missed box (Settings tab). In order to get your scan to wake up the PC, you might also have to turn off Require a password on wakeup in Power Options and/or enable wake timers in Power Options > Change when the computer sleeps > Change advanced power settings.

Also, if you want your scheduled scans to run at full speed, then you’ll have to run this command at the elevated PowerShell prompt:

Set-MpPreference -ScanAvgCPULoadFactor 100

Option Explicit
Function GetDescription(InputDescription As String) As String
Dim n As Long, x As Long, count As Long
     'For loop that checks each character of the description string
     For n = 1 To Len(InputDescription)
         'First if checks for a number character, ascii code 48 to 58
         If Asc(Mid(InputDescription, n, 1)) > 47 And Asc(Mid(InputDescription, n, 1)) < 58 Then

                 'For loop checks if the next five characters are numbers, ascii code 48 to 58
                 For x = 1 To 5
                     If Asc(Mid(InputDescription, (n + 1 + x), 1)) > 47 And Asc(Mid(InputDescription, (n + 1 + x), 1)) < 58 Then
                         count = count + 1
                     End If
                 Next x
                 If count = 5 Then
                     GetDescription = Mid(InputDescription, n, 7)
                     Exit Function
                 End If
             End If
         End If
     Next n
 End Function
