bobgreenblatt say: Assuming the names are in column A in both worksheets, then,on worksheet B in column B (or any other convenient column) enter the formula:
=Match(a1,worksheetb!$a:$a,0)

fill this formula down for the 46K rows. The column will show a number if the names match, or #N/A if they do not.

Remember that this assumes the names are identical, and first and last names are in a single cell.

This problem can occur if an Internet Explorer index file is damaged.

 

Method 1: Delete the Temporary Internet Files and check if that helps.

 

Refer the following link to do the same:

 

How to Delete the Contents of the Temporary Internet Files Folder

Method 2: Delete the Internet Explorer Index.dat file

a.    Click Start and type cmd. Right-click cmd on the Start menu, and then click Run As Administrator.. If you are prompted for an administrator password or confirmation, type your password or click  Continue. 

b.    Change the directory to the following path:

cd “drive:\Users\Username\AppData\Local\Microsoft\Windows\Temporary Internet Files\Content.IE5″ whereusername is the name of the user account that has the problem.

c.    Type del index.dat, and then press Enter.

d.    Close the command prompt, and then restart the computer.

The Index.dat file is recreated the next time that you log on to the computer.
