Hi LDH04,

Application and network related errors are the usual cause why issues with downloading notes are occurring. For initial troubleshooting, I suggest uninstalling and reinstalling your OneNote for Android app. Then log out and log back into OneNote (in your phone) to reset your credentials, this is to avoid authentication problems. You can also force OneNote to sync. Follow these steps to do so:


Reset credentials

  1. Open OneNote, tap the menu in the upper-right corner of the screen, and choose Settings.
  2. Tap Account.
  3. Tap Sign Out, and then click OK.
  4. Tap the recent apps button on the bottom of the screen (the one next to your home button), find the OneNote app and swipe it sideways to close it.
  5. Start OneNote and sign in to your account.

Force Sync

  1. In OneNote, find the notebook that's experiencing the sync issues.
  2. Tap the menu in the upper-right corner of the screen, and choose Sync all.
  3. Leave the device on until the sync is done. You may have to leave the device on overnight, especially in the case of very large notebooks or first-time syncs.

Private Function HimetricToPointsX(X As Single) As Single
  HimetricToPointsX = X / (254 / TwipsPerPixelX) * TwipsPerPixelX / 20
End Function

Private Function HimetricToPointsY(Y As Single) As Single
  HimetricToPointsY = Y / (254 / TwipsPerPixelY) * TwipsPerPixelY / 20
End Function

Function TwipsPerPixelX() As Single
  Dim hDC As Long
  Static Value As Long
  If Value = 0 Then
    hDC = GetDC(0)
    Value = TWIPSPERINCH / GetDeviceCaps(hDC, LOGPIXELSX)
    ReleaseDC 0, hDC
  End If
  TwipsPerPixelX = Value
End Function

Function TwipsPerPixelY() As Single
  Dim hDC As Long
  Static Value As Long
  If Value = 0 Then
    hDC = GetDC(0)
    Value = TWIPSPERINCH / GetDeviceCaps(hDC, LOGPIXELSY)
    ReleaseDC 0, hDC
  End If
  TwipsPerPixelY = Value
End Function

Function HimetricToPixelsY(Y As Single) As Single

Sub InsertPicture(ByVal FName As String, ByVal Where As Range, _
    Optional ByVal LinkToFile As Boolean = False)
  'Inserts the picture file FName as link or permanently into Where
  Dim SR As ShapeRange
  Dim S As Shape
  Dim W As Single, h As Single
  'This may take a while, let the user know that we work
  Application.Cursor = xlWait
  'Don't flicker with the screen
  Application.ScreenUpdating = False
  'Import the picture as link
  Set SR = Where.Parent.Pictures.Insert(FName).ShapeRange
  If LinkToFile Then
    'Move the picture to the cell
    SR.Left = Where.Left
    SR.Top = Where.Top
    'Scale it to fit the cell
    SR.Width = Where.Width
    If SR.Height > Where.Height Then SR.Height = Where.Height
  Else
    'Save the width and height
    W = SR.Width
    h = SR.Height
    'Remove the link
    SR.Delete
    'Insert it permanently
    Set S = Where.Parent.Shapes.AddPicture(FName, False, True, Where.Left, Where.Top, W, h)
    'Keep the proportions
    S.LockAspectRatio = True
    'Scale it to fit the cell
    S.Width = Where.Width
    If S.Height > Where.Height Then S.Height = Where.Height
  End If
  Application.Cursor = xlDefault
  Application.ScreenUpdating = True
End Sub

Using the shape.AddPicture method the size of the picture is a required parameter and as such the photo isresized without considering aspect ratio.


The code example below uses Shapes.AddPictures to insert the picture with the aspect ratio plus how to resize and maintain the aspect ratio.

   Sub InsertPicture()
        
    Dim strFileName As String
    Dim lngLeft As Long
    Dim lngTop As Long
    Dim shpPic As Shape
   
    strFileName = "C:\Users\User\Pictures\MyPicture.jpg"
       
    lngLeft = Columns("C:C").Left
    lngTop = Rows(10).Top
   
    'Note: The -1 parameters for Width and Height to maintain the aspect ratio
    Set shpPic = ActiveSheet.Shapes.AddPicture(strFileName, False, True, lngLeft, lngTop, -1, -1)
   
    'If re-sizing only re-set the Height or the Width (Not both) to retain aspect ratio
    With shpPic
        .Height = 100
    End With
   
End Sub

Chrysler Uconnect compatibility and features with WP 8.1 and 10. - Microsoft Community

ritzybiscuit7 say:

Chrysler Uconnect compatibility and features with WP 8.1 and 10.

 

Here is the compatibility of using FCA (Fiat Chrysler Automobiles) Uconnect system for Windows phones.

I tested the Uconnect system on a Chrysler 200 limited 2016 with Uconnect 8.4 (non navigation-- which should cover Uconnect 5.0) on two Nokia Lumia 635(s). (both Virgin Mobile/Sprint phones)

One of the phones was running Windows phone 8.1 and the other was running Windows 10 Mobile (November update).

Both were connected via Uconnect's Bluetooth paring system and with the latest updates for Groove Music and Xbox music.  (note Uconnect 8.4 or 8.4 nav are the same just with different features and is the bigger version of the Uconnect systems.)

Windows phone 8.1

Cortana: does not work with the system at all.

Media: Xbox Music works perfectly with Uconnect 8.4, Users can browser and select music that is even in a users collection purchased selection for it or with music pass. (including songs not on the device that causes streaming to occur)  Uconnect sorts a users music by artist, album, genera, etc.  Pausing, skipping, shuffle, etc. works great. Playing videos from Youtube works too including pausing and playing from Uconnect. (But do note that if a person has a 3rd party music app they would have to select the music in the app rather than through the Uconnect system.)

contacts: Contacts show up and can call people.

calling: Can receive and make calls and plays through system. (Audio seemed low when in phone call but that could be my settings on voice calls)

Windows 10 mobile.

Cortana: Works with Uconnect in a weird way When I was playing around on my phone I hit Cortana with the search button and she came through the speakers and told a joke. (or use commands like play my music or an artist) But I noticed that Uconnect acted like it made a call on the "Phone" section. So I hit "redial" and it called 555-555-9876 and when it was dialing, my phone pulled up Cortana from sleep and started talking to me. Long story short, use redial after making Cortana come up through the search button first on the phone. Then the Uconnect system acts like a user is talking to Cortana.  All the features of Cortana works: like checking the weather, set  an alarm, Play (song) by (artist) or "what's Playing."

Media: Pause and play work, Shuffle sometimes works. But selecting music like in Windows phone 8.1 is grayed out and users have to end up using Cortana or using the phone to select music 

Contacts: Cannot bring up phonebook on Uconnect for some odd reason.Calling: Works like in 8.1

Bugs and fixes.

(Windows 10 Mobile only) If a user attempts to use hands free Cortana after turning off the car and turning the car back on. Uconnect will attempt to call a blocked number and require cell connection which would not work because it's a fake phone number. To resolve this issue force Cortana to open voice command by holding down the search button on W10M. Then the Uconnect user can hit "redial" on the "phone" page and pull up Cortana.

(Both 8.1 and 10) If a Uconnect user turns on their vehicle and manually connect a device to the Uconnect system and it shows a box with a blue music note on the "media" page and no text shows up after 15 seconds then the user will have to unpair and repair the device back to the car's Uconnect system (under Bluetooth pairing settings on Uconnect)

 

(both 8.1 and 10) if Audio seems low confirm that the Windows phone Bluetooth audio volume is set to 22-30. (30 normally works the best.)

(Windows 10 Mobile only) If your device does not automatically connect, pull down the action center and hit the connect app. Which will scan for devices and connect to the Uconnect system.

I hope this helps owners of Windows phones Fiat Chrysler products know what they can do or not do with a Uconnect system. When a owner of a car goes to Mopar.com to look at help on connecting Windows phones are not an option.

If anyone else has anything to add or can test out Uconnect 5.0 (small screen) or Radio 3.0 (no screen) that would be great.
