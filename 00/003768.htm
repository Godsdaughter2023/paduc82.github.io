gord dibben say:

This macro will filter each sheet by the criterion you enter for a chosen column.

Sub Filter_Stuff()
    Dim whatwant As String
    Dim colnum As Integer
    Dim wkSht As Worksheet
    colnum = InputBox("enter a column number")  '3 for column C
    whatwant = InputBox("enter criterion. e.g ABC  or *ABC*")
    For Each wkSht In ThisWorkbook.Worksheets
        Range("A1").Select
        With ActiveSheet.UsedRange
            Selection.AutoFilter
            Selection.AutoFilter Field:=colnum, _
                    Criteria1:=whatwant, Operator:=xlAnd
        End With
    Next wkSht
End Sub

With your workbook open hit Alt + F11 to go to Visual Basic Editor.

Select your workbook/project and right-click>insert>module.
Paste the macro into that module.
Alt + q to return to the Excel window.
Save the file as is then Developer>Macros
Select the macro name and "Run" to activate the procedure
