suzanne s. barnhill say:

When you have "Different Odd and Even" enabled, the Odd Page Footer is completely separate from the Even Page Footer and (if "Different First Page" is also enabled) the First Page Footer. And of course any of these can be different in any given section if the specific footer is unlinked from the previous one.

In the section where the Odd Page Footer is incorrect, check to see whether it is Linked to Previous. If so, unlink it and correct it.

Or go back to the previous section, which may be too short to have any odd pages, in which case the Odd Page Footer will be inaccessible unless you add an odd page.

chip pearson say:

Try something like

Sub AAA()
    On Error Resume Next
    Debug.Print ThisWorkbook.Worksheets("Sheet2").Name
    If Err.Number <> 0 Then
        ThisWorkbook.Worksheets.Add.Name = "Sheet2"
    End If
End Sub

Try this modified macro:

Sub CheckAddSheet2()
Dim WS As Worksheet
Dim CurrentSheet As Worksheet
Dim NewSheet As Worksheet
Set CurrentSheet = ActiveSheet
For Each WS In Worksheets
  If WS.Name = "Sheet2" Then Exit Sub
Next
Set NewSheet = Worksheets.Add(After:=CurrentSheet)
NewSheet.Name = "Sheet2"
CurrentSheet.Activate
End Sub
