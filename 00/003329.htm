teylyn say:

Hello,

Excel does not have a function for adding a comment to a formula, but you can use a workaround. The N() function tries to convert its argument to a number, and if it can't it will return a 0. That can be used to put comments into formulas, for example

=SUM(A1:A10,N("the last ten sales values"))*(C2+N("the expected margin"))

joeu2004 say:

And if the formula is text, you can do something like:

=A1 & B1 & IF(FALSE, "This is my comment", "")

[EDIT] Better:

=A1 & B1 & T(N("This is my comment"))

It shows up normal while typing, but disappears in print preview and ultimately does not show when printing. The margins are normal at 1" around. I have no idea what happened and why is is no longer appearing. Never have had this problem before.
doug robbins - mvp office apps & services (word) say:

Print Preview normally gives a pretty accurate representation of what the active printer is capable of printing, so, even though I would have thought 0.5" would not cause a problem, maybe the printer cannot print in that area of the page.  (nearly all printers have an unprintable area around the perimeter of the sheet.

 

Try downloading the free PrimoPDF utility which will install itself as a printer and can be used to print a document to a pdf file and see if you have the same issue with that.  If not, I would suspect that the issue is with the printer\printer driver combination.  You might check if there is an updated driver on the printer manufacturer's website and see if that overcomes the problem.  You could also try printing the pdf created by Primopdf (assuming that it does not have the same problem).  Otherwise, it may be necessary to move the header further away from the top of the page.
