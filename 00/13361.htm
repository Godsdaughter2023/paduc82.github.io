ken sheridan say: Are you by any chance trying to format the return value as hours:minutes?  If so then calling the DateDiff function with "h" as the interval won't work; it returns the difference in hours as an integer.  You can use an expression which returns the difference in minutes and then, by using integer division and the Mod operator convert this to hours:minutes.  You can see how this works in the debug window, e.g.

? DateDiff("n", #2012-11-06 10:38#,  #2012-11-08 16:41#)\60 & ":" & Format(DateDiff("n", #2012-11-06 10:38#,  #2012-11-08 16:41#) Mod 60,"00")
54:03

Or you can use the following function which returns the difference in the format hh:nn:ss or optionally d:hh:nn:ss:

Public Function TimeDuration(dtmFrom As Date, dtmTo As Date, _
            Optional blnShowdays As Boolean = False) As String
            
    ' Returns duration between two date/time values
    ' in format hh:nn:ss, or d:hh:nn:ss if optional
    ' blnShowDays argument is True.
    
    ' If 'time values' only passed into function and
    ' 'from' time is later than or equal to 'to' time, assumed that
    ' this relates to a 'shift' spanning midnight and one day
    ' is therefore subtracted from 'from' time

    Dim dtmTime As Date
    Dim lngDays As Long
    Dim strDays As String
    Dim strHours As String
    
    ' subtract one day from 'from' time if later than or same as 'to' time
    If dtmTo <= dtmFrom Then
        If Int(dtmFrom) + Int(dtmTo) = 0 Then
            dtmFrom = dtmFrom - 1
        End If
    End If
    
    ' get duration as date time data type
    dtmTime = dtmTo - dtmFrom
    
    ' get whole days
    lngDays = Int(dtmTime)
    strDays = CStr(lngDays)
    ' get hours
    strHours = Format(dtmTime, "hh")
    
    If blnShowdays Then
        TimeDuration = lngDays & ":" & strHours & Format(dtmTime, ":nn:ss")
    Else
        TimeDuration = Format((Val(strDays) * 24) + Val(strHours), "00") & _
            Format(dtmTime, ":nn:ss")
    End If
    
End Function

? TimeDuration(#2012-11-06 10:38#,  #2012-11-08 16:41#)
54:03:00

? TimeDuration(#2012-11-06 10:38#,  #2012-11-08 16:41#, True)
2:06:03:00

Note that when you return a formatted value like this the value is a string expression, so you cannot do further arithmetic on it.  You should always do any arithmetic on the underlying date/time values and only format the results as the final stage.
