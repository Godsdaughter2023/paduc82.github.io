erdeslawe say:

This thread is getting very confusing:

The windows.old folder is not a Windows 10 installation file.  It is what it says, i.e. an image of your old system which you can use to roll-back to that system for up to 30 days, after which it will be automatically removed from your system.

If you've already carried out the Windows 10 upgrade and haven't created any Windows 10 Media, you can download the relevant ISO image from Microsoft for future use.  The media creation Tool is only really of use if you are still running an 'old' OS (Windows 7/8.1).  Make sure you use the inbuilt utilities to create Startup Media and System/Data Backups as with any OS.

If you are running an activated version of Windows 10 then you shouldn't be asked for a Product Key when you try to do a clean install as the Product Code is embedded on your upgraded 'device'.  So, check your Windows 10 activation status.
