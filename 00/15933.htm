rena yu msft say:

Hi Luxury,

Please follow the steps below to see if the issue can be resolved:

  1. Make sure the file you want to sync isn’t larger than 15GB.
  2. Make sure the file path isn’t too long. Example of a file path - C:\Users\<UserName>\Pictures\Saved\2014\January\Holiday\NewYears\Family…The entire path, including the file name, must contain fewer than 400 characters. Shorten the name of your file or the name of subfolders in OneDrive, or select a subfolder that's closer to the top-level folder.
  3. Go to Site settings>Site Libraries and Lists>Customize “Documents”>Check if there’s any required column. OneDrive can’t start to sync the library or list including a required column.
  4. Go to Site settings>Site Libraries and Lists>Customize “Documents”>Advanced Settings>Check if you have enabled “allow items from this document library to be downloaded to offline clients”. If the option is set to “no”, you are not able to start to sync.
