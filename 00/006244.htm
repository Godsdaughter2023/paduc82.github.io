jackrabbit181 say:

You can make use of the INDIRECT function.

 

Assuming you have your data in Column A1:A500, you can get 50 lines to a page and 5 columns to a page when you print. Then for your "PRINTED" version;

  1. In cell B1 you use the formula =INDIRECT("A"&ROW()) and copy this down to row 50.

  2. In cell C1 you use the formula =INDIRECT("A"&ROW()+50) and copy this down to row 50.

  3. In cell D1 you use the formula =INDIRECT("A"&ROW()+100) and copy this down to row 50.

  4. In cell E1 you use the formula =INDIRECT("A"&ROW()+150) and copy this down to row 50.

  5. In cell F1 you use the formula =INDIRECT("A"&ROW()+200) and copy this down to row 50.

  6. and so on until you have covered the number of rows in your data table.

Now if you delete any of the entries in the original list the entries in your "PRINTED" version will update automatically.

 

If you have the "PRINTED" data starting on different rows because of headings etc, just subtract that many rows less 1 from the above formula. The ROW() function picks up the row in which the formula currently sits so you need to tell it how many more or less to look at if the data you want is higher or lower. For example if you place the first formula in B10 but the data still starts in A1 then you should -9 so it picks up from A1 not A10.

Lightly tested...

I would leave the actual data in a single column and put the data for printing in a separate location.

So, suppose your data are in Sheet1 column A starting with A1.

Then, in Sheet2 cell A1 suppose you have the number of entries you want in each column before wrap around occurs.

Now, in Sheet2 Cell A2 enter the formula =IF(OR(ROW()-ROW($A$2)+1>$A$1,$A$1*(COLUMN()-COLUMN($A$2))+ROW()-ROW($A$2)+1>COUNTA(Sheet1!$A:$A)),"", INDEX(Sheet1!$A:$A,Sheet2!$A$1*(COLUMN()-COLUMN($A$2))+ROW()-ROW($A$2)+1))

Copy A2 down column A until you get one blank entry.  Now copy A2:A{last-cell-with-formula} across the columns until you get a column with all blank entries.

For my test I had 1000 entries in Sheet1, and in Sheet2!A1 I had 100, i.e., 100 rows per column before wrap around.

So, I copied A2 all the way down to A102 and then copied A2:A102 across to B:K.
