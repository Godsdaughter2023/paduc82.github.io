a. user say:

My Dell Inspiron 13-5368 still hasn't received Update 1809 ...

You're not alone. AdDuplex reports that only 12.4% of PCs were running the October 2018 Update in January (2019.)

You may have heard that the October 2018 Update has been plagued with one problem after another. For that reason, Microsoft has been very conservative about deploying this update and as a result, very few people are getting it. I haven't received it yet either.

The smartest thing to do is to wait for the update to come to you, for two important reasons:

1- Microsoft is using its telemetry to try to determine which PCs can successfully get v.1809. This helps it to avoid sending the update to PCs that might not work properly after the update. Microsoft can even rollback an update before it makes any changes to your computer, if Setup determines that the update is not going to install successfully. That's why people sometimes see the "No changes were made to your computer" message. You forgo both these advantages if you install the update manually.

2- Microsoft recently admitted that users who manually download updates are considered to be advanced users, and for that reason the version of the update they receive can include experimental code. Do you really want to have experimental code running on your computer?

For both these reasons the smart thing to do is to wait until Microsoft sends the update to you. A new update is scheduled to be released in April - barely two months away - and since feature updates are cumulative you will receive the October 2018 Update along with the new update.

Nobody needs to have a new update as soon as possible. For decades, large organizations (and smart users) have waited months after the release of a new, major feature update before adding it to their computers. This allows time for early bugs to be discovered and hopefully fixed. Only then is it time for the rest of us to install the update.

If more people had followed the conventional wisdom, the bad effects of the October 2018 Update would have hurt far fewer people.
