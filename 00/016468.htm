Here's a quick and easy way to duplicate the behavior. 

On Sheet1, select C3. Go into Formulas, Name Manager and create a named reference called TinCan which Refers to: Sheet1!B2. Note that the Refers to: is a relative reference.

Exit the Name Manager and select Sheet1's A1. Go back into the Name Manager and check the Refers to: address. It should now be Sheet1!XFD1048576. _charles_williams say:
The difficult with using relative addressing in defined names is that the anchor cell does not really exist until you actually use the defined name in a formula in a cell.

So if you want to use relative references in Defined Names its much better to switch to R1C1 mode rather than A1 mode: its then much easier to work out what Excel is going to do (internally Excel always works in R1C1 mode).

So in your example the refersto of TinCan is Sheet1!R[-1]C[-1] and it never changes regardless of whatever cell is selected.
But the cell it actually refers to is relative to either the active cell (when you look at it in Name manager) or the formula that you use it in.

jay freedman say:

The little triangles that you see next to headings in Word 2013 appear only in Word 2013. Although earlier versions of Word can open documents created in 2013, those users will see only the fully expanded text.

There is no such thing as a "collapsible/expandable file" -- it's strictly a matter of Word 2013 adding functionality to headings that have particular styles (technically, what outline level is applied to each paragraph, but that's usually a property of the paragraph style). The same document opened in earlier versions still has the same styles, but in that case the program doesn't supply the functionality.

The answer to your last question depends on what you want the "protection" to do. If you expect to be able to prevent collapsed headings in the document from being expanded, the answer is no. Passwords can be applied to determine whether the document can be opened, or whether it can be edited/reformatted.

For a method that does work in all versions, see http://gregmaxey.mvps.org/word_tip_pages/toggle_data_display.html.

a. user say:

Change the keyboard shortcut for a macro
  1. Click the Microsoft Office Button , and then click Word Options.
  2. Click Customize.
  3. Next to Keyboard shortcuts, click Customize.
  4. In the Categories list, click Macros.
  5. In the Macros list, click the macro that you want to change.
  6. In the Press new shortcut key box, type the key combination that you want to choose.
  7. Check the Current keys box to make sure that you aren't assigning a key combination that you already use to perform a different task.
  8. In the Save changes in list, click the option that matches where you want to run your macro.

     Important   To make your macro available in all documents, be sure to click Normal.dotm.

  9. Click Close.

I hope this helps.

Good luck!

The document has probably always been in [Compatibility Mode] and you just did not notice it.
