jeeped say:
Excel 2007/2010 has the AVERAGEIFS() formula for multiple criteria. Try,

=AVERAGEIFS(A1:A10,A1:A10,">=25",A1:A10,"<=50")

.. where A1:A10 is the range to be both averaged and the range to base the criteria upon. Note that I've used less than or equal to and greater than  or equal to as the comparison operators.

Earlier versions of Excel can use SUMPRODUCT() to construct the component parts of an average, e.g. sum/count.

=SUMPRODUCT((A1:A10>=25)*(A1:A10<=50)*(A1:A10))/SUMPRODUCT((A1:A10>=25)*(A1:A10<=50))

Resolution depends on a bunch of factors... here's some of them.

  • What is the AutoRecovery save time set at? Less than 10 minutes can become a problem depending on what else is running and making copies.
  • Do you have Time Machine running? What is the frequency of backup there?
  • Where are the AutoRecovery files being placed? Are you using the default location or have you set a custom location from Word > Preferences > File Locations?
  • When was the last time you powered down your system?
  • What is your version of Office? Is is fully up-to-date?
  • Have you run the Disk Utility.app program and repaired permissions on your Primary Drive? If not you should and then restart your system after it complete.
  • Does the document you are trying to save have any special characters in the file name?
