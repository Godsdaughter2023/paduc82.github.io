wunders say: Log in as a different user and see if that user has the same problem.  If not, and it only affects the one user account, then you probably have a corrupted profile.  To recover from a corrupted profile, try one of the following:

"Windows XP Problems if Your Profile Is Damaged"
  < http://support.microsoft.com/kb/326688 >

"How to recover the damaged user profile in Windows XP"
  < http://support.microsoft.com/kb/555473 >

Usually this is accompanied by disk file system corruption as well, so it is recommend that you perform a chkdsk operation with the /F or /R option.  This is a good regular maintenance step but be warned that it can take a couple of hours to perform and can't be interrupted.

"How to perform disk error checking in Windows XP"
  < http://support.microsoft.com/kb/315265 >
