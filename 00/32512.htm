From the previously available FAQ on Vault Credits:

I claimed Rewards Credits from the vault. When will I receive them?

If you claim Rewards Credits from the MyVIP Vault, they should appear as pending in your account within 96 hours. Once they are pending in your account, the normal deposit process kicks in. bill smithers say:

Hi,

No matter whether you did a clean install of Windows 7 or upgraded to windows 7 from Vista, that Vista OEM Product Key is locked to that particular computer and can not be transferred to another machine.

You can only use that Vista OEM Key to reinstall Vista on that particular machine and start again.

Licensing Rules (my words)

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) or is an OEM System Builders License, that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) and OEM System Builders License are tied to the first machine it is installed on and dies when that machine dies.

Only Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

And Retail Licenses can also only be installed on one machine at any particular time.

@@@@@@@@@@@@@@@@@@@@@@@

You rec'd the free upgrade to 10 from 7, as Vista never qualified for the free upgrade to 10.

The Windows Product Key/License for the Qualifying Operating System, Windows 7, Windows 8.1, etc. is absorbed into the Windows 10 Upgrade during the installation process and becomes part of the Activated final install of Windows 10.

The Qualifying Operating System License, whether it be Windows 7 or Windows 8.1, etc. can not be used elsewhere, or in a dual boot with Windows 10.

But there is no Product Key for your free upgrade to 10.

You Laptop's Hardware configuration is stored on Microsoft's Servers and that how your machine will be recognised if you ever wish to reinstall 10.
