Press CTRL+ALT+DEL or CTRL+SHIFT+ESC and examine the running processes. (click the header to sort them alphabetically). Is the process that fails to start already listed e.g. Outlook.exe (a frequent offender)?  If it is, end the process. You should then be able to restart it.
