rxaverius say:

Scrolling a docx exactly page after page with a keyboard shortcut

Which would be the way to scroll a doc EXACTLY page after page, that means, that the insert point appears exactly at the first line for each page and that could be seen

when I press "Page Down" over the vertical bar it is not doing that, it goes down but it is not going to look exactly from the beginning to the end of the following page.

my ultimate desire is to create a keyboard shortcut for that

suzanne s. barnhill say:

There is a keyboard shortcut that can be made to do that. Historically, Ctrl+PageDown and Ctrl+PageUp were assigned to the NextPage and PrevPage commands. When the browse arrows and "browse object" were introduced, the shortcuts were reassigned to BrowseNext and BrowsePrev. When you launch Word, the "browse object" is Browse by Page, so you get the expected behavior, but if you perform a search, then the browse object becomes Find/GoTo, and you can't use the browse arrows or the shortcuts to go up or down a page unless you reset the browse object to Browse by Page.

Unfortunately, Word 2013 dispensed with the Select Browse Object dialog, so it is not possible to change the browse object through the UI. However, through the Customize Keyboard dialog, you can unassign the BrowseNext and BrowsePrev commands and restore NextPage and PrevPage.

Even these will perhaps not do what you want, as they do take you to the top of the next/previous page. If what you want is to scroll by exactly one page, then I'm not sure there is a command that does that.
