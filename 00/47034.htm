Can pausing a video playback on Windows Media Player for a longer - Microsoft Community

xubu say:

Can pausing a video playback on Windows Media Player for a longer period of time cause damage to the hard disc drive?


Recently, as I was watching a video that was stored on my hard disc drive using Windows Media Player, I became very tired. Instead of stopping Windows Media Player, I pressed pause by mistake and fell asleep for around an hour.  Generally, does leaving the video playback on pause for a longer period of time cause damage to the hard disc drive?  Is there possibly a default protection mode on Windows 7 that keeps the hard disc from being damaged in case the playback is paused for a longer period of time?

Thank you,

 

Xubu

ralf_g say:

Recently, as I was watching a video that was stored on my hard disc drive using Windows Media Player, I became very tired. Instead of stopping Windows Media Player, I pressed pause by mistake and fell asleep for around an hour.  Generally, does leaving the video playback on pause for a longer period of time cause damage to the hard disc drive?  Is there possibly a default protection mode on Windows 7 that keeps the hard disc from being damaged in case the playback is paused for a longer period of time?

Thank you,

 

Xubu


The hard disc doesn't work the way you imagine it does. It isn't like a paused VHS tape subject to wear on one spot from the constantly spinning heads rubbing against the tape. Tapes are sequential access devices, harddrives are random access. Video data is read from the drive and then loaded into memory in large chunks as needed (dependant on how much RAM is available). That takes but a moment each time, with the harddrive moving on to other tasks as the computer requires it. The part of the video you are watching at any given moment is already loaded in memory, with the harddrive freed up for doing other tasks until it has to go back and read the next segment of the video file(s). Segments can be many minutes long.

davem121 say:

These files are Windows System Files and are always present, however you must have changed settings in Windows Explorer to be seeing these files . . .

Desktop.ini contains information on icon location, along with other settings on the way your desktop is arranged

Every folder in Windows which contains images had a Thumbs.db file which contains information on the thumbnail cache for that folder

Open Windows Explorer, on the View Ribbon at the right hand side click Options.

In the resulting dialog click on the view tab and check the box marked 'Hide Protected Operating System Files' 

Click Apply and OK

Then you will not be able to see these files . . .
