jjme say:

 

WM5 didn’t properly handle international phone numbers stored in these formats.  We attempted to fix this oversight in WM6 but it looks like we have a bug for French numbers which we’ll address in a future release.  For now, the below registry work-around should solve the issue.  

 

1-  Get an application to edit the registry on your phone. Usually HTC phones are shipped with an application called “RegistryEdit”.

 

2 - Add “33,00,0” to the following regkey :

[HKEY_LOCAL_MACHINE\Security\Phone] “DialingPrefixData” = ”33,00,0”

 

Note: This workaround requires you to have security privileges to edit the secure part of the registry. Some phones won’t let you modify this reg key.
