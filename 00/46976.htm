considerate_guy say:

There isn't a built-in highlighting feature, but you could do this by taking a snapshot, then editing that snapshot with the highlights that you want.  If you stick that edited snapshot into your movie, the action will pause with your highlighted image, then the action will continue.

Here's a sample video

that shows how it might look.

To do this in Windows Live Movie Maker, you would set the playback marker at the point where you want to highlight the action.  Use the Video Tools | Edit tab and make a Split (click the button or just press 'M' on the keyboard). Now go back to the Home tab and click on the Snapshot button.  Keep track of where you save the image on your computer.  A copy of the picture will be placed at the end of your movie, so drag it back to the point where you made the split.

At this point, if you preview the movie, it will play, then pause, then continue.  But you want to add some highlights to this image.  So save your project, and find the image on the computer.  Open it up in Paint or another image editor.  You can add a spot shadow (highlight the area and darken everything around it), or draw arrows, or circles as needed.  I kind of overdid it in my sample video (did all three), but you get the idea.

Now when you reopen your project, it should pull the latest version of the image from the hard disk and you should see the highlights when it pauses.

greg carmack - mvp 2010-2020 say:
Hi Jari. I'm Greg, a volunteer installation specialist and 8 year Windows MVP, here to help you.

You can install EasyBCD to Windows on the 32gb drive, add the cloned install on F to the Boot Menu. Save changes. http://neosmart.net/Download/Register/1
http://neosmart.net/wiki/display/EBCD/EasyBCD+B...

Then restart PC to see if you get a Dual Boot menu. If so, install from EasyBCD access Edit Boot Menu, change the defaults to the time you want during the transition phase to get into the other OS (suggest 10 seconds), set the clone as Default. Save.

I would consider moving the Paging file to the SSD and leave the boot files there.

When ready you can use Easy to hide the Boot Menu so you don't see it at all and just use the SSD for boot and paging speed.

If you correctly cloned to F it will appear as C when booted. If not I'd do it over using free Macrium saving a backup image to your data partition and apply it to deleted F. Macrium will give you the tools to delete and partition during imaging from a Tools link beneath the partition. http://www.howtogeek.com/howto/7363/macrium-ref...

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.

If it wasn't clear, the above approach is to preserve the speed of the SSD for booting and paging file which are what it will benefit the most if the OS itself is not there.
