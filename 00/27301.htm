You can't "rollback". You will have to uninstall and install to get back to the point you want.

The Office 2016 Home and Student license was improved to allow it to only be installed on one computer at a time. You have to decide if you want it on the new or old computer.

If you do want it on the new computer try doing this:

  1. Uninstall Office 2019 from the new computer
  2. still on the new computer, log in to the MyAccount site https://account.microsoft.com/
  3. Find the part of the site that lists installed activations
  4. Find the 2016 installation
  5. There should be a deactivate link beside the 2016 computer name
  6. Install Office 2016 on the new computer

You can leave 2016 installed on the old computer. It will be in "limited function mode". It will be able to read and print existing documents. It cannot save new files or changes made to existing files.

Or if 2019 was activated, you could install it on the old computer.

DWORD: LoadBehavior
Value: 3 = add-in is enabled, 2 = add-in is not enabled

You can use the following steps to check the registry:

3. In Registry Editor, locate the following subkey:
HKEY_CURRENT_USER\Software\Microsoft\Office\Outlook\Addins\UcAddin.LyncAddin.1
4. In the details pane, double-click LoadBehavior. If the value in the Value data box is not 3, change it to 3 and then click OK.
5. Exit Registry Editor.
6. Start Outlook.
