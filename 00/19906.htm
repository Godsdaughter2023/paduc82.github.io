Typically Malwarebytes has been known to be more aggressive with detection and removal of Potentially Unwanted Software (PUPs), so that's why they're often recommended, occasionally even by me, when such items are involved.

Windows Defender can often detect these as well, but Microsoft's detection criteria are different and for the less malicious variants require manually changing a registry setting (really intended only for Enterprise business use) and thus it's less "friendly" in it's handling of PUPs for the consumer.

Does Defender scan automatically, if so, how frequently?
greginmich say:

Does Defender scan automatically, if so, how frequently?

In versions of Windows 10 prior to Version 1607, Automatic Maintenance runs a Defender Quick Scan on a daily schedule, although the scan will only run consistently on PCs that are kept powered up and allowed sufficient idle for the scan to complete, since the Automatic Maintenance scans are limited to idle time and are canceled (not paused) by any user activity.

In Version 1607, the Automatic Maintenance scans don't seem to run consistently even when the above conditions are met, unless you change the user account for the Windows Defender Scheduled Scan task (which is the task run by Automatic Maintenance) to the local administrator.

There are several viable options for dealing with this issue:

1. Just ignore it and wait for a fix ( since there’s very little added risk involved in depending solely on Defender’s real-time protection).

2. Run occasional Quick Scans manually.

3. Schedule your own daily Quick Scan with the Create Basic Task Wizard under the (default) local administrator account by following the steps in this thread (when you get to step 7, don’t change the user account – just leave it at its default (local administrator):

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-do-i-set-windows-defender-to-a-custom-auto/ec19aab4-10ce-4f41-9990-343056fd104f?page=1  

4. Change the user account for the Windows Defender Scheduled Scan task to the local administrator – exactly as described here:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/windows-defender-daily-quick-scan-windows-10-os/af9b6bf8-7880-48c1-afde-4043534b55d3

If your PC isn’t password protected, then first change the account to the local administrator, and then check the Do not store password. The task will only have access to local resources option.

Mike

 

Len() is not limited to 255 characters UNLESS you are using an agregate query. This is how Access works.

 

To use a function as a wrapper you have to pass the primary key as well as the field like this example:

Public Function GetLen(lngPK As Long, strMemo As String) As Long
    GetLen = Len(strMemo)
   
End Function


Here is the query:

SELECT GetLen([MyPK],[MyMemo]) AS LenOfMemeo, Table1.FirstName
FROM Table1
GROUP BY GetLen([MyPK],[MyMemo]), Table1.FirstName;
