How to Find SSID (Network Name) - Microsoft Community

xb1r athena say:

How to Find SSID (Network Name)

Login Information

  • Gateway - 192.168.0.1
  • Username - admin
  • Password - (blank)

How To Find SSID

  1. Open a web browser.
  2. In the address bar type in the default gateway (http://192.168.0.1) or click the link.
  3. When prompted type in the user name “admin” and leave the password field blank.
  4. With the “Home” tab at the top selected, click on the “Wireless” from the menu on the left.
  5. The SSID will be listed at the top of this page in the box labeled SSID.

andre da costa say:

Can I do a clean install using the Free upgrade?

No, it will require that you are running a previous qualifying version and start the upgrade from within the qualifying version. You can initiate a clean install after completing the Upgrade.

Please be aware that you cannot use the free upgrade offer to perform a clean install on first attempt. You must first upgrade from the qualifying version of Windows 7 or Windows 8/8.1 (whether you do it through Windows Update or using the .ISO file). Ensure the upgrade is completed successfully and then ensure that it is activated. You can then proceed to do a clean install by using recovery media or using the Reset function in Start > Settings > Update & Security > Recovery > Reset this PC (Get Started).

aerthand say:

No, you cannot do a "clean" install of Windows 10 using the free upgrade.  Here is how it the upgrade really works:

1.) You have an existing install of Windows 7/8/8.1

2.) You "upgrade" to Windows 10

3.) You use the "Reset my PC" feature to then re-install Windows 10 which they say is a "clean" install, but really isn't since it's using pre-staged files on the same partition without doing a zero-fill.

If you want to do a full genuine clean install where you re-partition your hard disk, reformat it, and then drop a new installation of Windows 10, you need to buy the full version.  $200 for Pro.
