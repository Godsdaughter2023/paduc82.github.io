2. After your PC restarts to the Choose an option screen, select Troubleshoot > Advanced options > Startup Settings > Restart.
3. After your PC restarts, you'll see a list of options. Select 4 or F4 to start your PC in Safe Mode. Or if you'll need to use the Internet, select 5 or F5 for Safe Mode with Networking.

Method 2: Perform Clean Boot. If the issue does not occur in safe mode, it could be some application or programs, which runs on the computer at the background, leading to this issue. Please follow steps on this link to perform a clean boot in Windows 10.

Note: Please refer to the section: How to reset the computer to start as usual after troubleshooting with clean boot of the Kb article to boot the computer in normal startup.

Method 3: Run a full computer scan with the Microsoft Safety Scanner to make sure that the computer is free from viruses.

Disclaimer: Any data files that are infected may only be cleaned by deleting the file entirely, which means there is a potential for data loss.
