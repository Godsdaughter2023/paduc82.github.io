As Hadyn1 pointed out a reboot is a good idea, but in Windows 10 not all reboots are really a reboot. Microsoft has enabled a feature called Fast Startup, which behaves like a hibernation, and does not really restart the PC but puts it to sleep so it can start up faster the next time you turn it on.

You can overcome this by opening Powershell, and entering the following command, and the PC will do a reboot:

restart-computer -force

you can also run a Powershell command to see when the PC last booted with the following command:

get-wmiobject win32_operatingsystem | select lastbootuptime

the result will look something like this:

LastBootUpTime           
--------------           
20180108064831.492409+630

This is the date and time in reverse order, i.e. yyyymmddhhmmss etc.
