This works great as long as the document in question doesn't have formatted page numbering (i.e. more than one section that restarts numbering at some point in the document).

Next I tried:

{If {Page } = {SectionPages} {DOCPROPERTY DocID}}

Which worked great as long as the document either had only one section or multiple sections and they DID restart numbering at some point.

In other words, neither works in all situations.

Is there any way to reference the REAL page number instead of the formatted page number in the IF field?

{If {RealPage } = {NumPages} {DOCPROPERTY DocID}}

 

Thanks in advance,

Jeri

jay freedman say:
Unfortunately, there is no such field.

The closest workaround is to insert a bookmark (named something like LastPage) on the last page of the last section, unlink the footer of the last section from the footer of the section before it (if there is one), and insert a field in that footer with a field code like

{ If {Page } = {PageRef "LastPage"} {DOCPROPERTY DocID }}

This workaround has several drawbacks. Bookmarks are too easily overwritten or deleted, especially if the option to display them is turned off. More text could be added to the end of the document, so the page containing the bookmark would no longer be the last page. If the Different First Page option is turned on for the last section, but the macro inserts the field only in the primary footer, the field won't appear when the section has only one page. There may be other odd situations that behave badly.

pacificparrot say:

I had the same problem when manually entering or copy/pasting all the values but if I put the list in an Excel worksheet, I was able to import the entire list by selecting that cell range using the Excel Options in the Microsoft Office Button.  Here is a link (scroll down to "Create a Custom List from a Cell Range" under the "Create a Custom List" heading. 

http://office.microsoft.com/en-us/excel-help/create-or-delete-a-custom-list-for-sorting-and-filling-data-HA010222142.aspx

 

EXCERPT from that webpage:

 

Create a custom list from a cell range    

  1. In a range of cells, enter the values that you want to sort or fill by, in the order that you want them, from top to bottom. For example:
A
1    High
2    Medium
3    Low
  1. Select the range that you just typed. In the preceding example , you would select cells A1:A3.
  2. Click the Microsoft Office Button , click Excel Options, click the Popular category, and then under Top options for working with Excel, click Edit Custom Lists.
  3. In the Custom Lists dialog box, verify that the cell reference of the list of items that you selected is displayed in the Import list from cells box, and then click Import.

The items in the list that you selected are added to the Custom Lists box.

  1. Click OK twice.
