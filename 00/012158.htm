jeeped say:

This is possible, but impractical unless you are talking about a large worksheet(s) with numerous formulas and/or values sporadically placed around the worksheet's extents. For my example here, I'll assume that you have Sheet1 with numerous formulas and/or values populating A1:J1000.

First, define two UDF's (User Defined Functions) to assist you.

Press ALT+F11 to open the VBE, then Insert, Module. Paste these following two UDFs in:

Function fcnIsFormula(Check_Cell As Range)
    fcnIsFormula = Check_Cell.HasFormula
End Function

Function fcnGetFormula(Cell As Range) As String
   fcnGetFormula = Cell.Formula
End Function

Now press ALT+Q (e.g. close the VBE and return to Excel).

Start a new worksheet (e.g. Sheet2). This is to be used as an intermediary worksheet and could be eliminated through CTRL+A, Copy, Paste Special, Values on the same sheet rather than a new one but the purpose of this exercise, I'll leave the extra sheet in.

In A1 (on Sheet2) input the following formula:

=IF(fcnIsFormula(Sheet1!A1),fcnGetFormula(Sheet1!A1),"")

... and Fill Right to column J, then Fill Down A:J to row 1000. While A1:J1000 is highlighted, Copy it.

Go directly to A1 on Sheet3 and Paste Special, Values, Skip Blanks. (YES! That is Values, NOT Formulas!)

Now at this point, Sheet3 has confused Excel a bit and it doesn't know whether to process formulas or display textual representations of formulas. To resolve this, perform a search and replace on Sheet3's A1:J1000 range (or the whole worksheet, it matters not).

Press CTRL+H, and search for (, replace with (, Replace All. By searching for a left bracket (which every formula contains) and replacing it with the same left bracket, you have asked Excel to re-evaluate the cell contents.

At this point, all formulas should be formulas, and all direct values from Sheet1 should be blank. As mentioned, this would appear to be a lot of work if you didn't have a large amount of formulas to segregate and copy.
