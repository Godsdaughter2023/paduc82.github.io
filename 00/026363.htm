john korchok say:

Most likely the sound files are linked rather than embedded. Probably, they are in a different folder than the presentation. Linked files include the path to the sound files. When you move the files to a new computer, the path is different and the sound files can't be found.

The best practice is to keep the sound files in the same folder and the presentation., then when you link them, the path is not included and the folder containing the deck and sounds can be copied to another machine.

With modern version of PowerPoint, you can also embed the sound files in the presentation. This makes the file larger, but the sounds are always available.
