seeker55 say:

syntax error (missing operator)

the following query

INSERT INTO member_personal_goal (member_id,Goal,GoalStartDate,ServiceID) VALUES (2211,'"socialize more - don't isolate in my apartment"',#8/20/2010#,'PR-AG')

john w. vinson say:

the following query

INSERT INTO member_personal_goal (member_id,Goal,GoalStartDate,ServiceID) VALUES (2211,'"socialize more - don't isolate in my apartment"',#8/20/2010#,'PR-AG')
gives me a syntax error (missing operator) -- I need another set of eyes.


You've got a ' (in the word don't) inside a string delimited by ' marks. Your string is being interpreted as

"socialize more - don

and Access is unable to parse the remaining

t isolate in my apartment"

Either double up the ' or delimit the whole string with doublequotes: try

strQry = strQry & "VALUES (" & Forms!f_diag_main!member_nbr & ",""" & txtGoal.Value & """,#" & Nz(txtStartDate.Value, #12/30/1899#) & "#,""" & NewService & """)"

using "" within the doublequote delimited string to encode a single doublequote (how's that for doubletalk!)

You may want to leave a null date as NULL:

& IIF(IsNull(txtStartDate, Null, CDate(txtStartDate)))

 

John W. Vinson/MVP
