Sync Center is used to manage your Offline Files from your network. The ability to use Offline Files is enabled by default in Windows 10, but can be disabled.
 
You may follow the steps below to disable the use of offline files.
 
a.     Open the Control Panel (All Items view), and click on the Sync Center icon.
b.     In the left pane, click on the Manage offline files link.
c.     Click on the Disable offline files button.
d.     If prompted by UAC (User Account Control), then click on Yes.
e.     Click on OK.
f.      Click on Yes to restart the computer.
NOTE: This will restart the computer immediately. Be sure to save anything that you have open first.
 
g.     If you enabled Offline Files, then all you need to do now is to make the network files or network folders you want always available offline.

nathant_06 say:
Hi ArvidFenske,

You can add permissions by doing the following:
- Right-click the hard drive in File Explorer and select Properties
- Select the Security tab
- Select Edit
- Select Add
- Enter the name of the user account you want to give permissions to
- Select the appropriate level of permission after selecting the username entered above (Full access or modify will both have read/write permissions)
- Select OK, and then OK again

Hope this helps,
