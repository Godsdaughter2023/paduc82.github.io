gord dibben say:

Pop-up messages are usually the result of VBA event code.

 

You can have a message box pop-up generated by a calculated value.

 

Right-click on your worksheet tab and select "View Code".

 

Copy/paste the code below into the module that opens.

 

Private Sub Worksheet_Calculate()
    On Error GoTo stoppit
    Application.EnableEvents = False
    With Me.Range("D8")
        If .Value >= 9798.79 Then
            MsgBox "Value in D8 is too high. Please rectify"
        End If
    End With
stoppit:
    Application.EnableEvents = True
End Sub

 

Alt + q to go back to Excel.

 

When the calculated value in D8 reaches or exceeds the parameter set you will get a message box that user has to deal with.

 

As the code is written user can close the message box and do nothing.

 

If you need something more forceful I suggest some none-too-subtle behaviour modification.

System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files.

To run SFC scan, follow the steps:

Open an elevated command prompt - Right click on the Start Button and select Command Prompt admin. If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

Enter below formula in cell H2 and copy down:

=DATEDIF(A2,B2,"M")*C2+DATEDIF(A2,B2,"MD")/(EDATE(A2,DATEDIF(A2,B2,"M")+1)-EDATE(A2,DATEDIF(A2,B2,"M")))*C2
