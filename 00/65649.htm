jetta48 say:
Whats the best way to wipe out an old xp computer before disposing of it so there's no personal stuff left or recoverable?


There are 2 main ways of making your data unrecoverable in your scenario (assuming standard HDD):

1. Phyisical - angle grinders, plasma torches, drills, big hammers (there are industrial scale, dedicated HDD shredders that munch hundreds at a time)
2. Software - deletion/overwriting of HDD.

If done by itself, Option 1 is fairly obvious - just be thorough. HDDs are surprisingly tough.

Option 2 cannot be achieved simply by deleting files & emptying the recycle bin alone. This leaves your data potentially recoverable by readily available legal programs. You need to use a utility that overwrites your HDD with random data multiple times. The more times the better - the Gutmann standard (x35) is usually considered good. One such utility is Darik's Boot and Nuke.

The last time I wiped a HDD, I used the following proagmatic compromise method:
1. Backed-up data
2. Removed old HDD from old PC and temporarily connected to new PC.
3. Booted new PC quick formatted old HDD
4. Used CCleaner (set to overwriting x35) to wipe free space on old HDD.
5. Removed old HDD & bashed up the PCB & connector sockets - just for good measure.

There are other combinations of the above, but I'm sure you get the gist of it.

If you have a Solid State Drive (SSD) the software method will not work. Physical annihilation is your best bet.

sharief k say:

Hi Reynolds,

 

1.      Do you get any error message?

2.      If yes, what is the exact message?

3.      What all diagnostics have you done so far?

 

This issue could be due to Windows Update database is corrupted or Windows Update service stopped during the installation.

 

Here are few troubleshooting methods you may try in order to fix the issue.

 

Method 1: Reset Windows update components from the following link and check.

http://support.microsoft.com/kb/971058

 

Important: Back up the registry

 

Method 2:

 

a. Click in Start and in Start search box type CMD.

b. In the list Right click on CMD and click Run as Administrator.

Type the following commands, and press Enter after each command:

 

o   net stop cryptsvc

o   md %systemroot%\system32\catroot2.old

o   xcopy %systemroot%\system32\catroot2 %systemroot%\system32\catroot2.old /s

o   Net stop wuauserv

o   cd %systemroot%

o   Ren SoftwareDistribution SoftwareDistribution.old

o   Net start wuauserv

 

Try to install the updates and check.

 

Method 3: Unable to install updates in Windows Vista, Windows 7, Windows Server 2008 and Windows Server 2008 R2

 

http://support.microsoft.com/kb/2509997 

 

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.

For more information, refer to the article.

 

Troubleshoot problems with installing updates

 

Do let us know about the results. We will be happy to assist you further.
