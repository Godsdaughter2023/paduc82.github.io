VPN screen resolution - Microsoft Community

rksn say:

VPN screen resolution

I got a Pro 3. Everything is good till I connect to VPN. Every font reduces to a ridiculous small size. Need a magnifying glass to read. How to adjust the screen size.
cmdrkeene say:

The Surface Pro 3 has a high-definition screen, like the "Apple Retina Display".  Since it has so many more pixels but closer together, a 100x100 pixel square is much smaller than on an older monitor with less pixels that are further spaced out.

Normally, Windows uses scaling to adjust this. For example, the Surface Pro 3 multiples the size of everything by 150% in the DPI Scaling (Dots Per Inch) so it appears to be larger.

Your VPN is doing a remote-desktop view so it isn't aware of the scaling.  You can try adjusting some scaling options under "right-click desktop > screen resolution > make text and other items larger or smaller", but that is not likely to make much difference because the change needs to be made on the remote VPN system.

In that case, you would likely need to contact your VPN administrator for an updated VPN program, or possibly use another VPN vendor entirely (one that supports HD displays).

Windows 7 - upgrade vs. full version - Microsoft Community

slickuser say:

Windows 7 - upgrade vs. full version

I want to buy Windows 7, what is the difference betwen upgrade and full?

Is the "upgrade" mean that I must have some other Windows install already?

Example:
Windows 7 Home Premium Upgrade
Windows 7 Home Premium

Currently, I have Linux installed on my machine, can I buy Windows 7 Home Premium Upgrade and installed and use it?
andre da costa say:
In order to install the upgrade version of Windows 7, you must have a qualifying Windows operating system installed and activated.  You cannot install an upgrade version of Windows 7 on a blank hard drive.  The installation procedure does not ask you to insert a Windows disc in the drive for verification, the actual qualifying operating system must be installed.

    If you do not have a qualifying Windows operating system installed with a genuine license activated, then you cannot use the upgrade version of Windows 7 - you would need a "full version" Windows 7 license.

    In summary:

       1. A qualifying Windows operating system must be installed.
       2. The qualifying Windows operating system must have a genuine license (product key) and it must be activated.
       3. To upgrade, boot to the qualifying Windows desktop, insert the Windows 7 Upgrade disc in the DVD drive.
       4. When the setup menu appears, select Custom (advanced) to initiate a clean install procedure.

Windows 7 Home Premium full version, does not require a previous qualifying license, you can use it do a upgrade or clean install (which you can also do with the upgrade version, just that the upgrade version requires a qualifying operating system).

Another thing to note about the Upgrade version, due to the fact the original full version software and the upgrade version software together constitute a single software program, you will need to retain your copy of the prior version of the software program (including all media, manuals and the Certificate of Authenticity) in order to establish your eligibility for the upgrade license.

No, you cannot use Linux as a qualifying operating system, you need Windows XP or Vista to use the Windows 7 Home Premium upgrade (activated and installed).

Did you already try the Safe Mode options? F8 during boot-up then either start in Safe Mode and run System Restore or run System Restore without starting safe mode. From the same menu there should also be  an option to start with the "Last Known Good" configuration. If the last option gets Windows running you can do a system restore to the most recent point when Windows booted normally (or before you made the change that corrupted it).

You need to have created a system image of your installed Windows beforehand in order for the System Image recovery option to work. System startup repair may need to be run multiple times because it doesn't fix everything it finds at once.

Your best bet to clean up the mess if nothing else works is to use the Gateway system recovery function to put the computer back as it was when you first got it. You can either run it from the harddrive boot options (not sure what key Gateway uses to access it) or from the recovery discs you no doubt dutifully made.  There are often two recovery modes so pay close attention to the options you get. One mode will wipe out everything on the harddrive and reinstall the original OS and software in its place, the other mode will reinstall the OS and original software but without wiping your existing files. That of course assumes that the data on your harddrive hasn't already been corrupted.
