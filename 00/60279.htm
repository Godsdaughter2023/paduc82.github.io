    Technically, when you create an account there is no longer an option to use a preferred Skype Name. If you have created your account using an email address or phone number, Skype will generate a Skype Name based on those information.

    • If account is created using email address, your Skype name will be live:(youremail) For example: Email is julie123@hotmail.com Skype Name is live:julie123
    • If account is created using phone number, your Skype name will be live:(alphanumeric) For example: live:123ghj45klz
