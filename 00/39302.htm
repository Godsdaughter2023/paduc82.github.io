System restore points may be created on the drives having the operating system. A restore point is a representation of a stored state of your computer's system files. You can use a restore point to restore your computer's system files to an earlier point in time. Restore points are automatically created by System Restore weekly and when System Restore detects the beginning of a change to your computer, such as when you install a program or a driver.

System image backups stored on hard disks can also be used for System Restore, just like the restore points created by system protection. Even though system image backups contain your system files and personal data, your data files won't be affected by System Restore.
