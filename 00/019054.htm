To your original quesiton, Graham has problably already shown you that it is far better to write macros than it is to attempt to record them.

 

It seems that Graham's code is working fine.  One suggestion if you will do this sort of thing regularily and if the manipulation of the two tables should get complex.

Create a building block from the first table format.  Call it TableA.  Create a building block of the second table format.  Call it TableB.  Put a bookmark in the template where the table should go.  Call it TargetBM:

Insert TableA as the bookmark using:


Dim oRng As Word.Range
Set oRng = ActiveDocument.Bookmarks("TargetBM").Range
On Error Resume Next
oRng.Tables(1).Delete
On Error GoTo 0
Set oRng = ActiveDocument.AttachedTemplate.BuildingBlockEntries("TableA").Insert(Where:=oRng, RichText:=True)
ActiveDocument.Bookmarks.Add "TargetBM", oRng

 

Then use:


Set oRng = ActiveDocument.Bookmarks("TargetBM").Range
On Error Resume Next
oRng.Tables(1).Delete
On Error GoTo 0
Set oRng = ActiveDocument.AttachedTemplate.BuildingBlockEntries("TableA").Insert(Where:=oRng, RichText:=True)
ActiveDocument.Bookmarks.Add "TargetBM", oRng

at each qualified date condition changing "TableA" to "TableB" as required.

This simply swaps TableA with TableB or TableB with TableA at the bookmark and preserves the bookmark.

 

You would of course distribute the template to other users to create the documents with.

macropod say:

Word doesn't provide a built-in facility for excluding citations. However, the following macro will report the word counts for citations & bibliography, so you can subtract them from the word counts.

Sub CountReferenceWords()
Dim FtNt As Footnote, EndNt As Endnote
Dim Fld As Field, i As Long, j As Long, k As Long
With ActiveDocument
  For Each Fld In .Fields
    With Fld
      If .Type = wdFieldCitation Then
        i = i + .Result.ComputeStatistics(wdStatisticWords)
      ElseIf .Type = wdFieldBibliography Then
        j = j + .Result.ComputeStatistics(wdStatisticWords)
      End If
    End With
  Next
  For Each FtNt In .Footnotes
    For Each Fld In FtNt.Range.Fields
      With Fld
        If .Type = wdFieldCitation Then
          k = k + .Result.ComputeStatistics(wdStatisticWords)
        End If
      End With
    Next
  Next
  For Each EndNt In .Endnotes
    For Each Fld In EndNt.Range.Fields
      With Fld
        If .Type = wdFieldCitation Then
          k = k + .Result.ComputeStatistics(wdStatisticWords)
        End If
      End With
    Next
  Next
End With
MsgBox "Citation words in the body: " & i & vbCr & _
  "Citation words in footnotes & endnotes: " & k _
  & vbCr & "Bibliography words : " & j
End Sub

zorvek say:

To reference the same cells across multiple contiguous sheets in a cell formula use the syntax below.

   Sheet1:Sheet2!A1
   Sheet1:Sheet10!A1:B2

The syntax references the first and last sheets in the set of worksheets to be aggregated. Any number of contiguous worksheets can be referenced but only the first and last worksheets are specifically referenced. This format can be used in most functions such as SUM, MIN, and MAX. For example, the formula

   =SUM(Sheet1:Sheet2!A1:A100)

sums all values in cells A1:A100 on both sheets Sheet1 and Sheet2.

Note that this technique cannot be used in VBA as the VBA range object cannot handle a multiple sheet reference.

Kevin

amit tandon say:

Try below formula:

If the multiple sheets are not congruous (ie. not consecutive), enter the sheet names in cells A2, B2, C2 & D2, below formula will return the sum of F5 from each of these sheets (ensure sheets names on each of the mentioned cells).

=SUMPRODUCT(SUMIF(INDIRECT(A2:D2&"!F5"),"<>0"))

To SUM multiple cells (say F5:F7) from each of these sheets, use below:

=SUMPRODUCT(SUMIF(INDIRECT(A2:D2&"!F5:F7"),"<>0"))

If the multiple sheets are congruous (ie. consecutive) - specify the first (say Sheet1) & last sheet (say, Sheet6) names in the formula. This will sum all sheets starting from Sheet1 to Sheet6.

=SUM(Sheet1:Sheet6!F5)

or for multiple cells:

=SUM(Sheet1:Sheet6!F5:F7)

zorvek say:

"I am expecting to see added together the TOTAL of a specific cell from multiple worksheets."

If the sheets are contiguous (in sequence with no sheets in between that you do not want in the total) then this formula will work:

   =SUM('First Sheet:Last Sheet'!A1)

The formula above sums (totals) all values in cell A1 on sheets "First Sheet" to "Last Sheet". Note the use of single quotes: these are required if the sheet names contain blanks. Also note that for the formula to work, you have to enter it by pressing CTRL+SHIFT+ENTER.
