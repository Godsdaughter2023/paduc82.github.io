efmorris say:

Two people entering information at the same time

Is it possible for two people to enter information in ACCESS at the same time without it crashing? If so, how?

hansv mvp say:

Access is by default multi-user. You don't have to do anything special to let several users enter and edit data simultaneously.

You can specify what happens when a user starts to edit a record in the Record Locks property of a form:

All Records: other users will not be able to edit any record in the record source of the form.

Current Record: other users can edit other records, but not the same record.

No Locks: other users can edit the same record. If you start to save a record that has been saved by another user since you started editing, Access will display a dialog with several choices: keep your own changes (and overwrite those from the other user), discard your own changes, or copy the record to the clipboard so that you can paste the data into another record (or elsewhere).

bob jones aka: cybertaz say:
Microsoft discontinued bar codes beginning with Word 2007 (PC) because the USPS switched specs faster than MS could keep up. What was offered in recent earlier versions were typically invalid as a result. From Word 2007 Help:

Why can't I include a POSTNET bar code or FIM-A code on envelopes or labels?

Symptoms
In an earlier version of Microsoft Office Word, I was able to get a bulk mailing discount on my mailings by inserting a POSTNET bar code, as well as a FIM-A code on envelopes and labels. In Microsoft Office Word 2007, I cannot find the options for adding these codes to my envelopes and labels.

Cause
Periodically, the United States Postal Service (USPS) changes the way it calculates postal bar codes. When this happens, bulk mail that uses the bar codes generated by Word may not be accepted by the USPS. To avoid providing bar codes that may become obsolete, Microsoft has removed this feature from Office Word 2007.

Resolution
Your bulk mailing can still qualify for a discount, but you must work with a third-party provider and the USPS.

hansv mvp say:

AVERAGEIF will ignore blank and text values, but it will include 0 values in its calculation. If you want to ignore 0 values:

=AVERAGEIFS(G$7G$37,$B$7$B$37,A48,G$7G$37,">0")

Note that this formula uses AVERAGEIFS instead of AVERAGEIF.
