Hi, 

 

The PAGE FAULT IN NONPAGED AREA error happens when Windows can't find a system file within the memory that it needs to be able to run properly. This may happen due to failed Windows updates, driver conflicts, or even hard drive errors. We suggest that you follow the steps below to fix it: 

  1. First, press and hold the power button until the computer restarts. Do this three times in a row and you should be taken to the Recovery screen.  

  2. Click See advanced repair options.  

  3. Click Troubleshoot.   

  4. Click Advanced options.  

  5. Click Command Prompt. 

  6. Type chkdsk /f /r and press Enter. Wait for it to finish. This will check your hard drive for errors. 

  7. Type sfc /scannow and press Enter. Wait for it to finish. This will find and fix corrupted or missing system files. 

  8. Type shutdown /r and press Enter. 

 

Let us know what happens after following these steps.

    Solution: Windows 10 internet not working (in Opera , Firefox or Chrome)

    1. Run command prompt as an administrator.

    2. Type the following command: 

    netsh winsock reset

    and press Enter.

    4. You will see success message.

    5. Restart PC and everything should be working fine.
