If you want Photo Gallery to be as it was the first time you
launched it...you have to rename the proprietary data-base
Pictures.pd6.

This file is not affected by uninstalling / reinstalling Photo
Gallery....it must be renamed manually...then Photo Gallery
will create a new file.

If you rename the file...it will force Photo Gallery to create a
new database file the next time it's launched and the gallery
will be as it was the first time it was launched.

Just rename it to OLDPictures.pd6. This may cause you to
lose some tags and metadata that were not written back to
the photo files.

The path to the file should be something like:

Windows\Users\%Username%\AppData\Local\Microsoft\
Windows Live Photo Gallery\Pictures.pd6

Pictures.pd6 is a hidden file...the following article explains
how to show hidden files:

Windows 7 - Show hidden files
http://windows.microsoft.com/en-US/windows7/Show-hidden-files

If you still cannot find the file...try doing a system search
for Pictures.pd6.

If this does not improve the problem...you can delete the
new Pictures.pd6 file and rename OLDPictures.p6 back to
Pictures.pd6 and the gallery should be as it was before.

    If you have been running a copy of Windows 7 that isn't activated, it's no surprise that you think it's "crappy."  When Microsoft detects that a copy of Windows 7 isn't properly licensed (e.g., it hasn't been activated), you will suffer the following consequences:

    • Your desktop background goes black. You can change it, but every 60 minutes, it will return to black until you activate Windows.
    • You will have a "This copy of windows is not genuine" watermark on the bottom right corner of your desktop.
    • You will get a notification balloon stating that you may be a victim of software piracy.
    • You will be prompted to activate Windows every time you log in.
    • You will receive periodic reminders to activate Windows.
    • Windows Updates will be limited, but you will still be able to get critical security updates. However, access to optional updates or benefits available exclusively to genuine Windows customers, such as Microsoft Security Essentials, might be restricted.
