trevor rabey say:

That's what resource leveling is for.

Make the tasks, estimate the durations, link up the predecessors and successors, assign the resources to the tasks.

There will inevitably be resource over-allocation (resource is double or triple booked).

Go to resource, leveling options. Accept the defaults, except uncheck all of the options boxes, level all.

Job done. Tasks have been delayed until the over-allocation goes away.

If you don't like the answer, adjust the task priorities, then level again but select "priority, standard" in leveling order.

My sound is not muted on the iPad. I can play the videos with sound in both the Files App and the Dropbox app.

john korchok say:

Video formats have 2 parts. The container is MP4 in your case. The codec is the software format in which the audio and video is encoded. PowerPoint for iOS supports a smaller range of containers and codecs than desktop versions. If your MP4s use an audio codec that is not supported by PowerPoint for iOS, you won't hear anything.

You can try converting to other formats using the free Zamzar or CloudConvert services, then see which format works better with PowerPoint for iOS.

Do you have “Fun Tools” add-in? A number of people have reported that removing it overcomes problems with 2013

Do you have Toshiba Book Reader installed on the PC?  It has been reported that uninstalling that brings Office back to working status.

Do you have the Acrobat PDFMaker add-in.  It has been reported that uninstalling that overcomes an issue with a message appearing that states "The document contains macros. Hotmail doesn't support flagging for mobile clients in its current version but supports it for access through  http://mail.live.com or Windows Live Mail PC client.

The Outlook client in Windows Phone 7 supports flagging if the email server is using protocol Exchange ActiveSync 12.0 and higher versions (i.e. Exchange Server 2007 and higher or third-party servers using ActiveSync).
