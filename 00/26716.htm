If you have a table with the other table names, then you can use a VBA procedure to automate running the Append queries.  The code might look something like this untested air code:

Dim db As Database
Dim rs As Recordset
Const cstrAppend As String ="INSERT INTO maintable SELECT * FROM "

   Set db = CurrentDb()
   Set rs = OpenRecordset("tableoftables")

   Do Until rs.EOF
      db.Execute cstrAppend & rs!tablename, dbFailOnError
      rs.MoveNext
   Loop

   rs.Close : Set rs = Nothing
   Set db = Nothing
