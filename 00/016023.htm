suzanne s. barnhill say:

The list of paper sizes you see in Word is provided by your printer driver and reflects (or should reflect) the paper sizes your printer is capable of handling. If the specs for your printer indicate that it should be able to handle A5, then make sure you have the latest driver (which you can download from the manufacturer's website).

Alternatively, you can create the document with the Microsoft XPS Document Writer selected as the active printer. That will allow you to select the A5 paper size. Whether your printer will be able to print it or not is another matter.

doug robbins - mvp office apps & services (word) say:
Create the document as an A4 document, increasing the font size if necessary, and then when printing, select the option to print 2 pages to the sheet.  You will then get an a5 sized document on half of an A4 sheet of paper.
