ruwim.b say:

The new Microsoft Skype for Windows 10 (the 12.X version) is not any longer saving the chat history in the main.db file, but in skype.db database file.

 

https://answers.microsoft.com/en-us/skype/forum/skype_windesk-skype_messms-skype_sms/sqlite-maindb-file/75258f84-4b2d-4284-82b0-958bed9f3b36

 

This means that the tools designed to read main.db file cannot be used to view and export chat history from Skype for Windows 10.

 

Location of the skype.db file is given by this path:

 

%localappdata%\Packages\Microsoft.SkypeApp_kzf8qxf38zg5c\LocalState\<Skype Name>\skype.db

 

skype.db is a standard SQLite database file and can be opened by a suitable SQLite browser.

 

The browser I would recommend, is DB Browser for SQLite:

 

http://sqlitebrowser.org/

 

Using this browser, it is possible to access all the content of the skype.db file and export the chat data to e.g. a text file (.csv file).

You don't need to execute any SQL commands.
