Try this:

- Set up the data validation for the A,B,C,D,E list as normal in column A

Hello Charlie,

Please be informed, when you open a document in Word 2016 that was created in Word 2010 or earlier, you see Compatibility Mode in the title bar of the document window. You can work in Compatibility Mode or you can convert your document to the Word 2016 file format. The Word Convert command clears the compatibility options so that the document layout appears as it would if it had been created in Word 2016. If the file is in .doc format, the Convert command also upgrades the file to the .docx format.

Doing this may change some formatting done in the document file that are not compatible with current version. For more information you may refer to the suggestion mentioned in the following Microsoft article and check if it helps.

https://support.office.com/en-us/article/Use-Word-2016-to-open-documents-created-in-earlier-versions-of-Word-5b38a00a-840b-4719-a8a3-ce155df82554
