I looks like your queries are all action queries.  If so, it's better to use the Execute method to run them:

 

    CurrentDb.Execute "qry_clean_tbl_data", dbFailOnError
    CurrentDb.Execute "qry_create_tbl_supplamental", dbFailOnError
    CurrentDb.Execute "qry_create_tbl_data", dbFailOnError
    CurrentDb.Execute "qry_update_tbl_data", dbFailOnError

 

Note, it would be more efficient to set a database object variable to CurrentDb once and then use that on each Execute line.

 

Because the Execute method generate no messages for a valid query, you probably won't need the SetMessages.

You're very welcome, Jo.  Thank you for your kind comments.

Regarding your question about date/time formats.  When you export the data to Excel it arrives in Excel as a text entry.  In the Excel spreadsheet you can fairly quickly convert the text to a date value and format.

Let's say your first start date is in cell C2 in the export and Cell D2 holds the first finish date.  In cell E2 write the following formula:

=DateValue(c2)

C2 should now contain some number above 41000.  Change the format of that cell to Short Date.  Now fill the formula (and format) to the rest of the cells needed to show the start in just date format.

The fill right (to cell F2) to convert the finish fields.  Then fill down to convert the finish field.

  6. Delete all of the files in this folder.
  7. Restart Microsoft Outlook and you should be able to open your attachment.

Error 45: This hardware device is not connected to the computer (Code 45) will occur if hardware device is not properly connected to the computer. The only solution is reconnecting the device to the respective port.

Check Windows Updates for the latest drivers.

Check in Windows updates if any driver updates are available. If so, install the same and check if it helps.

a) Press “Windows Logo” + “W” keys from the keyboard.

b) Type “Updates” in the search bar and click on the icon “Windows Update” from the search result.

c) On the “Windows Update” window, click on the link: “Check for updates” from the left hand side of the window to receive a fresh list of updates.

d) If the updates are available, then install all the updates listed there.
