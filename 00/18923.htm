roady [mvp] say:

An Exchange account syncs on a push-based method. This means that Outlook doesn't check for new messages itself but will receive a notification of the server whenever a new email arrives and Outlook will then download this message (and any other message that may have arrived) after 30 seconds.

The only way to stop this is by placing Outlook in Offline Mode. This will still allow you to automatically receive IMAP emails every 15 minutes.

Since the template formatâs conditional formatting is set for numbers not dates, it canât be changed easily. If you need the date format, I suggest you create a new one and it is quick and easier.
