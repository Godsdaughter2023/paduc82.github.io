frederik long say:

When I recreate the partition on the destination computer and deploy the Windows Image is it also possible to create the paging file for Windows 10 so this error doesn't happen and having the virtual memory setup at the same time.

Windows will create the paging file automatically at boot time, provided that the specified location exists. You can see it here:

HKLM\SYSTEM\ControlSet001\Control\Session Manager\Memory Management/PagingFiles

If your target machines do not have the partition that you use on the source machine then you must move the paging file on the source machine before running sysprep.
