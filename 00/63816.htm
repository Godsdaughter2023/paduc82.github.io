Hi Raymond,

Hopefully I can be of some help here. Here's a TenForums article link in context to what I will be suggesting, hopefully it paints a vivid picture here: https://www.tenforums.com/tutorials/3162-set-wi...

Open Command Prompt as an administrator (Start > type in cmd then right-click Command Prompt, then run as an administrator).

Paste in the following: netsh wlan show profiles

See the name of the wireless network you're connected to?: netsh wlan show profile name="thename"

...with thename being the name of the profile of course.

Then when you have your sights set on fixing it, here's the command to do so: netsh wlan set profileparameter name="thename" cost=cost

This can go one of two ways in being metered, which I will quote directly from the forum article:

Fixed (metered) = To set this wireless connection is not restricted by usage charges and capacity constraints until you reach a certain data limit.

Variable (metered) = To set this wireless connection is charged on a per byte basis.

So if you, say, want to try Variable, you would paste it in like so: netsh wlan set profileparameter name="thename" cost=Variable

Then hit Enter. It should be saved now. I realize this is a lot to follow, and I wish this process was easier, but this is a method that Windows saves wireless network profiles that I hope would commit the changes.
