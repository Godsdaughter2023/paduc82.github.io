In Excel, select File > Options.

Select Advanced in the navigation pane on the left hand side of the Excel Options dialog.

Scroll down to the section labeled General.

Make sure that the check box labeled 'Ignore other applications that use Dynamic Data Exchange (DDE)' is clear (not ticked).

Click OK.

If that doesn't help:

Right-click the Start button in Windows.

Select 'Programs and Features' from the context menu.

Select your version of Microsoft Office 2016 in the list of programs.

Click Change above the list of programs.

Select the option to repair Microsoft Office.

In case, options provided by HansV MVP do not work -

Please look into C:\Users\User_Name\AppData\Roaming\Microsoft\Excel (replace user name with your user name, you can directly reach here by copy and pasting following without quotes in Explorer address bar - "%appdata%\Microsoft\Excel\") and see if is contains few files / folders. 4. Type "? application.StartupPath" without quotes and press enter
5. You will have your path below.
