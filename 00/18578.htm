daviddonalds say:

2011 Excel for Mac - How to highlight duplicate numbers in 2 columns

Hi,

Can you please help me with this request.
How to compare the contents in 2 columns and highlight the duplicate numbers/cells. The contents in both columns are only numbers.
For example: if A1 match B64 - highlight them both in yellow; if A12 match B2 - highlight them both in yellow. 
bobgreenblatt say: You can use conditional formatting for this. Assume your numbers go for 100 rows (change this if different).
 - chngeHighlight A1, and drag down till A1:A100 are selected
 - Go to Format - Conditional formatting.
 - click + at the bottom to add a rule
 - Under style at the top, select classic
 - Change the next drop down to "use a formula to determine which cells to format"
 - in the formula field, type: =or(a1=$b$1:$b$100)
 - Change the Format with to whatever highlight you want.
 - Click OK Twice.
This highlights the duplicates in column A. For Column B, do the same beginning with highlighting B1:B100. In the formula section type: =or(b1=$a$1:$a$100). Select the highlighting and click OK twice.

brian tillman [outlook mvp 2007-2019] say:

Autocomplete doesn't access your contacts in any way whatsoever, so there's no way to autocomplete by searching contacts.

Autocomplete looks at two strings: the display name and the address itself.  When you start typing, if either of those values starts with the string you type, the autocomplete entry will appear.  So, for example, if your autocomplete cache containg the name/address pair John Doe <john.doe@someaddress>, typing "J" will show it because both the display name and the address start with that letter.  If the pair is, instead Doe, John <john.doe@someaddress>, then entering either "D" (the beginning of the display name) or "J" (the beginning of the address) will display the entry as a suggestion.
