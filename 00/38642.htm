Follow the steps below to place your computer in “clean boot” 

  • Click the Start orb on your Desktop
  • Type msconfig in the Start Search box, and then press ENTER.



If you are prompted for an administrator password or for a confirmation, type the password, or click Continue. 

  • On the General tab, click Selective Startup.
  • Under Selective Startup, click to clear the Load Startup Items check box. 
  • Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.

http://support.microsoft.com/kb/331796

  • Click the Start orb on your Desktop
  • In the Start Search box. Type msconfig, and then press ENTER.


If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

  • Click Normal Startup - load all device drivers and services, and then click OK.

Trouble setting up a homegroup? I think I've cracked it. - Microsoft Community

skyreburn say:

Trouble setting up a homegroup? I think I've cracked it.

It's taken me about two hours to get this set up, so I'm hoping if I share what finally worked, it might save some of the rest of you some aggro.

I have two home computers, both running 64-bit Win 7 Professional.

To create a successful Homegroup all this had to happen:

1.  If you've accidentally created a homegroup in the process of setting the machines up or you've already set one up and it doesn't work, start by going into both computers and 'leaving' the homegroup.  (Control Panel/Homegroup)  Reboot both computers.

2. Disable or temporarily remove your firewall.  I use Norton 360 ver 3.  I ended up having to take it off altogether and reload it when everything was done.  It actually does say in the fine print that it will work automatically with home file sharing as long as you have shared files first before Norton is put on.  So I just took it off until I'd set up the homegroup.

3.  Make sure you have ownership of the files on both computers.  Look this one up in the help files, if you don't know how to do it.  Do it before you try to set up a homegroup.

4.  Make sure you have right-clicked on all the things you want to share, like Documents and Pictures, and chosen 'Share With' from the drop down menu.  Do it on both computers before trying to set up the homegroup.

5. Make sure you have both computers turned on and Power Saving off, so that neither accidentally powers down while you are working on them. 

6. Set up the homegroup on your main computer - the one that is attached to your router and/or peripherals.  The homegroup computer seems to see itself as the boss computer - sort of rooster on the dunghill, as it were - and, at least in my case, it took serious exception to not having possession of/direct contact with the peripherals.  My second computer is wi-fi networked and when I tried setting the homegroup up from that one, it wouldn't have it, despite the two computers being virtually identical in specs and the wi-fi connection good.  It went through very straightforwardly, however, when I 'left' the original homegroup with both computers and then set up a new one using the main computer.

Good luck.  May you crack it faster than I did.
