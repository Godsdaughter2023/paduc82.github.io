¡firedog say:

Thank you. I think it's clear why you're facing problems with your Bellsouth account. AT&T are changing the way users are authenticated when they access its mail servers, and the system they are introducing - called OAuth - isn't supported by many mail programs, including Windows Live Mail. The change now seems to have been applied to your account. This means that you can no longer use your account password in Windows Live Mail, so that's most probably the reason why you're seeing these errors. 

The solution is simple enough: you have to generate a 'secure key' and use that instead of the account password. The instructions are on this page: Create a secure mail key 

Please don't think that your email is any less secure because of this - it isn't. The communication between Windows Live Mail and AT&T's servers is just as safe and just as strongly encrypted as it was before since you're using TLS.

The error message you're seeing about the Windows Live Calendar Service arises because you're signing in in Windows Live Mail. The calendar no longer synchronizes with the online one because of changes at Outlook.com. You can read more about this fault in this (I'm afraid rather dated) article: Error when attempting to connect to the Windows Live Calendar service
