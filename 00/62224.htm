a. user say:

The file that you download contains the installer for the program. Program Files and Program Files (x86) are where your programs are installed.

You want to keep the installer separate from where the program is installed. In fact, after the program has been installed it would be best to move the installer to an external USB hard drive or a USB flash drive, and include a short text file with the name and version of the program, to whom the program is registered and (most important) the license key.

Best practices for security are central to the design of any operating system. The Program Files and Program Files (x86) folders can only be written to by an account that has Administrator permissions. Since you are (hopefully) using an account with Standard permissions for your day-to-day computing, you would not be able to save the installer to those folders without temporarily elevating your permissions (if you are able to supply an Administrator password) and we don't want to do this unless it's really necessary.

You want to save all your installers in a single, easy to access location in case you need to reinstall the program. And you want to keep your installers outside your computer in the event that your computer becomes unusable.

Most web browsers give you the choice of specifying a download location every time you download something, which can get tedious, or specifying a default  location to hold all downloads. For convenience, most people specify a default location which either the Downloads folder their desktop.
