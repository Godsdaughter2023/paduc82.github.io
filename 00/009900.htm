   =Forms!yourform!txt1

What do you mean by "sufficient space"? The distance between the last line of a footnote & the Footer content (i.e., page numbers), by default, should be consistent regardless of the length or number of footnotes. The position of Footer content is fixed relative to the bottom edge of the page - it doesn't vary from one page to another within a Section. The number of lines per page typically will differ, however, due to Widow/Orphan Control as well as other formatting variables.

Hi Renay,

No, AuthorTec File Doctor does not save the DOCX file as a filtered HTML file, it is a standard HTML file with all of Word's style attributes embedded.

Since you need the actual HTML file and not just the images, use the following VBA macro.

Sub SaveDocxAsFilteredHTML()
    Dim strFileName As String
    Application.DisplayAlerts = False
    strFileName = ActiveDocument.FullName
    strFileName = Left(strFileName, Len(strFileName) - 5) & ".htm"
    ActiveDocument.SaveAs2 FileName:=strFileName, AddToRecentFiles:=False, FileFormat:=Word.WdSaveFormat.wdFormatFilteredHTML
End Sub

It will place the ".htm" file you need in the same folder as the source document.

Hope this helps
