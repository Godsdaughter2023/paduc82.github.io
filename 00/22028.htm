You can't step into a procedure that takes arguments.
The SelectionChange and Change event procedures are parsed a object (Target) that is a range object of the range that has been selected or changed.

You could test these event procedure like this:

 

Private Sub TestWsSelChng()
    Dim rngTargTest As Range

Hi,

Please let us know the model of your PC?

The driver needs to be installed before the device can start properly working.

  • Determine which among your PCI devices is causing the problem. Click "Start." Type "device" in the search box provided and hit Enter. Select "Device Manager" under the "Control Panel" group.

  • Click the arrow beside "Other devices" and right-click "PCI Simple Communications Controller." Select "Properties" and click the "Details" tab.

  • Take note of the entry in the "Details" tab. The entry should be something like "PCI\VEN_1093&DEV_70B8." This indicates the vendor (VEN) and device ID (DEV) of the PCI device. Take note of these values because you will need them to search for the driver that the computer needs.

  • Open a Web browser and navigate to PCIdatabase.com to match the vendor and device details, and find out which PCI device is malfunctioning in your computer. Once you know which device is malfunctioning, use the installation disc that comes with the device to install the driver to the computer. If you do not have a copy of the PCI device driver, search for the driver online (see Resources).

mike h.. say:

Hi,

Because you have a text string in B4 that cannot be coerced into a number I don't know of any way to get SUMPRODUCT to work.

The apostrophe in front of the 800.... is making it a text value so IMO the simplest way to do what you want is a user defined function and they're easy. Do this:-

ALT+F11 to open VB editor and you should see 'Project Explorer' on the left side of the editor. If you can't see it hold down the CTRL key and tap R. On the left you will see VBAProject(Book1) where Book1 is your workbook name.

Below that right click 'ThisWorkbook' and insert a module and paste the code below in on the right. Close VB editor, we're finished in there.

Back on the worksheet we call this with a formula the same as you would any native excel function. Put this in a cell:-

=MySum(B3:B100,6)

So B1:B100 is the range with your (text) numbers in a we sum the column offset by 6 rows from this which is Col H. You can change the range and the offset to anything you want.

The last thing is you must now save your workbook as a macro enabled workbook with the .XLSM extension.

Function MySum(Rng1 As Range, oset As Long) As Double
Dim c As Range
On Error Resume Next
For Each c In Rng1
    If Val(c.Value) >= 8000000 And Val(c.Value) <= 8900000 Then
    MySum = MySum + c.Offset(, oset).Value
    End If
Next
On Error GoTo 0
End Function
