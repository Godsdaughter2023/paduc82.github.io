marshall barton say: As long as you have the group KeepTogether property set, you can do that by adding a text box (named txtDtlCnt) to the detail section.  Set its expression to =1 and RunningSum property to Over Group.  Then a group header text box can use an expression like:

ken sheridan say: Forget that.  The other thread was by somebody else.  You can force a group header to repeat by setting its RepeatSection property to True (Yes).  Then:

1.  In the report's module's Declarations area declare a variable:

    Dim intCount As Integer

2.  Add a text box TxtCont with a ControlSource of ="(cont.)" to the group header.

3.  In the group header's Format event procedure put:

    intCount = intCount + 1
    Me.txtCont.Visible = (intCount > 1)

4.  In the group footer's Format event procedure put:

    intCount = 0

When you remove all your security info, there’s a 30-day wait during which we help protect sensitive parts of your account so nobody can change your security settings or use your payment options. In another words:

When account security information is changed, a verification message is sent.  This can be an email or an SMS text message sent to a phone.  The email or message will contain a verification link or a link to cancel the changes.

To make the security information changes, you open the message and use the verification link.  if you do not use the cancel link or you do nothing, then after 30 days, the changes will be made.
 
After clicking on a verification or cancel link, you have to login, so there is no problem with someone else getting your old verification address.
 
You have started the process to remove the old verification address.  After the 30 days, the address will be removed.
