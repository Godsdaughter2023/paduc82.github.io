Hi Sharon,

If you want to change the name of your Windows computer, see the steps below:

  1. Click the Start Button and right-click on Computer, then select Properties.
  2. Under the Computer name, domain, and workgroup settings section, click on Change Settings.
  3.  Click Change button under the Computer Name Tab.
  4. Under the Computer Name field, you can type the new name for your computer.
  5. Click on OK.
  6. Click on OK, then Apply.
  7. You will be prompted to reboot the computer, and once your reboot is complete, your new computer name will be in effect.

Note: You need to be logged in with an account that has Administrator rights.

Let us know if you have any other questions.

Try resetting WordPad's options

  1. Close WordPad if it is open
  2. Open Regedit (Windows + R keys together and type regedit in the run dialog)
  3. Navigate to this Key and expand it: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Applets\Wordpad
  4. Delete the Options Key (it looks like a folder)
  5. Close the Registry Editor

The next time you start WordPad, the options key will be recreated with default values
