Hi,

 

 

 

Method 1:

 

Reopen last browser session is still an available option in Internet Explorer 10.

a. Open Internet Explorer 10.
b. Click on Tools and Reopen Last Browsing Session.

c. Internet Explorer will now open the tabs with the websites that you had open in the last Internet Explorer browsing session.

 

 

Method 2:

Try setting Internet Explorer to open with tabs from the last session automatically. This can be set by:

a) Open the desktop Internet Explorer

b) Click the Tools button

c) Click Internet Options

d) Under the General tab, under the Startup heading, select Start with tabs from the last session

e) Click OK

f) Close Internet Explorer

g) Open Internet Explorer

 

 

This setting only affects Internet Explorer with non pinned sites, so you will need to use the normal Internet Explorer shortcut to open Internet Explorer, but this should provide the functionality you are looking for.
