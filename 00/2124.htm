So, I got an idea. About 12 hrs later. The installer that you download from Comodo is a compressed package. So, I created an empty folder on my desktop and put the installer package inside of it. Then, with WinZip, I just right clicked and selected "Extract Here". That extracted the entire package.

Once extracted, your journey begins! Open the file 'cis', then download> installs> xml_binaries> cis.

Scroll to the bottom and right click on the cis_setup_x64.msi and select "Run as Administrator".

This will install Comodo Internet Security, placing all files and folders in "Program Files". Then, of course, a system restart. And your done.

I hope that you have found this useful. I do not know if this process will work on other security software suites. But, it might put you on the path.

“There is a problem with this website’s security certificate”. This error message appears when the security certificate presented by the website is not issued by a trusted certificate authority. The computer warns you about the possible threat this website can cause if it’s opened ignoring this fact that it does not contain the genuine trusted certificate.

However, if it’s a known website and you were able to work on this website earlier and suddenly it stopped working with this error message, we can make some changes in the browser to get this working.

The permanent resolution for this issue would be that the organization that host this website purchase a certificate for each webserver from a third-party provider.
