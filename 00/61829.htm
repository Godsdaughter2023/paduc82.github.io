mr__grey say:

I had the same issue after my Window 10 64 bit upgrade to version 1079 yesterday. i.e. Saitek Cyborg keyboard would not work. This is a keyboard driver problem.  HERE IS THE SOLUTION THAT WORKED FOR ME :-

A) First identify the current Saitek cyborg device in Device Manager

If you are sure which is the device you can skip this and go to section (B). further down.

To identify the driver I followed subsections 1 to 5 on the Saitek website titled “Locate and update the driver for your device’s HID ” https://logitechsupport.force.com/saitek/s/article/Saitek-device-driver-installer-does-not-complete as follows :-

Locate and update the driver for your device’s HID

Open Device Manager: 

  • Windows 7, 8, 8.1: Control Panel > System > Device Manager

  • Windows 10: In the Task Bar, right-click the Windows flag and select Device Manager

  1. In Device Manager, search for your device. If it’s a mouse or keyboard, navigate to the Mice or Keyboard sections and expand. If it’s a joystick or any other device, expand the Human Interface Devices section.

For example, if you have an X52 Pro, this is a joystick and appears as an HID-Compliant game controller.  (A mouse will be a HID-compliant mouse and a keyboard will appear as an HID Keyboard Device.) Starting from the bottom entry, right-click the HID part and then select Properties.

  1. In the Properties window, select the Details tab.

  2. In Property drop down menu, select Hardware Ids. Under Value, you will see a string of letters and numbers. Use this information to find the correct HID for your device. For example with the X52 Pro, you’ll see HID/VID_06A3&PID_0762&REV_0123:

  • VID = Vendor ID. This is specific to each manufacturer.  For Saitek products, this will always be 06A3. 

  • PID = Product ID. This value is specific for each product within the VID. 

  • REV = Firmware revision number. In this case, for the X52 Pro in this example, the  revision number is v1.23.

  1. Once you’ve determined the right HID part of the device, click OK. 

NOTE: In my case , in Control Panel, Device Manager, the Saitek device was not listed under Keyboards, but under a heading “other / unknown device” or similar. So the identification process above was very useful to ensure I had the correct device.

B) Update driver for the Saitek device

1) In Device Manager, right click the Saitek Keyboard Device identified above.

2) select "update driver software" --> browse my computer for driver software

3) select "let me pick from a list of drivers on my computer"

4) From the list select "HID keyboard Device"  (NOTE I DID NOT SELECTEITHER OF THE OTHER 2 OPTIONS i.e HID COMPLIANT DEVICE OR SAITEK CYBORG KEYBOARD (HID)

Hope that helps.

Is Windows 7 better than Windows XP - Microsoft Community

albert johnson say:

Is Windows 7 better than Windows XP

I have Windows XP so Is Winows 7

a. user say:

On a new computer with decent hardware Windows 7 is much better than Windows XP.

I have put Windows 7 on some older Dell desktops and with enough memory 2 GB or more (preferably 4 GB or more) it is definitely better. As I wrote earlier:

Many things are better in 7 than in XP...

• Aerosnap (can drag a window to the far left or right of the screen and it fills half a screen) - very useful!

• Internet Explorer 9 much better than Internet Explorer 8.

• Multiple monitor settings; Windows and P switch monitor settings fast (Although Actual Windows Manager helps both out alot).

• Increased security - UAC, Firewall.

• Taskbar/looks - pinning items.

• Drivers alot of things work straight out the box/Windows Update is far superior downloading/updating drivers.

• Support in XP ends in 2013. However now things are becoming unsupported in Windows XP. Internet Explorer 9, Windows Live Mesh are excellent Microsoft Services which you may use for Free with Vista/7 and are unavailable for XP.

• Shortcut keys, Windows and P, Windows and a number (loads the first 9 items pinned on the toolbar).

• MSPaint is much better. Snipping Tool, Sticky Notes.

• Library organisation is much better, folder layout etc. (Alt and ↑ arrow key) goes to parent folder.

• In general just alot more stable than XP (if done with a clean install) and runs alot faster if you have a decent computer with 4 GB of RAM or superior.

I am a fan of Windows 7 actually I like it alot more than XP. Vista with SP2 (clean install) is almost identical to 7 (now). Taskbar pinning and aerosnap aren't there which I keep forgetting when I am on a Vista PC. Also it tends to demand more RAM. Performance of 7 is slightly better than Vista (for me). Many people experienced significant problems with Vista.

If you add Office 2010/Windows 7 combination vs Office 2003/Windows XP then there are alot more advantages.

rainbowdash loc say:

In short, the Xbox One is not reverse compatible, it is possible that games could be downloaded for use on the new console if that option is explored.

Xbox One is basically a Multi-Processing device designed to easily swap from gaming; apps; television; movies; etc.

The unit is running multiple operating systems on start up, one specifically designed to switch between applications. To make a console reverse compatible they must add the operating system to run those games too. This would sacrifice performance.

From a technological standpoint I don't believe there is a Blue-Ray disc reader that reads HDD also. "Not 100% on that"

IGN stated that "Xbox Live Vice President Marc Whitten elaborated on the situation, saying, "The system is based on a different core architecture, so back-compat doesn't really work from that perspective.""

www.ign.com/.../xbox-one-not-backwards-compatible
