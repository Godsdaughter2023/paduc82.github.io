a. user say:

Hi,

Thank you for posting your query in Microsoft Community.

I can understand how frustrating it could be when things go unusual.

I will certainly help you to resolve the issue.

Try the below methods and check.

Method 1: Run Power troubleshooter

Power troubleshooter is an automated Windows tool which finds and fixes the common issues with the Power settings in your computer.

  1. Press the Windows key on the keyboard.

  2. Type troubleshooting and hit Enter.

  3. Click on View all on the left panel.

  4. Click on Power.

  5. Click on Next and follow the onscreen. 

Method 2: Allow the computer to turn off the device to save power setting controls how the keyboard and mouse is handled when the computer enters sleep and can be used if a driver misrepresents how it handles sleep states. When this setting is checked (enabled), Windows puts the device to sleep and when it resumes it puts it back to D0. When this setting is not checked (disabled), Windows completely halts the device and on resume reinitializes it.

  1. Open Device Manager Press Windows Key and X Key, select Device Manager.
  2. Right-click on "keyboard", "network adapter" and "mouse" drivers on at a time. After that, click on "Properties" and choose "Power Management" tab. Uncheck the option which says "Allow the computer to turn off this device to save power".

If the issue persist follow the below method.

Method 3: Clean Boot.

Follow the article How to perform a clean boot in Windows in the below link and check if it helps:

https://support.microsoft.com/en-us/kb/929135

Disclaimer: A clean boot is performed to start Windows by using a minimal set of drivers and startup programs. This helps eliminate software conflicts that occur when you install a program or an update or when you run a program in Windows. You may also troubleshoot or determine what conflict is causing the problem by performing a clean boot. You must log on to the computer as an administrator to be able to perform a clean boot. Your computer may temporarily lose some functionality when you perform a clean boot. When you start the computer normally, the functionality returns. However, you may receive the original error message, or experience the original behavior if the problem still exists. If the computer is connected to a network, network policy settings may prevent you from following these steps.

NOTE: Once you perform clean boot do refer to the section “How to reset the computer to start normally after clean boot troubleshooting” to boot the computer in normal mode.

Hope it helps.

Let us know if you have any further query on Windows, we are glad to assist you.
