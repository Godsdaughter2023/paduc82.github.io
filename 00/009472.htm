The file you deleted is thumbnail cache, which is automatically created by your phone's operating system. It contains thumbnails of your media files for quicker access, and as a system created file, deleting it is useless as it will be automatically rebuilt.

 

If you have a lot of pictures and videos on your device the thumbnail cache will, of course, be larger in size. In order to prevent the cache file ballooning, you could move some of your media files off the device with Nokia Suite, thus reducing the file size.
