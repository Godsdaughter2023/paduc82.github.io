Hi Bussoboy,

Let’s follow these steps & check if the issue persists:

Important:  This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, see the article to view the article in the Microsoft Knowledge Base: 322756 (http://support.microsoft.com/kb/322756/) How to back up and restore the registry in Windows.

a. Click Start, type ‘regedit’ in the Start, Run box, and then click regedit.exe in the Programs list.

b. Locate and then click the following registry subkey:

 HKEY_CLASSES_ROOT\AllFilesystemObjects\shellex\ContextMenuHandlers

c. Once you are at that key, right click and choose the New Key option.

d. Now you will have to double-click on the Default value and enter the following:

 {C2FBB630-2971-11D1-A18C-00C04FD75D13}

e. Click OK and continue.

f. If you want to enable Move To, you will have to repeat the same steps, except creating a new key named Move To, and using this value:

{C2FBB631-2971-11D1-A18C-00C04FD75D13}

g. Now when you right click on a file or folder to check if the options are available.
