I got this from Chris DeHerrera's website (he is one of our moderators) which I think answers your question:

Multiple Users on a Single PC

ActiveSync does not allow multiple users to login to a PC and synchronize like you may expect. The first user that connects is able to use ActiveSync normally. When the second user logs into the PC, if the Pocket PC is in the cradle, ActiveSync will attempt to setup another partnership because the default partnership Pocket_PC is in use. To work around this issue, I suggest installing ActiveSync Keys by Information Appliance Associates from http://www.pocketpcfaq.com/applications/ and configure ActiveSync to connect Pocket PCs as Guest Only. This allows all other users that login to not have to remove the Pocket PC from the cradle in order to avoid the new partnership wizard. Don’t worry you can still sync with the original partnership because it is already established
