Your first post did not indicate which build you were on, so it sounded possible that the builds had installed properly and were showing in your installed update history, but you didn't notice anything different. That is why I first suggested to check your current build number and how to find it.

During a Windows build upgrade, there are various stages of downloading and installation. jeanette cunningham say: You can make the database check for reminders when it opens and open a form showing reminders for the next day or week or month.

You would need a date field that shows the date for the activity - you probably already have this in the relevant table.
Create a query based on the table and include that date field and create a calculated field using that date field to get all activities due within the next week. Use the DateAdd function in the calculated field.
Search help in A2010 for examples of using DateAdd.

Check this query when the db opens and if there are any records in it (check with DCount), then open the form and show the remincers.
