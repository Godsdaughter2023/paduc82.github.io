Dim LastControl As Control

Property Let Freeze(Value As Boolean)
  Dim C As Control
  IsFreezed = Value

  For Each C In Me.Controls
    If TypeOf C Is MSForms.OptionButton Then
      If Value Then
        'Change to green if freezed

    End If
  Next

Yes, this can be done.

Add a class module to your project, name it "FreezeController" and put all code below my name in there.

In my current user form interface interaction code, e.g.;

Private Sub optBS_Click()
        Me.optStdy.Enabled = False  'to disable the option

You must remove (or better mark as comment) any of this lines and add a new line like this:
  FC.Freezed(optStdy) = False

Unfortunately we can not cover the Enter event nor the GetFocus event of a control, so you must write an Enter event for each control that should have the freezed feature.

In this way you can "ask a control" if it is freezed or not and switch to the last control:
  if FC.Freezed(optStdy) then FC.SetLastFocus

BTW, the FreezeController supports any kind of control, not only optionbuttons.

....FontColor :: Green
....(text) Appearance :: Dimmed

For the colors I gave you 4 values
  ForeColorEnabled
  ForeColorDisabled
  BackColorEnabled
  BackColorDisabled
which can be set to the desired colors.

Or you can use this methods to get the colors from an existing control:
  GetEnabledColors(Control)
  GetDisabledColors(Control)

For example your userform code should look like this:

--- schnipp ---
Option Explicit

Dim FC As New FreezeController

Private Sub CommandButton1_Click()
  'Toggle freezed state
  FC.Freezed(OptionButton1) = Not FC.Freezed(OptionButton1)
  FC.Freezed(OptionButton2) = Not FC.Freezed(OptionButton2)
End Sub

Private Sub OptionButton1_Enter()
  'Leave this OptionButton if freezed
  If FC.Freezed(OptionButton1) Then FC.SetLastFocus
End Sub

Private Sub OptionButton2_Enter()
  'Leave this OptionButton if freezed
  If FC.Freezed(OptionButton2) Then FC.SetLastFocus
End Sub

Private Sub UserForm_Initialize()
  'Here is the best place to setup your colors, for example:
  FC.GetEnabledColors OptionButton1
  FC.BackColorDisabled = FC.BackColorEnabled
  FC.ForeColorDisabled = 32768
  'Enable this userform
  FC.EnableUserForm Me
  'Add needed optionbuttons
  FC.Add OptionButton1, True 'Freezed by default
  FC.Add OptionButton2       'Unfreezed
End Sub

Option Explicit

Dim Controls As Object 'Dictionary
Dim WithEvents fcUserForm As MSForms.UserForm
Dim LastControl As Control
Public ForeColorEnabled As Long
Public ForeColorDisabled As Long
Public BackColorEnabled As Long
Public BackColorDisabled As Long

Sub EnableUserForm(UserForm As Object)
  If UserForm Is Nothing Then Exit Sub
  Set fcUserForm = UserForm
End Sub

Sub Add(Control As MSForms.Control, Optional Freeze As Boolean = False)
  If Control Is Nothing Then Exit Sub
  Freezed(Control) = Freeze
End Sub

Property Get Freezed(Control As MSForms.Control) As Boolean
  If Control Is Nothing Then Exit Property
  If Controls.Exists(Control.Name) Then Freezed = Controls(Control.Name)
End Property

Property Let Freezed(Control As MSForms.Control, Value As Boolean)
  If Control Is Nothing Then Exit Property
  If Controls.Exists(Control.Name) Then
    Controls(Control.Name) = Value
  Else
    Controls.Add Control.Name, Value
  End If
  On Error Resume Next
  If Value Then
    Control.ForeColor = ForeColorDisabled
    Control.BackColor = BackColorDisabled
  Else
    Control.ForeColor = ForeColorEnabled
    Control.BackColor = BackColorEnabled
  End If
End Property

Sub SetLastFocus()
  If LastControl Is Nothing Then
    'Should not happen, just to be sure
  Else
    LastControl.SetFocus
  End If
End Sub

Sub GetEnabledColors(Control As MSForms.Control)
  If Control Is Nothing Then Exit Sub
  ForeColorEnabled = Control.ForeColor
  BackColorEnabled = Control.BackColor
End Sub

Sub GetDisabledColors(Control As MSForms.Control)
  If Control Is Nothing Then Exit Sub
  ForeColorDisabled = Control.ForeColor
  BackColorDisabled = Control.BackColor
End Sub

Private Sub Class_Initialize()
  'Set default colors
  ForeColorEnabled = &H80000012
  ForeColorDisabled = &H808080
  BackColorEnabled = &H8000000F
  BackColorDisabled = &H8000000F
  Set Controls = CreateObject("Scripting.Dictionary")
End Sub

Private Sub fcUserForm_Layout()
  'Get first active control
  Set LastControl = fcUserForm.ActiveControl
End Sub

Private Sub fcUserForm_MouseMove(ByVal Button As Integer, _
    ByVal Shift As Integer, ByVal X As Single, ByVal Y As Single)
  'Get the last active control if the mouse moves
  Set LastControl = fcUserForm.ActiveControl
End Sub

To set the linked cell, right click the DropDown combo and select Format control -> Control tab.

The Input range for the list is the first field and the second field is the Cell link. Just click the icon at the right of the Cell link field, select the cell that you want to link, click the icon at the right of the displayed field and then OK.

The linked cell only displays the numeric Index of the selected item; not the actual item value. To get the actual value you need to insert a formula in the cell beside the linked cell and use the Offset function to find the actual value from the Input range. See the screen snippet below where:

A2:A12 is the Input range for the DropDown.

B4 is the linked cell showing the 7th item is the list is selected.

Formula in cell C4 is:   =OFFSET(A2,B4-1,0,1,1)

Note that with the Offset function the number of rows to offset is the number of times you would need to press the Down arrow to get to it from the Reference which is A2 and hence B4-1 because although it is the 7th item in the list, if the selected cell is A2 then the down arrow only needs to be pressed 6 times to get to the 7th cell in the list.
