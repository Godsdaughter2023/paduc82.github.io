    Calculated field in a query - Microsoft Community

    irammcp say:

    Calculated field in a query

    Hello,

    I need to create a column in a query that calculates date differences, rounded to Year and Months.

    Example: ¨Date Signed Up¨ 07/11/2011, minus todays date, equals 1yr 1month.

    Can you help me create this calculated date difference formula?

    Iram/mcp

    vanderghast say:

    Note that DateDiff returns the number of BOUNDARIES between the two dates.

     

    ? Datediff( "yyyy",  #12-31-2010#,  #1-1-2011#)

    1

     

    even if it is only ONE day of difference, the interval of the dates cross one year boundary.

     

    ? DateDiff( "yyyy", #1-1-2010#, #12-31-2010#)
    0

     

    even if there is 364 days between the two dates, you don't cross any year boundary.

     

     

    So, basically someone can get the "age" in months, then makes an integer divison by 12 to get the age in year, and the modulo 12 to get the remaining number of  months:

     

    ageInTotalMonth = DateDiff("m", date1, date2) + iff( Day(date1) <= Day(date2), 1, 0 )

     

    ageInYear:   (  DateDiff("m", date1, date2) + iff( Day(date1) <= Day(date2), 1, 0 ) ) \ 12
