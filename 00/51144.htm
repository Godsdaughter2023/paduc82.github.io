Port Forwarding

Netgear CG3100D

Login Information:

  • Default Gateway - 192.168.0.1
  • Username – admin
  • Password – password

Port Forwarding

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Port Forwarding under the Advanced section.
  4. Type a Service Name that has something to do with Xbox.
  5. Set the Protocol to Both.
  6. Set both the Start Port and End Port to 3074.
  7. Enter the IP Address of your Xbox console in the Server IP Address space.
  8. Click the Add button at the bottom of the page.

Disable Port Forwarding

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Port Forwarding under the Advanced section.
  4. Select the rule that pertains to port 3074.
  5. Click the Delete button at the bottom of the screen.

jay freedman say:

For the most part you can ignore anything in any folder named Temp. Some of these are "temporary" files created while a program is running as a sort of scratch pad. Others are there as backups or for other (undisclosed) reasons. Sometimes whatever program created them will delete them when you close the program, but others can become orphaned. Unless they're very large or there are a huge number of them, they aren't taking up much disk space and they aren't causing problems.

There are some temporary files that should be cleaned out after a program such as Word crashes or is abruptly shut down (for example, by terminating it from Task Manager). Graham Mayor discusses this, and methods for removing them, in http://www.gmayor.com/what_to_do_when_word_crashes.htm. He doesn't mention anything about AppData\Local, which is not at all the same as AppData\Roaming. I'd advise you not to mess with anything in AppData\Local unless you know exactly what you're doing and what those files are used for.
