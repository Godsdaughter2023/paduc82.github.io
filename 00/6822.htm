romfordbluenose say:

I found out a work around on my laptop/PC but still don't know the root cause.

After upgrading to windows 10 and perhaps with some updates later then if you have a network connected printer which is classed as a web-services printer then the pdf never gets output to it correctly and the printer server just hangs.

You can get round this by changing the port from being a WSD port to the actual IP address.

You do this by going into control panel/hardware and sound/devices and printers.

You should then see your printer in the list. Right click on the printer and select properties. You should then get a popup box with a lot of tabs. Click on ports. You will probably find WSD is ticked. Untick this box and tick the box with an IP address in the title. If one doesn't exist then add a new port and create the port with the IP address of the printer.

Thank you for replying.

Outlook treats anniversaries, holidays, and birthdays as all-day events (an all-day event is defined from midnight to midnight in the local time zone). When the time zones change, all-day events move. Therefore, when you change the computer's time zone, the all-day event moves to display in the new time zone. An event, such as a holiday or birthday, will appear in the Calendar on two consecutive days. For example, a birthday could appear from 6 A.M. one day to 6 A.M. the next day.

You can do one of the following:

  • Use the additional time zone feature in Outlook to display the local time zone and do not change the operating system time zone.
  • Correct each all-day event for the current time zone setting.
  • Change the time zone back to the one used when the items were created.
