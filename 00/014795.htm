Way to get pages to print double sided and some to print single - Microsoft Community

taylorkyrzyk say:

Way to get pages to print double sided and some to print single sided?

Trying to create packages for work and some sheets I need to come out double sided while some I need to stay single sided. Not sure if there's a way to do this but it would save me so much time figured it was worth an ask.

 

Thanks everyone!

 

 

jay freedman say:
Usually the simplest solution is to make a note of which pages are single-sided; then do two print runs, one of just the single-sided pages and one of just the double-sided pages. Then manually combine them in the correct order. In the Print dialog, enter the numbers of the pages to print, separated by commas or in ranges (for instance, 1-4,7,12).

If for some reason you need to print the whole document in one pass, you need to add an Odd-Page section break at the end of each single-sided page (and remove any manual page break that might be there). Then you can print the whole document as double-sided, and the section breaks will force a blank page on the back of each single-sided page.

If the document has printable page numbers, the blank pages will also have numbers (although they won't be printed). For instance, if pages 1 and 2 are double-sided and page 3 is single-sided, the next page after that will show the number 5 (and page 4 will be blank). Trying to "fix" this so the printed pages have consecutive numbers is far more trouble than leaving out the section breaks and using the two-print-run method.

By the way, don't try to print multiple copies of a package from Word; instead, print one copy and then use a copier to make more. It's faster, cheaper, and less work.

The formula in B2 returns the text string "January", not the range named January.

To get around this, you can use the INDIRECT function. It converts a text string to the corresponding range, if possible.

=VLOOKUP(A2,INDIRECT(VLOOKUP(MONTH(VALUE(A2)),MonthLookup,2)),3)
