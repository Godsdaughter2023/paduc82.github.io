talking moose say:
Sorry, you can't make Outlook use Word's dictionary. Outlook for Mac is the first of the Office for Mac applications to do away with using Microsoft's dictionaries and instead use the built-in Mac OS X dictionaries. I would assume future versions of the product will do the same.

You can edit your Mac OS X exceptions dictionary a couple of ways. If you find that a word is being flagged as misspelled (red dotted underscore) then you can right-click or Control-click that word and select Add to Dictionary from the contextual menu that appears. You'll need to log out and log in again for this to take effect or simply restart. Or if you can create a plain text list of words then you can edit the following file (where " ~ " denotes your home folder) using TextEdit and add them all:

~/Library/Spelling/LocalDictionary

I don't believe this will autocorrect words for you but it should stop them from being flagged as misspelled.

Hope this helps!

First condition: (Stops formatting of empty cells.)

Second condition: (Sets format for months/years previous to current month)

Tips for setting conditional format to a range:

Method 1: Select the entire range to be conditionally formatted and enter the formula as if it is only for the first cell and Excel will look after the rest.

 

Method 2: Select the first cell only and set conditional format and then copy the cell and paste special -> formats to the additional cells.
