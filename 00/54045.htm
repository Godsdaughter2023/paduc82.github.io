Overview

BEC 5102 Series

Login Information

Gateway - 192.168.1.254

Username - admin

Password - admin

 How to Find SSID (Network Name)

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Wireless.
  5. Your network name will be listed as SSID.


 Wireless Security

How To Find Wireless Passphrase (WPA/WEP)

  1.  Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Wireless.
  5. Your passphrase will be listed as Key 1 or Pre-Shared key

 

Changing Wireless Security Type/ Disabling Wireless Security

  1.  Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Wireless.
  5. Use the Authentication Type drop down to choose your desired security
  6. If WEP is selected, select the WEP Key Length (either 64-bit or 128-bit).  The password will need to be 10 charecters in length (if 64-bit) or 26 characters in length (if 128-bit) and can only use 0-9 and A-F.
  7. If WPA is selected use the Encryption drop down to select your desired cipher type (TKIP preferred).  The password can be between 8-63 characters, is case sensitive and can use any character on your keyboard.
  8. Select Apply after inputting your desired passphrase.

 

MAC Filtering

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Wireless.
  5. In the Wireless MAC Address Filter section set the radio button to Deactivated.
  6. Select Apply.

 

 

Change Wireless Channel

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Wireless.
  5. From the drop down menu labeled Channel ID select the desired channel. (Channels 1,3,9,11)

 

Change Wireless Mode

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Wireless.
  5. From the drop down labeled 802.11b/g select 802.11g.
  6. Select Apply.

 

DHCP Reservation

*This feature is unavailable on this device.*

 

UPnP  

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Access/Management.
  4. From the sub-menu select UPNP.
  5. Enable or disable UpnP as needed.
  6. Select Apply.

 

 

DMZ

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Advanced Setup.
  4. From the sub-menu select NAT.
  5. Select DMZ.
  6. Set the radio button to Enabled.
  7. Add the Xbox’s IP address into the DMZ Host IP Address box.
  8. Select Apply.

 

 

 

Disable DMZ

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Advanced Setup.
  4. From the sub-menu select NAT.
  5. Select DMZ.
  6. Set the radio button to Disabled.
  7. Select Apply.

 

 

Port Forwarding

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Advanced Setup.
  4. From the sub-menu select NAT.
  5. Select Virtual Servers.
  6. Set the Rule Index to a number not in use in that table below.
  7. For the port start and end numbers enter 3074.
  8. Enter the xbox’s IP address to the Local IP Address box.
  9. Select Apply.

 

 

Disable Port Forwarding

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Advanced Setup.
  4. From the sub-menu select NAT.
  5. Select Virtual Servers.
  6. Select the rule number using the Rule Index drop down.
  7. Select Delete.
  8. Select Apply.

 

Firewall

*This feature is unavailable on this device.*

 

How to Find Firmware Version

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu at the top select Maintenance.
  4. From the sub-menu select Firmware.
  5. The firmware version will be listed as Current Firmware Ver.

 

 

Firmware Update

*This feature is unavailable for this device.*

 

Restore Factory Defaults

 

Through Hardware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Using a toothpick or tip of a ballpoint pen press and hold the reset button located on the bottom of the device for 15 seconds or until all the ethernet lights flash in unison.

 

Entering PPPoE

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Internet.
  5. In the section marked PPPoE/PPPoA enter your pppoe user name and password.
  6. Select Apply.
