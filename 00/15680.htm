rich007a say:

Hi,

Assuming you created the chart within PowerPoint, the data is stored in an Excel workbook, in sheet1, in a Table ("Table1").  So you need to work with Excel's ListObjects object.

 

First you must create a reference to the Excel Object Library.

 

Sub EditPptChartData()

' Add a reference to 'Microsoft Excel xx.0 Object Library' via VBE > Tools > Rerferences...

'
    Dim cht As Chart
    Dim wb As Excel.Workbook
    Dim ws As Excel.Worksheet
   
    Set cht = ActiveWindow.Selection.ShapeRange(1).Chart     'get the chart object
   
    cht.ChartData.Activate                              'open the charts data in Excel
    Set wb = cht.ChartData.Workbook
    Set ws = wb.Sheets(1)
   
    ws.ListObjects(1).ListColumns(3).Delete    'delete series 2
'    ws.ListObjects(1).ListColumns.Add(3)        'use to put series 2 back in
    ws.ListObjects(1).ListColumns.Add            'add a new series at the end
   
     'Fill data in the new series
    ws.Cells(1, "D").Value = "New series"
    ws.Cells(2, "D").Value = 2.1
    ws.Cells(3, "D").Value = 2.2
    ws.Cells(4, "D").Value = 2.3
    ws.Cells(5, "D").Value = 2.4
   
     'If you want to close Excel:
    Dim xl As Excel.Application
    Set xl = wb.Parent
    wb.Close
   
     'Check there are no other xl files open:
    For Each wb In xl.Workbooks
        If wb.Windows(1).Visible = True Then Exit Sub
    Next wb
    xl.Quit
    Set xl = Nothing
End Sub

 

Use     ws.ListObjects(1).ListRows    to delete or add rows to the data (i.e. to change the number of cateogries).  So you could use something like:

 

    Do While ws.ListObjects(1).ListRows.Count > 7
        ws.ListObjects(1).ListRows(8).Delete
    Loop

 

You may find this page very useful: http://www.jkp-ads.com/articles/Excel2007TablesVBA.asp

 

Hope that helps.

Cheers
Rich

greginmich say:

Is there a way I can stop windows defender from automatically deleting files it considers to be threats? I would prefer if it would notify me on a file by file basis, because it keeps telling me it "fixed" a malicious file that I know was not a threat. I want to be able to add files to a whitelist, instead of having windows defender just delete them, but I don't see that option.

You can turn off Automatic Remediation by either manually editing the registry:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/disable-automatic-quarantine-in-w10/8183c443-6ad0-4e28-a114-085a3db32bbf

Or by running a REG command: Right-click on the Start button, select Command Prompt (Admin), and then copy, paste, and enter the appropriate command:

Turn off Automatic Remediation:

REG ADD "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableRoutinelyTakingAction /t REG_DWORD /d 1 /f

Turn on Automatic Remediation:

REG DELETE "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableRoutinelyTakingAction /f

GreginMich
