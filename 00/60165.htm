Wireless Security

Login Information

  • Gateway: 192.168.1.1
  • Username: root
  • Password: 12345

 


 

How to Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Security" from the sub menu.
  6. If the "Network Authentication" is showing "802.1X" the passphrase will be based on the "Current Network Key". (By default it will be key 1, and the wireless passphrase will be "Network Key 1".)
  7. If the "Network Authentication" is showing "WPA-PSK", "WPA2-PSK", or "Mixed WPA2/WPA-PSK", the wireless passphrase will be labeled "WPA Pre-Shared Key".

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Security" from the sub menu.
  6. Select the desired security type from the "Network Authentication" drop down menu. (To disable security, select "Open" and then "Disabled" from the "WEP Encryption" drop down menu.)
  7. Select Save/Apply.

Instead, clone your current hard drive to your new hard drive.  All you need is an external drive enclosure and disk cloning software.  Plus, once you have made certain that everything is working properly, put the old drive in the external enclosure and use it for backups.

Some of the cloning apps below also make disk "images" (e.g., Acronis True Image).  A disk image can be compressed so that for example 58 GB of data might produce a 40 GB image file.  And some of the apps also can selectively backup specified files.

Most external drive enclosures connect to a USB port.  Just make sure that you buy one with the correct "form factor" (laptop = 2.5 inch; desktop/tower = 3.5 inch) and internal drive connector.  Many but not all of the enclosures can deal with both SATA and PATA drives.

Sounds like you somehow changed the defaults but now don't have enough authority to change them back to the real defaults.  E.g. the Restore plan defaults button is only allowing you to undo your personal changes.

FWIW I launched the Control Panel Item I mentioned from Administrator Windows PowerShell and with it I have more choices in

Control Panel\All Control Panel Items\Power Options\System Settings

e.g. instances of Change settings that are currently unavailable.  

Click that  and the Shutdown settings checkboxes become active.  I suspect you may find that Sleep in there is unchecked and either hidden or greyed out otherwise.

This doesn't have exactly the intended effect, e.g. no difference from specifying a Category or not but it's close enough and it documents what you want to do:

Get-ControlPanelItem -Name "Power Options" -Category "System and Security" | Show-ControlPanelItem

So, in order to get into the System Settings dialog from there click on Choose what the power buttons do.
