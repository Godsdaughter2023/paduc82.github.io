Inventory spreadsheet using barcode scanner to enter count - Microsoft Community

paul425 say:

Inventory spreadsheet using barcode scanner to enter count

Is there a formula for adding and subtracting barcode entries and return the value back to the item list, and the item in the list?

sheen valentine ali msft say:

Hi Paul,

Assuming you have your barcode entries in one column, you would just need to use COUNTIF to count how many times each unique barcode entry is entered.

Note: Your barcode entries would need to be in a readable format (numbers) instead of bars.

As an example, I have created a spreadsheet with some barcode entries and an Inventory List as displayed below.

Note: I put the barcode entries into column A.

With this, I used a COUNTIF formula to count each unique entry (1111, 2222, and 3333) using the formula below:
=COUNTIF(A:A,1111)

With the range of all column A (A:A in the formula), and count the entry 1111.

I then substract this count from the initial quantity and put them in a new column using a simple substraction formula:
=L3-M3

Please note that the formula above depends on the cells you put your barcode entries and the inventory list in. You would need to adjust them according to your cell placement. If you would like to add instead of subtract, simply change the subtraction formula into an addition formula (For example: =L3+M3).

Sheen

daniel jackson say:
Hi,

Thanks for posting in Microsoft Community.

There are 2 methods to add the .vcf files into Outlook’s Contact.

1) You may open the. VCard file (*.vcf) and click on Save and Close to add it into Outlook’s Contact.

2) You may copy all the vCard files (*.vcf) and paste it into a folder and then import it into Outlook following the below mentioned instructions:

1. On the File menu, click Open and Import.
2. Click to select the Import a vCard file (*.vcf), and then click Next.
3. Click to select the vCard file, and then click Open.
