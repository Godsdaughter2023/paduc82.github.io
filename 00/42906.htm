Can you please tell me how to correct this error. cannot find a script file c:\ progm files\xvid\checkupdatelauncher.vbs".
lemp say:

Xvid is a video codec used with various video files.  You likely had installed some video file player or editor.

You do not say, but I assume that you get this message when your computer starts.

The script appears to be an automatic updater, presumably to check for new versions of Xvid.  It also appears that you uninstalled that program, but the uninstaller failed to remove the entry that attempts to run the now-deleted script.

Because this entry could be in several different locations, the easiest way to find and remove it is to download (free) Autoruns (or run it "live"):  https://docs.microsoft.com/en-us/sysinternals/downloads/autoruns

When Autoruns has completed its scan (it will say "Ready" at the bottom left), click the "Logon" tab.  Search for "Xvid" in the "Image Path" column.  When you find it, UNcheck the box at the left end of the line and close Autoruns.  The error message should no longer appear when Windows starts.

ruwim.b say:

There is no longer an easy way to move chat history from older Skype installation to the new Microsoft Skype for Windows 10 (Skype UWP) installation.

 

The local chat history used by Skype UWP is not located in the %appdata% folder, but in this location:

 

%localappdata%\Packages\Microsoft.SkypeApp_kzf8qxf38zg5c\LocalState\<Skype Name>

 

However, it will not help to move the old chat history to this folder either, because it is now stored in different database files. There is still a main.db database file, where some data is stored, but the main history is saved in skype.db file. The structure of this new skype.db file is different from the main.db file, so they cannot be replaced.

 

If you want to be able to move the history from the old installation to a Skype application on your Windows 10 computer, then you need to install the classic Skype for Windows Desktop on this computer, and so you will be able to follow the procedure described in the mentioned FAQ.

 

The classic Skype for Windows Desktop can be installed using this installer link:

 

http://www.skype.com/go/getskype-full

You're partly right but not in the way you think.

Blu-ray drives have a higher data transfer rate than DVD but the speed of the drive is constant, compared to DVD which is variable.

So with a game on the DVD you can speed up the drive when it is needed, as the lens can move more quickly to the part of the disc the needed data is on. So a DVD can layout the data 1,2,3,4.

However since blu-ray always runs at one speed the drive can't speed up and so data needs to be replicated on the disc more than once. Giving us a layout of say 1,2,3,4,2,1,3,1,4. Essentially rendering the extra storage of the disc nearly pointless.

Also hard drives are faster than optical media, so you install the game from blu-ray in order to maximise the space available on the disc and get a faster loading game.
