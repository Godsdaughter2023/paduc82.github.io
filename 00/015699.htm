There is a slim possibility you can get back your file. In the same folder where is was, choose Tools>Folder Options>View. Check View hidden files and folders and OK out. You may see a file with mostly the same name as your file, but the first character is a tilde ~.

Files missing icons in Office 365 on Windows 10 - Microsoft Community

timothyberto say:

Files missing icons in Office 365 on Windows 10

Hello.  I recently upgraded to Office 365 and uninstalled a really old version of Office.  When I did this, I noticed that despite my default apps being set properly, when you view the files in Windows Explorer, the files had no icons.  They were just these generic white boxes for random file types (.docx, .xlsx, etc).  I was able to fix the problem by doing the following:

Specifically for Windows 10, running Office 365 -

Go to Start > Settings > System > Default Apps

Scroll down to the bottom and click "Set Defaults by App"

The Set Default Programs window will pop up

Go down the list and find your Office programs one-by-one (for example Word 2016) and select it.  Repeat the next two steps for each program (Excel 2016, etc)

To the right you will see two options "Set this program as default" and "Choose defaults for this program", right above that, you will probably see something like "This program has 11 of 24 of it's defaults"

I just simply clicked  "Set this program as default" and I got 24 of 24 and my icons all started working

That is, of course, allowing Microsoft to choose them all automatically so some of you may need to go back and change certain files individually if you want to be more selective on which programs open which files. 

Hope that helps.
