In Word 2010 you will have to make use of a macro to add commands to the right-click (context) menus. Is there a reason why you can't simply use the keyboard shortcut Ctrl+A for Select All?

weaseldlb say: The reason for wanting to use the mouse instead of the keyboard is that I will already use that hand on the mouse to click copy, cut or paste (i.e. one less move)

stefan blom say:
There are shortcuts for copy (Ctrl+C), cut (Ctrl+X), and paste (Ctrl+V) as well, if that helps.

In the simplest case, to add Select All at the beginning of the "standard" Text context menu, you can run the following macro:

Sub AddSelectAllCommand()
CustomizationContext = NormalTemplate
CommandBars("Text").Controls.Add ID:=756, Before:=1
End Sub

For assistance, see http://www.gmayor.com/installing_macro.htm.

Note that if you are prompted to save the template after you run the macro, choose to do so.

ken sheridan say: I've always used this function:

Public Function GetAgeYearsMonths(varDoB As Variant, Optional varAgeAt As Variant) As Variant

    Dim intYears As Integer
    Dim intMonths As Integer

    If IsMissing(varAgeAt) Or IsNull(varAgeAt) Then varAgeAt = VBA.Date
    
    If Not IsNull(varDoB) Then
        intYears = DateDiff("yyyy", varDoB, varAgeAt) - _
            IIf(Format(varAgeAt, "mmdd") < Format(varDoB, "mmdd"), 1, 0)
           
        intMonths = DateDiff("m", varDoB, varAgeAt) - (intYears * 12)
            
        If Day(varDoB) > Day(varAgeAt) Then
            intMonths = intMonths - 1
        End If
            
        GetAgeYearsMonths = intYears & ":" & Format(intMonths, "00")
    End If
    
End Function
