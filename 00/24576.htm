Workbooks("Book1.xls")

' NOT

Workbooks("C:\Folder\Book1.xls")

 

Try the code below

 

    Dim s As String
    Dim WB As Workbook
    s = Application.GetOpenFilename()
    Set WB = Workbooks.Open(Filename:=s)
    Sheets("DEPR US ALLOCATION REPORT").Select
    Application.Goto Reference:="DB_COMPANY_ALLOCATIONS"
    Selection.Copy
    '
    '    Switch back to original file
    '
    Windows("TXFILE.xlsm").Activate

I have tried looking for the font, copying the deck to a new document, replacing the font using the "replace fonts" menu, and nothing seems to work. There is also nothing about this in the Microsoft help centre. 

john korchok say:

PowerPoint cannot apply a non-existent font.

System Font Regular is a pointer to a real font, but that font pointed to changes with different versions of macOS. The pointer name remains the same. The System Font used to be Lucida Grande, then it was Helvetica Neue Desk Interface, today it's San Francisco.

The System Font can't normally be used as a regular font, but it appears that there's a bug in Office for Mac that can allow it to happen. You wouldn't see the name San Francisco in your presentation, because the OS refers to it as System Font. You are seeing that.
