Probably the best way is to use File and Printer sharing which is built into all versions of Windows.

 

Whether it is WiFi or hard-wired does not make so much a difference as the type of connection that you have.  The higher bit-rate, the faster the file will transfer.  There are three speeds of USB, at least three common speeds of wireless WiFi (b,g,n), and three speeds of wired Ethernet (10-Base-T, 100-Base-T, and Gigabit).  The approximate maximum bit-rates of different connectons are listed as follows.  I use a 1/10 factor to convert to Bytes/sec.  Numbers are approximate maximums (ie you will never see these rates but good for comparison) and many conditions can reduce these speeds (particularly with wireless):

 

USB 3.0             = 4000 Mbit/sec (wired)

Gigabit Ethernet = 1000 Mbit/sec  (wired)

USB 2.0             = 280 Mbit/sec  (wired)

Wireless "N"      = 150 Mbit/sec (wireless)

100-Base-T Ethernet = 100 Mbit/sec (wired)

Wireless "G"      =  54 Mbit/sec (wireless)

USB 1.x             =  12 Mbit/sec (wired)

Wireless "B"       = 11  Mbit/sec (wireless)

10-Base-T Ethernet = 10  Mbit/sec  (wired)

You can fix the ERR_NETWORK_CHANGE by resetting IP/TCP. This is a rather simple procedure and you can do it by following these steps:

  1. Open Command Prompt as administrator.
  2. When Command Prompt opens enter the following lines:
  • netsh int ip set dns
  • netsh winsock reset

  3. Close Command Prompt and check if the issue is resolved.
