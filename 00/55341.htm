santosh y say:

Hi RichardSteel,

The NTuser.dat file is the registry portion of the user profile. When a user logs off of the computer, the system unloads the user-specific section of the registry (that is,  HKEY_CURRENT_USER) into NTuser.dat and updates it. If there is a user profile corruption then these files are not copied to the new user profile because it has the corrupt registry information.
