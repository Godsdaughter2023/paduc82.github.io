frederik long say:

It occurred to me that copying all but three of the files to a new profile directory is logically the same as deleting those three files from the existing profile directory. What would that accomplish?

-> I assume you are thinking of the user's registry files, e.g. ntuser.dat. Deleting these files would remove most of the user-specific settings.

Would copying the files (all but those three named files) to a new profile produce a better result?

-> No, it would not. To fully restore a corrupted profile you must restore a healthy version of all files. Here are two ways to do it:

  • Use System Restore to set Windows back to a point before the damage occurred.
  • Restore all files from an image that you created some time ago, either by restoring the whole System partition or by manually restoring the affected user profile folder (which requires advanced skills).

The KB article you used as a basis for your repair work is misleading. It creates the impression that one can fully restore a corrupted profile with a few copy commands. It should clearly state that in most cases, it delivers full recovery of all data files but not user-specific settings.
