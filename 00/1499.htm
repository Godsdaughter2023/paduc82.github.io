… Is there a way I can get the comments to change language along with the titles?


With the following language data on the Language Settings worksheet,

                           

First, delete anything that you have in the active worksheet's A2 cell and set a Data Validation list in A1 that references the 'Language Settings'!$AA$2:$AA$15. Right-click the name tab of the active worksheet and select View Code. When the VBE opens, paste the following into the pane titled something like Book1 - Sheet1 (Code),

Private Sub Worksheet_Change(ByVal Target As Range)
    If Target.Count > 1 Then Exit Sub
    On Error GoTo FallThrough
    Dim iLang As Long
    If Not Intersect(Target, Range("A1")) Is Nothing Then
        Application.EnableEvents = False
        With Sheets("Language Settings")
            iLang = Application.Match(Me.Range("A1"), .Range("AA2:AA15"), False)
            Me.Range("A2").Value = .Range("AA1").Offset(iLang, 2) & Chr(32) & .Range("AA1").Offset(iLang, 1)
            If Me.Range("A2").Comment Is Nothing Then Me.Range("A2").AddComment
            Me.Range("A2").Comment.Text Text:=.Range("AA1").Offset(iLang, 3).Text
        End With
    End If
FallThrough:
    Application.EnableEvents = True
End Sub

Tap Alt+Q when done to return to your worksheet. You can now use the drop-down list in A1 to change this,
