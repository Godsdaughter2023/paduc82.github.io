john mcghie say:
Artus:

You're indulging in a circular argument here :-)

The Line AutoShape tool is context-sensitive.  It cannot work on a toolbar because the toolbar cannot detect the context of the insertion point.  Several of the graphics tools are like that: they can't work without context, so they can't work on a toolbar.  Not in any Office version, not in any file format.

You can draw a line, then save it as an AutoText.

You can then record a macro to insert the AutoText.  You can put the macro button on the toolbar and thus insert your line with one click.

If you want to do this, the Word Help explains how.

Cheers

jim gordon mvp say:

Here are the results of my testing with VBA.


The CommandID for the line tool in older versions of Word is 130. When I try to add this tool in Word 2011 I get an automation error.


There is a new CommandID for HorizontalLineInsertClassic. That is number 3824. This displays the dialog box that has a handful of pictures, including a horizontal line. It's not what you want, I suspect.


However, I found a code snippet at MrExcel.com and modified it to work with Word. You could modify this code as desired and assign it to a toolbar button or menu.


Sub test()

ActiveDocument.Shapes.AddLine(100, 300#, 300, 300).Select

MsgBox "Here is a line"

Selection.ShapeRange.Line.EndArrowheadStyle = msoArrowheadTriangle

MsgBox "Now it has a tip"

Selection.ShapeRange.Line.Weight = 3#

Selection.ShapeRange.Line.DashStyle = msoLineSolid

MsgBox "do you ike it?"

End Sub
