What is firmware and what is symbian OS? - Microsoft Community

rossonero22 say:

What is firmware and what is symbian OS?

What's the difference between them?I lost some of the application,what should I do?Reinstall the firmware or the OS?

donald1---01 say: Some explanation.
Your phone is hardware, electronics you can touch. Software is soft and you can not touch it. Software are commands you enter via eg an application in the phone / computer / CPU to enable a certain task/function.
You have several layers of software. There is software which describes how the hardware should act but you have also software which describes how other software should program the hardware (or computer).
The firmware is the first software layer which connects the hardware with other software.
The OS (Operation System) is also a basic layer of software and uses the firmware commands to manage the hardware functions. The OS defines the interface between the user or applications and between the phone/computer hardware (via the firmware). Typical the OS can not be changed and is needed otherwise the phone / computer will not function.
The firmware is typical updated by Nokia. Sometimes Nokia issues a new version of the firmware software to resolve bugs or enable new enhancements/functionality.
The OS on the phone is Symbian and is build by a seperate company. You can not remove or change the OS on the phone.

Applications is software which uses software functions from the OS and/or firmware. The user can remove and add application software. In case of losing an application it is not needed for the user to remove the OS. Most propably you need to re-install the software. Or maybe checking on a different location for the program is sufficient ?

I hope this helps to understand.

paymanm say:

Hey mate,

 

I had the same issue and finally installed it. Im not going to pretend I know how it happened but I will run you through my steps prior to my success.

 

I purchased a copy of Office Home and Student 2013 from a store so I had a CD. I would put the CD in and an autoplay feature popped up, I clicked run and it would say it was getting ready with a percentage. It would then have a window that said "Background Installation ran into a problem" and said something about connect to the internet.

 

I then tried to uninstall office but it would get stuck on "Step 2 of 4" so I used Task Manager to end the task. There was also a process running called "integratedoffice.exe", I ended that process. I went to this page and downloaded a "Fix It" file: http://support.microsoft.com/kb/2739501 

Theres a picture of a man holding a wrench, clicked on that and saved file to the desktop. I ran the program and it took a few minutes but it managed to uninstall Office.

 

Next I performed a "Clean Boot" by following the directions on this page: http://support.microsoft.com/kb/929135

Under the title 'How to perform a Clean Boot'. At the end of the process it prompted me to restart the computer so I did. When Windows started up again, I inserted the CD and copied the files from the CD to a folder on the desktop and took the CD out of the drive. In the folder there was a setup application which I right clicked on and went to 'Run as Administrator'. From there everything was smooth sailing, box came up asking to sign in or use product key, selected product key, installed and everything is working fine.

 

So to conclude;

  1. I uninstalled existing Office software with the use of Fix It;
  2. Performed a Clean Boot;
  3. Restarted the computer;
  4. Pulled files off CD and put onto the computer;
  5. Ran the Setup as Administrator.

Goodluck!
