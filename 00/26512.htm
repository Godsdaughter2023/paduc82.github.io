mike h.. say:

Hi,

 

Attempting this with sendkeys is probably the worst possible way to do it. It can easily be done in a loop with no selection of any cells or resorting to the very unreliable sendkeys method.

 

 

Have a look at the code below. It starts off with the activecell as did your code but then does all the filling in of cells using the OFFSET method. Now I may not have got it exactly right about what you want in the cells but it should be easy enough for you to correct that. It's the bold bit that you may need to change.

 

 

Sub No_Tabbing()
Dim lastrow As Long, x As Long
Dim Col As Long
Col = ActiveCell.Column
lastrow = InputBox("What is the last row number of the entries?", "Stop!")
For x = ActiveCell.Row To lastrow
    With Cells(x, Col)
        .Offset(, 3) = "O"
        .Offset(, 6) = 123
        .Offset(, 8) = Cells(x, Col)
    End With
Next
End Sub

skateboarder873 say: I was able to find the solution from another forum, and in case someone finds this thread in the future, I'll post what worked.  Basically, if I understand it, I needed to add Wait time, and then instead of using ALT-TAB equivalents of SendKeys, I should use AppActivate.  Additionally, I learned that you can use variables with SendKeys.


It was suggested:
"When I've tried similar things in the past I've found adding a slight delay between sent keys can help.

Edit: Or using AppActivate to activate your application window rather than alt-tab!"

So, here's my code that works:


Sub CopyWaitCopy()
Dim lastrow As Integer
lastrow = InputBox("What is the last row number of the entries?", "Stop!")
Dim awsched As String
Dim Amount As String
Dim eRef As String
Start:
eRef = ActiveCell.Value
awsched = ActiveCell.Offset(0, -1).Value
Amount = ActiveCell.Offset(0, 1).Value
AppActivate "Entering The Receipts"
Call AppActivate("Entering The Receipts")
Application.Wait Now + TimeValue("0:00:01")
Application.SendKeys "O", True
Application.Wait Now + TimeValue("0:00:01")
Application.SendKeys "{Tab}", True
Application.Wait Now + TimeValue("0:00:01")
Application.SendKeys "{BS}"
Application.Wait Now + TimeValue("0:00:01")
Application.SendKeys eRef
Application.Wait Now + TimeValue("0:00:01")
Application.SendKeys "{Tab}", True
Application.SendKeys Amount
Application.Wait Now + TimeValue("0:00:01")
Application.SendKeys "{Tab}", True
Application.Wait Now + TimeValue("0:00:01")
Application.SendKeys "~", True
Application.Wait Now + TimeValue("0:00:01")
AppActivate "Microsoft Excel"
Application.Wait Now + TimeValue("0:00:01")
ActiveCell.Offset(1, 0).Select
If ActiveCell.row < lastrow Then
GoTo Start
Else
Exit Sub
End If
End Sub
