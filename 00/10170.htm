Solved. I fixed the group policies by:
1. Windows + R and then type gpedit.msc to open Group Policy Editor. 

2. Navigate via: Computer Configuration > Administrative Templates > Windows Components > Data Collection and Preview Builds
3. Double click on "Allow Telemetry", and change it to Enabled and the value to "3 - Full" 
4. Apply changes.
5. Double click on it again and change it to "Not Configured", and click apply. 
6. Done.

This will work for any other issues regarding Group Policy. It worked for me and will work for other issues regarding group policy.

hansv mvp say:

Thanks!

Let's say the numbers are in A2 and down, and that your time zone is GMT-5 (Eastern Standard Time in the USA).

In B2 (or another cell in the same row), enter the formula

 

=MOD(TIMEVALUE(INT(A2/1000000)&":"&INT(MOD(A2,1000000)/10000)&":"&INT(MOD(A2,10000)/100)+MOD(A2,100)/100)-5/24,1)

 

The bold -5 is the GMT offset.

Format the cell with the custom number format hh:mm:ss.00

This formula can be filled down.

mike h.. say:

Hi,

This finds the last used row so add 1 to the variable to get the empty cell below that

LastRow = Cells.Find(What:="*", After:=[A1], _
SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Row

If this post answers your question, please mark it as the Answer.
Mike H

rick rothstein, mvp - excel say:

Just to point out.... you do not have to specify the "After" argument as the top left corner of the range is the default starting point which means, for the range Cells, A1 will be used by default. However, I do think you should specify the "LookIn" parameter to make sure you do not accidentally skip over (meaning you would overwrite) formulas dispalying the empty string ("")... the Find function uses the last "LookIn" setting whether set by code or via the menu bar. So, I would use this to find the Last used row...

LastUsedRow = Cells.Find(What:="*", SearchOrder:=xlRows, SearchDirection:=xlPrevious, LookIn:=xlFormulas).Row

IF you really want to find the last row with a displayed value (meaning formulas displaying the empty string, if any, would be ignored), then use this...

LastUsedRow = ActiveSheet.Cells.Find(What:="*", SearchOrder:=xlRows, SearchDirection:=xlPrevious, LookIn:=xlValues).Row
