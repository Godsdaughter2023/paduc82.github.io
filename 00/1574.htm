In Column A, there would be a total of 4 dates (offenses) because 09/27/12 and 10/04/12 fall within 365 days of each other and also 10/7/14 and 07/15/15 fall within 365 of each other. The 10/5/13 date does not fit within the 365 day period so it would not get counted.



paulcordts say: I don't think COUNTIF will work alone because of the need to do the subtraction operation


You could also use the DATEDIF Function in column B and C
=DATEDIF(A1,A2,"D") gives you the days between A2 and A1 (A2 must be the later date)
Starting in B2 with this formula and copying down would give the elapsed days since the prior sentence.  

You also have to check days elapsed to the next sentence
C1=DATEDIF(A1,A2,"D") copied down to row 4 (Exclude the last row) would do this

9/27/2012                 7
10/4/2012      7 366
10/5/2013   366  367
10/7/2014   367  281
7/15/2015   281

So you end up with this.  Counting the values in B and C when either (but not both) are less than or equal to 365  and then subtracting the count when both are under would get you to the 4 that is the answer.

9/27/2012                     7
10/4/2012 7 1
10/5/2012 1 2
10/7/2012 2 1011
7/15/2015 1011

The subtraction is needed when the dates are close like in the example above

The count formula would be =COUNTIF(B1:B5,"<=365")+COUNTIF(C1:C5,"<=365")-COUNTIFS(B1:B5,"<=365",C1:C5,"<=365")

e.j. gun say:

You could use a User Defined Function (UDF) to do this also.  I have created one below.  Put the code in a general code Module, and then enter the function (similar to how you would enter any Excel function) into one or more cells to see the result.

Using your data, here are two examples.

Example 1:  Return just the count.

    In Cell B1, enter "=CountDateOffenses(A2:A5,365)" and press ENTER

    (You can change 365 to whatever date interval you want)

    The result should be the number "2", which tells us that there are 2 violating pairs of dates.

Example 2:  Return the count, and the date pairs that are in violation.

    Select Cells B1:D5.  In cell B1, enter the same formula as above.  Now, however, press SHIFT-CTRL-ENTER to make this an ARRAY FORMULA.  The result you see should look like:

2
9/27/2012       10/4/2012       7
10/7/2014       7/15/2015       281
#N/A       #N/A       #N/A
#N/A       #N/A       #N/A

The #N/A means that the function did not find enough violators to fill all the cells you selected.  Note the three columns:  early date, late date, difference between the two.

 

HTH,

Eric

 

'===== BEGIN CODE =====

Option Explicit

Function CountDateOffenses(inRange As Range, delDate As Single) As Variant
    Dim i As Long, j As Long
    Dim callRange As Range
    Dim callRows As Long, callCols As Long
    Dim nDates As Long
    Dim dateArray() As Variant
'
    Dim nPairs As Long
    Dim datePairs() As Variant
    Dim resArray() As Variant
'
    Set callRange = Application.Caller
    callRows = callRange.Rows.Count
    callCols = callRange.Columns.Count
'
    nDates = inRange.Rows.Count
    dateArray = inRange.Value2
'
' Find date pairs that are within "delDate" days of each other and
' save them
'
    For i = 1 To nDates
        For j = i + 1 To nDates
            If (i <> j) Then
                If (Abs(dateArray(i, 1) - dateArray(j, 1)) <= delDate) Then
                    nPairs = nPairs + 1
                    ReDim Preserve datePairs(1 To 3, 1 To nPairs)
                    datePairs(1, nPairs) = Application.WorksheetFunction.Min(dateArray(i, 1), dateArray(j, 1))
                    datePairs(2, nPairs) = Application.WorksheetFunction.Max(dateArray(i, 1), dateArray(j, 1))
                    datePairs(3, nPairs) = datePairs(2, nPairs) - datePairs(1, nPairs)
                End If
            End If
        Next j
    Next i
'
' Transfer results to the output array
'
    ReDim resArray(1 To Application.WorksheetFunction.Min(callRows, nPairs + 1), _
                   1 To Application.WorksheetFunction.Min(callCols, 3))
'
    resArray(1, 1) = nPairs
    If (callCols > 1) Then
        For i = 2 To Application.WorksheetFunction.Min(callCols, 3)
            resArray(1, i) = ""
        Next i
    End If
    If (callRows > 1) Then
        For i = 1 To Application.WorksheetFunction.Min(callRows - 1, nPairs)
            For j = 1 To Application.WorksheetFunction.Min(callCols, 3)
                resArray(i + 1, j) = datePairs(j, i)
            Next j
        Next i
    End If
'
    CountDateOffenses = resArray
'
End Function
'===== END CODE =====
