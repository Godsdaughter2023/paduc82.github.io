  1. From Start, search for msconfig 

  2. Select System Configuration from the search results. 

  3. On the Services tab of the System Configuration dialog box, tap or click to select the Hide all Microsoft services check box, and then tap or click Disable all. 

  4. On the Startup tab of the System Configuration dialog box, tap or click Open Task Manager. 

  5. On the Startup tab in Task Manager, for each startup item, select the item and then click Disable. 

  6. Close Task Manager. 

  7. On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.

hairyfool say:

The Digital Entitlement License means that you no longer need a key to reinstall Windows 10, nor do you have to reinstall qualifying Windows software because the Windows 10 license is an upgrade.

On a clean computer use this tool to download a fresh install of Windows 10. The version hosted will include the feature Creators Update (1703) but not subsequent security and bug updates.

http://go.microsoft.com/fwlink/p/?LinkId=616447

Use the option to create the image for another computer and select the options for the version of license on those other computers (if they are different you will need each one). If they support USB boot (most modern computers do) then create a USB drive as this is fastest option, the same key can be used for multiple computers.

You can use the key to do a repair install which will reset all Windows files and retain user files but given the nature of the problem retaining anything may be suspect, a clean install is the safest  even if the most work. If you go for a clean install skip over the license key entry and once Windows setup is complete it should activate automatically. Make sure you fully update the computer

If a clean install involves a significant amount of work to rebuild the computer ready for the end user then a system image is probably advisable once all updates are applied but before user mods and files installed. How often you do that and where it is stored is dependent on your exact environment.

Remember that the image will not include the next feature update so the release of that may be the trigger to refresh the system image. In a large organisation then you would normally implement a dedicated 3rd party backup utility which would have its own system recovery procedure. In a network ideally no user files would be stored on the local PC but centrally hosted and that host should have its own backup system.
