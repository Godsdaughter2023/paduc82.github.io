Finally figured it out myself (with some internet searching as well):

Bypass Windows 10 Logon Screen at Boot-Up

1. Press WindowsKey / R simultaneously; enter netplwiz; and press Enter

2. If prompted by User Account Control, click Yes

3. Uncheck Users must enter a username and password to use this computer

4. At the Automatically Log On window, enter your name and password; click OK; then X out

5. Restart the computer to see if it automatically logs on without a password logon screen

6. If it logs on automatically, you’re done.

7. If it doesn’t log on automatically:

    - Repeat the above procedure and re-check User must enter a username and password...

    - Repeat procedure again and uncheck User must enter a username and password...

    - Now it should work
