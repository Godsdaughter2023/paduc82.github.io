2001...........2001

2001...........2001

2093...........2001

The first filter would return the 3 records with 2001 in the first column.   The second filter (with 2001 in column 20) would only return the 4th record since the 1st and 3rd records were returned in the first filter.  I'm looking for a total of all records returned since 2001 appears in either column 5 or column 20.

e.j. gun say:

The easiest way to do this is to use a helper column.  Let's say column 26 ("Z") is empty, and we'll use that.  In that column, we want to enter a formula that meets your criteria.  In the first cell after the header (Z2) we enter:

=OR($E2=2001,$T2=2001)

Drag the formula down the length of your data.

For your sample data above, we would get:

TRUE

TRUE

TRUE

TRUE

Now you can filter that Column Z for "TRUE" to get those rows that meet your criteria.

Now, since you want a VBA solution, you would simply use the following code (or something similar):

Sub TestIt()

    Dim nRows as Long

    Dim BrnVar as Long

    BrnVar = 2001

    nRows = ActiveSheet.Cells(1, 5).End(xlDown).Row

    With ActiveSheet

        .Range("$Z$2").Formula = "=OR($E2=" & BrnVar & ",$T2=" & BrnVar & ")"

        .Range("$Z$2").Copy Destination:=.Range("$Z3:$Z" & nRows)

        .AutoFilterMode = False

        .Range("$A1:$Z1").AutoFilter Field:=26, Criteria1:="TRUE"

    End With

End Sub
