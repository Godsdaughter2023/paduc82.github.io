To start with, you would be better off using the SUMIFS() function introduced in Excel 2007. This is a vastly improved function over the SUM(IF(IF())) array formula and should speed up your calculation cycle by as much as 70%. In addition, it will happily skip over #VALUE! errors in the data that are causing your original problem.

=SUMIFS('Data   Import'!$P$10:$P$848107,'Data   Import'!$B$10:$B$848107,$B8,'Data   Import'!$T$10:$T$848107,"ins pymt")

HOWEVER, that doesn't get rid of hte error which may skew your results. Tap F5 and when the GoTo dialog opens, click Special in the lower left. When the GoTo Special dialog opens, choose Formulas and clear all check boxes except Errors. Click OK in the lower right.

mjftn say: It could be that Outlook changed the default mail folders from the Mailbox to the Personal Folder. If you want new emails to go to the Mailbox Inbox (the one iPhone can see), then go to Tools, Account Settings, Data Files tab.
Highlight the Mailbox, then click "Set as Default". You will have to exit outlook and reopen for that change to take effect.

If you just need to move the contacts and Calendar items back to the mailbox:

For Contacts:
You can move the Contacts from the personal folder contacts to the Mailbox Contacts.
Open Outlook, Click on the Folder List view button, at the bottom left of outlook (looks like yellow file folder).
On the folder list, click on the + next to Mailbox and Personal Folders, so you can see the contents of each.
Go to the Contacts folder that has the contacts you wish to keep. Highlight one of the contacts on the right, and then hit Ctrl+A to select them all. While all contacts are highlighted, Right-Click on one of them again and this time choose Move to Folder...
Select the Mailbox Contacts, (location you wish to move the contacts), and then click OK. It should move all of those contacts to the mailbox, which is what the iPhone can see.

For Calendar:
While in the "Folder List" view of Outlook, highlight the Calendar that contains the appointments you need (I assume the Calendar appointments are now in your Personal Folder calendar, and not mailbox).
Click on File, Import and Export
Choose "Export to a file", click Next
Select Personal Folder File (.pst), Click Next
Verify the calendar is selected, click Next
Click Browse to save the file to a location you can get to, such as My Docs or Desktop.
Give the file a name, such as Backup.pst or Calendar.pst - it does not matter.
Click Finish

That will just export all of the appointments to a pst. Then go back to File, Import and Export
Select "import from another program or file", click Next
Select "Personal Folder File (.pst)", click Highlight the Calendar at the top, select "Import items to the same folder in:" and make sure the Mailbox is selected.
Click Finish
