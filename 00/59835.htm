hey there demonicreaper6,

In computer networking, the maximum transmission unit (MTU) of a communications protocol of a layer is the size (in bytes) of the largest protocol data unit that the layer can pass onwards. MTU parameters usually appear in association with a communications interface (NIC, serial port, router etc.). Standards (Ethernet, for example) can fix the size of an MTU; or systems (such as point-to-point serial links) may decide MTU at connect time.

Large packets can occupy a slow link for some time, causing greater delays to following packets and increasing lag and minimum latency. For example, a 1500-byte packet, the largest allowed by Ethernet at the network layer (and hence over most of the Internet), ties up a 14.4k modem for about one second.

you need to either check your router control page on your PC to change the MTU settings or reach out to your ISP to allow larger MTU with your service.

From my experience, the MTU error is thrown whenever the Xbox is out of ideas why a connection didn't properly work, when the console "thinks" everything is OK.

So please try the following, to check if it's really a MTU issue on your Internet providers side.

Open up a command prompt on your computer. Do this by pressing "Windowskey + R" and then typing "cmd" into the small window. Then press OK.

In the black window now type:

ping -f -l 1364 www.xbox.com

(to avoid confusion between a capital i (I) and lowercase L (l), the command "spoken out" is: "ping (space) minus EFF (space) minus ELL (space) one-thousand-three-hundred-sixtyfour (space) www.xbox.com")

Do you get a message about the packet needing to be fragmented, but the don't fragment flag being set? Then you have a MTU problem, and need to contact your Internet provider.

If you get a few rows with responses and return times, then you do NOT have an MTU issue, and the Xbox is clueless ;-)

Please then check the following: With the black window still open, type:

IPCONFIG

and press Enter. Make note of the computer's IP address, and the Default Gateway address.

Now go to your console and go all the way to the right to the settings hub. There go to the console system settings tile in the upper left. There go to the network settings.

In the network settings choose your WLAN or LAN connection if necessary.

Then go right once more to the advanced network settings.

There choose the option to reset the network settings to factory default.

Go back, and if necessary connect to your WLAN with the proper passkey, and check the IP address and default Gateway your Xbox shows you.

Does the default Gateway match EXACTLY the PC's default gateway? Is the IP address ALMOST match with the PC address (i.e. matches the first three parts of the IP address with the PC or default gateway IP)?

If it still does not work, follow the instructions in the last part of this post here, and write down the information that is asked in bold.

3. Connect the appropriate video cable from your computer's output to your TV.

4. Turn on your TV and set it to the input that matches the output on your computer. You might need to refer to the user's guide included with your TV for instructions on how to do this.

5. Turn on your computer.

You should see Windows displayed on your TV. If the screen looks stretched, or part of the screen seems cut off, you probably need to adjust the screen resolution in Windows. If you can't see Windows on your TV, you might need to reconnect the computer to your monitor and adjust the screen resolution before you connect to your TV.

With audio and video files, the files tend to be containers that can contain different formats. So one avi file may not be the same format as another.

 

To See if a Fix is Available

 

If it involves a crash.

 

In Control Panel (and select Classic view in the left hand pane) choose Problem Reports and Solutions (type problem in Start's search box), go to Problem History, right click your error and choose Check For Solution.

 

You may also right click and choose Details for more info. Post those details here. The Fault Module Name is the important information.

 

Get the Error Code

 

When you try to play the file and it can't be played you should get a dialog offering Web Help. This contains the error details. Post the web address it takes you too so we can read it too.

 

These two links list most specific errors

http://support.microsoft.com/kb/886273/en-au

http://msdn.microsoft.com/en-us/library/windows/desktop/dd798067(v=vs.85).aspx

What Codecs Are Installed

In Media Player, press Alt + H then About Windows Media Player, Technical Support Information.

 

 
To See What Codec is being Used

 

While playing the file right click it in the Now Playing pane and choose Properties. On the file tab will list the codecs being used. If Now Playing pane is not showing right click Now Playing on the command bar and tick Show List Pane.

 

Disable Codecs
