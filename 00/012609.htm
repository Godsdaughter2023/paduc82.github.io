daniel g. grau say:
OK, this is what you need to paste as code in the Editor. Jim says copy and paste doesn't work for him, but I constantly copy and paste code from web pages into my Word editor. So try it.

I've tried the following steps on my own system and it works fine: it adds a Bold, Italic, Reset Characters and Insert Comment to three contextual menus (see screenshot). The first two options are toggles (apply if absent, take it off if present). But as Bob J. pointed out in the other thread, these are undocumented patches, so we learn as we go along. Thus, you may  find that they don't always work in some circumstances, given that there are several contextual menus that need to be patched (this macro modifies four of those menus).

The four new options are shown below, at the end of the contextual menu.

              

You must understand that if you don't follow the instructions accurately, it won't work. If your Word is an Italian version, it may give you an error in Step 8 below, but we won't know until you try.

If you have any doubts while following the instructions, stop, exit Word (if asked to save Normal answer "No") and post back here. 

You only need to do the following steps once, then save the Normal template when asked. Here are the instructions:

  1. In your main menu bar, select Tools > Macro > Visual Basic Editor
  2. You should see a Project box
  3. In there, select Normal (Normal.dotm)
  4. In the main menu bar of the Editor, select Insert > Module
  5. A new window will open
  6. Paste the code include below
  7. Click anywhere on the first Subroutine (It starts with "Sub AddToContextualMenu")
  8. In the main menu bar, select Run > Run Sub/User Form (this will add the four new contextual options you requested, at the end)
  9. You can save your Normal.dotm template now (File > Save Normal.dotm) or Word will prompt you when quitting
  10. In the main menu, select Close and return to Microsoft Word (or do Command Q)
 
Should you want to delete the four added options: Do Step 1 above, place your cursor anywhere on "Sub DeleteFromContextualMenu" and then do steps 8-10.

The code appears below this line.
--------------------------------------------------------------------------------


Sub AddToContextualMenu()

'First contextual menu ("Text w/ Thesaurus") appears when there is no selection
'or there is a single word selected.
    With Application.CommandBars("Text w/ Thesaurus").Controls.Add(Type:=msoControlButton)
        .Caption = "Bold Toggle"
        .OnAction = "BoldToggle"
        .BeginGroup = True
    End With

    With Application.CommandBars("Text w/ Thesaurus").Controls.Add(Type:=msoControlButton)
        .Caption = "Italics Toggle"
        .OnAction = "ItalicsToggle"
    End With
        
    With Application.CommandBars("Text w/ Thesaurus").Controls.Add(Type:=msoControlButton)
        .Caption = "Reset Characters"
        .OnAction = "ResetCharacters"
    End With
        
    With Application.CommandBars("Text w/ Thesaurus").Controls.Add(Type:=msoControlButton)
        .Caption = "Insert Comment"
        .OnAction = "InsertComment"
    End With
    
'The second contextual menu ("Text") is used when selecting more than one word
'or when clicking in blank areas.
    With Application.CommandBars("Text").Controls.Add(Type:=msoControlButton)
        .Caption = "Bold Toggle"
        .OnAction = "BoldToggle"
        .BeginGroup = True
    End With
        
    With Application.CommandBars("Text").Controls.Add(Type:=msoControlButton)
        .Caption = "Italics Toggle"
        .OnAction = "ItalicsToggle"
    End With
        
    With Application.CommandBars("Text").Controls.Add(Type:=msoControlButton)
        .Caption = "Reset Characters"
        .OnAction = "ResetCharacters"
    End With
        
    With Application.CommandBars("Text").Controls.Add(Type:=msoControlButton)
        .Caption = "Insert Comment"
        .OnAction = "InsertComment"
    End With
    
'The third contextual menu ("Comment") is used when selecting text that is already part of a comment.
    With Application.CommandBars("Comment").Controls.Add(Type:=msoControlButton)
        .Caption = "Bold Toggle"
        .OnAction = "BoldToggle"
        .BeginGroup = True
    End With
        
    With Application.CommandBars("Comment").Controls.Add(Type:=msoControlButton)
        .Caption = "Italics Toggle"
        .OnAction = "ItalicsToggle"
    End With
        
    With Application.CommandBars("Comment").Controls.Add(Type:=msoControlButton)
        .Caption = "Reset Characters"
        .OnAction = "ResetCharacters"
    End With
        
    With Application.CommandBars("Comment").Controls.Add(Type:=msoControlButton)
        .Caption = "Insert Comment"
        .OnAction = "InsertComment"
    End With
    
'The fourth contextual menu ("Lists") is used when selecting bulleted text.
    With Application.CommandBars("Lists").Controls.Add(Type:=msoControlButton)
        .Caption = "Bold Toggle"
        .OnAction = "BoldToggle"
        .BeginGroup = True
    End With
        
    With Application.CommandBars("Lists").Controls.Add(Type:=msoControlButton)
        .Caption = "Italics Toggle"
        .OnAction = "ItalicsToggle"
    End With
        
    With Application.CommandBars("Lists").Controls.Add(Type:=msoControlButton)
        .Caption = "Reset Characters"
        .OnAction = "ResetCharacters"
    End With
        
    With Application.CommandBars("Lists").Controls.Add(Type:=msoControlButton)
        .Caption = "Insert Comment"
        .OnAction = "InsertComment"
    End With


End Sub
Sub BoldToggle()
    Selection.Font.Bold = Not Selection.Font.Bold
End Sub

Sub ItalicsToggle()
    Selection.Font.Italic = Not Selection.Font.Italic
End Sub

Sub ResetCharacters()
    Selection.Font.Reset
End Sub

Sub InsertComment()
    Selection.Comments.Add Range:=Selection.Range
End Sub

Sub DeleteFromContextualMenu()
'Run this macro to delete the contextual menus
    On Error Resume Next
        Application.CommandBars("Text w/ Thesaurus").Controls("Bold Toggle").Delete
        Application.CommandBars("Text w/ Thesaurus").Controls("Italics Toggle").Delete
        Application.CommandBars("Text w/ Thesaurus").Controls("Reset Characters").Delete
        Application.CommandBars("Text w/ Thesaurus").Controls("Insert Comment").Delete
        
        Application.CommandBars("Text").Controls("Bold Toggle").Delete
        Application.CommandBars("Text").Controls("Italics Toggle").Delete
        Application.CommandBars("Text").Controls("Reset Characters").Delete
        Application.CommandBars("Text").Controls("Insert Comment").Delete
        
        Application.CommandBars("Comment").Controls("Bold Toggle").Delete
        Application.CommandBars("Comment").Controls("Italics Toggle").Delete
        Application.CommandBars("Comment").Controls("Reset Characters").Delete
        Application.CommandBars("Comment").Controls("Insert Comment").Delete
        
        Application.CommandBars("Lists").Controls("Bold Toggle").Delete
        Application.CommandBars("Lists").Controls("Italics Toggle").Delete
        Application.CommandBars("Lists").Controls("Reset Characters").Delete
        Application.CommandBars("Lists").Controls("Insert Comment").Delete
    On Error GoTo 0
End Sub
