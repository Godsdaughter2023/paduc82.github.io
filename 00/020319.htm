pikeyspikey say:

using vba as below which as i say works for the other table that is re-linked.  I set the original link up manually

Function LinkCSV(strDb As String, strDir As String, strName As String) As Boolean

On Error GoTo Error_Handler

Dim strLinkCode As String
Dim strTblName As String
Dim db As DAO.Database
Dim tdfLink As DAO.TableDef

LinkCSV = False

Set db = OpenDatabase(strDb, False, False, "MS Access;PWD=" & CSTR_TRADE_PSWD)

strTblName = Left(strName, Len(strName) - 4)
Set tdfLink = db.CreateTableDef(strTblName)
tdfLink.SourceTableName = strName
strLinkCode = "Text;DATABASE=" & strDir & ";TABLE=" & strName
tdfLink.Connect = strLinkCode
db.TableDefs.Append tdfLink
db.TableDefs.Refresh

Set tdfLink = Nothing
db.Close
Set db = Nothing

LinkCSV = True

Exit Function

andreas killer say:

Open the downloaded template

Right-click the sheet tab and choose "Unhide sheets"

You'll see the sheet "Chart Data"

Click Ok to unihde

In sheet "Chart Data" in cell B5 is a formula
  =MIN(TotalMonthlyExpenses/TotalMonthlyIncome,1)
The arguments are named cells, you can find them in the Name Manager.

These names refers to cells in sheet "Current Month".
