Problem opening Excel files from flash drive - Microsoft Community

suecoit say:

Problem opening Excel files from flash drive

Running Windows 7, Office 2010, when opening Excel files from a flash drive, Excel wants to open a "copy" of the requested file.  You must press ESC to get the requested file to open, every single time.  Does not do this on work computer, just home computer.  What does this mean??
rohn007 say:
I'll second Mike's recommendation.

The reason has to do with technical limitations of Windows and Thumb Drives.  The memory used in flash drives has a number of write-cycles. To minimize wear on the drives Windows saves all write operations to a "cache" file on the local HD. When a full "block" worth of information is in the cache, it gets sent to the flash drive in a single write operation.

If you don't properly "flush" the cache by properly closing the flash drive before disconnecting it you stand the risk of leaving part of the file behind on the HD ... OOPS !  That causes file corruption.  Unfortunately it also appears that for DOCX files, the body text portion of the file structure is at the end of the file. Making it the part most likely to be lost.

By working from a copy on the HD, you get in to a process. Copy files from HD to Flash, then shut down the flash drive.

mike middleton say: Regarding "opening Excel files from flash drive," the usual advice is "Don't."

Instead, first copy the files from the flash drive to your computer, and then use Excel to open the files from your computer.

And, save files from Excel to your computer, and then copy the files from your computer to the flash drive.

-  Mike Middleton, www.TreePlan.com
