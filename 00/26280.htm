The subroutine below looks at the volume added: if not zero it copies average from F to G column, otherwise it does the calculation you specified and put new data in G column

Let me know if it works. I doubt the rest of forum readers are interested, so go to my website to get my email address and we can continue my private email.

See file 1April2012.xlsm at https://skydrive.live.com/?id=162119FF97286948!2988&cid=162119ff97286948&

If new to VBA: see http://msdn.microsoft.com/en-us/library/office/ee814737(v=office.14).aspx

(its about Excel 2010 but is applicable to Excel 2007 also)

Sub tryme()
  mycode = "OK"
  mycount = 0
  mylast = Cells(Cells.Rows.Count, "E").End(xlUp).Row
  For j = 2 To mylast
    If Cells(j, "E") <> 0 And mycode = "OK" Then
     Cells(j, "G") = Cells(j, "F")
    mycount = 0
   End If
  I f Cells(j, "E") = 0 Then
    mycode = "Zero"
   mycount = mycount + 1
   End If
   If Cells(j, "E") > 0 And mycode = "Zero" Then
     myvol = Round(Cells(j, "E") / (mycount + 1), 0)
    Cells(j, "G") = myvol
    Do While mycount > 0
       Cells(j - mycount, "G") = myvol
       mycount = mycount - 1
   Loop
   mycode = "OK"
   mycount = 0
  End If
  Next j
End Sub
