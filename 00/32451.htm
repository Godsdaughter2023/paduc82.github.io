andre da costa say:

Either will do, its up to you which edition/license distribution is suitable. The OEM license would avoid the activation issues required of the Home Premium Upgrade license.

OEM licenses are much cheaper and are identical to full version licenses:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system  

 In order to install the upgrade version of Windows 7, you must have a qualifying Windows operating system installed and activated.  You cannot install an upgrade version of Windows 7 on a blank hard drive.  The installation procedure does not ask you to insert a Windows disc in the drive for verification, the actual qualifying operating system must be installed.

    If you do not have a qualifying Windows operating system installed with a genuine license activated, then you cannot use the upgrade version of Windows 7 - you would need a "full version" Windows 7 license.

    In summary:

       1. A qualifying Windows operating system must be installed.
       2. The qualifying Windows operating system must have a genuine license (product key) and it must be activated.
       3. To upgrade, boot to the qualifying Windows desktop, insert the Windows 7 Upgrade disc in the DVD drive.
       4. When the setup menu appears, select Custom (advanced) to initiate a clean install procedure.

Workaround:

Boot from the Windows 7 DVD
Click Install Now
Accept License Agreement
When the option is displayed to select an installation type, click (Custom Advanced)
Click Drive Options
Select the drive
Click Format and click Next to proceed with the installation

Skip entering your Windows 7 product key and complete the installation.

Once you reach the desktop, click Start, right click Computer

Click Properties

Scroll down to Windows Activation

Click the link to change your product key

Enter your product key and use Phone Activation to activate your license. Remember to explain to the call agent your situation.

Problem Type1: Feed Synchronization Issue
First and fore most verify if the feed synchronization is corrupted. Here is how you do it.

• Click on Start; then on All Programs.

• Click on Accessories. Right click on Command Prompt and click on “Run as administrator”.

• Type in the following command
schtasks /query | findstr /i "user_feed

If you see error in the output then your feed synchronization is broken.

Errors look something similar to
ERROR: The task image is corrupt or has been tampered with.
ERROR: Task cannot be loaded: User_Feed_Synchronization-{.

To fix the Feed synchronization issue, run the following commands at the command prompt.

msfeedssync disable
msfeedssync enable

This should take care of the feed sync problem.

Problem Type2: Corrupted RSS feed database

If you do not see any errors in the Feed Synchronization then the problem might be with the Feed Database.  To fix this issue. lets start with exporting all out feeds so that we don’t have re-subscribe again.

• Open Internet Ezplorer, and click Alt key to get to the menu bar.

• Go to the File menu and click “Import and Export .”

• Select Export to a file and click next

• Select feeds and click next.

• Export the OPML file to some place where you remember.

Now lets delete the feed databases. Close all Internet Explorer windows.

• Browse to the following folder C:\Users\YourUserName\AppData\Local\Microsoft\Feeds (Do not forget to replace YourUserName with your actual user name).
• Select all the contents in the folder and delete them.

Now, lets import our saved feeds. Open Internet Explorer and go to File menu (Click Alt button to get the menu bar)

• Select Import And Export. and click next

• Select Import from a file and click next

• Check the feeds box and click next

• Select the previously saved OPML file and click next.

You should be all done, and hopefully your feeds are now working as expected.

Important information about your email service

Many users of Windows Live Mail 2012 have received an email under this heading from the Outlook Team advising them of the changes that are taking place at Outlook.com as accounts are migrated to the same platform as that used by Office 365. Apart from advice to switch to using the built-in Mail apps on Windows 8.1 and 10, users are instructed to apply the patch KB3093594. A few points to note:

  1. The change means that migrated accounts can no longer be synchronized using the old DeltaSync protocol. To synchronize these accounts, users can set up the account again to use IMAP. This will enable mail messages and folders to synchronize, but not the calendar and contacts.
     
  2. The patch enables support for Exchange ActiveSync (EAS) to be used instead of DeltaSync in Windows Live Mail 2012 on Windows 7, 8 and 8.1 (see System requirements on the download page).
     
  3. Mail synchronization errors can often be cleared by removing the account, closing down the program and waiting a few minutes, then re-launching it and adding the account back again.
     
  4. We don't know what is going to happen with Windows Live Mail 2009 on XP and 2011 on Vista. It is clear that users will no longer be able to sign in with a migrated account to synchronize contacts and calendar, and they won't be able to synchronize mail messages and folders using DeltaSync. Whether the programs will continue to work in other respects (e.g. as a POP3, IMAP and NNTP client) remains to be seen.
     
  5. Apart from this question of synchronizing mail, calendars and contacts with a Microsoft Email account, there is no change now or in the foreseeable future for Windows Live Mail 2012. Mail accounts configured to use POP3 or IMAP will continue to work as before, the default calendar and contacts lists will still be available and the other functions - RSS feeds and newsgroups - continue as before.

Windows 10 - how to make the command prompt text easier to read - Microsoft Community

j w stuart say:

Windows 10 - how to make the command prompt text easier to read

Technical Level : Basic

Summary

If you use the command prompt to perform certain task and find the text is of not very high quality and may be hard to read on your display then here is how you can make it easier to read.

Windows 7’s command prompt was never a problem on any of my displays but Windows 10 is. The reason why is the default font has changed in Windows 10.


Details

As you can see from the top (Windows7) to bottom (Windows 10) screen shots, the text in Windows 10 is much less clear.

To find out the default font style and size in Windows 7 right click and select ‘Properties’.

Next, click on the ‘Font’ tab. You will see the font is “Raster Fonts” and the size is 8 x 12

In Windows 10 the font is “Consolas” and the ‘Size’ = 12

To improve readability, change the Windows 10 font to ‘Raster’ and the change the ‘Size’ to 8x 12

Note that you may find that the font size that is best for you is dependent on the size of your display and the screen resolution settings.

Screen Buffer Size:

a) The ‘Width’ allows you to configure how many characters are displayed in a single line (before the text is wrapped to the next line) in the Command Prompt window.

b) The ‘Height’ value determines the number of lines that are stored in memory.

Window Size:

You can change the size of the Command Prompt window by changing either the ‘Width’ or ‘Height’ from the default ( 80 x 25 ) to for example: 80 x 40  

Window Position:

Changes the position of the Command Prompt window location referenced from the left and top edges of the screen. You need to ‘uncheck’ the "Let System Position Window" box to make any changes.

Let System Position Window - if it's checked, you can still use the mouse to resize the Command Prompt window.

Make the text a little brighter:

The default values for the RGB (Red, Green, Blue) intensity = 192. To increase the brightness by about 15% change these values from 192 to 220. You can see the change in real time inside the inset box.

Note: The ‘Command Prompt’ and ‘Command Prompt’ (Admin) are independent of each other and you will need to change the settings for each separately or you will end up with this for a standard non-admin command prompt.
