travelor say:

Hello,

 

There are only a couple of reason why the Zune Software will "auto run" -

1. You've connected a Zune/WP7 device through USB cable -

If you have your WP7/Zune device connected and exited the Zune Software, it shouldn't load, unless the cable or usb ports are bad and the computer recognizes the disconnect/re-connect and thus starts the Zune Software

2. You've setup the "Wireless Sync" option

If you've setup the Wireless Sync option, connected to the network selected to sync on, and you're Zune/WP7 device is connected to an external power source, the Zune Software will auto start and sync wirelessly. Only way to stop this is to end the wireless sync option.

Trick:

Open the System Configuration (MSCONFIG) and go to Startup (tab) - Uncheck the "Zune" startup item - Reboot your computer. This should stop the Zune Software from auto running when you plug in a Zune Device and/or WP7 device.
