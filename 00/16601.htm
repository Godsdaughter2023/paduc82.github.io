Can someone please help with the code?

hansv mvp say:

Try this code:

    Dim r As Long
    r = Range("A:C").Find(What:="*", SearchOrder:=xlByRows, _
        SearchDirection:=xlPrevious).Row
    ActiveSheet.PageSetup.PrintArea = "$A$1:$C$" & r

hansv mvp say:

When searching, * is a so-called wildcard meaning "anything".

r = Range("A:C").Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row

looks at columns A:C and looks for a cell containing anything (i.e. a non-empty cell). Since we specify SearchDirection:=xlPrevious, it starts from the bottom of the columns and works its way up.

SearchOrder:=xlByRows tells Excel to search row by row. The search stops when a non-empty cell is found, and the variable r is set to the row number of this cell.

In other words, r is the last used row in columns A:C.

ActiveSheet.PageSetup.PrintArea = "$A$1:$C$" & r

sets the print area to the range from A1 to the cell in column C and the last used row.

As Doug suggests, the likelihood is that you have a plain-text number rather than a PAGE field. The fact that you have 1,2,2,... suggests that you have "Different First Page" enabled and have either a PAGE field or the number 1 in the First Page Header. So open the primary Header (from the first or any subsequent page), select the 2, and press Alt+Shift+P to replace it with a PAGE field.
