qa guy commelec say:

You can use a Ranking query like this --

SELECT YourTable.Area, YourTable.Name, YourTable.Sales, (SELECT Count(*) FROM YourTable AS [XX] WHERE [XX].Sales <= YourTable.Sales )+1 AS Rank
FROM YourTable
ORDER BY YourTable.Sales;

Ties will have the same (lowest) number.  If you use a unique (autonumber) field for rank there will be no duplicates.
