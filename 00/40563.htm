If you had XP Mode, it is gone and is blocked from running on Windows 10.  Even XP Mode vms converted by VMWare and other virtualization programs are blocked.  User created XP vms also will not run if in Microsoft Virtual PC format because Virtual PC does not run on Windows 10.  The only Microsoft virtualization software that runs on Windows 10 is Hyper-V.  Hyper-V is only available on Windows 10 Pro.  XP is not supported in Windows Hyper-V (no integration tools).  It runs, but it is pretty useless without the integration tools.

The most important thing is to match the aspect ratio (16 units wide and 9 units high) if you want to avoid black bars.

Width = 1920 = 16 units x 120 pixels/unit
Height = 1080 = 9 units x 120 pixels/unit

So in the example above, you could crop to 16 inches by 9 inches where your pixels/inch was 120 ppi.
