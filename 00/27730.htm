doug robbins - mvp office apps & services (word) say:

The problem could actually be caused by the printer failing to feed the envelope properly with the result that it is not as far into the printer as Word is expecting when it starts depositing the  text.

However, to change the position of the Return Address, add an Envelope to a document and then put the selection in the return address area and then press CTRL+SHIFT+s to display the Apply Styles dialog (which should show the Envelope Return as the style in use) and you click on the Modify button in that dialog

in the Modify Style dialog that appears, click on Format and then on Frame

and then in the Frame dialog, make an appropriate change to the Horizontal position of the Frame.

After clicking OK in that dialog, and before clicking OK back in the Modify Style dialog, select the radio button for "New documents based on this template" so that the changes that you made will be retained.
