ariel aba say:

Hi Dan, 

The reason as to why that laptop is not connecting while it's outside of the network is possible due to how the account was set up. In other words, the administrator who created the user possibly configured it so that the user won't be able to login outside of the specified domain. If you have access to the registry, the key that allows you to login to a different network is contained in HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\Current Version\Winlogon\. The value should be anything asides from 0, which is the number of log ins allowed outside of the domain. Check more information on this link. Another might be due to the Group Policy. You can change it by going to Computer Configuration\Windows Settings\Security Settings\Local Policies\Security Options\Interactive logon. Again the value should be anything but 0. If you do not have access to the registry or Group Policy, it would be best to talk with your institution's admin.
