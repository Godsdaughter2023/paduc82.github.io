andreas killer say:

If you have a header you can sort the columns by this value. Execute the sub Init in an empty sheet, have a look at the values and run sub SortColumns afterwards.

 

Note: It doesn't matter for sub SortColumns if a header exists or not. All existing headers are sorted to the left side in the given order.

 

Andreas.

 

Sub Init()
  'Setup up a test scenario
  Dim Headers(), Header
  Dim P As Long
  Cells.ClearContents
  Headers = Array("A", "B", "C", "D", "E", "F")
  For Each Header In Headers
    Do
      P = Int((UBound(Headers) + 1) * Rnd) + 1
    Loop Until IsEmpty(Cells(1, P))
    Cells(1, P) = Header
  Next
End Sub

Sub SortColumns()
  Dim Headers(), Header
  Dim R As Range
  Dim P As Long
  Headers = Array("F", "D", "E", "C", "B", "A")
 
  For Each Header In Headers
    'Search the header in row 1
    Set R = Rows(1).Find(Header, LookIn:=xlValues, LookAt:=xlWhole)
    'Found?
    If Not R Is Nothing Then
      'Calculate the next position
      P = P + 1
      'Move the column if necessary
      If R.Column <> P Then
        R.EntireColumn.Cut
        Columns(P).Insert
      End If
    End If
  Next
End Sub

tasos_k say:

Hi,

 

Assuming that names are in column A (in all sheet tabs)

 

1) add a new sheet named: 'myData'

2) in cell A1 (myData sheet), add a drop down list/menu

with desired names (make sure that are correct..)

 

3) right click on myData tab, select View Code and paste in the following:

 

 

Private Sub Worksheet_Change(ByVal Target As Range)
If Not Intersect(Target, Range("A1")) Is Nothing Then
Const myCol As String = "A" '<<< names in column A, change
Const mySht As String = "myData" '<<< expected results in sht 'myData', change
Dim vSh As Variant
vSh = Array("Sheet1", "Sheet2", "Sheet3", "Sheet4") '<<< sheet names, change

Dim r As Long, j As Long, t As Long, X As Long
Application.ScreenUpdating = False
If Sheets(mySht).Range("A1") = "" Then Exit Sub
r = Sheets(mySht).Cells(Rows.Count, "A").End(xlUp).Row
If r <=2 Then r = 3
Sheets("myData").Rows("3:" & r).ClearContents
t = 3
For j = 0 To UBound(vSh)
r = Sheets(vSh(j)).Cells(Rows.Count, myCol).End(xlUp).Row
For X = 1 To r
If Sheets(vSh(j)).Cells(X, myCol) = Sheets(mySht).Range("A1") Then
Sheets(vSh(j)).Rows(X).Copy
Sheets("myData").Cells(t, "A").PasteSpecial xlPasteValues
t = t + 1
End If
Next
Application.CutCopyMode = False
Next
Application.ScreenUpdating = True
End If
End Sub

jay freedman say:
When you say "shifting from Home back to File", do you mean that it shows you the Backstage view (with Save, Save As, Open, etc. down the left side)? If not -- if you continue to see the ribbon, but maybe not the buttons on the Home tab -- then it is NOT shifting to File.

By design, there is no way to keep the Home tab selected at all times. If you insert any of a number of "special" items (a table, a picture, a chart, an equation, and so forth), the ribbon automatically switches to a tab containing tools for working with that item. If you then click in "regular" text, that tab disappears and the ribbon goes back to the Home tab. But Word never switches to the File tab by itself; you have to click it to go there.
