Okay, you have data in Sheet2. This has numbers in both columns C and D which is associated with names in Sheet2's column B. There are some duplicate records which may hold blank amounts and you want the first non-blank amount retrieved from Sheet2's columns C and D into Sheet1's columns E and F based upon a cross-reference on the names from Sheet2's column B to Sheet1's column A.

You haven't mentioned where the data actually starts and ends, but I'm going to assume that you have column heading labels so the data starts in row 2 and doesn't extend beyond row 999. =IFERROR(INDEX(Sheet2!C$2:C$999,MATCH($A2,IF(LEN(Sheet2!C$2:C$999),Sheet2!$B$2:$B$999),0)),"")
... which requires Ctrl+Shift+Enter↵ rather than just Enter↵. Once entered correctly, copy it to Sheet1's F2 and then use fill down on E2:F2 as far as necessary.

brian tillman [outlook mvp 2007-2019] say:

So, as I said, they're two separate accounts.  Therefore, logging into the Hotmail account doesn't "send you to the Outlook.com account".  Hotmail IS Outlook.com now.  So, logging in with the Hotmail.com address logs you into that account's mailbox, but with the Outlook.com user interface for that mailbox.  Logging in with the Outlook.com address logs you into that account's mailbox, also with the Outlook.com user interface.  You're still seeing completely separate mailboxes.  You're not being "sent" anywhere.
