hansv mvp say:

If you want Selection to be selected by default in the Print dialog, you can create the following macro in your Normal.dotm template:

 

Sub PrintSelection()
    With Application.Dialogs(wdDialogFilePrint)
        .Range = wdPrintSelection
        .Show
    End With
End Sub

 

and assign it to a Quick Access Toolbar button and/or keyboard shortcut.

 

Or if you want to print directly to the default printer:

 

Sub PrintSelection()
    ActiveDocument.PrintOut Range:=wdPrintSelection
End Sub

tasos_k say:

Hi,

assuming that you want to copy from Sheet1 into Sheet2 the rows 3, 5,10

 

Select the cells A3, A5, A10

(hold down Ctrl key and select the desired cells)

 

and run this code...

 

Sub Selection_CopyRows()
Dim ws1 As Worksheet, ws2 As Worksheet
Set ws1 = Sheets("Sheet1")   '<< Source sht, change name
Set ws2 = Sheets("Sheet2")   '<< Target sht, change name
Dim r As Long
r = ws2.Cells(Rows.Count, "A").End(xlUp).Row + 1
For Each rr In Selection
Rows(rr.Row).Copy ws2.Cells(r, "A")
r = r + 1
Next
End Sub
