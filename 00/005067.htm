myall_blues say:

Hi

The following bit of (rough) VBA code creates a new sheet in your workbook, then loops through all the other sheets copying from D7 to the end and consecutively placing it on the new sheet from D7 onwards.

Note: Doesn't contain niceties such as checking to see if the sheet being added already exists etc.

Option Explicit
Sub test()
    Dim lastRow As Long, prevRow As Long
    Dim csheetName As String
    Dim WS_Count As Integer
    Dim I As Integer
    '
    ' Create and name a new sheet
    '
    Sheets.Add
    ActiveSheet.Name = "VeryUnlikelyExistingName"
    prevRow = 7
    '
    ' Set WS_Count equal to the number of worksheets in the active workbook
    '
    WS_Count = ActiveWorkbook.Worksheets.Count
    '
    ' Loop through each sheet
    '
    For I = 1 To WS_Count
        If Not ActiveWorkbook.Worksheets(I).Name = "VeryUnlikelyExistingName" Then
            csheetName = ActiveWorkbook.Worksheets(I).Name
            lastRow = Sheets(csheetName).Range("D" & Rows.Count).End(xlUp).Row
            Sheets("VeryUnlikelyExistingName").Range("D" & prevRow & ":D" & lastRow - 7 + prevRow).Value _
                   = Sheets(csheetName).Range("D7:D" & lastRow).Value
            prevRow = lastRow + 1
        End If
    Next I
End Sub

bhavani raoreddi say:

How do I transfer my Office 2013 licence from my old Asus tablet to a new Dell Laptop.

My old Asus tablet screen was broken. I have product key of Office 2013 that was loaded into it. Can I download a Office 2013 into my Dell laptop and activate it with the product key of Office 2013 ?

rohn007 say:

Maybe.

It depends on exactly how you bought Office 2013.  If you bought an "OEM" license, factory installed AND activated then sorry you cannot transfer to the new machine.

More likely you do not have an OEM license. MS "improved" the Office license terms in Office 2013 so that retail licenses could only be run on one machine at a time. Normally you would have to uninstal from the old machine. But since the old machine is "dead" you can skip that step.

You need the 25 character Product key and the appropriate installation file to install on the new machine.  If you already have both go ahead and run the install file on the new machine. Submit the product key when prompted.  If the activation fails with something like a "too many activations" message you will have to switch to a phone activation.  Unfortunately MS has "improved" phone activation so it does not work for many people. Here is an alternate way to get the required phone number.

"Telephone activation is no longer supported for your product" error when activating Office

Hide zero values in Excel 2010 column chart - Microsoft Community

jeff0614 say:

Hide zero values in Excel 2010 column chart

How can I hide the zero values that appear flat and level with the floor of a 3-D column chart prepared in Excel 2010?  I was able to hide the zeros returned by formulas in the source data by applying a custom number format or by changing the zero value display options on File, Options, Advanced, but the zeros still display in the column chart.

luke moraga say:

Assuming their series labels (and not the 0's on the axis), you should be able to select the data labels, right-click and select 'Format data label'. Go to the Number section, and apply a custom format of

#,##0;;

Make sure you hit the Add button, then click Ok. That will suppress the 0 value in the chart.
