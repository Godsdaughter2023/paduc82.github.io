Hello 13057532768
When Windows 10 goes to download a new version of Windows, the system itself analyzes your hard drive to verify that it has enough space and, if not, it shows us a window indicating that you do not have enough space.

In order to proceed with the update of the operating system it is necessary to fulfill two points: the first one is to have enough space in C to install the new version, an essential step to continue, while the second one is optional, and we can choose between free up at least 10 GB more space so that the system can process the update without problems and make the backup or connect a USB memory and delegate this task to that memory.

If we have a memory on hand and connect it, there is no problem, when selecting Windows it will be used to carry out the update. However, if we do not want to use external storage, we will have to free more space on the disk, and what better way to do it than from the Windows 10 space healer, which will allow us to erase all temporary files from our system.

Once Windows has enough space to update, it will automatically recognize it and tell us that everything is correct and we can continue. If Windows does not automatically recognize the released space, we must click on the icon that appears next to the release button to update the status of the hard disk and, then, should appear as correct with its green mark.

The Windows Update wizard can now continue with its update work. This will automatically continue to perform the additional checks and configurations to begin the installation and ensure that it is done normally.

Now we just have to wait and follow the steps that will appear on the screen, if any, to continue with the process of updating our Windows 10. Once the update process is complete, we will have the latest version of Windows installed. in our system.
