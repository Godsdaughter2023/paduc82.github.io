Where documents are stored in Windows 7 - Microsoft Community

jyaqui say:

Where documents are stored in Windows 7

Help me understand where documents should go. I have libraries which has documents, but then my real documents are in Users\Owners or Users\public. Can I change the Owners to my name and what should not go into Public?

a. user say:

A couple of issues here.

Presumably then you are using the User account that was used by the Vendor to install Windows? Presumably also this account is an Administrator account?

You should a) create a new User account in YOUR name because having only ONE account on the machine (the built-in Administrator account is disabled by default on Windows 7) lays you open to a nasty scenario where you cannot access your machine because your account is corrupted and b) set it as a Standard user with a password. You should also add a password to the the Owner account and leave it there for elevation purposes and also for emergency access if your normal user account gets corrupted.

As to Documents - the Library is just a collection of Shortcuts. The main storage for data is \Users\{Your Account name}\Documents.

Only use the Public folder if you wish to share documents with others.
