A simple way to do it is to start with an Excel Workbook.

In Cell A2, insert the formula =Today()

In Cell A3, insert the formula = A2+1

Then copy that formula down as many rows as required.

Then save this workbook and then in Word, with a new document on the screen, go to the Mailings tab of the ribbon and click on the Start Mail Merge dropdown in the Start Mail Merge section of the ribbon and select "Letters". 

Then, click on Select Recipients and then on Use Existing List and then navigate to, and select the Excel  Workbook that you created, selecting Sheet 1 as the source of the data.

Then, in the Word document, with the selection where you want the date to appear, using the Insert Merge Field dropdown, select the Date field and insert it into the document.  Add any other text that you want to appear on each page, and save this document, which will be your Mail Merge Main Document.  Then, with that document as the active document, in the Finish section of the Mailings tab of the ribbon, click on Finish & Merge and select Edit Individual Documents.

When the process has finished running, you will have a document that contains a page for each day with the date where you inserted the merge field.

dan parish say:

You can add a new shortcut key to anything listed in the menu bar using Mac OSX's built-in shortcut functionality.  It actually works for all apps not just Office, and works like this:

Let's say you want to add a keyboard shortcut for Trace Precedents in Excel 2016 for Mac.  To add this keyboard shortcut simply do the following:

  1. Open System Preferences
  2. Select Keyboard
  3. Select the Shortcuts tab
  4. Click App Shortcuts on the left
  5. Click the + button to add a new one
  6. For "Application" choose "Microsoft Excel"
  7. For "Menu Title" enter "Trace Precedents"
  8. For "Keyboard Shortcut" press CTRL+Q
  9. Click Add

That's it.  CTRL+Q will now launch Trace Precedents in Excel 2016 for Mac!  If you aren't sure what the menu title for a particular command is, you can simply click Help in that app and search for what you want, which will then show you the exact menu name.

To get a keyboard shortcut to the "Paste Special..." menu control dialog do the following:

From your Dock open System Preferences > Keyboard > Shortcuts and you should be at a dialog that looks like this.

Now click the "+" button to add a new control and a dialog like this opens.

Pull down the Application menu and choose Microsoft Word.app. Type "Paste Special..." into the Menu Title box and then with your cursor insertion point flashing in the Shortcut box press the keyboard sequence you want to assign. For example hold down the Ctrl and Option key and then press P. 

Finally, click Add then fully quit and restart Word. Your shortcut should now be active.

This shortcut assignment method only works with "Named" controls on ribbon tab menus. It does not work with named commands that do not appear on the ribbon. For these you must use Word's keyboard shortcut assignment, which is accessible from the Tools menu of Word 2016..
