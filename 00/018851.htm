If you are looking for a specific birth date, the Jun/Jul 2013 marker point isn't much to go on. You need a specific date from which to calculate backwards from. Let's call it July 1, 2013.

With 01-Jul-2013 in E1 the formula in C2 is,

=DATE(YEAR($E$1),MONTH($E$1)-A2,DAY($E$1))

D2 uses,

=INT(($E$1-C2)/365.25)

julie fka profjulie say: The active contacts file is the PIM.vol file that is found in main memory.  There is no way to move this file to the storage card.  Of course, you can always backup the data or copy the PIM.vol file to the storage card from time to time for safe keeping.  But any PIM.vol file that is on the storage card will not be updated with new data, so you'd have to copy the file over to the storage card regularly.
