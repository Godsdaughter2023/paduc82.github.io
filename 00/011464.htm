ossiemac say:

Depends on exactly what you want the user to be able to do.

Allow formatting of cells is turned on by checking the box when protecting a worksheet. However, this allows formatting of both locked and unlocked cells if user is allowed to select the locked cells.

If you don't want the user to be able to change locked cells then is there any need for them to be able to select locked cells? If not, when protecting the worksheet, uncheck select locked cells and check the box against Format cells and then the user can format the unlocked cells but cannot select the locked cells so they can't format the locked cells.

Record the code while setting the protection to get the required code and syntax. You can add the parameter wsh.Protect UserInterfaceOnly:=True to the end of the list of parameters that is recorded.
