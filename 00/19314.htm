suzanne s. barnhill say:

As Doug says, accessing Page Setup has changed a little, but in any version of Word, the easiest way to access that dialog is to double-click at the top of the horizontal ruler.

Instead of inserting a Continuous section break, you should just convert the Next Page break. With the insertion point in the last section (following the Next Page break), go to the Page Setup dialog and select the Layout tab. In the "Section start" dropdown, change the type to Continuous.

If the shape is truly an image, it would behave exactly as you describe.  An image contains no text as such; conceptually, it's just a grid of colored dots.  The dots may be arranged to LOOK like text, but it's not text, so can't be edited as though it were text.

Considering that it's a template, its author might have dropped a "dummy" image into an image placeholder (which is exactly what the existing text suggests), to give you an idea of what the slide will look like when you add your own images.

Word always uses the Envelope Return style to format the whole return address. You can change this for one envelope by selecting text in the return-address subwindow, right-clicking it, and clicking Font. Function OldestEPA(Branch As String) As Integer

    Dim TempEPA As Integer

    RowCount = 0

        If TempEPA < Worksheets(1).Range("M2").Offset(RowCount, 0) _
            And Worksheets(1).Range("J2").Offset(RowCount, 0).Value = Branch Then
           
            TempEPA = Worksheets(1).Range("M2").Offset(RowCount, 0)
        End If
        RowCount = RowCount + 1
    Loop
   
    OldestEPA = TempEPA
   
End Function
