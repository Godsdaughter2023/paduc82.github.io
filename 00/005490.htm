You can right-click on the OneDrive folder in the File Explorer, choose Properties, under General tab, see the difference if you look at the Size and Size on disk.

Files you save to OneDrive are available online at cloud location (such as personal OneDrive, OneDrive for business, SharePoint online) and offline on your PC. Hi Ni.Pe,

For the “attachment file”, you can consider using lookup column to see if you can fulfil your requirement. It can add another file to a file as attachment by using a library column.

To create a lookup column, please go to the library > LIBRARY in the ribbon > Create Column > Lookup, select the destination file’s library in the Get information from section > OK. Then click Edit Properties in the ribbon to add an attach file to this document.

For more information about lookup column, please refer to the following articles:

•    Add or change a lookup column

•    Create list relationships by using unique and lookup columns

doug robbins - mvp office apps & services (word) say:

It would seem that somehow the «Next Record» field, that should be before the first mergefield on all of the labels EXCEPT for the first, has been deleted.

You can either insert that field into each of the labels where it is required by selecting it from the Rules dropdown.  Alternatively, if you create a new label type mail merge main document selecting the Avery 8395 product, the «Next Record» fields should be present. 

Note, use the Update Labels button to replicate the set up of the merge fields from the first label to the other labels on the sheet.  If you manually copy and paste, you will delete the «Next Record» fields.

cristhianescobarpq say: Hello emon_lewis

1 Activate the Bluetooth audio device and make it detectable.

The way to make it detectable depends on the device. Check the device's help or visit the manufacturer's website to find out.

2 Activate Bluetooth on your PC if it is not already activated. To do this, select the Start button and then choose Settings> Devices> Bluetooth and other devices and activate Bluetooth.

You can also use the activity center to activate Bluetooth. To do this, in the taskbar, select activity center> Bluetooth.

3 In the activity center, select Connect and then the device.

4 Follow other instructions that may appear. If nothing else appears, everything will be ready and connected.

Normally, the Bluetooth device and your PC will connect automatically whenever both are within range with Bluetooth activated.
