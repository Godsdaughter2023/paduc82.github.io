To restore the View menu (assuming you have no menu customizations you don't want to lose):

  1. Right-Click a button on either the Standard or the Formatting Toolbar
  2. Select Customize Toolbars & Menus
  3. On the Toolbars & Menus page click the name of the Worksheet Menu Bar

There is no Ribbon in the Office 2008 apps -- The Ribbon UI was introduced in Office 2011. Also, there is no "formula toolbar" in Excel 2008.

From what you described above, I understand your existing AD is being set up Hybrid with Office 365 tenant. Meanwhile, a new corp AD forest needs to be incorporated to this Office 365 tenant as well.

If you want the AD forest (@abc.com) also to be Hybrid with the existing Office 365 tenant (@xyz.com), this is the Hybrid deployments with multiple Active Directory forests scenario. Raised lower case c with two dots under it - Microsoft Community

danielmclalin say:

Raised lower case c with two dots under it

How do I raise a lower case c with two dots under it. That is how my anestors signed their name: Mc Lalin
jay freedman say:

Because that isn't a standard character in any font I know about, you'll do best to build it, format it, and save it as an AutoCorrect entry. Here are the steps:

  • In whatever font you use the most (such as the default font of your Normal style in the Normal.dotm template), type McLalin with an ordinary letter c. Position the cursor between the c and the L.
  • Click Insert > down arrow on the Symbol button > More Symbols.
  • In the Symbol dialog, make sure the dropdown in the bottom right is set to Unicode and the dropdown in the top left is set to (normal text). In the dropdown in the top right, select Combining Diacritical Marks.
  • In the Character Code box at the bottom, type the number 0324. That's the Unicode value of the character "Combining Diaeresis Below", which is the two dots you want.
  • Click the Insert button and then click Cancel to close the dialog.
  • In the document, you now have a letter c with two dots below, but the dots are below the baseline of the text. To raise it, select the letter and press Ctrl+D to open the Font dialog. Click the Advanced tab of the dialog, click in the "By" box to the right of the "Position" box, and enter the value 2 or 3 (you may have to experiment to see how much looks best). Click OK.
  • You can save your entire last name as an AutoCorrect entry. Select the whole word. Go to File > Options > Proofing and click the AutoCorrect Options button. In the dialog, click the option button for "Formatted text" so the properly formatted name appears in the box under "With". In the box under "Replace", type a short trigger that, when typed, will insert the formatted name. I suggest \mc as something that's easy to remember but won't be typed by accident. Click the Add button and then OK.
  • Whenever you want to include the formatted name in a document, just type the trigger followed by a space or a punctuation character.
