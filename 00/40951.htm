git design say:

I'm sure they will fix it from the inside eventually, but what I do for any updates that hang is this:

  1. Restart Windows to stop the update
  2. Go to the Microsoft®Update Catalog in Internet Explorer
  3. Search for the needed updates and add them to the basket (in this case: "KB3189866" and add "Cumulative Update for Windows 10 Version 1607 for x64-based Systems (KB3189866)" to the basket)
  4. Click the "view basket" and then click "Download" button
  5. Run the install files once they finish downloading (this may end in requiring a restart)

In this case, once I updated that one manually I ran the update tool again and Windows 10 identified that this update was installed and moved on to the other two updates waiting in queue. Sorry it's not exactly a fix, but more of a work around. Some people prefer to only update windows manually though, so to each his own.

a. Press Windows key + R to open Run.

b. If you are prompted for an administrator password or for confirmation, type the password, or click Continue.

c. In the Open box, type %systemdrive%\programdata\microsoft\ehome, and then click OK.

d. In the details pane, right-click mediacenterdatastore.db, and then click Delete.

e. Click yes to confirm that you want to move the Mediacenterdatastore.db file to the Recycle Bin.

f. Restart the computer for the changes to take effect after executing the steps.

g. Start Windows Media Center/Media Player.

 

 

Method 2: Perform a System File Checker (SFC) scan.

 

a) Move the mouse pointer to bottom left of the screen where you see preview of start menu and Right-click to bring up a list of Windows settings.


b) Click on Command Prompt (Admin) to bring up elevated command prompt.

Note: Click ‘Yes’ if it prompts for permission.


c) In the command prompt type in the following command: ‘sfc /scannow’.
