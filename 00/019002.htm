Table of Contents withOUT page numbers? - Microsoft Community

willbeesea say:

Table of Contents withOUT page numbers?

I have a ToC using Heading 1s and 2s.  I want them all listed, but I want only the Heading 1s to have page numbers.  I want to list the Heading 2 lines, but without their page numbers.

I know I can manually remove the dots and page numbers from the Heading 2 lines in the ToC and then that continues as long as I only "Update Page Numbers."  But if I "update entire table" they come back (of course).  Is there a way to display Heading 2s, 3s, etc. without their page numbers?

hansv mvp say:

Press Alt+F9 to view field codes. You should see something like

{ TOC \o "1-2" \h \z \u }

Add a switch \n 2-2, so that it looks like this:

{ TOC \o "1-2" \h \z \u \n 2-2 }

Press Alt+F9 again to hide field codes.

Select the entire TOC, then press F9 to update it, and specify that you want to update the entire table.

If your TOC contains Heading 1 to Heading 3, you'd see \o "1-3", and you'd want to add \n 2-3.
