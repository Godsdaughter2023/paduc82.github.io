Enter below formula in cell E2 & copy down:

=IFERROR(IFERROR(IFERROR(INDEX($A$2:$A$100,MATCH(0,INDEX(COUNTIF(E$1:E1,$A$2:$A$100)+($A$2:$A$100=""),,),0)),INDEX($B$2:$B$100,MATCH(0,INDEX(COUNTIF(E$1:E1,$B$2:$B$100)+($B$2:$B$100=""),,),0))),INDEX($C$2:$C$100,MATCH(0,INDEX(COUNTIF(E$1:E1,$C$2:$C$100)+($C$2:$C$100=""),,),0))),"")

Its a normal (non-array) formula. Presently it extracts from 3 columns. The formula can be expanded to cover additional columns by further nesting with IFERROR, for example below part covers column A and similarly column B & C have been included in the above formula.:

INDEX($A$2:$A$100,MATCH(0,INDEX(COUNTIF(E$1:E1,$A$2:$A$100)+($A$2:$A$100=""),,),0))

The above formula will skip duplicates being mentioned multiple times and also skips blanks.

That would be Error 421 (too many connection attempts within a short timeframe). This typically affects users who try to send hundreds or more emails per session (eg. over 15 mins). If your IMAP error was 421, were you syncing your Hotmail account (including lots of send) every often, say every 1-5 mins ?

# 421 RP-001  The mail server IP connecting to Microsoft server has exceeded the rate limit allowed.

Reason for rate limitation is related to IP/domain reputation problems.

Wikipedia describes Error 421 as:
Misdirected Request: The request was directed at a server that is not able to produce a response (for example because a connection reuse).

See also:

  • Received error code 421-RP-001 (Microsoft Answers)
  • Delivery problem: Too many connections from your host (hmailserver.com)
