Technically, that's a status code, not an error code, and it doesn't come from Windows. Rather, it comes from the website you want to visit. It means that the web page or web resource you are trying to access is forbidden to you.

Either it actually is forbidden to you, or there's some problem with the web page. In either case, you'll need to contact the owner of the website for help.

windows system error: a duplicate name exists on network. - Microsoft Community

cwlong52 say:

windows system error: a duplicate name exists on network.

I am getting a windows error message at initial boot. The message is "Windows System Error: a duplciate name exists on the network." How can I correct this?
lemp say:
What version of Windows, including service pack, is installed?

After the error, can you use the computer to access the Internet?

Do you have a home network (i.e., more than one computer sharing an Internet connection) or is this error appearing on a computer used in an office/corporation/business?

Assuming this is a home situation, what has changed recently in terms of other devices (computer, printers, game boxes, etc.) added to the mix?

How is this computer connected to the Internet?

Right-click My Computer and select Properties.  Then click the "Computer Name" tab.  The "Full computer name" should not be the same as the "Workgroup" name.

The error is a complaint that some other device on whatever network to which your computer is connected has the same name as the "Full computer name" shown in System Properties (I suppose it's also possible that the error is reporting that there are two devices on the network with the same name even if neither of those devices is your computer).

Although there are utilities that scan your network and report the names of the connected devices, they are somewhat non-straightforward to use and might not work from a computer that's exhibiting the error you report.

If your computer is on a home network, sharing an Internet connection using a router, you can manually check all of the other devices.  You may not easily be able to determine the "computer name" of non-computer network devices, but on the other hand, it is highly unlikely that any such device, e.g., a network printer, would have the same name as your computer.

Just to be sure that there's not something peculiar going on, disconnect your computer from your network (and/or your Internet connection) and then reboot it.  The duplicate name error should NOT appear if the computer is not connected to a network.
