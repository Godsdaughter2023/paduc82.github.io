Hello Odings,

Thank you for keeping us updated on the status of the issue and apology for the delayed response.

I would suggest you to completely remove the entries of the Canon scanner from the Windows registry.

In order to do so, refer the steps given bellow:

Step 1: Uninstall the Canon scanner software.

Refer the link: Uninstall or change a program

Step 2:

a. Press Windows key + R key. Run window opens.

b. Type services.msc in the start search box and press Enter.

c. Select Print Spooler service.

d. Right click on Print Spooler service and select properties.

e. Click on the Stop button.

Step 3:

1. Go to “Program & Features” & uninstall any drivers related to the Canon scanner.

2. Create a restore point.

3. Browse  to C:\WINDOWS\SYSTEM32\SPOOL\DRIVERS\w32x86 & delete its contents.

4. Browse  to C:\WINDOWS\SYSTEM32\SPOOL\PRINTERS & delete its contents.

Step 4: Next, edit the registry.

Important this section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base:

1. Press Windows key + R key. Run window opens.

2. Type regedt.exe in the start search box and press Enter.

3. Registry editor windows will open.

4. Navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Print\Environments\Windows NT x86

5. View the list of subkeys (subfolders). There should only be the following subkeys:

a. Drivers

b. Print Processors

If, any other subkey is present then take registry backup & delete it.

6.  Navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Print\Environments\Windows NT x86\Drivers & then look for Version 2 or Version 3, subkey.

Delete the entries except Microsoft Shared Fax Driver & Microsoft XPS Document Viewer* under the –Version-x. (after taking registry backup).

7. Navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Print\ \Monitors

View the list of subkeys. There should only be the following subkeys for the default print monitors*:

a. Local Port

b. Microsoft Shared Fax Monitor

c. Standard TCP/IP Port

d. USB Monitor

e. WSD Port

If, any other subkey is present then take registry backup & delete it.

8.  Exit Registry Editor.

If the scanner is still connected then unplug it & restart the computer.

9. Restart the Print Spooler service. Status should be Started & Automatic.

10. Try to install the Scanner drivers again from the manufacturer’s website.

Hope this information is helpful. Please do let us know if you need further assistance, we’ll be glad to assist you.

Hi,

If the problem persist, please follow the below methods:

Method 1: I suggest you to run in-built Power troubleshooter and check. Follow the steps:

  1. Press Windows Key + X.
  2. Click on Control Panel.
  3. Select Troubleshooting on the left pane, click on View All.
  4. Click on Power to run the troubleshooter.

Method 2: If the above method doesn’t help, I would suggest you to reset and restore all of the default Power Plans back with their default settings. Follow the steps. 

  1. Press Windows Key + X.
  2. Select Command Prompt (Admin).
  3. At the command prompt, type powercfg -restoredefaultschemes, and then press ENTER.
  4. At the command prompt, type exit to exit the Command Prompt window.
  5. Restart the computer and check with the issue.

Let us know if the issue is resolved or if you require further assistance.
