tgunda say: There are too much entries in the registry to correct them manually one by one. 
An easier and quicker solution is to copy the full MSOCache folder to a new place and to make a soft link to it:

Create a new folder, e.g. f:\MSOCache
Copy everything from C:\MSOCache to the new one.
Rename the old folder  C:\xMSOCache  (Don't delete it, just in case).
Open a command prompt window in administrator mode.
Write:  mklink /d c:\MSOCache f:\MSOCache
Now there is an MSOCache link at C, pointing to the new place.

If everything is OK, you can delete  c:\xMSOCache

How do I find what file formats work in what countries? - Microsoft Community

agdiii say:

How do I find what file formats work in what countries?

We are trying to create a portal that would send any type of file or attachment to 38 different countries without the receiving country needing to convert or reformat the file.  Is there a way to find out what file types are supported by different countries?
steve rindsberg say:
Support for file types has little or nothing to do with what country the recipient is in, other than language issues.  

For example, if you need to send files in languages like Hebrew or Arabic, the format you choose must be able to support right-to-left text.  Chinese, Korean and Japanese require special support as well.  

Of course, the software you produce these documents must support any of the languages that you plan to use.

You also need to work out whether the recipient needs only to be able to read the contents of the file, or do they need to be able to edit/save as well.

The biggest question is whether the recipient of the email has the software needed to open the documents you send.

I'd follow Gina's advice and look into PDF.  It supports multiple languages, nearly any program that can print can also produce PDFs.  Recent versions of Office can make PDFs natively from Word/PPT/Excel (Gina ... Access too?) so no additional software is needed.

Most computers come with a copy of the free Reader program for PDFs, but if not, anyone can download it for free.

The only hitch would be if you need recipients to be able to edit the files.  That's pretty much a non-starter for PDFs, at least for the general audience.
