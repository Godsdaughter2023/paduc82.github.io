e.j. gun say:

Your If statement should check if the rounding of the time equals zero.  If it does, than you rounded up to midnight, which means the day should be incremented by one.  Except in the case where you rounded DOWN to midnight, in which case you don't change the day.  It looks like this:

=IF(AND(TIME(HOUR($G6+1),MROUND(MINUTE($G6),6),0)=0,MINUTE($G6)>50),DATE(YEAR($G6),MONTH($G6),DAY($G6+1))+TIME(HOUR($G6+1),MROUND(MINUTE($G6),6),0),DATE(YEAR($G6),MONTH($G6),DAY($G6))+TIME(HOUR($G6+1),MROUND(MINUTE($G6),6),0))

Note the "AND" that says both the rounded minute must be zero and the minutes must be 50 (you could choose any number > 6 I guess).  That ensures that dates just after midnite don't adjust.

doug robbins - mvp office apps & services (word) say:

If put the mergefield that you want to use as the file name into the footer of the mail merge main document, along with the other text, and you execute the merge to a new document, by create a pdf file for each record with the filename being the taken from the text in the footer of the document created from each record.

Dim i As Long, Source As Document, Target As Document, Letter As Range

Dim fname As Range

Set Source = ActiveDocument

With Source

    For i = 1 To .Sections.count

        Set fname = .Sections(i).Footers(wdHeaderFooterPrimary).Range

        fname.End = fname.End - 1

        Set Letter = .Sections(i).Range

        Letter.End = Letter.End - 1

        Set Target = Documents.Add

        With Target

            .Range.FormattedText = Letter.FormattedText

            .Sections(1).Footers(wdHeaderFooterPrimary).Range.Text = ""

            .SaveAs filename:=fname.Text & " - Contributions", FileFormat:=wdFormatPDF

            .Close wdDoNotSaveChanges

        End With

    Next i

End With

  ReDim fileNames(1 To 1) ' create the array

    ReDim Preserve fileNames(1 To UBound(fileNames) + 1)
    sFile = Dir() ' get next filename or empty string
  Loop
  'array filenames() will have an empty element at the upper end
  'if the array size is only 1, then no files were found
  If UBound(fileNames) > 1 Then
