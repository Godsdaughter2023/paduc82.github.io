mike middleton say:

An article describing some of the design decisions made by the "Excel team" is

https://www.microsoft.com/en-us/microsoft-365/blog/2015/08/18/visualize-statistics-with-histogram-pareto-and-box-and-whisker-charts/

which includes

"For outliers, we designed the Box and Whisker chart to follow the Tukey industry standard, which states that values are considered outliers only if they lie 1.5 times the length of the box (known as the interquartile range) from either end of the box."

For more insight, it may be helpful if you post a data set for one of your box-whisker charts so we can replicate your results.

Also, specify the version of Excel (from Excel's topmost menu choose Excel > About Microsoft Excel). I'm currently using general release 16.15.
