ossiemac say:

It is only a long shot but I have at times been able to get rid of unexplained behaviour by deleting the temp files for the application so that the application is forced to re-create them when opened. It doesn't hurt to clean these files out occasionally anyway.

  1. Close any open applications. (don't worry too much about the ones that open automatically on start and have nothing to do with the application giving the problem)
  2. Click the Start button bottom left of screen
  3. In the field directly above (Search programs and files) enter   %temp%
  4. Press Enter
  5. Select all of the files displayed. (Click on any file in the displayed windows explorer and press Ctrl and A to select all)
  6. Press Delete key and at the message confirm to delete.
  7. When you get a dialog box message that can't delete file because in use click Skip button.
  8. Close the windows explorer window and re-start Excel and test.

How to calculate cost price where margin percent and selling price is - Microsoft Community

christopher_teo say:

How to calculate cost price where margin percent and selling price is given?

If selling price is $20.00 with profit margin of 20%.
How do I write the formula to find out the cost price?

Thank You
hansv mvp say:

The cost price will be selling price - 20% of the selling price.

So with the selling price in A1 and the margin in B1, the formula is

 

=A1-B1*A1

 

You can also write it as (100%-20%) of the selling price:

 

=(100%-B1)*A1

 

or, if you keep in mind that 100% is the same as 1:

 

=(1-B1)*A1

Conditional Mean, Median and Standard Deviation - Microsoft Community

mahateja say:

Conditional Mean, Median and Standard Deviation

I need to find the Mean value of a range when the following conditions are met- The value of another corresponding range lies between 70 and 90.
Similarly I want to find the Median and Standard Deviation values for a similar condition. What should the formulae be?
jeeped say:
I need to find the Mean value of a range when the following conditions are met- The value of another corresponding range lies between 70 and 90.

Standard practice would be to put the MEDIAN, AVERAGE and STDEV() functions into an array formula with IF(IF()) conditions. If the range of values to be evaluated in in A2:A10 and the criteria range (>=70 & <=90) is in Z2:Z10 then try these,

=AVERAGE(IF(Z2:Z10>=70,IF(Z2:Z10<=90,A2:A10)))
=MEDIAN(IF(Z2:Z10>=70,IF(Z2:Z10<=90,A2:A10)))
=STDEV(IF(Z2:Z10>=70,IF(Z2:Z10<=90,A2:A10)))

Each of those formulas must be finalized with Ctrl+Shift+Enter↵ rather than just Enter↵. If you need to use a STDEV.P() function or some other, I'm sure you can pick up the pattern from those three. Post back if you have any trouble transcribing the formulas for your own purposes. Remember to keep the rows of the ranges matching in size.

Edit: Excel 2007/2010 has a new function called AVERAGEIFS() that can perform one of the above like this,
=AVERAGEIFS(A2:A10,Z2:Z10,">=70",Z2:Z10,"<=90")
... but you would still have to provide for the other two formulas.
