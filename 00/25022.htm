mike middleton say:

 how did you get Excel to give you the correct value of chi squared?  

(A) If you have the observed and expected frequencies, you use worksheet formulas to calculate the chi-square statistic.

A PDF file "Chi-Square Test of Independence" includes an example of Excel formulas using mixed references (particularly useful for calculations when the observed data displayed in a large table). The file is available at http://www.MikeMiddleton.com in the "Books, Papers, and Presentations for Data Analysis" section.

(B) If you have a right-tail probability and degrees of freedom , you use the CHIINV function to get the chi-square value.

mike middleton say:

The Chi-Square statistic value is the sum of (observed-expected)^2/expected.

For observed 100,90,10,20 and expected 95,95,15,15, the individual terms are approximately 0.263158,0.263158,1.666667,1.666667, and the sum is approximately 3.859649.

With the 2x2 tables, the CHISQ.TEST result is 0.049, and

CHIDIST(x,df) = CHIDIST(3.859649,1) = 0.049.

With the 1x4 arrangement, the CHISQ.TEST result is 0.277, and

CHIDIST(x,df) = CHIDIST(3.859649,3) = 0.277.

The CHISQ.TEST function seems to be using the correct degrees of freedom depending on the way the data are organized on the worksheet.
