andre da costa say:

You will have to purchase a new license for the new PC.

OEM :

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

Buy Windows 10 Home/Pro

A full license permits transfer rights even when the motherboard dies or you no longer want to use it on that computer.

Hi Claudiu,

Thank you for writing to Microsoft Community Forums.

I understand the inconvenience you are facing in accessing the preferred location on Windows Map application.

Provide us additional information to assist you better with this issue:

  1. Since when are you facing this issue?

  2. What is the current OS build running on your device?

Try the below suggested steps to fix the issue.

Method 1: Run the Store app troubleshooter

  1. Click on Start, type in Troubleshoot.

  2. Select Windows store apps and click on Run the Troubleshooter.

  3. And then check the status of the issue.

Method 2

  1. Click on Start > Settings > Privacy > Location.

  2. Check Location for this device is ON under Allow access to location on this device.

  3. Also check Allow apps to access your location is ON and check the status of the issue.

Method 3

  1. Click on Start > Settings > Time and Language > Region and Language.

  2. Ensure your home country is correct under Region and Language.

Method 4: Reset the Maps app

  1. Go to Search, type powershell, and open PowerShell as Administrator.

  2. Paste the following command, and press Enter:

Get-AppxPackage *windowsmaps* | Remove-AppxPackage

Your Maps app will be uninstalled.

 3.  Now head to Microsoft store, search for Maps and install it again to check the status of the issue.

You may also refer the article Windows 10 location service and privacy for more information.

steve winograd say: To select multiple files, hold down the CTRL key while clicking each of them.

To select a continuous range of files, click the first one and hold down the SHIFT key while clicking the last one.

Then right-click any of the files, click Cut , right-click the folder where you want to move them, and click Paste .
