How to extract data between 2 open and close brackets - Microsoft Community

billmark say:

How to extract data between 2 open and close brackets

Hi

I have the following codes to extract information from the text with only 1 open/close bracket.  Please advise how to handle more than one open/close brackets in the cell.

eg Mug(s) (7.1%)

I want the Sector/Country and Percentage to 

FundSector Fund Sector/Country Percentage
AAA ABC Mug(s) 7.1

//

    

Sub copydata()

Dim i As Integer
Dim qusector As String
Dim quper As String
Dim fundno As String
Dim openbracketpos As Integer
Dim closebracketpos As Integer
Dim FolderPath As String
Dim Filepath As String
Dim Filename As String
Dim lrow As Long
Dim lastrow As Long
Dim lastcolumn As Long
Dim wb As Workbook

Application.ScreenUpdating = False

'clear content
Worksheets("Summary").Select
Worksheets("Summary").Columns("A:C").Select
Selection.ClearContents
Worksheets("Summary").Range("A1").Value = "FundSector"
Worksheets("Summary").Range("B1").Value = "Fund"
Worksheets("Summary").Range("C1").Value = "Sector/Country"
Worksheets("Summary").Range("D1").Value = "Percentage"

FolderPath = Range("G1")
Filepath = FolderPath & Range("G3")
Filename = Dir(Filepath)

Do While Filename <> ""
Set wb = Workbooks.Open(FolderPath & Filename)
wb.Worksheets("portfolio").Activate
lastrow = ActiveSheet.Cells(Rows.Count, 1).End(xlUp).Row

fundno = Sheets("variables").Cells(1, 105)

For i = 7 To lastrow
wb.Worksheets("portfolio").Activate
openbracketpos = InStr(wb.Worksheets("portfolio").Cells(i, 3), "(")
closebracketpos = InStr(wb.Worksheets("portfolio").Cells(i, 3), ")")


If closebracketpos * openbracketpos > 0 Then
If Cells(i, 3) = UCase(Cells(i, 3)) And Cells(i, 3).Font.Bold = True Then
    qusector = Left(Cells(i, 3), openbracketpos - 2)
    quper = Mid(Cells(i, 3), (openbracketpos + 1), (closebracketpos - openbracketpos - 2))

Workbooks("MasterNEWRevised.xlsm").Worksheets("Summary").Activate
lrow = Worksheets("Summary").Cells(Rows.Count, 1).End(xlUp).Offset(1, 0).Row
Worksheets("Summary").Cells(lrow, 1) = fundno & Trim(qusector)
Worksheets("Summary").Cells(lrow, 2) = fundno
Worksheets("Summary").Cells(lrow, 3) = Trim(qusector)
Worksheets("Summary").Cells(lrow, 4) = quper

End If
End If
Next i

wb.Close savechanges:=False

Filename = Dir

Loop
Worksheets("Summary").Range("A1").Select
Worksheets("Summary").Range("A:D").Columns.AutoFit

Application.ScreenUpdating = True


End Sub

The three most likely cures in order of probability:

  1. Go to Word> Preferences> Print, make certain that the boxes are checked for Drawings & for Background colors & images,
  2. Lacking updates: Make certain that your installation of Office is current [Version 15.40], confirm that your version of macOS is updated, & check the HP support site to make certain you have the latest driver for your model,
  3. Watermark Settings: They often involve Transparency which some printers do not handle well (or at all).
