Dim i As Long, j As Long

suzanne s. barnhill say:

To clarify, are you trying to print on paper that is actually 4.625"x7" or are you trying to print two pages this size on 8.5"x11" paper? If the latter, then enabling 2 pages per sheet in the Page Setup dialog and adjusting margins to the correct print area would be the equivalent of using the Rite-in-the-Rain template (except that you could create one page at a time). But if you want to print an 8.5"x11" page twice on 8.5"x11" paper, then you would need to use the 2 pages per sheet setting in the Print dialog, and the margins of the Letter-sized document would have to be large enough to keep the print area within the target paper size.

If you are actually trying to print on 4.625"x7" paper, then the best you can do is hope that your printer allows you to define custom sizes. If so, then you could define this as a custom size in the printer driver and use "Scale to paper size" to select that size as the output for your Letter-sized document.

Word 2011 Macro to convert textboxes to plain text? - Microsoft Community

maximzodal say:

Word 2011 Macro to convert textboxes to plain text?

OCR software inserts numerous textboxes into Word 2011 file. Converting them requires first formatting each to convert to frame then formatting frame and remove frame.  I've searched but been unable to find a macro that can do this.

a. user say:

You could start with a macro like this (I just wrote this now, so I would expect problems) and see what problems you encounter:

Sub tbtxts2txts()
Dim lng As Long
Dim targetrange As Word.Range
Dim sourcerange As Word.Range
Dim shp As Word.Shape
For lng = ActiveDocument.Shapes.Count To 1 Step -1
  Set shp = ActiveDocument.Shapes(lng)
  If shp.Type = msoTextBox Then
    Set sourcerange = shp.TextFrame.TextRange
    Set targetrange = shp.Anchor.Paragraphs(1).Range
    targetrange.Collapse direction:=WdCollapseDirection.wdCollapseStart
    targetrange.FormattedText = sourcerange.FormattedText
    Set targetrange = Nothing
    Set sourcerange = Nothing
    shp.Delete
  End If
  Set shp = Nothing
Next
End Sub

The idea in this case is that the macro copies the text to the beginning of the paragraph where the textbox is anchored, preferably complete with formatting. Then it deletes the textbox. But if that is, on average, the *wrong place* then you may find that copying/pasting the text to the right place is not much simpler than copying it out of the text box and pasting it manually.
