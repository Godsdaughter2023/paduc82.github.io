Neither the creator version nor the file type is the issue. The problem is that the file contains Frames which are ActiveX Controls. ActiveX Controls are not supported by Excel Online [which the "browser" message suggests you're actually trying to use] or by any version of Excel for Mac.

It further sounds like the file may be stored on OneDrive or other cloud service. I don't have any way to test, but unless there is other unsupported content I believe the Online app should be able to open the file, but the unsupported content either will not be displayed or will not be usable.

hansv mvp say:

Try this:

Sub Increase()
    Dim r As Long
    Application.ScreenUpdating = False
    For r = 35 To 70
        Do While Range("FO" & r).Value < 0.8
            Range("FN" & r).Value = Range("FN" & r).Value + 1
        Loop
    Next r
    Application.ScreenUpdating = True
End Sub

Warning: if your function in column FO does not reach 80% by increasing the value in column FN, the macro will get stuck in an endless loop...
