Sub SoftenAll()
    Dim oSld as Slide
    Dim oShape as Shape

    For Each oSld in ActivePresentation.Slides
        For Each oShp in oSld.Shapes
            If oShp.Type = msoPicture Then
                oShp.SoftEdge.Type = msoSoftEdgeType1
                'You probably need some other stuff to get the right soft edge effect here
            End If
        Next oShp
    Next oSld
End Sub

Sub setEdge()

For Each osld In ActivePresentation.Slides

In Build 10041, sometimes the user name and password boxes do not show up on the logon screen at all.  This can be triggered by locking Windows, swiping up to show the logon screen, letting the 30-second timeout kick in to show the lock screen again, and then try to sign in again.    Once this happens, the only recourse is to press the power button.
