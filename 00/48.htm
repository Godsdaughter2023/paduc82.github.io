Further to my reply to John, the fields in footnotes do not update when you select the document & use the keystroke to update [although I prefer F9 (fn+F9), myself] because the footnotes are seated in a different layer of the document. How to link/unlink contacts - Microsoft Community

jorge e. - wp7 support say:

How to link/unlink contacts

How to link/unlink contacts

 
jorge e. - wp7 support say:

If you have more than one contact card for the same person (e.g., one created by signing in to a social network, one that you’ve created on your phone, and one created using information from an Exchange account), you can link the cards together so they’ll all appear when you view that contact.

To link cards:

1.      Open a contact card.

2.      Tap More > Contact linking to open the Linked Contacts list. Currently-linked contacts are shown at the top of the list. Contact cards with similarities are shown under Linking suggestions.

3.      Tap Link another to open the list of contact cards.

4.      Tap the contact card you want to link, then tap Done.

When you search for and find a contact with linked cards, each linked card will appear on its own screen. The first screen will display any card you created on the phone, then any Exchange card, then any social network cards.  Just slide your finger to the left or right to move between the cards.

Notes:

·         If you want to unlink any of the currently-linked contact cards, tap Unlink.

·         You can’t link the same contact to more than one set of linked cards.

·         Each editable linked card will have its own edit menu; edits to one card won’t affect the other linked cards. (Only phone and Exchange cards are editable.)

·         When you add a card to Favorites, all its linked cards will be automatically added to Favorites as well, but they will appear as a single contact in Favorites.

·         Linked cards appear only once on each contact screen.

·         If you unfavorite a contact, all the associated contact cards will be removed from Favorites as well.

·         If you delete a contact with linked cards, only the card you delete will be removed. The other linked cards will remain on your phone.

I'm not clear what cell you have this data in but try this

=SUMPRODUCT(--(ISNUMBER(SEARCH(E1,$A$1:$A$1000))))

=SUMPRODUCT(--(ISNUMBER(SEARCH(SUBSTITUTE(E1," ",""),SUBSTITUTE($A$1:$A$1000," ","")))))

cindy meister say:

Hi Worf

Mmmm. That version of Word will insert page numbering in two different ways:

- "free floating": This is a TextBox object - a graphic. Most experienced Word users don't care for this approach as it doesn't follow all the rules in complex documents and can be difficult to manage.

- A field code, directly in the header or footer (not free floating). Most of us prefer this, as it does follow all the rules and is more predictable.

But the command Word offers at the top level (Insert/Page Number) is for the first, not the latter. In order to try the second option, use Insert/Header and Footer. The buttons on the Header and Footer toolbar will create the "right kind" of page number. See if that works any better.

Note: I don't know why the page numbers would disappear from your new Word document (as opposed to the conversion files mentioned in your other message) when you save and close, assuming Word is saving to its native *.doc file format. The only thing that occurs to me, right off-hand, is that you might have a macro executing on save or close that's removing things.

lgodiva say:

Excel - how to conditionally highlight range of rows based on one cell value that is dynamic (not hardcoded)?

I have searched extensively and have not found an answer to this question yet.  I would like to conditionally highlight a range of rows based on the value of one cell in the row, but I will not know the values ahead of time, so I cannot hardcode the values into the formatting logic.  One column contains people's names...I want to highlight rows with person #1 as one color, rows with person #2 as a different color, and so forth (or alternating the same two colors).  Is there an easy way to do this?  Thanks!
rajesh c. say:

Hi,

Assume that you want to highlight range from C2:E2 based on Name value in cell C2
Assume that you will be putting the search value ( not known now - will be known in future only )  in cells  A2  & A3

Carryout the following steps:
1. Select range C2:E2
2. Click on -  < Home >  -  < Conditional Formatting > - < NewRule > -
3. Click on -  < Use a formula to determine which cells to format >
4. Key in the following formula in the box under "Format values where this formula is true:"
5. =C2=$A$2      ( here we assume that the Person#1 name which you wish to search and highlight will be keyed in in cell A2 )
6. Format as required/desired by you - for person #1
7. Click OK
8. Once again Select Range C2:E2
9. Click on -  < Home >  -  < Conditional Formatting > - < Manage Rules > -  < New Rule..>
10. Click on -  < Use a formula to determine which cells to format >
11. Key in the following formula in the box under "Format values where this formula is true:"
12. =C2=$A$3      ( here we assume that the Person#1 name which you wish to search and highlight will be keyed in in cell A3 )
13. Format as required/desired by you  - for person #2
14. Click OK
15. Using Paintbrush apply the above formatting to the number of rows ( say for example C3:E50 )
16. Key in some person names for #1  & #2  in cells A2 & A3

You should be able to get something like this..

You can similarly apply to your actual data incase if it is different.

Hope this is Helpful.

Feel free to put your query in case of any issues.
