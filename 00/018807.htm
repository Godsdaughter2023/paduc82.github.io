I just started working at a place that sends out massive amounts of overtime requests via email.  These requests can come from anyone and the subject is usually something like, "OT available in building 10" or "Anyone want some OT on Friday?"  I can get anywhere from ten to fifty of these a day.  So obviously I want to set up a rule to filter them out, but since there's no set list of people who send them out, and no specific subject line, the only constant in all of them is the word "OT".  

In my ignorance, I set up a rule to move any email with the word "OT" in the subject line to the Overtime folder.  I realized a few days later that it was also filtering email with words like "Note" and "Remote."  So I then set up a rule to filter the word " OT ".  Notice the space before and after.  Well that worked for the emails where OT is in the middle of the subject somewhere, but not for the emails that start with the word OT.  I added the word "OT " with a space after, but that captured words like "Hot" and "Not."  I tried adding OT with quotes around it, but that didn't work at all.  

So how can I create a rule that filters out all subject lines with the whole word "OT" and not words that contain the letters "OT"?
brian tillman [outlook mvp 2007-2019] say:
Rule searches search for substrings.  It doesn't matter where in the string the substring is.  However, if you put a delimiter of some kind in front and behind "OT" you can find it if it's in the middle of a string of words, like your second example.  Use " OT " (i.e., space, oh, tee, space) and it will match "Anyone want some OT on Friday?"  It will miss your first string "OT available in building 10, however, or "Anyone want some OT" without a question mark on the end.

What you ask is not so easy. And you can not strongly protect the Excel file and source code. We can hide all the sheets (except one) when we close the file. When you will open file, will be requested a password.

Try this:

  • Open a new file with 7 sheets.
  • Try to copy/paste this code in ThisWorkbook module

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    Dim sh As Worksheet
    For Each sh In Worksheets
        If sh.Name <> "Sheet1" Then
            sh.Visible = xlSheetVeryHidden
        End If
    Next
End Sub

Private Sub Workbook_Open()

    Dim sh As Worksheet
    Dim s As Variant
    Dim shRSM1 As Sheets
    Dim shRSM2 As Sheets
   
    Set shRSM1 = Worksheets(Array("Sheet2", "Sheet4", "Sheet5"))
    Set shRSM2 = Worksheets(Array("Sheet3", "Sheet6", "Sheet7"))
   
    For Each sh In Worksheets
        If sh.Name <> "Sheet1" Then
            sh.Visible = xlSheetVeryHidden
        End If
    Next
   
    s = Application.InputBox("Insert password")
   
    If s = False Then Exit Sub
    If s <> "" Then
        If s = "abc" Then
            For Each sh In shRSM1
                sh.Visible = xlSheetVisible
            Next
        ElseIf s = "qwe" Then
            For Each sh In shRSM2
                sh.Visible = xlSheetVisible
            Next
        End If
    End If
   
    Set sh = Nothing
    Set shRSM1 = Nothing
    Set shRSM2 = Nothing
   
End Sub

  • Close and open the file
  • Each password(abc or qwe) shows a different set of sheets

Note. In the sample Sheet1 is the sheet always visible and I use only two RSM. This technique can be extended to individual users.
