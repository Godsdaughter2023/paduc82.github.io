Does Microsoft Security Essentials have anti-fishing? - Microsoft Community

fabiofeim say:

Does Microsoft Security Essentials have anti-fishing?

Does Microsoft Security Essentials have anti-fishing?
kosh vorlon say:

Hi fabiofeim,

 

It's called "anti-phishing" and it's a technique to avoid scams.  Here's a good article with many references that will help you understand more about it and how to use IE, Outlook, and Hotmail to protect yourself from most of them. http://www.microsoft.com/security/online-privacy/phishing-scams.aspx.  Some include methods to report them and activities and features of the products (or add-ons) that can enhance their capabilities if used properly. 

 

MSE offers some but not nearly full protection against such schemes, because for the most part they require you to do something to fully activate or enable the process, and so the major focus is on blocking such websites or popups (where IE can help) - though MSE does stop many of the fake security scams (but it can be assisted greatly by what you do and some other actions you can take that I'll go into shortly, but some still do get through - though if you spot them early and take the proper steps, you can limit the infection and make it easier to remove).  Here's more on those fake security scams: http://www.microsoft.com/security/pc-security/antivirus-rogue.aspx. 

 

Here's more general information on how to protect yourself from malware which can sometimes be part of a phishing scheme: http://www.microsoft.com/security/pc-security/protect-pc.aspx.  Here's additional information of recognizing and defending against phishing attacks. http://www.microsoft.com/security/online-privacy/phishing-interests.aspx.

 

Here's what I do (and it isn't 100% but nothing is, and in some cases only catches things after they've already gotten into the system, but it's a good process while remaining reasonable and they can all work together - as proven by the fact that I use them):

 

I use MSE for real-time protection and run 2-3 quick scans a week with a scheduled bi-weekly full scan.

 

To backup MSE in real-time, I also use Prevx. http://info.prevx.com/downloadcsi.asp. This is only a scanner (get only the free version - the paid version not only isn't worth the money, but includes things which would cause conflicts) - so if something is found, some other product or process must be used to remove it.

 

I update and periodically (every couple of weeks or so) run full scans with Malwarebytes: http://www.malwarebytes.org/ and SuperAntiSpyware: http://superantispyware.com/ and ESET online scanner: http://www.eset.com/us/online-scanner. Get only the free versions. Any that also have paid versions include features that cannot be used along with MSE anyway (in these cases) or have limited value, so there's no point in spending the money anyway.

 

I use Secunia PSI http://secunia.com/vulnerability_scanning/personal/ to make sure all my programs are updated and current (since updates often plug security holes in the software and thus can be important as many people never check and/or update their software unless prompted - and this way I don't need to check everything since notifications and sometimes updates are automatic).

 

I use a standard user account unless I have a specific need for administrative priviledges since if anyone does get into my account, they too will have limited access which in terms means it will limit the potential harm that can be caused.

 

I always follow the following advice from Stephen Boots - MVP:

 

-Make sure that the Windows Firewall is enabled. No other firewall truly offers better protection or is needed. This one is perfectly fine.

 

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

 

-Make sure Internet Explorer is at version 8 or 9 and updated with all patches.

 

-In Internet Explorer 8 or 9, use the SmartScreen Filter.

 

-Make sure that IE Internet Security settings are at least set to medium-high (default).

 

-Enable the pop-up blocker in IE.

 

-On Vista and Windows 7 make sure that User Account Control (UAC) ON and not running with elevated privileges.

 

-Make sure that Windows Automatic Updates are set to at least notify, but the preferred setting is to download and install automatically. If you update manually, be sure to update as soon as possible after being notified of available updates.

 

-Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.

 

-Never click through links from unknown sources and use caution even if they are from a "trusted" source.

 

-Never open unsolicited email attachments.

 

-Practice safe web browsing.

 

I hope this helps.

 

Good luck!

Now I have regular page numbering for each page in the document in the Footer of each page (for example Page 1, Page 2, Page 3 etc.).

My issue is that when I insert an automatic table of contents, it inserts the page number from the Header of each section. So for each section in the table of contents, the page number restarts at Page 1. The TOC is essentially displaying Page X in the header as the page numbers.

How do I get the automatic table of contents to display the page numbering from the Footer of each section instead so that it can show consecutive numbering?

doug robbins - mvp office apps & services (word) say:

In those multiple "Section" part of your, you will need to use

{ = { PAGE } - { PAGEREF Sn } } of { = { PAGEREF Sny } - { PAGEREF Sn }}

Where Sn is the bookmark inserted at the end of the Previous Section and and Sny is the bookmark inserted at the end of the multiple "section" part of your document

For example, if the previous Section ended on Page 4 and that is where you inserted the bookmark Sn

{ PAGEREF Sn } will return 4.

If then in that part of your document, you have say 2 portrait pages, a landscape page and another 3 portrait page2, at the end of that page, you would insert the bookmark Sny and { PAGEREF Sny } would return 4 + 2+1+3 = 10

So for the first page of that part of the document, the above construction would return

5 - 4 of 10 - 4 = 1 of 6

As before, I uninstalled Nokia Suite, Windows Driver Package – Nokia, PC Connectivity Solution, and Nokia Connectivity Cable Driver.   I had also found a folder C:\ProgramData\PC Suite which contained settings and server check files for my phone, but from November 2011.   I deleted this PC Suite folder as well.   I then cleaned the registry using CCleaner, emptied the recycle bin and restarted my PC.  When restarted, I disabled my Firewall and Anti-virus software.   I found Nokia Software Updater from another post in the discussions, downloaded it and ran it, but it hung before finishing the process.   As mikkodss suggested and I agree with him, I don’t think this was actually necessary.   I connected the phone and a driver was loaded.   Then the Windows Update appeared and was loaded successfully.   More drivers were loaded, including accessing “PC Suite Services” which recreated a fresh version of the PC Suite folder (is this what was wrong all the time ?).
