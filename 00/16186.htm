ashish mathur say:

Hi,

 

Given that the your data to Transpose is limited, both methods should be equally efficient.  I assume by Transpose you mean Paste Special > Transpose.  Alternatively, if you mean TRANSPOSE() array formula, then the second method would be more efficient

 

Furthermore, the second method will take quite some time for you to link each cell manually (since you will not be able to copy the formula across).

 

Try this alternative

 

1. In cell B1, type A1

2. With the fill handle, drag all the way till AA1.  The result will be A1, A2, A3 etc.

3. Select B1:AA1 and press Ctrl+H

4. In the Find What box, type a

5. In the Replace With box, type =a

6. Click on Replace All

 

Hope this helps.
