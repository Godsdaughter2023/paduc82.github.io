I'll post the code below and if you do want it to automatically change and select the default value again and overwrite the previously selected value then just comment out (or delete) the If and End If lines in the code where I have bolded the section of code that I added. The following is a copy of the code with the additional code in bold.

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    'Note: Me is the generic object variable for the worksheet to which this module belongs
    Dim strListAddr As String
    Dim cboTemp As OLEObject
   
    If Target.Cells.Count > 1 Then Exit Sub     'Do not process if more than one cell selected
   
    If Application.CutCopyMode Then
        'Allow copying and pasting on the worksheet
        GoTo errHandler
    End If
   
    Set cboTemp = Me.OLEObjects("TempCombo")
   
    On Error GoTo errHandler
    Application.EnableEvents = False 'Required before test for validation cell
   
    If Intersect(Target, Me.UsedRange.SpecialCells(xlCellTypeAllValidation)) Is Nothing Then
        'If nothing then not a validation cell so hide combo and exit the sub
        cboTemp.Visible = False
        GoTo errHandler
    Else
        'If IS validation cell then test for Type 3 (List type) validation
        If Target.Validation.Type <> 3 Then
            cboTemp.Visible = False
            GoTo errHandler    'Not Type 3 then Exit sub
        End If
    End If
       
    'Test 3 cells in columns B, C and D to ensure Gas Selection has been made
    'Otherwise the code will error evaluating the Validation formula
    If Me.Cells(Target.Row, "B") = "" Or Me.Cells(Target.Row, "C") = "" Or Me.Cells(Target.Row, "D") = "" Then
        cboTemp.Visible = False
        GoTo errHandler
    End If
   
    'Evaluate the address (Including the worksheet name) returned by the validation formula
    'and save address to a string variable
    strListAddr = Evaluate(Target.Validation.Formula1).Address(external:=True)
   
    With cboTemp
        'Show the combobox with the list and position and size relative to Target cell
        .Visible = False  'Without this line the DropDowns on previous cells can remain visible
        .Visible = True
        .Left = Target.Left
        .Top = Target.Top
        .Width = Target.Width '+ 15   'OssieMac thinks that width of Target cell should be OK
        .Height = Target.Height '+ 15 'OssieMac thinks that height of Target cell should be OK
        .ListFillRange = strListAddr
        .LinkedCell = Target.Address
        .Object.ListRows = .Object.ListCount  'Length of displayed list matches number of options.
        'Next If/End If. If user has already selected a value and returns to cell
        'then does not automatically change previously selected value to default value.
        If Target.Value = "" Then
          .Object.ListIndex = 0     'Defaults to first value in the list
        End If
        .Activate
        .Object.DropDown
    End With
   
errHandler:
     If Err.Number <> 0 Then
         MsgBox "Error occurred in Module Sheet1 Private Sub Worksheet_SelectionChange." _
             & vbCrLf & "Contact the Administrator of this application."
     End If
     On Error GoTo 0
    
     Application.EnableEvents = True
    
End Sub

kpagewq say:

"Too many spelling or grammatical errors..." error message in Word 2007

I'm getting the "too many spelling or grammatical errors" message on a document that is primarily tables AND I'm not seeing any wavy underlines telling me where the problems are.  Any suggestions?  This just started today with this document; it has never occurred before.
suzanne s. barnhill say:

The reason you're not seeing any wavy underlines is that Word has enabled "Hide spelling errors in this document" and "Hide grammar errors in this document" because you have "too many" of what it considers errors. This can be a problem in any document that contains a lot of text that is not, strictly speaking, English (or whatever your proofing language is). That could include programming code, technical jargon, slang, unusual names, etc., and the amount Word can keep up with is at least partially dependent on your resources: amount of RAM, free space on your HD, etc.

The best workaround for this problem is to put Word out of its misery by telling it not to bother with things that don't need to be spell-checked. You do this by creating a "Do not check spelling or grammar" character style and applying it to text that is always going to be "misspelled." For instructions on doing this, see http://sbarnhill.mvps.org/WordFAQs/MasterSpellCheck.htm#ExemptingText. This will prevent Word from considering the marked text and allow it to do its job on the rest of the text; you should now be able to turn off "Hide spelling/grammar errors"and have Word flag only your actual typos.
