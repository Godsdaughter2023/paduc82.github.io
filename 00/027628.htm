techfreak_ say:

If you want to call Skype call yourself (Skype to Skype) you would need a separate account to accomplish that task.  Think of it similar to a phone number.  You cannot pick up the same phone in your house, dial your home phone and expect to not get a busy signal.

If you want to place phone calls using the existing credit you have, you would have to sign into the same account on both devices.  So if you are logged into another account to be able to Skype call yourself (on that device), simply sign out when you no longer require that, and sign into the account with credit to place your phone calls.   If you do not want to do this, you would have to purchase credit for both accounts.

Understand that Skype to Skype calls use your Skype/Microsoft account and does not require credit or a subscription (free).  Credit or a subscription are required when dialing actual phone numbers.  If you want to uniquely call your laptop or your tablet using a phone number, you would need to get a Skype Number (not free) for one or both of your accounts.

Simply correcting your code. (You were trying to paste again at the selected range and the selected range is a column and you were trying to transpose. When transposing you can only paste to the first cell and allow the code to work out where the rest of the range fits.)

 Sub test()
  Dim rng1 As Range
  Set rng1 = ActiveSheet.Cells(Rows.Count, 1).End(xlUp) _
             .Offset(1, 0)
  Application.Goto Reference:="Extract_Fields1"
  Selection.Offset(0, 1).Select
  Selection.Copy
 
  rng1.PasteSpecial Paste:=xlPasteAll, Operation:=xlNone, SkipBlanks:= _
         False, Transpose:=True

 End Sub

It is almost never necessary to select ranges. The following code is more professional. Note that instead of ActiveSheet I believe it is better to use the worksheet name.

Sub test_2()
    Dim rng1 As Range
   
    Set rng1 = Worksheets("Sheet2").Cells(Rows.Count, 1).End(xlUp) _
                .Offset(1, 0)
   
    Range("Extract_Fields1").Offset(0, 1).Copy
   
    rng1.PasteSpecial Paste:=xlPasteAll, Operation:=xlNone, _
            SkipBlanks:=False, Transpose:=True
   
 End Sub
