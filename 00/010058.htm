How to set up a dual boot for the Insider Program without a new - Microsoft Community

cbarnhorst say:

How to set up a dual boot for the Insider Program without a new license for Windows

Windows 10 supports multibooting on the same license.  This is a change from previous versions of Windows.  

Procedure:  After you select the volume you are going to use for a reinstallation of Windows 10 start a clean install using media.  Make sure you are using the Fall Update (build 10586) as this is harder to do with the July 29 release (build 10240).

When Setup reaches the Product Key page click the link at the bottom center of the page indicating this is a reinstallation (you are not using a product key).  On the next page choose the same edition for which your computer is already registered as a Windows 10 device.

Proceed with the installation normally.

This will result in an activated (by digital entitlement) copy of Windows 10.  

Discussion"  Objections to "violating the license" as a result of wording in the license for some reason are reasonable but don't seem to apply.  (I am fully conversant with Microsft EULAs.)  This procedure would have violated the EULA of any previous version, but does not appear to in Windows 10, the present wording notwithstanding.  Whether this is by Microsoft tolerance or simply a need to update the EULA, I don't know.  Perhaps it is because a second installation is described as a reinstallation, as becomes clear on the product key page when you select the link at the bottom middle of the page to continue without a product key.

I did this procedure with Build 10240 a couple of months ago when the Product Key page would not advance without a product key (clean installs) and supplied the dummy key just to get Setup to continue.  I contacted Support for activation help and was up front that I was trying to dual boot on one license.  A second level Answer tech remoted into my computer and, using a retail key, activated Windows for me.  This is also consistent with a tweet from Gabe Aul (last summer) responding to a user asking about dual booting this way.  Aul commented that dual booting on the same license was possible but virtual machines each required their own licenses.  I have tried to recover the quote from the net but apparently that string is now archived.

What has changed is that the first installation of Windows 10 registers the computer as a Windows 10 device.  The focus of anti-piracy measures has changed from the software to the hardware.  Your motherboard's serial number is now accessible in your MSA account at account.microsoft.com by clicking to the right of the device icon. The motherboard serial number is the basis for the device's registration as a Windows 10 device.  The device list shows those devices you have associated with your MSA.  The link you click on the product key page for subsequent installations defines what you are doing as a reinstallation.  

Functionally, it just works and does not require any "workarounds" or "tricks."  In cooperation with fellow insider John AC Nolan we have verified this using the Fall Update.  This provides a way for someone wanting to be an insider but not having a spare license for Windows 7 or 8 to participate without succumbing to the temptation to use their primary copy of Windows.
