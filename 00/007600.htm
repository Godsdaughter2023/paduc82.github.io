toasterscheme say: D'oh - I knew as soon as I posted this it would just start working.

Well, one of the following two things I found elsewhere caused the registry keys for HKU\S-1-5-19 and S-1-5-20 to reappear and after that the install went through no problem.

1) In LocalDisk(C:) Properties->Security->select user "SYSTEM". Tick "allow full control" (it already was) ->Advanced->Replace permission entries on all child entries [and wait for a long long time].

or 2) In Control Panel->Administrative Tools->Computer Management->Local Users and groups->groups->administrators->add->advanced->find now->, add both LOCAL SERVICE and NETWORK SERVICE to the Administrators group.

herb tyson say:

For the comment text, set the desired font in the Comment Text style--make this change in the template you use for creating these documents (usually Normal.dotm).

Except... I wonder if you really mean the Word comment feature... since that feature does not let you resize or otherwise change the comment text box.

If you really mean a text box (AutoShape)... you can create a text box with the desired shape and format the default paragraph mark inside it to have the desired text formatting. Then select that text box, and save it to the Quick Parts Gallery (or as a building block... your choice). This then serves as a "template" of sorts each time you want to insert a "comment" of that type.

Alternatively, once created, you can create a formatted AutoCorrect entry that automatically inserts the comment. Select the comment box, and choose  Office button, Word Options, Proofing, AutoCorrect Options. Set With: to Formatted Text, and type the shortcut you'd like to use into the Replace: field (e.g., cmmnt, or some other non-word, since each instance will automatically be converted into the text box.
