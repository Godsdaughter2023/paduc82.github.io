Dear Minnie,

I came across your issue and I wanted to provide you with a solution, I've had similar issues with a much smaller Toshiba Tablet (16GB and 1GB RAM).

Make sure you have installed a SD card with a capacity of at least 16GB

  1. Plug in the tablet to the power outlet
  2. Go to microsoft.com windows 10 download page and download the latest W10 version in ISO format 
  3. Save the ISO to the SD card
  4. Use a free app to unpack the ISO file (winrar or winzip)
  5. Make sure you unpack it in a folder again in the SD card
  6. In that folder, right click the setup.exe file and choose "run as admin"
  7. Select to reinstall W10 keeping apps and files intact
  8. When the app asks for more space, choose once again to use the SD card
  9. Windows latest version will be installed
  10. One month after the new install the folder "old.windows" in your tablet and on the SD card will be deleted. Don't delete them yourself in case you want to revert to the previous installation

Hope this will help.
