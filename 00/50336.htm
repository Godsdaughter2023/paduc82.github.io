    Hi, 

    Corrupted components of your system is one possible reason why you are experiencing FPS drop. Here are methods on how to troubleshoot low FPS on Windows 10: 

    Method 1: Switch the Game bar off. 

    The Game bar is switched on by default. First, switch the Game bar off via the Settings app. You can configure the Game bar’s settings as follows. 

    • First, press the Cortana button and enter ‘settings’ in the search box. Select to open the Settings app. 

    • Alternatively, you can also press the Win key + I hotkey to open Settings. 

    • Next, click Gaming to open the options shown in the snapshot directly below. 

    • Select Game bar on the left of the Settings window.

    • Then toggle the Record game clips, screenshots, and broadcast using Game bar setting switch to Off. 

    • Below that setting there’s a Show game bar when I play full-screen games Microsoft has verified option. Uncheck that option’s check box so that it’s not selected.

    Method 2: Switch off Game DVR. 

    The Game bar includes Game DVR to record video so that you can capture clips of those more exciting gaming moments. Even though you’ve switched the game bar off, Game DVR still remains on. This is how you can disable the Game DVR. 

    • Open Cortana and enter ‘xbox’ in the search box.

    If you have Windows Media Player 11 (or newer)…

    1. While connected to the Internet, open OverDrive Media Console.

    4. Click 'Upgrade'.

    5. Click the 'Close' button in the 'Windows Media Player' dialog box.
      The 'Media Usage Rights Acquisition' dialog box is displayed.
    6. Click the 'Play' button.

    4. Click 'Yes' to perform the upgrade.
