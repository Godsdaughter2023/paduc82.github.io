Graph colors be auto formatted to negative #'s? - Microsoft Community

lbrown1 say:

Graph colors be auto formatted to negative #'s?

Is there a way you automatically have colors in an Excel graph be red or green based on negative or positive numbers?
pbezucha say:

There are solutions on several sites. This one with macro is relatively comfortable and can be applied to many types of charts.

Option Explicit

Sub NegativeRed()
'Petr Bezucha, 2010
'Sub shades markers, lines or bars green or red according to positive or negative values, also trends in line chart can be followed
'A series must be selected before deployment.
'Both interior and border colors are set up.

Dim ChT As String, ChType As String, ErrMsg As String, _
  I As Long, MarkCol As Long, _
  Ser As Series
ErrMsg = "No series has been selected"
On Error GoTo ErrExit
Set Ser = Selection
ChT = Selection.ChartType
Select Case ChT
Case xlXYScatter, xlXYScatterLines, xlXYScatterLinesNoMarkers, xlXYScatterSmooth, _
  xlXYScatterSmoothNoMarkers, xl3DLine, xlLine, xlLineMarkersStacked, xlLineStacked
  ChType = "XY"
Case 51 To 59
  ChType = "Bar"
Case Else
  ErrMsg = "This chart cannot be adapted in this way": GoTo ErrExit
End Select
Application.ScreenUpdating = False
For I = 1 To UBound(Ser.Values)
  With Ser.Points(I)
    On Error GoTo Skip
    If Ser.Values(I) >= 0 Then MarkCol = RGB(0, 255, 0) Else MarkCol = RGB(255, 0, 0)
    If ChType = "XY" Then
      .MarkerForegroundColor = MarkCol
      .MarkerBackgroundColor = MarkCol
    ElseIf ChType = "Bar" Then
      .Border.Color = MarkCol
      .Interior.Color = MarkCol
    End If
  End With
Skip:
Resume Next
Next I
Application.ScreenUpdating = True
Exit Sub
ErrExit:
  MsgBox ErrMsg, vbOKOnly, "Negative red"
End Sub

Regards

Petr Bezucha

The timestamp for an SMS message can be determined either by the mobile network the device is connected to, or by the device's internal time.

 

If the operators's network is set to time zone GMT+0 and the handset is set to use the operator's network time then the timestamps may be offset from the time on the customer's device by the same number of hours the operator's network time zone is offset from the actual local time zone.

 

The operator can also prevent the issue by setting the time zone of their network to the local time zone in which it is operating.

Full Justification - Outlook - Microsoft Community

halmemphis say:

Full Justification - Outlook

Hi.  Is full justification possible in Outlook 2010?  Thank you.

graham mayor say:
You can if you insert a table cell without borders, then format the text in Word and paste it into the cell. The justification you set should remain.

It is not possible to justify text in the main body of the message as the text will wrap to the window.

Frankly if you want to send out formatted messages you should either create them completely as html or (simpler) format in Word and send as an attachment in PDF format which will ensure the recipient sees what you intend.
