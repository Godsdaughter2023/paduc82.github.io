florencemyhre-walton say:

How to recover the content of a document on Word 2010

By accident I deleted the content of an important document which I saved on Word 2010. If I search for the file I find it, but when I open it all I can see is a blank page. I saved the document various times, at least 5/6 times, under the same name. I was wondering, is it possible to recover one of the previously saved files?

jay freedman say:

It's unlikely that you'll be able to recover anything, with two exceptions.

#1

In the File > Options dialog, on the Advanced page, scroll down to the "Save" section. If the box for "Always create backup copy" is checked, then open the same folder where the document is saved and look for a file named "Backup of <document name>.wbk". If that file exists, it's the version of the document as it was just before the last save.

If that box is not checked, then Word hasn't saved anything you had before the last (blank) save.

#2

Open the blank version of the document. Press Alt+F9. If the text appears, then you somehow got it all enclosed in a field. To get it out, select everything except the curly brace at the beginning and the curly brace at the end (these are field markers). Cut that selection to the clipboard, delete the braces, and paste the text back into the document.

If neither of these possibilities works out, you can try doing a search of the entire drive for a word or phrase that would be only in that document. You just might find a temporary file that Word left behind, although it isn't supposed to do that.

After trolling this site I found my answer here after doing a few minor tweaks:

http://answers.microsoft.com/en-us/office/forum/office_2007-word/is-it-possible-to-convert-a-batch-of-doc-files-to/8f5549be-8143-4f0a-8f82-06c855fcb092

Sub SaveAllAsPDF()

Dim strFilename As String
 Dim strDocName As String
 Dim strPath As String
 Dim oDoc As Document
 Dim fDialog As FileDialog
 Dim intPos As Integer
 Set fDialog = Application.FileDialog(msoFileDialogFolderPicker)
 With fDialog
     .Title = "Select folder and click OK"
     .AllowMultiSelect = False
     .InitialView = msoFileDialogViewList
     If .Show <> -1 Then
         MsgBox "Cancelled By User", , "List Folder Contents"
         Exit Sub
     End If
     strPath = fDialog.SelectedItems.Item(1)
     If Right(strPath, 1) <> "\" Then strPath = strPath + "\"
 End With
 If Documents.Count > 0 Then
     Documents.Close SaveChanges:=wdPromptToSaveChanges
 End If
 If Left(strPath, 1) = Chr(34) Then
     strPath = Mid(strPath, 2, Len(strPath) - 2)
 End If
 strFilename = Dir$(strPath & "*.doc")
 While Len(strFilename) <> 0
     Set oDoc = Documents.Open(strPath & strFilename)
     strDocName = ActiveDocument.FullName
     intPos = InStrRev(strDocName, ".")
     strDocName = Left(strDocName, intPos - 1)

'This instruction converts to PDF 

       strDocName = strDocName & ".pdf"
     oDoc.SaveAs FileName:=strDocName, _
         FileFormat:=wdFormatPDF
        
     oDoc.Close SaveChanges:=wdDoNotSaveChanges
     strFilename = Dir$()
 Wend
 End Sub

Here are a few things to check:

Press Ctrl + Alt + Del together then choose Task Manager.

Click the Startup tab in Task Manager.

If PowerPoint appears there, rightclick it and choose Disable.

Then ...

Click the Windows icon at lower left of your screen

Type START MENU and click Start Menu when it appears at the top of the list

Doubleclick the Programs folder in the window that appears

Doubleclick the Startup folder

Any programs/shortcuts/documents in that folder will automatically start when Windows starts; delete any you don't want.
