My XP laptop keeps sending "low disc space" on my D disc. - Microsoft Community

tim mary alicethompson say:

My XP laptop keeps sending "low disc space" on my D disc.

I have an XP HP pavilion laptop and I keep getting the message: "low disk space, You are running out of disk space on HP_Recovery Disk D".  How do I repair this problem?
lemp say:
This is a bit tricky to deal with.  The "HP_Recovery Disk" is supposed to be a hidden partition that contains all the files necessary to restore your computer to its "fresh from the factory" state.  Normally, you should not access this disk or delete any of its files.

Typically, the restore partition is set up to have just enough room to contain the restore files, so adding only a little bit more will trigger the warning. 

You will find several places on the Internet that describe how to turn off the warning (e.g. http://www.jvfconsulting.com/blog/89/How_To_Fix_You_Are_Running_Out_Of_Disk_Space_On_HPRECOVERY.html) but that's a bit like taking the batteries out of your smoke alarm because you don't like the beep.

Instead, you need to figure out what is writing files to D: and stop it, and -- preferably, but not absolutely necessarily -- delete those extraneous files from D.

  • Check to see that System Restore is not monitoring the D drive.  Right-click "My Computer", select "Properties", and click on the System Restore tab.  System Restore should be monitoring ONLY the C drive.  If any other drives are being monitored, click "Settings" and turn them off.

  • Check to see that Disk Indexing is running on the D drive.  Open "My Computer."  Assuming that the D drive is shown (it may not be), right click on its icon and select Properties.  On the "General" tab, the two boxes at the bottom should both be NOT checked.  If the D drive is not shown in My Computer, report that fact.

  • The most likely culprit is something like Norton 360.  Apparently at least some versions of Norton have an auto backup feature that by default writes the data backup to the D: drive, meaning the recovery partition, which in short order fills it up. You will need to either turn off the auto backup feature in Norton (or what ever it has) or set it to backup to a different drive, like an external drive or a DVD or CD (if your system is capable of writing to optical media).

    You cannot set Norton to write to anywhere on the C: drive, it won't let you.

    If the D: drive is already full, you will need to find the Backup files and move them (or delete) off the D: partition. Be careful not to delete the 'Recovery partition's data.

lemp say:
Because I have neither Norton nor an HP computer, I don't know which files are the Norton backup files and which files are the HP restore files -- that's one of the reasons that this was a tricky problem to deal with.

Maybe I'll be able to recognize them if I see them.

Do the following. 

  • Open a Command Prompt window (Start > Run > cmd > OK)

  • In the black Command Prompt window, type each of the following lines and press Enter after each line.  If you like, copy the lines (one by one), right-click in the Command Prompt window, select Paste, and then press Enter.

                         dir d: > "%userprofile%\Desktop\info.txt"
                         dir d: /AH >> "%userprofile%\Desktop\info.txt"
                         dir d: /AS >> "%userprofile%\Desktop\info.txt"
                         notepad "%userprofile%\Desktop\info.txt"
                         exit

Copy the contents of the Notepad document that opens and paste it into your reply here.  You can then close and delete info.txt, which you will find on your desktop.
