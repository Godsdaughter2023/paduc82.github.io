hansv mvp say:

You can use ([EndTime]-[StartTime])*86400

 

EndTime-StartTime yields the elapsed time as a date/time value. Multiplying with 86400 returns the number of seconds.

Or

DateDiff("s", [StartTime], [EndTime])

john p spencer say:

The DateDiff function can calculate the difference between two datetime values in seconds, minutes, hours, days, weeks, quarters, months, or years.

    DateDiff("s",StartTime, EndTime)

As long as the difference is less than 86400 seconds (0ne day), you can convert the seconds back to a time using the DateAdd function.

   DateAdd("s",DateDiff("s",StartTime, EndTime),0)

If the time in seconds is more than one day, then you need to use (x is time in seconds)

Hours: Int(X/86400)

Minutes: Int(X/60) Mod 60

Seconds: X Mod 60

justin cavida msft say:

Hi Naoma,

As an initial recommendation, we suggest reloading your Class Notebook Add-in. To do that, you can follow the steps below:

1. Open OneNote
2. Select File > Options > Add-ins
3. Class Notebook should be listed as an Active Application Add-in
4. If it is listed as an Inactive Application Add-in, select Go
5. In the next window, check Load Behavior. If it reads "Load at Startup", select Class Notebook then OK to enable Class Notebook Add-in
6. If you see this message: “Not loaded. A runtime error occurred during the loading of the COM add-in,” try the following steps

If the issue persists, try installing Class Notebook Add-in as an administrator:

1. Close OneNote
2. Download the Class Notebook Add-in
3. When prompted, select Save
4. After installation finishes, select Open folder. You’ll see the Class Notebook installer listed as ClassNotebook.Setup in your Downloads folder
5. Right click on ClassNotebook.Setup and select Run as administrator
6. When Class Notebook installation finishes, open OneNote. If you see the Class Notebook tab in the OneNote ribbon, you’re done. If Class Notebook is still not showing up in the OneNote ribbon, repeat the Load Class Notebook Add-in steps above

Lastly, try to repair your Office through Control Panel. Go to Control Panel > Under Programs, click Uninstall a Program > spot Microsoft Office, right-click on it and select Change. Go with Quick Repair or Online Repair accordingly to fix the problem.

If you can't still see the Class Notebook tab in the Ribbon, try again the reload steps above.
