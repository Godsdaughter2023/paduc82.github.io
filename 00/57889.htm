Hi Wade,

To disable password expiration for a user on Windows 10, kindly refer to the steps below:

  1. Press Windows key + R on your keyboard.
  2. In the run command box, copy or type lusrmgr.msc and hit enter.
  3. Click Users.
     
  4. Double-click the username that you want to disable password expiration.
     
  5. Put a check mark under "Password never expires".
     
  6. Click Apply and hit OK.
