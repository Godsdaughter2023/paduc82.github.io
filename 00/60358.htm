"Error Message: STOP: C0000221 Unknown Hard Error or STOP: C0000221 STATUS_IMAGE_CHECKSUM_MISMATCH"

The flashing power light can indicate that there was a hardware problem detected or failure in the Power-On Self Test (POST) which is run before Windows even starts to boot.  One of the above articles suggests that  a bad RAM module can cause this error.  This could be detected on POST and be the reason for the flashing light.  The last time I had a flashing light, I ended up an excuse for a new computer.

Assuming a RAM problem, you might try opening up your computer, removing the RAM modules, then re-inserting them back into their sockets (this might fix a dirty connector or oxidation problem).  If that doesn't help, and you have more than one RAM module installed, try removing one at a time and rebooting.  Your computer may run much slower on less memory but you may be able to isolate a bad module this way.

%USERPROFILE%/AppData\Local\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\LocalState\Assets

Rename each respectively settings.dat.bak and roaming.lock.bak

Restart your computer

Other things you can try.

Open Start, type: regedit
Right click Regedit
Click Run as administrator

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Authentication\LogonUI\Creative\S-1-5-21-826134325-3229497062-2753576501-1001

this will be your first user, if you got more change on all of them:

Change the Value data to 4

Click OK

Exit the Registry then restart.
