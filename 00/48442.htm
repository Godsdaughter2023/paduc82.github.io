Reboot after completing all the scans.  You may also want to try the free OneCare at http://onecare.live.com/site/en-us/default.htm and let it run all the options because that’s good maintenance (it will take some time to complete but can be done in the background) - and this one you can run while your other AV program is installed and running.

Then you will know you've tried everything in addition to Windows Update drivers.
