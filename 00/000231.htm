The following is a workaround:

  • Start Word.
  • Press Alt+F11 to activate the Visual Basic Editor.
  • Press Ctrl+G to activate the Immediate window.
  • Enter the following line, or copy/paste it from this post into the Immediate window:
     
    Application.CommandBars("Research").Enabled = False

  • The Research task pane shouldn't appear any more.

My Outlook periodically asks me to archive and to be honest without knowing what will happen to my email I'm reluctant to let it happen

My email is held on a server at head office

Does archiving remove it from the server? 

Is the archive just stored on my local PC (whichever one i'm using at the time)

brian tillman [outlook mvp 2007-2019] say:

Archiving moves items whose Modified data value is older than the date criterion you set in the archive settings from one data file to another, local data file.  That source data file can also be an Exchange mailbox.  So, yes, it can reduce the amount of space you're using on the server, and, yes, the archive will be on whatever PC you're using at the time.  The archived data won't be available on other PCs unless you copy the archive data file to a network share.

I do not believe IMAP data files support the archive function.  I don't know if Hotmail Connector mailboxes support archiving.
