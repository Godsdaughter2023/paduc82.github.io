The Created, Modified, Changed, and Accessed dates in the file properties are coming from the hard drive's file system itself, not actually a record stored with/in the file.

You'll want to use meta data stored in the file, rather than the hard drive.  Something like the embedded ID3 tags in the media file like "published date" or "released date". These are stored inside the media file itself.
