Jump to a Random Slide

This is the code to jump to a random slide within a given range. Give an action button an action of Run > Macro > randjump to use it.

To generate the random number use this formula:

Number =Int((highest number in range - one less that lowest number)*rnd + lowest number in range)

eg For a number between 4 and 10

number = Int(10-3)*rnd+4) ---OR -- Int(7*rnd +4)

So here's the code to jump to a random slide between 4 and 10

Sub randjump()
randomize
Dim Inum As Integer
Inum = Int(7 * Rnd + 4)
ActivePresentation.SlideShowWindow.View.GotoSlide (Inum)
End Sub

Most people will tell you random slide choice can only be done with vba. If vba isn't suitable for your use look at random powerpoint slides without vba


Shuffle Part of Presentation

You can use rnd in a similar way to shuffle a section of a presentation. Input the highest and lowest slide numbers to be affected.

Sub shufflerange()
Dim Iupper As Integer
Dim Ilower As Integer
Dim Ifrom As Integer
Dim Ito As Integer
Dim i As Integer
Iupper = InputBox("What is the highest slide number to shuffle")
Ilower = InputBox("What is the lowest slide number to shuffle")
If Iupper > ActivePresentation.Slides.Count Or Ilower < 1 Then GoTo err
For i = 1 To 2*Iupper
Randomize
Ifrom = Int((Iupper - Ilower + 1) * Rnd + Ilower)
Ito = Int((Iupper - Ilower + 1) * Rnd + Ilower)
ActivePresentation.Slides(Ifrom).MoveTo (Ito)
Next i
Exit Sub
err:
MsgBox "Your choices are out of range", vbCritical
End Sub

graham mayor say:
400 x 300 is a small picture and will need a lot of scaling to make it into a desktop background. When you make a bitmapped  image larger the same amount of data from the image has to go farther and the perceived result is almost inevitably poorer than the original. Better to start with a larger image with greater resolution.

There are several applications that can scale an image for use as desktop wallpaper e.g. Google Picasa, but it certainly makes things a lot better if you have a decent image to begin with.

rohn007 say:
I'll second Graham's points.

At the current picture size, all you will get are blurry blowups when you expand them to desktop size because each pixel will be "golfball sized".

As he said, you will have to use special software to "resize" the images.  The tools will calculate / guess / interpolate the color of pixels between existing pixels. But they cannot add more detail. So although the resized image won't be as blurry as simply zooming a small image, it won't be as clear and detailed as a picture taken originally at a higher resolution.
