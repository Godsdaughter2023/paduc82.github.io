jackpot. say:

Excel will treat the range A1:F2 as a relative range. Since you want this range to be constant through out the range where conditional formatting is applied you need to lock the range using abosolute referencing $A$1:$F$2

 

Below are the different reference styles. 


A1 Relative referencing. Both column and row will change if you copy or drag the formula.
$A1 The column reference is fixed and will not change
A$1 The row reference is fixed and will not change.
$A$1 Column and row reference are fixed.
