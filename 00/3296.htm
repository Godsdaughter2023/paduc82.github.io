daniel g. grau say:
Probably it's not multiple blank lines, but the effect of two paragraph attributes called "Space Before" and "Space After."

To modify them, select the problem paragraphs, then:

  • Select Format > Paragraph in the main menu bar and set the "Before" and "After" parameters to zero; or
  • Activate the Ribbon's Home tab and set the spacings to zero using the pink-arrow icons.

I should tell you that these attributes are a lot more convenient than manually adding blank lines, since the latter can fall at the end or the beginning of a page and create uneven spacing relative to the header or footer. Furthermore, they mess any "Keep with next paragraph" attributes you care to add to titles and subtitles (to prevent them from ever separating from the paragraph that follows).

If you start using them, as I think you should, use only one of them (Before or After) throughout your document. This will avoid addition of spacings in successive paragraphs.
