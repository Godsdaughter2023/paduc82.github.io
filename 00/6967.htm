      Jack Sons.

      ronrosenfeld say:

      To find the row number with the last non-blank cell, you can use:

      LastRow = Range("A1:BA60").Find(what:="*", after:=[a1], searchorder:=xlByRows, searchdirection:=xlPrevious).Row

      To find the row with the last non-ZERO cell, where a cell that contains a zero would be excluded, you have to look at each cell.  Blanks would also be non-ZERO.  Is that what you really want?
