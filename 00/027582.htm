Sub ScratchMacro()
'A quick macro scratch pad created by Greg Maxey
Dim oRng As Word.Range
Set oRng = ActiveDocument.Content
With oRng.Find

  .MatchWildcards = True

greg maxey say:

Graham is correct of course.  That was my last post of the night and I woke up thinking about it.  There is really no way to completely automate this, unless you are satisfied with the indescrimate process I first posted.

 

In addition to the first word of a paragraph, there are initials and common names e.g, B. Jones, British Petroleum, etc.  There is title case The Count of Monte Cristo. There are abbreviations Dr. Mr. Mrs. etc.

About all you can do is refine the code as your experience dictates.  You might start with:

Sub FindAndConvertUCaseToLCase()
Dim oRngStart As Word.Range
Dim oRng As Word.Range
  Set oRngStart = Selection.Range.Duplicate
  Set oRng = ActiveDocument.Content
  With oRng.Find
    'You probably don't want to find the first word in each paragragh.
    .Text = "([!^13])[A-Z]?"
    .MatchWildcards = True
    While .Execute
      'You will probably want to bypass initials (e.g., the A. in A. Smith)
      If Not oRng.Characters.Last = "." Then
        'You will probably want to bypass common abbreviations.
        Select Case oRng.Words(2)
          Case "Mr", "Mrs", "Dr"
            'Do nothing.
          Case Else
            With oRng
              .Characters.First.Next.Select
              Select Case MsgBox("Do you want to convert this character?", _
                                 vbQuestion + vbYesNoCancel, "Convert ??")
                Case Is = vbYes
                  .Characters.First.Next = LCase(oRng.Characters.First.Next)
                  .Collapse wdCollapseEnd
                  Application.ScreenRefresh
                Case Is = vbNo
                  'Do nothing.
                Case Else
                  GoTo lbl_Exit
              End Select
            End With
        End Select
      End If
    Wend
  End With
lbl_Exit:
  oRngStart.Select
  Exit Sub
End Sub
 

suzanne s. barnhill say:
A different approach that does not use Find and Replace is just to select the entire document, go to Format | Change Case, and choose "Sentence case." This will result in only the first word of every "sentence" being capitalized. The drawbacks to this approach are that (1) proper nouns and adjectives within sentences will need to be re-capitalized and (2) pretty much every period will be treated as the end of a sentence and the next word capitalized (though this may be subject to any exceptions you've entered in Tools | AutoCorrect Options). So in the end this might require so much cleanup work that just stepping through the document with Find/Replace might be more efficient.
