erdeslawe say:

Windows 10 doesn't include OCR (Optical Character Recognition) Software; and nor has any previous version of the Windows OS.  

Multi-Function Printers sometimes come with an included OCR application, which has to be installed as part of the Printer setup process and your Printer seems to be one of those, but the software provided with the printer must be relatively old, given the age of the Printer Model, so it's a bonus if it still works with Windows 10.

If the OCR software previously worked with an earlier version of Windows, then it's possible that Windows 10 has identified that it is not directly compatible with the OS and has either not re-installed it after upgrade, or it was re-installed but can't be recognised because of a compatibility problem.  You could possibly try reinstalling the OCR software in (Windows 7?) Compatibility Mode to see if it will work. Right-click on the software installation file and select the Troubleshoot Compatibility Option.
