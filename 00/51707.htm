kerry waterfield say:
You mentioned you got the key by looking it up on the web? Any key you find there is likely no good by the way. The factory Toshiba OEM Windows 7 key will only activate via OEM-SLP activation, which is setup by the factory install. OEM-SLP keys will not activate online. They are purposely blocked from online activation. The original factory install activated against a SLIC table in the BIOS, not online. To activate your PC online you need to use the product code on the COA sticker. If you can't see the sticker remove the battery and look in the battery compartment. The key on the sticker has to match the currently installed version or it will not work.
