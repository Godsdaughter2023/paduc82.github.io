Go to File > Options > Customize Ribbon, and click the Customize button in the bottom left corner to open the Customize Keyboard dialog.

In the dialog, select "Macros" near the bottom of the Categories list. In the other list, which is now a list of the available macros, click one of the names. If there is anything in the Current Keys box, that will be the shortcut currently assigned to the selected macro. In any case, you can click in the "Press new shortcut key" box and type the shortcut you want. Make sure the "Save changes in" box says Normal.dotm. Then click the Assign button. You can repeat this for each of the other macros, and finally click the Close button.

When you next exit Word, if you're prompted whether to save changes in the Normal template, click Yes.

If the shortcuts again don't work after exiting and restarting Word, there is probably a problem with saving the Normal template, such as incorrect permissions. If that's the case, see these:

deanh say:

As mentioned before, ensure that the headings have the appropriate style assigned and that these styles have the appropriate Outline level assigned.

Also check that the TOC is set to capture down to the 5th level.

Select the whole TOC, press Shft-F9, what code to you see? Probably something like "1-3" within the field, if so change the 3 to a 5, then press F9 to refresh the entire TOC.

If you see something different, probably style names, it would be best to create a new TOC, replacing the old one, and change the "Show Levels" option on the Table of Contents tab from the default 3 to 5.

Seehttp://www.shaunakelly.com/word/toc/CreateATOC.html for an easy guide to create a TOC.

If some headings do not show up in the TOC, as per the start of this posting and check the styles of these headings that are not showing up.

Hope this helps
