techie007l say:

    If you used the Media Creation Tool to write the Windows 10 ISO's contents to your USB drive, you'll need to boot your computer from the USB drive.  Some computers will automatically do this if you insert the drive and reboot.  With others, you'll need to call up the BIOS boot menu and ask your computer to boot from the USB drive.  The hotkey for this usually is [F12] (although newer HPs tend to use [Escape]), which you would press repeatedly as soon as you power on your computer, or as soon as your computer starts rebooting.  In rare cases, you'll have to enter BIOS setup and actually change the boot order.  Rapidly pressing [Escape], [F1], [F2], [F10] and [Delete] works on most computers to enter BIOS setup.  Every once in a while that doesn't work and I'll actually have to Google how to do it for that particular model.

    If you just copied the ISO file to a USB drive, that's not quite going to work.  Before it will work, you will need to extract the contents of the ISO file to a folder on your HDD (7-zip can do this, but it will require administrative privileges to install), format your USB drive in FAT32, copy all the extracted files back to the USB drive.  When you're finished here, the root folder of the USB drive should have four folders and four files (including setup) on it.  Next, you will need enable the bootable flag on the USB drive (you will need administrative privileges here too):

  1. Right-click Start and click Command Prompt (Admin).
  2. Type the following command and press [Enter]: diskpart
  3. Type the following command and press [Enter]: list disk
  4. A list of all the storage devices connected to your computer should appear.  Identify your USB drive.  It will probably be Disk 1 or Disk 2.  You can usually tell by looking at the disk sizes.  If you're still not sure, you can unplug/replug and repeat the list disk command to see which one appears and disappears.
  5. Once you have identified your USB drive, type the following command, replacing the number with the USB drive's number, and press [Enter]: select disk 1
  6. Type the following command and press [Enter]: active
  7. Type the following command and press [Enter]: exit
  8. Close the command window.  The USB drive is now ready to use as a Windows 10 installation disc.  If step 6 fails, the USB drive was partitioned incorrectly (either by the manufacturer, or when it was first formatted).  This usually only happens with older USB drives; but it can be fixed, so let me know if it happens.

pranavmishra5582 say:

Hi,

Networks without an active security protocol are identified by network name (SSID) only, or by indicating Unsecured or No authentication. SSID is service set identifier, is the name of your wireless local area network. Networks with an active security protocol indicate the type of encryption next to the SSID, or by an icon such as a small lock. When a network broadcasts its name or SSID, it will appear in a list of available networks on all computers or devices within its range.

Follow the steps given below to connect to a secure network on your Windows 8 computer.

  1. Press Windows Key + C to bring up the Charms Bar.
  2. Click on
  3. In the WiFi section:
    • Hover your cursor over a network name to view a pop-up that lists the security type.
    • If that does not work, right-click a network and select View Connection Properties. The current security type is displayed in the Security type drop-down list.
    • If the security type is something such as WEP or WPA2, your network is secured.
    • If the security type is something such as Unsecured or No authentication (open), your network is not secured.

WEP uses static 64-bit or 128-bit encryption keys, which can protect every device on the network. WPA is a stronger form of encryption than WEP; WPA2 is considered an even more secure form of wireless secure.
