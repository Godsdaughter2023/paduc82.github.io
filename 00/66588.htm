This method resolves issues where the currently loaded USB driver has become unstable or corrupt.

  1. Type Device Manager in the search box, then press Enter.
  2. Double-click Universal Serial Bus controllers to expand the list. 
  3. Right-click the USB external hard drive with the issue, and select Uninstall.
  4. After the hard drive is uninstalled, unplug the USB cable.
  5. Wait for 1 minute and then reconnect the USB cable. The driver should automatically load.
  6. Check for the USB drive in Windows Explorer.

Note: Connecting your USB external hard drive into a non-powered USB hub can cause a lack of enough power to operate the external drive. Instead, plug it directly into your computer.

If this method does not resolve your issue, proceed to Method 2.

Method 2: Reinstall USB controllers.
This method resolves steps where the currently loaded USB driver has become unstable or corrupted.

  1. Type Device Manager in the search box, then press Enter.
  2. Expand Universal Serial Bus controllers. Right-click a device and select Uninstall.
  3. Repeat for each device.
  4. Once complete, restart your computer. Your USB controllers will automatically install.

If your problem still exists, proceed to Method 3.

Method 3: Disable USB selective suspend setting.
This method prevents your USB external drive from powering down.

  1. Type Power plan in the search box, then press Enter.
  2. Click Change advanced power settings.
  3. Select the box to expand USB Settings > USB selective suspend settings.
  4. On the Settings, select Disabled.
  5. If you're using a laptop, select Battery, select the drop-down menu, and then select disabled.

This method updates your motherboard's chipset drivers, so your computer will recognize your USB external hard drive.
