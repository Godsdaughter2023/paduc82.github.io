Excel & Identical Data - Microsoft Community

fabledmyth say:

Excel & Identical Data

What I want to do is, have Excel automatically identify when 2 (or more) cells in a range contain the same (identical) data.  So, if I am entering numbers in a row of cells and in one cell I enter, 2525, and then farther down the row I again enter 2525, some note or "!" or something pops up to let me know this number has already been entered.
jeeped say:
... if I am entering numbers in a row of cells and in one cell I enter, 2525, and then farther down the row I again enter 2525, some note or "!" or something pops up to let me know this number has already been entered...

  1. Select the entire column, (let's call it column A) and then select Conditional Formatting, New Rule from the Home tab's Styles group.
  2. Select Use a formula to determine which cells to format then supply one of the following formulas in the Format values where this formula is true text box,
            =COUNTIF(A:A,A1)>1                                            ◄ any duplicate value
            =AND(ISNUMBER(A1),COUNTIF(A:A,A1)>1)           ◄ duplicate numbers only
      Note that you may have to change the A:A and A1 references if the column you are using is not column A.
  3. Click Format and select some distinct formatting. I suggest a Red Fill.
  4. Click OK to save the formatting and OK again to save the rule.
If you enter duplicate values into the column A, both the original and the duplicate will be formatted with a red fill.
