Thank you for replying.

 

In the IMAP account setup, the data such as Contacts and Calendar will not be synchronized between the Webmail and the email client (Outlook). You will be able to synchronize emails and email folders only. This is the limitation of IMAP accounts.

 

When you configure a Hotmail or a Windows Live email account using Exchange Active sync, you will be able to sync all the data in Outlook as well as the webmail.

guruprasad ra say:

Hi Carlo,

 

As suggested in the earlier reply, when you use an IMAP account in Outlook and receives a calendar invite or you update the local contacts, it will not be synchronized with the IMAP contacts / calendar folders present in the Outlook but it will be saved in the local outlook data file (PST). This is how the IMAP protocol works. Hence you will not be able to see this data on the IMAP webmail.

 

Please refer the following link for more information on the email account:
