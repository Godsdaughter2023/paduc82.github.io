cbarnhorst say:

The key is there.  It is the same key for everybody and you can view it with the latest versions of programs like Belarc and Keyfinder.  

Windows 10 introduces a new licensing technology, called the "Registered Windows 10 Device."  That is why all the hoopla over upgrading before clean installing.  

The computer ("device") itself has to be registered by upgrade setup.exe.  Why it has to be an upgrade the first time is discussed below.

Once a registered device always a registered device so subsequent installations, including clean intallations, don't require a product key.  

Upgrade setup needs to determine who owns the original Windows license (for legal reasons) so the successor Windows 10 license inherits the correct rights.  Upgrade setup.exe then generates the Windows 10 license showing who the license is being granted to.  

To read the ownership of a Windows 10 license, run winver and read the bottom of the doc that appears.  In the case of original OEM Windows, the computer manufacturer was granted the original license and Microsoft does not have the legal right to change the ownership during the upgrade.  That gets recorded when the "device" is registered so after that the correct license will be generated even during a clean install.  

That is why Microsoft is so keen about the initial upgrade and is not going to change their minds.  It's a legal issue, not a software issue.

The bottom line is that you have a generic product key that is not much more than a placeholder in the system and there are no product keys for Microsoft to distribute like they do for retail copies of the software.  The real licensing and activation functions are now bound up in the Registered Windows 10 Device process.

There is another method that can be used to set your Documents/Pictures/Videos folders to be in OneDrive.  To be clear: this doesn't mean they are only in OneDrive.  They will still be in a folder on your computer's hard drive, available offline and such. But they'll be inside the OneDrive folder so that any changes will get synced to the cloud.

This article does a great job outlining this alternative method for accomplishing the same task.
