  12. Click OK.

Method 3: Make sure that there are no commas in the PDF File Name.

If the PDF file name includes commas or any special notations, it will be created to 0 Bytes size and you won’t be able to find it in the saving folder. This is the reason why Microsoft Print to PDF not working.

Method 4: Set the Microsoft Print to PDF as the Default Printer.

  1. Type Control Panel in Search.
  2. Choose Hardware and Sound.
  3. Select Devices and Printers.
  4. Right-click on Microsoft Print to PDF.
  5. Select Set as default printer.

Method 5: Change the Output Directory for PDF files.

There are times that an issue with the default browsers affects the Microsoft Print to PDF function. No notification or PDF reader opening when printing files to PDF but the output goes directly to the root of C:\Users\%username%\%internetDocName%site%.pdf. Use a different browser to choose or change the output folder.

Method 6: Update the Microsoft Print to PDF driver.

  1. Type Control Panel in Search.
  2. Choose Hardware and Sound.
  3. Select Devices and Printers.
  4. Right-click on Microsoft Print to PDF.
  5. Select Remove device.
  6. Click Yes.
  7. Right-click on the blank area on the Devices and Printers window.
  8. Select Add devices and printers.
  9. Choose The printer that I want isn’t listed.
  10. Select Add a local printer or network printer with manual settings.
  11. Click Next.
  12. Under the Use an existing port, select PORTPROMPT: (Local Port).
  13. Select Microsoft as the Manufacturer.
  14. Select Microsoft Print to PDF.
  15. Choose Replace the current driver.
  16. Hit Next.

Basically, in order to use touch gestures (pan and zoom) in OneNote, you have to open OneNote in the monitor that's set to primary. So if your touch monitor is set to secondary (to extend your display), OneNote touch gestures won't work on your touch monitor.

I think this is a grave oversight that hinders productivity. For people who use portable touch&pen monitors like Wacom Companion and Lenovo LT1423p, or people who attach their Surface to a big monitor and only want to use Surface as a secondary monitor, touch gestures won't work on the secondary monitors unless they are set to primary. Setting your much smaller and portable touch display to primary instead of your much bigger external monitor to primary is anti-productive and doesn't make sense, because programs launch on your primary monitor by default and you have to drag them to your bigger monitor every time. I really hope Microsoft can pay attention to this issue and allow OneNote touch gestures be used in secondary touch monitors.
