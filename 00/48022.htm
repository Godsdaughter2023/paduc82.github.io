How do you change the default Installation Directory in Windows 8

I have a SSD harddrive and a second larger hard drive. How do I change the default installation from my SSD to my larger HDD ?
srimadhwa b say:

Hi Damien,

 

When installing any program you will get an option to choose where the files needs to be saved. You need to click on “browse” option and choose the desired path or location and continue with the installation. Once you complete the installation the files will be saved in the new location rather than the default location.

 

By default Windows installs all the programs in the drive in which it is installed. In most cases it would be “C:\Program Files”.

You can always change the installation path for each program during its install. But if you install all your programs on a different drive, you might want to consider changing the default installation path.

To make it happen, Press Windows key + R and type “regedit” to open registry edit and navigate to the following key:

“HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion”

Right click on the value named “ProgramFilesDir” & click “modify” to change the default value “C:\Program Files” to the path you want to install all your programs in.

Click OK Exit.

Windows, by default, will now install your programs in the new drive or path.
