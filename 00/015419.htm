amit tandon say:

Not sure about your context, but below is how the array formulas function.

={1,2;3,4}

This array formula will enter the numbers 1,2,3 & 4 over 2 columns and 2 rows, and rest will display #N/A.

={1,2,3;4,5,6}

This array formula will enter the numbers 1,2,3,4,5 & 6 over 3 columns and 2 rows, and rest will display #N/A. The ";" symbol is used to split rows.

={1,1;1,1}

This array formula will enter the number 1 over 2 columns and 2 rows, and rest will display #N/A.

={1;1}

This array formula will enter the number 1 over all the selected columns and 2 rows, and rest will display #N/A.

Please note that The ";" symbol is used for rows and if you only enter the array formula ={5}, it'll repeat the number 5 across all the selected cells.
