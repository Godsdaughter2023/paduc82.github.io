patricia yu say:

Hi Abhi,

To resolve your concern, we suggest following the methods below:

Method 1: Run System File Checker.

  1. Press Windows + X, and then select Command Prompt (Admin).
  2. Type SFC /Scannow on the prompt, and then press Enter. This scan will repair system files.
  3. Restart the PC and then try to start Windows Firewall service.

Method 2: Allow Windows Firewall full control.

  1. Open Run by pressing Windows + R.
  2. On the Run dialog box, type REGEDIT, and then click OK.
  3. Follow this path:HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\SharedAccess.
  4. Right-click on ShareAccess and then select Permissions.
  5. Click on Add.
  6. On the Enter the object names to select field, type NT SERVICE\mpssvc.



  7. Click Check Names. MppSvc will appear on the field, and then click OK.



  8. Under Group or user names. select MppSvc, and then allow Full Control by clicking on the Allow checkbox under Permissions for MppSvc.



  9. Click Apply, and then OK.
  10. Try to start the Windows Firewall service.
