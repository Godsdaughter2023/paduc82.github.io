Hi,
Thank you for your interest in Microsoft Windows 10 Technical preview.
 
As you have aptly guessed, the issue that you are facing right now might as well be due to an outdated display driver.
However, as you have told that your machine is experiencing issues with internet connectivity, as of now we shall have to install the particular driver manually using the driver "inf" file.
 
Here's how to do it:
How To Update Drivers in Windows using "inf" files:
1.Locate, download, and extract the latest drivers for the hardware in a different machine that has an internet connection and transfer the files in any portable media (like a flash drive). You should always check with the hardware manufacturer first when looking for an updated Windows  driver. When downloaded direct from the hardware maker, you'll know the driver is both valid and the most recent available.
Note: If no drivers are available from the hardware maker, check Windows Update or even the disc that came with the computer or piece of hardware, if you received one.

2.Open Device Manager (of the machine that requires a Driver Update) from the Windows Control Panel. There are actually several ways to get to Device Manager in Windows but doing so from the Control Panel is pretty simple.

3.With Device Manager open, click or touch the > icon to open the category that you think contains the device you want to update the drivers for.

 
Tip: If you don't find the device you're after, just open some other categories until you do. Windows  doesn't always categorize hardware the way we think about a device and what it does.
  
4.Once you've found the device you're updating drivers for, right click or press and hold on the hardware's name or icon and choose Update Driver Software.
 The Update Driver Software wizard will begin, which will completely step through to finish the Windows driver update for this piece of hardware.
 
5.To the "How do you want to search for driver software?" question, click or touch Browse my computer for driver software.
 
6.Next, on the Browse for driver software on your computer window, click or touch Let me pick from a list of device drivers on my computer, located near the bottom of the window.
 
7.Touch or click on the Have Disk button, located on the bottom-right, under the text box.
 
8.On the Install From Disk window that appeared, click or touch the Browse button on the bottom-right corner of the window.
 
9.On the Locate File window you see now, work your way to the folder that you created as part of the driver download and extraction in Step 1.

10.Touch or click any INF file in the file list and then touch or click the Open button.

 INF files are the only files that Device Manager accepts for driver setup information and so are the only types of files you'll be shown.
If you find several INF files in one folder don't worry . The Windows Update Driver Software wizard loads information from all the INF files in the folder you're in automatically so it doesn't matter which one you choose.
 
11.Touch or click OK back on the Install From Disk window.
 
12.Choose the newly added hardware in the text box and then click or touch Next.
 
13.While on the Installing driver software screen, which should only last a few to several seconds, Windows will use the instructions included in the INF file from Step 10 to install the updated drivers for your hardware.
Note: Depending on the drivers you happen to be installing, you may be required to enter additional information or make certain choices during this process, but this isn't very common.
 
15.Once the Windows driver update process is complete, you should see a Windows has successfully updated your driver software window.

 Touch or click on the Close button. You can also now close Device Manager.

Hope this information was helpful.
Please get back to us with the updated status on the issue and we would be glad to assist you.
