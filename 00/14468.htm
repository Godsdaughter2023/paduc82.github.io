Microsoft has removed the connection to the OneNote popup when taking a screenshot ein the Windows 10 creators update, most likely this is not a bug but intended. Now the screenshot option is handled (hijacked) by the Explorer and the screenshot only lands in the clipboard.

It is possible to block the Explorer interference adding a certain registry entry which brings the old underlying OneNote function back. But I assume this will be blocked in a future Windows 10 update as well.

You could use the following command to add the mentioned registry entry, but I cannot guarantee there are no side effects and this workaround will keep functioning after future updates: 

reg.exe add HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced /v DisabledHotkeys /t REG_SZ /d S /f
