andre for directly say:
You can create a stop and start script for Windows Update.

Create a Notepad file with each of the following and save them with the .bat extension.

stop-updates.bat:

sc config wuauserv start= disabled
net stop wuauserv

sc config bits start= disabled
net stop bits

sc config dosvc start= disabled
net stop dosvc

pause

start-updates.bat:

sc config wuauserv start= auto
net start wuauserv

sc config bits start= auto
net start bits

sc config dosvc start= auto
net start dosvc

pause

Now you can simply right click on stop-updates.bat and select Run as Administrator


Or launch command prompt then do the following:

Stop the BITS, Cryptographic, MSI Installer and the Windows Update Services. To do this, type the following commands at a command prompt. Press the ENTER key after you type each command.

net stop wuauserv

net stop cryptSvc

net stop bits

net stop msiserver

How do I insert the word count into my document? - Microsoft Community

rodneyconner say:

How do I insert the word count into my document?

How do I insert the word count into my document?

a. user say:

Click where you want the count.

Click Insert-Field...

Select category "Document Information"

Select NumWords under "Field Names"

Click OK.

If you see something like { NUMWORDS \*Mergeformat }, that's the field code. You can toggle between field codes view and field results view using Option-F9 (Alt-F9). Many field results do not update automatically. To update the field when the word count changes, select it and press F9.

(You can also insert a field by clicking cmd-F9 to insert a pair of the special field code braces, and typing the field code, i.e. NUMWORDS, between them) 

NUMWORDS gives you Word's view of what the word count is. It's not what everyone expects.
