ririchiyo say:

Windows login A is an administrator, and B is also an administrator in Windows 7; However, I want B has less permissions that it just cannot alter group policies or using 'gpedit.msc' to change the system settings.

Windows has three types of accounts:

  • Administrators
  • Users
  • Guest

All administrators have the same rights. If B is an admin account then it can do the same things as A can.

The question is, whether you can use Group Policy to restrict the privilege of each account even an administrator, to perform different access and privilege practice, such as if this administrator account can perform allocating page files (which is already a control in the account policies).

So it is obviously possible to have administrators having distinctive rights rather than 'the same rights' because of the significance of Group Policy. Hence the question is not about the types of Windows accounts, is about the Group Policy to further maintain the privilege of the Windows users. The inquiry here is about the further control of Windows policies that whether some privileges can be restricted for some users to what extent. Again, not talking about the different Windows editions that Home/Basic users cannot access Group Policy at all.

If it is not possible, then it would be wise to use other approach like HIPS to make the similar result.

    Windows Media Player won't play CDs - Microsoft Community

    jimbob9292 say:

    Windows Media Player won't play CDs

    Windows Media Player has stopped playing CDs. It was working fine a few days ago but now when I put a CD into my laptop WMP recognises the disk - the tracklisting etc will be displayed properly, but when I select a track to play a red error 'x' appears next to it.

    I have now tried a number of CDs, incase the problem was the disc.

    If the disc has other features (videos, photo galleries, links to online content etc) then those all work fine - and WMP will have recognised it, have the tracks and artist info, but it will not play the tracks.

    The CDs are all regular CDs - official purchased CDs from my music collection not home burnt cds.

    I can still play mp3s etc.

    vinod n say:

    Hi,

    1.       Are you getting any error message?

    2.       Were there any recent changes made to the computer prior to the issue?

     

    I would suggest you to follow the steps mentioned below and check if it helps.

     

     

    Method 1: Try to run the Windows Media Player Settings troubleshooter:

    http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Settings-troubleshooter

     

     

    Method 2: If the issue persists then try to disable and enable Media Player and check.

    Step 1: Disable the Windows Media Player:
    a. Go to start and in the search type "Turn Windows features on or off".
    b. Click on "Turn Windows features on or off".
    c. Browse to the Media Features and uncheck Windows Media Player.
    d. Restart the computer.
     
    Step 2: Enable the Windows Media Player:
    a. Go to start and in the search type "Turn Windows features on or off".
    b. Click on "Turn Windows features on or off".
    c. Browse to the Media Features and place a check mark in front of Windows Media Player.
    d. Restart the Computer.

     

     For more information refer to the article:

    http://windows.microsoft.com/en-US/windows7/Turn-Windows-features-on-or-off

     

    Hope this information is helpful.
