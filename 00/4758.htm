The usual way is to use the DataObject, this requires a reference to "Microsoft Forms 2.0 Object Library". Set this reference manually in the VBA editor under Tools\References or execute this sub once:

Private Sub SetRef()
  On Error Resume Next
  'Microsoft Forms 2.0 Object Library
  Application.VBE.ActiveVBProject.References.AddFromGuid _
    "{0D452EE1-E08F-101A-852E-02608C4D0BB4}", 2, 0
End Sub

And paste the code below into a normal module.

Andreas.


Option Explicit
Option Private Module

Private MyData As DataObject

Function GetClip() As String
  If MyData Is Nothing Then Set MyData = New DataObject
  MyData.GetFromClipboard
  GetClip = MyData.GetText
End Function

Sub SetClip(ByVal S As String)
  If MyData Is Nothing Then Set MyData = New DataObject
  MyData.SetText S
  MyData.PutInClipboard
End Sub
