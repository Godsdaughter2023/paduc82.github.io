If you are using Windows XP (you posted in the XP forum), then you can.  This feature was removed in Windows 7 (I don't know about Vista).

In XP:

1.  Create the zip file by right-clicking a file and selecting Send to > Compressed (zipped) folder

2.  Open the zip file by double-clicking it

3.  Click File > Add a password

4.  Enter the password twice and click OK

Notes:

  • You will be able to open the zip file and see the files that are included in it.  You will be required to use the password to open the compressed files.
  • Zip file passwords are relatively easy to crack using readily available tools.
