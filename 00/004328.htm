hansv mvp say:

Remove the following from Preview_Template_Click:

     ' Create the Outlook session.
    Set objOutlook = CreateObject("Outlook.Application")

and

     Set objOutlookMsg = Nothing
     Set objOutlook = Nothing

for the Outlook variables are defined locally within CreateMessage.

In CreateMessage, remove the lines

     Dim objSubject As Object
     Dim objBody As Object

Subject and body of the message are text strings, not objects. Change the lines

     Set objSubject = objMI.Subject.Add(Subject)
     Set objBody = objMI.Body.Add(Body)

to

     objMI.Subject = Subject
     objMI.Body = Body

BTW, Preview_Template_Click can be simplified to

Private Sub Preview_Template_Click()
    Dim strPath As String
    Dim strFile As String
 
    ' Show the dialog box. If the .Show method returns True, the
    ' user picked at least one file. If the .Show method returns
    ' False, the user clicked Cancel.
 
    With Application.FileDialog(3)
        .Title = "Please select an Email Template"
        If .Show Then
            strPath = .SelectedItems(1)
        End If
    End With
 
    With Application.FileDialog(3)
        .Title = "Please select an Attachment"
        If .Show Then
            strFile = .SelectedItems(1)
        End If
    End With
 
    With Me.Recordset
        If .RecordCount Then
            .MoveFirst
            Do Until .EOF
                If CreateMessage(!Email, !Subject, !MainText, _
                        strFile, strPath) = False Then
                    MsgBox "Something went wrong!", vbExclamation
                End If
                .MoveNext
            Loop
        End If
    End With
End Sub

frankkenna say:

I actually had some trouble following those directions. But I did figure out another way to do it. For the benefit of anyone else trying to do this, here's what I did:

I created my list with the checkmarks:

✓1. Start the car

I selected the checkmarks and made them transparent. I set the text to appear at once (all together) on a mouse click. 

I then duplicated the text block. On this copy, I made the checkmarks solid and all the words transparent. I then set the animation to appear one line at a time. Then I placed this text bock directly over the first one. 

Result: 1st mouse click, the entire list appears without checkmarks. 2nd click, a checkmark appears next to the 1st item, 3rd click a checkmark appears next to the 2nd item, etc. 

This may not be the most elegant way to do this, but it was easy and it works.
