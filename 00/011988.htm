bob jones aka: cybertaz say:

You can't find it because there isn't one :-) You're trying to retrieve info that is external to the file. The built-in functions can't do that.

One approach:  If you're familiar with VBA you can add this code to a Module in the workbook to create a user-defined function:   

Function LastSaved() As Date
  LastSavedTimeStamp = ActiveWorkbook.BuiltinDocumentProperties("Last Save Time")
End Function

Format a cell with Date formatting of your choice & enter the function as =LastSaved() . No arguments are required between the parens. It will update each time the workbook is opened.

The way I've found to get this working is... (it's somewhat convoluted but bear with me)

CNAMES can only be added ONCE to the primary domain name and can also be added to sub-domains, so:

Firstly, select the domain you want to edit:

1) Create a sub-domain selector1.digitalsigns.com

2) Edit the DNS settings for THAT sub-domain.

3) Add the CNAME of selector1-mail-****._domainkey.digitalsigns.onmicrosoft.com

Then

4) Create another sub-domain selector2.digitalsigns.com

5) Edit the DNS settings for THAT sub-domain.

6) Add the CNAME of selector2-mail-****._domainkey.digitalsigns.onmicrosoft.com

SAVE the settings and wait.

Not quite. I would create an AgeSort table like you show with TWO fields. One for the sort order and one for the age range. 

I would have a field in your data table where you store the sort order. I would then add this table to your query joined in the sort order so that you can bring in the description into the query.
