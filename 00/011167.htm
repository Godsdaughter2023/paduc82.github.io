There’s no way to password protect a subsite or a document library. If you want to restrict some users from accessing the subsite and its content, you can, however, go to Site Setting page of the subsite and break permission inheritance.

When you break permissions inheritance for subsite and then define new permission settings, the subsite becomes a parent for items in it. Items under that parent now inherit the new permission settings (unless the items have uniquely defined permissions.)

To break inheritance and assign unique permissions, follow these steps:

  1. Navigate to your subsite.
  2. Click Settings>Site Settings.
  3. Under Users and Permissions, click Site permissions.

scottgem say:

I assume tbl1 is named Products and tbl2 OrderDetails or something similar. Use real names, not hypothetical ones.

First, you have to understand that you don't have fields on a form, you have controls that may or may not be bound to fields in a table. This is a subtle but important distinction. Second, you should NOT be storing ProductName in Products. You should have a ProductID field (preferably an Autonumber) and that is what should be stored in OrderDetails. 

The Productname control should be a combobox with a RowSource of:

SELECT ProductID, ProductrName, Price, Discount

FROM Products

ORDER BY ProductName

The Bound column should 1, the Column Count 4 and the Column widths should be 0;2;0;0

In the After Update of the combo should be:

Me.Price = ProductName.Column(2)

Me.Discount = ProductName.Column(3)

These are the main exceptions to the rule that data should exist in only one table. This is because this data is time sensitive. You need to capture the price and discount at the time of the order. So that, if you change the price and discount in the future it doesn't affect past orders. On the other hand, the product name is unlikely to change and if it does you probably want past orders to reflect the change. 

Also you should not have Total in Order Details. As a general rule we don't store calculated values. These can be calculated on the fly as needed. The expression (Quantity*Price)*Discount will calcuate the total wherever you need to display it.

ossiemac say:

I am assuming that the top left cell of each rectangle in within the same row as the cell in column A and that all rectangles on the worksheet are to be colored.

The following VBA code will color the rectangle the same as the interior color of the column A cell in the same row.

Ensure that you have a backup of your workbook before you run the code in case I have misinterpreted what you require and you finish up with something other than your requirements.

AFAIK the shapes can only be colored with RGB because standard color numbers do not to work but if someone knows how to do this without RGB then I am interested. The UDF (User Defined Function) creates an array of the RGB numbers so that the RGB parameters can be extracted from the interior color of the cell.

Sub ColorShapes()
    Dim ws As Worksheet
    Dim shp As Shape
    Dim r As Long   'for Row number of shape's TopLeftCell
    Dim arrRGB()
   
    Set ws = Worksheets("Sheet1")       'Edit "Sheet1" to your worksheet name

    For Each shp In ws.Shapes
        If shp.AutoShapeType = msoShapeRectangle Then
            r = shp.TopLeftCell.Row     'Get row number of shape's TopLeftCell
            arrRGB() = GetRGB(ws.Cells(r, "A").Interior.Color)  'Get the RGB array for Interior color
            shp.Fill.ForeColor.RGB = RGB(arrRGB(1), arrRGB(2), arrRGB(3))   'ForeColor of Shape
        End If
    Next shp
End Sub

Function GetRGB(lngColor As Long) As Variant
    'Returns an array with the RGB components of a color number
    Dim strTemp As String
    Dim arrRGB(1 To 3)  'One based array with 3 elements
   
    strTemp = Right("000000" & Hex(lngColor), 6)    'Convert number to Hex with 6 characters
   
    arrRGB(1) = WorksheetFunction.Hex2Dec(Right(strTemp, 2))    'Last 2 chars of Hex number converted to decimal
    arrRGB(2) = WorksheetFunction.Hex2Dec(Mid(strTemp, 3, 2))   'Mid 2 chars of Hex number converted to decimal
    arrRGB(3) = WorksheetFunction.Hex2Dec(Left(strTemp, 2))     'First 2 chars of Hex number converted to decimal
   
    GetRGB = arrRGB     'Assign the array to the Function variant
End Function
