qa guy commelec say:
These 2 queries will produce the results you want --
   CallRecRank --
SELECT dbo_AskCall6401.CallID, dbo_AskCall6401.customerId, dbo_AskCall6401.calldate, Sum(IIf([XX].[calldate]<=[dbo_AskCall6401].[calldate],1,0)) AS Rank
FROM dbo_AskCall6401, dbo_AskCall6401 AS XX
WHERE (((XX.customerId)=[dbo_AskCall6401].[customerId]) AND ((XX.calldate)<=[dbo_AskCall6401].[calldate]))
GROUP BY dbo_AskCall6401.CallID, dbo_AskCall6401.customerId, dbo_AskCall6401.calldate
ORDER BY dbo_AskCall6401.customerId, dbo_AskCall6401.calldate;

TRANSFORM First(CallRecRank.[calldate]) AS FirstOfcalldate
SELECT CallRecRank.[customerId]
FROM CallRecRank
GROUP BY CallRecRank.[customerId]
PIVOT CallRecRank.[Rank];

    stefan blom say:

    When you change the default options for spacing via the Design tab, Word modifies the Normal style, which you may not have used for all paragraphs. Click in a paragraph whose spacing did not change, and press Ctrl+Shift+S; see which style name is listed in the Apply Styles pane.
