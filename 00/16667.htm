bob jones aka: cybertaz say:
The reason is that the link is to a location on the sheet [the cell that contains the data], not to the data, itself. When you sort the 'master sheet' the data gets rearranged but the cells remain in the same location.

One option is to set the Excel> Preferences - Calculation to Manually. You can update the slave sheets individually as needed by using the Recalculate Sheet button on the Formulas tab of the Ribbon.
