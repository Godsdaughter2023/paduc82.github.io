I'm not suggesting that you should reduce the resolution :-) I mentioned the 72 ppi simply in reference to the image I was working with.

What you might try is opening the original JPEG in Photoshop, then go to Image> Image Size to change the Document Size (print dimensions) to what you need them to be - I'm guessing that the current dimensions are more than twice what you need. You can retain the 150 ppi resolution but the file will be considerably smaller. Also, by setting its Document Size in Photoshop you shouldn't have to scale it down in Word.
