I believe what you are seeing is an HP Security login (as opposed to a native Windows login) screen.

(has an icon, "other user", a box which would normally allow choice of Password/Fingerprint)

entering the [correct] password yields "(X) Invalid Pointer"

took a while, but I believe I have resolved this on an Win 7 pro HP 455G1 :

1) boot into Safe Mode

2) run MSConfig

3) select the Diagnostic startup option, on the General tab

restart....

you will get a "generic" windows login screen and be able to get to your desktop...but almost all services will be disabled and stopped, so not much will work

4) open Services.msc, find and double click on Windows Installer service, change to Manual, and Start it...

(this will allow you to actually uninstall programs)

5) Control Panel, Programs and Features

6) uninstall HP software: file sanitizer, drive encryption, device access manager, and finally Client Security Manager (it is the Security one that needs to be uninstalled, but the others have to be removed first) - I chose the "don't save stored configurations" option (as I believe this is what is effectively foo-barred (the "pointers" between user names, passwords, and stored fingerprint database settings)).

7) MSConfig again, change to Normal startup.

system reboot after those uninstalls and final msconfig, seems to be ok now. (no HP login screen, no fingerprint capability - only generic Windows password login).

new version download/installs of the HP Security Suite are available on their website, if you want to reinstall them.
