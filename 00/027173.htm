Word - Creating a gap in Page numbering - Microsoft Community

david lindsey (dlindsey) say:

Word - Creating a gap in Page numbering

HI everyone

I am working on a large report.

I need to leave a 16 page gap in the page numbering to insert some graphics, when the documented finally PDF'ed.

The document is large and constantly being edited. Therefore I cannot use a simple equation in the page number field such as { = { PAGE } + 16 }.

I need to reference the page number of the previous section (which an change) and add 16 pages to that.

Any ideas how I  can do this would be greatly appreciated.

jay freedman say:

You certainly can use the field code {= { PAGE } + 16 }, as long as you turn off the Link to Previous button in the section after the break.

Think of it this way: If you did nothing, the page before the break would be X (maybe 30), and the page after the break would be the next number (31). The result of the field {= { PAGE } + 16 } on the latter page would be 47. Bear in mind that as far as Word knows, that page's number is still really 31, which would be the result of the field {PAGE} alone.

Now you edit the first part of the document, and that section now ends on page 34. The page after the break has the real page number 35, and the {= { PAGE } + 16 } field displays the result 51. That's what you want.
