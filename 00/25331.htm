diane poremsky mvp (slipstick.com) say:

No, not a bug - you don't need compatibility mode with any of the Office apps and enabling it will cause issues. You also shouldn't use the compatibility troubleshooter because it will set outlook to win7 compatibility.

If Outlook is running as admin and the checkboxes aren't checked, you can look in the registry -

HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\AppCompatFlags\Layers and the same key under HKey_local_machine.  If there is an entry for Outlook, delete it (and delete any for BCM, if using).
