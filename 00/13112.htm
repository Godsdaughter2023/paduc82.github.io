If Target.Column = 3 Or True Or True Then

If True Then

Table positioning of page - Microsoft Community

suzannexn say:

Table positioning of page

I have 4 tables on page 5 of a document and I cannot get them to sit on the page where I want them.  there is no text between the tables (It is essentially a form that has been put together using multiple tables).  Does anyone have a good link to explain positioning of tables?
john mcghie say:
Suzanne:

Generally, tables sit in the paragraph they were created in, and are positioned by the properties of that paragraph.

The exception is if you adjust the Table Properties so the Text Wrapping is "Around".  When you do that, the table is encased in an invisible Frame, and positioned by the properties of the frame.

Once things go wrong, it can be difficult to determine which formatting properties are in control.  Under those conditions, I will often set the tables to Inline With Text (Text Wrapping = "None") so I can see where each table "wants to be".

Tables are just a special case of "Paragraph".  A table is a nested set of dependent paragraphs.

You could solve this one by constructing the tables as a set of "nested tables".  Begin with an outer table that encompasses the whole page.  Hide the borders so the readers don't have top see it.  Set up columns and rows to position each of the other tables.  Then you can drag the other tables into this outer table, one per cell.

Don't do too much of this: a Word document becomes fairly fragile with nested tables in it.  But that should get you out of today's difficulty.

Hope this helps
