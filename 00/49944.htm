jessen p say:

Hi,

 

Error Code 43 is a Device Manager error that means the following:

 

Windows has stopped this device because it has reported problems. (Code 43)

 

When you get this error with USB devices it’s because one of the drivers controlling the device notified the operating system that the device failed in some manner. This can be because of corruption in one of the loaded drivers and the driver cache just needs to be flushed. Unplugging the computer sometimes doesn’t resolve the problem because the battery still holds power.

 

To fix this, unplug all of your USB devices, then turn off the PC and take out the battery (This is assuming that you’re on a laptop), let the PC set for about 5 minutes, then put the battery back in and restart the PC. Next, plug your USB devices back in one at a time and make sure that they work.

 

If this does not work or you’re on a desktop, follow the steps in this KB article

Code 43: Windows has stopped this device because it has reported problems

Hello Schizo,

We regret the inconvenience. Let me assist you with some important information from here.

Please be informed that OEM licenses cannot be transferred from one PC to another. However, if it is a full Retail license key you are using, you can migrate the license easily.

If everything is in place and you have a retail copy of Windows 8.1 you can use the product key to activate the copy of Windows seamlessly on the new PC

Follow the below steps and see how it goes.

1. In the present PC, from where you want to migrate the licnese status from open command prompt(Admin).

2. Now, Right-click on cmd.exe from the result and select Run As Administrator.

3. User Account Control or UAC dialog box will appear, click on Yes.

4. The first thing we need to do is get our Activation ID, to do this use the following administrative command prompt and run:

                        slmgr /dlv

5. Windows will open a message box with information about the license status of your PC, take note of your activation ID.

6. To uninstall your product key you need to use the /upk switch, along with your

The command will look like:

                       slmgr /upk XXXXXXXX-XXXX-XXXX-XXXXXXXXXXXXXXXX

7. To install a different or the same product key again you need to use the following administrative command prompt and use the /ipk switch this time followed by the 25 character

                       slmgr /ipk XXXXX-XXXXX-XXXXX-XXXXX

The above steps needs to be performed because activation pairs your product key with your hardware configuration and helps verify that your copy of Windows hasn't been used on more computers than the Microsoft Software License Terms allow.

Also, once your device upgrades to Windows 10 using the free upgrade offer and activates online automatically, an entitlement is registered to your PC’s hardware for your edition of Windows 10. The entitlement is unique based on the hardware signatures which cannot be altered/changed from the Microsoft Server point of view.

Moving ahead, I regret to inform you that the newly built PC wont qualify for a free Windows 10 upgrade, as because the last deadline for free Windows 10 upgrade offer has expired on July 29th. So, the only option lies ahead is to buy Windows 10 Pro from Microsoft Store and install it afresh on the new PC.

a. user say:

Hello Yadec,

Thank you for posting your concern in Microsoft Community and welcome to the Windows 10 Family.

To be able to assist you better, let us know if the computers you are referring to came preinstalled with any operating system(OEM licensing). Because, if you have a fresh Windows 10 retail product key, the key wont be accepted if the PC has already got one OEM license in it.

Now, please be informed that once your device upgrades to Windows 10 using the free upgrade offer and activates online automatically, an entitlement is registered to your PC’s hardware for your edition of Windows 10. Swapping licenses wont be a feasible option anymore in this case. 

If everything is in place and you have a a retail copy of Windows 10 you can use the product key to activate the copy of Windows seamlessly.

The prerequisite in this process are as follows:

  • Both the PC shouldn't have any OEM licensing in it.
  • Ensure that you uninstall the product key first before you reinstall it in another PC, so that no two devices are active at the same time with the same license/product key.

Follow the below steps and see how it goes.

1. In the Search Bar type cmd.

2. Now, Right-click on cmd.exe from the result and select Run As Administrator.

3. User Account Control or UAC dialog box will appear, click on Yes.

4. The first thing we need to do is get our Activation ID, to do this use the following administrative command prompt and run:

                        slmgr /dlv

5. Windows will open a message box with information about the license status of your PC, take note of your activation ID.

6. To uninstall your product key you need to use the /upk switch, along with your

The command will look like:

                       slmgr /upk XXXXXXXX-XXXX-XXXX-XXXXXXXXXXXXXXXX

7. To install a different or the same product key again you need to use the following administrative command prompt and use the /ipk switch this time followed by the 25 character

                       slmgr /ipk XXXXX-XXXXX-XXXXX-XXXXX

Hope the above information is helpful. If you need further assistance, feel free to write to us
