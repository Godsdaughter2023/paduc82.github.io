john mcghie say:
The short answers are "No" and "No".  Sorry about that...

Word does not HAVE an "Auto Save" function.  What it does have is the ability to save the "changes" to a document.  This list of changes is no use at all if the original document fails.  When Word opens the Autorecover file, it discoverers that it cannot open the original so it has nothing to apply the changes to.

Unless you have manually performed a "real" save, your day's work never made it to the disc and so it is not there to be recovered.

If you look in Word>Preferences>Save, there is an option to "Always make backup copy".  I suggest that you turn that on.  If you do, you will see a file labelled "Backup of..." appear in the same folder as the document you are working on.

That file will be updated every time you manually save.  If you get into the habbit of hitting Command + s every time you pause to think, Word will update the current version of the file, and the "Backup of" file will contain the version as it existed at the previous save.

Both of these files are complete documents, not a simple list of changes.  So either can be opened independently, they do not depend on each other like Autorecover files do.

I would also suggest setting your Autorecover period to a value of 10 minutes or longer.  If you set this period too short, the operating system may not have released the autorecover file by the time the next autorecover save comes around.  This will cause lockups and freezes.

In the meantime, "something" is causing those freezes, and chances are, it's something in the document.  It would be worth finding out what it is and fixing it.

Before you begin, check that you have all of the most recent updates applied, and clean up your fonts to remove any old fonts that can get into mischief: See http://word.mvps.org/Mac/fontweeding.html

Chances are the freezes are caused by corruption.  That can be bullets and numbering, tables, pictures, citations... Allmost anything, really.  Do the updates and the font weeding, then come back and we can help you de-louse the document.

Cheers
