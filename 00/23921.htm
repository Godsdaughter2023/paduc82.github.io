jllatham say:

There is no formula that will insert rows or columns.

Here is a Macro that will take your list after you sort it and put a blank row at hour changes.  All you have to do is put it into a regular code module and then edit the two Const values to point to the column with your date/time entries in it and to the first row with a date/time entry in it.  Then you can run the macro from the [View] or [Developer] tab after you sort your list.

Instructions for placing the code into a regular code module can be found here:

http://www.contextures.com/xlvba01.html#videoreg

Here is the code:

Sub SeparateByHour()
  'note: 'Date/Dates means date time entry)
  Const firstRowWithDate = 2 ' change as needed
  Const columnWithDates = "A" ' change as needed
  'working values you do not need to be concerned with
  Dim lastRow As Long
  Dim rowPtr As Long

  lastRow = Range(columnWithDates & Rows.Count).End(xlUp).Row
  For rowPtr = lastRow To (firstRowWithDate + 1) Step -1
    If Not IsEmpty(Range(columnWithDates & rowPtr)) _
     And Not IsEmpty(Range(columnWithDates & rowPtr - 1)) Then
      If Hour(Range(columnWithDates & rowPtr)) <> _
       Hour(Range(columnWithDates & rowPtr - 1)) Then
        Range(columnWithDates & rowPtr).EntireRow.Insert
      End If
    End If
  Next
End Sub
