scottgem say:

Tempvars are a replacement for global variables. They can hold a variable's value beyond the scope of the current code module. The big advantage over tempvars to global variables is that they can be used in queries.

But they are not appropriate here because you are working with the the scope of the single form. 

A combobox is used to select from a list of values. That's why we questioned its use here. If you need to just allow the user to enter any date, then a textbox is the more correct choice. 

Ken gave AN correct answer. not necessarily THE correct answer. The point is that you use the After Update event of your control to assign its value to another control.

So the correct answer is 

Me.newcontrolname = Me.Currentcontrolname

obviously changing controlnames to suit. Ken's solution is more generic as it pulls the value from whatever the current control is,

I seem to have missed all the fun while I've been in town!  Now I'm back at the coalface, if you want to open a form specifically for entering new records only, then you don't really need to make a copy of the form and change its DataEntry property to True (Yes).  You can use the same form as you do for viewing/editing existing records and control how it opens.  Say you are opening the form from a 'menu' form with two buttons, one to view/edit records, the other to add new records, the code in the first button's Click event procedure would be like this:

   DoCmd.OpenForm "frmWorkOrder"

the code for the second button's Click event procedure would be like this:

DoCmd.OpenForm "frmWorkOrder", DataMode:=acFormAdd

The first would open the form to show all records, the second showing only an empty form in which you can add a new record.  You can then add further new records while the form is open if necessary.  There will be a macro equivalent to this, but never having occasion to use macros myself I'm not very familiar with their actions.  I think you'll find that by selecting the OpenForm action in the macro designer setting the data mode to add a new record will be pretty straightforward.
