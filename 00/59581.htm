jetta48 say:

Firstly, it is a very bad idea to back up files/folders to the same hard drive as the original. Hard drives can & do fail on a regular basis. An external USB hard drive is generally considered the most cost effective solution.

As for creating a sub-folder, the solution is the same for an internal or external hard drive. In Windows Explorer, select the parent folder, then right click and select New then Folder. Once the new sub-folder is created you can right-click on it and select Rename to rename it. You can then drag and drop or copy and paste the files/folders you want to back up into the new sub-folder.

There is no New-File option although you can right-click and select New then a data file for a program (for example Word Document). This is not the method to do backups though like you need - it is for creating brand new data files of the appropriate type.

stephenguo say:

How to share specific folder with specific user?

Hi,
I got one computer call Eight-Server and another called Eight-PC.
I want to share specific folder in Eight-Server with a user called Eight on Eight-PC, but I don't want other people on the homegroup network able to access it.
I tried permission tab in the file properties option in Eight-Server, but i was not able to find the user in Eight-PC for me to add him on to the permission.
In the location Select Users or Groups, I was not able to find Eight-PC.

Please help, I've looked everywhere online, can't find anything.

I tried creating same user on both computer and giving permission to //Eight-Server/Eight, it still doesn't work. Tried give permission to //Eight-PC/Eight, it said not find.

Please help.

Thanks.

jonathanlackman say:

This looks like kind of an old question, but I was just working through this myself. I got it to work by doing this;

  1. go to the PC that has the files you want to share (share FROM)
  2. right click on the folder or files
  3. you should get a menu asking if you want to share with the homegroup, if so, it should be complete at this point
  4. if you do not get this right click menu item, then the homegroup is not set up right. You need both PCs joined to the same homegroup and that needs to be working prior to sharing specific files.
