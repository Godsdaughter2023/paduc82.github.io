a. user say:

Hi,

Creating a new partition in Windows 10.

To create a new partition, the hard disk drive must contain unallocated space that is not part of an existing partition. Shrink the volume on an existing drive to create unallocated space, then create and format a new partition, using the following steps.

  1. In Windows, search for and open Create and format hard disk partitions.
  2. Right-click the Windows (C:) drive, or another drive on the computer.
    Do not partition volumes named System Reserved, Recovery, Factory_Image or HP_Tools. Do not use these volumes for file storage. Doing so might prevent you from being able to troubleshoot startup issues or perform a recovery in the event your computer encounters a major problem.
  3. Click Shrink Volume... from the drop-down menu to create unallocated space.
    Figure : Shrink volume
    The Shrink window opens.
  4. Without making any changes to the settings in the Shrink window, click Shrink.
    Figure : Shrink volume C:
    When the shrink process is complete, the new partition displays in the Disk Management utility as Unallocated space.
    Figure : Unallocated space
  5. Right-click the new partition, then select New Simple Volume... from the drop-down menu.
    Figure : Creating a new simple volume
    The New Simple Volume Wizard displays.
  6. Click Next.
    The Specify Volume Size window displays.
  7. Click Next to accept the default setting (recommended), or change the value in the Simple volume size in MB field, and then click Next.
    Figure : Specify Volume Size
    The Assign Drive Letter or Path window displays.
  8. Select a drive letter from the drop-down menu, then click Next.
    Figure : Assign Drive Letter or Path
    The Format Partition window displays.
  9. To name the drive, type a name in the Volume label field.
    Make sure Perform a quick format and Enable file and folder compression are not selected.
  10. Click Next.
    Figure : New Simple Volume Wizard
    The Completing the New Simple Volume Wizard window displays.
  11. Click Finish.
    Figure : Settings in Completing the New Simple Volume Wizard
    If an AutoPlay window displays, click the X in the top right corner to close it.
    The Disk Management utility shows the formatting progress, and then displays the New Volume.
    Figure : New volume and its size

Hope this helps. Thank you.

drunkenacid say:

When you removed the game, did you remove the save file that is stored locally on the Xbox? If so, it sounds like the save file might actually be corrupted on the cloud server that is syncing with your profile. If that is the case, unfortunately you will have to remove the save file from both the cloud server and your Xbox. This will however cause you to have to restart in your game.

How to remove both the cloud save and local save:

1. Go to you My Games & Apps

2. Find NBA2k17 and press the menu button and select manage game

3. Go to the save data, press the menu button witht he save data selected and hit Delete.

4. A window will pop-up asking if you want to delete everywhere or just from the console, select everywhere. This will remove all saved data from the cloud as well as the console.

I hope this helps!
