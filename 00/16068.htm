Ok, you are starting out with a PC with no operating system (os) which is just like a new PC.  The routine is rather simple -

1   On another PC go to Microsoft and download the Media Creation Tool

2   When using this download, ask to prepare for use on another PC and you will get a bootable USB

3   You will need at least a 4GB USB and current data will be deleted

4   On the dead PC, you will need to go to BIOS to change the load order to USB

5   Entry to BIOS can be different per manufacturer.  My HP uses the ESC key on a new start before the HP icon gets shown

6   Upon entry to BIOS, look for boot order and change to USB

7   Insert USB and call for a new start

8   You should see USB activity as the data gets transferred to Drive C

9   Go default for partitions choosing the largest one

10 You should  be asked to remove the USB

11  At this point I return to BIOS and reset the boot order back to Drive C.   In actual fact, without the USB it may default to Drive C but it must be changed some time

12  During the early install part, you will be asked if you want to do updates.  If you select updates, the final install should give you the latest public build

That should be about it.  You need a bootable device and you need to get into BIOS since you have no O/S.  If you are an Insider, just sign back to Fast Ring.

Your activation for W10 should stay on place since the PC  is already activated.  I swap out replacement hard drives on my HP fleet without issue.  Time will tell on this one.

This is a proven routine that I do often if I fry my Insider or any PC with a dead O/S.
