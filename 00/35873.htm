a. user say:

Hello Steve,

We have checked the error code that you have provided, and we found out that one of the possible reasons is a third-party anti-virus (if you are using one). In this case, we highly suggest disabling them first before troubleshooting this issue.

Once third-party anti-virus is disabled, let's repair the system at boot. Here's how:

  1. Reboot your computer.
  2. Press Shift key several times at boot and select Repair your computer.
  3. Then select Troubleshoot and click Advanced options.
  4. Now select System Restore. If this process go through, then error 0x80070091 should be gone.

If that didn't work, let's try another way in resolving this. Let us rename WindowsApps folder and perform the System Restore again, since the error code 0x80070091 indicates that the problem is with “AppxStaging” at “%ProgramFiles%\WindowsApps” folder. Here is how to rename the “C:\ProgramFiles\WindowsApps” folder:

  1. Press the Shift key and go to Power then Restart.
  2. Once your computer restarts, select Troubleshoot, then Advanced Options, and Command Prompt. Your computer will restart again.
  3. Choose an account (with Administrative privileges), type in the password (if any), and click Next.
  4. At the command prompt, enter the following command (hit enter after giving the following command):
    • diskpart
    • list volume
  5. Locate the Windows Drive letter. (It's where the Windows is installed).
  6. Type Exit to close the DISKPART tool.
  7. Navigate to the Program Files folder in the Windows drive, by typing this command:
    • cd / d D:\Program Files
  8. Give the following command to unhide the WindowsApps folder.
    • attrib -h WindowsApps
  9. The rename the WindowsApps folder to WindowsAppsOLD, by giving this command:
    • ren WindowsApps WindowsAppsOLD
  10. Then type rstrui to start System Restore.
  11. Follow the on screen instructions to restore your system.

NOTE: Once System Restore is successful, we suggest disabling Windows Updates until Microsoft correct the problem.

Let us know if you need anything else, and we will be happy to assist you.
