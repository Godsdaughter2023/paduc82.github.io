Lgross246, I am not sure what exactly you were trying with re-opening notebook etc. but you can use macros with Onetastic addin, which makes it much more simpler than XML parsing. Here is a macro to change page title:

 

<?xml version="1.0" encoding="utf-16"?>
<Macro name="Change Title" category="Title" description="Changes title of the current page" version="4">
  <For each="Paragraph">
    <Under a="Title" />
    <ModifyProp name="text" op="set" value="New title" />
  </For>
</Macro>

 

To use it you need to first install Onetastic addin (http://omeratay.com/onetastic) then import it by:

  1. Click on New Macro form Home tab
  2. Click Import
  3. Paste the XML
  4. Click OK
  5. Click Save

The macro will be available in the Home tab under the Macro group.

SUMPRODUCT will work only if no. of columns in second array is either 1 or has as many columns as in first array. Hence, E9:E43 will work as column is 1. But E9:F43 will not work as columns are 2. Your first array ABS9:ABY43 has 7 columns.

=SUMPRODUCT(ISNUMBER(FIND("PC-G",'New Updates - Version 3C'!ABS9:ABY43))*(('New Updates - Version 3C'!E9:E43=1)+('New Updates - Version 3C'!F9:F43=1)))
