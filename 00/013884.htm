I had difficulty visualising what you are describing.
Let's try and establish some terminology.

A single running instance of Excel has only one main window. 
Let's call that the Application window.
That instance of Excel may have multiple workbooks open.
The workbooks each have their own window, within the Application window. 
Indeed one workbook may have more than one such window.
One or more workbook windows may be visible within the Application window.

If you want to see 2 workbooks on the screen simultaneously you can do it in 2 ways:
- 2 instances of Excel, each with 1 workbook open
  (to open a second instance, start Excel again from the Start menu)
- 1 instance of Excel with 2 workbooks open and the workbook windows tiled within the application window

Does that help your understanding?

Brief instructions

Long instructions

  1. Quit Word
  2. Open the Terminal application in Applications/Utilities. You'll see an old-style terminal screen, much like the following:

  4. defaults write /Applications/Microsoft\ Office\ 2011/Microsoft\ Word.app/Contents/Info NSHighResolutionCapable True

  6. For Excel and PowerPoint, repeat the "Paste + Return" operation using the bold commands in steps #7 & 8:
  7. defaults write /Applications/Microsoft\ Office\ 2011/Microsoft\ Excel.app/Contents/Info NSHighResolutionCapable True
  8. defaults write /Applications/Microsoft\ Office\ 2011/Microsoft\ PowerPoint.app/Contents/Info NSHighResolutionCapable True
  9. Quit Terminal and test.

When you read messages from people you know, Outlook can show extra content from popular social networks. It is possible that the setting to Show public content from third-party networks is enabled in your account. To make sure that this is not causing the issue, please follow these steps and check if it will resolve the issue:

  1. Log in to your account.
  2. Hover your mouse to the Gear Icon on the top right corner beside your name.
  3. Select Mail more settings.
  4. Under Reading email, select Content from third-party networks.
  5. Under Showing public content, select Don't show public content from third-party networks.
