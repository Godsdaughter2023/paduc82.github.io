Hi David,

It is possible to do this in Office 365. An admin of Office 365 can change the room mailbox properties to define delegates who can accept or decline booking requests in Office 365 EAC (Exchange Admin Center). 3. In booking delegates, choose Select delegates who can accept or decline booking requests and add the manager as a delegate.

Set scheduling permissions for a room mailbox
