Something portable I have seen the BaseQI but those speeds are to slow.

patrick b.ludwig say:

The fastest you will get are usb-3.1 gen.1 based speeds as this is a general limitation on SB2. 

You will get these with external SSD Harddisks, for example the Samsung portable (up to 2 TB) and similar ones. 

For smaller solutions you could try USB flash drives with high specifications. 

You could also try SDXC cards or microcards with uhs-ii specifications, albeit not in the built in card slot. You will need a uhs-ii capable usb-3 adapter. This is included with some Lexar and Samsung drives. 

This is a good way to make use of the usb-c port on the SB-2.

You could also try a usb-c to GB Ethernet adapter (like the one from Microsoft) and ethernet based storage or even Wlan storage if you have a really fast router.

john john __ say:

It could be corrupt or it could just be that you lost permissions on the folder, the folder renaming scheme is explained here:

How To Restore a User Profile in Windows Server 2003

http://support.microsoft.com/kb/324734

If you are using XP Pro disable Simple File Sharing (Tools>Folder Options>View tab) and then right click on your profile folder and select Properties>Security and make sure that you have Full Control on the folder and all its child objects.  If you are using XP Home you have to log on to Safe Mode to access the Security Tab, or you can use the CACLS command at a command prompt to verify and grant yourself Full Control on the folder:

To verify your permissions:

cacls "C:\Documents and Settings\User Name"

You should have :F  listed for your account, if you don't:

cacls C:"\Documents and Settings\username" /t /e /g "user name":f

will grant you full control on the folder.  You must use "quotation marks" (as above) when  there are spaces in the path or in the user name.

If you do have full control on the folder then the profile is corrupt.

John
