amit tandon say:

Refer below image:

Formula in cell E2:

=IF(OR($D2>MAX($A$2:$A$100),$D2<MIN($A$2:$A$100)),"NA",IFERROR(INDEX($B$2:$B$100,MATCH($D2,$A$2:$A$100,0)),INDEX($B$2:$B$100,MATCH($D2,$A$2:$A$100,1))+(INDEX($B$2:$B$100,MATCH($D2,$A$2:$A$100,1)+1)-INDEX($B$2:$B$100,MATCH($D2,$A$2:$A$100,1)))/(INDEX($A$2:$A$100,MATCH($D2,$A$2:$A$100,1)+1)-INDEX($A$2:$A$100,MATCH($D2,$A$2:$A$100,1)))*($D2-INDEX($A$2:$A$100,MATCH($D2,$A$2:$A$100,1)))))

Formula is a bit long ... if exact revenue is matched, then the precise profit is returned. Else if revenue is inbetween 2 revenues, then proportionate profit level is returned which may not be average (will be average only if revenue is exactly midway 2 revenues). If revenue is less than min (ie. 200) and more than the max (ie. 585) then "NA" is returned.

joeu2004 say:

A26_andreas wrote:

A B
1 Revenue Profit
2 200 175
3 300 275
4 400 375
5 500 475

[....] I would need to do that for the rest (i.e. $250, $350, etc.) [....] So is there a way to calculate this with VLOOKUP?

Note that for those particular values, the relationship is linear; that is, of the form y = x - 25.  So you could use:

=FORECAST(X1, $B$2:$B$5, $A$2:$A$5)

where X1 is the revenue to "look up".

I realize that the values might change from time to time.  But if the relationship is always linear, that formula should suffice.

More generally, you indicate that you want to do pairwise linear interpolation.  In that case, you could use:

=IF(X1<$A$2, FORECAST(X1, $B$2:$B$3, $A$2:$A$3),

FORECAST(X1, INDEX($B$2:$B$4,MATCH(X1,$A$2:$A$4)):INDEX($B$3:$B$5,MATCH(X1,$A$2:$A$4)),

INDEX($A$2:$A$4,MATCH(X1,$A$2:$A$4)):INDEX($A$3:$A$5,MATCH(X1,$A$2:$A$4))))

Note that values less than A2 are assumed to be linear with respect to the A2:A3; and values more than A5 are assumed to be linear with respect to A4:A5.

Programming notes:

1. It would be better to calculate =MATCH(X1,$A$2:$A$4) in a helper cell, and replace MATCH(X1,$A$2:$A$4) with a reference to the helper cell in the formula above.  Your design may or may not allow for that easily.

2. We might simplify the formula by using OFFSET.  But that function is "volatile":  the formula and any directly or indirectly dependent formulas would be recalculated every time any cell in any worksheet is modified.  The form INDEX:INDEX avoids that.
