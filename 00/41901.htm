cbarnhorst say:

Have you considered downloading the Windows 10 Home recovery software from your Asus support website?

Have you tried to use the Asus recovery partition (if there is one)?  The website will explain how.

Have contacted Asus support for help?

There are several ways to remove a partition.  One is to use Disk Management (right click on the Start Button and choose it) in a computer to which you attached the SSD drive, internally or externally.  The second is to boot the computer with the installation media and when you reach the volume selection screen highlight the SSD and click on the red "X" in the drive tools.  Then select "New" in the same place.

If you have a Samsung SSD the Samsung tools has a "secure erase" which is even better.

However, none of this may work since the computer itself is what is registered as a Windows 10 device by edition.  I had this problem with my wife's all in one where we had to reinstall the OS.  The standard .iso file from Microsoft includes manifests for both editions and for some reason we could not get it to install Home.  What I am saying is that your issue probably does not relate to the SSD at all.  In previous versions of Windows it did, but not Windows 10.  Once Windows 10 Pro has been installed on a device it sometimes wants to reinstall Pro without offering the choice.  You may have to use the Contact Support app on the Start Menu (of another computer) to fix this.

But try working with the SSD first.  Strange things happen.

It does not look like it is booting properly from the Installation Media

If you have problems booting from a boot disc, you may have UEFI BIOS, go into your BIOS and make the following changes:
1. Insert the Bootable Media
2. Disable Secure Boot
3. Enable Legacy Boot
4. If Available enable CSM
5. If Required enable USB Boot
6. Move the device with the bootable disc to the top of the boot order
7. Save BIOS changes and restart your System
