Don't work in the Outbox view and you shouldn't have the problem.

If you open the Outbox, the first item is selected and that selection will change the date to none. You would have to open the message and send it again, with the Outbox view closed, for it to retain its date.

The following Outlook macro will force a send of items in the Outbox no matter what the date. The commented out section is optional.

Sub SendMessages()
Dim olItems As Outlook.Items
Dim olItem As Outlook.MailItem
Dim olNS As Outlook.NameSpace
Dim olSycs As Outlook.SyncObjects
Dim olSyc As Outlook.SyncObject
Dim i As Long
Set olNS = Application.GetNamespace("MAPI")
Set olSycs = olNS.SyncObjects
Set olItems = Session.GetDefaultFolder(olFolderOutbox).Items
For i = olItems.Count To 1 Step -1
    Set olItem = olItems(i)
    olItem.sEnd
Next i

For i = 1 To olSycs.Count
    Set olSyc = olSycs.item(i)
    'If MsgBox( _
     "Do you wish to synchronize " & _
     olSyc.Name & "?", _
     vbYesNo) = vbYes Then
    olSyc.Start
    'End If
Next
cleanup:
Set olItems = Nothing
Set olItem = Nothing
Set olNS = Nothing
Set olSycs = Nothing
Set olSyc = Nothing
End Sub
