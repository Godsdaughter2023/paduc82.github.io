froeba say:

Excel File Caching

Hello,
i receive the daily finance reports via email with an excel sheete attached. The file name of that attached excel sheet is the same.
When i open the attached excel file, excel (2011) shows me a old version of this file.
When i open the same attachment with Apple preview, the actual file is shown.
Why shows excel a old Version of this file? Is this a kind of file chaching? How can i disable these behaviour?

Best Regards
bob jones aka: cybertaz say:
The main cause of the problem is that you are repeatedly saving a file of the same type with the same name to the same location. You haven't detailed your workflow, but my guess is that it goes something like this...

When you hit the Save button in the message window the file is saved to the folder you have specified in Mail Preferences (I'm assuming you're using Apple's Mail app.). However, in the case of attachments, Wednesday's file does not overwrite/replace the one you saved Tuesday. Instead, the file name is retained but appended with the number 1, so you now have report-1.xlsx in the folder. When you d/l Thursday's it will be names report-2.xlsx & added to the same folder... and the beat goes on, day after day. If you open the folder the files are being save to [Mail uses
Excel knows only about the most recently opened file. If you're selecting report.xlsx from File> Open Recent you repeatedly open the same "version" of the file.

No offense intended, but if this sounds at all familiar it suggests that you'd do well to modify the way you work :-) If the files can't be delivered with unique names you might at least

  1. Right-click the attachment in the message
  2. Choose Save Attachment
  3. Specify a name & location for saving that version
  4. Use File> Open to open the file in Excel

Alternatively, move &/or rename each day's report. Or, if you don't need the previous version, delete it.


Whether you follow these suggestions or prefer some other method, each day's attachment has to be uniquely identified.

Surely you want the width of the column spacing to be the left margin plus the right margin? Dim m As Long

Turn off picture compression

To keep the maximum picture quality, you can turn off compression for all pictures in a file. However, turning off compression can cause very large file sizes without an upper limit on the size of the file.

  1. Click the File tab.
  1. Under Help, click Options, and then click Advanced.
  2. Next to Image Size and Quality, click the file that you want to turn off picture compression for.
  3. Under Image Size and Quality, select the Do not compress images in file check box.

 Note   This setting applies only to pictures in the current file or the file selected in the list next to Image Size and Quality.

 

 

Select ALL NEW DOCUMENTS next to Image Size and Quality if you want to apply this to your new documents automaticly
