Rather than post an off topic rant as a reply I'll type this up as a new thread. It started as a reply and expanded.

@Microsoft reviewers, there will be several topics mentioned here about the overall usability of the Operating System as compared to 7 which was excellent. I'm just listing off stuff as I think of it in one long post. I've tried to be detailed and explain my reasoning.

Background: I've been running 10 since it came out as a secondary partially useful system. I skipped 8 after I managed to tolerate it only for an hour. Maybe even a replacement for the old Control Panel as well ??

I got a Feedback notification question that asked me if I thought PC Settings would be a good replacement for Control Panel in 9841, so obviously yes, this is the direction MS is thinking of taking.

There sure is a ton of stuff MS need to do to replicate all of the functionality Control Panel offers, though.  I hope they don't force anything on us half-baked...  again...

I got the same popup and promptly told it that I prefer control panel over pc settings in a big way.

User friendliness and function have been removed from this environment across the whole system for the same of simplifying things. That's fine if you want to run an Apple computer. If I wanted a non-customizable dumb brick I'd buy an Apple. It's the same with their phones. I buy Android so I can control things and customize them.

That is the same reason I use Windows. And that is why the first thing I do on an install is go to control panel and go through each of the relevant items and change settings to how I use them. Things like manual updates, date/time format, pagefile location and making it fixed size, folder advanced settings like view extensions, view hidden etc. and so on.

How the hang do I set view extensions in PC Settings?!? Beyond that the way things are stuffed into sub menus with no idea of what is actually in them until you go and look at every single item is a complete waste of time. I'd likely never remember half of what was buried in there until I use the items several times each which with advanced options could take years. How often do you change folder view settings?

I go to control panel and set list view. BAM! there is everything I need all at once.

Stop stuffing useful features deep down where they can't be found.

On the topic of usefulness, I also have problems with the lack of informative feedback from the system. Examples from the last 24 hours: How long do these apps have left to update? Answer: bunch of flying dots across the top of the screen. How big is this build download and how long roughly will it take to download and install? Answer: bunch of flying dots in a circle (I think that's what it was. There was nothing at all if not.)

Yes, Explorer has a nice fancy copy progress bar graph and stats. I like that a lot and want to experiment with multiple transfers at once to compare graph speeds sometime.

But you've dropped useful data feedback from a lot of other areas. You've put settings for customisation down in the depths of sub-menus. And everything looks like Windows 3.1 in 16bit colours 640 resolution styled icons and no border windows. My laptop screen is not a phone screen. I can see more than 5 inches diagonal. Additionally, I don't use a touch interface but much of the text and buttons on my screen are spread out like I'm trying to hit them with fat thumbs so I don't get to mash more than one at once. I use a mouse. I have accuracy to something like 300dpi or more with thins thing. Most of the screen is wasted space with that approach.

I appreciate you are trying to merge desktop and touch systems into one platform. I see why, but I think it is a mistake to treat desktop systems the same as a touch environment where people want only a few Apple type big blocky tiles. Desktops are supposed to be the power platforms where the hard stuff gets done in several apps at once parallel. Facebook popups and notifications like on your cell phones are not really wanted in work environments where I'm moving avis and noise filtering audio... ooh somebody likes my profile picture. Sending a video across the network... my army is ready for battle.

My view of 10 is actually for the most part positive. In desktop mode, with the metro (“block world” as some call it) removed from the Start menu, things function mostly the way we expect a desktop to work. The Start right click option btw is handy, tho I haven't got used to going there instead of keyboard shortcuts and links I have already made. I'm on an old i5 laptop about 4 years old, 4gb ram and shared graphics. Things run smooth. I have not had any problems that aren't alpha/beta testing type issues. Most day to day things work as expected. Things do get rather slow and sluggish when doing multiple things at once (updating games, windows updates and firefox browsing at the same time). There are lots of good things and I want to express my appreciation for them even tho I can't think of them all to list them.

More useful feedback with the download/install type time estimates and percentage bars across the system.

Not stuffing configuration options way down in 3rd or 4th level sub-menus like they are now. I would like to be able to get to any setting within only 3 maybe 4 clicks total and I can do that with control panel linked on the Start menu.

Finally, the feedback app is half baked. That's not important to a RTM release without it I guess but its a problem for alpha/beta. Eg: I searched for an item for that progress bar stats issue I mentioned above and didn't know which one was appropriate.
