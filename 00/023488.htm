ken sheridan say: It should not be difficult to write a little function in Access 97 which addresses your task specifically.  Let's assume you have a table Products with primary key ProductID and a table Comments with foreign key ProductID and columns Comment and CommentIndex, the last being to indicate the order in which you wish the comments to appear in the string expression:

Public Function ListComments(lngProductID As Long) As String

    Dim dbs As DAO.Database
    Dim rst As DAO.Recordset
    Dim strSQL As String
    Dim strComments As String
   
    strSQL = "SELECT Comment FROM Comments " & _
        "WHERE ProductID = " & lngProductID & _
        " ORDER BY CommentIndex"
       
    Set dbs = CurrentDb
    Set rst = dbs.OpenRecordset(strSQL)
   
    With rst
        Do While Not .EOF
            strComments = strComments & ". " & !Comment
            .MoveNext
        Loop
    End With
   
    ListComments = Mid(strComments, 3)
   
End Function

You can then call the function in a query like the following for use as the report's RecordSource:

SELECT *, ListComments(ProductID) AS CommentList
FROM Products;
