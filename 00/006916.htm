How do 1 open a tsv file into excel? - Microsoft Community

dericksapien say:

How do 1 open a tsv file into excel?

How do I open a tsv file in Excel? 

ossiemac say:

Try the following.

  • Open a blank workbook or just go to a blank worksheet.
  • Select Data ribbon
  • Select "From text" icon in the "Get External Data" block at left of Data ribbon
  • Select the file from the dialog and then click Import
  • In the Text Import Wizard select "Delimited" (Probably will be default)
  • Click Next
  • Check Tab delimiter box only (all other boxes unchecked)
  • Click Next
  • The next dialog allows you to specify the type of data in the tsv columns. If you have any Dates then select the column of Dates at the bottom and then Select "Date data format". To the right of the date select the date format that is in the tsv file (NOT the date format you want to finish up with because Excel needs to know what date format is in the tsv file so that it can convert to your regional date format).
  • You can also select any other columns and set the data type. eg if you have numbers that are in text format and you want them to remain in text format so as to preserve leading zeros then select the column and set to Text format.
  • Click finish .
  • At this point the data is connected to the source. If you want to break the connection and keep the data then click "Connections" icon in the Connections block and Remove the connection.

How do Windows 8 Bing Apps sync across devices? - Microsoft Community

liffeys_mum say:

How do Windows 8 Bing Apps sync across devices?

How do Windows 8 Bing Apps sync across devices?
debbie_h say:

Windows 8 allows certain features and settings to sync across Windows 8 devices that are tied to a specific Microsoft Account.  Therefore, application settings and data may sync across your Windows 8 devices as well, depending on the App.

 

Bing Apps:

 

1.       Search history settings and SafeSearch settings DO NOT sync

2.       Search history will sync across Windows 8 devices and the web experience (Bing.com)

3.       Search history will only sync across to instances where Search History settings are ON

 

Bing Maps App:

 

1.       All application settings under the Options menu will sync across Windows 8 devices

2.       Search history and settings will sync across Windows 8 devices

3.       Application settings and search history DO NOT sync from the Windows 8 application to the web

          experience (Bing.com)

4.       Items saved on the web experience (Bing.com/maps) such as favorite places or saved places, will

          not sync into the Windows 8 Bing Maps app

 

Bing News App

 

1.       Topics, My News and Pinned tiles will sync across Windows 8 devices

 

Bing Sports App

 

1.       Favorite teams, along with the sports and leagues you have added to the application will sync

          across Windows 8 devices.

 

Bing Weather App

 

1.       Favorite locations, Units preferences (Fahrenheit/Celsius), Search History (On/Off), Pinned Tiles and

          default Home Location will sync across Windows 8 devices

2.       On the very first machine you run the Weather app, the app will always request your permission to

          use your location

3.       On subsequent machines where you sign in, the Weather app will not ask your permission to use

          your location again
