jeeped say:

Page numbers can be restarted (or continued) in new sections. After inserting a next page section break (Page Layout ► Breaks ► Next page section break), position the insertion point (aka Cursor) in the header or footer of the new section and go to Header & Footer Tools ► Page Number ► Format Page Numbers. The resulting Page Number Format dialog will have an option to Continue from previous section or Start at:.

 

If adding a new section (or modifying an existing one) is not an option, then the page number *can* be manipulated mathematically but the preferred method is through sections.

graham mayor say:
You can do it with a macro e.g.

Sub ReplyUsingTemplate()
Dim oAccount As Outlook.Account
Dim objItem As Outlook.MailItem
Dim oMail As Outlook.MailItem
Dim i As Long
    Set objItem = ActiveExplorer.Selection.Item(1)
    Set oMail = Application.CreateItemFromTemplate("D:\Path\TemplateName.oft")
    With oMail
        .To = objItem.SenderEmailAddress
        .Subject = "Re: " & objItem.Subject
        .HTMLBody = .HTMLBody & objItem.Reply.HTMLBody
        .Display
    End With
Set objItem = Nothing
Set oMail = Nothing
End Sub

graham mayor say:

I don't think it is possible to do what you ask, using a pre-configured template, but you could format the reply document and add text e.g.

Sub ReplyUsingTemplate()
Dim oAccount As Outlook.Account
Dim olInsp As Outlook.Inspector
Dim wdDoc As Object
Dim objItem As Outlook.MailItem
Dim oMail As Outlook.MailItem
Dim oRng As Object
    Set objItem = ActiveExplorer.Selection.Item(1)
    Set oMail = objItem.ReplyAll
    With oMail
        Set olInsp = .GetInspector
        Set wdDoc = olInsp.WordEditor
        Set oRng = wdDoc.Range(0, 0)
        oRng.Text = "This is your reply text"

        'Format oRng as required. The original message and sig will be retained.
        .Display
    End With
    Set objItem = Nothing
    Set oMail = Nothing
    Set olInsp = Nothing
    Set wdDoc = Nothing
lbl_Exit:
    Exit Sub
End Sub
