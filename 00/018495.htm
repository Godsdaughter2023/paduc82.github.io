There is a difference between having updates installed and applied.  To check if you have everything applied that you should have you can use msinfo32.exe to list Loaded Modules.  Sort by Version and see what you have for the latest version at the level that you think you are at.  For example it sounds like you are still only at IE8, so look for a group starting with 8.0 and see what version is the highest and lowest and compare with the version that you should have by looking at the File Information section in here (currently the latest cumulative security update).  If you are all up-to-date your Help About dialog (Alt-H A) will have a link besides Update versions: which would take you there too.

If you don't have your IE8 up-to-date security wise, then that should be your first focus, not trying to change the version of your IE at a lower level of security.  Chances are that the new version only started after a higher level of security had been reached and that is what your problem symptoms would be about.

If you add the caption to an object in the drawing layer (meaning that the Text Wrapping for the object set to something other than "In line with text"), Word places it in a text box.
