bruce hagen ~ mvp say:

The Address Book is a hidden file so you have two options.

1: To view these files in Windows Explorer, you must enable Show Hidden Files and Folders under Start | Control Panel | Folder Options Icon | View, or in Windows Explorer | Tools | Folder Options | View.

2: Do an Advanced search and include hidden files and folders and search for:  *.wab

If you unhide the files, the old path would be similar to this:

C:\Documents and Settings\Your User Name\Application Data\Microsoft\Address Book\Username.wab

When you find a good AB, open the Address Book and File | Export | Address Book (wab) to any place on your HDD that is easy for you.  Then, open the Address Book in the new identity and File | Import | Address Book (wab) and point to where you saved it.
