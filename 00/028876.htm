jeeped say:

... Can anyone tell me how I can get all of these conditions to work together?


With the limitations of Excel 2003's conditional formatting and the speedbump that the text in B3 puts into your conditions, you might want to consider formatting through VBA based upon a trigger on a cell's value changing.

Right click the worksheet's name tab and select View Code. When the VBE opens paste the following into the pane titled something like Book1 - Sheets1 (Code),

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim rngCondFormat As Range
    Set rngCondFormat = Range("D:F")
    If Not Intersect(Target, Range("B3")) Is Nothing Then
        lngFntClr = xlAutomatic
        If Range("B3") = "1mock" Then lngFntClr = 4  'green
        rngCondFormat.Font.ColorIndex = lngFntClr
    ElseIf Not Intersect(Target, rngCondFormat) Is Nothing Then
        For Each c In Target
            If Not Intersect(c, rngCondFormat) Is Nothing Then
                c.Interior.Pattern = xlNone
                If c > 0 Then
                    If c <= 5 Then c.Interior.ColorIndex = 5  'blue
                    If c < 3 Then c.Interior.ColorIndex = 3  'red
                End If
            End If
        Next c
    End If
End Sub

Edit the second line to address the range that you want to target with conditional formatting. In addition, I've used ColorIndex as the change mechanism so you can refer to the image I've supplied below to change the colors being assigned.
