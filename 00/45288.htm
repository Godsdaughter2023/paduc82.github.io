Hi Everyone,

I've got a Windows 8.1 with Media Centre machine, that seems to run fine, which is also hooked up to a TV. Unfortunately, although we are able to get visuals we aren't able to get audio through to the TV. Any other device that is connected is able to receive sound, like a headset, though the TV doesn't seem to get audio through the HDMI.

ainsley26 say: Hi to anyone again,

If anyone's interested, it appears we solved the problem, it was a bit of a combination of a number of things...

Firstly, we managed to find a new graphics driver aside from the original Intel driver, I believe it to be called "Intel Iris", if you have a 4th gen CPU, then check that out. Also, it might be handy just to make sure that you have the updated Realtek driver which you can get from their website.

Also if you are using a AV Receiver like I was, try to not let the HDMI audio pass through, instead stop it from going through to the TV. If you let it go through, apparently the "EDID" of the TV is given to the PC and not the Receiver and then you only end up with Stereo sound and not full surround. 

I had an Onkyo TX-NR905 which passes through the audio, if you have an Onkyo, then check the setup menu under "Hardware" >> "HDMI" >> "HDMI Audio Out:" "On/Off", and select off.

Hopefully, this helps some people who have a similar problem.
