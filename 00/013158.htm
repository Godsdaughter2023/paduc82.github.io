If Zip codes apply to only one state then VLOOKUP should work

Suppose you have a table like

Zip Code      State

0                 New York 

15000          DC

16500          RI

If these are in A2:B52 then

=VLOOKUP(C1,A2:B52,2,False)

should lookup the zipcode in C1 against the sorted table in A2:B52.

  1. Open Internet Explorer by clicking the Start button , and then clicking Internet Explorer.

  2. To turn the panning hand on or off, click the Panning Hand button .
