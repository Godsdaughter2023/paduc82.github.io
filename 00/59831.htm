considerate_guy say:

Which version of Movie Maker are you running?  You can see in the About Movie Maker dialog.

 

In the latest versions of Movie Maker, the last song should end when you run out of a visual element (e.g. when you have no more photos or videos).  In the prior versions, you could actually run the music longer than the video and it would appear with a black screen.

 

In either case, if you have music you want to shorten, you just need to select it in the project window.  In the classic versions, the timeline is at the bottom of the screen.  There should be an audio/music track and there you can select the rectangle.  Drag the right edge of the rectangle to make it shorter.

 

In the newer versions of Movie Maker, you have a wrapping storyboard on the right hand side in your project area.  Below that you should see a rectangle representing your song.  Select it by double-clicking which will bring you to the Music Tools tab.  From there you can change the end point of the song.
