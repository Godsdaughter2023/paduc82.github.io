    DNG saving malfunctioning on Windows 10 Mobile

    Alright, this issue has been mentioned before and has been an issue for a long time. And yes, I know the Lumia 1020 isn't supported officially on Windows 10 Mobile (it will remain on WP8.1), but every Lumia 1020 user I know has been complaining about this specific issue. In general, the Windows 10 Insider builds work pretty ok on the 1020, but the DNG saving bug is a real showstopper.

    So down to the actual problem at hand. The issue is that it's not possible with Windows 10 Mobile to save DNG/RAW files on the Lumia 1020. As the 1020 has a 41MP camera, the DNG files are very useful and it's one of the key-features that makes it stand out over the other Lumia's.

    How to reproduce:

    1) start up the Lumia Camera app and set it to save a 5MP JPEG + 34/38MP DNG

    2) try to take a picture

    3) the "Saving..." animation will pop up and will continue forever

    4) you'll notice that the camera module isn't closed (or the shutter not closing?)

    5) starting any camera app won't work as the camera module isn't closed yet, only after a couple of minutes it closes itself and you can use camera apps again

    Setup:

    - Nokia Lumia 1020

    - OS Build: 10.0.10586.164 (latest available for Lumia 1020)

    - Lumia Camera app version: 4.10.0.6 (latest available)

    Things already tried (by me and other Lumia 1020 users) and found not to resolve the issue:

    - soft reset

    - hard reset

    - recovery tool and reinstalling Windows 10 Insider build

    Problem analysis:

    This issue does not occur when setting the Lumia Camera app to its "5MP JPEG" mode, nor does it occur when setting it to "5MP JPEG + 34/38MP JPEG" mode. It only happens when setting it to the DNG mode. Perhaps the issue has something to do with the filesize of the DNG files, as these are typically 40+ MB versus the normal 9MB for the 34/38MP JPEGs. Another option might be that the interface to save DNG files is different for the 1020 than it is for the 930 (which does save DNG files successfully, albeit by using the Windows Camera app).

    1) fix it in the Lumia Camera app and send out one last update for all Lumia 1020 users

    2) add support for the Lumia 1020 to the Windows Camera app. At this moment, the Windows Camera app does not recognize the PureView capabilities of the 1020 and doesn't allow to select any kind of size/resolution modes

    3) ask help to the community to try and fix this bug, I'd be happy to beta-test a variant of the Lumia Camera or Windows Camera app and give feedback

    I'd like to speak for all Lumia 1020 users here, please fix this and help keep the Lumia 1020 around a bit longer.
