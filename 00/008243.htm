Title page

stefan blom say:
You can link footers (using the controls on the Header & Footer Tools Design tab), but you can only link the current footer to that of the preceding section. If you want to duplicate the footer of section 1 in section 4, you can bookmark the content and insert a cross-reference to bookmark text in the section 4 footer.

Note that each Word section can have up to three footers (and three headers); this is the case if you have enabled "Different odd and even" and "Different first page" in Pager Setup. These must all be dealt with separately.

suzanne s. barnhill say:

If your legal jargon is consistent across a number of documents, you might want to save it as an AutoText building block. If it's used only in documents based on a specific template, save it in that template.

 

Or you can save it as a Footer building block; just type the desired text in your footer, then click the Footer button on the contextual Header & Footer Tools | Design tab and select "Save Selection to Footer Gallery..." Give it an appropriate name and save it. 

charles kenyon say:
You can link footers (using the controls on the Header & Footer Tools Design tab), but you can only link the current footer to that of the preceding section. If you want to duplicate the footer of section 1 in section 4, you can bookmark the content and insert a cross-reference to bookmark text in the section 4 footer.

Interesting, this is one instance where the REF field is superior to the StyleRef field (which will not repeat text in a style found only in a header or footer).

stefan blom say:
Even if STYLEREF could repeat content located within footers (or headers), we wouldn't want to repeat the "current" footer in this case. In other words, a REF (cross-reference) field is what we want.
