simpler way of changing charts reference worksheet - Microsoft Community

rene29 say:

simpler way of changing charts reference worksheet

Is there a simpler way of changing a chart's reference worksheet?  I copied the referenced sheet then edited its layout and added data, now I would like the chart to grab the data on the new worksheet.  I know I will still have to modify the chart but I am trying to avoid having to redo the new charts (100+) or changing each data series reference.

 

Thanks

shane devenshire say:

If all the sheets have 1 and only 1 chart on them and no other objects and they use the same range then the following macro will work. 

Sub ChangeReference()
    Dim ws As Worksheet
    Application.ScreenUpdating = False
    For Each ws In Worksheets
        ws.Activate
        ActiveSheet.DrawingObjects.Select
        ActiveChart.SetSourceData Source:=Range("A1:B12")
    Next ws
End Sub

This assumes that the chart range is A1:B12, but you could adjust that.

To add this code to the workbook press Alt+F11 and choose Insert, Module then paste the code into the window on the right.  Close the VBA window and you should be back in Excel. Save your file as a macro enabled workbook. You can run the macro by choosing View, Macros, View Macros, pick the macro and click Run.

This macro may be slow if there are 100 sheets.
