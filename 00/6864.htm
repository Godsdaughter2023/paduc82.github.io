john korchok say:

One old workaround for this is to keep the file being linked to at the root of your C: drive. Then all users move the Excel file to the same spot and the links work on everyone's computer. This doesn't work with Macs, because they don't have a C: drive.

The alternative is to make the Edit Links to Files command visible so you can use it:

  1. In PowerPoint, choose File>Options>Quick Access Toolbar.
  2. Change Choose command from: to All Commands.
  3. Scroll down to Edit links to files and select it.
  4. Click on the Add button. A copy of the command is moved to the right-hand list. OK out. A new icon appears in the Quick Access Toolbar.
  5. Open your linked PowerPoint file, then click on the Edit links to files icon. A dialog appears list all links in the deck and allows you to edit them.
