Adding Space Above and Space Below to the Home Ribbon - Microsoft Community

nickhelm say:

Adding Space Above and Space Below to the Home Ribbon

This isn't so much a question as an answer to a question that has been locked. See https://answers.microsoft.com/en-us/msoffice/forum/all/word-2016-paragraph-space-abovebelow-command-to/206d716b-381f-41ad-a7d9-bbfafd9e6596

These controls are hidden away in the Layout ribbon, but I far prefer them among all the other normal formatting tools in the Home Ribbon. How to get them there?

The answer is quite straightforward, if you know what you are looking for. 

In Word (Mac) in the Ribbon & Toolbar preference pane, you need the Before and After commands from the All Commands list on the left. 

To add these to the Home Ribbon, click on the Home group in the right pane and then the + below, and create a new group. I called this 'Spacing'.

Then on the left pane scroll to find Below, click on it and then click the right arrow to add it to your new group. Do the same for Above.

You can change the horizontal position that this group takes in the ribbon by dragging it to the position you want among the other groups in the Home Ribbon. 

Then Save and you will find you have your Paragraph Space Above and Space Below controls where you want them in the home ribbon!

ronrosenfeld say:

Here is a VBA macro to do that.  You need to select the range of cells you wish to process first.

 

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.

==========================

Option Explicit
Sub ChangeDigit()
    Dim rg As Range, c As Range
    Const DigitToChange As Long = 3
    Const NewDigit As Long = 2
Set rg = Application.InputBox(Prompt:="Select Cells to Change", _
                Default:=Selection.Address, Type:=8)
For Each c In rg
    If c.Text Like "#####" Then _
        c.Value = WorksheetFunction.Replace(c.Text, 3, 1, NewDigit)
Next c
End Sub

teylyn say:

Hello,

Vlookup has the syntax Vlookup(value,lookup table, return column, match type). You list something you call "the reference value". I don't understand how that comes in.

You also don't need SUM to add the results of two Vlookups. You are already using the + operator. That should be sufficient. 

Also, the Concatenate function is a waste of space. You can use the & operator to concatenate cells. From you description, I've come up with this formula

=VLOOKUP(A2&B2&C1,Sheet1!A1:V20,3,FALSE)+VLOOKUP(A2&B2&C1,Sheet2!A1:V20,3,FALSE)

I still don't know what you want after the Sheet1!A1:V20 reference where you mention "the reference value". If you concatenate something, the result will be text, but the parameter needs to be a number that identifies which column to return.
