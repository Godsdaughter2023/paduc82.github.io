mark isi say:

Hello,


The System Reserved Partition contains two important things, which are the Boot Manager and Boot Configuration Data, and the Startup files used for BitLocker Drive Encryption. It is hidden by default and a very crucial part of the operating system, there fore deleting them entirely is not recommended.

If you still want to proceed with deleting the System Reserved, feel free to follow the steps below:

  1. Backup your PC, so you can revert the changes in case any errors occurred during the process.
  2. Open command prompt in an elevated mode.
  3. Run bcdboot c:\windows /s c:
  4. Go to disk management and right click on the C: partiton and select the option to mark it as "Active partition"
  5. Reboot the server and then delete the system reserver or the BDE partition.
  6. Right click on the C: partiton and select the "Expand" option to expand the C: partiton.
  7. Restart your PC.

Let us know the outcome.

tom ogilvy say:

Jon K. Jr.

this code should approach what you want:

Sub abc()
Dim rTotal As Range, rBody As Range
Dim rKR As Range, r As Range
Set rTotal = ActiveSheet.ListObjects(1).TotalsRowRange
Set rBody = ActiveSheet.ListObjects(1).DataBodyRange
Set rKR = ActiveSheet.Columns("K:R")
Set r = Intersect(rKR, Union(rBody, rTotal).EntireRow)
r.FillDown
End Sub

If you want to use R1C1 notation, then you can change

Set rKR = ActiveSheet.Columns("K:R")

to

Set rKR = Activesheet.Cells(11,1).Resize(1,8).EntireColumn

Cells(row, column) is the closest equivalent to R1C1.
