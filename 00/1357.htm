How to create a self-running presentation - Microsoft Community

glenna shaw say:

How to create a self-running presentation

Summary
Technical Level: Basic Applies to: Windows 7, Windows 8
Details

We usually think of PowerPoint as being used by a live presenter who stands in front of a live audience. But PowerPoint can be used in other ways:

·         PowerPoint is often used an an elearning tool. In this situation, the learner controls the presentation.

·         PowerPoint is also used as a marketing tool to give to potential clients. In this situation, the presentation often runs itself.

 

How do you create a presentation that others will run or control?

 

Add narration

Usually, this type of presentation is narrated. You can use PowerPoint’s narration feature (see “Secrets for successfully narrating a presentation“) or insert audio files that you recorded separately on each slide. (Choose Insert> Audio or Sound from File.)

  • If you want the presentation to advance automatically from slide to slide, typical of a marketing presentation that you send to others, add slide timing. For specific instructions, see “Create a video effect.”
  • If you want the viewer to control when the slides change, you can just let the person click or add hyperlinks for navigation. You can use the provided Action buttons because they have Next and Previous buttons that most people instantly understand.

 

Add navigation for people to move through your presentation

Here are the steps for adding an Action button to the next slide:

  1. Go to Home tab> Shapes> Action buttons and choose the one you want, such as the right-facing arrow Next button. The Action Settings dialog box automatically opens.
  2. On the Mouse Click tab, the Hyperlink To option is selected and set to Next Slide. If you want, you can change this, but it usually works well.
  3. Click OK.
  4. Format the Next button any way you want.
  5. Go into Slide Show view and test your button. Note that clicking off the button will also get you to the next slide.

You can add navigation to the previous slide and the first slide as well. You can even put action buttons on the Slide Master.

 

Disable clicking to advance slides

If you don’t want users to be able to click from slide to slide but only to use the navigation, you can turn off mouse clicking to go to the next slide. Here are the steps:

  1. Go into Slide Sorter view (View> Slide Sorter).
  2. Select all of the slides.
  3. In PowerPoint 2010 and 2013, click the Transitions tab. (In PowerPoint 2007, click the Animations tab. In PowerPoint 2003, choose Slide Show> Slide Transition.) In the Advance Slide section, uncheck the On Mouse Click checkbox. (This is where you would add slide timing if you want the slides to advance automatically after a specific amount of time.)

Another thing you can do to disable clicking is to set the presentation to run in kiosk mode. This mode is specifically meant for self-running presentations (running in a kiosk or computer at a trade show or company lobby, for example). Go to Slide Show> Set Up Slide Show (or Set Up Show). In the Set Up Show dialog box, under Show Type, choose Browsed at a Kiosk (full screen) and click OK.

 

Save the presentation as a show

Finally, you can save the presentation as a “show. This simply means that the presentation opens immediately in Slide Show view. Choose File (or Application button)> Save As. (In PowerPoint 2013, you then have to choose your location.) In the Save As dialog box, click the Save as Type drop-down list and choose PowerPoint Show (.ppsx or .pps). Note that a savvy user will know how to change the presentation name’s extension from .ppsx to .pptx and get access to the presentation, so don’t think of this as a measure to secure your content from being edited.

rangao_msft4227 say:

Hi ARPANCHATTERJEE,

After checking the case in the thread you provided, I can confirm that it is a normal behavior that Check out and Open is not showing when you try opening a PDF file directly from a SharePoint Online library.

To edit the PDF file and save it back to SharePoint Online, I suggest you open PDF documents from search results page as a workaround.

Here I’d like to explain the normal behaviors about PDF integration in SharePoint Online.

There are two add-ons handle PDF integration behavior in SharePoint Online, one is Adobe PDF Reader and the other one is Adobe Acrobat SharePoint OpenDocuments Component. Both add ons were installed together with Adobe PDF clients. We can find them in IE-> Manage add-ons.

Scenario 1# When we click a PDF document in a Documents library, IE will launch Adobe PDF Reader to open it in the browser directly. This add on is the default add on to open PDF document in the browser, it is configured by the local PDF clients (i.e. Adobe Reader)

Scenario 2# When we click a PDF document from the search results page, IE will launch Adobe Acrobat SharePoint OpenDocuments Component to open PDF document, and this add-on will attempt to use your local PDF clients to open PDF documents, so we can see the integration behavior.

Display templates control the opening behaviors of the items on the search results page. For the PDF documents, there is a specific Display template called "Item_PDF". We can find it by Site settings-> Master pages gallery-> Display Templates-> Search-> Item_PDF.html.

Basically, it means once we click a PDF document, IE will attempt to contact with PdfFile.OpenDocuments to open it. PdfFile.OpenDocuments is a Registry key and its value is Adobe Acrobat SharePoint OpenDocuments Component.

Now we can explain the reason IE launches Adobe Acrobat SharePoint OpenDocuments Component to open a PDF document if we click it from the search results page.

Regarding the opening behavior in a document library, actually it's control by a XML document called DOCICON.XML, you can refer to the following documents for details:
https://msdn.microsoft.com/en-us/library/office/ms463701(v=office.14).aspx

If there is no specific value defined, Internet Explorer will just launch the default add-ons for opening PDF documents, which is Adobe PDF Reader in this case.

However, we cannot modify the DOCICON.XML in SharePoint Online, because the settings in DOCICON.XML are global to a SharePoint deployment and affect all site definitions on all front-end web servers.

It is a pleasure to help you with your issue in turning off junk filtering in Outlook.com. Let's find out what are the possible options that we can do to resolve your issue.

 

You can adjust the filtering settings of Outlook.com. We recommend that you set your junk mail filtering to standard. Here's how:

 

1. Go to www.outlook.com and sign in with your Microsoft account

2. Go to your Inbox

3. Click Options and then select More options

4. Under Preventing junk email category, select Filters and reporting

5. Under Choose a junk email filter category, select Standard

6. Click Save to confirm changes

 

We also suggest that you mark the email sender as safe. When you mark it as safe, the emails that you will receive from the sender will be directed to your inbox. All you have to do is:

 

1. Go to www.outlook.com and sign in with your Microsoft account

2. Go to your Junk folder

3. Open the email you want to mark as safe

4. Click on the link that says Wait, it's safe!

 

You may also add your sender in your safe senders list. Here's how to add it:

 

1. Go to www.outlook.com and sign in with your Microsoft account
2. Click the Gear icon and select More mail settings
3. Under Preventing junk email category , please click the Safe and blocked senders link.
4. Click the Safe senders link.
5. On the box under the Sender or domain to mark as safe, type in the e-mail address or domain of the sender.
6. Click the Add to list >> button to add the email address to safe sender list


Let us know if you have any questions.

Hi Lilian,

 

Please follow the steps below to see the result:

  1. Go to SharePoint admin center>Search>managed properties>New Managed Porperty
  2. Give a property name, for example, searchDocID, then check the “searchable” 
  3. Add the “ows__dlc_docid” crawled property to it. 
  4. Click ok
  5. Wait for up to 4 hours then you should be able to search this.
