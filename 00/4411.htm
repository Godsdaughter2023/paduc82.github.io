page size options in Word 2010 - Microsoft Community

doinmypart say:

page size options in Word 2010

Is it possible to do large pages in Word?  Like 9 x 12?  If not, is there a recommended version or software that would allow me to format larger pages?

jay freedman say:
The maximum page size in Word (all versions) is 22 inches in either direction, but it will limit you to the maximum size that your printer driver can handle. The typical personal printer in the US can't fit paper wider than 8.5 inches, and Word won't let you choose a wider page while that kind of printer is selected. If you have a wider printer, then Word will accept 9x12 inches in the Page Layout > Size > More Paper Sizes dialog -- otherwise you'll be prompted to "fix" the setting.

For paper larger than 22x22 inches, you can use PowerPoint or Publisher, or a word processor from another source.

jay freedman say:
You'll probably be happiest going with Publisher.

If you want to stick with Word for some reason, you could install the driver for a wide-carriage printer -- you don't have to have the hardware for it -- and select that printer in Word's Print dialog.

Windows comes with drivers for lots of printer models. In the Windows Start menu, click Devices & Printers; at the top of the dialog click Add A Printer. Select "Local printer" and then "Use an existing port". Choose a wide-carriage printer such as the Canon Inkjet iP9910. Just remember that you can't really print the document, although you can save it as a PDF file.
