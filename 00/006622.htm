jeeped say:
You can automate this by creating a custom Worksheet_Change function.

Right click the name of hte worksheet tab and select View Code. Paste this into the right-hand window titled something like Book1 - Sheet1 (Code),

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Range("D:D,K:K")) Is Nothing Then
        Application.EnableEvents = False
        If UCase(Range("K" & Target.Row).Value) = "D" Then
            Range("D" & Target.Row).Value = 0 - Abs(Range("D" & Target.Row).Value)
        ElseIf UCase(Range("K" & Target.Row).Value) = "C" Then
            Range("D" & Target.Row).Value = Abs(Range("D" & Target.Row).Value)
        End If
        Application.EnableEvents = True
    End If
End Sub

When this is pasted in, press Alt+Q to close the VBE and return to Excel.

This workbook will have to be saved as macro-enabled.

Edit: Originally I had this set up as accrual accounting where a credit is a negative. I've changed the sign of the output to more closely reflect the OP's request.
