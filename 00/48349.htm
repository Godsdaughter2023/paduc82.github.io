    How Do I Use a Windows Explorer Saved Search? - Microsoft Community

    jedisql say:

    How Do I Use a Windows Explorer Saved Search?

    One can use the "Search" field in the upper right of a Windows Explorer window to find files matching specified criteria (Windows 10, 1511).  Once the search has started, a new ribbon tab appears - "Search Tools"/"Search."  In the Options section of the Search tab is "Save search."  I clicked that, and a Save dialog box opened with the save location defaulted to "My Name > Searches" (C:\Users\MyLogin\Searches).  I gave it a name, "My First Search," and clicked Save.

    I then used Windows Explorer to go to C:\Users\MyLogin\Searches, and I saw My First Search.  I double clicked it, and Windows Explorer just opened it like an empty folder (including the Search field in the upper right with the gray suggestion text "Search My First Search").  It did not produce any search results.

    Once I have created a saved search in C:\Users\MyLogin\Searches, how do I use that to run a search and get the current specification-matching contents of the folder I originally searched?

    jedisql say:

    OK, I found the problem.  The problem is that Windows 10.1511 does not properly store this search argument:
    *{*}    (asterisk, left brace, asterisk, right brace).
    I tried all sorts of things, and it seems that when a search argument contains a left brace, {, the Saved Search fails.  Just my luck that the very first search argument that I ever tried to save as a Saved Search is one that Windows glitches on.  The search argument works correctly when first entered, it just doesn't save properly.

    I found a Windows 7 "Advanced tips for searching in Windows" article*.  I was looking to see whether braces were special characters for search.  That page did not indicate that braces are special characters.  However, it provided the info that allowed me to rephrase my search argument as:
    System.FileName:~="{" AND System.FileName:~="}"
    When I searched with that and saved the search, the Saved Search then worked.

    *http://windows.microsoft.com/en-us/windows7/Advanced-tips-for-searching-in-Windows

    Moderator, please mark this as the answer.
