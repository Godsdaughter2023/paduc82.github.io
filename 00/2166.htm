Thank you for any suggestions.

zorvek say:

Unfortunately, the date picker is an ActiveX control and ActiveX controls are notorious for behaving badly on worksheets. Undesired font sizes is just one of many issues commonly witnessed. The worst is outright corruption of the worksheet/workbook. Most Excel developers avoid using ActiveX controls on worksheets and use Forms controls instead--but there is no Forms control date picker.

You're only dependable option is to build your own date picker or acquire a third party date picker control.

If the workbook is a macro-enabled workbook, you can try to work around the issue by resetting the control font size when necessary but this is akin to chasing your tail since you never really know when or how the control will start misbehaving.

Kevin
