At one time I reformatted my hard drive and did a clean install of Windows 8 64 bit and found no improvement. About a month ago I again reformatted my hard drive and did a clean install of Windows 7 64 bit. Adobe Flash player now installs smoothly, but the problem with certain web sites is just the same.


The ABC.com web site is a perfect example. Until yesterday, whenever I went from the yahoo.com page to follow a link to an ABC.com video clip, I got the message: YOU MUST INSTALL FLASH PLAYER  to play the video. ABC.com would also offer a button to install Flash Player 11.


Several times I encountered the suggestion that there is no problem with 32 bit versions of Windows, and though I am not certain, I believe this is true. I am now also certain that the problem is not with Windows 7 64 bit nor with Windows 8 64 bit nor with Windows 8.1 64 bit. The problem is actually not with IE 10 64 bit and IE 11 64 bit, but only with the    ACTIVE X FILTERING    component that comes enabled with IE 10 and IE 11.


ACTIVE X FILTERING blocks certain TYPES of content and I believe the   POP UP BLOCKER   that is embedded in the ACTIVE X FILTER  is the source of the trouble. I believe that the ACTIVE X FILTER blocks web sites that use pop up windows to play video clips ( like ABC.com and CBC.com) . The ACTIVE X FILTER may only block the POP UP ADDS that precede the video clips, but I believe the problem web sites may also use a pop up window to play their video.  When their content is blocked, it appears to ABC.com and other similar sites,  that  Flash Player is not installed on your browser. So ABC.com warns you that    YOU MUST INSTALL FLASH PLAYER    to play their video. And ABC.com offers you a button to install Flash Player.


You can use the   TOOLS   menu in IE to turn off ACTIVE X FILTERING, but that would leave you vulnerable to all the malicious applets that hide on Facebook and elsewhere, and to all the Pop Up Adds that ACTIVE X FILTERING is currently blocking.


THE SOLUTION


Find the ADDRESS box at the top left of the browser page. At the far right side of the ADDRESS box ( just before the TABS begin ) is a clockwise circular arrow that   REFRESHES    the current web page. Just to the left of the   REFRESH   button is a small circle with a diagonal line thru it ( the universal symbol of negation as in   NO LEFT TURN   traffic sign ). Left Click the small circle, and the click on the   TURN OFF ACTIVE X FILTERING    button that appears in the drop down box. I believe that this method    TURNS OFF ACTIVE X FILTERING    only for this visit and only for this specific web site. The various videos will now play and   POP UPS   on other web sites will still be blocked.


20 minutes ago when I checked the ABC.com web site, the    YOU MUST INSTALL FLASH PLAYER    did not appear. Instead the video would just not queue up and would not play. When I    TURNED OFF ACTIVE X FILTERING   first a    POP UP ADD    played and then the video clip itself played. Half way thru the video, I    TURNED ACTIVE X FILTERING ON    and immediately the video stopped. There also was an alert that internet explorer had blocked a    POP UP   on the web site.

If you want to multiply all values with 3.6%:

  • Enter 3.6% in a cell.
  • Copy that cell.
  • Select the range with the Low/High numbers.
  • In Excel 2003 or before: select Edit | Paste Special...
  • In Excel 2007 or later: click the lower half of the Paste button on the Home tab of the ribbon, then click Paste Special...
  • Click Multiply under Operation.
  • Click OK.
  • Excel will probably change the number format of the selected range to Percentage; you can correct this if desired.
  • You can now clear the cell with 3.6%.

 

If you want to increase each value by 3.6%:

  • Enter 103.6% in a cell.
  • Copy that cell.
  • Select the range with the Low/High numbers.
  • In Excel 2003 or before: select Edit | Paste Special...
  • In Excel 2007 or later: click the lower half of the Paste button on the Home tab of the ribbon, then click Paste Special...
  • Click Multiply under Operation.
  • Click OK.
  • Excel will probably change the number format of the selected range to Percentage; you can correct this if desired.
  • You can now clear the cell with 103.6%.
