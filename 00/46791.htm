101a say:  

Hi,

 

As I know, there are  three types of deleting files.

 

First type: Move the files to the Recycle bin. With this method you just temporarily stored the files in the Recycle Bin. This usually gives you an opportunity to restore the file to its original location if you discover that the file should not have been deleted.

 

Second type: Either move the file to the Recycle bin and then empty it or just press SHIFT+DELETE. With this method you removed files from the computer and reclaimed any hard disk space they were using. However, this  does not erase the contents of these files from the disk.  it only deletes 'references' to these files from files system tables. The contents of all deleted files remain on the disk and can be easily restored using any recovery utility.

 

Third type: Erase the hard disk after deleting the file. With this method Windows overwrite the contents of free space on your hard disk to prevent deleted or encrypted files from being recovered. It is absolutely impossible to restore any data that has been erased.
