There's no way to create them programmatically, but it's pretty easy to do it manually.

 

First, just create a page that you want to be the template.

bernie deitrick say:

Select one of the selection change events below, copy it, right-clcik the sheet tab, select "View Code" and paste the code into the window that appears.

 

'This version just prompts the user to enter the value
Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Range("A1").Value <> "" And Range("B10").Value = "" Then
        Application.EnableEvents = False
        Range("B10").Select
        MsgBox "Enter a value into cell B10, please"
        Application.EnableEvents = True
    End If
End Sub

 

'This version forces the user to enter a specific type of data - in this example, a number

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Range("A1").Value <> "" And Range("B10").Value = "" Then
        Application.EnableEvents = False
        Range("B10").Value = Application.InputBox("Enter a value for cell B10, please", "Enter Value", , , , , , 1)
        Application.EnableEvents = True
    End If
End Sub
