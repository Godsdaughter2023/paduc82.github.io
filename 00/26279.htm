      Rounding to nearest nickel in Access - Microsoft Community

      neil s wcc say:

      Rounding to nearest nickel in Access

      Can anyone tell me how to create a query expression to round a field to the nearest nickel?  I know in Excel you can use the ceiling or the floor functions. 

       

      Thanks in advance.

       

      Neil.

      jeeped say:

      Can anyone tell me how to create a query expression to round a field to the nearest nickel?


      The function for rounding to the nearest 5¢ or $0.05 is accomplished with,

      Round([myFld] * 20, 0) / 20

      20 is used as 5¢ is 1/20th of 1. That will get you the nearest nickel.

      ... I know in Excel you can use the ceiling or the floor functions ...


       To round up to the nearest 0.05 use,

      Round(([mtFld]+([mtFld]<>(Round([mtFld]* 20,0) / 20)) / 40) * 20,0) / 20

      ... or to round down to the nearest nickel,

      Round(([mtFld]-([mtFld]<>(Round([mtFld]* 20,0) / 20)) / 40) * 20,0) / 20

      These account of the fact that an entry like $0.05 is not rounded since it is already its own nearest nickel.

      Here is the routine that I finished up with that accesses SQL Server returning rows and then those rows are transferred into an array named data.  With this, I am able to then do table lookups on the items that are in the array without having to make multiple return trips to the SQL Server.   I hope that this will be useful to anyone who may need something like it.

       

      You can also query with a stored procedure instead of a SELECT.  In my case, because the number of rows in the Item table is quite large, I use the stored procedure method passing a comma delimited string to the SP containing the only the item numbers that are on my current worksheet.  That way I minimize the size of my array and greatly improve performance.

       

      Sub TestSqlConnection()
          Dim sqlstring As String
          Dim connstring As String
          Dim sLogin As String
          sLogon = "Uid=myUserID;Pwd=myPassowrkd;"
          sqlstring = "select * from zitemloc"
          connstring = "ODBC;DSN=myDSN;" & sLogon
          ActiveSheet.Range("B1:t2000").Clear
         
          Dim qt As QueryTable
          For Each qt In ActiveSheet.QueryTables
              qt.Delete
          Next qt
          With ActiveSheet.QueryTables.Add(Connection:=connstring, Destination:=Range("B1"), Sql:=sqlstring)
              .BackgroundQuery = False
              .Refresh
          End With


          Set qt = ActiveSheet.QueryTables(1)
          Dim rowCount As Integer
          rowCount = UBound(qt.ResultRange.Value)
         
         
          Dim ix, iy As Integer
          Dim data As Variant
          Dim colCount As Integer
          colCount = qt.ResultRange.Columns.Count
         
          For ix = 1 To rowCount
              If Not IsArray(data) Then
                      ReDim data(rowCount - 1, colCount - 1)
              End If
              For iy = 1 To qt.ResultRange.Columns.Count
                  data(ix - 1, iy - 1) = qt.ResultRange.Value2(ix, iy)
             
              Next
          Next
      End Sub
