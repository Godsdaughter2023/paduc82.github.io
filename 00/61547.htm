6. Save BIOS changes, restart your System and it should boot from the Installation Media

a. user say:

Hi,

Thanks for posting your query on Microsoft Community.

As I understand you ran troubleshooter for network icon by turning off the internet router. Later when you opened Internet Explorer you found .xml files so you opened them and it automatically opened with internet explorer with the text it had.

Whenever you open .xml files system will select the default application to open it. It could be internet explorer or any other browser. Sometimes it may open in excel or notepad. If you want to open with a different application then you need to right click and change default option. Sometimes if no application is downloaded on the computer to open .xml files it may not open.

But you need to be careful when you open different kinds of executable .xml files that supports some ability to run an automatic task. If it is for viewing some data or playing a video or music, etc. then you can open with whichever application you want.

Take great care when opening executable file formats received via email or downloaded from websites you're not familiar with.

Diagnostic files are executable files you can open only with internet explorer or other browser. It will not affect your system in anyway.
