Are you clicking the Formatted Text button on the AutoCorrect dialog? See screen clip below.

For creating AutoCorrect entries the process to follow is:

  1. Type the formula in a Word document
  2. Format the text and other content exactly as it should be (this becomes your replacement content)
  3. Select the formatted content (text and anything else you want stored as a reusable object) 
  4. Open the AutoCorrect dialog
  5. Click the Formatted Text button
  6. Type in the text string that your formatted text will replace
  7. Click the Add button

The process of saving AutoText and AutoCorrect entries are similar. Both functions were probably long ago misnamed because the content you store/record does not just have to be Text or even Formatted Text (meaning character formatting) the stored content can also include links, tables, shapes, images, etc. The difference between the functions is with AutoCorrect you create a key that is a string of text that when typed into a document triggers the insertion of your stored content. With AutoText you create a label that can be assigned to a keyboard shortcut, a macro, or just selected from a sorted list of stored entries.

Macro to put all values on sheet into a single column - Microsoft Community

sancarloscyclist say:

Macro to put all values on sheet into a single column

I have data in many columns and rows. Can one of you experts create a macro that would take all of the data and put it into column A?  It would be great if that macro also got rid of any blank rows after the data got moved to column A. Is this possible?

 

I as always appreciate your help.

chip pearson say:

The code to use depends on how the 2D table is to be mapped into a single column. You can move across each row, spanning several columns before moving down to the next row. Or, you can move down each column, spanning several rows before moving to the right to the next column. Both approches are shown below:

 

 

 

Sub RangeToColumn()
    Dim InputRange As Range
    Dim R As Long
    Dim C As Long
    Dim Destination As Range
   
    Set InputRange = Worksheets("Sheet1").Range("A1:C3") '<<< CHANGE
    Set Destination = Worksheets("Sheet2").Range("A1") '<<< CHANGE

    For R = 1 To InputRange.Rows.Count
        For C = 1 To InputRange.Columns.Count
            If InputRange(R, C).Value <> vbNullString Then
                Destination.Value = InputRange(R, C)
                Set Destination = Destination(2, 1)
            End If
        Next C
    Next R
End Sub

This loops across each row, column by column, then moving to the next row.

 

To loop down each column row by row, then moving to the next column, use:

 

 

Sub RangeToColumn()
    Dim InputRange As Range
    Dim R As Long
    Dim C As Long
    Dim Destination As Range
   
    Set InputRange = Worksheets("Sheet1").Range("A1:C3") '<<< CHANGE
    Set Destination = Worksheets("Sheet2").Range("A1") '<<< CHANGE
    For C = 1 To InputRange.Columns.Count
        For R = 1 To InputRange.Rows.Count
            If InputRange(R, C).Value <> vbNullString Then
                Destination.Value = InputRange(R, C)
                Set Destination = Destination(2, 1)
            End If
        Next R
    Next C
End Sub

 

In both procedures, change the lines marked with '<<< to the appropriate values. InputRange is the range of the original data and Destination is the first cell where the columnar data is to be written.
