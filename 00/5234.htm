    Powershell suddenly started working again.

    So I was able to create and run Powershell scripts to repair issue #1.

    1. Right-Click on Desktop and create a new Text Document.

    2. Paste this:

    powershell.exe -ExecutionPolicy bypass -command "& '%~dpn0.ps1'"
    pause

    And name the file fixit.bat

    3. Create another Text Document and paste this:

    Get-AppXPackage -AllUsers |Where-Object {$_.InstallLocation -like "*SystemApps*"} | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

    Name that file fixit.ps1

    4. Run each file. That fixed the problem for me.

    You may or may not need to run those files every time you boot into your desktop.
