arminatorx say:

The first number 4476 signifies that your console is on version 17526 which is the latest system version already. 

The second number (0000) says, that the console didn't find a new update. 

This combination usually happens, if your system on a storage device became corrupted. Usually because the storage device is defective.

If you have a 4GB console with built in flash storage, your only way to resolve that is to send in the console for repair. 

If you have a console with an internal hard drive, try to remove the hard drive, and use a new USB drive connected to the console. 

In the storage settings, have that USB drive prepared as system storage, and attempt to delete the corrupted update by pressing 

LB RB X LB RB X

on the storage drive selection screen. 

Have the console reboot, and hopefully apply the latest update again. 

If that doesn't work either, you need to send the console in for repair.
