If you've somehow managed to delete just the endnote reference without affecting the numbering of other endnotes, that suggests what you deleted was the reference in the endnote, not the reference in the document body. One possibility, depending on the effect you're after, would be to enter the text normally but make it rather smaller than you want it to end up.

With the text box selected, press Ctrl+C to copy it.

Then on the Home tab, click the triangle under "Paste" and choose Paste Special | JPG.

Enlarge the pasted image to the final text size you want.

Experiment by starting at different text sizes and/or pasting as PNG to get different results.

I don't remember if this is available in 2013, but in 2016, you can also start with text the size you want it, copy it, paste special as PNG then use the Artistic Effects (Picture Tools | Format | Adjust group on the ribbon)

 

I'd try this first, in fact.

jessen p say:

Hi Tracey,

Thank you for the update.

You have to manually set up a wireless profile. The process is very simple and painless.

Go to Control Panel/Network and Sharing Center
Select "Set up a new connection or network"
Select "Manually connect to a wireless network"
Fill in "Network Name" (this is the SSID)
Select the "Security type" (choose WEP or other type from dropdown list)
Select the "Encryption type" (choose from dropdown list)
Fill in "Security Key" (such as 10 character hex or passcode)
