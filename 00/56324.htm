Your computer can't come out of hibernation 0xc000007b - Microsoft Community

hefe44 say:

Your computer can't come out of hibernation 0xc000007b

On my windows 7 professional, 64 bit when i  wake up my laptop i receive this message: Your computer can't come out of hibernation. Status: 0xc000007b Info: A fatal error occurred processing the restoration data. File: \hiberfil.sys  Any information that was not saved before the computer went into hibernation will be lost.             Does anyone have a solution to this error? Thanks in advance!
balaji kundalam say:

0xc000007b error code is shown whenever an application or service failed to start. In this case, the issue is with the hibernation.

Hence, to begin with, kindly answer the questions:

1. Do you turn off the power/remove battery while hibernating (hibernation doesn't show any sign of progress, hence people used to pull off the power; since it's related to power, failed battery also causes this issue)?

2. Is this issue happening even when resuming from sleep?

Case 1:

If the answer is yes to question 1, kindly make sure sufficient power is present until the computer turn off completely. If there is no LED/any other indicator present, kindly wait for 3 to 5 minutes before turning off the power.

Case 2:

If the answer to the question 2 is yes, you can try upgrading the display drivers (graphics card drivers). If the upgrade fails to resolve the issue, reinstall the drivers: uninstall the drivers, restart the PC, install the drivers, restart the PC and then check.

Case 3:

If the answer to both the above questions is no, kindly follow the steps:

Go to start -> type cmd -> right click on it -> click on Run as administrator. If UAC prompts click Yes. Then, in the Command Prompt, type the following and press enter (disabling hibernation):

powercfg -h off

Then, restart the PC a couple of times. Then, the same way open the Command Prompt, type the following command and press enter (enabling hibernation):

powercfg -h on

Then try hibernation.

If the issue persists, again, disable hibernation, perform disk defragmentation, restart the computer and the enable hibernation. Follow the link for more details on disk defragmentation:

http://windows.microsoft.com/en-us/windows/improve-performance-defragmenting-hard-disk#1TC=windows-7

Cause:
This behavior may occur if you try to register a DLL by using Regsrv32 while you are logged on using an account that does not have administrative credentials, such as an account that is a member of the standard users group. An account that does not have administrative credentials cannot write to the registry or change files in the System32 folder.
 
Resolution:
To resolve this behavior, log on by using an administrator account, and then register the DLL.
Perform the following workaround to enable the default built in administrator account and set the password.
Let us run the same command in the elevated command prompt and check.
Here is how:
a. Click on Start and type cmd in the search box.
b. Right click on the cmd.exe and choose "Run as administrator".
c. Now run the command that your are running and check if it is successfully executed.

You Receive a "DllRegisterServer" Error When You Try to Register a DLL by Using Regsvr32.exe

Having a corrupted game file is the common reasons why you are having this problem. To better assist you with this concern, we highly recommend that you perform the troubleshooting steps below:

Method 1: Check for updates for apps and games from Windows Store. You may refer to this link for the steps.

Method 2: Reset the game from the Settings menu.

  1. Click Start and click Settings.
  2. Select System.
  3. Select Apps and features.
  4. Find the game and click Advanced options.
  5. Click Reset.
  6. You need to restart your computer to let the changes take effect.

Method 3: Delete Temporary files. This will help delete temporary files that have been stored on your computer.

Click Start and click Settings.  

  1. Select System.  
  2. Select Storage.  
  3. Select This PC.  
  4. Click Temporary files.  
  5. Choose Temporary files.   
  6. Click Remove files.

Method 4: Reset Store cache. Follow the steps below:

  1. Press the Win+R keys to open the Run dialog. 
  2. Type WSReset.exe, and click/tap on OK.  
    Note: The WSReset tool resets the Windows Store without changing account settings or deleting installed apps.
  3. The  Store and a command prompt will now open without any message. The command prompt will automatically close when the Store cache has been reset.
