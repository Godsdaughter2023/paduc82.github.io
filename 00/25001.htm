OK I'll try to explain some of this.

Skip the first link, you are right, it is too technical.

Going back to the first idea, a "helper" column
In excel often we create a new column based on an existing one that will help with our calculations. In this case we want to create a company name field that omits the "AB" abbreviation so that the company names will sort correctly, or better.

In the past you would create a formula to populate this helper column. But in 2016 there is some powerful automation that can help you do this WITHOU fancy formulas (formulaii if you want to be pendantic).

First step, we are going to convert your data into an Excel Table. To do this the first thing you need to do is make sure your data does not have any completely blank rows or columns. If there are you have to delete them.

Second step, with the insertion point located inside of the data to be defined as a table. Go to the Insert tab > Tables group > click on the Table button. The Create table dialog will display. It will show the range excel thinks you are talking about, and it will draw a dotted box around that range so you can easily identify the range visually. There is also a checkbox to idenfity if your data has column headings or not. click on OK to define the table. I don't think that there is any way that you can use such a workbook directly as a datasource for a merge in which you want to include the pictures.  However, assuming that the pictures are on Sheet 1 of the workbook, if you run a macro containing the following code in Excel, it will create a Word document with a Table in it with the word "Picture" in the first row and each of the pictures in individual rows of the table. 

Sub SendpicturestoWord()

Dim wdApp As Object
Dim wdDoc As Object
Dim wdTable As Object
Dim wdRow As Object
Dim i As Long
On Error Resume Next
Set wdApp = GetObject(, Word.Application)
If Err Then
    Set wdApp = CreateObject("Word.Application")
End If
On Error GoTo 0
wdApp.Visible = True
wdApp.Activate
Set wdDoc = wdApp.Documents.Add
Set wdTable = wdDoc.Tables.Add(wdDoc.Range, 1, 1)
wdTable.Cell(1, 1).Range.Text = "Picture"
For i = 1 To ThisWorkbook.Sheets(1).Pictures.Count
    ThisWorkbook.Sheets(1).Pictures(i).Copy
    Set wdRow = wdTable.Rows.Add
    wdRow.Cells(1).Range.Paste
Next i
End Sub

If that Word document is then be used as the data source for a mail merge, with the «Picture» mergefield in that main document, when that merge is executed, the pictures will be displayed in that field.
