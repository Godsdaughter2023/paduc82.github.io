jason of oz say:

If all your PCs are running versions of Windows 7, take a look at setting up a HomeGroup, there's some handy links from Al Jarvi already posted.

If you're running a mixed network, you'll need to take a few more steps.

  1. If you're logged on as AccountA, make sure AccountA exists on all PCs you're trying to connect to.
  2. Make sure AccountA has the same password on all PCs you're connecting to (not necessary, but easier).
  3. If AccountA has no password, think seriously about setting one, if you don't want to you'll need to change Local Security Policies to allow network access by accounts with blank passwords on each machine you want to connect to.
  4. Don't try to connect to \\Machine\C unless you've explicity shared your HDD under that name...\\Machine\C$ is the default share.

That should just about do it. Best of luck.
