jay freedman say:

Normal behavior in a text box is that if you press and hold the Ctrl key while pressing another key, you get the shortcut. You get the layout options popup only if you press and release the Ctrl key.

If you do get the popup when you press and hold Ctrl, you may have a "key bounce" problem in your keyboard. If that's the case, try using a different keyboard to see whether the popup stops.
