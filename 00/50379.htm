Did you try to troubleshoot issues or just expect Windows 10 to work properly  after doing an in-place Upgrade which is an inferior install to begin with, and one which no tech enthusiast would even run?  How about instead of retreating trying to fix the issues?

When you power up and plug in the printer via USB, watch for the animation in the System Tray at right end of Task Bar, click on it and monitor install of driver.  If driver installs then printer should print and may have basic scanning via Windows Fax and Scan utility.  To find any such utility type part of its name in Search box. 

If driver doesn't load type Devices and Printers in Start Search box, open it, see if Printer is already listed, right click on it to open Properties and try printing a Test page. If it doesn't print right click Printer again to Troubleshoot. 

If printer isn't listed  go to the printer's Support Downloads webpage to download and install the basic driver.  If a wireless printer you will need the full software package to get it set up fully.  If you want to be able to scan from the printer top to the PC then you'll also need the full package.  Follow the setup screens closely as you install the software. 

When you plug in the iPhone look for activity in System Tray like a popup to click to choose how you want to interact with phone.  If it's not there you'll find the phone as a drive in Explorer to browse.  You can also type Autoplay settings in Start Search while phone is plugged in, choose the iPhone to choose what you want it to do  in dropdown. 

If Turbotax is an installed program and doesn't work properly then download the lastest version to  run it as a Repair Install.  If that fails, uninstall it in Control Panel, restart and reinstall. 

Your Internet Explorer Favorites are loaded in IE which you can access by typing Internet Explorer in Start Search box, right click to Pin to Task Bar.  To load them into Edge, click the three lines icon for Settings, choose Favorites Settings, import Favorites from IE.

Step 1:Download and save the driver

  • Go to the Compatibility tab.
  • Check the box for Run this program in compatibility mode for:

  • Click OK.
  • Double-click the driver file to install the driver and follow the onscreen instructions.
