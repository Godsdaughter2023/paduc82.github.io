How do I stop the print command and empty the queue? - Microsoft Community

charlesendreola say:

How do I stop the print command and empty the queue?

* Original Title: printer commands

My printer was malfunctioning and the queue was filling up by the time I realized it and cured the printer problem. Now I want to stop the  print command and empty the queue. How?

sharief k say:

Hi Charles,

The most obvious way to cancel a print job is to find a cancel button or a cancel menu command on the printer and to press it. Next, try turning the printer off and then on. With certain printer models, this cancels the print job.

If this doesn’t help, try these steps:

a) Press Windows key + R on the keyboard together.
b) Type “Services.msc” and press Enter.
c) Right click on Print Spooler service.
d) Click the "Stop" button to stop the Print Spooler Service if it is running.
e) Check whether the Startup type is set to Automatic. If not, then set it to Automatic.
f) Go to the following location and delete all the pending documents from the “Printer”. “C:\WINDOWS\system32\spool\PRINTERS”
g) Delete all the queued print jobs (select and delete all the files) in the folder so the folder is devoid of pending print jobs. The folder should be empty when you are done.
h) Restart your system and check the Print Spooler Service again to be sure the Print Spooler Service restarts automatically on reboot (since the Startup type is set to Automatic).

i) After rebooting, check the Print Spooler service again and the "Status" column for the Print Spooler should read "Started".

Hope this information helps. If you have any concerns, please let us know.

tryx3 say:

Windows 7 Recovery partitions are created by PC makers and they cannot generally be manipulated by anything other than a utility put on the PC for that purpose by the PC maker.  There are no universal tools for doing the job.

For example, Dell recovery partitions ["Dell Factory Restore" partitions] can be transferred to a USB stick using the Dell DataSafe Backup Local utility provided by Dell.  Using their utility to create the bootable USB stick is an alternative to the normal procedure of using the recovery partition in situ and is an excellent precaution in case the HDD ever fails.

I suggest that you check your computer manual and the PC maker's support site to find out how to do what you want to do.
