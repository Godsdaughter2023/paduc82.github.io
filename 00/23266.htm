per jessen say:

Hi

 

First, when you use .Resize, you can not use 0, as Resize(1,0) will result in a range with one row but 0 columns!.

 

Second, your approach in the If.. Then ... End If construction will not work, as rngWeeksList, will point to the last cell in the desired year when you use the resize statement, ie rngWeeksList will finally refere to the last cell in the year and the 51/52 cells below!

 

Using the For Each... construction we can find the first cell in the year , and then use .End(xlDown)  to find the last cell in the year, as years are seperated by an empty cell.

 

    For Each rngWeeksCell In rngAllWeeksList
       
        If rngWeeksCell = intCurYear Then
            Set rngWeeksList = rngWeeksCell
            rngWeeksList = Range(rngWeeksList, rngWeeksList.End(xlDown))
            Exit For 'No need to continue
        End If
    Next
    returnedVal = rngWeeksList.Rows.Count

 

But I would not loop at all, but use .Find to find the first occurance of the year, and then use the approach above to find the number of weeks in the year.

 

Sub weekItemCount(control As IRibbonControl, ByRef returnedVal)
    Set wksSheet = ThisWorkbook.Worksheets(msWKSWEEKS)
    intCurYear = wksWeeks.Range("N3")
   
    Set rngWeeksList = Nothing
    With wksSheet
        Set rngAllWeeksList = .Range("A1", .Range("A" & Rows.Count).End(xlUp))
    End With
   
    Set rngWeeksList = rngAllWeeksList.Find(what:=intCurYear, after:=wksWeeks.Range("A1"), _
        LookIn:=xlValues, lookat:=xlWhole)
    rngWeeksList = Range(rngWeeksList, rngWeeksList.End(xlDown))
 
    returnedVal = rngWeeksList.Rows.Count
 
End Sub

jay freedman say:

Word does not normally print any of the spelling and grammar marks (red, blue, or green wavy underlines). If you're seeing those in your printouts, there's something wrong with your installation. Repairing Office may help.

If you can't find any other solution, you can temporarily hide the marks: Go to File > Options > Proofing, and check the box at the bottom of the dialog for "Hide spelling errors in this document only". If grammar errors are also enabled, then also check "Hide grammar errors in this document only". After printing the document, you can go back and uncheck the boxes to return the marks on screen.

When I embed audio, PowerPoint inserts a speaker icon. I know I can individually change the picture, but I'd like to be able to change the default icon.

Can you change the default audio icon in a theme? 

Can you change all of the icons in at presentation all at once?

thanks for you help.

john sr wilson say:
You can and as I mentioned earlier you could change the whole presentation sound images at once if you used code.

This is pretty "top of head" but I think it works.

Sub switchPic()
    Dim osld   As Slide
    Dim oshp   As Shape
    Dim strPath As String
    Dim sngW   As Single
    Dim sngT   As Single
    Dim sngL   As Single
    'change to suit
    strPath = "C:\Users\John\Desktop\banner.jpg"
    For Each osld In ActivePresentation.Slides
        For Each oshp In osld.Shapes
            If oshp.Type = msoMedia Then
                If oshp.MediaType = ppMediaTypeSound Then
                    sngW = oshp.Width
                    sngT = oshp.Top
                    sngL = oshp.Left
                    oshp.MediaFormat.SetDisplayPictureFromFile (strPath)
                    oshp.Top = sngT
                    oshp.Left = sngL
                    oshp.Width = sngW
                End If
            End If
        Next oshp
    Next osld
End Sub
