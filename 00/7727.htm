Private Sub Worksheet_Change(ByVal Target As Range)
  'Which cell is changed?

I have achieved this where the images are stored in an Excel workbook as opposed to a database.

Please note that this code adds an image to a Word document but does so from an Excel workbook which is where the code resides.  If you're replicating the code in Word you would eliminate any references to "Word." because this would be implied.

All images pasted into Excel can be accessed by looping through the Shapes array:

Dim shpShape As Shape

    'this is where the image resides
    shpShape.TopLeftCell.Address
Next shpShape

You can retrieve a particular image from the workbook by knowing within which cell the top left hand corner of the image is located.  So if you setup a nice matrix of wide columns and high rows such that each cell totally encloses one image only, you can quite easily enforce that the cat is in A1, the dog in B1 etc.

With this known, you can use code similar to the following to copy the image at a given location and paste into the Word document.  The logic I took this from generates a sequential word document so the location of the image is the current cursor position.  In your instance you will likely do better by creating a bookmark, going to the bookmark and then pasting the image.  In the below, the "strImageAddress" parameter is address of the cell containing the image.  I loop through my own array based on the results of the above .Shapes logic until the image with that cell reference is found.  It's then copied to the clipboard and pasted into the Word document.

<EDIT - simplified the below code by removing anything extraneous to the OP's question>

Sub AddImage(wdDoc As Document, strImageAddress As String)
     Dim iShapeIndex As Integer
     Dim shpShape As Shape
    
     'loop through all pictures in array
     For iShapeIndex = LBound(c_objMyShapeObject) To UBound(c_objMyShapeObject)
         'with current elemenet in array
         With c_objMyShapeObject(iShapeIndex)
             'is the picture located in the target cell?
             If .strCell = strImageAddress Then
                 'picture is located in the target cell
                
                 'get the picture
                 Set shpShape = .shpShape
                 'copy to clipboard
                 shpShape.CopyPicture
                 'paste into Word document at current caret position
                 wdDoc.Application.Selection.Paste
                
                 Exit For
             End If
         End With
     Next iShapeIndex
 End Function

I hope this helps you out.
