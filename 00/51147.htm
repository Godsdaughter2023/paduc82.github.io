HomeGroup Network = Works only between Win 7 computers. This type of configuration makes it very easy to Entry Level Users to start Network sharing.

canadian tech say:

Wow.  What an interesting question.  I just had a colleague ask me this question.  I did not know the answer.  He figured it out and here is the solution.  The strange part of this story is that he asked me if there might be others with the same question.  I was doubtful.  And here 2 days later, you ask it

I would really appreciate your confirmation that this works or other comments.

Customize the file attributes listed in Explorer on the title bar

 

Windows file Details can contain a lot more than the usual File, Date, Modified, Size, etc. information

 

The first step is to select a folder. Right click and follow the context-sensitive menu down to properties (or Alt-click on the folder). Then select the Customize tab. The first option is to select the type of folder that you want to make it be e.g. Pictures and select whether you want to apply the template to all Sub-folders. You can also put a custom icon on the folder (e.g. a camera). This is how you Select or differentiate folders so that it is NOT all or nothing.

 

Then instead of right-clicking on the Title Bar in the right-hand panel, select View and go down to Choose Details. (So you were on the right track after all!). A drop-down menu appears with the attributes that you can select by ticking the checkboxes (you will have to click on More at the bottom of the menu to scroll and see all the available options). This has the same effect as right-clicking on the Title Bar EXCEPT that when you then go up to Tools >Folder Options, and click on the View tab, the option to Apply to All Folders is NOT greyed out. Click on it and you will receive another prompt to the effect of, Do you want to apply this to all folders of this type. Choose Yes and you are done.

 

There is also a Tag Attribute which would appear to allow you to create your own specific attribute (e.g. mine).  However, it only works with certain kinds of files e.g. MS Office files and doesn’t seem t work at all with folders. Also, when you do a Search, you can specify a particular atribute as a search term.

 

The Mac OS also has a Tag attribute, but it can be applied to files (all I think) and to folders. The default tags are based on colours, but you can define your own.
