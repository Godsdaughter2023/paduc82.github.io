stefan blom say:

As Charles said, the anchor basically defines which page the object should appear on. Note that you can lock the anchor to prevent it from being moved via drag and drop.

However, cutting and pasting the object to some other page in the document will also move the anchor. The converse is also true: if you move the paragraph containing the anchor to another page in the document, the object will move with the anchor to that same page. (This applies no matter if the anchor is locked or not.)

It is worth noticing that the position of the actual object on the page is not (directly) related to the anchor. It is possible to position an object relative to a text paragraph, so that the object moves up or down the page along with the paragraph, but, as an alternative, you can specify the position relative to the page edges, for example. In both cases, the object will always move to a new page if the anchor moves to another page, though.

jack_norris say:

Audio issues / build 10159: Failed to play test tone

Having issues changing speaker configuration and sample rate settings in Windows 10 (builds 10158 and 10159).

Issue:

  • Changing sample rate or speaker configuration results in no audio. Testing in the Windows playback control panel results in message "Failed to play test tone".
  • This issue can be worked around by running through the "Default rate" settings under the Advanced tab and clicking test for each one. Eventually you will find a configuration that works.
  • Reboot, audio still works but again changing any settings results in "Failed to play test tone" and no audio. Fix is to once again scroll through the Default rate list and find which one works (interestingly it seems to be different every time you make a change to the audio configuration). Additionally I find that 24bit/96KHz works with all my applications when previously on Windows 8 it would only work with a select few - so I'm not sure if it is actually running at 24bit/96KHz.

I have tested this with the following sound cards on Insider Preview X64 Pro (10158, 10159):

  • ASUS Xonar DX 7.1
  • VIA VT2021
