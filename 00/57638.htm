¡firedog say:

Windows Live Mail Error ID: 0x800420CD

This error indicates that there's a message on the server that Windows Live Mail can't or won't down­load. Log in to the account in webmail and start by emptying the spam and deleted folders. Then move all of the messages in the inbox to a user-created temporary folder, leaving the inbox empty. Now do a send/receive in Windows Live Mail; you should see a No new email messages status mes­sage.

Back in webmail, move half of the messages from the temporary folder back into the inbox, and again send/receive in Windows Live Mail. Repeat, halving the number of messages moved each time until the error recurs; you should soon be able to identify the message that Windows Live Mail is choking on and deal with it. It will probably be the oldest that hadn't been downloaded.
