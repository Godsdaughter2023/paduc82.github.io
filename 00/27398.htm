MID Function in query - Microsoft Community

revned12 say:

MID Function in query

Hello,

 

I have a Field name:Document Number to my Query:qry_Log, I want to extract the first 4 digits in this field and put it in a new Field I want to name it

with Field name: Facility. Can anyone show me how the MID function works in access 2003??

 

Many thanks,

revned12

ken sheridan say: Is the column a number or text data type?  If text execute the following two queries, firstly to populate the facility column:

UPDATE [YourTable]
SET [Facility] = LEFT([Document Number],4);

Then to remove the first 4 characters from the Document Number column from the:

UPDATE [YourTable]
SET [Document Number] = MID([Document Number],5);

As you see the Mid function returns a string of characters starting at the point specified by the second argument, 5 in this case.  If the length of the string to be returned is not specified it will return all character sot the end of the string.

If the column is an number data type then the queries would be:

UPDATE [YourTable]
SET [Facility] = VAL(LEFT([Document Number],4));

UPDATE [YourTable]
SET [Document Number] = VAL(MID([Document Number],5));

Do not populate the facility column without updating the document number column.  That introduces redundancy.  Do either both or neither.  If the latter you can always return the first four characters in a query with the LEFT function.

Hi Matthew,

 

If OneDrive.exe is installed on your Windows 10 computer but cannot be started, you may check whether the following two settings have been configured correctly.

  1. Check Registry.
    a)  Press (Windows key) + R to open the Run box.
    b)  Type regedit and click OK.
    c)  Navigate to HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\Windows\OneDrive. Make sure DisableFileSyncNGSC is set to "0". (It is also OK if the registry doesn’t exist).
  2. Check Group Policy settings.
    a)  Press (Windows key) + R to open the Run box.
    b)  Type gpedit.msc and click OK.
    c)  In the Local Group Policy Editor, in the folder list under Local Computer Policy, navigate to Local Computer Policy > Computer Configuration > Administrative Templates > Windows Components > OneDrive > Prevent the usage of OneDrive for file storage. Make sure it is set to "Not configured" or "Disabled".

hansv mvp say:

I use this macro to "clean up" blank cells. It replaces clears all cells within the selected range that contain an empty string "" after replacing non-breaking spaces with ordinary spaces and trimming leading/trailing spaces. If there are formulas within the selection that result in an empty string, they will be removed.

 

Sub ClearBlanks()
    Dim oCell As Range
    On Error GoTo ExitHandler
    Application.ScreenUpdating = False
    For Each oCell In Selection
        If Trim(Replace(oCell.Value, Chr(160), " ")) = "" Then
            oCell.ClearContents
        End If
    Next oCell
ExitHandler:
    Application.ScreenUpdating = True
End Sub
