00,3

04,7
98,1
99,2

How can I apply this custom sort order to the results of a query?

The details are as follows.  Previously, we named our projects xx-yyyyyyyyyy...   where xx is the last 2 digits of the year.  The issue is that the order put 99-  before 11-....  so I thought a custom order table would be the way to go but don't know how to implement.

Thank you

QuestionBoy

john p spencer say:

SELECT x, y, z
FROM SomeTable INNER JOIN tbl_Pro_SortOrder
ON Left(SomeTable.ProjectCode,2) = tbl_Pro_SortOrder.fPrefix
ORDER BY tbl_Pro_SortOrder.fSortOrder

Another way would be to use the VAL function and a test of whether or not the value returned is less than 98.
ORDER BY Val(ProjectCode)<98, ProjectCode
