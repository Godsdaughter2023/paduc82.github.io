Hi Gavin,

If you suspect virus is corrupting the files online which are synced on your local drive using the OneDrive sync client, then in this situation you must stop the OneDrive sync to save the data online. Disabling sync on OneDrive client will protect your cloud article which will give more information on versioning.

Data protection services are provided to prevent the loss of SharePoint Online data. Backups are performed every 12 hours and retained for 14 days. If the data is infected, it can be recovered by contacting Microsoft support. Assisted support Engineers should help you evaluate the period which will be needed to recover the data.

As far as your another question is concerned “How do I allow our users to continue syncing and editing documents in OneDrive / SharePoint without allowing File Operations; particularly mass deletion?”, well is this case, if you want the user to edit the files, you will have to give them Edit or Contribute permission.  When any of these two permissions are given, user gets delete permission automatically.

There’s no way to not have the users delete the file while giving them the edit permission.  

Reference: Understanding permission levels in SharePoint

doug robbins - mvp office apps & services (word) say:

You need to modify the paragraph formatting in the Envelope Address and Envelope Return styles.

The easiest way to do it is to create an Envelope and enter some text into the Delivery address space (and also into the Return address space if it is blank) and then click on Add to document and then press CTRL+SHIFT+s to display the Apply Styles dialog and then when you put the selection into either the Delivery address or the Return address, the relevant style will appear in the Style Name control in the Apply Styles dialog.  Use the Modify button on that dialog to access the Modify Style dialog and make the necessary adjustments to the formatting of the style and check the "New documents based on this template" radio button before clicking on OK in the Modify Style dialog to dismiss it.
