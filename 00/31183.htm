To format the External drive, we suggest you to first copy and save the files from the Drive in your Windows XP system and then connect the drive in Windows 10 and follow the below steps to format the drive:

1. Press Windows Key + X from your keyboard.

2. Select Command Prompt (Admin).

3.  Now in the Command prompt, type diskpart.

4. Next Type list disk.

5. Now select the disk (your external drive) where you want to perform the format. For example, type Select disk 1 where Disk 1 is the drive you want to format.

6. Type clean.

7. Now to make the drive from FAT to NTFS format, Type create partition primary.

8. Type format fs=ntfs quick.

9. Type active.

10. Type assign.

11. Type exit.

12. Type exit.

Let us know the status of the issue. we will be glad to assist you further.
