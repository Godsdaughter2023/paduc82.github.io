malkeleah say: All the hardware inside your computer (and outside, like your printer) has a corresponding piece of software called "drivers" that allows the operating system (Vista or Windows 7 for example) to use that hardware. Without the correct drivers, Windows might know you have the wireless hardware installed but it can't use it (no connectivity).

Before installing an operating system on a computer, the first thing the person should do is go to the computer mftr.'s website (Fujitsu Siemans in your case) and make sure that drivers are available for that operating system. You apparently didn't do this. On a generic/home-built computer you would get drivers from the individual hardware component mftrs.'s websites but on a laptop you only get drivers from the laptop mftr. It is particularly important on laptops to have drivers and laptop software (like power management software) for a specific operating system because the hardware inside laptops is proprietary. IOW, laptop hardware is made specially for that laptop make/model.

In some cases older drivers will work; i.e., you might be able to use Vista drivers in Windows 7. You never can use 32-bit drivers on a 64-bit operating system.

All that said, if there are no wireless network adapter drivers for Windows 7 on Fujitsu Siemen's website then your choices are:

1. Use an external USB wireless network adapter that will work with Windows 7; or
2. Contact Fujitsu Siemens tech support and find out if this laptop will be supported in Windows 7 (wait for the drivers to be written and made available); or
3. Restore your computer to factory condition by whatever method provided by Fujitsu Siemens and run the operating system for which the computer was designed.

Run the Vista System File Checker Tool:

http://www.tech-recipes.com/rx/2231/vista_run_system_file_checker/


1. Open a Command Window in Administrator mode:

* clickStart
* click All Programs, thenAccessories
* right click on the Command Prompt option,
* on the drop down menu which appears, click on the Run as Administratoroption.

* If you haven’t disabled User Account Control (and you shouldn’t!) you will be asked for authorisation. Click the Continue button if you are the administrator or insert the administrator password.

2. Start the System File Checker

* In the Command Prompt window, type:sfc /scannow,
* press Enter.
* You’ll see “the system scan will begin”.

The scan may take some time and windows will repair/replace any corrupt or missing files. You will be asked to insert your Vista DVD if it’s needed.

Close the Command Prompt Window when the job is finished.

One caveat: you need a Windows DVD to enable SFC to make more than minor repairs.

“How to analyze the log file entries that the Microsoft Windows Resource Checker (SFC.exe) program generates in Windows Vista”
