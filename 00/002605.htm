Based on what you have mentioned (some cells in the range C4:K5 are drop down lists and some of them are dependent drop downs), one thing is for sure that you cannot achieve what you want (All cells in range C4:K5 get blanked out when you delete the value in cell B4).

This may be possible only by way of writing a excel macro.

Your macro code can be like

Private Sub Worksheet_Change(ByVal Target As Range)
    If Target.Address = "$B$4" Then
            If Target.Value = "" Then
                Range(C4:K5).clearcontents
            End If
    End If
End Sub

This is only a structure which I have given. You may have to fine tune the code based on your actual scenario.
