Several points:

1. You can't do an update from XP to 7. You would have to clean install it.

2. Your computer might not be compatible with 7. You should first check to make sure.

3. You might need Windows 7 drivers for some of your hardware, and they might not be available.

4. You have to buy a copy of Windows 7 and follow the installation instructions that come with it.

5. Windows 7 is obsolescent. You can find copies for sale, but most of them are pirated. Be *very* careful.

6. It's likely that Windows 7's performance on such on old computer would be very poor.

7. Since Windows 7, there has been Windows 8, 8.1, and several releases of 10. As far as I'm concerned, it make no sense to go from one obsolete version to another. You should either stick with what you have or go to Windows 10.

8. Considering points 1, 2 3, and 6 above, you would very likely need to buy hardware upgrades. Those hardware upgrades plus the cost of Windows 7 might be as expensive as (or even more expensive than) buying a new computer computer with Windows 10.

So here's my advice: don't do this. Either stick with what you have or buy a new computer with Windows 10.
