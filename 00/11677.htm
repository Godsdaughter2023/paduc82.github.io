richie weldon - msft say:

There is no method for specifying which types of items (.wav, .mp3, etc) to synchronize. However, if you do not want certain files to be synchronized you can tether (connect via cable) your device to your computer and use Windows Explorer to mark those items as "Read Only" (Right-click the item, select Properties, check the Read Only box).  Items marked as "Read Only" will not be synchronized.  Do keep in mind that marking a folder as "Read Only" does not automatically mark the contents of that folder with the same attribute!  You will need to modify the attributes of the items in the folder directly.

creedo say:

If anyone is still struggling with this, I've had some luck fixing it.

Some PDFs are very complex and don't want to tag easily without spitting up errors. If you have full acrobat you can attempt to tag the files with advanced -> accessibility -> Add tags.

If you get errors like "acrobat was unable to make this document accessible" errors, the best thing you can do is:

1. Go advanced --> PDF optimizer and check every box under "discard objects". Optimize the PDF. Try to tag it now. If it still gives errors, then try...

2. Delete pages that might be causing problems. For example if you have a full book with a cover and back cover, delete those, along with any indexes or tables of contents that fall outside the usual numbered pages. Ideally just leave page 1 though (whatever) and get rid of pages that have been numbered with roman numerals or letters. Complex pages with forms and charts might also choke things up. After deleting these pages try tagging again. If it seems to complete (even with warnings or errors) then the PDF should be good to go on your phone.

The file size will usually increase noticeably when it works.
