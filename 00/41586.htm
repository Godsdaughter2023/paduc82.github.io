  4. To reinstall Windows Store in Windows 10 after removing it with PowerShell, run PowerShell as administrator and type the following command: Get-Appxpackage -Allusers
  5. In the output, locate the Microsoft.WindowsStore entry. Look for the text PackageFamilyName.
  6. Then, run this PowerShell command, still elevated, replacing the ****** with the PackageFamilyName: Add-AppxPackage -register "C:\Program Files\WindowsApps\******\AppxManifest.xml" -DisableDevelopmentMode

If you get an error message, you might need to take ownership of the WindowsApps folder. You may refer to the steps below:

  1. Go to C:\Program files using the File Explorer.
  2. Change the File Explorer settings temporarily to show hidden files and folders.
  3. Select the WindowsApps directory. Right-click on it and select Properties.
  4. Go to the Security tab and click on Advanced.
  5. The window Advanced Security Settings for WindowsApps is now visible. Click on the Change link next to the Owner label.
  6. The window Select user or group is now available. Type your user account in the area Enter the object name to select and  click OK.
  7. Set full access to the app folder for your account. Right-click on the WindowsApps folder > Properties > Security > Add.
  8. The window Permission entry for data is now visible. Click on Select a principal, then add your account. Set permissions to Full control.
  9. Run PowerShell as an admin and re-register all of the packages by typing the command: Get-AppXPackage | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}
