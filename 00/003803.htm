chip pearson say:

I think you would need a VBA function. E.g.,

 

Function IsWorkbookOpen(BookName As String) As Long
    Dim WB As Workbook
    Dim N As Long
    Dim FileName As String
   
    N = InStrRev(BookName, "\")
    If N = 0 Then
        FileName = BookName
    Else
        FileName = Mid(BookName, N + 1)
    End If
    For Each WB In Application.Workbooks
        If StrComp(WB.FullName, BookName, vbTextCompare) = 0 Or _
            StrComp(WB.Name, FileName, vbTextCompare) = 0 Then
            IsWorkbookOpen = 1 ' found
            Exit Function
        End If
    Next WB
    IsWorkbookOpen = 0 ' not found
End Function

Here, you pass in the workbook's name (e.g., "Book1.xls") or the full file name of a workbook (e.g., "C:\Folder\Book1.xls") and the function will return 1 if the specified workbook is open or 0 if the specified workbook is not open.

You can call the function from cell A1 as:

=IsWorkbookOpen("C:\Folder\Book1.xls")

Function IsOpen(FN As String) As Boolean
Dim myF As Variant
myF = Split(FN, "\")
On Error GoTo NotOpen:
    If Workbooks(myF(UBound(myF))).FullName = FN Then IsOpen = True
    Exit Function
NotOpen:
IsOpen = False
End Function
