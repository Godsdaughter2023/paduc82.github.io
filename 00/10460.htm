The "restricted" icon means that permission to modify an installed toolbar/extension by the user is denied.  By right-clicking a toolbar, you were usually given the option to "customize" it to some extent, allowing you to change functions inherent in the toolbar.  Unfortunately, in so doing, the user also inadvertently allowed new threats to enter the computer.  As a security measure, the security software has "turned off" this capability with a "No Toolbar Customization" key in the Windows registry.

You can’t import directly in Outlook because the app has fairly limited import options, but there is an easy workaround.
