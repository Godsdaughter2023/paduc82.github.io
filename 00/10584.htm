suzanne s. barnhill say:

If you want Saros to be the default font for the document (and it sounds as if you do), then one way to accomplish this is through the document defaults:

  1. Open the Styles pane and click the Manage Styles button (third from the left at the bottom).
  2. In the Manage Styles dialog, select the Set Defaults tab.
  3. On that tab select the font and font size you want as the default for the document.
  4. The radio button for "Only in this document" will be selected by default; leave it and click OK.

Alternatively (and more simply), you can do this through the Font dialog:

  1. Right-click in any paragraph and choose Font...
  2. In the Font dialog, select the desired font and font size.
  3. Click Set As Default.
  4. You'll get another dialog with "This document only" as the default selection; click OK.

The advantage of Method 1 is that you can also, at the same time, if you choose, set the defaults for paragraph spacing, indents, etc.

macropod say:

Hi Orcas,

The wildcard Find/Replace I gave you cannot do as you say. If you recorded the as a macro, you'd get code like:

Selection.Find.ClearFormatting
Selection.Find.Replacement.ClearFormatting
With Selection.Find
  .Text = "[^11^13]{1,}"
  .Replacement.Text = "^p"
  .Forward = True
  .Wrap = wdFindContinue
  .Format = False
  .MatchCase = False
  .MatchWholeWord = False
  .MatchAllWordForms = False
  .MatchSoundsLike = False
  .MatchWildcards = True
End With
Selection.Find.Execute Replace:=wdReplaceAll

This could be simplified a bit, but all it would do in any file is to change a document with:

The quick brown fox jumps over the lazy dog.¶
¶
¶
¶
¶
¶
The quick brown fox jumps over the lazy dog.¶
The quick brown fox jumps over the lazy dog.¶

to:

The quick brown fox jumps over the lazy dog.¶
The quick brown fox jumps over the lazy dog.¶
The quick brown fox jumps over the lazy dog.¶

It can have no effect on HTML tags, since it neither looks for nor replaces them.

What your latest post suggests is that you're not simply parsing the HTML but are opening the HTML file for editing as a document. You can solve any paragraph style issues by defining the style you need with the formatting you require, then applying them to every paragraph needing them throughout the document. However, you can't simply remove all styles - every paragraph must have a style - that's the way Word works.
