daniel g. grau say:
The language setting is a text attribute: a given portion of text can be defined as Bold, as 10 Points, as Red, and as English (CAN), French Canadian, etc. Just like there will always be a color to the text, there will always be a language to it.

What you need to do is tell Word to apply English (CAN) to the text. To do that:
  • Select the text
  • Go to Tools > Language
  • Choose the language
  • Make sure there is no checkmark in Do not check spelling or grammar

If you click on the Default button you add that setting to the document's original template.

 Another way to do it is to redefine the Normal style in your document:
  • Go to Format > Style
  • Select Normal in the list to the left
  • Click on Modify
  • Choose Language in the bottom dropdown menu and continue as before

vistas ad india say:

Follow these steps to create a Windows 8 Start Screen tile for a web site:

  1. Start the Internet Explorer app and visit the site you wish to add.
  2. After the site loads, right-click on an empty section of the web page. When you right-click you will see the address bar and favorites panel appear.
  3. To create a tile for the web site, click on the Pin button. This will open a menu asking if you wish to add the site to your favorites or to pin it to the start screen.
  4. Click on the Pin to Start menu option and a new dialog box will appear asking what you would like to name the tile.
  5. Internet Explorer will automatically populate the tile name with text recommended by the site. If you wish to change the name, type something else in the text field. When you are satisfied with the name, click on the Pin to Start button. The site will now be added as a tile to the Windows 8 Start Screen.
  6. To launch the site in the future all you have to do is click on the tile and Internet Explorer will open directly to the site. If the site has been programmed to use custom jump lists, you will see the Pin menu change to a jump list icon.
