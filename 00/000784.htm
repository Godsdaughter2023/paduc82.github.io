norman david jones say:

Hi EmeroGorK2,

A sheet module is normally used to house event code for that sheet; the ThisWorkbook module normally houses event code which relates to the workbook or multiple sheets. Non event code should normally be placed in a standard module.

For a detailed discussion, see Chip Pearson at:

             http://cpearson.com/excel/WhereToPutTheCode.aspx

To rename a module, highlight the module in the project explorer window and press F4 to open the corresponding properties window.
