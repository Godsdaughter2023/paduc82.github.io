dave's not here say:

I have run into this error on every Windows 7 machine I have worked on which has either just had a clean install of Windows 7 or has had Windows update turned off for an extended period of time.

The solution is that worked for me was:

1. Hide the update KB4056894.
2. Check for updates again.  You will be offered the update KB4054518.
3. Install update KB4054518 and restart the computer to complete the update.
4. Restore the hidden update KB4056894. 
5. Check for updates again.  You will be offered the update KB4056894.
6. Install the update and restart.  Problem solved!
