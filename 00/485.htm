Save Excel to Tab Delimited without quotes - Microsoft Community

ribower say:

Save Excel to Tab Delimited without quotes

I need to save a worksheet to a tab delimited file. some of the cells contain text strings that include commas. Excel is putting double quotes around the text string if it contains a comma. I need to get rid of the quotes.
peter schleif say:

The code below saves the active sheet in a text document. Same folder, same name as the active workbook - except the file extension .txt.

Peter


Sub SaveExcelToTabDelimitedWithoutQuotes()
   Dim r    As Long
   Dim arr  As Variant
   Dim file As Variant

   file = Left(ActiveWorkbook.FullName, InStrRev(ActiveWorkbook.FullName, ".") - 1) & ".txt"
   Set file = CreateObject("Scripting.FileSystemObject").CreateTextFile(file, True, True)

   With ActiveSheet
      arr = .Range(.[A1], .UsedRange.Cells(.UsedRange.Cells.Count))
      For r = 1 To UBound(arr)
         file.WriteLine Join(WorksheetFunction.Index(arr, r), vbTab)
      Next
   End With

   file.Close
End Sub

Follow these steps to run the troubleshooter:

a) Click Start button.

  3. After you get message “The operation completed successfully” type second command and press enter. It will take some time to finish.
    dism.exe /online /cleanup-image /restorehealth
Important: When you run this command, DISM uses Windows Update to provide the files that are required to fix corruptions.

ghizah say: i have bought a second hand nokia 3250 do you know how to remove the memory card if i go to organsier and memory card then remove it does not remove the memory card please help

xero90 say: Hi Ghizah,

The MMC slot on the 3250 is well hidden... I took me for ever to find it.

When you go into organiser and remove memory card it doesn't physically do so. All it does is closes any application that may be using the MMC at that time therefore preventing any damage to the MMC or handset.

Now to answer you question...

If you twist the bottom half a quarter turn and then look at the underside of the handset you will find a small symbol for the memory card. If you then look at the front of the handset and then down to the extreme bottom left corner (below menu key) you will find a small groove. If you take you nail and push down and away from the handset you will be given access to the memory card slot.
