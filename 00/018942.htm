Because you use the phrase "required rate of return", it is not clear to me exactly what you want to calculate using Goal Seek.  But I'll offer an example of my own.  I hope it is not too far afield.

Suppose you want to calculate the internal rate of return (IRR) for the following regular cash flows:  -10000 in A1, and 2000 in A2:A13 (12 periods).

Of course, we could use =RATE(12,2000,-10000) to calculate the "exact" IRR, namely 16.9425508579397%.

But you want to learn how to use GS.  (BTW, Solver is typically better to use because it offers more control.  But GS is certainly easier to use.)  The set-up might be as follows:

The IRR is defined as the discount rate that causes the "net present value" (NPV) of a series of cash flows to be zero (or relatively close to zero).  The math formula can be found in the help page for the Excel NPV function (click here) [1].  The Excel formula might be [2]:

B1: =SUMPRODUCT(A1:A13 / (1+B2)^(ROW(A1:A13)-ROW(A1)))

where B2 is the IRR.  So the GS set-up would be:

Set cell:  B1

To value:  0

By changing: B2

The result in B1 (Excel 2007) is about 3.55E-7 ("close" to zero).  And the result in B2 is 16.9425508570867%.  Compare with the "exact" IRR above.

Actually, there is one other alternative.  Place the chart on its own Chart Sheet...

 

Right-click on the chart > Move Chart > New Sheet > OK.

 

Review tab > Protect Sheet ("Contents" checked, "Objects" checked) > OK

 

That chart can no longer be editited (as long as the sheet is protected).  But you can still do Home tab > Copy and then paste it into another worksheet, or paste-special as a picture.

I want to copy the comments from word document into excel sheet - Microsoft Community

prakash devarasu say:

I want to copy the comments from word document into excel sheet

After I deliver word document report to the client, I am receiveing comments for mistakes that I have done in the report (Comments are given using the tab Review   -> New Comment ). I want to document the comments one by one in an excel sheet along with the page number without manually copying and pasting it. Atleast to copy comments from the report and paste them on another word document.

 

hansv mvp say:

Try this:

 

Sub CopyComments()
    Dim xlApp As Object
    Dim xlWbk As Object
    Dim xlWsh As Object
    Dim r As Long
    Dim cmt As Comment
    On Error Resume Next
    Set xlApp = GetObject(Class:="Excel.Application")
    If xlApp Is Nothing Then
        Set xlApp = CreateObject(Class:="Excel.Application")
        If xlApp Is Nothing Then
            MsgBox "Can't start Excel.", vbExclamation
            Exit Sub
        End If
    End If
    On Error GoTo ErrHandler
    Set xlWbk = xlApp.Workbooks.Add(Template:=-4167) ' xlWBATWorksheet
    Set xlWsh = xlWbk.Worksheets(1)
    With xlWsh
        r = 1
        .Cells(r, 1) = "Comment"
        .Cells(r, 2) = "Page"
        .Cells(r, 3) = "Author"
        .Cells(r, 4) = "Timestamp"
        .Columns(1).WrapText = True
        .Columns(4).NumberFormat = "dd-mmm-yyyy hh:mm"
        For Each cmt In ActiveDocument.Comments
            r = r + 1
            .Cells(r, 1) = cmt.Range.Text
            .Cells(r, 2) = cmt.Scope.Information(wdActiveEndPageNumber)
            .Cells(r, 3) = cmt.Author
            .Cells(r, 4) = cmt.Date
        Next cmt
        .Range("A1:D1").EntireColumn.AutoFit
    End With
ExitHandler:
    xlApp.Visible = True
    Exit Sub
ErrHandler:
    MsgBox Err.Description, vbExclamation
    Resume ExitHandler
End Sub
