Start each chapter (after the first) with a new page section break.

Each section of a document has up to three header/footer ranges (first page, primary and even pages).

You can set which header/footer ranges are used for each section from page setup. You are probably not using odd/even page header/footers so you are only interested in the first and primary header/footers. Set the different first page option for the whole document.

Unlink the primary header/footer from the first page header/footer for each section and put the page field in the primary header/footers. You could, if you wished, restart the numbering for each chapter, but that would seem an unlikely requirement here.
