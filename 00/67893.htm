malkeleah say: There is no magic way around this without any installation discs. Most laptops come with a factory restore image on a special partition on the hard drive. The factory restore process is normally invoked by pressing a Function key (like F11) at computer startup. There will be a message as to which Function key to press on the screen when you first start your computer. You can also refer to your computer manual, or the computer mftr.'s website, or contact its tech support to see how to restore your computer to factory condition since each computer mftr. does it differently - sometimes even between models.

These machines also usually have a utility with which to create physical recovery discs in case the hard drive needs to be replaced or the factory restore image is somehow damaged. If you did not create the physical recovery discs and damaged the recovery image, contact the computer mftr.'s tech support to order a recovery disc set. This is normally very inexpensive, around $20-25.

Before you attempt to restore the computer, you need to determine if the hardware is good. You can test it yourself:

http://www.elephantboycomputers.com/page2.html#Hardware_Tshoot

A good troubleshooting step is to boot with a Linux Live CD. I like Knoppix, but there are others. This has two advantages: 1) if the system will run, you can back up any data that missed your last backup; and 2) running the machine with Linux helps to determine if the problems are caused by software or hardware. If the machine is unbearably slow and/or Linux can't run correctly, you know the hardware is at fault. If the machine behaves perfectly under Linux, you know that software (Windows) is at fault.

If the hardware is faulty, whether you replace it or not depends on what component is bad and the age of the machine.

If the hardware is good (perhaps Windows was severely infected by viruses/malware and that's why it won't start any more for instance), then do a restore to factory condition using whatever method was provided by the laptop mftr.

If you can't do the work yourself (and there is no shame in admitting this isn't your cup of tea), take the machine to a professional computer repair shop (not your local equivalent of BigComputerStore/GeekSquad).

how to allow child to install app "virtual dj" - Microsoft Community

emmamccabe say:

how to allow child to install app "virtual dj"

Could I please have some help.

 

My son has downloaded the program/app "VIRTUAL DJ" which I don't have a problem with, but the Family Safety Network settings won't allow him to install it. I have logged in as administrator and installed the app without a problem, but it does not show up on his profile. Where have we gone wrong?

a. user say:

Hello Emma,

 

Family Safety normally blocks installation on a child account to prevent installation of softwares that might prevent Family Safety from working properly. What you can do is to disassociate first the child account then install again the software you want to install. Here's how:

 

1. Sign in to Family Safety website(https://fss.live.com).

2. Under Family Summary, click Edit Settings under the affected child account.

3. Under Accounts for <Name of the Child>, remove the username of the child by clicking Remove.

4. Now, log in using the child account and you can now install the software.

5. After installing, you can now add the child account by signing in again to Family Safety client and associate the user account to your child.

 

For additional information, some applications might ask you if you want to install the program on other users too. If this was asked to you and you set it that you want to install only on that specific user account, the program will not be visible on other accounts.

 

Thanks.
