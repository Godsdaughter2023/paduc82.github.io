cmdrkeene say:

That's not true.  This has been told many different ways but the base hasn't changed.  If you have transfer rights today, you keep them.  Here's the full details:

  • Did your current copy of Windows (Windows Vista, 7, or 8) come bundled with your computer?
    1. Yes, it came with my computer.
      You have no transfer rights. It doesn't matter if you install Windows 10 or not, any copy of Windows that comes with a PC is locked to that PC forever and can never be transferred to a new computer (self-bought or otherwise).  This is locked by OEM License and stays locked even if you upgrade it. That's why upgrade editions are cheaper (or even free).

    2. No, I bought it separately from my computer (on a CD/DVD/USB or download)
      You have transfer rights. It doesn't matter if you install Windows 10 or not, any copy of Windows you hold a single user license for is yours to take to any new computer you get or build. You aren't allowed to install it more than once at a time (you have to delete it from an old computer before you install it again elsewhere), but you keep that license forever, it's not tied to any particular computer.

The above remains the same before and after Windows 10. The only thing you need to know is that rule #1 stays the same as always.  You can upgrade the pre-bundled version of Windows, but not transfer it after.  You can't take a computer that shipped with Vista, upgrade it to 7 using a "Windows 7 Upgrade" CD (which is cheaper), and then take that Windows 7 to another computer later. It's against the license.

Windows 10 is the exact same way.  If you just install a free upgrade to your OEM-bundled system, you still don't get transfer rights, those stay locked to the original license even after the upgrade. If you want transfer rights, you have to buy the OS separately like it's been for 30 years. 

Everyone gets this way out of proportion because Windows 10's free upgrade is free instead of "just cheaper" like Windows 7, but the base rules didn't change.  Upgrade copies get the legal rights of whatever your existing license is, so either you have the license or it's tied to an OEM. If you bought your copy outright, it's yours to use anywhere, no matter if it's Windows XP, Vista, 7, 8, or 10.

ramesh srinivasan say:

There is a registry edit where you can set the default file name format for copies.


HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\NamingTemplates

Create a String value named "CopyNameTemplate" and set it value data as follows:

%s _

where %s represents the file name.

The copies would be named as:

"C:\Capture.PNG" (original file)
"C:\Capture _.PNG" (1st copy)

"C:\Capture _ (2).PNG" (2nd copy)

"C:\Capture _ (3).PNG" (3rd copy)

Without the registry edit, the file names appear as:

"C:\Capture.PNG" (original file)
