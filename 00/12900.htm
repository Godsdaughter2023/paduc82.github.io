johnx71 say:
I'm having the same issue with this template. Is anyone able to help yet? This would be a really useful template to use!
I hopefully found the solution - if you right-mouse click on the 'Compile List' button and select 'Assign Macro' - select the "CompileShoppingList" Macro, then click EDIT - It will open a VBA Coding window.

Scroll to line:

Sheet2.ListObjects(1).Range.AdvancedFilter xlFilterCopy, [WeekIngredientsCriteria], [WeekIngredients], False

This is a different colour from the rest of the code because it's been commented out! Just delete the apostrophe at the start of the line and then save the macro (top left of the screen).

Your button will now work! It took a little bit of figuring out but I can now use it! Remember to save the document after updating the Macro,

I hope this helps & that an MS Office employee can update this download.
