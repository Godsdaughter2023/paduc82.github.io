rick rothstein, mvp - excel say:

I'm pretty sure Mike is right and that the Integer declaration is limiting your code. However, I'd like you to consider this modification to your code which I think will be quicker (it processes the selection in memory and then writes the entire text string out to the file all at once rather than looping the columns and writing it out one row at a time)...

Sub QuoteCommaExport()
   ' Dimension all variables.
   Dim DestFile As String
   Dim OutText As String
   Dim FileNum As Long
   Dim RowCount As Long
  
   ' Prompt user for destination file name.
   DestFile = InputBox("Enter the destination filename" _
      & Chr(10) & "(with complete path):", "Quote-Comma Exporter")

   ' Obtain next free file handle number.
   FileNum = FreeFile()

   ' Turn error checking ON.
   On Error Resume Next

   ' Attempt to open destination file for OutText.
   Open DestFile For Output As #FileNum

   ' If an error occurs report it and end.
   If Err <> 0 Then
      MsgBox "Cannot open filename " & DestFile
      End
   End If

   ' Turn error checking OFF.
   On Error GoTo 0

   ' Process selection all at once
   For RowCount = 1 To Selection.Rows.Count
     OutText = OutText & """" & Join(WorksheetFunction.Transpose(WorksheetFunction. _
                                Transpose(Selection.Rows(RowCount))), """""") & """"
     If RowCount < Selection.Rows.Count Then OutText = OutText & ","
   Next
  
   ' Write OutText to file all at once
   Print #FileNum, OutText
  
   ' Close destination file.
   Close #FileNum
End Sub

jay freedman say:

The shortcut for EditCut hasn't exactly "changed" to Shift+Del. By default both Ctrl+X *and* Shift+Del are shortcuts for that same command. Something has reassigned Ctrl+X to another command, leaving only Shift+Del for Cut. The odd part is that there is no built-in command that can close Word without prompting to save changed documents -- that would require a specially written macro. I've never heard of shortcuts "spontaneously" changing. Have you installed any new add-ins, or other software that might include Word add-ins?

Anyway, you can restore the shortcut:

  1. Choose Customize on the Tools menu, then click Keyboard ...
  2. In the Categories list, choose “Edit ” .
  3. In the list on the right, choose the EditCut command.
  4. Enter the Ctrl+X key combination in the space provided.
  5. Before clicking Assign , select the template where you want the assignment to be stored.
  6. Click Assign , then Close .
  7. Hold the Shift key down and select File + Save All , to save your changes to the template.

If the shortcut gets reassigned the next time you start Word, then you're going to have to track down and exterminate the responsible macro or add-in.

2. Type or paste the URL using the ticker you are interested in 

3. Use File > Save As and change the file format to Plain Text

4. Choose MS DOS encoding

5. Save the file in Applications:Microsoft Office 2011:Office:Queries

6. In Finder, change the file extension of your saved query from TXT to IQY


Done!


Now in Excel go to the Data menu and choose Run Saved Query. Your saved query will be in the list. Give Excel a few seconds to digest the query and display the results.
