Hi,

 

OK I get it now I think the one thing I'm not fully sure about is the table row numbers which you said in your last post are in column Q but I'm not entirely clear which is row 1 of your table but in this formula I assume that row 1 of the table is the same row as row 88 of the worksheet.

 

So what we do is

 

1. Find the table row

2. In that row find the value in Col AG of the same row

3. Return the header row of that column.

 

 

The formula

 

=INDIRECT(ADDRESS(85,COLUMN(INDIRECT(ADDRESS(O98+87,MATCH(INDIRECT("AG" & O98+87),INDIRECT("W" & O98+87 & ":AF" & O98+87),0)+22)))))

 

If you look at the formula you will see 4 instances of the number 87. This number works out the row on the worksheet from the row in the table. As noted above I believe you're saying row 1 of your table is row 88 of the spreadsheet. so 87+1=88. If the actual row 1 of the table is different that row 88 then you will need to change the 87 in 4 places to get the rows correct.
