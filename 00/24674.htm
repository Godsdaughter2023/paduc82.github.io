#d09r dinesh say:

This error message occur because of recent malware removal done by installed anti-malware application, which lefts it's malicious loadpoints entries in registry or task scheduler.

To get rid of systemmenu.dll error, follow below steps;

  1. Press Windows Key + R
  2. In run dialog, type regedit and press Enter
  3. Navigate to HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run and Delete systemmenu.dll
  4. And press F3 or Ctrl+F, in search dialog box type systemmenu.dll and delete all the founded entries in search result.
  5. If didn't find any entries in the registry, then check it in Task Scheduler
  6. Open Control Panel > Administrative Tools > Task Scheduler
  7. Click on Task Scheduler Library
  8. Scroll through the list until you find systemmenu
  9. Right click on the entry, click on Del

    Note: You must use extreme caution when you use it to change registry values. Missing or incorrect values in the registry can make the Windows crash.
