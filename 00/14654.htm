faysal farooqui say:

Use Index-Match function in D column of Sheet1.

- Assume that, in Sheet1 your data starts from A2:C100000 where A1:C1 is the column header.

- In Sheet2 you have the final status in D column.

- Use the below Index-Match function in D2 cell of your Sheet1 and copy down to the end of the data table:

=IFERROR(INDEX(Sheet2!D:D,MATCH(Sheet1!C2,Sheet2!A:A,0)),"-")

This formula will return the result "DNC" or "NR" from D column of Sheet2 and if nothing found, then it will return a "-".

talking moose say:
A few things:

I have both CrashPlan and Time Machine enabled on my Mac. I've noticed that when Outlook hangs one or both of these may be backing up or starting a backup. Temporarily disable either of these to test whether it's causing the issue.

Also, if you're on a portable then you may be using Time Machine's feature to back up files locally without knowing it. See this for more information:

http://osxdaily.com/2011/09/28/disable-time-machine-local-backups-in-mac-os-x-lion/

If you're using an Exchange or IMAP account then keep your Inbox as clean as possible. Do not store sub-folders in your Inbox but rather outside the Inbox. Check your Drafts, Sent items and Deleted Items folder for messages that may be trying to sync to your server. I had a client with a 450 MB attachment in the Deleted Items folder. Deleting this message stopped her slow Outlook issues. Activity Monitor will show high network activity if this is happening.

Hope this helps!

To save an Excel chart in .PNG format, please follow these steps:

  1. In Excel, right-click on the chart border and choose Copy.
  2. In windows, open Paint.
  3. Paste the chart.
  4. Use Paint's File>Save As command and select PNG picture as the format.
