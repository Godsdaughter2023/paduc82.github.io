jay freedman say:

Put a name for the dropdown in its Title (in the content control's Properties dialog).

Put the following macro in the ThisDocument module of the template and modify it according to the comments. Note that the text in quotes in the "If CC.Title" statement must match what you put in the Title property with the same capitalization.

The user's choice will be copied to the table when the cursor exits from the content control, either by mouse or by keyboard.

If the place in the table where the text must appear isn't the entire content of the cell -- for example, if it's part of a sentence -- then you'll have to put a named bookmark in each of the three locations and change the macro to set the correct bookmark's .Range.Text to the CC.Range.Text.

Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, Cancel As Boolean)
    If CC.Title = "choices" Then        ' if CC title is something else, change quoted string
        With ActiveDocument.Tables(1)   ' if the table isn't the first in doc, change (1)
            Select Case CC.Range.Text
            
                ' CC.DropdownListEntries(1) is the placeholder,
                ' handled in the Case Else
                Case CC.DropdownListEntries(2)
                    ' if the text should go in a different column than column 2,
                    ' change the number after the comma in every .Cell()
                    .Cell(1, 2).Range.Text = CC.Range.Text
                    .Cell(2, 2).Range.Text = ""
                    .Cell(3, 2).Range.Text = ""
                Case CC.DropdownListEntries(3)
                    .Cell(1, 2).Range.Text = ""
                    .Cell(2, 2).Range.Text = CC.Range.Text
                    .Cell(3, 2).Range.Text = ""
                Case CC.DropdownListEntries(4)
                    .Cell(1, 2).Range.Text = ""
                    .Cell(2, 2).Range.Text = ""
                    .Cell(3, 2).Range.Text = CC.Range.Text
                Case Else
                    .Cell(1, 2).Range.Text = ""
                    .Cell(2, 2).Range.Text = ""
                    .Cell(3, 2).Range.Text = ""
            End Select
        End With
    End If
End Sub
