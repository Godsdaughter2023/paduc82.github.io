tricky300 say:

Hi,

If you are using XP home edition, the in-built administrator account probably does not have a password set. You can boot to safe mode where this account will be selectable at logon. Use it to logon and change the password of your normal logon.

To boot in safe mode, keep hitting F8 twice a second after turning on the PC until you get the boot menu where safe mode is selectable by the up/down arrow keys, then hit enter.

If you are using XP professional, try logging on using the built-in administrator account. You do not need to do this in safe mode (though you could if you want). If your normal logon screen is a user icon display hit ctrl-alt-del twice to get to a text input logon. Enter Administrator as the user name and try a blank password.

See also http://support.microsoft.com/kb/321305

The Mail app in Windows 10 has no way of storing messages like Windows Live Mail does. What you can do is upload your messages using an IMAP account, e.g. at Outlook.com, Gmail or Yahoo and then add that same account in Mail.

If you're normally signed in in Windows Live Mail, all you have to do to is to add the same account as you sign in with to the People app in Windows 10.

If you're not normally signed in in Windows Live Mail, then you will have to export your contacts to CSV, then sign in with a Microsoft Account at people.live.com and import them there. Add this same account to People in Windows 10 to synchronize the contacts.

Alternatively, you could just install Windows Live Mail and carry on using it. This would take some preparation, so please post back if you'd like to explore this possibility. In your reply, give the version of the program you were using. To find it, in Windows Live Mail press Alt-F B.
