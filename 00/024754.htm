**The macro will performs as per your requirements

- The moment you type a Completion Date that row will be transferred to "Completed Sites" sheet beneath the previous completed job.

- If by mistake the user type a Completion Date that is before Starting Date (i.e. Start Date =20/06/2019,   Completion Date=20/05/2019), a warning message box will prompt, and date in selected cell will be deleted.

Please check pictures

This is the macro

**********************************************************************

Private Sub Worksheet_Change(ByVal Target As Range)


Dim MyRng, TransfRng, Destination As Range


Set MyRng = Range("G:G")  ''' Sets the "Completion Date" Range


If Not Application.Intersect(Target, MyRng) Is Nothing Then

        ''Sets the Range (row) to copy

        Set TransfRng = Target.EntireRow

        ''Sets the Destination row beneath the previous completed job

        Set Destination = Sheets("Completed Sites").Cells(Rows.Count, "A").End(xlUp).Offset(1, 0)

                 If IsDate(Target.Value) Then

                 ''Case the entered Date is before Starting Date

                        If Target.Value < Target.Offset(0, -1).Value Then

                                Target.ClearContents

                                Target.Select

                                MsgBox "Please, enter a Valid Date"

                                Exit Sub

                        Else

                                TransfRng.Copy Destination

                                TransfRng.Delete

                        End If

               End If

End If
