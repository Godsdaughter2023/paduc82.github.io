Create your own graphics with bursts and stars - Microsoft Community

ellen finkelstein say:

Create your own graphics with bursts and stars

Technical Level : Basic

Summary
Use PowerPoint to create graphics for multiple uses.

Details

Do you need to create striking graphics? Here are some possible uses:

  • Blog posts
  • The first slide of a presentation
  • Ads and sales pages
  • Company posters and announcements

A nice way to do this in PowerPoint is with one of the burst  or star shapes. Here you see an example. This is easy to create in PowerPoint:

  1. Right-click off any object and choose Layout, Blank.
  2. On the Home tab, in the Drawing group, click the Shapes drop-down arrow and go to the Stars and Banners section. Choose any shape you like and drag it to the desired size on the slide.
  3. On the Format tab, use the Shape Outline button to format the outline. The burst you see here doesn’t have an outline; I chose No Outline.
  4. Again on the Format tab, use the Shape Fill button to format the fill. I used a radial gradient. You can see the settings here.The white color at the center looks like a  highlight.
  5. If you want, you can add a shadow to make the shape appear to hover off the page.  On the Format tab, choose Shape Effect, Shadow and choose one of the preset options or choose Shadow Options to customize your shadow. Note: I explain how to use the shadow settings in PowerPoint 2007 and later in this  video tutorial: Add drama with PowerPoint’s shadows.
  6. From the same Shapes drop-down list, choose Text Box and drag a text box on the burst or star. Type the text you want. PowerPoint always defaults to 18 point text in a text box, so you’ll probably want to change it.
  7. Click the text box’s border and press Ctrl + D to duplicate it as many times as you need to for your text.
  8. For most uses of PowerPoint, I say to be consistent and not use too many fonts. In this case, you can go crazy! Different sizes, fonts, and colors all add to the mayhem and make the shape more striking, so select the text in each text box and format it in any way you want.
  9. Adjust the spacing and size of the text boxes to make a pleasing array.  If you want, you can also rotate some of them, as I did.  You’ll probably also need to resize the burst or star and you can rotate that as well.
  10. Select everything. (If you used the Blank layout, you can press Ctrl + A to select everything.)
  11. Right-click anywhere inside the objects and choose Save as Picture and save it. Now, you’ll be able to insert it anywhere you need it.

Currently, there are more than a dozen versions [variations] of what is loosely referred to as "Times New Roman". I have 3 versions installed here [v3.05, v5.01.3x, & v5.05] which range in glyph count from 1,320 to nearly 4,000 characters. Although the basic design of the characters is consistent from one version to another, there are other differences, as well. If the document was created using one version of TNR it isn't uncommon for it to display somewhat differently on another computer - PC or Mac - which is using a significantly different flavor of the same font. Just because a certain font name is shown in the Font List & other dialogs does not mean that it is the font actually being used to display the text.

Try this macro:

Sub RenameSheets()
    Dim i As Long
    On Error GoTo ErrHandler
    For i = 89 To 1 Step -1
        Worksheets(CStr(i)).Name = CStr(i + 1)
    Next i

    Resume Next
