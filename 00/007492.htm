How to import all modules & user forms into an workbook - Microsoft Community

light blue programmer1 say:

How to import all modules & user forms into an workbook

 How can i import all user forms & modules at once from 1 workbook to the other?

I know how to do it with a single form. (export & import), but not with 2 together or more.

ossiemac say:

andreas killer say:

I know how to do it with a single form. (export & import), but not with 2 together or more.

There is no need to export/import modules or forms.

Open both files, within the VBA project explorer use drag&drop to copy the modules/Userforms.

When you need that often and/or with many modules, you can to that also with code, see the example below.

Requires a reference to Microsoft Visual Basic for Applications Extensibility 5.3

Andreas.

Option Explicit

Sub ExportPrim()
  Dim VBComp As VBComponent
  Dim Extension As String
  For Each VBComp In ThisWorkbook.VBProject.VBComponents
    Select Case VBComp.Type
      Case vbext_ct_ClassModule
        'Class modules
        Extension = ".cls"
      Case vbext_ct_StdModule
        'Modules
        Extension = ".bas"
      Case vbext_ct_Document
        'Thisworkbook, Sheets
        Extension = ".cls"
      Case vbext_ct_MSForm
        'Userforms
        Extension = ".frm"
    End Select
    VBComp.Export ThisWorkbook.Path & "\" & VBComp.Name & Extension
  Next
End Sub

Sub ImportPrim()
  Dim Extension, Extensions
  Dim FName As String

  Extensions = Array("cls", "bas", "frm")

  For Each Extension In Extensions
    FName = Dir(ThisWorkbook.Path & "\*." & Extension)
    Do While FName <> ""
      ThisWorkbook.VBProject.VBComponents.Import ThisWorkbook.Path & "\" & FName
      FName = Dir
    Loop
  Next
End Sub
