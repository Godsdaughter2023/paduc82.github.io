I ran this by the formula PM on the Excel team, and it's because we've effectively stopped supporting Lotus compatibility with the dot stop (.) for range references, so you can't use =SUM(A1.A10), but need to use =SUM(A1:A10). It had to go to be able to support linked data type references.

Also note that entering a formula with @ is going away to be able to support @Mentions, so you'll need to start formulas with =.
