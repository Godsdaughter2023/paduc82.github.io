techfreak_ say:
All international multi-country toll-free numbers have a registered area. This could be two or thirty countries. Skype only supports toll-free numbers in some countries which may have no toll invovled, others which have tolls involved or are totally unreachable. If you do not know which countries the international 800 number is registered with, you could start with the countries that are free from Skype which is France, Poland and the US. Simply select the France, Poland or US country code and input the international 800 number without any preceding zeros. Enabling Caller ID is always wise, in case that 800 number blocks anonymous calls, which is a seperate issue.
