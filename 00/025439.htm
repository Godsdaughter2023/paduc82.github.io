amit tandon say:

Your query:

What I wanted was;
If both the cell content of A2 matched & the cell content in D2 matched with the same cells in the file UK Supplier Orders OS_OLD_Sep12.xlsx, then return the content of cell K2.

In the file UK Supplier Orders OS_OLD_Sep12.xlsx, you are finding a match in column A for cell A2 value and a match in column D for cell D2 value. Both these matches can happen in a certain row, say row number 11 of both Column A & Column D. Your formula will return the value from column K, row number 11 (ie. where both A2 & D2 values are matched). In this sense your formula seems to be working fine.

In case your requirement is otherwise, please clarify.

You may alternatively use below to find a match from column K:

=IFERROR(INDEX('C:\Users\admin\Desktop\[UK Supplier Orders OS_OLD_Sep12.xlsx]UK Supplier Orders'!$K$1:$K$65536,MATCH(1,('C:\Users\admin\Desktop\[UK Supplier Orders OS_OLD_Sep12.xlsx]UK Supplier Orders'!$A$1:$A$65536=A2)*('C:\Users\admin\Desktop\[UK Supplier Orders OS_OLD_Sep12.xlsx]UK Supplier Orders'!$D$1:$D$65536=D2),0)),"")

What is new in IE9 Beta? - Microsoft Community

keith- say:

What is new in IE9 Beta?

How is IE9 different from IE8? What’s new? How is IE9 different from IE8? What’s new?
Keith
Microsoft Answers Support Engineer
Visit our Microsoft Answers Feedback Forum and let us know what you think
steven. s say:

1.      Browser Interface - The new interface in IE9 is minimal with all toolbars turned off by default. Also the address bar and search bar are now merged. The favorites button has been moved to the right hand side next to the Home and Tools buttons and along with it the Favorites and History columns. The status bar is also disabled by default, and most of the messaging is done through the notification bar that pops up towards the bottom of the page. All toolbars can be enabled/disabled by right clicking on the Home Button.

2.      Download Manager - There is a new download manager where we can view past downloads and their locations and also set a default save location. Like most explorer windows, the download manager also has an integrated search bar in the top right corner. The short cut for opening the manager is Ctrl+J. When a file is being downloaded the download bar pops up at the bottom of the IE window and provides options for saving or running files and also to pause, cancel or view downloads. If IE is closed in the middle of a download, the download will pause and then continue once IE is opened again.

3.      Add-on Manager – There are no major changes to the looks of the Add-on manager. However, the messaging and background functionality has been completely overhauled. There is a notification on first start-up which allows the user to disable any add-ons they need. If one clicks the ‘Disable Add-ons’ button, it opens up the IE Add-on Performance Advisor. When IE hangs or slows down due to an add-on the notification bar announces the same so as to keep the user aware of any problematic add-ons. If customers install any new add-on, on first restart of the browser, IE9 will pop-up a list of the add-ons that have been installed and the user will need to enable the add-ons for it to work.

4.      Backend Changes – IE9 will utilize the power of the GPU to render HTML5 multimedia elements. This includes played audio and video and allows for very fast rendering of these elements. Also the new Chakra JavaScript engine allows for faster page loads for existing webpages that use JavaScript. The overall memory usage as well as power consumption has also been improved allowing for better experience on lesser memory systems and a longer battery life.

You can also find additional information and demonstrations of the new features of Internet Explorer 9 by visiting the Internet Explorer 9 Beta – Product Guide.
