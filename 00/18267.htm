andreas killer say:
I use the following code to open Word reference from Excel (at the openning)

Your code did not "open a reference" and a reference could not be opened, it is only possible to set or remove a reference. Your code test if that reference is set, that's all.

Anyway, if you have Word 11 you can determine the correct GUID for it. Set a reference manually to Word 11 and execute the sub GenRef below. Copy the GUID from the direct window.

Andreas.

Sub GenRef()
  'Listet alle gesetzten Verweise auf
  Dim ref
  Dim s As String
  s = "Sub SetRef()" & vbCrLf
  s = s & "  On Error Resume Next" & vbCrLf
  For Each ref In Application.VBE.ActiveVBProject.References
    s = s & "  '" & ref.Description & vbCrLf
    s = s & "  Application.VBE.ActiveVBProject.References.AddFromGuid _" & vbCrLf
    s = s & "    """ & ref.GUID & """, " & ref.Major & ", " & ref.Minor & vbCrLf
  Next
  s = s & "End Sub"

  Debug.Print S
End Sub
