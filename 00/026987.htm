Hi A. Eng,

They are different.

The Setup.exe in the ODT cannot be directly used to install Office suite. In command portal we use command to install Office with this Setup.exe. To learn more details about ODT, see the following articles:

Overview of the Office 2016 Deployment Tool (in Command Portal, we use Setup.exe /download to download the office installer, use Setup.exe /configure to install the Office suite.)

Deploy Office 365 ProPlus from a local source

Configuration options for the Office 2016 Deployment Tool(with these properties in configuration.xml file, admin custom the installation of Office suite)

The Setup.exe downloaded from Office portal is a installer which can be directly used to install Office suite (double click it). It's different from the Setup.exe in ODT. Default footnote font - Microsoft Community

quintaluz say:

Default footnote font

How do I set the default footnote font in Word 2016?  I usually type in 13pt.  I'm a lawyer and that's the type size the courts I mainly practice in require.  Word, however, puts the footnotes in 10pt.  Word also sometimes changes the font.  My default is Times New Roman, but Word sometimes puts footnotes in Calibri.  I have computers running Win 7 Pro and Win 10.
jay freedman say:

Word automatically formats footnotes in the Footnote Text style, which defaults to 10 pt. You need to modify that style.

Press Ctrl+Shift+S to open the Apply Styles dialog. Type Footnote Text in the box and click the Modify button. In the Modify Style dialog, change the font size in the middle of the dialog, make sure the font name is Times New Roman, and click the "New documents based on this template" option at the bottom; then click OK. When you next exit from Word, if you're prompted to save changes to the Normal.dotm template, click the Save button.

You'll have to make the change separately on each computer.
