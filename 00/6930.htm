Misbehaviour of Word when using background colors and then saving as pdf. How can I avoid this?

Hi,

I want to export an word document to pdf by Save As function.

The word document has a two colored background:

But the exported pdf has a different color pattern:

Does anybody know how I can avoid this and export the correct color pattern to pdf?

Using a PDF Printer instead of "save as" shows the same problem.

Best

jay freedman say:

This is a longstanding bug in the last three or four versions of Word -- page backgrounds that use a gradient or a picture always print as tiles rather than as a full-page pattern. The workaround is to use a rectangle shape from the Insert > Shape button instead.

If you want the same background for all pages in the document, place the cursor in the header before inserting the rectangle. To have it only on a specific page, place the cursor in regular text on that page, and make the rectangle's text wrapping Behind Text.

In either case, drag the rectangle to the same size as the page. On the Drawing Tools tab, set the Shape Outline to No Outline. For the Shape Fill, choose Gradient > More Gradients, and set up the color and location of each of the gradient stops. To duplicate the gradient you showed, I set the stops at positions 0 and 100 to "Blue, Accent 4, Lighter 40%", and stops at positions 45% and 65% to "Light Yellow, Accent 1" (which is actually white).

If you print to paper, the printer driver will usually cut off the edges of the background (unless your printer supports "bleed" all the way to the edge). Saving to a PDF file, however, will retain the full-page gradient.

john sr wilson say:

It is:

If you right click the box > View code and in the right drop menu choose KeyDown

Add this code so you end up with:

Private Sub TextBox1_KeyDown(ByVal KeyCode As MSForms.ReturnInteger, ByVal Shift As Integer)
Select Case KeyCode
Case Is = 88 'X
Me.TextBox1.ForeColor = &HFF0000 'Blue
Case Is = 79 'O
Me.TextBox1.ForeColor = &HFF80FF 'Pink
Case Else
Me.TextBox1.ForeColor = &H0 'Black
End Select
End Sub
