Unfortunately, the Picture Tools don't seem to have any "Set as Default" or "Add to Gallery" provision for picture formatting. Your question intrigued me, as I've just finished working on a document in which I had added a black line around almost all of the pictures, and I wondered whether there would have been a quicker way, but apparently there isn't.

What I did was add the Picture Border command to the QAT, making it one click to add the default border. Word remembers the color and weight you last selected in a given Word session. julie sheets say:

A summary task (Task A in part 2) doesn't show a split like a "normal" task.  If you change task A to be autoscheduled, then its duration (and bar) will only be as long as its subtask.  Usually if a summary task has only one subtask - you can just eliminate one and go with one task.

trevor rabey say:

Task B has two parts, so make it two tasks, B1 and B2. If there is no "data", then estimate start date and duration of each part. In task information, check the rollup box.

You can't make A look the same in both situations because in 1 A is a task while in 2 A is a summary, not a task.
