Port Forwarding

Dynex DX-E401

Login info

Gateway – 192.168.0.1

Username – admin

Password – *none*

 

Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.0.1 in the address bar. (Or click the link)
  3. Login using the Username “admin” and there is no password.
  4. Select the Advanced tab at the top of the menu.
  5. Click the Virtual Server button on the left side of the screen.
  6. Enter Xbox Live for the name.
  7. Enter the IP Address of the Xbox in Private IP.
  8. Select TCP from the Protocol drop down.
  9. Enter 3074 for both the Private and Public Port.
  10. Select Always for schedule.
  11. Click the Apply.
  12. Repeat steps 6-11 with one change, select UDP from the protocol drop down.

 

Disable Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.0.1 in the address bar. (Or click the link)
  3. Login using the Username “admin” and there is no password.
  4. Select the Advanced tab at the top of the menu.
  5. Click the Virtual Server button on the left side of the screen.
  6. Locate the Xbox Live entries in the table at the bottom of the screen.
  7. Click the garbage can icon next to these entries.
