keith f williams say:

Hi,

Had the very same issue and have spent ages hunting around to find that it is in some pesky settings...

Head to File -> Options -> Advanced in Visio.

Scroll down to the display section where you will find a section called "Stencil spacing", for some reason this is set way too low and short -> like 5 characters and 1 row.

So beef this up with using say 15 characters over 3 lines per master or whatever makes it work for the text on your stencils.

The next pass if this does not work is then to close out form Visio, find a stencil file and edit it.  then choose say "Icons & Names" to see the text to the side or "Names under Icons" to see the text below them.

Hopefully that should fix your and also others with the same problem as it has worked for me.
