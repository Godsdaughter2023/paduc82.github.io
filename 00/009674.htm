petrib say: The phone's email support is for Internet standards for email access.

For sending/outgoing email that means SMTP (Simple Mail Transfer Protocol). In other words, you need to specify your email service providers SMTP server for the outgoing mail (or your phone network operators SMTP server, if the operator requires that their server is used).

For receiving/incoming email, that means either POP3 (Post Office Protocol version 3), or IMAP4 (Internet Mail Access Protocol version 4). Depending on the mail service provider, they could limit you to use one or the other, or allow either one.

In the case of Hotmail the problem is that Microsoft has chosen not to support these Internet email standards at all for accessing Hotmail mailboxes, as far as I know.

Until Microsoft does that, you can't use the phone's email support for direct email access.

However, you should be able to use the phone's built-in WAP browser ("Services", or whatever it is called on the 6230i) to access Hotmail through http://mobile.msn.com/wml/

There could be some intermediate email service providers that can act as a front-end service to provide this SMTP and POP3/IMAP4 access to their servers, and they access Hotmail using Hotmail's proprietary/non-standard mechanisms, or whatever. With such an intermediary, you would have to give your Hotmail/Passport username and password to the intermediary. Not necessarily very safe.

Sub CountByFillColor()

Const oRw As Long = 4 'Row offset for data range

Const oCol As Long = 4 'Column offset for data range

 

Dim lR As Long, ctColor As Long, sTxt As String, vA As Variant, R As Range

Dim sColor As String, vColor As Variant, Colors As Variant, Chk As Boolean

 

lR = Range("A" & Rows.Count).End(xlUp).Row

Set R = Range("E5", "AG" & lR)

vA = R.Value

Colors = Array("blue", "green", "red", "white", "yellow", "none")

sTxt = InputBox("Enter the letter or phrase to count")

If sTxt = "" Then Exit Sub

sColor = InputBox("Enter the color fill to count as: blue, green, red, white, yellow or none")

If sColor = "" Then Exit Sub

For i = LBound(Colors) To UBound(Colors)

    If LCase(sColor) = Colors(i) Then

        Chk = True

        Exit For

    End If

Next i

If Not Chk Then

    MsgBox "The color you entered - " & sColor & " - is not allowed - try again"

    Exit Sub

End If

Select Case LCase(sColor)

    Case "none": vColor = xlNone

    Case "blue": vColor = vbBlue

    Case "green": vColor = vbGreen

    Case "red": vColor = vbRed

    Case "white": vColor = vbWhite

    Case "yellow": vColor = vbYellow

End Select

For i = LBound(vA, 1) To UBound(vA, 1)

        If CStr(vA(i, j)) = sTxt Then

                    vColor Then ctColor = ctColor + 1

            End If

        End If

    Next j

Next i

MsgBox ctColor & " Cells contain the text " & sTxt & " and are " & sColor & " filled"

End Sub
