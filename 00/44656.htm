How To Find SSID (Network Name)

Login Information

 

  • Gateway: 192.168.1.1
  • Username: root
  • Password: 12345

 

 

How To Find SSID (Network Name)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Basic" from the sub menu.
  6. The wireless network name will be labeled as "SSID".

a. user say:

There are two types of 'bad sector:'

A physical bad sector is bad because of some physical defect. It's not uncommon for a hard drive to have a few of these. Windows simply marks them as 'bad' so they won't be used for storage. You only need to worry if bad sectors begin to proliferate.

A logical bad sector is bad because of a software problem, e.g., Windows can't find it or its description has contradictory information. Running CHKDSK will usually fix that, but again, it's not uncommon and only a problem if Windows tells you it's a problem.

All modern hard drives ship with a utility called S.M.A.R.T. whose purpose is to monitor the drive for problems and notify the user before problems become serious. If you're not getting a notification from S.M.A.R.T. and you're not receiving error messages from Windows, you've got nothing to worry about.

If you still think there's a problem, then run the hard drive manufacturer's own drive diagnostic utility. You can download it from the manufacturer's website, and the manufacturer will show you how to interpret the results.

If you're determined to replace your hard drive, then I guess you'll do it anyway. But why go to the trouble if you don't really need to?
