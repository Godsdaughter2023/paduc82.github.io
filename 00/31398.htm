Hi,

 

Thank you for posting your question in the Microsoft Community Forums.

 

This is happening because Windows 7 added support for eSATA functionality. Some hardware vendors have incorrectly configured all SATA ports as eSATA ports. This incorrectly configured settings causes all internal and external SATA devices to be treated as removable drives. This issue can affect hard disks, DVD drives, and other hardware attached to the internal SATA ports of the system. To resolve this issue you should contact the computer manufacturer for a BIOS update or other type of resolution to correct the misconfiguration of the internal SATA ports.

 

Note: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

what is windows process activation - Microsoft Community

earlgriffis iv say:

what is windows process activation

Taking a look at the windows features section in Installed Programs for windows 7 - what is the Windows Process Activation Service?  I just installed Win7 Ultimate and looking to initiate some added features to make my computer run better.

ronnie vernon say:

Hi

The Windows Process Activation Service is one of the optional components in the Windows 7 features.

Unless you are a developer, or you use the IIS component, this service is not usually required in Windows 7.

Open Control Panel / Programs and Features / Turn Windows features on or off (left side menu).

After the list populates, scroll down to the Windows Process Activation Service and click the + sign to expand the component. This will show the 3 components that are installed with this service.

More Information:

"Windows Process Activation Service" : The Official Microsoft IIS Site:
http://www.iis.net/search.aspx?q=Windows+Process+Activation+Service

WAS Activation Architecture:
http://msdn.microsoft.com/en-us/library/ms789006.aspx

Hope this helps.
