constanceenglert say:

Ok.  I have painfully figured this out.  Disregard the various suggestions below from others to the contrary.  No nuclear option is necessary.

In order to use OneNote across varying Win platforms (Win10M, Win10D, WinRT), you MUST have the OneNote file in the specific OneDrive\Documents\OneNote\Notebooks. There is no redirecting - the ON file MUST reside in this directory if you want Win10 Mobile to pick this file up.  You can redirect your Win10D OneNote to any location but Win10 Mobile can only pick up a ON file from this specific OneDrive file location - no exceptions.  If it isn't there, it isn't going to pick anything up and creates a new blank notebook.

If you haven't shared your existing ON file, it's easy enough to move your old notebook to this specific location.  If you have "shared" it - even with yourself (work/personal), you must create a new ON file in the specific directory and then (painfully) move each section independently from the old location to your new notebook in the new location.  This is covered in the Office support section.   I suggest the old school DOS trick of renaming your existing notebook OLD and then copying into a newly named file in the new location to preserve the original file in case something terrible happens.

Lastly, in your Win10 Mobile device, go to your OneDrive application, open the above OneNote\Notebooks directory, and then open each of the notebooks you want to use from OneDrive - NOT initially OneNote.  OneDrive controls the file, OneNote lets you use it.
