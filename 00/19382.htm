andrea jones - all about resources say:

Is the actual value entered in the cell 2.9M (you can look in the formula bar to check)?  It is possible to make 2900000 entered in a cell look like 2.9M using a custom format.

If the actual value in the cell is 2.9M then you can use a formula like

=VALUE(LEFT(A1,LEN(A1)-1))*1000000

to convert a value in cell A1 from 2.9M to 2900000.  However, if some cells have values like 1.5K for thousands then you will need a different formula.
