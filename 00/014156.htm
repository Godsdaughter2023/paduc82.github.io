sllee say:

Another user has asked me the same question so I looked into it some more....

 

It's not fully automated as, you are right it cannot be done by Outlook, but it is a lot easier.

 

In the end I created a script as below:

 

Sub Item_PropertyChange(ByVal Name)

Select Case Name

          Case "Status"

                  if Item.Status = 2 then '2 = Completed   

                  Set NewItem = Application.CreateItem(0)

                  NewItem.To =

                  NewItem.Recipients.ResolveAll

                  NewItem.Subject =

                  NewItem.Body =

                  NewItem.Display

            End IF

End Select

End Sub

 

If you publish this as a custom task and then create a task from it with a reminder, at the time specified it will prompt you to send the email. Once it is marked complete it will generate the email and then all you have to do is hit Send.
