van_m say:

Hi,

 

You may try to check on it by using this steps.

 

  1. Sign in to the Skype application.
  2. Go to one of your Skype contact’s messaging field.
  3. Type in the command “/showplaces” and send it as a message. This will show all of the end point on where you have logged in on a Skype application.
  4. In order for you to be logged out on the other device. Go again to the messaging field of any of your Skype contacts and Type in the command “/remotelogout” and send it as a message.

 

The command wouldn’t appear as a message on the other end user as it is a special command for signing out on all Skype application for your account.
