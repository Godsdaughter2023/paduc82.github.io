It seems it's people who upgraded who are having trouble. Try a clean install of Windows:

  1. Since you've already upgraded to Windows 10, your computer is now licensed with Microsoft, so there's no need for a key.
  2. Download Windows 10 from Microsoft and put it onto a USB stick or a DVD: https://www.microsoft.com/en-us/software-download/windows10
  3. Back up all your files since you'll be wiping your disk
  4. Boot from the device where you downloaded Windows 10
  5. Choose "Custom Install", and select the disk where Windows 10 is to be installed
  6. Select "Format", or if "Format" is greyed, select "Delete"
  7. Click Next to install Windows 10
  8. When prompted for a key, just click Skip, your computer will activate automatically after Windows starts
  9. When done, reinstall any software and restore your files, and install all Windows Updates

That's it. You'll now have a clean Windows 10 installation and everything should work now.
