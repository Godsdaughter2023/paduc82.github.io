bossdweeb say:

Nope! Nothing 'fake' about the dictatorial forced Win10 update routine.

I would think having to do the 'in place upgrade' is a desperate last hope measure, but it does look like the updates

may be failing (Win10 used to show failed updates on the list, but I don't know if it still does).

- Have you tried running Windows Updates 'manually' from Settings rather than allowing Windows to do them 'automatically' ?

You cannot prevent WU from downloading updates in the background if you allow WU to do them automatically.

The second Tuesday of each month is generally the day to watch out for, so I usually run WU around that time to avoid

being caught out.

Some updates come at other times (eg- The Meltdown/Spectre fix), so I also run WU before doing anything 'big' on line

as the updates can slow other downloads or on line gaming.

I also go off line to play single player games to avoid surprises.

- Similarly, when the bi-annual 'feature' updates roll around it's safer to use the Update Assistant that MS releases for each

new feature update, and install the update at a time that suits you rather than allow WU to download it.

You didn't say exactly how you have set up WU, but...

- Newer versions of Win10 have extended the Active Hours from 12 to 18 hours a day, so set up Active hours to cover

the time you are most  likely to be using your machine (8AM > 2AM works for me).

- The Restart option > Schedule a time, on my Win10 is greyed out, but should be on 'off' (I set this option when first installing Win10, but don't know why it's not available now).

- I turn on > Show more notifications.  This just lets you know an update that has been downloaded in the background without

your knowledge is going to be installed when you shut down, and gives you the option to update and shut down or update and reboot.

I use the reboot option so I can verify the update has not corrupted my Windows, rather than wait for a surprise later.

- Disable  both options in Choose how updates are installed

- Disable Delivery Optimisation > Allow downloads from other PC's

- I also have the Download/upload settings all at minimum (maybe not a good idea if you allow WU to do the

large 'feature' updates).
