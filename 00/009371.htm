The updates for Windows Defender definitions are cumulative (e.g. contain all previous updates) and are completely refreshed when the engines are updated approximately once each month.

For this reason, there's no need to reinstall older definitions updates, so the only question is whether the current definitions are installing properly, which indicates everything's fine if so?
