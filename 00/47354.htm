Hi Jerry,

By default, Windows 10 opens applications in a "Normal" window size, meaning it won't fill the entire screen. While users can maximize the app window's size manually by clicking the middle icon in the upper right-hand corner, many would rather not have to repeat the action every time they start an application. To force Windows 10 to open a application in full-screen mode always, kindly follow the steps below:

  1. Right-click on any Windows 10 application or the application shortcut and select Properties.
  2. In the properties window, click the Shortcut tab (as shown in the image below).
  3. Locate the Run: section, and then click the down arrow on the right-hand side.
  4. In the drop-down menu that appears, choose Maximized.
  5. Click Apply, and then OK. The application should now open in a full window.

    Deleting "System Volume Information" and similar such as "recycler" must *not* be removed, removal will result in corrupt File system!

    If you have several user accounts you need to empty recycle bin for each user by normal means, not by taking ownership of any kind. ( that is log into user account and empty it's recycle bin, and run disk cleanup as explained below )

    do the same for each partition and hard drive ( plug external HDD in while clearing recycle bin )

    The best method to backup your personal data is by using at least 1 external HDD or USB stick.

    Online storage is also good but only for non confidential data.

    Do not encrypt your external storage, sooner or later you'll lock your stuff out and there is no way back in that case.

    System backups are good for system recovery ( backup of system files *only* ) and these do not take too much space.

    You can reduce or configure this space in:

    Control Panel\System and Security\System

    Advanced system settings

    System protection tab

    Configure button

    And

    System restore button

    Once you configure above make a disk cleanup for each user account.

    Control Panel\System and Security\Administrative Tools > disk cleanup

    I lost a lot of data while doing backup on laptops ( laptops die or get stolen sooner or later )

    I lost data while trying to encrypt stuff ( you loose your decryption key sooner or later )

    I lost data while using only one external storage ( hard drives die over time or get hit by corrosion )

    I lost data while doing backups on CD/DVD ( CD/DVD backup s**uks )

    Keep confidential stuff on EXTERNAL HDD/USB, stuff on external storage can't be "hacked" like other pluged in storage devices.

    do not use programs of any kind to transfer data to external storage, use copy/paste *manually*.

    Therefore currently, I use 1 external HDD, 2 USB sticks and 1 SD card + online storage.
