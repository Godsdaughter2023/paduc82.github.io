Did the other users get the font from you, or did they download it independently? Fonts from different sources can be set up either as single fonts, mini-families (regular/italic pairs) or full families (regular, bold, italic and bold italic) and all these variants will interact differently with PowerPoint.

Have you tried embedding the fonts when saving? Here are the steps:

  1. Choose File>Save As>Browse.
  2. Click on the Tools dropdown beside the Save button.
  3. Select Save Options.
  4. Check Embed fonts in the file.
  5. If the presentation is to be editable by the recipients, choose the Embed all characters option. OK out and save the file and reditribute.
