graham mayor say:

It's been a while since I used Word 2003, but I think this is a limitation of that version. Later versions save such reference labels in the normal template. Word 2003 does not appear to do that even if you open the normal template directly. However it is easy enough to workaround.

Add the following lines to an autoopen macro in the normal template.

Dim oCapLabel As CaptionLabel
Dim bFound As Boolean
For Each oCapLabel In CaptionLabels
    If oCapLabel.name = "fig." Then
        bFound = True
        Exit For
    End If
Next oCapLabel
If Not bFound Then CaptionLabels.Add name:="fig."
