When using Word 2003's Find & Replace, the Special Options are - Microsoft Community

cherylcuddeback say:

When using Word 2003's Find & Replace, the Special Options are only available for the Find Field. Is there a work-a-round for letting these same options be available in the Replace Field?

I would like to Find a paragraph mark followed by any digit and a space and Replace the space with a tab.  In my Find Field I type ^p^# and a space.  In my Replace Field the Any Digit Option is not available.  How do I get around this?
hansv mvp say:

Tick the check box Use wildcards.

In the Find what box, enter   (^13[0-9])   followed by a space.

In the Replace with box, enter   \1^t

Click Replace All.

 

Explanation: ^13 has to be used instead of ^p in the Find what box if you use wildcards.

[0-9] means any character in the range 0 ... 9, i.e. any digit.

The parentheses ( ) mark a part of the Find what string. The first part can be referred to in the Replace with box as \1, the second part as \2 etc.

The code \1^t in the Replace with box means: part 1 of the Find string followed by a tab character.

hansv mvp say:

If you want to allow for 1 to 3 digits, you can also use (^13[0-9]{1,3})

 

{1,3} means that the previous character (in this example a digit) occurs at least 1 and at most 3 times.
