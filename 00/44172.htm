From JW Stuart's post

"Acer: press and hold ALT + F10 as soon as you see the logo"

This should work even if the installed operating system is corrupt and will not boot. What do you get?

The two main reason why even this fails are the user has removed the recovery partition or the disk boot record is damaged and the BIOS cannot access the recovery operating system.

If this is the case then you have to be suspicious of the drive. You say the disk has been tested and is OK but how? Some drive faults cannot be identified using rudimentary tests.

If the disk is OK but you cannot get the Recovery utility to run then you have to assume that has no further use. Boot from the Windows 7 DVD. Select the repair option, go to command prompt and enter DIskPart. (link provided in 1st post for a guide)

Use that to clean the drive which will remove ALL partitions and reset the master boot record. Exit DiskPart and reboot and boot Windows 7 setup as normal, it should see the HDD as a completely empty blank drive.

You can sometimes request a replacement disk set from the manufacturer for the ones that should have been made when the PC was new. This would be the preferred option as it contains all drivers and Acer provided utilities from new.
