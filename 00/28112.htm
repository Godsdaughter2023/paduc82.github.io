Just a quick additional information for your future reference.  If you are looking up from a query and you don't have the same field name in more than one table, you don't need to include the table name (see my example below) and if you are looking up from a table you also don't need to put the table name in the criteria:

john w. vinson say:

The three arguments to a Domain Function such as DLookUp must all be text strings;you're passing the actual fieldnames and so on - not strings. Try

=DLookUp("[Wall Thick]", "[tblSSPipeSizes]", "[tblSSPipeSizes]![Pipe OD]=" & [cmbHXRShellOD] & " And [tblSSPipeSizes]![SS Schedule]='10S'")

This will give literal strings for the first two arguments (you don't need the tablename unless there are two fields both named [Wall Thick]) and will construct a text string for the criteria by concatenating literal strings with a form reference (I'm assuming that there is a combo box on the form named cmbHXRShellOD and that its value is numeric).

Note that I delimted the 10S criterion with singlequotes ' since it needs to be embedded in a text literal delimited with " characters.

moving pages around in one document - Microsoft Community

safety 7 say:

moving pages around in one document

I woul like to know how to move whole pages around within a single document.
jay freedman say:

Word doesn't have a firm idea of what constitutes a "page" -- it constantly recalculates the page breaks based on the document content and the feedback from the printer driver.

If you have inserted manual page breaks, you can select, cut&paste a page (including the break) just like any other text.

If each page starts with a heading that has an outline level, you can use the Outline view; collapse the outline so that level shows only the heading, and then drag the headings up or down through the outline. The text hidden within the heading will move with it.

If you have Publisher, export the document to that program, which does have a fixed container for each page.

Jay Freedman
MS Word MVP  FAQ: http://word.mvps.org

Hi dineshk0870,

You can synchronize contacts, calendar and mail from gmail to the N8 by creating a Mail for Exchange account. Goto Mail and create a new mailbox, enter your gmail address and password and choose to create a Mail for Exchange account.

 

You now need to enter the following parameters:

 

Username and mail address  - your mail address

Password

Domain - empty or google

server address - m.google.com

 

When you have set this up you can change settings specific to the google account by going to m.google.com on your N8 and logging in with your gmail credentials.
