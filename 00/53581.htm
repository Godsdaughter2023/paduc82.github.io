brian tillman [outlook mvp 2007-2019] say:

I don't see what this has to do with Hotmail, a Microsoft account, or SkyDrive.

 

If you want to rip the contents of a cassette, connect the earphone jack of the cassette player to the line in or microsoft jack on your PC using a cable with a stereo miniplug on each end.  Then download Audacity from http://audacity.sourceforge.net/.  You can use it to capture the output from the cassette.

 

What I use is an old program called "Analog Recorder" from a package called "Microsoft Plus! Digital Media Edition". released in 2003.  It was meant for Windows XP Media Edition, but it works on Windows 7.  I have a small mixer to which I've connected a cassette player and a turn table.  The mixer plugs into the Line In of my sound card.  Analog Record samples the sound from the mixer and can detect the silence between tracks, so it will automatically split a cassette or LP into tracks for which you can then supply tags like album, track name, artist, and genre.  They're saved as Windows Media Audio files.

 

Another approach would be to purchase a product from Ion called "Tape 2 PC" (see http://www.ionaudio.com/products/details/tape-2-pc) which converts cassette audio into AU files for iTunes.  This is the cassette player I use, although I prefer using my sound card rather than the direct USB interface because I don't want iTunes on my Windows system.

 

If you have LPs to convert, Ion makes products for that as well.  One that converts both cassettes and LPs is http://www.ionaudio.com/products/details/duodeck.  For $80, it's hard to go wrong.

jvdev# say:

Secure Connection Failed - Solution

Avast Anti-Virus was the cause in my case

Here was my situation -

  • I had two PC's that I upgraded to Windows 10
  • I changed the default browser to Firefox (version 39) in both of them.
  • When using the Search Window at bottom left of Windows 10 screen labelled "Search the web and Windows", I consistently came up with "Secure Connection Failed" regardless of what I was searching for.  This also occurred within Firefox.  This occurred on one of my two PC's, but not the other.
  • I discovered on another forum that Avast might be the problem
  • Sure enough, on the PC that worked, I was using Windows Defender for anti-virus.  On the PC that failed I was using Avast Anti-Virus.
  • On the PC that failed, I temporarily disabled Avast, and by golly the "Secure Connection Failed" message was no longer a problem when I entered something in the Search window.
  • I re-enabled Avast and the "Secure Connection Failed" error message reappeared.
  • So it appears that using Avast Anti-virus along with Firefox and Windows 10 somehow created the problem.
  • I have switched over to Windows Defender on the PC that was causing problems.
