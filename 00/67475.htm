considerate_guy say:

If you just want a backup copy of the file, you can simply use Windows to copy a video file (e.g. *.mp4) onto an optical disc (e.g. DVD) and you'll have made a data disc copy of your video.

This copy will be understood by other PCs that understand the .mp4 format as well as Blu-ray players that can directly play .mp4 files.

However, the disc will NOT be understood by a standard DVD player. The video DVD standard requires different software to create the disc in the standard format by transcoding the video to MPEG-2, adding optional menus, etc. For that you'd need separate software for DVD authoring. While Windows Vista and Windows 7 had Windows DVD Maker that could do this, the program has not been included with Windows 8, 8.1 or 10.

If you are on Windows 7 and are using Movie Maker, then it will figure out that you have Windows DVD Maker. Movie Maker will then give you the option to "Burn a DVD" using Windows DVD Maker. If you don't see that option, it's because you are on a later version of Windows where the program isn't present.

Note: The video DVD standard is a maximum of 480p so isn't really good for HD video.
