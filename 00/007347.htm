brian tillman [outlook mvp 2007-2019] say:

You should log into http://mail.live.com/ using the exact address and password you've used for the Outlook Connector account and see if the banner at the top says "Outlook.com" or "Outlook Mail (Preview)". If it says the former, the Outlook Connector should still work and if it doesn't there's a problem. If the banner shows the latter, then your Outlook Connector will no longer work and you should change to using an Exchange account by using automatic account configuration. Click File>Account Settings>Account Settings>New>Next. Supply the same credentials you supplied in the Outlook Connector (Name, Address, Password), then click Next and allow Outlook to configure the account. Once the account configuration is complete, you'll be able to remove the Outlook Connector account for that address.

If you have any accounts on Outlook.com that still show "Outlook.com" in the banner, keep the Outlook Connector installed until they, too, transition to "Outlook Mail (Preview)". Once all Outlook.com accounts have transitioned, uninstall the Connector because it will then be obsolete.

vassilischristopoulos say:

HTC HDT, WP7 7.5, "Can't send sms"

Although I receive sms messages, I cannot send any sms using my new HTC HD7, running WP7 7.5 (OS version 7.1.7720.68, firmware version 2250.21.40502.401, Bootloader version 4.5.2250.0 (129185)).

My phone provider is Greek Cosmote, have double-checked the sms center number (+3097100000). Have also tried HTC's Connection Setup without success.

Have tried also to send sms having de-activated the Wi-Fi or 3G, but nothing!

Have tried also using the SIM card of other providers (Wind and Vodafone), nothing!

I cannot understand if this problem is related to Microsoft WP7 version 7.5 or HTC's firmware or finally is a hardware problem (although I have seen in several internet forums that the same problem appears on other manufacturers devices like Samsung Omnia 7).

I have contacted also my provider (Greek COSMOTE) and HTC's support, but they do not have an idea how to resolve this.

 

PS. My HTC HD7 is now Chevron WP7 unlocked, but the "can't send sms" problem was there even before doing the Chevron WP7 unlock.

vassilischristopoulos say:

Problem solved!!!

There was an invisible space character at the end of sms center number: it was "+3097100000_" instead of the correct "+3097100000". After removing the space character at the end, everything is ok!

 It seems that "HTC connect" application, which is used to setup automatically the device, has the wrong sms center number format for the Greek provider Cosmote. I have informed them.
