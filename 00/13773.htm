gary's student say: First enter the following User Defined Function:

Public Function Pressure(temp As Double) As Double
Dim I As Long, Y0 As Double, Y1 As Double
Dim X1 As Double, X2 As Double
Pressure = 0
X = Array(98.5, 131.8, 148.2, 166.2, 199.8, 215.5)
Y = Array(1, 5, 10, 20, 60, 100)
For I = 0 To 4
    If temp >= X(I) And temp < X(I + 1) Then
        X0 = X(I)
        X1 = X(I + 1)
        Y0 = Y(I)
        Y1 = Y(I + 1)
        GoTo CALC
    End If
Next
Exit Function
CALC:
Pressure = Y0 + (Y1 - Y0) * (temp - X0) / (X1 - X0)
End Function

I would expect =lookup(J2, E:E, G:G) to work in the normal way and find the value of J2 in EE and return the corresponding value in G:G but there are circumstances where it might not.

It does not seem like you are using macros. But it may be necessary to do what you want to do. Formulas can only affect the value of the cell they are in, and con not change the value of another cell.

P.S. this is the way Excel works, has always worked, and will continue to work. I think you are getting confused about how COPY and PASTE are supposed to work. ENTER simply terminates entry into a cell and "fixes" that value or string in the cell. If you COPY a cell and then press enter in the same cell it will cancel copy mode. You are expecting Enter to behave like PASTE when this was never its function.
