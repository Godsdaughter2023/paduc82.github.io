    Windows 10 Insider Preview is a preview into what should have already - Microsoft Community

 jacob t hamblett  say:


            Windows 10 Insider Preview is a preview into what should have already been...

                    
I personally regard Windows and other Microsoft products very highly. Windows 8 / 8.1 for me was a superb transformation of the GUI for computing - that was well over-due a transformation / modernisation. Windows 8 had offered a refined, more mobile and
 interactive userface; yet retained all original functionality of previous Windows operating systems (e.g PC settings, and full Control Panel). The main advancement of Windows 8 / 8.1 is the synchronisation of account settings, apps, credentials and files via
 OneDrive - as associated to the users Microsoft Account. 




This is where my first issue lies with the Windows 10 o.s. that I have previewed. OneDrive was a main feature of Windows 8 /8.1, and was fully integrated with the PC and User Interface. It does feel that this level of integration is not maintained in Windows
 10, as there are no options for OneDrive in PC Settings, and OneDrive has to install the original application-software-package that was used on pre-Windows 8 operating systems.





Another issue I have is the cluttered Start Menu layout in the Windows 10 insider preview. I understand that there was previous scrutiny over the lack of traditional start menu in Windows 8 /8.1. It seems that Microsoft have decided to act on this by taking
 a step back and overcomplicating the start menu feature with:


    

  • many configuration options for a full screen menu, or a menu that just fills 70% + of the screen
  • different lists of applications down the side of the screen
  • no full screen list of apps and software installed on the computer, one has to scroll through what can be an extensive list on a tiny scrolling menu on the left of the start menu
  • buttons and user options are scattered, cluttering the page and over-complicating the start menu. For instance, the used to just be a Spyglass, User Name with their account picture, and a power button - all taking up a small part of the start menu in the
 top right. Now the power button is under the messy list of recent apps, new apps and all other apps - along with an All Apps button that fills these lists even further. There are buttons for settings, Documents and File Explorer listed above the list
 of apps, under the User Name and picture. Other than the clutter, why isn't the power button with the
sign out options on the Username Button?


Lack of consistency is something else I have noticed between different areas of the interface. This is admittedly something that became present when the
PC Settings menu was developed for Windows 8, to refine the basic setting of the computer - in addition to the extensive
Control Panel, which retains the original windows theme. This is something that became familiar; however the insider preview of Windows 10 has provided indication into further lack of consistency. This is lack of constancy between Windows 8 / 8.1
PC Settings theme and menu's, the new Windows 10 PC Settings theme and layout, the new interface (start menu, icons and side-menus) of Windows 10, and the more traditional Windows interface/menu's like the Control Panel and its various tools.



An example of this is the WiFi menu. This was updated on the taskbar in Windows 8, and retained this Windows 9 theme on the side-menu access, and the
PC Settings menu. There was still the more traditional theme of this option in
Network and Sharing centre, and control panel for Windows 8. In Windows 10 insider preview, the taskbar menu for WiFi (and network connectivity) has a new windows 10 theme. Yet, when accessing this setting from the updated
PC Settings interface, the WiFi menu appears using the Windows 8 theme. Then when accessing this setting in the traditional
Network and Sharing centre, and control panel interfaces, the WiFi menu is in the more traditional windows theme. That's three themes for 1 option alone, something that could continue to be present throughout the rest of Windows 10 Operating
 System. 




These are just some of the issues I have with the Preview of Windows 10, other issues with regard to the updating of apps such as
Mail (, calendar and people) I have raised in the form of questions (due to on-going issues with those applications). But all I all, I would say that Windows 10 is looking to be a bit of a step backwards for Microsoft. It appears
 to be bridging the gap between Windows 7 and Windows 8 / 8.1, in what initially looks like a modern theme. Yet, I feel that Windows 8 was a step in the right direction, modernising the traditional, dated operating systems (pre Windows 8). Microsoft probably
 should have maintained the advancements made I Windows 8, with its simplicity, clean layout, interactive Start Menu & applications, Cloud synchronisation of user settings & files, etc. These are advancements that I feel could be lost in the launch of Windows
 10. The integration of these features are not tight enough, the preview of Windows 10 operating system is not clear, is not refined - it is like a mix between Windows 7, 8 / 8.1 and some elements of a new system (Windows 10). I don't think this is going to
 make windows, not for the younger generations of Windows users that were so appreciative of Windows 8.
