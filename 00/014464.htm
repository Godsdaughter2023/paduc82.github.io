Trying to hide columns based on a Form Control Combo Box selection - Microsoft Community

mags0930 say:

Trying to hide columns based on a Form Control Combo Box selection

I have to use Form Control. I'm having trouble with the VBA code for hiding columns based on a Form Control Combo Box selection. I have logic built into my tab that returns a "Yes" or "No" in cell D7 based on the selection of the drop down. If the answer is "Yes" I do not need to hide columns D:M and P:Q; however, if the answer is "No" I need to hide columns D:M and P:Q.

I have figured out the VBA code using Data Validation and ActiveX, but neither of these functions are an option for my model.

Thanks in advance!

Moved from  Office / Excel / Windows 10 / Office 2016

ossiemac say:

I am assuming from your comment "I have logic built into my tab that returns a "Yes" or "No" in cell D7 based on the selection of the drop down" that you have that working.

Use a macro code assigned to the Forms Control ComboBox.

If you have not already got a standard module in your VBA editor then add one as follows.

Alt and F11 to open the VBA Editor.

  • Select Menu Item Insert -> Module to insert Module1 (or if Module1 already exists then last numeric increments)
  • Copy the code below into the VBA Standard Module.

Assign the code to the ComboBox. as follows

  • Right click the Combobox and select Assign Macro. (If when you right click the combobox you get options "Move here, Copy here, Cancel" then right click the combobox again until you get more options that includes "Assign macro")
  • Select the macro name from the list of macros and click OK.

Copy all of the following code into the VBA editor.

Sub DropDown1_Change()
    If Range("$D$7").Value = "Yes" Then
        Range("D:M,P:Q").EntireColumn.Hidden = False
    Else
        If Range("$D$7").Value = "No" Then
            Range("D:M,P:Q").EntireColumn.Hidden = True
        End If
    End If
End Sub
