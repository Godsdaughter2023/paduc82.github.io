rudidudi say:

See the answer by Lee_G3 on how to change the Run Tile settings, here.

I just discovered that, in these settings, you can choose to display "Instantaneous Pace" instead of "Average Pace"! This sounds like exactly what you (and I) want!
This is a critical feature for any serious runner in training, so I'm surprised it isn't mentioned anywhere in the help pages!
I have not yet tried it on an actual run though.

Update, July 7:

I tried using "Instantaneous Pace" on an actual run, and found that it was not very useful because the pace displayed fluctuates wildly. While running at a steady pace of around 8:30 min/mile, I glanced at my Band 2 every 15 seconds or so and saw bogus paces like 13:32, 10:50, 7:23, and in between.

I suspect this is because the band is using too short a time interval to measure the instantaneous pace. Thus it is overly impacted by (1) GPS inaccuracies and (2) changes in the instantaneous speed of the wrist as the arm swings back and forth. (When the arm swings forward, the wrist is moving forward much faster than the torso, and when swinging backward, its speed relative to the ground is maybe half that of the torso.) To iron out these fluctuations, even an "instantaneous" pace needs to be averaged over at least the last 10-15 seconds or so in order to be useful.

It looks like Microsoft needs to do some more tuning of the "Instantaneous Pace" feature by field testing it with actual runners.

Update, July 11:

Curiously, the Instantaneous Pace while running stays steadier if I hold my wrist in front of me, with my elbow steady at my side. This suggests that factor (2) above indeed introduces a significant error.

On the other hand, when I switch from running to walking, the Instantaneous Pace drops gradually. It takes about 20-30 sec to drop to my walking pace. So it appears that the Instantaneous Pace is in fact an average over the last several seconds.

It would be great if the Band development team shared more details about the true meaning of Instantaneous Pace. This is important for runners in training. And I would love to hear other runners' experiences with the Instantaneous Pace feature.

I had a similar issue earlier today. When I turned on my Samsung laptop, all my Word document icons and Excel icons had changed to orange icons. My solution was to right click on a Word document. From the list of options that appeared, I hovered the cursor on the 'Open with' option and then I clicked on 'Choose default program'. I was presented with a couple of choices, one of which was Microsoft Office and the other was 'Microsoft Office Client Virtualization Handler'. I left clicked on the latter option and made sure the box below which read 'Always use the selected program to open this kind of file' was ticked and then clicked on OK. Hey presto, my Word icons were reinstated and opened without any further issue.

 

I had to do a similar process for Excel spreadsheets. Instead of the option to open with 'Microsoft Office Client Virtualization Handler' in the example above, I had to select 'Microsoft Virtualization Application DDE Launcher'.

 

If you have old and new Word documents (.doc and .docx) then you will need to perform the process twice (once for each type of document) and also once for each type of Excel file type (.xls and .xlsx) to fix the problem.

 

Hope that makes sense and hope it helps someone, somewhere...
