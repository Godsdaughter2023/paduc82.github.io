When Excel is started, the default delimiter in the Text to Columns dialog is the tab . You can change the delimiter to another setting. Excel will remember that setting while it remains open. mike h.. say:

Hi,

As you've already established the formula is fine so it must be the real data that's the problem. Very often when pasting data from outside sources it's dirty with odd and unwanted and more importantly invisible characters. Chief among these are spaces and CHAR(160).

Try this. For a pair of cells you think should be getting counted enter these formula

=LEN(A1)

=LEN(B1)

Do they return a number that's the same as the characters you can see in the cell? If they return more than you think that are there then we need to clean the data and the way I use is VB code.

ALT+F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in on the right. Back on the worksheet select all the cells in your 2 columns of data. Don't select full columns, it will take to long and run the code and then try your COUNTIFS.

Sub CleanMe()
Dim c As Range
    For Each c In Selection
    If Not c.HasFormula Then
    With c
        .Value = WorksheetFunction.Trim(.Value)
        .Value = Replace(.Value, Chr(160), "")
    End With
    End If
    Next
End Sub

You can access the apps in one of two ways:

WP7 "usb device not recognized"

Trying to connect my HTC 8x WP7.8 to my Windows 7 PC via USB but all I get is repeating "USB Device Not Recognized" error.

a. user say:

Hello Microsaur:

Thank you for posting to the forum regarding the issues you are having between your computer and your Windows Phone.  The error you are getting tends to indicate an issue with the cable you are using, the computer you are using or the phone you are using.  I would like to start by trying to narrow down where your issue lies.

Please start by connecting your phone to your computer via all available USB ports.  I have seen similar issues when a USB port on a computer is bad.  Next, I would try using another sync cable.  If you try another cable in all ports and it still does not work it is safe to say this issue is related to the computer or the phone.

The next thing I would suggest you do is take this phone and connect it via another computer.  I would suggest you try more then one other computer and try one with Windows 7 and with another operating system.  When you connect, look for the same error as well as go to device manager to see if the phone is listed here without errors.  You can get to device manager by pressing and holding the Windows key + R and typing control panel in the search box.  From here select hardware and sound and click on "device manager".  Here you should see your phone listed.  Double click this phone and review properties for any error messages or codes.

One thing you can do on your computer is go to device manager as above.  Right click the listed phone within the portable device/unknown device section and select properties.  Select the drivers tab and then try to first update the driver.  If you continue to have issues select the uninstall option > disconnect the phone for about 1 minute > reconnect the phone.  If you find any errors in device manage please detail for us what you found.

I would like to stop the initial troubleshooting here.  If you continue to have issues provide the following details and we will move on with our troubleshooting.

  • Make and model of your phone.  You can find this by going to settings > about > more info.
  • Current OS version.  See Above.
  • Current firmware revision.  See Above.
  • Country you are located in.
  • Regional setting of your phone.
  • Language setting of your phone.
  • Speech setting of your phone.
  • Last time phone was properly synced with your computer.

Please let us know if you have any other questions or concerns.

Mark W.
