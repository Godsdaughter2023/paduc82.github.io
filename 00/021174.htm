Removing partial data from Excel Cells - Microsoft Community

pgawannabe75 say:

Removing partial data from Excel Cells

I am trying to remove part of the contents of a cell.  I could have upwards of 20000 rows and I need to remove the text below.

for example

Row 1     ABCDEFGHIJK Q/A

Row 2     ABCDEFG  Q/T

Row 3     ABCDFT Q/B/X

 What I want to do is REMOVE anything to the right of the Q/

Is there any way to do this with a Macro?

 

jllatham say:

Sure, the code below should do it for you, just change the "Const" values to match up with your worksheet and its layout.

NOTE: this code is going to leave that space just before the Q as part of what's left behind.  If you want to get rid of that then change this line below

  Const findInData = "Q/"

to become

  Const findInData = " Q/"

Sub RemoveQSlash()
  'change these Const values as needed
  'name of the sheet with the data on it
  Const dataWSName = "Sheet1"
  'column with the data in it
  Const dataColID = "A"
  'first row with an entry to examine
  Const dataFirstRow = 1 ' might be 2 if 1 has a label
  'what we are looking for
  Const findInData = "Q/"
  Dim dataWS As Worksheet
  Dim dataRange As Range
  Dim anyDataCell As Range
  Dim foundPos As Integer
 
  Set dataWS = ThisWorkbook.Worksheets(dataWSName)
  Set dataRange = dataWS.Range(dataColID & dataFirstRow & ":" _
   & dataWS.Range(dataColID & Rows.Count).End(xlUp).Address)
  Application.ScreenUpdating = False
  For Each anyDataCell In dataRange
    foundPos = InStr(UCase(anyDataCell), findInData)
    If foundPos > 0 Then
      anyDataCell = Left(anyDataCell, foundPos - 1)
    End If
  Next
  Set dataRange = Nothing
  Set dataWS = Nothing
  MsgBox "Task Finished", vbOKOnly + vbInformation, "Job Done"
End Sub

    cataholic say:

    Yes, the battery will not charge if the charger cannot supply a minimum charge current - apart from the mA rating of the charger this may be driven by the low voltage of the charger as well, many older chargers are 3.7v or similar, not 5v.

     

    The E52, like most other modern phones are moving to using the USB standard for charging (they are being driven to do so by regulation if they wish to sell phones in some countries), so uses the USB bus 5v line voltage for charge, but many USB bus's on computers cannot supply the required charge current when the battery is low on charge. The USB 2.0 specification is that a unit load is 100mA with a total load of 500mA but as more devices are now using the USB bus for charging or for higher powered applications this has been raised in USB 3.0.

     

    There is also provision in the USB standard for current to be limited to 100mA from the USB bus until the device has notified its actual current requirement in order to protect the computer USB board from overload but I suspect this is not used for lower load devices such as phones.

     

    So none of this is the fault of the phone, if one has a 5v charger that can supply the same current as the phone's supplied Nokia charger (1,200mA) then one should be fine. Also, the fact that older USB buses may not be able to supply sufficient current is also not the phone manufacturer's fault.

     

    I had previously done a few checks on this and found:

     

    - a 350mA 3.7v Nokia charger would not reliably charge the phone

    conversion from picture to editable word text - Microsoft Community

    shack1430 say:

    conversion from picture to editable word text

    I need to change a document that when converted from PDF to word is a picture, to an editable text document

    jay freedman say:

    If you use Word 2016 to open the PDF, it will try to convert it to an editable document. Whether that succeeds depends on how the PDF was created.

    If it was saved as PDF from Word or another source that embedded editable text in the file, then you'll get an editable document (although its formatting may not match what you see in a PDF reader).

    If it was created by scanning a document directly into PDF, the file contains only pictures of the pages, and that's what will appear in Word. In that case, the PDF must be submitted to an Optical Character Recognition (OCR) program, which examines the outline of each character and outputs the text equivalent.

    You can find online OCR services where you upload the PDF and receive an editable document in return. You can get standalone OCR that you install on your PC, although that tends to be expensive and worthwhile only if you have a continuing need for it. If you have OneNote as part of Office, it has a basic OCR capability -- you paste the OCR pictures into a OneNote tab, right-click and choose Copy Text from Picture (which puts the text on the clipboard), and then paste the text wherever you need it such as Word.
