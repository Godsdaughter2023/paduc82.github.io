john w. vinson say:

Unfortunately, most Totals queries cannot be updated. But... two ideas.

One: Do you need to store this data redundantly in a table? If you need the total, the count and so on, why not just use the SELECT query directly? Sure, you can't update it, but you don'twant to be able to update the count or the sum! In addition, if you do store the data in a table, as soon as you edit the tbl_temp_import_data, or add a record, or delete a record, the data in your new table will bewrong, with no easy way to detect that it's wrong. Note that you can base a Form, or a Report, or an Export directly on a Query; it is certainly not essential to store the information in a table.

Two: You cannot run an Update query (directly or simply), but if this is new data, you can run an Append query. If you create (or have already created) NEW_FLIGHTMAWB with appropriate fields, you can change your SELECT query to an APPEND query:

INSERT INTO NEW_FLIGHTMAWB ([MAWB No], [CountOfFlightMAWB ID1], [MaxOfFlightMAWB ID]  (SELECT tbl_temp_import_data.[MAWB No], Count(tbl_temp_import_data.[FlightMAWB ID]) AS [CountOfFlightMAWB ID1], Max(tbl_temp_import_data.[FlightMAWB ID]) AS [MaxOfFlightMAWB ID]
FROM tbl_temp_import_data
GROUP BY tbl_temp_import_data.[MAWB No]
HAVING (((Count(tbl_temp_import_data.[FlightMAWB ID]))>1)));

This data will still be redundant, unnecessary, and a Bad Idea, but you can do it.

Run Automatic Repair using a Windows 10 Technical Preview installation disc. Automatic Repair is a Windows recovery tool that can fix certain system problems that might prevent Windows from starting. Automatic Repair scans your computer for the problem and then tries to fix it so your computer can start correctly.

a) After you boot your computer using Windows DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.

b) Select the correct time and Keyboard type.

c) Click Repair your computer in the lower left corner.

d) Select Troubleshoot from Choose an option screen.

e) Click advanced options in Troubleshoot screen.

f) Select Automatic repair.

a) Refer to the Steps a to e
