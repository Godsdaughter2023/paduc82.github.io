    2)       Is it a 32 bit or a 64 bit Operating System that you are using on the computer?

    Try these steps and check if that helps:

    Method 1: Hardware and Devices Troubleshooter

    A.       Type troubleshooter in the search bar

    B.       Click on view all.

    C.     From the list, select hardware and devices troubleshooter and follow the on-screen instructions.

    Method 2: Uninstall the damaged driver and then reinstall it. These are few changes to make and try : 

    A.       Press “Windows key + X”.

    B.       Click on “Device Manager” and then right click on the speaker device listed in the Device Manager window and then select “Uninstall”.

    C.       Restart the computer and let the Windows to detect the device and install the drivers suited to it automatically.

    D.       If it does not detect automatically, then download the latest driver from the manufacturer’s website and install it manually.

    Check if the speaker is working. If not, then try the next step.

    Method 3: Install the driver in compatibility mode

    A.       Download the latest drivers for the speaker that you are using from the manufacturer’s website and save it in your computer.

    B.       Right click on the setup file, scroll down to Properties and click Properties.

    C.       On the window that pops up, click Compatibility tab.

    D.       Under Compatibility mode, check the box “Run this program in compatibility mode for”.

    E.        In the scroll down select “Windows 7/8”

    F.        Under Privilege Level, check the box “Run this program as an administrator”

    G.       Click Apply and Ok.
