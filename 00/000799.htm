Unique number creation - Microsoft Community

jennifertozer say:

Unique number creation

I am new to Access and I am trying to create an app that will track work orders for my organization.  The work order number needs to be automatically generated using the organization's model of yyjjjhhmm (y=year, j=julian date, hhmm=hour/minute).  Does anyone know if this is possible to do & if so, how?  Thanks.

john w. vinson say: This will of course not be unique if two users (at different computers naturally) happen to enter a workorder in the same minute.

That said, you can use an expression

Format(Now, "yy") & Format(Now, "yhhnn")

to generate this string. You could do this in some VBA code in the work order form's BeforeInsert event:

Private Sub Form_BeforeInsert(Cancel as Integer)
Me!txtWONumber = Format(Now, "yy") & Format(Now, "yhhnn")
End Sub

scottgem say:
John gave you the way to do it, but I would offer two additional suggestions. First, I would still use an Autonumber as the tables Primary Key. Second I would convert the number back to a numeric value before storing. The code John gave you will store i as a text value. Change the code to:

Me!txtWONumber = Val(Format(Now, "yy") & Format(Now, "yhhnn"))

And it will convert back to a number.
