andreas killer say:
The array separators can be different, it depends on the local settings which on are used. If the current array separator is the same as the decimal separator, the alternate array item separator has to be used instead.

See code and link below.

Andreas. 

Sub DetermineArraySeperators()

  'https://msdn.microsoft.com/en-us/library/micros...
  Dim ColSep As String, RowSep As String, AltSep As String, DecSep As String
  With Application
    ColSep = .International(xlColumnSeparator)
    RowSep = .International(xlRowSeparator)
    AltSep = .International(xlAlternateArraySeparator)
    DecSep = .International(xlDecimalSeparator)
  End With
  If ColSep = DecSep Then
    ColSep = AltSep
  ElseIf RowSep = DecSep Then
    RowSep = AltSep
  End If
  MsgBox "Column " & ColSep & "  Row " & RowSep, , "Seperators"
End Sub
