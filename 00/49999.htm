jacklebek say:

I just talked to Windows support. The issue is if your PC wasn't made with Windows 10. If you upgraded it from Windows 7 or 8, it's using another harddrive setting - GPT. 

Win 7 and 8 used GPT disk settings, and Windows 10 use UEFI. This isn't a problem because windows made a "bridge" between these to settings, but if something happens during the update - well. There goes trouble.

I had to make a recovery USB disk to go to the windows installation, and had to go to BIOS to change the harddisk settings. Then I could create a new clean windows install.

I don't know if it is possible to do this without making a clean windows install - but at least this way you can get your system up and running again.
