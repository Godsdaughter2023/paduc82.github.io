julie sheets say:

When you grab the split task from Wednesday and move it back to Tuesday, you are shifting the work back to Tuesday -- it does not stay on Wednesday.

If Tuesday is a non-working day, you must modify the calendar to make Tuesday a non-working day -- not manually trying to split tasks.  If it is a non-working day only for the resource, change the resource's calendar. Go to Tools > Change Working time and select either the Project calendar or the resource calendar.

Once you have made Tuesday a non-working day, the task bar will span from Monday through Wednesday but no work will be scheduled for Tuesday.  If you want the appearance of the task bar not spanning the non-working time (Tuesday, Saturday, Sunday) draw the non-working time in front of the bars.  Right click in the Gantt chart background and select "Non-working time".  On the Non-working time tab in the format section, select "In front of task bars".
