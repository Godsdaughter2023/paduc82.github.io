Bing Ads Conversion Attribution Date

Hello All,

I need to clarify on the Bing Ads Conversion attribution date. If the click occur today and user converted after 2 to 3 days, for which date this conversion will attribute? Google will attribute the conversion for the Click date. How about Bing?

For more clarification, If the first click occurred on day 1 and the conversion occurred on day 10, Google AdWords conversion date is day 1. How about Bing? Its Day 1 or Day 10?

microsoft advertising - shelly say:

Hi Karimulla Shaik,

That is a great question! Bing Ads will not count any user activity as a conversion, even if it matches a conversion goal, unless it is able to find that the user clicked on an ad before finalizing the action. The process is only counted as a conversion once the conversion process has been completed.

      Ex. If a customer clicks on the ad on Wednesday and put the item in the shopping cart; however, the customer does not purchase the item till Friday the conversion will be counted on Friday. Even if the user pauses the campaign on Thursday.

The period of time after the user completed the action that Bing Ads uses to record conversion data is known as the conversion window. The default is 30 days; however, you can set the window from 1 minute to 90 days. You also have the flexibility to tally conversion according to your business needs:

doug robbins - mvp office apps & services (word) say:

Assuming that the text in the footnotes for which you want to retain the cross reference are ALL of the form 

See p #

where # is the number of the page, and that book 1 will contain pages 1 to 250, book 2 will contain the pages that were 251 to 500, book 3 will contain the pages that were 501 to 750 and that book 4 will contain the pages that were >750, running the following macro

Dim numrange As Range
With ActiveDocument
    For i = 1 To .Footnotes.Count
        With .Footnotes(i).Range
            If InStr(.Text, " p ") > 0 Then
                Set numrange = .Duplicate
                numrange.Start = numrange.Start + InStr(.Text, " p ") + 2
                numrange.End = numrange.End - 1
                Select Case Val(numrange.Text)
                    Case Is > 750
                        .Text = "See Book 4, Page " & Val(numrange.Text) - 750
                    Case 501 To 750
                        .Text = "See Book 3, Page " & Val(numrange.Text) - 500
                    Case 251 To 500
                        .Text = "See Book 2, Page " & Val(numrange.Text) - 250
                    Case Else
                        .Text = "See Page " & numrange.Text & " of this book"
                End Select
            End If
        End With
    Next i
End With

will modify footnotes such as:

to:

If the separation of the book will take place at different page numbers, by making appropriate changes to the numbers used in the Case statements, such differences can be accommodated.

Using this macro, there is no need to insert Section Breaks in the document and modify the formatting of the page numbers so that the numbering in each Section starts at 1 as after the book is separated into the four parts, the numbering in each part will start at 1 when the page numbers are updated.

Of course, the page number in the modified footers will no longer act as a hyperlink, but as the books are going to be printed, that should not be an issue
