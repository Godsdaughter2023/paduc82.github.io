bob jones aka: cybertaz say:
Object Linking & Embedding [OLE] is not as fully implemented in OS X as it is in Windows. The only embedded object file types supported in Office documents are Word documents, Microsoft Equations, Excel spreadsheets & Charts.

Additionally, I believe that specific type of sound file is Windows-specific, so even if they can be extracted from a .docx Word document there are no Mac apps capable of playing them.

Your best bet is to have the sender resave the sound files in a more universal format [.wav, .aiff, .mp3, etc.] & forward them to you separately. You can then play them using a Mac-compatible media player.

Sub HideColumns()

    Dim ws As Worksheet
    Dim rngCol As Range
   
    Set ws = Worksheets("Sheet1")   'Replace "Sheet1" with your sheet name

    For Each rngCol In ws.UsedRange.Columns
        If WorksheetFunction.CountIf(rngCol, "=*Dogs*") > 1 Then
            rngCol.Hidden = True
        End If
    Next rngCol

End Sub


Sub UnHideAllCols()
    Dim ws As Worksheet
    Dim rngCol As Range
   
    Set ws = Worksheets("Sheet1")   'Replace "Sheet1" with your sheet name
    ws.Columns.Hidden = False
