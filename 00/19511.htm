OK, so these are just my thoughts and are what I have done just based on my personal experience. If I have a memory issue I do not try to move apps from my device to the SD card. So at first glance Paul, it seems to me that probably the best and quickest way to get your space back (and more importantly to get your upgrades) is to strongly consider performing a hard reset. It will not take that long and the advantage is that you get a fresh start. If it were me, I would NOT restore from a backup but rather start fresh.

There are a couple of reasons for doing that. All too often we have (overtime) installed a lot of programs that we never use or rarely use so they just take up space. Most likely, if you did not reinstall them, you would not even miss them. If you restore from a backup, it will set all those programs as pending and it always seems to take forever to download.

I find that not restoring from a backup, my device gets setup very quickly and I am up and running in a short period of time.

Use your same Live ID so all your contacts and calendar return and as long as you have enabled auto backup to OneDrive when near a Wi-Fi, all your photos have been stored in OneDrive. You might want to check that your photos are there before you hard reset if you choose to go this route. 

Once it is setup and BEFORE you add back any must-have apps, check for updates. It is a perfect time to install them since there is all sorts of room on a freshly setup device. Once completed you can then add back the programs you need and slowly add back others that are not as important or rarely used.

That is my advice, I have been doing it this way for a long time and it has always worked well for me. In fact, I just hard reset my 950XL two weeks ago. Over time I had accumulated too much and it is nice to start fresh ... it works well!

Keep in mind that if you are running out of space, you probably have too much installed and if you look at it carefully, most likely there is a lot of stuff you never or rarely use. This is not a PC but a device to help you perform those important tasks you need to do. Just my thoughts....

Good luck with your decision!!

$ - what does this mean in a formula and when and why would I use - Microsoft Community

helpmequick say:

$ - what does this mean in a formula and when and why would I use it? Thanks.

$ - what does this mean in a formula and when and why would I use it? Thanks.
rick rothstein, mvp - excel say:

See if this link helps clarify things for you...

http://www.cpearson.com/excel/relative.aspx

NOTE: Please mark the message or messages (yes, you can mark more than one) that answer your question as the "Answer" so that others will know your question has been resolved.

hansv mvp say:

It means that the cell reference is absolute, i.e. it will not change if you copy or move the cell with the formula.

Example 1: in cell B1, enter the formula A1.

Now copy cell B1 to B4, The formula in the copied cell is =A4: Excel has automatically adjusted the cell reference. Such a reference is called relative.

Example 2: change the formula in cell B1 to =$A$1

Now copy cell B1 again to B4. The formula in the copied cell is still =$A$1: Excel has *not* adjusted the cell reference. This reference is called absolute.

Example 3: in cell D1, enter a tax rate of 6%.

In E1, E2, etc., enter some amounts, e.g. 10, 20, ...

In column F, we are going to compute the tax amounts.

In F1, enter the formula =E1*$D$1

Copy cell F1 to F2, F3 etc.

In cell F3, you'll see =E3*$D$1, i.e. the reference to the amount has been adjusted, but that to the tax rate is fixed.

Hello,

yes, it is possible to install Office 2013 on Windows 7. The computer I am using to write this runs Windows 7 Pro 64 bit. I have Office 2010 and 2013 installed and working.

When you install Office 2013, there will be a dialog early in the process that asks if you want to upgrade the existing Office version. Make sure that you do NOT upgrade the existing Office version but select the Custom installation. That way you can run both versions on one machine.

Hello,

Excel 2013 has changed the user interface to SDI (just like Word has been doing for a long time) and each Excel file will open in its own window.
