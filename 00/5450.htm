If you know which column will always have the last piece of data in it, you can use this (where I assume Column A is that column)...

LastRow = Cells(Rows.Count, "A").End(xlUp).Row

If you don't know which row in your range will always have the last piece of data, then you can use this for the range you showed us...

LastRow = Range("A:AC").Find(What:="*", SearchOrder:=xlColumns, SearchDirection:=xlPrevious, LookIn:=xlValues).Row

Whichever one you end up using, you would incorporate it into your code line like this...

ActiveSheet.ListObjects.Add(xlSrcRange, Range("$A$2:$AC$" & LastRow), , xlYes).Name = "Table2"

susanpyi say:

Remove #DIV/0

If my formula in a cell is =(L4-K4)/ABS(K4) then what will be the revised formula to remove the #DIV/0 errors from appearing in the cells where there is no data to calculate.  

jeeped say:
Did you want to display nothing on a #DIV/0 or substitute a value? This will leave the cell blank,

=IFERROR((L4-K4)/ABS(K4), "")

Substitute a number for the empty quotes or add some text if you want something other than a blank cell.

IFERROR function
