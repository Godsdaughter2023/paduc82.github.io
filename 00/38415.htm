Normally you receive the Error Code 10: The device cannot start, if the Webcam is not found on the computer and if the drivers are not installed properly.

 

Firstly unplug the webcam from the computer. Later uninstall all the webcam drivers from programs and features and device manager and restart the computer.

Also you can try this solution to resolve Error Code 10 issues by following the steps below:

 

1.     Click on Start and right click on My Computer.

2.     Click on Properties

3.     Choose the Hardware tab and click on it.

4.     Click on Device Manager.

5.     Navigate to expand USB – Universal Serial Bus controllers section.

6.     Right-click every device under the Universal Serial Bus controllers node

7.     Click Uninstall to remove them one at a time.

8.     Restart the computer.

 

After restart the computer, allow the computer to reinstall the USB controllers. Finally plug the removable USB Webcam, and then test it to make sure that the issue is resolved.

For the Windows Update issue, follow the steps mentioned below: 


I) Rename the Catroot2 folder

To rename the Catroot2 folder, follow these steps:

1. Click Start, type cmd in the Start Search box, right-click cmd.exe, and then click Run as administrator.
2. At the command prompt, type the following commands, and then press ENTER after each line: 

net stop cryptsvc 
ren %systemroot%\System32\Catroot2 oldcatroot2 
net start cryptsvc 
exit

3. Delete all tmp*.cat files from the following folder: 

%systemroot%\system32\CatRoot\{F750E6C3-38EE-11D1-85E5-00C04FC295EE} 


If no files that start with tmp exist in this folder, do not remove any other files. The .cat files in this folder are necessary for installing hotfixes and service packs.


Important: Do not rename the Catroot folder. The Catroot2 folder is automatically recreated by Windows, but the Catroot folder is not recreated if the Catroot folder is renamed.



II) Reregister DLL files that are associated with Cryptographic Services


To register .dll files that are associated with Cryptographic Services, follow these steps:

  1. Click Start, type cmd in the Start Search box, right-click cmd.exe, and then click Run as administrator.


  2. At the command prompt, type the following commands, and press ENTER after each command:

regsvr32 /u softpub.dll
regsvr32 /u wintrust.dll
regsvr32 /u initpki.dll
regsvr32 /u dssenh.dll
regsvr32 /u rsaenh.dll
regsvr32 /u gpkcsp.dll
regsvr32 /u sccbase.dll
regsvr32 /u slbcsp.dll
regsvr32 /u mssip32.dll

regsvr32 sccbase.dll
regsvr32 slbcsp.dll
regsvr32 mssip32.dll
regsvr32 cryptdlg.dll
exit

Note: Click OK if you are prompted.

Restart the computer.


III) Empty the software distribution folder

  1. Click Start, type services.msc in the Start Search box, right-click services.msc, and then click Run as administrator.
  2. In the Services (Local) pane, right-click Automatic Updates, and then click Stop.
  3. Minimize the Services (local) window.
  4. Select all the contents of the Windows distribution folder, and then delete them.

    Note By default, the Windows distribution folder is located in the drive:\Windows\SoftwareDistribution folder. In this location, drive is a placeholder for the drive where Windows is installed.
  5. Make sure that the Windows distribution folder is empty, and then maximize the Services (local) window.
  6. In the Services (Local) pane, right-click Automatic Updates, and then click Start.
  7. Restart the computer, and then run Windows Update again.

IV) Disable the security software

For the “low disk space” popup, you can try following steps:

I) Run Disk Cleanup. This tool removes temporary files, empties the Recycle Bin, and removes a variety of system files and other items that you no longer need. For more information, see Delete files using Disk Cleanup.

II) Uninstall old programs. In addition to deleting old files you no longer want, you can do the same for programs that you no longer use. For more information, see Uninstall or change a program.


III) Archive old files to CD or DVD. If you have old photos that you want to keep but don't necessarily want on your computer, consider archiving them to removable media, such as a DVD or CD. That way, you’ll be able to view them whenever you'd like, but they won't take up space on your computer. For more information, see Burn a CD or DVD.
