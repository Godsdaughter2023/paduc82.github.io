bernie deitrick say:

The linked cell property of a checkbox is not updated by a sort, as you have found. If you move the cells or insert/delete rows or columns above or to their left, that property is updated.

Since you want to sort those cells, it is better to not rely on the cell location for the link. Link the checkboxs to cells in a table (with row labels) that does not get sorted, then use formulas to extract the correct value for whatever it is that you are doing that requires sorting.
