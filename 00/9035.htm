jay freedman say:

To add the group headings to the "from template" index, after you've inserted the index, right-click anywhere inside the index and click Toggle Field Codes. In the code, after the keyword INDEX, add the switch

   \h "A"

to make it look like

   { INDEX \h "A" \c "2" \z "1033" }

(some of the other switches may be different). Then right-click again and click Update Field.

If you would rather start with the Classic style and change the formatting, here's how:

The only thing the Modify button in the Index dialog lets you do is to modify the nine paragraph styles named Index 1, Index 2, ... Index 9. Although that button is disabled when you choose any of the built-in indexes (and no, I don't understand why the program does that!), you can use the standard style-modification methods outside of that dialog.

Because the Index N styles aren't ordinarily included in the Quick Styles gallery on the Home ribbon, you'll need to use one of the other ways to get to the Modify Style dialog. The simplest is to press Ctrl+Shift+S to open the Apply Style box, type Index 1 in the Style Name box, and click the Modify button.

In the Modify Style dialog, click the "New documents based on this template" if you want to change the default for all future documents based on the current template. Then change the font name and size. Click the Format button at the bottom left, click Paragraph, and change the line spacing. Click OK.

Repeat for the Index 2 style that's applied to second-level subentries. Repeat for Index 3 through 9 if you use those levels. You may also want to modify the Index Heading style, which formats the letter at the start of each group.

mccbleue say:

The ringtone search should find all compatible file types from anywhere in the phone, so there should not be an issue.

 

However, it does depend on the format - the X3 can only use Midi, MP3, AMR, AAC, eAAC+ and WMA files as ringtones, whilst thre are other formats that the phone can play in media player (including M4A, WAV etc.) which cannot be used as tones, so if the music is in a non-compatible format for tones it won't be found.

 

Also note that some DRM-protected audio files are prevented by their DRM from being used as tones.

Private Sub cmdLaunchReport_Click()

Dim strCrit As String

Resource Monitor - CPU - Secure System process - Microsoft Community

a. user say:

Resource Monitor - CPU - Secure System process

I discovered through the Resource Monitor (CPU) that the "Secure System" process is suspended, and when i try to resume it, i get a response indicating that access is denied.  Is this suspension and the denial to access it normal, or are hackers disabling it and preventing me from resuming it?

rob koch say:

It appears from the following article on an external website titled Virtual Secure Mode (VSM) in Windows 10 Enterprise  Windows OS Hub, that this indicates the system is operating in Virtual Secure Mode with the Secure System process displayed in a Suspended status in the Task Manager screenshot found in the section titled "How to Make Sure That the VSM Is On".

Since my own much older Windows 10 system was upgraded from Win7 and so doesn't support Secure Boot, this process wouldn't be present, since only those with newer systems capable of this feature would find this process present to confirm its state.

The other thing I discovered while searching was that the Windows 10 operating system and the Universal Windows Platform (UWP) apps that it contains typically transition to suspended status upon exit, since as with most modern mobile app designs this uses less resources on often limited hardware.  Though, of course, this is less critical when using a traditional PC type device, it's still the way that these apps and operating system components are designed to operate.

Note that the following Microsoft Dev Center article titled Windows 10 UWP App lifecycle - UWP app developer  Microsoft Docs discusses this suspended state in detail as having been introduced in the Windows 8 new application model with UWP apps.
