a. user say:

An upgrade is supposed to erase existing restore points. If you restored an old restore point after an upgrade, you would corrupt your Windows installation.

For the same reason, restore points are automatically erased after a certain amount of time. In Windows 7 it was 90 days; I don't know the time limit for Windows 10. If you tried to restore an old restore point, you could corrupt your Windows installation.

A restore point is not a backup. These are two different things.

igor leyko say:
Hi papachra,
Laptops usually have preinstalled (OEM) Windows and such Windows cannot be transferred to another PC. So sorry but you'll need to buy another license.

davem121 say:
Hi papachra

There will be an activation problem if you restore form that image, even on the exact same make and model, however this is easily resolved . .

Before restoring form that image, make sure Windows is activated on the new laptop

Then restore form that image . . .

Then in command prompt as Admin, run this command to remove the product key from Windows:

slmgr.vbs -upk

When that completes, once that laptop is connected to the Internet the original activation on that new laptop will automatically re-activate

That's It!
