amanda zhang msft say:

Hi Chris,

 

Based on my test, to achieve your goal, I suggest you follow the following steps:

  1. To prepend the subject line with the name of the group the message was sent to, you can log your Office 365 admin account into Exchange Admin Center> mail flow>rules, then create a new rule> More options, choose the conditions that Apply this rule if Any recipient address includes ‘your Office 365 group name’, add action that Prepend the subject of the message with ‘ your customized subject line’ and click Save.
  2. To add a custom message header for e-mails sent out via an Office 365 group, you can log your Office 365 admin account into Exchange Admin Center> mail flow>rules, then create a new rule> More options, choose the conditions that Apply this rule if The sender address includes ‘your Office 365 group name’, add action that Set the message header to this value, and Set the message header 'name of message header' to the value 'customized value' and click Save.

Amanda

hansv mvp say:

You can merge cells:

  • Select two or more cells.
  • Select Format | Cells...
  • Activate the Alignment tab.
  • Tick the "Merge Cells" check box.
  • Select Center from the Vertical dropdown list to center the text between the top and bottom, or Justified if you want a line at the top and at the bottom, as in the cells with a diagonal line.
  • To enter multiple lines in a cell, tick the check box "Wrap Text". Later, when you enter text, press Alt+Enter to start a new line.
  • For a diagonal line, activate the Border tab and click the button with the appropriate diagonal.

pete_uk say:

It is mainly a matter of highlighting ranges of cells and using Format | Format Cells | Border and choosing the border width and Outline to set up large parts of your table. However, you will need to merge cells A1 with A2, B1 with B1 and C1 with C2 (from the Alignment tab of Format cells). In C1/C2 you will need to enter the following:

<sp><sp><sp><sp><sp>Size/<alt-Enter><sp><sp><sp><sp><sp><sp><sp><sp>Col<alt-Enter><alt-Enter>Spec

where <sp> means space and <alt-Enter> means that you hold the Alt key down and press the Enter key - this gives you a line feed within the cell. The merged C1/C2 cell can be formatted with a border around together with a diagonal top-left to bottom-right. In the merged cells A1/A2 and B1/B2 you can put an outline border and on the Alignment tab you can set the text to center vertically as well as horizontally. Adjust the columns widths to suit.

Hope this helps.

Pete
