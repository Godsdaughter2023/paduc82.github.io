Mail merge into multiple letter templates - Microsoft Community

dmquatt say:

Mail merge into multiple letter templates

We have a largish mailout program we undertake for our clients on a weekly basis. We use a data from excel 2007 datasheet which has client specific information which works fine when mail merged into a MS Word 2007 letter. The problem we run into is we can have up to 4 different letters which are used depending on a variable in the spreadsheet (a turnover category in actual fact). The content of the letter is very different between each so using if or include text fields doesn't seem to be the answer. Is anyone aware of either a relatively simple manner to acheive this using standard Word tools, alternatively if there is a third party addon which would allow us to do this?

a. user say:

 

It is worth noting that

 a. if you use { IF condition "text and fields" } then you had better know how to deal with double-quote characters in literal text (you can use { QUOTE 34 } fields, for example)

 b. either way, /if/ you are using different headers/footers for each letter, you have either to put header/footer variants in separate IF statements, or INCLUDETEXT different headers/footers separately from INCLUDETEXTing the body. I think.

Personally, these days I would be much more inclined to define completely separately letters on the grounds that they can be maintained independently more easily, particularly if they have different layouts/margins. There is no need to be concerned about clashing paragraph styles, and so on. But then you either have to do several separate merges as Suzanne originally suggested or use some other method, cf. the one suggested by Doug.

FWIW  if there are definitely only a fixed number of values of turnover_category and the INCLUDETEXT approach works OK for you, then you can also consider using a single nested INCLUDETEXT field as follows. If you have 4 letters called c:\\myletters\\myletter1.docx, c:\\myletters\\myletter2.docx, etc. then you can use

 

 

{ INCLUDETEXT "c:\\myletters\\myletter{ MERGEFIELD turnover_category }.docx" }

 

whereas if you have 4 letters bookmarked (say) L1, L2, L3, L4 you can try

 

{ INCLUDETEXT "c:\\myletters\\myletter.docx" "L{ MERGEFIELD turnover_category }" }
