Third party AV programs (McAfee, Norton, Avira, etc.) are notorious and often associated with update installation failures as they are protecting your system from changes.

I have NEVER seen a Windows Update installation fail when a system is using Microsoft Security Essentials (MSE), but there are beaucoup Windows Update failures reported when a system is running a third party AV program.  In other words - suspect McAfee but of course McAfee will tell you the problem is with Windows.

    Hi, Jody.
    You're running Windows 7, correct?  If that's the case, the MediaCenterDataStore.db won't be there - that was for Vista.  Media Center for Win7 stores most of its configuration data in mcepg2-0.db (which is located in the same folder that Vista held its database in.)
    If you open an Explorer window, and go to %SYSTEMDRIVE%\ProgramData\Microsoft\eHome (just copy/paste that into the address bar), you'll see that said file isn't there (this is normal.)

    What issue are you having with Media Center and/or Media Player?  We can help you fix it; we just need to know what needs fixing.  ;)

    If you truly do need to do a database reset of Media Center (which will reset all settings and customizations, delete your recording schedule and history, and require re-running TV Setup (actually, it'll require you to re-run Media Center Setup - it's literally, a clean setup), you can do this from the command line:
    Open an Elevated command prompt, and paste the following into the window:
    mcupdate -mediacenterrecoverytask

    Firmware Update - Microsoft Community

    xbr dionysus say:

    Firmware Update

    Login Information

    • Gateway – 192.168.1.1
    • Username – admin
    • Password – admin
    How to Find Firmware Version
    1. Open a web browser.
    2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
    3. If prompted to login enter “admin” as the username and “admin” for password.
    4. Select Management from the menu on the left.
    5. Select Status from the sub menu.
    6. The firmware is located on this page labeled Firmware Version.

     

    Firmware Update

    1. Plug a computer into the LAN port on the back of the device.
    2. Open a web browser.
    3. Go to the Engenius support page and find the newest version of firmware for your router.
    4. Download it and save it to the desktop.
    5. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
    6. If prompted to login enter “admin” as the username and “admin” for password.
    7. Select Management from the menu on the left.
    8. Select Upgrade Firmware from the sub menu.
    9. Select the Browse button.
    10. Find the file you just downloaded and click Open
    11. Select the Upload button.
    12. If asked to confirm click Yes.
    13. Wait for the router to finish the update and reboot.
