If they're all the same date, simply insert a DATE field somewhere and use Cut or Copy to put it in the Windows clipboard. Then, using Find/Replace, input the date as the Find expression and ^c as the Replace expression.

macropod say:

I have twenty-two 200+ page Word docs that are the forms for a database built on Access.  The docs are forms to use.  When the form is used (letter, memo, document, etc.), it needs to have the current date in it.


In that case, you probably shouldn't be using a DATE field, as the date on the completed form will continue to change. Other options that might be more suitable are the CREATEDATE and SAVEDATE fields.

If you create templates based on your forms and produce documents from those, a CREATEDATE field will show the date the document was created on. Alternatively, if you continue using documents, you can update the CREATEDATE field by saving the document via File|SaveAs.

The SAVEDATE field updates to show the date the document was last saved - which is useful if you want to know when the document was completed.

If the last word is of 2 characters only, there is no reason why this formula (already given in the forum) should not work - formula in cell B5:

=RIGHT(A5,2)

However, if the last word could be of varying number of characters (because you have mentioned "usually" two characters), below formula in cell B2 will extract the word after last space:

=TRIM(RIGHT(SUBSTITUTE(A2," ",REPT(" ",LEN(A2))),LEN(A2)))

tom_b# say:

I uninstalled and reinstalled Microsoft Office twice, without success. As I made clear in my original post, drive space is not the issue - there were hundreds of GB of free space on all drives.

I managed to fix the problem myself. The issue was an invalid Cache location, as a result of creating a new User.

The solution was to 

1. Run regedit.exe

2. Navigate to HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders

3. Set the Cache to a valid location

This would have been much easier to diagnose if the error messages had been even vaguely meaningful.

Office should have been able to tell that the Cache location was invalid. If the error message had said "Invalid cache location" instead of "Sorry, we couldn't open \\COMPUTER\filename" or "Microsoft Excel cannot open or save any more documents because there is not enough available memory or disk space", it would have saved me many hours of trouble.

Hopefully this helps someone else.

Text wrapping in a Active X textbox in a Word form - Microsoft Community

custardkabuki say:

Text wrapping in a Active X textbox in a Word form

I created a form in Word using Active X textboxes but I can't get the text to wrap for paragraphs or long sentences.  How do you do this?

stefan blom say:
I created a form in Word using Active X textboxes but I can't get the text to wrap for paragraphs or long sentences.  How do you do this?

In Design mode, right-click the control and choose Properties from the context menu. Change "Multiline" to "True."

Note, however, that ActiveX controls are tricky in Word. For one thing, macro security usually forces Design mode as you open the document. It will be easier to make use of the form fields instead.

For more on form fields, see http://word.mvps.org/faqs/tblsfldsfms/LinesInForms.htm and http://gregmaxey.mvps.org/Repeating_Data.htm.

Stefan Blom, Microsoft Word MVP

  2. Right click on the textbox. 
  3. Click Properties.
  4. Change MultiLine to True.

how do i disable in private browsing - Microsoft Community

dan moore(2) say:

how do i disable in private browsing

How do I disable in private browsing?

a. user say:

Hi Dan,

 

Thank you for asking in Microsoft Community!

 

Are you referring to desktop version or the Modern User Version of internet explorer?

 

To disable InPrivate browsing in Internet Explorer 10 desktop version, please follow the steps given below.

 

a)  Press the Windows +R keys to open the Run dialog, type iexplore -private and press Enter.

b)  Open the Internet Explorer 10 desktop version, press the Ctrl+Shift+P keys.

c)  A new Internet explorer will now open with InPrivate turned on. You can now browse to whatever site you like in InPrivate mode.

d)  To turn off InPrivate, you would just need to close the InPrivate window.

 

To disable InPrivate browsing in Internet Explorer 10 Modern UI version.

 

a)  Open Internet Explorer from the start screen,

b)  press the Ctrl+Shift+P keys.

c)  A new Internet explorer will now open with InPrivate turned on. You can now browse to whatever site you like in InPrivate mode.

d) To turn off InPrivate, you would just need to close the InPrivate window.

 

Hope the information provided was helpful, Please feel free to post for further assistance.
