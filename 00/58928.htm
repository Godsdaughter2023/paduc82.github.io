1/ Boot on Bios (depends on your brand)

Safe and reboot. Voilà

You will need to run and install Virtual PC in XP Mode to have any chance of a fully operational 16 bit subsystem. If you are running Windows 7 Professional or Ultimate, you can use Windows XP Mode and Virtual PC without a second Windows license.

jimmywin10 say:

When you start your computer up and The “Please Wait” loading appears and hasn’t made an progress in 5-10 minutes. You are probably in a start up loop. We will need to access Safe mode and go into services to end this loop. Follow these steps to exit out of this loop.

 

Step 1: You will need to access the advance options menu at start up. If you have safe mode set. Select the safe mode you want. If you select it and still get stuck at the please wait loop, that means the safe mode isn’t working. And you will have to enter through advanced option.

Step 2: manually power the computer Down. And start it up. Once the machine has reached the windows logo and you see the loading sign of spinning dots restart it manually again. Due this till the window symbol underneath reads Preparing for automatic repair.

Step 3: we need to navigate to start up settings to reach safe mode,  Once on the main page we are going want to click on trouble shoot. On the next page click on Advanced Options. On The following page click on Start up Settings and the hit the restart button on that page.

Step 4: On the restart a menu will appear with options. We are going to want to choose Safe MOdE.

Step 5: login when prompted.

Step 6: Windows key + R to open a run box

Step 7: type in services.msc in the run box.

Step 8: In the Services menu They are three services we want to locate. They Are Network List Service, Network Location Awareness, and Windows Error Reporting Service. We will want to stop all three of these and disable them for the start up. Remember we will have to re enable these once we get out of the loop. So make sure we know if they are manual or automatic

Windows Error Reporting Service is “Manual” Startup

Network List Service is “Manual” Startup

Network Location Awareness is “Automatic” Start up

Now since we know what to turn them back to we can disable them and do a reboot

 

Step 9: Reboot and now we should be able to access the login screen  after the load/start up is finished

Step 10: Go back to services and turn the ones we turn off back on.
