MS does not give users ANYTHING to display the Product Key after it as been installed. All you can see is the "Product ID", which is generated by a 1-way Hash from the Product Key.  There is a 1:1 correspondence between the Product Key and Product ID, so you can use that to match them.

The reason for the duplication is that you are writing at the bookmark then not resetting the bookmark to include the added content, so next time it is inserted at the old bookmark and therefore duplicated. graham mayor say:

FillBM inserts text strings so you need to convert the value of the checkbox to a meaningful text string. You can do that one of two ways. If you simply want to enter True or False then you can use

FillBM "BookmarkName", CStr(CheckBox1.Value)

or you could use

IF CheckBox1.Value = True then

    FillBM "BookmarkName", "The TRUE value text"

Else  

   FillBM "BookmarkName", "The FALSE value text"

End If

Option buttons are usually grouped in sets of more than one option, so you can use

Select Case True

     Case Option1.Value: FillBM "BookmarkName", "The Option1 text"

     Case Option2.Value: FillBM "BookmarkName", "The Option2 text"

     Case Option3.Value: FillBM "BookmarkName", "The Option3 text"

     'etc

End Select
