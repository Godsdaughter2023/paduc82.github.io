jasles say:

Say column B has one activity in it. If you put the student limit in cell B1, use data validation in all cells in column B with this formula:

=(COUNTA(B:B)-1)<=$B$1

And then put the message you want to pop up when the limit is reached.

Use this same process for the other columns.

If you have a title in the column or other entries in the column, you'll need to adjust the 1 in the formula to the appropriate number and not use cell B1 in the formula, but use the appropriate cell that you use to enter the limit. 

yea so say:

lock the cells you don't want editable and password protect the worksheet      

by default all cells are in locked mode so when you password protect a worksheet all cells are immediately uneditable.

Select the cells you want not locked and editable for that activity then right click the selected cells, and choose format, when the dialog box comes up, select the protection tab and uncheck the lock checkbox.

after you have done this for all the activities, password protect the worksheet and it will lock the cells and be uneditable by anyone except for the cells that had been unlocked during the format.
