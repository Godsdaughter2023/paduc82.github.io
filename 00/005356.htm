jenrg say:

I'm adding to this because we've found the source of the problem.

First we did upgrade the user to Office 2010 (yes, we're fully aware that support ends next year).

Once we did the upgrade, the user started getting lots of "Undelivered Mail Returned to Sender" mails.  It turns out that since the user only downloads the headers and deletes the unwanted mails, the system was sending "Not Read" receipts even though the machine was set to never send a read receipt.  Those not read receipt which were stuck in the outbox, got sent after the upgrade.

From what I can see, this had been a problem with IMAP accounts, however a few users (like ours) had the same problem with POP3 accounts.

The reason we'd never figured this out before is because the whole internet talks about ghost read receipts getting stuck in the outbox, but in our searches, we've never come across the concept that it's not a "read receipt" but a "not read receipt".

I'm posting this just so that others who are having the same issue understand the cause.

No we're hoping that the problem doesn't come back with Outlook 2010, but we've got a better idea of how to handle it if it does.

vijay a. verma say:
Try following one by one and see after every step whether it resolves the issue-

1. Try repairing your office installation (Quick repair is good enough). Try repairing your Office installation. First try Quick Repair (takes 5 - 10 mins) , if that doesn't help then try Online Repair (Takes more than 30 mins but less than 1 hour in general)

https://support.office.com/en-us/article/Repair...

2. Sometimes, add-ins can cause problems. To determine, if add-in is a problem start your application in safe mode and see if problem has gone away.

To start in Safe Mode -

Hold CTRL key and click on application icon and don't release the CTRL key till it asks you for Safe mode confirmation.

If yes, then start application normally and disable add-ins one by one and start application again every time you disable an add-in to determine the culprit add-in.
To disable add-in > File > Options > Add-ins > Down below you have Manage: Com Add-in : Press Go > Untick the Add-ins to disable and tick to enable them

3. Login to Windows with a new user profile and see if problem is resolved.

4. Sometimes, this is caused due to application not being able to access the default printer. Close all office applications and change your default printer through Start > Devices and Printers to another printer. It may be some physical one or One Note or XPS or PDF or anything else. If this is the problem, then you will need to update your printer driver for default printer.

5. Many third party applications also cause the problem like Tuneup Utilities, Abby Finereader, Kaspersky & other Anti Viruses etc. If you want to know if some program is conflicting -

Boot your computer in Clean boot mode and then start the application. If it works successfully, then some program is conflicting with this.
Refer to the section - How to determine what is causing the problem after you do a clean boot.
Don't forget to do - Reset the computer to start normally after clean boot troubleshooting.
https://support.microsoft.com/en-us/help/929135...
