rob koch say:

Let me add to Steve's response to hopefully clarify:

The only updates that MSE has ever done automatically are a daily silent update that has no visible means of control and an optional setting to "Check for the latest virus & spyware definitions before running a scheduled scan."

The Microsoft [Windows] Update, Automatic Updates optional MSE updates that are the subject of this thread have quite simply never installed themselves automatically, which is why many here complained so loudly.  Unless and until either of the automatic update processes mentioned above occured, this delivery method simply sat there waiting for you to install them.

The only change that has occured in conjunction with the latest monthly engine update release is that these MSE updates are now classified as 'Optional' rather than 'Recommended' (Important is a display category that includes Recommended updates), so rather than displaying the notification icon in the tray, they are hidden and must be installed directly from the Microsoft Update interface itself.

This in no way is intended to classify these updates as unimportant, it's simply a control selection that allows the MSE updates to be offered via the Automatic Updates channels without setting off the notification icon, which was the primary complaint.

To put the situation in context, "You can't have it both ways".  It either will display the notification icon or it won't, so the current attempt is to still offer them, but not put it in everyone's face, which should satisfy most.  The gain is that you still have the option to install these optional (and they really are) MSE updates more often if you prefer, since otherwise you'd only receive them automatically at 24 hour intervals, as has always been the case.

BTW, the reason they can offer this option is that the updates delivered via Automatic Updates are actually smaller (~450KB) packages of three to four of the most recent (3 times daily) releases, while a normal delta update package starts off about 100KB in size and quickly increases to several MB by the end of a typical monthly cycle.  By offering these smaller packages more often, they are able to keep the package size small and so not overload the servers as the month wears on.
