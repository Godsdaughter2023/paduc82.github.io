Follow this method:
You must complete the upgrade before the 29th, and activate it.
Once that is done, your licence has been "claimed".
That means: the activation server has saved it's "digital fingerprint".
Afterwards you can use the Go-back feature to return to Windows 7 and keep using it as always.
Once the POS software has been updated by it's vendor, you can upgrade again to Windows 10.

Then it will activate automatically since it will recognise the "digital fingerprint".
Beware: this will ONLY work if no hardware is replaced in the mean time!
Otherwise the "fingerprint" will not be recognised and activation will fail (and then you'd need to buy a licence).
