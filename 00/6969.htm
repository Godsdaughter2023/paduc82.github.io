shashirekha say:

To sort text in one column, the text must be in a table. You can create a table in MS Word 2007 from the Insert tab by clicking on the down arrow under the Table group, or you can create a table from a tabbed list in your document. Here’s how you create a table:-

1.     Make sure each data column is on the same tab stop in the ruler at the top of the document. (Otherwise, you will end up with more columns than you need.)

2.     Select the text.

3.     Click the Table down arrow (second icon on the right on the Insert tab. Select Convert Text to Table (third selection down in the menu).

Sorting the Table

1.     Click anywhere in the table. The Table Tools Tab will appear at the top of the window with two additional Tabs Design and Layout.

2.     Select Layout and click on Sort in the Data group on the far right. Select which column(s) you wish to sort by.

Note: Select the My List has a header row button to make sure your header rows are not sorted with the rest of the columns.

scottgem say:

There are two ways to send mails from Access. The simpler way is using the SendObject method. One of the parameters of that method is the message body. You can construct whatever body text you want. Usually, you create a variable, assign the text you want, then use the variable in the SendObject method.

 

The other way uses Office Automation to generate the e-mail. A property of an e-mail generated using Office Automation is the Body property. And similarly you would generate the text you want and assign it to the Body property.

 

You should be able to find many examples of using either method.

 

If your letters involve multiple records, then you should be using a report to generate the form letter you want to use. What I do in such a circumstance is use the OutputTo method to create an HTML file from the report then import the HTML file as the HTMLBody property of the e-mail.
