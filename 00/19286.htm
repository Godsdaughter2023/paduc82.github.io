Brief answer: you don't.

In the screen shot above you can see such a border applied to the word "Themes." It is about to be applied to the word "look" which is selected. It can be confusing because the borders menu is in the paragraph group of the ribbon.

Sub SelectionBorder()
'
    With Selection.Font.Borders(1)
        .LineStyle = Options.DefaultBorderLineStyle
        .LineWidth = Options.DefaultBorderLineWidth
        .Color = Options.DefaultBorderColor
    End With
End Sub

Hello TrudiePegler,

 

 

You can only delete messages you've received or sent. There is no option to prevent or delete incoming messages.

 

Depending on the Skype version you have installed, you can follow the steps below:

 

*For classic Skype 7.41, there is no option to delete a particular conversation. You can only hide it by right clicking on the chat, then select Hide conversation. To delete your entire chat history (all of the messages from all of your contacts) using this version:

 

1. Click Tools.

2. Select Options...

3. Click IM Settings.

4. Select Clear History.

 

*For the new Skype version 8.xx, we do not have an option to delete entire conversations at once, but you can delete them one by one. To do this, just right-click on the chat you want to delete and select Delete chat. Then confirm removal by selecting Delete.

doug robbins - mvp office apps & services (word) say:

Starting in Safe mode will prevent any add-ins from loading, so start Word normally and check to see what add-ins are loaded via the Office Button>Word Options>Add-ins.

That dialog will show you the path of the Add-ins, so with Word shut down, go to that folder and create a sub folder under it named Store and move all of the add-ins to that folder.  If when you restart Word, the problem has gone away, you can then move them back one at a time to identify the one that is causing the problem.

As an alternative, does Ctrl+C (copy), Ctrl+X (cut) and Ctrl+V (paste) work?
