Thanks for the comprehensive documentation. Sorry to be so long getting back to you.

Before we go any further, please make sure that TLS 1.0 is enabled: in Control Panel, open Internet Options. The secure connection options should be like this:

Otherwise, I can only suggest that something's gone wrong with that account on that machine. I've not before come across a case where the user signs in apparently successfully but then shows none of the symptoms of being signed in.

If this is the case, then removing the account and then adding it back again should dispense with any corruption in the account itself.
