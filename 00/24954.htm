We had created a questionnaire in word with radio buttons that prompted the user to select "Yes" or "No".  We have the responses back and we wish to count the number of "Yes" responses and "No" responses.  Can this be done through Word without having to use VB or exporting it to Excel?  We have over 100 questions with responses.  Thanks.
doug robbins - mvp office apps & services (word) say:

If the Captions that you have used for the Radio Buttons are "Yes" and "No", running a macro containing the following code will give you the number of "Yes" answers:

Dim lngYes As Long
Dim i As Long
lngYes = 0
With ActiveDocument
    For i = 1 To .InlineShapes.count
        With .InlineShapes(i)
            If .Type = 5 Then
                If .OLEFormat.Object.Caption = "Yes" And .OLEFormat.Object.Value = True Then
                    lngYes = lngYes + 1
                End If
            End If
        End With
    Next i
End With
MsgBox "The number of Yes answers is: " & lngYes

john korchok say:

Inserting section breaks is the first step. Then, after that, choose View>Header and Footer to open the header. As soon as you do, You'll see a light blue rectangle drop from the bottom boundary of the Header that reads something like;

Header -Section 2- | Same as Previous | Close

This is telling you that the section 2 header is linked to the previous header, so whatever formatting changes you make to one section will affect the other. To change this, look for the Formatting Palette, a small dialog that floats off to the side. If it's not showing, choose View>Formatting Palette.

In the formatting Palette, expand Header and Footer, then uncheck Link to Previous. As soon as you do this, the light blue area reads:

Header -Section 2- | Close

indicating that the header is no longer linked. Now you can make your changes and it won't affect the previous one.

Because you have both even and odd headers, you must unlink both of them. Use the steps above on both an even and odd page in the new section and you should be good to go.

ronrosenfeld say: The following will return the addresses of the red cells, according to the rule which you have stated above:

===========================
Option Explicit
Sub foo()
  Dim c As Range
  Dim rApplies As Range
  Dim addresses() As String
  Dim i As Long
  Dim s As String

Set rApplies = Range("E3:E20")
For Each c In rApplies
    If Val(c.Offset(columnoffset:=1)) Then
        i = i + 1
        ReDim Preserve addresses(1 To i)
        addresses(i) = c.Address
    End If
Next c
       
For i = 1 To UBound(addresses)
    s = s & vbLf & addresses(i)
Next i

MsgBox ("Red Cells: " & s)

End Sub
