louie del say:

We appreciate the details that you’ve provided. We can try to reset the Start menu layout on your PC as it may be corrupted and can be the reason why it still shows the duplicate icon.

 

For us to reset the Start menu lay out on Windows 10, we’ll be needing to have three user account on your computer:

  • Your account – The account that has the duplicate icon in the Start menu.
  • Local account – This will serve as a temporary account that we’ll need to create a copy of the Start menu files so we can reset the menu on your account.
  • Administrator account – This account will be used to copy the Start menu files between account since Windows won’t allow to overwrite files while they’re in memory.

Here’s an article that you can use to create a new local user account in Windows 10.

Below are the steps on how to enable the built-in Administrator account.

To copy database files between account, another account is needed to be used. You can either create another account with permissions, or the Windows 10 built-in Administrator account can be utilized to complete this task.

Enabling the built-in Administrator account:

  1. In the search box, type Computer Management, click on the result.
  2. Expand Local Users and Groups.
  3. Select Users.
  4. Double-click the Administrator account.
  5. Uncheck the Account is disabled option to enable the account.
  6. Click Apply.
  7. Click OK to complete the task.

Steps on how to reset the Start menu layout:

  1. Sign out the account with the duplicate icon.
  2. Sign in to the local account that you’ve created to populate and set the environment.
  3. Sign out the local account that you've created.
  4. Sign in to the Administrator account.
  5. Open File Explorer.
  6. On the View tab, check the Hidden items option to show hidden files.
  7. Navigate the following path:
    C:\Users\[Username of the account that has the duplicate icon]\AppData\Local\TileDataLayer

    Remember: Make sure to change the Username in the path with name of the user account that has the duplicate icon.
  8. Right-click the Database folder that has the Start menu layout settings and select Rename.
  9. Rename the folder Database.bak and press Enter.
  10. Navigate to the TileDataLayer folder of the new account that was created:
    C:\Users\[Username of the account without the duplicate icon]\AppData\Local\TileDataLayer

    Remember: Make sure to change the Username in the path with name of the user account that you’ve created earlier without the duplicate icon.
  11. Inside the TileDataLayer folder, look for the Database folder that contains the working layout of the Start menu, right-click the folder and select Copy.
  12. Navigate to this path:
    C:\Users\[Username of the account that has the duplicate icon]\AppData\Local\TileDataLayer

    Remember: Make sure to change the Username in the path with name of the user account that has the duplicate icon.
  13. Right-click the inside the folder and paste it from there to copy the new Database folder to your account.
  14. Sign out to the Administrator account.
  15. Sign in to the original user or the account that has the duplicate icon.

As soon as everything is completed, the Start menu is expected to be reset and duplicate icon should be removed already.

If just in case that it resolved the issue with the duplicate icon, the Database.bak folder that has the corrupted Start menu data can be deleted now.

  1. Press the Windows key + R from the keyboard to open the Run command.
  2. Type this command to open the TileDataLayer folder and hit Enter:
    %HOMEPATH%\AppData\Local\TileDataLayer
  3. Right-click the Database.bak folder and select Delete to perform the action.

Also, make sure to disable the built-in Administrator account:

 

  1. Press the Windows key + X to launch the power user menu and choose Computer Management.
  2. Expand Local Users and Groups.
  3. Select Users.
  4. Double-click the Administrator account.
  5. Check the Account is disabled option to disable the built-in account.
  6. Click Apply.
  7. Click OK to complete the action.

To delete the temporary local user account that was created:

  1. Go to Settings.
  2. Click on Account.
  3. Click on Family & other people.
  4. Click on the account that you’ve created.
  5. Click the Remove button.
  6. Click the Delete account and data button to perform the action.

For any results, please send us a response.
