skat0r vista say:

Remotely Shut Down a Computer

Remote Shutdown Tutorial

Read this article if you would like to remotely shut down a computer on your network.

Compatible with:

Windows 10, 8/8.1, and 7

1. Go to the Start Menu.

2. Type cmd and right-click to Run as Administrator.

3. In the new window, type net view.

4. A list of computer names will come up.  Decide on which one you wish to shut down.

5. Type shutdown -i

6. A box will pop up.  Press on the Add button on the top right.

7. Fill out the rest of the boxes.  They are pretty self explanatory.

8. Press OK.

9. The computer(s) will shut down.

NOTE: Shutting down computers while files are unsaved can cause data loss.  To abort a shutdown on your computer, type shutdown /a.

Only by installing 64-bit Windows 8.

 

Any 32-bit Windows can only address 4GB of RAM, and part of that is reserved for video memory and system BIOSes.

ken blake - microsoft mvp say:

On Sat, 15 Dec 2012 07:21:39 +0000, nitin786 wrote:

before installing windows 8, i was using windows 7 professional 64 bit edition and all the 4 gb ram was usable but after installing windows 8 pro 32 bit the usable ram is only 2.69 gb.

All 32-bit client versions of Windows (not just XP/Vista/7/8) have a
4GB address space (64-bit versions can use much more). That's the
theoretical upper limit beyond which you can not go.

But you can't use the entire  address space. Even though you have a
4GB address space, you can only use around 3.1GB of RAM (in your
case, a little less than that). That's because some of that space is
used by hardware and is not available to the operating system and
applications. The amount you can use varies, depending on what
hardware you have installed, but can range from as little as 2GB to as
much as 3.5GB. It's usually around 3.1GB.
