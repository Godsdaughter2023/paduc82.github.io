 I thought in Outlook 2010 the contacts were in the PST file as well.


Contacts are in the PST file but the AddressBook and contact folders are 2 different things so a lot depends on what actions you have taken as part of the restore process.

If the restored PST file has been set as the "default" PST file for the profile, then generally the contact folder in the same tree as the INBOX is automatically set to be shown as part of the AB. The AddressBook is nothing more than a "view" of one or more contact folders which are set to "Show this folder as an email addressbook" by right clicking on the folder --> properties --> AddressBook tab --> make sure the check box is checked.

If the above checkbox is greyed out and unchecked - then the best solution would be to create a new profile via the Control Panel mail applet.

Difference between .xlt and .xls files - Microsoft Community

donmckelvay say:

Difference between .xlt and .xls files

For some reason I have created a file and saved it as .xlt.  I understand it is a template but do not understand the differences between a .xlt or .xls, or which I should use and why.  Appreciate your advice.

hansv mvp say:

When you double-click a .xls file in File Explorer, Excel opens that .xls file. If you then select File > Save or press Ctrl+S, Excel saves the changes to the existing .xls file.

When you double-click a .xlt file in File Explorer, Excel creates a new, unsaved workbook in memory that is a copy of the .xlt file. If you then select File > Save or press Ctrl+S, Excel displays the 'Save As' dialog, letting you save the workbook as a .xls file.

You can create a template if you frequently need to create a new workbook with the same design and/or formatting. Let's say, for example, that you need to produce a monthly report. The data will change from month to month, but the layout must be the same each time. You could save a workbook with the required layout as a template. Each month, you can double-click the template to create a new report.
