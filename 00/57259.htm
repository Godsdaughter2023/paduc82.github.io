considerate_guy say:

Windows DVD Maker was created for Windows Vista. It was also in Windows 7. It hasn't been changed since then. Back when Windows DVD Maker was in active development, Movie Maker only created output in .wmv format so that's why Windows DVD Maker doesn't have support for .mp4 files.

In order to make this process work, you need to save your Movie from Movie Maker in .wmv format. Start the save process as usual. File > Save movie. At the point where you can change the file name, you can also change the file type.

The drop-down for 'Save as type' defaults to MPEG-4/H.264 (*.mp4) in Movie Maker 2012. But you can change this to Windows Media Video File (*.wmv). That file should be recognized by Windows DVD Maker.

P.S. Windows DVD Maker is not being updated, so don't expect this to change. In addition, Windows DVD Maker isn't even present in Windows 8 or later. Assuming you have plans to update Windows, be aware you may lose the ability to use Windows DVD Maker. I would suggest either looking for alternate software for burning DVDs, or foregoing the process altogether. Video DVDs are in MPEG-2 format and can only handle standard definition (SD) video. If you give it HD video, it will be down-rezzed to 480p (NTSC) video. If you want other options, consider just streaming HD video via your network to your HDTV and/or sharing video on sharing sites like YouTube, OneDrive or Vimeo.

Hi,

Code 10

If the device has a FailReasonString value in its hardware key, that string is displayed as the error message. The driver or enumerator puts this registry string value there. If there is no FailReasonString in the hardware key, you receive the following error message:

This device cannot start. (Code 10)

Recommended resolutions

Update the driver

In the device Properties dialog box, click the Driver tab, and then click Update Driver to start the Hardware Update Wizard. Follow the instructions to update the driver.

You may be prompted to provide the path of the driver. Windows may have the driver built-in, or may still have the driver files installed from the last time that you set up the device. If you are asked for the driver and you do not have it, you can try to download the latest driver from the hardware vendor’s Web site.

Probably because the index is too large, but remember it's also tuned to receive instant results from user documents with human-readable content:  documents, presentations, emails, spreadsheets, powerpoint slides.  It's not tuned to instantly search the file system folders.  It can do it, but you need to set some options for how it searches in non-indexed locations (for starters, tell it to only search the file name in those locations so it can speed through them by name only).  And remember the power of wildcards like ? and * which can be used just like the old days to find items.
