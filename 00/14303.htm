I interpret that to mean you want to get the sum total of unique numbers in a filtered list.

Let's assume this is your data in the range A3:B15 -

Type Value
B 1
A 1
B 1
B 2
B 2
B 2
A 3
A 4
B 4

Try this array formula**:

=SUM(IF(FREQUENCY(IF(SUBTOTAL(2,OFFSET(B3:B15,ROW(B3:B15)-ROW(B3),,1)),B3:B15),B3:B15),B3:B15))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

The sum total of the unique numbers is 28 (1+2+3+4+5+6+7)

If you filter on Type = B then you'll get:

Type Value
B 1
B 1
B 2
B 2
B 2
B 4
B 7
B 7

The sum total of the unique numbers is 14 (1+2+4+7)

macropod say:

By design, code that addresses a document range (e.g. .Content) acts only on the body of the document. To process, headers, footers, textboxes, footnotes, endnotes, etc., you need to either loop through all story ranges or specify each of the story ranges to process. For example:

Sub Demo()
Application.ScreenUpdating = False
Dim Rng As Range, Sctn As Section, HdFt As HeaderFooter
With ActiveDocument
  For Each Rng In .StoryRanges
    Call FndRepRng(Rng)
  Next
  For Each Sctn In .Sections
    For Each HdFt In Sctn.Headers
      With HdFt
        If .LinkToPrevious = False Then
          Call FndRepRng(HdFt.Range)
        End If
      End With
    Next
    For Each HdFt In Sctn.Footers
      With HdFt
        If .LinkToPrevious = False Then
          Call FndRepRng(HdFt.Range)
        End If
      End With
    Next
  Next
End With
End Sub

Sub FndRepRng(Rng As Range)
With Rng.Find
  .ClearFormatting
  .Text = "~CASE_REF~"
  .Replacement.ClearFormatting
  .Replacement.Text = "ABC123"
  .Forward = True
  .Wrap = wdFindContinue
  .Format = True
  .Execute Replace:=wdReplaceAll
End With
End Sub

Note the additional circumlocution for headers & footers, to take account of Section breaks.
