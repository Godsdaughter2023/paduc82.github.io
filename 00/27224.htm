Any function to determine delta within Black scholes model? - Microsoft Community

oemsoft say:

Any function to determine delta within Black scholes model?

Does anyone have any suggestions on any function to determine delta [Black scholes model] within Excel?

Thanks in advance for any suggestions

joeu2004 say: OEM wrote:

Does anyone have any suggestions on any function to determine delta [Black scholes model] within Excel?


Not an expert in Black Scholes, and I agree with Ossie's pointer as a paradigm.  But to answer your question....

Based on the Black Scholes wiki page, you would use the Excel NORMSDIST function to compute "delta".

For a European call option for a non-dividend-paying stock, the "delta" is [1]:

=NORMSDIST((LN(S1/K1)+(R1+V1^2/2)*T1)/V1/SQRT(T1))

where:

S1:  current stock price
K1:  option strike price
R1:  annual risk-free rate
V1:  annual historical volatility
T1:  time to maturity, in years

The "delta" for a European put option is that value minus 1.

According to the free option pricing workbook that Ossie points to, the "delta" for a European call option for a dividend-paying stock is the formula above, replacing R1 with R1-D1, where:

D1:  annual dividend yield
