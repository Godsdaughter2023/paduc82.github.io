jonavn say:

I figured out my issue, although I'm not sure it will be the same case for everyone.  Here's what I did:

  • Go to Settings
  • Search for "Screen Saver"
  • If the Wait time is set to 0 and the screen saver is disabled, enable the screen saver, set the time to 15 minutes (or whatever you want other than 0), and then disable it again (if you want).

After I did that, the screen timeout behaved normally.

Hopefully this solves the issue for others, as well.

jonavn say: I did a little more experimenting, and apparently the screen saver wait time overrides the other power settings, even if the screen saver is set to None.  Also, 1 minute is the minimum time (so if you set it to 0 it will automatically get changed to 1), so that should explain the timeout everyone is seeing.  If you really want the power setting, just set the screen saver wait time to something bigger than that, and you should be covered.
