To unlink OneDrive from Office 2013 do the following:

  1. Open Word 2013 and then click the "File" tab, then click the "Account" tab
  2. Click "Sign out" under your User Information

That will disconnect your entire Microsoft account from Office 2013. If you purchased Office 365 though your subscription is activated through your Microsoft account, so it always must be signed in. If you want to prevent Office from saving files by default to OneDrive without disconnecting anything, follow these instructions:

  1. Open Word 2013 and then click the "File" tab, then click the "Options" tab
  2. Click the "Save" tab
  3. Select "Save to Computer by default" under the Save documents section

how to read/verify Windows Regional Settings in Excel vba - i need - Microsoft Community

svetlana_ say:

how to read/verify Windows Regional Settings in Excel vba - i need to know Date format

I need to know date format for user's PC before I will start my code execution. Please help me to solve this. Thanks a lot.

ronrosenfeld say: Examine HELP for the International property of the Application object.

For example:

========================
Option Explicit
Sub RegionalDateFormat()
    Dim DateOrder As String
    Dim DateSeparator As String
   
With Application
    Select Case .International(xlDateOrder)
        Case Is = 0
            DateOrder = "month-day-year"
        Case Is = 1
            DateOrder = "day-month-year"
        Case Is = 2
            DateOrder = "year-month-day"
        Case Else
            DateOrder = "Error"
    End Select
   
    DateSeparator = .International(xlDateSeparator)
End With

Debug.Print "Date Order: ", DateOrder
Debug.Print "Date Separator: ", DateSeparator
End Sub

jay freedman say:

You can't alter the  main dictionary at all -- it's binary code. What you can do is place a custom dictionary on a server and add it to each workstation's list of custom dictionaries. Don't make
it the "default" custom dictionary, to prevent the "Add to dictionary" option in the spell-checker from trying to add more words to it (you may also want to make the file read-only).

See  http://sbarnhill.mvps.org/WordFAQs/MasterSpellCheck.htm for more info.
