suzanne s. barnhill say:

The only way to do this is to set the compatibility to Word 2010, as follows:

  1. Press Alt+F11 to open the VBA Editor and Ctrl+G to get to the Immediate window.
  2. In that window, type (or paste)
             ActiveDocument.SetCompatibilityMode 14
    and press Enter.
  3. Close the VBA Editor.
  4. Go to File | Options | Advanced and scroll to the bottom.
  5. Underneath Compatibility options for, there will now be a dropdown to "Lay out this document as if created in." Select Word 2010.
  6. You'll now get a new list of Compatibility Options that will include the "Suppress..." option (which will not be checked).
