Hi Danny,

Firstly, it is possible to send emails with the size larger than 10MB and change the default size limit of a mailbox. Normally, in Outlook client, the maximum limit size of a message in Exchange Online Plan 1 mailbox is 150MB. In Office 365 OWA (Outlook Web App), the maximum limit size is 35MB. So if an admin has set the default size limit to 10MB, this admin can also change this default limit to larger than 10MB via PowerShell. If you are not an admin, please contact an admin of your Office 365 tenant to help you. Here are the detailed steps:
1. Connect to Exchange Online with PowerShell.
2. Run the following commands.
1) Set the default size limit of a single user.
Set-Mailbox -Identity *** Email address is removed for privacy *** -MaxSendSize 75MB -MaxReceiveSize 75MB
2) Set the default size limit of all users.
Get-Mailbox | Set-Mailbox -MaxSendSize 75MB -MaxReceiveSize 75MB

A questionnaire database has a very specific structure. This is the basic strucutre I normally use:

tblQuestion
QuestionID (Primary Key Autonumber)
Question

tblRespondent
RespondentID (PK Autonumber)
Firstname
Lastname
other info about respondent

tblResponse
ResponseID (PK Autonumber)
RespondentID (Foreign Key)
ResponseDate

tblAnswer
AnswerID (PK Autonumber)
QuestionID (FK)
Answer

You would add a field in tblQuestion for the path to the image file you want to display with your question.
