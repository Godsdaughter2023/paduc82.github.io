andre da costa say:

If Windows crashes, because of bad drivers or any other reason, during the upgrade process we’ll fail the upgrade and restore your previous operating system.  On the upgrade progress screen you'll see three steps listed along the bottom "Copying files," “Installing features and drivers”, and “Configuring settings.”

During "Copying files", we actually boot into a minimal version of Windows and copy important files and drivers from the old operating system into the new one.  If we fail in this first stage you’ll see an error code in the Windows Update control panel called c1900101 x200xx.  For example, 20017 means we couldn’t even boot into this operating system and usually points to bad storage drivers.

The “Installing features and drivers” phase is the trickiest part of upgrade.  If you see a failure c1900101 x300xx, it means we failed during this phase.  The most common failure is x30018, which generally means a driver crashed when we loaded it for the first time.  Any driver could cause the failure, but it may not crash 100 percent of the time.  If you try again, it may just work.

"Configuring settings" is the final phase and failures codes look like c1900101 x400xx.  The most common failure here is 40017, which means we failed to boot.  Since this is the first time we boot into Windows 10 with all of the drivers in place, again, drivers are usually the cause of the crash.  These issues tend to be less transient than the 30018s, so a second try is less likely to solve the issue. Still, it never hurts to try.

Source, Windows 10 Insider

¡firedog say:

file>options>mail>read>fonts and/or file>options>mail>compose

As you'll realize, those are different settings.

The first changes the default font for reading plain text messages. Most emails these days are composed in HTML, and the sender usually specifies which font styles to use, so you have no control over them. When reading a message, you can use the keyboard shortcuts Ctrl + (Control Plus) to increase the size of the text and Ctrl - (Control Minus) to decrease it, or Ctrl 0 (Control Zero) to set it back to 100%.

You can elect to read all messages in plain text on the Read tab of the program options dialogue, where you found the Fonts button. If you do this, then you can elect to read a particular message in its original HTML format by opening it and pressing Alt-Shift-H.

The second setting only changes the font style to be used when composing HTML messages. For plain text messages, you can select any font you like so long as it's 12 point Black Calibri.

The text size setting in Internet Explorer (View menu > Text size, or Alt-V X) will affect the size of some text on some web pages (not this one, for example), of all text in plain text messages in Windows Live Mail and of some text in some HTML messages (notifications from this site, for example). It will only take effect in Windows Live Mail after closing down the program, waiting a few minutes and then re-launching it.

You should be able to find some combination of these settings that will help you.

Try following and check after each step

1. Reset Photos App
Windows+X > Settings > Apps > Apps & Features > Locate Photos by Scrolling Down > Click Advanced Options > Click Reset

2. If Reset doesn't work, then you should uninstall using Powershell and reinstall the Photos App
Windows+X > Click Windows Powershell (Admin) > Copy the below command and right click in Powershell to paste it

get-appxpackage *photos* | remove-appxpackage

Now, you need to reinstall it
Use below Powershell command to install

Get-AppxPackage -allusers *photos* | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

3. Reset the Database of Photos App
Copy following and paste in Windows Start Search box (Replace Your User Name appropriately)
C:\Users\Your User Name\AppData\Local\Packages\Microsoft.Windows.Photos_8wekyb3d8bbwe\LocalState
Click on result and delete everything from inside the folder
