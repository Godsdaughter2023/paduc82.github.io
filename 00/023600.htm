The formula is checking if there is a value entered in B6, if there is a value entered in B6 then it will look if a value is entered into B7... If both of the cells have a value it will calculate B14-B7 otherwise it will return nothing in the cell.

 

An alternative way to write the formula could be =if(OR(B6="",B7=""),"",B14-B7).
