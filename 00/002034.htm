When you transfer contacts from a SIM card, they are copied to the phone's memory and if a Microsoft account has been set up on the phone, the contacts will be automatically linked to the Microsoft account and synced to it. jay freedman say:
Two further clarifications to what's in that article:

  • When you're seeing the error message "Compile error in hidden module", the culprit is definitely a template add-in, not a COM add-in, so you can concentrate on that portion of the solution.
  • There are two locations that need to be checked for templates:

Word 2003 on Windows XP
    C:\Program Files\Microsoft Office\Office11\Startup
    C:\Documents and Settings\<user name>\Application Data\Microsoft\Word\Startup


Word 2003 on Windows Vista or later

    C:\Program Files\Microsoft Office\Office11\Startup
    C:\Users\<user name>\AppData\Roaming\Microsoft\Word\Startup


If you're running 32-bit Office on 64-bit Windows, then substitute C:\Program Files (x86) for C:\Program Files.


There are some add-ins -- the Avery Label Wizard is one, and WinCalendar may be another -- that install a macro-containing template as an interface to the main program, but fail to delete the template when you uninstall the package.

An Excel documant is being passed around a group of people. In this way the reader can see when the version he is reading was last updated. mike h.. say:

Hi,

To do that automatically requires code. ALT+F11 to open VB editor. Double click 'ThisWorkbook' and paste the code below in on the right. Change the sheet and cell to the correct one

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
Sheets("Sheet1").Range("A1") = Now()
End Sub

Animations - Microsoft Community

karimcol say:

Animations

Can someone please explain to me how to completely stop animations please? I can stop it at my end but as soon as I share the link, the animations start again. This is not good for some of my readers because they have vertigo. Thank you! 

jason (office engineer) say:

As the author, you can go into the styles pane and set animation level to low, but that still won't avoid all animations.

Viewers can turn animations off via the settings button that appears in the upper right when the Sway is loaded.  You can load the page with them disabled by adding ?animationsdisabled=true to the end of your Sway URL.  Disabling animations like this doesn't work in the Slides layout though.

Another thing you can consider to totally eliminate all animations is using accessibility view which is available in the ... menu after opening a Sway (or by adding ?accessible=true to the end of your URL).  This will change the layout of your Sway, but could be your best bet.
