sheen valentine ali msft say:

Hi Atoosa,

Unfortunately it is not feasible to exclude the contents of a document from the search results. If you would like to search for only a specific column (or several specific columns), as a workaround, we suggest mapping your column(s) into a managed property.

You can map more than one column into a managed property and set an alias for it. Afterwards, you can search for this specific column / columns by using the query:
alias/managedPropertyName:"insert your search query here"

This will show only results from the columns that you map to that specific managed property.

For example, I have two columns named "Alias" and "DocAlias" respectively. I then map these columns to the managed property Refinablestring2 and added an alias "alternateName".

If I search using either of the query below:

Refinablestring2:"test"

Alias:"test"

I will get only results that has the value "test" from the column Alias and DocAlias.

Sheen
