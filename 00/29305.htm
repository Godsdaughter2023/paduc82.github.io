  1. Right-click the Start button then select Command Prompt(admin).
  2. Enter this command schtasks /Change /TN "Microsoft\Windows\LanguageComponentsInstaller\Installation" /Disable
  3. If it does not help, enable it back by entering this command schtasks /Change /TN "Microsoft\Windows\LanguageComponentsInstaller\Installation" /Enable
