a. user say:

Hi Meran,

A few possible reason can cause your external drive to not show in File Explorer. To diagnose and fix this issue, we suggest that you check the Disk Management window. To open it, follow the steps below:

  1. Right-click on the Start button.
  2. Select Disk Management.
  3. Alternatively, you can search diskmgmt.msc in Cortana.

Examine the list of drives in the Disk Management window and look for your external drive. Even if it doesn’t show up in File Explorer, it should appear here. If the drive appears, it may not be appearing in File Explorer because it needs to be assigned a drive letter. To do this, right-click the partition on the drive in Disk Management and select Change Drive Letter and Paths.

If you don’t see the drive in the Disk Management window at all, we suggest that you try connecting the drive to a different USB port or use a different USB cable. We also recommend that you avoid using USB hubs, and connect the drive directly to your computer.

mlg63 say: I had the same problem and discovered that a removable drive does not show up unless it is connected when the computer boots up everyday and Windows checks to see what drives are available. A backup drive should be available all the time but disconnecting is good security especially for drives with system images. Consider a separate removable drive for disaster recovery and an always connected one for Windows 10 constant backups. Or in your case, always connect the drive before  booting the computer on the days you intend to make backups. See if that makes a difference.

  1. Press Windows key +R.
  2. Type “devmgmt.msc” (without quotation marks) and press "Enter". Click “Continue” if necessary.
