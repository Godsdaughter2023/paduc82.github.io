firstace say:

"Default User" in Documents & Settings

I do not know why a Default User folder is there, or what if anything it does: so can it be deleted without compromising any necessary data?
azeez n say: Hi,

The My Documents folder is your own personal folder in which you can store your documents, graphics, and other personal files. When there is more that one person using the computer, Windows creates a My Documents folder for each user on the computer.

By default, the target or actual location of the My Documents folder is C:\Documents and Settings\\My Documents, where C is the drive in which Windows is installed, and is the currently logged-on user.

I'd not recommend you to delete the folder because it is created by Windows for each user on the computer.

Personalize new computer for my wife

I purchased a new Dell computer for my wife.  I used my Microsoft Account information to setup the computer because I want to add the device to my Office 365 account.  After setup I can't personalize her computer for her,  it shows my name at logon and uses my screen saver and display settings.  She  can still access her content Gmail and Chrome Apps.

marc cai say:

Hi Ellis,

We're glad that you're a part of the Microsoft Community. Hopefully we can provide best ideas to you.

Regarding your query, it will always be safe to remove an account from a PC. This should not harm other device settings. Meanwhile, you have the option to create a Local user profile (Administrator is recommended) in the new Windows PC and remove the existing one, or sign-up for a Microsoft Account for free, create a user profile and bind the newly created Microsoft Account:

- Create a local user or administrator account in Windows 10.

- Sign up for a Microsoft account.

To remove an existing User profile in Windows 10: Open the Settings app located on the Start menu's sidebar, click Accounts, and navigate to Family and other users. Click on the account you want to remove from your computer, and click Remove.

When things are done and you've successfully created wifey's profile, you can now help her personalize the new PC.

Should there be a need of further assistance, please feel free to chime us back.

All the best.
