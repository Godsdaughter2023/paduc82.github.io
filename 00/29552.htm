burrwalnut say:

Selective startup is showing because a program (or more than one) has been disabled from starting. Open a Run window (Windows Logo key+R), type msconfig and press Enter. Go to the Startup Tab and you will see a list of items, with and without ticks alongside them. Those that are unticked have been disabled from starting up.

 

It is when you make a change here and restart the computer that you will receive the message. Just put a tick in ‘Don‘t show this again‘ as you‘re effectively doing a selective start up.

lemp say:
This is why using msconfig to deal with startup issues is the wrong way to go.

To answer your specific question, click Start, then Click Run, then type msconfig in the "Open" box and click OK.

Running msconfig starts the System Configuration Utility.  It has several tabs, the left-most of which is the General tab that you are looking for. Click on that General tab and then click on the radio button for "Normal  Startup."  Then click OK.

When you reboot, you will be back in normal mode -- but likely whatever problem prompted "telephone support" to run msconfig in the first place will return.

If you want to fix things rather than merely hide them with msconfig, please post back describing the problem that returns when you restart in Normal mode.
