recovery - Microsoft Community

joshua a dalglesih say:

recovery

I have just accidently closed a document I was working on without saving it. Is there a way to get the document back?

 

beth melton say:

If you don't have some type of backup created then no, there really isn't a way to recover a document you closed without saving if using Word 2007 and below.

You might be able to dig through your temp files and Autorecover files but the chances of recovering any portion of your document using these methods are slim. Most temp files Word creates cannot be opened in Word and when you close your document any AutoRecover files that were created are automatically deleted. Plus, if you're using Word 2007 it's been my experience that AutoRecover files can't be opened. And there's the time factor involved in digging -- it may be faster to recreate the document. With this in mind, if you're interested in trying this here's an article that provides the steps:

http://support.microsoft.com/default.aspx/kb/316951?p=1

Now, if you're using Word 2010 and worked on the document then it now has the ability to recover unsaved versions of your files.

If the document was previously saved then open the document again. If a newer version of the file is available a message bar should appear below the Ribbon along with the ability to open the newer version. You can also click the File tab to open Backstage view and on the Info tab under the Versions heading you'll see the available Autosave versions.

If the document wasn't previously saved then click the File tab to open Backstage view and on the Recent tab, below the Recent Locations list, click Recover Unsaved Documents. This will open your UnsavedFiles folder and you may be able to find your document there.

~Beth Melton Microsoft Office MVP

Dim i As Long

    Next i
End With

Are you saying you don't have the VBA password?  If not, and if that prevents you from opening the References dialog, you're going to find it difficult to fix the reference.  You might try executing the following line in the Immediate Window to list the references:

You purchased monthly subscriptions. Purchasing multiples of the same subscription does not provide you with any new minutes for the month. The old subscription needs to complete its monthly cycle before new minutes are awarded. This holds true whether you consume all the minutes or cancel the old subscription. I suggest you contact Skype support over live chat to get a better understanding of how subscriptions work and to ensure you are not getting double billed.
