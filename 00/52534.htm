    Method 1: Update the Bluetooth drivers.

    The issue that you face could be due to some of the driver corruptions, hence, please try to update the Bluetooth drivers and check if the issue persists. To do this, please follow the below steps.

    2. In there, search for the Bluetooth drivers, and select the Bluetooth driver and right click it and select "Update driver software...".

    3. Restart the computer.

    Method 2: Remove the pairing and again pair the Bush TV with the computer.

    You can try to remove the Bluetooth pairing and thereby remove the BUSH TV from the Bluetooth devices list and then.

    Connect to Bluetooth devices

    https://support.microsoft.com/en-us/help/17156/windows-10-connect-to-bluetooth-devices

    If the issue persists, please follow the below method.

    Method 3: Run the hardware troubleshooter.

    The Hardware troubleshooter will try to troubleshooter any hardware related issue like the bluetooth, or mice or keyboard etc. Hence, please try to run the same and check if the issue persists. Please follow the below steps to do the same.

    a. Type “Troubleshooting” without quotes in the Search box and hit enter.
