Note: Please back up your registry first, you can refer to:

How to back up and restore the registry in Windows

a. Press Windows + R keys The 'Run' dialog should show up.

b. Copy %localappdata%\Microsoft\Office\16.0\WebServiceCache\AllUsers\officeclient.microsoft.com into the dialogue box and delete the files in this folder.

Note: Please back up the files in this folder before you delete them.
