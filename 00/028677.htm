stefan blom say:

What you are describing is the Click and Type function in Word. You can turn it off at File | Options | Advanced. Go to the "Editing options" section and clear the check mark next to "Enable click and type." Click OK.

As you have noticed, when you make use of click and type, blank paragraphs as well as excessive tab characters are added to the document.

A document never needs blank paragraphs, because having them present only means that you duplicate paragraph formatting for no good reason (instead use Spacing Before/After, Line Spacing, etc. to position text).

Therefore, it is best to turn off Click and Type.

Try the following:

Dim Source As Document, Target As Document
Dim i As Long
Dim ffname As String
Set Source = Documents.Open("path\nameofsourcedocument.doc")
Set Target = Documents.Open("path\nameoftargetdocument.doc")
With Source
    For i = 1 To .FormFields.Count
        ffname = .FormFields(i).Name
        Target.FormFields(ffname).result = .FormFields(i).result
    Next i
End With

ubes say:

moving pst files from one user to another

I have recently activated an admin user account and transferred daily email traffic to a standard account from the admin account. The last 2 years contacts and outlook folders are still in the admin account and I would like to transfer them to the standard account and de-activate Outlook from the admin account. ruchi bisht say:

       You can login to your Standard account and try the below steps:

 

1.      Browse to C:\Users\<user name of admin account>\AppData\Local\Microsoft\Outlook.

2.      Right click on PST file-> copy.

3.      Browse to C:\Users\< user name of standard account>\AppData\Local\Microsoft\Outlook.

4.      Right click-> paste.

5.      Open Outlook-> File-> Open -> Outlook data file and select outlook.pst  from the location C:\Users\< user name of standard account>\AppData\Local\Microsoft\Outlook->ok.

6.      If you want to set this pst file as default click on Tools-> Account settings ->Data files-> click on the pst file ->click set as default at the top-> click ok, close and open outlook.

7.      Copy your account setting from Admin account and enter the same account settings in Standard account.

8.      Once the  email account is set up in the Standard  account, login to Admin account -> open Outlook-> Tools-> Account settings -> Click on account -> remove account, that will remove your email account in old Admin account.

 

Note: If you don’t find App data folder visible, you need to ensure hidden files and folders are displayed. To do this

Open Windows Explorer

1.      Select Organize> Folder and Search options

2.      Click on the view tab and check the radio button Show hidden files and folders under hidden files and folders column.

 

Ruchi Bisht
