From the issue description, I understand that you are getting the verification code via your alternate email address or mobile phone, however, after you enter the verification code and click "Submit" by selecting the option "I sign in frequently on this device. Don't ask me for a code.", it takes you back to the original page where it again asks to trust the device and due to which you are unable to edit your security info linked with this Microsoft Account.

Normally, when you click on "Submit" button after entering the code, it will not give you any notification/message that the device is now trusted under your account. But instead, it will directly take you to the "Security info" page with the heading "Your security inform protects your account" showing all your security information like "Phone Number", "Alternate email address", "Recovery code" etc. with the options to add or edit them.

If you get this page, then it means that your device is now trusted under this Microsoft Account.

If this is not the page that you get and if it prompts you again to trust the device when you try to edit your security info, then there could be some issues either with the security code or your Microsoft Account. So, the best option here would be to contact the Microsoft Account Support from this link:

Microsoft Account Support
