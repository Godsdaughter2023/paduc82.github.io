Excel Hyperlink Options - Microsoft Community

robinkooy say:

Excel Hyperlink Options

I am creating a database in Excel with hyperlinks to various PDF files within a company server. Is there any way that I can lock the hyperlinks to the specific PDF file so that even if the file name gets changed the hyperlink will still pull that PDF? On the same topic, is there a way that I can prevent the hyperlinks from failing if I move the location of the excel document or any of the PDF files?
bill manville say:

Using normal hyperlinks you can make it safe to move the Excel document.

Before creating the hyperlinks set the Hyperlink Base for the workbook to \\NoServer\NoFolder or some other destination that does not exist.

That will ensure the hyperlinks are created as absolute rather than relative.

To set the hyperlink base: File > Info > Show all properties

To make it possible to move all the PDF files to a different location, use the HYPERLINK worksheet function.  Have a cell named Path containing the path to the folder and a cell for each file containing the filename.  Then
=HYPERLINK(Path & "\" & B2,B2)
will create a hyperlink in the cell to the file whose name is in B2.
If you subsequently move all the files you need only change the content of the Path cell.

This approach also avoids the potential problem of moving the Excel file since the hyperlinks are all absolute.

For filename changes - no chance.
