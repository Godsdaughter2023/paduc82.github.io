    1. Press windows key+R

    2. Type in regedit

    3. Navigate to HKEY_CURRENT_USER > Software > Microsoft > WIndows > Current Version > Policies > System

    4. Right click on Noclose

    5. Choose the option to delete Noclose

    6. Confirm that you wish to delete Noclose

    7. Reboot your computer

    on 2013 win8 laptop with 500gb internal hard drive:

Is it feasible to do system restore to a 64gb external drive in drive E (internal drive will be used for another operating system install). Can system restore be "guided" to the E drive 64gb drive?

    My HP User Guide says if hard disk is removed, use system reset (would result in a 2012 win 8 version).

    A system restore point is nothing to do with a new install of Windows.

    A system restore just puts your computer settings back to an earlier date in time on the same hard drive.

    It is nothing to do with transferring anything to another hard drive.

    A 64GB hard drive is not feasible anyway.

    If you remove the hard drive there is nothing to reset.

    HP recovery media is designed to reinstall on the same hard drive or a REPLACEMENT hard drive after the original has failed.

    It is not designed to install where you want it to.

    And F11 only works on the original HD as that is where the HP recovery image is imaged to.

    Skype for Windows 10 (14.x) is a UWP app that functions more like a service, integrated with the OS. It takes more steps to disable this integration. I have covered how in several other posts. The alternative is uninstalling it.

    Can't uninstall skype

    Skype for Windows 10 comes already packaged with the OS. By default it loads but does not sign in. The only recent change was the presence of a visible icon/shortcut to it. You have the option to stop it from loading or uninstall it.
