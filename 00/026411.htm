Your 81% is a good indicator that you have a non-Microsoft antivirus program installed.

When an upgrade to a new build fails, these steps may enable it to succeed:

  • Completely uninstall any non-Microsoft anti-virus software.
  • Perform a clean boot before upgrading.
  • Disconnect any un-needed peripherals, such as devices connected via USB.
  • Disable all network adapters after the build has downloaded but before it Restarts, or before running setup.exe, if you are upgrading via a mounted ISO or Windows 10 install media.

Sometimes disconnecting the Internet, such as unplugging the Ethernet cable, will enable a stalled upgrade to complete.

For other users with the same problem who come across this thread, this is what I did:

1. Went to my Control panel (start menu)

2. Went to User Accounts and family safety

3. Selected add or remove user accounts

4. Selected create a new account  and selected administrator

5. Selected create account

6. I then went to my start menu and logged off

7. I then logged back in as my original administrator user

8. I then selected the link provided in the email from HUP when given the option to download or save I saved it to my desktop.

9. When the icon showed on my desktop I right clicked it and selected run as administrator and then followed the instructions displayed during the software installation

Hope this helps anyone else with the problem. Thanks again Nayana
