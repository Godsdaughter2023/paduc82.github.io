jeeped say:

When you create your conditional formatting rule, option for Use a formula to determine which cells to format and then supply a formula for Format values where this formula is true: that resolves to some form of TRUE or FALSE. In your case it could as simple as,

=A2>9

  ... or for both upper and lower,

=OR(A2>9, A2<5)

One thing to note: Excel considers FALSE to equate to zero and by strict definition, anything that is not FALSE is TRUE. Any formula that results in a number will be TRUE if not a zero and FALSE if a zero.

One other thing to note: You can test your CF rule's formula on the worksheet and observing what the formula is returning. This can save some time as there is limited functionality in the Create New Rule dialog.
