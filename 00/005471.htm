lz. say:

Hi,

The following code does the job whatever the number of values exist in columns A & B:

Private Sub foo()

   Dim lastRowA As Long, lastRowB As Long
   lastRowA = ActiveSheet.Cells(Rows.Count, "A").End(xlUp).Row
   lastRowB = ActiveSheet.Cells(Rows.Count, "B").End(xlUp).Row

      For j = 2 To lastRowB
         Cells(j + k, 3).Value = Cells(i, 1).Value
         Cells(j + k, 4).Value = Cells(j, 2).Value
       Next j
       k = k + lastRowB - 1
   Next i

Note thank blank cells in columns A & B are not ignored.

Hope this helps
