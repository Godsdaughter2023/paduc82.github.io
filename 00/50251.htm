slance310 say:

Hi,

 

A couple of ways to get to Safe Mode...

 

1) Boot from the Windows 8 DVD
 2) Select Next
 3) Select Repair your computer

 4) Select Troubleshoot
 5) Select Advanced Options
 6) Select Command Prompt

 7) Type - Bcdedit /set {bootmgr} displaybootmenu yes       (take note of spaces)
 8) Type - exit

 9) Reboot

10) At the boot screen select F8
11) Select Safe mode with Networking

This can then be turned off when finished.

 1) Start Command Prompt (Admin) by right clicking lower left corner when Start pops up.

 2) Type - Bcdedit /set {bootmgr} displaybootmenu no       (take note of spaces)
 3) Type - exit

 

 

Another Way...if you don't have a Win 8 Disk....

 

Some of the newer machines POST so fast, you may not be able to get the Ctrl+Alt+Del in time....

1st Restart your machine....then hit CTRL-ALT-DEL...BEFORE the BLUE WINDOW....this causes the
machine to restart.

2nd Again hit CTRL-ALT-DEL... BEFORE the BLUE WINDOW.

3rd Let the machine boot this time....it will start at PREPARING AUTOMATIC REPAIR, Then DIAGNOSE,


And finally, ATTEMPTING REPAIRS. (about 10 minutes)

THEN the options will appear.

1) Select ... ADVANCED OPTIONS

2) This Window...Choose TROUBLESHOOT

3) Again select ADVANCED OPTIONS

4) Select ... WINDOWS STARTUP SETTINGS

5) Select ... RESTART

After Restart you will end up at the window F8 used to produce.
Total time on my machine...about 15 minutes...not fast, but no installation media necessary.

When you get to the SAFE MODE window, select Safe Mode with Networking.

lemp say:

You're trying to open the *.zip file with Word.  You first have to extract the compressed file and then open the extracted file with Word.


Method 1

Double-click the zip file.  This will show the contents of the zip file, which presumably are several *.doc or *.docx Word files.

Double-click whichever of the Word files you want to work on.

Method 1a

Rather than double-clicking on one of the compressed files, drag 'n' drop it to your Desktop.  This is in effect "extract one" rather than the "extract all" of Method 2. 

Method 2

Right-click on the zip file and select "Extract all."  This will create a new folder (with the same name as the zip file).  All of the files you had compressed into the zip file will be in this new folder.  These are copies of the compressed (zipped) files.

Double-click on whichever of the files in the new folder you want to work on.

The problem with all of these methods comes when you want to save your work. 

For Method 1, you should use "Save as" (rather than clicking the "Save" icon or using File > Save) and be sure to change the location (folder) where you're going to save the document from the temporary directory that will be the default to someplace more convenient, such as your Desktop.

For Method 1a, you can use the regular "save" function.  The file will be saved wherever you had dragged it (e.g., your Desktop).

For Method 2, you can use the regular "Save" function, but be aware that the file will be saved in the new folder that was created when you extracted all of the files.

In all of the above methods, the version of the file in your original zip file will not be changed.  Assuming that's not the outcome you want, after you save the modified file, drag 'n' drop it onto the zip file.  You'll get a window offering to "Copy and replace" or "Don't copy."  Choose "Copy and replace."  Your modified file will be compressed and placed into the zip file in place of the original compressed file.

You can then delete the saved modified file (Method 1 or 1a) or the entire new folder with all of the extracted files (Method 2).
