studentny say:

How to delete an existing image in a cell?

Hello.

I have a picture in cell "A11" and I want now to deleted that picture.

tasos_k say:

Hi,

try this...

 

Sub Delete_One_Picture()
Dim s As Shape, r As Range
Set r = [A11]
On Error Resume Next
For Each s In ActiveSheet.Shapes
If Not Intersect(s.TopLeftCell, r) Is Nothing Then
If s.Type = 13 Then s.Delete
End If
Next
On Error GoTo 0
End Sub

 

Note

Check if the top left of the image is in cell A11.

No matter if the image is located in one or more cells.

stefan blom say:

You have to apply your color theme to the template for it to be available in new documents based on the template. Note that the color theme is saved as a separate (XML) file; it isn't really stored in the template.

elsieb66 say:

It is saved in the template.  I don't see the point of setting up a colour palette if it's not available to use in fonts, shading etc!

elsieb66 say:

Yes, I have set up the colour palette and theme.  It does not show the colours any of the colour options - font/shading, borders, etc.

stefan blom say:

Custom colors applied to content and/or to styles in the template will be present for new documents created from the template.

However, the particular Theme Color set will only be available and selectable from Design | Colors if the corresponding xml file is available in the user's user profile (look in the C:\Users\YourUserNameHere\AppData\Roaming\Microsoft\Templates\Document Themes\Theme Colors folder).
