HPGL format picture insert - Microsoft Community

chriscarlson_1143 say:

HPGL format picture insert

Is there any hope that Microsoft will restore the ability to import HPGL files onto office documents?  I know it is no longer mainstream, but I see that I am not alone with this need.  A recent web search revealed that scientists at the LHC at CERN are struggling with HPGL imports too.  It can't be too difficult to add as on Office feature, and it would be a feature not currently found in Open Office.  The format has been around forever, and the instruction set is short and simple.  The fact is that HPGL has a number of unique advantages over other graphic formats.  

  • The HPGL files themselves are very small. A typical graph with thousands of data points is typically only a few kb.  When imported into old MS Word documents, they added very little to the file size.  Hundreds of images could be added and the document could still be sent by email.  Since the files are ASCII text, they can be compressed dramatically too. 
  • The HPGL files can be edited with any simple text editor like Notepad.  The text is readable and easy to understand.
  • The image is in vector format, so it doesn't degrade quality when the size is changed.  This isn't a unique quality, but it is important for images that have technical content.
  • There are many people (old-timers?) in the technical community who are familiar with HPGL and appreciate it for its simplicity, efficiency and elegance.

I'll get off my soap box now.  If this could bed passed up the food chain at Microsoft  I would appreciate it, and I know a few other who would too.

There is an HPGL filter that can be downloaded from https://support.microsoft.com/en-us/kb/196506.

I can't tell whether it works properly in versions of Word later than 2000 or in later versions of Windows, because I don't have any HPGL files to try it with. However, after following the installation instructions, it at least didn't cause Word 2003 or 2013 to crash.

Two things to note about the installation: The filter is 32-bit code, so it won't work at all with 64-bit Word (but it should be ok with 32-bit Word on 64-bit Windows). vanessa ouyang msft say:

Hi RJL_web,

If you directly click "New Contact Group", it will expand individual addresses when you send a meeting invitation, because it doesn't have own email address.

To achieve your goal, I suggest that you create a group by click "New Group" and then add a Group ID for it.

In this way, the meeting invitation will show group name rather than individual addresses.
