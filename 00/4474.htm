rick rothstein, mvp - excel say:

This macro may do what you want...

Sub ConvertDatesInA()
  Dim X As Long, LastRow As Long
  Const StartRow As Long = 2
  LastRow = Cells(Rows.Count, "A").End(xlUp).Row
  For X = StartRow To LastRow
    Cells(X, "B").Value = Format(Cells(X, "A").Value, "0000-00-00")
  Next
End Sub

You can adjust the StartRow constant (in the Const statement) for the actual row number containing the first piece of data that needs to be converted (the code takes care of determining the last row of data in Column A).

stefan blom say:
Strange behavior connected to the creation of bulleted and numbered lists may be caused by a corrupt ListGal.dat file (that file stores the settings for the built-in list formats in Word). Do the following: Exit Word. In File Explorer, choose to display hidden files and folders and then search for ListGal.dat. Delete the file.
