nettlebane say:

It appears a clean boot did not help, but thank you for your help though, however I did happen by chance to run across the solution. What I had to do was simply edit the 'Interactive logon: Machine inactivity limit' field in Group Policy Editor to '0.' Strange, but it seems to have worked. I post this solution in hopes to help anyone else with the same problem, however unfortunately this will ONLY work in Windows 10 Pro or higher due to that the Group Policy Editor is not available in Windows 10 Home. 

Instructions:

1. Type 'group policy' into Windows search(from Start) and open the gear looking icon that comes up.

2. Use the navigation tree to the left of the screen: Computer Configuration>Windows Settings>Security Settings>Local Policies>Security Options>

3. Find the 'Interactive logon: Machine inactivity limit' field and double click to open it.

4. Insert '0' into the box on the 'Local Security Setting' tab within the 'Interactive logon: Machine inactivity limit' field.

5. Click 'Apply,' then click 'OK' and let your PC sit for a minute to test whether or not it fixed the problem.

I can't gaurantee that this will work for everyone, even if you do have access to the Group Policy Editor, but it's worth a shot if you are experiencing the same problem as me.

As far as I can recall, this behavior is expected and normal when the call is audio on the receiving end as you do not really need to see the person's avatar like you would need to see the person on a video-video call.  While it may not have been like this since Skype's inception, it has been like this for quite some time.

Generally when the dimensions of the window are smaller than that of the whole screen, it will center the advertisement in the middle or adjust it in some manner. If you increase the size of the chat/call window past a certain point, the advertisement will revert to the right side of the screen.  If the other party enables video, this will also remove it from the center. 

I actually did test this out using Skype 7.40 and could get the advertisement to move from the center to the right side by increasing the window size or maximizing the window.  Decreasing the window size moved it back to the center.

I can only come to the conclusions that your window has been proportionally resized smaller than it was in the past, your monitor has been set to a different resolution, you have generally had mostly video to video calls or that proportionality threshold has been adjusted.  One thing that does affect window size and proportionality is the view.  You can try toggling the view between Single/Split Window (with and without the docked contact list) and see if that changes things.

An upgrade - whether it's the Creators upgrade, Anniversary upgrade, Fall upgrade or any other upgrade - is essentially no different than upgrading from one version of Windows to another, and the same principles apply.

The upgrade will eventually cause problems and possibly fail entirely if any of the following are true:

  • the hardware is not compatible with Windows 10;
  • the software - whether pre-existing or added afterward - is not supported on Windows 10;
  • the pre-existing Windows configuration is corrupted; and/or
  • unwise configuration choices have been made since the upgrade.

And the solution is the same:

  • Check to be sure that the hardware will work properly on Windows 10.
  • Do a clean install.
  • Only add supported software.
  • Before making changes, read about what you want to do to learn the best way to do it.

I understand that the computer is unable to boot to the desktop after interrupting the installation of Windows updates.

I suggest you to perform the following steps and check if it helps.

How to create installation media for a PC refresh or reset

BIOS Disclaimer:

 

Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

 

Method 1:

 

Since you can be able to view command prompt window, I suggest you to run System restore from here and check.

 

Type rstrui.exe. and try to perform System Restore.

 

Method 2: Perform an Automatic repair using Windows 8 bootable DVD or USB drive and check if that helps.

 

Follow these steps to perform Automatic repair

 

a) Insert the installation DVD or USB and boot Windows 8 from it.
b) In the ‘Windows setup’ page select the ‘Language to install’, ‘Time and currency format’ and the ‘keyboard or input method’ and click on ‘next’.
c) Click on ‘Repair your computer’ and select ‘Troubleshoot’.
d) Click on ‘Advanced options’ and select ‘Automatic Repair’ and select the operating system.

 

Method 3:

 

If the issue persists even after performing automatic repair you may try to boot from the Windows 8 DVD and try performing a System Restore from Windows Recovery Environment (WinRE).

To perform System Restore on Windows 8 computer follow these steps:

a. Restart the Windows 8 computer, press & HOLD Shift+F8, until you see the option Choose an operating system.
b. When you see the screen Choose an operating system, click on Change defaults or choose other options.
c. Choose the option Troubleshoot and select Advanced options.
d. You may choose System Restore from Advanced boot option.
