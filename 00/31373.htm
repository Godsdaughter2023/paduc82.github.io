Windows 10 Goes to Sleep Too Early - Microsoft Community

smlbizmgr say:

Windows 10 Goes to Sleep Too Early

Why does Windows 10 go to sleep earlier than my settings say it should?S
rpgarrett say: Your solution worked for me. In my case it was a screen saver issue as well. I have 2 laptop computers running Windows 10 Pro; one is new with Windows 10 Pro pre-installed and the other is older where I did a fresh install of Windows 10 Pro. Both had “Blank” screen savers enabled with a “Wait” time to turn on the screen saver of 2 minutes. As you stated the “Blank” screen saver gave the false impression that the display was actually off when it wasn’t. All I had to do to fix the problem was increase the wait time to my preferred time of 30 minutes. I also changed the screen saver from “Blank” to one of the non-Blank choices. That solved the problem for me. No other changes were necessary. I would have thought that something like a screen saver would be disabled by default but that was not the case. As I write this my second computer has been idle for at least 15 min and the display is neither off nor has the screen saver kicked in. So I suspect that the setting that actually controls when the display is turned off actually does work as this setting has nothing to do with the screen saver timeout setting although both times could be set to the same value.

jim balcomb say: 1. Right click on any open area on the desktop.
2. A tab will open > goto Properties.
3. This will open 'Display Properties' there will be 5 tabs.. Theme, Desktop, Screen Saver, Appearance, Settings.
4. Make your changes to any or all tabs depending on your personal preferences. NOTE: Click "apply" on each tab after you make changes and before going to next tab.
5. When all done with your settings go back to the Theme tab goto SAVE AS and give or chose a theme name i.e.'my favorite theme'. Save this to the desktop.
6. Every time you reboot your setting will be lost, as per psych0fred post above. Before using IE or open any other windows double click on he theme file you created, it will take you to 'Display Properties', make sure your theme name is in the save as box > click apply>ok. NOTE: If you open IE before applying changes you wil need to close and reopen for changes take effect.

I also had to put my Microsoft password in to access the computer even though the computer had been in the sleep mode. I never had to type in the password prior to the upgrade. Now I need to do it each time I want to use the computer. I have 10.1.

spiritx say:
Hi,

Control Panel - Power Options - Require a password on wake-up - Change
settings that are currently unavailable - tick Don't require a password.

Start button - Settings - Personalization - Lock Screen - Scroll down to

Screen Saver Settings - click it - untick "On resume, display login of screen".

Start button - Settings - Accounts -Sign-in Options - Require Sign-in - set
to Never.

Hope this helps.

a. user say:

Hi,
 
Thank you for posting in Microsoft community. I  understand your inconvenience and will certainly assist you with the issue.
 
Follow the steps below and check if it helps:
 
1. Right click on start button.
2. Go to control panel.
3. Select power options.
4. Click the “Require a password on wakeup”.
5. Select “Change settings that are currently unavailable".
6. Choose "Don’t require a password” .
7. Save the changes.
