Sprocket       002             Company B

So on the report, all serial numbers for the sprockets at Company A need to be in red font.  All serial numbers for the sprockets in Company B need to be in blue font.  There are five companies, so I need to set it up for up to five colors.

Can anyone help?

boblarson say:

Since you are doing this in a report you can use VBA to set the font color based on the company name.

In the section where you are printing the SN

Private Sub Detail_Format(Cancel As Integer, FormatCount As Integer)
SELECT Case Me.Organization
   Case "Company A"
       Me.SN.ForeColor = vbRed
   Case "Company B"
       Me.SN.Forecolor = vbGreen
   ...

   Case Else
      Me.SN.ForeColor = vbBlack

END SELECT

You can use a number to define the color if you wish instead of the constants.  For instance vbYellow = 65535, but you could have a custom color.  16711680 is a Dark Blue.  I just set the color I wanted and then copied the number from the control's forecolor property.

Hello Routten,

Thank you for using the Microsoft Vista Answers Forum!

Let’s start troubleshooting by running Internet Explorer with no add-ons.
Even though you may not have installed any add-on but something may have installed itself as part of a malicious program.

To temporarily disable all add-ons

Click the Start button, click All Programs, click Accessories, click System Tools, and then click Internet Explorer (No Add-ons).

If your browser work fine after these steps, add-on may be responsible for the error, use the steps below to enable add-ons individually to determine which of the add-ons may be causing the problem.

1.      Open Internet Explorer.

2.      Click the Tools button, click Manage Add-ons, and then click Enable or Disable Add-ons.

3.      In the Show list, click Add-ons that have been used by Internet Explorer to display all add-ons.

4.      Click the add-on you want to enable, and then click Enable.

5.      Repeat step 4 for every add-on you want to enable. When you are finished, click OK.

If still not working after the above steps, you can reset IE to the default setting using Control Panel.  By resetting Internet Explorer to its default settings, you return it to the state it was in when it was first installed on your computer. This is useful for troubleshooting problems that might be caused by settings that were changed after installation.

To reset Internet Explorer settings

1.      Close any Internet Explorer or Windows Explorer windows that are currently open.

2.      Click  Start > Control Panel >  Internet Options

3.      Click the Advanced tab, and then click Reset.

4.      In the Reset Internet Explorer Settings dialog box, click Reset.

5.      When Internet Explorer finishes restoring the default settings, click Close, and then click OK.

6.      Close Internet Explorer.

Your changes will take effect the next time you open Internet Explorer.
