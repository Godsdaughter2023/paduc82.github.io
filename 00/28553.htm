shyam pillai say:

Hi,

You will not be able to edit a picture in normal view if it is placed on the slide master or custom layout. This is by design. If you wish to make it editable then you have to make multiple copies of the same image on all the slides.

Use this macro:

Sub DuplicationImageAcrossPresentation()
Dim shp As Shape
Dim sld As Slide

Set shp = ActiveWindow.Selection.ShapeRange(1)

shp.Copy

For Each sld In ActivePresentation.Slides
    If sld.SlideIndex <> shp.Parent.SlideIndex Then
        sld.Shapes.Paste
    End If
Next
End Sub

Select the image and then run the macro, it will duplicate the shape across the presentation.

philipradler say:

If you worked directly on the download file (never a good idea, by the way), it may be saved in some weird temporary Internet folder. One quick way to find it: Go to File>Recent and review the ensuing list. You should be able to identify the missing file (if you haven't worked on too many other Word files since). Just left-click on the file to open it, and immediately save it in some location on your computer where you would normally expect to find it next time you go looking (like in My Documents)!

 

Recommended practice with downloaded files and e-mail attachments is to save them to your computer's hard drive before working on them at all. That will (a) make the file easy to locate, and (b) avoid losing your work when temporary storage files are deleted upon program exit.

 

Hope this helps!

-Phil
