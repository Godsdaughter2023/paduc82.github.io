Gotta be a better way but this tested in 2003 and should work in 2007

Sub CopyOnlyFormatCondionColorSAS()
Dim c As Range
Dim x As Long
On Error Resume Next
For Each c In ActiveSheet.UsedRange. _
SpecialCells(xlCellTypeAllFormatConditions)
x = c.FormatConditions(1).Interior.ColorIndex
c.FormatConditions.Delete
c.Interior.ColorIndex = x
Next c
End Sub

doug robbins - mvp office apps & services (word) say:

Select a paragraph that has the Style that you want to modify assigned to it and then press CTRL+SHIFT+s.

The Apply Styles dialog will appear with the name of that Style in the Style Name control.  Click on the Modify button in that dialog to open the Modify Style dialog and then expand the Format drop down and then click on Paragraph to display the Paragraph dialog.  In that dialog, check the box for "Don't add space between paragraphs of the same style" and then click on the OK button.

Back in the Modify Style dialog, select the radio button for "New documents based on this template and then click on the OK button.

You can make this change in already created templates by opening those templates and follow the above procedure.

eli qian msft say:

Hi Feargal,

The banner of update appears when Microsoft AutoUpdate Tool detects that you have available release. According to our test previously, this should only appear when Office 2016 is three or more versions behind in updates. Manually Check is the option to avoid prompt as soon as there is any new release. As an Office 365 subscriber, we would suggest you keep Office in the latest version to ensure that you can get the latest experience and service.

To address your requirement, you won't see this message if you remove the Microsoft AutoUpdate application found in "/Library/Application Support/Microsoft/MAU2.0" folder. In other words, you could uninstall MAU application if it's necessary to disable this notification.

Hope this information helps.
