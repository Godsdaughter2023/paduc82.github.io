a. user say:

Hi,

Sorry, I now understand your question, here's how you fix this;

Because the cell with the link in it has the value of text and the actual hyperlink is hidden behind it, we need to extract the hyperlink and not just the text face value of the cell.

Unfortunately, Excel doesn't have a formula to do that, but we can make one.

Firstly, you need to save the workbook as an XLXM to be macro enabled, then go to file, options, trust center, trust center settings, macro settings, and then Enable all macros.

If you open up files from unknown users that you don't trust, you may want to choose "Disable all macros with notification" or alternatively, just digitally sign your macro's but that's another topic.

Open up your workbook, press ALT-F11 which opens the VBA editor, select insert and then module.

Within the module that has been created (click on it in the left window) and paste this text;

Function GetURL(rng As Range) As String
    On Error Resume Next
    GetURL = rng.Hyperlinks(1).Address
End Function

Now press close the VBA editor and then go back to the workbook.

If Sheet 1 had a hyperlink in cell A1 and you wanted cell A1 in Sheet 2 to reference that link, go to Sheet 2, A1 and type in: =GETURLSheet1!A1)

This will give you the link as text, to turn it into a hyperlink, enter this:

=HYPERLINK(geturl(Sheet1!A1),"Test")

This will show the word "Test" in the cell with the hyperlink.

Does this solve your problem?
