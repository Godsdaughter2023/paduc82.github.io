How to find out data source of a mail merge document - Microsoft Community

kathydu say:

How to find out data source of a mail merge document

How do I know the name of the data source and its location that sticked to an existing world document that has got the mail merge function.

macropod say:
Word doesn’t make it obvious what the Data Source for a Mailmerge is. You can locate it by answering ‘Yes’ to the mailmerge SQL prompt - but only if the data source can’t be found, in which case Word will tell you what datasource it's looking for. If you answer ‘No’, the document disconnects from the Data Source and becomes a normal document. For a mailmerge main document that has successfully connected to its data source (i.e. after answering ‘Yes’ to the mailmerge SQL prompt), go to Mailings>Edit Recipient List and hover over the file named in the 'Data Source' box. You should now see the full name & path of the data source.
