tom_bpc say:

IMI THERESA,

In addition to Dale's fine advice, I suspect the first order of business is to ensure that you are using a common resource pool.  If the resources have been created separately for each subproject, then MS Project will treat them as distinct (i.e. separate) resources, regardless of their shared name.  Thus, IMI LORENZO is treated as two distinct, local resources in the two subprojects you have shown, and no overallocation is detected.  The only way for MS Project to detect the overallocation (and to implement a leveling delay) is to use the same (Pool-resident) resource in both subprojects.

lorenaaguileracuevas say:

Restore Down button on word is not working

I used to be able to click in the Restore Button in word and the document would minimize to half the screen. That doesn't happen anymore, the document stays basically the same size. I am not able to get to the corners to make it smaller. How can I fix this?

jay freedman say:

It's certainly possible to have the "restore down" size be the same as the maximized size, making it difficult or impossible to grab an edge or corner with the mouse. When that happens, there's a keyboard-only method for resizing the window.

Press the Alt key and the spacebar together and release both. That opens a small menu in the top left corner. Press the S key to activate the Size command, and the cursor will change to a four-way arrow. Next, press one of the four arrow keys -- the left arrow moves the cursor to the left edge of the window, the up arrow moves to the top, and so on.

Now you can use either the arrow keys or the mouse to drag the selected edge toward the center of the screen. Press the Enter key to set the edge at its current location and exit the Size command. At this point you may be able to grab a corner, or you can use the Size command again to move another edge.

An alternative is to use the Move command on the same menu, so pressing the arrow keys moves the window down and sideways. Once a corner comes into view, you can use the mouse to resize the window.
