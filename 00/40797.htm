mathew mirabito say:

For other people having the same or a similar issue, it MAY be that you have an SD card inserted and that's where Windows put the update files for installation.

In that case, you can try: -

* Removing the SD card

* Inserting the SD card into a USB SD card reader

* Inserting that into a free USB port

* Waiting about 30 seconds, and trying again

If you don't have a USB SD card reader, you could also try copying the files off the SD card directly onto a USB key and inserting that before trying again.

What I believe has happened, is that the drivers are not loaded for the SD card reader, and therefore after the reboot it is unable to access the files required for updating - a USB SD card reader is plug-and-play, and gets you back up and running again.

Overview

If you see error code 80004005 when you try to access the Games Store or the Apps Store after you have signed in to Xbox LIVE, it means that you cannot access the Games Store or the Apps Store through your Xbox 360 console.

Read this page to learn more about error code 80004005 and what you can do about it.

On this page
Topics
Error code and message
Problem
Solutions
Error code and message

Error code 80004005 is preceded by the following message:

Can't retrieve information from Xbox LIVE. Please try again later.
Status code: 80004005

Problem

If you’re seeing this message and error code, it means that you cannot access the Games Store or the Apps Store through your Xbox 360 console.

Some Internet connection types and Internet Service Providers (ISPs) cannot provide access to the Games Store or the Apps Store.

Note We are aware of this issue and working with the affected ISPs to resolve the problem.

Solutions

To resolve this issue, try the following solutions:

Solution 1: Use the Xbox LIVE Marketplace on Xbox.com

You can purchase items from Xbox.com and add them to your console’s download queue. Here’s how:

  1. Visit Xbox.com.
  2. In the upper right-hand corner, click Sign In.
  3. Type the email address and password that you used when you signed up for Xbox LIVE.
    Note This is your Microsoft account.
  4. Click Games, and then browse the games, add-ons or avatar items to find something that you want to download.
  5. Click Download to Xbox 360.

The next time you turn on your console, the content will start downloading on your console.

Solution 2: Use a different Internet connection

Connect your Xbox 360 console to the Internet at another location that uses a different ISP. For example, take the console to a friend’s house and connect to the Internet there.
