helpster say: 'Cache' most notably is like Pirate treasure...a hidden supply. :-)

In Windows Update terms, it is a place where files are stored for later use. Oh...these files are set with a 'Hidden' attribute so they are not seen by most users.
Windows Update uses a cache for a few reasons.
One is to allow for a rollback to a previous version. Basically, uninstall the latest update and reapply the older files.
A second reason is so the the file are present on the system so newly installed features can get the update automatically applied and/or a repair can be run to re-apply the files should something happen.

Here is a quick senario to help understand what could be happening.

You download an update for Windows. It is downloaded to a 'cache'. We'll say the update is 100MB for this senario. 
It then Extracts the compressed files to say 150MB, to a temporary location for installation purposes.
Then integrates the update into Windows, adding the new/updated files...another 150MB.
Then save/cache the older files in case we need to uninstall the new update. We won't add this ammount...but we are not deleting it either...as you might assume. 
It then deletes the temporary folder. At least we're not keeping the download AND the extracted files! :-)

So one 100MB update is actually 250MB installed. The 100MB download and 150MB of installed files.

You can see that if you download a 500MB update and it extracted to 650MB, you would now have added 1.15GB to your hard drive.

Mind you that all updates don't work exactly like this, but most are similar. Many older updates also get deleted after they have been superceded by certain updates. Also, older cached updates will be deleted as the 15% rule kicks in. There is a rule that only 15% of the drive space can be used to store 'Cached' files for Windows Update.

So here is the reality of the issue. If you have a 100GB C partition, 15GB(15%) can be used by cached updates with default settings, PLUS growth from the updates.
So if Windows installed using 20GB, you then installed 100 updates...unfortuantely all too easy to do, your size could easily grow to 30GB. Then add in the 15% cache and you have 45GB used up....YES over double its original size!
Good thing hard drives are so large these days!!!

All that said, you very likely are just seeing normal growth. And it will only continue to grow. Not only from Updates...both Microsoft and other companies, but installed programs and saved files.

It is a good practice to keep your computers file system as clean as possible. Only install programs that you 'really' need...or will use. Delete files, email, etc. that you no longer need, and uninstall older programs that you no longer use. And importantly, back up files to CD, DVD, or an external drive.
