Hello,

You will need to make an edit in the registry.

Note:

Incorrectly editing the registry may severely damage your system. At the very least, you should back up any valued data on the computer before making changes to the registry.

  1. From the start screen (or search in the start menu if you're using Windows 7) type run.
  2. Select Run from the search results.
  3. In the Run command box, type regedit. This will display the Registry Editor.
  4. Navigate to the HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\16.0\User Settings\OneNoteToIEAddin registry key
  5. Click on Count
  6. Change the Count data from 1 to 0 and click OK.
  7. Exit the Registry Editor and reboot the system.

Why is Microsoft discontinuing MSN apps? - Microsoft Community

randomalianfromvietnam say:

Why is Microsoft discontinuing MSN apps?

Let me break into details the reasons why Microsoft shouldn't discontinue these applications, I know that Microsoft will continue to discontinue MSN Health & Fitness, MSN Travel, and MSN Food & Drink regardless of how Microsoft's users and costumers will react, but from a consumer's stand-point this move makes little to no sense, let me break it down to why this doesn't fit.

Is this the end of an era?

Historically Microsoft has almost never discontinued anything, usually they merge services (Windows Live Mesh and Windows Live SkyDrive is a good example of this), or rebrand them (Zune -> Xbox Music -> Groove Music), or split them (Zune -> Xbox Video, Xbox Music, Windows Phone App for Desktop), so far this has always been the order of things at Microsoft, in fact this is the prime reason I choose Microsoft as my primary software (and even hardware) provider over Google, with Google you never know if your favourite service will exist tomorrow, in fact they pretty much delete half of their products every year with their ''spring cleaning'', and as for ''the other alternatives'' AOL and Yahoo! these are pretty much fully powered by Microsoft except for AIM, Yahoo! Messenger, AOL Mail, and Yahoo! Mail but these services can hardly compare with Hotmail.

So the recent decision to discontinue these applications shock me.

Is low app usage really an excuse?

What do I mean with this? Well on Windows Phone MSN Health & Fitness is the #1 downloaded fitness application, and I personally use every day because the Microsoft Health application simply lacks many features that MSN Health & Fitness has, personally I think that part of the reason that Microsoft decides to terminate this service is because it's the only Microsoft-made Fitness application for the current platforms that isn't powered by Microsoft HealthVault, which is Microsoft's own fault as the old Bing Health & Fitness application was.

How about the others? MSN Food & Drink was universally praised upon its arival for its ''hands-free'' functionality, many newsppapers named it a truly innovate feature, maybe the reason why MSN Food & Drink, MSN Health & Fitness, and MSN Travel are less downloaded than the other MSN applications is because they're more recent, not all Windows 8 PCs have updated to Windows 8.1 yet, in fact 1/3rd due to a variety of issues, the one I most commonly found was that most people's internet wouldn't allow them to download such a large file from the Windows Store, MSN Travel grew mostly out of Microsoft's acquisition of Farecast, and though MSN Food & Drink did absorb the MSN Healthy Living site with the rebranding, it was still mostly software developed by the Bing AppEx Team.

Now compare that with MSN Money, MSN News, MSN Sports, and MSN Weather, all of these have been around since 1995, they have grown their own dedicated users over the years, but MSN Travel is from 2012 and is limited only to Windows platforms, and MSN Food & Drink and MSN Health & Fitness are both from 2013 and on mobile since 2014 so a low number of users compared to the other MSN apps is simply not an excuse, in fact I know a lot of developers who would kill for ''low'' numbers like that.

Is this move shortsighted?

Right now Bing has had a Travel Search Engine, and a Health Search Engine years before Google had, and Google only started to implement these features very recently, though there used to be Google Health but they gave all their data to the Microsoft HealthVault so Microsoft has a double advantage in health over Google, and a part of the reason that Apple and Google were suddenly writing their own fitness applications is because Microsoft pre-installed their own fitness app on their Windows tablets, now imagine if Android will come pre-loaded with Google Travel in the near future, then Microsoft will have to completely start over again from the ground up to make a new travel application despite MSN Travel being a beautiful well-designed service that can take on any make-shift Google alternative.

Windows Phone needs these full-featured applications

Here's something Microsoft forgot about, Windows Phone has a Shopping List app, an official Microsoft application that hasn't been updated for years, akin to several others like these, all of them but 1 are going to be bundled with Windows 10 Mobile, Unit Converter is going to be in the Calculator, World Clock, Nokia Timer, Nokia Stopwatch, and a few others are going to be into the Windows Alarms & Clock application, but the Shopping List is going to be orphaned (along side the Level app), the most commonly requested feature on that app is the ability to sync it or to view it online, but Microsoft has never marketed the MSN Food & Drink app as an online shopping list so many potential users will be stuck thinking that Microsoft never listened.

Furthermore Cortana relies heavily on them, Cortana needs MSN Travel if you want to track your flight (something I do multiple times a year), Cortana needs MSN Health & Fitness for health information, and Cortana needs MSN Food & Drink for recipes, often when I want to search recipes I use Cortana, if Microsoft isn't going to implement these features in Cortana natively then we'll have an assistant on our hands that has lowered down to the level of Siri, and now she offers a powerful alternative to Google Now thanks to the MSN, but soon that advantage might be gone as well.

So if all these odds are playing in favour of Microsoft, then why is Microsoft going to discontinue these services?

doug robbins - mvp office apps & services (word) say:

Use a Drop-Down List Content controls that contain the values Red, Amber and Green and assign the word Status as the Title of each control

Then in the ThisDocument object in the Visual Basic Editor, insert the following code:

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
If ContentControl.Title = "Status" Then
    With ContentControl.Range
        Select Case .Text
            Case "Red"
                .Cells(1).Shading.BackgroundPatternColor = wdColorRed
            Case "Amber"
                .Cells(1).Shading.BackgroundPatternColor = wdColorOrange
            Case "Green"
                .Cells(1).Shading.BackgroundPatternColor = wdColorGreen
            Case Else
                .Cells(1).Shading.BackgroundPatternColor = wdColorAutomatic
        End Select
    End With
End If
End Sub

You should save the file as a Macro Enabled Template (.dotm) and the users should use File>New and select that template when they need to use it.
