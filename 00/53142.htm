Get More Details On the Error

 

To decode 0x80070002. The 0x means it's a hexadecimal number, the 8 means error, the first 7 means it a windows error, and the rest of the number, 2, is the actual Windows error.
 
To look up the error we need it in decimal format. Start Calculator (Start - All Programs - Accessories - Calculator) and choose View menu - Scientific, then View menu - Hex. Enter 2. Then View menu -  Decimal. It will say 2.
 
Start a Command Prompt (Start - All Programs - Accessories - Command Prompt) and type
 
net helpmsg 2
 
and it will say
 
The system cannot find the file specified.
 
Use Notepad to search for FATAL and and the specific error code (eg, 80070002) in C:\Windows\Windowsupdate.log.

Also check cbs.log. You must follow these steps. You must check it at the beginning and the end of troubleshooting. To make life easy, empty the file first, do the update, read the file. Else there will be tens of thousands of lines to read.

 

 

Look for lines saying error or failed.

http://support.microsoft.com/mats/windows_update/en-us

http://support.microsoft.com/fixit/en-us

del %SYSTEMROOT%\Logs\CBS\CBS.log
