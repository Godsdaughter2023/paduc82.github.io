If there are no previous versions of the  file  available, you can restore a previous version of the  folder  where the file was stored to get a previous version of the file. 

1. Open  Windows Explorer  and navigate to the folder where the file was stored.

3. After the previous versions list populates,  select  a Previous Version of the folder that has a Date and Time stamp from before you edited the file.  Click the  Copy Button.

 

4. In the  Copy Dialog  that pops up, select the Desktop  folder. Click the  Copy Button. This will copy the older version of the folder to the Desktop.

 

5. Go to your Desktop and open the copied folder. Look for the deleted file.  Move or Copy the file to a new location.

Delete the previous version of the folder from the Desktop.
