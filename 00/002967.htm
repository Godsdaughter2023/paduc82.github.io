You cannot add an Outlook.com account or a Hotmail account using the automated process. You must create the account manually by entering the imap-mail.outlook.com and the smtp-mail.outlook.com data and then specifying the IMAP 993 incoming server with SSL and the SMTP 587 outgoing server with TLS.

Doing anything else locks up your Outlook in the "Loading Profile" message for hours and hours. The only solution then is to hold CTRL before starting Outlook, starting in Safe mode, creating a new Profile, adding all your email addresses one at a time (making SURE not to auto-add the Hotmail address), exiting, removing the nonworking Outlook profile through Regedit thus making the newly created one the only profile, then running Outlook to get all your emails back.

WORD 2013 Instructional Text/Notes that is not printed - Microsoft Community

metalray_ say:

WORD 2013 Instructional Text/Notes that is not printed

Hi Word Experts,

Is there a way to add instructional notes or text in any form to a WORD document that later on does not get printed?

Like you can in MS Excel define a printable area and hence exclude instructions or headings.

Thanks

Bob

jay freedman say:

Word doesn't make this easy.

One method is to mark the instruction text as Hidden (this is a check box in the Font dialog, or you can use the keyboard shortcut Ctrl+Shift+H; preferably, define a style that includes Hidden format as one of its settings). In the Options dialog, you can enable display of Hidden text on the screen, and disable printing of Hidden text.

The drawback is that those options are not saved with the document, so each user will see or not see the Hidden text according to how the options are set on their computer. It's possible to write macros to set the options when the user opens the document and change them back to the user's preference when the document closes. However, there's no way to force the user's computer to allow macros to run, if their Trust Center options disable macros.

Other methods are even more complicated and less reliable. You can try to use the ScreenTip (mouseover text) of hyperlinks, but that's limited to about 250 characters. You can put the instruction text in comments, and (either manually or with a macro) uncheck the "Print markup" setting.
