Keyloggers are a form of malware. Defender protects against them.

However, no antivirus solution, Defender included, will protect against everything.

The following recommendations will assist in protecting the PC from infection:

-Make sure that the Windows Firewall is enabled.

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

-Make sure Internet Explorer is updated with all patches.

-In Internet Explorer use the SmartScreen Filter.

-Make sure that IE Internet Security settings are at least set to medium-high (default).

-Enable the pop-up blocker in IE.

-On Vista and later make sure that User Account Control (UAC) ON  and not running with elevated privileges.

-Make sure that Windows Automatic Updates are set to at least notify, but the preferred setting is to download and install automatically. If you update manually, be sure to update as soon as possible after being notified of available updates.

-Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.

-Never click through links from unknown sources and use caution even if they are from a "trusted" source.

-Never open unsolicited email attachments.

-Practice safe web browsing.

Read from word and write to Access - Microsoft Community

anfy2010 say:

Read from word and write to Access

I have word documents that I want to read all the paragraphs that have the style AIO and send its data to a field in access database

any help with that please ?

 

hansv mvp say:

Select Tools | References... in the Visual Basic Editor.

Scroll down until you see Microsoft DAO 3.6 Object Library.

Tick its check box, then click OK.

Copy the following code into a code module:

Sub SendToAccess()
  Dim dbs As DAO.Database
  Dim rst As DAO.Recordset
  Dim par As Paragraph
  Dim strText As String
  Set dbs = DBEngine.OpenDatabase("C:\Databases\MyDatabase.mdb")
  Set rst = dbs.OpenRecordset("MyTable", dbOpenDynaset)
  For Each par In ActiveDocument.Paragraphs
    If par.Style = ActiveDocument.Styles("AIO") Then
      strText = par.Range.Text
      strText = Left(strText, Len(strText) - 1)
      rst.AddNew
      rst.Fields("MyField") = strText
      rst.Update
    End If
  Next par
  rst.Close
  Set rst = Nothing
  dbs.Close
  Set dbs = Nothing
End Sub

Modify the path and name of the database, the name of the table and the name of the field (which should probably be a memo field) to those you use.

(Make a backup copy of the database before testing the code)

Function GetParaText(docName As String, paraStyle As String)
Dim para As Paragraph
Dim paraText As String
Dim checkDoc As Document
    Set checkDoc = Documents.Open(FileName:=docName, Visible:=False)
    For Each para In checkDoc.Paragraphs
        If para.Style = paraStyle Then
            paraText = paraText & para.Range.Text
        End If
    Next para
    checkDoc.Close False
    GetParaText = paraText
End Function

julie sheets say:

Hello Peter,

The Milestone Report can be customized through the customization of the table.

  1. Create a new table through View > Tables > More Tables.  Select the Entry table, copy and rename.
  2. Apply the new table to your current view and add or delete fields as needed.
  3. Then customize the Milestone Report -- Report > Reports, select the Milestone report and click the Edit button.
  4. On the Definitions tab, select the custom table you just created.
  5. You'll need to adjust the column widths through trial and error in the view and then check report to see if you have the information you need.

To Export to Excel.

  1. File > Save AS -- select Excel.
  2. In the Export Wizard, select "Selected Data"
  3. On the Map page, select "New Map".
  4. On the Options, select "Tasks"
  5. On the Task Mapping Page, select "Milestones" as the Export Filter
  6. Click the "Base on Table" button and select the custom table you created.
  7. Save the map and export to Excel.

I hope this helps.
