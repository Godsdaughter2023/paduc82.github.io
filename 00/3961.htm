Is there a way to horizontally center radio buttons in Excel 2010?

I'm creating an excel sheet that I use to keep track of progress made on my projects.  I'd like to use radio buttons and have them centered on the cell.  I currently have the size of the radio button (Form Control) to snap to the size of the cell.  Is there a way, either within Excel or by adding VBA code, to center the radio button so it's in the middle of the control rather than on the left hand side by default?
hansv mvp say:

Try this - you may have to experiment with the value of the Fudge constant.

 

Sub CenterOptionButtons()
    Const Fudge = 7
    Dim opt
    Dim rng As Range
    For Each opt In ActiveSheet.OptionButtons
        opt.Characters.Text = ""
        Set rng = opt.TopLeftCell
        opt.Top = rng.Top
        opt.Height = rng.Height
        opt.Left = rng.Left + rng.Width / 2 - Fudge
        opt.Width = rng.Width / 2 + Fudge
    Next opt
End Sub

Word 2010: How do I remove “content control” status - Microsoft Community

paty 81 say:

Word 2010: How do I remove “content control” status from text?

I have some text that is "content control" and some that is not, and I cannot integrate the 2 together.  How do I remove the "content control" status? I I have some text that is "content control" and some that is not, and I cannot integrate the 2 together.  How do I remove the "content control" status? I am merging several documents together for my thesis. Each document has its own bibliography.  I used the references tab to create it for each section. I copied and pasted each bibliography together in one section for the entire paper, but some of these are in content control and some aren’t…
yves dhondt say:

You can't have different bibliographies in one document, not even when they are in separate sections. A bibliography in Word is nothing more than a field code and will therefore always return the same result (with the exception of language tag) no matter where you place it in a document.

You have two options:

  1. In each of your documents, convert the bibliography to static text. This can be done by selecting the blue arrow next to the content control and choosing the appropriate option. You can then copy each of the bibliographies into your final text. If you want to sort the result in your final text, you can select all the paragraphs containing the combined bibliographies and use the sort function.
  2. Copy all source materials to your final document. This can be done by moving all sources to a separate master list and then importing all sources from that master list into your final document. Then, in your final document, just add a new bibliography.

Note that the second solution will also ensure that your in-text citations, which are also content controls if you haven't converted them to static text, will not display something like "Invalid Source Specified" if fields get updated.
