A sheet is one side of a piece of paper, as printed.

A page is a part of a document, as defined by the page setup of the document.

rich007a say:

An alternative, similar to HansV's.

In cell A1, put the day of the week you want, either M, TU, W, TH, F, SA or SU

In B1 put the start date

In C1 put the end date

In D1 (or anywhere else), put:

=IF(COUNT(B1:C1)<>2,"",SUMPRODUCT(--(WEEKDAY(B1-1+ROW(INDIRECT("1:"&C1-B1+1)),2)=MATCH(A1,{"M","TU","W","TH","F","SA","SU",0},0))))

(I got this from here: http://www.excelforum.com/excel-worksheet-functions/443685-calculating-number-of-days-e-g-mondays-between-two-dates.html, and I can't explain it, but it does appear to work.  Note I had to tweak it a bit, I added the +1 because it didn't work properly if the last day was the same day of the week as the number of days you were counting.  I guess that depends on your definition of "days between" vs "days in month", I would think "days in month" is INCLUSIVE of the end date.)

Cheers
Rich

PS. note, that's one single formula above, the line break just happened to ocurr before the "=" that is half way through the formula.

Conditionally hide columns

I am using a spreadsheet for a training matrix for a construction site. There are hundreds of columns, listing all possible machine types that could be sent to site, however only a small proportion of those columns will have data in them as not all possible machines would be sent to site. Normally the columns for machines that are not on site are hidden manually. I want to know if it is possible to unhide the columns and then rehide only the columns that do not contain cells with data.

jllatham say:

It is possible using macros, not with formulas or conditional formatting.

 

Here's the easy one: unhiding all columns:

 

Sub UnhideAllColumns()
'
' UnhideAllColumns Macro
' Select the sheet to unhide columns on and then
' execute (Run) this macro.  Sheet must be unprotected
' at the time the macro is run.
'
    Rows("1:1").EntireColumn.Hidden = False

End Sub

Now here's the reverse operation, hide unused columns - see comments for limitations and restrictions. 

 

To put these into a workbook for use, copy them both into a Regular Code Module as instructed on this web page: http://www.contextures.com/xlvba01.html#videoreg

And save the workbook as a "macro enabled" workbook, type .xlsm (or .xlsb).

 

Sub HideUnusedColumns()
'
'HideUnusedColumns Macro
'Select the sheet to hide unused columns on and then
'execute (Run) this macro. Sheet must be unprotected
'at the time the macro is run.  Initially all columns
'are unhidden, and then columns that only have 1 (or no)
'cells with entries are hidden: this allows you to leave
'labels in row 1, but still hide if otherwise unused.
'
'Functions against all columns from A to the last column
'with an entry (as a label) in row 1.  Ignores all beyond
'that point.
  Dim lastCol As Long
  Dim colPtr As Long
  Dim colLtr As String
 
  lastCol = Cells(1, Columns.Count).End(xlToLeft).Column
  If lastCol = 1 Then
    'nothing to do, no labels or only a label in column A
    Exit Sub
  End If
  'this improves speed
  Application.ScreenUpdating = False
  'begin by unhiding all columns
  Rows("1:1").EntireColumn.Hidden = False
  For colPtr = 1 To lastCol
    colLtr = Left(Cells(1, colPtr).Address, InStr(2, Cells(1, colPtr).Address, "$") - 1)
    If Application.WorksheetFunction.CountA(Range(colLtr & ":" & colLtr)) < 2 Then
      Cells(1, colPtr).EntireColumn.Hidden = True
    End If
  Next
End Sub
