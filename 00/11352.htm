Application.FollowHyperlink "C:\Drawing\"& Forms!Form1.txtPartNumber & ".xls"

don varnau say:

Since you have the old Favorites folder, you cannot import those Favorites into the new computer, but you can copy them to the new Favorites folder then import them from IE into other browsers.

Access the new Favorites folder at "C:\Users\your account\Favorites"

Open the old Favorites folder.

It may be tempting to simply replace the new Favorites folder with the old, but that might cause "permission" problems. It might work; it might not. It's safer to copy Favorites from the old folder to the new.

After you have the new Favorites folder filled and working, import Favorites into the other browsers.
