If you are going to use VBA, why not add a permanent hyperlink to the symbol just typed in addition to launching the web page?

Right-click your worksheet's name tab and choose View Code. When the VBE opens, paste the following into the pane titled something like Book1 - Sheet1 (Code),

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Me.Columns(2)) Is Nothing Then
        On Error GoTo FallThrough
        Application.EnableEvents = False
        Dim b As Range
        For Each b In Intersect(Target, Me.Columns(2))
            If Len(Trim(b.Value)) = 5 Then
                b.Hyperlinks.Delete
                b.Parent.Hyperlinks.Add Anchor:=b, _
                  Address:="http://finance.yahoo.com/q/hl?s=" & Trim(b.Value), _
                  ScreenTip:="Yahoo stock page for " & UCase(Trim(b.Value)), _
                  TextToDisplay:=UCase(Trim(b.Value))
                b.Hyperlinks(1).Follow NewWindow:=False, AddHistory:=True
                b.Font.Underline = False
            End If
        Next b
    End If
FallThrough:
    Application.EnableEvents = True
End Sub

I never liked underlined hyperlinks. Blue is sufficient. If you want the hyperlinks to be underlined, remove the b.Font.Underline = False line. Tap Alt+Q to return to your worksheet. Any five alphanumeric characters typed into column B will create a hyperlink and launch the Yahoo stock ticker site to the appropriate web page.

suzanne s. barnhill say:
What you are experiencing is by design. A StyleRef field always references the first instance of the referenced style on the page in question (unless you use the \l switch, in which case it will reference the last instance on the page). Only if the style does not appear on the page will Word continue the last previous instance (on a previous page). That's just the way it works.
