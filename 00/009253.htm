andreas killer say:
Doesn't matter, the Autofilter knows where the headings are. .-)

Study the comments in the code below, customize the sheet names you want to refer to and the filter setting. The rest of the code should work without changes.

Andreas. 

Option Explicit
Sub Main()
  Dim WsSource As Worksheet, WsDest As Worksheet
  Dim Source As Range, Dest As Range
    
  'Refer to the source sheet
  Set WsSource = Worksheets(1)
  'Refer to the destination sheet
  Set WsDest = Worksheets("Whatever")
  
  'Do we have an Autofilter?
  If Not WsSource.AutoFilterMode Then
    MsgBox "Please enable the Autofilter"
    Exit Sub
  End If
  'Refer to the Autofilter
  With WsSource.AutoFilter
    'Remove any exsiting filters
    .ShowAllData
    'Filter first column > 50
    .Range.AutoFilter 1, ">50"
  End With
  
  'Get the filtered data
  Set Source = GetAutoFilterRange(WsSource)
  'Do we have any?
  If Source Is Nothing Then
    MsgBox "No data match the criteria"
    Exit Sub
  End If
  'We need columns B:G only
  Set Source = Intersect(Source, WsSource.Range("B:G"))
  'Just to be sure
  If Source Is Nothing Then
    MsgBox "Uuups, there is something wrong with the Autofilter, no data in B:G"
    Exit Sub
  End If
  
  'Find the next available cell in the destination sheet
  With WsDest
    Set Dest = .Range("A" & Rows.Count).End(xlUp).Offset(1)
  End With
  'Copy the data
  Source.Copy Dest
End Sub
Private Function GetAutoFilterRange(Optional ByVal Parent As Object, _
    Optional WithoutHeader As Boolean = True) As Range
  'Returns the visible range of an Autofilter, Excel 2010 and above
  Dim R As Range
  If Parent Is Nothing Then
    Set Parent = ActiveSheet
    If Parent Is Nothing Then Exit Function
  End If
  'No filter, return nothing
  If TypeOf Parent Is Worksheet Then
    If Not Parent.AutoFilterMode Then Exit Function
  ElseIf TypeOf Parent Is ListObject Then
    If Parent.AutoFilter Is Nothing Then Exit Function
  Else
    Err.Raise 438, "GetAutoFilterRange", "Object " & TypeName(Parent) & " not supported"
  End If
  
  With Parent.AutoFilter
    'Get the whole range
    Set R = .Range
    'Remove headings?
    If WithoutHeader Then
      If R.Rows.Count = 1 Then Exit Function
      Set R = R.Resize(R.Rows.Count - 1).Offset(1)
    End If
    'Filter active?
    If .FilterMode Then
      'Error's off, we get an error if no cells are visible
      On Error GoTo ExitPoint
      Set R = R.SpecialCells(xlCellTypeVisible)
    End If
  End With
  'Return the result
  Set GetAutoFilterRange = R
ExitPoint:
End Function

scottgem say:
That is simple enough. But, rather than printing the "order" groupfooter at the end of the previous section, I would like to print it at the bottom of the page (same location all of the time). I'm not sure this is possible, unless there is away to place it in the page footer section.

Oddly enough, this is being discussed right now with the Access Development team.

One solution offered was to create a dummy text box in the footer and size it on the fly to push the bottom of the group header.

This was contributed by Brent Spaulding (another MVP and my co-author)

Add a text box control (named "txtSpacer") to the very right of your Report Footer section with a control source of ="" and a height that spans the height of the design height of the Report Footer section.  Set txtSpacer's CanGrow = Yes, CanShrink = No, and Visible = Yes.  The Report Footer should be set to CanGrow = Yes.  Then in the OnFormat of the PageFooter section I set a module variable to the Top of that section.  Then in the OnFormat of the Report Footer section, I get the bottom of the section by adding the sections' Top and Height properties.  I then set the height of Report Footer section by adjusting the txtSpacer control, then I move all the other controls as if they are anchored to the bottom of the Report Footer section.

Here is some code that does I have just laid out.  This is code behind a Report object.

 ----------

Option Compare Database
Option Explicit

Private lngPageHeaderTop As Long

Private Sub PageFooter_Format(Cancel As Integer, FormatCount As Integer)
    lngPageHeaderTop = Me.Top
End Sub

Private Sub ReportFooter_Format(Cancel As Integer, FormatCount As Integer)
   
    Dim lngReportFooterBottom As Long
   
    lngReportFooterBottom = Me.Top + Me.Height
   
    If lngPageHeaderTop > lngReportFooterBottom Then
       
        'Calc the adjustment needed to place the Report Footer on top of
        'the Page Footer.
        Dim lngAdjust As Long
        lngAdjust = (lngPageHeaderTop - lngReportFooterBottom)
       
        'Adjust the height of the Report Footer section
        Me.txtSpacer.Height = lngAdjust + Me.txtSpacer.Height
       
        'Move all the controls down to accomodate the change in height (ie: Anchor to the bottom)
        Me.txtRptFooterControl.Top = Me.txtRptFooterControl.Top + lngAdjust
   
    End If
       
End Sub

While this is designed for the Report footer, the code should be modifiable to use in a Group footer.

Hi Yahalias,

Generally, when we use Vlookup, it only retrieves the value of a cell, it does not retrieve the hyperlink of the cell.

As a workaround, here are a few suggestions, you can check them out to see whether your requirement can be achieved:

How do i convert a VLOOKUP function result as a link to that result

VLOOKUP and hyperlinks problem.

Let us try using VBA code mixed with hyperlink, index, match and vlookup function to check if this can meet your requirement.

 

First, open your Excel file, press Alt + F11 to open VBA code editor.

Click Insert>Module, and paste the following code into the code area in the right side.

 

Function getlink(HyperlinkCell As Range)

    If HyperlinkCell.Hyperlinks.Count > 0 Then

        getlink = HyperlinkCell.Hyperlinks(1).Address

   Else

        getlink = ""

   End If

End Function

 

Next, assuming the data I want to look up from is in Sheet 2 as the following picture shows.


 

Now in Sheet 1 (Main Sheet), for example, I'd like to retrieve the value google along with the hyperlink from Sheet 2.

We need to input the following formula into a cell of Sheet 1 and press Enter.


The formula I input is as below, you can make some changes depending on your situation:

=HYPERLINK(getlink(INDEX(Sheet2!$B$1:$B$3,MATCH(Sheet2!A1,Sheet2!$A$1:$A$3,0))),VLOOKUP(Sheet2!A1,Sheet2!$A$1:$B$3,2,0))
