gerry c j cornell say:

To clear an event log by using Event Viewer
Start Event Viewer.
In the console tree, navigate to the event log you want to clear.
On the Action menu, click Clear Log.
You can either clear the event log or save a copy of the event log and then clear it.
To clear the event log without saving: Click Clear.
To clear the event log after saving: Click Save and Clear, type a name for the saved file in File name on the Save As dialog box and click Save.

Source: Event Viewer Help.

gerry c j cornell say:

OK let's start again and take the System log as an example. All events are recorded in the system.evtx file. The file is located at %SystemRoot%\System32\Winevt\Logs\System.evtx. This normally translates to c:\Windows\System32\Winevt\Logs\System.evtx which is a different location to the one you are looking at, namely c:\ ProgramData etc. I do not know the function of the files at the location where you have been looking.

The system.evtx file is a permanent file and should not be deleted. You can clear the contents in the way I have previously described. If you clear the contents you can reduce the file size. The default file size is overgenerous unless you want to keep a log of events long past the time they remain relevant. You cannot reduce the size when the log (file) is full.
