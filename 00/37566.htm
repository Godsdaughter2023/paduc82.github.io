kristine arp say:

Hi Jawnny,

Optimization option is a great way to help your PC stay at the top speeds, however it is only applicable to traditional rotating platters hard drive (Hard Disk Drive). Before we proceed to any troubleshooting steps, we would like to confirm if you have a Hard Disk Drive or a Solid-State drive (SSD).

If you have a Hard Disk Drive, you can follow these steps to start optimizing it:

1.  Type Defragment and Optimize Drives in Cortana Search.

2.  Press Enter.

3.  Select the hard drive you want to optimize.

4.  Click Optimize.

Take note that the optimization may take some time depending on the number of files, drive size and fragmentation.

If you have a Solid-State Drive, it is automatically being optimized by the TRIM feature. To check if TRIM is enabled, follow these steps:

1.  Press the Windows logo key + X.

2.  Select Command Prompt (Admin).

3.  Type fsutil behavior query DisableDeleteNotify.

4.  Press Enter.

If the command result is zero (0), TRIM is enabled. TRIM is disabled if the result is one (1). If you have to enable TRIM feature, follow the steps below:

1.  Press the Windows logo key + X.

2.  Select Command Prompt (Admin).

3.  Type fsutil behavior query DisableDeleteNotify 0.

4.  Press Enter.

Don’t hesitate to reach out if you have other questions.

Proxy server can be found in the Internet option in Control Panel. You can follow these steps to check it:
