davem121 say:
Hi Claudette

This is caused by a recent Windows 10 update

That update deleted all files in the Downloads folder and also added the Downloads folder to the Disk Cleanup routine by default

Honestly, the Downloads folder is not the place to store files, it is meant to be a temporary download location and any files downloaded that need to be retained, should be then moved to another folder for more permanent storage . . .

This is not an excuse - that update should not have deleted the contents of your downloads folder, it seems to be a bug in that update, because a lot of users are reporting the exact same thing . . .

Hopefully Microsoft are looking into resolving this bug with a future patch

For the moment, try the free version of Recuva to see if it can recover any important files that were deleted from the downloads folder . . .

If you have office try following steps:

1. Start Word and click the “Insert” tab.

3. Scroll through the “Object type” menu to the type of file the scanned document was saved in, such as a PDF or image file format.

4. Double-click the type of file. Note that these only list file types, not your actual scanned document.

5. Browse through the window that opens and locate your scanned document. When you are returned to the Object window, click the “OK” button to insert the document.

6. Click the cursor after the end of the scanned document or in the middle of the pages if you prefer to add the second document in between. Press “Enter” to add space for the second document or “Ctrl” and “Enter” together for a page break.

7. Repeat the process to add the second scanned document into the Word document.

8. Click the “File” tab, then click “Save As” and type a name for the combined document into the “File name” text box. Click the “Save” button.
