zenilogix say:

Solved my own problem.

Found a very odd combination of symptoms:

  • Email settings worked from a desktop computer (inside my home network)
  • Email settings worked from the phone if outside my home network (WiFi switched off, using cellular data)
  • Email settings DID NOT WORK from phone if INSIDE my home network on WiFi (using my ISP).

Contacted my ISP tech support who instructed me to use SSL/TLS and specified the port numbers; they are requiring encryption.

Problem: email settings on the phone don't have boxes to enter port numbers.

Solution: append port number to server name e.g. smtp.myisp.com:575

NOWHERE is it documented that you can do this.

Not sure why my GoDaddy-hosted account manifested the same problem, but it may just be co-incidence.

ANYWAY... important fact to know is that you CAN specify a POP or SMTP port number simply by appending it to the server as shown above, which gives the needed flexibility to get email working.

alexsun_2012 say:

Hi Nick,

When a calendar is shared to the users, they can add the shared calendar in Outlook desktop or Outlook Web App to view the events on it. For your first question, the answer is No. There is official solution to set the calendar to email everyone about the events. If they add the shared calendar in Outlook desktop or Outlook Web App, they can view the details just like their own calendar.

For your second and third questions, are you referring to ‘Event reminders’ by notifications? If so, the answer is No. Event reminders for a shared calendar is not available in Outlook desktop or Outlook Web App. Only the user’s default calendar supports the event reminders.

For the last question, do you mean you don’t want others to see the updates after some users modify the calendar events? If that is your requirement, the answer is No. The calendar is synced with the server side in real-time as well as with the other users’ clients. If you want them to have a different calendar, yes, a separate calendar is required for them.

I understand the limitations on a shared calendar may lead to some inconvenience to you.
