sacjug say: I fixed the Launch4J message by editing the JavaSoft registry key values:

Start RegEdit
Open this key HKEY_LOCAL_MACHINE\SOFTWARE\JavaSoft\Java Development Kit
Expand any sub keys and change any JavaHome values to your current JAVA_HOME value.

If you installed Java Development Kit in C:\Program Files\Oracle\6 then use this value "C:\Program Files\Oracle\6".
If you installed Java Run-time Environment in C:\Program Files\JRE then use this value "C:\Program Files\JRE".

You might be able to open a DOS command window and run this command to find the current Java path:
echo %JAVA_HOME%
