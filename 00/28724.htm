ben m. schorr -bschorr say: To some extent having THAT many notebooks all syncing across the network is going to inherently be slower than having much fewer syncing.

Several reasons:

1. Your NAS device can only read/write so many file blocks at a time. 
2. Your network may be bandwidth constrained.
3. The size of the notebooks can be a big factor.
4. How much RAM your machine has can be a big factor.

My first thoughts would be, if you have 300 open notebooks and performance is poor would be:

1. Examine how much RAM you have.
2. Close notebooks that don't need to be open.
3. Consider a faster disk system on the local machine.
