trekdozer say: How to Grant Permissions in Windows 7

1. Locate the file or folder in which you want to take ownership in Windows Explorer.
2. Right click on file or folder and select Properties from the Menu
3. Click on the Security tab, then click the Edit button. Click OK to confirm UAC elevation request if needed.
4. Select user/group from permission windows or click add to add other user or group (you may need to do this for multiple users - ex: administrator and your own user account).
5. Now under the Permission section check the rights which you want for the user i.e check "Full Control" under the "Allow" column to assign full access rights control permissions to the user.
6. Click OK for changes to take effect, and click OK again to exit from the Properties window.

ackis say:

[Guide] Windows Game DVR Troubleshooting Tips

The Game DVR for Windows 10 is great when it works, but it has some issues.


General support for the DVR can be found here:

http://support.xbox.com/en-US/browse/xbox-on-windows/social/Game%20DVR


Windows Game DVR System Requirements:


Supported Video Cards:

AMD:

  • Radeon HD 7000 series
  • Radeon HD 7000M series
  • Radeon HD 8000 series
  • Radeon HD 8000M series
  • Radeon R7 series
  • Radeon R9 series

NVIDIA:

  • GeForce 600 Series or later
  • GeFOrce 800M series or later
  • Quadro Kxxx series or later

Intel:

  • HD 4000 or later
  • Iris 5100 or later


Troubleshooting Steps:

Trouble shooter guides can be found at:

http://support.xbox.com/en-US/xbox-on-windows/social/troubleshoot-game-dvr-windows-10


Try Full-Screen Mode:

Sometimes you'll get an error "There's nothing to record" when running a game in windowed mode.  It'll refuse to run unless the game is in full-screen mode.  You'll have to change the settings in your game to get it to fill screen.  Another option is to press Alt-Enter and that may switch it to full screen.


You won't see the game bar in full screen so you'll need to use your shortcuts to start/stop recording.

http://support.xbox.com/en-US/xbox-on-windows/social/record-game-clips-game-dvr-windows-10


According to that article you need to press these keys in sequence to start recording.  Other keyboard shortcuts won't work.

  1. Windows + Alt + G
  2. Windows + Alt + R


Restart GAME DVR Task:

  1. Open up the Task Manager (Right click on the taskbar and go to Task Manager OR Press Windows-X and then press T)
  2. Find the DVR Server task (bcastdvr.exe).  Click it and go to End Task.


Delete Temporary Files:

  1. Open Settings App and go to System
  2. Go to Storage and choose This PC
  3. Scroll down and find Temporary files
  4. Click on it and then click on Delete temporary files.

This process may take a while to complete - it all depends on how many you have.  Wait until it's complete and then reboot your PC and try again.


Change the Shortcut Keys:

Sometimes it's possible that the shortcut keys are blocked by another program - you can't record by pressing Windows + Alt + R for example.

You can change these or assign a secondary shortcut by:

  1. Start menu
  2. Xbox
  3. Settings
  4. Game DVR
  5. Keyboard shortcuts


Reinstall or Upgrade Your Graphics Card Drivers:

Visit your graphics card manufacturer's website, locate your model and download the latest drivers.  Install/reinstall them, reboot and try the DVR again.
