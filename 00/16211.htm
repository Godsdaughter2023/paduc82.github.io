You have been given the answer multiple times, but here it is again:

  • The latest versions of Microsoft Word do not use MathType.
  • They use Microsoft Equation -- different software, built by Microsoft, not Design Science.
  • Microsoft Equation is no longer based on MathType.  The contract with Design Science has ended.
  • If the equation was made in any version of MathType, MathType can edit it. That includes old Word documents, which did use a version of MathType.

If you use MathType to create the equation, MathType will be able to edit it.  If you use Word Microsoft Equation, MathType can't read or edit that code.

There will never be any updates to enable Word to edit MathType code: Microsoft no longer has the licence from Design Science for that.

bernd p. say:

Some clarification is really necessary ;-)

If you open a notebook Onenote always copies the notebook to its local (= on your computer) Onenote cache (more or less hidden from you). Always means independent of the place where this notebook is stored. This guarantees that you can work offline under all circumstances. That's the reason why one suggests never to close a notebook you're working with. That's the way Onenote works, different from all other Office programs. Closing the notebook kills its cached copy which results in a performance loss at the next open, which is forced to renew the cached copy.

Locally stored notebooks = not on Onedrive

Notebooks are called local notebooks if they are stored on your computer or on a network share on another computer in your LAN.

Such a notebook is stored as a folder (with the name of the notebook) containing one or more section files of file type .one.

Normally they are stored under

C:\Users\<user>\Documents\OneNote-Notebooks

Online/Shared notebooks = on Onedrive

Those notebooks are either created on Onedrive by using your Internet browser and Office Online or by a mobile client.

And they can be "created" by moving an existing local notebook to Onedrive. But this move is in reality a transformation into another format.

The ominous Onedrive folder in your Windows

It's for synchronizing files from Word, Excel etc. to Onedrive for backup or sharing.

It's not for Onenote notebooks !

The dumb workaround from MS is creating only a shortcut in case of a Onenote notebook which allows you to open this notebook in your browser.

How to relocate a notebook from Onedrive to your computer

I assume you have it open in Onenote (otherwise open it from Onedrive). Right click the notebook. Select Properties. Then use Change Location

to C:\Users\<user>\Documents\OneNote-Notebooks

Using coefficients from trendline in Excel Chart - Microsoft Community

rg46873 say:

Using coefficients from trendline in Excel Chart

I want to be able to use the coefficients shown for a trendline in an Excel chart.  One way is to look at the values shown in the textbox and then type them into cells in the spreadsheet.  However, when more data or different data is used in the chart, one has to retype the values into the cells.  Is there some way to access the source of the coefficients shown on the chart  in the text box and link them into the spreadsheet cells?

I have tried the Regression pack add in but that does not refresh automatically as does the trendline in a chart

Adding in additional functions such as Trend does not appear to be a promising direction since that duplicates what has already been done in the trendline generation process.

hansv mvp say:

You can use the LINEST function for this.

For a linear trendline:

Select two cells next to each other in the same row.

For a scatter chart, enter the formula

=LINEST(y-range,x-range)

where y-range is the range specifying the y-values, and x-range is the range specifying the x-values.

Confirm the formula with Ctrl+Shift+Enter to make it an array formula.

The first cell will return the slope and the second cell the intercept.

For other charts, use =LINEST(y-range)
