you then have a choice

greg carmack - windows mvp 2010-2020 say:

The key will still work but the Digital LIcense takes precedence until you want to change a retail license to an entirely different PC using the Product Key.  Hardware changes using the DL are limited to parts changes including motherboards which require reactivating using the Activation Troubleshooter:  Reactivating Windows 10 after a hardware change

You said you presently have the Digital License linked to an old MS account email address you changed.

I showed you how to change the email address on the same account if you haven't already used it to create a new MS Account, after which it cannot be used to change the old MS account email address since you must use a fresh email address to change MS Account email or start a new MS account.

If you've already created the new MS account then I also showed you how to change it in Windows 10, either by converting your old MS to a Local account first so you can then change it back to the new MS Account; or by creating a new Admin MS account to move into and then when ready delete the old MS account.

Once you are signing into Windows 10 with the new Admin MS Account, it will change the Digital LIcense to the new account.

If you have any problems navigating the steps with illustrated tutorials I gave, just ask back any questions.
