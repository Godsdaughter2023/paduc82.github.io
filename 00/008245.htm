hansv mvp say:

Try this:

Sub UnmergeAll()
    Dim wsh As Worksheet
    Application.ScreenUpdating = False
    For Each wsh In Worksheets
        wsh.Cells.MergeCells = False
    Next wsh
    Application.ScreenUpdating = True
End Sub

hansv mvp say:

You could use this - change the list of sheet names as needed.

Sub UnmergeSome()
    Dim wsh As Worksheet
    On Error Resume Next
    Application.ScreenUpdating = False
    For Each wsh In Worksheets(Array("Sheet1", "Sheet2", "SheetC"))
        wsh.Cells.MergeCells = False
    Next wsh
    Application.ScreenUpdating = True
End Sub
