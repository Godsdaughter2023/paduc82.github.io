andreas killer say:

It's up to you to change the filename and location. You can e.g. get the filename from the model and change the extension. Sample:

Sub Main()
  Dim swApp As SldWorks.SldWorks
  Dim Part As SldWorks.ModelDoc2
  Dim Fullname As String

  Set swApp = Application.SldWorks
  Set Part = swApp.ActiveDoc
  Fullname = Part.GetPathName
  Fullname = Left(Fullname, InStrRev(Fullname, ".")) & "pdf"
  Part.SaveAs3 Fullname, 0, 0
End Sub

If you need further help please ask in the Solidworks forum.

Andreas.
