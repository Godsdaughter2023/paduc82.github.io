scottgem say:

Access 2010 introduced a new datatype called a calculated datatype. This datatype can store a simple expression that will display a value based on fields in the record. For more details on how this works, see my blog on the Calculated datatype. Fields of individual pieces of data that compose a table. So fields only exist in tables. 

The more traditional way to create a calculation is to add a column to a query that uses an expression. Queries have columns, not fields.

So you can create a query and add a column with the expression:

Alias: IIf([TS_LESS_CO3] Between 1000 And 2399, [TS_LESS_CO3] *0.0000007143+1.000286, Null)

If that doesn't work then try:

Alias: IIf([TS_LESS_CO3] >= 1000 And [TS_LESS_CO3] <= 2399, [TS_LESS_CO3] *0.0000007143+1.000286, Null)

tom_bpc say:

Hi bkei,

Yes, if you try to do this with custom flags and bar styles then you will quickly run out of room, as John highlights.

The simplest solution to me is to use vba to individually format the appropriate bars based on resource names.  These are basically individual exceptions applied to the bar style at each task (like right-clicking on an individual bar and selecting "format bar").

If you are willing to go the vba route, here's a bit of code to get you started.  Not very user friendly, but a place to start for your own use.  This code re-colors the middle of the 5th bar style to "Aqua" for each task with the resource "R1" assigned.

            

Sub BarColor()

Dim t As Task
Dim GS As Integer
Dim Color As PjColor

'Select which bar style is to be modified (row number from style list)
GS = 5

    For Each t In ActiveProject.Tasks
        If Not t Is Nothing Then
            If t.ResourceNames <> "" Then
            ' Select Colors for Different Resources
                If InStr(1, t.ResourceNames, "R1") <> 0 Then Color = pjAqua
                'Repeat for each resource and color combination
  
            ' Apply Color Exception
                GanttBarFormat TaskID:=t.ID, GanttStyle:=GS, MiddleColor:=Color
            End If

        End If
    Next t

End Sub

I think you also need a method to undo what you have done, so here it is:

        

Sub BarReset()

Dim t As Task
Dim GS As Integer
Dim Color As PjColor

'Select which bar style is to be modified (row number from style list)
GS = 5

    For Each t In ActiveProject.Tasks
        If Not t Is Nothing Then
            If t.ResourceNames <> "" Then
            ' Reset Color Exception
                GanttBarFormat TaskID:=t.ID, GanttStyle:=GS, Reset:=True
            End If

        End If
    Next t

End Sub

The way it is set up, the last resource entered in this code will color the bar for tasks with multiple resources assigned.  After all, there can be only one, as John notes.  You could very quickly get real sophisticated with this code - applying multiple colors to multiple bar styles, for example - but I'm not a fan. 

Good luck, tom

marites_p say:

Hi

           

To unblock your contacts, kindly follow the steps below:

For the classic Skype version (7.xxx):

  1. In the main menu, select Tools > Options.
  2. In the Skype Options box, select Privacy > Blocked Contacts. A list of blocked contacts is displayed.
  3. Select the contact you'd like to unblock and click Unblock this person.

For the Skype for Windows 10 UWP (12.xx):

Click the (...) icon > Choose Settings > Scroll down to Contacts > Click Manage blocked users > Click Unblock next to the name of the Skype contact.

For the new Skype (8.xx):

Go to settings with the ellipses (...) > right click > Application settings > select Manage blocked users.

Should you have further questions, do not hesitate to contact us back.
