tom-s say:

Hi Monika,

A couple of different ways you could tackle a 0% price increase in B6:

1) Extend the lookup table so the first row is 0% (price inc)  0% (quant dec)

    And amend the lookup range in the formula to A9:B16

2) Amend the formula to check first for 0% in B6, something like:

     =IF(B6=0,B5*B3,(100%+VLOOKUP(B6,A9:B15,1,FALSE))*B5*(100%-VLOOKUP(B6,A9:B15,2,FALSE))*B3)

3) If you want to use IFERROR then amend the formula something like:

     =IFERROR((100%+VLOOKUP(B6,A9:B15,1,FALSE))*B5*(100%-VLOOKUP(B6,A9:B15,2,FALSE))*B3,B5*B3)

 

I should have added last time that the problem with the original formula was that the multiplication (of B5 and B3) was occurring before the addition/subtraction - for more information on calculation operators and their order of precedence see this help page: http://office.microsoft.com/en-us/excel-help/calculation-operators-and-precedence-HP010342223.aspx

 

********* EDIT **********

I also meant to add that in the original formula you don't really need the first of the 2 Vlookup's because it's just returning the same value as it's looking up - so the formula can be re-written as:

=(100%+B6)*B5*(100%-VLOOKUP(B6,A9:B15,2,FALSE))*B3

rohn007 say:

IE has some features that help resist "malware", many types of bad software not just virus, but is not enough to protect your machine alone. Not by a long shot.

Install Microsoft Security Essentials, https://www.microsoft.com/security_essentials/ , it is free. That will help some more.

There are many types of malware and many types of protection. Your best bet is to search for newsletters and magazines that review antimalware and make recommendations. The quality/success of malware packages varies over time.

Ultimately you should aim for a plan that gives you "protection in depth" so that you are not relying on any one thing. One of the key things to remember is to actively only run one package for each type of malware ie virus, spyware.  It is OK, acutally a good idea to run periodic (weekly or at most monthly) scans with other packages. No one package will catch every bug, so running scans will hopefully catch the ones that slip through.

Some of the things you can/should do include:

  • install a router between your computer and the internet (yes, even at home)
  • use HOSTS file to provide protection ie http://www.mvps.org/winhelp2002/hosts.htm
  • install a firewall application Windows default firewall at absolute minimum or 3rd party freeware
  • install an anti-malware suite, or selected freeware antimalware applications
  • activate hardware and software DEP: http://windows.microsoft.com/en-US/windows-vista/Change-Data-Execution-Prevention-settings
  • install all windows updates on a timely basis - windows update on auto
  • make sure all your software is up to date - ie run Secunia PSI or other similar software
  • have a regular backup schedule: ie Monthly full image copy, weekly incremental
There are many more things you can do. The key is, don't depend on only one thing to protect you. And, know that no matter what you do, sooner or later you will get "infected". That is where the backups come in. In a worst case scenario you can restore from the backup to minimize your losses.

I think you are misunderstanding something here, what you are referring to could be misleading branded by Virgin. The 6300 has no WLAN hardware installed whatsoever, the best you have is EDGE services. The 6300 does not even have 3G or 3.5G.

Access to wi-fi internet requires no SIM card whatsoever because you can connect to a wireless local area network (WLAN) using hardware built into the handset and bypassing your operator's network altogether.

Now that networks are stopping strangling the bandwidths it is possible to get faster access to network-based services, but you are not hooking up to wireless internet in quite the same way as via a wireless internet router.

To recap:

6300 - Only GPRS/EDGE internet connectivity via the operator' network;
6301 - UMA for VOIP calling, still via the operator's network only;
6300i - intergrated WLAN functionality for full, network-independent wireless network access.
