For an unlimited number of predecessors, I would try (untested at this point) an array formula, something like the following which I'll build step by step to show the logic.

=index($D:$D,match(F62,$A:$A,false))

Gives me the value of one item

=index($D:$D,match(F62:H62,$A:$A,false))

isn't valid, but if you enter it as an array formula and wrap it in a MAX, I think it should work

=MAX(index($D:$D,match(F62:H62,$A:$A,false)))

entered with Cntrl-Shift-Enter to get the automatic curly brackets (entering these by hand will not make it an array formula)

{=MAX(index($D:$D,match(F62:H62,$A:$A,false)))}

Assuming that this works in an ideal case where you have 3 cells with precedent information- chances are an individual match/index will error for empty cells (or if you have anything other than a real match). So the overall MAX would error as well. If this is true, your options are to either (a) default all cells in all predecessor columns to the value of your initial step (easiest way to go), or (b) set the range of F62:H62 dynamically so it only references cells with contents, something like

=F62

gives you the first cell

=CountA(F62:K62) ... ending column is however many columns you are reserving for precedent info

=offset(F62,0,CountA(F62:K62)) gives you a dynamic range based on how many columns have data in them. This requires that precedent info be filled in from left to right with no blank columns in the middle.

{=MAX(index($D:$D,match(offset(F62,0,CountA(F62:K62)) ,$A:$A,false)))}

Truthfully, that is a lot of meat and without testing I'm not 100% confident that the array formula will handle it the way I think it will. You might want to build up the pieces in helper columns so you can see which parts are working (and how they are working), and build up the same way to get to the final array formula and test it.

If I understand you correctly, you want to determine the latest of the completion dates for the tasks listed in the Predecessor columns to the right of the Compliance entry and add one to it.  If that is the case, then:

Assume ID is the named range of your ID numbers (e.g: $A$3:$A$73 in your example), and Finish the range of Finish dates ($D$3:$D$73), then:

This formula must be **array-entered**:

    =MAX(IF(ID=F62:H62,Finish,""))+1

----------------------------------------

To **array-enter** a formula, after entering
the formula into the cell or formula bar, hold down
<ctrl><shift> while hitting <enter>.  If you did this
correctly, Excel will place braces {...} around the formula.
