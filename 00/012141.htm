Multiple Columns to produce one field on a pivot table - Microsoft Community

prazz_914 say:

Multiple Columns to produce one field on a pivot table

I have a spreadsheet that logs issues, resolutions and the assignee. Some issues have more than one Assignee so I have columns for Assignee, Assignee2 and Assignee3. When I created  my pivot table, it nest assignee2 and assignee3 under assignee. I want each of those columns (assignees) to be considered separate. So the same issue can be under Assignee and Assignee2 or Assignee3. Does this make sense? I need to have each person whether they are assignee or assign 2 or 3 to be counted separately on an issue. Thanks for your help !
max, singapore say:

You need to transform your source data to be "pivot-ready" before creating the pivot. Here's an example illustrating the use of formulas to easily achieve that motive

Assume this original source table is in Sheet1, in A1:D5

Desc ASK1 ASK2 ASK3
text1 1271 1649 1815
text2 1049 1320 1447
text3 1030 1641 1928
text4 1494 1523 1171
.....

In another sheet:
Put in A2: =OFFSET(Sheet1!$A$2,INT((ROWS($1:1)-1)/3),)
Put in B2: =INDEX(Sheet1!$B$1:$D$1,MOD(ROWS($1:1)-1,3)+1)
Put in C2: =OFFSET(Sheet1!$B$2,INT((ROWS($1:1)-1)/3),MOD(ROWS($1:1)-1,3))
Copy A2:C2 down as far as required to exhaust the desired data transformation ...
Note: Just modify the "3" to suit the number of cols to be made vertical (ie the # of ASK cols in the example)

The "pivot-ready" results will look like this:

text1 ASK1 1271
text1 ASK2 1649
text1 ASK3 1815
text2 ASK1 1049
text2 ASK2 1320
text2 ASK3 1447
text3 ASK1 1030
 ....
