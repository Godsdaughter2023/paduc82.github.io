Replace the User Name in the Personalize field.

hansv mvp say:

The following macro will protect Sheet1 while allowing outlining:

 

Sub ProtectWithOutlining()
    With Worksheets("Sheet1")
        .EnableOutlining = True
        .Protect Contents:=True, UserInterfaceOnly:=True
    End With
End Sub

 

This setting is not stored with the workbook, so you have to apply it each time the workbook is opened. You can do so automatically:

In the Visual Basic Editor, double-click ThisWorkbook (on the left hand side, below Microsoft Excel Objects).

Copy the following code into the ThisWorkbook module:

 

Private Sub Workbook_Open()
    Call ProtectWithOutlining
End Sub
