gonçalocouto say:

in the A1 cell, but using VBA code. Is it possible?

 

Also, I need to know if there is any VBA control that will allow me to select the last active sheet, before the one now active in my code...

 

Thanks and best regards,

Gonçalo Couto from Portugal

hansv mvp say:

Activate the Visual Basic Editor.

If you already have a code module, open that. Otherwise, select Insert | Module.

Insert the following line near the top, before all Subs and Functions if you're using an existing module:

 

Public LastSheet As Worksheet

 

In the project explorer on the left hand side (the Windows Explorer-like treeview), under Microsoft Excel Objects, double-click ThisWorkbook.

Copy the following code into the ThisWorkbook module:

 

Private Sub Workbook_SheetDeactivate(ByVal Sh As Object)
    Set LastSheet = Sh
End Sub

 

This code should *not* go into a standard module. It is a so-called event procedure of the Workbook object. This code will run automatically each time you deactivate a worksheet by moving to another sheet. You can now use the variable LastSheet in your own code. For example, to refer to cell C12 in the sheet that was active before the currently active one, you can use

 

LastSheet.Range("C12")

 

You may want to check whether LastSheet is valid - if the user hasn't moved between sheets yet, LastSheet will be Nothing. For example:

 

Dim rngCell As Range
If LastSheet Is Nothing Then
    ' User hasn't moved to a different sheet yet
Else
    Set rngCell = LastSheet.Range("C12")
    ...
End If

brian tillman [outlook mvp 2007-2019] say:
Suppose you create a rule that says "Sender's address does not contain outlook.com", then when you get a message from any address other than one that uses "outlook.com", then the rule's action (Move to Inbox, Move to a new folder, Delete these messages, etc.) will take effect.  So, if you choose the action "Forward to" and specify another address you own, then all the messages you get from people who do not use the "outlook.com" domain will be forwarded to that other address you own and only messages from addresses containing "outlook.com" will stay in your Outlook.com mailbox.  If this more clear?
