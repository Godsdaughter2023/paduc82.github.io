I found my own solution :-)

I will give it here, but my windows 7 is in danish, I hope to translate it so it can be understood.

Problem: Windows Media Center dont find any TV-tuner after I have used another applikation (Cyberlink Power Cinema 6).

Solution:

1. Go to Control Panel.

2. Under programs - klick on "remove a program."

3. In this window - upper left corner - you find "switch windows-functions on or off" - klik on that message.

4. Find the map "Media-functions" and open it with the + sign.

5. Remove the "hack" in "Windows Media Center" and klick on OK.

6. Wait for a restart.

7. After the restart - do the same things again, but this time you must place a "hack" again in Windows Media Center and klick OK.

8. After this you can start Windows Media Center and after a short message "searching for tuners" - you have picture from the TV-tuner.
