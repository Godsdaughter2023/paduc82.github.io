What does "A network cable is not properly plugged in or may be broken" mean? 

How do you connect to the Internet?  Full path.  E.g. if there is a cable to a wireless router and there is no Internet connection detected the message could be referring to that.  So that would explain the first "problem" but then in the course of diagnosing that it may have reset the Wi-Fi driver, found that there actually was an Internet connection and so declared that aspect of the problem "fixed".

If it happens a lot you should do some deeper analysis of your symptoms.  For example, one user discovered that it may have been a situation where there was more than one channel with the same SSID and switching between those would have been causing instability and confusion.

The Error Code: 0xC1900101-0x20017 is caused by a driver bug check while booting the computer during the Windows 8.1 installation process.

This error could be either due to the outdated device drivers or the conflicts from certain third party desktop utilities (For Example: Certain utilities from IOBit or Intel Desktop Utilities like including Extreme Tuning software but excluding Managed Engine).
