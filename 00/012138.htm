greginmich say:

My wife's computer displays a pop up claiming to be Microsoft and saying that the computer has been blocked. They want to call and control the computer. Not happening.

How do I get rid of it?

No genuine Microsoft alert will ever pop up in a web browser and ask you to call a telephone number for assistance. So there’s really no need to even identify the source – all of these security/virus alerts are fake. These Tech Support Scam websites lock up a browser by just running a script that immediately loads a modal alert dialog – and then reloads that dialog every time you try to close it by clicking on the OK button. The modal alert dialog was originally designed to force users to respond to the alert by locking up the browser until they closed the dialog – so by simply looping the dialog like this, the malicious script can prevent you from ever regaining access to the browser’s controls.

In Microsoft Edge, and also in several third-party browsers, you can close one of these browser-locking websites by using the Dialog Loop Protection checkbox that appears in the alert dialog after you click on the OK button:

Microsoft provided Dialog Loop Protection for the Edge browser over a year ago so that users could stop the malicious script that keeps looping the modal dialog and then close the scam website – without having to go through the trouble of ending the browser process in Task Manager and restarting Edge indirectly. Microsoft’s belated version of Dialog Loop Protection followed the precedent that had already been set by other browser developers prior to the release Edge – and this feature actually turned out to be even more important in the case of Edge – because Edge doesn’t provide any way to prevent the automatic recovery of malicious websites:

https://wilde-it.co.uk/blog/post/locked-in-microsofts-edge-browser

Fortunately for everyone concerned, the third-party browser developers are also leading the effort to first deprecate, and then finally eradicate, the showModalDialog method. For a recent discussion of the Tech Support Scam issue, see this thread:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/defender-ineffective-against-the-windows-warning/dfa1ffd3-56cf-403d-b8a9-96f2a4846909

Since Microsoft still seems to be having a little bit of trouble appreciating the true scope of this issue, and the severity of its impact; it might help things along if more people reported this scam directly:

https://www.microsoft.com/en-us/safety/online-privacy/avoid-phone-scams.aspx#reptech

If that link is broken, here's the list of resources provided in the document:

How to report tech support scams

  • Help Microsoft stop cybercriminals by reporting
  • In the United Kingdom, you can report

    Of course you should be sure to include the telephone number and business name (if available) in your report, in addition to the URL – because the scammers already have the ability to spoof the URL, according to this source:

    https://www.bleepingcomputer.com/news/security/microsoft-edge-smartscreen-security-feature-can-be-abused-for-tech-support-scams/

    There’s also some great information in this Malwarebytes blog that shows how the scammers are able to fake Google Chrome’s address bar URLs and Dialog Loop Protection checkbox:

    https://blog.malwarebytes.com/cybercrime/social-engineering-cybercrime/2016/08/tech-support-scams-and-google-chrome-tricks/
