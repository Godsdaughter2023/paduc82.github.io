graham mayor say:
Whether 'half' the paragraph is the dividing point I cannot say, but the premise is certainly true. Format  the majority of a paragraph with italics and then hover over the quick styles selections.

I suspect the only way to do it would be along the lines suggested already e.g. the following macro will apply the named style to the selected paragraph(s) (or the paragraph containing the cursor) retaining italics. It could easily be modified to include underlines, bold etc. as appropriate.

Sub ApplyStyle()
Dim oPara As Paragraph
Dim oWord As Range
Const strStyle As String = "List Paragraph"
    For Each oPara In Selection.Paragraphs
        For Each oWord In oPara.Range.Words
            If oWord.Font.Italic = True Then
                oWord.HighlightColorIndex = wdYellow
            End If
        Next oWord
        oPara.Style = strStyle
        For Each oWord In oPara.Range.Words
            If oWord.HighlightColorIndex = wdYellow Then
                oWord.Font.Italic = True
                oWord.HighlightColorIndex = wdAuto
            End If
        Next oWord
    Next oPara
End Sub
