jeeped say:
I'll assume that you do not have a list of holidays anywhere and that you are concerned with federal holidays in the USA. Try this snippet that takes advantage of the native worksheet WORKDAY() function,

    With ThisWorkbook.Sheets("Service Invoice")
        Dim dtHolidays As Variant
        dtHolidays = Array("1-Jan-2013", "21-Jan-2013", "18-Feb-2013", "27-May-2013", "4-Jul-2013", _
                           "2-Sep-2013", "14-Oct-2013", "11-Nov-2013", "28-Nov-2013", "25-Dec-2013")
        With .Range("E3")
            If IsEmpty(.Value) Then
                .Value = Application.WorkDay(Application.EoMonth(Date, -1), 1, dtHolidays)
            End If
        End With
    End With

Edit: alternate code that uses VBA to get the last day of the previous month rather than a second worksheet function.

       .Value = Application.WorkDay(DateSerial(Year(Date), Month(Date), 0), 1, dtHolidays)
