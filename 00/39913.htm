Understanding downgrade rights

Consumer versions of Windows 8 Pro that come preinstalled on PCs have downgrade rights to a previous version of Windows (example: Windows 7, Windows Vista).

If your copy came pre-installed on your system, you may downgrade your copy to Windows 7.

If not you need purchase a copy of Windows 7 and install the same on your computer.

• Downgrade rights are only available for licenses obtained through OEM or Volume Licensing (VL) and not apply to Full version (FPP).

How to export Windows movie maker project 7 to another pc - Microsoft Community

thomas anthony buttigieg say:

How to export Windows movie maker project 7 to another pc

Can someone pls help me exporting windows movie maker project 7 to continue working on it on another pc pls ??
considerate_guy say:

In order to edit on another PC you need your project file (.wlmm) and all the source files.  That means you must take all your pictures, video and music too.  A project file just refers to those files where they exist on your computer. If you just copy the project file to another PC, Movie Maker won't be able to access the source files and you'll see yellow triangles.

 

Assuming you have copied the project file and the source files, you may still see yellow triangles on the new computer. That's because the project file is looking for an exact match with the location of the files. If you have copied the files, but put them in a different folder, you'll have to double-click on a yellow triangle and Find the new location. Then the yellow triangles will go away.

 

One last caveat, you must make sure that the version of Movie Maker is the same on both PCs.  In particular, if the version of Movie Maker on the second PC is older, it won't be able to open the newer format project file.  In order to go back and forth, you want the versions to be exactly the same.

each installation.

shows up. This is because some drivers rollback before the latest is installed (sound drivers
particularly do this) so install a driver - reboot - check to be sure it is installed and repeat as
needed.

Repeat at Device Makers - BTW at Device Makers DO NOT RUN THEIR SCANNER - check

http://msdn2.microsoft.com/en-us/xml/bb190622.aspx
Method 2: Replace the corrupted files To resolve this problem, follow these steps:
  1. Copy the following DLL files from the C:\WINDOWS\SYSTEM folder on a computer that is working to the computer that is experiencing the problem:
    • Msxml3.dll
    • Msxml3a.dll
    • Msxml3r.dll
  2. After you replace the files, you must register the files. To register the files, follow these steps:
    1. Click Start, and then clickRun
    2. In the Open box, type REGSVR32 C:\WINDOWS\SYSTEM\MSXML.DLL, and then click OK. You receive a message when the registration succeeds.
    3. Repeat steps a and b with the following files:
      • C:\WINDOWS\SYSTEM\MSXML2.DLL
      • C:\WINDOWS\SYSTEM\MSXML3.DLL
    4. Restart your computer.
If the above doesn't work try the following:

1. Open a Command Prompt as and Administrator.

2. Type the following Command


sfc /scannow

(This could take some time but let the scan complete on it's own and please do not close the Command Prompt Window or Shut Down/Power Off your Computer until the scan is Complete. It is Recommended that you Close out of all other Open Windows and Save all work before Starting the Scan)


3. Your Done! If you followed the steps correctly you should be good to go!

How does the Windows 7 Upgrade Option work?

The Windows 7 Upgrade Option is available through participating PC manufacturers on selected PCs and through Microsoft on qualifying purchases of the Windows Vista packaged product. The Windows 7 Upgrade Option applies to Windows Vista Home Premium, Business, and Ultimate editions. Your Windows 7 upgrade will be to the comparable version: so you'll get either Windows 7 Home Premium, Windows 7 Professional (the upgrade to Windows Vista Business), or Windows 7 Ultimate, respectively. The Windows 7 Upgrade Option program runs from June 26, 2009 through January 31, 2010. Some Manufacturers may choose to shorten the Upgrade Option time frame.
