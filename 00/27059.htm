Restrict entries based on cell fill colour - Microsoft Community

devendraram say:

Restrict entries based on cell fill colour

Hi,

I want to allow the users to enter the details only in the cells with grey background in below screenshot. Thus I want to deny access to cells with no fill (white). Can anyone help me with step wise procedure for this?. It may be noted that while locking the cells, the other functionalities like filter etc should be available.

ossiemac say:

All cells on a worksheet are locked by default but the locking only becomes effective when you protect the worksheet.

  1. If you have been experimenting with locking cells and protecting the worksheet then best that you get it back to the default situation.
  2. If the worksheet is protected then select "Format" icon and unprotect and then the following.
  3. Click the button top left of worksheet where the column Id's and row Id's meet and this will select all cells.
  4. Select "Format" icon and in the DropDown click the "Lock Cell" line.
  5. Select "Format" again. If the padlock adjacent to "Lock cell" has an orange border and background then all cells are locked as per the default. If no orange border around the padlock then click "Lock cell" again and then select "Format" again and re-check that orange border is present.
  6. After you have set the default situation for the cells then Select the Grey cells. (You can select a range of grey cells and then hold the Ctrl key while selecting the other grey cells in separate ranges.)
  7. Select Format icon (on Home ribbon).
  8. In the DropDown you will see an Orange border around the padlock adjacent to "Lock cell" indicating that the selected cells are locked. Click the "Lock cell" and the orange border around the padlock will disappear to indicate the cells are unlocked. (Need to select format again to see that orange border is gone)
  9. Select Format icon again and then select Protect Sheet.
  10. The first 2 boxes should be checked by default and can remain as is. Check the boxes against the remaining functions that you want the user to be able to perform. ("Use AutoFilter" is one of them but you might need to scroll down the list to see it.)
  11. Enter the password (you will be prompted to enter a second time) and then click OK.
