-> In current terminology you might call this "Create an image file of your system drive"

2. Install the copy of the "C" drive onto a new internal drive. What do you call this?

-> Again in current terminology you restore the image file to the new disk.

1. What exactly are the steps I must take to get the contents of the "C" drive to the external drive from within then Disk Management dialogue box?

-> Nothing. The Disk Manager is not designed to do this. Control Panel / Backup and Restore / Create a System Image is the correct place.

2. How do I then install the contents of the external drive to my new internal "C" drive. (I understand I must first disconnect the old "C" drive as you previously stated.)

-> You boot the machine with your System Repair CD, then select the option to restore an imge.

I only want to copy the contents of the "C" drive to the external drive. I do not want to copy the "M" drive which is also designated as a system drive.

-> On some machines the image creation tool insists on including partitions other than the System partition in the image. In these cases you must first remove the drive letter for that partition. You can restore it later on.

I encourage you to make use of the extensive help built into Windows. When you click the Start globe, then Help, then look for help on "System image" then you will get comprehensive answers to all your questions.
