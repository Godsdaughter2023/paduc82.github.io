How to Calculate The Percentage Total of a Column - In Power Query

I have a large data set that is grouped by month with budget dollars assigned to each month. For each month, I want to insert a formula in Power Query to calculate the percentage of grand totals to get an aggregate for each month.

Thanks for you help.

ashish mathur say:

Hi,

Try this

1. Select your dataset (which is tow columns - Month and Amount) and press Ctrl+T.  Excel will automatically assign a name to it - Table1

2. Select the dataset and take it to Power Query

3. In the Power Query window, go to View > Advanced Editor and paste this code there

let
    Source = Excel.CurrentWorkbook(){[Name="Table1"]}[Content],
    #"Changed Type" = Table.TransformColumnTypes(Source,{{"Month", type text}, {"Amount", Int64.Type}}),
    #"Added Custom" = Table.AddColumn(#"Changed Type", "Total", each List.Sum(#"Changed Type"[Amount])),
    #"Added Custom1" = Table.AddColumn(#"Added Custom", "Contribution (%)", each [Amount]/[Total]),
    #"Changed Type1" = Table.TransformColumnTypes(#"Added Custom1",{{"Contribution (%)", Percentage.Type}}),
    #"Removed Columns" = Table.RemoveColumns(#"Changed Type1",{"Total"})
in
    #"Removed Columns"

4. Go to Home > Close and Load
