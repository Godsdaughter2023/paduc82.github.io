steve scoltock say:

See my post "Windows 8 - TV recording problem with Windows Media Center"

 

The answer to my problem is in the last but one post ....

 

I have managed to solve it now by running a recovery task in the Computer Management application.

 

In case anyone else has this problem, here is what I did:

 

1. Close down Windows Media Center

2. Open the Computer Management application and navigate down the tree in the left hand panel by selecting System Tools->Task Scheduler->Task Scheduler Library->Microsoft->Windows->Media Center

3. In the top, centre panel find the task "MediaCenterRecoveryTask" which should have a status "Ready".  Right click on this task and select Run (status goes to "Running")

4. Wait a while for the task to do its stuff (the status doesn't seem to change) then right click on the task again and select End (status goes back to "Ready")

5. Open Windows Media Center which will report that there has been a problem and that it is necessary to run through the setup again

 

After running the above task and having successfully performed a Windows Media Center setup, I found that I could once again record tv!
