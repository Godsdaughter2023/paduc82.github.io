Right click the sheet tab where you want this to happen, view code and paste the code below in on the right. It currently works on column 8 (H) and it will autofit when selected and resize to normal when not. Change the 8 to your notes column.

 

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
Dim MyColumn As Long
MyColumn = 8 'means column H. Change to your notes column
If Target.Column = MyColumn Then
    Columns(Target.Column).EntireColumn.AutoFit
Else
    Columns(MyColumn).ColumnWidth = 8.43
End If
End Sub

>> I can save it as PPTX and it will reduce in size to 5602KB.  But it has to be in PPT format due to the variety of people that need to use it so if I open the new PPTX and save it again as PPT the file size goes back up to 7995KB.

 

When you save as PPT, PowerPoint 2007 and 2010 save some features in two different versions.  For example, if you've applied gradient shadows, glows or any of the new features that aren't supported in previous versions, it saves an IMAGE of the feature, which can cause file sizes to balloon.

The fact that it shows as 100% suggests that PPT thinks it's compressed the image.  Whether it's made the image appreciably smaller will depend on how large the image was originally (in pixels) vs how large it's set to in the slide.
