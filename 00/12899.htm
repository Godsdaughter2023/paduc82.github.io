Refer below image:

Main tab is named Sheet1 which contains all data. Name the next tab as 6000 (one of the Dept#). Now in the tab named 6000:

Enter Below formula as an array formula (CTRL+SHIFT+ENTER) in cell B2 & copy down:

=IFERROR(INDEX(Sheet1!$B$2:$B$9,SMALL(IF(Sheet1!$A$2:$A$9=VALUE(MID(CELL("filename",$A$1),FIND("]",CELL("filename",$A$1))+1,255)),ROW(Sheet1!$B$2:$B$9)-ROW(Sheet1!$B$2)+1),ROWS(Sheet1!$B$2:$B2))),"")

Enter Below formula as an array formula (CTRL+SHIFT+ENTER) in cell D2 & copy down:

=IFERROR(INDEX(Sheet1!$D$2:$D$9,SMALL(IF(Sheet1!$A$2:$A$9=VALUE(MID(CELL("filename",$A$1),FIND("]",CELL("filename",$A$1))+1,255)),ROW(Sheet1!$B$2:$B$9)-ROW(Sheet1!$B$2)+1),ROWS(Sheet1!$B$2:$B2))),"")

Enter below formula in cell A2 & copy down:

=IF($B2<>"",MID(CELL("filename",$A$1),FIND("]",CELL("filename",$A$1))+1,255),"")

Your first tab 6000 is complete & will display data related to Dept 6000. Now copy the 6000 tab and create a copy. Name the new tab 6005. This tab will display data related to Dept 6005.
