rob koch say:

How do I tell windows defender that I am using a private network.

Why would you want to change this setting from Public to Private if it's otherwise working fine for you?

The reason I say this is that setting the Public network type is always more secure, since your PC will be hidden from other devices on the network by the firewall, meaning it can't be directly attacked via the network.

Make a Wi-Fi network public or private in Windows 10 - Windows Help

The disadvantage of choosing Public is that you can’t use your PC for file and printer sharing, such as being a member of a HomeGroup.  In some cases this can also create issues with printing to a network attached printer, though in theory this should still work and if it doesn't it's typically due to problems with the printer's driver.

In my own case, I've operated my laptops in the Public configuration for years, since as a security professional I'd often travel into and out of many different customer networks, so I never wanted to take the risk of malware or other network attacks breaching my own PC.  For that reason, I always ran all networks in the Public configuration, which since I'd never need to share anything from the laptop's drives (files were stored on servers) and no printers were directly attached to it either, there was really nothing to share from that device.

suzanne s. barnhill say:

In the example that you gave, you would define a multilevel list that had Heading 1 linked to Level 1 and NewLevel linked to Level 2, with Level 2 set (as it is by default) to restart after Level 1. If you want NewLevel to restart after any heading style, then you will have to link the number of headings you're using to the corresponding levels (Heading 1 at Level 1, Heading 2 at Level 2, Heading 3 at Level 3, and so on) and the paragraph style at a lower level (say Level 4). If it is set to restart after the next-higher level, it will restart after any higher level.
