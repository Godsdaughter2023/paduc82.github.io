e.gray say:

I was able to transfer my Favorites from Explorer to Edge but I cannot add any new websites to Favorites. That "star" icon doesn't work. How to fix?

There are actually two "star" icons.... the first one is next to the "hub" (the 3 little lines), the draw icon,  share icon, and the "more" icon (the 3 little dots).   This star icon is used to add a new favorite to your list.

...the other "star" icon is after you've clicked on the "hub".  This star show the favorites in the "hub pane" (as opposed to the reading list, history, and downloads).

Yeah, it's kinda strange that they'd use the exact same icon to mean two different things in the same application.

So, make sure you're clicking on the correct "star"!

Hi

You shouldn't be able to create a folder called On My Computer because by default, this folder is generated automatically. Did you create this folder within some other folder?

The problem you've encountered is that your old profile containing your old On My Computer content is corrupt and can't be recovered, correct?

It's very rare to encounter a corrupt profile. It's so rare, in fact, that it makes me wonder if there is something wrong in Mac OS on your computer. The first thing I want you to do before attempting to fix anything is to perform a Safe Boot.

macOS Sierra: Start up in safe mode - Apple Support

A safe boot checks the volume, directory, makes any necessary repairs, and clears the font cache among other things. Once you reach the desktop from a Safe Boot, go ahead and do a regular boot and continue from there.

Next, be sure Mac OS and Office are up to date. Go to the Outlook menu and choose About Outlook. You should have version 15.37. If it is an older version, please follow these instructions and install updates:

Make sure your Mac has a working internet connection before checking for updates. Install Mac OS X updates followed by Office updates.

Install Mac OS X updates first:

To get Mac OS updates, go to the App store on your dock, and then click the Updates button to reveal available updates.

Then Install Office Updates

To get the update:

1. Open any Office application:

3. Make sure your Mac has a working internet connection

4. Go to the Help menu and choose Check for Updates

5. Let AutoUpdate check for and install updates

6. Restart your Mac
