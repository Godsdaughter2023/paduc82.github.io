john korchok say:

Those versions are quite different with audio. 2007 links most files rather than embedding them. Whenever you're working with 2 different versions, get it to work in the older version first, then it will be easier for it to function in the newer software. With 2007, you can embed the audio if it's a .WAV file. Otherwise, place the audio file in the same folder as the presentation, then link it to the presentation.
