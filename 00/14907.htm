'------ start of example code ------

Private Sub cmdCopyPrevious_Click()


    On Error GoTo Err_cmdCopyPrevious_Click

    Dim fld As DAO.Field

            End If
        Else

        End If
        If .BOF Then

            Next fld
        End If
    End With

Exit_cmdCopyPrevious_Click:
    Exit Sub

You'll have to post a little code so we can see how you are using the IsNumeric() function. IsNumeric() evaluates to True for positive or negative values and blank cells (but not cells with formulas that produce an empty string). Additionally, dates are not reported as numbers with VBA's IsNumeric() but are considered numbers by a worksheet's TYPE() or ISNUMBER() function. Use VBA's IsDate() for that functionality. Conversely, boolean (i.e. logical or TRUE/FALSE) are considered numbers by IsNumeric() but not by the TYPE() or ISNUMBER() worksheet functions.

You might want to run this formula on your data,
=LOOKUP(TYPE(A1),{1,2,4,16,64},{"Number","Text","Boolean","Error","Array"})
... where A1 is the first cell with a value, then filled down as necessary.

Word 2016 normally opens to whatever subfolder you opened the last document from. To ensure you're not seeing the Backstage view, please open File>Options>Save and check the option for Don't show the Backstage when opening or saving files. Now using Ctrl + O will open the normal File Open dialog and it should remember the last folder location after you open the first document.

If you want the File>Open folder to always be the same, you can set it using File>Options>Advanced. Scroll down the the General section, click on the File Locations button, select Documents, click on the Modify button, navigate to your preferred folder and click on OK when you've opened it. Now File Open will always open there.

how to make a table invisible - Microsoft Community

nice to know say:

how to make a table invisible

I created a document that is with a table I would like to printed without the table being visible
gero rometsch say:

From my viewpoint, there are 2 ways to interpret your issue. The first is that when you print, you don't want any of the text and the table structure to print. If that is the case, then Mr. Tyson's response is the easiest to use.

If you want the text in the table to appear but without all the boxes (or borders) around the text, that's a design change. To remove these borders is simple:

  1. Select the table
  2. Click on the Table Tools|Design tab on the Ribbon (2nd from right).
  3. Click the drop-down arrow that's next to the "Borders" control on that Ribbon, then click on "All Borders" on the drop-down menu that appears.

This should erase all the borders. If you are seeing dashed lines that's light in color, the "View Gridlines" option is on, but no border will print.

Let us know which solution works for you.

terence soh [msft] say:

kmason5727,

 

Unfortunately with WM5, there is no option to leave messages on the POP3 server after deleting them from the device. There are special cases where this appear to be contradictory, that is because those servers ignore the delete requests sent from other POP3 clients.  Your options are 1) upgrade to WM6 (hopefully your mobile operator or device manufacturer would provide that), or 2) use an IMAP email account that support folders, in which case deleted messages will be moved to a trash folder on both the device and on server.

 

WATERBUFFALOBILL,

 

With the Messaging app in WM6, there is now an account specific option to leave messages on the POP3 server after deleting them from the device.  Depends on the specific WM6 version, it is inside one of the send/receive related advanced settings and should title “When deleting messages”.  This should work, unless the server has special settings like the one Jaider26 described.

 

Jaider26,

 

I’m not familiar with CenturyTel’s email, but from your description it looks like it’s a server specific setting where it simply deletes all messages from server upon downloading them to any POP3 client (your WM6 device or a desktop mail application like Outlook).   Try checking “Keep a copy of forwarded mail in my inbox” option.  See http://community.centurytel.net/help2/
