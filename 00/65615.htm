mohammad _ imran say:

Hi Durdle,

 You can use the Volume Shadow Copy Administrative Command-line tool to change how much space you want System restore to use.
Run the command prompt with administrative rights by typing "cmd" in the search box on the start menu, then pressing Ctrl-Shift-enter.

To change the amount of space you devote to system restore, use the vssadmin command with these switches and parameters.

 

1) From within an administrator account open the command prompt. To do so:

 Click on Start – All Programs – Accessories and right click on the Command Prompt, select Run as administrator.

2)  To view lists existing volume shadow copies type or paste the following command then press enter: 

vssadmin list shadows

3) Displays all shadow copy storage associations on the system and their size:

 vssadmin list shadowstorage

 4) The “vssadmin resize shadowstorage” command can be used to resize the storage area.

Here’s a example of the command used to resize the shadow storage area on the C:\ partition to a maximum size of 6 GB’s.

 vssadmin resize shadowstorage /For=C: /On=C: /MaxSize=6GB

 Note: As soon as you disable/enable System Restore, the original values will be back.
