frederik long say:

I would rather avoid resetting the hard drive in the hopes of saving my data.

I wince when I read statements like these. They confirm that there is only one really effective incentive to get people to back up their data regularly (e.g. weekly) to an external disk: The catastrophic loss of all data, or the threat of such a loss.

In your case you are half-lucky: It's only the threat of a loss. Saving your data is tedious but it's your only option. Here is how to do it:

  1. Buy a 500 GByte hard disk in a USB case. It costs less than $50.00.
  2. Ask a friend to create a bootable Windows Repair flash disk on his Windows 10 machine. The command RecoveryDrive.exe will do it. Do NOT copy the system files.
  3. Boot your PC with this disk.
  4. Select advanced options, then open a Command Prompt.
  5. Connect the USB hard disk.
  6. Type this command to save your data to the USB disk:
    robocopy /s  c:\Users\Epicness\Documents  E:\Documents
    (assuming that the USB disk is visible on drive E:)
  7. Repeat the process for your Music and Pictures folder.
  8. Do a few spot checks on a different PC to ensure that you can read the backed up files.
  9. Continue with your repair or restore efforts.

If you use a local E-Mail application such as Outlook then you might want to save its files too. Locating them is a little more demanding. The same goes for your browser favorites.

Bevor Du anfängst, gleich eine Warnung! Wie Weidhas schon erwähnt hat, ist es über den Cisco Client möglich, alle anderen lokalen Netzwerkverbindungen abzuschalten. Es gibt zwar in dem Programm einen Haken, bei dem Du einstellen kannst, ob lokales Netzwerkeln möglich sein darf oder nicht, aber wenn die VPN Gegenstelle des Campus dem Client mitteilt "Egal was der User anhakt oder nicht: LAN bleibt deaktiviert!" wirst Du die Xbox niemals über das Campus Netzwerk online bringen können.

Ist das Aktivieren der LAN Schnittstellen möglich, kannst Du aber dennoch nur eingeschränkt Multiplayer Partien spielen, je nachdem wie Dein Campus das Studentenwohnheim Routet und NATed.

Am Ende gibts noch eine "unkonventionelle" Lösung, wenn alle Stricke reißen.

Erstmal die "Hilf Dir selbst so hilft Dir Gott" Lösung:

Wie weiter oben schon erwähnt, brauchst Du eine zweite Netzwerkkarte. Entweder einen USB LAN Adapter kaufen, oder wenn die Xbox WLAN-Fähig ist, und Du Windows 7 oder höher verwendest, kannst Du den Virtuellen Accesspoint verwenden.

Auf Deinem Laptop sollte in der Eigenschaftenseite für Netzwerkadapter (Windowstaste + R, dann "ncpa.cpl" eingeben und OK klicken) eine virtuelle Netzwerkkarte für den Cisco Client vorhanden sein. Rechtsklicke darauf und in den Eigenschaften sollte ein Freigabe Karteireiter existieren.

Erlaube die Freigabe der Netzwerkverbindung für andere Geräte im Netzwerk. Falls Du mehrere Netzwerke im Laptop hast, wähle das Netzwerk aus, an das Du die Xbox anschließt. Sei es nun eine zweite LAN Netzwerkkarte, oder WLAN.

Um das WLAN nutzen zu können, musst Du bei Windows 7/8 erst Virtual WiFi aktivieren:

Öffne als Administrator eine Eingabeaufforderung (Startbutton klicken, "cmd" eingeben, und bei der Eingabeaufforderung rechtsklicken und "als Administrator" ausführen). Gib dann folgenden Befehl ein:

netsh wlan set hostednetwork mode=allow ssid="WLAN NAME" key="SCHLÜSSELWORT" keyUsage=persistent

Ersetze "WLAN NAME" durch Deinen gewünschten Namen für das WLAN, und "SCHLÜSSELWORT" ist der Netzwerkschlüssel.

Bedenke, dass nicht jede WLAN Karte den Virtual WiFi Modus unterstützt. In dem Falle wieder eine neue (USB-)LAN Karte für den Laptop kaufen.

¡firedog say:

Server Response: -ERR [IN-USE] Unable to lock maildrop: Mailbox is locked by POP server


Yes, this is caused by your accessing the same mailbox from a phone-type device. If you configure the device for push mail, it will monopolize the POP server, and there can only be one POP3 connection at once. The account may also be provisioned for push mail at the server, which will effectively rule out any access from a humble POP3 client like Windows Live Mail.

The only thing I can suggest is disabling push mail everywhere you can, then set up the accounts on your mobile devices to use POP3, polling for new messages at different intervals from Windows Live Mail. It might take a day or two for the server to realize that you're not pushing mail any more; if it doesn't come right, talk to your mail service provider's tech support and ask them to reset the account for POP3 access.

As I noted earlier, the Windows.old folder is basically a backup of your old installation of Windows, which you can use to reinstall Windows if you decide to return to your previous installation. System Restore – System Restore keeps a large collection of System Restore Points in addition to storing them in your System Images of Windows 10. It was then I discovered that System Protection had been configured to use about 200 GBs of  disk space to keep System Restore Points.
Configuring System Protection settings
To configure System Protection, click Start, type: System Protection, hit Enter. Under Protection Settings, click the Configure button. A dialog will appear with various settings for configuring Restoration and Disk space usage. Under the Disk space usage tab, you will see the current amount of space in use by System Restore while the Max usage displays the amount allocated. 
Use the Max usage knob to configure how much disk space you want to allocate to System Protection. In the above screenshot, you will see that I am using 21.05 GBs for Restore Points, while I have allocated 22.36 GBs of disk space for System Protection. You also have the option of deleting all Restore Points which stores System settings and previous versions of files. I suggest you leave this, since you will be able to individually restore changes to files you are working on. So for instance, if you edited an image and would like to restore it to a previous version or original version of the file. 
Archive Large Files to an External Hard Disk
Storing large files on your local drive that you are not using doesn’t really make much sense these days. Moving infrequently used multimedia, like photos and videos, to an external drive can free up significant amounts of disk space.  Because of my Internet limitations, I still need to keep my music locally, so that means my iTunes library probably consumes the most space on my Surface Pro.
Archiving files to an external drive is easy. Before you do that though, you need to invest in an external drive. It’s fairly easy to find one that is really cheap with lots of space.  Once you have one, I recommend you check out our guide on how to partition a hard drive for macOS Time Machine and Windows 10 Backups.
