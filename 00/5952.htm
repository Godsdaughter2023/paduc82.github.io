There is a way that's a little tedious but not horrible.

Start by going to the Page Layout tab of the ribbon and clicking the little arrow in the bottom right corner of the Page Setup group to open the Page Setup dialog. In the Margins page of the dialog, set the Multiple Pages dropdown to "Mirror margins". It doesn't matter whether the left and right margins are the same or different.

Now the tedious part: In each red text box, right-click the border (not the internal text) and click More Layout Options. In the Layout dialog, click the option button next to "Book layout", and set the two dropdowns on that row to "Outside" and "Margin". Click OK to apply the settings.

If the document won't be edited in a way that might change the pagination, then technically you need to change only the ones on odd pages, but it's safer to do them all.

How can you have the page number of a document sequentially increase - Microsoft Community

bigzee say:

How can you have the page number of a document sequentially increase (single page document) based on the number of pages printed?

What is the 'field code' command in Microsoft office word 2010? IE. I print 50 pages of a single page document but they all numbered 1,2,3,4...etc.
macropod say:

Hi BigZee,

Here's one way:
• Add a custom document property named 'Counter' to your document, with the value 0
• At the location where you want the number to appear, insert a DOCPROPERTY field, for the 'Counter' property.
• Add the following macro to the document:

Sub MultiPrint()
Dim i As Long, j As Long
j = CLng(InputBox("How many copies to print?", "Numbered-copy Printout", 0))
For i = 1 To j
  With ThisDocument
    .CustomDocumentProperties("Counter") = i
    .Fields.Update
    .PrintOut
  End With
Next
End Sub

Now, when you want to print multiple copies, simply press Alt-F8, select MultiPrint then input the count when prompted.

To see how to install the macro, go to: http://www.gmayor.com/installing_macro.htm

Are you trying to change the notes text as seen in the notes pane beneath the slide in normal view or as seen in Presenter view.  Or as printed when you print speaker notes?

I don't think you can change the color in either of the first two, but you can change the font and size.  You format the text as you would any other text but to get it to APPEAR as you've formatted, do this:

Click the View ribbon
Click Outline View.
RIGHT-click anywhere in the outline pane
Click Show Text Formatting

Your font and size choices will now appear in Notes pane and in Presenter view.
