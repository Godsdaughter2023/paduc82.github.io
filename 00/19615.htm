hansv mvp say:

Arrow? By default, you select the cell to the right if you press Tab and the cell below the active cell if you press Enter.

To change what happens if you press Enter:

Select File > Options.

Click Advanced.

Make sure that the check box "After pressing Enter, move selection" is ticked.

Select the desired direction from the Direction dropdown.

Click Enter.
