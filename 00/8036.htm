carnet451 say: I also found a PowerPoint template who's default language was set to French, so every word I typed came up with a recommended spelling in French.  To fix, on the left, select all your slides, then switch to the "Review" tab.  From there, you should see a "Language" section, with two selections: A "Translate" drop-down and a "Language" drop-down.  To change the spell-check (master) language for all the slides you selected, click on the Language button, and click "Set Proofing Language".  It should come up with a small dialogue box that reads "French" for the default language (It might read "French (Canadian)").  Select English and hit the "Default" button in the bottom-left corner.  A pop-up will appear to ask you to confirm that you want this and all future presentations to be spell-checked in English.  Hit Yes, then on the previous dialogue, hit OK.  That should change the spell-check for it permanently.  Go ahead and save it afterwards just in case

In Outlook rules what does 'stop processing more rules' - Microsoft Community

mreds2k say:

In Outlook rules what does 'stop processing more rules' do?

Whenever I create a rule it automatically checks 'stop processing more rules'. What does this do? Is this normal?

a. user say:

Hi,

To better assist you, we suggest that you refer to the links below:

  • Inbox rules in Outlook.
  • Stop processing more rules in Outlook on the web.
  • Organize email by using inbox rules in Outlook.com or Outlook on the web.
  • After you configure one or more client-only message rules in Outlook, other message rules do not take effect.

By default, the option to stop processing more rules is turned on. With this option on, when a message comes in that meets the criteria for more than one rule, only the first rule will be applied. Without this setting, all rules that the message meets the criteria for are applied.

For example, without Stop processing more rules selected, if you have a rule to move all messages sent to a public group to a particular folder and another rule to move anything from your manager to another folder, and your manager then sends a message to that group, you'll find a copy of the message in both folders. If you want only the rule that moves messages from your manager to be applied, put that rule higher in the list than the rule that moves messages sent to the group, and then edit the first rule to add the option to stop processing more rules.

Let us know if you need further help.
