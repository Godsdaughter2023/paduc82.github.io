On that new laptop:

Click your Start Button, type cmd then right click Command Prompt and choose 'Run as Administrator'

slmgr.vbs /dlv

Close that dialog

Make sure Windows is activated with a digital licence


If both the above check out - the Windows 10 version installed is genuine!
