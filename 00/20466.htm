formatting currency value in a table - Microsoft Community

2bc say:

formatting currency value in a table

I am merging data in to a word document. the formatting of the data isn't imported. I am inserting the merged data which is a currency value and I can not format the data as currency.
hansv mvp say:

Right click the relevant merge field.

Select Toggle Field Codes from the context menu.

You should see something like

 

{ MERGEFIELD Amount }

 

or

 

{ MERGEFIELD Amount \* MERGEFORMAT }

 

where Amount is the name of the merge field. Change it to look like this:

 

{ MERGEFIELD Amount \# "$,0.00" }

 

You can use a different currency symbol of course. Press F9 to hide the field code and to update the result.
