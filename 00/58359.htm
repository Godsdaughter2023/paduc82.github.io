I somehow deleted the Accessories/System Tools (i.e., defrag., etc.) folder from my Program files.  I tried locating this program in Windows Download Center but was unable to do so.

Each of the System Tools has its own "exe" or "msc" program. Most of these programs are located in the C:\Windows\System32 folder.  Accessories\System Tools is a folder in your Start Menu, which contains shortcuts to the various System Tools.  It is most unlikely that you could have deleted the individual programs from Win XP - so I suspect you have merely deleted or moved the folder that contains the shortcuts.

The first thing to check is whether you have really deleted the "missing" folder .  If it is still in your Recycle bin, you can restore it to its proper location..

If the System Tools Folder is not in the Recycle bin, you might have moved it to an incorrect location.  To check this possibility, click on Start / Run... - and type explorer.exe in the "Open" field.  This will open Windows Explorer.  Expand the folder tree and scroll down to "C:\Documents and Settings\All Users\Start Menu\Programs\Accessories".

Your "System Tools" folder might be very close to its proper location, so look nearby for the "missing" folder.  If you find it, move it back to C:\Documents and Settings\All Users\Start Menu\Programs\Accessories\.

If you can't see the "System Tools" folder anywhere in Windows Explorer, you will need to search your entire hard drive.  Open the search menu and click on "All files and folders".  Type System Tools in the file name field.  Then, under Advanced Options, you should include system folders, hidden files and folders, and sub-folders in the search criteria.  Perform a full search for the missing file.

If the search locates the missing folder, move it back to C:\Documents and Settings\All Users\Start Menu\Programs\Accessories\.

If the System Tools  folder really is missing from your system, you could re-create the folder.  To do so, open Windows Explorer and scroll down to C:\Documents and Settings\All Users\Start Menu\Programs\Accessories.  Then, click on File / New and create a new sub-folder named System Tools.

Open the new, System Tools folder and create a separate shortcut for each of the system tools.  Each shortcut should point to the correct location, as follows:

System Information = "C:\Program Files\Common Files\Microsoft Shared\MSInfo\msinfo32.exe"
Files and Settings Transfer Wizard = "C:\Windows\system32\usmt\migwiz.exe"
Character Map = "C:\Windows\system32\charmap.exe"
Backup = "C:\Windows\system32\ntbackup.exe"
System Restore = "C:\Windows\system32\restore\rstrui.exe"
Disk Defragmenter = "C:\Windows\system32\dfrg.msc"
Disk Cleanup = "C:\Windows\system32\cleanmgr.exe"

     l.     Now you can go to the General tab, and Print a Test Page.

      So is there are any way the free space in E: be used to extend C:?

      john_barnett say: Hi Daniel,

      To extend any partition you need either unallocated space or a spare partition right next to it, i.e, to the right of the C: drive; and even then I would trust Windows Vista disk managment tools to extend the drive, I would much prefer third party partitioning software such as Paragon Disk Manager Suite (see this review on my website of the 2008 edition); http://www.winuser.co.uk/reviews/paragon_hard_disk_manager_suite_2008.html

      What you basically need to do is to delete the E partition (assuming there is nothing on it, if there is then thins needs backing up to a removable hard drive). This will now give you unallocated space. Next create a temporary partition andf copy the conents of youe D: partition to it (remember that your D: drive is currently FAT32 so make sure the new tempoery partiton you create is also FAT32) Once the D: drive is empty delete the D: drive. You shoud now have unallocated space right next to your C: partition. Now using something like Paragon Disk manager you should be able to extend your C: drive into, what originally, was your D: drive. Your temporary partition you used to copy your D: drive will then appear next to the C: partition. It may not have the same drive letter as before, but that can be changed to bring it back to D:, even though it is somewhat fiddly to do so.

      Either way you are going to have to move the D: partition in order to accomplish what you want to do.
