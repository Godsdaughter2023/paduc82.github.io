    I am working on a mileage log work sheet. insted of listing each day of the month in a coluum. is there an easier way to list all the days of the month in a coluum. and how do I do that.

    THANKS

    jeeped say:

    ... is there an easier way to list all the days of the month in a coluum. and how do I do that ...


    You've posted in Feedback so I moved this inquiry to Office, Excel and assumed you have Excel 2007 or 2010. For a series of days-of-the-month in every second column,
    1. Type 1/1/12 in A1.
    2. Put this formula in A2,
         =IF(AND(LEN(A1),IFERROR(MONTH(A1+1)=MONTH(A1),FALSE)),A1+1,"")
    3. Copy or Fill down to A31
    4. Put this formula in C1,
          =EDATE(A1,1)
    5. Copy A2:A31 to C2:C31
    6. Copy C1:C31 to E1:E31, G1:G31, etc
    7. Format the date cells with any of the supplied Date formats or your own custom number format

    IF function
    LEN function
    IFERROR function
    MONTH function
    EDATE function
