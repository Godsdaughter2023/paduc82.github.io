a. user say:

Hi Mark,

Thank you for posting your query in Microsoft Community. Let me assist you on this.

We regret for the inconvenience caused to you.

I suggest you to Enable the build in administrator account and then set your user account as Administrator Account. Follow the below steps to do so:

Follow the below steps to enable build in administrator account.

1. Right click on the Start Menu and select Command prompt (admin).

2. On the command line type: net user administrator /activate:yes and hit Enter on your keyboard. 

This will enable your build in Administrator account. Log out and then restart your computer and login with the administrator account. Once you are in the Build in administrator account you can make your primary user account as administrator account.

1. Right click on the Start Menu and select Control Panel.

2. Select User accounts.

3. Now Select the option Manage another account and select the primary user account.

4. Now select the option Change the account type and click on Administrator and save it.

Now sign out of the build in administrator account and login with your primary account. Now disable the Build in administrator account by following steps given below:

1. Right click on the start menu and select Command prompt (admin).

2. On the command line type: net user administrator /activate:no and hit Enter on your keyboard. 

This will disable the build in administrator account.

Hope the above information was helpful. If you need further assistance, please do let us know and we would be happy to help you.

Thank you.

ken blake - microsoft mvp say:
My Cannon Lide 2009 scanner wil not work with Windows 7.   It ran fine with XP.   What is wrong?

 

Whenever you change to a new version of Windows, you need to install drivers (software that provides the interface between the hardware and Windows) for printers, scanners, video cards, etc., that have been written for that new version of Windows.  And also note that you need different drivers for a 32-bit version of Windows and a 64-bit version.

 

It's up to the manufacturer of the hardware to write and make available for download those drivers. Especially if your hardware is older, it may  well be that the manufacturer has decided that it's not worth his time and effort to develop drivers for hardware that he is no longer selling.

 

 

Check with the manufacturer or on their web site to see if a driver exists. If it doesn't, you're out of luck. Also check their web sites in other countries, if there are any; sometimes you can find the driver you need that way.

 

So in the future, before you do such an upgrade, be sure that you've done the appropriate research to find out what drivers don't exist, and therefore what hardware you will have to replace if you upgrade.

Windows 8 takes a long time to authenticate showing the Welcome screen - Microsoft Community

fexpro say:

Windows 8 takes a long time to authenticate showing the Welcome screen when an incorrect password is entered to login.

Original Title: Wrong "Welcome"!!

 

Entering wrong password for a couple of times at windows logon screen shows "Welcome" for a long time, instead of showing any error message(experienced in windows 7 and 8). Although, usual warning note appears after that.

rajesh govind say:

 

Hi,

 

 

First of all let me give you a brief idea on how Windows 8 authenticates its users when they login to it using their Microsoft Account. In Windows 8, when you login to it for the first time using a Microsoft Account, you need the Internet Connection. At this time it creates a local cache on your computer to store the account login credentials. Thus, after this, it is not necessary to have the Internet Connection to login to the computer using the Microsoft Account.

 

Now, when the computer is connected to the Internet and you enter an incorrect password for your Microsoft Account to login to your computer, then it first verify the password that you have entered with the credentials that are locally stored on your computer. If it finds that the password is incorrect, then it checks for the Internet Connection and if available, it will try to connect to the Microsoft Account Server to recheck the password. In this case, if you have a slow connection, then it may take some time for the computer to get connected with the server. It may also take a long time if the server is busy or is not reachable.

 

This might be the reason why it remains on the “Welcome” screen for a long time when you enter the incorrect login credentials for Windows 8.

 

Hope this information is helpful.
