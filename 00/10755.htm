Text flow in document - Microsoft Community

icydoc say:

Text flow in document

how do I create a document with any number of pages and two columns per page, where the text in the left column flows to the left column on the next page and the text in the right column flows to the right column on the next page?
daniel g. grau say:
Try using a two-column table.* The cells will spill over onto the next page.

Just make sure to allow the rows (which would actually be a single cell) to break across pages. To do this:
  • Select the table with Table > Select >Table
  • Select Table > Properties
  • Click on the "Row" tab at the top
  • Place a checkmark next to "Allow row to break across pages."

If you need to align two titles (one on the left column and another on the right column), insert a new row (as oppossed to inserting Carriage Returns).

Caveat: You may find scrolling up and down difficult in Draft View, with Word jerking you to the top of the table (page 1). If this happens, use Print Layout View.



* Important detail: The table has to be in the main text flow (i.e., not floating). This is controlled through Table > Table Properties > Text Wrapping > None

bob jones aka: cybertaz say:

In Word, "Columns" refers to newspaper-type or "snaking" columns that flow from left to right. Side-by-side can best be done by creating a 2-column Table. I'm not sure what you mean by "any number of pages" but it's best to not allow a single row to run on for page after page.

 

That type of layout can also be done using linked text boxes, but that's more work & can lead to an unstable document.

Bob J.
