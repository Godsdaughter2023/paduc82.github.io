The Windows Store cache may get damaged due to corruption. You can try to re-register the Windows Store apps to resolve this. To do that, follow these steps:

  1. Press Windows Key + Q, type powershell and select Windows PowerShell from the results, right-click on it and choose Run as administrator.
  2. Type following command and hit Enter.
