jllatham say:

How are you doing the sort now?  From the keyboard or have you got a macro to do it?

 

If you have a macro, and post the code here, we could help change it so that the insert would be done automatically right after the sort completes.

 

Here is a 'stand-alone' routine that would do the job for you after the sort is completed.  You'll need to change the values of the two Const entries to match things on the sheet itself.   Afterthought: if you already have a macro to do the sorting, you could add the code below into the same module with that code and then just before the sorting code's Exit Sub statement you could enter a one-liner:

 AddSeparatorRows

which would call this routine after the sorting has been done, without further action on your part.

 

You can find instructions on how to put the code into a regular code module here:

http://www.contextures.com/xlvba01.html#videoreg  Then just run the macro from

Tools --> Macro --> Macros in Excel 2003 when needed.

 

Sub AddSeparatorRows()
  'change these Const values as required
  'the column that DEPARTMENT info is in
  Const deptCol = "A"
  'the first row that has department/name data
  Const firstDataRow = 2
  'that's all you need to change
  Dim lastRow As Long
  Dim rowPtr As Long
 
  lastRow = Range(deptCol & Rows.Count).End(xlUp).Row
  Application.ScreenUpdating = False
  For rowPtr = lastRow To (firstDataRow + 1) Step -1
    If Range(deptCol & rowPtr - 1) <> Range(deptCol & rowPtr) Then
      Range(deptCol & rowPtr).EntireRow.Insert
    End If
  Next
End Sub

bill manville say:

As I said:

Without seeing the layout I can't devise the formula for you but you could compute the best fit data number in one cell (BestFitNumber) using =MATCH(MIN(SumOfRSquaredValues),SumOfRSquaredValues,0)
and then use it as an offset from the first series to create the 18th:
=OFFSET(FirstSeriesCell,BestFitNumber-1,0)

You would use that 18th series, with the points selected using the OFFSET function, as the source of the best fit chart.

myveryownself say: Maybe something like this would help getting started.

In AF2 put
   =IF(OR($U2>EOMONTH(AF$1,0),$V2<AF$1),"",
          $AE2*(MIN(EOMONTH(AF$1,0),$V2)-MAX(AF$1,$U2)+1)/
                  IF(MOD(YEAR($AF$1),4)=0,366,365))
