Feedback - W10 is more like W8.2 at this stage; too much Modern on - Microsoft Community

wazhai say:

Feedback - W10 is more like W8.2 at this stage; too much Modern on the desktop.

I tried Windows 10 on a virtual PC and I am not impressed. It seems more like Windows 8.2 than a completely new version of Windows. There will need to be very big changes and improvements over what we currently have to convince me to upgrade from Windows 7. There is still too much Metro/Modern on the desktop.

Here are some of my thoughts:

* There is as much inconsistency as in W8 because of the two different UIs. There is a big divide between Modern apps and desktop programs. Title bars, window controls and borders are not consistent between the two. Some functionality can be accessed from both Modern and desktop.

* Most settings can be changed in the Control Panel or the Settings app, while some are exclusive to each. There should be a centralized place for everything.
* The taskbar is incredibly inconsistent. Clicking on the network icon, the charms networks sidebar is brought up. "View Connection settings" sends you to the Metro setting app instead of the Control Panel network settings. The Windows 7 style of taskbar pop-ups (Clock/Calendar, Volume, Action Center) is mixed with Modern sidebars (Network) and Modern pop-ups (Keyboard layout switcher).
* Modern apps use a design and control paradigm suited to touch screen. The fonts are too big, too much screen space is wasted, navigating them is not intuitive with a keyboard and mouse. The three dots menu offers essential functions, but can also lead to dead ends and is obfuscated. All options are always offered in this menu despite that Play and Share don't do anything in a lot of apps.
* An example why I think Modern is unusable on a traditional desktop PC or laptop without a touchscreen - Modern Skype that comes pre-installed with W10. When you open it up, there is a little tutorial that tells you to "swipe" and "tap" stuff. This version is inferior to the desktop version in every way.
* Right now Modern on the desktop is like a smartphone/tablet OS emulator that runs mobile apps. I don't see why anyone on a computer with KB/M and no touchscreen would use the News, Weather, Facebook, Skype, OneNote, etc. Modern apps instead of an equivalent website or the desktop version of the program.
* The flat UI design and lack of window borders makes everything blend together. Buttons and program controls don't stand out.
* The option to use a local user account should not be obfuscated behind a failed MS account login or offered as a barely noticeable text link that blends in with the background while registering for a MS account.
