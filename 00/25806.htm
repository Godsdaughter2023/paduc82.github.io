barry xie msft say:

Hi Keith,

To change the shared mailbox’s profile picture, we need to use Windows PowerShell. Please perform the following steps to change the shared mailbox’s profile picture:

1. Connect to Exchange Online PowerShell with an Office 365 admin account.

2. Run this commend:

Set-UserPhoto “John Smith” -PictureData ([System.IO.File]::ReadAllBytes(“C:\Users\Administrator\Desktop\JohnSmith.jpg”))

Note: Please change “John Smith” to your shared mailbox’s display name.
