craig lieboff say:

arrow down to page down or arrow up to page up redefinition

Is there any way to change the input for word 2016 when using arrow up/down to page up/down on windows 10.  it seems most ereaders and other platforms support this, but not word.

any insight would be greatly appreciated.

rohn007 say:

The arrow keys are "system" keys. They are hard to reassign functions.

I created this macro, then used the Customize Keyboard dialog to assign the down arrow to it (as described below).  It appeared to work, but when I tried to use it, the macro was not triggered.

Sub Adown()
'
' Move down one "Page" keypress
' created by: Rohn007
' date created: 2016 10 12
'
    Selection.MoveDown Unit:=wdScreen, Count:=1
End Sub

Save it in your NORMAL.DOTM

Then go to File menu > Options command > Customize Ribbon option >

Keyboard Shortcuts Customize ... button.

In the Customize Keyboard dialog, in Categories select Macros, and select the ADown (or what ever name you give it).

Click in the "Press new shortcut key" and then press the down button.  This should assign the macro to the down key.

Close the dialog and give it a try.

paul van der heu say:

 

Your contacts woudl normally be synced to your Microsoft account. If you log in to http://accounts.live.com you will find them there. Normally changing a battery will not delete any data and if you need to reset you will get your contacts back once you enter your microsoft account during te device first use (DFU) setup.

 

Messages, (account)settings and your applist is backed up to the same Microsoft account if your have set  up the options for this in Settings > backup and as with contacts they will restore when you enter the Microsoft account during DFU.

 

If the SIM you use does not have the settings for cellular data and the access point settings does not have the settings for your carrier you will need to activate Wi-Fi before you set up the Microsoft account. At the point you are asked for the account details during DFU press the windows key, goto Settings > Wi-Fi and activate it. Once connected go back to the DFU screen and enter the details.

 

Normally documents usedin the Office Hub will be on Onedrive or you should (IMO) always keep a backup on yoru local PC so you can write them back.

 

If you have any other questions let us know..
