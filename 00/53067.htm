shenan stanley say:

That's not exact.

I don't believe you are not speaking of the "Control Panel\System and Security\System", so... after reading what you put, I believe you are actually (this is what is meant by "exactly") looking at the "Control Panel\All Control Panel Items\Display\Screen Resolution" and clicking on the "Advanced Settings" text on the screen to bring up the Properties of your display, specifically the "Adapter" tab.

That window normally has two sections - truthfully, not sure what it has if you have multiple video devices in your computer - although it may show the active one decided by which monitor you clicked on before pressing the "Advanced Settings" text.  In any case the two sections usually are "Adapter type" and "Adapter Information".

The "Adapter Type" usually gives the specific Adapter Make/Model presented to the operating system throught he drivers.

The "Adapter Information" is what I believe you have presented, in part at least, here.

What I would like you to do is present the entirety of the text in both sections here.

For example, I only have one video card in my laptop.  So here are the sections of that:

Adapter Type

NVIDIA Quadro FX 370M

Adapter Information

Chip Type: Quadro FX 370M
DAC Type: Integrated RAMDAC
Adapter String: Quadro FX 370M
BIOS Information: Version 62.98.43.0.2
Total Available Graphics Memory: 3839 MB
Dedicated Video Memory: 256 MB
System Video Memory: 0 MB
Shared System Memory: 3583 MB

 

Given that it is very easy to answer your question.

Your video adapter has 64MB of Dedicated Video Memory - memory that belongs to your video device that is not used for/by anything else.  Just like mine (above) has 256MB of Dedicated Video Memory.

That means the video device will use its memory first and ONLY when (if) it has used up its own memory will it dip into the "Shared System Memory" pool - which is nothing more than the memory (RAM) that is installed in your computer (or some portion of it.)

Above I have 3583MB of Shared System Memory it can use - and if you add those two numbers up, you get the Total Available Graphics Memory number of 3839 in my case.

In your case, you have 1696MB of Total Available Graphics Memory and 64MB of Dedicated Video Memory, so you must have listed 1632MB of Shared System Memory.

Essentially - if you were to go down and purchase whatever video adapter that is reporting that to you - it would be advertised as a 64MB video card.

Error: 0x8078002A occurs when backing up to a 2.5 or 3.0 TB drive in Windows 7 Backup and Restore
Answer ID 6618 | Last Updated 05/19/2011    -->

Problem:
When using Windows 7 Backup and Restore with a 2.5 TB or 3.0 TB drive, the backup fails with an error code of (0x8078002A).

Cause:
This problem is caused by an issue with the Windows 7 Backup and Restore utility when using drives with 4K Logical sector sizes, which are found on WD 2.5 TB and larger external drives. This is different from an "Advanced Format" (AFD) drive which use 4K physical sector sizes. AFD drives should not cause this problem.

Solution:
It is still possible to use a 2.5 TB drive or larger with Windows 7 Backup and Restore, as long as you don't need to use the drive with Windows XP. Using the WD Quick Formatter tool, select the Factory Default configuration to reformat the drive.

Warning: This tool will erase all data on your drive, so take the necessary precautions.

After formatting the drive to Factory Default mode, the drive will be usable in Windows Vista and Windows 7. However, the drive will not be recognized in Windows XP. This is the only way to get Windows 7 Backup and Restore to work correctly.
