graham mayor say:

You need something like

Dim oRng As Range
Dim newDoc As Document
Set oRng = ActiveDocument.Range
Set newDoc = Documents.Add
With oRng.Find
    .Text = "[a-z0-9_.]{2,}\@[a-z0-9_.]{2,}"
    .Replacement.Text = "^&"
    .ClearFormatting
    .Replacement.ClearFormatting
    .MatchWildcards = True
    Do While .Execute(Forward:=True)
        newDoc.Range.InsertAfter oRng & vbCr
    Loop
End With
newDoc.Paragraphs.Last.Range.Delete

jeeped say:

You could change the original value to use extended UNICODE characters instead of inner cell formatting. For instance, your PbSO4 could be written as PbSO₄using the U+2084 character from the Callibri set. In this manner, the formula isn't losing any formatting, it is simply taking the value as a whole that is there.

You can find a Character Map program within Accessories, or simply click Start and type charmap<enter>. The 0-9 superscripts and subscripts are almost all the way to the end. Look in the lower right for the character code (e.g. U+2084). Click Select, then Copy to put your selection on the clipboard for pasting elsewhere.
