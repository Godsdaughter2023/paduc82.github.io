        #If VBA7 Then

glenprouty say:

Rakesh,

Dave's advice is right on, however, you will need the full path of the App that you

want Controlled Folder Access to pass. If you don't have that information, you can

find it in the event viewer.

Navigate through the viewer Applications and Services >Microsoft >Windows >

Windows Defender >Operational. In the log, find Event ID 1123 or 1127. Right

click on the log entry, and select Properties. In the properties panel, you can

find the full path that you will need.

If you do not see ID 1123 or 1127 easily, you can filter the list by clicking on

"Create Custom View" at the right side of the screen.  Click on <All Event IDs>

and enter 1123,1127 and click OK. Click OK on the next screen, and these two

Event IDs will be displayed. Find what you need as described above.

With this information in hand, return to "Controlled Folder Access", and allow

the App that you want to pass.  Leaving CFA active is a good idea.
