Hi, if you are trying to match a criteria in SUMIFS function, then use below criteria as per your need:

(1) if you are trying to use Not Equal To a Cell Reference:

"<>"&C2

(2) if you are trying to Not Equal To a Number:

(3) if you are trying to Not Equal To a Number which is Text in Lookup range:

"<>"&TEXT(1234,"0")

amit tandon say:

To differentiate text you may use SUMPRODUCT, as below:

=SUMPRODUCT((A$2:A$25<>D2)*(B$2:B$25))

This will treat D2 as number if D2 has a number but if D2 has the text "1234" then it will be treated as text.

or

To treat as text:

=SUMPRODUCT((A$2:A$25<>"1234")*(B$2:B$25))
