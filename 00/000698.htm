I had a look at your file and I think that you can probably modify the following example code that copies the Conditional Format Displayed color from Sheet1 to the corresponding cell in Sheet2.

Note that DisplayFormat was only introduced in xl2010 and will not work in earlier versions of xl.

Sub CopyConditFormatColors()
    Dim ws1 As Worksheet
    Dim ws2 As Worksheet
    Dim cel As Range
   
    Set ws1 = Worksheets("Sheet1")  'Worksheet with Conditional format
    Set ws2 = Worksheets("Sheet2")  'Worksheet with the Copy of Sheet1
    For Each cel In ws2.UsedRange   'UsedRange in the worksheet with Copy of Sheet1
        cel.Interior.Color = ws1.Range(cel.Address).DisplayFormat.Interior.Color
    Next cel
End Sub

The code below (Test sub and Function) can be used to count the displayed colors in Conditional Formatting.

To find the number for the displayed color select a cell with the color and then in the Immediate window enter the following and it will return the numeric for the color.

? ActiveCell.DisplayFormat.Interior.Color

Sub TestCountConditFormats()
    Dim ws1 As Worksheet
    Dim sglColor As Single
   
    Set ws1 = Worksheets("Sheet1")
    sglColor = 49407
   
    MsgBox CountConditFormats(ws1.Range("A2:F2"), sglColor)
   
End Sub


Function CountConditFormats(rng As Range, sglCol As Single)
    Dim rngCel As Range
    Dim lngCount As Long
   
    lngCount = 0    'Good programming to initialize
    For Each rngCel In rng
        If rngCel.DisplayFormat.Interior.Color = sglCol Then
            lngCount = lngCount + 1
        End If
    Next rngCel
    CountConditFormats = lngCount
End Function
