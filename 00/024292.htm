trevor rabey say:

Do you have a problem?

When you say "actual work", do you mean people x hours = people hours.

This is what MSP (and everyone) calls work. In MSP, if you assign a person at 100% to a task with duration = 1 day, where 1 day has 8 hours, then work = 8 hours. MSP has a field % work complete = actual work/work where work = actual work + remaining work.

Three things only:

Duration: actual, remaining, total, %

Work: actual, remaining, total, %

Cost: actual, remaining, total, but no % although it is easy to make one from a spare field if required

You cannot force the summary % complete (duration) or % work complete to be what you want them to be because they are whatever they are calculated to be from the % of the tasks under each summary.

hansv mvp say:

OK, set the default formatting to blue fill.

Remove the existing conditional formatting rules and create new ones:

On the Home tab of the ribbon, click Conditional Formatting > Manage Rules...

Click New Rule...

Select "Format only cells that contain".

Leave the first dropdown set to "Cell Value".

Select "greater than" in the second dropdown.

Enter =AB32 in the box next to it.

Click Format...

Activate the Fill tab.

Select amber.

Click OK to return to the New Formatting Rule dialog.

Click OK to return to the Conditional Formatting Rule Manager dialog.

Click New Rule...

Select "Format only cells that contain".

Leave the first dropdown set to "Cell Value".

Select "greater than or equal to" in the second dropdown.

Enter =M8+AB32 in the box next to it.

Click Format...

Select red.

Click OK to return to the New Formatting Rule dialog.

Click OK to return to the Conditional Formatting Rule Manager dialog.

Click OK to close the dialog.
