garrie 5150 say:

If you’re receiving an invalid error usually that means that one of the letters or digits in the code is incorrect. Codes are 25-digit alphanumeric strings but sometimes some numbers or letters may look similar.

For example:

  • The number "8" may resemble the uppercase letter "B."
  • Two instances of the uppercase letter "V" may resemble the uppercase letter "W."
  • The number "0" may resemble the uppercase letter "O."

If it still says the codes are invalid, you’ll want to reach out to phone support via support.xbox.com/contact-us.

janeane cru say:

Thank you for getting back to us. In this case, we suggest that you enable the hidden administrator account by following these steps:

  1. Click on Start.
  2. Type cmd, and right-click on it.
  3. Select Run as administrator.
  4. Type this command line, and then press Enter:
    net user administrator /active:yes
  5. You should see a message that the command completed successfully. Log out, and you’ll now see the Administrator account as a choice.
    Note: There's no password for the hidden administrator account.
  6. Access the hidden administrator account and check if the thumbnails of your picture files are showing.

If the issue persists, we suggest that you use System File Checker tool. This tool will help us to determine if there are system file corruption that caused the issue to happen.

After doing the above steps, you must disable the hidden administrator account again because we are just using it for troubleshooting:

  1. Click on Start.
  2. Type cmd, and right-click on it.
  3. Select Run as administrator.
  4. Type this command line, and then press Enter:
    net user administrator /active:no
  5. The administrator account will now be disabled, and shouldn’t show up on the login screen anymore.
