Biggest problem when reference is made to contacts and address books is "terminology"

 

The definition of an Outlook AddressBook is a specialized "view" representing contact folders that have been designated to be included when the "Show folder as an email address book" is checked

(right click a contact folder --> Properties --> Address Book tab)

 

In short, you have one Outlook AddressBook where the contents of one contact folder (which is a member of the AB) is set to be displayed when the AB is opened. Which contact folder is displayed first is a user controlled via the AB options (open the AB --> Tools --> Options)

poppy yang msft say:

Hi PattiSummerville,

 

How did you configure your account on the Outlook client, via POP/IMAP or Exchange?

 

This option is available only if your mail account is a Microsoft Exchange account.
If you are not using a Microsoft Exchange account, the Out of Office option will be dimmed.

 

You can still set up out of office replies for a POP or IMAP account by setting up a mail rule to automatically reply to new email messages.

How to set up an out of office rule for POP and IMAP accounts

To create a new rule in Outlook for Mac 2011 that automatically sends custom responses to all messages that you receive while you are out of the office, follow these steps:

  1. In Outlook, click Tools, and then select Rules.
  2. In the Rules window, select POP or IMAP from the list.
  3. Click the plus [+] sign.
  4. In the Rule name field, type Out of Office.
  5. In the If section, in the drop-down menu next to Execute, select If all criteria are met.
  6. In the If section, make sure that All messages is selected in the second drop-down menu.
  7. In the Then section, select Reply from the first drop-down menu.
  8. Click Reply Text, and then type your out of office message.
  9. Click to clear the Do not apply other rules to messages that meet these criteria check box.
  10. Click to select the Enabled box.
  11. Click OK.
  12. Close the window.
