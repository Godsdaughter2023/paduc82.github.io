jay freedman say:

For this, don't use the Font dialog at all. The control you want is in the Paragraph dialog. To get there, select all the text of the sign, right-click, and click Paragraph.

On the Indents and Spacing tab of the Paragraph dialog, under the Spacing heading, set the "Line Spacing" box to Exactly and the "At" box to some number of points, approximately the point size of the font you're using or slightly less. You'll have to experiment to find the number that reduces the spacing between lines but doesn't cut off the tops of the letters. The number also depends on the particular font, because different fonts have different letter heights for the same nominal point size. For example, in 72 pt Calibri font, a line spacing of Exactly 60 pt looks reasonable; but 72 pt Lucida Sans Typewriter requires line spacing of Exactly 66 pt or more.

Also set the Before and After boxes in the same dialog to 0 pt.

When you revert WU writes the build number reverted from in the registry so that it will not offer it to you again (the assumption is that you rejected the build) but WU will resume offering builds with the next build to flight.

john mcghie say:
Hi Steve:

The problem is obvious: your screen is too big.  Send that computer to me, and you can have this one :-)

Seriously:  A Word document will open at the percentage Zoom it was last saved at.  The zoom ratio is saved in each document, not in Word, so each document will open at the size it was last saved.

If you set the zoom to whatever size you want, make a change to the document (add and remove a space) then Save it, it will open next time with the zoom set to your desired size and the window sized to hold that.  If you do this to your Normal template, each new document you create will get the new size.  Old documents will not be affected unless you give them each the "treatment".

You need to make a change to the document to force it to save, otherwise the save command is ignored because Word thinks the document has not changed.

Hope this helps
