jetta48 say:

Hi


A shortcut is just a pointer to the executable file for a program. It is not the program itself. Deleting a shortcut does not delete the associated program. Desktop shortcuts normally have a small curled arrow in the corner of the icon to identify them.


Right-clicking and selecting delete is one method of deleting a shortcut. Selecting a shortcut with a single-left click to highlight it then hitting the Delete key on your keyboard also works (you can select multiple shortcuts if you wish for this latter method).


If you are not logged in as an administrator then it is normal for you to get asked for an administrator password when deleting shortcuts for installed programs.

New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee.

OEM computer manufacturers are required to provide at least one way for you to reinstall Windows if you buy a computer with Windows preinstalled - your computer should have included either a restore partition on your hard drive, or Windows install/restore disks. ﻿Most computer manufacturers also have a feature to make your own install disks.

If the restore partition is still there and was not removed, it will reinstall Windows 7. There is usually a prompt when you first start the computer telling you what key to press to start the restore﻿ - check your manual if you're not sure how to.
