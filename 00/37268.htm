    Hi,

    This usually happens if the services related to media players are not working or stopped and the player’s settings are changed. Many users have fixed this issue by starting their respective services and restoring the settings to default.

    Try the following steps and check the result.
    Step 1: Start media center and media player services.
    1. Login to the computer as an administrator.
    2. Click Start, type services.msc and hit enter.
    3. From the list of items, double click on all the services those start with name "Windows Media Center" and make sure the startup type should be Automatic and the status Started.
    4. Double click on Windows Media Player Network Sharing Service and click Start and make sure the startup type should be Automatic.

    Step 2: Run sfc scan to fix missing and corrupt system files.
    1. Open an elevated command prompt. To do this, click Start, click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator.
    If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.
    2. Type the following command, and then press ENTER: sfc /scannow

    The sfc /scannow command scans all protected system files and replaces incorrect versions with correct Microsoft versions.
