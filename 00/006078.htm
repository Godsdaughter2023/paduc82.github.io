jllatham say:

One hidden slow-down area is Conditional Formatting.  Do you have a good bit of that going on in the worksheets?  The conditional formatting rules are actually formulas in one way or another, and they are what is known as "volatile", which means that ANY change made anywhere on a sheet causes all of the conditional rules to be 'tested' again even if the change was made to a cell that has nothing to do with any of them.

Another kind of hidden slowdown, if used in large numbers, are references to entire columns in formulas like MATCH() or VLOOKUP().  This feature is only available in Excel 2007 and later.  In 2003 and earlier you have to specify the row numbers, Like this:

Excel 2003 & earlier:

=VLOOKUP(A1,G2:R455,3,False)

which can legitimately be written as

=VLOOKUP(A1,G:R,3,False)

in Excel 2007/2010.  The problem is that each of those may spend lots of time looking at empty cells below the last used cell in a column when a match wasn't found in the used area.

Those are two areas I can think of.  I'm sure others will come along and suggest other areas to look at.  Especially considering you're discussing Office Web Apps, which I actually don't use.

If you want to try printing comments "in line with text," you can test the macro below. Ensure that you do not save the document after you have run the macro, unless you want to save the inline comments with the document.

Sub InsertCommentTextInline()
'Macro created by MVP Stefan Blom, March 2017
 Dim c As Comment
For Each c In ActiveDocument.Comments
Set myrange = c.Reference.Duplicate
With myrange
.InsertAfter "["
.InsertAfter c.Range.Text
.InsertAfter "]"
End With
With myrange.Font
.Bold = True   'Make comment text stand out
End With
Next c
End Sub

Hello,

There are several possible reasons why this is happening and most maybe due to hardware issues. We would like to clarify though if you're referring to your SIM card or SD card?

If your SIM is not being read by your phone, can you try to insert it in another handset to see if it is being detected and if there is a network signal? If it is not detected then more likely, you would need to replace it.

For both cases, trying the card in another device is one reason to check if it is an issue with the card or the device. If the other device reads the card, there might be a problem with your handset, more specifically, a problem with the card reader.
