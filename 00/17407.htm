PowerPoint stuck in endless loop of "PowerPoint found a problem - Microsoft Community

alex long seattle say:

PowerPoint stuck in endless loop of "PowerPoint found a problem with content in <filename> ... Repair" Loop (Mac)

So I didn't find a good answer to this and had to figure it out.

There are two separate issues that have to do with this (at least on the Mac version of office).  Note I had this problem with both Office 2016 and Office 2019.  

The two separate issues:

1. You have either a corrupt file or inappropriate permissions.  The most effective thing that I have found so far was to move the file to a different location (if you want to get fancy bring up a terminal and do a quick chmod / chown to see if you can fix it, but frankly that can get a little dangerous if you don't know what you are doing)

2. The autosave file can't be read of there is a problem.  It was a nagging problem for me, and eventually I got tired of dealing with it.  Going to command line I was able to track down the directory (for a Mac) on where AutoCorrect was kept.  I moved the files in that location out and it was fixed.

So for those looking in Office 2019 where those files are kept:

/Users/<username>/Library/Containers/com.microsoft.Powerpoint/Data/

I moved those files out of the way and the problem was fixed.

I tried to find a summary thread on this, and got nowhere.
