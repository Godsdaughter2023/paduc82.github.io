Ensure that you create a backup of your workbook before using the code example below in case it is not doing exactly as you required.

I have assumed that your list of abbreviations are on Sheet1. If not then edit the code where indicated with the comment. The code loops through the words and then references the abbreviation in the column to the right of th word with Offset function.

Further, I think you mean that the Text where the replacements are to occur are on Sheet2, Column A. If not correct and you don't know how to edit the code then please get back to me with information of the range where the text is located.

The Find and Replace is coded so as NOT be case sensitive.

Because you have indicated you are new to Macros, I will provide the following guidelines to install the code in case you require it.

To install the VBA code:

  1. Alt and F11 to open the VBA editor window
  2. In the VBA editor select menu item Insert -> Module
  3. Copy the VBA code below and paste into the VBA editor.
  4. Close the VBA editor (Cross very top right of VBA editor window)
  5. Save the workbook as Macro enabled.
  6. Ensure macros are enabled as follows.
  7. If Developer ribbon not displaying then right click anywhere in one of the ribbons and select "Customize the ribbon".
  8. On the right side of the dialog check the box against "Developer" and click OK.
  9. Select Developer ribbon.
  10. Select Macro Security (In the Code block at left end of Developer ribbon)
  11. The dialog that displays should default to Macros in left column.
  12. Select required security option button. (Option to "Disable all macros with notification" should be OK.)
  13. Click OK to close the dialog.

Copy all of the following code into the VBA editor.

Sub SubstituteAbbrev()

    Dim wsText As Worksheet
    Dim wsAbbrev As Worksheet
    Dim rngWords As Range
    Dim rcel As Range
    Dim rngToFind As Range
   
   
    Set wsAbbrev = Worksheets("Sheet1")     'Edit "Sheet1 to name of sheet with the Word/Abbrev table
    Set wsText = Worksheets("Sheet2")       'Edit "Sheet2 to name of sheet with the text requiring replacements.
   
   
    With wsAbbrev
        'Following line assumes Words/Abbrev list starts row 2 (after the column header)
        'Column for words is column "A". (If not, then edit both instances of "A" to the correct column)
        Set rngWords = .Range(.Cells(2, "A"), .Cells(.Rows.Count, "A").End(xlUp))
    End With
   
    'Following code between With and End With inserts a column to right of column A
    'and then copies column A and Pastes to Column B
    With wsText
        Application.CutCopyMode = False
       
        'Following line can be deleted if column B is currently blank
        'and it is not necessary to insert a column
        .Columns("B:B").Insert Shift:=xlToRight, CopyOrigin:=xlFormatFromLeftOrAbove
       
        Application.CutCopyMode = False
        .Columns("A:A").Copy Destination:=.Cells(1, "B")
    End With
   
    With wsText.Range("B:B")
        For Each rcel In rngWords
            Set rngToFind = .Find(What:=rcel.Value, _
                            LookIn:=xlFormulas, _
                            LookAt:=xlPart, _
                            SearchOrder:=xlByRows, _
                            SearchDirection:=xlNext, _
                            MatchCase:=False, _
                            SearchFormat:=False)
                           
            If Not rngToFind Is Nothing Then
                Do
                    rngToFind.Value = Replace(rngToFind, rcel.Value, rcel.Offset(0, 1), 1, -1, vbTextCompare)
                    Set rngToFind = .FindNext(rngToFind)
                    If rngToFind Is Nothing Then Exit Do
                Loop While Not rngToFind Is Nothing
            End If
        Next rcel
    End With
   
End Sub

you can drag music from the zune software collection to the zune device icon to force that item to sync (unless your sync mode is 'all' in which case it will sync automatically anyway.)  You can also drag and drop from windows explorer to the zune software to add content to your collection.  But you cannot drag straight from explorer to the zune device.
