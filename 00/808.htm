velsenv say:

So after some time trying different ways to get my email signature to look good, I have found a solution (which worked for me and should work for everyone else in similar situations).

I am using Outlook 2016.

Steps:

1. Create your email signature as you normally would and then save (File>Options>Mail>Signatures)

2. Now compose a new email (Home>New Email) and make sure your signature is showing.

3. Select the image that would normally show as blurry/pixelated. Replace with the same image using the insert function (Insert>Pictures> select image). Repeat for all images in your signature

4. Add any links to the images if needed. You have to do this now or the image will get lost if done later.

5. Send the email to yourself.

6. When you receive the email from yourself, copy the whole signature.

7. Go to File>Options>Mail>Signatures and delete the email signature.

8. Create a new email signature and paste the new signature. Do not add any links to the images. If you do, the images will get lost (links should have been added as stated in step 4).

9. Save and try it out by sending an email to yourself. The email received should now have images at their full crisp resolution.

I know this is an old topic but I still had the issue of blurry email signatures so though I'd post a solution for anyone else still in this situation.

Hope this helps!

angtur31 say:

Sharing Auto Text

Is there a way to put standard auto text for Word say on a shared drive, where all users could go and access it? So the purpose would be to have a central, standardized auto text file that could be periodically updated/maintained by one person, and then all users with Word could go into the location of the auto text on the shared drive and update their's periodically?
jay freedman say:
Yes, you can do that.

In Word 2007, 2010, and 2013 (and presumably future versions), AutoText is just one particular gallery within the Building Blocks feature. The last paragraph of this article on the Microsoft Word Blog  explains how to set up a shared location on a network server for Building Blocks.

Word will load building blocks from any templates it finds in the local and/or network locations, so you can have multiple templates there. You could have one that's standard and maintained by one person, and one or more that may have entries added by anyone.
