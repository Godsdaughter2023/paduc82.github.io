        sht.Select

    Next sht

but I keep get an Object doesn't support this property or method error. Any ideas?

jllatham say:

Try this code instead (I've also shown routines that would call it to hide/unhide the sheets).

First problem: the .Worksheets at the end of the array definition was causing the error.

Second problem, when trying to make the sheets visible, you would have gotten another error since you cannot .Select a hidden sheet.  The code below has been tested and does work for me.

Third potential problem while values for .Visible properties do map to True (xlSheetVisible) and False (xlSheetHidden), I prefer not to risk Microsoft changing the rules mid-stream, so I explicitly set the property based on Excel constants.

Sub HideThem()
  hideUnhideSheets False ' hide then
End Sub
Sub UnhideThem()
  hideUnhideSheets True
End Sub
Sub hideUnhideSheets(visibleState As Boolean)
  Dim sht As Worksheet
  For Each sht In Sheets(Array("PTN Lookup", "PTN General Info", "Norad Lookup", "PE Lookup"))
    If not visibleState Then
      sht.Visible = xlSheetHidden
    Else
      sht.Visible = xlSheetVisible
    End If
  Next sht
End Sub
