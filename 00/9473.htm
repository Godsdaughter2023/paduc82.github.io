sharonmary say:

Hi Duecy,

 

If you are incorporating many documents into one - and they have different formatting - I would recommend that you separate them into sections to avoid any “surprises” with the formatting. You can take care of consistent formatting when you have all the chapters together.

 

Here's how:

 

1.     Create a new document to preserve the existing chapters "just in case"

 

 

2.     Go the Insert tab

 

 

3.     Click the arrow next to Object in the Text group

 

4.     Click on Text from File

 

 

5.     In the Insert File dialog box, locate the file that with the chapter content and then double-click it

 

6.     Go to the end of the text and insert a section break (Page Layout tab > Page Setup group > Breaks link > Section break (Next Page or Odd Page, as required)

 

7.     Click after the section break and repeat steps 2-6 until all your chapters are incorporated.

 

 

Hope that achieves what you wanted.

 

 

 

 

All the best,

 

Sharon

jay freedman say:

I don't think it's possible to turn the recent file/folder feature off and on.

What you can do is preserve its current state, and restore it when you've finished your project. To do that:

  • From the Start > Run box, start the registry editor, regedit.exe. (In Windows 10, you can ask Cortana or the search box for regedit.)
  • Navigate to the key HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\16.0\Word\User MRU.
  • Right-click that key and click Export. Select a folder, such as the Desktop, and enter a name for the file. The extension .reg will be added automatically.
  • Close the registry editor.

Carry out your project. (If you prefer, you can temporarily set the number of recent files that Word will record to zero. That isn't necessary, though.)

After you finish, double-click the .reg file wherever you saved it, and answer the prompt. That will put back the recent file/folder lists as they were when the file was saved, overwriting the current contents.
