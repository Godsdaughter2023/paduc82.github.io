jaeson cardaño say: Hi,

This can be done through a UDF (user defined function), in the workbook you are working press Alt+F11 / right click in the Project - VBAProject field / Insert > Module / then insert this code:

'---------------------------------------------------------------------------------------------------------
Function ConcatIf(ByVal compareRange As Range, ByVal xCriteria As Variant, Optional ByVal stringsRange As Range, _
Optional Delimiter As String, Optional NoDuplicates As Boolean) As String
Dim i As Long, j As Long
With compareRange.Parent
Set compareRange = Application.Intersect(compareRange, Range(.UsedRange, .Range("a1")))
End With
If compareRange Is Nothing Then Exit Function
If stringsRange Is Nothing Then Set stringsRange = compareRange
Set stringsRange = compareRange.Offset(stringsRange.Row - compareRange.Row, _
stringsRange.Column - compareRange.Column)

For i = 1 To compareRange.Rows.Count
For j = 1 To compareRange.Columns.Count
If (Application.CountIf(compareRange.Cells(i, j), xCriteria) = 1) Then
If InStr(ConcatIf, Delimiter & CStr(stringsRange.Cells(i, j))) <> 0 Imp Not (NoDuplicates) Then
ConcatIf = ConcatIf & Delimiter & CStr(stringsRange.Cells(i, j))
End If
End If
Next j
Next i
ConcatIf = Mid(ConcatIf, Len(Delimiter) + 1)
End Function

'---------------------------------------------------------------------------------------------------------

Press Alt+Q to close VBA and return to your workbook.

Now assuming that the list you have is in Column A, then the lookup value is in B2, then enter this in C2:

=ConcatIf(A:A,"*"&B1&"*",A:A,", ",TRUE)

ronrosenfeld say: More concrete examples would have been useful.  But from your examples and description, it would seem that if Column 1 has only a B, you would want returned all of the strings that contain a single B, and so forth.

The following macro assumes that your strings are in column 1 and column 2 as you show; and that the results will start in column 3 (C) and extend rightward.

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.

====================================
Option Explicit
Sub ReturnStrings()
    Dim rCol1 As Range
    Dim rCol2 As Range
    Dim c As Range, d As Range, rDest As Range
    Dim s As String
    Dim sPattern As String
    Dim i As Long
Set rCol1 = Range("A1", Cells(Rows.Count, "A").End(xlUp))
Set rCol2 = Range("B1", Cells(Rows.Count, "B").End(xlUp))

Range(rCol1.Offset(columnoffset:=2), Cells(rCol1.Row, Columns.Count)).Clear

For Each c In rCol1
    sPattern = ""
    s = c.Text
    For i = 1 To Len(c.Text)
        sPattern = sPattern & "*" & Mid(c.Text, i, 1)
    Next i
    sPattern = sPattern & "*"
   
    i = 3
    For Each d In rCol2
        If d.Text Like sPattern Then
            c(columnindex:=i).Value = d.Text
            i = i + 1
        End If
    Next d
Next c
End Sub

1. Sign in to your Hotmail account (via a Computer/Laptop).
2. Go to your Deleted folder (found on the left pane under Folders).
3. Look for "Lost a message?", and click "recover deleted messages".

You can also check out the following article for more information on how to recover lost/deleted messages from your Hotmail account:
What should I do if email messages are missing from my Hotmail account?
http://windows.microsoft.com/en-US/hotmail/emails-missing-from-account
http://windows.microsoft.com/en-US/hotmail/how-recover-deleted-messages  Don’t forget to click on Show All to see all the recommendations in the article

We can only recover recently deleted emails, specifically not more than 5 days old. Beyond that won't be possible.
