Block Global Admin from SharePoint Folder

The folder currently has one group with Full Control that doesn't include the global admins.  Is there a way to block the Global Admin users from accessing/viewing content?

james yan msft say: Hi Marc,

Generally, if the site collection (where the folder is stored) is created by the global admin, he is the site collection administrator of the site and he should be able to access all the resources in the site collection.
If you only want to block global admin from accessing the folder while keeping his Full Control permission level to this site, this is not feasible. But here is the workaround:

1. Go to the library > select the folder > click Download from the top menu, download the folder to the local.
2. Create a new site collection from SharePoint admin center > go to the site > add a new library and a new folder in it > upload all the files in the downloaded folder to the new folder.
3. Go to Site settings > Site collection administrators > add another user (for example "user A") into the list > OK.
4. Go to the site as "user A" > Site settings > Site collection administrators > remove the global admin from the list > OK.
5. Share the folder with the group with Full Control permission.
6. Go to the original site, remove the folder.

=IF(COUNTIF(Calculations!$D$2:$D$20000,"=YES")>=ROW(A1),INDEX(ROW($A$2:$A$20000),SMALL(IF(Calculations!$D$2:$D$20000="YES",ROW($A$1:$A$19999),20001),ROW(A1))),"")

This is an array formula and to enter it copy it from here paste it into cell A2 then double click in A2 and then press the key combination Ctrl+Shift+Enter. The formula should cover you for up to 20,000 entries but if you want to alter it you will have to redo the Ctrl+Shift+Enter. Copy the formula down as far as you need it. In sheet Group B cell A2 the formula needs to alter to find Group B records so you will need to change

Calculations!$D$2:$D$20000      into       Calculations!$E$2:$E$20000      and then Ctrl+Shift+Enter. and then change again in any following sheets you might have. This formula creates a record of which rows it finds a "YES" for each of the groups. Once entered you could hide this column if you wish.

=IF($A2="","",IF(INDIRECT(ADDRESS($A2,COLUMN()-1,,,"Calculations"))="","",INDIRECT(ADDRESS($A2,COLUMN()-1,,,"Calculations"))))

And copy/fill across to cell FE2 and down as far as you need. It's the same formula for all the Group sheets because it looks at column A and then gets the records from that row on the Calculations sheet. jonathanmatthis say:

Press the Windows Key + X to launch the System Settings snap in console above the start menu on the screen. Select Device Manager. Check in this list to be sure that you do not have any driver bangs. This will be immediately apparent as the list is supposed to be minimized but if there is an issue, the area with the driver bang will automaticaly be expanded and the item with the issue will have a Exclaimation mark in a yellow triangle with it. If there is an issue it will usualy be the UFEI firmware or Surface Touch or even Surface Embedded Controller Firmware. This is all found under the firmware tree. Assuming there are no bangs apparent, find the Human Interface Devices tree and expand it.

Locate the two HID-Complaint touch screen entires, right click and select Uninstall click OK.

**WARNING if there is a check box present asking to delete the drivers ensure that it remains unchecked**

After both of these entires have been uninstalled, restart your surface by completing a driver push button reset. This is done by holding the power button and volume up button until the system turns off and continue to hold the buttons for a period of 15 seconds.

Release the buttons then turn on your machine normally and check for touch functionality again.

When you download/install Google Chrome, it becomes your default browser, replacing Internet Explorer.  You need to set Internet Explorer as your default browser......

 

At the Start screen type default  (there is no box to type it into.....just type)

Select "Default Programs"

Select "Set Your Default Programs"

Select Internet Explorer

Select "Set This Program as Default"

Click OK 

 

Then.........

Open Internet Explorer on the Desktop:

Tools

Internet Options

Programs (tab on top)

Select "Let Internet Explorer Decide" (from the drop down box)

Make sure there is NO CHECKMARK next to "Open Internet Explorer Tiles on the Desktop"

Click APPLY

Click OK

 

restart your computer for these changes to take effect.
