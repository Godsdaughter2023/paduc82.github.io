Because of this,any change made to Heading 1 will be inherited to Heading 2, etc. However, I wanted to change that, in order to make the styles independent from the point of view of font-related changes. So I chose, for HEading 2, "style based on = (no style)". However, more than making it independent from Heading 1, this also took Heading 2 out of the numbering system, such that now, titles who have Heading 2 applied to them are no longer numbered.

How can I make Heading 2 be independent from Heading 1 while keeping it in the numberning system?

jay freedman say:

Your heading numbering should be set up according to the full procedure in http://shaunakelly.com/word/numbering/numbering20072010.html.  In the Modify Multilevel List dialog, you select Level 1 and then set the "Link level to style" dropdown to Heading 1; then select Level 2 and set the "Link level to style" dropdown to Heading 2; and so on. 

The other thing to verify (although it should still be set correctly) is to go into the Modify Style dialog for Heading 2, click the Format button at the bottom left and click Paragraph, and check that the Outline Level is still set to level 2.

mlg63 say:

You should repartition the C and D partitions and combine them into one (warning--this might require reinstalling windows and everything else.) Disk management will allow you to delete the big D partition- if it is not the boot drive. You will lose everything on D. Then extend the C boot partition to include the whole adjacent D partition. That will correct the old partitioning which was a failed  experiment of separating windows and data on some older computers. The D partition must be on the right side of the C partition with nothing in between to do this. There is still a chance you could lose everything so copy your files and save them on external media. Have your reinstallation disks ready to reinstall windows. I would delete the E partition too if possible. Open Disk Management to see how the drive is partitioned.

Windows 10 likes one big partition instead of splitting the drive into smaller ones. Combining the C, D and E partitions would be better than just taking a portion of D.
