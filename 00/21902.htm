Use a macro containing the following code (with revision tracking turned off)

Dim arev As Revision
With ActiveDocument
    For Each arev In .Revisions
        If arev.Type = wdRevisionDelete Then 'And arev.Range = "Bob" Then
            If arev.Range = "Bob " Then
                arev.Reject
            End If
        End If
    Next arev
    For Each arev In .Revisions
        If arev.Type = wdRevisionInsert Then
            If arev.Range = "John " Then
                arev.Reject
            End If
        End If
    Next arev
End With

You may or may not need the space after the names (Bob and John)  By experimentation here, I found that I needed the space.

howard31 say:
If you put the button outside the multipage control you can use the following to move next through all pages:

=======================================
With Me.MultiPage1
        If .Value + 1 < .Pages.Count Then .Value = .Value + 1
End With
========================================

To move backwards put another button OUTSIDE the Multipage control and add the follwoing code:

=======================================
With Me.MultiPage1
        If .Value + 1 > 1 Then .Value = .Value - 1
End With
=======================================
