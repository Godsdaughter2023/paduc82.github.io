How exactly did you try setting the font size as default?
In Word, go to Home tab > Font section > click on the arrow mark on the bottom right corner > type 124 in the size column > click ‘Set As Default’ > click OK. Now check if it retains the font size when you type.

Here's how I got round this problem. What the Lumia does is vibrate 3 times at the start of your ringtone, and that's all.
