I'm not quite sure I understand your question. But I'll take a shot. The File history feature is used to backup your personal data files, like DOCX, PDF, XLSX etc to an external drive. It keeps versions and is intended to allow you to recover older versions of files that have become corrupted or have been deleted accidentally. An Image copy backup is a "Snapshot" of your whole drive, typically taken at a point in time the drive is in a known state, good or bad. It is intended for "disaster" recovery. Restoring from an image copy means that you do not have to re-install Windows and all of your applications. Some image copy backup software allows you to extract individual files, but that is a feature you have to look for. Does that answer your question?

Windows' own System Image Backup utility will make a backup image, but you'll have to use a recovery disk or restore Windows first before you can restore the image.
