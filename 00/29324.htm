steveg47 say:

I'm more confused than I thought.  Apparently a "repair disc" is not a "recovery drive."  In Windows 10, one can write a "recovery drive" to a USB thumb drive.  But he cannot write a "repair disc" to one.  What's the difference?

In case that's not clear.  You get to the repair disc applet by going to File History, create image, create repair disc.  The only way I know to find the create repair drive applet is to go to the start menu and type "create recovery drive."

I cannot find any documentation on the difference.

Forget about creating a repair disc.  It is a hold over from Windows 7 and requires a dvd/cd reader/writer.  Your should create a usb Recovery Drive from your Windows 10 installation or download one from the manufacturer of your pc; in this case from Microsoft for your SP4.  The Recovery drive will allow you to repair or if necessary reinstall your operating system with all the installed drivers and default pre-installed applications that normally come with Windows 10 even if your installation will not boot.  The drivers and Windows 10 version will be as up to date as when the Recovery Drive was created.  Any additional applications you have added will not be recovered and will have to be reinstalled.  You can also create a System Image backup to an external drive which can be used to restore a corrupted operating system to it's exact state at the time the system image was created including all apps, settings, data, registry settings, etc.  It will also restore any existing problems and corruption.  Since it backs up the whole disc and everything on it, it will take time and a lot of disc space to create.  It's biggest disadvantage is that individual files cannot not be restored, it's an all or nothing situation. There are 3rd party applications which get around this limitation.  There are also restore points which can be created periodically which allow the system to be rolled back to the conditions that existed at the time the restore point was created.  Restore points are quick and easy but sometimes don't work as expected. 

The topic of repair and recovery is a complex subject and requires some study and experience to be able to use  effectively.  This brief write-up barely covers the basic concepts.  Trying to learn it after the disaster has occurred is not recommended.  If you have created the Recovery Drive an experienced tech can always help you to use it.  If you want to learn more get yourself an older pc you don't mind experimenting on.

And always backup your data; tomorrow will be too late!
