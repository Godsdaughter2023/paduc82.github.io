samuel tai say:

There are 2x copies of WMP on 64-bit. The 32-bit one is in C:\Program Files (x86)\Windows Media Player. The 64-bit one is in C:\Program Files\Windows Media Player.

32-bit and 64-bit WMP don't share codecs at all. 32-bit WMP only uses 32-bit codecs. 64-bit WMP only uses 64-bit codecs.

If you've installed a codec pack, you might want to see if there's a 64-bit version available.

A couple of utilities to use:

1. http://www.codecguide.com/windows7_preferred_filter_tweaker.htm. This will allow you to set your preferred codecs for both 32-bit and 64-bit environments.

2. http://gspot.headbands.com/. This 32-bit only tool will scan your 1080p files and tell you which codecs are used.

3. http://blog.monogram.sk/janos/2009/06/14/monogram-graphstudio-0320/. Comes in both 32-bit and 64-bit flavors. This will show you the DirectShow graph that WMP builds to apply the codecs to your media files. It also allows you to set the merit of DirectShow codecs.

You may have to run these tools as Administrator to get things to stick.
