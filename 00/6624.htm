braytonfisher say:

It is actually easy to hilight.

1. Select a shape, like a rectangle

2. Draw the rectangle where you want it.  Don't panic when part of the picture disappears.

3. Right click on the shape, select Format Shape (or select Format - Shape from the top)

4. Select Fill and change to a solid color, like the Standard Color Yellow.  Change the Transparency to about 75%, and click OK

5. Click on Layout, and select "In front of text"

Copyand paste this shape to the various places on the picture.

Cost Depreciation Rate Year 1 Year 2 Year 3 Year 4 Year 5

$80 50% $40 $40 0 0 0

So my table here is about spreading the cost into smaller one, accordingly to the depreciation rate. So for cost $100, and 25% rate, the cost will be spread out for 4 years, with each year holds 25% of the actual full cost ($25). For cost $80, and 50$ rate, the cost will be spread out for 2 years, with each year holds 50% of the actual full cost ($40).

So I can just put formula like Cost*Depreciation Rate into the years to get the depreciated cost. But my question is how do I make it stop when it has been fully depreciated? Like for cost $100, stops at year 5 and cost $80 stops at year 3.

amit tandon say:

Enter below formula in cell C2 & copy to the right and down:

=IF(COLUMN()-COLUMN($C2)+1>ROUNDUP(1/$B2,0),"",IF(COLUMN()-COLUMN($C2)+1=ROUNDUP(1/$B2,0),$A2-$A2*$B2*(ROUNDUP(1/$B2,0)-1),$A2*$B2))

This also adjusts the balance depreciation in the last year if the depreciation rate does not fit in precisely with each year (ex. depreciation rate of 27% would cover 3.7 years & not precisely 3 or 4 years, so in the 4th year the depreciation would be the balancing amount).

Note: "$C2" in the formula is the start cell.

In case this is your requirement.

Is there a way to display page numbers in the TOC, the way they are formatted in the document?

     

    Thank You

    suzanne s. barnhill say:
    When you add the \* roman switch to a PAGE field, the page number is displayed on the page as a roman numeral but still reported as arabic in the TOC. In order to get roman numerals in the TOC, remove the switch and use the Page Number Format dialog to select the i, ii, iii format instead of 1, 2, 3.

    amirulislamot say:

    @Suzanne, Thank you for your kind help. The method is working.

     

    I have one more question. Instead of using heading and style, I am building my TOC mannually from the TC fields. Similarly, is there a way to build table of figures and list of tables from such fields manually, instead of using the "insert caption" feature?

     

    Thank you

    suzanne s. barnhill say:

    While the table of figures is intended to be based on a specific series of captions (distinguished by the identifier of the SEQ field used to number them--e.g. { SEQ figure } or { SEQ table }), it can also be based on a specific style. If you apply a different style to each type of caption (the styles can all be formatted the same; they just have to have different names), then you can select that style in the Table of Figures Options dialog. See this help article.

     

    Alternatively, you can use TC fields for those as well. You just need to add an \f switch to identify the entries. By default, with no \f switch, TC fields become TOC entries, but if you include switches such as \f f for figures and \f t for tables, then you can specify that identifier in the Table of Figures Options. See this help article for more.

    To print pages 1-3 in the first section, you would enter

    p1s1-p3s1.
