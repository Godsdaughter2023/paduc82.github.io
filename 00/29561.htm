rcwhite say:

﻿Hi, Nathan.

Nope!

I purchased a lap top with Windows 7 already installed.

Then this is an OEM (Original Equipment Manufacturer) version, which is forever tied to the computer on which it was pre-installed.  Doesn't matter who owns it or who "registers" it.

You'll need to buy another license for your other computer.  If that other computer is running an activated copy of a qualifying version of Windows, you should be able to use an "upgrade" package, saving perhaps $100 off the price of the "full" retail Windows 7, depending on which edition (Home, Professional, Ultimate?).
