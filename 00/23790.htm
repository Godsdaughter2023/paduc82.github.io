For Exchange Online mailboxes, you need to manage user photos by using the PowerShell cmdlet Set-UserPhoto, or by accessing the user's Outlook Web App Options page. Set-UserPhoto  "Neo"  -PictureData  ([System.IO.File]::ReadAllBytes("C:\Users\Administrator\Desktop\Neo.jpg"))
