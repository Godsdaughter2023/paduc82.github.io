cbarnhorst say:

I am answering your question in one of your above replies.  To further clarify, a home-built computer is not an OEM computer.  It is a do-it-yourself (DIY) computer.  It requires a retail license.  An OEM license is quite different.  Microsoft grants the purchaser of a retail license the rights to use the software.  Since Microsoft grants the license, Microsoft has the right to also grant the right to transfer it. 

On the other hand, OEM's do not buy licenses.  They pay royalties to Microsoft and then grant the right to use their royalty-based licenses to their customers.  Since Microsoft does not grant the right to use a royalty-based license (the OEM owns it), Microsoft cannot grant transferability (because it is not their license in the first place).  

Microsoft requires that any copy of Windows that is distributed be supported by the distributor.  That means that OEMs agree to support Windows on the devices they distribute.  If an OEM allowed its license to be transferred, the OEM would continue to be obligated to support the transferred software, even if it were transferred to a competitor's device or a DIY device.  I can't imagine HP wanting to support Windows on a Dell.

rich007a say:

Errr, in your original question you asked for code to loop through each cell in a range, checking whether the cell in col D is empty, and if col D is not empty then "run a function".

What you have inserted does nothing with the "current" row being checked in the loop.  Instead you set up another loop (For Each r In Range("C9:C62")) which checks each cell (r in this case is a single cell range) in the range C9:C62.  The value of the "current" cell in col C is asigned to a variable v (v = r.Value).  Then you test if v is empty (If v = "" Then...) and if it is empty you Exit Sub!

That's why your code is stopping at the first empty cell in col C, because that's what you're telling it to do...

I suspect you just need this:

Sub Miss_Rows3()
    Dim c As Range                       'c is a single cell range
    Dim parts As Variant                 'parts will be used as an array
   
    For Each c In Range("C9:C62")
        If c.Value <> "" Then            'If c is not empty then..
            parts = Split(c.Value, ", ") 'Split current cell value into an array
           
             'Test if the first two elements of the array are numeric:
            If IsNumeric(parts(0)) And IsNumeric(parts(1)) Then
                c.Offset(0, 1) = --parts(0) 'Put 1st element of array in col D
                c.Offset(0, 2) = --parts(1) 'Put 2nd element of array in col E
            Else
                MsgBox "Cell " & c.Address(False, False) & " is not numeric!" _
                    , vbCritical + vbOKOnly, "Error"
                Exit Sub 'Comment this out if you want the code to continue.
            End If
        End If
    Next c                               'Loop to next cell in range C9:C62
End Sub


Note I have changed r to c.  I like to use this notation, for me "c" is always a single cell range.

I also added a check to make sure the first two elements of the array are numeric, otherwise the -- will cause an error.
Technically, you don't need this.  If you get VBA to enter a numeric string into a cell formatted as General (or pretty much anything other than "Text") it will be converted to a number.
So you could just use:

Sub Miss_Rows4()
    Dim c As Range                       'c is a single cell range
    Dim parts As Variant                 'will be used as an array
   
    For Each c In Range("C9:C62")
        If c.Value <> "" Then            'If c is not empty then..
            parts = Split(c.Value, ", ") 'Split current cell into an array
            c.Offset(0, 1) = parts(0) 'Put 1st element of array in col D
            c.Offset(0, 2) = parts(1) 'Put 2nd element of array in col E
        End If
    Next c                               'Loop to next cell in range C9:C62
End Sub

Hope that helps.

graham mayor say:

You could use a macro e.g. as follows - http://www.gmayor.com/installing_macro.htm  

Sub Send_As_Mail_Attachment()
'Graham Mayor = http://www.gmayor.com
'Send the document as an attachment _
  in an Outlook Email message
'Requires the code from - http://www.rondebruin.nl/win/s1/outlook/openclose.htm
'to either retrieve an open instance of Outlook or open Outlook if it is closed.
Dim bStarted As Boolean
Dim olApp As Object
Dim oItem As Object
Dim oDoc As Document
Dim strName As String
Dim strDocName As String
Dim strPath As String
Dim intPos As Integer
Dim iFormat As Long
    Set oDoc = ActiveDocument
    'Prompt the user to save the document
    On Error GoTo err_Handler:
    oDoc.Save
    strDocName = oDoc.Name
    iFormat = MsgBox("Send as PDF format?", vbYesNoCancel)
    If iFormat = 2 Then GoTo lbl_Exit
    If iFormat = 7 Then strDocName = oDoc.FullName: strName = oDoc.Name
    If iFormat = 6 Then
        'Get the document name and path
        strPath = oDoc.path & "\"
        intPos = InStrRev(strDocName, ".")
        strDocName = Left(strDocName, intPos - 1)
        strName = strDocName & ".pdf"
        strDocName = strPath & strDocName & ".pdf"

        'And save the document as PDF
        oDoc.ExportAsFixedFormat OutputFilename:=strDocName, _
                                 ExportFormat:=wdExportFormatPDF, _
                                 OpenAfterExport:=False, _
                                 OptimizeFor:=wdExportOptimizeForPrint, _
                                 Range:=wdExportAllDocument, from:=1, to:=1, _
                                 Item:=wdExportDocumentContent, _
                                 IncludeDocProps:=True, _
                                 KeepIRM:=True, _
                                 CreateBookmarks:=wdExportCreateHeadingBookmarks, _
                                 DocStructureTags:=True, _
                                 BitmapMissingFonts:=True, _
                                 UseISO19005_1:=False

        'Now close the document without saving as we have finished with it
        oDoc.Close 0
    End If
    'Get Outlook if it's running
    Set olApp = OutlookApp()
    On Error GoTo 0
    'Create a new mailitem
    Set oItem = olApp.CreateItem(0)

    With oItem
        .Subject = strName
        .Attachments.Add strDocName
        .Display
    End With

lbl_Exit:
    Set oItem = Nothing
    Set olApp = Nothing
    Exit Sub
err_Handler:
    Err.Clear
    GoTo lbl_Exit
End Sub
