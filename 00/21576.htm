Don't try to do this from the Print dialog. Instead, in the Page Setup dialog, change the paper size to A4 and select "2 pages per sheet" or "Book fold" from the "Multiple pages" dropdown. This will give you two A5 pages on an A4 sheet and will not change your page layout at all.

suzanne s. barnhill say:
Indeed. Just one example: I frequently have to create 3" x 4" inserts for name badges. I can use "2 pages per sheet" (portrait) to print them 2-up on 4" x 6" cards. Since each badge is a separate logical page, any selection of "pages" can be selected to print on the same card. Creation of the badges in bulk is absurdly simple: a First Name style (72-point Arial), formatted as "Page break before," with the Last Name style (48-point Arial) assigned as the "Style for following paragraph."
