This issue could be due to an incorrect color profile in the Color Management settings used for the monitor. This usually happens when we install an incompatible monitor drivers on the computer. To fix this issue, let us first check if there are any incorrect color profile under Color Management settings on the computer and remove it, if available. Please follow these steps:

b) Type “Color Management” in the search bar and then click on the option “Color Management” from the search results.

c) On the “Color Management” window, on the “Devices” tab, select the check box “Use my setting for this device”

d) Click on “Add”.

e) Double-click on the profile “sRGB IEC61966-2.1” (This is the color profile that IE is using) listed under “ICC Profiles” and set it as the default profile.

f) Click on “Close” and check if the issue persists.

 

Hope this resolves the issue. Please get back to us with the result.
