a. user say:

It sounds like you need to figure out of there is one or more updates that cause the problem.

If you can check the system when it is broken and get into Windows Update, View update history you can click the Date Installed column to sort and get a list of the updates installed today (or recently) and make a list.  If the problem occurs after Windows Update one of those recently installed things sounds suspicious.

You can include a screenshot of the list (see below).

Then change Windows Update Settings to Never check for updates and save that.

Then restore your system to a working state.

Then go to Windows update and manually click the Check for updates button and see what is offered which does not mean you have to install anything at the moment.

It could be that one of those offered updates is the problem so you can post a list/screenshot of them and maybe somebody can spot something troublesome and give some advice.

Or you can use a process of elimination and install the offered updates one at a time (or a few at a time) restarting after each to see which one brings on the problem.

Once you have it narrowed down you can uninstall the troublesome update and consider hiding it and hope you never see it again.  Of course it would be good to share the KB number and description of what you think is the troublemaker.

Setting Windows Update to Never check for updates and checking yourself manually puts you in charge if what gets installed on your system instead of Microsoft.  Then you might not have these problems in the future.

1. This is more likely to be a hardware issue or CD/DVD issue. Please check your CD/DVDs are not scratched and in good shape.

2. Try to clean your CD/DVD with a clean cloth to remove dirt etc.

3. From Windows side, you can do following -

A. Use the Hardware & Devices Troubleshooter

Type Troubleshoot in Windows Start Search box > Click Troubleshoot > Scroll down & Click Hardware & Devices > Click Run the Troubleshooter & follow on-screen directions > Restart the computer after troubleshooting is completed

B. Update driver

Windows Key+X (Or right click on Start) > Click Device Manager > Expand DVD/CD Drives > Right click on entry below > Uninstall Device > Reboot your computer

C. Review following Microsoft Help
