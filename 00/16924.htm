suzanne s. barnhill say:

On the Home tab, in the Paragraph group, click the arrow beside the Multilevel List button and choose Define New Multilevel List. You may want to first select the list template that shows bullets, then reopen the menu and choose Define New Multilevel List.

 

In the Define New Multilevel List dialog, click More to expand the dialog. Set the desired bullet and indent for each level and (very important) link each level to a style. I suggest linking to List Bullet, List Bullet 2, List Bullet 3, and so on. If you have used the List Bullet series, then you can apply the List Bullet style using the built-in keyboard shortcut Ctrl+Shift+L. Do not use the Bullets button on the Home tab, as this will apply the List Paragraph style with standard round bullets, which is not what you want. Once you have started the list, you can use the usual shortcuts to demote/promote levels.

jim gordon mvp say:

Please go to the Word menu and choose About Word. The dialog will tell your your license type.

It is not impossible, but a few people did obtain Office 2011 using an Office 365 subscription. If you did that, your license will say Office 365 Subscription. With a subscription you go to portal.office.com where you can deactivate and activate installs as you like.

If you have any other kind of subscription, the number of activations is limited by how many you purchased. Home and student was sold with single user and multi user licenses. Office for Business likewise was sold with single and multiple user licenses.

If you purchased a single install license, then you can't install Office 2011 on another machine using the same product key. If you purchased a multi-pack, then you can.

If you do need to purchase a new license, they're now really cheap on sites like eBay.com where they are selling for about $20 US right now.
