wunders say:

I referred to KB307874 mainly for the disabling Simple File Sharing part.

 

Looking back, KB193826 only seems to apply to Windows NT.  However the same file does exist in Windows XP, so the quick check to see if you have read permission to this file would be to make a copy of the file to somewhere else on the disk.  If you can do that, you have read access to that file.

 

There is one more super-basic requirement that is needed to see the Security tab (so basic but rare that I overlooked it)....  The disk on which the target file or directory is located must be formatted as NTFS (not FAT-16 or FAT-32).  So double-click on "My Computer", then right-click on the hard drive and select "Properties" and verify that the "File System" is NTFS.  If it isn't, you can convert it to NTFS if you want using the following procedure:

 

"How to use Convert.exe to Convert a Partition to the NTFS File System"

 < http://support.microsoft.com/kb/314097 >
