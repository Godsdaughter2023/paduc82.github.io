Is there a simple and quick way of deleting these temporary files? - Microsoft Community

brian8686 say:

Is there a simple and quick way of deleting these temporary files?

To clear my comp. of clutter and to help speed up my PC, my ISP told me to put Prefetch and %temp5 and Temp into Start-Run in order to delete the temp files. This is a bit long winded as each group has to be deleted separately. Disc Clean up does not get rid of them. Is there a simple and quick way of deleting these temporary files? Thanks Brian
lemp say:
To clear my comp. of clutter and to help speed up my PC, my ISP told me to put Prefetch and %temp5 and Temp into Start-Run in order to delete the temp files. This is a bit long winded as each group has to be deleted separately. Disc Clean up does not get rid of them. Is there a simple and quick way of deleting these temporary files? Thanks Brian

Deleting contents of the prefetch folder will not help speed things up.  In fact, it will likely slow things down.  See http://answers.microsoft.com/en-us/windows/forum/windows_vista-system/can-i-delete-the-data-in-the-prefetch-folder/c54a8e56-c8cf-451d-a88f-07f06a2f2d54

I assume that %temp5 is a typo and you really meant %temp%.  The percent signs indicate that this is an "environment variable" the usual value of which is C:\Documents and Settings\,yourusername\Local Settings\Temp

The directory specified by %temp% is, in fact, the directory cleaned by Disk Cleanup under the name "Temporary Files."  Why do you think Disk Cleanup isn't removing the files?  Note that Disk Cleanup does not delete files from this directory that have been accessed in the last 7 days (this time period can be changed by modifying a registry value).

What Disk Cleanup does NOT do, however, is clean temporary files, Internet browser cache files, and other files in user-profile-specific locations from profiles other than the one that you are currently using.  Disk Cleanup also does not delete the contents of C:\Windows\Temp (although this doesn't get filled with very much).  C:\Windows\Temp is the directory that opens when you type Temp into Start > Run (without the percent signs).  And, of course, Disk Cleanup will not empty the browser cache of non-Microsoft browsers, e.g., Firefox.

There are third-party applications that do a "more thorough" cleaning.  Some of them clean files from all users without having to login to each profile separately.  Most include features that you should not use.

Here are 2 -- there are others, but I haven't used them.

ATF-Cleaner  (don't clear prefetch)
CCleaner (don't use the registry tool)

You might be interested in "How to Automate the Disk Cleanup Tool In Windows XP"

2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
