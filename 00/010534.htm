rena yu msft say:

Hi Theking2,

About the “after reopening Excel the Document Recovery sported the old document but not the new one.”, it’s related to the auto recovery settings via File>Options>Save>Save AutoRecover information every XXX minutes.

The lower value you set, the more frequently it auto saves your unsaved files. The default value is “10” minutes. That means if the crash happens within 10 minutes after you open the file, the Excel hasn’t run the auto recovery function for any lost information. That’s should be the reason why you can’t get the old file in the auto recovery pane this time. We suggest you set the value smaller in case of losing unsaved data because of the crash happens again in the future.

The default AutoRecover file location is C:\Users\<user>\AppData\Roaming\Microsoft\Excel

You should find all the AutoRecover file there. If you can't find out the unsaved version, it means they can't be got back.

About the crash issue, please first repair Office from the Control Panel to see if the crash issue can be fixed. (This repair step will not delete your AutoRecovery files). After that, go to the AutoRecover file location to try to open those files again to see if you can open them.
