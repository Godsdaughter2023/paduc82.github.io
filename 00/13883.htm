cindy meister say:

Hi frumbert

Word takes the language basically from the Windows keyboard settings. And you're right, U.S. English is in there by default...

Go to Control Panel, Regional settings and language. You want the Keyboard tab in the dialog box. In that, Change Keyboards (I'm looking at Vista, so the exact commands may be slightly different, but the functionality is still the same). If the keyboard language you're using is listed, click the "Add" button and make sure you add Australian English as a language. graham mayor say:

It is the input language that determines the spelling, not the keyboard mapping. The input languages are set from the languages dialog of the regional settings and can be associated with any keyboard mapping. The Location setting of the regional settings is a separate entity that determines what local services, number and date formats etc you see on your PC.

You can have as many or as few input languages set as you require. If you have (as you previously indicated) set the language detection in Word to off, then the only language issue you should encounter is when pasting formatted material from the web or from documents that already contain alternative language formatted text. The formatting stays with a document so old documents formatted with US English.

I can, for example, set up Australian English input language with a US keyboard mapping and if I select that, that is what it will use until I switch to something else. As I live in a Greek speaking country I have UK English and Greek languages set. I removed the English US input language and I can switch between the two available languages at will.

The easiest way to make the changes is to add the language bar to the Windows task bar and right click it to change settings.
