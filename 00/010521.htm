jamalbdeir_msft say:

Same Domain Name for Exchange 2007 and Office 365

Hi Team,

Is it possible for one of the customers to move few users to Office 365 while keeping the rest on an Exchange Server 2007 email environment while using the same domain name for both environments?

If yes, please advise on the process of completing this.

Thanks,

Jamal

brook zhang msft say:

Hi Jamal,

 

Yes, you can use Simple Domain Sharing for the existing Exchange 2007 and Office 365. It’s not like hybrid deployment. No federation relationship is set for them. Actually, you can count them as two dependent organization. MX is pointed to Exchange 2007. All emails are received on-premise then be forwarded to Office 365 if the target mailboxes are hosted online. Emails from an Office 365 mailbox is sent by using Office 365 SMTP service. And Exchange 2007 mailboxes directly send emails to the internet too.
