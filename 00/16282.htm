In addition to what Stefan said, I'll note that you aren't doing your students any favors by teaching them to use macros for creating a footer -- it's a WordPerfect habit that shouldn't be carried over to Word. It's more in the Word way of things to create a template for a specific type of document, and to include the footer information in the template so it automatically appears in each document based on the template.

If you want to use a simple task just to teach how to record macros, I can applaud that, but the task should be one that's more appropriate. Maybe a Replace operation that they would want to run against several documents would make a good candidate.

Also, refer to http://word.mvps.org/FAQs/MacrosVBA/ModifyRecordedMacro.htm for some useful information about the recorder.

If the task is to be able to start new documents with this information, teach them how to create a template that has the information in the footer. Here they would want to use a DocProperty Title field or a StyleRef field referencing the Title Style and a Docproperty Author field. If they wanted to be able to insert that in any footer, they could save an AutoText entry with all of these for future use.

These will give your students permanent skills to produce documents.

Please note that macros can be written to put information into a header footer but each document has at least three spots for header/footer information. They can have many more!

A really good exercise might be to have them create a letterhead following instructions at http://sbarnhill.mvps.org/WordFAQs/Letterhead.htm. Have them set up a template for multiple page letters. Perhaps have them use a StyleRef field to pull information into the continuation header/footr from the body of the letter template like the date or addressee name.  Then have them set one up with a list of the board of directors in one of the margins on the first page. Then have them do the same thing to accommodate preprinted letterhead.

Your students will wow their employers.
