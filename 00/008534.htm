john korchok say:

I think you're expecting that after you compress the pictures, that the choice displayed in the Picture Quality dropdown should change to what you chose previously, so it might read Email (96 ppi) instead of High fidelity (Maximum ppi). While that would be a useful and intuitive feature, Word doesn't track the picture resolution of the document photos. Since it has no idea, it always displays the first choice in the dropdown.

Instead, open a Finder window and check the file size of your document first. Then use File>Reduce File Size... and save. Recheck the file size to see that it is smaller.

bob jones aka: cybertaz say:

One of the key questions is: What is the image resolution to begin with? If the images are at or below what you select the feature will have no effect.

Also, Delete cropped areas... has no effect on file size if the image hasn't been cropped. Even if minor cropping has been done the impact may be minimal.

Other contributing factors are the file type of the images as well as their print dimensions. Scaling the image in Word has no impact on the actual print dimensions of the image file stored in the document.

Just a suggestion, but for optimum results images should be processed for their intended purpose using a decent image editing program. Save a version with the appropriate file type, resolution & print dimensions in order to minimize or eliminate the amount of modification you need to do with the limited convenience tools of a word processor.
