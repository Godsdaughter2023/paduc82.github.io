History Sheet without Sharing a Workbook - Microsoft Community

kevinpc06 say:

History Sheet without Sharing a Workbook

Hello,

Is it possible to create a changes history worksheet in Excel without turning on sharing?

gord dibben say:

Hello,

Is it possible to create a changes history worksheet in Excel without turning on sharing?

You can keep a logsheet of changes made to your input sheet by using VBA Event code.

Copy/paste the code into the sheet module of "Data Input" sheet

Tracking is recorded on "Log" sheet

'Input Sheet is a sheet named Data Input
'The code below writes the change to a sheet called "Log"
'List of addresses and changes are stored in column A of "Log"
'The row number used to write the last change is stored in "Log" B1

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim Lastrow As Long
    Lastrow = ActiveWorkbook.Sheets("Log").Range("B1").Value
    'Write the Change
    ActiveWorkbook.Sheets("Log").Cells(Lastrow + 1, 1) = "Data Input " & _
    ActiveCell.Address & " changed to: " & Target.Value
    'Increase the LastRow value by 1
    ActiveWorkbook.Sheets("Log").Range("B1").Value = Lastrow + 1
End Sub
