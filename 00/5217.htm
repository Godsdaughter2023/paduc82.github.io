Click Start and in the start search field, type

Regedit

HKEY_CURRENT_USER\Software\Microsoft\Windows\Current Version\Explorer\User Shell Folders

%USERPROFILE%\AppData\Local\Microsoft\Windows\Temporary Internet Files

under value data and click ok. Close the registry and  restart the computer and check if it works.

Microsoft will continue the existing support for Windows Phone 8.1 to the end date of the support on July 11, 2017. The Windows Phone 8.1 support date can be found here.

3. How does this policy apply to the actual phone (for example, a mobile hardware) made by other original equipment manufacturers (OEMs)? 

Third-party products that integrate Microsoft products, such as the Windows Phone operating system, are not included in the Microsoft Support Lifecycle policy. Contact your mobile or OEM carrier to learn more about the support and warranty of your mobile hardware.

What is the support Lifecycle policy for services and/or subscription applications that are part of Windows Phone?
Windows Phone 8 Lifecycle policy covers the Windows Phone 8 operating system. Download applications and subscription services receive vendor support.

That last question is of extreme importance.

**Microsoft supports the operating system and not the applications or programs that are installed in this system. So it fits only the application developer, to offer the support. Obviously, many developers will prefer to adhere to a more current system: Windows 10 Mobile.**

Extended support lasts for at least 5 years and includes free security updates and non-security and support-related upgrades. Additionally, Microsoft will not accept requests for support for design changes or new features during the extended Support phase.
