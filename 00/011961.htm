Hi Steven,

The Full Access permission is required. After assigning this permission, users can open the mailboxes in both the Outlook client and OWA (Outlook Web App), and manage the mailbox like the mailbox owner. The Send on Behalf permission is not required if you only want to add it as an additional mailbox. chevia say:

Hi Steven,

Grant someone(let’s say user1) Full Access of another mailbox(user2), then user1 will have the permission to view/edit/modify/delete all the email folders, calendar, contacts and tasks.


If you grant user1 delegate permission of user2’s inbox, such as reviewer permission, then user1 will only be able to view user2’s inbox, but not other folders.

mel77084 say:

'My BIOS displays a "pls insert bootable media, then hit any key to boot from it." Doing this has no effect.'

Your bios thought you would understand that it meant to boot with the bootable media in the drive, and hit any key to boot from it when prompted to do so.

Installing the upgrade wouldn't have changed the settings that you had in your bios.

If your computer booted ok before the upgrade, it should still boot ok after the upgrade, if the upgrade completed successfully, and you didn't have an incompatible 3rd party security program installed. That seems to leave a question of your drive's condition.

Running the "repair your computer", as Krishna has explained, might be able to repair the boot failure issue, for now; however, it's likely that your blue drive has done as they often do. There might be a good reason that the blue drives are always the cheapest offered for you to buy. You might want to consider getting one with a better statistical record.

And, if you really think you need a TB drive, you should make an OS partition on it of 100 to 150 GB, and use the rest for other partitions to store stuff such as data, pictures, music, etc.
