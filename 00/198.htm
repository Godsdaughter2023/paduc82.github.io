lz. say:

@DarylFagala

Can someone please explain why the products of the individual multiplications aren't showing in column C?

Easy (when you know - as usual ;-) ) but not easy to explain in writting (++ for a non-English native). Let's see...

It's all about array calculation. And BTW your formula can be enterred as a non-array formula (Enter only) if you use SUMPRODUCT(A2:A6*B2:B6) - this would produce the same result as your formula

Basically the calculations are done (in order) as followin with the above example or your formula :

  1. A2*B2 (result (328.87)
  2. A3*B3 (result (141.40)
  3. A4*B4 (result (158.22)
  4. A5*B5 (result (189.33)
  5. A6*B6 (result (112.39)
  6. SUM(Step 1 + Step 2 + Step 3 + ... Step 5)
    SUM(    328.87+141.40+158.22+189.33+112.39)

So, as your formula or the SUMPRODUCT one produce only 1 result, when you select C2:C6 > enter the formula > and finally hit Ctrl+Shift+Enter you "ask" Excel to replicate the same value/result in each cell (C2:C6). I hope this helps a bit understanding array calculations...

That being said I suspect what you expect is:

  1. Select C2:C6
  2. Enter formula: =$A$2:$A$6*$B$2:$B$6 or =($A$2:$A$6)*($B$2:$B$6)
  3. Validate with Ctrl+Enter (not Ctrl+Shift+Enter)

Using the DeskPins application, a small piece of freeware that pins windows to the foreground, I set up an 'Autopin' assigment that automatically pins all reminder windows to the foreground.  Here are the steps I followed:

1. Close Outlook
2. Download and install DeskPins from http://users.forthnet.gr/pat/efotinis/programs/deskpins.html

Tips
Try to be as specific as possible when defining rules. Also, try to use both a window title and a class when possible.
Use a single asterisk to match any text. Leaving a field empty will only match an empty title or class. Note, however, that a window class cannot be empty.
Don't rely on the class of dialog windows being unique. Almost all dialog boxes in all applications have the same class ("#32770"). If you must make a rule depending on a dialog box, you should also use the dialog title.
