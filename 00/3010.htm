hansv mvp say:

You might use a macro for this. Ideally, the macro wouldn't select text but for some reason I can't get it to work that way. The following macro is a bit clumsy for that reason:

 

Sub RemoveTrailingSpaceInCells()
    Dim arng As Range
    Dim cel As Cell
    Selection.Tables(1).Select
    For Each cel In Selection.Cells
        Set arng = cel.Range
        arng.End = arng.End - 1
        If Right(arng.Text, 1) = " " Then
            arng.Select
            Selection.Collapse Direction:=wdCollapseEnd
            Selection.TypeBackspace
        End If
    Next cel
End Sub

Actually, you can get rid of leading and trailing spaces in a selection by changing the horizontal alignment to "Center" (Ctrl+E) and then back to "Left" (Ctrl+L). 

When dealing with table cells, be sure to select all the cells, but do not include the end-of-row markers (that is, select column content, but not the entire table).

rajesh c. say:

Try the following formula :   =COUNTIF(J3:J95,"<="&1000)+COUNTIF(J3:J95,">"&1000)*2

Have a look at the following image:

Here, you can change      the   "<="    to  "<"   if you do not wish to include 1000 in that part.

If you use     =COUNTIF(J3:J95,"<"&1000)+COUNTIF(J3:J95,">"&1000)*2

                      then all values of 1000 will be ignored and not counted

If you use     =COUNTIF(J3:J95,"<="&1000)+COUNTIF(J3:J95,">"&1000)*2

                      then all values of 1000 will be counted with first part (without doubling)

and

If you use     =COUNTIF(J3:J95,"<"&1000)+COUNTIF(J3:J95,">="&1000)*2

                      then all values of 1000 will be counted with second part and will be doubled

You can use the one which you desire.

Hope you find this Helpful.

ossiemac say:

Not sure that I am on the same wave length as your self. I am assuming that you want to count the values less than 1000 and then count the values greater than 1000. Is this correct. If so, then one formula needs to have 2 operators or the value 1000 will be left out of both criteria. the following formula for less than or equal to 1000 so it includes 1000.

=COUNTIF($J$3:$J$95,"<=1000")

The next formula for greater than 100 so it excludes 1000 and below and then multiplies the result by 2.

=COUNTIF($J$3:$J$95,">1000")*2

The following formula counts both less than and equal to 1000 and plus greater than 1000 and multiplies by 2 and adds the results together.

=COUNTIF($J$3:$J$95,"<=1000")+COUNTIF($J$3:$J$95,">1000")*2
