charles kenyon say:

In part, it depends on how you inserted the Cover Page. Did you use the Insert > Cover Page button?

See Cover Page Anomalies.

If inserted in Word 2010, it should actually have a page number of 0.

If this is simply a page that you typed, you'll have to fiddle around it. There are two methods:

First method: Simply set your headers/footers to have a different first page and then set the page number for your cover page at 0. Then you would not need a section break.

Second method: Put an odd-page section break at the bottom of your cover page and unlink the headers/footers for the following pages. Then you can use your different odd and even h/f and start your page numbering at 1 for the page you want numbered 1.

  • Numbering Front Matter (Ribbon) by Suzanne Barnhill, MVP

split workbook into worksheets - Microsoft Community

ginny michaels say:

split workbook into worksheets

how do I split an excel workbook into multiple worksheets based on data in column.   IT will always be the data in column B that is used.   In other words column b contains a list of names (around 20) I want to create a worksheet for each name.

vijay a. verma say:
1. Make a backup of your workbook.
2. Open your workbook and ALT+F11
3. Locate your Workbook name in Project Explorer Window
4. Right click on your workbook name > Insert > Module
5. Copy paste the Macro code given. CHANGE Sheet1 in Set Ws = Worksheets("Sheet1") as per your master sheet
6. Go back to your Workbook and ALT+F8 to display Macro Window
7. Run your Macro from here
8. Delete you Macro if the Macro was needed to be run only once.
9. Otherwise save your file as .xlsm if you intend to reuse Macro again.

Sample file @ https://1drv.ms/x/s!Akd5y6ruJhvhkT6rKMaynG3kxF91 

Sub SplitSheets()
    Dim Rng As Range, Cell As Range
    Dim Ws As Worksheet
    Set Ws = Worksheets("Sheet1")
    Set Rng = Ws.Range("B2:B" & Ws.Range("B" & Rows.Count).End(xlUp).Row)
    Application.DisplayAlerts = False
    Application.ScreenUpdating = False
    For Each Cell In Rng
        On Error Resume Next
        Worksheets(Cell.Value).Delete
        On Error GoTo 0
        Worksheets.Add(After:=Worksheets(Worksheets.Count)).Name = Cell
    Next Cell
    Application.DisplayAlerts = True
    Application.ScreenUpdating = True
End Sub
