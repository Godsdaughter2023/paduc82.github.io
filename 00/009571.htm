stanimal21 say:

It's been a while since I posted this thread, but I think I figured out why it does this.


When it formats your email, it uses HTML paragraph elements to make basic paragraphs, however I'm not sure if Outlook will actually sets any padding after the paragraph element, so it's left up to the recipient to judge for themselves, which can lead to odd email messages. The biggest problem was when I was sending email to Gmail recipients; all of my emails looked changed slightly.


Here's what I did to change that: increase the spacing between paragraph elements in Outlook 2013. Originally, I would hit the Enter key twice to add the additional space between paragraphs. That is a big no-no because it affectively puts two paragraph elements in there, and different email clients will interpret the spacing differently if you don't have something explicitly defined. In Outlook 2013 do the following:

  1. Create a new email message and then click the "Format Text" tab
  2. Click "Line and Paragraph Spacing" in the Paragraph group
  3. Select "Line Spacing Options..."
  4. Set the "spacing after" to be 9pt with Line Spacing equal to "Single"; that is the default in Word

Whenever you create email messages you won't have to press the enter key twice to get the necessary space between paragraphs. You'll have to fix Bullet spacing next:

  1. Enter some text in your email, press enter to create a new line, and then select an ordered list format
  2. Select "List Paragraph" in the Styles section of the ribbon
  3. Select "Format" on the bottom-left and then select "Paragraph"
  4. Set spacing after to 9 pt and Line spacing to Single, but then also select "Don't add space between paragraphs of the same style

Now your list styles won't have increased spacing between elements, only after the list style is complete. It seems a little weird, but that is actually a very logical and correct way to create additional spacing in emails and in Word too. Make sure you save your style changes to "New documents based on this template" so new emails will always carry these changes.


This doesn't make it perfect when someone else is viewing your email, but it does definitely help and it looks professional again. Give this a try, hope it helps. :)
