You can zoom in/out by holding down the Ctrl key while scrolling with the mouse scroll wheel. So either you're inadvertently doing this, or your keyboard has a stuck Ctrl key, or you have a mouse with side buttons which has zoom set for one of these buttons and you're inadvertently pressing this button. So:

1. What kind of mouse do you have? If you have a more-than-plain-vanilla mouse with programmable buttons, go to the Mouse applet in Control Panel to change its behavior.

2. If you only have a simple 2-button mouse (left/right click and the scroll wheel) try a different keyboard.

This is a new type of spam. It purports to have been digitally signed, and the icon you see indicates that the message contains a digital signature. In fact, the alleged signature (just a bunch of code sent with the message) is garbage. It is also classed as an attachment, which is why you see the paperclip as well.

To delete this type of message, try selecting it in the message list and pressing Shift-Delete on the keyboard. This bypasses the Deleted items folder, so you may be asked to confirm that you really want to delete the message permanently. If that works, try selecting a number of messages and shift-deleting them.

I deleted dozens of these last week and haven't seen any more since then, so perhaps the spam filter at the server (in my case Outlook.com) has been adjusted to trap them.
