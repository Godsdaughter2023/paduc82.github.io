cortig say:

Since you're syncing, I can only assume you're using an Exchange account.
With Syncing involved, it's really hard to figure out the source of a duplication. It could be an issue on any part of the chain, including on the server itself. with other apps like Contacts, an iPhone etc.
It's a major pain!
There are many AppleScripts out there that help with duplications in Outlook, but Outlook 15 blocks them all.
One option is to configure the Exchange account in Contacts and to use an app like Contacts Clearner (from Spanning Sync) to clean things up in Contacts and let it sync clean back to the server.
As to why your deletions are not syncing back… well that could be an Exchange cache corruption (which could be what got you there in the first place).
You could consider wiping out the Exchange cache, but it will force-redownload everything off the server again.

Macros are very easy to install and use:

1. ALT-F11  brings up the VBE window
2. ALT-I
    ALT-M opens a fresh module
3. paste the stuff in and close the VBE window

If you save the workbook, the macro will be saved with it.

To remove the macro:

   1. bring up the VBE window as above
   2. clear the code out
   3. close the VBE window

To use the macro from Excel:

   1. ALT-F8
   2. Select the macro
   3. Touch RUN
