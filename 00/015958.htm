I next want to add 9 months to it and save as another field (DOB+9mos).

Can anyone help me with this?

john w. vinson say:

The function you want is DateAdd. Press Ctrl-G to open the VBA editor (to get at the right help files!) and search Help for DateAdd; it lets you add integer values of any interval from seconds to years to a Date field, and it's quite aware of month lengths, leap years and so on. If you have a specific problem, please post a specific example.

Note that it would generally be BAD design to put such a calculated value into a stored field in a table. If you calculate and store a date based on the DOB, and then realize that the DOB was incorrectly entered, the derived fields will NOT be automatically corrected if they're stored; but they will if you calculate them on the fly in a Query or in an expression in a Form or Report textbox.

Once you create a document from a template (or however you attach the template) the document's variables are independent for the template's variables.

 b. use makeVariableTableFromTemplate to save the variables from the activedocument's Attached template then

 c. use copyVariablesFromTable to update the Variables in the document
