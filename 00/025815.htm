To remove the new entry that is a duplicate I have added code to my original: 

Private Sub Worksheet_Change(ByVal Target As Range)
Dim r As Range, r1 As Range, res As Variant, v As Variant
Set r = Range("A:A").Cells
On Error GoTo ErrHandler
If Application.CountIf(r.EntireColumn, Target) > 1 Then
  Application.EnableEvents = False
  v = Target.Value
  Target.ClearContents
  Debug.Print Target.Address, v, Application.Match(v, r.EntireColumn, 0)
  res = Application.Match(v, r.EntireColumn, 0)
  If Not IsError(res) Then
     Set r1 = r(res)
     r1.Select
     MsgBox "Dup here"
  End If
End If
ErrHandler:
Application.EnableEvents = True
End Sub

If you are entering the number 5 and then in another cell in column A you enter the string "5", countif counts these as being the same, but match will not see them as the same.  So this can cause problems, but I am assuming that you won't have that type of situation.

This problem may be easily solved if - and only if - you have not already linked or merged an old or legacy Skype account with what appears to be a Microsoft account.  If I am correct, linking the two accounts should import the Skype account's paid feature - the Skype Number - and credit into the Microsoft account.

Be careful about this!  If you have already linked or merged a Skype account with a different Microsoft account, this process can not be reversed.
