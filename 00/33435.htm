**** update*** I found a way to start the BITS

1. press win + R key

2. Type "regedit"

3. on /HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlSet/Control/BackupRestore check if there is a key named FilesNotToBackup, if there ain't just right click the BackupRestore and add new key named "FilesNotToBackup" without a value.

4. Leave regedit and press win + R key again, This time type services.msc

5. Navigate to BITS and try to start it.
