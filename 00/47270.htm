The 350 MB partition was the first replacement Recovery Tools partition placed there during your first 8 to 8.1 update.  The 450 MB partition was the one placed there during your second update and is probably the active partition.

You can check this by using the reagentc /info command in an administrative command prompt and if it shows the Windows RE as in Harddisk0 and Partition 5, then it is confirmed to be the active partition.

Windows RE location:       \\?\GLOBALROOT\device\harddisk0\partition5\Recovery\WindowsRE

I could not guarantee moving partitions would not cause damage to your system, but you should be able to slide the 450 MB partition over and allow space to extend the OS partition.  Removing one of the other primary partitions will leave unallocated space to do this.  And again, this can be dangerous regarding your install in general and the operation of your Recovery Options, assuming they work correctly now.

There is a possibility removing a partition after an 8 to 8.1 update might create a need to re-register the Recovery Image partition to make sure it is identified correctly. From the earlier command, that image should show in another partition, but hard to guess from your configuration what that might be. Including your readout from the command and a Diskpart listing of your partitions might be helpful.

Steps to perform SFC scan

1. Click the Start button
2. From the Start Menu, Click All programs followed by Accessories
3. In the Accessories menu, Right Click on the Command Prompt option
4. From the drop down menu that appears, Click on the 'Run as administrator' option
5. If you have the User Account Control (UAC) enabled you will be asked for authorization prior to the command prompt opening. You may simply need to press the Continue button if you are the administrator or insert the administrator password etc.
6. In the Command Prompt window, type: sfc /scannow and then press Enter

8. Be patient because the scan may take some time
9. If any files require replacing SFC will replace them. You may be asked to insert your Vista DVD for this process to continue
10. If everything is okay you should, after the scan, see the following message "Windows resource protection did not find any integrity violations"
11. After the scan has completed, Close the command prompt window restart the computer and check.
