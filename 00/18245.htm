jllatham say:

First, ActiveX buttons/controls are not macros - macros in Excel are sections of code written in the Visual Basic for Applications (VBA) for Excel language.  ActiveX controls often 'associate' themselves with one or more macros so they can respond to actions like CLICK or CHANGE - but you have to write the code to tell them what to do when they are clicked or their value changes.

 

There is also another type of control that can be placed on a worksheet; it is a Forms control.  And the solution I'm going to try to give you here will use them rather than ActiveX controls.

 

First, your question sheet layout:  It's going to be kind of long with 70 questions because one thing we need to do is set the Row Height for rows with questions and option buttons to go with them to a value of about 25 to have room to present the option buttons neatly.

 

So start by setting the row heights to 25 for all rows or just the ones that will have questions in them.  You can change non question row heights later if you choose all at this point.

 

Next let us say that your questions are going to be put into a rather wide column A.  Then there will be 3 option buttons, one each for "Yes", "No" and "Undecided" on each row.  So we could put labels in a row above the start of all of this like:

 ------A----------------------------B--------------C---------D

1----Question-------------------YES------------NO------Undecided

 

 

Now lets give the worksheet a name on its tab - for simplicity we will just name this sheet SURVEY.

Name a second sheet in the workbook RESULTS - this is where we will put the indications of which of each 3 sets of questions were selected.  By using option button groups, only 1 choice on each row can be made.  Clicking any one of them will "unclick" the other two.

 

To make this easy for you, I have prepared 2 vba/macros: one will create the option button groups on the SURVEY sheet and set up to capture their choices on the RESULTS sheet.  The second one will delete all of them on a sheet so you can start over if needed.  I will continue this discussion in a second post because it's going to get long with the code and discussion.  Here is the code you need and the instructions for putting it into your workbook (be sure to save as a .xlsm -- macro enabled workbook for reuse later) can be found at:

http://www.contextures.com/xlvba01.html#Regular

 

The code:


Sub MakeSurveyButtons()
  Const surveySheetName = "SURVEY" ' change as necessary
  Const yesColumn = "B" 'column for Yes responses on SURVEY sheet
  Const noColumn = "C" ' column for No responses
  Const maybeColumn = "D" ' column for Undecided option buttons
  Const startRow = 2 ' row the first question is on
  Const endRow = 71 ' row the last question will be on
 
  Const resultsSheetName = "RESULTS" ' change as necessary
  Const linkCol = "A" ' just need 1 column for this sheet
'end of things you may need/want to change

  Dim boxObject As Object
  Dim optObject As Object
 
  Dim whatCol(1 To 3) As String
  whatCol(1) = yesColumn
  whatCol(2) = noColumn
  whatCol(3) = maybeColumn
  Dim boxLeft As Single
  Dim boxTop As Single
  Dim optWidth As Single
  Dim optHeight As Single
  Dim optLeft As Single
  Dim optTop As Single
 
  Dim boxWidth As Single
  Dim boxHeight As Single
 
  Dim rowPtr As Integer
  Dim optCount As Integer
 
  Worksheets(surveySheetName).Select
 
  boxWidth = Range(whatCol(1) & 1).Width + Range(whatCol(2) & 1).Width _
   + Range(whatCol(3) & 1).Width
  boxHeight = Range(whatCol(1) & startRow).Height
  boxLeft = Range(whatCol(1) & startRow).Left

'gives nice neat appearance to it all
  optWidth = Range(whatCol(1) & 1).Width / UBound(whatCol)
  optHeight = boxHeight - 6
 
  For rowPtr = startRow To endRow
    boxTop = Range(whatCol(1) & rowPtr).Top
    '.add(left, top, width, height)
   
    Set boxObject = ActiveSheet.GroupBoxes.Add(boxLeft, boxTop, boxWidth, boxHeight)
    boxObject.Characters.Text = ""
    For optCount = LBound(whatCol) To UBound(whatCol)
      optTop = boxTop + 3
      optLeft = Range(whatCol(optCount) & rowPtr).Left + _
       (Range(whatCol(optCount) & rowPtr).Width / 2) - (optWidth / 2)
      Set optObject = ActiveSheet.OptionButtons.Add(optLeft, optTop, optWidth, optHeight)
      With optObject
        .Characters.Text = ""
        If optCount = LBound(whatCol) Then
          .LinkedCell = "'" & resultsSheetName & "'!$" & linkCol & "$" & rowPtr
        End If
      End With
    Next
  Next
 
End Sub

Sub DeleteALL_OptionObjectsOnSheet()
'this will delete all stuff associated with our option button groups
  Const surveySheetName = "SURVEY" ' change as necessary

  Dim anyObj As Shape
  Dim varSomething As Variant
  Dim objName As Variant
  ThisWorkbook.Worksheets(surveySheetName).Select
  For Each anyObj In ActiveSheet.Shapes
    objName = anyObj.OLEFormat.Object.Name
    If Left(objName, Len("Group Box")) = "Group Box" Or _
     Left(objName, Len("Option Button")) = "Option Button" Then
      anyObj.Delete
    End If
  Next
End Sub

jllatham say:

OK we have some code that should help you out.  If you need to change any of the Const values in the MakeSurveyButtons() module to make it agree with your questionnaire sheet layout, go right ahead.

 

To run the code and create the option button groups, you can run it from either the [View] tab or the [Developer] tab (if it is visible).

 

Initially none of the option buttons will be selected.  As people make their choices, the values on the RESULTS sheet will change:

YES choices = 1

NO choices = 2

UNDECIDED choices = 3

 

So you can use formulas like this to count them and give you the results:

Total number of yes responses:

=COUNTIF(RESULTS!A:A,1)

Total number of no responses:

=COUNTIF(RESULTS!A:A,2)

Total number of undecideds:

=COUNTIF(RESULTS!A:A,3)

 

For percentages you could go with something like these formulas where you know the number of questions asked (70 in these examples) just format the cells as % to show as many decimal places as you want.

=COUNTIF(RESULTS!A:A,1)/70 ' for % yes

=COUNTIF(RESULTS!A:A,2)/70 ' for % no

=COUNTIF(RESULTS!A:A,3)/70 ' for % undecided

 

If you think they may not even make a choice for every question, you could get % of number they responded to with formulas like these

=COUNTIF(RESULTS!A:A,1) / COUNTIF(RESULTS!A:A,">0") ' true yes percentage

=COUNTIF(RESULTS!A:A,2) / COUNTIF(RESULTS!A:A,">0") ' true no percentage

=COUNTIF(RESULTS!A:A,3) / COUNTIF(RESULTS!A:A,">0") ' true undecided percentage

 

By the way: after you set up your questionnaire and are ready to send it/them out, then people will probably be concerned if they get alerted that the workbook has macros in it.  So to get rid of the macros (which aren't needed after you get it all setup) you can either use File -- Save As and choose the .xlsx file type, or just go into the VBA editor and delete the code module itself and save the file.

 

I hope this helps you get moving with your project.

 

Forgot to mention that you can hide that RESULTS sheet so that people don't get curious about it.  Probably a good thing to do.  The VBA code doesn't care if it is hidden or not.
