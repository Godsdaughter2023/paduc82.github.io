jessen p say:

Hi Sayuri,

 

AppLocale is a tool for Windows XP and Windows Server 2003 by Microsoft.  It is a launcher application that makes it possible to run non-Unicode (code page-based) applications in a locale of the user's choice.

 

The program installs itself in a subfolder of the Windows directory called "AppPatch", and when launched prompts the user for an executable to run and the desired codepage. It can also create a shortcut in the start menu, located under Microsoft AppLocale, however you will be prompted by AppLocale before the program's launch.

 

Running AppLocale on Windows Vista or Windows 7 is not officially supported and it fails to install on default configurations of these OS if attempted.

 

Hope this information helps.
