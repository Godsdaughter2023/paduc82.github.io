Pass value between two form within Navigation form control - Microsoft Community

abdul-majid say:

Pass value between two form within Navigation form control

I have a question

I have a access database with many forms and also with Main navigation control form. I have also attached the forms in navigation form. Now within this navigation form, I want to pass a text box value on other form.

If any body help in this regard, it will be really appreciated.

Thanks in advance.

karl donaubauer say:

Hi,

you can't. At least not directly. Unlike a classic main form with tab control + subforms a navigation form always has only 1 currently active/present subform that changes to the next single current subform as soon as you choose another tab. That's the main reason why I don't use navigation forms.

So to pass a value to a different subform you would e.g. have to use a kind of "buffer" control/variable or tag property etc. in the navigation form or elsewhere to hold the value until the other subform can pull it (after becoming the current subform).

abdul-majid say:

Thank you Karl Donaubauer for your kind reply.

Can you please help me how can I pass value using a variable...?

How to store a textbox value in a variable and then access this variable on other subform?

karl donaubauer say:

Hi,

there are many ways to do this that may depend on your details. e.g.

In a fitting event of the first subform, e.g. AfterUpdate of the textbox, you store the textbox value in the Tag property of the navigation form:

Me.Parent.Tag = Me!MyTextbox

In a fitting event of the second subform, e.g. OnOpen, you check the Tag property for a value:

If Me.Parent.Tag > "" Then
    'do sth with the value

    MsgBox Me.Parent.Tag
   'empty the tag for next use
    Me.Parent.Tag = ""
End If
