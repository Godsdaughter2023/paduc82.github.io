rich007a say:

Something like this will do what you need, although there are probably more efficent methods.  You could also consider using a VLOOKUP function in your spreadsheet, instead of using VBA.

 

Here's the code:

Sub CopyLookup()
    Dim rng1 As Range, c1 As Range, rng2 As Range, c2 As Range
    Dim ws1 As Worksheet, ws2 As Worksheet
    Dim lnLastRow1 As Long, lnLastRow2 As Long
   
     'Create an object for each worksheet:
    Set ws1 = Worksheets("Wrksht1")
    Set ws2 = Worksheets("Wrksht2")
   
     'Get the row number of the last cell containing data in each sheet:
    lnLastRow1 = ws1.Cells(ws1.Cells.Rows.Count, "B").End(xlUp).Row
    lnLastRow2 = ws2.Cells(ws2.Cells.Rows.Count, "A").End(xlUp).Row
   
     'Create range objects for the two columns to be compared:
    Set rng1 = ws1.Range("B2:B" & lnLastRow1)
    Set rng2 = ws2.Range("A2:A" & lnLastRow2)
   
     'Loop through each cell in col A in sheet 2:
    For Each c2 In rng2
       
         'Check if the cell is not blank:
        If c2.Value <> "" Then
       
             'Loop through each cell in col B in sheet 1:
            For Each c1 In rng1
               
                 'Test if cells match:
                If c1.Value = c2.Value Then
                   
                     'Copy value from sheet 2 to sheet 1:
                    c1.Offset(0, 5).Value = c2.Offset(0, 5).Value
                   
                     'Move on to next cell in sheet 2:
                    Exit For '(exits the "For Each c1 In rng1" loop)
                   
                End If
            Next c1
        End If
    Next c2
End Sub

a. user say:

You can't transfer your activation from one computer to another. Starting with Windows 10, activation binds your copy of Windows 10 to the device on which it is installed.

This may or may not matter in your case. You are able to upgrade any number of computers to Windows 10 for free, as long as they're currently running Windows 7 SP1 or Windows 8.1. If your 'other' computer is already running one of those, just wait until July 29 (in reality, you may need to wait a few days after July 29) then download the bits from Microsoft and do the deed.

If your other computer is not running one of those two editions of Windows, you can still get the bits but you'll have to pay to activate Windows 10.

The thing to remember going forward is that activation binds an installation of Windows 10 to the hardware on which it's installed. Once activated, you can reinstall or clean install on the same device until the cows come home, with no need for a license key. You'll only need a Windows 10 license key the first time you install on a device that isn't already running one of those earlier versions of Windows, and the license key only proves that you have the right to install Windows 10 (because you paid for it.) Once installed, you can toss your license key into the trash because you'll never need it again.

Blue arrows on word - Microsoft Community

a. user say:

Blue arrows on word

My word has little blue arrows on every single document after I press retun/enter and I can't work out how to get rid of them. For example::: How do I get rid of the blue arrows

bob jones aka: cybertaz say:

These are some of what are referred to as non-printing characters which include Tabs, Spaces, Section Breaks, Optional Hyphens & Hidden Text in addition to Hard/Soft returns (Paragraph Marks & Line Breaks). IOW, they indicate impositions that are responsible for how the text flows as it does, but they do not print. They are indispensable when editing, revising or 'troubleshooting' a document.

The article for which Charles provided the link pertains to Windows versions of the program. Since you're using a Mac they can be turned Off by clicking the Show/Hide ¶ tool in the Paragraph group on the Home tab of the Ribbon or the Mac Word keyboard shortcut Command+8.

However, certain aspects of your screen shot suggest that something else may be at play such as having an East Asian Language enabled. With an East Asian Language enabled the blue ↵ is the equivalent of the ¶ - It denotes the end of a paragraph. Unlike the ¶, though, the default is for it to remain visible even when other no-printing characters are not being displayed. In order to have it toggled Off it's necessary to go to Word> Preferences - View to clear the check for Paragraph Marks.

If you really don't need to use East Asian scripts, restore Word to your Default Language by going to Word> Preferences - East Asian Languages to select it from the dropdown, then confirm updating the Normal.dotm when you Quit Word..

If these suggestions don't toggle them Off please include your full release & Version update level of Mac Office: That info can be obtained by going to Word> About Word in the main menu.

john korchok say:

In my tests, once you zoom in, the screen stays zoomed while you pan around. Moving your mouse moves the spellcheck dialog out of the viewing area. Does that sound like what's happening at your end?

Unfortunately the zoom feature is a poor workaround for constant use, but here are the settings I find most useful in System Preferences>Accessibility>Zoom:

  • Check Use keyboard shortcuts to zoom. This makes it convenient to turn the feature on and off.
  • Set Zoom style to Picture-in-picture, then click on the Options button.
  • Set Window position: to Tiled along edge.

With these settings, most of your screen remains unzoomed. You move your mouse over the spellcheck box to see it magnified at the side of the screen.

Another possible workaround it to plug a secondary monitor into the display port of your MacBook. In System Preferences>Displays, set the secondary monitor to Scaled and choose a lower resolution. This turns the secondary monitor into a permanently zoomed display. Drag the spellcheck box onto this monitor and leave it open between spellcheck use so you don't have to keep dragging it back from your main display.
