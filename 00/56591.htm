Character Maps/Glyphs

I'm trying to put glyphs in with my fonts.  I'm in character maps in the advance part, and have selected the glyphs I want. The tutorial I am using tells me to move them into the silhouette studio. I have the cricut, but is there any way to put them in with my fonts?

Thank you in advance.

naveen_m say:

Hi,

Thank you for writing to Microsoft Community Forums.

To get your font into Silhouette Studio software:

1. Close the Silhouette Studio software if it is open.

2. Download and install the font you want on your system.

3. The font file is compressed. Unzip the folder, and open it.

4. Once it is open, you'll get a prompt to install.

5. Open Silhouette Studio.

6. Click the font icon, and you can see the new font in the font menu. 

To add glyphs in Silhouette Studio:

1. Open Character Map.

2. Change Character Set to Unicode, and change Group by to Unicode Subrange.

3. Select Private Characters in the selection box.

4. Find your font.

5. Select the glyph you want. Press 'Control + C' key together to copy it.

6. Go back to Silhouette Studio, and highlight the letter you want to change.

7. Press 'Control + V' key together to paste the glyph in.

Hope it helps.

Naveen M
