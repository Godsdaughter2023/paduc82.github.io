I had this very same problem when trying to send email using Windows Live Mail from my default Outlook.com address.  Up popped Server Error: 5204, Windows Live Mail Error ID: 0x8DE00006.  I could send emails through Windows Live Mail if I changed to a different (Gmail or Road Runner address).

I talked to my ISP and also Microsoft support and they could see nothing wrong with the way things were set up and could provide no solution to the problem.  After trying some of the other suggestions I had found online and getting nowhere, I did some investigating into my browser add-ons and discovered that the extensions for Windows Live had been disabled.  I enabled them, restarted the computer and now Windows Live Mail is working fine.  I should note that I do not bother signing in to Windows Live.  There is no need to.  I can send and receive all accounts (Outlook, Road Runner and Gmail) at once, automatically when the program opens up.  When it works like it is supposed to, it is a very convenient program.

What caused this problem for so many people?  Looks like it may have come through a software update for either the browser, email program or operating system.
