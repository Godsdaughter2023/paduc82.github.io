grschinon say: You haven't given many details so I'm going to assume that you installed the applications in question onto your memory card and that you then transferred that physical memory card to another phone and expected the applications to be there.

It doesn't work that way, I'm afraid. Unless you actually install the applications on the new phone, it can have no knowledge of the fact that they're there on the memory card and therefore won't show them in your menu.

You have to re-install them from the original packages for the new phone to know they're there. If they were applications pre-loaded onto the phone before purchase then you can't re-use them because they were packaged for use on that original phone alone.
