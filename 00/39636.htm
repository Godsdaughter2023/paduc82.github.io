No help there. It seems to be a threshold problem with memory manager in Windows7. I get the warning when free memory gets below 1GB, but I still have plenty of available memory in standby.
Standby memory is not available... it is being used to hold cached data or cached application code for some multi-tasked process.  To make that memory available, windows would have to write it to the page file (windows swap drive) or finish the process.  It would appear that you are running multiple applications or are running application(s) that run multiple processes.  You can take a look at the task manager for some clues, but that may or may not be easy to decipher.   I'd look at what you have set to start up automatically at boot- that's probably a more fruitful approach.  And a brand new laptop is probably loaded with 'crapware' (all that pre-installed stuff including free trials of this or that...) that you should go thru and prune...
In any event, this is how Windows multi-tasks.
