a. user say:

You're right that the clicking sound almost always means the hard drive has failed, is about to fail, or has some serious problems and will fail shortly.  Why they only occur with Vista and not with Kubuntu is beyond me - it should make no difference (unless the drive is partitioned and the problem is only on the Vista partition).

To test your hard drive, check the manufacturer and then get the diagnostic utility from:http://www.techsupportforum.com/hardware-support/hard-drive-support/302602-hard-drive-diagnostic-utilities.html (and/or get one from your hard drive or computer manufacturer if they have one available – it wouldn’t hurt to try both).  If it fails the test, replace it.  If it passes the test, then there's probably nothing wrong with it.

If  that checks out, try checking the drive for corruption.

Go to Start / All Programs / Accessories / Command prompt and right click on command prompt and click run as Administrator.

While in Command Prompt, type chkdsk /f /r and enter and let it run.  It will want to schedule itself to run at the next restart.   Answer yes and then reboot to run the program.  It will scan and try to fix any corruption or bad sectors on your hard drive and mostly remove that as a potential cause.

Let's see what these procedures reveal before deciding on next steps.

I hope this helps.

Good luck!

rajithr say:

Hi,

 

Have you you looked Program and Features list and see if you .Net Framework 3.5 is enabled.

a. Press Windows key + X, choose Program and Features.

b. Click on Turn on Windows Features on or off.
c. Check if .NET Framework (includes .NET 2.0 and 3.0) is enabled.

 

You can also refer to this article: http://blogs.technet.com/b/aviraj/archive/2012/08/04/windows-8-enable-net-framework-3-5-includes-net-2-0-and-3-0-i-e-netfx3-feature-in-online-amp-offline-mode.aspx

 

You can use the Windows installation media as the file source when you enable the .NET Framework 3.5 feature.
To do this, follow these steps:


a. Insert the Windows installation media.

b. At an elevated command prompt, run the following command:  To open Command prompt in elevated mode, follow these steps.
a. From Start screen, type cmd, right-click on Command Prompt and choose Run as Adminstrator.

 

Now,type the command (Please note the spaces between commands)


Dism /online /enable-feature /featurename:NetFx3 /All /Source:<drive>:\sources\sxs /LimitAccess

Note In this command, <drive> is a placeholder for the drive letter for the DVD drive or for the Windows 8 installation media. For example, you run the following command:

Dism /online /enable-feature /featurename:NetFx3 /All /Source:D:\sources\sxs /LimitAccess

 

Hope this helps.
