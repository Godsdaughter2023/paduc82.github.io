Vista Win Mail Contacts to WM6 without using Outlook - Microsoft Community

mikep224 say:

Vista Win Mail Contacts to WM6 without using Outlook

I've been going nuts trying to find a way to get my Windows Mail contacts from my PC to my WM6 phone (without installing Outlook on the PC) and finally found a way to do it!  Let me preface this by saying I'm by no means an expert at any of this and I am using an AT&T Tilt.  I don't know if it will work the same on other devices but, here's what worked for me:

 

1. Create a folder called "Contacts" on your PC (where ever you want. I placed it on the desktop)

2. Export your Windows Mail contacts as a V-card file to the folder "Contacts"

3. Sync this folder to your WM6 device (I placed it in the "Documents" folder)

4. Navigate to this folder on the device and open it

5. Click on the V-card name.  When the message asking if you want to save it pops up click "yes".  You'll have to click on each file and save it separately (I had 52 v-cards and it took about a minute do all of them).

6. Once you've saved all the v-card files they can be deleted to save space.

 

Hope this helps...

Just a note on your assumption that you can only paste 2 charts into placeholders. This is only true if you stick to the stock slide layouts that Microsoft designed. If you open the slide master view, you can create additional slide layouts and each one of them can have as many content placeholders as you want. Since link editing is very important to you, I recommend you consider using the Windows version of Office, as it has better (though still imperfect) tools for this.
