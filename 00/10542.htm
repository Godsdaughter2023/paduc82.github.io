ken sheridan say:

Most of them are all showing for example as 12.029856656 and I want to change the format so it only shows as 12.02

Note that formatting 12.029856656 to two decimal places will not return 12.02, but 12.03.  This is because the number is rounded up or down on the basis of the numbers at the third decimal place and beyond.  By default Access uses 'banker's rounding' (round half even) which is designed to balance out rounded values over a random set.

If you want to round by truncation towards zero to two decimal places then you can do so with a function such as the following, which is based on a method originally described by James Fortune in the old MS Access newsgroup:

Public Function RoundToZero(dblVal As Double, dblTo As Double) As Double

    Dim intUpDown As Integer
    Dim lngTestValue As Long
    Dim dblTestValue As Double
    Dim dblDenominator As Double
   
    If dblVal < 0 Then
        intUpDown = -1
    Else
        intUpDown = 1
    End If
   
    dblDenominator = intUpDown * dblTo
    dblTestValue = dblVal / dblDenominator
    lngTestValue = Int(dblTestValue)
    RoundToZero = intUpDown * lngTestValue * dblTo

End Function

You can see how it works in the debug (aka 'immediate') window:

? RoundToZero(12.029856656,0.01)
 12.02

The second argument determines the interval to which to round the value, in this case 0.01.

cbarnhorst say:

Build 14393.105 is the current release build, not an insider build, so you are safe to stop getting insider builds (because you have not received one yet) without having to reinstall anything.  It also means that you are not in the Fast Ring.  However, you could be in the Slow ring and about to receive a preview (we don't know when Slow ring will get its first preview build).  If you are in the Slow ring you need to stop receiving insider builds. 

If you are on the Release Preview ring you will NOT receive any preview builds so it doesn't matter if you "stop receiving insider builds" or not because the RP ring doesn't ever get them anyway.  All the RP ring insiders get are the updates to the Anniversary Update (build 14393) that everybody else does, but about a week early.  The idea is that you may spot a problem with the update and give Microsoft advance warning before the whole world gets the update.  Hence the name "release preview."  You don't need to bother making any changes if you are in the Release Preview ring unless you just want to.  You will stay on build 14393 until a new version of Windows 10 is released to the world next year.

The ring is indicated in a drop down menu further down the page you are having issues with.  If you really are set to receive insider builds.  So far it is not clear to me that you really are.  When you finally get to that point you should be able to tell easily.  All I can confirm is that you are not in the Fast Ring.

Frames anchored to a given paragraph can be moved to any location relative to the paragraph. Just drag the frame where you want it or use the Frame dialog to change the vertical position relative to the paragraph.

A text box can be positioned "In line with text," which means that it behaves as a (large) text character, if that helps.
