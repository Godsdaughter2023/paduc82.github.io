jay freedman say:

First, double-click in the header area (or click Insert > Header > Edit Header) to open the header pane. That causes the Header & Footer Tools ribbon to display. If you already have anything in the header, select it -- if that's the material you want on the first page header, cut it to the clipboard, or otherwise just delete it. _elaine_ say:

Hello, DonnaWilliams1

It depends: unfortunately, in most cases, that is not the purpose of the conference ID.  The conference ID is a code that you will need to type using the dial pad to access the actual conference call.

Look also in the email for a list of the conference bridge access numbers, most of which should be toll-free numbers.  If there is no toll-free number listed, and you call from Skype, you will need at minimum Skype Credit to cover call costs.

If the "conference ID #" is indeed a stand-alone phone number, then you could use that number to dial in to the conference call.

Another option is the conference ID could be a link.  When you click on the link, you may be asked to install a web browser plug-in, which would take the place of a telephone number.  Web conference services such as Skype for Business offer this option.

Kind regards,

Elaine
