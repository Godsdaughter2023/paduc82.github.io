Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
    Dim rngCell As Range
    Dim TimeStr As String
    On Error GoTo EndMacro
    Application.EnableEvents = False
    For Each rngCell In Target
        With rngCell
            If .Value <> "" And .HasFormula = False And InStr(rngCell.NumberFormat, ":") > 0 Then
                Select Case Len(.Value)
                    Case 1 ' e.g., 1 = 00:01 AM
                        TimeStr = "00:0" & .Value
                    Case 2 ' e.g., 12 = 00:12 AM

                    Case 3 ' e.g., 735 = 7:35 AM

        End With

    Next rngCell
    Application.EnableEvents = True
    Exit Sub
EndMacro:
    Resume NextCell

As I understand Outlook do not download images for emails which are marked as safe senders.

 

By selecting the option Never block sender or Never block sender's domain will add the recipient email address or domain to Safe Senders list. Hence the email sent by that person or domain will not be blocked and will be received in Inbox folder and not in Junk email folder.

·         Open Outlook.

You may add approximately a total of 2000 email addresses in safe and blocked senders list because the total size of all lists (both safe and block senders combined) cannot exceed 512K.
