You will always get that warning when saving in any of the file types you mention. They are all basically text/data-only file types -- a list of records (Rows) with fields (Columns) separated by commas. Some examples:
  • There are no A,B,C.../1,2,3 column/row headings, gridlines or any other aspects of  worksheets,
  • Only the content of the currently displayed sheet is saved [those file types don't support multiple worksheets]
  • No formatting other than currency is preserved,
  • No graphic content is included,
  • Only the results of calculations are retained [not the actual formulas]

steve rindsberg say:

This probably has more to do with the physical limitations of your printer than it does with PowerPoint.

Most printers have an unprintable margin at the edges.  They physically can't print any closer to the edge than that.  PowerPoint generally scales its output to fit the printable area reported by the printer driver.  That's what the "Scale to fit paper" setting does.

If the printer driver has a setting that allows you to print clear to the edge, enable that then try again.

And also try removing the check next to "Scale to fit paper"
