tom van stiphout say:

Ignoring your current SQL statement for a moment, and renaming fields to be without funny characters (thus avoiding potential notational difficulties), the total cost of an invoice is:
select ip.InvoiceID, Sum(p.ProductCost*ip.QuantityOrdered) as TotalCost
from Product p inner join InvoiceProduct ip on ip.ProductID=p.ProductID
group by ip.InvoiceID

scottgem say:

Let me add some other points.

I don't think your structure works. What happens when a Product's cost changes? Your query will show the current value no matter when the invoice was generated. Time sensitive data is the main exception to the no redundancy rule.

Another question is whether invoices are different from orders.

Your structure should look more like this:

Invoice

InvoiceID (PK)

EmployeeID (FK)

InvoiceDate

InvoiceProduct

InvoiceProductID (PK)

InvoiceID (FK)

ProductID (FK)

Quantity

UnitCost

Product

ProductID (PK)

ProductName

UnitCost

From there I would use 2 queries. One for your invoice line items

SELECT InvoiceID, ProductID, Quantity, UnitCost, Quantity*UnitCost AS ExtendedPrice

FROM InvoiceProduct

ORDER BY InvoiceID;

Call it qryInvoiceLine

Then to get the total per Invoice:

SELECT InvoiceID, Sum(ExtendedPrice) AS TotalInvoice

FROM qryInvoiceLine

GROUP BY InvoiceID;

You can then join these with your other tables to pull all the info you need for an Invoice.

bernie deitrick say:

I have assumed your example is in columns A and B.  You can do what you want if you add data validation with your list of allowed values to column C, and use the worksheet's change event to add the new selection to the cell on the same row in column B: copy this code, right-click the sheet tab, select "View Code" and paste the code into the window that appears:

 

Private Sub Worksheet_Change(ByVal Target As Range)
    If Target.Cells.Count > 1 Then Exit Sub
    If Target.Column <> 3 Then Exit Sub
    Application.EnableEvents = False
    If Target.Offset(0, -1).Value = "" Then
        Target.Offset(0, -1).Value = Target.Value
    Else
        If InStr(1, Target.Offset(0, -1).Value, Target.Value) = 0 Then
            Target.Offset(0, -1).Value = _
            Target.Offset(0, -1).Value & ", " & Target.Value
        End If
    End If
    Application.EnableEvents = True
End Sub

 

When you filter, choose the custom filter "includes" then type fred into the value.
