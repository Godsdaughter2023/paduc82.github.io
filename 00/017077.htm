rohn007 say:

Yes, the "Retail" license version of Office in the "big box" for Office 2010 (and earlier) allows you to install on 2 computers at the same time.  That license term was originally written at a time when "everyone" had desktop computers and laptops were rare and exceptional.  But as far as MS is concerned they just care about you limiting installations to your "licensed" device and a secondary "portable" device. There is no explicit exclusion preventing you from installing on 2 "portable" devices.

Here are the specific license terms copied from the MS license terms document:

<snip>

One Copy per Device.

You may install one copy of the software on one device. That device is the “licensed device.”

c. Portable Device.

You may install another copy of the software on a portable device for use by the single primary user of the licensed device.

</snip>

rohn007 say:

it depends on exactly what type of Office 2010 license you bought, ie "how" you bought it. 

If you bought the "retail" license in the "big-ugly-yellow-plastic-CD-holder-box" then the answer is yes. This type of license allows you to install on up to 2 machines at the same time and to transfer the installation(s) to new computers as required.  (NOTE: in Office 2013 the license terms were "improved" and now only allow install on one computer at a time. <sigh> )

If you bought a "product Key Card" or "online download" then the answer is no. This type of license only allows you to install on one computer period. It cannot be transferred to a new computer. When the computer dies or is sold, the license goes with it.

maxwell joseph say:

Font Substitution in Publisher

How to avoid font substitution in Publisher?
mukherjee ashutosh say:

To avoid font substitution, do one or more of the following:

Embed TrueType fonts (TrueType font: A scaleable font that displays and prints smoothly at any point size. TrueType fonts are directly supported by the Windows operating system and can be embedded in publications.) in your publication. When you embed TrueType fonts, they are saved within your publication. Publications with embedded fonts can display and print the text in the original fonts, even if those fonts are not normally installed on the computer you are using.

 Note   Only TrueType fonts can be embedded and only if they are licensed for embedding.

If you are taking your publication to another computer, verify that the computer has the same fonts installed that you used in your publication.

If you used PostScript fonts (PostScript font: A scaleable font that displays and prints smoothly at any point size. PostScript fonts are based on the PostScript page description language. Also called a Type 1 font.) in your publication and you are taking it to a commercial printer, ask whether the printer has the fonts that you used or will purchase them.
