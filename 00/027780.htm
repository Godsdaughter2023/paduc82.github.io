hansv mvp say:

Use the / operator for division.

Examples:

Divide 12 by 3:   =12/3

Divide the value of cell A1 by 3:   =A1/3

Divide 12 by the value of cell A2:   =12/A2

Divide the value of cell A1 by the value of cell A2:   =A1/A2

teylyn say:

Hello,

in addition to what Hans wrote, you can use the / operator inside a formula as well, for example

=match(A1/A2,B1:B10,0)

This will first divide the value of A1 by the value of A2 and then perform a match with the result.
