In Word 2010, there seems to be no alternative to selecting Header, Footer, or Page Number from the Insert tab on the ribbon and then selecting from a very long drop-down list of decorative entries that don't do what I want; for example, all the "Page X of Y" entries are in boldface and separate the values with only a single space.

Is it possible to enter the specs the old way, by (say) tabbing to the center of a footer and typing "Page  " then inserting the number placeholder and "  of  " and inserting the total-pages placeholder?
hansv mvp say:

To insert the current page number:

  • Place the insertion point where you want the page number.
  • Press Ctrl+F9 to insert field brackets { } (do *not* type { } yourself)
  • Type the word PAGE between the brackets.
  • Press F9 to hide the field code and to update the number.

 

To insert the total number of pages:

  • Place the insertion point where you want the total number of pages.
  • Press Ctrl+F9 to insert field brackets { } (do *not* type { } yourself)
  • Type the word NUMPAGES between the brackets.
  • Press F9 to hide the field code and to update the number.

This is how I figured it out using my touchscreen laptop with 8.1 and Office 2013 installed.

 

Go the Tile START screen (says Start upper left),

 

Scroll down or use your finger to move to the screen hidden below which shows all the apps installed as small tiles (mine says Apps by name). 

 

Then scroll right or use your finger to move to the screen hidden to the right.  Keep scrolling until your find the Office list with all the apps (Word, Excel, PowerPoint, etc).

 

Either use your mouse to right click or finger to touch and hold to select the tile.

 

A toolbar will appear at the bottom (mine is green). 

 

Select "Pin to Start" and the larger tile will appear on the Tile Start screen.  Move it around where you want it.

 

I had to do this one at a time.  I'm sure there is a batch way to do it but I have not figured it out.

jack cook - aka help_line say:
The display languages included on your Windows Phone are set during manufacturing by the original equipment manufacturer (OEM) and mobile operator based on languages that are available from Microsoft. There may be multiple display languages installed on the phone by the OEM, but each OEM installs a different set of languages.

The Windows Phone update process cannot add or remove display languages. Although newer versions of the Windows Phone software might add support for additional display languages, the new languages will only be available on new phones. After you update your phone, you'll continue to have the same display languages that you had before the update.

Here are the steps to view available display languages:
  1. From the Start screen, flick left and tap Settings .
  2. Tap Region+language.
  3. Tap the language listed under Display language to see which other languages are available.
