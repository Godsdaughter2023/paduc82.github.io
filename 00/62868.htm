Windows 7 SP1 is provided in two ways, either as a full .ISO file download weighing in at 2.4 GBs or as an update available through Windows Update. 1. Make sure you install the latest updates for Windows 7, your applications and device drivers. This can ensure a smooth experience I noticed, although Windows 7 SP1 is minor, you don’t want to take any chances. If you are updating through Windows Update, it also guarantees a smaller update since only missing patches will be applied.

Windows 7 has made it very easy tobackup your computer . The built in system imaging and backup and restore center makes it convenient to back up the operating system and all your critical data. ramesh srinivasan say:

I think the correct answer to my question at the beginning of this thread is:

Win 10 is designed to have three Desktop folders (four if the user pins one to Quick Access) and all three are system folders that cannot be removed or renamed. One of them (C:\Users\(user name)\Desktop) is the actual Desktop folder. The second one (This PC\Desktop) is shortcut to the first. The third one (Desktop\This PC, etc) is not is not a desktop folder, but rather a super folder that would be more correctly called Namespace.

(and if you want to make a post of the above paragraph, I will mark it as the answer to my question.)

It's 4 if you turn on "Show all folders". Otherwise it's 3.

All of them point to "C:\Users\{Username}\Desktop"

1st one in the Quick Access

2nd one is "This PC\Desktop"

3rd one is under the the drive letter, which is the real path to user desktop.

Yes. "Desktop\This PC" is the "super folder" as you say. Shows everything in your computer, files/folders, namespace objects, your profile, all users profile etc.
