keith a. miller say:

You don't see 'Open Command Prompt' because it's been replaced by 'Open PowerShell Window Here'. That's fine, because PowerShell is even better suited to your need.

PowerShell can export the output of a dir (= gci = get-childitem) to a comma-seperated (.csv) file which can be gracefully imported into Excel.

Take a moment to check it out. Right-click in the background of a folder & select 'Open PowerShell window here', Type:

dir<Enter>

You'll see output very similar to the 'dir' command in Command Prompt. It's a default view that only displays a few properties of the items.  But when you export the output, you get a complete set of properties. The format for this is:

dir | export-csv c:\users\keith\documents\filelist.csv

If you want listings of subdirectories as well, use:

dir -recurse | export-csv c:\users\keith\documents\filelist.csv

devlin sco say:

Hi Jacob,

Changing the default installation path of programs can be done through the Registry Editor.

Please make sure to back up your registry first before making any changes to avoid issues.

To change the default installation path of programs, please follow these steps:

  1. Click the Start button.
  2. Type regedit.
  3. Right-click regedit and select Run as administrator. Click Yes or type your password if prompted by the User Account Control (UAC).
  4. On the Registry Editor window, navigate to HKEY_LOCALMACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion.
  5. Double-click both ProgramFilesDir and ProgramFilesDir (x86).
  6. Change the Value data to the new path for the 2TB hard drive.
  7. Click OK
  8. Close the Registry Editor window.
  9. Restart your computer.

IMPORTANT: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, refer to the following Microsoft Knowledge Base article.

How to back up and restore the registry in Windows.
