ken blake - microsoft mvp say:

On Fri, 18 Feb 2011 20:26:25 +0000, Destanee wrote:

I was wondering what the D drive is used for.

The D: drive is simply the second drive in your list of drives. There
is no standard as to what kind of drive it is or how it is used. It
could be a second hard drive, a second partition on your only hard
drive, a CD drive, a DVD drive, a thumb drive, an external hard drive,
etc. None of us can be sure what it is unless you tell us.

I'll make a guess (but again, I'm not sure) that on your machine it's
a second partition on your single physical drive. If my guess is
right, read on.

It can be either of two things:

1. A recovery drive, created by your computer's manufacturer to
hold what you need to restore the computer to factory condition,
should you ever need to or want to. It is not there for you to write
anything to it, or do anything else to it.

2. An empty drive for you to use to store whatever files you want to.
Think of your drive as a two-drawer file cabinet, with one drawer
called C: and the other called D:. You have no choice as to where
Windows goes (it will be on C:) but all the data files you create can
be put wherever you want them.

balaji kundalam say:

As Thomas Ferguson says, C partition/drive contains system files that handle Windows boot process and so, can't be formatted the way other partitions are being formatted. Similarly, it's not possible to change the drive letter of C partition.

The reason is that all the system files will be in use, so obviously locked from modification. Hence Windows will not be able to make any changes to the files. Thus, the partition can't be formatted.So to format C partition, as Vishal Singhal says, you need to boot from Windows installation media and then enter Repair your computer -> Command Prompt and format manually. Here, kindly note that Windows may fail to boot the next time, so you'll be performing Startup Repair, which will again create files in C partition. 

Furthermore, if you've installed Windows in both C and D partitions, note that the real C partition appears as D when viewed from Windows installed in D partition and the real D will appear as C. This is because, when Windows (1.0) first came, it was floppy disks being used as primary storage and starting Windows 3.1, hard disk is being used as primary storage. After two floppy drives, drive letters A and B, the obvious character for hard drive is C (which is the primary storage then), and that legacy is being maintained throughout Windows family, even in Windows 10.

Now, the question that strikes immediately is, why do you have to format C drive?

How can i share my large files with my friends? - Microsoft Community

a. user say:

How can i share my large files with my friends?

How can i share my large files with my friends?

a. user say:

I'm tempted to say "the same way that you share the smaller files" but I suspect you're having some sort of problem that you didn't describe sharing larger files.  At what file size do you start hving problems?  How are you sharing your files (email, messenger, over a network, whatever) and what is the EXACT name and version of the program you are using?  What error message do you get when you try to share a file too large (the EXACT message including any codes)?

One solution is to compress the files so they are smaller in size.  This sometimes helps a lot and sometimes does hardly anything (mostly depending on the file type)  - but it's worth a try.

Here are two articles on setting up right-click compression in Vista: http://www.vistax64.com/tutorials/65254-zip-file-support-restore.htmlandhttp://www.vistax64.com/tutorials/93827-zip-file-compressed-folders.html.These articles should tell you how to turn on and turn off compression and reset compression options in Vista.

If you can't get Vista compression/decompression to work properly despite these instructions, you might want to just try downloading and installing 7-Zip http://www.7-zip.org/ which provides essentially the same functionality in a free, easy-to-use program.  Many people use this program (or others like it) instead of Vista's program for a variety of reasons.

Amother option with certain file types is to convert them to other file types which take up less space (like converting a TIFF file to a JPG file).  It's not exactly the same, but most people would find it difficult to notice the difference (and sometimes the difference isn't visible to the naked eye or on a monitor).  To help here, we would need to know the file types of these large files.

I hope this helps.

Good luck!
