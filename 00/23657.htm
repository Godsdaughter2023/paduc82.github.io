The code as you have it written closes the workbook running the code from what I can see.

Assume you want to paste to the sheet that is active when the macro is run:

Dim bk as workbook, sh as worksheet
Dim sh1 as Worksheet

  myfilename1 = Application.GetOpenFilename

  set sh1 = Activesheet

  set bk = Workbooks.Open(Filename:=myfilename1)

  sh.Range("A1:IV65000")copy sh1.Range("A1")
  Application.CutcopyMode = False
  bk.Close Savechanges:=False

The Creators Update removes the 'Internet Explorer' entry from the 'Turn Windows Features on or off' menu in the old Control Panel. Various people have reported that it may also remove iexplore.exe itself, on systems where it was previously installed.

To reinstall the Internet Explorer, you need to use the new 'Windows Settings' app: the one that is represented by a cogwheel and uses the new 'Universal Windows Platform' style.

These are the steps:

1. Launch the Windows Settings app.

2. Click on Apps.

3. Beneath the heading 'Apps and Features' there's an entry called 'Manage Optional Features'. Click on it.

5. A new menu opens. Click on 'Add a Feature'.

6. Internet Explorer 11 should be among the options. Click on it.

7. Wait briefly, then reboot.

8. The Internet Explorer should now be installed.
