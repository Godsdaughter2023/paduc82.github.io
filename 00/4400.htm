Hello,

 

I was finally able to figure out why my variables where not working.  Below is the working code for anyone who may find it useful:

 

Option Explicit
Sub DwnldFXTable()

Dim oWeb As QueryTable
Dim sWebTable As String
Dim strValue As Date
Dim strDay As Integer
Dim strMonth As Integer
Dim strYear As Integer

sWebTable = 2

strValue = Range("A1").Value
strDay = Day(strValue)
strMonth = Month(strValue)
strYear = Year(strValue)

Set oWeb = ActiveSheet.QueryTables.Add( _
Connection:="URL;http://www.xe.com/ict/?basecur=USD&historical=true&month=" & strMonth & "&day=" & strDay & "&year=" & strYear & "&sort_by=name", _
Destination:=Range("A3"))

With oWeb
     .WebSelectionType = xlSpecifiedTables
     .WebTables = sWebTable
     .Refresh BackgroundQuery:=False
     .SaveData = True
End With

Set oWeb = Nothing

ActiveSheet.Columns("A:D").Insert Shift:=xlToLeft
Range("A1").Value = Cells(1, 5) - 1


End Sub

kathyklk say:

last page of document with sections

I'm inserting Section Breaks into a document of "stacked" files so that each new file can start with Page X (Page field) of Y (Section Page field).

When I do this, it always generates a last page which is blank except for a footer (not the footer from the previous file, but the footer from the form file I created the document from). I want neither the extra page nor the footer. If I delete the final Section Break (Next Page), I lose the footer that I do want.

Is there any way around this? I'd like the document to simply end with the text for that page and the correct footer showing Page X of X, and have that be that. 

Thanks!

stefan blom say:

Sections are tricky in Word, because a section break stores the formatting of the preceding section, and the last paragraph mark (¶) of the document contains an invisible section break. In other words, if you delete a section break, you delete the section formatting (headers, footers, margins, etc.) stored in that break.

To work around the issue, do the following: Go to the very last section of the document. Double-click the header to activate the header/footer view. On the Header & Footer Tools Design tab, click to activate "Link to Previous." Do the same for the footer. Double-click in the main body of the document. Place the insertion point in the section immediately before the section break. Display the Page Setup dialog box (for example by double-clicking the vertical ruler). Click OK to confirm the settings (and make Word "remember" them). Again, go the last section. Press F4 to apply the section formatting. Delete the section break.

For more on the topic, see http://word.mvps.org/faqs/formatting/workwithsections.htm.

charles kenyon say:
When you have multiple sections the headers and footers may be continued from section to section. If this is the case, the settings are in the last section. You can break these connections so that each section is independent.

Do you want your page x of y to restart (and end) with each section? If so, you need to use the SectionPages field instead of the NumPages field.
