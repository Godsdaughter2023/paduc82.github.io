frederik long say:
There are several ways to set up file sharing and I do not know which method you use. On my machines I do this:
  • Create identical accounts & passwords on each machine.
  • On each machine, create one or several folders that you wish to share.
  • Create shares for these folders.
  • Set the share permissions to "Full control for everyone".
  • Set the NTFS permissions to "Full control" for the accounts to which you wish to grant access.
When I do this then I have no problem accessing the shares across the network.
