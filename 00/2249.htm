The control you are refering to is not a list box, it's a combo box.  Karl's solution works without a parent recordset in the main form.  Basically the LinkChildFields and LinkParentFields properties work as a criteria method of the subform.  WHat he left out is that you will need to requery the subform after making a selection in your combo box.  That is done in the After Update event.

Privae Sub Combobox_AfterUpdate()

   Me.[Subform].Requery

End Sub

Alternatively, If you choose to provide additional criteria in which to query your subform you can create a filter to apply to the sub form.  In this case you would not use the LinkChildFields and LinkParentFields properties.  Instead you would build your criteria statement based however many parameters you choose, the set the subform's Filter property = to the criteria string and set the FilterOn property to true.

Private Sub SomeEvent()

    Dim strCriteria

    'Build Criteria
    strCriteria = "CustomerID = " & Me.Cbx_CustomerID
    strCriteria = strCriteria & "CustomerAcct = " & Me.Cbx_CustomerAcct
    ...

   Me.[Subform].Form.Filter=strCriteria
   Me.[Subform].Form.FilterOn=True

You would probably want to create a Display button and use the OnClick event to launch this.
