kga17 say:

Analysis Toolpak Single Factor ANOVA vs. Regression ANOVA

Thank you,

mike middleton say:
... I know how to calculate ANOVAs and have done so in a number of ways. The problem that I'm having is that for the ANOVA that is in the regression tool,  not finding documentation specific to that tool that shows why it provides a different result from the Microsoft Single Factor ANOVA tool.  It looks to me that for the data I provided earlier the Regression tool's ANOVA and the Single Factor ANOVA should be returning the same value.  I've been able to replicate the Single Factor tool's result  using an alternate method that is widely available, even though I haven't found any documentation from Microsoft that describes how they do it in the Single Factor tool.  So for the time being, I can't recommend to my students that they use the ANOVA that is in the regression tool.  Instead, they should use the Single Factor tool. 
Single-factor ANOVA and Regression are usually used for very different purposes, and the appropriate data sets are also very different.

Single-factor ANOVA is used to test the null hypothesis that the means of two or more populations are equal, and it requires that the random samples from each population are independent. And each sample (column) may have a different number of values.

Regression is used to determine a relationship between the Y variable of interest and one or more X explanatory variables. Each observation (row) has a value for Y and the corresponding values of X, so each column must have the same number of values.

(With single-factor ANOVA, I think of each value of the data set as a value of the Y variable of interest, and the corresponding value of the X explanatory variable, usually not explicitly shown, is the population from which the random sample was selected.)

The ANOVA section of the Regression tool is testing the null hypothesis that there is no relationship between Y and any of the X variables, sometimes called a test of overall fit. That is, the null hypothesis is that the population slope or all partial slopes (net regression coefficients) are equal to zero. This Excel feature is a standard part of any multivariate regression software package.

There is generally no reason to expect that single-factor ANOVA will yield the same results as the ANOVA summary of the Regression tool. The appropriate tool depends on the meaning of the data and the statistical questions that you are pursuing.

I doubt that you will find documentation of how Excel does its computations for single-factor ANOVA. As I recall, the Analysis ToolPak was developed by a third party for Excel 4. In general, the computations for single-factor ANOVA are very standard and well-known, and Excel likely uses standard sum-of-squared-deviations for intermediate calculations.

-  Mike Middleton, www.MikeMiddleton.com
