jllatham say:

Ok all we need to deal with now is that first formula:

=CONCATENATE(RC[-2]," ",RC[-1])

We can write it very simply as:

Function dsCC2(Cell1 As Range,Cell2 As Range) As String
  dsCC2 = Cell1.Value & " " & Cell2.Value
End Function

to call it, you actually put the cell references for the 2 strings to be joined together.  Example using C4 as the cell to put the formula into:

=PERSONAL.XLSB!dsCC2(A4,B4)

I named this one dsCC2 rather than dsC2 because a 3-letter name can be confused in Excel 2007/2010 as a cell address, so safer to use 4-character name.  Think of it as:

DavidSextonConCatenate2Cells

vasil l. michev say:

Disable dirsync, remove the on-prem accounts, clear the ImmutableID attribute of the cloud users and re-enable dirsync.

A faster, but a bit scarier option si to delete the Office 365 account, then recover it from the recycle bin. This will make it a "disconnector" and you can clear the ImmutableID attribute, breaking the link with the on-prem account. Then you can safely delete the AD ones.

steve rindsberg say:

The function below is one way of doing it.  

Sub TestHREF()
    Dim sLinkString As String
    sLinkString = "<a class=""rdwebsite mb0 fRight"" href=""http://www.microsoft.com/"" itemprop=""url"">Microsoft</a>"
    MsgBox HREF(sLinkString)
End Sub

Function HREF(sLinkString As String) As String

    Dim x As Long
    Dim y As Long
    Dim sTemp As String

    ' where in the string is href="
    x = InStr(sLinkString, "href=""")

    ' start at the character past href=" and append
    ' to a temp string until we find the next "
    ' Return the temp string and exit
    For y = x + Len("href=""") To Len(sLinkString)
        If Mid$(sLinkString, y, 1) <> Chr$(34) Then
            sTemp = sTemp & Mid$(sLinkString, y, 1)
        Else    ' we found the trailing quote
            HREF = sTemp
            Exit Function
        End If
    Next ' y

End Function
