colm o'brien say:

Trouble with inserting a file in a document using user form

I have a user form which has a checkbox on it.

if the checkbox is true then i want to insert a file at a specific bookmark in the document. if false i want another document in the bookmark.

the files are on a mapped network drive w:\filename.doc

the code is below

Private Sub OKbut_Click()
Dim DocName As String

DocName = "W:\elcert.doc"
ElseIf Var1 = "No" Then
DocName = "W:\doc4.doc"
End If
Set Target = ActiveDocument.Bookmarks("Target").Range
Target.InsertFile DocName

Unload Me
End Sub

when i run this i get the error message

Run-time error 5273

the document name or path is not valid

I have looked in the w:\ drive and can see both docs there have i missed something.

 

jay freedman say:

The value of a checkbox is not "Yes" or "No". It's a boolean value (True or False).

Since your code checks explicitly for both strings "Yes" and "No", and both If conditions are always false, the value of DocName is always an empty string (the value of a string that has no other value assigned to it). That's why the error message says it's an invalid file name.

For a boolean value, you don't have to check both possibilities explicitly. Just say

  If Var1.Value = True Then
    DocName = "W:\elcert.doc"
  Else
    DocName = "W:\doc4.doc"
  End If

The other thing to note is that you should use the full property name, Var1.Value; don't just rely on Value being the default property of the checkbox.
