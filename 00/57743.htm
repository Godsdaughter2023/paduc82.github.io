Hi there!

When it comes to installing disc games to an Xbox One system, you can install it in offline mode and play the game (if its not multiplayer) in that mode as well; however, the internet connection would be needed for signing in your profile and any updates that are available for the game as well.

This being said, you should still be able to play and if you have a smartphone with wifi capabilities, you may be able to connect to its network wirelessly as a temporary fix, if only so that you're able to sign your profile into the console to ensure that any offline play that you do (not connected to internet) will be saved to the Cloud once you get your apartment set up with a network. Use your console offline

Follow these steps:
