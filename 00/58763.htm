a. user say:

Even though it is a home only computer, it requires an administrator account (for such things as installing programs and other functions limited to administrators - and there are quite a few even if you haven't encountered many yet because you're using an administrator account - and if you think you're asked too often for your administrator password now, just wait until you're using a standard user account).  This has nothing to do with the use you make of your computer.  You do NOT want to delete or change your only administrator account because believe me you will need it (and probably sooner than you think).  It's perfectly fine to use a standard user account for your day-to-day work (in fact, it is recommended for security purposes) but you still need the administrator account (in fact, I would create two just to be sure one doesn't become corrupt or otherwise unusable and then you'll be stuck).

Once you've created the new administrator accounts (do that first or you won't be able to create the administrator accounts with your standard user account), then go to your current administrator account and change it to a standard user account which is how you say you want it.

This is all done in Start / Control Panel / User Accounts.  Click on add an account to add the new administrator accounts (with their passwords).  Click on manage a current account to find your current "administrator" account and click on it and click on change type and then check standard user.  You may from time to time be requested for your password (especially at logon though we can change that if you want but I strongly advise against it for security reasons - but rarely any other time for a standard user account) and the only way I know to avoid this (or workaround it) is to remove the password (set it as nothing) and then you have nothing to enter (but this isn't very good for security purposes).  Whenever you do something that requires an administrator (like install a program or many other things - mostly configuration changes or system changes), you will be asked for the administrator password and there's no way around that (again unless you remove the password and that's an even worse idea for an administrator account).  This will occur much more often now that your main account is a standard user account.  You may need to reboot for these changes to take effect.

I hope this helps.

Good luck!
