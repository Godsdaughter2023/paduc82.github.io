dslepak say:

Is there a way move my aol mail folders to my outlook.com account?

Is there a way move my aol mail folders to my outlook.com account?

brian tillman [outlook mvp 2007-2019] say:

I can think of a way, but it's fairly laborious.

outlook.com can be configured to POP messages from other mailboxes.  AOL supports POP, according to http://help.aol.com/help/microsites/search.do?cmd=displayKC&docType=kc&externalId=217449.  So, create a new folder in your outlook.com mailbox and move the contents of your Inbox to that folder.  Now, click the Settings gear to the left of your name at the top right of the Inbox and choose "More mail settings".  Click "Sending/receiving email from other accounts" in the left-hand column.  Click "Add an email account".  Click the "Advanced options" and enter the AOL server names, ports, encryption, and your AOL credentials into the appropriate fields.  Select the option to leave copies of messages on the AOL server or deselect if you want them deleted.  Click Next.  Choose whether you want your AOL mail to be delivered to your existing Inbox or another folder whose name is based on the AOL user.  Click Save.  Return to your outlook.com Inbox.

Now log into your AOL mailbox.  You'll see there a message from outlook.com asking you to verify that you want to access this mailbox from outlook.com.  Click it and outlook.com will finish configuring yor account to get messages from AOL.  It will also configure outlook.com to allow you to send mail from your outlook.com mailbox using your AOL address.  At this point, outlook.com should pull messages from your AOL Inbox and put them in the folder you specified in the previous setup.  Once that has been done, you can them move your AOL folder messages fo the AOL Inbox and have outlook.com pull them over one folder at a time.  On the outlook.com side, you can move them to whatever folder you want; either an existing folder or a new one that you create.  Eventually, you'll have all of your mail moved from AOL to outlook.com

https://support.microsoft.com/en-us/kb/2752583

For changing PST file location, please refer to the below mentioned steps.

Outlook Data file (PST) are stored in Documents\Outlook Files folder. If you upgraded to Outlook on a computer that already had data files that were created in Microsoft Office Outlook 2007 or earlier, these files are saved in a different location in a hidden folder at drive:\Users\user\AppData\Local\Microsoft\Outlook.

 

  1. Exit Outlook.

  2. Open Windows Explorer(location as mentioned above) and move the Outlook Data File (.pst) to the new location.

  3. Restart Outlook.

  4. An error dialog box appears that notifies you that an Outlook Data File (.pst) could not be found.

  5. Click OK.

  6. In the Create/Open Outlook Data File dialog box, browse to the new folder location, select the Outlook Data File (.pst) that you moved, and then click Open.

don cuthbert say:
When an upgrade to a new build fails, these steps may enable it to succeed:
  • Completely uninstall any non-Microsoft anti-virus software.
  • Perform a clean boot before upgrading. https://support.microsoft.com/en-us/kb/929135 
  • Disconnect any unneeded peripherals, such as devices connected via USB and SD cards.
  • Disable all network adapters after the build has downloaded, but before it Restarts, or before running setup.exe, if you are upgrading via a mounted ISO or Windows 10 install media.
Sometimes disconnecting the Internet, such as unplugging the Ethernet cable, will enable a stalled upgrade to complete.

There should also be a log that recorded what happened. Check C:\Windows\Panther\setupact.log and setuperr.log. There may be some details there.

The Reliability history may also indicate what error was encountered. Type reliability in Cortana search and choose View Reliability history.
