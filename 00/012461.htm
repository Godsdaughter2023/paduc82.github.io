suzanne s. barnhill say:

Red is the default color for the Font Color button. It does not indicate the Automatic color of text and does not affect existing text or styles. If you change the color and apply it to text, the  color displayed by the Font Color button will reflect the color you change it to (I keep mine set to white). Even if you Undo the change in font color, the button will still reflect the color change.

The primary difference is that, while Word 2003 will retain the selected color between Word sessions, Word 2010 apparently resets the button every time you start the program; it does seem to persist during an editing session, but only for new documents; existing documents open with it back to red. That is just the way it works, and presumably we will have to get used to it.
