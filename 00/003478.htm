When you first create the table and caption, both are inline with text. Then you drag the table, that turns it into a floating object. But you haven't changed your caption into a floating object, so it stays with the text where it was created. This is exactly what I was referring to in point #2 of my previous reply.

Instead, please select both the table and the caption, Cut them, then Paste them into your preferred location. This keeps both items inline with text and together.

A more complicated solution would be to cut the caption, paste it into a text box placed below the table, move the text box anchor to be attached to the table and lock it, then set the text box position to be relative to the table. Then you can drag the table and the caption will follow it. This method could interfere with the caption appearing in a Table of Tables, if you use that. I can give you complete steps if you don't understand this description.
