andreas killer say:

You have to create a formula and a hyperlink to the same cell in one cell, that is not possible woth copy&paste.

Andreas.

Option Explicit

Sub Test()
  'Create a link in Sheet2!F4 to Sheet1!B3
  CreateFormulaWithLink Sheets("Sheet1").Range("B3"), Sheets("Sheet2").Range("F4")
End Sub

Sub CreateFormulaWithLink(ByVal Source As Range, ByVal Dest As Range, _
    Optional ByVal ScreenTip As String)
  If Dest.Hyperlinks.Count > 0 Then Dest.Hyperlinks.Delete
  Dest.Formula = "=" & Source.Address(External:=True)
  If Len(ScreenTip) > 0 Then
    Dest.Parent.Hyperlinks.Add Dest, "", Source.Address(External:=True), ScreenTip
  Else
    Dest.Parent.Hyperlinks.Add Dest, "", Source.Address(External:=True)
  End If
End Sub

According to my test in Skype for Business 16.0.9126.2072, if I am already in a conference call, then I start a new separate call, the previous call will automatically be in hold. And the 2 separate calls will not be merged automatically. You can check the picture below.
