Here's an approach that is slightly different but is extremely easy to maintain.

Suppose you have data in B3:C{n}.

Then, in A3 enter the formula =SIGN(B3-C3) and in D3 the formula =-A3

Copy A3 as far down col. A as you have data in B.  To do this quickly, move the cursor over the bottom-right corner of A3.  The cursor will become a bold + sign.  Double-click.

Copy D3 as far down col. D as you have data in C.

Select A3:A{n} and D3:D{n}

Create a c.f. with the desired icon set.

Change the conditions so that if >=1 then use green icon.  For all other conditions use no icon.

japanzaman say:

There's more than one way you could go about doing this, but tags would probably be the best way to go. I'm assuming there is a question on each slide, and you have these slides grouped consecutively based on their content. If not, you won't be able to use this method.

After a question is answered you should have a macro add a tag to that slide (you can name it whatever you want). This macro will also take you back to the first slide, so you can use it in the place of a normal hyperlink.

---------------------------

Sub addtag()

ActivePresentation.Slides(Application.SlideShowWindows(1).View.CurrentShowPosition).Tags.Add "Done", "Done"

Application.SlideShowWindows(1).View.GotoSlide 1

End Sub

---------------------------

Then, when you are about to access a slide, check to see if the tag exists. If it does, skip to the next question. If you have already tagged the slides with your own tags, you'll need to adjust the value from 0. Shouldn't be a problem unless some slides have more tags than others.

---------------------------

Sub getquestion()

 

For I = 2 To 6   'This should be the slide range of the group you want to test, like slides 2 through 6

    With ActivePresentation.Slides(I)

        If .Tags.Count = 0 Then

             Application.SlideShowWindows(1).View.GotoSlide I

             Exit Sub

        End If

    End With

Next

MsgBox ("All the questions for this group have been answered!")     'in case all the questions have been answered.

End Sub

 

---------------------------

If you want to do the test again, you'll need to either close down the presentation WITHOUT saving and restart, or run a macro that will delete the tags, which will essentially restart the quiz.

---------------------------

 

Sub reset()

On Error Resume Next   'in case there isn't a tag

For Each sld In ActivePresentation.Slides

    sld.Tags.Delete ("Done")    'or whatever you called your tag

Next sld

End Sub

 

 

I hope this gives you at least an idea of what needs to be done. Basically, you need someway to tell powerpoint what slides have been answered, and tags are the easiest way to do this. You could do this without tags, but it would be rather cumbersome. I can't see your presentation so I can't be sure what I have suggested is what you need, but this will at least allow you to mark slides that have been answered and skip over them. You can cut and paste these macros into the VBA editor, and then have buttons execute them on the slideshow screen.
