jllatham say:

You could do this with two macros - one would work automatically as you change information on the sheet you want 'color coded', the other you would have to manually cause to run when you wanted to reset the colors so that you can start color coding them again.

 

First, the automatic code: this will change the cell fill color for all cells on the sheet when you change the value in the cells, such as by [Del] or typing in or pasting in new information on it.  This is a Worksheet event associated process, so to put it into the workbook properly:

select the sheet that you want this to work with

right-click on the sheet's name tab and choose [View Code] from the pop-up list

Copy the code below and paste it into the module presented to you

press [Alt]+[Q] to close the code editor

Important: save the file as a 'Macro Enabled.." workbook, type .xlsm or .xlsb (or 97-2003 type .xls)

 

Test it by simply changing a value on the sheet.  The cell should change color.  Here is the code:

 

Private Sub Worksheet_Change(ByVal Target As Range)
'built-in color settings available:
'vbBlack, vbBlue, vbCyan, vbGreen, vbMagenta, vbRed, vbWhite, vbYellow
  Target.Interior.Color = vbYellow
End Sub

 

Now, here is some code that performs the action described in its comments - if you prefer not to have the code, then you could perform those steps manually each day before beginning to make the daily entries.  Doing them all by hand takes about as many mouse-clicks as it does to select the macro ([View] tab, Macros, choose the macro from the list, click [Run]) - so it is your choice.

 

This code goes into a regular code module, and instructions for placing it into one of those can be found here:  http://www.contextures.com/xlvba01.html#videoreg

 

Here is the simple routine:

Sub ResetCellColors()
  'removes all shading from cells
  'this is the same as selecting the sheet
  'clicking the 'select all' button in the
  'upper left corner of the sheet:
  ' left of the "A" column indicator and
  ' above the "1" row indicator
  'and then setting cell Fill to 'No fill'
  'from the [Home] tab.
  ActiveSheet.Cells.Interior.Color = xlNone
End Sub
