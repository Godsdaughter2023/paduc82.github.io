ben m. schorr -bschorr say:

Create a new PST file (File | New | Outlook Data File) and copy the folders/messages you want to take with you to that file.

Copy that file to a flash drive and take it to your spouse's computer. Copy it to the spouse's computer and File | Open | Outlook Data File on that machine.  DON'T copy or move any items from those folders - leave them in your transport PST file.  Until you get a new computer, then move the PST file to that new computer and you can copy/move the items to it.

EXCEL 2007

(I think the process is the same in the 2010 version).

 

On the screen on front of me I have a Chart with 2 series one of which I want to move to a secondary axis. These are the actions I take to achieve that:-

1. Click the Chart in the top right hand corner.

This should cause the Chart Tools to become enlivened in the Title Bar.

2. Chart Tools

Format contextual tab

Current Selection group (on the far left hand side).

In this Current Selection group there is a drop down arrow - click this then click the series to which you wish to apply a secondary axis.

The series name that you have just chosen should now appear in the drop down window.

3. Still in the Current Selection group click:-

Format Selection

The Format Data Series window should now open and it should default to:-

Series Options

 - in its uppper left hand corner.

4. Click:-

Secondary Axis

 - towards the lower right hand corner then click:-

Close

You should now have achieved the desired effect.
