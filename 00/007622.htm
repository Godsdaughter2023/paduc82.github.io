If you just need to return the formatting to that of the styles, use Ctrl+A and then Ctrl+Space.  That will remove all of the direct formatting.  To identify where the direct formatting was applied, you could then save that document with a new name and use the Compare utility to compare it to the original with the direct formatting.  You could then probably use a macro to extract the changes from that resulting comparison document.
