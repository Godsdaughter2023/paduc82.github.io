Code 52: Windows cannot verify the digital signature for the drivers required for this device.

 

Method 1:

Run the hardware and devices troubleshooter and check if it helps:

http://windows.microsoft.com/en-us/windows7/Open-the-Hardware-and-Devices-troubleshooter

 

Method 2:

I would suggest you to refresh device manager and check if it helps:

a.     Click on Start.

b.    Click on Run and type “devmgmt.msc” (without quotes), and then Enter. The Device Manager window opens.

c.     Click to select your computer as the location for the scan.

d.    On the Action menu, click Scan for hardware changes.

e.    Close the Device Manager window and restart the computer.

 

Method 3:

I would suggest you to uninstall drivers from the device manager and try reinstall them by downloading from manufacturer’s website in clean boot mode:                                                           

a. Click on Start.

b. Type “Device Manager” without quotes in the search bar.

c. Click on Device Manager.

d. Select the device you wish to uninstall in the Device Manager window and uninstall it.

e. You can also uninstall by right clicking on the device and selecting Uninstall from the context menu. A dialog box will open asking if you really wish to uninstall the driver. Click Yes, then OK in the next window.

 

Note: Make sure you put the computer back to normal startup after the troubleshooting.
