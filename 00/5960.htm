Moving Favorites from a computer with XP to a new computer with - Microsoft Community

keith mcg say:

Moving Favorites from a computer with XP to a new computer with Windows 8

I just bought a new computer with Windows 8. I want to copy my IE Favorites from my old computer which uses XP. I searched the Internet for suggestions. I now have my Favorites on a flash drive in a file called "Bookmark". I cannot find an Import/Export on the Windows 8 IE. The favorites toolbar only allows me to add the current page. Help please.
santanu suna nayak say:
Hi


Insert a USB flash drive into the computer with Windows XP on it.

  1. Open Internet Explorer.
  2. Click the "File" menu at the top-left of the Internet Explorer window, and then click on "Import and Export..." to open the "Import/Export Settings" window.
  3. Select "Export to a file" and click "Next."
  4. Click the box next to "Favorites" to put a check in it. Remove any checks in other boxes you see, and then click "Next."
  5. Select the "Favorites" folder you want to export. If you want to export all your bookmarks, select "Favorites"; otherwise click on the subfolder you want and then click "Next."
  6. Click "Browse" and navigate to the flash drive you inserted. You can find it by clicking the drop-down menu by the words "Save in" and then selecting "My Computer," which displays all your drives, including the USB flash drive.
  7. Name the file anything you want in the "File name" text field; the name is unimportant as long as it's something you'll remember.
  8. Click "Save" and remove the USB flash drive.

Import to Windows 8

  1. Insert the USB flash drive in a free USB port on the computer with Windows 8.
  2. Open Internet Explorer.
  3. Click the "Favorites" button in the top-right corner of the window; the button has the picture of a star on it. Clicking this opens the "Favorites" tab.
  4. Click the drop-down arrow next to "Add to favorites" and select "Import and Export..." to open the "Import/Export Settings" window.
  5. Select "Import from a file" and click "Next."
  6. Check the box beside "Favorites" and remove any checks by other options. Click "Next."
  7. Click "Browse" and navigate to the flash drive you inserted, and then double-click the file you exported from the XP computer. Click "Next."
  8.  Select the location you want to import your "Favorites" to. If you want them immediately under your "Favorites" menu, just select the "Favorites" folder. Otherwise select the subfolder you prefer and click "Import."

VLookup - Sum mulitple return values - Microsoft Community

marionmcguire say:

VLookup - Sum mulitple return values

Hello

Can I use VLookup function to sum the total of multiple return values that match the lookup value?

I hope this makes sense?

Thank you.

myall_blues say:

Not really - VLOOKUP is primarily for looking up a single, unique value and returning a corresponding value from another column.

The SUMIF or SUMIFS formulae are better suited to this purpose. For SUMIF:

=SUMIF(range,criteria,sum_range) where:

range = the column containing the values you want to lookup

criteria = the lookup value or a cell containing the lookup value

sum_range = the column containing the values you want to sum
