There are a few factors that may be at play.

  1. First, normally the name that someone else will see is controlled by whatever name is listed at http://account.live.com for whatever Microsoft Account is used to login to your computer.  Unless you have multiple accounts setup in your Mail program.

  2. An exception (usually) is that if a recipient has that email address in their address book, their screen will override the name and use whatever their "pet name" is for that person.

  3. When you start to write a new email message in the Mail app, you can choose who you are sending "from" by going to the little "more" link near the subject line, then clicking the drop-down at the top.

  4. Finally, the email account that is selected at the bottom-left of the screen, is the account that the outgoing message will be sent "through", which relates back to #1 above.

Working with these in mind, you can get a handle on what the recipient will see.  Or even try using another email program.  You mentioned you were new to Windows 8, so let me suggest that you consider using whatever email program you used before you got Windows 8:  it will still work for you just fine, and since you're already familiar with it, there's no reason to exclude it as a choice.

Disconnect all the external devices (printers, scanner, USB (universal serial bus) drives, etc...) Except mouse and the keyboard and then boot up.

If this corrects the issue then add the devices back one at a time until you discover the piece of hardware causing the issue. Then get updated drivers software for that particular device.

System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files
