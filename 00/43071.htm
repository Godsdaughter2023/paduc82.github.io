Network Error:  Windows has detected an IP address conflict

More details are available in the windows system event log.


Why do I receive this message from Windows®?
You experience a message as this one when a couple of PCs are connected
to a network. Each PC has an IP-address, however, when two PCs have each one
IP-address that is identical to the other, and then there may be a problem. Windows®
then notifies one of the clients about this, and requires that the administrator make some
changes on the network settings. This can be an issue with the DHCP on the router.

Resolution:
Check Dynamic Host Configuration Host Protocol (DHCP) settings on the router.
Check each IPv4 address on each PC connected to your network.

Another thing to try:
1. Click Start Menu and in the Instant Search type “Command Prompt” then
right-click and left-click Run as administrator.

2. Type the first command ipconfig /release  and press Enter.
3. Type then the second command ipconfig /renew and press Enter. 


Explanation: 
The fist command will send a DHCPRELEASE message to the DHCP server to release the current
DHCP configuration and discard the IP address configuration for either all adapters, the second
command Renews DHCP configuration for all adapters.

For more details see:
http://technet.microsoft.com/en-us/library/bb490921.aspx

Err.. the Motorola is the modem and the Linksys is the router. Do you only have one Xbox, or more than one?

 

A sequential power cycle is the first step in clearing connection problems, and the order and timing of the steps is important to success:

 

1) turn off the Xbox and unplug the power supply cord from the back of the Xbox then

2) unplug power from the Linksys.

3) unplug power from the Motorola.

4) Wait at least 5 minutes then replug the Motorola power.

5) Wait 'til the light on the front of the Motorola which says ONLINE, READY, CABLE or BROADBAND is on solid (not flashing), then plug in the Linksys.

6) Wait until the WiFi light on the Linksys is on; you may not use WiFi, but it's still a good indicator your router is finished booting up.

7) Plug in and turn on the Xbox.  For more background on Ports, see whatismyipaddress.com/port
