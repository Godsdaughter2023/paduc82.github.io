Include the following line as the first line of the code. (Replace "OssieMac" with your login user name).

If Environ("UserName") = "OssieMac" Then Exit Sub

The above line is case sensitive. You can use the following so that case is ignored but then use all upper case for the username in the double quotes.

If UCase(Environ("UserName")) = "OSSIEMAC" Then Exit Sub

? Environ("UserName")
