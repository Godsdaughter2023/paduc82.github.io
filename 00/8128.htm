jdmorgan say: You do NOT need to have 64-bit Office to import/export Excel and Access datafiles from 64-bit SAS.  You DO. however, need to install the 32-bit PC File Server (NOT 64-bit PC File Server) in order to import/export 32-bit Excel and Access files from 64-bit SAS.  (That is, the version of the PC File server must match the Office version, NOT the SAS version.)  You can download it for free from the sas.com website.  After the PC File Server is installed, you need to use the options for the PC File Server in the Import/Export wizards.  If you import/export from your SAS program, the simplest way to get a sample of the correct code is to run the Import/Export Wizard and then save the code in the last step.
