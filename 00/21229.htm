progressive numbers in one cell in Excel - Microsoft Community

lyndoudiet say:

progressive numbers in one cell in Excel

I am trying to set up a spreadsheet that will increase the number in only one cell after each print, can anyone help me out with a code to write?

 

I am trying to set up cell "E2" starting with 000001 and when I print off 50 pages the numbers in that cell will increase by 1 on each print  (ie: 000002, 000003, 0000004 - 000050)

chip pearson say:

You can't really do this well for a couple of reasons. Although Excel has a BeforePrint event procedure that runs automatically before printing something, it runs when you print anything. There is no way to determine what is being printed. Also, BeforePrint will run only once when multiple copies are printed. So, if you had code in  BeforePrint to increment a cell's value and you printed 50 copies of the worksheet in one print operation, the  value would increment only once, not fifty times. You would have to have fifty individual print jobs to get the number to properly increment.
