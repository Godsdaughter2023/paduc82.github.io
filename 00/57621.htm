undelete sticky note

How do I undelete a stick note in Windows 8.1?

ronnie vernon say:

How do I undelete a stick note in Windows 8.1?


Hi

 

There is no Undelete option, or Save option for Sticky Notes. They are designed to be temporary reminders.

 

If it is critical that you recover the text, there is a complex way to recover the text that was written on a sticky note.

 

Open File Explorer.

 

In the Address Bar Type or Paste the following location.

 

 

%AppData%\Microsoft\Sticky Notes\StickyNotes.snt

 

Press Enter.

 

You will see a prompt to choose a program to display the file.

 

Select Wordpad.

 

The file will display the Raw Code for each of the Sticky Notes that have been created, including the deleted notes.

 

Mixed in with all of that raw code will be the text for each of the Sticky Notes.

 

If you have critical information in a Sticky Note, it's best to copy that information and paste it in Notepad where you can Save the file.


You can also Hide all of the current Sticky Notes without deleting them.

 

Right Click the Taskbar Icon for the Sticky Notes and select the Close Window option. The next time you start Sticky Notes all of the current notes will be displayed again.

 


Regards
