npr6 say:

See http://npr.freei.me

Try:

Open "windows firewall with advanced security"

Click on "Outbound rules"

Right click "Outbound Rules" select new rule
Select Custom > next
Select "All  programs" > next
Select "protocol type" Any > next
Under "Which local IP ......" select Any IP address
Under "Which remote IP ...." select These IP addresses > Add > Predefined set of computers > select "Local subnet" > OK > next
Select "Allow the connection" > next
Leave all three ticked > next
Give the rule a name > finish

That should get you started. If you feel the need to tighten the rule, get info from the log file and cports.exe and experiment. May also try changing the profile to "private". As this only gives permission to the local subnet I would be inclined to leave it, provided the machine is unlikely to be connected to a public wireless network at some time.
