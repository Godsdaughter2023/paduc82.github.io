    ActiveCell.Address - Microsoft Community

    gary's student say:

    ActiveCell.Address

    Assuming we have a workbook with a number of worksheets.  Is it possible, without Activating/Selecting each worksheet in turn, to determine what the ActiveCell will be when the worksheet is selected??
    mike h.. say:

    Hi,

     

    If a sheets isn't active then no cell(s) have the focus, none are selected and there is no activecell so I don't think what you're trying to do is dooable.

     

    You could use something like this below on a hidden sheet to keep track of the last selected cell(s) on each sheet. I haven'e spent a lot of time on it in case it's of no use but to turn it into a workable solution on workbook open you would need to loop throught the sheets once to nsure you had a value for each sheet.

     

    Private Sub Workbook_SheetSelectionChange(ByVal Sh As Object, ByVal Target As Range)
    Dim Myrange As Range
        lastrow = Sheets("Cell Tracker").Cells(Cells.Rows.Count, "A").End(xlUp).Row
        Set Myrange = Sheets("Cell Tracker").Range("A1:A" & lastrow)
        If IsError(Application.Match(Sh.Name, Myrange, 0)) Then
            Sheets("Cell Tracker").Cells(lastrow + 1, 1) = Sh.Name
            Sheets("Cell Tracker").Cells(lastrow + 1, 2) = Target.Address
        Else
            Sheets("Cell Tracker").Cells(Application.Match(Sh.Name, Myrange, 0), 2) = Target.Address
        End If
    End Sub
