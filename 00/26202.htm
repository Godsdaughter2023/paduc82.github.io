david mcuk say:
Hi Diggsy

IF Statements only support 7 IF Statements Nested.

Not sure how your data is set out, so taking a few assumptions here.
Essentially, I am assuming that your Data Tables on each worksheet are the same structure and that each Data Table has a unique code, hence so many data tables.

In my example I have 8 worksheets with data is cells A1 to B4.I am assuming that you tables are all the same size as well

There is a new function is Excel 2016, called IFS, with this function that can have up to 127 elements, however it does not have an else section, ie there is no match.

https://support.office.com/en-us/article/ifs-fu...

You can combine this with a IFERROR function that would return the result it no match https://support.office.com/en-us/article/iferro...

Below is an example of my formula

=IFERROR(IFS(VLOOKUP(C16,Data01!A2:B4,2,FALSE),VLOOKUP(C16,Data01!A2:B4,2,FALSE),VLOOKUP(C16,Data02!A2:B4,2,FALSE),VLOOKUP(C16,Data02!A2:B4,2,FALSE),VLOOKUP(C16,Data03!A2:B4,2,FALSE),VLOOKUP(C16,Data03!A2:B4,2,FALSE),VLOOKUP(C16,Data04!A2:B4,2,FALSE),VLOOKUP(C16,Data04!A2:B4,2,FALSE),VLOOKUP(C16,Data05!A2:B4,2,FALSE),VLOOKUP(C16,Data05!A2:B4,2,FALSE),VLOOKUP(C16,Data06!A2:B4,2,FALSE),VLOOKUP(C16,Data06!A2:B4,2,FALSE),VLOOKUP(C16,Data07!A2:B4,2,FALSE),VLOOKUP(C16,Data07!A2:B4,2,FALSE),VLOOKUP(C16,Data08!A2:B4,2,FALSE),VLOOKUP(C16,Data08!A2:B4,2,FALSE)),"No Match")

Just remember that IFS consist of
IFS(Condition1,Result1, Condition2, Result2, Condition3, Result3, etc)

Changing the DPI settings in Windows

If font sizes is a general issue throughout Outlook, Windows and other applications, you might want to consider customizing the DPI settings in Windows instead.

This is especially recommended when you are using a smaller screen with a high resolution and also on a larger screen with a 4K, 5K or higher resolution.

Windows 10
  1. Right click on an empty spot on your Desktop and choose: Display Settings.
  2. Use the slider directly under “Change the size of text apps, and other items” to set your preferred text size level.

Windows 7 and Windows 8
  1. Right click on an empty spot on your Desktop and choose: Screen resolution.
  2. Click on the blue text link: Make text and other items larger or smaller.
  3. Here you can directly set your DPI settings to 125%.
    To set a custom DPI value:
    • Windows 7
      Use the “Set custom text size (DPI)” link in the Task Pane on the left.
    • Windows 8 and Windows 8.1
      Use the “Custom sizing options” text link below the presets.
    • Windows 8.1 with multiple monitors
      Select the option “Let me choose one scaling level for all my displays”.
  4. In the new dialog that pops up you can directly choose a predefined value to make the fonts bigger.You can also type any percentage you like.

Extract from list: average value & count Bottom 25%, Top - Microsoft Community

philip bendall say:

Extract from list: average value & count Bottom 25%, Top 25%, Mid 50%

Column A: List of sites numbered 1 to 200 site, site 2, site3, etc.

Column B: Cost per hour to service

Minimum = 11.97

Maximum=87.00

Median=25.5

I am looking for help on a formula to extract:

Average cost  and number of sites this relate to of the:

 and number of sites this relate to of the:

Bottom 25% - number of sites

Highest 25% - number of sites

Mid 50% - number of sites

ronrosenfeld say:

I'm not sure exactly what you want. 

 

If by bottom 25% you mean the 50 sites with the lowest costs, then:

=AVERAGE(SMALL(Cost,ROW(INDIRECT("1:"&COUNTA(Sites)/4))))

 

Middle 50%

=AVERAGE(SMALL(Cost,ROW(INDIRECT(COUNTA(Sites)/4 & ":" &COUNTA(Sites)/4*3))))

 

Upper 25%

=AVERAGE(LARGE(Cost,ROW(INDIRECT("1:"&COUNTA(Sites)/4))))

 

The above three formula must be **array-entered**

To **array-enter** a formula, after entering
the formula into the cell or formula bar, hold down
<ctrl><shift> while hitting <enter>. If you did this
correctly, Excel will place braces {...} around the formula.

 

 

If you want to relate it the costs, rather than the raw counst of sites, then you can use the QUARTILE function to get the break points for the different segments:

 

Lowest 25%:

     Minimum:  =QUARTILE(Cost,0)  or MIN(Cost)

     Maximum:  =QUARTILE(Cost,1)

 

Mid 50%

     Minimum: =QUARTILE(Cost,1)

     Maximum: =QUARTILE(Cost,3)

 

Upper 25%

      Minimum:  =QUARTILE(Cost,3)

      Maximum:  =QUARTILE(Cost,4)   or MAX(Cost)

 

So to get AVERAGES for the ranges, you can use AVERAGEIF  or AVERAGEIFS

 

Average of lowest 25%:     =AVERAGEIF(Cost,"<="& QUARTILE(Cost,1))

Average of middle 50%:    =AVERAGEIFS(Cost,Cost,">="&QUARTILE(Cost,1),Cost,"<="&QUARTILE(Cost,3))

Average of upper 25%:     =AVERAGEIF(Cost,">="&QUARTILE(Cost,3))
