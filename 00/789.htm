alan wu msft say:

Hi Shehan,

It is my happiness to reply to your post here. 

Firstly, I should say it is supported for multiple forests and single Azure AD Tenant. Secondly, I'd like to answer your questions one by one as the followings.

1. Do I need to consider any join rules while configuring the second forest in AAD Connect? (We don’t have ADFS running on either forest)

The answer should be not needed since you haven't deployed the AD FS. The only thing you need to do is to choose the correct installation wizard for your topology. 

2. Will adding the Forest ms-DS-ConsistencyGuid make any impact on the Forest A's ObjectGuid attribute or to the users in Forest A who are already synced in Azure AD?

The answer should be no. You can set up a 2-way trusts for the 2 forests as each of the forest is independent from the other. 

3. Forest B has got a non-routable domain (Eg: ABC.local) – In that case do I need to verify and add the custom domain prior adding the Forest B to the AAD connect?The answer should be you need to verify the forest B's domain in the Office 365 Tenant prior to configuring the AAD Connect. So it doesn't impact the current email service or any other service of forest B's domain as you do not set up the (EXO and other) services for that domain in Office 365. 

For more information, you can refer to https://docs.microsoft.com/en-us/azure/active-directory/hybrid/plan-connect-topologies (Multiple forests, single Azure AD tenant).

Alan

tasos_k say:

try this...

(but i'm not sure for japanese ideograms)

 

 

step1

Save As, your Workbook with extension .xlsm (macros enabled)

 

step2

1) press ALT+F11 to open Visual Basic

2) Insert > module and paste the code below on the right

 

 

Sub macro_01()
Const myDelim As String = ","
Dim WS As Worksheet

Set WS = ActiveSheet  '<<< convert data from active sheet
Dim r As Long, c As Long, i As Long, j As Long
r = WS.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
c = WS.Cells.Find(What:="*", SearchOrder:=xlByColumns, SearchDirection:=xlPrevious).Column
Dim myPath As String
myPath = ThisWorkbook.Path
Dim myFile As String
myFile = myPath & "\new File.csv"
Dim obj As Object
Set obj = CreateObject("ADODB.Stream")
obj.Type = 2
obj.Charset = "unicode"
'obj.Charset = "shift-jis"

''obj.Charset = "utf-8"
obj.Open
Dim v() As Variant
ReDim v(1 To c)
For i = 1 To r
For j = 1 To c
v(j) = WS.Cells(i, j)
Next
obj.WriteText Join(v, myDelim), 1
Next
obj.SaveToFile myFile, 2
obj.Close
Application.ScreenUpdating = False
Dim wb As Workbook
Set wb = Workbooks.Open(myFile)
Dim t As Long, x As Long
t = 1
Do
vv = Split(Cells(t, "A"), myDelim)
For x = 0 To UBound(vv)
Cells(t, x + 1) = vv(x)
Next
t = t + 1
Loop Until Cells(t, "A") = ""
Application.DisplayAlerts = False
wb.Save
wb.Close
Application.DisplayAlerts = True
Application.ScreenUpdating = True
MsgBox "file is ready"
End Sub

 

 

3) Press ALT+Q to Close Visual Basic

 

 

step3

To run the macro, press ALT+F8, 

select macro_01 from the list and click the run button.

 

 

expected results
