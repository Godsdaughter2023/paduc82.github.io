suzanne s. barnhill say:

What works for me (I have Word 2003, 2007, 2010, and 2013 installed but have 2003 set as the default) is to reregister the specific Word version I want to use. "Reregistering" allows the program to reclaim its "ownership" of the Word file types through settings in the Windows Registry. The general command for reregistering Word is:

winword.exe /r

but this will always call the most recent (or currently dominant) version. To reregister a specific version, you have to include the entire file path. For Word 2003 on my system, for example, the command line is:

C:\Program Files (x86)\Microsoft Office\OFFICE11\WINWORD.EXE /r

To reregister Word 2007, first determine the full path to the executable (you've obviously done this--when you browse to it, you can select and Ctrl+C copy the path from the Address Bar), then press Windows Key + R to open the Run box. Ctrl+V to copy the path and add /r, leaving a space between the path and the switch, and press Enter. You'll get a message box. In my case it says,  "Please wait while Windows configures Microsoft Office Professional 2003." When it finishes, you should be good to go.

vijay a. verma say:
You don't require a product key for MS Office if you have installed and activated it. You require it just once, for the first time installation and activation.

Once you install MS Office 365 / 2016 through Microsoft site and activate it, a digital signature corresponding to your computer gets stored in Microsoft servers.

Now whenever you login to your Microsoft account, it will check whether you are trying to install on the same computer or not. It won't ask for the product key again.

Hence, go ahead and reinstall. Follow the Help below to reinstall -

https://support.office.com/en-us/article/Downlo...

In case, you want to install on a different computer, you will have to first deactivate Office license which was installed on old computer otherwise it will be counted as a license. Hence, if you were allowed installation on one computer, you will not be able to activate on new computer. If your license was for say 5 computer as in case of Office 365 Home, now your validity will be for 4 computers.

Hence, deactivate first and then install and activate.

Deactivate your existing installation - https://support.office.com/en-us/article/Deacti...

doug robbins - mvp office apps & services (word) say:

I do not know what you mean by a solid bar.   Whatever that is, it is not a page break, though it might appear in the same position.

There are two types of page break, an automatic one and a manual one.  A manual one, that is inserted by using Ctrl+Enter will have the words Page Break in the middle of it.

An automatic one, which will only see in draft view, and even then, not always, appears as a row of dots right across the width of the screen.

Do you have the white space between pages being shown.  If not, the division between one page and another will appear as a grey line and maybe it is that to which you are referring.  If that is what you are seeing, to show the white space, move the mouse cursor until it is over one of those lines and the cursor changes to two vertically opposed arrows and a screen tip "Double-click to show white space" appears and then do that.
