john korchok say:

Publisher, unlike InDesign, is not a color-managed application. Publisher uses whatever profiles are installed in Windows, so changing the Windows profiles will change Publisher's rendering.

I've read about, but haven't tested, a kludge for embedding a profile whereby you create a customer printer that prints to a PostScript file. Then you attach a profile to the printer, print your Publisher file to and get a PostScript file that your print shop should be able to read and print. This is only worthwhile if your print shop has a set of Windows color profiles that they are willing to share with you and if they are willing to spend some time experimenting to get the print results you're after. Otherwise, just print a PDF and let them worry about the color balance.

When "Different first page" is selected, the section has two headers: a first page header and a main header. (Of course, the corresponding statements are true for footers.)

You can use the Link to Previous button on the Header & Footer Tools Design tab to link the first page header of section 2 to the first page header of section 1.

In other words, linking a header only applies to a header of the same type (in the preceding section). The same applies to footers.
