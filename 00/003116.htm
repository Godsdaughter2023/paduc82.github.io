I've tried "fooling" the document by making sections but have not been successful in getting the Watermark on all pages including the Cover.

Can anybody help?

jay freedman say:
The official Watermark button is programmed to remove any existing watermark -- stupidly, from everywhere in the document -- before inserting the new watermark. The way to defeat this is to recognize that a "watermark" is really just a graphic (WordArt or a picture) that's inserted with its anchor in a header.

That means you can copy the existing watermark by opening the header pane of a page where it's displayed, clicking on it, and copying (Ctrl+C, or the Copy button on the ribbon, or the Copy command on the right-click menu). Then move the cursor to the header pane of the other page where you want it, and paste.

When you insert a cover page, you automatically get Different First Page layout (when the cursor is in the cover page's header, you'll see that checked on the Header & Footer Tools ribbon). That creates a separate header just for the first (cover) page. So if you have a watermark in the main document, you need to copy it from there to the first page header; or if the watermark is on the cover page, you need to copy it to the main header.

Private Sub Worksheet_Change(ByVal Target As Range)
Dim dt As Date
For Each rr In Range("B5:B15")
Range(Cells(rr.Row, "A"), Cells(rr.Row, "E")).Interior.ColorIndex = 0
dt = Format(rr, "hh:mm:ss")
If (dt <= TimeValue("06:00:00") And dt >= TimeValue("01:00:00")) Then _
Range(Cells(rr.Row, "A"), Cells(rr.Row, "E")).Interior.ColorIndex = 3

Starting with A1 as the active cell, tap F5 and then type A5:E15 as the Reference: and click OK. With A5:E15 selected and with A5 as the active cell, create three new Conditional Formatting Rules.
  1. Click the Home tab's Conditional Formatting, New Rule from the Styles group.
  2. Select Use a formula to determine which cells to format and supply the following as the Format values where this formula is true:, 
    =AND(ISNUMBER($B5),OR(AND($B5>=TIME(0,0,0),$B5<=TIME(6,0,0)),AND($B5>=TIME(18,0,0),$B5<=TIME(23,0,0))))
  3. Click Format and select a red Fill then click OK to accept the format.
  4. Click OK to create your new rule.
Repeat this procedure to create two additional rules with,
=AND($B5>=TIME(7,0,0),$B5<=TIME(11,0,0))
... and a blue Fill then,
=AND($B5>=TIME(12,0,0),$B5<=TIME(17,0,0))
... and a green Fill.

These formulas represent your criteria verbatim and leave certain periods between the shifts without highlight.

Download the 5380_Driver_for_OS Deployment Sept 2014 Files.

Run setup.exe as an administrator

Download and extract the 5380_Network_Driver Files

right-click and install the qcbtuart.inf from the drivers/production/windows8.1-x86/Bluetooth-Driver directory.

Restart the machine.
