e.j. gun say:

The code below is an example of what you could do to synchronize views on selected worksheet tabs within a workbook.  It intercepts the events created when you click on an inactive tab, and makes the view of the inactive tab look like the view of the active tab before the new tab is brought to the front.

Try it on a new workbook.  Scroll Sheet1 to some location, and then click on Sheet2.  The view on Sheet2 will go to the same location.

Is this what you're trying to accomplish?

Eric

Option Explicit

Public lastRange As Range

    If (lastSheet = "") Then lastSheet = ActiveSheet.Name
End Sub

    Dim i As Long
'
Application.EnableEvents = False
    Application.Goto Reference:=ActiveSheet.Range(lastRange.Address), Scroll:=True
Application.EnableEvents = True
End Sub

    Dim i As Long
    Dim newsh As String
'
    Select Case Sh.Name
    Case "Sheet1", "Sheet2", "Sheet3"
Application.EnableEvents = False
        lastSheet = Sh.Name
        newsh = ActiveSheet.Name
        ActiveWorkbook.Worksheets(lastSheet).Select
        lastSheet = Sh.Name
        Set lastRange = Sh.Cells(ActiveWindow.ScrollRow, ActiveWindow.ScrollColumn)
        ActiveWorkbook.Worksheets(newsh).Select
        Application.Goto Reference:=ActiveSheet.Range(lastRange.Address), Scroll:=True
Application.EnableEvents = True
    End Select
End Sub
