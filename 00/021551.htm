rick rothstein, mvp - excel say:

Well, this does not do everything you want, but it does flush out the invalid characters from the inputted string and returns the empty string if the resultant string is empty or equal to HISTORY (case insensitive)...

Function MakeValidSheetName(ByVal S As String) As String
  Dim X As Long
  Const InvalidCharacters As String = "[*?/\."
  For X = 1 To Len(InvalidCharacters)
    If InStr(S, Mid(InvalidCharacters, X, 1)) Then S = Replace(S, Mid(InvalidCharacters, X, 1), "")
  Next
  MakeValidSheetName = Left(Replace(S, Chr$(1), ""), 31)
  If UCase(MakeValidSheetName) = "HISTORY" Then MakeValidSheetName = ""
End Function

The problem I saw taking it further is the need handle incrementing 31 character strings and the problems with modified strings having to be tested against existing sheet names to make sure each time you modify the text, you don't accidently duplicate a different sheetname in the process. Anyway, it should serve as a starting point for you.

Function ValidSheetName(ByVal SheetName As String) As String
  Const InvalidChars = ":\/?*[]"
  Dim I As Integer
  For I = 1 To Len(InvalidChars)
    SheetName = Replace(SheetName, Mid(InvalidChars, I, 1), "")
  Next
  ValidSheetName = Mid(SheetName, 1, 31)
End Function

Function NewSheetName(ByVal SheetName As String) As String
  Dim I As Integer, NewName As String, SheetExt As String
  SheetName = ValidSheetName(SheetName)
  NewName = SheetName
  If SheetExists(SheetName) Then
    I = 0
    Do
      I = I + 1
      SheetExt = " (" & I & ")"
      If Len(SheetName) + Len(SheetExt) > 31 Then _
        SheetName = Mid(SheetName, 1, 31 - Len(SheetExt))
      NewName = SheetName & SheetExt
    Loop Until Not SheetExists(NewName)
  End If
  NewSheetName = NewName
End Function

Function SheetExists(ByVal SheetName As String, _
    Optional WB As Workbook = Nothing) As Boolean
  Dim s As Worksheet
  If WB Is Nothing Then Set WB = ActiveWorkbook
  On Error GoTo ExitPoint
  Set s = WB.Sheets(SheetName)
  SheetExists = True
ExitPoint:
End Function

  Dim I As Long, LeftParen As Long, NewName As String, SheetExt As String
  SheetName = ValidSheetName(SheetName)

Set recent files to zero in Powerpoint Mac - Microsoft Community

zeke wn say:

Set recent files to zero in Powerpoint Mac

How do I set the number of recent documents to zero in Powerpoint for Mac v. 15.3
john korchok say:

I hope you're not really on version 15.3. Maybe 15.30? Otherwise, head for Help>Check for Updates and get up to the current version before proceeding.

PowerPoint 2016 gets it's recent file list from 2 sources. If you're signed into your Microsoft Account or OneDrive, it will give you a list of online files that you've edited. To remove them from the list, you must sign out.

The other source are files that you've edited locally on your computer. You can remove those files by following these steps:

  1. Close PowerPoint.
  2. In the Finder, hold down the Alt key while clicking on the Go menu and choosing Library. Your user Library folder opens.
  3. Open Library>Containers>com.microsoft.PowerPoint>Data>Library>Preferences.
  4. Drag com.microsoft.PowerPoint.securebookmarks.plist to your desktop.
  5. Reopen PowerPoint. Viola, no more file list!
