To calculate SUM/ Max / Min / Avg, use these basic formulas. Even if C2 = A2/B2, it does not matter.


=SUM(C2:C100)

=MAX(C2:C100)

=MIN(C2:C100)

=AVERAGE(C2:C100)

If there are any errors in column C, DIV error or Value error, or any ther, then use the following - note all these except SUMIF are Array formulas (Ctrl+Shift+Enter):

=SUMIF(C2:C100,"<"&1E+100,C2:C100)

Array formulas (Ctrl+Shift+Enter):

=MAX(IFERROR(C2:C100,""))

=MIN(IF(ISNUMBER(C2:C100),C2:C100))

=AVERAGE(IF(ISNUMBER(C2:C100),C2:C100))

graham mayor say:

There are two options, neither of which entirely fits your requirements.


1. Set the Word Startup folder for all the users to a common location, set that folder as a trusted folder and put your global templates there.

2. Leave the Word Startup folder where it is and provide each user with a macro for their normal template to check (and if necessary add) the added, but unchecked templates from their central location when Word is started e.g. as follows


Sub AutoExec()
Dim oTemplate As AddIn
Dim bTemplate As Boolean
Const strAddInPath As String = "C:\Path\" 'The common add-in template location

    'repeat for each add-in
    bTemplate = False
    For Each oTemplate In Application.AddIns
        If oTemplate.name = "Add-in Name.dotm" Then
            AddIns(strAddInPath & oTemplate.name).Installed = True
            bTemplate = True
            Exit For
        End If
    Next oTemplate
    If Not bTemplate Then
        AddIns.Add Filename:=strAddInPath & "Add-in Name.dotm", Install:=True
    End If

macropod say:

Basically, you have two options -

1. Insert 12 sets of IF fields, each coded along the lines of:

{IF{DATE \@ M} = 1 "January"}, {IF{DATE \@ M} = 2 "February"}, etc., or nested, as in {IF{DATE \@ M} = 1 "January" {IF{DATE \@ M} = 2 "February" {IF{DATE \@ M} = 3 "March" }}}, etc

2. Insert a single compound field coded along the lines of:

{QUOTE"{SET MM {Date \@ "M"}}{IF{=INT(MM/4)}= 0 {=MM-2 \# "'March';'January';'February'"} {IF{=INT(MM/7)}= 0 {=MM-5 \# "'June';'April';'May'"} {IF{=INT(MM/10)}= 0 {=MM-8 \# "'September';'July';'August'"} {=MM-11 \# "'December';'October';'November'"}}}}"}

with each of the month name replaced by their Bengali equivalents (note the month orders in the code).

In either case, the fields may not update to reflect the current month until you do a print preview or something else to cause it (e.g. Ctrl-A, F9).

Note: The field brace pairs (i.e. '{ }') for the above example are created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practicable to add them via any of the standard Word dialogues.

For a macro to convert field text strings to working field codes, see:  http://www.gmayor.com/export_field.htm#TextToField
