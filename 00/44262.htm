hal hostetler - mvp-outlook say:
Windlows Live Mail would be the best solution, as it supports multiple email accounts with separate Inboxes, however, if you don't mind ALL email arriving with your business email address, go  to 'Tools|Accounts|Mail|(your email account)|Properties' and add your business email address to the "Reply To", box.  If this box is blank, Windows Mail uses the entry in the "E-mail Address" box as the reply address for all email.  If the box has an entry, it will use this "Reply To" address for all email instead.  Note that this means personal email, too.

Note that it's the server that terminated the connection, not the client (Windows Live Mail).

This problem often occurs with Yahoo accounts if there's a dodgy message on the server that Windows Live Mail doesn't want to download. Log in to the mailbox in webmail and start by emptying the spam and trash folders. Then move all of the messages out of the inbox and into one or more user-created folders. Conduct a send/receive in Windows Live Mail - is there any error message now? Do you see a No new email messages status message?

Once you have established a working connection, move half of the messages back into the inbox and repeat the send/receive. Repeat, halving the number of messages each time until the fault arises again. You should eventually be able to identify any dodgy one and deal with it appropriately.
