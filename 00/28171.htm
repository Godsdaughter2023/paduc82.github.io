john korchok say:

This is a pure guess, but I have seen a similar effect if a photo or other floating object is anchored to a paragraph, but also has wrapping turned on. When the object with wrapping is placed close to the paragraph it is anchored to, it tries to impose wrapping on the paragraph. This changes the shape of the paragraph and can make it jump to a different page in the document, taking the anchored object with it. The net visual effect is of a page you can't scroll to. If this is the case, you can probably view it in Draft or Outline View and see all content.

If this sounds like your issue, try making the floating objects on those pages into inline shapes by selecting them, then choosing the Format Picture tab of the Ribbon and selecting Position>In Line With Text. Then switch back to Print Layout View to see if the display is more stable.
