hotspot problem ? - Microsoft Community

jayprakashkamat say:

hotspot problem ?

when i change my wifi mac address then windows mobile hotspot not working why?

andre da costa say:
Because you basically made a completely different machine by changing it, so it would be unrecognizable to Mobile hotspot.

Press Windows key + X
Click Device Manager
Expand Network adapters
Right click then uninstall all virtual adapters

Open Start then type: regedit
Right click Regedit then click Run as administrator
Navigate to the following registry key:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WlanSvc\Parameters\HostedNetworkSettings\

Delete the following key:
HostedNetworkSettings

Restart then try again.

Run the Network adapter troubleshooter

Open Start > Settings > Update & security > Troubleshoot
Scroll down, select Network adapters
Click Run the troubleshooter

When complete, restart to see if it works.

If not, you can try updating both your Network adapter drivers or uninstall them.

Press Windows key + X
Click Device Manager
Expand network adapters
Right click your adapter, click uninstall

Exit Device Manager then restart

Windows 10 will re-detect them and reinstall them.

mouneshwar r say:

Hi b3ti3,

 

If you’re receiving printer offline message when you try to print a document, then there could have been a stuck print job that would cause the printer to go offline.

I would suggest you to follow the steps mentioned below

 

Step 1:Printing problems and printing errors

http://support.microsoft.com/gp/printing_problems

 

Step 2:Clear the print queue

a.    Click on Start; in the start search type “cmd” (without quotes).

b.    Right click on cmd.exe and choose to “Run as administrator”

c.    Type the following command and press Enter

              net stop spooler

d.    Type “del %systemroot%\System32\spool\printers\* /Q”(without quotes) and press Enter

e.    And now type “net start spooler”(without quotes) and press Enter

 

That is it. You have deleted and cleared printer queue.

 

Now try to print a test page and see if you’re able to print.

 

Step 3:Update your printer drivers

If the issue still persists beyond these steps, then I would suggest you to update your printer drivers. Visit the link below for the same.

 

http://windows.microsoft.com/en-US/windows7/Find-and-install-printer-drivers

 

Hope this information is helpful and let me know if you need any further assistance.
