vijay a. verma say:

Replace

TEXT(MONTH(TODAY())+1,"MMMM)

With

=TEXT(29*(MONTH(TODAY())+1),"MMMM")

Your month MONTH(TODAY())+1 will give 9 and excel treats this as day and assume month as 1 and year as 1900, hence it gives January. Multiply by 29 will move the date to that particular month and then Excel gives right month. 

You can also use a formula which makes sure that year, month and day all are available. Year may be anything and date may be anything as you need to know only month which has to be accurate i.e. MONTH(TODAY())+1 -

=TEXT(DATE(2015,MONTH(TODAY())+1,1),"MMMM")

You can also write as (but this is dependent upon your date format and below assumes MDY date format. Whereas above formulas are not dependent upon date format.

=TEXT(MONTH(TODAY())+1&"/1","MMMM")

pamcaswell say: First, go to the Styles pane > Manage Styles button > Set Defaults tab.   Make sure  that  WP MathB is not the font set in the defaults AND then make sure that the default settings match those in the normal paragraph style. This last bit is key to getting well behaved table text.  Back in the text, check to see if WP MathB still shows up as the table font when the cursor is just before the end of row marker.

If it is still there, it may have been set in the table style.

If the table style is Table Grid (its icon will show up in the tables gallery with a halo around it), the easiest way to get rid of unwanted settings is to  delete Table Grid.  Because it is a built-in style, Word will delete it from the document but leave the style available for use with its built-in settings.  The table will now be in the Table Normal style (just like table Grid but not borders.)  You can reapply the Table Grid style to get the  borders back. 

If the table style is user defined, modify the style to get rid of the unwanted font setting or delete and recreate the table style.

Pam

And about how to change the (-) sign to delta, here is I'm telling you:

- Insert a Delta symbol in any cell.

- Copy it.

- Select the cell, you want to change the negative sign to delta

- Press "CTRL+SHIFT+F" at once

- Click on "Number" tab.

- Choose "Custom" from "category"

- Type the below code there:

#,##0.00;#,##0.00;0

- Paste the delta sign after first ";"

#,##0.00;(Paste the delta symbol here)#,##0.00;0

- it will looks like this:

#,##0.00;▼#,##0.00;0

(for negative direction, Delta symbol is down)
