How to create 4 sections on Word 2007 - Microsoft Community

a. user say:

How to create 4 sections on Word 2007

On Microsoft Word 2007, how can a split a page into 4 sections and have a border around each section? (I am referring to 4 square sections in the page of Word 2007).
suzanne s. barnhill say:

As Graham says, a 2x2 table will do it. But you don't have to create this from scratch:

  1. Select the Mailings tab.
  2. In the Create group, click Labels.
  3. In the Labels dialog, click Options.
  4. In the "Label vendors" dropdown, select Microsoft.
  5. Scroll down to 1/4 Letter and make the appropriate choice (portrait or landscape).
  6. Click OK to close the Options dialog.
  7. Click New Document to open a new document and close the Labels dialog.

This is a little deceptive, as Word displays each quadrant as a single page; in effect, it prints four pages per sheet. The margins are set to 0" all around, so you may have to change them.

For a more conventional view of this layout, choose "Avery US Letter" as the vendor, then select stock number 3263 (landscape) or 8315 (portrait), then click OK and New Document.

You can do this on a form. Let's say that the control bound to the attachment field is named MyAttachment.

Create an On Got Focus event procedure for the control:

Private Sub MyAttachment_GotFocus()
    RunCommand acCmdManageAttachments
End Sub

Alternatively, use the Macro Builder to create a macro for the On Got Focus event.
