  1. Changes on Computer Visual Styles
  2. Disabled Services
  3. Incompetible Theme
  4. Security Issues (spyware & malware problem)

There are some fixes for Theme Problems. Lets discuss some of them:

  • Go to Start > Settings > Control Panel > System > Advanced and click onSettings button under Performance tab. If "custom" option is used, ensure that "Use visual styles on windows and buttons" checkbox is checked.
  • In Windows Explorer go to Tools > Folder Options. InGeneral tab, ensure that "Show common tasks in folders" checkbox is checked underTasks section.
  • Go to Start > Settings > Control Panel > Display > Appearance and ensure that "Windows XP Style" is selected underWindows and buttons drop-down list.
  • Click Start > Run and type services.msc to viewWindows XP Services Management Console. Ensure that Themes service is configured to run automatically and check status to verify that it has already started. (Note: You may want to right click on a item on the Name column to configure a specific service)
  • Click Start > Run and type services.msc again. Double click onTerminal Services item on the name column. Click on Log on tab. Ensure that service is enabled for defaulthardware profile.

http://windows.microsoft.com/en-us/windows-8/install-apps#1TC=t1

 

 

 

By default Windows 8 stores all apps in a folder "Windows Apps" present in "Program Files" folder. To access this folder follow these simple steps:

 

 

1) Type Computer on the Start Screen and go to C: drive. Now double-click on "Program Files" folder.

 

 

2) This folder contains "Windows Apps" folder but you'll not be able to see it because it is a hidden folder.

 

 

3) You'll need to enable the option to show hidden files and folders. You can enable the option by clicking on "View" menu in ribbon and enable "Hidden items" checkbox. Now you'll be able to see "Windows Apps" folder.

 

 

4) If you try to open this folder, you'll get following error message: “You don't currently have permission to access this folder”.

 

 

5) Click Continue to permanently get access to this folder.

When you click on continue button, you'll get another error message: “You have been denied permission to access this folder”.

 

To gain access to this folder you will need to use the security tab. It happens because you don't have sufficient permissions to open the folder. Hence, you'll need to take ownership of the folder.

 

 

6) I would suggest you to follow the steps outlined from “Issue 1: I get an "Access Denied" error message when accessing or working with files and folders” from the below provided link.

 

http://support.microsoft.com/kb/2623670#method4

 

 

7) Once you take ownership, you'll be able to view the content of "Windows Apps" folder.

How do I select which photos I wish to import in the Windows 7 Photo - Microsoft Community

h.m.hatcher.iv say:

How do I select which photos I wish to import in the Windows 7 Photo Import Wizard, and how do I change their names to a custom name group during import?

Has functionality been lost from Windows XP in Windows 7?

I am trying to do a simple import of photos using the native "Import Pictures and Videos Using Windows" Autoplay option.

In Windows XP, I could choose which pictures to import and give them all a custom base name during the import process, with each photo being subsequently appended with a "001", "002" etc. using the Scanner and Camera Wizard.  This was very handy as since there might have been several hundred photos on some of my cards from a long period of shots, I could run the tool multiple times for different groups of photos I would select for import, and the photos would get named sequentially and stored in their desired folders right away.  I imported thousands of photos this way.

This new tool for Windows 7 lets me add tags to the entire group of photos on the card/device, and name the photos after their date, or the tag, or other combinations of that... but why does this new tool in Windows 7 assume that I want to import every single photo on the card/device I just connected, and furthermore that I want them all to have the same tag assigned to them?  This is stupid and pointless - who out there actually only has photos of one topic on their card at any given time that they plug in to import their photos, and always wants to import everything every time? 

To illustrate this issue, let's look at my current situation.  I was on a vacation recently in Europe and took photos at more than one location I visited, of course, and now I would like to tag the photos with the location names individually... or even more, would like to actually name the photos during the import with the name of the place in the picture.  Scanner & Camera Wizard would do this, no problem - you just had to run the tool once for each different group name, and select the appropriate pictures to import with that group name.  It was easy.  This type of naming is far superior to adding tags to simple-list or date-wise named pictures because you can tell what picture file in your library you want to open just by browsing to (or searching for) the name group.  Want to see the pictures of Saint Peter's Basilica?  If you named the photos with that (appended with 001, 002 etc.), then just browse to that name in your "Italy Vacation 2010" pictures folder.  No trying to find the tagged photos in the sea of "IMG20100324005" name-soup, and no photo-album program like Windows Live Photos or Picasa needed either.

This new program in Windows 7 for importing pictures doesn't appear to be able to handle the very handy functionality of its predecessor from Windows XP.  It is very frustrating to have switched to the newest and supposedly "best" Windows version but find that several programs that were very useful in XP have had valuable functionality removed or are no longer present at all (like how Windows 7 doesn't have a Clipboard Viewer either - who's genius idea was that?).

So is there any way to recover within Windows the picture import functionality I have apparently lost by "upgrading" to Windows 7?  I of course could go buy a picture importing and editing program (Smart Photo Import looks nice for the price, from a quick search I just did), but since this functionality was native in Windows XP it's again stupid to have to now buy a 3rd-party program just because Microsoft doesn't understand the idea of retaining popular functions in their software and instead seems to "dumb it down."

Any help on selecting which photos I want to import and naming them during the import with a custom group name using the Import Wizard would be greatly appreciated... 10+ gigs of photos and videos from this trip are sitting & waiting for some solution!

Hi Stephane,

I realize the inconvenience you are facing with black screen issues on your computer. Now that you have me with you let me take care of this.

We appreciate your interest and time spent in resolving the issue.

This issue might have occurred due to corrupted/ outdated display drivers after the latest update. Let us try the troubleshooting steps below and check if it helps.

Method 1: If black screen error occurs before sign-in or after sign-in, I suggest you to refer to the below link replied by Sharath Srinivasa on August 7, 2015 and check if it helps:

Black screen after upgrading to Windows 10

If the issue persists, 

Method 2: I recommend you to download and install latest drivers from Manufacturer's website manually.

Step 1: I suggest you to download the display drivers for your device from the Manufacturer's website.

Refer: http://support.hp.com/in-en/drivers

Enter my HP model number and download the drivers.

Step 2: Boot the device to Safe mode and uninstall the existing drivers and restart the computer.

Refer: Start your PC in safe mode in Windows 10

Steps to Uninstall the Display Adapter driver.

  1. Press Windows key + X key when you are at desktop.
  2. Select Device Manager.
  3. Expand Display Adapter.
  4. Right click on the Display Adapter and click on Uninstall.
  5. Restart your computer.

Refer to the article by Ander Da Costa about How to: Install and Update drivers in Windows 10, try to install the drivers in compatibility mode if no latest drivers for Windows 10.

Step 3: Follow the below steps to place the computer in normal mode. 

  1. Press Windows + R keys to open Run command.
  2. Type msconfig and press Enter.
  3. In the System Configuration window, click on General tab.
  4. Select Start-up selection as Normal startup, click on Apply and Ok.
  5. Follow onscreen instructions.

Hope it helps. Reply to the post with an updated status of black screen issues on your computer for further assistance.
