Troubleshooting General Errors while Updating - Microsoft Community

prashantkumar96 say:

Troubleshooting General Errors while Updating

Technical Level : Intermediate

Summary
This Wiki discuss and provide solution to general errors encountered while updating Windows Defender and Microsoft Security Essentials.
Windows Defender and Microsoft Security Essentials.

Details
Windows Defender (WD)/Microsoft Security Essentials (MSE) use Windows Update as its backbone for updating (for full details you Windows Defender (WD)/Microsoft Security Essentials (MSE) use Windows Update as its backbone for updating (for full details you may want to read Wiki article MSE Definitions/Signatures Update FAQ). If Windows Update is not working properly then WD/MSE will also not work properly. So, there can be two cases when updating WD/MSE gives error-

  1. Windows Update is not working, thus WD/MSE is not working.
  2. Windows Update is working and problem is with WD/MSE itself.

Before you proceed:

1. Check if internet connection is working properly or not.

2. Check PC date/time.

3. Restart your PC and see if problem persists or not.

4. See if Windows Updates isn't already installing any update.

If Windows Update is not working properly

If Windows Update is not working properly then you should follow this article:

Fix Windows Update issues 

If for some reason the solutions provided above does not work, then below are some methods that you can try. Please try them serially:

1. Restart Windows Update Service

Follow these steps:

  • Press Win key + R. It will open Run dialog.
  • Type "services.msc" and hit enter.
  • In Services, scroll down to bottom and find 'Windows Update'.
  • Right click on 'Windows Update' and click on 'Restart' or 'Start'.

2. Resetting Windows Update Components

You can follow this article for some advanced methods of resetting Windows Update Components:

How do I reset Windows Update Components?

Warning: The article above requires bit of technical expertise in Windows command line.

If nothing solves your problem then you should start a new thread in Windows Update forum and get it fixed. Hopefully, WD/MSE will be fixed after Windows Update starts working properly.

If Windows Update is working properly

If Windows Update is working properly then problem is with WD/MSE. In this case, try these methods:

1. Check for conflicting software

Many antiviruses leave their traces after uninstalling them which may result in conflict with WD/MSE, causing it to malfunction. So their remnants should be removed using their removal tools. Removal tools can be downloaded from here:
List of antivirus product removal tools

2. Remove installed Virus Definitions

Start Command Prompt as Administrator –> Type these codes (hit enter after each code):

cd %ProgramFiles%

cd "Windows Defender" [cd "Microsoft Security Client" for MSE]
mpcmdrun.exe -removedefinitions -all
exit

It will remove installed definitions. Now try to update again.

3. Manually install Virus Definitions

By manually installing virus defintions, sometimes the problem within WD/MSE is fixed. Download definitions from here:
https://www.microsoft.com/en-us/wdsi/definitions

Install these definitions and see if it solves the problem or not.

4. Reinstall MSE

Uninstall MSE using any one method given here:

Uninstalling MSE

Then download and install latest version of MSE from here:
Microsoft Security Essentials

If everything fails then post a question in Virus & Malware forum with as much information as possible and the results of every method you tried.


Further reading:

How to troubleshoot definition update issues for Windows Defender

Windows Defender Event ID descriptions for troubleshooting
Troubleshooting update issues for Microsoft Security Essentials

MSE Definitions/Signatures Update FAQ

=SUMIFS($G$2:$G$6,$A$2:$A$6,$A12,$B$2:$B$6,B$11)

Conditionally adding/hiding text - Microsoft Community

servien say:

Conditionally adding/hiding text

How do I conditionally add or hide text in Word. For example in a standard letter I would like to add the text "Please see the enclosure for more details" if there is an enclosure and if there is no enclosure this text should be hidden, is it possible to do this using fill-in fields or something? More than one person uses this document so autocorrection isn't an option.
graham mayor say:

Save the following macro in the document template and add a docvariable field
{ DocVariable varEnclosure }
at the place you want the text to appear - then run the macro

Sub UpdateDocVariable()
Dim oVars As Variables
Dim sEnc As String
Set oVars = ActiveDocument.Variables
sEnc = MsgBox("Insert enclosure statement?", vbYesNo, "Enclosure")
If sEnc = vbYes Then
    oVars("varEnclosure").Value = "Please see the enclosure for more details"
Else
    oVars("varEnclosure").Value = " "
End If
ActiveDocument.Fields.Update
End Sub
