martin bar say:

Hi,

Uninstalling default apps in Windows 10 can be done using Windows PowerShell (Admin). We don't recommend uninstalling built-in apps If you wish to remove the apps, follow the steps below:

  1. Type Windows PowerShell in the search bar, then right-click Windows PowerShell from the search result. Select Run as Administrator.
  2. In the PowerShell prompt window, type in the following command to remove the Xbox:
    • Get-AppxPackage *xboxapp* | Remove-AppxPackage
  3. Press Enter after typing the command to complete the process.
  4. To remove other default apps, use the same command but replace xboxapp with the following text:
    • Calculator: windowscalculator
    • 3D Builder: 3dbuilder
    • Alarms and Clock: windowsalarms
    • Calendar and Mail: windowscommunicationsapps
    • Camera: windowscamera
    • Groove Music: zunemusic
    • Maps: windowsmaps
    • Movies & TV: zunevideo
    • OneNote: onenote
    • People: people
    • Photos: photos
    • Store: windowsstore
    • Voice Recorder: soundrecorder

Let us know if you need anything else. We're here to help.

Please read these instructions carefully and completely before you begin installing Windows 7: There are important steps that you need to follow to upgrade from your PC from Windows XP to Windows 7 in order to preserve your files and settings.
