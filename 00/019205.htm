The default contact/calendar folders are governed by the data store that is configured as the "default data file" for the profile. From OL menu, go to File --> Account Settings --> Data File tab and set the data file for your IMAP account as the default

However, would advise against using the IMAP "This Computer Only" contact folder as the default since if you ever delete the account, all the data in "This Computer Only" folders will be lost unless you explicitly backup that data manually. You're far better off keeping your default contacts in the Outlook.com account or create a new PST file and make that the default data file for your account and move all your contact/calendar data to the corresponding folders in the PST file.
