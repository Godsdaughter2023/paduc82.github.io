scottgem say:

First, lets make sure we have some terminology straight. In access terms a Database is a collection of objects. Objects are tables, forms, reports queries etc. A database may be in a single file or it may span multiple files. Generally, multi-user databases, span at least 2 files. A back end containing the tables and a front end containing everything else.

So when you said files, I assumed that you have a single user (at least at a time) app on one computer contained in one file and the entire file is copied over each month. 

Again, this is not good design and makes it difficult to compare one month against each other. 

If Compact and Repair doesn't work, then I would create a blank file and import all the objects into from the problem file and see if that fixes the problem.

Angle symbol - Microsoft Community

keerthisagar say:

Angle symbol

How to insert angle symbol in ms word 13?
suzanne s. barnhill say:

The angle symbol is Unicode character 2220. "Measured angle" and "spherical angle" are 2221 and 2222. They do not appear in all fonts but can be found in Cambria Math. As Mary says, they are in the Mathematical Operators character subset.

You can insert one by typing the number (such as 2220) and pressing Alt+X; the result will depend on the Word version: in Word 2010, I do get Cambria Math regardless of the font in use, but in Word 2003, if I start with Times New Roman, I get MS Mincho, in Calibri, I get MS Gothic, and only in Cambria do I get Cambria Math.
