The text must be inserted at a specified location (before the final -but may vary in number - section break), and there are several different styles applied to the various paragraphs - i.e. heading level 1, heading level 2, body text etc.

The code i have written to achieve this firstly finds the reference point (by searching for some non-printing text which i've included for other reasons), and then inserts a line of text. I then use the line of text as my insertion point and am inserting paragraph by paragraph using the code below. Is there a slicker way to achieve this? I'm currently repeating a lot of the code at each With .Range.HighlightColorIndex = wdNoHighlight         .Collapse wdCollapseEnd etc There are another 22 clauses to include, each with a heading and several paragraphs.

I would have designed the macro to simply delete the T+Cs which were inserted already into the template, but people change their minds and want them back in after all, so this seemed a more fail-safe and flexible option.

 

Private Sub Find_Location_Insert_Text()
Dim rngStory As Range
  Dim MyText As String
    MyText = "found it working" & Chr(10)
 For Each rngStory In ActiveDocument.StoryRanges
    With rngStory.Find
    .ClearFormatting
    .Text = "DO NOT DELETE THIS SECTION BREAK" 'this is the non-printing text preceding the final section break
    .Execute Forward:=True, _
                Wrap:=wdFindStop
   rngStory.InsertBefore MyText
   rngStory.Collapse wdCollapseEnd
End With
Next rngStory

Dim MyString As String
'T+Cs heading
    With Selection
        .ClearFormatting
        .Text = "General Terms and Conditions" & Chr(10)
        .Style = ActiveDocument.Styles("Heading 1")
        .Range.HighlightColorIndex = wdNoHighlight
        .Collapse wdCollapseEnd
        End With
'T+Cs para 1
    With Selection
        .ClearFormatting
        .Text = "DEFINITIONS" & Chr(10)
        .Style = ActiveDocument.Styles("Terms and Conditions Level 1")
        .Range.HighlightColorIndex = wdNoHighlight
        .Collapse wdCollapseEnd
    End With
'T+Cs para 1
    With Selection
        .ClearFormatting
        .Text = "" & Chr(34) & "Client" & Chr(34) & " shall mean the company or organisation .......etc;" & Chr(10) & Chr(34) & "Contract" & Chr(34) & " shall mean the agreement between ...... etc;" & Chr(10)
        .Style = ActiveDocument.Styles("Terms and Conditions Body Text")
        .Range.HighlightColorIndex = wdNoHighlight
        .Collapse wdCollapseEnd
    End With

'etc etc.....
   
'Remove "found it working"
With Selection.Find
        .ClearFormatting
        .Text = "found it working"
        .Replacement.Text = ""
        .Execute Replace:=wdReplaceOne, Forward:=True, _
                Wrap:=wdFindStop
End With
End Sub

Set oRng = ActiveDocument.Sections(iSection).Range
iParas = oRng.Paragraphs.Count - 2
oRng.Start = oRng.Paragraphs(iParas).Range.Start
oRng.End = oRng.Start
Application.Templates(strPath & "MyTemplateName.dot").AutoTextEntries("MyFormattedText").Insert Where:=oRng, RichText:=True

This will insert a formatted autotext entry called "MyFormattedText" before your "DO NOT DELETE THIS SECTION BREAK" text (assuming that there is a paragraph break at the end of that text followed immediately bgy a section break) from the document template called "MyTemplateName.dot" which would be the name of the template that creates the document.

Insert the full text into your document, format it then select and Press ALT+F3. Save it as an autotext entry in the template itself, then delete the text. Running the macro will replace it. Change the names of template and autotext entry as appropriate.
