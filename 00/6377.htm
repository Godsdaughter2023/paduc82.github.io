When the object to which you are adding a caption is In Line With Text, the caption will also be inserted inline. When you say “For some reason the archiving is not working”, may I know whether you have enabled the archive mailbox for the users in your organization. If the archive mailbox isn't enabled for a user, then no action is taken on items. Those items will remain in the user's primary mailbox.

•    In the EAC, go to Recipients > Mailboxes, and then select the mailbox from the list. In the details pane, under In-Place Archive, confirm that it's set to Enabled. Click View details to view information such as archive status, usage, and quotas

Get-Mailbox <Name> | FL Name,*Archive*

For more detailed information, please refer to the article: Enable or disable an archive mailbox in Exchange Online

Yes, it will work. However, when you put a mailbox on Litigation Hold or In-Place Hold, the hold is placed on both the primary and the archive mailbox.

Archive mailboxes in Exchange Online
In-Place Hold and Litigation Hold

You can add .jpg  pictures in mail merge. To do this, you need to type either the file name or path (path: The route that the operating system uses to locate a folder or file; for example, C:\House finances\March.doc.) for each picture file that you want to use in the appropriate column or data field for each record. Let’s follow one of the following:

1.If you saved your picture files and your data source in the same folder, type the file name for the relevant picture. For example, type firstphoto.jpg.

steve rindsberg say:
There are two fundamentally different types of computer graphics.  

One kind describes graphics as data that represents shapes like squares, circles and so on.  Ungroup one of these in PowerPoint and you get the individual circles/squares etc. that make up the graphic.  PowerPoint can edit these.

The other kind describes them like a pointillist painting ... lots of dots (pixels) that, together, happen to look like a square or circle.  It wouldn't make sense to ungroup them because it'd leave nothing to edit, at least not in PowerPoint.   These are called images or raster graphics.

There are programs (image editors, paint programs, names like that) that let you edit these images but once in PPT, you can't do much with them other than resize/move them around.

There are also programs that trace the patterns of dots in images and create new graphics of the first type, consisting of editable shapes.  CorelDraw is one program that includes a trace utility; I'm sure there are others.
