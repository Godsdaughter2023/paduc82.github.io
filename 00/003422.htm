A nested IF can be done along the lines of my formula above, but it will be hard to read and maintain. 

I suggest you build a lookup table and use Index/Match to lookup the result based on that table. Consider the following screenshot:

The formula in cell N1 is

=INDEX($B$3:$C$8,MATCH(L1,$A$3:$A$8,0),MATCH(M1,$B$2:$C$2,0))

It finds the row in column A where the value matches L1, then it finds the column in row 2 where the value matches M1 and returns the value at the intersection of the found row and column.
