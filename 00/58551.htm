ken blake say:
Back in the days of DOS, the PrtScn key used to print the screen. But in all versions of Windows, this works differently, and the name of the key is now an anachronism. The key does *not* print the screen. PrtScrn captures the whole screen, and Alt-PrtScrn captures the active window.

Either one captures the image to the Windows clipboard. Once it's in the clipboard you can paste (Ctrl-V) it into any application that supports graphics (Windows Paint, other graphics programs, even your favorite word processor). You can edit or add to the image as you wish, then print it.

This ability to manipulate the image in a program before printing it is an improvement over the original DOS method of just printing it. But if you'd like that old facility back, there are several third-party freeware/shareware programs that can do this, such as PrintKey2000 at http://www.sharewarejunkies.com/00zwd2/printkey2000.htm

in order to delete the files from trusted installer, you have to first take ownership of the files or folders and then grant yourself full control permissions and rights!
You have to do all of that just to be able to rename, delete, or edit these files and folders. In this article, I’ll walk you through the steps in order to do this.
1. First, go to the folder or set of files that you need to change permissions for, right-click on them and choose Properties.
2. Then Click on Security.
3. Click on Advanced at the bottom
4. click on the Owner tab and you’ll now see that the current owner is
5. click on the Edit button and choose who you would like to change the owner to, either your account or the Administrators. If your account is an Administrator account, I would suggest just picking Administrators.

8. Now click on the Security tab again, but instead of clicking on Advanced, you need to click the Edit button.
9. Now click on the user name in the list that you want to change the permissions for, which should be the same as who you changed the current owner too. If the user name is not in the list, click Add, type in the name and click OK.
10. Since I had changed the current owner to Administrators, I clicked on Administrators here and then clicked on the check box next to Full Control. When you do that, all the other boxes get checked too.

lemp say:

You should not be installing from a Command Prompt.

You need to use the ISO file to burn a DVD (or USB flash drive) and boot from that.

To create a bootable DVD, download, install, and run (free) ImgBurn:  http://www.imgburn.com/

If your computer is capable of booting from a USB flash drive, then to create a bootable USB flash drive, download, install and run (free) Rufus:  https://rufus.ie/  The USB flash drive must be at least 4GB.  When you create the bootable USB flash drive from the ISO file, everything previously on the flash drive will be deleted.

Once you have created the bootable media, insert the DVD into your computer's DVD drive or connect the USB flash drive to the computer and reboot.

Depending on your computer, you may see a menu just after power on that will instruct you what key to press in order to select the boot device.  If your computer does not offer such a "boot menu," you will have to press the appropriate key just after power on in order to enter the computer's BIOS setup utility.  In the BIOS setup, change the boot sequence so that the computer will start from the optical drive or the USB drive (as appropriate).

The key to press to enter the BIOS should be shown just after power on.  Or see the list here:  https://www.tomshardware.com/reviews/bios-keys-to-access-your-firmware,5732.html

If you must use Microsoft tools instead of ImgBurn or Rufus, you can use the Microsoft USB/DVD Download Tool:  https://www.microsoft.com/en-us/download/windows-usb-dvd-download-tool

See this answer by the developer of Rufus:

Not much that can be done when Rufus especially indicates that "The device is in use by another process". You will need to find what process is accessing the drive and prevent it to do so. It may be a security solution or one of the software listed at the end of the FAQ.
