If you double click a file in a non left-most column where the selected file (or any other file in the same column) has even 1 character of its filename 'cut off' by the Explorer window boundaries, the file does not execute. As you can see the file name is cut off but you can see what the file is.

 

When you go to execute the file you double click it and it should open.

 

Instead of executing explorer automatically scrolls over requiring you to double click once more.
