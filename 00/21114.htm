steve rindsberg say:

If at all possible, save the graphic to a different file format.  PNG or TIFF both support transparency or if it's a vector graphic, try EMF.

If you have Illustrator, Photoshop, CorelDraw or the like, try importing the EPS into one of those, then saving in one of the formats I mentioned above.

Or if you have Adobe Acrobat, Distiller can convert the EPS to a PDF and from there you can export to PNG or TIFF.

Office programs don't really handle EPS properly.  Instead, they try to convert the EPS into EMF format using a built-in PostScript interpreter.  Unfortunately for you, the interpreter is only PS Level 2 and can't handle transparency.  What you're seeing is an error message from the interpreter.

I found a solution.

I also was frustrated that the clean up function in people/outlook.com was not only ineffective but also combined several different guys named Bill into one contact. Luckily I had copies.

This is the solution I worked out:

1. I had (or you can create) a new pst file in Outlook (2010 in my case) and copied my contacts to that pst file to edit out the duplicates. (File/account settings/data files/add)

2. Then I deleted all the contacts in the outlook.com (ost file) .i.e., change view to a list and remove categories and other sorts, click on one, the control-A to select all, then delete.

3. Wait a while. Then log in to outlook.com with a browser and you should find all the people are gone.

4. If so, open the contact file in your pst and for safety sake, either copy to another pst file (fastest) or export to a csv.

5. Then again list and select all the contacts (crtrl-A) and right-click, move to the outlook.com contact file. I prefer move to keep track of those not transfering. If like me you have contacts that you stuck little notes in the email field, it will reject these along with ones whose company name exceeds 40 characters or whose total length exceeds 1024 characters. In the first case the copy stops and you can edit the contact at the top of the list to clean up the email (or email 2 or email 3 or im address or web address which are all part of that failure). In the other cases saying ok will truncate the company name or notes that are probably pushing the 1024 envelope.

I'm an old guy who's had email since 1977 so forgive me if I misremembered sizes above.

6. You could take a chance and import the clean file from the pst using the function in the people screen, but I preferred to just copy all the contacts from my clean pst file to the outlook.com file on my pc in hopes of preserving my categories. (again in contacts view, select one; then control A to select all; then drag them to outlook.com contact file.);

I have been looking for a solution to this problem all over.  thank you for all the info provided that led me to clean up the "duplicates" in People/Contacts on outlook.com and my Windows Phone and later my iphone.  OP suggested to delete all contacts and replace with a clean import, so that's the way:

1. login to outlook.com or outlook.live.com, click the upper left 9-tile apps button, then click "People".

2. go to "Manage" -> Export contacts, and in the Export window select "All contacts" or a folder under "Contacts from this folder".  click Export.

3. save the "contacts.csv" file to a known folder location.  double check the file in Excel that all the contacts are there.  the order of the contacts may be in reverse order or whatever your lists were saved in the exchange.  but the counts should be correct, including duplicates.

4. Edit the file in Excel painstakingly: highlight all the columns except first row, and use data->sort: column A (first name) then column C (last name).  delete dup rows and merge items manually.  you can reorder them after edits to re-edit.  you can probably use some macros or formulas to merge the data.  I didn't move the columns - not sure if that matters.  save the edited file to .csv format,

5. back to outlook.com People window, click on any contact in the second pane showing the list, then hit Ctrl-A.  alternatively hover your mouse over "Your contacts" just above the list.  you'd see a little checkbox to the left.  click it.  that selects all the checkmarks of the contacts. (you can try unchecking some if you want to "save" some that have "notes" in there, but it barked at me with an error message).

6. hit Delete.  in a few minutes all contacts would be deleted from outlook.com and phone.  gone are those custom contact tiles that you might've pinned to the windows phone as well.

7. now to import: click "Manage" -> Import Contacts, and select "Windows Live Mail" option and follow the instructions.  the other options may work as well - not sure.  choose the edited contacts.csv file you painstakingly cleaned. 

8. if you go to your Windows phone, People (or phone book), and click the "..." on the lower right and "settings", scroll all the way down to "accounts", you'd see the Windows Live account showing update activities of the contacts counts.

9. wait a long while (half an hour or more until exchange is caught up), then click "Manage" -> Restore deleted items, if you want to save any previous contacts from deleted list.  you have 30 days to do so.  note that the deleted items have the same date 7/9/2016 next to them.  I think that's when outlook.com messed up my phone's contacts sync.

9. on iphone and androids if you had already linked up to outlook.com/Live account the contacts should be updating during the same time.  on iphone "Contacts" -> Groups, you can turn on each group, icloud or Outlook or both.

10. re-pin the quick contacts to your windows phone as needed.

I spent a day paring 1000 lines down to 400 unique contacts.  there were contacts with dups up to a dozen lines - ridiculous!  maybe some were old contacts from feature phones or old sim cards with weird formats that outlook.com or Live/exchange didn't recognized.

the exchange/outlook.com dup bug(s) probably didn't do a good job with merging and syncing, especially with outdated windows phones unsupported or tested.  items that had number format xxxxxxxxxx might've gone thru some conversion to (xxx) xxx xxxx.  or it might've tried to combine multiple entries of different phones and emails in different columns and created multiple entries.  outlook.com and Live treat those dups as "linked", as one contact.  it's fine within windows but not when you need to work with google android or ios phones.  also the .csv formats are different between them.  outlook.com only exports one Microsoft format and loses some of the info like "notes" and "company" info.  if you "recover" some contacts with same names, outlook.com would link them, as "one contact" for Live and windows phone.  but they would be separate contacts for iphone and google.  the "Manage" -> Clean up contacts don't quite work, as it may not see that as a problem.  also, until 30days is over or you manually purged the deleted items, that clean up tool checks also the deleted folder, now treating each contact as separate.  hey maybe there's a better way to do this: delete all contacts, then use clean up tool to get rid of the dups, then recover them? try at your own risk.
