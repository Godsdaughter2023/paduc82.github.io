shishira d say:

Hi stinky000001,

 

Desktop.ini and index.dat are system files which are created for initialization and configuration.

 

Desktop.ini is a Initialization/Configuration File found in folders, it will be created automatically by system even after you delete it. It stores the information needed to display your folder contents the way you’ve customized.

 

Index.dat is a kind of database file which runs as long as the user is logged in, so in most cases it is not possible to delete it. It stores information about your search queries and url paths typed, etc.

 

If it’s deleted, it will be recreated by system automatically and run as long as Internet Explorer is running, though you have deleted it from the folder.
