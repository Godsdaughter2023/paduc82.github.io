canadian tech say:

32 bit is slower than 64, but you could not run 64 with just 1G of memory.  You need to add memory.  If you keep the 32 bit install, there is no point to more than 3 G, but the reason you have such slow performance is that you do not have nearly enough memory.  As a result of not enough memory your system is constantly thrashing the hard drive as it caches out stuff to the hard drive to get enough working memory to do anything.  If you go to 64, you should go to at least 4 Gb.  Always make certain when working with memory that you balance the pieces.

On top of that, your hard drive is full.  That may surprise you.  When you exceed 75% of capacity on a hard drive, performance drops like a rock.  When you add to that the virtual memory space your computer is using on the hard drive, your hard drive is over-worked.  If it is original OEM, it is likely to fail soon.

You are expecting way too much of your OLD computer.  80 to 90 applications is way more than you have a right to expect.

Bottom line,

  1. you can not run that many applications
  2. You must add memory
  3. It would be advisable to replace the hard drive with a larger one, while you are at that, install 64 bit.

I've just spent three days finding a solution as to why Windows Update would not work after carrying out a "Factory Restore" on my two Windows XP Home Premium netbooks (both with S.P.3). In the end, the solution was quite easy & requires absolutely no messing around with the registry;
  1. Download Internet Explorer 8, but UNTICK the search for updates box & let it install
  2. After installation, open I.E.8 & follow instructions/preferences, then click ok to finish
  3. Search online for Cumulative Updates for I.E.8 & install (KB2888505-x86-ENU.exe)
  4. Click on Windows Update Website, after which you will be invited to install ActiveX... click yes to install
  5. Problem solved... Windows Update works immediately....Happy Days... At least until 8th April 2014
Both netbooks are now successfully updating from the Windows Update Website even as I type.
