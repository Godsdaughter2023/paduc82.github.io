aljon_t say:

Hi there,

Question 1: Profile name doesn't match skype name? should it? can I change it to match? How?

Answer: Your Skype name serves as the unique identification of your Skype account and cannot be changed. After migration, customizing a Skype name is no longer available. It will be a system-generated that has a "live:" prefix followed by the word or whatever characters come before @ in the email address (given that you used email address to create the account).

On the other hand, your profile name or display name is your preferred name when you create the account. It can be changed and modified.

Question 2: What is skype user id and Microsoft user id? I have both. what to do? yeah I have a pc and I have an iPhone.

Answer: Since Skype is part of Microsoft, after the migration, all Skype accounts are considered as Microsoft accounts. If your Skype user ID starts with "live:", it is considered as Microsoft ID/account.

Question 3: their relation to skype? should I delete one of them, Having two will confuse the system?

Answer: Technically, if you delete a Skype account, it will also delete the associated Microsoft account with it. Please be guided that you can use a Microsoft account to sign in to any Microsoft products including Skype. How do I close my Skype account?

Question 4: I want to use skype only on iPhone 5s.

Answer: If you want to use Skype on a preferred device (iPhone), then I suggest to only sign in to the device you want to use Skype on.

Question 5: so why is skype so confusing? why is the roll out so bungled?

Answer: Are you referring to the latest version of Skype? If so, please be advised that we gradually releasing updates based on the feedback and needs of our valued users.

Question 6: can't seem to edit information already entered; can I?

Answer: You can edit your profile information on your account dashboard. Refer to the steps below.

  • Sign in to this link.
  • Scroll down to "Settings and preferences" > Edit profile.
  • Click "Edit profile".
  • Make your preferred changes.
  • Hit Save.

Question 7: afraid to link such a nebulous system to my iPhone contacts? doesn't appear to offer much control of the listing nor explain what happens if I say OK; what does happen if I say ok? what if I don't want all those people to skype me? how to control?

Answer: If you sync your iPhone contacts to Skype, it will show contacts that have Skype account in the contact icon. You will be prompted to add these suggested contacts.

Hope this information helps. Feel free to post back if you have questions and clarifications

In the event that the same Skype Name appears on both apps on both devices, please respond to this thread.
