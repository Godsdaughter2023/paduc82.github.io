jeeped say:
My son uses this laptop and every icon has changed to the Mozilla/Firefox emblem.

I've seen this happen when upon opening a program through a shortcut (e.g. .LNK file), you are asked what program to open it with. After selecting Mozilla/Firefox (or some other program) via the Select a program from a list of installed programs, and checking the Always use the selected program to open this type of file (circled in red below), the document or program referenced by the original shortcut opens in Mozilla/Firefox as it should.

The trouble is that now all shortcuts (e.g. .LNK files) on the desktop or throughout Explorer will want to be opened by Mozilla/Firefox and use the Mozilla/Firefox icon to display the shortcut regardless of what the shortcut was originally intended to open. This is known as a file association and default icon. The same condition could occur if you performed those 'Open With' actions on a .JPG or even an .EXE, but the .LNK explanation is far more likely for items on the desktop.

The good news is that a small free utility has been written to undo the .LNK file association that was created. There is also a registry hack to accomplish this, but I recommend the unASSOC utility for its ease of use.

You can get a copy from my SkyDrive here. I used to send people somewhere else to download a copy, but that page is getting increasingly confusing with the misleading ads to download this and that.

Just download and then run the utility to unassociate the .LNK files. Select .LNK from the list on the left and click either Remove file association (User) or Delete file type and restart the computer.

Hi Chris,

This issue might have occurred due to incompatible or corrupt hardware drivers. Follow these methods to troubleshoot the issue:

 

Method 1:

I would also suggest you to run the Hardware and devices troubleshooter to troubleshoot the device:

a. Press Windows key to launch the Start screen.

b. Type “Troubleshooting” without quotes and click on Troubleshooting.

c. In the new window, click View all on the left pane.

d. In the list of options, click on Hardware and devices.

e. Click Next and follow the on-screen instructions.

 

Method 2:

I would suggest you to uninstall the drivers completely and re-install them by downloading it from the manufacturer’s website:

a.     Swipe in from the right edge of the screen, and then tap Search.

(If you're using a mouse, point to the upper-right corner of the screen, move the mouse pointer down, and then click Search.)

b.    Enter “Device manger” without quotes in the search box.

c.     Tap on Device manager.

d.    Right-click on the device driver and click Uninstall.

e.    Restart the computer.

 

Refer to the link to update the drivers:

Download and install drivers

http://windows.microsoft.com/en-in/windows-8/all-drivers

 

Method 3:

I would suggest you to disable and re-enable the USB controllers. The following steps will uninstall all the USB controllers and then re-install them:

a.     Right click on Start, and then click Run.

b.    Type “devmgmt.msc” without quotes, and then click OK. Device Manager opens.

c.     Expand Universal Serial Bus controllers.
Note: You might have to scroll down the list to find this item.

d.    Right click the first USB controller under Universal Serial Bus controllers, and then click Uninstall to remove it.

e.    Repeat step 4 for each USB controller that is listed under Universal Serial Bus controllers.

 

Note: Restart the computer. After the computer starts, Windows will automatically scan for hardware changes and reinstall all the USB controllers that you uninstalled.

I had a M6600 for a while - nice machine (had an SSD and a couple of other drives for data.)
 
Hibernation is not something I have ever recommended because it often takes longer (and with unreliable results in my experience) than just turning on a properly shutdown computer and starting up the applications needed at the time.  This is even more true - in my experience - with Windows 8 now and SSDs.
 
Given your drive size, you don't have an SSD most likely.  You have a regular disk drive.  What "speed" is that drive (5400?  7200?)
 
Is the BIOS up to date?
 
I found that getting the drivers direct from Intel for much of my M6600 made a lot of difference in performance (newer drivers than Dell was providing.)  You may want to do this.
 
What antivirus software are you using?
 
Any other security software other than the built in stuff (firewall, etc.)?
 
Is this something that just started happening or has it always been this way?
 
What all is opened when you hibernate?
 
Have you timed a shutdown and subsequent power up + logon until time you can use the system/load what you need and compared it to a hibernation test for the same stuff?
 
How much FREE space do you have on the drive?
 
Have you turned off Hibernation, cleared the file off the system and defragmented then followed it up by turning Hibernation back on and trying again?

seashore44 say:

My winsxs file is at 13GB and growing. Any chance of deleting anything in this file?

My winsxs file is at 13GB and growing. Anything I can delete from this folder? IE8 Windows 7.

a. user say:

Hi,

 

I request you to not delete the WinSxS folder or its contents. All the necessary files for specific additions, roles, applications will be contained in the %windir%\WinSxS directory. The %windir%\WinSxS folder (also referred to as the component store) is used to store all the installation source files that are needed for Windows to service itself and its optional components, which takes the place of the traditional flat from media.

 

Deleting, moving or adjusting the WinSxS folder will be fatal. Clean installation of Windows 7 is the only resolution.

 

However, you can un-install the programs which you do not use. This will help with the size of the WinSxS folder. Refer to the following link:

 

How to address disk space issues that are caused by a large Windows component store (WinSxS) directory
http://support.microsoft.com/kb/2795190

 

I hope this information is helpful. If you have any further questions regarding Windows operating systems, reach out to us on this forum.
