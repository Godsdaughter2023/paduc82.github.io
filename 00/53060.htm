carolina calling say:

Click on Start->Control Panel->Network Connections.  This should open Network Connections in Windows Explorer.  You should see a "Local Area Connection" or a Wireless connection icon.  (The name may vary depending on how many time connection setup has been run, etc.)  Right click on this icon and select, "Properties".  The properties panel has two panes under the "General" tab: "Connection using" and "This connection uses the following items:"  Connection using should be your Network Interface Card (NIC) and there should be serveral protocols installed including "Internet Protocol (TCP/IP)".  If it's not there, "install" it.  Make sure the check box is selected, highlight the entry by clicking on it and click "Properties" (down to the right).  This should open the "Internet Protocol (TCP/IP) Properties" window at the "General" tab.  Be sure the "Obtain an IP address automatically" radio button is selected or that you have typed in a valid IP address, netmask and default gateway.  Select the "Obtain DNS server address automatically" radio button or enter at least one valid DNS server address.  Click "OK" on the two open windows and close the network connections window.  Now, run "cmd".  In CMD, first, run: ipconfig /release

then run: ipconfig /renew

Hopefully, after a little wait, you've now got an Internet connection (again).

Good luck!
