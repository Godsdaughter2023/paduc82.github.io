andrew sayers say:

There's always a risk with any operating system upgrade, particularly unfinished systems such as the tech preview.

Ideally back up all your files first to a separate drive such as a flash drive and then go for the install. I've done it on a couple of machines, over-writing Windows 8.1 on one and as a clean install on another and didn't encounter any problems with either so you'll probably be okay too.

The advantage of installing over an existing system is that you'll not need to install all your programs again. As long as your data is securely backed up you've not really got a lot to lose.

Sub GetOpenWB()
On Error GoTo errh
Dim myWB As Workbook, WB As Workbook
Dim myWS As Worksheet
Set myWB = ThisWorkbook
Dim myFile As Variant
Dim v As Variant, v1 As Variant
v1 = Array("Sheet1", "Sheet2", "DataSheet", "Table2_Sht")   '<<< change Names
Dim L As Long
myFile = Application.GetOpenFilename("Excel Files (*.xls*), *.xls*", , MultiSelect:=True)
Application.ScreenUpdating = False
For Each v In myFile
Set WB = Workbooks.Open(v)
For x = 0 To UBound(v1)
Set myWS = myWB.Sheets.Add
r = WB.Sheets(v1(x)).UsedRange.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
WB.Sheets(v1(x)).Rows("1:" & r).Copy
myWS.Cells(1, 1).PasteSpecial xlPasteFormats
myWS.Cells(1, 1).PasteSpecial xlPasteValues
Application.CutCopyMode = False
Next x
WB.Close False
Next
myWB.Save
Application.ScreenUpdating = True
Exit Sub
errh:
MsgBox "wrong.."
End Sub
