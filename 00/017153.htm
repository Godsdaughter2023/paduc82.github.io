vba save - filename = ContentControl

jay freedman say:

The following code may seem like overkill, but it behaves properly if one or more of the content controls has no entry, or has been deleted.

The code as written assumes that the content controls have been given unique Tag values in their Property dialogs, as "first", "last", and "date" respectively. If you use different tag values, change the code accordingly.

                                        

Sub FileSave()
    If ActiveDocument.Path = "" Then
        Call FileSaveAs
    Else
        ActiveDocument.Save
    End If
End Sub
Sub FileSaveAs()
    Dim strTag As String
    Dim strFirst As String
    Dim strLast As String
    Dim strDate As String
    Dim strPath As String
    Dim strFilename As String

    strPath = "\\filesrv\test\"  ' <== change as needed

    strTag = "first"
    strFirst = GetCCcontentbyTag(strTag)
    If strFirst = "**EmptyCC**" Then
        MsgNotReady strTag
        Exit Sub
    ElseIf strFirst = "**MissingCC**" Then
        MsgMissing strTag
        Exit Sub
    End If

    strTag = "last"
    strLast = GetCCcontentbyTag(strTag)
    If strLast = "**EmptyCC**" Then
        MsgNotReady strTag
        Exit Sub
    ElseIf strLast = "**MissingCC**" Then
        MsgMissing strTag
        Exit Sub
    End If

    strTag = "date"
    strDate = GetCCcontentbyTag(strTag)
    If strDate = "**EmptyCC**" Then
        MsgNotReady strTag
        Exit Sub
    ElseIf strDate = "**MissingCC**" Then
        MsgMissing strTag
        Exit Sub
    End If

    ActiveDocument.SaveAs2 FileName:=strPath & strFirst & "_" & _
        strLast & "_" & strDate & ".docx"
End Sub

Function GetCCcontentbyTag(theTag As String) As String
    Dim CCs As ContentControls
    Dim CCcontent As String

    Set CCs = ActiveDocument.SelectContentControlsByTag(theTag)
    If CCs.Count = 0 Then
        ' Should not get here if CCs are marked "cannot be deleted"
        CCcontent = "**MissingCC**"
    Else
        If CCs(1).ShowingPlaceholderText Then
            CCcontent = "**EmptyCC**"
        Else
            CCcontent = CCs(1).Range.Text
        End If
    End If
    GetCCcontentbyTag = CCcontent
End Function

Sub MsgNotReady(theTag As String)
    Dim strTitle As String
    Dim strMsg As String

    Select Case theTag
        Case "first"
            strTitle = "First Name"
        Case "last"
            strTitle = "Last Name"
        Case "date"
            strTitle = "Date"
        Case Else
            ' can't get here except by error
            strTitle = "unknown"
    End Select

    strMsg = "The " & strTitle & " box needs an entry." & vbCr & _
        "Please complete it and try again to save."
    MsgBox strMsg
End Sub

Sub MsgMissing(theTag As String)
    Dim strTitle As String
    Dim strMsg As String

    Select Case theTag
        Case "first"
            strTitle = "First Name"
        Case "last"
            strTitle = "Last Name"
        Case "date"
            strTitle = "Date"
        Case Else
            ' can't get here except by error
            strTitle = "unknown"
    End Select

    strMsg = "The " & strTitle & " box is missing." & vbCr & _
        "Please contact <<name of programmer>>."
    MsgBox strMsg
End Sub

Instead of writing separate FileSave and FileSaveAs macros, you could write a class module that contains an event handler for the DocumentBeforeSave event. That would probably be more reliable (but if you go that way, be aware of http://blogs.msmvps.com/wordmeister/2013/03/17/word2013-isinautosave/).

Since you're working with dates, I don't think you can do this with a Calculated Field. Also, your date field is text. For these reasons you need helper columns in your data source.

Assuming your dates are in column A, with a single row of headers in row 1, 'Issue Value' is in column E, and data starting in row 2, enter these two helper columns with formulas in row 2:

MAT Date (assumed to be column F):
