You don't need the code in your post above.
A combo will filter itself as the user types.

To get the value of the combo to use somewhere else, you just use the value property of the combo in its after update event.

Private Sub ComboName_AfterUpdate()
Dim lngValue as Long

If Not IsNull(Me.ComboName) Then
lngValue = Me.ComboName
End If

End Sub

Replace ComboName with your combo's name.
