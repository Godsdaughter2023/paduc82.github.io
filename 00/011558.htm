bob jones aka: cybertaz say:

What you're seeing is a residual graphic or artifact which is the result of a useless screen tip that identifies the gray space as the Markup Area. The box should be yellow & contain that phrase... but it also should go away if you click in the gray space or move the pointer into the main document area. Apparently that isn't happening for you.

See what happens if you go to The Review tab & choose Final from the viewing options or deselect Comments form the Show Markup dropdown. When you reverse the change it may refresh the Markup Area & eliminate the ghost.

If that doesn't work you may have a bad .plist (Preferences) file or possibly a corrupt Normal.dotm template. To replace either it's first necessary to Quit Word - don't just close the document windows. Then;

To replace the .plist file

  1. In Finder, hold the Option key while you open the Go menu,
  2. Select Library
  3. Open the Preferences folder
  4. Locate the file: com.microsoft.Word.plist & drag it to the Trash
  5. Launch Word to see if the artifact still appears. If the artifact is gone you shouldn't have to do anything more except possibly reset some of your personal Preferences settings. Word will generate a new .plist file when you next Quit the program.

If that doesn't do the job, test the Normal.dotm template;

  1. Make sure you have Quit Word
  2. In the same library folder navigate to ~/Application Support/Microsoft/Office/User Templates
  3. Rename Normal.dotm to OldNormal.dotm
  4. Launch Word to see if that corrects the problem. If so you need do nothing more unless you have customization you wish to recover [Macros, AutoText, Custom Toolbars or Styles]. That can be done using Organizer, otherwise you can delete the OldNormal.dotm file. Word will generate a fresh Normal.dotm template when you quit the program.
