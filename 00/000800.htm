bob umlas, excel mvp say:
Function GetB(rg As Range) As String
      temp = rg & " "
      For i = 1 To Len(rg)
          If Mid(rg, i, 3) Like " B#" Then
              j = InStr(Mid(temp, i + 1), " ")
              GetB = Mid(rg, i + 1, j)
              Exit Function
          End If
          If Mid(rg, i, 4) = " BAF" Then
              j = InStr(Mid(temp, i + 1), " ")
              GetB = Mid(rg, i + 1, j)
              Exit Function
          End If
         If Mid(rg, i, 4) = " BAL" Then
             j = InStr(Mid(temp, i + 1), " ")
             GetB = Mid(rg, i + 1, j)
             Exit Function
         End If
      Next
  End Function

Your cells are having date also apart from time. That it why your formulas are not working. You select a cell and see the formula bar.
For cell A2, the value is 2/21/2019 9:49:44 AM

For these kind of situations, you will need to use following formula

=SUMPRODUCT((MOD(A2:A57,1)>TIME(0,0,0))*(MOD(A2:A57,1)<=TIME(12,0,0)))
