Starting with your existing 5.5 x 8.5" handouts, change the paper size to Letter and set "Multiple pages" to "2 pages per sheet." Your document will still look exactly the same, but you can print two of them on a sheet.

Sub ListClipPrim()
  Dim S As String
  Dim Length As Long, CF_Format As Long

  If OpenClipboard(0) Then
    CF_Format = EnumClipboardFormats(0&)
    Do Until CF_Format = 0
      S = String(255, Chr(0))
      Length = GetClipboardFormatName(CF_Format, S, 255)
      If Length = 0 Then
        S = "unknown"
      Else
        S = Left(S, Length)
      End If
      Debug.Print CF_Format, S
      CF_Format = EnumClipboardFormats(CF_Format)
    Loop
    CloseClipboard
  End If
End Sub

When you File > Save and Send > Publish as HTML then Publisher 2010 will produce an index.htm file (your home page) and an index_files folder that contains all your supporting grahphics and images plus any other pages in your website (.htm files). When you upload to the web host you must upload both the index.htm file and then index_files folder for the site to render correctly. When you publish your webfiles, save them to your desktop where you can easily see both the index.htm file and the index_files folder.

 

If you are creating a single page newsletter then you can also do a Save As > Single File Web Page (.mht) and this file will contain all the images in the one file, and you can upload just that one file to your webhost.

 

Alternatively if you want a single file you could also save your newsletter as a pdf file and upload it. Most people have a pdf reader installed and would be able to open and read it.

 

DavidF
