How to set a system restore point in Win 7? - Microsoft Community

lynniem say:

How to set a system restore point in Win 7?

How do I set a restore point in Windows 7?
bill smithers say:

Hi,

How to Create a System Restore Point in Windows 7 - HowToGeek

You can't create a restore point today for yesterday, last week, etc..

That is past history

You create a restore point today to use at some time in the future to restore computer settings back to today's date, if necessary.

New restore points are being created automatically/regularly and as that happens older ones are deleted to fit that space allocation.

If you find that you need earlier restore points as they are no longer there you will have to alter that space allocation

You can't alter the restore points that are available presently. What's there is there, what's gone are gone.

But remember that you use too much hard drive space if you wish to keep restore points back to the year "dot" for your machine and which you would have to keep increasing to accommodate them and it is not practical to do that as you would lose installed programs/app, Windows updates, computer settings, etc. when you do a SR onwards from that earlier restore point.

"How to Change Maximum Disk Space Used by System Protection and System Images in Windows 7"

How to Change the System Protection Disk Space Usage in Window 7

Cheers.

canadian tech say:

Restore is a valuable tool.  Most systems should be put into Safe mode before a restore.  The "security" software often prevents it from working.

In addition, there are usually not enough restore points to be useful:

The default restore point space allotment in Windows 7 is 1 to 3% of drive capacity.  Typically, this does not allow for more than a few restore points.  In almost all cases, the computer is using a very small proportion of the space available. That makes it very practical to raise that 3% to say 8 to 10%.  This strategy will yield far more restore points that will give you much more flexibility in recovering a bad situation.

To change the allotment, type create in the text box above the start globe.  Choose Create a restore point.  Click configure.  Move the slide to the right as much as you see fit.  Setting it at 50 Gigabytes will have no affect if you have a 500 Gigabyte drive and are only using about 100 or so, which is highly typical.

This may help :

You probably need to reload the camera drivers and any camera control software.

Logon as Admin.

Control Panel - Device Manager - Imaging - write down make and model of camera - double click
on it and on driver's tab get version. Now click Update Drivers (which may not do anything as MS
is far behind certifying drivers). The RIGHT CLICK on the camera - UNINSTALL - REBOOT - this
will refresh the driver stack.

Now go to System Maker's site and download the latest camera driver and other related camera
software (if no newer get the same ones).

Download - SAVE - go to them and RIGHT CLICK - RUN AS ADMIN - reboot after each driver.

cesar_e. say: Hello Paul,

Welcome to Microsoft Community Forum. I understand that you want to sort your emails by date. I will assist you on your inquiry.

Within Windows Live Mail, there is an option to sort your emails by Date, Subject, Size etc. To do this, login to your Windows Live Mail and go to your Inbox. To the right where you can see your emails, click the Sort by and choose your preferred sorting order. In this case, Date. You'll also have the option whether you want to choose Descending or Ascending to further sort your emails.
