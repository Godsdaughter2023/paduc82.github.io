sachin shetty say:

Your AOL account is configured as IMAP account in Windows 7 but on Windows XP it was POP account.

 

Difference between the IMAP and POP account.

 

IMAP account is a mirror image of what you have on the server will be displayed in Outlook. If you delete any mail from your server (AOL account) will be deleted from outlook as well. Also if you delete any mail from outlook it will be deleted from the server as well.

 

POP account will download the mails into outlook and you can store mails in both server as well as outlook and it would not make any changes if you delete mails from both that is Outlook as well as the AOL server.

Now to configure the POP account with AOL you can configure AOL support.

 

Have you upgraded from Windows XP to Windows 7? If so search for the

*.pst file from Window‚Äôs Search, If you find the .pst file take a backup of it and go to Outlook and import that data file into Outlook.

 

How to import .pst file data into Outlook

 

Use the following steps to import a .pst file into Outlook:

1. On the File menu, click Import and Export.

2. Click Import from another program or file, and then click next.

3. Click Personal Folder File .pst, and then click next.

4. Type the path and the name of the .pst file that you want to import, and then click next.  And Finish

The data that you have in the .pst file will be imported into Outlook.
