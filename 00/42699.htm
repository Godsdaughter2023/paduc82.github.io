USB headsets should have their own driver. The Realtek audio only works on the audio jack outputs on the

front and back panels of your machine.

The AMD audio driver is for when you use an HDMI or Displayport output to a monitor or TV with built in speakers.

This would also disable the Realtek driver when in use.

If you look in Control Panel > Sound I believe you will see your monitors speakers are set to default.

Having said that, your Realtek audio should still show in Device Manager if the drivers are installed, even if not

designated as the default audio in Sound.

The audio chip on motherboards have been know to fail as well.

Code et message d'erreur

Le code d'erreur 801613c9 contient le message suivant :

Il y a un problème avec le service Xbox. Veuillez réessayer plus ***.

Code d’état : 801613c9

Problème

Si ce message et ce code d'erreur s'affichent, cela signifie que le code prépayé que vous tentez d'utiliser sur Xbox Live a déjà été utilisé. Un problème lié à votre connexion à Xbox Live vous a peut-être empêché de recevoir la confirmation relative à l'utilisation initiale du code.

Solutions

Pour vérifier que votre code prépayé a été utilisé, essayez la solution suivante.

Solution 1 : vérifier votre connexion à Xbox Live et s'assurer que votre code prépayé a été utilisé

Vérifiez d'abord votre connexion à Xbox Live. Voici comment procéder :

1.Sur la manette, appuyez sur la touche Xbox Guide, accédez à Paramètres, puis sélectionnez Paramètres système.

2.Sélectionnez Paramètres réseau.
