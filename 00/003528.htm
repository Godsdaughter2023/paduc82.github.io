Hi tootdAy, 

Thanks for posting your question. This is a Windows Vista support forum.

 

If you have disabled "Always ask before opening this type of file" during file downloads.  To reset the default behavior so that explorer will prompt you to open or save the file type, perform the following steps

 

 

1. Open the Control Panel Folder Options applet.

2. Select the File Types tab.

3. Scroll down to the extension type that you want to reset.

4. Select the extension type, and then click Advanced.

5. Check the "Confirm open after download" check box, then click OK.

6. Click Close to the Folder Options dialog box.

 

If you do not get the prompt for downloads at all then try to the steps below

 

1. In Internet Explorer, click Tools, and then click Internet Options.

2. Click the Security tab.

3. There are four security zones listed:

   Internet

   Local intranet

   Trusted sites

   Restricted sites

4. Click the zone where you want to change the settings.

5. Under Security level for this zone, click Custom Level.

6. Under Downloads, under Automatic prompting for file downloads, click Enable, and then click OK two times.

 

Hope the above information helps.

    Selection.Delete Shift:=xlUp
End Sub

 


Just so you know, you do not have to keep selecting things like that. The code above could be written this way instead...

Sub Shift_Up()
  ActiveCell.Offset(-1, 0).Resize(, 3).Delete Shift:=xlUp
End Sub

By the way, your code requires you to select a cell in Column A in order to delete the values above it in Columns A, B and C. Here is code that allows you to select a cell anywhere on the row below whose Column A, B and C values you want to delete (just in case that would be of use to you)...

Sub Shift_Up()
  Intersect(ActiveCell.Offset(-1).EntireRow, Columns("A:C")).Delete Shift:=xlUp
End Sub
