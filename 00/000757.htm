sum total in column if date falls between two dates

Hello

I want to write a formula that will sum the total in one column if a date in another column falls between two dates.

Example:

Date is entered into column E, sum I want totaled is in column D

I would also want it to look at columns G and I and sum up totals in columns J & K for their totals.

      column D       column E                              column G                                    column I

So if the date falls between 10/01/2019 and 10/31/2019 in column E, I want it to sum up the totals in column D for that month.

Any help is appreciated.
Thanks

hansv mvp say:

For example:

=SUMIFS(D2:D1000, E2:E1000, ">="&DATE(2019,10,1), E2:E1000, "<="&DATE(2019,10,31))

Adjust the ranges as needed.

If you have the start and end dates in cells, e.g. V1 and V2, you can also use:

=SUMIFS(D2:D1000, E2:E1000, ">="&V1, E2:E1000, "<="&V2)

macropod say:

It's not clear what you want.

If you're trying to concatenate references inserted via References>Insert Citation or References>Bibliography, Word doesn't support the treatment of citations that way and it's not how you'd list them in a bibliography.

The only built-in reference format that equates with your (1)(2)(3)(4)(5) layout is ISO 690 Numerical, which can combine the citations as (1; 2; 3; 4; 5). The IEEE format would could generate [1] [2] [3] [4] [5] and can combine them as [1, 2, 3, 4, 5]. You can create the combined citations by selecting the first one after you've inserted it, then inserting each of the others in turn.

If you're trying to insert cross-references to citations, you'd bookmark at least the first and last ones, then use the technique in the link.
