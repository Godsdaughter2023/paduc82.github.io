UPnP - Microsoft Community

xbr dionysus say:

UPnP

Login Information

  • Gateway - 192.168.0.1
  • Username - admin
  • Password - password
UPnP
  1. Open web browser.
  2. Login to router by entering http://192.168.0.1 into the address bar
  3. The router will then prompt you to enter the user name and password. The default user name and password are: "admin", "password".
  4. From the menu on the left under the advanced section choose UPnP.
  5. Check or uncheck UPnP.
  6. Select Apply.

trekdozer say:

One of the things the OP asked for was an iso of loaded system image.

I've not tried it, but if I plug my USB flash drive in before starting 'Create a system image', its one of the listed options in the dropdown as to where I can save the iso.

But when you make a system image of Windows 7 it doesn't create an ISO file. An ISO is an image of a CD or DVD and not the same as a system image. While both use the word "image" they are very different.

To make a bootable flash drive to install Windows 7 from, which is what he wants, you need to copy the install DVD.

Copying/imaging the installed system wont work, but it would allow you to restore Windows 7 to it's current state instead of installing Windows 7.
