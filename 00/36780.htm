The Asus AC-3200 does not reset like most other routers. The reset button actually boots the router into a "Rescue" mode which is used for the "Firmware restoration utility". According to the manual this is only to be used when your device is pretty much bricked. Say a firmware update gone wrong.

janeane cru say:

Hi Seshasai,

Unmovable files refer to the data generated by Windows that cannot be automatically moved by Windows built-in tool Disk Management, such as Hibernation, Pagefile and System protection. You cannot see these files in File Explorer, because they are hidden by default. In this case, we suggest that you perform the following methods:

Method 1: Disable Hibernation.

  1. Click on Start.
  2. Type cmd.
  3. Right-click on Command Prompt, select Run as administrator.
  4. In the Command Prompt window, type powercfg.exe /hibernate off, and then press Enter.
  5. Exit Command Prompt window.

Method 2: Disable the Page File.

  1. Click on Start.
  2. Type This PC.
  3. Right-click on it and select Properties.
  4. On the left pane, click on Advanced system settings.
  5. Select the Advanced tab.
  6. Under Performance, click on the Settings button
  7. Select Advanced tab.
  8. Under Virtual memory, click on Change button.
  9. Uncheck the Automatically manage paging file size for all drives option.
  10. Click on No paging file radio button.
  11. Click on Set button.
  12. Click on Yes button to confirm.
  13. Click on OK.
  14. Restart your PC.

Method 3: Disable System Protection.

  1. Click on Start.
  2. Type Control Panel, and then press Enter.
  3. At the top-right corner of Control Panel window, set View by from Category to Small icons or Large icons.
  4. Select System.
  5. On the left pane, click on System protection link.
  6. Click on Configure button.
  7. Under Restore Settings, click on Disable system protection radio button.
  8. Click on Apply.
  9. Restart your PC.

After doing the steps above, you can now try to shrink your drive.

Note: The 3 methods above are temporary steps. After shrinking your disk drive, do the steps again and enable all the options that we have disabled.

Let us know if you need further assistance.

Hi,

It is possible that the display and camera drivers got corrputed. If reinstalling the drivers did not work, we suggest that you try to following solutions.

Roll back the drivers.

  1. Select the Start button, then in the search box next to Start, type device manager and select Device Manager from the list of results.
  2. Find the problematic device in the list.
  3. Press and hold (or right-click) the name of the device, and select Properties. In Properties, select the Driver tab, select Roll Back Driver.

Install Surface and Windows updates.

On how to run the updates, please click on this link.

Run the hardware troubleshooter.

To run the troubleshooter, follow these steps:

  1. Type Troubleshooting in the search box, click Troubleshooting.
  2. Under Hardware and Sound, choose Configure a device.
  3. The Hardware Troubleshooter will open.
  4. Click on Next to run the Hardware and Devices Troubleshooter.
  5. Choose the one you want to fix and click on Next.

If none of the solutions provided work, try to restore your device to an earlier point in time.  Restoring won’t affect your personal files, but it will remove apps, drivers, and updates installed after the restore point was made.

  1. Press and hold (or right-click) Start , and select Control Panel.
  2. In the search box in the upper-right corner, enter recovery.
  3. Select Recovery > Open System Restore > Next.
  4. Choose the restore point related to the problematic app, driver, or update and select Next > Finish.

Reset your Surface if restoring did not work. Click on this article for more information on how you can reset your Surface.
