The only reason to replace or restore the ntoskrnl.exe log in Windows XP is if your system is not booting to Windows XP properly and if you're getting an error report saying that the file is missing or corrupt. If this is the case, you will need to boot from the Windows XP installation CD to repair and replace the file. This task is something that anyone can do, even those with little to no computer experience.


 

 

  • Insert the Windows XP installation CD into the CD-ROM drive of your computer and then restart your computer.
