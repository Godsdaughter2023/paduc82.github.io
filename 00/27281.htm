mike h.. say:

Hi,

 

Yes you can do that and you should be able to adapty this.

 

It looks at B1 in the enquiries sheet and if it contains a 1 it puuls in the number with the E replaced with a Q

 

=IF(Enquiries!B1=1,SUBSTITUTE(Enquiries!A1,"E","Q"),"")

 

Note the underlined "" bit at the end of the formula. This is what happens if there is a 1 in b1 and you can put anyting within those quotes. For example. Not quoted

 

=IF(Enquiries!B1=1,SUBSTITUTE(Enquiries!A1,"E","Q"),"Not Quoted")

 

or even return the original number with the E prefix

 

=IF(Enquiries!B1=1,SUBSTITUTE(Enquiries!A1,"E","Q"),Enquiries!A1)

rik_uk say:

OK this might help...

You will need to amend certain items to suit, as there are a number of assumptions I have made.

1. The websource data sheet has a header row so the data starts in row 2, column A.
2. The 'formatted report' template form spreadsheet is in the same workbook.
3. The new sheets per row are kept in the same workbook.
4. Column A holds a unique record number or unique ID reference code

Copy the following code into a standard module...

Option Explicit
Option Base 1

Sub XferRowsToForm()
Dim varDataRow As Variant
Dim lngLastRow As Long, lngLastCol As Long
Dim arrData() As Variant, rngData As Range
Dim lngRowCnt As Integer, lngRowStart As Integer

lngRowStart = 2          ' Change this if the data starts on row 1 or any other row...
Application.ScreenUpdating = False
    With Sheets("Data")     ' Change the name here to match the tab name of the websource sheet
        lngLastRow = .Cells(.Rows.Count, "A").End(xlUp).Row
        lngLastCol = .Cells(1, .Columns.Count).End(xlToLeft).Column
        Set rngData = .Range(.Cells(lngRowStart, 1), .Cells(lngLastRow, lngLastCol))
        ReDim arrData(lngRowStart To lngLastRow, 1 To lngLastCol)
        arrData = (rngData)
    End With
    For lngRowCnt = 1 To lngLastRow - lngRowStart + 1
        Sheets("Template Form").Copy After:=Sheets(Sheets.Count)   ' Change the name to the tab name of the Template Form sheet
        ActiveSheet.Name = arrData(lngRowCnt, 1)
        Range("A1") = arrData(lngRowCnt,1)  ' Change cell reference to first cell in form, 1 in last part represents 1st column
        Range("B2") = arrData(lngRowCnt,2)  ' Change cell reference to second cell in form, 2 in last part represents 2nd column
        Range("C3") = arrData(lngRowCnt,3)  ' Change cell reference to third cell in form, 3 in last part represents 3rd column etc......
        'Range("Cell in form") = arrData(lngRowCnt,Column in data)
        '............
        '............
    Next
Application.ScreenUpdating = True
End Sub

Thank you for the help!

- Ryan

stefan blom say:

Clearing "Automatically resize to fit contents" prevents Word from changing the widths of cells. To prevent cells (rows) from expanding vertically, you have to set a fixed height for table rows. Select the relevant rows and, on the Table Tools Layout tab, click Properties. On the Row tab, select "Specify height" and then choose "Exactly" for "Row height is." Specify the desired amount.
