cmdrkeene say:

A reinstall doesn't have to affect the D:\ partition, just don't modify it during the install process. Although you know an external copy or backup image is never a bad idea.

A refresh is an operation that makes a temporary copy of your user profile folders (files, settings, preferences, data), then erases the windows drive and restores a backup image (factory image). After Windows is replaced with the stock version, the user profiles are copied back into place.  Since Windows Store ('app store') apps are stored inside a profile, those are preserved too, so following a refresh, the only things you need to replace are desktop programs like Office (and even then, only if it wasn't included with your PC).  You can even make your own refresh points after installing all your favorite desktop software, so that after a refresh it's already present by default.
