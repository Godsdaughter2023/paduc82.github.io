dale howard [mvp] say:

kkc153 --

Microsoft Project keeps track of both the date AND the time each task starts and ends.  If you have a task with a Duration of 1.5 days, and the task starts at 8:00 AM on the first day, the task will finish at 12:00 Noon on the second day.  That means a successor task would start at 1:00 PM on the second day.  If you click File > Options > General, you can set the Date Format value to display both the date AND time in all date fields, such as Start and Finish.  If you look at the task in question, do you see that the Finish time is earlier than 5:00 PM?  If so, that would explain why you are seeing what you currently see.  Please let us know and we will try to help you.
