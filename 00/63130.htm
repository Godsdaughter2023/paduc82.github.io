andre da costa say:

No, Windows 8 OEM is not transferable to a new motherboard.

If Windows 8 came preinstalled on your computer, it is an OEM license.

OEM versions of Windows 8 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

------------------------------------

If you would like to be able to transfer to a new motherboard, then you will need a full version System Builder license.

Full version Windows 8

There are no full retail versions of Windows 8. If you need a full version, you will need to purchase the Windows 8 Pro OEM from New Egg. This requires a custom clean install.
