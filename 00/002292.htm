albert d. kallal say:

SquareSpace to my knowledge is a “store front” type of web service.

 

Access is like word, or power-point – a desktop application that has rather “little” to do with that type of web site.

 

SquareSpace certainly has some interfaces that allow you to “pull” tables of data that could THEN be imported to an Access table. This local table of course then can be used by Access, and searched, or reported on.

 

However such a process would be an “import” type of process, and Access would not have a “live” and “interactive” connection to some web site.

 

So Access is not a web tool, but really a tool that lets you edit and work with tables of data.

 

So most if not all of the data you have in SquareSpace can be downloaded (the data format is json).

 

So Access can’t work directly with SqureSpace anymore than it can say with eBay or Amazon. However, such web sites do support “web services” in which data can be pulled down to your desktop for use in say Excel, or in this case Access.

 

You have to check the SquareSpace forums to see what kind of “export” options exist for SquareSpace, but such a process has VERY little to do with Access.

 

So you could download some data (check out their forums for options in this regards). You could then import such data into Access.

 

However, having Access “interact” directly with the web site is not really what Access is designed for, and is primarily a desktop tool for working with tables of data on your pc – not data from a web site. Some push and pull of data from that web site is likely possible, but it will require web development skills, and Access is not really the appropriate tool for such a setup.

How do I format the font in an IF statement?  For example, I have a column of numbers in Column A and another column of numbers in Column B.  In Column C, I have written an IF statement that compares the two columns.  If Column B  is greater than Column A, I want to have "NO" show up in Column C in red and boldface.  If Column A is greater than Column B, I want "YES" to show up in Column C in green and boldface.

Thanks!

hansv mvp say:

You cannot do that in a formula, but you can create conditional formatting rules for this purpose.

  • Select the cells with the formulas.
  • On the Home tab of the ribbon, click Conditional Formatting > Highlight Cells Rules >Equal To...
  • Enter the value NO.
  • Select Custom Format... from the dropdown.
  • If necessary, activate the Font tab of the Format Cells dialog.
  • Select Bold in the Font style list, and select Red from the color dropdown.
  • Click OK, then OK again.
  • Repeat the above steps, but with YES as value and Green as color.
