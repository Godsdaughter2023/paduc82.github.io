Hi,

 

I have assumed this data are in columns A thru F and column G is where you want to return the previous job. Put this in G2 and drag down as required.

 

=IFERROR(LOOKUP(2,1/($E$1:E1=E2),$A$1:A1),"")

mike h.. say:

Hi,

 

Let's examine the formula without the iferror bit which look like this when dragged down to G6 working on the data below

 

=LOOKUP(2,1/($E$1:E5=E6),$A$1:A5)

 

First we look at the underlined bit which returns this. The TRUE and FALSE refer to whether the formula found the string In E6 in any of the cells E1:E5 and as you can see it finds it twice in E2 and again in E4

 

=LOOKUP(2,1/({FALSE;TRUE;FALSE;TRUE;FALSE}),$A$1:A5)

 

Now in Excel TRUE & FALSE can be coerced into 1 and zero by any mathmatical operation so in the next part of the formula we divide 1/ TRUE or FALSE to get this. As you can see we now have errors and 1s becuase 1/0 gives a #DIV/0 error and 1/1 =1. We can be certain that this array will always be either 1 or the error, no other result is possible.

 

=LOOKUP(2,{#DIV/0!;1;#DIV/0!;1;#DIV/0!},$A$1:A5)

 

The lookup now tries to lookup 2 in that array (2 is the lookup value) and we know it will never find it and we use this to our advantage because if LOOKUP can't find the value it will return the last value that is less than the lookup value so the formula finds the last 1 in E4 and returns the corresponding value from column A.

 

Does that help?
