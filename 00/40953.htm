To resolve this problem, remove the download directory, and then visit the update locations again. To do this, follow these steps:
  1. Stop the Automatic Updates service. To do this, follow these steps:
    1. Click Start, click Run, type services.msc, and then click OK.
    2. Right-click the Automatic Updates service, and then click Stop.
    3. Minimize the Services snap-in.
  2. Rename the SoftwareDistribution directory. To do this, follow these steps:
    1. Click Start, click Run, type cmd, and then click OK.
    2. Type cd %windir%, and then press ENTER.
    3. Type ren SoftwareDistribution SDTemp, and then press ENTER
    4. Type exit, and then press ENTER .


    Note When you remove the %windir%\softwaredistribution folder, the history list is erased from the "Review your Updates" section of the Windows Update Web site. This action will not affect the currently installed updates on your computer. Subsequent updates will appear in the history list.
  3. Start the Automatic Updates service. To do this, follow these steps:
    1. Maximize the Services snap-in.
    2. Right-click the Automatic Updates service, and then click Start.
    3. Close the Services snap-in window.
  4. Visit the Windows Update Web site, the Microsoft Update Web site, or the WSUS server again, and then download an update.
  5. If you still receive the same errors, verify that you have correctly performed steps 1 through 3.

¡firedog say:

Windows Live Mail Error ID: 0x8007007A

The error code indicates that the Microsoft Account you're signed in with (or at least the OneDrive bit of it) has been migrated to the new platform, which means that the photo email feature you've been using no longer works.

You'll either have to send the pictures as attachments in the traditional way, in which case they will be at full size, or embed them in the message body where you can resize them and apply other effects. You'll find instructions - and a solution for cases where you want to send a lot of pictures - in this article:

You must remove unsent messages from the outbox (below the account folders in the folder pane) before trying to send again.

NB: The error in this case is caused by a change at OneDrive, which means that it no longer carries out the photo email function it used to. It has nothing to do with the mail account being used to send messages, so removing and re-configuring a mail account will have no effect. Nor does it have anything to do with the Windows Essentials program files, so repairing the installation won't help, either (but luckily it won't hurt anything). It won't help to try a different Windows user account.

3.       Now you should have a window called “Windows Task Manager”.

4.       From there click “File” in the menu bar, and then click “New Task (Run…)”

5.       In the next window that appears type “explorer” where is says “Open:”, then press OK.

1.     Restart your computer if it is powered on.

3.     Once you see the Advanced Boot Options menu you can stop tapping.

Hi,

This is a default message that doesn't mean what it says most of the time. For 99% of the time, this happens because the lobby is full. Since FSX will drop you in the Free Flight lobby by default, you might have trouble logging in at busy times of the day. But there is a way to overcome this issue: 

When you're logged in, select another lobby than the Free Flight lobby and then disconnect. Next time you login, FSX will drop you in that lobby and not in the Free Flight lobby. Since there aren't that many players in the other lobbies, it is unlikely that GameSpy will disconnect you. Make sure you logout in a non-Free Flight lobby. Remember that this is another way and is only valid if you don't care about the Free Flight lobby, because when you select the Free Flight lobby after you logged in via another lobby, you will be disconnected anyway if the Free Flight lobby is full.
