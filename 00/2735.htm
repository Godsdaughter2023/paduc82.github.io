Hi smcquality

In order to achieve this, you'll need to insert a "section break".

Position the insertion point (cursor) at the end of page 2. From the Page Layout tab, Page Set up group, click on the "Breaks" button. choose "Next Page" from the "Section Breaks".

cimjet say: Hi
Here is the macro, if you have problems, I will send you a link with the modification but before lets give it a try.
Right click on your sheet tab at the bottom and select >View Code<

VBA editor will open and you should see the macro. you will need to delete the old one and paste the new one instead.

Private Sub Worksheet_Change(ByVal Target As Range)
Set sh2 = Sheets("Sheet2")
On Error Resume Next
ltrw = sh2.Cells(Rows.Count, "A").End(xlUp).Row + 1
    If Target.Column <> Columns(5).Column Then Exit Sub
If Target.Value = "Yes" Then
  sh2.Cells(ltrw, 1) = Cells(Target.Row, 1).Value And Clear
  Cells(Target.Row, 1).Clear
    sh2.Cells(ltrw, 2) = Cells(Target.Row, 2).Value
    Cells(Target.Row, 2).Clear
        sh2.Cells(ltrw, 3) = Cells(Target.Row, 3).Value
        Cells(Target.Row, 3).Clear
            sh2.Cells(ltrw, 4) = Cells(Target.Row, 4).Value
            Cells(Target.Row, 4).Clear
  End If
End Sub
