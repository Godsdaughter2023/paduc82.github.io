This sort of problem often occurs with Yahoo accounts if there's a dodgy message on the server that Windows Live Mail doesn't want to download. Log in to the mailbox in webmail and start by emptying the spam and trash folders. Then move all of the messages out of the inbox and into one or more user-created folders. Conduct a send/receive in Windows Live Mail - is there any error message now? Do you see a No new email messages status message?

Once you have established a working connection, move half of the messages back into the inbox and repeat the send/receive. Repeat, halving the number of messages each time until the fault arises again. You should eventually be able to identify any dodgy one and deal with it appropriately.

To perform Automatic repair on Windows 8 computer follow these steps:

  1. Insert the media such as (USB or DVD) and restart your computer.

  2. Press F12 key (typically this is F12, but it can differ between computer manufacturers) and choose the drive that you inserted the installation media into.

  3. Once the Windows Setup window appears, Click Next and select Repair your computer.

  4. You will then see a blue screen and an option to choose. Choose the option Troubleshoot and select Advanced options.

  5. You may choose Automatic Repair from Advanced boot option and follow the onscreen instructions

Method 2:

If the issue persist, you may try performing a check disk booting from the DVD by following above steps.

  1. From Repair your computer choose the option Troubleshoot and select Advanced options.

  2. You may choose Command prompt.

  3. At a command prompt, type the following command, and then press ENTER
    chkdsk /f /r X:    ( drive letter) (for ex. C or D )

  4. Press Y when you are prompted to check the disk the next time that the system restarts.

  5. Restart the computer.

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.
