Options for freeing up space:
Windows over time will accumulate data outside of the normal types of data we store. This can include things like copies of Windows Updates, memory dump files used to diagnose and troubleshoot error messages, temporary internet files for web pages we often visit and Windows Upgrade files, commonly used to assist with installing updates for Windows. Sometimes we don't need this type of information and removing it can free up precious disk space inaddition to improving overall system performace.

Sub Indexing()

'### by Rick Rothstein ###

  Dim X As Long, Z As Long, LastCol As Long, LastRow As Long

  Dim OutStr As String, Arr() As String

  Const StartCol As Long = 2

  Const StartRow As Long = 2

  Const LastUsedCol As Long = 6

  LastRow = Cells.Find(What:="*", SearchOrder:=xlRows, _

            SearchDirection:=xlPrevious, LookIn:=xlValues).Row

  'LastUsedCol = Cells.Find(What:="*", SearchOrder:=xlByColumns, _

                SearchDirection:=xlPrevious, LookIn:=xlValues).Column

  ReDim Arr(1 To LastUsedCol)

  For X = StartRow To LastRow

    LastCol = Cells(X, Columns.Count).End(xlToLeft).Column

    For Z = StartCol To LastUsedCol

      If Z > LastCol Then

        Arr(Z) = ""

      ElseIf Len(Cells(X, Z).Value) Then

        Arr(Z) = Val(Arr(Z)) + 1

      Else

        Arr(Z) = Val(Arr(Z))

      End If

      Cells(X, StartCol).Offset(, -1).Value = Replace(Trim(Join(Arr)), " ", " .")

    Next

  Next

End Sub

steve rindsberg say:

What you're seeing may just be PowerPoint doing what it's supposed to do.

For example, in normal view, it won't scroll side to side at all unless there's content (shapes/whatever) sitting off the slide to the left or right.  Once there's content off-slide, a scrollbar will appear at the bottom of the window; you can use that to scroll left/right then.  Your mouse might then work for this; I don't know;  don't have a mouse that includes left/right scrolling.

Vertical scrolling in PPT is different from Word/etc too.

If there's no content off the slide above or below, turning the scroll wheel on the mouse will take you from one slide to the next.  If there's content, say, below the slide in the work area, scrolling will shift the view down to make the off-slide content visible.  Scrolling a bit more will take you to the next slide.  

It's a bit startling.

But if that's what you're seeing, it's not the mouse, it's just PowerPoint being PowerPoint.

krishy77 say:

Hi,

Suggestions given did not solve the problem and as mentioned i had already tried most options before posting the query. That said, after quite of searching and trial and error, finally found the solution that worked for the issue i was facing.

Setting the following registry keys solved the issue in my case

[HKEY_CURRENT_USER\Software\Policies\Microsoft\Office\16.0\Outlook\security]

"ShowOLEPackageObj"=dword:00000001

"AdminSecurityMode"=dword:00000003

"ShowLevel1Attach"=dword:00000001

Posting it here so that it can help anybody else who is facing a similar problem.
