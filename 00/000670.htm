beth melton say:

If you are opening the Paragraph dialog box, modifying the spacing, and clicking Set as Default, and confirming the change, then these are the correct steps to take. You indicated the change isn't sticking after restarting Word which indicates changes to the Normal template are being discarded when you exit Word. The most common cause of this error is the Adobe PDFMaker add-in. In order to rectify another issue, a prompt to save changes to your Normal template each time you exit Word, they opted to discard all of your modifications to make sure you don't encounter the prompt. The best way to rectify the issue is to disable the add-in. To do so:

1.  Click the Office Button

2.  Click Word Options (below the recently used file list)

3.  Click Add-ins

4.  In the list of Add-ins, determine if the PDFMaker add-in is active. If so, select it, and not the file path you see below the list.

5.  Locate the file using the path provided in the Windows Explorer

6.  Exit Word

7.  Move the PDFMaker file out of the folder to prevent it from loading when Word starts

I think I heard Adobe has an updated add-in that corrects this issue so if the built-in PDF capability doesn't work for you (Office Button/Save As) and you need the PDF add-in then you might check the Adobe site for an update.

ruel mayoya msft say:

Hi That’One’Guy,

 

It seems that the cause of the issue is related to your network connection. Outlook seems to detect different connections between the dock (LAN/cable connection) and Wifi whenever you undock/dock the laptop.

If you undock or disconnect your computer from the network, Outlook will not sense the disconnection and switch to offline mode. This will cause errors because Outlook thinks it's still online. However, if you remove the network PC card while Outlook is running, it will switch to Work Offline automatically. Check if you connect back to LAN connection or dock the laptop, if it will return you to Work Online.

 

Try manually switching from Online to Offline mode in Outlook (Alt + S + W), before attempting to undock the laptop. Now after Wifi is connected, click the Work Offline button and check if there is any improvement.

 

To narrow down the issue, please follow the steps below and provide us with the following information:

1.) Check if there is any Antivirus that blocks or controls the network. Temporarily disable AV then check if the Work Offline button works.

2.) Run Outlook in safe mode. To do this, go to Run (Win + R) > type Outlook.exe /safe.

3.) Flush the DNS cache on the laptop. To do this, go to command prompt (Win + R) > type cmd > then type ipconfig /flushdns.

3. Under Managing your account, click Account details (password, aliases, time zone).

4. Retype your password if you're asked to sign in again.

5. At the Account overview page, click the “Change” link beside the Country/region field. You should be redirected to the Registered Information page.

6. Under Home Location, choose your correct country and time zone.

7. Click Save.
