superfranky say:

Hi Jan,

Please open Outlook in Safe Mode and check how it works.
To do that, press the ‘Windows + R’ key on the keyboard and type in ‘Outlook /safe’ and hit OK in the Run open box.
Note: There is a space between Outlook and /.

If application safe mode works, try disabling the add-ins associated with Outlook and check.

Here are the steps:
1. Open Outlook.
2. Click on File and then click on options.
3. In the Options Window click on Add-ins.
4. Select Com Add-ins in the Manage list in the Add-ins section and click Go. Now uncheck the all the add-ins and click OK.

If disabling the add-ins do not resolve your issue, please create a new profile to check if the issue persists.

Here are the steps:
1. Open Control Panel.
2. Click the Mail icon.
3. Click Show Profiles.
4. Click Add.
5. In the Profile Name box, type a new name for the profile.
6. Specify the user name, the primary SMPT address, and the password, and then click Next to verify the account settings, click Finish to save it.
7. Under Always use this profile, select the profile name that you used in step 5.
8. Click OK.

If the issue persists after doing the steps above, try run the /resetnavpane command.
To do that, close Outlook and any other Office applications, press the ‘Windows + R’ key on the keyboard, type in ‘Outlook.exe /resetnavpane’ and hit OK in the Run open box.

Feel free to let me know if there is any update.

Thanks,
Franky

arunrajsekar say: Sub Insert_Tab_For_Selected_Cell()
    Dim oSh As Shape
    Dim oTbl As Table
    Dim lRow As Long
    Dim n As String
    Dim lCol As Long
    Set oSh = ActiveWindow.Selection.ShapeRange(1)
    Set oTbl = oSh.Table

    With oTbl
    For lRow = 1 To .Rows.Count
    For lCol = 1 To .Columns.Count
        If .Cell(lRow, lCol).Selected Then
        n = .Cell(lRow, lCol).Shape.TextFrame.TextRange
        With .Cell(lRow, lCol).Shape.TextFrame
        .TextRange = vbTab & n
        End With
        End If
    Next
    Next
    End With
End Sub
