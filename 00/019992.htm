How to use or create style to highlight text in Word 2016? - Microsoft Community

os11 say:

How to use or create style to highlight text in Word 2016?

How to use or create style to highlight text in Word 2016? The color code should include 2 different colors.

jay freedman say:

You cannot include actual highlighting -- as applied by the button on the Home tab -- in any style. However, you can include shading in either a paragraph style (so the entire paragraph appears to be highlighted) or in a character style (which can be applied to as little as a single character).

To do that, create or modify a style, assigning it a name and a style type. In the dialog, click the Format button in the bottom left corner and choose Border in the Menu. In the next dialog, click the Shading tab, select a Fill color, and click OK.

Any given style can have at most one shading color. If you want two different colors in the same paragraph, define two character styles and put one color in each of them. Each style will have to be applied separately to the text that needs its color.

    Dim ILS As InlineShape
    Dim Shp As Shape
   
    For Each ILS In ActiveDocument.InlineShapes
        If ILS.HasChart Then
            With ILS.LinkFormat
                If .Type = wdLinkTypeChart Then
                    .Update
                    .BreakLink
                End If
            End With
        End If
    Next
   
    For Each Shp In ActiveDocument.Shapes
        If Shp.HasChart Then
            With Shp.LinkFormat
                If .Type = wdLinkTypeChart Then
                    .Update
                    .BreakLink
                End If
            End With
        End If
    Next
End Sub

As you can see, charts could have their text wrapping set to In Line With Text (making them InlineShapes) or any of the "floating" values (making them Shapes), and these have to be handled separately. In either case, each chart that has a link will be updated and the link will then be broken, converting the chart to an embedded one.

In a very simple test, with Excel not running when the macro starts, it took several seconds for each chart to be updated. If you're absolutely sure all the charts are already up to date, you could comment out the two .Update lines to speed up the process.
