jay freedman say:
Frankly, I'm not surprised that the Trust Center settings don't behave the same in PowerPoint and Word. I wouldn't have even thought of using that mechanism.

Install the following macro in a module in your Normal.dotm template (see http://www.gmayor.com/installing_macro.htm if needed). The effect is that any time you open a document that would be in compatibility mode (both .doc and any .docx that was last edited in any earlier version), it will be converted to Word 2013 -- the same as clicking File > Convert. If you make any change in the document and save, it will be saved as a .docx.

Sub AutoOpen()
    With ActiveDocument
        If .CompatibilityMode < Application.Version Then .Convert
    End With
End Sub

Of course, saving it as a .docx doesn't remove or overwrite the original .doc file.

Old-format documents are going to be opened in compatibility mode. doug robbins - mvp office apps & services (word) say:

The following modified version of Jay's macro will ask if you want to convert the document.


Sub AutoOpen()
     With ActiveDocument
         If .CompatibilityMode < Application.Version Then
            If MsgBox("The document was created in an earlier version of Word. Do you want to convert it to the latest version?", vbQuestion + vbYesNo) = vbYes Then
                .Convert
            End If
        End If
     End With
End Sub


The conversion should not really affect the layout of the document.  More likely differences between the printers\printer drivers that are active between the machine on which the document was created and that on which it is being opened.
