Hi,

Sorry for the delay.

Its a very common problem among Windows users. Whenever Windows starts, it automatically turns "CapsLock", "NumLock" or "ScrollLock" keys On at login screen even if you turned the keys off before the login or boot screen.

 

If you want to permanently change their behavior, then you can use following steps to do so:

 

1. Type regedit in RUN dialog box and press Enter. It'll open Registry Editor.

2. Now go to following key:

 

To change the behavior for currently logged in user:


HKEY_CURRENT_USER\Control Panel\Keyboard

 

To change the behavior for all users:

HKEY_USERS\.DEFAULT\Control Panel\Keyboard

3. Now in right-side pane, change value of "InitialKeyboardIndicators" string value to any of following:

0- Turn all indicators Off (NumLock, CapsLock, ScrollLock)
1- Turn CapsLock On
2- Turn NumLock On
3- Turn CapsLock and NumLock On
4- Turn ScrollLock On
5- Turn CapsLock and ScrollLock On
6- Turn NumLock and ScrollLock On
7 - Turn all indicators On (NumLock, CapsLock, ScrollLock)

4. That's it. Exit registry editor and restart your system to take effect.

Dim iFilecount, aHandles, iIndex

if oFSO.FileExists(sUnlock) then oFSO.DeleteFile(sUnlock)

 

WScript.Echo Now(), oWshShell.ExpandEnvironmentStrings("script started. User=%UserName%")

 

Do

            LockFiles         'Lock files then wait until the unlock file appears

            UnlockFiles     'Unlock files then wait until the unlock file disapears

Loop

 

'-----------------------------------------

'Count the number of files, then lock them

'-----------------------------------------

Sub LockFiles

            iFilecount = 0

            For Each sFolder In aFolders

                        CountFiles(sFolder)                 'Count the files to be locked

            Next

 

            ReDim aHandles(iFilecount)

            iIndex = 0

            For Each sFolder In aFolders

                        Process sFolder, iIndex           'Lock all files in the nominated folder

            Next

            Sleep(True)      'Sleep until the unlock file appears

End Sub

'Recursively lock all files in the nominated folder

'--------------------------------------------------

Sub Process(sFldr, j)

            On Error Resume Next

            For Each oFile In oFSO.GetFolder(sFldr).Files

                        Set aHandles(j) = oFSO.OpenTextFile(oFile.Path, 1)  'Lock all files

                        If Err.number = 0 _

                        Then j = j + 1 _

                        Else WScript.Echo "Cannot lock """ & oFile.Path & """"

            Next

            On Error Goto 0

           

            For Each oSubFldr In oFSO.GetFolder(sFldr).Subfolders

                        Process oSubFldr.Path, j         'Process subfolders

            Next

End Sub

'-----------------------

'Unlock all locked files

'-----------------------

Sub UnlockFiles

            For Each oFile In aHandles

                        if not IsEmpty(oFile) then oFile.close  'Unlock all files

            Next

            Sleep False      'Sleep for 15 minutes or until the unlock file disappears

End Sub

'Count the files in the nominated folder

'---------------------------------------

Sub CountFiles(sName)

            If Not oFSO.FolderExists("C:\FileLocker") then oFSO.CreateFolder("C:\FileLocker")

            sTempfile = oWshShell.ExpandEnvironmentStrings("%temp%\Filecount.txt")

            oWshShell.Run "cmd /c dir """ & sName & """ /s /a-d | find /i ""file(s)"">""" & sTempfile & """", 0, True

            aLines = Split(oFSO.OpenTextFile(sTempfile, 1).ReadAll, VbCrLf)

            iFilecount = iFilecount + Split(LTrim(aLines(UBound(alines)-1)))(0)

End Sub

'Pause until the unlock file appears or disappears

'-------------------------------------------------

Sub Sleep(Blocked)

            tTime = Now()

            Do

                        If oFSO.FileExists(sUnlock) And DateDiff("s", tTime, Now()) > iGrace _

                        If oFSO.FileExists(sUnlock) = Blocked Then

                                    WScript.Echo Now(), "script changed mode"

                                    Exit Sub

                        End If

            Loop

End Sub

patrat46 say:

Step 1: LOGING IN

By default the IP address should be set to 192.168.1.1 Type that in to a web browser and go.

You should see a box prompting you for your username and password. Enter your username and password now. By default both the username and password are admin.

Step 2: SETTINGS

On left Select Advanced (Site appearance may change)

Under thhe advance section select UPnP

Check the box to Enable UPnP

(reboot router and check nat status on xbox if not open continue)

[View:http://screenshots.portforward.com/routers/Huawei/EchoLife_HG532/UPnP.jpg]

On left under DMZ Select NAT

ON top Select DMZ

Add the Xboxs IP to the Host address line

Check Enable.

Press submit

(reboot router and check nat status on xbox if not open continue)

[View:http://screenshots.portforward.com/routers/Huawei/HG532b_-_B013/NAT_DMZ.jpg]

meghmala say:

Hi Maximusflavius2,

 

I would suggest you to remove all the folders included and add them back in the library of media center one at a time again. Removing from the library does not remove the folders from the original location. This procedure will rebuild your library. You may use the steps below and check if these help –

 

a. Click the Start button, click  All Programs, and then click  Windows Media Center.

b. On the Windows Media Center start screen, scroll to  Tasks, and then click  Settings.

c. Click Media Libraries.

d. Click the type of media that you want to remove (Recorded TV), and then click next.

e. Click Remove folders to the library, and then click  next.

f. Depending on where your media files are stored, click one of the options under where are the folders you want to add? And then click  next.

g. Navigate to the folders that you want to add by clicking the Plus‌ or minus buttons, and then select the check box next to each folder. (Files in subfolders will be included in your selection).

h. When you've selected all the folders that you want Media Center to monitor, click next, click yes, use these locations, and then click  Finish.


Now use the same procedure as above provided to add the media files in the library. You may refer to the below link for further assistance on the issue -

http://windows.microsoft.com/en-US/windows7/Add-media-files-to-Windows-Media-Center

http://windows.microsoft.com/en-US/windows7/Shared-libraries-in-Windows-Media-Center

 

Hope this helps.
