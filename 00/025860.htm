You could try this approach to fix files with DISM at an Administrator Command prompt:

dism /Online /Cleanup-Image /RestoreHealth

If it does not comlete successfully, you can point it to Windows 10 Install media which you can create here:

http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install

dism /Online /Cleanup-Image /RestoreHealth /Source:Wim:E:\Sources\install.wim:1

Replace E:\ in the line above with whatever drive letter you have Windows 10 media mounted to.

* do not have roaming rights

PKC:
- Can activate on 1 system only
- transfer >> only once to another system within 90 day only (eg, failed system board)
- downgrade to prior version – may require assistance from Microsoft to do so
