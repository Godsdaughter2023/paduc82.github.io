Q. If my customer asks me to upgrade a PC with new hardware components, when is a new operating system needed? When would the PC be considered "new"?

A. Generally, an end user can upgrade or replace all of the hardware components on a computer—except the motherboard—and still retain the licence for the original Microsoft OEM operating system software. If the motherboard is upgraded or replaced for reasons other than a defect, then a new computer has been created. If the motherboard is replaced because it is defective, you do not need to acquire a new operating system license for the PC as long as the replacement motherboard is the same make/model or the same manufacturer's replacement/equivalent, as defined by the manufacturer's warranty.

You can not move a hard drive with Windows 8 on it from a machine and put it into another machine that previously ran Windows 7 and expect it to work.

It won't.

@@@@@@@@@@@@@@@@@@@@@@@@@@@

And even if it physically would work, legally you can't transfer the Windows 8 License from the MSI laptop to the Acer laptop in your circumstances:

Read this information:

If the Laptop came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

And Retail Licenses can also only be installed on one machine at any particular time.

018D5C66-4533-4307-9B53-224DE2ED1FE6

Change the DWORD value from 1 to 0.

Close the Registry to complete the task.

Restart your computer and check the file explorer for the folder.

tluscher say:

I have found a solution, which disappointingly doesn't include a Microsoft solution.

  • Download DriveClone (Free version for home use) from Farstone.
  • Create a clone of your Windows 8.1 Pro system drive.
  • Download the UEFI boot enabler ISO from Farstone (See DriveClone user guide pdf for URL).
  • Burn a DVD using the aforementioned boot enabler ISO.
  • Boot your pc/laptop into the boot enabler DVD and set the cloned hard disk to be UEFI bootable.
  • END

Note that whilst DriveClone works it took about 8 hours to finish the clone process!!!! Not ideal, but it works. I did try Acronis True Image 2014, but it doesn't support Seagate Hybrid Hard Discs.
