Hello,

It sounds like the computer is booting into safe mode. When a computer boots into safe mode automatically, there is usually a problem preventing normal startup or there is a specific setting enabled.

Check this website to see if the computer is set to boot into safe mode on every startup.

http://www.howtogeek.com/howto/windows-vista/force-windows-to-boot-into-safe-mode-without-using-the-f8-key/

Safe mode starts the computer with only the minimum necessary drivers and programs so if there is a program preventing startup, you can resolve the issue by using safe mode. You cannot use any network connections in safe mode unless you select safe mode with networking. This is likely the reason you cannot connect to any networks.

If there is an issue preventing startup, you can try running startup repair, system restore, or back up the documents and re-install the operating system. If you re-install the OS, you will use ALL programs installed on the computer, so unless you don't care about the programs, you may not prefer that option. You can also boot from the windows 7 installation disc and run the system file checker.

I would definitely recommend backing up files regularly in case of any hardware or software failures.
