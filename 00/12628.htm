Solution:

I had the same issue with PowerPoint 2013, and I think the culprit is MS Upload Center which is used to sync the offline files in the background. So this is what I did:

  1.    Go to Task Manager by doing taskmgr in Windows icon -> search box
  2.    In "Process" tab, look for MSOSYNC and kill the process
  3.    In PowerPoint, go to "File" and select "Options". Under "Save", and in "Save checked-out files to", tick the box "The server drafts location on this computer" instead of "The Office Document Cache". Then OK.
  4.    Restart PowerPoint and you can save it without that intermittent read-only message
  5.    To ensure that the MSOSYNC is not started automatically on reboot, then go to "Services" by typing services.msc in Windows icon -> search box. And in that window, locate "Offline Files" and right click, and select "Properties". From there, select change "Start-up type" to "Manual"

Hope this helps everyone.

steve house say: Welcome to Microsoft's brave new world of software licensing.  They made a huge change in the terms of the non-subscription licenses with Office 2013, though it actually started with the key-card licenses for pre-installed OEM versions of Office 2010.  When they say you can download Office again, strictly speaking that is true.  You can download and install it as many times as you need BUT a given key can only be used on one and only one PC.  Once a product key has been activated on a particular PC it is forever locked to that one specific physical computer and cannot be transferred to another one.  You can reinstall it on the same computer without a problem should you need to - if for example you've reformatted the drive - but you cannot move it to another computer even if it has been completely removed from the first one.  If you replace the computer you first installed it on with a new one because of system failure, theft, destruction of the computer, hardware upgrading, etc and want to continue to use Offce you must buy a fresh new full-price copy with a new unused product key.  The other alternative is to rent your software by purchasing an Office 365 subscription as they can be moved from device to device as needed.

IMHO it's rather like being forced to buy a separate copy of a book for each chair in your house that you might happen to want to sit in while you read.
