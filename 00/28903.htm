Hi,

As for how to have the heading that appears in the brief bolded without pulling the bold attribute into the TOC:

The trick is to apply bolding within the heading style itself. That is, modify the heading style and apply the bold attribute in the style, rather than using direct formatting to apply bold to the heading text as you type it. As long as the corresponding TOC style (TOC 2 for a Heading 2 style, etc.) does not incorporate the bold attribute, the heading should appear in regular type (not bolded) in the generated TOC.

I learned this trick from the venerable Word MVP Suzanne Barnhill (on a similar issue, see this somewhat dated but still relevant exchange: http://www.tech-archive.net/Archive/Word/micros...).
