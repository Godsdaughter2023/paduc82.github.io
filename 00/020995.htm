jllatham say:

It may be that you did not install Excel when you installed the package - you probably did, but I mention this because it is possible to install any version of Office without all of the pieces.

First thing to try:  Use the Start button and choose All Programs scroll down to the Microsoft Office entry and expand it - you should see all of what are considered Office applications listed in it, and Excel should be one of those.  You can right click the entry and use "Pin to Taskbar" or "Send to..." and choose Desktop to create handy shortcuts to Excel for use later.

If you don't see Excel in the list under Microsoft Office, see if Excel.exe is on your computer: Select the C:\ drive and do a search for Excel.exe - if it is installed, it's going to show up in a path like

C:\Program Files\Microsoft Office\Office14

or one like

C:\Program Files (x86)\Microsoft Office\Office14

Although the version may show up as something other than Office14. 

If you still cannot find any sign of it, we have to fall back to the possibility it didn't get installed initially.  You can get it installed if you have the original media through the Control Panel.  If it is set in Category display then choose "Uninstall a program" under Programs and Features.  If you have the view set to small icons, choose the "Programs and Features" entry.  A list of all installed programs will appear.  Microsoft Office is going to appear in that list (Excel doesn't show up separately).  Highlight the Microsoft Office entry and choose the "Change" Option up at the top of the list.  This will bring up the Office installation window with some choices:

Add or Remove Features

Repair

Remove

Enter a Product Key

You might start with a Repair.  But more than likely you will need to use the Add or Remove Features option (you could try Repair and if that doesn't fix things, then try this one).  In the Installation Options section, make sure you expand the Microsoft Office group and ensure that it is to be installed.  Note that you can choose them all very simply by going to the top of the tree and click on the 'drop down arrow' for Microsoft Office and choose "Run All from My Computer"  and that should definitely install everything in your package that can be installed.

Good luck, and Merry Christmas.
