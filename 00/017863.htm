a) the type of account involved

b) what you currently have residing on the email server

 

*** to clarify that more specifically, if you are using a POP account that has been set to delete everything from the server when an email is downloaded, the only emails that will subsequently get downloaded are new emails received. For anything else, depends what you have left on the server and will download what exists the first time since as far as it's concerned, it's the first time you are accessing the account from this particular computer.
