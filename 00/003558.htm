ossiemac say:

Your initial post indicated one cell per page and your later post one column per page. I am assuming one column per page is correct.

The following code should not require any tailoring. Simply insert it in a module and run it. From your comments in your first post, I am assuming that you know how to do this.

The code removes any page breaks first and also clears the print area. If a print area is set then there are circumstances where setting page breaks errors. You can manually set the print area after running the code.

The code only runs on the active (selected) worksheet.

 

Sub SetPageBreaks()
  Dim lngColumns As Long
  Dim c As Long
 
  With ActiveSheet
    'Find number of columns with data. Relies on having _
     column headers or data in all columns of row 1.
    lngColumns = .Cells(1, .Columns.Count).End(xlToLeft).Column
   
    'Remove any existing page breaks
    .Cells.PageBreak = xlPageBreakNone
   
    'Remove print area otherwise setting page breaks can error
    .PageSetup.PrintArea = ""
  End With
 
  'Start at column 2 because page break is inserted to left
  For c = 2 To lngColumns
    ActiveWindow.SelectedSheets.VPageBreaks.Add before:=Cells(1, c)
  Next c

End Sub
