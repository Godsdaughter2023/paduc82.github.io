andreas killer say:
Believe me, it does work. :-) But I guess it does not work as you expect.

Problem: The list a,b,c is included in the list a,b,c,d,e,f, etc. ,x,y,z

So if you start with just an a in a cell, Excel search for the first list that has an a at front. So the fill order might be a,b,c,a,b,c or a,b,c,d,e,... that depends how the lists appears in your custom lists.

Let create a custom list
A,B,C
and
a,b,c,d,e,f, etc. ,x,y,z
that makes sure that A,B,C comes first.

If you now start with an a in a cell and fill down, you get a,b,c,a,b,c
If you start with a list a,b,c,d and select this cells and drag down, you get the alphabet.

That's the way it works.
