paul van der heu say:

What goes into 'other' storage;

 

  • Files you can manage that don’t fit into another category. These include emails and messages (including attachments), documents and downloaded maps. See the More tips for clearing spacesection below for how to manage these.
  • File types your phone doesn’t recognise. These are file types that Windows Phone can't play or view. For example, an .ogv video file. Try deleting your browser history to remove any you may have downloaded using Internet Explorer. And if you've copied some to your phone from your computer, use File Explorer to find and delete these from your phone.
  • Temporary files. Cached files (such as photo and video uploads to SkyDrive) and thumbnails from photos and online photo albums.
  • Files downloaded by apps like UCbrowser, whatsApp, YouTube downloaders and such. These apps can't store the downloaded data outside their own space which is allocated from 'other' storage and will remain there until you delete them from within the app.

jay freedman say:
In Word, a tab stop setting always affects a minimum of one complete paragraph (the text from one paragraph mark to the next paragraph mark). The only way to make a tab stop affect only one line is to have only one line per paragraph -- that is, put a paragraph mark (press Enter) at the end of each line.

This is incompatible with trying to justify the entire group of lines, which requires that they all belong to the same single paragraph.

There is a possible solution. If you do make each line a separate paragraph, you can select them all except the last one and press Ctrl+Shift+J to apply "distributed" alignment. The difference between this and "justified" alignment is that "distributed" will justify the first (or only) line of a paragraph. Then a tab stop you set on one line will affect only that line.

The problem with this is that if the text typed into the form fields extends too far, the text at the end of the line will be forced to the next line, and will have huge spaces between the letters. This looks really bad.

The usual alternative is to use a single paragraph and not try to use tabs to position the fields at specific places -- just let Word wrap the lines as needed. Another alternative is to put everything into a table (with its borders turned off) with the form fields placed one-per-cell in cells with fixed sizes and positions.
