The stop error code 0xC000021A is displayed when either the Winlogon.exe or Csrss.exe fails to load. d)      After you boot your computer using Windows DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.

e)      Select the correct time and Keyboard type.

f)       Click Repair your computer in the lower left corner

g)      Click on Troubleshooting, advanced option and then click Command prompt.

In the command prompt type the following command: sfc/scannow and press enter

Now, restart the computer.

 

If the above steps didn’t resolve the issue, I would suggest you to once again try the automatic repair option after setting the boot priority to CD\DVD disc as mentioned in the above steps.
