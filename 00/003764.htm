suzanne s. barnhill say:

As marysully says, in order to have a landscape last page, you must have a section break. Since you already have two sections, it is no problem to do what you need. You do not need to "restart" page numbering. "Continue from previous section" (which is the opposite of restarting) is the default for new sections unless you have previously restarted (unlikely if you have just a single previous section).

 

What you do need to do, since you want the footer on your landscape page to be different from the footer in the previous section, is unlink the section 2 footer from the footer in section 1. To do this, on the contextual Header & Footer Tools | Design tab, click "Link to Previous" to turn it off. Then you can type your text in the section 2 footer without affecting the footer in section 1.

 

If you don't know how to create a landscape section (and especially if you want a portrait footer on your landscape page), see http://word.mvps.org/FAQs/Formatting/LandscapeSection.htm.

I'm guessing that in the Section 1 footer you are just typing a page number, either 1 or 2, and that number appears on both pages. That isn't the correct way to do it.

Instead of typing a number, you need to insert a PAGE field. The simplest way to do that is to press Alt+Shift+P. That will display the number 1 on page 1 and the number 2 on page 2.

Technically you should also put a PAGE field in the Section 2 footer on page 3. But if you don't plan to add anything to the document that would make more pages, it probably isn't worth the effort.

Using Word 2003, I was not familiar with this keystroke and never needed it because the Insert Page Number button on the Header and Footer toolbar does just exactly what it's supposed to. Unfortunately, in Word 2007 and above, the situation becomes much complicated. If a user has already added a header or footer building block and wants to add a page number to the existing header/footer, and if s/he makes the mistake of choosing one of the page number building blocks, s/he will find that it replaces the current header/footer building block. Only by using Current Position is it possible to insert just a page number, and even then you have to choose from a dizzying array of "fancy" page numbers. This can be quite daunting to someone who wants to replicate the action of the Insert Page Number button on the Header and Footer toolbar! That's why I suggest the keystroke (which can be used anywhere in a document, too--not just in the header or footer).

hansv mvp say:

Try this:

Sub CopyComments()
    Dim wsh As Worksheet
    Dim cmt As Comment
    Application.ScreenUpdating = False
    Set wsh = ActiveSheet
    For Each cmt In wsh.Comments
        If cmt.Parent.Column = 1 Then
            cmt.Parent.Offset(0, 1).Value = cmt.Text
        End If
    Next cmt
    Application.ScreenUpdating = True
End Sub
