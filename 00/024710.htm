FYI - tethering is setting up a device to 'forward' its internet connection to some other device. You set up your phone to support this by turning on internet sharing. It does not connect the devices.

jim gordon mvp say:

The default format for cells is General. When using General format, Excel formats cells based on its internal rules on a cell-by-cell basis. Type text in a cell, Excel will treat it as text. Type a number, Excel treats it as a number - unless it looks like a date or a fraction. Start a cell entry with an = sign and Excel tries to interpret your typing as a cell formula.

When you are performing an operation on a column have mixed data types in General format, Excel shows you only options that make sense for the entire column. Sorting alphabetically makes no sense when you have numbers to sort. Doing logic with conditional formatting is tough when you are trying to evaluate text as numbers. 

Excel already does the best it can with mixed data. The solution is that you either clean up the data (put only one data type in a column) or tell Excel to format the entire column as text. You can click on the column letter above a column, press Command-1 and then choose Text as the format for the column. Then Excel will behave consistently for you.

https://www.youtube.com/watch?v=IXyrgvtQrVE

To automatically autofit you will need a macro that fires on the sheet_change event

Place this in the code pane of your worksheet. To add this macro, press Alt-F11, double-click Sheet1 (or whatever your sheet is named) in the Project Pane at the left, and copy the code in the right-hand code pane.

You will need to change the line - Set rngCellsToAutofit = Range("$D$5:$E$6") to the range of cells that contain the text you wish to fit.

Private Sub Worksheet_Change(ByVal Target As Range)
Dim rngCellsToAutofit As Range
Set rngCellsToAutofit = Range("$D$5:$E$6")
If Intersect(Target, rngCellsToAutofit).Address = Target.Address Then
  Target.Rows.AutoFit
End If
End Sub
