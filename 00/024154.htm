Multiple Indexes in Word for Mac - Microsoft Community

unclehefty say:

Multiple Indexes in Word for Mac

I am an author and currently involved in a biography that has a number of areas that I would like to index separately.  It is possible that entries to each index can be found anywhere throughout the document.  I am looking at creating five separate indexes:

Artiste

Band

Song

Album

Place

Can anyone tell me if this is possible and if so where can I find references as to how to achieve it.

Thanks

a. user say:

Yes, you can use a "switch" in each index entry (XE) field and in each INDEX field. 

So for example, if you have songs indexed using { XE "The Song Remains the Same" }, { XE "This is My Song" }

you could use { XE "The Song Remains the Same \f "s" } and { XE "This is My Song" \f "s" }

Then, to produce an index of songs, you could use

{ INDEX \f "s" }

However, the standard index and index entry dialogs do not let you specify this "f" switch, so you will need to modify each field by hand. To do that, you will need to

  • ensure that hidden text is visible and use option-F9 (or fn-option-F9 depending on your keyboard) to view the "field code",
  • type the '\f "song"' into each field that needs one
  • finally, select the INDEX fields and update them (F9 or fn-F9)

If you need an entry to appear in more than one index (e.g. in a "song" index and in a general index) you will need two { XE } entries, one with \f "s" and one without.

(Notes - as far as I know:

you can use category names such as "song", but the XE and INDEX fields only recognise the first letter.

you don't have to have the "" around the category name

Also, you might find it useful to set up some autocorrect entries or some such to insert 'empty" XE fields such as { XE   \f "a" }, { XE \f "b" } and insert the index entry text manually, rather than go via the dialog box.

How can I link two Word documents located on the same hard drive? - Microsoft Community

jupiter blue say:

How can I link two Word documents located on the same hard drive?

In my master document I want to have an active phrase or word that, when clicked, will automatically open another document. I've tried Hyperlink, but keep getting an error message that says, "Cannot Open the Specified File."  Of course, when I use Insert File it places the whole file in my master document. Not what I want.  Help please!!
john mcghie say:
You're on the right track, but you need to specify the correct protocol and file type :-)

Select some words in the document and hit Command + k (or Insert>Hyperlink) to open the Hyperlink dialogue.

Now change to the DOCUMENT pane.

Choose your document, and OK it.  BOTH documents must be saved when you do this (or you need to hand-code the entire thing).

If you then right-click the hyperlink, you will expose the field code.  You will see something like this:

HYPERLINK "file://localhost/Volumes/Data/Users/john/Documents/John%20McGhie%20Hothouse.doc"

Note that the link begins with the word "file://".  That's the protocol designator.  It tells the computer that the target is a file on a computer.  If you have HTTP:// in that position, that's a command to the computer to hand the file over to your default browser to look it up on the web.

Note the forward slashes: those are the Unix path separators.  Colons may work, but will often give strange results on a modern system.

Note the extension ".doc" at the end of the file name: that is a command to the computer to hand the target over to Word.  If that's wrong, you live in misery.

You don't HAVE to use Command + k to do this.  You can use Insert>Field and choose Hyperlink, or you can use Command + F9 to create an empty field.  But then you have to code everything yourself...


Hope this helps

Conditional Formatting Based on the time fo Day - Microsoft Community

mark1974dalton say:

Conditional Formatting Based on the time fo Day

I have a work control sheet where columns represent 12 minute blocks of the day. I want to shade cells in the column when the actual time of day is the same time as is represented by that column. Can this be done using a formula in conditional formatting. eg.

        A                  B                 C                D                E

08:00-08:12   08:13-08:24  08:25-08:36  08:37-08:48  08:49-09:00

 

I have a cell with the NOW() formula in it that I thought I could use a reference cell to conditionally format the cells in my columns, but whatever I try it just doesn't work.

ossiemac say:

OK I am assuming that the times you have posted are the column headers. Is this ssumption correct? If so, then I think you should make a change because except for the first column, the headers only represent 11 minutes instead of 12. eg. column A is 08:00-08:12 and then column B is 08:13-08:24. What do you want to do with the 1 minute in between end of column A and start of column B. I suggest that you change as follows:

 

08:00-08:12 08:12-08:24 08:24-08:36 08:36-08:48 08:48-09:00

 

That way current time (NOW() ) can be compared to >= to the first value and < than the second value.

 

Select all of the columns required for the conditional format. (The formula assumes that cell A1 will be the first cell of the selection.)

 

Select Conditional Format and then select" Use a formula to determine which cells to format" and then copy the following formula and paste into the formula field and then set the required format.

 

=AND(MOD(NOW(),1)>=TIMEVALUE(LEFT(A$1,5)),MOD(NOW(),1)<TIMEVALUE(RIGHT(A$1,5)))

 

You should note that NOW() function only updates when the sheet calculates so you might need to force calculate with F9 to get it to update. The MOD function is to extract the time only from NOW() which is actually Date and Time.
