How can I covert a PDF file to a word file for editing?

How can I covert a PDF file to a word file for editing?

charles kenyon say:
Hi Marcia,

Since Word 2013, Word has had the ability to directly open pdf document. However, some pdf documents (from scanners mostly) are really pictures of text rather than the text itself. Just as you can't look under the hood of a picture of a car, you can't edit a picture of a document.

In that case, you need to convert the picture to text. This is a process known as optical character recognition. This is built into Adobe Acrobat and is also in Office OneNote. Most scanner software comes with an OCR component as well. Once you have text, you can edit it with Word.

If you simply want to write on the document (but not in it) you can add a Text Box floating on top of the document layer, whether or not it has been put through the OCR process.

Word documents that have been saved as PDF will not need the OCR process, they retain their text, although not all of their Word structure and formatting.

Finally, documents converted from pdf (or really any other format) to Word can be tough to edit because the conversion process never has a one-to-one matching of how formatting is done. This means that a converted document will seldom be formatted in Word in a way that makes sense. An example is multiple section breaks to change margins, where in Word you would simply change the paragraph indent.
I hope this information helps. Please let me know if you have any more questions or require further help.

Regards

    greginmich say:

    Windows Defender has real-time protection in Windows 8.1, which means that it’s continuously scanning your PC’s file system activity. So a lapse in the automatic scanning is really not a major issue, and Defender hasn’t really stopped working unless the Windows Defender Service has stopped, or its real-time protection has been turned off. In fact, the automatic scanning feature isn’t even handled by the Windows Defender program itself – it’s handled by Task Scheduler, which is the primary interface for Windows Automation. The Windows Defender Scheduled Scan task is one of several Windows Defender scheduled tasks that are triggered daily by Windows Automatic Maintenance (a higher order, “management” task).

    So if you’re unable to resolve your issue with a malware cleaner, or with a program uninstaller, you do have the option for setting up your own scheduled task to run the automatic Defender scans, and a user-scheduled task actually makes for a much more reliable and timely scheduled scan. The steps required are listed in this thread:

    https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-do-i-set-windows-defender-to-a-custom-auto/ec19aab4-10ce-4f41-9990-343056fd104f?page=1  

    But we do still have an issue with scheduled scans not running consistently under the System account in Windows 10 (which actually might also be one of the issues with your scans in Windows 8.1) – so if you decide to try this, you’ll have to be sure to skip step 7, as I’ve indicated in this thread:

    https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-to-set-up-windows-defender-to-scan-pc/0d2d6ffb-bb94-4cbf-9e12-33aaf1999df1

    And if all of this is just a little too techy for your liking, then you should be just fine if you only run a manual scan every now and then, or whenever you’re prompted to do so – because Defender is always working full time in the background, scanning files from startup to shutdown.

    GreginMich
