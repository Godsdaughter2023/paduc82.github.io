Converting a typed bibliography into a format suitable for import into Word's sources.xml file is far from straightforward, as the data parsing varies according to the numbers of authors & editors (and how much information there is for each), publication type (book, journal, chapter in a book, url, etc.), and so on.

It's probably easiest to copy the existing data into a separate app like Notepad, where you can toggle back & forth between there and Word's Source Manager for the purpose of copying & pasting the source data into the Source Manager.

One thing to note when copying author/editor details is that they should be in the format of:

surname, given name/initials

for a single name or:

surname, given name/initials; surname, given name/initials; surname, given name/initials; etc.

for multiple names.

julie sheets say:
In File> Options, Advanced, select the option "Calculate multiple critical paths."

rod gill say:

Julie has the correct answer, but you need to know it calculates a critical path for every path ending in a task with no successor.

So make sure the milestones that flags the end of each phase are the only tasks with no successor.
