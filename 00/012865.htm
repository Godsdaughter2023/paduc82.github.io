Pure speculation here, I haven't tried it; but what I can imagine is code in the Form's Change event and its BeforeUpdate event.

 

The Change event fires the instant a keystroke is entered into the field - at the very beginning of the update process. It could capture the current state of the field into a variable. hassan el-meligy say: Adding AutoCorrect entries

If you use Word as your e-mail editor

  1. In Word, click AutoCorrect Options on the Tools menu.
  2. In the Replace box, type a word or phrase that you often mistype or misspell — for example, typeusualy.
  3. In the With box, type the correct spelling of the word — for example, typeusually.
  4. Click Add.

If you use Outlook as your e-mail editor

  1. From the main Outlook window, click Options on the Tools menu, and then click theSpelling tab.
  2. Click AutoCorrect Options.
  3. In the Replace box, type an abbreviation that you want to use as a shortcut.
  4. In the With box, type the text you want to appear as a replacement.
  5. Click Add.

More information here http://office.microsoft.com/en-us/outlook-help/use-shortcuts-for-commonly-typed-phrases-HA001034791.aspx

Hi SWatkinsFSB，

 

Regarding “Mail Encryption”, if you refer to the Office 365 Message Encryption service in Exchange Online or Exchange Online Protection, you can use the Read Receipt when sending the encrypted emails in Outlook client or Outlook Web App. To send the Read Receipt request email, please see the steps below:

Outlook Web App: open a new message > More commands “…” > Show message options > Request a read receipt.
