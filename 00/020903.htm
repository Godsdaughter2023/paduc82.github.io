Sub RossinOz()
Set r = Intersect(Range("C:C"), ActiveSheet.UsedRange)
st = "DROPPED"
For Each rr In r
    If rr.Value = st Then
        ActiveWindow.SelectedSheets.HPageBreaks.Add before:=rr.Offset(1, 0)
    End If
Next
End Sub

karl timmermans [outlook mvp 2012-2018] say:

You can stop looking since there is no "real solution" - the field names for standard Outlook field names cannot be changed.

 

Background.............

 

Outlook utilizes "one" set of "internal" field names for every version in existence regardless of country or language in use enabling 3rd party programs to be written without regard of language. The "Display Name" fields are for external use only and depend on the Outlook (Office) language settings in use. Those internal field names are fixed and cannot be changed.

 

While the capability to change field names may seem a trivial request, the impact of allowing that on the universe of 3rd party Outlook programs depending on those fields would be a complete "nightmare" for the population at large especially since it would involve every version of Outlook going back to at least O'2000 which some people still use.

 

Correlation between Outlook "internal" field names and Display Names for 9 different languages can be found below if of interest

 

http://www.contactgenie.com/outlook_fields.htm
