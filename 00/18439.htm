ipellant say:

Isaac,

I don't have a Nextbook (no Walmarts in Australia). But did buy a couple of cheap Tablets; one of which was sacrificed to Windows 10... so I can only offer generic advice.

To restore to Windows 8.1 was a little tricky. I did it a few times on the Windows 8.1 Bing tablet after trying out Windows 10.

I assume (hope) you created a Windows 8.1 Recovery USB stick from the previous Windows 8.1 original install using the Recovery Create USB drive in Windows 8.1.

Windows 10 does not seem to recognize Windows 81. Recovery drives as valid... it only seems to look for a Windows 10 Recovery media drive.

What I needed to do was force the tablet into the hardware BIOS screens with the USB stick plugged in, it then appears as a boot device and can be selected. It then boots to Windows 8.1 Recovery where an option appears for a complete disc re-partition and install. It will then restore everything back to factory state.

If it boots into Win 10, try using Recovery to boot into UEFI BIOS under the advanced options.

The higher level options may not recognize the Windows 8.1 media and will only show "Restore Windows 10"... which becomes an infinite loop to nowhere. At that level, the Windows 8.1 drive will create an invalid image error.

A brute way is to force the tablet into BIOS recovery mode by using a hardware reset - there may be a small hole in the case to poke a paper clip into. When the tablet is turned on, force a reset a few times and it will eventually boot into a BIOS recovery mode and the hardware boot device option can be found. Some tablets can get there by holding the volume button down (or up) when powering on. Sometimes holding the power button down for a half a minute then release and press again for half a minute may force it into recovery mode - may need to do that 3 times.

Problem with a low level / hardware BIOS is that you will need a keyboard that works at the low hardware level. I assume that your Nextbook 2 in 1 came with the keyboard; otherwise a powered USB hub with USB keyboard, USB mouse and the USB recovery stick need to all be plugged in. (Fully charge the tablet before starting).

There are many variables involved - it may take a few attempts to get the USB stick booting.
