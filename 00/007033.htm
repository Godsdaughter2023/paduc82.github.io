doug robbins - mvp office apps & services (word) say:

The "Black Heading" problem occurs because the font characteristics in the list template have become corrupted.  It can be fixed by running a macro containing the following code:

Sub Fixit()

Dim templ As ListTemplate

Dim lev As ListLevel

For Each templ In ActiveDocument.ListTemplates

    For Each lev In templ.ListLevels

        lev.Font.Reset

    Next lev

Next templ

End Sub
