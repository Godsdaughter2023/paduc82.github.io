ossiemac say:

The following change event code is fairly generic in that it copies the format from the selected item in the list and pastes it into the target cell.

Caveat: If the target has any existing formatting that is not included in the corresponding cell of the list then the existing formatting will be lost and replaced with the formatting of the found cell in the list.

 

To install code:

Right click on the worksheet name tab and select View code.

Copy the following code and paste it into the white area of the VBA editor.

To close the editor, click the cross with red background top right of VBA editor.

Save as Macro enabled workbook.

Enable macros if not already enabled. See Help for how.

 

Private Sub Worksheet_Change(ByVal Target As Range)
  On Error GoTo ReEnableEvents
  Application.EnableEvents = False
  Dim rngToFind As Range
 
  'Test if change is in the required range
  'Edit Columns("D:D") to required column
  'or maybe range as Range("D2:D5") to suit your project.
  If Not Intersect(Target, Columns("D:D")) Is Nothing Then
 
    If Target = "" Then
      Target.ClearFormats   'If cell is empty (Optional)
   
    Else
   
      'Edit to sheet and range of dropdown list.
      With Sheets("Sheet3").Range("A1:A5")
          Set rngToFind = .Find(What:=Target.Value, _
              LookIn:=xlFormulas, _
              LookAt:=xlWhole, _
              SearchOrder:=xlByRows, _
              SearchDirection:=xlNext, _
              MatchCase:=False)
     
          If Not rngToFind Is Nothing Then
            rngToFind.Copy
            Target.PasteSpecial Paste:=xlPasteFormats
          End If
      End With
    End If
  End If
 
ReEnableEvents:
  Application.EnableEvents = True
End Sub

 

If you make changes to the code and for any reason it fails withoug going to the error routine then events will remain turned off until after restarting Excel. Alternatively insert the following code in the VBA editor (anywhere in the vba editor and can be under the code above). Just click anywhere in the sub and press F5 to run the code which will turn events on again.

 

Sub Re_EnableEvents()
  Application.EnableEvents = True
End Sub

 

 

jllatham say:

Chip is correct, you cannot format the entry as it appears in the list.  But!...

You can get it to show up the way you want it to once it is selected using Conditional Formatting.  Depending on the length of your list and how many unique entries you want to format separately, you could sure push the limits of conditional formatting.

Just set up a bunch of conditional formats using Cell Value Is Equal To and then the particular word in the list and set formatting accordingly.  Repeat as needed.  Another way to do a similar thing would be to use a Worksheet_Change() event to provide almost limitless formatting possibilities.

Worksheet event code to do it might look something like this, where your dropdown list is in cell A1. And instructions on how to put it into a worksheet code module are at http://www.contextures.com/xlvba01.html#Worksheet

 

Private Sub Worksheet_Change(ByVal Target As Range)
  Const testCellAddress = "$A$1" ' address of cell with your drop-down list
  If Target.Address <> testCellAddress Then
    Exit Sub ' no action to take
  End If
  'clear any previous formatting
  With Target.Font
    .Bold = False
    .Italic = False
    .ColorIndex = xlAutomatic
    .Underline = False
  End With
  Select Case Target
    Case Is = "Two"
      Target.Font.Bold = True
    Case Is = "Three"
      Target.Font.Underline = True
    Case Is = "Four"
      With Target.Font
        .Italic = True
        .Underline = True
      End With
    Case Is = "Five"
      With Target.Font
        .Bold = True
        .Italic = True
        .Underline = True
        .ColorIndex = 3 ' red
      End With
    Case Else
      'ignore, leave at default
  End Select
End Sub
