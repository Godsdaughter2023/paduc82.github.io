steve rindsberg say:

You can use a shape's TextRange.BoundLeft, .BoundTop, .BoundHeight, .BoundWidth to get the coordinates of the text (as opposed to those of the text box itself).

Here's a simple example you can adapt:

' First, a sub to call the function, passing it the currently selected shape

    Call AddXparentBox(ActiveWindow.Selection.ShapeRange(1))
End Sub

    Dim oBox As Shape
    Dim oParentSlide As Slide
   
    Set oParentSlide = oShp.Parent
   
    With oShp.TextFrame.TextRange
        Set oBox = oParentSlide.Shapes.AddShape(msoShapeRectangle, _
            .BoundLeft, .BoundTop, .BoundWidth, .BoundHeight)
    End With
   
    ' set properties for the rectangle shape here
    With oBox
        With .Fill
            .ForeColor.RGB = RGB(0, 0, 255)
            .Visible = True
            .Transparency = 0.99
        End With
        With .Line
            .Visible = False
        End With
    End With
   
    Set AddXparentBox = oBox
   
End Function

bill manville say:

I would agree that writing to a worksheet and printing that would be the best approach.

 

In fact you could have all the data in a worksheet and load the listbox from the worksheet range.

 

Then, if there is a userform frmPrintOptions with option buttons optPortrait and optLandscape and a multi-select listbox lbColumns containing the 12 column names in the order they are on the sheet
your code would be something like this:


Dim iColumn As Integer
For iColumn = 1 To frmPrintOptions.lbColumns.ListCount
  Sheets("Sheet1").Columns(iColumn).Hidden = Not frmPrintOptions.lbColumns.Selected(iColumn - 1)
Next
Sheets("Sheet1").PageSetup.Orientation = IIf(frmPrintOptions.optPortrait, xlPortrait, xlLandscape)
Sheets("Sheet1").PrintArea = Sheets("Sheet1").Range("A1").CurrentRegion.Address
Sheets("Sheet1").PrintOut

 

Selecting a printer on your own form is not quite so straight-forward.

In fact, the best option might be to have a button on the form that calls up Excel's printer dialog using

  Application.Dialogs(xlDialogPrinterSetup).Show
