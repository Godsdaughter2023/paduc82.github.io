rick-m say: Virtual PC, Windows XP Mode was only made for Windows 7. It's not supported in Windows 8.x or Windows 10. Hyper-V is the replacement for it, but you need to have the 64-bit version of windows to run it. The only other option is to try VM-Ware, or Virtual Box, or configure your computer for Dual / Multi-booting. 

c.b. wilsha say: Thank you very much, Rick.  That explains why mine is not working! My W10 is a 64 bit version so can I get Hyper-V from Microsoft as I did VM and can I install XP on it??

rick-m say: Actually, it's built into Windows 10. You need to go to your control panel > programs and features > turn windows features on or off. When you go through the list, Hyper-V should be in there. You said you did an upgrade to Windows 10 from Windows 7, your WindowsXP.vhd file should still on your hard drive some place which you should be able to reuse. Unfortunately, I can't give you the exact path as I have changed the default location on my computer.
