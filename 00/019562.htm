Hi Jon,

If you want to add external users to the Modern Group in OWA, it’s not feasible to do this. Thanks for your understanding.

If you want to add external users to the Distribution or Security Group, you can add them to the contacts in Exchange admin center. Please follow the steps below (You need to be an admin or at least an Exchange administrator):
1.Sign in to the Office portal and choose the Admin icon.
2.Click Admin in the left and choose Exchange.
3.In the Exchange admin center, click recipients>contacts.
4.Click “+”> mail contact and fill in the blanks.
5.Click Save.
6.After adding the external user to the contacts, click groups in the same page.
7.Double click the group you want to add this user to and click membership in the pop-up.
8.Add the external user and click OK.
9.Click Save.

If you want to add members from a domain outside your tenant to your SharePoint Group, you need to turn on the external sharing feature and then add them to your group. Below are the steps:
1.Turn on the external sharing future in the site collection level.
  a.Sign in to the Office portal with your admin work/school account and click Admin icon.
  b.Click Service Setting>Sites.
  c.Turn on the button for external sharing.
2.Add the external users to the group. They will appear in the group after accepting the invitation.
To learn more, see Manage sharing with external users in Office 365 Small Business.

joe men say:

Hello John,

The function keys F1 - F12 are mapped to the numerical keys except for the - (Hyphen) key which is used to execute the F11 key and the = (Equal) key which is used to execute the F12 key. You will have to press the Fn key + the numerical value of the function key that you wanted to perform.

For example, you wanted to execute the F5 key to refresh a certain website. Press the Fn + 5 keys on the keyboard to perform the refresh.

Here are the corresponding commands:

  • Fn + 1 keys corresponds to (F1)
  • Fn + 2 keys corresponds to (F2)
  • Fn + 3 keys corresponds to (F3)
  • Fn + 4 keys corresponds to (F4)
  • Fn + 5 keys corresponds to (F5)
  • Fn + 6 keys corresponds to (F6)
  • Fn + 7 keys corresponds to (F7)
  • Fn + 8 keys corresponds to (F8)
  • Fn + 9 keys corresponds to (F9)
  • Fn + 0 keys corresponds to (F10)
  • Fn + - keys corresponds t

    Feel free to reply if you have further questions.
