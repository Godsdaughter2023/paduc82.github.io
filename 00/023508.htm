Use VBA to count the number of tables in a word doc?__ - Microsoft Community

brettm winteam say:

Use VBA to count the number of tables in a word doc?__

Anyone know how I can use VBA to count the number of tables in a word doc? Ultimately I am trying to insert a new table at the end of the rest of the tables but the table number could be 2 or 10 or 1.. Anyone know how I can use VBA to count the number of tables in a word doc? Ultimately I am trying to insert a new table at the end of the rest of the tables but the table number could be 2 or 10 or 1..
gerry knight say:

ActiveDocument.Tables.Count gives the total count of tables.  Note however, that NESTED tables are NOT counted.

We would need more details on "I am trying to insert a new table at the end of the rest of the tables".  How are you going to do this?  Do you want to insert a new table right after the LAST table?  Is there a paragraph mark after the last table?

If there IS a paragraph mark after the last table, then something like:

Dim r As Range
Set r = ActiveDocument.Range( _
   Start:=ActiveDocument.Tables(ActiveDocument.Tables.Count).Range.End + 1, _
   End:=ActiveDocument.Tables(ActiveDocument.Tables.Count).Range.End + 1)
    
   ActiveDocument.Tables.Add Range:=r, NumRows:=3, NumColumns:= _
      2, DefaultTableBehavior:=wdWord9TableBehavior, AutoFitBehavior:= _
      wdAutoFitFixed

creates a range object of LAST table + 1, and uses that range to insert a table.

The new table is after the last table, where ever it is.  If you want something different from that you will have to actually state what it is you want.

Gerry Word MVP

jeeped say:

I often use a caption under each table or graphic that incorporated the{SEQ} field. Consider this,

Table 1 - My First Table
Table 2 - My Second Table
Table 3 - My Third Table

I've used the field code {SEQ} to create an numbered entry. This can be brought into the document through Word 2007's Insert, Quick Parts, Field command or you can simply tap Ctrl+F9 and type it in like this:

  Table { SEQ tbls \* MERGEFORMAT } - My First table

... where tbls is any name you decide that is used to sequence the occurences of SEQ tbls.

Once the first has been created, it can be copied and pasted to subsequent tables. Press Ctrl+A (e.g. Select All), then F9 to force update all fields. They will be updated automatically before printing in any event.

Note that if you use the captions (titles?) above the tables, you can modify the Paragraph, Line and Page Breaks to incorporate Keep with next to avoid orphaning them from their associated table.
