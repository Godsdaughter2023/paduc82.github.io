e.j. gun say:

These appear to be ActiveX option buttons.  They will be automatically grouped on a worksheet such that you can only select one at a time.

For your VBA, you need to know the name for each option button.  Go into Design Mode.  Right-click on each button and select "Properties".  Look at the "(Name)" field.

In VBA, you will have code that looks like that below.  For my example, I assume the worksheet is "Sheet1".

If (Sheet1.OptionButton1) then ' First option selected

    MsgBox "First option is selected"

ElseIf (Sheet1.OptionButton2) Then

    MsgBox "Second option is selected"

ElseIf (Sheet1.OptionButton3) Then

    MsgBox "Third option is selected"

etc., etc.

End If

For check boxes, you can have multiple selected at the same time, so the code will be a little different:

If (Sheet1.CheckBox1) Then

' Do stuff here because CheckBox1 is selected

End If

'

If (Sheet1.CheckBox2) Then

' Do stuff here because CheckBox2 is selected

End If

In this case, you have to check each CheckBox individually, since they are not linked together like the OptionButtons.

Hope this helps,

Eric
