There's a few places you can look.  It sounds like you've checked a few of the more common ones.  I may repeat these below:

  1. The Startup folder
    Press +R to show the Run box, type shell:startup and click OK.

  2. The Registry for Current User Account
    Press +R to show the Run box, type regedit and click OK.  Navigate to HKCU\Software\Microsoft\Windows\CurrentVersion\Run

    Press +R to show the Run box, type regedit and click OK.  Navigate to HKLM\Software\Microsoft\Windows\CurrentVersion\Run

  4. Check MSCONFIG
    Press +R to show the Run box, type msconfig and click OK.  Check the Startup tab.
