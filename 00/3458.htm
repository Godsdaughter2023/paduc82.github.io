The challenge of designing Excel for smaller size screens/touch - Microsoft Community

mauricedurbin say:

The challenge of designing Excel for smaller size screens/touch screens.

I have used Excel on 15 inch and 13 inch laptops, on a 10 inch hybrid in Keyboard/mouse mode and a 10 inch tablet. mauricedurbin say:

We need to maximise screen space available to the worksheet.

The ability to choose Full Screen mode is a must.

When in Full Screen mode the two wide rows at the top of the screen occupied by the Tabs (Home, Insert etc) and their associated functions should be removed. This frees up a lot of screen space.

Make them available by a swipe up from the bottom of the screen (or right click) which would give the tabs, and then after touching/clicking the desired tab the associated functions would overlay the tabs using a minimum amount of screen space. To return to full screen needs a further up-swipe or right click. This entirely consistent with the operation of other Apps such as Mail, for example.

You could follow the same strategy to save space used at the bottom of the screen by adding a new 'sheets' tab to those at the top (home through view plus 'sheets'). Selection of sheets would overlay an equivalent of the worksheet tabs giving the user the opportunity to choose the worksheet they want to work in.

In this way you add screen space for 6 to 8 more visible rows of cells in the worksheet which taking in to account space occupied by an on screen keyboard near enough doubles useful space.

Creation of a spreadsheet is more likely to be done using a larger screen. The user would not be short of space so would not need to select Full Screen which leaves them with the traditional UI as now, so both scenarios, 'creation' and 'use', are catered for.

Hello,

"Microsoft account" is the new name for what used to be called a "Windows Live ID." Your Microsoft account is the combination of an email address and a password that you use to sign in to services like Outlook.com, OneDrive, Windows Phone, or Xbox LIVE. If you use an email address and password to sign in to these or other services, you already have a Microsoft account—but you can also sign up for a new one at any time.

An administrator/Local account is a user account that lets you make changes that will affect other users. Administrators can change security settings, install software and hardware, and access all files on the computer. Administrators can also make changes to other user accounts.

Using Microsoft Account you will be able to access all the apps and other programs that requires you to login with the Microsoft Account. You can Create multiple number of user account according to your requirements.

"Edit, Find, find next, find previous— these first there don't even work from edit menu— nothing comes up."

Not exactly accurate... Initially only the Find command does anything -- It sends the insertion point to the Search in Document field which has the magnifying glass. But it doesn't cause any type of dialog to "come up". Until you have done a Find for something, Find Next & Find Previous do nothing at all. When you enter text into the Search in Document field & press return the Sidebar/Search Pane appears on the left side of the window. It's a bit misleading because the phrase "Find and Replace" appears at the top of it rather than its actual name.

"But the ones below them do: replace, go to , and advanced."

Yes, they are a totally different set of commands so they evoke different behaviors... That's why there is a separator line between one group of commands & the next.

"Gear dropdown, you mean magnifier icon,  not gear?"

Really, Kevs55 -- I don't make any claim of flawlessness, but even for an old goat like me it's a bit difficult to confuse a magnifying glass with a gear. As I wrote, the gear icon is in the Sidebar/Search Pane (left side of the window), not in the Search in Document oval (upper right corner). It's next to the button labeled Find, near the top of the pane.
