rohn007 say:

Exactly what do you mean by "scan into Word"? Basically, it can't be done in one step.

Scanning only creates images, "photocopies", of the scanned text that you can paste into Word, but the words on the scan are a picture so they can't be edited as text. Your printer probably came with scanner software to capture the initial image.

1.1.1 Replace MODI in 2010

http://en.wikipedia.org/wiki/Microsoft_Office_Document_Imaging

1.1.1 A Few General Scanning Tips

http://www.scantips.com/

If you want to scan a document and be able to text edit the words, say in Word, then you need a second step which is called "OCR", Optical Character Recognition. OCR software takes images, such as scanner output or even PDF's and "extracts" the text out of the image and presents it in editable text format.

Since you have Office 2010, you have built in OCR. After you get a scanned image, select and copy it, then paste it into OneNote. After it has been pasted into OneNote, right click on it and select "Copy text from Picture". You can now paste the extracted text into word. You will have to carefully review it and spell check it. Depending on the quality of the original image OCR programs have problems interpretting the text so sometimes the output spelling errors.

1.1.1 Use OneNote Printer to Extract Text from Images and PDFs

http://www.howtogeek.com/61012/how-to-use-onenote-printer-to-extract-text-from-images-and-pdfs/

1.1.1 OCR- Using Microsoft Office Document Imaging in Office 2010 (OneNote)

http://www.brighthub.com/computing/windows-platform/articles/91749.aspx

jeeped say:
... What I need to do is format rows 20 up to the row just above the row with the word "Total" in it to have all borders ...

I'll assume that you have a fairly large range to cover so I'll call it A20:F999. Press F5 and type of paste in A20:F999 to the Reference: text box and click OK.

Click Conditional Formatting, New Rule from the Home tab's Styles group. Choose Use a formula to determine which cells to format then use the following for the Format values where this formula is true: text box,
=IFERROR(MATCH("Total",$A:$A,0)-ROW()>0,FALSE)
Click Format and apply a Border pattern. Click OK to save the format and then OK again to save the rule.

Your cells should remain plain until Total is put into a cell in column A. Then row 20 to the row above Total will have the border pattern you assigned. Note that the IFERROR() function is specific to Excel 2007/2010.

BTW, I used the formula =E20-D20 in F20 and applied a custom number format of.
[Blue]"Up "0_);[Red]"Down "0_);[Green]_("Even"_);_(@_)
... which seemed to work well.

hansv mvp say:

Create an event procedure for the On Current event of the form as a whole:

 

Private Sub Form_Current()
    If [field1]="other" Then
        [field2].Visible = True
    Else
        [field2].Visible = False
    End If
End Sub

 

or even

 

Private Sub Form_Current()
    [field1].Visible = (Nz([field1]) = "other")
End Sub

 

(The Nz function is used to prevent errors when field1 is entirely blank)
