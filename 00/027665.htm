john sr wilson say:

You should be able to do it with a macro NOTE this code is "top of head" so use with care on a copy of your pres.

Sub replaceR()
Dim osld As Slide
Dim oshp As Shape
Dim ipos As Integer
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
If oshp.HasTextFrame Then
ipos = InStr(1, oshp.TextFrame.TextRange, Chr(174))
If ipos > 0 Then
Do
oshp.TextFrame.TextRange.Characters(ipos).Font.Superscript = msoTrue
ipos = InStr(ipos + 1, oshp.TextFrame.TextRange, Chr(174))
Loop Until ipos = 0
End If
End If
Next oshp
Next osld
End Sub

Merging Cells in protected Sheet in Xls - Microsoft Community

syoufi say:

Merging Cells in protected Sheet in Xls

I'm trying to allow users to merge cell ranges in a protected sheet .......... it proving really difficult

hansv mvp say:

You'll have to create a macro that

  • Unprotects the sheet
  • Merges the selected cells
  • Protects the sheet again

You can assign the macro to a custom keyboard shortcut and/or a custom Quick Access Toolbar button.

Sub MergeSelection()
    On Error Resume Next
    ActiveSheet.Unprotect Password:="Secret"
    Selection.Merge
    ActiveSheet.Protect Password:="Secret"
End Sub

Change "Secret" to the password that you used to protect the sheet - use "" if you didn't specify a password.

You can also provide a macro to unmerge cells:

Sub UnMergeSelection()
    On Error Resume Next
    ActiveSheet.Unprotect Password:="Secret"
    Selection.UnMerge
    ActiveSheet.Protect Password:="Secret"
End Sub

Don't forget to save the workbook as a macro-enabled workbook (.xlsm).

Users will have to allow macros.
