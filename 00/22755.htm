First create a font theme. You can do this with any text editor, you can even use TextEdit if you're careful. Here are instructions: XML Hacking: Font Themes

Then go to the Master slide and set the placeholder text to the theme fonts. In the font dropdown these will appear as Fontname (Headings) or Fontname (Body).
