Family Safety: Frequently Asked Questions

Important Note: Antivirus software can help protect your computer against viruses and other security threats. In most cases, you shouldn't disable your antivirus software. If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

  4. Now, click on the account for which you want to set family safety.

  5. Click on ‘Set up family safety’.

  6. Now go back to the control panel main page and select ‘Family safety’.

  7. Check if family safety is turned ON for that account.

  8. Now try changing that account to child account and check if it works.

Hi,

There are several reasons as to why you're encountering issues downloading the game. Here are some of the factors:

  • Network interruption during the download process
  • An outdated Windows 10 OS
  • Corrupted user data

Here are the troubleshooting solutions that you may perform to resolve this issue:

Solution 1: Run troubleshooter
The Windows apps troubleshooter can help resolve issues with apps and games downloaded from the Windows Store.

Solution 2: Check for updates

We also recommend keeping your Windows 10 device up-to-date. Refer to How to get software and driver updates for Windows 10 for the steps.

Solution 3: Reset the Windows Store app

Resetting the Windows Store app from the Settings menu can help restore corrupted data associated with the app. Here are the steps:

  1. Click Start and click Settings.
  2. Select System.
  3. Select Apps & features.
  4. Find the app and click Advanced options.
  5. Click Reset.
  6. Restart your device so that the changes can take effect. Download the game and check if the same issue occurs.

Solution 4: Sign in to a different user account

Signing in to a different account can help isolate if this is caused by a corrupted profile data.
  1. Press the Windows button on your device or keyboard, or select the Windows icon from the bottom-left corner of the main screen.
  2. Select your account name from the top-left corner of the menu, and then select Switch account.
  3. Sign in with an account other than the one you were using. Download the game using the new account. Make sure to sign in to the Windows Store using the account you've used to purchase the game.

If the same issue occurs after performing the solutions above, you may contact our Xbox support team via chat or request a call at your preferred date and time. They'll provide you options on how to get a game refund. Refer to this link for their contact information.
