But I thought I'd ask first.

Thanks in advance for any help.

troy

macropod say:

You could use a wildcard Find/Replace, where:

Find = <*>

Replace = <i>^&</i>

and you set the Find font parameter to italics. Note that the entire replacement will be italicised and that multi-word strings will have the <i> & </i> around each word.

A second wildcard Find/Replace could be used to remove the </i> & </i> between words, where:

Find = \<[!\>]@\> \<[!\>]@\>

Replace = ^32

Another wildcard Find/Replace could then be used to remove the italics from the outer <i> & </i>, where:

Find = \<[!\>]@\>

Replace = ^&

and you set the Replacement font parameter to not italics.

steve rindsberg say:

For occasional problems with older files, it's worth trying an online conversion service.  Zamzar is free and a few users have reported that it's able to convert older PPT files to formats that will open in current versions.  Whether it'll work with really old Mac PPT or not, I don't know, but at these prices, it's worth a try.

Another approach is to create virtual machines to hold older versions of software.  I'm running PPT 2016 in a Yosemite VM created in Parallels on a 4gb Air.  It doesn't win any speed contests (I'd enter it in Slow Contests if they had 'em) but it'd be plenty good for converting the occasional file.  Older versions of MacOS and PPT will be far less demanding, so might run quite reasonably.  You'd need installation media for both MacOS and Office to create the virtual machine, of course.
