You can edit the .wpl file to .m3u file, which is supported in most media players. You can do so by opening the playlist file in Notepad. It'll resemble something like this:

 

#EXTM3U
#EXTINF:10,Sample.jpg

#EXTINF:10,Sample 2.jpg
Sample 2.jpg

neilpzz say:

With the option to Run as Administrator you should select Yes for it to make a difference, so using that setting doesn't change anything.

You should be able to see a Compatibility tab for the shortcut you use to run the program. Are you using a button in Start to run Money. If so right click the icon and go to More then select Open File Location. That will highlight a file in File Explorer in the Money folder. If you right click that file and go to Properties you should see Compatibility. Tick the box and select your last operating system.
