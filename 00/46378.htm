Remove Account:

a. Open the Messaging app.
b. Press Windows key + C.
c. Select the option Settings.
d. Click on Account, Select the facebook account.
e. Select the option Remove account.

Add Account:

a. Open the Messaging app.
b. Press Windows key + C.
c. Select the option settings.
d. Click on Account, select add an Account.
e. Select the option facebook and follow the onscreen instructions.
