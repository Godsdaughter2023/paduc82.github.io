nathan o. - microsoft support say:

Hi Mike,

 

Usually this error means you have some files on your machine that are not registered properly and / or some missing / invalid registry keys.  If this is the case, I would first start by trying the steps in KB 958846 (it states Access 2007, but the info still applies to Access 2010).

 

958846 - Error message when you try to create a new database in Access 2007: "Template could not be instantiated"

http://support.microsoft.com/default.aspx?scid=kb;EN-US;958846

 

After going through these steps, I would check to see what your default file format for new databases in Access 2010 is set to.  If it isn’t set to the Access 2007 file format, I would change it to this and test again.  To check / change this you can do the following:

+>Open Access 2010.

+>Click on the File tab and then Options.

+>On the General tab, check the “Default file format for Blank Database” setting.  If this isn’t set to Access 2007, change it to this and test again.

 

If you still have the problem after this, I would then probably recommend opening up a support ticket with Microsoft so that we can look at some specific Ace and ADO registry keys on your machine.  To open a case, you can do so online, using the following link:

 

Online Assisted Support Options (create cases online) - http://support.microsoft.com/select/default.aspx?target=assistance

 

Best Regards,

Nathan Ost

Microsoft Online Community Support

5)  Now try to open IE11 (or IE10) normally to see if it is working.

brian tillman [outlook mvp 2007-2019] say:

If you created a new mail profile and didn't add a data file, then you'd see the "Cannot start Microsoft Office Outlook.  Cannot open the Outlook window.  The set of folders cannot be opened.  The information store could not be opened." error.


Export exists mainly to transfer data from Outlook to a non-Outlook program.  Export to a PST also exists for selecting only specific portions of a PST.


If I wanted to transfer some of my contacts, I'd create a separate PST, create in that PST a new Contacts folder, then copy to that folder the specific contacts I wished to transfer.  I'd then close Outlook and copy the PST to the other Outlook, opening it there.
