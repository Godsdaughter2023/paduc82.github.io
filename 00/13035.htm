A                                                 B            C            D                 E

School Name                          School #    Area       Date Out       Makep-up date

White School                             01              1         3/25/13         5/20/13

Red School                                12              2         3/25/13         5/20/13

Blue School                               23               3         3/25/13         5/20/13

Green School                            24               2         3/25/13         5/20/13

Orange School                          30               1         3/25/13         5/20/13

 

I need all area 1 school on one sheet with all data, all area 2 schools on another sheet with all data, etc............. Any help you could give would be most appreciated. I've done this in the past with a function, but my brain is getting older and I just can't figure it out! Thanks!

mike h.. say:

Hi,

 

Try the code below. You will need to change the sheet name with the data on (currently Sheet1) to the correct name.

 

 

The code will create new sheets called Area1, Area2 etc and copy the data to the relevant sheets. I

 

 

 

Sub Copy_Data()
 Dim r As Range, LastRow As Long, ws As Worksheet
 Dim LastRow1 As Long, src As Worksheet
 'Change this to sheet with data on
 Set src = Sheets("Sheet1")
 LastRow = src.Cells(Cells.Rows.Count, "A").End(xlUp).Row
 For Each r In src.Range("A2:A" & LastRow)
         On Error Resume Next
         Set ws = Sheets("Area" & r.Offset(, 2).Value)
         On Error GoTo 0
         If ws Is Nothing Then
             Worksheets.Add(After:=Worksheets(Worksheets.Count)).Name = "Area" & r.Offset(, 2).Value
             src.Rows(1).Copy ActiveSheet.Range("A1")
             LastRow1 = Sheets("Area" & r.Offset(, 2)).Cells(Cells.Rows.Count, "A").End(xlUp).Row
             src.Rows(r.Row).Copy Sheets("Area" & r.Offset(, 2)).Cells(LastRow1 + 1, 1)
             Set ws = Nothing
         Else
             LastRow1 = Sheets("Area" & r.Offset(, 2)).Cells(Cells.Rows.Count, "A").End(xlUp).Row
             src.Rows(r.Row).Copy Sheets("Area" & r.Offset(, 2)).Cells(LastRow1 + 1, 1)
             Set ws = Nothing
         End If
Next r
 End Sub

      Reasons why the file might be read-only:
      - the file is marked as read-only
      - the parent folder is marked as read-only
      - the file has been saved as Read-Only recommended and you accepted the invitation to leave it so when you opened it
      - the file is open for updating by some other user and you accepted the invitation to open it as read-only
      - the file was opened by a macro in another workbook that specified read-only
      - there is a macro in the file that changes it to being read-only.

      suzanne s. barnhill say:

      You should be able to unassign it.

      1. Select a U200C space in your document and open the Symbol dialog. The "zero-width non-joiner" will be selected (though of course it is invisible).
      2. Click Shortcut Key... The Customize Keyboard dialog will open with your shortcut shown in the Current keys box.
      3. Select that shortcut, and the Remove button will become enabled.
      4. Click that button to remove the key assignment.

      Alternatively, you can open the Customize Keyboard dialog, select Common Symbols in the Categories list, select No-Width Optional Break, and then do steps 3 and 4.
