dale howard [mvp] say:

Daryl --

Based on the behavior you describe for this project, I would be very concerned that this project has gotten corrupted.  Though there are no hard and fast symptoms of corruption, the one thing you always look for is weird behavior in a project that does not behave like all other projects.  My friend, you are seeing weird behavior, which makes me think it might be corrupted.

To attempt to resolve the corruption, I would recommend you try using the Save for Sharing method with it.  You will need to add the Save for Sharing button to your Quick Access Toolbar first.  Then you will need to do the following:

  1. Open the problematic project.
  2. Click the Save for Sharing button.
  3. Save the MPP file in a convenient location, such as your Desktop.
  4. Close the MPP file.
  5. Reopen the MPP file.
  6. Click File > Save As and then click the Save button to save it in Project Server.

Though this process does not resolve every possible kind of corruption, it might fix your problem.  Please let us know if this helps.
