gina whipp say:

Excel Issue - I would expect that, Excel is not *set up* for shared use.  Usually, the first one to open it is the lucky one, everyone else is just opening a *snapshot* of the data at it's prior Save.  If they select Save it will default to Save As because the first User is the only one who can actually save.  No fix that I am aware of.

Access Issue - I would suggest recreating the Linked Tables because it sound like the few are connected to a different Backend.  It may even be an archived one.  Also, are you using a mapped drive or UNC?
