dominicp say: The unknown device will probably go away if you disable Away Mode in your BIOS. Google for the Device ID shown in the device's Properties in Device Manager to find what it is.

If you upgraded online, you will get the same bitness as the system you download on - 32-bit if you were running XP.

There may be no drivers for your DTV card. If it is an old card the chipset manufacturer will not supply any reference drivers for it. Check on the TwinHan site and, if you can determine which chipset the card has, check for generic Windows 8 drivers for that chipset.
