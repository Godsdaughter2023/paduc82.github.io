jim gordon mvp say:

Office 2016 is the same install whether you purchase a stand-alone perpetual license or an Office 365 subscription.

Uninstall/reinstall will have no effect and provides no benefits, other than it will probably install updates along the way, and that can be done without uninstall/reinstall.

You can easily check to see which kind of license you currently are using. Open Microsoft Word and go to the Word menu, then choose About Microsoft Word. The dialog will tell you the license type. If it says Office 365 Subscription, then you have an Office 365 subscription. If it says anything else, the software license is not a subscription type. Also check the version number shown. You should have version 16.14.1 and if that is not the version, before you do anything else, install updates.

Make sure your Mac has a working internet connection before checking for updates. Install Mac OS X updates followed by Office updates.

Install Mac OS X updates first:

To get Mac OS updates, go to the App store on your dock, and then click the Updates button to reveal available updates.

Then Install Office Updates

To get the update:

1. Open any Office application

2. Go to the Help menu and choose Check for Updates

3. Let AutoUpdate check for and install updates

4. Restart your Mac

If you have any add-ins such as WebEx, Mendelay, EndNote, Zotero, TypeIt4me, etc it is essential that you install the latest updates for your add-ins.

To change your subscription type, log out of the current log-in and then log in with the credentials for the license you want to use. Here's how to change your login:

Go to the templates gallery in Word (File > New from template). Click the circle in the upper left corner of the gallery. Log out.

Restart your Mac

Return to the circle and then log in using the credentials of the other account license you want to use.

That said, crashes are almost always caused by not having updates installed. Before bothering with the license type, install updates.
