steve rindsberg say:
This is both simpler and more difficult than you might like.

Each slide contains just a bitmap image, so you won't be able to edit anything, but since there's just the one shape on the slide, it's easy enough to work out which shape to adjust:  THE shape.  The ONLY shape.

This will set all of the white pixels in the image transparent so you can see the background beneath

Sub SetXparency()

Dim oSl As Slide
For Each oSl In ActivePresentation.Slides
    With oSl.Shapes(1)
        With .PictureFormat
            .TransparentBackground = msoTrue
            .TransparencyColor = RGB(255, 255, 255)
        End With
    End With
Next
End Sub

mel77084 say:

Is your first boot in the morning a cold boot and the others during the day all restarts? If so, you need to turn off "fast startup" in power plan.

With "fast startup" on, your computer saves your running processes in ram during shutdown, and picks them back up during a cold boot; however, during restart, fast startup doesn't operate, so it takes longer, but starts all of the programs that are set to start on boot.

If your ram won't hold all of the processes that you had running before shutdown, any that couldn't be held in ram upon shutdown do not start in the cold boot; then, if you do a restart, they are all loaded.

Error message 0x8004FF0B and 0x80070715 when you attempt to uninstall - Microsoft Community

aiscer say:

Error message 0x8004FF0B and 0x80070715 when you attempt to uninstall or reinstall Microsoft Security Essentials

This is a consolidation of all resolutions for the errors 0x8004FF0B and 0x80070715 when attempting to uninstall, remove or reinstall Microsoft Security Essentials.

The best method for uninstalling Microsoft Security Essentials is still through Add/Remove programs or from Programs and Features if you have Vista or Windows 7. These are alternate resolutions for removing MSE if the natural method fails.

METHOD 1

1. Download the MSE installer for your Operating System from the Home Page: http://microsoft.com/security_essentials
http://microsoft.com/security_essentials
2. Save the file to your Desktop. This will ensure that you can find it easily after the download is done.
3. Open a Command Prompt console:
    FOR XP: Go to Start. Choose RUN delete any entries. Type CMD then hit enter.
    FOR Vista and W7:Go to Start Search. Type CMD. Right-click on CMD and select "Run as Administrator".
4. In the Command Prompt console type: CD %USERPROFILE%\Desktop <-hit ENTER
5. Type: msse <-then hit the TAB button.
6. Depending on your Operating System, after hitting tab it should either appear as any of the three below:
          mssefullinstall-x86fre-en-us-xp.exe
          mssefullinstall-x86fre-en-us-vista-win7
          mssefullinstall-amd64fre-en-us-vista-win7.exe
7. Now put a space in between the whole line and put /U as seen below:
          mssefullinstall-x86fre-en-us-xp.exe /U
          mssefullinstall-x86fre-en-us-vista-win7 /U
          mssefullinstall-amd64fre-en-us-vista-win7.exe /U
8. Hit ENTER and this should uninstall Microsoft Security Essentials.
9. After the uninstall you can now attempt installation. It would be best that you restart your system first after the uninstall.

*Do not forget the space between CD and the % sign.

METHOD 2

1. Download the MSE installer for your Operating System from the Home Page: http://microsoft.com/security_essentials
2. Save the file to your Desktop. This will ensure that you can find it easily after the download is done.
3. Open a Command Prompt console:
    FOR XP: Go to Start. Choose RUN delete any entries. Type CMD then hit enter.
    FOR Vista and W7:Go to Start Search. Type CMD. Right-click on CMD and select "Run as Administrator".
4. Now drag the icon for the installer of MSE into the Command Prompt window.
5. After dragging the icon, the shorcut for the location or path should appear in Command Prompt. Put a Space and then /U. Hit ENTER.

METHOD 3

1. Download the MSE installer for your Operating System from the Home Page: http://microsoft.com/security_essentials
http://microsoft.com/security_essentials
2. Save the file to your Desktop. This will ensure that you can find it easily after the download is done.
3. Now press the Windows Key and R all at the same time. This will open the RUN dialog box.

         C:\Users\[YOUR USERNAME]\Desktop\mssefullinstall-amd64fre-en-us-vista-win7.exe /U
