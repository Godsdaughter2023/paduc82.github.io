Instead, if you must test it on a single machine, the best recommendation is to create a partition or have an a spare hard disk ready where you can install it.     Conditional formatting of a cell based on another cell's value - Microsoft Community

    disrinieuwoudt. mnr say:

    Conditional formatting of a cell based on another cell's value

    Hello Everyone

    I use conditional formatting to assign background colors to cells based on their values.  I.e. when cell A3>4, it's colored green, when cell A3>3, it's colored yellow etc.

    At the same time, when cell A3 has a value of higher than 4 (cell A3 is now background colored green), cell A4 displays "GOOD".  Is there a way to get cell A4 the same background color as cell A3.  In other words cell A3 determines what the value in cell A4 is going to be (as text), but I want A4 to automatically take on the background color that corresponds with the background color of cell A3.

    Thank you so much

    hansv mvp say:

    The idea would be:

    Select A3:V4.

    On the Home tab of the ribbon, select Conditional Formatting > New Rule...

    Select 'Use a formula to determine which cells to format'.

    Enter the formula   =A$3>0

    Click Format...

    Activate the Fill tab.

    Select red.

    Activate the Font tab.

    Select the desired text color.

    Click OK, then click OK again.

    Repeat the above with   =A$3>1   and orange as fill color and another text color.

    Also with   =A$3>2   and yellow as fill color and black as text color.

    Etc.
