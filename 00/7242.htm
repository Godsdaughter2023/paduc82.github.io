donleverton say:

I do something somewhat similar, in that I delete / re-create a  LOCAL Access table from a LINKED Excel sheet in VBA.

My method uses a "template" table that has the desired structure. I make a copy of the blank template, and then append the new data into it. One advantage is that your AutoNumber gets "re-set" so that it doesn't continue to grow during recursion.

I have run into a bit of a snag, in that Access will not allow you to delete a table that is participating in a relationship.

Yeah, there is a work-around to that as well.;)

=============== Code ==============

'Temporarily DELETE the Relationship
Call DelRelations

'Delete the old Inventory table.
If IsTableQuery("", "tblInventory") Then
    With DoCmd
        .SetWarnings False
        .DeleteObject acTable, "tblInventory"
        .SetWarnings True
    End With
End If

'Copy the blank(template)table, renaming it to "tblInventory"
DoCmd.CopyObject , "tblInventory", acTable, "tblInventoryTemplate"
'Now that we have correctly structured table we can import the data.
    
Me![txtProgress].Text = "Repopulating LOCAL Inventory Table"
Me.Refresh
DoCmd.Beep
Me![txtProgress].SelStart = 0
Wait 2, False


' Note that "irrelevant" IVCLS# items will not be appended to the local table.

MySQL = ""
MySQL = MySQL & "INSERT INTO tblInventory " & CR
MySQL = MySQL & "SELECT InventoryExcel.* " & CR
MySQL = MySQL & "FROM InventoryExcel " & CR
MySQL = MySQL & "WHERE (((InventoryExcel.[IVCLS#]) Not In ('CHEM','DSL','GAS','GRAV')));"
MyDB.Execute MySQL, dbFailOnError

'Restore the Relationship
Call MakeRelations

============ modRelations Code ===========

Sub DelRelations()
Dim Myrel As DAO.Relation
Dim MyDB As DAO.Database
Set MyDB = CurrentDb

'Delete the Relationship
MyDB.Relations.Delete "IVITM"
Set MyDB = Nothing
End Sub

Sub MakeRelations()
Dim Myrel As DAO.Relation
Dim MyDB As DAO.Database
Dim MyFld As DAO.Field

Set MyDB = CurrentDb

'Create the relationship on IVITM
Set Myrel = MyDB.CreateRelation("IVITM")
Myrel.Table = "tblInventory"
Myrel.ForeignTable = "tblLocation"
Myrel.Attributes = dbRelationDeleteCascade + dbRelationUpdateCascade
Set MyFld = Myrel.CreateField("IVITM#")
MyFld.ForeignName = "IVITM#"
Myrel.Fields.Append MyFld
MyDB.Relations.Append Myrel

Set MyDB = Nothing
Set MyFld = Nothing
End Sub
