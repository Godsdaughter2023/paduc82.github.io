considerate_guy say:

Contrary to what it may seem like, Movie Maker doesn't store any media files within the project. All that you'll find in the .WLMP file is a list of file names. If you were pulling those directly off of an SD card and that card is lost, then you have no video files.

Usually the best practice is to import all your videos (and photos) off the SD card and onto your computer's hard disk. In that way you have a copy of them and can work from the faster hard disk anyway. If you did that, then maybe the project file will point to the copy on the hard disk? You can hover over a clip in your Movie Maker project and it will show you the location of that media item.

But if you are seeing exclamation points in yellow triangles, that means that Movie Maker is not finding the file at that location. If it was on an SD card or other removable media, that item must be missing. Or if it is pointing to your hard disk, the files have either been moved, renamed or deleted.

Remember:

A project file is like a recipe for how to build your movie. But just like a recipe, it has a list of ingredients, not the actual ingredients themselves.
