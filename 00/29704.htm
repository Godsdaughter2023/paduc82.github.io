There can be a lot of factors when there are unresponsive errors when installing Windows 10 on an SSD. Since this is a new SSD, the Page File setting is set as default. We suggest changing it to the recommended settings. Here's how you can do this

1. Press Windows key on your keyboard or click Start menu, then on the Cortana search box,  type "Adjust the appearance and performance of Windows" and click the search results.
2. Go to Advanced tab and click Change... button in the Virtual memory section.
3. At the top, uncheck the box right before "Automatically manage paging file size for all drives" option.

5. Select Custom size: by ticking its radio button, and change the Initial size (MB) and Maximum size (MB) to the Recommended value that you will find just below the Total paging file size for all drivers section.

This method will help improve the system performance of Windows 10, and will gradually decrease the freeze issues.
