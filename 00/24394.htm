l  Outlook > Tools > Account Settings > double-click on the target email account > More Settings

l  Select the Advanced tab on the More Settings Window to uncheck the ‘Leave a copy of the Messages on the Server’ option

bill manville say:

You haven't said what version of Excel you are using.

I am guessing 2003 or earlier.

First thing to do is to see if they remain disabled if you start Excel in safe mode (hold down Ctrl while launching Excel).  If not then the disabling is being done by an add-in or a program that is running during startup.
- in that case, disable the add-ins and remove the startup programs and use a process of elimination to find the culprit.

If they are disabled in safe mode then you could run this macro:

Sub ReEnableCutCopyPaste()
  ReEnable 19
  ReEnable 21
  ReEnable 22
  ReEnable 6002
End Sub

Sub ReEnable(ID As Long)
  Dim CB As CommandBar
  Dim CTL As CommandBarControl
  For Each CB In Application.CommandBars
    Set CTL = CB.FindControl(ID:=ID, recursive:=True)
    If Not CTL Is Nothing Then CTL.Enabled = True
  Next
End Sub
