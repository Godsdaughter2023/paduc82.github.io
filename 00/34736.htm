Error 80072af9 occurs when you sign in or download Xbox LIVE content

Overview

If you see error code 80072af9 when you try to sign in to Xbox LIVE or download content from Xbox LIVE, there may be a network problem or a temporary service issue.

Read this page to learn more about error 80072af9 and what you can do about it.

On this page

Topics

Error code and message

Problem

Solutions

Error code and message

Error code 80072af9 contains one of the following messages:

Some Xbox LIVE content is temporarily unavailable... You can still access your profile, communicate with other players and play games online.

The content will automatically appear when it's available.

Status Code: 80072af9

The download could not be completed. Please try again later. Any points you used for this transaction have already been deducted from your balance and will not be deducted again.

Status code: 80072af9

Problem

If you get this message and error code, it means one of the following:

•Some Xbox features are not available because of system maintenance or a temporary Xbox LIVE issue.

•You can't access some Xbox features because of a network configuration issue.

Solutions

Solution 1: Check the status of the Xbox LIVE service

During scheduled system maintenance or unforeseen Xbox LIVE issues, some Xbox LIVE features may temporarily be unavailable. To determine the Xbox LIVE service status, check the Service status page.

Solution 2: Test your Xbox LIVE connection:

To test your connection to Xbox LIVE, follow these steps:

1.Open the Settings hub and select System.

2.Select Network Settings.

3.Select Wired Network or the name of your wireless network, if you are prompted to do so.

4.Select Test Xbox LIVE Connection.

Solution 3: Download the content again

Try to download the content again to see if the connection problem was temporary. For information about how to download previously purchased content, see Buy or redownload content from Xbox LIVE.

If you still can’t download the content, try the next solution.

Solution 4: Open network ports

To access Xbox LIVE, several ports need to be open on your network hardware or firewall. For information about which ports must be open for Xbox LIVE, see Network ports used by Xbox LIVE.

If you still can’t download the content, try the next solution.

Solution 5: Improve network performance

If you can't connect to Xbox LIVE, are frequently disconnected, or can't see some content, try to improve your network performance. For help, see Issues with connection speed or disconnects.

If you still can’t download the content, try the next solution.

Solution 6: Check for network connectivity or ISP issues

You may be unable to connect to Xbox LIVE or use some features if your Internet service provider (ISP) has set bandwidth or connection time limits. Contact your ISP to determine whether there are any restrictions that might cause your connection to time out.
