mccbleue say: Check you memory (Menu>Tools>File Manager>Options>Memory details) to make sure your memory isn't overfull, it may just be a case of deleting some data or moving things like pictures and music to a memory card.

Sometimes the phones get confused over their memory. If this is the case, back up your personal data (either using PC Suite or to memory card, reformat the phone with the codr *#7370#. After this, the phone will ask for a lock code which is 12345 if you haven't changed it, then the phone will switch off and restart. Try to send a message and you should be successful. After this, you can restore your personal stuff to the phone.
