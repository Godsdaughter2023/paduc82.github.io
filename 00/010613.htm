bernie deitrick say:

Rounding - you shuld be summing the hours and minutes and not the decimal values. Your displayed values are rounded to two decimal places, so, for example 8.32 is actually 8.31666666666667. Your sum is 39.983333333, which is 39 hours and 59 minutes. Using time math in Excel will return 39 hours and 59 minutes

 

The easiest fix is to change to H:M from decimal, so change all 24s to 1s in your formula (and take out the ones not needed, and all those extra parens:

 

=IF(OR(G8="",G7=""),0,IF(G8<G7,G8-G7+1,G8-G7))


 

and format for all your time cells for [h]:mm

 

And really, since you are using AM and PM, your formula could just be

 

=IF(OR(G8="",G7=""),0,G8-G7)


If you need to calculate anything (like salaries), multiply the final time sum by 24 before using it

By default Internet Explorer disable running unsigned ActiveX controls. Unsigned activeX controls are programs that try to execute a command on your PC, like CAB install files or a ZIP file which do not have a digital signature.

While this is an important security measure on the Internet Explorer browser, if you are sure that the website is a trusted one then you can temporarily disable this security feature to get the website run the required ActiveX and later on re-enable the security.

To disable this feature:
a.    Open internet explorer.
b.    Click Tools – Internet Options.
c.    Click the Security tab – Custom Level button.
d.    Under ActiveX ensure the following are set to  enabled
        o   Run ActiveX and plug-ins – Enabled
        o   Download Signed ActiveX Control – Enabled
        o   Script ActiveX controls marked as safe for scripting – Enabled
e.    Set the following to “Prompt”
        o   Download unsigned ActiveX Control – Prompt
        o   Initialize and script ActiveX controls not mark as safe – Prompt
f.    Click OK and Apply and OK.
g.    Restart the browser.

You should now be able to install the ActiveX control.

rohn007 say:

Monthly payments mean Office 365, not "Office 2013".

Assuming that you mean Office 365 "Home" (5 machine/user) or "Personal" (1 computer) it is actually simple enough to do.

  • On the new computer, open your browser
  • Log in to http://office.microsoft.com/MyAccount using your email address and password
  • Click on the "Install" option. In there you should see info about the install on the broken machine
  • There should be a "disable activation" link there that you can click on
  • Then you can find the "Install" link to install and activate on the new computer.

If you really mean Office 2013, like "Home and Student" or "Home and Business", then you will need to find your product key first.
