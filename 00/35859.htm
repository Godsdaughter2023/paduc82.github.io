No, you won't lose the operating system or its activation status:

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

hairyfool say:

Formatting a partition involves removing all file allocation tables and clearing the root directory of all entries. By definition that partition (otherwise termed as a volume) is therefore empty of all data and programs (the "C drive" may not be the entire disk) What you cannot do is format a partition from within the Windows that is booted from it because Windows protects itself. In order to format an operating system drive you have to have booted from another disk.

There are ways to recover from a newly formatted partition but as soon as you start writing files to the empty drive to all intents the data is unrecoverable. Formatting is not exclusive to Windows, all operating systems can carry out the same basic task.

When you Reset Windows you remove all customisations, re-initialise the registry and replace all the operating system files with known good copies. There are two options when resetting, to remove everything that has been added and to keep your personal files. Because the Registry is reset normally all Apps are removed apart from the built in ones. With a factory image HP can integrate their Apps into that Windows build so the reset does not removes their additional Apps.

There is an argument that even those manufacturer modifications could be detrimental and the only truly optimised system arises from a clean install. HP in particular were past masters at including a vast amount of "bloatware" with earlier versions of Windows not needed by the majority of users. How bad it was for the computer is open to opinion and frequently overstated. Of course it does not help when Microsoft themselves start doing it in the "clean" setup (how many really want Films and TV?)

Not all that HP include on their build can be downloaded from the support site so if you use any of it then your only option is a reset, if you don't use them (never found anyone that uses LightScribe) then a clean build with possibly drivers from HP should give you the best configuration.

andre da costa say:

Both Reset and booting from install media do the same thing.

Booting from install media then deleting and formatting the partition is much faster than Reset and will of course provide a faster out of box experience.

A)     First, right-click on the Sidebar icon in the system tray, and then choose Exit.

%localappdata%\microsoft\Windows Sidebar

a. user say:

Windows 7 has a feature to safely shut down all programs when the system shuts down. It is due to this reason that windows is taking so much time to shut down.

However you can easily solve this problem. Just press [Win+R] and type regedit to open the registry editor. Go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control. On the folder on the right double click on the string to open the WaitToKillServiceTimeout field.In the > Edit String window, change the > Value data from the default of 12000 (12 seconds) to 2000 (2 seconds). Click > OK to save the change.

If this entry does not exist, you can create it. Right-click into the folder and click through > New > String Value, and name it > WaitToKillAppTimeout.

Set > Value data to 2000, click > OK.

In the same registry folder, if you see an entry called > AutoEndTasks, set its > Value data to 1, then close the registry editor.

greg carmack - mvp 2010-2020 say:
I'm Greg, a volunteer installation specialist and 8 year WIndows MVP here to help you.

1) Fix Windows 10 Camera Not Working in Windows Settings

Some users may not know how to launch the laptop camera when they change to Windows 10 from its predecessors. In fact, it's easy to find the camera on the Start menu. But you may also see the "We need your permission" error after opening the Camera app. In this case, you need to turn on the laptop camera via the Settings.

Step 1. Use Windows shortcut keys Win + I to launch the Settings.

Step 2. Go to the Privacy menu.

Step 3. Select the Camera option on the left side.

Step 4. On the right side, toggle the switch under the "Let apps use my camera hardware" to the ON position.

Step 5. Scroll down and turn on the apps that can access the camera.

Now back to the Camera app and you will find your image has appeared on the screen. This is an easy tutorial for those who fail to open camera on Windows 10 Creators Update/Fall Creators Update. If the laptop camera not working still gets on your nerves, or if you see "We can't find your camera" error in the app, then go on reading the following content.


2) Enable Camera in Device Manager of Windows 10 Creators Update/Fall Creators Update

A disabled camera device can result in the Windows 10 camera not working issue. So to fix the problem and make the camera available on your Windows 10 Creators Update/Fall Creators Update laptop, you also need to enable the camera hardware device.

Step 1. Right-click the Start menu.

Step 2. Select the Device Manager.

Step 3. Expand the Imaging devices entry.

Step 4. If you find your camera device has a special mark on the icon, then you should right-click it and select the Enable device.

enable-camera-device-manager-windows-10.png

After that, the device list will refresh by itself and you can check the device in the Camera app to see if this has resolved the problem. If not, try the camera driver solution below.

3) Update Camera driver in Device Manager.

Start at the PC's Support Downloads webpage. Download and install the latest driver.

Other methods here: http://answers.microsoft.com/en-us/windows/wiki...

If no newer or different drivers to try, on Cam device Driver tab click Uninstall and restart PC to see if it works after reinstalling driver.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
