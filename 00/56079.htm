angelcarreno say: If your computer is connected to wireless network, then this is the way to find out the network status:

1) Go to lower right corner (taskbar area) in Windows desktop screen to click on the network icon, after that click on Network settings.
Windows 10 network settings
2) Scroll down the Network Settings window and click on Change adapter options.

Note:The other way to go to this Change adapter options is via: Start->Settings->Network & Internet->Wi-Fi->Change adapter options
change network adapter options in Windows 10
3) Right click the Wi-Fi adapter in Network Connections window and click on Status.

Note: Here you can do the same on Ethernet 2 wired adapter to check the network connection status too.
check wifi adapter status in Windows 10
4) The Wi-Fi status window will appear, it briefly shows you the following connection info: IPv4 connectivity, IPv6 connectivity, media state, SSID, duration, speed, signal quality and also bytes sent and received. Don’t click Disable unless you want to disable this wireless adapter.

Note: If you find out there are only bytes sent but very limited bytes received, most likely it’s not connected properly on network. In this case you could click Diagnose to troubleshoot this network problem.
wifi status Windows 10
Note: If you want to find out more info, go ahead to click on Details. Here basically it shows you the network related info, such as IP address, subnet mask, default gateway, DNS server, etc
network connection details in Windows 10

shina schatten say:

To break the code apart with you and see if you can get your profile working try the following:

8015 in your code means your account has something missing/unverified in the security and needs to be fixed

Go to account.live.com/proofs  (if possible use a PC and Internet Explorer as the Microsoft security pages don't always work with other browsers/devices)

Login with your account (login email and password) and verify access if your PC is not yet trusted

Check 3 things on the Protect Your Account page

1. Make sure you either have Two Step Verification OFF

OR set up TSV correctly with a smartphone ID app (and that you are using the ID app passcode instead of the password)

2. Make sure you have two verified security options (email and/or phone number) other than your login email address (your login email will not count for the 2 verified security options)

3. Set up the 25 character recovery code (this code causes a back end change so the system knows you have updated the security)  Also keep the code somewhere safe since it can be used to reset the password and wipe all account security options (replacing them instantly and not having to wait 30 days for the update)

Then check "recent activity" listed on the left and see if your recent attempts show a really weird location  (like the IP is in another state or country)

If it looks really odd but it was you; see if you have the option to agree that the activity was your activity

Next go to account.xbox.com   (or sign in on www.xbox.com and click My Account)

Click Manage Devices on the My Account page

select See Xbox 360 Profile Protection

If you have more than one console showing as having your password saved (and you do not have more than one 360 in use) hit the green "require profile download" button

---

Now the ####19## part of the code.

There is something NOT being trusted about your network with regards to the console/account connecting to Live

The best way to deal with this is to restore your network stetting on the console to defaults (if you use manual settings record them somewhere)

-to restore to factory defaults go to settings>network settings>select your network>configure network>additional settings>restore to factory defaults

Once the network settings are restored, turn off the console (if you use Ethernet disconnect it); go to your modem and router.  After you unplug the power from both, locate the little button on the modem that resets its entire connection to your ISP (normally a little button on the back or bottom that can only be pressed with a pin)

Let the modem and router fully power cycle for 5-10 minutes

When the modem and router are back on if you have Ethernet Reconnect it

Then turn on the console

When the console is back on either test your wired connection or reset up and test your wireless connection

If the test passes and shows connected to Live head to the next step (otherwise you will need further networking assistance)

Now that the console connects to Live:

Delete the following system item files if you have them:  Rights Database, Marketplace System data, Content Cache and Product Improvement Data

-to get to your system items:  Settings>system settings>storage>primary storage device>system items

Clear the system cache 3 times ( support.xbox.com/.../clear-system-cache )

Now re-download your profile

Do not save the password, but please sign in

Make sure you can get online

If you can get online, please restart the console.  This time login normally (you can save the password now) and make sure you can still get online and play

***if the profile is not corrupted this will simply refresh the profile and help the console see all the security updates.  If the profile is corrupted then the profile will fail to download and seem to vanish from the console's sign in options.  If this happens go to settings>system settings>storage>primary storage device>profiles>locate the corrupted profile file (if still present) and then Delete Profile ONLY do NOT delete profile and items unless you want to lose your game saves

Download profile
