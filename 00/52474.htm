a. user say:

The most common reason for this error is that you have one or more device drivers that are not supported on Windows 10.

For your computer to work properly with Windows 10, all your computer's hardware must be supported on Windows 10. If any of your computer's hardware is not supported on Windows 10, it's only a matter of time before problems develop. Sometimes unsupported hardware can work successfully for months or years, then one day an update to Windows is delivered that your hardware isn't prepared for, and Windows crashes.

Hardware support is the hardware manufacturer's responsibility. There isn't anything Microsoft can do about it. You need to make a list of all the hardware in your computer and find out from the manufacturer whether it is supporting your particular model number on Windows 10. If it is supporting Windows 10, the manufacturer will have a driver for Windows 10, or they might say that the Windows 8 driver will also work on Windows 10.

Start with your computer's video card or chip, since that's a common problem area.

A less likely reason for the error is corruption in the operating system itself. That can happen when people fiddle with Windows configuration settings without knowing what they're doing. I used to be famous for this.

There are other causes for this error that are even less common. But in your case, make sure all your hardware has Windows 10 drivers or earlier drivers that the hardware manufacturer supports for Windows 10.

When a computer starts up a Windows splash screen is displayed. On computers manufactured by HP, the HP logo is also displayed during the boot-up process. This is part of the standard branding of an HP computer. If you do not want to look at the HP branding every time you start your computer, there is a way to remove it. You can configure the options in the Windows System Configuration utility to remove the HP branding at start-up.

a. Press Windows key + R and type msconfig.

b. Click the "Boot" tab, then click the check box next to "No GUI boot" in the "Boot options" section. Click the "Apply" button to save the changes, then click "OK." The HP branded splash screen will no longer display.

If the issue persists you can also turn on the fast startup setting and check if it helps.

a) Press Windows Key +W, type "Shutdown Settings"

b) Click the "Settings" Icon on the right of the screen.

c) Click "Change what the power buttons do" on the left of the screen.

d) Click "Change settings that are not currently available"

e) Scroll to bottom of Dialog box.

f) Check the box titled "Turn on fast startup (recommended)."

g) Click "Save Changes" button.

h) Exit Power Options Dialog box.

  1. Incorrectly configured, old, or corrupted device drivers.

  2. Corruption in Windows registry from a recent software change.

  3. Virus or malware infection that has corrupted Windows system files or Windows Operating System-related program files.

  5. Damaged or removed system files after you’ve installed software or drivers related to Windows Operating System.

  6. Corruptions in hard drive
