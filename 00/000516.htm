If not, try this code from Rick Rothstein which keeps numbers sequential.

Sub NumberCellsAndMergedCells()
'rick rothstein
  Dim Cell As Range, Number As Long
  Set Cell = Selection(1)
  Number = 1
  Do While Not Intersect(Cell, Selection) Is Nothing
    Cell.Value = Number
    Number = Number + 1
    Set Cell = Cell.MergeArea.Offset(1)
  Loop
End Sub

O365 Word “Access Denied” Error

11.  While all affected users are members of that security group, not ALL members of the group have this problem. At least (2) have been identified as not having this problem.
