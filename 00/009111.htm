Where, why, and how did MS decide to reformat my text with paragraph spacing??

How can I stop this reformatting?

I have tried this on a variety of files and if the blank doc always changes my para spacing.

Andy

a. user say:

The most likely possibility:

In Word, every piece of text is associated with a named style of some kind. So for example, the "Normal" paragraph style in one document may be set up to have 0pt space following, but in another, it may be set up to have 10pt spacing. It may also have formatting that you applied "directly", e.g. 9pt spacing.

But when you insert one document into another, Word has somehow to resolve conflicts between styles that have the same name. If you did not apply direct formatting, then the text from one document will take on the layout characteristics of the same-named style in the other document. So your direct formatting may stay as 9pt, but all of a sudden, text that had 0pt spacing now has 10pt spacing or vice versa.

doug robbins - mvp office apps & services (word) say:

You need to unlink the headers\footers in the second and third sections from those in the first section of the document.

You can do it manually via the Header & Footers>Design tab of the ribbon, which will appear when you have the selection in the header or footer, or, by running a macro that contains the following code

Dim i As Long, j As Long
With ActiveDocument
    For i = 2 To .Sections.count
        With .Sections(i)
            For j = 1 To .Headers.count
                .Headers(j).LinkToPrevious = False
            Next j
            For j = 1 To .Footers.count
                .Footers(j).LinkToPrevious = False
            Next j
        End With
    Next i
End With

dermottxp say:

Changing Paper Size under Page Setup.

Word 2003 won't allow me to change 'Paper Size' under Page Setup from 'Letter' to 'A4' in an existing document. This problem cropped up once before (about 2003/4) and was easily solved, but I can't now track down the solution that was set out in this forum.
suzanne s. barnhill say:

I was probably the one who previously answered the question, though it was Office MVP Beth Melton, IIRC, who originally identified the solution. Go to Format | Columns and click on the preset button for One (even if 1 column is already selected). Then click OK. Doing this checks the box for "Equal column width," and you may notice that the column width changes.

What causes this and other problems seems to be having a set column width that doesn't adapt to changes in margin width (sometimes it's larger than the margin width, and sometimes there are multiple columns with one consuming the entire margin width, which is another issue). When the "Equal column width" box is checked, it becomes flexible again. If you are able to select "Whole document" under "Apply to," do so. Otherwise, you may have to repeat the procedure for each section.
