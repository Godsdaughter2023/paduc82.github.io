greginmich say:

Windows Defender scheduled scans should also work just fine if you have a third-party AV app installed and you’re running Windows Defender in its Limited Periodic Scanning mode. There’s absolutely no limitation on Windows Defender’s on-demand scans when you have a third-party AV app installed – you just can’t turn on Defender's real-time protection. In fact, I’m currently testing these Windows Defender scheduled scans on a machine that’s running a third-party AV app for its primary (real-time) protection. The instructions for setting up a scheduled scan that I posted in this thread are a little more inclusive:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-do-i-set-windows-defender-to-a-custom-auto/ec19aab4-10ce-4f41-9990-343056fd104f

But neither of these threads tells the whole story – and if you’re actually looking to combine a signature update and a Quick Scan in a single scheduled task, then you’ll need to have access to some undocumented parameters…

The parameter for the update and scan combo in the Windows Defender UI (MSASCui.exe) would be:

-UpdateandQuickScan

And in the Command Line Utility (MpCmdRun.exe), the combo parameter would be: 

-SignatureUpdateandQuickScan 

And it’s also important to note that a scan scheduled by following the steps that I provided in these old threads simply won’t run; because user-scheduled scans are currently subject to the same issue as the Automatic Maintenance Scan – they won’t run under a System account; and my instructions explicitly call for changing the user account for the task to the System account. This will cause the scan to fail – so if you decide to try scheduling your own scan, you’ll have to be sure to skip step 7.

You are just printing a preview of the labels, rather than actually executing the merge, which you must do by using the Finish & Merge facility in the Finish section of the Mailings tab of the ribbon.  That facility allows you to either execute the merge directly to the Printer, or to a new document, which will give you the opportunity to check the labels if necessary and then print that document.
