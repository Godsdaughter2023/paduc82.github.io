You have to have both the phone and Outlook syncing to the same service, and by syncing, I mean either through IMAP4 or Exchange. POP3 will NOT work.

So, if you set up a GMail account and set it up as an Exchange account on your phone, an IMAP4 account in Outlook (GMail only supports mobile phones via Exchange ActiveSync, not desktop clients), then when you delete from the phone, it deletes it on Gmail's servers (or archives it) and it will vanish from Outlook's inbox the next time it syncs.

OK, then, it seems that instead of an equation for a line, you would like to display the value of a calculation on the chart,  So here's how I would do it.

First, a little setup.

Going back to my earlier example, let's say I want to put the sum of the number of cars on the chart so that when the sum changes, the value displayed on the chart is updated.

Let's put the result of the calculation in cell D5 (it could go anywhere except in a cell already busy).I also need to add some text so that it displays in the formula, too. So I put this formula into cell D5
="The number of cars is " & SUM(B2:D2)

Cell D5 now displays "The number of cars is 90"

Next, we need a text box. Form the Insert menu choose Textbox. Drag diagonally on the worksheet to make a text box.

=TheTotal

The text box now has the total, and it is linked to cell D5. Go ahead and change the values in the table and the text box will update.

This is the "markup area," the space allocated for the use of the balloons used for "markup" (tracked changes and comments). It is displayed only in Final Showing Markup or Original Showing Markup view (find these views in the Tracking group on the Review tab).

It will appear if the document contains any kind of markup, whether or not Track Changes is turned on. If you turn Track Changes on, with balloons enabled, the markup area will usually not appear until you have made a change or inserted a comment, so it is likely that the document does contain some markup.

On the Review tab, in the Tracking group, turn on the Reviewing pane and scroll through the pane to make sure that "(none)" is displayed for all the elements that can be tracked. If you find markup, you can accept changes or delete comments as needed.

If no markup is found and the markup area is still being displayed, click on the Accept button in the Changes group and click Accept All Changes in Document. If Track Changes is turned on, turn it off.
