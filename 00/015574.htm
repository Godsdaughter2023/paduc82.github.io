ossiemac say:

I have interpreted your question a little differently to Ashish Mathur's reply.

My understanding is that you want the address of the cells immediately before and after the maximum value. Is this correct? If so then the following formulas where B2:B17 is the range containing the numbers.

Max-1:     =CELL("address",OFFSET(INDEX(B2:B17,MATCH(MAX(B2:B17),B2:B17,0)),-1,0,1,1))

Max:        =CELL("address",INDEX(B2:B17,MATCH(MAX(B2:B17),B2:B17,0)))

Max+1:    =CELL("address",OFFSET(INDEX(B2:B17,MATCH(MAX(B2:B17),B2:B17,0)),1,0,1,1))

ossiemac say:

Additional information to my previous post. If you want the result in relative format (without the $ signs) then the following formulas.

Max -1 :  =SUBSTITUTE(CELL("address",OFFSET(INDEX(B2:B17,MATCH(MAX(B2:B17),B2:B17,0)),-1,0,1,1)),"$","")

Formula in cell B18 - returns the value of the cell one above the Max value:

=INDEX(B1:B16,MATCH(MAX(B1:B16),B1:B16,0)-1)

Formula in cell B19 - returns the Max value:

=MAX(B1:B16)

=INDEX(B1:B16,MATCH(MAX(B1:B16),B1:B16,0)+1)
