i get error code 351, and what is it? - Microsoft Community

danieldolan say:

i get error code 351, and what is it?

how do i fix this error 351

 

a. user say:

Hi, Daniel,

 

System error code 351 means "The shutdown operation failed." This error code may also display as "ERROR_FAIL_SHUTDOWN" or as the value 0x15F.

 

The shutdown of the system may be caused by the user or by the system itself. The process can be initialized due to a system error or when completing an installation. Shutting down the system ends all processes. This is done one-by-one including the processes started by the users and the system processes. The error may happen if a program cannot be ended properly. In cases where the shutdown of the system was caused by another program the error may happen when the user continues to run programs or ends some of the processes needed by the shutdown.

 

When you look at the Error Code in Event Viewer, Click the Detail Tab.  This will list which process or program is causing the error if applicable.

techfreak_ say:
USB 1.1 has a full speed of around 12 Mbps (full speed). USB 2.0 has a high speed of around 480 Mbps. If the cable was only rated for USB 1.1 then the build quality might only ensure it functions at a lesser speed.  This is insufficient for the HD-6000 camera as it requires USB 2.0 transfer rates especially at higher resolutions (lower resolutions/preview modes may work intermittently).  Using a higher rated cable would be recommended.  The extender cable I am using is a USB 2.0 rated extender cable.

techfreak_ say:

I own several HD-6000 cameras as well as other LifeCam and Logitech cameras. The key thing to USB extender cables or any cable is the quality of the cable and its length. Sometimes the thicker the cabling, the higher the quality (not always as you cannot see inside the cable). It really depends on how well it is designed.  Usually high speed USB or fast charging rated cables have a higher quality build with possibly thicker /multiple internal cables and better grounding/return.  Slower transfer devices tend to function on lower quality and longer cables whereas higher speed devices will fail. I am currently using a 3 foot Logitech cable that originally came with a Logitech mouse and it works fine for me.  Another option is to use a powered USB hub to help maintain line speed and data/power stability. It may or may not help depending on the cable you are using.  Other times you might still need it even if the cable is half-way decent because the onboard USB ports may be insufficient in some capacity due to age or other quality-related reasons.
