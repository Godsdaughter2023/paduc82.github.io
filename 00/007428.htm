1. Messages were manually deleted either due to accidental deletion or were moved or deleted due to your account filter settings.

Autogenerate next tracking number - Microsoft Community

lords1979 say:

Autogenerate next tracking number

I have a word form I built with the embedded VBA utilizing an access database to pull in information via combo box.  As part of this form a user must enter a tracking number in the format "Julian Date"-00, where Julian Date is the current date in Julian date format and the 00 is a unique value for that day.  The information from the form is then added to fields within the document when user makes a change to appropriate combo box/text box etc. on form.  I am looking for a way to auto generate this number starting with 01 for that date.  As I am already connected to an access database, I am guessing the easiest way would be to create a table with a tracking number field, but not sure how to get from here to there.  Any help would be appreciated.

a. user say:

One approach you could try (I haven't used it, and it can probably be simplified with a bit more thought):

Suppose your database is at \\device\share\db.accdb. Or for this purpose, you could consider creating a separate database with just one table.

Create a table in your database called "tracking" with two fields.

Field 1 is called "daynum" and is numeric

Field 2 is called ID and is AUTOINCREMENT. Make sure the New Values property says "Increment" rather than "Random"

Mage field 1 + field 2 the primary key.

Initialise the table by creating 1 record with daynum = 0. Access will fill in the Autoincrement field automatically.

In a suitable VBA routine in Word, make a reference to the Microsoft ActiveX Data Objects 6.x Library (the version may differ on different versions of Windows/Office.

Use a routine along the following lines to create a new number and retrieve it:

Function getNextNum() As String

' Returns the next number for today's date starting at "01"

' Only allows for 99 per day

' Requires that all PCs have the same date and time.

Dim cn As ADODB.Connection
Dim todaynum As String ' date as string in "YYYYMMDD" format
Dim rs As ADODB.recordset

' Create today's date string

todaynum = CStr(Year(Date)) & Right("0" & CStr(Month(Date)), 2) & Right("0" & CStr(Day(Date)), 2)

Set cn = New ADODB.Connection
cn.ConnectionString = "Provider=Microsoft.ACE.OLEDB.12.0;Data Source=\\device\share\db.accdb;"
' You may need to deal with database security

cn.Open

' Create a new record using today's date. Jet/ACE will fill in the ID field with the next available number

cn.Execute "INSERT INTO tracknum (daynum) VALUES (" & todaynum & ")", , _
  adCmdText + adExecuteNoRecords

' Retrieve the most recently created ID using @@Identity, and the largest ID from the latest date

' before today
Set rs = cn.Execute("SELECT @@Identity,max(ID) FROM tracknum WHERE daynum < " & todaynum, , adCmdText)

' This gives us a sequence number for today

getNextNum = Right("0" & CStr(rs.Fields(0).Value - rs.Fields(1).Value), 2)
rs.Close
Set rs = Nothing
cn.Close
Set cn = Nothing
End Function
