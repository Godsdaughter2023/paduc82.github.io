Images not being displayed in Outlook 2003 messages.

Issue has to do with the Outlook Temporary Items folder.

Close Outlook before preceding to the steps below.

1. Locate the Outlook Temporary Items folder by opening the Registry (do Run in the start menu and then regedit) and finding HKEY_CURRENT_USER\Software\Microsoft\Office\11.0\Outlook\Security. Double click on the OutlookSecureTempFolder key. The value data is the path of the storage folder for picturse and starts with OLK. Change the OLK* to a number larger than the current value. (Best to increase the number by 1. i.e. OLK4 becomes OLK5)

2. Right click the Value Data field and select Copy. It should be in the form of %USERPROFILE%\Local Settings\Temporary Internet Files\OLK* where OLK* is some randomly generated string ending with a number that always starts with OLK.

3. Open Windows Explorer, paste in the Address field and press Enter (or select GO). You will be navigated to this folder.

4. Select All the files in this folder and Delete.

5. Launch Outlook. The images should now appear in the Email.

ISBLANK() will only return TRUE is the argument is REALLY blank.

ISBLANK() will return FALSE if the argument contains a null character.  For example, in A1 enter:

 

=""

 

then Copy A1 and Paste/Special/Values into A2.  Both A1 & A2 look blank, but they both contain a null character.  So both:

 

=ISBLANK(A1)

and

=ISBLANK(A2)

 

will return FALSE

Hello DMTodd

Just to clarify something - from your description, suspect that what you are really asking is to set Outlook so that it asks you which "Outlook Profile" to use as opposed to "which email account to log into".

For sake of accuracy, Outlook has no facility to select which email account to log into/use when multiple email accounts are configured in one profile.

So to answer your question regarding the laptop it would require the following

#1 - Create an Outlook profile and configure one email account in each profile.

#2 - In your existing environment, create a new Outlook profile and configure one of the two email accounts currently configured in the existing profile
#3 - When #2 is done, open the existing Outlook profile and remove the email account you added to the profile in step#2
#4 - When done, be sure to set having Outlook ask which profile you want to use when Outlook starts as per the attached screenshot

You can create/manage your Outlook profiles in one of two ways

a) - From within Outlook, go to File --> Account Settings drop down menu --> Manage Profiles
OR
b) - With Outlook closed, open Control Panel <Mail> app --> Show Profiles button

I am assuming you mean that obviously misspelled words are not being flagged. This is most likely a proofing language problem. The "proofing language" is a key to several Word features including spelling and grammar and AutoCorrect. It is not an application-wide setting nor even a document setting; it is set at the character level! To see if this is the source of your problem:
•    Select (or create and select) a misspelled word.
•    Then under the Review tab, click on the "Language" button or click on the language in the Status Bar.
•    Pick "Select Proofing Language" from the short menu.
•    Look at the bottom of the dialog that appears.
•    If the checkbox that says "Do not check spelling or grammar" is not empty, click on it until it is clear.
•    Then OK.

The read out loud button is grayed out in a specific document - Microsoft Community

sw111 say:

The read out loud button is grayed out in a specific document

The read out loud button is grayed out for part of a specific Microsoft Word document. Oddly, it worked for the first four pages of the 17-page document. And it appears to work for at least part of what follows the two paragraphs that won't work. I haven't tested it extensively yet.

What could be wrong? is there a fix?

jay freedman say:

Try this in a copy of the document (keep the original in case something goes wrong).

Select the two paragraphs (including the paragraph mark at the end of the second paragraph; click the ¶ button so you can see the marks) that can't be read aloud and cut them to the clipboard. Then click the down arrow on the Paste button and choose the icon with the large letter A (the Text Only command). Now try the Read Aloud to see whether it reads those paragraphs.

If it does work, apply any style or direct formatting that was in those paragraphs in the original document. Test again to be sure Read Aloud is still working.

If the Text Only paste doesn't help, it's possible that there's some corruption in the original document. First make a new blank document. In the original document, select and copy everything except the last paragraph mark at the end of the document. Paste that into the blank document and test Read Aloud. If the original has headers and/or footers, recreate them (don't copy/paste) in the new document.
