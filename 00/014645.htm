But with some of the more sophisticated scam pages, you’ll have to right-click on the taskbar (or press Ctrl + Alt + Delete) to start Task Manager, and then end the browser process. And Microsoft Edge still isn’t handling these scam pages properly, because if you kill the Edge browser process with Task Manager, then you’ll have to restart Edge indirectly with a web search, and then right-click on the search page tab and select “Close other tabs” in order to remove the scam page.

 

If you’re able to escape from the Tech Support Scam site like this, then it was just locking up your browser with JavaScript, and didn’t actually install any malware – so using the malware-removal tools will be optional. These scam pages will naturally tend to pop up again whenever you revisit a site that’s hosting a compromised advertising domain, just because the compromised domain will get another chance to deliver a malware-site redirect, and not because there’s some kind of persistent malware involved – so the best way to deal with this malvertising issue is to just install an ad-blocker extension like uBlock Origin, which will selectively prevent any suspect domains from connecting to the host site.
