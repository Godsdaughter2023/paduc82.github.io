Sub CreateFormula()
    On Error Resume Next
    If IsNumeric(ActiveCell.Value) Then
        ActiveCell.Formula = "=" & ActiveCell.Value & "*1.6"

Date format changed in Excel for Mac 2011 - Microsoft Community

alfie_j_uk say:

Date format changed in Excel for Mac 2011

All of my date formats in Excel appear to have changed. All hyphens within dates have changed to the letter 'M'
e.g. previous format dd-mm-yyyy has become ddMmmMyyyy or mmm-yy is now mmmMyy

This only happened in the last couple of days so I don't know if it relates to the version of Excel (v14.3.8) or my Mac's OS - I just upgraded to 10.9, Mavericks.

I'm having to create a custom date format then reformat all my dates!

Anyone else experienced this?

Alfie 
alfie_j_uk say: Ok, I solved my own issue - it's because I changed the Date format in my Mac's System Preferences (System Preferences, Language and Region, Advanced, Dates). I'd chosen a yyyymmdd format (and deleted the separators '/') which caused the problem but no idea why it added the extra 'M's !

Clicking the Restore Defaults button then restarting Excel solved the issue.
