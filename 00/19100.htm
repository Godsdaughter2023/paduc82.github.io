karl timmermans [outlook mvp 2012-2018] say:
Would it be correct to assume that your contact folder is in the IMAP folder tree and the folder has the added description of "This Computer Only"?

Assuming the above is correct, categories wouldn't work as expected. However, with that said, strongly suggest that the contact/calendar folders be moved. Should you delete the IMAP account, all data in "This Computer Only" folders will be lost unless you explicitly manually export (backup) those folders to a PST file. Similarly, should the OST file where those folders are located, ever get orphaned (orphaned = separated from the Outlook profile/email account that originally created it), you would require a 3rd party OST2PST conversion utility to recover that data.

There are two solutions to avoiding the above, both of which support Categories

Option #1 - described in the following article

How to Protect Outlook '2013/'2016 Imap Contact/Calendar Data
https://www.contactgenie.info/how-to-protect-ou...

Option #2 -

- create a free Outlook.com email account
- configure this account in Outlook
- make the .ost file associated with the Outlook.com account the "default data file" for the profile (go to File --> Account Settings --> Data files tab). Changing the default data file is completely separate and distinct from setting a default email account
- copy the data in the "This Computer Only" folders to the same folders under the Outlook.com folder tree

Using option #2, allows the contact/calendar data to be available on any device where the Outlook.com account is configured. There is also no requirement to use the Outlook.com for email

adding or subtracting page numbers. HOW?

adding or subtracting page numbers in MS Word 13. HOW?
suzanne s. barnhill say:

You can use a formula field to adjust page numbers. For example, if you want the number printed on the page to be 2 less than the actual continuous page number, you can use the field { = { PAGE } - 2 }. Note, however, that this number will not be reflected in the TOC, if you have one, or in PAGEREF fields, both of which always report the actual value of the { PAGE } field.

You can change the starting number for a given section using the Format Page Numbers dialog; this requires inserting a section break at the point where you want numbering to restart. For more information on this, see http://wordfaqs.mvps.org/FrontMatterRibbon.htm.
