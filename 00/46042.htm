keydown  keyCode=9         which=9         charCode=0

keydown  keyCode=192       which=192       charCode=0

The folders and files that you see in the Start / All Programs menu are stored 2 separate locations.

C:\ProgramData\Microsoft\Windows\Start Menu\Programs folder.

 

This is the folder where shortcuts that appear in all user accounts, All Programs menu.

 

C:\Users\your name\AppData\Roaming\Microsoft\Windows\Start Menu\Programs folder. 


This is the folder where the shortcuts that only appear in the All Programs folder for your specific user account is located.
