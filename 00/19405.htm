andreas killer say:

 I have values in cells A1:I3 (27 cells) - a 3 x 9 matrix.

However, I would like it to go down the matrix (A1, A2, A3...I1,I2,I3).

It is possible to access each cell and read/write the values seperately, but much faster is to read all values into an array at once, use a collection to store the results and then generate the output.

Andreas.

Sub Test()
  Dim Data, Item
  Dim Temp As New Collection
  Dim i As Long, j As Long, m As Long
 
  With ThisWorkbook
    'Read in all values
    Data = .Sheets("Sheet1").Range("A1:I3")
    
    'By columns
    For j = 1 To UBound(Data, 2)
      'By rows
      For i = 1 To UBound(Data)
        If Not IsEmpty(Data(i, j)) Then
          'Split by line breaks
          Item = Split(Data(i, j), vbLf)
          'Remember the max. nr. of elements (Note: "Item" is zero based!)
          If UBound(Item) > m Then m = UBound(Item)
          'Store into the collection
          Temp.Add Item
        End If
      Next
    Next
 
    'Do we have some data?
    If Temp.Count = 0 Then Exit Sub
    
    'Create the output array and copy the values from the collection
    ReDim Data(1 To Temp.Count, 1 To m + 1)
    i = 0
    For Each Item In Temp
      i = i + 1
      'Copy the data
      For j = 0 To UBound(Item)
        Data(i, j + 1) = Item(j)
      Next
      'Fill unused fields with blanks
      For j = j + 1 To UBound(Data, 2)
        Data(i, j) = ""
      Next
    Next
    
    'Write into the sheet
    .Sheets("Sheet2").Range("A1").Resize(UBound(Data), UBound(Data, 2)).Value = Data
  End With
End Sub

If you don't like SUMPRODUCT then you can do it like this. With this formula it doesn't matter if 2013 is text or a number it will be counted

 

=COUNTIF('101'!C1:C324,"2010-11")+COUNTIF('101'!C1:C324,"2011-12")+COUNTIF('101'!C1:C324,"2013")

 

 

With the sumproduct formula this ISNUMBER(MATCH bit test every cell in the range for any of the 3 values and it returns an array of TRUE:FALSE etc. SUMPRODUCT can't work with TRUE ot FALSE so we use the -- to coerce TRUE into 1 and FALSE into zero and then sumproduct adds them up.

=SUMPRODUCT(--(ISNUMBER(MATCH('101'!C1:C234,{"2010-11","2011-12","2013"},0))))

Hi Mike,

From your issue description, I understand that the printer has stopped working as it showing offline.
 

Let’s start with some questions.
1. What is the make and model of the computer?
2. Have you made any changes to the computer prior to this issue?
 
I suggest you try the following methods to resolve the issue

 
Method 1.

Step 1:
The printer displays offline when the printer is not connected to the network or is turned off or has gone offline.
Verify that the printer is running and that all cables are secure. There might be a button to push to bring the printer online.
http://technet.microsoft.com/en-us/library/cc938175.aspx
You can follow this link as well.
http://windows.microsoft.com/en-us/windows7/Open-the-Printer-troubleshooter
 

Step 2:Set your printer as the default printer and see if it resets it to online status.

To select a default printer
1. Open Devices and Printers by clicking the Start button.
2. Right-click the printer you want to use, and then click Set as default printer. (You'll see a check mark on the printer's icon signifying that it's now your default printer.)
For more information visit the following link:
http://windows.microsoft.com/en-us/windows7/Change-your-default-printer
 

Method 2: Follow the link and check if it helps.
The new printer status is "Offline" after you delete a print queue and then restart a Windows-based computer
http://support.microsoft.com/kb/2027593?wa=wsignin1.0
 

Method 3: Next thing to check is if your printer is set to sleep, if the printer is set to sleep, It would appear offline.
 

a. Open Device Manager by clicking the Start button, clicking Control Panel, clicking System and Maintenance, and then clicking Device Manager. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
b. In the list of device categories, expand Ports.
c. Right-click the entry for the printer, and then click Properties.
d. In the printer Properties dialog box, click the Power Management tab, Uncheck the Allow this computer to turn off this device to save power check box, and then click OK.
 

Method 4: If the above method does not help, check for the following setting:
1. Open the Start menu and click on “Devices and Printers”. This will open a window with a list of the printers currently set up on your computer. Now click on the printer that you have under the "Printers and Faxes" icon.
2. Double-click on the icon of the printer you want to change to online. A pop-up window detailing all current print jobs will open.
3. Go to "Printer" in the menu bar of the pop-up window and uncheck "Use Printer Offline". This action will change the printer from offline to online.
 

Hope the above information helps.
