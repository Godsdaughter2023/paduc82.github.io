charles kenyon say:

Like you, I would use a print > scan to pdf (not save as pdf or print to pdf).

Otherwise it depends in large part how the redacting is done. If it is done through text boxes, shapes, shading or background the text is still there. If it is done through deletion, the redacted material is not present once the document is saved and closed. If done through deletion, I would copy the redacted text to a new document. This is a very laborious process. This all also assumes that track changes is not turned on.

If they really want a Word document, I suppose you could print, then scan into pdf, then convert to Word. This would require Word 2013 or later to directly convert the pdf to a Word document.

Remember, when you transfer the original word document, even with deletions, confidential information may be included in metadata. This includes headers and fooers, including ones that you may not be displaying.

Hi Steven,

With Word 2011 the Normal template is located here: 

/Users/<your login name>/Library/Application Support/Microsoft/Office/User Templates

With Word 2016 (Mac) the Normal template is located here:

/Users/<your login name>/Library/Group Containers/UBF8T346G9.Office/User Content/Templates

I don't recommend using the 2011 Normal template with 2016. I believe you will encounter issues if you do. I would make a copy of the 2011 version, giving it a new name. Then I would attach that renamed version to an open blank Word 2016 (Mac) document and use Organizer to transfer the Macros, AutoText, and Styles you want from the 2011 renamed version of Normal to the actual Word 2016 (Mac) template.

The function that attaches a different template to a document is the Templates and Add-ins function. The same function also opens the Organizer function of Word. You access the Templates and Add-ins function from the Tools menu on Word's Menu Bar.

Can you try a few things?

First run Disk Utility.app, located in your Applications/Utilities folder and use it to Repair Permissions on your Primary Drive. When that completes restart your computer.

Next run the Font Book.app program, which is located in your Applications folder. After it opens go to File > Validate Fonts and let it analyze all of them. If it finds problems you must resolve them. Usually that means trashing duplicate fonts, regardless if they are enabled or disabled. When that completes you need to cold start your computer. Which means a full power down, and then waiting 2 - 3 minutes before powering it back up.

When you then start Word, do so while holding down the Shift key and keep holding it down until the blank new document appears. Now try testing Word again. Does the problem still happen?
