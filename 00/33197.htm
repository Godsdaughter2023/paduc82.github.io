robert aldwinckle on forums say:
Is there another way to stop/start the Windows Search service

Could be a permissions problem.   Instead of using services.msc  (apparently unelevated) you could try in an elevated cmd window (aka Administrator Command Prompt) to do a stop.  Also do some sc commands to try to see what is happening.


sc  qc  wsearch


sc  config  wsearch  start= disabled


sc  queryex  wsearch


net  stop  wsearch


sc queryex  wsearch


I am suggesting before and after  queryex  specifically to see if there are any Flags changing?


You could use ProcMon to find out what is going on e.g. it could be as simple a filtering on Result Is Access Denied  to see a problem with that tool.


Note that my second command would disable the service and thus prevent restart once stopped.   The command to undo this and make it automatic is:


sc  config  wsearch  start= auto


net  start  wsearch
