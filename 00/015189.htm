Exporting records from Access that have been added or modified. - Microsoft Community

billfleming43 say:

Exporting records from Access that have been added or modified.

I want to create an export process that repeats, either scheduled or at will, an export of records in an Access DB that have either been modified or added to the DB since the last such export was executed. For example, we create a view of Contact records that shows 5 fields of data. We export them for subsequent import to an Excel spreadsheet.  During the next 15 minutes 2 new records are created 10 existing records are modified. When we run the export again, we want only the new and modified records that have occurred in the last 15 minutes to be exported. Can this be easily done?
john w. vinson say:

Access doesn't keep track when records are added, but you can easily do so yourself. You can add a Date/Time field, ModifiedAt say, to your table; set its Default property to =Now() to timestamp new entries. In the Form you use to update the table (and yes, you should use a Form, not edit directly in the table!!) you can put code in the Form's BeforeUpdate event to set the ModifiedAt field to Now as well. You can then use a Query selecting records over any desired time span.

You might also want to add a field ExportedAt; set it to Now in the code that you use to run the export using an Update query. With this, your query could find all records with ModifiedAt greater than ExportedAt to identify which records to export.

scottgem say:

As John said you have store when the record was created or modified to be able to do what you want. However I would do it slightly differently.

I would have 2 fields in your table: CreatedWhen and EditedWhen. Set the Default value of CreatedWhen at the table level to =Now(). On your Form use the BeforeUpdate event of the form to set EditedWhen to =Now().

Then I would have a one record parameters table with a field named LastExport. 

So the final pieces would be to create a query:

SELECT * FROM table 

WHERE CreatedWhen > DLookup("[LastExport]","tblParameter") OR EditedWhen >DLookup("[LastExport]","tblParameter");

Then use a TransferSpreadsheet method to export the file and afterwards run the following:

CurrentDB.Execute "UPDATE tblParameter SET LastExport = " & Now() &";",dbFailOnError)
