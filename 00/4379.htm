nancydocken say:

Can I use constants in a query?

Thank you in advance.

Nancy

tom van stiphout say:

By creating a public function in a standard module to return the value:

(in a standard module)

public const MY_INTEGER = 5
public const MY_STRING = "abc"

public function GetConstant(byval strConstantName as string) as Variant

select case strConstantName

  case "myInteger"
    GetConstant = MY_INTEGER

  case "myString"
    GetConstant = MY_STRING

end function

 

In your query:

select GetConstant("myInteger") from myTable

Try setting Work Offline before doing the Print Preview.   One possibility would be that the site doesn't want to let you print that way and so, if the Print Preview involves a check with the server it could be preventing the image from being shown again.   Setting Work Offline would be a way of trying to avoid that case; however, with IE9 it frequently just ignores the Working Offline state, goes ahead and does the request and *then* informs you (uselessly) that it has done that.   So to avoid that annoying "feechur" you might have to actually disconnect from the Internet to make sure that there is no interaction with the server during the Print Preview request.
