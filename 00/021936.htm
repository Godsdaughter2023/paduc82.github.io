Parallel page numbering in Word 2010 - Microsoft Community

basketcase24 say:

Parallel page numbering in Word 2010

We have documents with multiple attachments. We format each attachment as a section. The whole document is page numbered (page X of Y). However, each attachment section also needs to be page 1 of S, 2 of S, etc. 

I think there may be a way to do this with Formulas, but I'm not knowledgeable enough to figure it out. Anyone help?

basketcase24 say:

I asked the original question. Before I got the reply above, I figured out a way to do this. I am using Word 2010 with Windows 7.

Our documents require "Page X of Y" for the whole document in the header.

For attachments, each attachment needs "page s of S" for that section, also in the header.

I inserted the field {SECTIONPAGES} for the number of pages in the section.

I set a bookmark in each section on the last page of each section. I bookmarked the section break, so it won't get accidentally deleted.

Then I used an equation to subtract the last page number of the previous section from the current document page number.  For the second section it looked like this when the field codes are showing:

{ = { PAGE \* Arabic \* MERGEFORMAT } - { PAGEREF Sec_1_end \h }}

Then for each subsequent section I just copied the whole phrase "Page s of S" into the header of the next session and changed the number; e.g., Sec_1_end became Sec_2_end.

I learned that you have to use Ctrl-F9 when adding each step of the equation. I started with Ctrl-F9, Insert Field PAGE, then Ctrl-F9, = (inside the first bracket), then Insert Cross-Reference to Bookmark page number (inside the last bracket), then added the "-" (minus sign) before the Bookmark page reference.

I think it was novice luck...
