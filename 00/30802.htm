net stop cryptSvc

net stop bits

net stop msiserver

ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

ren C:\Windows\System32\catroot2 Catroot2.old

  • Now, let’s restart the BITS, Cryptographic, MSI Installer and the Windows Update Services. Typethe following commands in the Command Prompt for this. Press the ENTER key after you type each command.

net start wuauserv

net start cryptSvc

net start bits

net start msiserver

  • Type Exit in the Command Prompt to close it and then restart the computer

Now you may try running the Windows Updates and check if the above steps resolve the issue.

  • Press Windows key +R to bring up the run command window.
