jeeped say:
If I am interpreting your problem correctly you need either,

    Worksheets(i).Range("B8").Formula = "='[stu.xlsx]" & i & "'!L11"
... or,
    Worksheets(i).Range("B8").Formula = "='[stu.xlsx]Sheet" & i & "'!L11"

It's a little unclear to me if your actual worksheet names are Sheet1, Sheet2 and Sheet3 or 1, 2 and 3. Note that the single quotes start before the first square bracket wrapping the external workbook name and finish just prior to the exclamation mark separating the worksheet name and the cell reference. The first half of the formula uses i as the index ordinal (integer position) of the worksheet in the workbook receiving the formula while the latter half uses i as either the worksheet name or part of the name. Two different methods of referencing worksheets within a workbook that is a little confusing in your case.

Syncing does not happen between client devices, syncing happens between a client device and the server.  What syncs and what doesn't depends entirely on the type of email account involved; POP accounts don't sync anything without "fudging" the account settings; IMAP accounts sync folder structure an email content, but not contacts, calendar, notes, to-dos, etc.; Exchange ActiveSync (Hotmail/Live Mail/Outlook.com) and Exchange Server accounts sync everything.  What kind of account do you have?

jeovany cv say:

Hi pfjh1520

The solution to your problem requires

1-VBA macros,

2-or the use of Windows Task Scheduler + .vbs file + .bat file + .xlsm file

My approach is using the 1st one.

a). Save your workbook as macro enabled workbook "*.xlsm"

b). Insert a module in your VBA editor project and paste this 2 macros

Note: Portion of Code highlighted in RED is the one to change according to the path of the folder where you want save the files.

Sub DayBackUp()

Dim MyFilePath As String

Dim MyFileDate As String

Dim MyFileName As String

 '''This code line stores the path of the folder where you want save the files

''' Don't forget the backslash at the end.

MyFilePath = "D:\Save2\"

MyFileDate = Format(Date, "mm-dd-yyyy")

MyFileName = "Workbook_" & MyFileDate & "_Days"

ActiveWorkbook.SaveCopyAs Filename:=MyFilePath & MyFileName & ".xlsx"

End Sub

Sub NightBackUp()

Dim MyFilePath As String

Dim MyFileDate As String

Dim MyFileName As String

  '''This code line stores the path of the folder where you want save the files

 ''' Don't forget the backslash at the end.

MyFilePath = "D:\Save2\"

MyFileDate = Format(Date, "mm-dd-yyyy")

MyFileName = "Workbook_" & MyFileDate & "_Nights"

ActiveWorkbook.SaveCopyAs Filename:=MyFilePath & MyFileName & ".xlsx"

End Sub

************************************************************************

c). Paste this macro in the workbook code procedure

Private Sub Workbook_Open()

''This line will execute macro for Days BackUps

Application.OnTime TimeValue("08:00:00"), "DayBackUp"

''This line will execute macro for Nights BackUps

Application.OnTime TimeValue("20:00:00"), "NightBackUp"


End Sub

****************************************************************

Doing so, a copy of your workbook will be stored in the folder of your choice at the specific time you requested in your post.

Today's files will be named as

Workbook_03-31-2019_Days.xlsx

Workbook_03-31-2019_Nights.xlsx
