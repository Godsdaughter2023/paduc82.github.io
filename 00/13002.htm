4.       Ensure you got all your appointments in Calendar 1 which is your default calendar. And then right click on calendar 2 and delete it.

jim gordon mvp say:

While you can't set the default line style for comment boxes, you can use a macro to change all the comment lines on a worksheet to the format you like.


Sub FormatCommentLines()

 Dim oComment As Comment, i As Integer

 For Each oComment In ActiveSheet.Comments

    Selection.ShapeRange.Line.Weight = 3#

    Selection.ShapeRange.Line.DashStyle = msoLineRoundDot

    Selection.ShapeRange.Line.Style = msoLineSingle

    Selection.ShapeRange.Line.Transparency = 0#

    Selection.ShapeRange.Line.Visible = msoTrue

    Selection.ShapeRange.Line.ForeColor.SchemeColor = 14

    Selection.ShapeRange.Line.BackColor.RGB = RGB(255, 255, 255)

 Next

 End Sub
