leonard_su say: Hi maxla,

The best way to delete the duplicate photos in Windows Live Photo Gallery is by selecting all the duplicate photos in the same folder and manually deleting them. Please refer to the steps below:

1. Open Windows Photo Gallery to view your photos.

2. Find the location of the folder where the duplicate photos are saved. Scroll through all of your pictures or click to view pictures in a specific folder.

3. Right-click on the duplicate photo. Select Delete from the menu that opens. Click Yes when you are asked if you are sure you want to move the photo to the Recycle Bin. The duplicate photo will be deleted.

4. Click CTRL and select each duplicate photo you want to delete if there are multiples in the same folder. The photos will be highlighted; right-click over the photos and click Delete from the menu that opens. Confirm to send the items to the Recycle Bin.

5. Repeat this process if you have any more duplicate photos you want to delete in other folders.

But if you're referring to the option that automatically detects all duplicate photos and delete them, that option is currently unavailable in Windows Live Photo Gallery.

Dell Digital Delivery is a service which enables you to purchase software at the time you order your computer and then download and install it automatically once you receive the computer and connect to the internet.

When your new computer is powered on for the first time, you will be walked through the regular computer setup including connecting to the internet. The second time the computer is powered on, the Dell Digital Delivery helper application already installed on your computer will run automatically. Such software is installed by the manufacture when delivering the PC to the end user.

   D) Click on "Repair your computer" then select the operating system to repair.

syed jamaluddinkamal shah say:

can system image of windows 7 backup and recover from a usb drive

I tried DVDs to burn for such task but its not working and consuming long time the i try to backup it on my usb drive the i got a massege "this is not a valid backup drive so plz help me if I backups system image on a network computer then copy it on my usb will windows recover from that usb? tryx3 say:

1  If you have a big enough USB then yes you can put an image on any other NTFS USB / hard drive then move it as long as it is kept intact & the entire folder structure is put into the USB drive's root level before you try to restore from it e.g. J:\WindowsImageBackup or H:\WindowsImageBackup.  You cannot tamper with any of its subfolders as that can corrupt the image.  Personally, I have never created an image onto a network drive but Windows Help says that this is okay [I have a feeling I have seen reports of people having problems with saving images in network locations but I don't have any details noted down].

2  My Win7 Home Premium x64 system image takes up 29-33GB depending what mood it is in so you will need a big USB for this job.  I use my 500GB external drive so I have never done any experiments to find out how small a drive I could use.  Your datafiles will add to the size of your image - my 29-33GB is just Windows 7 & applications since my datafiles are on a different partition.

3  It is possible that the error message you had was caused by the USB being too small for the job or not being NTFS-formatted - try a larger USB / external drive & it may work but do check that the external drive or USB stick is formatted using NTFS.

4  Your Dell also has a Factory restore partition & the backup utility bundled by Dell can create an image of that partition as an extra insurance against drive failure.  I used a 16GB USB stick for this but I think I could have got away with just 8GB.

5  Please be aware that both Windows system images & the Dell Factory restore partition have problems restoring to different hardware so if you ever have to change hard drives [if your internal hard drive fails or you just decide to get a bigger one], you may have to reinstall Windows from scratch.  You can try the system image but it may not work correctly.  You should therefore regard a system image as being a method of recovery from software corruption of your system [by far the most likely event] rather than protection against hardware failure.  There are other applications such as Acronis TI that make images that can cope with hardware changes.

6  You can also make an installation DVD/USB yourself. Download the correct ISO from Official Windows 7 SP1 ISO from Digital River then transfer it to a DVD or USB using the Windows 7 USB-DVD download tool The installation DVD/USB can also be used as a Windows repair disk for repairs & for restoring system images.

foxfifth say: Note that according to the following you can "manually" create a system image on a USB flash drive if it is formatted to NTFS:  http://windows.microsoft.com/en-US/windows7/Back-up-your-programs-system-settings-and-files

Also note that Home Premium and lesser versions will not backup to a network location via Windows 7 Backup and Restore.
