This typically indicates a Windows Installer error.  The most likely cause is remnants from prior installtions of other AV software products on the computer that were not completely (or properly) removed using appropriate removal tools, but it could be an issue with Windows Installer itself, so let's address that first as it's a simple procedure and may quickly resolve the problem.

 

*********************************************************************

 

  • Click “Start” >> Run (or Start / All programs / Accessories / Run
  • Type msiexec /unreg in the Open box, and press ENTER
  • Again click Start >> Run (or however you got there before)
  • Type msiexec /regserver in the Open box, and press ENTER

 

That is all. Does that resolve your windows installer error and solve your problem?

 

*******************************************************************
