bob jones aka: cybertaz say:

As for the missing balloons, there are a couple of possibilities;

  1. It sounds like the document simply may be displayed in Draft View, which does not show balloons. Go to the View Menu & select Print Layout (or use the button on the Status Bar to select it).
  2. If the balloons still don't display, go to the Review tab of the Ribbon to confirm that All Markup is set as the Display for Review mode & that the correct settings are checked in the Markup Options.

The markup in the document will be removed when the changes are Accepted/Rejected. Make sure that Track Changes is turned Off, then use the Accept/Reject features on the Review tab to resolve the changes... simply choosing other settings as described in #2 above does not remove the markup, they simply hide it.
