This can be caused if Active X control is blocking the upgrade. I suggest that you log on to the Genuine Microsoft Software – Diagnostic Site to check if scripting is turned on.

Genuine Microsoft Software

 

Method 1: If ActiveX is disabled, you may do the following.

 

a.        Launch Internet Explorer.

b.       Click "Tools" from the toolbar and select "Internet Options."

c.        Click the "Security" tab and then click "Internet" in the zone box. Click the "Custom Level" button to launch the Security Settings window.
