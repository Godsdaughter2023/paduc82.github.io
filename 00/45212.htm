Step 1

Make sure that Windows Installer Service is set to automatic and started. Later try to install the programs and check if it installs fine. To do this, follow the steps below:

  1. Click on Start.
  2. In the start search box type services.msc and press enter
  3. In the services Window, scroll down to Windows installer.
  4. Right click on it and click on Properties.
  5. Check if the startup type is set to Automatic, if not set it to automatic.
  6. Click on start button, this will start the windows installer services.
  7. Click on Apply and Ok.

 

Step 2

 

If the above steps do not help, you may unregister and reregister windows installer service on your computer and check if it helps you fix the installation issue.

a.   Click on the Start menu, click onRun.
b.   In the Open box, typemsiexec /unreg, and then press ENTER.
c.   On the Start menu, clickRun.
