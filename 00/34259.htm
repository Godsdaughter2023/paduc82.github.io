¡firedog say:

Please check all your account settings, to ensure that they conform with TWC's recommended ones: right-click on the account name in the folder pane and select

 

  • On the Servers tab 
     
    • Ensure that both server names are mail.twc.com 
                   
    • Ensure that the Email username is your full RR email address
       
    • Select Log on using clear text authentication
       
    • Select My server requires authentication
       
  • On the Advanced tab 
     
    • Ensure that This server requires a secure connection (SSL) is not selected for both incoming and outgoing mail.
       
    • Ensure that the server port for outgoing mail is 587
                           
  • Click Apply and  

 
Remove any unsent messages from the outbox (below the account folders in the folder pane), then compose a new test message with a unique subject and send it to yourself. Post any new error message in your reply. You can click on an error message to select it, then use right-click to copy it and paste it into the text entry box.
