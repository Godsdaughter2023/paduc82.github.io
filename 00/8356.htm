Can I create a consolidated view in Microsoft project 2013? I knew - Microsoft Community

ronald warrenfeltz say:

Can I create a consolidated view in Microsoft project 2013? I knew it was possible in 2007.

Can I create a consolidated view in Microsoft project 2013?  I knew it was possible in 2007.

john - project say:

Ronald,

Yes you can but be advised that creating a dynamic master (linked master/subproject structure) opens the door for file corruption. A better option is to create a static master (i.e. un-check the "link to project" option in the lower right corner of the Insert Project window). A static master is a snapshot in time and has to be re-created anytime an update of all files is needed but that can be automated by recording a macro the first time the static master is created. Thereafter, just run the macro.

Hope this helps.

John

dale howard [mvp] say:

Ronald --

In addition to the excellent answer given by John, I personally like to use a temporary master project.  To create a temporary master project, complete the following steps:

  1. Open each of the projects you want to include in the temporary master project.
  2. Click the View tab to display the View ribbon.
  3. In the Window section of the ribbon, click the New Window button.
  4. In the New Window dialog, select all of the open projects, select the Gantt Chart view in the pick list, and then click the OK button.

Now, work with the master project and its subprojects the way you would do normally.  When you close the master project, however, click the No button to save the changes to it.  Microsoft Project will ask you if you want to save the changes to the subprojects.  Click the Yes to All button to save the changes to all of the subprojects.

Just an additional thought.  Hope this helps.
