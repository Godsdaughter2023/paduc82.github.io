1. Open Notepad.

2. Copy the following text into Notepad:

@echo off

taskkill /f /im explorer.exe

taskkill /f /im shellexperiencehost.exe

timeout /t 3 /NOBREAK > nul

del %localappdata%\Packages\Microsoft.Windows.ShellExperienceHost_cw5n1h2txyewy\TempState\* /q

timeout /t 2 /NOBREAK > nul

start explorer

@echo on

3. Click File and Save.

4. Choose a location to save the file.  Enter a “File name” of ClearTileCache.bat and select “Save as type” of All files (*.*).  Click Save.

andre da costa say:
Open Start > Settings > Personalization > Themes

Toggle off Start full screen.

andre da costa say:
Another thing you can do is try placing your cursor at the edge of the Start menu to resize it manually.

Run the Start menu troubleshooter:

http://aka.ms/diag_StartMenu

If that does not work:

Press Windows key + X
Click Task Manager
Select the Processes tab
Scroll down to Windows Explorer then select it
Click Restart

If that does not work:

uninstall your display driver:

Press Windows key + X
Click Device Manager
Expand Display adapters
Right click the current display adapter
Click Uninstall
Exit Device Manager
Restart
