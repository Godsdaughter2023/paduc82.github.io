Thanks for the responses, I understand you have loaded XP on the system at this time.  If you do choose to go back to Windows 7, it sounds as though you may have encountered at least a couple of different issues.

1.  Possibly the Driver for the DVD ROM should have been reinstalled using the Troubleshoot Compatibility mode.  I would download the most recent driver for the DVD onto the desktop of the system, if it's not Windows 7, then at least XP and written in the correct bit ---X86 or X64--of what you are running-----browse to the folder you have the driver downloaded, right click the file that is the equivalent to SETUP.EXE and Select TROUBLESHOOT COMPATIBILITY.  It will run the Troubleshoot Compatibility TOOL and then give and Option to TRY WINDOWS RECOMMENDED SETTINGS if it finds a possible workaround.  -------This is if you are actually seeing the DVD drive in Windows Explorer, because from the sound of the information provided you were also encounter the Upper and Lower Filters needing to be reset. ---I'll get to that in a moment.

2.  Next the original issue with the DVD freezing sounds as though it may have been a CODEC's issue.  A codecs is the COMPRESSION/DECOMPRESSION of the file that is being played.  There are many CODECS available to all kinds of media.  If that occurs again, check the file extension of the file you are playing and try to download the latest codec for it.

3.  Last, if you are having issues where the DVD viewed in Windows Explorer is the same as the one that was removed from the system previously, the UPPER AND LOWER FILTERS should be deleted.  Here are the steps for resolving that;

  1. Click Start

    , and then click All Programs.
  2. Click Accessories, and then click Run.
  3. Type regedit, and then click OK. If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.
  4. In the navigation pane, locate and then click the following registry subkey:
    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{4D36E965-E325-11CE-BFC1-08002BE10318}
  5. In the right pane, click UpperFilters.

    Note You may also see an UpperFilters.bak registry entry. You do not have to remove that entry. Click UpperFilters only. If you do not see the UpperFilters registry entry, you still might have to remove the LowerFilters registry entry. To do this, go to step 8.
  6. On the Edit menu, click Delete.
  7. When you are prompted to confirm the deletion, click Yes.
  8. In the right pane, click LowerFilters.

    Note If you do not see the LowerFilters registry entry, unfortunately this content cannot help you any further. Go to the "Next Steps" section for information about how you can find more solutions or more help on the Microsoft Web site.
  9. On the Edit menu, click Delete.
  10. When you are prompted to confirm the deletion, click Yes.
  11. Exit Registry Editor.
  12. Restart the computer

colo_queen say: Reverse Key or Reverse 10-key in data entry refers to the use of numbers within the letter keys. The original keypunch machines and early data entry terminals used what is now considered reverse key. The nine on the top row is the 0, U the 1, I the 2, O the 3, J the 4, K the 5, L the 6, M the 7, the command key is the 8 and the period key is the 9. To change from alpha to numerical the keyer holds down the shift key. Actually data entry is much easier with reverse key as it eliminates the transition from alpha keys to the numeric keypad.
