bossdweeb say:

There should be no problem upgrading to a 1080, but when the time comes you would be wanting to upgrade

to the new series card that will have been released by then. Power requirements have been falling on gpu's over the

last few series, but we won't know until the next series has been released if this trend will continue.

Nvidia's minimum recommended psu for a 1080 is 500W (under full load a 1080 uses 180W +\- (add cpu/HDD ), so your machines

psu should be adequate. 

- Due to games putting full load on the system for prolonged periods of time many gamers traditionally preferred a psu with 50-100W headroom as the psu will run cooler and last longer if not being pushed to it's limits (also to allow for an SLI upgrade).

However, with modern psu's this is not as critical as most quality psu's intended for gaming machines  will have enough Amps

on the +12V rail alone to equal, or nearly equal, the Watts shown on the psu's label,  eg- 41Amps yields approximately 500W,

so  +12V rail with 35 Amps/420W +/- or more on a quality psu should be adequate.

- Note that OEM builders unfortunately do tend to use cheap psu's .

It depends on how Windows 7 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

It it originally came with the computer, it is not transferable.

﻿
OEM versions of Windows 7 are identical to Full License Retail versions except for the following:
- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel
- OEM licenses are tied to the very first computer you install and activate it on
- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard
- OEM versions cannot be used to directly upgrade from an older Windows operating system

rohn007 say:
The activation issue has been covered.

In addition to that, the only time it is OK to transfer a image from one computer to another is if they are running the same hardware. Or very similar hardware. The reason is that hardware requires 'drivers' to run. Windows installation files/disks come with generic drivers for all of the general classes of hardware: audio, video, keyboard, mouse,network, etc. If those generic drivers have been removed by hardware maker and replaced by hardware specific drivers, when the image is moved to another computer it may not work because some key piece of hardware cannot talk to windows.
