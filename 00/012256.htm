rasyha say:

Try use this vba code

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
  Dim lZoom As Long
  Dim lZoomDV As Long
  Dim lDVType As Long
  lZoom = 100
  lZoomDV = 120
  lDVType = 0

  Application.EnableEvents = False
  On Error Resume Next
  lDVType = Target.Validation.Type
 
    On Error GoTo errHandler
    If lDVType <> 3 Then
      With ActiveWindow
        If .Zoom <> lZoom Then
          .Zoom = lZoom
        End If
      End With
    Else
      With ActiveWindow
        If .Zoom <> lZoomDV Then
          .Zoom = lZoomDV
        End If
      End With
    End If

exitHandler:
  Application.EnableEvents = True
  Exit Sub
errHandler:
  GoTo exitHandler
End Sub

locking cells with randomly generated codes - Microsoft Community

michelledavid1 say:

locking cells with randomly generated codes

Hello,

I used the rand function to create random generated codes, but want to be able to use the other cells on the sheet without changing the codes. Even when I supposedly lock the cells that have the random codes generated, the codes still change when I enter information in the other unlocked cell and press enter. 

gord dibben say:

If the locked cells contain a volatile function like RAND, they will still update when Calculation takes place.

The results of RAND formula must be converted to values to prevent recalculation.

Gord

michelledavid1 say: Thanks so much! How do I do that?

gord dibben say:

Assuming you entered a formula like

=RAND()*100     in A1  and copied down 100 rows

Select that 100 cells and Copy>Paste Special(in place)>Values>OK>Esc

Now you have static values.

Unlock all cells then select A1:A100 and lock them.

Now protect the sheet.

Gord
