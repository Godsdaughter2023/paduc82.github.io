suzanne s. barnhill say:

It's easy enough to format the Endnote Reference style to Hidden. It would be handy if the NoteRef field would reference the page on which the endnote is referenced; unfortunately, it references the page on which the note occurs, so that's no help. So you'd have to bookmark the endnote reference (or text adjacent to it) in the text, then insert a PageRef field in the note. You can do this using References | Captions | Cross-reference, inserting a cross-reference to "Page number" of a "Bookmark."

Note that cross-references do not update dynamically; you can update them manually with F9, or they will update automatically when the document is repaginated (for printing, for example).
