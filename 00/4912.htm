2 options.

Array formula (Ctrl+Shift+Enter) in cell D2:

=COUNT(1/FREQUENCY(IF(SUBTOTAL(3,OFFSET(A2,ROW(A2:A100)-ROW(A2),)),IF(B2:B100=C2,MATCH(A2:A100,A2:A100,0))),ROW(A2:A100)-ROW(A2)))

=SUM(IF(FREQUENCY(IF(SUBTOTAL(3,OFFSET(A2,ROW(A2:A100)-ROW(A2),,1)),IF(B2:B100=C2,MATCH(A2:A100,A2:A100,0))),ROW(A2:A100)-ROW(A2)+1),1))

Today when I tried opening one of those .docx files on my desktop computer Word 2013 crashes on startup, while at the bottom saying "Opening myTemplate.dotx". I can open those .docx file without problems in the web version of Word, and on my other computers (Laptop & Tablet) running Word 2013.

Another, maybe related issue is if I open the myTemplate.dotx file in Word 2013 I can't save it because it is "Upload failed: Locked for editing by Johan Falk". This issue is present on all three computers, all running Windows 8.1.

My question is essentially, how can I get Word 2013 not to crash anymore on my desktop computer and how to make the myTemplate.dotx file unlocked?

Steps I've done so far, with no difference at all:

- Online Repair Office 2013, resulted in an error message.

- Removed Office using the uninstall tool (http://support.microsoft.com/kb/2739501/sv)

- Reinstalled again from office.com/myoffice

- Ran the Onedrive troubleshooter (http://go.microsoft.com/?linkid=9835055)

Hello Johan,

You may save the document locally and open it on the computer.

You may go to Word, Options, Add-in(s), Templates and check the location of the template used by the document.

You may look for the event viewer logs and upload the error log to check and help you better.

You may follow the steps below to access the event viewer logs
Press the ‘Windows + R’ (Without the quotes) key on the keyboard.
In the ‘Run’ windows type ‘eventvwr’(Without the quotes) and click ‘Ok’.
