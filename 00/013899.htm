Whether you see one page or two in a document depends on the zoom and the screen resolution. The only way to force a particular number of pages and a particular zoom is to run a macro as you open documents.

Sub AutoOpen()
'Macro created by MVP Stefan Blom, March 2017
On Error GoTo errhandler
With ActiveWindow.View
.Type = 3  'Print Layout view
.Zoom.Percentage = 100  'the desired zoom
.Zoom.PageColumns = 1
End With
errhandler:
Exit Sub
End Sub

When you use SUMIFS, each criteria range must have the same number of rows and columns as the sum range or it returns a #VALUE! error. In your case the sum range is 3Rx4C, the second range is 3Rx1C and the third range is 1Rx4C.

Hi,

 

You mentioned that the fake account was associated to your email address, most likely the account was created by accidentally signing in to Skype using the Microsoft email address.

 

Signing in to Skype using the Microsoft email address will automatically create a Skype account for you and we call this a Microsoft Skype account. It has a “live:” at the beginning of the Skype Name.

 

Deleting the account with a “live:” Skype Name will also delete the Microsoft account email address. This is because technically, you’ve created the Skype account using a Microsoft email address.

 

To close the live account without affecting the other one, I advise you to merge the personalized Skype Name to a different Microsoft account first. The Skype account with a personalized Skype Name will be now a Microsoft account and you can sign in to any Skype services using the Skype Name. If you have an existing Microsoft account that you’re using, that’s much better. Click here to know more on how to merge Skype account to a Microsoft account.

 

After that, you can now go ahead and close the account that has a “live:” username by signing in to Microsoft website using the email address and proceed to the account deletion process.

 

Click here to know more about account deletion process.
