Further to John's reply: The actual constraint is that you cannot modify the properties of a toolbar while the bar is Docked... You can undock the Formatting toolbar but not the Standard toolbar.

Instead of formulas to put in another cell use a macro to replace the original with the new cell

Sub trimcell()
Dim c As Range
For Each c In Range("a1:a10")
If Right(c, 1) = "," Then
mc = 1
Else
mc = 0
End If
c.Value = Application.Trim(Left(c, Len(c) - mc))
Next c
End Sub

Sub trimcell()
