doug robbins - mvp office apps & services (word) say:

Use the following code:

Selection.HomeKey wdStory
Selection.Find.ClearFormatting
With Selection.Find
    Do While .Execute(FindText:="\[[0-9]{1,}\]", MatchWildcards:=True, Forward:=True, Wrap:=wdFindStop) = True
        With Selection
            .Text = Replace(.Text, "[", "")
            .Text = Replace(.Text, "]", "")
            ActiveDocument.Footnotes.Add Selection.Range, .Text, "Test" & .Text
            .Delete
        End With
    Loop
End With

gininesmith say:

Automatically create new Tabs from List in sheet 1

Hi

I have a list of people names in sheet 1.

I'd to automatically create a new sheet in the workbook for each - with their name on each tab.

mike h.. say:

Hi,

ALT+F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in on the right.

Select the names and run the code.

Sub addsheets()
Dim c As Range
For Each c In Selection
Worksheets.Add(After:=Worksheets(Worksheets.Count)).Name = c.Value
Next
End Sub

If they’re not in the trash, then there’s no way to recover them just like that.
