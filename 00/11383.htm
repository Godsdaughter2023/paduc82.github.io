I'm prettty sure you cannot change the default setting, but the following code changes all the series on the active chart to line thickness of 1.5.

Sub FormatAllSeries()
    Dim SeriesNum As Series
    With ActiveChart
        For Each SeriesNum In.SeriesCollection
            SeriesNum.Format.Line.Weight = 1.5
        Next
    End With
End Sub

Refer below image:

Some options to exclude items in C2:C4:

Formula in cell D2:

=SUMPRODUCT(($A$2:$A$100<>"")*ISNA(MATCH($A$2:$A$100,$C$2:$C$4,0)))

Formula in cell D3:

=ROWS($A$2:$A$100)-SUM(COUNTIF($A$2:$A$100,{"","Orange","Mango","Apple"}))

Formula in cell D4:

=COUNTIFS($A$2:$A$100,"<>Orange",$A$2:$A$100,"<>Mango",$A$2:$A$100,"<>Apple",$A$2:$A$100,"<>")
