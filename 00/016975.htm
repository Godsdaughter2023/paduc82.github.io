stefan blom say:

AutoRecover has never been intended as a replacement for saving. If you save locally, you can use Graham Mayor's add-in to create backups. See http://www.gmayor.com/SaveInTwoPlacesAddIn.htm.

If you are saving to OneDrive, you do get automatic saves (with version history). Of course, there may be many reasons why you don't want to store documents in the cloud.

OneNote 2016 cannot insert screenshot - Microsoft Community

c hanmer say:

OneNote 2016 cannot insert screenshot

I was happily using OnNote 2013 on a Windows 10 laptop as part of the Home Office package but last weekend was upgraded to  Office 2016 and now seem to have a much inferior version of OneNote that looks like the app on my Anrdoid tablet - for example on the insert menu there is no option to insert a screenshot. How can I restore the functionality I am used to?  
c hanmer say:

I found the solution to this -there are 2 versions of Outlook, the full Outlook 2016 and a mobile one both installed on my laptop. For some reason after the upgrade to Office 2016 the default was set to the mobile version, I have now changed this and am back to the OneNote functionality I want.

**Proceed At Your Own Risk**

The issue sounds like it could be video driver related.

If you happen to have Automatic Windows Updates...it's
possible that an updated video driver was downloaded
and installed that is not as compatible with your system
as the one you already had.

It may be worth a try to so a System Restore to a time
before the issue began.
