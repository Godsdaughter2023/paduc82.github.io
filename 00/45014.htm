asunnyside say:

Which usb 2.0 converter cable will connect to a sidewinder 3D pro plus joystick with15 pin gameport?

I want to connect the joystick to a PC with no game ports, only USB,  running Windows 7 Home Premium.  I need to find a suitable converter cable, so i can check whether joystick will work. 
gndattveh say:

Sorry to resurrect this thread but I have been researching this recently and cam across this.

The answer to this is a little complicated.

MS produced a joystick (follow up to the "3D Pro")  that was boxed as the "Precision Pro" however if you looked under the joystick the embossed name was "3D Pro Plus".  This joystick is not easily USB compatible.  The follow up joystick, that looked identical, was marked underneath as "Precision Pro" and this came with a simple USB adapter (simple as just a series of wires links between the Joystick Game Port and a USB).

Obviously care needs to be taken when buying either of these second hand as most sellers describe the item based on the name used on the box which was "Precision Pro".  Though I believe the latter unit had USB compatible written on the box.

The individual units can be identified by the embossed name on the underside of the unit and the first 5 letters of the Product Id (also on the underside):

Product ID: 97482-xxx is a "3D Pro Plus" - NOT USB compatible.

Product ID: 85791-xxx is a "Precision Pro" - USB compatible with a simple adapter.

The follow up to the "Precision Pro" was the "Precision 2" was only USB but this was a new physical unit and was alleged to suffer more static build up problems than its predecessor and had the shift button removed.

All these units would have been USB 1.x which should be usable on a USB 2.0 connector.

a. user say:

Hi Leigh,

Thank you for posting in Microsoft Community.

You can follow the steps to do the changes from registry.

  1. HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services this path should contain gpsvc key (a folder), which is responsible for service parameters and configuration.  I found that the key wasintact, so, you do not touch anything here.
  2. HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SVCHOST This is the most important path you should look into, as it must contain the keys and values referred in the key #1.  Below are descriptions what must be present there.
  3. There must be Multi-String value called GPSvcGroup. My laptop was missing it.  So, you should create multi-string value named GPSvcGroup and assign it value GPSvc.
  • Next, you must create a key (a folder) and name it GPSvcGroup
  • Then open newly-created GPSvcGroup folder and create 2 DWORD values:
  1. First called AuthenticationCapabilities and you must give it a value of 0x00003020 (or 12320 in decimal)
  2. Second is called CoInitializeSecurityParam and it must have value of 1.

Once you completed all steps above, reboot the computer.
