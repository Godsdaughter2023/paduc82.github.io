You will be able to revert back to the previous operating system only if you have windows.old folder in C: drive and within 30 days of upgrading to Windows 10.

 

When you upgrade to Windows 10, the previous Windows files and folders are automatically stored in C:\Windows.old folder. This folder is kept by Windows only for one month. keyboard stops working at winows setup - Microsoft Community

headache tablet say:

keyboard stops working at winows setup

I'm trying to install Windows XP Tablet PC Edition 2005 on a replacement HDD in a Samsung Q1 Tablet using the supplied Samsung Recovery Media disc. The usb keyboard works OK at BIOS and USB Legacy Support is enabled in BIOS; however the keyboard stops functioning at the Windows Setup stage when being prompted to install windows or create/delete partition, and I am unable to proceed further as there is no touchscreen option. Does anyone know how to correct this?

headache tablet say:

I am pleased to report that I was able to sort out the problem by trial an error; and the answer for anyone else encountering the same problem with a Samsung Q1 tablet is as follows:-

 

DON'T USE THE DEDICATED SAMSUNG KEYBOARD AA-SK0TKBD/UK WHEN TRYING TO RESTORE THE O/S ON A SAMSUNG Q1 NP-Q1-V000/SUK, AS IT IS SEEN AS A 'HUMAN INTERFACE DEVICE' AND THE USB KEYBOARD DRIVER DOESN'T RECOGNISE IT. INSTEAD YOU SHOULD USE A STANDARD USB KEYBOARD ALTERNATING IT WITH A STANDARD USB MOUSE VIA THE RIGHT HAND USB PORT WHEN REQUIRED TO DO SO. AFTER THE INSTALLATION OF THE O/S THE SAMSUNG KEYBOARD WITH ITS' INTEGRAL JOYSTICK MOUSE IS THEN RECOGNISED.
