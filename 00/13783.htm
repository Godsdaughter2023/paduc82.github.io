jay freedman say:
Choosing the proper Paste As variant as John suggests will help, but it probably won't cure the "newspaper column format" you mentioned. That's usually caused by line breaks or paragraph breaks that come with the pasted text, not by "margin" settings. The article http://wordfaqs.mvps.org/CleanWebText.htm describes the steps to remove those breaks, after which the text will obey the margins you set in Word.

amy baggott say: The problem is that Access 2007 and 2010 do not recognize the mdw security.  I had the same problem when we converted our Access 2003 database to Access 2010; all of my database objects were looking for permissions that the database no longer knew how to read.  I solved it by creating a blank Access 2010 database and importing all my objects into it, which stripped all the old Access 2003 security information out of them.  Now I don't have any problem.
