jay freedman say:

As usual when a dialog or other feature doesn't work exactly as you want, the alternative is a macro. Install the following macro and assign a keyboard shortcut, Quick Access Toolbar button, or Ribbon button to run it.

Sub SavePDF_NoResult()
    With Dialogs(wdDialogFileSaveAs)
        .Format = wdFormatPDF
        SendKeys "%e": .Show
    End With
End Sub

The statement SendKeys "%e" sends the keystroke Alt+e to the dialog, because Alt+e is the accelerator for the label "Open file after publishing". [To use this macro in languages other than English, replace the letter e with the character that's underlined in your version of Word.]

doug robbins - mvp office apps & services (word) say:
Use File>Info>Inspect Document>Check for Issues.

jay freedman say:

First, to be notified that there are comments or tracked changes in a document, check this box in Word's Options > Trust Center > Trust Center Settings > Privacy Options page:

If you don't remove comments and tracked changes from a document, regardless of whether you can see them, they will remain in the document and can be displayed by anybody you send to. Not selecting "All Markup" merely hides them. 

One way to remove comments is to go to the Review tab, click Show Comments, click the down arrow on the Delete button, and click Delete All Comments in Document.

(The Delete button will be grayed out if there are no comments in the document.)

To remove tracked changes, click the down arrow on either the Accept button or the Reject button and choose one of the All Changes items.

To remove both comments and tracked changes at once (and possibly other things you don't want to send out), click File > Info > Check for Issues > Inspect Document:

In the dialog that appears, check or uncheck the boxes to specify which items to remove and which to keep; then click the Inspect button. If the inspection report indicates that there are comments or tracked changes, you can click the Remove All button and then save the document.
