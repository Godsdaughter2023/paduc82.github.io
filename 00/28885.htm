This is a bug with Win8 and Publisher files. It appears that text is rendered differently on Windows 8 machines. It has been reported that you will get about 5% more text in each box although all the settings for font size, spacing, etc are all the same as before. The only workaround reported thus far is to replace the Win8 fonts with the fonts from the older OS. The examples provided were with Win7 but might also work with XP.

In your case you might want to just create your PDF on the Win8 machine and transfer it to the XP machine for printing, rather than transfer the Pub files. I would think this the easiest solution short term. Long term you will be retiring the XP machine anyway and then you can tweak the layout of your old Pub files on the new machine.
