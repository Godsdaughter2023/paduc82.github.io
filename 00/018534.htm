hansv mvp say:

Let's say you want the date in cell B3 on Sheet1.

Press Alt+F11 to activate the Visual Basic Editor.

Double-click ThisWorkbook in the project explorer on the left hand side.

Create the following code:

 

Private Sub Workbook_Open()
    With Worksheets("Sheet1").Range("B3")
        If .Value = "" Then
            .Value = Date
            .NumberFormat = "m/d/yyyy"
        End If
    End With
End Sub

 

You must save the workbook as a macro-enabled workbook (.xlsm) and the user must allow macros to run.
