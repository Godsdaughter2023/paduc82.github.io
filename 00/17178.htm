allen z. msft say:
Hi Nadeem,

I see that you have created both Office 365 cloud users and on-premises AD users. Yes, they can match with each other, as long as one cloud user and one on-premises user have the same SMTP address. This is what we call SMTP match or soft match. And you can see more details in How to use SMTP matching to match on-premises user accounts to Office 365 user accounts for directory synchronization.

Generally, running an SMTP match will not affect the services provided by Exchange Online and Skype for Business Online. Please just be aware that one user cannot have one Exchange Online mailbox and one Exchange on-premises mailbox at the same time. If you don’t have an Exchange on-premises environment, you don’t need to worry about this part.
