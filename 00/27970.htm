jedisql say:

If some sort of VBA would be needed, it would need to be something that could be hidden in a template and used by people that have no clue about VBA.

 

Our office uses a mix of Office 2007 and 2010.

jay freedman say:
Recognizing that Word is not a web/HTML kind of application, the one kind of object in Word that can reliably display a tooltip -- especially if the tooltip must be different from the text -- is a hyperlink. A cross-reference field with the \h switch acts something like a hyperlink, but its tooltip is always the same as its text.

With some planning ahead, you can use the macro below to create the hyperlinks you want. This macro does not have to be distributed with the document -- you just run it once before you distribute it to the world at large. If you edit the document to add more glossary entries, you can run the macro again to make hyperlinks for them.

The up-front work is this:
  • Create a custom character style. Name the style "Gloss" (or, if you give it a different name, change the expression ActiveDocument.Styles("Gloss") in the macro to match). The macro will remove the instances of this style and replace them with hyperlinks. It would be best to give the style a color that stands out, so you can see places that still need to have hyperlinks added to them.
  • Apply the character style to each place in the text where a glossary term appears and you want the hyperlink and its popup.
  • Save the document.

With the document open, run the macro to create the hyperlinks. Afterward, scroll through the document to see if any instances of the character style still appear; that might happen if the text in the style doesn't match any entries in the table, maybe because of a typo.


If the glossary table isn't the first table in the document, change the number in the expression ActiveDocument.Tables(1) to point to the correct table.


Sub MarkGlossaryEntries()
    Dim tbl As Table
    Dim rg As Range
    Dim tblRow As Long
    Dim rgDef As Range
    Dim tip As String
   
    If ActiveDocument.Tables.Count > 0 Then
        Set tbl = ActiveDocument.Tables(1)    ' <== use the correct number
    Else
        MsgBox "No table found in this document.", , "Error"
        Exit Sub
    End If
   
    Set rg = ActiveDocument.Range
    With rg.Find
        .Format = True
        .Style = ActiveDocument.Styles("Gloss")     ' <== use the correct style name
        .Wrap = wdFindContinue
        While .Execute
            tblRow = GetRgTbl(tbl, Trim(rg.Text))
            If tblRow <> 0 Then
                Set rgDef = tbl.Rows(tblRow).Cells(2).Range
                rgDef.MoveEnd wdCharacter, -1
                tip = rgDef.Text
                rg.Style = ActiveDocument.Styles(wdStyleDefaultParagraphFont)
                ActiveDocument.Hyperlinks.Add _
                    Anchor:=rg, _
                    Address:="", SubAddress:=rg.Text, _
                    ScreenTip:=tip
            End If
        Wend
    End With
End Sub

Function GetRgTbl(tbl As Table, txt As String) As Long
    Dim testRg As Range
    Dim testRow As Long, aRow As Long
   
    aRow = 0
    ' note that the function may return 0 (not found) as its value
   
    For testRow = 1 To tbl.Rows.Count
        Set testRg = tbl.Rows(testRow).Cells(1).Range
        testRg.MoveEnd wdCharacter, -1 ' exclude cell marker
        If LCase(testRg.Text) = LCase(txt) Then
            ActiveDocument.Bookmarks.Add Name:=txt, Range:=testRg
            aRow = testRow
            Exit For
        End If
    Next
   
    GetRgTbl = aRow
End Function

If .Execute doesn't return True, that means the Find operation couldn't find any text in the document that is marked with the style named in the macro (where the comment "use the correct style name" points to).

How do I do this.

charles kenyon say:

See Portrait and Landscape in the same document

Select the text on the page you want to be different.

Go to the Page Layout Tab

Click on the dialog launcher for the Page Setup Dialog (bottom right corner of Page Setup group on ribbon).

On the margins page of the dialog, change "Applies to" to read "selected text."

Click on landscape and then OK.

That inserts two next-page section breaks around your landscape page. It is now a separate Word "section."

If you have headers or footers, you may want to use alignment tabs to position text horizontally.
