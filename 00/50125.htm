a. user say:

Hi,      

Thank you for posting your query on Microsoft Community. Let

me assist you.

I understand you would like to use your personal signature on the emails sent by the Windows Mail App.

'Sent from Mail for Windows 10' is the default signature on the app and you can change it according to your preference. I suggest you to refer the steps below to change the signature on the Mail App and check if it helps.

Steps:-

1. Open Mail App.

2. Click on Settings (gear symbol at the bottom left)

3. Go to Signature.

4. Type you preferable signature on the dialogue box and turn the Use signature On.

5. Click on the check box beside Apply to all accounts.

6. Open a new mail and check if the change reflects.

Possible fix for Microsoft Wireless Display Adapter - Worked for me - From Windows 10 build 14295 for PC and Mobile Release

Saw the known issues list had this in it.  So I thought I would try it on my released to public version and found the same .173

Trying to connect to Miracast will fail. The workaround to get out of this state is to go to this registry key HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\SharedAccess\Parameters, select StandaloneDhcpAddress, and change the value from 192.168.173.1 to 192.168.137.1 (value type is REG_SZ).
