milanovic milan say:

I had this same problem. I solved it by deleting Personal.xlsb file. I was using Windows 8.1 and Office 2013.

All recorded macros that should be always accessible are placed in Personal.xlsb file. Deleting this file could mean that you are going to lose your recorded macros. Before deleting this file you could copy your recorded macros to make a backup.

These are some location where Personal.xlsb is located:

Win7

C:\Users\AppData\Roaming\Microsoft\Excel\XLSTART\PERSONAL.XLSB

Win XP

C:\Documents and Settings\Application Data\Microsoft\Excel\XLSTART\PERSONAL.XLSB

Win8.1

C:\Users\User\AppData\Roaming\Microsoft\Excel\XLSTART\PERSONAL.XLSB

You can find location for Personal.xlsb by typing "? Application.StartupPath" in immediate window in VBE.

Personal.xlsb file will be recreated next time you record another macro and save it in "Personal macro workook".
