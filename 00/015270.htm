I have found creating a batch .cmd file that deletes the registry key works better.
The registry key gets created again when checking for updates so doing it twice worked for me.

Instructions:
1. Click File Explorer on the taskbar, click view, tick [/] Hidden Items
2. Navigate to your Desktop, right-click New > Text Document
3. Open "New Text Document.txt" and copy & paste the lines in black below

(they are two command lines the first command ends with /f)

and save the file New Text Document.txt

reg delete "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate\Auto Update\RequestedAppCategories\8b24b027-1dee-babb-9a95-3517dfb9c552" /f
pause

4. Right-click New Text Document.txt, click rename.
5. Highlight all "New Text Document.txt" rename file to fix.cmd (not fix.txt.cmd but fix.cmd)
6. Right-click fix.cmd, run as Administrator
7. It should say "The operation completed successfully"
8. Press the [x] to close the command

9. Click start, settings, update & Security
10. Click [Check for updates]
11. Go back to your desktop & Right-click fix.cmd, run as Administrator again
