khernn say:

windows\system32\config\system is corrupt or missing

I rebooted Windows XP Home edition using the operating system software/setup CD.

the sasquatch say: Boot off your Windows XP CD and go into the Recovery Console - follow this document to do it http://support.microsoft.com/kb/307654

go to the command prompt.

Then change to the C: by typing C: and hit enter.



Then type CD\SYSTEM VOLUME INFORMATION and hit enter

Then type DIR /AH and hit enter (This will show you directories even if they are hidden directories)

This will show a few directories... one in particular starts out with the directory name being _RESTORE{ There will be a long series of random letters and it ends with }

Type CD _restore{then the random set of letters, dashes and numbers shown in the directory listing from the command above and ending with the } then hit enter.

Then type dir and hit enter.

It will show a bunch of directories that start with RP - something like RP1366.

Look at the last RP directory entry.  If it is say RP1366... then look up a couple of directory entries..

Then type CD RP1364 of whatever restore point directory that is relevent to your directory entry.  Just do not do the last restore point directory - I would go back 2 or 3 restore point directories.

Then type DIR /AD and hit enter

This should show a directory called SNAPSHOT

Then type CD SNAPSHOT

Then type DIR and hit enter

You should see a few files but two in particular.

_REGISTRY_MACHINE_SYSTEM
_REGISTRY_MACHINE_SOFTWARE

Then type COPY _REGISTRY_MACHINE_SYSTEM C:\windows\system32\config\system and hit enter
Then type COPY _REGISTRY_MACHINE_SOFTWARE c:\windows\system32\config\software and hit enter

Then reboot your PC and see if you can log in.
