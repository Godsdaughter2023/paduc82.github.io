karl timmermans (outlook mvp 2012-2018) say:
>>
#1 - The new Outlook update does not allow changes to the email accounts as the older version.
#2 - I tried to change pop to imap and there is no provision for this.
<<

Two parts to your question so to answer this in reverse order

Re: #2 - Change from POP to IMAP

You can't simply change an account from one protocol to another. The old account needs to be removed from the profile and then re-added. For IMAP, just need to enter email address/password.

In your scenario, that will result in the PST file for the POP3 account that was removed to remain in the profile and also still be designated as the default data file for the profile (assuming that you only had one account configured). This is what you want so that your default contact/calendar folders remain in the PST file. Your IMAP folder tree will not have a contact or calendar folder in this instance.

Re: #1 - Email Account settings

What you're seeing is the "simplified" UI for email account settings.

To change your account settings - do the following

- Press CTRL+ALT+S (this brings up the Send/Receive Settings window)
- Select the group you want to edit and click on the <Edit> button
- Select the email account you want to modify --> click on <Account Properties>

You can get to the exact same place via File --> Account Settings --> Options --> Advanced --> click on the Send/Receive button

You can also use the Control Panel <Mail> app (with Outlook closed) - highlight the email account --> select <Change> (not applicable if you have the MS Office "Store" version installed)
