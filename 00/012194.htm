john korchok say:

Steps to open Customize Keyboard (for other readers):

  1. In Word 2007, click on the Office button, then on Word Options.
  2. Click on Customize, then beside Keyboards shortcuts, click Customize again.

To find the assigned shortcut for your macro:

  1. In Categories (left-hand list), scroll down to Macros and select it.
  2. In Macros (right-hand list), scroll down to your macro and select it.
  3. The currently assigned keyboard shortcut displays in the Current Keys field.
  4. If you click in the Press new shortcut key field and enter a key combination, you will see the current assignment for that combination below the Current keys field. If it just says Currently assigned to: followed by a blank, the combination is unassigned.

jay freedman say:

There's a little-known shortcut (!) to the Shortcuts dialog: With NumLock turned on, press the Ctrl, Alt, and number-pad plus keys together. The cursor becomes a clover-leaf shape. Now press the shortcut combination you're interested in -- e.g., Ctrl+2 -- and the Shortcuts dialog will open with the assigned command already selected (in fact, all other commands are grayed out). If the shortcut you press has no assignment, the dialog won't open.

While the clover-leaf cursor is in effect, you can also click any button or menu item to see its shortcut (if any is assigned). Press the Esc key to turn it off.

Hello,

Try booting into Safe Mode with Networking.  This runs minimal drivers and prevents malware from executing.

Start > Type MSconfig

Press Enter when you see it appear at the top of the menu

Click on Boot tab

Click on Safe Boot

Click on Networking

Click Apply, Ok

Reboot and the system will load into safe mode with networking

Download this Rootkit Remover from Bitdefender.  Save the file.

http://labs.bitdefender.com/projects/rootkit-remover/rootkit-remover/

Once you've downloaded the file, close open windows.  Locate the file > Right click on it > Select Run As Administrator

When finished, go back to MSconfig window (follow steps above)

Click on Boot Tab > Uncheck Safe Boot

Click Apply, Ok

Reboot and Windows will load normally

Let us know if this helps.
