Method 1: Turn off the Airplane mode

2. Click on Settings window 
3. Go to Network icon 
4. Choose Network & Internet category.
5. The second setting on the left column is Airplane mode.
6. Turn it Off

Method 2: Update the drivers.

 Step 1: Check for the latest driver update through Windows Updates.
1. At the Start screen, type Windows Update.
2. Click or tap Settings on the right side of the screen, and then select Install optional updates from the left results.
3. When the Windows Update control panel opens, click on “Check for Updates” on the top left corner to receive a fresh list of updates.

Step 2: Follow the steps below to update the Wifi drivers.
1. Press Windows + X key on the keyboard.
2. Go to device manager.
3. Right-click your Wireless/Wi-Fi devices.
4. Click Update Driver Software and then follow the steps in the wizard that appears.

Restart the computer and check.

Let us know how it goes.

No section breaks needed. Here are the steps:

  1. On your first page, insert your cover page photo. By default, it will be inline with text and will fill the width of the text area.
  2. With the photo still selected, click on the Picture Format tab and choose Position>More Layout Options.
  3. Click on the Text Wrapping tab and choose Behind Text.
  4. Click on the Position tab and set to the right of and below to Page. Click on OK.
  5. Choose Position>More Layout Options again, then select the Position tab again. Set both Absolute position fields to 0".
  6. Click on the Size tab and set the Absolute dimensions for Height to 11" and Width to 8.5". OK Out.

The photo should now fill the page and sit behind your cover page text.

jack cook - aka help_line say:

 
I've not tried ucweb or nimbuzz but Skyfire should work if you can browse the net with IE.  Try uninstalling, soft reset then reinstalling Skyfire.

I'm not sure what country you are in but Skyfire is only active in 3 countries:  US, Canada or the UK

If you live in one of those countries then check this:

The most likely reason that you cannot connect is the availability of an appropriate APN or Access Point configuration. Check with your carrier (or the internet) for appropriate configurations that allow Internet connections (including TCP port 443 and UDP) for your specific carrier.

If Skyfire is stuck on "locating" and fails to connect, then you might have an incorrect connection setting on your phone. Skyfire requires TCP (port 443) and UDP connectivity to function properly. This type of connection is blocked on some carriers when WAP Gateways or similar proxies are configured for your phone. Please check that you select a configuration that allows 'internet' connectivity.



To turn off the proxy setting (in Windows Mobile):

Go to Settings, then to the Connections tab
Connections > Advanced > Select Networks
Under "Programs that automatically connect to the Internet, should connect using"
Select (in the US):
MediaNet for AT&T users
T-Mobile Data for T-Mobile users
Sprint - MyISP - PowerVision
Verizon - MyISP - Broadband Access
In Proxy Settings, make sure to UNCHECK: "This network connects to the internet" and "This network uses a proxy server to connect to the internet"
If you are a Sprint or Verizon customer who is using "My Work Network", make sure to UNCHECK usage of the Proxy server.


If you are a T-mobile T-zones plan user, your plan only allows WAP web access and connections only through port 80. Skyfire cannot connect using this plan. Please us the T-Mobile 'Internet' plan, or you can use Wi-Fi to access the internet.

In addition to checking your proxy setting, please also check your signal strength as that might be effecting your connection as well.
