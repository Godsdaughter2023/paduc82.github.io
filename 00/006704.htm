daniel g. grau say:
They might be field references to footnotes:
  • Press Shift F9 (this is a toggle control that switches between field codes and their result).
  • If you see something like { NOTEREF name_here } or { NOTEREF name_here \f \*MERGEFORMAT}, then it's a reference to the footnote number.

A NOTEREF field acts like a duplicate of the original footnote reference, and if you add or delete footnotes, it will renumber as well. The "name_here" text would be the name of a bookmark in the footnote reference number itself. You can check, add and delete Bookmarks through Insert > Bookmark.

To help you distinguish fields from other text, enable field shading, which adds a gray non-printing shade to fields: Word > Preferences > View >Field shading.

jim64---01 say: Few options for file transfer:

1) Using phone in mass storage mode is probably easiest option for file transfer.

For photos connect phone to PC and select Mass Storage mode when prompted by phone. It will show as a drive in My Computer, H: in my case. Open My Pictures or wherever your pics are stored on PC, and from here you can transfer selected files to Images folder in phone by copy/move/drag & drop. Use safely remove hardware function in taskbar before disconnecting to avoid any data loss/corruption. Open Photos on phone and transfered pics will appear in Downloads folder.

Same method for videos, transfer to Videos folder in phone, and you'll find them in Videos & TV>My videos>All video clips/Other. Videos will also appear in Photos app as this serves as gallery where all pics and vids can be accessed.

Music can be transfered to Sounds folder on phone using same method. Then open music player and refresh library.

Depending on amount of data you've sent to phone it may take a few mins for new files to appear.

2) You can also transfer files using Bluetooth if your PC is enabled. Bit slower, not so good for large files. Files are sent to phone's messaging inbox, you'll hear the message alert(s). Open and select to save to E drive on phone.

3) Ovi/PC Suite will sync/transfer files to phone when connected in PC Suite mode, don't use it myself as Windows does the job perfectly well for me.

The following from Nokia's N97 spec:

Video playback formats:  3GPP formats (H.263), Flash Video, H.264/AVC, MPEG-4, RealVideo 7,8,9/10, WMV 9.
 
Graphic formats: BMP, EXIF, GIF87a, GIF89a, JPEG, JPEG 2000, MBM, OTA, PNG, TIFF, WBMP, WMF.
 
Audio formats: AAC, AAC+, AMR-NB, AMR-WB, AU, AWB, eAAC+, M4A, MIDI Tones (poly 64), Mobile XMF, MP3, MP4, RealAudio 7, 8, 10, RMF, SND, SP-MIDI, True tones, WAV, WMA, WVE.

The following should provide good video playback on N97:
Container: MP4.
Video: MPEG-4, 640x360, 16x9, 25fps, 1200kbps.
Audio: AAC, 48000 Hz, 2 channels, 96 kbps.

Hope that helps.

You mean you want to see that number on your PC?

Well, the 1st 3 examples are possible with an easy way and you don't need a language pack for that.

The 4th one is the tricky one, I don't believe that such a number format is possible with a cell format in Excel.

Excel get the decimal and thousands separator usually from the system, so it is possible that you change the system setting to whatever you need. But that has many site effects... so better don't touch that.

You can overwrite the settings inside Excel, view: File\Options\Advanced\Use system separators

If you uncheck that box you can set the separators to whatever you want.

And the 1st 3 examples are done... but the 4th one...?

A simple way is to use VBA, e.g. you can format the cell as usual in your language and replace the separators later, e.g. using a UDF in a separate cell:

A2:  =MyText(A1,"'",".",",","â‚¬ ")

The code below must be placed in a regular module.

Andreas.

Function MyText(ByVal Where As Range, Optional FirstThousandOperator, _
    Optional ThousandOperator, Optional DecimalSeparator, Optional CurrenyCode) As String
  Dim i As Long
  If IsMissing(CurrenyCode) Then _
    CurrenyCode = Application.International(xlCurrencyCode)
  If IsMissing(DecimalSeparator) Then _
    DecimalSeparator = Application.International(xlDecimalSeparator)
  If IsMissing(DecimalSeparator) Then _
    DecimalSeparator = Application.International(xlDecimalSeparator)
  If IsMissing(ThousandOperator) Then _
    ThousandOperator = Application.International(xlThousandsSeparator)
  If IsMissing(FirstThousandOperator) Then _
    FirstThousandOperator = ThousandOperator
  MyText = Where.Text
  MyText = Replace(MyText, Application.International(xlDecimalSeparator), Chr(1))
  MyText = Replace(MyText, Application.International(xlThousandsSeparator), Chr(2))
  MyText = Replace(MyText, Chr(1), DecimalSeparator)
  MyText = Replace(MyText, Chr(2), ThousandOperator)
  If Where.Value >= 1000000 Then
    i = InStr(MyText, ThousandOperator)
    If i > 0 Then Mid$(MyText, i, 1) = FirstThousandOperator
  End If
  MyText = Replace(MyText, Application.International(xlCurrencyCode), CurrenyCode)
End Function
