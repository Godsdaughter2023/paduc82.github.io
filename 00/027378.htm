There is no such option in the Mac version. Internally recorded audio is stored in the presentation file. The option to Link only pertains to external sound files added to the presentation.

A [PITA] Workaround:

The audio is stored as 1 or more .wav files. You can use StuffIt Expander to expand the .pptx file as a folder containing the OOXML components. It's probably because a table field needs to be either Text or Number, not both.

If you can live with it, try using:

=IIF([SX undiluted]*10/[dry weight]/1000<0.01, "<0.01", Str([SX undiluted]*10/ [dry weight] / 1000))
