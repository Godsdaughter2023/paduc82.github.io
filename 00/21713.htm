dana.d say:

Hello.  Here's is just an idea.

I can't do these as a 1-liner, so I like to break them up.

This is a little longer, but I find it easier to debug by doing it this way.

 

Sub PlayJoin()
    Dim s, add1, add2
   
    Const Fx = "=CONCATENATE(#,#)"
   
    add1 = Sheets("Edit RMA").Range("B43").Address(True, True, , True)
    add2 = Sheets("Edit RMA").Range("J43").Address(True, True, , True)
   
    s = Replace(Fx, "#", add1, 1, 1)
    s = Replace(s, "#", add2, 1, 1)
   
    Sheets("List").Range("V5").Formula = s
End Sub
