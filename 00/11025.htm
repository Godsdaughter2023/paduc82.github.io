don cuthbert say:

You can make this sort of master install Windows disk, although there is significant change between Windows XP and Vista with how it boots. So you probably couldn't easily put every version of Windows on it as you are suggesting. There are articles online as to how to do this, but basically you can put multiple Windows Editions in a single Install.wim file. The official ISO for the Anniversary Update is done this way and has multiple Editions of Windows 10 in it.

You can also "slipstream" all of the Windows Service Packs and updates into your install image, so that it is as up-to-date as was available on the day that you made it.

One issue is that DVDs only hold so much (and I suppose some XP-grade PCs couldn't read a DVD) but a USB stick could be used instead.

Ok, now autosave is working reliably for me.

In my case, the key info was:

 
  1. "If the file is saved to another location, then AutoSave is disabled. This will happen if your file is ..... saved to a local folder".  I was opening from a local folder that syncs to OneDrive.
  2. You must open your file from the File menu within Word, Excel, or PowerPoint in order for AutoSave to be activated.

Well, these things I would not have discovered by myself, so there may be room to improve the online help for autosave, but anyway I am happy because I am off and running with autosave which give me peace of mind.
