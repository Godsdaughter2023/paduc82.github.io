bernie deitrick say:

To make it more flexible I added a variable that you can change, though I hope your table only goes through Y5, which is 24 columns of pallet prices.

Sub TestMacro24()
Dim i As Integer
Dim j As Integer
Dim k As Integer
Dim l As Integer
Dim lRow As Long
Dim iTotal As Integer
Dim iSize As Integer

iSize = 24  'Set Size Here

lRow = 8
Cells(lRow, 1).Value = "A1"
Cells(lRow, 2).Value = "B1"
Cells(lRow, 3).Value = "C1"
Cells(lRow, 4).Value = "D1"
Cells(lRow, 5).Value = "Total"

For i = 0 To iSize
For j = 0 To iSize - i
For k = 0 To iSize - i - j
l = iSize - i - j - k
lRow = lRow + 1
iTotal = IIf(l <> 0, Cells(2, l + 1).Value, 0) + _
    IIf(k <> 0, Cells(3, k + 1).Value, 0) + _
    IIf(j <> 0, Cells(4, j + 1).Value, 0) + _
    IIf(i <> 0, Cells(5, i + 1).Value, 0)
Cells(lRow, 1).Value = l
Cells(lRow, 2).Value = k
Cells(lRow, 3).Value = j
Cells(lRow, 4).Value = i
Cells(lRow, 5).Value = iTotal
Next k
Next j
Next i

End Sub

Significant Digits vs Decimal Places - Microsoft Community

keithdowling say:

Significant Digits vs Decimal Places

Is there a simple way to round fractions to significant digits rather than decimal places? 

If I use the Round function I must select a number of decimal places but this means that my values must always be within the same 10x range which I cannot do. 

For example... 0.000002645 to two significant figures becomes 0.0000026 and 0.0002645 becomes 0.00026.  Same significant figures but different decimal places. 

Is there a simple function for this in Excel?

Thanks in advance for your help,
Keith
joeu2004 say: Keith wrote:
Is there a simple way to round fractions to significant digits rather than decimal places?
[....]
For example... 0.000002645 to two significant figures becomes 0.0000026 and 0.0002645 becomes 0.00026.  Same significant figures but different decimal places. 

For 2 significant digits:

=--TEXT(A1,"0.0E+0")

More generally:

=--TEXT(A1,"0." & REPT("0",C1-1) & "E+0")

where C1 is the number of significant digits.

Remember that Excel will format only up to 15 significant digits, even though the internal representation might be more precise.
