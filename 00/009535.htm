I work with documents prepared in Word that, due to old-fashioned formatting being used by the author, are too bulky and need reformatting.  I created a macro to automate this, but it only works with the main text because I cannot figure out how to include language in the macro to select the header/footer fields and then update the font size and type.  Does anyone have the fix?  I thought it would be simple.  
stefan blom say:

Well, to access the headers and footers in all sections, you can use code such as the following:

Sub FormatHeadersAndFooters()
Dim s As Section
For Each s In ActiveDocument.Sections
s.Headers(wdHeaderFooterEvenPages).Range.Style = wdStyleHeader
s.Headers(wdHeaderFooterFirstPage).Range.Style = wdStyleHeader
s.Headers(wdHeaderFooterPrimary).Range.Style = wdStyleHeader

s.Footers(wdHeaderFooterEvenPages).Range.Style = wdStyleFooter
s.Footers(wdHeaderFooterFirstPage).Range.Style = wdStyleFooter
s.Footers(wdHeaderFooterPrimary).Range.Style = wdStyleFooter

Next s

End Sub

The macro reapplies the Header and Footer styles to headers and footers. You can then modify the styles as required.
