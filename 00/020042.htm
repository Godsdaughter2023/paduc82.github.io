Setting Up Outlook.com with Mail App

We’ll assume you already signed up for a free outlook.com email address, if not head over to Outlook.com and make one.

  1. Launch Mail app and pull down the “Mail” menu to select “Preferences”
  2. Click on “Accounts” tab and then click the + icon to add a new account
  3. Enter the full name you want attached to the mail account, enter the email address, and password, click “Continue” to start the automatic setup
  4. For the incoming mail server, choose “POP” as account type, and set the incoming mail server to pop3.live.com
  5. For the outgoing mail server, use SMTP outgoing mail server smtp.live.com, and set to default ports

With the setup finished, create and send a new email message to verify that everything works.

    There are three basic ways to approach your issue: the first two are workarounds, the third is the better option and also allows you to more easily query your data, use pivot tables, use grouping, or apply subtotals.

    1) Select just the bottom-most of your three header rows prior to filtering

    2) Insert a blank row between the second and third rows of headers

    3) Follow good data practices and remove the first two rows of headers and make them into columns of data. This may require some rearrangement of your data - search for "Excel Unpivot Power Query instructions"

    hanson831 say:

    Had same problem here on multiple macbooks running 10.8 and 10.9 OSX with Office 2011. Web app worked fine, outlook just asked for password. I tried deleting keychains, changing passwords, etc. The account worked fine on another mac, so it is not a server or connection problem.

    Turned out to just be an outdated Office, as they were set to manually check for updates.

    To resolve the outlook issue:

    • Open Word (or any Office program)
    • Click HELP on the top menu bar
    • Select Check for Updates.
    • When the Automatic Updates Screen Appears, select Check for Updates button
    • Install all available updates (this will require all apps to close, so save any open docs, etc.). This takes about 5-10 minutes.
    • After all updates have been installed, open Outlook. It will then update the office database. This will take approximately 20-30 minutes (possibly longer) depending on the size of your outlook file.
    • After it has updated the database, you can open Outlook and enter your password. It should sync and work like normal.

    Hope that helps someone.

    Another option to try if you have all current office updates (although I did not confirm this) is to just rebuild the office database.

    Hold Option key when opening up outlook, select the database, and click Rebuild.
