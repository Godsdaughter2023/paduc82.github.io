sarahkong say:
You can do this via reghack. But not sure if it works 100% anymore.
I know some versions of Windows 10 it doesn't work.
Before doing this make sure you back up your registry and\or do a system restore point.

Go to search and type powershell then right click it and select run as administrator.
Type the following all on one line and hit enter.

Set-ItemProperty -Path 'HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Authentication\LogonUI\UserSwitch' -Name Enabled -Value 1

You will then need to set a task to keep it set to 1 cause it will want to set it 0 after logon.
So still in Powershell type the following.

$Trigger= New-ScheduledTaskTrigger -AtLogOn (one line hit enter)
$User= "NT AUTHORITY\SYSTEM" (one line hit enter)
$Action= New-ScheduledTaskAction -Execute "PowerShell.exe" -Argument "Set-ItemProperty -Path HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Authentication\LogonUI\UserSwitch -Name Enabled -Value 1"
Register-ScheduledTask -TaskName "UserSwitch_Enable" -Trigger $Trigger -User $User -Action $Action -RunLevel Highest –Force (one line hit enter)

Go to search and type task scheduler and make sure the task is there.
Should be called UserSwitch_Enable.

Reboot a couple times to make sure it works.
