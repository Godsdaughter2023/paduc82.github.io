Function DigitStrings(S As String, Optional I As Long = 1) As String
    Dim RE As Object, MC As Object

    .Pattern = "\d+"

        DigitStrings = MC(I - 1)
    End If
End With
End Function

Thanks for the complete description Sally, since that makes what's happening completely clear.

The core problem is that you are using two antivirus products together which aren't designed to be, resulting in an operational conflict where one is detecting the functions of the other as malware and reacting as well.

In your case, the process which Avast uses to detect potential malicious links or other issues within email messages is creating a temporary file in order to examine these.  From the detection of PWS:HTML/Phish by MSE, we can see that it's detecting an apparent phishing message within that temporary file, most likely assuming that the temporary file itself has been created by whatever email program you are using.

So the problem is that both security programs are performing similar tasks, which while the Avast program may never provide an alert or simply moves the message to an alternative email folder or silently deletes, MSE detects and alerts upon whenever such a temporary file is created by Avast.  The file can never be found in that location because Avast had already finished its task and deleted the file normally before MSE could take action.

The reason so many are seeing this is obviously that it will happen for any person using both MSE and Avast together, which anyone in these forums has been recommending against for several years, precisely due to the potential for conflicts such as this one.

So to fix this, choose between Avast and MSE and uninstall the other.

david j parker say:
Let's say you have the following formula in the EvtDblClick cell of a shape that is linked to data: 

=CALLTHIS("GetDSRow",)

Then the following code in a VBA module will set the selected data recordset row:

 

Public Sub GetDSRow(ByVal shpIn As Visio.Shape)
On Error GoTo errHandler
Dim drsIDs() As Long

    shpIn.GetLinkedDataRecordsetIDs drsIDs
   
Dim winData As Visio.Window
Dim win As Visio.Window
    For Each win In ActiveWindow.Windows
        If win.ID = visWinIDExternalData Then
            Set winData = win
        End If
    Next win
   
    If winData Is Nothing Then
        Exit Sub
    End If
   
Dim i As Integer
Dim rowID As Long

    For i = 0 To UBound(drsIDs)
        If drsIDs(i) = winData.SelectedDataRecordset.ID Then
            winData.SelectedDataRowID = shpIn.GetLinkedDataRow(drsIDs(i))
            Exit Sub
        End If
   
    Next i
   
    winData.SelectedDataRecordset = shpIn.Document.DataRecordsets(drsIDs(i))
    winData.SelectedDataRowID = shpIn.GetLinkedDataRow(drsIDs(i))
   
exitHere:
    Exit Sub
errHandler:
    MsgBox Err.Description
    Resume exitHere
End Sub
