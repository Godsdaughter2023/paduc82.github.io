macropod say:

What are you trying to accomplish?

If it's to merge only the records with the X, or vice-versa, use the mailmerge filters to exclude the other records.

If it's to insert varying content according to the city depending on the presence of the X, you could use field coding like:

{IF{MERGEFIELD City} = "Atlanta" {IF{MERGEFIELD X}= "X" "Text for Atlanta with X" "Text for Atlanta without X"}}

{IF{MERGEFIELD City} = "Baltimore" {IF{MERGEFIELD X}= "X" "Text for Baltimore with X" "Text for Baltimore without X"}}

{IF{MERGEFIELD City} = "Charlotte" {IF{MERGEFIELD X}= "X" "Text for Charlotte with X" "Text for Charlotte without X"}}

{IF{MERGEFIELD City} = "Dallas" {IF{MERGEFIELD X}= "X" "Text for Dallas with X" "Text for Dallas without X"}}

Although the above depicts line breaks between the field sets, that's only for posting purposes - in your mailmerge main document you'd have them one after the other on the same line.

If it's to only insert text where the X is present,  you could use field coding like:

{IF{MERGEFIELD X}= "X" {IF{MERGEFIELD City} = "Atlanta" "Atlanta Text" {IF{MERGEFIELD City} = "Baltimore" "Baltimore Text" {IF{MERGEFIELD City} = "Charlotte" "Charlotte Text" {IF{MERGEFIELD City} = "Dallas" "Dallas Text"}}}}}

If it's to only insert text where the X is absent, simply change:
{IF{MERGEFIELD X}= "X"

to:

{IF{MERGEFIELD X}<> "X"

Note: The field brace pairs (i.e. '{ }') for the above example are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practical to add them via any of the standard Word dialogues. The spaces represented in the field construction are all required.

Sub TableTo2ColumnsList()

On Error GoTo Stp

For Each Ns In NameList

Set Data = Range(Ns.Offset(0, 1), Cells(Ns.Row, Columns.Count).End(xlToLeft))

                Destination.Worksheet.Activate

                End If

        Next Dp

hansv mvp say:

Here is a macro you can run. It will create a new worksheet with the desired layout.

Sub Transform()
    Dim wshS As Worksheet
    Dim wshT As Worksheet
    Dim s As Long
    Dim m As Long
    Dim c As Long
    Dim n As Long
    Dim t As Long
    Application.ScreenUpdating = False
    Set wshS = ActiveSheet
    Set wshT = Worksheets.Add(After:=wshS)
    m = wshS.Cells(wshS.Rows.Count, 1).End(xlUp).Row
    For s = 1 To m
        n = wshS.Cells(s, wshS.Columns.Count).End(xlToLeft).Column
        For c = 2 To n
            If wshS.Cells(s, c).Value <> "" Then
                t = t + 1
                wshT.Cells(t, 1).Value = wshS.Cells(s, 1).Value
                wshT.Cells(t, 2).Value = wshS.Cells(s, c).Value
            End If
        Next c
    Next s
    Application.ScreenUpdating = True
End Sub
