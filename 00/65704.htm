davem121 say:
Hi FranH37, this a common enough problem, it will require a registry fix . . .

Click your Start Button, type regedit and hit Enter - this will open the Registry Editor

In the left pand, navigate to:
HKEY_LOCAL_MACHINE - SYSTEM - CurrentControlSet - Control

Then in the Right Pane double click WaitToKillServiceTimeout and change the value to 1000

Click OK

Then back in the left Pane, close up all those folders (keys) again (click in the left pane and press left arrow on your keyboard a few times)

In the left pand, navigate to:
HKEY_CURRENT_USER - Control Panel - Desktop

Then in the right pane, right click in the blank space and choose New - String Value

Name the String Value WaitToKillServiceTimeout

Then double click WaitToKillServiceTimeout

Set its value to 1000

Save and close the Registry Editor

That's It!

magstriker say:

So, you should just have to do what Nyima Yonten said, but in order to get there, you will need admin access.

**NOTE** For convenience I have bolded all the important bits so it is easier to skim.

To do this you will need a windows 10 installation DVD or, preferably, a bootable USB stick. All instructions work for both. Just replace the work usb for DVD in this solution and you should be just fine.

After you acquire one, restart your computer with the usb stick still plugged into your computer.

If you don't boot into the windows 10 install usb by default you will have to change your motherboard settings so you can boot from your windows 10 usb. Basically, the process involves mashing your F1 thru F12 keys right when you computer begins to start up until you get to your BIOS/UEFI menu. Usually the key you want is between F10 and F12 inclusive (includes F10 and F12) but it's different for every machine.

Then, (since every BIOS/UEFI is different I can't really show pictures), navigate to "boot options" and then "boot order" then move USB to the first boot device. You can change this back to your Hard Drive or SSD afterwards, just make sure that even when setting the usb as the first boot device that your HDD or SSD is still in the list somewhere.

Now that that's been taken care of, save your settings and restart. You should be in the windows 10 install usb.

When you get there follow the directions in this video

from 1:10-5:00 and then follow Nyima Yonten's steps.

**NOTE BEFORE WATCHING** Your home PC's main drive is probably C: instead of D: in the video. You will be able to tell which is which because when you type DIR you should see the same files that he does. If you don't see these then keep going through the alphabet until you do.

Alternatively, if in the future this video does not exist or if you have slow internet and don't feel like waiting, here's my summary:

Press shift and F10 to make a command prompt appear.

Then type the following:

C:

dir

(At this point you should see a Users, Program Files, and Windows Folder among other things. Again, If you don't see these then keep going through the alphabet until you do.)

(cd means Change Directory and ren is short for REName)

cd windows\system32

ren utilman.exe utilman1.exe

ren cmd.exe utilman.exe

(make sure you change the names back afterwards)

At this point. Restart into normal windows by restarting and pulling out the usb stick quickly before it starts to fully boot again.

Now that you should be back at the good ol' windows 10 login screen, click on the "Ease of Access" button in the bottom right corner of the screen and voila! The command prompt with admin access is at your disposal!

Now type the following:

net user administrator /active|yes

(The | character is made by pressing shift and \ )

Then close the command prompt and restart.

Congrats! you should now be able to select "Administrator" from the login screen and, without any required password, log in and follow Nyima Yonten's steps.

Afterwards:

You should set the admin account back to being hidden for security reasons by navigating to a command prompt with admin access from anywhere, it doesn't matter if you do it from the hidden admin account, and type the following:

net user administrator /active|no

Now, before restarting, navigate in windows, or the command prompt if that's more your thing, to Windows\System32 and rename utilman.exe back to cmd.exe and rename utilman1.exe back to utilman.exe

Restart your computer.

Now, if you want to keep the ability to boot from a usb if you have one plugged in that you're done at this point. [Yay! :)] However, if for whatever reason you want to change it back, just mash the same function keys F10-F12  and navigate back to the boot order and set your hard drive or SSD to the first boot device. 

Finally, save your settings and restart your computer and log back into your normal account.

Hopefully this helped! :)
