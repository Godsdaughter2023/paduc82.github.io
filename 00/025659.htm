Message title of "<This item contains active content. Open the item to read its contents.>"

creating a new mailbox profile
changing cached mode's time period

    john sr wilson say:
    Your code should work but are you sure you have the slide references correct.

    Slide1.TextBox2 is not necessarily on Slide 1 and Slide2.TextBox2 may not be on Slide 2! You need to right click > View Code and look in the left pane to check.

    An alternate method to reference by the actual slide number

    Private Sub TextBox2_Change()
    Dim oTB2 As Object
    Set oTB2 = ActivePresentation.Slides(2).Shapes("TextBox2").OLEFormat.Object
    oTB2.Text = Me.TextBox2.Text
    End Sub
