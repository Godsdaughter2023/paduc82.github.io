Many thanks

Alex

jay freedman say:

Use content controls to create the data entry points in the form. They enable you to make non-printing "labels" that appear only when the cursor is in the control.

The first step is getting Word to show you the buttons needed. They're on the Developer tab of the ribbon, which is hidden by default. Go to File > Options > Customize Ribbon and check the box for Developer in the right-hand list.

The next step is to set up a borderless table to fix the data entry points at the places on the page that match the preprinted forms. The best method is to scan the printed page, and insert the scanned graphic image while the cursor is in the header pane of your Word document. Size and position the graphic to match the printed page. This image will be behind anything you put on the document's body. Insert one or more tables in the document and drag the cell borders to match the needed locations. With the cursor in a table, press Ctrl+Alt+U to turn off all the table's borders.

Place the cursor in a cell where you want the first content control to appear. On the Developer tab, click the Plain Text Content Control button.

Initially the control will have no label, and it will contain placeholder text. Click the Properties button on the Developer tab, and type the label for this control into the Title box in the dialog box, then click OK.

The title will appear as a label whenever the cursor is in the content control.

If you leave the placeholder text in the control, it will be printed. That's undesirable, so you can remove it. Click the Design Mode button on the Developer tab, which changes the appearance of the control (in fact, all the controls in the document at once) like this:

Now you can select the placeholder text and replace it with one or a few space characters. Click the Design Mode button again to turn it off.

There is no way to explicitly change the size of content in Sway.  Sways are designed to layout great no matter where they are viewed, regardless of the device's size or orientation.  In order to achieve this, Sways appear with a layout tailored for the device of each individual viewer.  Because of this, there is not a way to set the exact coordinates or size of content because those values wouldn't allow for the best possible end result.  Instead, in the iOS app, authors can specify emphasis (by selecting an image and then tapping on the star button that appears) to indicate that a particular piece of content is more important relative to the rest of the content around it and cropping control (by selecting an image and then tapping on the crop button) to zoom in on the most important part of the image.

Animated text in a fixed position - Microsoft Community

adminuser8 say:

Animated text in a fixed position

I want to do a build/animation of text but I want them to appear in the same position on the slide - help!
john korchok say:

Here are the steps:

  1. Copy each name into its own text box.
  2. From the Shape Format tab, open the Format Pane.
  3. Select the Size & Properties tab of Shape Options.
  4. Set all text boxes to be exactly the same dimensions and position using the numeric fields in the Format Pane (this is more repeatable than dragging/dropping/resizing with your mouse).
  5. Choose Home>Arrange>Selection Pane. This allows you to select, show/hide and change the stacking order of the text boxes. Items at the bottom of the list are also at the bottom of the stack.
  6. Select the bottommost text box, then choose Animations>Appear and set Start: to how you want to begin the animation series. If your text boxes have an opaque background, you don't have to worry about hiding the text boxes, the next one will just appear on top.
  7. In the Selection Pane, select all the other text boxes, give them an Appear animation, set Start to After Previous. In the Animation Pane, expand the Timing section and set the Delay to a short time like .25 seconds or so.
  8. Click the Preview button to see a preview. Adjust the delay time until the names appear at the rate you prefer.
