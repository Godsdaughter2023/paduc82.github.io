Hi Irene,

 

 

When the printer is printing, you will see a small printer icon at the taskbar (right bottom). Click on it and you'll see the files that are printing. If you see the document, please delete it from the print queue.

 

 

If this does not help, then do the following.

 

1. Type Services.msc in the Start search box and press enter.

2. Locate Print Spooler and right click on the service and then click on Stop.

3. Once the Print Spooler service is stopped, go to C:\WINDOWS\System32\spool\PRINTERS and delete all the files mentioned in the folder. This will clear the present print jobs.

4. Once the files are deleted, go back to Services window and start the Print Spooler service.

This is how it works. When you connect the Bluetooth device to the Surface, you will be asked to pair the device with the system. Once, it gets paired, the device will automatically get connected to the Surface when it’s turned on within the range.

 

Well, you can remove the headset device from the list of Bluetooth device on the Surface. In this way, it won’t connect automatically the next time.

 

Follow the steps:

•At the Start screen, type Devices.

•Tap Settings on the Search screen.

•Tap Devices.

•Find Bluetooth headset device in the list on the right.

•Tap on the device.

•Tap on the minus sign in the circle (-)

•Tap Remove.

 

Check if it connects automatically when you try to connect the Bluetooth headset to the Windows 8 laptop.
