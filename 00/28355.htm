Dynamic Dropdown - Microsoft Community

upickyrizek say:

Dynamic Dropdown



I have in column A about 300 names of chemicals.
I would like to create dropdown menu in cell C1 which will contain only chemicals which contain text in cell B1 .. would anybody know how to do it ?
I am using Excel 2000


jllatham say:

This worksheet event code should do the trick.  To put the code into the proper place in the workbook, choose the sheet that has the list, cell B1 and C1 you need to work with and right-click on its name tab and choose [View Code] from the list that pops up.  Copy the code below and paste it into the code module that appears when you do that.  You can then close the VB Editor by clicking the red-x for it.  Type an entry into B1 and check the list in C1 to verify it is working properly.  I have made the code case-insensitive, so if you type "pal" into B1, it will match any entries in column A with "PAL", "pal","Pal", etc in them.

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim ListRange As Range
  Dim anyListEntry As Range
  Dim keyPhrase As String
  Dim matchedEntries As String
 
  If Target.Cells.Count > 1 Or _
   Target.Address <> "$B$1" Or _
   IsEmpty(Range("B1")) Then
    Exit Sub
  End If

  Set ListRange = Range("A2:" & _
   Range("A" & Rows.Count).End(xlUp).Address)
  keyPhrase = UCase(Trim(Range("B1").Text))
  For Each anyListEntry In ListRange
    If InStr(UCase(anyListEntry), keyPhrase) > 0 Then
      matchedEntries = matchedEntries & anyListEntry & ","
    End If
  Next
  If Right(matchedEntries, 1) = "," Then
    matchedEntries = Left(matchedEntries, Len(matchedEntries) - 1)
  End If
  If Len(matchedEntries) > 0 Then
  With Range("C1").Validation
      .Delete
      .Add Type:=xlValidateList, AlertStyle:=xlValidAlertStop, Operator:= _
      xlBetween, Formula1:=matchedEntries
  End With
  End If
  Set ListRange = Nothing
End Sub

As you know (but it is worth noting), since 64-bit setup.exe cannot run in a 32-bit OS, there is no way to upgrade the 32-bit version to 64-bits.  The 64-bit version now can only be clean installed.  So if you still want to dual-boot after you sort the hard drive issues, start by reinstalling W7 x64 on one drive and then reinstall W10 x64 on the other by running Setup.exe from the W7 desktop.
