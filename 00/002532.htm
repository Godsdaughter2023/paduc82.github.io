Sub Hide()
 LastRow = Cells(Rows.Count, "G").End(xlUp).Row
 For j = 2 To LastRow
  If Cells(j, "G") = "Y" Then Rows(j).EntireRow.Hidden = True
 Next j
End Sub

Sub UnHide()
 ActiveSheet.Cells.EntireRow.Hidden = False
End Sub

Right-click the worksheet's name tab (at the bottom) and choose View Code. When the VBE opens, paste the following into the pane titled something like Book1 - Sheet1 (Code),

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
    If Not Intersect(Target, Cells(4, 7)) Is Nothing Then
        Cancel = True
        With Range(Cells(4, 7), Cells(Rows.Count, 7).End(xlUp))
            If Me.AutoFilterMode Then
                .AutoFilter
            Else
                .AutoFilter field:=1, Criteria1:="<>Y"
            End If
        End With
    End If
End Sub

... into this,

The filter icon is added to G4. Double-clicking G4 a second time removes the filter. G4 is identified in the code as Cells(4, 7) literally the fourth row, seventh column. Note that I've used,

=SUBTOTAL(109,$F5:$F23)

  ... in F25 instead of a standard =SUM($F5:$F23) to show the filtered sum. You can make your own determination on that point.

bill manville say:

Try opening Excel in Safe mode (hold down the Ctrl key while launching it).
This will disable all add-ins and startup files.

If Excel opens OK then you can remove the check marks from all add-ins in Tools > Add-Ins and restart Excel normally.  If it works then you know that the problem was add-in related and you can selectively re-introduce them until the problem recurs, so you know which was the culprit.
