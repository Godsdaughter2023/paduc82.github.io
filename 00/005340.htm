Connections between MS Project and MS Access/Excel - Microsoft Community

dbpotter808 say:

Connections between MS Project and MS Access/Excel

is there a way to connect/link MS Project to external files?  I know you can connect/link files between MS Access and Excel but how can you do the same with MS Projects 2007?

Thanks for any help.

 

dale howard [mvp] say:
dbpotter808 --
 
If I can jump into this thread, the only way to accomplish what you want to do is to copy a cell in one application, such as in Excel, select a cell in Microsoft Project, and then use the Paste Special - Paste Link functionality.  For example, suppose you have an Excel workbook that contains a list of additional expense items that you want to total and then to place the total value automatically in the Fixed Cost column for a task in Microsoft Project.  Using this example, create the workbook and create a formula to total the additional costs.  Select the cell with the total cost value and copy it to the Clipboard.  In Microsoft Project, apply any task view and then click View - Table - Cost.  Select the Fixed Cost cell for the task on which you want the cost value from Excel and then click Edit - Paste Special - Paste Link.  This will create a "hot link" between the cell in Excel and the corresponding cell in Microsoft Project.  If at any point you add new costs into the Excel workbook and the total value changes, the next time you open Microsoft Project, the software will update the new cost value automatically.
 
Keep in mind that links to an external application are very sensitive in Microsoft Project.  If you move the Excel file to a different location, for example, you will break the link in Microsoft Project.  Also, be forewarned that if you link data from an external application in a cell in Microsoft Project, every time you launch Microsoft Project, the software will always prompt you to update the external data.  Hope this helps.
