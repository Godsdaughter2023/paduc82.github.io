hansv mvp say:

The method that I posted is only suitable for a relatively small number of rows.

As an alternative, you can use a helper column. In the following, I'll use column H, but you can use any empty column. After setting it up, you can hide this column if you wish.

Leave H1 empty.

In H2, enter the formula   =IF(B2=B1,H1,NOT(H1))   where B is the column with the school names.

Fill down from H2 as far down as the data go.

Change the conditional formatting formula to   =$H2

About the formula from my previous reply:

Let's say B2 and B3 contain School A, B4 contains School B, and B5, B6 and B7 contain School C.

The formula for row 7 is

=MOD(SUM(1/COUNTIF($B$2:$B7,$B$2:$B7)),2)

COUNTIF($B$2:$B7,$B$2:$B7) returns an array of values: COUNTIF($B$2:$B7,$B$2), COUNTIF($B$2:$B7,$B$3), ..., COUNTIF($B$2:$B7,$B$7). With the values mentioned above, this evaluates to 2, 2, 1, 3, 3, 3.

1/COUNTIF($B$2:$B7,$B$2:$B7) inverts these values: 1/2, 1/2, 1, 1/3, 1/3, 1/3

SUM(1/COUNTIF($B$2:$B7,$B$2:$B7)) adds these numbers: 1/2+1/2+1+1/3+1/3+1/3 = 3. This Is the number of unique schools in B2:B7.

=MOD(SUM(1/COUNTIF($B$2:$B7,$B$2:$B7)),2) returns 0 if the number of unique schools is even, 1 if it is odd.

Conditional formatting will be applied if the number is odd.

On Error GoTo 0

         ' If we can find the file then get the file attributes
        If (Not objFolderItem Is Nothing) Then
'For testing only, to print list of attribute names:
