Thank you for posting.  These messages are telling you that you either have an incorrect SMTP server defined or you have the correct SMTP server defined but you are using an incorrect security setting, or have an incorrect mail port defined in Outlook.  Comcast blocks many of the most common mail ports.  Comcast is blocking port 25 (both incoming and outgoing) and port 110 (again, incoming and outgoing).  This is done to prevent spam going through there server but it is also preventing their customers from being able to send and receive email.  Unfortunately, since Comcast is blocking the most common mail ports we are not going to be able to assist you.  You MUST have the open port information from Comcast in order to setup your Outlook.  I would suggest calling Comcast back and if their frontline support cannot provide you with this information ask to be escalated to second level support.  Once you have the correct port information entered into Outlook your email will work correctly.

b.    In the command prompt, type the command below:

fsutil resource setautoreset true C:\

c.    This assumes that C: is the drive in which Vista is installed. If it is installed on another drive like D:, please change the drive letter appropriately.

d.    Reboot the system.

e.    Install the software and check.

“How to perform Clean Boot” http://support.microsoft.com/kb/929135

Repair the Microsoft .NET Framework installation

a.    Click Start, click Control Panel, and then click Add or Remove Programs.

b.    In the Currently installed programs list, click Microsoft .NET Framework, and then click Change/Remove.

c.    Click Repair, and then click Next.
