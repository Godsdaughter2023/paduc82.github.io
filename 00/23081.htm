    Replace words in WordDoc from Excel VBA - Microsoft Community

    barb reinhardt say:

    Replace words in WordDoc from Excel VBA

    I am trying to replace some text in multiple word documents based on data in an excel file.   I have found this code BUT need to know how to refer to the "SELECTION.RANGE" in the document I'm calling myDoc.   Can someone assist?

     

    Sub Demo()
    With Selection.Range
      With .Find
        .ClearFormatting
        .Text = "Heading"
        .Replacement.Text = ""
        .Forward = True
        .Wrap = wdFindStop
        .Format = True
        .MatchCase = False
        .MatchWholeWord = False
        .MatchWildcards = False
        .MatchSoundsLike = False
        .MatchAllWordForms = False
        .Execute
      End With
      Do While .Find.Found
        ' get the next word
        MsgBox .Words.Last.Next.Words.First
        ' get the whole paragraph
        MsgBox .Paragraphs.Last.Range.Text
        .Collapse wdCollapseEnd
        .Find.Execute
      Loop
    End With
    End Sub
