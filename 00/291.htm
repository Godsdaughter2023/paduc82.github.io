Review: Windows 10 build 9926 - Microsoft Community

andre da costa say:

Review: Windows 10 build 9926

Technical Level : Basic

Summary

INTRODUCTION

The end of an era! That is how it honestly feels to start this journey writing about Windows 10. It’s no surprise, the 90s and even the 2000s have been filled with reviews of technologies such as operating systems that are written in thousands of words; just check out some of our own previous reviews and even those from technology websites that have written about new releases of Windows from an encyclopaedic point of view. I personally looked forward to reading them, but looking at the future, this methodology of covering the latest version of a Windows release will eventually become obsolete. Windows 10 is certainly the last major (what they call big bang) release of the company’s popular desktop operating system.


Details

A lot has changed since 2006 when Windows Vista came to market, a colossus overhaul of a platform that almost brought down the entire industry with it and at the same time giving rise to old and new players. The Internet certainly facilitated the changing landscape of how we use and consume technology. The 80’s and 90’s encouraged the upgrade treadmill which encouraged users and companies to take advantage of the latest advances in computing when they became available, processors, memory, graphics, software. In a sense, we are almost going back to those days but in a way that does not present the disruptive painstaking migration that was always a certainty of migrating to newer technologies.

Windows 10 presents a challenge in a market that has pretty much migrated to the Internet. It’s at a cross roads between a world that is old and new. Since I started writing this preview, Microsoft has released four revisions of the still in development operating system. Normally I would have jumped on the first build and started preparing content for readers about the new operating system with the first preview build, but I felt a bit reluctant. I think that was within good reason, considering that Windows 10 has undergone significant changes since October and the most recent build, 9926 is still a reflection of the platforms infancy.

BACKGROUND

Since Windows 8 came to market in late 2012, it has been met with mixed reviews. The bold strategy behind that release left behind a market of more than a billion users who have enjoyed working on desktop and laptop computers using mouse and keyboard for more than two decades. The consistent evolution of the Windows interface since Windows 95 gave users the confidence about how they operated a PC. Windows 8 took most of its influence from its mobile sibling Windows Phone OS which was practically unknown and lacked the market penetration to provide a familiar transplant to the desktop. A mixture of bad execution, lack of modern applications and questionable design decisions made Windows 8, the company’s Vista 2. This even resulted in the company having to scramble to tone down a lot of the earlier design choices a year later with a revision called Windows 8.1 and another major update seven months later called Update 1. With the continued rapid growth of competing platforms such as Android and iOS, both of which own nearly 100 percent of the mobile market have left Microsoft in a position of defending its legacy.

Since the launch of the Windows Insider Program in October 2014, Microsoft has provided two editions of Windows 10 for evaluation, Pro and Enterprise. A Core SKU exist for PCs running consumer editions of Windows 7 or Windows 8/8.1, but is only applied when setup is initiated through Windows Update. For the purposes of this review, I will be using the Pro edition, build 9926. Pro includes additional premium business functions such as Domain Join, Group Policy, BitLocker Drive Encryption, Network Backup and Remote Desktop. Microsoft has not finalized all the details regarding pricing, but existing Windows 7, 8/8.1 users will be able to upgrade to the final release of Windows 10 for free within the first 12 months of release. If you are running volume license clients, you will still procure Windows 10 the way you do licenses today. Windows XP/Vista users will need to purchase a new license or you can work around this now by migrating to Windows 10 Preview today which will provide a direct upgrade path to the final release. Keep in mind, you will be installing pre-production software.

Familiar Desktop Experience: Users coming from older versions of Windows such as Windows 7 will feel more at home. The start menu makes a triumphant return in Windows 10, but holds on to much of its Touch functions. Notification Center also centralizes notifications and makes it easier to access common functions like Wi-Fi, Display, orientation and Location.

System requirements for Windows 10 remain remarkably unchanged since Windows Vista, although users should expect some caveats. Microsoft has noted that some older AMD and 64 bit capable CPUs are not compatible with Windows 10 at this time. Although the final requirements have not been defined, if your system can run Windows 7, it should be able to run Windows 10. Do keep in mind, meeting the system requirements is one aspect, low level requirements like PAE, SSE2, NX support are still necessary. The Windows 10 user interface remains pretty much the same as Windows 8.1, some would argue more subdued, yet refined, with bolder brighter icon palettes, re-emphasized window shadows, loss of window chrome and subtle window controls that are more desktop friendly. Long-time Windows 7, XP, Vista users are sure to feel more at home in Windows 10 than its predecessor. Immediate changes such as direct boot to desktop, removal of the controversial Charms bar and Alt-Tab menu are sure to bring a lot more cheers than sneers.

SETUP

It is evident that the Windows Setup experience in Windows 10 is a place holder, hints of changes to come can be seen in the Out of Box Experience. If you have installed Windows Vista or later, you pretty much have an idea of the setup routine. Rumour sites have shown leaks suggesting a refresh to the wizard interface adhering to more of the subdued design theme for this release. For the purposes of this article, I have done mostly clean installs on an existing partition or in a Virtual Machine utility such as Oracle Virtualbox. I thought build 9926 would have been the build that I committed to, but since build 9860, Windows 10 has gone through more unstableness than stability. In fact, I would describe the development experience more like Windows Vista, where features are built as the team goes along. This does not feel like a Windows 7 or Windows 8 at this stage of development where the planned, frozen feature set is being engineered. Then again, I can’t blame the Windows Team considering that Windows 10 was originally intended to be Update 2 for Windows 8.1. Evidence of this can even be found in the Group Policy editor. It was even noted that the OS at one point was called Windows 9. Overall, the traditional setup experience is about as stable as you can get.

The Windows Team and Microsoft have clearly learned from the disaster that was Windows 8.1 acquisition using the Windows Store. Since 2013, the experience upgrading from Windows 8 to Windows 8.1 remains unstable and uncertain, so much so that Microsoft recently released a Media Creation Tool (which is what they should have done in the first place) that makes it easy to do an in place upgrade using media from Windows 8.0. Existing Windows 7/Windows 8.1 users will be able to upgrade to Windows 10 directly through Windows Update by downloading a small tool that enables the capability. Once this tool is installed, all you need to do is check for updates and download the 2 to 3 GB Update and this will start into setup after a reboot. . I always wondered why Microsoft never thought of it this way from the get go? Windows XP and Vista users will have to do a custom install by booting from install media, you can’t even start setup from the desktop in those versions of Windows. The Windows 10 development story is not expected to end at RTM as with previous releases. The Windows Insider program which started in October 2014 will be an ongoing part of the Windows Development model, providing a vast community of users who are risk takers interested in evaluating fresh code when it becomes available. Microsoft will push out revisions of Windows 10 using the Preview Build mechanism built into Windows Update for such testers. Business users will of course be able to avoid this by using a slower mechanism for deploying updates and revisions to Windows 10 in the future. In fact, you can’t do build to build updates unless you have Administrative privileges. Microsoft will give company’s the choice though to move to revisions much quicker if they wish, providing a Current Branch and a Long Term Branch, both of which will be well tested after going through Microsoft’s rigorous internal development processes and Windows Insider Program participant paces.

Windows 10 setup continues the excellence in hardware detection introduced in Windows Vista. Although all the systems I tried Windows 10 on required that I connected to Windows Update to download Video drivers. I was of course pleased to have out of the box network support. Only thing I had to do was run Windows Update, download the drivers without even doing a reboot. The Out of Box Experience continues to be refined although the company needs to understand not everyone wants to use a Microsoft Account. So I would like to see more prominence for setting up a Local Account instead of hiding it to the point it feels like you are coerced into setting up or using a Microsoft Account. I do like the benefits of using my Microsoft Account with Windows 10 though. Lets face it, Windows 10 is a heavily cloud centric revision of Windows, it is heavily based on a servicing model, if you really want to harness the benefits of using the OS, a Microsoft Account will be strongly recommended (not required). Otherwise, you will be just upgrading I guess to use a revised Windows 7, although there is much more to the OS than just the online experience.

If you are just coming from older versions of Windows other than Windows 8/8.1, a lot has changed since. For instance, Control Panel is being deprecated, replaced by a more modern configuration tool called Settings. Settings simplifies and consolidates the myriad of Control Panel items into a set of logical, stoic groups that lead to more verbose options. Of course, this might take some getting used to and so far there have been some outcry, especially with the revised Windows Update that Microsoft has gone too far with. I agree to some degree, for instance, Windows Update no longer provides options for customizing and defining which updates are installed. You will have to settle for all or nothing. Although you can view the details of an update, there is not much more you can do. I hope Microsoft listens and refines these areas before RTM.

DAILY USAGE

Windows 10 might be going in the direction of being the best desktop operating system yet. It basically includes the best of Windows 8.1, with the advantages of what made Windows 7 a great desktop operating system, probably the best version to date. Of course, your first experiencing logging into the OS might not be desired, users will have to adjust to a reality of what it means to be connected to the cloud. As I noted, if you really want to get the most out of a Windows 10 device, a Microsoft Account is preferred, although you are not required to have one, if you want to utilize Windows 10, you should jump in.

The Windows 10 desktop at first glance feels, subtle, yet underwhelming. You are greeted by a revised Start menu that you can immediately feel is trying its utmost best to please two audiences. So far, it feels disjointed. I personally prefer the ideas that were introduced in Windows 10 build 9841. Windows 10 build 9926 the Start menu still feels like it’s trying hard to hold on to its Touch Centric roots too much. I have to be honest with you, the tile apps while nice are just not for me, I hardly use them, so I would have preferred if Microsoft had none at all. Regardless of my opinion, much of these choices need to be understood, Windows run on over 1.5 billion systems worldwide, you are not going to be able to reach a perfect compromise. Regardless you might not agree with all the design choices, it actually works. Just to make it feel like I am using the OS, I will pin apps I do use, mostly desktop apps such as Office and VLC for instance. I particularly despise the new integrated Cortana Digital Assistant menu on the Taskbar (I have it disabled). Although I appreciate the benefits, the implementation looks awkward, I personally would rather have Cortana be more omnipresent than having an actual visual presentation.

Desktop productivity: Windows 10 introduces a long awaited feature, multiple desktops. Called, Task View, Microsoft's implementation uses an app centric approach to managing your opening applications. Snap Assist improves Microsoft's earlier implementation, making it easier to snap and organize available windows you have open.

The Start menu as I noted tries to reapply much of what was lost in Windows 7, but does so in a way that feels unfamiliar at times. The new alphabetically grouped application listing under All Apps which replaces the Start Tip (8.1) and All Programs (7/XP/Vista), feels rough and is more akin to Windows Phone users who have limited navigation options. On a desktop computer with generous real estate, this does not seem necessary. Again, I resolve this by pinning or using the run command to launch apps. Windows 10 reintroduces old favourites like the Jump List, although users complain that it’s not like Windows 7, for instance, you can’t pin your favourite personal folders like Favorites to the Start menu like you did in previous versions. The menu features basic customization, such as the ability to resize on the fly or maximize to a full screen Start Screen and of course, you can pin, drag and drop items. Power Options and Fast User Switching have been split into two separate menus.

Taking into account over 300 million persons seem to like running Windows 8/8.1, Microsoft is not leaving those users behind. If you are running a Tablet or 2 in 1 device, the operating system still provides the familiar Start Screen. If desktop users wish, they can enable this menu on the fly by clicking the expand button on the menu which will immediately provide a familiar environment. The All Apps menu is now a part of this improved Start screen. Users will notice that the Taskbar is always on screen instead of being hidden. Called Continuum, the aim is for users to get the best of both worlds. Microsoft is working to ensure that there is little manual interaction required. For instance, 2 in 1 devices will automatically switch to the appropriate mode depending on the mode the Tablet is in.

The desktop might seem unchanged, but as noted, Microsoft has removed a lot of what made Windows 8.1 unique and different to existing Windows 7 users. Activating menus such as Charms or Alt-History are no more. Two new additions that desktop users will like though include Multiple Desktops called Task View and Notification Area provide more productive benefits. Although I do find the Notification Area to limit options I preferred such as the quick access to Wireless Networks and defining connection settings. Notification Area consolidates all notifications into a centralized pane; for example when you connect a storage device or disconnect one, receive messages, app updates from the Store. Notification Centre also lets you can quickly adjust settings like screen orientation when on a Tablet, Quite Hours and connect to a Virtual Private Network. Task View in Windows 10 is unlike Spaces in OS X or Linux implementations. Instead, Windows 10 takes an app centric approach by making each app define the experience while keeping a single manageable approach using the Taskbar. I would like to see some customizable options such as Setting a unique Desktop Wallpaper, drag and drop options. Another productive improvement is Snap Assist. Windows 7 first introduced an iteration of this function which allowed you to snap together to windows on screen by simply making it possible to drag the window to opposite sides of your screen; automatically each window will fill its half of the screen. Snap Assist in Windows 10 is more intelligent, when you drag the first window to the other side of the screen, it immediately organizes the other windows into small thumbnails on screen, you can quickly choose from. So, it’s more of a less effort process.

Probably the biggest change users will experience if you are coming from the Windows 8x releases is the fact that Modern apps are now windowed. Unlike the apps in Windows 8x which were full screen, you can now have modern apps floating on your desktop alongside traditional desktop applications such as WordPad. They feature most of the functionality of a desktop app such as window controls for minimizing, maximizing, closing and even resizing and taking advantage of functions such as window snapping. Modern apps include a couple more window controls though, a full screen function lets you go back to the traditional full screen mode. Also, functions of the Charms menu are now tucked under a separate title bar menu called App commands, it offers functions such as Searching, Projection, Play To and app settings.

FILE MANAGEMENT

Not a lot has changed in File Explorer with exception for the addition of a Home Directory (something that might sound familiar to OS X/Linux/Unix users. Unlike what you see in those operating systems, the Home Directory is more of a host for recently accessed locations. I like it, it makes opening frequently used directories faster. For instance, I am always browsing directories on my external hard disks, whether it’s backed up files, music or movies. These are often tucked away into sub folders on an external storage device. File Explorer will learn my behaviour and keep a shortcut of the most frequently accessed directories on the drive in the Home directory. So all I need to do is just open it when I connect my external drive. This works just as well with network shares or folders you frequently accessed within Windows. Privacy options are included so you can easily clear any traces of recently accessed locations from the folder options.

The Windows Team has also added some customization options such as the ability to define the default directory you would like to have open by default when you launch File Explorer, whether your Personal Folder or Computer environment. Overall, File Explorer remains unchanged since its introduction in Windows 8, the now familiar Ribbon still remains same.

CONNECTIVITY

The days of knowing protocols such as TCP/IP or 802.11 are really behind us. I remember reading about Internet Connection Sharing in Windows XP. Today, I connect to the Internet on my Windows 10 PC over wireless by simply sharing the Internet from my Nokia Lumia 625 smartphone. It doesn’t get any easier than that.

Settings: Windows 10 introduces a less confusing and refined settings app which focuses on bringing pretty much all the functionality found in the classic desktop Control Panel. In fact, managing updates are exclusively done through this new Settings app.

INTERNET EXPLORER and PROJECT SPARTAN

Windows 10 includes Internet Explorer 11 which is expected to be the last revision of the company’s 20 year old web browser. Microsoft is preparing its next generation web browser code named Spartan that will focus on providing a streamlined interface and enhancing web site compatibility. Internet Explorer remains unchanged from current versions available for Windows 8 and Windows 7. The browser of course offers better site compatibility, I notice many sites that I often browse with IE display pages better, without broken elements or having me resort to compatibility mode. Spartan which is an exciting project promises to provide both compatibility but with greater emphasis on modern standards. Spartan uses a heavily revised code based off the existing Trident rendering called EdgeHTML. Spartan will support advances in Windows 10 such as Cortana for finding information on the web, Annotation, PDF support and a superior reading experience.

MULTIMEDIA

Users of Media Center which was Microsoft’s major push for bringing the PC into the living room should get the message early that Windows 10 is the end of the road. Unlike Windows 8 which still offered support, Microsoft decided to end all support with this release. Users running Windows 8/8.1 with Media Center will have it removed automatically when they upgrade. If they need this feature, they should think twice. Apart from that, legacy applications like Windows Media Player are still around, but are hidden. The OS now comes with out of box support for standards such as .MKV and FLAC which would have been nice 10 years ago, but I guess better late than never. This of course provides one less reason to use a third party solution such as VLC Player which I have been loyal to for years.

Microsoft’s modern multimedia apps such as Music and Video are greatly improved reincarnations of earlier versions. They are much more stable, full featured and integrates better with the desktop and mouse/keyboard. I especially like Music which supports many of the standard features of a desktop jukebox. Of course, I still can’t stand the fact that these apps are not as quick as their traditional desktop counter parts. I am still greeted by a splash screen that leaves me waiting. Don’t get me wrong, they are much improved over what were obviously buggy iterations in Windows 8.0, but I thought by Windows 10, they would have matured to the level of desktop app performance. I can have a movie playing in VLC or Media Player quicker than I would in Video. I honestly would love to make apps like Video, Music and Photos my default choices, but I tend to be an impatient guy. Also, they still lack some desktop support like Jump List support or Live Tile controls.

Modern apps: Modern apps take on desktop functionality with the ability to be windowed.

Speaking of Photos, the approach to library management in Windows 8.0 was complicated, but I liked the premise of bringing in everything from the different sources such as Flickr, Facebook and other services. That has ended and Microsoft is focusing exclusively on making the Photos app a mature, robust app for library management and editing. The integration with OneDrive is also a critical component, providing syncing with the service, so not only are your photos backed up to OneDrive, but they can also be accessed naturally from within the app.

Navigating the application is so much more logical than its predecessor. You are greeted by a Library view with all folders in which you store your pictures displayed in thumbnail view. One of the things I still need to get use to is navigation in these modern apps. For instance, getting out of a folder requires that I right click to activate the back button which takes me to the main folder. It’s a small learning curve and at the same time you learn that Microsoft is trying to appease two audiences here.
