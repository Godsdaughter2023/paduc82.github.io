Data Validation to Prevent Formulas - Microsoft Community

phil hageman say:

Data Validation to Prevent Formulas

Need a formula to prevent users from entering a formula in a cell range.
mike h.. say:

Hi,

 

As I noted in my last post you can only have 1 worksheet change event so I've combined these 2 into 1.

 

Note this is worksheet code so right click the worksheet tab and paste this code in

 

Option Explicit
 Private Sub Worksheet_Change(ByVal Target As Range)
 Dim C As Range
 'User enters an "X" in column C, date appears in adjacent column B cell.
 If Target.Cells.Count > 1 Or IsEmpty(Target) Then Exit Sub
     If Not Intersect(Target, Range("E:E")) Is Nothing Then
         Application.EnableEvents = False
             If UCase(Target.Value) = "X" Then
             Target.Offset(, -1) = Date 'Enters the date in firs column to the left, the -1.
         Application.EnableEvents = True
             End If
     End If
 
'Forces text to UPPER case for the range E4:E33
 If Target.Cells.Count > 1 Then Exit Sub
     On Error Resume Next
     If Not Intersect(Target, Range("E4:E33")) Is Nothing Then
         Application.EnableEvents = False
         Target = UCase(Target)
         Application.EnableEvents = True
     End If
    
 If Not Intersect(Target, Range("F4:F45")) Is Nothing Then
 Application.EnableEvents = False
 For Each C In Target.Cells
     If C.HasFormula Then
         Application.Undo
         MsgBox "No formulae allowed"
     End If
 Next
 Application.EnableEvents = True
 End If
    
     Application.EnableEvents = True
     On Error GoTo 0
 End Sub
