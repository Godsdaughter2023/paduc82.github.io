calvinlenhart say:

Keyboard Failure in Windows 10

On August 29, 2015 my Windows 10 upgraded Desk Top computer's keyboard would click anytime that a key was pressed and the desired character or action would not occur. It is the same sound that you get when the keyboard buffer is full. I was unable to continue my work. I also have a Laptop computer that was upgraded to Windows 10. I tried to use it to continue my work. I found that it had the exact same problem. Through much effort and research I found a solution to the problem and present it here for others to use. Both of my computers failed at the same time. I believe it was a portion of the Microsoft update that changed a setting.

Click the start menu and select settings. In the upper right corner of the settings screen there is a "Find a setting" bar. In that bar put in a word keyboard. There will be a click before each letter appears. In the same way there will be a click when you press enter before the enter happens. In the keyboard settings menu one is marked "Beep when key are pressed or accepted" turn that one off.

ramesh.kumar say:

Hi Kristenlowe,

Thank you for visiting Microsoft Community.

As per the issue description your computer get stuck at installing 15/19 updates and you are getting error message "do not turn off or unplug computer". If it is too much time then there could be a chance that there is some corruption either with system files or with the downloaded files.

The computer is attempting to install Windows Updates and is taking time to complete the process. Check to see if the following helps.

Method 1

Step 1:

If it is still stuck on installing update page, then restart the computer and choose boot normally and check what happens. 

Step 2: Disconnect any non-essential external devices and restart the computer. Check to see if it boots fine.

Method 2:

Step 1: Check if you can  boot the computer in safe mode.

 

Step 2:  If you are able to boot then  perform a system restore to the time when you installed the updates. If this helps you boot fine normally then install one update at a time in case of multiple updates.

 

Method 3:  If you are unable to boot then try renaming “pending.xml” file from the  Windows Recovery Environment (WinRe).

 

a.       Restart your computer and start pressing the F8 key on your keyboard. On a computer that is configured for booting to multiple operating systems, you can press the F8 key when the Boot Menu appears.

b.      Use the arrow keys to choose Repair your Computer in the Windows Advanced Boot Menu Options and then press ENTER.

c.       Select Command Prompt from the System Recovery Options.

d.      Type “cd C:\windows\winsxs”, and then press ENTER.

e.       Type “ren pending.xml pending.old”, and then press ENTER. Restart the computer.

Hope this helps. Please let us know the results, feel free to write us back for any further assistance, we’ll be glad to assist you.
