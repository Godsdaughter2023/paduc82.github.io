suzanne s. barnhill say:

Labels are set up as tables. You can print portions of tables but not necessarily individual selected cells (if you could, they wouldn't be in the same position on the page).

If you're wanting to print just one or two labels on a sheet, you can do so by formatting the text in all the other cells as Hidden or Font Color: White, but the labels will print in the position where they appear, not on the first labels on a sheet.

If you want to print the contents of specific labels on the first few labels on a sheet, you can create a new blank labels document (in the Labels dialog, with the correct label stock selected, click on New Document), then copy/paste those labels into the blank sheet.

Ellipsis button instead of usual arrow - Microsoft Community

soggycashew say:

Ellipsis button instead of usual arrow

Hello, I have access 2007 and wanted to use the ellipsis instead of the arrow on all my combobox(s). Is there a way I can? Thanks!

hansv mvp say:

As far as I know, the arrow is hard-coded.

 

You could place a command button with an ellipsis as caption on top of the dropdown arrow, and make the combo box list drop down in its On Click event:

 

Private Sub cmdEllipsis_Click()
    With Me.cboMyCombo
        .SetFocus
        .Dropdown
    End With
End Sub

 

but as soon as the combo box receives focus, the dropdown arrow will be displayed and the command button will be hidden behind it. So that is a very unsatisfactory workaround.
