jay freedman say:

Create a table in the document. In the Table Tools > Layout tab of the ribbon, click the AutoFit button and choose Fixed Column Width in the menu. (This doesn't prevent you from resizing the columns, but it will prevent Word form automatically changing column widths as the contents of the cells change.)

Put each field and its label into a separate cell of the table. You can split or merge cells on the same row to make the desired spaces for the fields.

Hint: If the table's height completely fills the depth from the top margin to the bottom margin, and an empty second page appears, that means there is a paragraph mark on the second page. That paragraph mark can't be deleted. Click the ¶ button to see it. You can format it to a size of 1 point and/or make it Hidden to get rid of the empty page. It may also help to adjust the sizes of the top and bottom margins, if possible.
