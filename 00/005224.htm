copestoo say:

Combine 2 spreadsheets that have only 1 column of data in common

Hello all,

  I have an Excel spreadsheet that contains the complete inventory of a companies including part numbers, descriptions, billing codes, pricing, etc... . I have another spreadsheet that has the same part numbers plus or minus a few hundred. However, it also has descriptions of the parts that are not the same as the descriptions in the first spreadsheet as well as different pricing. I need to match a few thousand parts by part number between the two spreadsheets and combine the descriptions from the second spreadsheet with the descriptions in the first AND overwrite the pricing of the first spreadsheet with the pricing from the second. There are no similarities between the two spread sheets other than the part numbers. One has considerably more than the other but that is not my issue at the moment I just need to combine the information on spreadsheet 2 with spreadsheet 1 and overwrite the pricing. How would I accomplish this? 

I appreciate any help you all might provide.

lianayada say:

I have done this using VLOOKUP function.  In the "more complete" worksheet (the one with more part numbers than the other one), enter this formula in the columns to the right of the existing information.
=VLOOKUP([PART NUMBER CELL],[ARRAY OF ENTIRE OTHER WORKSHEET],[COLUMN NUMBER TO RETURN INFO FROM],FALSE)

For example, assume one worksheet is called "COMPLETE" and the other is "PARTIAL", and Part Number is in Column A of both sheets. After the last populated column in "COMPLETE", enter this:
=VLOOKUP(A1,PARTIAL!A1:Z26,2,FALSE)

Change the "2" to 3, 4, 5, 6, etc. depending on how many columns of information you need to replicate.

This will return all the values from the other worksheet which match the part number. Then Copy | Paste Values to replace all the VLOOKUP formulas with their actual values and you will have a complete worksheet with all information.  Then you can delete the old Pricing and Description columns, or copy/paste the values, etc.

This is what you are asking:

=COUNTA(A2:A100)-COUNTIF(D2:D100,"draft")

Counts all non blank cells in column A and deduct the number of cells in column D containing "Draft". This will also deduct cells in column D with "Draft" even if corresponding column A cell is blank.

In case you want to deduct "Draft" cells in column D ONLY if corresponding column A cell is non-blank, then you may use below:

=COUNTIFS(A2:A100,"<>",D2:D100,"<>draft")

a. user say:

Can I create new folders or delete old folders in the Mail app?

I've been asked this question quite a bit more than I expected, and the plain answer is 'No.' It's not possible to use the Windows 10 Mail app to create new folders or to delete folders for your email account. You have to create or delete folders using either the web interface or another email client (Outlook if using a company Exchange account) and then they will show up automatically in the Windows 10 Mail app.
