doug robbins - mvp office apps & services (word) say:

I think that Terry got the Ctrl+t and the Ctrl+m back to front.

Here, Ctrl+T sets a hanging indent.  That is first line aligned at the left margin and the subsequent lines in the paragraph indented and each successive Ctrl+t increases the amount by which the second and subsequent lines are indented.

Ctrl+M will shift the whole arrangement to the right.  That is, it will move the first and subsequent lines by the same amount so that the hanging indent is maintained.

As Terry said, Ctrl+Shift+t or Ctrl+Shift+m decreases the amount of the indents.

You can also do this by clicking on the arrow on the right hand end of the Paragraph bar on the Home tab of the Ribbon to access the Format Paragraph dialog and manually set the indents in that dialog.

pesach shelnitz say:

First, press Ctrl+Q in the paragraph that you want to format to restore the default formatting. Then on the Home tab, in the Paragraph group, click the dialog box launcher (the arrow in the lower right-hand corner). In the dialog box that opens, on the Indents and Spacing tab, under Special, select Hanging, and in By, set the number to match your first tab stop.

If you want to align the second word on the first line with the first word on the remaining lines, insert a tab after the first word on the first line.

If you want to define a style to easily apply this formatting to other paragraphs, while your cursor is still in the newly formatted paragraph, perform the following steps.
1. In the Styles group, click the Dialog Box Launcher.
2. In the Styles dialog box, click the New Style button (the first of the three buttons at the bottom of the dialog box).
3.  In the dialog box that opens, change Style1 to a more meaningful name, such as ScriptPar, and click OK.

The ScriptPar style will now be available in Quick Styles to easily apply anywhere.

Macro To Delete Blank Rows Or Columns?

Like, if i have to delete the blank rows, it should prompt me for as "Rows Or Columns", if put Rows, it should delete the blanks rows and same for Blank Column Deletion..

Warm regards Dharmesh
rick rothstein, mvp - excel say:

Hi,

Sub DeleteEmptyRowsInActiveSheet()
Dim lngI As Long
  For lngI = ActiveCell.SpecialCells(xlCellTypeLastCell).Row To 1 Step -1
    If Application.WorksheetFunction.CountA(ActiveSheet.Rows(lngI)) = 0 Then
      ActiveSheet.Rows(lngI).Delete
    End If
  Next
End Sub

Sub DeleteEmptyColumnsInActiveSheet()
Dim lngI As Long
  For lngI = ActiveCell.SpecialCells(xlCellTypeLastCell).Column To 1 Step -1
    If Application.WorksheetFunction.CountA(ActiveSheet.Columns(lngI)) = 0 Then
      ActiveSheet.Columns(lngI).Delete
    End If
  Next 
End Sub

Sub DeleteEmtyRowsAndColumnsInActiveSheet()
  DeleteEmptyRowsInActiveSheet
  DeleteEmptyColumnsInActiveSheet
End Sub

Hi Franks,

Thanks for the above superb Macro. However, Is it not possibl to allow it to prompt me delete the empty rows or columns.. Can it be modified a bit? There should be a prompt option for the deletion of empty column or rows?

You could combine them, but it might just be easier to add another macro to do the prompting and let that macro call the appropriate one that Frank posted. Just add the following to the macros Frank gave you and then call it whenever you want to delete either columns or rows...

Sub DeleteRowsOrColumns
    Dim Answer As String
    Answer = InputBox("Delete entire Rows or Columns? (Enter R or C only)")
    If Answer Like "[Rr]" Then
        DeleteEmptyRowsInActiveSheet
    ElseIf Answer Like "[Cc]" Then
        DeleteEmptyColumnsInActiveSheet
    Else
        MsgBox "You did not respond with an R or a C (single letter only)!"
    End If
End Sub

mike h.. say:

Hi Dharmesh,

Try this which prompts for row or column

Sub Blank_Rows_Columns()
Dim response As String
Dim i As Long, LastRow As Long, LastCol As Long
response = InputBox("Enter R or C to delete Rows or columns")
If UCase(response) <> "R" And UCase(response) <> "C" Then
    MsgBox "Invalid response"
    Exit Sub
End If
With Application
    .Calculation = xlCalculationManual
    .ScreenUpdating = False
    If UCase(response) = "R" Then
    LastRow = Cells(Rows.Count, "A").End(xlUp).Row
    Else
    LastCol = ActiveSheet.Cells(1, Columns.Count).End(xlToLeft).Column
    End If
    If UCase(response) = "R" Then
    For i = LastRow To 1 Step -1
        If WorksheetFunction.CountA(Rows(i)) = 0 Then
            Rows(i).EntireRow.Delete
        End If
    Next i
    Else
    For i = LastCol To 1 Step -1
        If WorksheetFunction.CountA(Columns(i)) = 0 Then
            Columns(i).EntireColumn.Delete
        End If
    Next i
    End If
        .Calculation = xlCalculationAutomatic
        .ScreenUpdating = True
    End With
End Sub

jay freedman say:

Start by going to the Insert tab and clicking WordArt. Select any of the samples (you can modify the appearance later).

Type the word 'rainbow' into the box to replace the "Your text here" placeholder. Choose a font and size.

Because the WordArt is a kind of graphic, the Drawing Tools tab appears on the ribbon. On that tab, in the WordArt Styles group, click the Text Effects button, point to the Transform item at the bottom of the menu, and click the Arch Up item in the fly-out panel:

If you want to make the word colored like a rainbow, right-click the word and choose Format Shape at the bottom of the menu. That opens the Format Shape panel at the right side of the Word window. At the top of the pane, click Text Options. In the Text Fill section, click Gradient Fill. Move the gradient stops apart on the slider. Click each stop in turn and choose a color for it:

You can also experiment with the type, direction, and angle settings and the positions of the stops. You can get something like these:

although I haven't found a way to make the center of the gradient coincide with the center of the arch to make it really look like a rainbow.
