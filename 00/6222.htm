I increased the gap per shape to demonstrate. The code is:

Sub Shapes()
   Dim S As Shape
   Dim varShapeInfo As Variant
   Dim i As Integer, j As Integer
   Dim varFontColor As Variant
   Dim varFontColors As Variant
   Dim intSpace As Integer
   Dim intLen  As Integer
   Dim intWordLen As Integer
   Dim dblSL      As Double
   Dim dblSW      As Double
   varShapeInfo = Sheets("Sheet2").Range("A1").CurrentRegion  '''<= change this
   Sheets.Add
   For i = 2 To UBound(varShapeInfo, 1)
      Set S = ActiveSheet.Shapes.AddShape(CLng(varShapeInfo(i, 1)), Range(varShapeInfo(i, 6)).Left, Range(varShapeInfo(i, 6)).Top, varShapeInfo(i, 7), varShapeInfo(i, 8))
      S.Select
      With Selection.ShapeRange
         .Fill.ForeColor.RGB = RGB(Split(varShapeInfo(i, 3), ",")(0), Split(varShapeInfo(i, 3), ",")(1), Split(varShapeInfo(i, 3), ",")(2))
         .Line.ForeColor.RGB = RGB(Split(varShapeInfo(i, 5), ",")(0), Split(varShapeInfo(i, 5), ",")(1), Split(varShapeInfo(i, 5), ",")(2))
         .TextFrame2.TextRange.Characters.Text = varShapeInfo(i, 2)
         On Error Resume Next
         varFontColors = Split(varShapeInfo(i, 4), ";")
         intSpace = 0: intLen = 0
         For j = 0 To UBound(varFontColors, 1)
            
            intLen = intSpace + 1
            intSpace = InStr(intLen, .TextFrame2.TextRange.Characters.Text, " ")
            If intSpace = 0 Then
               intSpace = .TextFrame2.TextRange.Characters.Count - intLen + 1
               intWordLen = intSpace
            Else
               intWordLen = intSpace - intLen
            End If
            .TextFrame2.TextRange.Characters(intLen, intWordLen).Font.Fill.ForeColor.RGB = RGB(Split(varFontColors(j), ",")(0), Split(varFontColors(j), ",")(1), Split(varFontColors(j), ",")(2))
            .TextFrame.AutoSize = True
            
         Next j
      ''' distribute the shapes  
      End With
      If dblSL = 0 Then

         dblSW = Selection.ShapeRange.Width
      Else

         dblSL = Selection.ShapeRange.Left
         dblSW = Selection.ShapeRange.Width
      End If
   Next i
   
   On Error GoTo 0
End Sub

- click OK

scottbolander say:

I just went through a similar issue.

Just had a tech here and installed cable modem. After he left I went to check my email and error connecting to Gmail Pop server.

I spent hours trying everything I could remember from my days long ago working computer networking. Everything was set right. Only thing that changed was the cable modem. I tried port forwarding, getting into the back end of the cable modem to enable DMZ, static IP on computer, changed to different DNS, nothing was working.

Did a cmd and ipconfig and although I saw it in the network card status, something clicked with the IPV6. Not much uses it yet, and certainly not my old outlook 2003.

I disabled the IPV6 in the network properties on the computer and BOOM fixed.

To make sure that was the issue, I re-enabled IPV6, and broke again.

Disabled IPV6 and fixed again.

I have seen this problem countless times in searches and when people contact their ISP, Microsoft or the email provider (Gmail) is blamed, and in turn the ISP is blamed with the ISP skipping the fact many times it started with a modem swap.

SO if everything else has failed, try disabling IPV6 in your network settings. You don't have to uninstall, just uncheck it.
