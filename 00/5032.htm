Before you start troubleshooting, determine if you can connect to the Internet.

There opens the Mail Settings dialog:

1. Click Start, click Run, type Control in the Open box, and then click OK.

2. Depending on the version of Windows running on your computer, follow one of these procedures:

• Windows 10: If you are in the category view, click User Accounts, and then click mail. If you are not in the category view, double-click on mail.

• Windows Vista, Xp, or earlier versions: Click User Accounts, and then click Mail.
3. Open the Mail Settings dialog box.

After this, set the default profile:

1. In the mail dialog box, in the When starting Microsoft Outlook box, use this profile, click to select the new profile.

2. Click on OK.

3. Use Outlook to send you an email.
