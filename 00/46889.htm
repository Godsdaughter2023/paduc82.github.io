The usual reason that the Print Spooler stops isn't because spoolsv.exe has become corrupt but because a printer driver or a print job has become damaged.  The "usual directions" for dealing with those issues are below the line of **********.

If you have XP with service pack 3 and subsequent updates, spoolsv.exe is version 5.1.2600.6024.  You should have copies of that in %windir%\system 32 (the operative copy); %windir%\system32\dllcache (the WFP backup copy), and %windir%\$hf_mig$\KB2347290\SP3QFE (the file downloaded during Windows Update).

If you don't have the file in any of those locations, the updated file was part of the update for MS 10-061, so you could uninstall that and download and reinstall it: 

read about it, including how to uninstall: http://technet.microsoft.com/en-us/security/bulletin/MS10-061
download and reinstall: http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=15901

MalwareBytes AntiMailware
SUPERAntiSpyware
