Is the only way for me to get them is to RESTORE this previous image from the DVD's; THEN copy the files to my external drive; THEN RESTORE the most current WindowsImage Backup?

Any help would be very appreciated..

Moe

frederik long say:

When an image is stored on a hard disk then you can mount the .vhd file and make it visible as a separate drive which allows you to access all its files and folders. Maybe the method works for DVDs too. I suggest you give it a try. Note also that images are not intended for data backups. Windows has a separate tool for this.

  1. Use Windows Explorer to locate the .vhd (Virtual Hard Disk) file in the folder \WindowsImageBackup on your external disk. Note the .vhd extension.
  2. Launch the Disk Manager (diskmgmt.msc).
  3. Click Action, then Attach Virtual Hard Disk.
  4. Use the Browse button to locate the file you found in Step 1 above, then click OK.
  5. Note the new drive letter that Windows assigned. It could be G:. If no drive letter was assigned, right-click the new disk and give it a drive letter.
  6. Use Windows Explorer to open drive G: and retrieve your files.
  7. When finished, close Windows Explorer.
  8. Go back to the Disk Manager.
  9. Right-click the new disk on the left side of the panel where it says "Basic" and "Online".
  10. Click "Detach VHD".
