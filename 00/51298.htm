This is the reason why I always have two admin accounts: One for admin work plus a spare admin account for much the same reason as I have a spare key. Here is a complex method to reset your password. Ask a computer-savvy friend to help you if you lack experience in this area.

 1. Set your BIOS to use the DVD drive as the primary boot device.
 2. Boot with a Windows Repair CD.
 3. Press a key when the message indicating "Press any key to boot from CD or DVD â¦" appears.
 4. Select a language, a time and currency, and a keyboard or input method, and then click Next. You are now in Repair Mode.
 5. Select an Operating System, then click Next.
 6. Select the Command Prompt.
 7. Identify the correct drive letter for your Windows installation. It could be a drive other than C: but it won't be X:. You do it by typing the commands
    dir C:\Win* {Enter}
    dir D:\Win* {Enter}
    dir E:\Win*  {Enter}
    etc.
    until you find the Windows folder.
    I will call that drive "Q:".
 8. Type these commands exactly:
    cd  /d  Q:\windows\System32 {Enter}
    ren  sethc.exe  sethc.org  {Enter}
    copy  cmd.exe  sethc.exe  {Enter}
 9. Reboot normally.
10. When challenged for a password, tap the Shift key five times. You will see a small black screen.
11. Type these commands and press the Enter key after each of them:
    net user
    (To see a list of all accounts that exist on your machine)
    net  user  "Emily"  (or whatever)
    (To see when you last logged on or changed your password)
    net user  "KT"  123456
    (to set your password to 123456)
    net user Spare abcdef  /add
    (to create the badly needed Spare admin account. Its password will be abcdef.)
    net localgroup administrators Spare  /add
    (to promote the spare account to admin level)
    exit
13. Log on under the KT account, using the new password.
14. Log on under the Spare account (because an untested account is close to useless!)

15. Document the spare admin account!
