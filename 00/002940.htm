shannon_irind say:

Never mind, I found it in another question on the forum.

 

FYI for anyone else who asks, this is from the other question:

 

To make the 'last" record the form's current record, add a line of code to the form's Load event procedure:

Me.Recordset.MoveLast

 

Worked perfect. I knew it was something simple. Thank you to the other guy who wrote that!! :)

dirk goldgar say:

 

To make the 'last" record the form's current record, add a line of code to the form's Load event procedure:

Me.Recordset.MoveLast

 

Worked perfect. I knew it was something simple. Thank you to the other guy who wrote that!! :)


Be aware that, if the form has no records to display, that line of code will raise an error ("no current record").  You could avoid that error by either suppressing it:

 

    On Error Resume Next

    Me.Recordset.MoveLast

 

or by checking first to see if there are any records:

 

    If Me.Recordset.RecordCount <> 0 Then Me.Recordset.MoveLast

stephenpsa say:

No, restore points are a 'backup' of the registry. As there are (almost) always changes to the way things are stored in the registry between builds, there is not much point in keeping them across builds. In order to return to a prior build *), you must perform a clean install from suitable media. This is not too hard to do but you should certainly not install insider builds on a machine you depend on (and always make sure you have backups of important files).

*) For a while after an upgrade, you have the option to roll back to the prior build. Just don't count on it being available indefinitely.

mel77084 say:

if i upgraded my win 10 to win 10 insider preview to try it out but before the upgrade i made a restore point , so my question is can i restore my system to normal win 10 with a restore point? or after the upgrade i cant do it or deleted?

If you use a free (for personal use) image backup program such as Macrium Reflect to make an image of your OS partition and save it on a different drive, (and to make the rescue disk) you can quickly and easily restore that image without relying on any OS to do so. It typically takes less than 10 minutes to make the image and about the same to restore the OS and all of your installed programs onto your drive in your OS partition.

In my lab, it's standard (required) procedure to make such an image for each build, on each machine. We save the last two, as well the last public release, and the licensed OS prior to the upgrade to Windows 10.

If you have ever had a hard drive failure, you'll know how to appreciate being fully back in operation in 10 minutes or so after physically replacing the bad drive.

The restore point you asked about is only for use in restoring to an earlier state within that build, not restoring to an earlier build.

insert arc symbol

hello  i  want to  know if  i  can  insert  the  arc  symbol  in  a  doc  with  keyboard(without  equation  from  word)
suzanne s. barnhill say:

On the Insert tab, in the Symbols group, click Symbol and choose More Symbols... This will open the Symbol dialog. In the Font dropdown, choose the Symbol font. There are arc symbols at 199 and 200. You can insert them from the dialog (if you do, they will be on the MRU palette the next time you click Symbol on the Ribbon), or you can click Shortcut Key... and assign a keyboard shortcut.

Or you can insert the desired character in your document, select it, open the AutoCorrect dialog (File | Options | Proofing | AutoCorrect Options) and create an AutoCorrect entry.

If you have the Arial Unicode MS font installed, you can also insert an arc character from the Miscellaneous Technical character subset (U2312).

If this is not the arc symbol you're looking for, please explain.

I rotated that counterclockwise to make the arc run across, and adjusted the outline color and weight, adjusted the size to fit across the letters, and set the text wrapping to "in front of text".
