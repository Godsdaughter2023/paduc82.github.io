    Step 1: Press the buttons "Ctrl, alt, and delete" simultaneously.  A list of options should come up.  Click on Task Manager from the options that display.  Note, if Task Manager is unable to open OR pressing the buttons "Ctrl, alt, and delete" simultaneously does not do anything, jump to Step 5.

    Step 2: In task manager select Notepad.  Now click the "End Task" button.  If a confirmation message comes up, then click "End Task" in the confirmation message.  If Notepad closes then problem is solved.  If Notepad does not close please repeat this step.  If repeating this step after 5 times does not work, go on to Step 3. 

    Step 3: Select Notepad on task manager, right click on the selected Notepad in Task Manager.  A drop down menu should come up.  Select "Go to Details".  After doing this DO NOT click on anything yet.  Go to step 4. 

    Step 4: There should already be something highlighted, right click the highlighted object, a drop down menu should appear.  Click "End Task" option from that menu.  A prompt will come up, click the "End Process" option on the prompt.  If Notepad closes then problem is solved.  If Notepad does not close, please repeat this step.  If repeating this step does not work, got to step 5. 

    Step 5:  Hard shutdown the computer, by pressing onto the power button.  DO NOT LET GO OF the power button until the computer power indication sign has turned off.  Now turn the computer back on.  If the issue is gone, then problem is fixed.  If "04sall-notepad" opens up again on startup, please reply back stating that.
