hpholm say:

Microsoft have inherited Nokias way of doing things, so they work with local operators to have the phones approved to use on the particular network before releasing updates, and before enabling new features. Other vendors may release phones and updates and hope it will work. My friends often complain about dropped calls on expensive, competing phones, something that has never been an issue on my Nokia phones.

 

The particular trouble with LTE/4G is that phone calls still need 2G or 3G in most areas. This fallback is not trivial to do, since it is very different technology. Also since there are at least three major network hardware suppliers (Nokia, Ericsson, Huawei) to provide the equipment for the cell towers, this in itself can be challenge. There are official industry standards to ensure compatibility of course, but I prefer something that is tested and approved for my region.

 

LTE is still work-in-progress. I keep the money for a new phone in my pocket until my operator enables phone calls on 4G (VoLTE) which completes a phone call in less than a second, instead of an up to 10 second wait. the long-range 700Mhz frequencies for rural areas will be nice too, and then there is carrier aggregation for even higher data speed. It is nerdy stuff, but that is how I am.

 

It is very unlikely that you will get an official response from Microsoft, so I hope this helps. ;)

  5. Click on Type your message here to start composing your message.
  6. Enter your recipients on the To box. Click on Cc & Bcc if you want to put Cc and Bcc recipients to your email.
  7. Click Send.

Long running script - Microsoft Community

terrycrouch1 say:

Long running script

How can I stop a long running script
don varnau say:

If this is happening only on one (or a few) website(s) there really is (by Internet Explorer standards) a long running script on the page and IE is doing you a favor by providing the "Stop script" option. Click on the Stop script button.

Again, if only a site or two are affected...

It might break a desirable feature on the site, but placing a site in Restricted Sites will disable JavaScript and the script messages will stop.

Internet Options> Security> Restricted Sites...

If you're seeing the message on all or nearly all sites, a browser add-on might be causing it.

Note: The Google Toolbar has caused script problems for some users. See if disabling that toolbar stops the messages.

Start Internet Explorer without add-ons from Start> Programs> Accessories> System tools> Internet Explorer (no add-ons)
or

from Start> Run (or Windows logo key + r) type in iexplore.exe -extoff

(iexplore.exe<space>-extoff) then hit Enter.

If the problem goes away, an add-on is causing it.

To determine which add-on is causing the problem see this old (but still relevant) article:

IEBlog  Troubleshooting and Internet Explorer’s (No Add-ons) Mode:
http://blogs.msdn.com/ie/archive/2006/07/25/678113.aspx

Another option: Other browsers like Edge, Chrome, Opera and Firefox are more forgiving of these script problems. Consider using another browser on affected sites.

Don
