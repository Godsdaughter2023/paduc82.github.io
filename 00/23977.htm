Generally these are the few methods people use to handle mail of resigned staff -

1. PST it to a file, and keep the file for future reference

<excerpt>
Let me fix it myself To unregister the mswrd632 converter yourself, edit the registry as follows:
  1. Click Start, click Run, type regedit, and then click OK.
  2. Locate and then click the following registry subkey:
    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Shared Tools\Text Converters\Import\MSWord6.wpc
  3. On the Edit menu, click Delete.
  4. Click Yes.
  5. Exit Registry Editor.
This change will effectively unregister the converter and disable it for third-party applications and for Microsoft Office. Microsoft Office will use its own text converters to open these kinds of files.

chris stubbs say: Packed/rlpack isn't the name of a specific malware. There are executable compressors that modify an executable file to make it smaller. One of them is called "rlpack"; a detection of "packed/rlpack" just means a file packed with "rlpack" was found. It might be malware, it might not.

You can send a file for analysis to http://www.microsoft.com/security/portal/Submission/Submit.aspx
