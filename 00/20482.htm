dale howard [mvp] say:

JAG1832 --

If you are not using Microsoft Project with Project Server or Project Online, you will first need to create a shared resource pool file and connect each of your projects to the shared resource pool file as sharer projects.  In the shared resource pool file, this is where you will add the Timber Mats resource as a Work resource, and you will need to set the Max. Units value for this resource at 250,000.  This is how you indicate to Microsoft Project that you have 2,500 timber mats available for project work.

Next, you will need to assign the Timber Mats resource to tasks in your project.  Set the Units value for the Timber Mats resource at 100% multiplied by the number of timber mats needed for the task.

Finally, you will need to create a master project consisting of each of the projects as subprojects.  In the master project, you can apply the Resource Usage view, which will show you the resource utilization for the Timber Mats resource across all projects.  Right-click anywhere in the timephased grid on the right side of the Resource Usage view and select the Details Styles item on the shortcut menu.  In the Detail Styles dialog, remove the Work item on the right and add the Percent Allocation item instead.  Zoom the timescale to the time periods you want to see, such as months, and now you can see the percentage of allocation for the timber mats resource across all projects.  If you see a Percent Allocation value greater than 250,000 in any time period in the future, then this means you will not have enough timber mats and will need to rent them.

Hope this helps.

julie sheets say:

There are a couple of ways of approaching this.  You can create the timber mats as material resources and then assign the mats to the jobs.  Using the Resource Usage view, you can see how many mats you have assigned at any time.  However, you will not be alerted if you exceed the 2500 limit.  The advantage of creating the mats as material resources is they will not affect the calculated work, assuming you are also assigning work resources to the tasks.

To create the resource as a material resource:

  • Go to the resource sheet and enter the Timber Mat resource.
  • Set the type to Material.
  • You may also set up a price per mat if needed.

You can then assign the resource at X amount per day using the Task Form in the lower pane.  Right click in a blank area of the Gantt chart (right side) and choose "Show Split" to show the Task form in the lower pane.  You can select the resource and enter 50/d in the units field to indicate that you are using 50 mats for each day of the task.

In order to be alerted if you have exceeded the 2500 limit, you'll need to set up the timber mats as work resources.  By default, Project shows resource units in percentage, but you may find it easier to work with decimals as units.  

  • Go to File > Options, Schedule and set Show Assignment Units as to Decimal.  

Then create the resource.

  • Go to the Resource Sheet and enter the Timber Mat resource with a max unit of 2500.
  • You can then assign the mats to the tasks.
  • If you exceed the 2500 limit, you'll see red indicators in the indicator column in the Gantt chart view.

1.  There are no spaces vertically between the bottom of one control and the top of that beneath it.  Unused vertical design area does not shrink.

Sub Slow()
  Dim i As Long, j As Long
