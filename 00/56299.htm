I'm not suggesting going to integrated graphics permanently. That would be horrible.

a. user say:

Some monitors come with automatic brightness similar to your TV, if you are using an external monitor, try looking around in the menus and turn any such features off.

If you have a laptop, it might be a power issue, does it happen if it's plugged into the mains or only on the battery? in the latter case, replace the battery, in the former case, you might need to replace the device. 

Check in the BIOS (press del, F2 or F12 during the POST screen) for similar brightness control settings. DO NOT adjust any setting in the BIOS you do not understand as it can make real changes to your machine's performance.

Check that any software supplied by the OEM isn't providing similar services to Window's adaptive brightness feature, this may be easy or hard depending on the OEM, and the software provided, if you built the machine yourself, or used a Windows CD instead of the Recovery image provided by the OEM this will not be an issue.

¡firedog say:

I have in the c:\users\anne\appdata\local\microsoft\windows live mail

I do not see a contacts folder or anything like that,


Contacts used by Windows Live Mail 2011/2012 are stored in databases called DBStore. They are located at
%localappdata%\Microsoft\Windows Live\Contacts

If Anne did not normally sign in in Windows Live Mail, her contacts will be in a folder called Default. There may be a sub-folder named with the version; the latest is 15.5. If you have the same sub-folder in both places, you may be able to copy the old DBStore folder to replace the new one. If you don't have the same sub-folder, this is not likely to work, because the database structure changed between versions. Try this with all programs closed, then follow with a repair of Windows (Live) Essentials to re-register everything - just run wlarp.exe and select the Repair... option. Reboot when it's finished.

If she did normally sign in, there's no problem: all she has to do is to sign in with the same
Microsoft Account to populate her contacts list automatically.
