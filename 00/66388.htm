jeeped say:

Your keyboard has one or two additional input languages available, probably US which change the keyboard layout. The additional keyboard layouts are likely being triggered from assigned hot keys. You need to get to the Advanced Key Settings within Text Services and Input Languages.

Admittedly, this is a hard dialog to find. I have had the same trouble between US (English), Canadian French and Canadian Multilingual (standard). I left the languages intact, but removed the hot keys.

If you can locate a small keyboard icon in the Notification Tray (right hand side of the Taskbar, just left of the clock and a couple of other icons), right click it and select Settings.

If not, you'll need the Control Panel's Region and Language applet, Region and Language dialog box,Keyboards and Languages tab's Change keyboards... button.

Either way, you should be at the Text Services and Input Languages dialog. Two options here:

  1. Delete any languages you do not want to remain on the system from the General tab.
  2. Click the Advanced Key Settings tab. In the Hot keys for input languages section, use the Change Key Sequence button to remove any Key Sequence that may be interfering with your work. This includes the setting for Between input languages (usually at the top of the list and probably the one causing your troubles).

Click OK as you exit each progressive dialog so as to save any modifications you may have made.

techfreak_ say:

Current TVs with HDMI ports are similar to standard computer monitors with HDMI ports. As long as the HDMI port is connected to a desktop computer or laptop, you should be able to extend the Windows desktop or mirror it, like any monitor. As long as there are no issues with the cabling or length it should work. You simply set the TV to the HDMI input source and Windows should detect it.

I am unaware of casting being supported in Skype. However, if you are using a ChromeCast device you should be able to still cast your desktop or browser screen to that device. This can be initiated in Chrome by clicking on the three dots and selecting Cast. This should allow you to display Skype running on the desktop or in a web browser.
