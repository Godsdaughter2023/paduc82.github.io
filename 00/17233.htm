pie chart greater than 100%

Is there any way to do a pie chart for something that equals more than 100%? For example, I want to show that we went over budget by more than 200% on a particular line item... we spent $3500 when only $1500 had been budget/allocated. I know I can do it on a simple bar chart--just show the two entries, budgeted ($1500) right beside actual ($3500), and that will work, but I wanted something that was really going to illustrate the percentage of overage...

kergga say:

If I were to do this, I would create an all blue pie chart representing 100% of budget, then have some other data series to represent the savings (performance was better than budget) or overage (performance is worse than budget). Note that if your overage was greater than 199% of budget you might have to extend this approach and use another color for 200%-300% over.

 

Think of it as a dial, where straight up is budget, rotate counterclockwise to show underbudget (green wedge) or rotate right to show overbudget (up to 200%, but the idea is extendable).

 

A             B             C   D etc

Budget   Actual

 

C= % of budget = B/A = (may be larger or smaller than 100%)

D= savings = if(C<1,1-C,0) = green wedge

E= BudgetFill = if(C>1,C-1,0) = revised amount of the 'remainder' of the circle

F = Red wedge = if(E>0,1-E,0)

G = OverallCoreAmount = if(E>1,E,C)

 

Then graph D, F, and G (if I got that right). red or green will only show if the amount is above or below 1, and G should be the fill for the remainder of the circle.

 

You will want to order your data series so that F (red) is first, G (blue) is second, and D (green) is third.

Insert a bookmark at the end of the TOC and then the following field construction

 

{ = { NUMPAGES } - { PAGEREF bookmarkname } }

 

Replace bookmarkname with the name of the bookmark that you inserted.  You must use CTRL+F9 to insert each pair of field delimiters { } and use ALT+F9 to toggle off their display.

It could look like this:

    With Application.FileDialog(2) ' msoFileDialogSaveAs
        If .Show Then
            Me!txtFilePath = .SelectedItems(1)

            MsgBox "Export canceled!", vbExclamation
