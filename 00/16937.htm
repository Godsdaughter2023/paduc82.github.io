Try adding the following code to the worksheet in question's VBA macro.  It caused all of the conditional formatting on my screen to be repainted properly without having to scroll down every time I do a change.

-------------------------------------------------------------------------------------------------------

Private Sub Worksheet_Change(ByVal Target As Range)
    Application.ScreenUpdating = False
    Application.ScreenUpdating = True
End Sub
