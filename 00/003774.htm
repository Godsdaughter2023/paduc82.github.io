The Digital License for Windows 10 is assigned to the device that it is currently activated on. If the Windows 10 software was acquired as stand-alone software, or upgraded from Windows 7 or 8.1, that was acquired as stand-alone software, it does include Transfer rights to any other device that you own (such as your Parallels VM on a Mac) or that someone else owns. You need a separate license for each device, whether virtual or physical.

See section 4. b. of the Microsoft Software License Terms:

Stand-alone software. If you acquired the software as stand-alone software (and also if you upgraded from software you acquired as stand-alone software), you may transfer the software to another device that belongs to you. You may also transfer the software to a device owned by someone else if (i) you are the first licensed user of the software and (ii) the new user agrees to the terms of this agreement. You may use the backup copy we allow you to make or the media that the software came on to transfer the software. Every time you transfer the software to a new device, you must remove the software from the prior device. You may not transfer the software to share licenses between devices.

But if it is activated on a PC that shipped with OEM-supplied Windows 7 or 8 or 10, you can't transfer that License to another device.

If you upgraded through the Windows Store from Home to Pro, I do not know if that changes the license to a Retail, stand-alone license or not.

You do not have to stay running Insider builds. Your device, if Windows 10 Pro is activated, is permanently licensed for Windows 10 Pro. I have recently been able to reproduce the issue that you describe. I'm not sure when a fix for this issue will be available - but in the meantime I have reviewed the issue and have a suggested Work around:

  • Click "File" and then choose "Options"
  • Choose "Advanced" option category
  • Scroll down on the right until you see the "General" subcategory of options
  • Uncheck "Provide feedback with animation"

This should prevent the crash when you add or remove columns in a RTL document with freeze panes enabled.

tom hoke say:

If you search on Bing for "How to disable Animation in Excel 2013" you should find a few good sites that discuss in detail how to disable animations.

Quick summary - there are three options of progressively greater impact:

  • disable hardware graphics acceleration (from Options/Advanced/Display)
  • change registry to add DisableHardwareAcceleration in HKCM\Software\Microsoft\Office\15.0\Common\Graphics
  • Windows "Use computer without display" option "Turn off all unnecessary animations"

Hopefully the first or second of these will be sufficient to avoid the crash for you.

Best Regards,

-Tom

amit tandon say:

A small typo in ediardp's formula - insert a comma after 1:

=INDEX($D$2:$D$19,MATCH(1,($A$2:$A$19=H2)*($B$2:$B$19=I2)*($C$2:$C$19=J2),0))

and entered with CTRL+Shift+Enter

If you want to use it as non-array, enter below as normal formula:

=INDEX($D$2:$D$19,MATCH(1,INDEX(($A$2:$A$19=H2)*($B$2:$B$19=I2)*($C$2:$C$19=J2),,),0))
