Hi idigweeds,

Thank you for visiting Microsoft Answers.

Windows wav files are stored in directory C:\Windows\Media. The files are locked and require permission to make changes to the files.

Here are the steps to modify ownership and permissions of folders and files:

1. Right-click the file or folder, click Properties, and then click the Security tab.

2. Click Advanced, and then click the Owner tab.

3. Click Edit, and then do one of the following:

• To change the owner to a user or group that is not listed, click Other users and groups and, in Enter the object name to select (examples), type the name of the user
   or group, and then click OK.

• To change the owner to a user or group that is listed, in the Change owner to box, click the new owner.

4. If you want to take ownership of the contents of the folder, select the Replace owner on sub containers and objects check box.

5. Click OK, and then click Yes when you receive the following message:

You do not have permission to read the contents of directory folder name. Do you want to replace the directory permissions with permissions granting you Full Control?

All permissions will be replaced if you click Yes.

Note folder name is the name of the folder that you want to take ownership of.

6. Click OK, and then reapply the permissions and security settings that you want for the folder and its contents.

Additional considerations

• An administrator can take ownership of any file on the computer.

• Assigning ownership of a file or a folder might require you to elevate your permissions using User Access Control.

Now follow these steps:

1.  Right-click on Desktop and select Personalize.

2.  Click on the link named “Sounds” at the bottom of the window.

3.  Next, select a sound event from the list and then hit Browse button. Note that you need to have your customized new sound file in .wav format.

4.  Select your desired .wav sound track and click Open and then click Apply.

5.  Once you change the sound, just hit the Play button to test the customized sound.

Repeat the procedure from step 3 to step 5 to customize all or any sound event. Only thing you need to remember is that Windows supports only .wav files.
