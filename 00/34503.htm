The HVR-850 is an analog cable / ATSC tuner.  It cannot tune Clear-QAM (unencrypted digital cable.)  Sorry.

The HVR-950Q can tune Clear-QAM, though.  Note, though, that this tuner cannot be switched on-the-fly between modes - you must choose which you want (analog cable, or clear-QAM) during TV Setup.

Re-install the operating systems using the OEM installation media that
goes with the OEM Product Keys.
 
Product Keys are bound to the specific type and language of
CD/license (OEM, Volume, retail, or full) with which they are purchased.
For example, a WinXP Home OEM Product Key won't work for any retail
version of WinXP Home, or for any version of WinXP Pro, and vice versa.
An OEM Product Key will not work to install a retail product. An
Italian Product Key will not work with an English CD. Bottom line:
Product Keys and CD/license types cannot usually (it's been reported
that the Product Key for a full license will work with an Upgrade CD) be
mixed & matched.

jess can say:

Hi Mike,

Windows 10 allows you to move your apps to an external storage media and yes, it does need the storage device to be on NTFS format. So far there have been no issues doing so- or update issues relating to it.

If you were able to mount your SD card to your Windows computer and able to move apps, it is likely that the SD card is already on NTFS format; otherwise Windows would have prompted you to format the card.

To check whether your external storage device is on NTFS format or not, we recommend the steps below:

1. Open File Explorer.

2. Right-click on the (SD card) Storage drive.

3. Click Properties. In the General tab look at the File System, this will indicate the system format of the storage media.

Should you need any more assistance, feel free to contact us.
