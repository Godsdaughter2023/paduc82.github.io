or like this in VBA

strLet = LAN(Range("A2").Value)

And if you ever had a string like

ThisStreet 17 B, Other, 12D

It will return D in preference to B.

Function LAN(s As String) As String
    Dim re As Object
    Dim mc As Object
    Dim m As Object
    Dim vA As Variant
    Dim i As Integer
    Dim j As Integer
    Dim iVal As Integer
    Dim sTemp As String

    LAN = ""
    Set re = CreateObject("vbscript.regexp")

    sTemp = Replace(s, ",", " ")
    With re
        .Pattern = "[0-9][A-Z]"
        .Global = True
    End With

    vA = Split(sTemp, " ")
    For i = LBound(vA) To UBound(vA)

            If re.test(vA(i)) = True Then
                For j = 0 To 9
                    vA(i) = Replace(vA(i), j, "")
                Next j
                If Len(vA(i)) = 1 Then
                    LAN = vA(i)
                    Exit Function
                End If
            End If
        End If

A Default Policy Tag (DPT) applies to all items that do not have a retention policy tag applied (inherited/explicit). This applies to the entire mailbox.
