a. user say:

If you mean you are seeing a message like this:

Windows could not start because the following file is missing or corrupt:
\WINDOWS\SYSTEM\vgaoem.fon

You can attempt to repair this file by starting Windows Setup using the original Setup CD-ROM.
Select 'r' at the first screen to start.

You can't access the floppy (or many other folders on your HDD) from the XP Recovery Console

unless your system has been set up in advance through a local security policy or in the registry to allow such things (and most  systems are not).


To do so, you would have to enter this command:

To access files on a floppy or USB you would need to enter this command:

set  AllowRemovableMedia  =  TRUE     (it is NOT removEablemedia, there is no E).

To allow access to other folders you would need to enter this command:

set   allowallpaths  =  TRUE

If you do not have the security policy set up in advance (most people don't), you will see a message like this when entering the SET commands:

The SET command is currently disabled. The SET command is only an option Recovery Console command that can only be enabled by using the the Security Configuration and Analysis snap-in...

From the security policy, you can also tell XP not to require a password for the RC.  This is good if somebody can't get past the prompt for Administrator password or just to save time.

Here is a link that tells you more about these variables and the SET command:
http://support.microsoft.com/kb/235364
http://winhlp.com/node/78

If you are going to use any SET commands in the Recovery Console (and want them to work), you need to have previously set up the security options to allow the use of the SET command in the Recovery Console.  You can do this in a variety of ways, but GPEdit is what I use (won't work on XP Home of course - so you will have to do it in the Registry by hand).

So what you should do is make a Hiren's boot USB since it does not have these limitations.

Here's how:

I am going to recommend you use Hiren's boot CD (it will also go on a USB flash/thumb stick/drive).

This is good for you because it has many more tools on it that on the XP Recovery Console CD, does not care about your Administrator passwords and you will not have to futz around in your BIOS if any afflicted system has SATA drives - Hiren's can deal with that.

You will have a whole bunch of cool tools that you don't have in the XP Recovery Console...  a registry editor, password resetter, and a desktop that looks like Windows XP so you will feel comfortable maneuvering.

You can also easily copy your personal data (documents, images, music) to an external drive.

From a working system, first download the latest version on Hiren's Boot CD from here (it is a substantial download but worth it):

http://www.hirensbootcd.org/download/    (look near the bottom of the page).

Unzip the Hiren's to some folder where you can find it.  There is a Hiren's.BootCD.15.x.iso in there that you are going to need next.

Hiren's has instructions to make a bootable USB that you can use, but it requires you to first burn the .iso to a CD and some other steps, so I suggest another way and here's how:

Download RUFUS 1.4.x or whatever the latest version is from here (read some stuff on the page so you can know more about it):

http://rufus.akeo.ie/

I will caution you to be careful that you don't accidentally format any of your hard drives - be sure your USB stick is in and know what the drive letter is!  This part always makes me a little nervous, so be careful.

Launch RUFUS and all the defaults should be okay, for the Device, choose your USB drive letter, Quick format, FAT32, label it if you want to and in the Format options box, click the little icon that looks like a CD and a window will open.  Navigate that dialogue to point to the folder that contains the Hiren's.BootCD.15.x.iso that you unzipped earlier and the box should change to say ISO image (RUFUS understands the Hiren's ISO file).

Double check you have the right Device selected in the top (NOT you HDD) Click Start, acknowledge the warning and let it finish (it will take a little while) as it copies the files.   The Hiren's ISO is also good size.

When RUFUS is done, it will say 'DONE' in the bottom.

Put the USB stick in the afflicted machine and reset/reboot and press whatever key you need to press to get to a boot menu where you can select the USB as the first boot device (that is F11 for me).  If you don't see a boot menu choice, you will have to adjust your BIOS to boot from the USB first instead of the HDD.

When the Hiren's menu comes up, choose the Mini XP Mode and it will start loading (slowly from a USB drive) and eventually you should see a Windowsy looking desktop.  You should recognize that part and feel comfortable, but it is not your desktop - it is the Hiren's desktop!

Remember:  You did not boot on your hard disk - you booted into the Hiren's desktop.

I just want to not have to fresh install ever ever again !! The first 100+ was enough just want to relax and enjoy my time online :)

How would you like to never need to do a clean install of Windows 10 ever again - unless you want to? It's easy when you know how to backup.

You'll need image backup software to do this, and you'll need to spend some time learning how to use it. Once you're up to speed, do this:

1- Do a clean install of Windows 10.

2- After the installation has completed, run Windows Update until the computer is completely up to date. Make a note of the most recent installed update.

3- Make an image backup of your C: drive. The image backup is an exact copy of your hard drive at that point in time. It will take about 15 minutes for the software to make the image. Store the image on an external hard drive.

4- The next time you feel the need to re-install Windows 10, don't do it. Instead, restore the backup image that you made, which will also take about 15 minutes. When you restore the image, the computer literally goes back in time to when the image was made.

Isn't that so much better than spending a day re-installing everything?

After some time goes by, you might want to revise the original backup so it includes the updates to Windows that were released since you made the original backup. Watch how easy it is to do that:

1- Make a backup image of your computer as it is now.

2- Restore the backup of the original clean install.

3- Run Windows Update until the computer is completely up to date, again taking note of the most recent installed update.

4- Make an image of your newly updated computer, and store the image on an external hard drive.

5- Restore the image you made in step 1.

That's it! Why would anyone want to waste time on recovery disks and reinstalling when backup image software makes it so easy to recover your computer from trouble?
