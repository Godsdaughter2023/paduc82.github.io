jim gordon mvp say:

There's an old-fashioned way of moving an Excel color palette from one workbook to another. If you have to do this a lot you could probably automate the process with a macro.


1. Open the source workbook and the destination workbook.

2. Switch to the destination workbook (the destination workbook should now be the active workbook.

3. In Excel preferences, choose Compatibility preferences

4. Click the pop-up next to Copy Colors From

5. Choose the source workbook

6. Click the OK button


This method will work with workbooks saved in current Excel file formats made in Excel 2007 or later. Using this method with workbooks saved in old file formats made in Excel 2004 or earlier will result in incorrect color mappings.
