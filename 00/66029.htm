If you are referring to XP Mode when it takes that much time to load, you have to remember that it is only a virtual machine operating with only part of the resources available compared to XP which would have all your system resources.  What i mean by that is XP Mode by default only loads with 512MB of RAM and uses the virtualization aspect of the CPU.  So things will take a bit more time to load since it doesnt have access to all of the available resources compared to if you were running XP directly.
