frank fei msft say:

Hi FX8314,

You may enable only the .key layout file from custom keyboard bundle to see the outcome:

  1. Go to your ~/Library/Keyboard Layouts folder.
  2. Find the custom keyboard layout file that you want to continue to use. It should end in ".bundle".
  3. Move it out of that folder to the desktop or another folder.
  4. In its new location, Control-click on the file and choose Show Package Contents.
  5. Find the file that ends in ".keylayout" in the folder that appears (most likely under Contents > Resources.)
  6. Make a copy of the file that ends in ".keylayout" and put it in the Keyboard Layouts folder you opened in step 1.
  7. Log out and log back in.
  8. Add the custom keyboard layout in the Input Sources tab of Keyboard System Preferences. (It will be listed under Other.)

Thanks,
Frank
