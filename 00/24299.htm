wayne-i-m say:

You have this

SELECT Product_Master.[Product ID], Product_Master.[Current BU Name], Product_Master.[Product Name], Products_by_IntlAttainmentCenter.[Product ID], Products_by_IntlAttainmentCenter.[Product Name], Products_by_IntlAttainmentCenter.[Current BU Name], StrComp([Products_by_IntlAttainmentCenter]![Product ID],[Product_Master]![Product ID],0) AS Expr1
FROM Product_Master LEFT JOIN Products_by_IntlAttainmentCenter ON Product_Master.[Product ID] = Products_by_IntlAttainmentCenter.[Product ID]
WHERE (((StrComp([Products_by_IntlAttainmentCenter]![Product ID],[Product_Master]![Product ID],0))=0));

By adding the criteria =0 you are restricting the left join.  It is still a left join but it can't act as one as you have restricted the non comparable records

If you really need the criteria "=0" (we can't see what else is happening) you could use something like this - note the "SomeName"

SELECT Product_Master.[Product ID], Product_Master.[Current BU Name], Product_Master.[Product Name], Products_by_IntlAttainmentCenter.[Product ID], Products_by_IntlAttainmentCenter.[Product Name], Products_by_IntlAttainmentCenter.[Current BU Name], StrComp([Products_by_IntlAttainmentCenter]![Product ID],[Product_Master]![Product ID],0) AS SomeName
FROM Product_Master LEFT JOIN Products_by_IntlAttainmentCenter ON Product_Master.[Product ID] = Products_by_IntlAttainmentCenter.[Product ID];

Another option may be to remove the compare from the strcomp. 
So this
StrComp([Products_by_IntlAttainmentCenter]![Product ID],[Product_Master]![Product ID])
and not this
StrComp([Products_by_IntlAttainmentCenter]![Product ID],[Product_Master]![Product ID],0)
Although this is not really needed, the left join should give the same results

See the following pages of Greg Maxey's website :

http://gregmaxey.mvps.org/Create_and_employ_a_UserForm.htm

http://gregmaxey.mvps.org/Populate_UserForm_ListBox.htm

You can also do it using Content Controls - If you have a ComboBox Content Control to which you have assigned the Title "Category" with the items A, B and C, and another one to which you have assigned the Title "Items" and to the This Document Object for the Template in the Visual Basic Editor, you add the following code

Dim CC1 As ContentControl
Dim CC2 As ContentControl
Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
With ActiveDocument
    For Each CC1 In .ContentControls
        If CC1.Title = "Category" Then
            For Each CC2 In .ContentControls
                If CC2.Title = "Items" Then
                    CC2.Range.Text = "Select Item"
                    Exit For
                End If
            Next
            Select Case CC1.Range.Text
                Case "A"
                    For Each CC2 In .ContentControls
                        If CC2.Title = "Items" Then
                            With CC2.DropdownListEntries
                                .Clear
                                .Add "Item A1"
                                .Add "Item A2"
                                .Add "Item A3"
                            End With
                            Exit For
                        End If
                    Next
                Case "B"
                    For Each CC2 In .ContentControls
                        If CC2.Title = "Items" Then
                            With CC2.DropdownListEntries
                                .Clear
                                .Add "Item B1"
                                .Add "Item B2"
                                .Add "Item B3"
                            End With
                            Exit For
                        End If
                    Next
                Case "C"
                    For Each CC2 In .ContentControls
                        If CC2.Title = "Items" Then
                            With CC2.DropdownListEntries
                                .Clear
                                .Add "Item C1"
                                .Add "Item C2"
                                .Add "Item C3"
                            End With
                            Exit For
                        End If
                    Next
            End Select
            Exit For
        End If
    Next
End With
End Sub

The items in the ComboBox with the Title of "Items" will be based on the selection made in the ComboBox with Title of "Category"

The various methods of loading data into comboboxes\listboxes from external sources detailed on Greg's website could be modified to work with Content Controls.
