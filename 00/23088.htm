ActiveSheet.Protect Password:=str

Sub UnProtectSheet()
On Error Resume Next
Dim str As String

ActiveSheet.Unprotect Password:=str

Sub Protect_All()
Dim x As Long
On Error Resume Next
Dim str As String
str = InputBox("Write the password = ")
If str = "" Then Exit Sub
For x = 1 To Worksheets.Count

End Sub

If a user’s ForwardingSMTPAddress value is not null in the result, the user has set to forward their email to another email address.

If the users use inbox rules to forward emails, you can change the second PowerShell command above to the following command:
Get-InboxRule | Select Identity,Name,Description | fl
The result of this command will be the inbox rules of all users. You can identify which user has an inbox rule to forward emails by checking the Identities of the users and the detailed rule description.
