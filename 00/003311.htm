Drop Cap Settings - Microsoft Community

joan leacott say:

Drop Cap Settings

I have to create 271 drop caps! Going through the dialogue box gets tedious fast! How can I change the default settings, font and size, in Drop Caps so I don't have to go through that dialogue every... single... time?

Thanks, Joan

jay freedman say:

Install the following macro in your Normal.dotm template and assign a keyboard shortcut or a Quick Access Toolbar button to it:

Sub MyDropCap()
    With Selection.Paragraphs(1).DropCap
        .Position = wdDropNormal
        .FontName = "Castellar"
        .LinesToDrop = 2
        .DistanceFromText = 3 ' in points
    End With
End Sub

Edit the bold items to use the desired font name, number of lines of text, and distance from the text in points.

To use the macro, click in a paragraph that needs a drop cap and then press the shortcut or click the button.
