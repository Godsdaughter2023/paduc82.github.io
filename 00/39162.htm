greg carmack - mvp 2010-2020 say:

Files not backed up at all times are valued as worthless since the hard drive can die at any time.  You can set up automatic backup to sync them to another hard drive or external as often as you want, or to sync across all your devices use a solution like Sync, Backup and Store your Files to the Cloud with OneDrive - Microsoft Community .

Your most immediate fix would be to

 boot using the BIOS Boot Menu Key.   If a UEFI or EFI verison of the media is present choose it only.  Then navigate to second screen "Repair Your Computer" choice, under Troubleshoot do a Startup Repair, or if present Go Back to Old OS - but this may be a damaged recovery compared to going forward to do the best possible install of Windows 10. 

If all these will not start Windows then do a Clean Install Windows 10 Upgrade

choosing the C Windows partition to click Next without formatting so that it doesn't wipe out the data but preserves it in the c:\windows.old archive. 

You can also use the same media to first rescue the files to USB or another hard drive following Copy & Paste - in Windows Recovery Console.  If you get the files out then you can delete all partitions during the booted install to get install cleanest, as illustrated in Clean Install wiki. 


As to problems with booting a disk, these are almost always user failures having to do with bad timing triggering the media to boot, waiting too late to press the correct BIOS Boot Menu Key, not heeding "Press any key to boot disk" in time.  If nothing works then try a DVD instead of flash stick, or vice versa.  You can also take the hard drive out to connect to another PC to rescue data, then delete all partitions in Disk Mgmt which should force installer to autostart when returned to old PC.
