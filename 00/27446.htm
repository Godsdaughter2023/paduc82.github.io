    hansv mvp say:

    You could create a pivot table based on the data, add the Created Date, Call Status and Assigned User fields to the Report Filter area and any of the fields to the Values area. You can then filter the data any way you want.

     

    If you prefer formulas:

    Enter the date 08-Jul-11 in a cell, say H1.

    Enter the user name Priya in another cell, say H2.

    Enter the status Held in a third cell, say H3.

     

    Total number of calls on 08-Jul-11: =COUNTIF(A2:A100,H1)

    Number of calls for Priya on 08-Jul-11: =COUNTIFS(A2:A100,H1,D2:D100,H2)

    Number of calls for Priya with status Held on 08-Jul-11: =COUNTIFS(A2:A100,H1,D2:D100,H2,C2:C100,H3)

     

    Adjust the ranges as needed. If you change the value of H1, H2 and/or H3, the result of the formulas will be updated automatically.
