Well, we could do that, but it's very special and would only work with a few strings.

Identifying duplicates is an usual procedure, but a programmer thinks more universal.

I suggest to separate the words by a given set of delimiters and search for duplicate occurrences of a word with a min. length.

Boris met; Bor went, Boris wept

this macro would return true (because it would find Boris twice).  If the cell was:

Boris met, Bor went, Bori sent

the result would be false.

Put all code below my name in a normal module and use this formula in your worksheet (I assume that your string is in cell A1).

=HasDuplicates(A1,"; ,",5,1)

Option Explicit

Function HasDuplicates(ByVal S As String, _
    ByVal Delimiter As String, _
    Optional MinLength As Integer = -1, _
    Optional Compare As VbCompareMethod = vbTextCompare) As Boolean
  'Searches for duplicate occurrences in S, words are separated by any char in Delimiter
  Dim Data, i As Long, j As Long
  'Eliminate double delimiters
  S = OneDelimiter(S, Delimiter)
  'Seperate the words
  Data = Split(S, Left$(Delimiter, 1))
  'Search for a double occurrence
  For i = 0 To UBound(Data) - 1
    If MinLength < 0 Or Len(Data(i)) >= MinLength Then
      For j = i + 1 To UBound(Data)
        If StrComp(Data(i), Data(j), Compare) = 0 Then
          HasDuplicates = True
          Exit Function
        End If
      Next
    End If
  Next
End Function

Function OneDelimiter(ByVal S As String, ByVal Delimiter As String, _
    Optional Compare As VbCompareMethod = vbTextCompare) As String
  'Replace all occurrences of each char in Delimiter with the first char in Delimiter _
    and remove double delimiters
  Dim i As Long
  For i = 2 To Len(Delimiter)
    S = Replace(S, Mid$(Delimiter, i, 1), Left$(Delimiter, 1), Compare:=Compare)
  Next
  Delimiter = String(2, Left$(Delimiter, 1))
  Do
    i = Len(S)
    S = Replace(S, Delimiter, Left$(Delimiter, 1), Compare:=Compare)
  Loop Until Len(S) = i
  OneDelimiter = S
End Function
