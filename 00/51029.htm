Advanced users might find the details in event logs helpful when troubleshooting problems with Windows and other programs.


Event Viewer tracks information in several different logs. Windows Logs include:

 

a. Application (program) events. Events are classified as error, warning, or information, depending on the severity of the event. An error is a significant problem, such as loss of data.
