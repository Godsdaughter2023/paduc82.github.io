swischner say:

I don’t have an actual solution to your problem but an explanation that might help understand why this is happening.

Every time you copy some content to the clipboard in certain Windows applications (including OneNote), there are several different versions of that content stored in the clipboard. So, if you select some text and press CTRL+C, the clipboard gets a plain text version, jpg, bmp and tiff bit map image versions, a HTML version, even an icon version and others.

I could verify this using a small tool called CopyQ which, besides other functions, lets you inspect the clipboard content.

Now, which of the versions of the clipboard content is transferred to a target app when you press CRTL-V is totally controlled by the target app. So, when you expect text to be inserted but you get a bitmap image of the text instead, the target program has chosen the incorrect version of the clipboard content. I guess this is the problem here. Some programs like Word or Excel do allow you to somewhat control the version that is getting inserted via the tiny control bar that appears when pasting from the clipboard (plain text, image..), while most others don’t.

The mentioned CopyQ tool also let’s you select a specific version of the content to be inserted, but of course that’s not very convenient as it requires a few mouse clicks.

Adding external emails into group - Microsoft Community

a. user say:

Adding external emails into group

I need to send emails from my corporate email to external parties which majority of them having emails(yahoo,gmail,etc).I have added them into the contacts but I find it difficult to differentiate them since most are under common company/corporate organization.It looks messy on my contact list because I have to rename it with different name to indicate.Is it possible to add them into groups?

zoeyzan_msft6423 say:

hi nash,

there are two ways to achieve it:
1.doing this in your personal mailbox. create a contact list and add these external users in it. (only you can send emails to this contact list)
  create a contact list
  1.login to the portal->click people.
  2.click list->new contact list.

 

  add external users to list
  choose the user and click add to as below

 

finish creating the list named 123, you can input 123 in the from field when you are composing the emails.

2.doing this in an organization level. create mail users and then add them in a distribution group. (all users in your organization can send emails to this distribution group)
  create the mail user
  1.login to exchange admin center as an admin.
  2.click recipient->contacts, and create a mail user.
  3.in the pop-up window, input the information especially external email address.

after creating the mail user, you can add this user into a distribution group.

  create a distribution group
  1.click recipients->groups.
  2.create a new distribution group.
  3.add the mail user as a member.

please note: the information which put into an organization level will share will all members in your organization.
