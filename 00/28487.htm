jim gordon mvp say:

Normal.dotm is the template Microsoft Word uses to make a new, blank document. It is sometimes called the "Global" template, or simply the "Normal" template.

If there is something wrong with your Normal template, you can rename it or delete it while Word is not running. You must complete quit Word before renaming or deleting this file. When you open Word again, it will make a new, pristine blank Normal template to use.

Changing all the images brightness,contrast and Saturation levels at - Microsoft Community

malliga k say:

Changing all the images brightness,contrast and Saturation levels at once!!

I am using MS word 13 in my PC. I have created one document which has around 100 images. I need to change all the images brightness,contrast and Saturation levels at once. Is there any way to do it ? If so please let me know.

jay freedman say:

Any time you need to do the same thing to many objects in a document, you need a macro.

In this case, you can do something like this (see http://www.gmayor.com/installing_macro.htm if needed):

Sub Macro1()
    Dim ILS As InlineShape
    For Each ILS In ActiveDocument.InlineShapes
        With ILS.Fill.PictureEffects

         ' Insert a 170% Saturation effect.
         .Insert(msoEffectSaturation).EffectParameters(1).Value = 1.7

         ' Insert Brightness/Contrast effect and set values to 30% Brightness and 25% Contrast.
         Dim brightnessContrast As PictureEffect
         Set brightnessContrast = .Insert(msoEffectBrightnessContrast)
         brightnessContrast.EffectParameters(1).Value = 0.3
         brightnessContrast.EffectParameters(2).Value = 0.25


        End With
    Next ILS
End Sub

You can change the numbers as needed. If your pictures are not in line with text, you'll have to change "InlineShape" in the second line to "Shape" and change "InlineShapes" in the third line to "Shapes". If your document has a mixture of inline and floating shapes, use two macros (each one will affect only the specific kind of picture).
