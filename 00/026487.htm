This solution involves using Conditional Formatting to highlight the Active Row. However, it also requires a small amount of VBA code to force the screen to update.

Not sure of your expertise here so I will include full detailed guidelines.

  • Select the entire Worksheet (Click the top left corner where the Column Id's and Row Id's meet.)
  • Select Home ribbon -> Conditional Formatting -> New Rule
  • Select "Use a formula to determine which cells to format"
  • Copy the following formula and paste into the blank field.           =CELL("row")=ROW()
  • Click "Format" button.
  • Select "Fill" tab (at top of dialog)
  • Select a color of choice for the fill. Needs to contrast well with your font color. (Yellow contrasts well with black font)
  • Continue clicking OK until Conditional Formatting dialog closes.

Install the VBA code as follows.

  • Right click the worksheet tab name.
  • Select "View Code" (the VBA editor will open at the worksheets module)
  • Copy the following code and paste into the VBA editor. (3 lines of code between the asterisk lines but not the asterisk lines)

'****************************************************************
Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    Application.ScreenUpdating = True
End Sub
'****************************************************************

  • Close the VBA editor (Cross very top right of window)
  • Use "SaveAs" to save the Workbook as Macro Enabled.
  • Ensure Macros are enabled as follows.
  • If Developer ribbon not displaying then right click anywhere in one of the ribbons and select "Customize the ribbon".
  • On the right side of the dialog check the box against "Developer" and click OK. (Ensure you click OK; not just close the window or the choice will not be save.)
  • Select Developer ribbon.
  • Select Macro Security (In the Code block at left end of Developer ribbon)
  • The dialog that displays should default to Macros in left column. (If not, select Macros)
  • Select required security option button. (Option to "Disable all macros with notification" should be OK.)
  • Click OK to close the dialog.

Now when you select any cell in the worksheet then the row will be highlighted with the color you selected in the Conditional formatting.

I don't expect you will have problems with this but feel free to get back to me if you do have problems.

john mcghie say:
Mmmmm...  Track Changes is very flaky: I am not sure I would trust it to show "moves" in any document.

Basically, I do not use Track Changes because on complex documents it is almost certain to break the document eventually, particularly if you have "keep track of formatting" switched on.  I keep a copy of the original and use "Compare Documents" at the end.  It achieves the same effect, but won't break the document.

Track Changes does observe a hierarchy when deciding how to show text.  If you have changed a sentence, deleted a sentence, added a sentence, then moved the whole paragraph, I would not expect it to show that as a "move", because the added text has been "added", it was not there before to be "moved", while the "deleted" text should not be there now to be shown as "moved".

From my testing, it will show a "Move" only if:

a)  You move a whole unchanged paragraph
b)  You move by cut-and-paste (not drag and drop).

Personally, as I said, I would use "Compare Documents" because I suspect your document is now living on borrowed time :-)

Hope this helps
