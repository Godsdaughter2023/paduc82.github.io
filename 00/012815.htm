In excel 2010 check box option, if i insert the check  box in cell i want the next cell to reflect as Active.

For eg: If in Column A, cell A3 is inserted Check Box, i want B4 cell to reflect as Active.

    Column A    Column B
   Check Box       Active
  Unchecked Box    Blocked

Please show the way for this.

ronaldoonenil say: Right click your check box control and select format control. On the control tab you need to enter a cell link (This can be a cell somewhere on your spreadsheet that isn't displayed e.g. Z2)

When you check or uncheck the box, this will put True or False in Z2

Your Formula in B2 can then be =IF(Z2=True,"Active","Blocked")

Repeat for all the check boxes in column A using link cells in column Z
