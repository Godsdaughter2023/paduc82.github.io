jay freedman say:

Let me be as explicit as I know how:

You only need ONE template -- which does not have to contain any text in its body, just macros -- stored in the Startup folder. Pick one of the three you posted, and delete the others from that folder. (I didn't check to see if the _DocumentBeforePrint event handlers in all the posted templates are the same, but the one you keep should contain every case that appears in any of these class modules.)

If any of the document templates (in the Templates folder, not in Startup) also have _DocumentBeforePrint event handlers, delete any code from them that has to do with those handlers -- all the class modules, all AutoExec, FileOpen, and FileNew macros, and any Word.Application declarations. Leave only code that's specific to documents based on the template.

Do not create new documents based on any template in the Startup folder. Those templates aren't supposed to supply anything but macros.
