Because the status bar text is updated with all sorts of information during editing sessions, it isn't a good location to hold the file name. There is a solution, though: Go to File > Options > Quick Access Toolbar. Select the "All Commands" category, scroll to and click on the Document Location control, and click the Add button.

The control is available in Word, Excel, and PowerPoint. It is not in Access, OneNote, or Outlook.

jay freedman say:

The following macro will put the file name in the status bar, but that will be overwritten with the usual status information as soon as any other event occurs, such as moving the cursor or typing a character. You would have to put a button for the macro on the Quick Access Toolbar or the ribbon, or create a keyboard shortcut, and run it each time you want to see the file name in the status bar.

Sub FN_statusbar()
    Application.StatusBar = ActiveDocument.Name
End Sub

There is no way to add anything to the built-in set of items that can appear on the status bar (the list of things in the bar's right-click menu) through a macro or any other customization. You can suggest that the file name be added to that list by Microsoft, by posting at https://word.uservoice.com/.
