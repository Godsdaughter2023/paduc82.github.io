bob jones aka: cybertaz say:
If I understand you correctly, you want the result in Word to mimic the formatting you see in the source you copy from? That may not be possible.

Much is determined by the source app you're using when copying, the nature of the source you're copying from, as well as exactly how you go about executing the copying.

Copy/Paste is an OS functionality supported at different levels by the programs you use. Especially if you're copying from web pages [source] using a browser [source app] that level can be very limited,  so what you actually see may not be exactly what gets copied. In a nutshell:

  • The source you copy from determines what is available to be copied.
  • The app in use when you copy determines what of that goes onto the Clipboard.
  • The app you use when pasting can pick only from what's available on the clipboard, regardless of how powerful that app may be.
I don't see how a Theme would help with this, but you most definitely can format Hyperlinks using different Character Styles. Also, you can have as many Hyperlink Character Styles as you wish as long as each one is given a unique name. The character formatting you choose to use in each one is up to you. Base each custom style on the built-in Hyperlink style &/or base one custom style on another.

Before you go to that much trouble, though, just a couple of admittedly biased opinions, but I believe them to be well-founded on experience :-)

Technicolor documents are an eyesore, the beauty of which usually lies strictly within the eye of the creator,
Inconsistent formatting of like content in the same document is confusing & misleading to the reader,
Fluctuation in fonts used, font sizes & other formatting not only are hard on the reader's eyes but also make a document more complex to control & revise for its editors.

Track changes - multiple colors - Microsoft Community

simonagi say:

Track changes - multiple colors

is it possible to track changes in subsequent versions in a different color if you are the same author?  When I change colors in the options menu it makes all the old changes the same color as the new ones.
jay freedman say:
Yes.... Go to File > Options > General and make some change to the User Name shown there -- for example, add a number to the end of it. Leave the Track Changes options set to "By Author". Track Changes will now see any new changes as being made by a different author, so it will use a different color.

You can change your user name multiple times to get a different color each time. After a certain number, though, Word will have cycled through all the colors in the list and will start repeating.
