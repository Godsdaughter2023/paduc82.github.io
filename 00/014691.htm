Aligning center and right on the same row

jay freedman say:
There are two basic approaches to doing this.

In the first approach, insert a center-aligned tab stop at the middle of the text column (http://www.word.mvps.org/FAQs/Formatting/SettingTabs.htm) and a right-aligned tab stop at the right margin. (The paragraph itself should be left-aligned.) Starting at the left margin, insert a tab character, enter the formula, insert another tab character, and enter the formula number.

As a refinement, you can define a style that includes the tab stops, and assign a keyboard shortcut or a button on the Quick Access Toolbar to apply the style when you want to use it.

In the second approach, insert a two-column, one-row table. Press Ctrl+Alt+U to quickly turn off the table's borders (you can go to the Table Tools | Layout tab of the ribbon and click View Gridlines so you can see nonprinting lines where the borders were). Drag the vertical edge between the two cells so the cell on the right is just big enough to hold the formula number. In the left cell, set the paragraph alignment to center (Ctrl+E); in the right cell, set it to right (Ctrl+R). You can save the formatted (but empty) table as a building block by pressing Alt+F3 and assigning a name to it.

jay freedman say:

One method is given in http://wordfaqs.mvps.org/CleanWebText.htm.

If the text that precedes has any direct (non-style-based) paragraph formatting, that method would remove it. Another method can preserve the formatting:

  • Open the Replace dialog and click the More button to expand it.
  • Check the "Use wildcards" box.
  • In the Find What box, insert this expression:

(^13)[ ^13]{1,}

  • In the Replace With box, insert this:

\1

  • Click the Replace All button.

The ^13 code matches a paragraph mark. The part of the expression in square brackets includes a space character, so the replacement will pick up paragraphs that contain only spaces.

Microsoft Word 2010.

Is there a way to do this?  So that it stays the same look, but still works as a hyperlink? 

 

Thank you.

jay freedman say:
A hyperlink that hasn't been followed yet has the Hyperlink style applied to it. When you click it, Word applies the FollowedHyperlink style to it.

The colors of these two styles are determined by the definition of the theme applied to the document. You can define a custom theme that has the same blue color for both the Hyperlink and FollowedHyperlink, so the link's formatting won't change when you click it.

  • On the Page Layout tab of the ribbon, click the Themes button and choose a theme you like. A theme includes both a set of colors and a pair of fonts (headings and text).
  • Next click the Colors button to the right of the Themes button. At the bottom of the dropdown, click Create New Theme Colors.
  • In the dialog, click the little arrow next to the Hyperlink color sample and note which color block is selected. Close that dropdown without changing anything.
  • Click the little arrow next to the FollowedHyperlink color sample, and click the same color block as seen for the Hyperlink color. The two samples should now look the same.
  • Type a name for this color set in the Name box and click Save.
  • Click the Themes button again, and choose Save Current Theme. A dialog like the Save As dialog will open, with the Document Themes folder already selected. Either accept the suggested name or type a more recognizable name, and click Save. The theme will be saved in a file with a .thmx extension.


For any future documents, you can choose this theme from the Themes button, where it will appear at the top of the gallery.

The Change Text Direction button is only available when a vertical text language (such as Japanese) is enabled. Try following steps after adding 'Change Text Direction' button to the toolbar:

Enable a vertical text language

1) Click Start, click All Programs, click Microsoft Office, click Microsoft Office Tools, and then click Microsoft Office 2007 Language Settings.

2) Click the Editing Languages tab.

3) Select the vertical text language you want to enable in the Available editing languages list, and then click Add.

Change text direction

1) Select the shape.

2) Click the Change Text Direction button on the Formatting toolbar. Observe that the shape's text rotates 90° clockwise and appears in a vertical font.
