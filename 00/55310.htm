aakkam22 say:

Hi, 

Your error code indicates a Windows Update problem. That is how Windows Defender is updated, so if updates aren't working, then Defender won't work either.

Check to verify that Windows Update's services are correctly started and configured. Here is how:

1. Open the services console (open the Start menu, type services.msc and hit Enter)

2. Find the Windows Update service (it's at the bottom)

3. Under the Status column, check that the service is running. Under the Startup Type column, check to make sure that the service is set to Automatic. If it is not, double-click the service, and then use the drop-down to select the Automatic startup type.

4. Repeat this process for the following services:

a) Background Intelligent Transfer Service (BITS)

b) Cryptographic Service

If this did not work for you, you can try running the Windows Update Reset Tool to reset Windows Update and its components.

•Ntuser.dat
•Ntuser.dat.log
•Ntuser.ini

e) Click/tap on “delete the account”.
f) Depending on whether you want to “Keep the files” or “delete the files” choose the appropriate option.

anitamills53 say:

This problem is now resolved by using the AVG PC TuneUp utility to resolve other windows problems, selecting the option for the correct display of folders in Windows explorer. This revealed that the folders had all been moved into another directory (Mes Images not Mes documents), leaving a copy of just one subfolders and two of its subfolders containing  some previously deleted files. How this occurred is not clear, these are big complex directories and require admisitrator access, not something I could move by mistake and not notice!

But when the folder I pinned actually revealed the correct path, I was able to delete the corrupt folders and files, make backups and remake the directories which now display correctly under My PC. And I can also see two peripherals not previously visible in the network.
