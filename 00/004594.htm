This is certainly not something that should be happening. When you open a Word 2007 document in a newer version, the title bar will tell you that the document is in Compatibility Mode. Unless you explicitly resave the document in the newer format, it will remain a Word 2007 document and should be displayed the same. If you save in the newer format, it's conceivable that some formatting would be changed, but you shouldn't be experiencing what you're seeing.

I'm guessing that this is a display issue only and that you don't see this in Print Preview (or at least not when you print)? If so, there are a couple of settings that may help:

  1. At File | Options | Advanced: Display, check the box for "Disable hardware graphics acceleration."
  2. At File | Options | Advanced: Display, change the setting for "Optimize character positioning for layout rather than readability."

If neither of those helps, you might check for an updated display driver for the computer.

In the Table Properties dialog, make sure that the Text wrapping is set to None.  To get the header row to repeat, you must click on the Repeat Header Rows item in the Data section of the Table Tools>Layout tab of the Ribbon.
