    frederik long say: You could do something like this:
    1. Create a scheduled task on each machine. It must run at boot time under an admin account and with the highest privileges.
    2. The task should check if a certain semaphore file is present on the local disk. The semaphore file gets created by the standard logon script.
    3. If the semaphore file is present then the scheduled task should invoke a batch file that you keep in a proteced location on the server. The batch file has two tasks:
      - Delete the semaphore file
      - Perform whatever task you want it to perform, e.g. run regedit.exe.
    It is a lot of trouble to set up a scheme like this and it might be faster for you to run the regedit command manually at each machine.

    "How to Refresh Windows 8 and Windows 8.1"

    • The PC boots into Windows RE (Recovery Environment).
    • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
    • Windows RE installs a fresh copy of Windows.
    • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
    • The PC restarts into the newly installed copy of Windows.

    "When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it.
