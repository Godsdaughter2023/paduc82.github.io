echo s say:

As Steve said, if you're using a click-to-run installation, you won't have any options to do custom installation settings (e.g., add the old equation editor, etc.).

You can copy your themes to this folder and they will appear in the themes gallery on the Design tab: 

C:\Users\UserName\AppData\Roaming\Microsoft\Templates\Document Themes

You can also put them in your default personal templates location, which you can find (or change) by going to File | Options | Save in PowerPoint. Templates and themes placed here will cause you to have a PERSONAL or CUSTOM option next to FEATURED on the PowerPoint File | New tab (start screen). Templates and themes saved in the default personal templates location are available there. You can add subfolders there to help you organize. (See bottom pix.)

As an aside, I'm not sure why I now see a folder named PERSONAL whenI have been seeing it named CUSTOM. I'm not sure if this is a difference that MSFT has made in a recent update or if it's the difference between a MSI install vs a Click-to-Run install, or if it's corporate vs Home & Student or what. But you should have one or the other available.

PowerPoint doesn't actually ship with more themes. Instead, MSFT controls what shows up on the File | New tab, and they occasionally roll out new "stock" themes there. Recently there's been a problem where that live content isn't showing up, so you may only see 5 or 6 stock themes there.  You may want to get in the habit of right-clicking and pinning themes to the list if you want to keep some specific themes available regardless. 

 If you want to download more themes, use the search box at the top of the File | New page where it says, "Search for online templates and themes."
