Add "Size and Position" (Layout) to Command ribbon - Microsoft Community

kamyot say:

Add "Size and Position" (Layout) to Command ribbon

Hello,

I would like to be able to access the Layout dialogue box in the Command ribbon. (This is so that I can record a macro to specify the absolute width of a graphic on the Size tab.) However, in the All Commands menu, I can't find a command to display this display this dialogue box. Is there one?

The "Picture Size" command works most of the time, but only the  Layout dialogue box works all the time

Thanks.

jay freedman say:

There are actually four commands in the All Commands list, each of which opens the Layout dialog. The one most suited for your purpose is "Advanced Layout: Size", which opens the dialog with the Size tab already visible. It's equivalent to clicking the dialog launcher (the little arrow in the bottom right corner of the Size group on the Picture Tools > Format ribbon).

However, although this command works perfectly while not recording a macro, using the dialog while recording causes Word to crash.

Instead of recording a macro, use the following code (see http://www.gmayor.com/installing_macro.htm if needed):

Sub PictureAbsWidth()
    Dim sngWidth As Single
    
    ' change the 3.5 in the next line to the desired
    ' width in inches
    sngWidth = InchesToPoints(3.5)
    ' or replace the line above with the following,
    ' using the desired width in centimeters
    ' sngWidth = CentimetersToPoints(10.5)
    
    Select Case Selection.Type
        Case wdSelectionShape
            Selection.ShapeRange(1).Width = sngWidth
        Case wdSelectionInlineShape
            Selection.InlineShapes(1).Width = sngWidth
        Case Else
            MsgBox "Please select the picture to resize"
    End Select
End Sub

suzanne s. barnhill say:
There's no simple way to do this. You can change the direction of the text in any given cell, but in order to rotate the entire table by this method, you must first paste the content into Excel, select and copy it, then use Paste Special with the box for "Transpose" checked. This converts rows to columns and vice versa, and you can then paste the content back into Word and change the text direction. The downside to this is that, since your headings are the first column rather than the first row, you cannot have them repeated.

yes, you are right. currently, we are not able to see the shared calendar in the outlook for ios & android app. as a workaround, you can use owa for iphone or owa for android. your understanding is highly appreciated.

please note: to make the shared mailbox show up on the owa for ios & android, please first add the shared mailbox into the outlook web app on your pc by referring to the following steps:

1. sign in to owa.

2. right click the display name, then click add shared folder…, type in the email address of the shared mailbox, then click add.

to make the shared calendar show up on the owa for ios & android app, please also add the shared calendar into the owa calendar on your pc by referring to the following steps:

1. sign in to owa.

2. click calendar, right click “my calendar” > open calendar.

after adding the shared calendar & mailbox in owa, the change will be synced to your owa for ios & android app.
