Hi there! :-)

Been working on this for half a day. This solution worked for me. I was trying to gain access to the folder shared to me by my boss. I can only get to it via browser access but I need to edit the files and sync them automatically since it contains a huge number of documents.

STEPS I TOOK:
1. Open Onedrive in browser and enter your credentials.
2. Click on "Return to classic OneDrive" on the bottom left corner of the page
3. Select Shared with me on the left pane of the page.
4. Select the files and folders you need to sync to your PC. NOTE: A checkmark should appear right before the file.
5. Click on the Sync button at the top.
6. A dialogue box will appear. It's "Sync your files to this PC".
7. Check if everything you need syncing has been selected.
8. If everything is OK, click the Start sync button.


Voila! You're good to go. Hope this helps! :-)
