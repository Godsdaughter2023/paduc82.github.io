avrilw10 say:

I had the same problem with Night Light. This is how I fixed it quickly.

  • Open the Settings icon - little cog wheel - accessed via Start icon or via the Notifications icon on bottom right.
  • Click on Update & Security.
  • Click on Troubleshoot from left side menu.
  • Scroll down. Under the heading Find and Fix Other Problems, click on Hardware and Devices, then click on Run the troubleshooter. It found a problem with the monitor and fixed it.
  • You will then be requested to restart your computer. Your Night Light should now be fully functional.

This is much simpler than any other method I have seen on the forum. Hope this helps.

BTW, if you have ANY problems with Windows 10, try the troubleshooter first. If it doesn't help, click on Recovery (the menu option under Troubleshoot) to see the options available to fix Windows. You should also create a Recovery Drive periodically. This makes a mirror image of your system, including files, saved onto a USB drive. It is then easy to go back to the last known pristine version of Windows 10.

To create a Recovery Drive, open Settings. Above the icons you will see a search bar. Type in Create Recovery Drive. A drop down menu appears. Click on Create Recovery Drive and follow the instructions. You will need an 8GB USB drive for a 64 bit system.
