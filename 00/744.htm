jim gordon mvp say:

Hi Maoira

I split your reply off so that people can respond to your specific situation rather than simply address things in general.

Let's start by clearing up some vocabulary.

Office 365 is a license type that provides a subscription to Microsoft Office and other Microsoft products. Office 365 is not a product. Microsoft Office 2011, 2016 and 2019 desktop products are all products that were or are still offered as part of an Office 365 subscription.

There is a completely free on-line (cloud-based) version of Microsoft Office at OneDrive.com. The free version is sufficient for many users. If that version has everything you need, you don't have to worry about the desktop version, which has many more features.

Apple is threatening to withdraw support for 32-bit applications, including printer drivers. So far, Apple has not carried out the threat. Apple throws warnings up about the upcoming demise of support so that you can go buy new hardware now and not be stranded when it happens. On the other hand, if you don't update Mac OS past the current update, you can continue to use your old hardware and software (Office 2011) indefinitely.

Since you have access to an Office 365 subscription, you are entitled to obtain Microsoft Office 2019 desktop version. It requires Mac OS 10.12 or newer. Ask your company's subscription administrator for a username and password and you can log into portal.Microsoft.com, download the software and install it. The desktop version is partially "cloud-based" in that in order to obtain and use it, you'll need an internet connection. When you open an Office application, it needs a connection to verify its authenticity. There are some features that require an internet connection. You can store files locally, or in a cloud location such as SharePoint or OneDrive.
