xbr poseidon say:

Overview

SMC8014WG-SHW

Login Information

  • Gateway – 192.168.0.1
  • Username – cusadmin
  • Password– password
    *The login password may be the wireless passphrase which depending on the install tech could be in all capital letters.*

How to find the SSID (Network name)

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Select the Channel and SSID link in the sub menu.
  6. The current SSID (Wireless network name) will be located here.

 

 

 Changing the Wireless Password
*The Wireless passphrase along with the encryption type and network name can be found on a sticker on the side of the device*

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Select Encryption from the sub category.
  6. Verify the wireless passphrase or security key.

 

MAC Filtering

  1. Connect a computer/laptop with an Ethernet cable to the gateway.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  4. When prompted to login enter “cusadmin” as the username and “password” for password.
  5. Select Wireless from the menu on the left.
  6. Select the MAC filtering sub category.
  7. Select the Trusted PC’s Only option.
  8. Towards the bottom of the page type in a device name (like “Xbox”).
  9. Type in the Xbox’s MAC address in the boxes provided.
  10. Click on the Add button.
  11. Repeat this process for all wireless devices connecting on your network.

 

Disabling MAC Filtering

  1. Connect a computer/laptop with an Ethernet cable to the gateway.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  4. When prompted to login enter “cusadmin” as the username and “password” for password.
  5. Select Wireless from the menu on the left.
  6. Select the MAC filtering sub category.
  7. Under the Wireless Access List check the devices you would like to remove/disable.
  8. Once you have checked the devices select the delete button to remove them.
  9. Repeat this process for all desired wireless device connecting on your network.

 

 

Changing the Wireless Channel

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Select Channel and SSID.
  6. From the drop down menu labeled Channel select the desired channel. (Channels 1,3,9,11)
  7. Click the Apply button.

 

Changing Wireless Mode

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Set the mode to 11G only.
  6. Click the Apply button.

 

Change Wireless Band (width)

The default wireless band is 20MHz and cannot be changed.

 

 

DHCP Reservation

Not available

 

 

UPnP

Not Available

 

 

DMZ

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select Firewall.
  5. From within the Firewall subcategory select the DMZ option.
  6. Check the Enable DMZ Host box.
  7. Enter the consoles IP Address in the boxes to the right.
  8. Click the Apply button at the bottom of the page.

 

Disable DMZ

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select Firewall.
  5. From within the Firewall subcategory select the DMZ option.
  6. Uncheck the Enable DMZ Host box.
  7. Click the Apply button at the bottom of the page.

 

 

Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select NAT.
  5. From the NAT subcategory select Port Forwarding.
  6. Click the Add button under the Customer Defined Service Table.
  7. In the Name field enter XBL or something that indicates Xbox Live.
  8. In the type drop down select TCP/UDP.
  9. In the LAN Server IP enter the IP address of the console.
  10. For the Remote IPs select Any.
  11. From Public IP Ports drop down select Single Port.
  12. In the Start Public Port box enter 3074.
  13. Leave End Public Port blank.
  14. In the Private Ports box enter 3074.
  15. Click the Apply button.

 

Disable Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select NAT.
  5. From the NAT subcategory select Port Forwarding.
  6. Check the ports that have been forwarded.
  7. Select the delete button.
  8. Click the Apply button.

 

 

Firewall

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select Firewall.
  5. On this screen set the Firewall to Disable.
  6. Click the Apply button.

 

 

Updating Firmware

Not Available

 

 

Restoring Defaults

Through Hardware

  1. Locate the Reset button.
  2. Using the tip of a ballpoint pen or toothpick press and hold the reset button for 15 seconds.
  3. Release the reset button.
  4. Wait for the device to restart.

 

Through Firmware

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Tools from the menu on the left.
  5. Select Configuration Tools from the subcategory on the left.
  6. Click the Factory Reset button.

 

 

Multiple Console Support

We have confirmed that this network device is unable to support more than one console with an Open NAT. 

 

 

Known Issues

Wireless passphrase is locked and can not be changed
