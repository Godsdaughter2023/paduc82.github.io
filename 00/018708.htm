Find and Change on Table Cell Shading - Microsoft Community

gh541 say:

Find and Change on Table Cell Shading

Hi everyone,

I have a heap of tables in a very large Word document that currently have black shading in the cells. I wondered if there was a way to target these cells using find/change and have the shading replaced with a lighter shade of grey? I couldn't find where to do this in the find/change dialogue but thought maybe someone has any suggestions on how to do this?

Appreciate any advice.
daniel g. grau say:
OK, I discovered the problem.

Cells can have a fill, which is what we wanted to use, but can also have a superimposed pattern in another color. Usually, these are too intrusive, but may tolerable if used sparingly. For instance, you can have a red cell fill (background) with blue dots (foreground). These dots should be few, as in 25% at most. Try it on a blank cell: Format > Borders and shading; select a color for a fill and another for the pattern, and select a low density value (called Style).

Now, the higher the Style value, the more intrusive the pattern becomes. When it reaches 100% (solid), the pattern dots connect and cover the whole cell fill. And this effect cannot be distinguished from a regular cell fill. What we were doing before was look for a fill (background) and since most cells had none, all cells with a None fill would change into the new fill. That explains your results in testing: everything but the black cells changed!

The following macro works by matching these three values (taken from the cell where the cursor is originally placed):

  • Texture (i.e., dot density, from 0% = clear to 100% = solid) 
  • Foreground (color of the dots)
  • Background (color of the cell fill, which in the sample tables seems to be none, but I'm considering it anyway)

So you start with your cursor in a cell and run the macro. There will be a dialog to select the replacement color. Then the macro will look for similar cells matching the above three parameters (taken from the original cell). If all three match, the macro:

  • Clears the texture (sets the pattern to 0%)
  • Clears the foreground color (the dot color; not really needed, as now there is no texture)
  • Replaces whatever fill color the cell has (even if it's none) by the color previously selected in the dialog

I tested it on the sample tables you sent and had to run it twice—the second time because some cells had a slighty different foreground color.

----------------------------------------------------------

Sub ChangeCellPatternToCellFill()

If Not Selection.Information(wdWithInTable) Then
   MsgBox "Place the cursor in a table cell with a colored pattern."
   Exit Sub
End If

Dim myRange As Range
CurrentTexture = Selection.Range.Cells(1).Shading.Texture
CurrentForegroundPatternColor = Selection.Range.Cells(1).Shading.ForegroundPatternColor
CurrentBackgroundPatternColor = Selection.Range.Cells(1).Shading.BackgroundPatternColor

Set myDialog = Dialogs(wdDialogFormatBordersAndShading)
myDialog.DefaultTab = wdDialogFormatBordersAndShadingTabShading
myDialog.Display
NewFillColor = myDialog.BackgroundRGB

For Each myTable In ActiveDocument.Tables
    Set myRange = myTable.Range
    For Each myCell In myRange.Cells
            If myCell.Shading.Texture = CurrentTexture _
                And myCell.Shading.ForegroundPatternColor = CurrentForegroundPatternColor _
                And myCell.Shading.BackgroundPatternColor = CurrentBackgroundPatternColor Then
                    myCell.Shading.Texture = wdTextureNone
                    myCell.Shading.ForegroundPatternColor = wdColorAutomatic
                    myCell.Shading.BackgroundPatternColor = NewFillColor
            End If
        Next myCell
Next myTable

End Sub
