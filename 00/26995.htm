hansv mvp say:
  1. Start with a document without page numbers.
  2. At the end of page 3, insert a next-page section break (from the Breaks button in the Page Setup group of the Page Layout tab of the ribbon.
  3. Do the same at the end of page 6 and at the end of page 21.
  4. Move back to page 4 and activate the header or footer where you want the page number (Insert tab of the ribbon, Header > Edit Header or Footer > Edit Footer). You should see 'Header - Section 2' or 'Footer - Section 2'.
  5. This will activate the Design tab of the ribbon.
  6. By definition, the 'Link to Previous' button in the Navigation group will be "on" (highlighted). Click it to turn it "off".
  7. If you want the page number to be centered, press Tab once. If you want it to be right-aligned, press Tab twice.
  8. In the Header & Footer group, click Page Number > Current Position > Plain Number.
  9. Click Page Number > Format Page Numbers...
  10. Select I, II, III... from the 'Number format' dropdo
