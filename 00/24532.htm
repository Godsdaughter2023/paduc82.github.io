You use the Switchboard Manager to create multiple pages. If you look in the Switchboard Items table, it will list the Switchboard ID for each page.

If you look at the On Open Event of the Switchboard form it shows you the code. The first piece is to set 

Set TempVar

macropod say:

Word's implementation of the Chicago style isn't even close to reflecting the requirements of either of the two Chicago referencing formats:  'Author-Date'; and 'Notes and Bibliography', both of which you can see at: http://www.chicagomanualofstyle.org/tools_citationguide.html.

This situation has existed ever since Office 2007 was released and nothing has been done to address the issue. IMHO, you'd do far better using a 3rd-party Addin such as EndNote.
