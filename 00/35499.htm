I experienced the same problem with an ACER laptop provided by TN government. I deleted the partition which was displayed free in Windows but originally had Boss Linux installed. I got File System Error message and went in to GRUB recovery mode.

You just have to repair the MBR, which can be done as follows.

1. Make a bootable USB flash drive with Windows 7 System Recovery. (You need the help of another working PC with a valid Windows 7 installation for this)
   http://en.kioskea.net/faq/29462-create-a-system-repair-usb-under-windows-7

2. Restart the laptop and enter setup (F2 in my case).

3. In Boot Options, make the USB HDD - Flash drive to boot first by bringing to the top (I used F6 to move up) and make sure to Save and Exit.

4. Plug in the bootable USB flash drive and restart the laptop. Windows System Recovery should load automatically and wait for keyboard selection (US). Press Next and when a valid Windows 7 installation is found, it will be displayed along with the partition size and location. Select it and click Next.

5. In 'Choose a recovery tool' window, select Command Prompt.

6. Type the following in the command prompt and press Enter.

bootrec /fixmbr

and then

bootrec /fixboot

If successful, "The operation completed successfully" message will be displayed for each.

7. Restart the laptop.

Now Windows 7 should boot normally.
