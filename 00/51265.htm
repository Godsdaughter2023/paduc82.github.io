a. user say:

Hi,

You might be experiencing this issue because of corrupted system files or outdated device drivers. Since this issue happened after an update, it's possible that the current drivers on your computer are not compatible with Windows 10.

Let's get you laptops sorted. For the Asus laptop, we recommend that you start your laptop in safe mode. This mode starts Windows in a basic state, using a limited set of files and drivers. It can help you troubleshoot problems on your computer.

Once you are in safe mode, we suggest that you use the System File Checker tool. This tool scans for corruptions in Windows system files and restores corrupted files.

We also recommend that you check if you have the latest device drivers on your computer. While it's likely you already have the most recent driver, if you're having trouble with a device, you can try fixing it by updating the driver or reinstalling it.

For your Toshiba laptop, we suggest that you repair your laptop using an installation USB or disc. If you do not have a Windows 10 installation USB or disc, you can download and create one from the Microsoft software download website. To create an installation media and use it to repair your computer, follow the steps below:

  1. On a working PC, go to the Microsoft software download website.
  2. Download the media creation tool, and then run it.
  3. Select Create installation media for another PC.
  4. Choose a language, edition, and architecture (64-bit or 32-bit).
  5. Follow the steps to create installation media, and then select Finish.
  6. Connect the installation media you created to your nonfunctional laptop, and then turn it on.
  7. On the initial setup screen, enter your language and other preferences, and then select Next. If you do not see the setup screen, your laptop might not be set up to boot from a drive. Check your laptop manufacturer's website for info on how to change your laptop's boot order, and then try again.
  8. Select Repair your computer. You will then be able to perform Startup Repair to repair your laptop.
  9. On the Choose an option screen, select Troubleshoot > Advanced options > Startup Repair.

Regarding your Windows 10 license, have you upgraded to Windows 7 or Windows 8.1 from Vista before upgrading to Windows 10? If you have upgraded from Windows 7 or Windows 8.1, then you should have a Windows 10 Digital License. With Digital License, there is no need to know your product key or purchase another product key if you lose it. Anytime you need to reinstall Windows 10 on the same computer, just proceed to do so using any of the available options. Windows 10 will automatically reactivate.

If you need further assistance, please do not hesitate to post a reply.

Hi LAFredrikNilsson,
First of all try to restart the PC and try again. Otherwise, follow this procedure: Go to network connections, access the properties of the network adapter connected to the internet> Sharing tab, enable Internet Connection Sharing (ICS) and confirm by pressing OK. Skip this step if ICS is already enabled.) Return to the Sharing tab, disable ICS and confirm by pressing OK. Enable the Mobile Hotspot from the Settings panel and try again.
Note: when using the Mobile Hotspot, the checkboxes on the Sharing tab must remain unchecked, otherwise the Mobile Hotspot will not work.
If it doesn't work, try resetting the network: Settings> Network and Internet> Status. Click on Network Restore on the right side and follow the instructions.
