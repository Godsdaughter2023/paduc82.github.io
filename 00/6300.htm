In the list of figures i would like to include only a part of the - Microsoft Community

nederlands_808 say:

In the list of figures i would like to include only a part of the figure's caption. How can I achieve this?

The caption of the various figures (and also tables) I used in my thesis is too long to appear as a whole in the list of figures/tables. So I would like for each caption to choose only a few words which then appear in the list of figures/tables. Thanks for your help...
john mcghie say:
Arshad: What utter nonsense...

Nederlands:  You do not need a "workaround", what you need is a standard function of the List of Figures, and I would have expected someone employed to give help in here to know that...

You need to know that the "List of Figures" is actually built by the Table of Contents generator, which has various switches and controls to modify the way it goes about its business.

The TOC generator is controlled by a TOC field.  You can specify that it looks for "TC" (Table of Contents) tags instead of entire captions.  If it does, it will include in the list of Figures only the text you tag in the captions, regardless of how long the captions are.

First, you need to pore through the detailed explanations here:

http://office.microsoft.com/en-us/word-help/field-codes-toc-table-of-contents-field-HP005186201.aspx

http://office.microsoft.com/en-us/word-help/field-codes-tc-table-of-contents-entry-field-HP005186197.aspx?CTT=1

Now:  to see what you are doing, you need to display the codes that generate the List of Figures, instead of their results.  Click in the field and hit Shift + F9 (or right-click and "Toggle Field Codes".

Instead of the standard List of Figures code, { TOC \c "Figure" } you need to change the code to { TOC \f f }  Just edit the code directly.  The curly braces are not really curly braces, they are Field Bounding Characters.  If you need them you can add them by hitting Command + F9.

That tells the TOC generator to look through the document only for entries marked with Field Codes (the \f switch) and to include them in the TOC only if they have a type letter of "f".

Now you can go tag your captions.  For each caption, we need to insert a tag with the code { TC "The text I want in the List of Figures" \f f }.  The contents of the TC tag are set to invisible text, so they will neither display nor print.  You need to turn on non-printing text so you can see what you are doing.  The quick way to do this is to:
  1. In each caption, select the text you want in the LOF
  2. Copy it
  3. Paste it inside the caption
  4. Add quotes around the text -- "around the text"
  5. Select it again
  6. Hit Command + F9 to insert the field bounding braces
  7. Inside the left brace type the characters "TC" without the quotes.
  8. Inside the right brace, type "\f f" without the quotes.
It's important to copy the text, because once it's inside the TC field it won't print.

It is critical to use Command + F9 to create the field, it can't be done any other way.  It is important to have the text selected when you do, otherwise you will have to drag the text inside the braces later.

It is critical to add the "\f f" switch at the end of the text: if you do not, the content of the tag will appear in the Table of Contents instead of in the List of Figures.

By now I am sure you are muttering to yourself that this is so much work that next time you will use shorter captions.  Yep!  This mechanism is hardly ever used because it is a lot of work.  But it does exactly what you need: it was developed for just this purpose.

Pity Arshad didn't look in the Help: it's described there... that's what the URLs I sent you to are from.

Hope this helps

scott mcdaniel say:

You can select an item in a Listbox like this:

Me.YourListbox.Selected(0) = True

Where "0" is the index of the item you want to select. If you want to select everything to the "bottom" of the list from the user's current selection, you could do this:

Dim i As Integer
For i = Me.List0.ListIndex To Me.List0.ListCount - 1
  Me.List0.Selected(i) = True
Next i

To go up:

Dim i As Integer
For i = Me.List0.ListIndex To 0 Step -1
    Me.List0.Selected(i) = True
Next i

debmel say:

Creating lines - can't move cursor - Word for Mac 2011

I have tried creating horizontal lines using 'borders' in Word for Mac 2011.  Once I create the line, I cannot move the curser below the line to add additional text. I use the lines to separate sections on a resume template.  Can anyone help?  Thanks.
daniel g. grau say:

Horizontal lines can be added via Format > Borders and Shading:

  1. Above and/or below a single paragraph
  2. Above the first and/or below the last paragraph in a selected group of paragraphs (a general instance of case 1)
  3. As in 2, but also adding lines between intervening paragraphs

Depending on where you place the cursor (before, within or after in cases 1-3), you'll get different effects. Your best bet would perhaps be to add several blank paragraphs before selecting them and choosing which of the three instances you need. Then move up/down in the text with the cursor or mouse.

bob jones aka: cybertaz say:

Point your mouse directly beneath the Paragraph Border @ the left margin, then double-click (note that this is assuming you have the Enable click and type preference checked in Word> Preferences> Edit).

What your running into is that you're applying a bottom border to what is currently the last paragraph in the document, so there is no place beyond it in which to click. The double-click overcomes that by creating a new paragraph.
