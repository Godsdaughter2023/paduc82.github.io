maxwell joseph say:

Hi Ashutosh,

If you receive the error message “there are no products affected by this package installed on this system” when attempting to install Service Pack 1, either

Your have an Office 365 subscription that automatically prompts you to update. This is called Click-to-Run and you are trying to update your Click-to-Run version by using a Windows Installer patch (.msp).

Or

You have an MSI-based (Windows Installer) edition that has already been updated.

For MSI-based versions, you can check to see if Service Pack 1 is already installed by looking for the program version in the list of installed programs.

  • Open the Control Panel
  • Click Programs and Features
  • Click View Installed Updates in the left-hand menu
  • Under Office, you should see Service pack 1 for Office 2013

For Click-to-Run-based versions, you can check to see if Service Pack 1 is already installed by looking at the program information from within the Office applications.

  • From within an Office application, click File and Account
  • Under Office Updates note the Version
  • The version will be 15.0.4569.1507 or higher
