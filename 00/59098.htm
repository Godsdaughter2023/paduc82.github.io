Windows 10 reinstall - Microsoft Community

geoffnash1 say:

Windows 10 reinstall

   

do I have to link windows 10 with a microsoft account to reinstall windows 10?

davem121 say:
Hi geoffnash1, when you activate Windows 10, it will automatically create a digital entitlement on the Microsoft Activation servers, which is generated from your PC's hardware.

If you ever need to re-install Windows 10, as long as you re-install the same Edition of Windows 10 (Home, Pro . . .etc.) it will automatically activate once it is put on the Internet - even without being attached to your Microsoft Account

When you log into your PC with a Microsoft Account, that device will automatically be added to your Microsoft Account as will that digital Windows 10 Licence - this may happen immediately, it might take a few days - you do not need to do anything it is automatic



If you ever need to re-install Windows:

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

FIX for the shaky screen on Surface Pro 4! (For out of - Microsoft Community

louisfeldman say:

FIX for the shaky screen on Surface Pro 4! (For out of replacement SP4 devices)

As a WARNING, this fix has only been tested on my Surface Pro 4 on Windows 10 Pro Ver.1809. I have no other damage to this device, this is solely for the shaky screen issue due to idle screen. The fix I describe below involves adding to the Registry Editor, I DO NOT TAKE RESPONSIBILITY for any damage to your device if you choose to follow these instructions. I highly recommend going through with the recall program  if your device is within the 3 year recall limit; this post is primarily for SP4 uses that are BEYOND the 3 years and have the shaky idle screen.  

This problem is an idle screen problem, if your computer is not updating any section of the screen after about 5 seconds it will start to shake. Many of the other big discussions mention having task manager open or other programs that constantly update and sit them on the side of the screen while they work. My fix works the same way by updating the screen every second, but it does not take up any additional screen space. Have you guessed what we are adding to the screen to fix this issue that has bamboozled everyone for the last three years? 

We are expanding the taskbar time to include seconds. That's it.

Since I have added seconds to the taskbar time about 3 months ago I have not had a shaky screen. The screen is updating every second to show the new time, not letting the screen go into its idle state. The issue is turning this option on, it is not a setting you can just toggle in your personalization or taskbar settings. You have to manually add a DWORD Value and turn it on in the Registry Editor. If you are not comfortable or do not know how to use the Registry Editor, DO NOT try this fix yourself. I will make it as simple as possible, but you should work with someone that has experience with the Registry Editor.

The first step is opening the Registry Editor, on Windows 10 use the search bar to search for the "regedit" 

application and start it. This will bring up a window with the Registry Editor shown below.

Copy and paste this address into the address bar below the file, edit, etc. options and press enter. This will take you exactly to the file where the information is going.

Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

Once at this location, right click in the empty space to the right, and select New>DWORD (32-bit) Value.

Like creating a new file, the new DWORD Value will ask for a name. It is critical that you spell this name exactly as shown below. I recommend copying and pasting it into the field. (Renaming is a right click option if it sets to the default name) 

ShowSecondsInSystemClock

At this point you will have the new DWORD Value in the file, but it will be off by default. To turn it on, right click ShowSecondsInSystemClock and choose modify. 

Set the value to 1 (on), make sure the base is checked on Hexadecimal, and press okay.

At this point, you are done making changes to your system, but changes will not take place until your next login. You can just exit out of the Registry Editor and sign out, then sign back in. You should see the seconds ticking on in your taskbar clock. Wait about five to ten seconds to verify that the screen shake is gone or still persists. If it is gone, congratulations! If not, there may be more hardware prolems that should be looked at by Microsoft.   

If you do not see a change in the taskbar time, open the Registry editor again (it will open to the last used folder) and verify that you are in the right folder, spelled the name of the DWORD Value correctly, and that you remembered to turned the binary to 1. If you are still having problems, reference this article's troubleshooting section. The article also has a downloadable file that makes this addition automatically, but I recommend doing it yourself.
