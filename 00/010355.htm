Sub Send_Range_Or_Whole_Worksheet_with_MailEnvelope()
Dim AWorksheet As Worksheet
Dim Sendrng As Range
Dim rng As Range
On Error GoTo StopMacro
With Application
.ScreenUpdating = False
.EnableEvents = False
End With
Set Sendrng = Worksheets("Sheet1").Range("C3:P11")
Set AWorksheet = ActiveSheet
With Sendrng
.Parent.Select
Set rng = ActiveCell
.Select
ActiveWorkbook.EnvelopeVisible = True
With .Parent.MailEnvelope
.Introduction = "Hello, here it is."
With .Item
.To = "*** Email address is removed for privacy ***"
.Subject = "Hello"
.Open
End With
End With
rng.Select
End With
AWorksheet.Select
StopMacro:
With Application
.ScreenUpdating = True
.EnableEvents = True
End With
End Sub

OK with that saved, activated VM, you can try using different virtual hard disks. It is the VM that is licensed, not the VM's hard disk. This means that you could swap out the hard disk (leaving the current version 1607 intact) reinstall Windows 10, either public build, or Insider build, to a new virtual hard disk and it should auto-activate. If at any time you want to go back to the existing Windows 10 install, just swap the Virtual hard disk file with the original one that it uses now. With Hyper-V I can export my VM's to a new, identical file to ensure that I can keep a copy of my licensed VM's somewhere.

When you clean-install Windows to the new virtual hard disk (attached to the licensed VM) start with an unpartitioned virtual disk and let Windows configure it.
