How do I change the default location of user files in windows 7?

 

Thanks

abdul s say: Hi,
 
 
Changing the default system folders to different location is not recommended. If you would change the location, it would be on your own risk at Microsoft we do not recommend.
 
If you want to change the location for all the files including the Application data and personal files then you can follow the below steps by logging into each account separately:

 
a. Open Windows explorer and create a new folder is the desired drive (Ex: D: Drive).
b. Re-name the folder as desired name (Ex: Documents).
c. Click the start menu, in the search box type in ‘shell:UsersFilesFolder’ (Without quotes).
d. Right click on the folder that you want to move to a different location.
e. Select Properties.
f.   Click on Location tab, select move.
g. Navigate to the location you want to move (the new folder you had created in step 1).
h. Click on select folder, click OK.
i.   If you are prompted for permission of Move folder, click OK.
 

If you check back at the C:\Users\ (user-name) location, you will notice that the user folder (ex: Documents) that you moved is no longer there, but is still at the Start Menu (shell: UsersFilesFolder) location. You will also notice that the folder at the location (ex: D drive) you moved the user folder to, now has the default user folder (ex: Document) icon.
 
If you have already tried these steps, alternatively, you can select only the folders you like to move by following the below steps:
 
a. Right click on the documents folder Go to Properties.
b. Under the library location tab, add a location where you like to save the files.

Windows 7 Update Error 8024000b, 0x8024000B, WindowsUpdate_8024000B

Cause

■ If you receive Windows Update error 8024000b, it means that Windows Update can't read a file (Update Manifest) that is required for installation.


Solution

1.Click the Start button, type Run in the search box, and then, in the results list, click Run.

2.Copy the following text, and then paste it into the Run text box:

Cmd /c ren %systemroot%\System32\Spupdsvc.exe Spupdsvc.old

3.Click OK.

4.Run Windows Update again.

How to access Windows Mail from another PC on in-home wireless - Microsoft Community

r l santee say:

How to access Windows Mail from another PC on in-home wireless network.

I have Windows Mail already set-up on Dell XPS using Vista. We have added a second Netbook using Windows 7 on in-home wireless network. How can I access the Widows Mail account from the second pc? Thank you,
gary vandermolen say:
The solution mentioned by the other responders will only allow you to see
new incoming messages on multiple computers.  It won't allow you to see the
messages sent from one computer to be seen on multiple computers.  In order
to do the latter, you would need to switch to an IMAP or HTTP account rather
than POP3.
 
Note that both Windows Mail and Windows Live Mail require that their message
store reside on a local hard drive, not on a network-accessible non-local
drive.
