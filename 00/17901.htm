ken sheridan say: I'd agree with Karl that a report which groups by Division and Team and breaks the page after each team group would be the best way to go.  You should be able to base this on a fairly simple query.  The report can easily be filtered by one or more teams, or can show all.  If you are storing 'historical' data from previous years where a team may be in a different division you'd need to include the year in the data and group the report by this too, along with filtering it by year if necessary.

Let's assume for the moment that either only one year's fixtures are stored, or if multiple years are stored the query already restricts the report to the current year, so the report only needs to be filtered by team.  To select one or more teams I'd suggest a multi-select list box on a dialogue form which lists all teams (possibly with their current division in a second column), so any number from one to all can be selected.  Also on the form would be a button to open the report, or maybe one to preview it and another to print it.  The code would be near identical for each button, and in the case of the preview button would be along these lines:

    Dim varItem As Variant
    Dim strTeamList As String
    Dim strCriteria As String
    Dim ctrl As Control
   
    Set ctrl = Me.lstTeams
   
    If ctrl.ItemsSelected.Count > 0 Then
        For Each varItem In ctrl.ItemsSelected
            strTeamList = strTeamList & ",""" & ctrl.ItemData(varItem) & """"
        Next varItem
       
        ' remove leading comma
        strTeamList = Mid(strTeamList, 2)
       
        strCriteria = "Team In(" & strTeamList & ")"
       
        DoCmd.OpenReport "rptFixtures", _
            View:=acViewPreview, _
            WhereCondition:=strCriteria
    Else
        MsgBox "No teams selected", vbInformation, "Warning"
    End If

If you also need to filter by year, it would be a simple task to add a combo box to the form for this and extend the code slightly.

You can add a 'Select All' button to the form if you wish with the following code in its Click event procedure:

    Dim n As Integer
   
    For n = 0 To Me.lstTeams.ListCount - 1
        Me.lstTeams.Selected(n) = True
    Next n

and a 'Clear Selections' button with:

    Dim n As Integer
   
    For n = 0 To Me.lstTeams.ListCount - 1
        Me.lstTeams.Selected(n) = False
    Next n
