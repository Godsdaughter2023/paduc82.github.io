I have the same problem. Based on another thread, Microsoft has reproduced the problem and is working on a fix. https://answers.microsoft.com/en-us/msoffice/forum/msoffice_excel-mso_win10-mso_365hp/lost-formatting-in-excel-2016/87ca9032-aba6-4a03-86bb-a46a875f12be

1) The font change applies to text after subscripts and superscripts and changes in font size, and basically any change in font properties within the same cell. For instance, ex: F_d = m*a; the font would change starting right before "d".

2) Excel changes the font to whatever properties you set in your "Normal" style font. (In styles, right-click the style and click "Modify" to see the properties.)

3) If you open the file through Excel (rather than directly through the desktop or folder), the font issue does not occur. If you have Excel open already (i.e. a new workbook), the font issue does not occur in other files when you open them directly through the desktop or folder. So definitely a software startup issue. 

This has worked for me and several other people.

  • Test that the ActiveCell is the correct column.
  • Test that the ActiveCell contains validation and contains List type validation and that the List Formula is "=liner"

Explanation of remaining code.

  • The rows are added and the cells in columns A:C are merged and border formatted.
  • The Manufacturer is copied to the merged cells. Optional so remove the line of code if not required.
  • The data validation is removed from column "E" of inserted rows. Optional but I should think that you do not require it in the inserted rows. Remove if not required in 2 places.
  • The code adds a blank row between the added rows and the next Manufacturer. If not required then get back to me and I will edit the code.

The code is below and just in case you need it, the test workbook is uploaded as a zipped file to the following link. Download the zipped file and extract. Note that the file is missing some key components so I suggest that you simply copy the code to your working project and not "Save as" to the working project name.

Private Sub CommandButton1_Click()

    Dim i As Integer
    Dim n As Integer
    Dim m As Long
    Dim X As Variant
    Dim strMsge As String
    Dim currentCell As Range
    Dim rngToFormat As Range
    Dim strManufacturer As String
   
    Set currentCell = ActiveCell
   
    '************************************************************************************
    'Code between asterisk lines tests that Active Cell is correct column and
    'contains correct validation so that processing does not take place with an incorrect cell selected
    strMsge = "Select cell adjacent to " & Chr(34) & "Enter # of rows..." & Chr(34) _
                & vbCrLf & "Processing terminated."     'Create message and assign to variable
               
    If currentCell.Column <> 5 Then
        MsgBox strMsge
        Exit Sub
    End If
               
    On Error Resume Next
    X = currentCell.Validation.Type
    On Error GoTo 0
    If X = xlValidateList Then  'Tests if List type validation
        If currentCell.Validation.Formula1 <> "=liner" Then 'Test for correct list
            MsgBox strMsge
            Exit Sub
        End If
    Else
        MsgBox strMsge
        Exit Sub
    End If
    '*************************************************************************************
   
    Do While Not IsEmpty(currentCell)
        strManufacturer = currentCell.Offset(-1, -4).Value
      
        n = currentCell.Value + 1
        m = currentCell.Row
   
        If n > 0 Then
            Rows(m + 1 & ":" & m + n).Insert
           
            Set rngToFormat = Range(Cells(m + 1, "A"), Cells(m + n - 1, "C"))
            For i = m + 1 To m + n - 1
           
                Cells(i, "E").Validation.Delete    '<---*** Optional to delete validation in the added rows.
               
                Cells(i, "A").Value = strManufacturer   '<---*** Optional to populate manufacturer cell
               
                With Range(Cells(i, "A"), Cells(i, "C"))
                    .Merge
                    .HorizontalAlignment = xlCenter
                    .VerticalAlignment = xlBottom
                    .WrapText = False
                    .Orientation = 0
                    .AddIndent = False
                    .IndentLevel = 0
                    .ShrinkToFit = False
                    .ReadingOrder = xlContext
                   
                    .Borders(xlDiagonalDown).LineStyle = xlNone
                    .Borders(xlDiagonalUp).LineStyle = xlNone
                   
                    With .Borders(xlEdgeLeft)
                        .LineStyle = xlContinuous
                        .ColorIndex = 0
                        .TintAndShade = 0
                        .Weight = xlThin
                    End With
                     
                    With .Borders(xlEdgeTop)
                        .LineStyle = xlContinuous
                        .ColorIndex = 0
                        .TintAndShade = 0
                        .Weight = xlThin
                    End With
                   
                    With .Borders(xlEdgeBottom)
                        .LineStyle = xlContinuous
                        .ColorIndex = 0
                        .TintAndShade = 0
                        .Weight = xlThin
                    End With
                    With .Borders(xlEdgeRight)
                        .LineStyle = xlContinuous
                        .ColorIndex = 0
                        .TintAndShade = 0
                        .Weight = xlThin
                    End With
                   
                    .Borders(xlInsideVertical).LineStyle = xlNone
                    .Borders(xlInsideHorizontal).LineStyle = xlNone
                End With
            Next i
           
            'Note FYI: After the end of loop i is one greater than "To" value of loop
            Cells(i, "E").Validation.Delete    '<---*** Optional.Delete validation last added row.
           
            Set currentCell = currentCell.Offset(n + 2, 0)
        End If
    Loop
   
    Range("A1").Select  'Optional
   
End Sub
