ossiemac say:

Only way I know of doing this is to use VBA (Visual Basic for Applications) to AutoFit each time the worksheet calculates as per the code below.

To install the code:

  1. Right click the worksheet tab name and select View code to open the VBA editor.
  2. Copy the code below and paste into the editor.
  3. Close the VBA editor. (Cross with red background top right)
  4. Save the workbook as Macro Enabled workbook.
  5. Ensure that macros are enabled. See Help for how to do this. The option to "Diaable all with notification should be OK.)

Private Sub Worksheet_Calculate()

    'With Me.UsedRange      'Can use this in lieu of next line
    With Me.ListObjects("Table1").Range   'Edit "Table1" to your table name
        .WrapText = True
        .Rows.AutoFit
    End With
End Sub

Is it Possible to Create a Rule in Outlook 2013 that will Move E-mails - Microsoft Community

jason scoggins say:

Is it Possible to Create a Rule in Outlook 2013 that will Move E-mails with Chinese Characters to a Specific Folder?

I have a client who receives e-mail from English and Chinese speaking customers. My client would like to sort all e-mail received with Chinese characters into a specific sub folder.

Note - I am not trying to block the messages. Instead, the goal is to filter them and store them in a specific folder.

So far I have come up with two possible solutions, neither one I like 100%:

Solution 1 - Create an e-mail alias for the client, and filter all messages sent to that alias

One option is to have the user send out a new e-mail address for all Chinese-speaking contacts. And then filter any messages sent to that specific address.

Solution 2 - Create a rule in outlook that will filter messages based on 'specific words' in the Subject or Body, and add commonly used Chinese characters

This is using a standard e-mail rule, and simply adding new Chinese characters to filter the majority of the e-mail as it comes in.

Any other thoughts?

This is on Outlook 2013 with an on-prem Exchange server (Exchange 2013)

jason scoggins say:

We were able to get this working.

However, we could not use the "with specific words in the message header" option, as there wasn't a common string of text that we could use to filter the messages with Chinese characters.

Instead, we went ahead and created a rule to look for messages "with specific words in the message body" and simply pasted a few common Chinese characters. The client typed up about six characters and I added each one to the rule, one at a time, which means they are processed as OR and not AND. This makes a big difference.

We ran this rule on his inbox containing about 20,000 messages, with nearly 4,000 that had Chinese characters, and the process took less than a minute. So my early concerns of having Outlook search each message for several different characters might slow down the performance seems to be a non-issue.
