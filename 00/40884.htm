xbox360italy say:

Hello

If you have a laptop computer with both wired and wireless network adapters, you can use your laptop to connect your Xbox 360 console to Xbox LIVE. This type of connection is called a network bridge connection. This connection is quick and easy, but you cannot access the Internet on the laptop when the laptop is connected to your Xbox 360 console.

To use a network bridge connection, you must have the following:

  • A computer with two network adapters
  • Windows XP, Windows Vista, Windows 7, or Windows 8 
    Note The Windows 7 Starter edition does not support this feature.
  • High-speed Internet access
  • A standard Ethernet cable

Set up a network bridge connection

Note The following steps show Windows 7 and Windows 8 images. However, the steps are the same for earlier versions of Microsoft Windows, unless otherwise indicated.

  1. Click the Start button.
  2. In the Search programs and files box, type ncpa.cpl, and then press ENTER. 
    Windows XP: Click the Start button, click Run, type ncpa.cpl in the Open box, and then click OK.
Windows 7 Windows 8
  1. Select the two network connections that you want to bridge. To do this, click the first network connection, press and hold CTRL, and then click the second network connection.
  1. Right-click either of the selected network connections, and then click Bridge Connections.
  1. Wait for the new connection icon to be displayed. 
    Note The new icon will be labeled Network Bridge. When the icon appears, the new connection is ready to use.
    1. Plug one end of the Ethernet cable into the network port on your portable computer. Plug the other end of the network cable into the back of your Xbox 360 console.

    1. On your Xbox 360 console, make sure that all network settings are set to Automatic.
      Here’s how:

Note
       If you are connecting to a wireless network, or if you have PPPoE settings entered, note the settings because you might have to enter them later.


      1. Press the Guide button on your controller, go to Settings, and then select System Settings.
      2. Select Network Settings.
      3. Select Wired Network if you are prompted to do so.
      4. Select the Additional Settings tab.
      5. Select Restore to Factory defaults.

        This resets the network settings to their default setting. This does not affect any other console settings.

    1. Test your Xbox LIVE connection. Here's how:

    1. Press the Guide button on your controller, go to Settings, and then select System Settings.
    2. Select Network Settings.
    3. Select Wired Network if you are prompted to do so.
    4. Select Test Xbox LIVE Connection.
