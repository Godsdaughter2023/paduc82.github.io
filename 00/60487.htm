Windows 10 Update 0x80070011 error Temp Fix - Microsoft Community

zack sims say:

Windows 10 Update 0x80070011 error Temp Fix

This is how I fixed the 0x80070011 for Window's Update and it will need to be done every time a new cumulative update gets released.

Prerequisites: 

Regedit or Registry Workshop
Windows 10 Recovery Media USB, DVD.
A Local user profile for testing and not your everyday one. 
Always backup your Registry and Windows Partition before making any changes just in case something goes wrong.

Step 1: In Regedit or Registry Workshop change any key in these locations 

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion
HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion

that has C:\Program Files (x86) and C:\Program Files (x86)\Common Files to X:\Program Files (x86) and X:\Program Files (x86)\Common Files.

Note: Replace X with your drive letter to where Program Files (x86) is located on your PC.

Step 2: (Only needed if you don't have a secondary user profile for testing or whatever you use it for): Go to Start > Settings > Accounts > Family and Other Users and click Add Someone Else To This PC and when the windows loads click I Don't Have This person's Sign In Information then click Add A User Without A Microsoft Account. This will make sure no shortcuts or pinned applications will break when the junction is reimplemented later on when used on your regular everyday account. 

Step 3: Log into this secondary user account and delete the junction for Program Files (x86) in C:\ and after deleting, create a new folder and call it Program Files (x86).

Step 4: After creating the new folder go to the registry by your preferred means and change all the keys back from Step 1 to C:\Program Files (x86) then reboot and log back into the secondary profile.

Step 5: Go to Windows update and check and install the latest update then reboot and when done log back into the secondary profile.

Step 6:  Once logged in go to C:\Program Files (x86) and copy all the contents to X:\Program Files (x86) then prepare your recovery media and reboot.

Step 7: Boot into your Windows 10 recovery media and go to command prompt then type notepad then go to File > Open then find C:\Program Files (x86) and delete it.

Step 8: After deleting C:\Program Files (x86) in the notepad explorer, close notepad then go back to the command prompt window and type mklink /J "C:\Program Files (x86)" "X:\Program Files (x86)" After it has successfully created the junction you can reboot.

I hope someone finds this helpful.

The BCC (Blind Carbon Copy) field enables you to sent mail to multiple targets without the list of recipients being displayed. This is particularly useful if you have an email that has to be sent to a number of your friends or colleagues. At least you can sent the mail knowing that whoever receives it will not know who else the mail has been sent to.

When you perform a repair installation, this restores the current Windows  installation to the version on the installation DVD. This requires you to reinstall all updates that are not included in that version of Windows.
