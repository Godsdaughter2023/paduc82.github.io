scottgem say:
Actually, that's not the best way to go about it. The RowSource you have created creates a text string for month and year. I'm assuming your date field is a Date/Time Datatype, so you can't use it for a filter. Also, Date is a reserved word in Access and shouldn't be used for object names.

If you want to filter a form for a specific month and year, then I would use a RowSource like so:

SELECT DISTINCT Format([qryWEEKLY TITHES].[TitheDATE],"mmmm yyyy") AS [MthYr], Month([TitheDate]) AS Mth, Year([TitheDate]) As Yr 
FROM [qryWEEKLY TITHES]
ORDER BY Yr, Mth; 

Then you can use the After Update event of the combo to run code like:

Dim dteStart As Date, dteEnd As Date

dteStart = DateSerial(Me.cboDate.Column(2),Me.cboDate.Column(1),1)
dteEnd = DateSerial(Me.cboDate.Column(2),Me.cboDate.Column(1)+1,0)

Me.Filter = "[TitheDate] > #" & dteStart & "# AND [TitheDate] < #" & dteEnd & "#"
Me.FilterOn = True

Note" cboDate is the name of the combo. You can set the Column Widths property to 1";0";0" so that the mth and Yr columns don't show.
