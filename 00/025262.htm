bernd p. say: I think you're missing some knowledge about Onenotes file/folder structure.

A Onenote notebook is not a file, it's a folder within the Windows file system. Its name is the name of the notebook, which appears at the top level in the bar at the left of the Onenote window.

Such notebook folder contains one or more Onenote sections, the *.one files. Which are shown as tabs in the Onenote window (under the ribbon).

The trouble starts with the ability of Onenote to open either sections or notebooks.
Using File > Open you can point to a section = *.one file and open it. But this is not a notebook and it will instantly "disappear" after closing Onenote.

To open permantly a Onenote notebook you have to use again File > Open, but point to the notebook's folder and open that folder with all its sections.
Such a correctly opened notebook will stay open after closing Onenote. So will see it open after the next start of Onenote. As it should be.

Bernd
