irfan h say:

Hi,

It looks like his secondary drive (z:) has the boot/startup files (listed as “System”).

The other drive (c:) has the Windows 7 system files on it (listed as “Boot”).

Since you have recently reinstalled Windows, and it appears the “System” files ended up on the other drive (z).

This is likely the cause of the conflict.

They can work around this issue though:

a.     Create a shared folder on the secondary internal hard drive (z).

b.     In the Backup program, at the “Select where you want to save your backup” section, click “Save on a network”

c.      Browse to the shared folder.

d.     Click “OK” to use the shared folder as the backup location.

e.     That should allow you to save the backup file without any issues.
