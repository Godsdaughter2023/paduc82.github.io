Count word occurrences in a Word Table - Microsoft Community

heathermaran say:

Count word occurrences in a Word Table

I have a table in Word that has three columns. I need to count the number of "Y's" in the third column (rows 3-33) and total them at the bottom. How do I do this? I know it is easy in Excel, and I know I can paste a worksheet into Word from Excel, but that looks sloppy. Can someone tell me how to do this?

Thanks!

doug robbins - mvp office apps & services (word) say:

To do that, use:

Dim i As Long
Dim count As Long
With Selection
    With .Tables(1)
        count = 0
        For i = 3 To 33
            If Left(.Cell(i, 3).Range.Text, 1) = "Y" Then
                count = count + 1
            End If
        Next i
    End With
    .Range.Text = count
    Select Case count
        Case Is < 10
            .Cells(1).Shading.BackgroundPatternColor = wdColorRed
        Case 10 To 14
            .Cells(1).Shading.BackgroundPatternColor = wdColorOrange
        Case 15 To 19
            .Cells(1).Shading.BackgroundPatternColor = wdColorYellow
        Case Else
            .Cells(1).Shading.BackgroundPatternColor = wdColorGreen
    End Select
End With
