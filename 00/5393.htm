zack barresse say: Khuram,

You should really start your own thread for these things.  Putting a link in your thread to this one, to show similarities, is perfectly acceptable.

This code should get you what you are asking for...


Sub MoveMyData()
   
    Dim WS As Worksheet
    Dim iRow As Long
    Dim iCol As Long
    Dim iStep As Long
    Dim iLastRow As Long
   
    Set WS = ActiveSheet
    iLastRow = WS.Cells(WS.Rows.Count, 1).End(xlUp).Row
   
    'Start at B2
    iRow = 2
    iCol = 2
   
    For iStep = 1 To iLastRow
        WS.Cells(iRow, iCol).Value = WS.Cells(iStep, 1).Value
        iCol = iCol + 1
        If iStep Mod 24 = 0 Then
            iRow = iRow + 1
            iCol = 2
        End If
    Next iStep
   
End Sub

Cells B1 and B2 contain formulas =A1 and =A2, respectively. A1 contains nothing, but B1 contains 0. A2 contains two lines (using Alt-Enter), but B2 has both lines concatinated as one line. How can I make col B exactly match col A?
rick rothstein, mvp - excel say:
But I don't want to copy cells, I want B1 to always reflect whatever is entered in A1, exactly as it was entered and is displayed in A1. I.e. if A1 has 2 lines, B1 should have two lines. If A1 is empty, B1 should be empty, not 0. If A1 is changed, I want that change immediately reflected in B1, without any other action such as copying.


You cannot do that with formulas, but I think the following VB event code does what you want. Right-click the name tab at the bottom of the worksheet and select View Code from the popup menu that appears, then copy/paste the following code into the code window that opened up...

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim Cell As Range
  On Error Resume Next
  For Each Cell In Target
    If Not Intersect(Cell.Dependents, Range("B1:B2")) Is Nothing Then
      If Cell.Value = "" Then
        Cell.Dependents.NumberFormat = ";;"
      Else
        Cell.Dependents.WrapText = Cell.WrapText
        Cell.Dependents.NumberFormat = Cell.NumberFormat
      End If
    End If
  Next
  On Error GoTo 0
End Sub

Okay, now go back to your worksheet and make a one or two line entry in either A1 or A2... B1 or B2 (depending on which cell you entered your value in) should now reflect the cell format (almost) as well as the line wrapping (we can add other mirroring properties as needed). The reason I said the cell almost reflects the cell formatting is because when you blank out either A1 or A2, its corresponding Column B cell wants to display 0, so I changed the cell formatting for this one instance only to ";;" which, in essence, displays the empty string.
