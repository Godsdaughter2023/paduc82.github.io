My only recommendation would be is to reinstall Windows Vista, restore the backup, then use Windows Easy Transfer from the Windows 7 disk to do the appropriate backup. Its going to be a number of hurdles, but if you believe your data is important, its best option.

install the Windows 7 Windows Easy Transfer Utility from the DVD (To find it, just navigate to x:\support\migwiz, 'x:' represents the name of your disk drive. http://www.notebooks.com/2009/10/24/how-to-backup-your-installation-of-windows-7/

andre da costa say:
Yes, Windows 7 64 bit will work on your computer.
Personally, I would recommend you stick with 32 bit Windows 7 for a number of reasons including the fact that you won't see any immediate advantage because you don't have 4 GBs or more RAM in your system.

As for compatibility with your Adobe software, depends on which versions of Adobe Photoshop and or Creative Suite you have installed, CS3, CS4 are known to work, I personally running those versions on 64 bit Windows 7.

64 vs 32
So I suggest you stick with Windows 7 32 bit. Windows 7 64 bit comes in handy when you need to address at least 4 GBs or more of RAM. Windows 7 32 bit can utilize up to 3.2 GBs of RAM and you are right within that range. Because the memory address space is much larger for 64 bit Windows, that means, you need twice as much memory than 32 bit Windows to accomplish some of the same task.

As for compatibility, you will need 64 bit device drivers for any hardware devices you might have. Also, there is no 16 bit subsystem in Windows 7 64 bit, which means, your applications must be 32 bit only, not 16 bit installer or uninstallers. Also, if you decide to move to Windows 7 64 bit in the future, there is no upgrade path from 32 bit Windows, clean install only.
