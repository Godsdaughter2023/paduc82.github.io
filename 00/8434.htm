amit tandon say:

Try below formula:

=MAX(0,NETWORKDAYS(AK3,MIN(TODAY(),AO3),0))

If cell AO3 is blank, the formula will consider TODAY();

If cell AO3 is less than TODAY(), formula will consider the lesser date;

If cell AO3 is more than / later than cell TODAY(), formula will consider TODAY();

If date in AO3 is less than date in AK3, formula will return 0;

michelle v [msft] say:

Hi Jim,

I believe you can tackle this by using "IF" and "AND" statements. I'm assuming with the above you would like to calculate dates network days lapsed (i.e., no weekends) from the start date to today, not the from start date to the complete date, is that correct? If so, here's how I would solve this:

For the columns, I'm using A, B, C to simplify this a bit.

In column B, I use this formula:

=IF(AND(A2<>"",C2<>"")=TRUE,NETWORKDAYS(A2,TODAY()-1,0),"")

Basically, this tells you that if the Start column (A) is not blank, and Complete column (C) is not blank, then calculate the time laps (network days from start date to today's date). The result above is 12 days counting from 5/9 to 5/24. Otherwise, leave the cells blank (meaning, don't calculate).

If you'd rather calculate the time lapse from Start Date to Complete date, then the Network formula calculation will be this:

=IF(AND(A2<>"",C2<>"")=TRUE,NETWORKDAYS(A2,C2-1,0),"")

Hope this helps!
