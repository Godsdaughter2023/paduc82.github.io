What happens to my previous operating system when I upgrade to Windows 10?

Windows 10 replaces your previous version of Windows by moving it to the Windows.old folder. Windows 10 becomes the default operating system.

This is an important thing the Windows 10 install routine does. Because if you need to go back to Windows 8.1 in case something goes wrong, incompatible hardware, you don't like Windows 10, you can roll back to it easily.

The Windows.old folder facilitates this.

So, I would suggest you not rush to delete it after completing Windows 10 setup, see how the OS functions. If its to your liking, then you can run Disk Cleanup and remove it from your system completely.
