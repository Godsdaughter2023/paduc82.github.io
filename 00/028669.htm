x-axial data distribution in line chart

I have a group of data e.g. x= 1,1.2,3,4,8;y=10,15,12,20,13. When I use line chart to plot x-y relationship, the x-axial does not give me correct seperation between those values. For example, the seperation between first two x points 1.2 and 1 is 0.2 and the seperation between 2nd and 3rd (1.2 and 3) is 1.8. So it is expected the 2nd point should be 1.2 units away from the origion and the 3rd point should be 3 units away from the origion. However, excel 2007 gives me equally distribute x axial. it means all 5 points on x-axial seperate equally.So please help me out!!
jon peltier (excel mvp) say:

A few things to check:

1. Is it really an XY chart, or did you inadvertently create a Line chart? They can be formatted the same, but the XY chart treats X data as numerical values, while a line chart treats X data as non-numerical labels.

2. Does the chart use the X data? Select the data series in the chart and look in the formula bar. You will see a formula like:

=SERIES(,Sheet2!$A$1:$A$5,Sheet2!$B$1:$B$5,1)

In this formula there are four slots for information (series name, X values, Y values, plot order), separated by commas. Is there data indicated in the second position, for X values? If not, Excel just uses the number of the point (1, 2, 3,...).

3. Is the data numerical? Sometimes if it's imported from another source it may be treated as text, despite it looking like numbers. In an XY chart, Excel treats text X values as if they are the same counting numbers as used in item 2 above.
