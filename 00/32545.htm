brent-- say: Hi Lewis54,

The TFTP client in Windows Vista is off by default.  If you haven't already turned it on, do the following:

  • Click on Start, then Control Panel.
  • Choose Programs, then Programs and Features.
  • On the left hand side, select Turn Windows features on or off.
  • When the list of features comes up, scroll down to TFTP Client and check the box next to it.

If you have already enabled it, you may want to check any firewalls or security software you have running.  TFTP uses UDP port 69 by default and needs to be allowed to pass.
