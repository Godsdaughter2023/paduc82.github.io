jesusbeerblog say:

Here's a *COUGH* short list of things that can cause this particular process to rob system resources.

* Having P2P enabled for Windows Update.  Solution: Disable

* Having something stuck in Windows Update.  Solution: Force it to update.

* Windows Spotlight enabled for Lock Screen.  Solution: Disable

If you've done all three of these and it still goes crazy, here's what fixed my problem.

Download Process Monitor from TechNet.

Filter results to only include Runtimebroker.exe

On the tool bar there is a file cabinet with magnifying glass to view file accessing.  You can see what files it is accessing and, importantly, what folder they are in.  I found that it was accessing a bunch of Spotlight files from when I first installed 10.  I moved those file into a temporary folder, restarted, and TADA, problem solved.

If you search the folder path in your search engine of choice, you can find what that folder is; if it's just something stupid and pointless, empty the folder into a temp folder to make sure you don't break something permanently and then restart.  Should fix your problem.
