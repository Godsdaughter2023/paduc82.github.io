jeeped say:
This looks like you are tryhing to find hte last Grand Total in any of the columns A through E and hide any data or formulas below that. Your use of ActiveSheet may not be appropriate here since (from your description of the problem) another sheets may be active when this is event macro is triggered. Fixing this could be as simple as as substituting Me for ActiveSheet., but it may be better to define all of the cell/range parent references and shut off the event trigger while it is working.

The following unhides everything from row 1 to the last row containing Grand Total and hides everything down to Rows.Count below it.

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim gt As Long, lr As Long
    If Not Intersect(Target, Me.Columns("A:E")) Is Nothing Then
        On Error GoTo FallThrough
        Application.ScreenUpdating = False
        Application.EnableEvents = False
        gt = Me.Columns("A:E").Find(What:="Grand Total", SearchDirection:=xlPrevious, LookAt:=xlPart).Row
        Me.Cells(1, 1).Resize(gt, 1).EntireRow.Hidden = False
        Me.Cells(gt + 1, 1).Resize(Rows.Count - gt, 1).EntireRow.Hidden = True
    End If
FallThrough:
    Me.Range("A5").Activate
    Application.EnableEvents = True
    Application.ScreenUpdating = True
End Sub

Edit: added two Me parent references to the Columns("A:E") range references.
