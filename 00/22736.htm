joeu2004 say: Vijay wrote:
Thanks for this alternate solution,  BUT I am more interested to know why COUNTIF formula not giving correct result.  In this case, should it consider "225" and "225." as same  when LEN and EQUAL TO both formulas are showing that both cells are different.

COUNTIF treats numeric text as a number.  Consider the following example (note the single-quote prefix in all except A1):
A1:  225      (numeric)
A2:  '225.
A3:  '225      (text)
A4:  '225.0
A5:  '000225
A6:  '+225.0

=COUNTIF(A1:A6,"+225") returns 6.

=SUMPRODUCT(--(A1:A6=225)) returns 1 because 225 is numeric, and it matches only the numeric 225 in A1.

=SUMPRODUCT(--(--A1:A6=225)) returns 6 because the double-negate (--) converts any numeric text to numeric, so they all match numeric 225.

tom ogilvy say:

this is what I would think:

 

ActiveWorkbook.RefreshAll

 

depends on the BackgroundQuery setting for each query table or pivot table.

 

I suspect that for the table you are processing it is set to TRUE do it does a refresh in the background (which can take a measurable amount of time).  So when your macro first runs and initiates the refresh, the table hasn't finished refreshing and since you are looking for the latest data (my surmise), it doesn't exist yet.  But when you stop you code and run it again, the query has completed and your code finds the value sought.

 

Solution:  Change the BackgroundQuery property of the table to FALSE 

 

or if it must remain as false, in your macro add  the refresh command specifically for that table.  Then you can specify the the BackgroundQuery value just for that refresh.

 

 Sheets("Sales Import").QueryTables(1).Refresh BackgroundQuery:=False

 

this is all just a guess since your code and description only alludes to that being the case.

Different header on a column page in Word 2010 - Microsoft Community

rebecca_kuss say:

Different header on a column page in Word 2010

Hi, I have a multipage document with 3 columns on each page. I have text in the header above each of the columns  but I need the text in the header to be different on each page.

mikek703 say: You have to separate your pages with section breaks instead of page breaks.  Then go into header 2 and in the header/footer tab click on "link to previous" to unlink it from header 1 and when that "same as previous" disappears from the right side of your header, you can type new text in the header without worrying about changing header 1.  Do this for each page (don't forget -- section breaks not page breaks).

stefan blom say:

If each page should have a new header you need a section break at the end of each page. This will be a nightmare to maintain!

On the other hand, if you are saying that you want to repeat, say, the current heading on each page. you can do so without section breaks. Make sure that the heading is formatted in a specific style and insert a STYLEREF field in the header pointing to that style.

To insert a STYLEREF field: Double-click the header to activate it. Click Insert tab | Quick Parts | Field. Locate STYLEREF in the list. Click it and choose the style name. Click OK.
