stefan blom say:

"Automatically update document styles" (in the Templates and Add-Ins dialog box) copies the style definitions of the template into the document, each time the document is opened, yes. Styles will be overwritten if they exist in the document.

There is also an option to "Automatically update" an individual style (this option is in the Modify Style dialog box for all paragraph styles except for the Normal style). It incorporates direct formatting applied to a paragraph into the style.

Note that if you want to reformat a document using a template, there is a different approach which is easier in many ways: Create a new document based on the template and then use the Insert File dialog box to add existing content.

To display the dialog box, click the Insert tab, click the arrow on the Object button, and then click Text from File (the command is misleading, because not only text will be inserted, but all content, from the inserted file).

3.       Remember the current server is ( the offline address book location).

4.       Completely close all office applications ( like outlook client , skype for business etc.)

5.       Open the windows file explorer and navigate to the offline address book location ( you need to copy/remember from the steps 3)

6.       Find the offline address book files folder and delete it.

7.       Go back to re-open outlook client.

8.       Click File > Account Settings > Download Address Book, then choose offline global address book and click OK to re-download it.

9.       restart the outlook client.
