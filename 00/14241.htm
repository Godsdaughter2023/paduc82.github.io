Automatic e-mail trigger at outlook startup

Is this possible to configure via Options or else VBA??

rich007a say:

Yes, you can do this in Outlook VBA.

 

In Outlook, press Alt+F11, then paste the following in the ThisOutlookSession module:

 

 

Private Sub Application_Startup()
    Dim msg As MailItem
   
    Set msg = Application.CreateItem(olMailItem)
    msg.To = "address_AT_somewhere.com"
    msg.Subject = "Message Subject here"
    msg.Body = "Your message text here"
    msg.Display     'to show the message on your screen while debugging
'    msg.Send         'to send the message
End Sub

 

 

When you close Outlook you will be prompted to Save the VBA project.

When you next open Outlook you may be prompted to Enable Macros.

 

Hope that helps.

There's no such thing as a "normal" line in Mac Office.

How you change them depends on how you added the line, and to what artefact you added it.  If you drew the line as a "Shape":

Right-click the line to format it. Then right-click again and choose "Set as default" so every other line you draw in that document will have the same formatting.
