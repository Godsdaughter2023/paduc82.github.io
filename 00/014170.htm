To create a dynamic named range, where the range is defined from O14 to the last numerical value in Column O, try...

='TM197'!$O$14:INDEX('TM197'!$O$14:INDEX('TM197'!$O:$O,ROWS('TM197'!$O:O)),MATCH(9.99999999999999E+307,'TM197'!$O$14:INDEX('TM197'!$O:$O,ROWS('TM197'!$O:$O)),1))

You'll notice that other data can be added above this range without affecting the it.
