amit tandon say:

Try below to calculate hours (in decimals) between start time and end time:

=MOD(D3-C3,1)*24

Where D3 is end time and C3 is start time. In your file C3 = 10AM and D3 = 7:45AM, so the formula returns 21.75 (which is 21 hrs and 45 minutes).

If C3 = 10PM and D3 = 7:45AM, the formula will return 9.75 (which is 9 hours and 45 minutes). Please ensure that the cell which has the formula is formatted as General.

Using MOD presumes next day if D3 less than C3.
