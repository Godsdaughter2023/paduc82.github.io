Try this:

 

Sub CopyData()
    Dim wshR As Worksheet
    Dim wshA As Worksheet
    Dim m As Long
    Dim t As Long
    Application.ScreenUpdating = True
    Set wshR = Worksheets("ReqSht")
    m = wshR.Range("A" & wshR.Rows.Count).End(xlUp).Row
    Set wshA = Worksheets("ApprSht")
    t = wshA.Range("A" & wshA.Rows.Count).End(xlUp).Row + 1
    wshR.Range("A2:C" & m).Copy wshA.Range("A" & t)
    wshR.Range("E2:E" & m).Copy wshA.Range("E" & t)
    wshR.Range("K2:K" & m).Copy wshA.Range("H" & t)
    wshR.Range("M2:M" & m).Copy wshA.Range("J" & t)
    wshR.Range("N2:O" & m).Copy wshA.Range("M" & t)
    Application.CutCopyMode = False
    Application.ScreenUpdating = True
End Sub

ken sheridan say: Nevertheless, data is being hard-coded, which is not good practice.  A fundamental principle of the database relational model is the Information Principle.  This requires that all data be stored as values at column positions in rows in tables, and in no other way. This was in fact Codd's Rule #1 when he originally introduced the database relational model.

A better solution, therefore, is to store the values in a table like so:

A    1
A    2
A    3
A    4
A    5
B    8
B    4
and so on

The RowSource property can then be a query on this table referencing the first combo box as a parameter in the usual way to correlate the controls.

hansv mvp say:

Set the Row Source Type of the 2nd combo box to Value List.

Create an After Update event procedure for the first combo box:

 

Private Sub Combo1_AfterUpdate()
  Me.Combo2 = Null
  Select Case Me.Combo1
    Case "A"
      Me.Combo2.RowSource = "1;2;3;4;5"
    Case "B"
      Me.Combo2.RowSource = "8;4;27;123"
    Case "C"
      Me.Combo2.RowSource = "456;789;123"
    Case Else
      Me.Combo2.RowSource = ""
  End Select
End Sub

susan_s. say:

Hello 1v4n0,

 

Welcome to the Skype Community forum!

The last 30 days of your conversation history are stored in the cloud, so you can access it when you’re signed in to Skype on any device.

History older than 30 days is saved on your hard drive, according to your history settings. 

You can back up your history if for any reason you need to uninstall Skype, or if you want to transfer your history from one computer to another – for example, from a Windows PC to another Windows PC, or a Windows machine to a Mac. You just need to copy the main database file in your Skype Name folder. 

To save and transfer your conversation history, follow this:

  1. Press and hold the Windows key       
  2. Type %appdata%\Skype into the Run window and press OK.
  3. Navigate to the folder named after your Skype Name.
  4. Right click on the folder itself and select Copy. This will copy the entire folder including all its contents.
  5. Save the folder to an external media device.
  6. On your new computer, install and sign in to Skype so that the necessary folder is created, then sign out of Skype again.
  7. Still on your new computer, go to the Run window again, type %appdata%\Skype and press OK.
  8. Navigate to the folder named after your Skype Name.
  9. Replace the entire folder (named after your Skype Name) with the folder that you saved in step 4 and 5.
  10. Sign in to Skype. Your chat history will be there when you browse through your conversations.

For more information, click here.
