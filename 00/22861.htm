With the new Skype, the conversation history is now saved on the Cloud for the 2 years period. When Skype is installed first time, the chat history is pulled out of the Cloud, and currently this will correspond to about 1½ history period. This is why the oldest chat messages you are seeing now on Skype are from 2017.

 

If you had previously the classic Skype (Skype 7.X version) installed on your computer, then the older messages might be still saved in the local settings folder of this Skype application.

 

%appdata%\skype\<your Skype name>main.db

 

You can access the chat history saved in this main.db database file using 3rd party free SkypeLogView application.

 

https://www.nirsoft.net/utils/skype_log_view.html

 

Also, if you install Skype for Desktop application, you will have there an option to export the chat history saved in the main.db file.

 

The currently latest 8.34.0.78 version can be installed using this direct download link:

john korchok say:

126mb is not a big video. But the flavor of QuickTime may not be what PowerPoint is expecting. Try opening the problem file in QuickTime Player. The Inspector will say whether the file is H.264, which is what PowerPoint likes. Then use File>Save... to resave with a different name. If that doesn't fix it, try using File>Export to a different QuickTime format.

If the codec is something like Sorenson instead of H.264, I believe you need QuickTime Pro to convert it.
