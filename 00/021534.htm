marshall barton say:

As I said, users can enter the same kinds of things you can enter in the query designer's Criteria row.

For example, if you have a form with a text box (named txtSrchA) where users can enter the search conditions for a Text field named FieldA and another text box (named txtSrchB) for a Currency field named FieldB, then the BuildCriteria function can be used in a VBA procedure like this:

   If Not IsNull(Me.txtSrchA) Then
      strWhere = strWhere  & " AND " & BuildCriteria("FieldA" , dbText, Me.txtSrchA)
   End If
   If Not IsNull(Me.txtSrchB) Then
      strWhere = strWhere & " AND " & BuildCriteria("FieldB" , dbCurrency, Me.txtSrchB)
   End If
   DoCmd.OpenForm "SearchResults", WhereCondition:= Mid(strWhere, 6)


 and that would take care of users searching FieldA for criteria like:

   "Red"
   Like "*ABC*"
   Not Like "*##/##/##*"
   ="Red" Or ="Blue" Or ="Teal"
   IN("Red", "Blue")
   Not IN("Pink", "Purple", "Lime")

and things like this for FieldB:

   12.95
   >=15.00 And < 20.00
   Between 15 And 19.99

Let me start by stating that I'm a former computer security consultant. Computer security over the years has been a cat&mouse game. OS'es get minor security improvements - hackers get more sophisticated tools. The problem is that the hackers are getting better at a faster pace than the OS.

$adminUPN="<the full email address of a SharePoint administrator account, example: jdoe@contosotoycompany.onmicrosoft.com>"

Function Add-OnedriveSecondaryAdmin($AdminURL,$SecondaryAdmin)

{

    #connect Spo service.

    Connect-SPOService -Url $AdminURL

    #Get all Onedrive URL's.

    $OneDriveURLs = Get-SPOSite -IncludePersonalSite $true -Limit All -Filter "Url -like '-my.sharepoint.com/personal/'"

    foreach($OneDriveURL in $OneDriveURLs)

    {

        #Add Secondary administrator to Onedrive Site.

        Set-SPOUser -Site $OneDriveURL.URL -LoginName $SecondaryAdmin -IsSiteCollectionAdmin $True -ErrorAction SilentlyContinue

        Write-Host "Added secondary admin to the site $($OneDriveURL.URL)"

    }

}
