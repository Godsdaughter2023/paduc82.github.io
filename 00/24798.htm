cj. eichholz [msft] say:

Yes - cause it depends on what you are scanning. 

Full Scan runs a scan across the entire files on each drive.  So if you have any ISO's or large ZIP/CABs it will go into each one and scan those too. 

Running a full scan isn't a bad idea, but you may just want to run a custom scan cross just the system drive (this is usually c:\) to build up the Windows Defender cache. 

Since it is the first time you are running a full scan on the system, you are building your cache, so each file gets a check before added to the cache.  On the next scan (same goes for Quick Scan) it will be faster since it will know which files to rescan vs trust since its in the cache.

a. user say:

Some array concepts...

ShtArray1(3, 50) is an array with 4 rows and 51 columns.
Arrays are zero based unless you specify otherwise. 
So you are actually sizing the arrary as ShtArray1( 0 to 3, 0 to 50)

Also,
Range ("A1:C50) has 50 rows and 3 columns. and will not fit in your array.
Using a Variant variable to contain a set of range values is usually a better option...
  Dim arrSht as Variant

  arrSht = Range ("A1:C50).Value
 'the above creates a 2 dimensional array (1 to 50, 1 to 3) containing the range values.
  So arrSht(3, 2) contains the value in cell B3

In addition,
A 3 (or more) dimensional array (2, 3, 50) is a difficult concept for most (it is not whatever you think it is), and is usually not needed.

chatmaker say:

I have the same problem.  I was hoping for help here but...Here is how I'm working around this. 

 

Observations

When this happens, the error you reported shows in a rectangle where the image should be.  The rectangle has the same name as the Picture.  Pictures are shape type 13.  Rectangles are shape type 11.

 

Strategy

Find all Shapes of Type 11 and replace with the original picture.

 

Fix

I don't know how to "Change Picture" through VBA.  Research came up empty.  But we can delete the shape and reload the Picture.  Here is a sample:

 

    With ActiveSheet
        If .Shapes("Logo").Type = 11 Then
            With .Shapes("Logo")
                h = .Height
                w = .Width
                t = .Top
                l = .Left
                .Delete
            End with
            With .Pictures.Insert(ThisWorkbook.Path & "\Logo.gif")
                .Name = "Logo"
                .Height = h
                .Width = w
                .Top = t
                .Left = l
            End With
        End If
    End With

 

This, of course, assumes the image is in the same directory as the workbook which may, or may not be your situation and will probably not be the situation when the workbook is emailed (sorry).
