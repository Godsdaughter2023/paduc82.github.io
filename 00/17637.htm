rob koch say:

As the card company customer support stated, it's likely just a coincidence.  That's especially true since it's unlikely the information could have been misused that fast, since it still takes time for the thief to create a fake card and go to a store or even simply use it online, which is less likely since that would require a false delivery address and identity as well.

I work in the computer security field and have sometimes had literally dozens, if not hundreds of individual account names and passwords to remember.  I used various sets of similar, but slightly different passwords for similar purposes, with major differences only for the handful of highly critical accounts like the bank, other financial institutions or social security.  For example, using the same base password for multiple utility company accounts, with an additional letter or number added to the beginning or end for each, which makes them all different but easy to remember.

You can also use other common tricks such as converting certain letters like A to 4, E to 3, o to 0 (Zero) or similar, but these depend on your ability to apply these in memory.  If you search for "strong password" online you'll find lots of articles and tools, but typically simpler to remember is better, with password length more important than complexity.  Using a phrase with either the trick above and/or replacing spaces with numbers or an easily remembered sequence like the following might work for important accounts you access rarely.  Note the upper case letters and/or look-alike number substitutions in the second example.

First1Second2Third3Fourth4Word#!  - (Note: Special characters such as # or ! are not allowed on all systems)

F1r5T 53c0nd Th1rd F0urth W0rd  - (You could remove spaces here, I left them for clarity)

Th3 P455w0rd F0r My B4nk$$$  - (This example should now be obvious, spaces again included for clarity)

You can come up with many variations, but picking one or two and using them consistently is best, since that's easier to remember.  The only difficulty are exceptions like those sites that limit passwords to short lengths like 10 or 12 characters, which only leaves complexity (e.g. numbers, upper/lower case letters and special characters, no dictionary words) to insure they aren't simple to crack.

If someone either manages to capture your passwords using a keylogger or malware, it doesn't matter how complex they are, the computer will get every character exactly as it's typed.  This is the fallacy behind complex passwords, they only make remembering the password difficult for the human using it and do nothing to protect from leaving them lying around on a piece of paper or in a spreadsheet on the computer.

Rob

The answer is in your Settings App under Activation:

You might possibly have noticed that even if you do a clean install of the public build of Windows, without logging in with a Microsoft Account, it will still auto-activate. The section shown above would then read the same, except for the phrase "linked to your Microsoft account". This is because it is your device that has the Digital License for Windows. That license can be linked to your Microsoft Account, but it does not depend on that link. If you were to sell your computer to someone else it would still auto-activate Windows 10 Pro if they were to clean install it.

So there is no dependence of your digital license on the Insiders Program at all. In fact Insider Builds would not activate if you did not already have a digital license for Windows 10 for that device. There is nothing stopping you from never using Insider Builds again if you so choose.

What linking to your Microsoft Account does provide is a means to look up your license and activate Windows if you were to make a major hardware change, such as changing the motherboard, which would prevent the auto-activation process from seeing it as the same device. In that case, the Activation Troubleshooter can help to find the Windows 10 Pro license and re-assign it to the new hardware.

wangyanyu say:

Hi Gamchi,

Office 365 Business Premium doesn’t include Office 365 ProPlus and it instead includes the Office suite that is named 365 Business. Office 365 ProPlus in the Office 365 Enterprise E3 plan instead of Office 365 Business premium can be used by multiple users on a shared device or virtual machine. The feature is not available for Office 365 Business plan.

Shared computer activation gets enabled during the installation of Office 365 ProPlus using the Office Deployment Tool. Once enabled, Office installs without being activated. When a user signs in to a computer with Office installed via shared computer activation, Office will check to see if the user has been provisioned for Office 365 ProPlus and temporarily activates Office 365 ProPlus. If a second user signs in to the same computer, the activation does not persist from the first user and process is repeated. Deploying Office 365 ProPlus using shared computer activation does not count against a user’s five total installations of Office 365 ProPlus or Office for Mac

The most common shared computer activation scenario is to deploy Office 365 ProPlus to shared computers by using Remote Desktop Services (RDS).The following are two common RDS scenarios:
1. Install Office 365 ProPlus on an RDS server.
2. Install Office 365 ProPlus on a shared virtual machine.

For specific guidance on how to deploy Office 365 ProPlus by using Remote Desktop Services (RDS) and shared computer activation, see Deploy Office 365 ProPlus by using Remote Desktop Services. More information on how to enable shared computer activation for Office 365 ProPlus can be found in the article you mentioned above.
