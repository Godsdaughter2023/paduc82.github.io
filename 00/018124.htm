ossiemac say:

Empty is a reserved word in VBA. Best to prefix your variables with characters that represent the type of variable.

You don't have any parameter variables in Function Colorcount to accept the parameters (or arguments) to the User Defined Function.

Try the following code. The formula in cell B1 to return the count will be   =Colorcount(A1,C3:E3)

Function ColorCount(rngExample As Range, rngCountColors As Range) As Long

    Dim lngEmpty As Long
    Dim lngColor As Long
    Dim rngData As Range
   
    lngColor = rngExample.Interior.Color
    For Each rngData In rngCountColors
        If rngData.Interior.Color = lngColor Then
            lngEmpty = lngEmpty + 1
        End If
    Next rngData
    ColorCount = lngEmpty
End Function

robert j stroud say: Here's a workaround for the missing Document.VBProject property based on Peter's approach:

Function FindProject(d As Document) As VBProject
    Dim p As VBProject       
    For Each p In Application.VBE.VBProjects
        'On Error Resume Next
        If (p.FileName = d.FullName) Then
            Set FindProject = p
            Exit Function
        End If
    Next
End Function

Note that the "On Error" statement is commented out - it needs to be enabled on a PC because p.FileName is undefined for a document with no template, so if you have any such documents open, the test will throw an exception.

I hope this is useful to someone.

Best wishes,

Robert
