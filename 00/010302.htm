I have a list of names in range G:J

Each cell in the range contains a string of several names

I have extracted the individual names to Col E and placed their corresponding email address in col D.

I want search the range G:J for a name located in col E and replace it with the persons email address, which is in cell D.

Any help greatly appreciated!

Sandra

 

 

sandracw say:

Solved!

Sub SearchMacro

Dim sName, sEmail As String
 

For a = 2 To 464

  sName = Cells(a, 5)
  sEmail = Cells(a, 6)
 
  Range("G2:J61").Select
 
  Cells.Replace What:=sName, Replacement:=sEmail, LookAt:=xlPart, _
    SearchOrder:=xlByRows, MatchCase:=False, SearchFormat:=False, _
    ReplaceFormat:=False

next a

End Sub

cholmesrtb say:

You can't see the double-headed diagonal arrow because you need to have outlook running in cache mode. I'm assuming the people pane could become a huge drain on the exchange server in large organisations. Outlook running in cache mode will eleviate this problem and allow you to see full details in the people pane.

steve rindsberg say:

When you APPLY a template to an existing presentation, the presentation inherits formatting but not content from the template file; in this case, your VBA doesn't come over.

Try basing a new presentation on the POTX to see what happens; that should bring over the content, including macros, into the new presentation.

On the whole, though, you don't want to mess with distributing macros via templates; it works out well for Word and Excel but not for PowerPoint.

Consider creating an add-in instead.
