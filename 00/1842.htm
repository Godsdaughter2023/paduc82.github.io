ossiemac say:

In the rule type select "Use a formula to determine which cells to format" and enter the following formula.

=AND(A1>=B1*0.9,A1<=B1*1.1)

The formula is applied to the first cell of the range to be formatted and Excel looks after applying to the remaining cells in the Refers to range.

To alter the Refers to range

  1. Select a cell that contains the conditional formatting
  2. Select Conditional formatting
  3. Select Manage rules
  4. Select the required rule
  5. Click in the Applies to field (to right of displayed formatting)
  6. Press F2 to enter editing mode otherwise moving arrow keys selects cells.
  7. Click the icon at right hand end of the Refers to field (Opens another field)
  8. Select the required range (eg column A)
  9. Click the icon at right hand end of field to return to previous dialog
  10. Click Apply and OK.

Note that with above formula the blank cells will always be within the 10% range if you select all of column A as the Refers to range. Therefore maybe include another condition in the AND operator formula to exclude blank cells like following.

=AND(A1>=B1*0.9,A1<=B1*1.1,A1<>"")
