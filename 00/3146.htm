dorilynn say:

How do I format a content control with numbers and a dollar sign?

I work at Montana Department of Transportation in User Support.  One of my customers created a form in Word 2010 and is using the text content control.  We cannot find in properties where you can select the format - similar to how you select what number format you want in Excel 2010. 

 

She is running Windows 7 with the Office 2010 products. 

 

Please let us know how to accomplish this.

 

Thanks,

 

DoriLynn

macropod say:

Basically, you don't. Word doesn't have that kind of automatic formatting. Instead, you can use a ContentControl_OnExit macro to apply the formatting. Such a macro, which goes in the document's 'ThisDocument' module, might look like:

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
Dim CCtrl As ContentControl
With ActiveDocument
  For Each CCtrl In .ContentControls
    With CCtrl
      If .Title = "MyCCtrl" Then
        With .Range
          If IsNumeric(.Text) Then
            If Format(.Text, "0.00") > 999.99 Then
              .Text = Format(.Text, "$0,000.00")
            Else
              .Text = Format(.Text, "$0.00")
            End If
          End If
        End With
      End If
    End With
  Next
End With
End Sub

Simply give the content control a title and replace 'MyCCtrl' with that title.

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)

jay freedman say:

If you know what text will be in the text box when the userform appears (because your code is creating that text), you can assign a specific number to the text box's .SelStart property. The correct number is the number of characters to the left of the insertion point.

Private Sub UserForm_Initialize()
    With TextBox1
        .Text = "This is the text when the userform starts."
        .SelStart = 12  'before the t in "text"
        .SetFocus
    End With
End Sub

If you want one or more characters to be selected instead of just an insertion point, also set the .SelLength property's value.

If you don't know what text will be there (for example, you're copying in text from the document body), you can use the InStr function on the box's .Text property to search the string for a particular substring. To put the insertion point at the end of the text, use

   .SelStart = Len(.Text)

palcouk say:

You cannot locate a connected outlook data file on a Network/LAN/OneDrive location, its not supported and will lead to data corruption

You can Archive or move mail to a local *.pst data file, and then include that pst in your data backup process.

An *.ost data file is used under an IMAP or Exchange mail account and is simply a local cached copy of what is on the mail server. And as such if you add the same mail account on another device mail will be downloaded/synced with what is on the server, ie all mail data will be reproduced.

However if Contacts/Calendar/Tasks are in folders marked 'This PC only' they will be lost if the mail account is re-set (moved)
