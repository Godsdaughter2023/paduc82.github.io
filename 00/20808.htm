Adding extra payment to Loan Schedule - Microsoft Community

rick 01810 say:

Adding extra payment to Loan Schedule

I have created a loan schedule for a 60 month car loan. I want to increase my payment beginning in year 3. How do i edit my loan schedule to reflect this change?

joeu2004 say: Rick wrote:

I have created a loan schedule for a 60 month car loan. I want to increase my payment beginning in year 3. How do i edit my loan schedule to reflect this change?


Depends on the design of your loan schedule.  If you are using a MS template, which one?

Many loan amortization schedules that I have seen do not compute principal in a manner that allows you to change the payment.  Some do not even handle payments that are properly rounded (at least to the cent); so the final payment is incorrect.

But in a well-designed schedule, it should be trivial.  Generally, payment should be one column, interest should be another column, and principal should be computed as:  payment minus interest.  Remaining balance can be previous balance minus payment.

Download my "loan schedule.xls" from http://www.box.net/shared/s9mk5g438b0gyh77ps32 for a bare-bones loan amortization schedule that provides for additional payments.

tasos_k say:

Hi,

Assuming that drop down menu list is in Sheet1, in cell A1

Since you are interesting, only for cells a2-a9 (lock-unlock- protect-unprotect)

if Yes  then [a1-a9: lock and  protect, rest area: unlock and unprotect] , else [unprotect and unlock all]

try this..

right click on Sheet1 tab, select View Code and paste in....

Private Sub Worksheet_Change(ByVal Target As Range)
'Mar. 26, 2016
Dim sPass
sPass = "123" '<< change password
Dim rng As Range
Set rng = [A2:A9]
If Not Intersect(Target, [A1]) Is Nothing Then
With ActiveSheet
.Unprotect Password:=sPass
.Cells.Locked = False
If Target.Value = "Yes" Then
rng.Locked = True
.Protect Password:=sPass
End If

End With

End If
End Sub
