a. user say:

For a workbook that has been saved to disk (i.e. has a file name), you can use ADO to retrieve data in a sheet, a named range, or a range in a sheet. But there are some well-known problems when using ADO to get data from a worksheet - the data may not look like what you actually see in the sheet, and the Jet/ACE provider can play havoc with the data when there are mixed data types in a column. 

But as an example

' Use VBE Tools->References to make a reference to the Microsoft ActiveX Data Objects library

Sub getdatafromrange()
Dim cn As ADODB.Connection
Dim rs As ADODB.Recordset
Dim s As String
' Type 35 for an Excel 97-2003 .xls, type 37 for a .xlsx, possibly other values
s = "Provider=Microsoft.ACE.OLEDB.12.0;User ID=Admin;Data Source=" & Me.FullName & ";Mode=Share Deny Write;Extended Properties=""HDR=YES;"";Jet OLEDB:Engine Type=37;"
Set cn = New ADODB.Connection
cn.Open s
Set rs = New ADODB.Recordset
' For a sheet called abc, use [abc$]
' For a range name called def, use [def]
' For a range A4:B7 in sheet ghi, use [ghi$A4:B7]
' (Use Am:Bn notation even if Excel is set to use R1C1:RnCn notation)
rs.Open "SELECT * FROM [Sheet1$]", cn
Debug.Print rs.Fields(0).Name
While Not rs.EOF
  Debug.Print rs.Fields(0).Value
  rs.MoveNext
Wend
rs.Close
Set rs = Nothing
cn.Close
Set cn = Nothing
End Sub

You could also use DAO. It may be possible to use a WorkbookConnection but I do not see how.

ruel mayoya msft say:

Hi Pstoric,

I understand that you want to share your calendar with another user. You can share your calendar with internal and external users.

Based on your description, I’d like to collect some information,  also please follow these steps to narrow down the issue:

Suppose there are two users: User-A and User-B

1. The screenshot of the error regarding “cannot create events on that shared calendar”.

2. When you are about to share the calendar of User-A to User-B, you may select other View Details such as: Availability only, Limited details and Full details.

3. On the Calendar of User-A, On the Ribbon, click Calendar Permissions icon, and you can Add User-B, then choose the Permission level, such as “Owner”.


4. On the Calendar of User-B, highlight the shared calendar (User-A), then click again Calendar Permissions in the Ribbon. Now you can change the setting here with the permission given to you by User-A’s Calendar. You can also refer to this article. If you want to share in OWA, go to this article.

5. Regarding the error ” the delegates page is not available. Cannot access outlook folder”, check if the issue can be fixed by creating a new profile, Online Repair and applying new updates.
