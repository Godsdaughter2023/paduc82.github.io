stefan blom say:

Paragraph styles are the foundation of text formatting in Word. Often, paragraphs are formatted in Normal style, unless you have explicitly picked something else (for example in the Styles pane or the Apply Styles pane).

Whenever you modify text directly, for example by selecting it and choosing a different font, you apply direct formatting, that is, formatting not stored in the current style. Paragraph styles include not only font formatting, such as font and size, but also paragraph level formatting, such as line spacing and spacing before/after. Ctrl+Q runs the ResetPara command, which removes direct paragraph formatting. There is a corresponding command, Ctrl+SpaceBar, that removes direct font formatting.

A bullet format on the Home tab applies bullet formatting on top of the current style. You can get rid of it again by pressing Ctrl+Q (the shortcut runs the ResetPara command).

A bulleted paragraph style, on the other hand, preserves bullet formatting until you modify the style or apply some other style. The article I linked to describes how you can set up a multilevel list of bullet paragraphs at different levels, with one built-in bullet style for each level. 

For more on styles in general, see http://shaunakelly.com/word/concepts/styles.html.

karl timmermans (outlook mvp 2012-2018) say:
>>
how to import contacts from windows 7 to windows 10
<<

First, for the sake of accuracy, importing contacts is not an operating system based issue.

What is important is

#1 - What version of Office/Outlook is being used on Win7 (am assuming that you are using Outlook on Win7 and are or intending to use Office/Outlook 365 on Win10 for the purpose of this response)

#2 - What type (POP3, Imap or Exchange) of email account is configured on your Win7 system? If not sure and you're using Outlook '2010 or later, you can check by going to Outlook menu --> File --> Account Settings -> email tab --> account type is on the right column. If you have more then one email account configured, switch to the Data files tab and check to see if what data file is set as the default data file for the profile --> the default data contains the default contact/calendar folders

Without knowing anything else at this point

#3 - If this involves a POP3 account AND you're contacts are contained in a PST file (default data file) - best approach would be to copy that PST file to the Win10 system. Will provide more instructions if this is applicable

#4 - If this involves an IMAP account using Outlook '2010 or earlier AND the default file is a PST file (which would be the case normally) - same as #3

#5 - If this involves an IMAP account using OL'2013 or later AND the contacts are stored in a folder with the added words "This Computer Only" in the folder description, then you would need to export the contacts (and calendar if applicable) to a PST file and copy that file to Win 10 - same as item #3

#6 - If this involves an Outlook.com/Exchange account - nothing needs to be done. The info will sync when you configure the account on the Win10 system
