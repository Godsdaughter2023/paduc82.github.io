auto populate form based on combobox selection

I have a form that includes a combo box with style codes.  I would like two other fields to auto populate based on what I select in the style code combo box.  The two fields that auto populate need to be editable since there are some exceptions and I would like them to also be drop down lists, so if they do need to be updated they pick from a list.

john w. vinson say:

You can "push" data from the combo box into bound controls on the form, using the style code combo box's AfterUpdate event. Include the two fields in the style combo's RowSource query (you can set the ColumnWidths property to zero for these columns so they're not visible), and use code like

 

Private Sub cboStyleCode_AfterUpdate()

Me!cboFirstOtherCombo = Me!cboStyleCode.Column(2)

Me!cboSecondOtherCombo = Me!cboStyleCode.Column(3)

End Sub

 

Note that the Column property is zero based, so this will copy the third and fourth fields in the combo's Rowsource into the bound column of the other combos.
