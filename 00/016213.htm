I want to create a Word document where every page has fixed headings or prompts, where the user adds more info, like this (fixed part shown in bold, comments after //):

Trip: London to Paris

// next page

Date: Monday 17 May 2010
Trip: Paris to Rome

When a new page is created, the parts in bold should appear. Using Header and Footer doesn't work, as they can't be different every page. I have Office 2007, but ideally this would work with earlier versions.

-- Nigel M

doug robbins - mvp office apps & services (word) say:
If you use a macro containing the following code when you want to begin a new entry, it will insert a Next Page Section break at the end of the document, unlink the header in the new section from that in the previous section and then ask you for the date and the trip details which it will then insert into the header of the new section in the format required and finally position the cursor at the beginning of the section ready for you to begin typing:
 
Dim rng As Range
Dim strDate As String
Dim strTrip As String
strDate = Format(InputBox("Enter the date", "Date", Format(Date, "dd/mm/yyyy")), "dddd, dd mmmm yyyy")
strTrip = InputBox("Enter to Start and Finish locations", "Trip Details", "From here to there")
With ActiveDocument
    Set rng = ..Range
    rng.Collapse wdCollapseEnd
    .Sections.Add rng, wdSectionNewPage
    With ..Sections(.Sections.Count).Headers(wdHeaderFooterPrimary)
        ..LinkToPrevious = False
        ..Range.Text = "Date: " & strDate & vbCr & "Trip: " & strTrip
        ..Range.Paragraphs(1).Range.Words(1).Font.Bold = True
        ..Range.Paragraphs(2).Range.Words(1).Font.Bold = True
    End With
    Set rng = .Range
    rng.Collapse wdCollapseEnd
    rng.Select
End With

--
Hope this helps.

graham mayor say:
If you are starting from scratch, then this is relatively easy to achieve. The following macro will prompt for the date and the trip information and write it at the start of a new page. When you have finished writing about your trip, run the macro again to create a new page with the required heading on the new page in the same format.
 
If you want to pick the date from a date selector then instead of the InputBox you will need to call a userform - see http://www.gmayor.com/popup_calendar.htm . If you are calling userforms then you may as well combine the calendar picker and the trip field on the same userform - For the basics, see Word MVP FAQ - Userforms
http://word.mvps.org/FAQs/Userforms.htm
 
for a more in depth explanation, see
http://gregmaxey.mvps.org/Create_and_employ_a_UserForm.htm 
 
Dim PromptForNewPage()
Dim oRng As Range
Set oRng = ActiveDocument.Range
oRng.Start = oRng.End
If Len(ActiveDocument.Range) > 1 Then
    oRng.InsertBreak Type:=wdSectionBreakNextPage
End If
oRng.Start = ActiveDocument.Range.End
oRng.Text = "Date: " & InputBox("Enter Date", _
"Date", Format(Date, "dddd d MMMM yyyy")) & vbCr
oRng.Words(1).Bold = True
oRng.Start = ActiveDocument.Range.End
oRng.Text = "Trip: " & InputBox("Enter Trip title", "Trip") & vbCr
oRng.Words(1).Bold = True
Selection.EndKey wdStory
End Sub
 
If the pages are already written, then the task is somewhat more complicated. For that it would be necessary to know how the pages are broken. Frankly I would rather not go there unless essential.
