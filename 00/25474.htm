michael g bennett say:

Can you stop co-authoring and require checkout / check in

This is are really great feature and we are excited to see this and start using it within our company.  I have a question from one of our users about no allowing co-authoring, but require single editing with Check-out / Check-In, is this an option to disable co-authoring; or does turning off auto-save remove the co-authoring?

Thanks,

micah [msft] say:

Hi Michael,

Checking out the file does prevent coauthoring. When a file is checked out anyone else that tries to open it will get the "File is Locked" message that you may be familiar with from other versions of Excel.

Turning off AutoSave does not disable co-authoring but the answer is a little more nuanced than that. The live features of coauthoring (such as seeing others changes as a person makes them and seeing where they currently are in the file) will be disabled, but multiple people can still make changes and when each person presses the save button, their changes will be added to the file. It's also possible to have any combination of people with AutoSave on and AutoSave off. For example: two people have AutoSave on and a third has AutoSave off. In this scenario, the two people with AutoSave on would see each others changes as they are made, but they wouldn't see the changes from the person with AutoSave off until that person manually presses the save button. Similarly, the person with AutoSave off will not see any of the changes from the other two people until pressing the save button.
