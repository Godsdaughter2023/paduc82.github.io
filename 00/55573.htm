Try these

I) Reset the TCP/IP stack.

Meanwhile, you receive an error "The driver itself it not digitally signed" due to the Windows 10's Driver Signature Enforcement. This feature only load drivers that have been signed by Microsoft. If manually replacing Intel driver netwfw02.dat somehow works for you, then you may try disabling the Driver Signature Enforcement:

1. Click on Start, search for Command Prompt, right-click on it and select Run as administrator.

2. Type the commands below and press enter after each command:

bcdedit.exe -set loadoptions DISABLE_INTEGRITY_CHECKS

bcdedit.exe -set TESTSIGNING ON

3. Reboot your device.
