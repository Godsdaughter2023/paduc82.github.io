Hi All,

Based on my experience and test, all downloaded files from Internet will be regarded as unsafe and “blocked” by Windows system. An exception is that the site we download files from is listed under Trusted Sites.

Therefore, in this case, the best practice is to add the site (you regularly download files from and trust) to Trusted Sites list. To do this, you can open IE > tools > Internet Options > Security > Trusted sites > Sites > input the site address > click Add > close > OK.
Note: Make sure the exact file download address is included in the site Url you add to trusted sites. In some cases, it could be that file is downloaded from another site other than the site you see it.

For my example:
 

Meanwhile, I'd like to summarize the two options you have mentioned,
1.    Go to "download" folder > right click on the file > click properties > unblock the file
       This option is recommended when you want to unblock an individual file and keep blocking all files for security reasons.
2.    Open Excel > Files > Options > Trust Center > Trust Center Settings > Protected View > Enable Protected View for file originating from the Internet
       This option can be used when you understand the potential risk and still want to unblock all the files downloaded from untrusted Internet locations.

Let us know if suggestion above can resolve the issue.
