Hi,

 

I was having the same problem with my 6600 slide, when trying to use the software updater. I noticed that the phone was connected on port COM11, i simply changed this to the lowest COM port available (in my case COM1) and the software updater picked it up.

 

To change COM port right click

  1. My Computer and select Properties
  2. Select Device Manager

  4. Right click your Nokia phone USB and select Properties
  5. Select Port Settings
  6. Click Advanced button
  7. Choose a new COM port from the drop down list
  8. Click OK when done
