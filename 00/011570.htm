joeu2004 say: James wrote:

I'm trying to make a cell auto fill to the color of the RGB values in the cells before it.  Example if A2 has the red value, B2 has the blue value, and C2 has the green value, D2 would autofill with the RGB color from cells A2:C2


Not sure what you mean by "autofill" in this context.  It might mislead you into unnecessarily complex solutions.

The following macro sets the background color of the selected range (presumed to be a single column of cells) to the RBG values in the 3 columns adjacent to the left.

(Note:  Beware of periods (.) just to left of some words like .Interior and .Offset.)

Option Explicit
Sub doit()
Dim cel As Range
Application.ScreenUpdating = False
For Each cel In Selection
    With cel
        .Interior.Color = RGB(.Offset(0, -3), .Offset(0, -1), .Offset(0, -2))
    End With
Next
Application.ScreenUpdating = True
End Sub

Design issue....  Since we (and you) call it R G B, it would be nice if their component values were in that order.

So instead of "A2 has the red value, B2 has the blue value, and C2 has the green value", it would be nice if B2 were green and C2 were blue.  (What a concept!  Patent pending. ;->)

If you accept the reasonableness of that expectation, change RGB call above as follows:

.Interior.Color = RGB(.Offset(0, -3), .Offset(0, -2), .Offset(0,
