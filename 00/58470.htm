a. user say:

Hi Dennis,

Thank you for posting your query in Microsoft Community.

Follow the below mentioned steps to create a Standard Local user account:

* Press Windows key + X, select Command Prompt(Admin)

* Type "net user useracc /add" (without quotes) and hit Enter Key

Note: useracc should be the name of the User account your want to create. For example: If you want account name as 'computer', then the command will be "net user computer /add"

*Type "net localgroup administrators useracc /add" (without quotes). Using this command you can make the desired account with administrator rights.

* Type "net user useracc *" (without quotes). This command allows you to create a password to your account

* Once done, type Exit to close the command prompt. Sign-out from the current account to login to the created New account, which you can find on the login screen.

Hope this will address your query.
