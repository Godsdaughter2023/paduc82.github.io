  2. Try updating first in Windows Defender's Update interface and if it failed, try Windows Update.
  3. If it didn't work, make sure to install all updates (if any) and restart your PC and try the update again.
  4. If problems persist,

For more information you may refer to the article create by our community contributors in the following link: Troubleshoot Windows Defender Update Issues in Windows 10.
