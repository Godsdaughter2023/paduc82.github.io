michael. o say:

This sounds like your device is moving from a USB 3.0 host controller, an eXtensible Host Controller (XHCI), to a USB 2.0 host controller, an Enhanced Host Controller (EHCI).  To validate, when your device is operating slower than expected, ensure it is connected to a USB 3.0 port.  Open Device Manager and view "Devices by Connection."  Find your USB 3.0 device and see what type of host controller it is connected to.  If it's connected to an EHCI Host Controller, it will only operate at High Speed.

 

To address this issue, you will need a BIOS update from your PC OEM.  Please contact them and ask for an update.
