Hello,

Thank you for posting your concern here in Microsoft Community and welcome to the Windows 10 Family.

From the description of the issue, I understand that System Restore Points are automatically created in Windows 10.

Please follow the below steps and check if the below steps helps to fix the issue.

1. Press the keyboard shortcut Windows key+R to open the Run Command Dialog Box,

2. Type gpedit.msc in the Dialog Box and press Enter. this will open the Group Policy Editor.

3. Now navigate to: Computer Configuration -> Administrative Templates -> System -> System Restore.

4. In right-side pane, make sure both "Turn off Configuration" and "Turn off System Restore" options are set to "Not Configured". If both or any of them is set to "Enabled", double-click on the option and set its value to "Not Configured".

Now in order to delete the  delete all Restore Points, which will clear up the most space, follow the below steps.

1. Follow the path: Start> Control Panel> System

2. Click on System Protection.

3. In the System Properties dialog select the System Protection tab, highlight Local Disk C: then choose Configure.

4. Click on Delete. This will delete all System Restore Points, if created any.

Note: Also refer to the Disk Space Usage section and check for the Max Usage slider.
