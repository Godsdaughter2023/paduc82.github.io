In the macro editor (Alt+F11), locate the module that contains the macro. If you recorded the macro, that module will probably be the one named NewMacros in the Normal template project.

If the macro is the only one in that module, you can right-click the module's icon in the Project Explorer pane and click Export. The result will be a text file with a .bas extension, with whatever name and folder you choose in the Save dialog. The other user can use the Import command on the right-click menu to import the module from the .bas file.

If there are other macros in the same module, and you want to send only the one of interest, you can select and copy the code in the right side of the editor, and paste it into Word or Notepad. Save it as a text (*.txt) file to send to the other user. The other user will have to copy/paste it from that file into a module (either new or existing).
