A collection of fonts that is usable with the bold and italic buttons is a font family. In the Windows font folder, a font family shows as a multipage icon. By contrast, single fonts which ware not part of a family have a single page icon:

If a font family has all four weights installed, using the bold and italic buttons switches to use the true font. The bold is a real bold, the italic a true italic. If only the regular weight is installed, using bold and italic will fake the look.

steaxa say:

if in worksheet 2 you have the customer's code over the customer's address you can use in worksheet 1 (if you have the customer's code):

 =VLOOKUP(customer's code; table with customer's code and address; number of the columns in which is the address)

You can use vlookup if you have the two sheets in the same workbook, otherwise you must construct a macro.

1. From the Outlook.com Options screens under "Preventing junk email", click Safe and blocked senders.

1. In the Outlook.com Inbox, click the cog icon at the upper right part of the screen and then select More email settings.

3. Click Blocked senders.

4. In the text box, type in the email address or domain you’d like to stop email receive from, and then click Add to list >>.
