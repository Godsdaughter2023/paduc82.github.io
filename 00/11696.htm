For on-demand scans, you always have the option to quarantine or allow the threat at the time of detection by clicking on will let you choose the action for threats that are detected by real-time protection:

 

Copy, paste, and run (enter) these commands at the Administrator Command Prompt or PowerShell Prompt to configure Automatic Remediation:

 

Turn off Automatic Remediation:

 

REG ADD "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableRoutinelyTakingAction /t REG_DWORD /d 1 /f

 

Turn on Automatic Remediation:

 

REG DELETE "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableRoutinelyTakingAction /f

 

 

-QuarantinePurgeItemsAfterDelay

 

Specifies the number of days to keep items in the Quarantine folder. If you specify a value of zero or do not specify a value for this parameter, items stay in the Quarantine folder indefinitely.

 

https://docs.microsoft.com/en-us/powershell/module/defender/set-mppreference?view=win10-ps

 

The documentation is wrong for this setting: The default value is actually 90 days – so you can either modify the schedule or turn off automatic removal by setting the value to 0.

 

Copy, paste, and run (enter) this command at the Administrator PowerShell Prompt:

 

Set-MpPreference -QuarantinePurgeItemsAfterDelay 0

Hello

Unfortunately, if you no longer have an access to the account it's not possible to close the account, You will need to recover the account first then close it after. To recover your account, please answer the verification form: https://account.live.com/acsr

The chances of passing the validation increase if you fill out this form from a computer that was previously used to sign in to that account, and was successful. Also, it is much better if you can do this process from a network location (i.e. home Wi-Fi or work network) where a previous sign-in occurred.

Be sure to use the information you had associated with the account originally, even if there were recent changes. The 4 basic information (First Name, Last Name, Birthdate and Current/Old Passwords are the best and easiest way to prove your Online Identity) And also, please be reminded that you just need to fill out information that you are just 100% sure and in the best of your knowledge.

You don't need to fill out everything but make sure that you have provided enough for you to pass this validation. The 4 basic information is more than enough to prove your identity - as long as it matches the system's record.

As soon as you recovered the account, please follow the steps here to delete your account: https://support.skype.com/en/faq/FA142/how-do-i-close-my-skype-account?q=account+deletion

Let me know if you need further assistance.
