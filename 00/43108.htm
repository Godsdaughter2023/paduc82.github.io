hairyfool say:

You don't want to back up a folder as the backup utility creates a compressed single file which would then not be accessible to other computers.


You can use a command to refresh the network folder with the local contents on a file by file basis but the real problem is the "automatic" bit. You could manually run a script but there is no trigger for the script to run if the network resource becomes available.


The alternative would be consider the master to be the network store not the Surface and sync the folders both ways by date. With standard Windows the solution is Offline folders where you make a network store available even when not connected, unfortunately Windows RT does not support that option. There is one facility which you should look at and that is SkyDrive which is Microsoft's preferred method but the files are stored off your network altogether and depends on the size of the folder as well.

angelmesa say:
Hi Debasish,

When encrypting files and folders, Windows will use a self-generated certificate that contains keys used to encrypt and decrypt the data. When opening encrypted data when logged into the user account that generated the certificate, the decryption process is transparent and the files are opened normally.

However, if another user or system tries to access those same data files or if the files are moved to another location, then they cannot be opened unless the original certificate is installed.

If you do not have a backup of that certificate, you will not be able to decrypt the data. Unfortunately, there is no way around this since the encryption is very strong and cannot be broken easily.

Hi Mustafa,

Thank you for your interest in Windows 10!

You can certainly upgrade your current PC to Windows 10 by going to the Windows 10 Software Download page to download the Media Creation Tool. The Media Creation Tool can be used to create a USB or DVD media to upgrade your current PC to Windows 10 or install Windows 10 on a different PC.

Follow the instructions on Windows 10 Software Download page: http://www.microsoft.com/software-download/windows10

Note:

When upgrading to Windows 10 using media, you must run setup.exe from within your Genuine and Qualified version of Windows 7 or Windows 8. If you want to boot from media, you must have upgraded to Windows 10 previously. Boot from Windows 10 media directly without upgrading to Windows 10 previously is not supported and will have activation issues.

If, however, issues activating your system having upgraded from a genuine Windows copy, then I would suggest you to refer to the information provided in the following article:

"Why can't I activate?"
