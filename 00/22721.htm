A master file is only a collection of links, and indeed removing the link from a master file doesn't remove the subroject from the resouce pool. How to do that depends on whether the subproject still exists. If yes, open the subproject and the resouce pool together, then in the subproject Resource Ribbon, Resource Pool, Share resources, Use own resoruces.

When however the subis deleted and can no longer be opened, open the pool, Resource ribbon, Resource Pool, Share resoruces, select the disappeared project, Break link.
