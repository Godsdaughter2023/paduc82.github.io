a. user say:

Welcome to the Microsoft Windows Vista Communities site.

 

 

Windows Backup and Restore Center…

 

The backup that you’ve described is a Complete PC Backup which is a “snapshot” of your entire computer from a given date and time. This method of backup is intended as disaster recovery in the event that your hard drive or entire computer fails. It is not intended for individual file or folder restore, but rather to restore the entire computer to that date and time. Unless you would like to restore your computer to (2) months ago you will be unable to restore using this method.

 

The Complete PC Backup is intended to restore your programs, system settings, and personal data – all at once. The Back Up Files Wizard is intended to specifically backup user data, and to be run on a more frequent basis (daily/weekly). However, you should include both methods in your backup regiment. See below for a description of the two backup methods. Also, note that the Complete PC Backup is only available in select editions of Windows Vista.

 

Complete PC Backup

This method of backup is supported in Business, Enterprise, and Ultimate editions of Windows Vista. This method is considered an image backup as it takes a “snapshot” of your programs, system settings, and the files on your computer. You can use this backup if your computer ever stops working. Although this type of backup includes your personal files, we recommend that you backup your files regularly using the Back Up Files Wizard.

Backup Files Wizard

This method of backup is supported in the Home Basic, Home Premium, Business, Enterprise, and Ultimate editions of Windows Vista. This method backs-up the following file categories: Pictures, Music, Videos, E-mail, Documents, TV shows, Compressed files, and Additional files.

 

Workaround to allow you access to the VHD…

 

There are “ways” to access the contents of your backup which is stored in VHD (Virtual Hard Disk) format. However, they can be complex for the average user. If you are familiar with the Microsoft Virtual PC 2007 software than you should be able to easily mount this backup VHD as a secondary hard drive. Once you’ve done that you can browse the file structure within the Virtual PC.

 

If you are not familiar with Virtual PC 2007, or do not have it installed, you can use the below method to install VHD Mount from the Microsoft Virtual Server 2005 suite. This utility is a command-line utility that allows you to mount, or temporarily add, a VHD (Virtual Hard Disk) to your computer for access. Once you are done accessing the VHD you can simply run another command to dismount the drive, and discard any changes you made.

 

Please be aware that these steps can be complex for the average user, and I do not encourage you to do them unless you are confident you can. Also, making any direct modifications to the VHD can result in it being unusable for the purposes of a Complete PC Restore – should you need to use this backup for disaster recovery in the future.

 

How to access Windows Vista Complete PC Backups using VHD Mount…

1.     VHD Mount is part of “Microsoft Virtual Server 2005 R2 SP1.

2.     Download Microsoft Virtual Server 2005 R2 SP1 – Enterprise Edition from the following link:

http://www.microsoft.com/downloads/details.aspx?familyid=bc49c7c8-4840-4e67-8dc4-1e6e218acce4&displaylang=en

3.     Install only the VHD Mount tool by following the below steps.

4.     Download the setup package from the above link that is designed for your Operating System (x86 or x64).

·          Once the download is complete click Run twice.

·          Click Install Microsoft Virtual Server 2005 R2 SP1.

·          Click I accept… if you agree to the licensing terms, and then click Next.

·          Click Next again, select Custom, and then click Next again.

·          Click the down arrow next to each feature to be installed, and select This feature will not be enabled.

·          Each feature listed should have a red X next to it.

·          Click the down arrow next to VHD Mount and select This feature will be installed on local hard drive.

·          Click Next, then click Install, and then Finish once setup is complete.

5.     Once VHD Mount has been installed you will need to use the Command Prompt to mount/dismount your VHD.

6.     Click Start, click All Programs, click Accessories, right-click Command Prompt, and select Run as Administrator.

7.     Click on Continue, or provide Administrator credentials as necessary.

8.     Now type the following at the Command Prompt, and press Enter after each line:

·          cd %programfiles%

·          cd “Microsoft Virtual Server”

·          cd vhdmount

·          vhdmount.exe /p “VHD PATH HERE”

Note: You will need to replace VHD PATH HERE with the full path name of your backup VHD.

9.     This will install your VHD backup, and you will be able to browse it as if it where another hard drive.

10.  You can then copy any files you need to recover to your current computer.

11.  Once you have finished with the drive you will need to unmount it.

12.  Open the command prompt again by following (Step 6) and then type the following commands:

·          cd %programfiles%

·          cd “Microsoft Virtual Server”

·          cd vhdmount

·          vhdmount.exe /u /d “VHD PATH HERE”

Note: You will need to replace VHD PATH HERE with the full path name of your backup VHD.

13.  This command will unmount the drive and discard any changes you have made to the VHD backup.

14.  This ensures the integrity of the backup should you need to use it for a Complete PC Restore in the future.
