link text between documents

a. user say:

In one document (A), select the material you want to appear in the other document, and do Edit-Copy (or e.g. ctrl-C)

In the other document (B), do Edit->Paste Special and select "Paste Link" and Word Document object.

Bear in mind that

andy pope say: You will get the triangles if the error bar value is zero or less.

Whilst the actual custom value may not be zero when told to use the value in a minus direction it will cause the triangle if the value becomes invalid.

Here is a simple test, enter formula in A1:B1  and drag down the A4:B4

A1: =10^(ROW())
B1: =A1*2

create column chart on A1:A4 and set scale to log.
Add error bar with custom values referecing B1:B4, you should get black fills running to top left of chart.

change B1 to be =A1/2 and drag down to revert to normal error bars.
