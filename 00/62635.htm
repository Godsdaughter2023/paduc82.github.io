    How to upgrade a 1GB tablet to the anniversary update - Microsoft Community

    cbarnhorst say:

    How to upgrade a 1GB tablet to the anniversary update

    A 1gb tablet can be upgraded to the Anniversary Update (version 1607).  Here is how.  Performance is the same on my tablets as it was prior to the upgrades.

    The problem is not performance or unsupported features.  It is getting installation to work.  The problems are that Windows Update and the Media Creation Tool both use too much memory to permit the installation to complete.  Also, 1GB is not enough to permit the user to keep anything so be sure to back up important files and have access to installation media for apps.

    Microsoft has provided a special .iso download that does install Windows 10 version 1607 on a 1GB device.  You must have sufficient free storage (I think 8GB but possibly only 4GB) as always.

    Download the .iso from here

    https://www.microsoft.com/en-us/software-download/windows10ISO/

    It must be on the C:\ drive to work and you must not have Program Files set to another drive.  Mount the .iso and double-click Setup.exe when the File Explorer window opens.  Of course if you have an external dvd drive you can burn to disc or create a usb and use the "Boot from device" option in the UEFI firmware.  I ran the upgrade from the desktop but it may work as a clean install also.  The result would be a Modern Clean Install in either way.  It is not necessary to disable or remove third-party anti-virus programs first since everything is removed anyway.

    The web page says in pertinent part, "You’ve been routed to this page because the operating system you’re using won’t support the Windows 10 media creation tool and we want to make sure you can download Windows 10."  The link was provided to me by a Microsoft Answer Desk agent so it is a supported procedure.  Keep this link for use after version 1703 releases in the Spring.
