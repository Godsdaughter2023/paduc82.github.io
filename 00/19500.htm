john sr wilson say:
I'm pretty sure that File menu > Options command > Advanced option > Image Size and Quality  doesn't change the export resolution when you save as images. You have to alter the registry. Steve's AddIn will do a better job though.

perpetual office newbie say:

So it sounds like the PPT control for image resolution has no effect on the resolution of images in a slide that is itself saved as an image. That makes no sense, but if that's what PPT does, that's what PPT does.

What if I save the slide as PDF? In that case, would the images in the PDF be at 220 ppi resolution, meaning that if I found a utility to save the PDF as a high-resolution image, I'd get the result I want?

steve rindsberg say:

That's correct. The compression setting determines whether (and if so, by how much) PPT compresses images you've inserted onto a slide when it saves the presentation.

It doesn't control the resolution of images you get when you save a slide or shape AS an image, other than in a backwards way, whereby it limits the maximum resolution you can get back out:  if you insert a high resolution image and let PPT compress it, you'll have thrown away the resolution you might want to get back out.

Better to turn compression OFF.

What you might do instead is to create a new blank presentation, set it's slide size to be exactly the same as the group of shapes that make up your collage, then copy /paste the shapes into the new presentation.

Then you can use a bit of the code in the link Rohn gave you to export the entire slide at higher resolution.

perpetual office newbie say:
Also relevant is this information, which indicates that the maximum dimension of a slide exported as an image in PPT is 3072 pixels.

steve rindsberg say:

Yes, thanks for adding that.  Still, by following some of the other suggestions, you should be able to get exports at more than 1920 pixels wide.

In case very high resolution images are important to you, PowerPoint 2013 and up can export at vastly higher resolutions.  15,000 pixels wide seems to be no problem (but you need to use VBA to get there).

I came across a similar situation setting up one of my Virtual Machines and then trying to join Insider to use it just for Fixes, Drivers etc. 

What happened in my case I was using a program called Windows 10 Manager, and used a setting to totally disable User Account Control. This setting completely shuts down UAC, not like the slider does in windows UAC settings.
