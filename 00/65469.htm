How to fix Win 7 Hybrid drive problem - Microsoft Community

jan boen say:

How to fix Win 7 Hybrid drive problem

This is actually not a question but a solution.

As I've spent days to find a solution and the forums not providing a clear answer for my problem I thought of sharing my problem and how I solved it.

 

Windows 7 - 64 bit, clean install

New hard disk Seagate ST750LX003 (Hybrid drive with SSD and classical disk)

Whole install goes well BUT after the install the following elements are not working:

1- Windows Update

2- Disk indexing / search

3- Search in Outlook (2010)

 

Turns out that the Intel  ICH9M-E/M SATA AHCI Controller driver is not compatible with the new drive which then generates problems with the MS Jet Database and results in above.

After a long quest for a solution I finally disabled the Intel driver and selected the much older Microsoft "Standard AHCI 1.0 Serial ATA Controller" driver.

Reboot PC and hooray all works!

 

Hope this posting saves someone else several hours or even days of effort to find a solution...

a. user say:

Hi,

 

I would suggest you to run the internet connections troubleshooter and check if there is anything wrong with the connection.

Follow these steps to run Internet Connections troubleshooter.

a.      Press Windows key + W and type troubleshooting.

b.      Select troubleshooting.

c.       Select View all on the top left corner.

d.      Run Internet Connections troubleshooter.

 

I would also suggest you to flush the DNS cache and check. When you type a web address into your web browser and press ENTER, you are sending a query to a DNS server. If the query is successful, the website you want opens; if not, you'll see an error message. A record of these successful and unsuccessful queries is stored in a temporary storage location on your computer called the DNS cache. DNS always checks the cache before querying any DNS server, and if a record is found that matches the query, DNS uses that record instead of querying the server. This makes queries faster and decreases network and Internet traffic. So, I would suggest you to flush, release and renew the DNS cache and check for the issue.

Follow these steps to do so.

a. Press Windows key + X and select Command Prompt (Admin).

b. Type the following commands and hit enter after each of them.

ipconfig /flushdns

ipconfig /release

ipconfig /renew
