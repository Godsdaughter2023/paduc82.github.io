Add/Remove programs does not entirely remove a program-I search and find files left in "prefetch"...how may I COMPLETELY remove a program and any association of it from my system-as if it never existed?

lemp say:

Add or Remove Programs (or an application's uninstall routine) typically remove enough so that what's left won't cause problems.

Sometimes, the uninstall deliberately leaves the application's data files and/or customization information behind so that you can, for example, re-install the application or a newer version of it and pick up where you left off.  Apple's iTunes is a good example of this: uninstalling iTunes does NOT remove all your music files -- which for most people is a very good thing.

If you are concerned about space (or about leaving traces of a program that you don't want others to know that you had used), you can always go to C:\Program Files, find the folder that corresponds to the now-removed application, and delete it.

The prefetch folder is something you should leave alone. Windows uses this to speed up the loading of applications. The *.pf files are NOT generated by the application. Rather, they are created by Windows.  The prefetch folder is self-maintaining, which means that after a while, any *.pf files that relate to an uninstalled application will automatically be deleted.

Aside from folders in C:\Program Files and any entries in the Prefetch folder, there are likely to be traces of an application scattered throughout the file system and in the Windows registry.  You can search for and delete files and folders, but you may end up deleting something that only looks like it belonged to the uninstalled application.  The same is even more so in the case of the Windows registry.  I really suggest that you NOT attempt to "weed out" registry entries.

The only method guaranteed to remove all traces of an application, aside from a clean reinstall of Windows, is to make a disk image BEFORE you install the application and then restore that image when you no longer want the application.  Of course, you'll lose any other changes you made after you created the disk image, but that's a price you'll have to pay to get the results you want.
