If you have a Router (or Modem/Router) connect the switch to the Router and then all the computers can go into the Switch forgoing the need for double NAT (ICS).  The minimum included Norton 2010 can be the core of the problem.

Home Network and Homegroup are two different things.

Homegroup is based on IPv6 and works only between Win 7 computers. Home Network, and Work Network, work between all Windows client OS’ (my personal preference is Work Network).

The issue is not changing the names "Unidentified Network" and is classified as "Public". These are a result of wrong network configuration and will go away once you are configured correctly.

HomeGroup Network = Works only between Win 7 computers. This type of configuration makes it very easy to Entry Level Users to start Network sharing.

 Work Network = Basically similar to the previous methods of sharing that let you control what, how, and to whom folders would be shared with.
