vijay a. verma say:
When you upgrade from one version to another version of Windows or also within a version you install a major upgrade say Fall Creator's update, Windows.old folder is created. This contains necessary files and folders which can help you to restore to previous version of Windows if you need.

This consumes space on your hard drive. This Windows.old folders clears off within 10 days or 30 days depending upon the type of update.

In case, you don't want to go back to previous version of Windows, you can delete it. Rather I would recommend that you can archive this to some external backup drive.

You can execute following to gain space on your computer

1. Use Storage Sense to delete unnecessary files
Type Storage in Windows Start Search Box > Click Storage > Click Change how we free up space > Down below there is button Clean now under Free up space now which you should press

2. Use Disk Cleanup (You have already tried this)
Type Disk Cleanup in Windows Start Search box > Click Disk Cleanup > Check relevant options (you can also clean system files)

3. Analyze disk space
You should analyze your existing space with help of third party softwares and remove un-necessary files. TreeSize is most popular one but my personal favorite is WinDirStat.

TreeSize Freeware version - https://www.jam-software.com/treesize_free/

WinDirStat (Released under GPL license) - https://windirstat.net/download.html

3. Check for log files in CBS directory
Go to C:\Windows\Logs\CBS > Check the size of CBS.Log and if size is huge >
Type services in Windows Start Search box > Click Services > Scroll down to find the Windows Modules Installer service and double-click on it > Set its startup type to Manual > Reboot your computer

Go to C:\Windows\Logs\CBS > Delete all of the files in the CBS folder (or rename and move to a backup drive) > Reboot your computer > Then go back into the Services App and set Windows Modules Installer Service back to Automatic Startup (Note CBS log files contain information about windows update.

4. Microsoft has also some suggestions to improve disk space.

https://support.microsoft.com/en-us/help/12425/...

We appreciate your response. To further isolate your concern, we suggest manually updating the USB drivers on your device using the hidden Administrator account. After updating the drivers, check if the issue persist on hidden Administrator account before checking it on your original User Profile.

To enable the hidden Administrator account, follow these steps:

  1. On Search type CMD to open Command Prompt. Right click on CMD then choose Run as administrator. If you are prompted for an administrator password or for a confirmation, type the password, or click Yes.
  2. In the Command Prompt window, type net user administrator /active:yes then hit the Enter key.
  3. You should see a message that the command completed successfully. Log out, and you’ll now see the Administrator account as a choice on the login screen.

Note: The hidden Administrator account has a lot more privileges than a regular administrator account. Privileges that can easily generate a conflict on your system. We only recommend enabling the built-in Administrator account for troubleshooting purposes and you have to disable it once you are done.

To disable the hidden Administrator account, follow these steps:

  1. On Search type CMD to open Command Prompt. Right click on CMD then choose Run as administrator. If you are prompted for an administrator password or for a confirmation, type the password, or click Yes.
  2. In the Command Prompt window, type net user administrator /active:no then hit the Enter key.
  3. The administrator account will now be disabled, and shouldn’t show up on the login screen anymore.
