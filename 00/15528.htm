daniel g. grau say:
Jacky, what's happening is this: your text is defined as "U.S. English." Note that language is not a setting or preference in Word, but rather a text attribute, just like bold, font and color. Therefore, you don't choose the working language, you need to apply it to the selected text.

If the text in your document is already defined as "U.S. English," selecting "U.K. English" in Tools > Language… will only apply the language change at the cursor's position (or to the selected text). So select the text and apply the language.

There is also an additional language attribute: "Do not check spelling or grammar," which is applied in addition to the language. You've seen the checkmark in the Language dialog. 

For good measure, go to Format > Style, click on Normal in the list, then on the Modify button. There will be a dropdown menu at the bottom left—select Language and define the proper language for the Normal style. This redefinition will only affect your current document. If you want all your new documents to be in U.K. English, select Tools > Language, make a language selection and click on the Default button.

As for not being able to add a new word to a dictionary, this is probably what's happening: Each custom dictionary can have a language associated to it (when you see the list of dictionaries, there is a dropdown menu for this). If the language of the currently-selected custom dictionary does not match that of the text being spell checked, the Add button is grayed (logically Word won't let you add a French word to a Dutch custom dictionary). Thus, you need to make sure that the custom dictionary's language matches the language of your document.

Choosing None as the language makes Word use the dictionary regardless of language (i.e., always).

And of course you know that the squiggly red underline is a consequence of setting Word > Preferences > Spelling and Grammar > Check spelling as you type.

brian tillman [outlook mvp 2007-2019] say:

There is no such thing as "Outlook Expresses 2010".  Sounds like you are using Outlook 2010, since you are using Windows 7 and there is no Outlook Express on Windows 7.

Open your  Contacts and then click View>Change View>List.  You'll see a list of your Contacts and one of the fields will be "Country/Region".  Right-click the header for that column and choose "Group By This Field".  This will give you one of more groupings.  One will be "none" and the others will be whatever Country/Region values you have entered.

Select whatever entries in the "none" group you want changed to, say, "United States of America", then drag the selection to that group.  They'll all change at once.  Do the same for the contacts you wish to put in the other groups you have.  If you do not yet have a Country/Region value group, manually change any contact you want to be in that group, then you can change the rest with the select and drag method.  WHen you're done, you can change the view back to one of th card views, if that's what you had been using.
