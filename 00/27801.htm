    Dim strDecimalSeparator As String

j.e. mcgimpsey say: One way is use Offset() or Index(). For instance, if your table is in B2:F20, you could use

=INDEX(Sheet1!$B$2, 3, 2)

to reference the 3rd, row, 2nd column (i.e., cell C4) even if you insert additional rows above row 4.

To help with consistency, use a defined name (Insert/Name/Define, and assuming you have a header in B1, hence the -1 after COUNTA):

  Name in workbook:   Table
  Refers to:                     =OFFSET(Sheet1!$B$2, 0, 0, COUNTA(Sheet1!$B:$B)-1, 5)

You can then refer to rows and columns of the table using the defined name. For instance:

  =INDEX(Table, 1,1)

returns the value in B2.

   =SUM(INDEX(Table,3,))

returns the sum of the third row of the table, while

   =AVERAGE(INDEX(Table,,2))

returns the average of the 2nd column.

Resetting back to the default location (/Users/<username>/Library/Group Containers/Microsoft/Office/User Content/Templates) in Word's preferences (prefs -> File Locations -> User templates) fixes the problem.

Hi Vanice,

If you are trying to move a list from Office 365 group to classic SharePoint site, then you can save the list as the template from Office 365 and upload it in the SharePoint site.

For this, first you must enable scripting in the Office 365 group to use list templates. For more info on how to enable scripting, see section To allow custom script on other SharePoint site collections in Turn scripting capabilities on or off.

Once you’ve enabled the custom script, please follow the steps from this article under the section Online,2016,2013 to know how to Create a list template, download a template and Upload a template to the list template gallery in SharePoint Online.

After following the last step from the article i.e. Upload a template to the list template gallery, in SharePoint Online, go to the site, click Settings>Site Content> New>App.

In the search box, type the name of the list template which you saved and from the search result click on the list template. It’ll ask you to provide a new name. After giving the name, you can use this list.

Please note, if you are trying to move the list from one Office 365 group to another, then this feature is currently not available.

To move the document library, use Copy to feature. It may take time to move the files depending on the size.

  1. Select the items that you want to copy, and then click Copy to.
  2. Under Choose a destination, select the location where you want a copy of the files, folders, or links to go. If you're copying to a different site or subsite and don't see it listed, click Browse sites to see the full list of sites that you can copy to.
