amit tandon say:

A vba solution is suggested because you need a sum of cells adjacent to the selection. Enter below code in the sheet module ie. in the code module of the sheet wherein your data is present.

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
'on selection of any non-numeric cell in the Target range (A1:A10), display the sum of adjacent cells in column B

On Error GoTo ErrHandler
Application.EnableEvents = False

'if selected cell(s) selected is NOT in Range of A1:A10, then exit sub
If Application.Intersect(Target, Range("A1:A10")) Is Nothing Then GoTo ErrHandler

'----------------------------
'displays the sum of adjacent cells in column B - does not distinguish between  numeric / non-numeric selection
'MsgBox Application.Sum(Target.Offset(, 1))
'the sum of adjacent cells in column B is entered in cell D1
'Range("D1").Value = Application.Sum(Target.Offset(, 1))
'----------------------------

Dim lSum As Long
lSum = 0
For Each cell In Target
    'if selected cell value is numeric, exclude from total
    If Not IsNumeric(cell) Then
        lSum = cell.Offset(0, 1) + lSum
    End If
Next

'displays the sum of adjacent cells in column B
MsgBox lSum
'the sum of adjacent cells in column B is entered in cell D1
Range("D1").Value = lSum

ErrHandler:
    Application.EnableEvents = True

End Sub

Refer below image:

If you select a cell(s) in the range A1:A10, the sum of adjacent cells in column B will be entered in cell D1. Note that this code will sum the adjacent cells only for selected cells which are non-numeric. So in the above image, cell D1 returns the sum of cells B3:B5 only,

In case you wish to sum all adjacent cells irrespective whether the selected cells are text or numeric, the code has been given above itself, though commented out:

'----------------------------
'displays the sum of adjacent cells in column B - does not distinguish between  numeric / non-numeric selection
MsgBox Application.Sum(Target.Offset(, 1))
'the sum of adjacent cells in column B is entered in cell D1
Range("D1").Value = Application.Sum(Target.Offset(, 1))
'----------------------------
