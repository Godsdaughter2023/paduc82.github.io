Powerpoint 2016 für Mac – Hyperlink Farbe ändern: Wie?

Wie ändere ich die Hyperlink Farbe von Text in Powerpoint für Mac (Office 2016)?

john korchok say:

You can set the color for hyperlinks using the following steps:

  1. Choose the View tab on the Ribbon and select Slide Master.
  2. On the Slide Master tab, click on Colors, then on Customize Colors.
  3. Click on the color square beside Hyperlink and set your preferred color.
  4. Give the Color Theme a name and click on the Save button.
  5. After saving the Color Theme, you can apply it to other presentations and the hyperlink color will be changed.

Sie können die Farbe für Hyperlinks legen Sie die folgenden Schritte aus:

  1. Wählen Sie die Registerkarte Ansicht auf dem Band, und wählen Sie Folienmaster.
  2. Auf der Folienmaster Registerkarte, klicken Sie auf Farben, dann auf Farben anpassen.
  3. Klicken Sie auf das Farbquadrat neben Link und stellen Sie Ihre bevorzugte Farbe.
  4. Geben Sie dem Farbthema einen Namen und klicken Sie auf die Schaltfläche Speichern.
  5. Nach dem Farbthema speichern, können Sie es zu anderen Präsentationen und der Hyperlink Farbe wird geändert gelten.
