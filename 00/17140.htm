dale howard [mvp] say:

Pete213 --

In Microsoft Project, the definition of any view is as follows:

View = Table + Filter + Group + Screen

Notice that the Outline level is not a part of the definition of any view.  Because of this, you cannot specify that the Resource Usage view always show the resources, with their task assignments collapsed.  In each individual project, however, you can use the Outline button ribbon to collapse all of the assignments .  Apply the Resource Usage view and then click the Select All button to select every resource and task assignment.  In the Data section of the View ribbon, click Outline >  Hide Subtasks to collapse each resource's task assignments so that you see only the resources.

Ref. to Mail-Merge in Word 365 - Microsoft Community

bobwescott say:

Ref. to Mail-Merge in Word 365

Is it possible to mail merge for desktop printing so that one letter and one envelope print for each record, without running letters and envelopes separately?
suzanne s. barnhill say:

Whether this is possible depends more on your printer than on Word. Assuming your printer has a tray that feeds Letter-sized paper and another that is used for envelopes, try this: Create a dummy letter, then go to Mailings | Create | Envelope, create a dummy envelope, and click Add to Document. This will give you a letter (Section 2, page 1) with an envelope attached as Section 1, page 0. For each section, go to the Paper tab of Page Setup (the easiest way to access it is to double-click at the top of the horizontal ruler or anywhere on the vertical ruler) and set the correct tray for "This section."* Then try printing. If the "letter" and envelope print from the correct trays, then you should be able to print the merge that way.

So create your merge letter, add an envelope as you did before (but without content), and copy the name and address mergefields onto the envelope. You should then be able to merge to a new document that will alternate envelopes and letters.

Personally, I would find it a lot simpler (and more versatile) to create separate letter and envelope merges from the same data source. Admittedly, this would require collating the letters and envelopes, but some manipulation will be required in any case if you have to fold and stuff.

*Note this caveat. The Envelope Address paragraph in an envelope is in a frame anchored to the Address Return paragraph, which is above the section break. In order to set the paper tray for the envelope, you have to click in the return address when opening the Page Setup dialog.

Hi Lucian,

To clarify, at least once every 30 days, Office 365 ProPlus needs to connect to internet, communicates with the license/activation service to extend the product key to keep the Office application full functional. If there is no connection for over 30 days or your subscription ends, the Office application will enter reduced functionality mode after grace period ends.

Answers:
1. The “remaining grace: 31 days” means that the Office application are active for 31 days and you are required to connect your computer to internet to extend the product key within 31 days period.
2. Yes, if you don’t connect the computer to internet in 31 days, the activation will expire and the Office application will goes to reduced functionality mode until the computer has been connected to the internet for license verification.

Thus, we recommend you connect the computer to internet at least once every 30 days to keep the Office application full functionality.
