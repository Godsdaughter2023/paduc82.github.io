Hi,

It Is possible only with functions as:

Function myMin(myString)
Dim ARR() As String
Dim i As Integer
   myMin = "2050"
   ARR = Split(myString, "(")
   For i = 0 To UBound(ARR)
       If Left(ARR(i), 4) >= "1900" And _
          Left(ARR(i), 4) <= "2050" Then
          If myMin > Left(ARR(i), 4) Then
             myMin = Left(ARR(i), 4)
          End If
       End If
   Next
End Function

Function myMax(myString)
Dim ARR() As String
Dim i As Integer
   myMax = "1900"
   ARR = Split(myString, "(")
   For i = 0 To UBound(ARR)
       If Left(ARR(i), 4) >= "1900" And _
          Left(ARR(i), 4) <= "2050" Then
          If myMax < Left(ARR(i), 4) Then
             myMax = Left(ARR(i), 4)
          End If
       End If
   Next
End Function

scottgem say:

Actually Mimmo has a good idea there. 

To explain what his code does, it loops through the code looking for strings of 4 digits that fall within the values 1900-2050 (you may need to adjust that range for your data). When it find a string that fits within that range it writes the value to a temporary variable. it then compares that variable to the upper end of the range and if its is less, then it replaces it as the new value. So the end result would be the highest or lowest value. 

The only issue is if there is a number within the field that falls within the range but is not a year.
