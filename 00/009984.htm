3D Builder is a provisioned app. If you want to get rid of it completely, you not only have to de-install it from your user account (which, for example, is all what CCleaner does), but also remove it from the list of provisioned apps.

Unless you are somewhat familiar with PowerShell and Windows Image Maintenance, I suggest you take a look at this: Windows App Boss .

Be aware, though, that with every new Windows build, these provisioned apps are back.

Not sure whether this still works with the latest build, but: If you are of the adventurous type, here's a PowerShell script that should get you started (use at you own risk!):

$AppsList =
 "Microsoft.3DBuilder",
 "Microsoft.BingWeather",
 "Microsoft.Getstarted",
 "Microsoft.MicrosoftOfficeHub",
 "Microsoft.WindowsPhone",
 "Microsoft.XboxApp",
 "Microsoft.MicrosoftSolitaireCollection"

foreach ($App in $AppsList)
{
 $Packages = Get-AppxPackage | Where-Object {$_.Name -eq $App}
 if ($Packages -ne $null)
 {
  "Removing Appx Package: $App"
  foreach ($Package in $Packages) { Remove-AppxPackage -package $Package.PackageFullName }
 }
 else { "Can't find AppxPackage: $App" }

 if ($ProvisionedPackage -ne $null)
 {
  "Removing Appx Provisioned Package: $App"
  remove-AppxProvisionedPackage -online -packagename $ProvisionedPackage.PackageName
 }
 else { "Can't find AppxProvisionedPackage: $App" }
}

Note that the script tries to remove both you personal copy of an app and the provisioned version of that app. Also note, that this script works on your currently running Windows image (the "-online" in those get- and remove- commands); it can easily be modified to work with an offline Windows image, too.
