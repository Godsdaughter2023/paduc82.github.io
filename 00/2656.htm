OneNote cannot sync this notebook because the network is disconnected - Microsoft Community

varunsrin say:

OneNote cannot sync this notebook because the network is disconnected or the network location is not available. ( Error code: 0xE4010641 )

Your notebook is most likely stored on SkyDrive, SharePoint or a network location, and OneNote is unable to access this location. This could happen for any of the following reasons:

  • Your network or internet connection is disconnected
  • The server hosting your notebooks has gone offline temporarily – if you were storing your notebooks on a SharePoint server or a network share, someone may have turned the machine off or taken it down for maintenance. Check to see if you can access the root address of the server. For instance, if your notebook is located at \\contoso\documents\teams\research\onenote, check to see if \\contoso\ is available. If it isn’t, talk to your system administrator.
  • Your notebook no longer exists at the network location. This could happen if you (or someone else with permissions) deleted the notebook from the server. If your notebook is on SkyDrive, go to skydrive.com and login to check if your notebook still exists.  If you are unsure about where your notebook is located, you can go to File -> Info to find the URL to your notebook.  
  • Occasionally the disconnected error can be caused by intermittent server issues. Try syncing again after a few minutes by hitting Shift + F9, or selecting Sync Now from Notebook Sync Status.

joeu2004 say:

To simply display 1234 as 12:34, use the Custom cell format "00\:00" without quotes.

To interpret 1234 as numeric time 12:34 in time calculations and convert to decimal minutes, use the following formula:

=ROUND( ( TEXT(E11,"00\:00") - TEXT(C11, "00\:00") + (E11 <= C11) ) * 1440, 0)

The term "(E11 <= C11)" handles the case where the times span midnight.

It is prudent to explicitly round the calculation in order to avoid infinitesimal anomalies that arise from the internal binary arithmetic.  The point is:  hh:mm cannot be represented exactly internally.
