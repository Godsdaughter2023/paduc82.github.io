Sub UnderstandingVariants()
    On Error Resume Next
    Dim X As Variant
    X = ActiveCell.Offset(2, 0).Resize(4).Value
    ActiveCell.Offset(2, 1).Resize(4).Value = X
    MsgBox X.Value
    MsgBox X(1).Value
End Sub

per jessen say:

A one dimensional array will always consist of one row of data.

As your data is in a column your macro is creating a two dimensional array (one column with 4 rows).

 

To display the first element in the array use: MsgBox X(1,1) 'you don't need the value part, as you have only loaded values to the array.

 

To display the second element use: MsgBox X(2,1)

 

Íf you want to create a one dimensional array from your date you has to use:

 

X = WorksheetFunction.Transpose(ActiveCell.Offset(2, 0).Resize(4).Value)

MsgBox X(1)

shift f2 textbox zoom not working with Access 2007 runtime - Microsoft Community

douglev say:

shift f2 textbox zoom not working with Access 2007 runtime

I recently upgraded my Access 2002 application to Access 2007.  When running it with Access 2007 runtime, the shift f2 textbox zoom capability does not work.  It works when you run it with the full Access 2007.

kathy s - microsoft support say:

Hi Doug,

Unfortunately, the zoom box functionality is contained in one of the wizard files (utility.accda). Wizard files are not redistributable and are only meant to be used with full versions of Access.

You could build your own Zoom control with AutoKeys to call F2 (Thanks Michael!).

'// Private implementation of the Zoom Builder
'//
'// 1) Create an AutoKeys macro with a submacro named +{F2}.  This traps the Shift+F2 key stroke
'// 2) The submacro calls a public function (ex: InvokeZoomBuilder) in a standard module that:
'//       a) Builds a delimited string that contains the name of the activeform (i.e. Form1), the active control (Control1), and the value (Hello, World) of the active control
'//       b) Calls a function to invoke the builder form.  The function performs an OpenForm on the builder form, passing the delimited string in the OpenArgs argument


            '// Builds a semi-colon delimited list to pass to a zoom builder form
            '// First item in list:  Name of the active form
            '// Second item in list: Name of the active control
            '// Third item in list:  Contents of the active control
            Public Function InvokeZoomBuilder()

            Dim strArg As String
   
                  strArg = Screen.ActiveForm.Name & ";" & Screen.ActiveControl.Name & ";" & Screen.ActiveControl
                  DoCmd.OpenForm "ZoomForm", acNormal, , , acFormEdit, acDialog, strArg
   
            End Function

'// 3) Create a builder form.  In the OnLoad event of the builder form:
'//       c) Use the Split function to parse the OpenArgs property into a string array.  item(0) = "Form1", item(1) = "Control1", item(2) = "Hello, world".
'//       d) Copy the value of the third string array element (item(2)) into the text control on the builder form.

            '// Parse the openargs argument.  We expect a semi-colon delimited list of three items.
            Private Sub Form_Load()

                  If Nz(Me.OpenArgs, "") <> vbNullString Then
                     strArgs = Split(Me.OpenArgs, ";")
          
                     '// ensure we have a three-item list in the OpenArgs string
                     If LBound(strArgs) <> 0 Or UBound(strArgs) <> 2 Then
                            MsgBox "Invalid openargs string: " & Me.OpenArgs
                            Exit Sub
                     End If
       
                     '// Copy the contents of the third item to the zoom control text box on this form
                     Me!ZoomControl = strArgs(2)
       
                  End If
   
             End Sub


'// 4) Create cancel and OK buttons on the builder form.  Cancel merely closes the form.  OK copies the value of the
'//    text control on the builder form to Forms(arg(0)).Controls(arg(1)), and then closes the form. For example, if the user had changed the
'//    value of the text control from "Hello, World"  to "Good night, world", the text "Good night, world" would
'//    be copied into the text control at Forms("Form1").Controls("Control1").

            Private Sub cmdOk_Click()
                 ReturnValue
                 DoCmd.Close acForm, Me.Name
            End Sub

            Private Sub cmdCancel_Click()
                 DoCmd.Close acForm, Me.Name
            End Sub

            Private Sub ReturnValue()
                On Error Resume Next
                Forms(strArgs(0)).Controls(strArgs(1)) = Me!ZoomControl
                On Error GoTo 0    
            End Sub

Hope this helps!

bigeye say:

Thanks for your input.  Late last nite I found the solution myself.

As I said earlier it seems that if you change the value for Shell Folders Favorites to point to your network location.  When you launch IE8 that value gets changed back.  I found that what seemed to be changing it back was the value set for User Shell Folders Favorites.  So I achieved success by doing the following:

 

Launch the registry editor (regedit)

At the following location:

HKEY_CURRENT_USERS\Software\Microsoff\Windows\CurrentVersion\Explorer\User Shell Folders

Change the value for Favorites form %USERPROFILE%\Favorites to the network location of your favorites

At the following location:

HKEY_USERS\S-1-5-21-1708537768-1677128483-1343024091-1003\Software\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders

 

Change the value for Favorits from %USERPROFILE%\Favorites to the network location of your favorites

¡firedog say:
You don't need to have access to a Hotmail account to be able to use the same email address as the username for Facebook. Once Facebook has established that the address is genuine when the Facebook account is opened, there is no further need for the Hotmail account. The password for the Facebook account doesn't change simply because the password changes for the Hotmail account.

A Hotmail account will lapse if the account holder doesn't sign in at least once every 270 days. Once a year has passed since the last sign-in, the account will be closed and the address will be available for someone else to use. It won't be possible to create another Facebook account using that address, though.

The page number itself is the part that gives you 20, 21, 22, etc. You'll need, at least temporarily, to set this to start at 20. The II is the part that must be a "chapter number."
