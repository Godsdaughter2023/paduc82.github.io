hansv mvp say:

SourceType:=xlConsolidation is used if you want to create a pivot table from identically sized ranges on multiple sheets.

For a 'standard' pivot table, based on a single range, use SourceType:=xlDatabase. You can then use

SourceData:=Worksheets("Sheet1").Range("H4:BE36")

or

SourceData:=Worksheets("Sheet1").Range("H4").CurrentRegion

If you want the source range to be dynamic, make it a table: select the range, click Table on the Insert tab of the ribbon). Note the name of the table - it is listed and can be changed in the Design tab of the ribbon when the table is selected.

Then use

SourceData:=Worksheets("Sheet1").ListObjects("Table1")

If you add rows to the table or remove them, then refresh the pivot table, it will take the added/removed rows into account.

how to duplicate business cards in word 2011 for Mac

I've created a business card in a Word 10 card template. How do I duplicate that design throughout the other nine panes of the template? Have tried copy and paste but nothing works. 
daniel g. grau say:
It's very difficult, unless all elements are totally contained by that first cell. Do this:

  • Click on the cell
  • Do Table > Select > Cell
  • Copy to the clipboard
  • Go to the end of the document, insert a page break (Insert > Break > Page break) to make sure things don't mutually interfere. [You can do this into a new document, but the format may not look right if you've customized your styles. Try it, anyway.]
  • Paste the cell at the end, in its own page
  • Add any missing elements, always keeping your cursor within that single cell
  • Repeat the "select cell, copy and paste" checking procedure until you know all elements are being placed in the clipboard when you copy the cell.
  • Finally, go back to the original table
  • Do Table > Select > Table
  • Paste the contents of the clipboard (it will paste in all cells)
