I too had this problem on Windows 8, and got to this as the root cause. 

When you launch the app and it fails, run the following process :

 

From a Command Prompt : eventvwr.exe

From Windows 8 Metro : View Event Logs

On the right hand frame, you'll see a list of events, and hopefully, one of them will be an Error Level with a source of "Application Error"

If you want to encrypt a file or folder using Encrypting File System (EFS), but the Advanced button isn't displayed on the Properties dialog box or the Encrypt contents to secure data check box is grayed out, it could be because of one of the following reasons:

·         You don't have an edition of Windows that supports EFS. You can use EFS to encrypt files on Windows 7 Professional, Windows 7 Ultimate, and Windows 7 Enterprise.
