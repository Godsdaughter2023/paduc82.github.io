Hi there,

Run The Internet Explorer Performance Troubleshooter

If your computer is having problems browsing the web with Internet Explorer, try running the Internet Explorer Performance troubleshooter to fix the problem. It checks for common issues, such as whether you have enough space on your computer's hard disk to store temporary Internet files.

1 Open the Start Menu and select Control Panel.

2. Click in the search bar and type in Troubleshooting.

3. On the left panel, select View All.

4. Select the icon saying Internet Explorer Performance.

Open Internet Explorer in Safe Mode.

If the problem still occurs after you attempt the previous suggestion, an add-on or a toolbar may be causing the problem in Internet Explorer. Run Internet Explorer 7 in "No Add-Ons" mode. For more information about how to do  this look at the instructions below.

1. Press Windows Key + R.

2. Type in iexplore -extoff.

If the issue doesn't occur in safe mode, it will be a problem with one of the add-ons you have installed on Internet Explorer and you will have to disable them and re-enable them one by one to see which add-on is the cause of the problem.

You could also run the troubleshooter to fix Internet Explorer add-on problems when IE hangs or freezes, here's the link:

http://support.microsoft.com/mats/ie_freezes_or_crashes

Reset Internet Explorer settings: http://support.microsoft.com/kb/923737/en-gb
