You are going to have to re-install Windows 7.

http://www.canadiantech.info/for-techies/re-build-windows-7/

If you are going to do a re-install, you may be able to take your data off first.  Remove the hard drive and put it into a USB external drive housing.  Connect to a working PC, copy the data off.

Always check to ensure the hard drive you are about to rebuild is actually sound from a hardware perspective.  Download and run your hard drive manufacturer’s drive testing software.

First off, you want to try a System Repair disk to boot the PC.  It may be more successful in fixing and offer other alternatives.  If you do not have one, go to any other PC that has windows 7 with the same bitness (32/64).  Start globe, All Programs, Maintenance.  You need just a blank CD.  When you boot, ensure that you are in fact booting on that CD and not the hard drive.  Usually pressing F12 will lead you to the option to do so.

If that does not work, you are likely into a re-install.  Takes a lot of time, but not all attended time.

  1. Factory restore — replaces everything on the hard drive with exactly what it had on it when you bought it
  2. “Upgrade” install using the Win7 install disk.  You start as if you are re-installing from scratch but when it gets to the point where it offers the Upgrade choice you choose that.
  3. Full Clean install.  The only absolutely sure way to have a perfectly clean system.

You will need a Win7 install disk for 2 or 3.  You will need a legal product code for 3 and the win7 disk must match the edition of the code and its bitness (32 or 64).  If you do not have the original Windows 7 install disk, borrow one from a friend.

Mouse freezes, then jumps; keyboard sometimes gets stuck. - Microsoft Community

rapidstrike say:

Mouse freezes, then jumps; keyboard sometimes gets stuck.

Hello,

When I run something that generally requires at least a medium amount of CPU power (i.e. 2006-2008 era video games or a web browser with multiple tabs open), I've noticed that, almost frequently, two things would happen concurrently:

  • Mouse freezes in place for around 2 seconds before it jumps to where it should. It would not respond to mouse movements or clicks. If it freezes right in the middle of clicking, it would repeat that click action.
  • When typing something, the keyboard would stop printing characters for around the same time (2 seconds) before it all jumps onto the screen. If it freezes right in the middle of holding a key down or typing, it would spam that key until I tap it again.

I've tried multiple things to resolve this issue, including:

  • Updating my drivers
  • Trying different USB ports
  • Trying a different keyboard and mouse
  • Unplugging other USB peripherals
  • Running the programs in compatibility mode (This wouldn't actually solve the problem since it also happens outside of the high-CPU applications, but it was worth a shot)

For reference, here are my specs:

  • Intel Core i7-5820K 3.3GHz
  • MSI X99S SLI Plus ATX LGA2011-3 Motherboard
  • Mushkin Blackline 16GB (4 x 4GB) DDR4-2133 Memory
  • Sapphire Radeon R9 290X 4GB Tr-X OC Video Card
  • Rosewill 750W 80+ Gold Certified Fully-Modular ATX Power Supply
  • Zowie EC1 eVo Gaming Mouse
  • Corsair Vengeance K70 Keyboard

I can't tell if this is a motherboard issue, a peripheral hardware issue, or a Windows 8.1 issue. It's been bothering me ever since I built this computer 2 years ago, and it's come to a point where it's almost unusable. Any solutions?

rapidstrike say:

Turns out it was a motherboard issue, most likely related to the USB Controller; upgraded to an MSI Z270-A and the mouse/keyboard freezing completely disappeared.

This was not a Windows issue after all, but this question may serve to help someone diagnose a problem in the future.
