lemp say:

You've got several different issues packed into your question.

For the "Start" command, the /min and /max parameters refer to the behavior of the Command Prompt window that opens when you run the Start command.  It doesn't apply to a program like Media Player. 

The Start Command "Starts a separate Command Prompt window to run a specified program or command. "  https://technet.microsoft.com/en-us/library/cc770297.aspx

As far as the "/close" parameter is concerned, that option seems to have been available for Windows Media Player 6.4 (see https://support.microsoft.com/kb/241422/en-us?wa=wsignin1.0) but the options for WMP have changed and "/close" is no longer available.  See https://msdn.microsoft.com/en-us/library/dd562624%28VS.85%29.aspx.  The "/play" parameter is also gone, but the command worked because the new syntax to play a file is just

     wmplayer "c:\filename.wma"

If you want Media Player to open minimized, you should create a shortcut to Media Player, configure the shortcut to open Media Player minimized, and then run the shortcut.
