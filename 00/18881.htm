Although it could be accomplished in the Open or Save As dialogs of any Office programs, that isn't really a convenient or appropriate tool. Instead, use the File Explorer (which you'll find on the Windows 10 Start menu).

If you can plug in both drives at the same time, you can simply drag the files from one drive to the other. The default action for dragging between drives is Copy.

If you have only one USB port available, plug in the source drive and drag the files from it to a new folder on the hard drive.
