if you are using Business Essential, premium, E1 or Exchange plans you need to purchase Azure Information protection then follow the steps below.

 If you are on E3 or E5 follow the steps below

- Look for the Microsoft Azure Information protection 

- Click the manage Microsoft Azure Information protection 

- CLick Activate

Afterthat

- Install MSOnline Sign-Assistant 

- Then on your computer look for Windows Powershell and run as admin 

- run the ff commands one by one:

  • Set-ExecutionPolicy RemoteSigned
  • $credential = Get-Credential
  • $exchangeSession = New-PSSession -ConfigurationName Microsoft.Exchange -ConnectionUri "https://outlook.office365.com/powershell-liveid/" -Credential $credential -Authentication "Basic" -AllowRedirection
  • Import-PSSession $exchangeSession -DisableNameChecking
  • Set-IRMConfiguration –RMSOnlineKeySharingLocation "https://sp-rms.na.aadrm.com/TenantManagement/ServicePartner.svc"
  • Import-RMSTrustedPublishingDomain -RMSOnline -name "RMS Online"
  • Set-IRMConfiguration -InternalLicensingEnabled $true

- After the powershell, go back to  your Exchange Admin center and go to Mailflow 

- Create a RULE and click first the "More Options" then set the condition that you want (example is if the subject contains "Confidential")
