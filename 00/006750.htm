Ok, now I think I know what you want.

You want a dynamic hyperlink formula (I think!!!).

Let's assume you want the first hyperlink in cell A2 and want to copy down the column.

Enter this formula in A2 and copy down as needed:

=HYPERLINK("#Sheet1!E"&ROWS(A$2:A2)*20,"Print Page "&ROWS(A$2:A2))

  • A2 will be a link named Print Page 1 and link to Sheet1!E20
  • A3 will be a link named Print Page 2 and link to Sheet1!E40
  • A4 will be a link named Print Page 3 and link to Sheet1!E60
  • A5 will be a link named Print Page 4 and link to Sheet1!E80
  • etc
