amit tandon say:

Below formula will return "Approved" if A2 finds a match in F2:F15 and the match in F2:F15 has case sensitive "End" as last 3 letters. The other text need not be a case sensitive match so that "CatherinedesignerEnd" in F2:F5 will also return "Approved". In this formula cell A2 last 3 letters need not be case-sensitive "End" so that "CatherineDesignerend" in A2 will also return "Approved".

=IFERROR(IF(EXACT(RIGHT(VLOOKUP(A2,F2:F5,1,FALSE),3),"End"),"Approved",""),"")

In case you want cell A2 last 3 letters to be case-sensitive "End" so that "CatherineDesignerend" in A2 should return a blank, use below formula:

=IFERROR(IF(EXACT(RIGHT(A2,3),"End"),IF(EXACT(RIGHT(VLOOKUP(A2,F2:F5,1,FALSE),3),"End"),"Approved",""),""),"")

In case you want a case-sensitive match for the full text, then the above formulas can be modified. Presently, in the first formula the last 3 letters ("End") matching text in F2:F5 is tested as case-sensitive while in the second formula the last 3 letters ("End") in both A2 & F2:F5 are tested as case-sensitive.
