To change the drive letter, make sure that the external hard drive is connected and follow these steps:

slance310 say:

Hi,

 

Win 8 has a new feature called Fast Startup. It can sometimes cause problems like you mention as well as others.

 

The easiest way to tell if this is the problem is to turn it off and see what happens.

 

This feature only works when you do a SHUTDOWN and then Boot.

It doesn't effect a RESTART.

 

To turn it off or on...

Go to Control panel...Power Options, and select Choose What the Power Buttons Do on the left.

Then select Change Settings That Are Currently Unavailable near the top center of screen...

Lower down on the window, uncheck Fast Startup.

 

This may or may not help your problem, but is easy to check.

If this resolves the problem, then you can investigate what isn't loading properly.

 

The problem is most likely related to the drivers for your network adapter, but fast startup will aggravate the problem if they are not right.

I appreciate your interest on windows 10.

You might have unpinned the Store app from the start screen. Type store in the search box Right click and choose Pin to Start and check.

Method 1

  1. Type powershell in the search box and right click on it and click on run as an administrator.

  2. Run the following PowerShell command: Get-Appxpackage –Allusers

  3. When the command completes, you'll be presented with a long list of installed apps. Use the PowerShell window Find feature and search for WindowsStore.

  4. Locate the PackageFullName and copy it.

  5. Now, use the Add-AppxPackage command to reinstall the Windows Store. Use the following command:Add-AppxPackage -register "C:\Program Files\WindowsApps\<PackageFullName>" –DisableDevelopmentMode

  6. In my case, the Add-AppxPackage command line is: Add-AppxPackage -register "C:\Program Files\WindowsApps\Microsoft.WindowsStore_2015.8.25.0_x64__8wekyb3d8bbwe\AppxManifest.xml" –DisableDevelopmentMode
