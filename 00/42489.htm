a. user say:

You can test the RAM yourself as follows and give your $59 to charity.  Dell would probably next tell you to just reinstall your XP from scratch (sometimes they don't try very hard to help).

Here's how:

Run a test of your RAM with memtest86+ (I know it is boring and will cost you a CD).

Memtest86+ is a more up to date version of the old memtest86 program and they are not the same.

The memtest86+ will not run under Windows, so you will need to download the ISO file and create a bootable CD, boot on that and then run the memtest86+ program. 

If even a single error is reported that is a failure and should make you suspicious of your RAM.

If you have multiple sticks of RAM you may need to run the test on them one at a time and change them out to isolate the failure to a particular single stick.

Always keep at least the first bank of RAM occupied so the test will find something to do and there is enough to boot your system.

Sometimes, reseating the RAM in the slots will relieve the error but a failure is still cause for suspicion.

The file and instructions are here:

http://www.memtest.org/

Here is a link that shows you have to create and use the memtest86+ CD:

http://www.geekstogo.com/forum/Guide-to-using-Memtest86-t246994.html

If someone says to run memtest86, you can tell them to go pound sand and that you know memtest86+ supercedes memtest86 and here's why:

1. There is no way to merge two gamertags.

2. Any Microsoft account can only have one gamertag associated.

3. Migrating a gamertag two a different Microsoft account will take over all digital purchases and subscriptions as they are bounded to the gamertag, not the Microsoft account. Any existing credits will not be transferred as they are bounded to the Microsoft account.

5. There is a difference in the license models of the 360 and the One. In the 360 digital purchases are bound to the console and the purchasing gamertag. On the One they are primarily bound to the purchasing gamertag. There is no license transferring on the One.

So in summary, there is unfortunately no way to have both gamertags merged. To migrate gamertag 1 into account 2 you have to get rid of gamertag 2 which means the games purchased by gamertag 2 could get lost if not transferred to the new 360 then, purchases for the One will be lost anyhow by this. So for the One it would be best to have both accounts installed on the One and have the one set as home console for both accounts, so games purchased by any of the accounts could be played by both of them. For the 360 the licenses of both gamertags should be transferred to the new 360 so both accounts are able to play them.

But no merging of the accounts at all.
