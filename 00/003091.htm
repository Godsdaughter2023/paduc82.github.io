jim gordon mvp say:

Three different combo boxes are supported in Excel for Mac

  1. Worksheet combo box
  2. Dialog sheet combo box
  3. Userform combo box

Worksheet and dialog sheet combo boxes do not have a listrow property.

Userform combo boxes do have a listrow property that can be programmed via VBA.

The way worksheet and dialog sheet combo boxes display is consistent with the way combo boxes display on the Mac in general. You'll find that same behavior when you click a long list in a combo box control on a web page. Scrolling is not broken. This behavior is by design.

If you want control over the listrow property you must use a combo box on a userform.

If you want a more manageable way to display a list on a worksheet or dialog sheet choose a Listbox control, instead. You can programmatically control the dimensions of the Listbox, which determines how many rows will be displayed.

A note of caution about Listboxes: while they display and work great in Excel for Mac, in Excel 2016 for Windows the mouse cursor is erratic, making it difficult to choose items from the list. I sent feedback to the Excel for Windows team about this and hope they will fix it.

Microsoft Equation 3.0 - caption and cross referencing do not work properly

Hi

It is also too late to switch to the new equation editor, as my thesis is already 80% written in Word 2007 using Microsoft Equation 3.0.

suzanne s. barnhill say:

The conventional approach to this problem is to put the equation and caption in separate cells of a single-row two- or three-cell table. If you use three cells, the first cell, which is the same size as the last, is left empty to balance the equation caption so that the equation itself, in the center cell, will be centered between the margins.

Word does not make it easy to insert a caption in a table cell. You have to insert the caption outside the table and then copy/paste or drag it into the table. When you are starting from scratch, the simplest solution is to save a table with dummy equation and caption as an AutoText entry. You can then easily insert this, replace the equation with a new one, and update the caption. You will then be able to cross-reference the caption without including the equation.
