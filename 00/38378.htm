Hi Deidra,

Thank you for posting your query in Microsoft Community.

  • Is the issue specific to some of the tracks or any?

If you're having trouble playing a song or video, it might be because the file is protected with digital rights management (DRM). DRM is a technology that content providers sometimes use to make sure that the digital music and video files you get from them are used and distributed according to their media usage rights.

Let's look at the possible situations:

a. When you try to play a protected file in the Player, the Player checks to see if you have valid media usage rights on your PC. If you do, the Player plays the file.

If the Player doesn't find valid media usage rights on your PC, or if the rights limit what you can do with the file, the Player can't play the file. In this case, you'll see an error message that tells you why the file couldn't be played.

b. Is it rights-protected?

(i) Swipe in from the right edge of the screen, and then tap Search (If you're using a mouse, point to the lower-right corner of the screen, move the mouse pointer up, and then click Search.).

(ii) Enter Windows Media Player in the search box, and then tap or click Windows Media Player.

(iii) Swipe across or right-click a media file, tap or click Properties, and then tap or click the Media Usage Rights tab. If the file is rights-protected, the license terms appear in the Media usage rights box, and you'll see what usage rights you have. If nothing appears in the Media usage rights box, it might be because the file isn't protected, the usage rights aren't found on your PC, or you're playing a file that's being streamed to your PC (for example, you're playing subscription content directly from an online store's website).

If the issue still persists, I would suggest you to run Windows Media Player troubleshooter and check.

a. Press Windows key + X and select Control Panel.
b. In the search box, type troubleshooter, and then click Troubleshooting.

mann_manohar say:

Hello Dennis,

Welcome to Microsoft Community Forum,

As per the description, I understand that you get beeping sound from your computer randomly, you checked the sound settings and did not find any alarm.

I would appreciate if you can provide us the following information to help us understand the issue better.

  1. What is the make and model of your computer?
  2. At what stage from startup do you get this beeping sound from the computer?
  3. Is there any pattern of this beep and does it change or remain constant every time?
  4. How long does this sound last?
  5. Does this problem appears while using any particular type or application?

If you're hearing beep codes after you turn your computer on, it typically means that the system has encountered some kind of problem before it was able to display any kind of error information on the monitor. Different beep patterns at different stages signifies different issues with the computer.

Pay close attention to the number of beeps, if the beeps are long or short and if the beeping repeats or not. There is a big difference between a "beep-beep-beep" beep code and a "beep-beep" beep code.

The long beeping sound is likely coming from your motherboard. Generally, these indicate either a video adapter or memory issue, but not exclusively. Your motherboard or system manual may contain information on these, or you can look up the error codes on the manufacturer's web site. Replacing the faulty component will be the most likely solution.

If the computer beeps at startup before loading Windows, I would suggest you to contact your computer manufacturer to get hardware diagnosis performed on the computer.

However, if the problem appears after loading Windows or while using any application, please reply with the required information and other relevant details about the issue, we’ll be glad to assist you.

Hi

Go to Start / All Programs / Startup folder. If there is a shortcut for a program in that folder, the program will start when the computer boots.

Press the Windows Key+R to bring up the Run dialog box.
