taidara say:

Hey T4NGO!

Since you're connecting more then one Xbox to the same router Internet connection, you'll have a port competition for Xbox traffic. In this scenario, you can't configure static NAT, to favor one console from the other.

Not many end-user NAT devices/routers cannot handle multiple console support with an open NAT/full cone NAT/Endpoint-indepedent mapping.

UPnP (Universal Plug and Play) is the only feature (on your router) that support such a set up, depending on how well-behaved UPnP implementation works on your router.

Restore both console's network settings to factory defaults and have each console's network settings set to obtain IP configuration from the DHCP server (Automatic).

Press "B" on your controller and select Test Xbox LIVE Connection.

Disable Port Forwarding and DMZ and make sure these entries are cleared. Save/Apply.

Disable UPnP. Save/Apply and reboot your gateway.

Enable UPnP. Save/Apply.

Go to 'Advanced Settings' and select respectively console to have, 'Always use this IP address'. Save/Apply and reboot.

Post back and let us know how that goes.
