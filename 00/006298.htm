Doing what you need is going to be a process with several steps.

  • Start with "Text from original PDF".
  • Select all
  • use <CTL><ALT><K> (autoformat) to apply standard formatting to all text. (Hint: on my system that shortcut has been hijacked by Windows as a system command, so I had to add the "autoformat" command to my QAT to make it work)

This will apply the 'normal' style formatting to all of the text, converting font from courier to (probably) Calibri (MS default). 

It will remove the extra para marks from the ends of the lines.

After you do that you'll see that the lines are too close together. No problem, you just have to modify the style that is being used. In my case, since there were leading numbers it applied the "List" style.  I would have to modify it to increase the line spacing. 

In the modify style dialog click on the "Format" button, Paragraph command, Indents and Spacing tab.  Change the "Space before" OR "Space After" amounts until it looks like you want.

Hint: make sure to turn off the option "Don't add space between paragraphs of the same style".

As well, you will probably want to to play with the size of the indent to get text lined up the way you want.

These articles tell you how to do this

@ modify styles in Microsoft Word
Word has dozens of built-in styles. But the pre-set formatting may not suit your needs. For example, in Word 2002 and Word 2003, Word's default Body Text is Times New Roman, 12pt. In Word 2007 and Word 2010 it's 11pt Calibri. Perhaps you'd rather it was 11pt Century Schoolbook, or 10pt Lucida Sans.
You can modify any of Word's built-in styles. This page describes two different methods for modifying paragraph or character styles

@ Customize or create new styles in Word - Office Support
https://support.office.com/en-us/article/Customize-or-create-new-styles-in-Word-d38d6e47-f6fc-48eb-a607-1eb120dec563
To learn more, see Apply a style to text in Word. Modify an existing style. Modify a style by updating it to match formatting in your document

Another style thing you will have to do is create or modify a "character" style to apply the italics and size you use to identify the speaker, ie "the commissioner:", ie the "Emphasis" or "Intense Emphasis", or "Subtle Emphasis"

You can probably use an existing style.  Display the "Style pane" (this article tells how: https://www.dummies.com/software/microsoft-office/word/how-to-find-styles-in-word-2016/ )

#1 click on highlighted button to display #2 styles pane.

#3 click on the Options button to display the Style Pane Options dialog

My personal preference is to set #4 Select styles to "All Styles". That is all default and custom styles in the document

#5 I also set sort to "Alphabetical" order

#6 then click OK to save the change.

You now can see all of the default styles and custom styles in the document. Pick one to modify.


You can use Find and Replace to apply the emphasis style much faster.

#1 In the replace dialog, type the text to apply the style to, ie "Mr. Branson :", including the space and colon.

#2a type (or copy/paste) the same text in the Replace dialog

#2b click on the "More" button to expand the Find/Replace dialog

#2c click on the Format button to show the drop down list

#3a click on style to display the Styles dialog

#3b select the style to apply, ie "Emphasis"

#4 after the style dialog closes you will see the style name under Replace input box.

Click on Replace All to apply the style to all occurrences of the specific Find text. Since you included the space and colon, the style would NOT be applied to any occurrences of the name in the body of the text, just to the identifiers.

The last step in the process is to carefully edit the text. The quality of text generated by OCR (Optical Character Recognition) depends on several factors. The bottom line is that you have to carefully read through the text, comparing to the original PDF, looking for transcription/translation errors (like the ones you identified with the yellow highlights in your example document).  But also look carefully for missing text or misspelled text that word does not recognize and highlight.

One editing trick is to read the text out loud. Another is to read the text from bottom to top, one sentence at a time. Both of these techniques force you to focus on the text. Many people find that if they read silently they are more likely to skip over errors.  And another trick I use is reading from a printed copy of the document.
