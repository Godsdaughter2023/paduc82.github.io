How do or can I format placement content control for credit card - Microsoft Community

flooc say:

How do or can I format placement content control for credit card numbers?

Hello - I'm creating a fill-in document for credit card authorization and would like the placement formatted for the credit card types: Visa/MasterCard/Discover and American Express. 

Please advise 

Thank you

macropod say:

Content Controls don't take formatting arguments the way formfields do; instead you need to use a ContentControlOnExit macro to apply the required formatting. And, since Diners & Amex card #s are shorter than the others (14 & 15 digits, respectively vs 16 digits) and have a different formats, the macro would need to handle all formats - and be able to recognise whether a Visa/MasterCard # is missing one or more digits or an Amex/Diners card # has too many digits.

For example, if the content control's title is "CreditCard#", you could add the following code to the 'ThisDocument' code module of the document or its template. If you add the code to the 'ThisDocument' code module of the document, you'll need to save it in the .docm format. . If you add the code to the 'ThisDocument' code module of the template, you'll need to save it in the .dotm format.

Private Sub Document_ContentControlOnExit(ByVal CCtrl As ContentControl, Cancel As Boolean)
Dim StrTmp As String, bValid As Boolean
With CCtrl
  If .Title = "CreditCard#" Then
    If .Range.Text = .PlaceholderText Then Exit Sub
    StrTmp = Replace(Replace(.Range.Text, " ", ""), "-", "")
    'Validate the Card #
    If IsNumeric(StrTmp) Then
      If Len(StrTmp) Mod 2 = 1 Then
        bValid = Validate("0" & StrTmp)
      Else
        bValid = Validate(StrTmp)
      End If
      If bValid = False Then
        Cancel = True
        MsgBox "Invalid card #. Please try again", vbCritical
        Exit Sub
      End If
      Select Case Len(StrTmp)
        'Diners Club
        Case Is = 14
          If Left(StrTmp, 1) = 3 Then
            StrTmp = Left(StrTmp, 4) & " " & Mid(StrTmp, 5, 6) & " " & Right(StrTmp, 4)
          Else
            Cancel = True
            MsgBox "Invalid card #. Please try again", vbCritical
            Exit Sub
          End If
        'Amex
        Case Is = 15
          If Left(StrTmp, 1) = 3 Then
            StrTmp = Left(StrTmp, 4) & " " & Mid(StrTmp, 5, 6) & " " & Right(StrTmp, 5)
          Else
            Cancel = True
            MsgBox "Invalid card #. Please try again", vbCritical
            Exit Sub
          End If
          'Visa = 4, MasterCard = 5, Discover = 6
        Case Is = 16
          Select Case Left(StrTmp, 1)
            Case 4 To 6
              StrTmp = Left(StrTmp, 4) & " " & Mid(StrTmp, 5, 4) & " " & Mid(StrTmp, 9, 4) & " " & Right(StrTmp, 4)
            Case Else
              Cancel = True
              MsgBox "Invalid card #. Please try again", vbCritical
              Exit Sub
          End Select
        Case Else
          Cancel = True
          MsgBox "Invalid card #. Please try again", vbCritical
          Exit Sub
      End Select
    Else
      Cancel = True
      MsgBox "Invalid card #. Please try again", vbCritical
      Exit Sub
    End If
    .Range.Text = StrTmp
  End If
End With
End Sub

Function Validate(StrTmp As String) As Boolean
Dim i As Long, j As Long, k As Long
For i = 1 To Len(StrTmp)
  j = Mid(StrTmp, i, 1) * (i Mod 2 + 1)
  k = k + Int(j / 10) + j Mod 10
Next
If k Mod 10 = 0 Then
  Validate = True
Else
  Validate = False
End If
End Function

rich007a say:

I agree this is strange behaviour.

As a work around, try pasting this code into the ThisWorkbook module of the Workbook:

 

Private Sub Workbook_WindowActivate(ByVal Wn As Window)
    ActiveWindow.DisplayGridlines = False
End Sub

 

In Excel with the relevant workbook open, press Alt+F11, then in the top left pane (Project Explorer) find the ThisWorkbook module in your workbook.  Paste the code.

Does that fix it?  You'd have to make that change in all your workbooks.  There is a way to have that code run when any workbook is activated[*], but I can't think of it right now.  Maybe one of the other helpful chaps on this site can point you at that.
