karl timmermans (outlook mvp 2012-2018) say:
My apologies for over-looking that this is for OL'2010 (had '2016 on the brain when I replied).

For OL'2010 - an Imap account requires 2 data files, one for the default contact/calendar folders and the other for the email folders. The default folders are stored in the default data file which in this case is the PST file that was used for your POP account. You can't remove this data file since in OL'2010, the data file for the Imap account cannot be set as the default.

In short, there is no way to remove the PST data file that used to belong to the POP3 account. Just as an FYI, if you had configured the IMAP account as the first account in this profile, OL'2010 would have created 2 PST files with the only difference being that the default data file would have been called "Outlook Data File" instead of having the name of an email account.
