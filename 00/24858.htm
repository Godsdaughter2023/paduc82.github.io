Outlook 2011 raw query syntax question - Microsoft Community

scottgbg say:

Outlook 2011 raw query syntax question

I'd like to use a smart folder to display email from "john smith" OR to "john smith".  Looks like i can use a raw query to do this since "OR" is not supported in the standard query tool.  Could someone point me to the correct syntax?
talking moose say:
Below is an example of a raw query string with three parts separated by "||", which is "or" in raw query syntax.

com_microsoft_outlook_author_email_addresses == *** Email address is removed for privacy *** || kMDItemAuthors == *Richard* || kMDItemAuthors == "Joe Cool"

com_microsoft_outlook_author_email_addresses == *** Email address is removed for privacy ***
This example shows how to search based on email address of the sender.

kMDItemAuthors == *Richard*
This example shows how to search based on the display name of the sender. In this case it will match "Mr. Richards", "Richard Branson" or "Sir Richard".

kMDItemAuthors == "Joe Cool"
This example shows that you can also search based on an exact display name of someone by using quotes.

You really don't need to use a raw query to search "from" or "to" someone. Just search for "John Smith" and Outlook should show you both. However, the syntax for that would be:

kMDItemRecipients == "John Smith"c

The final "c" at the end makes the search case insensitive.
