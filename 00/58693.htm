Not all computers provide Power Management options (Standby or Hibernation).  Consequently, the first step is to check whether your computer hardware supports Power Management (or ACPI). Check the user manual (if you have one).

Then, you will need to enter the BIOS configuration menu.  The manual may provide instructions.  If not, re-boot and press the F10 key repeatedly - BEFORE Windows loads.  Computers differ, so if F10 doesn't work, try again using Esc, F12 or F9.

BIOS settings differ, so the following options may, or may not be available.
If the option is available, check the power management settings and make sure power management has been enabled.
If the options are available, enable Standby (S3) and Hibernate (S4)
Save any changes and exit from the BIOS configuration menu.

Boot into Windows
Click on Start / Run
Type "cmd" (without the quotation marks)
Click on OK
At the command prompt, type "powercfg /hibernate on" (without the
quotation marks).  NB.  There is a space between powercfg and /hibernate.
Press the return key.
Type "exit"
Press the return key.

Open Control Panel
Click on Power Options
Select the Hibernate tab
Enable hibernation, and click on OK.

If these suggestions don't help, you will find additional information in the Microsoft Knowledgebase -
http://support.microsoft.com/kb/907477
