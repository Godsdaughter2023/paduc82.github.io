You should not use the "three finger" reset (pressing volume down + camera key + menu key and pressing power button) to hard reset the Nokia N8-00 device. Doing this will delete some applications (Ovi Store, Ovi Maps, some preinstalled widgets) which cannot be restored.

The "three finger" reset should be used only as a last resort if the device cannot be booted up in any other way.

Search function in MS Access

Hi

john w. vinson say:

First off, data is not stored in Forms, but only in Tables. A Form is just a tool, a window to let you work with data in Tables.

Secondly, 52 fields is a extremely wide table, and probably not properly normalized.

That said... is the criterion always in the same field, or in different fields from one search to the next? Are you familiar with Queries, or with setting the Filter property of a Form? Either will probably be better than the builtin (but limited) Find button.

scottgem say:

First I agree with John. You need to review your table structure.

However, if you are always searching on the same field, then you might at the search combo. The Combobox wizard allows you to create a combobox where you can select a value and retrieve that records.
