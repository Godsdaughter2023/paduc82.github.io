You need to insert a <<Next record>> field at each point where you want the merge to move to the next record in the data source.

 b. use ctrl-F9 to insert a pair of the special field braces { }, then type "NEXT" between them, so you have

{ NEXT }

then use alt-F9 as necessary to display field results view rather than field code view. (You cannot just type the "{" and "}" using the ordinary keyboard keys).
