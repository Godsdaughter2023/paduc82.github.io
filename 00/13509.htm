Hi All,

I finally could build a working solution based on the Jaccard similarity.

From Wikipedia, the free encyclopaedia

The Jaccard index, also known as the Jaccard similarity coefficient (originally coined coefficient de communauté by Paul Jaccard), is a statistic used for comparing the similarity and diversity of sample sets. The Jaccard coefficient measures similarity between finite sample sets, and is defined as the size of the intersection divided by the size of the union of the sample sets:

The UDF is basically splitting the lookup_values in groups of 3 letters, e.g.

JP Ronse => jpronse => jpr, pro, ron, ons, nse

Say one of the values in the lookup_array is jp ronsse, that would result in:

jpr, pro, ron, ons, nss, sse

The intersection of both is: jpr, pro, ron, ons and the union is:

jpr, pro, ron, ons, nse, nss, sse.

The Jaccard index is 4/7 = 0.57

Say that the lookup_array contain also jpr ronse:

jpr, prr, rro, ron, ons, nse

intersection: jpr, ron, ons, nse

union: jpr, prr, pro, rro, ron, ons, nse

Gives also a Jaccard index of 0.57

Therefore, I had to include some options to return several results and let the user selecting the match he/she want to use.

The UDF has following arguments:

Lookup_Value As Variant, Lookup_Array As Range, LookupColumn As Integer, ReturnColumn As Integer, Optional ExcludeWords As Range, Optional BuiltInWords, Optional IndexNr, Optional ReturnIndex, Optional PctMatch = 0.85

Lookup_Value and Lookup_array are self explaining..

The LookupColumn is the column in the lookup_array where to find the lookup_value. The returncolumn is the index to return from the lookup_array.

The arguments ExcludeWords & BuiltInWords  can be used to remove a string from the lookup_value and lookup_array.

Referring to the Portfolio.xlsx included in the Fuzzy Lookup add-in, it replaces "corp" by "", "inc" by "", etc.

IndexNr defines the returned matches to display. ReturnIndex to return the match by the index number. And finally a percentage of match we would like to get.

Option Explicit

Public dicExcludeWords As New Dictionary

Function JaccardLookup(Lookup_Value As Variant, Lookup_Array As Range, LookupColumn As Integer, ReturnColumn As Integer, Optional ExcludeWords As Range, Optional BuiltInWords, Optional IndexNr, Optional ReturnIndex, Optional PctMatch = 0.85) As Variant
   Const k                 As Integer = 3
   
   Dim dicIntersect        As Dictionary
   Dim dicUnion            As Dictionary
   Dim lngRow              As Long
   Dim intCount            As Integer
   Dim varLookUpArray      As Variant
   Dim strTemp             As String
   Dim sngJaccard          As Single
   Dim dicLookupValue      As Dictionary
   Dim rngLastCell         As Range
   Dim varMatches          As Variant
   Dim varSplitAddresses   As Variant
   Dim varMatch            As Variant
   Dim i As Integer, j As Integer
   Application.Volatile False
   Application.ScreenUpdating = False
   
   If IsMissing(BuiltInWords) Then BuiltInWords = True

   Set dicLookupValue = New Dictionary
   dicLookupValue.CompareMode = TextCompare
   
   If Not ExcludeWords Is Nothing Then
      If dicExcludeWords.Count = 0 Then
         dicExcludeWords.CompareMode = TextCompare
         For intCount = 1 To ExcludeWords.Cells.Count
            If Not dicExcludeWords.Exists(CStr(ExcludeWords.Cells(intCount))) Then dicExcludeWords.Add CStr(ExcludeWords.Cells(intCount)), Item:=0
         Next intCount
      End If
   End If
   Lookup_Value = WorksheetFunction.Trim(Normalize(Lookup_Value, BuiltInWords))
   For intCount = 1 To Len(Lookup_Value) - (k - 1)
      If Not dicLookupValue.Exists(CStr(Mid(Lookup_Value, intCount, k))) Then dicLookupValue.Add Key:=CStr(Mid(Lookup_Value, intCount, k)), Item:=0
   Next intCount
   
   ''' limit Lookup_Array to the cells containing values
   ''' avoid walking through each row
   varSplitAddresses = Split(Lookup_Array.Address, "$")
   If UBound(varSplitAddresses, 1) = 2 Then
      With Workbooks(Lookup_Array.Parent.Parent.Name).Sheets(Lookup_Array.Parent.Name)
         Set rngLastCell = .Cells(2 ^ 20, 1).End(xlUp)
         Set Lookup_Array = .Range(Lookup_Array.Cells(1, 1).Address & ":" & varSplitAddresses(2) & rngLastCell.Row)
      End With
   End If
   ReDim varMatches(1 To IndexNr)
   ReDim varMatch(1 To IndexNr)
   
   varLookUpArray = Lookup_Array.Columns(LookupColumn)
   For i = 1 To IndexNr
         varMatch(i) = 0
         varMatches(i) = 0
      Next i
   For lngRow = 1 To UBound(varLookUpArray, 1)
      
      Set dicIntersect = New Dictionary
      Set dicUnion = New Dictionary
      dicIntersect.CompareMode = TextCompare
      dicUnion.CompareMode = TextCompare
      For intCount = 0 To dicLookupValue.Count - 1
         dicUnion.Add Key:=dicLookupValue.Keys(intCount), Item:=0
      Next
      If varLookUpArray(lngRow, 1) = "" Then GoTo Next_Row
      varLookUpArray(lngRow, 1) = WorksheetFunction.Trim(Normalize(varLookUpArray(lngRow, 1), BuiltInWords))
      For intCount = 1 To Len(varLookUpArray(lngRow, 1)) - (k - 1)
         strTemp = Mid(varLookUpArray(lngRow, 1), intCount, k)
         ''' add it to the union if it not exists
         If Not dicUnion.Exists(CStr(strTemp)) Then dicUnion.Add Key:=CStr(strTemp), Item:=0
         ''' mark it as found if so
         If dicLookupValue.Exists(CStr(strTemp)) Then
            If Not dicIntersect.Exists(CStr(strTemp)) Then dicIntersect.Add Key:=CStr(strTemp), Item:=0
         End If
      Next intCount
      
      If dicIntersect.Count / dicUnion.Count > 0 Then
         sngJaccard = dicIntersect.Count / dicUnion.Count
         
         For i = 1 To IndexNr
            
            If sngJaccard >= WorksheetFunction.Large(varMatch, i) Then
               For j = IndexNr To i + 1 Step -1
                  ''' insert sngJaccard at i
                  ''' move the values to the end
                  varMatch(j) = varMatch(j - 1)
                  varMatches(j) = varMatches(j - 1)
               Next j
               varMatch(i) = sngJaccard
               varMatches(i) = lngRow
               Exit For
            End If
            
         Next i
         
         If 1 - sngJaccard < 0.00001 Then GoTo Exit_Function
      End If
      
Next_Row:
   Next lngRow
Exit_Function:
   
   If PctMatch > 1 Then PctMatch = PctMatch / 100
   If IsMissing(ReturnIndex) Or ReturnIndex = "" Then
      intCount = 1
      Do While intCount <= IndexNr
         If varMatches(intCount) > 0 And varMatch(intCount) >= PctMatch Then
            JaccardLookup = JaccardLookup & WorksheetFunction.Index(Lookup_Array, varMatches(intCount), ReturnColumn) & ", "  '''& "(" & varMatches(intCount, 2) & "), "
            intCount = intCount + 1
         Else
            Exit Do
         End If
      Loop
      If JaccardLookup = "" Then
         JaccardLookup = CVErr(xlErrNA)
      Else
         JaccardLookup = Left(JaccardLookup, Len(JaccardLookup) - 2)
      End If
   Else
      If varMatch(ReturnIndex) >= PctMatch Then
         JaccardLookup = WorksheetFunction.Index(Lookup_Array, varMatches(ReturnIndex), ReturnColumn)
      Else
         JaccardLookup = CVErr(xlErrNA)
      End If
   End If
   
   Application.ScreenUpdating = True
End Function




Function Normalize(strNormalize, ByVal BuiltInWords As Boolean) As String
   Dim strTemp       As String
   Dim intCount      As Integer
   
   ''' remove all signs
   strTemp = Replace(Replace(Replace(Replace(Replace(Replace(Replace(Replace( _
             Replace(Replace(Replace(Replace(Replace(Replace(Replace(Replace(Replace(Replace(Replace(strNormalize, _
             ".", ""), ",", ""), "?", ""), "!", ""), ";", ""), ":", ""), """", ""), "'", ""), "<", ""), _
             ">", ""), "(", ""), ")", ""), "[", ""), "]", ""), "_", ""), "-", ""), "/", ""), "&", ""), " ", "")
   ''' remove some common words
   If BuiltInWords Then

      strTemp = Replace(strTemp, "company", "", , , vbTextCompare)

      strTemp = Replace(strTemp, "inc", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "incorporated", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "group", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "groep", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "nv", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "sa", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "Corp", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "enterprise", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "enterprises", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "ltd", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "limited", "", , , vbTextCompare)
      strTemp = Replace(strTemp, "incorporated", "", , , vbTextCompare)
      strTemp = Replace(strTemp, " & ", "")
      strTemp = Replace(strTemp, "&", "")
   End If
   ''' remove the listed words
   If dicExcludeWords.Count > 0 Then
      For intCount = 0 To dicExcludeWords.Count - 1

         End If
      Next intCount
   End If
   Normalize = strTemp
End Function



Function GetLastCell(InRange As Range, SearchOrder As XlSearchOrder, _
   Optional ProhibitEmptyFormula As Boolean = False) As Range
'''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''

' By Chip Pearson, *** Email address is removed for privacy ***, www.cpearson.com
'
' This returns the last used cell in a worksheet or range. If InRange
' is a single cell, the last cell of the entire worksheet if found. If

'''''''''''''''''''''''''

' containing 'e'. This cell has no value in this example.
'
'''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''

Dim r As Range

If ProhibitEmptyFormula = False Then
    LookIn = xlFormulas
Else
    LookIn = xlValues
End If

Select Case SearchOrder
    Case XlSearchOrder.xlByColumns, XlSearchOrder.xlByRows, _
            XlSearchOrder.xlByColumns + XlSearchOrder.xlByRows
        ' OK
    Case Else
        Err.Raise 5
        Exit Function
End Select

With Ws
    If InRange.Cells.Count = 1 Then
        Set RR = .UsedRange
    Else
       Set RR = InRange
    End If
    Set r = RR(RR.Cells.Count)
    
    If SearchOrder = xlByColumns Then
        Set LastCell = RR.Find(what:="*", LookIn:=LookIn, _
                LookAt:=xlPart, SearchOrder:=xlByColumns, _
                SearchDirection:=xlPrevious, MatchCase:=False)
    ElseIf SearchOrder = xlByRows Then
        Set LastCell = RR.Find(what:="*", LookIn:=LookIn, _
                LookAt:=xlPart, SearchOrder:=xlByRows, _
                SearchDirection:=xlPrevious, MatchCase:=False)
    ElseIf SearchOrder = xlByColumns + xlByRows Then
        Set LastC = RR.Find(what:="*", after:=r, LookIn:=LookIn, _
                LookAt:=xlPart, SearchOrder:=xlByColumns, _
                SearchDirection:=xlPrevious, MatchCase:=False)
        Set LastR = RR.Find(what:="*", after:=r, LookIn:=LookIn, _
                LookAt:=xlPart, SearchOrder:=xlByRows, _
                SearchDirection:=xlPrevious, MatchCase:=False)
        Set LastCell = Application.Intersect(LastR.EntireRow, LastC.EntireColumn)
    Else
        Err.Raise 5
        Exit Function
    End If
End With

Set GetLastCell = LastCell

End Function

scottgem say:

First you need to understand how Access stores DateTime values. They are stored as a double precision number where the integer portion is the number of days since 12/30/1899 and the decimal portion is a fraction of a day (.25 = 6AM). So yes you will have a problem with totals over 24 hours.

What you should be doing is using a DateDiff function to determine the elapsed time in minutes. Then you can calculate the number of hours and minutes from that total. You can find functions that will take the minutes and return a string of hours and minutes.
