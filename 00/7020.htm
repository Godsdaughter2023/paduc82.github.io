djsteele, mvp say:

The Len function will tell you.

You could add an event procedure like the following to your form:

Private Sub Text0_KeyDown(KeyCode As Integer, Shift As Integer)
    If Len(Me.Text0.Text) > 500 Then
        MsgBox "You've written too much!"
    End If
End Sub

To run a query that tells you any existing entries that are too long, you'd do something like

SELECT Field1, Field2, Field3
FROM MyTable
WHERE Len(Field3) > 500

 

marshall barton say:

Another way would be to use the text box's Change event.

If you like, you could add an unbound text box (named txtNumChars) to display the number of characters (in red?) when there are more than 500.  This way they would be aware of the length of text without having to dismiss a warning dialog after typing each character:

   If Len(Me.thefield.Text) > 500 Then
      Me.txtNumChars = Len(Me.thefield.Text)
   Else
      Me.txtNumChars = Null
   End If

Or you could always display the number of characters and make the text box red when the length exceeds 500:

   Me.txtNumChars = Len(Me.thefield.Text)
   If Len(Me.thefield.Text) > 500 Then
      Me.txtNumChars.BackColor = vbRed
   Else
      Me.txtNumChars.BackColor = vbBlack
   End If

Or maybe you really don't care about exactly how many characters over 500.  You could just make an anoying sound for each character beyond 500:

   If Len(Me.thefield.Text) > 500 Then Beep
