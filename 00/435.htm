jllatham say:

You are correct, we cannot truly detect the F9 - and F9 is recalculate, not convert to a static value.  The code below will convert a formula to its calculated value, losing the formula in the process.  I'm not sure that's what you really want.  Since we cannot detect [F9] what we will look for is a Double-Click in a cell, and respond to that.

 

This code goes into the worksheet's code module (you'd have to put it into each sheet's code module individually if you want it on more than one sheet, although you could put  it into the ThisWorkbook code module using the Workbook_SheetBeforeDoubleClick() event if you wanted it to be used on all sheets in the workbook). 

 

To put it into a sheet's code module: open the workbook, select the sheet it needs to work with, right-click on the sheet's name tab and select [View Code] from the popup list.  Copy the code below and paste it into the module presented to you, close the VB Editor, save the workbook as type .xlsm (macro enabled), and try it out.

 

 

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
  'this will take a cell with a formula in it
  'and convert the contents to the value returned by the
  'formula (the formula is "lost"/deleted)
  'It only works with one cell at a time since you cannot
  'double-click on more than one at a time
  ' in case you double-click
  'on a locked cell on a protected sheet
  On Error Resume Next
  'in case there is a Worksheet_Change() event in place
  Application.EnableEvents = False
  Target.Formula = Target.Value
  Application.EnableEvents = True
  On Error GoTo 0
  Cancel = True ' do not go into edit mode
End Sub

jllatham say:

If you really want to just update the value of a formula without losing the formula, then use this code instead of what I provided above:

 

 

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
'recalculates formulas on a sheet when any cell
'is double-clicked - leaves formulas in place.
  On Error Resume Next
  'in case there is a Worksheet_Calculate() event in place
  Application.EnableEvents = False
  ActiveSheet.Calculate
  Application.EnableEvents = True
  On Error GoTo 0
  Cancel = True ' do not go into edit mode
End Sub
