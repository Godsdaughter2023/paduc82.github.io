Package Identity : Package_for_KBxxxxxx~31bf3856ad364e35~x86~~6.1.1.0

dism /image:c:\ /remove-package /PackageName:Package_for_KB978262~31bf3856ad364e35~x86~~6.1.1.0 (name of update from previous step)

Note : You will want to start with the most recent and remove them one at a time to find the problem update.
