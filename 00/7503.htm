lzw_ve say:

I did it, and works perfectly.

 

It's not difficult, but requires some skills (Apache and OpenSSL knowledge).  You can enable WebDAV access if you use Apache.  It's free.

 

Your steps are:

 

1) Download and install Apache.  If you're using Linux, it's easy: install Apache and enable mod_dav.  In Windows, you must find Apache Win32, and enable WebDAV by editing the httpd.conf

 

2) Create a self-signed SSL certificate and send it to the phone. E72 requires the file in DER format

 

3) Activate SSL in Apache (enabling mod_ssl)

 

4) Configure in httpd.conf, some directory to support WebDAV:

 

 

Alias /webdav /some/disk/path


<Location /webdav>

Dav On

AuthType Basic

AuthName "Remote Drive"

AuthUserFile /path/to/htpasswd

</Location>

 

5) Create an user and a password with htpasswd

 

You can use this directory as a Remote Drive.  The URL will be https://your-server/webdav

 

Ready.

 

----

 

Note: you can skip SSL but I don't know if the E72 supports it, and I don't recommend it, mainly because the password is sent in plain form, so anyone can sniff the connection and decode the password using a simple Base64 decoder.
