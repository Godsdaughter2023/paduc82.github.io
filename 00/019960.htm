stephanie sal say:

Hi,

There are several factors that might cause this issue. To troubleshoot your concern, we advise that you delete the temporary update files in the SoftwareDistribution folder on your computer. You may refer to the steps below on how to delete the temporary update files:

  1. Open File Explorer.
  2. Double-click the local hard disk where Windows is installed.
  3. Double-click the Windows folder, and then double-click the SoftwareDistribution folder.
  4. Double-click the DataStore folder to open it.
  5. Delete all files and folders in the folder.
  6. Click on the Back button.
  7. In the SoftwareDistribution folder, double-click the Download folder to open it, and then delete all files in the folder.

Note: If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

Do update us with the result and if you need further assistance.

Regards.

chris charette say:

Adding an incoming call to a current conversation

The latest version of both the Windows and Mac versions of Skype seem to have left off an invaluable feature that was in the previous versions. The ability to add an incoming call to a current conversation is a critical feature that allows call continuity when two or more people are expected to get on a call together with you when not everyone is available at the exact same time.

Being able to add people to a call (using the 3 dots menu on Windows or the + menu on Mac) is fine when you're responsible for setting up a call but when some people are not available at the start of a conference and they need to call you, we should be able to add them to the call already in progress. The older desktop version of Skype permitted this (and still does), but this new "Metro" interface seems to have removed this functionality completely or buried it somewhere that makes it ridiculously hard to find...if it's available at all, I still don't know for sure.

I'm using:

   Windows: Skype 12.9.604.0

Mac:

   Skype 8.11.0.4

   Calling Version 2017.23.01.38

Both of these versions fail miserably at many other things (address book, ease of use, speed, reliability, etc.), but this inability to add an incoming call to a current call is a critical one.
