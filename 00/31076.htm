The games select which version DirectX file they use, and Win10 has DX9,10&11, as well a 12.

You can verify if all DirectX files are installed by running the DirectX Web Installer. This will install any missing

files, or show a message to the effect your DX is up to date.

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

Your Intel HD 3000 graphics would be your main issue. HD3000 is limited to DX9&10, it is not capable of using DX11&12 .

Intel and laptop mfg's have not released any proper dedicated Intel HD 3000 graphics drivers for Win10, so your

HD 3000 would be using generic drivers provided by MS which are likely to reduce it's already limited performance.

- The only emulator I'm familiar with is DOX BOX. If the others have options to select DX I would think you would want to

select whatever version the game was developed to use. Some DX11 games also have options for DX9 effects so these

games may still function if you select DX9.

- Beyond this, I believe you would probably get better answers at dedicated forums for one or more of the emulators you

are using that uses, or runs games that use DirectX.

btw - Your cpu is a quad core with Hyper-threading that adds another four 'virtual' cores.

A lot of games don't run well with Hyper-threading enabled, but old games only use one or two physical cores anyway.

On desktop machines Hyper-threading is enabled/disabled in BIOS, but I'm not positive if laptops do it the same way.
