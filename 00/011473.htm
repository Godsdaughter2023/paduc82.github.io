jim gordon mvp say:
Page Layout view is the replacement for Print Preview in Excel 2011. in VBA it's:

ActiveWindow.View = xlPageLayoutView

If you want to display the old style preview it is possible by using an Automator Action. Open the AutoMator application and record the steps as you choose File > Print and then click the Preview button. Your Automator action will look like this:
Click the "File" menu
Click the "Preview" menu

Save the Automator action as an application if you want to use it often. See this page for more about using Automator:
http://www.agentjim.com/MVP/Automator/IntroToAutomator.html
