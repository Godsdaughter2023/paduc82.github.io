Sub CopyPaste_VisibleCells()
'Dec 18, 2015
Dim ws1 As Worksheet, ws2 As Worksheet
Set ws1 = Sheets("Sheet1") '<< source sht/Sheet1
Set ws2 = Sheets("Sheet2") '<< target sht/Sheet2
Dim r1 As Long, r2 As Long
r1 = ws1.Cells(Rows.Count, 1).End(xlUp).Row
r2 = ws2.Cells(Rows.Count, 1).End(xlUp).Row
Set rng1 = ws1.Range("A2:A" & r1).SpecialCells(xlCellTypeVisible)
Set rng2 = ws2.Range("A2:A" & r2).SpecialCells(xlCellTypeVisible)
If rng2.Rows.Count >= rng1.Rows.Count Then
For Each rr In rng1
s = rng2(1).Address
ws2.Range(s).Value = rr.Value
Set rng2 = ws2.Range(ws2.Cells(ws2.Range(s).Row + 1, 1), ws2.Cells(r2, 1)).SpecialCells(xlCellTypeVisible)
Next
Else
MsgBox "wrong"
End If
End Sub

Say I've got a list of 8 numbers, Assume they are in column A and the first number is in Row 1

How would one go about calculating(Excel formulae) the streak of each of the following types of numbers separated

A streak of positive numbers

A streak of negative numbers

A streak of zeros

And also calculate the average length of each streak

So for example.

In the dataset above we have 2 streaks of 1's(Positive numbers), the average streak length of the positive numbers is 2.

Please note the formula must group the numbers according to positive, negative and zeros, the 1's used here are purely for demonstration

Please assist in giving me a formula that will help me with a much larger dataset

amit tandon say:

All formulas are Array formula - Ctrl+Shift+Enter.

Array Formula in cell C2 - Number of Positive Streaks:

=SUM(--(FREQUENCY(IF(A2:A100>0,ROW(A2:A100)),IF(A2:A100<=0,ROW(A2:A100)))>1))

Array Formula in cell C3 - Number of Negative Streaks:

=SUM(--(FREQUENCY(IF(A2:A100<0, ROW(A2:A100)),IF(A2:A100>=0, ROW(A2:A100)))>1))

Array Formula in cell C4 - Number of Zero Streaks:

=SUM(--(FREQUENCY(IF((A2:A100=0)*(A2:A100<>""),ROW(A2:A100)),IF(A2:A100<>0,ROW(A2:A100)))>1))

Array Formula in cell G2 - Average of Positive Streaks:

=AVERAGE(IF(FREQUENCY(IF(A2:A100>0,ROW(A2:A100)),IF(A2:A100<=0,ROW(A2:A100)))>1, FREQUENCY(IF(A2:A100>0, ROW(A2:A100)), IF(A2:A100<=0, ROW(A2:A100)))))

Array Formula in cell G3 - Average of Negative Streaks:

=AVERAGE(IF(FREQUENCY(IF(A2:A100<0,ROW(A2:A100)),IF(A2:A100>=0, ROW(A2:A100)))>1, FREQUENCY(IF(A2:A100<0, ROW(A2:A100)), IF(A2:A100>=0,ROW(A2:A100)))))

Array Formula in cell G4 - Average  of Zero Streaks:

=AVERAGE(IF(FREQUENCY(IF((A2:A100=0)*(A2:A100<>""), ROW(A2:A100)), IF(A2:A100<>0, ROW(A2:A100)))>1, FREQUENCY(IF((A2:A100=0)*(A2:A100<>""),ROW(A2:A100)), IF(A2:A100<>0,ROW(A2:A100)))))

Note: Streaks are considered where there are minimum 2 consecutive positives / negatives / zeros.

This could be something as simple as Internet Explorer not being set to use Outlook as the default email program.

Close Outlook

For I E 8, you'll have to click on "Set Programs" in the bottom right.
