This should be fairly easy if you have 2 separate tables, 1 for each of the products.

The formula logic would be something like this:

If the cell contains this product use table1, if the cell contains this product use table2, if the cell doesn't contain either product do nothing and leave this cell blank.

Let's assume you have these 2 tables on Sheet2:

Complete  
1 75
10 91
20 45
50 38
100 16

Core  
1 10
20 32
75 80
150 4
300 46

You can give each table a unique defined name to make it easy to identify which table needs to be used. In this case the logical names might be Complete and Core.

Ok, now let's assume that sheet1 cell A1 has a drop down list and the choices are either Complete or Core.

Cell B1 contains a number of users.

Then, you could use a formula like this in cell C1:

=IF(OR(A1="",B1=""),"",LOOKUP(B1,IF(A1="Complete",Complete,Core)))

swischner say:

Selective sharing like you want to needs you to store notebooks on OneDrive or OneDrive for Business. As the owner of your cloud space you just share single notebooks to others and may decide to keep other notebooks (even in the same storage space / folder) private to you.

If you store notebooks on a network shared folder (which is possible only with the Office OneNote although not "very supported" by Microsoft anymore) the access rights are not handled by OneNote but by the file system. 

So if you want to share notebooks with other users this way you would have to store them in a folder / network share where all participants have access rights (either just read or read/write). 

Your private notebooks have to be stored in another place where only you can access them. 

Please also note that storing notebooks locally or on a network share instead of Microsofts cloud services you won't be able to access those notebooks with OneNote Win 10 App, OneNote for iOS, Mac, Android or the free OneNote 2016 at all! You have to use OneNote 2010/2013/2016 which comes with the corresponding Office package.

j.e. mcgimpsey say: One way:

   =B2 * $A$1 / 100

Note that both SUM() and the parentheses are superfluous. In SUM(B2*A1), B2 is multiplied by A1, then the result is passed to SUM() which just returns the same result.
