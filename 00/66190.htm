How to install and activate Windows XP after hardware upgradation - Microsoft Community

ashoke biswas say:

How to install and activate Windows XP after hardware upgradation

I am using Windows XP SP3 and it is activated. I have the original and genuine installation CD
with product key in my possession. I intend to upgrade my hardware by replacing the HDD, motherboard,CPU,
 RAM and DVD drive. Kindly advise how to install and then activate Windows XP for the new
hardware configuartion. At present, I do not have any intention to migrate to Windows 7 operating system.
john john __ say: I take it that you are wanting to do a 'clean' installation and that you don't want to keep your old installation?  If so just boot the computer with your Windows XP CD and follow the on screen instructions.

With your new hardware your hard disk will almost certainly be SATA.  The Windows XP setup program doesn't include SATA drivers so it will not be able to detect the drive, you have several options:

1-  Fit a floppy diskette drive to your system and use the F6 driver installation method to supply the SATA controller drivers to the Setup program.  When using the F6 installation method the Setup program will only accept drivers on a floppy diskette,
Limited OEM driver support is available with F6 during Windows XP and Windows Server 2003 setup

2-  Slipstream the SATA drivers to the Windows XP installation CD.  This is usually the preferred method, the slipstreamed CD will be useful if you ever need to use the Recovery Console.  While you are at it you may as well also slipstream the latest Service Pack (SP3) into the installation, your installation will be that more recent and it will be one less thing to do when the installation is completed.

Slipstream SATA drivers

3-  Go in the BIOS and set the SATA controller to IDE mode for the Windows XP installation.  You can then later install the SATA controller drivers while Windows is up and running and after the drivers are installed you can reboot the machine and go in the BIOS and change the IDE mode back to SATA and Windows will be able to boot using the SATA drivers.

If you want to keep your old installation you can just clone it to your new drive then do an in-place upgrade of the operating system.

John

    I think that if MSE brings your computer to a crawl, you must have some other problem(s).

    I would take a good hard look at your AVG setup if you decide to keep it, for example, if you have the AVG firewall installed and the XP firewall is also running, you re going to have a problem with two firewalls running.  Pick one or the other but never two at the same time.

    To try to be reasonably confident your system has no malicious software on it, I would to this no matter what else you are using for malicious software protection.

    Download, install, update and do at least one full scan with these free malware detection programs:

    Malwarebytes (MBAM):  http://malwarebytes.org/
    SUPERAntiSpyware: (SAS):  http://www.superantispyware.com/

    These full scans can take a long time, but you really need to run them.  SAS will probably just report a bunch of Internet tracking cookies, but you can delete those.  Once you have done at least one full scan, you can do the quick scans in the future to save time and save the full scans for when you have more time or are really suspicious of a system infection.

    They can be uninstalled later if desired.

    I believe that MSE and regular scanning with MBAM and SAS is good protection.  I will never install AVG again.

    If you care to have someone look over your startup items for things that you might not need, do this:

    Download CCleaner from here:

    http://www.piriform.com/ccleaner

    CCleaner will let you manage most Startup items and also do some other cleanup oriented operations.  Some people will recommend you not to use the registry functions in CCleaner so avoid those features for now.  I have no issue with it myself, but it is sometimes a contentious issue I prefer to avoid whenever possible!

    You can uninstall CCleaner later if you don't like it.

    You can disable startup items, see how things look and if you want - totally delete unneeded startup items, or just leave them disabled while you continue troubleshooting.  Note that disabling things does not uninstall anything and if there is a problem, you can just enable the item again.  If you are sure you don't want it at all, then you can delete it.

    After launching CCleaner and seeing what startup items you can disable/delete if you still are not sure about your startup items, do this:

    Launch CCleaner and choose Tools, Startup and in the bottom right, click the "Save to text file" button to save the Startup information to a text file on your desktop with a name you can remember - something like startup.txt

    Save the startup information text file to your desktop (or someplace you can find it) open the text file with a text editor (notepad or WordPad), select all the text (Ctrl-A), copy all the highlighted text to the Windows clipboard (Ctrl-C) and then paste (Ctrl-V) the contents of the startup list back here in your next reply and perhaps somebody can tell you other things to consider.

    A more revealing startup analyzer is Autoruns which will allow manipulation of all areas of startups and does not actually install anything:

    http://technet.microsoft.com/en-us/sysinternals/bb963902.aspx
