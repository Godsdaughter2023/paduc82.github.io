Use data validation with the list option on the cells that you want to control and:

1) Copy this code.
2) Right-Click the sheet tab of interest.
3) Select "View Code"
4) Paste the code into the window that appears.
5) Save the file as a macro-enabled .xlsm file.
6) Make changes to the range as needed

Private Sub Worksheet_BeforeRightClick(ByVal Target As Range, Cancel As Boolean)
    If Intersect(Target, Range("A3,B4,C5,D6")) Is Nothing Then Exit Sub
    Application.CutCopyMode = False
End Sub

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Intersect(Target, Range("A3,B4,C5,D6")) Is Nothing Then Exit Sub
    Application.CutCopyMode = False
End Sub
