Having set a Word document to print on a custom-sized paper a thick white border, about 2cm deep, appears at the top and the bottom of the document. The document is the same size as the custom paper. The printer manufacturer, Canon, says it is a Microsoft issue rather than an issue with the printer. Any help much appreciated!
stefan blom say:
Note that most printers have a nonprinting area. To find out which margins are accepted for your default printer, do the following: Exit and restart Word (easiest way to get back to the default printer, in case you selected a different one recently, for the current Word session). Display the Page Setup dialog box, for example by double-clicking the upper part of the horizontal ruler. On the Margins tab, set all margins to zero. Click OK. If the printer doesn't support zero margins, Word will present an error message offering to "fix" the margins. Choose to do so and you'll see the smallest possible top, bottom, right, and left margins for the printer.
