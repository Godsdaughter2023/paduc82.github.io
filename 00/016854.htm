Drop-down box length? - Microsoft Community

maryhonaker say:

Drop-down box length?

I'm trying to add a drop box to a word document to create a form.  Is there a way to expand the number of characters that you can put in a drop box? I would like to put a complete sentence but I'm only getting 50 characters.

a. user say:

It looks as if you can push the limit up using VBA, but...
 a. I don't know how far exactly and
 b. there are at least two limits: the text length that you can achieve via VBA, and the length that Word is actually going to display. AFAICS 
    - you can only have 25 options ("sentences")
    - as long as each sentence has 222 characters or fewer, you can have 25 entries. Longer texts may reduce the number of options you have.
    - Word displays longer texts by chopping out the middle and substituting "...". I don't know what determines the width limit - a character count, a "text width can't be more than so many pixels" limit, a display width limit, or what.

But all I did was macro record the creation of a single dropdown and a single entry, then modify the macro. The following macro should let you create a form dropdown in a blank document with the texts you need. Then, I think you could probably copy that dropdown where you need it. If that's not enough, you'd need to do som more work on the macro.

If this is unfamiliar territory,
 a. you need to enable the Developer tab in the Word 2011 ribbon Word->Preferences->Ribbon->Customize
 b. there are some hints on how to install and use VBA macros here:

http://word.mvps.org/Mac/InstallMacro.html

Sub Macro25()
'
' Macro25 Macro
'
'
    Selection.FormFields.Add Range:=Selection.Range, Type:=wdFieldFormDropDown
    Selection.PreviousField.Select
    With Selection.FormFields(1)
        .Name = "Dropdown1"
        .EntryMacro = ""
        .ExitMacro = ""
        .Enabled = True
        .OwnHelp = False
        .HelpText = ""
        .OwnStatus = False
        .StatusText = ""
    End With
    Selection.FormFields("Dropdown1").DropDown.ListEntries.Clear
    Selection.FormFields("Dropdown1").DropDown.ListEntries.Add Name:="Put the sentence 1 text in here"
    Selection.FormFields("Dropdown1").DropDown.ListEntries.Add Name:="Put the sentence 2 text in here"
    Selection.FormFields("Dropdown1").DropDown.ListEntries.Add Name:="Put the sentence 3 text in here"
    ' and so on...
End Sub
