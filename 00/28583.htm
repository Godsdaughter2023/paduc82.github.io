Hi,

By default, Excel reads data only from OPEN workbooks - it cannot read data from a closed workbook.

However, there can be a workaround - you can give it a try :

1. You can have the data read from a closed (un opened) workbook by making use of a macro

alternatively

2. - Assume that you have two workbooks WKB-A and WKB-B

    - WKB-A has your raw data in Sheet1   and you have a summary data in Sheet2

       ( This summary data is going to be the data using which you can create your Dashboard ) 

    - In WKB-B, in (say) Sheet-2 you create a link to WKB-A Sheet2 by giving formula as

       'SomeDriveLetter:\fullpath\[WKB-A.xlsx]Sheet2'!cellreference

       ( example - 'E:\ABCD\PQRS\[WKB-A.xlsx]Sheet2'!B24 )

                   (Normally we do not give full path while writing a formula).

      This way you have to fetch each and every cell of your summary data.

    - Now in Sheet1 of WKB-B you create your Dashboard using data from Sheet2 of the same workbook (WKB-B).

   

    This way the workbook WKB-B Sheet2 cells will be linked to cells of workbook WKB-A Sheet2 and would

    give results even when the WKB-A workbook is closed.

I have personally tried this and it works.

andre da costa say:
This is a bug that has been affecting some Insider Builds since build 17063.

Does the edition of Windows 10 correspond with your Windows 7 license?

The copy of Windows 10 you download and upgrade from will correspond with the edition of Windows you have installed, if it does not, this is why you might be experiencing problems activating:

Windows 7 Starter, Home Basic, Home Premium, Windows 8.0 Core, Windows 8.1 Core will install Windows 10 Home

Windows 7 Professional, Windows 7 Ultimate, Windows 8.0 Pro, Windows 8.1 Pro will install Windows 10 Pro

To find out if you have the right edition installed:

Click Start > Settings > Update & security > Activation > Edition.

If the edition of Windows 10 installed does not correspond with the edition of Windows 7 or Windows 8 you were previously running, then you need to reinstall the right edition.

If its the right edition, try using your Windows 7 product key to activate:

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

If you are unable to activate Windows 10 after all the above and you are certain you have the correct edition installed, then please attempt phone activation:

Activate Your Windows 10 License via Microsoft Chat Support
https://partner.support.services.microsoft.com/...

If none of the above help...

Because you are running a Windows Insider Preview build, you might be seeing defects or bugs in the build you are using, which is still under development. I suggest you use the Feedback Hub to submit a bug report detailing your problem.

Press Windows key + F
This will open the Feedback hub
Proceed to submit your report.

Feedback:

You can reach out to key folks on the Windows Engineering team through Twitter

https://twitter.com/northfacehiker
