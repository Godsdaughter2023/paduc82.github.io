Install chipset drivers first, then the latest graphics, sound and any other device drivers. Once the chipset drivers are installed the installation order should not matter.

Windows Updates should be applied, but be careful about any hardware driver updates offered.

Windows Virtual PC and XP Mode are applications, so should be installed after the system hardware drivers.
