brian tillman [outlook mvp 2007-2019] say:
The 5.6.2 error is "Bad domain or address.  Invalid or improper domain or address in MAIL FROM, RCPT TO, or DATA."  This is an error issued by the receiving SMTP server.  You are also seeing error 554.  The definition of this is "Reply code 554 is to be used to reject a MAIL FROM, RCPT TO, or DATA command which contains improper domain references."  This is, essentially the same thing.  The headers you supplied prove that the issuing server is icp-osb-irony-out7.iinet.net.au.  So, the Outlook.com server contacted the iinet server, gave the iinet server the address you included in the message, and the iinet server said, "bad address".
