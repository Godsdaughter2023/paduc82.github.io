Now, here's the key step:  DO NOT BOOT from the media.  Let me repeat that.  DO NOT BOOT from the media.  Instead, boot the existing installation of Windows and run the setup from the flash drive user data.  It also preserved my license key so I didn't need to re-enter it again.

The inconvenience of having to re-install all of one's programs is usually the key deterrent to re-installing Windows, but by following the procedure I outlined, that becomes a non-issue.

After the re-installation completed (which did take multiple hours), I naturally had to go through the process of downloading and installing all the necessary Windows updates.  I also needed to re-install a handful of drivers.  However, those minor inconveniences were well worth the hassle since I now have a machine that functions as it should.  The key takeaway is that I did not have to go hunting for installation media and re-install the dozens of other programs on the machine.
