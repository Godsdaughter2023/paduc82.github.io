If you currently have Windows 7 installed and you plan to keep using it on that machine, the best recommendation is to backup the current installation, which you can easily restore to a new hard disk in case of failure:

 http://www.notebooks.com/2009/10/24/how-to-backup-your-installation-of-windows-7/

If your intention is to keep an emergency copy in case you purchase a new computer with Windows 10, your options will be very limited.

Say your computer fails within 3 years, any computer you purchase at that point will work with Windows 10 only, even if you have a copy of Windows 7 you can install.

Unless you plan to purchase a second hand computer capable of running it and you plan to furnish your own license.

As for getting a copy.

Because Windows 7 is in limited stock because Microsoft ended sales in October 2013, this has created demand because of its scarcity. This means retail boxed copies demand a premium, while OEM System Builder copies remain affordable.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM System Builder version:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

There is nothing wrong with using it too, especially since all future releases of Windows will be full versions anyway.

You can find OEM System Builder software from dozens of online merchants. The current price for OEM Windows 7 Professional at Newegg, for example, is $140. When I checked a few minutes ago, Amazon was offering OEM Windows 7 Professional packages from multiple sellersat prices ranging from $101 to $150. When I checked just now, a package specifically intended for refurbished PCs cost only $50 for a 64-bit copy.

There are no technical limitations to prevent you from using OEM software on your own PC, although this software will work only for a clean installation and not for an upgrade. In the past, Microsoft has been remarkably inconsistent in its advice to customers about whether this practice is allowed. (See "Is it OK to use OEM Windows on your own PC? Don't ask Microsoft.")

bossdweeb say:

I am assuming your machine is using the older Intel GMA graphics, which was a chip on the motherboard.

The newer Intel HD graphics is built into the cpu and is much more capable ( especially HD4000 and higher).

There are a number of recent games that may be limited to DX10 or 11, but these are still somewhat

rare (Just Cause 2 is DX10 only). These games (commonly referred to as AAA titles) have high requirements,

and your Intel graphics would struggle to run them at all, even using only DX9.

The vast majority of games that use DX 10 and 11 would run using DX9 by default. DX10 or 11 effects are options selected  in the games setup menus. The only way to implement DX10/11 effects without turning the game into

a slide-show would be to lower other more important graphics settings.

I say more important because DX10/11 effects are more a frosting on the cake - nicer lighting, smoke and similar

subtle effects.

- There is no way to upgrade any graphics (dedicated card or integrated) to higher DX levels, and

there is no way to upgrade integrated graphics chip-sets. These are mounted to the motherboard.

If this is a PC you should be able to install a dedicated graphics card, but the power supply has to be

powerful enough. The age and capability of the other components needs to be considered as well.

Laptops - no hope.

Easiest access to try first is holding Cntrl + Alt + Del keys to see if you get a screen with the Power icon to hold Shift +  Restart which will take you into the Options screen.   You may also need to Create Windows 10 Installation Media to boot to access Repair console on second screen.  You can also use this later to do the Upgrade over, do booted repairs, or reinstall.  Once Upgrade activates you can Clean Install Windows 10  as often as you please.

At any time you can rescue your files, which should always be backed up externally if you value them, by booting into a Windows installation disk to Copy & Paste - in Windows Recovery Console, a Linux disk, or Paragon Rescue Kit Free Edition 11.0 Free to copy files to USB flash stick or an external or another hard drive.

Then immediately without installing any programs or anti-virus besides Microsoft Security Essentials in Windows 7 or  built-in Defender in Windows 8.1,  fully update, Activate, save a System Image in Backup Center so you can go back at any time.

Install the tool, Tick the checkbox in first option to automate the process, or select "For another PC" to create bootable flash media or download an ISO to burn to a new DVD.  Now you have media for reinstalls or needed repairs.  But to run the Upgrade the media must be opened in Windows 7/8.1, then right click Setup file to Run as Administrator.

In Windows 7, type msconfig in Start Search Box, uncheck everything on Startup and Services tabs,

In Windows 8.1, right click Task Bar to open Task Manager, disable everything in Startup Tab, Apply, Reboot.

Immediately after install, type Update in Start Search box to Check for Updates, expedite these through requested restart, then check again until no more are offered. 

Next right click Start button to open Device Manager, import any drivers still missing from the PC or Device's Support Downloads webpage.  How to: Install and Update drivers in Windows 10 - Microsoft Community
