joeu2004 say: ToneML wrote:
VLOOKUP gives #N/A in a number of cells in a column where there's no result found. I need to change them all to 0 but there seems to be no way to do this with the Find and Replace function. How else can I simplify this process?

As to the latter question, use the following form in Excel 2007 and later:

=IFERROR(VLOOKUP(...),0)

where "..." represents your parameters to VLOOKUP.  IFERROR returns the result of VLOOKUP if no error; but if there is an error, it returns the 2nd parameter, zero in this case.

stefan blom say:

There is no automatic saving in Office unless you save to OneDrive.

When saving locally, all you have is "Save AutoRecover information every X minutes" which is not designed to be a replacement for saving a file manually.

You can use Windows 10 File History to set up backups, or you can use a third-party solution of course.

You may also be interested in http://www.gmayor.com/SaveInTwoPlacesAddIn.htm.
