When you are done with your editing of your project, first save the project file (.wlmp) with File > Save project. That will save the current state of your project in case you want to make further edits later and/or want to save the movie in a different format.

Next, you want to save your movie. This is a separate option called File > Save movie. When you select that, you'll be given a choice of output formats. Scroll down through the list until you get a format you like (e.g. Android (large) or Android (medium) or perhaps Android 720p or Android 1080p.

Hi WRG2012,

It’s unfortunate that you are unable to connect to Internet. Do not worry; let me see what best I can do for you to get the issue fixed.

 

I would suggest you to perform the following methods and check if it helps to resolve the issue.

 

Method 1: Reset Winsock

a. Click the Start button.

b. Type cmd in the Start Search text box and press Ctrl-Shift-Enter (keyboard shortcut to run Command Prompt as Administrator). Click Continue to allow elevation request.

c. Type netsh winsock reset in the Command Prompt shell, and then press the Enter key.

d. Type netsh int ipv4 reset in the Command Prompt shell, and then press the Enter key.

e. If you use ipv6, type netsh int ipv6 reset in the Command Prompt shell, and then press the Enter key.

Note: You may need to run Network Diagnostics afterwards to repair your connection after this.

 

Method 2: Reset TCP/IP settings

You can try manual steps provided in the link mentioned below to rest TCP/IP settings
http://support.microsoft.com/kb/299357 

Note: This applies to Windows 7 Operating System as well.

 

Method 3: Perform System Restore

System Restore can return your PC's system files and programs to a time when everything was working fine, potentially preventing hours of troubleshooting headaches. It will not affect your documents, pictures, or other data.

Please refer the link: http://windows.microsoft.com/en-US/windows7/products/features/system-restore and follow the steps mentioned in the link.

If the above methods don’t work, I think the problem is with a setting on the router. If you have MAC filter configured on the router I would suggest you to disable it and check if it helps to resolve the issue.
