a. user say:

Actually, File History was not designed to "copy literally every single file on a hard drive." For that you should be using a third-party backup program. File History was designed to backup individual files as they change.

To copy all your user files at one time (documents, pictures, music, etc.) you'll want a folder-and-file backup program. There are lots of great examples - one is Backup4All.

To backup your entire C: drive at once - which you can easily restore in the case of system failure - you'll want an image backup program like Macrium Reflect (which also has a free version.) Here too there are several great programs from which to choose.
