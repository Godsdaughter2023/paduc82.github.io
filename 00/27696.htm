    Andy,

    the driver that you download directly from Realtek is, I believe, different from the beta that they have furnished to Microsoft to be offered through Windows updates.

    As you probably know, there has been a time-out problem with installing the driver provided by Windows Updates, and, there is the fix (that works) of lengthening the time allowed for driver installs; however, I have been working on the other end of the problem, believing that the real cause of taking so long to install it is that the old driver must first be uninstalled, and the new driver has difficulty accomplishing that ... and therefore, the long time.
