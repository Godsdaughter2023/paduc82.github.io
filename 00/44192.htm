xircal say:

Assassin's Creed Revelations dates from 2011 according to Wikipedia so your best bet here will be to run the game in Compatibility mode for Windows 7.

So right click the setup file and click Properties. Click the Compatibility tab and then checkmark the box to run the game in compatibility mode for Windows 7. 

Further down the same Compatibility menu, checkmark the box to Run as an administrator. Click Apply/Ok and then run the setup file.
