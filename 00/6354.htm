jim gordon mvp say:

Hi,

You're on the right track.

Put both videos on the same slide

A very basic way is to use the PLAY animation

The first video should be set to play with previous

The second video should be set to play after previous

A variation of the above would be to set both videos to play with previous, but set the second video to start x amount of time after the first one (based on how long that video takes to complete)

You can get very fancy, if you want. You could use other animation effects for entrance and exit. You could use motion paths to move the videos around on the screen. But. you'll have to think about the order in which things play on the slide and know how "with previous" behaves compared with "after previous."

If you have an office 365 subscription, you could put the first video on the first slide, then apply Morph transition and then duplicate the slide. On the copy, delete the first video and insert the second video.

To enable looping of the show, go to the Slide Show tab and choose Set Up Show.

in Set Up show, in the Show Options choose to loop continuously until Escape key is pressed.
