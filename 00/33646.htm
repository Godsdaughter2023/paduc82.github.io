considerate_guy say:

First, Movie Maker never touches your original source files, so it can never "delete" the audio from your video. All that Movie Maker can do is create a new movie from other videos, photos and music.

So if your question is can you remove (mute) the original audio and replace it with music and then save that as a new movie, the answer is yes.

To mute the original audio, double-click on the video clip to get to the Video Tools tab. Set the Video volume to zero. Now you can do whatever you want with added audio such as adding music, sound effects, etc.

Another potential way to do this, if you don't want the original audio from any of your original video clips is to go to the Project tab and set the Audio mix. Normally it is set in the middle between your original audio and the added audio. If you only want to hear added audio, set it all the way to the right.

The first approach is useful if you want to only mute the audio from certain clips in your project. The second approach is useful if you know you want to do it completely for all clips in the project, not just some of them.

a. user say:

The first question to ask for those kinds of errors is:

Did you attempt an upgrade to Windows 10 and then revert back to Windows 7?

If the answer is yes, forget about any KB articles or advice from the Microsoft engaged alleged "experts", Methods and Steps and ideas that begin with the word "try".

What has happened is when reverting back to Windows 7 sometimes the registry parts of the Scheduled Tasks are restored but the Windows 7 tasks files themselves are not restored leaving behind Windows 10 tasks so now there is a conflict and hence the "corruption" of the tasks - things are not matching up like they should.  

The chances are very high that ALL your Scheduled Tasks are corrupted (about 80 of them) and some of them are pretty important if you want your Windows 7 to behave properly.

The problem usually surfaces first when folks discover their backups are no longer working and try to set their backup schedule up again but you will probably not be able to look at any of the Windows Scheduled Tasks - they are all corrupted.
 
From Control Panel search for Scheduled Tasks and see if you can even open the Windows Task scheduler without getting some error messages.

Some nice person named Dijji has already figured out this problem and developed a nifty method/program to straighten out the Scheduled Tasks and people report that this method works pretty well most of the time and you can read about it here:

https://repairtasks.codeplex.com/

It might help to understand the problem better if you to read through the Home and Documentation tabs first so you can walk through the repair process.  Note that there is a separate Discussion tab where folks report success, issues, problems and questions.  The Discussions are helpful too when you read them and see what kinds of things folks
like you have run into and how to fix them.

In the Windows 7 community most of the time the CodePlex method works just fine (whew!).

Example where the CodePlex method worked:

http://answers.microsoft.com/en-us/windows/forum/windows_7-update/task-scheduler-multiple-image-is-corrupt-or-has/6205cb04-a749-4f4f-a5be-632e01989158

But if there are any issues or problems consider pursuing them in the Discussion tab on their page first (they know what they are doing).

There are also many examples where the CodePlex method is not able to restore all the tasks and you can decide if you want to try to fix them yourself individually and that might work - maybe.

Finally you have to consider that if reverting back to Windows 7 corrupted all your Scheduled Tasks what other things on your system are now afflicted/broken that you don't know about yet?  In the future if you have a problem are you always going to wonder if the problem was caused by some Windows 10 leftovers?

Some frequent contributors in the Windows 7 community feel the only way to be 100% sure your Windows 7 system is whole again is to backup all your personal files and folders, whack your hard drive and do a clean/fresh install of Windows from scratch but that is a lot of work.

It is an all too frequent problem when installing Windows 10 and reverting back to Windows 7.

foxfifth say: Two hard drives in a laptop is very unusual and I would be extremely surprised if your laptop has two hard drives -- the following review indicates that one 640gGB drive is standard: http://www.notebookcheck.net/Review-Packard-Bell-EasyNote-TX86-540M-Notebook.36957.0.html

If you remember two hard drives, what is more likely is that the one physical hard drive had two partitions on it.  This would make it look like it has two drives because the one drive had been subdivided.  That is frequently done on laptops so the factory can deliver it with software that makes it easy for you to set the laptop back to the way it was when it shipped from the factory after you have had a problem with it.  That second partition is sometimes hidden but not all manufacturers have always done that.  Backing up to that partition is not a good idea as the space is normally minimal and backing up to any partition on the main hard drive is also a bad idea because if there is a hardware problem with the drive, both partitions will become unusable and there goes your backup.  External USB hard drives are becoming relatively inexpensive and are a much better place for a backup.

jjafolla say: Hey, I actually got it working again.  What you have to do is restore File Manager as the default handler of .VHDs but for some reason it won't let you manually select it.

What I had to do was go into the registry and go to Computer\HKEY_CLASSES_ROOT\.vhd

I changed the default key to 
(Default)  REG_SZ  Windows.VhdFile

and I added another key under that called "OpenWithProgids" with the values

(Default)  REG_SZ  (value not set)
Windows.VhdFile  REG_SZ

For the second one you'll have to right click and add a new String Value and just don't type anything into the data field.
