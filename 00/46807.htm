Deleting recent documents from WordPad - Microsoft Community

jennifer graha say:

Deleting recent documents from WordPad

What steps should I take to delete recent documents from WordPad on Windows 10?

alexander bertnikov say:
Hello Jennifer, my name is Alexander. I'm an Independent Advisor and I'm happy to assist you!

You have to make some changes in the registry if you decided to delete recent documents in WordPad.
Please, refer to the article on ‘How to back up and restore the registry in Windows.’ https://support.microsoft.com/en-us/help/322756...
Note: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs.
Please, follow the steps below:
- Close WordPad.
- Go to the Start and search for ‘Run’.
- Rype regedit and press Enter.
- Locate the following registry subkey: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Applets\Wordpad\Recent File List.
- On the right pane, serach for Files#.
- Right click on the registry keys and press Delete.
- Close the Registry editor.
- Restart the PC.

Best regards,
Alexander B.

davem121 say:
Hi Jennifer, you will need a registry fix . . .

Click your Start Button, type regedit and hit Enter to open the Registry Editor

Click View and make sure 'Address Bar' is turned on

Paste this into the Address Bar at the top and hit Enter

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Applets\Wordpad\Recent File List

In the right hand pane, you will see the list - File1, File2 . . .etc.

Right click each one and choose Delete

Make sure to leave the 'Default' value there . . .

ethan b. say:
Hi Jennifer,

I'm an Independent Advisor here to help.

This will be a bit of a process, but hopefully I can help you get it done as simply as possible.

Hit Windows key + R to go into Run, enter: regedit

That will take you to the Registry Editor. Copy and paste the following path, then paste it into the address bar of that Registry Editor window: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Applets\Wordpad

On the left pane (after you navigate to that path), go into Recent File List, and on the right, you should see File1, File2, File3, so on and so forth. Just go ahead and delete all of those.

It's not a user friendly process granted, but I hope this helps.

http://social.answers.microsoft.com/Forums/en-US/w7files/thread/09d5a83e-dadc-441d-8ac7-46f5641781dd

 

If you do not have more than 50 folders in the Libraries and yet receive the error message, I’d recommend you to restore the libraries to default state and check if that helps in fixing the issue.

 

To do this, follow these steps:

 

1. Click Start and select Computer.

2. Right-click Documents in the navigation pane and select Delete.

3. Click Yes if asked for confirmation.

 

Note: Deleting a library does not delete the contents of the included folders. However, you may need to re-add or include your folders manually, if needed.

 

Then, in the navigation pane of Windows Explorer, right-click Libraries and click Restore default libraries.

 
 

This recreates the library that was deleted earlier. This option only restores the default libraries.

 

For further information, refer the following article:

 

Libraries: frequently asked questions
