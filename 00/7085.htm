Here's an extended macro that starts linking combo boxes to cell D2, and continues on from there (assuming you have enough combo boxes).

Sub LinkBoxes()
StartingRow = 2 'Starting row
StartingColumn = 4 'Starting column

i = StartingRow
c = StartingColumn
For Each sh In ActiveSheet.Shapes
If sh.Type = 8 Then 'Combo Box type
sh.Select
Selection.LinkedCell = Cells(i, c).Address
i = i + 1
If i > 100 Then 'Max row to goto
c = c + 1
i = StartingRow
End If

End If
Next
End Sub

roady [mvp] say:

The current EAS method to connect to Hotmail will be deprecated by the time that Outlook 2016 will be released as Outlook.com will be migrated to Exchange Online and will make an actual Exchange connection then. Your sync issues should be no more then.

The fact that you only see your Inbox folder when connecting via POP3 is expected since the POP3 protocol only supports downloading (not synching) from the main mailbox folder (Inbox).

As an alternative, you could also connect via IMAP but that has some limitations of its own as IMAP only synchs mail folders and doesn't support Calendar and Contacts folders. For configuration details see:

http://www.howto-outlook.com/howto/accountsettings.htm
