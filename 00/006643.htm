Refresh a Pivot Table on Hidden Sheet - Microsoft Community

faraz a. qureshi say:

Refresh a Pivot Table on Hidden Sheet

What piece of code could have a couple of Pivot Tables on Hidden Sheets be refreshed?
Thanx in advance.
hansv mvp say:

It doesn't matter whether a sheet is hidden or not; the code to refresh a pivottable remains the same.

For a single pivottable:

 

    Worksheets("SheetName").PivotTables("PivotTableName").RefreshTable

 

or

 

    Worksheets("SheetName").PivotTables("PivotTableName").PivotCache.Refresh

 

For all pivottables on a sheet:

 

    Dim pvt As PivotTable
    For Each pvt In Worksheets("SheetName").PivotTables
        pvt.RefreshTable
    Next pvt

I have had a similar issue where I have to give the same survey to new classes. I find the easiest way is to save as to a new file. Then open that file and delete all the data except those that need to carryover. I create quizzes and I use the first two lines as correct answers and grading check statements respectively. Since it is an Excel file, it can be saved to a central location and used as a template, emailed, shared on OneDrive, etc. This technique however assumes that each survey is a standalone.
