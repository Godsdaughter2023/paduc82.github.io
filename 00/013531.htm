If that didn't work, then try to manually add the shared mailbox to Outlook:
• Open Outlook.
• Choose the File tab in the ribbon.
• Choose Account Settings, then select Account Settings from the menu.
• Select the Email tab.
• Make sure the correct account is highlighted, then choose Change.
• Choose More Settings > Advanced > Add.
• Type the shared email address, such as info@contoso.com.
• Choose OK > OK.
• Choose Next > Finish > Close.

For detailed information, please refer to the article: Open and use a shared mailbox in Outlook

To set up rules for shared mailbox via OWA, you need to open the shared mailbox in a separate browser window and then set up a rule in Outlook on the web for a shared mailbox. 

• Sign in to your own user account in Outlook on the web.
• On the Outlook on the web navigation bar, click your name. A list appears.
• Click Open another mailbox.
• Type the email address of the Shared mailbox that you want to open, and then click Open. Another Outlook on the web session opens in a separate window, allowing access to the Shared mailbox.
• And then select the Settings menu icon and choose Mail.
• In the Mail > Automatic processing section choose Inbox and sweep rules.

For you reference, see under Open the shared mailbox in a separate browser window

For detailed information, please refer to the article see under Outlook on the web: Add rules to a shared mailbox
