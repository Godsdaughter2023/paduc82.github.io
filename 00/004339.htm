techfreak_ say:

A camera light can remain on if the mic is accessed by any program in Windows, not just Skype.   It does not necessarily mean the video portion of your camera is in use.   In my past tests, I have noticed a camera light can be triggered by simply having its mic being set as the default communication device.  Under the recording tab in your Windows sound properties you can try one or more of the following:

1.    Right clicking on a recording device that is not your onboard mic and select "Set as Default Communication Device".

2.    Disable your webcam mic.  This will either be a separate mic entry for your laptop or be part of some audio array.

3.   Separate the inputs which will break up an audio array into separate devices in the Windows Sound properties.  This is usually found in you audio configuration software found in control panel or your Start Menu.  Sometimes the act of unbinding them will stop this problem.  In other cases you may have to accomplish a combination of 1 or 2.

Be aware that leaving the Windows sound properties open may be enough to trigger the camera light when viewing active devices in the tabs.

dnago4 say:

Macro or style for shading alternate table rows, some with cells vertically merged

Alternate row shading can be accomplished with rows containing merged cells by discontiguous row selection using the Ctrl key and choosing all appropriate cells, but setting up a style to accomplish the same task is challenging. Applying a stock style (e.g., Grid Table 2 - Accent 1) skips some cells and misinterprets row positions in others. The style customization menu either does not accommodate merged cells, or this option is deeply hidden. However, the Ctrl-select method moving by row can select vertically merged cells as if they all belong to the same row.

macropod say:

You could use a macro like:

Sub Demo()
Application.ScreenUpdating = False
Dim Cll As Cell
For Each Cll In Selection.Tables(1).Range.Cells
  If Cll.RowIndex Mod 2 = 1 Then Cll.Shading.BackgroundPatternColor = RGB(237, 244, 253)
Next
Application.ScreenUpdating = True
End Sub

but don't be surprised if Word's idea of alternate rows isn't the same as yours, especially for the row following any cell that's split into an even number of rows.
