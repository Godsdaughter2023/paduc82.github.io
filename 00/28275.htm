Hi ArminN,

I notice you “would like to import our office 365 users into active directory and have on going synchronization”. As far as I know, we can only sync local AD users to Office 365 via DirSync tools.
But it is not feasible to sync Office 365 users to on-premises active directory.

For the concept of SSO, if you just deploy ADFS, it is true that it only applies to domain joined computers. However, if you deploy ADFS proxy, then you would be able to experience SSO on non-domain-joined machines.
For ADFS proxy, you may refer to the following article:

peterthwaites49 say:

Using the DLookUp function

I am struggling to sort out the method of using the DLookUp function and would really love some help before it drives me mad!

I have a form, 'Inspections Recorded' that has 2 fields, 'Inspection Code' and 'Code Description'

( All taken from the table 'Inspections' as described below. )

I also have two tables, one called 'Inspection Item' with two fields, 'Code' and 'Description', and the other called 'Inspections' with two fields 'Inspection Code' and 'Code Description'.

What I am trying to do, is when data is added to the the form field 'Inspection Code',  the next field 'Code Description' is automatically completed using the DLookUp function. 

Where the 'Inspection Code' field in the form 'Inspections Recorded' is the same as the 'Code' field in the 'Inspection Item' table, then the data in the 'Description' field is copied to the field 'Code Description' in the form 'Inspections Recorded'.

I have tried this as follows: =DLookup("[Description]", "Inspection Item", "[Code] =" & Forms![Inspections Recorded]!Inspection Code) 

Nothing happens at all !

Thanks

Peter

dhookom say: To fix your immediate issue you need to:
- wrap text values in quotes
- wrap object names with spaces in square brackets

Assuming Code is a text field you should try:
=DLookup("[Description]", "Inspection Item", "[Code] =""" & [Inspection Code] & """")

If Code is numeric, try:
=DLookup("[Description]", "Inspection Item", "[Code] =" & [Inspection Code])

I would expect you could use a combo box for the [Inspection Code]. The Row Source would be

SELECT [Code], [Description] FROM [Inspection Item] ORDER BY [Code];

Then you could replace the DLookup() with:

=[Your Combo Box Name].Column(1)
