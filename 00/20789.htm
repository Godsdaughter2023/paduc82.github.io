If the cells are truly empty, then the chart will plot correctly.

If you use a formula to calculate the X axis value and that formula returns an empty string like "" or a blank like " " then the chart will indeed number the data points on the X axis since the data is no longer truly numeric.

suzanne s. barnhill say:

If by "newline" you mean a line break (Shift+Enter), then this is not true, but if each item is in a separate paragraph (separated by paragraph breaks, Enter), then yes, you can do this. To convert the list:

  1. Press Ctrl+H to open the Replace dialog.
  2. In the "Find what" box, type a comma and a space (assuming there is a space after each comma).
  3. In the "Replace with" box, type "^p" (without the paragraph marks).
  4. Click Replace All (if there is other text in the document, select the list before replacing).
  5. Once you've done this, you can select the new list and use Table | Sort to sort it.

greg maxey say:

You can sort a comma separated list with a simple macro.  Just select the list and run:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim arrItems() As String
  arrItems = Split(Selection.Text, ",")
  WordBasic.SortArray arrItems
  Selection.Text = Join(arrItems, ",")
lbl_Exit:
  Exit Sub
End Sub
