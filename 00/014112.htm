Hi Robbie,

There's actually two ways that come to mind.

The first is called AutoCorrect. What you do is put the letter in a document and select it. Then open the AutoCorrect pane, which you do from Word > Preferences > AutoCorrect. Your selected Lambda symbol will show in the AC pane and you then provide a shortcut key combination that will work every time you type those letters followed by a space. In my sample below I used "=L" (without the quotes) as my replacement trigger.

The second method is AutoText, which starts the same way by inserting the letter, symbol or phrase into a document, select it, open the AutoCorrect pane... but then you choose the AutoText button which in part of the menu bar in the AutoCorrect pane itself. You give the entry a name, see example below, and add it to the AutoText collection of the template. Note that both formatted AC entries, which in the case of a symbol are formatted, and AutoText entries are stored in a template. It can be any template but to use them the template must be attached to the active document. That is why often the Normal template becomes the storage facility for the AC and AT entries.

Finally with your AutoText entry stored, you assign a custom keystroke to it, for example Control + L, and you use the Customize Keyboard function from the Tools menu to do that.

I hope this helps, and of course if you have any further question about this, just post back.

suzanne s. barnhill say:

In the Footnotes group on the References tab, click the "dialog launcher" (the small arrow in the bottom right corner) to open the Footnote and Endnote dialog. Make sure "Apply changes to" is set to "Whole document," and set the "Number format" to the desired type, making sure that there's nothing selected for "Custom mark."

 

If everything appears to be set correctly, then it's possible that the Footnote Reference style is formatted as Hidden or white. You'll need to access the Modify Style dialog for that style. One way is to open the Apply Styles dialog (Ctrl+Shift+S), type "Footnote Reference," and then click Modify. Or you can open the Styles pane (Alt+Ctrl+Shift+S) and set the Options to display All Styles in alphabetical order, then scroll to Footnote Reference and choose Modify... In the Modify Style dialog, click Format, then Font and see if anything besides Superscript is selected.

Useless - I couldn't make an image backup of my system disc and recovery partitions because the recovery partitions were not NTFS format and you can't remove them from the devices to be imaged.  It seems as though this was due to the issue which was causing problems with the machine as I've tried again and this time it seems to be OK.
