Hi,


You can try the below steps


1. On a page, add the text box for the title

2. Select the text box, go to Developer tab -> shape design group -> Shape Name and note down the name

3. Add another text box on the far right edge. While in editing mode, go to Insert Tab -> Text Group -> Field

4. In the field dialog, select custom formula under category and in custom formula text field enter the formula - SHAPETEXT(<ShapeNameOfTheTitleTextBoxGotInStep2>!TheText) e.g. SHAPETEXT(Sheet.1!TheText)

5. Rotate the text box etc.

6. Now whenever you update the title, text box on the far right edge will get updated automatically

7. Use a copy of this page as a starting point for all your sheets to avoid doing the above steps for each page separately
