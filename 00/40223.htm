Hi,

Both of those are only available via Windows Update.

If you've ever run a troubleshooter on your machine, it deletes the history. That causes the menu to show "Never" when the user clicks "View update history" giving the impression that no updates have been installed.

If that's what you're seeing, do the following.

  1. Right click the Start button and go to Programs and Features at the top.
  2. In the menu which opens, click the link on the left to "View installed updates". You'll see them all there.
  3. Copy/paste KB3035583 into the Search field top right and if it's installed, you'll see it appear below. Do the same with the other one.

If you don't see them, copy/paste KB2919355 into the Search field.This is a prerequisite to obtain Windows 10 as well as other patches.

If you don't have that installed, you'll have to download it manually from here: http://www.microsoft.com/en-us/download/details.aspx?id=42335

Use Internet Explorer to download it because that provides the option to install it immediately. Click Run instead of Save.

After you install that, go to Windows Update in Control Panel and click the link to "Check for updates". Those two will be among the updates to install.
