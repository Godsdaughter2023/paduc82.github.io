Background Intelligent Transfer Service (BITS) is used to transfer files asynchronously between a client and a server. There are three types of transfer jobs. A download job downloads files to the client, an upload job uploads a file to the server, and an upload-reply job uploads a file to the server and receives a reply file from the server application. This service is used for Windows Update, app updates, and file transfers.
