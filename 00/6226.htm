julie fka profjulie say: If you've got a WM6.1 windows mobile device you can change the location where bluetoothed files are placed as follows:

Tap Start > Settings > Connections tab > Wireless Manager icon
Tap Menu> Bluetooth Settings > tab the File transfer tab > input in the file folder name and path where you'd like bluetoothed files to go (i.e., \Storage Card\My Documents)
Tap OK and try it out to see how well it works.

If your device is running an earlier OS, the place where'd you'd change the location is found by going to Bluetooth Settings, then tapping the Services tab > tapping the File Transfer option in the Services window > tap the Advanced button > change the place where you'd like bluetoothed files to go in the "My shared folder" box.
