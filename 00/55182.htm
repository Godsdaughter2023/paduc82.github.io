john john __ say:

It's not a driver problem, its a hardware problem at the BIOS level.  First go in the BIOS and make sure that the disks are not disabled and make sure that the disk are properly identified, it is usually best to set the disks to AUTO and the BIOS should properly recognize them.  If the BIOS does not see the disks then open the computer and verify the power cables to the disks, make sure that they are properly seated, also verify the data cable and make sure that it is properly seated on the motherboard and on the disks.

Third Party Network Setup Instead of Homegroup - Microsoft Community

daddyman say:

Third Party Network Setup Instead of Homegroup

I appreciate Microsoft's efforts to make it simpler for users to setup a home network through the Homegroup and Windows Connect Now. Nevertheless, for those who are familiar with home networking or prefer to use third party tools, or who wish to include computers that don't run Win 7 in a home network, can we forgo setting up a Homegroup altogether?
daddyman say: For anyone else with the same question: I was able to find the answer elsewhere.

It is not necessary to use Windows 7's Homegroup to setup your home network; use any third-party tool you like. (And yes, Homegroup is one word.)

If you are relying on Windows Firewall (WF) as your software firewall, you will have to tell WF whether your computer is on a public network, home network or work network. If you choose 'home network' you will be offered the opportunity to setup or join a Homegroup, but there's no need to do so. A Homegroup makes network discovery and sharing easy...as long as you're okay with doing it Microsoft's way.

  2. Change the view by option on the top right to Large icons.
  3. Click on “Administrative tools” and select “Windows memory diagnostic.
  4. Click “Restart now and check for problems”.
