marshall barton say:

I never heard of using a object/method in an event PROPERTY.  Event properties only allow the name of a standard module public function following an = sign.  The other choices.in an event property are the name of a macro or "[Event Procedure]".

DoCmd and other objects/methods can only be used in VBA procedures and sandbox mode has nothing to do with it.
