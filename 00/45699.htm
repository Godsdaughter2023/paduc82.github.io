brian tillman [outlook mvp 2007-2019] say:

DHCP stands for Dynamic Host Configuration Protocol.  A DHCP server is a device on a network whose job it is to hand out network addressing information to the computers that appear on a network.  In a home network, the router is usually the DHCP server, handing out IP addresses to the devices that are connected to the network.  Often, a router's address will be something like 172.160.1.1 and it will hand out addresses in that network.

Verify your printer is acquiring a valid network address.  The first two octets of the IP address should match those of your router or your other PCs on the LAN.  If the address starts with 169, though, your printer is not acquiring a valid address, which indicates that the network settings you've entered into the printer are incorrect.  If your printer is acquiring a valid address, then you'll have to start breaking the problem down into smaller parts.  Trying to ping an address of a device that is working is a good place to start.  Look at the router's internal web page to see if it detects the printer.  (My router gives a list of all active devices on the network.)  Disable all internet security on the PC and then try to reach the printer.  Visit HP's Wireless Printing Center for more help.  Try using their Wireless Network Assistant to see if it can help also.
