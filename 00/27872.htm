Sub newWB_shtNames()

Dim x As Long
Dim v As Variant
Application.ScreenUpdating = False

wb.Close False

End Sub

Can you use it as the logical argument in the IF function as follows?  =IF(A1={"Midwest","Northwest","Southwest"},2*A2,0.75*A3)

Thanks!

You can but it might not be a good idea! The "test" in that case will result in an array of TRUE/FALSE values, so if A1="Northwest" you'd get {FALSE,TRUE,FALSE}...and then the final result of the formula would also be an array - you might want that if you were to "array enter" that formula in an array of cells.......or use in another function that could process the resulting array, like SUM or AVERAGE......but not for a single result in a single cell.

The OR works because applied to the array of TRUE/FALSE values it will return a single TRUE/FALSE result - TRUE if there are any TRUEs, FALSE otherwise. AND wouldn't make much sense because A1 couldn't be equal to more than one of those......so an AND like this

=AND(A1={"Midwest","Northwest","Southwest"})

would always return FALSE
