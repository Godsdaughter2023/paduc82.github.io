vijay a. verma say:

COUNTIF is very poor in taking formulas as arguments. In these cases, you will need to use SUMPRODUCT -

=SUMPRODUCT((MONTH(B1:B6)=7)*(C1:C6="Closed")*(A1:A6="Approved"))

But if you insist to use COUNTIF, then you will have to use a helper column. In D1, put following formula and drag down -

=MONTH(B1)

Then you can use following COUNTIF formula

=COUNTIFS(D1:D6,7,C1:C6,"Closed",A1:A6,"Approved")

vijay a. verma say:

Simply use YEAR(B1:B6) = 2014 (If 2014 is the year which you are referring to)

=SUMPRODUCT((YEAR(B1:B6)=2014)*(MONTH(B1:B6)=7)*(C1:C6="Closed")*(A1:A6="Approved"))

But a better way would be following -

=SUMPRODUCT((TEXT(B1:B6,"mm/yy")="07/14")*(C1:C6="Closed")*(A1:A6="Approved"))
