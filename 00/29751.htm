    john_barnett say:

    Hello S.Panchatcharam,

     

    Is this a brand new Dell machine pre-installed with Windows 8? If so is it running a UEFI Bios?

     

    The problem with shrinking a drive is that Windows will only shrink the drive until it encounters unmoveable files. This is the reason why you cannot shrink the drive as far as you would wish.

     

    I take it that you want to shrink the C: drive to 200GB then you want to create another drive of 150GB followed by a third drive at 102GB?

     

    Santanu's post states that "Why because they have already given 4 Primary drive,,3 hidden and 1 is C drive..all are primary.So cant creat another primary". While this is correct for drive that use the MBR (Master Boot record) partition table it does not apply to drives using GPT (GUID Partition Table). A MBR partition table can support 4 primary partitions; a GPT partition table can supports 128 primary partitions. Transition to GPT drives started around 2010. Unfortunately, Windows Disk management doesn't actually tell you whether the Boot record is MBR or GPT simply by looking at the disk management graphical interface.

     

    Download the third party partitioning software called EaseUS Partition Master http://download.cnet.com/Easeus-Partition-Master-Free-Edition/3000-2248_4-10863346.html Install this to your PC and then run it. Once the Partition manager window opens you will see a list of your partitions. Look to the right of the list and see if the 'type' is MBR or GPT. If it is GPT you can create extra partitions by shrinking your drive. Actually third party partitioning software is far better at partitioning than the built in Windows Disk management console.

     

    I have a brand new Samsung Intel i7 laptop. The hard drive has a GPT boot record. Originally there were four partitions. 3 were basic recovery partitions and the other was the C: drive. The C: drive had a total space allocation of over 800GB. I managed to shrink the C: drive (the one with the operating system on) to 300GB leaving me 500GB of unallocated space to create 1 partition of 200GB for my data files, 3 partitions of 40GB each for VM (virtual Machine) systems and I still have approximately 180GB of unallocated space to create further partitions if I want them.

     

    Before you do decide to try any partitioning make sure you have a full backup of your system - that includes recovery partitions. If your system has a UEFI Bios then you will not be able to create recovery disks as the UEFI Bios usually has the 'secure boot' option enabled which does not allow any other operating systems to be installed. In other words you cannot boot from a CD/DVD operating system disk such as Linux etc. Ordinary software disks will, of course, work as normal.

    2. turn on external communication and public im connectivity in skype for business admin center.

    3. add the skype user as a contact in the skype for business client.
    you can refer to the article below for more detailed steps.
    let skype for business online users communicate with external skype for business or skype contacts.

    also for external skype users, they may not be able to directly join into a skype for business online meeting in their skype client. however, they can use skype for business web app to join into skype for business online meetings.
    when an external skype user receives a skype for business online meeting invitation email, there will be a link for the user to join in the meeting.

    the user can click on the link and then will be redirected to the skype for business web app. then the user can follow the instruction to install the plug-in and then join the meeting as a guest.
