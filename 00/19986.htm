You will be asked to close out Excel and restart for your changes to be in effect.

I had to leave this for a while but lately realized I had to update my operating system and solving this problem was critical to moving on.

After some failures, I tried Goal Seek but could not get answers with adequate precision. I finally found an application of Application.OnTime to sequential solutions on the same cell that was adaptable to my need. Here is the code:

Sub SecurityReturns()

    With ActiveSheet
    
    'Place values in fixed locations for use by other sheets
    
    Range("B4").End(xlDown).Select
        Application.CutCopyMode = False
        Selection.Copy
    Range("Q2").Select
        ActiveSheet.Paste
    Range("B4").End(xlDown).Select
    ActiveCell.Offset(0, 7).Select
        Range("$V$2") = ActiveCell
    ActiveCell.Offset(0, 1).Select
        Range("$R$2") = ActiveCell
    
    'Calculates the Internal Rate of Return for 1 year.
    
    ActiveCell.Offset(0, 4).Select
        SolverOk SetCell:=ActiveCell, MaxMinVal:=2, ValueOf:="0", ByChange:="$S$2"
        SolverOptions MaxTime:=2
        SolverSolve (True)
        SolverFinish
    
    'Set up to run next solver solution after fixed time then exit vba
    
    Application.OnTime Now() + TimeValue("00:00:03"), "SecRetTwo"
    
End With
End Sub

Sub SecRetTwo()

With ActiveSheet

    Range("B4").End(xlDown).Select
    ActiveCell.Offset(0, 13).Select
        SolverOk SetCell:=ActiveCell, MaxMinVal:=2, ValueOf:="0", ByChange:="$T$2"
        SolverOptions MaxTime:=2
        SolverSolve (True)
        SolverFinish
    
    'Set up to run next solver solution after fixed time then exit vba
    
    Application.OnTime Now() + TimeValue("00:00:03"), "SecRetThree"
    
End With
End Sub

Sub SecRetThree()
    
With ActiveSheet
   
    Range("B4").End(xlDown).Select
    ActiveCell.Offset(0, 14).Select
        SolverOk SetCell:=ActiveCell, MaxMinVal:=2, ValueOf:="0", ByChange:="$U$2"
        SolverOptions MaxTime:=2
        SolverSolve (True)
        SolverFinish

End With
End Sub

ossiemac say:

If I am interpreting correctly what you are attempting to do then like the code below.

In the following line of your code Rows.Count is referencing the ActiveSheet; not necessarily Worksheet Register.

LastRow = Worksheets("Register").Cells(Rows.Count, 1).End(xlUp).Offset(1).Row

Should be like the following line but personally I prefer to use With / End With as per my example code.

LastRow = Worksheets("Register").Cells(Worksheets("Register").Rows.Count, 1).End(xlUp).Offset(1).Row

Note that the space and underscore at the end of a line is a line break in an otherwise single line of code. (Used below after Copy).

The bold code in the Destination was the major correction to your code..

Sub CopysourcedatetoRegister()
     Dim LastRow As Long
    
     With Worksheets("Register")
        LastRow = .Cells(.Rows.Count, 1).End(xlUp).Offset(1).Row
     End With
    
     Dim header As Range, headers As Range
    
     Set headers = Worksheets("source data").Range("A1:v1")

    For Each header In headers
         If GetHeaderColumn(header.Value) > 0 Then
            Range(header.Offset(1, 0), header.End(xlDown)).Copy _
                Destination:=Worksheets("register").Cells(LastRow, GetHeaderColumn(header.Value))
         End If
     Next
 End Sub

Try this:

 

Public Function Loader()
    On Error GoTo ErrHandler
    DoCmd.SetWarnings False
    DoCmd.OpenQuery "qry_tony_test", acViewNormal
    DoCmd.OutputTo acOutputQuery, "qry_tony_test", "ExcelWorkbook(*.xlsx)", _

soshan. say:

I have been having this issue and finally figured it out.

  • Go to the REVIEW tab and click the down arrow on Show Markup, found on the Tracking section.
  • There are check marks next to a number of items:
    • Comments, Ink, Insertions and Deletions, Formatting
  • Click each of those, one at a time (because the drop down collapses when you click it).
    • See the check marks disappear!
  • LEAVE A CHECK MARK next to COMMENTS.

Check your print preview. You may now dance with joy.
