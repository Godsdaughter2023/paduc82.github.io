To sync Outlook with Live using the Hotmail Connector, you need to make the Live Contacts and Calendar folders the default data folders, and cut/paste your Outlook data there, because these are the folders that sync with Live (and the phone).


Obviously, the same Live account also has to be added to the phone.


Typically, when using Outlook your Contacts and Calendar data are in "Personal Folders" that exist in Outlook only. To copy the data and set the Live folders as default, you can follow these steps:


 


1. Go into Outlook’s Contacts view, and select all of your contacts by  holding down the “Ctrl” key and pressing “A”.  (This is usually abbreviated in  help and how-to guides as “CTRL-A.”)  This will highlight all of your contacts.

 

 

2.  Now select Move > Copy to Folder (you could also select “Other  Folder,” but this would move the contacts rather than copy them.  By copying  them, we’re leaving a backup set in the original folder in case something goes  awry.)

 

3.  In the pop-up dialog box, expand your Hotmail/Live account with the symbol on  its left, then select the Contacts folder under it and click “OK” (you might want to view the folder by selecting it in the left hand  pane to verify your contacts were copied.)




The Contacts will now sync to the Live server, and be available in the  Live web interface, the Windows Live Mail clients, or any phone that supports  Exchange Activesync, like Windows Phones.  Copying your Outlook Calendar data is  a similar process, but requires
 an extra step to insure we get all of the  Calendar data.







4. Switch to your Calendar, and then select File > Change View > List View  to see all of your Calendar items in a single list.





5. Now, like with Contacts, we select all of the Calendar items with “CTRL-A”





6. Copy them with Move > Copy to Folder

 



7. Select the Calendar folder under the Hotmail/Live Account, then click “OK”. (You might check to see the Calendar appointments copied to the folder by  selecting it in the left pane, like we did with Contacts.)

 


Now, like our Contacts, our Calendar items are now in the Live folder, and  will sync to the Live server via the Outlook Connector, to be available on our  devices, in Windows Live Mail or the Live.com web interface.

 

Once you are satisfied that all of your data has synced to the Live cloud,  you can now make the Live folders the Default Personal folders in Outlook, so  that they are pre-selected each time you start Outlook.

 

8. Select File > Info > Account Settings,

  


9. Then, in the “Account Settings” pop-up box, select the Data Files tab,  your Live account, and then “Set as Default”

 


10. A dialog box will pop-up warning that you are changing the default  folders.  Select “OK” and restart Outlook.


 


Now anything you add in Outlook will go into the new default folders, which sync with Live, and should propagate to the phone, and vice versa.  Keep in mind there may be a delay- by default, Live/Hotmail accounts only sync with Outlook every 30 minutes,
 but you change that in the account settings.  (Whenever Outlook fetches your Live email, it syncs Cobracts and Calendar as well.)


 


 


You mentioned you can only get email on the phone by forwarding them to Hotmail. What type of account is this you can't get on the phone directly?  Gmail, Yahoo, your ISP email?


 


Typically you set up a separate account for that email, using the same settings you used in Outlook.  If you tell us the provider, we can probably track down the settings.
