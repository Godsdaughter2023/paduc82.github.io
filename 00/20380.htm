'===== BEGIN CODE =====

Option Explicit

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim myShape As Shape
   
    If (Not Intersect(Target, ActiveSheet.Range("$B$1")) Is Nothing) Then
        Select Case ActiveSheet.Range("$B$1").Value
        Case "Yes"
            For Each myShape In ActiveSheet.Shapes
                myShape.Fill.ForeColor.RGB = RGB(255, 255, 255)
            Next myShape
        Case "No"
            For Each myShape In ActiveSheet.Shapes

            Next myShape
        Case Else
            For Each myShape In ActiveSheet.Shapes

            Next myShape

    End If

e.j. gun say: Strange, my original still has the shapes in it.  Maybe MS stripped them out for some reason.  Anyway, the code is very simple.  Remember that the shape that you want linked to a cell should have the same name as the cell (e.g. the shape for cell D11 will be named "D11").  The list of cells in the variable "MyCells" is what you will change to meet your needs.

Place this code in the code module of the worksheet with the shapes on it.

Eric  

'===== BEGIN CODE =====
Option Explicit

Private Sub Worksheet_Change(ByVal Target As Range)
     Dim myShape As Shape
     Dim myCells As Range
     Dim intRange As Range
     Dim intCell As Range
     Dim shpName As String
     
     Set myCells = Range("B3,B7,C10,D13,E16,F13,G10,H7,H3") ' These are the "Target" cells
     Set intRange = Intersect(myCells, Target)
     
     If (Not intRange Is Nothing) Then ' This means that that changed cells intersect our selected cells
        For Each intCell In intRange.Cells
            shpName = intCell.Address(False, False) ' Shape name same as cell address
            Select Case intCell.Value
            Case "Yes"
                ActiveSheet.Shapes(shpName).Fill.ForeColor.RGB = RGB(255, 255, 255)
            Case "No"
                ActiveSheet.Shapes(shpName).Fill.ForeColor.RGB = RGB(255, 0, 0)
            Case Else
                ActiveSheet.Shapes(shpName).Fill.ForeColor.RGB = RGB(255, 255, 255)
            End Select
        Next intCell
    End If
 End Sub

Simply reuse the existing PST.  It will still contain all of your contacts and your calendar (and your old mail, but you can delete that if you wish once you're using the other account).  Since IMAP accounts are mail-only, you'll still need that PST for your contacts and calendar.  WIth Outlook 2010 you'll need a minimum of two folder sets: the PST holding your default folders and the IMAP folder set, which cannot contain contacts or calendar and which cannot be your default folder set.  (Note: this is different in Outlook 2013.)

Features that are unavailable in shared workbooks

Unavailable feature Alternatives
Insert or delete blocks of cells You can insert entire rows and columns.
Delete worksheets None
Merge cells or split merged cells None
Add or change conditional formats Existing conditional formats continue to appear as cell values change, but you can't change these formats or redefine the conditions.
Add or change data validation Cells continue to be validated when you type new values, but you can't change existing data validation settings.
Create or change charts or PivotChart reports You can view existing charts and reports.
Insert or change pictures or other objects You can view existing pictures and objects.
