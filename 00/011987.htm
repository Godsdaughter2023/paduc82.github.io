teylyn say:
Hello,

the Excel Mobile functionality is a very reduced sub-set compared to Excel on a PC or Mac. You cannot expect the full functionality of Excel to work on a mobile phone, especially an iPhone, which has a completely different operating system from whatever Excel has been developed for. 

Printing a spreadsheet is not a trivial task. Cells need to be sized, pages need to be broken, etc. 

Excel Mobile for iOS currently struggles with some VERY basic Excel functionality like conditional formatting or data validation. 

In this context it does not surprise that printing does not work. It's about a hundred times more complex than data validation.

So, if you really, really need to print the Excel file on your iPhone, and you happen to be in range of a printer that actually talks to your phone, chances are that this printer is also connected to a computer and that said computer is connected to the internet. 

The easiest and fastest way to print your file is to log in to your Skydrive account on the computer and print from there. 

I'm sure Excel Mobile will get there in the end. But the product has only been released a few months ago as an iPhone application and there are still a lot of functionality gaps compared to the mature Excel on Windows that people have been using on PCs for the last 20 years.
