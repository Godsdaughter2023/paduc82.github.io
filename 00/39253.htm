Hi,

There are two types of bad sectors. One resulting from physical damage which can't be repaired and one resulting from software errors which can be fixed.

Regarding the specific speed, we highly suggest contacting the hardware manufacturer as this is beyond the scope of support for Windows operating system.


Furthermore, if you want to determine if there's a bad sector on your hard disk, you can use the Windows built-in Disk Check tool. This will scan your hard drives for bad sectors, marking hard ones as bad and will automatically repair soft bad sectors to make them usable again.

Note: While performing Disk Check Tool on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.
