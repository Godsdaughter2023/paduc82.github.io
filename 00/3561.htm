merge work doc direct from outlook contact - Microsoft Community

graham mayor say:

If you want to insert the address of the currently selected contact into a letter, then create the following macro in Outlook (http://www.gmayor.com/installing_macro.htm is aimed at Word, but the principles involved are the similar for Outlook).

Change the name and path of the template (here "D:\Word 2007 Templates\MyLetter.dotx") to that of the letter template you want to use. Open the template in Word and in that letter template insert a docvariable field {DocVariable "varAddress"} at the place you want the address to appear. Then  toggle the view without updating the field (which would show an error) and save and close the template. With a contact selected in Outlook, run the macro and it will open a new document in Word based on the named template and add the address as a variable before updating the field to show the value of the variable in the document.

Sub LetterToContact()
Dim olItem As ContactItem
Dim wdApp As Word.Application
Dim wdDoc As Word.Document
Dim oVars As Variables
Dim sAddress As String
Dim bStarted As Boolean
Dim strMyTemplate As String
strMyTemplate = "D:\Word 2007 Templates\MyLetter.dotx"
Set olItem = ActiveExplorer.Selection.Item(1)
On Error Resume Next
Set wdApp = GetObject(, "Word.Application")
If Err Then
    Set wdApp = CreateObject("Word.Application")
    bStarted = True
End If
Set wdDoc = wdApp.Documents.Add(strMyTemplate)
Set oVars = wdDoc.Variables
sAddress = olItem.FullName
If olItem.CompanyName <> "" Then
    sAddress = sAddress & vbCr & olItem.CompanyName
End If
sAddress = sAddress & vbCr & olItem.MailingAddress
oVars("varAddress").Value = sAddress
wdDoc.Fields.Update
Set wdDoc = Nothing
Set wdApp = Nothing
Set olItem = Nothing
End Sub
