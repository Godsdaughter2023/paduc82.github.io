VLOOKUP - Returns #N/A - Cell Format???? - Microsoft Community

cindym10 say:

VLOOKUP - Returns #N/A - Cell Format????

One sheet contains a master list of all items & descriptions.  Column A = Item #, Column B = Description

Want to pull the item description into a list of random item numbers...My active sheet has the item # entered in column A and the following formula in column B

=VLOOKUP(A1029,'Item Master'!A:B,2,FALSE)

Some of the correct descriptions worked while many others came back with #N/A - In the case that a #N/A was returned if I manually type the item number again (i.e. in A1029) or if I simply double click the cell (A1029) and hit enter then the formula correctly calculates.  However, I am looking at approx 5,000 cells that I would have to either manually type or singularly select...

What am I doing wrong - why did it work on some but not on others????

sloopjohns say:

Cindy - data is a weird thing.  It may look the same to us but it is different to the computer.  VLOOKUP is a function that highlights this.  Often "invisible" or non printed characters make their way into data (think of a leading or trailing space).  This could be the issue.  To fix your immediate issue, i recommend the following:

Create a new column in the Item Master worksheet.  Name it 'Clean Item #"  Then, in this new column, use the trim formula referencing your original Item # column e.g. '=Trim(B2)'.  Copy therelative formula down for all records within the master list.

Now create a vlookup that references the new column but include one other change (in bold) to trim the lookup value.

=VLOOKUP(trim(A1029),'Item Master'!A:C,3,FALSE)

Also note that this formula uses three columns and references column 3.  This is required b/c the added column of trimmed data.

I do this as standard practice where data comes from sources where non printed or invisible characters can be included.  It helps me get more matches.

Also, you can add a Clean function in front of the trim functions for further normalizations:

=clean(Trim(B2))

and

=VLOOKUP(clean(trim(A1029)),'Item Master'!A:C,3,FALSE)

Let me know how it works.
