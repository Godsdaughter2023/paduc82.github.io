That's the kind of thing that's liable to happen if someone unlinks fields in the document (e.g., via Ctrl-A, Ctrl-Shift-F9) instead of updating them (e.g., via Ctrl-A, F9) or by editing the document in, say, WordPad. It's also possible the document has acquired some form of corruption. Corrupt documents can often be 'repaired' by inserting a new, empty, paragraph at the very end, copying everything except that new paragraph to a new document based on the same template (headers & footers may need to be copied separately), closing the old document and saving the new one over it. This still won't recover any fields that have been converted to text, though - you'd still have to recreate them manually.

biddu say: Hi.

You can use a sum of COUNTIFs for all the sheets within a single formula as shown below.  In the below formula I have used cell A2 as the input cell for the string.  You can replace A2 with the actual string within quotes that you are counting in all these sheets. Also, I have used the tab names as provided by you.

=COUNTIF('Logistics1'!$F:$F,A2)+COUNTIF('061410.SOUTH CENTRAL'!$F:$F,A2)+COUNTIF('060210.MID-ATLANTIC'!$F:$F,A2)+COUNTIF('060910.SOUTHWEST'!$F:$F,A2)+COUNTIF('061210.CENTRAL'!$F:$F,A2)+COUNTIF('064410.GULF STATES AREA'!$F:$F,A2)+COUNTIF('Logistics2'!$F:$F,A2)+COUNTIF('061710.NORTH CENTRAL'!$F:$F,A2)+COUNTIF('061110.METRO NEW YORK'!$F:$F,A2)+COUNTIF('064210.MOUNTAIN AREA'!$F:$F,A2)+COUNTIF('060610.MID SOUTH'!$F:$F,A2)+COUNTIF('060710.FLORIDA'!$F:$F,A2)+COUNTIF('060810.MIDWEST'!$F:$F,A2)+COUNTIF('061610.CAROLINAS'!$F:$F,A2)+COUNTIF('062110.EASTERN CANADA AREA'!$F:$F,A2)+COUNTIF('060310.NORTHEAST'!$F:$F,A2)+COUNTIF('061010.NORTHWEST'!$F:$F,A2)+COUNTIF('062310.WESTERN CANADA AREA'!$F:$F,A2)
