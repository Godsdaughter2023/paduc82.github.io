marshall barton say:

If a customer never has two visits on the same day or the visit date includes the time as well as the date, then you can use a TOP subquery to get that:

 

SELECT *, (SELECT TOP 1 X.score FROM table As X
                 WHERE X.customer = table.customer
                      And X.visitdate < table.visitdate
                 ORDER BY X.visitdate DESC) As PreviousScore
FROM table

 

Correlated subqueries like that can be slow on a large table so you should probably make sure the customer and visitdate fields have an index.
