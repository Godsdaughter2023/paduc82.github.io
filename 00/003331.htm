To get rid of the alphas use this UDF in an adjacent column.

 

Function RemAlpha(str As String) As String
'Remove Alphas from a string
Dim re As Object
Set re = CreateObject("vbscript.regexp")
    re.Global = True
    re.Pattern = "\D"
RemAlpha = re.Replace(str, "")
End Function

Usage is  =RemAlpha(cellref)      or     =RemAlpha(cellref)*1     to convert to real numbers
