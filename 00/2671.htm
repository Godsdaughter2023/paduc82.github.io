When a paragraph style includes numbering, all the indent and tab settings on the left side must be set in the Define New Multilevel List dialog; these settings override any settings in the Paragraph dialog.

You cannot access the Define New Multilevel List dialog from the Modify Style dialog; you have to open it directly and make sure that the list level in question is linked to the relevant style.
