The INF folder contains setup information for the Windows Installer. The files there will allow the uninstall of software and drivers you've installed. I can't find a good answer for the differences between the other two folders, other that to say that since Windows 7, DriverStore is the default location for hardware drivers. How to delete "Windows Mail" - Microsoft Community

richardshiel say:

How to delete "Windows Mail"

I have a Toshiba laptop and have "Windows Live Mail" and "Windows Mail", how do I delete "Windows Mail" from my computer,

I have been into Install & Delete programme on control panel, but can't find any title with "Windows Mail" to delete. ?

Rick

john_barnett say:

Hello RichardShiel,

Windows mail is installed with the Windows Vista operating system and therefore cannot be uninstalled. Surprisingly enough it can't even be turned off via the Turn Windows features on or off option in Programs and features.

However, if you have the Ultimate versions of Vista ( please note that I have edited this post as originally I had said Pro and Ultimate; this was incorrect, my apologies) you should be able to turn it off via Group Policies.

1. Press the Windows Button plus R to bring up the Run menu.

2. In the Run menu type: gpedit.msc

3 If you don't have gpedit on your version of Vista an error message will appear. If you do have gpedit on your machine the Group Policy editor will now appear.

4. Once the group policy editor is open look  in the left hand pane and click on the Computer Configuration option. A list will now expand.

5. Next expand the list under Administrative Templates.

6. Look down this list and the expand the Windows Components option.

7. Now look in the right hand pane and scroll down until you come to Windows Mail.

8. Double click on this.

9. You should now see two options. a) Turn off the communities feature and b) Turn off Windows Mail application.

10. Double click on the Turn Off Windows Mail option to open the Properties window.

11. In the properties window click on the radio button marked Enabled, then click on the OK button.

12 Finally Exit properties and then the Group Policy editor.

13. The next time you boot your machine Windows Mail should be disabled.

As i've said this should work if you have the group policy editor.
