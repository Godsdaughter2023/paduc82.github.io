rob koch say:

To provide you more complete and useful information, at one point in the distant past the MSRT was the only tool Microsoft freely provided for this purpose.  However, at that time the tool also didn't allow the selection of a full scan, since it had virtually no visible interface and was only intended for use in a one time pass, typically when included with the monthly Windows Update packages on Black Tuesday each month.  This is also usually the only time that the MSRT definitions are updated each month.

More recently a more complete interface was added, but with mixed results since the tool is now both relatively old and has been superseded by more broadly effective tools such as the Microsoft Safety Scanner, which provides a more complete set of antimalware engines and definitions more similar to those provided within the Microsoft Security Essentials or Windows Defender security products.

The problem with the MSRT tool is that though you are allowed to select a "full" scan, this is clearly a misnomer, since the definitions included with it are a very limited subset, though the time required in order to perform this scan remains roughly the same as for the other Microsoft antimalware based products, since it must still access each file or registry entry contained in the available disk drives.  On top of this, the process will be delayed by any overhead created by an installed antivirus application, since in effect both of these will be operating in parallel.

Though there's nothing truly wrong with using the MSRT in this way, it should now be clear why there are far better tools available to perform the more complete scan you really desire.  The problem isn't lack of function, it's lack of efficiency and effective tool selection.

In general, most here would recommend selecting a manual scanning tool that's entirely different than what is already installed on the PC, since that's more likely to find something your existing antivirus might be missing and so operates as a "sanity check" or second opinion scanner rather than just an alternative method of performing the same exact scan.

Rob
