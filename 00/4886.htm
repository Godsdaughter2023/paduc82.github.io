Using Formulas across Worksheets - Microsoft Community

northwest woman say:

Using Formulas across Worksheets

I am trying to create a file with 12 worksheets, one for each month.  The purpose of the sheets is to track rental amounts due, amounts paid and balances for several addresses. 

 

For Example: 

worksheet Jan 2013:  amount due, amount paid and balance; worksheet Feb 2013: same info but it must reflect the balances from the Jan 2013 worksheet; and so on.

 

Additionally, the first three columns of each worksheet will show name and apt number;  this doesn't change often, but if, for example,  a name changes in March 2013 worksheet, I need that change to carry forward on  the April 2013 worksheet and so on.

 

 I tried really hard to find this in the Help section, but have been unable to find it.  Can anyone give me pointers on this and also tell me where to find the info in the Help section?  Thank you!

ossiemac say:

Hello Northwest Woman,

 

To do what you want you are looking at some complexity with the formulas but if you are reasonably comfortable with this then I will try to explain.

 

However, if you are not comfortable with the complexity, then I suggest that you simply put all of your data in one worksheet and work across the columns for each month. You can hide the columns that you do not want to see. (It is effectively the same as changing worksheets; you will still only see a month of data.)

 

Because you are using dates in mmm yyyy format as the worksheet names, formulas can be used to calculate the previous month worksheet which is what you are going to need.

 

It is then necessary to use Indirect function to reference the previous sheet. Set up a new test workbook and name the worksheets Jan 2013, Feb 2013, Mar 2013 .. . . . . .  Dec 2013.

 

In worksheet Jan 2013, enter some dummy data in column A (10 rows or so will be sufficient).

 

Save the workbook. (The workbook must have a saved name for this to work.)

 

Change worksheets to Feb 2013.

 

For the exercise I am going to stipulate the cells in which to insert the formulas. Normally these will be on the worksheet out of the way to the right of you data. I have used column AA

 

Enter the following formula in cell  AA3 on worksheet Feb 2013.

 

=CELL("filename",A1)

 

The A1 parameter can be any cell in the worksheet. You will see that it returns the full path and filename plus the displayed worksheet name.

 

The following formula has been expanded to extract just the worksheet name. Copy the following formula and paste it into cell AA2 on worksheet Feb 2013

 

=MID(CELL("filename",A1),FIND("]",CELL("filename",A1))+1,256)

 

Now to extract the worksheet name for the month prior to Feb 2013, enter the following formula in cell AA1

 

=TEXT(EDATE(MID(CELL("filename",A1),FIND("]",CELL("filename",A1))+1,256),-1),"mmm yyyy")

 

The first 2 formulas were only examples so that you might be able to follow what is occurring. The only one you really want is the last one that returns the previous months sheet name.

 

I assume column headers in row 1 so copy the following formula and paste it into cell A2.

 

=INDIRECT("'" &$AA$1&"'!"& ADDRESS(ROW(),COLUMN(),4))

 

Drag the formula down and it should return the data that you entered in worksheet Jan 2013.

 

The INDIRECT function converts a string into valid references. Note that because the worksheet names have a space, the string must include the single quotes around the worksheet name and these must be concatenated into the formula. Also the address of the cell references need to be turned into a string from the row and column.

 

Now a little trick you might like to try.

Select worksheet Feb 2013

Click on the tab name for worksheets Feb 2013 and then hold the Shift key and click on worksheet Dec 2013 and it will select all the worksheets from Feb 2013 to Dec 2013 (with Feb 2013 as the top sheet.)

 

Click in cell AA1 and then click in the formula bar where the formula is and without changing the formula, press Enter.

 

Click in cell A2 and then click in the formula bar where the formula is and without changing the formula, press Enter.

 

Now click the tab name of the unselected sheet (Jan 2013) and it will unselect the multiple sheet selection. Check each sheet in turn and you will see the formulas copied to each worksheet and cell AA1 will return the previous months worksheet name.

 

In say worksheet Mar 2013, alter the displayed data in one of the cells in column A and you will see that it is reflected to all sheets after it.

hansv mvp say:

You can add a test:

 

Sub PrintFromList()
    Dim wshInput As Worksheet
    Dim wshPrint As Worksheet
    Dim lngRow As Long
    Dim lngLastRow As Long
    Set wshInput = Worksheets("Data Input")
    Set wshPrint = Worksheets("Print Template")
    With wshInput
        lngLastRow = .Range("A" & .Rows.Count).End(xlUp).Row
        For lngRow = 28 To lngLastRow
            If .Range("A" & lngRow).Value >= 1 Then
                wshPrint.Range("A8").Value = .Range("A" & lngRow).Value
                wshPrint.PrintOut
            End If
        Next lngRow
    End With
End Sub
