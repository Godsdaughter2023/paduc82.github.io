ken sheridan say: The OutputTo method does not have a WhereCondition argument.  To restrict the result set of the query you'd need to include a parameter on the CourseType column which references the cboType control on your form.

You'll find a file InvoicePDF.zip in my public databases folder at:

https://skydrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

which outputs a report to a PDF file in this way.  The principle is much the same for outputting a query to an Excel file.  If you amend the relevant lines of code in the click event procedure of the 'Create Invoice as PDF File' button on the form to:

            strFullPath = varFolder & "\" & Me.Customer.Column(1) & Me.InvoiceNumber & ".xls"
            ' ensure current record is saved before creating Excel file
            Me.Dirty = False
            DoCmd.OutputTo acOutputQuery, "qryPDFInvoice", acFormatXLS, strFullPath, True

it will output the query for the current invoice to an Excel file.

Hi,

Welcome to Microsoft Community and thank you for posting the question. As I understand you are having tough time importing the contacts from the CSV file into Outlook 2013.

 

Let me help you in providing information on the error ‘Cannot complete because service provider does not support’ which would give you fair idea for the cause of the error .

 

This error is the result of trying to import into an Exchange ActiveSync (EAS) account. The EAS service provider (Outlook.com / Hotmail) does not support importing (or moving), only copying into the folder.

 

If you need calendar or contacts imported, you’ll need to import them into a local PST file then copy the appointments or contacts into the EAS account’s calendar or contacts folders.

 

To troubleshoot the issue related to Outlook, you may try the steps below:

  • Use a list view, especially in a Calendar, then select all and right-click, drag and copy the items to the EAS account.
  • If the items you copied are not synced up to the server after a few minutes, press F9 or the Send and Receive button.
