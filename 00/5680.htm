Actually, yes there is.  It is a feature called "Tags".

You can also see the tags, or add or change them in the File menu > Info command > Info Pane:

You can then see the tags in Windows Explorer:

click on the Search Window to display the Search TAB where you can select the option to search by tag names:

That is all of the "good news". In theory you can apply tags to files and subsequently view and even search by them.  You have to have Windows "Indexing" turned on to do this search.  

Bad News

Well, that is the theory. Unfortunately I was not able to get it to work consistently for me in the past (on Win 8).  So I stopped trying to make it work.

More Good News

I fell back to an "old school" technique to do the same sort of thing.  I add these "tags" directly to the File name.

I do it a couple of ways.  I add the article date to the file name as a prefix. That way when I do an alphabetical sort the files are displayed date.  I added the date to the name because the date the article is created/published is not the same as the date I create the file. And on IE favorites (which are Windows shortcuts) the date create does not stay stable, so if I do not want to copy and save the text of an article, but want to be able to find it again, I create an IE favorite with the date embedded.

As well as the date, I add text "tags" to the file name.  These are short forms of words that I can use to search for.  If I want to categorize a file sort, I put the most important "tag" first in the file name, followed by the date. That way I see all of the related articles in the same place in Explorer, sorted by date.  If the article fits into more than one "tag" category, I can add as many tags as I wish.  Whether I put those "extra" tags at the start or end of the name is optional.  Finally, I use hyphens to separate my tags from the file name content.

Let's say you want formulas in B1:B10 that refer to Sheet1!A13:J13.

Option 1:

Select B1:B10.

Enter the formula =TRANSPOSE(Sheet1!A13:J13)

Press Ctrl+Shift+Enter to confirm the formula; this turns it into an array formula.

Option 2:

Select B1.

Enter the formula =OFFSET(Sheet1!$A$13,ROW(B1)-ROW($B$1),0)

Fill down to B10, or as far as needed.
