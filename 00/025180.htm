In WORD 2016, how do I make all page headers the same? - Microsoft Community

robertgjudge say:

In WORD 2016, how do I make all page headers the same?

I have received a Microsoft WORD document that has many sections.  Each section has the same header.  I want to change the page header for the entire document.  However, it looks like because of the sections, even though each section has the same header, I would have to go into each section separately and change the header in each section one at a time.  That would be very time consuming.

How can I change the header so that the header in each section changes at the same time, rather than having to change each section header one at a time?

I will appreciate help.

jay freedman say:

If all the headers after the first one have Link to Previous turned on (in the Header & Footer Tools ribbon, which appears when the cursor is in the header or footer pane), then you only have to change one section. See http://wordfaqs.ssbarnhill.com/HeaderFooter.htm, particularly the parts about "Types of headers and footers" and "What happens when you have more than one section".

If either or both of the "Different First Page" and "Different Odd & Even Pages" check boxes are checked, you'll have to change each of those types of headers separately. The Link to Previous applies only to headers and footers of the same type.

The only time you would have to visit the header(s) in every section is if the original author didn't use Link to Previous and instead pasted a copy of the header into each section. Even in that case, though, you wouldn't have to edit the headers; just turn on Link to Previous (and OK the prompt asking whether to discard the existing header).
