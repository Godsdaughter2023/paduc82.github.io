cyndie browning say:

<< I'm using styles to format the text, but it's redefining the style very often when I don't want it to. >>

(1) Are all your styles in the document based on Normal style?? If not, they should be, to create consistency and so that the appearance of your other styles do not change just because you've copied formatting from anywhere in the document or pasted in text from one page to another.

For instance, if you have a style MyParagraph1 that's based on Body Text, and then you make a change to Body Text style, it's going to change the appearance of MyParagraph1, too. All of your styles should be based on Normal style, with additional style features added in and updated.

(2) Under File/Options/Proofing/AutoFormat As You Type, under the subheading "Automatically as you type," is there a check in the box next to "Define styles based on your formatting"?? If so, UNcheck that box so that _you_ have control of how your styles look instead of letting Microsoft's idea of the way styles should look make that decision for you.

(3) Make sure that you do NOT have "Automatically update" turned ON for any style (except TOC styles). Otherwise, any change to one paragraph could change the appearance of any or all the rest of your paragraphs.

I can see that you have input values for the % work complete. Once you do this, MSP will be forced to give the tasks an actual start start date and an actual duration, and what was previously a scheduled start date will become the actual start date. You can see this in the tracking table. MSP will not change actuals after you have input them.

I suspect (because you haven't mentioned it) that you assigned the predecessors after you input the % work complete, that is, after the start date was locked in. You have forced it to ignore the predecessors, or to put it another way the predecessors can no longer have any effect. They are just a decoration at this point.

Remove the progress. Delete the actual start date, actual duration and actual work in the tracking table. The tasks will all take up their scheduled position according to the predecessors. Then you can re-do the progress.

graham mayor say:
The addresses are placed on the envelope with the envelope address and envelope return paragraph styles in the current document template. If you add an envelope to a document you can easily modify the envelope return paragraph style to include a negative indent as suggested by Doug. Then save that indent as part of the style.
