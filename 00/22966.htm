mel77084 say:

as of recently I keep seeing under Windows Update the following yellow hint: "Einige Einstellungen werden von Ihrer Organisation verwaltet" (German) - Insider Updates seems to be 'Stuck' and wont install. This machine belongs to me (private) and is not under any commercial domain. please advise.

"Some settings are controlled by your organization" is meaningless for you ... and that it seems to be stuck (if the upgrade is build 15007) is the norm ... before the download actually begins, it usually indicates (incorrectly) that the download has reached 100% (that is referring to the Windows Defender definitions download that preceded it), and what is actually going on is it is checking to see what files will need to be downloaded (only the new ones) when the download begins. Eventually, after that has finished, the download progress indication will change to 0%, and within a few minutes, it will begin to show the download status beginning at 1%.

If this all seems ridiculous, it is ... you'll have plenty of time to go to lunch, work in the garden, take a nap, etc. If, after a few hours, nothing seems to be happening, it won't hurt to restart the machine and check for updates ... again.

The good news is that, due to the few hours of delay, the actual download will be somewhat smaller than we were used to getting, so will take 15 minutes or so less.

Once the download finishes, it only takes a few minutes to start "preparing to install", and again, the indication of progress is phony ... it will seem to have stopped somewhere around 75%, but then, suddenly gives the restart now, etc.

All this is to allow for a slightly smaller download for those need to limit their downloaded data, or who have slow internet service. Very considerate of Microsoft, eh?

Why will my "Find Related Messages in the Conversation" - Microsoft Community

kimberlykd say:

Why will my "Find Related Messages in the Conversation" feature not pick up all of the emails?

The last few days this feature has become more and more unreliable. My organization has recently switched to Office 365 so I suspect this may have something to do with it.

The search works sporadically but usually will not include the email I am searching on, or anything from the inbox.

I've also noticed that my Quick Steps are not functioning from the search screen.

I use both of these features all day long to manage email and am at a loss. Does anyone have any suggestions that might repair this issue?

barry xie msft say:

Hi KimberlyKD,

 

Thanks for choosing Office 365.

 

Regarding “Find Related Messages in the conversation,” it’s a kind of search function in Outlook client. According to your description, this feature may encounter some problem when you search emails for a same title or from a specific email address.

 

Since you are facing a problem about search function in the Outlook client, I recommend that you follow the steps below to rebuild your Outlook client’s index. Here are the steps:

1. Open your Outlook client. Select File > Options > Search > Indexing Options.

2. Click “Modify” to check if “Microsoft Outlook” is marked. If it is marked, you should also try disabling and then re-enabling Outlook. To do so, follow these steps:

a. Deselect Outlook from Indexing.

b. Close the Outlook client.

c. Wait 5 minutes, check in the task manager that Outlook.exe is not in the processes tab.

d. Now open the Outlook client again, go back and select Outlook for indexing

3. Click “Advanced” on the “Indexing Options” page. Select “Rebuild” button on the following page.

Note: When the index is being rebuilt, your search will not work. You should wait until the index is done rebuilding. The time it will take for the index to be rebuilt depends on the system you are using, the amount of files that need to be indexed, and file size.

 

Meanwhile, I recommend that you try using the Conversation feature on the Outlook client. Navigate to View > Messages > select “Show as Conversation” button and all emails with the same title will be grouped in a conversation. The messages within each Conversation are sorted with the newest message on top. When a new message is received, the whole Conversation moves to the top of your message list. About searching emails from a specific email address, you may try manually type the email address in the search field as a temporarily workaround.

 

If the problem still persists, please provide the following information for us:

1. What version of Outlook client do you use? It looks like Version 1609 Build 7369.2130.

2. Do any other users in your company encounter the same problem?

3. Some screenshots about the symptom.

 

Thanks,

Barry
