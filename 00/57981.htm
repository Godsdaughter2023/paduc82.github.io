You say you updated from Windows 7 to Windows 10.  How did you do that?

If you used the two ways I know of to do that you should have not been prompted for your product key.

The first way to do it would be to wait for the "Get Windows 10" to prompt you to do the update.

The second way is to use this MediaCreation tool and use the "Update This PC" option.https://www.microsoft.com/en-us/software-download/windows10

The first upgrade has to be done while running from the old operating system.

What is happening is that during this update the update software is checking that your license is valid, and then it is also going to register your device with the Microsoft register server (I believe it is done when you first log in using your Microsoft Online account on that device).

That is what makes your Windows 7/8/8.1 product key valid in Windows 10.

If you on the other hand tried to just start by doing a clean install of Windows 10 using the software above and using your old Windows product code I think it will not be valid.

After the update from your old operating system to your new one is registered then you can actually then use the above software to do a clean install on that machine and skip entering the product Id and when you get to the point where it asks for your Microsoft Online account name/password it will automatically activate it.

If you did use one of the two processes I mentioned to update then the other possibility is Microsoft's register servers.  On July 29th and maybe for few days they were having problems with it where for some people it wouldn't register/activate, and that is where the advice to wait a few days came in, as in they would fix the problems with their servers, and it would automatically activate.  I don't see people complaining about that now so I got the impression that it has been fixed, but I certainly have no real insight into that.

You will hear the standard power on beep, but keep holding the buttons untill you hear a second beep.

Now release the buttons and watch your xbox getting started in the lowest resolution possible.

From here you can go to the settings to change them back to correct values.
