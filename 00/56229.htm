No. I think you'll find that the paperclip shows each time there is an attached file, whether you regard that as an 'actual attachment' or not. Attachments sometimes have a Content-Disposition: header, which will usually be either attachment or inline; it's this latter that is probably confusing you, because an image in a signature will be displayed inline in Windows Live Mail and probably all other modern mail clients. It's an attachment, nonetheless, so it will generate a paperclip icon in the message list.

You may be able to use the message size as a guide to whether there's an 'actual' attachment or not.
