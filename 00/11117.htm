    Hi Miriam,

     

    I understand that you need to change the font size while printing a webpage.

     

    1.       Is the font size on the website readable on Internet Explorer 10?

    2.       Does this happen with all the websites?

    3.       With which version of Internet Explorer 10 (Modern User Interface or Desktop) does this happen?

    4.       What is the make and model of your printer?

     

    When you print a webpage, Internet Explorer shrinks the content of the webpage to fit the width of your paper. If the webpage is wide, reducing the text size to fit the page can cause the text to be unreadably small. To fix this, try one of the following:

     

    Let’s try these methods and check if the issue persists.

     

    Click the Tools button.

    Point to Print, and then click Print preview.

    From the Change Print Size menu, choose Custom.

    Specify how large you would like the webpage to be printed by setting a percentage in the Custom Size text box.

     

     

    Note : This will enlarge the printed size of the entire webpage, but it might result in some of the webpage being cut off on the printed document.

     

     

    If the issue persists you can zoom the webpage on the Internet Explorer 10 and then take print out.

     

    Method 2: Zoom the webpage on Internet Explorer 10 and then take a print out.

     

    You can follow these steps to increase the fonts of the Internet Explorer 10 and then try printing the webpage and check if the fonts on the print out are still small.

     

    Press “Ctrl” and “+” keys on the keyboard to increase the font.

     

    If that doesn’t help, then try the next method and check.

     

    Method 3: Make the text on your screen larger or smaller

     

    Refer the article: “Make the text on your screen larger or smaller” to change the font size larger from the given link and then try printing the webpage and check.

     

    http://windows.microsoft.com/en-us/windows-8/make-text-screen-larger-smaller

     

     

    Hope it helps.
