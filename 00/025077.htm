joshabilly say:

vba Isnumeric, Case and zero

Hi, I'm coding in Windows 10 using Office 365, Access and VBA and hit a problem when using Isnumeric in a Case statement. The code below illustrates the issue

Public Sub Foobar()
  Dim var As Variant
  var = 0
  If IsNumeric(var) Then Debug.Print "var is numeric"
  Select Case var
    Case IsNumeric(var)
      Debug.Print "var is numeric"
    Case Else
      Debug.Print "var is NOT numeric"
  End Select
End Sub

This code outputs the the following to the immediate window.......!!

var is numeric
var is NOT numeric

hansv mvp say:

Your Select Case statement compares the value of var with IsNumeric(var). It is equivalent with

    If var = IsNumeric(var) Then
        ...
    Else
        ...
    End If

Since 0 does not equal True, the Else part is executed. If you want to do this with Select Case - not very useful, but OK - you can use:

    Select Case True
        Case IsNumeric(var)
            Debug.Print "var is numeric"
        Case Else
            Debug.Print "var is NOT numeric"
    End Select

or

    Select Case IsNumeric(var)
        Case True
            Debug.Print "var is numeric"
        Case Else
            Debug.Print "var is NOT numeric"
    End Select
