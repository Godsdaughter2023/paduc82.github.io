john korchok say:

Your version is out of date. In PowerPoint, choose Help>Check for Updates, then click on Check for Updates. Possibly an update will help the videos that are not playing.

A presentation saved to video normally does not include the narration. You can extract the audio, then include it with the video when saving it. Please follow these steps to extract the audio:

  1. Close PowerPoint. Make a copy of your presentation and rename it, adding .zip to the end of the file.
  2. Right-click on the file and choose Open With>Archive Utility. The file is unzipped.
  3. Archive Utility creates a folder with the same name as the presentation. Open it.
  4. Open ppt>media.
  5. Your narration will be in a WAV file. If you only have one narration and no music, then there will only be one file. If there are more, play the files to find the narration.

Now you can re-export the movie:

  1. Open the presentation in PowerPoint.
  2. Choose File>Save as Movie.
  3. Click on the Movie Options... button
  4. Beside Background soundtrack, click the dropdown and choose Select soundtrack.
  5. Choose the WAV file that holds your narration.
  6. Click OK, then Save.

It's possible you may need to edit the soundtrack to get it to fit the altered timings in you presentation. You can do this by downloading the free audio editor Audacity from here: Audacity for Mac Download

tychicusthefriendly say:

If you search this forum there are a few people with questions on how to set up a dual boot.  You might have to go back a couple of pages but they are in here, I know I have answered about three of them.  If you don't want to search them out I will give you the quick and dirty.

  1. Create a partition on the drive you want to install Windows 10
  2. Burn the .iso image to a 'disk', either a DVD or USB drive.  This is simple in Windows 8/8.1 because all you have to do is right click the .iso and select the option to burn to disk.  I think it is the second one down.
  3. Restart your computer and make sure it is set to boot from your 'disk', either the usb drive or the DVD.  You might have to go into your BIOS to set the boot order.
  4. When installing Windows 10 make sure you pick custom instead of express.
  5. When it asks you where to install make sure you pick the partition you created.  If you pick any other drive/partition data will get over written.
  6. When you start your computer it will ask which OS to boot from, it might need to restart to boot from the OS you did not use last.

As for the size of the partition, Windows 8 needed at least 20 GB for it's preview so that is the least I would recommend.  If you want to install more programs then you will need a bigger partition.
