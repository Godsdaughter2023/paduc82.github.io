brook zhang msft say:

Hi Edward,

 

When you run the command Convert-MsolDomainToFederated –DomainName domain_name.com, not only will the domain be changed to federated but also an online instance containing the AD FS server URL will be created online. If you try to login to Office 365 at the login page, Office 365 will return you a URL of the local AD FS server (based on the online instance).
