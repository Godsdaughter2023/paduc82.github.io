roady [mvp] say:

No, you cannot do that since rules execute upong send/receive time. If you want to execute rules against messages that have already been received, then you'll have to run the rule manually.

In your case, I would recommend creating a Search Folder or a custom view. This allows you to filter out the messages that only the messages are shown which are unread and older than 2 weeks. From the list generated, you can easily move all the messages found to a different folder.
