marshall barton say:

I think Hans is not stating it strongly enough.  For desktop applications, VBA code is faster, more concise, more versatile, more easily documented, easier to debug, etc, etc, etc.  AFAICS, there is no reason to conert for an old app to use, much less convert to, macros.

If a desktop app is going to be convertd to a web app, then you will have a whole lot more to worry about than just not being able to use VBA.  Since you will be redoing so much of the app, converting VBA to macros in anticipation that it might be nice to run on the internet is just a big waste of time.

For an internet application, preventing damage to the system running the app in a browser is the driving force.  Here the versatility of VBA and all the libraries it can use could unintentially or maliciously run amok just by viewing the app's startup page.  Therefore, only the "safe" macros allowed in a web database can be used.

hansv mvp say:

If you want to use the databases directly, I wouldn't bother converting VBA code to macros. Although the macro editor has been improved a lot, especially in Access 2010, it is still more cumbersome to use than the Visual Basic Editor (in my opinion).

If you want to convert the desktop databases to web databases, you must convert the VBA code to macros, for VBA won't work in a web database.

bernie deitrick say: Create a range named "Equipment" in an otherwise empty Excel file, and enter a list of all your equipment short names - from your example, they would be (I think):

Copy the first macro below and paste it into a codemodule in that new workbook. Then save the file and run the macro - after, you can check columns B through E against the original data in A - when you are sure the macro works, uncomment the line at the bottom as directed... the code will create a new workbook with a date-time stamp in its name. If you need the dates "Sep, 2012" converted to real dates, then use the second macro below instead.

Option Explicit
Sub ImportTXTMacro()
    Dim rngC As Range
    Dim rngA As Range
    Dim strTemp As String
    Dim strVal As String
    Dim i As Integer
    Dim myFN As String

    'Get the filename
    myFN = Application.GetOpenFilename("TXT Files (*.txt), *.txt")

    Workbooks.OpenText Filename:=myFN, _
                       Origin:=437, StartRow:=1, DataType:=xlDelimited, TextQualifier:= _
                       xlDoubleQuote, ConsecutiveDelimiter:=False, Tab:=True, Semicolon:=False, _
                       Comma:=False, Space:=False, Other:=False, FieldInfo:=Array(1, 1), _
                       TrailingMinusNumbers:=True

    With Range(Range("B1"), Range("A65536").End(xlUp)(1, 2))
        .FormulaR1C1 = "=TRIM(RC[-1])"
        .Value = .Value
        For Each rngC In .Cells
            If rngC.Value = "" Then rngC.Clear
        Next rngC
        With .SpecialCells(xlCellTypeConstants).Areas
            For i = 3 To .Count
                If .Item(i).Cells.Count = 1 And .Item(i).Cells(1).Value <> "-------" Then
                    For Each rngC In ThisWorkbook.Worksheets(1).Range("Equipment")
                        If Not IsError(Application.Find(rngC.Value, .Item(i).Cells(1).Value)) Then
                            .Item(i).Cells(1).Offset(0, 1).Value = rngC.Value
                            .Item(i).Cells(1).Offset(0, 2).Value = Trim(Replace(.Item(i).Cells(1).Value, rngC.Value, ""))
                            GoTo NextArea
                        End If
                    Next rngC
                End If
                If .Item(i).Cells.Count = 1 And .Item(i).Cells(1).Value = "-------" Then
                    .Item(i).Cells(1).Offset(0, 1).Value = "-------"
                    GoTo NextArea
                End If
                If .Item(i).Cells.Count >= 1 And .Item(i).Cells(1).Value <> "-------" Then
                    For Each rngC In .Item(i).Cells
                        rngC.Offset(0, 1).Value = Trim(Left(rngC.Value, InStr(InStr(1, rngC.Value, " ") + 1, rngC.Value, " ")))
                        rngC.Offset(0, 2).Value = Trim(Replace(rngC.Value, rngC.Offset(0, 1).Value, ""))
                        If Not IsNumeric(rngC.Offset(0, 2).Value) Then
                            strTemp = rngC.Offset(0, 2).Value
                            strVal = Trim(Left(strTemp, InStr(1, strTemp, " ")))
                            rngC.Offset(0, 2).Value = Val(strVal)
                            rngC.Offset(0, 3).Value = Trim(Replace(strTemp, strVal, ""))
                        End If
                    Next rngC
                End If

NextArea:
            Next i
        End With
    End With

    Columns("A:E").EntireColumn.AutoFit
    'un-comment the next line after you have fully tested the code
    'Columns("A:A").Delete
    ActiveWorkbook.SaveAs ThisWorkbook.Path & "\Imported data " & Format(Now, "yyyy-mm-dd hh-mm") & ".xls"
End Sub
