aaron zhu msft say:

Hi David,

There are two types of rules in the Outlook desktop client — server-based and client-only. For the server-based rules, they will be synced to the server side. For the client-only rules, they will not. Client-only rules are rules that run only on your computer. Here is an article for your reference: Server-based rules versus client-only rules.

As a tip, the client-only rules display “(client-only)” besides them when you manage your rules.

But if you set up the inbox rules in OWA, they are stored in the server side directly and will be synced to the client side automatically.

However, inbox rules of the Outlook client have more rich conditions and actions than inbox rules of OWA, for some specific requirements, you need to use the client-only rules to achieve them. For example, if you want to use a inbox rule to print the specific emails, you need to create a client-only rule in the Outlook client other than OWA.

Feel free to let us know if anything is unclear.

bob jones aka: cybertaz say:

Paper size is a property of the document, established when the document is created. If the document is defined as US Letter Word cannot arbitrarily open it as a different Paper Size regardless of what your Default Paper Size is set for. The Default pertains only to new documents you create.

Once you open the document you can change the document dimensions if you wish, but keep in mind that it will cause the content to reflow based on the new dimensions. In all probability that will have a decidedly negative impact.

martinmc cartan say:

filling in a pdf form/document

can i use Dobe  Acrobat to fill in a downloaded pdf form /document ,? or do i have to download new software?
graham mayor say:
If the downloaded form has been designed to be filled in on line, then you can do so in Adobe Reader. If not then you can probably convert it into a form with Adobe Acrobat, some versions of which - certainly Acrobat X Pro - have the ability to create fillable forms from PDF forms. Though it works quite well, it is still probably not worth the effort for a one-off.

Make sure too that any form you intend to fill - particularly with respect to job applications - does not require you to fill the form with a pen and return it. Failure to follow simple instructions will guarantee that the form only travels as far as the recipients waste paper basket.
