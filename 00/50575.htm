How To Find SSID (Network Name)

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Wireless.
  4. From the submenu, click on Basic.
  5. The SSID is located where it says SSID.

 

Wireless Passphrase

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Wireless.
  4. Select Security from the submenu on the left.
  5. The Wireless Passphrase will be listed next to "WPA Pre-Shared Key" or "Network Key 1-4".

 

Changing Wireless Security Type/ Disabling

  1. Open a web browser.
  2. Enter http://192.168.1.1  or http://192.168.1.254 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Wireless.
  4. Select Security from the submenu on the left.
  5. In the drop down box labeled “Network Authentication” you can disable or change the wireless security.
  6. If WEP is selected, select the WEP Encryption Strength (either 64-bit or 128-bit).
  7. If WPA-Personal is selected, select the WPA Mode (Auto, WPA or WPA2) and the WPA Encryption (TKIP, AES or Both). WPA2 with AES is recommended.
  8. Once the passphrase has been entered click Save/Apply.

 

MAC Filtering

**Filtering is only available when bridging PVC is Configured**

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter admin for the user name and admin for the password.
  4. In menu to the left, select Advanced Setup
  5. In the sub-menu that populates, select Security.
  6. In Following sub-menu select MAC Filtering
  7. Choose add or remove to configure the MAC filtering rules
  8. Select one or more conditions (i.e. Destination, Source, Frame Direction)
  9. Click “Apply” to activate the filter.

 

Changing Wireless Channel

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Wireless.
  4. Select Advanced from the submenu on the left.
  5. From the drop down menu labeled Channel select the desired channel. (Channels 1,3,9,11).
  6. Select Apply/Save

  3. From the menu on the left, select Wireless.

  4. Select LAN from the submenu on the left.

  6. Enter the MAC Address and the IP Address of the Xbox.

  2. Enter http://192.168.2.1/ or http://192.168.254.254/ into the address bar. (Or click the link) When prompted enter the admin for the username and admin for the password if the gateway asks.
  3. From the menu on the left, select Advanced Setup.
  4. Select NAT from the submenu on the left.
  5. Select Virtual Servers on the next submenu.
  6. Select the Add button.

  8. Enter the IP Address of the Xbox in "Server IP Address".
  9. Input 3074 for the External Port Start/End and Internal Port Start/End.
  10. Select TCP/UDP for the Protocol.
  11. Select Apply/Save.

  4. Select NAT from the sub-menu on the left.

There are several things to try:

 

·         The device and your computer could be at the edge of their range or too close to each other. Move the device closer to your computer or farther away. The device information should tell you the correct distance.

 

·         The device might be too busy or have too many connections. For example, other devices might be connecting to a Bluetooth enabled printer, preventing this connection. Wait until the device is less busy, and then try again. Or, to reduce the number of connections, set the other devices so that they are not discoverable. If you're not sure how to turn off discoverability, check the information that came with the device or go to the manufacturer's website.

 

·         If the device is close to other devices that use radio frequencies (such as microwave ovens, cordless phones, remote controls for electronics or lighting, or 802.11 wireless networks), those devices might be creating interference. Try moving the device that isn't working farther away from the other devices.

 

·         Security might have been reset on either device. This is especially common with keyboards and mice. Other devices can have similar problems and might need to be reinstalled.

UnInstall Bluetooth drivers and re-install Bluetooth drivers meant for Windows 7 or Windows 8.0 in "Windows 7 Compatibility Mode".

I had tried all suggestions and troubleshooting steps including disabling power saving options, doing a factory reset and clean install of Windows 8.0 / 8.1 but I continued to have problems with Microsoft Bluetooth mice (w/o USB dongles) randomly disconnecting. The ONLY solution that worked for me was un-installing existing Bluetooth drivers from Control Panel as well as Device Manager and re-installing Bluetooth drivers meant for Windows 7 or Windows 8.0 in "Windows 7 Compatibility Mode". The key here is installing Bluetooth drivers in "Windows 7 Compatibility Mode" by right-clicking the Bluetooth EXE file > Properties & selecting "Windows 7 Compatibility Mode". Avoid Bluetooth drivers meant for Windows 8.1. Luckily, my touchscreen laptop manufacturer (Acer) had both Windows 7 & Window 8.0 Atheros Bluetooth drivers for my model on their support website.   3. Windows Power Management started working perfectly with the laptop going into sleep / standby nicely and waking up instantly without any lockups or WiFi issues
  4. CPU / Fan noise is much quieter-- it seems the Bluetooth drivers installed as I suggest are more "in harmony" with the rest of the system.
  5. Everything including the Bluetooth mouse is working great for almost a week since I made these changes. I promptly updated my complete system image backup with the new setup.
