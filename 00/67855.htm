I've read some evidence online that this may be caused by a corrupted or replaced system file.

Try starting an elevated CMD window (i.e., Run As Administrator) and running:  SFC /VERIFYONLY
If it reports any problems you may want to use SFC to restore your protected system files:  SFC /SCANNOW

If you have complete faith in Microsoft's system file checker, you can skip the /VERIFYFILES and just do the second command, as it will both verify the files and automatically restore any file that's been corrupted or replaced.  I've seen no reason to distrust SFC, but I like to know what's going to happen before it happens.

Try these steps in order to fix the issue.

 

  1. Go to start, type in the program "msconfig" (Without the quotes.)
  2. Select the button "Diagnostic startup", and then click apply, then reboot when it prompts you to do so.
  3. After logging in again, run the program services.msc, and enable the Windows Update service, BITS service, along with the RPC service. (Also depending on your situation, you may want to enable the WiredAuto config service or the WirelessAuto config serivce.)
  4. Once the services you need are running, open the Windows Update program, and install your updates like normal.
  5. Once they are completed, run the program msconfig again, and this time select the "Normal startup" button, and apply, then restart.

System requirements

If you want to upgrade to Windows 10 on your PC or tablet, here’s what it takes.

Latest OS:  Make sure you are running the latest version either Windows 7 SP1 or Windows 8.1 Update.

For Windows 8.1, see the following article:

http://windows.microsoft.com/en-us/windows-8/install-latest-update-windows-8-1

For Windows 7, see the following article:

http://windows.microsoft.com/en-US/windows7/learn-how-to-install-windows-7-service-pack-1-sp1

Processor:  1 gigahertz (GHz) or faster processor or SoC

RAM:  1 gigabyte (GB) for 32-bit or 2 GB for 64-bit
Hard disk space:  16 GB for 32-bit OS 20 GB for 64-bit OS
Graphics card:  DirectX 9 or later with WDDM 1.0 driver
Display:  1024x600

How much actual free disk space is required by Windows 10 setup?

Windows 10 uses approximately 10 GBs of disk space. The minimum disk space available is to facilitate the installation process. Some install files are compressed and need to be expanded during setup. Some files are temporary and are deleted during or after installation. Also, because you might have data and apps on the hard disk, they will be moved around during migration.

msvcp90.dll errors could indicate a registry problem, a virus or malware issue or even a hardware failure.

Method 1: As a first step, boot into Safe mode and try to install the program and check if you can install the program successfully.

Microsoft Visual C++ 2008 Redistributable Package (x86)

Verify whether you are able to install the program successfully after this.
