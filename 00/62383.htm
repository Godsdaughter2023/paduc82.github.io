steve winograd say: When more than one network connection is available, Windows uses the one with the lowest metric value.  By default, it automatically assigns a metric value based on the network connection's rated speed.  See An explanation of the Automatic Metric feature for Internet Protocol routes .

To force Windows to use a specific network connection, assign a metric value to each one, giving the lowest value to the desired connection:

1. Open the Network Connections folder (Start > Run > ncpa.cpl)
2. Right click the desired connection.
3. Click Properties > Internet Protocol Version 4.
4. Click Properties > Advanced.
5. Un-check "Automatic metric".
6. Enter a number between 1 and 9999 for the "Interface metric".

sandm1tch say:

If HDMI to DVI doesn't work, next best thing is getting the HD VGA cable for Xbox 360 and use that in 720p/1280x720 or the component cable at 720p. The reason I recommend 720p instead of 1080i is for a couple of reasons.

-

1. You have an LCD projector which is a digital progessive type of display. Interlace is not suited for this type of display (interlace was intended for CRT displays), but if you can't use a digital connection, then analog is the only way (but with progressive formats like 720p instead of 1080i which is interlaced).

-

2. Progressive allows for faster display of your controller button presses on screen than over interlace. Progressive shows the 30 or 60 frames in one 60 hertz field each second. Interlace splits those same 30 or 60 frames into seperate 30 hertz fields to be combined into 60 hertz. It takes longer because only half of the resolution (why people say its actually 1920x540 instead of 1920x1080/1080p) is flickered each second.

-

So in summary, you want to try and use Progressive resolutions with LCD, Plasma, and DLP. It will make your actions less delayed onscreen and less blurring will occur. This is why 1080i is removed from Xbox one. But with an analog connection if you can't get the HDMI to DVI cable working...

zigzag3143 (ms -mvp) say:

This is from Microsoft


In previous OS's the dxdiag tool reported the DirectX level as the highest level that the OS supported.   In Windows 10 that was changed to show the highest Direct X level that the video card driver installed supports.  There has been some confusion in the forums with customers thinking that Windows 10 does not have DirectX 12.


Your best bet is to refer to (NVidia\AMD\Intel) sites to see what cards\drivers support full DX12.
