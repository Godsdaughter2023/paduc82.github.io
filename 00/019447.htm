brian spittle say:

Resolved:

For posterity:

You cannot have an auto-reply (AutoReplyState) AND a forward rule (ForwardingSmtpAddress) when DeliverToMailboxAndForward is FALSE.  Changing DeliverToMailboxAndForward to TRUE resolves this.

It appears the trigger for an auto-reply is that the email lands in the mailbox.  If it is never actually delivered, there is no response.

graham mayor say:
The key in Works 2001 suite is for the Word part - in this case Word 2000. Word 2000 may work with Windows 7, though it is not a recommended combination, and the chances of getting a replacement disc or a legitimate download of it are slim, but if you find one, your key should still work - but whether Word will work without problems is open to debate.

The Works version is version 6. This too has long been replaced by versions up to version 9. That version is still available for download from http://msft-dnl.digitalrivercontent.net/msoffice/pub/X14-02658/X14-02658.exe and will install and work on Windows 7, but it does not include Word.
