ken sheridan say: I take a different view with regard to time durations from that expressed here.  There is no reason why a time duration should not be stored as a date time data type, and consequently entered as hh:nn, provided that the individual values in the table are each less than 24 hours.  In fact it is also possible to store longer individual durations as a date/time data type, but a little more work has to be done on the interface.

The important thing to bear in mind is that the date/time data type is no more than a 64 bit floating point number which counts days (the integer part) and the fractions of a day (the fractional part).  When used to express a date, whether with or without a time of day, the count is from 30 December 1899 00:00:00, either towards positive infinity in the case of dates after that value, or towards negative infinity in the case of dates before that value (in my line of work such early dates are commonly used).

Consequently a duration of less than 1 day can be stored as value of date/time data type, as it merely represents the duration in days, e.g. 4 hours and 30 minutes is 0.1875 days.  To show the value in hours and minutes it simply has to be formatted as 'hh:nn'.  

With a little bit of processing such values can be aggregated and, even where the aggregated value exceeds 24 hours can be expressed as hh:nn.  This cannot be done by simple formatting as 'hh:nn' as this would only show the hours and minutes represented by the fractional part of the value, e.g. 28 hours and 30 minutes, whose underlying value in date/time data type is 1.1875 would still be shown as 04:30 rather than 28:30.  It is not difficult to write a little function to do the latter, however, and you'll find such functions illustrated in TimeArithmetic.zip in my public databases folder at:

https://onedrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

Note that if you are using an earlier version of Access you might find that the colour of some form objects such as buttons shows incorrectly and you will need to  amend the form design accordingly.  

If you have difficulty opening the link copy its text (NB, not the link location) and paste it into your browser's address bar.

In the same OneDrive folder you'll also find TimeSheet.zip, which applies the functions to a simple time-keeping database.

In your case the durations could thus be entered as 04:12 for 4.2 hours, but rather than having to mentally calculate this each time I'd suggest you create table similar to the TimeSlots table in my TimeSheet demo, but with slots of 6 minute duration rather than 15 as in my case.  If you prefer to select the values in the 4.2 format, just add another column to the table with the equivalent value in decimal notation, so the row with this example would be:

04:12:00    4.2

To enter the times you can then use a combo box with a RowSource which returns both columns, so if you want to store the durations as date/time values, but select them in decimal notation the BoundCoumn property of the combo box would be 1, and this column would be hidden by setting the combo box's ColumnWidths property to 0 (zero).

With the durations stored as date/time values, if you want to use them as hours in decimal notation, all you have to do is multiply them by 24, e.g. in a query SUM(AbsenceTime)*24 would give you the sum of the durations in decimal notation, e.g. if the durations sum to 76 hours and 30 minutes then the query would return a total duration of 76.5.  To return the total duration as 76:30 you can use the functions from my files.

Hello.

Welcome to Microsoft Community. To add your Google account on your Lumia 950, please follow the steps below:

  1. Go to "Settings".
  2. Tap on "Accounts".
  3. Tap on "Your email and accounts".
  4. Tap "Add an account".
  5. Tap "Other accounts POP, IMAP".
  6. Enter your Email address and Password.
  7. Tap the Sign in button.
  8. Once you're signed in Open the Edge browser.
  9. Visit gmail.com and log in to your Google account.
  10. Open the message with the subject "Blocked sign-in attempt for security reasons"
  11. Tap on the Allow Access button.
  12. You will be forwarded to the Google security setting page. "Under the Access for less secure apps", select "Turn on".

Wait for the settings on the phone to update, and your Google account will work on your phone.

Margins for printing a certificate to fit into an 8 by 10" - Microsoft Community

peggyhempelmann say:

Margins for printing a certificate to fit into an 8 by 10" frame?

I want to print a certificate that fits into an 8 by 10" frame but I cannot figure out how to set the correct margins for this in Microsoft Publisher. I am wasting a lot of paper and ink! Please help! Thank you.
john korchok say:

It's a multi-step procedure, but you need a printer that can print custom page sizes (not all can):

  1. If you are starting from scratch, create a custom page size of 8"x10" by using File>New>More Blank Page Sizes>Custom>Create new page size
    If you already have an existing page and the size is not already 8"x10", use Page Design>Size>Create New Size. Note that you have to set both Page and Target paper size to 8"x10".
  2. Create or modify your art, if necessary.
  3. Click on File>Print and click on Printer Properties under your chosen printer. Set up a custom paper size of 8"x10", if your printer driver has that capability. Every printer driver has different options, so you're on your own here.
  4. Load 8"x10" paper into the printer and print.

Unless you're printing lots of these, it's probably simpler to just design and print it to an 8 1/2" x 11" sheet and trim it by hand.

don cuthbert say:

To get an activated Windows 10 you have to first upgrade from the device's activated Windows 7.

What you could do is install and activate Windows 7 on a new drive partition (a dual boot of Windows 7).  Use your new computer's Windows 7 Product Key to activate the new install.  This will become the default boot OS, but you can switch that later as you wish by running msconfig.

Then upgrade that new Windows 7 partition to Windows 10.  Now your computer is registered as a licensed Windows 10 device.  You can now wipe the upgraded partition and clean install Windows 10 64-bit there.  Note that the free Windows 10 upgrade does not use any product key to activate, so do not enter any product key for Windows 10.  You will have to use the Edition of Windows 10 that matches your Windows 7 license.

Windows Pro has downgrade rights.

As for burning to a DVD, I recommend you use a USB flash drive.  They read much faster than a disk.  I got a few 16GB USB 3.0 from Best Buy yesterday for $9.99 each.  You can re-use them for each new Windows Build you want to try, or put your Windows 7 install on it if you want and re-use it for Windows 10 installs.

cbarnhorst say:

As the other replies indicate, you need to set up a Windows 7 Sp1 or Windows 8.1 on the SSD, activated and fully updated.  You will need a second Windows 7 or a Windows 8.1 license to do this.  The second license is the one that will be consumed by the Windows 10 upgrade and you will not be able to use it on another computer while Windows 10 is installed on your second partition.

Windows 10 is a free upgrade license, not a free FFP license for Windows.  

Or you can simply buy a retail FPP copy of Windows 10 and clean install it on the SSD.

If you want the free copy of Windows you have to perform an upgrade for the intial installation.  Once installed you are free to perform a clean install if that is your wish, but you have to do the upgrade to get the copy for free.  Windows 10 is not a software activation but a device activation technology, so don't try to do things "the old way."
