jeeped say:
If the Owner field is RT or Fusion and the wildcard added to your COUNTIF() was purely speculative, then you can wrap the process in a SUMPRODUCT() that divides each count by the number of common values. Example: if there where three identical owners and dates then each would be counted as 1 ÷ 3 or ¹⁄₃ + ¹⁄₃ +¹⁄₃ = 1.

This sample formula removes the sheet name and only deals with RT for clarity,

=SUMPRODUCT(($B$2:$B$9999="Y")*($F$2:$F$9999="RT")*(MONTH($J$2:$J$9999)=6)*(YEAR($J$2:$J$9999)=2012)/COUNTIFS($F$2:$F$9999,$F$2:$F$9999&"",$J$2:$J$9999,$J$2:$J$9999&""))

If you require actually the wildcard capabilities of RT* then FIND() or SEARCH() might be appropriate but they do not guarantee that the Owner start with RT or Fusion, only that the text is within the cell. However, both RT and Fusion can be accounted for without adding two formulas together.

=SUMPRODUCT(($B$2:$B$9999="Y")*ISNUMBER(SEARCH({"rt","fusion"},$F$2:$F$9999))*(MONTH($J$2:$J$9999)=6)*(YEAR($J$2:$J$9999)=2012)/COUNTIFS($F$2:$F$9999,$F$2:$F$9999&"",$J$2:$J$9999,$J$2:$J$9999&""))

If RT or Fusion absolutely have to be the start of a text string in column F, then some array processing with LEFT() might work.

=SUMPRODUCT(($B$2:$B$9999="Y")*(LEFT($F$2:$F$9999,LEN({"rt","fusion"}))={"rt","fusion"})*(MONTH($J$2:$J$9999)=6)*(YEAR($J$2:$J$9999)=2012)/COUNTIFS($F$2:$F$9999,$F$2:$F$9999&"",$J$2:$J$9999,$J$2:$J$9999&""))

These formulas are starting to get complicated and you may experience some calculation lag. When you transcribe them for your own purposes, knock the cell range references down to a minimum number of rows.

entegy say:

No problem. Windows 8 right? Libraries are a Windows 7 and Windows 8 feature.

  1. Close Zune and connect the drive to your computer.
  2. Open up File Explorer. You can right-click the bottom left corner of your screen and find it in the menu that will appear.
  3. From File Explorer's lefthand sidebar, click "Music" under "Libraries"
  4. From the Ribbon, move to the Library Manage tab. Click "Manage save locations"
  5. Click "add" and choose the external hard drive (or any specific sub-folder on the drive).
  6. Click Ok until all your windows are gone.

Now in the Music library, you should see both your hard drive's music folder and the external hard drive's contents.

 

By default on Windows 7 and Windows 8, Zune uses your Windows Libraries to manage music. But we can double-check it IS using this setup. Open Zune, then go to Settings>Software>Collection. The header on this page should say "Windows Libraries" and you should see 2 or 3 entries under the music library, one of them being your external.

If it says "Monitored folders" as the header, click the "Align folders" button at the bottom of the page to switch back to using Windows Libraries.
