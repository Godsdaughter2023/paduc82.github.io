Why is Windows Defender running?! - Microsoft Community

stb100 say:

Why is Windows Defender running?!

I disabled Windows Defender a long time ago. Today, I look at my bottom-right taskbar, I see an unfamiliar shield icon, and it turns out Windows Defender is running. Why?!!?

I'm running Windows 10. I use Avast Free Antivirus.

(Can I disable it (again)? Probably. But why, Microsoft, is it running on its own again? I'd call that a bug - or extremely poor, intrusive practice. Did I recently receive the Windows 10 update? Yes. Maybe it had to do with that. But that is no excuse.)

Thanks.

a. user say:

Not to worry...

It's not wise to have two malware detectors running in the background at the same time, so when you install a third-party malware program like Avast, Windows stops Windows Defender from running in the background.

However, it's fine to have another malware program scanning on demand, without running in the background, and that's what Windows Defender is doing. In fact, it's good security practice to run a different malware scanner on demand from time to time, because no one anti-malware program can detect every possible threat.

For example, I happen to use BitDefender Internet Security, which runs in the background. But every now and then I'll do an on-demand scan with Malwarebytes.

Windows Defender is simply running an on-demand scan, which doesn't interfere with Avast. You're getting something for nothing and it isn't causing a problem.
