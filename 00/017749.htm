yaqub k say:

Hi,

 

Thank you for posting the query on Microsoft Community and we are happy to help.

 

Content Advisor is a tool for controlling the types of content that your computer can access on the Internet. After you turn on Content Advisor, only rated content that meets or exceeds your criteria can be viewed. You can adjust the settings to suit your preferences. Content Advisor allows you to control the types of Internet content that can be viewed on your computer. Depending on how you configure it. Content Advisor uses ratings that websites voluntarily provide to block or allow certain content. Because not all websites are rated, unrated websites are automatically blocked (but you can choose to allow them). Once you've adjusted the settings and turn Content Advisor on, it checks websites as you visit them.

 

As you want to reset the content advisor password, you may try to follow the steps mentioned below.

a) Click on Start

b) Click on Run.

c) Type regedit, and press Enter.

d) Click on the + sign to the left of to expand HKEY_LOCAL_MACHINE.

e) Repeat Step (d) until you reach Software → Microsoft → Windows → Current Version → Policies.

f) Expand Policies and click on the Ratings folder.

g) On the right side of the Registry Editor window, you would see an item named as "Key", right-click on it and select Delete.

h) Close the Registry Editor.

i) Restart the computer and open Internet Explorer.

j) Click on Tools to open Internet Options.

k) Click on the Content tab and click on Disable. When IE asks for the password, leave it as blank, and click on OK. This will disable the Content Advisor Password in Internet Explorer.

Important: The above mentioned article contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs.

Right-click the worksheet tab and paste the following code in the worksheet's

code module:

'===========>>
Option Explicit

'---------->>
Private Sub Worksheet_Change(ByVal Target As Range)
    Dim Rng As Range
    Dim Rng2 As Range
    Dim rCell As Range, aCell As Range
    Dim oDic As Object

    Set Rng = Intersect(Me.Range("D:D"), Me.UsedRange)

    If Not Intersect(Rng, Target) Is Nothing Then
        Set oDic = CreateObject("Scripting.Dictionary")
        For Each rCell In Rng.Cells
            With rCell
                If Intersect(rCell, Target) Is Nothing Then
                    If Not .Value = vbNullString Then
                        If Not oDic.exists(.Value) Then
                            oDic.Add .Value, .Value
                        End If
                    End If
                End If
            End With
        Next rCell

        For Each aCell In Target.Cells
            With aCell
                If oDic.exists(.Value) Then
                    MsgBox Prompt:="The value " & .Value & " in cell " _
                                   & aCell.Address(0, 0) _
                                   & " is already in use. Please assign a new one.", _
                           Buttons:=vbCritical, _
                           Title:="Pleases Change!"
                End If
            End With
        Next aCell
    End If
End Sub
'<<===========
