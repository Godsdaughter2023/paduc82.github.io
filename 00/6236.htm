    rich007a say: A generic comment.
    If you have dates in column A, and data in column B that you want to use in a formula that only looks at data next to and above that formula (ie. historic data) and the start of that data is fixed (1-Jan) you just need to use a range which uses absolute references in start cell and relative references in the end cell:
    e.g in cel C1 type:   =SUM(B$1:B1)
    You can now drag that down, so if you drag it to cell C5, it becomes   =SUM(B$1:B5)
    Thus, in this example using SUM, it gives you the total of column B to date since the date in A1.
