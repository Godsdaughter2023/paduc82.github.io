Option Explicit

Sub Main()
  Dim R As Range
  Dim Dict As Scripting.Dictionary
  Dim Key As String
  Dim Childs
 
  'Get all Parent-Child kombinations
  Set Dict = CreateObject("Scripting.Dictionary")
  For Each R In Range("B1", Range("B" & Rows.Count).End(xlUp))
    Key = R.Value
    If Dict.Exists(Key) Then
      'Add another chlid for this parent
      Childs = Dict(Key)
      ReDim Preserve Childs(0 To UBound(Childs) + 1)
      Childs(UBound(Childs)) = R.Offset(, -1).Value
      Dict(Key) = Childs
    Else
      Dict.Add Key, Array(R.Offset(, -1).Value)
    End If
  Next
 
  'Build the relationships
  For Each R In Range("B1", Range("B" & Rows.Count).End(xlUp))
    Key = GenPC(R.Value, Dict, 0)
    'Remove the last "," or "X"
    R.Offset(, 1) = Left(Key, Len(Key) - 1)
  Next
End Sub

Private Function GenPC(ByVal Parent As String, ByVal Dict As Dictionary, ByVal Level As Integer) As String
  Dim Child, Childs
  'Done if no more nodes
  If Not Dict.Exists(Parent) Then Exit Function
  'Prevent and mark endless loops
  If Level > 10 Then
    GenPC = GenPC & "XXXXX"
    Exit Function
  End If
  'Get all childs for this node
  Childs = Dict(Parent)
  'Add the top level node only once
  If Level = 0 Then GenPC = GenPC & Parent & ","
  For Each Child In Childs
    'Go down in each sub treee
    GenPC = GenPC & Child & "," & GenPC(Child, Dict, Level + 1)
    'Stop if max. depth reached
    If Right(GenPC, 1) <> "," Then Exit Function
  Next
End Function

How to drag down the same numbers - Microsoft Community

brianentz say:

How to drag down the same numbers

Every time I try to highlight a cell with numbers in it and then drag it down to fill all the cells below it with the same number the numbers instead progressively increase by one number. Sometimes that is helpful but other times I want it to be the same number all the way down. how to do this?
bob jones aka: cybertaz say:

Yet another method:

  1. Select the range into which you want the repeating value
  2. Type the value
  3. Press Command+return to finish

The type of content in the cell you drag from determines what happens when you drag...

  • If it's recognized as a member of a series it will extend the series [e.g., Date, Day Name or abbreviation, etc.] - Contrary to popular misconception, though, neither a single cell containing a number nor an alpha character is recognized as a series.*
  • If it isn't recognized as a member of a series, dragging the Fill Handle will copy the entry rather than extending it.

*[There are many variations on the behavior of the technique. For example, if the cell contains the number 1, dragging the handle results in 1 being copied. OTOH, if you enter 1 in the first cell & 2 in the next cell, then select both, dragging the handle results in 1 2 3 4 5 6 7 8... Or if you start with 0 & 5, dragging the handle results in 0 5 10 15 20 25...]
