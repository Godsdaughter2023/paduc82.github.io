'105.110200.388250.811400.000000

hansv mvp say:

If you place the search string in a cell, a formula such as

 

=VLOOKUP(D1,A1:B13,2,FALSE)

 

should work, where D1 is the cell with the lookup value and A1:A13 is the set of string values.

If you want to use a literal lookup value, it should *not* include the apostrophe, and it should be enclosed in quotes:

 

=VLOOKUP("105.110200.388250.811400.000000",A1:B13,2,FALSE)

 

You *must* use strings for this type of lookup, for Excel uses 15 significant digits when processing numbers.

Tab name

t. valko say:
Is it possible to have the tab name come from a cell within the worksheet?


You would need an event macro to do that.

Let's assume you want the sheet (tab) name to be that of the entry in cell A1.

  • Select the sheet (tab) that you want this to happen in
  • Right click the sheet (tab) name
  • Select View Code
  • Copy then paste the code below into the right side pane of the window that opens

Private Sub Worksheet_Change(ByVal Target As Range)
 On Error Resume Next
    ActiveSheet.Name = Range("A1").Value
End Sub

  • Close the window to return to Excel

Enter a valid sheet name in cell A1. The sheet (tab) name will change as desired.
