chip pearson say:

Where are you declaring the variable Brand? You need to declare it  (Dim Brand As String) outside of and before any procedure in the code module in which it is used. If you are not declaring the Brand variable, the compiler will create a new variable named Brand when it encounters it on the first line of your ComboBox1_Click code. Then, code execution moves to Select_Brand and the compiler creates a completely new variable named Brand, unrelated to the Brand in the ComboBox1_Click procedure, when it encounters "Brand" in your IF statement. It sets that variable to the default value of an empty string. Since Brand in Select_Brand is an empty string, it will not match any of the tests in your If/ElseIf structure.

 

You should ALWAYS explicitly declare your variables. Not doing so can cause problems that can be very difficult to debug. You can enforce this in all future modules. In VBA, go to the Tools menu, choose Options, then the Editor tab. There, check "Required Variable Declaration".  You can enforce it in an existing module by entering "Option Explicit" as the very first line of the module, above and outside of any procedure or code.

 

It is also possible that the problem lies in case sensitivity. In Binary comparison, upper and lower case are considered different ("abc" <> "AbC"). At the very top of the module, above everything else, enter

 

Option Compare Text

 

This will force text comparisons to ignore upper and lower case (e.g., "abc" = "AbC").

ossiemac say:

If you want a VBA code solution then the following VBA code should do it. However, it tests all cells in the used range on the worksheet so if you want to restrict it to a specific range then please get back to me.

Also if you don't know how to install the code then get back to me.

Sub ReplaceNumerals()
    Dim rngCel As Range
    Dim regExp As Object
   
    Set regExp = CreateObject("VBScript.RegExp")
   
    For Each rngCel In Worksheets("Sheet1").UsedRange
        If rngCel.Value <> "" Then Debug.Print rngCel.Value
        With regExp
            .Global = True
            .Pattern = "[0-9]{5}"   'Any character 0-9 total of 5 characters
            rngCel.Value = .Replace(rngCel.Value, "this ")   'Replace any string matching pattern with "this".
        End With
    Next rngCel
End Sub
