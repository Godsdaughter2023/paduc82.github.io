I finally found the answer to my own problem ! One is never been useful better than by oneself as it said.

The USB cable connection process was stopped because the memory card was used by an another application (this is told in a message box on the Nokia E61).

The solution is the background Image on the desktop in the interface was defined from a image file on the memory card. You need to copy the image file in the smartphone memory to solve the problem with the USB cable connection process.

Thanks to that, the process reachs its goal and load the memory card as a removable hard disk on the computer.
