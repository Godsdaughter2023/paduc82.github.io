michael m. knight say:

The main issue people have when customising Outlook toolbars is the actual toolbar. You need to select All Tabs and customise this first. Even for the new Simplified Ribbon. 

Here's how:

  1. In simplified view, right click a menu item
  2. Select Customise the Ribbon...
  3. On the Options dialog box, select the tabs drop down box on the right and choose All Tabs
  4. Customise how you want the tool bar to look and rearrange as you like them. Add or remove items here too if what you need isn't listed
  5. Once your happy, export your settings. This helps if you have a new installation or somehow your settings change. You'll be able to restore them
  6. Click OK and you're Done

As mentioned, this works for the full ribbon or the simplified ribbon. The reason most people fail to customise correctly is that they edit the default setting which is Main Tabs. You need to set it to All Tabs first.

Microsoft My Phone uses AGPS to get an approximate location of where your phone is located.  This information is transmitted from the GPS coordinates of the cell tower it is currently engaged with for sending/receiving information. Address Book Error 81: - Microsoft Community

howdoidoit say:

Address Book Error 81:

Cannot connect to Internet Directory Service (LDAP) server:  mail.optusnet.com.au.  Check your network connection or modify your Address Book settings
shashirekha say:

Follow the steps below to get rid of this error message:-

  • If you are not using Lightweight Directory Access Protocol (LDAP) Internet directory services, remove the LDAP address book. To do this, follow these steps:

1.    Start Outlook.

2.    On the File menu, clickAccount Settings.

3.    On the Address Books tab, click the LDAP address book that you want to remove, and then clickRemove.

4.    Click Close.

5.    Exit and restart Outlook.

  • If you are using Lightweight Directory Access Protocol (LDAP) Internet directory services, make sure that the settings for the LDAP address book are configured correctly. To do this, follow these steps:

1.    Start Outlook.

2.    On the File menu, clickAccount Settings.

3.    On the Address Books tab, click the LDAP address book that you want to configure, and then clickChange.

4.    Verify that the settings for the LDAP server are correct.

5.    Click Next, and then clickClose.

How do you know your computer has been scanned? - Microsoft Community

denise laurao'brien say:

How do you know your computer has been scanned?

I installed Microsoft Security Essentials which I like; there is no logs, no dates, so How can I be sure that the full scan was done at 1 am?
le boule say:

Open MSE, open Home Tab and look at the bottom of the page under Scan Details.

 

If malware is found MSE will address it or if you need to take action you will be notified.

 

MSE does not identify or alert to the presence of cookies which are not malware.

 

Recommend you set all default actions to quarantine.  If malware is found you can read about it by opening MSE and clicking on the History tab.

 

Recommend you thoroughly review the following paying close attention to Common issues that can affect MSE and items 11, 30 and 31 under Frequently Asked Questions and General Concerns: 

Microsoft Security Essentials – Installation Checklist and Frequently Asked Questions 

 

 

And

 

 

The following thread includes a discussion about scans that you may find helpful - see the post of Kosh Vorlon: http://answers.microsoft.com/en-us/protect/forum/mse-protect_scanning/will-mse-scan-if-computer-is-turned-off/4bca54aa-0fa6-4b50-8172-ef188c3810b9

 

 

On this computer that I'm using MSE is not set to do any scheduled scans...I prefer an "occasional" manual scan.  I really think routine scanning is a bit overrated but others don't share my view.

 

And here is an excerpt from a previous post by MSE Forum Moderator Stephen Boots reference the use of quick scans and full scans that sums it up quite well:

 

1. A periodic quick scan will check the common locations for the presence of malware.

2. A full scan will scan all local drives for any traces of malware

Typically, you do not need to perform a full scan.

Real time protection is monitoring activity and file access on your PC, so if you attempt to open an infected file, it should detect the infection and stop you.

A full scan is very time consuming and disk intensive. If you wish to perform a clean scan periodically, go for it. I rarely run one.

Your best defense is not scanning, but preventing infection in the first place:

In addition to MSE or other up to date antivirus software:

Make sure that the Windows Firewall is enabled.

Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

Make sure Internet Explorer is at version 8 or 9 and updated with all patches.

In Internet Explorer, use the SmartScreen Filter.

Make sure that IE Internet Security settings are at least set to medium-high (default).

Enable the pop-up blocker in IE.

On Vista and Windows 7 make sure that User Account Control (UAC) ON  and not running with elevated privileges.

Make sure that Windows Automatic Updates are set to at least notify, but the preferred setting is to download and install automatically. If you update manually, be sure to update as soon as possible after being notified of available updates.

Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.

Never click through links from unknown sources and use caution even if they are from a "trusted" source.

Never open unsolicited email attachments.

Practice safe web browsing.
