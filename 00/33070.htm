To solve the issue, I deleted a registry key "PortableOperatingSystem" with following steps:


1. run "regedit"

     - command+R

     - type regedit in the command box

     - click Run button

2. Go following Path in regedit

     - HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control

3. Find "PortableOperatingSystem" and delete the key

4. Reboot


Then I succeeded to install Windows 8.1 without any problem.

davem121 say:
Hi Beth,

Your network password is set on your Router, it is not a Windows Setting

In the back or bottom of your Router is a sticker with a UserName, Password and IP address (something like 192.168.0.1) printed on it

Note those down

Go to your PC and open a Web Browser

In the address type the IP address and hit Enter

That will bring up your routers configuration page

Somewhere on that page you will find LAN settings and from there you can change your password

Before you can change that, you will need to enter the UserName and Password you noted down from that sticker on the router

That's It!
