I notice the original poster did not continue with this thread. I used the Search function of the Windows Forum -MS Communities for an answer to the same question; i.e., changing folder locations.

 

It's always difficult to accept an answer to a reasonable question when the answer contains the impossible words "not possible".

 

You will probably remember the small utility program TWEAK UI and its function. In applicable OSs, TWEAK UI would allow a user to change locations of certain folders; even system folders. An example of this would be the "My Documents" system folder and any or all of its sub-folders.  The My Documents folder was significant to the OS, for example, XP, because the user could save just about anything in the My Documents folder; it was also a systemic share folder if wanted, and was a major piece of the XP folder/file system.  Using TWEAK UI, a user could choose to re-locate this system Folder to a data partition on a hard drive in order to save the folder and its contents from an OS failure, or any digital event that might do damage to the OS install partition, commonly the "C" volume.

 

Is the design of the W8 registry (or any part of the W8-OS that makes folder re-location "not possible") so advanced, complex, or rigid that a simple task performed by WXP can-not be performed by W8?  I think not.
