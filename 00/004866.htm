Multiple Selection Listbox - Microsoft Community

tjhudson say:

Multiple Selection Listbox

I can see how to add a drop-down list content control to a Word 2013 document, but is there a way to set it up to select more than one item in the list?
jay freedman say:

Short answer: No.

Long answer: Of the three types of list boxes that you can insert in the body of a document (content control, legacy form field, and ActiveX list box), only the ActiveX variety can support multiple selection.

To make that happen, turn on the Design Mode button in the Developer tab, right-click the list box and click Properties, and change the value of the MultiSelect property to "2 - fmMultiSelectExtended".

If you create a userform, the list box that you can use there will also support multiple selection, and the code that you write for the OK command button can transfer the selected item(s) to the body of the document.

stefan blom say:

I'm assuming that you are using the new equation editor in Word (as opposed to the older Microsoft Equation 3.0 objects). Click Insert tab | Equation | Insert New Equation and then pick a matrix format from the Equation Tools Design contextual tab on the ribbon.

     

Once you have inserted a matrix, click the arrow on the content control which encloses the equation, and click the "Linear" option. You will see the underlying "markup" for your matrix. Specifically, notice that there is an & before each new cell in a row, and that there is an @ before each new row. You can add or remove these to get exactly the layout you want.

      

When you are done, click the arrow again and choose Professional.
