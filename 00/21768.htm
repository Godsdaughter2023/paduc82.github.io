Insert blank line after sets of duplicates - Microsoft Community

roccofiero say:

Insert blank line after sets of duplicates

I have a spreadsheet that I have sorted several ways.  In column "A" I have a list of numbers, some duplicates, some not.  I want to insert a blan row in between each set of duplicates for easier reading. 
gord dibben say:
I have a spreadsheet that I have sorted several ways.  In column "A" I have a list of numbers, some duplicates, some not.  I want to insert a blan row in between each set of duplicates for easier reading. 

 

Run this macro.


Sub InsertRow_At_Change()
'Sandy Mann July 1st, 2007
    Dim lastrow As Long
    Dim x As Long
    lastrow = Cells(Rows.Count, 1).End(xlUp).Row
    Application.ScreenUpdating = False

    For x = lastrow To 2 Step -1
        If Cells(x, 1).Value <> Cells(x - 1, 1).Value Then
            If Cells(x, 1).Value <> "" Then
                If Cells(x - 1, 1).Value <> "" Then
                    Cells(x, 1).entirerow.Insert Shift:=xlDown
                End If
            End If
        End If
    Next x
    Application.ScreenUpdating = True
End Sub

    jeeped say:

    Put your cursor where you wish the information to be (e.g. header, footer, body) and press CTRL+F9. This opens a set of field braces. Type in FILENAME \p \* MERGEFORMAT so that it looks like this,

    { FILENAME \p \* MERGEFORMAT }

    ... and press F9 to update the field. You can use ALT+F9 to toggle between display field results and display field codes.

    Hint: If this is in a header or footer, you can quickly update the header/footer field(s) by entering Print Preview and immediately exiting.
