kathycurry say:

How do I turn a movie back into photos?

I have a DVD that was made using Windows Live Movie Maker.  How do I convert this back into individual photos?  I have no backup files of this and don't want to ruin the movie but would rather have photos for editing.  Windows Vista

larry henry jr. say:

Taking a Snapshot of a video…
Windows Live Movie Maker has the option of grabbing a screenshot during the previewing of your video.
Play the video in Windows Live Movie Maker, and during the preview [on the HOME tab] you'll see the option for Snapshot.
You click on Snapshot and then the application will capture the current video position and then ask you to save the Snapshot as a PNG file.

There are no other options to export, other than PNG.

The PNG file can be saved anywhere on your system [like your desktop].

NOTE:
The snapshot is taken from the preview window. The larger you can get the preview window, the larger the saved image will be.

     http://lehsys.blogspot.com/2011/11/windows-live-movie-maker-how-to-take.html

considerate_guy say:

Did you create the original DVD?  If so, you should hopefully still have the photos used on the DVD somewhere on your computer.  If someone else made it, you should check with them for the photos.

In either case, if you can go to the project file, it will show all the individual photos again.  The project file (.wlmp) is the file that lists the individual elements used to make your movie.  If you need to do more editing, you should use the project file, not the final movie.

If all you have is the video stream (completed movie), then the best you can do is to take screenshots as noted by Larry.  A DVD is a maximum of 720 x 480 pixels, so don't expect to get back the full resolution of the original photos.  Again, for that, I would go back to the original photos instead of using the DVD.
