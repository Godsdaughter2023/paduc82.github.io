bernie deitrick say:

    A) For the pop-up message

    1) Copy this code.
    2) Right-Click the sheet tab of "EHS Checklist"
    3) Select "View Code"
    4) Paste the code into the window that appears.
    5) Save the file as a macro-enabled .xlsm file.
    6) Make changes to the wording as needed

Private Sub Worksheet_Change(ByVal Target As Range)

    If Target.Cells.Count > 1 Then Exit Sub
    If Target.Value = "" Then Exit Sub
    If Target.Column <> 3 Then Exit Sub

    If Target.Value = "Partial" Then
        MsgBox "Please select percentage compliance from dropdown menu in next cell."
    End If

End Sub

=AND(D2="",C2="Partial")

and select red as the fill color.

bob jones aka: cybertaz say:

That's because Duplex printing isn't determined by the program used to create the document... It's a feature of the printer you're using. Where you'll find the setting varies depending on the Print dialog window configured for that particular program. Typically, it's in one of two places in the Print dialog:

  • On the main pane of the Print dialog window. It's often just a checkbox in the upper right corner, or
  • By selecting Layout from the Copies & Pages dropdown menu. There will be a Two-sided option on that pane, but wording may vary for different printers.

The Print dialog supplied to Word uses the latter of the 2 configurations.
