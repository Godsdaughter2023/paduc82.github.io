hosch250 say:

Remove Power Button on Start Screen

Technical Level : Intermediate

Summary
Explains how to remove the Power button on the Start screen.

Details

Do you dislike the Power button on the Start Screen?  There is a way to remove it, although you do have to edit the Registry to do it:


First, open the Run command by pressing Windows-R, by searching from the Start screen, or by right-clicking the lower left corner of the screen and type “regedit” without the quotation marks, then press Enter or click OK.  Navigate to “HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\ImmersiveShell”, then right-click “ImmersiveShell” and create a new key:


Next, name the key “Launcher”, without the quotation marks, then right-click in the right pane of the editor and create a new DWORD (32-bit) Value:


Finally, name this value as “Launcher_ShowPowerButtonOnStartScreen”, again without the quotation marks.  When this key’s value is 0, the power button will be off the Start Screen.  Set the value to 1 or delete the key to get the power button back.  The computer must be rebooted for changes to take effect.
