gary's student say:

Build the disable mechanism into the button code.  For example, for a button named buttton code like:

 

Private Sub buttton_Click()
If StopMe Then
    Exit Sub
End If
MsgBox "Good Morning Louise"
End Sub

 

 

and in a standard module:

 


Public StopMe As Boolean


Sub MAIN()
StopMe = False
'  do more stuff
End Sub


Sub routine()
' disable the button code
StopMe = True
End Sub

 

Running MAIN enables the button logic and running routine disables it.

 

The advantage to this approach is that the user can see the state of the button by examining the Boolean.

steve rindsberg say:
It seems you've found a bug.  

As a workaround, you can write your own function:

Function WorkingFindByName(oSl As Slide, sName As String) As Shape
    Dim oSh As Shape
    For Each oSh In oSl.Shapes
        If oSh.Name = sName Then
            Set WorkingFindByName = oSh
            Exit Function
        End If
    Next
End Function

This lets you find any named shape on a slide rather than having it work only with placeholders.  Or not work, as the case may be.  And in fact is.

toby ornott say:

The first five folders are Outlook Mail default folders which can not be removed or reordered. 

Archive (I believe) can be deleted (because I deleted mine); yet I believe that a new Archive is easily returned by clicking the selection in the Outlook Mail, sub-ribbon, menu (which appears whenever you mark a message) if you actually wish to have one.

All other folders are user-created folders and are arranged alphabetically. 

So if you want a different order, then you will need to rename your folders keeping in mind that your first-priority would be the letter-A.   Consequently if you want your MBA folder to be listed immediately after Archive, you could rename it AMBA.*  In like manner: you could reorder all your user-folders (and subfolders) in any peculiar alpha-numeric order keeping in mind that A will be followed by B (e.g. Aardvark comes before Abbacy) and A1-Cleaners are listed before A1.01-Cleaners.

*note: Although Archive is Ar, it takes precedent over AM because Archive is a default folder and its position is governed by the Outlook Mail software code.
