Worksheet in itself doesn't refer to anything. Use

 

ActiveSheet.EnableFormatConditionsCalculation = False

 

to disable recalculating conditional formatting in the currently active sheet, or

 

Worksheets("MySheet").EnableFormatConditionsCalculation = False

 

to do so for a specific sheet (MySheet in the above example) which is not necessarily the active sheet.
