One of the first things that you are going to need to confirm is "which of those folders is your <default> contact folder in your current Outlook profile?"

The default contact folder will be in the same tree as your Inbox and if that is the one that is empty - 2 things

1) You cannot delete that contact folder (on its own) under any circumstances
2) You probably should move the contacts from your secondary contact folder to your default

If, on the other hand, you have opened a secondary PST file which "used to contain" default folders - you will similarly not be able to delete any folder that used to be designated as a <default>. Again, if there is an Inbox in the same tree as the secondary contact folder - this would be the case.

You need to first move your PST file to the correct location from where Outlook 2016 can read/access the PST file. Then create your account as a POP account making the emails to be delivered to the default PST file and not the new PST file which is created with every new Profile.

 

To copy/move your PST file from the C: drive to another drive access the following PST file location on C: drive and cut and paste the PST file to the corresponding location on another drive.

 

PST file location on C: drive:

 

C:\Users\<UserName>\AppData\Local\Microsoft\Outlook\
