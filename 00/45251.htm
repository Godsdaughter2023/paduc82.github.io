a. user say:

Never defragment a SSD.

In the Disk Defragmenter tool, 'optimize' means defragment for a mechanical hard drive, and it means run the TRIM command for a SSD.

There's no need to manually defragment a HDD. Modern hard drives don't need defragmenting as older ones did. Whatever Windows does on its own is entirely sufficient.
