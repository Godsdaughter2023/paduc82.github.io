Hi juneiljj,


In order to open zipped or compress file from the web you will first need to download and save a local copy onto the computer.

Once you downloaded onto the computer try the below steps

• Right click on the zipped file, click on extract all.
• If you do not find extract all, it may be that default program associated with it is not set.
• Now right click on the file click on properties later click change button
• Select windows explorer from the list and click OK
• If you do not find windows explorer in the list of program, click browser button and navigate to c:\windows locate the file named “explorer.exe” and later click OK
• Now you should be able to open it.
