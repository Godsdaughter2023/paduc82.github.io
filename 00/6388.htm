How to populate a dependent dropdown list in Word based on 2 previous - Microsoft Community

andreadeighton say:

How to populate a dependent dropdown list in Word based on 2 previous dropdown lists

Is it possible to have a dropdown list populate based on the selection of TWO previous drop down lists?  I have found VBA coding which helps with populating based on ONE dropdown list, but I need it based on two.

eg:  List 1:  QLD or NSW

List 2: General or Social

List 3: (based on what is selected above, shows a Finance code for either QLD General, QLD Social, NSW General, or NSW Social).

Thanks for your help.

macropod say:

In that case, you might consider using three content controls - two dropdowns and one text one. The dropdowns would be titled 'State' and 'Category', with contents corresponding to your List 1 & List 2, respectively. The text content control might be titled 'Output'. Having populated the dropdown options, you could then add the following ContentControlOnExit macro to your document's 'ThisDocument' code module.

Private Sub Document_ContentControlOnExit(ByVal CCtrl As ContentControl, Cancel As Boolean)
Application.ScreenUpdating = False
Dim Rng1 As Range, Rng2 As Range, Rng3 As Range, Txt1 As String, Txt2 As String, Txt3 As String
With ActiveDocument
  With .SelectContentControlsByTitle("Output")(1)
    Set Rng1 = .Range
  End With
  With .SelectContentControlsByTitle("State")(1)
    Set Rng2 = .Range: Txt2 = .PlaceholderText
  End With
  With .SelectContentControlsByTitle("Category")(1)
    Set Rng3 = .Range: Txt3 = .PlaceholderText
  End With
End With
With CCtrl
  Select Case .Title
    Case "State", "Category"
      If (Rng2.Text = Txt2) Or (Rng3.Text = Txt3) Then
        Txt1 = ""
      Else
        Select Case UCase(Rng2.Text)
          Case "QLD"
            Select Case Rng3.Text
              Case "General": Txt1 = "QLD General"
              Case "Social": Txt1 = "QLD Social"
            End Select
          Case "NSW"
            Select Case Rng3.Text
              Case "General": Txt1 = "NSW General"
              Case "Social": Txt1 = "NSW Social"
            End Select
        End Select
      End If
    Rng1.Text = Txt1
  End Select
End With
Application.ScreenUpdating = True
End Sub
