There isn't a default shortcut, but you can assign one.

Go to File > Options > Customize Ribbon and click the Keyboard shortcuts: Customize button at the bottom. In the Customize Keyboard dialog, select "All Commands" in the Categories list. Then, in the Commands list, scroll almost to the bottom and click on ViewEndnoteArea. Click in the "Press new shortcut key" box and press the shortcut key combination you want to use. Despite the description "Opens a pane for viewing and editing the endnotes (toggle)", the command just jumps to the endnotes (at least in Print Layout view in Word 2016). If the cursor is on an endnote mark in the text, the command sends it to the corresponding endnote; if it's anywhere else in the text, the command sends it to the endnote whose mark is next in the text.
