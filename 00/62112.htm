After sharing a folder on a drive and moving that drive to another - Microsoft Community

kevin-- say:

After sharing a folder on a drive and moving that drive to another computer, the user does not have access to the folder

How do I access a folder that I shared and then moved to a different drive? How do I access a folder that I shared and then moved to a different drive?
james- say:

If you share a folder and then move the drive that folder is on to another computer, you won’t have access to the folder when you log onto the other computer.  Or, if you share a folder and then install another instance of Windows 7, when you are logged onto the new installation you won’t have access to the folder.  For instance, if I have my operating system installed on C: and I share out D:\Foo, if I reinstall the OS on C:, I will not have access to D:\Foo.

 

When you try to access the folder, you’ll be told that you don't have access to it and you’ll be asked to elevate.  If you elevate, you should regain access to the content.

 

However, in some cases this won’t work. If so, follow these steps to gain access:

1.            Click the Start button, and then type command prompt in the search box.

2.            In the list of results, right-click Command Prompt, and then click Run as administrator.  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

3.            Type the following at the command prompt:

   - "takeown /f <path to folder>"

   - "icacls <path to folder> /grant <user>:(CI)(OI)F"

 

In the above instructions, <path to folder> is the folder's path (for instance, "D:\Foo") and <user> is the user name (for instance, "Frank").  For example:

 

"takeown /f D:\Foo"

"icacls D:\Foo /grant Frank:(CI)(OI)F"
