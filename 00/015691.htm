Zune bitrate conversion process - Microsoft Community

sharpiefinepoint say:

Zune bitrate conversion process

I have filled up my Zune, but still have more music that I would like to put on it.  The debate I have is whether to use the Zune software bitrate conversion to decrease the bitrate on some of the files so I could put more music on the device.

So before I go ahead and risk this I want to know a little more about the conversion process.

1. Does the conversion process affect the individual files saved on the computer in any way? I have had enough problems with the Zune software messing up tags in the files, and I don't want to risk my music files being permanently converted.

2. I assume the conversion works in the following way: first a copy of the original file is made, except at the lower bitrate, that file is synced to the device, and then the copy is deleted, the device has a new lower size and bitrate file, while the original file is unchanged.  Is this correct?

3. If I already have files saved to the device which are higher than the bitrate I set as the maximum bitrate, will they be converted to a lower bitrate, or will I have to take them off of the device manually and then put them back on in order to get them to take up less space? This would be a very time consuming process, and if there is an easier way to do it I will do that.

4. Most (90% or more) of my music comes from cds ripped to 256 Kbps mp3s.  If I was to convert all of them to lets say 128 Kbps, how much memory should I expect to save?

I exported a WAV file to three separate mp3 files at bitrates 128, 256 and 320 kbps.  The 128 and 256 were about 40% and 80 percent of the 320 kbps, this implies a nearly exact linear correlation between the size of the file and the bitrate. Is this an accurate assumption?  I only ask because I am not an expert and wonder if this is an accurate way to estimate this.

5. On the files I experimented on I couldn't really notice a difference in the quality of sound between 256 kbps and 128 kbps, but I was using my laptop which has the standard sound card.  Should I expect a noticeable difference in quality or will it be unnoticeable?

This is kind of a long winded question but I just wondered if any had any good answers.

Thanks

nsilmike say:

1) No.  Only the file that is sync'd to the device.

2) A tmp file is created and saved on the PC in the transcoded file cache.  You can keep them or delete them as you see fit.  You would probably want to keep them if you sync content on and off... thus you would save the conversion time.  You would/could trash them if once you sync, that's it, they stay on the device indefinitely.

3) Not sure.  watch while you sync.  you should see file converting followed by sync'ing, and of course it will take some time to go thru everything you sync.

4) Roughly half.  But, just FYI... 128Kbps is pretty avg quality... try a few and see if you like the sound with your normal earbuds before you go after them all.

5) Yes, for a CBR (constant bit rate) conversion.  The conversion resolution is constant, the frequency of the samples is changing... and the bit rate is the product of the bit resolution x samples per second.
