greg maxey say:

Without using some API that I don't know about, this can be done with a little trickery.

Create another form the same size as your form of interest call it frmTemp.  It the frmTemp module place the following code:

Option Explicit
Private Declare Sub Sleep Lib "kernel32" (ByVal dwMilliseconds As Long)
Private Sub UserForm_Activate()
DoEvents
Sleep 5 'That is 5 milliseconds
Unload Me
ThisDocument.Activate
End Sub

In a code module call the frmTemp and your frmOfInterest as follows:

Public Sub CallForm()
  frmOfInterest.Show vbModeless
  frmTemp.Show vbModeless
End Sub

Basically your frmOfInterest is displayed.  Then it loses focus to frmTemp which loses focus to the document in 5 miliseconds.

john sr wilson say:
There are probably better ways but one way is to add the sound as a transition sound on a hidden slide.

Sub soundoff()
Dim se As SoundEffect
'slide 12 is hidden and so doen't usually play
ActivePresentation.Slides(12).SlideShowTransition _
.SoundEffect.ImportFromFile "C:\Program Files\Microsoft Office\Office14\MEDIA\APPLAUSE.wav"
Set se = ActivePresentation.Slides(12).SlideShowTransition.SoundEffect
se.Play
End Sub

The path to the sound may not be the same on your PC
WATCH OUT for the line breaks!
