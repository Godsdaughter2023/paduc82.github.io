mlg63 say: You cannot access the bios in pc settings. The bios is accessed as soon as you reboot and before windows starts. Tap F2 as soon as you reboot to enter Setup (bios) and change settings there--IF you know what you are doing. You can make your computer unbootable if you change something in the bios incorrectly. Explain what settings you want to change. Most likely those settings are not in the bios. The settings are more likely to be in windows and not the bios. For instance--If you need to turn a sata port on the motherboard on/off when adding or removing a hard drive, you use the bios. But if you are changing a windows setting it is not done in the bios.

Hello,

This issue may arise if the driver of the camera is outdated. We suggest that you follow these troubleshooting steps:

Method 1: Install updates from the manufacturer.

  1. Open the Device Manager on your PC.
    • In Windows 10, enter Device Manager in the search box next to Start, and then select Device Manager in the results.
    • In Windows 8.1, swipe in from the right edge of the screen and then tap Search. (If you're using a mouse, point to the lower-right corner of the screen, move the mouse pointer up, and then click Search.) Then enter Device Manager in the search box, select Settings, and select Device Manager in the results.
  2. Expand Imaging devices.
  3. Right-click the camera that's having problems, and select Update Driver Software.
  4. Select Search automatically for updated driver software, and then follow the on-screen instructions. This might include restarting your PC.
Method 2: Install your PC's default drivers.
  1. Open the Device Manager on your PC.
    • In Windows 10, enter Device Manager in the search box next to Start, and then select Device Manager in the results.
    • In Windows 8.1, swipe in from the right edge of the screen, and then tap Search. (If you're using a mouse, point to the lower-right corner of the screen, move the mouse pointer up, and then click Search.) Then enter Device Manager in the search box, select Settings, and select Device Manager in the results.
  2. Expand Imaging devices.
  3. Right-click the camera that's having problems, and select Update Driver Software.
  4. Select Browse my computer for driver software.
  5. Select Let me pick from a list of device drivers on my computer.
  6. Select USB Video Device, select Next, and then follow the on-screen instructions. This might include restarting your PC.

If the issue persists, kindly contact your device manufacturer for better assistance.

    kevin-- say: Hi nycshortie72,

    Can you try to ping 127.0.0.1 and ping www.yahoo.com.  Post back with the result.

    This is might be an issue with your ARP table.  To confirm that it’s indeed an ARP issue, try to ping both the loopback address (127.0.0.1) and the local computer’s IP address. Then try to any other IP addresses such as those of Google.com or Yahoo.com. If you can ping to local IP address and loopback address, but not any other IP addresses, then this is the typical symptom of ARP cache failure.

    Use the NetShell (netsh) command to clear, delete or refresh the Address Resolution Protocol (ARP) cache by following the following steps.

    1. Click on Start button.
    2. Click on Run command and type Cmd into the Run text box, and then press Enter. For Windows Vista, type Cmd in Start Search, and then press Ctrl-Shift-Enter keyboard shortcut to force Administrator privileges. Enter Administrator user ID and password and allow any permission request when prompted.
    3. In the Command Prompt window, type the following command then press Enter.

      netsh <interface ip> delete arpcache
