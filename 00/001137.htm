Windows defender is taking a long time to scan - Microsoft Community

springbender say:

Windows defender is taking a long time to scan

I am running my first full scan with windows defender 5 hrs. and still scanning. why is it taking so long?

stephen boots say:

Full scan times, especially the first one, depend on many factors:

- CPU speed

- Memory installed and available (RAM)

- Drive speed

- Drive capacity and number of files present

- Compressed files take longer to scan -- many compressed files can really add to the scan time.

Typically, subsequent full scans will be faster.

On the topic of full scans --

As long as real time protection is enabled, a full scan on a frequent basis is usually overkill. A quick scan looks for malware in the most common locations. A full scan has to touch every file. Malware found in a full scan, not in a quick scan is typically not active. Personally, I only ever run a full scan if an infection has been detected or if one is suspected.  I use a different scanner for the full scan (the malware was missed when it arrived, so why should I expect it to find it on a full scan?)

The following scans are good for a second opinion:

Try Malwarebytes Antimalware (free version, do not install the trial or Pro version) - http://www.malwarebytes.org/products/malwarebytes_free/

Try SuperAntispyware

Try the ESET free online scan:  http://www.eset.com/us/online-scanner/
