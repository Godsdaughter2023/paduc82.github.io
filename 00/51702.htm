You cannot. Contacting Skype support will not change that fact.

What is caller identification and how do I set it up?

Caller identification using your Skype Number is available in:

  • USA
  • UK
  • Chile
  • Denmark
  • Estonia
  • Poland
  • Sweden
  • Switzerland

Skype numbers in those areas will already be in your Caller ID settings, so there is no need to attempt to add those Skype numbers. Skype numbers outside those areas do not support Caller ID and attempting to add them will fail. You can only add a French mobile number if you require a French Caller ID.
