I was able to install Office Click-to-Run with Visio MSI by renaming the ProductCodes registry value in the Uninstall key.

Example

(32bit Visio on 64Bit OS)

HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Uninstall\Office15.VISPRO

Rename ProductCodes to ProductCodes.bak

Run Click-to-run install

I'm sure this is not supported (https://support.microsoft.com/en-us/kb/2753630), but it seems to be working fine for me. Also, even though I do not think it matters, I renamed the value back after the install completed.

vin55 say:

I have managed to reproduce _FlyDye_'s method for Visio 2016 and a volume licensed Office 2016 Professional Plus:

  1. Install Visio 2016 first.
  2. Once activated, open regedit (type it in the Start menu).
  3. Go to \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\VisioProRetail - **-** (**-** will be the region and language code for your version, eg. en-us).
  4. Rename ClickToRunComponent to something else (e.g. ClickToRunComponent.bak).
  5. Install and activate Office Professional Plus.
  6. Enjoy!

jay freedman say:
If the paragraph you want to go to doesn't start with a heading (in a built-in heading style), insert a bookmark at the beginning of the material you want to go to, surrounding some descriptive text.

At the point where you want the hyperlink, insert a cross-reference (on the References tab of the ribbon, in the Captions section, click Cross-reference). In the cross-reference dialog, set the reference type to Heading (if the reference is to a heading) or to Bookmark and select the heading or the bookmark's name from the list. The "Insert reference to" box is set to "Heading text" or "Bookmark text", and the  "Insert as hyperlink" box is checked by default, so leave them as is. Click the Insert button and then Close.

Unfortunately, the cross-reference won't look like a hyperlink, and applying Hyperlink style to it won't change anything. You'll have to select it and manually apply blue color and an underline.
