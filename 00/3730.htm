techfreak_ say:

The number you provided (partially removed from your image by me since this is a public forum), does not appear to be in the default Finland mobile area code. If the number you are trying to verify for Caller Id purposes is a landline number you will not be able to verify the number with Skype. Verification only occurs when the verification system sends a short code SMS (not audible) to the number provided. If the destination number cannot receive that incoming SMS, due to it being unreachable, you can receive the "Internal Error Occurred" message. Most landlines are incapable of receiving SMS messages (unless virtual), since it is typically a mobile service. If your number is a landline that was ported to mobile or a specific virtual system then that might create its own set of issues. As long as I can recall, Skype has never provided any other alternate methods of confirming ownership of a number for Caller Id purposes.
