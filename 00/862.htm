bernie deitrick say:
This is how to combine the two, but I have concerns that your autoNo is not properly written to work with this - it really should take a range argument (Target) or be incorporated into the event macro

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim rngC As Range
    If Intersect(Target, Range("B5:B39")) Is Nothing Then Exit Sub
    Application.EnableEvents = False
    For Each rngC In Intersect(Target, Range("B5:B39"))
        If rngC.Value = "" Then
            Intersect(Target.EntireRow, Range("C:M")).ClearContents
            Intersect(Target.EntireRow, Range("C:M")).Validation.Delete
        Else
            Call autoNo
        End If
    Next rngC
    Application.EnableEvents = True
End Sub

Note: Repairing Office Click-to-Run products require an Internet connection to be available during the entire repair process.
  1. In Windows 7, click Start and select Control Panel from the right-hand list then select Programs.
  2. Select Microsoft Office Starter 2010 from the list of installed programs and them click Change from the commands available above the list.
  3. Check Remove Office user settings if you want to reset all options to factory defaults. Uncheck it if you wish to retain any customization that you have performed on the program(s).
  4. Click Repair and follow the procedure through to its conclusion.
If you are not speaking of Office 2010 Starter edition, the repair for standard Office editions is found in the same place, e.g. Control Panel, Programs, right-click→Change and on first screen select Repair.
