Windows usually connects to networks in this priority order:

  1. Ethernet
  2. Wi‑Fi (wireless)
  3. Mobile broadband

When you connect to a new Wi‑Fi network, it’s added to the list, and Windows will connect to that network while it’s in range. If you connect to another Wi‑Fi network while in range of the first network, Windows will prefer the second network over the first one.

Mobile broadband networks are treated differently. If you manually connect to a mobile broadband network when there is a Wi‑Fi network in range, the mobile broadband network is preferred just for that session. The next time you’re in range of both networks, the Wi‑Fi network is preferred. This is because mobile broadband networks typically are metered.

If you want to force your PC to prefer a mobile broadband network over Wi‑Fi, tap or click the Wi‑Fi network in the list of networks, and then click Disconnect. Windows won’t automatically connect to that Wi‑Fi network.

xb1r athena say:

Overview

Vizio XWR100

Login Information:

To log into your routers configuration page open up a web browser on your computer and in the address bar type in the gateway address below. You won’t be prompted for a username and password because it is not required.

  • Gateway - 192.168.1.1 or http://vizio.home
  • Username - N/A (None)
  • Password - N/A (None)

How to Find SSID (Network Name)

  1. Open a web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the wireless symbol, (The symbol on the right).
  4. Click Edit button down at the bottom of the page.
  5. The network name will be listed where it says Network Name (SSID).

 

How To Find Wireless Passphrase (WPA/WEP)

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the wireless symbol, (The symbol on the right).
  4. Click Edit button down at the bottom of the page.
  5. The wireless password will be written at the bottom of the page where it says security key.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the wireless symbol, (The symbol on the right).
  4. Click Edit button down at the bottom of the page.
  5. There will be a drop down box on this screen where you can disable or change the wireless security type.
  6. If WEP is selected, select the WEP Key Length (either 64-bit or 128-bit).
  7. If WPA-Personal is selected, select the WPA Mode (Auto, WPA or WPA2) and the Cipher Type (TKIP, AES or Both).  WPA2 with AES is recommended.
  8. If WEP security is selected the password will need to be 10 charecters in length (if 64-bit) or 26 characters in length (if 128-bit) and can only use 0-9 and A-F.
  9. If WPA or WPA2 was selected the password can be between 8-63 characters, is case sensitive and can use any character on your keyboard.
  10. Enter the Pre-Shared Key or WEP Key.
  11. Once the passphrase has been entered click Save Settings

 

MAC Filtering

  1. Open a web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the wireless symbol, (The symbol on the right).
  4. Click on Advanced.
  5. Click on Wireless MAC Filter.
  6. Delete any addresses entered on this page.

 

Change Wireless Channel

This feature is not available on this device.

 

Change Wireless Mode

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the wireless symbol, (The symbol on the right).
  4. Click on the Advanced button at the bottom of the page.
  5. Select the desired network from the drop-menu at the top of the page.
  6. You will then be able to change the mode by clicking the network mode drop-down box and changing the mode to “g only”.
  7. Select Save Changes.

Change Wireless Band (Width)

  1. Open a web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://visio.home.
  3. From the menu at the top, click on the wireless symbol. (The symbol on the right).
  4. Click on the Advanced button at the bottom of the page.
  5. Select the desired network from the drop-menu at the top of the page.
  6. Select the 20MHz option from the Channel width drop down box.
  7. Select Save Changes.

 

DHCP Reservation

  1. Open a web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the router icon.
  4. Click Advanced next to router IP.
  5. Click DHCP Reservation button.
  6. Select the xbox from the client table or enter the IP address and MAC address manually.
  7. Click Save Changes.

 

UPnP

This option is not available. 

 

DMZ

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  4. Next to where is says firewall, click advanced.
  5. Next click on DMZ.
  6. Click on the “on” button.
  7. Under Destination IP address, select the MAC address of the xbox or enter the IP address of the xbox.
  8. Select Save Changes.

 

Disable DMZ

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  4. Next to where is says firewall, click advanced.
  5. Next click on DMZ.
  6. Click on the “off” button.
  7. Select Save Changes

 

Port Forwarding

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  4. Next to where is says firewall, click advanced.
  5. Next click on Port Forwarding.
  6. For Application, type in Xbox1.
  7. For external port, enter in 3074.
  8. For internal port, enter in 3074.
  9. For Protocol select UDP in the drop-down box.
  10. For the IP address, type in the last number of the IP address of the xbox.
  11. Check the box that says on.
  12. In the next line down, type in Xbox2.
  13. Repeat 7-9 for TCP instead.
  14. Click Save Changes.

 

Disable Port Forwarding

  1.  Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  4. Next to where is says firewall, click advanced.
  5. Next click on Port Forwarding.
  6. Uncheck any “on” boxes for port forwarding rules you have set up for the xbox.

 

Firewall

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  4. For the option that says firewall, move the button to off.
  5. Click Save Changes. 

 

 How To Find Firmware Version

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  4. The firmware version will be listed on this status page.

 

Firmware Update

Through the firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open web browser.
  3. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  4. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  5. Click Advanced next to Firmware.
  6. Click “Check Latest Firmware” .

By downloading from the website

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open web browser.
  3. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  4. Click Advanced next to Firmware.
  5. Click on the icon of the router at the top. (The symbol in the middle of the three icons
  6. Open another tab in the web browser.
  7. Go to http://www.vizio.com
  8. Click on Support at the top.
  9. Click on the downloads picture.
  10. Type in the router model where it says search by model.
  11. Click on the available link under available Firmware Downloads.
  12. Save the file to somewhere that is easy to locate , such as the desktop.
  13. Go back to the original tab that you opened the router's GUI page with.
  14. Click on browse.
  15. Find the firmware bin file and click open.
  16. Click update and let the software install without interruption.

 

Restore Factory Defaults

Via Firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open web browser.
  3. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  4. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  5. Click Advanced next to Firmware.
  6. Click on Restore Defaults.

Via Reset Button

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Locate the reset button on the device.
  3. With paperclip/pen/toothpick hold down the reset button down for 30 seconds.
  4. Wait for lights to turn off then return to normal.

 

Entering PPPoE

  1. Open a web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the internet symbol(The first symbol across the top).
  4. Click on the drop down box right below internet settings.
  5. Choose PPPoE from the drop down box.
  6. Enter the user name and password provided to you by your ISP.
  7. Click Save Changes.
