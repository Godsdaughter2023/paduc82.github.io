ron sommer mvp 2009-2017 say:

Only accounts accessed using http or imap protocol will sync messages.  All folders and messages would sync including the read/unread status.

Pop3 protocol will only download emails from the server.  When using pop3 protocol, there is a setting to leave a copy of messages on the server after the messages have been downloaded to your computer.

Hotmail/outlook.com can only use http or pop3 protocol.

Macs can only use imap or pop3 protocol.

 

You can move the messages back to the Inbox.

 

You are responsible for backing up your Hotmail/outlook.com emails.

 

I am not familiar with Mac programs.  If Microsoft Office for Macs could use http protocol to access your Hotmail account, then you could create backups and sync your messages.
