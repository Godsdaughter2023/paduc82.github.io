G sind Gamerscore Punkte, die Du (wie Du richtig erkannt hast) Punkte für Erfolge in Spielen sind. Falls Du vom Playstation Lager kommen solltest: Die Playstation Trophäen sind dasselbe, nur ohne Punkte.

[quote user="JOw4R"]wenn ich jetzt zb die Halo Master Chief edition kaufe steht bei xbox rewards das man 15000punkte bekommt, heißt das sie stehen dann neben meinem gamertag?[/quote]

Nein. Die Gamerscore Punkte sind unabhängig von den Rewards Punkten.

Gamerscore sind wie Punkte bei einem Spiel: Absolut wertlos und nur dazu gedacht, sich mit Freunden zu messen ("Ha! Ich habe bei Halo schon 350G, und Du nur 345!")

Die Rewards Punkte sind hingegen ein Rabattprogramm von Microsoft, bei dem Du für verschiedene Dinge Punkte bekommst (ähnlich Payback oder Vielfliegermeilen).

Diese Punkte werden NICHT zu Deinen G Punkten gezählt. Ebenso werden die G Punkte nicht zu Deinen Rabattpunkten gezählt.

Die Reward-Punkte sind irgendwann echtes Geld wert. Sobald Du 5000 Punkte zusammen hast, wird ein kleiner Geldbetrag auf Dein Guthabenkonto bei Xbox Live eingezahlt, von dem Du Dir dann auf dem Xbox Marktplatz was kaufen kannst (5000 Punkte sind etwas mehr als 4€).

Gamerscore Punkte sind nichts wert, und Du kannst damit auch nichts kaufen.

Hello Marcia,

Thank you for posting your query on Microsoft Community. I appreciate your efforts towards resolving the issue.

As per the description, I understand that you are experiencing the printer issue. I will certainly help you with this issue.

This issue can be caused due to print spooler service or due to driver corruption. However, before we proceed I would require some more information to assist you better. Let’s work together to get this problem resolved.

1. Is this issue confined with a particular application or with all the applications?

I would suggest you to try the following methods:

Method 1:

Try to run the printer troubleshooter from the Microsoft article given below:

Open the Printer troubleshooter

http://windows.microsoft.com/en-US/windows7/Open-the-Printer-troubleshooter

If the above troubleshooter doesn’t help then try method 2.

Method 2:

Try the following steps to clear Printer Spooler files and enable the Spooler service. Check if the error persists.

1. Press the Windows key + R, Run window will be open.

2. Type services.msc the search box and press Enter.

3. Double click on “Printer Spooler" service in the Services list.

4. Click on Stop and click OK.

5. Press the Windows key + R, Run window will be open.

6. Type %WINDIR%\system32\spool\printers in the search box and press Enter. In the open window, and delete all files in this folder.

7. Now, go to “Services” again. Double click “Printer Spooler."

8. On the General tab, next to Startup type, make sure that Automatic is selected.

9. If the service is not already running, under Service status, click on Start and then click OK.

10. Check if it helps.

If the issue still persists, then try method 3.

Method 3:

Stop and Start the Print Spooler service at Command prompt.

1. Click on Start, type command prompt or cmd in the Search box.

2. Right-click on command prompt from the start search list and then click Run as administrator.

Note: If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

3. Command prompt window will be open.

4. At the command prompt, type the following lines and pressing ENTER after each command:

net stop spooler

net start spooler

exit

5. Check if it helps.

scottb_924 say:

Monitor goes black w/ games or movies in full-screen

Since upgrading from Win 7 to Windows 10, any time I play a game or movie in full screen mode, if the action on screen is very dark then the monitor will immediately go black as if it were sleeping or turned off.  This happens even with short "fade to black" scene transitions in shows like True Detective when streaming, and makes playing games like the space-trading sim Elite:Dangerous almost impossible.  If the game or show is windowed this doesn't happen.

I don't see anything in display settings that would cause this, and Power Management settings only seem to reference monitor going to sleep after 30 minutes.  Any ideas what could cause this and how to fix?

billion debbarma say:

Hi Scott,

Welcome to Microsoft Community and thank you for your interest in Windows 10.

I understand the inconvenience that you are experiencing. I will definitely assist you.

This issue could be due to the display driver is outdated. I would suggest you to update the Display driver from Device Manager manually and check if this helps. Try the below steps:

  1. Press (Windows key + X) and click on “Device Manager”.

  2. Expand “Display Adaptor”.

  3. Right click on the video driver and select “Update Driver Software”. Once update is done, restart the computer and check if the issue persists.

Hope this helps. Post us with the status we will be glad to help you.

scottb_924 say: Thanks for the response.  I had actually upgraded the video drivers previously; the GPU is a NVidia GeForce GTX 680, with driver v 10.18.13.5560 released on 8/6/2015.  This didn't seem to be the cause of the issue.  Any other thoughts welcome

scottb_924 say: Discovered the solution on a thread in another forum.  Issue appears to be related to smart contrast settings on certain monitors (ASUS, and ACER known).  Turning off ASCR (Asus Smart Contrast Ratio) resolved the issue for me.
