What should I do if BIOS does not detect my SSD? 1) BIOS may fail to support the SSD if it is outdated. Please update the BIOS to the latest version.
2) Your system may fail to detect the SSD if the SSD is not properly connected. Disconnect the cable and reconnect.
3) When there is a problem with the SATA Port, BIOS may fail to detect the SSD. Connect the SSD a different SATA Port and try again.

What should I do if I don't see my SSD in the Device Manager?

1) Your PC may fail to detect the SSD automatically; please search for new devices in Windows Device Manager
2) When Hot Plug-in for each port in BIOS settings is set to Off by default, Windows may fail to detect the SSD
Reboot the system, enter BIOS setup, and go to SATA Configuration menu to ensure that Hot Plug-in for each port is set to "Enable"
- How to enter BIOS setup for different mainboards :
GIGABYTE -->F12, ASUS --> F8, ASRock --> F11, Biostar --> F9, MSI --> F11, Intel --> F10
3) When SSD is not properly connected, the system may fail to detect the SSD. Disconnect the cable and reconnect
4) When the partition has not been defined, it may cause failure in detecting the SSD
Go to Device Manager to set up a partition, format the drive, and then reboot.
- Management> Disk Management> Select SSD> Set up partition and format

I had this same problem and after more than 10 hours with Microsoft Support, 3 tickets, and support trying to get me to delete my gamertag and re-install a perfectly working Windows 10 installation I finally found the solution. Your issue is that your settings profile saved on the cloud (this is automatic, nothing you can do about it) is corrupt. If you have Xbox One as well you can verify this by loading the game on there and it will find a corrupted save game. You also probably didn't unlock achievements, but I can't confirm this. You can also verify this without an Xbox One by making a new gamertag, loading it up, and if ReCore launches to a campaign then that's definitely the issue. To fix this you have to delete the cloud save of your profile, which is no easy task and Windows/Xbox/Microsoft Support is clueless to help you at any level. Here's how I got rid of it. You have to get on an Xbox One and log into your gamertag. I could find no way to do this on Windows 10. If you don't own an Xbox One, go to a friend's house or somehow get your hands on one for a few hours.
