The quickest way is to insert a TOC and print that.

Alternatively, if all the text that is not headings is in the same style, you could modify that style (temporarily) to be Hidden. Unfortunately, if that style is Normal, you would end up making your headings Hidden as well unless you modify their styles to be based on "(no style)" instead of Normal.

Or, if it's Normal, you could use Find to search for all the Normal text and apply the Hidden property as direct formatting (not modifying the style). If you do this on a throwaway copy of your document, you won't even have to restore the Normal text afterward. In fact, if you use a throwaway copy, you could just delete all the body text (again by using Find).
