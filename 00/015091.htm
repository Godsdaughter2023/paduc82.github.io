chip pearson say:

You can use a VLOOKUP function to do this. In B1 on SheetA, enter

 

=IFERROR(VLOOKUP(A1,[B.xlsm]Sheet1!$B$1:$B$1000,1,FALSE),"no match")

 

and fill down column A as far as you need to go. For each element in column A (SheetA), it will try to find that element in SheetB, column B. If found, the value appears in SheetA column B. If no match is found, it returns the text "no match".

If you don't want the returned IP address to appear in column B, SheetA, you can use the following to enter the text you want:

 

=IF(ISERROR(VLOOKUP(A1,[B.xlsm]Sheet1!$B$1:$B$100,1,FALSE)),"no match","match")

 

Here, "no match" is displayed if no match is found in column B sheetB and "match" is display if a match is found. You can change these to the values you want.

 

I'm not sure about your second request. Perhaps you can modify the formulas above to do what you are trying to do.
