cindy meister say:

Hi Tim

Yes, you're completely on the right track :-)

File/Information. At the right of the "Backstage" there's the "Properties" list. Click on the arrow next to the title to display the dropdown and choose the second item. (I'm looking at a German language system, so I'm not sure of the label, probably something like "Advanced properties".)

Go to the Custom tab. Here you can specify a name, a data type (Text is the default and probably what you want) and a value. Click "Add" to add the property to the list.

When you want to change the name, come back to this dialog box, click the property, change the value, then click Change (the lable of Add becomes Change when you're editing a property) again.

To insert the property value in the document, press Ctrl+F9 to insert a pair of field codes. Type:
DocProperty [Name]

So, if I create a custom document property CustomerName with a value of "Smith", my field code looks like:
{ DocProperty CustomerName }

Press Alt+F9 to toggle from field code to result display. You can copy/Paste this throughout the document.

robert aldwinckle on forums say:

 There is no history in the 'View by Order Visited Today'. If I understand correctly from your post, this means there is no repopulation of the 'most visited' possible, right?

Yes.   To see what your History actually contains you can enter:   shell:History


Otherwise to reinitialize it you should delete it, either using the Ctrl+Shift+Delete dialog (checking only History) or, if necessary by using another administrator account to delete your account's entire History folder.  It would regenerate the next time you need it.

 

HTH

Robert

oscar_wi say:

Most visited sites were not visible in my new tabs either, in Internet Explorer, until I made this change:

Go to top-right Tools / Internet Options, General and click on Tabs button. In the new window that opens, named Tabbed Browsing Settings, go to drop down list ‘When a new tab is opened, open:’ and select instead of ‘A blank page’ , ‘The new tab page’. This solved the problem for me.
