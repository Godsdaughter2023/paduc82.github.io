suzanne s. barnhill say:
I beg to differ; the end-of-row and end-of-table markers should have nothing
to do with the case. By default, if you have a greater than zero value for
the side cell margins, then the table will extend into the margins by that
amount (in order to allow the text to extend to the margin).
 
If your table is unbordered, you can decrease the left margin of the
leftmost cell and the right margin of the rightmost cell to zero, and its
edges will align with the page or column margins
 
If your table has borders, then you have to allow for those to extend into
the margins, and you will need to set the space between columns adequate for
the overhang of both tables. Alternatively, you can size the table so that
the borders fit within the margins, but the text will then be narrower by
the width of the cell margins inside the borders.

max, singapore say:

Try this sub from a past posting by Ken Wright,
it'll give you that all important col of originating sheetnames

" .. The following will do that for you, and will also add an additional column at
the start, and then put the source sheet name against every record.  If you
don't want it then just delete the column, but it will allow you to keep some of
the granularity that the separate sheets gave you.  Dump it all into a Pivot
table and you could then even recreate the spearate sheets using 'ShowPages' on
that field:-

Sub SummaryCombineMultipleSheets()

Dim SumWks As Worksheet
Dim sd As Worksheet
Dim sht As Long
Dim lrow1 As Long
Dim lrow2 As Long
Dim StRow As Long

HeadRow = InputBox("What row are the Sheet's data headers in?")
DataRow = HeadRow + 1

On Error Resume Next
Application.DisplayAlerts = False
Sheets("Summary Sheet").Delete
Application.DisplayAlerts = False
On Error GoTo 0

Set SumWks = Worksheets.Add

With SumWks
   .Move Before:=Sheets(1)
   .Name = "Summary Sheet"
    Sheets(2).Rows(HeadRow).Copy .Range("1:1")
    Columns("A:A").Insert Shift:=xlToRight
    Range("A1").Value = "INDEX"
End With

With Sheets(2)
    ColW = .UsedRange.Column - 1 + .UsedRange.Columns.Count
End With

For sht = 2 To ActiveWorkbook.Sheets.Count
   Set sd = Sheets(sht)
   lrow1 = SumWks.Cells(Rows.Count, "B").End(xlUp).Row
   lrow2 = sd.Cells(Rows.Count, "B").End(xlUp).Row
   sd.Activate
   sd.Range(Cells(DataRow, 1), Cells(lrow2, ColW)).Copy SumWks.Cells(lrow1 + 1, 2)
   SumWks.Cells(lrow1 + 1, 1).Resize(lrow2 - (DataRow - 1), 1).Value = sd.Name
Next sht

SumWks.Activate

End Sub
