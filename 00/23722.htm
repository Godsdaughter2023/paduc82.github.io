ben ball say:

I thing the Open is failing because the %appdata%

you could try building the path using Environ("appdata")

to continue with your current approach.

 

However, you may have problems when rendering the HTML to the body of the email (in HTML format) if you try and use sendobject ...

 

You could try something like this ...

 

On Error Resume Next

Dim strSignatureFile As String
Dim fn As Integer
Dim olApp As Object
Dim olNs As Object
Dim olMail As Object
  
strSignatureFile = Environ("appdata") & "\Microsoft\Signatures\" & Environ("Username") & ".htm"
Open strSignatureFile For Input As #1

 Do While Not EOF(1)    ' Loop until end of file.
    Line Input #1, textline    ' Read line into variable.
    strHTML = strHTML & textline
Loop

Close #1    ' Close file.

Set olApp = CreateObject("Outlook.Application")
Set olNs = olApp.GetNamespace("MAPI")
olNs.Logon
Set olMail = olApp.CreateItem(olMailItem)

olMail.To = "*** Email address is removed for privacy ***"
olMail.Subject = "Test"
olMail.HTMLBody = strHTML
'olMail.Send
olMail.Display

olNs.Logoff

 

Set olNs = Nothing
Set olMail = Nothing
Set olItem = Nothing
Set olApp = Nothing

Lock Cells in a Worksheet. - Microsoft Community

sajjad zahid say:

Lock Cells in a Worksheet.

I want to lock few cells in an entire worksheet, how to do this(I use MS Office 2013)?

hansv mvp say:

By default, ALL cells are locked, but this only becomes effective when you protect the sheet.

If you want to lock only a few cells:

  • Select the entire worksheet, for example by pressing Ctrl+A Ctrl+A.
  • Press Ctrl+1 to activate the Format Cells dialog.
  • Activate the Protection tab.
  • Clear the Locked check box.
  • Click OK.
  • Now select the cells that you want to lock. If necessary, you can Ctrl+click on individual cells to select them.
  • Press Ctrl+1 to activate the Format Cells dialog.
  • Activate the Protection tab.
  • Ticked the Locked check box.
  • Click OK.
  • You have now set the Locked property of the cells, but it is not effective yet.
  • Activate the Review tab of the ribbon.
  • Click Protect Sheet.
  • If you wish to set a password that will be needed to unprotect the sheet, enter it in the box.
  • Set or clear check boxes to specify what users will be allowed to do.
  • Click OK. If you entered a password, you will have to confirm it. Don't forget this password!
  • From now on, you won't be able to change the contents of the locked cell
