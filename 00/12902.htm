mikek703 say:

Don't open Word but find your normal.dotm template and right click on it (don't double click).  Select Open.  This opens your normal template.  If there is that backward P character in this file, delete it and save the file.  Close Word and then open it again to see if it is gone.  Or ... find the normal.dotm file, change the file name to read normal.old then open Word.  This will create a new normal.dotm template and the backward P should not be there.  Be aware that if you do the second suggestion, any special formatting that you may have saved to your normal template (like special margins or opening fonts or paragraph settings) will be lost and you will have to do them over again.  If things don't work out right, you can always delete the new normal.dotm file and change the name of the normal.old file back to normal.dotm

graham mayor say:
This is an end of paragraph marker ¶. You cannot delete it, but you can hide it by clicking the corresponding ¶ symbol on the Home tab of the ribbon. What hidden formatting characters are displayed by this button are controlled by File > Options > Display > Always show these formatting marks on the screen.

The formatting marks are very useful when formatting your documents and some users like to have them displayed all the time.
