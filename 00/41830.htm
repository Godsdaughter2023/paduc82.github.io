before proceeding, be sure to shut off "Windows Update", else windows will run the device driver update again on reboot.  To shut off auto-updates I believe you can make your wireless connection a metered one.  That should prevent or at least delay the auto-updates from blowing away the working display drivers once you get them installed.  If that doesnt work, find another way to turn off windows updates then proceed.

For Windows 10 Professional users, the best solution I was able to find is as follows:
