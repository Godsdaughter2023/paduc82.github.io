Another way to do this is to go to the sheets where you don't want to display the zeros and 1/0/1900 dates and then set that sheet to prevent display of zero values.

Display options for this worksheet:

[OK] back out of all the dialogs.

It does have the sometimes unwanted effect of preventing real zero values that you'd like to see from appearing in cells.

The TOC styles (TOC 1 through TOC 9) determine the formatting of TOC entries. Also, note that direct font formatting applied to headings (that is, font formatting not part of the heading styles) will be reflected in the TOC.

stefan blom say:
The TOC 1, TOC 2, etc. styles (that control the formatting of TOC entries) are set to update automatically, which means that by applying font formatting to the entries, you probably triggered an update to the styles. If an update didn't occur, the formatting will be lost the next time you update the TOC.

stefan blom say:
If the automatic update failed, you can modify the style(s) manually. Here is one way: Press Ctrl+Shift+S to display the Apply Styles pane. To modify level 1 entries, type in TOC 1 and click Modify. In the Modify Style dialog box, change the desired font settings. When you are done, click OK. Repeat for the other TOC styles, if necessary (TOC 2 through TOC 9).

Note that changing a TOC style changes the entire entry; there is no way to change the settings for the text only, or for the page number only.

suzanne s. barnhill say:
In my experience, changes to paragraph formatting in the TOC entries are propagated to the styles, whereas changes to font formatting are not. This is probably by design, but it is certainly true that changes to the number formatting will probably be lost on an update of the TOC field.
