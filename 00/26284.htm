ptay456 say:

is there a way of removing all hyperlinks in a Word doc at one time rather than searching & removing each hyperlink?

I have Word docs without noticing embedded hyperlinks that are not always visible unless I scroll over each individual word in the doc - is there any easy way to remove all hyperlinks at once? Many thanks!
john mcghie say:
I have Word docs without noticing embedded hyperlinks that are not always visible unless I scroll over each individual word in the doc - is there any easy way to remove all hyperlinks at once? Many thanks!

Yes, yu need a little macro.  Here you are:

Sub RemoveHyperlinks()
' Removes hyperlinks from either the selection or the entire document,
' and resets the formatting.
' Macro written 30 June 2005
' (c) By John McGhie
' McGhie Information Engineering Pty Ltd

Dim ahyper As Hyperlink
Dim i As Long
Dim x As Long
Dim Rng As Range

x = MsgBox("Click YES to remove hyperlinks from the whole document, Click NO to remove them just from the selected area.", vbQuestion + vbYesNoCancel, "Remove Hyperlinks Macro")

If x = 6 Then
    For i = ActiveDocument.Hyperlinks.Count To 1 Step -1
        ActiveDocument.Hyperlinks(i).Range.Font.Reset
        ActiveDocument.Hyperlinks(i).Delete
   Next
ElseIf x = 7 Then
    Set Rng = Selection.Range
    For i = Rng.Hyperlinks.Count To 1 Step -1
        Rng.Hyperlinks(i).Range.Font.Reset
        Rng.Hyperlinks(i).Delete
    Next
End If

End Sub
