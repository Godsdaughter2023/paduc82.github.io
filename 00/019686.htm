daniel g. grau say:
The convenience of this macro is, once you know it works, all you need to do is open all documents requiring fixing, ALL AT ONCE, and run the macro one final time.

To force Australian English, change the language specification from wdEnglishUS to wdEnglishAUS
(in two places).

There is a Preference setting that, curiously, is saved with each document and is probably the root of your current problem: Word > Preferences > Spelling and Grammar > Hide spelling errors in this document. I've added it to the code as well, so here's a new version with the changes in boldface, which you can add manually:

Sub ApplyEnglishAUStoAllOpenDocuments()
For Each aDocument In Application.Documents
    aDocument.Activate
    'Apply language ----------------------
    Selection.WholeStory
    Selection.LanguageID = wdEnglishAUS
    Selection.NoProofing = False
    'Redefine the Normal style ----------------
    ActiveDocument.Styles("Normal").LanguageID = wdEnglishAUS
    ActiveDocument.Styles("Normal").NoProofing = False
    'Enable spelling errors --------------
    ActiveDocument.ShowSpellingErrors = True
    'Save changes and close -------------
    aDocument.Save
    aDocument.Close
Next aDocument
End Sub

In case you don't know anything about macros, the name that follows the starting "Sub" is arbitrary (thus, you can use pretty much any descriptive name there, like Sub ChangeLang(), etc.) while comment lines start with an apostrophe and are ignored by Word.

Hi Mohammad,

When it comes to flashing cells, I think that the best solution is don't!

However, try as follows:

Alt-F11 to open the VBA editor

Alt-IM to insert a new code module

In the new module, paste the following code

'===========>>
Option Explicit
Public blStop As Boolean
Public mySheet As Worksheet

'----------->>
Public Sub Blink()
    Dim Rng As Range
    Dim rCell As Range
    Dim dTime As Double
    Const sAddress As String = "A1, A3, A5"  '\\ Cells to flash       '<<==== Change 

    If blStop Then Exit Sub
    Set Rng = mySheet.Range(sAddress)
    For Each rCell In Rng.Cells
        With rCell
            .Interior.ColorIndex = IIf(.Interior.ColorIndex = xlNone, 3, xlNone)
        End With
    Next rCell
    dTime = Now() + TimeValue("00:00:01")
    Application.OnTime dTime, "Blink"
End Sub

'----------->>
Public Sub StartIt()
   blStop = False
   Blink
End Sub

'----------->>
Public Sub StopIt()
    blStop = True
End Sub
'<<===========

Right click the tab of the worksheet of interest

Select View Code and paste the following code:

'===========>>
Option Explicit

'----------->>
Private Sub Worksheet_Activate()
    Set mySheet = Me
    Call StartIt
End Sub

'----------->>
Private Sub Worksheet_Deactivate()
    Call StopIt
End Sub
'<<===========


Alt-Q to close the VBA editor and return to Excel.

On activating the sheet of interest, the designated cells will flash; on leaving the sheet,

the flashing will be suspended.

===

Regards,

Norman
