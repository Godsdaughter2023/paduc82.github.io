josé nestorgonzález sardi say:

Getting errors in Event Log Report in XP

I learned how to get to Event Viewer.  After a quick look I found the below described issues:

From the event viewer report There is an error with the following details:  Source: Service Control Manager Catergory: one; Type: Error; Event ID: 7000;  Description:  The SessionLauncher service failed to start due the following error:  The system cannot find the path specified  Shall I be worried about this type of error?  If yes please provide fix instructions.

 

 

alex_tech09 say: This type of error means that the file or program specified has created a run registry key and when windows tries to run the program, it can't find the file that the registry key points to. The best thing you can do is uninstall the program or try to reinstall it if you need it. In this case the program is "SessionLauncher". If you don't know what it is, first run a search on the name session launcher and try to identify what it is. If it's not needed just delete any files you found to stop that error message.

Errors like this can slow down your computer since your computer then has to report the error and tries to resolve it as well.
