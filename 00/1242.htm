How do you change the Navigation pane default open setting in Word - Microsoft Community

jainga say:

How do you change the Navigation pane default open setting in Word 2010

I have Word 2010.  Upon opening any Word document the default setting has the navigation pane open on one side of the document.  I do not use the the navigation pane nor will i, so all it is a waste of space, and an annoyance to manually close it everytime I open a document.  And since i open hundreds of documents a day, its not a small annoyance.

Is there a way to default the settings in Word so the navigation pane is Not open by default?

 

jay freedman say:

Word should start up with the Navigation pane in whatever state it was the last time you shut down. If you close the pane and shut down, it should still be closed when you restart. The setting is not
stored in document files, so opening another document won't affect whether the pane is shown.

If that isn't the behavior you see, the setting might not be getting stored in the registry, maybe because of a permissions problem. In that case, you can force the pane to be turned off every time
you open an existing document by placing this macro in your Normal.dotm template (see http://www.gmayor.com/installing_macro.htm if needed):

Sub AutoOpen()
ActiveWindow.DocumentMap = False
End Sub

Jay Freedman
MS Word MVP  FAQ: http://word.mvps.org
