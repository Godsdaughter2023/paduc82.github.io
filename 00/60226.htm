m1ent say:

Ok, I have resolved this issue.

It seems my display EDID claims global support for 12bpc color depth after upgrading to Windows 10 Fall Creators (v1709) Update which causes the picture to appear scrambled if the new display modes allowed by the OS is not technically supported by the display panel.

So for anyone experiencing this issue with Nvidia Graphics Cards, a Workaround for this issue is to change your color depth from 12bpc to 8bpc from the NVIDIA Control Panel and Everything will display as normal.

To do this, Open the NVIDIA Control Panel & select Change Resolution, Followed by Selecting your Display that has an issue. 


Select the option to Use Nvidia's Color Setting, & change the Output color depth setting from 12bpc to 8bpc. 

Select Apply to save your changes and It should resolve the issue.

Hope this helps out anyone else who experienced this issue.
