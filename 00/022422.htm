Convert AM to PM using vba - Microsoft Community

markburkhardt say:

Convert AM to PM using vba

I run a report that I am able to export to Excel 2010.  I have a column that has data stored the date and time together.  The problem I am having is all the times were saved as AM.  I would like to run some VBA code on the range that would convert all times less then say 7:00:00 to PM.
bernie deitrick say: You don't have date/time values but rather strings that look like date/times. This will convert to actual date/time values:

 
Sub AM2PM()
 
Dim rngC As Range
Dim rngT As Range
Set rngT = Range("D2", Cells(Rows.Count, "D").End(xlUp))

rngT.NumberFormat = "mm/dd/yyyy hh:mm:ss AM/PM"

For Each rngC In rngT
If Hour(TimeValue(rngC.Value)) < 7 Then
rngC.Value = DateValue(rngC.Value) + TimeValue(rngC.Value) + 0.5
Else
rngC.Value = DateValue(rngC.Value) + TimeValue(rngC.Value)
End If
Next rngC

End Sub

julie fka profjulie say:
If the hx4700 is running WM5.0, you'll need to perform a clean reset (this is comparable to a hard reset in a windows mobile 2003 device):

Press and hold down the Contacts and Inbox buttons. While holding down these buttons, use the stylus to lightly press the Reset button on the bottom of the iPaq Pocket PC for about two seconds.

A dialogue box will pop up asking for confirmation. Press the Calendar button and the unit will begin the clean reset process.



The hx4700 could be downgraded to windows mobile 2003se, but you'd need to have the WM5.0 upgrade CD that was used to upgrade the device to WM5.0 - the upgrade CD included a downgrade utility on it.
