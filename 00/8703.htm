Your description is unclear to me, and I suspect I do not understand it correctly.  When asking a question, it is always best to augment an abstract description with concrete examples, e.g. "if A1 and B1 have this and that, then the formula result should be this; but if A1 and B1 have that and this, then the formula result be that".

Perhaps the following will be useful:

=IF(COUNTIF(A1:B1,">18:00")>0,A1+B1,B1-A1)

If your "2 cells" are not contiguous, e.g. A1 and C1, you might use the following IF statement:

=IF(OR(A1>--"18:00",C1>--"18:00"),A1+C1,C1-A1)

Note that you cannot write simply >"18:00" in this context.  The double negation (any arithmetic operation) converts the string "18:00" to numeric time.

I would be surprised if that formula works for you as-is.  It does not make sense to me to sum hours of the day(?) in one case and take the difference in another case.

This issue may cause due to corruption in the favorites folder. I would suggest you to try the following methods and check if it helps.

Method 1:

Run Microsoft Safety Scanner and check if it helps.

Microsoft Safety Scanner: http://www.microsoft.com/security/scanner/en-us/default.aspx

Note: The Microsoft Safety Scanner expires 10 days after being downloaded. During this 10 days, it will remove all the virus infected files and folders. I suggest you to create a backup of your data and then Install Microsoft Safety Scanner.

 

Method 2:

Try the closing all the Internet Explorer windows and then try deleting the favorite’s folders. Once after deleting the favorite’s folder, open the Internet Explorer it will automatically create a new favorite’s folder, but before deleting the favorites folder take a backup of the folder that consist of the URL’s. Once the new favorite’s folder is created copy all the URL’s from the backed up favorite’s folder and check if the issue persists.

Location of the Favorite folder: c:\users\<User name>\Favorites

Refer the following link to take a backup of the file or folder:

http://windows.microsoft.com/en-US/windows7/Back-up-your-files
