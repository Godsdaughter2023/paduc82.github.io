NOTE: If you are running a Windows 7 with SP1 you will need to run the tool in compatibility mode with Windows 7. To do this right-click on it and select Properties , click on the Compatibility tab and check the box for “ Run this program in compatibility mode for ”, select Windows 7 from the drop down list, click on apply and close the window.

When you click on the upgrade button you should give it a few seconds and the page should refresh with the following message:

“You have successfully updated your windows media security component. Please close this browser window and try playing your content again.”

If you don’t get that message your security components upgrade has failed.

 

If you did get the message then run the Zune software log into it with your account and play your music again.

 

Let us know if your issue persists.

Hi,

Although it is not clear from your query, but I think you need to make the references to the lookup array absolute. In your formula =IFERROR(VLOOKUP($A22,circ!$A22:$C151,3,FALSE),"0")you have only made the column references absolute, and the row references will keep changing as you copy the formula down.

If your lookup array (on second worksheet) is in cells A22:C151 make it absolute using dollar signs before both the column and row references, i.e. $A$22:$C$151. Thus your formula becomes: =IFERROR(VLOOKUP($A22,circ!$A$22:$C$151,3,FALSE),0)

Adjust references according to your setup.
