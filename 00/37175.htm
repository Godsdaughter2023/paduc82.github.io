a. user say:

Hello Dale,

Thank you for visiting Microsoft Community and providing us with the detailed description about the issue.

As per the description, I understand that you are experiencing issue burning CD’s using Windows Media Player on the system.

I certainly understand your concern and will try my best to help you.

In order to understand the issue more clearly, I would request you to kindly provide me with the following information.

  1. Are you getting any error message? If yes, what is the exact error message?

There is a possibility that this issue could be due to corrupt Media Player settings or due to the Windows Media Player burning disc speed is high.

I would suggest you to try the following methods and check if it helps.

Method 1:

Please follow the steps given below to change the settings in Windows Media Player.

  1. Open Windows Media Player.
  2. In the Player Library, click the Burn tab.
  3. Click the Burn options button.
  4. Then click on more burn options.
  5. Please make the following changes:

Burn CD without gaps. Select this check box if you don't want two-second gaps between each song on audio CDs that you burn.

If the issue still persists, follow method 2.

Method 2:

Run inbuilt Windows Media Player Settings troubleshooter and check if it works. Please follow the steps given below.

  1. Press Windows + W keys on your keyboard to get the search option.
  2. Type Troubleshooting and click Troubleshooting.
  3. Now click on view all option.
  4. Click on Windows Media Player Settings and follow the prompts to run the troubleshooter.

If the issue still persists, follow method 3.

Method 3:

I would suggest you to uninstall then re-install Windows Media Player and then check if this fixes the issue.

Step 1:

Follow these steps to uninstall Windows Media Player: 

a)  Press “Windows key + W”.

b)  Type “Turn Windows Features on or off” without quotes in the search box and hit enter.

c)  Expand “Media Features” option.

d)  Un-check “Media Player”.

e)  Restart the computer. 

Step 2:

Follow these steps to re-install Windows Media Player: 

a)  Press “Windows key + W”.

b)  Type “Turn Windows Features on or off” without quotes in the search box and hit enter.

c)  Expand “Media Features” option.

d)  Make a check mark next to “Media Player”.

e)  Restart the computer.

For reference:

Please refer to the article given below and check if it helps.

Burn a CD or DVD in Windows Media Player

http://windows.microsoft.com/en-IN/windows-8/burn-cd-or-dvd-media-player

All the above steps holds good for Windows 10 as well.

I hope this information is helpful.
