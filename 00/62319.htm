yaqub k say:

Hi,

Error Code 9c57 generally corresponds to a failed installation of Internet Explorer 11. In addition to that, the KB 2882822 update requires validation of genuine Windows installed on the system, so you may visit the link referred in the third question and check if your Windows requires activation. In order to troubleshoot Windows updates, I would suggest you to follow the methods below and check if you are able to install the updates:

Method 1:

To troubleshoot the issue with Windows Updates, run the Windows update troubleshooter and then try installing the updates.

Open the Windows Update troubleshooter
http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Update-troubleshooter

Also check:

Troubleshoot problems with installing updates
http://windows.microsoft.com/en-in/windows7/troubleshoot-problems-with-installing-updates

Method 2:

Troubleshooting a failed installation of Internet Explorer 11

http://support.microsoft.com/kb/2872074/en-us

Note: Please make sure that you have installed the prerequisite updates for your system as mentioned in the following KB article:

Prerequisite updates for Internet Explorer 11

Method 3:

Try to install the updates one at a time. Follow the steps:

1. Open Windows updates.

2. Click on the option where it says “important updates available”.

3. Uncheck all the updates except one and then install it.

4. Check if you get the error on all the updates or only a few of them.

Method 4:

You can download and install the System Update Readiness Tool in accordance with your Windows type from the link below.

http://support.microsoft.com/kb/947821/en-us

Try installing the updates after having installed System update readiness tool.

Hope it helps.
