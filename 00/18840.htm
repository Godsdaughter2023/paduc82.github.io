Hi Cnick_24,

Here are 4 choices you may consider:

  1. When you complete your Word document, click File > Print > Send to OneNote version on your system under the Printer tab > pick a section or page  in the Select Location in OneNote prompt > click OK. See the Select Location in OneNote prompt below.

   4. Use the copy paste method to transfer the content in your Word document to OneNote.

BTW one thing I meant to mention earlier; the size of the image file will, not surprisingly, influence the time it takes to load into the control.  With the large size of .jpg files being created by cameras these days this can take a few seconds.  It's best to use relatively small versions of the files if you want to page through them without pausing unduly for each to load.  A good strategy is to load small versions of the file into the control in the form's Current event procedure, but include a mechanism, e.g. a button on the form or double clicking on the image, to load the full size version of the file into a larger image control on a separate dialogue form when needed.  My more complex two demos do something similar, but rather than showing the small versions of the images, they list them by name in a list box.  By default the name is the filename, but can be changed to something more meaningful.
