Hi,

 

Try this. ALT+F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in. Change the column to the correct one; currently B, and run the code.

 

Sub stripe_rows()
Dim MyCol As String
Dim x As Long
Rows(2).Interior.ColorIndex = 8
' Change to correct column
MyColumn = "B"
For x = 3 To Cells(Rows.Count, MyColumn).End(xlUp).Row
    If Cells(x - 1, MyColumn) = Cells(x, MyColumn) Then
        Rows(x).Interior.ColorIndex = Rows(x - 1).Interior.ColorIndex
    Else
        If Rows(x - 1).Interior.ColorIndex = 8 Then
        Rows(x).Interior.ColorIndex = 15
        Else
        Rows(x).Interior.ColorIndex = 8
        End If
       
    End If
Next x
End Sub

ronaldoonenil say: With your company name header in A1 and the first company name in A2 then the following macro will do it

Sub Format_Row_Colour()
    Dim iRow As Integer
    Dim strCoName As String
    Dim lColor As Long
    
    iRow = 3
    lColor = 15986395
    strCoName = ActiveSheet.Range("A2")
    ActiveSheet.Range("A2").EntireRow.Interior.Color = lColor
    Do Until ActiveSheet.Range("A" & iRow) = ""
        If ActiveSheet.Range("A" & iRow) <> strCoName Then
            If lColor = 15986395 Then
                lColor = 15921906
            Else
                lColor = 15986395
            End If
            strCoName = ActiveSheet.Range("A" & iRow)
        End If
        ActiveSheet.Range("A" & iRow).EntireRow.Interior.Color = lColor
        iRow = iRow + 1
    Loop
End Sub

dave peterson say:

I like this technique:

(saved from a previous post)

Select the range that would have held the checkboxes.
Format|cells|number tab|custom category
In the "type:" box, put this:
alt-0252;alt-0252;alt-0252;alt-0252

But hit and hold the alt key while you're typing the 0252 from the numeric
keypad.

It should look something like this when you're done.
ü;ü;ü;ü
(umlaut over the lower case u separated by semicolons)

And format that range of cells as Wingdings (make it as large as you want)

Now, no matter what you type (spacebar, x, anyoldtextatall), you'll see a check
mark.

Hit the delete key on the keyboard to clear the cell.

If you have to use that "checkmark" in later formulas:
=if(a1="","no checkmark","Yes checkmark")

SA wrote:
I wanted to know in general terms the algorithm/logic/method of working, of the function and why it is giving me different answers in a wrongly sorted column

Although we cannot know the internal implementation exactly (only a Microsoft engineer can know that), we can explain the concept of the algorithm.

Presumably, VLOOKUP(...,TRUE) uses a binary search algorithm, and it returns the value corresponding to the largest match less than or equal to the lookup value.  Consider the following example.



A binary search works like how you might (should) look up a word in a dictionary with no tabs.  Let's work through the examples.

A9:  Go to the middle of B2:B8 (B5).  Since 3<6, go to the middle of B2:B4 (B3).  Since 3>2, go to the middle of B4:B4 (B4).  Since 3 < 4 and the lookup range is now one cell, the cell below (B3) must be the largest value <= 3.  So return the value in relative column 2 (C2).  You might have expected c6.

A10:  Go to the middle of B2:B8 (B5).  Since 0<6, go to the middle of B2:B4 (B3).  Since 0<2, go to the middle of B2:B2 (B2).  Since 0<1 and the lookup range is now one cell, the cell below must be the largest value <= 0.  Oh, but there is no cell below within the lookup range B2:B8.  So return a #N/A error.  You might have expected c5.

bob jones aka: cybertaz say:

Excel does not have a feature to convert the case of content in cells. You have at least 2 other options though:

  1. In an another column use the =UPPER() function, then copy the result & use Edit> Paste Special - Values to paste into the original cells. You then can delete the column of functions, or
  2. Cut the column, paste into a blank Word document, convert the case there, then copy back into the Excel file.
