I found the error of my ways - I wasn't using a comma when I tried to put the code together with your code. This uses a semicolon (and space) delimiter - the send on behalf names dont run on and could be split using text to column.

franksam23 say:

How to configure email account in Outlook Express?

Some websites offer me a chance to e-mail them. Outlook Express comes up with a message that asks if I want to make it my default e-mail, and asks for POP (inbound) and SMTP (outbound) server names in order to do so. How do I find those names?

(Original title: POP3 and SMTP)

a. user say:

Hi Franksam23,

 

Based on your post, it looks like you want to know how you can add your email account in Microsoft Outlook Express which is an email client installed on your computer.

 

To add your email POP3 account in Outlook Express, please follow these steps:

 

Note: If you are moving from DAV to POP3, there are a few important differences. With POP3, there is no synchronizing. For example, if you read an e-mail message in Outlook Express the message will be unread if you view your Outlook.com account on the web. Also, only messages in your Outlook.com Inbox will display in Outlook Express. Other folders, and any messages in them, will not be displayed.

 

1. In Outlook Express, on the Tools menu, click Accounts.

2. Click Add, and then click Mail.

3. Type your name, and then click Next.

4. Type your full Outlook.com e-mail address, and then click Next.

5. Enter the following information, and then click Next:

     · In the My incoming mail server is a list, choose POP3.

     · In the Incoming mail (POP3, IMAP, or HTTP) server box, type pop3.live.com

     · In the Outgoing mail (SMTP) server box, type smtp.live.com

6. Type the account name (type your full MSN address if you have an @msn.com account) and the password for your Outlook.com  account. Clear the Remember password check box unless you're the only person with access to the computer. Make sure that the Log on using Secure Password Authentication (SPA) check box isn't selected.

7. Click Next, and then click Finish.

8. In the Internet Accounts window, click the Mail tab, click the Outlook.com account that you added, and then click Properties.

9. Click on the Servers tab and under 'Outgoing Mail Servers', please check the "My server requires authentication" box.

10. Click on the Settings button to the right of the "My server requires authentication" option. You may need to turn on the "Use same settings as my incoming server" option unless you are using a special account setup requiring a different Account name and password. Most people will need to use the default option of "Use same settings as my incoming server".

11. Click the Advanced tab, and under Server Port Numbers, enter the following information, and then click OK:

     · Under both Outgoing mail (SMTP) and Incoming mail (POP3), select the This server requires a secure connection (SSL) check box.

     . In the Incoming mail (POP3) box, ensure it is 995 and in the Outgoing mail (SMTP) box, type 25 or 587 (If port 25 has been blocked in your network or by your ISP, you can set SMTP port to 587)         

     · Under Delivery, select the Leave a copy of messages on server check box if you want to see your messages when you use Outlook.com in a web browser, your mobile phone, or other e-mail programs. By default, Outlook Express will delete the messages from the Outlook.com server when they are downloaded to your computer.

Reminder - If you have not selected the Leave a copy of messages on server checkbox, the emails will be deleted from the Outlook.com server upon synchronizing and will only be available on your computer.

 

12. Click OK.

13. On the Internet Accounts window, take a look at your mail accounts by selecting the Mail tab. If you see your Outlook.com  account with Type 'HTTP', select the account and click the Remove button (NOTE: this should apply if you are moving from the DAV protocol to the POP protocol - you've added an account with POP through the above instructions and now need to delete the previous account accessed through DAV).

 

In line with this, you can also review this article: Send and receive Outlook.com email from an app which gives details on how you can manage your email account in using email applications.

 

Note: All the steps provided on this post including the article applies if you have a Microsoft account which you want to configure in your Outlook  Express email client. Otherwise if it's not, please contact your email provider's support team for more information about it.

 

It would also be best to contact the Outlook Express support if you have additional questions about configuring your account on the said application.

 

If you have any questions and clarifications, please let us know.

Waiting for the update to come through is one option. Not the best choice in my opinion. I would ask you to go to task manager, click on the services tab, and make sure windows update is running. You may or may not have to opt back into the insiders program if you want to continue to be an insider. I have not tried that, so I don't know for sure.

Choose how you want to upgrade and the type of device/media being used.

When completed, just install it as you would any other program on media. Just be sure to do a backup if you have data that is important that you do not want to loose.

luc sanders say:

Otto,

You can only set one level bulleting in a textbox in PPT. If you indent on a second level you will get the same symbol. The settings you apply using set as default textbox are at a presentation level. If you want all your textboxes to have the same bulleting you will have to set them in a template or edit Blank.potx:

http://www.pptfaq.com/FAQ01010_Create_your_own_default_presentation.htm

At least that has been my understanding, are you sure you have not used a new slide layout created with a custom placeholder. Starting with PPT 2007 that could be the case.

Luc Sanders
MVP - PPT

ute-s say:

Text boxes are meant for short text. Thus you cannot define more than one bulleted text level.

If you need long text with more than one level of bulleted text, you should consider creating a new layout with custom text placeholders. There you can define bullets and indents for up to nine levels (though I do not recommend using more than three).

Best,
Ute

    You will have to do this for all horizontal ranges separately. Hence, do CF for D10 to FG10.

    Click D28 once to apply to D28:FG28
    Now press ESC to switch off Format Painter
