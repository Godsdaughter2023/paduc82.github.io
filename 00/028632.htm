As you already indicated based on...

"As far as the pdf goes: I only have Reader, so apparently I cannot adjust paper size for pdf. But the printer can just cut the paper to the right size and it should print out correctly anyway - it looks right on the pdf page except for the giant margins, a function of the standard letter-size paper that seems to be the only one available in Reader."

You should be at an acceptable point so I'll try to provide a condensed explanation:

  • You have to make the distinction that once the PDF is created Word is no longer involved. The dimensions of the document are what you specified in Word. If you want verification;
  1. In Preview, open the PDF, go to Tools> Show Inspector then make sure the General Info Inspector panel is displayed. What is indicated as
  2. In Adobe Reader go to File> Properties & see what Description panel.
  • Further, when you print the PDF you're using the OS X Print Services supplied to the pdf reader program you're printing from [i.e., not Word]. Both Preview & Adobe Reader automatically use your printer's default paper size, which is, typically, US Letter. Therefore, the result is that the Half Letter document is previewed or printed on a Full Letter sheet of paper.
  • OTOH, if you open the 5.5"x8.5" Word Document in Word (rather than opening the PDF file in a reader) then go to File> Print you'll see that the document does preview at its true dimensions. (It's the PDF app that imposes the US Letter paper size.)
  • If you actually want to display (preview) [or print] the document as Half Letter you need to make the change in the Print dialog of the PDF reader program you're using. IOW, you have to tell the program you want to print the smaller document on smaller sheet of paper -- the program doesn't automatically pick a paper size to match the document dimensions;
  1. In Preview, once you go to File> Print just open the Paper Size list & pick the preferred built-in or custom paper size.
  2. Reader is a bit more challenging... Once you go to File> Print you then need to click the Page Setup button -- Only to be met with the admonition that you not do what you need to do. Click the Yes button then select the preferred Paper Size.

BTW: The sizes you see in the Paper Size list are determined by the printer you have selected at the time. Each printer offers something a little different. [My Epson doesn't offer the Hagaki (Japanese postcard) size, for example.]

If the exact dimensions you want aren't included in your printer's offerings you can create a Custom Paper Size. I have one I named eBook which is 6"x9" & another I named SmallSquare which is 4"x4", as well as a few more.

        If IsNumeric(cell.Value) Then
           CellFormat = cell.NumberFormat
           If CellFormat <> "General" Then
              FormatCell = False
              
              If InStr(1, CellFormat, "%") = 0 And Not Percent Then FormatCell = True
