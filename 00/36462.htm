If you have updated Windows 10 recently, you may have to grant applications permission to use the camera. This is because, in the latest versions of Windows 10, some applications do not have default access to the camera.

To allow applications to access the camera, select the Start button and then:

Settings> Privacy> Camera. Next, enable Allow apps to access the camera.

Also, you can search for an obsolete webcam driver.

1. Select the Start button, type Device Manager and select Device Manager from the search results.

2. Search your webcam on Image Devices or on Sound and Video Controllers and game devices.

3. Press and hold the name of the webcam (or right-click on it) and select Properties.

4. Select the Driver tab, select the Driver Details button and look for a file name that includes stream.sys. If one exists, the webcam was designed before Windows 7 and you will have to replace it with a newer webcam.

5. If you can not find a file name that includes stream.sys, try reverting to the webcam driver.

Revert to the webcam driver

1. In Device Manager, press and hold the webcam (or right-click on it) and select Properties.

2. Select the Driver tab, select Revert to the previous driver, and then choose Yes. (Note that some drivers do not provide a rollback option).

3. Once the reversion has been made, restart your PC and try to open the Camera application again.

4. If the rollback has not worked or is not available, uninstall the webcam driver and look for hardware changes.

Uninstall the webcam driver and look for hardware changes

1. In Device Manager, press and hold the webcam (or right-click on it) and select Properties.

2. Select the Driver tab, select Uninstall> Remove the driver software from this device and then select OK.

3. In Device Manager, on the Action menu, select Search for hardware changes. Wait for the updated drivers to be found and reinstalled, restart your PC and try to open the Camera application again.

4. If the webcam still does not work, try to search for help on the camera manufacturer's website.

a. user say:

Hi Marilyn,

 

Thank you for posting your query in Microsoft Community. I regret the inconvenience caused to you. Let me help you.

 

After upgrading to Windows 10, the files and folders that was saved in previous Windows version should be present in the system. If it is not present, then it will be in Windows.old folder.

 

When you upgrade to Windows 10, the previous Windows files and folders are automatically stored in C:\Windows.old folder. This folder is kept by Windows only for one month. After one month Windows 10 will automatically delete the folder. So you can restore previous Windows version only within one month after Windows 10 upgrade. Also if you run Disk Cleanup in Windows 10, make sure to uncheck "Previous Windows installation(s)" option. If you don't uncheck it, Windows will delete all files and folders present in C:\Windows.old folder and you'll be unable to rollback to previous version of Windows.

 

As a work around, check if you can see the files and folders that are lost after checking Hidden items option

 

Step 1: Open File Explorer by pressing Windows key + E.

Step 2: On the ribbon file located on top of the screen go to View tab.

Step 3: Check the box hidden items.
