joe men say:

Hello Jeannine,

Depending on how you have set up your email accounts, some of your email messages will not be synced. For example one of your accounts is set up a POP (Post Office Protocol) from one device and IMAP (Internet Message Access Protocol) from another, you will not be able to see the messages if the device that is using POP that already downloaded the message from the email server.

You will need to make changes to the device that is using POP to leave a copy of the messages on the mail server so the device that is using IMAP will be able to view the messages.

The best solution for your concern is to configure all of your email accounts as IMAP across all of your devices. This way all of your messages can be viewed on any email clients that you are using.
