amit tandon say:

In case you may want to try below:

=IF(ISTEXT(B20),B20,IF(B20>0,(B20/1000)*($B$13*1000),0))

If B20 is text, the formula returns B20 itself;

If B20 > 0, formula returns (B20/1000)*($B$13*1000);

If B20 is <= 0, formula returns 0.
