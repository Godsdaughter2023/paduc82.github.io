How can I merge both .OST files so I can have all my folders and emails back.
<<

Unfortunately an ost file can only be opened by the original Outlook profile/email account used to create it. The only other option is getting a 3rd party OST2PST conversion program. Email folders for an Imap account get sync'd with the folder whenever the account is configured or when the .ost file is recreated whicch happens if Outlook can't find the file.

For reason provided above, there's no point retaining a backup file of the ost file. Also, contact/calendar folders would be marked as "This Computer Only" and info in these folders should be exported to a PST file to avoid risk of losing the data. The better option would be to maintain your contact/calendar data in a PST file set as the default data file for the profile. The default data file is where the default contact/calendar folders are located.

Speed up when new builds are installed - Revised 03/31/15 - Microsoft Community

j w stuart say:

Speed up when new builds are installed - Revised 03/31/15

Technical Level : Basic

Summary

Starting with build 9860 the Technical Preview has a new option to choose how quickly future new builds reach you, it’s called the “Fast Ring”. By default everyone is in the “Slow Ring”.

Recent changes to the later builds have also changed what steps you need to perform to change to the “Fast Ring”.


Details

Click on the ‘Start Menu’ and click on ‘PC Settings’.

In the list of settings, click on the ‘Update and recovery’ option.

In the Windows Update window, click on the ‘Advanced options’ link.

At the bottom of the ‘Advanced options’ window you will see the ‘Choose how preview builds are installed’ option, the default setting is ‘Slow’.

If you click on the drop down arrow you’ll see two options: ‘Fast’ or ‘Slow’. By default, your PC will be in the ‘slow’ ring (as Microsoft names it).

 

If you want to install a new build as soon as it becomes available, you’ll want to be in the ‘Fast’ ring.

Click on the drop down arrow, move you cursor over ‘Fast’ and click to change the setting.

Click on the ‘Back Arrow’ in the upper left corner of the ‘Advanced options’ windows. Fast means that you’ll get it downloaded via Windows Update on the day it is released. Then click the ‘Check for updates’ button.

If a new build is available, based on the type of build is that is being released, you will see an update named: fbl_awesome or fbl_impressive with the build number.

 

When the Windows Update download completes you will be prompted to “Install” the new build. Click to start the install. This may take some time to install and complete and your PC will restart. So relax and keep yourself occupied until you are prompted near the end of the installation.

 

Note: If you see a “Failed to check for new preview build, please try again later. 0x8024402C error message, reboot your computer to clear the message.

 

Per Microsoft: “If you want to wait a bit and let others find any gotchas so that the community gets caught up with known issues and workarounds, then you’ll want to stay in the ‘slow’ ring.”

Pre-requisites

  1. Office 2013 valid license key

Procedure:

  1. Open the regedit and make a backup of the following keys HKLC\Software\Microsoft\Office and HKCU\Software\Microsoft\Office. (This will be used if something gets wrong.)
  2. Go to "Add Remove Programs" and uninstall Office 2013.
  3. Restart the computer.
  4. Open the regedit and delete the following keys HKLC\Software\Microsoft\Office and HKCU\Software\Microsoft\Office. (One of them will show an error after deleting the key. Just ignore it and press Ok.)
  5. Restart the computer.
  6. Reinstall Office2013. (This will recreate the deleted registry keys)
  7. Configure the email Outlook.
  8. Now the user can edit the shared calendars and contacts.
