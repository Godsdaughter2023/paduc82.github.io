Command for "Don't add space between paragraphs of the same - Microsoft Community

rxaverius say:

Command for "Don't add space between paragraphs of the same style"

since I would like to create a shortcut for that option I wonder if there is a command to toogle "Don't add space between paragraph of the same style", an option of the Paragraph Dialgo

jay freedman say:

There's nothing you can add to the macro. Either it works or it doesn't. It's well known among VBA programmers that the SendKeys function is flaky, sometimes sending keystrokes into the great unknown instead of to the dialog that's being opened. However, the fact that the dialog closes immediately means that at least the {Enter} part is being sent correctly.

One thing you can do is, after running the macro, manually open the Paragraph dialog (with the cursor still in the same place) and see whether the "Don't add space" box has toggled from its previous value. If it has changed but you don't see a difference in the spacing of the text, then there's something about the formatting of the current paragraph and/or the ones before and after it that prevents the "Don't add space" setting from doing its job.

You can use the Reveal Formatting pane (Shift+F1) to compare the formatting as you move the cursor from one paragraph to another. (Hint: It's useful to check the box in the pane for "Distinguish style source" so you can see what's in the style vs. what's from direct formatting.)

jay freedman say:

The default assignment of the Ctrl+0 shortcut is the command OpenOrCloseUpPara. In the Keyboard Shortcut dialog it appears in the All Commands category, but I couldn't locate it in any of the more specific categories. It doesn't appear in the Quick Access Toolbar customization list at all.

However, it doesn't appear to do what you want -- if the paragraph's style includes Space Before, then pressing Ctrl+0 first adds more space before the paragraph, and a second press removes that extra space but leaves what's specified by the style. It doesn't affect Space After at all.

As to your other question: Word does not have a built-in way to manage the Space After of a selected paragraph and the Space Before of the following paragraph in the same action. As with many other things, that can be managed with a macro (and this one wouldn't involve SendKeys, so it should work as advertised). If it doesn't behave the way you want it to, tell me with before-and-after examples and the definitions of the styles involved.

Sub NoSpaceBetweenParasOfSameStyle()
    ' Remove any Space After of the first paragraph
    ' in the selection, and any Space Before of the
    ' paragraph that immediately follows it, if both
    ' paragraphs have the same style. If the styles
    ' are different, only the first paragraph is
    ' affected.
    
    ' Note: if the selection contains more than one
    ' paragraph, this macro acts only on the first two.
    
    Dim Para1 As Paragraph, Para2 As Paragraph
    
    Set Para1 = Selection.Paragraphs(1)
    Set Para2 = Para1.Next

    Para1.SpaceAfterAuto = False
    Para1.SpaceAfter = 0
    
    ' If the selection is in the last paragraph of
    ' the document, Para2 will be Nothing, so the
    ' settings would cause an error.
    If Not Para2 Is Nothing Then
        If Para1.Style = Para2.Style Then
            Para2.SpaceBeforeAuto = False
            Para2.SpaceBefore = 0
        End If
    End If
End Sub
