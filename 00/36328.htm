    Hi Ronald,

    A system image is an exact copy of a drive. By default, a system image includes the drives required for Windows to run. It also includes Windows and your system settings, programs, and files. You can use a system image to restore the contents of your computer if your hard disk or computer ever stops working. When you restore your computer from a system image, it's a complete restoration—you can't choose individual items to restore, and all of your current programs, system settings, and files are replaced with the contents of the system image.

    I suggest you try the below methods and check if it helps.

    Method 1:Change the Windows Installer service to Automatic.

    I suggest you to boot into the safe mode with Networking and try to change the Windows Installer service to Automatic.

    Follow the below steps to change the Windows Installer service to Automatic:

    1. Press Windows key + R from the Keyboard.
    2. Type services.msc and hit Enter.
    3. Search for Windows Installer Service.
    4. Double click on Windows Installer and click on Start button.
    5. Click on the dropdown list and set the Startup type to Automatic.

    To start your PC in safe mode in Windows 10, Refer to the below link:
    http://windows.microsoft.com/en-us/windows-10/start-your-pc-in-safe-mode

    Method 2: Reregister the installer engine.
     
    Open  Taskbar and search for the process MISexec.exe is running. If before installation you see this process running then it means that there is a process which is using this service. Follow the steps given below.
     
    a) Press Windows key + X key, select Command prompt (Admin) to bring up elevated command prompt.
    b) At the command prompt, type the following, and then press Enter after each line:
    MSIExec /unregister
    MSIExec /regserver
    c) Try to install or to uninstall again.
