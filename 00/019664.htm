Public Sub OpenFileList(Optional myString As String = "bad")
Dim fs, f, f1, fc, s As String, WB As Workbook, SourceWb As Workbook

Set fs = CreateObject("Scripting.FileSystemObject")
Set f = fs.GetFolder(ThisWorkbook.Path)
Set fc = f.Files

Application.ScreenUpdating = False

For Each f1 In fc

DoEvents

End If

Next

In WORD 2010 how do I remove all character and line spacing - Microsoft Community

terrybarnell say:

In WORD 2010 how do I remove all character and line spacing

In Word 2010, I would like to remove all horizontal and vertical spacing between each characters.

Therefore each character touches the next character on each line and each line of characters touches the next line of characters down the page.  As a result there will be no gaps between the letters and numbers.

Can this also be produced in a text box such that there is no blank margin inside the box. ie. the characters touch each other and touch the edges of the text box?

This is to be used for a graphical design using large formatted characters with a textured fill. 

macropod say:

To get characters on the same line to touch, you can condense the character spacing (Font|Advanced>Spacing>Condensed). A reduction of around 10% of the point size might be suitable, but that really depends on the font.

As for the vertical spacing, it's impossible to get a consistent result with lower-case letters, due to the presence of ascenders and descenders. Even with capitals it's impossible with some fonts due to their character glyphs including white space above/below. The only way around this is to use textboxes that you can overlap.

stefan blom say:
For the "vertical part," you will have to set the line spacing to an exact value, identical to (or a bit smaller than) the point size of text, and set the spacing before/after to zero. Select the text paragraphs and click Paragraph on the context menu. You'll find the options on the Indents and Spacing tab of the dialog box.
