¡firedog say:

I am using Windows Live Mail v 2009 on one PC and Windows Live Mail 2011 on another one.  

In another thread, you said that you had Windows 10 on two machines. Windows Live Mail 2009 is no longer supported, and Windows Live Mail 2011 only on Vista. You should upgrade both programs.

Both programs were have sub folders for "Backup" under Windows Live Mail in the App Data settings. There are no files in the Backup folder on the pc running v 2009 but there are files in "Backup" folder on the pc running v 2011.

The Backup folder doesn't contain any messages. It exists only to contain a recent version of the message database index in case anything should happen to the current one.

1.  Why isn't the messages being saved to the PC running v 2009?

Again, the Backup folder doesn't contain messages. It will only contain a copy of the database index after the current one has been 'compacted', by default after every 100 program starts.

2.  I am using a backup program with external hard drive.  Can I simply include the entire App Data/Windows Live Mail folder in my backup set

Yes.

... and be able to restore it to any Windows Live Mail client application

That is not so likely. You should be able to import the messages it contains, but it depends very much on how the backup program works whether this will be possible or not. On Windows 10, it's easy to use File History to do what your current backup program is doing. This will also enable you to see previous versions of the folders containing your messages.

3. Is there anyway to create automatic backups of my email to my One Drive cloud account?

I don't think there's anything to stop you from relocating your store folder to a folder in OneDrive. When viewing your message list, press Ctrl-Shift-O for Options. On the Advanced tab, click Maintenance and then Store folder. Select a new folder in your OneDrive folder for the new location.
