It's a good idea to do this every 2 or 3 days as preventive maintenance

6. Open Run from [Windows logo key + r] Type in iexplore.exe -extoff (iexplore.exe<space>-extoff) then hit Enter. This will open Internet Explorer in no add-ons mode. If performance improves, troubleshoot add-ons to find the troublesome one.

For anyone else running into this issue, I was able to fix it. Here are the steps

Right click on the CPA icon from the Start menu and select "Open File Location"
