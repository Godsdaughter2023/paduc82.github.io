xpat.dll

a. user say:

Xpat.dll (AOL Connectivity Service XML Parser) is a component from the software AOL Connectivity Service version 2.0.1 by America Online, Inc. Xpat.dll version 2.0.1 has a file size of 90,112 bytes, and is most commonly found under the directory "ACS" with a creation date of August 30, 2005. This is not a known spyware, adware, or trojan component.  If you use AOL Connectivity Service and the file meets these parameters (or you can't find it), then you probably need to reinstall AOL Connectivity Service to get the file back.  If the file is a different size, location, or date (or if you've never used AOL) then it is probably malware.  If you can't find the file, then your current AV program probably already cleared most of the infection and this is what remains (or your AOL program lost the file somehow if it is/was legitimate). 

If it is a remanant of an infection that wasn't fully deleted (you've never used AOL) then perhaps we can find it and remove it so it no longer appears.  Download and install Autoruns http://technet.microsoft.com/en-us/sysinternals/bb963902.aspx and run it to see if you can find the xpat.dll entry and then uncheck it or delete it and that should end the problem.

What is your current AV program and version?  Did you get any error messages or notification messagrs from it?  Try to run anti-malware programs (in safe mode with networking if necessary). To fix this problem (if it is a problem) download, install, and run the following two programs: http://www.malwarebytes.org/mbam.php and http://www.safer-networking.org/en/index.html.  You may also want to try a free trial of Pandasoft (http://www.pandasecurity.com/usa/homeusers/solutions/global-protection/).  Before downloading pandasoft, uninstall whatever anti-virus program you are currently using (because installing and running two at the same time can cause conflicts, freezes, and all sorts of problems).  Once you've uninstalled your current AV program, download, install, update, and scan using Pandasoft.  I think you might be amazed at how much it finds that the other software missed.  Once complete, uninstall pandasoft and re-install your current AV program (unless you decide to switch which I did when I first tried it about 4 years ago).  You may also want to try the new, free Microsoft Security Essentials http://www.softpedia.com/progDownload/Microsoft-Security-Essentials-Download-131683.html (with the same caveat that only ONE AV program can be installed and running on your system at any one time).  You may also want to try the free Avira at: http://www.free-av.com/ and Avast at: http://www.avast.com/index.  Reboot after completing all the scans.  You may also want to try the free OneCare at http://onecare.live.com/site/en-us/default.htm and let it run all the options (except the registry cleaner) because that’s good maintenance (it will take some time to complete but can be done in the background).
