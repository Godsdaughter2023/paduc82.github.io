How to coerce Solver to change two cells? - Microsoft Community

joeu2004 say:

How to coerce Solver to change two cells?

Refer to the "solver" worksheet in solver.xlsm (click here) [1].  Crossposted to mrexcel.com/forum.


I want Solver to modify D1 (interest rate) and H1 (date) until I7 (delta) is minimized.  That does not seem to work.


It does not even seem to work when I just change H1.  For example, for D1 = 352.50% (initial state), I7 is smaller when H1 is 6/30/2014.  Solver sets H1 or leaves H1 set to 1/1/2014 (initial state).


In the "bestAPR" worksheet, I implement a macro which discovers (one of) the best combinations of D1 and H1 by brute force.  It runs in less than 8 sec on my (ancient) laptop.  One "best" answer is D1 = 353.36% and H1 = 10/30/2014.  In that case, I7 = 0.06%.


How can I coerce Solver to find about the same or equivalent solution or better?


-----


I hope the solver.xlsm workbook is self-explanatory.  The following is for those who prefer to read a description here.



Formulas:

B2: =C2+D2

C2: =$E$1/5

D2: =E1*$D$1/12

E2 =E1-C2

Copy B2:E2 down through B6:E6

B7: =SUM(B2:B6)

D7: =SUM(D2:D6)

G1: =-(E1+F1)

G2: =B2

H2: =EDATE($H$1,ROW(H2)-ROW($H$1))

Copy G2:H2 down through G6:H6

G7: =XIRR(G1:G6,H1:H6)

G8: =XNPV(G7,G1:G6,H1:H6)

I7: =ABS(G7-2261%)


Solver set-up:

Objective cell:  I7
To:  Min
By changing:  H1, D1    (or just H1)
Constraints:
    D1 <= 353.5%
    D1 >= 352.5%
    H1 <= 12/31/2014    (or 42004)
    H1 >= 1/1/2014      (or 41640)
    H1 = integer
Select "Make unconstrained variables non-negative"
Solving method:  GRG nonlinear
Options:  default

bestAPR macro:

In the "bestAPR" worksheet, I implement a macro which discovers (one of) the best combinations of D1 and H1 by brute force.  It runs in less than 8 sec on my (ancient) laptop.  One "best" answer is D1 = 353.36% and H1 = 10/30/2014.  In that case, I7 = 0.06%.


Sub bestAPR()
    Dim d As Double, minD As Double
    Dim x As Double, minX As Double
    Dim r As Double, rint As Long, minR As Double
    Dim st As Single, et As Single
    Application.StatusBar = "Solving..."
    DoEvents
    Application.ScreenUpdating = False
    minX = 1E+300: minD = 0
    st = Timer
    For rint = 35250 To 35350
        r = rint / 10000
        Range("d1") = r
        For d = #1/1/2014# To #12/31/2014#
            Range("h1") = d
            x = Range("i7")
            If x < minX Then minR = r: minD = d: minX = x
        Next d
    Next rint
    et = Timer
    Range("d1") = minR
    Range("h1") = minD
    Application.ScreenUpdating = True
    Application.StatusBar = ""
    DoEvents
    MsgBox Format(CDbl(et - st), "0.000000") & " sec"
End Sub

    Y As Long

Private Declare Function GetCursorPos Lib "user32" (lpPoint As Point) As Long

Private XPixelsPerInch As Long
Private YPixelsPerInch As Long
Private Ratio As Single

Private Moving As Boolean
Private DragShp As Shape
Private TimerID As Long
Private HostObj As Class1

Private OrigShpLeft As Single
Private OrigShpTop As Single
Private OrigMouseLocation As Point

Sub MoveShape(ByVal Shp As Shape)
    Dim hDC As Long

    On Error Resume Next

    If SlideShowWindows.Count > 0 Then
        If Moving Then
            EndMoveShape
        Else
            hDC = GetDC(0)
            XPixelsPerInch = GetDeviceCaps(hDC, LOGPIXELSX)
            YPixelsPerInch = GetDeviceCaps(hDC, LOGPIXELSY)
            ReleaseDC 0, hDC
   
            Ratio = Shp.Parent.Parent.SlideShowWindow.View.Zoom / 100#
   
            Set DragShp = Shp
            OrigShpLeft = Shp.Left
            OrigShpTop = Shp.Top
            GetCursorPos OrigMouseLocation

            StartTimer
            Moving = True
            Set HostObj = New Class1
        End If
    End If
End Sub

Sub EndMoveShape()
    On Error Resume Next

    Set HostObj = Nothing
    Moving = False
    StopTimer
    Set DragShp = Nothing
End Sub

Private Sub StartTimer()
    On Error Resume Next

    TimerID = SetTimer(0, 0, 10, AddressOf TimerProc)
End Sub

Private Sub StopTimer()
    On Error Resume Next

    KillTimer 0, TimerID
End Sub

Private Sub TimerProc(ByVal hwnd As Long, _
    ByVal uMsg As Long, _
    ByVal wParam As Long, _
    ByVal lParam As Long)

    Dim CurMouseLocation As Point
    Dim DeltaX As Single
    Dim DeltaY As Single

    On Error Resume Next

    If Moving Then
        GetCursorPos CurMouseLocation
        DeltaX = (CurMouseLocation.X - OrigMouseLocation.X) * _
            TWIPSPERINCH / 20 / XPixelsPerInch / Ratio
        DeltaY = (CurMouseLocation.Y - OrigMouseLocation.Y) * _
            TWIPSPERINCH / 20 / XPixelsPerInch / Ratio
        DragShp.Left = OrigShpLeft + DeltaX
        DragShp.Top = OrigShpTop + DeltaY
    End If
End Sub
