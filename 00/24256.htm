Applying a theme applies theme fonts, theme colors and theme effects; it won't directly affect table styles, as you have noticed. For more on themes, see https://support.office.com/en-us/article/What-is-a-theme-7528ccc2-4327-4692-8bf5-9b5a3f2a5ef5?CorrelationId=475df827-96d6-48b5-bb81-7913d37a8225&ui=en-US&rs=en-US&ad=US&ocmsassetID=HA010338417.

What you can do is run a macro that applies the table style to all tables in a document. Something like this will do the trick:

Sub ApplyTableStylesToAllTablesInDoc()
Dim t As Table
For Each t In ActiveDocument.Tables
t.Style = "Table Style Name Here"  'Specify the style name
Next t
End Sub
