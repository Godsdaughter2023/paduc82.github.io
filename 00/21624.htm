bob jones aka: cybertaz say:

Command+W in Word is assigned to the FileClose command & has been since long before OS X came into being. My guess is that the assignment has been retained in order to maintain continuity for those of us who have been using the program since it was introduced. It wasn't until long after Word was well established that Apple penned new guidelines that contradicted software in use at the time. Be that as it may or may not be...

It's simple enough to create a 1 line macro to get the behavior you'd prefer.

  1. Open any 2 document windows [They need not be windows of the same file]
  2. Start recording a Macro
  3. Give it a name using no spaces or punctuation
  4. Assign a keyboard shortcut - use Command+W if you wish
  5. Click the Close button of either window
  6. Stop recording

Your keystroke will now be assigned to the ActiveWindowClose command instead of FileClose.
