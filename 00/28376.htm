bob jones aka: cybertaz say:
Jim makes a good point about being able to 'unwrap' the OOXML files to extract the media content. One point you'll discover, though, is that the stored file isn't always in the same format as the original... especially if copy/paste has been used repeatedly or the source was a different document type, etc. The image gets converted to whatever flavor of image file format is in vogue for the OS in use when the copy/paste is done.

I do suggest a different method than what Jim did for expanding the OOXML files... If you have a recent version of StuffIt Expander you can simply drag the file to that app's Dock icon. A new folder will be created in the same location as the file. The folder will have the same name as the file followed by the word Folder. The reasons I prefer this method are;

  1. You don't have to take the time to edit the extension of the file name.
  2. If you do change the extension you may not be able to put the file back together again once it's been unzipped, so additionally...
  3. You may be out of luck if you don't have a backup of the file before unzipping

And yes, the linked file names are included in the document.xml file when you unzip the OOXML but it can be a real picnic trying to locate it if you don't have some idea what it is to begin with [did someone say "conundrum"?] and I've yet been able to locate the actual path to where the file was when inserted.

a. user say:

Hello EdwardSmart,

Thank you for posting on Microsoft Community.

I understand that you put a wrong Microsoft account (spelling mistake) as primary on your phone, and that now you can’t update your phone or your apps, and that you can’t access the Store.

To change which Microsoft account to use, you will have to hard reset the phone. The soft reset won’t allow you to do this.

First of all, please check exactly what address is displayed as the Microsoft account (the first in the list) in settings > E-mail + accounts.

Then, try to login this address here:

https://people.live.com/

And check that you see all your phone’s contacts.

If you can login, that is good news, because that means that after the hard reset you will be able to add this misspelled account as a secondary account on your phone to restore your contacts.

If you can’t login, then you can try to reset the password from this page:

https://login.live.com/

If you can’t reset the password, then you will have to back up manually all your contacts, because the hard reset will wipe your phone’s memory.

You can also back up your photos/music/videos by plugging your phone on another device (like a pc or a tablet).

Once you are sure that you backed up everything you could, hard reset the phone:

http://www.windowsphone.com/en-GB/how-to/wp8/settings-and-personalization/reset-my-phone

Then, set it up with the right Microsoft account, and your problems should be solved.

 

Kind regards

Manon B.
