What is the make and specific model of your computer?

If you have a laptop, you probably have built-in Bluetooth capability.

If you have a desktop/tower, you probably don't have any Bluetooth capability.  You should buy a USB Bluetooth "dongle."  For example only, see:  https://www.newegg.com/Bluetooth-Adapters/SubCategory/ID-295 

Can you pair these headphones with a smartphone?

Even if you have a Bluetooth adapter (either a USB add-on or built in), you may need to update its driver.

Open Device Manager (click Start, type Device Manager into the Search box, press Enter)

If there is a category for Bluetooth, double-click it to expand the category.

Select the "Advanced" tab

What is the "LMP" version?

  • LMP 0.x - Bluetooth 1.0b
  • LMP 1.x - Bluetooth 1.1
  • LMP 2.x - Bluetooth 1.2
  • LMP 3.x - Bluetooth 2.0 + EDR
  • LMP 4.x - Bluetooth 2.1 + EDR
  • LMP 5.x - Bluetooth 3.0 + HS
  • LMP 6.x - Bluetooth 4.0
  • LMP 7.x - Bluetooth 4.1
  • LMP ​8.x - ​Bluetooth 4.2
  • LMP 9.x - Bluetooth 5.0
As far as I can determine, JBL Synchros E50BT headphones use Bluetooth 3.0.  If you already have Bluetooth capability, the LMP version needs to be 5 or higher.  If you have an LMP version lower than that, an updated driver may fix this.  If your Bluetooth adapter is built into your computer, check the support section of your computer manufacturer's website for an updated Bluetooth driver.
