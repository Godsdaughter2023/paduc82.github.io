***Moved from: Windows 10/Windows settings***.

ossiemac say:

Following solution requires Conditional Formatting plus a small amount of VBA coding (Macro). The VBA code forces an update of the screen each time a cell or range is selected.

If multiple cells are selected (ie, a range) then only the first selected cell is highlighted with the selected format but remaining selection will still be darkened as per normal selection.

Not sure of your expertise so I will provide step by step guidelines.

  • Select entire worksheet (click top left of worksheet where the column headers and row labels join)
  • Select Home ribbon
  • Select Conditional Formatting -> New rule
  • Select the option "Use a formula to determine which cells to format"
  • Copy the following formula and paste into the blank field in the Dialog under "Format values where this formula is true"

   =AND(CELL("row")=ROW(),CELL("col")=COLUMN())

  • Click Format button
  • Select the required format tab (top of dialog). Suggest Fill tab.
  • Select the required format
  • Click OK and OK again to close the dialog.

  • Open the VBA editor (Alt and F11)
  • Double click "ThisWorkbook" in Project Explorer (left column).  (f Project Explorer is not visible then select Menu item View -> Project Explorer to make it visible).
  • Copy the following code from Private Sub to End Sub and paste into the VBA editor (Large white space right side of window).

Private Sub Workbook_SheetSelectionChange(ByVal Sh As Object, ByVal Target As Range)
    Application.ScreenUpdating = True
End Sub

  • Close the VBA editor (Cross very top right of the VBA editor window)
  • Use "Save as" to save the workbook as Excel Macro Enabled workbook (.xlsm).
  • Ensure macros are enabled as follows.
  • If Developer ribbon not displaying then right click anywhere in one of the ribbons and select "Customize the ribbon".
  • On the right side of the dialog check the box against "Developer" and click OK.
  • Select Developer ribbon.
  • Select Macro Security (In the Code block at left end of Developer ribbon)
  • The dialog that displays should default to Macros in left column.
  • Select required security option button. (Option to "Disable all macros with notification" should be OK.)
  • Click OK to close the dialog.

Feel free to get back to me if any problems with this solution.

peter atherton say:

When you tab accross and reach a column with formulas then Excel will copy down the formula above. You need a macro to unrprotect the sheet anad then protect it again when the data is entered.

 

Copy this macro, between the lines '========, into the worksheet module (right-cick the sheet tab and choose View Code).

 

'==========================
Private Sub Worksheet_Calculate()
  Call LockCells
End Sub

'===========================

You are now in the VB Editor. Choose Insert, Module and paste the next code

'========================

Sub LockCells()
  Dim c As Range
  Dim addr As String
 
  addr = ActiveCell.Address
  Application.ScreenUpdating = False
  'assumes there is no password protect
  ActiveSheet.Unprotect
  With Cells.SpecialCells(xlCellTypeFormulas, 23)
    .Locked = True
    .FormulaHidden = False
  End With
  ActiveSheet.Protect
End Sub

 

'==================

If you have password protected your sheet then you will have to add the password to the code, but I wouldn't bother.
