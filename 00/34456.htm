gokul t say:

Hi KandiK68,

 

Firstly try this: Click Menu > Current view and select “Show all messages”.

 

If the above step does not show the messages, rebuild the Windows Mail folder and check if that fixes the issue.

 

Follow these steps:

 

Step 1: Rename the Windows Mail folder and create a new folder

 

a.    Exit Windows Mail.

b.    Locate the  Windows Mail folder. See "Step 1: Locate the message store" from Method 1 for information about finding this folder. It is typically located in:

 

hard disk\Users\user\AppData\Local\Microsoft\

 

c.    Right-click the  Windows Mail folder, and then click Rename.

d.    Add old to the end of the folder name so that the folder is now named  Windows Mailold.

 

Step 2: Create a new Windows Mail folder and copy the contents from the previous folder

 

a.    Start Windows Mail so that it can automatically create the new  Windows Mail folder, and then exit Windows Mail.

b.    Located the new  Windows Mail folder to confirm it is created. See "Step 1: Locate the message store" from Method 1 for information about finding this folder. It is typically located in:

hard disk\Users\user\AppData\Local\Microsoft\

 

Note If the new Windows Mail folder is compressed, you have to uncompress this folder. To uncompress it, right-click the Windows Mail folder, and then select Extract.

 

c.    Open the  Windows Mailold folder.

d.    Open the  Local Folder and copy the account{xxxxxxxxxxxxx}.oeaccount file. Notice thatxxxxxxxxxxxxx represents numbers or letters.

e.    Paste the file into the  Local Folder in the new Windows Mail folder (that was created in step 1 of this procedure).

f.     If the following folders are present in the  Windows Mailold folder, copy each folder and paste it into the new Windows Mail folder.

o   Stationery

o   Microsoft Community

g.    Close all folders.

 

Step 3: Import messages in Windows Mail

 

a.    Start Windows Mail.

b.    Import the messages from the previous Windows Mail folder to the new folder. To import the messages, follow these steps.

 

i.     Click File, point to Import, and then click Message.

ii.    Click Microsoft Windows Mail 7 in Select Program, and then click Next.

iii.   Click Browse, select the location of Windows Mailold, and then click Next.

iv.   Click All folders in Select the folder, and then click Next.

v.    Click Complete, and then the messages will be imported to Imported Folder.

 

c.    Move the messages that were imported to the  Imported Folder to the appropriate folder under theLocal Folder. For example, move messages located in the  Inbox folder under the Imported Folder to the Inbox folder under theLocal Folder.

d.    Check whether the stuck messages are removed. If not, try to send or delete the messages.

This should fix the issue.
