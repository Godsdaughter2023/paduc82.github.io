A couple lines of VBA might make this a great deal simpler, IF using VBA is in the cards.

Sub MoveLeft(oSh As Shape)

Optional paragraphs Word 2016 - Microsoft Community

victoriay say:

Optional paragraphs Word 2016

I'm creating a template for a document that has several optional paragraphs.  

Is there any way to have a paragraph optional/drop down added to a particular section to be selected - so that if we do need the paragraph it is added and if we do not need it it is deleted?

Rather than having to search quick parts for the paragraph and risk adding the wrong paragraph into the wrong section?

Any help is greatly appreciated.

Thanks,

Victoria

jay freedman say:

Assuming everyone who will use the template has Word 2007 or later, you can -- with some careful planning -- create this without any programming. The feature that makes it possible is the building block gallery content control, which you'll find in the Controls section of the Developer tab. (If the Developer tab isn't visible in your copy of Word, go to File > Options > Customize Ribbon and check the box next to Developer in the right-hand list.)

I'll get to the use of these controls eventually, but first you have to set up the building blocks that they'll use.

  • Open a document based on the template, and type or paste in one of the optional paragraphs.
  • Select the paragraph and press Alt+F3 to open the Create New Building Block dialog box.
    • Fill in a short descriptive name for the building block.
    • In the Gallery dropdown, scroll down and select Custom 1.
    • In the Category dropdown, click Create New Category. In the little box that appears, type a short descriptive name for the section in which this paragraph will be valid.
    • In the Save In dropdown, select the name of your template. That will ensure that the building block will be saved in that template, so other users will be able to see it (unlike Normal.dotm and Building Blocks.dotx, which are only on your computer).
    • Optionally, type a description in the Description box.
    • Click OK.

  • Repeat this for the other optional paragraphs. The only difference in the Create dialog for paragraphs that belong to the same section should be the name of each paragraph. For each additional section, create a new Category.
  • At this point, close Word. You can save the document or not, as you please, but be sure to click Save when prompted to save changes to the template.

Now you can add the content controls to the template.

  • Open the template for editing.
  • At the location where each optional paragraph may appear, click the Building Block Gallery Content Control button on the Developer tab.
  • Click the Properties button to the right of the content control buttons.
    • At the bottom of the Properties dialog, set the Gallery dropdown to Custom 1 and set the Category dropdown to the name you created for this section's category.
    • At the top of the dialog, in the Title box, type the Category name or something similar but more descriptive.
    • Click OK.
  • Repeat this for the other places where optional paragraphs may appear. You can include more than one content control in each section, in which case they'll have the same settings.

When a user clicks in a content control and clicks the down arrow in the control's name tag, a gallery opens with the list of optional paragraphs that are valid in that section. The controls in each section will present only the paragraphs for that section, because the Category settings in the building blocks match the Category setting in the controls' Properties dialog.

At any location where no optional paragraph is needed, the user can select the content control and the paragraph mark following it, and press Delete to remove them.
