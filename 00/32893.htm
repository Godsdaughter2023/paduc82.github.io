Windows 10 Update error code 0x80240439 may be due to corrupt cache or concerned service malfunctioning. You can perform these methods to address this problem:

Method 1: Run Windows Update Troubleshooter

You can run the automated troubleshooter for Windows Update as it will identify and resolve issues it detects.

As soon as you're done with the troubleshooter, restart your PC then run the update that triggered 0x80240439 error to check if the issue is resolved.

If the issue persists, proceed to Method 2.

Method 2: Reset Windows Update Cache

  1. Press Windows key + X.
  2. Click on Command Prompt (Admin).
  3. If you see the UAC pop up, click on Yes.
  4. Type net stop wuauserv then hit Enter. It may take a while since this command stops the Windows Update service.
  5. Type net stop appidsvc then hit Enter.
  6. Type net stop cryptsvc then hit Enter.
  7. Press Windows key + R.
  8. Copy C:\Windows\SoftwareDistribution\Download and paste it on the Run diaglog box then hit Enter.
  9. Select all the items available in the directory and press Delete.
  10. Go to the Command Prompt (Admin) window.
  11. Type net start wuauserv then hit Enter.
  12. Type net start appidsvc then hit Enter.
  13. Type net start cryptsvc then hit Enter.
  14. Restart your PC.

After rebooting, run the update that triggered 0x80240439 error to check if the issue is resolved.
