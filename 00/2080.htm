ben m. schorr -bschorr say:

Sure.  Create your master notebook in OneNote. Set it up just the way you like. Then right-click it and choose "Close This Notebook". 

Then whenever you want to create a new notebook from your template, go to Windows Explorer (or My Computer, if you prefer) and navigate to your My Documents folder and there should be a OneNote Notebooks folder.

In there you should see a folder with the same name as your template notebook.  Right-click it and choose "Copy".  Then Paste that notebook into the same folder and give it a new name for your new notebook.

That done go into OneNote click "File | Open" and browse to your OneNote Notebooks folder.  Open the new notebook you just made and it should open just fine in OneNote.

Rinse, Repeat for add'l notebooks.

If you ever want to edit your template notebook just open it the same way, make your changes, close it again.

tisky zheng msft say:

Hi TheTechWriter,

 

As you have known, the page title is your file name. To change the page title without changing the file name. You can embed the custom code to the page.

 

follow the steps below:

1. Open your page.

2. Click EDIT to edit the page.

3. Click EmbedCode under INSERT tab.

4. Insert the code below then click Insert.

 

<script type="text/javascript">

document.getElementById("DeltaPlaceHolderPageTitleInTitleArea").innerHTML="New Page Title";

</script>

 

If you want to hide the page title, just replace “New Page Title” with blanks.

5. Click save to save the changes.
