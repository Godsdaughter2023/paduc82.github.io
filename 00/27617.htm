Use ctrl-F9 (on Windows Word) to insert another pair of field code braces surrounding your NOTEREF field, so you have

theprobableadversary say: That's very helpful, thank you!

a. user say:

I would consider trying the following for each range. It might need to be adapted if you have over 1000 footnotes, and/or you want "23-6" and "123-6" rather than 23-26 and 123-26. Also, I haven't looked at how to deal with any loss of function of the \r switch ("formatted") or \h switch ("hyperlink"). The \h switch used in SEQ fields means "hidden".


Let's suppose the bookmark for the first footnote is _Ref1 and the bookmark for the second is _Ref2 

Then you use

{ SEQ ra \r{ NOTEREF _Ref1 } \h }{ SEQ rb \r{ NOTEREF _Ref2 } \h }{ SEQ rc \r{ =abs(int({ SEQ rb \c }/100)- int({ SEQ ra \c }/100)) } \h }{ ={ SEQ ra \c }-{ SEQ rb \c } \#"'{ SEQ rb \c }-{ SEQ rc \c \#"'{ SEQ ra \c }';;'{ =mod({ SEQ ra \c },100) \#00 }'"}';'{ SEQ ra \c }-{ SEQ rc \c \#"'{ SEQ rb \c }';;'{ =mod({ SEQ rb \c },100) \#00 }'"}';'{ SEQ ra \c }'" }

Apart from the first two { SEQ } fields, all the rest is the same and could go in an autotext/building block. Then the idea is that if
_Ref1 is 195 and _Ref2 is 197, you get 195-97
_Ref1 is 195 and _Ref2 is 207, you get 195-207

_Ref1 is 197 and _Ref2 is 195, you get 195-97 (not 197-95)

_Ref1 is 207 and _Ref2 is 195, you still get 195-207

_Ref1 is 195 and _Ref2 is 195, you get 195

rick rothstein, mvp - excel say:
Right now my results are in decimal form. I'd like the results to be displayed in x'y" (if possible) or any other ft-in formatting, i.e. x'-y", xft yin. Is this possible?

By "decimal form", do you mean feet and fractions of a foot (for example, 12.34 feet)? If so, consider this event code which keeps your decimal value (so you can calculate with it), but changes the cell display to the format you want (note - change both of the A1:A65 range references to whatever your actual range is for your measurements)...

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim Cell As Range, Pattern As String
  Application.Volatile
  If Not Intersect(Target, Range("A1:A65")) Is Nothing Then
    For Each Cell In Intersect(Target, Range("A1:A65"))
      If IsNumeric(Cell.Value) And Len(Cell.Value) > 0 Then
        Cell.NumberFormat = """" & Int(Cell.Value) & "'-" & 12 * (Cell.Value - Int(Cell.Value)) & "''"""
      End If
    Next
  End If
End Sub

One thing you didn't tell us is if you want the entire decimal inches or if you wanted it rounded to a certain number of decimal places. The above code does not round it; the code below rounds it to 1 decimal place...

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim Cell As Range, Pattern As String
  Application.Volatile 
  If Not Intersect(Target, Range("A1:A65")) Is Nothing Then
    For Each Cell In Intersect(Target, Range("A1:A65"))
      If IsNumeric(Cell.Value) And Len(Cell.Value) > 0 Then
        Cell.NumberFormat = """" & Int(Cell.Value) & "'-" & Round(12 * _
                            (Cell.Value - Int(Cell.Value)), 1) & "''"""
      End If
    Next
  End If
End Sub

Note: That is not a quote mark for the inches, it is two apostrophes as I couldn't get the quote mark to display in that location.

EDIT NOTE: That Application.Volatile statement is in there in case your measurements result from formula calculations. If your measurements are user-entered, then you can comment out that line (doing so will make your worksheet more efficient).

 

HOW TO INSTALL Event Code
---------------------------------------------------
If you are new to event code procedures, they are easy to install. To install it, right-click the name tab at the bottom of the worksheet that is to have the functionality to be provided by the event code and select "View Code" from the popup menu that appears. This will open up the code window for that worksheet. Copy/Paste the event code into that code window. That's it... the code will now operate automatically when its particular event procedure is raised by an action you take on the worksheet itself.

# script

# Establish a remote session to Exchange Online

$creds = Get-Credential

$Session = New-PSSession -ConfigurationName Microsoft.Exchange –ConnectionUri ` https://outlook.office365.com/powershell-liveid/ -Credential $creds -Authentication Basic -AllowRedirection
