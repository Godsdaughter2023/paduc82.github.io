Adam!

Depending on your hardware, like CPU speed and cores, RAM memory, internet speed, and running apps; this sort of update will take considerable time. I notice by the questions you are asking that you are anxious and that can spell disaster. Be patient when the OS performs an update of this size and complexity. You may want to save then close your apps. and let the update download and install on it's own terms if you want a successful update. But never interrupt or interfere with the update in any manner if you want success, patience is a virtue here.

Using Task Manager, you will notice high disk usage as Windows prepares your disk for certain updates (notice the reads and writes). Reads and writes can take extended time depending on the type of drive and configuration you are working with so don't be concerned with disk usage at this point as it is normal. Windows will need to get things in order before it downloads this update and it will seem like it is taking forever but do not fear when Windows stops at 95% as this is most likely a configuration pause but it will continue in time.

Not being able to use your system is most likely system level interrupts being called by Windows. This is why updates are best performed after hours or when you are not using your PC. Give it plenty of time; be nice to your computer if you want it to be nice to you!

So, go to Start>Settings>Update & Security>and under update status click on Update History to see if the update shows up as successfully installed. If it shows up successfully installed, you should be OK as long as you didn't interfere with the update. If it is listed and states it Failed to Install, click on the Uninstall Updates option and after it loads, find the right update and uninstall it then go back and Check for Updates and re-install it. However, if it doesn't show up, save and close your apps. and perform the Check for Update function and be patient as this will take time, lots of time.

Do not let a Failed to Install listing scare you. This often happens when Windows tries to update a feature that requires a prerequisite update and can be resolved by ordering the Check for Updates option a day or two later.

Good luck Adam!
