If error on date formual - Microsoft Community

chrisb421 say:

If error on date formual

This formula returns a date of 12/31/1900 and i'd like it to be blank

=DATE(YEAR(A4) + 1, MONTH(A4), DAY(A4))

i tried if error several different ways, but cannot get it right.  could someone help?  Thank you!

joeu2004 say:

=IF(A4="", "", DATE(YEAR(A4) + 1, MONTH(A4), DAY(A4))

or

=IF(N(A4)=0, "", DATE(YEAR(A4) + 1, MONTH(A4), DAY(A4))

The latter allows for A4 to contain zero as well as any text, not just the null string ("") or nothing at all.

Aside....  If A4 contains the date Feb 29 in some leap year, your DATE formula will return Mar 1 of the following year.  If you would prefer Feb 28 (I would), use EDATE(A4,12).

Half the problem is my fault but only to a small extent.  The other half of the problem is Microsoft not taking into account or changing things up in the USER PROFILE section where the problem resides.


I like to partition my hard drives in half or keep a secondary drive which I always MOVE the default folders to as protection from losing my data.  What do I mean, once I install a computer from scratch I always move My Documents, My Music, My Videos, Downloads, Desktop, Contacts, etc. to my D drive or Secondary Drives so that my data is protected from day one.

Step 1:  Run Command Prompt as Administrator and type the following command in:


mkdir %userprofile%\AppData\Local\Temp\Low


Step 2:  Run Command Prompt as Administrator and type the following command in:


icacls %userprofile%\AppData\Local\Temp\Low /setintegritylevel low

Hi Akhilesh,

Use the below mentioned formula to get the desired output.. Note that it is an array formula. You have press Ctrl+Shift+Enter after typing the formula.. Here in the below formula, B5:B11 is the Leader's Range, C5:C11 is the Subordinate Range. D4 is the Leader criteria...

=SUM(--(FREQUENCY(IF(B5:B11=D4,MATCH(C5:C11,C5:C11,0)),ROW(B5:B11)-ROW(B5)+1)>0))

If you want total summary, you can do the same by using pivot table. In pivot table drag Leader into Rows and Subordinate into values in the field list.. You will get the desired output..
