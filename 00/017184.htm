john mcghie say:

Just a note:

Do not use the .jpg format for signatures (or anything else where you want a crisp result, such as line drawings).

In the case of a "signature" you need only one colour, so use .gif for the smallest result.

If you need more than 256 colours (a line drawing with a shaded fill...) use .png.

.jpg should be used only for photos.

The jpg format throws out detail to preserve colour when compressing the file.  .gif and .png throw out colours to preserve detail.  .jpg can display unlimited colours, but it will remove resolution to do so.  .gif can store only 256 colours: it will use "dithering" to give the impression of more.  .png can store millions of colours, but the file size will be ten times larger than .gif.

Do signatures in .gif, especially if you are planning on printing or emailing them.

Hope this helps
