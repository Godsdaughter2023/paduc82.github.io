For a macro to clean up text copied from emails, web pages, pdfs and the like, see:

Although Office 2008 doesn't support macros, the following series of wildcard Find/Replace operations, as used in the link, will do the same when invoked via the GUI:

'Eliminate spaces & tabs before paragraph breaks.

Find = [ ^s^t]{1,}^13

Replace = ^p

'Replace single paragraph breaks & line breaks with a space

Find = ([!^13^l])([^13^l])([!^13^l])

Replace = \1 \3

'Replace all double spaces with single spaces

Find = [^s ]{2,}

Replace = ^32

'Delete hypens in hyphenated text formerly split across lines

Find = ([a-z])-[^s ]{1,}([a-z])

Replace = \1\2

'Limit paragraph breaks & line breaks to one paragraph break per 'real' paragraph

Find = [^13^l]{1,}

Replace = ^p

paul andrus say:

Eve,

SquirrelMail is not a mail server like Gmail. Think of it like Outlook Web Access. It is the web-based user interface to a mail server that your organization sets up.

You need to set Outlook up to connect to your organization's mail server. You will probably need to know whether the mail is POP3 or IMAP, and the incoming and outgoing server addresses. Then you go into Outlook, add an account, manually configure, and enter the information there. 

You will be setting up Outlook to go to your company's mail server directly, not through SquirrelMail.
