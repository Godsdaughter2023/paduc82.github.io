Thank you for posting in Microsoft Community, we appreciate your interest in Windows 10.  

  

I understand the inconvenience that you are facing on Windows 10, we will help you with this issue.

I suggest you to check if Internet Connection Sharing services is started and startup type set to automatic.

To check Internet Connection Sharing services is started please follow the steps below:

  1. Press Windows + R key.

  2. Type services.msc

  3. A new window will pop up.

  4. Find Internet Connection Sharing.

  5. Right click and go to properties.

  6. Change Startup type to automatic.

  7. Under Service status click start if it shows Stopped.
