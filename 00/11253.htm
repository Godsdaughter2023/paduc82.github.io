How to use a combo search to return matching multiple entries in - Microsoft Community

martin276 say:

How to use a combo search to return matching multiple entries in Access?

Hi - How can I create a form that uses a combo search box to return a list of all of my table entries that match the value selected in the combo search box?

For example; my table contains a list of all visits I have made. If I select London in the combo search box, I would like to see a list of all London visits.

Thanks for your help

 

ken sheridan say: Set the Filter property of the form in the unbound combo box's AfterUpdate event procedure, and set the FilterOn property to True, e.g. if the value is a string expression, the name of the city for instance, in a VisitLocation column:

   With Me.ActiveControl
       If IsNull(.Value) Then
           Me.FilterOn = False
        Else
            Me.Filter = "VisitLocation  = """ & .Value & """"
            Me.FilterOn = True
        End If
    End With
   
However, geographical location names such as city names can legitimately be duplicated (Paris, France and Paris, Texas for instance), so the column should really be a numeric value referencing the numeric primary key of a table of locations in which the location name is a non-key column which allows duplicates, and which contains also contains another column which differentiates between locations of the same name, e.g. by county in the UK.  

You'll find examples of the use of navigational combo boxes which search a table on the basis of a numeric foreign key in FindRecord.zip in my public databases folder at:

https://onedrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

Note that if you are using an earlier version of Access you might find that the colour of some form objects such as buttons shows incorrectly and you will need to  amend the form design accordingly.  

If you have difficulty opening the link, copy the link (NB, not the link location) and paste it into your browser's address bar.

In this little demo file the option to filter by a foreign key opens a form in which an unbound combo box's BoundColumn property references a hidden numeric column, but lists the non-key text values by virtue of the control's ColumnWidths property being 0cm.  The code in its AfterUpdate event procedure is:

    With Me.ActiveControl
        If Nz(.Value, 0) = 0 Then
            Me.FilterOn = False
        Else
            Me.Filter = "ContactID = " & .Value
            Me.FilterOn = True
        End If
    End With

Note how this combo box's list has 'All Contacts' as its first row, selecting which turns off the filter, rather than by the value of the control being Null, as in my first example above.  This is achieved by setting the control's RowSource property to:

SELECT ContactID, Firstname & " " & LastName, LastName, FirstName
FROM Contacts
UNION
SELECT 0, "All Contacts", NULL,NULL
FROM Contacts
ORDER BY LastName, FirstName;

Template for Publisher - Microsoft Community

joshhobart say:

Template for Publisher

I am trying to design a template for Publisher, all I really need is a page cut into four equal quadrants, and the to be able and design in one quarter of the page, then easily copy the entire quadrant into the other three quadrants, I haven't been able to find a way to do this, I am sure this might sound elementary to some designers, but I cant figure it out.
davidf2 say:

I will assume you are using Publisher 2010 or 2013.

Create the blank page of the size you need.

Go to the View tab > and make sure at least Boundaries, Guides and Rulers are all checked.

Switch to Page Design tab > Layout group > click the drop down arrow below Guides > select Add Horizontal Ruler Guide and go back and add Add Vertical Ruler Guide. This will give you the 4 equal page quadrants.

I would also Layout group > check Align to Guides and Objects which will help you align your pasted elements in the other quadrants.

Now you can complete your design in one of the quadrants and copy and paste to the other 3 quadrants. I would first select all the elements of your design by holding down your Ctrl key while you click on each element in the quadrant to select them all. Then go to Home tab > Arrange group and click Group (or Ctrl+Shift+G) all the elements together. Now you can copy that group and paste into the other 3 quadrants and drag or use the arrow keys to nudge the group into place.

If your original design included an element that touched one of the ruler guides you can easily orient  the pasted group by dragging the group until it 'snaps to' the ruler guide in the other quadrants because you previously checked the option to align to guides and objects.

If you want you could add a text box, a rectangle shape or even a line shape to an empty corner of your original design, group it with the other elements and when you paste that group it will be easy to 'snap to' the corner of the other quadrants. Then you can go back upgroup, and delete that 'orientation' object or shape.

With all that said there is an easier way if all you want to do is to print mulitple copies of your design on a sheet of paper or to a PDF. Open a business card template and explore it as an example. You could create a custom size publication that equals a quarter page and print multiple copies of that publication to the full size sheet of paper.

Please post future Publisher questions in the Publisher forum. This is the Photodraw, frontpage, works forum. Take time to specify which version of Publisher you are using and as many details as possible which makes it easier to help you.
