graham mayor say:

Running the macro from Excel presents some issues with the code I posted. You would need something like the following which defines WordApp and the Document "invoice_out_file".

Using Late Binding to the Word object you cannot use Dialogs(wdDialogFileSaveAs) but must use its numeric equivalent wordApp.Dialogs(84).

The Documents Count will be irrelevant as you will have either created a new document or opened an existing one and it is that which is being saved. Run the following as a stand alone macro from Excel and you will see then how to add your process.

Dim wordApp As Object
Dim invoice_out_file As Object
    On Error Resume Next
    Set wordApp = GetObject(, "Word.Application")
    If Err Then
        Set wordApp = CreateObject("Word.Application")
    End If
    On Error GoTo 0
    Set invoice_out_file = wordApp.Documents.Add 'or open a document
  
    'do stuff with the document invoice_out_file e.g.
    invoice_out_file.Range.Text = "This is some text"
    With wordApp.Dialogs(84)
        If Not .Show = 0 Then
            invoice_out_file.Close 0
        End If
    End With
    Set wordApp = Nothing
    Set invoice_out_file = Nothing
