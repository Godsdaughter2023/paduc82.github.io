melowin mar say:

Hi George,

Error 0x800F081E is a Windows status code for CBS_E_NOT_APPLICABLE, which essentially means that an update requirement is missing or the files that are already installed are already of a higher version than the ones that are pending. In line with this, it seems that the error has to do with the Media feature pack that you may have installed on your system, or it comes pre-installed. To get this sorted, I suggest that you disable the Media pack, install Windows 10 Springs Creators Update, and then reinstall the Media pack again.

Here's how:

  1. To remove the Media feature pack, go to Control Panel >Programs.
  2. Click Turn Windows features on or off.
  3. Another windows will open. Wait for the list of features and look for Media features in the list. Uncheck the options, then click OK and restart your PC.
  4. After the restart, run the update again.

Note: If the installation completes successfully, return to Windows Features and re-enable Windows Media Player since you might need it in the future.

"First" - correct, when you make a system image using Windows Backup or another utility, just image the C:\ drive if you want to keep it as small as possible but do make & retain at least one image that includes all the partitions that Windows Backup offers to include.  Please note that you will also need to make a system repair disk [this can be used to boot the system* so you can restore the system image - so you need it in case the system is so corrupted that Windows cannot boot on its own].  Both of these functions are on the left-hand side of the window when you start Windows Backup.
