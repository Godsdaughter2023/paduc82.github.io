Two questions - two answers

First - how to copy items from one folder to another

1) Make sure that the <view> for whatever it is that you want to copy is set to a "list" (open your folder --> goto <Change View> and select a "list" option --> then

2) Hightlight all items in the folder you want to copy from using CTRL+A --> right-click --> select <Move> --> at bottom of that menu, you have 2 options (a) Other Folder and (b) Copy to folder.  I suggest using (b) since that will leave a copy in the original folder in the event something goes wrong. Option (a) moves to destination and then deletes from origin.

Do that for whatever folders you want to move over to Outlook.com

Second - removing folders

Will be gone when you delete the account but no option in Outlook to prevent applicable folders from appearing in the corresponding lists

<Something to remember>

If you are adding a contact (for sake of example) and no contact folder has been selected, the item will always be added to the <default> contact folder (same for any other type of item). If you FIRST select the contact folder and then add a new item, it will be added to the folder you just opened.

a. user say:

Hello Devereux,

I have included two simple commands for your task... I am working on the script to combine both and will give you the information in single file, will post that soon..

To retrieve the users in Florida

Get-Msoluser -MaxResults 10000 | Where-Object {$_.State -eq 'Florida'} | Format-Table DisplayName,UserPrincipalName,City,State | Export-Csv -NoTypeInformation "C:\Floridausers.csv"

Mailbox Size INformaton

Get-Mailbox -ResultSize unlimited | Get-MailboxStatistics | Select-Object DisplayName,TotalItemSize,ItemCount,MailboxTypeDetail | Export-csv -NoTypeInformation "C:MailboxSize.csv"
