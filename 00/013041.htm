File Size is bigger than what it contains - Microsoft Community

aojiku01 say:

File Size is bigger than what it contains

I have a file that has one empty tab but contains macro, but the file size 17,356KB does make any sense. Why is the file size so huge but there file is practically empty?
zorvek say:
Occasionally a worksheet's used range can become larger than it needs to be and thus can result in the size of the workbook increasing unnecessarily. To remove unused rows and columns from a worksheet's used range, select the first unused row or column and press CTRL+SHIFT+DOWN for rows, CTRL+SHIFT+RIGHT for columns. Choose the menu command Edit->Delete and click OK. Repeat this process for all rows and columns in each worksheet suspected of having this problem.

An easy way to tell if the used range has been extended more than is needed is to use the scroll bars to scroll to the bottom or the right of the used area of a worksheet. If the scroll bar elevators (not the scroll bar arrows) allow the window to be scrolled to a point where no used cells are visible then the used range is probably too large. The scroll bar elevators (the boxes in the scroll bars) always reflect the size of the current used range on that worksheet as Excel has defined it.

jim gordon mvp say:

The best way is to have a copy of the font and/or its installer with you so that you can install it onto the computer you plan to use. Be sure to restart the computer after installing the font, before you run your presentation.


Installation steps vary by font and operating system. Font Book application comes with Mac OS. You can use File > Add Fonts in Font Book to add fonts. 


Be sure to ask permission before installing a font onto someone else's computer. Some fonts have restrictions as to whether or not you are allowed to install the font without purchasing additional copies of the font.

jim gordon mvp say:

Well, it's not the only way. 


Generally, you can use the fonts on the Font menu in the Font Collections > Windows Office Compatible submenu. It's a little risky, as those fonts are not guaranteed to be on all computers. People tend to customize the font collections. IT departments tend to use the bare minimum required fonts. 


However, if the font you chose isn't installed, a substitute font will be chosen by PowerPoint, and it might not look all that much like your nice font.
