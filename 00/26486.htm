Any assistance is appreciated. 

Thanks!

yangzheng_msft6300 say:

Hi Pw1981,

According to your description, do you mean that you want to restrict the permissions of non admin users to ‘view only’ another user’s calendar in a group? If so, an Office 365 admin can achieve this using PowerShell. Here are the detailed steps to do this.
1. Connect to Exchange Online with PowerShell.
2. Run the following command to set the default calendar permission to view only. Then users who haven’t been assigned other permissions can only view calendar items but can’t edit or create entries.
Get-Mailbox | foreach{ Set-MailboxFolderPermission $($_.UserPrincipalName+":\calendar") -User default -AccessRights  AvailabilityOnly }
3. Run the following command to assign the Editor permission to certain admins. Admins with Editor permission can view, create and edit entries of other calendars in the group.
Get-Mailbox | foreach{ Add-MailboxFolderPermission $($_.UserPrincipalName+":\calendar") -User user@domaincom -AccessRights  Editor }

If you meet any error during the steps above, please post back with the detailed error message.

When in Windows 7 you select "Default input language" in the "Text Services and Input Languages" dialog (Google/Bing how to get to it), you normally get choices like:
