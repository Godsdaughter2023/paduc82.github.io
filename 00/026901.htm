john halbach say:

This module evokes the Get External Data - Excel Spreadsheet wizard.

Option Compare Database

Sub GetExternalData()
DoCmd.RunCommand acCmdImportAttachExcel
End Sub

On an MSA switchboard run this VBA and the wizard will appear.
