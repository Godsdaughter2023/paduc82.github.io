amandamp say: Is there any other way to fix this problem?

wunders say:

I'll try to explain succinctly.  You are having problems because a power interruption occurred during the instant that data was being written to your System Registry file.  This caused that file to become corrupt and/or unreadable.  The System Registry is a file that contains every preference and setting needed to run Windows on your particular system.  It is a critical file for Windows to run and it is also very personalized to your particular computer and how it is currently set up.  If it goes bad, it must be either repaired or replaced by a fairly recent backup copy.  You simply can not get the file from someone else and replace it.

 

The only way to "fix" it is to run "chkdsk" (as you did).  Chkdsk does its best to put the disk back in order but in some instances like a power failure, sectors can be effectively erased or overwritten with garbage to the point where they will never be recoverable.

 

The second way to fix it is to restore the file from a backup.  Every time a Restore Point is created, it backs up this critical file as part of the process.  Restore points are normally created at the first boot-up of the day and whenever system updates or installations are performed.  Creation of Restore Points can be turned off and in your case, it looks like restore points for some reason have not been turned on.

 

The only other backup copy that you might have can be accessed by repeatedly tapping the F8 key as the system boots and, when the black-and-white screen comes up, select the "Last Successful Startup" (or words to that effect) option.  In theory, a backup copy is created at each boot which can be used should something like this happen.  My experience is that, for whatever reason, this never works -- so I rarely consider it -- but it might be worth a try in your case if you haven't already.

 

So unless you know where another backup of this file might be, the only option left is to backup what data files you can and re-install Windows.  There are no other options.

 

HTH,

   JW
