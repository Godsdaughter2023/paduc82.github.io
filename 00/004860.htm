lars-Åke aspelin say:

Not clear if you want to sort the four data sets "independent" of each other or if data should be moved "between" the data sets.

Assuming that no data should be moved between the data sets and that all data sets should be sorted by the first column, try this macro:

Sub sort_4_ranges()
  Dim ws As Worksheet
  Dim first_cell As Range
  Dim next_cell As Range
  Set ws = ActiveWorkbook.Worksheets("Sheet6")  'change name
  columns_to_sort = 10 'change to suit your data
 
  Set first_cell = ws.Range("E10") 'the first cell of the first data set to sort
  sorted_datasets = 0
  While sorted_datasets < 4
     Set next_cell = first_cell
     While next_cell.MergeArea.Count = 1
        Set next_cell = next_cell.Offset(1, 0)
     Wend
    
     'sorting code
     ws.Sort.SortFields.Clear
     ws.Sort.SortFields.Add Key:=Range(first_cell.Address), SortOn:=xlSortOnValues, Order:=xlAscending, DataOption:=xlSortTextAsNumbers
     With ws.Sort
        .SetRange Range(first_cell.Address & ":" & next_cell.Offset(-1, columns_to_sort).Address)
        .Header = xlNo
        .MatchCase = False
        .Orientation = xlTopToBottom
        .SortMethod = xlPinYin
        .Apply
     End With
    
     Set first_cell = next_cell.Offset(1, 0)   'change the 1 if the next data set it not immediately after the row with merged cells
     sorted_datasets = sorted_datasets + 1
  Wend
End Sub

If the data should be sorted by some other column than the first column, change

Key:=Range(first_cell.Address)
to
Key:=Range(first_cell.Offset(0,column_to_sort_by-1).Address)

Hope this helps / Lars-Åke

negative number and it subtracts twice the value in excel

jllatham say:

NO, it's not subtracting twice - you're being fooled by the DELTA or change in total values.

 

Try this (I'll put in commas to make visibility easier)

C1:  18,000,000

C2:       755,000

C3:       130,000

C4:       247,000

C5:     1,000,000

 

in C6 put =SUM(C1:C5)  and it will show you your 20,132,000 value

Now change the 247,000 entry to a negative entry and you should get 19,638,000

 

If you just add the 18,000,000 + 755,000 + 130,000 + 1,000,000 values, then you have your desired goal of 19,883,000  but the SUM() function is doing ((18,000,000 + 755,000 + 130,000 + 1,000,000) - 247,000) which is indeed 19,638,000

 

You might be happier with this formula:

=SUMIF(C1:C5,">0",C1:C5)

That will give you your anticipated 19,885,000.

 

Confusing isn't it?

 

ronrosenfeld say: That is how math works.

When you change the sign of a number in the series, you don't subtract that number from the previous total; rather you subtract that number for the total less that number whose sign you reversed.

Going back to your original example, and changing the sign of A5

A1: 100
A2: 100
A3: 100
A4: 100
A5: 100

100+100+100+100 = 400
400+100 = 500
SUM(A1:A5) = 500

A1: 100
A2: 100
A3: 100
A4: 100
A5: -100

100+100+100+100 = 400
400-100 = 300
SUM(A1:A5) = 300

In your second example, you want to show 20,132,000 less 247,000.
Therefore you need to subtract 247,000 from 20,132,000

If you merely reverse the sign of the 247,000, the other values only add up to 19,885,000.
