At the risk of compounding nonsense with more nonsense, perhaps the following gives you the control over calculation that you require (for reasons that I do not understand).


' setting EnableCalculation=True while in Manual mode
' will  not  cause recalculations on active worksheet

Application.Calculation = xlCalculationManual    ' in case it was Automatic
ActiveSheet.EnableCalculation = True                  ' in case it was False

' Section A

  Column("A:BZ").Calculate

  For Each ColumnText In Range("DY:IE").Columns

  Next ColumnText


' avoid recalculation on active worksheet when Automatic
' mode is re-enabled.

ActiveSheet.EnableCalculation = False
Application.Calculation = xlCalculationAutomatic    ' see footnote [1]


However, note that re-enabling Automatic mode might cause recalculation on other worksheets if changes were made there or if they depend on cells changed on active  worksheet while in Manual mode.


-----
[1] [EDIT] Arguably, instead of always setting xlCalculationAutomatic at the end, it might be better to set Application.Calculation to its original state, which might not be Automatic.  To wit:

Dim oldCalc
oldCalc = Application.Calculation
Application.Calculation = xlCalculationManual
'.... rest of the code ....
Application.Calculation = oldCalc

On the other hand, the arguable benefit of always setting xlCalculationAutomatic at the end is:  it is self-correcting.  That is, it corrects for the unintended situation where the workbook is in Manual mode initially.  It's a judgment call.
