An amazing number of problems is resolved by cold starting the X-Box One.  I had several issues just to get this game to run without sitting at an endless loading screen.  Turns out it wasn't the game it was the X-Box One.  By clearing the X-Box One cache that allows for quick starting in less then 30 seconds it fixed this problem as well as the later problem with only being able to Auto save.  A few days after the loading screen fix I had lost electricity while playing and had thought the save file damaged in some way.  I cold started the X-Box one and it fixed that problem as well.  To cold start: Make sure your X-Box One is turned off, remove the plug from the wall, wait 2 minutes, plug the X-Box power cord back into outlet, and turn on the X-Box One.  You should see the 3 little dots being highlights left to right.  It can take anywhere from 30 seconds to 3-4 minutes.  The second time I did this after I had lost power it took almost 4 minutes to start.
