a. user say:

Hi,

Windows 10 uses a page file when it has to. However, now when your computer’s memory starts to fill up, Windows 10 will start compressing old pages of memory so they take up less space, similar to what happens when you create a ZIP archive of multiple files. So, for example, if you have an app running that you haven’t touched in a while, rather than copying the information about that app to your hard drive, Windows will simply make it smaller, but keep it in memory.

To keep it low, I suggest you to close the unwanted applications and remove the unused apps.

Hope this information was useful.

If I am understanding your question correctly, as stated, your issue may not be exactly the same.

When you download the Media creation tool, the tool only is downloaded, not the installation files as well. The tool is then used to download the installation media files for either This PC (usually results in a slightly smaller file size) or for a Different PC (usually results in a somewhat larger file size). The choice then of creating an ISO file that can be used as such or burned to a dual layer DVD, or instead to create directly to a USB drive as a bootable USB drive must be made. Then, once all of the criteria has been chosen (including the choice of 32bit or 64bit) then the associated installation file package will be downloaded and created as per. It will then be able to be used from there as installation media.

Hello Emily,

To help you resolve the issue, follow the troubleshooting steps below:

Method 1: Run the Windows built-in troubleshooter

This troubleshooter automatically diagnose and repair common problems with audio playback.

1. Click on Start, search for Control Panel.

2. Search for Troubleshooting, then click on Troubleshooting.

3. On the left pane, click on View all. Select Playing Audio.

4. Click on Next and follow the on-screen instructions

Method 2: Update you sound drivers

1. Press Windows key + X, then select Device Manager.

2. Under Sound, video and game controllers, double-click on your sound card.

3. Go to Driver tab, and select Update Driver.

If that didn't work, try this:

I would assume the literal meaning of the message is correct. 

First, understand that this is the working memory of the computer that is being referred to, not the hard drive.

Two things are most likely:

1. one or more memory boards in the computer have failed

2. you need to increase the amount of memory in the computer.

Windows 7 has a built-in memory tester.  Simply type memory in the text box over the start globe.  It will bring up the link to the Windows memory tester program.  Run the test, possibly more than once.  If this does not identify a problem, the likelihood is that some program you are running needs more memory than you have installed.

Consider the possibility that your computer is infested with a mélange of start-up programs that is using a lot of memory.

If you need to add memory, you need to know something about what memory is installed in your computer and how many memory slots there are.  Be very certain you have the exact memory type.  An example would be DDR3 1333.  Also it is worth while to consider matching the CAS latency that is installed to what you are buying.  Few pay attention to this, but it can make a significant difference to performance. It is advisable to match memory components in pairs.

system restore only restores programs and system files.

 

system restore doesn't restore personal files, pics, etc....

 

however, you can upload and store 25 gigs of personal files for free on microsofts skydrive web site.

 

in any case, deleted files are still on the hard drive but are scheduled to be overwritten by windows if it needs to use the space that the deleted files are taking up.

 

so, as long as the windows hasn't overwritten them, there is a chance you can restore them with a program like, Recuva.

 

it will scan the disk and then produce a list of deleted but recoverable files.  

 

these files will have the first letter of the original filename replaced with a question mark.

 

you will simply need to replace the question mark with a valid character to re-instate the file(s)
