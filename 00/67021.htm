burrwalnut say:

It’s normal for Windows 7 to show, (a) all the networks you have ever connected to and (b) networks within the wireless range of your router.

They are displayed in the ‘Task bar’ so that you can connect to, say, a free hotspot when you’re in a different area. The fact that they appear will not cause any congestion on your network.

al jarvi say:
Windows 7 only shows the wireless connections I would like in Control Panel > Network and Sharing Center > Manage Wireless Netoworks however in the task bar Windows is still showing all my neighbors SSID's. How can I force Windows to simply ignore and no longer list these connections? The reason for removing them is because they are creating wireless congestion when connected to my own connection.
You can perform a blanket netsh denyall action then add your private wireless network back to the allow list if you want to block access to all neighboring wireless networks. Run these commands as an administrator.

netsh wlan add filter permission=denyall networktype=infrastructure
netsh wlan set blockednetworks display=hide
netsh wlan add filter permission=allow ssid=Your_SSID networktype=infrastructure

All nearby wireless networks will be blocked and hidden while you will still be able to connect to your private network, ie. the only one listed in the available networks list.

Note that I would only do that if your using a wireless desktop. If your using a wireless laptop then you would not be able to see or connect to any wireless networks at remote locations so this procedure is not recommended.

Additionally you can also block access to individual wireless networks if desired. See this for help with that.

http://theillustratednetwork.mvps.org/Netsh/Block_wireless_networks.html

If you use a unique home wireless network SSID and configure your wireless connection to automatically connect to your unique SSID and not to other preferred networks then you really should not have any issues even though multiple networks are nearby as noted by BurrWalnut.
