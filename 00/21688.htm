gary's student say:

This assumes the sheets are named Sheet1 and Sheet2.  Include the following event macro in the worksheet code area of Sheet1:

 

 

Private Sub Worksheet_Change(ByVal Target As Range)
Dim ady As String
ady = Target.Address
Sheets("Sheet2").Activate
Sheets("Sheet2").Range(ady).Select
End Sub

 

  Because it is worksheet code, it is very easy to install and automatic to use:   1. right-click the tab name near the bottom of the Excel window 2. select View Code - this brings up a VBE window 3. paste the stuff in and close the VBE window   If you have any concerns, first try it on a trial worksheet.   If you save the workbook, the macro will be saved with it.     To remove the macro:   1. bring up the VBE windows as above 2. clear the code out 3. close the VBE window   To learn more about macros in general, see:   http://www.mvps.org/dmcritchie/excel/getstarted.htm   To learn more about Event Macros (worksheet code), see:   http://www.mvps.org/dmcritchie/excel/event.htm

guessed say:

Each table must have the 'banded rows' setting turned on. You could resolve this by either turning this setting off for each table OR changing the table style definition so that that particular style doesn't have a difference between odd and even row formatting.

A macro to turn off the banding setting on every table and apply your preferred table style would look like this...

Dim objTable As Table
For Each objTable In ActiveDocument.Tables
   objTable.Style = "Table Grid"
   objTable.ApplyStyleRowBands = False   
Next objTable
