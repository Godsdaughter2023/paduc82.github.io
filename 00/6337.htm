stefan blom say:

What has happened is that the hidden bookmark which Word inserts when you create a cross-reference has expanded to include a manual page break. You could recreate the cross-reference or replace the manual break with "Page break before" formatting (look in the Paragraph dialog box).

To avoid the problem in the future, never add content (text, a blank paragraph/Enter) at the beginning of a cross-referenced paragraph. Instead, add content at the end of the preceding paragraph.

Keep Windows Defender or change to Windows Essentials?

Windows Defender was already preinstalled on my laptop by the shop I purchased my laptop from.  I've got Windows 7.  I've just kept downloading  Window Defender updates, but I am wondering if I should I switch to Windows Essentials - is it better?  If so, will I need to uninstall Defender to avoid conflicts?
rickcp say:
Windows Defender (WD) that comes preinstalled in Windows 7 (and Vista) is an anti-spyware software only. It's part of the system so it can not be removed (you won't see it listed as an installed program) and you may only disable it.

Microsoft Security Essentials (MSE) is a full anti-malware program (antivirus AND antispyware).

If you use MSE, you won't need WD. MSE is designed to disable WD 'automatically' (but it doesn't happen automatically sometimes) upon installation, in order to manage the PC’s real-time protection (AV+AS). It does so, to avoid system's performance degradation and other problems caused by the conflict of two services providing real time protection simultaneously.

For the same reason, you must ensure there is no free or trial security program(s) pre-installed you don't know about. All RTP programs must be uninstalled before installing MSE (or any other AV software).

Would recommend you install MSE following the MSE Checklist & FAQ's.
