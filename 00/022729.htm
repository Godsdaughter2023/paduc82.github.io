  1. Open an administrator command prompt
  2. Type net stop wuauserv and wait for the message "The Windows Update service was stopped successfully."
  3. Delete all files in C:\Windows\Softwaredistribution\ (this will take time and the folders will regenerate minus the files inside them)
  4. Delete the $WINDOWS.~BT folder in the root of c:\
  5. Go back to the command prompt and type Type net start wuauserv and wait for the message "The Windows Update service was started successfully.
  6. Reboot the machine
  7. Go back into the settings app and check for updates again.

The machine will re-download required files again.

You must stop the windows update service if you are going to delete files in the above mentioned folders (which is what the net stop wuauserv is doing.

mike h.. say:

Hi,

Try this code. Note the line in bold holds the names of the worksheets that will be tested and have the rows hidden. Change the names and add to the names as required but ensure the syntax for that line is maintained.

Sub Hide_Me()
Dim cel As Range, rng As Range
Dim v As Variant, s As String
Dim Sh As Worksheet, LastRow As Long
s = "Sheet1,Sheet2,Sheet3"
v = Split(s, ",")
For Each Sh In ThisWorkbook.Worksheets
If Not IsError(Application.Match(CStr(Sh.Name), v, 0)) Then
    LastRow = Sh.Cells(Rows.Count, "A").End(xlUp).Row
    Set rng = Sh.Range("A1:A" & LastRow)
    For Each cel In rng
        If cel.Value = "UNFILLED" Then
            cel.EntireRow.Hidden = True
        End If
    Next cel
End If
Next Sh
End Sub
