a. user say:

This has been discussed over and over again (ad nauseum) on these and many other forums.

 

Right now, the Windows Phone O/S and environment manage memory 'differently' than standard Windows, so there is no real concept of 'folders' and a folder hierarchy which could be exposed via a file manager.  AFAIK, there are no plans in the near future for MS to provide a file manager on the phone itself.  The closest thing is the Windows Phone application available for the PC which allows you to transfer media files to and from the PC and create 'folders' of a kind.

 

As far as the call log is concerned, I'm not sure what you mean, since if you press the phone tile on the start screen, you get a log of all calls received, made, etc.  You can also get a log of calls, emails, messages, etc. for each of your contacts from the People hub...

 

If you are looking for WP to be more like Android, then I'm afraid you're probably going to be disappointed.  WP is different for a reason, and it will continue to chart its own path just to be that -- different.  Personally, I've used both and much prefer WP.

marvinrvdv say:

this sounds like an issue with the part of windows that handles the metro/universal environment.

my grandfather had the same complaint, and it appears to be caused by that part of windows not recovering properly from standby or hibernation.

by default windows 10 hibernates partially instead of shutting down completely, which can make problems like this persist between sessions.

you could try a restart to see if it temporarily helps, but this needs to be a real restart and not turning your pc off and on.

if you can't locate a usable restart button you can try using the shutdown command:

  1. open the run screen with win+R, or open the command prompt by creating a desktop shortcut to cmd.exe
  2. execute the command shutdown -r -t 0
  3. check after the restart is complete if the start menu opens again.

if this fixes it, you can try making the fix permanent by disabling fast startup.

  1. in settings, go to power & sleep.
  2. click additional power settings
  3. in the side panel click choose what the power button does
  4. click change settings that are currently unavailable
  5. uncheck the box at turn on fast-startup
  6. click apply

it might make windows and some apps start a bit slower, but can also fix various stability issues with the metro/universal environment as well as other system components and drivers.
