Everyone, & not just the current version (which, BTW, is Office 2016 for Mac - "Office 365" is the marketing name of Microsoft's subscription service) :-)

This always has been the case in Mac Office. Object Linking & Embedding is significantly less robust than in Office for Windows. Unfortunately, the Insert> Object - From File dialog is quite deceptive on the matter until you attempt to use it.

The [first page of a] PDF can be inserted in the same manner as Picture file types, but only image file types can be linked/embedded & only Word or Excel documents can be linked/embedded as Icons. Further, if a document contains an object embedded (other than Word/Excel) as an icon using a Windows program the icon will display but the actual object will not be accessible.

Private Sub Worksheet_Change(ByVal Target As Range)
Dim rngDV As Range
Dim oldVal As String
Dim newVal As String
If Target.Count > 1 Then GoTo exitHandler

On Error Resume Next
Set rngDV = Cells.SpecialCells(xlCellTypeAllValidation)
On Error GoTo exitHandler

If rngDV Is Nothing Then GoTo exitHandler

If Intersect(Target, rngDV) Is Nothing Then
   'do nothing
Else
  Application.EnableEvents = False
  newVal = Target.Value
  Application.Undo
  oldVal = Target.Value
  Target.Value = newVal
  If Target.Column = 3 Then
    If oldVal = "" Then
      'do nothing
      Else
      If newVal = "" Then
      'do nothing
      Else
      Target.Value = oldVal _
        & ", " & newVal
      End If
    End If
  End If
End If

exitHandler:
  Application.EnableEvents = True
End Sub
