a. user say:

I've solved both of my problems, just updating this question in case someone comes across it and has the same issue -

 

To fix the C00D124D error, I did four registry edits (requires developer/chevron AND interop unlocked phone):

I went and found the following keys

HKEY_LOCAL_MACHINE>Drivers>SDCARD>ClientDrivers>Class>MMC_Class>DisablePowerManagement=1

HKEY_LOCAL_MACHINE>Drivers>SDCARD>ClientDrivers>Class>MMC_Class>High_Capacity>DisablePowerManagement=1

HKEY_LOCAL_MACHINE>Drivers>SDCARD>ClientDrivers>Class>SDMemory_Class>DisablePowerManagement=1

 

After changing all of those from 1 to 0, I no longer recieved the C00D124D error, while syncing, but only when my phone was unlocked (the display times out, "locking" your phone after a certain time)

 

To remedy that, I found this:

[HKEY_LOCAL_MACHINE>ControlPanel>Lock>DisableNever=1

 

Changed that to 0 which allowed me to pick the "never" option in the setting for display timeout. Phone syncs fine now, the only downside being the display has to be on the whole time. Not that big of a deal, just don't leave it over night, LED screens can burn-in.

 

 

 

As for my wireless sync issue, I have two wireless routers in my house, and basically I just connected my phone to the second one and it worked. Not really sure what the issue was or how it got fixed, sorry.

There can be 3 approaches to make it dynamic i.e. it doesn't depend upon the amount of data. All 3 approaches are in the sample file in respective tabs.

1. Formula Approach
Put following Array formula in E2 and drag down to generate unique list of buyers
=IFERROR(INDEX($A$2:$A$10000, MATCH(0, IF($A$2:$A$10000<>"",COUNTIF(E1:$E$1, $A$2:$A$10000)), 0)),"")
Note - Array Formula is not entered by pressing ENTER after entering your formula but by pressing CTRL+SHIFT+ENTER. If you are copying and pasting this formula, take F2 after pasting and CTRL+SHIFT+ENTER. This will put { } brackets around the formula which you can see in Formula Bar. If you edit again, you will have to do CTRL+SHIFT+ENTER again. Don't put { } manually.

Put following formula in F2 and drag down
=IF(E2="","",SUMIF(A:A,E2,B:B))

2. Pivot Table approach
Make a pivot table by selecting column A and B
Now apply a label filter to filter out blank values
Now, when you increase or decrease your data, just refresh the pivot table

3. Structured Table Approach
Select your data > CTRL+T to convert the range into table
Now click anywhere in this table and make a pivot table
Now, when you increase or decrease your data, just refresh the pivot table

Ok Bryan,

let's say that check Box 1 is in cell A1

FIRST un-check CHECK BOX 1, SELECT CELL A1 AND FILL COLOR RED

now,

step1

a) ALT+F11 to open Visual Basic

and paste the below two macros on the right

 

Sub myCheckBox1()
Application.Run "RedGreen1"
End Sub


Sub RedGreen1()
If [a1].Interior.ColorIndex = 3 Then
[a1].Interior.ColorIndex = 10
Exit Sub
End If
If [a1].Interior.ColorIndex = 10 Then
[a1].Interior.ColorIndex = 3
End If
End Sub

 

b) press ALT+q to close Visual Basic

 

step2

Select Cell A1

Format > Cells  > Custom > Type:

delete General Type and add three Semicolons ';;;' (in order to hide TRUE-FALSE)

press OK

 

step3

a) Right click on check box 1 > Assign Macro

and select  'myCheckBox1' .....OK

also

b) Right click on check box 1 > Format Contol > Control > Cell link: ....write A1

press OK

 

cell A1 is ready

XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

Note

if you want entire column A fill red-green

then

1) FIRST un-check CHECK BOX 1, SELECT ENTIRE COLUMN A AND FILL COLOR RED

2)

Sub RedGreen1()
If Columns(1).Interior.ColorIndex = 3 Then
Columns(1).Interior.ColorIndex = 10
Exit Sub
End If
If Columns(1).Interior.ColorIndex = 10 Then
Columns(1).Interior.ColorIndex = 3
End If
End Sub

 

instead of Columns(1), you can write: Columns("A:A")

norman david jones say:

Hi Stacie,

I have an excel file, two columns and 1884 lines of data.  One column contains dates from 1990 to 2015; the other column is rainfall amount, in inches.  I'm looking for the formula to easily identify rainfall events that take place over three or more consecutive days.

Try something like:

  • Alt+F11 to open the VBA editor
  • Alt+IM to insert a new code module
  • In the new module, paste the following code 

'=========>>
Option Explicit

'--------->>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet
    Dim Rng As Range, rCell As Range
    Dim LRow As Long
    Dim CalcMode As Long
    Dim iCtr As Long

    Const sSheetName As String = "Sheet1"      '<<=== Modify


    Set WB = ThisWorkbook
    Set SH = WB.Sheets(sSheetName)

    With Application
        CalcMode = .Calculation
        .Calculation = xlCalculationManual
        .ScreenUpdating = False
    End With

    With SH
        LRow = .Cells(Rows.Count, "B").End(xlUp).Row
        Set Rng = .Range("A2:A" & LRow)
    End With

    With Rng
        .Interior.ColorIndex = xlNone
        .Offset(, 2).ClearContents
        For Each rCell In .Cells
            With rCell
                If .Offset(1).Value = .Value + 1 Then
                    iCtr = iCtr + 1
                Else
                    If iCtr > 2 Then
                        .Offset(-iCtr).Resize(iCtr + 1) _
                                .Interior.Color = vbRed
                        .Offset(-iCtr, 2).Resize(iCtr + 1, 1).Value _
                                = ">= 3 Day Rainfall"
                    End If
                    iCtr = 0
                End If
            End With
        Next rCell
    End With

XIT:
    With Application
        .Calculation = CalcMode
        .ScreenUpdating = True
    End With
End Sub
'<<=========

  • Alt+Q to close the VBA editor and return to Excel
  • Save the file as a macro-enabled workbook with an xlsm extension
  • Alt+F8 to open the macro window
  • Select Tester | Run
