consider a small macro that you can call when a single shape has been selected. you of course can extend the process by creating a simple form with a text box that can act as input to the process.

al

Sub reNameMe()
    Dim visApp As Visio.Application
    Set visApp = Application
    Dim visWin As Visio.Window
    Set visWin = visApp.ActiveWindow
    Dim visSel As Visio.Selection
    Set visSel = visWin.Selection
    ' allow only one shape to be selected
    If visSel.Count = 1 Then
        Dim visShape As Visio.Shape
        Set visShape = visSel(1)
        visShape.Name = "newName"
        visShape.NameU = "newName"
    End If
   
End Sub

If the vertical lines go below the horizontal line at what you think is the bottom of the section. then the horizontal line is not really at the bottom of the section.  Check to make sure no other controls touch the line and then drag the bottom of the section up above the line.  It should end up with the line looking like its part of the section divider bar.

 

Note. vertical lines will automatically be clipped at the bottom of its section.  This means that you do not have to go to any pains to figure out the height of the section.  Just specify a huge height such as 65000 to go to the bottom regardless of how tall the section ends up being,

 

The report will open in the view specified in the OpenForm action or method.

   Me.Line (2.5 * 1440, 0)-(2.5 * 1440, 14400) 'Left of Total

could also be written:

   Me.Line (2.5 * 1440, 0)-Step(0, 14400) 'Left of Total
