a. user say:

You can if you want to. However, unlike earlier versions of Windows, in Windows 10 updates are cumulative. So you can recover with the original July 29, 2015 version of Windows 10 and Windows Update will do the rest.

But if you really want to do yourself a favor, forget about making a recovery disk. Instead, learn how to make a disk image. An image is an exact copy of your entire hard drive, compressed into a single file.

Now let's say I'm making some changes to my computer's configuration and I end up completely destroying my computer. Not that I would actually do that <ahem> but let's just say, for example. If the only thing I've got is a recovery disk, I'll have to run Windows update, re-install all my software, and set up my computer all over again. There goes the afternoon.

But if I have a recent disk image, I can restore that instead. It takes under 10 minutes, and I've got my computer and all my software back just the way I left it. Like nothing ever changed.
