bob jones aka: cybertaz say:

Content that 'shows up' on printed/PDF slides can be from any of 4 sources:

  1. On the slides, themselves
  2. On the Master on which the slide is based
  3. Inserted as a Slide Background -- which should still be visible but not accessible on either a slide or a Master
  4. Added by the printer -- Some printers have Watermark/Header/Footer options.

It sounds like you've checked the first 2 but what about the others? My bet is #3 :-) Right-click an open space on a slide that displays the logo in a PDF, then choose Format Background. See if anything has been inserted on the Picture or Texture page. If so, go to the Solid page, selectWhite (or appropriate alternative) from the Color dropdown, then click Apply to All.

In either of the first 3 cases, the image should be visible in the presentation file unless it has been set to 100% Transparency. It's also possible that it may be virtually invisible when you open the PPT file if the Transparency has been set to something a little lower. Regardless of the setting the image should not appear with any more intensity when printed to paper or PDF... except that some printers do not handle Transparency very well.
