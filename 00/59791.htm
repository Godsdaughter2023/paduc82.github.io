Method-

 

I would have you open Task Scheduler that will wake up the computer at a set time.

 

Refer the steps-

a)       Click on “Start” and type “Task Scheduler” (without the quotes).

b)       Now, select “settings”.

c)       Click on the Task Scheduler and create a task (not a basic task).

d)       Now, under the “General” tab place a dot next to “Run whether user is logged on or not”.

e)       Underneath “Triggers” set a recurring time to wake up.

f)        Underneath “Conditions” place a check mark next to “Wake the computer to run the task”.

g)       On “Actions” have it run a program.

h)       In the Program/script field enter the command: cmd.exe

i)         In the arguments field add: /c “exit”

j)         Click on “Ok” and check if the issue persists.
