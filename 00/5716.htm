If you did save the Merge Tools file in the Word Startup Folder, and you have re-started Word after doing so, there is

  • No need to open that file
  • The Merge Tools tab will remain on the ribbon

If you go to the Developer tab of the ribbon and then click on the Document Template item in the Templates section of the ribbon to display the Templates and Add-ins, you should see the MergeTools -20140218.dotm item in the list of items with a check mark in the box beside it showing that it is loaded.

IF you do not see that, then you have not saved the file in the correct location.
