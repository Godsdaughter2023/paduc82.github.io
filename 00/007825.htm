I had to find my older Mac to use PowerPoint 2004; I am so sorry for the delayed response -- follow these steps for PowerPoint 2004 for Mac:

  1. Select the slide object that you want to animate.
  2. Choose Slide Show | Custom Animation to bring up a dialog of the same name.
  3. Click the Add Effect button to bring up the Animation Effects dialog.
  4. Choose Entrance | Subtle | Fade. Click OK to get back to the Custom Animation dialog -- the animation you just added will be listed in the Animation order box.
  5. With that animation selected, change the Start option to After Previous.
  6. Click the Add Effect button again to bring up the Animation Effects dialog.
  7. Choose Exit | Subtle | Fade. Click OK to get back to the Custom Animation dialog -- the animation you just added will be listed below the existing animation in the Animation order box.
  8. With the exit animation selected, change the Start option to After Previous (or let it be On Click).
  9. Play your slide to see if this works for you.

phillip m jones say:
  1. Quit Word. (Save Document but have under slightly different name.)
  2. Now open a web browser and Click on this Link.
  3. Go o page named "Font Book Lion and Above"
  4. Open Font Book and follow directions on web page.
  5. Remove any Duplicates and defective fonts.
    (If removed due to defective, see if you can find replacements).
  6. Restart the computer in Safe Mode. 
  7. When completely loaded, restart again, this time normally.
  8. Reopen your Document (saved with the altered Name.
  9. Use Select All (⌘ - A) and choose you desired Font. Save again and reopen now try some new content.
  10. If successful, you have option to continue on using altered name, or paste all new content from time you quit using original Document and saving changes in the original document.
