suzanne s. barnhill say:

What this usually means is that the document is opening in a Normal window offscreen (sometimes this happens if you have dual monitors enabled but only one is turned on). You can move it back where it is visible.

  1. Hover over the document button the Taskbar until you get a floating button above the Taskbar (as shown below).
  2. Right-click on that button to get the menu shown below.
  3. If the menu offers Maximize and Minimize (Restore is dimmed), then click on Move.
  4. You will then be able to use the arrow keys to bring the window back onto the screen.
  5. When you can see it, grab the title bar with the mouse and drag it where you want it.
  6. Use the mouse to size it as desired, then quit Word.
  7. When you restart Word, the window should be where you left it.
