hansv mvp say:

You have to divide the document into sections by selecting Insert | Break... at the appropriate places in the document and selecting Continuous under Section Break Types.

Then select Tools | Protect Document...

This activates the Protect Document task pane.

Tick the check box "Allow only this type of editing in the document".

Select "Filling in forms" from the dropdown below it.

Click Sections...

Clear the check boxes for the sections that should remain unprotected.

Click OK.

Click "Yes, start enforcing protection".

Specify a password if you wish (if you do, don't forget it!), or leave it blank, then click OK.

How to find out when a Office 365 group was created? - Microsoft Community

a. user say:

How to find out when a Office 365 group was created?

Hi,

Is there any way to find out when a Office 365 group was created without having to join the group and looking at the earliest files uploaded (which is time-consuming)? Looked everywhere in the Admin Centre and also at EAC, but there is no such info. 

Any tips?

Thanks.

barry xie msft say:

Hi Rron,

 

Yes, you can use Windows PowerShell commands to check when the specific Office 365 group is created. Here are the steps:

1. Connect to Exchange Online PowerShell with an Office 365 admin account.

2. Run this commend and then check the “When Created” value:

Get-Group -Identity "Group name" | Format-List

 

About checking the earliest files uploaded data, you need to join the specific Office 365 group > Find it in Outlook Web App > Navigate to Files and then check the Modified value of the files.

 

Regards,

Barry
