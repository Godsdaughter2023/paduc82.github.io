WS2003 uses the SMB1 network sharing protocol and cannot negotiate SMB2 or SMB3.

a. user say:

Outlook exports the contact info. into a file called "OMMn.doc", where n is a number. If you go into Edit->Recipients you may be able to see the full path name of the file that Outlook has set up as the data source in Word (e.g. c:\Users\myname\AppData\Local\Temp\OMM0.doc ) You may be able to drag the right-hand edge of the column to the right so that you can see the whole path+file name.

If you can navigate to that folder using WIndows Explorer ("My Computer", etc.), you may see a bunch of other files - I suggest that you either rename them all, or delete any that yuou do not need.

That may help Outlook choose the current export file the next time (but I wouldn't be too certain). If not, the best you can probably do right now is use Word->Mailings->Select Recipients to select the correct file in that folder. Just be aware that despite the .doc extension these files are actually plain text files (the kind that you can open in Notepad)

skittles1313 say:

How do I changed the "Displayed Language" on my HTC HD7 to Chinese?

How do I changed the "Displayed Language" on my HD7 to Chinese? Is there a chinese language pack that I can download from either MS/HTC, or do I have to purchase a new 7.5 phone? Thanks.
steven _fyiguy_ hughes say:
The Windows Phone update process cannot add or remove display languages. Although newer versions of the Windows Phone software might add support for additional display languages, the new languages will only be available on new phones. After you update your phone, you'll continue to have the same display languages that you had before the update. 

Here are the steps to view available display languages:
  1. From the Start screen, flick left and tap Settings
    .
  2. Tap Region+language.
  3. Tap the language listed under Display language to see which other languages are available.

If available choose the region format of your version of Chinese;System Locale and choose your version of Chinese from the drop down list; Browser and search language and Display Language as well. Some changes may require a restart of your phone.

Most phones display language is locked to the carrier and region that the phone was sold in.

You can also change the keyboard under settings->system->keyboard to different input languages of Chinese.

Hope this helps
