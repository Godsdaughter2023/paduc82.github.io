dianeoforegon say:
  1. Quit Outlook.
  2. Move your Identity to the Desktop (delete after you complete the process
  3. Upgrade Outlook to 14.2.5
  4. Start over in a new Identity. There was an upgrade process from SP1 to SP2 that failed. It's much safer to start over in a new Identity.

Updaters need to be applied in order:
14.1.0 or higher
14.2.0
14.2.3
14.2.5

Follow these steps using the Microsoft Autoupdater (MAU)

  • Open any Office application and select "Check for Updates" under Help in the Menu bar. Once MAU is launched, select "Keep in Dock"
  • Log out/in as described below.
  • Select the MAU in the Dock and download the updater.
  • After it downloads, quit the MAU then install.
  • RESTART

Restarting after each updater is key to a good install. However for the Microsoft AutoUpdate (MAU)  and the Microsoft Error Reporting tool (MERP) it is not necessary to restart.

Log out under the Apple in the Menu bar.
When you log in, hold down the Shift Key. This disables all login items including the hidden ones that the all_quit script is trying to quit.

Service Pack 2 (SP2) 14.2.0 makes some major changes to file locations. Unfortunately, some of the files are not moved as expected. I suggest you rename the Microsoft User Data folder to Microsoft User Data OLD. When you launch Outlook it will create a new folder. Compare the two folders. You'll see Microsoft  moved some folders, but left the old ones in MUD folder.

If you are a Lion/Mountain Lion user you will need to enable the User’s Library folder. Hold down the Option key and select Go in the Menu bar to view your User’s Library folder.

Folder changes

It's possible you will need to log into your account to clean up your Inbox before downloading into Outlook.

Exchange Basics

The Inbox syncs more frequently than any other and should be kept as clean as possible. (this is the number one cause of problems with sync). Do NOT make subfolders under the Inbox.

Any folder with more than a few thousand messages is going to take some time to fully come down when you first sync your account. Due to the design of Outlook's sync engine (and how exchange works), you will see the 512 newest messages in a folder at the initial start of sync then the rest of the folder's contents will be back filled before you see any mail that arrives after this point. This is why it will look like you are not getting new mail when a folder is still going through initial sync. This is also further exacerbated by an Exchange issue where it gets unnecessary change events before the newer mail arrives. If you carefully watch the progress during the sync of a folder with greater than a couple thousand messages, you will see "Updating local..." a lot before the newer mail starts arriving (newer than the initial 512 that come down).

Outlook checks the server for updates every minute, any folder that has updates will subsequently be synced. Since Outlook has a limit on how many folders can be synced at a time, there can be a queue of folders waiting. The Inbox does get high priority so it will generally sync before other folders that also need to sync.

There could be multiple causes when you are unable to play any games. Technical Preview should work with the same devices and programs that work with Windows 8.1, but you might need to update or reinstall some of them. How to perform a clean boot to troubleshoot a problem in Windows 8
http://support.microsoft.com/kb/929135

Note: Once you are done with clean boot, please follow the section ‘How to reset the computer to start as usual after troubleshooting with clean boot’ under ‘more information’ in the article to put your computer back to normal startup.

Note: The steps are also applicable to Windows Technical Preview.

If the issue still persists, you may try to install the game in Windows Compatibility mode.

a. Locate the executable file (.exe file) for the game installation program.

b. Right-click the file, and then click Properties.

c. In the Package Name Properties dialog box, click the Compatibility tab.

d. Click to select the Run this program in compatibility mode for check box, click Windows 8 with which it is compatible with in the Run this program in compatibility mode for list, and then clickOK.

e. Double-click the executable file to start the installation program.
