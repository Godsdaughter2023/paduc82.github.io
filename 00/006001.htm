No, it is not a joke. As Phillip pointed out, links change over time. The link no longer works, so I deleted it.

One of Excel's most important features is that you can build your own custom solutions In Microsoft Visual Basic for Applications (VBA), AppleScript, and Automator to solve problems like this one involving window size and position. 

Here is a code snippet that you can use in VBA. Just copy and paste this into the VB Editor.

Sub Auto_Open()
'
'This runs when the workbook opens.
'
On Error GoTo errorhandler
'
'Make sure version is 2007 or later
If Application.Version < 12 Then
    msgbox "Requires Excel 2007 or later for Windows" & Chr(13) & "or Excel 2011 or later for Mac."
    Exit Sub
End If
'
'maximize & set zoom, allow Mac users to choose
    If Application.OperatingSystem Like "*Mac*" Then
        With ActiveWindow
            .WindowState = xlMaximized
            .Top = 1
            .Left = 1
            .Height = Application.UsableHeight
            .Width = Application.UsableWidth
        End With
        With Application
        'Allow user to choose zoom
            .Dialogs(xlDialogZoom).Show 130
        End With
    Else
        With ActiveWindow
            .WindowState = xlMaximized
        End With
        ActiveWindow.Zoom = 100
    End If
'

End Sub

You'll need the following macros stored in an add-in in each user's Startup folder or in the Normal template:
 
Sub FileSaveAs()
SetName
 
End Sub
 
Sub FileSave()
If ActiveDocument.Path = "" Then
    SetName
Else
    ActiveDocument.Save
End If
End Sub
 
Sub SetName()
Dim SuggestedFileName As String
Const CompanyName$ = "My company"
 
SuggestedFileName = CompanyName & " Agreement"
 
With Dialogs(wdDialogFileSaveAs)
.Name = SuggestedFileName
.Show
 
End With
End Sub
