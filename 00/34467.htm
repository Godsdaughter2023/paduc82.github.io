Let's try to nail down what we can find out for certain.  Open up the People app, then go to Settings and choose Accounts.  You'll see a list of all the places the people app is search for contacts (the People app then combines all these contacts into a single address book).

Recovering Windows 7 Administrator Password - Microsoft Community

f. shuaib say:

Recovering Windows 7 Administrator Password

My dad forgot the password to his administrator account. Unfortunately, he has the only admin account on our PC. Is there anything I can do from a standard account to recover it?

cmdrkeene say:

Not from a Standard account, but you can use the hidden administrator account as long as that one was not touched by anyone.  By default, the one true administrator account is hidden until you boot into safe mode, just for occasions like this.

Obviously some customers decide to ignore the warnings and use the administrator account anyway, instead of creating their own admin-level account.  If that is what happened, then the only way to recover from a forgotten password is to erase the computer and start over from factory conditions.

Assuming that your dad did not use the hidden administrator account, it can be used to help you.

You can boot to Safe Mode to access the Built-in Administrator account.  From this fail-safe account, you can force-reset your account's password.

  1. Reboot your computer, tapping F8 as it starts.
  2. On the first screen that appears, select Safe Mode.

Now that you are in safe mode, you can use the built-in backup Administrator account to perform you necessary recovery. The built in Administrator account should not be used for any other purpose, or it won't be available in situations like this when you need it.

  1. At the login screen, select the Administrator account.
  2. After Safe Mode starts, open Control Panel / User Accounts.
  3. Select Manage another account and select the account you wish to change to an Administrator account. You can also use the Change Password option here to reset the other account's password.

When you're done, restart the computer normally.

A OEM_SLP Key self-activates when all things are right - it doesn't need to connect to MS's Activation servers to do so.

If conditions are not right for an OEM_SLP Key to self-activate, then it will attempt activation with the MS servers - at which they are blocked from activation, and so return a 0xC004C003 error.
