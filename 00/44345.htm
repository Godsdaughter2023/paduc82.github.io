lemp say:

In "Standard mode," Windows Calculator evaluates operations in the order in which you type them rather than waiting until an entire equation is typed to begin calculating.

Thus, when you type 3 + 3, as soon as you enter the second 3, Calculator computes the result.  If at that point, you press the = key, this value is shown.  If at that point you enter another operator, it operates on that already-calculated value.

If you use Scientific mode, the calculator operates differently, honoring the "order of operations" rules (and you can also use parentheses).

See, for example, https://mathblag.wordpress.com/2011/01/02/some-misunderstandings-about-order-of-operations/

I like to introduce the topic by showing my students two ways to calculate and asking them which one is correct. I also show them that different calculators will give different answers to the same problem, depending on the type of logic that they use. A four-function calculator gives the answer 35, but a scientific calculator gives 23. In fact, Windows calculator gives both answers, depending on the mode (standard or scientific).

xircal say:

On an OEM laptop, that is to say, one which has Windows preinstalled, the product key is embedded in the BIOS in an encrypted format and therefore isn't accessible to you.

But provided you follow the standard procedure to rollback to Windows 8.1, it shouldn't  be necessary.

The following site provides a good tutorial with screenshots on how to rollback to your previous operating system: https://techingiteasy.wordpress.com/2015/03/24/how-to-rollback-to-a-previous-version-of-windows-from-windows-10/

The only other option is a reset. For that you'll need the recovery media you created when you purchased the laptop.

If you didn't create it then your only option will be to contact the manufacturer and see if you can purchase the recovery discs. If it's a Dell laptop, you can order those from this site: http://www.dell.com/support/contents/us/en/19/category/Product-Support/Self-support-Knowledgebase/software-and-downloads
