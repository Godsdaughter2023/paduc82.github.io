Word 2010 Macro that sets default drop down value - Microsoft Community

artgirl91 say:

Word 2010 Macro that sets default drop down value

Hello,

I have been trying to run a macro that will update a dropdown form field from an external list. I got the code below from Graham Mayor here:

http://answers.microsoft.com/en-us/office/forum/officeversion_other-word/using-the-same-drop-down-box-in-multiple-documents/4161f581-2934-40c6-9fa6-a129991abee5

Everything works perfectly as far as being able to update 100+ documents with the same drop down, however I can not figure out how to set a default value in the drop down box. It currently is blank when I open a document and I have to select a value from the drop down. I would like it to already be filled with a default drop down value. Can someone help me figure out what I need to add to the code below in order to make that possible? Thanks for the help!

Sub UpdateDropDown()
Dim oDD As DropDown
Dim strFilename As String
Dim strPath As String
Dim strNameDoc As String
Dim oDoc As Document
Dim oListDoc As Document
Dim oRng As Range
Dim fDialog As FileDialog
Dim i As Long, iName As Long

'Change the path to match the name of your namelist document
'which must comprise of one name per line with no blanks
'Maximum number of lines is 25!
strNameDoc = "LINKED DOCUMENT HERE (already changed on my macro)"

Set fDialog = Application.FileDialog(msoFileDialogFolderPicker)
With fDialog
.Title = "Select folder and click OK"
.AllowMultiSelect = False
.InitialView = msoFileDialogViewList
If .Show <> -1 Then
MsgBox "Cancelled By User", , _
"List Folder Contents"
Exit Sub
End If
strPath = fDialog.SelectedItems.Item(1)
If Right(strPath, 1) <> "\" _
Then strPath = strPath + "\"
End With
If Documents.Count > 0 Then
Documents.Close SaveChanges:=wdPromptToSaveChanges
End If
Set oListDoc = Documents.Open(FileName:=strNameDoc, _
Visible:=False)
For i = oListDoc.Paragraphs.Count To 1 Step -1
If Len(oListDoc.Paragraphs(i).Range) <= 2 Then
oListDoc.Paragraphs(i).Range.Delete
End If
Next i
If Left(strPath, 1) = Chr(34) Then
strPath = Mid(strPath, 2, Len(strPath) - 2)
End If
strFilename = Dir$(strPath & "*.doc")

While Len(strFilename) <> 0
Set oDoc = Documents.Open(strPath & strFilename)
Set oDD = ActiveDocument.FormFields("DropDown7").DropDown
End If

With oDD.ListEntries
.Clear
If oListDoc.Paragraphs.Count > 25 Then
iName = 25
Else
iName = oListDoc.Paragraphs.Count
End If
For i = 1 To iName
Set oRng = oListDoc.Paragraphs(i).Range
oRng.End = oRng.End - 1
.Add oRng.Text
Next i
End With
oDoc.Close SaveChanges:=wdSaveChanges
strFilename = Dir$()
Wend
End Sub

greg maxey say: You have to set the dd value to 1.  That is if the first entry is the defualt entry:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oDD As DropDown
Set oDD = ActiveDocument.FormFields("DropDown7").DropDown
With oDD.ListEntries
 .Clear
 .Add "A"
 .Add "B"
 .Add "C"
End With
oDD.Value = 1
End Sub
