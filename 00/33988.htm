Hi,

This is simple but should not be done as it might have an effect with your Windows Licensing. You can do it with your own risk.

What you need to do is to use the Registry Editor to edit the information on the Windows version information window. Since we are dealing with Registry Editor, please ensure to back up your Registry or create a system restore point in case something along the way goes unexpected.

Please follow the steps below to edit the desired information:

  1. Click Start Menu and type RegEdit in the Search box.


  2. In the Registry Editor, locate the following registry keys from the left pane:
    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion

  3. On the right pane, check if the following Registry Strings are available as shown on the screenshot:


  4. Double-click on the on each Strings and change it to your desired information.
    • RegisteredOrganization - you can leave it blank or change it as well
    • RegisteredOwner - i.e. Windows User

  5. If the said strings are not available, you can create it by right-clicking on any areas on the right pane and select String Value.

  6. Name it as RegisteredOwner if you opted to only change the registered username and press Enter. Then follow the Step 4.

  7. Do the same for RegisteredOrganization if you opted to add one.

The screenshots provided are evidences that it's working on my end. Hope this will for you too.

Note: Applies to Windows Vista, Windows 7, Windows 8, Windows 8.1, and Windows 10.

And for your last concern about "Also, I don't think I got an option to configure the Username which I see in C;\Users or elsewhere. It just chose "amuly" by default and I'm stuck with it. How do I also change this? Thank you.", there will be no option to rename the user folder unless you create a new Local account. Follow the steps on the article on how to Create a local user or administrator account in Windows 10. Once done, go to Settings > Accounts > Your info > Sign in with a Microsoft account for you to use your email address as login information.
