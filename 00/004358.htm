halifax user say:

Display only 2 levels of a table of contents

 when the entries were done they were done by marking table entry fields through alt+shift+O.  I have marked three levels but when I generate this time I only want 1 level to show.  In the The Table of Contents dialog box you can limit the TOC entries that were added from heading styles not field codes.  Is there a way to do this without removing level 2 and 3 codes?
suzanne s. barnhill say:

You'll need to edit the TOC field to add a switch, \l, which is described in Help as follows:

\l Levels

Builds a table of contents from TC fields that assign entries to one of the specified levels. For example, { TOC \l 1-4 } builds a table of contents from TC fields that assign entries to levels 1-4. TC fields that assign entries to lower levels are skipped.

If this switch behaves the same way as the \n switch (which omits specified levels), you will need to specify inclusive levels. That means that if you want two levels (as your title states), you will need \l 1-2, but if you want just one level, the switch you need will be \l 1-1.
