jllatham say:

Basic problem seems to be with the

(data!C4:E4=C3) portion of it - all arguments must reference the same number of cells.  Also as presented here, you're missing a final closing parenthesis at the very end.

 

[Edit]

Also, you've got Contract spelled wrong on the data sheet - so the result is always going to be zero even with everything else fixed.

 

Another issue, you're testing each row in column B to see if it matches either of 2 values: C1 and C2, but since the item in column B can't match both at the same time, once again the result will always be zero.

 

Let me see what I can do with it for you.

--

OK, try this formula for cell D4 on your Result sheet:

=SUMPRODUCT(--(data!A$6:A$35=B4),--(data!B$6:B$35=$D$1),data!D$6:D$35)+SUMPRODUCT(--(data!A$6:A$35=B4),--(data!B$6:B$35=$D$2),data!D$6:D$35)

 

And this one for cell E4 on the Result sheet:

=SUMPRODUCT(--(data!A$6:A$35=$B4),--(data!B$6:B$35=E$1),data!E$6:E$35)+SUMPRODUCT(--(data!A$6:A$35=B4),--(data!B$6:B$35=E$2),data!E$6:E$35)

 

I basically ignore the date completely - you'd need the date entered into each row on the data sheet to incorporate that into the SUMPRODUCT() formulas. 

 

I split it into 2 formulas, the first SUMPRODUCT() gets the totals of the "Labour" entries for a contract, and the second SUMPRODUCT() gets the totals of "Subcontractors" entries for a contract.

 

hansv mvp say:

The formula should be

 

=SUMPRODUCT((data!$A$6:$A$35=$B4)*(data!$B$6:$B$35=C$1)*(data!$B$6:$B$35=C$2)*(data!$C$4:$E$4=C$3)*data!$C$6:$E$35)

 

Note the different placement of a parenthesis.

The formula will return 0 since

1) You have "Contract 1" on the Result sheet and "Contrcat 1" on the data sheet.

2) The conditions data!$B$6:$B$35=C$1 and data!$B$6:$B$35=C$2 are mutually exclusive.

 

If you intended an OR condition on C$1 and C$2 instead of an AND condition:

 

=SUMPRODUCT((data!$A$6:$A$35=$B4)*((data!$B$6:$B$35=C$1)+(data!$B$6:$B$35=C$2))*(data!$C$4:$E$4=C$3)*data!$C$6:$E$35)
