    andreas killer say:
    I want sheet1 to print ONLY IF there is no RED FILL, which is formatted in the normal sense, in any of the cells. It is not in column M due to conditional formatting.  

    Sorry, I had you probably misunderstood.

    It is not possible to skip the print of a single sheet when you print the whole file.

    There is a BeforePrint event that we can use, unfortunately there is not argument in that event that tells you which sheet is printed.

    And you have to access each cell in column M to check if it has a fill or not, means its a time consuming process.

    So when you copy the sub below into the code module ThisWorkbook, it cancels the print when the active sheet has a fill in column M.

    Andreas.

    Private Sub Workbook_BeforePrint(Cancel As Boolean)
      Dim R As Range
      'Check each cell
      For Each R In Intersect(ActiveSheet.UsedRange, Columns("M"))
        If R.Interior.ColorIndex <> xlColorIndexNone Then
          'This cell has a fill, cancel the print
          Cancel = True
          MsgBox "Print canceled"
          Exit Sub
        End If
      Next
    End Sub

    rob rolnick say:

    Hi mw83041,

    Image in front of page number - Microsoft Community

    angel lulu say:

    Image in front of page number

    How to display page number in front of the picture? (I've set the image to the backmost. The number is still behind.)

    Thank you very much for your time.

    charles kenyon say:

    Think about the body layer as being a clear piece of plastic that lies on top of the header/footer layer. You can see things in the header/footer layer so long as you don't put things on the piece of plastic that block the view.

    You have set the image as behind text in the layer that is on top of the headers and footers, the main layer of the document. The image must be behind text in either the header or the footer, not in the body of the document. This is somewhat problematic if you only want the image on one page, this must then be a separate header/footer from those for the rest of your document. This can be done, but is most easily done by creating a new section with headers and footers not linked to previous.

    Header/Footer Settings Recap

    Alternatively, place a borderless, backgroundless, textbox or frame in front of your picture where you want the page number and insert a page number field in the textbox. Alt+Shift+P

    This has the advantage of not requiring a section break, so your page numbers will automatically be consistent. The disadvantage is that you have to remember about the textbox.

    BTW, the technical term in Word jargon for these layers is "stories."

    Try following one by one as see after every step whether it resolves the issue-

    1. Sometimes, add-ins can cause problems. To determine, if add-in is a problem start your application in safe mode and see if problem has gone away.

    To start in Safe Mode -

    Hold CTRL key and click on application icon and don't release the CTRL key till it asks you for Safe mode confirmation.

    If yes, then start application normally and disable add-ins one by one and start application again every time you disable an add-in to determine the culprit add-in.

    2. Try repairing your office installation (Quick repair is good enough). Try repairing your Office installation. First try Quick Repair (takes 5 - 10 mins) , if that doesn't help then try Online Repair (Takes more than 30 mins but less than 1 hour in general)

    3. Login to Windows with a new user profile and see if problem is resolved.

    4. Sometimes, this is caused due to application not being able to access the default printer. Close all office applications and change your default printer through Start > Devices and Printers to another printer. It may be some physical one or One Note or XPS or PDF or anything else. If this is the problem, then you will need to update your printer driver for default printer.
