Try the code below. I have adapted it for you from a previous answer that I gave to someone.

Because you say you are using Form control checkboxes, the code can be in one sub and that single sub is assigned to all of the checkboxes. Application caller determines which Checkbox called the code.

The positions of all the checkboxes are all saved based on the TopLeftCell property of the contro. After Hiding/Unhiding the rows the controls are all repositioned based on the original saved locations.

Of course if your controls are not Forms controls and are ActiveX controls then the code will not work. However, the same principles can be used for ActiveX controls but the individual controls must have their own code and you would need to call a subroutine to save the Checkbox positions and for repositioning the controls after Hiding/Unhiding the rows.

'Code below edited since initial post. Improved method.

Sub CheckBox_Click()
    'All forms Control CheckBoxes call this code
    Dim chkBox As Shape
    Dim shp As Shape
    Dim arrPos()    'to save position of checkboxes
    Dim i As Long
    Dim lngRow As Long
   
    With ActiveSheet
        Set chkBox = .Shapes(Application.Caller)
    End With
   
    i = 1
    ReDim arrPos(1 To 2, 1 To i) 'Initialize to 2 x 1 elements
    For Each shp In ActiveSheet.Shapes
        If shp.Type = msoFormControl Then
            If shp.FormControlType = xlCheckBox Then
                ReDim Preserve arrPos(1 To 2, 1 To i)
                arrPos(1, i) = shp.Name
                arrPos(2, i) = shp.TopLeftCell.Address 'Save address of the checkbox
                i = i + 1
            End If
        End If
    Next shp
   
    lngRow = chkBox.TopLeftCell.Row + 1     'first row to hide
    With ActiveSheet
        If chkBox.ControlFormat = 1 Then
            .Rows(lngRow & ":" & lngRow + 3).Hidden = True  'Hide next 4 rows after CheckBox
        Else
            .Rows(lngRow & ":" & lngRow + 3).Hidden = False 'Unhide next 4 rows after CheckBox
        End If
    End With
   
    With ActiveSheet
        For i = LBound(arrPos, 2) To UBound(arrPos, 2)
            .Shapes(arrPos(1, i)).TopLeftCell = .Range(arrPos(2, i))
        Next i
    End With

End Sub

This only works with Forms Controls. It will not work with ActiveX controls that need their own individual Event code. I have changed the Hide/Unhide to Unhide on Check and Hide on Uncheck. This is better than setting Hidden to opposite of current Hidden/Unhidden condition because there are circumstances where you could have some rows hiding on check and some hiding on uncheck and vice versa.

See my comments in the code. I have included code to size and align the CheckBoxes. Sub HideUnhide(ctrl As Object, rngRows As Range)

    With ActiveSheet
        If ctrl.ControlFormat = 1 Then    '1 is Checked
            rngRows.Hidden = False  'Unhide rows when CheckBox is checked
        Else
            rngRows.Hidden = True 'Hide rows when CheckBox is unchecked
        End If
    End With
   
End Sub

Switching the BIOS from AHCI to IDE mode

The following steps describe how to successfully switch the BIOS from AHCI to IDE mode without risking system crashes during boot:

  1. Open an elevated command prompt and set the machine to boot into Safe mode by typing
    bcdedit /set {current} safeboot minimal
  2. Reboot the machine into Safe mode by typing:
    shutdown /r /t 0
  3. During reboot, enter the system BIOS and switch the SATA emulation from AHCI to IDE, then choose SAVE and Reboot.
    The machine should now boot into Safe mode. In Safe mode all boot-start drivers will be enabled and loaded. Since the boot device is now used via IDE, the necessary drivers will also be loaded on subsequent boots.
  4. Reset the bcdedit settings to allow the machine to boot into the Normal mode by typing:
    bcdedit /deletevalue {current} safeboot
  5. Reboot the machine into Normal mode by typing:
    shutdown /r /t 0
This should successfully reboot into normal mode using IDE.
