jim gordon mvp say:
If you want to change all the .doc files to .docx, you can use an Automator action.
http://www.agentjim.com/MVP/Automator/ConvertWordFiles.html

You can't use the Automator action in Home and Student edition of Microsoft Office.

In Word, when you open a (.doc) file it will say Compatibility Mode at the top of the window and many features will be disabled. To update the file and turn on all of Word's features, on Word's File menu choose Convert Document, and then click the OK button.
