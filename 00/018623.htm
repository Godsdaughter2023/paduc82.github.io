Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, _

This is a great question.

Negative keywords can now be shared from campaign to campaign by using negative keyword lists.  With the addition of Negative Keyword Lists - you now have the option to share a list among several campaigns.

To use a shared negative keyword list

First you must create a negative keyword list and add your negative keywords. After this is done, you can add the list to one or more campaigns.

Step 1: Create a negative keyword list
  1. Click Shared Library in the left navigation pane.
  2. Click Create or edit your negative keyword lists.
  3. Click Create negative keyword list.
  4. Give your list a name, and enter your negative keywords, one per line. You can enter up to 500 negative keywords per list.
  5. Click Save.
Step 2: Add a negative keyword list to one or more campaigns
  1. Click Shared Library in the left navigation pane.
  2. Click Campaign negative keywords in the left navigation pane.
  3. Select the checkbox next to the list you want to add to one or more campaigns.
  4. Click Add to campaigns.
  5. Click the arrow button (>>) next to the campaigns that you want to associate with this negative keyword list.
  6. Click Save.

"Data Execution Prevention"
"No Execute Memory Protection"
"Execute Disable Function" with a sub-menu of  'No-Execute Page Protection Technology."
"CPU protection"
or similar wording
If you do not have any of these in your BIOS ignore it.
