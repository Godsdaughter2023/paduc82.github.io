    Because you have upgraded once to Windows 10, now you can clean-install Windows 10 using the Media Creation Tool ISO,

    Clean the drive does not return you to Windows 8.  It is a more thorough wipe of the disk, in case you are selling or giving the computer to someone.  Both options will reinstall Windows 10, but the last one takes a very long time.

    The quickest and easiest is to boot with a USB flash drive with bootable Windows 10 install media on it.  You can then clean the partitions from your current C: disk drive (assuming that you have nothing on that hard disk that you need) and reinstall Windows 10 without a product key.

    If your E: and F: drives are different hard disks you should be fine.  If they are just partitions of the same hard disk, then you can simply delete or format the C: partition and clean-install Windows 10 to that empty disk space after booting with the Windows 10 install media.

    Accounts become inactive if you don't sign in for more than 270 days via Web Browser.

    After an account becomes inactive, all messages, folders, and contacts are deleted.

    Incoming messages will be sent back to the sender as undeliverable.

    If you don't use your Windows Live ID for 365 days, your Windows Live ID will be permanently deleted.  

    Once the account is marked as in-active all data is purged so even if you could get the email account back the data would be gone
