tryx3 say:

William,

[You were right to dismiss the first reply as irrelevant.]

Windows is designed to work the way you are experiencing. 

  • When an Admin user logs in, it is given the same rights as a Standard user. 
  • The Admin prompts you are referring to are there so that specific processes can be given "elevated" permission [Admin permission] to proceed. 
  • You do not need to log out of anything or back in to anything, you are simply authorising that specific process [such as an installation].

If you are logged in to an Admin account, the Admin prompt looks like this and you just click on the Yes button if you want to approve the process.

If you are logged in to a Standard account, the Admin prompt looks like this and you have to select the name of an Admin account & enter its password then click on the Yes button to approve the process.  If the different users were really different people, an Admin account holder would have to walk over to the computer to enter the info.

Admin prompts appear when the process being run needs to alter something that affects other users. 

You can alter the level slightly without endangering your computer - see Control panel, User accounts, Change user account control settings [UAC].

  • As you will see there, you can have UAC at its highest level to include changes to Windows settings triggering the Admin prompt routine or you can have it at its second highest level [default] if you are not fussed about changes to Windows settings.
  • The two lowest settings make it easier for online hackers to get into your computer and start running their own installations without you even noticing.
  • I would urge you never to be connected to the internet if you had UAC at its lowest setting, which is turned off. 

It is the whole set of security improvements that have established Windows 10 as a pretty secure OS but UAC is a significant part of it [even though it was actually invented just for the convenience of office admins [who got fed up having to log into their colleagues' computers every time something needed to be set up].

Denis

tryx3 say:

Tori,

You have not found the answer.  There are serious ramifications of what you are doing that your linked webpage does not bother to explain.

The Built-In Admin account you have enabled is a recovery tool and cannot be used for day-to-day computing. 

  • Unlike its equivalent in Windows 7, the Windows 10 Built-In Admin account has its capabilities deliberately restricted.
  • If you keep using it for day-to-day computing then it is exposed to the risk of user profile corruption which would stop it being able to act in its recovery role.
  • Everything you need to do can be done in an Admin account of your own creation but that is not true of the Built-In Admin account.

Denis

andre da costa say:
Windows 10 came preinstalled on my computer, how do I find the product key?
Press Windows key + X
Click Windows PowerShell (admin)

Enter the following command:
wmic path SoftwareLicensingService get OA3xOriginalProductKey
Hit Enter
The product key will be revealed, copy the product key then enter it

If its an OEM system builder license, use the product key from the package.

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

If you are unable to activate Windows 10 after all the above and you are certain you have the correct edition installed, then please attempt phone activation:

Activate Your Windows 10 License via Microsoft Chat Support

Corrupted user profile in windows 7,I am unable to copy corrupted files to a new profile

This is the advice given to create a new user profile:

To create a new user profile

To create a new profile, you must first create a new user account. When the account is created, a profile will also be created.

  1. Open User Accounts by clicking the Start button , clicking Control Panel, clicking User Accounts and Family Safety (or clicking User Accounts, if you are connected to a network domain), and then clicking User Accounts, if you are connected to a network domain), and then clicking User Accounts.

    User Accounts.

  2. Click Manage another account.  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

  3. Click Create a new account.

  4. Type the name you want to give the user account, click an account type, and then click Create Account.

    Create Account.

To copy files to the new user profile

After you create the profile, you can copy the files from the existing profile. You must have at least three user accounts on the computer to complete these steps, including the new account you just created.

  1. Log on as a user other than the new user you just created or the user that you want to copy files from.

  2. Open Documents by clicking the Start button , and then clicking Documents.

  3. Click the Tools menu, and then click Folder Options.

    If you don't see the Tools menu, press ALT.

  4. Click the View tab, click Show hidden files and folders, clear the Hide protected operating system files check box, and then click OK.

    OK.

  5. Locate the C:\Users\Old_Username folder, where C is the drive that C is the drive that Windows is installed, and Old_Username is the name of the profile you want to copy files from.

  6. Select all of the files and folders in this folder, except the following files:

    • Ntuser.dat

    • Ntuser.dat.log

    • Ntuser.ini

  7. Click the Edit menu, and then click Copy.

    If you don't see the Edit menu, press ALT.

  8. Locate the C:\Users\New_Username folder, where C is the drive that C is the drive that Windows is installed, and New_Username is the name of the new user profile you created.

  9. Click the Edit menu, and then click Paste.

    If you don't see the Edit menu, press ALT.

  10. Log off, and then log back on as the new user.
