scottgem say:
First, lets get some terms straight. There are two basic types of LANs; Server based and Peer to Peer. In a server based LAN you have a separate computer that functions as a domain controller. With only 3 PCs you probably have a peer to peer. This uses the Workgroup feature of Windows to allow communications between the 3 PCs.

While you can certainly use any NAS as a file server on a peer to peer LAN, it is not necessary to do so. The advantage of using a NAS is that every PC can map the NAS as a the same drive letter. But you can use any one of the PCs as your file server, you just have to use UNC coding (\\computername\sharename\foldername) to link the back end. However, a NAS would be my preferred choice (I have one for my home machine). The NAS can remain on all the time. It can be easier to backup and can be addressed from every PC on the LAN.

The FBI lock screen is actually ransomware\maleware. 
Here is a guide howto remove virus with the sysinternals tools:

Autoruns - http://technet.microsoft.com/en-us/sysinternals/bb963902
 

You are looking for process that have: ( taken from Zero Day malware cleaning with the sysinternal Tools pdf )

- no icon
- have no description or company name
- unsiged Microsoft images
- live in Windows directory or user profile
- are packed
- inlcude strange URLS in their strings
- have open TCP/IP endpoints
- host suspicisous DLLS or services

Howto do an manually cleaning of maleware
 
Safe Mode With Command Promt:

1 ) Boot into Safe mode with command promt. In the command promt window type "explorer.exe" to get the start menu back 
 
 
Process Explorer:

1 ) Start Process Explorer ( as administrator CTRL+SHIFT+Enter ).  First we need to customize it a bit:

- Go to "View" --> Select Columns..In the "Process Image tab, turn on "Verified signer" and press "ok"

2 ) Go to "Options" and choose: "Verify Image signatures". In the "Verified signer column" you will se wich process that are signed ( it means that the process are legitimate )

- You are looking for process that have no icon, are not verified, missing description, company name and have the colour purple. 

Side note: the reason why you are looking for these things are that virus\maleware writers are lazy, they dont fill out company name,description etc. A very important thing to remember
is the colour "purple". Processes that have the purple colour means that the process are encrypted\packed, virus writers usually do this to avoid antivirus detection

- So hopefully by now you have spotted a process that are purple and are not verified! Right click on the suspect process and choose "suspend"

Side note: If you are choosing "Kill Process", the virus\maleware have a buddy system that will just review\respawn it.  Therefore you choose "suspsend"
 
Autoruns:

1 ) Open up Autoruns ( as administrator CTRL+SHIFT+ENTER ). First we need to customize it a bit:

- Go to "Options". Then "Filter options" and enable the "Verify code signatures" and "Hide Microsoft entries". Now you will only see the 3.part that are starting with Windows

Go to the "Logon tab" in autoruns. You are looking for the same characteristics as you did with process explorer ( No icon, missing describtion and publisher  ). 
Remember that the process are hiding in either Windows directory or appdata ( Example: C:\Users\USER NAME\AppData ).

When you have found the prosess that are matching some of the characteristics from above, just uncheck the process. Then restart Windows, hopefully you will not the see the FBI lockscreen anymore
 
The Aftermath:
 
- Download the latest antivirus definitions and rescan Your computer. Let the antivirus clean the remanings of the maleware.
