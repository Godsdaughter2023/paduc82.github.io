bob jones aka: cybertaz say:

OK, you've merged cells horizontally rather than vertically.

In that case, first select the data range beginning with that 'Subtitle' row (without including the row above it). Then click the Filter button or go to Data> Filter.

A suggestion: Rather than using Merge Cells for the Title, select those same cells, then go to Format> Cells - Alignment & use Center Across Selection from the Horizontal popup. It provides the same visual effect but is much more stable than merging cells & won't interfere with Filtering.
