rick rothstein, mvp - excel say:

Where can I find the number associated with a color? I need to change the cell color to yellow or any other color. At the red ? marks I had # 6 but give me black I though yellow was # 6  

 Master.Range(Cells(ingl11, ingl10), Cells(ingl13, ingl12)).Interior.Color =?? ‘yellow 

The #6 is the number used for yellow in the ColorIndex property, not the Color property. You can get the Number/ColorIndex equivalents by running this code in the Immediate Window (or you can bundle it inside a Sub/End Sub and run it from any code window)...

For X = 1 To 56: Cells(X, "A").Interior.ColorIndex = X: Next

What it will happen is the first 56 cells in Column A will fill with colors. Find the color you want to use... its row number is the number you should assign to the ColorIndex property. You can then clear the cell format for Column A when through to remove the colors (and any other formatting you have in that column, so you should conside activating an empty worksheet before running the code so you don't accidentally overwrite and existing cell format).
