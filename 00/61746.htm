Here is one way of displaying/printing folders and files:

1. To display the folder hierarchy, open File Explorer (was Windows Explorer), navigate to the folder you wish to start at, hold down the Shift key, right-click or press and hold on the folder name and choose Open command window here. 2. To display the files in each folder, type tree /f |clip and press Enter.
