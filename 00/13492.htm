a. user say:

Hello StuartZipper,

Microsoft won't really have a solution for this issue as it is unrelated to their products and services. DNS stands for Domain Name System and the system works as a sort of address book for everything on the internet. DNS service is usually handled by a provider of internet service. If that DNS server is blacklisted (usually due to issues with email spamming) it would return the email as undeliverable. All users trying to go through that blacklisted server would be affected even if they took no part in any email spamming.

Messages would not be able to be sent until the blacklist was removed. The barracudacentral.org website that was listed there has a searchable database as well as more information about other blacklisted servers.

Microsoft doesn't have any access to any of these DNS servers and further inquiries should be directed to the Internet service provider responsible for the DNS sever in question for any additional information.
