If you find that text is too small in Windows Live Mail, you should consider increasing the screen DPI. This will make text and other elements larger and easier to see without affecting images and other size attributes. Go to your "Task Manager" and close both SkyDrive services.


Next, go to Windows Explorer and navigate to:

C:\Users\%USER%\AppData\Local\Microsoft\Windows\SkyDrive


Delete the "SkyDrive" folder after backing up your SkyDrive date elsewhere. (just encase.  This is just a settings folder but any corrupted files will go missing after this.


Do this t all computers using the same account.  Takes a few minutes or so depending on your account size.  It will start with nothing then quickly fill in with all your files from the cloud.


Note: %USER% is replaced with your windows login; you will recognize I when you get to the user folder area i.e. C:\Users\
