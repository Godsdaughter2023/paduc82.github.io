How To Detect A Cell Got the Focus? - Microsoft Community

g70115 say:

How To Detect A Cell Got the Focus?

I am using the change event to detect the fact that my cell was changed.

 

How can I see the contain of the selected cell before the change occur ?   Is there an event in excel that will trigger a VBA code when I just select a cell ???

 

This way I can save the contain before changes.

 

Thank You.

 

I am using Excel 2003 current with VBA code.

hansv mvp say:

If you want to keep track of a single cell, you could use a static variable in the Worksheet_Change event procedure:

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Static oldValue As Variant
    Dim newValue As Variant
    If Not Intersect(Range("B2"), Target) Is Nothing Then
        newValue = Range("B2").Value
        MsgBox "Old=" & oldValue & ", New=" & newValue
        oldValue = newValue
    End If
End Sub

 

Alternatively, you can use the Worksheet_SelectionChange event - this occurs when the user moves to a cell:

 

' This declaration should be above all Subs and Functions
Private oldValue As Variant

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim newValue As Variant
    If Not Intersect(Range("B2"), Target) Is Nothing Then
        newValue = Range("B2").Value
        MsgBox "Old=" & oldValue & ", new=" & newValue
        oldValue = newValue
    End If
End Sub

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Not Intersect(Range("B2"), Target) Is Nothing Then
        oldValue = Range("B2").Value
    End If
End Sub

nokkika say:

No, COM4 is not locked, except when the **bleep** Ovi Maps Gadget is running!!!

Close the Ovi Map Gadget on the desktop, make sure Google Maps is not running, and no other GPS software is running.

 

Then open TeraTerm on COM5 and turn off the GPS (AT_OGPS=0).

 

Run the Driver exe (gps7s32.exe), set GPS Port as COM4, Baud Rate = 4800,Click on Install, confirm you want to add the Sensor, then leave this Window Open.

 

Open Teraterm on COM5 and turn the GPS back on (AT_OGPS=2)

 

Switch back to the Driver Window that you left open, and you should now see NMEA data spewing out, and eventually a Fix.  The API Sensor is now active.

 

If you want to test it out on the Turbo GPS app, then install it, open it up, Go to Tools, GPS, GPS Setup, tick the API Sensor (no need to change COM ports or baud rate here cos the API Sensor has taken over), then click on Tools, GPS, Connect and it should Connect and give you fix data etc.
