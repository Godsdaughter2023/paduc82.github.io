steve rindsberg say:

For starters, don't save as PPT unless you've first saved as PPTX.  PowerPoint 2010 and onward will save narrations as EMBEDDED MP3 sounds.  The PPT format doesn't allow for embedded MP3s so you're liable to lose the sound.

But let's look at something else:  I'm assuming that by "BB" you mean Blackboard.  How does it display PowerPoint presentations that you've uploaded?  If it converts them to web pages, the software may not have any way of accessing the embedded MP3 sounds (whereas it used to be fairly simple to access the embedded or linked sound files in earlier versions of PowerPoint). 

Can up upload the presentation as a PPTX file that students can simply download and play on their own PCs/Macs? 

joanne k say:

4courseload say:

john korchok say:

I realize this answer is too late for your deck, but I'm providing an answer for other users who want to recover a narration:

  1. Make a copy of the presentation to a separate folder.
  2. Change the file ending to .zip. Click on Yes when Windows asks if you really want to do that.
  3. Right-click on the zip file and choose Expand All.
  4. Open the ppt/media folder. Your narration files will be in there, one for each slide. They'll be named media1.wav, media2.wav, etc.
