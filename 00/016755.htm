gord dibben say:

This revision does not add extra rows.

Change any or all the numbers in column B then run the macro.

Sub InsertRows_Revised()
    Dim LastNumber As Long: LastNumber = ActiveSheet.Range("B" & Rows.Count).End(xlUp).Row
    Dim LastRow As Long:  LastRow = ActiveSheet.Cells(Rows.Count, 1).End(xlUp).Row
    For r = LastRow To 1 Step -1
        If Application.CountA(Rows(r)) = 0 Then
        Application.ScreenUpdating = False
            Rows(r).Delete
        End If
    Next r
    While LastNumber >= 2
        If Not IsEmpty(ActiveSheet.Range("B" & LastNumber)) Then
            Dim NewRows As Long: NewRows = ActiveSheet.Range("B" & LastNumber).Value
            Dim InsertIndex As Long: InsertIndex = 1
            While InsertIndex <= NewRows
                ActiveSheet.Range("B" & LastNumber + 1).EntireRow.Insert
                InsertIndex = InsertIndex + 1
            Wend
        End If
        LastNumber = LastNumber - 1
    Wend
    Application.ScreenUpdating = True
End Sub

Cadence Release timeframe of the Windows Insider Preview builds - Microsoft Community

meghraj_sq say:

Cadence Release timeframe of the Windows Insider Preview builds

Hi,

Can we get to know the cadence release timeframe of the Windows insider Preview builds of Window Server 2019 and Windows 10 or Microsoft office 2019 ?

Thanks!

cbarnhorst say:

In general, the WIP team's practice is to flight a new Windows 10 build once a week for Fast Ring Active Development and once a month for Slow Ring. 

Having said that, the WIP team flights builds based on the suitability of candidate builds and the stage of the development cycle.  Right now Fast Ring builds for RS5 are flighting more frequently than once a week because the team is fine-tuning RS5 for release in a couple of months or so. 

There are frequent interruptions to the flighting patterns due to blocking bugs so do not be surprised if the team goes as long as two weeks without flighting a build in Fast Ring.  Slow Ring has been very unpredictable.

ISO files are only posted following a Slow Ring flight.  In theory, each Slow Ring build is posted but clean install bugs have prevented ISO files from being posted a couple of times.  It is sometimes months between new ISO files.

Skip Ahead builds are only flighted for the first couple of months of a new cycle (right now 19H1).

Windows Server 2019 seems to be on a much less frequent cadence, perhaps once a month.  Office Insider updates appear to be frequent, as ready and is in two tracks;  the Monthly Channel and the Office 365 Channel.
