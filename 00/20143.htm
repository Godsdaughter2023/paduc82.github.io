To count the number of rows which are used (ie. not blank) in a sheet, you may try below:

Sub usedrows1()
Dim i As Long
For Each Row In ActiveSheet.UsedRange.Rows
    If Application.WorksheetFunction.CountA(Row) > 0 Then i = i + 1
Next
MsgBox i
End Sub

Note: this does not count the last row in a sheet but the NUMBER of rows which are used.

After testing many permutations, this is what I have learned.

The problem is not simply that Outlook must be restarted.

If Outlook 2007 is open and has either the IMAP or the POP3 Inbox folders highlighted, incoming messages to the IMAP account will not be received.  To receive messages, either Outlook must must be restarted, or one of the other IMAP folders must be highlighted followed by highlighting the IMAP Inbox.  

If any of the other IMAP folders are highlighted, e.g. Spam, Sent, Trash, messages to the IMAP account will be received. 

In summary, for those who are having a similar problem:

1.  Open Outlook 2007, which in my case highlights the Pop3 Inbox folder, which does not allow reception of new messages.

2. Highlight any folder other than the IMAP Inbox or the Pop3 Inbox.  Then messages to the IMAP account should be received normally.  Also, the envelope icon indicating new mail will appear on the task bar.
