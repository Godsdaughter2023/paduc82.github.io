bob jones aka: cybertaz say:
At the end of the first document's content create a Section Break (Next Page), then use Insert>File to insert the second document. At the end of it create another Section Break (Next Page) then insert the third document, repeat as necessary.

Another option is to go into Outline View in the first document, then click the Master Document Toolbar. At the end of that document's content click the Insert (not Create) Subdocument button. Select the second document & repeat as necessary.

Suggestion: It's best to have all of the individual documents in the same folder, regardless of which method you use. If you use the Master Document method be sure to do any additional editing in the subdocument, itself, rather than in the Master. See Word Help on Section Breaks & related topics. Info on using Master documents safely is in the PDF available from the link on this page: http://www.techwr-l.com/articles/general/masterdocs

How much cleanup you'll have to do will be determined by the manner in which each document is constructed.

NOTE: The Tools> Merge Documents feature is for combining the content of 2 separate copies of the same document in order to have the differences marked up as Tracked Changes.

qa guy commelec say: You need 2 tables - Assets and Moves.  
Assets must have a primary key field - an autonumber works very well.

Moves would have a number, long integer field as foreign key.  Have a datetime field to indicate the latest move.  

Use a form/subform (Assets/Moves) with the Master/Child links set using the Assets primary key and Moves foreign key fields.

steve rindsberg say:

You need to delete objects in a collection starting at the last and go backwards.  Otherwise you find the first of, say, ten, delete it, then move on to the second, but since you've deleted the first one, the second one is now the first and the third is second; it gets deleted, leaving the original second item (now the new first) behind. 

Dim x as Long

With oStory

For x = .Hyperlinks.Count to 1 step -1

    .Hyperlinks(x).Delete

Next  ' x

End With   ' oStory
