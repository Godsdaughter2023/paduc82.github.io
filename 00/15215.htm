karl timmermans [outlook mvp 2012-2018] say:

If you are referring to moving items from one default folder to another - from one Inbox to an Inbox in another infostore (i.e. PST file) - then you will not be able to simply "move" the folder from one to another since that option does not exist and "copying" the folder will create a new (different) folder in the target destination.

 

What you will have to do is "highlight" all the actual items from the source folder and then drag them to the destination folder. If you want to maintain a copy of the items in the original folder, hold the <shift> key down prior to starting the drag operation.

hansv mvp say:

Try using the Before Update event:

 

Private Sub TextBox1_BeforeUpdate(ByVal Cancel As MSForms.ReturnBoolean)
    Dim LastRowFilled As Long
    Dim DataArray(1 To 4) As String
    Dim i As Integer
    If Me.TextBox1 = "" Then
        Me.ListBox1.SetFocus
    Else
        DataArray(1) = ListBox1.Value
        DataArray(2) = ListBox2.Value
        DataArray(3) = TextBox1.Value
        DataArray(4) = Date
        LastRowFilled = xlLastRow + 1
        For i = 1 To 4
            Worksheets("Sheet1").Cells(LastRowFilled, i).Value = DataArray(i)
        Next i
        Me.TextBox1 = ""
        Cancel = True
    End If
End Sub
