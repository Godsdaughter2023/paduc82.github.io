My guess is that your IT person copied the data to the DVD but did not "Finalize" the DVD.  When this happens, only computers that use the same software that the original person used to write to the CD/DVD can read the DVD.  (After a DVD is "finalized", then any computer can read the DVD).

Give your DVD back to the person who made it and ask him/her to finalize the DVD.

frederik long say:

Reboot and select proper boot device or insert boot media in selected boot device and press a key.
As you suspected, you need to instruct the BIOS to treat your current disk as the primary boot device. Since there are numerous different types of BIOSs it is not possible to give you any specific advice for your particular BIOS. Why not get into it and examine the various options? It's actually fairly easy!

Splitting the disk the way you suggest is a good idea. Unfortunately Microsoft makes it hard to implement such a scheme. Here are a few options:
  • Connect the disk at the same time as your existing Windows disk, then split it using the Disk Manager diskmgmt.msc.
  • Purchase a partition manager such as Acronis DiskDirector and use it to split your disk any time you like.
  • Connect the new disk only, then boot the machine with your Windows installation DVD. Now get into Repair Mode, then the Command Prompt. Ask a computer-savvy friend to show you how to use diskpart.exe to partition and format the disk. Diskpart is a maintenance tool that is not in the least intuitive.
