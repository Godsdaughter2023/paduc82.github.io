Hi Christos,

By default, the internal user will use the Integrated Windows authentication (IWA) when sign into Office 365 using IE. If you add the related office 365 sites in the intranet settings, the user will sign into Office 365 seamlessly, that’s exactly how ADFS/SSO should work in the internal network.

For other browsers and external network, it will use Forms-based authentication (FBA), which the user will be redirected to an organization sign in page (as you can see in Chrome). Sub CreateNewSheet()
    Dim sheet_name_to_create As String
    Dim rep As Long
    Dim strOrigPrevShtName As String
    Dim strNewPrevShtName As String
    Dim rngToFind As Range
   
    'Original name used in formula is sheet name previous to the current ActiveSheet
    strOrigPrevShtName = Worksheets(ActiveSheet.Index - 1).Name
   
    sheet_name_to_create = Format(ActiveSheet.Range("N5").Value, "d-mmm")
   
    For rep = 1 To (Worksheets.Count)
        If LCase(Sheets(rep).Name) = LCase(sheet_name_to_create) Then
        MsgBox "This Sheet Already Exists!"
        Exit Sub
        End If
    Next
   
    ActiveSheet.Copy After:=ActiveSheet
    Sheets(ActiveSheet.Name).Name = sheet_name_to_create
       
    'The new previous sheet name is now the sheet previous to the new sheet
    strNewPrevShtName = Worksheets(ActiveSheet.Index - 1).Name
      
    'Find all occurrences of the original previous sheet name and
    'replace with the new previous sheet name
    With ActiveSheet.Cells
        Set rngToFind = .Find(What:=strOrigPrevShtName, _
                        LookIn:=xlFormulas, _
                        LookAt:=xlPart, _
                        SearchOrder:=xlByRows, _
                        SearchDirection:=xlNext, _
                        MatchCase:=False, _
                        SearchFormat:=False)
                       
        If Not rngToFind Is Nothing Then
            Do
                'Note the replacement is in the Formula of rngToFind; Not in value
                rngToFind.Formula = Replace(rngToFind.Formula, strOrigPrevShtName, strNewPrevShtName)
                Set rngToFind = .FindNext(rngToFind)
                'When all replacements are made then the string should no longer be found.
                If rngToFind Is Nothing Then Exit Sub
            Loop
        End If
    End With
End Sub

  • The code first tests for an actual formula in cell N5.
  • If it is a formula then it assigns the previous sheet name to the string variable strOrigPrevShtName.
  • If it is not a formula then the string variable value will remain as zero length string.
  • A new sheet name is then created from the date in cell N5 and assigned to sheet_name_to_create.
  • The new sheet is created and named with the sheet_name_to_create variable value.
  • If IstrOrigPrevShtName <> "" (ie. N5 had a formula and a previous sheet was referenced) then it replaces all of the old previous sheet names with the new previous sheet names.
  • Else it replaces the constant value in cell N5 with a formula using the new previous sheet name and the value that was copied to N5 and displays a message re the first added sheet and formulas might need to be edited.
  • If you want then you could go through all of the cells that need to be edited and use the VBA code to insert the formulas in t

    Sub CreateNewSheet()

        Dim rep As Long
        Dim strOrigPrevShtName As String
        Dim strNewPrevShtName As String
       
        'If N5 is a formula referencing the sheet previous to the current ActiveSheet
        If ActiveSheet.Range("N5").HasFormula Then  'Should only have formula if more than one sheet.
            strOrigPrevShtName = Worksheets(ActiveSheet.Index - 1).Name
        End If
       
        sheet_name_to_create = Format(ActiveSheet.Range("N5").Value, "d-mmm")
           
        For rep = 1 To (Worksheets.Count)
            If LCase(Sheets(rep).Name) = LCase(sheet_name_to_create) Then
            MsgBox "This Sheet Already Exists!"
            Exit Sub
            End If
        Next
           
        ActiveSheet.Copy After:=ActiveSheet
        Sheets(ActiveSheet.Name).Name = sheet_name_to_create
           
        'The new previous sheet name is now the sheet previous to the new sheet
        strNewPrevShtName = Worksheets(ActiveSheet.Index - 1).Name
          
        'Find all occurrences of the original previous sheet name and
        'replace with the new previous sheet name
        If strOrigPrevShtName <> "" Then
            'Only run this code if a previous sheet is referenced in cell N5
            With ActiveSheet.Cells
                .Replace What:=strOrigPrevShtName, _
                        Replacement:=strNewPrevShtName, _
                        LookAt:=xlPart, _
                        SearchOrder:=xlByRows, _
                        MatchCase:=False, _
                        SearchFormat:=False, _
                        ReplaceFormat:=False
            End With
        Else
            'Insert the formula in cell N5 of the added worksheet to replace the constant copied
            ActiveSheet.Range("N5").Formula = "='" & strNewPrevShtName & "'!N5+1"
           
            'The MsgBox is optional to display the message when cell N5 is a constant
