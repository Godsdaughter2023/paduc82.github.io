Microsoft welcome screens for Word, Excel, Powerpoint, etc. - Microsoft Community

kenardouin say:

Microsoft welcome screens for Word, Excel, Powerpoint, etc.

How do I remove the welcome screens for Word, Excel, Powerpoint and Outlook?
daniel g. grau say:
This is the procedure for Word:

  1. Go to Applications > Microsoft Office 2011
  2. Right-click (or Control-click) on Microsoft Word and select Compress
    • This will ensure you have an original version in case something goes wrong
  3. Right-click (or Control-click) on Microsoft Word and select Show package contents
  4. Navigate to folder Contents/Resources/en.lproj
    • "en" if for "English". If your copy runs in another language, the folder will have the localized initials of the language. For instance, "es.lproj" for Spanish and "fr.lproj" is for French.
  5. Locate file Splash.png and delete it
  6. Close all folders and test by opening Word
  7. If you notice any problem, this method no longer works
    • Trash the modified copy of Word
    • Double-click on the .zip file to restore the original version
  8. If everything works OK
    • Keep the .zip file around, in case you ever need it (for instance, if a software update fails)

Repeat the procedure for the other titles.

Note that deleting the Welcome screen won't gain you any startup time, as Word does some stuff while the screen is being displayed, like checking fonts, loading your templates, setting preferences, etc. In fact, it's good to have it shown, because when it's gone, it indicates that Word is ready to be used.

Here is an array** formula which will do it assuming no blanks and no text

=OFFSET(A1:A5000,MATCH(FALSE,(A2:A5001=A1:A15000+1),0)-1,0)+1

**Array: you enter the formula by pressing Shift+Ctrl+Enter not by press Enter

This will work without sorting the values:

a. user say:

Hi,

Thank you for posting your query on Microsoft Community.

Kindly reply: 

What happens when you try to delete junk from your email? Kindly mention the error message you receive (if any)

 

Your email address might have been spoofed, therefore, it not necessarily mean that the emails were sent by you. You can use your favorite search engine to learn more about spoofing.

For now, let's try changing your Junk filter settings to Exclusive, this setting will only allow emails from your contacts list or safe sender's list to move to Inbox, rest all the emails will be moved to the Junk folder. Therefore, make sure you add all the important domains to the safe senders list. 

Follow the steps below:

Login to https://outlook.com/ through your Microsoft Account.

If you use Outlook.com interface, then follow the steps below.  

1. Click on the Gear (Settings) icon and select Options.

2. Under Preventing Junk Mail, select filters and reporting.

3. Select Exclusive to change the filter level and then select Save.

If you use Outlook.com (Preview) interface, then follow the steps below: 

1. Click on the Gear (Settings) icon and select Options.

2. Expand Mail > Junk Mail, click on filters and reporting.

3. Select Exclusive to change the filter level and then select Save.

Hope this helps. Please respond if you have further related queries.

Regards,

Jesinta Rozario
