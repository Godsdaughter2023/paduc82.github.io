Luis,

The absence of an entry for your WiFi card in Device manager, Network adapters implies that there is probably a hardware fault with the adapter.

As a final test, go to your computer maker's website [or WiFi card maker's website if you bought it separately].  Download the driver for your WiFi card and install it.  If the WiFi card still fails to appear in Device manager, Network adapters then I think it has definitely failed.

Even WiFi cards in laptops can be replaced. 

1  The procedure might not be shown in your user manual so check the computer maker's website & any user forums they host for advice [Dell, just by way of example, provided detailed instructions for changing the WiFi card in the user guide for one of my laptops whilst for the other one Dell's website had a "service manual" with the necessary information].
