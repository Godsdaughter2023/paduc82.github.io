mike h.. say:

Hi,

Put this in a helper column and fill down and it returns the decimal places for each number which you can use to sort.

=IF(A1=INT(A1),0,LEN(MID(A1-INT(A1),FIND(".",A1,1),LEN(A1)-FIND(".",A1,1))))

Note it returns the real number of places even if you have formatted the cell to show less.
