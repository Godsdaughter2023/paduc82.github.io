How to delete user profiles - Microsoft Community

yvalsondronkers say:

How to delete user profiles

i have a tablet (Acer iconia tab 8 W1-810) 

but im selling it to someone but i don't know how to delete my user profile so that guy can make a new one

a. user say:

Hi,

Welcome to Microsoft Community.

As per the query, I understand that you want to delete a user account in Windows 8.1. I appreciate your efforts to resolve this issue. I will surely help you.

To know the current user profile logged in, you may press Windows Key + X, select Control Panel and select User Accounts.

Once you are in user account page, you can find the active user account. Check the screen shot.

 To delete a user account try these steps:

Note: Back up all the data before deleting the user account.

  1. Login to your      user account which you want to keep.
  2. Press Windows      key + X, select “Control Panel”      and “User accounts”.
  3. Click on “Manage another account” and then      select the user account that you want to delete.
  4. Click on “delete the account”.
  5. Depending on      whether you want to “Keep the files”      or “delete the files” choose      the appropriate option.
  6. You can delete      the other accounts similarly.

Note: Once deleted, the user accounts cannot be recovered. Make sure you back up any data if required.

If you want to copy and paste the data files to a different user profile, then try the following steps.

Follow the steps to copy the files from the old Microsoft account to the new Microsoft account:

a) Press/tap Windows and x together and select Windows Explorer.

b) Click/tap on View tab and click/tap on Options at the top right corner.

c) Click/tap the View tab, and then click/tap Show hidden files, folders, and drives.

d) Uncheck the box which states Hide protected operating system files.

e) Click Yes to confirm, and then click OK.

F) Locate the C:\Users\Old_Username folder, where C is the drive that Windows is installed on, and Old Username is the name of the profile you want to copy files from.

g) Select all of the files and folders in this folder, except the following files:

•Ntuser.dat

•Ntuser.dat.log

•Ntuser.ini

h) Right click/tap and Select Copy.

i) Locate the C:\Users\New_Username folder, where C is the drive that Windows is installed on, and New Username is the name of the new user profile you created.

j) Right click/double tap inside the folder and select Paste.

k) Restart your PC and check if you are able to log in to the computer using the new Microsoft account with all the files of the old account.

Let us know the status of the issue. If you need further assistance, please post back. We will be glad to help you.

Thank you.
