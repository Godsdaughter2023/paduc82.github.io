Method 1;

 

Run the Search and Indexing troubleshooter. Follow the steps mentioned below.

b. Type “Troubleshooting” without quotes in the Search box and hit enter.

c. Select “View all” option from the left panel.

d. Select “Search and Indexing”.

e. Follow the on-screen instructions to run the troubleshooter.

 

Method 2:

Turn off and turn on Windows Search, this might help you.

a. Press the Windows Key + R on your keyboard to open the Run dialog.

b. Enter services.msc and click OK.

c. The Services window appears. Scroll down, right click on Windows Search and select Properties.

d. Click on stop, save the changes.

e. Restart the computer.

Note: Refer to the above mentioned steps to turn on the Windows Search.
