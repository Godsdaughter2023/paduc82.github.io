Internet Explorer 11 on Windows 10 homepage buttons greyed out - - Microsoft Community

paulkinney say:

Internet Explorer 11 on Windows 10 homepage buttons greyed out - Regedit Solution

While searching for an answer to this question, I found the same question posted on the MS Community at this link, but was locked from further replies.

https://answers.microsoft.com/en-us/ie/forum/ie11-iewindows_10/internet-explorer-homepage-buttons-greyed-out/b25370a6-7741-430d-b8b6-e367d5319e76

Symptom Summary:  In Internet Explorer, select Tools->internet options menu option.  Under General tab, the three home page setting buttons (Use current, Use default, Use new tab) are all greyed out with the caption at bottom of dialog "Some setting are managed by your system administrator."

I then took to searching the Registry via Regedit, then found the Registry key that enables/disables the Home Page setting buttons.  Note, you must have administrator privileges on your system to edit any Registry key values.  I am sure Microsoft also would like me to state that directly editing Registry values could result in stability issues, so you do so at your own risk.  With the disclaimer blurb delivered, here is what worked for me.

  1. Click the Start (Windows icon) button, located on the task bar (usually at the bottom of the screen).
  2. Just type regedit and press Enter.
  3. In the entry field, just under the File/Edit/View/Favorite/Help menu choices, enter the following, then press Enter to display the Registry key values:
    Computer\HKEY_CURRENT_USER\Software\Policies\Microsoft\Internet Explorer\Control Panel
  4. On the right pane, find the HomePage key. 
  5. If the HomePage Data (value) is set to 0x0000001 (1), then double-click on HomePage then change the Value data from 1 to 0, then press OK. 
  6. Open a new instance of Internet Explorer, then select the Tools->internet options menu option.  You should (hopefully) see the Home Page setting buttons enabled (not greyed out).

Hope this helps others.
