Hi Don,


This can be caused by a software compatibility issue, or due to a malware or virus infection. Run time Error 339 occurs if the software was released without sufficient run time files.

Meanwhile, kindly make sure if the software is compatible with Windows 10. You can run the System File Checker and DISM tool. System File Checker (SFC) is a utility in Windows that scans for corrupted Windows system files. If it finds a problem, it will try to replace the problematic system files from a cached version that’s on your PC. If SFC can’t replace the files, the Deployment Image Service and Management Tool (DISM) can check the corrupted files then download and replace the files through Windows Update.

  1. Press Windows key + X on your keyboard.
  2. Choose Command Prompt (Admin).
  3. Copy or type each command followed by hitting Enter.
    sfc /scannow
    dism /online /cleanup-image /restorehealth

Running a full scan of Windows Defender can help. This is to check any malicious software or adware that might be causing the issue. You can check this article on how to protect your computer using Windows Defender.
