hansv mvp say:

I'm not sure it's a good idea to do this in a formula - it is very calculation-intensive. But here is a custom VBA function you could use:

 

Function MostFrequentWord(rng As Range) As String
    Dim cel As Range
    Dim arr As Variant
    Dim i As Long
    Dim wrd As String
    Dim col As Object
    Dim itm As Variant
    Dim hi As Long
    Set col = CreateObject("Scripting.Dictionary")
    For Each cel In rng
        arr = Split(cel.Value)
        For i = 0 To UBound(arr)
            wrd = arr(i)
            If col.Exists(wrd) Then
                col(wrd) = col(wrd) + 1
            Else
                col.Add Item:=1, Key:=wrd
            End If
        Next i
    Next cel
    For Each itm In col.Keys
        If col(itm) > hi Then
            hi = col(itm)
            wrd = itm
        End If
    Next itm
    MostFrequentWord = wrd
End Function

 

Use like this:

 

=MostFrequentWord(A2:B2)

 

If there are multiple most-frequent words, the function will return the first of these that it encounters.

Option Explicit

 

Function MostFrequentWord(Rng As Range)
    Dim strInput As String, strOutput As String
    Dim c As Range
    Dim dbMax As Double
    Dim vArr As Variant, vWord As Variant
   
     'Build a single string of all the words in the range:
    For Each c In Rng
        strInput = strInput & " " & c.Value
    Next c
   
     'Remove any punctuation and double spaces:
    strInput = ReplacePunctuationWithSpace(strInput)
    strInput = RemoveDoubleSpaces(strInput)
   
     'Split into an array of words:
    vArr = Split(strInput, " ")
   
     'Create a dictionary of all the words:
    With CreateObject("scripting.dictionary")
        .CompareMode = vbTextCompare
        For Each vWord In vArr
            If Not .Exists(vWord) Then
                .Add vWord, 1
            Else
                .Item(vWord) = .Item(vWord) + 1
            End If
        Next vWord
        
         'Find the max occurance:
        dbMax = Application.Max(Application.Transpose(.Items))
       
         'Build a list of the word(s) which have appeared dbMax times:
        For Each vWord In .keys
            If .Item(vWord) = dbMax Then
                strOutput = strOutput & "'" & vWord & "' & "
            End If
        Next vWord
       
         'Remove the last " & " from the list string:
        strOutput = Left(strOutput, Len(strOutput) - 3)
       
         'Output the result:
        MostFrequentWord = strOutput & " appeared " & dbMax & " times"
    End With
End Function

 

Function ReplacePunctuationWithSpace(strInput As String) As String
    Dim regex As Object
    Set regex = CreateObject("VBScript.RegExp")
    regex.Global = True
    regex.Pattern = "[^ 0-9A-Za-z]"
    ReplacePunctuationWithSpace = regex.Replace(strInput, " ")
End Function
 

Function RemoveDoubleSpaces(strInput As String)
    Dim regex As Object
    Set regex = CreateObject("VBScript.RegExp")
    regex.Global = True
    regex.Pattern = "\s{2,}"
    RemoveDoubleSpaces = regex.Replace(strInput, " ")
End Function

 

In Excel, you would do something like   =MostFrequentWord(A2:B2)

This UDF will return something like:

 

    'men' appeared 2 times

 

and if more than one word appears twice, it would return:

 

    'men' & 'women' appeared 2 times

 

Note also that this code will remove any punctuation, so "men," will be treated as "men"

jeeped say:
You can use the IFERROR() function, introduced in Excel 2007. This avoids having to repeat the formula once for an error check and again for an actual result. IFERROR() simply wraps around tour existing formula and you supply the desired result in the case that there is no valid result.

=IFERROR(INDEX(Table7[Batch Ref '#],MATCH(Sheet1!F5,Table7[Life Left],0)),"")

If you need to provide compatibility to Excel 2003, you will have to double up the MATCH() portion of your formula to check for #N/A.

=IF(ISERROR(MATCH(Sheet1!F5,Table7[Life Left],0)),"",INDEX(Table7[Batch Ref '#],MATCH(Sheet1!F5,Table7[Life Left],0)))
... or,
=IF(ISNA(MATCH(Sheet1!F5,Table7[Life Left],0)),0,INDEX(Table7[Batch Ref '#],MATCH(Sheet1!F5,Table7[Life Left],0)))

Try this code in sheet module of the Index sheet.

Also includes a goto index sheet link on every sheet.

Index changes as sheets are added or removed from workbook.

Private Sub Worksheet_Activate()
Dim wSheet As Worksheet
Dim l As Long

l = 1

    With Me
        .Columns(1).ClearContents
        .Cells(1, 1) = "INDEX"
        .Cells(1, 1).Name = "Index"
    End With
   
    For Each wSheet In Worksheets
        If wSheet.Name <> Me.Name Then
            l = l + 1
                With wSheet
                    .Range("A1").Name = "Start_" & wSheet.Index
                    .Hyperlinks.Add Anchor:=.Range("A1"), Address:="", _
                    SubAddress:="Index", TextToDisplay:="Back to Index"
                End With

                Me.Hyperlinks.Add Anchor:=Me.Cells(l, 1), Address:="", _
                SubAddress:="Start_" & wSheet.Index, TextToDisplay:=wSheet.Name
        End If
    Next wSheet

End Sub

I don't use 2010 but in eatlier versions you cannot simply copy a checkbox control and have the cell link change to the new location.  You must copy then manually change the link or build an array of CB's with code.

 

Sample code from Dave Peterson.

 

Sub CellCheckbox()
    Dim myCell As Range
    Dim myRng As Range
    Dim CBX As CheckBox


    With ActiveSheet
        .CheckBoxes.Delete 'nice for testing!
        Set myRng = .Range("A1:A10")  'change to the range you want
    End With


    For Each myCell In myRng.Cells
        With myCell
            Set CBX = .Parent.CheckBoxes.Add _
                        (Top:=.Top, _
                         Left:=.Left, _
                         Width:=.Width, _
                         Height:=.Height)
            CBX.Name = "Allan_" & .Address(0, 0)
            CBX.Caption = "" 'or what you want
            CBX.Value = xlOff


            'I like to use the same cell as the linked cell
            'with a number format of ;;;
            'I can't see the true/false in the worksheet, but it
            'still appears in the formula bar
            CBX.LinkedCell = .Address(external:=True)
            .NumberFormat = ";;;"
        End With
    Next myCell
End Sub
