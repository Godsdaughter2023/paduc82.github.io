Find what: ^13(  DGMAIN)
Replace with: ^m\1

Can Word search for whitespace? (Before someone suggests using a simple [non-wildcard] find "^p  DGMAIN" and replace with "^m  DGMAIN", I should note that I've simplified my search for the sake of brevity here, but considering how I want to search and that I want to use this in a macro, I _do_ need to use a wildcard search.)

Any suggestions? TIA.

doug robbins - mvp office apps & services (word) say:

Use a macro containing the following code:

 

Dim i As Long
With ActiveDocument
    For i = 1 To .Paragraphs.Count
        With .Paragraphs(i)
            If Left(.Range.Text = "  DGMAIN", 8) Then
                .PageBreakBefore = True
            End If
        End With
    Next i
End With

Note: There are two spaces before DGMAIN in the above.

macropod say:

Try:

 

Sub Demo()
Application.ScreenUpdating = False
With ActiveDocument.Content.Find
  .ClearFormatting
  .Replacement.ClearFormatting
  .Text = "^13  (DGMAIN[!^13]{1,}^13)"
  .Replacement.Text = "^12\1"
  .Forward = True
  .Wrap = wdFindContinue
  .Format = False
  .MatchWildcards = True
  .Execute Replace:=wdReplaceAll
End With
Application.ScreenUpdating = True
End Sub

 

Note: The above removes the two spaces before DGMAIN. If you want to keep them move the '(' two spaces forward.

There are two different ways to buy Office:
