Hi Mr_Humidty,

Thank you for taking the time to post.

The below method is a more advanced way to cleanup phantom/ghosted (Non-present) devices from your computer, it is similar to what you tried “Deleting USB root hubs and controllers and allowing Windows to re-install them.” but different.

Please be sure to backup any critical data and create a system restore point before following the steps below.

 

·         Click the Start button on the Desktop

·         In the Search field type cmd (do not hit enter)

·         Right-click the cmd.exe and select Run as Administrator

·          Once in the command prompt copy and paste into the command prompt window set devmgr_show_nonpresent_devices=1 and hit enter

·          Next type start devmgmt.msc and hit enter, Device manager should open

·          Once in Device manager, select the View drop down and choose show hidden devices

What you need to do here is clean up the ghosted (Non-present) devices from here, you should see some as “unknown” and the others will have a grayed out look to the icon to the left of them.

Hi Rohit,

Thank you for posting your query in Microsoft Community.

We can try the following methods and check the status.

I would suggest to make sure if you have turned on Bluetooth on other devices.

 

Method 1: I would also suggest you to run the Hardware and devices troubleshooter to troubleshoot the device:

a. Press Windows key to launch the Start screen.

b. Type “Troubleshooting” without quotes and click on Troubleshooting.

c. In the new window, click View all on the left pane.

d. In the list of options, click on Hardware and devices.

e. Click Next and follow the on-screen instructions.

 

Method 2: I would also suggest you to start the Bluetooth service and set it to Automatic:

a. Go to the Start screen.

b. Type “Services.msc” without quotes and press Enter.

c. Right click on Bluetooth support service and click on Properties.

d. Set the Startup type as Automatic and then click Start under Service status.

e. Click on OK.

 

Also, refer to: Why isn't Windows finding my Bluetooth or other wireless device?
