wunders say:

Yes. Possible.

 

Control Panel -> Folder Options

then click on the "File Types" tab at the top.  Wait for window to populate.

then scroll down in the window and find the file extension you are interested in.

then single-click to select that file extension and then click the "Advanced" button.

an "Edit File Type" window will then open.  At the top-middle, next to the icon, you can change the file type to anything that you want.

Then "OK" your way out.
