check marks on double click in multiple columns? - Microsoft Community

meenakshil say:

check marks on double click in multiple columns?

how to get check marks in excel file on double click in multiple columns?
amit tandon say:

Enter below code in the worksheet module - in the module of the worksheet wherein you wish to place the tick marks. In the macro, double click action is valid for column A & will put a check mark in 3 columns so taht if the double click is on A10, then cells A10:C10 will get the tick marks. This can be altered per your requirement.

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)

Dim rng As Range
On Error GoTo ErrHandler
Application.DisplayAlerts = False

'double click action valid for column 1 - will put a check mark in 3 columns
If Not Application.Intersect(Target, Columns("A")) Is Nothing Then
    'resize range to 3 columns in same row
    Set rng = Target.Resize(1, 3)
    rng.ClearContents
    rng.Value = ChrW(&H2713)
    Cancel = True
End If

ErrHandler:
    Application.DisplayAlerts = True

End Sub
