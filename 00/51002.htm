BACKGROUND

 

In Windows 8, Windows Media Player erroneously regards some (not all) of the internal SATA hard drives as sync devices.

 

QUESTION

Is there any way to stop an internal hard drive from showing up in Windows Media Player as a sync device?

Instructions given for Windows Vista (eject disk, right click in WMP, select End Sync Partnership) are not applicable here, as this option no longer exists in Windows 8 (ejecting the disk removes it from WMP, so there is nothing to right-click).

ganesh achar b say:

Hi Paul,

 

You can go through the following work around and check.

a. Open Control Panel.

b. Click on Hardware and Sound category.

c. Under Device and Printers, click on Device Manager.

d. Click on Portable Devices.

e. Right click on your "Device" if it's listed under there.

f. Select Disable.

 

It removes the device from Windows Media Player.  If you tries to uninstall it completely but it just show back up once you reboot the computer.  It shouldn't appear again unless you Enable it.

 

If you are unable to see the Portable Devices, click on “view” and “show hidden devices”.

 

If the issue still persists, refer to the following method. Run the Windows Media Player settings Troubleshooter to diagnose the exact issue. Follow these steps to run the Troubleshooter.

a. Press “Windows Key + C” to open Charms Bar.

b. Select Search option.

c. Select “Settings” then type “Troubleshooting” in the Search box and hit enter.

d. Click on “View all” and choose the Windows Media Player settings

e. Follow the on-line screen instructions.

 

Hope this information helps you. Please do let us know if you need any assistance, we’ll be glad to assist you.

You need Windows 7 64 bit files to install Windows 7 64 bit, you can't install it from 32 bit media.

 

You will have to do a clean installation, there is no in place upgrade path from Windows 7 32 Bit to 64 bit.

 

Using the Windows 7 64 bit disk, install the Windows 7 Windows Easy Transfer Utility from the DVD (To find it, just navigate to x:\support\migwiz, 'x:' represents the name of your disk drive. In this folder, you will see a collection of files, the important one you need to look for and double click is 'migsetup'.) , run it and create a WET file (Windows Easy Transfer file), if you have an external hard disk, you can have it stored there so you can access it later when you do a clean install of Windows 7 64 bit disk.

 

Once you have done that, simply do a clean install of Windows 7 64 bit disk. In Windows 7 64 bit disk, launch Windows Easy Transfer, follow the on screen wizard, point to where you stored the the WET file (external hard disk), and it will restore your personal files and settings. Reinstall any necessary applications.
