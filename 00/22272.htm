    Hyperlink Fields are odd beasts, and most generally avoid them. Instead, use a standard Text field and then use the Application.FollowHyperlink method if you want to allow users to navigate to the item. 

    For example, if I have a Text field named "MyPath", and I want to allow the users to navigate to the file at the path found in MyPath, I'd do this on my form:

    Application.FollowHyperlink MyPath

    If you're trying to get this to work at the Table or Query level, then always remember that Tables are for storing data, and users should never work directly with them. Queries are for combining or manipulating data in preparation for other processes (like reporting, perhaps) and it is quite rare that users would work with them directly.

    rick rothstein, mvp - excel say:

    Give this macro a try (testing on a copy of your workbook just in case)...

    Sub Move100PercentRows()
      Dim LastRow As Long
      Const StartRow As Long = 2
      Const PercentColumn As String = "U"
      LastRow = Worksheets("Sheet2").Cells.Find(What:="*", SearchOrder:=xlRows, _
                          SearchDirection:=xlPrevious, LookIn:=xlFormulas).Row
      With Worksheets("Sheet1")
        .Columns(PercentColumn).Replace "100%~*", "#N/A", xlWhole
        With .Columns(PercentColumn).SpecialCells(xlCellTypeConstants, xlErrors).EntireRow
          .Copy Worksheets("Sheet2").Cells(LastRow + 1, "A")
          .Delete
        End With
      End With
    End Sub

    rick rothstein, mvp - excel say:

    You can also do what you want without the overhead of loading a RegExp engine...

    Function ANU(ByVal strVal As String) As String
      Dim X As Long
      For X = 1 To Len(strVal)
        If Mid(strVal, X, 1) Like "[!0-9A-Za-z_]" Then Mid(strVal, X) = Chr(1)
      Next
      ANU = Replace(strVal, Chr(1), "")
    End Function

    Note: I called the function ANU, short for Alpha Numeric Underscore, to keep the name short (I hate typing<g>), but you can change it to whatever you like, of course.

    Your pattern is fine but you must set GLOBAL to TRUE

     

    Function RegExpReplace(strVal As String) As String
    Dim reRegExp As RegExp
    Set reRegExp = New RegExp
    With reRegExp
        .Global = True
        .Pattern = "[^A-Za-z0-9]+"
        RegExpReplace = .Replace(strVal, "")
    End With
     End Function

    Whilst watching Sky sports on my laptop the screen freezes constantly - Microsoft Community

    calumben say:

    Whilst watching Sky sports on my laptop the screen freezes constantly and the narrative states buffering

    Whilst watching SKy sports on my lap top the screen freezes constantly and the narrative states buffering; why does this happen; what is buffering
    sandeep ghatuary say:

    Hi calumben,

     

    A buffer contains data that is stored for a short amount of time, typically in the computer's memory. The purpose of a buffer is to hold data right before it is used. For example, when you download an audio or video file from the Internet, it may load the first 20% of it into a buffer and then begin to play. While the clip plays back, the computer continually downloads the rest of the clip and stores it in the buffer. Because the clip is being played from the buffer, not directly from the Internet, there is less of a chance that the audio or video will stall or skip when there is network congestion.

    Buffering is used to improve several other areas of computer performance as well. Most hard disks use a buffer to enable more efficient access to the data on the disk. Video cards send images to a buffer before they are displayed on the screen (known as a screen buffer). Computer programs use buffers to store data while they are running. If it were not for buffers, computers would run a lot less efficiently and we would be waiting around a lot more.
