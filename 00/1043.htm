mike h.. say:

Hi,

One way. In the Weekdayname function the FALSE argument means you get the full weekday name.

Sub somesub()
Dim DayOfWeek As String
Dim MyDate As Date
MyDate = Now
DayOfWeek = WeekdayName(Weekday(MyDate), False, 1)
End Sub
