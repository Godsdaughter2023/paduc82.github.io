jennn1978 say:

Highlight a character at a specific text position in each column

For example in the first row the sequence is 100 characters long and I want to highlight or change the color of character 15. In row 2 I have 112 characters and want to highlight character 28... and so forth. In total I have about 1000 rows, each with a single character position (which varies) that I want to highlight. Is this possible?
jeeped say:
.. I have a column with series of text characters (DNA sequence) and for each cell I want to highlight a specific character at a specific position (from a second column with the text position in integer form) ...

You could accomplish this with a macro. Tap Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert, Module. Paste the following into new pane titled something like Book1 - Module1 (Code),

Sub mcrRedBoldCharacter()
    Dim c As Range
    For Each c In Selection
        With c.Characters(Start:=c.Offset(0, 1).Value, Length:=1).Font
            .FontStyle = "Bold"
            .Color = -16777024  'dark red
        End With
    Next c
End Sub

Tap Alt+Q when done to return to your worksheet. Select your column of DNA sequences and run the macro (e.g. Alt+F8). You do not need to select the column of position numbers as they are assumed to be one cell to the right of the cell being changed. This macro will set the character at position x as bold and dark red.
