I have been having this problem all day, and I finally fixed it thanks to a couple of posts I saw elsewhere on here and adapted to my specific version and installation of Office 365 (64-bit) on Windows 8.1 (64-bit).

cd /d "%SystemDrive%\Program Files\Microsoft Office 15\root\office15"

Groove.exe /clean /all

cd /d "%SystemDrive%\Program Files (x86)\Microsoft Office 15\root\office15"

Groove.exe /clean /all

Hope this helps someone. It's too late for the chunk of hair I pulled out, but the rest of my head is now safe. Best wishes.

jon k. jr. say:

Smartly copying tabs with VBA

I have a bunch of single-tab worksheets produced by colleagues which I combine into a single working file.  Each input tab is named quasi-randomly, though each ends with "_INPUT" (e.g. "SmithNov2014_INPUT", "Bill_INPUT", "2014-12-02_Mary_INPUT").

I will be reviewing these tabs individually, and when my review is complete, I would like to create a values-only copy of each, to be named similarly to the source tab, but where the tab name ends "_FINAL" (e.g. SmithNov2014_FINAL", "Bill_FINAL", 2014-12-02_Mary_FINAL").  Thus, my working file with 3 INPUT tabs will end up with the 3 INPUT and 3 new FINAL tabs.

When done reviewing all the INPUT tabs, and the FINAL tabs have been created, I would like to then copy all the created "_FINAL" tabs (however many there are) to a stand-alone file -- the typical Excel default new file "Book1" would suffice for my purposes.

I'm a VBA neophyte (virtually nothing beyond recorded keystroke macros) and I would like to be set on the path to (1) test the name of the current tab; (2) make a copy of the validated "xxx_INPUT" file as a values-only version named "xxx_FINAL"; and (3) copy all existing "..._FINAL" tabs to a stand-alone file.

Excel 2010 (Windows 7 Enterprise) applies.

Thanks in advance,

bernie deitrick say:

After you have finished reviewing a sheet, with that INPUT tab still active, run this macro

Sub MakeFINAL()
    Dim shtInput As Worksheet
    Set shtInput = ActiveSheet
    If Not shtInput.Name Like "*INPUT" Then
        MsgBox "The current sheet is not an INPUT sheet"
    Else
        shtInput.Copy After:=Worksheets(Worksheets.Count)
        Worksheets(Worksheets.Count).Name = _
            Replace(Worksheets(Worksheets.Count).Name, "INPUT (2)", "FINAL")
        Worksheets(Worksheets.Count).UsedRange.Cells.Value = _
            Worksheets(Worksheets.Count).UsedRange.Cells.Value
        shtInput.Activate
    End If
End Sub

Then to export all the FINAL tabs to a new workbook from the activeworkbook, - you will be prompted for a new file name:

Sub ExportFINAL()
    Dim sht As Worksheet
    For Each sht In Worksheets
    If sht.Name Like "*FINAL" Then
        If ActiveSheet.Name Like "*FINAL" Then
            sht.Select False
        Else
            sht.Select True
        End If
    End If
    Next sht
    ActiveWindow.SelectedSheets.Copy
    ActiveWorkbook.SaveAs Application.GetSaveAsFilename
End Sub
