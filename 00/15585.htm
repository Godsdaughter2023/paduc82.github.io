Hide subform until main form is populated - Microsoft Community

questionboy say:

Hide subform until main form is populated

I have a simple form with a subform and they are linked by master-child fields.

 

How can I set it up so the subform isn't available until the main form is actually populated so as to actually generate a PK value to link the subform value on?

 

QuestionBoy

hansv mvp say:

1) Create an On Current event procedure for the main form:

 

Private Sub Form_Current()
    Me.MySubform.Visible = Not IsNull(Me.PrimaryKeyField)
End Sub

 

where MySubform is the name of the subform and PrimaryKeyField is the name of control bound to the primary key.

 

2) If the primary key of the main form is an AutoNumber field, it will be generated the moment the user starts entering data, so you can use the Before Insert event of the main form to unhide the subform:

 

Private Sub Form_BeforeInsert(Cancel As Integer)
    Me.MySubform.Visible = True
End Sub

 

But if the user enters the primary key, you should use the After Update event of the control bound to the primary key:

 

Private Sub PrimaryKeyField_AfterUpdate()
    Me.subSubTotal.Visible = Not IsNull(Me.PrimaryKeyField)
End Sub
