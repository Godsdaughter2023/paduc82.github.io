As the router needs just up to 64MB of ram (mine is using 34mb at the moment), it's possible to run the DD-WRT on a old laptop or anything you have around that can boot Linux (doesn't have to be virtual! It can even be physical and run directly on the hardware).
It's even possible to boot it from a USB Stick. I've thought about trying to use my Raspberry-Pi (http://www.raspberrypi.org/) as a router (there is Open-WRT support for it) but that's a project for some cold, dark day during the winter.
