Be aware that this is not how the reporting or enforcement system works.

Whenever someone reports you, the console involved will collect data and telemetry and sends that in with the report as evidence. In the case of a message that is reported, the message gets attached to the report. 

If someone (or multiple people) report you "100 times", then the Enforcement system consolidates all 100 reports into one single report.

Reporting someone over and over will NOT get them banned automatically. It just causes this one single report to move up the report queue faster. 

In the end, a real human being from the enforcement team then looks at the report and the attached evidence. If the evidence shows that you did nothing, then the Enforcement Team member will do nothing, and you will not get banned. No matter if one, two or one hundred reports were created about you.

If however your message contained something that violated the code of conduct (for example your message put someone down or belittled them without actually using swear words), then the Enforcement Team member will issue a suspension. 

If you still think the suspension is unjustified, you can try to file a case review on https://enforcement.xbox.com

atif akhtar ali say: Use this information it will work......i have tried n it workd for me.......

The first thing to do is to stop the Windows Update service on your Windows 7 machine. You can do this by going to Start and typing in services.msc in the search box.

Next, press Enter and the Windows Services dialog will appear. Now scroll down until you see the Windows Update service, right-click on it and choose Stop.

Now that the Windows Update service has been disabled, you need to go to the following folder and delete all the files and folders inside of it:

C:\Windows\SoftwareDistribution\

Once you have deleted everything, go back to the Services dialog and right-click on the Windows Update service and choose Start. All the files and folders will be re-created and hopefully you can start downloading updates for your PC.

    a. user say:

    Unfortunately, there is no website that can tell you this. Neither Microsoft nor any other website owner can possibly be aware of all the millions of different types of hardware being used throughout the world.

    But the hardware manufacturers know whether their hardware will work with Windows 10 ... because they're the ones who make that decision!

    So here's what you do:

    Make a list of the most important hardware in your computer:

    • motherboard
    • graphics adapter (chip or card)
    • network adapter (chip or card; you may also have a USB WiFi adapter)
    • audio adapter (chip or card)
    • printer (laser or inkjet)
    • keyboard and mouse

    Aside from the motherboard, get the manufacturer's name and model number for each hardware component. You won't need that information for the motherboard (unless you built the computer.)

    Now go the websites for the respective hardware manufacturers and look for the technical support page for your model number. That's typically where the manufacturers show the drivers that are available for the hardware. (I'll deal with the motherboard in a second.)

    If the manufacturer shows a Windows 10 driver for the hardware - life is good - because that means the manufacturer has decided to keep that model compatible with Windows 10; they'll also tell you which version of Windows 10 (e.g., version 1709) is compatible.

    For the motherboard, ask the computer manufacturer if it's supported on Windows 10.

    Sometimes you'll discover that - oh no! - there's no Windows 10 driver for the hardware. That doesn't necessarily mean you're out of luck. If there's a driver for Windows 8.1 that might work with Windows 10 - for now, at least. Sometimes, even a Windows 7 driver will work. But you won't know in advance. And the manufacturer doesn't know because they haven't tested it on Windows 10.

    In that case, you could try the Win 8.1 or 7 driver. But you're taking a risk - what if it messes up your computer? So the smart thing to do is to ask a question in the manufacturer's own community forum. Find out if other users of the hardware have had success with Windows 10, and how they did it.

    Sounds like a lot of work, yes? But that's how to find out for sure.

    You could also just go for it, install v1709 (or wait for the upcoming Spring Creators Update,) and see what happens. In that case, make sure you make an image backup before you start, so you'll have a way of easily going back to the way things were before.

    On Thu, 15 Jul 2010 12:53:36 +0000, dkalling wrote:

    Is there a different 25-character Product Key for the *Corporate edition*of Windows XP PRO SP3 than for Win XP PRO in general?

    Three points:

    1. Microsoft has no "Corporate Edition." The term is used only for
    pirated versions. You are perhaps thinking of a volume license.

    2. Every copy of Windows has a different product key.

    3. Each edition of Windows XP has a different series of product keys
    than the other editions. So you can not use a product key from one
    edition (for example Windows XP Professional) on another edition (for
    example, Windows XP volume licensed). It will not work on another
    edition.

    Ken Blake
