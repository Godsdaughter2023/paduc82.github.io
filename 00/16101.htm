Public Function do_conversion()
On Error GoTo Err_func_err
Dim db As DAO.Database
Dim rs As DAO.Recordset
Dim fso As New Scripting.FileSystemObject

Dim the_pathdb As String
Dim the_id As Integer
Dim the_path As String, the_wp As String, the_doc As String
Set db = CurrentDb()

Set rs = db.OpenRecordset("SELECT * FROM wpd WHERE the_status=0")
rs.MoveFirst
Do Until rs.EOF
    the_pathdb = rs!the_doc
    the_id = rs!the_id
    the_path = fso.GetParentFolderName(the_pathdb) & "\"
    the_wp = fso.GetBaseName(the_pathdb) & ".wpd"
    the_doc = fso.GetBaseName(the_pathdb) & ".doc"
    Responce = open_word(the_path, the_wp, the_doc, the_id)
    rs.MoveNext
    Loop
rs.Close

'Close DB
Set rs = Nothing
Set db = Nothing

Exit_func_err:
    Exit Function
Err_func_err:
    MsgBox Err.Description
    Resume Exit_func_err
End Function

Public Function open_word(the_path As String, the_wp As String, the_doc As String, the_id As Integer)

Dim objWord As Word.Application
Set objWord = New Word.Application

objWord.Visible = True
objWord.Documents.Open Application.CurrentProject.Path & "\doc_to_wpd.doc"

objWord.Quit
Set objWord = Nothing

Exit_func_err:
    Exit Function

    MsgBox Err.Description & " " & Err.Number

a. user say:

1. How to turn that automatic downloading of Windows 10 upgrade files as soon as I start my computer which is connected to the internet.

You need to uninstall then hide update KB3035583 - (GWX - get windows 10).

a) click start, control panel, Programs and features
View installed updates
Locate - update KB3035583 & right-click and uninstall.

b) Reboot your computer

c) click start, Control Panel, Windows Update
Check for updates
It should show 1 important update
Click 1 important update
KB3035583 - Right - click - hide update

If it comes back at a later date, hide KB3035583 again.
When you are ready to install windows 10 then unhide KB3035583.

https://support.microsoft.com/en-us/kb/3035583

Update installs Get Windows 10 app in Windows 8.1 and Windows 7 SP1

This update installs the Get Windows 10 app, which helps users understand their Windows 10 upgrade
options and device readiness. For more information about Windows 10, see Windows 10.

This update applies to Windows 8.1 or Windows 7 Service Pack 1 (SP1) and Internet Explorer 11 (IE11).
Before you install this update, check out the Prerequisites section.

2. Should I need to wait for more time for that BLACK screen to disappear once that 30% installation portion is finished.

Yes, it will go black a few times and reboot when it needs to during the update process.

3. Can I make the recovery DVD of Windows 10 after I upgrade my laptop. Will the system show an option to make the recovery DVD. Since there is already a recovery drive made for Windows 7 in my HP laptop and I have also made the recovery disks which it allows to make once in life time.

Yes, download windows 10 media creation tool and create an ISO / DVD / USB

Make sure you pick English, Home & either 32 bit or 64 bit.

http://www.microsoft.com/en-us/software-download/windows10

Use Download Tool Now (32bit version)
http://go.microsoft.com/fwlink/?LinkId=616935

Use Download Tool Now (64bit version)
http://go.microsoft.com/fwlink/?LinkId=616936


4. What to press on the keyboard during that BLACK screen (while upgrading to Windows 10) so that I am able to get the options to roll back to my previous version of Windows 7.

There is no option to roll back while upgrading to windows 10. You need to wait until windows 10 is fully installed.

Then click start, settings, Update & Security, Recovery. Under Restore back to Windows 7 click [get started].

You have only one month to restore back to your previous windows.

5. How to STOP the auto upgrade process since the laptop automatically restarts forcefully and starts with the windows 10 installation process. I want to upgrade but at a later date. I also don't want to download it right now but yes definitely at a later date. How to solve all this.

Same as answer for question 1.


6. If I close my laptop while the windows update is downloading the windows 10 upgrade - Will it create a problen in the installation file.

Yes it will pause and crash the update. You need to change the options to turn off the screen display, sleep
& Hibernation to never during the windows 10 update. Also uninstall any third-party antivirus software like Norton, McAfee, etc.

Yes, it will download the windows 10 update files and continue each time you power on your pc.

It will download the update to C:\Windows\SoftwareDistribution\Download, then

It will create a hidden folder C:\windows.~BT and copy the windows 10 setup files there ready to install.

To view this hidden folder, click start, My computer, organize - folder and search options, view tab

Tick [/] Show hidden files, folders and drives.

The same directory is created as well as C:\windows.~WS when using the Windows 10 media creation tool.
