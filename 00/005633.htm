Fri, 3rd, June, 2016.

Hello. I just want to add to this thread and hopefully help the community with my experience and findings.

I have an old XT2 and have tried to upgrade from Windows 7 64-bit to Windows 10 64-bit but experienced the same problem and frustration as everyone here with the phantom / ghost touches; even from doing a scratch/straight Windows 10 (Pro version 1511) install.

After long hours and days of testing and re-installing, what I have found out is the N-Trig DuoSense MultiTouch HID Device driver maybe the culprit. After my most recent straight Windows 10 Pro install was done, to prevent the phantom / ghost touch, I disable the N-Trig DuoSense MultiTouch HID Device driver from Device Manager and it seems to have suppressed the phantom/ghost touches. And with one additional consolation; the pen/stylus still works with the tablet. It is not a perfect solution but it suppresses the annoying/frustrating phantom / ghost touches.

Here are the steps I did to get to this point:

  1. Installed Windows 10 (Pro version 1511) from scratch. Either via external USB DVD drive or via bootable USB Flash Drive.
  2. After installation is completed, leave XT2 connected to the internet and let all the automatic updates go through. The video display adapter driver should be one of the first to get updated.
  3. You can then go to Dell's XT2 Product Support site and select latest updates to BIOS and chipsets. Install them, if you haven't already done this in the past.
  4. Also set/change OS to Windows 7 64-bit and download the latest ControlPoint System Manager and Security Device Drivers.  The System Manager adds the button functionalities (buttons next to power button) and the Security Device Driver is for the Fingerprint Scanner (or 5880 device manager error you see initially). You need to run the compatibility check on the fingerprint scanner driver to get it to install.
  5. At this point, most Windows updates are done and you will notice that the N-Trig DuoSense drivers are being updated. (After it reboots, you smile when you noticed you can now touch your screen but you will also notice the start of the erratic phantom screen movements). 
  6. Depending where the phantom touch occurs on your display, quickly get Device Manager up move it out of the way of the phantom touches and disable the N-Trig DuoSense MultiTouch HID device.**

**Note: I installed an external display and moved Device Manager there so the phantom touches don't accidentally set or disable anything else.

I hope that Microsoft and/or Dell releases a newer/updated version of the N-Trig DuoSense MultiTouch HID device driver to fix this problem. Although the XT2 is old, it still functions nicely with more RAM and larger hard drive/SSD.

Good luck to all.

Inserting an image that way will make it autosize based on the width of the Reading Pane or opened message window. If they were to enlarge that window, the picture will show bigger as well.
