If the WAP server is a non-domain joined machine, there are a couple of additional requirements:
On the target machine (WAP machine):
1. Please run ‘services.msc’ and ensure ‘Windows Remote Management / WS-Management’ service is running.

  c. To validate remote PowerShell connectivity, in the Server Manager All Servers tab: right click WAP server and choose ‘Windows PowerShell (PSH)’. A remote PSH session should open to ensure remote PowerShell sessions can be established.


After that, please re-run wizard.

Thanks,
Edward

rob steuer say:

I just had the same problem, but found the solution.

The reason it was failing is because in the Add Servers, you entered the IP address.  I have a feeling that when you ran "Set-Item WSMan:\localhost\Client\TrustedHosts –Value <DMZServerFQDN> -Force –Concatenate" you entered the name of the box.

Instead, run "Set-Item WSMan:\localhost\Client\TrustedHosts –Value <DMZServerFQDN> -Force –Concatenate" with the DMZServerFQDN just like it says...

So for example, to get mine to work, I did the following:

1. On the AD Connect server, for good measure, I created a hosts file with entry 192.168.3.30 server1.contoso.com

2. Run "Set-Item WSMan:\localhost\Client\TrustedHosts –Value server1.contoso.com -Force –Concatenate"

Then try to add the server again and follow it up with right clicking on the server and clicking Manage As... enter the local credentials for that DMZ box and it should work.  It did for me.

Really the key here is that whatever you enter for the DMZServerFQDN has to be the exact same value for what you enter into the Server Manager -> Manage -> Add Server.

        PartA = HYPERLINK(

        PartC = "A1"

        PartD = ".jpg"   - or whatever the other cells may contain

3.  Look at the cell reference "A1" and its content

4. Rebuild the formula in Column-C as    PartA+PartB+PartC+PartD+PartE

jan karel pieterse say:

Change the HYPERLINK formula to:

="www.longpathname.page" & A1 & ".jpg"

Copy.paste as values.

Select the cells and run this little macro

Sub Convert2Links()
    Dim oCell As Range
    For Each oCell In Selection
        oCell.Hyperlinks.Add oCell, oCell.Value, oCell.Value, , oCell.Value
    Next
End Sub
