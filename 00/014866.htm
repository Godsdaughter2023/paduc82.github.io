stefan blom say:

The AutoFormat As You Type options are global; that is, the selected options apply to all documents in Word. The replacement will only occur while you are typing; existing text in a document won't be affected.

More importantly in this case, though, make sure that language formatting is correctly specified, not because AutoFormat As You Type is language-specific (it is not), but because Word depends on language to recognize that "1st" (etc.) is the kind of expression that should be replaced.

To prevent Word from guessing the language, the first thing to do is turn off the "Detect language automatically" option in the Language dialog box (Review | Language | Set Proofing Language). For existing text, you should select all and (re-)apply the correct language, again via the Language dialog box.

talking moose say:
This is unsupported by Microsoft and you’re at your own risk of using it.
  1. Quit Outlook and any other Microsoft products currently running.
  2. Open Activity Monitor found in /Applications/Utilities.
  3. Search for all “Microsoft” processes and quit them from within Activity Monitor.
  4. If you’re running Office 365’s version of Office for Mac then quit the Office365Service too.
  5. In your home folder and within Documents, locate the “Microsoft User Data” folder.
  6. Drag this folder in its entirety to your SD card to copy it.
  7. When the copy is done delete the original “Microsoft User Data” folder.
  8. Hold Option + Command as you drag the “Microsoft User Data” folder from the SD card back into the Documents folder. This will create an alias to the new location with the same name.
Launch Outlook and any other Office for Mac applications you need. Everything should get redirected through the alias to the folder on your SD card.

Hope this helps!

hansv mvp say:

Try

 

Sub SetCF()
    With Range("B6:H50").FormatConditions
        ' Remove any existing conditional formatting
        .Delete
        ' Add condition: cell value equals today
        With .Add(xlCellValue, xlEqual, "=TODAY()")
            ' Set fill color to green
            .Interior.Color = vbGreen
        End With
    End With
End Sub

 

You can specify different formatting, of course.
