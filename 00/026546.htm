graham mayor say:
There is no PDF function in Excel XP (2002), so you are limited to third party products. Office 2007 and later include a PDF creation function and so are easier to program.

Acrobat 9 is not compatible with recent Office versions, other than as a simple printer driver, and I know nothing about the abilities of PDF_Xchange.

I suspect an answer might be to install PDF Creator - it's an open source PDF tool and is programmable from VBA. (It comes with extras so use Expert install mode and only install the core program - unless you want all the extras). You could then use something like:

Option Explicit

Sub Test()
ActiveSheet.PageSetup.PrintArea = "$A$1:$E$19" 'the area to be printed to PDF
PrintToPDFCreator Range("C9") & ".pdf", ActiveWorkbook
End Sub

Sub PrintToPDFCreator(sPDFName As String, xlBook As Workbook)

Dim pdfjob As Object
Dim sPDFPath As String
Dim defPrinter As String

    sPDFPath = xlBook.Path & Application.PathSeparator 'the folder where you wish to save
    Set pdfjob = CreateObject("PDFCreator.clsPDFCreator")

    With pdfjob
        .cStart "/NoProcessingAtStartup"
        .cOption("UseAutosave") = 1
        .cOption("UseAutosaveDirectory") = 1
        .cOption("AutosaveDirectory") = sPDFPath
        .cOption("AutosaveFilename") = sPDFName
        .cOption("AutosaveFormat") = 0        ' 0 = PDF
        .cClearCache
    End With

    'Print the document to PDF
    xlBook.PrintOut

    'Wait until the print job has entered the print queue
    Do Until pdfjob.cCountOfPrintjobs = 1
        DoEvents
    Loop
    pdfjob.cPrinterStop = False

    'Wait until PDF creator is finished then release the objects
    Do Until pdfjob.cCountOfPrintjobs = 0
        DoEvents
    Loop
    pdfjob.cClose
    Set pdfjob = Nothing
End Sub






shane devenshire say:

First question - at the top you say you are using Excel XP, that would be 2002?  But at the bottom you say Excel XP - 2010.

 

The Save as PDF functionality was introduced in 2007 and continues to be supported in 2010 and 2013.  If you are running 2002, which was release in 2001, you might consider upgrading.  Unfortunately most of us are on one of the 4 later versions so it is very difficult to test.

 

Something like this will work in 2007, 2010, or 2013

 

Sub PrintAsPDF()
    myPath = Range("C9") & "\"
    With ActiveSheet.PageSetup
        .PrintArea = "A1:H40"
        .FitToPagesWide = 1
        .FitToPagesTall = 1
        .Orientation = xlPortrait
        .CenterHeader = ""
        .LeftFooter = "&L&D - &T&CPage &P of &N"
        .CenterFooter = "Page &P of &N"
        .PrintComments = xlPrintNoComments
        .CenterHorizontally = True
    End With
    ActiveSheet.ExportAsFixedFormat _
        Type:=xlTypePDF, _
        Filename:=myPath & ".pdf"
End Sub
