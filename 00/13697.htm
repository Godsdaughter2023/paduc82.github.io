jay freedman say:
Whether the page numbers are consecutive through the sections is completely independent of whether the sections (strictly speaking, the headers and footers of the sections) are linked.

When all the even pages in a section show the same number, that means you typed a specific number instead of inserting a Page field.

Further, to have different content on the even pages and the odd pages, you need to check the box for Different Odd & Even Pages and insert the different texts -- and each kind of header needs its own Page field. The Even headers in each section should be linked to the Even headers in the previous section, and the Odd headers in each section should be linked to the Odd headers in the previous section (these links are independent for each kind of header).

If the page numbers still come out not consecutive, then a section has been set to restart numbering. I doubt that you did that.

You can not store a pst file in onedrive if it is open in outlook. You need to move the outlook files folder out of the folders that sync.

You will not lose the email you have downloaded - you just need to move the pst files and point outlook to the new location.

Move the folder
Go to Control panel, find mail and open your profile.
Select the Data files tab and select each pst then choose Change and select the new folder location.

boardman say:

Wow it is sorted.  I followed instructions from another site as follows and it worked:

  1. Go to File, Account Settings, open the account settings dialog. On the Email tab, select the email account and look at the bottom, click Change folder
  2. Click “New Outlook Data File”.
  3. Accept the name for the new data file (My outlook data file (1)) and press “OK”.
  4. Expand the new data file  and select the “Inbox” subfolder. Choose “OK”.
  5. Now go back, highlight name of email with the problem.  Go to change folder and this time select the folder from the original data file. 
  6. Choose “OK”.

Close and try send/receive again.  It works.  I repeated the process with my second account and called the new data file (My outlook data file (2)).
