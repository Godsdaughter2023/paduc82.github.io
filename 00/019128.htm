daniel g. grau say:

One solution is to use a macro:

Sub SaveAllOpenDocsAsDocx()

Copy that macro to the VBA Editor. Running it will convert all open documents and save converted copies of them by adding the .docx extension (originals are not deleted).

If you want to delete the originals as well, use this version instead:

Sub SaveAllOpenDocsAsDocxAndDelete()

             RememberFullName = aDoc.FullName
             aDoc.SaveAs FileName:=aDoc.FullName & ".docx", FileFormat:=wdFormatDocument
             aDoc.Close
             Kill RememberFullName
        Next aDoc
        End Sub

Test their behavior within a folder containing copies of the old files. Open them all and run one of the macros. Once you know they are working, use them on the target files.


How to paste a macro
  • In your main menu bar, select Tools > Macro > Visual Basic Editor
  • You should see a Project box
  • In there, select Normal (Normal.dotm)
  • In the main menu bar of the Editor, select Insert > Module
  • A new window will open (there will be a default name, but you can type your own)
  • Paste the macro code 
  • Save your Normal.dotm template (File > Save Normal.dotm). Otherwise, Word will prompt you when quitting. You can skip these steps if you don't want to save the macro.
  • In the Editor's main menu, select Close and return to Microsoft Word (or type Command Q)
  • To run a macro, select it through Tools > Macro > Macros.

If you need more help on how to install a macro, see Install a Macro in Word or Macros in Mac Word.

Sub Hide_rows_3shts()
'Dec 20, 2016
Const N1 As Long = 10 'data/first row
Const N2 As Long = 1045 'data/last row
Dim vWS As Variant, vCol1 As Variant, vCol2 As Variant
vWS = Array("data1", "data2", "data3") 'sht names
vCol1 = Array("H", "H", "H") 'first col in each sht
vCol2 = Array("O", "AC", "AC") 'last col in each sht
Dim x As Long, y As Long
For x = 0 To UBound(vWS)
For y = N1 To N2

carlos fernandez - pr say:

How to create a retention tag to archive just inbox and sent items

Hi

I need to configure a Retention policy to archive all mail items older than 60 days. I created a Retention Policy with the 60 days Archive setting and it works, but when I apply the retention policy to the mailbox, all the content, older than 60 days is moved to the archive.

Is there a way to only apply the archive rule to mail content (inbox/sent items)?

THanks

Carlos

mouran yan msft support say:

Hi Carlos,

As we know, the retention policy is made up of the retention tags. And there are 3 types of retention tags: Default policy tag (DPT), Retention policy tag (RPT) and Personal tag. DPT automatically applies to the entire mailbox. RPT automatically applies to a default folder such as Inbox, Deleted Items and sent items. However, the RPT only has the delete action, not the move to archive action. Personal tag is used to apply to messages and other self-defined folders, it cannot be used to the inbox and sent items folder.

In conclusion, it’s not feasible to create an archive retention policy which only applies to the inbox and Sent items folder. Thanks for your understanding.

You can create a custom retention policy which only contains some personal tags such as 60 days move to archive, 5 year move to archive, etc. After that, apply it to the mailbox by going to EAC > recipients > mailboxes > mailbox features. Then the personal tag can be assigned to Inbox and Sent Items folder. Below are the detailed steps:

1. After assigning the custom retention policy to the mailbox, run the PowerShell command:

Start-ManagedFolderAssistant -Identity "EmailAddress"

Before that, you need to connect to Exchange online using remote PowerShell.

2. After runing the PowerShell command to force the retention policy to take effect, you can ask the owner of the mailbox to sign in to his Outlook Web App (OWA).

3. In OWA, he can right click the Inbox folder > Assign policy and  choose applying which archive policy to the inbox folder.

Let's assume the range of scores is G3:G20.

Array entered**:

=IF(COUNT(G3:G20),AVERAGE(SMALL(INDEX(G3:G20,LARGE(IF(G3:G20<>"",ROW(G3:G20)-ROW(G3)+1),MIN(COUNT(G3:G20),10))):G20,ROW(INDIRECT("1:"&MIN(COUNT(G3:G20),5))))),"")

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

That formula will get the average based on these conditions:

  • If there are 0 scores entered the formula will leave the cell blank.
  • If there are <5 scores the formula will average whatever scores are available.
  • If there are >=5 and <=10 scores the formula will average the lowest 5 scores.
  • If there are >10 scores the formula will average the lowest 5 scores out of the last 10 scores.
