Hello,

It's clear that the number of people with this problem is vast.

While I don't claim to be able to solve the problem for all of you, this is what worked for me:

My problem was that no matter where I printed a PDF to, it would allways result in an empty file.

I found that if I ran the program I was printing from as Administrator, a PDF print would work.

I found that the problem was with the permissions granted to the Print Server, not the the Pinter device.

To change the permissions, you need an Administrator level user account.

Press the Windows key an type "print", that should be enought to show "Printers and Scanners", select it to open the Printers and Scanners settings window.

On the right hand side, look for Related Settings and click the Print server properties link.

In the window that opens, select the Advanced tab. See the button at the bottom called Change Advanced Settings? Click it.

In the next window that pops up, select the Security tab.

Now, click the Add... button.

In the new window that pops up, enter the user name or group name into the Enter the object names to select text box and click the Check Names button.

If you've entered a name that Windows recognises, then you should see the name you typed in change to the computer name, a back slash "\" and the name you typed in, all underligned.

Assuming this is the case, click OK.

This will return you to the prior security window, you should now see your user name listed with all the others.

Click on the user name to highlight it.

Now, in the Permissions for [your username] make sure that all the Allow boxes are ticked.

Click OK.

Try printing a document using the Microsoft Print to PDF printer. If all is well, you should now have a working PDF file.

Here's how I fixed it!

1) Plug in the thumbdrive to the xbox360, turn on the console.

2) Log into the profile that has the game you want to transfer.

3) Copy the game(s) (through the system settings, memory, harddrive, games, skyrim, select the game, copy) to the thumbdrive.

4) Log into the profile that you want to copy the game TO.

5) From this profile, either copy a game (owned by this profile) to the thumbdrive, or start up skyrim and save a new game under this profile to the thumbdrive.

6) Turn off the console.

7) On the PC, download Modio ( www.game-tuts.com/.../modio.php ). Run it.

8) Plug in the thumbdrive from the xbox into your PC

9) In MOdio, click the "Explore a device"

10) IN the new window, there should be at least two directories that have a series of letters and numbers (something like E5234AF3234). These are the profile ID's. FIgure out which one is the profile you are copying from, and which is the one you are copying to (I used the names of the saved game files inside the directories to figure out which was which). Write down the profile ID for the profile you are copying the game to.

11) In the old profile, find the game file that you want to copy.

11) Right click on the file and click the "Open In Resigner".

12) Change the ProfileID to the profile id of the profile that you want to copy to.

13) Click the "Rehash and resign" button.

14) Memory is a little foggy, but there should be a "Save to device" or something similiar button. CLick it. It should show you a list of the profile ID's on the thumbdrive. Select the profile you want to copy the game to.

15) CLose the drive (File menu, "OPen/CLose drive")

16) Eject the thumbdrive, stick it back into your xbox.

Now if you turn on the console, go to system settings, memory, thumbdrive, games, skyrim, you should see the saved game with the new profile as the owner. You can either copy that to the harddrive, or just play the game off your thumbdrive.
