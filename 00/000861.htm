greg maxey say:

Ah  moving the goal posts after the start of the game  ;-)

The code I posted was based on the assumption that all of the text in the main body of the document was initial the same size (i.e., 11 pt.)  If there are different sized fonts (i.e., the case with Headings, Titles, or sections that need to stay 11 pt.) then it won't work.

You really need to learn to apply styles to your text.  For example, in the following code:

Sub ScratchMacro()
'A quick macro scratch pad created by Greg Maxey
Dim i As Long
Dim j As Long
Dim oSec As Section
j = ActiveDocument.Styles("Body Text").Font.Size
ActiveDocument.Styles("Body").Font.Size = j + 1
For Each oSec In ActiveDocument.Sections
For i = 1 To 3
  oSec.Footers(i).Range.Font.Size = j
Next i
Next oSec
End Sub

Only the text that has Body Text style applied will be increased by 1 pt. Other text (i.e., the text in lines 8-14) that have a different style applied will not be affected.

You could create a style for this document named "Address Block" and apply it to those lines of text.

Concerning this block moving.  Insert a frame at the location where the text is to appear.  Set the properties of the frame so that it does not move with surrounding text.

Good luck.

a. user say:

Hi Michael,

 

Thank you for posting your query in Microsoft Office Community.

 

As per your issue, there is no default setting to add columns in Publisher text box. However, as a workaround I suggest you to add a text box and then add columns from the format tab. Then insert an auto-shape and then make the text box border ‘No fill’ and then do the same for the auto shape.

 

Hope this information helps. Please post to us if you need any further assistance with Office.
