dale koetke [msft] say:

 

Problem solved by starting from scratch...

 

1. On the Windows Mobile device, go to ActiveSync and delete all partnerships with computers (under the "Options" menu)

2. On the Windows Computer, in Windows Mobile Device Center (WMDC), delete the partnership.

3. Uninstalled OneNote Mobile on the mobile device

4. Uninstalled OneNote on the computer

5. Uninstalled Windows Mobile Device Center.

 

Then I reinstalled:

 

1. Install OneNote

2. Install Windows Mobile Device Center 6.1

3. Plug in device

4. When prompted, install OneNote Mobile

5. In WMDC, I created a partnership to the device.  Now OneNote shows up as a sync option and after the first sync, the "OneNote Mobile" notebook was created as expected.

Percentage:([ActualEndDateTime]-[StartDateTime])/([TargetEndDateTime]-[StartDateTime])*100

cummingsjp say:

Replace Text Macro in PPT

This is sort of the inverse of kt.mcdaniel's post in early 2010.

 

I have a series of 10 slides with distinct titles.  I want to replace each title with the same string (i.e. each slide title will be the same).

I ended up with

Sub ReplaceText()
    Dim oSld As Slide
    Dim oShp As Shape
    Dim oTxtRng As TextRange
    Dim oTmpRng As TextRange
    Dim strWhatReplace As String, strReplaceText As String
    Dim WhatReplace(10) As String
    
    WhatReplace(1) = "Title 1"
    WhatReplace(2) = "Title 2"
    WhatReplace(3) = "Title 3"
    WhatReplace(4) = "Title 4"
    WhatReplace(5) = "Title 5"
    WhatReplace(6) = "Title 6"
    WhatReplace(7) = "Title 7"
    WhatReplace(8) = "Title 8"
    WhatReplace(9) = "Title 9"
    WhatReplace(10) = "Title 10"

   
     ' write change text
    strReplaceText = "Same Title for Each Slide"
    
     ' go during each slides
    For Each oSld In ActivePresentation.Slides
         ' go during each shapes and textRanges
        For i = 1 To 10
           
     ' write find text
    strWhatReplace = WhatReplace(i)
    ActivePresentation.Slides(i).Select
       
       
        For Each oShp In oSld.Shapes
             ' replace in TextFrame
            Set oTxtRng = oShp.TextFrame.TextRange
            Set oTmpRng = oTxtRng.Replace( _
            FindWhat:=strWhatReplace, _
            Replacewhat:=strReplaceText, _
            WholeWords:=False)
            
            Do While Not oTmpRng Is Nothing
                
                Set oTxtRng = oTxtRng.Characters _
                (oTmpRng.Start + oTmpRng.Length, oTxtRng.Length)
                Set oTmpRng = oTxtRng.Replace( _
                FindWhat:=strWhatReplace, _
                Replacewhat:=strReplaceText, _
                WholeWords:=False)
            Loop
        Next oShp
        Next i
    Next oSld
End Sub

Got around to putting together a routine to do the job you described. Here you go.

 

 

Sub replace()

Dim textcollection(1 To 10) As String, sld As Slide, shp As Shape  'define our variables

 

For I = 1 To 10   'fill the string array automatically

textcollection(I) = "Slide " & I & " Title"

Next

 

For Each sld In ActivePresentation.Slides 'go through each slide

   For Each shp In sld.Shapes             'go through each shape

      If shp.Type = msoPlaceholder Then   'if the shape is a title box then fill it

          shp.TextFrame.TextRange = textcollection(sld.SlideIndex)

      End If

   Next shp

Next sld

 

End Sub


If you want to fill the string array manually then do as you were doing in your post. You can also have it only fill the title slide box by entering a Exit For command after it fills the first shape. You can also make it only fill textboxes by replacing "msoPlaceHolder" with "msoTextBox."

Public Function udfShowFormula(r As Range) As String

    If r.HasFormula Then udfShowFormula = r.Formula
