    That would be the "Lock" screen.  In order to dismiss it you may have to press Ctrl-Alt-Delete and then bring up the Login prompt.  However, some have found that even from the Lock screen that they can start Narrator with Win-Enter.  So doing that may help you navigate further in the "darkness".  Note that if you can get to the Login prompt you should find at its bottom right a Power menu where you can then use 

    A related possibility would be to attach a second monitor and see if there is anything showing on it.  E.g. a common "blank" screen at boot scenario is that the video system mistakenly displays the necessary details on a phantom monitor and leaves the real monitor blank, as if it was handling an Extended display.  In that case it has previously been possible just to immediately press Win-P, CursorUp, Enter; e.g., as if imagining that the Project dialog being rendered on the phantom monitor could be used normally via the keyboard.  This would have to be done from the Login prompt (or later).   It used to be that the mere pressing down of the Win- key was enough to switch the state from showing the Lock screen to showing the Login screen; so that's why it would have worked as a general procedure.  Unfortunately, lately it seems that pressing the Win- key is not enough to dismiss an unlocked Lock screen; so you may have to do something else such as press CursorUp first.
