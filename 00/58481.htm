chris cupler say: Yes!

When you buy a copy of Windows, you're not buying the media (disc.)  Rather, what you're buying is a license to use the product; the license includes a license key.  This license is (for Retail, Upgrade, and OEM versions) for use on one computer only.  Retail and Upgrade licenses are transferrable to new hardware; OEM licenses are tied to the hardware on which they're first activated.
You are free to reinstall (and, of course, activate) that copy as many times as you wish (within the terms as listed above.)

The EULA specifically prohibits the purchaser from sharing the license key with others.

<edit>
Your Technet subscriber agreement specifically prohibits the sharing, distribution, etc. of product keys, as well.

a. user say:

Hello Joseph,

Thank you for posting your query in Microsoft Community.

What troubleshooting steps did you tried?

Are you able to see the Biometric device in Device Manager?

Regret the inconvenience you have experienced.

First ensure that BIOS is Up to date for your device and that biometric device is enabled in BIOS. Check the status of biometric device under System Configuration in BIOS. If a biometric option does not exist in your BIOS, then the fingerprint reader is always enabled. Take help of device Manual or Manufacturer support to check this.

Warning

  • Be careful when changing BIOS settings. The BIOS interface is designed for advanced users, and it's possible to change a setting that could prevent your computer from starting correctly.

  • Updating BIOS should only be done if necessary (to solve a compatibility problem, for example). It can be a complicated process, and if an error occurs, your computer could be rendered inoperable. Be sure to follow the manufacturer's instructions exactly.

Once done with the above criteria, follow the below methods.

Method 1:

If you’re able to see the Biometric device in device manager, then update the drivers from Device Manager manually and check if this helps.

a) Open run command by pressing Windows key + R and type “devmgmt.msc” (without quotes) and hit Enter.

b) Spot and expand the “Biometric or Biometric Devices”.

c). Right click on the Fingerprint reader driver and select “Update Driver Software”. When it prompts to choose an option How do you want to search driver, then go with Search automatically for updated driver software. If you have a driver available on your PC, then select Browse my computer for driver software, then select the appropriate location, where the driver located on your PC for updating it.

d). Once this is done, restart the computer to check the problem fixed. If it doesn't fix the issue, then Right click on the same driver and select “Uninstall” then restart the system and let the system install the driver automatically on Start. Check and see if it resolve the issue.

Method 2:

Check and install latest Windows 10 Fingerprint reader device driver for your device from Manufacture's website and check if it helps. You can search them using your device model on Manufacture's website.

If no updated drivers for Fingerprint reader available on the computer manufacturer’s website for windows 10, then trying installing available drivers in compatibility mode. Compatibility mode runs the program using settings from a previous version of Windows.

Follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the driver from the manufacturer’s website.

b. Right-click on the driver setup file and click on ‘properties’.

c. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 8/8.1 or 7 operating system from the drop down menu and proceed with the installation. Once this is done, restart the computer and check if the issue persists.

Keep us posted how it went.

ski555 say:

I experienced a similar issue after installing Windows 10; I wasn't getting an option to create a HomeGroup. I just got done with a 2 hour live chat session with a Microsoft tech. After a couple of hours of trying to edit my register, it occurred to him to try a much easier fix. I know this sounds incredibly simplistic, but it actually worked for me. I'll keep my fingers crossed that you have success too!

1. Turn OFF *all* other computers/devices from your Network except the computer that you want to use to create a HomeGroup.  Leave all of these devices OFF until Step 6. Seems there can only be 1 computer on when trying to create a HomeGroup in Windows 10.

2. On the computer you want to use to create a HomeGroup:

a. If it's hardwired - disconnect the Ethernet cable for 30 seconds, then plug it back in.

b. If it's a wireless connection, turn off your wireless connection for 30 seconds, and then turn it back on.

3. Restart the computer that you are using to create a HomeGroup.  Type your password, hit 'Enter', and wait a full 5 minutes for everything to load.

4. Go to your Control Panel and check your HomeGroup options. You *should* (fingers crossed!) now be able to create a HomeGroup.

5. Follow the instructions to create a HomeGroup and wait 5 more minutes.

6. Turn ON the other devices that you want to join the HomeGroup. Access your HomeGroup options from the Control Panel. You should now see an option to join.

Good luck!
