bruce hagen ~ mvp say:
Are you sure you like WLMail more than OE?  :)
 
If you're on XP, use V: 2009.  2011 will not work on XP. (There is no V: 2010).
 
 
 
Transferring data from Outlook Express to Windows Live Mail:
 

For Messages:
 
Copy the *ENTIRE* OE message store folder to a flash drive. (Folders.dbx must be included). Place this on the Desktop or other location on the machine using WLMail. Open WLMail and: File | Import | Messages | Microsoft Outlook Express 6 and point to where you saved it.
 
OE Message Store Location:
 
In OE: Tools | Options | Maintenance | Store Folder will reveal the location of your Outlook Express files. Write the location down and navigate to it in Windows Explorer or, copy and paste it into Start | Run.
 
In WinXP, Win2K & Win2K3, the OE user files (DBX and WAB) are by default marked as hidden. To view these files in Windows Explorer, you must enable Show Hidden Files and Folders under Start | Control Panel | Folder Options Icon | View, or in Windows Explorer | Tools | Folder Options | View. 
 

For Addresses:
 
Open the Address Book in OE and File | Export | Address Book (wab) and save it to the Desktop. Copy to a flash drive. Place this on the Desktop or other location on the machine using WLMail.
 
For WLMail 2009: Open the Contacts list in WLMail, (Go | Contacts on the Menu Bar), and File | Import | Windows Address Book (wab) and point to where you saved it. 
 
For WLMail 2011: Switch to the contacts window (click the address book icon at the foot of the folder pane, or press Ctrl-3). Click the Import button, select Windows Address Book (wab) and point to where you saved it.
 
Note: If you use a CD or DVD instead of a flash drive, after placing on the new machine you must remove the Read Only attribute in Properties before you import.
 

For Account Settings:
 
In OE: Tools | Accounts, select the account and export it to the Desktop. This will be a .iaf file. Copy it to the new machine's Desktop and in WLMail: Tools | Accounts and import the settings from the location you saved them.
 

Help for Windows Live Mail is found in this forum
http://answers.microsoft.com/en-us/windowslive/forum/livemail?tab=all

The suggestion from kathleencollins to send copies
of the DVD to the colleges may be the best idea. You
could include a cover sheet that includes a photo and
other info that would be informative for the recipients.

Email providers have filesize limits that would
prevent large files from being sent. The following
article offers an idea of what those limits are:

Pushing the Envelope? Webmail & ISP File Size Limits
http://www.apartmenttherapy.com/click-image-above-for-fullsize-140393
(scroll down to : 'ISP and email provider file attachment size limits')

However...

If you wanted to email 'short' clips of the DVD...the
freeware VLC media Player can Record the DVD
content as it is being played. A 30 second .mpg clip
would be approximately 16 MB.

(The filesize of the .mpg clips could be reduced even
more if they were converted to the .wmv format using
a freeware converter like Format Factory)

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)
VLC Media Player
http://www.filehippo.com/download_vlc/
(runs on XP/Vista/7)
(the filename to download is : vlc-2.0.1-win32.exe)

See the following article:

VLC 1.0 Records Video from DVDs
http://lifehacker.com/5318807/vlc-10-records-video-from-dvds

I realize the article refers to an earlier version of VLC
but the steps are still the same...it can record any portion
of any video while it is playing in VLC.

After recording...find your clips in your My Videos folder
with filenames similar to the following:

vlc-record-2012-04-05-01h07m16s-dvd____F__-.mpg
vlc-record-2012-04-05-01h13m57s-dvd____F__-.mpg

(you can rename these to anything you like as long as
you retain the .mpg extension)

For longer clips that would exceed the sending limits
of your email provider you might consider uploading
the clips to the following site and then send the download
link to your recipients.

"SendSpace"
http://www.sendspace.com/

It's very simple...on the SendSpace site the following
steps are all you have to do.

Left click the Browse button and it will open a window
that allows you to access the files on your hard drive.

Navigate to the folder your videos are in and select one.

Left click the Upload button.

When the Download Link appears...left click the Copy
button. (this copies the link to your Windows Clipboard)

Paste the link in an email message to your recipients.

When the recipient goes to the site.

(***these steps are important so the recipient downloads
the correct clip and doesn't get re-directed somewhere
else by clicking the wrong link***)

Scroll down slightly and left click the "blue text":

**"Click here to start download from sendspace"**

Save the file to desktop.

lemp say:

If you have Microsoft Edge, you must have Windows 10, so I will move your question to the proper forum.

Unless your computer is having a slightly early Halloween, the only thing I can think of that would produce a very loud high pitched "scream" is a failing bearing on a fan (or much less likely, on a hard drive).

Is the reason that the update didn't complete is because you shut the computer down?

Did the screaming resume when you started the computer again?

Is the error message you received:

Error: The application has failed to start because the side by side configuration is incorrect please see the application event log or use the command line sxstrace.exe tool for more detail

If so, you may be able to fix the error by installing the 2008 and 2010 versions of the Visual C++ Redistributable Package as explained here:  https://answers.microsoft.com/en-us/windows/forum/windows_7-pictures/error-the-application-has-failed-to-start-because/df019c0d-746e-42d0-ad68-465e18e3f3ef  Or you might have to dig deeper as explained here:  https://blogs.msdn.microsoft.com/cesardelatorre/2011/03/27/the-application-has-failed-to-start-because-its-side-by-side-configuration-is-incorrect-error-related-to-mmc-exe-programs-and-weird-cause-simple-solution/

However, given the "scream," unless you are willing to open your computer case and check and possibly replace a fan, I suggest that you take your computer to a local independent computer repair shop.  Don't go to a chain store, a big box store, an electronics store, or an office supply store.  If a fan has indeed failed (which "screaming" followed by silence suggests), your computer is at risk for overheating and subsequent further hardware damage.

If it came with WIndows 10 installed then it has the Product key permanently embedded in BIOS which will reactivate whenever you reinstall Windows - unless the motherboard was replaced in which case the Key will be gone anyway and you need to buy a new one.
