Powerpoint, French and Spanish language accents - Microsoft Community

michel tonneijck say:

Powerpoint, French and Spanish language accents

Adding accents é è ê ñ à ç È Ñ ô etc. seems virtually impossible in Powerpoint unless you use the bothersome Insert Symbol or Alt-Numkey combinations.

These are solutions if you type one French word a year, but not a whole text or presentation.

After hours of searching internet I found one 50% answer, which helped to find the rest of it.

It seemed a shame that Microsoft ignores this problem, but actually they have a keyboard option thats suits the problem using only two keystrokes for accented letters.

My recommended option is now and I hope it will help many:

1. In Windows Settings (Time & Language) install French language (or Spanish) and combine it with the US - International keyboard (for my own language and for English, I use the US keyboard, which doesn't help much for French or Spanish in Powerpoint)

2. Make sure that when you use Powerpoint you also select French as default with the US - International keyboard (easy to switch to this language-keyboard combination in the taskbar, once having set the language button in the taskbar, or use the Windows key + Space to toggle between keyboards and languages)

3. typing accents above letters has become very easy now: some examples:

   - you will get è by typing ` and e

   - you will get Ñ by typing Shift ~ and shift N

   - you will get ê by typing Shift ^ and e

4. There are specific language letters and symbols that require a different approach, the Ç and ç: you will get these using the suddenly important AltGr key (to the right of the Space bar) in combination with the <, key
    This AltGr key will also bring you the é (AltGr+e)

                                                               á (AltGr+a)

                                                               ¿ (AltGr+?)

5. Since it is a Windows solution, this option will also work for all other applications, not restricted to MS Office only.

6. Adavantage is also, that you - not being French or Spanish - can stick with the QWERTY layout that you are used to and you do not have to adjust or switch to the AZERTY keyboard.

7. This AltGr key may also help you in other languages (e.g. umlaut or ß for German) but that is not for this topic.

if data in sheet1, column b matches data in sheet 2, column b then i need it to pull the cost from sheet2, column e and place it into sheet1, column d. How do i do this?


by putting this formula in cell D2 of Sheet1:

 

=VLOOKUP(B2,Sheet2!B:E,4,0)

 

or better still:

 

=IF(ISNA(MATCH(B2,Sheet2!B:B,0)),"not found",VLOOKUP(B2,Sheet2!B:E,4,0))

pete_uk say:

Try it like this:

 

=VLOOKUP(B4,MASTERPRICE!B$3:E$271,4,0)

 

If you still get #N/A then B4 is not present in B3:B271 of the MASTERPRICE sheet. You might think there is a match, but you may have spaces in the entries of your table (if text) that aren't in B4, or vice-versa. You can check for these by using a formula like:

 

=LEN(B3)

 

in the MASTERPRICE sheet and copying this down, and using a similar formula looking at B4 of the first sheet. This formula tells you how many characters you have in the cell (spaces not being very easy to count).

 

Or, you may be using numbers and one set is actually text values that look like numbers. To avoid this, try these variations:

 

=VLOOKUP(B4&"",MASTERPRICE!B$3:E$271,4,0)

or

=VLOOKUP(B4*1,MASTERPRICE!B$3:E$271,4,0)

 

The first one converts the number in B4 into a text value, the second converts a text value which looks like a number in B4 into a proper number.

 

Hope this helps.

 

Pete
