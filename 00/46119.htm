ronnie vernon say:

Hi

What you are seeing is by design.

In Windows 7, the Documents and Settings and All Users folders (as well as many other folders) are no longer used.

All of the data in the former C:\Documents and Settings folder is now stored in the C:\Users folder.

The data formerly stored in the C:\All Users folder is now stored in the C:\ProgramData folder.

 

The folders called Documents and Settings and All Users that you see in Windows Explorer are called Junction Points. They have a folder icon with the Shortcut Arrow. These are only there for backward compatibility purposes and cannot be used like a normal folder. They all have the permissions set to Everyone=Access Denied.

 

Tip: Please do not attempt to change the permissions on a Junction Point. Doing so can lead to problems that may require you to reinstall Windows 7 to recover.

 

Here is a link with more information (Advanced).

Application Compatibility: Junction Points
