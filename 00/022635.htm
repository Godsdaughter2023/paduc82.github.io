Thank you for posting your concern on Microsoft Answer Community.

The Windows Media Player in Windows 10 does not have an option to create a Library other than the existing libraries. However, you can follow the steps below and check if it works for you.

Steps: Create a folder for Audiobook and add to Windows Media Player.

  1. Create a folder on the computer and for example name it Audiobook.

  2. Please save all the Audiobook file under this folder.

  3. Open Media player from Start Menu.

  4. Click the Organize button and choose Manage Libraries from the drop-down menu.

  5. Click the Add button, select the folder with your files, click the Include Folder button, and click OK. This will add all the files to the media player.

  6. Right click on Music from the left pane, from the dropdown select “Customize navigation pane”.

  7. In the new window, under Music check the Folder box and click Ok. This will create a category under music.

chip pearson say:

If your cells are not contiguous in a manner such that you can't reference multiple cells with one reference (e.g., A1:A10), you can group cells in parentheses. Each set of parentheses can have up to 31 cells and you can have 31 sets of parentheses, giving you about a thousand cell references. E..g,

 

=SUM((A1,A2,A3),(A4,A5,A6),(A7,A8),(A9,A10))
