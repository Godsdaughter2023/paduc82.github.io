husham---01 say:

Internetcalls.com SIP settings for my nokia n95 8gb

I have searched up and down for people to give me some feedback or some comments/help on settings up internetcalls.com as a sip for N95 8GB but found verylittle in the important sites. so when I found the solution i thought its best to be posted here to help others.

 

This guide is based upon using InternetCalls.com.


You will need:

A Nokia S60 phone with the Internet Telephony application installed
An account with a internetcalls.com
Access to a WIFI hotspot or unlimited 3G bundle

What to do:

01) Go to Menu > Tools > Settings > Connection > SIP settings

02) Press the "Options" key then "New SIP profile" then "Use default profile" ( Dont ammend a current profile try to start from scratch)

03) Give the profile a meaningful name (I called mine InternetCalls)

04) Leave "Service profile" set to "IETF"

05) Set the "Default access point" to the data bearer to use to make calls

06) Set "Public user name" to your SIP username (for InternetCalls.com, you must use @internetcalls.com where is your username. ( in my case it was *** Email address is removed for privacy *** )

07) Set "Use compression" to "No", "Registration" to "When needed", "Use security" to "No", leave "Proxy server" alone. I set my registertion to always on but that can be a your choice and can be changed later.

08) Enter the "Registrar server" sub-menu and enter the server address (in the form sip.provider.com). In my case I typed sip.internetcalls.com

09) The "Realm" is simply the domain name without prefix (internetcalls.com,)

10) Enter your username and password as provided by your VoIP provider

                User name : hushammemar33

                password : xxxxxxxxxxxx

11) Leave "Transport type" set to "Auto" and enter 5060 in the "Port" box then press the "Back" button 3 times until you are returned to the "Connection" Menu

12) Go to the "Internet tel." option in the "Connection" menu

13) Press the "Options" key then "New profile"

14) Give the profile a meaningful name (I used InternetCalls) then set the "SIP profiles" option to the profile you have just created (in my case, also called InternetCalls).

15) Return to the home screen then go to Menu > Tools > Connectivity > "Internet tel." >

The handset should say "Discovering Services" then present you with a list of access points. One of these should be the one you defined in step 5. To connect, simply click on the connection name. You will then get a message along the lines of:

"Connecting to VOIPPROVIDER using ACCESSPOINT" then "Connected to VOIPPROVIDER". You should also notice a new icon in the top right hand corner, a globe with a telephone.

You are now connected and should have a new option in your contacts for "Internet Call" under "Voice Call" and "Video Call" (if enabled).

Now just select a contact, select "Internet Call" and you should connect to your contact using VoIP.

Skype for Business is part of Office. Generally the way to remove it is to uninstall Office. You also have the option of just not having it in startup (this can be controlled through the settings gear in the application) or renaming the lync.exe executable. Some people also utilize the Office deployment tool to reinstall Office excluding Skype for Business. 

Removing Skype for Windows 10

The version of Skype that keeps appearing is probably the Skype for Windows 10 App. You can uninstall it from Apps & Features. Appwiz.cpl is Programs and Features and only applies to Desktop applications, not UWP Apps.  You can also try to uninstall it by right clicking on it and selecting Uninstall. If the program keeps reinstalling when new users sign on or something specific to the build of Windows 10, you can try my removal tool (SRT (.NET 4.0 version)[pcdust.com]) by selecting Skype for Windows App and clicking remove. Keeping Skype for Windows 10

Another option is to stop it from loading at startup. If you do not want it to startup, you can stop if from launching by signing out every time. This will ensure that even if components of Skype are loaded in the background, they will load in a suspended state. If you do not want any components of that version to load, you can try to stop it from Startup Apps or disable the Skype App in your Background apps located in your Windows 10 Privacy settings.

Setting up call forwarding:

1. Sign In to Skype Manager.

2. Click on Features in the toolbar.

3. In the Features menu on the left, click Call Forwarding to display the Call Forwarding page. Only members with business accounts will be listed.

4. Check the box next to each member to whom you wish to setup call forwarding. When you select a member, options for setting up call forwarding are automatically displayed.

5. If you want calls to the selected members forward to a specific number, select the country for the number from the drop-down list displayed, then enter the number in the field displayed.

6. If you want calls to the selected members forwarded to a specific Skyoe Name, select Skype Name from the drop-down list displayed, then enter the Skype Name is field displayed.

7. Click Add.

All calls to the selected member will be forwarded to the nymber or skype that you specified.
