You should make an image of your hard drive and restore the image to the new hard drive. This will be straightforward for you if you are familiar with making and restoring backup images. If not, have no fear:

slance310 say:

Hi,

As to the question... Do I create a system image or do a back up under Windows 7 backup/restore option ?....   They are the same thing...

The problem with using this feature is that you are going to a different size drive...

This tool will not resize your partitions, and if it did, maybe not to your liking...

There is also the possibility that you will end up with a EFI or Recovery Partition after your OS partition that will make it difficult to expand your C:\ into the unused space...

You may also run into some of these same issues with a free 3rd party imaging program... Some of the "Free" programs will not allow the rearranging of partitions when restoring an image...

Some of the programs that are available with the purchase of a HDD will not even create an image, but will only "Clone"... Which will end up with possibly the same problem as using the Windows Imaging feature...

These programs will also require that they are installed, which leaves you with a program on your machine that you really don't want...

The best thing to do would be to search for a free disk management program that downloads as a bootable ISO that can be the put on a CD or Flash Drive... Make sure it will allow you to relocate partitions and\or resize them...

I personally use Acronis TrueImage which comes on a bootable CD, or you can download a bootable ISO for versions older than 2017...

This is the same program that may be furnished by a HDD manufacturer, but without the restrictions, and not requiring installation...

Most imaging programs require bootable media to reload the image anyway... You can create the image from within windows, but not reload it...

Simply "Cloning" may not accomplish what you want, but since you will still have the original drive, you can experiment as needed...

I wasn't planning on posting all this stuff, but once I started I felt obligated to warn you of some of the shortcomings of some of your options...

gerry c j cornell say:

Matteo

 

You are doing things which will not help resolve your problems and in many cases will make the situation far worse.

 

"tried to disable many drivers". Don't do it! You will stop devices working.

 

"Ati radeon hd 4200 that I've seen freezes are often due to (once I tried also to delete its drivers)." You should not delete drivers, especially those relating to graphics. If you succeed you will not see anything on your monitor and as a result not be able fix anything.

 

"clean the registry" Totally unnecessary and capable of creating insoluble  system errors that can only be fixed by reinstalling Windows 7.

 

"disable most of the start up programs". Not a smart idea if you want a working computer. You need a good reason to disable a start up item. 

 

I will now try to help you undo the damage and solve your original problem.

 

Click on the Start button and select Control Panel, Administrative Tools, Services. Locate Windows Management Instrumentation in the list. Check that it has started and that the StartUp type is Automatic. If it is not set to Automatic right click and select Properties, General tab. There is a box after StartUp type. The options are available on a drop down by clicking on the arrow down to the right of the box. Make the change and click on Apply and OK. Restart the computer and check whether it has started.

 

Please let me know when you have completed this task.

 

The next time you encounter  a freeze select Ctrl+Shift+Esc to bring up Task Manager and click on the Applications tab. Place the cursor on a running application to select it and click on the End Task button. As the application is removed from the list repeat the procedure on the next application in the list until all are gone from the list. You may need to be patient as a freezes in this situation means that your computer is trying to carry out too many tasks at one time and it takes time for Task Manager to be able to intervenue and end the task causing the freeze. Pressing the Power button can seriously damage files on your computer as the files can corrupt and cease to be usable.
