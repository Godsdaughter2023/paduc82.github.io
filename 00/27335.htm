I didn't understand your question exactly but as you may know, using the tab key is the best key to jump between the fields (cells in a row) one by one.

rohn007 say:
In Win 10 the easiest way to arrange application windows is using the <WIN><Arrow> (all 4 arrows) to "Snap" the application windows around the screen. First one, then the other.

In Excel there is an additional consideration. EXCEL is a "MDI", multi document interface, user interface. You can have multiple spreadsheets open inside of a single application window. Go to View Tab > Zoom group > click on Arrange All drop down. Select Horizontal. This will arrange any open spreadsheets to fill the application window, stacked on above the other. If there is only one spreadsheet open, it will fill the whole window.
