daniel g. grau say:

Like Beth suggested, your document is vertically justified, so it will occupy the whole page. Vertical justification is a Section attribute, so you must disable it for every section (or just do Select All before disabling it).

Given such conditions, widow/orphan control will have a bearing as to what page the extra space shows in, because it sends a line to another page. This causes one line less to appear on the page and vertical justification does the rest to "compensate" for the page-shifted line.

beth melton mvp say:

I'm glad to hear it was something that simple, because if it wasn't vertical justification we'd likely never solve it. Vertical justification is the only thing that explains why turning off Widow/Orphan control would make it better and why the space between the paragraphs kept changing. In your screenshot, it appeared to have roughly 15 pts of space between paragraphs. That's not typical and is an indication of Word is automatically controlling the space in order to balance the text between the margins.

rajesh govind say:

 

Hi,

 

You can definitely copy a part of an email and then paste on any text editor program like Microsoft Word and then you should be able to print it from there. I would like to ask this question regarding this issue.

 

Are you using Internet Explorer browser to access the email?

 

You may follow these steps and check if you are able to print:

 

a)       Select the portion of the email using the mouse and then press “CTRL” + “C” keys on the keyboard to copy it.

b)       Now, open the text editor program (Microsoft Word), click on it and then press “CTRL” + “V” keys on the keyboard to paste the copied portion of the email on the text editor program.

c)       You may now edit the text, if required and then press “CTRL” + “P” keys on the keyboard to print it.

 

Hope this information is helpful. If you face any difficult in performing the above steps, please do get back to us with the details.
