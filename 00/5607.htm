Sub CopyDataToNewSheet()

  ActiveSheet.UsedRange = Data

Do you NEED or simply WANT 64 bit Office (to "match" 64 bit windows)?

The main reason for NEEDING 64 bit Office is to take advantage of the file support for files that are larger than 2GB in size or uses more than 2GB of RAM. This is typically large business related spreadsheets.

MS recommends that everyone still uses 32 bit Office unless you have a specific need for 64 bit.  The main reason is that many addons still have NOT been converted to 64 bit. This includes a bunch from MS!
