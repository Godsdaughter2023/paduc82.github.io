lastgas say:

File history is the worst idea from Microsoft ever

It sounded good. Make a backup of your files when they change. It's easy to find something and restore it. Only in practice it's a disaster.

It's a disaster from the design point of view and from the implementation.

Design problems:

1. File History makes lots of copies of very large files. The default is to back up your files every hour. About the biggest file I have that needs backing up is my archived emails. They goes back for years, and have attachments. It changes hourly. So in a month, I have 720 copies x 3.25 GB.

2. It's difficult to know what is backed up. I use the Thunderbird email client. It puts its data in X:\Users\me\AppData, which by default isn't backed up with File History! How many users know where that hidden folder is? Imagine the shock when one finds their emails not backed up!

3. The user interface is a mess. On Windows 10 Home you can't set your backup to a network drive in Settings, but you can in Control Panel. The Stop Using Drive and Add drive are not on the same screen.

4. There is a one-size fits all limitation on the retention of backup copies. Some files with frequent changes shouldn't be backed up as often.

5. There is no progress indicator, a step back from Windows 7.

Implementation:

1. On two of my computers, the drive filled up with File History repeatedly making copies of the same unchanged file over and over. I'm deleting 49,000 of those right now. On top of that, I have 16,000 entries in my Favorites backup folder. I have 843 copies of the same Skype tracing file. I don't even run Skype on this computer! I got the repeated files to stop on one machine, but another one keeps going with the duplicates.

2. The user interface is buggy, giving wrong information about the status of a backup. It's supposed to offer a Cancel button in Settings while a backup is running, but sometimes it shows BACKUP NOW at the same time it says the backup is running. 

3. If a  backup fails, chances are you don't get notified. I think in general the error handling is inept.

4. When you do try to add a drive, chances are you get a "No Usable Drives" found error in File History Settings, even though there are drives available that are listed under the Control Panel File History.

So users don't know what's being backed up. What is being backed up is probably backed up at an inappropriate time. The whole thing is buggy and unreliable. It's a disaster. I've fired people for better work than this.

  1. Press Windows + X keys, select Command prompt (Admin) to bring up elevated Command prompt.

  2. Stop the BITS service, the Windows Update service, and the Cryptographic service. To do this, at a command prompt, type the following commands. Make sure that you press Enter after you type each command.

    • net stop bits

    • net stop wuauserv

    • net stop appidsvc

    • net stop cryptsvc

  3. Rename the software distribution folders backup copies. To do this, at a command prompt, type the following commands. Make sure that you press Enter after you type each command.

    • Ren %systemroot%\SoftwareDistribution SoftwareDistribution.bak

    • Ren %systemroot%\system32\catroot2 catroot2.bak

  4. Restart the BITS service, the Windows Update service, and the Cryptographic service. To do this, at a command prompt, type the following commands. Make sure that you press Enter after you type each command.

  • net start bits

How do I add Parents and Computers to Family Safety?

Here is the problem. We want to be able to have either one of us log on to the family safety site and control the settings for my child - authorize access, etc while we are at work. When I log on to Family Safety using my Live ID I can see both computers and my child's account, but I cannot see anything about my wife. When she logs in using her Live ID she can see her account and our child's account and ONLY her computer. She cannot see the family computer or me as as user.

I am at my wit's end trying to figure out how to give us both access to both computers.

Please help!

leonard_su say:

Hi ChuckSena,

 

We appreciate the details you have provided and we see that you are trying to setup two primary parent accounts to manage a child account in Family Safety. At this moment this is currently not available and it is part of the limitation of Family Safety.

 

However, we were able to produce a workaround. The best way to manage your child account on both PCs is to set the primary parent to manage the child and use a secondary parent account to manage the child account on your wife's computer.

 

To do this, you will need to configure the primary account on your (Family computer) to manage the child account.

 

After that perform a clean re-installation of Family Safety on your wife's computer to remove all her credentials so you can add her as a secondary parent account to your primary parent account.

 

Here are the steps on how to purge the Family Safety settings on her computer and at https://familysafety.microsoft.com.

 

1. Uninstall Family Safety by following the instruction in this article: How do I remove Family Safety?

2. Remove all the current members in your account:

a. Sign in to https://familysafety.microsoft.com using the parent account. (Wife's Microsoft account)

b. Click the option Manage the list of family members, it is located on the bottom part of the page.

c. Select all the members and click the Remove button and it will prompt you to confirm the action.

d. Click Remove button again to update the changes and it will automatically sign you out in Family Safety website.

3. Reinstall Family Safety. You can download the latest version of Family Safety from this link.

 

After purging the settings in your wife's account, please associate her account to the primary parent account (Your Microsoft account you used on Family computer). To do this, follow these steps:

 

1. Sign in at https://familysafety.microsoft.com using the primary parent account (Microsoft account you used to setup on Family computer).

2. Click the Add a new parent link in the bottom part of the web page.

3. It will ask you Does this parent have a Microsoft account? click Sign In then sign in using the wife's account.

4. Once it has been successfully added as a parent account. Please go back to your wife's computer and establish the child account that you want to manage.

 

To re-establish the child account in Family Safety client, please follow the instruction in this solution article. (Please use the secondary account (Wife's Microsoft account) to manage the child account on your wife's computer).

 

Note: Both of you can now manage your child's account via https://familysafety.microsoft.com.

 

If you need further assistance, please don't hesitate to reply back to this post.

 

Thank you.
