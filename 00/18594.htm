It could occur if the profile is corrupt in Outlook 2003 and also due to some add-ins.

Step A: Start Outlook in safe mode

=============================

Starting Outlook in safe mode will disable all the add-ins in Outlook to test if any add-ins are causing the issue

1.       Click Start >In the search tab type Run > Click on the Run from the list

2.       Type "Outlook.exe /safe" and click OK. (Without quotation marks) Note: There is one space between Outlook.exe and /safe.

3.       Does the problem still persist in Safe Mode? If it does not please follow the next step to disable add-ins.

 

 If the step one works, try step2 as mentioned in the post

 

Step B: How to disable add-ins in Outlook 2003

1.       Click Tools menu > click Options > under other tab > click Advanced Options

2.       Click Add-in manager > uncheck all the add-ins.

3.       Click COM Add-ins > uncheck all the add-ins.

       If you are not able to disable any of the add-ins then try the following:

Registry disclaimer: Modifying REGISTRY settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of REGISTRY settings can be solved. Modifications of these settings are at your own risk. 

4.       Click Tools menu > click Options > under other tab > click Advanced Options

5.       Click Add-in manager > uncheck all the add-ins.

6.       Click COM Add-ins > uncheck all the add-ins.

7.       If you are not able to disable any of the add ins then try the following:

8.       Click on Start and then in the start search box type “regedit” (without quotation marks) and then press the Enter key.

9.       Now in the registry, let’s take a backup by clicking on File > Export > (under Export Range select ALL) and then click on Save.

10.   Now please delete the following keys (Ignore if you don’t find or if you are not able to).

13.   HKEY_LOCAL_MACHINE \Software\Microsoft\Exchange\Client\Extensions (remove Extensions)

14.   Restart Outlook and check if it works.
