jay freedman say:
There are several kinds of styles. The kind you're talking about is a paragraph style. Another kind, a character style, applies only to the selected text (as little as one character) and does not affect the rest of the paragraph. Read Word's Character Styles for more info.

If you use the "Save Selection as a New Quick Style" item on the Styles gallery and just click OK in the little dialog where you enter the new style's name, you'll get a paragraph style -- which cannot be changed later into a character style. If you go through that dialog, you must click the Modify button instead of the OK button; in the large dialog, you change the Style Type dropdown to say Character.

An alternative is to open the Styles pane and click the New Style button at the bottom left of the pane, which gets you to the same large dialog (and again you have to change the Style Type setting).

techfreak_ say:

Got "Start"- "Run" (or Win key +R) and type in "msconfig", then click "OK".  Click the "Startup" tab.  Locate anything referencing Skype and uncheck it.  Click "OK", then reboot your computer.  If you reboot and Skype still starts or Skype is not on the list, then another program on the list or off it, is starting Skype.

If you are unable to determine what program that is, a workaround would be to simply rename your main Skype executable (Skype.exe) to something different like Skype1.exe.  That should stop Skype from starting when another program calls it.  You can then update your Skype shortcut to point to Skype1.exe.
