VBA to change color when cell updated - Microsoft Community

nagarajkp say:

VBA to change color when cell updated

I am tring to change color of  the cell in range("A10:F100"), when ever a cell in the range(which has as formula and is linked to other worksheet)is updated from previous value due to change in value of linked cell in other worksheet. Please help in this regard
ossiemac say:

Actually D column contains component Name and Column E15: I00 has formula like

=OFFSET(Components!$A$1,42,MATCH(D17,(Components!$A$9:$CM$9),0))

which is linked to component worksheet.

I am looking for vba which changes background color of the cell range( E15: I00) when ever it value changes due to change in component worksheet.

regards

nagaraj


The above is a change to range which you indicated in your original question. However, because you have indicated that you only want to know which cells have changed since opening the workbook then it can be done as follows.

  • For the description, the worksheet where you want the color highlighting I am calling "Main Sheet"
  • Add a worksheet to the workbook. (For description purposes I am naming it "Copy of values")
  • In the Workbook_BeforeClose event, copy the values from your  "Main Sheet" to the "Copy of values" worksheet. (See the example VBA code below).
  • On your "Main sheet" use Conditional formatting to compare the values in "Main Sheet" to the values in the "Copy of values" worksheet and color the non matches.
  • Copy the following code to Thisworkbook module. Edit the ranges to suit the ranges you want to monitor with the color. Copy to a corresponding range in "Copy of values" as per the example VBA code. (Note that values are copied, not formulas).

Private Sub Workbook_BeforeClose(Cancel As Boolean)
   
    'The following line copies the values only from Main Sheet A10:F100 to Copy of Data A10:F100
    Worksheets("Copy of values").Range("A10:F100").Value = Worksheets("Main Sheet").Range("A10:F100").Value
   
End Sub

Note that the user will need to confirm to save changes when closing the workbook due to the changes made with the copying of the values. (Excel pops up an alert re saving.)

The "Copy of values" worksheet can be hidden. If you use VBA code to hide the worksheet then it cannot be unhidden from the interactive mode.

Example code to hide the worksheet:

Sub HideWorksheet()
    Worksheets("Copy of values").Visible = xlVeryHidden
End Sub

Example code to unhide the worksheet:

Sub UnhideWorksheet()
    Worksheets("Copy of values").Visible = True
End Sub

Conditional formatting:

Select the range to be monitored on "Main sheet"

In the Conditional formatting select "Use a formula to determine which cells to format" and then use the following formula where A10 is the first cell of the selected range. Note that the cells are relative format (No $ signs).

=A10<>'Copy of values'!A10

Feel free to get back to me if you have any problems with it.

jay freedman say:
Word does not have a way to restore that special view, except by repeating the original Compare operation (assuming that neither the original document nor the revised document has been changed in the meantime).

The closest you will get is to open all three documents at the same time. In the Options dialog, on the Advanced page, in the Display group, clear the check box for "Show all windows in the Taskbar". Then you will be able to resize each document's window and position them in the same way as in the Compare display. Also, on the Review ribbon, click the Reviewing Pane button to enable it.

However, the documents won't scroll synchronously the way they would in Compare; they're just three separate windows. You'll have to scroll each one manually to the same location.
