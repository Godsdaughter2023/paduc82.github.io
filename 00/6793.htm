Change the Font Size in Outlook's Folder Pane

Many users want to change the font size used for the folder names in Outlook.  Unfortunately, Outlook no longer has an option to change the font size in the Folder List (Navigation pane); this was an option only in Outlook 2010. 

At this time, if you use Windows 10 and need larger text in Outlook's Folder List, you'll need to change scaling % in Settings > Display > Scale and Layout, or change the Font size in Ease of Access Settings > Display settings, either of which will affect all applications.  Raising either setting up to 25% will usually increase the folder fonts enough without messing other applications up.
