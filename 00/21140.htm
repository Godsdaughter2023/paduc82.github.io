As you are using Outlook I might be more inclined to use a script run from an Outlook rule to download the files automatically upon receipt e.g. as follows. Change the path where indicated to the path where you want to save the files.

Option Explicit
Sub SaveSomeAttachments(Item As Outlook.MailItem)
Dim olAtt As Attachment
Dim strFilename As String
Const strFolder As String = "D:\My Documents\Test\Outlook Attachments\"
    If Item.Attachments.Count > 0 Then
        For Each olAtt In Item.Attachments
            If olAtt.Filename Like "PUR2*" Or _
               strFilename Like "VAT2*" Or _
               strFilename Like "TAX2*" Then
                strFilename = strFolder & olAtt.Filename
                olAtt.SaveAsFile strFilename
            End If
        Next olAtt
    End If
End Sub

No manual interference required.
