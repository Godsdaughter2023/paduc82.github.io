hansv mvp say:

If a cell contains a time before 7:00, subtracting 7 hours would result in a negative time. Excel cannot handle that.

You could use

=MOD($C$3:$C$102-7/24,1)

hansv mvp say:
=IF($C$3:$C$103="","",MOD($C$3:$C$102-7/24,1))

"" is an empty string. The formula first tests whether the cell in column C is empty. If so, it returns an empty string. Otherwise, it subtracts 7 hours from the time.
