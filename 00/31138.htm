Scan for a TV code

You can send a sequence of codes to from your remote to your TV to find the right code for your TV, if you do not know it.

   Turn on your TV and point the media remote toward it.

   Press and hold the Mute and 5 buttons simultaneously for two seconds to enter TV programming mode.

   Press the Channel Up button repeatedly to send one code per button press from your media remote to your TV. By pressing Channel Up, you advance the code each time.

If your TV turns off but you then accidently continue to press Channel Up to continue past the correct code, press Channel Down to go back to the correct code. Your TV will turn on again when you go back to the correct code. Press the Select button to confirm your code.

lemp say:

You only need to post once.  In fact, if you post the same thing multiple times, that might decrease the chance that someone will respond.

There probably is a "Stop Code" associated with the Unmountable_Boot_Volume error message.  If the message you get is STOP 0x000000ED (0x<var>aaaaaaaa</var> ,0x<var>bbbbbbbb</var> ,0x<var>cccccccc</var> ,0x<var>dddddddd</var> ), it is describedhere .  Essentially, there are two possibilities.  If the second parameter (0x<var>bbbbbbbb</var> ) of theStop error is 0xC0000032 , the cause of the error is that the file system is damaged. Otherwise, it's supposedly a problem with the cable connecting your hard drive to your motherboard.

If the computer was on during the power outage, there's a reasonable possibility that the file system is damaged.  If you have access to any Windows XP CD (not necessarily the one that came with this computer), you can boot to the Windows Recovery Console and use chkdsk /r as described in the article linked above.

I've also seen suggestions that executing the "fixboot" command from the Windows Recovery Console has helped on occasion.

See: Description of the Windows XP Recovery Console for advanced users

If you don't have a Windows XP CD, you can still get the Recovery Console.  Microsoft provides a set of 6 floppy disks that include the Recovery Console -- but these days odds are that you don't have a floppy disk drive.  There are various sites on the Internet that have iso files for a bootable CD including the recovery console. This is one (I haven't used it).

Finally, you are correct to be concerned about your data.  Even the MS KB article first linked above warns that running CHKDSK /R "may" result in data loss.   To be most conservative, remove your hard drive from the Dell, put it in an external enclosure (these can be quite inexpensive; just make sure that you get one with the correct interface for your drive -- SATA or parallel ATA), connect it to a working computer and copy off the files you really don't want to lose. 

In fact, I won't delete everything I just wrote, but I realized as I wrote the last paragraph that if you put the drive in an external enclosure you can run the disk manufacturer's diagnostic utility and/or chkdsk /f from the working computer rather than go through all of the nonsense to get to the Recovery Console.
