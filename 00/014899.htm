Fix 2: Disable hardware acceleration (This fixed MY problem!)

1. Run regedit (Win + R ; "regedit")
2. Browse to HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Common
3. Create a New Key and name it "Graphics"
4. Select Graphics, right-click on the right panel and create a New DWORD (32-bit) Value and name it DisableHardwareAcceleration.
5. Enter Value data as 1
Try to run Excel normally now.

Backup files in earlier (much earlier) versions of Word had the .BAK file extension; currently the backup fille will be named "Backup of <filename>.wbk." The backup file is not created until the second time you save. Its content is that of the first saved version. Each time you save subsequently, the previously saved version is renamed as the backup.
