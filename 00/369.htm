paulcordts say: The recorder can only do so much.  You have to do a little tweaking to make a recorded macro flexible.

Your macro may look something like this

Sub Macro1()
   Selection.End(xlToRight).Select
    Range("J1").Select
    ActiveCell.FormulaR1C1 = "=SUM(RC[-9]:RC[-1])"
    Range("J1").Select
    Selection.AutoFill Destination:=Range("J1:J18")
    Range("J1:J18").Select
End Sub

The issue is that the recorder captures the current condition  (In this case data in A1:I18).

You need to analyze and modify the macro 

 Selection.End(xlToRight).Select   THIS LINE IS OK ASSUMING YOU START IN A1
    Range("J1").Select  THE SECOND RIGHT ARROW - THIS IS SPECIFIC
    ActiveCell.FormulaR1C1 = "=SUM(RC[-9]:RC[-1])"  
    Range("J1").Select AN UP ARROW TO GET BACK AFTER ENTERING THE FORMULA - AGAIN SPECIFIC
    Selection.AutoFill Destination:=Range("J1:J18")  BASED ON THE RANGE OF CURRENT DATA - SPECIFIC
    Range("J1:J18").Select BASED ON THE RANGE OF CURRENT DATA - SPECIFIC


This is the flexible version

Sub Macro1()
    Range("A1").End(xlToRight).Offset(0, 1).Select
    ActiveCell.FormulaR1C1 = "=SUM(RC[-9]:RC[-1])"
    Selection.AutoFill Destination:=Range(Cells(1, ActiveCell.Column), Cells(Range("A1").End(xlDown).Row, ActiveCell.Column))
End Sub

Range("A1").End(xlToRight).Offset(0, 1).Select
This starts with A1 regardless of where the cursor is and the Offset(0,1) moves over 1 column.

 Range(Cells(1, ActiveCell.Column), Cells(Range("A1").End(xlDown).Row, ActiveCell.Column))

Cells(1,Activecell.column) is the first row of the column now selected.
Cells(Range("A1").End(xlDown).Row, ActiveCell.Column) is the last row (This assumes that column A has no empty spaces)

Just change the formula to whatever you need.
