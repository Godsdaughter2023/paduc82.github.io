stefan blom say:

In the Table Properties dialog box, see if the table width is set in percent. In that case, the table as a whole would resize when you paste it into a new document (depending on the width of the "container" into which you are pasting).

You can set a fixed width for the table before copying it and then it will also paste in that width. To do so, click inside the table and, on the Table Tools Layout tab, click AutoFit, and then click Fixed Column Width.

Yes, I was able to do so with my personal VoIP/SIP over TLS service and its self-signed certificate.

  1. took my certificate authority file not as Base64 (PEM) but as binary file (DER)
    because that is required in Nokia Symbian/S60
  2. changed the file extension to .cer
    because that is required in Windows Phone
  3. put that file on my web-server with the MIME media type application/x-x509-ca-cert
    because that is required in Nokia Series 40

That way, my certificate file worked in my Nokia Asha 501 with the new Asha Software Platform 1.1, as well. However, please forgive me, I did not test, what other options exist (PEM, file extension, Bluetooth). Do you succeed?

If you have any concerns, suggestions and complaints regarding Windows 10, Windows feedback hub is the right place for you. If you haven't been introduced to Windows 10 feedback hub yet, we encourage that you submit your issue using this application. You can either vote on an existing submission or submit a new issue. In order to submit a new feedback, please follow the steps below:

    4. Important: If possible, reproduce the problem(s) after clicking Begin monitoring (or Start capture) near the end of the form; Stop monitoring when you're done.

    6. Click Continue using Feedback Hub.
    7. Click My feedback at the top, then find the item you submitted (it may take a while to appear).
