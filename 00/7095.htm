doug robbins - mvp office apps & services (word) say:

While you can make changes to the appearance of the text by the use of direct formatting such as increasing the font size, applying bold or italics, or changing the color, it is better, particularly if you have many instances of where you want to apply the same attributes, to define the necessary characters styles for each of those sets of attributes.

When defining a new style, you can base it upon one that has already been created so that for instance if you had a blue style already defined, any style that you create that is based on that style will inherit the blue color.  Then, if you decide that you want to change all of the blue to green, if you make that change to the base style, it will flow through to each of the styles that are based on that style AS LONG AS the color or the font in those styles has not been changed, either when creating the style, or my manually changing it.

How to disable JUNK filter on Outlook 2016 MAC - Workaround - Microsoft Community

seimn say:

How to disable JUNK filter on Outlook 2016 MAC - Workaround

Junk filter in Outlook 2016 for MAC has usually far too many false positives.

If you receive a few junk emails over the week, it is far more work than actually helping you.

You can't disable the junk-email folder in outlook 2016 so far (Version 16.13.1 (180523)

See: https://answers.microsoft.com/en-us/msoffice/forum/msoffice_outlook-mso_mac-mso_o365b/how-to-disable-junk-filter-on-outlook-2016-mac/28e2a3e1-95fe-489c-9692-765e4ef3d07d

This thread "How to disable JUNK filter on Outlook 2016 MAC?" is locked (for whatever reason)

If you like to accomplish the same task as disabling JUNK Filter you can create a ruleset as below:

Tools -> Rules -> Client Rules -> Exchange -> Add:

When a new message arrives (if all conditions are met):

Is Junk E-Mail

Account Is: (your account "A")

Do The following:

Change Status: Not Junk E-mail

Move Message: Inbox (account "A")

(you can create these rules for all your accounts)

hope that helps

Simon
