tina zhou msft say:

Hi Robert,

Thanks for the query.

As the files are inside the folder and per your requirements you would like to set different permission level for the affected users or groups for the files, the permission inheritance is stopped. You will need to stop the inheritance for every file in the folder one by one and assign Edit permission level to the affected users or groups. We can not achieve your requirements.

In the perspectives of manageability and performance, and cost of maintenance, I'd like to suggest the below workaround:

1. Create a sub folder inside the folder, stop the permission inheritance, and assign Edit permission level for the affected users or groups.

2. Files inside the sub folder will inherit the Edit permission level.

Okay that was really close.  I guess I just needed some caffeine.  I got it! This will mostly give the desired behavior.  For some reason if you close Excel with pending changes, it prompts you to save, and you tell it yes, it will use the save as dialog and ask if you want to overwrite the file, but it will default to the same file name. 

 

What I needed to do was manage the sub indirectly calling itself again when it calls ActiveWorkbook.Save.   

 

Public WithEvents App As Application

 

 

Private Sub App_WorkbookBeforeSave(ByVal Wb As Workbook, ByVal SaveAsUI As Boolean, Cancel As Boolean)
    If ((Wb.FileFormat = xlCSV) Or (Wb.FileFormat = xlCSVMac) Or (Wb.FileFormat = xlCSVMSDOS) Or (Wb.FileFormat = xlCSVWindows)) Then
        If Application.DisplayAlerts = False Then
            Cancel = False
        Else
            Application.DisplayAlerts = False
            ActiveWorkbook.Save
            Cancel = True

            Application.DisplayAlerts = True
        End If
    End If
End Sub

The keys issued with Insider builds in 2015 no longer work.  In 2015 MSFT introduced "digital entitlement"  (search online).  If you upgraded at that time on a device which had an activated copy of Windows 7 or 8.1 then that edition of Windows 10 will automatically activate if you re-install it on a new hard drive.  If you did not have an activated copy of Windows 7 or 8.1 to upgrade from you need to purchase Windows 10 when you reinstall.  Simply install Windows 10 on your new hard drive and see what happens.  Nothing to lose.
