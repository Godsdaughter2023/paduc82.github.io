considerate_guy say:

That isn't actually a movie file. The .mswmm file is a *project* file. It only contains the instructions on how to assemble your movie but it doesn't have any media within it.

Unless you still have all the original media files (photos, videos, music), the project file is quite useless. It's like having a recipe but no ingredients.

If you want to try opening the project file, download Windows Movie Maker 6.0. It understands the contents of a project file. However, if you notice that your clips are all showing with Red Xs, that means it can't find any of the source media files.

This issue usually occurs because of Boot configuration data (BCD) is corrupted.

The Boot Configuration Data (BCD) store, contains boot configuration parameters and controls how the operating system is started in Microsoft windows.

-          Do you have a copy of windows reinstallation disk?

You may follow the below mentioned methods to fix the issue.

Method 1:

Unplug all the external devices and try booting and check if you are able to boot to the desktop.

If you are not able to boot, Please follow method 2, If and only if you have windows 7 reinstallation disk.

Method 2:

Rebuild BCD

Please follow the following method to rebuild BCD.

  1. Put the Windows 7 media in the DVD drive, and then start the computer.
  2. Press a key when you are prompted.
  3. Select a language, a time, a currency, a keyboard, or an input method, and then click next.
  4. Click Repair your computer.
  5. Select the operating system that you want to repair, and then click next.
  6. In the System Recovery Options dialog box, click Command Prompt.
  7. Type Bootrec.exe, and then press Enter.Collapse this imageExpand this image
  8. Type “/RebuildBcd”.
  9. Please take the disk and restart the computer and check if issue persists.

Note: To start the computer from the Windows 7 DVD, the computer must be configured to start from the DVD drive. For information about how to do this, see the documentation that was included with the computer, or contact the computer manufacturer.
