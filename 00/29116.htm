The most likely cause is that Simple File Sharing is enabled.  The security tab will only appear if simple file sharing is disabled.

"How to disable simple file sharing and how to set permissions on a shared folder in Windows XP"
  < http://support.microsoft.com/kb/307874 >

In the case of the security tab being missing for files, the tab will only appear if the disk the file is on has been formatted as NTFS. (No security tab for FAT volumes).

An alternate way to check file permissions is to use the "cacls" command from a command prompt window.
