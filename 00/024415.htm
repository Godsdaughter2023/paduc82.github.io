Macros with pop up boxes in word

Hi there

I currently have the below macro which highlights certain words that I choose.

I'm hoping there is a way that I can have a pop up box come up which when they run the macro, prompts people to enter the words they want to highlight, separated by commas. For example, if users wanted to search "With" and "Experience of" and "football", they could do this by typing 

with, experience of, football

Once this is done, I'd also like them to be able to run the macro in various documents using the same key words, without having to reenter them every time they want to search a document, ie. once the words are set, then running another macro just performs the search on those already selected words.

Dim range As range
Dim i As Long
Dim TargetList

TargetList = Array("Array", "linkedin", "With", "experience of", "football") ' put list of terms to find here

For i = 0 To UBound(TargetList)

Set range = ActiveDocument.range

With range.Find
.Text = TargetList(i)
.Format = True
.MatchCase = False
.MatchWholeWord = False
.MatchWildcards = False
.MatchSoundsLike = False
.MatchAllWordForms = False

Do While .Execute(Forward:=True) = True
range.HighlightColorIndex = wdYellow

Loop

End With
Next

End Sub

greg maxey say: Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim range As range
Dim i As Long
Dim TargetList
Dim strList As String
  On Error Resume Next
  strList = ThisDocument.Variables("TargetList")
  If Not Err.Number = 0 Then
    strList = InputBox("Enter list here separated by commas, e.g.", "List", "With,experience of,football")
    ThisDocument.Variables("TargetList") = strList
  End If
  On Error GoTo 0
  TargetList = Split(strList, ",")
  For i = 0 To UBound(TargetList)
    Set range = ActiveDocument.range
    With range.Find
      .Text = TargetList(i)
      .Format = True
      .MatchCase = False
      .MatchWholeWord = False
      .MatchWildcards = False
      .MatchSoundsLike = False
      .MatchAllWordForms = False
      Do While .Execute(Forward:=True) = True
        range.HighlightColorIndex = wdYellow
      Loop
    End With
  Next
End Sub
