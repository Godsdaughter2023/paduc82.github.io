considerate_guy say:

I'll add a few more options:

1) If you still want the classic version of Movie Maker (rather than going to Windows Live Movie Maker), you might consider installing WMM 6.0 on Windows 7 instead.  It does a better job of handling memory, can take advantage of a 64-bit computer and uses the GPU (graphics card) for acceleration of the playback preview window.  It can read the .mswmm project files from WMM 2.6, and if you are ever planning to move to WLMM, it can read WMM 6.0 project files, where it can't read WMM 2.x project files.

2) Most often when I'm unable to save a movie the culprits are one of two things:

a) Incompatible file types.  If you are using files other than .wmv for video and .wma for audio, consider converting them using something like Freemake Video Converter and Freemake Audio Converter (freemake.com).  Or search for other conversion programs using your favorite search engine.  I would recommend the .wmv and .wma formats because that's what Movie Maker obviously prefers.  If you do install Freemake, just be sure to watch other add-ons it might offer to install.  I generally uncheck them all and just install the program.

b) Too complex a project.  If you have lots of photos, transitions, etc. that can cause problems for WMM during the saving.  Make several copies of your original project and in one of the copies delete from the end until you are able to pare it down to a project that will save as a movie.  If you can save one part as a movie, then open the next project copy and remove what is already saved and repeat the process of paring the project down.  Repeat until you have all the parts.  At the end, you can create one final project where you add the movies for each of the parts and join them all together.  Incidentally Freemake Video Converter also has a join feature, so that's another option for this final step.

How do I remove safe mode from my computer?

bill smithers say:

How do I remove safe mode from my computer?

You don't remove Safe Mode.

You change your settings to start in Normal mode.

  1. Close any running programs and open Windows so you are back at the desktop.
  2. Click on the Start () button.
  3. In the Start Menu Search Box type msconfig
  4. Then press enter on your keyboard.
  5. The System Configuration utility will open
  6. Make sure Startup Selection is set for Normal Startup.
  7. Click on the Boot tab.
  8. Make sure  Safe boot is unchecked
  9. Press the Apply button and then press the OK button.
  10. Click on the Restart button to reboot your computer.
  11. Cheers.

Hi Thomas,

If the icons changed after you restart Windows, any of the following can be true: The icon cache (This is where Windows keeps copies of each icon.) is not updated correctly, corrupted icon database or icon layouts are not being saved upon restarting Windows.

We need a bit more information to help you better, please answer these questions:

  • Can you describe how did you customize the icons?
  • Did you use the default icons or you've downloaded new icons from the internet?
  • Are you pertaining to the Windows Explorer icons or the Desktop icons?

In this type of scenario, you may need to delete the cache files in order to rebuild the Icon cache. Check out these steps:

  1. Open Windows File Explorer.
  2. Navigate to this folder: C:\Users\%username%\AppData\Local\Microsoft\Windows\Explorer
  3. Now, you will see a lot of files like iconcache_32.db, iconcache_48.db, iconcache_96.db, iconcache_256.db, iconcache_1024.db, iconcache_1280.db, iconcache_1600.db, iconcache_1920.db, iconcache_2560.db, iconcache_exif.db, iconcache_idx.db, iconcache_sr.db, iconcache_wide.dd, iconcache_wide_alternate.db, etc.
  4. Delete them all to purge and rebuild your icon cache.
  5. Reboot your computer and the icon cache will automatically rebuild upon reboot.

We'll be here if you need additional assistance.
