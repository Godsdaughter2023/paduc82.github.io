Sub SplitWorkbook3()
    '
    Dim ws As Worksheet
    Dim DisplayStatusBar As Boolean
    Dim xpathname As String, dtimestamp As String
    Dim shtcnt(3) As Long
    Dim n As Long, strPath As String

    dtimestamp = Format(Now, " dd-mmm-yy")
   
    DisplayStatusBar = Application.DisplayStatusBar
    Application.DisplayStatusBar = True
    Application.ScreenUpdating = False
    Application.EnableEvents = False
   
    strPath = "c:\temp\Reports" & dtimestamp '& " " & n
    If Len(Dir(strPath, vbDirectory)) = 0 Then
        MkDir strPath
    Else
        n = 0
        Do
            n = n + 1
            If Len(Dir(strPath & n, vbDirectory)) = 0 Then
                MkDir strPath & n
                strPath = "c:\temp\Reports" & dtimestamp & n
                Exit Do
            End If
            If n = 10000 Then Exit Sub
        Loop
    End If
   
    nDay = Sheets(1).Range("XFB7").Value
   
    Set wb = ActiveWorkbook
   
    For Each ws In wb.Sheets

            shtcnt(1) = shtcnt(1) + 1

vijay a. verma say:

I think what you want is following -

1. Column A should allow only Yes, Column B only No and Column C only Maybe.

2. In a row, only one option can be chosen. For example, if No is chosen in Column B, then Yes in Column A and Maybe in Column C can't be chosen.

Assuming row 1 is header -


1. Select B2 and Data Tab>Data Validation>Data Validation


2. In Settings Tab, in Allow: box, select Custom. And in Formula: box, put following formula -

=AND($B2="Yes",COUNTA($B2:$D2)<=1)


3. In Error Alert Tab, In Error Message: Box, you may choose to put an appropriate error message. For example - Only Option Yes can be Written in Column B And from Column B, C and D only one can be entered.

4. Repeat steps 1 to 3 for C2 and put following formula and appropriate error message -

=AND($C2="No",COUNTA($B2:$D2)<=1)

5. Repeat steps 1 to 3 for D2 and put following formula and appropriate error message -

=AND($D2="Maybe",COUNTA($B2:$D2)<=1)

6. Now select B2 to D2 together and drag the cells to as many rows as you want to apply the validation to all other rows. (Or you can just copy B2 to D2 and select B3 to D3 to as many rows as you want. Paste > Paste Special > Validation)

mike middleton say: Goal Seek uses an iterative search process.

The precision is controlled by the "Maximum Change" setting.

In Excel 2013, choose File > Options > Formulas > (Calculation options) Maximum Change.

The default precision setting is 0.001.

For your example, the first iteration of the Goal Seek process yields 4.44 in A2. You can observe this result if you use a Maximum Change setting 0.6.

The primary purpose of the Maximum Iterations setting is to control calculation in a worksheet with circular references. I have seen conflicting opinions about whether the Maximum Iterations setting applies to Goal Seek.
