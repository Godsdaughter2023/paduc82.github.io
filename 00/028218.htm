Sub macro_01()

ossiemac say:

Not sure why you want this because during the creation of a formula when the cursor is at the position where the cell reference is required all you need to do is click on the merged group of cells and it will automatically insert the first cell reference of the merged group.

However, the UDF below will return either the first or last cell address of a merged group. Enter the formula on the worksheet as follows:

=FirstLastPos(E1:H1)        'Will return the first cell of the first merged area in the range E1:H1

=FirstLastPos(E1:H1,FALSE)       'Will return the last cell of the first merged range in the range E1:H1

The second parameter is optional and if left out it defaults to True for the first cell. For the last cell then the second parameter of false is required.

Not sure of your expertise with VBA so some guide lines to install. To install the code Alt and F11 to Open the VBA editor

Copy the code and paste into the editor

Alt and Q will close the editor

Save as macro enabled workbook

Ensure macros enabled in options. See Help for how to do this. Use the option Disable all with notification.

Code edited since initial post to remove a couple of lines used during testing

Function FirstLastPos(rng As Range, Optional FirstLast As Boolean = True) As Variant  'Must be variant
    Dim cel As Range
    For Each cel In rng
        If cel.MergeCells Then
            With cel.MergeArea
                If FirstLast = True Then    'Return first cell address
                    FirstLastPos = .Cells(1).Address(0, 0)
                Else
                    FirstLastPos = .Cells(.Cells.Count).Address(0, 0)   'Return last cell address
                End If
            End With
            Exit Function
        End If
    Next cel
    'If gets to here then no merged cells found in range
    FirstLastPos = CVErr(xlErrRef)
End Function

Note the editing of the code since initial post. Also forgot to mention a couple of things.

If the range in the formula does not have any merged cells then the code returns #Ref error.

To use the returned address in a formula then it needs to be used in conjunction with the Indirect function like the following.

=SUM(INDIRECT(FirstLastPos(A1:C1)),F1,A5)
