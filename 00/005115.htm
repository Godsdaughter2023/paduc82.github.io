Access: Cumulative total in an embedded sub-report. - Microsoft Community

krisna777 say:

Access: Cumulative total in an embedded sub-report.

I am trying to get a cumulative total in a sub-report which is embedded in another report.  Anyway I set the property to "over-all" and this seems to work in the sub-report but when I put the subreport in another report it totals the group instead of running a cumulative property.
allen browne say:

Each subreport is a separate instance, so you're right: it will not accumulate totals across instances.

It is possible to:

  1. pass the total from the subreport back to the main report, and
  2. accumulate it there as a running sum.

For details of step 1, see Bring the total from a subreport onto a main report .

For step 2, create another text box on the main report, with its Control Source set to equal the first, and its Running Sum set to Over All.
