Hi Vincent,

The kernel power event error occurs when the computer is shut down, or it restarts unexpectedly. When a computer that is running Windows starts, a check is performed to determine whether the computer was shut down cleanly. If the computer was not shut down cleanly, a Kernel Power Event error message is generated.

Method 1:

First, I would suggest you to follow these steps and run the System maintenance troubleshooter:

a. Press Windows key

b. Type “Troubleshoot” without quotes and click on Troubleshooting in the results displayed on the left side of the screen.

Pin to Taskbar issue:

 

Does the application actually launch from its source folder?

 

Many at times it so happen that when you try to pin an application whose filename is included in the reserved word list, they don’t get proper taskbar representation. Windows 7 contains a list of reserved words that it will check for when you try to pin a program to the Taskbar. If the program shortcut text contains any of the words that are in the reserved word list, you may face issues with the pinned application.

 

To identify if this is the cause, rename your .exe file to another unique name and try pinning the application to taskbar.
