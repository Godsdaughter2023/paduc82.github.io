talking moose say:
FYI, Outlook for Mac has no internal mechanism for tracking Daylight Saving Time. It relies completely on the Mac OS for this information. During DST time the difference from Central to UTC is 5:00 hours and during standard time the difference is 6:00 hours. (Note that it never mentions whether you're in DST or not. That's for the Mac OS to decide.)

What you see in the Calendars preference has nothing to do with displaying existing event times. That is only for adding new events to your calendar. If your calendar is off by an hour then the Mac clock, time zone or date is off by an hour.

k. m. mccormick say:

This is the original poster of this. I discovered the following:

  • This issue had absolutely nothing to do with the Personalization Settings (Theme, Background, Display, etc.). Anyone who selects high-contrast with a dark background (including the out of the box themes High Contrast #1, High Contrast #2, High Contrast Black) will run into this exact issue.
  • The issue occurs because Outlook has a fixed, internal setting of black for certain text (such as Read Email titles), even when the user has selected an out-of-the-box high-contrast mode with a black background.
  • The only way to fix this is by "customizing" Outlook--which can be difficult to do, given that various aspects of the Outlook program are "hidden" from the user because not only is the black text on a black background, but in some cases, you can't even highlight the text/see it's there.

This is how I fixed this issue:

  1. Go to "View" and then click on "View Settings." This opens the "Advanced View Settings: Compact" window.
  2. Click "Conditional Formatting." This opens the "Conditional Formatting" window.
  3. Select "Message List Font Size" (or, if it doesn't exist, create a new rule). Then click the "Font" button under "Properties of selected rule" of the same menu, which will open the "Font" window.
  4. Select any color other than black for this font. Then click "OK." This will close the "Font" window and take you back to the "Conditional Formatting" Window.
  5. Click "OK" on the "Conditional Formatting" window. You will receive the following warning prompt: "You have not created a condition for the rule "Message List Font Size." All items will be formatted with the selected font. Are you sure this is what you want to do?" Click "Yes." This will close the prompt and the "Conditional Formatting" window, taking you to the "Advanced View Settings: Compact" window.
  6. Click "OK" on the Advanced View Settings: Compact window to close it.

I'm adding this in case anyone else runs into this issue when using Outlook in  high-contrast mode with a black background.

I can only Microsoft fixes this obvious issue (enabling Outlook to have more control over the Font color than the theme before the user chooses to customize anything). Again, this was 100% out of the box behavior, even using the three high-contrast themes(with black background) direct from Windows.

"How to deactivate Psiloc World Traveler?
Occasionally you may wish to deactivate World Traveler.

On certain models (for example, the Nokia E52, E55 and E72) Psiloc World Traveler is built into the ROM of some phones and can only be deactivated, not deleted.

Following the steps below to deactivate World Traveler:

World Traveler Settings > Access point > Always ask

To set your access point to Always ask; from World Traveler Settings, select where Access Point is indicated and you will be directed to select from among three choices: Always ask, Automatic and Always on. Select Always ask.

World Traveler Settings > Internet connection > Always ask

To set your internet connection to Always ask; from World Traveler Settings, select where Internet connection is indicated and you will be directed to select from among three choices: Always ask, Automatic and Always on. Select Always ask.

Flight Assistant > Options > Settings > Flight alert update > Manual

To set Flight Assistant to Manual, from Flight assistant settings select Flight alert update to toggle the setting from Automatic to Manual.

Currencies > Options > Settings > Update interval > Manually

To set Currencies to Manually, from Currencies settings select Update interval and select Manually.

Weather > Options > Settings > Update interval > Manually

To set Weather to Manually, from Weather settings select Update interval and select Manually.

If you have followed all of these steps correctly you will have deactivated World Traveler."
