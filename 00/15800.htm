Assuming that you have the ID as the 1st column in your combo.
Set the text box like this
=[MyCombo].[Column](1) & " " & [MyCombo].[Column](2)

Note the combo column numbering starts from 0

or do it with VBA AfterUpdate
Private Sub Combo7_AfterUpdate()
'Me.MyTextBox = Me.MyCombo.Column(1) & " " & Me.MyCombo.Column(2)

Private Sub MyTextBox_Click()

scottgem say:

First, what you are talking about may not be good design. IN a well designed relational database data should exist in ONE place only. If you have an employees table and you need to assign an employee to a record, then you should have EmployeeID as a Foreign Key and that's the only info you need about the employee from the Employees table. 

So we need to know the Recordsource of the form, the Rowsource of the employee combobox and what table is being updated with what info. You should then, be able to just eliminate the text boxes. 

When you set the ControlSource of a control to an expression then you remove the link between the control and the table. If you want to timestamp a record, then set the DEFAULT VALUE of the control to =Now(). Leave the Controlsource as the date field you want to populate. 

Also, the Format function returns a text value not a date value.
