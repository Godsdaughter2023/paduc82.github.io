I would do this differently. I would start with the table that tracks completions. I would design it like this:

 

tblCompletedGoal

CompletedGoalID (PK Autonumber)

GoalID (FK)

EmployeeID (FK)

CompletionDate

VerifiedBy (FK lookup to employee table)

 

This should be all you need to produce reports of completions for any period.

tblGoal

GoalID (PK Autonumber)

Description

DeptID (FK) (Note: You will need a choice to indicate a company wide goal)

FirstName

DeptID (FK)

Supervisor (Y/N)

Other info about employees
