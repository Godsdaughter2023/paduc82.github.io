john mcghie say:
Multiple Versions in a single file is a feature that has been withdrawn, because it was an almost guaranteed "breakage" of the document.  And if anything went wrong with the document, you lost the whole thing: all of the content of all of the versions.

In previous versions of Word, the "Versions" entry will appear on the File menu.  See:

http://wordprocessing.about.com/library/WordTutorials/blversions2.htm

In Word 2011, only the most recent version is accessible when you open a document containing versions.  If there are other versions, you can't see them, and they will be removed on save.

The only way to get the previous versions back is to use an old copy of Word (2008 or earlier) that will enable you to save the previous versions out as separate documents.

dr. incredible say:

I am posting this reply because I think this question most closely resembles a problem I was having. 

I too was having difficulty downloading mail with large attachments from AT&T Yahoo mail.  I use the latest version of Norton Internet Security.   If I turned off Anti-Virus and Anti-Spam scans for e-mails, I could receive the large attachments with no problem.  Upon further investigation, for Norton Internet Security, the anti-virus and anti-spam features of its e-mail scans will not work for Pop3 and SMTP servers requiring SSL encryption.  Thus, if these scan features are turned off, mail comes through just fine.  Otherwise the mail server times out with errors because the e-mail scan is just stuck and non-functional. 

Hope this helps someone struggling with this problem.  I have wondered from some time why the Anti-Spam feature just did not work.  Not until I hit this problem did I realize it was related to SSL encryption.
