How do I get Outlook 2007 to read .eml files and attachments? - Microsoft Community

mrbinky3000 say:

How do I get Outlook 2007 to read .eml files and attachments?

I am a web developer working from home.  A lot of my clients talk to me via chat programs like Yahoo Messenger and Skype.  Often times, they send mail messages from a end-users to me over yahoo or skype that are .eml files.  These files describe bugs or other errors that end-users experienced.

I can't understand why I can't read "eml" files in Outlook 2007.  Right now, windows 2007 wants to use windows live mail to open these files.

Is there a third-party add-on that will give Outlook 2007 this functionality?
mrbinky3000 say: I solved the problem for Windows 7 64 bit users and Vista 64 bit users

1) Open up regedit.exe (The windows registry editor)
2) Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Microsoft Internet Mail Message WLMail\shell\open\command
3) Change "C:\Program Files (x86)\Windows Live\Mail\wlmail.exe" /eml:%1 to "C:\Program Files (x86)\Microsoft Office\Office12\OUTLOOK.EXE" /eml "%1"   (do not forget the double quotes!)
4) Restart your computer

This way, windows thinks that "Windows Live Mail" is still the default program for opening .eml files.  However, we changed the registry so that instead of opening Windows Live Mail, we are opening OUTLOOK.EXE.  What I mean is that if you opened the control panel, clicked on Default Programs and associated .eml files with Windows Live Mail, we've tricked out the registry so that it will open outlook instead--even though Windows still says it is associated with Windows Live.  Confusing, I know.

The best analogy are those old commercials where "We've secretly changed Lisa's favorite detergent with new all temperature Cheer!"  In this case we secretly changed Windows Live Mail with Outlook when it comes to opening .eml attachments.

Anyhow, this trick worked for me.  I can open .eml files just by double-clicking on them.  You can always change things back if you decide to switch to Windows Live.

Windows Live Mail Not Downloading messages - Microsoft Community

davexy say:

Windows Live Mail Not Downloading messages

Hi

I have a Windows 8 pc, but use the Windows Live Mail for my email client, I have finally managed to get it working but when I shut down my pc at night and come in to work the following morning and start up the program it will not download any messages that were received when the pc was off? Is there a simple setting that I am missing?

Thanks in advance for any help

a. user say:

Hi,

 

The email client will not download any messages when your computer is turned off. Please be informed that it downloads messages when the program is running from your computer. Furthermore, please be informed that there is a configured time when it download your messages. You may try to edit this setting by following the steps below and changing the time it check for new messages.

 

1. Open Windows Live Mail.

2. Press and hold Alt key + F + O + M.

3. Under General tab look for Check for new messages every.

4. Set it to 1 minute.

5. Click Apply and select OK.

 

Thanks!

davexy say:

Hi

Thanks for the answer. When I used the windows Live Mail program before I upgraded to Windows 8, any emails that were received when pc was off would come through when I logged on in the morning, now any emails that I get outside working hours for example are not coming through, what has changed?

a. user say:

Hi,

 

Alright, it is possible that the setting send and receive messages at startup has been disabled. Please follow these steps and enable this setting.

 

1. Open your Windows Live Mail.

2. Press and hold Alt key + F + O + M.

3. Under General tab look for Send and receive messages at startup and tick the button beside it.

4. Click Apply and select OK.

 

Thanks!
