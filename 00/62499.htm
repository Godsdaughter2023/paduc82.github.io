Windows 10 doesn't care what kind of monitor you have. You could use an old-school green monochrome CRT with Windows 10 (remember those?) if you wanted to. aakkam22 say:

Not easily. 

First, there is a big licensing headache associated with this type of move. It is possible, however you would have to call Microsoft to obtain an activation ID once the move is finished.

Also, there could be issues with hardware compatibility, unless you use a tool like sysprep. 

If you would like to know more about using sysprep, you can see this excellent guide on SevenForums. Sysprep will do exactly what you want, but it takes a lot of time and setup work, not something I would recommend in your case.

The only downside to this is that you would have to reinstall programs, but this really shouldn't take too long unless she has a lot of licensed programs.

Hi Davevan,

The Microsoft Precision Racing Wheel was made several years ago and it comes with the drivers that, at the best using Windows XP. Incompatibility is one of the possible causes why the gaming tool does not work with Windows 10 device. Another issue that I also consider is the older version of the driver that you installed in your device. Since it's an old driver, it may work with previous version of Windows but not with the current version that you have. It's essential for you to install the up-to-date driver for your device.

I highly recommend you run the Program Compatibility Troubleshooter. It will help you to run old programs designated from older version and make it compatible to the newest version that you have. To do this, you may refer to this link.

As an alternative, you may also follow these instructions to run the Microsoft Precision Racing Wheel in the compatibility mode:

  1. Create a New folder.
  2. Insert the supporting CD you’ve got with the wheel.
  3. Copy all files from the supporting CD to a New folder.
  4. Open the folder, right-click on the Setup and open Properties.
  5. Select the Compatibility tab.
  6. Check the ”Run this program in compatibility mode for:” box and select Windows XP (Service Pack 2) from the drop-down menu.
  7. Check the ”Run this program as an administrator” box.
  8. Click OK. link.

    Note: Always make sure to restart your PC after doing the said steps to see the effects.

    Should you have further questions, please let us know.
