It is very much possible indeed :) As I struggled to solve the same functional issue and got a few useful tricks here and there on the forums, my turn to suggest a solution!

How to synchronize the user templates

- chose a folder where to store your personal templates

- check that it is synchronized with OneDrive across all devices

(that's what has been explained above)

How to create default templates

Now, because the default templates are to be stored in C:\Users\username\AppData\Roaming\Microsoft\Templates, you will have to make a copy of your default templates in this folder for all your devices. For Word and Outlook, they are respectively:

- Normal.dotx

- NormalEmail.dotx

Shortcuts to personal templates

If you don't create personalized default templates on each of your device, you will need to chose a personal template every time you create a new document or email (new doc/email, select from personal template). This is a viable option if you create shortcuts to the templates:

- in the Windows task bar (like any recently open document, right-click and pin)

- in the Office application Quick Access toolbar (display all commands and select "new document or template")

For Microsoft: I would expect a setting in OneDrive and/or Office that would do this automatically (i.e. synchronize \appdata\... and override the default templates when there is a user template of the same name).

For the users: The Design tab is a super convenient and powerful tool for 1. customizing the overall appearance of the documents (fonts, colors, layout = styles) and 2. saving the results as the default if you like it.
