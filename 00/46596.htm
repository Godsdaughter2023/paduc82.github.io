Hi JH,

Restore partition or the Recovery partition is a special partition that can be found on a system hard drive. It is being used to restore the system to factory settings in case system issues occur.

A recovery partition is a partition in hard drive which contains the operating system in a state when the computer or device was originally purchased, while a restore partition is usually set by the original equipment manufacturers to serve as a storage for personal files or back up files in case of a serious system error.
