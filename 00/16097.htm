john korchok say:

To collect the information at the beginning, you'll want to create a UserForm. But just before doing that edit your module code to declare Public variables at the top for the name, email and date. These variables will communicate the information from the userform back to the main macro. Add a final score variable as well:

Dim userName As String

Adding a dollar sign on the end automatically declares the variable as a string. A percent sign makes it an integer.

TheDate$ = Date

With your project selected in the VBA editor, use Insert>UserForm to create the userform.

A window opens with the blank form and near it should be another window with controls you can add to the form. You'll want to add 2 or 3 TextBoxes and at least one CommandButton for an OK button. After laying out the form, double-click on the button and it will open the Code window for the form and create a subroutine for the button click. In this subroutine, assign the textbox values to the variables and close the userform:

Private Sub CommandButton1_Click()
  UserName$ = TextBox1.Value

Finally, assign the final score to the variable:

Sub FeedBack()

End Sub

and print it, along with the variable for name, email and date. You wrote that you had the printing figured out, but if you need help, please post again.
