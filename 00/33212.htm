jeeped say:
Do I have to do anything else in order to update my driver?

First, follow my original instructions and download the updated driver then unZIP the internal files to a temporary folder, noting its location for future reference.
  1. Next, click Start or tap the  key and type device but do not hit Enter. Locate Device Manager from the Control Panel section of the lost above and select that.
  2. When the Device Manager window opens, expand Network Adapters then double click on your Atheros AR5B91 to open Properties.
  3. On the Driver tab of the Atheros AR5B91 Properties dialog, click Update Driver.
  4. In the Update Driver Software window, select Browse my computer for driver software and then select Let me pick from a list of device drivers on my computer.
  5. Click Have Disk and select the netathrx.inf file (for your Windows 64-bit) from the folder you put it in.
  6. Click Next.
The computer should recognize that the files you are pointing to are updated version of the network adapter driver and install them. If this was a 32-bit system, you would point to the netathr.inf file in step 5.
