jeeped say:

You shouldn't do this on a user workbook template. Especially so when it is opened. If two users opened the template within a short period of time, they would receive the same number. If a user opened the template to a new workbook and then opened another they would receive the same number. If a PO was started and then discarded, the next PO might skip a number. The number can only be generated when the PO is saved.

I've written some ERP software that had a PO component and that opened the PO with PO-XXXXX. When the PO was saved (and stored in a central local SQL) the PONO was was generated and the PO (for the user) became PO-00001.

jay freedman say:
Rename the unwanted Normal.dotm to something else. Restart Word, which should create a new factory-fresh Normal.dotm (if you don't see it appear, do something such as adding an AutoText building block to it to force it to be created).

Open the new Normal.dotm and recreate the formatting used to create your documents. Then close it.

Now you can open your documents, turn off the automatic updating, and save them. This could be automated with a macro.

Finally, you can put back the organization's Normal.dotm, which will no longer affect the formatting of the old documents.

As should now be apparent, no document should ever be left with automatic updating turned on. Turn it on, use it, and immediately turn it off again.

If you are familiar with VB, this little sub is all what you need. Execute it in your sheet and watch out the output.

Sub Test()
  Dim R As Range
  Set R = ActiveWindow.VisibleRange
  MsgBox _
    "VisibleRange " & R.Address(0, 0) & vbCrLf & _
    "UsedRange   " & ActiveSheet.UsedRange.Address(0, 0)
End Sub

You have to compare the size of this two ranges, and adjust the width of each column. Record a macro to get some code how to shrink or expand a column.

But why want you do that? Why not zoom to show all data?

Sub MakeAllDataVisible()
  If ActiveSheet.UsedRange.Columns.Count > ActiveWindow.VisibleRange.Columns.Count Then
    Intersect(ActiveSheet.UsedRange.EntireColumn, _
      ActiveWindow.VisibleRange.EntireRow).Select
    ActiveWindow.Zoom = True
    ActiveCell.Select
  End If
End Sub
