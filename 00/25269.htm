rob koch say:

The true difference is that Malwarebytes includes many less malicious items termed Potentially Unwanted Programs or PUPs, that though Microsoft also has some of these in its detections as well, they're disabled by default and really only intended for easy use by businesses.

Since Malwarebytes became aggressive in their detection of these PUPs a few years ago and also has a simpler set of controls for selecting how to handle them on an individual basis, it's often recommended here and elsewhere as the easiest method for their management by consumers.

Selecting all PUPs - Malwarebytes Labs

Malwarebytes Adopts Aggressive PUP Policy - Malwarebytes Labs

So it isn't that Malwarebytes is more accurate, it's simply more aggressive at broadly detecting such items and displays them in an easier to understand and operate interface, making it a better choice to use in cases where consumer users encounter these items.

Microsoft Defender is more concerned with the truly malicious malware items that are truly intending to do harm, while the PUPs category often includes less malicious, but still annoying items like adware or other software that though some consider them malware, others might wish to install them for the free or reduced price items they sometimes offer in return for displaying ads.

How Microsoft identifies malware and potentially unwanted applications  Microsoft Docs

Microsoft terms these items Potentially Unwanted Applications (PUA) in that criteria document, but as I mentioned above, they're disabled by default, not easily enabled and controlling them individually is less simple than with Malwarebytes.

Rob

charleskenyon (ia) say:
Hi

I do not have a Mac and am writing from my experiences with the Windows version. That version is similar, but the two are not the same. While I've used Word on a Mac, that was a long time ago. Nevertheless, I hope that I can be of some help. If you only want help from a Mac user, you can stop reading.

I am assuming you mean that obviously misspelled words are not being flagged. This is most likely a proofing language problem. The "proofing language" is a key to several Word features including spelling and grammar and AutoCorrect. It is not an application-wide setting nor even a document setting; it is set at the character level! To see if this is the source of your problem:
• Select (or create and select) a misspelled word.
• Then under the Review tab, click on the "Language" button.
• Pick "Select Proofing Language" from the short menu.
• Look at the bottom of the dialog that appears.
• If the checkbox that says "Do not check spelling or grammar" is not empty, click on it until it is clear.
• Then OK.
Does this make a difference?

If it does make a difference, then Ctrl+A (Command+A) to select all text in your document and repeat the language selection.
If the problem occurs regularly, it is probably a problem with one or more Style definitions. If this is the problem, write back for more help.
You may want to look at Suzanne Barnhill's article on the spelling checker. It is the definitive article. http://wordfaqs.ssbarnhill.com/MasterSpellCheck...

I hope this information helps.
