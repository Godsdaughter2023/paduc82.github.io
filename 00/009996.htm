How to Apply Formula's to Drop Down Menu Selections

Hey all,

I run a youtube channel with three different people, and we have to split the earnings based on how much our videos make. So if one of my videos makes $5, I get that $5 since it's my video. We see that to be fair as oppose to splitting it 3 ways regardless of whose video makes what.

We have almost 1,000 videos, so it's a pain in the butt to go through and figure out who gets what. So I decided to make an excel sheet where I can just copy and paste the name of the videos in one column, the money it made in the next, and a drop down menu next to that to select whose video it is.

What I would like to do is have a formula that changes based on what name(s) I select. So if I select "Player 1", it will add the sum of the money made for that video to a cell that is for Player 1's pay, etc. Also, if there is a video with more than one player, like 2 or 3, then another formula will make it so it will split it between the 2 or 3 players for their respective cells. So if I select from the drop down "Player 1 and Player 2", it will split that evenly and be summed up in their respective cells.

I have looked everywhere online and I can't seem to find any formula to make this work. All I keep finding is how to make more drop down menus from other drop down menus. Does anyone know a way to make this work, or know another method?

I should also mention with our network we get a 80/20 split (we get the 80%), so that is a factor in the formula.

If I didn't explain it properly, I can provide a picture once I return to the office.

Thanks in advance!
teylyn say:
Hello,

the table in the first three columns is an excellent starting point. You don't need to pre-fill empty rows, though. Delete the empty rows, then click any cell in the first column and click Insert > Table. Now Excel will automatically put the drop-down in column C into any new row that you add to the table. 

You can then use this data to create a pivot table. In my screenshots I have used this data:

  1. Click any cell in the data table and then Insert > Pivot Table 

  2. In the dialog, select "Existing Worksheet", click in the Location field and then click cell F3. Then hit OK. (By selecting cell F3 the pivot table will have the same location as in my screenshot)



  3. In the Pivot Table panel on the right, drag "Who" to the Row Labels and "Earnings" to the Values. This will result in a pivot table like this:

The row labels will show all the values that are in the "Who" column and the Sum of Earnings will show the total for that person or group. Next you need to work out what each individual person has earned.

For Casey you need to add the values for Casey, half of the values for Casey+Elsie and Casey+Rob and a third of the value in "Everyone". Also, since the network takes a 20% cut, Casey will get 80% of that. The formula looks like this:


=(G4+(G5/2)+(G6/2)+(G8/3))*0.8

So, take Casey's value in cell G4, add to that half of the value in Casey+Elsie in G5, add to that half of the value in Casey+Rob in G6, add to that a third of the value for Everyone in G8. Multiply that total by 0.8 to return 80% only.

In a similar way, calculate the earnings for Elsie and Rob. The formula for the Network takes the Grand total in cell G11 and calculates 20% of that by multiplying with 0.2

As a check, create a sum of the Pay table. It should return the same value as the pivot table grand total.

Let me know if this works for you.

3. Reopen InfoPath form by Right-click the file and choose Design.
