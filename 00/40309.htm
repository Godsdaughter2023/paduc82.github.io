How to backup File History to two different external hard disk - Microsoft Community

roy ang say:

How to backup File History to two different external hard disk

I have two external portable hard disks. One keep in my office and another keep at home. I will need to do File History backup at both places. How can I set two target disk in File History?

rajesh govind say:

Hi Roy,

 

 

Thank you so much for posting your query on this forum.

 

Regarding your query, I would like to inform you that it is not possible to have 2 different external hard disks as the File History Backup Drive at a time.

 

In this case, the File History does not recognize the second hard disk when the first one was already set as the backup drive in the past.

 

The reason behind this is because, whenever we setup the file history on an external hard disk, a specific hardware ID is linked to the whole process. If in between we end up removing this hard drive and connect another one, the File History will not recognize it; as the hardware ID for the new hard disk will be different from that of the first one.

 

So, in this case if you want to use two hard drives to create file history back up, then you may run the File History and complete the backup on one of the hard drive and then turn OFF the file history. Then, you may connect the other hard disk and start file history all over again.

 

Hope this information is helpful. If you need any further assistance with Windows Operating System, please post your query on Microsoft Community Forums and we’ll be glad to assist you with it.
