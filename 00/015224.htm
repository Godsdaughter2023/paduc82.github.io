suzanne s. barnhill say:

Okay, here's what works for me.

  1. Insert the image (In Line With Text) in a separate paragraph.
  2. Add a letter to the beginning of the paragraph where you want the drop cap.
  3. Select that letter and use Insert | Text | Drop Cap to insert your drop cap.
  4. Select the image you inserted and Cut.
  5. Select the framed dropped capital and Paste; the image will be substituted for the letter.
  6. Delete the empty paragraph in which you inserted the image.

FWIW, I can confirm that an inline image at the beginning of a paragraph can be formatted as a drop cap in Word 2003 (Format | Drop Cap...) but not in Word 2007, 2010, or 2013, so this is a change that came with the Ribbon and .docx format, apparently. The difference between your two documents, therefore, may be that one is still in Word 97-2003 .doc format and the other has been updated to .docx format.

suzanne s. barnhill say:

The Drop Cap is just text in a frame. You should be able to select your image and click the Insert Horizontal Frame button that you have (in preparation) added to your Quick Access Toolbar (find it in Commands Not in the Ribbon). You can then format the size and position of the frame (you'll need to remove its default border as well).

But that may well not be necessary. I suspect that the difference between the two documents is the wrapping style of the images. Word creates drop caps from inline text, and text that is in a frame is also In Line With Text. I suspect that the images you're able to format as drop caps have In Line with Text wrapping and the ones that don't work have a different wrapping style (Square, Tight, etc.). Try changing them to In Line With Text and then try again.

joshua zhang msft say:

Hi Linda,

 

The Design Manager is not activated by default when SharePoint sites are provisioned. Please follow along these steps mentioned below to get it up:

Step 1. Activate the feature “SharePoint Server Publishing Infrastructure” present in the site collection level. This can be navigated as Settings > Site settings > Site collection features (under Site Collection Administration) > SharePoint Server Publishing Infrastructure > Click on Activate

Step 2. Once the site collection feature has been activated in the above step, then navigate to the subsite, and activate the Site Feature – “SharePoint Server Publishing”. This can be navigated as –

Settings > Site settings > Manage site features (under Site Actions) > SharePoint Server Publishing > Click on Activate

N.B. Many times it happens that, the Design Manager isn’t available in the settings menu, even after the two required features are activated. In that case, just deactivate both the features (site feature first, followed by site collection feature), and then follow the steps again. This will bring in “Design Manager” to the settings menu.
