Having a number of older laptops of the same basic model (Lenovo ThinkPad R500) that had different BIOS speeds,  Is the upgrading of the BIOS to one with a more recent speed a change to the motherboard?

Speeds for the BIOS on this model vary from 2GHz, 2.26GHz to 2.8GHz.

azrael's kiss say:

It actually shouldn't be, but who knows exactly what Microsoft activation servers are really looking when they identify the motherboard.  It should be a hard coded descriptor in the BIOS (such as a serial number or UUID) but not all BIOSes have these codes available to the OS and they can be changed in a flash, so a BIOS update could, theoretically, create confusion for activation if they are used.  If it's a general hash of the information passed by the BIOS, then there's definitely room for BIOS updates to remove activation.  If it's a list of hardware components collected by the system at activation, a BIOS update shouldn't effect that (hopefully).  If it's a collection of available serial numbers, UUIDs, MAC address, and other unique identifiers, it would hopefully not be confused by a BIOS update.  I'm guessing it's like anything else when it comes to activation... It's a coin flip.  If nothing else... Hopefully Microsoft customer service will help with the activation if a BIOS update does cause issue.

Have a good one!
