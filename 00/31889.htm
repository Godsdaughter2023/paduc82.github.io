carlo fer say:

Hello DGK75,

Based on the information you've provided, it is possible that your laptop doesn't support 5ghz network. To check if your laptop supports 5Ghz bandwidth, please follow the steps below:

  1. Press the Windows key + R. A run dialog box with open.
  2. Type cmd, then press Enter.
  3. On the command prompt, type "netsh wlan show drivers" then press Enter.
  4. Look for "Radio types supported" section.
  • 802.11g and 802.11n network modes - 2.4 GHz network capability only.
  • 802.11n 802.11g and 802.11b network modes - 2.4GHz network capability only.
  • 802.11a and 802.11g and 802.11n network modes -  2.4 GHz and 5GHz network capability.

Also, make sure that you have installed the latest drivers for your wireless card. Drivers can normally be found in the Support section from the manufacturers website where you will find a Software and Drivers sub-section. You can also refer to this support article: Update drivers in Windows 10.
