Being that the laptop has no CD-ROM drive you have will have to remove the disk from the laptop and you will have to use a 2.5" to 3.5" drive adapter and mount the disk in another Windows 2000/XP computer or in a USB disk enclosure, or, if your laptop can boot from  a USB device, (it probably does, look in the BIOS or consult the manual for the laptop), use an external USB CD-ROM drive.

If you use a USB CD-Rom you can create a Recovery Console CD by downloading a Recovery Console .iso image from the net and burning it to CD:http://search.yahoo.com/search;_ylt=A0geuyhRhaJKDhEAKpWl87UF?p=%22recovery+console%22%2B.iso&fr=yfp-t-501&fr2=sfp&iscqry=

If you mount the disk in another computer (instead of using the Recovery Console) you can dispense with Steps 1 and 3 altogether in KB307545, the object of the exercise in KB307545 is to copy the registry files in the Snapshot folder to the Config folder.  From the Snapshot folder copy the files:

•     _REGISTRY_USER_.DEFAULT
•     _REGISTRY_MACHINE_SECURITY
•     _REGISTRY_MACHINE_SOFTWARE
•     _REGISTRY_MACHINE_SYSTEM
•     _REGISTRY_MACHINE_SAM

to the System32\config folder directly, making sure to rename them as follows:

•     Rename _REGISTRY_USER_.DEFAULT to DEFAULT
•     Rename _REGISTRY_MACHINE_SECURITY to SECURITY
•     Rename _REGISTRY_MACHINE_SOFTWARE to SOFTWARE
•     Rename _REGISTRY_MACHINE_SYSTEM to SYSTEM
•     Rename _REGISTRY_MACHINE_SAM to SAM

These will in fact replace the present  DEFAULT, SECURITY, SOFTWARE, SYSTEM, and SAM files in the Config folder.

Once that is done, return the disk to the laptop and if it boots proceed to Step 4 as instructed in the article.

lemp say:
Wifi requires a particular piece of hardware (wireless network adapter).  Assuming that you no longer have the documentation that came with your 7-year-old PC, the first thing you should do is open Device Manager (Start > Run > devmgmt.msc > OK), click the + to the left of "Network adapters," and see what devices are listed there.

If the function of a network adapter isn't immediately obvious from its name, Google it.  If you have only one network adapter (plus, perhaps, something with "1394" in its name), odds are that you do NOT have a wireless network adapter.

If you don't have a wireless network adapter, you can buy one.  Although desktop/tower PCs can use an internal device (PCI wireless network adapter), USB-connected models are very common today.  I suggest getting one with a "real" antenna, rather than one that looks like a USB flash drive (or worse, a "micro" USB flash drive).  Links are to examples only, not recommendations.
