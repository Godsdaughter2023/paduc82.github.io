Is there some kind of 'relative' cross reference that could be used when creating the Quick Part content to have 'Figure 1' and 'Table 1' in the above example relate themselves to the caption number in the newly 'inserted' content (e.g. 'Figure 2' and 'Table 2' or 'Figure 47' and Table 47' as the case may be depending on how many times the building block is inserted)?

Any suggestions will be most welcome and appreciated.

(Note: I will not be permitted to deploy a macro-containing template.)

jay freedman say:

I don't see any way to make that work.

The problem is this: A cross-reference is a REF field pointing to a hidden bookmark that's created by the internal code of the Insert Cross-Reference dialog. The bookmark's name is "_Refxxxxxxxx" where "xxxxxxx" is a pseudo-random number.

But your large building block will contain only one set of these bookmarks. When you insert a second building block, containing the same set of hidden bookmarks with identical names, all the bookmarks in the first building block will be removed -- because no two bookmarks in a given document are allowed to have the same name. The result is that the cross-references in the first building block will now point to the captions in the second building block.

As more building blocks are inserted, all the cross-references will wind up pointing to the captions in the last building block.

To work around this, after inserting the second and later building blocks, you would have to go back to the preceding section and re-insert its cross-references, which would create new bookmarks with names different from those elsewhere in the document.

macropod say:

Although you say you're using a Mac, you posted in a forum for Office 2010 on a Windows 10 PC.

On Macs, F9 on its own is sufficient to update all fields.

When cross-referencing footnotes, the cross-reference always points to the footnote reference in the body of the document, not to the footnote content. Hi,

 

Try this

 

1.  Assume data is in range D6:F14.  Headings are in row 5

2. In cell D18, enter Online and in cell F18, enter the following array formula (Ctrl+Shift+Enter).  These students have taken online and onground courses

 

=SUMPRODUCT(1*(FREQUENCY(IF($F$6:$F$14=$D18,IF($D$6:$D$14<>"",MATCH("~"&$D$6:$D$14,$D$6:$D$14&"",0))),ROW($D$6:$D$14)-ROW($D$5))>0))

 

3. In cell D19, enter On Ground and in cell F19, enter the following array formula (Ctrl+Shift+Enter).  These students have taken onground and online courses

 

=SUMPRODUCT(1*(FREQUENCY(IF($F$6:$F$14=$D19,IF($D$6:$D$14<>"",MATCH("~"&$D$6:$D$14,$D$6:$D$14&"",0))),ROW($D$6:$D$14)-ROW($D$5))>0))

 

4. In cell F24, enter the following formula to get number of students who have taken hybrid courses

 

=F18+F19-SUMPRODUCT(1/COUNTIF($D$6:$D$14,D6:D14))
