nicopechin say:

After two private chats with tech support, and some further research, I pieced together a solution.

  1. Restart your PC using the physical power button *SIX* times, at minimum.
  2. Let it perform an auto repair.
  3. Go to advanced options.
  4. Go to advanced options again.
  5. Go to Command Prompt.
  6. Enter the command bcdedit
  7. There will be two paragraphs of output. Focus on the second one.
  8. The first Line of that paragraph should be identifier                 {default}
  9. Note: {default} may say {current} or some other word. Use the word that appears in the next step.
  10. Enter the command bcdedit/ deletevalue {conditionhere} safeboot        (conditionhere being whatever appears within the { } ).
  11. To check your work, enter the command bcdedit . The bottom paragraph should be altered by some degree.
  12. Close the command prompt and turn off your PC. Situation normal! At least for me...

Question about =ADDRESS() function - Microsoft Community

keith171 say:

Question about =ADDRESS() function

Hello,

I have used the spreadsheet ADDRESS() function to specify which cells are to be included in a series.  For example, I have set it up so the entry =ADDRESS(f10,g10) returns the value $C$3.  Now, I need to show the contents of $C$3.  How do I use the ADDRESS() function to now return the contents of the address that was specified?

Thank you,

Keith

 

joeu2004 say: Keith wrote:
I have used the spreadsheet ADDRESS() function to specify which cells are to be included in a series.  For example, I have set it up so the entry =ADDRESS(f10,g10) returns the value $C$3.  Now, I need to show the contents of $C$3.  How do I use the ADDRESS() function to now return the contents of the address that was specified?

Although you could use INDIRECT(ADDRESS(F10,G10)), if you know the limits of values that F10 and G10 can have, the following avoids the use of volatile functions:

=INDEX(A:E,F10,G10)

=INDEX(B5:E100,F10,G10)

Note that in the latter example, F10 and G10 are relative to the range B5:E100.  So if F10 and G10 are 1, they refer to B5, not A1.

  • F10 = 3
  • G10 = 3

Try it like this...

=INDIRECT(ADDRESS(F10,G10))

Or...

=INDIRECT("R"&F10&"C"&G10,0)
