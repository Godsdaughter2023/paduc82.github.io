I have a Image.

And I want to read the text from the Images

Is is possible to do with the help of the Macros.

steve rindsberg say:

Rich is correct.

Images do not contain text.  They may contain patterns of dots that *look* like text.  That's all (other than special formats that Rich alludes to... PDF, PSD and the like, which can contain both text and images).

There are OCR (optical character recognition) programs that can scan images and (sometimes) figure out which patterns of dots look like characters; some of them are quite good, but none of this is built into VB/VBA.

Try googling terms like ocr+api to see what external programs are available that you might be able to control from VBA

steve rindsberg say:

It's possible to control *some* software from Excel (or any other instance of) VBA; it depends on whether the software exposes an object model to automation.

A quick google on the term:  automate pdfxchange produces:

http://www.tracker-software.com/knowledgebase/124-I-receive-an-error-message-Cannot-create-automation-object

myall_blues say:

A possible formula solution - your workbook must be saved first.

Select Sheet2.

Hold down SHIFT and select SheetN (e.g. Sheet50). This will group the sheets together so that the formula you type next will be reproduced on all the sheets.

In Sheet2 cell C9 enter the formula:

=INDIRECT("Sheet1!A"&SHEET(MID(CELL("filename",A1),FIND("]",CELL("filename",A1))+1,255))-1)

Right click the Sheet2 tab and select 'Ungroup sheets'
