Office 365 / Live@edu / Exchange F&F users: - Microsoft Community

mike (microsoft) say:

Office 365 / Live@edu / Exchange F&F users: problems with Rooms or no text message backup option on your new Windows Phone 8?

Symptoms

After joining or creating a Room on Windows Phone 8, you may notice the following problems:

  • Unable to view any room members.
  • Unable to see room chat message or use chat in the room.
  • Unable to see calendar appointments or add calendar entries in the room.
  • (However, you may have access to photos and notes)

When trying to setup text message back, you’ll find there isn’t a setting to enable in Messaging > Settings. 

mike (microsoft) say:

Cause

Your primary Microsoft account on Windows Phone 8 is an Office 365 account (for business or education), Live@edu, or an Exchange Friends & Family account. And, that you’ve changed the mail server from “m.hotmail.com” to “m.outlook.com”.

 

Note: you may have changed the mail server due to older work arounds that have you’ve read about or being used to changing it for your account on Windows Phone 7. For Windows Phone 8, you should not change the mail server.

 

Resolution

It is no longer needed to change the mail server for these account types, as was once thought necessary on Windows Phone. Instead, users should leave the primary Microsoft account with default settings (m.hotmail.com) and setup a secondary account to get their mail, calendar, and contacts for Office 365, Live@edu, or Exchange Friends & Family. 

 

The below should work for those who are setting up their Windows Phone 8 for the first time with these account types:

  1. Sign into the Microsoft Account with your account. 
  2. Do *not* modify the server that you are pointing to for the primary Microsoft account.  This should point to m.hotmail.com.
  3. Set up a secondary Outlook (O365) account to get your mail, contacts, and calendar – this account should point to outlook.com (m.outlook.com). To learn more about how to setup this account you can visit this page for more info: http://www.windowsphone.com/en-US/how-to/wp8/people/set-up-or-delete-an-email-account.  

 

If you are already in this state, due to manually setting the server to m.outlook.com and you have backed up this state, you'll have to go through the following to use Rooms and other features appropriately:  

  1. Sync all accounts to backup as much information as you can.
  2. Copy any pictures or other information you want to save to the cloud or a PC.
  3. Hard reset the phone. To do this, go to Settings > tap about, and tap reset your phone.
  4. During the initial setup wizard (OOBE), opt to NOT restore the phone from a previous backup. Note: This step is important because otherwise the primary Microsoft account will restore with the wrong mail server again.   
  5. Sign into the Microsoft Account with your account.
  6. Do *not* modify the server that you are pointing to for the primary Microsoft account.  This should point to m.hotmail.com.
  7. Set up a separate Outlook (O365) account to get your mail, contacts, and calendar – this account should point to outlook.com (m.outlook.com). To learn more about how to setup this account you can visit this page for more info: http://www.windowsphone.com/en-US/how-to/wp8/people/set-up-or-delete-an-email-account.

From those details it sounds like it's genuine. The RM, model, IMEI and code all relate to each other.

If the device works with Nokia software such as the nokia software updater, PC suite, Ovi Suite or the ovi store on the phone then it's genuine.

If when you connect it to a PC it gives web cam or com port as options then it is a fake.
