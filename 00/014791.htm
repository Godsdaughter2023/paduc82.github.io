Could you please tell me how we can do same thing in CODE.???

andreas killer say:

You must ask the object to execute the appropriate verb. The code below is just an example, you have to customize it.

Andreas.

 

Sub Test()
  With ActivePresentation.Slides(1).Shapes(3)
    If .Type = msoEmbeddedOLEObject Or .Type = msoLinkedOLEObject Then
      For Each sVerb In .OLEFormat.ObjectVerbs
        Debug.Print sVerb
        nCount = nCount + 1
        If sVerb = "Save Contents" Then
          .OLEFormat.DoVerb nCount
          Exit For
        End If
      Next
    End If
  End With
End Sub
