You need to use VBA:

Sub ReplaceCFwithFormatting()
    Dim rngC As Range
    For Each rngC In Selection
        rngC.Interior.Color = rngC.DisplayFormat.Interior.Color
        rngC.Font.FontStyle = rngC.Font.FontStyle
        'and so on for each specific format

    Next rngC
    Selection.FormatConditions.Delete

"Splash screen" usually refers to the little window you see immediately before a program starts. To suppress that window when Word starts, you can use a command line with the /q switch. For example, in the Run dialog box, you can type

winword /q

and press Enter.

The Start screen is the one you see after Word has started (unless you disable it in Word Options):

Hello I’m Independet Advisor

Thanks for contacting Microsoft Community.

To resolve Office error code 0xe06d7363 please follow simple instructions in this document.

1. System File Checker.

System File Checker is a Windows utility that allows users to scan for errors in Windows system files and helps restore corrupted / damaged / missing files.

1. Swipe in from the right edge of the screen, and then tap Search.
2. Type ‘Command Prompt’ in the Search box, right-click Command Prompt.
3. Click Run as administrator.
4. At the command prompt, type command sfc /scannow, and press ENTER.

sfc /scannow command will scan all protected system files, and replace corrupted files with a cached copy.

Note: Do not close Command Prompt window until verification is 100% complete. Scan results will be shown after the process is finished.
