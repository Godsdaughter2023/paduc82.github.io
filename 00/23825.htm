rey bel1 say:

Hi Yong,

Since the Inbox Repair Tool was unable to repair the errors found, we suggest that you create another profile instead and insert the PST files to it. To do this, follow the step below:

1. Go to Control Panel, search Mail and double click to open it.

2. Click Show Profiles and click Add.

3. Create the name of the new profile and click OK.

4. Choose Manual setup or additional server types and click Next.

5. Choose POP or IMAP and click Next.

6. Fill in those information and type in pop.gmail.com for Incoming mail server and smtp.gmail.com for Outgoing mail server (SMTP).

7. If you still have the data files from your Windows computer or from previous Outlook application, you can import those files, but first you need to save it on your Windows 10 computer. To import those data files, choose Existing Outlook Data File and click Browse to locate it on your computer, after that click More Settings.

8. Go to Outgoing Server tab and tick My outgoing server (SMTP) requires authentication.

9. Go to the Advanced tab, port 995 for the Incoming server (POP3) and tick This server requires an encrypted connection (SSL).

10. Type in port 465 for Outgoing server (SMTP) and click the drop down arrow, choose SSL to enable the encrypted connection and click OK.

11. To complete the new profile, click Next.

12. To open the new profiles, press the Windows key and R on the keyboard and in the Run command, type outlook /profiles and click OK.

OK. Check if you can now get all of the folders that's what you have from your previous Outlook application.
