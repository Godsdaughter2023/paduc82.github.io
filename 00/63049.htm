Windows update Error 0x80246008 and BITS error 2147024894 - Microsoft Community

a. user say:

Windows update Error 0x80246008 and BITS error 2147024894

Here’s the problem description:

  • When the customer tries to check (download/install) for updates in Windows 10, they may encounter the error 0x80246008
  • When you try to start BITS  (Background Intelligent Transfer Service), the customer encounters the error 2147024894.  You can manually start the service via the services console (Start > services.msc)

Windows Update depends on BITS.  If BITS is broken, Windows update will not work.  This applies to many version of Windows (introduced with Windows 2000). 

Here’s the quick easy way to fix this error:

  1. Open an admin CMD
  2. Type the following command:
    Reg add HKLM\SYSTEM\CurrentControlSet\Control\BackupRestore\FilesNotToBackup
  3. Start the BITS again via the service console (services.msc)

no cursor appearing and touchpad not responding - Microsoft Community

sue1988 say:

no cursor appearing and touchpad not responding

The cursor stopped appearing on the screen and the touchpad isn't responding.  

davem121 say:
Hi Sue1988, try booting into Safe Mode to see if your touchpad works in that environment . . .

Start your PC, just as Windows attempts to load (spinning dots), press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down

Do this twice

On the third start Windows will boot into the Recovery Environment and from there you can access System Repair, Safe Mode, Command Prompt . . . etc.

Go to Troubleshoot - Advanced Option - Startup Settings and click Restart
Upon restart, press 4 to enter Safe Mode

Once in Safe Mode, backup your data if you do not already have a backup

Open Device Manager

Expand out the Mice and other Pointing devices section

Right click your Touchppad driver and choose Uninstall

Restart your PC
