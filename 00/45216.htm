halima s say:

Hi uberduner,

During the ripping process, the Player makes a copy of each song, stores it on your hard disk as a compressed Windows Media Audio (WMA) or MP3 file, and then adds it to your library.

You will need to change the location to store the ripped files in a different folder for each disc so that you will create single library list for each disc and later burn them accordingly in single discs.

Use the below steps to change the location of ripped files.

1. Open Windows media player by clicking start menu
2. Click the arrow below the Rip tab, and then click More Options.
3. Make one or more of the following changes:
• Location. Click Change if you want to store ripped files in a different folder on your computer.

Change settings for ripping music
http://windows.microsoft.com/en-us/windows-vista/Change-settings-for-ripping-music
