An IMAP mail account is a copy of what is on the server, and if you add that IMAP account elsewhere it will be repopulated from the server.

A pop mail data file of 256kb is empty, or virtually so.

You can backup a pop mail data file, the pst, by simply copying it, whilst outlook is closed or by using your backup program

A pst data file cannot be incrementally backed up.

A pst data file opened in OL, requires full read/write permissions, so the internal modified  date is reset whenever outlook is opened.

If you have a pst data file you must have either created it, or have upgraded from an earlier version of outlook, as the pst is not used in OL 2013/2016 with an IMAP account
