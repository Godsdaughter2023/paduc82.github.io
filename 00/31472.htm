Check out 'will my devices be compatible' and try the upgrade assistant from here:

<<http://windows.microsoft.com/en-US/windows-8/upgrade-to-windows-8>>


It is possible that incorrect configurations, corrupt or broken-missing files in your version of windows could cause updating issues, if it has nothing to do with any installed software clashing with or preventing the update, such as a third party antivirus or firewall, which you may want to temporarily switch off whilst updating.


Try running the command prompt tool as administrator (in windows 8's metro screen, after right clicking and selecting the 'all apps' screen, it should be listed on the far right), and then use the system file checker function by entering

'sfc /scannow'

into the black box and hitting ENTER.


If some files cannot be fixed, try running the command prompt tool as administrator again, with the Deployment Image Servicing and Management function, which can restore system configurations, by entering


'dism /online /cleanup-image /scanhealth'

and then ENTER

(this will scan for issues)


'dism /online /cleanup-image /checkhealth'

and then ENTER

(this will scan for corruption)


and again with

'dism /online /cleanup-image /restorehealth'

and ENTER.

(fix corruption)


Note the spaces after certain text above for the functions to work.

You may also want to try a hard drive diagnostic tool to check your drives həalth before attempting another update.


If none of that helps then you may want to attempt an update to 8.1 after using windows System Restore, to take your installation back to a time when the update may possibly show no software exception error (depending on the size of your drive a restore can literally take hours), or if you have a windows 8 restoration disc, attempt an update to 8.1 after a fresh windows 8.0 install.


Make sure you always back up all of your important data before attempting any system update or restoration.

    8 people found this helpful

    dirty loco say:

    There are a few ways to tell.  First is manufacture date.  Second is wattage of the power supply.  Third is the system amperage. 

    Here is how it breaks down.

    Xenon:

    • Manufactured between 2005 -2006
    • 203W power supply.
    • 16.5A

    Zephyr:

    • Manufactured between 2006 - July 2007
    • 203W power supply.
    • 16.5A
    • Added a HDMI port (only for the premium, not the Arcade console)

    Opus:

    • Manufactured between July 2007 - September 2007
    • 175W power supply.
    • 14.2A

    Falcon:

    • Manufactured between August 24th 2007 - October 2008
    • 175W power supply.
    • 14.2A

    Jasper:

    • Manufactured between October 18th 2008 -  the Xbox 360 redesign
    • 150W power supply.
    • 12.1A
