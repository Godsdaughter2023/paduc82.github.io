rohn007 say:
Glad to be of help.

Now that it has worked, you should mark the replies that formed the answer <grin> so that others know exactly what worked for you.

Hiding file extensions. That is a MS/Windows thing that I personally strongly disagree with.  Along with the GUI it is one of MS/Windows first "user friendly" changes to dumb down computing for the "average user". But there is no intrinsic value to hiding the extension.   Part of the that dumbing down included a programming standard that "let" programs depend on Windows to match file extensions to programs. In DOS, and other OS's the program recognizes it's "own" files by reading the file header.

Over the years there have been a number of "hacks" that depended on mis naming files, showing the  "wrong" file extension.

Windows depends on a very simplist system of text entries in the registry to associate programs and file types. Windows "file associations" get messed up all of the time and have to be fixed.

Bottom line, I prefer to keep file extensions showing all the time. That is why I didn't mention it in the tip.  Actually, that is on my "new computer setup list" of configuration and customization I apply to a new computer.

"How do we change the ringout time on a Win 8 Nokia Lumia 920 phone?"

 

All mobile phones include this function, whether set by the provider or customisable, otherwise your phone would never divert to answerphone, although on pay as you go phones you may not be able to divert anywhere other than your answerphone due to the cost of the divert (you pay for the time the other person spends on the phone if you divert to another number).  Suggesting that a phone does not have such a basic option is just unhelpful and makes people think there is no fix for this question! 

 

On orange my answerphone is at 07973 100 123.  So, using this in my example, the response to this question is;

 

try *#61# to view the setting,

mine replies

Forward if no reply is enabled to +447973100123 for voice after 20 seconds

 

The next part will allow you to change the divert for no answer;

*61*07973100123*11*30#

then press call

will divert to +447973100123 after 30 seconds

phone will respond saying confirmed if it worked

 

Next type *#61# again to confirm the details are set correctly

mine now replies

"Forward if no reply is enabled to +447973100123 for voice after 30 seconds"

 

I use Orange in the UK, but this is a standard code it appears for all the Windows Phones (I also used it on MY HTC 8X, works exactly the same), also people on the wpcentral forum are using the same code for other models of Windows Phone.
