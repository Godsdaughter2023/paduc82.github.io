WHEA_UNCORRECTABLE_ERROR: This error is a bugcheck generated directly by the CPU. Most often it is caused because of incorrect voltage being applied to the CPU. This can happen because of incorrect settings being applied to the BIOS (often resetting the BIOS to the defaults or updating the BIOS helps), sometimes it indicates that a Power supply is outside of its correct voltage range. (Power supply voltage regulators or mother board voltage regulator is in the process of failing.

hal.dll: The Windows Hardware Abstraction Layer (HAL) is implemented in Hal.dll. Windows Hardware Abstraction Layer (HAL), a file that hides hardware complexities from Windows applications. hal.dll is a system process that is needed for your PC to work properly. In simple, the interaction between the computer operating system and a hardware device is the effect of a software program layer known as the hardware abstraction layer (hal.dll). The HAL implements a number of functions that are implemented in different ways by different hardware platforms.

Causes of the "missing or corrupt hal.dll" error include, naturally, a damaged hal.dll DLL file or a hal.dll file that has been deleted or moved from its intended location. Additional causes may include a damaged or missing boot.ini file or possibly a physically damaged hard drive. The hal.dll is called either from the operating system's kernel or from a device driver.

Purpose of HAL is, it prevents other applications from directly accessing the computer's system memory, CPU or hardware devices such as the video and sound cards.

HAL Conflicts: HAL sometime does not work well with the Disk operating system (DOS) games and programs. DOS games and programs need to load their own memory managers or control hardware.
