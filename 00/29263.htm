A runtime error is a software or hardware problem that prevents a program from working correctly. Runtime errors might cause you to lose information in the file you're working on, cause errors in the file (corrupt the file) so you can't work with it, or prevent you from using a feature. Unlike stop errors, runtime errors don't typically cause Windows or a program to stop working.

 

Runtime errors can occur if you are running two software programs that aren't compatible, if your computer has memory problems, or if the computer has been infected with malicious software. If a runtime error keeps occurring, try the following methods.

 

Method 1: You may Re-register comdlg32.ocx file from an elevated command prompt and check if it works.

 

a) Type CMD from the start screen

b) Right click on CMD and select Run as administrator.

c) Type the following command and hit Enter after each command

regsvr32 /u Comdlg32.ocx
regsvr32 /i Comdlg32.ocx

d)  Close the command prompt and then try to run the program.

 

You may also try performing System File checker scan.

 

Method 2: Run a system file checker scan and check if that helps.

 

Refer the following steps to run SFC scan:

 

a) Move the mouse pointer to bottom left of the screen where you see preview of start menu and Right-click to bring up a list of Windows settings.
b) Click on ‘Command Prompt (Admin) to bring up elevated command prompt. Note: Click ‘Yes’ if it prompts for permission.
c) In the command prompt type in the following command: ‘sfc /scannow’.

 

Note: It may ask for windows installation DVD to fix and to enable SFC to make more than minor repairs.

 

Install the latest updates for Windows and for the program displaying the error. Note that the update for the program might be listed as an optional update on the Windows Update page.

 

Method 3: Install Windows update through Control panel.

maximumextreme say:

NAT is Network Address Translation. It allows your network to communicate with devices that are hooked up to your router or modem. The best type of NAT is open. If your NAT is moderate then only some communication is being received. If you go to network in the settings of your Xbox One you can click on advanced settings on the bottom of the screen, then select the third option which I believe is clear MAC address. After you select this option your console will restart. Once it restarts go into network settings again to see if your NAT changed to open.
