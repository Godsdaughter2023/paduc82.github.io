sue mosher say:

The usual warnings against importing from a .pst file don't apply in this scenario, because you want to move to the new file format and there is no way to convert an existing file to the new format. Instead, you will need to copy or import your old data into a new .pst file. What you'll want to do is set up Outlook on the new machine and allow it to create a new .pst file. That new file will be in the new format with a 20GB maximum size. Copy your old .pst file to the new machine and then either open it in Outlook and copy the items you want into the new file or go ahead and import the whole thing. You still will lose some information in the copy or import process -- custom views, links to contacts from other items, links to Outlook items in the journal, possibly also distribution list functionality -- but there's really no way around that.
