PART 1 General

An Ideas - also how do you make the question single space without all the extra lines

 

suzanne s. barnhill say:
1. Place the insertion point in a paragraph in your top-level style. In the
Multilevel List gallery, select the numbering that has 1, 1.1, 1.1.1 or (if
you're using heading styles for all the numbered paragraphs) 1 Heading 1,
1.1 Heading 2, 1.1.1 Heading 3. This will apply numbering to the selected
paragraph (and also, if you've used the one with headings, to all your other
headings).
 
2. Open the gallery again and click "Define New Multilevel List." In the
Define new Multilevel List dialog, with Level 1 selected, type PART and a
space before the number in the "Number format" box.
 
3. Change the number style to I, II, III.
 
4. If you did not select the list style that is linked to headings, then
select the appropriate style for this list level.
 
5. Select Level 2 and click More >> to expand the dialog.
 
6. Check the box for "Legal style numbering." This will give you 1.1 instead
of I.1.
 
7. Again, link the level to the appropriate style if necessary.
 
8. Repeat for Levels 3, 4, etc., as required.
 
9. If the indents set by Word don't suit you, change them at each level as
desired.

You cannot delete any built-in styles (which include all the Heading N styles) from any template or document. However, you can prevent specific ones from appearing in the Styles gallery.

In the Manage Styles dialog that you showed, or in the Styles gallery or the Styles task pane, click the Modify button or the Modify item on the right-click menu. In the Modify Style dialog, clear the check box for "Add to the Styles gallery" and click the option button for "New documents based on this template".

Repeat for each of the styles you want to suppress. Remember to save the Normal.dotm template if you're prompted when you shut down Word.

neo wu msft say:

Hi Andrew,

 

For ruler units, you can follow steps below to change it:

1.    Go to System Preferences on your Mac and choose Language & Region preferences.

2.    Click Advanced button and then the General Tab. Under this tab, you will see there are many different locate options, such as Format language, number separators, currency and Measurement units.

3.    Click the Measurements unit and change it from Metric to US. Then, open PowerPoint and the unit of measurement should be changed.

 

For text box, you can add it by clicking Text Box button in Insert ribbon. Or click the edge of one existing text box then copy and paste it in the slide.

If you still can’t add text boxes by the methods above, please send us the info below to us via Private Message.

  1. The reproduce video. You can follow this article to record your screen.
  2. The version number of your OS.
  3. The version number of your Office.
