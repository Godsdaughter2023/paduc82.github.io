How To Activate The Hidden Administrator Account For Windows - Microsoft Community

jeff h ねむ゚ say:

How To Activate The Hidden Administrator Account For Windows

Activate the preloaded hidden Administrator account of Windows.

Windows 8 / 8.1 

Using any of your user accounts(admin), press and hold Windows Key and hit "X",
choose Command Prompt(admin) on the list, you will then have the command prompt.

Widows 7

Press or hit the Start Button or Windows Logo(lower left), type in to the Search bar, cmd - right click on it and choose the "Run As Administrator", accept the UAC and you'll get the command prompt.


Type this command in:

net user administrator /active:yes

and hit enter, the command prompt should state that the command was successfully loaded.

Sign out your current user, you will then able to see an Administrator account on the log in list, it has no password, just click on it.

And you have the Administrator.

---

To turn it off(do not turn it off if you have no other accounts - you'll not be able to see your desktop again)
here's the command:

net user administrator /active:no

---
