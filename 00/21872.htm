marshall barton say:

You don't gave to use a custom function to do that.  A calculated query field expression that uses built in function might be considered a little clumsy, but do the job:

 

NumDays:  IIf(TimeValue(Assigned) >= #15:30# And DateValue(Assigned) = DateValue(Shipped) - 1, 0, DateDiff("d", Assigned, Shipped))

 

Using a custom function might be somewhat easier to understand, but is not required.

 

IMPORTANT Note: this calculated value should NOT be saved in a table.  Instead, use the above expression (or a custom function), in a query any time you need to see the NumDays value on a form or report.
