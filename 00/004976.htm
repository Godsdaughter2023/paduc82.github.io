How to add a " IF(ISERROR " to as Sumproduct formula - Microsoft Community

d_gibson say:

How to add a " IF(ISERROR " to as Sumproduct formula

Hello All -

=SUMPRODUCT(--(B65:J65<>0),AR65:AZ65)/(SUMPRODUCT(--(B65:J65<>0)))

Some of the rows I'm summing up has all zeros, so of course I get the #DIV/0!

I can do it if I don't use the "(--" piece to it, but I need it.  Seems like it would be simple to do, but I'm not finding a solution on the web. 

TIA - Gibson
joeu2004 say: Gibson wrote:
Can someone suggest how to add a " IF(ISERROR " to this Sumproduct formula:
=SUMPRODUCT(--(B65:J65<>0),AR65:AZ65)/(SUMPRODUCT(--(B65:J65<>0)))
Some of the rows I'm summing up has all zeros, so of course I get the #DIV/0!

If you do not need Excel 2003 compatibility, you could write:

=IFERROR(AVERAGEIFS(AR65:AZ65,B65:J65,"<>0"),0)

You can replace 0 with anything else you want when AVERAGEIFS fails.

If you require Excel 2003 compatibility, you could write:

=IF(SUMPRODUCT(--(B65:J65<>0))=0,0,
SUMPRODUCT(--(B65:J65<>0),AR65:AZ65)/SUMPRODUCT(--(B65:J65<>0)))

ISERROR might be preferred if you also want the same result when any of B65:J65 and AR65:AZ65 contains an Excel error.  In that case, you could write:

=IF(ISERROR(SUMPRODUCT(--(B65:J65<>0),AR65:AZ65)/SUMPRODUCT(--(B65:J65<>0))),0,
SUMPRODUCT(--(B65:J65<>0),AR65:AZ65)/SUMPRODUCT(--(B65:J65<>0)))

Not a good idea, IMHO.


chip pearson say:

Excel 2007 introduced the IFERROR function, which traps an error and returns an alternative result. Wrap your existing formula in an IFERROR and specify what to return if an error like #DIV/0 occurs.

 

=IFERROR(SUMPRODUCT(--(B65:J65<>0),AR65:AZ65)/(SUMPRODUCT(--(B65:J65<>0))),"error text")

 

Chagne "error text" to whatever value you want to return if the formula returns an error.
