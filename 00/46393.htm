Hi Rebekah, I don't know if you've fixed your problem or not, but I've found a fix that's a lot less mucking about than what's suggested above.

1. Open your Apps screen and go to the complete list of apps using the arrow-in-a-circle at the bottom of the opening screen.

2. Look for app icons that don't have any image in them.  I had three.  When I clicked on them there was an error message that said Windows couldn't open them.  Underneath in smaller writing it said go-to-the-store.  Click on the link.

3. When the store opens, you'll get an error message that says the app won't work and there's a button that says Repair (or fix).  Press that and go make yourself some coffee while it downloads whatever it is.  You will see a run of dots going across the screen while it's downloading and the dots will go away when it's finished.  It takes a while if the repair is doing mail, calendar and contacts at the same time, which it did for me.

4. When you go back to your Apps screen the icon should now have an image inside it. If it doesn't, reboot your computer. 

This worked for me.  (I'm running Win 8.1).

how can i remove the write protected of my usb driive?

How can i remove the write protection of my usb drive on my netbook,, i cant erase the contents of my usb drive on my computer.
ganesh achar b say:
Hi,
 
You can remove write protection from your usb flash drive using the below method:

a. Open start menu, in the search bar type regedit and press enter. This will open the registry editor.

b. Navigate to the following path:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\StorageDevicePolicies

Note:
If the registry key StorageDevicePolicies does not exist, you will need to create it manually.
c. Highlight StorageDevicePolicies, and then create a New DWORD (32-bit) Value named as WriteProtect.
d. Double click the key WriteProtect in the right window and set the value to 0 in the Value
Data Box and press OK button
e. Restart your computer and try copying files into your USB drives.

Note: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article: Back up the registry in Windows 7

http://windows.microsoft.com/en-US/windows7/Back-up-the-registry
