lemp say:
In addition to the suggestions in other comments, when you create ("burn") a CD or DVD you need to "close" or "finalize" the disc before it can be read on another computer.  If you don't finalize the disc, you can add more files -- using the same computer that you used to originally create the disc -- but you likely will NOT be able to view any files on another computer.

So ... if you still have the Win98se computer, put the CD in its burner drive and use whatever burning software you have installed there to finalize the CD.

For more help (assuming that you do have the Win98se machine and it's operable), please identify the name and version of the software you used to create the CDs in the first place.

EDIT:
If you don't have the original equipment that was used to burn the CD, it's likely that IsoBuster may be able to recover the files anyway.  I don't have personal experience with this program, but it's worth a try. 

Differences between free and licensed versions HERE.  Note particularly:
Part of the functionality related to the UDF and HFS(+) file-systems is not free and needs to be paid for. This is the extraction of files that come from the UDF [drag & drop software] or HFS(+) [Macintosh] file-systems. regsvr32 /u mmcndmgr.dll

regsvr32 /u mmcshext.dll

IN CASE THERE IS ANY ERROR MESSAGE THAT THE FILE COULD NOT BE FOUND, THEN PLEASE CLOSE THE ERROR MESSAGE AND CONTINUE. The file may not be part of Windows Vista's MMC. regsvr32 mmcbase.dll

regsvr32 mmcex.dll 

regsvr32 mmcfxcommon.dll 

regsvr32 mmcndmgr.dll  

regsvr32 mmcshext.dll
