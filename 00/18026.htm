WLAN not working on my E5-00 - Microsoft Community

hellothere---01 say:

WLAN not working on my E5-00

Hi.

The WLAN is not working on my Nokia E5-00.It keeps saying ''Invalid server name''.Can anyone help?

Please Nokia reply!!!

Any suggestions welcome!!!

Thanks!!!

chinyi---01 say:

i have same problem before but solved it by using the following steps

1) go to menu->Ctrl.panel->Settings->Connection->Destinations->Internet->WLANNAME

2)go to option->Advance Settings->IPv4 settings->Phone IP address "192.168.1.xxx"

Subnet mask "255.255.255.0", Default Gateway "192.168.1.254", Primary DNS address "192.168.1.254", Secondary DNS address "192.168.1.254".

 

Go to your gateway/router home page http://gateway.2wire.net

Set your gateway/router to DHCP mode, it will auto allocate addresses from 192.168.1.1 to 192.168.1.127 for new devices in your home network.

Then choose your phone IP address from the pool, example 192.168.1.124.  make sure it is not used by other network devices in your home network.

 

Now you should be able to connect to your home WLAN.
