bruno giordano say:

Then I have a solution for you. Try the following thing (I tested it on my PC and it worked).

Create a new folder somewhere else. In this example it will be "C:\Images". The Images folder is the one you'll add on the Library later.

Open the command promt with administrative privileges, and go to that folder by typping this:

c:
cd Images

Note: In Windows 7 you may open a Command Promt on a specified folder just making [Shift]+[Right Click] on it and Selecting "Open command window here" option. But, depending on where you create the Images folder, you may need administrative privileges to use the commands I'll be writing next.

In there, use this command for each of the folders you wanted to be on the Pictures Library.

mklink /d "01012010" ".../photos/01012010/images"

That code will create a "directory symbolic link" named 01012010 witch is in fact your ".../photos/01012010/images" folder. Please, be sure to write the full path to your images folder (the ".../photos/01012010/images" one).

The only thing you need to do then is to add the Images folder you created to the Library. In that way, you're adding only one folder witch "contains" all those that you wanted to add.

For more information about mklink, follow this: http://technet.microsoft.com/en-us/library/cc753194(WS.10).aspx

I hope that fixes your problem.
