cmdrkeene say:
It's not the shortcut itself, but where the shortcut points to.  Open your shortcut, and see if there is any mention of Library or Libraries at the top of the folder like this:


If you are inside a Library, any sort/filter/icon size/arrangement you apply is applied to the library itself.  Any folder you open in the library will take on this arrangement.  The folders can and do still remember their individual preferences, but you aren't opening the folder itself, you're opening the library.

This is intentional for libraries to apply a single style to everything in or opened from the library. It's to provide for consistency when navigating similar types of files (for example, all pictures folders in the picture library will look the same).

If you want to use your per-folder remembered settings, just don't use the library.  Delete your shortcut, then open your folder without using libraries (open Computer > C Drive > Users > etc... until you get to your folder.  Then make the shortcut to that.  The views you set for that folder will still be remembered.

Hi Jeeva,

It could be due to missing/corrupt drivers or missing updates. 

Let us try these steps.

Restart the computer a few times and check for Windows Updates.

In the Search box on the taskbar, enter Windows Update, select Windows Update. On the Windows Update page, select Check for updates.

Roll back the network adapter driver:

  1. In Device Manager, select Network adapters>the network adapter name.
  2. Right-click (or press and hold) the network adapter, and then select Properties.
  3. In Properties, select the Driver tab, select Roll back driver, then follow the steps.
  4. In the search box on the taskbar, type Device Manager, and then select Device Manager from the list of results. If the button is unavailable, that means there's no driver to roll back to.
  5. After rolling back to the previous version of the driver, select the Start button Power>Restart if asked to restart, and see if that fixes the connection issue.
