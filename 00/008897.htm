wyman w say:

Although I don't fully understand the process, but I think I understand what you want to do for the hide and unhide part. If I am correct, you are trying to hide the complete row in Sheet 2 if any row in Sheet 2 is blank.

Below is an example, you can define stCol (start column number), endCol (end column number),stRow, endRow. My example demonstrate how to hide blank rows and unhide non-blank rows for data range A1:J20

To insert the code:  ALT+F11 > Double click Sheet2 in Project Explorer > Paste the code

Private Sub Worksheet_Activate()
    stCol = 1
    endCol = 10
    stRow = 1
    endRow = 20
    For r = stRow To endRow
        counter = 0
        For c = stCol To endCol
            If Cells(r, c).Value = "" Then
                counter = counter + 1
            End If
        Next c
        If counter = endCol Then
            Rows(r).EntireRow.Hidden = True
        Else
            Rows(r).EntireRow.Hidden = False
        End If
    Next r
End Sub

Installation:

 

At first, trying to install Zune from the complete package, if the error "0x800 ..." persists then run the installation as follows:

 

You must extract the files contained in ZunePackage.exe (with winzip for example right click> extract) and then try installing from the folder Zune x86/Packages (or x64 depending on your version: Control Panel> System to check)> then run-x86.msi zune (zune-x64.msi or if 64-bit Windows)

 

- If you encounter errors during installation ... ignore them...

 

-If the installation hangs at the end when setting, check although there is no notification in the taskbar asking to get Zune ... In this case, select the cancel on this window.
