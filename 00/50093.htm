steve winograd say:

The quota that's being exceeded isn't local storage and isn't on your computer.  It's in your mailbox on the server computer that belongs to your E-mail service provider.  You've probably been leaving messages on that server when you download them to your computer, and that mailbox is now full.

Many E-mail service providers let you access your mailbox by going to a web mail site in your Internet browser.  Once there, you can look at your mailbox and delete messages. Or, you can contact your E-mail service provider for help.

You can configure Windows Mail to automatically remove messages from your mailbox on the server computer:

  1. Click Tools > Accounts .
  2. Double-click your E-mail account.
  3. Click Advanced .
  4. Un-check Leave a copy of messages on server and select one or both of the removal options.

debleena s say:

Hi  Dragon Twins,

As you said you have tried copying the media file in other drive also, it means that this particular file has got corrupted by some reason so we are getting this error message. I would suggest you to download the file again and then try to copy the file and check if it works.

If the media file is present in a CD try to copy the file in a different computer and check.

If the .mkv file is saved on your computer I would suggest you to run chkdsk to the bad sectors of the hard drive.

1. Click on start.

2. Type cmd in start search box.

3. Right click on cmd.exe in the Program list and then select the option Run as administrator.

4. If you are prompted for an administrator password or for confirmation, type your password, or click Continue.

5. In the command prompt window, Type in the commands listed below i.e. (/f, /r, /i, /c).

   In the command prompt, type chkdsk or chkdsk C: followed by one or a combination of switches listed   below and press Enter.

Type One of these Commands to Run Chkdsk:

1. /f = The /f switch is the most common of the chkdsk switches. It tells chkdsk to try and fix any errors it finds.

2. /r = The /r switch tells chkdsk to attempt to recover any bad sectors of the hard drive if any are found. A bad sector is a spot on the hard      drive that cannot hold saved data anymore. (See screenshot below)

3. /i = The /i switch tells chkdsk to perform a less detailed but faster disk check. (See screenshot below)

4. /c = The /c switch tells chkdsk to skip the checking of cycles within a folder structure which reduces the scan time.

During the restart process, Windows checks the disk for errors, and then Windows starts.

Note: While repairing the bad sectors of the hard drive it will remove the corrupted files and repair the bad sectors so there is slight chance of losing data.

 

Hope the information helps. Please post back and let us know.
