hyrulebalverine say:

Whenever you purchase something you get two specific licenses for that item: a Profile license and a Console license. The profile license allows you to access the content you purchased from any console; you just have to be online to use it on any console other than the one with the console license. A console license allows any profile on said console access the licensed content, even if offline. The console licenses are what get moved, together, in a license transfer. The actual content is not moved, so you would need to redownload any content to the console you are license tansferring to that was not already on it.

Based on this and how I understand your situation, it seems your best option would be to license transfer to the living room console so that you can play the games offline with no issues. Note that you will still need to download any content to this console you wanted to play that is not currently there. You will still be fine with Stacking as the patches are not automatically downloaded; just make sure to say no to the update if you try to play the game while online with the living room console. Because you are always online with the console in your room, you should have no problems accessing any of you content after a license transfer to the living room console.
