blue13x say:

Copy MMC Back-up back to the memory card

Hi, I was told to use a memory card reader to make an image copy of the memory card.
This was in case I wanted to restore it the way it was when I fisrt got it.

Can I now just reformat the card and copy the files on to the card using the card reader. This also includes the hidden "system" folder.

petrib say: Yes.

On Windows (2000 or XP), the best way to copy every single file from the card to the PC is, in my opinion, by using the XCOPY command:

XCOPY s:\*.* d:\backup /S /V /H /K

Where:

"s:" = the drive letter that represents the memory card

"d:\backup" = is the drive and directory (folder) where you want to save the stuff off the card on the PC

"/S" = copies all subdirectories

"/V" = verifies each file (checks that it was copied correctly)

"/H" = copies also hidden files

"/K" = copies also file attributes

After the XCOPY is done, you can do it in reverse to put things to the new card:

XCOPY d:\backup\*.* s: /S /V /H /K

After you've verified that the new card contains everything, you can reformat the old card (or just delete all files and directories from it).

kenpremo say:

excel macro to rename tabs - some with duplicate names

I am trying to name tabs in a workbook with the name listed in a cell of that worksheet. My problem is that some of my worksheets will have tabs of the same name, making the macro stop. Is there any way to append duplicate sheet names with a number to make the tabs unique?

I use the following code to rename tabs in a workbook,

Sub RenameTabs2()

' Renames all worksheet tabs with each worksheet's cell B2 contents.

'If cell B2 has no content, then that tab is not renamed.

    Dim i As Integer

    For i = 3 To Worksheets.Count

        If Worksheets(i).Range("B2").Value <> "" Then

            Worksheets(i).Name = _

            Left(Replace(Replace(Replace(Replace(Replace(Replace(Replace(Replace(Replace( _

                Worksheets(i).Range("B2").Value, "|", ""), ">", ""), "<", ""), Chr(34), ""), "?", _

                ""), "*", ""), ":", ""), "/", ""), "\", ""), 31)

        End If

    Next i

End Sub
