My password has stopped working. - Microsoft Community

jt1982yr say:

My password has stopped working.

My password has stopped working - that is I cannot login with my correct password. I have an Admin password and understand I might be able to lLog on as Administrator in Safe Mode and then reset the password. How do I do that?

jane lla say:

Hi JT1982YR,

Roll up your sleeves. This is going to be long, but please bear with me. I'll try to be as detail as possible.

Boot your computer. Wait for the Windows logo to appear, then force shutdown your computer. This will trigger the system to boot into the Windows Error Recovery once you reboot the PC.

Make sure that the Launch Startup Repair (recommended) option is highlighted. Then press Enter.

It might take a while for Windows to commence the repair. A Startup Repair window will pop-up showing you the process of the scan. (Do not close this window. You'll need it for later.)

Another pop-up box will appear asking you if you want to restore the computer using System Restore. Click on Cancel.

Once the scan completes, your computer should display this window:

Click on the drop-down button for View problem details. Select the last link *.txt (the privacy statement hyperlink for offline reading). This will open a text file in Notepad. Now, follow this path:

File > Open > Computer > Local Disk (C:) > Windows

Click the drop-down option for the Files of type. Change it to All Files. Then, scroll down and look for the Microsoft.NET folder > System32 > sethc (this is a .exe file)

Once you locate this application (sethc), you need to rename it to sethc1 by right-clicking on your mouse.

After doing so, look for cmd. Make a copy of cmd, then paste it. Rename the copy of cmd to sethc.

Once done, click on the Cancel button. Close the notepad. Close the second pop-up box, the one that has the Hide problem details drop-down button.

On the Startup Repair window (very first window), click on the Cancel button. This will allow Windows to restart. A notification will ask you if you want to exit Startup Repair and restart the computer. Click on Yes.

Once your PC boots, wait for the sign-in screen, then press the Shift key five times (fast). This will launch the cmd box. A pop-up box might appear asking you if you want to turn on the Sticky keys. Just select No.

On thecmd box (sethc.exe) type: net user
It'll give you the list of all the users. Select the user you want to change the password for.

Type net user (the username of the account you want to regain access to) *

It will ask you to key in the new password. Just leave it blank. Then, press Enter. Exit cmd.

It will now allow you to access the account.

Sorry for the lengthy post. Hope you were able follow through.

Looking forward to your update.

All the best.
