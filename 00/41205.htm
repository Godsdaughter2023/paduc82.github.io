How will I know if my computer can run Windows 10?

System requirements

If you want to upgrade to Windows 10 on your PC or tablet, here’s what it takes.

Latest OS:  Make sure you are running the latest version either Windows 7 SP1 or Windows 8.1 Update.

For Windows 8.1, see the following article:

http://windows.microsoft.com/en-us/windows-8/install-latest-update-windows-8-1

For Windows 7, see the following article:

http://windows.microsoft.com/en-US/windows7/learn-how-to-install-windows-7-service-pack-1-sp1

Processor:  1 gigahertz (GHz) or faster processor or SoC

RAM:  1 gigabyte (GB) for 32-bit or 2 GB for 64-bit
Hard disk space:  16 GB for 32-bit OS 20 GB for 64-bit OS
Graphics card:  DirectX 9 or later with WDDM 1.0 driver
Display:  1024x600

How much actual free disk space is required by Windows 10 setup?

Windows 10 uses approximately 10 GBs of disk space. The minimum disk space available is to facilitate the installation process. Some install files are compressed and need to be expanded during setup. Some files are temporary and are deleted during or after installation. Also, because you might have data and apps on the hard disk, they will be moved around during migration.

Learn more:

http://www.microsoft.com/en-US/windows/windows-10-specifications

lastgas say:

The iCloud Photos process is repeatedly restarting Explorer.exe (Explorer also manages the desktop). That's what causes the refreshes that you're seeing, and killing this iCloud Photos process will stop the restart.

I presume that this problem is related to the general Folders and Permissions problem that many Windows 10 users are having, caused by OneDrive. What I suggest below is what I did to deal with that problem, which stopped the iCloud problem there.

What you'll probably see if you look at Properties | Security on your iCloud Pictures directory is that your personal account doesn't have an entry. (Administrator only helps you if you run a program As Administrator). If you walk that back up to X:\Users\<your account>\Pictures, you'll probably find the same problem. You can't just add the permissions because OneDrive will remove them at the next reboot. 

What I did was to go up to X:\Users and update the User (under Properties | Sharing) Permissions, adding your account with Full Control to the share. Then go to the Pictures subdirectory and Security | Advanced, makinge sure Inheritance is enabled. Then do the same with iCloud Photos. My guess is that will fix everything.
