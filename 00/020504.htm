ken sheridan say: I assume that you want the text box visible only if the location is any of the three values, in which case put the following in the Format event procedure of whichever section the text box is located in, probably the detail section:

If [LOCATION NO] = 1742 Or [LOCATION NO] = 1726 Or [LOCATION NO] = 1725 Then
    Me.Textbox125.Visible = True
Else
    Me.Textbox125.Visible = False
End if

This assumes that LOCATION NO is a number data type.  If it's a text data type put:

If [LOCATION NO] = "1742" Or [LOCATION NO] = "1726" Or [LOCATION NO] = "1725" Then
    Me.Textbox125.Visible = True
Else
    Me.Textbox125.Visible = False
End if

Make sure that the CanShrink property of both the text box and the section in which it is located is True (Yes).

If you are unfamiliar with entering code into a section's event procedures, this is how it's
done in report design view:

1. Select the section by clicking on its title bar and open its properties sheet if it's not already open.
2. Select the On Format event property and select the 'build' button (the one on the right with 3 dots).
3. Select Code Builder in the dialogue and click OK. This step won't be necessary if you've set up
Access to use event procedures by default.
4. The VBA editor window will open at the event procedure with the first and last lines already in
place. Enter or paste in the code as new lines between these.
