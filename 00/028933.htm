Round Robin Tournament - Microsoft Community

hughneedshelp say:

Round Robin Tournament

Hi,

I'm trying to organise a charity (field) hockey tournament, where each team plays every other team once.  The trouble is, I don't know how many teams will turn up on the day, so I need a dynamic way of doing this, and I just can't work it out.

I have two pitches, so two games can run concurrently, but clearly the same team can't play on two pitches at the same time.  Other than that, the order of the matches is irrelevant.

I know how many games there will be: Number of Games (G) =((E3*E3)-E3)/2, where E3 contains the number of teams (T).

I (think I) need a table with the first column "Game #" shows 1 to G (from the above formula), the next column "Team A" will show one team, the next column "Team B" will show Team A's opponents and the fourth column "Pitch" will show either 1 or 2 denoting the pitch the game is to be played on to avoid a clash.

I don't have Team Names yet, either, but that's ok as I can use numbers and substitute later with a vlookup.

All the templates I've founds are for knockout / bracket type tournaments, but none for a Round Robin.

Thanks in advance.
bernie deitrick say: Enter the number of teams in cell E3, then run this macro:

Sub RoundRobin()
    Dim i As Integer
    Dim j As Integer
    Dim k As Integer
    Dim boolConflict As Boolean
    Dim myC As Range
    Dim myR As Range
    Dim x As Single

    Range("A:D").Clear

    Range("A1").Value = "Game #"
    Range("B1").Value = "Team A"
    Range("C1").Value = "Team B"
    Range("D1").Value = "Field"

    k = 1
    For i = 1 To Range("E3").Value - 1
        For j = i + 1 To Range("E3").Value
            Cells(k + 1, 1).Value = k
            x = Rnd() * 100
            If x > 50 Then
                Cells(k + 1, 2).Value = i
                Cells(k + 1, 3).Value = j
            Else
                Cells(k + 1, 3).Value = i
                Cells(k + 1, 2).Value = j
            End If
            Cells(k + 1, 4).Formula = "=RAND()"
            k = k + 1
        Next j
    Next i

    boolConflict = True

    While boolConflict
        Range("B1:D" & k).Sort key1:=Range("D2"), order1:=xlAscending, Header:=xlYes
        boolConflict = False
        For i = 1 To k Step 2
            Set myR = Range("B1").Offset(i, 0).Resize(2, 2)
            For Each myC In myR
                If Application.WorksheetFunction.CountIf(myR, myC) > 1 Then boolConflict = True
            Next myC
        Next i
    Wend

    Range("A1:D1").Interior.ColorIndex = 34

    For i = 1 To k - 1 Step 2
        Set myR = Range("B1").Offset(i, 0).Resize(2, 2)
        Intersect(myR.EntireRow, Range("A:D")).Interior.ColorIndex = IIf(((i - 1) / 2 Mod 2) = 1, 36, 35)
        Cells(i + 1, 4).Value = 1
        If Cells(i + 2, 3).Value <> "" Then
            Cells(i + 2, 4).Value = 2
        Else
            Cells(i + 2, 4).EntireRow.Delete
