Dim ArrFind As Variant

For i = 0 To UBound(ArrFind)

 .ClearFormatting
 .Text = ArrFind(i)
 .Forward = True
 .Wrap = wdStop
 .MatchWildcards = True
 End With

While Selection.Find.Found = True
             
strLength = Selection.Range.Characters.Count
For j = 1 To strLength

   Selection.Range.Characters(j).Delete
 End If

Selection.Collapse Direction:=wdEnd
Selection.Find.Execute

Next i

macropod say:

For a generalised routine that will delete the periods from any UPPER-CASE acronym, you could use:

Sub Demo()
Application.ScreenUpdating = False
With ActiveDocument.Range
  With .Find
    .ClearFormatting
    .Replacement.ClearFormatting
    .Text = "<[A-Z].[A-Z.]{1,}"
    .Replacement.Text = ""
    .Forward = True
    .Wrap = wdFindStop
    .Format = False
    .MatchWildcards = True
    .Execute
  End With
  Do While .Find.Found
    If .Words.Last.Next.Characters.First Like "[a-z]" Then
      .Text = Replace(.Text, ".", "")
    Else
      .Text = Replace(.Text, ".", "") & "."
    End If
    .Collapse wdCollapseEnd
    .Find.Execute
  Loop
End With
Application.ScreenUpdating = True
End Sub

Const StrAcr As String = "|C.I.A.|I.B.M.|U.K.|U.N.I.C.E.F.|U.S.A.|"
With ActiveDocument.Range
  With .Find
    .ClearFormatting
    .Replacement.ClearFormatting
    .Text = "<[A-Z].[A-Z.]{1,}"
    .Replacement.Text = ""
    .Forward = True
    .Wrap = wdFindStop
    .Format = False
    .MatchWildcards = True
    .Execute
  End With
  Do While .Find.Found
    If InStr(StrAcr, "|" & .Text & "|") > 0 Then
      If .Words.Last.Next.Characters.First Like "[a-z]" Then
        .Text = Replace(.Text, ".", "")
      Else
        .Text = Replace(.Text, ".", "") & "."
      End If
    End If
    .Collapse wdCollapseEnd
    .Find.Execute
  Loop
End With
Application.ScreenUpdating = True
End Sub

To uninstall Internet Explorer 10, follow these steps:

 

a.     Log on to the computer as an Administrator or with an account that has administrative rights.

c.     Click Start, type cmd in the Search box and then click cmd.exe from the programs list.

FORFILES /P %WINDIR%\servicing\Packages /M Microsoft-Windows-InternetExplorer-*10.*.mum /c "cmd /c echo Uninstalling package @fname && start /w pkgmgr /up:@fname /norestart"

e.     Restart the computer to complete the uninstall of Internet Explorer 10.

jay freedman say:
Hello Abhay,

There is no shortcut key assigned to those functions by default, but you can assign one or more for your own use. Each shortcut you assign can only condense or expand the spacing by one predefined amount.

Click the Office button and click Word Options. In the Options dialog, click Customize on the left side and then click the Customize button at the bottom. That opens the Customize Keyboard dialog. Set the category in the left-hand list to All Commands.

In the command list on the right, click the entry for Condensed: . That makes a dropdown box appear, with a list of numbers of points. Let's say you want a shortcut key to condense the selected text by 0.3 point, so select 0.3 pt in the list. Click in the "Press new shortcut key" box and press the shortcut you want to use for this command -- for example, Ctrl+Alt+Shift+3. Then click the Assign button.

Now, you can pick a different amount to condense, and assign a different shortcut to that command. Or you can go back to the command list and click the entry for Expanded: , and assign a shortcut for expanding by a certain amount.
