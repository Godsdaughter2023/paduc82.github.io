mike h.. say:

Hi,

Here's a way with VB code. All 16 numbers appear once and 4 numbers appear twice.

ALT+F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in on the right and run it.

Sub somesub()
Dim c As Range
Dim FillRange As Range
Dim Low As Long, High As Long
Low = 1
High = 16
Set FillRange = Range("A1:A16")
For Each c In FillRange
Do
c.Value = Int((High * Rnd) + Low)
Loop Until WorksheetFunction.CountIf(FillRange, c.Value) < 2
Next
Set FillRange = Range("A17:A20")
For Each c In FillRange
Do
c.Value = Int((High * Rnd) + Low)
Loop Until WorksheetFunction.CountIf(FillRange, c.Value) < 2
Next
End Sub
