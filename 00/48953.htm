John,

A factory image restoration is only relevant for computers that came with Windows 10 preinstalled.  If it did not then do not pursue the notion of a factory restore at all. 

The computer maker might provide a download of the factory restore partition and instructions for how to use it to create a factory restore USB.  If they do then that would be the most direct solution. 

  • Dell use this approach for their latest Windows 10 computers but computer makers differ so you should go to their online support website and see how they advise you to go about doing a factory image restore if the intended procedure cannot be used.
  • For Dell, use the computer's Dell Service Tag to go to the Drivers and downloads section for her computer and you'll find the Dell Windows 10 "ISO" there together with a link to its instructions.

You can always make a standard reinstallation USB using http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install

It might be the case that booting from an installation disk but choosing Repair your computer [rather than Install now] then Advanced, Troubleshot, Start-up repair might fix the whole thing for less effort but it would be impossible to predict that in advance. 

  • Start-up repair can also be accessed by other means and your reference to have been playing  in the Command prompt implies that you might have already tried this.
  • Even if Windows would start afterwards, I think I would be concerned that it had other damage that would come back to bite me later so I think I would reinstall anyway.  But that's just my opinion.

Best of luck,

Hi,

When you create a simple storage space, it will write data in a stripe across physical disks (in your case, 3 disks) without any extra copies or parity information. This type of storage space will not offer resiliency or data protection. Data corruption and data loss will occur when one of these drives fails.

Since simple space, or simple layout does not provide any data protection, it is best suited for use on files with backups, or users who are okay with losing or recreating the data if a disk fails.

You may still be able to recover your data using any third-party recovery tools but data integrity is not guaranteed.
