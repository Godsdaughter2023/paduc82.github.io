bytes count - Microsoft Community

mvdec say:

bytes count

how can I find out the byte count on my document? 

jay freedman say:
Word can show you a "character count" in the Word Count dialog, but that isn't the same (it doesn't include graphics or a lot of other things). The simplest method is to open the folder in Windows Explorer and look at the file's size.

If you want to do this within Word, you need a macro (see http://www.gmayor.com/installing_macro.htm if needed):

Sub GetFileSize()
    Dim fs, f, s
    Set fs = CreateObject("Scripting.FileSystemObject")
    Set f = fs.GetFile(ActiveDocument.FullName)
    s = f.Name & " uses " & f.Size & " bytes."
    MsgBox s, 0, "File Size Info"
End Sub
