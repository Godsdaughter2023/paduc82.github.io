Welcome to Microsoft Community. I will certainly guide you in the concern you mentioned.

I would like to inform you that there is no such tweak or fix from Microsoft which would help you change the default System Restore settings. There might be some third-party tools which may help you with this, but Microsoft does not recommend to use any such software or tweak. However, if you want to schedule a System Restore point on the time and day you want, you may do so using the task scheduler in Windows 8.1. To do so, refer to the steps below:

  1. Press Windows + X. Click on Control Panel.
  2. At the top right corner, click on the search box and type 'Task Scheduler'.
  3. Click on Task Scheduler. Once it's open, look under Microsoft \ Windows \ SystemRestore on the left-hand pane.
  4. Now you’ll see a single item in the middle pane that says “SR Queued Multiple triggers defined”. Double-click on that item to change the details.
  5. Check the 'triggers' tab. To change the trigger, click on one of the items and then Edit to change the trigger.
  6. Choose the settings that you want and click Ok.

Note: You may also add conditions similarly according to your need.
