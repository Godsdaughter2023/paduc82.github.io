greginmich say:

Windows Defender along with the free versions of Malwarebytes and SuperAntiSpyware is a classic combination for AV protection. It’s exactly the combination that I was recommending here many years ago when the regulars still rejected the idea of using any third-party malware-removal tools at all. But in those days Security Essentials was a real contender, and it was getting respectable scores from the AV testing labs. These days, the scores for Microsoft AV apps are literally used as the baseline for the protection scores at the AV testing labs. That means that if you’re not extremely careful with your browsing, then your frontline (real-time) protection is going to miss more malware and leave it for your backup scanners to clean up. That’s what “supplementing your protection” really means. So if you stick with Windows Defender, and your browsing habits leave something to be desired, then you’ll most likely need to do a lot of “supplementing” – in which case you should definitely expand your arsenal with some of the more specialized malware-removal apps mentioned here:

http://www.bleepingcomputer.com/forums/t/407147/answers-to-common-security-questions-best-practices/?p=2413189

But taking this approach doesn’t just mean more cleanup work and system repair – it also involves substantially more risk, because once you lose your personal files, or your personal information, as the result of a breech in your frontline defense, then the damage is already done, and no amount of after-the-fact cleaning up can undo that damage. So having better frontline protection really means having more protection, as well as less cleanup work to do:

http://www.pcmag.com/article2/0,2817,2388652,00.asp

greginmich say:

Well wait a minute now: Your screenshot shows a Windows 10 desktop with Windows Defender pinned to the taskbar, and we know that no version of SuperAntiSpyware (looks like you have the free version) would be registered by the Security Center as your virus protection (“primary” protection) because SuperAntiSpyware doesn't include virus protection. So installing it wouldn't disable Windows Defender, and if Windows Defender’s status is “Protected” then everything is fine and the problem is just with the semantics.

Your primary security program is the one that provides virus protection, and you can always check its identity and status in the Security Center: right-click on the Start button and chose Control Panel  > Security and Maintenance > Security > Virus protection. If Windows Defender is running, then that would be your “primary” protection, because it provides continuous (real-time) protection against viruses (as well malware), by scanning files as they’re accessed (on-access); whereas the free versions of Malwarebytes and SuperAntiSpyware only scan for malware, and they only examine your files when you run a scan (on-demand). So Windows Defender has the ability to actually prevent viruses and malware from installing; whereas Malwarebytes and SuperAntiSpyware can only clean up malware that gets installed when it's not detected by Windows Defender. Therefore Windows Defender would be the “primary” protection; aka “real-time”, “on-access”, or “frontline” protection.

How to preserve original image quality for a inserted image in - Microsoft Community

yujaungkim say:

How to preserve original image quality for a inserted image in Word?

I made a figure and saved as PDF file.

Whenever inserting a PDF (containing an image in letter size), the image quality is considerably worse than the original.

And I tried also saved that PDF file as png file and inserted on Word, that didn't look different at all.

How to insert a picture into the Work without losing the quality?

Any suggestion?

john mcghie say:

Print it, and make sure you still have a problem!

When Word displays a PDF image in a document, it actually makes and displays a low-resolution image known as a "placable header".  That image is 72 dpi, and looks fairly average :-)

However, when you print it, Word will use the actual PDF image data that is also stored in the document.  That will print at unlimited resolution.

Mind you, you DO have to ensure that the PDF contains an EPS image!  If your PDF contains a raster graphic (GIF, TIFF etc) then the resolution was baked in to the PDF and there's nothing Word can do to improve it.

PDF is, after all, a "container": the key is "what does it contain?"

OH:  Almost forgot...  you ARE saving the file as a .docx, right?  There has been a nasty rumour running around that .doc and .docx formats are equivalent.  They are not, and never were.  If you downgrade a file to the obsolete .doc format, Word will remove information that can't be stored in the obsolete format.  That WILL give you a case of the fuzzies with vector images.

Hope this helps

rightpaddock say:

Defender - Update Automatically

As a general rule I don't like automatic updates, so I have WU set to inform me when any updates are available. I am very happy with this approach for infrequent Windows, Office, and other updates, nominally once a month. 

But I would like Defender definition updates to be installed automatically and silently - like MSE did on W7. 

Is there any way to do this, if not how about an option - preferably in Settings, or gpedit if you must ;)

Windows 10 Pro - 1607.

pa bear - ms mvp say:

...I would like Defender definition updates to be installed automatically and silently - like MSE did on W7

Don't hold your breath. Unlike MSE (and Defender) in Win7, Defender in Win10 (as well as Win8.1) will ONLY auto-update itself when Windows Update is set to the default Automatic configuration.

If you wanna leave it set to Notify Only, you'll have to update Defender manually.

Like MSE, Defender considers itself sufficiently up-to-date as long as its Definitions are no more than 72 hours old.

prashantkumar96 say:

You can maybe add a Task in Task Scheduler to run updates daily. See this guide on how to create a task:

http://www.sevenforums.com/tutorials/12444-task-scheduler-create-new-task.html

Create a task to run this program:

"C:\Program Files\Windows Defender\MpCmdRun.exe" -SignatureUpdate

Or if you want to see the interface while updating:

"C:\Program Files\Windows Defender\MSASCUI.exe" -Update
