First, let's see if this is an easy fix or if it's going to give us more trouble.

 

Icons in the notification area sometimes are hidden if inactive.  This can happen with the MSE icon as well.  First, check to see if there is a <  to the left of the notification area - if so, click it to see hidden icons.  If the MSE icon is there, then you can leave things as they are and check if you need to or, if you want it showing all the time, right click on the Start Menu and click on Properties.  Click on the Notification area Tab and click Customize.  Find the MSE icon and click the dropmenu next to it to change it from "Hide when idle" to "Show" and save the changes.  If it doesn't appear, open the notification area to see the hidden MSE icon and click on it.  That should bring it to the front and showing. That should do the trick.  If it disappears again, reboot and that should set the setting and it should now show all the time.

 

I hope this helps.  If not, post back and we'll look deeper into the problem.

 

Good luck!

The question about booting to the desktop means that you are not using a login ID and password. having that in place slows things down so that the services and such can start first -- as long as you don't immediately log in.
