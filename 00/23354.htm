gary's student say:

You can create a set of tiny macro to perform formatting and then assign keyboard shortcuts to these macros.  For example:

 

Sub FormatC1()
With Selection
    .HorizontalAlignment = xlCenter
    .NumberFormat = "0.0"
End With
End Sub

You can assign CNTRL-SHFT-G to this macro.  Just select the cells and touch the shortcut keys to format centered with one decimal place.

rich007a say:

Hi,

You can record a macro to do this.

First ensure one cell is selected.

In the bottom left corner of Excel, there is a button with a round red dot.  Click it.

You get a window which prompts for a macro name, give it something sensible like "Centre1DP" (align centre with 1 decimal place).

Enter G (shift+G) in the shortcut field.

Set save in Personal Macro Workbook (so it will be available to all your workbooks).

Now apply the required formatting to the selected cell.

Now stop the recorder with the blue sqaure in the bottom left corner of excel.

 

To do the super- and sub-script, do you want to apply that to an entire cell?  That's the only way you could do it with a macro.

 

You can now edit the code if necessary.  Press Alt+F11 to open the VB editor.  You should see your macro(s).

Some extra code will get recorded, but all you really need is to make them like this:

Sub Centre1DP()
' Makes selected cell aligned centre and 1 decimal place
'
' Keyboard Shortcut: Ctrl+Shift+G
'
    Selection.HorizontalAlignment = xlCenter
    Selection.NumberFormat = "0.0"
End Sub


Sub MakeCellSuperscript()
' Makes selected cell super-script
'
' Keyboard Shortcut: Ctrl+Shift+R
'
    Selection.Font.Superscript = True
End Sub

 

Sub MakeCellSubscript()
' Makes selected cell sub-script
'
' Keyboard Shortcut: Ctrl+Shift+C
'
    Selection.Font.Subscript = True
End Sub

 

Sub MakeCellNormalscript()
' Removes sub- or super-script from selected cell
'
' Keyboard Shortcut: Ctrl+Shift+???
'
    Selection.Font.Subscript = False
    Selection.Font.Superscript = False
End Sub

 

Note that the easiest (and, I have to admit, only way I currently know) to set the key-board shortcut is to record a macro.  You can then edit the macro if/as required.  Changing the text  " ' Keyboard Shortcut: Ctrl+Shift+??? " in the comment at the top of the macro will not change the keyboard assignment.

I included a fourth macro so you can undo any sub or super script formatting.

 

Hope that helps.

Cheers
Rich

[EDIT: OK, I do know how to change a keyboard shortcut for a macro.  Make sure you have the developer tab visible.  If not, go (round) Office button > Excel Options  and tick Show Developer Tab in the Ribbon. Ok.

Now go to the Developer tab > Macros.  You will see all of the macros you have created.  Select a macro and click the Options button and you get a similar screen you get when starting to record a macro.

So you could just copy and paste all of the above macros in, then assign the shortcuts.]
