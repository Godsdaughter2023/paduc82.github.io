If I go into Outlook Search options everything under the Indexing and Search catagory is also disabled. timofcourse say: In case anyone else stumbles upon this same issue, I found the answer on the Outlook discussion groups here .

Just needed to change a couple registry settings to get this working again.
HKLM/Software/ Policies/ Microsoft/Windows/Windows Search
PreventIndexingOutlook set to 1

HKLM/Software/Microsoft/Windows Search
SetupCompletedSuccessfully to 0

Reboot and all is well!
