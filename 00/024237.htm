derekgillespie- [o365] say:

I was trying to do something similar but for every user not just certian ones.  Here is what I did that worked for me.

1. I wanted to view the mailbox folder permissions for the Test Room calendar.

Get-MailboxFolderPermission test-room:\calendar

2. I then set the Test Room calendar to show limited details by default.

Set-MailboxFolderPermission -AccessRights LimitedDetails -Identity test-room:\calendar -User default

3. Finally, I set the Test Room calendar to show the Organizer and Subject of every meeting.

Set-CalendarProcessing -Identity test-room -AddOrganizerToSubject $true -DeleteComments $false -DeleteSubject $false

You can usally run the "Get" command before the "Set" to see what current values are.  All you have to do is figure out how to assign #2 to only certain users.  Please see example 1 in this link for reference: technet.microsoft.com/.../ff522363(v=exchg.160).aspx

Select the cells and run:

 

 

Sub RefreshCells()
Dim r As Range, rr As Range
Set rr = Selection
For Each r In rr
    r.Select
    Application.SendKeys "{F2}"
    Application.SendKeys "{ENTER}"
    DoEvents
Next
End Sub

 

 

joeu2004 say: Vishu wrote:

I have data in around 200 cells. I need to go to each cell and press “F2” button and then “Enter” button.  Can I have macro codes,  that should perform above task in selected cells. I could not get these code by recording.


Of course, this is needed, for example, when you entered numbers or formulas into cells originally formatted as Text, then you change the format to General or some other numeric formula.  As we know, Excel does not automatically change the cell contents immediately.  We must "edit" each cell by pressing F2, then Enter.

FYI, if your original data are constants, you can simply put the number 1 into some cell and copy it, select the cells to "edit", right-click and click on Paste Special, select Multiply, and press OK.  Then you can delete the number 1.

But if you have formulas, you can use the following macro.  Select the range of cells to change first.  The formula works only for cells with constants and non-array-entered formulas.  But then, so does your procedure of pressing F2, then just Enter.

Sub reEnter()
For Each r In Selection
    r.Formula = r.Formula
Next
End Sub

    How do I enter a variable with a subscript raised to a power? - Microsoft Community

    mdkimzey say:

    How do I enter a variable with a subscript raised to a power?

    In the equation editor in Microsoft Office 2010, is there a way to enter a variable with a subscript raised to a power.  If I use the script template that has both a subscript and superscript.  It will only allow me to enter either a superscript or a subscript but not both. 

     

    Is there a way to enter:

     

    \sigma_y^2

     

     

    jay freedman say:
    You can do this using either the linear entry method or the script templates.

    Linear:  Type  \sigma_y  and then press the space bar. The y becomes a subscript, and the cursor is on the main line to its right. Press the left arrow key to move the cursor down to the subscript line, but still to the right of the y. Type  ^2  and press the space bar again.

    Templates:  Insert the template with a main block and a subscript block. Put the cursor in the main block and insert a sigma. Move the cursor to the subscript block, and insert the template with a main block and a superscript block. Type the y in the main block, move to the superscript block, and type the 2.
