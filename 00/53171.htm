curmudgeonator say: Hello
Too bad this naming convention is so counter intuitive, but the "System" partition on a windows computer is where the files needed to "boot" the computer are located.  It is normally the first primary partition on the first disk enumerated by the computer's bios.  The "Boot" partition of a windows computer can be located on most any other disk and contains at least, the files for the windows os.  Looks like you have a dual boot system where one of your windows 7 versions is installed on the first primary partition which is where the files needed to "boot" the computer are located, that is why in each case, it is identified as the "System" partition.  It is good practice and can be less confusing with a multi-boot computer to create a separate "System" partition, then install each os on another partition.  That way if you have a problem with one of the os's, you can reformat without rendering the computer unbootable.  It also simplifies backups. This separate partition can be only 200-500mb in size.  You probably would not want to do this now unless you feel adventurous.

manuelcapiro say:
Hello MICHAELPMILLER,

To clear the files in the Downloads folder from your hard drive in Windows, follow the instructions below.

Press the Windows key on your keyboard, type Downloads, and then press Enter.
In the window that appears, highlight all of the downloaded files with your mouse or press Ctrl+A.
Press Delete to send the files to the Recycle Bin.
To permanently delete the files, right-click the Recycle Bin icon on your Desktop and select Empty Recycle Bin from the drop-down menu that appears.

If there isn't a Recycle Bin icon on your Desktop, you can restore it.
