The code for the HP_FYQ function contains a superfluous End If.

 

HP_FYQ expects a date as argument. If DET_Data.Date_Completed is Null (blank), it'll return an error, and you'll get the Type Mismatch error that you report.

Public Function HP_FYQ(ByRef TestDate As Variant) As Long

    If IsDate(TestDate) Then

            (Int((((Month(TestDate) + 1) Mod 12) / 3) + 1))
    End If
End Function

jay freedman say:

The key to this is to understand that a "watermark" in Word is really just a WordArt object or a picture that is anchored in a header (or it could be anchored in a footer). That means you can make your own "watermark" by clicking in a header and inserting a WordArt object or a picture without using the actual Watermark feature. 

Why would you want to do that? Well, Word has this "helpful" (ahem) habit that when you insert a second watermark anywhere in a document, it automatically deletes the first one. By inserting a "watermark" without using the Watermark button, you can avoid that.

I think I understand you to say that your document is exactly 4 pages, and that the third and fourth are duplicates of the first and second, respectively. You want one "watermark" on pages 1 and 2, and a different "watermark" on pages 3 and 4. If that's correct, then these steps will do:

  • At the end of page 2, click Insert > Breaks > Next Page to insert a section break. If you have a manual page break there, delete it.
  • Double-click in the header area on page 3 to activate it, which makes the Header & Footer Tools tab display on the ribbon.
  • In the Navigation group on that tab, click the Link to Previous button to turn it off. Then click the Previous button in the same group to go back to the Section 1 header.

How Many To, Cc and Bcc Recipients are Allowed Per Message

Again, suspicious activity can lead to a temporarily lower limit (as low as 10 (ten) recipients).

If you’ve been notified that you have hit some limit on a Microsoft product, it’s almost certainly part of an effort to reduce spam.
You can also increase your limit simply by continuing to use your email normally
It only takes a few days of normal activity to increase the limits—that gives us enough time to detect automated spam machines and accounts that have been hacked.
We’re constantly working to make things worse for spammers and better for real people.

If you are an established user, recent account activity may have sent us a false alert.
