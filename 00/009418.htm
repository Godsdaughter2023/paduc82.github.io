 b) In order to then adjust the players handicap, in line with their Stableford score and category, I need to set a formula in say A3 that says -  IF a given cell say A4 has a numeric value less than 36 then the value in A3 (their handicap) will increase by 0.1, If the the value in A4 is 36, nothing will happen to the A3 value and if the A4 value greater than 36, then the A3 value will decrease according to the player category noted above (A1) for every digit above 36.
