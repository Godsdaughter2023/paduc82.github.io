removing the #N/A error message from vlookup

Using Excel - Vlookup function to retrieve data and evaluate for decision.  Tables that data is retrieved from is a series of dates that are not present in all tables and end result is #N/A.  How do I remove the N/A after the evaluation?
jeeped say:
... How do I remove the N/A after the evaluation?

You are using Excel 2007 so you can take advantage of the reasonably new IFERROR() function. If you want blank cells on a #N/A return then wrap your VLOOKUP() with IFERROR() like this,
=IFERROR(VLOOKUP(A1,B1:C3,2,FALSE),"")
... or if you want a zero value then,
=IFERROR(VLOOKUP(A1,B1:C3,2,FALSE),0)

If compatibility with previous versions (e.g. 2003, 2002/XP, 200, etc) is required, the same method can be accomplished with IF(ISERROR()) like this,
=IF(ISERROR(VLOOKUP(A1,B1:C3,2,FALSE)),"",VLOOKUP(A1,B1:C3,2,FALSE))
... or,
=IF(ISERROR(VLOOKUP(A1,B1:C3,2,FALSE)),0,VLOOKUP(A1,B1:C3,2,FALSE))

In the first place, my apologies - I had left in the word Selection - that shouldn't have been there.

 

In the second place, VBA is case sensitive by default. So if you have a workbook Book1,

 

wbk.Name Like "book1*"

 

will not be true. The easiest way to get around this is to add a line

 

Option Compare Text

 

at the top of the module.

 

In the third place, Columns("A2:G5000") is not valid - that should be Range("A2:G5000").

 

Try this version:

 

Option Compare Text

Sub Macro11()
    Dim wbk As Workbook
    Dim wsh As Worksheet

    Set wsh = ActiveWorkbook.Worksheets("Download")

    For Each wbk In Workbooks
        If wbk.Name Like "book1*" Then
            wbk.Worksheets(1).Columns("A:G").Copy _
                Destination:=wsh.Range("A1")
            Exit For
        End If
    Next wbk

    For Each wbk In Workbooks
        If wbk.Name Like "book2*" Then
            wbk.Worksheets(1).Range("A2:G5000").Copy _
                Destination:=wsh.Range("A1").End(xlDown).Offset(1, 0)
            Exit For
        End If
    Next wbk

    For Each wbk In Workbooks
        If wbk.Name Like "book3*" Then
            wbk.Worksheets(1).Range("A2:G5000").Copy _
                Destination:=wsh.Range("A1").End(xlDown).Offset(1, 0)
            Exit For
        End If
    Next wbk
End Sub
