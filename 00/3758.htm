daniel g. grau say:
The "gutter" is a non-printable area used for binding the pages, so that the text won't be hidden into the stapled area and become hard to read. If you set the gutter to, say, 2 inches, then all text will be moved that far to the right (for odd pages) and to the left (if you have even and odd pages). That's not what you need.

First, you need to set the page orientation. Triptychs are usually done on a landscape page. If this is what you need, then do File > Page Setup and click on the second icon next to Orientation.

Then you need to move the margins real close to the edge of the page. To do this, Format > Document and change the first four values (Top, Bottom, Left and Right) as needed. Set Gutter at zero. Depending on your printer, you may get a "Fix margins?" warning. If your printer cannot print all the way to the edge of the page, there will be blank stripes all around the page, so heed this warning and Word will set the largest margins it can.

Then you need to define three columns (Format > Columns).

Try enabling this setting as a visual guide: Word > Preferences > View > Show > Text boundaries.

paulcordts say:


Assuming that Emp ID is a unique identifier, use that as a lookup for WS1.

Looks like the data is in columns so go to a row below

In row 8 put the formula =If(iserror(match (A3,Sheet2!$A$3:$Z$3,0)),"Not on Sheet 2","On Sheet 2)

Copy across.

The Match function (with the condition ,0) will tell you if an exact match in found is the range.  

If a Match is not found the function result will be the error #N/A thus when a match is not found the ISERROR function will be true and the formula result  "Not on Sheet 2".  If a match is found ISERROR function will be false and the formula result "On Sheet 2"
