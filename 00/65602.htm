andre da costa say:

There is no in place upgrade path from Windows 2000 Professional and you will need to purchase a full version Windows 7 license.

Also, you will need to make sure your system meets the minimum system requirements to run Windows 7:

System Requirements:

    *      1 gigahertz (GHz) or faster 32-bit (x86) or 64-bit (x64) processor
    *      1 gigabyte (GB) RAM (32-bit) or 2 GB RAM (64-bit)
    *      16 GB available hard disk space (32-bit) or 20 GB (64-bit)
    *      DirectX 9 graphics device with WDDM 1.0 or higher driver

Windows 7 Upgrade Advisor, find out how ready your system is:
http://www.microsoft.com/windows/windows-7/upgrade-advisor.aspx

Also, if you are using Windows 2000 Professional in a corporate environment, your logical path would be Windows 7 Professional (again, no in place upgrade path), but simply because they share  the same common core architecture (Encryption, Domain Join, Remote Desktop etc).

If the system is in shape to run Windows 7, the only option to get from Windows 2000 Professional to Windows 7 Professional while preserving your personal files and settings is to do what is called an a marathon upgrade. This means, you will do a series of upgrades, this can be a costly option if you don't have the licenses to do it. Meaning, you will need to upgrade from Windows 2000 Professional to Windows XP Professional to Windows Vista Business then to Windows 7 Professional. You would need to have legitimate licenses too for at least Windows XP Professional with exception if the Windows 2000 Professional is provided under Volume Licensing and you are upgrading to a Volume License distribution of Windows 7 Professional.

Create or change a regular playlist in Windows Media Player

a. Exit Windows Media Player.

b. For Windows 7: Click Start, click Run,type %LOCALAPPDATA%\Microsoft\Media Player,and then click OK.

c. Select all the files in the folder, and then click Delete on the File menu.

Note: You do not have to delete the folders that are in this folder.

d. Restart Windows Media Player.

Note: Windows Media Player automatically rebuilds the database.

a. Exit Windows Media Player.

b. If you are running Windows 7, click Start, click Run,type %LOCALAPPDATA%\Microsoft,and then click OK.

c. Select the Media Player folder, and then clickDelete on the File menu.

d. Restart Windows Media Player.
