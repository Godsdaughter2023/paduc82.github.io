macropod say:

To control number & currency formatting in Word, add a numeric picture switch to the mergefield. To do this:
• select the field;
• press Shift-F9 to reveal the field coding. It should look something like {MERGEFIELD MyData};
• edit the field so that you get {MERGEFIELD MyData \# $.0,00} (or whatever other numeric format you prefer - see below);
• position the cursor anywhere in this field and press F9 to update it.

 

Note: The '\# $.0,.00' in the field is referred to as a numeric picture switch. Other possibilities include:
• \# 0 for rounded whole numbers
• \# .0 for rounded whole numbers with a thousands separator
• \# .0,00 for numbers accurate to two decimal places, with a thousands separator
• \# $.0 for rounded whole dollars with a thousands separator

 

To control percentage formatting in Word, add a formula and numeric picture switch to the mergefield, as follows:
• select your mergefield, which will look something like «Percent»;
• press Ctrl-F9 to wrap another field around it, thus { «Percent» };
• edit the field so that you get {=«Percent»*100 \# 0,00%};
• position the cursor anywhere in this field and press F9 to update it.

 

Your mailmerge will now be ready to run. There is no need to modify the source data.

Auto hide/unhide rows in Excel - Microsoft Community

jeremygoldberg say:

Auto hide/unhide rows in Excel

Hi All!

I'm having difficulty revising existing macros found online my purposes, so I'm hoping to get some help. I need to auto hide/unhide one row based on the value of cell H32.

Here's what I need:

If cell H32 = cells X26 or X27, then do not hide (or unhide) row 34.

If cell H32 = cells X28 or X29 or X30, then hide row 34.

Thank you for the help!

- Jeremy

myall_blues say:

The worksheet_change event, for a specific tab, works by monitoring if the contents of the sheet change, and if it does the VBA code is triggered to do something.

So in your case I thought that you might want to trigger the row hide/unhide when the value in H32 changes, instead of having to run the macro manually.

The code would go something like this:

Private Sub Worksheet_Change(ByVal Target As Range)

       Select Case Range("H32").Value

        Case Range("X26").Value, Range("X27").Value

          Rows("34").EntireRow.Hidden = False

        Case Range("X28").Value, Range("X29").Value, Range("X30").Value

          Rows("34").EntireRow.Hidden = True

        Case Else

      End Select

    End If

End Sub

To make this work:

Open the VBA editor (ALT+F11)

On the left hand side, in the VBAProject window, right click on the relevant sheet ("Template" in your case) and select View Code

Paste the code above into the resultant window.

Now when you change the value in H32 on the template sheet it should hide or unhide row 34 as per the rules.
