doug robbins - mvp office apps & services (word) say:

Unless you used the Replace facility to replace ^b with nothing, the first thing that I would do is run a macro containing the following code:

MsgBox ActiveDocument.Sections.count

to see how many Sections there are in the document.  If the response is more than 2, then, you could either use the replace facility to remove them all this time and then put back the one that you want between the Table of Contents and the first page, or, run a macro containing the following code:

Dim i As Long
With ActiveDocument
    For i = 3 To .Sections.count
        .Sections(i).Headers(wdHeaderFooterPrimary).PageNumbers.RestartNumberingAtSection = False
    Next i
End With

to set the page numbering in each Section to continue from that in the previous Section.

john korchok say:

In general, people insert way too many unnecessary section breaks in documents and it causes these kind of problems. You may not be able to see the break, but it's there. It will help to find it if you right-click on the status bar at the bottom of the Word window. This displays the Customize Status Bar list that shows all the information you can display here. Choose the Section option. 

Now that you can see the section number, click at the bottom of Page 3 and use the right arrow key of your keyboard to step through one character at a time. At some point, the section number will increase, then you know you've just passed over the section break. Press the left arrow key once until the section number goes down by one. Hold the shift key and press the right arrow key once to select the section break. Press the delete key to delete it. Be prepared to undo if the appearance of the document changes drastically.

aaronitfc say:

A 404 error is an

In Internet Explorer, the message The webpage cannot be found usually indicates an HTTP 404 Internal Server Error but a 400 Bad Request error is another possibility. You can check to see which error IE is referring to by checking for either 404 or 400 in the title bar.

How To Fix the 404 Not Found Error

Retry the web page by pressing F5, clicking the refresh/reload button, or trying the URL from the address bar again.

The 404 Not Found error might appear for several reasons even though no real issue exists, so sometimes a simple refresh will often load the page you were looking for.

Check for errors in the URL. Often times the 404 Not Found error appears because the URL was typed wrong or the link that was clicked on points to the wrong URL.

Move up one directory level at a time in the URL until you find something.

hannelielabuschagné say:

Word 2016 - Page numbers

I recently upgraded to Windows 10, and my Office suite was automatically updated, I assume to Office 2016.

Since then I am unable to insert page numbers.  When I choose "Insert Page number" "Top of page" (or any other place for that matter) it only gives me the option:  "More Page Number from Office.com" But I can't click on that, as it is greyed out.  

I am working with legal documents, and I need to put "page x of y" on every single page that I work on.  

suzanne s. barnhill say:

Regardless of version, this problem usually indicates that the Building Blocks file is damaged. In Word 2010 and 2013, it is called Built-in Building Blocks.dotx, and if you delete the copy in your profile (in my case the path is C:\Users\Suzanne\AppData\Roaming\Microsoft\Document Building Blocks\1033), Word will build a new one (make sure you delete only the one in your profile, not in Program Files).

But you don't need a fancy building block to insert Page X of Y. You can easily create this yourself and even save it as a building block (AutoText entry).

  1. Where you want the number to appear, type "Page X of Y" (without the quotation marks.
  2. Select the X and press Alt+Shift+P. This will insert a { PAGE } field, which generates the page number.
  3. Select the Y and press Ctrl+F9. This will create a field that will look like this: { Y }.
  4. Select the Y and type "NUMPAGES" (without the quotation marks).
  5. With the insertion point in this field, press F9 to update the field, which will now show the number of pages.
  6. Select the entire expression and press Alt+F3. This will open the Create New Building Block dialog. By default, the name will be whatever number is displayed in your text, but you can change it to Page X of Y or whatever else you choose. Leave all the other settings as is and click OK.
  7. If you have not changed the name from Page [something] of [something], then you will get an AutoComplete tip every time you type "page" in the header or footer. Otherwise, you'll get the tip when you type the first four letters of the name you assigned. To accept the tip, press Enter or Tab.
