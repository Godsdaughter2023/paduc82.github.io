How to fix Wireless connectivity problem since update (please sticky)

Hey everyone. Ive been searching the forums a lot as i had a problem many people had or are currently having. Which is since the update around 24-28th of may time when the update launched which everyone connected to live got in that period of time. The update was suppose to support a new disk format through a software update. I think that microsoft may have made this update for better security seing as PSN was hacked last month. The reason why i think this is because the soultion to this is that you must change your password securtiy on your router to a later version which im guessing is more secure, hense why microsoft make you change this. I found the solution to my problem from searching the forums and finding solutions people posted many thanks for the help.

 

I dont know why Microsoft didnt tell us this, it has led a lot of people to not being able to not connect like me. The problem is most people and maybe some of you reading this may have no clue what to do with router settings which is a big problem. So Microsoft have made it complicated for us.

 

However the problem i had and solution to it is actually pretty simple if you know what you are doing.

What this post is for is if you had the same problem as me which i think that is the case i want to try and help so that everyone knows what to do simply. Hopefully Microsoft can sticky this.

My router password protection key was old, it was the WPA, I had to change to the later version to the WPA2-AES.

 

How to fix

What you must do is change your router settings. To do this you must go on the computer that is wired connected to your modem/router. Then you enter your IP address in the address bar on your internet browser. To find out your IP address go to Start>Accessories<Command Prompt. Then enter ipconfig. This will tell your IP address.

Then you go to your WI-FI password protection settings. Then change the WPA to WPA2-AES.

Now you will be wondering how do i get to my WI-FI password protection settings. Well this varies from different routers. I will try to keep this OP updated and post links how to from your chosen router.

andre da costa say:
This error message is displayed when attempting to install Windows on a computer and the EFI Boot Sources setting is enabled in the BIOS:
Windows cannot be installed to this disk. The Selected disk has an MBR Partition table. On EFI systems, Windows can only be installed to GPT.


The resolution to this issue depends on the the hard disk volume size:

Follow these steps if the hard disk volume size is less than 2.19 TB:
Temporarily disable the EFI Boot Sources setting in the BIOS:
Restart the computer, and then press F10 to enter the BIOS.
Navigate to Storage > Boot Order, and then disable the EFI Boot Sources.
Select File > Save Changes > Exit.
Install the Windows operating system.
Enable the EFI Boot Sources setting in the BIOS:
Restart the computer, and then press F10 to enter the BIOS.
Navigate to Storage > Boot Order, and then enable the EFI Boot Sources.
Select File > Save Changes > Exit.
