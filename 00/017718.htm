To timestamp a VBA project, you need to create three registry entries, one of which points to the timestamping server of the company from which you obtained your digital certificate. Here are the steps as described by Thawte: Timestamp VBA Projects

Some keyboard layouts allow two methods for producing a character with a diacritical mark : 1) first type the dead key combination assigned to the mark, then type the base character; or 2) first type the base character, then type the key combination assigned to the mark. jay freedman say:
According to http://office.microsoft.com/en-us/starter-help/word-features-that-are-not-fully-supported-in-word-starter-HA010374498.aspx,

"If you open a document that was changed while the Track Changes feature was turned on, you will see editing marks in the document. You cannot accept or reject the changes, and you cannot turn change tracking on or off in Word Starter."

You'll have to ask the reviewer to send you a copy of the paper without any tracked changes, which can be produced by accepting or rejecting each change (or all of them at once). This is not something you can do in the Starter edition.
