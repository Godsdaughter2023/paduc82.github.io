tryx3 say:

1  A useful step to take even before you have fixed these problems is to create two spare password-protected Admin-level accounts. 

  • User accounts & their security are taken much more seriously in Windows 7 so the tricks we learnt in XP cannot help us any more. 
  • Any account can suffer from user profile corruption and if it is your main user account and you have no spare Admin account to access the system with then you have a serious problem [that we are not even permitted to advise you about in this forum other than to suggest complete reinstallation of Windows 7]. 
  • If you have these two spare accounts then, because they are hardly ever used, they are less likely to suffer corruption as that happens in use. 
  • So even if you get serious problems in your normal account you can then log in as one of these and fix the problem.
  • You should write down the passwords and keep the piece of paper safe.  You could make "Password reset disks" instead but they are no more effective than a piece of paper and they would also need to be kept somewhere safe.

2  Did you run the Windows 7 compatibility tester before upgrading?  You can run it now and its results will still help you.  See Windows 7, 8.1, 10 compatibility testing for Windows XP, Vista - WiKi

3  Then see Where to get hardware drivers to address your current problems with your multi-media controller etc.

4  Once you get these things sorted out then you will probably want to ensure you don't get forced into a Windows 10 upgrade before you are ready - see Disable Windows 10 upgrade and ignore anybody who starts telling you to disable specific Windows updates or to disable Windows update itself as that is all nonsense.

5  Like many of us when we first came from XP, you might not be used to making "system images"  This is worth doing as soon as you have a system worth preserving - so you will probably want to put this off until you have fixed your hardware drivers problem. 

  • You can make a system image in less than half an hour. 
  • You can restore a system image in less than half an hour and thereby overwrite any software-based problems, including driver problems, that have appeared on your computer. 
  • Search for create a system image in Windows Help to get an introduction to making & using them,
  • Then look in Control panel, [View by - Small icons, if necessary], Backup & restore for the link on the left-hand side to Create a system image to actually run the utility. 
  • Note that you will not need to make the System repair disk that will find references to - your Windows 7 installation disk can do everything that a system repair disk can [and more]. 
  • The most convenient & reliable target to save system images on is an external hard drive.  A system image does not take over the entire target external hard drive so it can also be used for other purposes such as storage for your routine file backups.
  • Rewriteable DVDs are not as reliable and you would need several each time you made a new image.
  • Third party utilities for making system images can be better than the built-in Windows utility  I use Acronis TI which has a very good reputation.  Macrium Reflect free edition also has a good reputation.  But you can start off with the built-in utility and then worry about alternatives later on.

And this behavior is not a "Microsoft" or "Frontier" thing. 

Every game publisher needs to sell games, and therefore the publishers want to limit the "sharing" of their games. 

Microsoft knows that it's unreasonable for families to buy a game for every family member, so they introduced the "Home Console" concept, where every account can play the games of the one account that set the console as their "Home Console".

To prevent excessive sharing (and financial losses for all the game publishers), there are limits on how often you can switch the Home Console, and how many Home Consoles an account can have (one). 

This is, so you don't buy a game once, and share it with dozens of people, that won't buy the game from the publisher anymore.

dax1792 say:

To summarise:

USB 2.0 clock speed is 480 megabits per second. That's 60 megabytes per second. Given the protocol overhead and the fact that USB 2.0 is half-duplex, the maximum data rate will be 30-40 megabytes per second. The 480 megabits per second limit applies to the USB controller and is shared between the ports attached to it. The number of USB controllers per card or motherboard will vary.

salvation at the cross say:
Changing drive letters in Windows 7 usually takes less than 5 minutes
Here's How:

  1. Click on Start and then Control Panel.

    Type computer management in the search box after clicking Start. Choose Computer Management under Programs from the list of results and then skip to Step 4.

  2. Click on the System and Security link.

    Note: If you're viewing the Large icons or Small icons view of Control Panel, you won't see this link. Simply click on theAdministrative Tools icon and proceed to Step 4.

  3. Click on the Administrative Tools link.

  4. In the Administrative Tools window, double-click on Computer Management.

  5. Once Computer Management is open, locate the Disk Management option on the left andleft-click on it.

    If the Disk Management option is not visible, look under the Storage option by clicking the small arrow next to the wordStorage.

  6. On the right-hand side of the screen, locate and right-click on the drive that you wish to change the drive letter of and chooseChange Drive Letter and Paths....

  7. In the Change Drive Letter and Paths window, click on the Change... button.

    The Change Drive Letter or Path dialog box will appear.

  8. Choose the drive letter you wish to assign to this partition by clicking the letter from the drop-down box on the right.

    Click OK.

  9. The Disk Management window that appears advises you that "Some programs that rely on drive letters might not run correctly. Do you want to continue?".

    Note: If you have software installed to this drive, the software may stop functioning after a drive letter change. Nearly all software defaults to the primary drive though, so this is rarely a concern.

  10. If you're sure you want to make this change, click Yes.

    Depending on the speed of your PC, the change should be nearly instant.

  11. You can now close the Computer Management, Administrative Tools, andControl Panel windows.
