This is most likely a proofing language problem. The "proofing language" is a key to several Word features including spelling and grammar and AutoCorrect. • Pick "Select Proofing Language" from the short menu.
• Look at the bottom of the dialog that appears.
• If the checkbox that says "Do not check spelling or grammar" is not empty, click on it until it is clear.
• Then OK.

jay freedman say:
Your first If statement says that the inner part -- hiding the image -- will execute only if the result of the message box is vbYes. That's backwards; it should be If ... = vbNo Then ...

As a suggestion, I'd switch the order of the two If statements so the message box appears only if the .Shapes.Count is greater than zero. If there is no image in the header, there shouldn't be any message.

jay freedman say:
