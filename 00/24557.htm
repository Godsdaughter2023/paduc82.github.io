jay freedman say:
You don't have to go to page 4 in order to print page 4. All you need in the code of the command button is this one line:

    ActiveDocument.PrintOut Range:=wdPrintRangeOfPages, Pages:="4"

Since you aren't going to page 4, you don't need any button to go back to page 1 -- which you never left.

If you're saving the document in Word 2007 document format (as opposed to Word 97-2003 format), you must save it as a Word Macro-Enabled Document (*.docm). If you ask to save it as a regular document (*.docx) you'll be warned that the macro will be discarded.

To allow the macro to work, you must also store the file in a folder that is in the Trusted Locations list (at Office button > Word Options > Trust Center > Trust Center Settings > Trusted Locations), or else put the file in another folder and then add that folder to the Trusted Locations list.

jay freedman say:
OK, I understand wanting to look at the page, although I still don't see any point in going back to page 1 before picking a different page to look at. If you want to go back to the start of the document, that line of code is

Selection.HomeKey Unit:=wdStory, Extend:=False

This macro is probably more elaborate than what you had in mind -- it shows you page 4 and asks whether to print it; if you click No, you can enter another page number, and another, etc. until you finally click Yes. Or, if you cancel the dialog that asks for a page number, it stops without printing anything.

Sub PrintPage4Maybe()
    Dim pageToSee As Long
    pageToSee = 4
    Selection.GoTo What:=wdGoToPage, Which:=wdGoToAbsolute, Count:=pageToSee
    If MsgBox("Print this?", vbYesNo, "Print page") = vbYes Then
        ActiveDocument.PrintOut Range:=wdPrintCurrentPage, Background:=False
    Else
        pageToSee = Val(InputBox("Enter number of page to look at:"))
        While pageToSee > 0
            Selection.GoTo What:=wdGoToPage, Which:=wdGoToAbsolute, Count:=pageToSee
            If MsgBox("Print this?", vbYesNo, "Print page") = vbYes Then
                ActiveDocument.PrintOut Range:=wdPrintCurrentPage, Background:=False
            Else
                pageToSee = Val(InputBox("Enter number of page to look at:"))
            End If
        Wend
    End If
    Selection.HomeKey Unit:=wdStory, Extend:=False
End Sub

One other thing: Word 2007 (and all other versions) has a Print Preview, and you can add a button for it to the Quick Access Toolbar. In the preview, you can find the page you want and click on it; then click the Print button, and select Current Page in the Print dialog. No macro is necessary, and it's probably quicker than jumping page by page with the macro above.

Excel formula to keep track of Shirt Sizes - Microsoft Community

amybailey03 say:

Excel formula to keep track of Shirt Sizes

I have a spreadsheet to keep track of shirt sizes I need for our vacation bible school. I would like it to keep a running total of how many shirts I need in each size. The sheet is set up like this:

Name       Shirt

Susie         M

George     XL

Sally           S

Bobby       YM

Make sense?

mike h.. say:

Hi,

Try this

=COUNTIF(B2:B20,"S")

With the obvious change for M etc.

amybailey03 say:

Ok, after playing around I figured it out. I had 3 columns going (adults, youth leaders and youth attendees) so I was stuggling, but here is the formula I used:

=COUNTIF($C7:$C16:$G7:$G27:$K7:$K34,B36)

B36 was the shirt size
