john w. vinson say:

If you want it to be the last 24 hours as of the time you run the query - e.g. as I type, all records from 9/28/2014 2:10:42pm to 9/29/2014 2:10:42pm - you can use a criterion on the field of

>= DateAdd("h", -24, Now())

The Date() function returns today's date, as of midnight last night; the Now() function returns the current date and time, accurate to the second. They're both useful but they're different.

Does the field in fact contain a time component? If you edit it in the table or on a form does it show as 9/29/2014, or as 9/29/2014 02:13:33?

john w. vinson say:

Sure. The DateAdd() function is very flexible. Type Ctrl-G to get into the VBA editor (so you're connected to the right Help file) and search for help on DateAdd (and DateDiff and DateSerial among other functions). They can all be used in queries to manage date and time values.

srinivasu guntimadugu say:

Hi,

 

Thank you for contacting Microsoft community and We will be glad to assist you with your concern. It looks like there is a corruption with the Office application or the entire Office suite.

 

Let us follow the steps to determine the cause of the issue and fix it. Before we proceed, I have a couple of questions.

 

1. Are you able to open other Office applications apart from Word 2013?
2. Where there any updates installed recently or any 3 party software installed?
3. Have you installed the Click-to-Run based edition of MSI (Setup File) edition of Office 2013 on the compute?

 

If there are any updates installed recently or any software installed, it might conflict with the Office applications.

 

Method 1:
Let us open Word in safe mode.
1. Press the Windows + R key on the keyboard
2. Type winword /safe (leave a space after winword) -> Click OK.

 

If you are able to open word in safe mode, then let us disable the Add-ins and verify results.
1. Click on File -> Options -> Add-ins.
2. Under Manage select -> COM Add-ins.
3. Click on GO options beside Manage, uncheck all the add-ins and click Ok.

 

If you are unable to open and access other Office applications, then let us repair the Office 2013.
Follow the steps to repair Office 2013.
1. Press the Windows + R key on the keyboard.
2. In the Run windows type appwiz -> Click Ok.
3. Select Microsoft Office 2013 from the Programs & Features list.
4. Click on Change and repair Office 2013 and restart the computer.

 

Restart the computer and open the Office applications to check if the issue is fixed.

 

I hope the steps provided in the above methods fix the issue. If the issue persists, reply and we will be happy to help you further.

 

Thank You
