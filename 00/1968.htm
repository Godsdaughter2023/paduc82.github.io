Ken --

You will need to modify each Gantt-based view individually to accomplish this, which is what you would have done in versions prior to Microsoft Project 2016.  Open any project and apply the Gantt Chart view.  Double-click anywhere in the white part of the Gantt Chart pane to display the Bar Styles dialog.  In the dialog, select the Milestone item in the list at the top, and then set the desired formatting for milestones in the Bars section at the bottom.  Click the OK button and then do this for every Gantt-based view, such as the Tracking Gantt view, for example.

Bookmark? - Microsoft Community

boisebob1 say:

Bookmark?

When I save a large document and later reopen it, Word (Office 2013) offers to take me to the last place I was working in the document.  Unfortunately, this only happens in my office copy of Word.  Any idea how to turn this (really handy) feature on so I can fix my home copy?
suzanne s. barnhill say:
As Stefan said, the feature in Word 2013 and 2016 is called Resume Reading, but Shift+F5, which works in all versions of Word except Word 2007, executes the command GoBack. You can use it to return to the last edit point (actually the last three edit points) while you're working on a document. When you close the document, Word sets a bookmark at the last edit point, to which you can return with Shift+F5 when you reopen the document. The Resume Reading feature just automates this (and presumably enhances it, since it returns to the last insertion point position rather than the last actual edit).
