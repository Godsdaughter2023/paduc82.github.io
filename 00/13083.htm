rich michaels - mvp (greatcirclelearning.com) say:

Microsoft decided to remove access to the Title property field because they felt that it was confusing to those users who where also inserting Alt Text for Accessibility purpose. I disagreed with this change because the Title property field is used for more than just Accessibility purposes, particularly in document automation functions, and I expressed that to the development team ... but they refused to listen, at least to just me.

Data that you have included in the Title field is not lost, if you open the new Alt Text task pane you will see that existing Title data appears as the first line in the Alt Text description. However, don't be misled to believe that the first line you type into the Description box will be populated to the Title field ... that won't happen.

If you need to use the Title field for purposes other than Accessibility you can can add a Picture Content Control to you image and then set the Title from the Properties button for the Content Control. These buttons are on the Developer tab. You can also write your own custom VBA macro and access the Title property field. Finally, you can do a web search on AuthorTec Accessibility and investigate that free add-in which continues to provide editing access to the Title property field, regardless if you are using it for Accessibility purposes or not.

If you wish to provide direct feedback to Microsoft on this feature being removed from the User Interface, please use the Feedback function, which is on the File menu of Word.

tasos_k say:

Okay,

assuming that you want  4 pictures
in cells A1, B2, A3, B4

also, you can change cells' dimensions (90.75 and 25.75), as needed

 

try this code...

 

Sub InsertPictures_02()
'17Nov2013
Dim v As Variant, vCell As Variant
vCell = Array("A1", "B2", "A3", "B4") '<<< target cells, change
Dim pCell As Range
Dim x As Long
Dim p As Picture
v = Application.GetOpenFilename(FileFilter:="ALL files (*.*), *.*", _
Title:="open a folder, press Ctrl and select the pictures", MultiSelect:=True)
If Not IsArray(v) Then
MsgBox "no pictures"
Exit Sub
End If

Application.ScreenUpdating = False
On Error Resume Next
For x = LBound(v) To UBound(v)
Set p = ActiveSheet.Pictures.Insert(v(x))
Set pCell = ActiveSheet.Range(vCell(x - 1))
pCell.RowHeight = 90.75
pCell.ColumnWidth = 25.75
With p
.Height = pCell.Height
.Width = pCell.Width
.Top = pCell.Top
.Left = pCell.Left
End With
Next x
Application.ScreenUpdating = True
End Sub

 

 

note

1) don't worry, if you select more or less than four pictures,
no problem.

2) if you run the code twice,

then the new picture covers (not overwrites) the previous one.

Sub DeletePic_Selection()
