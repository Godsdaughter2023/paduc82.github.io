andreas killer say:
You want to sort more then 3 columns?

Sort the last columns first, then the first columns. Below is an example, Execute the sub in a new sheet.

Andreas.


Sub Example_SortMoreColumnns()
  Dim Data
  Const Columns = 5
  Dim Rows As Long, i As Long, j As Long
  Dim Divider As Long, Bit As Integer
  Dim Order As XlSortOrder
 
  'Create a field in ascending order
  Rows = 2 ^ Columns
  ReDim Data(0 To Rows - 1, 0 To Columns - 1)
  For i = 0 To UBound(Data)
    For j = 0 To UBound(Data, 2)
      '16,8,4,2,1
      Divider = 2 ^ (UBound(Data, 2) - j)
      '0,1
      Bit = i \ Divider Mod 2
      'A,B,C,D,E,F,G,H,I,J
