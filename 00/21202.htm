hansv mvp say:

If you see the blinking insertion bar, changes to tabs are applied to the current paragraph only.

If you select some text, changes to tabs are applied to all paragraphs in the selection.

If you modify a style, for instance the default Normal style, and modify the tabs there, changes are applied to all paragraphs formatted with that style, except where you have overridden the style by changing the tabs directly.

If you want to remove directly applied paragraph formatting, select some text and press Ctrl+Q. But keep in mind that this not only resets the tabs but also left and right indent, line spacing etc.

huni shen msft say:

Hi WWTweaver,

 

From my perspective, the issue you encountered may be that the PST file is corrupted. So I suggest you can try to repair your PST files first.

 

In addition, Outlook 2016 store the AutoComplete list as a hidden message in your primary message store,  if the issue persists after doing what I mentioned above, I suggest you can try to do the following steps:

Step1. Export the AutoComplete mailbox message

Step2. Clear the whole AutoComplete List. Restart Outlook and type in a few email addresses and then exit Outlook and open it again and see if the autocomplete works.

Step3. Import the Autocomplete mailbox message

 

For more detailed information about these operation, you can refer to the following article:
