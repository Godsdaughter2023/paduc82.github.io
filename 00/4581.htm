Yep, you can do this:

 

Dim objAcc As Object

 

Set objAcc = CreateObject("Access.Application")

 

With objAcc

   .OpenCurrentDatabase("G:\pathto\DB2.mdb")

   .DoCmd.OpenForm "Form2", acNormal

   .UserControl = True

End With

 

 

And in case you can't see the periods before them, here it is without the WITH statement:

 

Dim objAcc As Object

 

Set objAcc = CreateObject("Access.Application")

 

   objAcc.OpenCurrentDatabase("G:\pathto\DB2.mdb")

   objAcc.DoCmd.OpenForm "Form2", acNormal

   objAcc.UserControl = True

number occurrence formula - Microsoft Community

lea1993 say:

number occurrence formula

=SUM(LEN(A2:A7)-LEN(SUBSTITUTE(A2:A7,"apple","")))/LEN("apple")

I need a formula similar to this but that will count the occurrence of the number "16"

My current issue is that excel is not counting the number if it appears in more than once within a single cell....is there a way for me to count each occurrence of that specific number to include more that one entry within the cell?

vijay a. verma say:

Use following type of formula to count in a range -

=SUMPRODUCT((LEN(A1:A10)-LEN(SUBSTITUTE(A1:A10,16,"")))/LEN(16))

amit tandon say:

Your formula is correct, except that it needs to be entered as an array formula  (CTRL+SHIFT+ENTER):

=SUM((LEN(A2:A7)-LEN(SUBSTITUTE(A2:A7,16,"")))/LEN(16))

Vijay's formula uses SUMPRODUCT so it can be entered as a non-array formula.

Amit Tandon
