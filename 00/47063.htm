methylpropyl say:

When Task Manager shows the "System Idle Process" at 98/99% CPU, there is no problem whatsoever.  This just means that the active processes on your computer are only using 1-2% CPU.

 

The common issue is with "svchost.exe" running at 97-100% CPU when preparing the Windows Updates list for "wuauctl.exe" .  It this happens to a normally reliable computer the best thing you can do is to just let the process finish even if it takes hours.

Is is nothing unusual for a single instance of "svchost.exe" to run at 100% CPU for short periods of time.

It seems to me that this statement made by you earlier could well be incorrect. I suspect that WinDirStat  is not reporting the size of the System Volume Information folder and that if you added  the reported space used to the reported free disk space you would discover that it is notably less than the size of the SSD drive. The difference largely being the space occupied by the System Volume Information folder.

To ascertain the size of the System Volume Information folder in Windows 7 select Start, Control Panel, System, System Protection, click on Configure under Protection Settings. What is the Current Usage figure? This figure often changes daily and can fluctuate up and down by significant amounts.

You will be able to temporarily increase the amount of free disk space by removing redundant system restore points but the longer term solution lies in moving folders from your SSD drive to your second drive. If your second drive is an external drives this limits what you can move. Thus you can move a pagefile to a send internal drive but not to an external drive.

Select Start, All Programs, Accessories, System Tools, Disk CleanUp to empty your Recycle Bin, remove Temporary Internet Files and other selected options. I also recommend you click on the More Options tab, System Restore and remove all but the latest System Restore point.

A better programme than Disk CleanUp is cCleaner. Removing System Restore points can be important if you have limited free disk space. The ability to remove them is one of the Tools options. You can selectively remove those created at less important times.

cCleaner also comes with a Registry tool. Do not use this tool as like any registry cleaner it can lead to the creation of very difficult to resolve system errors.
