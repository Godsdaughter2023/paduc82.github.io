How do I create a printable document with windows 10? - Microsoft Community

denisehering say:

How do I create a printable document with windows 10?

Hi, I want to create a document using Windows 10.  Id like to revise it from time to time, send it by email to others and print it.  Please Hellp!

robertjwin say:

If you have Word for example (or whatever program you use to create and edit your documents), in Win10 you can Print to PDF and the file will be saved as a standard PDF file. The advantage is that while a Word DOC/DOCX might look slightly different depending on the version of Word installed on other people's PCs, PDFs are meant to look exactly the same everywhere. Also, people who don't have Word or your large document creation software installed can simply install any small PDF viewer instead (by default the Edge browser in Win10 itself can open PDFs) and view/print the file you've sent them.

PDFs can also be edited directly, but unless you have a PDF creator/editor installed, the easiest solution is to simply create your document, print to PDF and send the PDF to other people. Any time you want to make changes, edit the master document on your system, again print to PDF and send the new PDF. Simple.
