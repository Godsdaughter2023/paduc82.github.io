Hi Gopi,

If an employee leaves company and we delete his/her account today, the OneDrive for Business site ownership is transferred to the manager and a secondary My Site owner. The manager and the secondary owner can continue to access the personal site for 3650 days via the URL if the OneDrive for Business site.

For this to occur, the user profile Manager property and Access Delegation need to be set for the deleted user before deleting it. To do that, as a global admin or SharePoint admin, please follow the steps below:

1. Go to the SharePoint admin center.
2. Click User profiles > People > Manage User Profiles > Find the user account to deleted > Edit my profile > Manager > type the manager account into the box > Save and Close.

3. Go to SharePoint admin center > User profiles > My Site Settings > Setup My Sites > My Site Cleanup > tick Enable access delegation> Secondary Owner as below:

When a user's profile has been deleted, that user's My Site will be flagged for deletion after 3650 days. To prevent data loss, access to the former user's My Site can be granted to the user's manager or a secondary My Site owner. This gives the manager or the secondary owner an opportunity to retrieve content from the My Site before it is deleted. Select whether or not ownership of the Site should be transferred to a manager or secondary owner before the site is deleted. Set a secondary owner to receive access in situations in which a user's manager cannot be determined. Hi ShirleyWieneke,

 

Thank you for posting your query in Microsoft Community Forums. I'm glad to assist you with your issue.

 

We would like to inform you that it is impossible to absolutely stop any junk email from coming to your mailbox. But you can create an email rule that will automatically delete email based on the specific rule you created.

 

Here are the steps:

1. Sign in here
2. In the upper-right corner of the page, click the Cog icon, and then click More mail settings.
3. Under Customizing Outlook, click Rules for sorting new messages.
4. Click New.
5. Under Step 1, select the email attribute from the drop-down list and type the email address that you want to apply the rule to.
6. Under Step 2, determine where you want the messages to be moved or what you want to do to that email (e.g. if it would be auto Deleted):

1. To move the messages to an existing folder, choose Move to Inbox, and then click the folder name

2. To move the messages to a new folder, choose Move to a new folder and type the name for the folder you want to create.

3. To delete the messages, choose Delete these messages.

4. To forward messages, select forward messages and type the email address where you wish to forward the said emails.

5. Click Save.

Note: In an instance where the spammer just changes the domain (sending email address), it would be better if you would create an email rule that will specify the subject instead of the sender's email address. (e.g. If the email contains the word "Viagra", delete the email).
