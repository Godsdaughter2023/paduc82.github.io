vijay a. verma say:
Unlike majority of the default Windows apps, Cortana can not be disabled through Powershell. Microsoft wants people to use Cortana.

If you want to completely turn off Cortana, then you will need to resort to a registry fix (Note - It is recommended that you create a backup of your registry before modifying so that if you commit some mistake, you will be able to restore - https://support.microsoft.com/en-us/help/322756...)

Windows Key+R > Type regedit and Enter
Go to the key - HKEY_LOCAL_MACHINE > SOFTWARE > Policies > Microsoft > Windows
See if you have a directory named - Windows Search
If not, then right click on Windows folder > New Key > Give the name Windows Search
Click Windows Search > In the right-hand side pane, right-click in the empty area and choose New > DWORD (32-bit) Value. Type AllowCortana > Enter
Double-click AllowCortana, and type in 0 under Value Data.
Reboot your computer

crosseye jack say:

The issue is that the Windows Store currently tries to use all available bandwidth when downloading be it new content or updates. But when you are doing other heavy BW usage things on your machine at the same time those grind to a halt.

For an reproducible example of not being able to control the download speed of windows store is to open a long form YouTube video and then download something large from the Windows Store. 

Windows store will try and use all your available bandwidth to download the content not leaving any BW available to stream the YouTube Video without it buffering producing a less then ideal video Playback experience. You will quickly run out of buffer on YouTube before the item in the windows store completes its download. YouTube will then deem you are on a slow connection and reduce the quality of the video to try and prevent buffering.

Steam (and other download clients) get around this by giving you the option to limit how much Bandwidth they can consume meaning you can limit the client to say (random number plucked out of the air) 90% of your Download speed and the spare 10% is enough for your other day to day activities. Your content from Steam (and other clients) download speed takes a 10% hit but that hit doesn't give too much of a negative experience esp as the user may be using other things on their device while Windows Store content is downloading.

Here's how to change the screen resolution, color depth, and color space settings for an HDTV or monitor that's connected to an Xbox One console by an HDMI cable.

Screen resolution

  1. Press the Xbox button to open the guide.
  2. Select Settings.
  3. Select All Settings.
  4. Under Display & sound, select Video output.
  5. Select TV resolution.
  6. Select the appropriate screen resolution for your TV or monitor.

Note The Xbox One console supports screen resolutions of 720p and 1080p.

Color depth

  1. Press the Xbox button to open the guide.
  2. Select Settings.
  3. Select All Settings.
  4. Under Display & sound, select Video output.
  5. Select Color depth.
  6. Select the option that works best for your HDTV or monitor.

Color depth is how much color information per pixel is sent to the TV or monitor.

Note Color depth of 30 bits per pixel (bpp) or higher is known as "deep color." Not all HDTVs accept deep color information.

Color space

  1. Press the Xbox button to open the guide.
  2. Select Settings.
  3. Select All Settings.
  4. Under Display & sound, select Video output.
  5. Select Color space.
  6. Select the option appropriate for your TV or monitor:
    • Standard (recommended) – The optimal reference level. It should work for all displays.
    • PC RGB – Recommended only if you're using a PC monitor and colors from the "Standard" setting don't look good.
