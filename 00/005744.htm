rich michaels - mvp (greatcirclelearning.com) say:

Hi Alex,

When moving your personal Word templates and shared Word templates to a private or shared server location, such as OneDrive, SharePoint, Dropbox or any connected server location there are a few folder names and concepts that are very important to understand.

Not in any particular order:

  1. User Templates and My Templates are important folder names to Word.
  2. Personal templates are those for your use and you would be ill advised to “share” them with others. They are generally stored in a private location, which is only accessible by you.
  3. Shared templates are those that both you and others have rights to use and alter. They are stored in a location that is accessible to all who are authorized.
  4. The “User Templates” location is where the Normal.dotm template is stored. By design, the Normal template is considered a personal/private template.
  5. By default the “My Templates” location is a subfolder in the “User Templates” location. It contains personal templates that you have created, purchased, borrowed or stole. :-) By design, templates contained in the My Templates subfolder of the User Templates directory are considered private/personal templates.
  6. Not withstanding #1 above, the “designated” User Templates folder can be named something else but that location will only be recognized if the Word > Preferences > File Locations > User Templates location points to that unique location. Otherwise, Word will expect it to be in the default location with this path name: ~/Library/Application Support/Microsoft/Office/User Templates
  7. The Word Documents Gallery dialog that displays when the File > New from template… is clicked contains a fixed text button labeled “My Templates”. The link for that text button is derived from the designated User Templates location (default or custom) and the subfolder it expects to find must be labeled “My Templates”. If a folder with that exact name is not found within the designated User Templates folder, then the Word Documents Gallery display for My Templates will show empty.
  8. There is not a designated default location for Word Group Templates. If a custom location for Work Group Templates is set, the folder name containing those shared templates will be displayed on the Word Documents Gallery dialog.
  9. Since “My Templates” is a fixed label pointing to personal/private templates as described in #7 above, it is recommended that you not label your Work Group Templates folder “My Templates” as it will be confusing.

10.If your objective is to have all your personal/private templates, including the Normal template, shared across your various Mac computers then your Dropbox path should be Dropbox/User Templates/My Templates. This is a pragmatically simple way to do it. There are other methods and I will let others attempt to explain their alternative approaches.

To establish #10 above use Finder to copy your existing User Templates folder from its default location, listed in #6 above, and paste it into your local Dropbox folder. Open Word and use Word > Preferences > File Locations to point the User Templates setting to the User Templates folder in your Dropbox. Quit Word completely and then use Finder to remove the User Templates folder and all of its content from its default path in Application Support/Microsoft/Office. You will have to repoint the User Templates file location on each computer you will be using and remove the default folder that currently exists.

Regarding your last question about deleting the file location paths in the Preferences dialog. Unfortunately, on the Mac that does not appear easily done. I would expect the setting would be in a plist file but I can’t find it. Maybe someone else knows where it is.

Sorry for the long winded answer but I think there’s a lot of confusion about this and my intent is just to bring some clarity.
