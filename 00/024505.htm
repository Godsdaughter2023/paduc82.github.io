Do you want to import the Outlook.com calendar to the Office 365 account calendar?

If yes, you can refer to the following steps:
1. Set up the Outlook.com account into the Outlook client (e.g. Outlook 2013, Outlook 2016).
2. Export the Outlook.com account calendar into .CSV file by clicking File > Open & Export > Import/Export > Export to a file > Next > Comma Separated Values > Next. Then select Calendar, choose where you want to export the file, click Next > Finish.
3. Set up your Office 365 account into the Outlook client.
4. Import the CSV file into the Office 365 calendar by clicking File > Open & Export > Import/Export > Import from another program or file > Next > select Comma Separated Values > select File to import > Next > select the Office 365 account’s calendar as the destination folder > Next > Finish.
