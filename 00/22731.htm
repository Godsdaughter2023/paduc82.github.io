brian tillman [outlook mvp 2007-2019] say:

First, you never enter contacts using the Address Book interface.  The Address Book contains no data.  It is a view of your Contacts folders.  Enter contacts in the Contacts folder(s).

Second, your Address Book service is pointing to an invalid Contacts folder.  In Outlook, click Tools>E-mail Accounts>View or change existing directories or address books>Next.  Select the Outlook Address Book service and click Change.  Remove the Contacts folder you find there.  Finish the wizard.  Stop and restart Outlook.  Right-click your Contacts folder, choose Properties, select the Outlook Address Book tab, and check the box labeled "Show this folder as an e-mail Address Book".  Click OK.  Now the Address Book should show the correct folder.
