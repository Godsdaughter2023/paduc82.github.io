and with the following code you can DICTATE the width and height  :)


Private Sub Worksheet_SelectionChange(ByVal Target As Range)
Dim Wdth, Hgth, MW, MH

MW = InputBox("Please, enter the Chart Width")
MH = InputBox("Please, enter the Chart Height")

If MW > 10 And MW < 1000 Then
    Wdth = MW
  Else
    Wdth = 200
End If

If MH > 10 And MH < 1000 Then
    Hgth = MH
  Else
    Hgth = 100
End If

Set co = Sheets("Sheet1").ChartObjects.Add(50, 40, Wdth, Hgth)
co.Chart.ChartWizard Source:=Worksheets("Sheet1").Range("A2:" & "B10"), _
    Gallery:=xlColumn, Format:=6, PlotBy:=xlColumns, _
    CategoryLabels:=1, SeriesLabels:=0, HasLegend:=1
   
co.Activate

Sheet1.ChartObjects(co.Index).Top = Sheet1.Range("C2").Top
Sheet1.ChartObjects(co.Index).Left = Range("D2").Left

End Sub

tintombstone say:

Hi Melepoil

Try this revised code. Make sure that you have a macro enabled presentation (saved as .pptm) and you enable macros on open

Again drop the code into a standard module and run the ReplaceArabicNums() procedure

Sub ReplaceArabicNums()
Dim x As Integer
    For x = 0 To 9
        ReplaceNums2 CStr(x), ChrW$(&H660 + x)
    Next x
End Sub

Sub ReplaceNums2(sFind As String, sReplaceFor As String)
Dim Sld As Slide
Dim Shp As Shape
Dim frameText As TextRange
Dim tempText As String
Dim ch As Integer, r As Integer, c As Integer
    For Each Sld In ActivePresentation.Slides
        For Each Shp In Sld.Shapes
            If Shp.HasTextFrame And Shp.TextFrame.HasText Then
                    Set frameText = Shp.TextFrame.TextRange
                    tempText = frameText.Text
                        For ch = 1 To Len(tempText)
                            If Mid(tempText, ch, 1) = sFind Then Mid(tempText, ch, 1) = sReplaceFor
                        Next ch
                    frameText.Text = tempText
            End If
            If Shp.Type = msoTable Then
                For r = 1 To Shp.Table.Rows.Count
                    For c = 1 To Shp.Table.Columns.Count
                        With Shp.Table.Cell(r, c).Shape.TextFrame
                            Set frameText = .TextRange
                            tempText = frameText.Text
                            For ch = 1 To Len(tempText)
                                If Mid(tempText, ch, 1) = sFind Then Mid(tempText, ch, 1) = sReplaceFor
                            Next ch
                            frameText.Text = tempText
                        End With
                    Next c
                Next r
            End If
        Next Shp
    Next Sld
End Sub

Hi Vincent,

For your concerns:

1. Can and how can I use Office 365 in the Cloud to achieve this;

As you mentioned you are using POP3 account, items will be downloaded to Outlook client directly, and then it's not feasible to sync your items between different devices since they are not in server any more. If you want to sync your contacts and calendars in different devices, you should configure your account with Exchange type in client. And Office 365 for Business (cloud) can achieve this, you just need to have an Office 365 for Business account.

2. Would I need Exchange and therefore Office 365 Business Essentials or Business Premium to achieve this;

Actually, what you need should be an Exchange account. You can refer to this link, Office 365 Business Essentials don't include Office applications. 3. Can I use Exchange in the cloud as a single user;

Of course you can. You can just create one user (Global administrator) and purchase one license in your organization.

4. Do I need to have an IMAP account;

After you purchased Office 365 for Business subscription, you should register and create an Office 365 account. Then you don't need to have another IMAP account. The Office 365 for Business account can be configured in client with Exchange, IMAP, and POP3.

teylyn say:

Hello,

it took the MVP community years and years to finally get Microsoft to apply good practice data visualisation principles to Excel, following guidelines set out by Edward Tufte and perfected by Stephen Few. One of these principles is that the axis color and labels should be muted and not be a stark black, to help the data points stand out better. 

If you are interested in learning good data visualisation, take a look at these books:

Show Me The Numbers: Designing Tables and Graphs to Enlighten

Now You See It: Simple Visualization Techniques for Quantitative Analysis

With regards to your question: there is no Excel setting to change these colors by default. 

As a workaround you can create a template from a chart if you want to apply exactly the same formatting to other charts.
