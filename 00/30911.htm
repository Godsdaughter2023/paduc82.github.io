tryx3 say:

It sounds like Windows update is corrupted.  I'd suggest downloading & installing the last Cumulative update manually as it will reset Windows update whilst it is installing.

Go to Windows 10 update history to identify & download the latest Cumulative update for your current version. 

  • I always disconnect from the internet & restart the computer before installing Cumulative updates so I can be confident that Windows update is not in the middle of anything.
  • You then install the Cumulative update by double-clicking on the downloaded file.  Running the Cumulative update does not actually use Windows update itself so you will probably not have any problems with it.
  • It will almost certainly want to restart your computer when it has finished.
  • Once you have installed the latest Cumulative update & reconnected to the internet you will probably discover that Windows update runs correctly and installs updates that might include ones that are several weeks old [depending on how long you have had this problem].

If you start to get similar problems in the future, you can use a troubleshooter that is much better than the built-in Windows update troubleshooter. 

  • Download it from Windows Update - Reset - TenForums 
  • Then right-click on the downloaded file, select properties & set the checkbox for Unblock - this is Windows' way of ensuring that downloaded files do not run without your express agreement.
  • It is a batch file that you must run as Admin [right-click on the downloaded file & you will see this option].
  • You have to restart the computer when the batch file completes.

If you have any questions then please let me know your current version number at the same time.  You can get this by

  • right-clicking on the Start button,
  • selecting {whichever appears} Command prompt / Windows PowerShell,
  • entering winver
