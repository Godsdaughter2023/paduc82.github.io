Sub BlankAmp()
' change occurrence of &followed by characters up to a space to be white
  Dim C As Range
  Dim iStart As Integer
  Dim iEnd As Integer
  For Each C In Selection.Cells  ' Change Selection to Range("B7:H50")
    If Not C.HasFormula Then   ' can't do this for cells with formulas
      iStart = InStr(C.Value, "&")
      If iStart > 0 Then ' there is an &
        iEnd = InStr(iStart, C.Value, " ")
        C.Characters(iStart, iEnd - iStart).Font.ColorIndex = 2
      End If
    End If
  Next
End Sub

jeeped say:
Initiating a macro scrubs the .CutCopyMode to False just as it scrubs all Undo points. If you want to test for .CutCopyMode you must initiate a .Copy within the macro sequence of commands. Try,

Sub aaa()
    ' .CutCopyMode will be false at this point
    Range("A1").Copy
    If Application.CutCopyMode Then
       MsgBox ("Macro Works")
    End If
End Sub

BTW, since CutCopyMode is already a boolean (True or False) it does not have to be compared to one in an If statement.
