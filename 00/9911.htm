I am not sure if the values with the asterisk should have the format or the values without the asterisk should have the format.

If values without the asterisk to have format then use following formula (where A2 is the first cell in the range with the conditional format)

=AND(A2>=2000,A2<=17500,MOD(A2,500)=0)

If values with the asterisk to have the format then use the following formula (where A2 is the first cell in the range with the conditional format)

=AND(A2>=2000,A2<=17500,MOD(A2,500)=0)=FALSE

When you use references like B7, C7 and D7 in lieu of the actual numbers then you need to make them absolute references (with the $ signs) in the formula like the following.
