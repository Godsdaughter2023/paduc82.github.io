If you don't like this feature you can disable it:

Menu > Tools > Settings > General > Navi Wheel > Breathing

john mcghie say:
Really, you do exactly the same thing in Mac Word as you do in Win Word to define a list style.  The only difference being that Win Word has no menus so you must use the Ribbon.  But you get to the same dialog and use it the same way.

This is a complex area of Word (unnecessarily so, in my view...) which means you need to understand the principles behind the mechanism.  If the minor differences in the dialogs throw you, check that you have really understood the workings, because otherwise you may follow a wrong path.  I repeat: you do the same thing -- you are running the same code and calling the same dialogs either side of the fence.

Windows Word has deprecated the Style Separator in favour of using Linked Styles. The style separator was a nasty hack that philosophically was wrong in the way it made Word operate.

Mac Word will work with style separators if they are in the document, but it does not offer any way to insert one. I haven't checked, but I am fairly sure the latest versions of Win Word won't allow you to insert them, either.

If you select some text in a paragraph (less than half of it) and apply a style, Word will coerce that style into a Linked Style at that point, if it has not previously done so. The CHARACTER part of the style (the font attributes) will be applied to the text; the paragraph parts will not.

The style will then appear in the Organiser as two styles: say "Heading 3" and "Heading 3 Char". One half is a normal paragraph style, the other half is a Character style. The two are linked, so that changes you make to the font formatting in the character style will also update the paragraph style.  You could think of it as splitting the style in two so you can apply the character part independently.

Once a paragraph style has been split into a linked style, there's no way to put it back together: it will remain two styles forever.  Unless you look in the Organiser, you will never SEE the character style: it is automatically hidden from view. Yet if your selection does not contain a paragraph mark (whether you can see the paragraph mark or not) there is nothing to which paragraph properties can be applied (the paragraph mark is the container that holds the paragraph formatting). In that case, whenever you apply a style, Word will silently apply only the character portion of the style, splitting the style into its two linked style components at the time, if need be.

Since you started off talking about List Styles, you should be aware that a List Style links a List Template (a numbering or bullets definition) and a bunch of paragraph styles. Whether those paragraph styles have themselves been split into linked styles makes no difference to the operation of the List Style.  That's because the font formatting for the numbering is provided by the List Template in a list style, not by the character part of the paragraph style.  The List style uses only the paragraph portion of the embedded styles.

This is all getting very complex and convoluted! You need to know that Word's formatting is not in the text of the document.  It's contained in a set of "property tables" that look a bit like spreadsheets.  These are stored below the last paragraph mark at the bottom of the document.  In these tables are a set of "pointers" -- numbers that indicate which rows of information to use in other tables.

So a linked style is a row in a table, which has nine columns that can contain links to up to nine paragraph styles.  It also has a column that indicates which one of the 200 possible list templates in the document to use to format the bullets/numbering. A paragraph style is a row in a table that has a column indicating which row in the character styles table to use.

You don't need to know any of this to use Word, but I find it helps me to know what to do when defining styles if I have an understanding of the underlying mechanism.

Hope this helps

james yan msft say:

Hi Hapser,

You can follow these steps:
1. Share your site with the "Everyone except external users" group and choose "Read" permission level.

2. Go to SharePoint admin center > select the site collection > click Sharing button on the top > select "Don't allow sharing outside your organization".


Note: Each user needs to be assigned SharePoint Online license to be able to access any SharePoint Online site, so if the user only has Exchange Online license, he is not eligible to visit the site.
