This type of issue usually arises whenever there is an issue with the Print Spooler, either because it is not running or set to start up manually. The purpose of Print Spooler is to spools print jobs and handles interaction with the printer. If this service is turned off, you won’t be able to print or see your printers. To see if the Print Spooler service is running on your computer:

Open your Start Menu and type in "Services" and hit your Enter key. Once the Services window opens, you will need to look for "Print Spooler" and see if the Status for this service is set to "Running" and Startup Type is set to "Automatic".

https://i.imgur.com/L5aSmLu.png

If the Status for Print Spooler does not display "Running", then navigate your cursor to the top left of the Services window and click on "Start the service".

If Startup Type for Print Spooler is not set to Automatic, double click on the Print Spooler service and a new window called Print Spooler Properties (Local Computer) should appear. Click on the drop down menu next to Startup type and change it Automatic. Click OK and try installing your printer once again.

tryx3 say:

Press and hold the F8 key as your computer restarts - see http://windows.microsoft.com/en-us/windows7/What-are-the-system-recovery-options-in-Windows-7#

As soon as you get any functionality back, get to the appropriate folders & copy your data to an external drive in case you cannot do a complete recovery & need to go further.

If you really cannot complete the recovery, do a Factory Restore.  The instructions are in the PC manual.  I have extracted those for my PC just as an example [they might have been changed in the meantime]

Dell Factory Image Restore [extract from manual for Dell Inspiron 1545]
1. Turn on the computer.
2. When the Dell logo appears, press <F8> several times to access the Advanced Boot Options Window.
Note: If you wait too long and the operating system logo appears, continue to wait until you see the Microsoft Windows desktop; then, shut down your computer and try again.
3. Select Repair Your Computer. The System Recovery Options window appears.
4. Select a keyboard layout and click Next.
5. To access the recovery options, log on as a local user. To access the command prompt, type administrator in the User name field, then click OK.
6. Click Dell Factory Image Restore. The Dell Factory Image Restore welcome screen appears.
Note: Depending upon your configuration, you may need to select Dell Factory Tools, then Dell Factory Image Restore.
7. Click Next. The Confirm Data Deletion screen appears.
Note: If you do not want to proceed with Factory Image Restore, click Cancel.
8. Click the checkbox to confirm that you want to continue reformatting the hard drive and restoring the system software to the factory condition, then click Next. The restore process begins and may take five or more minutes to complete. A message appears when the operating system and factory-installed applications have been restored to factory condition.
9. Click Finish to reboot the system.
