daniel g. grau say:
To Chris:

You cannot reassign, but can code two macros and assign them the shortcut you need.

In your main menu bar, select Tools > Macro > Visual Basic Editor. In the main menu bar, select Insert > Module and paste the following code:

Sub OutlineMoveUp()
    Selection.Range.Relocate wdRelocateUp
End Sub

Sub OutlineMoveDown()
    Selection.Range.Relocate wdRelocateDown
End Sub

Then assign them a shortcut as you tried, except that these will be listed in the Macro category.

If you need help for this, see Install a Macro in MacWord.
