*Original Title - DAT.file extension

burrwalnut say:

DAT files are at the discretion of the programmer, so they don’t have a predefined program that opens them. You and many other people could write a program that uses a file with a .dat extension, but it needs its own program to open it. Most programmers make the extension meaningful, so DAT usually means data, in the same way that DB, or variants of it, usually means database.

 

Contact the email sender to find out which program created the file. It could, of course, be a mistake and its extension was changed, e.g. you could rename a text document from, say, letter.txt to letter.dat. When you double-click on the file, you will not be able to open it, but if you right-click and use Open with > Notepad, it will open.
