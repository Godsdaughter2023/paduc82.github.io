You can check for corrupted system files. Open an administrator command prompt and run SFC if the above doesn't help. Click START, then type sfc in the search box, right-click SFC.EXE and click Run as administrator. Then from the command prompt type sfc /scannow.

Finally if all else fails, you can check the rather cryptic system event log. To do so click Start -> Control Panel -> Administrative Tools -> Event Viewer. Thank you for the information.

You can follow the steps below on how to troubleshoot Power Plan settings in Windows 10:

Method 1: Run the Power troubleshooter.

  1. On the Start menu, type Troubleshoot and then hit Enter.
  2. Under Find and fix other problems, select Power, and then click Run troubleshooter.

Method 2: Restore Power Plan settings to default.

  1. On the Start menu, type Command Prompt. Right-click the result and then choose Run as administrator.
  2. Type or copy this command and hit Enter:
    powercfg –restoredefaultschemes
  3. Exit Command Prompt.
  4. Restart your computer to apply the changes.

Once you have done restoring the Power Plan settings, you can now customize your Power plan settings to which you prefer.
