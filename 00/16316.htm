tom hoke say:

Hi,

   In general when you enter into a cell that has not already been formatted in a specific way, Excel will try to determine the format that you are using (entering) and set that for the cell going forward. For example if you enter:   9:21 AM

into a cell Excel will say Oh! you want AM/PM date/times. If you select the cell and now enter "4:00" it will use the format that is now in the cell and you'll see "4:00 AM".

   You can override this "smart" set formatting for existing cells by choosing the "Time" format from the Number Format drop down. This will set all your times to look like:  6:30:00 AM  ( if your regional settings are US ).

   Note: if that Time format is not the one you want you can use the first feature to work around Excel Online not having full number formatting. In a cell that has not been formatted enter a number/date/time formatted as you want. Then copy that cell and select the range you want to look that way and choose from the Paste drop down menu Paste Formatting. There you go - all the selected cells now look the way you want!
