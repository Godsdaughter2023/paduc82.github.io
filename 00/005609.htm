jp ronse say: Hi,

It looks as you have opened a workbook with all these styles.

The following code (in a normal module) removes all custom styles

Public Sub DeleteAllCustomStyles()
   Dim lngCount As Long, lngDeletedStyles As Long, lngNotDeletedStyles As Long
  
   On Error GoTo Error_DeleteCustomStyles
  
   With Application
      .EnableCancelKey = xlErrorHandler
      .ScreenUpdating = False
   End With
  
   lngDeletedStyles = 0: lngNotDeletedStyles = 0
  
  
  
  
   With ActiveWorkbook
      For lngCount = .Styles.Count To 1 Step -1
        
         If Not .Styles(lngCount).BuiltIn Then
            .Styles(lngCount).Delete
            lngDeletedStyles = lngDeletedStyles + 1
         End If
NextStyle:
      

      Next lngCount
   End With
Exit_DeleteCustomStyles:
  
   ''' reset changed applications settings here
   If lngNotDeletedStyles = 0 Then
      ProgressBar glngHandled, glngToHandle, glngNewTime
      MsgBox Prompt:=lngDeletedStyles & " custom styles deleted.", Title:=JPE_APP_TITLE
   Else
      ProgressBar glngHandled, glngToHandle, glngNewTime
      MsgBox Prompt:=lngDeletedStyles & " custom styles deleted." & vbCr & "Unable to delete " & lngNotDeletedStyles & " styles." _
         & vbCr & "See the help file to manually delete custom styles." _
         , Buttons:=vbOKOnly + vbMsgBoxHelpButton, _
         Title:=JPE_APP_TITLE, HelpFile:=ThisWorkbook.Path & JPE_HELP_FILE, Context:=JPE_HELP_DELETE_CUSTOM_STYLE
   End If
   Application.ScreenUpdating = True
  
   Exit Sub

Error_DeleteCustomStyles:
   Select Case Err.Number
      Case 18
         MsgBox Prompt:=JPE_INTERRUPT_MSG, Title:=JPE_APP_TITLE
         Resume Exit_DeleteCustomStyles
      Case 1004
         lngNotDeletedStyles = lngNotDeletedStyles + 1
         Resume NextStyle
      Case Else
         MsgBox "Error " & Err.Number & " (" & Err.Description & ") in procedure DeleteCustomStyles of Module modUtilities", Title:=ThisWorkbook.VBProject.Name
         Resume Exit_DeleteCustomStyles
   End Select
End Sub

don cuthbert say:

Try shrinking C: with Disk Management (64 GB works if you have the space, although it could be less) and then Custom-install Windows 10 to the space you created. Do not use any Product Key during the install. After Windows 10 is installed, you can launch the Activation Troubleshooter by running changepk.exe /Troubleshoot. Indicate that you have changed hardware and see if it offers you the license from your previous PC. For this to work, you would have to have used a Microsoft Account (MSA) login for Windows on your previous PC and you need to install the same Edition (Pro or Home) as you had on your previous PC. However, if you install Home and it finds that you should have installed Pro, it may offer to upgrade. You also have to use your MSA login on the new install.

If you get Windows 10 activated, you can then just wipe your hard disk and do a full clean install to the entire disk, or mount the Windows 10 media and do an upgrade (now that your hardware has a digital license for Windows 10).

If you can't activate Windows 10, you may find it works well enough without activation.

Also, if you happen to use assistive technologies in Windows, there is still a free upgrade offer:  https://www.microsoft.com/en-us/accessibility/windows10upgrade
