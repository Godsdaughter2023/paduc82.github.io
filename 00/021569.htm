shyam pillai say:

Hi,

You can use this as a reference to rotated 3d shapes with VBA.

http://support.microsoft.com/kb/162236

 

I would specifically add DoEvents in the loops. They are missing in the sample code.

 

 For i = -45 To 45 Step Increment

         ' Rotate the shape to the degree specified.
         FirstShape.ThreeD.RotationY = i
         FirstShape.ThreeD.RotationX = i

         ' Refresh the slide. This step is needed to redraw the screen
         ' after the rotation step; Otherwise, the animation effect is
         ' invisible.
         show.View.GotoSlide 1
         DoEvents

      Next i

      For i = 45 To -45 Step -Increment

         FirstShape.ThreeD.RotationY = i
         FirstShape.ThreeD.RotationX = i

         ' Refresh the slide.
         show.View.GotoSlide 1
         DoEvents
      Next i

chris peng msft say:

Hi Rory0B,

 

For an Office 365 mailbox, I suggest you try the steps below to find the disappeared task:

1.  Sign into OWA (Outlook Web App), go to Tasks App. Make sure you select All in the Filter and see if the tasks appear.

2.  Go to Mail app, navigate to Deleted Items and check the folder.

3.  Right click the Deleted Items and select Recover deleted items…. Check if there are the tasks.

4.  If you still cannot find the tasks, I suggest you involve your admin to search them using the eDiscovery. Please note: To perform an eDiscovery search, you must be assigned the Discovery management admin role by navigating to Exchange Admin Center > permissions > admin roles. Then:
1). Go to Compliance management > In-place eDiscovery & Hold > click New +.

2). In the New In-Place eDiscovery & Hold window, on the Name and description page, type a name for the search, add an optional description, and then click Next.

3). Select Specify mailboxes to search. Then add your mailbox. Click Next.

4). On the Search query page, Select Filter based on criteria and select Tasks as the message type to search.

If the tasks can be searched, you can export the search results into .PST files and import the .PST files into the shared mailbox via Outlook client. For more information, please refer to:
Create an In-Place eDiscovery search: https://technet.microsoft.com/en-us/library/dd353189(v=exchg.150).aspx

Export eDiscovery search results to a PST file: https://technet.microsoft.com/en-us/library/dn440164(v=exchg.150).aspx

Import email, contacts, and calendar from an Outlook .pst file:
https://support.office.com/en-us/article/Import-email-contacts-and-calendar-from-an-Outlook-pst-file-431a8e9a-f99f-4d5f-ae48-ded54b3440ac?ui=en-US&rs=en-US&ad=US&fromAR=1
