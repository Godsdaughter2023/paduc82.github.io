Go to the Desktop in Windows 8, right-click on the Taskbar and select Task Manager.

In the Startup tab, see if the Upload Center is listed, if so, select it and click on Disable.

If you just want to 'kill' the process for this session of Windows, in the Task Manager click on the Processes tab, locate the Upload Service, select it and click on End Task.
