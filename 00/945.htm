steven stanley bayes say:

I have managed to find an answer to the question I had asked :

 

1.      Go to : http://aka.ms/Microsoft365BusinessPreview and request Microsoft 365 ( Windows 10 and Office 365 ). Go through their wizard to create a Microsoft 365 account. After all questions related to the account, the customer is directed to the customer’s page in the Office 365 web site. There is no specific web site for Microsoft 365, the Office 365 web site is used. This web site would show a table view with many things. Download and Install Office 365 is one of them. Do not click this before the installation of Windows 10. Wait for 15 minutes and a new cell would appear on the page table. This cell says Download and Install Windows 10 or alike. Click there.

2.      The customer is then brought to a page where the installation tool of Windows 10 is. There are a few options there. Select the option which provides the Windows 10 ISO and download the ISO file. Burn this file to a DVD or a flash drive ( ensure the PC can start from a bootable flash drive ). Best use DVD to keep the ISO file thereafter. Best keep the copy of the ISO on the hard drive too provided there is enough room and desire.

3.      Start the computer from the DVD or flash and the standard Windows 10 setup would come up. At a certain point, there would be a question whether to use a standard install ( over the old system ) or a custom install. Choose a custom install.

4.      At a given point there would be a question to choose a drive where Windows 10 would be installed. Choose the new drive. Ensure the disk is Primary which should be written on the right side.

Windows 10 will be installed on the new drive. However, the old drive will continue to be used as the only bootable drive. Only one bootable drive is allowed on the system and this will be the old drive. However, Windows 10 installation will alter the record and, at start up, there will be a question from the bootloader as to which system the customer desires to start : the old one on the old drive or the new, Windows 10, on the new drive. The default is the new one : Windows 10 on the new drive. Either do not do anything, in which case the bootloader will wait for a few seconds and then start Windows 10 from the new drive or select Windows 10. Then there will be a standard post setup of Windows.

 

Windows 10 will automatically change the names of the new drive and the old drive : the new drive will always be C : and the old drive will be D : or whatever.

 

All programs and data will remain on the old drive which will truly be untouched ( with the exception of the start sectors where the bootloader is which is, obviously, reconfigured to be able to start either system ). The data on the old drive is accessible from the Windows 10 ( which is on the new drive ).

 

Obviously, the programs which require registration may or may not work with the new Windows 10. Should work OK when the old Windows is chosen in the bootloader. Then, everything is supposed to be the same as before the Windows 10 installation.

 

One may wish to clone the old drive to the new drive before the installation of Windows 10 to make the new programs work. Backup to the new drive and then restore may or may not work.

 

However, the best way is to have a brand new clean install of Windows 10 on the new drive and then to reinstall all programs.

 

Those who do not want to have the bootloader on the old drive and want to have this on the new drive may have to physically disconnect the old drive by a removal of the data cable, insert the DVD or flash with the Windows ISO and start the PC from the DVD or flash. Then, the installation DVD or flash will run the Windows setup, the first window of which asks whether to install Windows 10 or to repair Windows 10. Choose REPAIR and there should be an option to repair the boot sectors. Choose this option and the system will be transferred to the new drive which will become the bootable drive. Then, reconnect the old drive. Be careful : after the reconnection of the old drive, the bootloader may or may or may not have been designed to look for any Windows installation on the old drive. Thus, to start the old system may or may not be possible. I do not know the answer of this question. There should be nothing wrong to use the old drive just for the bootloader, though.

vijay a. verma say:
1. You may login to Windows with a new user profile and see if it helps.

2. Zorvek (http://answers.microsoft.com/en-us/office/forum/office_2013_release-excel/windows-10-and-microsoft-excel-2013/f886877d-b486-4d57-bd61-73e30b7ded81) has given following solution which you can try -
Windows 10 is causing some issues with Office/Excel. Some of the issues are resolved by disabling a new function of the .net installation that comes with Windows 10 and Visual Studio 2015. This feature can be disabled with a registry entry.
To create the entry, run PowerShell with administrator privileges (open the Start menu, enter PowerShell in find box, right-click on PowerShell and select ""Run as administrator"". Enter the following command to add the registry entry:
Set-ItemProperty -Path HKLM:\Software\Microsoft\.NETFramework -Name useLegacyJit -Type DWord -Value 1
You can copy the above line and paste it into PowerShell by clicking on the top left part of the PowerShell window title bar and selecting Edit->Paste. Press ENTER to run the command.

3. If above don't, re-installation of office should solve your problem. Make sure that you have license keys ready before you attempt re-installation.
