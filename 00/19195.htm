bill manville say:

The code looks consistent with your intention.

One reason why it might not copy from all rows with "YES" in column P would be that there is not an exact match with "YES".
If P contained "Yes" or "YES " then the copy would not be done (VB text comparison is case-sensitive by default).

To cater for such anomalies in the data you could use

  If Trim(UCase(Delta)) = "YES" Then

The only other reason I can think of for early termination of the loop would be if you have a Worksheet_Change procedure in Sheet2 that executes an End statement.
