how to synch mail between a POP3 acct on Outlook 2010 client and - Microsoft Community

xjay1000 say:

how to synch mail between a POP3 acct on Outlook 2010 client and multiple android tablets?

currently using contracted POP3 email service and would like to synch email across a couple of android tablets. All the while archive all of the email on my local desktop computer's Outlook 2010 client. In essence a poor man's exchange server. Is there any logical way to accomplish this.
brian tillman [outlook mvp 2007-2019] say:

POP is one-way from the server to the client.  You can certainly access the mailbox from multiple clients and as long as you configure those clients to leave copies of messages on the server, each client will be able to download the messages that arrived since the last time it checked, so all your incoming mail can be available on each client.

Your outgoing mail, however, is another matter.  When POP accounts store outgoing mail, the folder where it is stored is local to the sending client and it cannot be seen by any other client.

In addition, POP isn't folder-oriented and it's a mail-nly protocol.  The only folder POP knows about is the Inbox in the mailbox and the Inbox on the client.  Anything stored in any other folder in the mailbox is not visible to a POP client.  No contacts and no calendar.

IMAP is a better protocol for sharing a mailbox.  It can access folders other than Inbox, it can store outgoing mail on the server so that all clients can see it, and it's two-way.  Any client (or the server, for than matter), can change something in the mailbox and all IMAP clients will see the change.  The problem with IMAP, though, is many mail providers don't support it.  It is also a mail-only protocol.  Like POP, it knows nothing of calendars and contacts.

Another possibility is Hotmail/Live.  Outlook has an add-in called the Outlook Hotmail Connector which, like IMAP, is a two-way link to a Hotmail mailbox, givbing .  Android devices have, I believe, an "Exchange" configuration in their mail clients that can access a Hotmail mailbox and sync with it.  The Outlook Hotmail Connectot support the synching of calendar and contacts as well as mail and I believe there are Android-based tools that also can sync that data with Hotmail.

Gmail and Androids go together well and can sync not only mail, but calendar and contacts.  Using IMAP in Outlook along with Google-supplied add-ins for Outlook, you can sync most of a grmail mailbox between Android devices and Outlook.
