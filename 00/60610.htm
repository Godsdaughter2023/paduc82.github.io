Problems with sending and receiving mail are very rarely caused by faults in the program or in Windows. They are almost always attributable to account misconfiguration either in Windows Live Mail or at the server.

Then there's the question of accessing the same account from your iPad. There can only be one connection at a time to a POP3 mailbox. If the iPad is also using POP3, and especially if it's configured to request push mail, then Windows Live Mail won't get a look in because the iPad is monopolizing the POP3 connection. The only way to check whether this is the case is by turning off the mail app in the iPad for a couple of days to see whether you receive anything via POP3 in Windows Live Mail. Last, there's the question of reverting to Windows 7 from Windows 10. Many of those who have gone through this process have experienced a corruption of Windows Live Mail's store folder, where account settings, mail messages and folders, calendars and custom dictionaries are kept. If this happens, the only sure remedy is to construct a fresh store folder and re-populate it. If you're not seeing problems with missing storage folders or with deleting old messages, you may have escaped this particular problem.

ASUS Q550LF Nvidia Driver Windows 10 Update Fix

Hello,

     So it seems Windows Update is conflicting with Nvidia users as it automatically Updates without your consent. The update for Windows Defender - KB2267602 is what is causing the problem. Ideally, we would have to reinstall the BIOS and disable the installation of the KB2267602 update. To fix this problem:

Step 1). Install WinFlash from the ASUS website.

Step 2). Install the BIOS corresponding to your notebook. (Q550LF in my case) Be sure to put the file somewhere convenient on your computer.

Note: For any specific Q550LF users out there, the latest BIOS update seems to be for the wrong computer type as it downloads for a N550LF notebook. So either download the second or third option.

Step 3). Download this and copy it to your desktop. This is to hide the KB2267602 update from Windows Update

Step 4). Run this command in CMD "C:\Program Files (x86)\ASUS\WinFlash\WinFlash.exe" /nodate This opens WinFlash.

Step 5). Locate the BIOS you want to install on your computer and click Flash and follow the instructions that were given to install your BIOS. This will take around 3-5 minutes.

Step 6). Now this is important. When your computer is in the Sign In Screen, disconnect from the internet right away. We don't want your computer installing or downloading any Windows Update.

Step 7). Run the Windows Update Show/Hide program and click Next, Hide Updates, and find the Windows Defender - KB2267602 and hide it.

At this point, your Nvidia card should be working. To make sure that your computer did not download the KB2267602 update, check your recent updates by pressing the Windows Key, Settings, Update and Security, Windows Update tab on the left, Advanced options, View your update history. If it says that it installed the Windows Defender Update ON THE DAY YOU WENT THROUGH THESE STEPS, then you have to redo the steps.

TL;DR:

1: Download WinFlash and your correct BIOS version from ASUS.com.

2: Download this and stick it on your desktop.

3: Run WinFlash by running "C:\Program Files (x86)\ASUS\WinFlash\WinFlash.exe" /nodate in cmd and "update" your BIOS

4: Disconnect from the internet on the Sign In Screen

5: Run the wushowhide.exe (Windows Update Show/Hide) and click Hide Updates

6: Find "Windows Defender - KB2267602" and hide it

7: You're Done!

You obviously cannot reformat the System Drive from within Windows, as the OS has to be functional throughout the formatting process to complete format, so the task has to be carried out using a Boot Disk/USB.
