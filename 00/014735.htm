Taking sbucki's case, if you have a { DOCVARIABlE Score } field in the document where you want the sum to be displayed, and you give each of the drop downs the Title of Score, the following code in the This Document object will cause the total to be displayed in the Docvariable field

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
Dim cc As ContentControl
Dim lngScore As Long
lngScore = 0
With ActiveDocument
    For Each cc In ActiveDocument.ContentControls
        If cc.Title = "Score" Then
            Select Case cc.Range.Text
                Case "Good"
                    lngScore = lngScore + 5
                Case "Needs improvement"
                    lngScore = lngScore - 5
                Case Else
                    lngScore = lngScore
            End Select
        End If
    Next cc
    .Variables("Score") = lngScore
    .Range.Fields.Update
End With
End Sub

If you assign titles such as Hazard# to the content controls in each row, and you have { DOCVARIABLE Hazard# } fields in the cells of the last column as shown below

and each of the items in the list are preceded by a number such as 1 - First Aid, 2 - Lost Time, 3 - Fatal  as indicated in your example, the following code in the ThisDocument object will do what you want

  Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
 Dim cc As ContentControl
 Dim lngHazard1 As Long
 Dim lngHazard2 As Long
 Dim lngHazard3 As Long
 lngHazard1 = 0
 lngHazard2 = 0
 lngHazard3 = 0
 With ActiveDocument
     For Each cc In ActiveDocument.ContentControls
         Select Case cc.Title
            Case "Hazard1"
                lngHazard1 = lngHazard1 + Val(Left(cc.Range.Text, 1))
            Case "Hazard2"
                lngHazard2 = lngHazard2 + Val(Left(cc.Range.Text, 1))
            Case "Hazard3"
                lngHazard3 = lngHazard3 + Val(Left(cc.Range.Text, 1))
            End Select
     Next cc
     .Variables("Hazard1").Value = lngHazard1
     .Variables("Hazard2").Value = lngHazard2
     .Variables("Hazard3").Value = lngHazard3
     .Range.Fields.Update
 End With
 End Sub

When inserting the DOCVARIABLE fields, you must use CTRL+F9 to insert the field delimiters { } and you use ALT+F9 to toggle off the display of the field codes.
