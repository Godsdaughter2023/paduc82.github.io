Method 1:

Run a System File Checker (SFC) scan to check for file corruption and restore it.

  1. Press Windows key + X, click Command Prompt (Admin).
  2. In the Command Prompt, type the following command, and then press ENTER:
    sfc /scannow

Method 2:

Run the Deployment Image Servicing and Management (DISM) tool as follows.
  1. Open an elevated command prompt. Press WinKey + X, click Command Prompt (Admin).
  2. Type the following command, and then press Enter. It may take several minutes for the command operation to be completed.
    dism.exe /online /cleanup-image /scanhealth
  3. After you get message “The operation completed successfully” type second command and press enter. It will take some time to finish.
    dism.exe /online /cleanup-image /restorehealth
Important: When you run this command, DISM uses Windows Update to provide the files that are required to fix corruptions.

This is the Santovec bug that you can read about on this page: Use Message Rules

With a bit of subterfuge, you can grab most of the Facebook notifications with a rule that looks at the message body. These notifications have two parts - an HTML part which most users will see, and a plain text part for those who can't or won't read messages in HTML. The plain text part seems always to end with the valedictory Thanks, The Facebook Team, although that doesn't appear in the HTML.

So, set up your rule like this:

Apply this rule after the message arrives


Where the message body contains 'The Facebook Team'
<do stuff>


    and Stop processing more rules
