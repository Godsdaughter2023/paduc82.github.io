bossdweeb say:

"It was working great for awhile until my sister changed the resolution settings in game. "

Generally 640x480 is a res that is most likely to cause problems. If 640x480 does not help with your monitor

try using 800 instead of 640.


=xres 640 ... Hmm, I believe that's supposed to be xres=640, or maybe  -xres=640.

Don't forget the space  after the file path.

And don't forget to delete this after you get back into the game and restore the correct resolution.

If you forget it will set you back to 640x480 again.

Alternatively (my choice), you could access the games configuration files ( .cfg, .ini, etc. ) and change the

resolution there instead of doing the shortcut trick. I don't know where these files are for this game,

but vintage games often keep saves and user configuration file in the main install folder.

If not, there should be a folder in your Username\Documents or AppData folders.

[ EDIT: If the game came from Ubisoft - UPlay the files may be elsewhere. ]

The files can be opened with notepad.

- A more desperate measure would be to reinstall the game. This would reset the setting to default.Back up the save folder/files first so you can use them after reinstalling.

"my estimated memory on my graphics card is over 25GB "

No way in the world. The game has a minimum requirement for a 16 MB (VRAM) graphics card

(32MB for higher settings) . This is a tiny amount of VRAM by current standards. 

No game has a 16GB system RAM requirement. 

- If you machine is using a dedicated graphics processor (card) the most  common amount of VRAM 

would be between 2- 4GB.  6+GB is available on some of the newer high end graphics cards though.

- If your machine is using an Integrated graphics processor,  eg- Intel HD, it has no dedicated VRAM.

Integrated graphics uses some of your system RAM. The amount varies from machine to machine,

but 2GB is a common maximum. Old games such as this generally run well on modern Intel HD graphics.

"I'm running DirectX 12 "

Win 10 has DX9,10,11 & 12. The game selects which DX file it requires. This one would use a DX9 file.
