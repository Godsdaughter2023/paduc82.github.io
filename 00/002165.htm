stefan blom say:
I'm wondering if the user is inadvertently pressing Enter twice, perhaps because of a damaged keyboard. It would certainly be worth testing a different keyboard.

On the other hand, what's the user's primary use of AutoComplete? To add the current date? If so, you could install a macro that inserts the date. You can attach the macro to a button on the Quick Access Toolbar and/or to a keyboard shortcut; that way, it will be easy to run it.

Here's one suggestion:

Sub InsertCurrentDateAtIP()
Selection.Collapse wdCollapseEnd
Selection.InsertAfter Format$(Now, "mmmm d, yyyy")
Selection.Collapse wdCollapseEnd
End Sub

RANDBETWEEN(...) will not reprocess if you do not like the result, but you might be able to use WORKDAY(...) to select the next Mon-Fri from (7/1/2112 - 1) to (7/31/2112 - 1). With 7/1/2112 in A1 try something like,

ossiemac say:

Use the WORKDAY function to create a list of the workdays in the month. (In some out of the way area on a worksheet. I have used column A in my example.)

Custom Format a column as dates to include the named day of the week. (This is just for convenience to display the week days together with the date.) I used the format "ddd dd mmm yyyy"

In the first cell (A1) enter the Date of  the last day of the previous month. (I entered 30 Jun 2012 because you said you want random days in July.)

In cell A2 enter the following formula:-

=WORKDAY(A1,1)

It will display the date of the first workday after the date in cell A1. (This date will be the first workday in July.)

Drag the formula down until the displayed date is the last workday of the required month.

Now note the range of row numbers because we are going to randomize the row numbers to get random selections of the workdays from the list. The row numbers will start on row 2 because row 1 contains last day of previous month. Therefore for the month of July 2012, the range of row numbers is 2 to 23.

Now in another cell enter the following formula:- (Formula edited since original post due to an error.)

=INDIRECT("A"& RANDBETWEEN(2,23))

Drag the formula down until you have a list of random dates. If you only want all unique days then create a long list that is most likely to include all dates at least once. Copy and paste special values and then  and use Advanced filter to create a unique list.
