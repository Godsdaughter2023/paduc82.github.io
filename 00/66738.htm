how to return to start screen after closing an app - Microsoft Community

denniscutchin say:

how to return to start screen after closing an app

when I close an app it go to the desktop screen

rajesh govind say:

 

Hi Dennis,

 

Please let me know these information for a better understanding:

 

1)       Does this happen with both the Modern User Interface Apps and the Desktop Apps on the computer?

2)       How did you install Windows 8.1 on the computer? Was it a free upgrade installation from Windows 8?

3)       If yes, did you face this issue on Windows 8 as well?

4)       After closing the app and when you come to the Desktop Interface, what happens when you press the “Windows Logo” key on the keyboard? Does this take you back to the Start Screen?

 

Let me first inform you that when you close a desktop app, you will be taken to the Desktop Interface rather than the Start Screen. This is because the desktop apps can be opened only from the Desktop Interface and so when you close it you will remain on the same interface. But, when you press the “Windows Logo” key on the keyboard, you will be taken to the Start Screen of the computer. However, when you close a Modern User Interface App, it shouldn’t take you to the Desktop Interface. If this happens, then there would be some incorrect settings preventing the Start Screen to load properly on the computer.

 

As the first step, I would suggest you to check the “Screen Resolution” of the computer. You need at least 1024 X 768 screen resolution for Metro apps and the Start Screen to work properly on the computer. The Windows apps will not work in screen resolution less than 1024 X 768. To snap apps, you need a screen resolution of at least 1366 x 768.  Make sure that the screen resolution meets these requirements. You can check this by:

 

a)       Right-click the mouse from the desktop interface and select “Screen Resolution” from the menu.

b)       Check the screen resolution shown there. It should be at least 1024 X 768.

 

Also, make sure the User Account Control (UAC) is enabled on the computer.  To enable the User Account Control on the computer follow these steps:

 

a)       Press “Windows Logo” + “W”, type “Account Control” in the search bar and press “Enter”.

b)       In the “User Account Control Settings” window, move the slider to the default settings and click “OK”.

 

If the issue still persists, then let’s check if this is caused by any missing or corrupted system files on the computer. To check this, let’s run an SFC Scan and check the result.

 

Run an SFC Scan

 

The sfc /scannow command scans all protected system files and replaces incorrect versions with correct Microsoft versions.

 

a)       Press “Windows Logo” + “Q” keys on the keyboard and type “cmd” in the search box.

b)       Right click on the “Command Prompt” icon from the search results and select “Run as administrator”

c)       On the command prompt window, type the following command, and then press “Enter”

 

sfc /scannow

 

d)       Follow the on-screen instructions and complete the scan.

 

Important: It may ask for windows installation DVD to fix and to enable SFC to make more than minor repairs. Some files saved on your computer might get deleted when fixing corrupted files.
