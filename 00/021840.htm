jeff toole say:

Unfortunately, many common malware do exactly what you are doing.  You have a few options:

 

1. Use a browser plugin to block sites instead of using the hosts file

2. Exclude the hosts file (as you have described).  This will prevent MSE from being able to detect any infection in your hosts file, so you may need to clean it manually in the future, if you get infected.

3. Exclude only the "SettingsModifier:Win32/PossibleHostsFileHijack" threat.  You can do this by making the change to your hosts file and running a quick scan.  When we detect the hosts file change, you can choose the "Allow" option.  We will now ignore this detection, but we may detect some other changes to the hosts file when it comes with actual malware.
