mike h.. say:

Hi,

 

I'm not sure I follow this but there are several methods. This one uses a formula. Enter this ARRAY formula in A2 then drag down. It will produce unique random numbers between 1 & 50.

 

Commit the formula with CTRL+Shift+Enter

 

 

=LARGE(ROW($1:$50)*NOT(COUNTIF($A$1:A1, ROW($1:$50))), RANDBETWEEN(1,50-ROW(A1)+1))

 

 

or another method using code.

 

 

 

Sub Unique_Randoms()
Dim FillRange As Range
Set FillRange = Range("A1:A50")
For Each c In FillRange
Do
c.Value = Int((1000 * Rnd) + 1)
Loop Until WorksheetFunction.CountIf(FillRange, c.Value) < 2
Next
End Sub

How to set up a document with front matter numbered separately in Word 2002.. Part of the document uses roman numerals and the remainder is normal numbering.

hansv mvp say:

Click at the end of the front matter.

If you have a hard page break there, remove it.

Then select Insert | Breaks, click Next Page and click OK.

Assuming that you have or want the page numbers in the page footer:

Go to page 1.

Select View > Header and Footer.

Use the Switch between Header and Footer button on the toolbar to activate the footer.
