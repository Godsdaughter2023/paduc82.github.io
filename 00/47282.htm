OK, I've solved it!

Writing the previous message actually made me think... It must be some sort of issue with the manufacturer's hotkey application. And it was. Or possibly user error. Looking at the app settings, I may have had the hotkey power plan set as 'power saver' or similar. I didn't check before uninstalling. But if this was the case, then that setting would presumably disable wifi and dim the screen.

Hello, I have some help for you two. Error code E73 indicates a possible hardware failure. Here is a solution for you guys.

Original xbox 360    

Solution 1:

1. Turn off your console.

2. Disconnect the power cord from the back of your console. Unplug the power cord from the wall outlet and the power supply unit.

3. Reconnect all three connections making sure they are secure.

4. Turn your console back on.

If problem persists go to the next solution

Solution 2:

1. Remove the Hard Drive from the console

2. Power on the console and check if the problem is still there.

      If yes: The Hard drive isnt the problem, reattach the hard drive and go to the next solution

      If no: reattach the hard drive and make sure the problem only occurs when the hard drive is attached

3. If you haven't already, Reattach the hard drive and power on the console

If problem persists go to the next solution

Solution 3:

*If this doesn't apply to you go to the next solution*

1. Remove any memory units or USB Flash Drives attached to the console

2. Power on the console

If the problem persists go to the next solution

Solution 4

1. Detach all accessories so that only the power cable and the a/v cable are attached to the console

2. Power on the console

If the problem persists see below

If the above four solutions have not solved your problem then the E73 error code is indicating a hardware failure and your xbox requires repair. You can submit a repair request by visiting https://myservice.xbox.com. Please Note though you have to register your console which you can do through the above site.
