USB functionality of Windows 10 is inferior to Windows 7 - Microsoft Community

terryq2 say:

USB functionality of Windows 10 is inferior to Windows 7

I've converted three computers from Windows 7 to Windows 10.  The two laptops converted fairly well.  Not so for my desktop, where I have a lot more USB connections.  On the desktop, I was running Windows 7 Pro 64 bit, and the Microsoft upgrade to me to Windows Pro 64.  It is installed on an HP Desktop with an Intel I7 Quad processor, with 16 GB of ram.  All of the USB issues I'm having worked fine when I was running on Windows 7 Pro 64.

After considerable effort to fix all of the problems, I've concluded that Windows 10 USB compatibility is inferior to Windows 7. 

Issue #1.  A USB connection from my multifunction printer stopped working after the conversion.  I tried several times to reinstall the printer.  Each time, I got the following message:

"USB Composite Device is an older USB device and might not work with USB 3.0. 

Plug the device into an available USB 2.0 port, and click Next.  Otherwise click Cancel."

First, USB 3.0 is designed to be backward compatible with USB 2.0 devices, so the first statement in the Microsoft error message is saying that Windows 10 doesn't follow the rules of USB 3.0. 

Secondly, my printer was already plugged into a USB 2.0 port on my computer when I got the Windows 10 error message.  My computer has six USB 2.0 ports and four USB 3.0 ports.  I know where they are. 

The only way I have been able to run the printer was to uninstall it, and reinstall it manually.  At the point in the Windows 10 where it asked what kind of connection to make, Windows 10 gave options for COM ports, LPT ports (my computer has none of those), some other ports that I've never heard of, but none of the open USB 2.0 or USB 3.0 ports.  This part of Windows 10 installation apparently doesn't even know that my computer has some of them available (although in Windows Device Manager, they are all listed).  But it did offer one "Virtual USB001 port.  I chose that.  Now the printer will print, but has lost some back-and-forth functionality, such as checking the ink levels. 

Issue #2.  A USB connection to a weather display, which worked fine in Windows 7, failed to work in Windows 10.  After troubleshooting this for a while, I have determined that if every time I restart the computer, and wait until the error message comes up, if I then unplug and replug the device, and wait a few seconds, I can click OK on the error message and the problem goes away.  But that was not necessary under Window 7.

Issue #3.  USB connection between a Palm personal digital assistant and the computer.  Initially after the upgrade to Windows 10, it operated okay.  But while working on other compatibility issues, I had to uninstall the driver for the Palm USB.   After that, the driver would not re-install in Windows 10 (I tried many things), even thought it previously would install in Windows 7,  I was only able to get it running again by going back to a restore point I set just after upgrading, so I could get the functionality back without having to reinstall the driver. 

Lesson-Learned:  After you upgrade, be sure you configure Windows 10 to make a "Restore Points."  After the conversion from Windows 7 to Windows 10, that is turned off, even if it was on in Windows 7..  Then make a restore point right away.

I've posted these comments as a question, because if anyone has an idea to fix any of these problems, please let me know. 

But my message to Microsoft is that there are some weaknesses in Windows 10 in its management of USB ports, compared to Windows 7.  For the benefit of your customers, it would be appreciated if you fixed these problems and included those fixes in a furture upgrade of the operating system. 

Terry
