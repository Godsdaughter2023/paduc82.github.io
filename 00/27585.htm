gary zhu msft support say:

Hi Vivek,

 

After checking with our backend team and your tenant is not affected with the issue EX94603. To fix this issue, please try the following:

a). Remove Outlook from Index:

  1. Go to Outlook Settings (File | Options)
  2. Go to the Search settings
  3. Click on Indexing Options
  4. Click Modify and deselect Microsoft Outlook
  5. Click Close to dismiss dialog box
  6. Exit Outlook.

 

b). Ensure Outlook Files Can Be Indexed:

Any Outlook OST or PST file that you’re using must be allowed to be indexed by Windows.  This is a default file attribute when Outlook indexing is enabled, but for some reason, it was changed during my recent OS upgrade.

  1. Find all OST and PST files that you’re using on your computer. In Windows 8, OST files are usually hidden and stored in the :\Users\[username]\AppData\Local\Microsoft\Outlook folder.
  2. For each file you’re using, right click on the file(s) and select Properties.
  3. Click on the Advanced button on the general tab.
  4. Ensure that the Allow this file to have contents indexed in addition to file is checked.

 

c). Re-Add Outlook to Index:

  1. Follow the same steps in the step “a).” and re-add Microsoft Outlook to the search index.  Start Outlook, go to the Indexing Options, and add Microsoft Outlook back to the list.

 

d). Rebuild Index (Optional):

If it’s still not working for you after you perform step a), b), and c), then you may need to rebuild the search index.  To rebuild the index:

  1. Go to Outlook Settings (File | Options).
  2. Go to the Search settings.
  3. Click on Indexing Options.
  4. Click Advanced.
  5. Click Rebuild.
  6. Click OK and then Close to dismiss the dialogs.

 

Moreover, to better understand you situation, I’d like to collect the following information:

  1. Regarding “I upgraded Office 2013 to Office 365. When I did so, my emails accounts were linked to Outlook 365”, do you mean you just use Office 365 account to download the Office suites Or use Office 365 Exchange Online account?
  2. The type of your account, like Gmail, Exchange on-premises.
  3. Has your Outlook 2013 search suddenly stopped working? When you perform search, is the result empty?
  4. The detailed version of your Outlook 2013 client.
