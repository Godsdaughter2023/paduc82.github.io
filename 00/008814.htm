rohn007 say:

What verison of Office do you use at home?  RTF is more portable than DOC, but even RTF structure changed over time. Trying to read an Office 97 version of RTF in a current version of office may be a problem.

Try opening the RTF file in Wordpad instead of Word.

Another possibility is that you are not following the proper process to remove a USB drive. You have to close the USB drive from the task bar. If you do not close the USB properly there is a chance that part of the file is not saved to the USB, corrupting the document. When windoze is writing to USB drives, it first saves file to a temporary cache file on the computer HardDrive. When there is enough in the cache to write a full "block" on the USB it gets moved there. Closing the USB properly "flushes" any partial blocks of data to the USB drive before allowing you to remove the USB drive.
