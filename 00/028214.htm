john mcghie say:
Hi Etaoin:

Well Tahoma is working on this Mac, and it's the 5.01.2x version you want.

All I can do is make some general remarks:

1)  I strongly suspect there is at least one other copy of Tahoma on your system "somewhere" that you have not found yet.  This problem is almost ALWAYS caused by duplicate fonts (but see below).

Remember:  OS X has potentially a LOT of font folders.  To solve this issue, I put ALL my fonts in the root library fonts folder /Library/Fonts.  I have no fonts anywhere else, which cuts the problem in half (and makes sure that all of the fonts on the system are available to all of the users, without me having to think about it...)

2)  Generally, I have found that "Disabling" fonts doesn't work very well.  I recommend that you enable all of the fonts you have, and leave them that way :-)  Word gets a bit confused if fonts start coming and going while it's operating.

In the case of duplicate fonts, I strongly recommend DELETING them from the system. (Then re-starting).

3)  If you are using other font managers, turn them off: I have found this leads to problems.  Stick with Font Book (at least until you solve this problem).

4)  This issue can also be caused by "other" fonts that are not duplicates, but which prevent the wanted fonts from loading.  To analyze this, remove all the fonts that were not provided by Apple, then bring them back, in batches, until the problem re-appears.

5)  You should not need to "clear the font caches", but if you do, some tools corrupt them so they are not writeable.  That will cause you to chase your tail forever, because the font caches will not register any changes.  I recommend a power-off system restart, which should bring all the font caches right.  If it doesn't, you need to DELETE them, not clear them, and empty the trash.

6)  I don't know why Microsoft is shipping outdated copies of its own fonts!  However, you do not NEED the "Microsoft" folder in your fonts folder.  You can delete it.  If you do, you will get it back again on the next update.  Or you can "Update it".  To do this, copy those fonts from your font folder into that Microsoft folder in your font folder.  That will overwrite any Microsoft versions with the Apple versions of the same name.  Then copy them all back to your font folder: that will ensure that your font folder has a copy of all the Microsoft fonts it did not have before, but that all of them will be the latest version.
