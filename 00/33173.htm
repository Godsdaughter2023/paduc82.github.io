harish b n say:

Hi,

 

You will not be able to customize System Reserved partition or make changes to the partition.

 

You will not be able to Copy contents form your old System Reserved partition to the one newly created customized System Reserved partition. So there is not point on working with this issue.

 

However, you will be able to customize or create a customized System Reserved partition while performing a Clean install of Windows 7 on your new hard drive that does not have any partitions on it.

 

When Windows 7 is installed on any partition, it creates a system reserved partition by itself on the same. It cannot be created manually.

2. In Device Manager list, look for the item you want to uninstall, select the item, right click on it and click Uninstall.
3. Once the driver is uninstalled, restart the computer for the changes to take effect.

NOTE: Uninstalling the drivers from the computer will result in malfunctioning of the appropriate device or may not work correctly.
