This option reverts your PC to an earlier point, called a system restore point. Restore points are created when you install a new app, driver, update, or when you create a system restore point manually. Choose a restore point created before the error occurred.


Restoring your PC to an earlier point will remove updates and drivers installed after the restore point was made. It won't affect your personal files.

brian tillman [outlook mvp 2007-2019] say:
A quick format rewrites all the pointers on the disk's allocation table to indicate all the space in empty.  Any write to that disk can grab an empty pointer and scribble on the sectors indicated.  Just because the header of a file indicates that there is still data in that file doesn't mean the data means anything, since all the other pointers to the sectors that file was using have been rewritten and point to what is considered unused space.  Without some fairly low-level tools, I don't think you stand much of a chance to recover your data.
Brian Tillman [MVP-Outlook]
