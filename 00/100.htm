teylyn say:

Hello,

Excel has functions that can get data from the web and display that data in Excel cells. You can refresh the data connection to update to the latest value of the data source.

You can use that data to build Excel charts. 

But Excel does not have a functionality that gets a chart from the web and updates in an Excel workbook to reflect the current state of the chart on the web. 

This can be done with code, but not with a formula or by ticking a few boxes in a dialog.

Keep in mind that data on web sites may be in any of hundreds of formats. Charts on a web site may be generated dynamically by scripts, and not be reproducible by data that is showing on the web site. These days, a lot of web content is generated dynamically, and the underlying data is not exposed to a site visitor who only sees the chart.

In such a case, there is no data that you could use with Excel, either.
