Hi Guy H(2),

Did you mean migrate the initial domain and accounts of your Office 365 tenant hosted by third party to Microsoft (in house)? So if you purchased Office 365 subscription provided by third party, you cannot directly migrate data from third party Exchange server to Microsoft Exchange server.

If you'd like to use the email service provided by Microsoft, you need to purchase a new subscription plan in Microsoft. User defined color palettes for highlight, font and borders - Microsoft Community

ccweems say:

User defined color palettes for highlight, font and borders

The default color palettes provided by Word and other Office applications most often do not work well for display and printing.  Far too often I see printed documents produced by others which undoubtedly looked fine on screen but when printed at the last minute prove to be a nightmare as printed.  In some cases there are substantial differences between color printers and to a lesser extent between monitors.  

I don't expect MS to resolve this by changing the default colors but I do suggest that at the very least they allow the user to design a template whereby the default "Standard Colors" can be defined and these selections will be durable when the distributed document is viewed or printed.  A more user friendly option would be to allow the user to define what is titled "Standard Colors" or perhaps add a line of "Custom Colors" in the font, fill and border dropdowns.  As an option with would be valuable  allow IT to add a modified standard palette to all installs.  Many graphic savvy companies have specific colors that are to be used for documents.  They employ some workarounds which crudely handle some applications but this is not a competent answer to the problem.  

It appears that much of this could be accomplished by adding a setup option at the bottom of the color selection dropdown which is essentially the same for font, fill and borders.  Modifying the highlight selection dropdown would much the same.  While it would be foolish for me to suggest that these changes are easy I am not a stranger to coding.  The changes discussed essentially affect the color definitions stored in a table which are looked up as needed.  These changes do not affect the behavior of other features nor do they limit the range of colors available.

In light of the many complicated and little used features that have been added to Word I think it is time to add some long needed features that will be put to immediate use by many.

    Hello Vikram,

    Can you kindly elaborate, which kind of reset you wish to perform?

    If you are looking for

    1) Soft reset --> Press and hold Volume and Power(ON/OFF) buttons simultaneously for 10 seconds. No changes will be made to the phone. This option is preferred mainly if phone freezes or not working smoothly.

    2) Hard Reset --> From phone navigate to All Settings --> System --> About. This screen will display details about OS and your device. Scroll down this screen till the end you will find a button named "Reset Your Phone".

    To perform Hard reset you should need to use this option. This option will reset your phone and will probably remove the application, and it will ask you whether you want to stick with current OS or wish to rollback to previous OS if any.

    You will also get option to select clean installation which will start over the phone with current OS but with no applications you have installed manually.

    3) Factory Reset --> This option requires Windows Recovery Tool provided by Microsoft which is available from https://support.microsoft.com/en-in/help/12379/windows-10-mobile-device-recovery-tool-faq link.

    You need to install this tool on your windows 7/8/10 desktop or laptop. Connect your phone to the desktop of Laptop. Launch the installed tool, it will display your connected mobile device. Follow the instructions. You will get button to recover the OS of your mobile to factory made OS (i.e. the version you received when you purchased the mobile). Once you click the reset option, the primary OS will start downloading. It will take some time based on your internet connection. Once the download completes, mobile configuration will begin.

    These configuration settings and steps are same which you carried out when you started your phone for first time after purchase. It will take about 10-15 minutes for completion.

    You will get the phone in the same state as you got it after purchase. You need to manually install everything again.

    I hope I provided you all the necessary options and steps to carry out them. Do reply if any additional details are necessary.
