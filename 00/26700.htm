Mirror Margins In Access Report

I have a report set up in Access with the following margins (Top .25, Bottom .5, Left 1.5, Right .50) that I would like to print double-sided to hole punch and place in a 3 ring binder.  I cannot find the Multiple Pages Mirror Margins fuction that is in Word.  Does anyone know how to do this in Access Report?
john p spencer say:

Nothing built in, but the following should work.  You may have to play with the left and right margin settings

 

Move the left position of each control as needed for each even page, then back for each odd page.

In the Code Window Declarations section, write:

Option Compare Database
Option Explicit
Dim MoveMargin As Integer
====
Code the Report Header Format event:

Private Sub ReportHeader_Format(Cancel As Integer, _
                      FormatCount As Integer)
MoveMargin = MoveMargin * -1
ChangeMargins
End Sub
=======

Code the Page Header Format event:

Private Sub PageHeader_Format(Cancel As Integer, _
                                               FormatCount As Integer)
If Me.Page = 1 Then
   MoveMargin = 0
ElseIf Me.[Page] Mod 2 = 0 Then
   MoveMargin = 1440
Else
    MoveMargin = -1440
End If
    ChangeMargins
End Sub
=========
Add a new Sub Procedure to the code window:

Public Sub ChangeMargins()
Dim C As Control
For Each C In Me.Controls
    C.Left = C.Left + MoveMargin
Next C
End Sub
====

Change the value of MoveMargin as needed.
1440 = 1 inch.
Make sure there is enough room to the right of the right-most controls to allow for the movement towards the right.

"If you want to add a footnote in a "forbidden mode" like in math mode, tables, or boxes, you have to do it in two easy steps. At the place you want the little superscript number (in the equation, for example) put
\footnotemark

Then, right after the "forbidden mode" ends, put the text of the footnote in the command

\footnotetext{The text of the footnote goes here.}

Admittedly, if there are multiple footnotes in the same "mode," this can get a little messy, but it can be done. Check the appropriate references. Suffice to say it can be done."

Has anyone tried to make multiple levels of footnotes work?

suzanne s. barnhill say:
You can have two series of footnotes, but not necessarily separately. You can use ordinary footnotes with ascending numbers, and you can also assign a "custom mark" to any given footnote. So, for example, you can use  *, †, ‡, §, ¶ (by inserting each character for a separate footnote, not by selecting that series in the "Number format" box), or you could use a, b, c, etc. Whatever character you assign is static; it will not update if you add other notes before it, and the number sequence of numbered footnotes will ignore the "custom" footnote and wrap around it.

Hi Ahmed Khan_,

 

Welcome to Microsoft Community. Let me assist you on how to transfer your emails from an Outlook.com or Gmail account to another Outlook.com.

 

For a Gmail account, you need first to add your account to an email client like Windows Live Mail. Once done, follow the steps below on how to import your emails to an Outlook.com account:

 

1. Sign-in to your Outlook.com account via a web browser.

2. Click the cog icon beside your Profile name.

3. Select More mail settings.

4. Under Managing your account, select Your email accounts.

5. Under Add an email account, click Add a send-and-receive account.

6. Below the page, click the link "upload your email to Outlook.com".

7. Follow the steps given in the page prior uploading your email messages to cloud.

 

Note: If you still do not have a Windows Live Mail installed in your computer, click here to download. To add an account in Mail, click here.

 

For Outlook.com account, you can still use Mail for importing emails. You just need to add another Outlook.com account to Mail and drag and drop the email messages from one Outlook.com account to another. It will sync the email messages online.

john korchok say:

Here's where to find it:

Perhaps you are providing a placeholder for someone else to insert a picture that will be a particular size. Jim mentioned a Frame (available on the Developer toolbar, which is hidden by default).

You can also create a one-cell table with the width and height set to exact measurements. A picture inserted here will fill one dimension if it doesn't match the aspect ratio of the table cell.

You can also use VBA to insert a Picture Content Control, which was designed to do this job:

  1. In Word, choose Tools>Macro>Visual Basic Editor. The Editor window appears.
  2. At the bottom should be a window called Immediate. Click inside it.
  3. Paste this command, then press Return:
    ActiveDocument.ContentControls.Add(wdContentControlPicture)
  4. Close the editor window to return to your document. It will have a light blue square windows that you can resize to your preferred dimensions. For a user to insert a picture, they would click on the icon in the middle of the Content Control to open an insert dialog.
