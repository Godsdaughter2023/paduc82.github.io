First of all, the easier way to get to the end of any document is to hold the Ctrl key while pressing and releasing the End key. If you don't mind remembering that, you don't need to do anything else.

If you want every document you open to jump automatically to the end, follow these steps:
  1. Open any document in Word.
  2. Click the Record Macro button on the status bar at the bottom of the Word window (if you don't see that button, right-click the status bar and check the box for Macro Recording).
  3. In the dialog box that opens, change the Macro Name box to say AutoOpen. Leave the Store Macro In box as "All Documents (Normal.dotm)". Click OK.
  4. Hold the Ctrl key while pressing and releasing the End key.
  5. Click the blue square button that the Record Macro button turned into to turn off the recorder.
  6. When you close Word, if you're prompted to save the Normal.dotm template, click Yes.

A fax, is a way to scan a hard copy piece of paper, transmit it and print a hard copy at the other end. Not trying to be cute here. Do you have a document on the phone that you wish to send to an actual fax machine?  Fax machines work on phone numbers. hansv mvp say:

This method is not foolproof, since a sentence can contain words that begin with an upper case letter, such as a name.

But you could use   (^13)(<[a-z][A-z0-9]{1,}>)   in the Find what box, and a space followed by \2 in the Replace with box. The parentheses in (^13)(<[a-z][A-z0-9]{1,}>) divide the found text into 2 parts, indicated by \1 and \2. \1 is the paragraph break and \2 is the word following it.

You could obtain the same result by entering   (^13)([a-z])   in the Find what box and a space followed by \2 in the Replace with box.

For others reading this: you need to tick the Use wildcards check box in the Replace dialog.

Setting fixed vertical distance between objects

davidf2 say:

I would question creating different text boxes for each paragraph. That doesn't make sense to me. But with that being said here is one way to get a consistent distance between two objects that I have used.

First go to View and makes sure that Boundaries and Guides are checked. It is hard to see the boundary of a an object if you don't with the new versions of Publisher; you must mouse over the text box to have the boundary appear...a real problem if you have an empty text box as it can disappear on the page.

Second go to Page design and make sure Guides and Objects are checked under Align to.

Now create a text box or even a rectangle object that is the height of the fixed vertical distance you want and perhaps 4 or 5 times as wide, to use as a spacer bar. For example if you want an the vertical distance between objects to be .125 inches then make a text box that is .125 X one inch or more. You can right click the text box > format text box > size tab > type in the exact desired height and width (or under the Drawing tools Format). Now fill that text box with some color so it is easy to see and you have a 'spacer bar'.

When you have two text boxes that you want to be .125 inches apart drag your newly created spacer bar text box down to either lower corner of the top text box and let part of it snap to the bottom corner of that text box. Grab the second text box and drag it into position until it snaps to the bottom of the spacer bar. Then grab the spacer bar and drag it off into the scratch area until you need it again. This is most easily done if you let about an inch of the spacer bar extend past the edges/corners of the two text boxes so it is easy to select, grab and drag. The result is the two text boxes remain your 'fixed vertical distance' apart.

You can use vertical guides to align the sides of the text boxes. And remember that if you hold down the shift key as you drag an object it will drag straight vertically or horizontally. So if you drag one text box over another and let the sides snap in alignment you can then hold down the shift key and drag and they will stay in alignment.

Another method I have used is to simply snap two objects together and then use the arrow keys to nudge the obects apart by a consistent number of 'nudges'...say 3 key strokes. You can set a custom nudge distance under Settings.

As I said, I don't think using separate text boxes for each paragraph makes sense. You can change the paragraph settings to set the line spacing between paragraphs...but there are a few ideas to get you started.
