stefan blom say:

To get rid of all auto numbers, do the following: Make sure that the document you want to convert is the active document in Word. Press Alt+F11 to display the Visual Basic Editor. Press Ctrl+G to activate the Immediate window. Type

ActiveDocument.ConvertNumbersToText

and press Enter. Close the Visual Basic Editor. Assuming that you did not create the numbered paragraphs by applying numbered styles, you are done. Save the document.

If the document includes numbered styles, you will have to clear their numbering as well, which you can do in the Modify Style dialog box (for each numbered style). Click the Format drop down, click Numbering, and select the "None" option (even if it already appears to be selected) and confirm with OK.
