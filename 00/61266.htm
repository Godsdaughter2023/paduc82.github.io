I want to keep a specific restore point indefinately. - Microsoft Community

neightwatts say:

I want to keep a specific restore point indefinately.

I want to hold on to a specific restore point. but vista automatically deletes old points as it creates new ones, and if I turn off protection then it will delete all of them.
how can I save the restore point indefinately?
anannya podder say:

Hi neightwatts, 

Thank you for posting on the Microsoft Answers Community Forums.

 

By default System Restore uses about 10-12% of the drive. However you can change this setting if you wish to. System Restore works by saving a copy of registry settings and other system information that Windows uses to some pre-allocated space on disk. Each time a new restore point is created it's placed in this space, and when that space fills up, the oldest restore points are removed to create room.

 

The system restore points are located in System Volume Information folder and many other places which interlink these files. In vista volume shadow copy service have restore points specific to hard drives and partitions. So there is no way we can backup one specific restore point indefinitely.

 

One workaround would use complete PC backup utility or third party ghosting programs, which takes an image of the computer and you can use it to restore it back. Although this doesn’t have the features of system restore, this program can be an alternative than nothing.

 

Hope this helps.

sumit (independent advisor) say:
Hi,
Unfortunately, pro packs cannot be moved if the qualifying Windows license of Home was preinstalled with the computer.

richardeiler say:
  • If you upgraded from a retail copy of Windows 7, Windows 8 or 8.1, the Windows 10 license carries the retail rights from which it was derived – can be transferred. But under Microsoft’s rules, you are only entitled to a one time transfer.
  • If you upgraded from an OEM Windows 7, Windows 8 or 8.1 license, these are licenses that come preinstalled on a new computer from a manufacturer, and then your Windows 10 license maintains the OEM rights – cannot be transferred.
  • If you have a full retail copy of Windows 10, you can transfer it as many times as you want.
  • If you performed an Easy Upgrade to the Windows 10 Pro Pack from Windows 10 Home, you can transfer it using Digital Licensing. This is possible because of the Pro Pack, while an upgrade, is a retail license attached to the Microsoft Account used to purchase it.

Protect YOUR PC - CREATE A SYSTEM IMAGE" (SHADOW COPY). Win 7 - Win 10


Here is the best advice to any PC user.


 System Restore is not guaranteed for like if you have issues on your system. Some times it fails and even if successful, now some apps, programs or system files may be corrupt.


 In back up and restore, backing up files is a waste of time, unless there are a few you want to save to thumb drive, CD, DVD or separate hard drive. And you can just use your "COPY and PASTE" ability or burn to CD or DVD.


My advice is to anyone is to use back up and restore and "CREATE A SYSTEM IMAGE" (SHADOW COPY).


This function is not available in all Windows 7 versions. I believe only the Home Version of Ultimate has this feature.

I am not sure about Win 8.

It is available for Windows 10 Pro and I am not sure about other versions of Windows 10.


 You need to check the size of drive C:\

If it is, for example: 250GB drive, go purchase a Western Digital 250GB (SATA or IDE - Check which your one system uses) or external USB Hard Drive, all a very cheap cost. Or you can use a network location such as another computer on your Home Network if its hard drive has the space to cover the info on your Hard drive. It may only be 150GB used on your 250GB hard drive. So that computer or device will only need that much free space (152GB) to accept your Image/shadow copy.


I have never had a Western Digital hard drive fail. But had 4 Seagate hard drives fail.


Go to:
 Control Panel\All Control Panel Items\Backup and Restore (Windows 7)


**Which is on Windows 10 and is for Windows 10.**


First, create a Repair "DVD" and Label it and put it in a safe place.

Left side: Create a repair disk


 On Windows 7, I realized that you can not find your system Image in the Back-Up/Restore>>Manage Space location. And using repair DVD disk to restore image may not be located by Windows during this process.


 So to cure this issue for Windows 7, You need to back up only one little file "One Time" to get it turned on and working properly.


So do this:

Control Panel\All Control Panel Items\Backup and Restore (Windows 7).

Select - Back Up>>Choose browse and pick one small file to back up.

Select location to back up. (Not your system Drive).

Now close Backup and Restore (Windows 7)


Re-Open Backup and Restore (Windows 7)

Select - "MANAGE SPACE".

Select - "View Backups".

Select - the Back Up you created and DELETE IT.

Now close Backup and Restore (Windows 7)

Go to the Back Up location Drive or Network location and if it is still there (The Back Up Icon), then DELETE it.

Empty Recycle bin.

NOTE: You might get a pop up box for ADMIN Privileges upon deleting file from location, If so, then click on Continue to Delete it. Or when you empty Recycle Bin.


NOTE: Windows Back Up in Windows 7 (Not sure about Windows 8), when it backs up your files, it actually makes a copy of the files you choose to back up, so the Back Up is large.


NOTE: Windows 10 seems to just back up all your files using same back up procedure you did above, but on my 10 Pro v1709 (16299.125 Build), it does not allow you to choose your separate files to back up. It backs up all your personal files automatically. But the Back Up file is only a few hundred KB. Mine is only 189 KB (VERY SMALL). Basically just a script to retrieve from your current drive.


**So I guess if you do Back Up first on Windows 10 of your personal files, you can just keep it. But always delete the previous Back Up through the BU/R Program and follow exact steps above to completely delete it from back up location before you start a NEW back up. I do not trust overwriting a previous back up or Image.

If you want to delete it use same steps above.

OK, now for the most important thing you can and must do.


"CREATE A SYSTEM IMAGE" (SHADOW COPY).


No matter what happens on your PC with anything to do with the System files, personal files, apps and programs or even some hardware issues like bad sectors on hard drive causing issues and you can repair hard drive with the "Error Checking" located in "Local Disk (C:) properties, in the "Tool" tab.


The IMAGE will restore your computer back to the EXACT STATE it was in when you created the IMAGE/SHADOW COPY. I always delete and create a new IMAGE about every week or two or usually a couple days after windows updates or after I add devices or new programs. If all works great then that is the time to do it.


Creating an Image can take 10 minutes to about 2 hours, depending on your systems number of Processors, Processors Speed, Memory and the size of data on your hard drive. My System drive C:\ excluding all my other storage drives, is about 160GB data on a 250GB Drive. I have 6 cores, 3.2 GHz speed, 8GB Memory. It takes about 15-20 minutes to create an Image. Not very long for a total life saver and is very easy and very fast to do. Restoring your System with the IMAGE you created with the repair DVD takes about the same time.


Open Backup and Restore (Windows 7)

Select on left - "Create a System Image"

Select location to back up. (Not your system Drive).

Choose what to back up - (C:\ System) and (Windows Recovery Environment System).

Note: They could already be selected and disabled for a selection.

Then follow process and start the back up for the image.

Upon Successful completion, select NO for create a Repair disk if you already have one. You only need 1 for all your "SAME" Windows 7, 8 and 10 releases. **Not sure if x86 (32 bit) and x64 (64 bit) OS use same repair DVD.


If you have a 32 and 64 bit system, I would create a Repair Disk for each type of OS version (32 or 64) and for each Windows release, 7, 8, 10.

Open Backup and Restore (Windows 7)

Select - "MANAGE SPACE".

Select - "CHANGE SETTINGS".

Select - Delete all system Images for this computer, click OK and DELETE IT.

Now close Backup and Restore (Windows 7)

Go to the Back Up location Drive or Network location and DELETE the Folder named "WindowsImageBackup".

Empty Recycle bin.

NOTE: You might get a pop up box for ADMIN Privileges upon deleting file from location, If so, then click on Continue to Delete it. Or when you empty Recycle Bin.

Now do the same process you did above and Create a System Image.

Put Repair DVD into DVD Drive.

Restart or Start up computer. After BIOS Screen on Start Up, hit any key (A FEW TIMES) to load repair files when it tells you to.

It takes awhile to load the files. Be patient and relax if the screen is black and seems to be doing nothing because it is.

Once you get the Language Screen, Choose your language (US) for America.

Choose Repair/Restore System.

Then choose ADVANCED.

Then choose IMAGE/Shadow Copy.

It will then show Windows 10. Select that.

Then it will scan and give you one IMAGE (UNLESS, you created more on more back up locations. Select it and continue.

1. Now on Format box, you can select NEXT and verify to Restore Drive C: and start the process.

I do not believe it Formats the Hard Drive. Not Needed but recommended.

All other Drives this way are automatically EXCLUDED FROM THIS PROCESS BY DEFAULT.


OR

In that window they are ALL CHECKED TO BE EXCLUDED AND WILL NOT BE BOTHERED, (FORMATTED).

**Formatting CLEARS ALL DATA AND IS LOST.**

EXCEPT THE HARD DRIVE C:\ YOUR SYSTEM DRIVE TO BE RESTORED TO IS UN-CHECKED". You want it "Un- Checked" and formatted. Apply Setting and select NEXT and verify to Restore Drive C: and start the process.

NOTE: The drive with the back up image is already also excluded because it has your image on it to restore your system. So if you only have 2 Drives connected, Then only one drive will show (C:) and it should already be un-checked to be formatted.

Once completed Restart Computer or it will automatically after 45 Seconds I believe.

On Start Up, do not hit any keys or it will load the repair files again and you are already done.

It takes a bit on boot up to load everything. Be Patient.

When the Log-On Screen Appears, Remove the Repair DVD and store in a safe location.

Log on and all back to how it was from Image created.

You should get this Information from your computer while it is running or if you know the manufacturer and specific model number of your Hard Drive then that works too.
