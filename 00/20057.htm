Hi Gerry,

You can use PowerShell commands to add the public folder permissions.

Get-PublicFolderClientPermission "\My Public Folder"

This example retrieves the permissions for all users of \My Public Folder.

Add-PublicFolderClientPermission -Identity "\My Public Folder" -User Chris -AccessRights CreateItemsThis example adds permission for the user Chris to create items in the public folder My Public Folder.

dingoyip say:

OneNote App Page Size

Is it possible to change the page size on the OneNote app? (Not OneNote 2013/2016).

I want an A4 Size page so that I can easily print my notes for study without having to reformat my entire notebook page.

Thanks

kristine arp say:

Hi Dingo,

 

 

You can change the page size before printing the notes from OneNote. If you’re using the OneNote desktop app, follow the steps below:

 

  1. Go to the note you want to print.
  2. Click File.
  3. Select Print.
  4. Choose Print Preview.
  5. Under the Paper size, click the drop-down list and choose A4.
  6. Hit Print.

 

Take note that you can also choose the Print option instead of Print Preview, and then follow these steps:

 

  1. Click the Preference button.
  2. Click Advanced.
  3. Click the drop-down list beside Paper size and choose A4.
  4. Hit OK.
  5. Click OK.
  6. Hit Print.

 

If you’re using OneNote online, follow the instructions below:

 

  1. Go to the page that you want to print.
  2. Click Print.
  3. Click the Properties button.
  4. Click Advanced.
  5. Click the drop-down list beside Paper size and choose A4.
  6. Hit OK.
  7. Click OK.
  8. Hit OK again.

 

You can also set the page template so you don’t have to configure the page size before printing your notes. Follow the instructions provided on this link.

 

Let us know if you need additional assistance.

Change distribution list SMTP to user alias

We're looking to move SMTP addresses that are in multiple distribution lists into a user's mailbox as an alias. Some of the distribution lists have a single SMTP address and others have more than one.

Our environment is all on Exchange online.

sylvie liang msft say:

Hello KMRL84,

For your concerns:

1. How to move SMTP addresses that are in multiple distribution lists into a user's mailbox as an alias.

As far as I know, you can sign into Exchange admin center > recipients > groups, then select the related group and the remove the address one-by-one. Then add the address into the user’s mailbox as an alias.

I understand this may cost a lot of time to do this. However, I did a lot of researches but haven’t found any single PowerShell cmdlet that can easily achieve this. This may need some more scripts to complete this. Thus, I would recommend you submit your requirement in PowerShell forum to seek further assistance. Engineers and members there have more professional experiences about PowerShell to help you with this symptom. Your understanding is highly appreciated.

2. What happens to existing email that has been sent to recipients of the distribution lists and can this email be backed up?

Based on my tests, after you remove the SMTP address from a distribution list, the emails that has been sent to recipients will still exists in recipients’ mailboxes. Of course, those emails can be backed up by the recipients via Outlook client. To do this, just click File > Open & Export > Import/Export.

For more details, you can see the information here: Back up your email.

Note: Just in case, I recommend you back up emails first then remove the address to prevent data loss.

I hope this can help. Please feel free to let me know if you need any further assistance.

how to I remove contact from safe sender in order to block?

 

Origianl title: how to I remove contact from safe sender in order to block?

a. user say:

Hi,

 

Thank you for posting here in Microsoft Community and we apologize for the delayed response.

 

I'm happy to help you with your issue.

 

If you wish to remove an entry from your safe senders list, please follow these steps:

 

1. Sign into your account.

2. Click the gear icon located on the upper right corner of Outlook homepage.

3. Click More mail settings.

4. Under Preventing junk mail click Safe and blocked senders.

5. Click Safe senders.

6. Select the email address that you wish to be removed.

7. Click << Remove from list.

 

To add an email address to your blocked senders list, please follow these steps:

 

1. On the same page where you selected Safe senders, click Blocked senders this time.

2. On the current page enter the email address in the textbox.

3. Click Add to list >>.

 

Should you have any other questions or concerns, please let us know.

 

Thank you.

stephen boots say:

MSE will restore it from where it quarantined it. I don't believe that you will have the opportunity to specify the location.

Note that an infected message would have potentially dormant code, only active if you open the message or attachment. (An exception would be an infected script in an HTML message and an email program that automaticlaly displays and allows active content.)

Detections change all the time. In a previous scan, the definitions may not have included the specific footprint match that a later definition set had. Furthermore, the messages are inside of a message store, so in a different format than is saved to the PC or loaded into memory. When it was loaded into memory on the way through the transfer, apparently written to a temp file first, the infection was caught.
