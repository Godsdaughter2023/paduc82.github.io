initial impressions of RT 8.1

After a day or so with 8.1, a few things I have noticed:

  • In general, it does not seem to be a big change.  I guess the big changes are more noticeable in Windows 8.1 and especially on PCs with no touch screen.
  • The Mail app seems about the same.  It has the same issue that made me stop using it, namely that it can't send email to my Yahoo account.  The odd thing is that it worked for one outgoing email last night, but is back in the mode of being unable to send mail tonight.
  • Another odd thing about the Mail app -- while I was composing this, and tabbing around to look at various apps, I got a pop-up that said I needed to log into Yahoo.; Once I did that, the Mail app was able to send an outgoing mail.  Very odd after being unable to for several weeks.
  • So I use the Yahoo mail app.  By coincidence, the Yahoo mail app was changed a few days ago.  I have had some trouble with that as well. 
  • I was glad to see there is a Facebook app.  I have not yet explored it enough to know how well I like it.
  • I was frustrated that the revised Music app showed the list of my radio stations, but would not play them when I clicked on them.  Instead I had to type in the artist's name again.  The one I tried came back with a result that the app "could not start that radio station now." 
  • I like the tiled view in the new photo app.  I was glad to see the editing tools (although I have not tried them).
  • Store app is improved.
  • I like the ability to change start page tile sizes. 
  • I like the ability to set the SD card as the default location for storing music, photo, and/or video files.  I did not see this mentioned in the publicity stories about this release even though the inadequate integration of the SD card was a frequent complaint by users. I have not yet explored how well it is integrated, but it would be terrific if the SD card is fully integrated with the libraries and file structure and fully accessible to the apps!
  • I look forward to exploring the new and revised features.

How to open Windows Media Centre in Windows 7 - Microsoft Community

robbie69 say:

How to open Windows Media Centre in Windows 7

Original title: install media center

 

 

i have win 7 home premium (ver 6.1, build 7601, SP1) and only have media player.  from what i have read i should have media center.  if i do i have no idea where it is or how to install/start it.  how do i get it going?

jetta48 say:

Try:

 

1. Click on Start (bottom left) then Control Panel.

2. Click on Programs and Features.

3. On left hand side of screen, click on "Turn Windows features on or off".

4. Find the entry called Media Features & untick in the box next to it.

5. Click OK.

6. Reboot.

7. Click on Start (bottom left) then Control Panel.

8. Click on Programs and Features.

9. On left hand side of screen, click on "Turn Windows features on or off".

10. Find the entry called Media Features & tick in the box next to it.

11. Click OK.

12. Reboot.

 

If the above doesn't work try:

1. Open Windows Explorer. (not Internet Explorer)

2. In left pane navigate to:

C:\Windows\ehome

and locate the file (if its there) ehshell.exe

3. Right-click on ehshell.exe and Create a shortcut/Pin to TaskBar/Pin to Start Menu.

 

If none of the above works please confirm that you are NOT using Windows 7 Home Premium N edition. See:

http://windows.microsoft.com/en-GB/windows7/products/What-is-Windows-7-N-edition .

sathish s v say:

Hi Robbie,

-          What is the edition of Windows 7 installed on your computer?

Windows Media Centre is available only on Windows 7 Home Premium, Professional, Ultimate, and Enterprise editions.

If you have any one the above Editions installed then you can open Windows Media Centre by,

a.      Clicking on Start button.

b.      Select All programs, then you will find Windows Media Centre there.

If it is not available in All Programs, then I would suggest you to enable it from Windows Features.

Follow the steps mentioned below.

a.      Click on Start button, then type Windows Features on search box and hit enter.

b.      You will get Turn Windows Features on or off window.

c.       Expand Media Features option.

d.      Place a check mark next Media Centre.

e.      Click on OK

 

Refer the following link to setup Windows Media centre and start with Windows Media Centre.

Windows Media Center in Windows 7

Getting started with Windows Media Center in Windows 7
