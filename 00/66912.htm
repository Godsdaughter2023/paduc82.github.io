A possible way to fix Game DVR not working after Windows 10 Creators - Microsoft Community

gentlytech say:

A possible way to fix Game DVR not working after Windows 10 Creators Update.

For many users, it seemed that Xbox Game DVR began displaying that the user PC's current graphics driver was not supported when opening the Game Bar and hitting Record or Broadcast, when clearly before the Creator's update, the bar worked perfectly normal.

According to theories from some posts, the Windows Creators Update installed a graphics driver that did not match the current driver version for the computer, possibly corrupting the driver itself, or the registry. That means you have to reinstall the correct driver version to fix the problem.

For me, the solution was to go into Device Manager (which can be found by right-clicking the start button), and then going into display adapters. Then, I right clicked on the corresponding display driver (For me, Intel Graphics 520), went to Driver - Update driver, and then clicked on browse on this computer. Then, I hit let me choose from a list of drivers on my computer, installed the latest one in the list (don't click on Microsoft basic display adapter), and then hit install. After it finished installing, I rebooted my PC, and Game DVR works again! Hope this helps those users out there who are baffled by the problem.

a. user say:

To answer your question, yes. It is always recommended to do a clean install when going to a new hard drive regardless of which one you choose. Installing the operating system on the SSD is the way to go in my opinion. The caching will be so much faster, plus the read and write is many times faster than a traditional mechanical drive. The SSHD are only good for caching information to appear to run faster but in all reality your still using the mechanical part for storage. Put the OS on the SSD and use the HDD for storage and you will thank me later. The only bad thing about SSD is that when they fail, they fail without warning but as long as you have software to monitor, you should be fine. Like for instance Samsung magician will tell you how much data you have written to the SSD. The solid state drives are so well manufactured you probably will never have a problem with it. I hope that helps!
