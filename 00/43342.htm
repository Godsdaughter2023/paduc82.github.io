¡firedog say:
every email appears blank with 2 attachments in them: ATT00XXX.txt and ATT00XXX.htm where "XXX" will be a random or sequential number of some kind.


These are the symptoms of having too many Temporary Internet Files. In Control Panel, open Internet options and (I think - I haven't got XP/IE7 or 8 handy to check) click Settings under Browsing history. Note the figure for Disk space to use. Note also the Current location of your TIF folder, then navigate to that folder in Windows Explorer. Right-click on the folder name in the navigation pane and select Properties. Do you see numbers of files and size on disk? How does the Size on disk compare with the figure for Disk space to use that you noted earlier? How many files?

If you have more than, say, 20,000 files, it's too many. WLMail uses TIF to create temporary files so that messages can be displayed within the program. If there are too many files in the folder to start with, you'll see the symptoms you describe.

Probably the easiest way to proceed is to use the Move folder option on the Browsing history | Settings page you visited earlier. Select a new location in your Documents and Settings folder. After rebooting the computer, you should start with a fresh, empty TIF folder. If WLMail now works properly, you should be able to delete the old, overfull TIF folder.
