benny.boy say:

Groove: Can't sign in, 0x80070005

I got this error on Windows 10 Enterprise build 1511 in the Groove Music app:

"We can't get to the file for this item. Make sure another app isn't using it, that you can access the place the file is stored, and that your network's proxy settings are correct. Then try again.

0x80070005"

I searched for answers and tried many things (including removing and reinstalling Groove several times).  Eventually, I went to one of my home computers running Windows 10 Home (with the Anniversary update), where Groove Music is working, and copied the settings.dat file to my ailing laptop (where Groove gives this error) and restarted Groove.  Viola!  I am signed in again and able to access my OneDrive Music.

So, for those who've experienced (or are experiencing) this problem, it's not the most elegant solution, but it worked for me:

1. Go to a Windows 10 computer where you are signed into Groove.

2. Ensure the Groove app is closed.

3. Copy the settings.dat file (e-mail it, copy it over the network, or use a flash drive, etc) to the ailing computer.

Found here: %USERPROFILE%\AppData\Local\Packages\Microsoft.ZuneMusic_8wekyb3d8bbwe\Settings\settings.dat

4. Ensure the Groove app is closed on the ailing computer and backup the the settings.dat file (i.e. rename it).

5. Replace the settings.dat file on the ailing computer with the one from the good Win 10 computer.

6. Restart Groove.

It successfully signed in and loaded the music (and playlists) I own from OneDrive.

I hope this helps someone else!
