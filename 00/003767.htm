When I try to do a mail merge using the labels (5160 - the most used address label) the size is incorrect.  How do I correct it or use a template from the avery site?
jay freedman say:

In the Mailings > Labels dialog, click the Options button. In the Label Options dialog, click the Details button. Compare the measurements in the next dialog with the default values in this picture:

If anything is different, change it to the default values. When you click OK, the resulting label definition will be saved in the "Other/Custom" group (in the Label Vendors dropdown). Print a test sheet of these labels.

If the values in your dialog already match those in the default definition, the problem could be in your printer driver, which may need an update to support Windows 10.
