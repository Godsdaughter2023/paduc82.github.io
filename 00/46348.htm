obsessive power say:

I personally don't think it's your connection to LIVE either.  Can you try removing your hard drive and powering up your system.  Does it load straight to the dashboard now, or does it still take nearly a minute?  If it now loads up quicker, the issue is a failing hard drive.  If removing the hard drive makes no difference then your console is suffering from what can only be described as a slow hardware failure.

For problems opening the disc tray see this trouble-shooter: http://support.xbox.com/xbox-360/disc-drive/manually-eject-disc
For freezing console issues see this trouble-shooter: http://support.xbox.com/xbox-360/console-freeze/xbox-360-screen-freeze

Unfortunately there is little that can be done with the noise levels of your console.  The Xbox 360 has several fans built in to keep the system cool, however over time these fans may become louder.  The most common cause is a build up of dust inside the machine, which impairs airflow.  This in-turn causes the console's cooling fans to spin faster.  You can cut down on some noise by installing some of your games to the console hard drive.  This will stop the DVD drive spinning.  But noise caused by dust build up can only truly be rectified by having the console serviced by Microsoft.
