vijay a. verma say:

Enter the below formula as Array Formula i.e. not by pressing ENTER after entering your formula but by pressing CTRL+SHIFT+ENTER. This will put { } brackets around the formula which you can see in Formula Bar. If you edit again, you will have to do CTRL+SHIFT+ENTER again. Don't put { } manually.

Put this formula in A2 of second sheet. Drag this to B2...Drag down A2 and B2 for other values -

=IFERROR(INDEX(Sheet1!A$2:A$10, SMALL(IF(Sheet1!$B$2:$B$10-TODAY()<=30, ROW(Sheet1!$A$2:$A$10)-ROW(Sheet1!$A$2)+1), ROW(1:1))),"")

Edit - This assumes that row1 is header in your sheet1 and values are in columns A & B.

stefan blom say:

The View Gridllines command on the Table Tools Layout tab refers to the nonprinting dotted lines that you see for tables which do not have a printable border.

If you are trying to clear the actual border (visible in the Backstage view and in a printout), select the whole table and use the Borders command on the Table Tools Design tab. You can click the arrow on the button and then choose No Border from the dropdown list.

jllatham say:

Let's say that you have data, even if scattered around, that goes from A1 over and down to AY9400.

You could go over to AZ1 and select it and then press [Ctrl]+[Shft]+[End] and then use Delete with the entire columns option.

Then go down to A9401 and again press [Ctrl]+[Shft]+[End] and this time Delete with entire rows option.

Save the workbook. Close it.  Open it again and the extra rows/columns should be gone on the sheet.
