Hello Steve,

  • First things is first - try ordinary clearing personal information.
  1. log in supposinly Local account   
  2. enter Start screen   
  3. enter Settings   
  4. enter Tiles section
  5. press Clear personal info from my tiles button   
  6. return to Start screen and check e-mail tile > If it asks for e-mail address - exit, your account is Local.
If it does not help, try steps below.

  • Steps I'd suggest to you are rather advanced then ordinary, so try it on your own risk.
  1. enter personal administrative account   
  2. enter Start screen
  3. enter Settings
  4. enter Tiles section
  5. switch Show administrative tools to Yes
  6. back in Start screen press Computer Management tile to open MMC
  7. inside MMC select: Local Users and Groups > Users
  8. from Menu Bar select: Action > New User...
  9. inside opened dialog window enter unique name (option regarding password setting you may learn yourself by [?] and click) and press Create button.
  10. press Close button and check that name of new user appeared in list on the right side.
  11. newly created user should be Local and have ordinary User rights. To check this double click on user name and select Member Of tab, under which you should see Users and HomeUsers group names. (If you want to add administrative privileges Add... them there).
  12. WARNING! Inside Users section of MMC you'll see many other user names. This tool allow you removal of any/all user accounts permanently, be careful if doing so. By removal of certain accounts or misentered password, you may easily end up unable to manage or even enter your device.
  13. restart device, log in with newly created user, enter e-mail tile to see whether it asks e-mail address. If it is - exit, you're done.

Method 2: Uninstall and reinstall the Printer drivers.

1. Press Windows key + R, then type in devmgmt.msc.

2. In the Device Manager window, expand the Network adapters.

3. Uninstall your printer driver.

4. Reboot your computer and Windows will attempt to reinstall the driver.

5. If Windows doesn't find a new driver, you can try looking for one on the device manufacturer's website and follow their instructions.
