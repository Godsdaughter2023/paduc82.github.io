Hi,

This an ARRAY formula, see below for how to enter it.

=ROUNDUP(SUM(D12:D15*37,D30:D33*33,D36:D38*91,D40*13,D48:D52*33),1)

This is an array formula which must be entered by pressing CTRL+Shift+Enter

around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

    hi kathrin,

    users who have the “edit” permission will also be able to save the document within sharepoint. there isn't a specific permission that lets a user only edit and not save a document.

    as a workaround, you may set the permission level of the users that you would like to be able to save and edit to “edit” in the document library, and the users that you would not like to be able to save to “view”.

    this way, users that has the “edit” permission will be able to open and edit, as well as save the document straight away.

    users that have the “view” permission however, will only be able to view the document. if they wish to edit it, they will have to download the document to their computer first. if a user tries to edit the workbook on excel, they will encounter this prompt.

    Niyamath is getting to this but it is possibly an add-in realted issue.  If you start Outlook in Safe Mode and the error does not happen it is likely an add-in causing the error.

    To disable potential add-ins open Outlook, File, Options, Add-ins.  At the bottom of the dialog choose Manage: COM Add-ins and click Go.  Uncheck any that you think are causing the issue.
