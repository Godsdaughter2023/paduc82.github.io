bob jones aka: cybertaz say:

Word does not "automatically" change anything about the file

If the file is in the .docx file type you can open & save any changes you make.

If the file is saved in the deprecated .doc file type you still can open it & save changes but Word will operate in what is referred to as Compatibility Mode... not all features of the newer software can be used & some elements (primarily graphic objects) may be lost if the file is re-saved using that same file type. IOW, the older file type does not support some of what the current file type supports.

However, Word 2008 does support the .docx file type, which is the default file type used by both versions. Before getting started on your project I'd suggest converting the file to .docx if it isn't already in that file type.
