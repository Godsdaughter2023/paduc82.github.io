roberta a.. say: Internet and Hardware
If your Internet connection isn't sufficient, you may experience interruptions while using Skype. Skype support recommends shutting down downloads and other bandwidth-heavy programs to optimize your connection. If your webcam and microphone are connected to your computer via USB, the connections might be loose or absent. Check them before making any changes in your Skype configuration.

Other Issues
Certain ports need to be unrestricted for Skype to function properly. Ports are the channels your computer uses to connect to other programs and computers via the Internet. Firewalls and other computer programs often close or restrict certain ports to prevent unauthorized access. Close other programs to see whether it makes a different in Skype's performance. Older versions of Skype can also cause problems, so make sure you're upgraded to the latest version. Usually you can open ports through your firewall program, your router or modem settings. You can also configure which ports Skype uses by selecting the Advanced from the Options menu under Connection in the Tools menu of the main Skype interface.

The optimum setup would be to open all ports above 1024, but you can also open ports 80 and 443 if the first option isn't possible. If none of these ports are open, Skype won't work correctly. The Skype Help page (link in Resources) provides detailed information about opening Skype ports.



Skype Software
If the entire Skype program is shutting down and no basic fixes are working, uninstal and reinstall the program. If there was an error, a fresh install could take care of the problem. Sometimes Skype itself has server problems or software problems that keep users off the site. In May 2011, for example, Skype was down for an afternoon because of a technical glitch. Skype's website should be able to alert you if there's a problem. If the website is down, chances are Skype won't function correctly.

The executable for IE 64-bit is located here:

How do I merge an existing outlook date file into a new one? - Microsoft Community

bryanwyles say:

How do I merge an existing outlook date file into a new one?

I am upgrading my email account server from a POP3 to an IMAP account.  Outlook creates a new data file when creating this account and I need to merge all of my old oulook data (data file) into a new outlook data file.  I need the new outlook data file to be the default data file and merge all of the old data like tasks, calendars and contacts into the new data file.  help!
srinivasu guntimadugu say:

Hi Bryan,

 

Thank you for contacting Microsoft Office Community and We will be glad to assist you with your concern. By the description, I understand that you are migrating to IMAP account from POP3 account type and would like to merge or move the data to a single data file.

 

The POP account created an Outlook data file (POP) locally on the computer whereas IMAP account will create and Offline Data file (OST).
The actual data for the IMAP account will be stored on the respective server and synchronizes with the OST file in Outlook.

 

You will need to transfer the data from the PST file to the OST file in Outlook. When we perform send/receive, OST file connects to server and will upload the data and merged on the server.

1. Select a folder or a bunch of e-mails in the PST file.
2. Right click on the folder or e-mails -> Select Move -> Click on Copy to folder.
3. Select the destination (IMAP data file) in the Outlook navigation pane -> Click OK.

 

Note: The default folders such as Inbox, Sent Items, Deleted Item cannot be moved, only the e-mail in these folders can be moved. However any custom folders created can be moved directly.

 

The best way to copy the data and move to IMAP account is to select a bunch of e-mails or each folder at a time and move them. Once you have moved/copied, click on Send/Receive so that it upload the changes on the server.
Similarly select each folder or a bunch of e-mails and copy/move them and perform send/receive.

 

You may also refer to the following article on how to manage the Outlook data file:
http://support.microsoft.com/kb/287070

 

I hope the steps provided helps. If you need further assistance, reply and we will be happy to help you.
