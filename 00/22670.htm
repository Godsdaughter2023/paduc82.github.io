andrea jones - all about resources say:

The clipboard is just a 'holding area', it's showing the things you've collected, you're not 'pasting' into the clipboard.  Normally it's more useful for the clipboard to work in the way it does, most people want to paste the most recent thing they've collected, so that appears at the top of the list and the older items appear further down.  Your pasting should be controlled by where you put your cursor in your document.

 

It's not clear what you're trying to achieve, but I guess from what you're saying that you may find the 'spike' useful.  To collect items in the order you want them pasted, highlight the text then press Ctrl + F3.  Carry on highlighting and collecting items by pressing Ctrl + F3.  When you've finished, click where you want the items pasted and press Ctrl + Shift + F3, they will now be pasted in the order you collected them - this is very useful for rearranging paragraphs.  Note that this method 'cuts' the items from the page, it doesn't copy them.

You can do it if you don't mind using a helper cell, and assuming as per JLLatham's post that the lowest "unique" value is 5 in 1,2,3,4,5,4,3,2,1,6.

Assume values in A1:A20

Then in B1, use array formula (CTRL+SHIFT+ENTER to enter) =MIN(IF(COUNTIF(A1:A20, A1:A20)=1, A1:A20, ""))

Then for range A1:A20, set conditional formatting, Format only cells that contain; Cell value equal to $B$1.

jllatham say:

Please clarify "unique number" - I am assuming that means a number that only appears once in the list of 20 cells.  That is, in this list

1 2 3 4 5 4 3 2 1 6

5 and 6 are unique (while 1, 2, 3 and 4 each appear twice). And you want the 5 to be highlighted.

I'm not sure how to do it with a conditional formatting formula either, at least not with built-in Excel functions.  MIN() is going to give you the minimum of the whole list, whether the number is repeated or not (it would return 1 for my list even though it appears twice).  So it would appear that job 1 is to get a list of the unique numbers - and that would take, at a minimum, a second column.

If we assume your list is from A1 to A20, then in B1 you could put this formula:

=IF(COUNTIF(A$1:A$20,A1) = 1, A1, "")

and fill it down, only the unique numbers would be pulled into column B, then you could build a conditional formula on that based on the MIN() of entries in column B.  Column B could be out in another column, or hidden.

Your conditional formula would be, using a formula and with all A1:A20 selected

=A1=MIN($B$1:B$20)

and set up the formatting options.

Another option is to use the UDF below which determines the unique numbers in the list and finds the lowest value of those unique entries.  The name of the UDF is FindMinimumUnique(). To use it with conditional formatting you'd again select all 20 cells and set up conditional formatting based on a formula, with this as the formula:

=A1=findminimumunique()

Here is the code for the UDF.  You can change constant rangeToExamineAddress to match the actual address of your list of 20 entries.  To add the UDF to your workbook, follow the instructions for adding code to a regular module here:

http://www.contextures.com/xlvba01.html#Regular  (there's a how-to video just above that)

Be sure to save the workbook as a macro enabled workbook, type .xlsm or .xlsb.

Here is a link to a sample workbook with both solutions in it

http://www.mediafire.com/download/isjwpvqrgh6vauj/for_JC881_UDF.xlsm

Function FindMinimumUnique()
  Const rangeToExamineAddress = "$A$1:$A$20"
  Dim wholeRange As Range
  Dim anyEntry As Range
  Dim uniqueList As Object
  Dim uniqueKey As Variant
  Dim maxVal As Variant
  Dim minVal As Variant
 
  Set uniqueList = CreateObject("Scripting.Dictionary")
  Set wholeRange = Range(rangeToExamineAddress)
  maxVal = Application.WorksheetFunction.Max(wholeRange)
  For Each anyEntry In wholeRange
    If Application.WorksheetFunction.CountIf(wholeRange, anyEntry) = 1 Then
      If Not uniqueList.exists(anyEntry.Value) Then
        uniqueList.Add anyEntry.Value, anyEntry.Row
      End If
    End If
  Next
  minVal = maxVal
  For Each uniqueKey In uniqueList.keys
    If uniqueKey < minVal Then
      minVal = uniqueKey
    End If
  Next
  Set uniqueList = Nothing
  Set wholeRange = Nothing
  FindMinimumUnique = minVal
End Function

a.       Press the Windows + C to activate the charms menu and click on Search.

b.      Click on Settings and type location, select Allow apps to use my location.

c.       Turn on the option to Allow apps to use my location.

 

Step 2: Change the regional settings and check if the issue persists.

a.       Press the Windows + C to activate the charms menu and click on Search.

b.      Click on Settings and type location and select Change location.

c.       Change the ‘Home location’.
