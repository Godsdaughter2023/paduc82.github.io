daniel g. grau say:
You can create it in Word using section breaks (Insert > Break > Section Break (Next Page)). Then, each section can have its own page size and orientation. However, when converting to PDF, Word will create one PDF document for each page size/orientation:

  • If you have two page sizes, you'll get two documents.
  • If you have one portrait, one landscape, one portrait—three PDFs.
  • etc.

There is an easy solution: open your PDF documents in Applications > Preview and combine the pages into a single document. You do this by dragging the page icons from the Drawer on top of each other.
