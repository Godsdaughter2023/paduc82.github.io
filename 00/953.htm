Macro for Combining Multiple Word Files Together in Word 2010 - Microsoft Community

cherylkcmo say:

Macro for Combining Multiple Word Files Together in Word 2010

I am trying to combine about 300 word files together into a single file, all are .rtf files and all are numbered consecutively in the file names so they should come in in order.  I don't want to lose any of the formatting.  I am not very familiar with using visual basic to create macros and am confused about where to save the macro and make it work on these documents, and be able to use it again as needed.  The 300 documents I have came to me from another source.  Anyway, in Word 2003 we were given the instructions below from a person no longer with our company, we are now using Word 2010.  I have been able to get the Developer on the Ribbon and can see where the Visual Basic Editor is and create the macro using instructions below, but am unclear where to save it and call it for use again.  I haven't gotten it to work...  Thank you

(1) Open up a WORD session (Word 2003 Instructions)

(2) Go to Tools -->Macros-->Visual Basic Editor. This will open up a separate VB editor window.

(3) Choose Insert --> Module. A new module editor window will be opened.

(4) Copy and paste the content of attached TXT file above in the module window (also copied and pasted below).

(5) Modify the SEARCHDIR and OUTFILE accordingly where SEARCHDIR indicates where the files you want to combine are located and FILEOUT is for path and destination file name for the eventual combined document.

(6) Click the green arrow button immediately above the module window OR select RUN --> run sub/user form.

 

This macro works for RTF file as well and all you need to do is to change *.doc to *.rtf on line ".FileName = "*.doc"";

 

 

Macro From TXT file  

Sub concat_rtfs()

'

' This does all the work of concatenating .rtf files into A SINGLE .RTF FILE

 

    searchdir = "\\kcifile\PFZ\PFZ1057X-1057XX\Listings\Report\ESMO2008Analysis"

    FileOut = "\\kcifile\PFZ\PFZ1057X-1057XX\Listings\Report\ESMO2008Analysis\meeting_docs\BDR_2008ESMO_DISP_GC.doc"

 

  With Application.FileSearch

    .NewSearch

    .LookIn = searchdir

    .SearchSubFolders = False

    .FileName = "*.doc"

    If .Execute > 0 Then

      Documents.Open FileName:=.FoundFiles(1), ConfirmConversions:=False, _

        ReadOnly:=False, AddToRecentFiles:=False, PasswordDocument:="", _

        PasswordTemplate:="", Revert:=False, WritePasswordDocument:="", _

        WritePasswordTemplate:="", Format:=wdOpenFormatAuto

 

      For i = 2 To .FoundFiles.Count

        Selection.EndKey Unit:=wdStory

        Selection.InsertBreak Type:=wdSectionBreakNextPage

        Selection.InsertFile FileName:=.FoundFiles(i), Range:="", _

        ConfirmConversions:=False, Link:=False, Attachment:=False

      Next i

 

      ActiveDocument.SaveAs FileName:=FileOut

      ActiveDocument.Close SaveChanges:=wdDoNotSaveChanges

 

    End If

  End With

End Sub

Hi,

Currently, editing a phone number of a contact is not an available option in the new Skype version, as well as editing an existing phone number.

I'll take this as your suggestion and forward it to our developers to help with upcoming Skype updates.

In the meantime, as a workaround, you can delete the number you wish to edit and create a new one. To do this you have to dial and call the number you wish to save using the dialpad. You don't have to complete the call, simply drop the call after calling it, as this is for the purpose of saving the number. After dropping the call, right click on the number from the recent list and select view profile. From the profile window, you will see an option to add the number to your Skype.

To delete an existing phone number in your contact list, right click on the number and select view profile. From the profile window you will see an option to delete the number. This is if you're using the new Skype version 8.x.

If you are using the classic version of Skype, you will only see an option to delete a number that you added to a Skype contact but not the phone number that is originally stored on your Skype account.
