Well, the Print Preferences look right. Also, your first image of how the document displays on screen is how it should be. The only problem I see in the PDF is that the background graphic seems to be missing -- the brighter blue is to be expected as I mentioned before.

Some printers simply don't like to print Watermarks due to the Washout effect [assuming it was inserted as a Watermark]. See if you have a better result by going to Insert> Watermark to clear the Washout check. If necessary, you can go into the Header/Footer, select the Watermark image, then use the controls on the Format Picture tab of the Ribbon to adjust the intensity of the image. If that doesn't do the trick...

As you create a new PDF check all of the settings in the Print dialog to make sure that the highest quality is selected. Check all of the other Print Settings as well. It may also be a good idea to confirm that Office [14.3.8] & the printer driver (check the mfr's web site) are fully updated.

Another option that may help if need be: In Word go to File> Page Setup. Open the Format For: dropdown, select Any Printer, then generate the PDF.

The setting you asked about is only active if a PostScript printer is being used. It's normal for that check box to be unusable if a non-PostScript printer is selected. If you point to that setting a more complete explanation of its purpose will be displayed in the Description of preference box at the bottom of the Preferences window.
