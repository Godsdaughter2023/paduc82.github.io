Yes, when you restart numbering in a section, any section created following that will also restart numbering. You have to manually set that section to "Continue from previous section." Any subsequent sections will then pick up that setting. Link to Previous just repeats the content of the footer, including the PAGE field; the result displayed for the PAGE field is determined by the settings in the Page Number Format dialog.
