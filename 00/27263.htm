mvan---01 say: A couple of choices depending on how much work you want for yourself (please ensure you itunes lib is backed up before, messing around with any solution) and a couple of issues first,

- The N95 does not support any files with DRM (m4p), so purchased tracks from the itunes music store will not transfer/play. A workaround is to burn these to a CD then reimport them as aac etc.. (a messy process).The recent itunes plus files (m4a) are not supported even though these are DRM free and in theory should play on any device, the problem is that apple have modified the tag data. There are third party solutions to this but again lots of manual operations involved. Nokia for some reason actually provide a solution to this for OSX users (Nokia Media Transfer app) but think all Windows users use WMP!

- The N95 will support files from your itunes library that have no drm, these are the tracks imported into your library from your CDs.

To actually get them onto your phone,
1. Manually copy across the music from the itunes library using windows explorer (need to have your phone connected in data transfer mode).
2. Use Nokia Music Manager to scan your itunes library then use this to import the files. Again only drm free files can be copied. I have found some annoying behaviour with this as NMM appears to start creating tmp folders with duplicate tracks the second time the phone is connected!
3. Use a third party app called itunes agent, do a search on google, this integrates the itunes library with any memory card and allows you to define a playlist in itunes (ie call it N95) that you can setup to only copy non-drm tracks, then the app copies these tracks to your phone when connected in data transfer mode.

As you can see there is no seamless method to integrate itunes with a Nokia, I have the same issue, I consider itunes to a better music managment app than either NMM or WMP but to get these files onto the phone involves poor solutions.

I hope Nokia provide Windows users with a similar app to the one they provide for OSX users to give a simple and seamless way to transfer files from itunes.

The alternative is buy an iphone (seamless integration)........or stop using itunes! The Nokia Music Store launched in the UK as a beta might offer a longer term solution but fails on 2 counts for me, use of WMA & low bit rates....

when you send the service command that you want a reply on, get the selection number you want then resend the code with the selection number after the *.

If you need to backup things to feel safe and your Computer boots into the Desktop and you have storage available you can make handy use of a Task manager trick.

Right-click the blank area on the Taskbar, and then select Task Manager. If this does not work press CTRL + ALT + DEL to select Task Manager. Select File at the top, then Run new task. Select Browse... You now have access to some of the Explorer functionality. Select 'All files' from the drop down list above the buttons at the right. Do not use the buttons. Use the left pane to select folders and the right pane to select files, like in normal Windows Explorer. You can locate files you want to backup (in Windows.old etc.). Open Word. Navigate to File, click Open. You will see a list of all your recent documents.
At the bottom
Click Recover Unsaved Documents.
It will open a folder that contains all of your unsaved documents from the last 4 days.

Simply double-click your file to open it. Below the ribbon, you’ll see a message that This is a recovered file that is temporarily stored on your computer. Click Save As and choose somewhere on your system to permanently store the file. Once chosen, click Save.

If you can’t get to this step and are encountering problems opening the file, it might be that it’s become corrupted.
Click your file from the list and click the dropdown arrow next to Open. Here you can select Open and Repair, which will try to automatically fix any problems.

Alternatively, if you want to navigate to the unsaved documents folder manually, press Windows key + R to open Run. Input the following and press OK:
C:\Users\%USERNAME%\AppData\Local\Microsoft\Office\UnsavedFiles

Check how to set you history files
https://support.office.com/en-us/article/custom...


Also you have "AutoRecover" is an Office feature that is helpful for when Word is disrupted, like if your system crashes or you have a power outage. It’s not there to replace saving via the usual method.

To use AutoRecover, launch Word and you should see the Document Recovery panel. This will list all of the available files with their title and time stamp.

You can click each file in turn to see what it contains. If there’s any you want to save, click the dropdown arrow next to the file and click Save As… Here you can choose somewhere permanent to save your file.

Word may only offer the recovered documents on this initial launch, so don’t delay if there’s something you want to save.
