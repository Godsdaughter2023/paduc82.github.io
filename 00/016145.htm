Why would anyone ever buy a Windows Phone 7, since Microsoft has - Microsoft Community

jdmorgan say:

Why would anyone ever buy a Windows Phone 7, since Microsoft has eliminated the features that made Windows Mobile worthwhile?

I had 2 Windows Mobile phones in a row, one version 6.0 and one version 6.5.  I was very pleased with both of them, and I considered the most important features that made Windows phones better than the competitors (Apple and Blackberry) were:

 

1. It allowed full access to the file system, so that you can create your own folders and organise your files the way you want them.

2. It allowed you to connect the phone via USB to a Windows computer and copy files back and forth freely (making use of feature number 1.)

3. It shipped with Mobile versions of Microsoft Office written by Microsoft and compatible with many of the features of the desktop Office software, including the ability to exchange documents between the two platforms.  This feature requires making use of both features 1 and 2 above, since you need to be able to know where your documents are stored in order to use them, and you need to be able to transfer documents between platforms to make use of the compatibility features.

 

I used all 3 of these features extensively on both of the Windows phones I had before as well as using the PDF reader.  Since the Owner's Manuals for both of my phones were supplied in PDF format on a CD, I copied the manuals for the phones from my desktop computer to where they would do the most good, ON THE PHONE!
