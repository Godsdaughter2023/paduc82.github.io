f) Topic should be: system
8. Click OK
9. Click Close

(xlt) or workspace (xlw), select the appropriate type in the File Types
tab and repeat from step (5)

Try it this way

=IF(OR(F1>DATEVALUE("5/31/14"),G1>DATEVALUE("5/31/14")),"New","Not New")

If that doesn't seem to work properly, check to make sure the entries in F1 and G1 are true dates, a quick way to do that is to take any cell and put a formula like this in it:

=F1

and format that cell as General - should show a number, which is the numeric equivalent of the date.

You could do the same to F1/G1 directly, but then you'd have to change them back, by using the formulas all you have to do is delete the formulas.
