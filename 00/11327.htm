Hi Mike,

I'm a software engineer on the OneNote team.  I have a few things for you to try.

Second, if you indeed are not using Office 365, try editing the page that contains the image you're having trouble with.  It's not been publicized, but simply editing a page that contains an image which needs to be processed by OCR will cause it to be processed.  So by editing the page (just type something somewhere on the page), it will cause our indexing service to detect that there was a change and look to see if there are any images needing OCR.  Typically the images will be processed it about 10 minutes, but it can take as long as an hour depending on how much traffic we're seeing.

Third, if that doesn't work, try saving the image to your machine (right click on the image and select "Save As...").  Then create a new page and add the image to the page using Insert->Pictures on the ribbon toolbar.  This will hopefully give you a fresh start with the image on a new page which has not been processed yet.  Again, you may need to wait as long as an hour for the "copy text from image" data to show up (though it usually only takes about 10 minutes).

Hopefully this will help!
