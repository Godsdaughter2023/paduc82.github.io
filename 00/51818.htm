Is it possible to set an automatic computer shut down after the backups are finished?

While using the Windows 7 Home Premium backup at a scheduled date and time, is it possible to set an automatic computer shut down after the last file is backed up?
david.chase say:

Unfortunately windows do not provide any such feature but you do it by making a scheduled tasking the steps are as follows: 

1. Click Start, and type Task Scheduler  serch bar,press Enter.
2. Click Create basic task in the Action area. 
3. Enter a description for the task, and click next. 
4. Choose "When a specific event is logged" in the trigger screen, click next.
5. Select Microsoft-Windows-Backup/Operational in drop down menu, choose Backup as the Source.
6. Enter 4098 as the event ID, and click next. 
7. Select Start a program in the Action screen.
8. Input C:\Windows\System32\shutdown.exe in the blank before the Browse button.
9. Enter "/s" into the Arguments blank, click next.
10 Click Finish.

Now you can try to make a backup to test.

auggy say:

Enabling the Driver Verifier can sometimes (rarely) cause a BSOD at startup which may result in not being able to start Windows.

So I would recommend to back up critical files and create a System Restore Point beforehand.

Also, have recovery media or installation media available which one can use to access System Restore or a command prompt if unable to start Windows. 

For the DRIVER_CORRUPTED_EXPOOL error try the following:

Start > type verifier.exe then click on "verifier Run command" > select  Create custom settings (for code developers) > Next >  select Special pool  > Next > Select driver names from a list > next > then select all non-Microsoft drivers (click on Provider to soft alphabetically if you wish) > Finish

Restart the computer and wait for a BSOD error to occur and then provide the resulting minidump file.

To turn off the Driver Verifier:

Start > type verifier.exe then click on "verifier Run command" > select Delete existing setting > Finish

If needed the Driver Verifier can be launched from a Command Prompt at boot using recovery or installation media by typing the following command and pressing "Enter" on the keyboard:

verifier.exe
