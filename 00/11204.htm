sumproduct of two columns based on criteria in third column - Microsoft Community

kliddicoat say:

sumproduct of two columns based on criteria in third column

Hi,

 

I am trying to return the sumproduct of values in columns AJ and AQ on my data sheet if the values in column J on the data sheet match cell E3,

I have the following but am getting an error message

 

=SUMPRODUCT((data!J:J=E3)*(data!AJ:AJ)*(data!AQ:AQ))

joeu2004 say: kliddicoat wrote:

I am trying to return the sumproduct of values in columns AJ and AQ on my data sheet if the values in column J on the data sheet match cell E3, I have the following but am getting an error message
=SUMPRODUCT((data!J:J=E3)*(data!AJ:AJ)*(data!AQ:AQ))


Would have been nice if you provided us with the error message.  I am going to guess it is a #VALUE error.  The following might avoid that:

=SUMPRODUCT(--(data!J:J=E3),data!AJ:AJ,data!AQ:AQ)

The double-negative (--) converts the logic values TRUE and FALSE to 1 and 0, which SUMPRODUCT needs in order to behave as intended.

But it is "poor practice" to use column ranges like J:J.  In Excel 2007 and later, it causes SUMPRODUCT to process 1+ million rows, in this case creating an array of over 1+ million elements, too.

It would be better to specify limited ranges like J1:J100000.  It is very rare that we might actually have 1+ million rows of data.

john sr wilson say:

You can use a small macro to list the linked files and the slide they appear on.

Sub linkChex()
Dim osld As Slide
Dim oshp As Shape
Dim strMessage As String
On Error Resume Next
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
Err.Clear
Debug.Print oshp.LinkFormat.SourceFullName
If Err = 0 Then
strMessage = strMessage & "Slide: " & osld.SlideIndex & _
"  Shape: '" & oshp.Name & "' Is Linked" & vbCrLf
End If
Next
Next

If strMessage = "" Then strMessage = "No linked files found" MsgBox strMessage
End Sub
