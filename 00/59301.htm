Hi HabtomYohannes,

Please be advised that Cortana and default Windows app cannot be removed by normal means. What we can do, however, disable Cortana features. Here's how:

To stop Cortana from using your location:

  1. On a Windows 10 device, open Cortana’s home from the search box on the taskbar.
  2. Go to Cortana’s Settings, and select Permissions & History.
  3. Select Manage the information Cortana can access from this device.
  4. Turn off Location.

To stop Cortana from seeing data in your calendar, email, contacts, and text messages:

  1. On a Windows 10 device, open Cortana’s home from the search box on the taskbar.
  2. Go to Cortana’s Settings, then select Permissions & History.
  3. Select Manage the information Cortana can access from this device.
  4. Turn off Contacts, email, calendar & communication history.
  5. Go back.
  6. Select Manage the information Cortana can access from other services.
  7. Select the name of any Connected Service you use for calendar, email, and contacts (such as Outlook.com or Office 365), and then select Disconnect.

To stop Cortana from using browsing history:

  1. On a Windows 10 device, open Cortana’s home from the Search box on the taskbar.
  2. Go to Cortana’s Settings, then select Permissions & History.
  3. Select Manage the information Cortana can access from this device.
  4. Turn off Browsing history.

To stop Cortana from accessing your voice data you can choose to interact with Cortana with the keyboard. Or you can turn speech recognition off

  1. If you don’t want Cortana to use your voice data, you can still interact with her using your keyboard. To turn off speech recognition, see Speech, inking, typing, and privacy. If you’re signed in to Cortana with a Microsoft account, you can view and delete your voice recordings at account.microsoft.com/privacy.

To turn off Cortana’s personalization, to stop data collection, and to clear all of the data Cortana has associated with a particular device:

  1. Go to Cortana > Notebook > About me.
  2. Select your Microsoft account.
  3. Select Sign out.

To hide Cortana:

  1. Press and hold (or right-click) anywhere on the taskbar.
  2. Select Cortana.
  3. Select Hidden.

As for running the application, launching it as administrator might help. Right-click on the application and select Run as administrator. Another thing is running the compatibility troubleshooter, right-click on the application and select Troubleshoot compatibility.

Let us know if you have other concerns.

jwrightlady say:

ruck1710 say:

Well if you are experiencing slow computing experience, its a serious performance issue.

There may be many reasons for the computer's performance dropping day by day, the most common one are as follows :
- Inadequate System Memory or RAM,
- Too much of junk stuff and fragmented files in the computer
- Unwanted programs running in the background and consuming system resources
- Virus, Spyware and Malware activity
- Multiple Antivirus running in the system


I guess if your computer was running good in the beginning means you have fair amount of System Memory or RAM, however it is recommended increase it. Else, you can do away with increasing the Virtual memory or paging file.
Right click on My Computer go to Properties. In the System Properties window go to the Advanced tab.
Now click on the settings buttons in Performance. The Performance Options window opens up click on the Advanced tab again and you'll see at Virtual memory at the bottom. Click on Change. Now, you can keep it either in System Managed or Custom. If you are using the custom , then make sure the initial size would be 1.5x the actual physical memory and the maximum size will be 3x times.
Then apply and close alll the windows and restart the computer, Notice any change in performance.

To clean all the junk files simply use the Disk Clean Utility, it will clean all the unwanted junk files from the computer. Then you can go ahead and use the Disk Defragmenter to defragment all the fragmented files in the c: Drive.

Preventing the unwanted programs to run should be done carefully. Programs at times load up automatically whenever the computer is turned on these programs keep on running in the background and consume system resources. The best thing to do is to prevent them from loading automatically. Click on START and then click on RUN. The RUN window opens up, in the run window type msconfig, the System configuration window opens up. Click on the last tab that says Startup. You will see list and most of them will have a check. First uncheck all of them, then selectively check the important ones like your Antivirus, Internet connection tool, modem driver, any other important application that may be required. You might find it difficult but understand that the ones you will check should sound familiar to you. Apply and then restart the computer. when the computer restarts you will see a prompt that saya that You Have used System Configuration Utility. Simplu uncheck the box at the bottom and click on OK. Notice the change in System performance.

If you are running more thn one Antivirus, make sure that you uninstall one. Since, most antivirus has a real time scanner or resident shield which consumes some system resources, using many will cause more consumption of System resoures. eventually, slowing down the computer.

If all the above contributes little in improving the performance the Virus issue might have to be declared. However, let me know and we'll do something if its a virus issue.
