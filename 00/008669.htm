vasil l. michev say:

You can certainly adjust the recipient filter for the Dynamic distribution group to exclude one (or several, or all) shared mailbox. But you will have to use PowerShell to do so. For example:

Set-DynamicDistributionGroup DDGname -RecipientFilter {((Company -eq 'blabla') -and (Name -ne "sharedmailbox1"))}

Replace with the relevant attributes for your case.

Is there a way that I may combine the entries so that instead of having 2000-2001 and 2001-2002, I have 2000-2002 for each respective student?

    Also, the school has very little money, I am donating my time to help them acquire statistics for grants. Is it worth their while to upgrade to office 2010?

    Thank you
    hansv mvp say:

    Create a query based on the attendance table.

    Add the student ID and student name to the query grid, and add the "year" field thrice.

    Select View | Totals...

    Leave the Total option for the student ID and name set to Group By.

    Set the Total option for the first "year" field to Count, for the second one to Min and for the last one to Max.

    Save this query.

     

    Create a new query based on the one you just saved.

    Add only the CountOfSchoolYear field (the name will be different, of course).

    Select View | Totals.

    Change the Total option to Avg.

    This query will return the average number of years a student spends at the school.

     

    Create another new query based on the first one.

    Add the student ID and name fields.

    Add a calculated column

    AllYears: Left([MinOfSchoolYear],4) & "-" & Right([MaxOfSchoolYear],4)

    Again, the actual field names will be different, of course.

    suzanne s. barnhill say:

    Although, as Stefan says, many advanced features are not supported in Word Starter, fields certainly are. There are several ways to get Page X of Y.

    1. I believe Word Starter has an Insert tab. On that tab you should see a Header & Footer group with buttons for Header, Footer, and Page Number. All open galleries. The Page Number gallery does include a Page X of Y option, though it has bold numbers (you can select these in the text and unbold them if desired).
    2. Also on the Insert tab is the Text group, with a button for Quick Parts. On the Quick Parts menu is Field. In the Field dialog, you can select Page and NumPages fields for the X and Y.
    3. Perhaps the easiest way is direct composition. You can press Alt+Shift+P to insert a PAGE field anywhere in the document. To insert the NUMPAGES field, type NUMPAGES (or numpages--Word doesn't care about the capitalization), select it, and press Ctrl+F9 to convert it to a field.

    how do I tell a cell with an Average formula to ignore - Microsoft Community

    cooperregina say:

    how do I tell a cell with an Average formula to ignore #DIV/0!

    I have a spreadsheet with YTD data, including some cells with Sum and Average formulas.  How do I get a cell with an Average formula to ignore #DIV/0! ?
    jllatham say:

    Assuming you now have some formula like

    =AVERAGE(A1:A5)

    Then you can write it this way for Excel 2007/2010 to hide the error

    =IFERROR(AVERAGE(A1:A5),"")  or even  =IFERROR(AVERAGE(A1:A5),0)

    If it needs to be compatible with Excel 2003 and earlier, then use this instead:

    =IF(ISERR(AVERAGE(A1:A5)),0,AVERAGE(A1:A5))

    Just as above, you could use ,"", instead of ,0, in the formula, depending on what you want to present in the cell when the #DIV/0! error does rear its ugly head.
