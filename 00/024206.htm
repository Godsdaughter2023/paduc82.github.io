Create a reference to document properties such as last modified date - Microsoft Community

lucyndisguise say:

Create a reference to document properties such as last modified date and created by

I'm trying to create a reference in a cell to document properties such as last modified date and created by. Does anyone know how?
ron coderre say: If you want to display the Created_by and Last_Modified values in a worksheet,
you'll need to create a User Defined Function (UDF) using Visual Basic for Applicatons (VBA)

Here's one way:
• ALT+F11.....to open the Visual Basic Editor
• Right-click on your workbook's name in the Project-VBA Project window
...Select: Insert Module

• Copy the below code and paste it into that module:

Function ShowAuthorLastSaved()
    Dim vResult As Variant
    Dim strAuthor As String
    Dim vSavedDateTime As Variant
    Dim strProp As String
   
    strAuthor = "Unavailable"
    vSavedDateTime = "Unavailable"
   
    With ThisWorkbook
        On Error Resume Next
        strProp = "Author"
        vResult = ActiveWorkbook.BuiltinDocumentProperties(strProp)
        If Err.Number = 0 Then
            strAuthor = vResult
        End If
       
        On Error Resume Next
        strProp = "Last Save Time"
        vResult = ActiveWorkbook.BuiltinDocumentProperties(strProp)
        If Err.Number = 0 Then
            vSavedDateTime = vResult
        End If
    End With
    ShowAuthorLastSaved = "Created by: " & strAuthor & " - File last saved: " & vSavedDateTime
End Function

• Save your workbook

Use that function this way:
• In any cell enter:
=ShowAuthorLastSaved()

The cell will display something like: Author: LucyNDisguise - File last saved: 2/23/2010 1:50:50 PM

Is that something you can work with?

Locking a template - Microsoft Community

artanis5204 say:

Locking a template

Hi,

I've just created a letterhead template by inserting images and text into a blank doc, and then saving it as a template. How do I lock it, so that the template elements don't appear as editable in subsequent documents that use this template? Thanks!

bob jones aka: cybertaz say:

The problem is that you have no Section Breaks so there is only the default Section 1... That's why the Options button is inactive. Section 1 comprises the entire document which you can either Protect or not. You need to have a Section Break preceding each segment of the document in order to vary the protected attribute from one Section (segment) to another. Once the document has 2 or more Sections the Options button will be active.

See Word Help on the topic: Insert, delete, or change a section break

If the Header/Footer has content to be protected you need a Section Break (Continuous) at the very beginning of the document & include Section 1 as a Protected section. Your most likely choice throughout will be to use Section Break (Continuous) if there are segments of body content to accommodate, but in a Letterhead template that's not very common except for the Signature Block at the end. If you want to account for the possibility of a multi-page letter which has letterhead on the first page but not subsequent pages see this tutorial by MVP Suzanne Barnhill which includes those instructions. It's written using Windows Word as the model but everything translates rather easily:

“How to set up a letter template”—how to create a letterhead (Word 2003 and earlier), have different margins on the first page, set up letter styles, etc.

bob jones aka: cybertaz say:

There is no provision for 'locking' individual elements in a Word document -- template or otherwise.

In order to prevent changes to certain areas of a document or template, Section Breaks need to be created. When Tools> Protection - Forms is applied the Sections button is active in a document that consists of more than one Section. There you can specify which Sections are to be Protected... Nothing in a Protected Section can be modified, anything in an Unprotected Section is fair game.
