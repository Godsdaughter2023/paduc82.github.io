    Solution 1 : inspecter et nettoyer le disque

    Examinez attentivement la face inférieure du disque contenant un jeu, un film ou de la musique (le côté sans image ou étiquette). La présence de rayures excessives et de taches peut empêcher la console de lire le disque.

    Pour nettoyer le disque :

    1. Tenez le disque par les bords sans toucher les surfaces inférieure et supérieure.
    2. À l'aide d'un chiffon doux, propre et légèrement humide, essuyez légèrement en partant du centre du disque vers l'extérieur.
    3. Lorsque le disque est propre, chargez-le à nouveau dans la console et vérifiez s'il est lu.

    Si le disque n'est toujours pas lu, essayez la solution 2.

    Solution 2 : essayer d'autres disques

    Essayez d'autres disques pour déterminer si le problème se produit avec n'importe quel disque ou uniquement avec celui-ci.

    Assurez-vous que le type et le code de région du disque sont compatibles avec la console. Pour plus d'informations sur les formats et les régions des disques, consultez Régions, formats et disques pris en charge.

    Si le code de région du disque est compatible avec la console mais que le problème de lecture persiste, essayez la solution 3.

    Solution 3 : vider le cache système (disques de jeu uniquement)

    Si la console Xbox 360 ne peut pas lire plusieurs disques de jeu, essayez de vider le cache système.

    S'il vous est toujours impossible de lire un disque de jeu, essayez la solution 4.

    Solution 4 : retirer le périphérique de stockage (disques de jeu uniquement)

    Si l'erreur se produit uniquement avec des disques de jeu, essayez de retirer le disque dur temporairement pour voir si cela résout le problème. Si votre console Xbox 360 n'est pas dotée d'un disque dur, passez à la solution 5.

    Si cette solution résout le problème, remettez le disque dur en place et commandez un nouveau disque dur Xbox 360 auprès de notre centre de service en ligne Xbox (connectez-vous à l'aide de votre compte Microsoft). L'inscription de la console est requise pour remplacer un disque dur sous garantie.

    S'il vous est toujours impossible de lire des disques de jeu, essayez la solution 5.

    Solution 5 : déconnecter les cartes mémoire et les clés USB (disques de jeu uniquement)

    Si le problème se produit uniquement avec des disques de jeu, essayez la solution suivante :

    1. Mettez la console hors tension.
    2. Déconnectez les cartes mémoire ou clés USB connectées à votre console.
    3. Mettez votre console sous tension.
    4. Essayez de jouer à un jeu.

    Si cette solution résout le problème, copiez le contenu de la carte mémoire ou de la clé USB sur votre disque dur et utilisez-en une autre.

    Solution 6 : effectuer une demande de réparation

    Si les solutions ci-dessus n'ont pas résolu le problème, la console a besoin d'être réparée. Effectuez une demande de réparation.
