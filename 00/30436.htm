robertjwin say:

These sorts of issues are par for the course because MS has never supported moving the Users or user account folders to any other non-system partition. As Dave mentioned, only the location of My Documents, My Pictures etc. can be officially modified, and that feature too took years to be introduced.

There are lots of workarounds including using junction points and so on, but invariably these break and cause big messes with updates and especially upgrades.

Your best bet would be to reinstall the programs to the exact same locations on D: as before, overwriting their existing files. The only other option would be to root through any config files and the registry to set right all relevant paths to point to D: again, but you'll most likely just waste your time doing that with no guarantee of success.

Maybe one day MS will think about users with small SSDs or those owning devices like tablets, 2-in-1s etc. with small eMMC/flash storage, but I'm guessing they'll just wait till larger SSDs are cheap enough that this ceases to become a problem. Much easier and cheaper for them, no?

a. user say:

Windows 10 activation is free if you upgrade a computer that is currently running Windows 8.1 or Windows 7 SP1. If a computer is not running either of those two operating systems at the time it is being upgraded, you have to buy a Windows 10 license to activate Windows 10.

In your case, to avoid buying a Windows 10 license you should do a clean install of Windows 8; upgrade to Windows 8.1; and then you can upgrade to Windows 10.

Before you start this process:

1- make sure you backup your personal data (Documents, Pictures, etc.); and

2- make sure you have license information for software you will have to install.

After you are done it would be wise to learn how to make image backups of your operating system. Had you been doing this it would have been fast and easy to restore a backup of Windows 8.1 before it got accidentally corrupted.
