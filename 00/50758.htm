1. If the internal clock is losing time or has a completely incorrect date/time, you may need to change the CMOS battery.

2. If the time is just a little adrift, it can be corrected by clicking the time display in the Notification Area (bottom right of screen), click Change date and time settings > Internet Time Tab and follow the prompts.

3. Additionally, you can change the interval between time checks, click the Windows Orb (Start), type regedit, press Enter and in the left pane navigate to HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Services\W32Time\TimeProviders\NtpClient and double-click SpecialPollInterval in the right pane.

The default setting is decimal 604800 (1 week in seconds), change it accordingly. Don’t forget to click the ‘decimal’ radio button.

Lastly, if you’re concerned about making registry changes, create a system restore point first.

seithennin say: Why are you having to set this option in msconfig?
normally you should leave this alone and windows will set it to the maximum memory installed..
windows sets it to 0 (automatic), as this is the default setting.
there is no reason you need to set this option in msconfig.
you should leave the msconfig options set to "Normal Startup".
if it does not show all 8GB of your memory in the windows task manager, then it may be a bios issue.
I recommend checking to see if there is a bios update for your computer's motherboard bios, which may correct the issue.
also 8GB of memory would be "8192mb"  NOT "8910mb"... which may be another reason why it resets it to "0" as 8910 is more than 8GB of memory.
