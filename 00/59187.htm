咲くやこの蚱 say:

Hello.

Permission change of the key is not necessary.

As you know, the REG_SZ values have never been changed.

(Hilight and HilightText in HKEY_CURRENT_USER\Control Panel\Colors)

You should edit system default REG_DWORD values.

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\DefaultColors\Standard

Hilight and HilightText

They are eight digit hexadecimal numbers with 00-Blue-Green-Red format.

You have to match these values (in different notation) to achieve steady highlight color.

But please note that key values in HKEY_LOCAL_MACHINE affect all users on that machine.

To check the effect, lock screen (press Win + L) and come back.

System REG_DWORD values take effect after screen lock and restore (including PC sleep or hibernation).

User REG_SZ values take effect after normal sign out and sign in (including PC reboot).

Default highlight colors are different among Windows releases.

Earlier releases used light blue Hilight.

(a): REG_SZ = "51 153 255" (Red=51, Green=153, Blue=255)

(b): REG_DWORD = 00ff9933 (Blue=FF(255), Green=99(153), Red=33(51))

Newer releases use dark blue Hilight.

(c): REG_SZ = "0 120 215" (Red=0, Green=120, Blue=215)

(d): REG_DWORD = 00d77800 (Blue=D7(215), Green=78(120), Red=00(0))

For example, version 1607 uses (b) and (c); version 1703 and later use (c) and (d).
