Are you getting an error when you do this? You should be, if the user cancels. 

You can handle this with an Error Handler:

On Error GoTo ErrHandler

  DoCmd.OpenReport "rptSalesReceipt", acViewReport

ErrHandler:

  DoCmd.Close acReport, "rptSalesReceipt"

ErrHandler:

  If Err.Number = 2501 Then

    Msgbox Err.Number & ": " & Err.Description

  End If
