Well, "what Scott said".

Again it's just a matter of using the right tool for the job. Tables are designed for data storage. Forms are designed for data display and interaction. Forms have a rich set of tools, controls, programming opportunites; tables are much more limited in what they let you do.

Microsoft has (I think unwisely) started putting things like Lookup Fields and Subdatasheets and Multivalue Fields into tables, blurring the distinction; but still, Forms are much more capable and flexible. No, it would not. The command line you're showing would connect to the mdw file, and then prompt for a u/pw.
