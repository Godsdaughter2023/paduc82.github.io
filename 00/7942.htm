By default, AutoCorrect uses a standard list of typical misspellings and symbols, but you can modify the entries in this list. Kindly follow these steps on how to change an entry on the AutoCorrect list:

  1. Go to the AutoCorrect tab.
  2. In the Replace box, type ' : -'.
  3. In the With box, type ' :- '.
  4. Click Add.

To change the replacement text for an entry, select it in the list of entries and type a new entry in the With box.

To delete an entry, select it in the list and click Delete.

Note: The Exceptions list tells the spellchecker not to autoreplace items it thinks are misspelled. Daniel have you actually tried going to Format cells? If I am not mistaken, if you click on Custom, there is a setting for Phone Numbers, zip and, zip+4.

If  one doesn't suit you can (at least you could in 2011) customize  by typing in the customize field:  (###) - ### - #### or 1 (###) - ### - #### and it will be added to custom List. Then if this format is selected your number will show up as formatted. 

The Procedure is:

  1. Click Column for "Phone:" at very top (selects entire Column).

  2. Go to Format menu.

  3. Click on Cells.

  4. Click on custom.

  5. Look for and select Phone number. Should be in format: 
    ### - ### - #### or
    (###) - ### - ####

  6. If second type is not shown in the customize field should already show:
    ### - ### - ####

    simply put  () around the First Group of character or add a 1 - then the () around the first three characters.
    Save as a custom format.

  7. Now type in you numbers, in row cell that corresponds to this formatting, example:

    Typing 5555551212 should now show as either (555) - 555 -1212 or as
    1 (555) - 555 - 1212
