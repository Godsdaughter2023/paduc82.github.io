Sub DeleteRows_SaveAs_xlsx()
Const sCr$ = "Y" '<< critera
Const sCol$ = "A" '<< search in column A
Dim wb1 As Workbook, wb2 As Workbook
Set wb1 = ThisWorkbook
Dim sFName As String
sFName = wb1.FullName
v = Array("Sheet1", "Sheet2") '<< sht names
Dim L As Long, x As Long
Application.ScreenUpdating = False
Application.DisplayAlerts = False
For x = 0 To UBound(v)
Sheets(v(x)).AutoFilterMode = False
L = Sheets(v(x)).Cells(Rows.Count, sCol).End(xlUp).Row
If WorksheetFunction.CountIf(Sheets(v(x)).Cells(1, sCol).Resize(L), sCr) > 0 Then
Sheets(v(x)).Range(sCol & "1").Resize(L).AutoFilter Field:=1, Criteria1:=sCr
Sheets(v(x)).Range(sCol & "2:" & sCol & L).SpecialCells(xlCellTypeVisible).EntireRow.Delete
Sheets(v(x)).AutoFilterMode = False
End If
Next
ActiveWorkbook.SaveAs Filename:="c:\users\username\desktop\new wb", FileFormat:=xlOpenXMLWorkbook
Set wb2 = ActiveWorkbook
Set wb1 = Workbooks.Open(sFName)
wb2.Close False
Application.DisplayAlerts = True
Application.ScreenUpdating = True
End Sub

Changing the default music player for “Hey Cortana play - Microsoft Community

jason d [msft] say:

Changing the default music player for “Hey Cortana play music”

In build 9926, by default Cortana will play music in the Windows Media Player, rather than the player selected in the Defaults settings page.
The first time you use this command to start playing music you will be presented with a notification to change the default player.  If you miss this notification, there is no way to see it again.  To work around this issue, you can do the following:

  1. Launch Windows Media player.

  2. Click Create playlist, enter a name for the playlist, and hit enter.

  3. Under the Playlists section, right click the name of the playlist you just created and then select “Open file location.”

  4. In file explorer, right click the playlist file and select “Open with” and then select “Choose default program…”

  5. Make sure the “Use this program for all .wpl files” is checked, and then select the program you want the “Hey Cortana play music” command to use. (The selected program must support .wpl files)

bob jones aka: cybertaz say:

You're correct that there is no Preference for disabling the tips. FWIW, there are a few ways to avoid them which involve a small amount of "retraining" :-) In general they all involve not positioning the pointer directly on the names, themselves... The location of the tip is relative to the tip of the arrow pointer.

  • If you place the pointer in the gray space on the left/right of the Style names (outside of the box surrounding the name) the tips don't display.
  • If you prefer to point to the rows the style names occupy, place it to the right of the names so that the tips don't display directly on the names on the adjacent rows.
