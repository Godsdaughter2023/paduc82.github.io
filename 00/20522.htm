yoga yao msft say:

Hi Albert,

Based on your description, you have questions about the modify date of files in local and OneDrive for Business. I would like to share with you the information below:
1. The modify date in our personal site and the modify date in the synced folder are not completely the same. A good example is that when we sync personal site at the first time, the local synced folder's modify date is when the folder created. It can be different from the cloud folder's modify date. All these new created folders will have the same local modify date.
2. They are both for the modify date of the files, but one is the date when cloud files are changed and the other is the date when the synced local files are changed. We can say not the modify date has been directly synced but the sync makes the files change. However, for the files which are not in folders, the modify date for cloud and local always keeps the same.
3. The situation is a little different for the folder and the files in the folder. Because not all the changes after syncing can make the modify date change. For example, if you change a file's name in the cloud folder, the name change will be synced to the local file, however, the local modify date for the file may be not changed during the sync.

To conclude, for the folder and the files in the folder, the modify date can be different. The most important thing is that there should be no sync issue, such as files can't be synced or the content change in a file can't be synced.

j.e. mcgimpsey say: An External Data Range (i.e., a query table) is a range that is defined in a name that specifies where data from an import or  query is placed. Since the range is or was linked to external data, XL won't let you break up that import range. Check your defined names (Insert/Names/Define) - you've probably got something named "Untitled" defined - just delete it.

As far as copying, you shouldn't need to format anything - copy the range and use Edit/Paste Special/Values to retain the formatting already set.
