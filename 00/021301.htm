MS, to the best of my knowledge, never blocked an AV in a OS beta, but I've only been doing this since Win95. a. user say:

Steve,
Re:  change formula depending on cell color

Can't be done in Excel using formulas.  You can determine if a cell has a color using the "Cell" function.  But that only tells you if it is colored and it won't work if the cell color is from conditional formatting.

ColorIndex can only be used with VBA code and in xl2007 and later versions it is problematical.

If you have some logic that determines what color a cell is, then you could use the same in your formula.  For instance, If a cell is colored Yellow if it has a value > 5, then use that same condition in your formulas.

In VBA, for xl2010 and later versions, the following code inserts the color number into the active cell.

 Yellow (vbYellow) is 65535, Green (vbGreen) is 65280, Red (vbRed) is 255
'---
Sub WhatColor()
  Dim N As Long
  N = ActiveCell.DisplayFormat.Interior.Color
  ActiveCell.Value = N
End Sub
