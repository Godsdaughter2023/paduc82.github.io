Hi Charlie,

Try to reinstall Windows 10 apps by following the steps below:

Reinstall preinstalled apps via PowerShell

  1. Open Windows PowerShell as admin. To do so, type windows powershell in the Start/Cortana, right-click  on Windows PowerShell, and click Run as administrator.

  2. In the PowerShell prompt, type the following command and press Enter key:

    Get-Appxpackage –Allusers

  3. Scroll down and locate the entry of the Store app and copy the package name. If you want to reinstall any other app, locate its entry and copy its PackageFullName. Note: Use the shortcut Ctrl+C to copy, as right-click won't work.

  4. Finally, execute the following command:

    Add-AppxPackage -register “C:\Program Files\WindowsApps\<PackageFullName>” –DisableDevelopmentMode

    In the above command, replace PackageFullName with the package name of the Windows Store or any other app that you copied in Step 3 and replace "C" with the drive letter of the drive where Windows 10 is installed.

Reinstalling all apps via PowerShell

  1. Open Windows PowerShell as administrator.

  2.  Make sure to close running apps, if any.

  3. In the PowerShell, copy and paste the following command and press Enter key  to execute it:

    Get-AppxPackage -AllUsers| Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

    This may take some time. Ignore all errors if any and wait for the command to complete its job.

  4. Once done, open the Start menu and search for the app you wanted to reinstall. It should be there.

If the same issue is encountered, ru-run the Windows apps troubleshooter. If the issue persists, try creating a new local user profile and check if the same issue occurs.

You have to ensure that you've got all the moisture out of the phone.

Remove the battery and wipe it to make sure it's completely dry.
