hansv mvp say:

If you want to return values from multiple columns in the same row: let's say that you want to return values from columns 2 through 5 of the lookup range.

Select 4 cells next to each other, and enter an array formula (confirm with Ctrl+Shift+Enter) like this:

=VLOOKUP(A1,P1:T100,COLUMN($B:$E),FALSE)

where A1 contains the lookup value and P1:T100 is the lookup range.

Option Explicit

Sub Test()
  Dim R As Range
  Set R = FindAll(Cells, 1)
  If R Is Nothing Then
    MsgBox "No cells with 1 found"
  Else
    R.Interior.ColorIndex = 3
    MsgBox "All cells with 1 are now red"
  End If
End Sub

Function FindAll(ByVal Where As Range, ByVal What, _
    Optional ByVal After As Variant, _
    Optional ByVal LookIn As XlFindLookIn = xlValues, _
    Optional ByVal LookAt As XlLookAt = xlWhole, _
    Optional ByVal SearchOrder As XlSearchOrder = xlByRows, _
    Optional ByVal SearchDirection As XlSearchDirection = xlNext, _
    Optional MatchCase As Boolean = False) As Range
  'Find all occurances of What in Where
  Dim FirstAddress As String, C As Range
  If Where Is Nothing Then Exit Function
  Set C = Where.Find(What, After, LookIn, LookAt, SearchOrder, SearchDirection, MatchCase)
  If Not C Is Nothing Then
    FirstAddress = C.Address
    Do
      If FindAll Is Nothing Then
        Set FindAll = C
      Else
        Set FindAll = Union(FindAll, C)
      End If
      If SearchDirection = xlNext Then
        Set C = Where.FindNext(C)
      Else
        Set C = Where.FindPrevious(C)
      End If
    Loop Until FirstAddress = C.Address
  End If
End Function

This PUP is also bundled within the custom installer on many download sites (e.g., CNet, Brothersoft, Softonic, etc..), so if you have downloaded a software from these websites, chances are that Hadsruda!bit (and probably some additional adware or PUP) was installed during the software setup process.

You should always pay attention when installing software because often, a software installer includes optional installs, such as this PUP. Be very careful what you agree to install.

To help with WD's remediation for this and any other PUP that might have been installed there with or without your knowledge or consent, I would suggest to try the following:
