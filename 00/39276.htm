To resolve this issue let us try the following steps:

 

  1. Press Windows Key + R to open Run window.

  2. Type wsreset.exe and press OK.

  3. This will clear the Windows Store cache and the Windows Store would open automatically. Check if you are able to search in the store.

 

If the above does not work, then try the following step:

 

  1. Open Cortana and type in the search keyword: Troubleshooting.

  2. On the search results click on Troubleshooting at the top to open the Control Panel.

  3. Click on View All in the left hand panel.

  4. Click on Windows Store Apps.

  5. Follow the on-screens steps to reset the store apps.

 

Hope the above steps help to resolve the issue.
