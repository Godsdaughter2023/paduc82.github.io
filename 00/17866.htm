jay freedman say:

It won't work in any version of Word. Any object (multipage TIF, PDF, even a Word document) will display only the first page and no more. The only workaround is to use a graphics program to save each page of the TIF as a separate file, and then insert each of the three individually.

Cross reference fields do not update automatically when the information they reference is moved. You will have to update the cross references to re-address them.
