Eliminating Account Unknown {S-1-5...} SID's - Microsoft Community

jjex say:

Eliminating Account Unknown {S-1-5...} SID's

This is not a question, but hopefully a solution to many of the posts on this forum. Only recently, did I discover that there were orphaned SID's on my system. I call them orphaned because they don't point to any existing user account.

I searched for answers on the Internet, including this Forum, and found mixed results. Most of the information was contradictory, vague or just plain wrong. Anyway, I discovered in my case that these objects had several things in common:

     - All objects are Shortcuts

     - All were created by the Applications they point to

     - All have a Special Permission which is to allow "DELETE" only

I have a Dell desktop upgraded to Windows 7 from Vista pre-installed by the manufacturer. Some desktop icons for Dell had the same problem, so I'm not sure that Windows 7 is the problem. I ran into the same 'Access Denied' error trying to delete them from the Properties/Security Tab. Then, I discovered that selecting Advanced under the Security Tab listed where the permissions are Inherited from.

In all cases, the Inherited Permission came from a Folder.

For Desktop Icons:     C:\USERS\PUBLIC\PUBLIC DESKTOP\

For Start Menu Programs:     C:\PROGRAMDATA\MICROSOFT\WINDOWS\START MENU\

Both of these locations have Hidden/System file Attributes, so you need to change Windows Explorer options to view files before you can make changes.

     Select -> Start -> Computer (drop down Organize Menu)

     Select -> Folder and Search Options

     Select -> View Tab

     Under Advanced Settings:

     Select -> Show Hidden Files

     Uncheck -> Hide Protected System Files

     Reply 'YES" to Warning

     Select -> APPLY

     Select -> OK

     Close Windows Explorer

Now, re-open Windows Explorer and navigate to each of the folders mentioned above. Right Click the Folder and select Properties.

     Select -> Security Tab

     Select -> Edit

     Highlight each Unknown Account entry and Select -> Remove

     Select -> OK Twice

That's it, your done exit Windows Explorer. Go back and check the original locations where the Unknown SID's were to see if they were removed. If all is OK don't forget to reset the View options in Windows Explorer back to they way they were.
