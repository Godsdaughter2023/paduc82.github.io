Why remove old version of MS Office Mac? - Microsoft Community

stevehutcherson say:

Why remove old version of MS Office Mac?

Instructions indicate that older versions of MS Office Mac before installing Office Mac 16? 
rohn007 say:
MS always recommends removing the older version.

There are 2 reasons. One is simple business, they want to encourage everyone to immediately buy the newest version and use it exclusively (MONEY MONEY MONEY!). That ignores the simple reality that many people have to work with others who have not yet upgraded. Using the same version helps avoid corruption issues (as small as that possibility is). The other reason is technical. Large programs like Office applications are not a single massive "chunks" of code. They are a collection of many smaller, separate sub programs. Some of these sub programs are shared, are used by other programs. When you install a newer version it may replace shared code that the older one uses with a newer version that will cause problems with the old program tries to use it.

I don't know if this applies on Mac, but on the PC side, Office is written to ASSUME it is always the only version installed. When you start any Office program it checks Windows to see if that Office bundle, ALL of the programs, are defined as Windows defaults. If it is not, then it triggers a "configuration" process to make it defualt. If you have 2 versions and keep switching between applications in the 2 versions you will constantly see this configuration process. It is annoying. Back in the 1980's MS developers created a "registry hack' to disable this check. It still works today.

Enter the below formulas as Array Formulas i.e. not by pressing ENTER after entering your formula but by pressing CTRL+SHIFT+ENTER. This will put { } brackets around the formula which you can see in Formula Bar. If you edit again, you will have to do CTRL+SHIFT+ENTER again. Don't put { } manually.

Drag down for other Start Depths.

In this End Depth and Depth Interval in column L and M become redundant.

Use following for +ve Slope

=SLOPE(IF(IF($C$2:$C$100>0,$C$2:$C$100,0)*($A$2:$A$100>=$K5)*($A$2:$A$100<1+$M$2)=0,"",IF($C$2:$C$100>0,$C$2:$C$100,0)*($A$2:$A$100>=$K5)*($A$2:$A$100<1+$M$2)),IF(IF($B$2:$B$100>0,$B$2:$B$100,0)*($A$2:$A$100>=$K5)*($A$2:$A$100<1+$M$2)=0,"0",IF($B$2:$B$100>0,$B$2:$B$100,0)*($A$2:$A$100>=$K5)*($A$2:$A$100<1+$M$2)))

=SLOPE(IF(IF($C$2:$C$100<0,$C$2:$C$100,0)*($A$2:$A$100>=$K5)*($A$2:$A$100<1+$M$2)=0,"",IF($C$2:$C$100<0,$C$2:$C$100,0)*($A$2:$A$100>=$K5)*($A$2:$A$100<1+$M$2)),IF(IF($B$2:$B$100<0,$B$2:$B$100,0)*($A$2:$A$100>=$K5)*($A$2:$A$100<1+$M$2)=0,"0",IF($B$2:$B$100<0,$B$2:$B$100,0)*($A$2:$A$100>=$K5)*($A$2:$A$100<1+$M$2)))
