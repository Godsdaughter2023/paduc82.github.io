graham mayor say:
With respect to headers and footers Word 2010 has the same functionality as the earlier version.

There are potentially three header/footer ranges in each section of a document. First Page, Primary (odd pages) and Even Pages. Each is separate from the others.

You can set the type of headers in the document from Page Layout > Page Setup (arrow bottom right of the group) > Layout.

You will need a document with three pages to access all the header/footers in order to set it up. Then from the Insert Menu > Header drop down. Edit Header, you will have access to the context sensitive header/footer Design tab.

If you are using all three header/footer ranges and want the same values in all the footers then you will have to copy the content into each of the three footers.

Once you have sorted out the headers and footers, delete the temporary page breaks and save the results as a template. Create new documents from that template.
