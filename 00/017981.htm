jason (office engineer) say:

Hi Francis,

As you have noted, the browser find feature will not be able to find all instances of text in Sway.  The reason for this is that for performance reasons, we don't actually download the entire Sway all at once onto your machine and only grab content as you go.  One way to work around this is to view the Sway in Accessibility view (from the … menu).  In this view, there is a Sway Find feature (available on the … menu or via Ctrl+f) that will allow you to find the content.  This find will not work for content embedded in the Sway.

Hope this helps!

roxanaanderson say:

invoice template with a detachable payment stub at the bottom of the page

Hello,

I want to create an invoice template in Word with a detachable payment stub at the bottom of the page.

Due to the constraints of the perforated paper on which the invoice will be printed, the payment stub has to stay fixed at the bottom of the page.

My current challenge is that I also have a table with products' description, quantity and price which pushes the payment section to the next page. Is there a way to make the table split and jump over to the next page but skip over the payment section without pushing it to the subsequent page?

Thank you.


john korchok say:

That can work as long as your document is set up for a different first page, otherwise the table would repeat on every subsequent page because it's in the footer.

To set a document up for a different first page, click on the Layout tab, then click on the down-pointing arrowhead to the right of Margins, then choose Custom Margins. In the dialog box that appears, click on the Layout tab. Check Different first page and OK out. Now the footer with table will appear only on the first page unless you insert a section break.
