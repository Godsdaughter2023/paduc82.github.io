e.j. gun say:

I assume you are getting the message about having values <= 0 on a log scale?  The interesting thing is, you only get that message the first time you change the scale to logarithmic.  If you change it back to normal and then back to logarithmic again, you no longer get the message.

I have not found a way around this, other than to ensure that my data does not contain non-positive values in it when I want to use a log scale.  You can use NA() in cases where your values become negative, and Excel will simply ignore those values.

For example:

=IF(A13>0,A13,NA())

would result in "#NA" in the cell with the formula if A13 is <= 0.
