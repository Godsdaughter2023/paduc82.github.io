Considering that, plus the fact that SFC /scannnow is completely automatic and very easy to use, the tool should usually be one of your first troubleshooting steps.

a)     Click Start, click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator. If you are prompted for an administrator password or for a confirmation, type the password, or click Allow

b)     Type the following command, and then press ENTER: sfc /scannow

c)      [The sfc /scannow command scans all protected system files and replaces incorrect versions with correct Microsoft versions.]A message will appear stating that 'The system scan will begin'. Be patient because the scan may take some time

d)     If any files require a replace SFC will replace them. You may be asked to insert your Win7 DVD for this process to continue

e)     If everything is okay you should, after the scan, see the following message "Windows resource protection did not find any integrity violations"

A few things.

First, a "system image" would not be the right thing to create in Windows 7, because a system image is a complete and total backup -- including the operating system.  So if you were ever to apply that saved image, you'd be 100% right back to where you were when you created it (so you'd be back to Windows 7).  But it's still a good idea to create one so you can if you need to.

You don't need to get Windows 8 first and then update to 8.1  You can just go directly from 7 ► 8.1.  Just put in your Windows 8.1 CD or run the installer, and choose to "keep everything" when it asks.

While this is a direct simple upgrade (like going from XP ► Vista or Vista ► 7), you should still start by making a full backup of all your files, writing down or saving all your product key serial numbers, and stuff. You should always be prepared before undertaking an OS upgrade.
