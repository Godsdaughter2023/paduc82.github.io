Private Function dateGregorian(sDate As String) As String
    Dim vVal As Variant

        On Error GoTo XIT

        VBA.Calendar = vbCalGreg
        dateGregorian = dtHijiri
    End If
    Exit Function

    dateGregorian = vbNullString

Hi DanielGerber1977,

Thanks for your post. According to your description, you want to preset an email with recipient’s address, subject and body content in Outlook client. In this way, you can compose the new message and then click the Save button in the upper left corner of the window. After that, this email will be saved in Drafts folder and you can continue editing it, discard it or send it.

Besides, you can try to create a Outlook Template in Outlook 2016 client by following the steps below:

1. Click New Email in the Outlook client.
2. In the New Message window, compose the new message with the recipient’s address, subject and body content.
3. Go to File > click Save As.
4. Type a name for the new template in the text box next to File name:.
5. Click the drop down menu of Save as type: and select Outlook Template.
6. Click Save button and close the current composing email message window.

The template will be automatically saved to the Outlook Template folder after you select Outlook Template in the Save as type drop menu. After creating the template, you can apply it when you want to create a new message with the same content in the template by following steps in Outlook 2016 client:

1. Click New Items > More Items > Choose Form….
2. Click the drop down menu of Look In: and select User Templates in File System.
3. Select and highlight the template you want in the list below.
4. Click Open and a new email message is created with the recipient’s address, subject and body content you saved in the template at once.

Normally there are 4 options for connection modes that appear on the screen as soon as you connect the phone to PC thru' USB..and the First one is the PC Suite Mode..(which is selected / highlighted by default). So press the center of the Navigation Key as if you are making a selection from the display immediately after you connect the phone to PC. If this does not work disconnect the phone.. and connect again..and this time press the lower Navigation key once and then the Center, thereby selecting the second option... If this also is not OK.. repeat with 2 times down and OK..and 3 times down and OK...

Hope one of these works and you are able to connect the Phone to PC.
