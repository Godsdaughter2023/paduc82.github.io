Activate Microsoft PowerPoint Online
1 Step
Go to the "Help" menu in PowerPoint 2003 and select "Activate Product." In PowerPoint 2007, click the "Office Button." Click "PowerPoint Options," click "Resources" and then click "Activate." In PowerPoint 2010, go to the "File" tab. Click "Help" and then select "Activate Product."

2 Step
Enter the "Product Key," which is a 25-character code on the CD case of your version of Microsoft PowerPoint. If PowerPoint was installed on your computer when you purchased it, look for the key on the computer box or with the included discs and paperwork. Click "Next" after entering the key.

3 Step
Click the "Accept" button to accept the license agreement and continue.

4 Step
Select "I Want to Activate the Software over the Internet." Click "Next."

5 Step
Select an option in the "Register with Microsoft" box. If you choose to register, fill out the required information. Click "Next" and wait for the wizard to complete the activation.

Activate Microsoft PowerPoint By Phone
Step
Follow steps one through three as listed in the previous section.

Step
Select "I Want to Activate the Software by Telephone." Click "Next."

Step
Select your location in the drop-down list. This will ensure you receive a phone number for your calling area.

Step
Call the activation number provided and speak to a customer service representative. After confirming your software with you, the representative will provide you with an activation code.

Step
Type the code the representative gave you into the "Confirmation ID" fields. Click "Next" to complete the wizard and activate PowerPoint.

A bit. I tried to re-think how this might be done. I suspect there is an easier way, but the following macro is the best I can do for now.

At the beginning, it opens the Find dialog box. Whatever criteria you choose, you then need to check "Highlight all items found in" then (probably) select Main Document in the dropdown.

Then click Find All. Word does its thing and you can then click Close.

The macro then goes through all the other open documents, performs the same search (AFAICS) and applies a highlight - you might prefer to change the colour. It should leave the last document it processes as the active document.

If you need to remove the highlights, the second macro should clear all find highlights for the active document.

AFAICS, the macro does not really need to run the find/replace dialog at the beginning - you could, I think run that in the usual way, then use the second part of the macro to "propagate the find."


Sub findInOpenDocuments()

Dim dlg As Word.Dialog

Dim doc As Word.Document

Dim f As Word.Find

Dim g As Word.Find

Dim strName As String

' this might not be quite enough...

strName = ActiveDocument.Name

Set dlg = Word.Dialogs(wdDialogEditFind)

dlg.Show

Set dlg = Nothing


' at this point, we have all the stuff we need to find/highlight 

' the same texts in all the other documents

For Each doc In Word.Documents

  If doc.Name <> strName Then

    doc.Select

    With Selection.Find

      .ClearHitHighlight

      .HitHighlight findtext:=Selection.Find.Text, highlightcolor:=WdColor.wdColorAqua

    End With

  End If

 Next

End Sub


Sub clearHighlights()
Selection.Find.ClearHitHighlight
End Sub

Sub UniqueDates()

Application.ScreenUpdating = False

Set ws2 = Worksheets.Add(before:=Worksheets(1))
r = ws1.Cells(Rows.Count, "E").End(xlUp).Row
ws1.Range("E1:E" & r).AdvancedFilter _
Action:=xlFilterInPlace, Unique:=True
ws1.Range("E2:E" & r).Copy
ws2.Range("B2").PasteSpecial xlValues
Application.CutCopyMode = False
ws1.Range("E1:E" & r).AdvancedFilter _
Action:=xlFilterInPlace, Unique:=False

ws2.Range("B2:B" & c).Copy
ws2.Range("B1").PasteSpecial (xlPasteValues), Transpose:=True
Application.CutCopyMode = False
ws2.Range("B2:B" & r).Clear
ws2.Range("a1") = "Visited"
ws2.Range("a2") = Str1
ws2.Range("a3") = Str2
ws2.Range("a4") = "Both"
i = 2
Do Until ws2.Cells(1, i) = ""
ws2.Cells(1, i) = Format(ws2.Cells(1, i), "mm/dd/yyyy")

i = i + 1
Loop
For i = 2 To 3
j = 2
Do Until ws2.Cells(1, j) = ""
x = 2
Do Until ws1.Cells(x, "E") = ""
If ws2.Cells(1, j) & ws2.Cells(i, 1) = _
ws1.Cells(x, "E") & ws1.Cells(x, "J") Then n = n + 1
x = x + 1
Loop
ws2.Cells(i, j) = n
n = 0
j = j + 1
Loop
Next i
i = 2
Do Until ws2.Cells(1, i) = ""
If ws2.Cells(2, i) > 0 And ws2.Cells(3, i) > 0 Then _
ws2.Cells(4, i) = 2 Else: ws2.Cells(4, i) = 0
ws2.Cells(1, i) = Format(ws2.Cells(1, i), "mmm dd yyyy")
Columns(i).AutoFit

ex-ms say: In reply to deleted message

Actually the "new" Skype version no longer provides an API. So it is not possible for add-on applications like Pamela to communicate with the Skype network anymore.

The only way to get addons to work is to install the Skype Desktop version (current version 7.40). You can download it from here: https://get.skype.com/go/getskype-full

Make sure you uninstall the newer Skype version.

Now you should be able to use all your Skype addon apps like Pamela again.

Note that Microsoft has decided to phase out the Skype API on Windows so that your addon apps will stop working eventually.

Till now Microsoft have officially stated they will still support call recording in the Skype API (see https://support.skype.com/en/faq/FA12395/how-can-i-record-my-skype-calls) but it seems that things are starting to break down......
