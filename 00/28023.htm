Hi Phil,

I noticed that the SharePoint site you are trying to access is still using the old format. Office 2016 may have some problems connecting to BPOS URLs, so this issue may be related to a similar one encountered recently.

As part of an ongoing effort, SharePoint Online sites that use the *.microsoftonline.com URL format will be renamed to use the *.sharepoint.com URL format.

As your site has not yet been renamed to *sharepoint.com URL format and you would like to use it with Office 2016, we would appreciate it if you would try to manually rename the site using the guide below:

$adminURL = 'https://<domain>-admin.sharepoint.com'

$site = Get-SPOSite $bposURL

The first 2 URLs should be the tenant admin URL and the BPOS URL. The 3rd is the target URL, it must start with the tenant name and hyphen but the highlighted part can be portal1 and portal2 according to the BPOS URL.

When you are installing an app from SD card, here are some things to remember.


1. If you visit the store on the phone, then open that app page, do you get INSTALL/BUY/TRIAL. If any of these buttons show up, that means the app is supported on your phone. If it does not, putting the XAP file on SD card won't change anything for you.

Remember that, you need to have an internet connection to install the app. It does not matter if the app is on SD card or it will be downloading. The phone store has to verify that your phone/account is allowed to install the app. If the store is not able to verify, the installation will not progress.

charles kenyon say:
Save each part of your Master Document and all subs (but not from within the Master). See how much you can rescue.

See Master Documents Feature in Microsoft Word.
John McGhie put it succinctly when he said that "there are two kinds of Master Documents: Those that are corrupt and those that will be corrupt soon." It was reported fixed in Word 2007, but "The jury is in: The Master Document feature in Word 2010 remains word processing's answer to Conan the Destroyer. Use it only if you enjoy pain and frustration." Microsoft Word 2010 Bible by Herb Tyson, MVP. "Since its inception, the Master Documents feature has been a bit quirky and buggy, and that fact hasn't changed with Word 2010, unfortunately." Microsoft Word 2010 In Depth by Faithe Wempen, p. 681

General notes on large documents and combining documents.

  • Combining documents is probably easiest if they are based on the same template.
  • If possible, all formatting should be done using styles. Direct formatting exponentially increases document complexity and makes editing much harder. With short documents this isn't a big problem, with long ones it is.
  • Hard page breaks should be avoided. Use paragraph formatting of page break before where a page break has to be forced.
  • Never, ever, use the Master Documents Feature in Microsoft Word - at least not until you read the articles at this link.
  • If you use automatic paragraph numbering or bullets read Shauna Kelly's articles on numbering and bullets. Start with How to create numbered headings or outline numbering in Word 2007 and Word 2010 For large documents you must follow these directions or you will lose your hair!

Downgrading New Skype to Skype 7 (previous version) on Samsung - Microsoft Community

vanzelot say:

Downgrading New Skype to Skype 7 (previous version) on Samsung Android Phones

Hello.

This is a guide on how to downgrade New Skype to Skype 7 (previous version) on Samsung Android Phones or try this procedure if it is applicable to your non Samsung phone.

1. Go to Settings.

2. Select Apps or Apps Manager

3. Select the Skype Apps

4. In the upper right corner, beside the Application Info, you see 3 dots.

5. Touch the 3 dots and select the Uninstall Updates.

6. After uninstalling update, go to the home screen and open your Skype.

7. Once you open your Skype, it will send you to Skype Update in the Galaxy Apps.

8. Press the Skype Update. This update will make your Skype from factory version to Skype 7.0

9. Finish. You can use your Skype now with no problems at all.

P.S. In the Google Play Setting, please check the "Do not auto update apps".
