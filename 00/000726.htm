    *** Moved from Outlook.com / Email / Sending, receiving, and reading email ***

    jose est say:

    Hi Gildas,

    Error 450 4.7.1 indicates that the Exchange server is performing the reverse lookups to identify the address from where you are sending the email. Example: Suppose you want to send an email from example@domain.com, as you send the emails, your system is stamped with the IP address of your system. Now, when the server receives the email, it verifies first whether the email is sent through the same domain by verifying its IP address. It checks whether the IP address is pointing to the same domain, i.e. example@domain.com. If the server was not able to detect the sender of email or is not able to identify the hostname, then Error code 450 4.7.1 Client host rejected: cannot find your reverse hostname is encountered.

    We strongly advise contacting your recipient's email provider to check the configuration of Mail server and verify whether domain records of DNS has been setup properly or not.

    If your IP address is blacklisted by the server, find out the main cause of blocking. An IP address is generally marked as blacklisted when they are sending spam link or content. Request the server to remove your IP address from the blacklist.
