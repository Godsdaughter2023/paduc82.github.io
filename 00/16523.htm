hansv mvp say:

That is more a Windows 10 question than a Word question, but here goes:

  • Right-click an empty part of the desktop.
  • Select Personalize from the context menu.
  • Alternatively, click Start > Settings > Personalization.
  • Select Colors in the navigation pane on the left.
  • Turn the slide 'Automatically pick an accent color from my background' to Off.
  • You will now see a palette of colors to choose from.
  • Select the color you like best.
  • If you only want colored tiles, set the slider 'Show color on Start, taskbar and action center' to Off.
  • If you want the background of the Start menu etc. to be colored, set this slider to On.
