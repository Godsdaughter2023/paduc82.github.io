ossiemac say:

Good solution suggested by Andreas Killer but to expand on his answer, you can use the following event code to determine if the user clicks on a cell that contains List Type Data Validation and if so then apply the SendKeys to display the DropDown.

Not sure if you require assistance to install the code but if you do then the following guidelines:

  • Right click the worksheet tab name
  • Select "View code" (will open the VBA editor window at worksheet's module)
  • Copy the VBA code below and paste into the VBA editor.
  • Do not rename the sub
  • Close the VBA editor (Cross very top right of VBA editor window)
  • Save the workbook as Macro enabled.
  • Ensure macros are enabled. See help for how to do this. (Option to "Disable all macros with notification" should be OK.)

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    Dim lngValType As Long
   
    On Error Resume Next
    lngValType = Target.Validation.Type
    On Error GoTo 0
    If lngValType = 3 Then SendKeys "%{DOWN}"
End Sub
