jay freedman say:

From the results of those tests, and the fact that you've tried with an external keyboard too, my guess is that there's a problem with either the circuitry that interprets keystrokes or with the keyboard driver software. The CPU just isn't seeing Ctrl+N or Ctrl+O keystrokes, so there's nothing to respond to.

There isn't much you can do about a hardware error, and in any case I'm not competent to tell you what a technician would do about it.

You can get some clues about the driver by looking at the Device Manager in Windows.

  • Click the Start button or  Cortana and type device. When Device Manager appears in the result, click it. 
  • Expand the Keyboards section.
  • If any of the icons there have a yellow question mark or other indication of an error, you can try updating the driver.

To update a driver for a keyboard, first it's best to make an image backup of at least the system drive, using a backup program that lets you make a bootable recovery disk. This is a fallback in case the keyboard becomes completely unusable after a botched update. (Trust no one!) Once that detail is taken care of, proceed:

  • Check the website of the laptop's manufacturer for the latest keyboard driver specific to your model, if there is one. If it's offered, download it and make a note of where you stored it. (A CD or a USB stick may be best.)
  • If you're logged in as a standard user, log in as an administrator user instead.
  • Open the Device Manager, expand Keyboards, and double-click the icon with the error sign.
  • In the Properties dialog, click the Driver tab.
  • Click the Update Driver button.
  • If you downloaded an update, choose the browse option; otherwise, choose to search automatically for an updated driver.
  • Let the update proceed. Good luck!

bob jones aka: cybertaz say:
Well, the Clear Filter button in the Filter pane is inactive because there is no data & no filter is applied. That button only removes the filtering [i.e., restores the records] that has been applied on the column.

If you want to turn the Filter feature off altogether click in the range then either
  1. Go to the Data tab of the Ribbon
  2. Click the big Filter button or
  1. Go to the Data menu
  2. Click Filter top remove the check
