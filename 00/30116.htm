john john __ say:

On 1/22/2011 11:49 PM, trnowry wrote:

Is there a way, for "out dated" hardware, like mine, to shift the
Virtual memory that windows 'makes' from the HD to something much
faster and cheaper like USB flash drives or camera cards?

This doesn't work on Windows operating systems as Windows does not allow the placement of the pagefile on removable disks.  Also, USB flash drives are not faster than hard disks, random reads are faster on flash drives but sequential read and writes are faster on hard disks.

ReadyBoost (available on Vista and newer operating Windows operating system) uses USB flash devices to cache portions of the pagefile on USB flash drives and the Memory Manager will then decide which is faster and where to get the cached information.  The point to note is that the pagefile itself is not on the flash drive and that there is nothing stored on the flash drive that is not present in and backed up by the pagefile, before anything makes it to the flash drive it must first be stored in the pagefile.

Oh boy. Lots to talk about.

1- You should be making an image backup every day, not every six months. Third-party software makes this easy peezy.

2- You should always store your backups outside the computer, not on an internal hard drive. If you store a backup inside the computer, and the computer is lost, destroyed or becomes unusable, there goes your backup.

3- You don't say what software you used to make your backup image. If you used System Image Backup (SIB):

  • SIB was deprecated with the Fall Creators Update - that means it's no longer being developed and might even be removed in a future update; and
  • to restore an image made with SIB you need to do a clean install first and then restore the image.

  • create and save different backup jobs for different purposes;

  • automatically run backups on a schedule you choose;
  • maintain a retention schedule; for example, you might want to keep the last month's backups or the last 10 backups - whatever you like
  • create a recovery drive to restore a backup even when the originating computer is unbootable

and a whole lot more. I happen to use Macrium Reflect (which also comes in a free version) but there are many other great applications that you can find with a search of the internet.
