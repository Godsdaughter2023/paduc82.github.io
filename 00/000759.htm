greg wilson 2 say:

This assumes that there are a large number of cells involved. My hunch is that it is caused by one or more of the following:

1)  Having too many cells containing the SumProduct function. This function does array calculations and, as such, has a high overhead.

3)  Having too many cells containing formula with complex dependencies on other cells (i.e. these cells are in turn dependent on others etc.). 

2)  Too many linkages between workbooks.

3)  Too many cell formats.

Suggestions:

1)  Experiment with reducing the number of cells involved and see at what point the problem develops.

2)  You may have to get rid of the SumProduct function and replace it with a VBA macro that loops through the cell range doing what is required. There is next to no overhead with this.

3)  Experiment with turning off worksheet calculation before making changes and turning it on again once all the changes have been made so that it only has to calculate things once. Programmatically, this is done respectively using:

Application.Calculation = xlCalculationManual

Application.Calculation = xlCalculationAutomatic

Sorry to hear the problem isn't resolved. I thought there was a lot of code. I get the impression there isn't that much. Mostly worksheet formula.

By using ActiveWorkbook without referring to your Excel.Application object, you create an extra instance of Excel that is not controlled by your code. Use WB instead.

Public Sub createExcelFile()
    Dim XL As Excel.Application, WB As Excel.Workbook, WKS As Excel.Worksheet
    Dim db As DAO.Database, rec As DAO.Recordset, f As DAO.Field
    Dim i As Integer, j As Integer
 
    Set XL = New Excel.Application
    XL.Visible = True
    Set WB = XL.Workbooks.Add
    Set WKS = WB.Worksheets(1)
 
    WB.SaveAs Filename:="C:\Users\MAG\Documents\Reports\Report1.xlsx", _
        FileFormat:=xlOpenXMLWorkbook
 
    WB.SaveAs Filename:="C:\Users\MAG\Documents\Reports\Report2.xlsx", _
        FileFormat:=xlOpenXMLWorkbook
 
    WB.SaveAs Filename:="C:\Users\MAG\Documents\Reports\Report3.xlsx", _
        FileFormat:=xlOpenXMLWorkbook
 
    WB.SaveAs Filename:="C:\Users\MAG\Documents\Reports\Report4.xlsx", _
       FileFormat:=xlOpenXMLWorkbook
 
    WB.Close SaveChanges:=False
    Set WB = Nothing
    XL.Quit
    Set XL = Nothing
End Sub
