Beginning a document in Office Word 2007, I'd like to have a "3 letter ident to EVERY paragraph."

   Whenever I type a personal letter, or condense a document (doc) for personal use, I always type the first body-line with a 3-letter indent, like herein, (instead of the usual 5-letter indent). I've been doing it by hitting the Space Bar. I would like to have it done automatically throughout the entire document instead.

   Upon starting a doc, I was hitting the Tab on the keyboard, but it indented too far (to where those little lines below the white part of the ruler are).  Most of the time, to save paper, I pull the thing which sets the margin of the paper to the left (though the white bar part of the ruler doesn't move), then moved the top little triangle ("1st line indent") to where the 3rd letter will be, and begin typing.

   How can I do this automatically for the entire document?

Mike

a. user say:

The simplest way is probably to create a style containing the desired first line indentation, and then apply this style to the document.

I only have Office 2013, so the steps may be different, but this is how I did it as a test:

  1. Type some text with the font size and other formatting that you want to use (but no indentation).
  2. Select the text, right click on the selection and choose Styles->Create a style
  3. Pick a name for Your style, and choose Modify
  4. Choose Format->Paragraph
  5. In the Field Special, choose First line, and then choose the size of the indentation. Note that you choose a measurement rather than a number of characters. This is because characters usually have different widths.
  6. You may choose to add the style to the Styles gallery, and to use it for all documents based on the same template.

When the style is created and saved, it can easily be applied to the entire document.

Hi CarolRay8,

 

From your description, you are not able to save the custom .dotm file to replace the original Normal.dotm file. Please try the steps below to see the result:

  1. Save the new custom “Normal.dotm” file to desktop.
  2. Quit the Word app. (as the template can’t be saved when you are opening it).
  3. Copy and paste the new custom template to ~/Users/<your login name>/Library/Group Containers/UBF8T346G9.Office/User Content/Templates. When you are asked if you want to replace the file, click yes to replace.

 

If the issue persists, please provide us the information below for more investigation:

  1. The screenshot about the “it says can’t save”.

  4. What’s the OS version?
