pa bear - ms mvp say:

I do allow the monthly Defender download to scan my system.

I think you're referring to the monthly Malicious Software Removal tool here, not Defender Definitions updates (KB2267602).

Assuming all is well with the computer, Windows Update is enabled, and the computer's connected to the internet, Defender will auto-update its Definitions at least once-a-day. That being said, Defender considers itself sufficiently up-to-date as long as its Defs are no more than 72 hours old.

If you (e.g., JimSmith42) feel that Defender needs to be updated more frequently, you can always update manually whenever you wish via the UPDATE button on Defender's Update tab (or use Scheduled Tasks).

Note that Defender includes heuristic detection of malware, etc., so MS feels that once-a-day updates is sufficient. YMMV
