macropod say:

You may be able to achieve what you're after via a Document_Close macro coded along the lines of:

Private Sub Document_Close()
Dim Rslt As Variant
On Error Resume Next
Restart:
With ActiveDocument
  If .Saved Then
    .SaveAs2 FileName:="C:\Users\" & Environ("UserName") & "\Documents\Backups\" & .Name, AddToRecentFiles:=False
  ElseIf InStr(.FullName, "/") > 0 Then
    .Save
    GoTo Restart
  Else
    With Application.Dialogs(wdDialogFileSaveAs)
      If .Show = False Then Exit Sub
      GoTo Restart
    End With
  End If
End With
End Sub

Simply change the code's save location to meet your own requirements then add the code to the 'ThisDocument' module of the document or its template, depending on whether you want it to apply to just the one document, or all documents based on the template.

Add a text field to your products table to store the full filename and path to the image of the product (ex: S:\Products\Images\1234.jpg or better yet \\servername\sharename\Products\images\1234.jpg). 

On the Form add an Unbound Image Control and a text box bound to the image path field.

In the On Current event of your form use Code like:

If DIR(Me.ImagePath) = "" Then

    Me.imgProduct.Picture = defaultimage

Else

ImagePath is the name of the control storing the location of the image file, imgProduct is the name of the Image control used to display the product. Defaultimage is the location of a default image file to use if there is no image file for the current product.

If you want to sort numbers from highest to lowest, enter below formula as an array formula (Ctrl+Shift+Enter) in cell B2 and copy down:

=IFERROR(LARGE(IFERROR(--SUBSTITUTE(IF(A$2:A$10<>"",A$2:A$10)," ",""),""),ROW(A1)),"")

If you also want to rank column A numbers, then enter below formula in cell C2 and copy down:

=IF(A2="","",RANK(--SUBSTITUTE(A2," ",""),B$2:B$10,))
