Word creates its own Preview for the EPS. This is some kind of SVG format and often it creates a very poor rendition. Missing lines, incorrect collars, missing text elements etc.

In a professional layout software (i.e Indesign) Eps is maintained and the EPS or PDF will be handed over to the Postscript printer to be rendered 99,99% of the time it is correctly rendered!

For best functionality:

  1. Create a large / or high resolution PNG file (pixel image)
