bernard liengme say:

This seems to fit the bill

To find your colour: select a coloured cell in Book1,type this into the VBA immediate window

msgbox Selection.Interior.ColorIndex

and press enter

Put the subroutine code in a module in Book1.xlsm

Note my files are Book1.xlsm and Book2.xlsx

best wishes

Sub TryMe()
  Worksheets("Sheet1").Activate
  LastRow1 = Cells(Rows.Count, "A").End(xlUp).Row
  With Workbooks("Book2.xlsx").Worksheets("Sheet1")
    LastRow2 = .Cells(Rows.Count, "A").End(xlUp).Row
    myrange = "A2:B" & LastRow2
    Set mytable = .Range(myrange)
    Debug.Print mytable(2, 2)
  End With
  myrow = LastRow1 + 2
  For j = 2 To LastRow1
     If Cells(j, "A").Interior.ColorIndex = 6 Then
        For k = 1 To LastRow2
          If Cells(j, "A") = mytable(k, 1) Then
             Cells(myrow, "A") = mytable(k, 1)
             Cells(myrow, "B") = mytable(k, 2)
             myrow = myrow + 1
             Exit For
          End If
        Next k
     End If
  Next j
End Sub

Yammer - how to stop the system proposing deactivated staff to new and existing users

Can this is stopped/suppressed?

patil bhalchandra say:

Hello Salos2k18,

 

Thank you for giving us opportunity to assist you.

 

Though, you might have deleted users from office 365 they might be in pending state on your Yammer network and hence they appear in suggested members list.

 

Now to resolve this issue we need to enable enforce office 365 identity and also delete users using bulk update method who are in pending state on your yammer network

 

If you are verified admin of the yammer network than, I would request you kindly download the export users file by navigating to the yammer home network as a verified admin > Settings > Network Admin > Export Users > Export all users > and click on export.

 

You will find a file as users.csv in the downloaded export folder.

 

We can differentiate in users.csv by Filter with column “state” and select soft_delete. Uncheck the box of “active”. Applying this filter would give us only soft_delete users.

Filter “joined_at” column which contains only blank value.

Filter “deleted_at” column which contains only blank value.

Filter “suspended_at” column which contains only blank value.

By doing above steps we can differentiate the users are currently in deleted state if they have “deleted_at” time stamp.

If the user with “suspended_at”  than that user is suspended or in block state.

If the user is having none of the states “deleted_at” and “suspended_at” then the user is in pending state

 

Kindly refer the article which talk about remove users from Yammer.

https://docs.microsoft.com/en-us/yammer/manage-yammer-users/add-block-or-remove-users

 

Note:- If you are using yammer free basic network and admin group than unfortunately you cannot download  users.csv file to see the pending user list. However, you can navigate to the group settings and click on add or remove user and can see list of the active user and admin of the group with the star mark below profile.

 

If you know the email id of the users whom you invited than kindly search the name of the user and you can click on the user name and there it will show as pending user.

 

If you find my reply useful, would appreciate if you can click it as helpful.

 

Please let me know if you have any further query or concerns. Please to assist you.
