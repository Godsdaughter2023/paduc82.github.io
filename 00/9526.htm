hairyfool say:

There are specific forums for Office issues, this is for general Operating System problems

However a Circular Reference means you are feeding the Total back into cells that are included when creating the total in the 1st place.

In simple terms Cells A+B+C+D = E. But if the value in B is a formula to include an element of E then E will never resolve. Watch out for hierarchical references. B could be the sum of X+Y+Z but if Z includes a reference to E then you have the same problem. Occasionally you want this to happen to produce an iterative calculation say to establish a maximum or minimum.

em-squared say:

In Word 2007, if you want your Headings to be numbered, try the following:

  1. Start with a blank document.
  2. Show the Styles by clicking the small icon at the bottom right of the Styles section of the Home ribbon. Docking it on the right helps to see it when working on your document.
  3. Create a line of text.
  4. Click the small arrow beside the Multilevel List button on the Paragraph section of the Home ribbon. Choose a list style that includes the "Header" on the sample together with the numbering style. That is, "1. Heading1____" not "1._____"
  5. You should find that when you create successive lines, set your cursor somewhere on text for a heading (make sure no characters are highlighted), and choose a Heading style, it should be numbered accordingly.

What I personally don't like about this style of number, though, is that it isn't easy to see the outlining. I prefer to have my successive headings nested or indented. To achive this, I did the following:

  1. I set up the Header Multilevel List, as above.
  2. With the cursor in Heading 1, I clicked the arrow beside the Multilevel List button and chose to Define a New List Style.
  3. In the ensuing dialogue box, I gave the new style a name.
  4. Choose Format | Numbering, and in the Modify Multilevel List dialogue box, click Set for All Levels.
  5. Set the Additional indent for each level. I would recommend starting with the same value as Text Position for first level and see if you like it. This setting makes each successive indent the same. You may like that or you may wish to adjust settings in the Modify Multilevel List dialogue box. You just need to make sure that the heading number doesn't push the title to the next tab stop (unless you don't mind).
  6. Click the More>> button. Choose the approriate Heading style in the Link Level to Style dropdown at the right to match the leve you have selected at the left under Click Level to Modify. You will have to do this for each level that you expect to use. Notice that the Style name shows up as one of the built-in ones was in my seciton above.

Great. Start using it. Ooops, now you will find that it doesn't look so great when you start adding normal body text. The normal text is stuck out at the right margin. Taking the lead from the built-in styles, I created a style for levels 2-9. For each level, I created a new Sytle, based on = Normal, and the ony thing I changed was the indent so that it uses Normal for eveything else. To get your positions, start with the value of Aligned At in the Modify Multilevel List dialogue box where you set the Link Level to Style, Set for All Levels button and Add Tab Stop At for the Multilevel List style you created. I have the following for my metric setup:

  1. 0.0 (don't need to set; Normal likely defaults to this on your machine)
  2. 0.76
  3. 1.52
  4. 2.28
  5. 3.04
  6. 3.80
  7. 4.56
  8. 5.32
  9. 6.08

I set up 8 styles for each level. It's not too tedious, and if you save this as a template, your pain now will be glory several times in the future. Do ensure that the Based on Style dropdown always says Normal.

Now, you should set the Style for Following Paragraph in the Header styles. Go to Headers 2-9, right-click in the Styles window and choose Modify. All you need to do here is set your speciall Normal style to be the one that is created after that Heading style when you press Enter.

Save your template again.

One last suggestion. If you have a need for such a document structure, you will likely benefit from working in Outline view some of the time. This is because documents which require such structure are long. If you first go into Outline view, set up the headings which you wish to communicate and start typing, you will find that Outline view is great for setting up the headings, whereas Normal view is great for typing text after the heading. You can type text after the heading in Outline view, but it doesn't use the Style for Following Paragraph setting.

HTH,

M<sup>2</sup>

neha singh msft say:

Hi Aaron,

How do you open the custom stencil? Do you double-click the stencil stored in your OneDrive synced folder?

If you double click the stencil stored in OneDrive folder, it is opened as read-only which disables the functionality to edit. When you open the stencil in this way, you must have noticed the ribbon with lesser tabs, as shown in the following screenshot:

 

To make it work, we have two options:

Option 1: Click Edit document under File>Info

  1. From the above screenshot, click File>Info.
  2. Under Info, you’ll notice the message “Read-only from Server”
  3. Click Edit Document to make it editable.

 

 

Option 2: Open the stencil from within Visio application

  1. Open Visio 2016 application on your desktop.
  2. Click File>Open, navigate to OneDrive folder where you’ve stored the stencil.
  3. Double click the stencil to edit the master shape.

Let me know if you need further help.

 

Thanks,

 Neha
