bazajaytee say:

You can't compare the price of a disc and digital copy of the game.

Publishers set the price and retailers then chose what they sell it at. Publishers can't go out of their way to under cut the retailers as that would start whole legal battles of unfair competition.

So for example. A publisher says a game should cost 70€. Amazon will then sell it at 60€ to entice people to buy from them but Microsoft would have to sell it at 70€ because that is what price was set out. Of course they do sales and launch offers but that's it.

The cost difference is purely down to retailers + cost of doing business in each region + cost of distribution + cost of staff in each region + taxes for all those things (fuel tax for the lorries that distribute the game etc) + accounting for cost of living + national average wage etc.

Take Sunset Overdrive. In the US it costs $59.96

UK - £39.85 - 64.0942 USD

Europe - EUR 59,99 - 76.8226 USD

Australia - 89.95 AUD - 78.7378 USD

You'll see this price difference in just about everything sold Internationally. For example the car I have cost me ~$27,000 and back home in the UK that same spec car cost ~27,000 British Pounds which, when converted is ~$43,500. Might not seem fair but it is the way International business works.

I had the same problem. If you have an Elantech touchpad, to fix it, you can edit a few registry values.

Under:

HKEY_CURRENT_USER\SOFTWARE\Elantech\SmartPad

I set the following four to 0:

EGS_InertialScroll_Enable
EGS_InertialScroll_ShowItem
SC_InertialScroll_Enable

SC_InertialScroll_ShowItem

It won't immediately go into effect, so just wait a bit and it should eventually detect these new settings. Memory Dump files:

A memory dump records all the contents of system memory when your computer stops unexpectedly. A complete memory dump may contain data from processes that were running when the memory dump was collected.

You can refer to the below link for details on Memory Dump Files:

Overview of memory dump file options for Windows 2000, Windows XP, Windows Server 2003, Windows Vista, Windows Server 2008, Windows 7 and Windows Server 2008 R2

http://support.microsoft.com/kb/254649

System Error Reporting Files:

Windows Error Reporting in Windows 7 is a feature that allows Microsoft to track and address errors relating to the operating system, Windows features, and applications. Windows Error Reporting gives users the opportunity to send data about errors to Microsoft and to receive information about solutions. Solution information can include instructions for working around a problem, or a link to the Windows Update Web site or another Web site for updated drivers, patches, or Microsoft Knowledge Base articles. Microsoft developers can use Windows Error Reporting as a problem-solving tool to address customer problems in a timely manner and to improve the quality of Microsoft products.

If you want to free some space on your computer, you can delete these files. Refer the below link on how to delete files using Disk cleanup:

Delete files using Disk Cleanup

http://windows.microsoft.com/en-us/windows7/Delete-files-using-Disk-Cleanup
