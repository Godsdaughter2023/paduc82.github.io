Sorting Data Sheets - Microsoft Community

jmast83 say:

Sorting Data Sheets

Is there any way to set up a worksheet to automatically sort new entries with previously entered entries as they are added within a specified group of cells?
vijay a. verma say:

This can be done only through VBA

1. Make a backup of your workbook. 
2. Open your workbook and ALT+F11
3. Locate your Workbook name in Project Explorer Window
4. Double click on your worksheet name
5. Copy paste the Macro code given below
6. Save your file as .xlsm

The macro is made for column A. If any other column, change bold entries appropriately. If you want, you can also change xlAscending and xlYes.

**** Macro by eforexcel.com *****

Private Sub Worksheet_Change(ByVal Target As Range)
    Application.ScreenUpdating = False
    On Error GoTo ExitSub
    If Intersect(Target, Me.Columns("A")) Is Nothing Or Target = "" Then Exit Sub
    Application.EnableEvents = False
    Me.Columns("A").Sort Key1:=Me.Range("A2"), order1:=xlAscending, Header:=xlYes
ExitSub:
    Application.EnableEvents = True
    Application.ScreenUpdating = True
End Sub

gwnatl say:

Picture shapes not being copied to clipboard

I paste a picture into a Word document then I go to Insert | Shapes and insert a callout onto the picture in the document. Then I copy the entire document to the clipboard. When I go to another application and paste the clipboard in, only the picture is included not the shape that I inserted on to the picture. Is there a way to copy include the shape as well as the picture when copying from the clipboard?
marysully say:

You have to Group the Shape and Picture together to have them Paste together.

Try this: At “Insert” tab click on “Shapes”  button and then click on “New Drawing Canvas.” Insert picture into DC. You may have to resize it. Then insert Shape. Next, while holding down Ctrl key select picture and then shape. Right click and select “Group.” -> “Group.”  At this point Copy and Paste and picture and shape should stay intact when pasted.

Can a Skype Number receive text messages? - Microsoft Community

lord shadowy say:

Can a Skype Number receive text messages?

I am looking into getting a skype number. Does it allow customers to text me at my skype number?

Thanks,

Subject/title expanded by Moderator to add detail.

danida_u say:

Hi

Skype Numbers are mainly use to receive calls from mobile and/or landline numbers, they cannot be used to receive SMS.

You can send SMS from Skype but you cannot receive SMS replies as Skype does not support this. However, you can set up your mobile number as your sender's ID to receive SMS replies from your recipient. The messages will go to your mobile number and not Skype.

You can refer to the following support links for additional information.

  • What are the different types of Skype subscriptions and pay-as-you-go options?
  • Can I send SMS text messages to any number?
  • How do I show my mobile number when sending SMS text messages from Skype?

Hope this helps! Let us know if you need additional information or assistance.

Thank you.
