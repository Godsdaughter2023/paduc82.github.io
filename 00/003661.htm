doug robbins - mvp office apps & services (word) say:

Run a macro containing the following code when the document containing the Hyperlinks is the active document

Dim docsource As Document, doctarget As Document
Dim alink As Hyperlink
Set docsource = ActiveDocument
Set doctarget = Documents.Add
With docsource
    For Each alink In .Hyperlinks
        doctarget.Range.InsertAfter alink.Address & vbCr
    Next alink
End With
doctarget.Activate
