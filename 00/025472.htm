maria fab say:

Hi Andrew,

To change the default "From" when sending a message, you need to set the default email account in Outlook. The default account is used when you start composing a new message. Here's how:

  1. Click File in Outlook.
  2. Make sure the Info category is open.
  3. Click Account Settings ▾ under Account Settings.
  4. Select Account Settings from the menu that has appeared.
  5. Highlight the account you want to be the default.
  6. Click Set as Default.
  7. Click Close.
  8. Go to File>Options>Mail.
  9. Under the Send messages section, put a check mark in "Always use the default account when composing new messages" and then click OK.

When you write a reply to a message, Outlook is pretty smart about selecting the email account used for sending your mail. If the original message was sent to an email address that appears in one of your Outlook accounts, the corresponding account will be selected for your reply automatically.

Only if none of your email addresses appear in the original message does Outlook use the default account for composing the message. Of course, the default account is also used when you start composing a new message (instead of a reply).

If the emails sent using Outlook are not saved to the Sent Items folder, check the option for Save copies of messages in the Sent Items folder in Outlook if it's disabled. Check this link.

In many cases, Outlook 2016 uses Automatic setup. If you are using an Exchange ActiveSync or IMAP, you can access your email wherever you are, from any device. Whilst POP email is downloaded from the Exchange Server onto your computer. Once the email is downloaded to your computer, it is deleted from the server. To access the email after it's downloaded, you have to use the same computer. If you access your email from a different computer or device, it will look like some of your previously downloaded messages have been deleted.

Let us know how it goes.
