I noticed this, too, but found a "workaround" (until Nokia updates their firmware to include this function).  USE the "Notes" APPLICATION, OPEN a “New Note”, then "touchpad" in the blank-lined area to start entering text.  The default "mini QWERTY keyboard" will pop up (well, that's what I have set for my text input).  In the lower right-hand corner of the "mini QWERTY keyboard" (also in the "Full Screen QWERTY keyboard" OR the "Handwriting keyboard", if you have either of these set for your default text input), there is a small image of a keyboard.  When selected, a MENU pops up, allowing you to SELECT an option for an "ALPHANUMERIC KEYPAD" for text input into the "Notes" application.  You can get the numerical correspondence for spelled-out phone numbers like 1-800-iLIKE-YOU (1-800-454-3968) [I don't know or care if that's a "real" number, I'm just using it here as an example] and can just remember them and input them manually into your “Dialer” application, or, you can “type” it directly into the blank “New Note” you already have open, SELECT and EDIT->COPY the number into the phone’s memory, then switch to the “Dialer” application (PRESS and HOLD the WHITE middle button on the lower touchpad until the "open items" display appears, allowing you to select the HOME or the previously-opened "Dialer" application), open the "Dialer" menu and EDIT->PASTE the previously-copied number into the “Dialer” application.  Press the “Dial” function icon and the number will be appropriately dialed.  Whew!  It gets a little easier with practice, especially if you use the GREEN BUTTON on the lower left of the phone’s touchpad, HOLDING IT DOWN until the “Voice Command” application initiates, then say out loud “Notes”, and the phone will switch to the “Notes” application with relative ease, so you’re just a few seconds away from having a “real” alphanumeric keypad that can be used to “decode” those alphanumeric phone numbers many businesses advertise.  Enjoy! Importing Word documents (docx) into Publisher 2013 - Microsoft Community

namesales say:

Importing Word documents (docx) into Publisher 2013

I have recently purchased a new computer running Windows 8.1 and Office 365.  I am wondering if it's possible to import my Word documents containing pictures into Publisher 2013.  The Word documents were mostly created using Word 2007.  Hope someone has an answer for me.
davidf2 say:

Create a blank publication > Insert > Insert File and browse to the Word file to select it. Publisher will import the file, creating text boxes and pages as necessary. The text boxes will be linked together. The pictures should come with the text but the formatting may need to be adjusted.

You can also insert a text box and insert the file into that text box.
