a. user say:

Hi,

Using the intitle: operator, you can restrict your search to find out webpages which contain a specific phrase or keyword in it’s Title attribute. An example search for the phrase “yellow pages” would be intitle:yellow pages.

The intitle operator is useful when you are searching for something very specific and want to filter out noise and useless information from search result pages.

Using the NOT operator is fairly easy, simply use the following search command:

[keyword] NOT [ keyword to exclude]

Example: Say you are searching for “Face” and want to exclude “Facebook” from search results. Use the following search command at Bing.com

Face NOT Facebook

Convert table into template

I am using Microsoft Word 2007.

I have a table that I would like to convert to a template form so that the size of the cells does not change when others complete the form.

jay freedman say:

The ideas of "template" and "table with cells of fixed size" don't really have much to do with each other -- you can make a table like that in either a document or a template. (Note that here I'm using the formal MS Word meaning of "template" as a *.dotx or .dotm file used as the base for new documents, not the informal meaning of "a starting point for making copies".)

Start with the cursor in the table you already have. If the form has protection, turn off the protection. Because the cursor is in a table, the Table Tools tab will appear on the ribbon. Click the Layout tab under Table Tools.

Click the AutoFit button in the Cell Size group, and choose Fixed Column Width.

Either drag the bottom edge of each row to where you want the maximum height to be, or use the vertical size box in the Cell Size group to enter a numerical size. Then click the little arrow at the right end of the title bar of the Cell Size group, and change the "At least" in the dropdown to "Exactly". (If you want all the rows to be the same height, you can select the rows before doing this and set them all at once.)

If the form needs to be protected, turn the protection on again.

Save, either as a document or (by changing the "Save as type" dropdown in the Save As dialog) as a template (.dotx file).
