Follow the below steps enable the Use original boot configuration:

  1. Press Windows + R keys from the Keyboard.
  2. Type msconfig and hit Enter.
  3. Click on the Boot tab.
  4. Under Boot options check mark on the Safe mode.
  5. Select the General tab.
  6. You can see that the Selective startup option will be selected and the Use original boot configuration will be enabled.

  2. Type msconfig and hit Enter.
  3. Click on the General tab.
  4. Select the Normal startup option.
  5. Click on Apply and Ok.
  6. Restart the system and check.
