How do you do Refresh and Restore without recovery media Windows - Microsoft Community

delrel say:

How do you do Refresh and Restore without recovery media Windows 8.1?

Obvious question, since Microsoft will not allow us to download and ISO, how can the Refresh and Restore advanced recovery features be used in Windows 8.1?  Clicking on either options after upgrade install from Windows Store, presents no valid options without media.  We can only uninstall, go back to 8 and then reinstall from store?  Dumb!

jj97403 say:

Obvious question, since Microsoft will not allow us to download and ISO, how can the Refresh and Restore advanced recovery features be used in Windows 8.1?  Clicking on either options after upgrade install from Windows Store, presents no valid options without media.  We can only uninstall, go back to 8 and then reinstall from store?  Dumb!


This is kind of a long way but it worked for me. You need a valid Windows 8 product key and I don't think it will work for a new or clean install. But it does provide the installation media when prompted you are missing installation files. So you can now "restore" or "refresh" within Windows 8.1 that has been upgraded from Windows 8 in the App Store.

1. You need a valid Windows 8 Product Key. I used mine from Amazon Windows 8 Pro Upgrade DVD I bought last year to upgrade my PC from Windows 7 to Windows 8.

2. Go to http://windows.microsoft.com/en-US/windows-8/upgrade-product-key-only

3. Download both "Install Windows 8.1" and "Install Windows 8." ; Don't install yet. Just save the files in your Download folder (or wherever).

4. Start with "Install Windows 8" ; click on "Install Windows 8: ; Enter your Windows 8 Product Key when prompted ; It should be accepted if valid (mine was) ; start download; once started, cancel by closing window; you'll get prompted with a warning 'are you sure you want to cancel?' ; go ahead and cancel by closing window once the download has started. I let mine download no more than 10 or 15 seconds once it started.

5. Now click on "Install Windows 8.1" ; the Windows 8.1 iso file should automatically start to download without being prompted for product key (mine did). The file might be as much as 3GB, so it could take a long time do download.

6. Once the file completes download, skip the install; you will be given a chance to make recovery media. I chose bootable USB flash drive but iso file is another option presented.

7. Create your Windows 8.1 iso recovery media by following instructions.

8. Exit, boot back into Windows 8.1 on your machine. You know have recovery media to perform a "refresh" or "reset" in Windows 8.1.

9. I did a "reset" in 8.1 and I was not prompted to enter any product key. My computer was reset back to Windows 8.1.


Keep in mind I originally went from Windows 8 (with a valid and legal license) to Windows 8.1 via the App Store. The only reason why I had to do steps 1 thru 9 was because "refresh" and "restore" in 8.1 is broken. It was prompting me for recovery media, but there was none since I had upgraded from the App Store.

Let me repeat also, I'm fairly certain you can't do a fresh or new install from this iso file because all you have is your Windows 8 Product Key and this is a Windows 8.1 install media. But if you've already upgraded to 8.1 from the App store, and are having trouble using "restore" and "refresh" feature, then this provides the missing installation files. (At least it did in my case).

I also know that sometimes, even when utilizing "refresh" or "restore" users can sometimes be prompted to re-enter Product Key if your files are corrupted. If you get prompted for a product key, I'm also pretty sure this won't work since your product key is for 8 and you'll be prompted for 8.1.

ralf_g say:

"when I try and extend the first partition is says I do not have enough space on the drive."

With the Windows disk management tools you can only extend a partition into empty drive space that is immediately adjacent to and on the right hand side of the partition you want to extend. There is third party partitioning software which is more flexible (ie. Easus -http://www.partition-tool.com/download.htm).

I see both an EFI and a 20gb OEM recovery partition on drive 0. Not sure if you want to keep those or not. Unless you possess recovery discs/media for that computer deleting them will remove the ability to recover the factory original operating system and software which came with that computer. 

Deleting the EFI partition might also interfere with the computer being able to boot normally so you might want to test that first by disconnecting drive 0 temporarily to see if the computer will boot directly from the SSD drive.

Copy any data from drive 0 that you want to keep onto another drive. In Disk Management, right click each of the partitions you want to remove and select Delete Volume. The highlighted recovery partitions on drive 0 are redundant and can be removed.

That should leave you with a single block of unallocated drive space which you can then re-allocate as a single partition and format. Assign an available drive letter and name the drive.
