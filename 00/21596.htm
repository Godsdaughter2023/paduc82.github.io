steven. s say:

Hey R_ibsen,

 

The first day I responded I was able to replicate the crash as stated, but now my computer and a couple other test machines are no longer crashing. (all using Windows 7)

 

A work around, for now, would be to add the website’s address to your trusted sites list since trusted sites load with Protected Mode off.

Security zones: adding or removing websites

 

When a crash occurs in Protected Mode it is usually tied to something being incompatible withLoosely Coupled IE (a feature of Internet Explorer 8 in Windows Vista/7).

 

Let’s check your add-ons to make sure they aren’t the cause of the crashes: (you’ll need to temporarily take site out of the trusted sites list to test this)

·        Click Start

·        Click All Programs

·        Click Accessories

·        Click System Tools

·        Click Internet Explorer (No add-ons)

·        Click Tools

·        Click Manage Add-ons

·        Click Toolbars and Extensions

·        Under Show: click All add-ons

·        Click any add-on

·        Press CTRL + A on your keyboard (this should select all add-ons)

·        Click Disable all

·        Click Close

·        Close Internet Explorer (No add-ons)

·        Open Internet Explorer

 

Now all add-ons are disabled in Internet Explorer, test to see if the same problem occurs without add-ons. If the problem is fixed then the next step is to enable the add-ons one by one to find which is causing the problem.

·        Click Tools

·        Click Manage Add-ons

·        Click the first disabled add-on

·        Click Enable

·        Click Close

·        Surf a few pages

·        Close Internet Explorer

·        Open Internet Explorer

 

Test to see if the problem has returned, if not the add-on you enabled is good to stay. Repeat the enabling process for the next disabled add-on and continue this process until the crashes return. Then the last enabled add-on will need to be disabled/uninstalled/updated to avoid further problems.

 

Look forward to hearing back from you.
