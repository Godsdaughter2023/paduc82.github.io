geetesh bajaj say:

Create a Default Theme in PowerPoint 2007 for Windows

Technical Level : Basic

Summary
Learn how you can set up any Theme in PowerPoint 2007 to be the default for new presentations you create -- this can be very helpful indeed when you want to use your corporate, organization or company Theme as the default.
Details

Do you want to use a Theme other than the blank white design that PowerPoint 2007 provides by default? Even though the basic white design is a great starting point,  there are situations when you need to use the approved Theme of your company or organization for all your presentations. If that's the situation, follow these steps to create a new default Theme:

  1. Open a new blank presentation. Access the Design tab of the Ribbon.
  2. Within the Themes group, click the More button to bring up the Themes drop-down gallery.

  3. Now, bring up a context menu by right-clicking any of the Themes. Within this menu select the Set as Default Theme option.

  4. This saves the selected Theme as the default Theme.

Hereafter, whenever you choose the File | New | Blank option, or even just launch PowerPoint, a presentation will be created based on the Theme you chose to be the default.

Note: What happens if you also have a default template in addition to a default Theme? As long as a default Theme called Default.thmx is available in the designated folder, PowerPoint will use that as the default for all new presentations created. If that's not available, it will use the Blank.potx file within the designated folder.

To learn more about what's different between a default template and a default Theme -- and to learn how you can create a default template, look at the Change the Default Template or Theme in PowerPoint 2007 page (external link).

The identity for Outlook 2100 is stored un ~/Documents/Microsoft User Data/Office 2011 Identities.

If you move or rename it, Outlook looses track of it and it will start a fresh new one at the default location.

If you play around with the content of the folders that compose the identity in the Finder, you are at high risk of damaging the identity.

It's hard to provide you with clear steps at this point without knowing exactly what you did in the Finder.
