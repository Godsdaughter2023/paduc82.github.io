Hi,

Thank you for posting your query on Microsoft Community.

This issue might occur when the SIMULATED value is set as 1024x768 in the registry.

 

I suggest you to change the SIMULATED value 3200x1800 in the resistry and check if it helps.

To change the SIMULATED value in registry, follow the below path.

1) Press Windows key + r and trpe regedit and press Enter.

2

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\GraphicsDrivers\Configuration\SIMULATED_...

This simulated display was activated when my monitor sleeps or is turned off.  Change the PrimSurfSize and ActiveSize to your full resolution - in my case, 2560x1440 (make sure to edit numbers in decimal), and my windows no longer resize when sleeping.
