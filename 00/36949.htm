john_barnett say:

Hello Thomasguy,

 

By default Wordpad is set to 1.15 lines after each carriage return. To change this to 1 line do the following:

 

1. Open Wordpad

2. On the wordpad toolbar ribbon look in the Paragraph section

3. Just above the text aligment icons you will see another set of icons.

4. Click on the icon to the far right. This icon has an up and down arrow on it and is situated to the left of the Picture Icon.

5. You should now see a list of sentence spaces. 1.15 is the default

6. Click on the 1 line space.

7 If you click on the icon again, to open it, you will see that a check mark (tick) now appears next to the 1 line space.

8 When you now press the Enter key the line space should be 1 instead of 1.5

 

From this same icon you can also change the space allocated between paragraphs. The default being 10pt (point).

You apparently have the 64-bit version of Windows 7. The "C:\Program
Files" folder and tree is for 64-bit applications, and the "C:\Programs
Files (x86)" folder and tree is for 32-bit applications. The separation
is necessary the applications will often share the same file names
between the 32 and 64-bit editions, but can't use one another's versions
of the files.

I had the simillar issue. I made lots of research but no one has the exact solution. That is why I wanted to write my solution. I solved mine with the frimware update.

According to my research, WD My Passport 0730 is not working on Windows 8 and 8.1 with USB 3.0 port. But if you convert it to USB 2.0 with an USB Hub, or if you plug it using USB 2.0 port to your PC, it will work without any problem.

So, plug your external using USB 2.0 and backup your WD 0730. After that, go to WD website and get the universal frimware updater. Download it, update WD 0730 frimware using the tool.

After the frimware update, you must be able to use your WD 0730 with USB 3.0 port.
