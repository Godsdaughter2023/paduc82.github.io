Data Validation Dropdown to have only unique values - Microsoft Community

aksuresh say:

Data Validation Dropdown to have only unique values

Good day!!!

I have a named range EMPID in B7:B350 contains employee ID in sheet Data.

This range contains many duplicate entries.

I am having data validation dropdown in A1 to select from EMPID.

I defined EMPID as =OFFSET(Data!$B$7,0,0,COUNTA(Data!$B$7:$B$350),1)

What changes are to be made to the above formula to have the dropdown with only unique values without duplicates?

Thanks and regards,

AK Suresh

vijay a. verma say:

You will need to generate a unique list somewhere and then create DV on this unique list

If your range is not having a blank, you can use following formula to generate starting C7 (adjust ranges of B and C appropriately)

=IFERROR(INDEX($B$7:$B$17,MATCH(0,INDEX(COUNTIF($C$6:$C6,$B$7:$B$17),0,0),)),"")

If your range is having blank, use below formula to generate unique entries

=IFERROR(INDEX($B$7:$B$17, MATCH(0, IF($B$7:$B$17<>"",COUNTIF(C6:$C$6, $B$7:$B$17)), 0)),"")

vijay a. verma say:

Additionally, if you are using unique formulas, you will need to replace COUNTA(Data!$B$7:$B$350) part in your EMPID with following as COUNTA treats "" (blank literals as non blanks.

SUMPRODUCT(--(Data!$B$7:$B$350<>""))

ashish mathur say:

Hi,

We will have to create a unique list of entries in a separate range first.  If you are looking for a non formula based solution, then we can use Power Query.  Power Query has a feature to Remove Duplicates.  Once we get the unique entries to a range, we can create the validation.

amit tandon say:

1. Named range EMPID, Refers to formula should be as below to accomodate any blanks (presuming last row in column B with data is > 7):

=OFFSET(Data!$B$7,0,0,SUMPRODUCT(MAX((ROW(Data!$B$1:$B$350))*(Data!$B$1:$B$350<>"")))-ROW(Data!$B$7)+1,1)

2. Create a unique list starting from cell C7 downwards - enter below formula in cell C7 as an array formula & copy down:

=INDEX(EMPID,MATCH(0,IF(EMPID<>"",COUNTIF($C$6:$C6,EMPID)),0))

3. In cell D7 create Data validation dropdown by creating a List & enter below formula in Source:

=OFFSET($C$7,0,0,MATCH("#",$C$7:$C$350,-1),1)
