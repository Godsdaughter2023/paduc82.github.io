jane lla say:
Hello Jeterp,

That sounds about right. Although please be advised that MBps and Mbps are two different things.

Task Manager uses Mbps (Megabits per sec). Steam uses MBps (Megabytes per sec) hence the difference in numbers, but technically if you're going to compute it, it will just give you the same result.

1 Megabyte = 8 Megabits. 5 Megabits per second (5 Mbps) should be equivalent to 625 Kilobytes per second (625 KBps). If you are getting around 400 KBps download speed in any app, that doesn't necessarily mean that it is the only network speed being used up. These apps may also have other background processes which allows the application to communicate with their servers or for other purposes aside from the current download you're running.

For assistance in converting Bytes to bits, we suggest using Bing to convert the units.

Teredo is a tunneling protocol designed to grant IPv6 connectivity to nodes that are located behind IPv6-unaware NAT(Network address translation) devices. It defines a way of encapsulating IPv6 packets within IPv4 UDP(User Datagram Protocol) datagrams that can be routed through NAT devices and on the IPv4 internet.

You may most probably have to enable IPv6 and check.

 
Method 1:
 
To enable IPv6 :
a. Go to your network connections under control panel and
b. Go to the properties menu for the connection that you want to config.
c. It will probably just be the Local Area Connection.
d. Enable the check box that says "Microsoft TCP/IP Version 6".
e. If you don't have that option, click the "install" button, choose protocols, and install it.
f. This will enable the teredo tunnel so it can tunnel through v4.

 

Disable IPv6:

Try uninstalling IPv6 on all interfaces. To remove IPv6,

a.  Click on Start and type in Network and Sharing Center

b. Select Local Area Network

c. Go into the properties for each network adapter

d. Either uncheck the box next to the protocol "Internet Protocol version 6(TCP/IPv6), that will disable it, or select it and click uninstall, which will remove it off the computer.

e. Restart the computer

NOTE: You should do this for each network connection.

While installing the IPV6 if its asking for the location then browse to c:\windows\inf and see if it installs the driver.

 

Method 2: Change TCP/IP settings: http://windows.microsoft.com/en-US/windows7/Change-TCP-IP-settings
