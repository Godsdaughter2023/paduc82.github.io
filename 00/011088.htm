ianinkl say:

I'm replying to my own post as I spent some time yesterday thinking about this problem and finally got it fixed.

here is what I did.

1.Deleted the mailbox

2. Created the mailbox from scratch.
  There are some "GOTCHAS" here - if you create a mailbox that the phone (mine is an E71) knows about the "wizard" goes ahead and completes many of the settings for you.
The only problem with this is that the wizard does NOT do these correctly for my email accounts.

Optusnet is a POP3 account. The incoming server is mail.optusnet.com.au, default ports, default security, username is usually your email address up to but NOT including the "@" - the setup wizard fills this in as you email address in full which will fail.

Similarly for outgoing mail: Optusnet is mail.optusnet.com.au, same username and password as incoming, SMTP protocol, default security.

I then set the "Reply-To" and Name settings to my preferences and all worked.

Connection is "Always Ask" as I toggle between my phone 3G network and my WiFi network or Starbucks etc

 

For Gmail it was a bit trickier - the wizard default to IMAP4 connection/mailbox but I wanted to keep it as POP3  so it functions like my laptop email. 

In this case you have to create a dummy email account - The Nokia Wizard will not recognise this and will make no assumptions asking you to complete all fields & settings your self.

Again - check carefully that the username is just your email address up to but NOT including the "@" character.

It worked for me first attempt after deleting the account thus ending 4 very frustrating weeks. I was resorting to using webmail which "sucks" big time!

I hope this helps others.

Works should not "disappear".

Check in Control Panel > Programs and Features.  Look to see if Works is still there in the list of programs.  If it is, right click on it to see if there is a "repair" option. 

But first, before doing the repair if you don't intend to buy a key to activate the 2010 trial, it is just wasting space on your drive and you might was well uninstall it before repairing Works.

MS has taken down all download links I knew of to get the Works installer.


If that doesn't work, the only other option I can suggest is to use the recovery partition to reset your computer to "factory fresh".  Works, and probably the 2010 trial should be there.

NOTE:

Using the recovery partition will wipe out ALL files on the drive. So BEFORE using recovery, make sure to create a full image copy backup to an external drive.  Also, copy your personal document files to a separate location so you can easily copy them back onto the computer after doing the recovery.
