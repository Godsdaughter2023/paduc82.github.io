This is a Windows 7 feature, not one specifically for Word. The Word option you changed affects only the list that appears in Word's File (or Office button) list, not the Windows Start menu.

To turn off the display of recent documents (for all programs, not just for Word):

- Click the Start Menu tab of the properties dialog.

- Uncheck the "Store and display recently opened items in the Start menu and the taskbar" box.
