rich007a say:

Hi Steve,

Is "last-touched" date the same as .LastModificationTime?

Try this:

Sub TouchNotes()
    Dim ns As Outlook.NameSpace
    Dim ntsFldr As Outlook.Folder
    Dim nt As Outlook.NoteItem
   
    Set ns = Application.GetNamespace("MAPI")
    Set ntsFldr = ns.GetDefaultFolder(olFolderNotes)
    For Each nt In ntsFldr.Items
        Debug.Print nt.Subject, nt.LastModificationTime
        nt.Body = nt.Body
        nt.Save
        Debug.Print nt.Subject, nt.LastModificationTime
    Next nt
End Sub

If the above has an effect on the sort order in your phone, you'd just need to set up an array to contain the .Subject of each Note, then grab Chip's QSortInPlace procedure from http://www.cpearson.com/excel/SortingArrays.aspx  to sort the .Subjects alphabetically and then run the above For Each nt... loop.  You can access a Note item directly with it's subject (from your sorted array):

    Set nt = ntsFldr.Items(strSubject)

young yang msft say:

Hi Marc,

I find the issue is resolved. Here is the closure summary.

 

Issue Description:

================

When external mailboxes send emails to an Exchange Online public folder, an NDR bounced with the error: 550 5.4.1 Recipient address rejected: Access denied.

 

Root Cause/Analysis:

================

When the domain type is set to Authoritative, Directory Based Edge Blocking (DBEB) allows any SMTP address that has been added to the service, except for mail-enabled public folders.

 

Resolution:

================

Keep the domain type as Internal Relay.

 

Thanks,

Young
