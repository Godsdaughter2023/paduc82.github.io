macropod say:

I've never used Salesforce or its Conga word template. Having looked at the Salesforce website, though, it seems you might be doing a mailmerge of some kind. In that case, what you need to test is probably not the checkbox but the data field that sets its value to true or false. That would be done via field coding in the mailmerge main document, not via a macro. Assuming your Conga word template allows you to use Word's normal field coding, such a field might be coded along the lines of:

{IF{MERGEFIELD Field_A}= True "Descriptive Text"}

or:

{IF«Field_A»= True "Descriptive Text"}

where 'Field_A' is the name of the data field you want to test.

Note: The field brace pairs (i.e. '{ }') for the above examples are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practical to add them via any of the standard Word dialogues. Likewise, you can't type or copy & paste the chevrons (i.e. '« »') - they're part of the actual mergefields, which you can insert from the 'Insert Merge Field' dropdown. The spaces represented in the field construction are all required.

As for screen sharing, no can do.

If you need specific advice about Salesforce products, I suggest you contact Salesforce support. I note that they have an FAQ page dealing with IF statements and the like at: https://www.congasphere.com/notice/WordTemplates.htm

Chip Pearson has a couple of macros that you can use to unselect a cell or groups of cells.

 

Sub UnSelectActiveCell()
Dim rng As Range
Dim FullRange As Range
'Chip Pearson
If Selection.Cells.Count > 1 Then
    For Each rng In Selection.Cells
        If rng.Address <> ActiveCell.Address Then
            If FullRange Is Nothing Then
                Set FullRange = rng
            Else
                Set FullRange = Application.Union(FullRange, rng)
            End If
        End If
    Next rng

    If FullRange.Cells.Count > 0 Then
        FullRange.Select
    End If
End If

End Sub


Sub UnSelectActiveArea()
'Chip Pearson
Dim rng As Range
Dim FullRange As Range
Dim Ndx As Integer
If Selection.Areas.Count > 1 Then
    For Each rng In Selection.Areas
        If Application.Intersect(ActiveCell, rng) Is Nothing Then
            If FullRange Is Nothing Then
                Set FullRange = rng
            Else
                Set FullRange = Application.Union(FullRange, rng)
            End If
        End If
    Next rng
    FullRange.Select
End If

End Sub
