hypochondrius say: To everyone who has been having problems with S805, I feel your pain.

My S805 headphones have always had issues when running in Vista regardless of what type of Bluetooth dongle I would use.  Fortunately, I only use them at work on a machine running XP.  However, I recently upgraded to Windows 7, which, unfortunately, does not have an existing driver that supports the S805.

This is quite a lengthy process, but, if you are desperate to make your S805 headphones work on Windows 7, you probably don't have any other choice.  You have to follow these steps in the order that they are written or it will not work.

Here's how I made it work:

1. Download the Broadcom Bluetooth driver stack here: http://www.broadcom.com/support/bluetooth/update.php
2. Right click the installation file and select Properties.
3. Click the Compatibility tab.
4. Under Compatibility mode, check the “Run this program in compatibility mode for:” option.
5. Select the Windows XP (Service Pack 3) from the available options.
6. Under Privilege Level, check the "Run this program as administrator" option.
7. Click Apply and OK.

Install the Broadcom Bluetooth driver stack:

1. Make sure your Bluetooth dongle is plugged into your computer.
2. Double-click the installer file.
3. Allow the installation to complete.
4. Restart your computer.

NOTE: You will not be able to click 'My Bluetooth Places' shortcut due to compatibility mode issues.  If you attempt to do this, Windows 7 will give you a 'Windows Explorer has closed' error.

Pair the S805 headphones:

1. Switch the S805 headphones to pairing mode (depress the call button until the blue LED light remains on).
2. Right-click on the Bluetooth icon in the notification area on the Windows taskbar.
3. Select 'Add a Bluetooth device.'
4. In the Bluetooth Setup dialog that appears, click Next.
5. Allow the search to complete.  Motorola S805 should show-up.
6. Select Motorola S805 and click Next.
7. In the Bluetooth Service Selection dialog, check 'Hands-Free unit' and 'Stereo Audio.'
8. Click Finish.

Connect the S805 Headphones via Bluetooth:

1. Right-click on the Bluetooth icon in the notification area on the Windows taskbar.
2. Select Quick Connect.
3. Select Stereo Audio Gateway.
4. Click Motorola S805.  The Bluetooth icon will turn from white to green, signifying that you are connected.

Set Your S805 Headphones as the Default Playback Device:

1. Right-click on the volume icon in the notification area on the Windows 7 taskbar.
2. Select Playback Devices.
3. On the Playback tab, select the Bluetooth Stereo Audio headphones icon.
4. Select 'Set Default' from the dropdown options.
5. Click on Apply then click on OK.

After following the above steps, your S805 headphones should now work with Windows 7.

Good luck!
