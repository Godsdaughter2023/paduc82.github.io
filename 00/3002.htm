    dhookom say: Hi William,
    You would need to use some code to loop through the selected items to build a WHERE statement. I include a generic function for this in every application I create:

    Function BuildIn(lboListBox As ListBox, _
    strFieldName As String, strDelim As String) As String
    'send in
    ' List box control object
    ' Name of the field
    ' Delimiter which is typically a single quote for text
    Dim strIn As String
    Dim varItem As Variant

    If lboListBox.ItemsSelected.Count > 0 Then
    strIn = " AND " & strFieldName & " In ("
    For Each varItem In lboListBox.ItemsSelected
    strIn = strIn & strDelim & lboListBox.ItemData(varItem) & strDelim & ", "
    Next
    'remove the last ", " and add the ")"
    strIn = Left(strIn, Len(strIn) - 2) & ") "
    End If
    BuildIn = strIn

    End Function

    You can use the results with some code to change the SQL property of your query. I also have code at https://www.tek-tips.com/faqs.cfm?fid=6099 which is a different solution but should work.

    macropod say:

    Since you'd need a wildcard Find/Replace to eliminate the space(s) following a field before a paragraph break, but you can't use ^d or ^p with a wildcard Find/Replace, it seems moot that you can't use:

    ^d^?^p

    The alternative is to use a macro like:

    Sub Demo()
    Application.ScreenUpdating = False
    ActiveWindow.View.ShowFieldCodes = True
    With ActiveDocument.Range
      With .Find
        .ClearFormatting
        .Replacement.ClearFormatting
        .Text = "^w^p"
        .Replacement.Text = "^p"
        .Wrap = wdFindStop
        .MatchWildcards = False
        .Execute Replace:=wdReplaceAll
        .Text = "^d"
        .Execute
      End With
      Do While .Find.Found
        With .Fields(1)
          If .Type = wdFieldHyperlink Then
            ActiveWindow.View.ShowFieldCodes = False
            If .Result.Characters.Last.Next = vbCr Then
              .Result.Characters.Last.Next = Chr(11)
            End If
            ActiveWindow.View.ShowFieldCodes = True
          End If
        End With
        .Collapse wdCollapseEnd
        .Find.Execute
      Loop
    End With
    ActiveWindow.View.ShowFieldCodes = False
    Application.ScreenUpdating = True
    End Sub

    Which is, of course, OS/Office version dependent in that it won't work with Office 2008.
