Handy Dandy Cell Text Bulleter - Microsoft Community

e.j. gun say:

Handy Dandy Cell Text Bulleter

Excel has no means of adding bullets to text in cells.  The little routine below will add a standard (dot-type) bullet to every selected cell.  It adds a bullet in front of the first character, and also adds a bullet right after every instance of ALT-ENTER (or chr(10)).  Put it in your Personal.xlsb and you'll have it handy when you need it.  There are probably more efficient ways to do it, but this works.

Could probably be improved to allow different types of bullets (e.g. check marks).  Probably ought to also have a "de-bulleter" to remove bullets from cells.  Maybe later! :-)

Eric

'
' This macro adds bullets to each line of text
' in a cell.  It adds a bullet in front of the
' first character, and then searches for any
' instances of ALT-ENTER (Chr(10)) and adds a
' bullet there.
'
Sub BulletText()
    Dim i As Long, j As Long
    Dim theRange As Range, theArea As Range, theCell As Range
    Dim theText As String, theChar As String
    Dim nChars As Long
'
    Set theRange = Selection
    For Each theArea In theRange.Areas
        For Each theCell In theArea.Cells
            theText = theCell.Text
            If (theText <> "") Then
                theText = ChrW(&H2022) & " " & theText
                nChars = Len(theText)
                For i = nChars To 3 Step -1
                    theChar = Mid(theText, i, 1)
                    If (theChar = Chr(10)) Then
                        theText = Left(theText, i) & _
                                  ChrW(&H2022) & " " & _
                                  Right(theText, Len(theText) - i)
                    End If
                Next i
                theCell = theText
            End If
        Next theCell
    Next theArea
End Sub

Why so much code? Here is a shorter macro for you to consider...

Sub AddBullets()
  Dim Cell As Range, Sentence As Variant, Lines() As String
  For Each Cell In Selection
    Cell.Value = ChrW(&H2022) & " " & Replace(Cell.Value, vbLf, vbLf & ChrW(&H2022) & " ")
  Next
End Sub

And, assuming the same bullet character, here is how to remove it...

Sub RemoveBullets()
  Dim Cell As Range, Sentence As Variant, Lines() As String
  For Each Cell In Selection
    Cell.Value = Replace(Cell.Value, ChrW(&H2022) & " ", "")
  Next
End Sub

Disable cell merging in excel - Microsoft Community

Disable cell merging in excel

Hello All,

Is there Any way we can Disable cell merging in excel

mike h.. say:

Hi,

Merging cells is a sure way to lead to future problems but sadly there's no setting that will stop it but you can use some VB code. Note this isn't foolproof because all a user has to do is not enable macros. ALT+F11 to opn VB editor and double click 'ThisWorkbook' and paste this code in on the right,

Private Sub Workbook_SheetSelectionChange(ByVal Sh As Object, ByVal Target As Range)
Dim IsMerged As Boolean
On Error Resume Next
IsMerged = LastSelection.MergeCells
On Error GoTo 0
If IsMerged Then
LastSelection.UnMerge
MsgBox "You're not allowed to merge cells"
End If
Set LastSelection = Selection
End Sub
