Hi,

 

You need to write your own VLOOKUP that doesn't have this limitation.

 

ALT+F11 to open VB editor, right click 'ThisWorkbook and insert module and paste the code below in. Close VB editor and back on the worksheet call with the formula

 

=MyVlookup(A1,Sheet5!A1:B100,2)

 

 

Where A1 is the lookup value, Sheet5A1:B100 is the lookup range and 2 is the column you want to return.

 

 

 

Function MyVlookup(Lval As Range, c As Range, oset As Long) As Variant
Dim cl As Range
For Each cl In c.Columns(1).Cells
    If UCase(Lval) = UCase(cl) Then
        MyVlookup = cl.Offset(, oset - 1)
        Exit Function
    End If
    Next
End Function

don cuthbert say:

On Windows 10 install media is \Sources\gatherosstate.exe. Copy that to a folder on an activated Windows 10 device.  Run it from there and it will generate an XML file GenuineTicket.xml.  This file is only of use on the same device that it was created on.  Keep it safe somewhere, labelled in a folder so you know where it came from, such as on OneDrive.

If you ever get in a situation where it does not activate again, copy that GenuineTicket.xml file to this location:

C:\ProgramData\Microsoft\Windows\ClipSVC\GenuineTicket\GenuineTicket.xml

Then run as Administrator: slmgr.vbs /ato or Restart.  Sometimes the XML file will vanish without activating Windows. If this happens, copy it back to that folder again and re-run slmgr.vbs or Restart again.

Remember that this XML file is only of use on the activated device that it was made on. It basically records that Windows on this specific device was Activated when gatherosstate.exe was run.

What is the make and model of the computer/wireless adapter?

Refer to the following suggestions and check if that helps.

Let’s change the Ethernet controller settings and check.

  1. Press Windows Key+ X key, click Control Panel

  2. Select System and Security.

  3. Select Device Manager under System.

  4. Select and expand the Network adapters from the list of devices.

  5. Find the wired network adapter and right-click on it and select Properties.

  6. Under the Power Management tab, make sure “Allow the computer to turn off this device to save power" is unchecked.

  7. Click Apply, Ok. Now check if the internet works fine.

If the above does not help, you may install the network adapter drivers in compatibility mode and check if that helps.
