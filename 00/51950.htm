bossdweeb say:

Once you have verified you are using a suitable graphics driver....

And with little background information to go on I'll suggest....

- What make and model laptop, and which games ?

- What kind of fps drop ? Fps is rarely constant, even within 5 fps, particularly if not using V-sync,

and should not be noticeable unless it is dropping below 30 . Well, not to most people (us mere humans ;D ),

but some do declare they can tell when there is a 3-5 fps change from 60 fps (without the meter running). 

If your eyes/brain are this sharp you may just have to live with it.

- A game may seem to be running well at the selected settings, but when the VRAM gets filled due to

more action (your character drives a vehicle, etc. ) it can overload the VRAM , or put more stress on the

graphics processor, which can cause the fps to suddenly drop (lag and stutter ).

If your laptop does not have a dedicated graphics processor the problem could occur because the laptop

uses part of the system RAM, which is not as fast as dedicated VRAM found on a dedicated graphics card.

If you have disabled the  Page File this could also contribute to slow downs in some games,

particularly if the laptop does not have dedicated VRAM.

- Try lowering a setting or two. Anti-aliasing would be my first go to, particularly if trying to use

more than 2X MSAA. 

Ambient Occlusion, SSAO, HBAO, etc. all have a big performance hit relative to the visual enhancement they

produce, so lowering these would be my next options.

Using very high texture options can also impact on VRAM/RAM, but I prefer higher res texture over fancy lighting,

so like to leave that until absolutely necessary.

- It may also help to enable V-sync, even if fps is not above 60, but if the fps is already below 30 using

V-sync may make things worse.

V-sync and Triple buffering sometimes need to be experimented with on a game by game basis.

- Then we have peculiarities in individual games. 

eg - Using the max lighting setting on Saints Row the 3rd over the next highest lighting setting drags

the fps down dramatically in this game.
