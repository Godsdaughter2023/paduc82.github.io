hansv mvp say:

MONTH($A$1) returns 12 (the month number). Hence the formula

=TEXT(MONTH($A$1), "mmmm")

is equivalent to

=TEXT(12, "mmmm")

The value 12, interpreted as a date, corresponds to January 12, 1900. The month of this date is January.

Instead of =TEXT(MONTH($A$1), "mmmm"), use

=TEXT($A$1, "mmmm")

This will return December.
