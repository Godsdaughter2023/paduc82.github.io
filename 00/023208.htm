Is there a way to do something similar to this in MS Project using the custom fields or by some other means?  I would basically like to insert a hyperlink column next to the column I keep the work order numbers in (Text6) and pull the work order number into the hyperlink.  Is this even possible in Project?

trevor rabey say: When you right click on any task, isn't the hyperlink option at the bottom of the menu?

tom_bpc say:

JeffGriffis,

As Trevor implies, each task already has the following fields: Hyperlink, Hyperlink Address, Hyperlink Href, and Hyperlink SubAddress, which you can populate using the dialog opened by the "Link" option in the menu.  Then you can insert these fields/columns into a table to see them.

You can also populate the fields automatically using vba.  The following code block creates the hyperlink you mention above for each task in your active project that has a value in Text6.

Sub CreateHL()

Dim t As Task


For Each t In ActiveProject.Tasks

    If Not t Is Nothing Then

        If Not t.Text6 = "" Then

            't.HyperlinkAddress = "http://www.tdg1.com/ProductionCentral/default.aspx?subject=WorkOrderDetail&WONum=" & t.Text6
        End If
    End If
Next t
End Sub

This will overwrite any existing hyperlinks, so be careful.

a. user say:

The following should work:

Set up your Heading numbering using Heading styles (1 and 2)


Apply the appropriate multi-level (outline numbered) - using the drop-down by the multi-level list numbering button in the Home Tab of the Ribbon, select the option that says

1 Heading 1

1.1 Heading 2

etc.
Insert a caption.

In the Caption dialog box, click the Numbering... button.

In the Caption Numbering dialog box, select 
Format 1, 2, 3 ...

Check Include Chapter Number

Next to Chapter starts with stye, select Heading 2

Next to Use Separator, select . (period)

The caption should be inserted. If you display the field codes that create it (option-F9 or fn-option-F9 depending on your keyboard and settings), you should see two fields like this, with a period between.

{ STYLEREF 2 \s }.{ SEQ Equation \* ARABIC \s 2 }

The first field should insert the number from the most recent Heading 2 paragraph, and the second field should insert the next number in the list, restarting with each Heading 2 paragraph.

NB, these fields do not auto-update. When the paragraph numbering changes (e.g. if you insert a Heading 1 or Heading 2 paragraph), you will need to select the relevant fields (often it's simpler to select all fields using cmd-A, especially if you do not have a Table of Contents) and update them using F9/fn-F9.
