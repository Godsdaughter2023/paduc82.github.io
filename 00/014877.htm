dale howard [mvp] say:

Michael A -

First of all, I would recommend that you DO NOT use Manually Scheduled tasks at all in your project.  Instead, convert all of your tasks to Auto Scheduled to give you the maximum scheduling capabilities in Microsoft Project.

Second, for these "Concrete Pour" tasks, I would recommend you create a special Base Calendar called something like "Concrete Pour".  On that calendar, set the working schedule so that only Tuesday and Saturday are working days, while all other days are nonworking time.

Third, assign that Concreter Pour calendar as a Task Calendar on each of the necessary tasks.  This will guarantee that they are scheduled on either a Tuesday or a Saturday.

Finally, link all of the concrete pour tasks with a Finish-to-Start dependency, beginning from the first pour to the last pour.  This process will schedule all of the tasks correctly.
