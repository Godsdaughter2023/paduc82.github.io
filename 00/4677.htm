ossiemac say:

There are 2 options below. The first one will clear the contents of each of the worksheets that have been nominated in the code. The second option will clear the contents of the active sheet only.

You can assign shortcut keys to either option but you cannot assign the same shortcut to both options; need different shortcuts for each option.

Anything you do not understand then feel free to get back to me.

Sub Test()
   'This code cycles through the nominated worksheets and clears contents of the nominated cells
    Dim strClrAddress As String
    Dim arrShts()
    Dim i As Long
   
    'Following line. Two dimensional one based array (For 4 worksheets)
    ReDim arrShts(1 To 4, 1 To 2)   'Edit 4 to number of worksheets required
   
    arrShts(1, 1) = "Target Sheet 1"    'Assigned to first dimension of array
    arrShts(1, 2) = "C2,C5,E2,E4,E6,E8,G2,I2,I4,I6,I8,I10,I12,I14"  '2nd dimension of array
   
    arrShts(2, 1) = "Target Sheet 2"
    arrShts(2, 2) = "C3,C6,E9,E11,E13,E15,G3,I5,I7,I8,I10:I12,I14"  'Note range I10:I12
   
    arrShts(3, 1) = "Target Sheet 3"
    arrShts(3, 2) = "C2:C5,E2:E8,G2,I2:I8,I10,I12,I14" 'Note ranges C2:C5, E2:E8, G2,I2:I8
   
   
    arrShts(4, 1) = "Target Sheet 4"
    arrShts(4, 2) = "C4,C6,E2,E4,E6,E8,G2,I2,I4,I6,I8,I10,I12,I14"
   
   
    For i = 1 To 4
        Worksheets(arrShts(i, 1)).Range(arrShts(i, 2)).ClearContents
    Next i
   
End Sub

Sub Test2()
    'This code clears the contents of nominated cells in the ActiveSheet only
    Dim ws As Worksheet
   
    Set ws = ActiveSheet
   
    Select Case ws.Name
        Case "Target Sheet 1"
            ws.Range("C2,C5,E2,E4,E6,E8,G2,I2,I4,I6,I8,I10,I12,I14").ClearContents
       
        Case "Target Sheet 2"
            ws.Range("C3,C6,E9,E11,E13,E15,G3,I5,I7,I8,I10:I12,I14").ClearContents
   
        Case "Target Sheet 3"

            ws.Range("C4,C6,E2,E4,E6,E8,G2,I2,I4,I6,I8,I10,I12,I14").ClearContents
    End Select
 
End Sub

I would suggest following modifications to formula of JP Ronse -

As WS1 contains order# in column C not in column A and also since FIND is case-sensitive and OUT is wanted

Hence, the formula should read as

=IF(SUMPRODUCT(('WS1'!C2='WS2'!$A$2:$A$11)*ISNUMBER(SEARCH("out",'WS2'!$D$2:$D$11))),"OUT","")

patricia yu say:

You can disable the integrated camera by disabling the device in Device Manager.

Here's how:

  1. Open Device Manager. Click Start and enter Device Manager into the search field.
  2. On Device Manager, locate Imaging Devices and double-click on your webcam to open Properties.
  3. Click on the Driver tab, and click on Disable to disable your webcam.
    Should you want to re-enable it, follow the same steps and select Enable in the same window.
