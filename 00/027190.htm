a. user say:

Hello Maria,

Thank you for your post.

In yammer you cannot stop the user to create the internal public or private group.

As Yammer is a social networking website developed at organizational level to share the messages and files with the colleague's.

Configure Exchange transport rule in Exchange admin center.

In order to restrict users from creating external groups and inviting external members we can configure a transport rule in Exchange admin center.

Create a Rule in Exchange Online admin center by following below steps.

  • Login to Exchange online admin center.

  • Navigate to Exchange Admin center> Rules

  • Click on + 

  • Click on "create new rule"

  • Click on "More option"

  •  Email with sender address as "*** Email address is removed for privacy ***, *** Email address is removed for privacy *** and *** Email address is removed for privacy ***" and the recipient located outside the organization.

  • Under "do the following "Deliver the message to hosted quarantine"

  • Click on Save.

Note: This will disable only the external messages from Yammer and will not affect other external messages.         

  • lease follow below steps to enable "Exchange Transport in Yammer"

  1. Please Login to Yammer as Global or verified Administrator
  2. Click on the gear icon on top and click on Network admin
  3. Click on security settings.
  4. Check Mark on "Enforce your Exchange Online Exchange Transport Rules (ETRs) in Yammer"
  5. Click on Save.

https://support.office.com/en-us/article/Control-external-messaging-in-a-Yammer-network-with-Exchange-Transport-Rules-f8fd6403-c8f3-4307-9230-65304d6000d9

  • Once the Exchange Online Transport rule is created you can check after 30 Minutes and see if you are able to create an external group from yammer. If you are not able to create an external group from yammer than the rule is successfully configured.

If its allowing the users to create the external group then kindly create a service request with Yammer support team to stop those settings in your network.

stefan blom say:

If heading numbering has been correctly set up from the start, there should be no reason to apply manual restarts. To fix the numbering, click in the first Heading 1 paragraph of your document and click Home tab | Multilevel List | Define New Multilevel List. Define numbering appropriately, level by level, following the instructions in the article to which Doug has already provided a link.

If you want to restart numbering because you need both heading and appendix numbering, you can use some of the un-used heading styles (such as Heading 7 and Heading 8) for the appendix. See http://www.shaunakelly.com/word/numbering/numberingappendixes.html.
