Warning: In current revisions of Windows 10, Cortana and the standard file search functionality you're used to have been completely integrated and are now one and the same. Deleting Cortana will delete the file search functionality for Explorer and the Start Menu. If you want to search for files and folders after this procedure, you'll have to use a third-party tool. If you're open to suggestions, I recommend "Classic Shell"; aside from restoring the file search functionality to machines stripped of Cortana, it also replaces your Start Menu with a highly customizable alternative, and allows some limited Taskbar customization as well. Also, Cortana is technically a system app, so if you do this, there may be other consequences for all I know. *shrug*

  1. Go to "C:\Windows\SystemApps."
  2. Right-click the folder that has "Cortana" in the name and select "Properties."
  3. Under "Security," go to "Advanced."
  4. Change the "Owner" property from "TrustedInstaller" to your username.
  5. Check "Replace owner on subcontainers and objects" and click OK.
  6. Open the same dialogue back up ("Properties" -> "Security" -> "Advanced").
  7. Click "Change permissions," and then "Add."
  8. Type your username and press Enter.
  9. Check "Full control," and click OK.
  10. Click "Enable inheritance" and check the checkbox underneath it. (Not sure if this step is actually necessary, but it won't hurt since you're going to delete the folder anyway.)
  11. OK out of the "Properties" dialogue.
  12. Open a command prompt in "C:\Windows\SystemApps."
  13. Type the following command: rename name_of_folder_that_has_Cortana_in_it TO_DELETE, replacing name_of_folder_that_has_Cortana_in_it with the name of the folder whose n the command yet.
  14. Open the Task Manager (Ctrl+Alt+Del, select Task Manager).
  15. Find the Cortana process. The Cortana Background Task Host process should be right underneath it. enough and should proceed from step 13.
  16. Select the newly renamed folder "TO_DELETE," and press Shift+Del. Answer "Yes" to the permanent deletion prompt.
