Split the macro into 2:

The 1st opens the files and reads the values into a global variable => Copy

The 2nd reads the values from the variable and writes into the sheet => Paste

Andreas.

Option Explicit

Dim Values As New Collection

Sub UpdateCopy()
  Set Values = New Collection
  If Not UpdateCopyPrim("ABC") Then Exit Sub
  If Not UpdateCopyPrim("DEF") Then Exit Sub
  UpdatePaste
End Sub

Private Function UpdateCopyPrim(ByVal SheetNameOrIndex) As Boolean
  Dim vFile As Variant
  Dim wbCopyFrom As Workbook
  Dim wsCopyFrom As Worksheet
  Dim ThisAddress As Variant
  On Error GoTo ExitPoint
  vFile = Application.GetOpenFilename("Excel Files (*.xl*)," & _
    "*.xl*", 1, "Select Excel File", "Open", False)
  If VarType(vFile) = vbBoolean Then Exit Function
  Set wbCopyFrom = Workbooks.Open(vFile, False, True)
  Set wsCopyFrom = wbCopyFrom.Worksheets(SheetNameOrIndex)
  For Each ThisAddress In Array("c11", "c12", "c13")
    Values.Add wsCopyFrom.Range(ThisAddress).Value, _
      SheetNameOrIndex & "!" & ThisAddress
  Next
  UpdateCopyPrim = True
ExitPoint:
  If Not wbCopyFrom Is Nothing Then wbCopyFrom.Close
End Function

Sub UpdatePaste()
  On Error GoTo ExitPoint
  Range("b3").Value = Values.Item("ABC!c11")
  Range("b6").Value = Values.Item("ABC!c12")
  Range("b7").Value = Values.Item("ABC!c13")
  Range("c3").Value = Values.Item("DEF!c11")
  Range("c6").Value = Values.Item("DEF!c12")
  Range("c7").Value = Values.Item("DEF!c13")
ExitPoint:
End Sub

What if I wanted to add ABC!c12 and ABC!c13 and paste this into Range ("b6")?

The collection returns the values like this

  x= Values.Item("ABC!c12")

  y= Values.Item("ABC!c13")

and you can add the variables like

  Range("b6").Value = x + y

The shorter version is
  Range("b6").Value = Values.Item("ABC!c12") + Values.Item("ABC!c13")

BTW, the Item property is the default property of a Collection, so this works too

  Range("b6").Value = Values("ABC!c12") + Values("ABC!c13")

BTW, the Value property is the default property of a Range, so this works too

  Range("b6") = Values("ABC!c12") + Values("ABC!c13")

If you want to combine strings into a cell with a line break:
  Range("b6") = Values("ABC!c12") & vbLf & Values("ABC!c13")
