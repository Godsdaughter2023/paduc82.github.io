bill smithers say:

Hi,

You can not create a restore point today for yesterday, last week or last month, etc.

That is past history and you can not re-create those settings now.

Either the Restore Points date/s you want are already made and are available or they no longer exist due to limitations on the amount of space allocated to System Restore.

You make a restore point today to use at some time in the future to revert back to today's date and computer settings from tomorrow, next month, etc. if you were to have a problem in the future.

Error 711: Cannot Load Remote Access Service Connection Manager

When trying to create a new network connection, one of the more commonly encountered errors is the Error 711. Error 711 is an error message that is displayed when the system fails to start one or more of the following required services:

  • Plug and Play
  • Remote Access Auto Connection Manager
  • Remote Access Connection Manager
  • Telephony

As a direct result of this error, Windows 7 computer fails to establish a connection with the Internet, and also disallows the user to install any additional network adapter(s).

Fortunately though, fixing Error 711 in Windows 7 is easy and quick. This article describes three methods on how to fix Error 711 in Windows 7, and ensure a smooth connection to the Internet.

Manually enable the required services

  1. Open the “Start” menu located at the bottom left and type “services.msc”.
  2. Right-click on “services” and then choose “Run as administrator”.
  3. Click “Yes” if prompted by the User Accounted Control dialog box and continue.
  4. Search and locate “Remote Access Auto Connection Manager”
  5. Right-click on “Remote Access Auto Connection Manager” and then click on “Properties”.
  6. Look for the General tab and select “Manual” from the Startup type drop-down menu.
  7. Click on “Apply” to save and apply your changes.
  8. Exit the Remote Access Auto Connection Manager by clicking on “OK”.
  9. Now, locate “Telephony” in the right pane of the “Services” window and right-click on it.
  10. Click on “Properties” to enter properties of Telephony.
  11. Under the General tab, select “Manual” as was done in Step 6, and then click “Apply” to save your changes
  12. Exit Telephony by clicking on “OK”
  13.  Repeat Steps 4,5,6,7 & 8 for enabling “Plug and Play” and “Remote Access Connection Manager”.

Check and Correct your Network Settings

One of the most commonly found causes for Error 711 is incorrect network settings. Follow the steps below and make sure your settings are properly entered:

  1. Open “Start” menu and click on “Control Panel”.
  2. Locate Network and Internet and click on Network Settings and Tasks found under it.
  3. Right-click on your Internet connection (e.g. Local Area Connection 1) and click on Properties
  4. Locate and click on Internet Protocol Version 4 (TCP/IPv4) and then click on Properties to enter its properties.
  5. Check your IP Address and make sure it is correctly entered. Note that your LAN IP Address always starts with 192.168.1.x. Contact your Internet Service Provider and ask for your IP Address.
  6. Click Close to close all boxes.

Delete a Specific Registry Key

Follow the exact steps mentioned below carefully, and delete the specific registry key:

  1. Press Windows Key + R to open Run.
  2. Type Regedit to enter the windows registry editor.
  3. Browse to the following path:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RasMan\PPP\EAP

  1. Delete all sub-keys after entering into the above hive
  2. Close the registry editor
  3. Restart your Windows

trekdozer say:

Custom Clean Install Steps  - With Hard Drive Format

Times to use:

Moving from XP to 7, or unsupported in place upgrade paths (ex: Vista Home to 7 Pro)

Moving from a 32 bit Windows system to a 64 bit system, or changing languages

Don't want to do an in place upgrade,


You can use an Upgrade or Full Windows 7 disk to do a custom clean install. This will remove all your programs, but you can save your files and settings to external storage before doing so. You will have to reinstall all your programs after installing Windows 7. Here are the steps:


1. Run Windows Upgrade Advisor to see if there are any known issues that might affect the installation and whether you can install the 32 or 64 bit version of Windows 7.

http://windows.microsoft.com/en-US/windows/downloads/upgrade-advisor


2. Backup all your files and settings to avoid losing docs, photos, and other information. Windows Easy Transfer is a free tool to copy files (not programs) from your PC, and then transfer them back after you install Windows 7. If you don't want to use Windows Easy Transfer, you can copy your files to a USB drive, or CD/DVD. Note: Easy Transfer will not work when changing languages, or when going from 64 bit to 32 bit (it will work going from 32 bit to 64 bit).

http://windows.microsoft.com/windows-easy-transfer


3. Locate the installation disks and any associated product/license keys for all your programs because you'll need to manually reinstall all programs. If you downloaded some programs from the Internet you can redownload them.


4.  You can format a partition with the Windows 7 installation DVD:

  1. Boot from the DVD

  2. Click Install Now

  3. At the setup screen, click Custom (Advanced)

  4. Click Drive Options

  5. Select the partition(s) you want to format - make sure you chose the CORRECT partition

  6. Click Format - this will delete EVERYTHING on that partition

  7. Create a new partition (if needed) and select a partition to install Windows on.

  8. Continue with install.


5. After Windows 7 is installed you can install your drivers,  and use Windows Easy Transfer (or another backup method you used) to restore your files & settings, and use your installation disks to reinstall all your programs. Then you're done - enjoy Windows 7.
