Downgrade Procedure - Microsoft Community

sudamlangote say:

Downgrade Procedure

I have windows 8 pro License and want to downgrade to windows 7 Enterprise.

What will be the Procedure.

shenan stanley say:
  1. Obtain/locate your legitimate Windows 7 Enterprise media and product key.
  2. Install Windows 7 Enterprise clean with your Windows 7 Enterprise product key.
  3. Update device drivers and install all software.
  4. Use your system with it's freshly installed Windows 7 Enterprise Edition operating system.

Windows 8/8.1 Pro cannot go to a 'higher' SKU in the previous OSes (such as "Enterprise") like you want:
http://download.microsoft.com/download/6/8/9/68964284-864d-4a6d-aed9-f2c1f8f23e14/downgrade_rights.pdf

See #4 in that document, "What are the downgrade rights for Windows 8.1?"

 

You can 'downgrade' to "Windows 7 Professional" from your "Windows 8 PRO" current install (or even "Windows Vista Business" if so inclined.)

 

The steps are still essentially the same...

  1. Obtain/locate your legitimate Windows 7 Professional (or Windows Vista Business) media and product key.
  2. Install Windows 7 Professional (or Windows Vista Business) clean with your Windows 7 Professional (or Windows Vista Business) product key.
  3. Update device drivers and install all software.
  4. Use your system with it's freshly installed Windows 7 Professional (or Windows Vista Business) Edition operating system.

Hope that helps!

To disable test-signing, use the following BCDEdit command using the command prompt:
