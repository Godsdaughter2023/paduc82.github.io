shane devenshire say:

You can select the entire pivot table by:

1. Placing the mouse alone the top edge of the cell at the top left corner of the pivot table and when a black arrow down facing arrow appears click.

2. Place the mouse along the left edge of the same cell as above and click when a black right facing arrow appears.

3.  Put your cursor anywhere in the pivot table and choose PivotTable Tools, Options, Select, Entire Pivot Table.

4.  Add this last command to the QAT by right click it and then in the future you can select the entire pivot table with 1 click.

5.  If there are no report filter fields put your cursor in the pivot table and press Ctrl+A or Ctrl+*

 

If this answer solves your problem, please check Mark as Answered. If this answer helps, please click the Vote as Helpful button. Cheers, Shane Devenshire

Sub PTSelectAndSetPR()
    Range("A5").CurrentRegion.Select
    Selection.Resize(Selection.Rows.Count - 5).Select
    ActiveSheet.PageSetup.PrintArea = Selection
End Sub
