neha singh msft say:

Hi Jody,

 

When you say 6 periods, by that if you mean 6 class notebooks and you want to create a new section in 6 class notebooks for different students in each class without duplicating the content for each notebook. If this is what you are looking for, then I would like to explain that it is not possible to achieve at once. You have to repeat the following process 6 times for each class.

 

         1. Let’s say, you have an “Assignment” section with content in Biology notebook, which you would like to distribute to your students’ notebooks.

 

       2.From the Distribution page under Class Notebook, you will select the section where you want to duplicate the “Assignment page”

 

 

Once you select the section in the student’s notebook the content section “Assignment” will be copied to their respective notebooks in Biology Class. Now you have to repeat this process for every class.

 

Let me know if something is misunderstood.
