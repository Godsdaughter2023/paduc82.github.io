This seems to be an issue with the Indexing feature not working in Office 2016. To resolve the issue we need to rebuild the Indexing. Since you have mentioned about Outlook Indexing errors, let us begin with the Outlook application and rebuild the Indexing in Outlook.

 

Follow these steps to rebuild Indexing in Outlook 2016:

 

  1. In Outlook click on File and then then click on Options.
  2. In the Outlook Options window click on Search and then under Sources click on the tab Indexing Options.
  3. In the Indexing Options window click on Advance.
  4. Under Index Settings click on Rebuild.
  5. As Indexing takes a long time, allow Outlook to rebuild the Indexing for about 12 to 24 hours and verify the result after about 24 hours.

Outlook hangs when opening a message - Microsoft Community

diane poremsky mvp (slipstick.com) say:

Outlook hangs when opening a message

When Outlook hangs while opening a message it often means the message contains external content and  Outlook is having problems contacting the external server or the web server is slow. 

Changing your Download External content settings at File > Options > Trust Center > Automatic Download, to prevent the automatic download of external content by default can help. 

My own preference is to not automatically download external content using these two options.

  • Don't download pictures or other content automatically in HTML e-mail option.
  • Warn me before downloading content when editing, forwarding, or replying e-mail.

By not downloading external content, Outlook will load the messages faster and "web bugs" will not load, preventing the sender from knowing if you viewed the message. 

See this Microsoft article for other causes: "Outlook not responding" error or Outlook freezes when you open a file or send mail

If Outlook displays a status bar message similar to: contacting:\\domain.com\ui\v1\icons\, the problem is a missing https: in the link to the external content. Outlook tries to contact a network server, not a web server, and appears to hang until it times outlook in 2 minutes (unless you have a higher timeout setting). 

Note: changing your Download External content settings won't help when Outlook hangs due to the missing HTTPS protocol.

If you only have an occasional message with the missing HTTPS protocol, waiting for Outlook to time out is the easiest (although annoying) solution. If it's a frequent occurrence, you can use a HOSTS file, convert the message to plain text, or remove the 'web bug' using a macro. I have more information on these solutions in this article:  Outlook hangs while opening messages
