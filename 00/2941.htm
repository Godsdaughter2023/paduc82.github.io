karl timmermans (outlook mvp 2012-2018) say:
>>
I intend to get Office 365. Can I just install Office 365 without uninstalling Office 2016? If I proceed with this will there be any problems with my emails and the email settings that are already in Outlook of Office 2016.
Or do I need to make a backup of the emails in a pst file.
<<

There are several pieces to all of this

#1 - Should you uninstall any existing version of Office? Highly recommended with the following caveats

a) if you uninstall Office through the conventional method (Control Panel - Programs), the your existing Outlook profile containing your email accounts and data files will remain in tact.

b) <IF> you use the MS uninstall <Fix-It> tool, the Outlook profiles will also be removed which potentially adds another problem if your have IMAP email accounts installed. Imap accounts use OST files to store the data locally including contact/calendar info. These .ost files can only be opened/read by the "original" Outlook profile and email account used to create the file. This is not a problem for email folders but data in contact/calendar folders will be lost or if you have a copy of the OST file, you would need to purchase a 3rd party OST2PST conversion utility.

#2 - Should you need to make backups?

Always a good idea to maintain backups of data as a normal function. In the case of POP3 email accounts, make a copy of the PST file associated with the account.

If you have IMAP accounts and you have any "This Computer Only" folders (i.e. contacts/calendar) in the IMAP folder tree, you absolutely want to back up (export) those folders to a PST file as a precaution. Contact/calendar data is not sync'd with the server.

If you're using Exchange accounts - nothing to worry about since all items types (i.e. email, contacts, calendar) are sync'd with the server.
