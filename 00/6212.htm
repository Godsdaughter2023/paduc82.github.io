rohn007 say:
By "full" license, I hope you mean a "Retail" or "FPP" Boxed CD.

In that case you have no problem, because the retail license allows install on up to 2 machines at the same time.  The no 'uninstall' is not a problem for another reason.  Uninstall does not actually contact the activation system.  The uninstall requirement is for licensing purposes, to meet the allowed number of installations. A "dead" computer qualifies as "uninstalled" equivalent.

So, all you have to do is pop the installation disk in the new computer. Run the setup program. Part way through you will be offered the option of activating it. Select Internet activation.  If this is only the second machine it is being installed on it should work no problem.  If you have had it installed on 2 machines at the same time in the past, you will have to do a phone activation.  When you phone in, ignore the initial prompts (they will give the same error as internet activation did).  Eventually you will be offred the option of talking to a live service rep. At that point, explain that you are transferring the license from a "dead" machine.  As long as you have a license type that allows transfers they should allow the activation to complete.
