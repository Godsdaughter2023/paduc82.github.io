aaron rykhus - msft say:

I did. Close PowerPoint and change the default printer to something else like Send to OneNote or something generic that's not a physical printer (e.g. PDF, OneNote, XPS printer, etc...). I saw the issue on a Windows XP SP3 box running Office 2010 with HP Color LaserJet 4550 set as the default printer. It only occured for one user on the machine, didn't happen for other users but did happen on other machines for that user (roaming profile). In this case there was only one printer installed for that user so we deleted the printer object from Printers and Faxes and that solved the problem.

aaron rykhus - msft say: One more note on my last reply, if you determine that the default printer is the cause, you can check with the print driver manufacturer for an updated driver (e.g. check support.hp.com) for updated driver from HP in my case. You could also run Windows Update and choose Custom scan, then look in Optional updates for an updated print driver.
