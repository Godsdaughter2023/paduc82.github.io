Hi

 

Just loop

 

Sub BubbleChartLabels2()
'
' BubbleChartLabels Macro
'
' Keyboard Shortcut: Ctrl+Shift+L
'
   Dim ch As Chart
   Dim x As Long, y As Long
   For x = 1 To ActiveSheet.ChartObjects.Count
      Set ch = ActiveSheet.ChartObjects(x).Chart
      With ch
         For y = 1 To .SeriesCollection.Count
            With .SeriesCollection(y)
               .ApplyDataLabels
               .DataLabels.ShowSeriesName = True
               .DataLabels.ShowValue = False
               .DataLabels.ShowBubbleSize = True
            End With
         Next y
      End With
   Next x
End Sub

    Data roaming should be turned on for it to work. Wifi setting is irrelevant if you are not going to use it. Even if you leave it on the phone will not automatically use available networks unless you have previously connected to them, i.e. done the required steps to register/log in/use the service.

    The recipients of your texts will only incur expense if they are located in a roaming location without free roaming. Where you are sending the texts from is completely irrelevant. Their location when receiving them will define whether it will cost them anything or not.

    I would recommend that you turn roaming completely off when at sea, because if you are sailing close to land the phone might pick up a foreign network which is NOT European - depending on how close you will come to North Africa, for example - and that could cost a lot in non-free roaming. The alternative is to set your network selection to Manual instead of automatic - refer to your device user guide for details about settings.
