When things start to get long and potentially complicated I like to write a vb function that behaves exactly how I like.

Here's an example that's based on what I think you're after:-

 

Function ExtractLetter(Param1 As String, _
                       Param2 As String, _
                       RtnLetter As String) As Boolean
                      
On Error GoTo ErrHdlr

    RtnLetter = Mid(Param1, InStr(1, Param2, "-", 1) - 1, 1)
    ExtractLetter = True

Exit Function

ErrHdlr:
    ExrtractLetter = False
End Function

 

AND here's a little test routine to show it works:-

Sub Test()
   
Dim MyLet As String

    If ExtractLetter("1234", "1-2", MyLet) Then
        MsgBox "Extracted: " & MyLet
    Else
        MsgBox "Invalid call to ExtractLetter"
    End If
End Sub

 

If you remove the - from "1-2" ExtractLetter returns FALSE and the test prints "Invalid call to ExtractLetter" otherwise it just reports the letter it extracted. HTH
