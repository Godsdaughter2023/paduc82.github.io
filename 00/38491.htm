Step 1:

a. Press Windows key + R and type services.msc in the search field and press enter.

       C:\Windows\system32\spool\PRINTERS

e. Delete all files in the PRINTERS folder until it is empty, and then close Windows Explorer.

f. Open Services, and start the Print Spooler service.

hairyfool say:

It varies according to the computer manufacturer but most modern computers have a boot menu option. During the initial BIOS phase of startup pressing one of the F keys will list all your boot options irrespective of the boot sequence set in the BIOS.

Asus (Award BIOS) use F8, Dell F12, HP F9 as examples

This shows all bootable devices detected by the BIOS, Hard Disks, Optical Drives (even if no disk is in the drawer) and USB devices. If the USB drive is not listed then it is not bootable. Windows Setup only needs to load once to run and transfer the image to the hard disk, any subsequent boot would be from the HDD so the boot menu is particularly useful for that purpose.

Either the drive not being properly configured or the BIOS not set up for USB boot are the commonest reasons for the setup USB key being ignored
