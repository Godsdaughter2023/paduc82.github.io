Excel Last Modified - Microsoft Community

angela319 say:

Excel Last Modified

When opening an excel document and closing it without making any changes, why does the last modified date change to the current date? 
vijay a. verma say:
This issue occurs because when you open an Excel workbook on a shared drive, Excel writes the name of the current user to the header of the file. This is necessary so that other users receive the "file in use" notification. The operating system then updates the last modified date.
That is why even if you don't save it Last Modified Date gets changed.

Although you have done some troubleshoot,  but maybe they are not completely. Please follow the steps below to troubleshoot the issue on your MAC to check whether you can access the OneNote

  1. Close all Office 365 for Mac applications.
  2. From Finder Go/Go to Folder (Shift+Cmd+G).
  3. Enter "~/Library/Containers" and press Go (Return key).
  4. Find the "com.microsoft.onenote.mac" folder.
  5. Select that folder Move to Trash (Cmd+Delete).
  6. Select and Move to Trash (Cmd+Delete) this folder: ~/Library/Group\ Containers/UBF8T346G9.Office.
  7. Open Spotlight by selecting the magnifying glass at the top right of your desktop.
  8. Search for and open Keychain.
  9. In the search field, enter ADAL.
  10. Select all items whose type is MicrosoftOffice15_2_Data:ADAL:<GUID>, and then Move to Trash (Cmd+Delete).
  11. In the search field, enter office.
  12. Select the items that are named Microsoft Office Identities Cache 2 and Microsoft Office Identities Settings 2, and then Move to Tras
