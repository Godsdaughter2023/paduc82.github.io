Basically you will need to create a template of a full sheet of labels instead of a single label which would be the approach if you are mail merging. Then you would manually type in the addresses on each label you want to print.

I would suggest that you reconsider this approach. As much as it seems to be frugal, I have read of many people who jam up their printers by trying to run a label sheet through the printer multiple times. The labels peel off in the printer and it is really hard to fix the printer jamb.

With all this said, I think I have read where you can do what you want with Word. It has more more options, so you might post the same question in the Word forum.
