As DaveM121 wrote, changing the zoom is the most efficient way to get a one-page view in Word 2007. Note that 100% may not suffice, depending on your display size and resolution.

To automate the process just a bit, you can run a macro as follows:

Sub AutoOpen()
On Error GoTo errhandler
With ActiveWindow.View
.Type = 3 'Print Layout view
.Zoom.PageColumns = 1
.Zoom.Percentage = 100  'specify the zoom you want
End With

errhandler:

Exit Sub

End Sub

Internet Explorer 11.0.96000.16IS and 11.0.9600.16476 is the same Internet Explorer version. Internet Explorer 11.0.96000.16IS is nothing but a Customized name of the same version.

 

All versions of Internet Explorer 5.0 (and later versions) that are customized with Microsoft Internet Explorer Administration Kit (IEAK) include one of the following strings after the version number.

 

IC = Internet content provider
IS = Internet service provider
CO = Corporate administrator

Information about Internet Explorer versions.
