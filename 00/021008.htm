I already had TextExpander installed on my Mac so I created a note with what I wanted in my template. Then I selected and cut that and pasted it into a new snippet in TextExpander (set to formatted text, not plain text).  Now I can create the new page and type in the keyboard shortcut and I get exactly what I wanted.  It's an extra step and you have to have TextExpander but it works.

Since you already have a template defined from a Windows version of OneNote (or it sounds like that) you could create a new page on a Windows workstation and then open it on your Mac to get the text.  That would save typing it again.
