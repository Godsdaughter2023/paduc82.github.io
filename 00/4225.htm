phillip m jones say:

Here is a Suggestion that might help at least with Operation after initial opening. It has helped in my case.

I use a MacBook Pro 17"

8GB System RAM

2.8GHz i7 Processor

Yosemite 10.10.5 System

Originally had these issues:

  1. Slow start up
  2. Herky-Jerky Scrolling
  3. Typing so slow You could type an entire sentence and watch it show letter by letter.

This is what worked for me: 

  1. Go to  Menu
  2. Click System Preferences
  3. Open Energy Saver.
  4. Uncheck the checkbox next to Automatic Graphics Switching. * †
  5. Close Energy Saver and Preferences.
  6. Now Open any of the Office Apps. 

*

  • Not all Laptops have this feature where you can turn off Automatic Switching.
  • In that case there is a Utility you can Download called: gfxCardStatus,
    it works better because It can be set to switch from one to the other depending on the application.
  • I would advise to plug in the Power Brick and run off Power and not Battery while using this. 

†

  • If you have an issue with slow opening of everything,
    this will not help actual opening of the application.

  • But once open, and everything drawn.
    Everything should speed up nicely.
