Hey all, I had the same issue as well.

It looks like I was able to get around it by doing the following:

  1. While logged into the account you use to do the upgrade, take note of your temp directory path (%temp%, which was C:\Users\jp\AppData\Local\Temp in my case)

  3. Navigate to the folder from step 1 and rename the folder (now C:\Users\jp\AppData\Local\Temp2)
  4. Log back into the account from step 1. Your upgrade should now be unblocked

The CK-600 can only be connected post amplifier as it needs to sense impedence load or audio will be disabled.

 

Therefore a simple head unit ISO adaptor on an amplified system will not suffice.

 

There can also be issues on the mute function with this car's system, therefore I would recommend it is professionally installed.

  

If the car has an auxilliary input then the installer should use this for the music output of the kit, as this will utilise the amplified system of the car.

 

If you are based in the UK, you can find your nearest Nokia Accredited Installer here:

 

www.nokia.co.uk/carkitinstallers
