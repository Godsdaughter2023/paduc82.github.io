Hi

To be able to calculate the number of Excellent/Good/etc boxes checked I would suggest the following.

Assign each check box (CB) to a cell, ie if you look at the properties of each CB and find the 'LinkedCell' (for ActiveX type) or 'Cell Link:' (for Form type) and for Exellent type B2 for Housekeeping CB; B4 for Production; etc... and for Good CBs use column C, etc...

Then at the bottom of excellent column of CBs use the formula =COUNTIF(B1:Bnn,"TRUE"); for good use =COUNTIF(C1:Cnn,"TRUE"); etc... where  nn  equals the last CB row

You will then get the total of checked boxes in each category.

What does "Archive" mean, in relation to a file property?

When I right click on a file and choose "Properties", I sometimes see the "Archive" box checked.  What does that mean?  I don't understand why some have this box checked and others do not.  Does it mean that there is ANOTHER copy of this file somewhere hidden on my computer?  Can I control this?
rohn007 say:

As Hans said, the archive flag is turned on whe a file is created or changed. Then, when you use backup software the archive flag is turned off until until the file is changed again.

The Archive flag is used by backup software to identify files that have changed since the last backup to allow you to do "incremental" backups. Obviously, incremental backups only copy files that have the archive flag turned on.

The archive flag can be reset manually, as you saw. It can also be reset by a user in a batch file using the XCOPY command with the /A parameter to recognize files with the archive attribute turned on.

Automatically get data from one sheet and copy it to another - Microsoft Community

willyfv say:

Automatically get data from one sheet and copy it to another

I have 2 Excel spreadsheets and some data is exactly the same in both. I would like to enter data in the first spreadsheet, perhaps code an entire line with an entry (in cell A1,2,3, etc.) in the first with an “m” (for medical) and have it automatically copied into the second spreadsheet. Is there a way to do such a thing? If so - - how???

gary's student say:

The original data will be in Sheet1.  Once a row is  complete  it will automatically be copied to Sheet2.  In a given row, the user will fill in column A last.

Filling in column A in that row tells Excel that the row is ready to be copied.

Enter the following event macro in the Sheet1 code area:

Private Sub Worksheet_Change(ByVal Target As Range)
Dim s1 As Worksheet, s2 As Worksheet
Set s1 = Sheets("Sheet1")
Set s2 = Sheets("Sheet2")
Dim t As Range, A As Range
Set t = Target
Set A = Range("A:A")
If Intersect(A, t) Is Nothing Then Exit Sub
Application.EnableEvents = False
If IsEmpty(s2.Range("A1")) Then
    n = 1
Else
    n = s2.Cells(Rows.Count, "A").End(xlUp).Row + 1
End If
m = t.Row
s1.Range("A" & m & ":P" & m).Copy s2.Range("A" & n)
Application.EnableEvents = True
End Sub

  Because it is worksheet code, it is very easy to install and automatic to use:   1. right-click the tab name near the bottom of the Excel window 2. select View Code - this brings up a VBE window 3. paste the stuff in and close the VBE window   If you have any concerns, first try it on a trial worksheet.   If you save the workbook, the macro will be saved with it.     To remove the macro:   1. bring up the VBE windows as above 2. clear the code out 3. close the VBE window   To learn more about macros in general, see:   http://www.mvps.org/dmcritchie/excel/getstarted.htm   To learn more about Event Macros (worksheet code), see:   http://www.mvps.org/dmcritchie/excel/event.htm
