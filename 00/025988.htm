Macro - Add Cross Reference Automatically to Footnotes - Microsoft Community

jjmadero say:

Macro - Add Cross Reference Automatically to Footnotes

Hi All,

I have the following pattern throughout lots of articles:

supra note # (# is an actual number, for instance 7). The # references another footnote. Example:

Footnote 10:

supra note 5, at 100 (this means that it's referencing the same source as footnote 5, at page 100)

What I'm hoping to do is update each # after "note" to be a cross-reference to the appropriate footnote. I am currently doing this manually but it seems like something that in theory could be automated.

As always, thanks in advance.

jay freedman say:

The following macro may do what you want. A cross-reference to a footnote, whether inserted by a macro or by the Cross-Reference dialog, hyperlinks to the footnote's number in the main text, not in the footnote area at the bottom of the page; to get around that, you'd have to put a bookmark in the footnote text and cross-reference to the bookmark rather than to the footnote itself.

Sub SupraNoteRef()
    Dim intNoteNum As Integer ' number of footnote where cursor is
    Dim intSupraNum As Integer ' number of footnote to refer to
    
    If Not Selection.Information(wdInFootnote) Then Exit Sub
    
    intNoteNum = Selection.Footnotes(1).Index
    
    intSupraNum = Val(InputBox("Enter supra number:", "Footnote Reference"))
    
    If intSupraNum < 1 Then Exit Sub ' canceled or non-number
    If intSupraNum < intNoteNum Then
        Selection.InsertCrossReference ReferenceType:="Footnote", _
            ReferenceKind:=wdFootnoteNumber, ReferenceItem:=CStr(intSupraNum), _
            InsertAsHyperlink:=True, IncludePosition:=False
    End If
End Sub
