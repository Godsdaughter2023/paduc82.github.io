ken sheridan say:

You should never expose data to users as a table's raw datasheet view.  Always use a bound form, which can be in datasheet view if you wish.

 

From your description it sounds like you have a table open as a datasheet, and are somehow using an unbound form in which to select rows which you want to delete on the basis of criteria selected in combo boxes.  How are you executing the delete operation?  I can only assume that this is done in code somewhere in the unbound form's module, in which case by using a bound form rather than a datasheet you can requery the bound form in the same code following the execution of the delete operation:

 

    Forms("NameOfYourBoundForm").Requery

 

If you are using a macro rather than code to execute the deletion operation, add the equivalent to the macro.

 

BTW Refresh and Requery are different operations.  The former only reflects changes to data in existing rows in the table, the latter also reflects insertions or deletions of rows into/from the table.

If you record the narration directly from PowerPoint, the program uses the built-in recorder of Windows, but without the optional quality settings. This will result in not the best-possible sound files. Either open the Windows audio recorder separately and use it for recording and then insert the saved file into the PowerPoint slide. Or - which I myself prefer - use a third-party audio editing program (I prefer the free Audacity) to record the sound, save it in mp3 format (Audacity's standard format is .au, which PowerPoint cannot use, chose to export as an mp3 file) and then use those files. It may take some minutes to get used to the program, but you have much more control over the file and will get better quality.

 

Best,

Ute

Yes, to convert the table into text, click inside of the table that is pasted into Word.  This will display the contextual table tabs: Design tab and Layout tab.  Click on the layout tab > Data group > Convert to Text button (it is right on the end of the ribbon),  From the displayed dialog, select "Separate with paragraph marks" option then click on OK.

suzanne s. barnhill say:

Here's how to do it without using Excel:

  1. Select the list.
  2. Press Ctrl+H to open the Replace dialog.
  3. In the "Find what" box, type "^p" (without the quotation marks).
  4. In the "Replace with" box, type a comma and a space.
  5. Click Replace All.
  6. When asked if you want to search the rest of the document, say No and close the dialog.
