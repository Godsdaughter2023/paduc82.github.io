Hi Bobbye,

 

Lexmark has not yet released any Windows 7 or Windows 8 compatible drivers for the X9350 printer model. The latest driver that they have released for this model is for Windows Vista Operating System. However, it is found that these drivers also work with Windows 7 Operating System. But, they may not work properly with Windows 8.

 

So, here we have 2 options:

 

1) Install the native printer drivers embedded in the operating system driver library.

2) Install the Windows Vista compatible drivers in the compatibility mode.

 

Method 1: Uninstall and Reinstall the Native Printer Drivers embedded in the Operating System

 

Step 1: Uninstall the Printer driver which is already installed

 

a)       Press “Windows Logo” + “X” keys on the keyboard.

b)       Click on “Device Manager” from that list.

c)       Search for the Lexmark X9350 Printer device from the device list, right click on it and then select “Uninstall”. On the un-installation window, if you have an option: “Delete the driver software for this device” you may select that and then remove the corrupted drivers from the computer.

d)       Follow the on-screen instructions to complete it and the restart the computer if prompted.

 

After the restart, please open the Device Manager again and then click on the “Scan for hardware changes” button. Check if Windows 8 detects the printer device and installs an appropriate driver automatically. If the issue still persists, then try the next step.

 

Step 2: Check for the Printer drivers in the Operating System Driver Library

 

a)       Press “Windows Logo” + “X” keys on the keyboard.

b)       Click on “Device Manager” from that list.

c)       Search for the Lexmark X9350 Printer device from the device list, right click on it and then select “Update Driver Software”.

d)       Select the option “Browse my computer for driver software” from the “Update Driver Software” window.

e)       Search for the drivers in the “Drivers” and “DriverStore” folders by browsing to the location “C:/Windows/System32”.

f)        If the correct driver is detected, then follow the on-screen instructions to complete the installation.

g)       Restart the computer, if prompted and check if that works.

 

If that doesn’t help, then let’s try the second option to install the Windows Vista compatible drivers in the compatibility mode.

 

Method 2: Download the Windows Vista compatible drivers from Lexmark website and install it in the Compatibility Mode.

 

Please download and save the Windows Vista compatible drivers for the Lexmark X9350 printer on the computer from this link:

 

Lexmark X9350 Drivers

 

Note: You need to select the operating system as “Windows Vista (64-bit)” (Or “Windows Vista (32-bit)” as according to your system architecture).

 

Now, to install the drivers in the Windows Vista compatibility mode, you may refer to this article:

 

Get your apps and devices working in Windows 8

Hi Rose,

Thank you for your response.

The problem is that that the UAC (user account control) related program "consent.exe" will not start properly. This is the program that pops up on your screen(alternatively dimming it) to ask you if you wantto let programs make changes to your computer.   

The following seems to only happen when "windows update" tries to update third party applications, such as graphics drivers.  

When "consent.exe" fails to start properly the prompt to accept changes to your computer never appears, halting the update you just started indefinitely. You can identify that this is the problem by opening your task manager and going through "processes" and "details" to see if "consent.exe" shows up. If it does, this is most likely your issue. 

This is the very easy work aroundwhen you understand what the problem is: 

1. open your "control panel" this is most easily done by right clicking the windows icon/start menu and selecting "control panel" 
 2. in the control panel window, next to the address bar, use the search field and search for "UAC" without quotations.  

3. In the results select "change user account control settings" 

4. Drag the slider to the lowest setting. Finnish by selecting "Ok" in the bottom right corner and then "yes" when prompted. 
 
This will disable any "consent.exe" prompts and let you update smoothly.

Note: If you feel unsafe not having these security prompts pop up, simply redo the steps after the update and put the slider to the level of security you want again.
