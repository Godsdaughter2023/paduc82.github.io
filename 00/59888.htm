How to Take Ownership and Grant Full Control Permissions in Windows

Follow the steps:

  1. In Windows Explorer, navigate and locate the file or folder which you want to take ownership and has full read write control access permission on.
  2. Right click on the file or folder (directory), and click Properties.
  3. On the File/Folder Properties dialog box, go to Security tab.

  5. In “Advanced Security Settings” dialog window, click on Owner tab, which will indicates the current owner as Trusted Installer or some other user account.

  10. Click OK to make the change.
  11. If you’re prompted with notice that you have just taken ownership of this object, and need to close and reopen this object’s properties before you can view and change permissions dialog box, click OK.
  12. Click OK to exit from all Properties dialog boxes.
  13. Repeat step 1 to step 3 to open the object’s Properties dialog box again.
  14. Under “Security” tab, click on Edit button to change permissions.
  15. Note: Press Continue or enter user name and password of administrator account if User Account Control prompts for permission to continue.
  16. Click and highlight the Administrators or the user who wants the permissions on the object be changed in the “Group or user names” box.

  1. In the Permissions for <user or group account> box, click on Full Control check box under the Allow  to assign full read-write access control permissions to the user or group.     

  1. Click OK when done to close off all object’s Properties dialog boxes.
  2. The file or folder now belongs to you with full control permissions.
