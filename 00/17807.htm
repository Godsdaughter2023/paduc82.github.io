pelqiurchan say:

 

HI

 

Basically, there are a few exceptions that could potentially turn off Protected Mode.

 

They are as follows:


• User Account Control (UAC) is disabled – If UAC is disabled, Protected Mode is turned OFF. When UAC is disabled, some of the protections which Protected Mode depends on are not available, for example, UI Privilege Isolation (UIPI) is disabled. Hence, Protected Mode is turned off in this scenario.

• IE is running with Administrator privileges – Protected Mode is turned off when Internet Explorer(IE) is launched by right clicking on the IE icon and selecting “Run as administrator” or when IE is launched with administrative privileges from another application. This generally occurs when an installer/setup program running with administrator privileges starts a new IE process.

• IE is navigated to a local HTML page – When the page being viewed is a local file, Protected Mode is turned OFF since the contents of the page are considered safe. Caveat: If the page was saved from a zone (for example Internet) which has Protected Mode enabled, then Protected Mode is turned ON.

Even though User account control is turned ON and the protected mode is still off, then try turning the protected mode ON in Safe mode with networking to see if it gets turned ON.

1. Use F8 while restarting the computer.
2. Under Windows Advanced Boot Options menu, select Safe mode with Networking and hit Enter.
3. Once you login to windows, open Internet explorer and try turning ON the protected mode.

 

You can also try to uncheck the option “Enable memory protection to mitigate online attacks”. To do so, follow the steps below.

1. Open Internet Explorer. Click on Tools.
2. Click on Internet Options.
3. Switch to Advanced Tab and then under the security tab, uncheck “Enable memory protection to mitigate online attacks”.
4. Click on OK. Restart the Internet Explorer and then try to turn on protected mode.

 

If all of the above steps are of no help, then I would suggest you to reset internet explorer settings by applying the fix given in the link below.

 

How to reset Internet Explorer settings
http://support.microsoft.com/kb/923737
Note: When you reset Internet Explorer settings, all add-ons and customizations are deleted, and you basically start with a fresh version of Internet Explorer.

If you've run the Windows 10 Update Troubleshooter and that hasn't fixed the problem then you'll need to take more drastic action and attempt to reset the Windows Update process yourself.

A simple method is to copy the text between the ++++ lines and paste it into Notepad

++++++++++++++++++++++++++++++++++++++++++++++++

net stop wuauserv

net stop cryptSvc

net stop bits

net stop msiserver

ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

ren C:\Windows\System32\catroot2 Catroot2.old

net start wuauserv

net start cryptSvc

net start bits

net start msiserver

+++++++++++++++++++++++++++++++++++++++++++++++++

Save as UpdateReset.BAT (note the .BAT and NOT .TXT) to your desktop and close it.

Go to the desktop and right click on the .BAT file and select "Run as Administrator"

Before trying to update again make sure you detach any non-essential peripherals, USB devices and remove any 3rd party anti-virus software.

If you've run the Windows 10 Update Troubleshooter and that hasn't fixed the problem then you'll need to take more drastic action and attempt to reset the Windows Update process yourself.

A simple method is to copy the text between the ++++ lines and paste it into Notepad

++++++++++++++++++++++++++++++++++++++++++++++++

net stop wuauserv

net stop cryptSvc

net stop bits

net stop msiserver

ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

ren C:\Windows\System32\catroot2 Catroot2.old

net start wuauserv

net start cryptSvc

net start bits

net start msiserver

+++++++++++++++++++++++++++++++++++++++++++++++++

Save as UpdateReset.BAT (note the .BAT and NOT .TXT) to your desktop and close it.

Go to the desktop and right click on the .BAT file and select "Run as Administrator"

That should flush out any corruption in the Windows update files and enable you to move on to download the latest build.

Before trying to update again make sure you detach any non-essential peripherals, USB devices and remove any 3rd party anti-virus software.

Hope that helps and good luck.
