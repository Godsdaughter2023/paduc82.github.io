Windows Management Instrumentation (WMI) is a key part of the Windows 7 operating system. It is used to gather system statistics, monitor system health, and manage system components.

 

This “This webpage wants to run the following add-on: 'WMI Scripting'” warning occurs whenever a web page attempts to execute code on your machine that has not been used previously and is not on the local pre-approved list. This is usually caused by the website trying to use older code.

 

This warning occurs when that particular website is developed based on the older versions of Internet Explorer.

 

I would suggest you to click on “Allow” option if you get it when this warning occurs.

 

If that doesn’t fix the issue, then I would suggest you to open the website in compatibility mode and check if this fixes the issue.
