¡firedog say:

Windows Live Mail is the only free mail client that can synchronize folders (and contacts and calendar) between client and server for Hotmail accounts. This configuration is naturally the default.

Most users want to use IMAP if it's available, so that's why IMAP is the default for Gmail and AOL accounts.

If you particularly don't want to synchronize folders and want to use POP3 instead, then you have to select Manually configure server settings when adding a Hotmail or Gmail account.

Both Hotmail and Gmail allow for message rules at the server. They are not as comprehensive as the ones available in WLMail, but if all you're interested in is diverting messages from the inbox to user-defined folders, the rules available will handle that.

a. user say:

Hi pargolfer(2).

 

Thank you for posting your query in Microsoft Community.

 

 

From your description I understand that you want to know if you can delete desktop.ini icon.

 

 

 

Windows uses desktop.ini to identify how a folder should be displayed, much like the .DS_Store files used in OS X.

When you create a new folder in Windows it is created with the standard folder icon, one common use of the desktop.ini file is to use to it set a custom icon for a folder, however this is not the only thing that it is used for.

 

 A typical desktop.ini will look like something like this:

 

[.ShellClassInfo]

ConfirmFileOp=0

IconFile=Folder.ico

IconIndex=0

InfoTip=Type Your InfoTip Here. 

 

   

The desktop.ini file can contain a lot of attributes but the following custom attributes are the most common:

  • ConfirmFileOp
  • IconFile
  • IconIndex
  • InfoTip.

 

They are hidden by default and will only be displayed if you choose to Display Protected Operating System Files by unchecking the checkbox.

 

I suggest you hide desktop.ini file instead of deleting it.

 

Follow the steps to hide this file.

 

a)     Open the file and now on top click the option view,

b)    Now uncheck the box show hidden files

c)     Or you can click options then click show folder and search options.

d)    Now click view and uncheck the box which says show hidden files, folders and drives.

e)     Now click apply then ok.

 

 

Hope the above information helps.

 

 

Do get back to us in case you need further assistance
