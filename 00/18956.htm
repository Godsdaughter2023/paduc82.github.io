rob koch say:

Have you already tried using the built-in Windows Disk Cleanup utility found by right-clicking the C: drive letter in Windows File Explorer, selecting the Properties option and then clicking the Disk Cleanup button found in the Local Disk (C:) Properties dialog?

Within the Disk Cleanup for (C:) dialog box that comes up is a Clean up system files button which after several minutes (maybe more) should display additional options to be deleted including a checkbox for Windows Defender,

Though it's possible that there is something wrong causing these files in the Windows Defender scans history, store folder not to be maintained properly, the above method is the official way that Microsoft has provided within Windows to manually maintain such files.

Browsing manually through such Windows system folders and deleting files by hand is dangerous, so always attempt to exhaust the official methods provided first, before randomly deleting items who's purpose you don't fully understand.
