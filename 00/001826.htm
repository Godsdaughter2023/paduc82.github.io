Option 2 - Use the below VBA code to fire the print out which will print last page without title but all other pages with titles 

Sub MyPrint()
    Dim PageCount As Long
    Dim i As Long
    Dim Ws As Worksheet
    Set Ws = ActiveSheet
    PageCount = Ws.PageSetup.Pages.Count
    Ws.PrintOut From:=1, To:=PageCount - 1
    Ws.PageSetup.PrintTitleRows = ""
    Ws.PrintOut From:=PageCount, To:=PageCount
End Sub

jay freedman say:

PowerPoint does have an Eyedropper tool to copy shading colors. You can use it this way:

Copy the table from Word, and paste it into a PowerPoint slide with the Keep Source Formatting option.

In the Table Tools | Design tab, click the down arrow of the Shading button and click Eyedropper in the menu.

Hover the mouse pointer over the cell with the desired shading. The tooltip will show you the RGB numbers for the color. If you prefer, you can just write down the numbers and go back to Word to insert the rest of the shading.

If you click the cell with the Eyedropper tool, two things will happen: the whole table will be given the color of the cell you clicked, and the Shading button will also be given that color as its current default. If you want the shading only in certain cells, press Ctrl+Z (Undo), select one or more cells, and click the left side of the Shading button. When the shading is the way you want it, copy the table and paste it back into Word, again using Keep Source Formatting.

And no, I have no idea why the Eyedropper tool is in PowerPoint but not in Word. Add your vote to the suggestion at https://word.uservoice.com/forums/304924-word-for-windows-desktop-application/suggestions/16713148-eyedropper-tool-for-word.
