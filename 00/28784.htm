How to find Ourlook e-mail folders on old drive? - Microsoft Community

steve bienenfeld say:

How to find Ourlook e-mail folders on old drive?

My old hard drive crashed and I had to reload everything on my new drive, including Microsoft Office, which has Outlook on it.  I copied my entire old drive onto an external drive.  My new Outlook only has my new e-mails since I reloaded Outlook--all my old folders and e-mails on somewhere on the external drive, but I don't know how to find them.  Help Please!!!!!!!!
Steve B.
brian tillman [outlook mvp 2007-2019] say:

You've been answered a few times.

The default location for Outlook's data files is a folder that is, by default, hidden.  For Windows 2000/XP, the path is x:\Documents and Settings\{username}\Local Settings\Application Data\Microsoft\Outlook.  For Windows Vista and 7, the path is x:\Users\{username}\AppData\Local\Microsoft\Outlook.  Open Windows Explorer and enter the appropriate path in the Address field, substituting the real drive letter for the "x" in the path as I wrote it and the username you were using in place of "{username}".  Click Go and you should see the Outlook data file.  It's file type will be ".pst".  You may have Windows configured to not show file types of registered files.  If that's the case, look for the one that is described as "Outlook Data FIle".  You may have more than one, depending on how you configured Outlook.  You should know.

Brian Tillman [MVP-Outlook]

 AndyWickens replied on January 1, 2014

 
FIXED ON SAMSUNG GALAXY S2?

 

I had the same problem with hundreds of duplicated emails in my outlook sent mail folder. I have a Samsung Galaxy S2 and I use the native email app (which I prefer after trying many alternatives). I cleared the problem using the native email app on my phone.  I deleted the outlook account [open app; settings; bin in top right corner; delete] then added it back selecting 'Manual Setup'.  When the app then asks 'What type of account?' I selected 'Microsoft Exchange ActiveSync'.


The settings needed are as follows...

 

Domain\user name: \[Your outlook email address]

Password: [Your outlook password]

Exchange Server: s.outlook.com

Use Secure connection (SSL) [Tick yes]

Use client certificate: [Leave blank]


All your emails will be back in place on the phone within a few minutes once you have worked through a few more settings which are all ok at their default values.


I can now send emails from my Samsung with only one instance appearing in my outlook sent folder :)


If I get any further problems I will report back.


I got the Exchange ActiveSync settings from the following
