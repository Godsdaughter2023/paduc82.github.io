Here,

 a. the key traditionally called "Option" on Mac is also labelled "alt"

 b. in System Preferences->Keyboard, there is an option called "Use F1, F2 etc. keys as standard function keys"

 c. with that option *checked*, I would use option-F9 (also known as alt-F9)

 d. with that option unchecked, I use fn-option-F9 (i.e. fn-alt-F9)

If none of those things work then it is possible that the relevant keystrokes have been assigned to something else, either in Word (although I doubt that as I think you would know about it) or in Mac OS(X).

Whatever the case, one thing you can do is assign another keystroke to the relevant command.

 1. Go into Word->Tools->Customize Keyboard

 2.Select All Commands from the list of Categories on the left.

 3. In the Search box under "Commands" on the right, type ViewField. The list of matching commands should reduce to "ViewFieldCodes". Select that.

 4. In the Current Keys, you should see a list of the keys that invoke the command. Here, it says "option-F9" (it does not take account of that function key setting in System Preferences and would never show the fn- part). It will probably be there even if Mac OS(X) is using it for something else.

 5. Click in the Press New Keyboard Shortcut box, and do a suitable multi-key combination. e.g. Control-Shift-V. Word should display whether that has been assigned to something else. If it says [Unassigned], it hasn't, so you can click Assign to assign it.

You may need to do much the same thing for some other field-related keyboard shortcuts.

There is another command called ToggleFieldDisplay (assigned to Shift-F9) which toggles the display of the currently selected field code.

hansv mvp say:

You can't sum a control across records, only fields in the record source and expressions depending on fields in the record source.

For example, if the Control Source of tbxTotalCases is

 

=[This]+[That]+[Other]

 

you should use the following as Control Source for the text box in the footer:

 

=Sum([This])+Sum([That])+Sum([Other])

 

Alternatively, if you have text boxes in the footer that sum individual fields, e.g.

 

tbxTotalThis:  =Sum([This])

tbxTotalThat: =Sum([That])

tbxTotalOther: =Sum([Other])

 

you can use

 

=[tbxTotalThis]+[tbxTotalThat]+[tbxTotalOther]

Thanks for coming back to us. Generally, we change MX records after all migrations are completed. Otherwise, on-premises users will not be able to receive emails.
