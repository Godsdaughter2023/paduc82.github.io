Reminders in Excel

I am running a excel 2013 spread sheet with multiple records that expire after 13 months. how do I:

b) set a reminder when it has expired.

bernie deitrick say:

  You can check a column of expiration dates on a specific sheet using the workbook open event.  For example, if your dates are in column C, and column D is used to indicate when the record is completed using the word "Finished" then this code will remind you when dates are within 2 weeks of expiring or if they have passed.  you may not need the "Finished" parts if you do something else with the row once the record is completed, like move them off the sheet or delete them.

    1) Copy this code.
    2) Right-Click the sheet tab of interest.
    3) Select "View Code"
    4) Open the Explorer window using Ctrl-R

    5) Expand your workbook's project

    6) Double-click the "ThisWorkbook" object in your project

    7) Paste the code into the window that appears.
    8) Save the file as a macro-enabled .xlsm file.
    9) Make changes as needed

Private Sub Workbook_Open()
    Dim rngC As Range

    For Each rngC In Worksheets("Sheet1").Range("C:C").SpecialCells(xlCellTypeConstants)
        If IsDate(rngC.Value) Then
            If rngC.Value >= Date And rngC.Value < Date + 14 And rngC.Offset(0, 1).Value <> "Finished" Then
                MsgBox "The date in " & rngC.Address(False, False) & " is getting close."
            ElseIf rngC.Value < Date And rngC.Offset(0, 1).Value <> "Finished" Then
                MsgBox "The date in " & rngC.Address(False, False) & " has passed."
            End If
        End If
    Next rngC
End Sub

slance310 say:

Hi,

If updating your drivers doesn't help, you might try turning off Fast Startup...

Sometimes certain drivers don't load properly when it is turned on...

This feature only works when you do a SHUTDOWN and then Boot.

It doesn't effect a RESTART.

To turn it off or on...

Go to Control panel...Power Options, and select Choose What the Power Buttons Do on the left.

Then select Change Settings That Are Currently Unavailable near the top center of screen...

Lower down on the window, uncheck Fast Startup.

This may or may not help your problem, but is easy to check.
