m4p - purchased audio file (locked)

daniel g. grau say:

Before I give you a solution, note that double spacing after a period may get you lambasted. (Personally, since the time QuarkXpress was king, I've never seen a typeset document using them.)

  • Highlight a period and two spaces in your text
  • Select Tools > AutoCorrect in Word's main menu
  • Enable Replace text as you type
  • In the Replace box, type a convenient sequence; I suggest using two periods (..)*
  • Check the contents of the With box—you'll probably need to add two spaces after the period
  • Select Plain text for a format-matching replacement; selecting Formatted text will always apply the current font, size, etc.
  • Click on Add

To test, type period period.

Bear in mind that AutoCorrect sequences are language-dependent. If you type in another language, you'll have to redefine the sequence for that language.

* You could also replace "period" by "period space" or "period" by "period space space." Experiment and decide.

** It is now possible to make Word 2016 insert two spaces between sentences.

karl timmermans [outlook mvp 2012-2018] say:

You can certainly customize your view but not aware of a way to include fields (include = getting the field to populate) from another item type (i.e. contact item). Aas far as I know, it won't work. You can add certainly the fields from any item type to a view but it won't pull the information.

That said, others may have far more experience than I have in doing what you're asking so always a possibility that there's something I've missed about combining info from multiple item types in a single view.

Also the fields that get displayed depend on whether or not you use the <Reading Pane>
