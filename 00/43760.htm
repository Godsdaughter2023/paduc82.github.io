Physical address is the card's MAC number.

The xx should be a number from 0 to 255 (all xx same number).

yy should be from 0 to 255

zz should be from 0 to 255 (all zz the same number.

a. user say:

Power supplies don't usually make a sound when they fail unless one of the internal parts blows, which is pretty spectacular.

The thing is, Windows can't directly access the power supply or the code stored in motherboard components like the CMOS or the chipset.  A Windows Update can only mess things up once Windows has started, but it seems your computer powers on and then immediately powers off, long before Windows has a chance to load.

Windows Updates sometimes require a restart, and if your computer hasn't been restarted in some time, the problem with the power supply wouldn't make itself known until then.

Someone who's handy with a multimeter can tell you in a few seconds if the power supply is the problem. I'd be happy to be wrong, but I think it's the power supply. Also try Andre's suggestion to start the laptop directly from the wall. If that works (hooray) then you may have a problem with the battery.
