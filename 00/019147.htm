to export the query to a workbook named AMAPS.xlsx.

ken sheridan say: You can export a query and create a specific sheet by specifying the sheet name as the Range argument, e.g.

Docmd.TransferSpreadsheet acExport, acSpreadsheetTypeExcel9, "NameOfQueryGoesHere", CurrentProject.Path & "/NameOfWorkbookGoesHere.xls", True, "NameOfWorksheetGoesHere"

This will firstly create the workbook in the current folder.  If you then change the values only of the query name and final argument to a different query and sheet name each time you call the method it will add a new worksheet to the workbook.
