Any control has a On Click event. so you can put an image control on your form and code the On click event to populate a hidden text box with the answer. For example:

This would be behind smiling face.

As an aside, a questionnaire type database requires a very specific structure. Something like this:

tblQuestions: QuestionID (PK Autonumber), Question

tblRespondent: RespondentID (PK autonumber), names, other info about respondent

tblResponse: ResponseID (PK Autonumber), RespondentID (FK), ResponseDate

tblAnswer: AnswerID (PK Autonumber), ResponseID (FK), QuestionID (FK), Answer
