gerry c j cornell say:

Sharon

Hard drives are disks for the storage of data which may be partitioned into one or more partitions (or volumes). RAM memory is a card which is placed in a memory slot in the motherboard. The motherboard may have a number of slots, commonly 2 or 4. As a backup to RAM memory the system will temporarily store data in a pagefile on the hard drive and this is virtual memory.

Hard drives may be internal (within the computer casing) or external (outside the computer case).

The system firstly uses RAM memory as it is faster to access, read and write, and for less immediate tasks the system swaps data out for later use from RAM to the pagefile on the hard drive.

Low memory messages are normally because the system is detecting that there is a danger that there will shortly be insufficient disk space to the meet the needs of the system. This may be because the allocated disk space is insufficent i.e. that the allocated maximum set needs to be increased or that there is not enough remaining free disk space to allow for the size of the pagefile size to be increased. In your case the messages are relating to something other than the more common situation.

In your case the problem relates to the Volume Shadow Copy service.
http://en.wikipedia.org/wiki/Shadow_Copy

This service has disk space requirements and the system reserves part of the disk solely for it's use. How it does this and how to fix it when things go wrong is not commonly understood which is why I am struggling to give you simple instructions on how to resolve your problem.

A possible solution could be the formatting of your hard drive and a new installation of Windows and programmes. This is a drastic solution for what is an irritating but probably not a problem which prevents you from using your computer. Reinstalling the operating system is not something I would recommend an inexperienced user undertaking.

What has caused your problem? Most likely it has been caused by your anti-virus programme or a a programme like Registry Mechanic, or is the result of malware (virus) damage. You should uninstall Registry Mechanic through Add / Remove Programs in Control Panel now and hope that it's presence has not done irreparable damage to the Registry.

What anti-virus software are you using? If it is Norton or McAfee that needs to go. Norton is especially good at messing up the Windows operating system. If you have Norton you will need specific advice as removal can be problematic. There are many suitable freeware alternatives.

rajesh govind say:

 

Hi Jeff,

 

By the term, "Column view" which shows the hierarchy of the folder in the computer, I think you are referring to the “Navigation Pane” on the Windows Explorer.

 

We can definitely enable the “Navigation Pane” on the Windows Explorer. To do this, you may follow these steps:

 

a)       Go to the Desktop Interface on the computer.

b)       Open the Windows Explorer and click on “View” tab at the top of the window.

c)       Click on “Navigation Pane” and select it from that list.

d)       Now, you will be able to see the “Navigation Pane” shown on the left hand side of the window.

 

To open the sub folders in the same window, you may follow these steps:

 

a)       Go to the Desktop Interface on the computer.

b)       Open the Windows Explorer and click on “View” tab at the top.

c)       Click on “Options” on the right hand side of the ribbon.

d)       Click on “Change Folder and Search options”

e)       In the “Folder Options” window and on the “General” tab, select the option “Open each folder in the same window”

f)        Click on “Apply” and then “Ok”.

 

Hope this information is helpful.

Here is the solution:

  1. Download the Intel Driver Utility, and run it.  It will install a new driver.
  2. Restart your computer
  3. Run Windows Update.  It will install the new driver.
  4. Restart your computer
  5. Go to Device Manager > Display Adapters > Intel(R) HD Graphics 4600 > Driver > Update Driver > "Browse My Computer for Driver Software" > "Let me pick from a list of device drivers on my computer"
  6. Select version 10.18.10.4252
  7. Restart your computer.
  8. Finished
