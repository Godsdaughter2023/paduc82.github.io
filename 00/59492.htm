considerate_guy say:

One test you might try is to break the project into two. So start by making two additional copies of the project (let's call them Part1 and Part2).

Open up the first project copy (Part1) and delete everything after about the 50% mark. Save that project and try to render it as a movie (File > Save movie). Does that complete?

Now open the second project copy (Part2) and delete everything you already have covered in the first part (everything *after* the 50% mark. Save that project and see if that will render as a movie. Does that complete.

If both parts complete, then it could be that your computer CPU/GPU is getting overwhelmed and is overheating. Or it could be that the combined project is too complex for Movie Maker.

If you can do it in parts, then start one new project and add the two movies that you previously saved (the movies, not the project files). Then see if you can save that as one joined movie.

To rectify the issue with PIN not working in Windows 10, you will have to boot your PC from a Windows 10 installation disc/media to access command prompt and create a new admin account, since you are unable to login to the current user profile.

If you don’t see the message, you might need to change the boot order in your computer's BIOS settings so that it first starts from the DVD or USB. 

Note: Modifying BIOS settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS settings can be solved. Modifications of the settings are at your own risk.

If you don't have installation media, you may download Windows 10 ISO files and create a Windows bootable media or DVD from this link. Then, follow these steps to create admin account for Windows 10:

  1. When you get to Windows Setup screen, press Shift + F10 to bring out a cmd.exe dialog.
  2. Execute the following command lines (Note: Replace d:\ with your system drive letter.):
    • Type move d:\windows\system32\utilman.exe d:\windows\system32\utilman.exe.bak and press Enter to make a copy of utilman.exe file.
    • Type copy d:windows\system32\cmd.exe d:\windows\system32\utilman.exe and press Enter to replace utilman.exe file with cmd.exe file.
    • Type wpeutil reboot, press Enter, and disconnect the Windows 10 disc from the computer to reboot Windows 10.
  3. When Windows 10 reboots up and comes to the sign-in screen, click the Ease of Access icon. This should open a command prompt dialog because you have replaced the Utilman.exe file with cmd.exe file in the second step above. Then execute the following two command lines to create a new admin account for your Windows 10. (Note: Replace the [username] with your desired user name and [password] with your desired password.) 
    • Type net user /add [username] [password] and press Enter to create a new user account.
    • Type net localgroup administrators [username] /add and press Enter to add the newly created user account to your Windows 10 administrator group.

Now that you are able to create a new admin account, you may follow the steps on how to delete the NGC folder and add a new PIN code, provided by Andre Da Costa on this similar thread to rectify the PIN issue.

GENERAL TROUBLESHOOTING GUIDE

Do you get any errors related to updates?

1. Open Windows Update by clicking the Start button , clicking All Programs, and then clicking Windows Update.
2. In the left pane, click View update history.
3. Find an update for Windows that has failed to install, and then double-click the update to view more information.
Updates that have failed to install will display Failed under Status, next to the update name.
4. In the Windows Update dialog box, next to Error details, review the error code for the failed update.
5. Either follow the links in the Windows Update dialog box under More Information or Help and Support to try to fix the problem, and then try installing updates again by using Windows Update in Control Panel or post back with the error code so perhaps someone may be able to assist further.

synnic say:

The Surface Pro is a full fledged PC and runs Windows 8, so you can install the Amazon downloader and download song files just like you do on a regular PC now. Surface RT runs Windows RT and does not run as many different apps as Surface Pro.

 

You can play songs by streaming from the web version of Amazon Cloud Player on both Windows 8 and Windows RT. Currently I'm not aware of any way to download songs from Amazon directly to Windows RT. You would have to download them to a regular PC first and then copy them to the Surface RT if you want to play the songs without a network connection.

 

There is no Google Drive app for Windows RT so far, but Microsoft does have something similar called SkyDrive.

 

Also, just FYI there is a Kindle application for Windows 8 and Windows RT, so you can read your Kindle eBooks.

 

I'm speaking from experience having recently purchased a Surface RT to play with. I hope this helps.

Press Windows key + X

Click Command Prompt (admin)

Enter the following command:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Hit Enter

Your product key should be generated, try entering it:

Click Start > Settings > Update & security > Activation > Change the product key
