Normally, the apps window can be moved from one point to another within the computer screen by pulling them using the mouse pointer and that is how we snap the apps on the screen.

 

The above instructions that you have given in the query, disables this movement of the apps window when they are in the full screen or enlarged mode. But, when the apps window is not in the full-screen/enlarged mode, then the above instructions does not work and the apps window can still be easily moved and snapping can be done on the screen. That may be the reason why it is not working in your case.

 

However, there is another way to disable this snap feature of the apps on Windows. It is done by performing a registry edit on the computer. You may follow these steps for the same and let us know the result.

 

a) Press “Windows Logo” + “R” keys on the keyboard to open the “Run” command box.

b) Type “regedit” in the “Run” command box and press “ENTER”.

c) This will open the Registry Editor window on the computer.

d) Now, from the registry editor, navigate to the location given below:

 

HKEY_CURRENT_USER \Control Panel\Desktop

 

e) On the right pane of the window, look for the option “WindowArrangementActive” and then double-click on it.

f) Now, change its value from the default “1” to “0” and this should disable the snap view feature on the computer.

kvm say:

Parse text and return value between 2 delimiters

Suppose, the sample text below is in Column A Cell A1,A2...so on, Could you please help me with a function that can parse a text and return text between 2 delimiters as shown in the examples below, the delimiters could be a ":" and/or "-".

 

 

Sample text 1: My Name is : Melinda Bill Gates :  and I work at Microsoft.

Expected output - "Melinda Bill Gates"

Sample Text 2: My Name is : Melinda Bill Gates -  and I work at Microsoft.

Expected output - "Melinda Bill Gates"

 

Thanks,

KVM

rich007a say:

Hi,

If your string is in A1, try:

 

=TRIM(MID(A1,FIND(":",SUBSTITUTE(A1,"-",":"))+1,FIND(":",SUBSTITUTE(A1,"-",":"),FIND(":",SUBSTITUTE(A1,"-",":"))+1)-FIND(":",SUBSTITUTE(A1,"-",":"))-1))

 

Hope that helps.

Cheers
Rich

 

PS, if you knew that the desired text would be sandwiched between ":" at each end, you can just use:

 

=TRIM(MID(A1,FIND(":",A1)+1,FIND(":",A1,FIND(":",A1)+1)-FIND(":",A1)-1))

 

but if the "bread of the sandwich" could be either ":" OR "-", then you need to use the longer formula with SUBSTITUTE, which effectively replaces any "-" with ":".

I always work in order or Theme>Master>Layouts>Slides. First, check whether your font theme is set to use Calibri by choosing View>Slide Master>Fonts. If the Theme is using Arial instead of Calibri, change it. The Calibri font theme uses Calibri for both Headings and Body.

Next, select the Slide Master (the larger slide at the top of the left-hand list of slides). Select each placeholder and check that the font is set to either Calibri (Headings) or Calibri (Body). When you see Headings or Body in the font listing, it means you are keyed to the font theme.

Then select each Slide Layout (the smaller slides below the Master) and check that each placeholder is set either to Calibri (Headings) or Calibri (Body).

Finally, close the Master view, select each slide in your presentation and click on Home>Reset. This reapplies the slide layout to the slide and removes any local formatting that could interfere with Calibri being used.

When you paste text, try Paste Special by clicking on the bottom half of the Paste button and choosing Paste Special...>Unformatted Text. This guarantees that no font formatting pasted with the text.

For pasting more formatted text, tables and slides, try Paste and choose the Use Destination Theme option.
