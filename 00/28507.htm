phillip m jones say:

The issue is not with Word. The message is part of safety feature built into the MacOS Sandboxing. 

After years and years of Apple of ignoring security concerns from Malware; in recent years the malware writers have begun to write Malware for Mac's. So Apple came up with Sandboxing.

Sandboxing is where Components of the Application and System Files associated are placed in safe zones. That way if a Malware tries to attack it could affect a part or two in the application but not the system or other applications. Thus limiting any possible damage to one or two items but not the entire system.

This technique complicates the ability to place files anywhere. Apple wants all documents placed in the Documents Folder. Anything placed in Documents (even in subfolders) are automatically protected, and the warning does not show up. However, if you are like me and thousand of others distrust the idea of putting all your eggs in one Basket.  You can create folders on the Hard Drive and place your files there.  But this warning will pop up So what you do is Locate the Folder where this file is and approve it as Safe area. After that any files placed in that folder are okay. 

There are some  conditions: 

  • First, the files should not be placed in many layers of subdirectories.
  • Avoid using Forbidden character in the naming the Directory, subdirectory or file naming. I notice from the sample snapshot that dates used as subdirectories are formatted wrong. In the case of dates you should avoid "1/2/2017" instead, they should be written as "1_2_2017".
  • Then to approve Locate the location of the file and approve the site.

scottgem say:

Does the elapsed time ever span more than 24 hours? If it doesn't then you should be fine. If it does, however, then you need to include the date as well as the time. 

You need to understand that Access stores time as a double precision number where the integer portion is the number of days since 12/30/1899 and the decimal portion a fraction of a day. So that .25 would be 6 AM (or 6 hours).

If you use this:

=(Stoptime - Starttime)*24 it should also work. 

To use your numbers 10:00-09:30 would = 0.0208333. Multiple that by 24 and you get .49999992
