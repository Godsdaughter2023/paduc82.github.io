wunders say:

More details about what was happening when this error appeared would be helpful, therefore my answer will be quite general...

 

Whenever you try to connect to any internet server using its internet name (e.g. "answers.microsoft.com"), a connection cannot be made to that server until an IP address for that server can be ascertained.  It's like telephoning "Joe"...  you can't call Joe unless and until you know his telephone number.  So out on the internet is a big "telephone book" that is called a "DNS server" which takes the name (e.g. answers.microsoft.com) and looks up the number that goes with that name (e.g. 157.56.56.109 ).  The error you cited will happen if your DNS server is unable to come up with an Internet Address for the server name that you gave it.

  JW

allendave say:

Dell Inspiron 2n1 will not auto rotate

I am writing this because I bought this device and couldn't figure out how to get auto rotate to work. Now that I have discovered the solution I wanted to share it with everyone.

If you have a problem getting your Dell 2n1 to auto rotate because the feature is grayed out these are the steps I used to solve the problem.

1. Click on the action center icon, it is located on the far right side of the task bar which is located on the bottom of the screen.

  The action center icon looks like this.

  The action center icon looks like this if you have notifications.

2.  When you click on the action center icon you will see the tablet mode icon and the auto rotate icon (auto rotate will be grayed out).

3. Select the tablet mode icon to put your device in tablet mode.

4. Now here is the trick, you must place your device into a position that requires auto rotate before auto rotate will become selectable.

5. Flip your device over so that the screen is upside down, you can do this by either placing the keyboard against the back of the screen and then flipping the device upside down or flip the device over and have it stand up like a tent so that the screen is upside down.

6. Now wait a few seconds and then go back to the action center and the auto rotate icon should be selectable.

7. Select the auto rotate icon and the screen should rotate, it may take a few seconds before it rotates. Now the device should auto rotate each time you place it in a position that requires auto rotate.

8. Now the next time you flip your device over you may see a dialog box appear asking if you want to switch to tablet mode. This is because you have "ask me before switching to tablet mode" selected in the display setting under tablet mode. To change this go to "setting" "display" "tablet mode".

Since you have already activated the auto rotate option, the device does not require you to be in tablet mode for it to auto rotate. If you don't want to be asked about going in and out of tablet mode just change the option in the display settings.   

9. However..... you must be in tablet mode if you want your device to auto rotate to portrait mode when you turn it side ways.

If you don't know where DMP files are located, here's how to get to them:

 1.    Navigate to the %systemroot%\Minidump folder.
 2.    Copy any and all DMP files in the Minidump folder to your Desktop and then zip up these files.
 3.    Upload the zip containing the DMP files to Skydrive or a hosting site of your choice and paste in your reply.

If you are going to use Skydrive but don't know how to upload to it, please visit the following:

http://www.wikihow.com/Use-SkyDrive

Please note that any "cleaner" programs such as TuneUp Utilities, CCleaner, etc, by default will delete DMP files upon use.

If your computer is not generating DMP files, please do the following:

1. Start > type %systemroot% which should show the Windows folder, click on it. Once inside that folder, ensure there is a Minidump folder created. If not, CTRL-SHIFT-N to make a New Folder and name it Minidump.

2. Windows key + Pause key. This should bring up System. Click Advanced System Settings on the left > Advanced > Performance > Settings > Advanced > Ensure there's a check-mark for 'Automatically manage paging file size for all drives'.

3. Windows key + Pause key. This should bring up System. Click Advanced System Settings on the left > Advanced > Startup and Recovery > Settings > System Failure > ensure there is a check mark next to 'Write an event to the system log'.

Ensure Small Memory Dump is selected and ensure the path is %systemroot%\Minidump.

4. Double check that the WERS is ENABLED:

Start > Search > type services.msc > Under the name tab, find Windows Error Reporting Service > If the status of the service is not Started then right click it and select Start. Also ensure that under Startup Type it is set to Automatic rather than Manual. You can do this by right clicking it, selecting properties, and under General selecting startup type to 'Automatic', and then click Apply.
