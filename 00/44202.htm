a. user say:

Hi,

Thank you for posting in Microsoft community.

We understand your concern and will certainly assist you to solve it.

Please be advised that the issue you face might be due to some of the Windows 10 audio driver related file component either goes missing or corrupt. So, please try the below steps to troubleshoot.

Method 1: Hardware troubleshooter.

Please follow the below steps to run the hardware troubleshooter:

1. Type Troubleshooting in search box and open it.

2. Click on Hardware and Sound

3. Click on Hardware and Devices.

4. Follow the instruction.

If the issue persists, please follow the below method.

Method 2: Disable and enable the speaker.

Please follow the below steps to troubleshoot.

1. Select the volume icon in the bottom right and pane and right click it and select "Playback Devices".

2. Select the realtek speaker there and right click it and select Disable.

3. Now, in there in the playback tab, select the "show disabled devices" and then select the Speaker and then right click it again and select Enable.

If the issue persists, please follow the below method.

Method 3: Clean boot.

Sometimes the improper audio mixing might be caused due to some software conflicts. So, please try to do a clean boot and check if the issue persists.

Putting your system in Clean Boot state helps in identifying if any third party applications or startup items are causing the issue. You need to follow the steps from the article mentioned below to perform a Clean Boot.

https://support.microsoft.com/en-us/kb/929135

DISCLAIMER: After troubleshooting, refer to this section "How to reset the computer to start normally after clean boot troubleshooting".

  • Devices must be able to turn on in less than 500 milliseconds
