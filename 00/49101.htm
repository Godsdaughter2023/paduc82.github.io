Hi,

A couple of things need clearing up with your question.

"My SSD on my new PC just crashed"

"am I allowed to install my copy of Windows 7 upgrade"

I was wondering why you would have a Windows 7 Upgrade on a new Computer.

You do not say where you got the Windows 7 Upgrade License from.

 

If it was part of a free Computer Manufacturer's Upgrade to Windows 7 from Vista (which expired long ago) it would be an OEM License

If Windows 7 came preinstalled, it would also be an OEM License.

An OEM license is tied to the original Computer it is installed on and can not be transferred to another Computer.

 

If it is a Retail Windows 7 License which you bought and installed yourself, you may transfer it to another Computer.

But, you can only have it installed on one Computer at any one time.

 

Cheers.
