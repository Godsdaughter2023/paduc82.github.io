Any idea what can cause this, or how to correct it?

jay freedman say:
When you want to print less than the entire document in which there is more than one section, especially if the sections have restarted page numbers, the p1s1 syntax is the only way to know exactly what you'll get printed. This is explained briefly in http://www.word.mvps.org/FAQs/Formatting/PrintMultipleSections.htm.

If you omit the section numbers, Word will print every page that has a number within the range you set. For example, suppose you have three sections, each of which restarts page numbers at 1, with 1 page, 4 pages, and 2 pages respectively. If you ask to print the range 1-3, Word will happily print one page in section 1, the first three pages of section 2, and both pages of section 3.

    srinivasu guntimadugu say:

    You may try to export data from BCM 2003 including History using the following steps:

     

    1.    Open Outlook

    2.    Click on File> Import and Export> BCM

    3.    Select Export file and click next

    4.    Select BCM (.bcm) and click next

    5.    Select Export everything click next

    6.    You should find an option with history or without history

    7.    Select the required option and set the location to export.

     

    For additional help, visit the following link:

    http://office.microsoft.com/en-us/outlook-help/export-a-file-from-business-contact-manager-HP001113493.aspx
