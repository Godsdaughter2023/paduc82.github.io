In order to view and edit the macros in that template, you must move it out of the Startup folder to any other folder where you have access rights (e.g., your Templates folder, or C:\temp, or some folder you create). Then use the Open dialog to open the template, and its macros will be editable.

rohn007 say:
If you want to import your 2003 styles and settings into 2007-2013 there is a simple technique.

  • Close all Word (and Outlook) sessions
  • Navigate to your personal 2007-2013 NORMAL.DOTM
  • rename or delete the NORMAL.DOTM file
  • navigate to the 2003 NORMAL.DOT
  • copy it
  • Paste the 2003 NORMAL.DOT into the 2007-2013 template folder
  • Start Word 2007-2013
  • Word will note that there is NORMAL.DOTM present and it will see the 2003 NORMAL.DOT. Word will extract the styles and settings from 2003 and use them in the new NORMAL.DOTM it creates.

Hi

The Programming Language for the new Modern UI Apps makes it very difficult to create shortcuts like you did in previous versions of Windows.

There are some tricks that you can use to make this easier. Here is one that will create a new folder that contains a shortcut to all of the Apps and Programs installed on the computer.

1. Right Click the Desktop and select New/Shortcut in the menu. In the Type the location of the item box, Copy/Paste the following text.

%windir%\Explorer.exe Shell:::{4234d49b-0245-4df3-b780-3893943456e1}

2. Click the Next button and name the shortcut  All Apps  and click the Finish button.

You can open this folder like any normal folder and you will see shortcuts to all of your apps. You can drag and drop the shortcuts anywhere you wish or right click one and select Create Shortcut and it will be placed on the Desktop.

You will notice a couple of anomalies in this folder. All of the Modern UI Apps have a white colored icon. This is because these icons are simply symbols from the Segoe UI Symbols Font. When you move a shortcut, try to place it on a dark background for better visibility.

Have fun.

Regards

tryx3 say:

dwelles,

In addition to using Ronnie's advice to create shortcuts for the Windows 10 apps, you can also organise them to suit yourself.

You can add a Taskbar Toolbar with shortcuts to whatever apps, proper applications, individual files or specific folders you want.  You can put subfolders in it and they fly out when hovered over in an XP-like fashion.

You can set up a folder called Menus* anywhere you want. Put whatever shortcuts you want in it in whatever hierarchy of subfolders you want.

You can have shortcuts to anything - you are not limited to those types of shortcuts that can be pinned to the Start menu or Taskbar. 

Then right-click on the Taskbar, unlock it, right-click again, go to Toolbars, New toolbar... and browse to your Menus* folder. Then lock the Taskbar again.

* I chose Menus because it is a short entry that takes up the least possible room on the Taskbar.
