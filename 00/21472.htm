    suzanne s. barnhill say:

    Odd pages must always be recto (right-side) pages; even pages are verso (left-side) pages. This is printing and publishing convention, and you will encounter considerable difficulty if you try to go against it.

    I would advise starting chapters on an odd (recto) page, which can be accomplished using an Odd Page section break; even if you choose to use just a Next Page break, enabling Different First Page will allow you to omit the header on the first page of chapter. If you also enable Different Odd and Even, you can have different recto and verso running heads.

    For more on documents with front matter numbered separately, see http://wordfaqs.mvps.org/FrontMatterRibbon.htm.
