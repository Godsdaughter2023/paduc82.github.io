Associate program with program type - Microsoft Community

captaink1 say:

Associate program with program type

How do I associate a program file type with a program to open it?

captaink1 say: Associate file type with a program.  Example .rar files to winzip.

lemp say:

A.  For some but not all programs

Click Start.

Click Default Programs.

Click "Associate a file type or protocol with a program"

B.  For some but not all programs

Click Start.

Click Default Programs.

Click "Set your default programs" (makes program the default for all the file types it can open)

C.  For most utility programs (should include WinRAR)

Open the program.

Go to the program's "Options" or "Tools" or similar menu

Set the file associations you want

D.  For any program

Right-click on a file of the type you want to associate (e.g., a .rar file)

Select "Open with ..."

Select "Choose default program"

Select the program you want associated with that file type from either the "Recommended Programs" or "Other Programs" or, if the program you want is not listed in either "Recommended Programs" or "Other Programs," click "Browse" and navigate to the exe file for the program in question.

Check the box to "Always use the selected program to open this kind of file" 

Click OK

Hi Mark,

I would like to inform you that the list of codecs was there prior to XP operating system from where you could have un-installed, but this feature is not there anymore.

In Windows 8.1, if you want to see the codecs installed, you need to go to System Information and see the list of the multimedia-audio and video codecs. Follow the steps:

  •  Press Windows Key + R.
  • Type msinfo32 and hit Enter.
  • Select Components.
  • Expand Multimedia
  • Click on Audio and Video codecs respectively  to view the installed codecs.

 I would like to inform you that you will only be able to see the name of the codecs, but you will not be able to un-install them.


If you recall the name of the codec, you may un-install them from add and remove programs.

You may refer to the steps mentioned in the following link and check if you are able to remove them:

Uninstall, change or repair a program

http://windows.microsoft.com/en-us/windows-8/uninstall-change-program
