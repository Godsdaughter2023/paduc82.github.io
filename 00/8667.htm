and again data in A4:A6 copied from another workbook as

To, remove duplicates, select A1:A6, and goto DATA > REMOVE DUPLICATES.

A
B
C
D
E

    With what option did you set up the Google account?

    2cakes say:

    Chose Google, Email/ Calendar only.

    2cakes say:

    FOUND IT!

    Went to Google Mail settings, in the Conversations section, I unchecked the box next to:

    "In conversations show replies and emails from all your synced folders."

    That seems to have fixed the issue.

    entegy say:
    If you chose "email only", then the phone is using IMAP instead of ActiveSync to sync to the Google account and syncing more folders can cause these weirdness. If you choose "email, calendars and contacts", then turn off the parts you don't want afterwards, the phone will use Exchange ActiveSync, which doesn't seem to get weird duplication problems. The downside is that an Exchange ActiveSync connection to a Google account has to be configured at m.google.com/sync.

    hansv mvp say:

    Your keyboard layout has probably been set to United States International. This is a Windows setting, not a Word setting.

    Quit all instances of Word.

    Click the Start button and type intl.cpl in the Search box, then press Enter.

    Activate the Keyboards and Languages tab, then click Change Keyboards.

    You'll probably see your language followed by United States-International in the Default input language dropdown.

    Click on this dropdown and select your language followed by United States (without International).

    If you don't see that in the dropdown list, click Add...

    Find your language in the list, expand it, then expand Keyboard, and tick the check box for United States. Then click OK.

    You should now be able to select it from the Default input language dropdown.

    Finally, click OK.

    Start Word.
