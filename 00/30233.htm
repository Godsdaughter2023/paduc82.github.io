I suspect you think you are booting from the System Repair disk, but in fact are not.  You need to invoke a startup list at the boot.  On Dell PCs that requires pressing F12 several times.  Then you should see a display from which you can select your optical disk drive.

Images are fine as long as the image is not of a corrupted system.  You can not recover a system by restoring a corrupted image.  You will simply re-creating the same issues.

Most systems are in fact very reliable when properly maintained.  Data backup is crucial if your data is valuable to you.  Re-installation of programs is usually the only way to recover a corrupted system.

I observe a practice you may want to consider some day when you have had to re-install Windows.

http://www.canadiantech.info/for-techies/clean-install-tip/ 

Clean Install tip

After Windows 7, system drivers and all updates are installed and any stable applications like Microsoft Office are installed and updated, create a system image.  It will take 3 or 4 DVD +Rs (not -Rs) and about an hour.  When you are done you will have a very nice bit of insurance.  Should you ever again need to re-build a corrupted system or replace a hard drive, you will have a precise duplicate of your system as it was at that point.  You can restore that image to a hard drive in about 20 minutes.  Creation of System Image is found in your menu under Maintenance, Backup and Restore. 

Another great feature about using the image is that you do not need an install disk or a product key to do the re-install the next time.

Also, here is a bit of an instructional on how to re-install Windows 7:

http://www.canadiantech.info/for-techies/re-build-windows-7/

Re-build Windows 7

If you are going to do a re-install, you may be able to take your data off first.  Remove the hard drive and put it into a USB external drive housing.  Connect to a working PC, copy the data off.

Always check to ensure the hard drive you are about to rebuild is actually sound from a hardware perspective.  Download and run your hard drive manufacturer’s drive testing software.

First off, if you have not already tried and failed, you want to try a System Repair disk to boot the PC.  It may be more successful in fixing and offer other alternatives.  If you do not have one, go to any other PC that has windows 7 with the same bitness (32/64).  Start globe, All Programs, Maintenance.  You need just a blank CD.  When you boot, ensure that you are in fact booting on that CD and not the hard drive.  Usually pressing F12 will lead you to the option to do so.

If that does not work, you are likely into a re-install.  Takes a lot of time, but not all attended time.

  1. Factory restore — replaces everything on the hard drive with exactly what it had on it when you bought it.  You access this before Windows starts using a special key that can vary by brand.  F11 is the most common
  2. “Upgrade” install using the Win7 install disk.  You start as if you are re-installing from scratch but when it gets to the point where it offers the Upgrade choice you choose that.
