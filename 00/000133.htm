doug robbins - mvp office apps & services (word) say:

OK, here's a way that you can do it.

For all of the Content Controls that you want to reset, give them a tag of CanEdit

Then, select everything in the document that you want to protect, and insert a RichText Content Control.  Go to the Properties dialog of that Content Control and assign the tag Group to it and tick the box for "Contents cannot be edited"

Then use the following code for you button

Private Sub CommandButton21_Click()
Dim oCC As ContentControl
With ActiveDocument
    For Each oCC In .ContentControls
        If oCC.Tag = "Group" Then
            oCC.LockContents = False
        End If
    Next oCC
    For Each oCC In .ContentControls
        If oCC.Tag = "CanEdit" Then
            oCC.Range.Text = " "
        End If
    Next oCC
    For Each oCC In .ContentControls
        If oCC.Tag = "Group" Then
            oCC.LockContents = True
        End If
    Next oCC
End With
End Sub

The first For Next routine unlocks the Group Content Control, the next one resets the CanEdit Content Controls and the last one re-locks the Group Content Control.

There is nothing sacred about those tag names.  Use whatever you want, but you will need the corresponding name in the code.

steve rindsberg say:

What you're asking for is a combination of what you can have from linked spreadsheets and embedded spreadsheets.  It's possible thus:

Open the spreadsheet in Excel.

Select the portion of data that you want to appear on the first slide, right-click and choose Copy.

Switch to PPT, go to the slide where you want the data to appear.

Choose File | Paste, Paste Special and pick Microsoft Worksheet Object.

Note that you have to click on the SLIDE first, not the thumbnail on the Slides pane.

Repeat for each chunk of content you want to bring in from Excel.

BUT NOTE:

There will be a copy of the ENTIRE workbook dropped into your PPTX for each instance of this that you produce.  

Side effects of this:

Your PPT can become quite a bit larger than you might expect, and

ALL of the data from the original workBOOK will be available to anyone you give the PPT file to.

All they need to do is doubleclick the worksheet object or right-click and choose Edit or Open.  Also, if they do this and change the view in the open instance of Excel, that'll be reflected on your PPT slide.
