nick andre say:

Sorry to hear about your problems! At this point your only bet to recovery is to track down the location of the local OneNote files. They will be stored as SectionName.one in a folder NoteBookName. The easiest way is to try running a search or you can check your settings for the notebook storage location.

If you're going to use OneNote, my recommendation would be to get a free Microsoft Account. Cloud sync through that account functions both for sharing between devices and users, but also for maintaining a backup should you ever run into a problem locally or change computers. You're correct, the current version of OneNote for Mac shouldn't let you do what you have done for the reasons that you're seeing now. The online version is treated as the "Truth" as to what is correct, and your local version is treated more as a cache given that OneNote has transitioned to a cloud-backed application with the most recent generation of apps.

You can also check if OneNote made any backups around your computer, but as the cloud functions as a backup I don't believe the OSX client offers that functionality.
