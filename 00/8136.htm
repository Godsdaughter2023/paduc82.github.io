  4. Click on the More Settings button and then click on the Outgoing Server tab.
  5. Check and make sure that the box for the option: My outgoing server (SMTP) requires authentication is checked (ticked).
  6. Also make sure to select the radio button option below: Use same settings as my incoming mail server.
  7. Also, click on the Advanced tab make sure that the Outgoing Server (SMTP) port number is set according to your local ISP’s information in USA.

Save these settings and test if the account works fine and is able to send emails normally.
