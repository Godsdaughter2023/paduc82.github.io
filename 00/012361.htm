Hi Albert,

 

Thank you so much for posting your query on this forum. You have reached the right place and I will try my best to assist you with this query.

 

I understand that whenever you try to use the “Save as” option with Internet Explorer, the “Save as” dialogue box fills the whole screen and you are unable to reduce its size, as there are no minimize, maximize, or restore buttons on the dialogue box.

 

It would be great if you can let us know – Does this happens with all the application from which try to save a file or is it only with the Internet Explorer?

 

I just checked this on my computer and was able to reduce the size of the “Save as” dialogue box. First, I would suggest you to check if the four edges and corners of the dialog box is "active" or not. This means that you should be able to hover the mouse pointer over one of the corners of the dialogue box and have it change to a double-headed arrow so you can click and drag the edge or corner.
