Never mind. I have already found a solution. Funny that one has to search for a solution to an entirely unrelated question (a.k.a. how to format the inline citation so as to remove the bracket on either sides) to find an answer to this one.

Onward to the answer then. This solution can also be applied to insert an 'et., al' to the second mentioning, and any nth number henceforward, of citations with multiple authors per APA regulation as well. What you require here is a Microsoft Word's concept called 'A field code'.

First, right click on the citation you wish to make an adjustment to-- the trick is that you have to go through each case manually. Then, select 'Edit Field'. To add '2013 as cited in', simply append the following code to the current format.

/f "2013 as cited in "
