chip pearson say:

First, I would run Excel with the RegServer switch enabled. Close Excel, then go to the Windows Start menu and choose Run. There, enter the following and click OK.

 

"C:\Program Files\Microsoft Office\Office14\Excel.exe"  /regserver

 

Of course, your path to Excel may be different.  You can get the path via VBA. Open Excel, then open the VBA Editor (ALT F11), and then open the Immediate window (CTRL G). In that window, enter the following and press ENTER:

 

?Application.Path

 

This will display the full folder name that contains the Excel.exe file.

 

The /regserver command switch causes Excel to start and reset all of its registry keys and file associations and other settings back to "factory defaults". 

 

If this doesn't work, try using Open With. Close Excel and then right-click an xlsx file in Windows Explorer. On the right-click menu, choose Open With. In the Open With popup window, click Choose Default Program... and navigate to Excel.exe, select Excel.exe and click OK. Make sure that the "Always use the selected program to open this kind of file" option is checked.

 

If neither of these work, run Setup.exe on your Office disc and choose the Repair option.

If all else fails, you can always uninstall and then re-install Office.
