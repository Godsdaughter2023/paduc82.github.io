charles kenyon say:
Do NOT replace the normal.dotm template. There is no need to do this to do what you want.
Instead, save your template as a dotx (or dotm if it has macros). This should be placed in each user's Word Startup Folder. If you want, you can have it copied there from a network location during user login.

You do not want to have it shared directly or you will have to have everyone else log off from Word to be able to edit this template.

This is what is called a Global Template.


graham mayor say:
Several issues here:

1. You may not share the normal template. It is a personal system file and copying your normal template to other users is not ideal.
2. You can save your building block in any template that is currently loaded. This can be a document template that uses the building block or an add-in (and of course the normal template and the building blocks template).
3. Given that you have Word 2013 you could setup a Workgroup folder (File > Options > Advanced > File Locations) to store shared document templates and this folder will then appear in the Custom templates when you create a new document.

andreas killer say:
My question is – is there a way to rewrite or add something to the below routine to have it pull in the data line exactly vs. Excel guessing and dropping data? 

Sure, try the code below.

Andreas.

 

Sub Page2()
  Dim strP As String, FName As String
  Dim LastRow As Long
  Dim fso As Object 'Scripting.FileSystemObject
  Dim TS As Object 'Scripting.TextStream
  Dim Lines
 
  strP = Sheets("Sheet1").Range("Q4")
  FName = "C:\Documents and Settings\DoeJane\Desktop\Today's File\" & strP & ".txt"
 
  'Open the file
  Set fso = CreateObject("Scripting.FileSystemObject")
  If Not fso.FileExists(FName) Then Exit Sub
  Set TS = fso.OpenTextFile(FName)
  'Read contents, split the lines into an array
  Lines = Split(TS.ReadAll, vbCrLf)
  TS.Close
 
  Sheets("sheet2").Activate
  ActiveSheet.Cells(1, 8) = strP
 
  'Clear previous results
  Range("A1", Range("A" & Rows.Count).End(xlUp)).ClearContents
  'Store all lines into column A
  LastRow = UBound(Lines) + 1
  Range("A1").Resize(LastRow) = WorksheetFunction.Transpose(Lines)
 
  Range("D1:E" & LastRow).FillDown
End Sub
