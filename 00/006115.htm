wryan1776 say:

This is not Outlook's problem. The problem lies in the way the marketer has designed the e-mail. Most of this stuff is designed in an HTML table. There are two ways to set table width -- at a percentage of the screen space available -- 50%, 80%, 100%, etc., or in a discrete number of pixels -- e.g., 600, 800, etc.  If the screen width is set in discrete pixels and you have a high resolution screen, the table will look narrower and certain items that won't condense, e.g., some typed characters, will cascade down the page. This is the fault of the marketer who should know better. There are several sites that explain all this to the marketers but some of these people seem to be insensitive to the way their advertisement looks on a high res display.

Hope this helps. You know, you could send an e-mail back to the marketer with a screen shot of how the thing looks and tell him/her to fix it in the future. I'd do it if the buggers would pay me for the advice. Most of the time I delete this stuff anyway.
