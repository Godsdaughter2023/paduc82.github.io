Sub RedParagraphMarks()
'
' Red Paragraph Marks
'
    With Selection.Find
        .ClearFormatting
        .Replacement.ClearFormatting
        .Find.Replacement.Font.Color = wdColorRed
        .Text = "^p"
        .Replacement.Text = "^p"
        .Forward = True
        .Wrap = wdFindContinue
        .Format = True
        .MatchWildcards = False
        .Execute Replace:=wdReplaceAll
    End With
End Sub

  2. Run the macro

hansv mvp say:

Create a workbook with the sheets, styles, headers, footers, etc. that you want.

Save it as an Excel template named Book.xltx (or Book.xltm if it contains macros) in your Excel startup folder, by default

C:\Users\<username>\AppData\Microsoft\Excel\XLSTART

The name Book.xltx is obligatory.

When you click the New button on the Quick Access Toolbar or press Ctrl+N, Excel will create a copy of Book.xltx as new workbook.

Create a copy of this template with a single worksheet in the same folder and name it Sheet.xltx (or Sheet.xltm if it contains macros). Again, this name is obligatory.

When you insert a new worksheet, Excel will insert a copy of Sheet.xltx.

This will dynamically calculate the number of columns against the first cell of each row so that some row may have 1 or 2 columns while some may have 25. However, due to the multiple calculations, for a large number of records the processing could become v slow. In that case you may use a vba code as below:

Enter below code in a standard module:

Sub RowsToColumn()
'Values in each row of Sheet1 (B2 till last value of row 2) will be displayed in column B _
of Sheet2 in each successive row
'Column A (Sheet1) values will display in column A (Sheet2), only if any Group values are _
present in the same row

Dim wsSource As Worksheet, wsDest As Worksheet
Dim lastRow As Long, lastCol As Long, rSource As Long, rDest As Long, c As Long

Set wsSource = Sheets("Sheet1")
Set wsDest = Sheets("Sheet2")

'enter headings in first row of Sheet2, after clearing all contents of the sheet
With wsDest
    .Cells.ClearContents
    .Range("A1") = "Name"
    .Range("B1") = "Group"
    .Range("A1:B1").Font.Bold = True
End With

'determines last row in column A of Sheet1
lastRow = wsSource.Range("A" & Rows.Count).End(xlUp).Row

rDest = 2
For rSource = 2 To lastRow
    'determines last column in each row of Sheet1
    lastCol = wsSource.Cells(rSource, Columns.Count).End(xlToLeft).Column
    If lastCol > 1 Then
        For c = 2 To lastCol
            wsDest.Cells(rDest, 1) = wsSource.Cells(rSource, 1)
            wsDest.Cells(rDest, 2) = wsSource.Cells(rSource, c)
            'increments row in Sheet2 for next entry
            rDest = rDest + 1
        Next c
    End If
Next rSource

'auto fit columns of Sheet2
wsDest.Columns.AutoFit

End Sub

Hi

There's an easier and faster way to do this:

Use Get & Transform (for Excel 2016) or Power Query (for Excel 2010/2013).

Load the table using 'from table' import option (within the same workbook), the query editor will open, and within it select the first column, go to the Transform ribbon and on the 'Unpivot Column' split button select ' Unpivot Other columns', this will create you the table you need (with an extra column you can delete).

Selecting 'Load to' at this stage will close the query editor and will create such a table in your workbook.

You can find this functionality in Excel 2016 in the Data ribbon -> Get& Transform section, or in Excel 2010/2013 in the Power Query ribbon (to download and install Power Query Add in for Excel 2010/2013 go here:https://www.microsoft.com/en-us/download/details.aspx?id=39379&CorrelationId=39021b6b-dd98-41f8-bfc3-6bffe8d96e2e.
