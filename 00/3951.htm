IMAP Folders - Microsoft Community

pr-51 say:

IMAP Folders

After years of using POP3 I am doing a partial migration to IMAP. I have a couple of questions regarding IMAP folders.

1. Outlook 2010 has a default "Junk E-mail" folder while the IMAP server has a system "Bulk Mail" folder and the two coexist. What is the best way to handle junk e-mail with IMAP in Outlook 2010? Delete the "Junk E-mail" folder and keep the "Bulk Mail" folder? Or unsubscribe to the "Bulk Mail" folder and use the "Junk E-mail" folder?

2. IMAP has a "Send_Later" folder. Does it serve the same purpose as Outlook's "Outbox" folder?

Thank you for your attention.

J.
roady [mvp] say:

That completely depends on the IMAP implementation on the server and the options you have available there.

 

If it supports XLIST and has the mail folders linked properly, then those folders will be recognized by their function in Outlook 2010.

 

If not, then you'll indeed be left with multiple folders; One for the function in Outlook and one for the function on the IMAP server. This is of course not ideal.

 

In Outlook 2010, you can only re-assign the Sent Items and Deleted Items folders as per:
http://www.msoutlook.info/question/486

 

Often, you can also set or rename the other folders on the IMAP server so that they will be recognized in Outlook. For instance, by renaming the "Bulk Mail" to "Junk E-mail". This might require you to first delete the "Junk E-mail" folder created by Outlook and also recreating the IMAP account in Outlook.

 

I don't know what your "Sent_Later" IMAP folder does. It could indeed be for scheduling emails like the Outbox folder but it could also be your Drafts folder. What purpose does it serve in the web based mailbox?

Use the following code:

Sub GetInfo()

Dim i As Long, j As Long, ID As Long
With Sheets(1).Range("A1")
    For i = 1 To .CurrentRegion.Rows.Count - 1
        ID = .Offset(i, 0)
        With Sheets(2).Range("A1")
            For j = 1 To .CurrentRegion.Rows.Count - 1
                If .Offset(j, 0) = ID Then
                    Sheets(1).Range("A1").Offset(i, 6) = .Offset(j, 4)
                    Sheets(1).Range("A1").Offset(i, 7) = .Offset(j, 5)
                    Exit For
                End If
            Next j
        End With
    Next i
End With
End Sub
