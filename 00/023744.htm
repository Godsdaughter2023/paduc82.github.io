To force Word to autocorrect incudes to includes, you can do that with autocorrect:

jay freedman say:

An alternative is to add "incudes" to your exclusion dictionary, as described at http://wordfaqs.ssbarnhill.com/ExcludeWordFromDic.htm. That is equivalent to removing the word from the main dictionary.

One note: After adding the word to the exclusion dictionary and restarting Word, go to File > Options > Proofing and make sure the option "Suggest from main dictionary only" is NOT checked. If it's checked, the exclusion dictionary is ignored.

Windows Defender is part of the Vista operating system. You cannot "lose" it. It lives and dies with the system, you cannot uninstall it. At the same token, it is already in the system, therefore you cannot install another copy to the system either.

If you have one of the following anti-viruses, AV , installed, they will shut down Windows Defender.

It is because those AVs have their own defenders. To run 2 defenders in the same computer with create conflicts.

The list of AVs are....

Norton
McAfee
AVG
Microsoft Security Essentials.

If you do have one of the above AVs, and you still need to access Windows Defender, please post back and explain why.

If you do not have one of the above AVs installed, this is a way to re-start Windows Defender.....

Start button > in the search box, type Services > press Enter key > scroll to find Windows Defender, click it > ....
under Status heading, it should say ......... Started
under Startup Type, it should say ............ Automatic
If it does not say that, Right click at Windows Defender > click Properties >
at Startup Type, click the pointer at far right and select Automatic
at Service Status, click Start
click OK when done.
