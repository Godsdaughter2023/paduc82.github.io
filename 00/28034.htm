Sub copyit()
Dim MyRange, MyRange1 As Range
lastrow = Sheets("Sheet1").Cells(Rows.Count, "A").End(xlUp).Row
Set MyRange = Sheets("Sheet1").Range("A1:A" & lastrow)
For Each c In MyRange
If c.Value = "Company #1" Then
If MyRange1 Is Nothing Then
Set MyRange1 = c.EntireRow
Else
Set MyRange1 = Union(MyRange1, c.EntireRow)
End If
End If
Next
If Not MyRange1 Is Nothing Then
MyRange1.Copy
End If
Sheets("Company #1").Select
lastrow = Sheets("Company #1").Cells(Rows.Count, "A").End(xlUp).Row
Sheets("Company #1").Range("A" & lastrow + 1).Select
ActiveSheet.Paste
MyRange1.Delete

jay freedman say:

The thread at https://social.technet.microsoft.com/Forums/office/en-US/f1e95b7e-3eca-4048-9d11-10e1249758a7/disable-proofing-spell-checker-using-gpo?forum=officeitproprevious contains information on how to do this in Office 2010. Most of the information is still applicable in Office 2016, but some of it needs updating, and there are some instructions that you'll need if you aren't familiar with the tools.

In the post by Rex Zhang on October 18, 2011:

First, you'll need to download the Group Policy administrative templates for Office 2016 and install them if you haven't already. Be sure to get the 32-bit or 64-bit set, depending on the "bitness" of Office that you'll work with (32-bit is most common).

Start the editor, gpedit.msc, as administrator (privileges are required to set policies).

On the left side, drill down through User Configuration > Administrative Templates > Microsoft Word 2016 > Word Options > Proofing > AutoCorrect. On the right side, double-click "Replace text as you type". In the dialog, click the Disabled option, then OK. That turns off automatic correction of spelling errors. You may also want to disable some of the other items, such as automatic capitalization.

Also under the Microsoft Word 2016 item, open Disable Items in User Interface > Custom. On the right, double-click the Disable commands item. In the dialog, click the Enabled option, then click the Show button in the Options box. Click in the box under Value and type the number 2 (the first one in Rex Zhang's list). Each time you type another number from that list, a new box appears below it. Enter the complete list, plus the ones in the post by B.G. Arets on January 31, 3012, and then click OK in both dialogs.

Double-click the Disable shortcut keys item, and enter the numbers from Rex Zhang's second list.

In the post by enjay_enjay on November 18, 2011:

Start the registry editor, regedit.exe, as administrator. There's a bit of a wrinkle here, because you need administrator privileges to make the changes, but running the editor as administrator means that changes made in the HKEY_CURRENT_USER part of the registry apply to the administrator, not to standard users.

First you'll have to determine which profile(s) belong to the standard users. Drill down to HKEY_USERS. Ignore the entries that only have three numbers, like S-1-5-18, and look at the ones that have a long string of numbers (but not ones that end with "Classes"). Under each of those, click on the Volatile Environment entry if it exists; the USERNAME item on the right side will show whose profile it is.

In each applicable profile, drill down to SOFTWARE\Microsoft\Shared Tools\Proofing Tools\1.0\Office. Double-click the WordSpellingOptions item and enter 0 as its value.

It appears to me that these settings lock down all the ways that students could get spelling and grammar assistance from Word 2016.
