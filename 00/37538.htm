The reason because of which Windows 8/8.1/10 displays this error message is that the network adapter that is used to connect to the network or Internet is by default configured to get turned off automatically to reduce the power consumption. This is mostly the case when you are using a laptop computer and it is running on batteries. Follow the below steps and check if you are able to sort out the issue.

  1. Press Windows + X keys on the keyboard, select Device manager.

  2. Expand the Network adapters from the device manager menu.

  3. Right click on the network adapter (which you use to connect to internet) and select properties.

  4. On you open the properties box, go to the Power Management tab.

  5. From the displayed interface, uncheck the Allow the computer to turn off this device to save power checkbox.

  6. Once done, click OK to save the changes that you have made.

  7. Finally restart the computer and check if it helps.

If the issue perist, follow the below step.

Step 2:

you may run the Network adapter troubleshooter and check if you are able to fix the issue. Follow the below steps to perform Network adapter troubleshooter.

  1. Press Windows + X keys on the keyboard and select Control Panel

  2. Type Troubleshooting in the search bar on the Control Panel.

  3. Click on View all on the left navigation pane.

  4. Click on Network adapter.

  5. Click on Advanced on the Network adapter dialog box and click on Run as administrator (Enter the credentials if required).

  6. Click on Next and follow onscreen instruction complete the Network adapter troubleshooter.

h) Select Next to continue.
