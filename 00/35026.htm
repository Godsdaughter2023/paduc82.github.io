In short, what you saw is not a problem. Running both simultaneously should not be a problem as they have differing priority levels (meaning one should not interfere with the other). It is normal to still see some fragmentation after a defrag routine completes, as not all files can be moved.

The only thing I would be concerned with is the file that your Norton software halted on. That normally only happens when it has either detected a possible infected file, or it is looking for confirmation from the user.
