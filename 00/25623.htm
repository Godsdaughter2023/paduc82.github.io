This error occurs in Windows 10 usually then, when in the background is running a virus scanner, which is certainly the case with many of you. That is why we recommend you to deactivate the virus scanner on your computer or to remove it before your Windows' downloads 10 update.

Should you already be affected by the error C1900208 when you upgrade to Windows 10, then you have now the first delete the failed update again from your PC. This is very simple with the following instructions:

1. Open the Windows Explorer and navigate to the following path: "C: \ Windows \ SoftwareDistribution \ Download"

2. Deletes all files in this folder

3. Now open the bottom left again the Start menu and then the command prompt as an administrator: All Programs -> Accessories -> Command Prompt (Run as administrator)

4. Give now in the console command "wuauclt.exe / UpdateNow"

5. Subsequently, the update from Windows 10 via the Update Center will restart.

The error C1900208 should no longer occur after you disable / uninstall anti-virus software on your computer. After the Windows 10 update is downloaded again, and then installed, you should of course as soon as possible an antivirus software enable or install on your computer.

t-rex1dr say:

box appears saying, "Internet Explorer has stopped working. Windows will now close the program"

A box often appears when using certain programs such as Outlook and YouTube.  It says, "Internet Explorer has stopped working. Windows will now close the program".

sathish s v say:

Hi,

 

-   Have you made any changes on the computer prior to the issue?

 

This might be caused due to various reasons:

-   Due to third party applications.

-   Due to corrupted Internet Explorer files.

-   Due to third party Add-ons.

-   Due to corrupted System files.

 

Perform the following troubleshooting methods:

Method 1:

 

Open Internet Explorer in No Add-ons mode and check how it works. This is to diagnose if the issue is caused by any third party Add-ons.

 

Follow these steps:

a.       Press “Windows key + R” from the keyboard.

b.      Type “iexplore –extoff” without quotes in the Run window and hit Enter.

c.       This will open Internet Explorer in No Add-ons mode.

 

Method 2:

 

I would suggest you to run Internet Explorer performance troubleshooter to diagnose and fix the exact issue with Internet Explorer.

 

Follow these steps:

a.       Press “Windows key + W” from the keyboard.

b.      Type “Troubleshooting” without quotes in the search box and hit Enter.

c.       Select “View all” option from the left panel.

d.      Select “Internet Explorer Performance” option and follow the On-screen instructions.

 

Method 3:

 

Place the computer in a clean boot state and check if it helps. This is to diagnose if any third party application is causing the issue.

You can start Windows by using a minimal set of drivers and startup programs. This kind of startup is known as a "clean boot." A clean boot helps eliminate software conflicts.

To set the computer in clean boot, follow the steps from the link: How to perform a clean boot

Note: After checking in clean boot, follow the step “How to reset the computer to start as usual after clean boot troubleshooting” from the article to reset the computer to normal boot.

 

 

Method 4:

 

Perform System File Check (SFC), and then check if this fixes the issue.

Follow the steps:

a.       Press “Windows Key + Q” to open Charms Bar.

b.      Type “cmd” without quotes in the search box.

c.       On the left pane, right click on the “cmd” option and select “Run as Administrator”.

d.      Type “sfc /scannow” without quotes and hit enter.

Note: It may ask for windows installation DVD to fix and to enable SFC to make more than minor repairs. Some files saved on your computer might get deleted when fixing corrupted files.

 

I hope this helps you to fix the issue. If the issue persists or you have any other issues with Windows, please do post it in Microsoft Community. We would be happy to help you.
