I would run a test, creating a new file in .docx format.  You WILL get resolution issues if you are working in the obsolete .doc format.

Enforce Date Format in Excel Cell - Microsoft Community

lizhughes say:

Enforce Date Format in Excel Cell

Hi.

 

Is it possible to have a cell in Excel default to show __/__/___ so that when the user enters the date it fills in the blanks?

 

I am trying to set up a form whereby users enter a series of dates but currently the cells are just blank and i think it would be clearer for people if they could see what is expected of them when they complete and also the format that the date should be in.


Is this at all possible or is there another alternate workaround?


Thanks.


Elizabeth.

jeeped say:
Conditionally formatting the cells may be an adequate workaround. 
  1. Put a <space> character in each of the form cells that are to receive dates.
  2. Select these cells noting the active cell. You can select cells from more than one location by holding one the Ctrl keys down as you click on them. The active cell should be last one you click. Let's call that C10.
  3. Create a new CF rule based upon a formula using =<active cell>=CHAR(32). In this example that would be =C10=CHAR(32).
  4. Click Format and supply a custom number format of "mm/dd/yyyy" (include the quotes). Optionally go to the Font tab and make it gray, 10pt, italic.
  5. Click OK to accept the formatting and OK again to create the new rule.
The cells with a single <space> should be showing mm/dd/yyyy in gray italic. Typing anything but a single space into them removes the conditional formatting. The mm/dd/yyyy is for display purposes only. The underlying value of the cell is still a single space until a date is entered.
