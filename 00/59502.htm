noor_azam say:

Hello,

Thank you for writing to Microsoft Community Forums.

Usually you will see this error message when the feature for map network drive are not enabled or the drive is disconnected from the network. Please make sure that the drive is connected and to make sure that all the features for network drive are in place, please follow the below mentioned methods and have a check with the issue.

Method 1: Enable the feature.

  1. Open Control Panel.
  2. Then click on all Control Panel Items.
  3. Then click on Programs and features.
  4. Then on the left hand side, click on Turn Windows features on or off.
  5. Windows Feature box will open. Please scroll down and under SMB 1.0 CIFS File Sharing Support, check the boxes next to SMB 1.0/CIFS Client and SMB 1.0/ CIFS Server. If the boxes are already ticked, please uncheck the box and check it again.
  6. It will ask you to restart your Computer. Please Restart it.

Method 2: Turn on Network sharing.

  1. Now open Control Panel once again.
  2. Then click on All Control Panel Items.
  3. Then click on Network and Sharing Center.
  4. Then click on Advanced Sharing Settings.
  5. Under Private, Turn on the Discovery and File & Printer sharing.

Method 3: Check services responsible for Network sharing.

  1. Now we will need to check four services.
  2. Open Run (Windows key + R). Then type "Services.msc" in the run box and hit enter.
  3. The names of the services which you will need to check are:
     Function Discovery Provide Host.
     Function Discovery Resource Publication.
     SSDP Discovery.
     UPnP Device Host.
    Also Check for DHCP Client and DNS Client.
  4. Right click on the above services and then go to properties.
  5. Under General tab, make sure that the Startup type is Automatic and Service status is running.
  6. Then go to recovery tab and change all the entries (next to 1st failure, 2nd failure and Subsequent failure) to Restart the Service.

After following the above methods, please refer this article

  • When do you get this error message?

  • Was there any external hard drive connected?

Hope it helps.
