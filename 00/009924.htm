the "Automatically" is always checked. 

Some security products act as registry protectors and thus prevent users updating their OS the way that they would like.  A commonly suggested fix for this is to try making the change in a safe mode boot, test that change in another safe mode boot and only then try seeing if the change will persist into a normal mode boot.  Chances are that it will, so then your "registry protector" will be helping you make the new values persist.

An alternative you could try which would have to be done each time is to use the Developer Tools, Network capture tool for its cache checking option, Always refresh from server.  (one of the buttons on that tab's Toolbar)
