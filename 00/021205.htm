At the METRO (Start) screen type fonts  (there is no box to type it into.....just type)

You wont see anything until you select "Settings" from the side menu.

 

Select "View Installed Fonts"

Select "Change Font size (left side)

There is a drop down.."Title Bar"  with another drop down to its right showing point size numbers.  (The higher the number, the bigger the letters will be...a 16 point font is larger than an 11 point font).

 

If you right-click on the type of font you selected earlier and select "Preview", it will show you how that font will look in all sizes (numbers to the left of line).

 

So............for example, using the font type you selected in your first post (Internet Options/General),  you can tell it that all Title Bars will be this large (15 points)...all Menus will be this large (11 points), etc.  Just play around with it awhile.  There are limitations, however.  If you want to change fonts on a web page, it might not do that, because a webpage is developed by someone other than Internet Explorer (Microsoft).

 

You can also click on Custom Settings an experiment there with overall size of ALL font.

Don't forget to click APPLY when you are done.
