From the memory.dmp file the driver that appears to be causing the crash is the Neo_VPN.sys :

BugCheck 7E, {ffffffffc0000005, fffff8075883db3b, ffff8d088d81f518, ffff8d088d81ed60}

Probably caused by : Neo_VPN.sys ( Neo_VPN+105a )

The Neo_VPN.sys does appear to be a component of SoftEther VPN which may have been installed at some time.

So, I will give you the following link on uninstalling SoftEther VPN :

https://www.softether.org/4-docs/1-manual/8._In...

First check Apps and Features and uninstall SoftEther VPN if it is listed.

Then delete the related virtual network adapter:

Open the Device Manager and expand Network adapters

Then right-click "VPN Client Adapter - VPN" and select "Uninstall device" and then also select "Delete the driver software for this device" > Uninstall and uninstall the device.

As a general precaution create a System Restore Point beforehand and restart the computer afterwards.

You could also delete the "SoftEther VPN Client" Program directory if it exists but you could leave it.

Note: The above link is a non-Microsoft website. The page appears to be providing accurate, safe information. Watch out for ads on the site that may advertise products frequently classified as a PUP (Potentially Unwanted Products). Thoroughly research any product advertised on the site before you decide to download and install it.

Removing the related virtual network adapter may also remove the netfilter2.sys.

These drivers, which may be older versions, may be conflicting with Bitdefender causing the blue screen error.
