Chroma-keying requires *two* video tracks (one for the green screen video, one for the replacement background image or video).

Movie Maker has only ever allowed one track of video and I don't foresee it changing anytime soon, especially since Movie Maker hasn't been updated since 2012.

Try re-registering the Microsoft Store:

  1. Press the Windows key on your keyboard and type in cmd
  2. Right-click the Command Prompt result, and choose Run as administrator. OR
  3. Press simultaneously the Windows and X keys on your keyboard.
  4. From the menu that opens, choose Command Prompt (Admin).
  5. Once the Command Prompt opens, type in the following command, and press Enter to execute it.
    PowerShell -ExecutionPolicy Unrestricted -Command “& {$manifest = (Get-AppxPackage Microsoft.WindowsStore).InstallLocation + ‘\AppxManifest.xml’ ; Add-AppxPackage -DisableDevelopmentMode -Register $manifest}”
  6. Once the command is executed, close the Command Prompt and try updating/downloading your apps again.

If it doesn't work then try disabling your proxy:

  1. Press simultaneously Windows and R on your keyboard. In the Run window that opens, type in inetcpl.cpl and press Enter or click OK.
  2. Navigate to the Connections tab, and click on LAN settings near the bottom.
  3. Find Proxy server, and make sure that Use a proxy server for your LAN is unchecked.
  4. Press OK twice to close the windows and save the settings. This should solve your issue.

Also, try:

  • Retrying the update/download. It could be a minor glitch
  • Allow it to use your location
  • Use dism.exe /online /Cleanup-Image /StartComponentGroup
  • Check your region and language.
