Is that the most current version of the Pulse VPN client and is the file being detected digitally signed (look for a digital signature tab in Properties of the file)?

These potential false detections can occur with older software which typically aren't signed, though some more current software isn't as well.  It's common for some vendors to forget that an installer is an executable as well and needs to be signed just as with the application files it contains.

If this isn't the problem, then I recommend checking with the application vendor's (Juniper?) support, since they should have a much better pathway to discuss this problem with Microsoft and get it resolved quickly, whether it's their fault or Microsoft's.

Here you go:

Function GenerateRandom()
    Dim Temp(1 To 3) As Long
    Dim Result(1 To 9) As Long
    Dim i As Long
    Dim t As Long
    Dim j As Long
    Application.ScreenUpdating = False
    For j = 1 To 1000
        Temp(1) = 1
        Temp(2) = 2
        Temp(3) = 3
        For i = 1 To 9
            Do
                t = Application.RandBetween(1, 3)
            Loop Until Temp(t) < 10
            Result(i) = Temp(t)
            Temp(t) = i + 3
        Next i

        Range("A" & j).Resize(1, 9).Value = Result
    Next j
    Application.ScreenUpdating = True
End Function
