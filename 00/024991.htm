    Can text flow from wide column to narrow column? - Microsoft Community

    catin dehat say:

    Can text flow from wide column to narrow column?

    A newsletter has two columns - they are not the same width for good reasons - on page 1. Pages 2 through N have two equal-width columns. Pages O through X do not have columns - just ordinary pages. Is it possible for text to naturally (be made to) flow from Page 1, Column 2 to Page 2 Column 1? The rest is drop-dead easy.
    jay freedman say:
    With ordinary text, no. Changing the number or widths of columns requires a section break, and text can't flow across a section break.

    A workaround is to use text boxes. Insert one text box in the first column of page 1, then another text box in the second column, and a third and fourth text box on page 2, and so on. With the cursor in the first text box, click the Create Link button in the Text group of the Drawing Tools tab, and then click in the second text box. From there, create a link to the third text box, and so on.

    You can chain together up to 32 text boxes in one story.

    If your newsletter has more than one story, it would be best to use the text boxes only for the story that starts on page 1, and then use regular columns for the rest.

    Something you may not need for this project but that might come in handy elsewhere: you can start two or three stories on page 1, and have separate chains of text boxes that lead to different inside pages.
