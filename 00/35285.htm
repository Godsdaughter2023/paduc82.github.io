With the exception of Product Key Cards (PKC’s) distributed with COA’s, Microsoft does not distribute products keys as standalone products.   1. Press the Windows key + R.
  2. Type msconfig.exe and press Enter.
  3. Click the Services tab.
  4. Tick the box that hides all Microsoft services.
  5. Untick the remaining services.
  6. Right-click the Task Bar, then left-click the Task Manager
  7. Click More Details
  8. Disable all tasks under the Startup tab.
  9. Reboot.
