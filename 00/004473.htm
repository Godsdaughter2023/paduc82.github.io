brian tillman [outlook mvp 2007-2019] say:

It's probably not your PC that's sending the messages.  The most likely scenario is that your mail address is being used as the sender address on some spammer's junk.  Many of the target addresses for that junk are invalid and so the receiving mail server generates a Non-Delivery Report and sends it to the address it sees as the sending address of the original message.  That address is yours and so you get the bounces even though you didn't send the messages.

Spammers change sending addresses often, so wait a bit and the bounces should stop, or at least decrease substantially.  It happens to everyone, particularly those who post their mail addresses in public forums like Twitter, Facebook, newsgroups, Yahoo! Groups, or any other Internet venue.

Brian Tillman [MVP-Outlook]

Prior upgrading your system to the latest build, I would once have you run the DISM tool and then try updating to the newer version. Refer to the following steps to do so:

  1. Open an elevated command prompt. Press WinKey + X, click Command Prompt (Admin).
  2. Type the following command, and then press Enter. It may take several minutes for the command operation to be completed.
    dism.exe /online /cleanup-image /scanhealth
