doug robbins - mvp office apps & services (word) say:

The following works fine in Word 2010 here:

Dim i As Long
Dim strFind As Variant
Dim strReplace As Variant
strFind = Split("A-C,ACCT#,ADD2,ADDR", ",")
strReplace = Split("|A-C|,|ACCT#|,|ADD2|,|ADDR|", ",")
For i = 0 To UBound(strFind)
Selection.HomeKey wdStory
With Selection.Find
   .Text = strFind(i)
   .Replacement.Text = strReplace(i)
   .Execute Replace:=wdReplaceAll
End With
Next i

Note that using the Split function is a much easier way of creating an array.
