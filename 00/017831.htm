julie sheets say:
Hello,

Where are you entering your 'target' dates?  Are you referring to the start and finish dates for tasks?  Generally, start and finish dates should be calculated by Project based upon: project start date, project calendar, resource calendars, task predecessors, lead, lag, duration.

If you need to adjust those start and finish dates -- what factor is changing?  Is the work increasing?  Are your resources not available as much as originally planned?  You cannot change start dates on tasks if there is progress recorded and there is an Actual Start date. 

Try applying the Tracking table and see if the tasks you are unsuccessful in altering have Actual Start dates.

Julie

bernd p. say:

It's not clear what you mean with offline.

If your notebook is stored in the cloud aka Onedrive you can always work offline if offline means: without Internet access. This is possible because you have a locally cached copy on your computer. Changes you make offline get automatically synched to Onedrive at the time of the next connection to the Internet.

But it sounds that you call it offline if you store your notebook only on your computer, not in the cloud.

This is a reasonable alternative.

You can store notebooks locally with Onenote 2016 as in 2013, but only in a paid version, not in the free one.
