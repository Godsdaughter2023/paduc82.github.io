xircal say:

There's a registry hack which will fix the problem.

Hit Windows key + R to open Run, type: regedit and click OK. Then navigate to the following key:  HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate\Auto Update\OSUpgrade

In the right hand pane, you'll see four values called respectively, AllowOSUpgrade, KickoffDownload, KickoffSource and OSUpgradeInteractive. The values of each of these needs to be modified to read "0" (without quotes) as shown in the screenshot.

To modify a registry value, right click it and go to Modify. In the "Value field", change the single figure from a 1 into a 0. Windows will add the 0x00000000 itself.

After modifying the values in the right hand pane, you have to reboot in order for the changes to take effect.

Before you do any of this, create a manual System Restore point which can be used to wind back the clock to the time you created it just in case you inadvertently delete a key or a value somewhere.

To create a manual System Restore point do the following.

  1. Right click the Start button and go to System.
  2. In the left hand menu, click the link to System Protection.
  3. Click the button at the bottom called Create and then give your Restore point a name, something you'll recognize later and then click OK.

If you have to restore to that point, repeat steps 1 & 2 and then click the System Restore button and find your Restore point.

Running System Restore undoes any changes you made to the Registry.
