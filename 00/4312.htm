It's my understanding that since the TH2 build, you were supposed to be able to use keys from previous versions of Windows to activate Windows 10.  It's very strange that it's not working for you--but then again I've never done a clean install in quite this way, I've always updated from something else.

Unfortunately, the plan you outlined likely won't work--hardware hashing wasn't introduced until Windows 8, and that's what the Insider build is actually looking for: to make sure that your system hash (that is, your particular hardware configuration) is licensed to run Windows 10.

If you have your formulas in column A, this macro will copy the values to column B and paint any starting "z" with white. If you like you can hide column A.

Sub whitez(myrange As Range)
  For Each c In myrange
    c.Offset(0, 1) = c.Value
    c.Offset(0, 1).Font.Color = RGB(0, 0, 0)
    If Left(c, 1) = "z" Then
      c.Offset(0, 1).Characters(Start:=1, Length:=1).Font.Color = RGB(255, 255, 255)
    End If
  Next c
End Sub

Sub test()
  whitez myrange:=Range("A1:A10")
End Sub
