beth melton say:

It's actually recommended that you leave the Normal style unchanged in Word 2007. :-) Instead, we can now modify the Document Defaults which provide your base formatting to your documents. To modify the Document Defaults, in the Paragraph dialog box (Home tab, Paragraph group, click the dialog box launcher), make your changes and then click Set Defaults. You can also use the Styles task pane (Home tab, Styles group, click the dialog box launcher) and use the Manage Styles option at the bottom to modify Document Defaults on the Set Defaults tab. Or you can use the Word 2003 Style Set and set it as the default as Stephan recommended.

The rationale for this is the Document Defaults is as noted above, actually set your base formatting for all styles. In previous versions you can't change your base defaults and we modify the Normal style instead. However, when you alter the Normal style you can encounter style hierarchy issues which are found primarily in table styles. One example is the inability to change the font size using a table style. Here are some steps that show the behavior if interested:

  1. Display the Styles task pane, click Manage Styles, and on the Set Defaults tab, make a note of the default font size, then close the dialog box.
  2. Insert a table (Insert tab, Table)
  3. Add some text in the first few table rows
  4. On the Table Tools Design tab, apply any table style
  5. Right-click the table style you applied and then click Modify Table Style
  6. In the Modify Style dialog box, make sure Whole Table is selected in "Apply formatting to" and change the font size to 14 pts.
    The font should change for the text in the table.
  7. Modify the Normal style and change the font to 10 pts.

    Note the font for the text in the table also changes to 10 pts instead of maintaining the font sized defined in the table style.
  8. If you change the font size for the Normal style back to match the default font size then the font size in the table will return to 14 pts.

Of course if at step 5 you don't see the font size change then the font size of your Normal style has already been changed.  You can still see the bug if you follow step 8 and change the font size of the Normal style to match the default font size.

The same style bug occurs when there is a conflicting format between the table style and the Normal style. For example if in a table you want 3 pts space after each paragraph (which is fairly common to add a little extra space between the table entries) you can't add it through the table style if you modified the paragraph spacing in your Normal style.

Unfortunately this is still an issue in previous versions and style bugs are encountered but in Word 2007 and later you can eliminate the style bugs if you leave the Normal style unchanged.

~Beth Melton Microsoft Office MVP

When you choose a mode with the Touch/Mouse Mode button, it should change the following two entries in the HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\16.0\Common key:

It's possible that an add-in or a permission problem in the registry prevents the mode settings from being written in the registry. A Word add-in would affect only the button in Word's Quick Access Toolbar, while a registry problem would affect all Office programs.

If the problem appears only in Word, use the troubleshooting steps in Methods 6 and 7 in http://support.microsoft.com/kb/921541. (Ignore the earlier steps, because they won't have anything to do with add-ins or the Quick Access Toolbar.)

bob jones aka: cybertaz say:
Word's Backgrounds are primarily for on-screen display -- primarily if you use Word to create basic web pages. They aren't intended for printing because:

  • They cause significantly longer print times
  • Large areas of 'solid' color are usually inconsistent when printed
  • The difference between print/display technology often results in color variance
  • Ink/toner consumption is outrageous
  • Over-saturation of a sheet of paper can cause bleeding [reduced readability]
  • Most printers will not print the Background edge-to-edge... there will typically be a minimum margin imposed resulting in an unprinted border

If you do want them to print go to Word> Preferences> Print then check the box to Print background colors & images.
