jeeped say:

I need a VBA code that will rename the file names of every TIFF with the reference number contained within it.


TIF (e.g. Tagged Image File Format or TIFF) files are image files with the distinction that they can be multiple pages. As such they cannot be read in the conventional sense. If you wanted to read the text it would first have to be processed through some OCR (e.g. Optical Character Recognition) software that will attempt to convert a picture of text into text that can be read, parsed and used for other purposes.

Sub OCRReader()

Dim doc1 As MODI.Document
Dim inputFile As String
Dim strRecText As String
Dim imageCounter As Integer

inputFile = Application.GetOpenFilename
strRecText = ""
Set doc1 = New MODI.Document
doc1.Create (inputFile)
doc1.OCR
For imageCounter = 0 To (doc1.Images.Count - 1)
    strRecText = strRecText & doc1.Images(imageCounter).Layout.Text
Next
fnum = FreeFile()
Open "C:\TEMP\testmodi.txt" For Output As fnum
    Print #fnum, strRecText
Close #fnum
doc1.Close

End Sub

don cuthbert say:

Enterprise is somewhat more complex. You can get a temporary Product Key for Windows 10 Enterprise or one issued through your corporate servers. There is no free upgrade from Windows 8.1 Enterprise to Windows 10 Enterprise (nor from any other Windows edition since July 28th) so having a valid Windows 8.1 Enterprise Product Key is irrelevant to Windows 10 Activation.

If you use Assistive Technologies, you can still upgrade for free to Windows 10 from Windows 7 or 8 Home or Pro, but that requires an activated install of Windows 7 or 8 for that device.

So if your corporation can issue a Windows 10 Enterprise Product Key, you can use that to Activate Windows 10 Enterprise. I don't know if Enterprise edition uses Digital Licenses for Windows (which are assigned per device) like Windows 10 Pro and Home do, but perhaps once Windows 10 Enterprise has been activated on a Device, it gains a Digital License which would auto-activate Insider builds installed on that same device.

There is something about "Windows as a Service" that may be applicable with Enterprise, which means Windows Licensing is based on a subscription. If a corporation sells a Windows 10 Enterprise device to a private individual, does it retain any Windows 10 license?

Perhaps you have no reason to use Windows 10 Enterprise instead of Home or Pro. You could always install Pro or Home without activating Windows and it will be functional, except for items under the Personalization section of the Settings App. Or you can purchase a License for Home or Pro.
