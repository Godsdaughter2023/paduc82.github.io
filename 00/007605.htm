zhongqing tang say: Hi spacekowboy,

There are two common causes for slow opening form.

1. creation of the LDB file/contention for it with existing users. 
This problem is usually resolved with some form of the solution from Tony's LDB Locking article. You can tell if this is the cause of the problem if opening the first form is slow, and opening successive forms are not slow if you leave the initial one open.

2. out-of-date metadata in linked tables. 
This can happen if, for instance, you work on the front end on your test server, move it to the production environment and update the connect strings to point to the production back end. Updating the connect string doesn't refresh all the metadata stored in the linked table definitions, and there's no way to actually fully refresh them. So, you have to delete and recreate the linked tables in the production environment. The symptom of this one is that in the test environment forms open immediately or in only a second or two, and in the production environment take a minute or more to open. After opening, they generally work just fine.

Saving and Deleting Attachments from Multiple E-mails - Microsoft Community

aks paul say:

Saving and Deleting Attachments from Multiple E-mails

In Entourage you had the ability to select multiple e-mails and then either save the attachments or delete them. In Outlook it appears you can only use these commands with a single e-mail message at a time. Is there any way to process attachments on multiple e-mails simultaneously? 
talking moose say:
You can accomplish this with a couple of rules.
  1. Go to Tools menu --> Rules and create a new rule based on the type of email account you're using (POP, IMAP or Exchange). Note that with IMAP accounts you will not have the option to delete attachments from messages.
  2. Name the rule something like "Save attachments...". Apply it to "All messages", choose the "Save Attachments" option and choose a destination folder such as your Desktop or Downloads folder.
  3. UNcheck the Enable option. You don't want this rule triggering when you receive new mail.
  4. Click the OK button to save the new rule.
Follow the same steps to create a "Delete attachments..." rule. You'll, of course, choose the option to "Delete attachments" instead. Again, be sure to UNcheck the Enable option.

To use these rules you'll select your multiple messages, right-click or Control-click one of them and select Rules --> Apply --> Save attachments... or Rules --> Apply --> Delete attachments...

Hope this helps!

Add work hours to a date and time - Microsoft Community

philip bendall say:

Add work hours to a date and time

Can anyone help-how do I add a specific number of hours to a date and time to exclude weekends. Work day 8.00-16.00. For example add 24 working hours to 21/01/11 14:00. Answer should be 24/01/11 14:00. Note 21/01/11 = 21 January. To further complicate this I would also like to exclude public holidays as well as weekend.
lars-Åke aspelin say:
Can anyone help-how do I add a specific number of hours to a date and time to exclude weekends. Work day 8.00-16.00. For example add 24 working hours to 21/01/11 14:00. Answer should be 24/01/11 14:00. Note 21/01/11 = 21 January. To further complicate this I would also like to exclude public holidays as well as weekend.

With your date and time in cell A1 and the number of hours to add in cell B1, and your list of holidays in cells C1:C20, try the following formula:

=WORKDAY(A1,INT(B1/8)+((MOD(A1,1)+MOD(B1,8)/24)>2/3),C1:C20)+MOD(A1,1)+MOD(B1,8)/24-1/3*((MOD(A1,1)+MOD(B1,8)/24)>2/3)

Note that the expression (MOD(A1,1)+MOD(B1,8)/24)>2/3 is calculated twice. Since there might be a rounding issue the evaluation might not be the same for both these occurances and that will lead to a mess in the result. I thus propose to use another formula where this expression is only calculated once:

=IF((MOD(A1,1)+MOD(B1,8)/24)>2/3, WORKDAY(A1,INT(B1/8)+1,C1:C20)+MOD(A1,1)+MOD(B1,8)/24-1/3, WORKDAY(A1,INT(B1/8),C1:C20)+ MOD(A1,1)+MOD(B1,8)/24)

You could also add a small number in order not to get a end time of 08:00 where you expect 16:00 the day before.

So this is the formula I suggest:

=IF((MOD(A1,1)+MOD(B1,8)/24)>2/3+0.0000000001, WORKDAY(A1,INT(B1/8)+1,C1:C20)+MOD(A1,1)+MOD(B1,8)/24-1/3,WORKDAY(A1,INT(B1/8),C1:C20)+MOD(A1,1)+MOD(B1,8)/24)

Make sure that the cell you put this formula in is formatted to show both date and time.

Hope this helps / Lars-Åke
