hairyfool say:

The licence itself is for one computer and therefore can only be active on one computer at a time. In strict terms it has to be removed from the original computer before installing on another. There is no specific limit how many times this can be done however if the same key is re-authenticated too many times and within a short period it may be flagged and you would have to call the authentication centre.

 

The main caveat is with OEM licences. Many buy this type of licence from on line retailers as it appears to be a significantly discounted version however this version can only ever be installed on one computer. You can re-install it on the same computer but it cannot be transferred to any other computer, even if that 1st one has failed.

a. user say:

Is there anything unique about the ones that don't work compared to the ones that work?  Are they all larger sized (sav over some set GB number)?  Are they all the same filetype (whereas the others are a different filetype)?

It is possible that the files on the DVD/CD are corrupted and you're freezing when it hits a corrupt file - have you tried to exclude those that don't work the first time through to see just how many you can get to copy over (there may only be one or a few exceptions)?  Have you tried playing a movie that won't copy to see if it will play all the way through - that would tell you pretty much if there was a corrupted file?

In the future, I would suggest you try copying at lower compression - that usually makes a copy with less risk of corruption and a higher quality of image/sound (though it does cut back on the length of the movie you can copy) - especially if the alternative is that you don't have a valid copy.  Here's more info on this: http://hometheater.about.com/od/dvdlaserdiscvideocd/qt/dvdmodewrite.htm.

Can your machine take any more RAM (mine can't - it's maxed at 2GB by the motherboard) - you could use a little more than 3GB (meaning add up to 4GB max) with a 32-bit OS (if your motherboard and processor can accommodate it - check with the manufacturer to find out the limitations).  I don't know if that would help (I honestly don't think so because of virtual memory but I could be wrong), but it certainly could do no harm (and might benefit you in other ways - like opening more windows and multi-tasking better (and doing it all faster)).

I hope this helps. Let me know how it goes no matter the case.  I'm not sure what else to suggest at this point, but maybe hearing the feedback will spark something that might be of use if none of this works.

Good luck!

Lorien - One - MCSE/MCSA/Network+/A+
