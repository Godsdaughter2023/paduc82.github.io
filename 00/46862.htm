john r h say:

Is Windows Defender Required if Avast Pro is running?

Avast Pro is running for anti-virus protection. Is Windows Defender and Mictosoft Security Essentials recommended?
ken blake say:
Avast Pro is running for anti-virus protection. Is Windows Defender and Mictosoft Security Essentials recommended?


 

Windows Defender is an anti-spyware program and Avast is an anti-virus.You should be protected against *both* viruses and spyware, so yes, having Defender running is wise.

 

However you should *not* run both Avast and Microsoft Security Essentials. They are both anti-virus programs, and running two at once is a very bad thing to do, It can cause problems as they conflict with each other, and also cause slowdowns in performance.

 

Either Avast or Microsoft Security Essentials is a good choice. Choose one, but don't run both.

 

le boule say:

As stated by Ken Blake in his post, MSE cannot be used with Avast.  Both are antivirus/antimalware products, both provide real-time scanning and as such would conflict with each other.

You should never have more than one security product installed on the PC providing active protection/scanning. This can cause performance issues, system instability and conflicts between programs, and can hinder the effectiveness all installed antivirus products.

Avast Pro does not have a firewall: http://www.avast.com/en-us/pro-antivirus#tab3

If you also want an Avast FW you need to use Avast Internet Security.

The Windows Firewall will work with Avast Free or Avast Pro.

Further questions re Avast suggest you contact: http://www.avast.com/support

Hi Edward,

To prevent automatic driver updates, try the following

Click the Start button, search Control Panel
Click System.
Click Advanced system settings
Switch to the Hardware tab.
Press the Device Installation Settings button.
When prompted for automatic updates, choose No, and then Save Changes.

This disabled automatic updates for all drivers, so you will need to manually update drives you wish to update.
