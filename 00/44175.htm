arminatorx say:

When you redeemed your Java Minecraft on Mojang, you have gotten a 5x5 code to redeem Minecraft on the Windows 10 store. 

This redemption is a separate service of your Mojang account, and it's possible to use a Microsoft Account email that is different from your Mojang account email. 

Mojang (which gives you Java Minecraft) is separate from the Microsoft Windows Store App, which uses a Microsoft Account email when you redeemed that 5x5 code. 

After redemption of that code, the Windows 10 Minecraft is tied to the Microsoft Account email you used on your Windows 10 PC. From then on, you don't really need the Mojang account anymore, if you only want to play on Windows 10. You will then get the game from the Windows 10 store only, with the E-Mail account you used on the Windows 10 store during code redemption.

So make sure you are signed in with the Microsoft Account email (which might not necessarily be the Mojang account email) when you use the Windows 10 Store app to re-download your Windows 10 Minecraft again.
