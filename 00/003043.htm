How do I double the value of a cell so that it can be copy pasted down the column for all cells in that colume

I have column of 1600 different prices values in each cell

Is there a formula that I can put in the 1st cell to double the value and copy paste it all cells in the column to double all values

hansv mvp say:

Let's say the prices are in C2:C1601.

Option 1:

In D2, enter the formula =2*C2

Fill down from D2 to D 1601.

Option 2:

Enter the number 2 in an empty cell outside the range with prices.

Select this cell and copy it (Ctrl+C).

Select C2:C1601.

Right-click anywhere in this range and select Paste Special... from the context menu.

Click Multiply, then click OK.

You can now clear the cell in which you entered 2.

Sub AddLegend(ChartName)

    ActiveChart.Legend.Border.Color = RGB(0, 0, 0)
    ActiveChart.Legend.Border.Weight = xlHairline
    ActiveChart.Legend.Position = xlLegendPositionLeft
   
End Sub

rich michaels - mvp (greatcirclelearning.com) say:

This macro will do the job ...

Sub DeleteSpellingErrors()
    Dim rng As word.Range, i As Integer
    If Selection.Range.Start = Selection.Range.End Then
        Set rng = ActiveDocument.Content
    Else
        Set rng = Selection.Range
    End If
    If rng.SpellingErrors.Count > 0 Then
        For i = rng.SpellingErrors.Count To 1 Step -1
            rng.SpellingErrors(i).Delete
        Next
    End If
End Sub
