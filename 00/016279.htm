hansv mvp say:

I'd use VBA instead of a macro. The VBA equivalent of the Output To action, DoCmd.OutputTo, overwrites the existing file without prompting.

For example:

 

Private Sub cmdExport_Click()
    DoCmd.OutputTo acOutputReport, "MyReport", acFormatXLS, "ExportedReport.xls"
End Sub
