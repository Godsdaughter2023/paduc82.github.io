rich michaels - mvp (greatcirclelearning.com) say:
Computers talk colors in RGB (Red, Green, Blue) codes, not Pantone codes.  So you have to convert from one to the other.  Here's a link to a site I just found that lists the conversion numbers. Converting Pantone Colors to RGB.

The color of a font is an attribute of Style and not the specific font type.  So to set a specific font's color you really assign the font type and color to a particular Style.

I recommend that your create specific Style name for your "Branded" colors.  In other words, create a Style named "Wendy Title Text" and set the Font type to what you want and assign the Color to the RGB value that matches your Pantone.

Word > Help has good information for creating Styles.

Hope this helps

bob jones aka: cybertaz say:
Another option:

If you go to the Font Color button on the Formatting Toolbar or the dropdown in Format> Font dialog you can select Colors dialog click the second icon from the left for Color Sliders then open the dropdown & select CMYK Sliders. Enter the CMYK values for your Pantone color then drag the large swatch at the top to one of the cells in the lower grid. You can then access the More Colors dialog to apply the custom color to text or drawing objects whenever you wish. However...

Regardless of anything else, Word does not do Color Separation, which is necessary to accurately produce CMYK. CMYK is based on fluid (ink) technology [percentages of Cyan, Magenta, Yellow & Black], RGB is based on light [intensity of Red, Green & Blue]. No matter what you do, the color will be converted to RGB as Rich suggests, and there is not a direct conversion from RGB to CMYK for all possible colors in the respective gamut [or range] each technology is able to produce. In short, unless the document is being printed commercially, CMYK means virtually nothing.

The overwhelming majority of local printing devices use RGB, so the color will be converted regardless of whether the printer is being fed RGB or CMYK values. It's just a matter of whether the conversion to RGB is done beforehand or is done by the printer's driver. Only when using software that supports color separation (such as Adobe InDesign)
