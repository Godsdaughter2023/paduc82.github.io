marilyn o say:

Hello ljrjr4,

In addition to what Praseetha has stated, you might try the following:

The cause of this error is due to the fact that the version of wmp.dll in the registry is out of sync with what is installed.

 

The wrong wploc.dll version error is a common problem that usually happens when the original “wmploc.dll” file in “C:\Windows\System32″ folder is replaced.  

To fix the incorrect wmploc.dll issue, follow the two Methods.

 

Method 1:

 

a.    Go to Start and type cmd.

b.    Right-Click on cmd and select “Run as administrator”.

c.     In the  Command Prompt type the following and press  Enter:

                       UNREGMP2.EXE /UPDATEWMPVERSION

             d.  Close the command prompt after running the above command. 

 

Run Windows Media Player.

 

Method 2: Re-register the wmp.dll file

 

Here’s how:

 

a.    Open elevated command prompt.

b.    Type “regsvr32 wmp.dll”without the quotes in the command prompt and press Enter.

 

See if Windows Media Player works.

 

Method 3: Disable and re-enable the player (You may have already tried this step)

 

This should fix the issue.

 

a.    Go to Start and in the search bar type "Turn Windows features On or Off".

b.    Click on "Turn Windows features On or Off".

c.    Browse to Media Features and place a check mark in front of Windows Media Player.

d.    Restart the Computer.

 

Please reply back with an update and let us know if you still need help or if your issue is resolved.

patrick barker say: Hi,

All of the attached DMP files are of the DRIVER_IRQL_NOT_LESS_OR_EQUAL (d1) bug check.

This indicates that a kernel-mode driver attempted to access pageable memory at a process IRQL that was too high.

A driver tried to access an address that is pageable (or that is completely invalid) while the IRQL was too high. This bug check is usually caused by drivers that have used improper addresses.

They are consistently faulting NETIO.sys which is the Network I/O Subsystem. Generally, in network related crashes like this, they are caused by one of two things:

1. Network drivers themselves need to be updated, or incompatible adapeter, NIC, etc.

2. 3rd party antivirus or firewall software causing network conflicts.

Do you have any security software installed? I took a quick look but did not see any.
