barb reinhardt say:

Several questions:  

1)  I presume at some point the first date column would be deleted.   I'd probalby put a worksheet level named range on.   Let's assume that the dates are in ROW 2 and the headers are in Row 1.    I'd create a worksheet Level range called Date_Range and define it like this

 

=OFFSET(Sheet1!$M$1,1,1,1,COUNT(Sheet1!$2:$2))

 

This would expand/contract based on the number of numeric entries in row 2.   This assumes that the only numeric entries are the dates.  

 

Here is the first part to hide the dates that are earlier than 24 weeks

 

Option Explicit

Sub HideDates()
Dim myWS As Excel.Worksheet
Dim myRange As Excel.Range
Dim r As Excel.Range
Dim myHideRange As Excel.Range
Set myWS = Sheet1  'Assumes that worksheet code name for sheet is Sheet1

Set myRange = myWS.Range("Date_Range")
Set myHideRange = Nothing
For Each r In myRange
    If r.Value2 <= Date - 24 * 7 Then
        If myHideRange Is Nothing Then
            Set myHideRange = r
        Else
            Set myHideRange = Union(myHideRange, r)
        End If
    End If
   
Next r

If Not myHideRange Is Nothing Then
    myHideRange.EntireColumn.Hidden = True
End If

End Sub

 

That should get you started.    I'd set a named range for the first non-numeric column and insert a column before that one.   Record a macro to do what you want and it can be modified.

jason (office engineer) say:

Sways created using an Office 365 account (granted by an organization like a business or school) are only available to other users in that organization, by default.  To change this so that people not signed in or signed in as an account outside that organization, the owner needs to open the Sway for edit, open the Share dropdown, and change the permissions to Anyone with link.

Some organization admins do block that ability for their users so if that option is not there you would need to contact the admin to make it available.  Another option, in this case, would be to duplicate the Sway to a personal, non-Office 365 account in which Sways are always publicly available.
