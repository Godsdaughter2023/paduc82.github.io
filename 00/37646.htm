tedburfeind say:

I have unsent messages in my Outbox

[Original title: Windows Live Mail 2012]

¡firedog say:
... an error message appears saying "You have unsent messages in your Outbox.

If this message appears, then there are unsent messages in your outbox (below the account folders in the folder pane). You may have to click Work offline to be able to remove them.

iwant2bhere say: Question for ifiredog - Your answer was to Click Work Offline but you did not explain how to get to the option Work Offline. I am having a similar problem with the Work Offline option not working and I don't know how to turn that option off or on. ¡firedog say:

Clicking Work offline will normally take Windows Live Mail offline for long enough to release the program's handle on unsent messages in the outbox. The state will rarely last long, though, because any HTTP request will change it to online again if IE11 is installed. If you're having difficulty going online, it should be enough to open IE11 and navigate to a web page (to send an HTTP request).

This sorry situation arose because third-party browsers, notably Firefox and Chrome, chose to interfere with the Windows registry setting GlobalUserOffline, which most programs had honoured previously. This is the registry key whose value - yes or no, 1 or 0 - could be toggled by the Work offline command on Internet Explorer's File menu. The problem you face is not experienced by those - like me - who have managed with IE since 1998 and eschew other browsers. But because these invasive third-party applications are now so prevalent, the IE developers decided to throw in the towel and ignore that registry value altogether. This is not good news for users of older software - like Windows Live Mail - some of whose functionality depends on it.

Summary

The Start menu and Taskbar have been hallmarks of the Windows desktop experience for more than 20 years. Both have evolved over the years with features and enhancements to make users productive and more agile at their everyday computing tasks. Windows 10 introduced an assortment of enhancements and productivity boosters. In particular, both are optimized for Touch, Keyboard and Mouse; making users feel right at home, no matter the device or form factor they are using.

Windows 11 features the most dramatic overhaul since the launch of Windows Vista in 2006. Not only does it look different, but the way users interact with it works a bit differently from Windows 10 and earlier versions. The Taskbar which hosts your app shortcuts and running programs looks familiar, but the immediate differences such as the center alignment and behaviors present new experiences for users. The Taskbar

The Taskbar is another important element of the Windows Desktop, it is the first thing you see when you sign into Windows. Every time you start a program or open a window, a shortcut is placed on the Taskbar with a visual indicator at the bottom of the program telling you it is running. When you exit a program, it will disappear from the Taskbar or the indicator will no longer appear below the program.

Read all this info on how to Network XP and Vista, including using the XP File Sharing Wizard.

Permissions/Share info is there as well.

 

If using Norton, McAfee, Trend Micro or Kaspersky I.S., make sure File and Printer Sharing is enabled in THEIR firewall (or LAN allowed, depending on how their Exceptions are worded in their Firewall)

 

1 thing to do is make sure that the Workgroup Name of ALL the computers is the SAME.

“How to Set Computer Name and Workgroup in Windows XP”

 

In Vista Network and Sharing:

 

Network Discovery: ON (So it can see the other computers)

 

Network set to Private (Public is for hotspots, airports, etc)

 

File Sharing: ON

 

Public Folder Sharing: ON (Vista’s Public Folder is the same as XP’s Shared Docs)

 

Password Protected: OFF (unless you want to set up identical usernames and passwords on ALL computers in your Network) If you have it ON, and no identical usernames/passwords set up, you will be asked for a username and password when you try to access a Vista computer from an XP computer, or a Vista computer.

 

XP File and Printer Sharing info:

 

Also, run the XP’s Home or Small Office Network File and Printer Sharing Wizard on the XP Computers, even if you had an XP Network set up prior to adding a Vista computer to it(redoing the Wizard seems to work for XP machines!).

 

In “My Network Places”: “Set up a Home or Small Office Network”

OR under Accessories > Communications > Network Setup Wizard > Allow File and Printer Sharing.

, delete high pitch alarm , noise from sound system - Microsoft Community

plantestephane say:

, delete high pitch alarm , noise from sound system

Need help ; loud high pitch squeak ,alarm , siren continuously on with volume . when I turn volume off ,way down ,no more sound 
RealTek system for windows 7 in Acer Aspire Notebook 7736-6080 
been at this for over 24 hrs can't shake it unless I disable sound system . Thank You !

lemp say:
If it's really a sound being generated through your speakers -- and from your description, it seems to be -- then you need to determine what application program or part of Windows is triggering the sound.

I don't know of any easy way to do this.  You could begin by restarting the system in a "clean boot" configuration, which minimizes the number of things that are running.

See http://support.microsoft.com/kb/929135

Does the sound stop when you reboot after having disabled all of the "Startup" items and non-Microsoft services? If so, see the "More information" section of the article linked above for suggestions on identifying the problem.

If the sound persists even though you have disabled all of the Startup items and non-Microsoft services, then it seems as if the sound is being triggered by Windows.  I don't know of any Windows condition that will continuously trigger a sound.  The only suggestion I can think of is to go to Control Panel > Sound and select the "Sounds" tab.  Select each event in turn and click the "Test" button until you identify which event and wav file is the one you are hearing.  This may or may not be possible if, in fact, you are continuously hearing the sound (that is, you might not be able to distinguish what happens when you click the Test button from what is happening all the time).
