hansv mvp say:

On the Insert tab of the ribbon, click Symbol > More Symbols...

Make sure that (normal text) is selected in the Font dropdown, and that Unicode (hex) is selected in the "from" dropdown near the bottom.

When you click on any character not in the "standard" character set, you'll see its character code in a text box. For example, the Yen character ¥ has character code 00A5. You can insert this character into Word by typing this character code (you may omit leading zeros), then pressing Alt+X.

Some letters with accents have built-in shortcuts in Word. For example, the letter é kan be inserted by pressing Ctrl+' then e. These shortcuts are also displayed in the Symbol dialog.

Finally, you can assign your own keyboard shortcut for characters you need frequently by clicking Shortcut Key... in the Symbol dialog.

Other applications are less flexible, so it may not always be possible to use or define keyboard shortcuts for special characters.
