How to add Alias, Remove and Set Primary Alias for Microsoft - Microsoft Community

jhenny m. say:

How to add Alias, Remove and Set Primary Alias for Microsoft Account?

An alias is an additional email address for your Microsoft account. It uses the same inbox, contact list, and account settings as the primary alias.  You can sign in to your account with any alias—they all use the same password. You can send email from an alias whether you’re using a mail app like Outlook or a device such as a phone.

An alias is also the best way to change your email, but keep all your mail. Add an alias, then make the new alias primary. Then you can keep or remove the original alias. If you have a @hotmail.com, @live.com, or @MSN.com email address and would like to add a new email address, the best way is to add an alias.

Sign in your account. "www.account.live.com"

Select "Your info".

 

Select "Manage your sign-in email or phone number".

Since you are going to access sensitive information. You need to request for a code. 

If you are using your Own device. You can CHECK the box to access sensitive information without the code.

Click SUBMIT. 

Select "Add email alias".

Enter a new email address, and then click Add alias. If the address you request is already taken, you’ll be prompted to enter a different address.
Add alias. If the address you request is already taken, you’ll be prompted to enter a different address.
- or -
Select Add an existing email address as a Microsoft account alias, enter an existing email address, and then select Add alias.

Add alias.

Select Make primary next to  the alias you want as your primary.

Select Remove next to  the alias you want to remove. As soon as you do this, that address will become available to anyone else who wants it after 30 days.

Note:

If you remove an alias from an EASI account, that address will immediately become available to anyone else who wants to use it.

If you remove an @Hotmail/outlook.com or an @MSN email address, it cannot be recovered.

REMINDERS:

 

❤You can create up to ten new aliases per year, for an overall maximum of ten. Deleting an alias removes it from the overall count, but not the yearly limit.

  • ❤Existing @hotmail.com, @live.com, and @msn.com addresses can't be added as aliases.

  • ❤You can create an alias using an existing email address from other providers such as Gmail, Yahoo! Plus, or  AIM Mail, as long as you haven't used the address to create a Microsoft account.

    ❤You can change your primary alias up to twice a week.

    ❤If your primary alias ends in @msn.com or @live.com, and you change it, you won't be able to make it primary again.

    stefan blom say:
    "Link to Previous" only controls what is displayed in a header (or footer). Click on the page where the numbering restarts and then click Insert | Page Number | Format Page Numbers. Which option for page number restart has been selected?

    jw aldridge say:

    Thanks! 

    I found out the real issue was, that I was using Brackets 2 for the page numbering format. Because of the size of the font, it would only display one of the digits starting after 9 so, it would appear that it was starting over at 0, 1,2,3...after getting to the number 9.

    It's a formatting issue... 

    Thanks again!

    stefan blom say:

    Thanks for the follow-up.

    Yes, the "Brackets 2" built-in page number format puts the page number inside a shape (whose dimensions are fixed) and that means that part of the page number may "disappear."
