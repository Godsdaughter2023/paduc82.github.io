Just how are you attempting to set it in the registry?  Where specifically and what are you putting in there?

 

Assuming you want it for the LOGON screen:

  1. Launch the Registry Editor (Regedit.exe).
  2. Go to HK_USERS\.DEFAULT\Control Panel\Desktop.
  3. Locate and double-click the SCRNSAVE.EXE string value.
  4. When you see the Edit String dialog box, type the name of the screensaver you want in the Value Data text box and click OK.  I would use the FULL PATH to the screensaver.
  5. Locate and double-click the ScreenSaveTimeOut string value and change the value from 600 seconds to another number, such as 120 for two minutes.
  6. Check the ScreenSaveActive value and make sure it is set to "1".
  7. Close the Registry Editor and restart Windows XP.

If it is your own screensaver:
2. Go to HKEY_CURRENT_USER\Software\Policies\Microsoft\Windows\Control Panel\Desktop
