A few thoughs on the forum and WinTP - Microsoft Community

eddwalker say:

A few thoughs on the forum and WinTP

After following this forum for several months and testing WinTP on a few machines I've come up with a few observations.

1. People seem to be split on the Metro interface - Personally I'm not a great fan of it but it does have it's uses. As a happy medium I'd suggest increasing the ability to personalise the interface so people can have it looking how they want? I know theres never going to be a way to make EVERYONE happy, but things like disabling windows classic shell seems a bit unessicary?

People also seem to get very... passionate... about some of the changes. It's good to see some people still care, but keep it civil? A number of discussion posts just seem to be people arguing for the sake of it and never getting anywhere.

2. The people with the "community moderator" tags are very good on this forum. They actually help people. Most of the "microsoft support engineer" posts seem to be copy/paste of 'solutions' without really reading the thread. For example; my first question post on here turned out to be a switchable graphics driver issue causing a bootloop after updating to build 9926. a fresh re-install yielded the same problem. the support engineer suggested it was a virus on a fresh install on WinTP, the community mod solved my problem on his first reply. I've seen this a few dozen times since then.

3. As this is a good forum for discussion I think some of the feedback into the current development builds should come from here. I've had problems with the feedback app on both PC's i have WinTP running on since one of the build 9926 updates meaning I can't leave feedback. However a single persons feedback compared to a community effort and agreement seems worlds apart?
I saw one discussion post last week (couldn't see its title on the first couple of pages) where people were making a list of suggestions they'd like to see enacted. But a single person was then going to submit it through the feedback app dispite the input from so many people working together! Especially since the Me Too was removed from the app (before it stopped working for me).

Also It would be nice to find out what was actually happening to our feedback, is it individually read? or generalised and aggregated into bulk form? or does it just vanish into the ether? out of the suggestions ive seen posted on here and the app I don't think I've seen many of them added into the following builds...

4. Drop the Microsoft Account nonsense. Or at least don't force it on people or they'll reject it. Having an option to opt out of having one should be mandatory. I ended up downloading and installing skype from their website because I didn't want to set up yet another online account that I don't want.

Everyone is going to want something different from this OS as it's going to be used for a multitude of things, so I think my main conclusion so far is that Win10 is going in a fairly good direction. However it would benefit from keeping high functionality and compatibility, certain programs seem to be missing like media center which people still use; even it it was a download and install from MS's website, at least give people the option to still use it even if it doesn't come pre-installed. And I dislike the dumbing down of the settings menu, reducing it's functionality may suit some people, but not me. Again, if there was the option to have a simple and an expert menu it would be a happy medium IMO.

And as I said above - a higher level of personalisation of the menus, desktop, metro interface etc. would be nice!

These are my opinions and my opinions alone. If you disagree with them it doesn't make my opinions wrong!

graham mayor say:

Office 2010 Home and Student does not include Outlook, but it includes a trial of Outlook 2010 which may have been installed. Uninstall the trial if present from (say) Word > File > Help > Trial Product >Microsoft Office Professional 2010 > Licensing Options > Remove Trial.

You should then be able to install Outlook 2007 (either as a stand alone or from the Office 2007 media). You will of course need the Outlook.Office 2007 product key.

Copy the Outlook PST data file from the old PC to the new one. Open it from Outlook 2007 (File > Open > Outlook Data File) then make that the default data file (File > Data File Management > Select the file > Set as Default). You can then close the original empty data file from the mail task pane - Right click the file > Close Personal Folders
