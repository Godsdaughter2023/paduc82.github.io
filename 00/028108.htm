robert aldwinckle on forums say:

When I reboot the computer Internet Explorer will open automatically two times, one right after the other.

Sounds like you have some automation in place that you don't want.  It might be something as simple as something extra in your Startup folder.   Use Task Manager (Ctrl-Shift-Esc) for its Startup tab to check.  Otherwise a good tool to help understand and control where automated things are coming from is  AutoRuns by SysInternals (available from TechNet).

Your "when I close the browser, after using it, and I close all tabs it automatically restarts it."  sounds related to your first "two times" observation but I would try using AutoRuns to try to see what is really going on.  A related tool is ProcMon  which would trace what is going on.

As for your problem about connecting to the Internet during Startup, that sounds like a timing problem caused by your Home page.  To avoid that you could set your Home page to something local only such as About:Blank.  Another possibility would be to have a Home page for a site which could be always completely cached and then use a cache checking option which would allow it to be used (e.g. Never).  However, then that option would apply to all of your other browsing and you would have to start using a manual Refresh command (e.g. F5 or Ctrl-F5) more often.

graham mayor say:

Word 2007 docx format is a compressed file format - a zip file. Inevitably a zip file will be smaller than an uncompressed file format like PDF.
The size of a given PDF file will be dependent on what it contains as opposed to what it expects the PC to provide e.g if you set the ISO 19005-1 option it will embed the fonts which results in a larger file size - but ensures that it can be read by the recipient as intended.
Ultimately the Office PDF creation tool is a compromise that will not give you the control over content that Acrobat provides.
How large your PDF document will be from 100 pages will be determined by what those pages contain. A sample document containing 192 pages had a docx file size of 375 kb and a PDF size of 1.25 mb. You will have to complete the document to establish the final size.

Hi cantumindy48765,

 

For your information, Internet Explorer do not track attempted downloads.
