Hi John, glad the formula worked. Below is an explanation how the formula works:

We need to determine ALL consecutive negatives in the first instance, and then Average these. The formula is valid for range A2:A100.

Step 1: First we determine the consecutive negatives LEAVING OUT the first negative from these:

(A$2:A$99<0)*(A$3:A$100<0) - condition to determine where there is a negative in a cell and negative in the previous cell ALSO

IF((A$2:A$99<0)*(A$3:A$100<0),A$3:A$100) - returns negative values where a cell & its previous cell both are negative. Note, this leaves out the first negative out of consecutive negatives because its previous cell will be a positive.

Step 2: Then we determine the FIRST negative among the consecutive negatives:

(A$2:A$99<0)*(A$3:A$100<0) - determines where there is a negative in a cell and negative in the previous cell ALSO

IF((A$2:A$99<0)*(A$3:A$100<0),IF(OFFSET(A$2:A$99,-1,0)>=0,A$2:A$99)) - where there is a negative in a cell and negative in the previous cell ALSO, we check the cell PRECEDING these and if it is positive, we take that into account - this determines the FIRST negative

FINAL Formula:

In Step 1 we determined the consecutive negatives LEAVING OUT the first negative; In Step 2 we determined the FIRST negative of the consecutive negatives.

In the final formula we AVERAGE both Step 1 & Step2 ie. ALL CONSECUTIVE NEGATIVES

Final Formula - array formula:

=AVERAGE(IF((A$2:A$99<0)*(A$3:A$100<0),A$3:A$100),IF((A$2:A$99<0)*(A$3:A$100<0),IF(OFFSET(A$2:A$99,-1,0)>=0,A$2:A$99)))

suzanne s. barnhill say:

The \h switch, which is present by default in all TOCs in Word 2000 and above, is the result of checking the box for "Use hyperlinks instead of page numbers" under the "Web preview" in the Table of Contents dialog. The result is that the entries themselves, and not just the page numbers, are hyperlinked to the  headings. It makes TOC styles more difficult to modify because, if you right-click in a TOC entry and choose Styles..., you get options for the Hyperlink style rather than the TOC style for the given level. For this reason, I usually clear that check box when I insert TOCs. You can also remove it manually: press Alt+F9 to display the TOC field code, delete the \h switch, Alt+F9 to toggle back, and then F9 to update.

 

That said, since TOC styles are, by default, set to update automatically, many modifications to the TOC styles can be made just by changing the format of a single entry at a given level. In my experience, this works well for paragraph formatting. Changes in font formatting often have to be made through the Modify Style dialog. If you find that making the style changes through the Table of Contents dialog doesn't "take," you can get to the Modify Style dialog after inserting the TOC by displaying All Styles in the Styles pane (in alphabetical order), right-clicking the relevant TOC style, and choosing Modify.

 

One "gotcha" about modifying TOC styles: it works only if you have "From template" selected for "Formats." If you have selected one of the other styles (Classic, Distinctive, Fancy, etc.), then any changes you have made in the styles will be wiped out every time you update the TOC.

  2. list disk
