andreas killer say:
I don't understand your problem, but here are some suggestions.

a) Never use SELECT and SELECTION in your macros, it's slow and error prone. Always access the objects directly. See Sub TestA as example.
b) Almost any object contains collections to get the sub objects, e.g. any sheet contains the ChartObjects collection that contains all charts and so on. See Sub TextB as example.

Andreas.

Option Explicit

Sub TestA()
  Dim C As Chart
  Dim mySeries As Series
  Set C = ActiveSheet.ChartObjects(1).Chart
  Set mySeries = C.SeriesCollection("f01_p033_t293_d0570_p005_t293_30mm")
  With mySeries
    .Format.Line.ForeColor.RGB = RGB(0, 0, 255)
    .MarkerStyle = xlCircle
    .MarkerBackgroundColor = RGB(200, 200, 200) ' marker fill colour
    .Format.Line.Visible = msoTrue
    .Format.Line.ForeColor.RGB = RGB(0, 0, 255) ' series line colour
    .MarkerForegroundColor = RGB(192, 0, 0) ' marker border line colour
    .MarkerSize = 6

    'Note: The interior and the border is the same in line charts!
    'Means the second lines overwrites what the first line has done.
    .Format.Line.DashStyle = xlContinuous ' marker border line style
    .Border.LineStyle = msoLineDash ' series line style
  End With
End Sub

Sub TestB()
  Dim Data
  Dim C As Chart
  Dim S As Series
  Dim P As Point
  Dim Scnt As Integer, Pcnt As Integer
 
  'Add a sheet and create some data
  Sheets.Add
  Data = WorksheetFunction.Transpose(Array( _
    Array("A", 1, 2, 3, 4, 5, 6), Array("B", 2, 3, 4, 5, 6, 7), _
    Array("C", 3, 4, 5, 6, 7, 8), Array("D", 4, 5, 6, 7, 8, 9)))
  Range("A1").Resize(UBound(Data), UBound(Data, 2)) = Data
 
  'Add a chart and assign our data
  Set C = ActiveSheet.Shapes.AddChart(xlLineMarkers).Chart
  C.SetSourceData Range("A1").CurrentRegion
 
  'Visit each series
  For Each S In C.SeriesCollection
    Scnt = Scnt + 1
    'Color the lines from blue to red
    S.Format.Line.ForeColor.RGB = ColorMorph(vbBlue, vbRed, C.SeriesCollection.Count, Scnt)
    Pcnt = 0
    'Visit each marker
    For Each P In S.Points
      Pcnt = Pcnt + 1
      'Color the interior from yellow to green
      P.Format.Fill.ForeColor.RGB = ColorMorph(vbYellow, vbGreen, S.Points.Count, Pcnt)
    Next
  Next
End Sub

Private Function ColorMorph(ByVal StartColor As Long, ByVal EndColor As Long, _
    ByVal MaxSteps As Long, ByVal Step As Long) As Long
  'Calculate color from Start- to EndColor for each Step from 1 to MaxSteps
  Dim NewRGB(1 To 3) As Integer
  Dim Start As Long, Ende As Long, i As Integer
  If MaxSteps < 2 Then
    ColorMorph = EndColor
    Exit Function
  End If
  For i = 1 To 3
    'Calculate RGB
    Start = (StartColor \ 256 ^ (i - 1)) And 255
    Ende = (EndColor \ 256 ^ (i - 1)) And 255
    'Calculate percentaged step
    NewRGB(i) = Ende + (Start - Ende) / (MaxSteps - 1) * (MaxSteps - Step)
  Next
  ColorMorph = RGB(NewRGB(1), NewRGB(2), NewRGB(3))
End Function

john korchok say:

When you link to a file, the file remains separate from PowerPoint. The link tells PowerPoint where to find the file to play it. By contrast, when you embed the file, it becomes part of the presentation, so the program always knows where to find it. Of course, video files embedded in a presentation make it larger by the size of the video.

Can you be specific about what didn't work when you tried to save as .pptx? A sound recording disappeared? PowerPoint crashed? You got an error message?

If you can save the file as .pptx, you can extract the audio by adding .zip to the end of the file name, expanding it, then looking inside the ppt/media folder. Otherwise, if you have access to a Windows computer with PowerPoint 2003 or 2007, you can save the presentation as HTML, then find the audio in Filename_files, where Filename is the name of your file saved as HTML.

1. Sign in to your Microsoft account.

2. Click the gear icon and select More mail settings.

3. Under Managing Your Account select Connect devices and apps with POP.

4. Tick on Enable and select Don't let devices or apps delete messages from Outlook.

5. Click Save.
