    I've tried locking/unlocking cells and also tried adding a piece of VBA code. Both work to skip the cells I want, the problem I'm having is that I can't then get the cursor to return to column A on data entry. With the VBA code, the cursor will only go back to the next cell after the last skipped section (F), not all the way back to A.

    Is there any way I can make the cursor always return to column A when editing A-G but still carry straight down for my checklist, with it still skipping over the columns I want protected?

    bernie deitrick say:

    Try it like this - if you select a cell in columns A through G, then you will be able to tab the way you want. If you want to enter values in H, first select a cell in a column to the right of H, and then select the cell in H, and you will be able to 'enter' you way down H. (H can be selected when tabbing from G, so it is kind of in both: thus the need to select to the right.)

    Note that I use the public variable boolTab, which needs to be declared at the top of your module, not just above the specific event.

    Dim boolTab As Boolean

    Private Sub Worksheet_SelectionChange(ByVal Target As Range)
        Application.EnableEvents = False
        If Selection.Column < 8 Then boolTab = True
        If Selection.Column > 8 Then boolTab = False
        If boolTab Then
            If Target.Column = 2 Or Target.Column = 5 Then Target.Offset(0, 1).Activate
            If Target.Column = 8 Then Target.Offset(1, -7).Activate
        End If
        Application.EnableEvents = True
    End Sub

    In case you are not familiar with using events:

    1) Copy this code.
    2) Right-Click the sheet tab of interest.
    3) Select "View Code"
    4) Paste the code into the window that appears.
    5) Save the file as a macro-enabled .xlsm file.

    bernie deitrick say:

    Use a macro with this structure

    Sub RenameAndSortSheets()
        'Deal with the sheets in the order you want them
        With Sheets("Avacado")
            .Name = .Name & "_1"
            .Move Before:=Sheets(1)
        End With
        With Sheets("Banana")
            .Name = .Name & "_1"
            .Move Before:=Sheets(2)
        End With
        With Sheets("Apple")
            .Name = .Name & "_1"
            .Move Before:=Sheets(3)
        End With

        ' Other sheets here
        ' increment the number in this line

        '  .Move Before:=Sheets(3)

        'Last one does not need to be moved
        With Sheets("Orange")
            .Name = .Name & "_1"
        End With

        'Hide that one sheet

        Sheets("Dragon").Visible = False

    End Sub
