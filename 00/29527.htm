thx1200 say:

To restart Windows without installing downloaded updates, you can use the command line tool shutdown.exe, which is available on all versions of Windows going back to XP.

 

Type this...

shutdown -r -t 0 -f

 

That means reboot (-r) now (-t 0 means no countdown time), and force shutdown (-f means don't do any "such and such application is open" prompts).

thx1200 say: No switch is needed.  The shutdown.exe will not install updates.  If you already installed updates, it will not prevent post-update actions from running during reboot, but it will not install updates that are in queue.

This will work in windows 8 too, but I haven't tested in 8.1.  I would assume it would work in 8.1.
