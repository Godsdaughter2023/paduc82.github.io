    jamesyan_0708 say:

    Hi John,

    1. In Office 365, only Password admins, User management admins and Global admins can receive the password reset request notification emails.

    If we have all the three types of admins above in your organization, then only the Password admins will receive the notification emails.

    If we only have User management admins and Global admins, then only the User management admins will receive the notification emails.

    If we only have Global admins, then the Global admins will receive the notification emails.

    So as a conclusion, the priority of receiving the password reset request notification emails is:
    Password admins > User management admins > Global admins .
    That's the reason why you as a global admin didn't receive those notification emails.

    2. If we already have Password admins, and we want to make sure other customized administrators can receive the notification emails, we need to ensure they are also assigned the Password admin role.

    Best Regards,
    James
