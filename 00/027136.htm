I guess I am just not in 2003 compatible mode today. The ISODD() and ISEVEN() will have to be replaced with an appropriate MOD() function as well to make the formula fully 2003 compatible.

Formula for D11 is,

=IF(AND(LEN(SUBSTITUTE(C11," ",""))=6,IF(ISERROR(SUMPRODUCT((ISERROR(--MID(SUBSTITUTE(C11," ",""),ROW($1:$5),1))*(MOD(ROW($1:$5),2)=1))+(ISNUMBER(--MID(SUBSTITUTE(C11," ",""),ROW($2:$6),1))*(MOD(ROW($2:$6),2)=0)))),FALSE,SUMPRODUCT((ISERROR(--MID(SUBSTITUTE(C11," ",""),ROW($1:$5),1))*(MOD(ROW($1:$5),2)=1))+(ISNUMBER(--MID(SUBSTITUTE(C11," ",""),ROW($2:$6),1))*(MOD(ROW($2:$6),2)=0)))=6)),"Valid","Not Valid")

Thanks. I assume that you're using HTML format. The HTMLBody property of the MailItem object contains the HTML source code for the body of your e-mail message. If you're familiar with HTML coding, you can manipulate the HTMLBody string.
