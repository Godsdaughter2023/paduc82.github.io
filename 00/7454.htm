davidf2 say:

Did you look in the recycle bin? It will be there unless it has been emptied.

Even if the recycle bin has been emptied you can sometimes get a file back using the free version of Recuva. https://www.piriform.com/recuva

If the file was not saved you can sometimes find a copy of it in the temp folder.

Start > Run > type in %temp% > find a recent file with the Works prefix, chnage the .tmp extension to .wps (assuming a Works word processing file), and try opening the file.

Use this as an educational opportunity and start backing up your files. Copy to a thumb drive, external hard drive or even someplace in the cloud such as OneDrive, Dropbox or perhaps Googledrive and next time something like this happens you won't have a crisis. Everyone looses one important file because they did not back it up...just don't loose two.

seasonh say:

Excel formula or pivot table: counting frequency of ratings

My Excel table looks like this, but much longer:

ID Rating 1 Rating 2
11111 E1 A3
11111 E1 A3
11111 E1 A5
22222 E2 A5
33333 E2 A5

I need to answer these two questions:

1. How many E1 ratings does each ID have?

For example: 11111 has 3, 22222 has 0, 33333 has 0.

2. How many E1 + A3 ratings does each ID have?

For example, 11111 has 2, 22222 has 0, 33333 has 0.

My spreadsheet is too long to count by hand. Should I use a formula or a pivot table to answer these questions?

Thank you for your help - I really appreciate it!

amit tandon say:

A solution using Excel formulas. Refer below image:

Enter below formula in cell E2 & copy down - this returns all unique Ids in column A:

=IFERROR(VALUE(INDEX($A$2:$A$100,MATCH(0,INDEX(COUNTIF(E$1:E1,$A$2:$A$100),,),0))&""),"")

Enter below formula in cell F2 & copy down - this returns for each Id, the E1 Rating1:

=SUMPRODUCT(($A$2:$A$100=$E2)*($B$2:$B$100=$F$1))

Enter below formula in cell G2 & copy down - this returns for each Id, the E1 Rating1 & the A3 Rating2:

=SUMPRODUCT(($A$2:$A$100=$E2)*($B$2:$B$100=$F$1)*($C$2:$C$100=$G$1))
