Keeping your device and the game version up-to-date may help resolve issues with games on Windows 10. Refer to the following links for the steps:

  • How to get software and driver updates for Windows 10

  1. Click Start and click Settings.

  5. Click Reset.
  6. Restart your device so that the changes can take effect. Launch the game and check if you'll encounter the same issue.

Uninstalling and reinstalling the game will replace corrupted game files. See How to uninstall and reinstall a game on Windows 10 for the steps.
