boblarson say:

From your description you don't really have a subform.  You just have ANOTHER form.  So, you can use code like this for your close button:

 

If Me.Dirty = True Then Me.Dirty = False

If CurrentProject.AllForms("YourMainFormNameHere").IsLoaded Then

   Forms!YourMainFormNameHere.Requery

End If

DoCmd.Close acForm, Me.Name, acSaveNo

 

 

Just an additional note - you can use that last line exactly as shown, with no modifications, and it will work fine.  It is the Me.Name part which passes the current form name which the code is on so you don't have to hard code it.  Also, the acSaveNo part has nothing to do with saving records (that part is handled in the If Me.Dirty part, but it has to do with saving DESIGN CHANGES and you don't want your users saving design changes for the form (which can also include filters).
