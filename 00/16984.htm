In Windows 7, an administrator is more akin to what used to be referred to as a power user rather than an actual administrator of the computer. The rights you require are now referred to as elevated privileges and are granted by right clicking your menu item or desktop shortcut and selecting Run as administrator then acknowledging the UAC warning and providing credentials if necessary. If you open the Insightly setup program in this manner, you should be able to proceed through the installation process.

Some background: Due to everyone under the sun running their computer logged in as either the or an administrator, many drive-by viruses and/or malware found their way on people's systems and to make it worse, they were installed with administrative rights as these were inherited from the currently logged on user. Microsoft created UAC (User Account Control) to counteract this and gave the Administrator of a computer no more rights than what was previously assigned to a Power User. If any action of real consequence was to be performed, the UAC would stop the process and ask for confirmation to proceed. If the current user was not a member of the Administrators user group, it would also ask for the username and password credentials of an administrator.

a. user say:

Hi,

 

Please be informed that if you will rename a @hotmail.co.uk to @outlook.com, the old email address will become an alias. An alias is a secondary email address added to an already existing Microsoft account address. To put it another way, you have a mailbox with an existing email address created when you signed up for your Microsoft account and sometime later you added another email address going to the same mailbox.  It's true that you can’t sign in with it, but you can send and receive email with it, that's what an alias is.
