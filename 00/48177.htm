Hi,

If your Windows-based computer is running low on storage, it will affect the performance of your device. To troubleshoot your issue, we recommend deleting files with storage sense.  If you have storage sense turned on in Storage settings, Windows can automatically free up space by getting rid of files you don't need, like temporary files and stuff in your recycle bin. Or, you can delete them manually.

To do this:

  1. Click Start, then select Settings.
  2. Select System, then Storage.

Do any of the following:

  • To have Windows delete unnecessary files automatically, turn on Storage sense.
  • To delete unnecessary files manually, select Change how we free up space, then Clean now.
  • To specify which files you want storage sense to delete automatically, select Change how we free up space, then select or deselect the check boxes for options under Temporary Files and Free up space now.

If you look at the trailing part of that file name (settingcontent-ms) this identifies the file as a Virtual File. These files do not behave like normal files.

Here is how you can pin that component to the Start Screen.

1. Open Control Panel / Hardware and Sound. Right Click Sound in that screen and click Create Shortcut. This will place the shortcut on the Desktop.

2. Open File Explorer and navigate to C:\ProgramData\Microsoft\Windows\Start Menu\Programs. Drag the File Explorer window to one side of the screen so you can see the Sound Shortcut.

3. Use the Right Mouse button and Drag/Drop the shortcut into the Programs folder and select the Move option.

4. Go to the Start Screen / All Apps view. Locate and Right Click the Sound shortcut. Select the Pin to Start option on the bottom toolbar.
