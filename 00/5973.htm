anthony.b say:

I don't think there is a way to do what you are asking for (modify the default template for new workbooks) in Excel 2013.  I would recommend setting up a workbook with all the formatting that you want, and then save it for future use. There are two ways to do this:


[1] When saving your Excel file, change the "Save as type" to "Excel Template". This will have extension .xltx. When opening the .xltx it will not overwrite your template, but ask you to give your file a new name.


[2] Just save your Excel file as a regular, standard "Excel Workbook" (will have the extension .xlsx). Then use your file you saved as a .xlsx as a starting point, to get the formatting that you want. Save the file with a different name to protect your original.

By the way, to set the color for hyperlinks, that is at PAGE LAYOUT -> Colors -> Customize Colors. This will require you to save it as a new Theme Color, if you want to quickly re-use your re-designed hyperlink colors. However, when you open a new blank Excel workbook, you will need to click on PAGE LAYOUT -> Colors to select your new color theme.
