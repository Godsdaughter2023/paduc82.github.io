graham mayor say:

The common causes of misaligned labels are as follows

1. You have added a header/footer to the normal template (rename the normal template)
2. The page size set in the printer properties doesn't match the page size of the label sheet.
3. You have a scaling option set in the zoom section of the print dialog
4. Your printer has poor paper handling. Some printers will slip when faced with label stock and thus are incapable of feeding the label correctly. Cleaning the rollers in the print path may help.

It depends on the type of license you got. Part of the problem is that I have not yet seen a "official" list of 2016 license types, so this is based on earlier versions.

If have an OEM license, Office installed and activated at the factory, or an Product Key Card (PKC) license. If you have one of these licenses, you are out of luck. These licenses are cheaper because one of the license limits is that it is permanently tied to the first computer it is installed on.

They can tell you what type of license you have. If you have the right one the will allow the activation to complete.

When the Windows 8 version of Defender was being developed, the equivalent version of Microsoft Security Essentails v4 was also operating in beta and this question was discussed in some depth.  In general the answer is that the detected item(s) are placed in quarantine, though in many cases the user is also prompted to make the determination whether to remove, quarantine or allow the item.

 

The only true documentation I can find for this outside of Defender is on the Windows 8, virus and malware web page at the following URL, which is where the short excerpt below is found.

http://windows.microsoft.com/en-us/windows-8/windows-defender#1TC=t1

 

Windows Defender found something! What do I do?

In addition to the color codes for your PC’s overall security status, Windows Defender applies an alert level to any suspected malware it detects. You can decide whether to remove an item entirely, research it further, or let it run because you recognize it.

Review quarantined items

If Windows Defender can’t determine whether a detected item is malware or something you intentionally installed, it prevents the software from running but quarantines it on your PC so that you can make the call later. Here's how:

  1. From the Search charm, search for defender and then open Windows Defender.

  2. On the History tab, select Quarantined items, and then tap or click View details. You might be asked for an admin password or to confirm your choice.

  3. Do one of the following:

    • Tap or click Remove all to get rid of all the quarantined software.

    • Select individual quarantined items, and then tap or click Remove or Restore.

Just a note to Jim's reply...

Most work involving currency does not use the ¢ symbol. Pennies simply are expressed as decimal fractions of a Dollar. A Custom Format can be constructed to include the ¢ but that does nothing to convert the value - IOW, what appeared to be 25¢ would still be treated as a whole number of 25 (dollars), not as "one fourth of a dollar".
