zedrick del say:

Hi,

This issue may occur when the user disables the use of caches and databases through Internet Options or Group Policy. To better assist you with your concern, we recommend that you change the configuration of your Desktop Search through Group Policy to enable the search option on Cortana. To do this, follow the steps below: 

1. Press Windows + R key to open Run box.

2. Type gpedit.msc and press Enter.

3. In the Group Policy Editor, navigate to the following location: 

Computer Configuration -> Administrative Templates -> Windows Components -> Search 

4. In the pane on the right, double-click Don't search the web or display web results in Search.

5. Select Enabled.

6. Once done, click Apply and then click OK. 

If the issue persist, we recommend that you run the Search and Indexing troubleshooter to scan and correct issues related to your query. To run this troubleshooter, follow the steps provided by Syed Abdul Jabbar on this thread.
