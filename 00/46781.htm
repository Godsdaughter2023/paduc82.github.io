You can reinstall Windows 10 at any time, you will not need a product key, the activation will be automatic and it will not cost you anything.

Since you previously had Windows 10 installed and activated on that PC during the installation process, skip the steps that require a product key and select the "I'm reinstalling Windows 10 on this PC" option, and activation will not be a problem, your PC will have a digital right stored in the Microsoft activation servers

Be sure to install the same edition of Windows 10 (Home, Pro ... etc.) that you previously installed to avoid activation problems

Click on this link:

https: //www.microsoft.com/en-us/software-downlo ...

To download the Media Creation Tool (click Download tool now), with which you can download the latest version of Windows 10 ISO (Select Create installation media for another PC), you can create a bootable USB flash drive (min 8 GB) with that tool

Then, start your PC from the installation medium you just created (change the boot order in your BIOS) to start installing Windows 10

If you have problems booting from a boot disk, you may have UEFI BIOS:
Insert the installation medium of startup, then enter your BIOS and make the following changes:

1. Disable safe boot

2. Enable inherited boot

3. If available, enable CSM

4. If necessary, enable USB booting

5. Move the device with the boot disk to the top of the boot order

6. Save the BIOS changes, restart your system and it should start from the installation media.
