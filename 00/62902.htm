Hi Michael,

  1. The old system restore points are not deleted as a new restore point is created. The old system restore points are deleted when the disk space allotted to restore points is reached. Generally 3-5% of disk space is allotted to each disk for restore points. If you have more disk space, you can allow more disk space as you desire for restore points. It is not mandatory to keep the disk space slider to maximum.
  2. For which purpose are you planning to use the EHD? If you are planning to use the external HD as a storage device, then yes, you can use the models you have mentioned earlier. However before purchasing, I would recommend checking the EHD features in manufacturerâ€™s website to make sure that it is compatible with Windows 8.1.
  3. A recovery partition will contain only the Windows system files. So if you use a recovery partition, you will be able to only set the system back to its factory settings. The system image is an exact copy of the drives when the system image was created. It includes all personal files, settings saved in the drives. So you can create a system image or recovery partition based on your needs.
  4. Yes, you are right. You can create a system image by clicking on the â€˜System image backupâ€™ link at the left bottom of File history settings.
  5. File history constantly creates a backup of included location when any changes are made. So it is advisable to use an EHD rather than a pendrive, so that there wonâ€™t be any space constraints. It is recommended to leave the EHD connected all times so that a backup would be created when you make any changes.
  6. Like I said earlier, a system image is an exact copy of the drive including all files. So any new files or changes made after the last system image backup would be not available in the last image backup. So, if you are creating a system image backup, it is recommended to make a new system image backup at regular intervals to keep the backup data up to date.
  7. To restore from a system image backup, you need to boot from the system image disc to enter the Windows recovery environment. You can refer to these steps:

 

  1. When your computer starts, wait for the manufacturer logo to check the option for boot menu, it will usually be F12.

  4. After you boot your computer using Windows DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.
  5. Select the correct time and Keyboard type.
  6. Click Repair your computer in the lower left corner
  7. Click on Troubleshooting, advanced option and then click System image recovery.

1) Open administrative command prompt and type or copy and paste:

2) bcdedit /enum all

3) wmic memorychip get banklabel, manufacturer, partnumber, speed, MemoryType, devicelocator

4) When these have completed > right click on the top bar or title bar of the administrative command prompt box > left click on edit then select all > right click on the top bar again > left click on edit then copy > paste into the thread

5) If the RAM is DDR3 then Run Memtest 86+ version 5.01 for 8 or more passes.

https://www.memtest.org/

6) If the RAM is DDR4 then Run Memtest86 version 8.0 for 8 or more passes

https://www.memtest86.com/

Testing is not done by time but is done by passes.

The more the passes the better the testing environment.

A false negative is a test pass when there is malfunctioning RAM.

The incidence of false negatives can be high when fewer than 8 passes are made.

Earlier in the thread version 4.3.7 was used instead of version 5.01

And earlier in the thread only 5 passes were made.

It sometimes takes passes into the 20's to find malfunctioning RAM.

Testing must be continuous.

A substantial portion of the testing can be performed overnight

Testing time is approximately 1 -2 hours / GB RAM

The computer had 16 GB RAM

If the motherboard has multichannel capability testing both RAM modules simultaneously can test this feature of the tested DIMMs.

If the RAM modules are tested one at a time the RAM modules can be tested but there will not be any multichannel testing

Just one error is a test fail and testing can be aborted.

Then test one RAM module at a time in the same DIMM for 8 or more passes to differentiate good from Malfunctioning RAM

7) For all RAM testing please make sure pictures are taken with a camera or smartphone camera and images are posted into the thread.

https://answers.microsoft.com/en-us/windows/wiki/windows_10-update/memory-problems/21c3f63f-f570-4522-b2ef-ecc7b7ff6461
https://www.tenforums.com/tutorials/14201-memtest86-test-ram.html
