Excel uses ASCII code 10 alone whereas Access uses both 10 & 13.

You can do a replace function like this --

  Replace(Replace([TextField], Chr(10), Chr(10)&Chr(13)), Chr(10)&Chr(10), Chr(10))

The second replace is to remove any double Chr(10)s.
