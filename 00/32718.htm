    ruwim.b say:

    If the main.db database file is corrupted, then it is practically impossible to fix it. Perhaps there might exist some highly specialised tools, but I'm not aware about any.

    When Skype for Windows Desktop detects corruption in main.db file, then this file is abounded (markt as main.corrupt) and a new fresh main.db file is created.

    Microsoft Skype for Windows 10 (Skype UWP) application is using it's own main.db file with no connection to the corresponding main.db file from classic Skype for Windows Desktop. The location of the Skype UWP main.db file is:

    %localappdata%\Packages\Microsoft.SkypeApp_kzf8qxf38zg5c\LocalState\

    If you don't have any backup of the main.db file, then, I'm afraid, your old conversation data are lost forever.

    When the main.db file gets corrupted, other database and settings files might be corrupted too, so it is best to reset all Skype settings.

    Quit Skype or use Windows Task Manager to kill any Skype.exe process. Go to Windows Start and in the Search/Run box type %appdata% and then press Enter or click the OK button. The Windows File Explorer will pop up. There locate a folder named “Skype”. Rename this folder to something different, e.g. Skype_old.


    Restart Skype.
