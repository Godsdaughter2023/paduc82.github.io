    Terkadang kita menemukan sebuah aplikasi yang menarik dan mungkin berguna bagi teman atau rekan kita. Lain waktu, kita menginstal sebuah game dan ingin mengajak teman kita untuk memainkan game yang sama. Untuk itu kita perlu menginformasikan alamat dari aplikasi/game tersebut kepada orang lain.

     

    Untuk melakukan itu yang harus kita lakukan adalah :

    1. Buka daftar aplikasi/ application list dengan cara sapu ke kiri dan home screen.

    2. Cari aplikasi/game tersbut dari daftar.

    3. Tekan lama nama aplikasi/game tersebut sampai muncul floating menu

    4. Pilih bagikan/share.

    5. Akan muncul beberapa pilihan dengan apa kita akan berbagi alamat aplikas tersebut, bisa lewat sms/messaging, facebook, twitter ataupun email. Pilih salah satu.

    Catatan :

    • Ada kalanya kita ingin berbagi alamat aplikasi/game melalui media yang tidak ada di pilihan pada langkah nomor 5, atau kita ingin berbagi  melalui facebok dengan pesan yang lebih personal atau melalui komenta paste/tempel di editor media yang kita inginkan, misalkan di jendela chat whatsapp, atau di kolom komentar facebook.

    • Cara yang lain adalah dengan menggunakan aplikasi Clipboarder (http://www.windowsphone.com/s?appid=83234c26-84eb-4002-9c31-0efb7e72e8aa) yang tersdia gratis di bursa/store. Bila aplikasi  tersebut sudah terinstall di lumia kita, maka pada langkah nomor 5, akan tersedia 1 pilihan lagi yaitu share ke clipboarder. Bila kita memilihnya, maka alamat aplikasi/game yang dibagikan akan tersimpan di clipboard dan siap di paste/tempelkan di editor aplikasi apa saja termasuk whatsapp dan facebook.

    Kontrol Persentase Office Kamu Menggunakan Lumia

     

    Mengkontrol Office Saat Persentasi,

    Microsoft menyediakan Tool yaitu Microsoft Office Remote, yg berfungsi sebagai pengendali office kamu disaat kamu melakukan persentasi Office baik itu Word - Excel - maupun Power Point di PC kamu dengan Windows Phone.


    Requirement :

     

    - Windows 7 / 8 / 8.1 (32-bit ataupun 64-bit) , Windows 8.1 recomended

    - Komputer/laptop Anda mendukung konektifitas Bluetooth (tidak masalah mau bluetooth internal ataupun external)

    - Microsoft Office 2013 (PC)

    - Smartphone Windows Phone 8 / 8.1 (salah satunya seri Lumia), Windows Phone 8.1 recomended

     

    Pada PC / Desktop

     

    1. Download Setup OfficeRemote.exe | 1,2MB

    2. Jalankan Setupnya, Ikuti Wizardnya dan Tunggu Process Installasi 

    3. Setelah Selesai, Buka salah satu aplikasi Office Word - Office Excel - PowerPoint, Pastikan ada tab baru add-on (Office Remote)

    4. Di dalam Tab tersebut, Klik menu Drop Down dan akan muncul 3 Pilihan yaitu

      • Turn On 

      • Install On My Windows Phone

      • Learn More

    5. Pilih Trun On . dan Nyalakan Bluetooth di PC.

     

    Pada Smartphone Windows Phone

     

    1. Install Office Remote Pada Windows Phone kamu, kamu bisa medownload nya di Bursa / Store melaui link ini

    2. Nyalakan Bluetooth dan Pairkan dengan PC Kamu

    3. Buka App Office Remotenya

    4. Muncul seperti ini, pilih Continue  

    5. Selanjutnya pilih koneksi yg sudah di pairkan, Pilih Nama PC kamu

    6. Ter Turn On dan kamu sedang membuka sebuah dokumen persentase di Office PC

    7. Ketika semua berjalan dengan lancar, akan keluar Document yg sedang kamu jalankan tadi di PC akan tampil di Smartphone Windows seperti ini

    8. Cukup buka saja dokumen tersebut dan saatnya kamu mengontrol persentase kamu!

    Remote Komputer/Desktop melalui perangkat Lumia Windows Phone

     

    Apa ini?

    Ini adalah bagai mana cara mengontrol komputer kamu hanya dengan sebuah Lumia Windows Phone! Jadi, kamu bisa melakukan apa yang bisa di lakukan di Laptop/Desktop seperti mengetik sebuah dokumen, mengirim sebuah file, memprint suatu foto/dokumen dan lain-lain hanya dengan seuah Smartphone Lumia Windows Phone!

     

    Bagai mana caranya?

    1. Dibutuhkan sebuah komputer ber-OS Windows7/Windows 8/8.1 atau yang terbaru dan perangkat ponsel dengan sistem operas Windows Phone yang sudah update ke Windows Phone 8.1 (smartphone Lumia misalnya).

    Kenapa Rekomendasi Windows 8?

    Karena nantinya lebih memudahkan urusan loginnya pada Windows 8. Tak masalah jika sistem operasi yang kamu gunakan adalah Windows 7, tapi yang digunakan minimum adalah harus Windows 7 Professional dan atau Ultimate, edisi Home Premium dan edisi kebawah nya tidak bisa.

     

    2. Komputer yang akan diremote dan perangkat yang akan meremote  haruslah terhubung dalam satu network yang sama, misalnya keduanya terhubung dalam suatu Wifi/hotspot yang sama. (Baca artikel berikut jika ingin: Cara membuat jaringan WiFi/Hotspot di Windows 8).

     

    Settingan pada Windows Desktop

     

    AKTIFKAN AKSES REMOTE DESKTOP DI WINDOWS

    3. Untuk mengaktifkan akses remote desktop di Windows caranya :

    • klik kanan pada Computer, pilih properties,

     

    • Pilih Remote Settings

    • Pada Tab Remote, pilih Allow remote connection to this computer, klik Apply, klik Ok, dan pada bagian Remote Desktop set settingnya seperti pada gambar.
    •  
      klik Selet User jika anda ingin menambahkan user tertentu, kalau accountnya sebagai member administrator, tidak perlu dimasukkan, karena account administrator sudah otomatis punya hak untuk meremote ketika option remote desktop ini diaktifkan.

    CATATAN:

    Sangat disarankan telah men-switch account ke account Microsoft agar proses masuk ke remote destop lebih mudah

    • Klik Apply, klik OK.

    Settingan remote desktop di komputer telah selesai!

     

    Settingan pada Windows Phone

     

    LANJUT SETTING REMOTE DESKTOP DI PERANGKAT WINDOWS PHONE 8/8.1

    4. Sangat direkomendasikan bahwa OS yang digunakan pada smartphone kamu telah menjalankan Windows Phone 8.1 , saya merekomendasikan Windows Phone 8.1 dari pada Windows Phone 8.

     

    5. Download aplikasi Microsoft Remote Destop Preview di store atau bisa klik tautan berikut ini.

    6. Buka aplikasinya, klik tanda Tambah untuk menambahkan client

     

    7. Pilih Add new Desktop

    8. Masukkan nama domain komputer atau ini juga bisa di isi dengan IP Address. Nama domain komputer bisa kamu cek di System Properties, jika kamu tidak mempunyai domain tersendiri, nama domain adalah sama dengan nama Fullname PC, sebagai contoh saya disini adalah ASUS-PC.

     

    9. pada kolom credential , isilah sesuai dengan data login komputer kamu, yakni username dan passwordnya. Komputer untuk remote desktop WAJIB memiliki password.

     

    10. Untuk memulai koneksi Remote Desktop, klik pada nama koneksi yang tadi kita buat.
    (sebelumnya pastikan baik komputer ataupun perangkat Windows Phone telah terkoneksi di wifi / network yang sama)

    Hasilnya, ini adalah tampilan meremote Windows 8.1 pada smartphone Windows Phone 8.1.
