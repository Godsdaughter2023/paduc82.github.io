Windows 98 - Microsoft Community

goldboyalex2005 say:

Windows 98

Hi. I have a desktop running windows 98 as a hobby and i am wondering if There is an anti-virus program made by microsoft for windows 98, and if not, would windows security essentials or windows defender work?

rob koch say:

No, No and No.

Windows 98 has been unsupported by Microsoft itself for over a decade now, since as the following Microsoft Blog article from June 2006 states, a critical update for newer operating system versions was not going to be made available for Windows 98/ME versions due to architectural limitations.

Windows 98, 98SE and ME Information about Support Lifecycle and MS06-015 – MSRC

Also note that the article's last line recommends; "that those of you who are still running these older versions of Windows upgrade to a newer, more secure version, such as Windows XP SP2, as soon as possible."  However, even Windows XP had been dropped from support several years ago, so clearly Windows 98 is extremely outdated and so unsafe to use in terms of security.

Nothing can protect such an outdated operating system, since beyond the fact that newer security applications won't operate on these older systems, even the most basic operating system security updates haven't been provided for these since the ended Microsoft support.

For these reasons, the only way to properly "secure" such a system is to keep it off of external networks like the Internet entirely, especially since these early Windows operating systems weren't truly designed to protect themselves from Internet types of attack in the first place, since direct connection to the Internet was relatively uncommon for most consumers and even many businesses at the time.

That Windows 98 system can only be considered a toy and a museum piece at this point, so understand that and if you'd like to keep it in operational condition, keep it off of all networks including internal except for very short periods when you're certain any other computers are either switched off or at least relatively certain not to contain any known malware.

This is what security professionals would have told any business attempting to use such a system as a warning once it approached and passed the end of Microsoft support back in the early 2000's.  I know, because I often did for financial and other businesses I supported or performed security analyses for.

Rob

Try:

'=========>>
Option Explicit

'--------->>
Private Sub Worksheet_SelectionChange(ByVal Target As Excel.Range)
    Dim Rng As Range
    Const myCells As String = "G7:G24, R7:R24"

    Set Rng = Me.Range(myCells)
    If Not Intersect(Target, Rng) Is Nothing Then
        teammembers.Show
    End If
End Sub
