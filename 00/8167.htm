a. user say:

Yes, I prefer not to have to turn to VBA etc., but...

...the following VBA should replace all paragraphs with "Level n" outline styles with the matching Heading style. If you have not used VBA for a while (or have never used it), the very first thing to do is enable the developer toolbar in Word->Preferences->Ribbon->Customize.

Then I suggest you use the developer toolbar to record a macro of a tiny action (e.g. record typing a single character. That will create a module called NewMacros n your Normal template if it doesn't exist, and create a macro in there.

Roughly speaking, you can then use the developer tab to open the VB Editor, find and open that module, replace the code of that recorded macro by the code I've posted below, and run the macro. I wouldn't waste more than 5 minutes on it.

Sub formatLevelsAsHeadings()
Dim level As Integer
For level = 0 To 8
  ActiveDocument.Select
  With Selection.Find
    .ClearFormatting
    .Replacement.ClearFormatting
    .Text = ""
    .Replacement.Text = ""
    .Forward = True
    .Wrap = wdFindStop
    .Format = True
    ' these two lines are a kludge
    .ParagraphFormat.OutlineLevel = wdOutlineLevel1 + level
    .Replacement.Style = WdBuiltinStyle.wdStyleHeading1 - level
    .Execute Replace:=WdReplace.wdReplaceAll
  End With
  Next
End Sub

imnoscrub say:

After tinkering around a bit, the following seems to be working... 

It's a simple C# user control (to create a task pane) with a single button that inserts inserts a slide from a separate PPTX file after the first selected slide in the currently open presentation:

public partial class MyUserControl : UserControl
{
    private const string SLIDE_TEMPLATE_PATH = @"C:\SlideTemplates\WoodTypeSlide.pptx";
 
    public MyUserControl()
    {
        InitializeComponent();
    }
 
    private void btnClickMe_Click(object sender, EventArgs e)
    {
        var presentation = Globals.ThisAddIn.Application.ActivePresentation;
        var selectedSlidesNumbers = this.GetSelectedSlideNumbers(presentation);
        var firstSelectedSlide = selectedSlidesNumbers[0];
 
        // Insert slide from file.  Note that this step will NOT bring the theme
        // from the slide file, only it's text.
        presentation.Slides.InsertFromFile(SLIDE_TEMPLATE_PATH, firstSelectedSlide);
 
        // Apply the theme of the newly inserted slide file.  Note that the new 
        // slide gets inserted after selected slide, so add 1 to slide index
        // to target new slide for theme.
        presentation.Slides[firstSelectedSlide + 1].ApplyTheme(SLIDE_TEMPLATE_PATH);
    }
 
    private int[] GetSelectedSlideNumbers(Presentation presentation)
    {
        var selectedSlides = new List<int>();
 
        foreach (Slide slide in presentation.Slides.Application.ActiveWindow.Selection.SlideRange)
        {
            selectedSlides.Add(slide.SlideNumber);
        }
 
        selectedSlides = selectedSlides.OrderByDescending(x => x).ToList();
 
        return selectedSlides.ToArray();
    }
}
