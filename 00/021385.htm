hansv mvp say:

I don't know if there is official terminology for this. A cell with a formula is never blank, even if the formula returns an empty string.

In a conditional formatting formula, you can test whether the value of a cell is an empty string, e.g.

=$C2=""

This formula will return TRUE if the formula in C2 returns "", while =ISBLANK($C2) will return FALSE since the cell contains a formula.
