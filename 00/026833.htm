Find text or partial text and vlookup associated part number - Microsoft Community

lorax49286 say:

Find text or partial text and vlookup associated part number

I work with tens of thousands of products that often have multiple chemical names.  I need to be able to look for text within a cell or table and return the associated part number. 

For example I may search for Ampicillin but the other cell will contain Ampicillin Sodium Salt. 

I've been digging through threads and I haven't found quite the right solution to be able to go down a list and search for the contents of each value in that list in a second list where the word being searched isn't always the same.  I'm not an excel guru and being able to do this would change my life, please help!

hansv mvp say:

Let's say the list with chemical names and part numbers is in Sheet2!A2:B20000

You enter a (partial) name in cell A1 on the current sheet. In the cell next to it, enter the following formula:

=VLOOKUP("*"&A1&"*",Sheet2!$A$2:$A$20000,2,FALSE)

"*"&A1&"*" means: look for the text of cell A1, with anything before and/or after it.

The formula searches in the first column of Sheet2!$A$2:$A$20000, and the 2 specifies that you want to return the value from the second column.

False means that you do want to find the exact text from cell A1.

The formula can be filled down if you have other (partial) names in A2, A3, etc.
