a. user say:

Hi JMC,

 

Virtual memory combines your computer’s RAM with temporary space on your hard disk. When RAM runs low, virtual memory moves data from RAM to a space called a paging file. Moving data to and from the paging file frees up RAM so your computer can complete its work.

This error message may also occur if Windows tries to create a paging file on an NTFS volume, but the System and Administrators accounts do not have the correct NTFS permissions on the volume.

 

I would suggest you to refer to the suggestion given by John Win 7 on November 24,2009 and check.
