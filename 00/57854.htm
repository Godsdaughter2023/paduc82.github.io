x86 is equal to 32 Bit.  The reson is that when PC's were first coming about the processor type was 8086.  It went on from there but always ended with the 86 at the end.  80486 was the last line of x86 processor before the Pentium line was released from Intel and that was still considered the 80586 by most.  Hence why the x86 has been referred to as 32 Bit as that was the processor line used back then.  It has nothing to do with 86 bits equalling 32 Bits or anything like that.  It is just an old CPU type and it stuck.

If you do have 64 Bit Version of Windows installed, you can find IE8 64 Bit on the Start Menu to use.  The reason why this isnt made the default, is that for the longest time there was no Flash Support for 64 Bit browsers.  Since Adobe released "Square" there is finally some support to use 64 Bit browsers.

Limited connection means the PC has successfully connected to the router, but the PC wasn't assigned a valid IP address, so you can't actually get to the Internet. It can also indicate that a valid IP address was assigned but that the PC doesn't have Internet connectivity. If you're connected to a home network, try restarting the router.

However Internet connection problems are most commonly caused by disconnected cables or by routers and modems that aren't working correctly.

 

I would suggest you to follow the steps provided below:

Method : Try resetting TCP/IP and check if it works.

a.    Boot to the desktop view.

b.    Open command prompt, right click in the left corner when the Start window appear and select command prompt (admin).

c.    At the command prompt, copy and paste (or type) the following command and then press ENTER:
netsh int ip reset c:\resetlog.txt
Note: If you do not want to specify a directory path for the log file, use the following command:
netsh int ip reset resetlog.txt

d.   Reboot the computer.
