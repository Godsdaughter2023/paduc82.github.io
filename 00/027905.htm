jay freedman say:

Based on watching what the For Each oPar loop did in a document that I made up, I think the problem is that VBA sees the section break at the end of section 1 as a "paragraph" with a length of 1 character and deletes it.

Instead, change the If statement to

If oPar.Range.Characters(1) = vbCr Then

Since the paragraph mark is ASCII 13 (= vbCr) and the section break character is an ASCII 12 ( = vbFormFeed), the condition will be false when oPar is a section break.

jay freedman say:

When you're going to exclude a certain number of items from a loop, it's easier to use a For loop with an index/counter and with the 'from' and 'to' indexes stated explicitly. Further, when you're using a numeric index to delete things, you must run the loop from the maximum index downward (otherwise at some point the index exceeds the number of remaining items and the macro throws an error). So the code for this will look something like...

Dim oPar As Word.Paragraph

Dim lngMax As Long

Dim lngMin As Long

Dim lngCtr As Long

     

     lngMax = ActiveDocument.Sections(1).Range.Paragraphs.Count

     lngMin = 9

    

     For lngCtr = lngMax To lngMin Step -1

        Set oPar = ActiveDocument.Sections(1).Range.Paragraphs(lngCtr)

        

        If oPar.Range.Characters(1) = vbCr Then

            oPar.Range.Delete

        End If

     Next lngCtr

The number for the minimum index (lngMin) needs to be set properly -- in this sample code it will be the ninth paragraph mark in the section, not necessarily the ninth visible paragraph of text.

To send a mass e-mail, you need to create a group. Then you need to enter all the contacts you want to be included in the group. Once done, you can enter the name of the group in the To fiels and click Send.
All the contact within that group will receive the same e-mail.

 

How to create Group :

Open outlook.com > place your cursor over the Outlook logo > a down arrow will appear > click it > select People > you are now in contact list window. They call it People.> click Manage at toolbar > select Manage Groups > click the + sign > a box will appear > enter the NAME of the group > click Apply.

Still in People window > click a contact you want to be added to the group > if you have multiple contacts to add, click the box at top left of the contact > continue clicking the top left box of each contact you want to add to the group until you are done> at the toolbar, click Groups > check the box in front of the group name > click Apply > all the contacts you have selected are now in the group.

Note : make sure you click squarely at the check box. If you click outside the box, all previous boxes you have checked will be nullified, and you’ll have to start all over.
