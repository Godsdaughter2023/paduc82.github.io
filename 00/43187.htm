Hi Satyaranjan,

The above issue might occur due to the following:

  1. Incompatible/non-updated device drivers.
  2. Third party applications conflicting with the device drivers installed.
  3. Hardware issue within the headphone.

Before we assist further, I would like to ask you a few questions:

  1. Did you try connecting another earphones/headphone and check if similar issue persists?
  2. Did you check if the headphone drivers shows up in Device manager on connecting to the PC?

Check if you are facing similar issue on connecting another earphones/headphone to the computer. If yes, try updating the earphones/headphone by following the steps below and check if the issue is resolved.

Since you have tried updating sound drivers under Sound, video and Game controllers, try updating the headphone drivers under Audio inputs and outputs in Device manager by following the steps below:

  1. Connect the earphones/headphone to the PC.
  2. Press Windows logo + X keys on the keyboard select Device Manager from the options.
  3. Expand Audio inputs and outputs option and check if the earphone/headphone drivers are visible in the window.
  4. If yes, right-click on the driver and select Update this software option.
  5. Follow the on-screen instructions and wait for the update to complete.
  6. Close the window after the update is complete and check if the changes are effective.
  7. If not visible, download and install the latest and most compatible device drivers from the manufacturer’s website and check if the issue is resolved.
