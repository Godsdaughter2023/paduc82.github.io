desiree yau say:

Hi,

Improper importing of a .pst file may cause missing emails. That being said, it is best if you will convert the .pst file into Outlook Data File before importing. Doing so emails will be imported into the computer first and will avoid losing emails when importing a large .pst file into Outlook.

Usually, you will import emails by using the Import/Export option in Outlook which is also correct, but if you have a large file, there is a big chance that some of the emails will not be imported.

In this case, we advise doing the steps below:

  1. Open Outlook.
  2. Click on the File tab.
  3. Select Open & Export.
  4. Under Open, choose Open Outlook Data File.
  5. Select the .pst file you want to import.
  6. Click on OK.
  7. Click on the File tab again.
  8. Select Open & Export.
  9. Choose Import/Export this time.
  10. Select the .pst file.
  11. Click on OK.

Once completed, kindly provide us with the result for further assistance.

I know of no bulit-in function.  Try this User Defined Function.  It works with both Inserted Hyperlinks and hyperlinks impleented the the =HYPERLINK() function:

Function hyp(r As Range) As String
hyp = ""
If r.Hyperlinks.Count > 0 Then
    hyp = r.Hyperlinks(1).Address
    Exit Function
End If
If r.HasFormula Then
    rf = r.Formula
    dq = Chr(34)
    If InStr(rf, dq) = 0 Then
    Else
        hyp = Split(r.Formula, dq)(1)
    End If
End If
End Function

How do I stop the share link from expiring? "Your organisation's policy requires this link to expire after 30 days" is the message I get when trying to set the expiry date of the link. The IT team don't seem to know how to change this and I can't find any guidance.

alex wang msft say:

Hi Steve,

The OneDrive for business in managed by your company. If your Office 365 account is removed, the content in OneDrive for business will be deleted eventually. You won't be able to access them once your account is removed. If you want to back up the content in your notebooks, there is no export method for notebooks on OneDrive for business, a simple simple workaround is to copy them to a notebook in your personal OneDrive. You can open the notebooks in OneNote for Windows 10, right-click the section > Move/Copy > select a notebook in your personal OneDrive.

About the expire data of the sharing links, your admins can control it in OneDrive Admin center > Sharing > Advanced settings for shareable links: Manage sharing in OneDrive and SharePoint. 

Regards,

Alex
