Hi CGSgtw, the only way Microsoft provides us to go from the 32bit version to the 64bit version is to clean install Windows 10 - there is no upgrade route . . .


You will need to backup your data, then clean install the 64bit version of Windows 10, restore your date, install a full set of device drivers and finally re-install your 3rd party applications, including Office (if applicable)

It is a lot of work, do you have a particular reason you want to go to the 64it version?



With regards to purchasing a new version, you can reinstall Windows 10 at any time and it will not cost you anything !

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

What will happen if I reset my Windows? - Microsoft Community

neilpzz say:

When you use Windows 10 Reset it effectively sets windows to a clean out of the box state. Any extra (non Windows 10 software) will be removed. You can select to Reset and Keep your files or Reset without keeping files.

If you bought your device with windows pre-installed then the makers software extra will be removed. You don't need a key it is embedded into the motherboard.

md. golam rabbi say:

I have software for Fingerprint Scanner. Do I need to reinstall them? This is my license.  I don't work any organization. It is my personal laptop.

neilpzz say:

You would need to install something like fingerprint scanner software. I'm not aware that comes as part of Windows 10, it sounds like it was software supplied by the manufacturer of the device.

With regards to your Activation question. It looks like activation came part of a business network. If you are absolutely sure this is not the case you can try to recover the embedded manufacturers key.

Open a Command Prompt (Admin) and type wmic path SoftwareLicensingService get OA3xOriginalProductKey press return and this should display the key. You can then enter this key by selecting Change product key. Again you do not need the key to perform a Reset.
