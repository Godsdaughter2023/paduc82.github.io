jeeped say:

No, the SUMIF() with INDIRECT() is not an updatable link. It will simply produce the #REF! when you open it without having the Commercial - January 2013 - December 2013.xlsx workbook open in the same workspace. If you subsequently open the external workbook, the SUMIF() will recalculate and provide the correct filtered total but if the external workbook is closed, the total will only hold until the next calculation cycle as INDIRECT() is considered a volatile formula which recalculates whenever any workbook-wide calculation cycle is processed, not just when a cell that affects the formula (like C7) is changed.

If this is a deal-breaker and you absolutely have to have the external references remain as INDIRECT() formulas, you might want to look into INDIRECT.EXT within the MoreFunc third-party add-on library. This does allow for indirect references to closed workbooks but I only recommend it as a last gasp resolution. Among other things, the add-on library would have to be installed in any Excel installation that uses the INDIRECT.EXT function.

There are a number of pirated and altered/malicious copies of the MoreFunc Excel add-on library but I have kept a clean copy on my OneDrive here.

Freeware MoreFunc Excel Library

jeeped say:
BTW, when I use the term current workspace, I am referring to a single application instance of Excel 2010. If you open a second application instance of Excel (another Excel application window), the INDIRECT() function cannot read data from that application window any more than it can read from a closed file. The two need to be opened within the same Excel application window.

The problem is that your formulas return text strings, so the conditional formatting rule compares values as text, not as number.

Change the formulas as follows:

=IF(D3=0,"",DATEDIF(TODAY(),D3,"m"))

and apply the following custom number format to the cells with the formulas:

0" months"

This way, the cell values are numbers but they display text.

You can now create a rule such as

Cell Value between 10 and 9
