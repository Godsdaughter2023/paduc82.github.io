jay freedman say:

The steps you listed are correct. It's possible that, even though you specified Normal.dotm as the place to store the macro, it wasn't saved there.

To check on this, click View > Macros > View Macros. In the dialog, set the "Macros in" dropdown to Normal.dotm. If the name you gave to the macro is in the list, click that name and then click the Edit button. You should see the recorded code in the macro editor.

If the name of the macro isn't in the list in the Macros dialog, the macro wasn't saved in Normal.dotm. This could happen if, after recording and creating the button, you shut down Word and clicked No when asked whether to save changes in Normal.dotm. It could also happen if you were never asked (maybe because in File > Options > Advanced > Save, the option "Prompt before saving Normal template" is unchecked) and the automatic save failed for some reason.

Try this: First, go to Options > Customize Quick Access Toolbar and remove the button. Then go through the whole process again. Before you shut down Word, press Alt+F11 to open the macro editor. Double-click any module under the Normal project:

Then press Ctrl+S to save Normal.dotm (rather than the currently open document).
