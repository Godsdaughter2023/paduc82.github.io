The best way to create an image backup is to use a third-party image backup program. Windows' own System Image Backup is deprecated (no longer being developed) and Microsoft recommends instead to use third-party programs.

Each image backup program does things its own way, but in general the steps are:

1- Choose the drive or drives you wish to backup.

2- Choose a destination for the backup (which should be somewhere outside your computer.)

3- Choose a time to execute the backup - You can do it immediately or on a schedule.

4- Choose how many previous backups you want to keep.

To find third-party image backup programs, use this as your internet search term:

best image backup for Windows 2018
