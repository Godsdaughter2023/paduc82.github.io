ken sheridan say: Don't calculate age by division, it's not a reliable method.  To return age in years use a function like this:

Public Function GetAge(varDoB As Variant, Optional varAgeAt As Variant) As Variant

    If IsMissing(varAgeAt) Then varAgeAt = VBA.Date
    
    GetAge = DateDiff("yyyy", varDoB, varAgeAt) - _
        IIf(Format(varAgeAt, "mmdd") < Format(varDoB, "mmdd"), 1, 0)

End Function

To return age in Years:Months use this function:

Public Function GetAgeYearsMonths(varDoB As Variant, Optional varAgeAt As Variant) As Variant

    Dim intYears As Integer
    Dim intMonths As Integer

    If IsMissing(varAgeAt) Or IsNull(varAgeAt) Then varAgeAt = VBA.Date
    
    If Not IsNull(varDoB) Then
        intYears = DateDiff("yyyy", varDoB, varAgeAt) - _
            IIf(Format(varAgeAt, "mmdd") < Format(varDoB, "mmdd"), 1, 0)
           
        intMonths = DateDiff("m", varDoB, varAgeAt) - (intYears * 12)
            
        If Day(varDoB) > Day(varAgeAt) Then
            intMonths = intMonths - 1
        End If
            
        GetAgeYearsMonths = intYears & ":" & Format(intMonths, "00")
    End If
    
End Function

In both cases they default to returning the age as of today for the date of birth entered.  To return someone's age on a specific date, pass the date into the function as the optional second argument.  So you'd use:

=GetAge([dob],[arrdate])

bob jones aka: cybertaz say:

Dynamic Guides work differently in PPt 2016.

One decided difference is that in prior releases the shape you were dragging remained visible at the point of origin, so the guides gave an indication of where you were positioning the 'instance' of the shape relative to its original location. In PPT 2016, that doesn't happen, if that's what you're expecting. Notice that as soon as you start to drag it, the shape moves accordingly without remaining visible at the point of origin. Hence, no guides appear because the instance being dragged is not aligned with anything. The guides display only when the edge or midpoint of what you're dragging align with that of another shape.

Another difference is that the Dynamic Guides are faint dashed lines rather than pronounced solid lines, so they can be difficult to see. I've also found that they require very precise positioning & will disappear if the alignment is even a fraction of a hair off.

Just in case you aren't aware: If you want to move a shape without affecting its Horizontal or Vertical plane, hold Shift while you drag it... or use the arrow keys to nudge it.

BTW: For clarity, the option for Color guides is available only in the Office 365 subscription release of Office 2016. It isn't available if you have the Retail License. Also, I believe it pertains only to Static Guides, not Dynamic Guides. I can't test because I have the Retail package, but I don't see how you could right-click a Dynamic Guide in the first place :-)

anna bon say:

Hello,

If you are using an Office 365 Home, usually the list of devices which have the Office 365 installed is visible on your account. If you have an Office 365, you can deactivate an Office install that you're no longer using so you can install Office on a different device. This is different from uninstalling Office, which doesn't deactivate the install. This is also different from canceling your Office 365 subscription. Please refer to the steps below:

  1. Go to https://office.com/myaccount and sign in using your Microsoft account, if prompted.

  2. Select Install >.

  3. Under Install information, locate the computer where you want to deactivate the install.

  4. Select Deactivate Install to deactivate the Office installs that you no longer use. If you don't see Deactivate Install, it's because you have a one-time (non-subscription) purchase of Office that can't be deactivated using this method.

If you need clarifications, feel free to message us again.


Thank you.
