doug robbins - mvp office apps & services (word) say:

If the Calendar that you mean is the 2012 Calendar Basic_one month evergreen, if you create a macro in your Normal.dotm template or in another template that you save in the Word Startup Folder and you start with a document that contains just the one page calendar.  when you run the macro, it will add a page for each day of the month to the calendar, inserting the date in the format that you want at the top of each page:

 

Sub AddPageperDay()

Dim i As Long
Dim rng As Range
With ActiveDocument
    .PageSetup.VerticalAlignment = wdAlignVerticalTop
    For i = 1 To Val(Format(.Variables("monthEnd"), "d"))
        .Range.InsertAfter Chr(12)
        Set rng = .Range
        rng.Collapse wdCollapseEnd
        .Fields.Add rng, wdFieldEmpty, "Docvariable MonthStart \@ MMMM"
        .Range.InsertAfter " " & i & ", "
        Set rng = .Range
        rng.Collapse wdCollapseEnd
        .Fields.Add rng, wdFieldEmpty, "Docvariable MonthStart \@ yyyy"
        .Range.InsertAfter vbCr
    Next i
    .Range.Fields.Update
End With

End Sub

suzanne s. barnhill say:

"Linked styles," which are indicated in the Styles pane by an icon that combines the pilcrow (paragraph symbol) of a paragraph style and the lowercase a of a character style. These can be applied to an entire paragraph, as a paragraph style, or to part of a paragraph, as a character style. If you apply a heading style to part of a paragraph, it will apply the font formatting of the heading style, and this "lead-in emphasis" will be considered a heading for TOC purposes, but will not be displayed in the Navigation pane. For that reason, it is generally less confusing to disable linked styles and use styles for only one purpose or another.

The "Automatically update" box in the Modify Style dialog is helpful when you are starting a document and not sure how you will want certain styles formatted. As you make changes and work toward a final design plan, you can change the formatting of one paragraph in the given style, and the style will be updated automatically, with the change propagated to all other paragraphs in that style. So you can constantly change your Heading 1 formatting till you get it to suit you. Then turn that option off. As long as it is enabled, you won't be able to apply any direct formatting to a given paragraph without changing others (and sometimes, no matter how many styles you're using, and no matter how carefully you've designed them, you do need to make manual adjustments). By design, the TOC styles are set to update automatically. This makes it easier to adjust the formatting of the TOC since those styles are sometimes hard to access for modification.
