bobgarrell say:

Unable to install or uninstall MS Office / Visio

I am running Win 7 Home Premium edition 64-bit on an HP Pavilion Elite with i5 processor, 8 Gb RAM.  It has run flawlessly for a year and a half. 

The other day we had multiple power outages.  The system was on a UPS, so I went to shut it down.  During shutdown it starts doing updates - it also seems to create restore points.  Halfway through, the power in the UPS runs out. 

Since power up, MS Office & Visio both fail.  (I have HUP licenses).  I have tried going into control panel and remove old version.  I get a setup error: "The language of this installation package is not supported by your system" and it fails.  I also can re-install over the existing version.

It can't find a restore point that works.  How do I reload?  Do I really have to install the OS (and everything else) since restores won't work?  I think the HUP license only allow two installs.  I think I might have already used both with the intial install and then one when I updated to a larger HDD.

Thanks in advance.

bob

dena b say:

Hi bobgarrell,

Were the updates completely installed? It's possible the updates are not completely installed and may be causing the issue. Since you are unable to find a suitable restore point, follow the steps below to remove the recently installed updates.

Boot from the Windows 7 DVD, select your keyboard and language options, select Repair your computer, then select the Command Prompt. From the Command Prompt run the following command:
Dism /image:c:\ /get-packages

This will produce a list of installed updates and packages. They will look like this:

Package Identity : Package_for_KB978262~31bf3856ad364e35~x86~~6.1.1.0
State : Installed
Release Type : Security Update
Install Time : 2/11/2010 1:01 PM

Find the most recent one and then make a note of the package identity.

 

Then run:

dism /image:c:\ /remove-package /PackageName:Package_for_KB978262~31bf3856ad364e35~x86~~6.1.1.0 (name of update from previous step)

 

You will get a status of the removal until it is complete. Then reboot your PC.

You will want to start with the most recent and remove them one at a time to find the problem update.
