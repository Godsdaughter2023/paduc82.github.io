tasos_k say:

Hi,

make a copy before you run this code

 

 

Sub DeleteHidden_RC()

'delete all Hidden rows-columns in Active sheet
Dim r As Long, c As Long, x As Long
r = ActiveSheet.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
c = ActiveSheet.Cells.Find(What:="*", SearchOrder:=xlByColumns, SearchDirection:=xlPrevious).Column

Application.ScreenUpdating = False
For x = r To 1 Step -1
If Rows(x).Hidden = True Then Rows(x).Delete
Next
For x = c To 1 Step -1
If Columns(x).Hidden = True Then Columns(x).Delete
Next

Application.ScreenUpdating = True
End Sub

    colbys say:

    Hi Mikki

    Had the same problem and found the answer.

    The print options available to you under the page setup menu is actually determined by your printer driver.  If you are using Windows 10, the default setting is to let Windows manage your default devices (printers included).  If, for whatever reason, you open Wordpad and Windows 10 is thinking you might print to PDF, or XPS, or whatever, the options may differ.

    To remedy the problem:

    Open your Control Panel

    Open Devices and Printers

    Right Click and "Set to Default" your printer that you use to print labels.

    Windows will throw up a prompt asking if you really want to stop Windows from managing your default printers.

    Select "Yes"

    Open Wordpad and your envelope option should be back.

    bob jones aka: cybertaz say:

    This is not at all uncommon, the primary explanation being that a Background was never intended to be printed :-) As a result, most printers cannot properly understand how they should be handled... That's why the default Print Preference is set to not include them.

    The Background feature was added back when Microsoft had the daft notion that Word effectively could be used for designing web pages. They were intended strictly for on-screen display & honestly are rather effective for that purpose. However, the feature was effectively done away with in Office 2011.

    Two alternatives you might consider;

    • Create your background graphic using a decent graphics program & saved as a JPEG image, then insert it into the Word document as a Watermark, or
    • Use Word's drawing tools to create a filled rectangle on the Header/Footer layer of the document.

    Just be aware that either of the 3 methods will slow down print jobs, consume a great deal of ink, well may over-saturate the paper, cause the main text to bleed, as well as other possible issues that adversely affect readability.

    hansv mvp say:

    Let's say you create a table tblFMonths that looks like this (the dates are just examples)

     

    FMonth     StartDate         EndDate
    201301      12/31/2012     01/27/2013
    201302      01/28/2013     02/24/2013
    201303      02/25/2013     03/24/2013
    ...

     

    The 'main' table is tblData

     

    TransactionDate    Amount
    01/04/2013            $100
    01/22/2013            $250
    01/30/2013            $120
    02/11/2013            $160
    ...

     

    You can create a query like this:

     

    SELECT tblData.ID, tblData.Amount, tblFMonths.FMonth
    FROM tblData, tblFMonths
    WHERE tblData.TransactionDate Between [StartDate] And [EndDate]

     

    (This is just a limited example, of course, in reality you'd have more fields)

    This query can act as record source for the report; you can group it on FMonth.

    1. On your inbox, click the gear icon and select More mail settings on the dropdown list.

    3. Click Safe Senders.

    4.In the text box, type in the email address or domain you’d like to always receive email from, and then click Add to list >>.

     

    D. Check your Email Rules

    1. On your inbox, click the gear icon and select More mail settings on the dropdown list.

    Click Edit next to the rule that you want to edit.
