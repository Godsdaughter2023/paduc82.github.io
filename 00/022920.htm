suzanne s. barnhill say:

Are you really using WordPad or Word 2003 as indicated? If it's WordPad, I don't think there's a way to get the action you want, but there is a choice in Word.

As Hans indicated, even in Word, if you select text and press Tab, you'll replace the text with a tab character, but if you are wanting to indent it, you need to change one of Word's options.

  1. On the Tools menu, select AutoCorrect Options.
  2. In the AutoCorrect dialog, select the AutoFormat As You Type Tab.
  3. Check the box for "Set left- and first-indent with tabs and backspaces."

Note that this setting is a little misleading: Tab will indent text only if an entire paragraph is selected (otherwise it just inserts a tab character). Backspace, on the other hand, will delete selected text; it works only if you place the insertion point at the beginning of a paragraph. What does work on selected text is Shift+Tab, which also works at the beginning of a paragraph.
