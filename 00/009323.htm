Saving a "Custom" font color for reuse. - Microsoft Community

muskmel0n say:

Saving a "Custom" font color for reuse.

In Word 2016 (Windows 7 Pro), Excel 2016 (Windows 7 Pro), and Outlook 2016 (Windows 7 Pro), I frequently change the font color to various colors approved in my employer's Brand Style Guide.

In order to select these colors,  I create a custom color by using the font color drop down menu to navigate to "More Colors" and then to "Custom" where I enter the RGB values.

After I create a custom color, it appears as a choice under the "Recent Colors" heading, but only during that session. If I close the application and reopen it, there are no Recent Colors options.

Is there a way to save the custom colors to the "Theme Colors" or "Standard Colors" choices?  Or is there a way to save Recent colors between sessions? I would like to enter the RGB values once, and then have them as a choice from that point on.

jay freedman say:

Create a character style that consists of the Default Paragraph Font plus your custom color. (Note: If you use the Create a Style command on the dropdown menu of the Quick Styles gallery on the Home tab, you must click the Modify button in the first popup box and change the style type to Character; once the style is created, you cannot then change its type from Paragraph or Linked to Character.)

The "New documents based on this template" saves the style definition in the template on which the current document is based (typically Normal.dotm).

You can then apply the style to any text in any document based on the same template. It won't change the font, size, etc. but only the color.
