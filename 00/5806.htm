bradstauf say: In case you are still looking for the answer, go to control panel, devices & printers (assuming windows 7, otherwise something like 'scanners and cameras').  Find the scanner, right-click, select 'scan properties' and then the 'events' tab.  You should be able to associate an action with each button, this will prevent the choice prompt from coming up.  I had the same problem and uninstalling publisher did not fix it, oddly enough.  Nor was there a way to set this from within the scanner software (canon mp navigator), it was unaware that I had publisher installed.

davidf2 say:

Of course there would be a setting on the printer to setup where to direct the scan. It could go as an image to your computer, to your desktop, to a program such as Publisher or Word, to a FAX, or directly to the printer to print a copy of the scanned page. Read the documentation for your scanner on how to use it.

 

Yes, you can uninstall Publisher but that is a bit overkill unless you never use the program. You would uninstall it just like you uninstall any program. If you are using XP go to control panel and uninstall.
