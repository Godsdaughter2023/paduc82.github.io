hansv mvp say:

Something like:

    Dim xlApp As Object
    Dim xlWbk As Object
    Dim xlWsh As Object
    Set xlApp = GetObject(Class:="Excel.Application")
    xlApp.Visible = True
    Set xlWbk = xlApp.Workbooks("Template.xlsx")
    Set xlWsh = xlWbk.Worksheets("Sheet1")

Change Template.xlsx to Template.xlsm if that workbook contains VBA code.

You can then refer to a range on xlWsh, for example

    xlWsh.Range("A1").Value = SomeValue
