beth melton say:

If you're asking how to add a Fax cover page to the Cover Page gallery the follow these steps:

  1. Create your Fax cover page in a Word document.
  2. Select your cover page and from the Page Layout tab, click Cover Page and then click Save Selection to Cover Page gallery
  3. Provide a name for your Cover Page building block, add a new category if desired (for example if you want yours to appear at the top of the gallery, create a new category that has a space before the name so it will sort to the top), and make sure the Options drop down reflects Insert content in its own page.
  4. Click OK and your custom cover page will be available to any Word document

Now, if your documents have page numbering and if you don't want your cover page to be numbered then before you save your custom cover page to the gallery you need to make these changes:

  1. After your cover page, from the Page Layout tab, insert a Next Page Section break.
  2. Right-click the header area for your cover page and then click Edit Header.
  3. From the Header & Footer Tools Design tab, in the Options group, click Different First Page
  4. Optional: From the Header & Footer Tools Design tab, in theHeader & Footer group, click Page Number, clickFormat Page Numbers, and then set your Start At page to 0.

 

~Beth Melton Microsoft Office MVP

bernie deitrick say:

The simplest thing to do is to use your correlation function / evaluation with reduced and offset sets of data.  For example, let's say that you had a function like this - 

=CORREL(B2:B30,C2:C30)

to check for a one year time lag, just use one fewer row in each range but offset the ranges by 1 row:

=CORREL(B2:B29,C3:C30)

(or the reverse!   =CORREL(B3:B30,C2:C29)     )

for a two year time lag - note the offsets:

=CORREL(B2:B28,C4:C30)

As an alternative, you can just move your data around - without changing all of your formulas. 

Create a copy your workbook/ worksheet, then copy the data from one column and move it up one row (don't move it - copy and paste it so that the formulas are not affected)  overwriting the first year's data and the delete the last row to remove it from the analysis. I'm not sure which column to move but you can figure that out easily enough.

Yes! That is correct.

I am assuming that you have an ActiveX control and if so then you could use the LostFocus event and then instead of selecting the required cell and then selecting the value in the ListBox you could select the value in the ListBox and then select the cell where you want the value. Note that if you select the required cell for the value first then you will need to re-select the same cell to trigger the lost focus event and the lost focus event will populate the cell that you select after selecting the ListBox value.

Private Sub ListBox1_LostFocus()
    ActiveCell.Value = ListBox1.Value
    ActiveCell.Offset(1, 0).Select
End Sub

bernie deitrick say: None of the available Excel buttons have a mouse-over event, but you can use the mouse move event with a commandbutton from the controls toolbar (the kind that does not get a macro assigned to it but uses code) to show alternative text when the lower right corner is crossed:

Private Sub CommandButton1_MouseMove(ByVal Button As Integer, ByVal Shift As Integer, ByVal X As Single, ByVal Y As Single)
    If X > CommandButton1.Width * 7 / 8 And Y < CommandButton1.Height * 3 / 4 Then
        CommandButton1.Caption = "Hover caption"
    Else
        CommandButton1.Caption = "Standard caption"
    End If
End Sub

Auto Insert with Outlook VBA

  1. In the first place, press “Alt + F11” key buttons to access VBA editor.
  2. Then double click the “ThisOutlookSession” project on the left side to open it.
  3. Next copy the following codes into it.
Public WithEvents objInspectors As Inspectors
Public WithEvents objMail As MailItem
Private Sub Initialize_handlers()
    Set objInspectors = Application.Inspectors
End Sub
Private Sub objInspectors_NewInspector(ByVal Inspector As Inspector)
    If TypeOf Inspector.CurrentItem Is MailItem Then
       Set objMail = Inspector.CurrentItem
    End If
End Sub
Private Sub objMail_Open(Cancel As Boolean)
    Dim strTime As String
 
    'the current time
    strTime = Now
 
    'insert to subject
    'If you don't want to insert to subject, remove the following line
    objMail.Subject = strTime
    'insert to body
    'If you don't want to insert to body, remove the following line
    objMail.Body = strTime & objMail.Body
End Sub

Note: The above codes will work to auto insert the current time into the subject and body. If you want to insert the current date, replace the “objMail_Open” sub with the followings:

Private Sub objMail_Open(Cancel As Boolean)
    Dim strDate As String
 
    'the current date
    strDate = Date
 
    'insert to subject
    objMail.Subject = strDate
    'insert to body
    objMail.Body = strDate & objMail.Body
End Sub
  1. Subsequently, sign this code as usual.
  2. Later, restart Outlook to activate the macro.
  3. Finally you can have a try.
  • Directly click on the “New Email” button to open a new email.
