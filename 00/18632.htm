With a defined name of Name1 identifying a vertical contiguous range of cells 1 column wide suitable to being the source for a data validation dropdown, this worked for me.

Sub Test()
Dim v As String
v = "Name1"
abc v
End Sub

Sub abc(LIST_NAME As String)

   With Selection.Validation
        .Delete
        .Add Type:=xlValidateList, AlertStyle:=xlValidAlertStop, Operator:= _
        xlBetween, Formula1:="=" & LIST_NAME

        .InputTitle = ""

  9. Restart the IIS service for the changes to take effect.

  • You can use the SSLChainSaver utility to determine which certificates are missing.

How to add a shared mailbox to the Outlook app on iPhone or Android

At this time, it's not possible to open a shared mailbox on a mobile device as a shared mailbox. Support for shared mailboxes in Outlook for iOS and Android is in development.  

*** Update: As of Fall 2019, users with Office 365 Exchange online accounts can add a Shared Mailbox or Shared Calendar to the mobile apps. Click the Home button (upper left) then the Add Account or Add Calendar button (you need to be in the Calendar module to add a shared calendar)  and select Add Shared MAilbox or Add Shared Calendar. ***

At this time, you need to have a password for the mailbox and add it as an account or open it in a web browser. 

Add a shared mailbox to Outlook on the web

Open and use a shared mailbox in Outlook Web App (Exchange 2010 and 2013)

If you don't have a password for the account, you may be able to add it to the mail app as an IMAP account and login using this format for the login address: youremail@address\sharedmailboxalias  and your own password.  You will not be able to send mail from the address. I have instructions for this method at Shared Mailboxes on a Smartphone.
