riel alc say:

Hi Robert,

The UpdatusUser account is automatically created by NVidia. If you have an Optimus-enabled notebook, NVidia update will be installed automatically. If you wish to delete this user, please follow the steps below:

  1. Press Windows key +X, then select Command Prompt (Admin).
  2. Copy and paste net user /delete updatususer, then hit Enter.
  3. Copy and paste sc delete nvupdatusservice, then hit Enter.

Note: These commands will remove both the user and services.
