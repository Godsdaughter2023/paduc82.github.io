How do i disable italics when writing a math equation? - Microsoft Community

mart687e say:

How do i disable italics when writing a math equation?

I use the hotkey shift + alt + = to write an equation in word, with the mathematics plugin, but i don't want it to default to italics. I can't seem to turn it off, can you help me?

jay freedman say:
You can't turn it off completely. What you can do is type the equation, then select the whole equation and click the Normal Text button at the left end of the Equation Tools tab of the ribbon.

mart687e say: Are you actually serious? Word has so many features i simply do not understand nor use, and this is not possible? That seems pretty ridiculous... 

jay freedman say:

Yes, I'm serious.

It's a standard convention in mathematical publishing that all variables (except sometimes vectors) are italicized. Microsoft had no reason to suspect you would want your equations to use roman characters instead.

There is an option that can be misapplied to do what you ask. Although variables are italicized, function names such as "sin" and "cos" are not italicized. Word has a list of recognized functions, and you can add to that list. With the cursor in an equation, click the dialog launcher arrow in the bottom right corner of the Tools group on the Equation Tools tab. In the dialog that opens, click the Recognized Functions button. In the next dialog, type one letter that you expect to use as a variable and click the Add button; then add other variables in the same way.
