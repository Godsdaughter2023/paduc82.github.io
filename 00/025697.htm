Pin a Document - Microsoft Community

gary's student say:

Pin a Document

Using VBA, how can I pin a document to the Recent Documents list?
hansv mvp say:

There is no direct support in VBA for pinning a document in the Recent Documents list, so you have to edit the registry. Unlike pinning a document in the interface, the result will only become visible after quitting and restarting Excel, so it appears to be of limited use to use VBA for this.

 

Here is a procedure you could use:

 

Sub Pin(strFullPath As String)
    Dim WSHShell As Object
    Dim strKey As String
    Dim strItem As String
    Dim i As Long
    Set WSHShell = CreateObject("WScript.Shell")
    strKey = "HKEY_CURRENT_USER\Software\Microsoft\Office\" & _
        Application.Version & "\Excel\File MRU\"
    For i = 1 To Application.RecentFiles.Count
        strItem = WSHShell.RegRead(strKey & "Item " & i)
        If InStr(strItem, strFullPath) > 0 Then
            If InStr(strItem, "[F00000001]") > 0 Then
                MsgBox strFullPath & " has already been pinned!", vbInformation
            Else
                WSHShell.Regwrite strKey & "Item " & i, Replace(strItem, _
                    "[F00000000]", "[F00000001]"), "REG_SZ"
            End If
            Exit Sub
        End If
    Next i
    MsgBox strFullPath & " not found in Recent Files!", vbExclamation
End Sub

How do I keep the rows on a Table to stay on the same page?

I am using VBA to create a table with a variable number of rows.  I want to prevent Word from splitting any row so that no row is on more than 1 page (each row is either 4 or 5 lines long).

How do I do that in Word?  Using VBA?
rohn007 say:
You can do the same thing by selecting the whole table or rows you want to affect then click on the Home tab > Paragraph group > dialog launcher. Then in the Paragraph dialog, click on the Line and Page Breaks tab.  You want to turn on the "Keep Lines Together" and or "Keep with Next" options.

jay freedman say:
Assuming you have a variable in the VBA declared as a Table object, like

   Dim tbl As Table

and you've assigned the table the macro creates to that variable, then you just need the line

   tbl.Rows.AllowBreakAcrossPages = False

This is the equivalent of selecting the whole table, clicking the Properties button on the Table Tools > Layout tab of the ribbon, going to the Row tab of the Properties dialog, and clearing the check box for "Allow row to break across pages".
