Hi Teri, I'm an independent adviser and will try to help.

One of the principles of a relational database is to eliminate redundancy. Another is to define information by data in rows and columns, not by object names. So having tables with duplicate structures for each book named for the book, would violate those principles.

You should have one table for all the books with a field that identifies the book. This will make it easier to search for references that might be in multiple books as well as being proper design.

ossiemac say:

Without a copy of the workbook then it is hard to be sure of the exact problem. However, try the code below. It resets the page breaks to remove any manually inserted page breaks (This should always be done when resetting the PrintArea.) and also uses a UDF to find the last used cell on the worksheet. This method is a very positive method of finding the last cell containing data. I am aware of instances where the method I used in my previous post finds the last formatted cell rather than the last cell containing

I have also included a commented out line that you can use if you wish to include the column and row Id's in the printout. If you cannot get it working then knowing exactly what columns and rows are printing rather than just blank pages might help to isolate the reason that blank pages are printing.

Note that you will need to install the SetPrintArea code as well as the Function LastRowOrCol. (I include this Function in all of my projects because it is so useful and very positive in finding the last cell with data.)

Sub SetPrintArea()
    Dim ws As Worksheet
    Dim lngLastRow As Long
    Dim lngLastCol As Long
    Dim rngPrint As Range
   
    Set ws = Worksheets("Sheet1")   'Edit "Sheet1" to your worksheet name
    lngLastRow = LastRowOrCol(ws, True)
    lngLastCol = LastRowOrCol(ws, False)
   
    With ws
        Set rngPrint = .Range(.Cells(1, 1), .Cells(lngLastRow, lngLastCol))
       
        .ResetAllPageBreaks 'Remove any manually inserted breaks
       
        .PageSetup.PrintArea = rngPrint.SpecialCells(xlCellTypeVisible).Address
           
        '.PageSetup.PrintHeadings = True    'Optional to print column and row headers
    End With
   
End Sub

Function LastRowOrCol(ws As Worksheet, bolRowCol As Boolean, Optional rng As Range) As Long
    'Finds the last used row or column in a worksheet
    'First parameter is Worksheet
    'Second parameter is True for Last Row or False for last Column

    'Third parameter is optional and finds last used cell of specific range on a worksheet

   
    Dim lngRowCol As Long
    Dim rngToFind As Range
   
    If rng Is Nothing Then
        Set rng = ws.Cells
    End If
   
    If bolRowCol Then
        lngRowCol = xlByRows
    Else
        lngRowCol = xlByColumns
    End If
   
    With ws
        Set rngToFind = rng.Find(What:="*", _
                LookIn:=xlFormulas, _
                LookAt:=xlPart, _
                SearchOrder:=lngRowCol, _
                SearchDirection:=xlPrevious, _
                MatchCase:=False)
    End With
   
    If Not rngToFind Is Nothing Then
        If bolRowCol Then
            LastRowOrCol = rngToFind.Row
        Else
            LastRowOrCol = rngToFind.Column
        End If
    End If
   
End Function
