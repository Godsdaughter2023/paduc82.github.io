Method 1: Run the Hardware Troubleshooter

 

The Hardware Troubleshooter is an automated tool which will check the hardware connected to the computer for any known issues with it and provides the details and on how to fix them. Follow these steps and run the troubleshooter.

 

a)       Press “Windows Logo” + “W” keys from the keyboard.

 

b)       Type “Troubleshooting” in the search bar and press “Enter”.

 

c)       In the “Troubleshooting” window, click on “View All” on the left pane.

 

d)       Click on “Hardware and Devices”.

 

e)       Click on “Advanced” and then click on “Run as Administrator”.

 

f)        Click “Next” and follow the on-screen instructions to complete the troubleshooting process.

 

If the troubleshooter was unable to find and fix any problems with the hardware devices installed on the computer, then try the next method and check.

 

 

Method 2: Check the Windows updates for any latest driver updates.

 

After running the troubleshooter, check in Windows updates if any driver updates are available for the ALAN SmartPad. If so, install the same and check if it helps.

 

a)       Press “Windows Logo” + “W” keys from the keyboard.

 

b)       Type “Updates” in the search bar and click on “Install optional updates” from the left pane.

 

c)       From the “Windows Update” window, click on “Check for Updates” on the top left corner to receive a fresh list of updates and install all the updates listed there.

 

If there are no driver updates available through the Windows update and if the SmartPad device is shown in the “Device Manager” then try the next method to uninstall the existing SmartPad drivers from the computer and then reinstall the latest driver from the computer manufacturer’s website.

 

 

Method 3: Uninstall the existing touch pad drivers and reinstall the latest drivers from the manufacturer’s website.

 

Step 1: Uninstall the existing SmartPad drivers from the Device Manager

 

a)       Press “Windows Logo” + “X”.

 

b)       Click on “Device Manager”.

 

c)       Search for the SmartPad drivers and expand it (It should be available under “Mice and other pointing devices).

 

d)       Right click on touch pad drivers and then select “Uninstall”.

 

e)       Follow the on-screen instructions and you may have to restart the computer to complete the un-installation process.

 

 

Step 2: Reinstall the latest touchpad drivers from the Acer website.

 

After the computer is restarted, you should go to the computer manufacturer’s website and then download and install the latest drivers available for the ELAN SmartPad.

 

 Hope this resolves the issue. Please feel free to get back to us for any further queries or issues related to Windows.
