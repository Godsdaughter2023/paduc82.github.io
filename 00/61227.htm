If its a retail store bought license, yes, you can transfer it to your Mac. If that license came preloaded on the computer you are selling, then you cannot transfer it.

 

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

"frequency out of range" when connecting laptop to projector - Microsoft Community

gurumurthysekhar say:

"frequency out of range" when connecting laptop to projector

original title: G Sekhar

 

laptop lenovo OS is Windows 8. Display not coming when connected to projector. Getting error message "frequency out of range" Tried in all resolutions but no display in screen. Computer recommended resolution is 1366 X 768

kevin d souza say:

Hi Gurumurthy,

 

If there is no picture and the screen is completely black, there may be a loose connection or an incorrect setting or the display driver might have been corrupted.

 

1. Make sure there are no loose connections. Unplug the connections from the laptop to the projector and turn off both. Restart them and connect them once again and check if that helps.

 

2. Uninstall the current display adapters installed on the computer and reinstall the latest display drivers from the Lenovo website.

 

- Type device Manager in the Search Charms.

- Browse display adapters, right click on the driver and click on Uninstall.

note: make sure you make a note of the video driver installed on the computer.

- Once the video driver is uninstalled, restart the computer and download/install the drivers from the Lenovo website.

http://support.lenovo.com/en_US/downloads/default.page?selector=expand#

 

3. Set the screen resolution to 1024×768 and also reduce the Refresh rate to 60Hz.

- Right click on the desktop and select Screen resolution.

- Change the screen resolution and then press apply.

- Click on Advanced settings and click on the Monitor tab.

- Change the  screen refresh rate to 60HZ and press okay.

- Restart the computer and the Projector and check.

 

Hope this information helps. If you have any concerns, let us know.
