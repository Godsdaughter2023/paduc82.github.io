melowin mar say:

Hi Mohammed,

As it turns out, merging two Microsoft account is currently not possible. However, you can change the way you sign in and show up to recipients by adding aliases to your Microsoft account. An alias is like a nickname for your account that can be an email address, phone number, or Skype name. It uses the same inbox, contact list, and account settings as the primary alias. You can sign in to your account with any alias, and you only have to remember a single password for all of them.

Notes:
  • You cannot add an alias that is already associated with another Microsoft account. Aliases cannot be used to share information between two Microsoft accounts—the ability to link Microsoft accounts was discontinued in 2013. 
  • Existing @hotmail.com, @live.com, @outlook.com, and @msn.com addresses can't be added as aliases.

By default, the email address or phone number that you use to sign up for your Microsoft account is considered the primary alias. Your primary alias shows up as the main account name when you view your profile information. This can be changed at any time. To add an alias, here's how:

  1. Sign in to the Microsoft account website.

  2. Select Your info.

  3. Select Manage how you sign in to Microsoft.

  4. Select Add email or Add phone number.

  5. Follow the instructions to set up and verify your new alias. If you chose Add email, you have the option to use an email address that you already have, or you can create a new Outlook.com email address.

Any ideas of how to accomplish this crazy task?

ken sheridan say: You just need to reference the combo or list box(whichever you've used), using the MONTH function to get the month of the date value in each column in the table:

PARAMETERS [Forms]![f_Special Dates Selection]![lstMonth] SHORT;
SELECT [First Name], [Last Name], [Date of Birth], [Contract Anniversary], Spouse, [Spouse DOB], [Wedding Anniversary]
FROM [Producer List]
WHERE MONTH([Date of Birth])= [Forms]![f_Special Dates Selection]![lstMonth]
OR MONTH([Contract Anniversary]) = [Forms]![f_Special Dates Selection]![lstMonth]
OR MONTH([Spouse DOB]) = [Forms]![f_Special Dates Selection]![lstMonth]
OR MONTH([Wedding Anniversary]) = [Forms]![f_Special Dates Selection]![lstMonth]
ORDER BY [Last Name];

where lstMonth is the name of the combo or list box.  Note that the parameter is now declared as SHORT, which means it's a short integer number.

How do I configure my account security information? - Microsoft Community

a. user say:

How do I configure my account security information?

How do I configure my account security information?

a. user say:

It's a good idea to add as much security info to your account as you can, because it will help make your account more secure. For example, if you forget your password, or if someone else tries to take over your account, you can use your security info to get your account back.

To get started with adding account security info, follow these steps:

1.       Sign In to your account using your Windows Live ID email and password.

2.       On your account page, under Account Security, you will see the Security Info section.

3.       Select the "Manage" link.

The Manage Security Info section allows you to configure the following security and password reset options:

  • Mobile Phone - SMS (Short Message Service) is the texting component provided through cell phone services. Configured mobile phone numbers are used to help you recover your account by receiving a text message code. You can register one or more mobile numbers.
    • Select Add New.
    • Select your location/country code.
    • Type in your mobile number.
    • Click the Save button.
  • Alternate email address - Add alternate contact email addresses for your Windows Live account. Windows Live will send any requested password resets to your configured alternate email address. You can have one or more alternate email addresses associated with your account. When a password reset is sent, it will be sent to all configured alternate email addresses.
    • Select Add New.
    • Type in the email address.
    • Click the Save button.
  • Trusted PC - A trusted device would be any computer registered as a validated system used to Sign In to Windows live Services with your Windows Live ID. Windows Live Essentials is required on any computer you want to configure as a Trusted PC.
    • Select Add New.
    • Type in your desired computer name.
    • Click the Save Button.
  • Security Question - This is a validation question. You can select from several questions and provide the answer that only you would know.
    • Select Add New.
    • Select a security question.
    • Type your answer to the security question.
    • Click the Save button.

Please note that after creating a new Windows Live account, you have 14 days to add, remove, or change password reset information without required confirmation. Once a Windows Live account is in service for more than 14 days, password reset information settings will require confirmation through existing configurations before they can be set as confirmed on the account.

Confirm your configurations: When configuring your password reset information, you will be required to confirm your settings. You can confirm your setting by following the instructions in the email sent to your alternate email address. Also, you can select "confirm" next to any password reset option to have a confirmation email sent to your configured alternate email address.

Remove password reset information: To remove any configurations, such as an email or SMS mobile number, just click delete next to the setting and following the instructions. You'll receive an email or SMS message to confirm any changes.

Have you lost access to all of your Password Reset Information?
If all of your Password Reset Information is no longer valid, you can mark it for removal. Once your information is marked for removal, you will receive an email to your alternate email address. If you configured SMS settings, you will also receive a text message to confirm the same. You can Click Here to remove all password reset information configured on your account.
