ken sheridan say: The following function returns the 'week-ending' date for any date:

Public Function WeekEnd(intEndDay As Integer, Optional varDate As Variant)

    ' Returns 'week ending date for any date
    
    ' Arguments:
    ' 1. intEndDay - weekday on which week ends, 1-7 (Sun - Sat)
    ' 2. vardate - optional date value for which week ending
    '   date to be returned.  Defaults to current date
    
    If IsMissing(varDate) Then varDate = VBA.Date
    
    If Not IsNull(varDate) Then
        WeekEnd = DateValue(varDate) - Weekday(varDate, intEndDay + 1) + 7
    End If
    
End Function

To return the week-ending date for the DateAssigned date in a query, where each week ends on a Thursday, call it with the following expression:

    WeekEnd(5,DateAssigned)

The query can then be grouped by this expression.

Outlook and calendar synchronisation between devices - Microsoft Community

paul patashnik say:

Outlook and calendar synchronisation between devices

How do I synchronise all Outlook/Hotmail movements (emails in and out and deleted, movements of emails to folders, contacts, calendar etc) between Outlook on a Dell PC, Outlook on an HP laptop and Hotmail and the calendar on an Apple 5S smartphone? I wish to set up the synchronisation between these 3 devices so that any movement on one device will update on the other two devices.

brian tillman [outlook mvp 2007-2019] say:

On the iPhone, tap Settings>Mail, Contacts, Calendar>Add Account. Choose "Outlook.com" from the list if you are still using the "old" Outlook.com or select Exchange is you are using the "new" Outlook.com (i.e., the Outlook Mail UI). Enter your Microsoft address and password, or an app password if you have two-step verification enabled. Enter a description. Tap Next. Select the remaining settings.

When you say "Outlook on a PC", are you talking about the outlook.com web site or are you talking about Office Outlook 2016 (or 2013 or 2010, or...)? If you mean the web site, you're done. The iPhone will show you what's on the web site and vice versa. You'll see the same mailbox on any PC you use if you use a web browser to connect to the outlook.com web site. Your mail, contacts, and calendar will all synchronize.

If you mean Office Outlook 2013 or 2016, open Outlook, click File>Account Settings>Account Settings>New>Next. Fill out the fields with your name, Microsoft address, and password (or app password if you have two-step verification enabled for your Microsoft account), and click Next. Outlook will configure the account and when it's done, you'll see the same data the iPhone sees. Anything you do on the phone, on the outlook.com web site, or in Outlook will sync so all will see the same information. Mail, contacts, and calendar will all synchronize.

For Office Outlook versions 2010 or 2007, you can connect to your outlook.com mailbox using IMAP. Your mail will all sync, but not contacts or calendar because IMAP is a mail-only protocol.

For Office Outlook 2003, you can connect to your outlook.com mailbox using POP. You'll see mail arriving in the Inbox, but you won't see any other folders and what you do in Outlook 2003 won't synchronize at all. You also won't see contacts or calendar from the mailbox.
