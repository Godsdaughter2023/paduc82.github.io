    robertwinz say: I got this to work. It required two work-arounds.


    First, there's a problem with Windows Update, you need to:

    1. Click the Start Button

    2. Type "services.msc" in the search/run box

    3. When the Services console opens, find Windows Update, right click and select "Stop"

    4. Open Window Explorer and browse to C:\Windows

    6. Delete the Folder "SoftwareDistribution"

    7. Go back to the Services console, right click on  Windows Update and click on Start.


    (modified from procedure found here: http://answers.microsoft.com/en-us/windows/forum/windows_7-hardware/windows-was-unable-to-get-a-list-of-devices-from/d396d3e0-f5f4-49ef-b2d1-e4bb0831a0f2?page=3&tm=1338466188671&auth=1)


    Now Windows Update will work when you add a printer, but I still couldn't add the network printer directly. But there is a second work-around for that:

    1. Add Printer (click)
    2. Choose local printer
    3. Choose use existing port: print to file
    4. Click windows update.
    5. Wait...
    6 MFC-9700 USB should now be listed, choose it.
    7. Printer will then install.

    after it's installed...

    1. Right click printer icon. Left click printer properties.
    2. Click port tab.
    3. Add new port. Local port
    4. Type in address of shared printer ie \\Name or ip address of computer with attached shared printer\Shared printer name

    (procedure found here: http://www.sevenforums.com/drivers/2726-printer-network-no-driver.html)

    Worked for me, Good Luck!
