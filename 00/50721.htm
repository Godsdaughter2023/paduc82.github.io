tryx3 say:

I urge you to leave the Built-In Admin [Default Admin] account alone despite what the previous reply said.  I think that poster misread your question.

Every Windows computer needs an Admin account to be set up [in addition to the Built-In Admin that is created by Windows].  Create an Admin account, make a "password reset disk" for it, password protect it [and write the password down somewhere].  Then, for the sake of complete assurance, repeat the process so you have two password-protected Admin accounts to choose from should you ever need to recover the system from serious problems [just in case one of these accounts has itself been corrupted].

Then you can change your son's account to be a Standard user account and he will have to ask for your assistance if he needs to install applications and so forth.  Despite this precaution, there are methods for bypassing this system protection [being limited by Standard user rights] and it might only be a matter of time before your son learns how to do this - so you could usefully consider how to reach an agreement with your son on the subject.  Good luck.
