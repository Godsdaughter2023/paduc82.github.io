Hello,

%userprofile% is a pre-defined environment variable. Each environment variable is assigned to a certain file/folder path on each computer.

A list can be found here:

https://technet.microsoft.com/en-us/library/cc749104%28v=ws.10%29.aspx

Alternately you can generate a list from your computer. Open up CMD -> Type in set -> press Enter. This will show you all the variables. If you look at the entry list the string on left is the Environment Variable and the string on the right is the file path, for example:

{Variable}={Folder path}

ProgramFiles=C:\Program Files

Each {Variable} will require '%' around it, this tells the system this an environment variable.

If you want to get a list of variables and export to notepad, use the following from a run box:

cmd /c set >1&&notepad 1

As for more information on Net Start:
