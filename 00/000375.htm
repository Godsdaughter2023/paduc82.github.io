7. Repair your Outlook data files - https://support.office.com/en-us/article/Repair...

8. Disable Anti-virus temporarily.

rich007a say:

You'll hve to create something for each control if you want something to happen....

There is away you can do it without using a class, but I don't know it and it wont be quick to master.

 

You could do this though:

Private Sub KeyPressWarning(ByVal KeyAscii As MSForms.ReturnInteger)
    Dim cb As ComboBox, i As Long
   
    If TypeName(Me.ActiveControl) = "ComboBox" Then
        Set cb = Me.ActiveControl
        For i = 0 To cb.ListCount - 1
            If Chr(KeyAscii) = cb.List(i) Then Exit Sub
   'or:     If Chr(KeyAscii) = Left(cb.List(i), 1) Then Exit Sub
        Next i
       
        MsgBox "Please use the dropdown!"
       
        cb.MatchRequired = False         'Could be set at design time
        cb.MatchEntry = fmMatchEntryNone 'Could be set at design time
       
        cb.ListIndex = -1
        cb.DropDown
    End If
End Sub

Private Sub cboAge_KeyPress(ByVal KeyAscii As MSForms.ReturnInteger)
    Call KeyPressWarning(KeyAscii)
End Sub
Private Sub cboGender_KeyPress(ByVal KeyAscii As MSForms.ReturnInteger)
    Call KeyPressWarning(KeyAscii)
End Sub
Private Sub cboOTHER_KeyPress(ByVal KeyAscii As MSForms.ReturnInteger)
    Call KeyPressWarning(KeyAscii)
End Sub
etc., etc.

 

One thing to note, I think I made a mistake about the cb.Clear method.  It removes all items from the list!  You don't want to use that.

 

An alternative to what I did here, would be to have an item in each of your lists that is says "(Please select)".  I think then you might be able to get away without all of this.  Try it and see.

You should set the cb's ListIndex to zero in the initialize event so (Please Select) shows at the start of the form.

To reset all the cb's, you can loop like this:

     'reset the fields:
    Dim cbo As Control
    For Each cbo In Me.Controls
        If TypeName(cbo) = "ComboBox" Then
            cbo.ListIndex = 0
        End If
    Next cbo
