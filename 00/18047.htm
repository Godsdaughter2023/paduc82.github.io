Access 2007 to Excel 2007 - Microsoft Community

michelleloweryjones say:

Access 2007 to Excel 2007

I have two queries I have exporting to Excel from Access and would like to know how I can get them to export into one spreadsheet separated by tabs. Right now I have a macro set up to export the two queries, then I am moving one spreadsheet to the other.

boblarson say:
I have two queries I have exporting to Excel from Access and would like to know how I can get them to export into one spreadsheet separated by tabs. Right now I have a macro set up to export the two queries, then I am moving one spreadsheet to the other.

Put this code into a standard module and then call it passing the queries to be included.

Function SendAllToXLFile(ParamArray strQueries() As Variant)
    Dim objXL As Object
    Dim xlWB As Object
    Dim xlWS As Object
    Dim rst As Object
    Dim lngCount As Long

    Set objXL = CreateObject("Excel.Application")
    objXL.Visible = True
    Set xlWB = objXL.Workbooks.Add

    For lngCount = 0 To UBound(strQueries)
        Set rst = CurrentDb.OpenRecordset(strQueries(lngCount))
        Set xlWS = xlWB.Worksheets.Add
        xlWS.Range("A1").CopyFromRecordset rst
        xlWS.Name = Left(strQueries(lngCount), 32)
        rst.Close
    Next
    objXL.UserControl = True
    Set objXL = Nothing
End Function


Then call it like this:

SendAllToXLFile "Order Summary","Inventory"

You can put as many queries in as you want for as many sheets as you want.

pats-n-kc say:

Need clarification on syncing OneNote workbooks with I-Phone and Win7 laptop.

After reviewing answers to other questions and reading the FAQ, I have come to the conclusion that:

1)  The actual workbook needs to reside on SkyDrive

2)  A laptop running MS Office 2007 or 2010 OneNote then sync's with SkyDrive

3)  The I-Phone app also syncs with SkyDrive

Currently all our pc’s in the office share Outlook calendars using Office Live/SkyDrive accounts. We also have OneNote workbooks that reside on our Windows servers that are shared with specific managers.

If we move all OneNote workbooks to a SkyDrive account, can we share them with the team via their pc’s OneNote AND their I-Phone / Win Phone?

benoit barabe say:

Almost - OneNote 2007 cannot sync with Skydrive. We made significant architecture change to enable cloud synching and that is only available in OneNote 2010.

But #1 an #3 is correct.

Currently the iPhone client only syncs with Skydrive and currently only personal notebooks - not shared one. But if they are on Skydrive, they can still be shared to others who will accessed them via PC, WP7 or the web interface. But the iPhone client itself only sees the personal notebooks associated with one Live ID.

PCs (2010) and Windows Phone 7 can sync with either Skydrive or Sharepoint 2010 - and can sync personal or shared notebooks.

Hope this helps,

Benoit Barabe, OneNote Development Manager.

Try the following: Make sure that only one document is open. Hold the Alt key and press F11 to display the Visual Basic Editor. Press Ctrl+G to activate the Immediate window. Type

Will Windows 10 run natively on a device with ARM CPU architecture - Microsoft Community

a windows 10 beta tester say:

Will Windows 10 run natively on a device with ARM CPU architecture

Original Title: ARM CPU?

Will Windows 10 run natively on a device with ARM CPU architecture (ie. Nexus 7 2013), or will there be an RT version as well?

entegy say:

There will be a version for ARM CPUs.

However, you have to remember that ARM devices do not have a unified boot structure like x86 PCs have. You can't just choose to boot another operating system. Microsoft's requirements for Windows RT have attempted to make a standard for ARM, but you still just can't install Windows on a Nexus for example.
