frenkelpartners say:

Solution for Windows 10 Store App problems - Windows 10 Store / Calculator / Photos

Hello I have been having lots of problems with the Windows 10 Store at my firm and found a solution that worked after trying so (so so) many things, that I'm hoping putting up my steps will help others.

What I found was in our firm, the Windows Store problems were isolated to a specific user. If another user was logged in, they wouldn't have any problems with the Store app or Calculator / Photos apps. Creating another user didn't have any affect on existing users or accounts already with this problem.

My Windows Version is Version 10.0 (Build 10240)

(run winver)

Problem

The Store would not do anything when clicked. They might flash a window, but that would just disappear. 

The Calculator and Photos Apps would get

This app can't open

Check the Windows Store for more info about Calculator

Go to the Store

Clicking on the store link wouldn't do anything.


If wsreset was run the window a message box

ms-windows-store:PurgeCaches

The application cannot be started. Try reinstalling the application to fix the problem.

Solution

1. Open Powershell 

(Start Button -> All Apps -> Windows Powershell -> Windows Powershell)

Open powershell AS THE USER. I found lots of other instructions telling me to run as an administrator. The issue is if your account is a standard account, to run a program "as Administrator" you need to run the app under another account, this actually can sometimes make changes to the administrator account, not the standard account.

2. Get Full Package Names of the broken Apps

type the following into the Powershell window

get-appxpackage | select name, packagefullname

Copy the right column of the Microsoft.WindowsStore and any other apps that you are having problems with.

Best to highlight with the mouse and use Ctrl+C

For me it was Microsoft.WindowsStore_11602.1.26.0_x64__8wekyb3d8bbwe

Open Notepad and paste (Ctrl+V) 

3. Reregister the apps

Copy the following line into notepad

Add-AppxPackage -register "C:\Program Files\WindowsApps\xxxxxxxx\AppxManifest.xml" -DisableDevelopmentMode

Now you want to replace the xxxxxxxx with the FullPackageName of the Windows Store.

Do the same for any other Apps that aren't working.

For me I had 

Add-AppxPackage -register "C:\Program Files\WindowsApps\Microsoft.WindowsStore_11602.1.26.0_x64__8wekyb3d8bbwe\AppxManifest.xml" -DisableDevelopmentMode

Add-AppxPackage -register "C:\Program Files\WindowsApps\Microsoft.Windows.Photos_16.1118.10000.0_x64__8wekyb3d8bbwe\AppxManifest.xml" -DisableDevelopmentMode
Add-AppxPackage -register "C:\Program Files\WindowsApps\Microsoft.WindowsCalculator_10.1604.21020.0_x64__8wekyb3d8bbwe\AppxManifest.xml" -DisableDevelopmentMode

Copy each line, paste this back into the Powershell Window and run each line separately starting with the WindowsStore line.

After I ran the windows 10 store, I was able to click the Store App and it was able to open.

Your post implies you cannot view pdf files.  Files with extension ".pdf" are opened by Adobe Reader.  If your problem is you cannot open pdf files, verify you have Adobe Reader installed.  Most web sites have links to Adobe.

 

If you know Adobe Reader is installed and when viewing a list of (or link to) pdf files do no see the Adobe icon or thumbnail, try the five steps below.

 

If you know Adobe Reader is installed but cannot open a pdf file by double clicking on the filename, try the five steps below.

 

    1. Right click on a pdf file

    2. You should see a dialog with a bunch of options

    3. Move the mouse pointer down to "Open With" and click on it

    4. There will be an option like "always open with," check that box

    5. Move the mouse pointer down to the Adobe Reader icon and click it

 

If, in step 5, you do not see Adobe Reader as an option, then Adobe Reader is not a registered application.  In this case you will have to install or reinstall it.

 

There is another possibility.  When you try to open a pdf file, if you get a message from Windows saying you do not have permission to open the file (location), you will have to contact the owner of the system.

 

Hope this helps.
