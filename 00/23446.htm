Hi,


I'm sorry you are having issues. You provided very little details so I'll try to clarify a few things about OneNote sync. Sorry if some of these are cryptic, I'm sharing in the hope that it helps you or others troubleshoot.


A few clarifications:

1) OneNote syncs file with SkyDrive directly - without the need of any other software. You cannot sync up or down .one files using any sync client other than OneNote. The protocol is meant to be very efficient and fast, and full-file sync just couldn't support the scenarios we support.

2) The SkyDrive client do not sync down the files to your machine - it creates small links instead - in order to avoid conflicts between the sync client and OneNote sync.

3) The only way to create a valid notebook on SkyDrive is to use a client - be it the web client or one of the software clients.

4) To open a notebook created on Skydrive web (OneNote web client) on a local client, you have two options: either use the File->Open functionality within the client, which should display the list of notebooks on SkyDrive, or open the notebook on the web client and click "Open in OneNote".


It appears your client is not able to authenticate with SkyDrive properly. Many things can interfere either the authentication or sync protocol we are using. Typically..

a) Software firewalls

b) Router firewalls

c) Malware or anti-malware

d) Administrator settings


Assuming you have the latest and greatest OneNote 2010 (which version/build do you have?), OneNote can use two mechanism to authenticate with the server:

- The built-in, out of the box mechanism. Works for most. Only supports 1 signed-in account, can have issues on some firewalls or machines that uses multiple accounts

- Windows Live Sign-in assistant (either installed as a standalone or through the Windows Live Essentials suite). Supports multiple accounts, deals with some firewalls better. OneNote 2010 will automatically use this component if it is installed.


OneNote 2013 does not require the sign-in assistant to support multiple account and has support for even more firewalls.


Hope some of the details above helps. If not, I would indeed recommend support, or at least provide lots of details about your experience:

- Which exact version you are using

- Do you have Windows Live Essentials or Windows Live Sign-in assistant installed

- Is this a personal or corporate system?

- Do you use any firewall (software or hardware?)

- Anything else interesting about your network?


Hope this helps,
