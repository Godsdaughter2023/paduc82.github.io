douglasnigelb say:

To seeing picture images in the thumbnails you need to check folder options and set how files appear in a folder.

Method 1 - Do this:

From Windows Desktop, Select Start Menu, then select Control Panel, then select Folder Options.

Click on the View Tab, and under Advanced Settings make sure the box next to Always show Icons, never Thumbnails is unticked (blank). If it is ticked, then uncheck it and Apply the changes.

Method 2 - Do this:

From the Windows Desktop, Start Windows Explorer and then, from a folder that contains samples of your camera's picture images, locate the Views button on Explorer's toolbar. (The views button is the third button in the upper right-hand corner of the toolbar.) Click this button several times and the file icons will all simultaneously change size.

Only when Folder Views is set to List or Details do you not see the picture icons. So it follows that if you can press the views button three times only, in a folder containing your picture files, then you should start seeing the picture thumbnails appear in the icons.

This behaviour works if the picture format are .JPEG files, as Microsoft Windows 7 supports these file formats by default.

However if you have your camera set to shoot pictures in uncompressed RAW and you upload these to the computer, then you still will not see the Thumbnails as Windows does not have installed the required codec by default.

The codec then is for the RAW file format supported by camera manufacturer, there is not a camera-model-specific codec, this might also be why you are also experiencing difficulties in finding the right codec. 

Method 2 - Do this:

Navigate to http://windows.microsoft.com/en-gb/windows7/how-do-i-view-raw-pictures and Download the Microsoft Camera Codec Pack. There is only one Codec pack for all available RAW formats. Canon Camera Raw (.CR2) is included in the pack. When download is complete, open the download folder and double-click on the Codec Installer. 

You will need to restart the computer for the changes to take effect. 

I hope this resolves your question.
