"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

I would like to inform you that the Bluetooth service will not be running when the computer is in sleep mode and if you are facing this issue also when you try to reconnect, I suggest you to uninstall and reinstall the Bluetooth drivers and check if it helps.

1. Press “Windows key + X” and click on “Device Manager”.

2. In the list of devices, you should see the Bluetooth Drivers.

4. In the uninstall dialog, make sure you select the option to delete the driver package from the system. This causes the driver package installed earlier to be deleted.
