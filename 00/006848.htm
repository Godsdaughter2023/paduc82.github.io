I had a chance to look into this a little more and here is what I ended up with:

I was correct about assigning the name directly in EffectInformation. However, this assumes that the name of the sound effect will come up on the sound effect list. You can't assign an any other kind of sound effect this way, so I was wrong in that respect. You can use that method for ActionSettings, but not for animations it appears.

Sub getsound()

Dim sound As Shape, partneranim As Effect

 

Set sound = ActivePresentation.Slides(1).Shapes("bag.wav")

Set partneranim = ActivePresentation.Slides(1).TimeLine.MainSequence(effectforaudiotoplaywith)

 

With ActivePresentation.Slides(1).TimeLine.MainSequence.AddEffect(sound, msoAnimEffectMediaPlay)

   .Timing.TriggerType = msoAnimTriggerWithPrevious

   .MoveAfter partneranim

End With

 

End Sub

Hi,

Try this:-

=SUMPRODUCT((MONTH(N3:N27)=1)*(N3:N27<>"")*(O3:O27="Yes"))

Note the 1 in bold represents January so can be changed to a 2 for February etc and likewise the Yes at the end can be changed to No.

amit tandon say:

Try below:

=SUMPRODUCT((O3:O27="YES")*(MONTH(N3:N27)=MONTH(P3)))

This refers to the month number of the Date in cell P3. It will return the number of dates in column N whose month is the same as the month of the date in cell P3.

If you dont want a reference to a date for a month, use below for "Jan". Repace "Jan" with "Feb" etc.

=SUMPRODUCT((O3:O27="YES")*(TEXT(N3:N27,"mmm")="Jan"))
