kevin510 say: None of that helps, here's what the solution is for other people using DLink routers (i personally have the DGL-4500).

For whatever reason in windows 7 the router will drop the VPN every 20 seconds.  I initially thought that maybe it was because of the lesser known port IP 47 which most people forget about.  So i created a rule for IP port 47 under advanced, virtual server.  I set it to 'other' protocol and typed in the 47 and pointed that to my local machine.

The other thing i did was in the firewall settings, i unchecked at the bottom where it says PPTS and IPSec so those check boxes are now unchecked.  YOu would expect that those being checked as they are by default would be the way to go but apparently not.
