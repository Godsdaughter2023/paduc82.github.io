do I need to keep old backup sets when I generate a new backup - Microsoft Community

charlesadams1315 say:

do I need to keep old backup sets when I generate a new backup set?

I have vista home premium.  I regularly backup my computer.  Each time I backup I now have a new set of disks.  Can I get rid of the old set when the new set is generated.?

a. user say:

No.

Because after the first initial files backup, all subsequent backups are incremental, meaning the system only back up new files and files that have changed since the last backup.

You need to keep the whole bunch.
However, eventually you will be asked to do another complete backup. Then, you can delete the old backups, but only after you have a new complete backup.
