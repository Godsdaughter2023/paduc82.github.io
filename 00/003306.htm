suzanne s. barnhill say:

You don't need nine different styles, just the number of levels you need. And you can start with Word's built-in styles--either headings or the List Number series. The primary caveat is that you need a different list template for each separate list.

I find that I can usually get away with using Restart Numbering or Continue Numbering for simple lists; for outline-numbered lists, you're better advised to use the list itself to force a restart; for this purpose you can use an unnumbered style at Level 1 (some style that will always appear between the paragraphs where Level 2 must restart at 1). See http://word.mvps.org/FAQs/Numbering/ListRestartMethods.htm.

How can I change the path of all linked documents by VBA ? - Microsoft Community

willykano say:

How can I change the path of all linked documents by VBA ?

How can I change the path of all linked documents (in the header , for example) by VBA for all the documents residing in one directory? (after implementation of a server with a different name, for example : old server name\\XXXX1, new server name (\\XXXX6)

doug robbins - mvp office apps & services (word) say:

The following will update the links in a single document:

' Macro created 26/10/01 by Doug Robbins to update links in a document
'
Dim alink As Field, linktype As Range, linkfile As Range
Dim linklocation As Range, i As Integer, j As Integer, linkcode As Range
Dim Message, Title, Default, Newfile
Dim counter As Integer

counter = 0
For Each alink In ActiveDocument.Fields
    If alink.Type = wdFieldLink Then
        Set linkcode = alink.Code
        i = InStr(linkcode, Chr(34))
        Set linktype = alink.Code
        linktype.End = linktype.Start + i
        j = InStr(Mid(linkcode, i + 1), Chr(34))
        Set linklocation = alink.Code
        linklocation.Start = linklocation.Start + i + j - 1
        If counter = 0 Then
            Set linkfile = alink.Code
            linkfile.End = linkfile.Start + i + j - 1
            linkfile.Start = linkfile.Start + i
            Message = "Enter the modified path and filename following this Format " & linkfile
            Title = "Update Link"
            Default = linkfile
            Newfile = InputBox(Message, Title, Default)
        End If
        linkcode.Text = linktype & Newfile & linklocation
        counter = counter + 1
    End If
Next alink

You could modify the macro in the article "Find & ReplaceAll on a batch of documents in the same folder” at:

http://www.word.mvps.org/FAQs/MacrosVBA/BatchFR.htm

so that it incorporated the above code to modify the links in all of the files in a folder.

How do you change the unit of measure from inches to pixels in - Microsoft Community

jonathankalkwarf say:

How do you change the unit of measure from inches to pixels in PowerPoint?

How do you change the unit of measure from inches to pixels in PowerPoint?
john korchok say:

To enter a dimension in pixels, enter the number with px at the end: 300px.

You can set dimensions this way, but not text sizes. This works on the Mac, and on older versions of Windows PowerPoint, but no longer works in PowerPoint 2016 for Windows. This is probably a good thing.

PowerPoint uses physical measurements internally. When you enter pixels, it looks up the current video resolution of the computer and converts to the physical size that corresponds. So you'll get different measurements on different computers, depending on what their display is set to.

If your heart is set on pixel-based measuring, you can vote on this suggestion in the PowerPoint User Voice forum: Change units from inches to pixels. Microsoft uses these votes to prioritize their programming.

Hi mrdanar,

 

Let me assist you with your concern regarding how to setup email rules on your email account.

 

If you wish to forward all your incoming emails into another email address, please refer to the steps below:

 

1. Sign-in to your Microsoft account.

2. Click the Gear icon then select More mail settings.

3. Under Customizing Outlook, click Rules for sorting new messages.

4. Click on the New button.

5. Under Step 1: Which messages do you want this rule to apply to,

a. First drop-down menu - choose To or Cc line

b. Second drop-down menu - choose contains
c. Third field - input your email account

6. Under Step 2: What action do you want to apply, click on Forward to then enter your preferred email address

7. Click Save button.
