If you're trying to apply a character style and also Bold, you must apply the character style first. jay freedman say:
In reply to deleted message

The English thesaurus in Office 2010 consists of the data file MSTH7EN.LEX and the matching code file MSTH7EN.DLL. (There are similar pairs for French and Spanish.) In Office 2013 and 2016 the equivalent files' names contain the number 8 instead of 7.

I don't believe these files are "hot-swappable", although it may be possible if you changed the the names after copying them into the appropriate folder. Just to further complicate matters, the .LEX file (which is binary, not text) contains the name of the .DLL, so you would have to use a hex editor to change that reference. If that works, I have no assurance that the newer thesaurus contains the synonyms you're looking for.

As best I can tell, there's no way to specify a third-party thesaurus to replace the Office one in situ. If you can find one that runs as a stand-alone program and takes a request on its command line, you could add a macro to Word to pass the selection to that program, using the Shell command. You probably won't get the function of dropping the synonym into the document to replace the original word, but would have to rely on copy/paste.

You may also want to look into some of the programs mentioned in https://www.howtogeek.com/131792/the-best-free-dictionary-and-thesaurus-programs-and-websites/.
