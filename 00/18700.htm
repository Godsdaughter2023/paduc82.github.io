a. user say:

Re:  Automating Excel

All objects must be set to nothing ( in the proper order - child then parent)
That includes the Excel application object.
Also, if the Application was created then it must be closed (quit) before the object reference is set to nothing.
All of this prevents "orphan" objects being left behind and creating the problems you describe.

In addition, PP does not know anything about xlConstants, so you should use their numeric value instead of the constant name.

'---

Dim bNewApp As Boolean
Dim xlApp As Object
Dim InputBook As Object

On Error Resume Next
Set xlApp = GetObject(, "Excel.Application")
If Err.Number > 0 Then
  On Error GoTo 0
  Set xlApp = CreateObject("Excel.Application")
  bNewApp = True
  xlApp.Visible = True
End If
On Error GoTo 0

Set InputBook = xlApp.Workbooks.Open(Filename:=InputFolderPath & InputFiles(FNum), _
                         UpdateLinks:=2, ReadOnly:=True)

'Processing here


InputBook.Close savechanges:=False

'Set all objects to nothing
Set InputBook = Nothing
If bNewApp Then
  xlApp.Quit
End If
Set xlApp = Nothing

dale howard [mvp] say:

salam.elias --

 

Your mistake is on the My Project task.  You should not use a regular task to show the name of the project.  Instead, you should display the Project Summary Task (Row 0 or Task 0) for this purpose.  Here is what you need to do to resolve your problem:

  1. Select all tasks from Phase 1 through the last subtask in Phase 4.
  2. In the Schedule section of the Task ribbon, click the Outdent Task button.
  3. Right-click on the My Project task and select the Delete Task item on the shortcut menu.
  4. Click the Format tab to display the Format ribbon.
  5. In the Show/Hide section of the Format ribbon, select the Project Summary task checkbox.

After completing the above steps, your outline numbers will be what you expect.  Hope this helps.

Tones that came with the phone are usually copyrighted and not transferable to another phone. The only tones you can transfer are tones you installed yourself.

cjlim say:

The 8250 did not come with many connectivity options. You can only send media files via IR to another phone device with IR or send it by mms. To do this, goto Menu>Gallery>Tones/music>select the tone you wish to transfer>Options>send>via infrared/via mms. You should be able to send all media files this way.
