hal hostetler - mvp-outlook say:
A scanned document is usually a .TIFF image file which is useless to a word processor.  You need an Optical Character Recognition (OCR) application to scan the image file and recover the text information from it. If Word is part of Office 2003 or Office 2007, you can optionally install Microsoft Office Document Imaging which includes OCR. To the best of my knowledge, Microsoft Works doesn't include OCR; a third party application will be required.  Here's an example of one:

SimpleOCR - freeware OCR software and royalty free OCR engine!
