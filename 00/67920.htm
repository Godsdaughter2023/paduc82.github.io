There are various methods to reinstall Vista.

 

There is no Windows Vista download available from Microsoft.

You can contact your Computer Manufacturer and ask them to send you a set of Recovery Disks.

dadsox say:

Hello Vivid. I am seeing that you are having trouble with the family timer on the account.  There is a very easy way to reset your family settings. Here is how you do it:

1) Turn on your XBOX 360 and go to the "Settings" tab on the dashboard.

2) Click on the "Family" option

3) Enter the pass code that it asks for. This was set when the family settings were originally placed.

4) Then you can turn off the family settings. Save and exit after you complete this step. This will eliminate the family timer altogether.

Hopefully, this will resolve your problem. I will follow up with this post to make sure you resolved this problem. Have a great night.

If you do not know the pass code, you can reformat your hard drive to restore it back to factory settings.  This is the only way I have found to make that work. I would suggest you back up any game saves or profiles onto a USB drive or the Cloud Saves option. Then follow these steps:

3) Press the Y button on your controller and select format.

marilyn o say: Hello barbara32ca,

Thank you for posting to the Microsoft Answers Forum.

If you have Windows 7 Professional or Ultimate, you should be able to use Picture IT in XP Mode. XP mode does not work in the Home versions. You can try 3rd party programs that have virtualization to run on the Home version.

Running Windows XP Mode requires:
 
1. Windows 7 Professional, Enterprise, or Ultimate editions.
2. A computer capable of hardware virtualization. This means your computer has a central processing unit (CPU) with either Intel-VT or AMD-V virtualization features.
3. Virtualization features turned on in your computer’s basic input/output system (BIOS).
 
To determine whether your computer’s CPU is capable of hardware virtualization, go to the Windows XP Mode and Windows Virtual PC support website and download a utility that can check whether your Intel or AMD CPU is compatible.
 
http://www.microsoft.com/windows/virtual-pc/support/configure-bios.aspx
 
To find your brand of processor so you can download the appropriate utility, click the Start button , right-click Computer, and then click Properties. Under System, next to Processor, you can see what type of CPU you have.
 
Even if your CPU can run Windows XP Mode, you might also need to turn on virtualization settings in your computer’s BIOS. Procedures vary depending on the BIOS manufacturer. For more information, see BIOS: frequently asked questions.

http://windows.microsoft.com/en-us/windows7/BIOS-frequently-asked-questions
 
For more information, please logon to:
http://windows.microsoft.com/en-us/windows7/install-and-use-windows-xp-mode-in-windows-7#section_3

Thanks,
