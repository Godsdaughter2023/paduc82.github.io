rickcp say:
Trojan:DOS/Alureon.A is only 'partially' removed by MSE.

DOS/Alureon is the detection for a variant of the Win32/Alureon malware family of data-stealing trojans that infects the Master Boot Record (MBR). It may also infect and corrupt certain driver files, causing them to become unusable. Recovering from this situation may require measures beyond removing the trojan itself from the computer.

Please note that Microsoft antimalware products don’t automatically repair the MBR when they clean this Trojan, so you would need to take further action manually. You will need to run the following commands using the "bootrec.exe" tool to ensure a complete repair of your computer:
- bootrec /fixmbr
- bootrec /fixboot
- bootrec /rebuildbcd
For more details on these commands, please refer to MS's KB927392, with specific focus to the options: "/fixmbr", "/fixboot" and "/rebuildbcd".

You should also be aware of the fact that it's a known issue for many offline scanners (including WDO) that can not 'automatically' repair the MBR (but only detect and partially remove threats) so this repairing needs to be done 'manually'.
