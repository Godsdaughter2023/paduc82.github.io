With first name in ColA and second name in ColB of Sheet2; the below formula applied in Sheet1 will retrive the information from Sheet2 ColC based on the firstname in Sheet1 cell A2 and last name in Sheet1 cell B2.

=INDEX(Sheet2!$C$1:$C$100,SUMPRODUCT((Sheet2!$A$1:$A$100=$A2)*(Sheet2!$B$1:$B$100=$B2),ROW($A$1:$A$100)))

dave peterson say:

I would do a little pre-work(??) first.

In the table worksheet, I'd insert a new column to the far left.  This new
column (column A??) would be the concatenation of the two fields I need to
match:

=B1&"|"&C1
(where | is a unique character that isn't used in either of the first or last
name fields)

Then I could use a formula like:

=vlookup(x9&"|"&z9,sheet2!a:e,5,false)

or even =index(match())
=index(sheet2!e:e,match(x9&"|"&z9,sheet2!a:a,0))

I would think these formulas would calculate/recalculate much faster than any
alternative.

But if you can't add that extra helper column...
(saved from a previous post)

There are other ways, too.

If you want exact matches for just two columns (and return a value from a
third), you could use:

=index(othersheet!$c$1:$c$100,
   match(1,(a2=othersheet!$a$1:$a$100)
          *(b2=othersheet!$b$1:$b$100),0))

(all in one cell)

This is an array formula.  Hit ctrl-shift-enter instead of enter.  If you do it
correctly, excel will wrap curly brackets {} around your formula.  (don't type
them yourself.)

Adjust the range to match--but you can only use the whole column in xl2007.

This returns the value in othersheet column C when column A and B (of
othersheet) match A2 and B2 of the sheet with the formula.

And you can add more conditions by just adding more stuff to that product
portion of the formula:

=index(othersheet!$d$1:$d$100,
   match(1,(a2=othersheet!$a$1:$a$100)
          *(b2=othersheet!$b$1:$b$100)
          *(c2=othersheet!$c$1:$c$100),0))

============
If there is only one match and you're bringing back a number (or 0 if there is
no match for all the criteria), you can use:

=sumproduct(--(othersheet!a1:a10=a1),
            --(othersheet!b1:b10=b1),
              (othersheet!c1:c10))

are the numbers like 92.00 really 92.00 or on the formula bar are they 92?

If the second case, try this:

1. Select all the cells containing numbers you care about, say A1:C100

2. Choose Home, click Decrease Decimals, (twice)

3.  With the cells still selected press Ctrl+H

4. In the Find what box enter . (period) and then click Find All

5. In the find window at the bottom of the dialog box select all of the entries, then click Close

6. Choose Home, Number Format, Currency.
