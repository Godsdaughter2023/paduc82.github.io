There are two ways to use the Freeform shape tool; you're using the wrong one. If you drag the mouse, yes, you get a very wobbly line. But if you just click the mouse, then move it without dragging, you will get a straight line to the next point where you click. Double-click to end the shape. You'll have one node for each click. You can use the Edit Points tool to fine-tune the shape, adding and removing points as needed.

There are dozens of threads in this forum with essentially the same title as this one, and they all point to a known issue where Windows Defender doesn't get properly turned off in its registry key when a third-party AV app is installed. In the current version of Windows 10, Windows Defender should be completely disabled when any third-party AV app is installed – which means that if you have McAfee installed, you shouldn’t be seeing the Defender UI at all when you try to start it – you should just see this:

But if Windows Defender was in the commonly seen error state, where it isn’t properly turned off in its registry key, then you’d be seeing the Defender UI – but with the specific message that the service has stopped and needs to be restarted, as shown here:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_updating/definition-update-for-windows-defender-kb2267602/ea933090-9301-4c27-b915-1d3ba92b301f

So if you’re seeing the Windows Defender UI with a message that real-time protection is turned off, then this would mean that Defender is in some kind of unknown error state.

If you want to use Windows Defender exclusively, then you should uninstall McAfee as described above. But if you’d rather keep McAfee, and resolve the definition update issue with Defender, then you should uninstall McAfee (as above) and then reinstall it, and see if that resolves the issue. Note that if you choose to keep McAfee, you’ll soon be able to run Windows Defender (minus real-time protection) right alongside of it, because Windows Defender will have a new operational state known as Limited Periodic Scanning as soon as the Windows 10 Anniversary Update arrives (deployment of the Anniversary Update starts today):

http://answers.microsoft.com/en-us/protect/wiki/protect_defender-protect_scanning/limited-periodic-scanning-in-windows-10-to-provide/3dcfc8dd-36b4-43f2-9a0a-732e43191cf3

How do I change excel comments to a box shape?

a. user say:

Hi,

You may change the shape of the comment box by following the steps below:

  1. Click the File tab, then click Options.
  2. Click the Quick Access Toolbar category.
  3. In the Choose commands from box, click All Commands.
    • In the list, select Edit Shape, click Add, and then click OK.
    • This adds the Edit Shape button to the Quick Access Toolbar, which is located in the upper-left corner of the Excel window.
  4. In the worksheet, select the cell that contains the comment for which you want to change the shape.
  5. To display the comment on the worksheet, on the Review tab, in the Comments group, click Show/Hide Comment.
  6. Click the border of the comment.
  7. On the Quick Access Toolbar, click Edit Shape, and then click Change Shape.
  8. Click the shape that you want to use.
