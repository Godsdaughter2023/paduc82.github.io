The normal way to remove an unwanted wireless connection in XP is:

1. If you were using Windows (Wireless Zero Configuration) to connect, right-click the Wireless Network Connection icon in the Notification Area (bottom right of screen), then click View Available Wireless Networks.                         net start wuauserv

                        net start cryptSvc

                        net start bits

                        net start msiserver

Exit the command prompt then restart your computer. Try starting setup again.
