lemp say:

Sometimes having the latest and greatest technology can cause problems if you're not careful.  Your Netgear Nighthawk is capable of a great many different configuration options.  The first thing I would do is check the network configuration settings on the printer and both computers.

On the printer: 

Make a note of the printer's IP Address, Default Gateway, and SSID (Network name).

Then do the following on both your desktop and your laptop; do it while connected via 2.4 GHz and via 5.0 GHz.  You'll end up with 4 sets of information.

  • Open a Command Prompt window (https://www.lifewire.com/how-to-open-command-prompt-2618089), type the following, and press Enter
         ipconfig
  • Make a note of the IPv4 Address and the Default Gateway

With each of your computers connected via 5.0GHz, determine the SSID to which you are connected.  Hover your mouse over the wifi connection icon next to the clock display or seehttp://support.brother.com/g/b/faqend.aspx?c=sg&lang=en&prod=p750weas&faqid=faqp00100049_000#7

At this point, you will have 5 sets of IP address information:

  1. printer
  2. desktop 2.4 GHz
  3. desktop 5.0 GHz
  4. laptop 2.4 GHz
  5. laptop 5.0 GHz
  • All of the Default Gateways should be the same (probably 192.168.1.1).  Are they?

  • All of the IP addresses should be on the same subnet, which in this case means that the first 3 "octets" of the address are identical.  For your Netgear, this means that the IP addresses should be 192.168.1.xxx.  Are they?

  • Compare the SSID information from the two computers when they are connected via 5.0 GHz.  Are they the same?

andre da costa say:
Press the Windows key + R key to launch the Run dialog box.
In the Run dialog box enter the following: %localappdata%\Microsoft\OneDrive\onedrive.exe /reset
Click OK. The OneDrive system tray icon on the desktop should disappear and re-appear after a minute or two.
If the OneDrive system tray icon does not re-appear after a few minutes, do the following:

In the Run dialog box enter:

%localappdata%\Microsoft\OneDrive\onedrive.exe

After resetting, right click on OneDrive icon present on the System Tray and click on Settings, then click on Choose Folder and make sure that all the folder which you want to sync must be checked. We recommend you to make a backup of your data before you reset, as after the reset you might have to re-download your files.
