Duratuion variance in MS Project 2007 - Microsoft Community

huguettejean-francois say:

Duratuion variance in MS Project 2007

Why MS Project 2007 is not showing whole numbers for duration variance? Why MS Project 2007 is not showing whole numbers for duration variance?
dale howard [mvp] say:

Huguette --

Duration Variance is calculated by MS Project using the formula Duration - Baseline Duration. By default, the software does not display whole numbers in any of these three columns. This is because the software measures all Work and Duration values in minutes, and then converts the values to Hours in Work-related fields, and Days in Duration-related fields.  If a task is scheduled to finish at 3:00 PM, this means you will see a Duration value such as 5.75 days.  Hope this helps.
