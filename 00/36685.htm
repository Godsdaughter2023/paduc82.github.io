farzadsadr say:

Hi guys,

I had a similar problem with English Windows suddenly becoming Arabic.

Here's a quick fix which doesn't require creation of installation media or reinstall of Windows:

  1. Windows 10 has a utility called "LPKSetup" which manages the installation and configuration of language packs within Windows. Run LPKSetup from a command line.
  2. You now see a list of installed language packs. in your case, I guess it's only Arabic.
  3. Uninstall the Arabic language pack and confirm to restart. This step removes all language packs from your Single-Language Windows and then  it decides to switch to the default language which is English/US!
  4. All done. Enjoy your English version of Windows 10!

Hope this information is helpful and save you some time in correcting the language settings.

Cheers!

It sounds like one of two things are happening.  Either:
1 - The picture file is of a type that isn't associated with a program (or has no extension), so it asks you to choose a program to open it.  This could be fixed by renaming the file to be .jpeg.  This sounds most likely since the file isn't recognized until you save it from Paint.  To fix this, just click the file, press F2 to rename it, and add .jpg to the end of the name.
or
