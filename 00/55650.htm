Firmware Update

  1. Download firmware at www.Zyxel.com in a bin file.  File names are usually the model name followed by the .bin extension.
  2. Plug the computer into a LAN port via an ethernet cable.
  3. Open a web browser.
  4. Log into the router by entering the default gateway, 192.168.1.1 into the address bar of the internet browser. 
  5. The router will then prompt you to enter the username and password. There is no username and the default password is 1234.
  6. Click SITE MAP in the top right corner of the menu.
  7. Click Firmware under the Maintenance header.
  8. Click the browse button and select the file that you downloaded earlier.
  9. Click the Upload button.

 

 

Zyxel P-660 Series (Dark Firmware)

Login Information

Gateway - 192.168.1.1

Username - N/A

Password - 1234

How to Find Firmware Version

  1. Log into the router by entering the default gateway, 192.168.1.1 into the address bar of the internet browser. 
  2. The router will then prompt you to enter the username and password. There is no username and the default password is 1234.
  3. From the menu on the left select Maintenance.
  4. From within the Maintenance menu select Tools.
  5. From the tabs at the top select Firmware.
  6. The Current Firmware Version will be listed after Current Firmware Version.

 

Firmware Update

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Locate the devices firmware at www.zyxel.com.
  4. Download and save this file to your computer.
  5. Log into the router by entering the default gateway, 192.168.1.1 into the address bar of the internet browser. 
  6. The router will then prompt you to enter the username and password. There is no username and the default password is 1234.
  7. From the menu on the left select Maintenance.
  8. From within the Maintenance menu select Tools.
  9. From the tabs at the top select Firmware.
  10. Click the browse button.
  11. Using the file browser that appears select the firmware file that was downloaded.
  12. Click the Upload button.
  13. Wait aprox. two minutes as the device restarts and upgrades.
