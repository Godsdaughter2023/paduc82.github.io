the third argument is the column from the lookup range/array to return the value from when a match is found in the leftmost column.  The leftmost column is column 1.  If you wanted to return the value from the third column in the lookup range/array, you would put in a 3 for the third argument

the fourth argument is the type of lookup to be done.  A false says to find an exact match and the sort order is not important.  A true says to find an approximate match and the data must be sorted on the lookup range/array column 1. 

If the dialog is showing you the correct answer, then I am hard pressed to say why is doesn't work when you click OK.

Heading, not header. They're 2 different things.

You appear to have put your heading text into a text box. This removes it from the normal text flow and makes it invisible to the cross-reference tool.
