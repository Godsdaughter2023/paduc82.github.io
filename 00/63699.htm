Stolen Laptop License Re-Use - Microsoft Community

ravi_desai say:

Stolen Laptop License Re-Use

I purchased a Laptop from Dell which has Windows 7 on it. It was stolen and has not been recovered. Can I use the same Windows 7 license to upgade my other PC from Vista to Windows 7?
trekdozer say:

If the laptop came with Windows 7 it is OEM and can not be used on a different computer.

 

Can You Move Windows 7 To New Computer

If it's a retail license (Full or Upgrade) - yes. You can move it to a different computer as long as it's only installed on one computer at a time (and if it's a Windows 7 Upgrade version the new computer must have it's own qualifying XP/Vista license). The previous Windows 7 installation on the old computer must be formatted/deleted. You might have to call Microsoft and explain what happened to complete the activation. Activating it on the second computer will automatically in effect "deactivate" the license for the first computer. The key will work with both 32 and 64 bit, but only one can be installed at a time.

If it's an OEM license - no. OEM licenses, including Windows preinstalled on a computer before purchase, are tied to the first computer they are installed on and can not be transferred to a different computer. To install Windows on a different computer you will need to buy another copy.

a. user say:

Is it possible that Windows signed you in to a temporary profile?

A temporary profile is used when your regular user account has become damaged to the point where it's no longer usable. When that happens, Windows signs you in to a temporary profile, to give you an opportunity to create a replacement user account and then move your personal files to the new account.

The temporary profile is, well, temporary, so you are not able to save files. They would only be deleted once you have created a new user account.

To see if you are signed in to a temporary profile, attempt to sign in to your regular user account with your username and password.

The amount of storage you have in the cloud has nothing to do with the amount of storage you have on your flash drive - they're two separate things.
