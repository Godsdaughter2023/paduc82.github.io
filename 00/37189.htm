If it looks the same (empty), your system is now or has been afflicted with malicious software and whatever you are using for malicious software protection has failed you.

Describe your current antivirus and anti malware situation:  McAfee, Symantec, Norton, Spybot, AVG, Avira!, MSE, Panda, Trend Micro, CA, Defender, ZoneAlarm, PC Tools, Comodo, etc.

You can fix the symptom of the problem by doing this:

It sounds like your jscript.dll file has somehow become unregistered, so you can reregister it.

There are probably a couple other things that don't work that you don't know about yet, so fix everything at once.

Reregistering these DLLs is okay even if they do not need reregistering.  You can also reregister the DLLs before the scans (listed below) which may fix the symptom, but the scans are still recommended.

Click Start, Run and in the box enter the following commands one at a time.  Click the OK button to execute each command one at a time.

regsvr32  jscript.dll
regsvr32  vbscript.dll
regsvr32  /i  mshtml.dll   (that's a slash eye)

The last command for mshtml will load the DLL but may also contain a harmless warning message depending on your environment.

Now take a look at your System Restore and see if you see the calendar.

System Restore is a frequent target for malicious software attacks.  If SR was working before and is now afflicted, run the scans below and then resolve any remaining issues.

Keep in mind that System Restore is not a time machine.

You can fix the symptom now and scan for malicious software later, or you can scan for malicious software now and fix the leftovers later.  I would not recommend just fixing the symptom of the empty page and forget to do the scans.  Fixing the symptom of the problem and fixing the problem are two different things.
