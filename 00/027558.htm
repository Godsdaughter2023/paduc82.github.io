julie fka profjulie say: It has been a long, long time since I've seen anyone post this error, but I remembered seeing it so I looked over some notes I'd made several years ago and, quoting from my notes, this is what I found that worked for other users who encountered the problem:

"The problem is usually caused by "mismatched" or outdated RAPI.DLL and CEUTILS.DLL files, which are located in \windows\system32 subdirectory. You can confirm if this is a problem for you by reviewing the WCESETUP.LOG file found in \documents and settings\{your name}\Recent subdirectory (or you can search for this file). This log file is created when you install ActiveSync, so if there was any problem during install, it will be documented here.

To resolve the problem, you should manually delete the RAPI.DLL and CEUTILS.DLL files and then reinstall ActiveSync."

emilygreen---01 say:

Hi stripegurl

 

Most of the time you can put password on your memory card by opening the File Mgr. application, select Memory card and open it, select Options > Memory card password > Set.

 

The password can be up to the 8 characters long and it is a case sensitive. The password is stored in your device, thus you do not need to enter it again while you use the memory card in the same device. If you use the memory card in another device, the password is requested.

 

To remove or change the memory card password, select Memory card > Options > Open > Options > Remove password or Change password and enter the current password.
