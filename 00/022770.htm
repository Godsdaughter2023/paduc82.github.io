One possible reason that you are seeing the error message: “There is a problem with this website’s security certificate” is if your computer’s clock is set so that the date/time is later than the expiry date of the web server’s SSL certificate. We can install the security certificate of this website so that you will no longer be prompted by this certificate warning again in the future. Just follow the steps below on how to do so:

 

1. Right-click Internet Explorer and choose Run as administrator.

2. Go to outlook.com and choose the option to Continue to this website (not recommended).

3. Click where is says Certificate error in the address bar, then select View certificates.

4. Click Install Certificate and select Next.

5. Choose Place all certificates in the following store option.

6. Click on Browse….

7. Select Trusted Root Certification Authorities, then OK.

8. When prompted with the security warning, click Yes.

9. Click OK on the The import was successful message, then OK again on the Certificate box.

 

Once these steps are done on all of your devices, you should no longer encounter the same error message again in the future, regardless if you are using Google Chrome or Internet Explorer.

When it saves, I need it to save in the normal way, but also to make a copy of itself in a folder on the network that only senior management can access.
Ask your administrator to add a directory to the server that is WRITE ONLY for every body.

Any user needs the write access rights for that folder. Then use a path like "\\server\directory\file" instead of "C:\directory\file" and make a copy of the file as usual, e.g. FileSystemObject.CopyFile
