how to change network security key wifi - Microsoft Community

saurabhdalal say:

how to change network security key wifi

Hi, I am using windows 7, i was trying to change network security key for my wifi connection, please suggest.
burrwalnut say:

Hi, I am using windows 7, i was trying to change network security key for my wifi connection, please suggest.

If your router appears in the Network folder, right-click it and choose View device webpage.

 

If it doesn’t appear, type 192.168.x.y into a browser’s Address bar. The values of x and y depend upon the make of router. If you don’t know the values, open a Run window (Windows Logo key+R), type cmd /k ipconfig (note the two spaces), press Enter, make a note of the Default Gateway address and type it into the browser’s Address bar.

 

2. If asked for the User Name/Password, type the default values, which are often admin and admin. If they are not known, look in the router’s user manual or Google for them.

 

3. Look for a section named wireless or similar wording and find an SSID (Network Name) and a Wireless Key. Change the Key accordingly, then save any changes that are made and exit.
