Carry out the following steps:

1. Right-click on the worksheet which you wish to copy to another workbook

2. Select "Move or Copy"

3. In the box under "To book:"  select the other workbook to which you wish to copy the worksheet in.

4. Under the "Before sheet:", you can select the worksheet, before which , this worksheet should be copied

4. Click OK

 

However, you must note that this method will definitely copy your worksheet to the other workbook, but the formulas in that worksheet will still refer to the old workbook.

In case if you wish that even the formulas (just in case if your design is identical) should refer to the new workbook, then there is another method.

Here in this case,

1. you need to replace all the formulas of your worksheet-to-be-copied as text

(you need to replace =A1+B1 type of formula with ~=A1+B1)

2. After the copying is done to the new workbook, replace all formula text  (~=.......)  with =.... by replacing ~= with =

Hope this Helps.

Unless you are using an older version of the Mac OS (10.6.8 or older), the iSync method is not entirely compatible.

 

For photos and video, you might do better to use the Mass Storage USB option. In that case you need to locate the DCIM folder on your N8's SD card (or wherever it's located in your case). This you can copy to the Mac's Desktop and then move the photos into iPhoto. iPhoto will let you know if any of the photos are duplicates of those already transferred.

 

Since the N8's Music Player program will find any music files on the phone, you can move any music you wish to transfer to the phone to the Music folder on your SD card. Remember to refresh your music library on the phone so that the phone discovers the additions.

 

Unfortunately, to sync calendar and contacts, you need to use iSync AND you need to have iCloud deactivated. This way, you can still use Mac OS 10.7 or perhaps even 10.8  (10.9?).

 

Another, rather complicated method involves the use of the Symbian app GoogaSync to sync with Google calendar, which is in turn synced to the Mac's calendar program.  I use a Mac with OS 10.9 and I don't bother syncing the calendar or contacts, as I don't keep them current on the Mac. But I do make sure that I do keep my phone backed up (via the phone's Files app) to the SD card.

 

If you have the funds, you could also use Parallels or VMware Fusion to run Windows in a virtual machine, on which you install Nokia Suite.......
