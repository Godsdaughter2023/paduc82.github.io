greg carmack - mvp 2010-2020 say:
Hi Crystal. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

There is really no way to know for sure. The rule of thumb I've always used on Amazon is mainline Amazon is totally safe but other sellers should give pause. You should be able to activate Home OEM Product Key on your Home install as it's not full-retail-specific.

As to transfer-ability this is more of a grey area. Support is known to grant exceptions and did so through all of Windows 7 years that I saw, and is said to still be doing so in Windows 10. If you change motherboards it will require contacting Support and them granting an exception.

Cause

This error can occur if two devices that are installed on your computer have been assigned the same I/O ports, the same interrupt, or the same Direct Memory Access channel (either by the BIOS, the operating system, or both). This error message can also appear if the BIOS did not allocate enough resources to the device.

Recommended resolution

You can use the Troubleshooting Wizard in Device Manager to determine where the conflict is, and then disable the conflicting device.

Disable the conflicting device
  1. On the device Properties dialog box, click the General tab.
  2. Click Troubleshoot to start the Troubleshooting Wizard. The wizard asks you some simple questions and provides a solution to your problem based on the answers that you provide.
  3. Follow the resolution steps provided by the wizard to resolve the problem.

daniellelehoux say:

Snipping tool won't work when trying to capture drop down menu

I recently upgraded to Office 2016 from 2013 and since then my Snipping Tool won't work correctly. I can still capture anything I want with the exception of drop down menus. There are eight other users in my company network; all Windows 7, all Office 2016 and it works on everyone's computer but mine. I open the Snipping Tool, click on cancel (tried Escape, too), then open the drop down menu, press Ctrl + Print Screen and nothing happens. Can anyone help me?
marensater say:

I just figured it out myself. Used to work in prior versions using the Esc, then Ctrl + Print Screen. Now it's a bit different.

  1. Get your screen ready for the capture you want, except for the drop down.
  2. Open the Snip Tool and choose the type of capture you want (I use Rectangle) and now the new part...choose a Delay for however many seconds you need from when you push New to when you can open the desired drop down.
  3. Click New. (The Snip Tool will not activate for however many seconds you chose in Delay.)
  4. Open your desired drop down and wait for the Tool.
  5. Proceed as normal.

Hope that helps!
