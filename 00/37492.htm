There is a big white swirling 360 splash screen every time you start a 360 game on the Xbox One that is pretty hard to miss. There is no stand alone emulator as it is built into every game that is backwards compatible. It is part of the digital download. Other than the built in emulator there isn't anything special relative to the Xbox One with regard to Fallout 3. It wouldn't run on Xbox One if the emulator wasn't working.

You want to make sure that upon starting the game that you point the game to the same save location every time or you will seem to start over. The game will always ask on start up for the save game location be it the cloud, hard drive, thumb drive, etc. but you have to choose the same location each time.
