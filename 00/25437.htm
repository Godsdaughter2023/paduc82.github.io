How do I add a date format within the options for Insert/Date - Microsoft Community

tricia h say:

How do I add a date format within the options for Insert/Date & Time feature?

Word for Mac 2011
When I use the Insert/Date & Time option, one of the date/time options I want is not available.  Highlighted one below.

How can I add this?





bob jones aka: cybertaz say:
You cannot modify the list provided in Insert> Date & Time.

One thing you can do is:

  1. Go to Insert> Field
  2. Choose Date & Time
  3. Select CreateDate [static based on creation date of the document] or Date [updates automatically each time the document is opened]
  4. Click the Options button
  5. Design the date format you wish using the list as examples
After OK-ing out:
  1. Select the inserted Field
  2. Go to Insert> AutoText> New
  3. Give it a unique name of at least 4 characters [e.g., mydate]
  4. Click OK.

From then on you can insert your custom date by typing the first 4 letters of the AutoText Item [i.e., myda] then press return.

This little demo file illustrates three possible approaches, in each of which the name is selected in an unbound combo box:

1.  Go to a record for a specific person where the names ( first + last in my demo)  are distinct.

2.  As 1, but in this case firstly restricting the combo box's list by the initial letter of the person's last name.  This can be useful if there is a large number of people from which to select a name.
