A cabinet file (.cab) is a compressed file archive (like a .zip or .rar). Cabinet files are commonly used to organize installation files because they can include information about other .cab files that are required for installation. However, .cab files donâ€™t include a mechanism for installing files onto systems; they are only compressed file archives. For this reason, Microsoft provides a separate way for users to extract data: self-extracting cabinet files.

Sfx.cab file may contain driver files. So, you may do not delete it.

Hello,

Seems that the Microsoft Photos app when it encounters an image file containing GPS data queries a reverse geolocation service to obtain a location name, when it does not have a location name it displays the geo coordinates instead.

You can still see the GPS coordinates in Windows File Explorer. Right click on a file and in the File Properties tab the values should be displayed there in DMS format.

Sadly, Windows Photos App does not display those geotags as its predecessor, Windows Photo Gallery did.
