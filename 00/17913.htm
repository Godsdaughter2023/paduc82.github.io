1.Access the document library you want to set

3.Delete all previous permission groups

4.Grant Permissions >Invite the users you want to assign with Read-only permission

5.Select a permission level with Read.

After that, if the users access the document library and click the file to open in Word Online, the files can shown up with Read-Only for them.

A "clean install" can mean different things to some. I interpret it as booting from Windows 10 install media, removing all partitions from the disk, and installing to the now unpartitioned space. This allows Windows 10 setup to partition the disk according to defaults.

As long as you use the same Edition of Windows 10 (usually Home or Pro) as you had activated previously on that device, and do not use any Product Key during install, you should normally be fine. It should auto-activate after install, once you connect to the Internet.

Before a clean install you should choose Factory defaults or Optimal defaults in BIOS and enable UEFI and Secure Boot, if available.
