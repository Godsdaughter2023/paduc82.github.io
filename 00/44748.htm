Hi Mike,

You will need to delete the account or accounts (if more than one was created) that are specified in this screen before you can go back.

IMPORTANT: Before deleting the user account please make sure that you have backed up any data that is associated with that user profile (Documents, Music, Videos etc.) as deleting the account will also delete the data associated with that user profile

3. Click on Advanced tab
4. Click on Settings under the User Profiles section
5. On the User Profiles dialog select the <COMPUTER NAME>\Administrator account and click  on Delete – confirm the delete

IMPORTANT: The above steps will also delete all data associated with the built-in Administrator account – please ensure data associated with the built-in Admin is backed up prior to performing the steps.

Please keep in mind the following actions will disable Go Back:

  • Refresh
  • Reset
  • Disk Cleanup (if it touched Windows.old or $windows.~bt)
  • Manually deleting files within Windows.old or $windows.~bt
  • Running on battery (must be plugged in)

If external device was used during upgrade to store Go Back data – then you must have it to perform a Go Back.
