Hi,

Be aware that that there are one or two ways Xbox One enhanced games work when it comes to 4K.  Some games are simply optimised to run in 4K resolution and nothing else is added to the game code.  The textures remain the same as the standard Xbox, but everything looks sharper and crisp because the game is running natively at 3840x2160. 

For example Assassins Creed Origins and Assassins Creed Odyssey run in 4K with HDR, but have no additional 4K assets.  Wolfenstein The New Colossus has no 4K assets either, just a resolution bump.

I don't think Shadow of the Tomb Raider has 4K assets.  The Division doesn't have 4K assets either, as far as I'm aware.  However, it does need the latest patch to run in 4K.  If you ran the Division in offline mode straight after the disc install on an Xbox One X, it will play as if it were running on a standard Xbox One.

I don't think RDR2 has any 4K assets, the game simply runs in 4K on the One X console. The Patch does not contain any additional texture data for the One X platform.
