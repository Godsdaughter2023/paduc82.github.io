The simplest way, as was said, is to get the official ISO files that are made available. ISO's for build 15063 should be available within a week.

The Windows Update downloaded files initially are in a subfolder under C:\Windows\SoftwareDistribution\Downloads. There used to be one main Install.esd file that gets moved to C:\$WINDOWS.~BT\Sources\Install.esd before the upgrade restarts for the first time. From that one file you could produce an ISO.

More recently, the new UUP process instead creates thousands of files under the SoftwareDistribution folder. These files end up in a matching location, but in a folder named Windows.old after the upgrade has completed. Users have created scripts to process these thousands of files into an installable ISO file too. So it can still be done if you want.

There is a website called MDL Forums that has users who have produced a working script to do this conversion. A user named abbodi1406 is one of the authors of it. You might have to join their forum to get to this link, but this will lead you to be able to download a script that works:

https://forums.mydigitallife.info/threads/59082-Windows-10-ESDs-Repository/page130?p=1323469&viewfull=1#post1323469 

This process could all be done using DISM and PowerShell commands too, but it's not very simple to figure out, as you will see if you read the script that they use.

=SUMPRODUCT(--IFERROR(REPLACE(LEFT(A1:A100,LEN(A1:A100)-1),1,FIND("(",A1:A100),""),0))/60
