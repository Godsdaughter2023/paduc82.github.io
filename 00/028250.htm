graye say:

How do I easily manage the favourites bars and bookmarks in Edge using File Explorer. I can not locate this folder? Any suggestions where this located?

Modern versions of the Edge browser no longer store your favorites as individual files and folders like you may have become accustomed to.  Instead, they are now stored in a database.  The database wasn't designed to be "human readable", so you'll need some specialized software to be able to do anything useful.  So, that means that you can't use the File Explorer to view or manage your favorites.

There is a free 3rd-party application called EdgeManage which is designed to make it easier for you to manage your favorites.  It has a more familiar interface, that you might find easier to use.

Take a look at this article to see if this is something that you'd be interested in: http://www.edgemanage.emmet-gray.com/index.html

pauli o say:

If you are using Windows 10 Pro you can use Group Policy editor (gpedit.msc) to set Edge and IE11 to synchronize bookmarks. Open gpedit and go to:

Computer Configuration (or User Configuration) / Windows Components / Microsoft Edge and change "Keep favorites in … " to Enabled.

After that setting anything you do in another browser bookmarks if sync to another browser bookmarks. You can also use File Explorer to handle "old" favorites folder and those changes are reflected to Edge as well.

bill manville say:

1) no.  If you make changes to the Master sheet layout, try to do it by cutting and pasting, inserting or deleting entire columns - that way my formulas will automatically adjust.

2) the code is in the module of the Filter worksheet:

Private Sub Worksheet_Change(ByVal Target As Range)
  If Application.Intersect(Target, Me.Range("Criteria")) Is Nothing Then Exit Sub
  Application.EnableEvents = False  ' don't come back for changes made here
  Me.Range("ExtractHead").Offset(1).Resize(Me.Cells(Me.Rows.Count, 2).End(xlUp).Row).ClearContents
  If Application.CountA(Me.Range("Criteria").Offset(1).Resize(Me.Range("Criteria").Rows.Count - 1)) <> 0 Then
    ' if no criteria do nothing
    With ThisWorkbook.Sheets("Master").Range("MasterHead")
      .Resize(.End(xlDown).Row - .Row + 1).AdvancedFilter xlFilterCopy, criteriarange:=Me.Range("Criteria"), CopyToRange:=Me.Range("ExtractHead")
    End With
  End If
  Application.EnableEvents = True ' don't come back for changes made here
End Sub
