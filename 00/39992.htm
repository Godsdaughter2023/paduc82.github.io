Hi,

Thank you for posting your query in Microsoft Community.I understand your concern, we will help you with the issue

Follow the steps below and check if it helps.

Method 1:


Run Hardware Troubleshooter
Follow these steps to run the troubleshooter.
a. Windows key + X and click on control panel.

b. select troubleshooter.
c. Click hardware and sound and run the hardware and device troubleshooter.

Method 2:

Try to set up the monitor manually and check if it works.
1)  Right-click the desktop, and select Properties or Personalize. 
2)  Next select Display Settings from the list of preferences.
3)  You should see icons with numbers representing your attached monitors. If one is disabled, it will be grayed out.
4)  Select the second monitor and choose to either extend your desktop or mirror your desktop.
5)  Your options may be dependent on the graphics card that you have installed. 
6)  You may need to make these changes from within your control panel. If you have these installed, they can be found in your Start menu.

Method 3:

I would suggest you to update the drivers and check if it helps.

Check the link below:

How to: Install and update drivers

http://answers.microsoft.com/en-us/insider/wiki/insider_wintp-insider_devices/how-to-install-and-update-drivers-in-windows-10/a97bbbd1-9973-4d66-9a5b-291300006293

If the issue persist you can go ahead and un-install and re-install the graphic driver and check if it helps:
You can follow the below steps:

1)  Press Windows Key + R, type devmgmt.msc.
2)  Select the graphic driver.
3)  Click on properties, click on uninstall tab.
4)  Click OK.
5)  Right-click on the driver (Downloaded from the manufacturer’s website) file.
