rasoul khoshravan say:

As Gord explained your problem is with relative and fixed references.

relative reference: When you drag the formula, the relative reference address will change according to original formula.

fixed reference: The fixed reference address will stay same as you drag it.

You fix the row part with a dollar mark before that.

You can fix the column part with a dollar mark before that.

So totally you can have four types of reference addressing:

A1 relative

A$1 column is fixed (row remains relative)

$A1 row is fixed

$A$1 both row and column is fixed

When you entered the formula it is a relative. You can enter the $ by hand but there is a shortcut for it (put cursor on the reference address and hit F4 key) which cycles between above 4 types.

An example

If you want to calculate the percentage of column D, if you enter D2/D7 in E2 and drag it down, it will become D3/D8 in next cell. In order to keep D7 unchanged, you fix its row part, as you drag along row.
