Hello, I know this question was posted eons ago but I came across the problem of connecting my PC to the TV wirelessly as recently as a few hours ago and I wanted to share the steps I took to solve this problem.

(Windows 8, 8.1 and 10)

Method 1:

Open your task manager CTRL + SHIFT + ESC

OR

Right-click an empty space on your taskbar and select Task Manager.

OR

CTRL + ALT + DEL and select Task Manager.

In the Task Manager look for a process named 'Play to Server' under 'Background Services.' This service needs to be Ended every time the media player is used then closed suddenly. For some computers, it may be the one causing the problem. Be sure to restart the Windows Media Player.

If this process cannot be found in your Task Manager, please continue reading.

<Task Manager><End Task 'Play to Server>

Method 2:

Enable Automatic Media Streaming.

This is the easiest method and probably the one that will help you.

First of all, you're going to want to 'forget the network' you're connected to. This deletes the password, disconnects the automatic wifi connection to that network and most importantly gets rid of all personalized settings set for the network.

This is done by left clicking on the network icon at the right of the taskbar (the notification tray), 

all available wifi connections

OR

Dragging your mouse to bottom right or top right of the screen and selecting the settings option then click on the networks icon to view all available wifi connections

OR

Press Windows Button + C then navigate to the settings option either with the mouse or the arrow keys on your keyboard then click on the networks icon to view all available wifi connections.

Once on this screen, you should right-click on the network you're on and then click forget this network.

Next, we're going to reconnect to the network now that the settings have wiped clean everything should go smoothly. The options that come up after the password has been accepted are the most important. This is where you enable automatic media streaming through the network. Click yes to enable.

Be sure to restart the windows media player and maybe even the TV for good measure.

<Networks><Forget Network><Reconnect to Network><Home or Public Network click 'Yes' for Home><Restart Windows Media Player and TV>

Method 3:

Network Discovery.

This option is found in the Network and Sharing Center which can be accessed by either right-clicking the network icon on the notification tray (right of the taskbar) and selecting Open Network and Sharing Center.

OR

Through the Control Panel. Windows + X then select Control Panel... Windows + U opens Ease of Access which has the Control Panel option to the left... or Windows + C, go to settings then select Control Panel. You can even use the 'Run application' (Windows + R) and type in Control Panel.

Once at the Control Panel if you are under 'Category' View look for Network and Internet option then Network and Sharing Center.

If you are under 'Large Icons' and 'Small Icons' view go straight to Network and Sharing Center.

Once at the Network and Sharing Center look for 'Change Advanced Sharing Settings' to the left of the screen.

Under the 'Private Network Settings', be sure to turn on Network Discovery, Turn on File and Printer Sharing and Allow Windows to manage Homegroup connections. Computers with Homegroup settings turned off may not have the last option but it's ok.

Save changes and Exit. Restart Windows Media Player.

<Control Panel><Network and Internet><Network and Sharing><Change Advanced Sharing Settings><Turn On Network Discovery><Turn On File and Print Sharing><Allow Windows to Manage Homegroup Connections>

Method 4:

Media Streaming Options

This option doesn't have anything to do with Windows Media Player but it will still allow you to wirelessly connect to your TV and play media that is found in 5 very specific folders, i.e, Music, Printers and Devices, Videos, Documents and Pictures. This is done using Homegroup.

Using the steps described in method 3 above navigate your way to 'Advanced Sharing Settings' in the 'Network and Sharing Center'.

Under 'All Networks' There should be a blue link written 'Choose Media Streaming Options'

Here you can view devices under the network you are currently on. Give it a few seconds to load up the devices and look for your TV Name. Click Allowed.

Now using the 'Source Button' on your TV Remote you should be able to see your PC together with where the HDMI options are. From here you can access the folders mentioned above and view media through here.
