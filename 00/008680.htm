Update one column of a table with data from another table - Microsoft Community

ronymaxwell say:

Update one column of a table with data from another table

I wish to use an UPDATE query to change data in one column of a table to the values from another table. It is simple enough to do so record by record but can I do so using something similar to the INSERT INTO ... SELECT type of query that does everything in one go? There can be a lot of records so the alternative may be slow.
scott mcdaniel say:

You need to JOIN those tables, and you can then update one from the other. For example:

UPDATE Table1 INNER JOIN Table2 ON Table1.JoinField=Table2.JoinField

SET Table1.SomeField=Table2.SomeOtherField

Obviously, Table1 and Table2 must have a common field. If not, you won't be able to update one table from another.

See this FMS page for lots of examples of different kinds of queries:

http://www.fmsinc.com/microsoftaccess/query/snytax/update-query.html

Also, a good way to learn how to do this is to create a standard SELECT query in the Access query grid, and make sure it outputs all the relevant data (i.e. the field to be updated, the field you're updating from, etc). Then, change the Query Type to an Update query, and Access will properly form the syntax for you. You'll have to make some changes, perhaps, but the basics will be there.

These modifications improve on your routine but I wish I could see some sample data in a publicly posted (redacted) workbook as I am sure that the loop could be improved with Union and perhaps For ... Step 9 Next.

Sub mcr_Mutli_Copy()
    Dim i As Long, x As Long, y As Long, lastcell As Long, countnonblank As Long
    Dim myrange As Range, wsLCW As Worksheet, wsDAT As Worksheet
    
    Application.ScreenUpdating = False
    Application.EnableEvents = False
    Set wsLCW = Sheets("Label Create Worksheet")
    Set wsDAT = Sheets("Data")
    
    With wsLCW
        Set myrange = .Columns("A:A")
        countnonblank = Application.CountA(myrange)
        lastcell = Int(countnonblank / 9) + 1
        For x = 0 To lastcell
            i = i + 1
            y = y + IIf(x = 0, 0, 9)
            
            .Cells(2 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 1)
            .Cells(3 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 11)
            .Cells(4 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 21)
            .Cells(5 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 31)
            .Cells(6 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 41)
            .Cells(7 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 51)
            .Cells(8 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 61)
            .Cells(9 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 71)
            .Cells(10 + y, 1).Resize(, 6).Copy Destination:=wsDAT.Cells(1 + i, 81)
        Next x
    End With
     
    Set myrange = Nothing
    Set wsLCW = Nothing
    Set wsDAT = Nothing
    Application.ScreenUpdating = True
    Application.EnableEvents = True

End Sub
