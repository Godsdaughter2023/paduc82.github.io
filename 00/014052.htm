cbarnhorst say:

So let's get you caught up.

Only the current insider build is available and only Windows Update offers it.  Unless you make your own installation media from an already downloaded copy separate installation media is not available.  Every so often MS will put a new .iso file on the Insider Media Creation Site but so far that has been rare.

Unlike the regular Media Creation Tool site, the downloads are like for like; i.e., Windows Update will only download the same edition and architecture as is already running on the device.

Use a robust backup plan so you can recover from catastrophic failures.

Do not use a preview build as a primary or production computer because preview builds are prerelease software and Microsoft does not provide support for recovery from problems.  Preview builds have broken or incomplete features in them someplace and our task is to report what we find.  It is NOT our task to fix these issues because they may not be fixable in their present state.  Fixes are Microsoft's task.  When you find a bug, report it using the Feedback Hub on your Start Menu.  Don't assume there is a workaround, but look in this forum in case someone has found one. 

Preview builds all expire after a few months so keep up with Windows Update.  Expiration dates on preview builds affect only the build and do NOT affect the status of your device as a registered Windows 10 device, i.e., activation.  An expired build will randomly restart the computer every few hours for a couple of weeks and warn you that it has expired.  After that the computer will not boot the expired build.

You can create a new partition on your hard drive and install a new copy of Windows to use for the insider program without the need for a new Windows license if you do not have a device you can devote solely to previews. 

If you decide you do not want to use preview builds anymore you usually will have to reinstall Windows in order to return to the current public build unless you have made a system image that you can restore to instead.  I image my system right after a new version of Windows releases (like 1511 and 1607) so I can return to it easily.  I store this as a separate archived backup for many months.

Did I mention that backups are your friend?

These are just highlights of what folks frequently ask about or run into trouble with when using preview builds.  Read the details on the Insider Program home page.

Now make your new backup.  :)
