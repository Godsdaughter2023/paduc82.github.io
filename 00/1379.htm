johnbogasky say:

I have finally resolved this error, at least for me.  In my case, it was caused by some corrupted Windows files.  Once I reinstalled windows, the problem disappeared.  That was a drastic fix, but Microsoft support and I spent many hours over several days trying everything else before resorting to this.  My case was eventually elevated above Level 2 support for resolution.

There is a Windows utility that will tell you if you have corrupted system files.  You run it from a command line in administrator mode.  To run the utility, key "cmd", without the quotes, in the search box.  Then press Ctrl-Shift-Enter to open a command line window in Administrator mode.  Click "Yes" on the User Account Control window that comes up.  Then in the command line enter "sfc /scannow", again without the quotes, and press Enter.  The utility takes a few minutes to run and will report if you have corrupted files that the utility cannot fix.  If you have corrupted files, then a Windows reset or reinstall is the only fix.

TURNING OFF PROTECTED MODE IS NOT A GOOD FIX!  Protected mode in Office is an important security protection for your PC.  You should not need to turn off.  Turning it off can be an interim workaround but something more fundamental is wrong with your system if you need to turn it off.
