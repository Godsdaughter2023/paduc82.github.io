rick rothstein, mvp - excel say:

This function is similar to Mike H's, but a little more compact. The function (note that I changed its name) takes up to three arguments... the first two are required and the third is optional. The first argument is either a quoted text string, a string variable (when call from other VBA code) or a cell address (when called from a worksheet as a UDF) and the second is the length of the upper case word you want to find. The third argument is optional and defaults to 1 if omitted... otherwise you can specify whichever word you want, but if you specify and instance number greater than the number of upper case words of the specified length, the function will return the last upper case word of that length in the text. If there are no upper case words of the specified length, the function returns the empty string (""). Here is the code for this function...

Function FindUpper(Text As String, Length As Long, Optional Instance As Long = 1) As String
  Dim Cnt As Long, Word As Variant
  For Each Word In Split(Text)
    If Word Like Application.Rept("[A-Z]", Length) Then
      FindUpper = Word
      Cnt = Cnt + 1
      If Cnt >= Instance Then Exit For
    End If
  Next
End Function

Sub SendMyWorkBook()

ActiveWorkbook.Save
