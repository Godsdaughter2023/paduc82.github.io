lemp say:
It depends on how/if the disk really failed and how much you want to spend (how much is your data worth?).

See if the manufacturer of the hard drive offers drive diagnostic tools as a bootable CD.  For example, Western Digital offers its DataLifeGuard tools as an ISO file that can be used to create a bootable CD with e.g., ImgBurn.

Alternatively, you can make a bootable "rescue" CD, e.g., Hiren's, or a live Linux distro, e.g., Knoppix

http://www.hirensbootcd.org/download/
http://knoppix.net/

After you download the ISO file and burn the CD, boot your computer with the CD (you may have to re-arrange the boot order in your BIOS; ask for directions if necessary).  If you are using a CD from the drive mfr, the use of the diagnostic tools should be pretty clear.  If using Hiren's, Knoppix, or similar, use whatever file utilities are available to see if you can access your drive. 

If the disk passes the diagnostic tests, you can either try to repair Windows (start another thread) or use the file utilities on Hiren's or a Linux CD to copy the files to a USB flash drive.

If the disk fails the diagnostic tests or if the file utilities on Hiren's or Linux can't access the files, your only option is professional data recovery.  Drive Savers has an excellent reputation for this, but it is quite expensive.
