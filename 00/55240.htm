hairyfool say:

Nothing!

By default System Image recovery will only work if the new drive is physically the same size or larger than the original, this applies even when the actual space used on that drive is a small part of the original disk.

However when the new disk is larger than the original Recovery will create a partition the same size as the original disk and leave the rest of the disk as unallocated.

Start Disk Manager and check the lower pane. You should find your new C: drive with the blue top bar occupies only a section of the disk with the remainder defined as unallocated with a black top bar.

Right click on the allocated section and the context menu should have the option Extend volume, click on that and extend the drive to include the unallocated space.
