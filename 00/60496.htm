Looks like there are some IPv6 settings in the mix here, particularly the first DNS server address of fe80::91b0:76c6:94dd:2b09 (which is actually a link-local address and probably not a real DNS server anyway). The home router probably doesn't run IPv6 either.

Look at the adapter properties for the Wireless Network Connection and disable the Internet Protocol Version 6 binding (uncheck it in the list). Disable and re-enable the adapter and see if this changes anything functionality-or-ipconifg-wise.

Check you are able to ping the other advertised DNS servers:

ping 24.220.0.10
