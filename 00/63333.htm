When using Live on backwards compatible games, there will actually be two instances of you signed in, one for Xbox One, and another for Xbox 360.  This is how it's meant to work be design.

For most programs, you don't delete the *.exe file but go to Control Panel > Programs and Features and "uninstall" the program.

From your sketchy description, it appears that you may be seeing this error message when Windows starts.  If that is the case, the most likely explanation is that you or your antimalware software deleted (rather then uninstalled) nielsenonline.exe but left the setting in the Windows Registry that attempts to start the program when Windows starts.

If you want this program, you should re-install it.  Re-installing the program and then uninstalling it properly also may eliminate the unwanted error message.

If you do not want the program and are unable to re-install it, and if the error message does appear when Windows starts, download Autoruns (free software) from https://technet.microsoft.com/en-us/sysinternals/bb963902.aspx?f=255&MSPPError=-2147217396  If you don't want to download and install this useful utility, you can run it "live" from the same web page.
