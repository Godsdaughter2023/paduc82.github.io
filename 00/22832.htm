jasles say:

It's hard to tell from what you're saying, but it seems that you have the stock quantity on one report that needs to be updated with data from another report?

If both reports are in Excel, you can use the VLOOKUP function to automatically update the data on the report.

Use it like this: = VLOOKUP(cell with lot number, range of data on the other report (with the column that has the matching lot number as the first column), which column number in that range has the stock quantity)
