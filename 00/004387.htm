albert d. kallal say:

You most certainly can install multiple versions. About the only issue is don't ever launch both versions at the same time.

 

I have 2003 and 2010 on my laptop. There is a "delay" in which the installer will launch and this delay can take a good 45 seconds.

 

Most suggest installing the lower version first but as a general rule this is not a requirement.

Multiple versions of office can often be problematic but for development use such a setup is quite common and such a setup should work fine for you.

 

Since 2000 is rather old, then on newer machines I suggest turning off UAC during the install, and when possible install as an admin. I also suggest disabling any virus software during such installs.

 

So a simple answer here is yes you can install multiple versions of Access on the one computer, but keep in mind that the default file extensions will be associated with the last version of Access you run. So to "switch" versions of Access you launch the version of Access you  want to use and THEN open the file (don’t' click on the file name).

 

You can as a rule still click on a mdb file in folder view but this results in the last version of Access being launched. So a bit of extra work and caution may be required to launch  the version of Access you want to use, but other than this issue installing 2 or even 4 versions of Access is possible.
