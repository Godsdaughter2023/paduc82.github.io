Enter below formula in cell F2:

=INDEX(A2:A10,MATCH(E2,INDEX(B2:D10,,MAX(INDEX((B2:D10=E2)*(COLUMN(B2:D10)-COLUMN(B2)+1),))),0))

This returns the name from column A, corresponding to the LAST column (from B to D columns) which has E2 value. This uses INDEX/MATCH - determines the last column number (column D) in the array (B2:D10) that matches the criteria (E2), and returns the Name from column A.
