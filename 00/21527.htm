Greetings Petruuus!

 

Here are some methods to save battery power.

 

Reduce the load on your battery:

  • Decrease screen brightness.
  • Set Timeout duration on Backlight.
  • Set Auto-Off Timeout duration for 1 minute or less (device turns off).
  • Throttle down your CPU speed.
  • Remove external devices (SD Cards, etc)

Turn off appropriate radios on your device:

  • Turn off Bluetooth transceiver when not in use.
  • Turn off WiFi transceiver when not in use.
  • Turn off GPS receiver when not in use.
  • Turn off Cell transceiver when not in use.

Change the way you do things with your device:

  • Minimize media playback use.
  • Minimize use of e-mail and browsing.
  • Once in a while (every few weeks), drain battery until dead, then recharge 100%

The last suggestion is that you may wish to buy a Charger for the car, home, and office so that you can recharge the device whenever you are any of the three locations...

echo s say:

The colors, fonts and effects travel with the theme and are included there already -- so there's no need for the user to select the color or font or effect set in the dropdowns on the Design tab. If you want there to be a color set, for example, that's named "My Company Colors," then yes, you'd need to send that XML file separately and have them save it in the appropriate folder. Otherwise, it's already there.

Only one set of colors travels with the template, so if you want more to be available, you'd have to add those manually to the theme's XML. But that original color set that's incorporated into the theme doesn't show up in that Design | Colors dropdown. It's just in the theme and shows up when you check the font or shape or outline colors. It's also what the colors in the Shape Styles (and tables and charts, etc.) galleries are based on.
