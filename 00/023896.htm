Upgrade Advisor app for Windows 10 Mobile

The Upgrade Advisor app will tell you if your Windows Phone 8.1 device is eligible for upgrade to Windows 10 Mobile. If your device is eligible, Upgrade Advisor can help free up space on your device to prepare for installation.

How to get the app
  1. On Start , swipe over to the All apps list, and then select Store.
  2. Search for the Upgrade Advisor app, and then install it on your device.
  3. Open the app on your device. It will check your device and let you know if it's ready for the upgrade, if another update is required before you can get the upgrade, or if your device can't be upgraded.
  4. If the update is available for your device, select Enable Windows 10 upgrade.

If your device is eligible for upgrade, you may need to free up some space before installing it. Upgrade Advisor will recommend files (such as videos or photos) to temporarily move to OneDrive or an SD card (if you have one installed). Accept the recommendations, or specify which files you want to move. You can also delete files you no longer want on your device. If you move files to OneDrive, you'll need to use the app to restore files back to your device after Windows 10 Mobile is installed.

Once your phone is upgraded to Windows 10 Mobile, learn how to get updates by going to How do I keep my Windows 10 Mobile device up to date.

Create and save nine union queries with ten queries each.

Then create a union query where you union the nine saved union queries.

And do use date values for dates, not fancy numbers, or the queries will not return date values:

#2018/08/01# AS [Date]

or:

DateSerial(2018, 8, 1) AS [Date]

Also, read up on normalising a database to prevent the need for nasty queries like this.

Using PowerPoint's built-in Save As Movie feature is not the best way to convert a PowerPoint presentation to a movie. You will lose all the sound, movies won't be captured, animations & transitions will be substituted or won't play. In Lion, you can use QuickTime player to capture just he portion of the screen that has the presentation playing.

How to save an excel 2003 file to caret delimited. - Microsoft Community

afloreseca say:

How to save an excel 2003 file to caret delimited.

How to save an excel file to caret delimited.
frank arendt-theilen say:

Hi,

use the following sub:

Sub PrintCSV()
Dim rngRange As Range, rngRow As Range, rngCell As Range
Dim strTemp As String

Const Path As String = "C:\yourpath"
Const Filename As String = "filename"
Const Extension As String = ".csv"
Const Delimiter As String = "#"           '<-- change it to your needs
Const strBackslash As String = "\"

  Set rngRange = ActiveSheet.UsedRange

  Open Path & strBackslash & Filename & Extension For Output As #1

  For Each rngRow In rngRange.Rows
    For Each Zelle In rngRow.Cells
      If InStr(1, rngCell.Text, Delimiter) > 0 Then
        strTemp = strTemp & """" & CStr(rngZelle.Text) & """" & Delimiter
      Else
        strTemp = strTemp & CStr(Zelle.Text) & Delimiter
      End If
    Next
    Print #1, strTemp
    strTemp = ""
  Next

  Close #1
  Set rngRange = Nothing

End Sub

If you want to change the list separator (e.g. delimitor), you need to set it system wide. For a system wide change:

From Control Panel, choose Region and Language. From the Region and Language dialog's Location page, click Additional Settings in the lower right. Two thirds of the way down this Customize Format dialoog, you can type in a new List separator. Click OK to save, and OK to save and exit the Region and Language dialog.

Using Excel's Save As, CSV (MS DOS) format will generate a delimited file with your new list separator.
