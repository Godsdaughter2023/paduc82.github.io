Create a template for mail messages - Microsoft Community

¡firedog say:

Create a template for mail messages

Technical Level : Basic

Summary
If you regularly send messages to the same group of people and/or with the same or largely unchanged content, you can save time and effort by using a template to store all the details that don't change from message to message. A template is simply a draft message which can be opened, completed and sent. You can create as many templates as you like for different purposes.
  
  
Details

Users frequently ask how to include a CC: or BCC: address in all outgoing messages automatically. There is no function within Windows Live Mail for doing this, but it is possible by using the method in this article.

Another scenario that causes problems for many users of Windows Live Mail 2012 is the unreliability of the group email feature, which relies on addressing a message to a category (a group of contacts) rather than to individual contacts. While there is a workaround for this fault in the program (http://answers.microsoft.com/en-us/windowslive/wiki/livemail-email/send-a-group-email-in-windows-live-mail-2012/cd98d675-e117-4425-bc44-09c10b8bb18f), it doesn't help if the category itself doesn't always return all the contacts it's supposed to. Again, using a template automates the group email process.

Create a template

Start a new mail message (e.g. by pressing Ctrl-N) and enter the bits that don't change, or don't change much, e.g. addressees, subject, body, pictures, signature and attachments. Format the message to suit. Figure 1 shows a simple template that will include a BCC address in the outgoing message, while Figure 2 shows a more complex one: the skeleton of a regular newsletter.

 

Press F12 to save the message as an .eml file in a convenient place, like the desktop (NB: the Save message As  dialogue will by default apply the subject line as the filename; edit this to suit). Double-clicking on this .eml file will open a new message ready for completing and sending. The program doesn't have to be running when you start a new message in this way. The template itself will stay unchanged where it is, ready to be used again next time.

Put a descriptive name for the distribution list in the To... box. When sending to a distribution list, it's polite to put the addressees in the BCC field so that none of the recipients will see who else has received the message.

A useful trick to ensure that the message makes it to its destinations is to put the sender in the To... box. In the example newsletter template above, a new contact has been created with WLMail as its first name, Newsletter as its last name and with the sender's address as its email address. If the To... box is left empty, the message will show as being sent to Undisclosed recipients, which isn't very helpful to them or to you when you need to find a specific message in your Sent items folder.
