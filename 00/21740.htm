    网游 say:

    Hi Gordon,

    I tried the suggestions/answers from multiple searches for "Turn off suggested sites for IE11", but they DO NOT REMOVE the "Turn on Suggested Sites..." or "See Suggested Sites..." buttons from the IE11 Favorites Center.

    The Internet Option... Internet Options>Advanced tab, "Enable Suggested Sites" (you must close and reopen IE and the Favorites hub for the changes to take affect), only changes the "See Suggested Sites" button  (on the favorites hub) back to "Turn on, Suggested Sites".

    I can understand how some users would be bamboozled by the wording used on the Buttons, and the wording used in the Internet Control Panel.

    There is however a Group Policy setting that DOES hide the "Turn on Suggested Sites..." button from the Favorites Hub in IE11.

    If you are using a Pro or Enterprise Edition of windows or you are using a company computer (managed by your company's IT department) then you or they can follow the instructions from the following link.

    http://www.gracecentered.com/christian_forums/computer-assistance/how-to-get-rid-of-annoying-'suggested-sites'-in-ie-favorites-toolbar-forever/

    Usually if you are using a company computer, you personally do not have access to change computer security settings... you should contact your company's Help desk or IT department.

    GPEDIT.msc>Local Computer Policy>User Configuration>Administrative Templates>Windows Components>Internet Explorer

    "Enable Suggested Sites"

    This policy setting controls the Suggested Sites feature, which recommends websites based on the user’s browsing activity. Suggested Sites reports a user’s browsing history to Microsoft to suggest sites that the user might want to visit.

    If you enable this policy setting, the user is not prompted to enable Suggested Sites. The user’s browsing history is sent to Microsoft to produce suggestions.

    If you disable this policy setting, the entry points and functionality associated with this feature are turned off.

    If you do not configure this policy setting, the user can turn on and turn off the Suggested Sites feature.

    If you are using a pro or enterprise edition of windows then, after following the above answer's you must close and re-open IE for the changes to the registry to take affect.

    If you are using a home edition of windows and have an advanced windows PC knowledge, then you can copy and paste the following text into notepad and save as a .reg file, then using file explorer run the .reg file(using admin rights) to merge it into the windows registry.

    Windows Registry Editor Version 5.00

    [HKEY_CURRENT_USER\Software\Policies\Microsoft\Internet Explorer\Suggested Sites]
    "Enabled"=dword:00000000
