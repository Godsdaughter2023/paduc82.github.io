rick rothstein, mvp - excel say:

If I understand you correctly, you want to put a formula into the first cell of the last column of a specified range and then copy it down that column to the end of the specified range (overwriting anything that already exists in that range with these formulas). If so, you can use this code...

Dim R As Range, LastColumn As Range
Set R = Range("H11:M19")
Set LastColumn = R.Columns(R.Columns.Count)
LastColumn.Formula = "=Sum(L11:L13)"

Simple change the specified example range to your actual range and, of course, substitute your formula for the example one I used. If, on the other hand, you wanted to put your formula in the empty column next to the last row of the specified range, then you could this code instead...

Dim R As Range, LastColumn As Range
Set R = Range("H11:M19")
Set LastColumn = R.Columns(R.Columns.Count)
LastColumn.Offset(, 1).Formula = "=Sum(L11:L13)"

The only difference between the two code snippets being the use of the Offset property in the last code line.

toddallcock say:

Honestly, the closure of the WM Market wasn't that big a deal. Unlike iOS or WP7, which requires all apps to be installed from a centralized official marketplace, WinMo's Market was really designed as a convenience, since apps could be installed without the market. WinMo existed for years before the official Market was even launched, and the Market only contained about a quarter of the apps available for WinMo, since many developers felt they could market their apps successfully without giving Microsoft a commission on each sale.

 

Windows Mobile apps can be found at at www.handango.com, www.freewarepocketpc.net, or freewareppc.com, and many other sites.
