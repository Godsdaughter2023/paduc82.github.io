grschinon say: You don't need a T-Mobile SIM card to remove T-Mobile's SIM restriction.

Simply insert your own SIM card. The phone will complain that the SIM card is invalid but that doesn't matter for now.

Enter the restriction code you were given this way:

#pw+...............+1#

You get the 'p' by pressing '*' 3 times.
You get the 'w' by pressing '*' 4 times.
'...............' replaces the restriction code you were given.

The phone will now be unlocked. You might have to switch it off and on again for the SIM to register.

Do not attempt to enter this key sequence too many times. If the wrong sequence is entered, I think, 3 times in total (either because you mistyped it or because you were given the wrong code in the first place, it does happen) then the phone will be locked permanently to T-Mobile and no code will be able to unlock it.

cjlim say: Try again as follows.

Enter the 15 digit Network Unlock Code as follows:

#pw+ followed by the 15-digit unlock code then +1#

For example: #pw+123456789012345+1#

The keypad will only allow you to enter numeric digits.

To enter p, press the * (star) key three times.

To enter w, press the * (star) key four times.

To enter +, press the * (star) key two times.

5. The phone will display ?Phone restriction off?. The handset is now unlocked.
