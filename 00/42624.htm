ADVICE for anyone using Ubuntu in a dual boot system with Win10 on a x64 architecture as a WIN10 patch upgrade WILL wipe your GRUB boot system out.

1) have a UBUNTU boot disk handy. Always!

2) When a Windows upgrade destroys GRUB, use the UBUNTU boot disk and boot up into "Trial" mode (i.e. don't reinstall UBUNTU). You may need to re-order your boot sequence and also ensure that secure boot is DISABLED else this may not work. Booting from dvd takes time - be patient!

3) Ensure you are connected to the internet - you may need hard wired connection as wireless does not always come up in Trial mode

4) Open a terminal and download the GRUB repair program:

  "sudo add-apt-repository ppa:yannubuntu/boot-repair" <CR>

  "sudo apt-get update" <CR>

  "sudo apt-get install -y boot-repair && boot-repair" <CR>

you should get a "success" message
