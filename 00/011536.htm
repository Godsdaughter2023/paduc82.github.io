bob jones aka: cybertaz say:

The first question is what type of keyboard do you have - full or condensed (laptop, Apple Wireless)? If the latter, the second question is what are your OS X Keyboard Preferences set to? The specific keyboard shortcut is Command+End, but what key combination you need to use depends largely on the answer to those questions.

On a full keyboard it's usually rather straightforward if no interfering keyboard customizing has been done. Since you did mention the word "function" I'm assuming it's a Condensed keyboard, which can be a bit more deviant :-)

Assuming all Apple default settings are in place, fn+Left Arrow is mapped as the Home key: fn+Right Arrow is mapped as the End key. However, fn+Left/Right Arrow are for beginning/end of line. For beginning/end of document the equivalent combination is fn+Command+Left/Right Arrow.

You'll find considerably more if you have a look in Help> Word Help using the search term keyboard shortcuts.
