A corrupted installation is a possible cause why you are unable to launch a game or an app on Windows 10. If Minecraft is not launching on your computer, there is a possibility that you need to install Windows Updates first. To check for new Windows Updates, open the Start Menu and click on Settings > Update & Security settings > click on Check for Updates.

It is also possible that Windows 10 apps won't open if the Windows Update service isn't running. To check the status of Windows Update service, follow the steps below:

  1. Press Windows + R on your keyboard, and then type services.msc.
  2. Hit Enter or click OK.
  3. In the list of services, look for Windows Update and make sure that Startup Type is set to Manual or Automatic.
  4. Set the Service status to Start.
  5. Click Apply and OK to save the changes.

Make sure that you are signed in to an administrator account before launching the game.

If the issue persists, reset Windows Store to clear the store cache.

  1. Press Windows key + X on your keyboard.      
  2. Click on Command prompt (Admin).      
  3. Type wsreset.exe, and then hit Enter.      

You can also reset the game from the Settings menu. This process can help restore corrupted game installation files.

  1. Press the Windows key on your keyboard, and then click Settings. 
  2. Select System, and then select Apps & features.  
  3. Find the game, and then click on Advanced options. 
  4. Click Reset. 
  5. Restart your computer for the changes to take effect.
