Fixit - When you run an .exe file on a Windows Vista-based or Windows 7-based computer, the file may start a different program

http://support.microsoft.com/kb/950505

 

 

Try the following:

1. Type command in the RUN dialog box to open Command Prompt

3. Type regedit to open up the Registries.

4. Expand HKEY_CLASSES_ROOT and find the folder of .exe

---- FRAPS  (free version) -  is one option (FRAPS is mainly a recording program but has an fps counter as well),

"how to I check if the 750m is also pushing 100%? Also how do I check temps under gaming load ?"

With the other two tools I gave links too. Both show the highest temps, and GPU-z shows the processor usage along with several

other parameters such as clock speed (see below)


- Also verify your laptop is actually switching to the Nvidia graphics processor when you launch these games.

If the games are running on the 750m the clock speed will ramp up (from a low idle speed to '3D' speed ) when you run the game.

It should happen automatically via the Optimus software used by laptops with Nvidia graphics processors, but it's not uncommon

for laptops to fail to  switch from the Intel HD integrated graphics when a game is launched.

I don't use a laptop, but I believe this issues is more common when running on battery vs mains power.

Some games with very basic graphics such as Candy Crush wouldn't really need to switch to the Nvidia graphics as they will run

well enough on the Intel HD.

If the games are running on the Intel HD graphics this would be a good reason for 100% cpu usage while gaming.

There are three options I am aware of to force games to run on the dedicated graphics.

- I believe if you R/click on the games shortcut or .exe file you will find an option in the drop down memu to force the game

to run on the Nvidia graphics.

- There should be something in your machines user manual regarding Optimus, but the GeForce Experience should also have an option (preferred processor) in the games settings to force the game to run on the 750m rather than the Intel HD graphics.

- There should also be a similar option in the Nvidia Control Panel 3D settings > Manage 3D Settings > Program Settings tab.

( If the games are not already on the Program Settings list you can create a new profile by browsing to the games .exe file

in the games install folder).
