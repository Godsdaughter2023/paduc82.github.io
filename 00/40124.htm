Saving Windows7 Solitaire games

I can save a Windows7 Solitaire game, but if do not choose to continue it when I reopen Solitaire, it's lost forever and will only reset @ (52). How can I keep the saved game and continue it if and when I choose?

syed n say:

Hi rds35,

 

When you re-open the game after saving and select not to continue the previously saved game then this saved game will be deleted and starts the new game. It is by design.

 

However, if you want, you can change the location of the saved game and put it back to the original location when you want to continue the same game. This is a tedious process though.

 

To do so, follow these steps.

 

a.       Save the game.

b.      Click Start, Computer and then open the local disk (C :)

c.       Open the Users folder and then “Saved Games”.

d.      Right click on the saved game and select copy.

e.       Navigate to the location where you want to put this file. Right click and then click Paste.

 

Now you can play a new game. Later, if you wish to play the previously saved game then go to the location where you saved the game. Copy and paste it to its default location under “Saved Games” as mentioned above and double click on it to continue the game.
