First, that's 79 individual items; 31 registry keys, 22 registry values and 26 files.  In truth there are about 4 or 5 different adware or potentially unwanted program (PUP) products, so not many when you consider these often come bundled together or install one another.

Adware and PUPs, are a major focus of Malwarebytes, which is far more aggressive than Windows Defender and other antivirus products at detecting and removing these typically less malicious items.

Though there is some additional PUP detection capability included in Windows Defender, it is disabled by default and only directly supported for use by enterprise business customers, where IT departments are available to use the management consoles normally required to control them.  Though it's possible to enable these detections manually in the registry, this isn't supported by Microsoft for consumers, so any issues that might result will be your own responsibility.

If you avoid downloading the useless junk programs these most often attach themselves to, you won't often see such PUPs, as I haven't in over 20 years online.  If you browse or download programs without paying close attention to optional selections or potentially unsafe web or download sites, you're almost guaranteed to get some of them.

don cuthbert say:

The Media Creation Tool method can create a bootable USB stick with the latest public build of Windows on it. Rufus will create a bootable USB from an ISO file.

To boot from it, you need to find the key on your PC to boot from USB at power-up, or switch the BIOS settings to do that. Often the key to tap at power-up is F12, F2 or ESC to get a one-time boot menu to point to the USB.

A simple way that might work is to run recoverydrive.exe and choose to backup System files. If that works, it will create a bootable USB that can do a clean install of your current Windows version. But you might prefer to have the public build as your primary install and not the build that you are running now.

Shrinking the C: drive is easy with Disk Management. You just select the C: drive, right click on it and choose Shrink Volume. It will tell you how much you can shrink it. 64 GB (65,536 MB) should be plenty for a partition for testing Windows Insider builds. Then if you boot with Windows 10 Insider install on a USB or DVD, you can just Custom install Windows Insider to the 64GB space created by shrinking C:. It will create a partition there.

bill manville say:

You need to change to using a text representation of the SSN.
e.g. enter it as 011-11-1111

I think you have a number (11111111) formatted as 000-00-0000.  Access will only get to see the number.
-----
If you have a lot of such cells, you could convert them all using a macro.

Sub ConvertSSNToText()
  Dim C As Range
  For Each C In Selection.Cells
    C.Value = "'" & C.Text
  Next
End Sub

Select the relevant cells and Developer > Macros > ConvertSSNToText > Run
