graham mayor say:

The macro as supplied will do most of that. The only thing that requires adding is to eliminate the sheet that contains the group members' details. To work around that you need to add a trap  e.g. the bold lines below.

For Each xlSheet In ActiveWorkbook.Sheets
    If Not xlSheet.Name = "Sheet1" Then        ' the name of the sheet to omit
        strName = xlSheet.Cells(3, 2) & ".pdf"
        xlSheet.ExportAsFixedFormat _
                Type:=xlTypePDF, _
                Filename:=strPath & strName, _
                Quality:=xlQualityStandard, IncludeDocProperties:=True, _
                IgnorePrintAreas:=False, OpenAfterPublish:=False
        'Substitute the cell with the e-mail address
        SendMessage xlSheet.Cells(1, 1), strPath & strName
    End If
Next xlSheet

The subject and body shown are not personalised, but you could personalise them by adding and passing the relevant variables to the Sub SendMessage(strTo As String, strAttach As String) macro and using those variables as required in the oRng.Text  and .Subject lines.

graham mayor say:

I think you are making up your own syntax, and you have an extra parameter in 

SendMessage xlSheet.Cells(1, 2), xlSheet.Cells(2, 4), strPath & strName

that is not accounted for in the sub you are calling (at least not the version I provided). :)

Sub SendPDFs()
Const strPath As String = "C:\exports\"
Dim xlSheet As Worksheet
Dim strName As String
    For Each xlSheet In ActiveWorkbook.Sheets
        If Not xlSheet.Name = "Sheet1" Then        ' the name of the sheet to omit
            If Not xlSheet.Cells(1, 4) = "" Then
                With xlSheet.PageSetup
                    .Zoom = False
                    .FitToPagesTall = 1
                    .FitToPagesWide = 1
                End With
                xlSheet.Rows(15).AutoFit
                xlSheet.Rows(19).AutoFit
                strName = xlSheet.Cells(1, 4) & ".pdf"
                xlSheet.ExportAsFixedFormat _
                        Type:=xlTypePDF, _
                        Filename:=strPath & strName, _
                        Quality:=xlQualityStandard, IncludeDocProperties:=True, _
                        IgnorePrintAreas:=False, OpenAfterPublish:=False
                'Substitute the cell with the e-mail address
                SendMessage xlSheet.Cells(1, 2), strPath & strName
            End If
        End If
    Next xlSheet
End Sub
