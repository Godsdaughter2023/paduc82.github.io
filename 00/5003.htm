Rollback from Windows 10 to the previous OS has a maximum limit of 30 days.  Some user changes to Windows 10 can cancel the rollback availabilty before 30 days.

You cannot use the Windows 7 key to reinstall Windows 7 without deleting the Windows 10 upgraded from it.  An upgrade from 7 to 10 for free is an entitlement upgrade.  The Windows 10 activation that results is an entitlement activation (Microsoft's terminology) as opposed to a product key activation (also Microsoft's terminology).  If you used the Windows 7 key to install another copy of Windows 7 your Windows 10 would no longer be entitled to activation, so when you enter the Windows 7 product key in the new Windows 7 installation you will see "product key in use" or "product key not available."

don cuthbert say:

It will be best to purge all existing traces of downloaded Insider build files. Delete C:\Windows\SoftwareDistribution and C:\$WINDOWS.~BT folders. Because of file rights or files in use, it may be easiest to boot with Windows 10 install media, such as a bootable USB, and use SHIFT-F10 to get a Command Prompt. Then:

 

rd /S /Q C:\Windows\SoftwareDistribution

 

rd /S /Q C:\$WINDOWS.~BT

Another way is to start an install of Windows 10 by running setup.exe from within Windows. Cancel it just before it actually proceeds to install. Apparently purging any Windows setup files is one of the first things it does prior to an in-place upgrade like this.

Disk Cleanup may also do it. I believe running it from an Administrator Command Prompt may be more thorough:

cleanmgr /sageset <-- select every option

cleanmgr /sagerun <-- deletes files preselected above
