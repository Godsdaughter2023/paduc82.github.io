cmdrkeene say:

If you've connected to any online service (like a Microsoft Account/Hotmail/Live/MSN/Xbox/etc.) that uses passwords, then the computer is going to require a password.

You have several options.  You can use a PIN code instead of the full password so that unlocking your computer works like your phone.

You can also just set the computer to login automatically for you. It doesn't get rid of the password, but you'll never have to type it.

A better compromise is to set it to "don't require a password when waking from sleep mode", but still require it for a full system restart. Many phones and tablets do this also, and it's a good mix of convenience (never having to type your password during regular use) and security (if the battery goes dead or it restarts, it needs your password again).
