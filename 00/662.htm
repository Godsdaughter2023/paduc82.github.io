scott mcdaniel say:

You must refer to the Subform CONTROL, which may or may not be named the same as the form you're using as the Subform. So if I have SubformA and SubformB, both of which are embedded on MainFormX, I could requery SubformA from SubformB like this:

Me.Parent.SubformB.Form.Requery

To determine the name of your subform, open your parent form in Design view, and then click ONCE on the subform, then review the Properties for that information. In the image below, after I click on the Subform note that the Property Sheet section reads "Subform/Subreport". In this case, the name of my Subform Control is "sfDetails".

ken sheridan say: Mike, if you still have my original StudentLog demo on which you based your attendances database a few years back, you'll find an example of requerying a control in one subform from another in the students subform in the courses form;

Private Sub Form_AfterInsert()

    ' requery attendances combo box to include new student
     Me.Parent.sfcCourseAttendances.Form.StudentID.Requery
     
End Sub

Note how the subform control is differentiated from its source form object by giving its name the sfc prefix.  A subform control and its source form object can have the same name, but I'd always recommend differentiating them in this way to avoid ambiguity.

john sr wilson say:

The Slides.Add method is legacy code from 2003 and earlier. This is before CUSTOM layouts were possible. You need to use the Slide.addSlide method which will accept custom layouts but not by name which makes it difficult.

This is top of head code but it would look something like this:

Sub addSlide()
Dim oCust As PowerPoint.CustomLayout
Dim pptSlide As PowerPoint.Slide
Dim pptApp As PowerPoint.Application
Dim pptPres As PowerPoint.Presentation
'Open PowerPoint
Set pptApp = New PowerPoint.Application
pptApp.Visible = True
Set pptPres = pptApp.Presentations.Add
pptPres.ApplyTemplate ("C:\ and rest of path.potx")
For Each oCust In pptPres.Designs(1).SlideMaster.CustomLayouts
If oCust.Name = "Display Content" Then Exit For 'use YOUR layout name
Next oCust
If Not oCust Is Nothing Then
Set pptSlide = pptPres.Slides.addSlide(1, oCust)

Else

MsgBox "Not Found"

End If
End Sub
