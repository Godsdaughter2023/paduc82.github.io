Just to add to what Marshall and John said. When Access encounters a reference to a field or variable name that has not been defined properly, it assumes this is a variable value you want the user to enter at run time, so it prompts for the data. So you may have a misspelled field name or a field that was used, but is no longer used in the report or something like that.

cawpo say:

Protect formulas

I am using 2007. How do I protect just the formulas on a worksheet? Does each worksheet need to be done, or can I do the workbook so each worksheet has the formulas protected.
mike h.. say:

Hi,

Select all cells by clicking the box at the intersect of column numbers/row letters.

Right click anywhere on the sheet - Format cells - protection tab and un-check locked. OK

Left click anywhere to de-select cells.

Tap F5 - Special and select 'Formulas' - OK  - and all cells with a formula in will be selected.

Right click any of these cells - Format cells - protection tab and check locked. OK

Protect the worksheet and your done.

You will need to do each sheet seperately.
