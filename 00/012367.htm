Welcome to the world of ARRAY formula. First off ARRAY formula are slow and particularly so if as you have done and use full columns. So avoid the use of full columns. In this formula I've cut it down to 15 rows, use more if you need to but only use sufficient to catch all your data.

john korchok say:

Watermarks can really interfere with legibility and usually add nothing to content.

If the picture is to be different on each page, you're going to have a harder time. You would have to use a placeholder on the slide layout, but then that placeholder will always appear in front of the footer graphic.

If the picture can be the same on each slide from the same layout, then try this:

  1. Choose View>Slide Master. The Slide Layout for your slide appears.
  2. Draw a large rectangle.
  3. Choose Drawing Tools Format>Edit Shape.
  4. Click on the bottom border and drag until you have a curve that follows the shape of the footer graphic:
  5. Right-click on the shape and choose Format Shape. The Format Shape pane opens at the side of your screen.
  6. Expand the Fill parameters.
  7. Choose Picture or texture fill.
  8. Click on the File... button to load your graphic.
  9. Adjust if required, then close the Slide Master to see how it looks in your slide.

Because this is applied to the layout, all slides with the same layout will have the same watermark. If this is to apply to just one slide, use the same steps, but do it on the slide. If this is to appear in all slides in the presentation, look for the Slide Master, the larger slide at the top of the Slide Layout thumbnails, and add the effect to that slide.
