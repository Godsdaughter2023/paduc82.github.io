jeeped say:
The F4 key is the correct key to use when cycling a cell reference through the available absolute/relative combinations; e.g. A1 → $A$1  → A$1  → $A1  → A1.

I'm guessing that you have a multi-purpose keyboard (laptop..?) that has reassigned the F1-F12 (i.e. function keys) to alternate commands. There may be an F-Mode or F-Lock button on your keyboard that toggles the functionality of the F-keys. You might also try holding down a (blue ...?) Fn key while tapping F4 to gain the true F-key functionality.

If you do not wish to keep the alternate commands and want to revert to true F-key functionality, this will be an option in your computer manufacturer's keyboard software or in the case of a laptop, typically within the BIOS.

teylyn say:
Hello,

the F4 function key does different things, depending in which situation it is applied.

1. Spreadsheet operations

If you perform an action in your spreadsheet, like insert a row, delete a cell, anything that happens on the sheet level, then F4 will repeat the last action. For example, format cell A1 with a red fill. Then click H6 and hit F4 and H6 will have a red fill, too.

2. Formula editing

The other functionality of the F4 key only applies when you are editing a formula. While the cursor is in or just before or after a cell reference, the F4 key will cycle through the different possibilities of absolute and relative referencing, i.e. A1, $A1, $A$1, A$1. If the cursor is not located in or near a cell reference, F4 won't do much. 

Chances are that you had just performed an activity to clear a cell on the spreadsheet level, selected the cell with the formula and hit F4. This would repeat the command to clear the cell, and not change the formula's cell references. 

So, be sure what mode Excel is in when you hit F4. Unless you are editing a formula, F4 will not change the reference mode, but repeat the last command. (Note: some versions of Excel are not consistent in the functionality to repeat the last command with F4, notably Excel 2007 and chart formatting commands).

cheers, teylyn
