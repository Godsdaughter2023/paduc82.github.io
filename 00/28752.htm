The switch to block automatic processing of recall requests does exist and it's basically the one Diane pointed you to:

Tools | Options | Preferences <Page> | E-mail Options | Tracking Options | Process requests and responses on arrival

From what you are saying I think this switch is working!

When automatic processing is on, a recall request would be immediately processed whiteout you even clicking on it.

When automatic processing is off, the recall will only work AFTER you click on the recall request. More on this here:

How Exchange Message Recalling Works

If you want to check what is being recalled you shouldn't open the recall request message. Just go through the emails in your Inbox first.

Setting up conditional formatting to help compare exam grades (in - Microsoft Community

dianebush say:

Setting up conditional formatting to help compare exam grades (in letters)

I have been asked to set up a spreadsheet in which i compared 2 different sets of grades, the target grade and the current with conditional formatting in place so if the current grade is less than the target it goes red, higher it goes green and so on.  The only issue that i am with this is, that if the grades were numeric values it would be easy, but they are letters i.e. A grade, B grade, and so.  So what i will have in A2 is a target grade of a B and in B2 a current grade of an E and so i want it to go red in the cell B2.  I have looked all over the internet to try and find a solution, which as of yet i have not found.

 

Please help.

 

Many Thanks

e.j. gun say:

This might seem counterintuitive, but you can use the following two conditional formatting rules to get what you want.  I'll explain after!

Assuming Column A has the target grades and Column B the actual grades, then:

Rule 1 (Higher than target grade, set fill color to green)

=B2<A2

Rule 2 (Lower than target grade, set fill color to red)

=B2>A2

 

Why does this work?  Excel will compare the "values" of the letters.  Those "values" are the ASCII character codes.  The character code for A is 65, for B it is 66 and so on.  Therefore, in the numeric sense, B is greater than A, so that the formula must use "less than" to see if the actual grade is higher than the target grade, and vice versa.

 

HTH,

Eric

P.S. - this works for either ALL CAPS or all small letters, but won't work if you compare a capital B to a small a.

 

P.P.S - you can eliminate the CAPS/small issue by modifying the formulas to:

=Upper(B2)<Upper(A2)

=Upper(B2)>Upper(A2)
