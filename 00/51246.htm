  1. Open web browser.

UPnP

Restoring Defaults

Through firmware

  1. Plug the computer into the LAN port of the device.
  2. Open web browser.
  3. Login to router by entering http://192.168.10.1 into the address bar.
  4. If prompted for a username and password use “admin” and leave the password blank.
  5. Select Tools on the left.
  6. Select Backup from the sub-menu on the left.
  7. Click the Reset button.
  8. Click Ok.
  9. Wait for lights on the device to turn off then return to normal.

 

Through Hardware

  1. Plug the computer into the LAN port of the device.
  2. Locate the reset button on the back of the device.
  3. Hold the reset button for at least 10 seconds.
  4. Wait for lights to turn off then return to normal.

 

Updating Firmware

  1. Plug the computer into the LAN port of the router.
  2. Open web browser.

  7. Select Tools on the left.
  8. Select Firmware from the sub-menu on the left.
  9. Select Browse.
  10. Locate the Firmware file that was downloaded and click Open.
  11. Wait for the update to finish before removing the ethernet cord or rebooting the router.

Methods used by large system builders such as Dell, HP etc. are specific and do not apply to small scale manufacturers. If you examine the key applied to these systems they differ from the COA label key on the case. In fact all Dell PCs with the same version of Windows have the same key, they do not pre-install the COA label key on each system. These SLP licenses are pre-authenticated, COA licenses are not.

Small scale builders have to use the individual OEM pack and load the disk using the OPK utility and reset the system to enable OOBE on the next restart. Even if you provide the key in the answer file for each build it will still ask for it during the user setup phase.
