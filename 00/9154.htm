Custom Office 2016 Installation from DVD

1. Download Office 2016 Deployment Tool  - http://www.microsoft.com/en-us/download/details.aspx?id=49117

2. Extract contents of Office 2016 DVD to C:\Archive\Office2016 (UltraISO is good for this).

3. Rename original setup.exe in the Office 2016 DVD to setup.exe-orig.  The old setup.exe
will be replaced with the setup.exe from the Office 2016 Deployment Tool.

4. Copy Office Deployment Tool exe to C:\Archive\office2016 folder.  
NOTE: You can run this deployment tool from any folder, but I chose to copy it to the Office 2016 install subfolder.

5. Execute the Office 2016 Deployment Tool and install it to office2016 folder - a configuration.xml file and a new setup.exe are created.

6. Edit the new Configuration.xml and exclude any undesired applications from the list.
Example xml changes that excludes all applications in Office 2016 Pro Plus but WORD and EXCEL.  If desired, cut and paste the following lines in the Configuration.xml file.

<Configuration>
<Add SourcePath="C:\Archive\Office2016\" OfficeClientEdition="64" >
<Product ID="ProPlusRetail">
<Language ID="en-us" />
<ExcludeApp ID="Access" />
<ExcludeApp ID="Groove" />
<ExcludeApp ID="InfoPath" />
<ExcludeApp ID="Lync" />
<ExcludeApp ID="OneNote" />
<ExcludeApp ID= "Outlook." />
<ExcludeApp ID="PowerPoint" />
<ExcludeApp ID="Publisher" />
<ExcludeApp ID="SharePointDesigner" />
<ExcludeApp ID="Skype" />
<ExcludeApp ID="Skypeforbusiness" />
<ExcludeApp ID="Visio" />
</Product>
</Add>
</Configuration>

6. Go to the C:\Archive\Office 2016 folder and Press SHIFT and right-click in any empty space in folder. Select "Open command window here" to display command prompt.

7. In the command prompt, enter the following command:

c:\Archive\office 2016> setup.exe /configure configuration.xml

If your custom new configuration file is edited correctly - the Office 2016 installation will install only the applications you selected in the configuration file.

P.S. Save this new custom configuration.xml file and setup.exe file so you can use it to create a different custom installation or just replace the setup.exe file on your original Office 2016 DVD.

ken sheridan say: I don't know if you've looked at my demo, but the equivalent code in it is:

    Dim varItem As Variant
    Dim strEmployeeIDList As String
    Dim strCriteria As String
    Dim ctrl As Control
    
    Set ctrl = Me.lstEmployees
    
    If ctrl.ItemsSelected.Count > 0 Then
        For Each varItem In ctrl.ItemsSelected
            strEmployeeIDList = strEmployeeIDList & "," & ctrl.ItemData(varItem)
        Next varItem
        
        ' remove leading comma
        strEmployeeIDList = Mid(strEmployeeIDList, 2)
        
        strCriteria = "EmployeeID In(" & strEmployeeIDList & ")"
        
        DoCmd.OpenReport "rptEmployees", _
            View:=acViewPreview, _
            WhereCondition:=strCriteria
    Else
        MsgBox "No employees selected", vbInformation, "Warning"
    End If

The RowSource property of the lstEmployees control is:

SELECT EmployeeID, FirstName & " " & LastName
FROM Employees
ORDER BY LastName, FirstName;

The autonumber EmployeeID column is hidden by virtue of the control's ColumnWidths property being zero, so the concatenated column of first and last names is what the user sees.  The hidden numeric value is what is used to build the value list, however, so the report is filtered on these values not the names.
