Hi Jesse,

The option in Settings that will allow you to turn on or off including driver updates when you update Windows is no longer available starting Windows 10 build 15019. Since your device is using Windows 10 Creators Update Version 1703 you will longer see it as an option.

In the meantime, there is still an option to do it by using through the Local Group Policy Editor. Please be advised that this feature is only available when you're using Windows 10 Pro, Enterprise, and Education editions. To set this policy, follow these steps:

  1. Click Start, type Local Group Policy Editor, then press Enter.
  2. In the left pane of Local Group Policy Editor, navigate to Computer Configuration/Administrative Templates/Windows Components/Windows Update .
  3. In the right pane on Windows Update, double-click Do not include drivers with Windows Updates policy to modify it.
  4. To enable it, click on the radio button Enabled.
  5. Click OK.
  6. Once done, close the Local Group Policy Editor, then restart your PC.

Hi MegaBoy740,

If you have access to a Windows 7 DVD, try Startup Repair you may also complete the following steps to remove any recently installed Windows Updates. Let us know if this helps.

1.      Boot from the Windows 7 DVD, select your keyboard and language options, select Repair your computer, and then select the Command Prompt. 

2.       From the Command Prompt run the following command:
Dism /image:c:\ /get-packages
This will produce a list of installed updates and packages.  They will look like this example:
Package Identity : Package_for_KB978262~31bf3856ad364e35~x86~~6.1.1.0
State : Installed
Release Type : Security Update
Install Time : 2/11/2010 1:01 PM

3.      Find the most recent one and then make a note of the package identity.

4.      Then run:

dism /image:c:\ /remove-package /PackageName:Package_for_KB978262~31bf3856ad364e35~x86~~6.1.1.0 (you will use the Package Identity from previous step as the PackageName so it may be different from the example)
You will get a status of the removal until it is complete. 

5.      Then reboot your PC.

You will want to start with the most recent and remove them one at a time to find the problem update
