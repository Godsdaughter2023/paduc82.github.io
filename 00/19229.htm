brian tillman [outlook mvp 2007-2019] say:

Don't confuse folders with accounts.  They're not strictly related.

With Outlook 2010, an IMAP account data file cannot be designated the default data file since IMAP cannot support contacts, calendar, tasks, or any non-mail folder.  Outlook requires those folders.  Thus, when you have an IMAP account, you will always have at least two folder sets: the one containing the default folders (Calendar, Contacts, Tasks, Journal, Notes, etc.) and one containing the IMAP account folders.  The default folder set will also contain Inbox, Outbox, Sent Items, and Junk E-mail folders, even though you may not use them because there is no account associated with them.

Your messages should all be in the folders associated with the IMAP account.
