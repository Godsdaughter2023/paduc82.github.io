superken67 say:

I figured out how to do it.  Word doesn't seem to have this option available.  For anyone else interested, it requires Adobe Acrobat Pro to set a "destination."  It's a bit difficult to explain, and I'm sure these directions will need to be adapted to different versions of the programs and different systems but the basic premise should be the same.  

1. First you should have the hyperlinks option enabled for the List of Figures (and Table of Contents or any other table) in the Word document.  To do this, you either toggle the field code of the List of Figures (right-click to find this option) and add \h at the end of the code in brackets, or right click to "edit field," which leads to a window with a "Table of Figures" button.  Click on that to find a "Use Hyperlinks instead of page numbers" box that you must check.  Then update the entire table again.

2. Convert to PDF.  Adobe Acrobat Pro (newer versions) offers an option to create PDF from a Word file, but some versions don't work with Word documents.  Newer versions of Word also have the option to save as a PDF with options to allow the document to be created with bookmarks enabled.

3. Open the PDF in Adobe Acrobat Pro, go to the List of Figures and check to see if you can click on the entries to go to the caption for the figures.  If not, go back and fix the hyperlink option in Word or you'll have to manually add the links in Adobe.

4. Click on a figure in the List of Figures to go to the page with the image.  Then scroll up or down to adjust exactly what section of the page you want the link to display.  In my case, that means scrolling up to include the full image and centering it on the screen. 

5. Once you're happy with the positioning of the page, click on the "Destinations" icon on the left side of Adobe.  You may have to right-click and check the Destinations option for the icon to show up.  It looks like a white, blue, and red target icon.

6. Then click the "Create New Destination" icon, which is the same target icon but with a golden gear/star/shape on top.  Give it a name that matches the figure you are linking.

7. Now go back to the List of Figures page and click on the "Tools" button on the Right.  Then select "Content," and finally, under "Add or Edit Interactive Object," choose "Link."  Now you should be able to select each hyperlink when you hover the pointer over an entry.

8. Right-click the figure entry for which you just created a destination and select "Properties..."  Then click on the "Action" tab to alter the destination of the link.  You'll click "edit," then "Use Named Destination."  Hit "Browse" and select the destination you created with the name of the figure you are linking.

9.  Then press ok for each window until you can test the new hyperlink to make sure it displays the page exactly as you intended.  Go back and repeat these steps for all figures you want to relink to a new destination.

The "set destination" option can also be used for Table of Contents links.  Bookmarks work the same and are even easier because you can set their destinations by right-clicking on the individual bookmark and selecting that option.  For the problem of the hyperlink boxes showing up on Macs having a black outline, you can right click on the link in Adobe, choose "Properties...," then make the box visible and change the color to white (for a normal black text on white paper document).

Hope this helps someone!

protecting word document from accidental editing - Microsoft Community

jsquareg say:

protecting word document from accidental editing

I have been using Office since, I think Office 97, mostly for Excel and Outlook with very little use of Word.  I used Word for a few letters and no report like documents. I am now at Office 365 level and find I now have to write reports or other such documents.  It is too easy for me to inadvertently  make changes to a report I  have just written and am viewing it.  Is there a way I can prevent me from over typing, deleting or otherwise making changes due to unintended key strokes.  Maybe something like make the report editable except when I want to deliberately nake a change?  (disable editing/enable editing.)

Thank you for help educating this Word newbie.

jay freedman say:

With the document open in Word, click the File tab. The Info page should appear, but click Info in the left column if another page appears. Now you can do either of the following:

  • Click the Protect Document button, click the Mark as Final choice, and click OK in the prompt that follows.

Click the leftward arrow in the top left corner or press the Esc key to return to the document. This banner will appear at the top of the document:

As long as you don't click the Edit Anyway button, any keys that would otherwise change the text will only display this message in the status bar at the bottom of the window:

When you finish your review, click the Edit Anyway button and save the document, which returns it to its ordinary condition.

  • Alternatively, on the Protect Document menu, click the Restrict Editing choice. The document reappears, along with the Restrict Editing pane on the right. (This is also available from the Restrict Editing button on the Review ribbon.)  Check the box under the heading "2. Editing restrictions" and then click the "Yes, Start Enforcing Protection" button near the bottom.

Click OK in the dialog box that opens -- don't change anything else. Stray keystrokes will display the same message in the status bar at the bottom of the window as shown above. When you finish the review, click the Stop Protection button at the bottom of the Restrict Editing pane and save the document.
