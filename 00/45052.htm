a. user say:

Hi,

 

Thank you for choosing Windows 8 and posting this question on Microsoft Community Forums.

 

I see that you get an error message: Documents.library.ms is no longer working (Documents.library-ms ne fonctionne plus- French).

 

Let us work together to resolve this issue.

 

Here’s what you can do:

Let us delete the Documents Library and restore it again. The library can be safely deleted. Folders that have been included will not be affected.

 

Follow the steps below to delete the Documents Library.

 

a.     Open File Explorer and right-click on Documents Library.

b.     Select Delete.

c.      Click Yes when asked for confirmation.

 

Note: Deleting a library does not delete the contents of the included folders. However, you may need to re-add or include your folders manually, if needed.

 

Follow the steps below to restore the Documents Library.

 

a.     Open File Explorer.

b.     In the Navigation Pane, right-click on Libraries and select Restore Default Libraries.

 

This will restore the Documents Library. This option only restores Default Libraries.
