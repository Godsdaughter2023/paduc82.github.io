Once I noticed that, I ungrouped the sheets and then was able to cut and paste with no problems. If you don't know to tell if sheets are grouped just look down at the bottom of your excel and you should see tabs labeled Sheet1, Sheet2, Sheet3...or personalized with labels someone else has set. If more than 1 tab is white (the others might be a pale blue) then that means the white ones are grouped together. Right click on a white tab and select "ungroup sheets" and voila - should be able to cut & paste with no problems. Sure hope this helps!

suzanne s. barnhill say:

You will need to use a different style for the text at each level. The built-in List Continue styles are designed to work with the built-in List, List Bullet, and List Number styles and have progressively larger indents. So, for example, you follow List Number with List Continue, List Number 2 with List Continue 2, etc.

Since you're using indented headings, then you can make the appropriately indented text style the "Style for following paragraph" for the given heading style. For example, Heading 1 would be followed by, say Body Text, Heading 2 by Body Text 2, and so on, with each body text style modified to include the required left indent.

mahayv say:

Hi,

 

In the majority of these instances, the following procedure will fix the issue:

 

From the main menu (and with the SIM Card still inserted), select Settings.
Select Configuration.
Select Preferred Access Point.
Select a listed access point, either WAP or Internet.
Go back to the main menu.
Select Messaging.
Open Email and proceed.


If a Communication Error still occurs, please follow these steps:

 

Go to your Network Operator's support page and request the Access Point configuration settings for WAP and Internet (GPRS).  The Operator might be able to send these settings to your phone via a text message.

 

From the main menu on your device, select Settings.
Select Configuration.
Select Personal Config Sett..
Select My Access Point.
Enter a name for your access point and input the settings obtained from your Network Operator for WAP settings.  The settings may already exist.  Please ensure the information is correct and create a new access point name if the Operator's most current settings are not in the list of specified access points.

 
For the Internet (GPRS) settings:

Enter a name for your access point and input the settings obtained from your Network Operator for GPRS  settings.  The settings may already exist.  Please ensure the information is correct and create a new access point name if the Operator's most current settings are not in the list of specified access points.
Select the newly configured access point name (either WAP or Internet or both).
Try connecting to Nokia Messaging Email again.
