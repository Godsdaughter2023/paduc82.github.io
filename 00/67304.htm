http://windows.microsoft.com/en-in/windows7/open-the-hardware-and-devices-troubleshooter

Method 2: Let’s remove and reinstall all USB controllers:

To remove and reinstall all USB controllers, follow these steps:

a. Press Windows key + R, type sysdm.cpl in the Open box, and then click OK.

b. Click the Hardware tab.

c. Click the Device Manager button.

d. Expand Universal Serial Bus controllers.

e. Right-click every device under the Universal Serial Bus controllers node, and then click Uninstall to remove them one at a time.

f. Restart the computer, and then reinstall the USB controllers.

g. Plug in the removable USB storage device, and then test to make sure that the issue is resolved.

Method 3:

You may check if the Windows Audio End Point Builder service is running correctly.

To do this, follow these steps:

a. Click Start, Run and type ‘services.msc’ in the open box, click OK.

b. In Services window, double-click on Help and Support.

c. Set the Startup type to Automatic & start the Service.

Method 4: Try loading the downloaded drivers for the USB Somic G927:

  1. Press Windows key + R, Type "Devmgmt.msc" and press enter.
  2. Look for the Somic G927, if it's not recognized by Windows, then it will most probably be listed as an unknown device.
  3. Right click the entry for the USB Headset G927
  4. Click on Update driver software
  5. Select, Browse my computer for driver software
  6. Then select, Let me pick from a list of device drivers on my computer
  7. Choose the Somic G927 and then click Next
  8. The driver for the Somic G927 should be installed
