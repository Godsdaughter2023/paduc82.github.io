Hi jmeush, because of the age of that system and indeed the integrated graphics on your i3 processor, it would be best to roll back the update or reset Windows and then turn off Automatic updates, that system is not going to be compatible going forward with Windows 10 updates

1.
Roll back Windows or Reset Windows:

Start your PC, just as Windows attempts to load (spinning dots), press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down

Do this twice

On the third start Windows will boot into the Recovery Environment and from there you can access System Repair, Safe Mode, Command Prompt . . . etc.

Go to Troubleshoot - Advanced Option - Revert to previous build of Windows

If the above fails, go to Troubleshoot - Advanced Options - Reset this PC
Choose to keep your files and apps,
Then start the Reset



2.
Disable Windows Updates permanently:

Click your Start Button, type services and hit Enter

Scroll down to find Windows Update Service

If it is Started, double click it and stop that service

CLose the services App

Click your Start Button, type cmd, right click Command Prompt and choose 'Run as Administrator'

Run this command and hit Enter

sc delete wuauserv

Close Command Prompt

Restart your PC


If you also have the Update Assistant installed on your system:

The Update Assistant in Windows 10 cannot be uninstalled, but you can get rid of it

Click your Start Button, type Task Scheduler and hit enter
In the left Pane, navigate to: Windows - Update Orchestrater
Right Pane double click 'Update Assistant'
Click on Triggers Tab
On each of the triggers ( At login . . . etc.) double click and uncheck 'Enabled'

Restart your system, that's It!

    a. user say:

    Hi,

    The Software Distribution folder is used by Windows Update Service to store the components that it will use to update your computer. You have the option to move this folder to Drive C or another location but it involves the following procedures in their correct order.

    1. Temporarily stopping Windows Update Service using Command Prompt.
    2. Moving the Software Distribution folder to your preferred location.
    3. Enabling Windows Update Service again using Command Prompt.
    4. Updating Windows Update Service about the new location of the Software Distribution folder.

    If Windows Update Service will not be updated about the new location of the folder, it will create a new one on the same location.

    These procedures can be complex. For in-depth support in performing these, we recommend posting your query on our Microsoft TechNet forum. Here's the link.

    DHCP - Microsoft Community

    shamssultan say:

    DHCP

    What is DHCP ?
    hairyfool say:

    Dynamic Host Configuration Protocol

     

    All devices require an address to be able to communicate on a network. The values required are normally different for each network. Instead of having to change them when moving a device between networks DHCP allows a device to ask what settings to use and a DHCP server will issue the required settings.

     

    There are additional benefits for network managers to use DHCP even when dealing with static devices that are not being moved around.
