a. user say:

Hi,

Thanks for posting your query to Microsoft forum. I will certainly help you with this.

You can transfer the local music files on your computer from the Music Library.

Follow these steps:

  

1. Attach the portable MP3 player to your PC.
2. Open Windows Media Player, if necessary.

3. The AutoPlay dialog box may prompt you to open Windows Media Player when you initially attach the device, or you can manually start Windows Media Player.

4. The portable device appears in the upper-right portion of the Windows Media Player window. 

5. Drag to the list pane the music you want to copy to the portable device. 

6. This step works just like creating a playlist. In fact, you can drag playlists to the device in the same manner.

7. Click the Start Sync button to copy music to the portable device.

8. Disconnect the device. 

 

Your portable MP3 player comes with special software for managing music. Although these steps get you through basic music synchronizing using Windows Media Player, you might still need to use the MP3 player’s extra software to help manage your music, remove tunes, or organize how the music is played.

gerry c j cornell say:

Steve

Microsoft hides these files because if they get deleted the system will not work properly. As long as you appreciate that system files should, in normal circumstances, not be deleted then there is no reason for them to be hidden.

To perform system maintenance it is inconvenient to have system files hidden. An example is BSOD dump files. If you experience a BSOD error and want help diagnosing what is causing the error, you can only provide a copy of the file if you can see the file to obtain a copy to pass to whoever is helping you.

The justification for Microsoft hiding the files is that many computer users cannot be trusted not to delete files when free disk space becomes limited, regardless of the type of file and consequences.

If files are unhidden they are just as easily restored to being hidden, by rechecking the boxes that were checked.
