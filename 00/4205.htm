Default Dictionary on Word for Mac 2016

bob jones aka: cybertaz say:

What you're doing does not change the "default dictionary" - there is only 1 main dictionary file used by Office software. That dictionary includes the lexicons for all languages supported by the software. Which lexicon it uses is determined on the fly by the Language attribute of the text being checked.

What you're doing is one way you can go about changing the Language attribute, but to do that, text has to be selected. Otherwise, picking a different language changes only the Language attribute of the current spot where the insertion marker sits.

Additionally, Language is not a global setting for a document. Parts of the content can be in one Language, other portions in other Languages. When you set the Default Language it merely determines the Language attribute used for documents you create from that point forward. It does nothing to change the Language attribute of the current document unless you have text selected as mentioned above - and only the selected text is affected. Language also can be determined by the Character & Paragraph Styles used for formatting documents.

English (AUS) is one of those which is supported. You should have no need for a Custom Dictionary unless you use technical, medical, atypical etc. terms which understandably would not be found in even a comprehensive bookshelf dictionary.

For more detail please see:  Spelling, Language, Dictionary
