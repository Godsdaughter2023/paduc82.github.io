Outlook 4266 does NOT support API set 1.4

It is not enough to update Outlook to pick up 1.4, you must also update osfshared.dll to pick up 1.4 support. 

This is the Sept 2017 KB update to Osf: 
https://support.microsoft.com/en-us/help/3191923/september-5-2017-update-for-office-2016-kb3191923

Here is the Oct 2017 Outlook KB: 

https://support.microsoft.com/en-us/help/4011162/description-of-the-security-update-for-outlook-2016-october-10-2017

There was also a bug in Outlook that caused Outlook to revert to 1.2 in the Feb 2017 Update (https://support.microsoft.com/en-us/help/3141511/february-7-2017-update-for-outlook-2016-kb3141511) 

This was fixed in the April 2017 Update: (https://support.microsoft.com/en-us/help/3178664/description-of-the-security-update-for-outlook-2016-april-11-2017

However, from your description it sounds like you are probably hitting the former issue (Osfshared.dll not being update)
