manuelcapiro say:
Hello ZORROGIRL,

To restore this option

Open the Run Command box pressing Windows+R keys and type %systemdrive%\Users\Default\AppData\Roaming\Microsoft\Windows\SendTo in the command box and click OK.

Now copy the Compressed (zipped) folder from this folder.
If you cannot find the file in this folder then you need to download the Compressed folder.zip file manually.

After obtaining the Compressed(zipped)folder , open the Run command box by pressing Windows+R keys and type theshell:sendto command in the command box and click OK.

Paste the previously copied Compressed (zipped) folder in this folder.

You can right click on any file and select the Send to menu and you will find the Compressed (zipped) folder option. Now you can create Compressed (zipped) folders.

Hope it helps, have a good day
