Hi,

UNEXPECTED_STORE_EXCEPTION and CRITICAL_PROCESS_DIED blue screen errors can be caused by a variety of hardware, firmware, driver, or software issues. Examples of the issues are:

  • Old or corrupted device drivers.
  • Virus or malware infection that has corrupted Windows system files.
  • Driver conflict after installing new hardware.
  • Damaged hard disk.
  • Memory (RAM) corruption.

To resolve this issue, we strongly recommend running both the DISM and SFC commands shown below:

  1. Open an elevated command prompt. To do this, right click the Windows button, and then select Command Prompt (Admin). If you are prompted for an administrator password or for a confirmation, type the password, or click Allow or Yes.
  2. Type the following command and press Enter.
    • sfc /scannow 
    • Note: It might take several minutes for the command operation to be completed.
  3. Type the following command, and then press Enter.
    • DISM.exe /Online /Cleanup-image /Restorehealth
    • Note: It might take several minutes for the command operation to be completed.
  4. Close the command prompt, and then restart your device.
