If this software doesn't start correctly, you might receive this error.


 

 Method 1:


Let`s try with these steps first.
 
 
1.       Remove the DVD from the DVD drive and then reinsert it. This will force the copy protection software to reset.
 
2.       While the DVD is out of the drive, make sure that your DVD is clean. If your DVD is scratched or dirty, the DVD drive may not be able to read it correctly, and you might receive this error.
 
Method 2:
 
If this doesn't work, try the following:
 
 
1.       if you have been able to play DVDs on your computer before, and you receive this error message, you might be able to solve the problem by restarting Windows Media Player or your computer.
 
2.       If you receive this error message the first time you try to play a DVD, you might not have a DVD decoder installed, or your decoder software might need to be updated.
