1     Just install Open Office on your new computer.  You can save yourself a lot of bother by doing this even if all you are going to do is use it to convert the files so Word & Excel can read them.  There are third-party converters available but I tried a lot of them whilst helping out somebody and not a single one produced good results.

2     If you intend to use MS Office Word & Excel instead then change the format of the files by opening them in Open Office.  Set up Open Office's default formats using this procedure:-

2.1       In the menu bar at the top of the application’s window, click on the Tools menu then select Options.

2.2       On the left hand side of the dialog that appears, click on the little plus sign next to Load/Save.

2.3       Select the General entry that now appears underneath Load/Save.

2.4       Look at the Document Type section on the right-hand side. 

2.5       Set up text document formats.  Under the words Document Type, click on the little dropdown arrow and select Text document.  To the right, under the words Always save as, click on its little dropdown arrow and select Microsoft Word 2003 XML.

2.6       Set up spreadsheet formats at the same time.  Under the words Document Type, click on the little dropdown arrow and select Spreadsheet.  To the right, under the words Always save as, click on its little dropdown arrow and select Microsoft Excel 97/2000/XP.

2.7       Click on OK.

2.8       Click on the Save button [if it is greyed out then it means you haven’t changed anything so just type any old rubbish anywhere you want first].

2.9       An annoying little dialog box appears.  Click in the checkbox next to Ask when not saving in ODF format to clear the checkbox then click on Keep current format to close the dialog.

2.10     That is all you need to do.  When you open Open Office files in the future, they will save in doc and xls formats without any special action being needed.

3     You can work through all your existing Open Office documents to change them all into Word- & Excel-compatible formats.  When you have done that, open them in Word & Excel then save them again to be on the safe side.  If you are using Word & Excel 2007 or later then you will need to make sure you save each of them in .docx & .xlsx format [Open Office will just have used the older Word .doc & Excel .xls formats].  Once you have done all this you can uninstall Open Office if you will not be using it again.

4     If you run into any problems then search the Office section of this forum [your current post is in the Windows 7 section because you were also asking about Easy transfer].  If you cannot find the help you need then post your problem in the Office section rather than here.  I would imagine that there are lots of Office users who have played around with Open Office at some stage and would therefore be able to help you.

General Recommendations For Setting Up Users In Windows 7

You absolutely do not want to have only one user account. Like XP, Vista and all other modern operating systems, Windows 7 is a multi-user operating system with built-in system accounts such as Administrator, Default, and Guest. These accounts should be left alone as they are part of the operating system structure.

You particularly don't want only one user account with administrative privileges in Windows 7 because the built-in Administrator account (normally only used in emergencies) is disabled by default. If you're running as Administrator for your daily work and that account gets corrupted, things will be Difficult. It isn't impossible to activate the built-in Administrator to rescue things, but it may be more work than you want to do. Better not to put yourself into a bad situation to begin with.

The user account that is for your daily work should be a Standard user, with the extra administrative user (call it something like "CompAdmin" or "Tech" or the like) only there for elevation purposes. Running as a Standard user is best practice for security purposes and will help protect your computer from infection. After you create "CompAdmin", log into it and change your regular user account to Standard. Then log back into your regular account.

If you want to go directly to the Desktop and skip the Welcome Screen with the icons of user accounts, you can do this:

Start Orb>Search box>type: netplwiz [enter]
Click on Continue (or supply an administrator's password) when prompted by UAC

Uncheck the option "Users must enter a user name and password to use this computer". Select a user account to automatically log on by clicking on the desired account to highlight it and then hit OK. Enter the correct password for that user account (if there is one) when prompted. Leave it blank if there is no password (null).
