If you can still see the key, try the following:

Click Start, type: cmd

Right click CMD and click Run as administrator

at the command prompt, type the following:

slmgr -ipk xxxx-xxxx-xxxx-xxxx (allows you to replace the current product key with the specified)

xxxx-xxxx-xxxx-xxxx represents the product key on the COA, enter those

Press Enter on your keyboard

Restart your system and attempt activation again

If activation does not work, try the following:

 

Click Start, type: cmd

Right click CMD and click Run as administrator

at the command prompt, type the following:

slmgr -ato (activates current pc’s license)
