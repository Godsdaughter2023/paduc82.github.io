Check for an outdated webcam driver

Select the Start button, type Device Manager, and then select Device Manager from the search results.
Find your webcam under Imaging devices or Sound, video and game controllers.
Press and hold (or right-click) the name of your webcam, and then select Properties.
Select the Driver tab, select the Driver Details button, and look for a file name that includes stream.sys. If it’s there, your webcam was designed before Windows 7 and you'll have to replace it with a newer webcam.
If you don't find a file name that includes stream.sys, try rolling back your webcam driver.

Roll back your webcam driver

In Device Manager, press and hold (or right-click) your webcam, and then select Properties.
Select the Driver tab, select Roll Back Driver, and then choose Yes. (Note that some drivers don’t provide a rollback option.)

If rolling back didn’t work or isn’t available, uninstall your webcam driver and scan for hardware changes.

Uninstall your webcam driver and scan for hardware changes

In Device Manager, press and hold (or right-click) your webcam, and then select Properties.
Select the Driver tab, select Uninstall > Delete the driver software for this device, and then select OK.
In Device Manager, on the Action menu, select Scan for hardware changes. Wait for it to scan and reinstall updated drivers, restart your PC, and then try opening the Camera app again.
If your webcam still doesn’t work, try looking for help on the webcam manufacturer’s website.

    andre da costa say:
    If Windows 7 came preinstalled on your computer, there should be a COA sticker at the bottom or in the battery compartment that will tell you if it is a OEM license.

    COA Certificate of Authenticity:
    http://www.microsoft.com/howtotell/content.aspx?pg=coa

    Your product id can also identify if your license is OEM.
    Click Start, right click Computer, click Properties

    Scroll down to Windows Activation

    Take note of your product ID, it will look something like the following:
    00000-OEM-0000000-00000

    The PID should have OEM in it.

    Computers, which are built by large manufactures that come with Windows Pre-Installed, come with two (2) Product Keys:

    A)   OEM SLP: This key comes pre-installed in Windows, when it comes from the Factory. This key is geared to work with the OEM Bios Flag found only on that Manufacturer's computer hardware. So when Windows was installed using the OEM SLP key (at the factory) Windows looks at the motherboard and sees the proper OEM Bios Flag (for that Manufacturer and that version of Windows) and Self-Activates. (that's why you did not need to Activate your computer after you brought it home)

     B)    COA SLP: This is the Product key that you see on the sticker on the side (or bottom) of your computer. It is a valid product key, but should only be used in limited situations (sush as if the OEM SLP key stops self-activating for whatever reason). The key must be activated by Phone. (Note: All manufacturers that use the OEM SLP system are required by contract to include a Certificate of Authenticity (COA) sticker, that has a COA SLP key, on the computer)

    shenan stanley say:

    If you have a product key for Windows that came with that laptop, you can format and install with that same product key on that same laptop as many times as you wish.  You just cannot transfer that product key (license) to another system.  If it came pre-instaleld on the laptop (and especially if there is a product key sticker on the laptop itself) then it is an OEM product.

    If you purchased Windows 7 yourself (retail/upgrade) and you remove it from the laptop (format the laptop) - then you can use it on another machine.  It might entail a phone call to get things sorted, but the license is transferrable because it is retail and you bought it separate from the laptop.

    So - direct answers....

    Hi, If reformat my notebook, can i reuse my license key?

    Depends on what you mean by "reuse my license key" (where?) and the license type.

    It's Asus K52j notebook pc. Running Windows 7 Home Premium, can i use the previous original license key?

    What do you mean by, "can i use the previous original license key?"?  Was this machine upgraded with Windows 7 or is it now in-use as it was shipped - with the original operating system that likely matches the product key sticker on the bottom (or someplace) on the laptop?

    And how to check the Windwos 7 is OEM Version or what ?

    Did the edition of Windows currently installed on the laptop come *with* the laptop?  Is there a product key sticker on the laptop for that version of Windows?

    And how to check the license key?

    If your current version of Windows is still running - you might try to use Belarc Advisor to get the product key that was entered on the machine at installation/activation time.  It may/may not match the one on the sticker on the machine itself.

    ramesh.kumar say:

    Hi,

    Which file you are trying to mount? 

    Method 1

    You can also try the following steps to mount VHD

    1. Open the Start menu
    2. Right click on Computer and select Manage
    3. Expand Storage and click on Disk Management
    4. Click on the Action menu and select Attach VHD
    5. Enter the Location and name of you virtual hard disk (there is a browse button you can use)
    6. Click OK

    And you are done - simple! To unmount the virtual hard disk you just need to right click on Disk entry for the virtual hard disk and select Detach VHD. 

    Method 2

    The other option is to use diskpart. To do this you will need to:

    1. Open up an administrative command prompt.
    2. Run diskpart
    3. Type in SELECT VDISK FILE=insert your VHD file path and name here
    4. Type in ATTACH VDISK

    When you are done you can unmount the VHD using the DETACH VDISK command under diskpart.

    For reference: http://technet.microsoft.com/en-us/magazine/ee872416.aspx
