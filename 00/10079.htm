macropod say:

Now that I know what you're trying to do, simply add the following macro to your mailmerge main document. It replaces the

Sub MailMergeToDoc()
Application.ScreenUpdating = False
Dim bFit As Boolean, i As Long, j As Long
ActiveDocument.Mailmerge.Execute
With ActiveDocument
  For i = .Tables.Count To 1 Step -1
    With .Tables(i)
      bFit = .AllowAutoFit
      .AllowAutoFit = False
      For j = .Rows.Count To 1 Step -1
        On Error Resume Next 'skip vertically merged cells
        With .Rows(j)
          'ChrW(8194) is the formfield space character. Trim erases any other spaces
          If Trim(Replace(Replace(Replace(.Range.Text, Chr(13) & Chr(7), ""), _
            ChrW(8194), ""), Chr(160), "")) = "" Then .Delete
        End With
        On Error GoTo 0
      Next
      On Error Resume Next 'skip deleted tables
      .AllowAutoFit = bFit
      On Error GoTo 0
    End With
  Next
End With
Application.ScreenUpdating = True
End Sub

Having problems with “Hey Cortana” hearing me - Microsoft Community

jason d [msft] say:

Having problems with “Hey Cortana” hearing me

Build 9926 does not automatically set the gain for microphones.  This issue will be addressed in a future release.  For now, you can manually adjust the gain by following these steps: 

  1. Right click on the speaker icon in the lower right hand corner of your desktop.

  2. Select "Recording Devices".

  3. Select your main input microphone.

  4. Speak into the microphone

  5. Note the sound level meter.  You want the level meter to peak every so often at the top of the bar but still show good bounce during your spoken text. 

  6. Ensure the microphone input is still selected and click properties.

  7. Click the Levels tab.

  8. If you have a microphone boost set it in the middle.

  9. Now adjust the microphone gain by moving the slider. 

  10. Click “Apply” and verify that the level meter is responding as described in step 5.
