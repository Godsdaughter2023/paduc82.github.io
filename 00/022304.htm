russ valentine say:

Many answers already posted here on that problem. The first thing to try is simply clearing the autocompletion cache in File > Options > Mail > Send Messages > Empty Autocompletion List. Then restart Outlook.

If that doesn't work, Microsoft Office Help files recommend closing Outlook and renaming your RoamCache folder:
 <drive:>:\Documents and Settings\<UserName>\Local Settings\Application Data\Microsoft\Outlook\RoamCache
 
Unfortunately, this solution just blows away your existing cache in the hopes you will be able to be happy to start from scratch to rebuild your cache. Many users aren't. Other users with this problem have reported that if they simply open their most recent Stream_Autocomplete.dat file that resides in the RoamCache folder using an NK2 editing program such as NK2edit, their problems are fixed without losing their existing cache. I can only pass this along without confirmation because I haven't encountered the problem nor tried to correct it myself. Since the Stream_Automplete file opens in Notepad by default, even just opening the file in NOtepad might work.

It is clear that Microsoft still doesn't know how to make this feature work reliably or consistently in any versions.
