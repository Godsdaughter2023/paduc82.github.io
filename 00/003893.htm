Windows 10 Party chat problem - Microsoft Community

wolfarc29 say:

Windows 10 Party chat problem

I've been having a problem with the Xbox app. I cant be in a party chat because of "internet problems". Every time I join a party with another person it says "your network settings are blocking party chat". Is there anyway I can fix this? 
bishopm say:

I will say to try this first Windows 10 Party Chat help and if it does not work or you have already tried it with no luck like me, then follow the steps below:

I was having the same problem.  After much aggravation, a lot of reading and trial and error, I was able to get mine working.  I tried every solution out there and nothing worked for me but what I did was find out is that my Teredo tunneling adapter was "offline" and no matter what I did, I could not get it to go online even after following the instructions of several web pages.  I tried reinstalling but that only helped for a short while until Windows did an update again and then it went back offline.  Anyways, first check and see if your Teredo tunneling adapter is offline and if it is, do this simple but annoying step.

To check and see if it is offline:  Right click on the "Start Menu" Windows app in the bottom left hand of your screen.  When the menu comes up, left click on the "Command Prompt (Admin)".  Once the command prompt is up type "netsh" and then hit enter.  It should change from "C:\Windows\system32" to "netsh".  Next, type "interface" and hit enter.  Once you hit interface a bunch of info with populate the screen starting with something along the lines of "In the future version of Windows...".  Don't worry, you don't have to read or pay any attention to that.  I am just telling you so that you don't think you did it right.  Finally type "teredo" and hit enter.  When that is done, you should see "netsh interface teredo>"  Once you have that up, type "show state" and the teredo parameters will show you the info about the tunneling adapter.  If you look at it and somewhere in that small bit of info you see it stating that it is offline then this is what you do to get it working again.  You might not like it but it is the only thing that I found to work and believe me, I tried a lot of things for weeks.

You have to reinstall Windows 10.  DO NOT ADD, INSTALL, OR LAUNCH ANY WINDOWS APPS OR PROGRAMS except to Activate Windows.  Once you have reinstalled Windows but BEFORE you update it, check the teredo tunneling adapter to see that it is working now by doing the steps listed above.  If it is, and you will know if it is because the information will be different than when you first checked it.  Don't worry if it says "Restricted"  in the NAT, it will still work.  Once you have established that it is working, type "set state client".  If it worked and you do "show state" again, the NAT Special Behaviour will change the PortPreserving from "No" to "Yes". 

Next, all you do is update Windows like normal but again do NOT launch or install any programs.  I accessed Windows Update by typing "Windows Update" in Cortana.  Once ALL the updates are done, you are free check the Xbox App and see if you can join and participate in party chats.  I hope this helped all of you who are struggling with it like I was.

    Application.FollowHyperlink Me.txtFileName

bob jones aka: cybertaz say:
No, it's not a bug, it's a faulty "Expected result" :-) The simplified explanation is...

When you "copy a slide" you're not actually copying the slide, you're only copying what is actually to a slide provided by the target file so it uses whatever Background is in effect based on the Slide Master of the target file.

If you want to have the slide appear as it does in the source file use the method designed for the purpose:


Be sure to choose the option to Select slides to insert & then use the check box in the Keep design of original slides. The specified slide will be inserted into the target presentation exactly as it appears in the source & its design elements (Slide Master) will be added as well.
