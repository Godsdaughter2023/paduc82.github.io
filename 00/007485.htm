However, failing that, you could make a suggestion for that at:

http://word.uservoice.com/

If you select the range of paragraphs that you want to be numbered in that format and run a macro containing the following code, it will insert the required ordinal number and a tab space before each paragraph and format the paragraph with a hanging indent.

Dim arrNumbers As Variant
Dim i As Long
Dim rng As Range
arrNumbers = Split("Primera Segunda Tercera Cuarta Quinta Sexta Séptima Octava Novenoa")
Set rng = Selection.Range
For i = 1 To rng.Paragraphs.count
    With rng.Paragraphs(i).Range
        .Text = arrNumbers(i - 1) & vbTab & .FormattedText

        .ParagraphFormat.TabHangingIndent 2

    End With
Next i
Exit Sub

Note, my assumptions in respect of the first nine numbers may not all be correct.  If you need to extend the range of numbers that can be used, insert the necessary numbers before the closing quote (") separating each from the preceding number by a space
