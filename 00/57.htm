This should be fairly easy.

First, let's assume you name each of the textboxes that contains scores as you've suggested.  HomestuckScore etc.

Sub ClearScores()

' Clears all of the scores to start with

' Run as needed or assign an action setting to fire it off

With ActivePresentation.Slides(1)

  .Shapes(HomestuckScore).TextFrame.TextRange.Text = "0"

  ' repeat with the other text names

End With

End Sub

Name the +5 button for Homestuck as 5_Homestuck

Name the +1 button for Homestuck as 1_Homestuck

Sub AddToScore(oSh as Shape)

  With ActivePresentation.Slides(1).Shapes(Mid$(oSh.Name,3) & "Score")

    .TextFrame.TextRange.Text = CStr(Clng(.TextFrame.TextRange.Text)+Clng(Mid$(oSh.Name1,2)))

  End With

End Sub

Assign each of the plus buttons an action setting of Run Macro:  AddToScore

The macro gets a reference to the clicked shape in the oSh parameter, then from its name, figures out which score to increment and by how much.  One Macro To Rule Them All.
