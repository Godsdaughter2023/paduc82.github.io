a. user say:

Yes, you can wake your computer from a power saving state, but you'll need a scheduled task to do that. Fortunately, Windows has several such scheduled tasks built in.

Windows Updates install in a particular way. If you have the Home Edition then there's no way to schedule when an update will be installed. If you have the Professional edition you can schedule Active Hours, which tells Windows when not to install updates. You can set aside up to 18 hours - might be longer by now - as Active Hours. For most people, Active Hours don't include late at night to early morning.

Outside Active Hours Windows will automatically wake your computer with one of the built in scheduled tasks.

It sounds like you might be managing a network. Network administrators have a completely separate arsenal of tools for waking computers, not only for Windows Updates, but for many maintenance tasks. You can wake a computer for PM even if the user powered off.
