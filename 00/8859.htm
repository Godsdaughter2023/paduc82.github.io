doug robbins - mvp office apps & services (word) say:

Try just using the one macro to toggle the protection, and specify the parameter (NoReset) for which you are providing the argument.

Sub ToggleProtection()
With ActiveDocument
    If .ProtectionType = wdNoProtection Then
         .Protect wdAllowOnlyFormFields, NoReset:=True

    Else
        .Unprotect    End If
End With

End Sub
