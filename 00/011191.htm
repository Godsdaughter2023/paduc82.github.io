Write both names as appropriate to DocVariable Name1 from the userform e.g.

Let's assume you have two text fields in your userform txtName1 and txtName2

If only one txt1Name field is filled then

ActiveDocument.Variables("Name1").Value = txtName1.Text

ActiveDocument.Variables("Name1").Value = txtName1.Text & " or " & txtName2.Text

If Not txtName1.Text = "" then

MSE icon missing from notification area - Microsoft Community

george632 say:

MSE icon missing from notification area

I just switched my mouse and it required downloading a new driver. After that the MSE icon did not show up in the notification area, I have been using MSE for several months now. I switched from Norton and it was completely removed from Windows 7.   Anyone have any suggestions.
kosh vorlon say:

Hi George,

 

I'd first like you to simply shutdown the PC normally and then reboot. You'd be surprised how many people with this issue said just doing that did the trick.

 

Next, please review the following thread: http://answers.microsoft.com/en-us/protect/forum/protect_start/no-icon-in-taskbar/82e7d541-84fd-4bf9-bea9-677a3c121e7f. Pay attention to ALL of the suggestions - not only the one marked as the answer (as that can sometimes be misleading and some ignore all but the answer to eventually find the solution to THEIR problem was clearly documented in the second response or on the third page of the thread).

 

***************************************************************

 

If that doesn't work, please try the following (which may repeat some of the above thread, but you'll know if you tried it before or not and if you did and did so EXACTLY as indicated then there's no need to repeat the procedure):

 

Since MSE places a status icon in the Notification Area to the right of the taskbar, to open MSE all you need to do is either double-click that icon or click it once and then click the small "Open" box that will appear over the icon. If you can't see the icon, then it may be hidden. To resolve that, right click on the Start orb and click on properties. Go to the Notification Area tab. Find the MSE icon and click on the dropmenu to the right of it to change the option to "Show." After the next time you reboot, it will always show in the Notification area. If it doesn't appear in the list or doesn't show at all, then please try the following:

 

There should be an entry in MSCONFIG for MSE. Go to the icon you use in the Start menu (it should be in Start / All Programs / Microsoft Security Essentials). Right click on it and click on properties to see the location. It should show the path to a file called msseces.exe. Make a note of that path.

 

Now go to Start / Run (or Start / All Programs / Accessories / Run or if using Vista or W7, go to Start and then the Search box) and type Regedit and enter. Go to HKEY_LOCAL_MACHINE\Software\Microsoft\CurrentVersion\Run and click on run to see the entries. Is there one that references the above file?

 

It should have the name MSC and be a string value and the command should say: "C:\Program Files\Microsoft Security Client\msseces.exe" -hide -runkey exactly as I wrote it with the quotes and spaces. If the path you noted was different from the path I listed, use the path you noted instead, but otherwise keep the quotes and the extra words with dashes in front of them at the end. If it exists but is different, then change it as indicated. If it does not exist, create a new string value under Run with those parameters.

 

Now reboot and see if it works to show the MSE icon in the notification area. If you needed to alter the registry to get it to appear, then you probably need to do the first paragraph again to make sure it remains visible all the time.

 

********************************************************************

 

I hope this helps. If not, post back with results of what happened and we'll see if we can find another solution (though it's likely to either be extensive or a referral to Microsoft Support for expert help - or both).

 

Good luck!

 

Kosh
