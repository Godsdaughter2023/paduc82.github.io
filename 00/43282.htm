considerate_guy say:

Let me ask you this:

  • If you create a project with photos, are you able to see them in the final movie?
  • If you create a project with a .wmv as the source video, are you able to see it in the final movie?
  • Is the only time you have problems with the specific video files you are using?

CASE 1 - All movies are black

If all these tests end up with black video, then I'd check your graphics driver. 

First figure out the maker of your GPU. To do this, hold down the Windows key + R. In the run dialog, type dxdiag. On the Display tab you'll see the Manufacturer and Model of your device. On the right you'll see the driver date and version.

Assuming it is more than a few months old, go to the manufacturer's website (e.g. Intel, AMD or NVIDIA) and find a driver update utility. Run it and then try Movie Maker again.

CASE 2 - Pictures and WMV videos are fine, but these specific video files are not.

In this case, the problem is that your video files aren't 100% compatible with Movie Maker. Try converting them to a known compatible format like WMV and see if the problem goes away.

Print Artist 8 was designed to run on Windows 95/98/NT, so it is fairly old. You could try installing it in compatibility mode on Windows 7. To find out how to do this, see:

http://windows.microsoft.com/en-US/windows7/Make-older-programs-run-in-this-version-of-Windows

A longer term and much better solution for you would be convert you Print Artist 8 files in to something that can be read by Print Artist 22 on Windows 7. In Print Artist 8 when you open the Save As (assuming it works this way) is there a list of file types (as in .xyz) in the drop down menu? If so, write down these down. Now in Print Artist 22 when you try to open a file is there a list of files types it can open. If so, write these down.

The trick now is to look for a file type in the "Save as" list for Print Artist 8 that also appears in the "Open" list for Print Artist 22. You would then be able to save your files out in this format in Print Artist 8 then Open them in Print Artist  22.

Hi

There are multiple reasons why they get a busy tone when calling your Skype Number. Possible reasons include:

  • You are signed in to multiple devices at the same time. We do not recommend being signed in to two or more devices at once because there is a possibility that calls, chat messages or notifications might get misrouted.

  • Your privacy settings are set to accept calls from contacts only. Change this to accept calls from anyone. Click Preferences then select Privacy. Choose Anyone on the drop-down for Allow calls from.
