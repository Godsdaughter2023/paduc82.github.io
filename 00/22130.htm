The paragraph mark character (called a pilcrow) is easy, because it occurs in every font. To insert one in text, type B6 and press Alt+X. (The B6 is the hexadecimal number equivalent to the ASCII value 182, the character number of the pilcrow.)

The line break character is harder because the exact character isn't available in any font I've seen, but there are very similar characters in several fonts including Arial Unicode MS. To insert one in a document, type 21B5 and press Alt+X.

=ROUND(..., 1)

=ROUND(A2*1.3, 1)

The formula for retail price:

=ROUND(..., 0)-0.05

For example, with the above data:

=ROUND(A2*1.3, 0)-0.05

  4. Repair your installation of Windows 10 - first try booting then using F8 to get into the repair environment [or just boot from a Recovery drive / installation disk] then use Troubleshoot, Refresh your PC.

That character is "used to replace an incoming character whose value is unknown or unrepresentable in Unicode". Information about the original character is lost, so a find and replace won't work.  The Unicode character hex character value is FFFD--the last character there.  It is in the Arial Unicode font.  In the past I have been able to search for it by converting the Unicode value to decimal (and using the ^).  But I sometimes  don't get a result.
