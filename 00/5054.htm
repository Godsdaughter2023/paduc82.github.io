julienlary say:

Hi,

You have to be a verified admin and you have to create the external network with the same account. If an other verified admin creates an external network and give admin rights to you, it will not work (step 5 will not work)

An other tricks to know: when you log off of Yammer (in browser), Yammer save the last opened network as your current network.

Here are steps to follow in order to invite external users to an external network :

  1. Create an app in Yammer
  2. Get your oAuth token using : https://www.yammer.com/oauth2/authorize?client_id=[client_id]&redirect_uri=[redirect_uri]&response_type=token

    Use client_id and redirect_uri you've used to configure your app.
    After giving credentials to your app (maybe signing-in) and being redirected, you'll see your oAuth token in URL of your browser.
    See https://developer.yammer.com/docs/oauth-2 for more information about oAuth token.

  3. Send a GET request to : https://www.yammer.com/api/v1/oauth/tokens.json using the access token as a “Bearer” token in the “Authorization” request header. Implementations depends of language you use. 
    See https://developer.yammer.com/docs/api-requests for more information about Making API Requests

  4. You'll get in return a list/array of all your network with a different oAuth token for each network. Copy your external network's token.
