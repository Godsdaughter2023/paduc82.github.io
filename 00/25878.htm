tom urtis say:

Not seeing exactly what you are working with in terms of chart location and complexity, looking at this from a distance, if it were me, I'd start with a Worksheet Calculate event for the worksheet where the formulas are in column A.

Make a copy of your workbook to test the crux of the code that monitors the calculated changes that produce or "unproduce" the NA() values in column A.

Right-click on your worksheet tab, and left-click to select View Code. Paste the following procedure into the large white area that is the worksheet module.   Press Alt+Q to return to the worksheet.

Private Sub Worksheet_Calculate()
Application.ScreenUpdating = False
ActiveSheet.AutoFilterMode = False
'Good idea to delete the current chart (if it exists) before you re-create a new one, example:

'ActiveSheet.Charts(1).Delete

Dim LastRow&, FilterRange As Range
LastRow = Cells(Rows.Count, 1).End(xlUp).Row
Set FilterRange = Range("A1:A" & LastRow)
FilterRange.AutoFilter Field:=1, Criteria1:="<>NA()"
'Your charting code goes here for visible cells only, example:
Dim ChartRange As Range
Set ChartRange = Range("A1").CurrentRegion.SpecialCells(12)
'Code to create your chart from the ChartRange goes here
Set ChartRange = Nothing
Set FilterRange = Nothing
'ActiveSheet.AutoFilterMode = False 'Optional
Application.ScreenUpdating = True
End Sub

Keep in mind, you'd insert your existing chart creation code where I indicated with the comments. This is only a first stab; you might need further suggestions if I am off the mark.
