Windows 10 FileHistory Update needed - Microsoft Community

bob_alan say:

Windows 10 FileHistory Update needed

OneDrive provides an effective method to mirror files as a form of backup.   The downside of "mirror" syncing -- older versions of files with the same name are lost.   The solution -- 'File History'.    Unfortunately File History is not overly flexible.

Since FH was introduced, more limited capacity SSD's are being used for the C:\ drive with high capacity HHD for data. If you have a limited capacity SSD for the c;\ drive you don't want to have FH 'offline storage' using it.  You want to have the offline be on your data drive.  If one is using a permanently connected USB Drive on a desktop computer to store FH files there's no reason for offline storage.    Unfortunately FH insists the 2% of the most recent FH file backup go on the SSD.   

Suggestion -- keep the default at 5% of the C Drive but let the user change the drive and pick the percentage.

Conversely there appears to be no way to limit the size of the space use by FH on the destination drive.   For example I don't want FH to use all 4GB of my external recipient drive.  

The product would also be more useful if it allowed the user to include and exclude file paths rather then just file types.
