harry stevens say:

Glad to hear that it's a profile issue. Since the new profile works fine, here is a list on what you need to do to fully use your system again.

-Turn your new profile to an Administrator Account

-Move your files from the old profile to the new one

-Set the new profile to login to a Microsoft Account (to import settings)

-Delete the old profile. (Optional)

Here's a step-by-step procedures on how to do each of the list.

For the administrator:

1. Login back to the old account. (Log off the new one!)

2. Run the Netplwiz application once more.

3. Select the new profile on the list and click on "Properties"

4. Click on the "Group Membership" tab and select the "Administrator" option. Apply and Ok.

Moving the files:

1. Open the C:\Users\[old profile name]\

2. Copy everything that you see there (Excluding the AppData folder, this is normally hidden)

3. Open the C:\Users\ [new profile name]\

4. Paste and replace everything

Using MSA Login for the new profile:

1. Login back to the new profile(Log off the old one) 

2. Click on the Start Menu, and select Settings.

3. Select Accounts, then click on "Sign in with a Microsoft Account instead".

4. Provide your 'current' password for the Local account.

5. Login your Microsoft account.

Deleting the old profile:

1. On the new profile, Run Netplwiz one more time.

2. Select the old profile on the list, click Remove.

Let us know if you have any issues performing any of these steps.

Extra: Unfortunately, moving files can only be done manually. Browser cookies and caches may also be saved on the AppData folder which is not copied. So it's best to export anything on your browser, and then import them back through the new profile. All kinds of browsers has this feature (Even sync features using emails), if you need help with that, just let us know.

It's also optional to delete the old profile in case you want a backup, but since this is already corrupted, I don't recommend keeping it.

Hi,

 

Thank you for posting your query on Microsoft Community. 

 

I suggest you to follow the methods below and check if this helps:

 

Method1:

Try to uninstall and re-install the Bluetooth drivers.

You may follow these steps to uninstall and reinstall the Bluetooth drivers:


1. Press “Windows Logo” + “R” keys on the keyboard and type “devmgmt.msc” in the “Run” Command box and press “Enter”.
2. In the “Device Manager” window, search for the Bluetooth drivers, right click on it and then select “Uninstall”. On the un-installation window, if you have an option: “Delete the driver software for this device”   you may select that and then remove the corrupted drivers from the computer.
3. Follow the on-screen instructions to complete it and the restart the computer if prompted.

 

Method 2:

Follow the steps below on how to install driver in compatibility mode:
1. Download the driver from the manufacturer’s website and save it on your local disk.
2. Right click on the setup file of the driver and select “Properties”.
3. Select “Compatibility” Tab.
4. Place a check mark next to “Run this program in Compatibility mode” and select operating system from the drop down list.
5. Let the driver install and then check the functionality.
