If E25 has the drop down chosen as 'no' I need E27 to display '0'

If E25 has the drop down chosen as 'yes' I need E27 to display the value created by F27/G27.

If it matters, F27 is the sum of several cells from column f, G27 is the same (sum of cells in G)

How do I make this happen? I hope the above is clear, if not please let me know what additional info I can provide.

amit tandon say:

Try below formula:

=IF(E25="no",0,IF(E25="yes",F27/G27,""))

If E25 is "no", formula returns zero;

If E25 is "yes", formula returns F27/G27;

For any other value in E25, formula returns a blank;

Regards,

Amit Tandon

faysal farooqui say:

Though you have set your drop down list as "Yes" or "No", then you can use the below simple formula:

In E27:

=IFERROR(IF(E25="no",0,F27/G27),"")

in F27 to calculate several cells, try the below formula:

=SUM(single_cell1,single_cell2,.......,cell_range1,cell_range2,....)

in G27 to calculate the G column, try below formula:

=SUM(G:G)

Hi Rick,

 

As I understand Excel enters the configuration process every time you open it. Let me assist you with the issue:

 

The issue might have occurred due to add-ins or might be due to missing DWORD value. Follow the steps mentioned and check:

 

Step 1: Add DWORD value to the registry:

·         Close all the Office applications.

·         Press the windows key+R on the keyboard to open the Run window.

·         In the Run window copy and paste the commands mentioned one by one and click Ok:

reg add HKCU\Software\Microsoft\Office\14.0\excel\Options /v NoReReg /t REG_DWORD /d 1

·         Open Excel application and check.

 

Step 2: Check if Excel is running in compatibility mode:

·         Close Excel.

·         Browse to the location mentioned below:

For Win8 32 bit:

C:\Program Files\Microsoft Office\Office14

For Win8 64 bit:

C:\Program Files (x86)\Microsoft Office\Office14

·         Right click on Excel.exe< Properties< Compatibility

·         Uncheck the box Run this Program in Compatibility Mode, Make sure you also click on the option change settings for all users tab at the bottom of the same screen

·         Click on Apply and Ok.

·         Open Excel and check if it works.

 

Step 3: Rename the add-ins key in registry:

Important: Ensure to back up the registry before changing the registry keys:

http://windows.microsoft.com/en-us/windows7/Back-up-the-registry

 

·         Close all the Office applications.

·         Press the windows key+R on the keyboard to open the Run window.

·         In the Run window, type Regedit and click Ok.

·         In the registry editor window browse to the location mentioned below and rename the add-ins key:

HKEY_CURRENT_USER\Software\Microsoft\Office\Excel\Addins

HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\Excel\Addins

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\14.0\Excel\Addins

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\ Excel\Addins

·         Close registry editor window, open Excel and check if it works fine.
