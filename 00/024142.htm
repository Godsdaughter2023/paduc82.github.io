Hi,

Select your list of names and run this code. It will add sheets after the active sheet in the same order as your list.

Sub add_Sheets()
Dim c As Range, MyArray, x As Long
MyArray = Range(Selection.Address).Value
For x = LBound(MyArray) To UBound(MyArray)
Worksheets.Add After:=ActiveSheet
ActiveSheet.Name = MyArray(x, 1)
Next
End Sub
