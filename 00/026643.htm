talking moose say:
You can do this with a rule.
  1. Go to Tools menu --> Rules... and create a new rule for the type of account you're using.
  2. Name this new rule something like "Save All Attachments".
  3. Set this rule to apply to "All Messages".
  4. Set the rule to "Save Attachments" and choose a Destination folder.
  5. Finally, UNcheck the Enabled option. This keeps the rule from running on new incoming messages.
  6. Save and close the new rule.
To use this new rule select your multiple messages and right-click or Control-click one of them. Select Rules from the contextual menu that appears --> Apply and select your new Save All Attachments rule. This will trigger the rule manually and act on all selected messages.

shane devenshire say:

To answer this we need to know more about your data.  In particular how would we determine the "last" cell in column M?  If column A is solid data down to the last row, you could just reverse your two arguments:

:=Range("A1", ActiveCell.End(xlDown).End(xlToRight))

(by the way the arguments are XLDown and XLToRight not XLToDown.

Alternatively you might use CurrentRegion, if your data meets the necessary criterion:

Range("A1").CurrentRegion.Select

What should I know about the 30-day waiting period?

Even when you choose to delete your out-of-date security info, MS don't fully remove it for 30 days. This waiting period helps to deter hackers from updating your security info and gaining access to your account without your knowledge.
