Combining tracked changes - Microsoft Community

briis62 say:

Combining tracked changes

If I write a paper and two people separately edit the same paper tracking the changes, can the tracked changes made in both the edited copies be combined into one paper with all the edits

jay freedman say:
Yes. On the Review tab, click the Compare button and select Combine in the dropdown menu.

In the dialog, use the two Browse buttons (the folder icons) at the top to select the two edited documents.

If the More button is visible, click it to open the rest of the dialog.

Select the kinds of changes you want to include.

At the bottom, it usually works best if you choose "Word level" and "New document".

Click OK, and the program does its thing.

If you have more than two edited documents, you have to combine two, then combine the result with the third, and so on.
