Richvg44,

Do you know if the firewire port in your laptop has 4 pins or 6? 4 pins are used for information transfer and the 6 pin is used for transferring power to the device. So if the port on your laptop only has four pins and the hardware requires power this could be a reason that it won’t work. If you do not know how to find out how many pins it has then I would suggest contacting HP for assistance on finding out how many pins your firewire port has. Is anything working when you plug it up to a firewire port? If not, it is a possibility that the port is bad even though it has just been replaced.

Another thing we could try would be to make sure that the drivers are installed correctly for the firewire device. Please follow the steps bellow.

1.      Click on start. And type “cmd”. Right click “Command Prompt” and select “Runs as administrator”.

2.      At a command prompt, type the following command , and then press ENTER:

set devmgr_show_nonpresent_devices=1

3.      Type the following command a command prompt, and then press ENTER:

start devmgmt.msc

4.      Click Show hidden devices on the View menu in Device Managers before you can see devices that are not connected to the computer.

5.      Now expand “IEEE 1394 Bus-Hostcontroller”

6.      I would suggest uninstalling everything under that category.

7.      When you finish troubleshooting, close Device Manager.

8.      Type exit at the command prompt.

9.      Restart your computer. And re-install the most current driver. (I would suggest to check with HP to see if they have any updates)

Note that when you close the command prompt window, Window clears the devmgr_show_nonpresent_devices=1 variable that you set in step 2 and prevents ghosted devices from being displayed when you click Show hidden devices.

a. user say:

Hi,

Thank you for posting your query on Microsoft Community.

A network adapter driver is software used by your PC to communicate with your wireless network adapter. Outdated, incompatible, or damaged network adapter drivers can prevent network connections or cause intermittent disconnections.

If you recently upgraded from one Windows operating system to another, it's possible that the current network adapter driver was designed for the previous Windows operating system. If you've had recent power outages, viruses, or other PC problems, it's possible that the driver has become damaged.

I would suggest you to Run the Network Adapter troubleshooter and check if it helps:

Click on the search icon below and type Troubleshooting.

Then click on the Troubleshooting on the top of the Window.

Tap or click Network and Internet, and then tap or click Network Adapter.

If the issue still persist I suggest you to update the drivers in compatibility mode and check if it helps:

a. Download the driver from the manufacturer’s website.

b. Right-click on the driver setup file and click on ‘properties’.

c. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select operating system from the drop down menu.

d. Click on ‘Apply’ and click ‘OK’ and run the file to install it.

Also refer to the below link:

http://windows.microsoft.com/en-IN/windows/network-connection-problem-help#network-problems=windows-81&v1h=win81tab5&v2h=win7tab1&v3h=winvistatab1&v4h=winxptab1
