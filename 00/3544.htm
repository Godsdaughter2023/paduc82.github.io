rohn007 say:
Consumer/Retail licensees do NOT have downgrade (or as you've found, some people consider it an upgrade <grin> ) rights.

Only Volume Agreements that include the "Software Assurance" (SA) option (most expensive, of course) include downgrade rights.

http://www.microsoft.com/licensing/about-licensing/briefs/downgrade-rights.aspx - 6 pg PDF documenting downgrade rights

In my experience Office Product Keys are version/release as well as bundle specific.  A key for Home and STudent 2013 is only good for that, it will NOT work for H&S 2010, etc.

Note: Some OEM licenses qualify to be migrated to an existing volume license to allow you to use the downgrade rights.

Your best bet is to find a local MS Volume licensing specialist who can nail down the license terms for you

This is a portal page for volume licensing: http://www.microsoft.com/licensing/default.aspx?navV3Index=3

This page will take you to vendors specializing in volume licensing: http://pinpoint.microsoft.com/en-US/SelectCulture

FYI: http://www.infoworld.com/t/office-software/how-buy-microsoft-volume-license-the-cheap-213214

To disable UAC via the registry, you’ll need to Press Windows + R keys which opens the Run Command Box, type in regedit.exe and browse down to the following key:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System

Over on the right-hand side, you should see a setting for EnableLUA, which you’ll want to customize as follows:

  • UAC Enabled: 1
  • UAC Disabled: 0

This include the previous suggestions:

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Count > 1 Then Exit Sub
Set isect = Intersect(Target, Range("F4:F100,I4:I100,L4:L100,Q4:Q100,R4:R100"))
If Not isect Is Nothing Then
