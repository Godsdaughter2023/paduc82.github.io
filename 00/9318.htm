bob jones aka: cybertaz say:

Additional to what has been offered: All Styles are available from 2 locations & possibly a third;

  1. The root source is Format> Style from the main menu. Select All Styles (or one of the other groupings) from the List: dropdown in the lower left corner of the dialog window.
  2. The Styles Pane which can be displayed by clicking the button at the right end of the Home tab or can be toggled on/off by using the built-in keyboard shortcut Shift+Option+Command+S. As with the above option, choose from the List: dropdown at the bottom of the pane.
  3. Styles which included the Add to Quick Style List property also are available from the Styles gallery on the Home tab.

All Heading styles are available from the first source at all times. However, Heading styles 2-9 are Hidden - They (as well as many others) are programmed to display in the other 2 UI sources only under certain conditions. Briefly;

  • Only Heading 1 displays in the Styles Pane when Recommended is selected from the List until you apply it. When a Heading style is used in the document the next level Heading style is automatically added to the Pane.
  • Initially, both Heading 1 & 2 are available from the Styles gallery. However, if another style having the Add to Quick Style List property is used before Heading 2 is used, Heading 2 gets bumped from the gallery. it isn't added back until it actually is applied in the document.

The Style management features in Mac Word are a bit lacking. This information is simply the gist of how these features work but it should point you in the right direction. BTW: You're to be commended for using Styles in the first place :-)

Use a macro.  Assuming they are inserted as inlineshapes then something like this:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oILS As InlineShape
  For Each oILS In ActiveDocument.InlineShapes

N96 - Microsoft Community

lazareth say:

N96

Does anyone know why my N96 "Say Caller's Name" does not work?

cjlim say: Does your phone display the name of then caller or just the number? Say callers name will only work is your phone recognises the caller.

The most common causes of caller ID problems are:

- duplicate entries in your contacts, including entries stored on your sim card. This is a problem becuase the phone does not know which name to display and so it doesn't display any. Duplication also includes the same number in different formats. Try searching for a PART of the number as if for example you search for +353 123 456 789, it won't find +353123456789, whereas if you search for 6789 it would.

Having different formats is a problem for caller ID.

- using your international dial code instead of "+", eg: using 00353123456789 (wrong) instead or +353123456789 (right).

- spaces in numbers (some phones, some firmware versions only), eg: +353 123 456 789 (wrong) instead or +353123456789 (right).

- MS Outlook putting "(0)" in number - if you define a country for a contact in MS Outlook it will automatically add a country code and insert a "(0)" in the number in front of the area code, eg: +353123456789 becomes +353 (0) 123456789. You need to remove the "(0)"s

- there has been reports of more than 7 digit numbers causing problems. Some phones may not differentiate between +353123456789 and +353993456789.
