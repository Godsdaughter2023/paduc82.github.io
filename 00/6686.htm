Hey Wood Worker,

 

To clear the list in the Address Bar on the Taskbar you will need to set the Start Menu to not store and display recently opened programs in the Start menu. This will clear the list of recently opened programs in the Start Menu and clear the list of addresses in the Address Bar on the Taskbar. You can do this by:

·         Right click the Start Button

·         Click Properties

·         Click the Start Menu tab

·         Uncheck the Store and display recently opened programs in the Start Menu box

·         Click OK

 

Repeat the above steps, but check the box you unchecked before to enable the list again. You will have to build up your recent programs list again, but this will clear the Address Bar list.

Hi Andy,

Here are the steps to get send as permission for all shared mailboxes:

1.Export all shared mailboxes to a CSV file:

Get-Mailbox -RecipientTypeDetails SharedMailbox | select Alias | Export-Csv C:\Users\username\Desktop\result.csv

2.Import the CSV file to PowerShell and get the send as permission:

Import-Csv C:\Users\username\Desktop\result.csv | foreach {Get-RecipientPermission -Identity $_.Alias} | Fl

Please note that the parameter -Identity $_.Alias needs to be in accord with the title bar (Here is Alias in this example) in the CSV file.
