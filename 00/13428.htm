rohn007 say:

OK, at first I could not recreate your problem, but I finally stumbled over a clue and eventually was able to consistently recreate your problem. it is a really finicky thing.

To fix the problem

  • click to the spot right next to existing word, no space between existing word and cursor (existing space is after cursor).
  • hit single space
  • turn on underline, <CTL><U>
  • type the word
  • turn off underline, <CTL><U> (should not turn off underline from new word)
  • hit space, no underline

It all appears to depend on what Word is treating as a "new word". If you start in the wrong place word acts up.

Hi FlyntM,

 

From your description, you would like to return the corresponding color in Choice 2 column according to the specific item in Choice 1 column.

 

If you would like to select the corresponding color via a drop down list, you can create dependent data validation to achieve it. Detailed steps are as follows:

1. Create named ranges in Name Manager under the Formulas Tab.

2. Create dynamic named ranges by using the following formula:

=OFFSET(Sheet!$E$1,1,0,COUNTA(Sheet!$E:$E)-1,1)

3. Create the main drop down list.

4. Create dependent data validation by using the following formula:

=CHOOSE(MATCH($C2$,Items,0),Cookie,Apple,Wagon,Bananna)

 

If you would like to return the corresponding color automatically according to the value in Choice 1, you can use the following formula to achieve it: =IF(C2="Cookie","Brown",IF(C2="Apple","Red",IF(C2="Wagon","Green",IF(C2="Bananna","Yellow",""))))

flyntm say:

Thanks to all those who attempted to help me.  Just in case someone comes across this later, here is how I finally  managed to get the results I wanted.

I had to resort to VBA but once I found the right code, it was pretty straight forward.

If Target.Address = "$E$8" Then

        If Target.Value = "LCD" Then

            Range("$J$8").Value = "NONE"

            Range("$K$8").Value = "NONE"

        End If

    End If

If the user chooses the value "LCD" from the drop down box in cell E8 then the values in J8 and K8 will be set to "NONE".
