madenna say: I stumbled across this code during my research.   If I understand what the original poster of this code is saying, he indicated that the DLL's could actually be stored in the FE and that this code could be used to install them. Public Sub DumpFilesToPath()
    Dim intFileNum     As Integer
    Dim bytBuffer()    As Byte
    Dim strPathAndFile As String

    With CurrentDb.OpenRecordset("tblUserFiles")
        Do Until .EOF
            strPathAndFile = CurrentProject.Path & "\" & !FileName
            
            If Dir(strPathAndFile) <> !FileName Then
                intFileNum = FreeFile
                
                Open strPathAndFile For Binary As intFileNum
                  
                ReDim bytBuffer(!File.FieldSize)
                bytBuffer = !File.GetChunk(0, !File.FieldSize)
            
                Put intFileNum, , bytBuffer()
                
                Close intFileNum
            End If
           
            .MoveNext
        Loop
    End With
    
End Sub

jllatham say:

I went back and read your request more closely and saw this:

"... and they are highlighted in that Monthly Calendar, so upto here verry nice and good big achievements for me but......"

I suspect that you are somehow using Conditional Formatting to get that highlighting.  What you would want to do is use the same test that you use for conditional formatting to see if you want to put the words "National Holiday" in a column. 

An example that is very simplistic and probably not close to how you test for October 26, 2014 being a holiday, but maybe is example will give you an idea.

Let us assume that your Conditional Formatting test is something like this (assumes 26/10/2014 in A2):

=A2=DATEVALUE("10/26/14")

and in B2 you probably have something like

=CHOOSE(WEEKDAY(A2),"Sunday","Monday","Tuesday","Wednesday","Thursday","Friday","Saturday")

To test specifically if A2 as the 10/26/14 holiday, and report it you might modify the formula in B2 to add something like this to it

 & IF(A2=DATEVALUE("10/26/14")," " & CHAR(10) & "National Holiday","")

So the whole formula might look like this (be sure to set word wrapping on in the cells):

=CHOOSE(WEEKDAY(A2),"Sunday","Monday","Tuesday","Wednesday","Thursday","Friday","Saturday") & IF(A2=DATEVALUE("10/26/14")," " & CHAR(10) & "National Holiday","")

But again, the test for the holiday date, A2=DATEVALUE("10/26/14"), is very simplistic and only works in this specific case.
