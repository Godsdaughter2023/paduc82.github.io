Hey Simon,

 

Here are few steps to increase the speed of Internet connection in Windows 7. Hopefully one of these steps should help you in boosting the speed of the connection.

 

Step 1: Register actxprxy.dll from the command prompt.

 

Close Internet Explorer and access an elevated command prompt.

 

1.     Go to Start and type cmd.

2.     Right-click on cmd and select “Run as administrator”.

3.     Once you have your elevated command prompt, type the following and press Enter.

regsvr32 actxprxy.dll

4.     A confirmation window will appear. Click the OK button.

 

Step 2: Disable reserved bandwidth.

 

Method 1: Using Group Policy Editor.

 

Note: Group policy editor is available only in Professional, Ultimate and Enterprise edition of Windows 7. If Group Policy Editor isn’t supported in your version of Windows, go to Method 2.

 

1.     Go to Start-> Run-> and type gpedit.msc.

2.     Expand the Administrative Templates branch.

3.     Expand the Network tab.

4.     Highlight QoS Packet Scheduler.

5.      Click on Limit Reservable Bandwidth and click on Properties.

6.     Check the box Enabled

7.      and type the Bandwidth limit 0%.

 

 

Method 2: Using Registry Editor.

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the windows help article below.

 

Back up the registry

http://windows.microsoft.com/en-us/windows7/Back-up-the-registry

 

1.     Go to start and type regedit.exe.

2.     Go to:
HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows

3.     In the left pane, right click on Windows and click on New and Key.

4.     Type Psched and press Enter.

5.     In the right pane, right click on a empty area and click on New and DWORD (32-bit) Value.

6.     Type NonBestEffortLimit and press Enter.

8.     To disable the reserved bandwidth, type 0 in the value data.

10.  Restart the computer to apply the changes.

ksalsample say:

Windows 8.1/8 comes with .NET Framework 4.5 already installed. But, for some applications to run properly, we might require an earlier version (such as 3.5 or 2.0). The .NET Framework 4.0 is replaced by .NET Framework 4.5 on Windows 8.1/8. So, you just need to enable 4.5 version on the computer and any application that needs 4.0 should work on the computer. 

 

If you need the older versions (such as 3.5 or 2.0), then there are two ways to install them: 1) install it on demand and 2) turn it on in Control Panel.
