Hi corrlines,

 

1.    What is the exact issue you have with the Windows media player?

 

You cannot start over with new library in Windows media player but you may clear the Windows Media Player database. To do this, follow these steps:

 

a)   Exit Windows Media Player.

b)   Click Start, click Run, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.

c)    Select all the files in the folder, and then click Delete on the File menu.

Note: You do not have to delete the folders that are in this folder.

d)   Restart Windows Media Player.

 

 Note Windows Media Player automatically rebuilds the database.

 

If the above does not fix your issue then clear the Windows Media Player database cache files. To do this, follow these steps:

a)   Exit Windows Media Player.

b)   Click Start, click Run, type %LOCALAPPDATA%\Microsoft, and then click OK.

c)    Select the Media Player folder, and then click Delete on the File menu.

d)   Restart Windows Media Player.

Speed of the internet and quality of streaming - Microsoft Community

piotr222222 say:

Speed of the internet and quality of streaming

Hello. Does the speed of the internet affect the quality of streaming between Xbox One and Windows 10 Laptop? Does the main role play the distance of the console and laptop from the router, with WiFi connection?
echo4ent say:

No, your internet speed does not affect the quality of streaming to a local PC.  If you are streaming over WiFi, then there are a couple of things that will affect the quality:

1. What type of WiFi (A/B/G/N/AC)  AC is capable of the fastest speeds

2. Distance to, and obstructions between you and the router.  The further away, and the more walls, furniture, etc. that are in the way will slow things down

3. How many users are on the same WiFi.  The more users, the less speed each person gets.

A wired connection is preferred for local streaming.

I hope that all made sense.

richminaya say:

"C:\ is not accessible. Access is denied" error message

original title: RIPPING OUT MY HAIR - I'm an idiot.

I was playing with advanced sharing and privileges and now I can not get access to my c drive.  I feel like an idiot.  I don't know what I have deleted, but it will not allow me to restore my access.  It tells me "C:\ is not accessible.  Access is denied" ... I right click on the c drive and attempt to change privileges.  For example, give access to everything to everyone, guests, users, etc - and the message I get is "An error occurred while trying to modify share administrator.  Access is denied" - - - HELP.  I am so frustrated. 

yusufat say:

In NTFS volume, only the owner of an object (file, folder, ...) that can change permissions of that object. You need to take ownership of your drive C: somehow, after that only you can grant permissions to users.

Try to enter safe mode, login as "Administrator" (enable that account first if it is currently disabled). Then right click C:\ drive, click Properties. Navigate to Security Tab, then click Advanced. Go to Owner tab, then change the ownership to "Administrator" first. Then, grant permissions like these to these users [CMIIW]:

Authenticated Users: Read, execute, write, modify,
Administrators: Full Control
TrustedInstaller: Full Control
SYSTEM: Full Control
Users: Read, execute

Finally, change the owner once more to "TrustedInstaller".
