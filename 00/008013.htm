You may have some volatile functions in the sheet(s), which automatically update, when the workbook is opened.

If you don't want to save the file, use this:

Workbooks("File1.xls").Close SaveChanges:=False

File2 should be closed too, unless the macro stop due to an error. You can use 'ThisWorkbook.Close' for File2, so you do not have to worry about changes in the workbook name.
