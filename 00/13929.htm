tom ogilvy say:

let's assume that =Sheet2!AP7 is in A1.   (Note, that you don't need single quotes for a sheet name that doesn't have spaces in it. 

Excel will not adjust the sheet reference if you drag fill so you will need to create a formula that creates the correct address and then use the indirect function to convert it to an actual reference. 

A useful function for calculating the address is the Address function.  the arguments are 

ADDRESS(row_num, column_num, [abs_num], [a1], [sheet_text])

you would enter the row and column numbers (no letter references) for your address. 

=Address(1,2)   would produce  $B$1 as a string/text value. 

If I want to produce the value in B1 in the cell with the formula I would put the address function inside the indirect function 

=Indirect(address(1,2))

If I wanted to produce   Sheet2!B1   is would do 

=Indirect(address(1,2,4,true,"Sheet2") 

The 4 as the third argument indicates that the reference should be relative vice absolute in both the column and row positions.   The key to the value in this position are:

1 or omitted:  Absolute
2 Absolute row; relative column

3 Relative row; absolute column
4 Relative

the fourth argument is  True (use A1 referencing) or False (us R1C1 referencing). 

the last Argument is the sheet name

Since address returns a string, dragging filling it will not change the results regardless of whether absolute or relative references are chosen.   

what we need to do is calculate the arguments for row and column so you can drag fill the formula. 

=Address(row(A1)+6,Column(A1)+15,4,True,"Sheet2")

entered anywhere will produce:    "Sheet2!AP7"

when I drag fill it down one row it will produce  "Sheet2!AP8"

If I drag fill the original cell to the right one column it will produce  "Sheet2!AQ7"

If i then drag fill that cell showing "Sheet2!AQ7" down one row it shows: 

"Sheet2!AQ8"    so both the column and row reference has been increased by one relative to the initial cell. 

This is not what you want but I am providing it to illustrate how address can be used.  To get what you want, you want the row to increment by 1 when you drag to the right and the sheet name to increment by 1 if you drag down a row.   So we would modify the formula to use the "column(A1)" in the row position and use the  row(a1) concatenated to the base "Sheet".  

in any cell enter:  

=Address(column(A1)+6,42,4,True,"Sheet"&row(A1)+1)  and drag fill to a total of 16 cells (4 rows and 4 columns)

this produces: 

Sheet2!AP7 Sheet2!AP8 Sheet2!AP9 Sheet2!AP10
Sheet3!AP7 Sheet3!AP8 Sheet3!AP9 Sheet3!AP10
Sheet4!AP7 Sheet4!AP8 Sheet4!AP9 Sheet4!AP10
Sheet5!AP7 Sheet5!AP8 Sheet5!AP9 Sheet5!AP10

We would modify the formula to this  

=INDIRECT(Address(column(A1)+6,42,4,True,"Sheet"&row(A1)+1) ) 

and again drag fill 4 rows and 4 columns to actually return the values in those cell referenced.
