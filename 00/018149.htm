VBA Vlookup of external workbook - Microsoft Community

chrisfly say:

VBA Vlookup of external workbook

Hi,

     I am trying to look up an external workbook using vlookup in vba.

I have a worksheet with a table of values Sheet1("I4:I500"), and for each value I want to lookup an external workbook wb2.Sheet2.("A1:G600") and see if that value appears in it column A, and then read the relevant value from column G.

Cheers

jllatham say:

I made some changes to the code.  Your compiler error was caused by 'wb' not being declared.  Fixed that.  This code will take the values in column I of the active workbook, compare them to the values in column A of the other workbook and when it finds a match, will bring the value from column G of the other workbook into column J next to the matched value in column I.  Same as putting a VLOOKUP() formula in column J of this workbook.  I changed the name of a couple of variables just to help me keep my head straight about it all.  I also removed some variables that were not being used. 

Sub LookupValues()

  Dim colI_Cell As Range
  Dim colI_Range As Range
  Dim rngLookupRange As Range
  Dim rngFound As Range
  'add this
  Dim wb As Workbook

  ThisWorkbook.Worksheets("MAIN").Activate
  Set colI_Range = ActiveSheet.Range("I4:I500").Cells
 
  'when you do this, Workbook2.xls becomes active
  Set wb = Workbooks.Open("W:\Workbook2.xls", True, True) ' open the source workbook, read only
  Set rngLookupRange = wb.Worksheets("CDE").Range("A1:A600")
  'moved from below to make this workbook active again
  'Workbooks("Workbook1.xlsm").Worksheets("MAIN").Activate ' old code
  ThisWorkbook.Activate ' leaves no doubt!
 
  For Each colI_Cell In colI_Range
    With rngLookupRange
      Set rngFound = .Find(What:=colI_Cell.Value, LookIn:=xlValues, LookAt:=xlWhole)
      If Not rngFound Is Nothing Then
        'rngFound points to a cell in col A in other workbook
        'we need to bring back value from G into col J in this workbook
        ' looked in col A, so offset (0, 7) returns column G value
        colI_Cell.Offset(0, 1) = rngFound.Offset(0, 7).Value
      End If
    End With
  Next
 
  'housekeeping - release resources back to the system
  'and close the other workbook
  Set rngLookupRange = Nothing
  wb.Close False ' close the source workbook without saving any changes
  Set wb = Nothing ' free memory
  Set colI_Range = Nothing
End Sub
