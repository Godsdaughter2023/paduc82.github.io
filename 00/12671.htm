VBA to make chart data series the same colour as another chart - Microsoft Community

mjxhx say:

VBA to make chart data series the same colour as another chart

I have 2 charts, one column and one pie. I need to make the data points in Chart2 (the pie) have the same colour as the data series with the same name (in the legend) from chart1.

i.e. if a data series in chart1 say the London data series is red, I want code to run through all the data points in my pie chart (chart2) and where it finds thhe same name, it will copy the colour from the chart1 data series, Therefore the 'London' data point in the pie will end up red.

Is that possible? I'm a bit stumped.

Thanks

M

e.j. gun say:

Try the subroutine below.  It assumes that "Chart 1" is the column chart, and that "Chart 2" is the pie chart.

Option Explicit

Sub SynchColors()

    Dim i as Long

    Dim theSeries as Series

    Dim thePieSlice as Variant

'

    i = 0

    For Each thePieSlice In ActiveSheet.ChartObjects("Chart 2").Chart.SeriesCollection(1).XValues

        i = i + 1

        For Each theSeries In ActiveSheet.ChartObjects("Chart 1").Chart.SeriesCollection

            If (theSeries.Name = thePieSlice) Then

                ActiveSheet.ChartObjects("Chart 2").Chart.SeriesCollection(1).Points(i).Format.Fill.ForeColor.RGB = theSeries.Format.Fill.ForeColor.RGB

                Exit For

            End If

        Next theSeries

    Next thePieSlice

End Sub

You definitely won't get any transition to play by using See All Slides. An automatic smooth transition back to the Summary Zoom slide will play when you use Page Up or when you reach the end of the section.

If you want to be able to jump back from any slide in the deck and have a smooth transition, you'll have to include a Slide Zoom on every slide that links back to the Summary Zoom.
