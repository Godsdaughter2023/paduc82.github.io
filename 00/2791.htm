dhookom say:

However assuming a query that returns

Seat     Passenger

  1. Create a report with a similar recordsource and make the detail section about 10" high.
  2. Add text boxes for the fields Seat and Passenger to the detail section and remove labels
  3. Make the Seat text box Visible: No
  4. Make the Passenger text box about 1/2" wide and 3/8" tall
  5. Make the Passenger border dark
  6. Make the report detail section background white and alternate color white
  7. Add code to the On Format event of the report
         Dim lngRowSpacing As Long
    Dim lngSeatSpacing As Long
    Dim intRow As Integer

    lngSeatSpacing = 0.75 * 1440    'spacing left to right 3/4"
    lngRowSpacing = 0.4 * 1440      'spacing between rows under 1/2"

    Select Case Right(Me.Seat, 1)   'set the left to right position
        Case "M"   'Middle
            If Left(Me.Seat, 2) = "DS" Then
                Me.Passenger.Left = lngSeatSpacing * 2
             Else
                Me.Passenger.Left = lngSeatSpacing * 6
            End If
        Case "W"    'Window
            If Left(Me.Seat, 2) = "DS" Then
                Me.Passenger.Left = lngSeatSpacing * 1
             Else
                Me.Passenger.Left = lngSeatSpacing * 7
            End If
        Case "I"    'Aisle
            If Left(Me.Seat, 2) = "DS" Then
                Me.Passenger.Left = lngSeatSpacing * 3
             Else
                Me.Passenger.Left = lngSeatSpacing * 5
            End If
    End Select
    intRow = Val(Mid(Me.Seat, 3))    'for the row
    Me.Passenger.Top = intRow * lngRowSpacing
    Me.MoveLayout = False
End Sub

If you do everything as I suggested your Print Preview should look like this. My passengers are the same as my seats for testing purposes.

john mcghie say:
Word 2008 is too old to have Style Sets.  Word 2011 has them.

You can achieve the same result in Word 2011 by making different templates, one for each style set you want to use.

Then use Tools>Templates and Add-ins>Attach to attach the correct template to your document.  When you do, check the box that says "Automatically update styles". The styles in the template you choose will instantly replace the styles in the document.

Or you can use File>New from Template to CREATE your new document from one of your templates.  That also enables you to bring in pre-defined text at the same time as you bring in the style set.

Using separate templates is the way skilled Word professionals always do it, because we always want not just the styles but also the text, the table formats, the auto-text entries, the toolbars, the macros, the headers, footers, TOC and Index, paper size and sections...

By using File>New From Template you get a complete document instantly, all set up ready to type.  Its much faster and more powerful than fiddling with style sets.  Style Sets was added to Office 2007 as a "lite" version of the mechanism for people who could not work out how to use Templates.

But if you spend five minutes figuring out Templates, you will find its faster, easier, more powerful, works better, and it works fine both sides of the Mac/PC divide.

Hope this helps
