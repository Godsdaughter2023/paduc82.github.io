     --> Other folder 
     --> select the newly created folder

Adjust the zoom of the document map (not the document itself) - Microsoft Community

paul domaskis say:

Adjust the zoom of the document map (not the document itself)

For years, I've been gnashing my teeth about the fact that Word's document map will change zoom level at unpredictable times, and the only way to compensate was to change the font size of the Document Map style.  It's hard to think of a less elegant way to work around the problem.  And at any time, it could change again.

Now, years later, I have accidentally discovered the circumstances under which the zoom level in the document map changes.  It is like magic!  I googled for the information, thinking that I must be a dunce not to know something so simple.  Surely, this is common knowledge.  What I found is that it isn't.

Well, here's the secret.  First, you have to make use of a trick that *is* common knowledge. That is, you control the zoom level of the window or pane with the windows focus by holding down Control and scrolling up or down with the mouse wheel.  Everyone knows this part.

The secret part is that you double-click in the Document Map in order to be able to use the above trick to adjust the zoom level in the Document Map.  Sure, the zoom field in the toolbars is grayed out, but Ctrl-scroll-wheel still works.  If you don't want to double-click on a heading (which might change the cursor position in the document), then double-click on the empty area after the last heading.

Amazing how such a useful tidbit is *not* common knowledge.  Perhaps it's not even known to the Microsoft Developers.  After all, when you evolve a complex software system, you define how different parts interact, but you might miss a few possibilities.

I HOPE they kept this feature in Office 2007 and 2010.  I discovered it in Office 2003, Windows XP.
