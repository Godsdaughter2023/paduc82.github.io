teylyn say:

Hello,

a data validation can have several conditions but only one error reply. Use a custom validation formula and combine several conditions in an AND() statement, like this:

=AND(A1*100=INT(A1*100),A1<1000)

Set the error alert message to something like "Please enter numbers below 1000 and with up to 2 decimal places only."

Alternatively, use only the formula

=A1*100=INT(A1*100)

and in another cell use 

=if(A1>1000,"Do you really want to enter "&A1&"?","")

This will allow entry of bigger numbers, but the prompt will show.

  2. Unchecking the notebooks you do not want to receive notifications
  3. Press save

  1. Sign into https://onedrive.live.com
  2. Navigate to the notebook location
  3. Right click on the notebook and click "Details"
