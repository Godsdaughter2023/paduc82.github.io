             task manager and chrome are somehow permanently minimized.

Often this is how users describe seeing windows which are open off-screen, e.g. on another monitor, phantom or otherwise.  Press Win-P to see if you are in Extended display mode.  Change to Duplicate if so, e.g. press CursorUp, and Enter after that.  

Otherwise, assuming both windows are Desktop windows, another way to move them back into view is to use the Taskbar's Cascade windows command.  Right-click the Taskbar to see that.  Note that this option will only work on windows which are not actually Minimized.  So, use right-click of a window's thumbnail to check on its state before using Cascade windows to make sure that it will be included in its operation.  The window will not be Minimized if you see the Minimized command as active, e.g. blue instead of grey.
