bob jones aka: cybertaz say:

What you describe cannot be done [effectively] in Microsoft Word. In theory, Linked text boxes could be used but that would be [IMHO] an absolute nightmare in virtually all respects.

For that type of document you need to use a page layout/desktop publishing program. Programs of that sort support individual pages as separate components of the document, so you can mix & match as need be. Word documents are structured differently, as a continuous text flow from start to finish. The concept of a page is purely a euphemistic part of the UI design.
