eileen gal say:

Hi Anshul,

Some background services can affect usage and disk usage even if there are no programs running. To resolved this concern, follow the troubleshooting steps recommended for each error that you have received.

Error-1: Event ID: 10016, Source: DistributedCOM
This error means that the COM Server application needs permission from your computer. To grant local activation permission, follow the instructions below:

  1. Click Start > Type Run > Type dcomcnfg, expand Component Services > Computers > My Computer > DCOM Config.
  2. Click the View tab > Go to Detail (Now you will get the Application Name and Application ID on the right side).
  3. Scroll down and find the application ID {4839DDB7-58C2-48F5-8283-E1D1807D0D7D} > Right Click > Go to Properties and select the Security tab.
  4.  Click Customize under Launch & Activation Permission > click Edit > Add in the account NT AUTHORITY\SYSTEM and set local launch and local activation.
  5. Restart the application Service linked to this Application ID or restart the server and monitor.

Error-2: Event ID: 5007 , Source: Netwtw04
To resolve this error, you can try uninstalling the network adapter driver and restart. Refer to the troubleshooting steps below:

Before uninstalling, make sure you have drivers available as a backup. Visit the PC manufacturer’s website and download the latest network adapter driver from there. If your PC can't connect to the Internet, you'll need to download a driver on a different PC and save it to a USB flash drive so you can install the driver on your PC. You’ll need to know the PC manufacturer and model name or number.

  1. Type Device Manager in the Search bar, and then select Device Manager from the list of results.
  2. In Device Manager, select Network adapters > the network adapter name.
  3. Press and hold (or right-click) the network adapter, and then select Uninstall device > Delete the driver software for this device check box > Uninstall.
  4. After uninstalling the driver, select the Start button > Power  > Restart.

Feel free to post back for an update and let us know if you need more help.
