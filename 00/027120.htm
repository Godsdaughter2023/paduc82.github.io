jllatham say:

I'm going to assume that you want to copy the cells over to another sheet at the same cell addresses that the original cells are in.

This routine would allow you to select the cells to be copied manually (you could even do a few at a time) then run the macro from the [View] tab, Macros icon, View Macros list item.

You said "another sheet or workbook" - and I've just coded to copy to another sheet in the same workbook.  Without more specifics, it's hard to know what to provide to you.

This page will give you the instructions for putting the code into a Regular Code Module in your workbook.  Be sure to save it as a macro-enabled workbook, type .xlsm or .xlsb, after adding the code to it.

http://www.contextures.com/xlvba01.html#videoreg

Sub CopyDispersedCells()
'you start by selecting the cells you want to copy
'Choose 1 then hold the [Ctrl] key while you select
'others.
'this routine copies them to the same cell address
'on another sheet you specify here:
  Const copyToSheetName = "Sheet2"
  Dim destWS As Worksheet
  Dim anyCell As Range

  Set destWS = ThisWorkbook.Worksheets(copyToSheetName)
  For Each anyCell In Selection
    anyCell.Copy
    'this copies just the value, not the formula
    Sheet2.Range(anyCell.Address).PasteSpecial xlPasteValues
    'then we also copy the formatting
    'you don't have to do this if you don't want to
    '-- just delete this line, or
    Sheet2.Range(anyCell.Address).PasteSpecial xlPasteFormats
    'if you wanted to just copy the value and number formats, use this line
    'instead of the two Sheet2.... lines above
    'Sheet2.Range(anyCell.Address).PasteSpecial xlPasteValuesAndNumberFormats
  Next
  Application.CutCopyMode = False
  Set destWS = Nothing
  MsgBox "Copy accomplished.", vbOKOnly + vbInformation, "Task Completed"
End Sub

jllatham say:

We might be able to work up a macro similar to the above that would look through all used cells on a sheet for cells that are formatted in a specific manner - but we would need to know the .Interior.Color or .Interior.ColorIndex of the cell(s).  Also, obviously you would only want to have the cells to be copied formatted that way.

The other limitation at this point is that the cells would actually have to have been formatted from the keyboard, NOT by conditional formatting.

Here is a macro that will give us the information about a cell's shaded properties that you could use to report that information back to us with.  Choose a single cell like one you want to be copied and run this macro:

Sub GetCellShading()
  If Selection.Cells.Count > 1 Then
    MsgBox "Sorry, please select ONE cell only and try again."
    Exit Sub
  End If
  MsgBox "The .Interior.Color is: " & Selection.Interior.Color & _
   vbCrLf & "The .Interior.ColorIndex is: " & Selection.Interior.ColorIndex
End Sub

gord dibben say:

Jerry

This macro will select all colored cells in any selected range.

Note: I use the older  Interior.ColorIndex   of  -4142    change to suit.

After running it then use your macro to copy those cells.

Sub Select_Colored_Cells()
    Dim C As Range, tempR As Range
    Set myrng = Selection

    For Each C In myrng
        If C.Interior.ColorIndex <> -4142 Then

            If tempR Is Nothing Then
                Set tempR = C
            Else
                Set tempR = Union(tempR, C)
            End If
        End If
    Next C

    If tempR Is Nothing Then
        MsgBox "There are no Rows " & _
               "that meet the criteria."
        End
    End If

    tempR.Select
End Sub
