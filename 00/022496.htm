There is no link to the macro.  The macro has created a data validation drop down for you.  But the macro is in no way linked to the cell.  The data validation is linked to the Distributor range but that is an entirely different issue.  If you removed the VBA macro from the file, all the existing data validations would continue to work you just wouldn't be able to add more with the macro.
