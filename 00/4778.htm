preciousleb say:

We have two solutions for this issue to be resolved. The first thing you need to do is to reset Windows 10 Store. Here are the steps:

1. Go to Search or I'm Cortana. Ask me anything.

2. Type WSReset.

3. Right click  and then select Run as administrator.

The second solution is to gain full control over the packages folder. Follow these steps carefully:

  1. Press Windows key + R to open Run command or type Run on I'm Cortana. Ask me anything.
  2. Enter the following path and then press Enter: C:\Users\<username>\AppData\Local
  3. If in case you can't find the username, click Advanced then after that, click Add.
  4. Find folder Packages, right click on it then select Properties.
  5. Go to Security window.
  6. Click Select a principal, then type users in Select User or Group box.
  7. Click Check Names.
  8. Check Full control for Basic permission section.
  9. Click Apply then OK.
  10. Reboot the computer.

Note: Make sure you have full control over the Packages folder.

Let us know how it goes.

Help. Is there symbols for vowel sounds available in Word 2008 Mac

Hi

I am creating some flashcards and want to put the symbol for a short vowel (inverted hat) over them and the symbol for a long vowel (short line) above letters.  Is this possible and if so please can you tell me where/how I do it.  Cannot find in symbols.  Thanks very much.
daniel g. grau say:
You mean these: āǎă? (Copy and Paste these into Word, then increase their size or zoom in to see them clearly.)

The simplest way:
  • Go to System Preferences > Language and Text > Input Sources and enable the "U.S. Extended" keyboard.
  • While you are there, place a checkmark on " Show input menu in menu bar," at the bottom. (This will add a new keyboard-selection menu in the top-right corner of your screen, next to the system clock.)
  • Then, in the new menu, select Show keyboard viewer.

Pressing the Option key will highlight the key combinations you can press. The first press will be a "dead" key, which means you then need to press a second key to obtain the combination. For the case above, I pressed Option a then a, Option v then a  and Option v then a.

If you want a phonetic alphabet, use the IPA English or the IPA Full online keyboards to type your text, then cut and paste into Word. It's a lot simpler than installing a phonetics keyboard and font.

daniel g. grau say:
The Option key is alto labeled Alt and may have an icon that looks like this: ⌥. There are other such "modifier keys": command, control, shift, and...

The fn key that you mention is used to alternate the commands triggered by the F keys (F1, F2, F3...) between the software's functions (e.g., as defined by Word) and the System's controls (e.g. volume, contrast, play, stop ...). This is controlled in System Preferences > Keyboard.
