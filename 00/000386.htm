nancydickinson say: Out of the inner depths of my mind, I solved my own problem after weeks of banging my head.

What I did:

  1. I created a section (right-click on page where you want the section break to appear BEFORE the page) to include pages 1 through 4.  For instructions on creating a section break see here:  http://answers.microsoft.com/en-us/office/forum/office_2010-office_other/publisher-2010-insert-section-break-page-numbering/aa903552-a9e8-4600-b470-c52dbf80d0b3

I then created a second master page, one that included page numbers in the footer, along with the link for the magazine.  This was a two-page view template so I could left-justify the left page, right-justify the right page.  I then applied this master page to the appropriate pages.  Make certain to change the setting for a page number of the first page and have it start at the appropriate page number using "Page number format".
For complete instructions on applying master pages see: http://office.microsoft.com/en-us/publisher-help/apply-master-pages-to-publication-pages-HP001004618.aspx?CTT=1
