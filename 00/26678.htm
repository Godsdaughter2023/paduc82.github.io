Adding HTML tags to selected text - Microsoft Community

matthew_m say:

Adding HTML tags to selected text

So lately I've had to to a lot of text prep for publishing on website, which requires me to add the HTML tags (mostly just H1, H2 and P) to the text. I was wondering if it's possible to be able to select a text, and then keybind a command (I'm guessing i'd have to macro this, possibly script?) to add chosen HTML <tag></tag> to the selected text, at the start and end, respectively.

So, for example, i'd select a paragraph in Word, press Ctrl+Alt+P, and it the paragraph would have <p> </p> tags added.

If this is possible, i'd love to know how, as this would greatly speedup my work.
guessed say: The following macro applies to the whole document but you could change ActiveDocument to Selection if you wanted to restrict it to the selected paragraphs
Sub TagParas()
  Dim aPara As Paragraph, sText As String, sOpenTag As String
  Dim i As Integer, rng As Range, iOL As Integer, aHL As Hyperlink
  Dim sCloseTag As String, sCloseTagH As String
  
  sCloseTag = "</p>"
  
  For Each aHL In ActiveDocument.Hyperlinks
    Set rng = aHL.Range
    rng.InsertAfter "</a>"
    rng.InsertBefore "<a href='" & aHL.Address & "'>"
  Next aHL
  
  For i = ActiveDocument.Paragraphs.Count To 1 Step -1
    Set rng = ActiveDocument.Paragraphs(i).Range
    rng.MoveEnd Unit:=wdCharacter, Count:=-1
    iOL = rng.ParagraphFormat.OutlineLevel
    Select Case iOL
      Case 10
        sOpenTag = "<p class='" & Replace(rng.Style, " ", "") & "'>"
        'sOpenTag = Replace(sOpenTag, "'Heading", "'h")
        rng.InsertAfter sCloseTag
        rng.InsertBefore sOpenTag
      Case Else
        sOpenTag = "<h" & iOL & ">"
        sCloseTagH = Replace(sOpenTag, "<", "</")
        rng.InsertAfter sCloseTagH
        rng.InsertBefore sOpenTag
      End Select
  Next i
End Sub

Sub AutoNew()
Dim i As Long

                .SpaceBefore = 2

            End With

        End With
    Next i
    .TablesOfContents(1).Update
    .TablesOfFigures(1).Update
    .TablesOfFigures(2).Update
    .SaveAs .AttachedTemplate.Path & "\Handbook.docx", FileFormat:=16
