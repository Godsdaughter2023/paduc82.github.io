Apply this code to range A2:A500, instead of active cell. - Microsoft Community

naseersheikh say:

Apply this code to range A2:A500, instead of active cell.

This code separates multiple lines in a cell (Alt+Enter) in separate cells. How do i apply this vba code to apply to a range A2:A:500, it looks like this code is limited to work on active cell. Basically i need to apply this to a range. Thank you!

Sub seperatealtenteritems()
Dim s As String, v As Variant, l As Long
s = ActiveCell.Value
v = Split(s, vbLf)
l = UBound(v) - LBound(v) + 1
If l > 1 Then
 ActiveCell.Offset(1, 0).Resize(l - 1, 1).EntireRow.Insert
 ActiveCell.Resize(l, 1).Value = Application.Transpose(v)
End If
End Sub
