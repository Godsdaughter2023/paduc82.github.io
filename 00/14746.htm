john korchok say:

If your presentation contains videos, consider converting them to a lower resolution, or place the video files in the same folder as the presentation and link them instead of embedding.

If  you have lots of photos, consider saving them as JPEG file with a higher compression or lower quality. There is usually only a small loss of visual quality for a large savings in file size.
