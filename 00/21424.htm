ashish mathur say:

Hi,

That formula returns the last filled up entry in the range A2:V2.  Here is what happens:

1. A2:V2<>"" - This portion checks for whether the cells in range A2:V2 are blank or not.  If a cell is blank, the result is FALSE, else TRUE

2. 1/(A2:V2<>"") - This portion return either 1 or #DIV/0! error

3. The LOOKUP() function has a property to search from right to left and return the first numeric value, if the lookup value (in this case 2) is a number greater than what is returned in step 2 above

4. The corresponding number is returned from A2:V2
