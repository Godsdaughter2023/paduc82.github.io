Network signals can be affected by what surrounds you when making calls, such as density of buildings in the area ,or even mountains can also mobile signal. We would suggest that you try the following, which can help you improve network signal:

  1. Restart your phone and make sure that your phone is not in the airplane mode.
  2. Check the signal strength on the phone screen, and move to an area with better reception. If you continuously experience low signal strength in a certain area, the mobile network coverage may be insufficient there. With Netradar you can check mobile operator coverage maps.   3. Change the network selection mode to automatic. Select Settings > Network & wireless > Cellular & SIM > SIM settings (Lumia with Windows Phone 8: Settings > cellular+SIM and tap SIM settings). Ensure that Network selection is set for automatic.

roxannenuckols say:

Auto-Name Spreadsheet Tabs

I am working in Excel 2010 and want each tab in the workbook named from the same cell location (A5) on each worksheet.

Each worksheet has a different name in cell A5 and that is what I want the tab to be named also.

I'm not a programmer and don't know code. is there an easy way?

hansv mvp say:

Press Alt+F11 to activate the Visual Basic Editor.

Select Insert > Module.

Copy the following code into the new module:

Sub RenameSheets()
    Dim wsh As Worksheet
    On Error Resume Next
    For Each wsh In Worksheets
        wsh.Name = CStr(wsh.Range("A5").Value)
    Next wsh
End Sub

With the insertion point anywhere in this line, press F5 to run the macro. This will initialize the worksheet names to the value of cell A5 in each sheet.

Double-click ThisWorkbook under Microsoft Excel Objects.

Copy the following code into the ThisWorkbook module:

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
    If Not Intersect(Sh.Range("A5"), Target) Is Nothing Then
        On Error Resume Next
        Sh.Name = CStr(Sh.Range("A5").Value)
        If Err Then
            MsgBox "'" & Sh.Range("A5") & "' is not a valid sheet name!", vbExclamation
        End If
    End If
End Sub

This code will update the worksheet name automatically when cell A5 is changed.

Don't forget to save the workbook in a format that supports macros (.xlsm, .xlsb or .xls but NOT .xlsx)
