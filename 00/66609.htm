cornpuff say:

I had the same problem, and I found a work around.

On the Xbox One, go to settings->TV->Devices

Enter your device as an "Epson EMP745". That model uses the same IR code for power as the 8350.

To get power down to work, I had to disable the confirmation prompt on the 8350. It's under Menu->Settings->Display...->Standby Confirmation. Turn the standby confirmation to Off.

In case you are curious, look up this phrase "NEC-Format (Custom Code = 8355h)" to find the projectors that use the same IR signals as the 8350.

Also, make sure to go to XboxOne->Settings->TV->Power Settings to make sure the Xbox can power off your projector.

Enjoy!

If the C drive is full, I suggest you to extend the C drive and add some space. Please follow the below steps to know how to add volume to c drive:

- Press Windows key and R to open the Run prompt.

- Type diskmgmt.msc and press enter.
