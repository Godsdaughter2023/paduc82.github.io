    You posted to a thread that's more than 8 months old.  No one is going to re-read it to find out what "a similar problem with the spooler log" means.

    The usual reason for the error message you received is that you had a Lexmark printer and it was incompletely uninstalled.

    To check,
    • Open Services (Start > Run > services.msc > OK)
    • Scroll down to Print Spooler and double click it.
    • Click the Dependencies tab and wait a few seconds.
    • If the top box ("This service depends on the following system components") has anything in it
    Assuming that there was an entry similar to LexBce in that box, fix as follows

    Open a Command Prompt window (Start > Run > cmd > OK) and type the following lines and press Enter after each one

         net stop spooler
         sc config spooler depend= RPCSS
         net start spooler

    Note that there is a space after the = but none before.
