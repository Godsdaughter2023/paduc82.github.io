shenan stanley say:

Not sure you recall that path quite correctly in Windows XP... ;-)  If you mean the Start Button menus..

 

  • "C:\Documents and Settings\<username>\Start Menu\Programs\" (User specific)
  • "C:\Documents and Settings\All Users\Start Menu\Programs\" (All users)

 

Now - for Windows 7 they are located at:

 

  • "C:\Users\<username>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\" (User Specific)
  • "C:\ProgramData\Microsoft\Windows\Start Menu\Programs" (All users)

 

Tip for the future...  You can "Explore" or "Open" those directly from the Start Menu.  The one folder that is consistently around (so far) in Windows OSes is the "Startup" folder.  So my suggestion is to locate the "Startup" folder in the Start Menu and right-click on it.  You should get a few choices including "Open" and "Explore" and "Open all users" and "Explore all users" and depending on what you choose - you can get to that folder (user specific or all users) and then navigate up a directory for the root of the "All Programs" list.
