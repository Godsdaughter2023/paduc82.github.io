All files have more than one date property: date created, date accessed, date modified. On top of that, .eml files also have date sent and date. If you view an .eml file in an Explorer folder, you will usually see the date modified. In the case of files imported from elsewhere, the file will probably have been modified by Windows Search, which adds the MIME properties of the file (like From, To, Subject and Date) to the file itself. To see the date of the messages, select the appropriate property in Explorer by right-clicking on a column heading in Details view.

ronnie vernon say:

Hi Jim

 

This is a new component that has been added to the Disk Cleanup utility by a recent Windows Update.

 

This option cleans out all of the outdated and redundant files that are installed in the Component Store (Winsxs folder).

 

Users have been complaining about the size of this folder ever since Windows 7 was first released.

 

It is safe to use and as you can see it will free up a lot of disk space.

 

You can get complete information about this new cleanup option and a brief tutorial on how to use it in the following article.
