Found a way to create user profile (which is not possible when doing it via Windows 10 settings UI). Right click on the windows start menu, select Command Prompt (Admin). Create a new user profile using the following syntax: net user username password /ADD (e.g.: net user johndoe 12345 /ADD).

clb7419 say:

PREVIEW OR PEEK FEATURE

I need to disable the "Peek" or "Preview" feature that is standard on Windows 7.  I created a D-word entry in the registry: DisablePreviewWindow without sucsess.  Am I missing something else?? 

ronnie vernon say:

I need to disable the "Peek" or "Preview" feature that is standard on Windows 7.  I created a D-word entry in the registry: DisablePreviewWindow without sucsess.  Am I missing something else?? 

There is some confusion about the names of these features.

 

1. Thumbnail Preview: When you have a program open and you hover the cursor over that programs Taskbar Icon,  you will see a small Thumbnail of the open window.

 

2. Preview Window: When you have a program open and you hover the cursor over the Thumbnail Preview of the open window,  the full Window that the Thumbnail represents will be displayed.

 

3. Aero Peek: When you hover the cursor over the small, rectangular, Show Desktop button at the right end of the Taskbar, all windows will disappear and you will see the Desktop, until you move the cursor away from that button.

 

4. Show Desktop: When you hover the cursor over the Show Desktop button andclick, all windows will be minimized to the Taskbar and you will be on the Desktop.

 


The registry value that you mentioned should disable # 2.
