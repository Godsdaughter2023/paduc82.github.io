Microsoft released the first beta back in June 2010 followed by a Release Candidate late October 2010 and the final RTM availability (February 16 2011 MSDN/TechNet subscribers) and public Availability (February 22 2011). Unlike previous Service Packs, Windows 7 SP1 does not contain any major improvements; it’s basically a rollup of updates that have been released for the operating system since it went to manufacturing July 22 2009. If you have been diligently updating your computer through Windows Update since then, you basically have all that SP1 has to offer.

Windows 7 SP1 is provided in two ways, either as a full .ISO file download weighing in at 2.4 GBs or as an update available through Windows Update. The .ISO file contains SP1 code for different processor architectures such as x86, x86-x64 and Itanium systems. Because I have a mixture of 32 and 64 bit systems, I went with the .ISO file. Hi,

How do you know there was a Vista COA sticker that was replaced with a XP COA?

 

These are general Licensing examples for transferring Operating Systems:

 

If the Laptop came with the Windows Operating System preinstalled (Dell, HP, Acer, etc), that Product Key that came with the Laptop is for the preinstalled OEM License and is non-transferable.

 

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

 

Only Retail Operating System License, that you buy and install yourself, are allowd to be transferred to another machine.

And Retail Licenses can also only be installed on one machine at any particular time.

 

There is also downgrade Rights from Vista Business and Vista Ultimate to XP Pro on OEM, etc machines.

http://www.microsoft.com/OEM/en-gb/licensing/sblicensing/Pages/what_to_do_downgrade_rights.aspx#fbid=6lmZhw78uY7

xcpf jedi say:

Bonjour,

 

Il ne te sera pas facturé deux fois l'abonnement, simplement, le fait d'avoir inscrit un moyen de paiement par prélèvement, en l'occurence une carte bancaire, implique que ce dernier soit valide, ce qui dans ton cas, ne l'est plus.

 

Donc, deux options possibles :

  1. L'enregistrement d'une carte bancaire valide à ce jour
  2. Demander au Service clientèle, la suppression du renouvellement automatique, dans ce cas tu aura la possibilité d'utiliser à ta guise des codes prépayés.

Ce qui peut se produire si l'une de ces options n'est pas remplie est une suspension temporaire du compte, en effet, pour le service facturation, la carte bancaire enregistrée n'est plus valide mais le renouvellement automatique est toujours activé, il y a donc risque de dysfonctionnement.

 

En conclusion, mon conseil serait de te rapprocher du Service clientèle afin de supprimer le renouvellement automatique, ainsi pas de problème en perspective et libre à toi ensuite de choisir la façon dont tu souhaite poursuivre ton abonnement.

 

Cordialement
