SCREENSHOTS IN WINDOWS 10 - Microsoft Community

breaking dad say:

SCREENSHOTS IN WINDOWS 10

Is it possible to change the default saving of screenshots from .png to .jpg? Why on earth did Microsoft make .png the default?

ronald j w say:

Is it possible to change the default saving of screenshots from .png to .jpg? Why on earth did Microsoft make .png the default?

PNG files are lossless, which means that they do not lose quality during editing. This is unlike jpegs, where they lose quality. PNG files tend to be larger than jpegs, because they contain more information. The screenshot and snipping tool have to Use data intelligently to make a pick on the spot.

ricw-nw say:

FWIW - Plan B would be to use the alternative method of saving screenshots (just hit the PrtScr key, and then open Paint - paste from clipboard icon - save as...jpeg). I use this method because I am normally editing screenshots immediately anyway, but I realize it wouldn't be convenient if you are having to take numerous shots quickly.

There's always the snipping tool which gives the jpeg option, but then you can't save dialogue boxes! - Ric.

sam___ say:

Breaking Dad,

Are you talking about the snipping tool? If so, it allows you to change the format to JPEG. PNG is the default because it yields better compression than JPEG when capturing simple images such as windows. JPEG achieves better compression than PNG when saving photos or rich graphics.
