stefan blom say:

In the Date and Time dialog box, the check box for "Update automatically" applies to the date or time that you are inserting. Previously inserted dates (in other documents or in the same document) won't be affected.

When an automatic date has been inserted, what you are dealing with is usually a DATE field (or a TIME field). You can replace the field with CREATEDATE in existing documents.

Do the following to fix the problem after it has occurred: In an affected document, press Alt+F9 (Alt+FN+F9 on some keyboards) to display field codes. Most likely, you'll see { DATE } (or { TIME }) in place of the date that keeps updating. Edit the field code so that it says CREATEDATE and press F9 (or FN+F9) to update. Press Alt+F9 again to hide field codes and redisplay results.

Private Function GetAllFiles(ByVal strPath As String, _
ByVal intRow As Integer, ByRef objFSO As Object) As Integer
Dim objFolder As Object
Dim objFile As Object
Dim i As integer, j As integer
Dim v(1 To 2) As String
Dim bPath As Boolean, sPath As String
sPath = strPath
If Right(sPath, 1) <> "\" Then sPath = sPath & "\"
' list the path substrings you want to check
v(1) = "\Documents\Main\"

bPath = False

     bPath = True
  End If
Next
Set objFolder = objFSO.GetFolder(strPath)
If bPath Then
For Each objFile In objFolder.Files

        'print file path

        i = i + 1
   End If
Next objFile
End If
GetAllFiles = i + ROW_FIRST - 1
End Function
