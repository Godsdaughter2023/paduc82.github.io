The problem may arise because:

Duplicating user settings - Microsoft Community

jjacobsen say:

Duplicating user settings

I want to set up multiple user accounts on the same computer.  After I have set up the first user (administrator), how can I duplicate the settings from that user to another user without having to set everything again?

fahimulla k say:

Hi,

To create a duplicate of a user profile (with a different user name), use these steps:

·         Create a new user account

·         Logon to that account to initialize the newly created profile

·         Log off from the newly created profile

·         Login as Administrator (the built-in administrator account)

·         Open Control Panel, double-click System

·         Click the Advanced tab

·         Click Settings under User Profiles

·         Select a profile to copy from and choose Copy To

·         Browse to the profile to copy to (C:\users\newuser)

·         In Permitted to use, click Change

·         In the Select User or Group dialog box, type the new user account name and press ENTER

 

Though you can move or rename the user profile folder, there may be some side effects after using this method. This is because of the reason that there may be some absolute path references (to the old user profile folder) in the registry

 

Also see:

Create a user account

http://windows.microsoft.com/en-US/windows7/Create-a-user-account

Fix a corrupted user profile

http://windows.microsoft.com/en-us/windows7/fix-a-corrupted-user-profile

Enable and Disable the Built-in Administrator Account

http://technet.microsoft.com/en-us/library/dd744293(v=ws.10).aspx
