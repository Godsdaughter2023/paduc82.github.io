You can get an effect like this:

For each element where you want to show the oxidation state, insert a 2x1 matrix structure from the Equation Tools ribbon. By default, the vertical center of the matrix aligns with the vertical center of the equation's main text.

Right-click the matrix structure, click Matrix Alignment, and click Bottom to make the lower block align with the main text.

bernie deitrick say:

The best approach is to name the shape (based on its position) just after it is created, but delete it before trying to create it again - this will never duplicate the shapes (so start by deleting all the original shapes):

Private Sub CommandButton1_Click()
    Dim ETD As Variant, ATD As Variant, ETA As Variant, ATA As Variant, Dur As Variant     ' estimated and actual times and duration of flight
    Dim Callsign As Shape
    Dim lngR As Long
    For lngR = 3 To Cells(Rows.Count, 2).End(xlUp).Row
        ETD = Cells(lngR, 2).Value
        ATD = Cells(lngR, 3).Value
        ETA = Cells(lngR, 4).Value
        ATA = Cells(lngR, 5).Value
        Dur = ETA - ETD
        
        On Error Resume Next
        ActiveSheet.Shapes("CallSignRow" & lngR).Delete
        On Error GoTo 0
        Set Callsign = ActiveSheet.Shapes.AddShape(1, (Cells(lngR, 7).Left + (ETD * 24) * Columns(7).Width), Cells(lngR, 7).Top, (Dur * 24) * Columns(7).Width, Cells(lngR, 7).Height)
        Callsign.Name = "CallSignRow" & lngR
        
        Callsign.Fill.ForeColor.SchemeColor = 43
        Callsign.Fill.Solid
        Callsign.Visible = True
        With Callsign.TextFrame
            .Characters.Text = Cells(lngR, 6).Value & " " & Format(Dur, "hh:mm")
            .Characters.Font.Size = 18
            .Characters.Font.ColorIndex = 3
        End With
    Next lngR
    
End Sub

I would also be remiss if I did not tell you that you could do something very close to this with conditional formatting and formulas
