ns.Folders is the collection of top-level folders, e.g. Personal Folders, Archive Folders etc. If the top-level folder that you want to use has a unique name, you can refer to that:

 

Set Inbox = ns.Folders("My Folders").Folders("Inbox")

Set Inbox = ns.Folders(2).Folders("Inbox")

First thing to do is remove the "Answer" flag from the other reply so other people will know to still look at the question to possibly make suggestions.

PDF is an Adobe format, so you should be talking to them to find out about creating an index directly on a PDF. I don't think it is possible, but who knows, maybe it is.

What you could do is OCR the PDF into an editable format, like a DOC/DOCX. Then you can generate the index and recreate the PDF, with the new index.

jim gordon mvp say:

The simplest web query is just a URL saved as a text file with a .iqy file extension. To make one type or paste a URL that has at least one HTML table into a blank word document, then use FIle > Save As and change the file format to Plain Text (.txt). When prompted about the file conversion click the MS-DOS option button and then click the OK button. You can change the file extension to .iqy in the Save As dialog. The web query will work whether it is saved as .txt or .iqy but if you use .iqy then you can visually see that the saved file is a web query.


Now, in Excel, use the Data menu and choose Get External Data > Run Saved Query. Choose your saved query file and Excel will then ask you what cell to put the query results into, as well as offer a host of other options that you can use with your query.


If you look in this directory on your Mac you will find 3 examples that are more robust using MSN MoneyCentral Applications:Microsoft Office 2011:Office:Queries
