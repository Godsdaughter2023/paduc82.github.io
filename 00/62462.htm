smwutches say:

When you download content it's tied to the gamertag that purchased it as well as the console that it was purchased on. All profiles that are on the console with the console license can access the content (avatar items cannot be accessed this way). If you get a new console, or use the gamertag on another console, only that profile will be able to use the content as long as it's signed into Live. Otherwise, you can do a license transfer to move the console license to the new console.
