a. user say:

The only way to restore your computer to the way it was on any given date is to restore an image backup that you made on that date.

System Restore is not a method for backing up. It's a method of restoring the basic guts of Windows, to give you an opportunity to (hopefully) work your way out of a jam. Only a backup is a backup.

Your restore points will eventually age out and be deleted. And all restore points, regardless of age, will be deleted after the next feature update. A backup lasts forever.

It's never too late to start making regular image backups. There is a learning curve to making backups, but it's nothing that you can't learn. Once you learn how it's done, and you see how easy and powerful it is, you'll never lose data and you'll never have to re-install Windows.

Changing Wireless Security Type/Disabling

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. In the address bar at the top, type in 192.168.2.1
  4. Click Login in the upper right.
  5. Input the password and click Login (by default, the password should be left blank).
  6. Select Security from the menu on the left under the Wireless section.
  7. To change or disable the Wireless Security, select the desired Security Mode from the dropdown (Disabled, WPA/WPA2-Personal(PSK), 128-bit WEP or 64-bit WEP).
  8. If WPA/WPA2-Personal(PSK) is selected, choose the Authentication (WPA-PSK, WPA2-PSK or WPA-PSK + WPA2-PSK) and Encryption Technique (TKIP or AES).  WPA2-PSK with AES is recommended.
  9. Enter the Pre-Shared Key or WEP Key.
  10. If WEP security is selected the password will need to be 10 charecters in length (if 64-bit) or 26 characters in length (if 128-bit) and can only use 0-9 and A-F.
  11. If WPA or WPA2 was selected the password can be between 8-63 characters, is case sensative and can use any character on your keyboard.
  12. Once the passphrase has been entered select Apply Changes at the bottom of the page.

Although by default the task bar appears on the bottom of the screen,
