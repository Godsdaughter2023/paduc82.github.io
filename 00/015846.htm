The problem is you are using an input mask on the Grand Total control. This makes Access see it as a text field, so it doesn't offer the Sum option for datasheet total. Format it as Currency and that should take care of it.

tom van stiphout say:

I didn't see an input mask.

The way I solved it is to go back to the Tasks table, delete the Grand Total field, and create a new GrandTotal field, make it Calculated field, and set formula to =[Cost]*[Quantity]*(1+([VAT]/100))

Then go into the Tasks subform and set the Control Source for the Grand Total field to GrandTotal.

 

As an aside, it is bad practice to use the range from 0-100 for percent fields like VAT. It's easier and more regular to use 0-1, and format the field as percent. Of course that changes your formula to =[Cost]*[Quantity]*(1+[VAT])

Some middle cases are a a bit tight which make the back case hard to open.

Instructions for opening case:

1. Place handset in left hand (screen down)

2. Place left thumb just under groove that comes from camera

3. Place right thumb at the bottom of case so that the top of the thumb can press the button but bottom of thumb will push very bottom of case down.

4. with both thumbs working together push down and up with right thumb and up with left thumb.

Case SHOULD click open, if the button has moved up inside the back case the plastic springs have bent or broken and the unit needs to go into a CNSP.

New Paragraph Pushed to Following Page - Microsoft Community

clarkuskent say:

New Paragraph Pushed to Following Page

In Word, when beginning a new paragraph toward the bottom of a page, Word often pushes that new paragraph to the next page.  This creates a bunch of unused white space at the bottom of the page -- where the new paragraph ought to have started.  Does anyone know how to disable this?  I'm in an academic program where page counts are strictly enforced, and I do not want Word to push the new paragraph to the next page.  I want instead to use every line of every page. 

Anyone know how to disable this feature?  Any expert help would be appreciated!

Thanks,

CK
pamcaswell say: Turn off widow and orphan control (Home tab > paragraph group > Arrow in lower right corner of group > Line and Page Breaks tab. Clear the Widow/orphan control check box). If you don't want to have to do this on every page, modify the normal paragraph style or whatever style you are using for regular text.

Pam 

jeeped say:
In Word, when beginning a new paragraph toward the bottom of a page, Word often pushes that new paragraph to the next page ...

There are paragraph formatting options like Widow/Orphan control, Keep with next, Keep lines together and Page break before that can force a a paragraph onto the following page depending on other conditions. Look at the Paragraph dialog's Line and Page Breaks tab for details on how your own paragraph style is set up.

More information at,
Control pagination
