You can't do that with a formula, it requires a bit of VBA.

Right-click the sheet tab.

Select 'View Code' from the context menu.

Copy the following code into the worksheet module:

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Range("B1"), Target) Is Nothing Then
        If Range("B1").Value <> 0 Then
            Application.EnableEvents = False
            Range("A1").Value = Range("B1").Value
            Application.EnableEvents = True
        End If
    End If
End Sub

Switch back to Excel.

Save the workbook as a macro-enabled workbook (.xlsm).

Make sure that you allow macros when you open the workbook.

hansv mvp say:

Activate the Visual Basic Editor (press Alt+F11).

Double-click ThisWorkbook under Microsoft Excel Objects in the pane on the left hand side.

Copy the following code into the ThisWorkbook module:

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
    Dim cell As Range
    If Not Intersect(Sh.Range("I2:I" & Sh.Rows.Count), Target) Is Nothing Then
        Application.EnableEvents = False
        For Each cell In Intersect(Sh.Range("I2:I" & Sh.Rows.Count), Target)
            If cell.Value <> 0 Then
                cell.Offset(0, -1).Value = cell.Value
            End If
        Next cell
        Application.EnableEvents = True
    End If
End Sub

I have assumed that you want to exclude row 1 from being handled the same way.

If you do want the code to apply to row 1 too, change both occurrences of I2 to I1.

If you want to exclude more rows, adjust I2 accordingly, for example if you want to exclude rows 1 to 3 and start applying the code in row 4, change both occurrences of I2 to I4.
