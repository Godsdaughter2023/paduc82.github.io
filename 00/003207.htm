While the collateral damage it may do may be unacceptable, a workaround is to insert another Heading 1 style paragraph before the one that you currently have set the numbering to start at 21 and set the numbering for that new Heading 1 to start at 20 (You can change the Heading 1 with the number 21 back to Continue from previous list.  Then to hide the Heading with the number 20, format it as white.

You may all use the same default printer but unfortunately, those with a different OS are using a different printer driver. The result is different text flow because the printer driver is providing Word with slightly differing layout parameters. Make sure that all the workstations have the latest printer driver appropriate to their OS. There's not much you can do about this except send the documents as PDFs.
