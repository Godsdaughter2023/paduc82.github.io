Maybe some fields don't have five field parts.

Try this modification:

Public Function SplitField(ByVal Value As String, ByVal Index As Integer) As Variant

    If Ubound(Split(Value, ",")) >= Index - 1 Then

        SplitField = Split(Value, ",")(Index - 1)

    Else

        SplitField = Null

    End If

End Function

Method 1:

To disable dial-up-connection dialogue , follow these steps:

  1. Start Internet Explorer. On the Tools menu, click Internet Options.

If issue persists follow method 2:

 

Method 2:

To set the AutoConnect value in the registry key to 0, follow these steps:

  1. Click Start, and then click Run.
  2. In the Open box, type regedit, and then press ENTER.
  3. Locate and select the following registry key:

HKEY_CURRENT_USER\RemoteAccess\Profile\<ConnectionName>

In the right pane, double-click the AutoConnect value, and then reset the value from 1 to 0.
