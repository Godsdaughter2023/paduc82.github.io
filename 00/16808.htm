scott mcdaniel say:

Generally speaking you don't store a calculated value, unless it is for historical reasons and you're flattening your data (and you're not doing that).

Any sort of "transactional" data should be stored in a Transaction table, and your query should refer to that table to determine the "status". The classic example is the Shipping/Receiving application where I'd have PurchaseOrder table  structure for those items I've ordered from my suppliers and a Transaction table structure to record the movement of those items in my system.

So let's assume I issue PO 1123 for an item named "Widget01" for a Quantity of 10.

I'd add a single record to my PO_Header table. This would record the PO_Number, PO_Date, Vendor, etc.

I'd add a single record to my PO_Detail table. This would record the PO_Number, Item_Number, Quantity, etc.

When I received the shipment from my vendor, I'd add at least one record to my Transaction table. If I received 4 items, for example, I'd add a record that would record the PO_Number, Receipt_Date, Item_Number, Qty_Received, etc. Note that I might need to record the PO_Detail line associated with that entry as well, depending on how much detail you will be storing.

So from this I could query my Transaction table for the PO_Number to determine the status of that PO:

SELECT SUM(Qty_Received) AS QtyReceived FROM Transactions WHERE PO_Number='1123' AND Item_Number='Widget01'

I could also query the PO_Detail table to determine how many of those items I should have received:

SELECT SUM(Quantity) AS QtyOrdered FROM PO_Detail WHERE PO_Number='1123' AND Item_Number='Widget01'

If I then subtract the two, I can determine the status.

You did ask for "Automate" so maybe you want a macro?

Sub Copy_PasteSpecial_In_Place()
    With ActiveSheet.Cells
        .Copy
        .PasteSpecial Paste:=xlPasteValues, Operation:=xlNone, SkipBlanks _
                :=False, Transpose:=False
    End With
    Application.CutCopyMode = False
    Range("A1").Select
End Sub

Sub CopyPasteSheetAsValues()
    Dim newSht As Worksheet
    Dim wkSht As Worksheet
    For Each wkSht In Worksheets
    Application.ScreenUpdating = False
    ' delete sheet if it exists
       With wkSht
            If .Name = "Overview-180831-New" Then
                Application.DisplayAlerts = False
                Sheets("Overview-180831-New").Delete
            End If
        End With
    Next
    Application.DisplayAlerts = True
    ' add and name new sheet
    Set newSht = Worksheets.Add
    newSht.Name = "Overview-180831-New"
    'copy from Overview sheet
    Sheets("Overview-180831").Cells.Copy
    'paste values and cell formats to new sheet
    With Sheets("Overview-180831-New")
        .Cells.PasteSpecial Paste:=xlPasteValues
        .Cells.PasteSpecial Paste:=xlPasteFormats
    End With
    Range("A1").Select
    With Application
           .CutCopyMode = False
           .ScreenUpdating = False
     End With
End Sub

The device timed out while waiting for the HTTP gateway

- OR -

Synchronization failed because of a server error. Try again.


Possible Resolution:

Try again later.

User Action:

Use a different proxy or report the issue to the proxy server administrator.
