Well your syntax appears a bit off.  You don't use the bang (!) between table name and field name.  Also, your field name DATE is using an Access Reserved Word (you should not use those as field or object names).  So yolu need it in square brackets.

 See if this helps at all:

 

SELECT [Retail Calendar].[Week Number], Salesperson_ID.Salesperson_Name, [Close Ratio].Ups, [Close Ratio].Sales, [Close Ratio].[Sales]/[Close Ratio].[Ups] AS [close rate]
FROM [Retail Calendar] INNER JOIN (Salesperson_ID INNER JOIN [Close Ratio] ON Salesperson_ID.Salesperson_Name = [Close Ratio].Salesperson) ON [Retail Calendar].[Sales Date] = [Close Ratio].[Date]
GROUP BY [Retail Calendar].[Week Number], Salesperson_ID.Salesperson_Name, [Close Ratio].Ups, [Close Ratio].Sales, [Close Ratio].[Sales]/[Close Ratio].[Ups]
HAVING ((([Retail Calendar].[Week Number])=[Forms]![week]![Combo0]));
