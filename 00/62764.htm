erwin jones say: Hi,

Check whether the Volume Shadow copy services is started.

To do this:
1. Go to start, type services.msc in start search box and press enter.
2. Now, check Volume shadow copy is running. If not set it to Start


Cause of Volume Shadow Copy Error 0x80042313 - flush writes timeout

The VSS backups are failing due the high disk activity. High disk activity does not allow VSS to create a shadow copy in the default time period when the volumes are frozen for snapshot.

Workaround:

Perform the backups or system restore when the disk activity is less or rather put the computer in clean boot state and then perform the system restore to avoid third party applications meddling around.

Clean boot- http://support.microsoft.com/kb/929135
