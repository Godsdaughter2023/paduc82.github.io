What would have been better is if the Microsoft Yammer folks would have simply made those of us going through such a downgrade completely aware of what would transpire and exactly when, instead of having us guess and hope.  But in the end, it all worked out perfectly, we're now a Basic account and for the time being, quite happy with that.

Hi Pam,

Here I'd like to explain that the Custom Site Collection Help Functionality is for on-premise SharePoint Server only, for the time being, it's not available for SharePoint Online. Though we can activate the Custom Site Collection Help feature in SharePoint Online Site Collection Features menu, and enabling this feature does create the Site Collection Help library, still the Help Settings button won't appear under the Site Collection Administration section after clicking Site settings.

For SharePoint Online, when we click the Help icon, it returns some reference articles. And we can also search for other reference articles or some relevant threads in the Office 365 Community in the search box. Nevertheless, it’s not feasible to customize the default content returned in SharePoint Online sites.
