Login Information

  • Gateway - 192.168.1.1 or 192.168.1.254
  • Username - admin |   root   |   user  |   user   | support
  • Password -  admin | 12345 | 12345 |   user   | support

Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254into the address bar. (Or click the link) When prompted enter the admin for the username and admin for the password if the gateway asks.

  4. Select NAT from the submenu on the left.
  5. Select Virtual Servers on the next submenu.
  6. Select the Add button.
  7. Select Custom Service and input the name "Xbox".
  8. Enter the IP Address of the Xbox in "Server IP Address".
  9. Input 3074 for the External Port Start/End and Internal Port Start/End.
  10. Select TCP/UDP for the Protocol.
  11. Select Apply/Save.

Disabling Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter the admin for the username and admin for the password if the gateway asks.

  5. Select NAT from the sub-menu on the left.
  6. Select Virtual Servers on the next sub-menu.
  7. Place a check in each port table to be removed.
  8. Select remove.
  9. Select Apply/Save.

bennyadamsky31 say:

Want to install and use wireless mouse

I want to simply install and use a Microsoft wireless mouse, does this require me to disable the touchpad? If so, how do I do that? I have a Dell Laptop with Windows 10. Sorry folks, I'm an old dude and have little much knowledge regarding pcs. Thanks in advance.

tryx3 say:

Benny,

You do not have to disable the Touchpad.  I use a Wireless mouse but I sometimes find it convenient to use the Touchpad instead so I just leave it as it is.

If, however, you find that you accidentally brush the edge of your hand against the Touchpad and so disrupt what you are doing, you can reduce its sensitivity or turn it off altogether -

  • Right-click on the Start button,
  • Control panel,
  • View by - Small icons [if necessary to see the next entry],
  • Mouse,
  • Dell Touchpad tab,
  • click in the middle of the big picture of the Dell Touchpad.

The remaining steps vary between different Dell models so yours might be a bit different but probably not by very much -

1  To disable it altogether - Device select, [under the label Touchpad], Disable, OK.  That's it so you can close the Dell Touchpad window and the TouchPad will not function at all.  A similar procedure can be used to turn it back on.

2  To disable it only when your external [wireless] mouse is connected - Device select, [under the picture of the Touchpad], set the checkbox "Disable Touchpad / Pointing stick when external USB mouse is present", OK.  That's it - when your wireless mouse is connected the Dell Touchpad will be turned off but when you disconnect your wireless mouse, the Touchpad will be turned back on automatically. 

3  To reduce its sensitivity to the edge of your hand brushing against it - Touchpad settings, under the heading TouchCheck, set the checkbox for Enable.  Now get some Dell Help by clicking on the ? in the upper-right corner of the Touchpad window and then clicking on the TouchCheck settings bar near the bottom of the window.  A screentip should appear explaining what TouchCheck does [it alters the sensitivity level].  Make the setting closer to Maximum if you are getting messed about when you accidentally brush your hand against the Dell TouchPad, OK.  That's it so you can close the Dell Touchpad window.  You can go back in later on if you have not set it as well as you wanted - it can be set/reset as many times as you want.

- - - - - - - - -

You also asked about setting up the wireless mouse itself.  This should be explained in the instructions that came with your mouse or that can be found on the Microsoft website.  The precise procedure varies between mice but in general:

  • Find its wireless dongle that looks like a USB connection with a lump of plastic at the end**.
  • Plug that dongle into a USB connection on your computer.
  • Fit the battery inside the mouse.
  • Wait for a bit for Windows to recognise the dongle and for the dongle & the mouse to set up their private communications channel.
  • Start using the wireless mouse.

**   If you cannot immediately find the dongle in the box the mouse came in, have a look at the underside of the mouse.  Many of them have a small slot that the dongle fits into for storage & transit.  Just pull the dongle out.  The point of storing it inside the mouse is that you can keep the mouse in your bag without any risk of losing the small dongle.  If, as is the case with many dongles, it only sticks out from the computer a small way then leaving it plugged in always would be the easiest way of making sure you didn't lose it.  The mouse and the dongle are an inseparable pair - if you lose one you have to replace both.

jimwynne say:

You can't do an in-place upgrade; you'll have to do a clean installation, and the free Windows 10 offer isn't open to Vista users.  Having said that:

  1. You can see the requirements here; pay special attention to the processor requirements.
  2. You should check the computer manufacturer's website to see if Windows 10 drivers are available.  They might not be necessary, but it's a good thing if they are.  The manufacturer might also advise that Windows 10 isn't supported or hasn't been tested on your hardware.
  3. Look for Windows 10 drivers for your hardware devices on the device manufacturers' website.
  4. Make sure you have a good backup of your data and make a system image, and you'll have to reinstall programs if the installation is successful.

mm pearson say: OK I am posting in case it will help anyone else who might find this.  I got it working. My solution after many tries to repair, reinstall, etc. was this:

In this order I performed these steps:

1) Uninstalled .Net 4 Expanded
2) Rebooted
3) Uninstalled .Net 4. Client
4) Rebooted
5) Tried first to uninstall all the MS Expression 4 programs but they were not listed on the ADD/REMOVE programs in the Control Panel, nor was there an "uninstall" option on the menu item for Expression. So I deleted the shortcut icons from the desktop, opened Windows Explorer, went to /Programs/Expression/ and deleted all files and sub-directories under that. Not the best way to uninstall, but it got the job done since couldn't find an uninstall avenue anywhere else.
6) Rebooted
7) downloaded & installed .Net 4 Client FIRST
8) downloaded & installed .Net Expanded/Full NEXT
9) rebooted
10) re-downloaded my Expression files (from the original link on my purchase receipt email) and installed Expression clean, like new
11)Voila! working Expression Web 4 is up and running!

Got the new .Net 4. client and expanded install files from the .Net support site on support.Microsoft.com
The order of the uninstall and clean installs mattered. AND the reboots in between each.

Hope it helps; sure it isn't as technical as Tech Support, but then, there didn't seem to be any of that around for this for some reason. Plenty of threads with this same problem however. Anyway, I'm running smooth.
