rich michaels - mvp (greatcirclelearning.com) say:

This will only run the first time from Excel if an instance of Word is already open. And why it fails the second time is due to the last command line of Set wordObj = Nothing with kills the open instance of Word.

Here is a clip from some code I use in PowerPoint for opening and using the Word object. The same technique can be applied to Excel and Word.

    On Error GoTo NoWord
    Word.Application.Activate
    Set wApp = Word.Application
    pApp.Activate
    On Error Resume Next
    GoTo GotWord
NoWord:
    Set wApp = CreateObject("Word.application")
GotWord:

You can code this in many different ways to suit your particular coding style but in essence it is doing this...

Due to the first On Error setting if "Word.Applicaiton.Activate" fails, which it will if Word is not already open, the thread resumes after the "NoWord" label where the Word.application instance is started. If Word is already open then the wApp object variable is set to the open instance of Word. The pApp.Activate command applies to activating PowerPoint, which in your case you'd substitute for activating Excel.

With the above done, both Word and Excel are open and Excel is the app with focus.

Hope that helps

To show every day in a month, you need a utility table (named Numbers) that containd one field (named Num) and populated with values 1 through 31.

With that taken care of, I think the query you want would look like:

SELECT Num, Count(personID)
FROM yourtable
WHERE Num <= Day(DateSerial(Year(Date()),[Enter month number]+1,0))
   And startdate <= DateSerial(Year(Date()), [Enter month number], Num)
   And enddate >=  DateSerial(Year(Date()), [Enter month number], Num)
GROUP BY Num

mike h.. say:

Hi,

 

A conditional format can't return a text response. You could have your text string in column F and have the font colour the same as the background and then use conditional formatting to change the font colour to make it visible or you could use code like this

 

Private Sub Worksheet_Change(ByVal Target As Range)
Dim Oset As Long
Dim Tsum As Double
If Target.Cells.Count > 1 Then Exit Sub
If Not Intersect(Target, Range("D:E")) Is Nothing _
And IsNumeric(Target.Value) Then
    Application.EnableEvents = False
     If Target.Column = 4 Then
        Oset = 2
        Tsum = Target.Value + Target.Offset(, 1).Value
    Else
         Tsum = Target.Value + Target.Offset(, -1).Value
         Oset = 1
    End If
   
    If Abs(Tsum) > 999.99 Then
        Target.Offset(, Oset) = "Don't forget the variance explanation"
    Else
        Target.Offset(, Oset) = ""
    End If
               
    
    Application.EnableEvents = True
    End If
End Sub
