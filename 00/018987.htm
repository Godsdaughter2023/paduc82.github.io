Private Sub Worksheet_Change(ByVal Target As Range)

a. user say:

Hello Sandeep,

If the admin has applied an Outlook Web App policy to restrict certain users from creating Office 365 groups, they will still be able to create groups in Yammer (as there is no restriction on group creation in Yammer).

https://support.office.com/en-us/article/Yammer-and-Office-365-Groups-d8c239dc-a48b-47ab-b85e-6b4b8191a869?ui=en-US&rs=en-US&ad=US&fromAR=1

However, the groups that these users create in Yammer will not be replicated to Office 365 as they have been restricted from creating OWA groups.

Applying Outlook Web App policies to restrict group creation in Office 365:

  1. Log in to Office 365 admin center and go to Exchange Admin Center.
  2. Navigate to permissions -> Outlook Web App policies.
  3. Create a new policy by giving a name of your choice.
  4. To apply this policy to users, go to Exchange Admin Center -> recipients -> mailboxes and select the name of the user.
  5. In the work pane, under Email Connectivity, click on View Details.
  6. Apply the policy you’ve just created.
  7. Open PowerShell and login to Exchange PowerShell.
  8. Run the command: Set-OWAMailboxPolicy -Identity “name of the OWA policy” -GroupCreationEnabled $false
  9. The changes will be applied in 15 minutes to 1 hour.
  10. The users will now be restricted from creating OWA groups.
