vaibhavpankaj say:

can i install windows 10 on oracle virtual box using the iso??

hi!

is it possible to install windows 10 technical preview on the oracle virtual box using the .iso file since there could be potential data loss??.

if the above thing is possible could i still be a part of the insider program and give feedback??

andre da costa say:

Yes, you can, but make sure you configure the Virtual Machine with the right settings.

Installing in a Virtual Machine is the safest way to test Windows 10.

To install W10 Tech Preview on Virtual Box, I made the following settings in V-Box:

Use the "Windows 8.1" selection for the version of windows that you are installing. (General Section of V-Box)

Check "Enable I/O APIC" (System Section, Motherboard)

Enabling PAE/NX doesn't seem to make any difference (System Section, Processor)

Enable VT-x/AMD-V and Enable Nested Paging (System Section, Acceleration)

Enable 3D Acceleration (Display Section, Video) - the V-Box extensions won't install unless you do this. (You need the V-Box extensions both to increase screen resolution and to access a network.)

Do not enable 2D Video acceleration - this caused my screen to flash on and off!

If V-Box does not work at all - with any OS - then you might need to enable virtualisation in the BIOS.

I had also tried Hyper-V, but it was too invasive, blocked V-Box execution and was impossible to uninstall without a System Restore.

Disable UEFI
