Your "Applies To" line says Excel 2003.

But I wonder if you are really using Excel 2007or later, and your references to Stmt, Credit and Debit refer to column headings of an Excel Table created by clicking on Insert, Table.

In that case, not only can a heading have leading spaces, but also if it does, you must including the leading spaces in the Table reference.

However, the correct syntax is [Stmt], [  Credit] or [[  Credit]], and [  Debit] or [[  Debit]] (Excel converts the first form to the latter form automagically.)

On the other hand, if you have Excel 2003, you cannot use Table references per se.  Instead, you must set up named ranges.  And in that case, the named ranges cannot have leading spaces, AFAIK.
