Hi,

It is possible to format headers in Word using VBA, but it has its pitfalls.  You should read, and re-read this page before attempting this:  http://www.word.mvps.org/FAQs/MacrosVBA/FindReplaceAllWithVBA.htm

Note the above was written about Word 2003.  I happen to be working on a PC with Word 2003 as I type this, so I have not tested this on 2007, but I doubt it will have changed much (famous last words).

That said, the other issue to be aware of is whether your first page header is different because "Use different first page headers" is checked in Page Setup (Layout), or because the first page is in Section 1, and the second page is in Section 2... and that Section 2's header is not linked to Section 1's header.... thus making Section 's header different.... (!).

Try the code below.  It should at least get you started:

 

Sub FormatAllButFirstPageHeader()
    Dim sec As Section
    For Each sec In ActiveDocument.Sections
        If sec.Index <> 1 Then     'FOR ALL SECTIONS OTHER THAN SECTION 1....
            With sec.Headers(wdHeaderFooterFirstPage).Range
                With .Font
                    .Name = "Arial"
                    .Size = 10
                    .Bold = False
                    .Italic = False
                End With
            End With
        'End If                             'XXXXXXXXXXXXXXXXXXXXXXXXXXXXX
            With sec.Headers(wdHeaderFooterPrimary).Range
                With .Font
                    .Name = "Arial"
                    .Size = 10
                    .Bold = False
                    .Italic = False
'                    .Underline = wdUnderlineNone
'                    .UnderlineColor = wdColorAutomatic
'                    .StrikeThrough = False
'                    .DoubleStrikeThrough = False
'                    .Outline = False
'                    .Emboss = False
'                    .Shadow = False
'                    .Hidden = False
'                    .SmallCaps = False
'                    .AllCaps = False
'                    .Color = wdColorAutomatic
'                    .Engrave = False
'                    .Superscript = False
'                    .Subscript = False
'                    .Spacing = 0
'                    .Scaling = 100
'                    .Position = 0
'                    .Kerning = 0
'                    .Animation = wdAnimationNone
                End With
            End With
        End If                                 'XXXXXXXXXXXXXXXXXXXXXXXXXXXXX
    Next sec
End Sub

 

Note the X's next to the two "End If"s.  Experiment by uncommenting the top one and commenting out the bottom one.  Post back if you need help understanding the difference - it's to do with .  I included, but commented out for speed, all the other settings that can be set while formatting the text.

Hope that helps.

Cheers
Rich

=AVERAGEIFS(A2:F2,A$1:F$1,"Hits",A2:F2,">0")

This formula will only take the average of numbers appearing in cells which have a heading as Hits.  So just ensure that the columns that you wish to ignore should not have Hits as the column heading.

amit tandon say:

Another option.

Enter below formula as an array formula (Ctrl+Shift+Enter) in cell G2 & copy down:

=AVERAGE(IF(MOD(COLUMN(A2:F2),2)=1,IF(A2:F2>0,A2:F2)))

This formula calculates average of all odd numbered columns. If your columns have a pattern. say odd numbered or even or every third column you may use the above.

In case you have non-congruous columns without a set pattern and also without common headings, you may use below - Enter below formula as an array formula (Ctrl+Shift+Enter) in cell H2 & copy down:

=AVERAGE(IF(COLUMN(A2:F2)={1;3;5},IF(A2:F2>0,A2:F2)))

This formula chooses column numbers 1, 2 & 3. You can update this to any column number(s) you wish to calculate average of.
