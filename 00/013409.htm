jeff schoner - msft say:

MyPhone will only sync the calendar on the phone with the calendar on the MyPhone web portal.

However, there are some other options here that may get you what you want:

1. You can sync your phone to Outlook when the phone is tethered to the computer (via USB, for example) through Windows Mobile Device Center (on Vista and newer) or ActiveSync (on older versions of Windows). Once the Outlook calendar is on your phone, it will be backed up to the MyPhone service assuming that you have the proper box checked in the list of items to sync.

2. If you get your mail through an Exchange server, you can configure both Outlook and your phone to synchronize with your Exchange server directly. On the phone side, you would do this in Outlook Mobile. However, if you're syncing to Exchange through EAS (Exchange ActiveSync, aka AirSync) on the phone, MyPhone will no longer backup and restore your contacts, calendar or email.

Extending Suzanne's suggestion, "IF" your example 1 through D text is five independent paragraphs then you could select it and run:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim lngIndex As Long
Dim oRng As Range
Set oRng = Selection.Range
  For lngIndex = 1 To oRng.Paragraphs.Count
     oRng.Paragraphs(lngIndex).Range.Characters.First.Delete

    If lngIndex = 1 Then
      oRng.Paragraphs(lngIndex).Style = "List Number"
    Else
      oRng.Paragraphs(lngIndex).Style = "List Number 2"
    End If
  Next
End Sub

Delete a section break (next page) - Microsoft Community

hollygardiner say:

Delete a section break (next page)

I have inserted a next page section break that I no longer need but cannot find a way to remove it. Does anyone know how to remove it?
suzanne s. barnhill say:

The easy answer, of course, is just to select it and press Delete. (If you have trouble seeing the break, switch to Draft view.) There are complications only if the section following the break is formatted differently from the section preceding it. In that case, you'll need to do a little preparation:

  1. If there is a different header or footer in the last section, access the header/footer pane and link the header/footer to the previous section (say yes when asked if you want to replace the current header/footer).
  2. With the insertion point in the next-to-last section, go to the Page Setup dialog (the easiest way to access it is to double-click at the top of the horizontal ruler). Without making any changes in the settings, click OK.
  3. Then go to the section following the break and press F4. This will make the Page Setup for that section the same as the previous one.
  4. Provided there is no difference in the number of columns, you can then safely delete the break.

There's more information on working with sections at https://wordmvp.com/FAQs/Formatting/WorkWithSections.htm
