I have now discovered that I can correct this using the GP editor (available in W7 Pro and above):

  1. Type gpedit.msc into the start search box and press Enter.
  2. Navigate in the left-hand pane to
    Computer Configuration > Administrative Templates > Windows Components > Internet Explorer
    and click on Internet Explorer.
  3. In the right-hand pane, double-click on Customize user agent string
    Select Enabled 
    In the Enter IE Version String box, type MSIE 10.0
  4. Click Apply and OK, then close down the GP editor.
  5. Re-start Internet Explorer and visit http://winware.org/en/ to check that the change has worked.

It's now reporting

Mozilla/5.0 (compatible; MSIE 10.0; Windows NT 6.1; WOW64; Trident/6.0)
