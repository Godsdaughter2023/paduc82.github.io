reiver william say:

Windows 7/10- no 5.1 Dolby or DTS test tone- my solution

This thread has been in discussion for years and I found a surprising solution.

I recently updated my home theater PC with new mobo/processor etc. My old board used a coaxial spdif to run to my receiver in another room. The new board only had a optical spdif so I bought  one of those $10 optical-to-coaxial adapters online. Seemed to work perfectly....the test tone in Windows worked in all the sampling frequencies. What did not work was the Windows Dolby and DTS test tones. It cycled through the speakers but no sound emerged. So my received played digital audio...just not in Dolby or DTS.

- After puzzling over this for weeks I noticed in the small print for the digital adapter ad that it could only pass 2-channel digital, not 5.1 or 7.1. In fact none of the cheap adapters I saw could do more than 2 channel.  So I bought a 35 foot optical cable online and ran it to the receiver. DD and DTS worked instantly.

a. user say:

Hello,

Thank you for posting your query in Microsoft Community.

Did you make any significant hardware or software changes to the computer prior to this issue?

I can imagine the inconvenience that you are experiencing connecting your devices via Bluetooth and will certainly help you in the right direction to fix the issue.

This error occurs if the service is started by the Local System account instead of by the Local Service account (NT AUTHORITY\LocalService). The Local System account may have insufficient permissions to start the service.

So, I would suggest you to follow the below provided steps and check if you are able to start the service and connect the Bluetooth devices.

  1. Press Windows key + R, type services.msc in the box, and then press ENTER.
  2. Locate and double click the Bluetooth support service.
  3. Click the Log On tab.
  4. In the This account text box, type “NT AUTHORITY\LocalService”.
  5. Set both Password fields blank.
  6. Click Apply and then OK.
  7. Restart the service and check.

As of now, try the above steps and check if the issue is reproduced.

Do let us know if you need any further assistance, we'll be glad to assist you.
