Hello!

NAT has to do with internal IP addressing on your network.  Most of the time a strict NAT type is caused from having more than one networking device doing NAT.  A common example would be you have a modem/router from your ISP that is doing NAT with your own router plugged into their device, also doing NAT. This results in double NAT and a strict NAT type on Xbox.  The best way to resolve this is to either remove your router from the equation or contact your ISP to assist with turning it off on their router.

arminatorx say:

You already have a Port forward rule for one console on port 3074. 

If that one console has an open NAT, go to the network settings of the second console. 

Set it to a static IP, and then in the network settings, set the second console to an alternative port. 

The second console HAS TO have a different Xbox Live port than 3074. 

After you have chosen a different Live port from the selection, go back to your router settings, and set up port forwarding to the second console with the alternative port you chose. 

Now with port 3074 forwarding to the first console (make sure the first console also uses a static IP, and that the console IP matches the forwarding rules you made above), and your custom port forwarding to the second console, both consoles should be able to get an open NAT.
