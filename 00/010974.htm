When I print labels how do I change fonts and font size. - Microsoft Community

gary & mary davies say:

When I print labels how do I change fonts and font size.

Labels & Fonts
When I print labels how do I change fonts and font size.  Also is there a way to change the left margine on a single label?
jay freedman say:
If you want to use the Print button in the Labels dialog to send directly to the printer, you can change the font by selecting the text in the Address box, right-clicking it, and choosing Font from the context menu. You get the standard Font dialog to choose font, size, color, etc.

If you're printing a whole page of labels and you need to change the left indent (not the left margin -- that's for the whole page) for just one of them, click the New Document button instead of the Print button. Go to the label of interest, select the content of the label, right-click it, and choose Paragraph. In the dialog, set a value for the left indent. It will affect only the selected lines.
