a. user say:

You are probably first seeing a message like this:

Windows cannot open this program because it has been prevented by a software restriction policy.  For more information, open Event Viewer or contact your system administrator.

Then when you look in the Event Viewer Application log, you find the message you posted - probably something like this:

Event Type:    Warning
Event Source:    Software Restriction Policies
Event Category:    None
Event ID:    866
Description:
Access to C:\Program Files\AVG\AVG PC TuneUp\OneClickStarter.exe has been restricted by your Administrator by location with policy rule {0134258e-52bc-4731-95c6-46633eed37d0} placed on path C:\Program Files\AVG

Right?

That message is telling you that somehow a Group Policy change has been made on your system so that the execution of OneClickStarter.exe has been disallowed (or blacklisted).

If you did not make the change yourself, somebody or something made the change for you - most likely malicious software. 

The malicious software does not want you to be able to run the AVG program because it does not want to be found and removed.  The malicious software would be happiest to fool you into thinking that you need to do something drastic (and unnecessary) like a Repair Install or a complete reinstall of your XP.  Neither are necessary.
