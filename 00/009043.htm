bob jones aka: cybertaz say:

Arrow keys are not reserved in any way.

On most condensed keyboards the secondary mapping of the Arrow Keys is modified by the fn key to these operations:

fn+Up Arrow = Page Up

fn+Down Arrow = Page Down

fn+Right Arrow = End of Line

fn+Left Arrow = Start of Line

The specific System Preference marked in your screen shot has absolutely nothing to do with key assignments. As described, it simply determines how the Tab Key operates in windows & dialogs. You might have a look in Mac Help on the subjects pertaining to Creating keyboard shortcuts for applications...

However, Word has its own mechanism for assigning keystrokes & in this case that's what needs to be used. The OS X method can only execute application commands explicitly listed in the application program's Menu structure, which the navigation key operations are not. In order to use the OS X method you'd first need to add the commands to menus in Word -- basically a redundancy.

bob jones aka: cybertaz say:

You can assign keystrokes to replicate the operation of the Arrow Keys, but you can't use the ones you mentioned... The fn key is not a universal modifier key. It has no impact on the operation of alphanumeric keys, punctuation keys or most other keys. IOW, fn+j is no different than j by itself.

The commands you need to assign keyboard shortcuts to are:

  • LineDown
  • LineUp
  • CharRight
  • CharLeft

If not already familiar with how to do so see Word 2011 Help on the topic: Create or delete a keyboard shortcut

[Note that you do not need to delete the existing Arrow assignments... Any command can have multiple keystroke combinations assigned to it.]

Change default minimum row height

You can do this with Macros.

1. Press Alt+F11

2. Click on the sheet on the left to which you want to make this adjustment (Say Sheet4)

3. On the right (editor window), at the top you see General written and a drop down.

4. Select the drop down to "Worksheet". In the next drop down, select "Change".

5. Paste the code below in the editor window.

Private Sub Worksheet_Change(ByVal Target As Range)
Set Target = ActiveCell
Cells.RowHeight = 15
End Sub

6. Save the file

There you go!
