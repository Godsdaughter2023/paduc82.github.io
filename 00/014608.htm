jay freedman say:

If the user has not used the "Default personal templates location" setting in File > Options > Save to move the Templates folder to a non-default folder, the .thmx file should be copied to %appdata%\Microsoft\Templates\Document Themes. Also, since you made a custom color scheme, you need to copy the corresponding .xml file from your folder %appdata%\Microsoft\Templates\Document Themes\Theme Colors to the same location on the other user's PC.

[Note: Windows creates the environment variable %appdata% to hold the path to the hidden folder C:\Users\<login name>\AppData\Roaming. The path will automatically expand when you type or paste %appdata% in the address bar of File Explorer and press Enter.]
