Random number generator with out duplicates (Draft Lottery) - Microsoft Community

travisoswald say:

Random number generator with out duplicates (Draft Lottery)

I want to create a random number generator for a draft.  I have 14 teams listed in order from A1 to A14 in column A.  In cell B1 I want to randomly select a number from 1-14.  In cell B2 I want to randomly select a number between 1-14 not including the number selected in B1.  In cell B3 I want to select a number between 1-14 not including the previous numbers selected in cells B1 and B2. And so one till I select the final remaining number.  Can someone help me with a formula?

alfranco17 say:

The easiest way would be this:

 

* Use RAND() function in column A.

* Enter numbers, 1 through 14, in column B.

* Sort both columns by the value in A.

* Delete the values in column A.

 

 

alfranco17 say:

Here's your macro:

 

Option Explicit

Function SpecialRandBetween(p_low As Integer, p_high As Integer, p_used_range As Range) As Integer
    Dim v_random As Integer, v_cell As Range, v_repeated As Boolean, v_tries As Integer
    Do
        v_random = Int(p_low + Rnd() * (p_high - p_low + 1))
        v_repeated = False
        For Each v_cell In p_used_range
            If v_cell = v_random Then
                v_repeated = True
                Exit For
            End If
        Next v_cell
        v_tries = v_tries + 1
        If v_tries = 1000 Then
            SpecialRandBetween = 1 / 0
            Exit Function
        End If
    Loop While v_repeated
    SpecialRandBetween = v_random
End Function

 

 

To use it, copy this formula from B2 downwards.

 

=SpecialRandBetween(1,14,$B$1:B1)

No, if the character style has the point size explicitly set to 10 points, that value would override the settings of the paragraph style.

On the other hand, if the point size has been left unchanged in a character style (meaning that the value displays blank in the Modify Style dialog box for the character style), the font of the underlying paragraph style applies. bernie deitrick say:

Sorry - I threw in an extra .End(xlUp) when I shouldn't have - worked for the one time I tried it.

Use this instead:

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim rngC As Range
    Dim lngR As Long
    Dim shtS As Worksheet
    
    Set shtS = Worksheets("Time Storage")
    
    If Intersect(Target, Range("B4:B12")) Is Nothing Then Exit Sub
    
    lngR = shtS.Cells(shtS.Rows.Count, "A").End(xlUp).Row
        
    If shtS.Cells(lngR, "A").Value <> Date Then
        lngR = lngR + 1
        shtS.Cells(lngR, "A").Value = Date
    End If
    
    For Each rngC In Intersect(Target, Range("B4:B12"))
        If rngC.Value <> "" Then
           shtS.Cells(lngR, rngC.Row - 2).Value = Target.Value
       End If
    Next rngC


End Sub
