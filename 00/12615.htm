How to insert a document property - Microsoft Community

nâmio evangelista say:

How to insert a document property

how can I insert a document property (like Author, Summary) into my document? Word for windows we have the quick parts, and what about word 2011 for mac?

a. user say:

I'd go along with Jim's approach, but it's probably worth mentioning a few differences between what you get on Windows Word (2007 and 2010) and Mac Word, particularly if you are a newer user of Word who hasn't actually noticed some of the older features.

1. Content controls v. Field codes

In Windows Word, Insert->Quick Parts, you get two submenus relating directly to "Properties" (and any Building Block might also contain property fields or controls).
 a. the Insert->Quick Parts->Field... option is very similar to the Insert->field... option in Mac Word 2011.
 b. the Insert->Quick Parts->Document Property option appears to insert the same kind of thing as the Field... option, but has no equivalent in Mac Word 2011.

(a) inserts a field code such as { AUTHOR }. The result of that field code reflects the value of the Author document property. Editing the text of the Author field itself does not change the value of the Property. If you have more than one { AUTHOR } field in the document, you may need to update all the Author fields to see any updates you make to the Author Property. 

(b) inserts a Content Control that is "connected to" the same Author property in the document but the link is two-way. Changing the value of the Author in that control changes the value of the Document Property, and other Author content control values will also update.

Although you cannot insert these content control fields in Mac Word 2011, if you create them in Windows Word and copy the document over to Mac Word, the controls do work in a limited way.

2. The list of "Document Property" content controls contains different types of "Property", some of which cannot be inserted using field codes. e.g. 
 a. the Author content control corresponds to a thing that both Windows and Mac Word (and older versions on both platforms going back to around Word 6, I think) recognise as a standard "Author" document property. But
 b. the Publication Date property is new in Word 2007, is not available as a field code on Mac Word, and is defined in a different way from "Author". It also behaves in a different way if you copy one from Win to Mac Word - on Win Word, this is a "date" content control with a date picker. On Mac, it looks much more like plain text, although in fact if you have a date such as Wednesday, 7 December 2011 and change the "7" to an *, Word will in fact change the day to Thursday even on Mac.
 c. If you are using templates/documents with additional new-style "Custom" properties defined on a SharePoint server, these properties also appear in the Document Property dropdown in Win Word but are not easily available on Mac Word (if at all).

(both Mac and Win Word also let you define old-style "Custom" Properties (now called "Advanced Properties" on Win Word) whose values can be inserted, but not updated, using DOCPROPERTY fields. And there are other similar things.)

The only way I know to natively hide the status bar would be to use Full Screen mode. You've mentioned that a Ctrl+<something> retrieves it but unless you have a custom macro assigned to a Ctrl+<something> keystroke combination you should not have this command in Excel 2010. Of course, someone else may have put a custom macro into your workbook.

Tap Alt+F8 and see if there are any macros listed in the Macros dialog. If there is and one looks like it is named to describe hiding and showing the status bar, select it from the list and click the Options button. The resulting dialog will tell you what keystroke combination is currently assigned to the macro.

If the Macros dialog is blank and you want to create a macro to show/hide the status bar, type

Show_Hide_Status_Bar

... in the Macro name: text box and click Create. When the VBE opens replace the empty Show_Hide_Status_Bar sub code with the following,

Sub Show_Hide_Status_Bar()
    Application.DisplayStatusBar = Not Application.DisplayStatusBar
End Sub

... and tap Alt+Q to return to your worksheet.

Tap Alt+F8 again to show the Macros dialog and select your Show_Hide_Status_Bar macro from the list. Click the Options button and assign a keystroke combination by holding down the Ctrl key or Ctrl+Shift keys while tapping a letter key. I typically use Q or W because they normally are not assigned to anything else. Click OK to close the Options dialog and OK again to close the macros dialog.

Test you new macro by tapping the keystroke combination you assigned. Tap it again to reverse the previous show/hide operation.

 Power Saving tips: Again... 😉
Here are some tips that may help you save battery power. In general, you should turn off functions when you do not need them.
- Turn Bluetooth off when not in use
- Turn off WLAN scanning (from Standby Screen)
- Reduce the brightness of the screen (Settings -> Personalization -> Display
- Turn off Navi "Breathing" for the light ring on the scroll key
- If your network operator does not offer 3G, turn off 3G usage
- Close down unused applications that run in the background, like Music, Browser, Maps, Fring, etc.
- Avoid using carry cases with magnets as this may cause the device to wake up
- If you are fairly low in battery level and want to extend the usage time with the remaining battery power, use the power savings mode (Power key--> Single press --> Power savings mode). It will reduce display brightness to the minimum level, and change the backlight time out to 5 seconds.
- Ensure that Maps application the Camera application (if location tagging is on) are not active when not needed
- Do not leave the USB cable connected as this causes power to be consumed even if no data is being transferred. Use the CA-126 USB cable for USB connections (it charges at the same time)
- DO NOT set Bluetooth visibility to "Define Period" this seems to drain the battery faster
- For GPS Navigation, set the backlight setting to "Optimized" rather than "Always on". Go to Maps-> Options -> Tools -> Settings -> Navigation -> Backlight set to "Optimized".
- Use "Headphones" rather than "Lineout" when playing music
