User Account Control permissionIf you are prompted for an administrator password or for confirmation, type your password, or click Continue.

7. Log off the system.

8.Log on to the system again.

After you log on to the system, the profile folder is re-created.

Hi Bob,

Welcome to Microsoft Community.

Sorry to know that the Bluetooth audio devices fails to work after on Windows 10. I will certainly help you.

This issue might occur if the audio drivers are corrupted or due to incorrect audio settings.

I suggest you to follow the troubleshooting methods listed below and check if it resolves the issue.

Method 1: Run the Playing Audio Troubleshooter

Windows 10 has a built-in troubleshooter to check and fix issues with playing sound. Here's how:

1. Open Control Panel by right click on Start Button.

2. In the search box, type troubleshooter, and then click Troubleshooting.

3. Click on the view all option on the left panel.

4. Run the Playing Audio Troubleshooter

Method 2: Restart the Bluetooth Support Service

Bluetooth Support Service will enables the detection of the bluetooth devices. Restarting this service can resolve these type of issues. Here's how:

1. Right click on Start, select Run and type services.msc, press Enter.

2. The Services window opens, search for Bluetooth Support Service.

3. Double click on the service, click on General tab, under Startup type, select Automatic.

4. Then click on Start > Apply >OK to save the changes.

5. Check if the issue is resolved, if not restart the system.

Hope this helps. If the issue remains unresolved, please get back to us and we would be happy to help.

I had the same issue with these headphones and found with a casual search that many others do, too.

To fix the problem:

1. Select Start > Settings > Devices > Bluetooth

2. Select the headphones in the list of devices, then click "Unpair".

3. (Optional) Turn Bluetooth to "Off", wait 10 seconds, then turn Bluetooth to "On".

4. Hold the power button on the Hesh 2 headphones until it is off, then hold it in for about 10 seconds until you hear it say "Pairing".

5. The headphones should again appear in the list of devices.  Select them and click "Pair".

6. The settings on the headphones should be back to Stereo.
