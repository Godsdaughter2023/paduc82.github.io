Hi Scott,

Normally, this stuck syncing issue may be caused by some files are stuck syncing from the sync server.

These stuck synced files may be the limit characters or the normal ones. If they are the sync limit characters, just rename them and re-sync. If they are normal ones, we suggest moving them out from the cloud and wait for a while, then remove them in again to sync.

So, for your scenario, could you try the following suggestions?

1. Check the sync limit characters from the KB below to see if there are some files names or files types are not supported to sync.

https://support.microsoft.com/en-us/kb/3125202.

2. Check if there are files with a red overlay in the synced folder. If so, remove them both from the cloud and local synced folder, then add them in to sync again. Note: please backup them before deleting them.

3. Check if there are some blank folders on the local synced folder even though there is a green overlay on them. If so, find out the files that are contained in them from the cloud, then remove them from the cloud and re-add them again to trigger sync.

4. Unlink and relink your account, and choose folder to sync to check the outcome.

Additionally, if possible, I 'd suggest you test on another Mac to check if the issue will occur.
