How Many To, Cc and Bcc Recipients Windows Live Hotmail Allows Per Message
You can add up to 100 (one hundred) recipients per message in Windows Live Hotmail.

It only takes a few days of normal activity to increase the limits—that gives us enough time to detect automated spam machines and accounts that have been hacked. 
We’re constantly working to make things worse for spammers and better for real people.

The send mail limit depends on your reputation within Hotmail. 
The system needs a few days to build your reputation.

If you are an established user, recent account activity may have sent us a false alert.

tim liu msft say:

Hi Anthony,

I'd like to explain that it's not feasible to directly sync all files/folders in Shared with me section of OneDrive for Business library. However, if we click and access the folder in Shared with me section, we are able to sync files in the folder with the new OneDrive sync client. It's not very convenience as we need to sync those folders in Share with me section one by one. For this part, please feel free to submit the feedback via OneDrive for Business Uservoice. If you want to sync files in a team site which is related to Office 365 group, you could directly access its document library and use the new sync client to sync files in it.

About some users mentioned to use personal OneDrive in old threads, that is because of consumer OneDrive has a feature which can add shared items to user's own OneDrive.
