How do you change the field name in a table - Microsoft Community

tim milstead say:

How do you change the field name in a table

I want to change the name of a field in a table?

dirk goldgar say:

If you want to actually change the field name, you can open the table in design view and change the field name there.  Having done so, if you have the Name Autocorrect option turned on, Access will change the field name everywhere it occurs (that Access can find -- you will have to change it if you mention it in VBA code).  If you don't have Name Autocorrect turned on, you will have to find all the places -- queries, forms, reports, macros, and modules -- where the field name is stated, and change it there.

If you don't actually want to change the field name, but just change how it appears in forms and reports, you can set its Caption property in the table design, and change the captions of any existing form/report labels that have the old name.

Also, for any given query, you can alias the name so that, for that query and queries that derive from it, the field can be referred to by the alias.

For those of you who, like me, don't know what a "non-breaking space" is a quick Google search turned up the following:

"When Word calculates line width and wraps text to the next line, it tries to break the line at either a space or a hyphen. Sometimes, however, you may not want Word to break a line at a certain space. For instance, you may want to make sure that two adjacent words appear on the same line as each other. The answer is to use non-breaking spaces instead of regular spaces when you don't want Word to break a line at a certain space. To do this, hold down the Ctrl and Shift keys as you press the Spacebar. Word will then not break the line at that point."
