1. Probably won't help, but won't hurt.

   Internet Options> General tab> Browsing History> Delete> uncheck "Preserve Favorites website data" then delete Temporary Internet Files. No reason, yet, to delete cookies, history,  passwords, etc.

2. This problem with the Back button and Google searches pops-up occasionally, but doesn't affect all users all the time.

Here are some workarounds:

  1. Use the down arrow next to Back to bypass any intermediate redirect or Google tracking pages and return to the original search.

  2. Google.com> Settings> Search settings> choose to have search results open in a new window or tab. If you ever delete cookies you will have to set this again.

  3. Open the search result in a new tab via right-click or Ctrl+click or middle (scroll wheel) click so the original search page is left there for you to return to
