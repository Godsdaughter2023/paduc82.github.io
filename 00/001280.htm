shereens. say:

there are many many many different workarounds that others have posted online; but i finally figured out how to poperly create a header on the first page ONLY. it is super simple, but not intuitive.

these instructions are for OFFICE 2010 word document, but may work for 2007 and other microsoft office applications.

1. click on INSERT at the top of the word document page.

2. click on HEADER

3. do NOT insert any header template or text--repeat: do NOT insert any header text or template; instead...

4. click on EDIT HEADER

5. then check the DIFFERENT FIRST PAGE box

6. now that you have formatted the document with instructions to have a header only on the first page, you can go back and insert whatever header template you wish--------> do this by clicking on the HEADER button that is now on the far left side of the ribbon at the top of the page

7. alternatively, you can manually type in your header text

8. after you are finished with the header, you can double click the body of the page (below the header) to type your text; after typing more than one page, you will notice that your header remained only on the first page

the trick is to format the document BEFORE inserting any header template or text. if one inserts header data BEFORE clicking on DIFFERENT FIRST PAGE, then--for some inexplicable reason--the header disappears from the first page and shows up on the 2nd page (at least that's what happens with me). so, the best thing to do is to choose DIFFERENT FIRST PAGE before inserting any header information.

hope that helps. best wishes.
