roady [mvp] say:

For organizing received messages via a message rule, there is most definitely a "Move to folder" action.

For organizing sent messages via a rule, there is indeed only a "Move a copy" action. his is because you also have the generic option to "Save sent items" option enabled. Disable that option and you'd only have a single copy.

Make sure you also create a rule to catch the remaining messages that you want to be saved in the Sent Items folder.

For more details see: http://www.howto-outlook.com/howto/sortoutgoingmail.htm

Note that when you are sending via the SMTP server of Gmail, Gmail always creates a copy in your Sent Items as well. They don't offer an option to disable that.

lz. say:

Hi Ralph

Assuming the following data:

YES within 2 days

=SUMPRODUCT( ($B$2:$B$6="Yes") * ( (($C$2:$C$6)-($A$2:$A$6))>=0 ) * ( (($C$2:$C$6)-($A$2:$A$6))<=2 ))

YES within 4 days and above

=SUMPRODUCT( ($B$2:$B$6="Yes") * ( (($C$2:$C$6)-($A$2:$A$6))>=0 ) * ( (($C$2:$C$6)-($A$2:$A$6))>=4 ))

NO within 5 days

=SUMPRODUCT( ($B$2:$B$6="No") * ( (($C$2:$C$6)-($A$2:$A$6))>=0 ) * ( (($C$2:$C$6)-($A$2:$A$6))<=5 ))

NO within 10 days

=SUMPRODUCT( ($B$2:$B$6="No") * ( (($C$2:$C$6)-($A$2:$A$6))>=0 ) * ( (($C$2:$C$6)-($A$2:$A$6))<=10 ))

NO above 10 days

=SUMPRODUCT( ($B$2:$B$6="No") * ( (($C$2:$C$6)-($A$2:$A$6))>=0 ) * ( (($C$2:$C$6)-($A$2:$A$6))>10 ))
