UPGRADE WINDOWS 10 AND FORMAT WINDOWS 7

im having windows 7 on my system and i have registered for windows 10 upgrade. Now i want to format my system to delete all the unwanted softwares and viruses. i want to ask that post that procedure would i be able to upgrade to windows 10 ?

cmdrkeene say:

Yes you sure can, but you will need to still have Windows 7 first.  The Windows 10 upgrade is only an upgrade, you can't "clean install it" the first time.  So your next steps would be like this:

  1. Format your drive
  2. Reinstall Windows 7 and get it fully updated/patched
  3. Upgrade to Windows 10

After you do the in-place upgrade from 7 to 10, your machine's hardware ID will be known to be authorized for Windows 10, and then you can clean install anytime in the future after wiping the drive, no "Windows 7 first" required.

frederik long say:
I have a Lenovo Thinkpad , running windows 7. I was using the internet in bed with my computer on my lap when I accidentally opened the disc drive.
This sounds mysterious. Disk drives come in a metal case that is screwed shut with around six screws. They are installed inside a laptop case and protected by a cover that is also screwed on. How can you "accidentally" open this disk case?

To find out if the disk is intact, do this:
  1.  Set your BIOS to use the DVD drive as the primary boot device.
  2.  Boot with a Windows Repair CD. 
  3.  Press a key when the message indicating "Press any key to boot from CD or DVD …" appears.
  4.  Select a language, a time and currency, and a keyboard or input method, and then click Next. You are now in Repair Mode.
  5.  Select an Operating System, then click Next.
  6.  Select the Command Prompt.
  7.  Identify the correct drive letter for your Windows installation. It could be a drive other than C: but it won't be X:. You do it by typing the commands
    dir C:\Win* {Enter}
    dir D:\Win* {Enter}
    dir E:\Win* {Enter}
  8. When you see lots of files, e.g. on drive D:, type this command:
    chkdsk   D:  /R {Enter}
