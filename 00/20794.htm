stefan blom say:
If you create an envelope and add it to a document, you can then modify the appearance of the envelope.
 
Word places the envelope in a separate section, so you could add a watermark if you first unlinked the header of the following section (watermarks are always anchored to a header paragraph): Place the insertion point in the first header following the envelope section, and click to deselect Link to Previous on the Header & Footer Tools Design tab. Then move the insertion point to the envelope section and add the desired watermark.

deepestblue---01 say:

You can very well try the soft reset by typing *#7370# at the stand-by screen before going for the hard-reset (which should be resorted to only if the phone does not start properly) mentioned in the post above yours. It requires a good deal of dexterity and many attempts to be able to do the 3-finger deep reset.The soft reset will give you the same results as the other one, if your phone is working properly.

 

The so called "solution" is not really a solution. The reset procedures simply clear your phone memory, so that if at all there is any corruption of system files it will be fixed. The "solution" is a generic one: it's commonly seen posted by many people- sometimes for reasons good enough and sometimes just for the sake of suggesting something.

 

It is always a good idea to backup your phone memory using PC Suite. This ensures that you can transfer your data to a different compatible phone and also, prevent loss of data in the event of an irreparable damage, theft, etc.
