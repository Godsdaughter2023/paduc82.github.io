If the document is protected by means of the Restrict Editing feature, it is, contrary to the response by Roberto Ven, NOT possible to access the header of the document via the user interface, without first removing the protection from the document.

It would however be possible to have an autonew or autoopen macro that displayed either an input box (OK for a single item of data) or a userform into which the user could enter the data for the header, which would then be used to set the value of a variable in the document so that it would then be displayed in the header in a Docvariable field, or Docvariable fields, that would need to have been inserted into the header before the protection was applied to the document.

Roberto,

If the document is protected by using of the Restrict Editing feature, it will NOT be possible for the user to access the header to enter information in it!

jllatham say:

I'll plug in your values, work the bugs out and probably post the revised code as an edit to this posting.  Probably p2 came up zero  since I see some entries with no "h" entry in them.

Might double check things.  When I set up the test data and applied the conditional formatting against the rows (with entries placed starting in A1), the highlighted entries were at A3, A6, A9, A10, and A11.  And some other checks verified that.  The code below provides the correct results based on those pre-tests.  Just delete the old code and replace it with this code and don't forget to change that sheet name as before.

Just a little note: on my system it took between 2 and 3 seconds to process 10584 rows made up of copies of the test data.  Removed 4411 rows.

Sub DeleteCF_Rows()
  Const testColumn = "A"
  Dim testWS As Worksheet
  Dim testRange As Range
  Dim anyTestCell As Range
  Dim testValue As String
  Dim LC As Long ' loop counter
  Dim p1 As Integer
  Dim p2 As Integer
  Dim dValue As Variant
  Dim hValue As Variant
 
  Set testWS = Worksheets("Sheet1") ' change as required

  'setup assumes labels in row 1
  Set testRange = testWS.Range(testColumn & "1:" & _
   testWS.Range(testColumn & Rows.Count).End(xlUp).Address)
  'when deleting things, best & easiest to work from bottom up
  'or from right to left
  Application.ScreenUpdating = False ' for performance improvement
  For LC = testRange.Cells.Count To 1 Step -1
    If testRange.Cells(LC).FormatConditions.Count > 0 Then
      'the cell has conditional formatting applied
      '
      'make it easy to reference the cell
      Set anyTestCell = testRange.Cells(LC)
      'get the string to evaluate
      ' remove leading/trailing white space and make sure
      ' any alpha characters are lowercase for testing
      testValue = LCase(Trim(anyTestCell))
      p1 = InStr(testValue, "d")
      p2 = InStr(testValue, "h")
      'quick test and possibly early out
      If p1 = 0 And p2 = 0 Then
        'definitely delete
        anyTestCell.EntireRow.Delete
      Else
        'either p1 <> 0 Or p2 <> 0
        'get 'd' value if p1>0
        'assumed to be leftmost entry
        dValue = 0
        If p1 > 0 Then
          dValue = Val(testValue)
        End If
        If dValue > 0 Then
          'do nothing, keep any with non-zero 'd' value
        Else
          'dValue = 0, but 'h' value must be considered
          'get 'h' value if p2>0
          hValue = 0
          If p2 > 0 Then
            'dig out the 'h' value
            'it's either leftmost (if no 'd') value or
            'get from mid() if p1 and p2 both >0
            If p1 = 0 Then
              hValue = Val(testValue)
            Else
              hValue = Mid(testValue, p1 + 1, p2 - p1)
            End If
            If hValue < 8 Then
              'the 'h' value is less than 8, delete the row
              anyTestCell.EntireRow.Delete
            End If
          Else
            'this part probably never gets used, but
            'it's here just in case.
            'there is no 'h' value, so treat as less than 8
            'delete the row
            anyTestCell.EntireRow.Delete
          End If ' end p2>0 test
        End If ' end dValue > 0 test
      End If ' end of p1=0 and p2=0 test
    End If ' end of test for cell has conditional format
  Next
  'good housekeeping/cleanup
  Set testRange = Nothing
  Set testWS = Nothing
End Sub
