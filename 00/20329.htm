  1. Select the word or words that you want to include as a table of contents entry.
  2. Press ALT+SHIFT+O.

    The Mark Table of Contents Entry dialog box appears.
  3. In the Entry box, modify the text if you want it to be different from the selected text.
  4. In the Table identifier list, click C for table of contents.
  5. In the Level list, select a level for the table of contents entry.

  7. Click Close.
  8. Repeat steps 1 through 7 for each table of contents entry that you want to create.

a. user say:

If you use the "Add text" feature in the Reference tab, you are using a different feature from the one described in the message that you have quoted. 

The Mark Table of Contents Entry approach that is described in the message creates a { TC } field.

The Add text feature applies a style to the currently selected paragraph. If you choose "Do not show in Table of Contents", Word will apply a style that (should) remove that paragraph from the ToC. If you choose one of the "Level" options, Word will apply a style that (should) add that paragraph to the ToC. 

In either case, you will only see the entries
 a. if the ToC has been set up to include that type of entry. 
 b. when you update the ToC

The ToC can gather entries from multiple types of source, e.g. TC fields as well as paragraphs with certain styles or outline levels.

To include { TC } fields, in the Table of Contents dialog box, Table of Contents tab, you need to click Options..., then check "Table Entry fields". A complication is that each TC entry can have an identifier ("C", in the example) The ToC can gather TC entries for all identifiers, or one specified identifier. If someone has already inserted a ToC in your document that specifies an identifier other than "C", your TC entries will not appear. I don't think you can specify the identifiers in the Table of Contents dialog box - you have to use alt-F9 to reveal the ToC field code. When you do that, you will see something like

{ TOC \o "1-3" \h \z \u }

You can include all TC fields as well by changing that to 

{ TOC \o "1-3" \f \h \z \u }

You can include just the "C" TC fields using

{ TOC \o "1-3" \f C \h \z \u }

toby ornott say:

When you originally established the signature, you most likely clicked the caret marked v which opened the More and then you clicked the Align Right.  So to return it to the left you probably need to do the following:

  1. First Copy your existing signature to some place safe where you can retrieve it without loosing it.
  2. Once you have completed step-1, then highlight the signature again and Cut it,**
  3. Now refresh your window so that you have a new signature graphic to work with.
  4. Clear-away anything unwanted and begin a new signature by pasting your original back into the window from the safe location where you stored it.

This should return your signature to Align Left; however there is a possibility that it will be indented slightly due to the Outlook default.

______________________

**Note: There is a very good chance that you will need to click the button marked Discard and then Save because the graphic-text editor works counter-productively to one's desires since there is no edit button.  Consequently it is even probably best to exit the program before proceeding to step-3, and then to return with a fresh, new attempt.  Then proceed with step-3.
