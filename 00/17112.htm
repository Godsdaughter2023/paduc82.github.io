  3. Click the cogwheel icon on the upper right corner of the page, then select More mail settings...
  4. Click Rules for sorting new messages under Customizing Outlook.
  5. See to it that there are no rules pertaining the email moving to your Junk folder. (Note: You might as well delete all your rules then recreate them again).

grschinon say:

There is no such thing as a virus for the 5800. If your phone is switching off spuriously then the problem lies elsewhere. The first thing to try is restoring factory default settings.

 

Dial *#7780# and supply the security code when prompted. The default on Nokia devices is 12345, so that's what it'll be unless you've changed it.

 

If that didn't work you should try a soft reset, but start by backing up the contents of your phone's memory to the memory card. Dial *#7370# and then supply the security code again.

 

Finally, if that doesn't work you'll want to think in terms of re-installing the firmware in the phone using Nokia Software Updater or OVI Suite.

 

If nothing helps then it could well be a hardware problem, in which case you should take the phone to your nearest Nokia Care Point.
