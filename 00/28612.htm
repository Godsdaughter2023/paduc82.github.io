pete_uk say:

Assuming your addresses are in column A, you can do something like this in a helper column:

=SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(A1," St "," St. ")," Ln "," Lane ")," PO Box "," P.O. Box ")

Then you can copy this down.

Hopefully you can see how they are nested - add other SUBSTITUTE functions as necessary.
