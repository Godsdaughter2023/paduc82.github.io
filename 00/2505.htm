Thank you for posting to the forum regarding the issue you are having playing your music files via the Xbox Music application.  I have seen this error a couple times in the past and for a couple different reasons.  I have a few things for you to try to get this resolved.

The first thing I have seen resolve this is to uninstall the reinstall the Xbox Music application from your computer.  Before doing the uninstall I would suggest you clear your computers temporary folder.  You can do this by pressing and holding the Windows key + R on your keyboard and then typing %temp% in the search box.  Highlight all the files and folders here, right click and select delete.  If any fail to delete, skip those.  Once the temporary files are deleted it is time to delete the application from the computer.  Once it is deleted, reboot your computer and then install the application again from the store.  This process will not effect your collection.

If you notice the same issue after the application is reinstalled the next thing to do is reset your default libraries.  This error can sometimes indicate that the application is not able to locate the proper path to where your music files are located.  You can do this by going to you desktop and then clicking the file icon to enter File Explorer.  Here you should see the word libraries along the left side of File Explorer.  If you do, right click this and select "Restore default libraries".  If you do not see the word libraries, right click the white space along the left panel and click on "show libraries" then do the steps suggested above.  After you complete this process, please close the Xbox Music application and then reopen the application and try accessing your content one more time.

If you do have this music stored on an external storage device, please be sure to have this location added to the Music app via the steps in this article, http://support.xbox.com/en-US/xbox-on-other-devices/windows/how-to-add-music .  Once you have added the location, please remove all other locations listed here.  Close the Music app. and reopen and try accessing your content again.

john w. vinson say:

Your tables appear to be set up exactly correctly. The preferred way to enter data into this structure is with a Form and a Subform.DON'T use table datasheets (with or without combo boxes or subdatasheets) for data entry; they are very limited in functionality and can be more confusing than helpful!

Create a Form based on one of the "one" side tables, whichever one makes sense; e.g. if you are focusing on a Member and want to assign Employees to that member, use the Members table as the source for the main form. On that form put a Subform based on the junction table, using the SSN as the master/child link field. On the subform put a Combo Box based on the Employees table, storing the EmpID but displaying the employee name (or whatever is meaningful to the user). Reverse the roles of the tables if you want to instead focus on employees.
