The wireless syncing that can be done between a windows phone and Zune on the computer is done when both devices are connected to the same wifi network - a Bluetooth connection won't work.

However, since syncing media files can take some time to complete and could potentially deplete the phone's battery, the wireless syncing is only performed when the phone is plugged into a power source and the battery is charged to 50% or more. 

So at the end of the day, plug your phone into AC power and while the phone charges up, your media gets synced and any pictures you snapped are backed up to your computer.

If you put in Subtotal(9,$E2:$E5)  in E6   and everywhere else you have a total, then at the bottom you can do

=Subtotal(9,E2:E168)  and it will compute the grand total.  Subtotal ignores values produced by subtotal formulas in the range it looks at so it is adding up all the raw numbers.

The 9 as the first argument tells subtotal  to sum the next argument (the range of values).   Subtotal can perform various math operations based on what the first argument is (if you are not familiar with subtotal).

if you currently have formulas like  =Sum(E2:E5)  in the sum cells and not in the raw data cells, then you can select the column E and do a find/replace

Hi,

A codec compresses or decompresses media files such as songs or videos.

Since the automatic download option of codecs has been removed in Windows Media Player version 12, you can search and download the required codecs from the Internet using your preferred browser.

Note: Similar to the AVI and ASF format, WAV is only a file container. Audio content that is compressed with a wide variety of codecs and that is stored in a .wav file can be played back in Windows Media Player if the appropriate codecs are installed on the computer. The most common audio codecs that are used in .wav files include Microsoft Adaptive Differential Pulse Code Modulation (MS ADPCM) and uncompressed Pulse Code Modulation (PCM).
