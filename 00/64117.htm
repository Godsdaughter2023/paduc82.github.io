Check for malware:

Download, install, run, update and perform full system scans with the following two applications:

  • MalwareBytes (FREE)
  • SuperAntiSpyware (FREE)


Removing everything they find.  Rebooting when needed.  (You can uninstall one or both when done.)

Then perform an online scan with the eSet Online Scanner.

 

The less you have running all the time, the better the things you want to run will perform:

Use Autoruns to figure out what all is starting up when your computer does/when you log on.  Look up anything you do not know about using Google (and/or ask here.)  You can hopefully figure out if there are things starting when you computer does (or you logon) that you do not need and then configure them (via their own built-in mechanisms is the preferred method) so they do not start up - using your resources without reason.

You can download and utilize Process Explorer to see exactly what is taking up your processor/CPU time and memory.  This can help you recognize applications you might want to look into alternatives for and/or get rid of all together.

 

Free up some hard drive space and get it organized (as well as check for errors):

Open an elevated command prompt by clicking on the start orb and browsing through All Programs --> Accessories and right-clicking on "Command Prompt" and choosing "Run as Administrator".

Type in the following in the command prompt:

dism /online /cleanup-image /spsuperseded

... and press ENTER.

The process takes a few minutes to complete, it ends with the sentences “Service Pack Cleanup operation completed. The operation completed successfully”.

Use Disk Cleanup to eliminate more temporary files safely.  May want to schedule this regularly.

You can use an application that scans your system for log files and temporary files and use that to get rid of those:

Ccleaner (Free!)
http://www.ccleaner.com/
( just the disk cleanup - don't play with the registry part for now )

Other ways to free up space..

SequoiaView
http://www.win.tue.nl/sequoiaview/

JDiskReport
http://www.jgoodies.com/freeware/jdiskreport/index.html

Those can help you visually discover where all the space is being used.  Then you can determine what to do and what to delete/archive.

After that - you will want to check for any physical errors and arrange everything for more efficient access...

CHKDSK

¡firedog say:

I don't remember exactly what happened right before they disappeared.


First, thanks for giving those details - it makes troubleshooting so much easier.

Even though you write that you have version 15.4.3538.0513, I'd like you to check again. This is because an update has been pushed over the past couple of weeks, and some users have only noticed because something went wrong with the program. Type   wlmail   into the start search box. When you see wlmail.exe in the results pane, right-click on it and select Properties. You'll see the product version on the Details tab.

If it really is the version you said, I can't say what's happened. However, the problem with not being able to restore contacts from your backup may be (I'm guessing) that the contacts database - the folder called DBStore - is marked as a protected operating system file. Windows Backup may not copy system files, only data files - I don't know. However, copies of system files are maintained by System Restore. So, if you can find a restore point dating from immediately before you noticed that your contacts were missing, you should be able to use that to restore them. Do this while not connected to the Internet to foil the automatic Windows Live updater. Any other changes to your system since the restore point will have to be reapplied.

If you manage successfully to recover your contacts database, I would recommend the following steps:

(a) back up the default contacts list by exporting to VCF:

  • Export your WLM contacts as individual *.vcf files to a new user-created folder on your desktop.
  • Open Windows Contacts: type   contacts   into the start search box and press Enter.
  • Import the .vcf files into Windows Contacts. You can select multiple files in the import wizard. This will create an individual *.contact file for each contact

Then close down WLMail.
(b) perform a full Windows Update.
(c) update WLMail to version 15.4.3555.0308 using the link on this page: Windows Live Essentials or if you prefer select the appropriate full installer on this page: How do I install Windows Live Essentials while offline? 
(d) check whether your contacts list is still intact. If it isn't, import Address book for current Windows user.
(e) carry out any other system changes that might have been reversed by System Restore - reinstalling programs, for example.

This will with any luck bypass the problems many users have had with the automatic update.

Noel
