a. user say:

Hi

Welcome to Microsoft Community.

 

I can understand that you are receiving a continuous message on the Action center icon as “Maintenance is in progress”.

 

Before proceeding further, I would like to know;

 

Do you see red X or exclamation next to the Action center flag?

 

Action Center lists important messages about security and maintenance settings that need your attention. Red items in Action Center are labeled Important, and indicate significant issues that should be addressed soon, such as an outdated antivirus program that needs updating. Yellow items are suggested tasks that you should consider addressing, like recommended maintenance tasks.

  

The “Maintenance in Progress” message will be displayed when the system is undergoing Automatic maintenance.

 

I would suggest you to follow the steps below to fix this issue:

 

Method 1. Run the System maintenance troubleshooter:

  1. Press Windows and X key and click on Control panel.
  2. Type Troubleshooting on search option and click on Troubleshooting.
  3. Click View all on the left pane of the screen.
  4. In the list of options, click on System maintenance.
  5. Click Next and follow the on-screen instructions.

 

If the issue still persists, follow the next method;

 

Method 2. Steps To Turn Off the Maintenance Messages in Windows 8

 

Step 1. Press Windows and X key and click on the Control Panel.

Step 2. Click on the System and Security.

Step 3. Open the Action Center under the System and Security then click on the Action Center.

Step 4. Click on the option 'Change Action Center Settings'

 

You can see the Maintenance Messages such as Windows Backup, Windows Troubleshooting, Automatic Maintenance, Drive Status, Storage Space, File History, HomeGroup and others.

 

 

In this you can see the list of options which you want to turn off the Maintenance Messages. If you want to disable the Maintenance Messages just uncheck the messages you want to disable and click OK.

 

Sometimes even if there is no action to be taken, Action Center flag does flash. Left click on Action Center flag on the taskbar and click “Open Action Center” and see if there are any actions it requests.

 

I would suggest you to hide the Action Center notification.

 

To do that just click the icon in the notification area and then drag it toward the desktop. To view hidden icons click the arrow next to the notification area. If the arrow isn't there, you don't have any hidden icons.

 

Please try rebooting the computer to see if the message is still there.

 

Also try to follow the help article below:

Automatic Maintenance

http://msdn.microsoft.com/en-us/library/windows/desktop/hh848037%28v=vs.85%29.aspx

 

Hope it can resolve the issue. Please reply us with the status of the issue.

Windows 10 has two big problems with respect to System Restore Points:

  • The feature is automatically turned off during the upgrade from Windows 7 or 8, even if it was fully active before.
  • It is also turned off during subsequent upgrades.

This greatly diminishes its values. On my machines I now run a script that monitors the facility on a daily basis.

About your Start button problem: Two weeks ago I had it on my machine too. I performed a thorough Google search and tried the various methods I found but none would work. Ultimately I solved it using, you guessed it, System Restore.

The problem is most likely caused by an inappropriate entry in the registry. It might be possible to restore a backup version of this file. Please have a look at the date stamp of the System file. Does it precede the date when you first noticed the problem? You need to do this quickly because Windows will soon overwrite this file.

C:\Windows\System32\config\Regback\System
