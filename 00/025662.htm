devlin sco say:

Hello Andres,

It seems that the fonts you mentioned, HGP Gothic M and HG Gothic M are custom fonts and doesn't ship with Excel 2016. If you still have access to the PCs that have the fonts, you can simply copy it and install it on the PC you're going to use.

  1. Select the Start button, type Fonts and then press Enter.
  2. Right-click the font you need.
  3. Select Copy and then Paste the font to a folder. Alternatively, you can drag and drop the font to a different folder to copy it.
  4. Transfer the font file to the PC you're going to use.
  5. Install the font by right-clicking the file and then select Install. If prompted, click Yes or enter the administrator password.
  6. Restart Excel 2016.

Let us know if you have other questions.
