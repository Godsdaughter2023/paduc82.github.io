Need Help Formatting Two-Page Letterhead in Word - Microsoft Community

andreabeth say:

Need Help Formatting Two-Page Letterhead in Word

I am working with my company's two-page pre-printed letterhead.  Page 1 and Page 2 have different margins, for which I've accommodated in my page setups. 

I am trying to create a Word document, keeping the different margins on Page 1 and Page 2, that will allow typed text to go directly from Page 1 to Page 2 when the end of Page 1 is reached. Right now, it is inserting an additional page that has the same margins as page 1. Can anyone help with this?
jay freedman say:
Go to http://wordfaqs.mvps.org/Letterhead.htm and read the section on "More complex letterhead". As it says, "The standard method of accomplishing this is a borderless text box or frame in the margin, anchored to the header paragraph."

The reason behind this is that margins are properties of an entire section, meaning that to change the margins you must insert a section break. But text can't flow across a section break, so changing the actual margins is ruled out. Instead, you have to use the "text wrapping" feature of a text box or frame to fake it, effectively making the margin on one of the pages appear larger than it is.

Since each section can have a First Page header in addition to the "primary" header, you can anchor the text box in one header or the other, depending on which page needs the larger margin. The Letterhead article, and the Headers article to which it links, explain what to do.

charles kenyon say:
You need borderless empty textboxes in the header or footer of the page that will need the larger margin. These should be set to wrap the text to the right (or left).

If you need more than the comprehensive article you've already been referred to, download the sample at Letterhead Textboxes and Styles Tutorial that shows how to do this. It contains a word document that is set up to have what appear to be different margins.

You do NOT want to be changing the page margins to do this unless you absolutely have to do so. Doing this will insert a section break and  will interrupt your document flow.

If the textbox is on the left side of the page, it will make automatic numbering on that page more difficult.

gp78 say:

Moving all footnotes between two documents.

Hi

I am finishing up my PhD and want to use two different style conventions for my footnotes one for publication the other for submitting. So I have two documents that are exactly the same except the footnote style (note by style I don't mean the style in word rather the academic style).

As I will be doing corrections to the thesis I don't want to have to do them twice so I would rather just do them in one document and then copy and paste the other footnote style into the corrected document.

Is this possible?  thanks
daniel g. grau say:
No, that cannot be done. The only way to replace a footnote's content is to overtype it.

Pasting footnotes from one document into another will at first seem to work, but the new footnotes will become part of an existing footnote, which can only be eliminated by deleting the reference in the main text flow.

Going the other way around (pasting text over the second document's text) will bring along the associated footnotes with it.

I would wait until one document is finalized, then make a copy and enter the changes to the footnotes. It makes no sense to keep an "almost final copy" if you are still making changes.

If you have all of the files in a folder, you could use a macro with the following code to open each document in turn and then print it out:

 

Dim fd As FileDialog

Dim strFolder As String

Dim strFile As String

Dim aDoc As Document

Set fd = Application.FileDialog(msoFileDialogFolderPicker)

With fd

    .Title = "Select the folder that contains the files that you want to convert to Portable Document Format."

    If .Show = -1 Then

        strFolder = .SelectedItems(1) & "\"

    Else

        MsgBox "You did not select a folder."

        Exit Sub

    End If

End With

strFile = Dir$(strFolder & "*.doc*")

While strFile <> ""

    Set aDoc = Documents.Open(strFolder & strFile)

    aDoc.PrintOut Background:=False

    aDoc.Close wdDoNotSaveChanges

    strFile = Dir$()

Wend
