Hi,

 

ALT+f11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in. It will select all unlocked cells on the active sheet.

 

If you want to delete the contents automatically then the last line could be r.clearcontents

 

 

 

 

Sub SelectUnlockedCells()
Dim c As Range, r As Range
For Each c In ActiveSheet.UsedRange
    If c.Locked = False Then
        If r Is Nothing Then
            Set r = c
        Else
            Set r = Union(r, c)
        End If
    End If
Next
If r Is Nothing Then Exit Sub
r.Select
End Sub
