marshall barton say:

To reduce the font size to make it "fit" in the text box is tricky.  Subject to the conditions below, you can use VBA code like:

Private Sub Detail_Format(Cancel As Integer, FormatCount As Integer)
Const FUDGE As Integer = 20
Dim fs As Integer

    With Me.[thetextbox]
        Me.FontName = .FontName
        Me.FontBold = .FontBold
       
        For fs = 11 To 5 Step -1
            Me.FontSize = fs
            If Me.TextWidth(.Value) < .Width - FUDGE Then Exit For
        Next fs
        .FontSize = fs
    End With
End Sub

Change "thetextbox" to the real name of your text box.

You'll have to fiddle with the fudge factor because it depends on the text box's border style/width, bold/italic and font properties.  If it's too small you may be missing the last character in the text.

Change the 11 and 5 in the For statement to the largest and smallest font sizes you want to use.

And, this will only work if the text in the text box is a single line (i.e. no wrapping to a second line).
