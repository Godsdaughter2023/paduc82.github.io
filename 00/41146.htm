shekhar s say:
Hi,
 
 
It appears that the file association on your computer is corrupted and the default program to open files is set to Adobe Reader.
 
Step 1.
To begin with troubleshooting, you can uninstall Adobe Reader and check the file associations are set to default.
To uninstall Adobe Reader, you can follow the below steps:
1. Open Programs and Features by clicking the Start button, clicking Control Panel, clicking Programs, and then clicking Programs and Features.
2. Select a Adobe Reader, and then click Uninstall.
 
Step 2.
Once you uninstall Adobe from your computer, follow the below link that will automatically fix the file associations‘When you run an .exe file on a Windows Vista-based or Windows 7-based computer, the file may start a different program’:
http://support.microsoft.com/kb/950505
 
Now re-install the Adobe Reader and check if the issue persists.
 
If you are able to fix the file associations, you should run a Malware scan on your computer to check if this is caused by a virus infection.
You can follow the below link and run a Malware scan on your computer:
http://onecare.live.com/site/en-my/default.htm?mkt=en-my 
 
 
Hope this helps.


Thanks and Regards:
