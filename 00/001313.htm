bob jones aka: cybertaz say:
There is no setting or Preference that controls it, but this seems to work:
  1. In Word, open the Normal.dotm template
  2. Insert a Figure of some sort
  3. Create a Caption using the Below... option
  4. Save the template
  5. Delete the Figure & the Caption (Don't use Undo)
  6. Close & Save the template again
  7. Quit Word [Note: When you Quit, Word may crash... That's OK so don't bother sending the Report]

From then on you should find that Below... is selected as the default position.

Caller ID is a feature set by your mobile operator. If your mobile operator supports and has enabled this feature, you will see the person's phone number when they call you.

You can turn on your Wp8's speech functions this way:

Other Speech commands you can turn on

In your phone's settings, you can turn on additional Speech commands and functions, including announcing caller ID for incoming calls and texts. You can even tell your phone to read a text message you receive, then dictate and send a reply.
