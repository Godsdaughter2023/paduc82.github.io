The code below can be used to transfer the filter from an existing filtered list to another similar list.

Sub FilterSecondTableSame()
'
'
y = Worksheets("Sheet1").ListObjects("Table1").AutoFilter.Filters(1).Criteria1

Worksheets("Sheet4").ListObjects("Table3").Range.AutoFilter Field:=1, Criteria1:= _
Array(y), Operator:=xlFilterValues
End Sub
