allenfei_msft say:

Hi lyncer2013,

After further searching, I've found the following information:

PROBLEM
After you delete an app password that's used for Azure Multi-Factor Authentication, the app password appears to continue to work. 
CAUSE
This problem occurs because the token that's acquired after a user successfully signs in by using an app password continues to work until the token expires. The token works only on devices on which the user successfully signed in. 
SOLUTION
Wait for the token to expire. This may take from 8 to 24 hours, depending on the service that the user is accessing. This practice follows the same guidelines for when passwords are changed or when users are deleted.

One important thing to become familiar with is the terminology as it relates to the types of licenses that are available

Office 365 is a subscription based product which includes all Office components and is associated with a yearly cost. Office 365 Personal is the same except for one user on multiple devices

Office 2019 is a one-time cost perpetual license for one user/one machine. It does provide any option to "upgrade" to a newer version of the Office suite

cbarnhorst say:

th2 = "Threshold 2"

rs_1 = "Redstone 1"

Threshold is the codename of the current version of Windows 10.  Threshold 1 released to the public on July 29th.  Threshold 2 releases this week and concludes the Threshold phase of Windows 10 develpment.

Redstone is the codename of the next phase of Windows 10 development.  There are no insider builds available for Redstone yet but there will be insider rs_1 builds soon.  The rs_1 you saw is probably the placeholder for the coming rs_1 builds.  rs_1 will end with a public release and then rs_2 will begin.  The final rs_x build is projected for July 2016 and then a new branch will begin (no hints yet about a codename).  This is how Windows-as-a-service works.  No more waiting 18 months for a service pack to get new features.
