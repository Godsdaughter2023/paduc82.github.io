How can I install a 64-bit version of Windows 7? - Microsoft Community

matthew forse say:

How can I install a 64-bit version of Windows 7?

how do i install a 64 version of windows 07 - what settings to change to get it install, i have originally a 64 bit version win 07 originally but iI could not get it to work
original title: Windows 64 bit?
andre da costa say:
Depends on what you have installed on your computer now.

If its a 32 bit version of Windows, you will have to do a clean installation, there is no in place upgrade path from Windows 32 Bit to 64 bit.

Using the Windows 7 64 bit disk, install the Windows 7 Windows Easy Transfer Utility from the DVD (To find it, just navigate to x:\support\migwiz, 'x:' represents the name of your disk drive. In this folder, you will see a collection of files, the important one you need to look for and double click is 'migsetup'.) , run it and create a WET file (Windows Easy Transfer file), if you have an external hard disk, you can have it stored there so you can access it later when you do a clean install of Windows 7 64 bit disk.

Once you have done that, simply do a clean install of Windows 7 64 bit disk. In Windows 7 64 bit disk, launch Windows Easy Transfer, follow the on screen wizard, point to where you stored the the WET file (external hard disk), and it will restore your personal files and settings. Reinstall any necessary applications.

32 vs 64 bit
http://windows.microsoft.com/en-US/windows7/32-bit-and-64-bit-Windows-frequently-asked-questions

Is my PC running the 32-bit or 64-bit version of Windows?
      http://windows.microsoft.com/en-us/windows7/find-out-32-or-64-bit

http://windows.microsoft.com/en-US/windows7/Installing-and-reinstalling-Windows-7

Windows 7 64 bit comes in handy when you need to address at least 4 GBs or more of RAM. Windows 7 32 bit can utilize up to 3.2 GBs of RAM. Because the memory address space is much larger for 64 bit Windows, that means, you need twice as much memory than 32 bit Windows to accomplish some of the same task, but you are able to do so much more, you can have more applications open, do things like run an Antivirus scan in the background without it affecting your system performance. Windows 7 64 bit is more secure too, malicious code cannot easily infiltrate it, drivers are more reliable since they must be signed before they can work with 64 bit Windows 7.

As for compatibility, you will need 64 bit device drivers for any hardware devices you might have. Also, there is no 16 bit subsystem in Windows 7 64 bit, which means, your applications must be 32 bit only, not 16 bit installer or uninstallers. Also, if you decide to move to Windows 7 64 bit in the future, there is no upgrade path from 32 bit Windows, clean install only.

    Outside temperature icon in System Tray - Microsoft Community

     em_1336  say:
    

                Outside temperature icon in System Tray
    
                    
    Microsoft,
    

    

    Please consider putting a System Tray icon in Windows to display the outside temperature of a person's location on Earth (if Windows can detect their current location), and maybe one additional location.  Nothing more, nothing less - just the temperature,
 in the System Tray.  To change from Celsius to Fahrenheit, or to add a secondary location, the user would right-click on the System Tray icon and see the options for "Celsius", "Fahrenheit", and "Add secondary location using city name or GPS coordinates".
    

    

    There are many third-party apps for this purpose, but most of them don't work properly, are bundled with adware or spyware, contain excessive religiosity, display too much detail, look cluttered, are gargantuan, or rely on technology that is reaching End-Of-Life. 
 For example, YoWindow is more than 16 MB and relies on Adobe Flash, which Adobe is killing in 2020.
    

    

    I realize that, in Windows 10, a user can pin the Weather app to the Taskbar, but that's not what I'm looking for.
    

    

    Why would someone want this feature?  Well I can't speak for other people, but for me, I like to know, when I wake up and turn on my computer, is it so cold outside that I need to run outside and plug in my engine block heater?  Is the temperature hovering
 near the freezing mark which will make for a harrowing drive to work and require that I leave the house earlier than usual?  Do I need a hoodie or a thick jacket?  Is the temperature rapidly descending, or not climbing?  Is the temperature near record cold
 (as it was here a few days ago, -5 F)?  Yes, a person could walk to their front door and step outside, but being able to know the outside temperature ahead of time, while sipping a cup of tea in your pajamas, is both useful and pleasant.
    

    

    The closest that I could find to what I'm advocating is a tiny portable application called WeatherMate.  The wm4 zip file is only 673 KB.  The program displays the temperature, in C or F, in the System Tray, and can display more weather details, in metric
 or US measurements, when you double-click on the System Tray icon.  Unfortunately, the temperature and degree symbol are often partially cut off unless you shrink the Windows text size, but in doing so, text in all other programs, Windows Explorer, and the
 Desktop is more difficult to read.  The program's map feature doesn't work (for me at least), and as of today, the author hasn't updated the program since Jan 2017.
    

    

    Another possible example of what I'm suggesting was a contribution 18 years ago on CodeProject entitled "Display the Current Temperature in the System Tray, Douglas Earl, 12 Nov 2004".  Mr. Earl's source code was 62 KB.  To run his program, the user would
 input the zip code and the zip code's outside temperature would display in the System Tray icon.  Of course, when you try to run his program now, the website no longer exists (where the program tries to pull the temperature from), but if he could do this 18
 years ago with only 62 KB of code, I would think Microsoft could easily implement the same functionality in Windows, no?
    

    

    Thank you.
