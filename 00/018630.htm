    Unify Control Panels and Settings - Microsoft Community

    ngmvista say:

    Unify Control Panels and Settings

    With each new release of Windows Microsoft has made attempts to make the UI / UX for configuring one's computer more intuitive.  There are a few issues with this that have limited success:

    • Many settings are left out from the new UI / UX layouts, perhaps the assumption was made that many settings are used by advanced users only.
    • Settings left out from the new UI / UX layouts are increasingly hidden in the form of legacy control panels (for instance certain wired network and adapter settings are now buried in the Windows 2000 / XP style control panel)
    • The UI / UX of the legacy control panel and that of new settings do not match or compliment each other well.

    Windows 10 could be more intuitive by striving for a single dash board to replace all control panels and settings from previous versions of Windows.  Two objectives should be:

    • Simple, clean and intuitive
    • Contains all settings (no retention of hidden control panels from legacy Windows)

    You cannot add a command directly to the main [  ] menu bar. When in the Customize... dialog the Worksheet Menu Bar appears directly below the main menu bar. (If is isn't displayed go to the Toolbars & Menus page of the dialog to put a check in the box.) You can then drag commands to the preferred menu pad (such as Edit) & when the menu pad opens, continue dragging to the preferred spot in the existing list.
