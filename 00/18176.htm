  4. Under Forward your mail to another email account, uncheck the Keep a copy of forwarded messages in your Outlook inbox check box.

Me![subformcontrol].Form.[controlname],Visible = False

where subformcontrol is the Name property of the Subform Control - the box on the mainform containing the subform; its name may or may not be the same as the name of the Form object within it.

bob jones aka: cybertaz say:

Just to clarify, "Word 365" (Office 365) is not a program -- it's a marketing concept :-) It provides Word 2011 [Office 2011] locally installed on your Mac [or Office 2013 on a PC] plus access to the Office web apps along with other services (such as OneDrive). The web apps support the same primary file types as do the full versions of the programs but they do not provide all of the same features as the locally installed versions.

Depending on your version of TextEdit you should be able to open the file, then use File> Save As & pick Word 2007 Format (.docx) from the File Format: list before you click Save.

Alternatively: Assuming that the file is saved in TextEdit's default format (.rtf), just open the file using Word, then save as a Word document. [Word fully understands the RTF file type as well as most of the others available in TextEdit.]

As Rich suggested, in either case it never hurts to make a backup copy first :-)
