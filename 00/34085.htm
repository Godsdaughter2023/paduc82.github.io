Hi Debra,

What's the hard disk capacity and how many partitions are there? johnanderson6 say:

This may or may not be helpful. I was finally able to get our failed updates installed on our affected devices. One thing that needed to be done was to decrypt the drives. The main thing that was causing the update failures was an outdated controller driver on the HP 745G3 laptops. I noticed this by chance. 

Each time the update was failing (at 33%) it would briefly blue screen an error and then revert to the previous update. The error was only displayed for a few seconds before it disappeared, not long enough to read. I was finally able to catch a picture of it with my phone and did some searching and found that I needed to update a controller driver for this particular model laptop. (By the way, do not buy HP 745 laptops, they are extremely problematic)

Here is the fix that worked for the 745G3 laptop.

  1. Open Device Manager (via Control Panel or by typing Device Manager into Windows Search.)
  2. Click System Devices
  3. Click AMD12C Controller Driver
  4. Click the Driver Tab
  5. Click Update Driver (Search Automatically for Updated Driver Software) - Make sure you have this version..

    • Windows 10 (64-bit)

    • 1.2.0.60
  6. Go into Windows Settings (click on Windows start button in the task bar and then click on the gear icon)
  7. Click on Update & Security
  8. Click to Check for Updates (This should find and download the Fall Creator's Update 1709)
    Note: Please make sure you download a fresh install of the update and not continue with a previously installed update. 
    Note: If you do not see an option to Check for Updates, you can download the update from https://www.microsoft.com/en-us/software-download/windows10
  9. Reboot after update installs
    At this point the device will go through different stages of the update process and will take some time. You may see your device reboot a few times during this process. 

Once your device starts back up, log on and check to make sure the update was successful

  1. Windows Search, type winver and hit enter. This will populate a window with the information. 

Note: Since I found this fix, Windows has pushed out a series of smaller updates. These updates need to be taken separately from the Fall Creator's Update. The next big updates is only a few weeks away so I hope that one goes smoother.. Good Luck!
