Hi,

You can't put an autofilter on empty cell so you need to find the last use column in row 2 and use that for the range like this.

Sub ddd()
Dim Lastcol As Long

With Worksheets("Supplemental Distribution")
Lastcol = .Cells(2, Columns.Count).End(xlToLeft).Column
   If Not .AutoFilterMode Then
   .Cells(2, 1).Resize(, Lastcol).AutoFilter
   End If
   .EnableAutoFilter = True
   .Protect Password:="xxxxxxxxx", _
   Contents:=True, UserInterfaceOnly:=True
 End With
End Sub

Reasons why a phone cannot update:

1. It is a country variant - country variants often get the updates later, or have updates skipped altogether. A Turkish country variant product code for example, skipped v40 and updated from v31 straight to v50.

2. It is network branded - phones acquired from network providers or these providers' stores SIM free are branded, and will not get updates until the networks approve and modify the updates to their liking.

3. Special editions - These special editions are chartered editions and may never get updates unless taken to a Nokia Care center.

sw dude say:

Well that should have worked if you UNchecked the check box.  Perhaps it would need to toggle.

I can understand not wanting users to figure out how to set their settings.  You can put this into a macro in Excel like the macro I made in the original post.  Here is an update to change the Word setting for field codes.  The macro will update all the links in the specified word document and make the filename the displayed text for the link.  I think you will have to modify it for your requirements but it should be a good head start.

‘You need to add “"Microsoft Word 12 Object Library" from the Excel VBA editor's “tools->References…” menu for this macro to work
Public Sub UpdateLinks()

    Dim myLink As Word.Hyperlink
    Dim myDoc As Word.Document
    Dim wordApp As Word.Application
    Dim arrFileFolders() As String ' This is an array of split elements
   
    Const SPLIT_CHAR As String = "\" ' CHANGE THIS TO "/" for web address
    Const DOCUMENT_FILE As String = "C:\SomeFile.docx"
   
    'Open the document to update links in
    Set wordApp = New Word.Application
    Set myDoc = wordApp.Documents.Open(DOCUMENT_FILE)
   
    'Loop through all links in the document
    For Each myLink In myDoc.Hyperlinks
   
        ' Split the address into elements
        arrFileFolders = Split(myLink.Address, SPLIT_CHAR)
       
        ' Update the display text
        myLink.TextToDisplay = arrFileFolders(UBound(arrFileFolders))
       
        ' Do not show the field codes
        wordApp.ActiveWindow.View.ShowFieldCodes = True ' Make sure we toggle
        wordApp.ActiveWindow.View.ShowFieldCodes = False
               
    Next myLink
   
EXIT_SUB:

    myDoc.Save
    myDoc.Close
    Set myDoc = Nothing
    Set wordApp = Nothing
   
End Sub
