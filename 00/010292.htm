How to convert MDE Database file to MDF ? - Microsoft Community

surya p dhakal say:

How to convert MDE Database file to MDF ?

How to convert .MDE Database file to .MDF ?

tom van stiphout say:
Most likely not possible, but what is MDF?

gina whipp say:
.MDF = SQL Server primary file

scottgem say:

How to convert .MDE Database file to .MDF ?

First, you need to understand that an Access database file is a container. It contains several Objects including tables, forms, reports etc. SQL Server is a database engine. So the only thing you can do is migrate the data to SQL Server. 

So assuming you can link to the tables in the MDE, you may be able to use SSMA to migrate the data. 

scott mcdaniel say:

Note that if you use SSMA (SQL Server Migration Assistant), your Access database would NOT be linked back to those new SQL tables. You'd have to somehow run a relink routing in that .mde file to link to the new SQL tables. Also, there are often changes which need to be made to an Access application to use a SQL backend.

So if your goal is to migrate your Access tables to SQL Server, then  you should think long and hard before doing this.

However, if you simply want to move your data to SQL Server to do other things with it, then you'd be okay.
