brian tillman [outlook mvp 2007-2019] say:

When a POP client wishes to check if there is new mail on the server, it asks the server for any new messages since the last time it checked.  It is up to the server to present this list to the client.  Google mail has a setting that limits the messages that it presents as new.  Log into your Google mail account using a web browser, click the Settings icon at the top right (the icon containing the gear) and choose Settings from the drop-down.  Click the Forwarding and POP/IMAP tab.  In Section 1 of the option list, select the radio button labeled "Enable POP for all mail (even mail that's already been downloaded)".  Otherwise, Google Mail won't show you all of the messages in the Inbox.

Alternatively, you could swich to using an IMAP account in Outlook.  That's what I use.

srinivasu guntimadugu say:

Check for file association for Word application. Follow the steps below.

1.    Click on Start> Default Programs

2.    Click on ‘Associate a file type or protocol with a program’

3.    Under the name select *.docx and click on change programs on top right corner.

4.    Click on browse in ‘Open with’ window and browse to the below location

         C:\Program file\Microsoft Office\Office 14\WINWORD.exe and click ok.

 

Note: Go to Program File (x86), if you are working with Windows 7 64bit.

 

Now check with hyperlink and see if it opens with Word.

 

Check the link for more information about File association.

http://windows.microsoft.com/en-US/windows-vista/Change-which-programs-Windows-uses-by-default
