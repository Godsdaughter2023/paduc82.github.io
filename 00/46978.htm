lemp say:

It's possibly the chipset driver.  See http://support.toshiba.com/support/driversResults?freeText=1200010656

If you've already installed the chipset driver, double-click on the unknown PCI device in Device Manager and click the "Details" tab.  Use the drop-down box to select "Hardware Ids."

You'll see a string like this:  VEN_5333&DEV_8901&SUBSYS_00000000&REV_16

Look up the VEN and DEV numbers (5333 and 8901 in this example) here: http://pcidatabase.com/

Once you know what it is, find the correct driver (likely on the Toshiba support page linked above) and install it.
