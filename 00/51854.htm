This is not really a Windows question (in a Windows forum) but a hardware question. Anyway, the fan in your power supply might have a worn-out bearing, causing the fan to seize. Here is how you can find out if it still spins the way it should.
  1. Cut a piece of paper about 3" long and 1/2" wide.
  2. Fold it so that it is still 3" long but only 1/4" wide.
  3. Turn on the PC.
  4. Stick the folded piece of paper through a slot at the back of the PC into the power supply. This is supposed to make a nice whirring sound when the fan blades chew away at the paper. If there is no sound or if it is irregular then the fan does not do its job.
  5. Open the computer case and do the same test to any other fans you might have.
  6. Check for dust accumulation on the heat sinks.

a. user say:

Hello Thomas,

Thank you for posting your concern in Microsoft Community and welcome to the Windows 10 Family.

Before we start with troubleshooting steps, let us know the make and model of the computer you are using

ACPI is the Advanced Configuration and Power Interface. The ACPI_BIOS_ERROR occurs when computer gets into a no boot loop due to BIOS incompatibility or power management.

As an initial recommendation, I suggest you to consider the below important points and check if it helps to isolate the main cause of the issue.

  • Disconnect all the external peripheral devices except for and check if it allows a successful boot up.

  • Try to clear the CMOS values, i.e load optimized BIOS defaults.
    Refer to the manufacturer provided manual on how to access the BIOS settings and how to load default values.

  • If the optimized BIOS defaults does not help, it's possible there's a device driver conflicting on startup. If that's the case, boot into safe mode with networking from Windows Recovery Environment and check if the issue still persists.

Now, in case you are trying to perform a Start up repair using Windows 10 ISO as mentioned in your query, you need to make sure that you might require to change the boot priority in BIOS in order to allow the computer to boot from CD/DVD or USB.

Let us know the results.
