ken sheridan say: What you are aiming at seems more like a candidate for a report. Using your original query as its RecordSource you can group the report by order and include the order data in a group header.  Each order will show only once.  In the detail section include the calls data.  The individual calls will be listed under each order header.

You can aggregate values from the calls table per order, or just count the calls per order in the order group footer, using simple aggregation functions like Sum, Count etc.  If you want to aggregate distinct order data, however, you cannot do so in the report footer by using aggregation operators like Count or Sum as this will aggregate over the total number of calls not the distinct orders.  What you can do, however, is include a hidden control in the order header section with its RunningSum property set to 'Over All'.  You can then reference this in the report footer.  Say you simply want to count the total number of orders, add a text box txtOrderCount to the order header, with a ControlSource property of =1 and a Running Sum property of 'Over All'.  In the report footer add a visible text box which references the hidden control with a ControlSource property of =[txtOrderCount ].  This will reference the last instance of the txtOrderCount  control, and show the total number of orders.

Similarly if you wanted to sum an OrderTotal currency field, you'd add a hidden text box, txtOrderTotal, to the order header, with [OrderTotal] as its ControlSource property and a Running Sum property of 'Over All'.  In the report footer reference this control in a visible text box in the same way to show the total OrderAmount value.

Ken Sheridan, Stafford, England

This is not a bug it is a math mistaken you have made (sorry to be blunt!)

jasles say:
Sorry, I misunderstood your question. Here's how to change the default settings for a text box:

1) Open the form in design mode
2) Open the Property sheet
3) On the Controls section of the Form Design Tools on the ribbon, select the text box, but don't click anywhere else
4) You'll notice that the text under the Property Sheet title should now say "Selection type: Default Text Box"
5) Change the default Scroll Bar setting to None
6) Any changes made now will affect the default text box on this form only, to change the default settings for any new form in this database, save the form as "Normal"

If a form called "Normal" is in a database, all newly created forms will inherit its properties.
