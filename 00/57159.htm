usasma say:
Windows likes to see 15% free space (9.6 gB on your 64 gB SSD)
I've seen systems with only 10% (6.4 gB) that seem to be working fine - but they were all systems with platter based hard drives.

It's possible to get so much stuff on the hard drive that you can't delete anything.  That means that you'd have to format/wipe the drive in order to get running again (again, this is based on my experience with platter based hard drives).

wilfred mac say:

Hi Steven,

The Xbox app is one of the default apps on Windows 10. It is created to make the gaming experience a lot better. If you wish to uninstall the Xbox app, you may follow the steps below:

Uninstall Xbox app

  1. Type powershell into the Start menu or Taskbar search box, right click it and choose "Run as administrator".
  2. PowerShell will be opened as administrator.
  3. Type or copy-paste the following command: Get-AppxPackage Microsoft.XboxApp | Remove-AppxPackage
  4. Wait till the process is done and then type "exit" to close PowerShell.

In any case you want to reinstall the Xbox app. You can reinstall it from Windows Store.
