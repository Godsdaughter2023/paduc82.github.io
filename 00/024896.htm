Any way to do this?

bob jones aka: cybertaz say:

AFAIK, Excel cannot translate from one language to another.

The Language feature you're referring to pertains only to Spell Checking, so that the spelling checker knows which language the text is. IOW, you can specify any workbook as German regardless of what the actual language of the text is. When you run the spelling check the text will be checked using the German proofing tools rather than your default Language.

There may be a better way, but this one will work until someone offers something better :-)

First, create this small table anywhere in the workbook:

1

Sonntag

2

Montag

3

Dienstag

4

Mittwoch

5

Donnerstag

6

Freitag

7

Samstag
  1. Select the table, then go to Insert> Name> Define to assign a name of your choice [as long as it doesn't begin with a number or contain any spaces]. For the example, I used a range name of GermanDays.
  2. In cell A4 enter the formula: =VLOOKUP(WEEKDAY(E14),GermanDays,2,FALSE)

If you prefer to not use a range name you can just use the range reference of the table, such as $X$1:$Y$7

bob jones aka: cybertaz say:
Yes, the TEXT() function using TTTT as the format argument will work only in the German version [or a version that recognizes T as the criterion for Day]. In the English version the TEXT() function works in the same manner but the format argument would be DDDD. However, it would return the day name in English.
