After an account becomes inactive, all messages, folders, and contacts are deleted. Incoming messages will be sent back to the sender as undeliverable.
Your account name is still reserved. During this time you wont be able to re-use the same email name.

However, if the account stays inactive for an additional 90 days, the account name may be permanently deleted.
If you don't use your Windows Live ID for 365 days, your Windows Live ID will be permanently deleted. Once the account is marked as in-active all data is purged so even if you could get the email account back the data would be gone....

What should I do if email messages are missing from my Outlook.com account?

entegy say:

In Windows 8.1 checking the OS version has changed. You must explicitly state what versions of Windows you support in your application manifest, otherwise, Windows will lie to you and tell you you're on the highest version of Windows you state you support.

Apps with no manifest will see Windows Vista

Apps that have a manifest will see Windows Vista, 7, 8, 8.1, or 10, whichever is the highest your app says it supports.

Apps that ignore the new methods, yet try to check OSVERSIONEX on Windows 8.1 and up will only see Windows 8. You

This was likely done due to poor version checks that made otherwise fine applications fail to run on future version of Windows.

Developers can read more here, and should ask more questions about this on MSDN, as Microsoft Community is a consumer-focused forum.

B42:  =B1
B41:  =B42/(1+$C$2*($A42-$A41))
