The text you pasted in used line feeds instead of carriage returns for separating paragraphs. This should be evident if you tap Ctrl+Shift+8 to turn on the display of formatting marks. A line feed is noted as ↵ and full carriage returns as ¶.

A carriage return is two characters, ASCII 0×13 followed by 0×10. A line feed is a solitary 0×10.

You should be able to quickly change all line feeds to carriage returns with a Find & Replace looking for ^l and replacing with ^p.
