basith m say:

Hi Michelinski,

There can be many reasons for booting to a black screen.  The main causes observed are: problems with the driver for your display adapter; or the video output is being sent to an unexpected port; or there is a problem with the Lock Screen app.  Please try these steps first if you’re booting to a black screen:

If you see the black screen or have a flickering screen before sign-in:

  1. Try connecting to different video outputs on the PC. Sometimes a change in driver can cause the video to be sent to a different display adapter or different output (such as from HDMI to DVI, or DisplayPort to VGA).  Or try projecting the output to a different monitor by pressing the Windows logo key and the P key at the same time, and then press P again and hit Enter.  Repeat this 4 times to cycle through the screen options. 
  2. Try to boot into Safe Mode with Networking or Enable low-resolution video to fix the display adapter driver.
    • Windows may automatically boot to the Recovery Environment after 3 unsuccessful attempts in a row.  If it doesn’t, insert Windows 10 installation media and boot to the media to enter the Recovery Environment.    You can create Windows 10 installation media from another computer using the Media Creation Tool.
    • Once you are within the Recovery Environment, select Troubleshoot, and then Advanced options, then Startup Settings, and Restart.  When it restarts, you should see a number of options.  Press 5 or F5 for Safe Mode with networking (or try Low Resolution Video).
    • Open Device Manager and expand Display Adapters.
      • If you see 2 items under Display Adapters, right-click the integrated video card (e.g. Intel HD 4000 or AMD Radeon HD 4200), select Disable and then restart your computer. If that doesn’t work, return to the Display adapter, re-enable the display adapter, and then disable the other adapter.
      • If you only see 1 item under Display Adapters, rollback the driver or uninstall the display adapter in Device Manager, and enable the option to delete driver software.

    • Check Windows Update or the display adapter manufacturer’s site for an updated driver.  We are seeing updated drivers from all of the major video chipset manufacturers. So the best driver will often be only days old.

If you see the black screen only after signing in and you can see a mouse cursor:

  1. While at the black screen, try using CTRL+SHIFT+ESC to start Task Manager.
  2. If it becomes visible, try finding LockAppHost.exe on the Details tab and ending the task.   If this resolves the black screen, then the Lock Screen app likely has damaged files.  To repair the damaged files, right-click on the Start button and then click Command Prompt (Admin).  In the command prompt window, type the following command and hit Enter:

Dism /online /cleanup-image /restorehealth

When it is finished, then type the command:

Sfc /scannow

And when this is finished, re-register the Lock Screen app by typing the command:

PowerShell -ExecutionPolicy Unrestricted
Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register
"$($_.InstallLocation)\AppXManifest.xml"}

  1. If this does not work, there may be a problem with the user account.  To test this, create a new user by typing the following commands in the same Administrator CMD window (changing “<password>” to something of your choice):

net user test <password> /add
net localgroup Administrators test /add

  1. This will give you a test user to try, which will confirm if you have a corrupted user profile.

Please feel free to visit our website for any help with Windows Operating System.

The System Restore tool

Other troubleshooting methods, such as using the System File Checker tool, may not resolve the issue. 1. Open a Command Window in Administrator mode:

* click Start
* click All Programs, then Accessories
* right click on the Command Prompt option,
* on the drop down menu which appears, click on the Run as Administrator option.
* You will be asked for authorization. Click the Continue button if you are the administrator or insert the administrator password.

2. Start the System File Checker

* In the Command Prompt window, type: sfc /scannow,
* press Enter.
* You’ll see “the system scan will begin”.
The scan may take some time and windows will repair/replace any corrupt or missing files.

Close the Command Prompt Window when the job is finished.

If for any reason it still does not give you the option to run as administrator, you may have to turn user account control off temporarily while you run the test, but be sure you turn it back on once you are done.  It is there to protect you from malicious files running without your permission.
User Account Control (UAC) can help you prevent unauthorized changes to your computer. It works by prompting you for permission when a task requires administrative rights, such as installing software or changing settings that affect other users.

We don't recommend turning User Account Control off. If you do turn it off, you should turn it back on as soon as possible.

2.  Click Turn User Account Control on or off.  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

Hello Terry, my name is Alexander. I'm an Independent Advisor and I'm happy to assist you!

You can try to fix this issue by enabling this service or by starting it from the services. I recommend you a few troubleshooting methods:

Method 1: Start the service.
- Hold down Windows + R keys together on your keyboard.
- Type in Services.msc and click on Enter.
- Look for Tablet PC Input Service. There are two other services responsible for Tablet input to function correctly. (SL UI Notification Service and Software Licensing).
- Double click on the service and choose ‘Automatic’ in the drop-down StartUp type.
- Tap on Start or Restart to start the service.

Method 2: Enable the service.
- Hold down Windows + R keys together on your keyboard.
- Type in msconfig and click on Enter.
- Find System Configuration window, navigate to Services tab and look for ‘Tablet PC Input Service’.
- Put the check mark next to ‘Tablet PC Input Service’ option to enable it.
- Tap on Restart and check if the issue is fixed.

I recommend that you perform the steps provided by Anthony Jose on 2 December 2016 in the following thread

https://answers.microsoft.com/en-us/windows/for...

What purpose is the KB898461 ? 

 

On a manual visit to Windows Update through the browser with automatic updates turned off, the response of the Windows Update is known.  It first starts checking  your computer to see if you have the latest version of the software required.  If you have the latest available downloadable version already installed it will recognize it and offer through an ActiveX permissions prompt to update.

  3. Under System and Security, click Fix problems with Windows Updates.

4. Click on "Download now" button to start downloading the new update build of Windows 10.

After downloading the build, your computer will automatically restart to install the new update.

The download size will be around 2 or 3 GB depending upon the computer architecture (32-bit or 64-bit) and system language.

karentitterington say:

I was having a similar problem.  I have a Surface book.  My buttons worked but wouldn't write.  What I had to do was:

  • Go to the Bluetooth settings and uninstall the surface pen.
  • I restarted the surface (not shut down) and while it is restarting, hold down the top button until the light blinks, about 7 or 8 seconds. You can release the button and it will continue to blink. 
  • Go back to the Bluetooth settings and choose add new device and choose to add pen.  Surface pen should appear, click on it and click done.  You should see the cross hairs when you hover over the screen.  

Hopes it helps.  It's just a temporary fix.  Every time I shut down the Surface book for any length of time it will not work again and I have to repeat the above steps.  There must be some bug between the Surface and the Pen.  Hope they find a fix.

mark ole say:

Hi,

If your Surface Pen no longer interacts with the screen but buttons are working, replacing the battery usually resolves the issue. However, if you've already replaced the battery and still your pen doesn't work, you may try these solutions that may help you resolve the issue.

Restart your Surface device

If after restarting the device doesn't work, force shutdown the device.

  1. Press and hold the power button on your Surface for 30 seconds and then release it.
  2. Press and hold the volume-up button and the power button at the same time for at least 15 seconds and then release both.
  3. The screen may flash the Surface logo, but continue holding the buttons down for at least 15 seconds.
  4. After you release the buttons, wait 10 seconds.
  5. Press and release the power button to turn your Surface back on. You should see the Surface logo.


Run Hardware and Devices Troubleshooter

  1. Type Troubleshooting in the search box, click Troubleshooting.
  2. Under Hardware and Sound, choose Configure a device.
  3. The Hardware Troubleshooter will open.
  4. Click on Next to run the Hardware and Devices Troubleshooter.
  5. Choose the one you want to fix and click on Next. The Hardware and Devices Troubleshooter will try to fix the problem with the pen.

You may also check this article about troubleshooting Microsoft Surface Pen issues.

Let us know if you need further assistance.
