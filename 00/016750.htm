The problem is that the xyscatter template, for some reason, does not work as expected. I treats all 3 columns as Y values, regardless of text over headers.

I was suggesting using the normal XYscatter as the default chart type so F11 function works. Then on the newly created chart apply your template formatting.

Here is some simple code to do that process from a macro.

Sub XX()
    If TypeOf Selection Is Range Then
        With ActiveSheet.Shapes.AddChart.Chart
            .SetSourceData Source:=Selection
            .ChartType = xlXYScatter

' change path to your own template folder location and file name.

            .ApplyChartTemplate "C:\Documents and Settings\Andy\Application Data\Microsoft\Templates\Charts\DefaultXY.crtx"
            .Location xlLocationAsNewSheet
        End With
    End If
End Sub
