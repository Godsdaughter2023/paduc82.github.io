Those numbers aren't meant to add up. The Cached memory is recently used data being held within physical memory. The Available memory is the total amount that can be used for new purposes at any given time. That should include memory from the Cache that can be reallocated. The Free memory is either empty or has data in it that isn't being used by anything and can be erased for reuse. The memory that is currently in active use by the system should be indicated in green under the 1st memory graph.

The Available memory plus the used memory indicated in green under the graph always adds up to close or equal to the total memory in my system. Currently 2141 available plus 930 used = 3071.
