rich007a say:

Hi,

I think you got things the wrong way round in your question...

An Excel Workbook (file, *.xlsx or *.xlsm, etc.) contains WorkSheets.

 

If you want to have hyperlinks to individual worksheets that are all contained within the same Workbook, you can create a named range on each sheet in Excel, then in PowerPoint, edit the hyperlink applied to the arrow shapes by adding #MyRangeName at the end (i.e. # followed by the names of your named ranges).

 

Alternatively, you can just hyperlink to a specific cell by using:

 

C:\path\filename.xlsx#Sheet1!C3

 

for the hyperlink address.

 

Enclose the sheet name in apostrophes if the sheet name contains a space:

 

C:\path\filename.xlsx#'Sheet Name With Spaces'!C3

 

Hope that helps.
