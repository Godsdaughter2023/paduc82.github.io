How to print a Word document without printing the images - Microsoft Community

jpsrww say:

How to print a Word document without printing the images

Hi, I am using Word 15.41 for Mac and have a 400-page document that I need to print that contains numerous photographs and other types of image. I would like to print it out WITHOUT printing the images (toner hogs!). How can I do this?

I have converted the doc to Draft mode, but when I try to print from Draft view, the images still print. Help, please!

Forgive me if this question has been answered, but after much searching I've been unable to find a solution for Mac.

bob jones aka: cybertaz say:

As you've found, the on-screen viewing mode being used does not "convert" the document in any way & has nothing to do with printing :-).

It depends on whether some form of Text Wrap has been applied to the images, nut it should be noted that any of these options will impact the text flow. That may be an undesirable effect.

  • If they have some form of Text Wrap applied [not In Line with Text] you can prevent them from printing by either:
    • Going to Word> Preferences - Print, then clear the check for Drawing objects, or
    • In the Print dialog open Copies & Pages, select Microsoft Word, then click Options & clear that same check, or
    • On the Layout tab activate the Selection Pane then click the Hide All button.

Neither of those methods have any effect on In Line images.

  • Any images which are In Line with Text, however, will have to be selected individually in order to apply the Hidden font formatting property by going to Format> Font to check the box for Hidden. It would be more expeditious to use the keyboard shortcut (Shift+Control+H).

If the text flow needs to be retained, the only option I know of is to select each image & set it's Transparency to 100% which is the equivalent of making them act as Placeholders. This can be done regardless of whether Text Wrap has been applied to the images. The potential down side to this is that some printers don't handle Transparency very well, so there may be some very faint rendering of the images. One way to avoid that may be to create a PDF, then print the PDF.

Depending on the intended purpose of the printed output it may be simplest to just save a copy of the document as Plain Text (.txt) then open & print the text file. That strips out any graphics but also alters the text flow as mentioned above.
