hansv mvp say:

Do you mean Access 2010 - autonumber, but allow me to create the number to start with?

After creating a table with an autonumber field, you can use an append query to add the first record with the desired starting value for the AutoNumber field; new records will continue from that value.

It's also possible to use VBA code to set the starting number.

 

Be aware that if a user starts to create a record, then cancels it, the autonumber value of the canceled record won't be reused, so there might be gaps in the sequence. If that is a problem, don't use an autonumber field but calculate the sequential numbers yourself - see Sequential Numbering on ScottGem's blog.

devlin sco say:

Hello,

You can use an inbox rule to manage your emails. We're going to set a condition, whenever the sender's address includes the words .space, it will be sent to the Junk Email folder.

  1. Click the gear icon located in the top-right corner of Outlook.com.
  2. Select Options.
  3. Under Mail, select Inbox and sweep rules.
  4. Under Inbox rules, click the add icon +.
  5. Set the condition to It includes these words in the sender's address and then set the value to .space.
  6. For the action, set it to Move the message to folder and then select Junk Email.
  7. Click OK to finish.

You can also set the action to Delete the message if you want to.
