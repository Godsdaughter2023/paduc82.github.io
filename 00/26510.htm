I cant save my document in word..

Do you mean save the document to your computer?  If so, you can do this in several ways.  Once hovering over the file, you can click the more option and then choose download.  This will download the file to your computer.  You can then modify the file in the Microsoft Word desktop application.  Once you are done making your changes you can manually upload it back to SkyDrive.

Alternatively if you click the file's name it will open in the Word Web App viewer.  From there you can click "Open in Word" from the toolbar to open it directly in the Microsoft Word desktop application.  (I'm not sure if there are any restrictions on which version of Microsoft Word is needed, it worked fine when I just tested it in Microsoft Word 2010.  Earlier versions, like Microsoft Word 2007 will probably work too.)  A similar option to "Open in Word" exists from inside the Word Web App editor, if you click "Edit in Browser" when hovering over your file.

The benefit of choosing Open in Word from within the Office Web Apps is that every time you save the file in the desktop applications any changes that you make against the file will automatically be pushed to the corresponding on SkyDrive!

 

i cant make changes
There are three ways to make changes to a Microsoft Word document being stored on SkyDrive.  The first two are outlined in the response just above this.  They were.
  1. Download the file, make changes, and re-upload it.
  2. Open the file in the Microsoft Office Web Applications, and then click "Open in Word" to have your changes be pushed to SkyDrive every time you save.
The third option is to use the Microsoft Office Web Applications, in this case the Word Web App, to make changes directly in your browser.  The Word Web App acts as a light web companion to the Microsoft Office desktop application.  It allows for light editing such as inserting images, tables, and editing the text of your document.  While it doesn't share all the functionality of the rich client, and may not display render all the contents of the document when editing them, the full fidelity of your file should be preserved every time you save.

To use the Word Web App editor to modify your file through your browser, you can click "Edit in browser" while hovering over the file's name.  Alternatively, if you've already loaded the Word Web App viewer, you can click "Edit in browser" from within that application.
i cant print

Another option to print is to click the "More" button, this drops down a menu with a few options.  Among these options is "Download."  This will download the corresponding document from the SkyDrive server to your computer.  Once the file is on your computer, you can choose to open it in the Microsoft Word desktop application.  If the desktop application is not on your computer, you can choose to open it in theWord Viewer to view or print the document.

Since you might just use Copy and Paste to post your picture, it cannot be shown up correctly in your post. For uploading a picture, you could use the button Insert Image here.

Back to your post, if you want to achieve the following display effect in Windows 10, we have to modify some values in Registry.

The steps as below:

1.Press Win logo key + R and type regedit to access Registry

2.Access Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

3.Create a DWORD (32-bits) Value called ExtendedUIHOverTime with 9000 value data.

4.Access Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Taskband

5.Create a DWORD (32-bits) Value called NumThumbnails with 0 value data.

6.Save your modification and restart your PC.

( Disclaimer: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall your operating system. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Edit at your own risk. )
