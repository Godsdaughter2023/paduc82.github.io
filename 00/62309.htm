¡firedog say:

How are you composing messages? Do you type into the message compose window, or are you pasting from some other program like Word?

There are several ways of indicating line breaks in HTML.

  • The basic one is simply a line break, using the single tag <br> or <br />. This does just what it says: the following text will start on a new line when the HTML is rendered.
  • Another common way is to create a new paragraph by enclosing text in a pair of <p>  </p> tags. Each new paragraph will start on a new line, with whatever formatting (e.g. space before) is specified for <p>.
  • Nowadays, it's usual to see text that's supposed to start on a new line enclosed in dividers <div>  </div>

Windows Live Mail interprets HTML tags pasted from another program in different ways, but I think you'll find that it normally uses the last of these three methods - <div>  </div>. If you only want text to start on a new line, with no other change in formatting, use the first method however you compose your message. To do this, use Shift-Enter to insert a line break. Do it twice to insert a blank line before the following text.

a. user say:

Hello Bryan,

Thank you for responding quickly with the answers and the status of the issue.

You may not be able to access the options to perform the above method due to the local user profile status. Enable built-in administrator by following the steps below, create a new administrator user profile and check if the issue is resolved.

Step 1: Enable built-in administrator account.

  1. Open elevated command prompt by entering Safe mode with Command prompt. To go to Safe mode, use the power button on the sign-in screen and press Shift (key on the keyboard) + Restart. This will take you to the recovery boot menu. Click Troubleshoot> Advanced options> Startup settings. You then see a list where you can choose Safe Mode with Command prompt.
  2. Type the following command: net user administrator /active:yes to activate the built-in administrator. 
  3. Login into the built in administrator user profile and create a new user profile with administrator privileges (steps given below).
  4. Then lock your screen by pressing Windows key + L keys on the keyboard and sign-in into the Administrator account.
  5. Disable built in administrator account by typing the command: net user administrator /active:no in the Command prompt (Admin).

Step 2: To create a new administrative user account follow the steps below:

  1. Open Elevated command prompt by pressing Windows logo + X keys on the keyboard and select Command prompt (Admin) from the context menu.
  2. Type net user NewAccount password /add and click Enter.
  3. Type net localgroup Administrators NewAccount /add and click Enter.
  4. Log off from the system and login to the new user account.

Write to us with the status of the issue for further assistance, we will be happy to help you. Your reply is most important for us to ensure we assist you accordingly.
