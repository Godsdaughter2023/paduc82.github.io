graham mayor say:
Save the document as Word 97-2003 DOC format and open it in Word 2000. Then use the following macro on the document. There are three issues with this

1. The macro will take some time to run on a long document. Be patient!
2. The macro looks for words formatted with the Highlight tool in red colour. It does not find any other type of highlight.
3. I do not have access to Word 2000 to test the macro, but it should work.

http://www.gmayor.com/installing_macro.htm


Sub HowMuchRed()
Dim iRed As Long
Dim iAll As Long
Dim i As Long
iAll = ActiveDocument.Words.Count
iRed = 0
MsgBox "This could take a while - please wait until the macro completes" & vbCr & _
"Click OK to continue", vbInformation
For i = 1 To ActiveDocument.Range.Words.Count
    If ActiveDocument.Range.Words(i).HighlightColorIndex = wdRed Then
        iRed = iRed + 1
    End If
Next i
MsgBox "The percentage of red highlighted words is " & Format(iRed / iAll, "0.00%")
End Sub
