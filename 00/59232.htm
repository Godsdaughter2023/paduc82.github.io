By default the cmd you are using to get the log file into a readable format uses the online Microsoft symbol library. However the symbols for the preview releases are not available there so that won't do any good. To set the symbol library manually you would use Get-WindowsUpdateLog -SymbolServer https://msdl.microsoft.com/download/symbols

If you went and downloaded one you would just type the location you saved it such as Get-WindowsUpdateLog -SymbolServer C:\Users\Documents\MySymbols. However that is usually only useful if you are programing something and have your own library

What you can do is run these

Get-WindowsUpdateLog -SymbolServer https://msdl.microsoft.com/download/symbols

Wait a few minutes for it to completely write the file

Run set-location C:\Users\YOURUSERNAME\Desktop and replace YOURUSERNAME with yours

Then run (Get-Content ".\WindowsUpdate.log") -notmatch "1600/12/31" | Out-File -Encoding ASCII ".\WindowsUpdateClean.log"

This will remove all of the useless info and give you a clean log to read.

Have you ran dism /online /cleanup-image /restorehealth and tried to install the updates? Are you getting error codes on the updates?
