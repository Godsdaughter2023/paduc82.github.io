Getting an #N/A error in VLookup formula only on certain words - Microsoft Community

joyakl say:

Getting an #N/A error in VLookup formula only on certain words - how do I fix?

I have a Vlookup to compare text from a cell (c6) to the first column of the table. If it matches, then return to the current cell the value from column two of the table.  If it is false, then calculate cell E6 *0.1.

This works great except when my search starts with the word Angle or Bearing. I've tested in different worksheets and rows. *Also doesn't work with numbers (1,2,3) but I'm ok with that. 

I would appreciate any advise!

Here is my formula:

=IF(C6=VLOOKUP(C6,COMITEMS,1,TRUE),VLOOKUP(C6,COMITEMS,2,TRUE),E6*0.1)

My Vlookup table (COMITEMS)

BELT OPERATOR 20
CHAIN OPERATOR 15
JACKSHAFT OPENER 20
LEAD 10

teylyn say:

Hello,

if the value is not found, then VLOOKUP will return the #N/A error, so your IF statement should check if Vlookup returns an error, and then execute the respective other bits of the IF statement. 

=IF(IFERROR(C6=VLOOKUP(C6,Comitems,1,TRUE),FALSE),VLOOKUP(C6,Comitems,2,TRUE),E6*0.1)

If you change the Vlookup to an exact match using FALSE as the last argument, then the formula gets a bit shorter. The approach with the double Vlookup is only beneficial if you have a very large dataset and a Vlookup with an exact match takes a long time. So, if that is not the case, you could try

=IFERROR(VLOOKUP(C6,Comitems,2,FALSE),E6*0.1)

If the Vlookup returns an error (because the value is not found) the calculation with E6 will be returned.
