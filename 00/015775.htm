Hi AndreThe

Circular references can cause your worksheet to go into a loop and may stop the sheet calculating

A circular reference is when a cell references itself and every time that cell updates, it runs the formula again, updates the cell and so on to infinity . . .

On the Formulas Ribbon, click the Error Checking drop Down - Circular reference

There you will see the cell(s) with circular references in them, you need to resolve those for your sheet to work correctly

Once those are resolved, press F9 to see if that sheet calculates correctly

If it does:

Go to File - Options

On the left, click Formulas

On the right under Worksheet Calculations, set that to Automatic . . .

12th Jan, 2010

 

Ok guys let's see if this helps...  the problem is NOT Nokia PC Suite, the USB to Nokia Enhancement connector ie; the CA-42 data cable, your cell phone or your computer. 

 

 

It's in your computer' settings... The answer to this problem with this particular cable is actually in the Nokia PC Suite "Help".  It's  under "Configuring connection types"  They  refer to it by name. 

 

Unfortunately, (1) you kinda have to stumble into it... thank goodness I did and,   (2) you kinda have to be a bit of a computer geek to understand what they are trying to say... 

 

The point is that they are basically tricking the computer to "see" the USB cable as a 9 or 25 pin serial DIN cable connection that would normally go into the COM1 or COM2 ports, which are still on the back of some PCs, but not available on modern laptops anymore. 

 

My PC is Windows XP Service Pack 3.  I will describe what I had to do to get the cable to work.  Vista and Windows 7 users… just follow the principle. 

 

You need to (1) configure Nokia PC Suite as well as (2) your PC/Laptop (in Control Panel) to "see" the USB connection as - COM3... here's how I did it for my PC. 

 

Step - 1:  Install the CA-42 Cable Drivers first using the install CD that was provided with the cable using "Found New Hardware"  remember this is the XP experience... 

 

Some preamble - I thought that installing the Nokia PC Suite does all the cable driver bits as well, but it looks like you actually have to install the cable drivers (Note – cable drivers first) and then the Nokia PC Suite app software separately.  I got a “file not found error - ser2pl.sys” installing Nokia PC Suite.  You don’t get that when you let the “Found New Hardware” wizard install it from the install CD. Don't restart your computer until the end of this rather lengthy two-step process... just click "Restart later"...  I'll tell you when to restart... 

 

START POINT (Still Step 1) - Ok, so now we are starting....

 

  1. Shut down your system...
  2. Plug in the CA-42 USB cable into a USB port.  Important here… if you have a PC, decide on which USB port you are going to use ie; front or back USB ports.  If you successfully install the cable drivers on one USB port, you cannot go taking out the USB cable and sticking it into another USB port on the same computer.  You will lose the cable driver install completely.  However if you plug the USB cable back into the original USB port used to install the cable driver software, it will re-appear in “Device Manager”.  Maybe it does the same with laptops as well, I do not know.. 
  3. Turn on your system...  do not connect your phone to the CA-42 cable yet. 

 

On restarting, Windows will find new device (CA-42 USB to Serial), try to install it automatically and come back and say... it was not installed properly (probably because of COM port conflicts)... let that finish, close it off and ....

 

While that was happening a "Found New Hardware"  wizard will appear...let it happen.. do not close this.. let it sit and wait, we want to use this.... it is going to ask for the install CD, but guess what... when you insert the install CD (at least, if it's the same as the on I got) it’s going to kick off yet another install process.   

 

Put the supplier's install CD in your player, that is going to kick off the CD's automatic install (autorun.inf),  cancel it off.. do not run the automatic CD's install, it will not work !.  Continue with Found New Hardware and let Windows find the CA-42 cable drivers on the install CD (from your drive D: or whatever) and install… do not restart your computer yet.

 

At the end of this, if you check Settings - Control Panel - System - Hardware - Device Manager - Ports (COM & LPT),  you should see an entry for - "CA-42 USB (COM something).  That COM something will probably be COM1 or COM2.  Highlight this and right click.. go Properties, Port Settings, Advanced – In XP, there is a box to select “COM Port No.”  Set this to COM3.  The driver should say that the “device is working properly”.  Click Apply, OK and exit Control Panel….  Do not restart yet.. 

 

So now we have XP set… let’s set the app…. Nokia PC Suite 

 

STEP 2 – Setting Nokia PC Suite. 

 

Some preamble - In July 2009, I bought version 6.4.8 of Nokia PC Suite with the CA-42 cable.  I installed it and I was unable to get the cell phone to see “Nokia PC Suite”.  After much trial and error, I flung it aside. 

 

Well this past weekend , I decided to have another go at it. 

 

I uninstalled it and downloaded and installed a more recent version from Nokia’s support site… version 7.1.40.1  … cell phone still giving “Enhancement not supported” message.  Could not believe that I was that unlucky… 

 

Again after much trial and error, I got it to work – a classic case of the simplest thing being so darn difficult…. So here goes.. 

 

Download the latest version of  “Nokia PC Suite”, if the version you get is the one I got (7.1.40.1), , then you need to -  

  1. Do not connect the CA-42 cable to your phone yet.  It should be connected to your computer on the USB end in it’s original install USB port !!
  2. Start up Nokia PC Suite - Go Settings, Manage Connections
  3. Tick off only “Serial Cable (CA-42), this should be in the list assuming you installed it correctly in Step 1
  4. Highlight it and the little spanner (Configure) at the bottom should now be enabled
  5. Click on Configure.. the little spanner
  6. If it is saying “COM1” click on the little down arrow to the right.  COM3 should come up in the list
  7. Click on COM3… stop right here,  do nothing

NOW….  you plug in the Enhancement end of the cable into your phone.  If it still gives the “Enhancement not Supported” error, don’t panic.  Make sure the cable is into the phone nice and secure.  By the way, this cable tends to get “Disconnected” with the slightest movement of my almost 4yr old Nokia 6101.  Make sure you are not using the phone and the phone is in standby mode.  The phone MUST NOT be in “Contacts” or some other menu available in the phone otherwise it will not connect to Nokia PC Suite… STANDBY MODE please… I am sure I frustrated myself unintentionally by not paying attention to this detail. 

 

Right, NOW you click on OK (the green check mark) … assuming everything else was done properly, after about 10 seconds, the model no. of your phone should appear in Nokia PC Suite.  If you get this, you are going to have a blast with this software. 

 

So cool – using my PC….. I have since been able to delete off a lot of junk (from inbox, outbox, images etc) and free up valuable memory, amend Contacts info, do a full backup of my phone onto my PC, load in a truncated MP3 file of Doobie Brothers, "China Grove" as my ringtone (...made it myself with a free MP3 editor - 1st 30 seconds – 433Kb… yeah huge… but worth it) etc…
