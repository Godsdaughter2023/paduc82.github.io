The numwords from the status bar indicator is closely tied to the Word Count dialog and will duplicate the number of words shown whether the toggle for Include textboxes, footnotes and endnotes option is on or off.
The { numwords } field never includes the word count from textboxes, endnotes and footnotes.

julie sheets say:

Why would you expect the duration to update if you have fixed it?  You have far more than 40 hours of work assigned to resources - I see a total of 98 hours of work.  If you assign additional resources to a Fixed Duration, effort driven task, all that happens is the peak units for each resource is reduced.  The 80% assignment unit you see in the Task Form is not the correct value - you'd need to add Peak to the Resource Usage or Task Usage view.

I'm not clear from your description what you expected.  If you want to figure out how long 40 hours of work will take if performed by 7 people all assigned at 80% - then setting the task as fixed duration will not do that.  I'd suggest Fixed Units, Effort driven.  With one person assigned all 40 hours, the duration is 6.25 days.  If you continue to add resources, the total work (40 hours) will stay the same and the duration will drop for each resource you add.  Assuming all are to perform the same proportion, by the time you have added all 7 resources, each resource will be scheduled for 5.72 hours of work and the task duration is .89 days.
