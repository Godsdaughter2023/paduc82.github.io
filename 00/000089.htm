albert d. kallal say:

Actually you don't need to download it, the help files are installed locally, but you have a choice of settings in the help system to use the online content, or the local help.  I simply suggest that you change the help to the local, you'll also find that it works quite a bit better in VBA when you do this.

To change this setting, simply bring up the help system, and click on the little drop down box just to the right of the search button that appears in the menu bar.  At that point you'll see a list of options to choose content from office. com or your local computer.

There are two options for help on local content, and I suggest you click on the developer reference.  One more tape, is in the lower right corner you'll see little box that says if you're offline or online, and changing the settings there will force access to remember this next time you launch it, else you have to go to the above ongoing process every time you want access.

So when the lower right choose the option

Show content only from this computer.

If you do the above, then the connection to the online help system will not occur anymore.

So there should be no need to download a separate copy of the help files (however if for some other reasons you do want a separate help copy there is copy of the help file that you can download local – I don't have the reference handy, but the above steps should suffice for your needs anyway).

Is there a quicker way of doing this besides checking each one individually. (Yes, I do have enough that it can be time consuming). Any assistance will be much appreciated.
steve rindsberg say:
There's nothing like this built into PPT but a bit of VBA and a link checking program like Xenu (google it ... it's free) would do it.

Here's the VBA.  You'd want to run it and press Ctrl+G to view the output in the immediate window if it's not already visible.  There's a limit to how much the window will show, so if the <html> line at the beginning has scrolled off, shout.  We'll show you how to toss the results off to a file.  Right now it's silly late, I'm sleepy and about to knock off for the night.  Simple=Good for now. ;-)

Anyhow, you'd just need to copy the results from the immediate window paste it into a notepad file and save as HTML.  Open in a browser to test it, then pass it off to Xenu or the like to check the links quickly.


Sub MakeLinkPage()

    Dim sTemp As String
    Dim oSl As Slide
    Dim oHl As Hyperlink
    
    sTemp = "<html>" & vbCrLf & "<body>" & vbCrLf
    
    
    For Each oSl In ActivePresentation.Slides
        For Each oHl In oSl.Hyperlinks
            sTemp = sTemp & "<a href=" & Chr$(34) & oHl.Address & Chr$(34) & ">" & oHl.Address & "</a><br />" & vbCrLf
        Next    ' hyperlink
    Next    ' slide
    
    sTemp = sTemp & "</body>" & vbCrLf & "</html>" & vbCrLf

    Debug.Print sTemp
    
End Sub

It would appear that the link is no longer available.  However, the same thing is accomplished by finding your ZuneStore.sdf file and renaming as follows:

 

First, close your Zune software.  Then do a search on your computer to find it the ZuneStore.sdf file.  You may need to change your folder options to check "Display contents of system folders" if you have that option turned off.  When you find the file, rename the file as ZuneStoreold.sdf (this will be your backup if you want to change back to the old data file).  Open your zune software (it will be creating a new ZuneStore.sdf file); the collection will be empty but you will see at the bottom on the status bar that it is finding your collection again.  This may take some time if you have a big collection.
