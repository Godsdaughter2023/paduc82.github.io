The default currency depends on your Windows system settings (Regional and Language Options).

 

You could create a macro in your personal macro workbook Personal.xlsb:

 

Sub DollarFormat()
    On Error Resume Next
    Selection.NumberFormat = _

End Sub

and assign this macro to a Quick Access Toolbar button. You can select the "$" icon for this button.
