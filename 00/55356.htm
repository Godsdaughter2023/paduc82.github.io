Hi Josh,

An unknown background program is possibly preventing the SFC tool from running. I suggest running both the DISM command-line tool and the SFC tool in safe mode. Both tools are designed to scan for corruptions in Windows system files and restore corrupted files. Safe mode will start your Surface Go in a basic state, using a limited set of files and drivers. This will help us troubleshoot the issue without interference from third-party programs. Refer to this support article for the steps on how to start your device in safe mode. Once booted in safe mode, follow these steps:

  1. Click the Start button and type Command Prompt.
  2. Right-click Command Prompt and choose Run as administrator.
  3. Enter DISM.exe /Online /Cleanup-image /Restorehealth and then press
    • Note the space before each "/".
    • This step may take a few minutes to start and up to 30 minutes to run and complete.
  4. Enter sfc /scannow and then hit Enter.
    • Note the space between "sfc" and "/".
  5. Restart your PC.

Should the issue remain, you can reset the Windows Update components by running the Reset Windows Update Agent tool by Manuel Gil. Please make sure that you're signed in as the administrator to run the tool.

  1. Go to the download page.
  2. Click ResetWUEng.zip and then choose I agree.
  3. Select download once prompted and then open the file.
  4. Double-click ResetWUEng and follow the on-screen instructions.
