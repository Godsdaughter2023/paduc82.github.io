A DVD may not play properly in some DVD players or DVD-ROM drives for any of the following reasons:

• The disc type may not be compatible. Some DVD players or DVD-ROM drives can only play certain kinds of DVDs. For example, some DVD players only play DVD+R or DVD+RW discs, while others only play DVD-R or DVD-RW discs.

I understand your concern, as Cortana uses Internet Explorer to access its resources, you might be prompted for credentials if Prompt for user name and password option is enabled in Internet Explorer, or if the Integrated Windows Authentication check box is not selected in the Internet Explorer. This issue might also occur if you are using a proxy server for the LAN settings in Internet Explorer.

Let us try few troubleshooting methods to resolve this issue and check if that helps:

Method 1: Set the proxy server setting in Internet Explorer.

To do that,

  1. Open Internet Explorer.
  2. Click on the settings icon on the top right corner.
  3. Now click Internet Options, and then click the Connections tab.
  4. Click on LAN settings, uncheck the Use a proxy server for your LAN check box, and then click OK.

Method 2: Enable Integrated Windows Authentication.

To do that,

  1. Open Internet Explorer.
  2. Click on the settings icon on the top right corner.
  3. Now click Internet Options, and then click the Advanced tab.
  4. Check if Enable Integrated Windows Authentication is check if not check mark the option.

Method 3: If the issue persists change few security settings in Internet Explorer.

To do that,

  1. Open Internet Explorer.
  2. Click on the settings icon on the top right corner.
  3. Now click on Internet option, and then click on the Custom Level.
  4. In the Security Settings dialog box, make sure that Automatic logon with current username and password is enabled under User Authentication.
  5. Click on OK.
