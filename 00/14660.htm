hansv mvp say:

Open the database.

Locate the relevant form in the Navigation Pane on the left hand side of the Access window.

Right-click the form in the Navigation Pane and select Design View from the context menu.

 

Click once on the chart control to select it.

If you don't see the Property Sheet, press F4 to activate it.

The name of the chart control will be displayed in the dropdown list near the top of the Property Sheet, and also in the Name property in the Other tab of the Property Sheet.

Make note of this name - you'll need it in the code.

 

Now click on the button at the intersection of the horizontal and vertical rulers to select the Form as such. Alternatively, click in the gray area outside the form, or select Form from the dropdown near the top of the Property Sheet.

Activate the Event tab of the Property Sheet.

Click in the On Current box (it should be listed at the top).

Select [Event Procedure] from the dropdown list in the On Current box.

Click the so-called builder dots. i.e. the ... button to the right of the dropdown arrow.

This will activate the Visual Basic Editor and open the code module of the form, with the insertion point in the blank line in the following code:

 

Private Sub Form_Current()
 
End Sub

 

This is the event procedure for the On Current event of the form.

Make the code look like this, either by copying the code between Private Sub Form_Current() and End Sub from this thread, and pasting it into the code module, or by typing it:

 

Private Sub Form_Current()
    ' Change axis scale
    With Me.Chart0.Object.Application.Chart.Axes(xlCategory)
        .MinimumScale = Date
        .MaximumScale = DateAdd("yyyy", 5, Date)
    End With
    ' Force update
    Me.Chart0.Requery
End Sub

 

Important: replace Chart0 with the name of your chart control - you made a note of it a bit earlier.

If the name of the chart control contains spaces or punctuation, you must enclose it in square brackets [ ]. For example, if the chart control is named My Chart, the code should look like this:

 

Private Sub Form_Current()
    ' Change axis scale
    With Me.[My Chart].Object.Application.Chart.Axes(xlCategory)
        .MinimumScale = Date
        .MaximumScale = DateAdd("yyyy", 5, Date)
    End With
    ' Force update
    Me.[My Chart].Requery
End Sub

 

When you have modified the code, press Alt+F4 to close the Visual Basic Editor and to return to Access.

Presumably your document has cross-references to some or all these bookmarks, but there's no indication of that in your code. Private Sub CommandButton1_Click()

End Sub

Sub UpdateBookmark(StrBkMk As String, StrTxt As String)
Dim BkMkRng As Range
With ActiveDocument
  If .Bookmarks.Exists(StrBkMk) Then
    Set BkMkRng = .Bookmarks(StrBkMk).Range
    BkMkRng.Text = StrTxt
    .Bookmarks.Add StrBkMk, BkMkRng
  End If
End With
Set BkMkRng = Nothing
End Sub
