daniel g. grau say:

THEORY

You've inserted the page number within a frame and there's no room there for both digits. A minor difference between like-font versions is enough to make the eighties digit disappear from view.

Although all digits take up the same room, you may have periods, spaces or hyphens involved, and these will surely vary with a different font or even a different version of the same font. So make sure you use a font that your recipient also has in their system.

If you click on the number, several handles should appear. Just make the box larger to ensure there's ample room for two digits.

teylyn say:

Hello,

if the values are real date/time values, you can simply sort them.

Chances are, however, that since they were imported from another system, they just look like date/time values, but are simply text. In that case, sorting won't show the desired result, since it will be an alphabetic sort, where values starting with "Jul" will be sorted before values starting with "May".

So, you need to convert the text to real date/time values.

To extract the date, you need to know what the order of month and day is in your computer settings. If it is DMY (for day, month, year) as in 24/12/2017, then you can use this to extract the date

=DATEVALUE(TRIM(MID(A2,FIND(",",A2)-2,2))&" "&LEFT(A2,3)&" "&MID(A2,FIND(",",A2)+2,4))

If you use MDY (for month, day, year) as in 12/24/2017, then use

=DATEVALUE(LEFT(A2,3)&" "&TRIM(MID(A2,FIND(",",A2)-2,2))&" "&&MID(A2,FIND(",",A2)+2,4))

To extract the time, use

=TIMEVALUE(MID(A2,FIND(" ",A2,10)+1,15))

To get the date and time into the same cell, add up the two formulas

=DATEVALUE(TRIM(MID(A2,FIND(",",A2)-2,2))&" "&LEFT(A2,3)&" "&MID(A2,FIND(",",A2)+2,4))+TIMEVALUE(MID(A2,FIND(" ",A2,10)+1,15))

The result is a real date/time value and can be sorted correctly.
