hansv mvp say:

Here is a macro you can run:

Sub ReplaceHyperlinks2()
    Const strOld = "..." ' old server address
    Const strNew = "..." ' new server address
    Dim hyp As Hyperlink
    For Each hyp In ActiveSheet.Hyperlinks
        With hyp
            .Address = Replace(.Address, strOld, strNew)
            .TextToDisplay = Replace(.TextToDisplay, strOld, strNew)
        End With
        hyp.Range.Replace strOld, strNew, xlPart
    Next hyp
End Sub

 

Substitute the correct addresses in the constants at the beginning of the code.
