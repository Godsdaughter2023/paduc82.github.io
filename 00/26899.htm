I've spent some time investigating this. What I've found is that the behavior is consistent in Mac Word at least as far back as Office 2004 (version 11.x). There are 2 possible behaviors:

  • If you add the envelope to the current document [Insert this envelope... check box] the page number
    The envelope feature in PC Word is a little different, so it doesn't apply the page numbers if you do print directly from the Envelopes & Labels dialog. Then, again, the PC versions do not provide for generating a separate envelope document -- You either Print.

    My conclusion is that by not adding the envelope to the document certain document parameters seem to be ignored --
    That having been said :-) I wouldn't expect to see any change made [at least not as an update to Word 2011] unless it is a rather simple fix. Just my guess, but I would expect that it's encountered by very few users because adding Page Numbering to the default Normal template isn't commonly done. Unless I'm quite wrong this issue would most likely rank rather low on the priority list because that's the only circumstance where it occurs.

    As I suggested earlier, modifying the Normal template to include any type of content [e.g., Page Numbering] generally is counterproductive. Please don't misunderstand -- I'm not telling you what to do or that how you choose to proceed is generate standard envelopes.. In my experience the Envelope mechanism has consistently worked fine... as long as the Normal template hasn't been altered in certain ways.

Due distinctions in print support offered by each of the operating systems it isn't uncommon for any given printer's Mac driver to present the printer differently than does the driver for Windows. It's also typical that one driver package may warrant updates not pertinent to its counterpart due to respective OS revisions.

    tryx3 say:

    1  No

    2  Yes, through Windows update

    3  No.  Your existing operating system will continue to operate as it always has.  You will continue to get the same sort of Windows updates [security updates] that you get now and you will keep getting them until 2010.

    4  [Edited a few hours later - I misread charges as changes]  No, there are no charges for using Windows 10.  The upgrade will be free for licensed Windows 7 & Windows 8.1 computers for the first year [i.e. to sometime in Summer 2016].  Subsequent Windows updates will, like Windows 7 updates, be free.

    Windows 10 does what Windows 7 does.  There are some differences but nothing to get excited about.  The changes to how it does it are significant so security is going to be better.  They bundle some small applications ["apps"] with it [Photos, Maps, Mail, Calendar, Music & Video] but, if you have already been using their Windows 7 equivalents, this is just a change not an improvement.

    Installing the Insider Preview is only worthwhile if you want to experiment with it and contribute your feedback to its development.  If you do consider installing the Insider Preview version then it ought to be on a spare computer or a spare disk.  You can get close to this by installing it as a "dual-boot" with your existing operating system but this is not risk-free.  If you want to find out more then search this forum for dual boot and you'll find the procedural guides that others have produced - one such is How to dual boot the latest Windows build with previous versions
