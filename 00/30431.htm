davepapalazarous say:

 

If the "show icons never thumbnails" option is checked then all explorer views [including all the different size choices for Icons] will show an icon only. So in effect you will loose the ability to view the Office, PDF, Picture files etc as "true" thumbnails. However the ability to show the true thumbnail images in explorer is not lost [as it is stored in the file configuration from the last save operation] and will reappear again if you uncheck the box.

The option two boxes down "display file icon on thumbnails" only effects wether an icon overlay [shown at bottom right of file] is visible in explorer when viewing files as true thumbnails is being used.

This option is totally down to user preference. For instance, I always leave this box checked as many of my folders contain both Word and PDF files containing duplicate information with identical filenames. If [as many people do] explorer is configured to hide file extensions for known file types [default], then with no Icon overlay [box unchecked] it would be difficult to differentiate between them as all you would see is a thumbnail of the documents' first page.
