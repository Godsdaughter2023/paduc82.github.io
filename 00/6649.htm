HOW TO Embed Album Art data into AAC files - Microsoft Community

m80116---01 say:

HOW TO Embed Album Art data into AAC files

Since I spent the whole day working hard to a solution for my (as I see) quite common problem, I would like to share with you my way to embed album art pictures in AAC files. It's a quite simple way honestly, but you need the right tools.

From now on I'll refer to the device based on my Nokia N95 8 GB (N95-2 or RM-320) with firmware v30.0.018.

I've chosen to concentrate on AAC file container since I suppose it being more common and flexible than M4A, supporting ID3 tags like used in MP3s. The obvious choice for coding was HE-AAC v2 (eAAC+) since it's very suitable for mobile devices regarding compression performance, size/quality ratio, and power efficiency.

The first way to embed the Album art is probably on the device, thing that I skip since I need to do it for a great number of files and wanted a semi-automated process, given the files you feed are already tagged either you have related cover art stored for them (to be used in the converted files), and need to be converted to the new format (i.e. a huge MP3 library to AAC).


The best way I could find to do that are these two programs: Easy CD-DA Extractor and WinAMP.

While WinAMP is a well equipped multimedia player Easy CD-DA Extractor with its File Converter is more aimed to extensive conversions and would be my first choice.

WinAMP 5.541 is able to do the job very straight forward, just chose "Send to..." then File Converter, you select the options (encoder, bitrate, etc) and you are done. The album arts are automatically transferred to the encoded files. You can of course insert a picture you provide in the original file before the process start, to do that just dobule click the song's title and go to the tab "artwork", from there you should be able to select a JPEG image as a "cover".

The other software is Easy CD-DA Extractor File Converter, which also has a wide array of formats and containers, lets you select your album cover art, would just be perfect for large collections BUT, we have reports this program is causing problems writing the cover art incorrectly.

I went through these problems myself and did spend my whole day figuring it out that was the program and NOT THE DEVICE !

Whether these problems are caused by the program itself or not is actually unknown. What I reckon based on my knowlege is that version 10.2.0 was able to write them correctly. Others reported versions above 11.x not being able anymore to embed album art metadata correctly. The new version 12.0.0 Build 1 if downloaded as a trial from their website is able to produce perfectly tagged files with embedded album art readable by the device.

On Easy CD-DA Ext. you are allowed to change the cover art directly from the main form under the tab "Additional ID3" and you can do that with multiple files at once.

If embedding album-art for your whole MP3 collection is too tedious I suggest you try a specific tagger program like Tag&Rename.

What I recommend is to always try an alternate program for tagging before you complain. Some files on which I had problems with the device clearly showed JPEG coding breakage at some point in the album art picture and they were not displayed on the device. Also try to download the newest versions of the respective software from manufacturer websites. Doing cross inspection like opening in onother player (WinAMP) what incorrectly tagged by Easy CD-DA halped me to isolate the problem.

I've also discoverd that if you already have the very same file on the device, on the same path, with the same metadata and you replace it with a correctly embedded album-art copy the image will show up the next time the file will be played, no matter if you refreshed the library or not.

When searching a contact, Skype will give you Skype names that are identical to the one that you are looking for.

 

If there’s more than one matching result and you’re not sure which one is your friend, right click on one and select View Profile. It's like their Skype business card which contains all the details they've added – like a photo, city, country or phone number.

 

If profile information can’t help you find your friend, ask them for their Skype Name or the email address they registered with.
