Header and Footer placement in a document - Microsoft Community

johng9210 say:

Header and Footer placement in a document

Word 2007. I'm having a tough time understanding the Page Setup dialog box and how it affects the placement of headers and footers in a document. It looks like the header takes up the space between the top paper margin and the start of the text area, and the footer takes up the space between the bottom paper margin and the end of the text area. Is this correct? Is there a good explanation of this anywhere?

suzanne s. barnhill say:

See http://wordfaqs.mvps.org/headerfooter.htm for general discussion of headers and footers, but here's the gist of it.

The header margin determines where the top of the header is. By default, this is 0.5" from the top of the page. With the default 1" top page margin, that gives you half an inch to play with before the header begins to encroach on the document--which it will do: the header expands to fit its content and will push the document body down as required (this can be helpful; see http://wordfaqs.mvps.org/DifferentMargins.htm).

The footer margin determines the bottom of the footer, which extends up as content as added, encroaching on the bottom of the document body if necessary. Again, the default footer margin is 0.5"; with the default 1" bottom margin, you have half an inch of footer space.

Word's representation of the header and footer is somewhat misleading, especially if you display text boundaries. The header space is shown intuitively, with the top at 0.5" and the Header line at 1", but the footer is confusing, since the space enclosed by the dotted text boundary line (and the highlighted portion of the vertical ruler) appears to extend from the top of the footer (with the text baseline at 0.5") to the bottom of the page. Once you know how headers and footers actually work, you can ignore this deceptive appearance and treat them intelligently.

How do I set up an email account to be send only and not receive? - Microsoft Community

emnh say:

How do I set up an email account to be send only and not receive?

I am using Outlook 2010 and Windows 7.  I would like to set up an email account that will send only and not receive emails.  (The account receives emails on a different computer.  I will be receiving and sending from this email address on a separate computer.)

 

How do I set up my email account to do this?  I cannot delete the incoming server in the account set-up.

jeeped say:
Go to Send/Receive, Send/Receive Goups, , Define Send/Receive Goups. With the Send/Receive Goups dialog open, select All Accounts and click Edit. With the new Define Send/Receive Settings dialog, select the account from the list down the left and uncheck Receive mail items from the Account Options.

Another option would be to open the account configuration through File, Account Settings and double-click on the account in question. On the Change Account dialog, click on More Settings and then in the new Internet E-mail Settings dialog, go to the Advanced tab and check Leave a copy of messages on the server. This will allow the messages to be downloaded but the emails will remain on the server for the other computer to receive as well.
