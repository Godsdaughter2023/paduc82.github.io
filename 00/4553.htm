jllatham say:

You could do it this way, and you only need code in the ThisWorkbook code module.  An explanation of CodeName aspect of referencing a worksheet is covered in:

http://www.jlathamsite.com/Teach/VBA/WritingBulletProofCode.pdf

Starting on page 2, with specifics on page 3.


Private Sub Workbook_BeforePrint(Cancel As Boolean)
'Shee1 and Sheet2 refer to the CodeName of the two
'sheets involved, not their tab names.
'R1 is just an available cell I chose at random.
'the way I am doing this with the
'Workbook_SheetActivate() process, we only need to
'put the flag on one of the 2 sheets and test it later.
  If MsgBox("Do you wish to delete data on sheets " _
   & Sheet1.Name & " and " & Sheet2.Name & " next time they are selected?", _
   vbYesNo + vbQuestion, "Prepare For Cleanup") = vbYes Then
    Sheet1.Range("R1") = True
  End If
End Sub

 

Private Sub Workbook_SheetActivate(ByVal Sh As Object)
'Sh refers to any sheet in the workbook when it is activated
'we take this opportunity to examine R1 on just one of the 2 special
'sheets and clean them up if they need to be.  This way
'we don't have to wait for either of those to to
'specifically be selected/activated -
  If Sheet1.Range("R1") = True Then
    Sheet1.Range("B17:K76").ClearContents
    Sheet2.Range("B17:K76").ClearContents
    Sheet1.Range("R1").ClearContents
  End If
End Sub
