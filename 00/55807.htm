Please be informed that playback of .mid/.midi files are provided by DirectX and the Sound card and not the player. You will want to make sure you have the latest version of Direct X as well as the updated drivers for your sound card first up.

How to install the latest version of DirectX: https://support.microsoft.com/en-us/kb/179113?wa=wsignin1.0

Also, refer to the below Microsoft help article on How to Install and Update drivers in Windows 10

http://answers.microsoft.com/en-us/insider/wiki/insider_wintp-insider_devices/how-to-install-and-update-drivers-in-windows-10/a97bbbd1-9973-4d66-9a5b-291300006293

Additionally, MIDI files are data and require a MIDI Synth to be able to play them and the appropriate CODEC. The Microsoft GS Wavetable MIDI synth is included with Windows. So the issue might also occur if the correct Codec is either missing or unavailable.

You can refer to following article for more information regarding this:

Codecs: frequently asked questions

http://windows.microsoft.com/en-us/windows/codecs-frequently-asked-questions#codecs-frequently-asked-questions=windows-8

Method 1

Follow the below steps to check if the sound settings are configured properly.

1. Try to play a different MIDI file from a different source to see if it is an issue of a corrupt .midi file.

2. Let the file play briefly to see if you get any sound, sometimes there will be a brief silence with header information at the beginning of the .midi file which will play as silence.

3. Locate the speaker icon in the right bottom notification area in task bar. Make sure that the volume slider for MIDI/SW Synth is not on mute and turn it up.

Method 2

Follow these steps to run the Windows in built troubleshooter:

a)      Write Troubleshooting in the search box and hit enter.

c)      Click on view all option on the upper left corner.

e)      Click Next to run the Troubleshooter.
