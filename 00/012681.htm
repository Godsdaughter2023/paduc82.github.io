WSL is a collection of components that enables native Linux ELF64 binaries to run on Windows. It contains both user mode and kernel mode components. It is primarily comprised of:

  2. Pico provider drivers (lxss.sys, lxcore.sys) that emulate a Linux kernel by translating Linux syscalls

a. user say:

Hi Bill,

You can duplex print in Outlook, depending on the printer that you are using. If your printer supports it, you can duplex print your file by enabling it on your printer properties. Most printers show it as 2-sided/Booklet. To enable it, kindly follow these steps:

  1. Launch Outlook.
  2. Click on File, then click on Print.
  3. Click on Print Options, then click on Properties.
  4. In the Basic tab, make sure that 2-sided/Booklet is set to 2-sided.
  5. Click OK.

Don't hesitate to reach out to us if you need further assistance with your concern.
