Saved pages do not display images when viewed offline. - Microsoft Community

anon7861 say:

Saved pages do not display images when viewed offline.

Why is it that when I have saved pages from the internet on my hard drive, when I view the saved pages offline only the text and not the images are displayed?  In place of the images, all one sees are blank boxes.  Even if the pages are e-mailed, only the text and not the images are displayed when viewed offline.  In both cases, only if I connect to the internet then both the text and images appear.  Is there any way to overcome this?

a. user say:

Hi,

 

This behavior can occur because images and other documents are not saved correctly if the Web address (URL) associated with the document is dynamically constructed by either JavaScript or VBScript within the Web page. Internet Explorer does not parse the script in a Web page when saving the Web page, and so it is unable to determine when an image or document URL is constructed from within a script.

 

To work around this behavior, save the Web page locally by clicking Add to Favorites on the Favorites menu, and then clicking the Make available offline check box to select it.

 

Hope this helps.

Styles: Footnote versus Footnote Reference - Microsoft Community

mcamz say:

Styles: Footnote versus Footnote Reference

Hi,

Can anybody please explain the difference between the style "footnote" and the style "footnote text".  I am adding footnotes to an existing document and when I insert a footnote the style is "footnote text".   However the existing footnotes have the style of "footnote". 

Thank you

Michelle

john korchok say:
"Footnote" may be a user-created style invented by the creator or a subsequent user of your document. Word creates 2 styles for footnotes: Footnote Reference for the styling of the superscript character and Footnote Text for the actual text of the footnote. You'll always get better results with automatically formatted Word text (like footnotes), if you use the automatically generated styles. Anything else involves a lot of manual labor.
