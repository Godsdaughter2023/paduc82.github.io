I don't know why Print Preview wouldn't show what will actually print, but I would advise against trying to align columns (even of numbers) using spaces and built-in tab stops. Even if you're using a monospaced font, this is sloppy formatting for a Word document. At the very least, set custom tab stops, but using a table is often simpler.

You can use Find and Replace to delete all the extra tab characters and spaces, leaving only one tab between intended columns. Then select all the text and set custom tab stops (you may need to use right-aligned or decimal-aligned). Once you've done that, you can easily convert the text to a table if desired (using Insert | Table | Convert Text to Table, separating at Tabs).

Slide duration in a slide show - Microsoft Community

jgltx say:

Slide duration in a slide show

I'm a first time Power Point user.  I am creating the slides OK, but once complete, how do I make the show run itsself with each slide showing for a specific time- say 30 seconds?  And then automatically start again so the show keeps showing until I stop it?

John

steve rindsberg say:

This is simplest to do from Slide Sorter view, so start by going there.  View | Slide Sorter.

Press CTRL+A to select all of your slides.

Go to the TRANSITIONS tab of the ribbon.

Choose the transition you'd like to apply to your slides (or None, if you prefer).

In the Timing section of the tab,  under Advance Slide, put a check mark next to AFTER and change the value next to it to 00:30.00 (30 seconds).  You'll see the time you've set under each of the slide thumbnails.

Next, go to the SLIDE SHOW tab of the ribbon.

Click SET UP SLIDE SHOW

Under Show Type, choose "Presented by a speaker (full screen)"

Under Show Options, put a check next to "Loop continuously until ESC"

Click OK.

That should about do it.
