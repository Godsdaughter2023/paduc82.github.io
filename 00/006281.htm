The IE Trusted zone, is mostly used by large company networks for B2B (business to business) linking of non-public (closed) networks.

A General Starting point when Word does something unexpected

As a general rule, when something seems to be going wrong, or at least not the way you want it to go, in Word “menu’ go to Tools and then Options, in Word “ribbon” go to File then Options, and study all of the various options, both turned on and off.

 

You may even get lucky and find some other settings that you want to try that have nothing to do with the current problem.

 

When I was teaching Word and other Microsoft Office programs at a private business college, I had three rules:

1.      Always check Tools Options

2.      When in doubt, right-click on something

3.      If you're wondering how to set up a special kind of material in a document, consider whether Word tables are suitable. The placement officer of the school came to me several times and asked how I thought she should set something up. Each time the answer was, "I'd use a Word table." EXCEL 2007

Simple example:-

Type the first three dates in cells A1, A2 and A3.

Highlight those three dates then hover the mouse over / towards the lower right hand corner of the range until the mouse pointer changed to a cross.

When the mouse pointer changes to a cross depress the mouse button and keep it depressed.

Release the mouse button when you want to stop the insertion of dates.

however when I use CTRL+F to search for the keyword and then clear the keyword from search box and close the search menu upon pressing CTRL+Page UP word takes me to the next keyword instead of scrolling to the next page. so CTRL+Page UP / Page Down is used as Find Previous / Find Next command instead of fulfilling page scrolling actions. 

I have cleared the keyword from search and from advanced search but it's still looking for that keyword when i press CTRL+Page Up/ Page Down.

suzanne s. barnhill say:
In Word 2010 and earlier, there are "browse arrows" at the bottom of the vertical scroll bar. They execute the commands Browse Previous and Browse Next, which have the keyboard shortcuts Ctrl+Page Up and Ctrl+Page Down. By default, they have the function Previous Page and Next Page, but using Find or Replace resets the "browse object" to Previous/Next Find/Go To. This can be changed using the Select Browse Object button between the arrow buttons.

Word 2013 does not have the browse arrows or any way to select the browse object, so once you've used search, you're stuck. Fortunately, however, a comment on the tip at http://wordribbon.tips.net/T010079_Using_the_Object_Browser.html provides a macro to change the browse object. Alternatively, as another user has commented, you can add Go To Next Page and Go To Previous Page buttons to the QAT; using either resets the browse object back to Page.
