    greg carmack - mvp 2010-2020 say:

    Almost all modern SATA controller drivers required to read the hard drive during install are in the installation media, so first we confirm the media is good since it can create a false positive error here. 

    Normally the driver to supply in the very rare case no hard drive shows up during install is the SATA controller from the motherboard's chipset drivers.  But with Intel Optane memory it has it's own  SATA controller hence the driver most likely needs to come from Intel.  

    Since Intel Optane is quite new it's possible it's SATA controller drivers are not in the Windows media.  In cases like this we need to locate the SATA controller driver, extract it to a flash stick and provide it at the link during install drive selection.
