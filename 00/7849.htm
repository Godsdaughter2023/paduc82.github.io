jack twist say:

How to enable the automatic backup option for Excel files

Instructions for enabling this valuable Excel option are well-hidden in both Excel for Mac itself and the help forums.  The basic “Help Center” doesn’t seem to mention it at all, and searches of this forum yield only hits to questions that seem at best only tangentially related to the auto-backup feature.

Thanks to Phillip Jones, one of the Community Stars, for providing the necessary sequence of commands, in the course of his response to one of those tangentially (barely) related questions:
  1. In the main Excel menu bar, select "File > Save as …"
  2. Click on the “Options” button near the lower-left corner of the resulting dialog box.
  3. Click the “Always create backup” check-box in the “Save Options” dialog box.
  4. Select “OK.”
Maybe posting the feature here as a separate question will make it easier to find and spare others the long and frustrating search for help that I experienced.  (BTW, in the Windows version of Excel 2007, the procedure for enabling this feature is very similar; the main difference is that the Windows “Office” button takes the place of the “Files” menu.  But in stark contrast to Excel for Mac’s help resources, instructions for this feature are extremely easy to find in the basic built-in Help feature of Excel for Windows.  It seems Windows does do some things better than Mac, after all!)

oemsoft say:

How to concate the cells together?

I would like to concate the cells under B column, but following code does not work,

Do I have to concatenate(B1,B2, ..., B100) in this way?

Is there any simple approach to concate a list of cells together?

hansv mvp say:

The CONCATENATE function is not very user-friendly.

You could use a custom VBA function:

 

Function Concat(rng As Range, Optional sep As String = ",") As String
    Dim rngCell As Range
    Dim strResult As String
    For Each rngCell In rng
        If rngCell.Value <> "" Then
            strResult = strResult & sep & rngCell.Value
        End If
    Next rngCell
    If strResult <> "" Then
        strResult = Mid(strResult, Len(sep) + 1)
    End If
    Concat = strResult
End Function

 

Use it like this:

 

=Concat(B1:B100,"")

 

If you prefer to separate the values by spaces:

 

=Concat(B1:B100," ")

 

and if you prefer to separate them by commas:

 

=Concat(B1:B100)

roy beaulieu say: You really can't. The way Skype has worked from day one is, your chat history is stored with your account info such as your contacts. When your Skype client logs into Microsoft, all your contacts and their associated chat are also downloaded to your device. You can delete chat history locally to remove it from your screen but, as soon as you wipe the local database, the client will redownload all contacts and chat history locally again. Hi Brian,

F4 is the cycle through all combinations of absolute and relative references in a formula if a cell reference or range is selected. If it's no longer working, it's possible that the Excel app is outdated.

To help you resolve your concern, kindly follow these steps:

  2. Follow the steps on the Repair an Office application article. It can automatically repair errors in Excel files.
  3. Update you Excel app. Go to File > Account > Update Options > Update Now. Installing any important recommendation and optimal updates can often correct problems by replacing out-of-date files and fixing vulnerabilities.

wildertiger68 say:

Hi Brian, 

I was having the same problem using a new Microsoft Sculpt Comfort Keyboard and couldn't figure out why F4 was no longer toggling cell references.  What I finally realized was that the switch above the Num Lock key next to the battery icon is not a power switch, it is the Function Lock.  With the switch in the up position locks F4 into Next Desktop command for virtual desktops. In the down position, you'll get the normal F4 functionality.

Hope this helps.
