Protocol: POP3

You can't connect to an IMAP server using the POP3 protocol.

Right-click on the account name in the folder pane and select Remove account. When it's gone, press Ctrl-Shift-T to start the Add your email accounts wizard. Fill in the first page and select Manually configure server settings, then click on Next.

Select IMAP from the dropdown on the next page, then complete the form with the same information as before:

  • Incoming Mail Server Name: imap.comcast.net
  • Incoming Mail Server Port Number: 993
  • Outgoing Mail Server Name: smtp.comcast.net
  • Outgoing Mail Server Port Number: 465
  • SSL Encryption: checked
  • Authentication: checked

Finish the wizard, then test the account.
