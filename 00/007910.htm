For example, I want to look up the number 100 in A1:A10, and when that number is located, I want it to return text from B1 ("hello"). So if A6 is 100, B1 will return "hello", but if none of A1:A10 is 100 it will return "goodbye" or nothing at all ideally. Any ideas? I appreciate the help I've been given by you generous experts on this forum.
jeeped say:
Try something like,

=IFERROR(VLOOKUP(100, $A1:$B10, 2, FALSE), "Goodbye")

If you require looking up a second instance of 100, then another formula (possibly an array formula) would be more correct.

Edit: minor correction to the lookup_range parameter.

trip_to_tokyo say:

1. In cell K1 (for example) enter the following formula:-

 

=IF(A1=100,$B$1,"")

 

Copy the above formula to (for example) cells:-

 

K2:K10

 

2. In cell B1 I have:-

 

hello

 

 - because you state:-

 

I want it to return text from B1 ("hello").

 

3. If you now type in:-

 

100

 

- into any of the cells in range:-

 

A1:A10

 

 - you will get:-

 

hello

 

 - in the corresponding cell in the range:-

 

K1:K10

 

4. If any of cells:-

 

A1: A10

 

 - does not contain:-

 

100

 

 - the corresponding cell in range:-

 

K1:K10

 

 - will be blank.

 

As far as I can see the above matches your requirement.
