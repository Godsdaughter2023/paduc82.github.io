Having set the row height and the column width for the hundred columns appropriately, try running the following code:


'=========>>
Option Explicit

'--------->>
Public Sub AddFormsComboBoxes()
    Dim WB As Workbook
    Dim SH As Worksheet
    Dim Rng As Range, rCell As Range
    Dim myDropDown As DropDown
    Dim dTop As Double, dLeft As Double
    Dim dWidth As Double, dHeight As Double

    Set WB = ThisWorkbook
    Set SH = WB.Sheets("Sheet1")                                       '<<=== Modify
    Set Rng = SH.Range("D2").Resize(1, 100)
    For Each rCell In Rng.Cells
        With rCell
            dTop = .Top
            dLeft = .Left
            dWidth = .Width - 2
            dHeight = .Height - 2
            Set myDropDown = SH.DropDowns. _
                                    Add(dLeft, dTop, dWidth, dHeight)
            myDropDown.LinkedCell = .Offset(1).Address
        End With
    Next rCell
End Sub
'<<=========

This code creates all of the 100 ComboBoxes, including the first one over cell D2 and sets the width and height of the ComboBoxes to accord with the pre-set column  widths and row height.

Note:  Before making the changes below, make a copy of the Excel file, just in case you make a mistake and need to revert to the original file.

There is nothing wrong with the form of the numeric text; in particular, the dollar signs ("$") do not cause any problems.

However, the first problem with my original TRIM/SUBSTITUTE formula is:  it does not account for the possibility that the result is the null string.  Ostensibly, the fix is:

1. Select K1, and enter the following formula:

=IFERROR(--TRIM(SUBSTITUTE(B1,CHAR(160),"")),"")

2. Copy K1 (press ctrl+C), select K1:R19, and paste (press ctrl+V). [1]

3. With K1:K19 still selected, copy (press ctrl+C), select B1:I19, and paste-value (right-click, click Paste Special, select Values, and click OK). [2]

4. Now you can clear K1:R19.

Note that =SUM(B1:I19) now returns the expected result, 816.61.

However, cells that appear to be empty now contain invisible text:  the null string.  That might cause problems later.  For example, ISNUMBER(B1) and ISBLANK(B1) return FALSE; and =B1+D1 returns #VALUE.

You can remedy that by selecting B1:B19, click Data, Text To Columns, Finish.  Repeat for each column of data, namely C1:C19, D1:D19, etc.  Unfortunately, TTC works on only one column at a time.

Alternatively, do the following:

5. Select B1:I19.

6. Right click the worksheet tab at the bottom, and click View Code.  That opens the VBA window.

7. In the VBA window, click Insert, Module.  Copy the following and paste into the VBA editor pane on the right:

Sub doit()

Selection.Value = Selection.Value

End Sub

8. Close the VBA window.

9. In Excel, press alt+F8, select the new macro ("doit"), and click Run.

10. Save and re-open the modified Excel file.  If you save to XLSX instead of XLSM, you will get a warning about the lost macro.  That is okay; we no longer need it.  Click OK.

gord dibben say:

Here are a couple of macros for you.

Sub FlipRowstoptobottom()
    Dim vTop As Variant
    Dim vEnd As Variant
    Dim iStart As Integer
    Dim iEnd As Integer
    Application.ScreenUpdating = False
    iStart = 1
    iEnd = Selection.Rows.Count
    Do While iStart < iEnd
        vTop = Selection.Rows(iStart)
        vEnd = Selection.Rows(iEnd)
        Selection.Rows(iEnd) = vTop
        Selection.Rows(iStart) = vEnd
        iStart = iStart + 1
        iEnd = iEnd - 1
    Loop
    Application.ScreenUpdating = True
End Sub

Sub Flipcolumnslefttoright()
    Dim vTop As Variant
    Dim vEnd As Variant
    Dim iStart As Integer
    Dim iEnd As Integer
    Application.ScreenUpdating = False
    iStart = 1
    iEnd = Selection.Columns.Count
    Do While iStart < iEnd
        vTop = Selection.Columns(iStart)
        vEnd = Selection.Columns(iEnd)
        Selection.Columns(iEnd) = vTop
        Selection.Columns(iStart) = vEnd
        iStart = iStart + 1
        iEnd = iEnd - 1
    Loop
    Application.ScreenUpdating = True
End Sub

TRANSFORM Count(qaoc_Action.[Action_ID]) AS CountOfAction_ID
