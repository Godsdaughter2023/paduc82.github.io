msft rudi r say:

How do Time allowance and Curfew work?

With Family Safety parents can control time ranges during which a child can logon and use a Windows PC. Also parents have the ability to control amount of time a child can spend on a computer during a day within the allowed time ranges.

msft rudi r say:

Time Limits functionality in Family Safety has changed recently. Time allowance settings is only available for Windows 8 PC’s. As an example, assume that you have one Windows 7 PC and one Windows 8 PC. Both PC’s are set up and using Family Safety. You log on to the Family Safety website, select a user to edit and then click the “Time Limits” option in the left navigation panel. You will see two tabs – one for allowance and one for curfew.  Allowance and curfew is available for Windows 8 PC’s. When editing Family Safety settings for Windows 7 PC’s , the Allowance option is not available but you can edit the Curfew settings except for the 30 minute increment aspect, this is not available for Windows 7 PC’s. Changing any settings on the Family Safety website will change the setting for all devices associated with that child account.

During the setup procedure, you are asked on which partition to install Windows 7. If there is no possible partition to choose from, possibly the hard disk is not recognized by Windows. The most common cause is an incorrect SATA controller or RAID configuration.

Some motherboard BIOS's have a feature to load a SATA hard disk as an IDE hard disk. Search the BIOS settings for an option like Native SATA support/AHCI (Advanced Host Controller Interface), and disable it. Although this setting works fine, it may be better to install the original drivers for the SATA controller afterwards followed by restoring this setting.

How problem started; Read my original post above on top. In short I set the security setting to Deny for

Windows/users in security tab which is reached after right clicking drive then properties > security. I set this and lost access to all drives!

How I fixed it.

Thanks to some nice people on this community page who have posted knowledge. One man had posted how to create "Administrator' account from command prompt which I did so now I could log out of my account and get on to this "Administrator" account with admin privileges so that I could fix the mess on my problem account. Trick was to take the mouse to the right of the screen and get that blue bar..then Settings> Change PC Settings > General - Advanced startup- press on button -<Restart Now>. Then it brings up blue screen where you have option called troubleshooting..go there then again advanced setting> look for booting to windows from command prompt. Get there you will have a command prompt with C:\windows\system32 on the prompt. There my command were accepted be it icacls or takeown. so I fired a command there: TAKEOWN /F C:\  /R /A  and I also tried icacls to give permissions after checking the syntax help on command line itself. All the command were executing successfully this time but be careful of what you give in the command. This is under some high privilege built in account feature of Windows 8.

Takeown command executed successfully and it solved my problem. I exit the command prompt then logged on to this Administrator account. This time I could get on to the security tab of C drive and set the Allow for users there.
