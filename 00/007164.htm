Use a column of formulas - in the column after Saturday. Since you don't show columns or row labels, for this example, let's say that Robert is on Row 10 and that Names are in column C and Saturday is column J.  So, in K9 Enter the fruit of interest (grapes) and in K10, enter the formula

and copy down to match your list. Then apply a filter to column K, show TRUE only, and the rows that contain grapes will still be visible.

Change to

=IF(COUNTIF(D10:J10,$J$9)>0,ROW(),"")

and then use this in another cell:

=IFERROR(INDEX(C:C,SMALL(K:K,ROW(A1))),"")

and copy that down until it returns blanks. When the value in J9 is changed, so will the list

Hi,

 

Try this ARRAY formula, see below on how to enter it.

 

Put it in a cell and ARRAY enter it and drag down as far as you want. It will return blanks when it runs out of matches.

pamcaswell say:

Put the cursor on the auto  number for the heading. Right click and choose Set numbering value > Continue from previous list > Advance value (skip numbers).  Set the values for all four numbering levels.  If you have nonprinting formatting characters showing, you'll see two heading 1s, two heading 2s, and two heading 3s before your heading 4.  They are all actually hidden--there's no text so the dotted underscores aren't there.  Do not delete them. When you turn off the formatting characters, you'll just see your heading number.  

If you  join this file to the main document, remember to reset the numbering to continuous.

Pam
