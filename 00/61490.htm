rajesh govind say:

 

Hi,

 

 

It seems that your apartment provides a “Public” WiFi service named “Internet” and such networks usually do not require a network key/password to get connected to it. The computer gets connected to it automatically as soon as it is detected.

 

When using such “Public” networks, the only thing that you need to be concerned is the safety of the computer. You should make sure that the computer is protected from unauthorized access or hacking and malicious software that may harm the computer.

 

So, you need to have a good Internet Security program on your computer. Windows 8 has Windows Firewall and Windows Defender anti-virus program inbuilt, so you need not worry about getting a good Internet Security program for your computer. You just need to make sure that both Windows Firewall and Windows Defender are working properly and that the computer is up-to-date.

 

The “Limited” connectivity issue with your other connection is because your PC is not assigned with a valid IP address and so you can't actually get to the Internet. To resolve this issue, you may refer to the section: “My connection is listed as “Limited”” from the article: “Why can't I connect to the Internet?” given in this link:

 

http://windows.microsoft.com/en-us/windows-8/cant-connect-to-internet

hairyfool say:

The Desktop\Marketing Files folder is going to be a different location to the Libraries\Documents\Marketing Files.

It can be a little confusing because of the mixture of hard and soft links. Also be careful about how you access the Library folders as they can be moved. By default they would be in C:\Users\username but the safer way is to view the Profile folder list.

Click on the Start button and then the User name on the top right. This opens a Window with all the profile folders in one view. If you then right click on the Documents folder and view Properties  the General tab shows the Physical location. However you will also see a Location tab which allows you to move the folder anywhere you want. The benefit of using this way to view the folder is even if you then move the folder to another drive for example the Profile view will still list the folder where C:\Users\username will not.

A Library folder can also contain folders from other parts of the drive. Click on the Library Icon then right click on the Documents Icon and you can Add the Desktop\Marketing Files directory to the Documents Library.
