jay freedman say:

There are two possibilities: one that's easy for you but not obvious for users, and one that's a miserable pain for you and obvious for users.

The former: Explain to users (maybe in a text box on the document) that they can press Alt+Left Arrow to return to the hyperlink -- this is the same as the shortcut that's common in web browsers, and in fact Word's name for the command is WebGoBack. They can also add a Back button to their Quick Access Toolbar (in the Options > Quick Access Toolbar dialog, the Back command is in the All Commands list).

The latter: Place a bookmark next to each hyperlink, and place another hyperlink next to the image that points back to that bookmark. That will be a lot of work, and may need to be revised if you edit the document.
