derekiwi say:

I know this thread was a long time ago, but for the benefit of anyone coming here searching for an answer (as I have):

If you start Windows from a recovery/rescue disk, and then start a command window, you will get this funny looking drive labeling. The X: drive is actually a ram drive created by the rescue software to boot and run from, which is why that is where the prompt is. The rescue disk system creates this ram drive, copies the files to up to make it bootable, and then starts from there.

Also, there is a difference between the physical drive letters and the logical drive letters. I believe this is because of the SATA drive system, which means you can plug a drive in on any connection. When Windows is running normally, it converts the drive physical location to C:\.

Upshot is - if you  go D: and then dir, and keep trying the various drive letters, you will eventually find the Windows boot drive.

If you have a Lexmark printer , these additional steps may be necessary (thanks to Robert Orleth [MSFT] for providing this information).  The Lexmarkprinter installation process sometimes installs a service that makes theprint spooler service dependent on itself.  If there is a problem with Lexmarkservice or a Lexmark printer driver (or you removed it using the steps above), theprint spooler service may not start (see also http://support.microsoft.com/default.aspx?scid=kb;en-us;324757 ).  The steps below make theprint spooler service only dependent on the Remote Procedure Call (RPC) service (RPCSS), which is normal.  Theprint spooler service dependencies are stored in the registry at

  2. key the command

            (note the space after the = but not before)
