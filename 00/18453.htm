2) When the folder opens, make backup copy of Normal.dotm, and save it in the default location, in case you want to remember the old style and return to it later.

3) Right-click on Normal.dotm. Choose Open. This will open default template Normal.dotm for editing in Word.

4) Select the first and only character in the template by pressing CTRL-a to Select all. On the Home Tab, change the Font in the Ribbon to Courier New and 10 points.

5) Save and exit Normal.dotm. Again, be advised: using this method, all new documents will open in Courier New, font size 10 points. However, it remains to be seen whether your existing text files will open in this new style using this method.

Yes, I agree completely ... and I FINALLY dug deep enough in Word 2010 to FIND the Plain Text style, modify it, and add it to the Quick Style list in the Styles Quick Access Toolbar in the Ribbon of the Home tab:

1) Open Word 2010. With the Ribbon for the Home tab displayed, click the tiny angular down arrow in the right-hand bottom corner of the Styles Quick Access Toolbar, below Change Styles. This pops out and displays the Styles window.

2) Click on the blue Options ... link in the right-hand bottom corner. The Style Pane Options window is displayed.

3) Click on the top drop-down list, Select styles to show, and select All styles. If you want this choice to be applied to all documents going forward opened with the Normal.dotm template (and I'm guessing that you do), select the New documents based on this template radio button in the right-hand bottom corner.

4) Now, ALL styles in Word 2010 are visible in the Styles window, and should be visible there going forward. Scroll down to Plain Text, and hover the cursor over it. You will see the current definition of this style. Among other choices, the defaults for the Plain Text typeface and size are apparently Consolas, and 10.5 pt.

5) Right-click on Plain Text. Select Modify in the context menu. The Modify Style window is displayed.

6) The combination of choices in this window make up the current definition of the Plain Text style, displayed at the bottom of the Formatting section. If you've never modified this style before, it likely says:

Line spacing:  single, Space
After:  0 pt, Style: Linked, Hide until used, Priority: 100
Based on: Normal

Notice that the style is based on the Normal (Normal.dotm) template.

7) In the Formatting section, choose the typeface and size you desire (Courier New and 10 pt if you want the defaults from earlier versions of Word).

8) Be sure to check the Add to Quick Style list checkbox, and select the New documents based on this template radio button at the bottom of the Modify Style window, if you want these changes to be applied to all documents going forward!

9) Choose OK, and click on the x in the top right-hand corner of the Styles window. Scroll down the Quick Style list in the Styles Quick Access Toolbar in the Ribbon, to see the Plain Text style now displayed there. Close word 2010.

10) Open Word 2010. Scroll down the Quick Style list in the Styles Quick Access Toolbar in the Ribbon, to verify that the Plain Text style remains displayed there for all documents going forward.

11) Right-click on Plain Text and select Modify. Verify that the changes to the typeface you made are there for all documents going forward. Select Cancel. Close Word 2010.

12) Enjoy and make good use of your new Plain Text style! As the man said, all text files opened in Word 2010 should now be displayed in Courier New, 10 pt. You can easily repeat these steps to use whatever choice of typeface and size that you require or desire.
