petrib say: Memory cards by default have no passwords at all set.

If you set the password, then it will have no impact on the device you are using the card in (except if you forget it and reformat/reset the device).

The card password is there to protect the card contents from being used in some other device (and when the person having the card does not know the password). And the case of reformat/reset is essentially just as if you had tried to use the card in a different device (because the device no has forgotten about the card and password).

So, if you set the password, then make sure you remember it, and disable it before you do a firmware update (just in case).

A backup before reformat/reset/firmware update, and a restore afterwards might recover the password, too, but I've never tested this (as I don't set the memory card password on any of my phones).

And one fundamental problem with this is, that anyone who gets access to the phone can set the memory card password, so somebody could do it "accidentally" by playing around the phone, or deliberately, and if they don't tell the owner, the owner will never know (until they reformat/reflash/upgrade the device, or try to use the card in some other device).

steve rindsberg say:

Minor quibble:  PNG isn't uncompressed, it's LOSSLESSLY compressed; the compression doesn't alter the image data that you see when you view the image.

JPG uses a kind of compression that tosses out or alters the image data to make it more compressible.  Or can.  IIRC, PPT by default uses the highest quality compression, meaning bigger files but better image quality.  As a result, there may not be a great deal of difference between its PNGs and JPGs.

john korchok say:
Indexed-color PNGs are the format of choice for logos and any artwork with areas of flat color. JPGs are better for photos, as they can compress to a smaller size while still looking good.

steve rindsberg say:
