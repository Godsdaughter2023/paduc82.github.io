If the issue persists, follow this:

Disable hardware acceleration. This has helped fix similar issue. 
1.      Press Windows key + R, type regedit and click OK.
2.      Browse to HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Common
3.      Create a New Key and name it "Graphics"
4.      Select Graphics, right-click on the right panel and create a New DWORD (32-bit) Value and name it DisableHardwareAcceleration.
5.      Enter Value data as 1.
