augustine alphypapali say:

SMALL CAPS

I have a word in lowercase. I can change it into SMALL CAPS. But, when I have a word all in uppercase, the SMALL CPAS format does not work. Is that how it is supposed to work?

greg maxey say:

That is the way it works. 

 

SmallCaps is a font property which you can apply to any text but its effects are only manifest in lower case text.  AllCaps is a similar property.

 

For example, you can type "TEST" select it and apply the following VBA procedure?

 

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
MsgBox CBool(Selection.Font.AllCaps) 'Returns False
Selection.Font.AllCaps = False
'It is already false so making it so explicitly doesn't change the font appearance.

'Now:
MsgBox CBool(Selection.Font.SmallCaps) 'Returns False
Selection.Font.SmallCaps = True
'We've applied the property but it doesn't appliy to Upper Case letters.

'However, change the font to lower case:
Selection.Range.Case = wdLowerCase
'Add the propery now applies'
End Sub

 

Similarly you can select a word typed with capital letters, select it and apply smallcaps then toggle the case to lower case by pressing Shift+F3

 

See: http://gregmaxey.mvps.org/word_tip_pages/installing_employing_macros.html for instructions to employ the VBA code provided above.
