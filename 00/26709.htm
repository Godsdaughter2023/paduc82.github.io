joel in vic say:

IF you want to go to Windows 10 Mobile now, it is possible.  I'd recommend resetting the phone back to bone stock WP8.1, preferably using the Windows Device Recovery Tool. 

The goal here is as clean an install as possible with minimal amounts of data to be migrated between WP8.1 and W10M.

Once the phone is reset:

  1. Log in with your Microsoft Account.
  2. Do NOT choose "restore from backup" if it is offered to you
  3. Allow your account to sync
  4. open the store and check for downloads, allow all apps to fully install
  5. restart the phone
  6. Settings --> check for OS updates
  7. check store again for app updates
  8. Install Windows Insider app
  9. Open Insider and choose Production Preview
  10. Settings --> check for updates
  11. Play your favourite elevator music while the gears turn
  12. Settings --> check for OS updates
  13. more gears if necessary
  14. open the store and check for downloads, allow all apps to fully install
  15. Store --> hamburger menu --> My Library --> reinstall all the apps you want back on the phone
