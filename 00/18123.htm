For a wave file, you should be able to use the code below.  You would have to put this code into a standard module, and then call the PlaySound function when you want to play your .wav file.

'------ start of module code ------

Option Explicit

Option Compare Database

Private Declare Function apiPlaySound Lib "Winmm.dll" Alias "sndPlaySoundA" _
    (ByVal lpszSoundName As String, ByVal uFlags As Long) As Long

Function PlaySound(strSoundFile As String) As Long

    ' Argument <strSoundFile> is the full path and name of a .wav file to be played.

    Dim lngRetCode As Long
    
    lngRetCode = apiPlaySound(strSoundFile, 1)

    If lngRetCode = 0 Then MsgBox "The Sound Did Not Play!"   ' comment out this line if you'll do error-handling elsewhere

    Return lngRetCode    

End Function
'------ end of module code ------

For example,

    PlaySound "C:\Windows\Media\tada.wav"
