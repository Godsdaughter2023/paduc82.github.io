pamcaswell say: The italics could have been set in the table style  (not the table text paragraph style) or from an italic setting in the normal paragraph style.

First check normal. If the italic button is selected in the style settings, click to deselect it.  That should take care of the italic end of row markers and unwanted italics in other places in the document. 

To check the table style settings, place the cursor in a table and go to the Table > Design tab. In the table styles gallery, the one with the orange halo is the table's parent style (usually Table Grid).  Right click the table style and left click Modify.  In the dialog, deselect  the italic icon, if necessary.   That will take care of italic formatting in the table and its end of row markers--as long as it is not in the normal paragraph style.

The takeaway from this is that the paragraph style settings you want in your regular text paragraphs (whether or not you are using normal) should be also set the the defaults (Styles pane > Manage Styles button > Set Defaults tab) and in normal.

Note that if we use Word the way the programmers intend--by changing the look of our documents by applying built-in and properly made custom style sets--we would not have these problems, because a style set includes default settings, so (a not manually changed) normal and the defaults would be the same.

pamcaswell say: I've never heard of "template layering".  A document  has one document template at a time.  After creation, the styles in a document and its template are independent unless manually done by updating document styles from template or by sending changes made to a documetnt style to its template ( by clicking New documents based on this template in the Modify styles dialog).

To get your style settings into another document, you can 1) attach your template to it and turn on Automatically update document styles in the Templates and Add-ons dialog (and  immediately go back into the dialog and turn it off) or 2) create a new document based on your template and paste all but the last paragraph mark of your source document into it. A regular paste is OK.  Pasting as unformatted text is unnecessary, particularly if you only need to change the style settings.


Note that the the problem of normal overriding the table formatting and conflicts with the defaults have been  in Word since W2002.  The changes  in W2007-13, fix those problems, but we have to change  our habits to take advantage of the improvements.
