doug robbins - mvp office apps & services (word) say:

The following is designed to be run from Word and will insert the table at the bookmark named "target" where the data is in the range of cells starting from cell A1 on the first worksheet in the selected workbook

Dim xlapp As Object
Dim xlbook As Object
Dim xlsheet As Object
Dim myarray As Variant
Dim i As Long, j As Long
Dim rownum As Long, colnum As Long
Dim tblTarget As Table
Dim fd As FileDialog
Dim strworkbook As String
Set fd = Application.FileDialog(msoFileDialogFilePicker)
With fd
   .Filters.Add "Workbooks", "*.xls; *.xlsx", 1
   .AllowMultiSelect = False
   If .Show = -1 Then
       strworkbook = .SelectedItems(1)
   End If
End With
On Error Resume Next
Set xlapp = GetObject(, "Excel.Application")
If Err Then
   bstartApp = True
   Set xlapp = CreateObject("Excel.Application")
End If
On Error GoTo 0
Set xlbook = xlapp.Workbooks.Open(strworkbook)
Set xlsheet = xlbook.Worksheets(1)
With xlsheet.Range("A1")
   rownum = .CurrentRegion.Rows.Count
   colnum = .CurrentRegion.Rows.Count
   myarray = .CurrentRegion.Value
End With
With ActiveDocument
   Set tblTarget = .Tables.Add(.Bookmarks("Target").Range, rownum, colnum)
End With
With tblTarget
   For i = 1 To rownum
       For j = 1 To colnum
           .Cell(i, j).Range = myarray(i, j)
       Next j
   Next i
End With

You can hardcode the source of the workbook if you do not want the user to have to select it.

Hope this helps.

There is no provision to wipe a document after a certain number of days or of viewings. Even if there were, it would be relatively easy to get around such a restriction.

General rule: If someone can see something, they can copy it. If they can copy it, they can edit it. All you can do is make it more difficult.

It would also be relatively easy to set up a document so that it would have the word "VOID" all over it if it was opened after a certain date. However, that would also be relatively easy to defeat.

Thinking about it, you could do the same thing with the content of your document, if that content was an image and not more than one page. That is, make it only display for a certain number of days. Again, however, this would be relatively easy to defeat. It might take me an hour to set up such a document but 2 minutes to defeat it. There are lots of people out there who could defeat such a protection even more quickly because the content would still be there.

Hi ArshadP and welcome to the Skype Community! 

There are a number of reasons why you are having issues with calling a mobile or a landline number. Reasons include: 

  • The number you are calling might not be covered by your subscription. If your subscription covers calls to landlines only, you wouldn't be able to call a mobile number unless you have Skype credits that will cover your calls. Check this link on how to troubleshoot Skype subscriptions. 

  • You are dialing an invalid number. Make sure that you are dialing the number correctly. Try dialing in this format: Enter a "+" sign and then the country code plus the number you want to call. 

  • If the issue is happening to a specific number, then it could be because of the fluctuations in the public switched telephone network (PSTN). We cannot always guarantee the quality or reliability of calls made from Skype. These issues are usually temporary, we suggest that you dial the number again later. 

  • Skype might not support the number you are dialing. Unfortunately, Skype does not support calls from all number rangers. There are certain premium-priced or geographic/country specific numbers which cannot be reached using Skype.

  • The connection you are using isn't stable. Calls not connecting usually indicates an issue with the Internet connection you used. Please try changing your connection and try to dial again. 

Be sure to check the above reasons why users experience issues with calling mobile/landline numbers. Let us know if the issue persists so I can send you a follow-up.
