Copy the following code into the worksheet module:

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
    ' Double-click in T3:T70?
    If Not Intersect(Range("T3:T70"), Target) Is Nothing Then

You can change Windows File Associations to make your VSDs open by default with Visio 2007.

 

To enable this, do the following.

1) Navigate to the .vsd file in Explorer

2) Hold shift, and right-click the .vsd file

3) Click on 'Open With...' option in the menu

4) Click on the 'Choose default program' and then choose Visio 2007.

 

Step #4 above is for Windows 8. It will be slightly different for Windows 7 or earlier.
