Hi sir_tonberry,

The error you are reading is about your file being too big for the E: disk; this is because you probably format it as FAT or FAT32, which doesn't allow to copy files bigger than 4GB.
You need to format your disk again as NTFS, in order to let files bigger than 4GB move on the E: disk.
You can check if your disk is a FAT or FAT32 formatted one following this procedure:

1. Open "This PC"
2. Right click on "E:"
3. Click on "Proprieties"
4. Check what's reported near file system

Please note that if you need to format your E: drive, then you will need to copy the files actually on E: to another drive, or you will lose them. To format E: drive:

1. Open "This PC"
2. Right click on "E:"
3. Click on "Format"
4. Select "NTFS" under File system option
5. Click on "Start"
