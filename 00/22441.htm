a. user say:

Hi Jorge Granja,

 

I will be glad to assist you with your concern.  This usually occurs if the email came from a third party domain. If the sender’s domain does not have SPF record configured/set up in the domain’s registrar, Hotmail or Outlook.com detects the email as potential fraud since it was forwarded from 3rd party IPs.

 

What is SPF Record?

Sender Policy Framework (SPF) is an email validation system designed to prevent email spam by detecting email spoofing, a common vulnerability, by verifying sender IP addresses. It is a security standard that has been implemented to stop forgery of sender address.

 

SPF allows administrators to specify which hosts are allowed to send mail from a given domain by creating a specific SPF record (or TXT record) in the Domain Name System (DNS).

 

To resolve this, the sender’s domain administrator should add the SPF record in the domain registrar.

don cuthbert say:

From that Command Prompt, you can run some things to check or repair Windows on the hard disk.

First check which drive letter has both Windows and Users folders on it.  Sometimes drive letters change in this mode, so Windows could be on C:, D:, or E:, etc.  Then run the below commands to check the drive and Windows. Adjust the drive letter used if yours is different:

chkdsk C: /scan

chkdsk C: /offlinescanandfix  <-- if /scan above found no errors, you can skip running this command

sfc /scannow /OFFBOOTDIR=C:\ /OFFWINDIR=C:\Windows

If sfc finds errors that it cannot fix, try running DISM.  First create a folder for temp files with md C:\Scratch (adjust drive letter as needed)

DISM /Image:C:\ /CleanupImage /RestoreHealth /ScratchDir:C:\Scratch

bootsect /nt60 SYS /force  <-- Updates the master boot code on the system partition used to boot Windows.

You can also try recopying the System files to the System partition with bcdboot.

bcdboot C:\Windows

Remember to adjust the drive letter C: in all above commands to whatever letter your Windows and Users folder appears on.  Do not use X:, as this is the RAM disk that Windows RE is running from.

Inbox priority and others - Microsoft Community

bitte♥ say:

Inbox priority and others

A change has come in the mail that splits the incoming mail in the inbox.
I do not like the emails sorted into priority and others.
How can I turn this off.

fry n. say:

Microsoft are currently implementing the Focused inbox feature in Outlook.com webmail to all Microsoft accounts. Focused Inbox helps you focus on the emails that matter most to you. It separates your inbox into two tabs Focused and Other. Your most important emails are on the Focused tab while the rest remain easily accessible but out of the way on the Other tab. You’ll be informed about email flowing to Other, and you can switch between tabs at any time to take a quick look. 

To turn if Off, kindly go to Settings/ Gear Icon > Display settings > Click Focused Inbox > Then click "Don't sort messages" > Hit Save.
