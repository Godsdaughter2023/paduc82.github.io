That's because the TOC is picking up the format of the page numbers from the current document. Even if you have not inserted page numbers, their format is still part of the document, and by default is Arabic numbers. You can:

A) Redefine the page number format
Go to a header or footer (View > Header and Footer), select Insert > Page Number and change the numeric format. (Then you can delete the number that was added, but by now you've already changed it's internal format.)

B) Do it the other way around
If you instead paste the cover letter before the TOC (i.e., in the TOC's document), the format of the page numbers should stay put.

You have a table tblparameters, with a Field StudentID. When the student logs in you update that tabel with the StudentID:

Dim strSQL As String

CurrentDB.Execute strSQL, dbFailonError
