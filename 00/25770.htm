suzanne s. barnhill say:

When a sufficient amount of text is selected, pretty much any check box in any dialog will be shaded, even if the formatting is consistent for all the selected text; that's just the way Word works. If you have explicitly cleared the check box for all the selected text, this is nothing to worry about.

As for Ctrl+Spacebar, this command (ResetChar) just removes direct formatting and returns the formatting of selected text (or, if no text is selected, text typed after the insertion point) to the formatting defined by the style. If selecting text and pressing Ctrl+Spacebar is resulting in its being set to "Do not check," then that is specified in the applied style, and you need to modify the style to correct that.

For more about how spell checking works, see http://wordfaqs.mvps.org/MasterSpellCheck.htm.

http://technet.microsoft.com/en-us/sysinternals/bb963902

 

With Autoruns you can uncheck an item, which disables it from starting, or you can right click an item and then delete it. If you uncheck you can recheck to re-enable the item. It is a much safer approach than editing the Registry and better than using msconfig.

The file name changes to AutoRuns.txt. Click the Save button and upload the file to your OneDrive as a shared file and post as a link here.

 

A useful feature of the programme is that you can right click an item and select Search Online to get information about the item selected.

In Hybrid deployment, Exchange server’s MRM policies will not be migrated along with the mailboxes to Office 365.
