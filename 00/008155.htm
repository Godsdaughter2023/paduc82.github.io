a. user say:

Hi Mark,

Thank you for posting your query in Microsoft Community.

I understand that you are unable to get images on some websites on Internet Explorer. However, I would have you answer a few questions to narrow down the issue and provide you with better assistance.

  1. Is the issue specific to some particular webpages?
  2. Are you able to click on other hyperlinks on that webpage?
  3. Did you make any significant hardware or software changes to the computer prior to this issue?

This can be caused by a compatibility problem between Internet Explorer and the site you're on. When a site is incompatible with Internet Explorer, you'll see the Compatibility View button in the Address bar. You can only turn on Compatibility View in Internet Explorer for the desktop.

 To turn on Compatibility View

  1. See if the Compatibility View button appears in the Address bar (If you don't see the button, there's no need to turn on Compatibility View) .
  2. Tap or click the Compatibility View button to display the site in Compatibility View.

If the issue still exists, I would suggest you try the following methods and check if the issue persists.

Method 1:

The problem may arise due to some add-ons that are enabled for the website. I would suggest you run Internet Explorer (IE) with no add-ons and then open the websites:
a. Press Windows key + R.
b. In the Run Window, type iexplore -extoff and hit ENTER.

c. The Internet Explorer will open up without add-ons.

If you are able to click on the buttons, then try turning on the add-ons one by one and check which particular add-on is causing the issue.

Manage add-ons in Internet Explorer

If the issue still exists, try the next method.

Method 2:

The issue may also arise due to the Protected mode turned on in Internet Explorer.

  1. Click the Tools icon in Internet Explorer.
  2. Click the Security tab and uncheck the checkbox beside Enable Protected Mode (requires restarting Internet Explorer).
  3. Click Apply, and then click OK.
  4. Close all open Internet Explorer windows, and then restart Internet Explorer.
  5. Browse to a website and try clicking on the buttons.

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.
