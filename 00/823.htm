UserAssist is absolutely made by Microsoft.  You're probably getting confused because there are also 3rd party programs, which are also called UserAssist, which were not created by Microsoft.  Those 3rd party programs are generally designed to let a person better interface (read) with Microsoft's UserAssist data.

You know how the Windows 7 Start Menu puts your most-used program at the top of the list, then your next-most-used program, and so forth down the list?  The number of times you use a program is kept track of by a little-known Microsoft Windows utility called UserAssist, which doesn't appear in the Microsoft Knowledge Base, probably because many people would freak out if they realized how much their use of programs was being tracked, but it is definitely a Microsoft tool.

Calculating 33% breaks for a list of data - Microsoft Community

kyleccu say:

Calculating 33% breaks for a list of data

I have been trying to figure out if there is a way to calculate breaks at intervals of 1/3 for a set of data, similar to the way the quartile function calculates breaks at 1/4.  So far I have only been able to find suggestions for ways to divide the data into thirds, not calculate new values representing the bottom, middle, and upper 33% break for the list of data.   If anyone can help with this I'd love a suggestion.   Thanks very much
j.e. mcgimpsey say: You can get any quantile of a data set by using the PERCENTILE.EXC(<data range>, k) function, where k = 0...1.

So if you want to determine where each data point falls in the 1/3 quantiles (i.e, 1st, 2nd or 3rd), you could use something like this (assume your data is in $A$1:$A$100):

  = 1 + (A1 > PERCENTILE.EXC($A$1:$A$100, 1/3)) + (A1 > PERCENTILE.EXC($A$1:$A$100, 2/3))

Note that if you need the formula to be compatible with pre-XL2007/8 versions of XL, use PERCENTILE() rather than PERCENTILE.EXC()
