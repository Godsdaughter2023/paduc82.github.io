1. Navigate to the Xbox app within Windows 10.

2. Select Settings and then Game DVR.

3. Turn ‘Record game clips and screenshots using Game DVR’ to off.

Turn the Game Bar on or off in Windows 10 using the registry:

1. Navigate to HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\GameDVR.


2. Change the AppCaptureEnabled DWORD value to 0 to turn it off. Change it to 1 to turn it back on.

3. Navigate to HKEY_CURRENT_USER\System\GameConfigStore.


4. Change the GameDVR_Enabled DWORD value to 0 to turn it off. Change it to 1 to turn it back on.

To remove the Game Bar altogether:
If you would like to remove the Xbox app altogether, try this.

1. Open a Windows PowerShell as an administrator.


2. Type or paste ‘Get-AppxPackage *xboxapp* | Remove-AppxPackage’


3. Let the process complete.

Let us know if you can now record again.

bossdweeb say:

What graphics card did you upgrade from ?

Are the BSOD happening even with no load on the gpu ?

- TRD errors always blame the driver. I don't think they know how to do anything else, because they come from a lag in

the gpu processing, but the driver is not always at fault.

Often as not the driver is the 'last man standing' in a chain of events that lead to the TDR.

- If a BIOS update did not help, and you have not already, you may want to try re-seating the card, and double checking

the power connectors are on fully. Particularly the 'extra' two pin on the eight pin connection.

You may also have bumped a RAM stick while installing the 2070, which can affect the connection, so you might want to

try re-seating the RAM as well.

-  Did you install any software with the new card that you didn't have before ?

- If you still have your old gpu you may want to try reinstalling that to help verify if your new card is faulty.

If not, try using your integrated graphics, if you have Intel HD or AMD IGPU, to see if you still get BSOD.

- It's not often an Nvidia driver is faulty. I haven't tried the new one, but  v417.71 is working well with my GTX980.

Check out the Nvidia forums to see if anyone is having problems with their 2000 series and the new driver.

you launch manually after booting into Windows as Clean Boot is only helpful with stuff that loads when booting into Windows.
