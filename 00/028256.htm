HTML format for replies - Microsoft Community

joseph zn say:

HTML format for replies

Is there a way to set HTML as the default format for replies?
shashirekha say:

You may run the following VBA macro to default HTML format for all replies in Outlook. To enable macro in Outlook 2007, refer to the following link:

http://office.microsoft.com/en-us/word/HA100310711033.aspx

To create a macro in Outlook, visit the link below:

http://msdn.microsoft.com/en-us/library/bb206736.aspx

Option Explicit

Private WithEvents oExpl As Explorer
Private WithEvents oItem As MailItem

Private bDiscardEvents As Boolean
Private olFormat As OlBodyFormat


Private Sub Application_Startup()
   
   Set oExpl = Application.ActiveExplorer
   
   bDiscardEvents = False    
 
   olFormat = olFormatHTML        '(*2) - uz.ywaj zawsze formatu HTML
   
End Sub

Private Sub oExpl_SelectionChange()

   On Error Resume Next
   Set oItem = oExpl.Selection.Item(1)
   
End Sub

' (*3) Uz.ytkownik wybra? polecenie "Odpowiedz"
Private Sub oItem_Reply(ByVal Response As Object, Cancel As Boolean)

   If bDiscardEvents Or oItem.BodyFormat = olFormat Then
       Exit Sub
   End If
   
   '(*4) Anuluj domys'lna; akcje;
   Cancel = True

   bDiscardEvents = True
   
   ' (*5) Utwo'rz odpowiedz' na wiadomos'c' w formacie tekstowym
   Dim oResponse As MailItem
   Set oResponse = oItem.Reply
   oResponse.BodyFormat = olFormat
   oResponse.Display
   
   bDiscardEvents = False
   
End Sub

' (*6) Uz.ytkownik wybra? polecenie "Odpowiedz wszystkim"
Private Sub oItem_ReplyAll(ByVal Response As Object, Cancel As Boolean)

   If bDiscardEvents Or oItem.BodyFormat = olFormat Then
       Exit Sub
   End If

   Cancel = True
 
   bDiscardEvents = True
   
   Dim oResponse As MailItem
   Set oResponse = oItem.ReplyAll
   oResponse.BodyFormat = olFormat
   oResponse.Display
   
   bDiscardEvents = False
   
End Sub

' (*7) Uz.ytkownik wybra? polecenie "Przes'lij dalej"
Private Sub oItem_Forward(ByVal Forward As Object, Cancel As Boolean)
   
   If bDiscardEvents Or oItem.BodyFormat = olFormat Then
       Exit Sub
   End If
   
   Cancel = True

   bDiscardEvents = True
   
   Dim oResponse As MailItem
   Set oResponse = oItem.Forward
   oResponse.BodyFormat = olFormat
   oResponse.Display
   
   bDiscardEvents = False
   
End Sub

NOTE: disable macro warnings in Outlook using Tools > Trust center > Macro security

mike h.. say:

Hi,

There isn't a worksheet function for summing by colour, you need VB. It's quite possible to write a user defined function to sum by colour or you can use a much simpler UDF to return the colorindex of a cell into a helper column and sum by this index.

This simple UDF returns the colrindex of a cell

Function WhatColour(rng As Range) As Long
    WhatColour = rng.Interior.ColorIndex
End Function

On the worksheet call with =WhatColour(a1)
It will reurn -4142 if the cell doesn't have a fill colour.

So now if we wanted to summ all red cell in column A and our helper column was B we could use

=SUMPRODUCT((B1:B16=3)*(A1:A16))

Note that this applies only to fill colours and NOT conditional format colours. It is possible to sum by CF colours but it's a lot more complicated. Bob Philips has a solution for that here but prepare yourself for a very technical paper

http://www.xldynamic.com/source/xld.CFConditions.html
