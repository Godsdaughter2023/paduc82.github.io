john w. vinson say:

All this could be done, with a fair bit of difficulty (building custom SQL strings in code and executing them) - but I'm afraid you're really way off on the wrong track!

For one thing, it's very rare indeed to need to create a new Table as part of routine operation.

For another, it's essentially never a good idea to store information (the expert's last name, address, etc.) redundantly in two or more tables.

And you certainly should NOT have multiple fields in your table for different kinds of experts! There should be only one People table, and it should have only one LastName (etc.) field.

It sounds to me like you may be better off with a properly normalized table structure such as:

tblPeople
  PersonID
  LastName
  FirstName
  <other biographical/contact info>

tblSource
  SourceID <autonumber primary key>
  Source <e.g. "Conference", "Referral", "Journal">

tblReferrals
  PersonID <link to tblPeople, who is being identified>
  SourceID <link to tblSource, how did this person get identified>
  IsTheExpert <yes/no, is this person "The Expert" that you want to emphasize>
  <possibly other fields about how this person came to be recognized>

It's always best to store your data in properly normalized tables, and - unless I'm grossly misunderstanding - your tables are NOT properly normalized.

vijay a. verma say:

1. To limit the number of characters -

Select that particular cell or range of cells, Data tab > Data Validation > Data Validation

In the Allow: select Text length

In the Data: Less than or equal to

In Maximum: Put your number of characters

Put appropriate messages in "Input Messages" and "Error Alert" tabs (If you want)

2. Colour can be changed if number of characters exceed. But if you take recourse to point 1, then you will not be able to exceed the characters. Hence giving it a colour will become meaningless as you will never be able to put more characters than required.

In case, you want to give color, select cell or range of cells

Home>Conditional Formatting>New Rule>Use a Formula to Determine which cells to format and put following formula and choose required Color in Fill Tab

If you selected cell is A2 or range start is A2 (says A2:D10), put following formula

=LEN(A2)>10 (replace 10 with required number of characters)

Implementation Idea:  Try controlling the calculation yourself.  You are likely set to Automatic, and you can control the level (workbook, worksheet, or cell) and timing of calculation to eliminate calculation redundancy.  You can save the calculation setting, set calculation to Manual, run your procedure (calculating only when necessary and at the level necessary), and then reset the calculation mode to the saved setting.  I've included some rough code below.

Matthew Herbert

Sub MyProcedure()
Dim xlCalc As XlCalculation

With Application
    xlCalc = .Calculation
    .Calculation = xlCalculationManual
End With

'   Worksheets(1).Range("A1:A3").Calculate

'return calculation to its original setting
Application.Calculation = xlCalc
End Sub
