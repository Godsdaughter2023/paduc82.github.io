Thanks for posting. You most likely will not be able to delete the system recovery partition inside Windows. If you are just looking to install XP and not use Windows 7 you can simply boot to your XP disk and delete and format all the partitions (WARNING: This will erase all your data).

If you want to use the recovery partition space to install XP you can delete the recovery partition only from the disk and load it there (WARNING: Doing this will NOT allow you to restore your Windows 7 installation).
