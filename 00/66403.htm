dan v [msft] say:
Hi Jon,

This error can appear during a clean installation of Windows 7 when the operating system install media is not in the drive. This typically occurs when a user is required to exchange disks in order to install required drivers on the system and the operating system disks were not placed back in the drive before proceeding with the install. 

The error would not occur if setup was started using a USB memory stick since the isntall media would not be switched.
 
The solution would be to change the driver CD or DVD to the Windows 7 install media DVD when this occurs so we can continue with the Windows 7 setup. In this particular situation it is likely that the RAID driver disk was left in the drive.

Note: This issue/error occurred less in Windows Vista since a message prompting for the install media was usually displayed similar to this:

"Please insert Windows installation disc 1"

cmdrkeene say:
Here's how:

Method One:  The Win-P shortcut
  1. While holding down the Windows Logo key, type the letter P until "Computer Only" is selected on the popup.  This will disable any "projector" or extra monitor that might be enabled, and move any windows from that other monitor to the one you can see.
Method Two: The Windows Key Shortcut
  1. Switch to the off-screen window by selecting it on the taskbar. Even though you can't see it, it needs to be the active program when you press the keys in the next step.
  2. Hold down the Windows Logo and either Shift key on the keyboard, then tap the left or right arrow.  So Windows-Shift-Left or Windows-Shift-Right.  This will attempt to move the window to your current monitor.
Method Three:  The Control Menu Method
  1. Switch to the off-screen window by selecting it on the taskbar. Even though you can't see it, it needs to be the active program when you press the keys in the next step.
  2. Press Alt-Spacebar, then release.  You may or may not see a menu appear, but do not do anything with it directly:  now type an M to select the "Move" option from the menu.
  3. Press any arrow key at all just once.
  4. Being careful not to click, move your mouse around until you see the window.  It should be "stuck" to your mouse pointer and moving with it.  Then you can move the window normally and click to drop it in place.
