hansv mvp say:

Place the following code in the ThisWorkbook module of the workbook.

 

 

Private blnMoveAfterReturn As Boolean
Private lngMoveAfterReturnDirection  As XlDirection
 

Private Sub Workbook_Activate()
    Application.MoveAfterReturnDirection = xlToRight
    Application.MoveAfterReturn = True
End Sub
 

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    Application.MoveAfterReturnDirection = lngMoveAfterReturnDirection
    Application.MoveAfterReturn = blnMoveAfterReturn
End Sub
 

Private Sub Workbook_Deactivate()
    Application.MoveAfterReturnDirection = lngMoveAfterReturnDirection
    Application.MoveAfterReturn = blnMoveAfterReturn
End Sub
 

Private Sub Workbook_Open()
    blnMoveAfterReturn = Application.MoveAfterReturn
    lngMoveAfterReturnDirection = Application.MoveAfterReturnDirection
    Application.MoveAfterReturnDirection = xlToRight
    Application.MoveAfterReturn = True
End Sub

 

 

Save the workbook as a macro-enabled workbook (.xlsm or .xlsb, not .xlsx).

 

Users will have to enable macros. The easiest way to do this is to make the folder containing the workbook a trusted location for Excel - see Add, remove, or change a trusted location. Each user will have to do this once.
