Can you pin a folder to the taskbar in Windows - Microsoft Community

drmudd say:

Can you pin a folder to the taskbar in Windows Vista/7/8.1/10? No....but yes you can.

I searched around and found a workaround besides creating a new toolbar of which can only utilize one row of a two-row taskbar. You can't specifically pin a folder to the taskbar, only an exe but here's how to trick Windows into "letting" you pin a folder shortcut and icon onto the taskbar.

1) Windows Vista/7

Open Control Panel/Folder and Search Options/View and uncheck "hide extensions for known file types"

     Windows 8.1/10

Open File Explorer. At the top of the top of the window click "View" then click the icon (not the drop down menu) on "Options" then click "View" and uncheck "hide extensions for known file types"

2) Close the windows and right-click on an empty space on your desktop. From the menu select "New" then select "Text Document"

3) Rename the text document to whatever you want. Example "Drive C". Then change the .txt extension to .exe and press enter. When asked if you are sure you want to change the extension click "yes"

4) Drag your "dummy" exe to the taskbar and drop it there. Your "Drive C" "exe" is now pinned to the taskbar

5) Right-click on the "Drive C" icon and right-click on Drive C. Then click "Properties" and in the "Target" box delete all text and type C:\ and then click on the "Start in" box and delete all text and type C:\

6) Then click apply/ok. And that's it. Left-click on your Drive C icon in the taskbar and it will open a window showing the contents of your C: Drive. When you restart your computer the icon in the taskbar for your Drive C will change to the Windows default icon for the C: Drive. You can now delete the "dummy" exe from your desktop and your new shortcut to Drive C will remain in the taskbar until you unpin it

Note: You can pin a taskbar shortcut to any folder you want by typing the exact folder path inside the "Target" and "Start in" boxes in the properties. For example, the path to your user folder would be C:\Users\YourUsername. The path to your documents folder would be C:\Users\YourUsername\Documents

Further note. If you want to change the icon to something other than the Windows default icon you must save your "dummy" exe somewhere and then you can click the "Change Icon" buttton in its properties and change the icon to whatever you want. Again, if you want your new shortcut to look like the original folder icon you can just delete the "dummy" exe from your computer - you only have to save that exe if you want to change the icon to something different

Enjoy your new taskbar folder shortcuts (I mean "programs"). Shhh...don't tell Windows and it will never know our little secret
