doug robbins - mvp office apps & services (word) say:

If put { DOCVARIABLE [varname] } fields in your document, where [varname] is the name of the variable from column A of the first sheet in the Excel workbook and you create an AutoNew macro in your template that contains the following code:

Dim xlapp As Object
Dim xlbook As Object
Dim xlsheet As Object
Dim bstartApp As Boolean
Dim i As Long
On Error Resume Next
Set xlapp = GetObject(, "Excel.Application")
If Err Then
    bstartApp = True
    Set xlapp = CreateObject("Excel.Application")
End If
On Error GoTo 0
Set xlbook = xlapp.Workbooks.Open("Path\FilenameofExcelWorkbook")
Set xlsheet = xlbook.Sheets(1)
With xlsheet.Range("A1")
    For i = 1 To .CurrentRegion.Rows.Count - 1
        ActiveDocument.Variables(.Offset(i, 0)).Value = .Offset(i, 2)
    Next i
xlbook.Close
If bstartApp = True Then
    xlapp.Quit
End If
Set xlapp = Nothing
Set xlbook = Nothing
Set xlsheet = Nothing
ActiveDocument.Range.Fields.Update

In which you specify the path and name of the Excel Workbook in the line

Set xlbook = xlapp.Workbooks.Open("Path\FilenameofExcelWorkbook")

when you create a new document from the template, the data from the Excel workbook will be displayed in the Document Variable fields in the document.

See the article "What do I do with macros sent to me by other newsgroup readers to help me out?” at:
