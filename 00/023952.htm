Applying a theme will only modify those elements that get their formatting from the theme. For example, the font of the document won't change to the theme font if you have selected a particular font, such as Arial. What you have to do is choose either "+Body" or "+Heading" for font "name." Similarly, the color of text or the color of a shape will only change if a theme color was applied to the object.

    Permanently remove text style from template - word 2016 - Microsoft Community

    nos4int say:

    Permanently remove text style from template - word 2016

    Since it was driving me mad, I would like to share the solution I found.

    This relates to the text styles including all preexisting and created styles in the 'home tab'.

    The option 'remove from style gallery' will only delete the style from the current document and not the template. There is no delete from template or import/export option available, as far as I know. The solution below will also work with all standards styles included in word. 

    To permanently remove the style:
    1) right click on the style -> Modify Style
    2) un-tick 'add to styles gallery'
    3) select 'new documents based on this template'
    4) OK and done

    I hope it helps someone else. Please let me know if there are other ways to deal with it.

    41 people found this helpful

    suzanne s. barnhill say:

    Note that this removes the style from the Style Gallery, but it does not remove it from the Styles pane (if you display All styles or (depending on the style) Recommended, or, if it's in use, from In use or In current document).

    Moreover, it does not remove the style from the given document if it is in use. You can search for paragraphs in that style and replace it with a different style. If it is not a built-in style, you can delete it entirely through the Manage Styles dialog, which does give you the option to apply the deletion to "New documents based on this template."
