    ben m. schorr -bschorr say:

    No. Collapsing the parent folder collapses all of the sub-folders.

     

    The tricks to working around long, unwieldy, folder trees is:

     

    1. Don't make unnecessary folders. Don't have structure just for the sake of structure. If a folder would only have a few dozen messages in it, there's rarely a need to sub-divide that folder.

     

    2. Use search more to find messages rather than relying upon navigation.

    3. Balance width with depth. Instead of one folder with 100 subfolders, consider breaking that folder into 5 subfolders, each with 20 sub-sub folders. Don't get carried away, balance is key.

    4. Use the Favorites bar in Outlook to pin current, frequently used, folders so you don't have to search for them.
