Hi MB_2017.

 

First, If you were using a preservation policy, that policy has been automatically converted to a retention policy, which is a new feature that does everything a preservation policy does and more.

 

If you are using the retention policy on SharePoint sites, the policy copies the changes of the content to the Preservation Hold library. You could browse this picture and article to get more information:

 

Meanwhile, anyone could not delete the items from Preservation Hold library.

 

About the timer job, it is a SharePoint default settings to check whether some content need to be deleted. It runs periodically to delete all the items whose retention period has expired within seven days of the end of the retention period. That means, it only automatically delete items whose retention period has expired. You could get more details from these picture and browse this article to get more details:

 

 

Therefore, you could not delete the content manually in the preservation library and the timer job deletion rule is a SharePoint default settings and you could change it. Given this situation, we welcome and encourage you to vote this  User Voice: Allow Admin to delete files from the Preservation Hold Library folder. The Skype client requires Internet access over Wi-FI or a data plan over a mobile network to be usable. A UK sim card providing data is one way to gain access. You might want to consider whether it is more advantageous/affordable to you, to use Skype or simply the UK mobile sim without Skype's services. If you use a sim card along with a Skype Number, you would have 2 phone numbers. One directly through the sim provider (no data required) and another through Skype (data required).
