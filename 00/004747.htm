margaretpate say:

Conditional Formatting for a Drop Down List in Word

I want to create a RAG status drop-down list in Word; when the User selects Red, I want the text &/or the box to turn red, Amber it should turn amber & Green should turn green.  I'm OK with creating the list but how do I get the conditional formatting to work?
doug robbins - mvp office apps & services (word) say:

If you use a ComboBox Content Control with the Title of RAG and you have it located in the cell of a table and you have the following code in the ThisDocument object, cell containing the Content Control will be shaded according to the selection made by the user:

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
If ContentControl.Title = "RAG" Then
    With ContentControl.Range
        Select Case .Text
            Case "Red"
                .Cells(1).Shading.BackgroundPatternColor = wdColorRed
            Case "Amber"
                .Cells(1).Shading.BackgroundPatternColor = wdColorOrange
            Case "Green"
                .Cells(1).Shading.BackgroundPatternColor = wdColorGreen
        End Select
    End With
End If
End Sub

jacques raubenheimer say:

In my opinion, Microsoft has really messed up the application of something that used to work really well (going from Word 2003 to Word 2007 and beyond).

Essentially, unless you're willing to muck around with the XML, Microsoft has provided no way to remove multilevel lists introduced into a document.

So what you want to do is:

1. Select a heading (one that still has one of your lists numbers).

2. Right click on the number and choose Adjust List Indents (or, making sure that step 1 has been done, click on the Multilevel list tool, and choose--very counter-intuitively--Define New Multilevel list)

3. This will open the Multilevel list dialog. Click on More at the bottom left.

4. Repeat this for each heading level

  a) click on the heading level (1 through 9, or however many you have used) at the top left

  b) Set Link level to Style (near the top right) to "(no style)"

5. Click OK.

Now you can still use your heading styles, and the numbers will not be added. But beware, that multilevel numbering list is still lurking in the unseen regions of your document, because, as I have said, the way Microsoft changed this in Word 2007 (and since) means they have given us no easy way of deleting these lists.

Excel rounding currency calculation to whole numbers when I want 2dp - Microsoft Community

np95 say:

Excel rounding currency calculation to whole numbers when I want 2dp

In the formula below, D12 is formatted as percentage, E12 and F12 are currency with 2 decimal places, and the output (G12) is currency with 2 decimal places.

=IF(IF(E12<0,0,PRODUCT(D12,E12))<IF(F12<0,0,PRODUCT(D12,F12,6)),QUOTIENT(SUM(IF(E12<0,0,PRODUCT(D12,E12)),IF(F12<0,0,PRODUCT(D12,F12,6))),2),IF(E12<0,0,PRODUCT(D12,E12)))

The output usually gets rounded to a whole number (but with large values of E12 and F12, sometimes the rounding doesn't occur).

Using a simple example where D12=100%, E12=£1.00 and F12=£1.00, the value I get in G12 is £3.00, when it should actually be £3.50.

I'm fairly confident I haven't made a mistake in the formula as entering £10.00 in both E12 and F12 produces a value of £35.00 in G12 

Does anyone know how to stop the rounding?

bernard liengme say:

Your formula is behaving correctly but (with respect) you have misunderstood the QUOTIENT function.

In =QUOTIENT(N, K) it computes N/K and returns just the integer part.

With the sample data you have given, click on G12 and in the Formula tab use the Evaluate Formula tool. You will eventually get to QUOTIENT(7,2), the result of the division is 3.5 but the integer part is 3.

I note you like to use function such as SUM and Product rather than math operators like + and *. Unfortunately there is no DIVISION function. So you need to replace

QUOTIENT(SUM(IF(E12<0,0,PRODUCT(D12,E12)),IF(F12<0,0,PRODUCT(D12,F12,6))),2)

by

SUM(IF(E12<0,0,PRODUCT(D12,E12)),IF(F12<0,0,PRODUCT(D12,F12,6)))/2

and all will be well

Best wishes

PS A simplified version is =((D12*E12)*(E12>0)+(D12*F12*6)*(F12>0))/2
