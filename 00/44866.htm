malex9 say: Hello,

I had same problem, but I just managed it to work. Here are the steps:

1. Open registry editor (on keyboard Win button + R then type "regedit")
2. In registry editor go to "HKEY_CURRENT_USER/AppEvents/Schemes/Explorer". Under Explorer you shouldn't see "SystemNotification" entry, so it comes to next step.
3. Right click on Explorer -> New -> Key, and rename it as "SystemNotification"
4. Right click on SystemNotification -> New -> Key, and rename it as ".current"
5. Left click on .current, and in right panel you'll see "(Default)" entry, double click on it.
6. In value data paste "C:\Windows\media\Windows Balloon.wav" and click OK (!!! make sure there exist Windows Balloon.wav file in mentioned folder "media").

That's it. It will work.
