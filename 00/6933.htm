Referring to the screen capture below. You need to create a column where the sum of the values in column B for the reference in column D matches the reference in column A.

The formula in cell G2 is:       =SUMIF($A:$A,D2,$B:$B)

Copy the formula down and in column G you will have the sum of the values from column B where the references in column A match the references in column D (and zero if no match found)

amit tandon say:

Refer below image:

Enter below formula in cell H2:

=SUMIF($A$2:$A$100,$G2,$B$2:$B$100)-VLOOKUP($G2,$D$2:$E$100,2,FALSE)

This adds up all values in column B where column A = "DEP22407" (cell G2) and then deducts the FIRST value in column E where column H = "DEP22407" (cell G2). In the above example: 1415.99+679.9+2279.95 - 4375.81 = 0.03

Alternatively, if you wish to deduct ALL values in column E where column H = "DEP22407" (cell G2), use below formula in cell I2:

=SUMIF($A$2:$A$100,$G2,$B$2:$B$100)-SUMIF($D$2:$D$100,$G2,$E$2:$E$100)

In the above example: 1415.99+679.9+2279.95 - 4375.81 - 0.01 = 0.02

a. user say:

Hello,

There is no direct way to export Search Folder criteria in Outlook. However, you may filter emails with the same criteria as search criteria when exporting. You may do so by following the steps below:

  1. Click File > Open & Export > Import/Export to open the Import and Export Wizard.
  2. Select Export to a file, then click Next.
  3. In the Export to a File dialog box, select Outlook Data File (.pst), then click Next.
