I think you are SOL.  AFAIK, this is normal Excel behavior:  we cannot do anything in VBA while editing a cell.  You can see this behavior by simply opening both the Excel and VBA windows, selecting a cell in Excel and puting the cursor in the Formula Bar (i.e. start "editing"), then selecting the VBA window and trying to position the cursor in the VBA editing pane.  Nothing will happen until you stop "editing" the Excel cell (e.g. press Esc in the Excel window).

I believe that VBA and Excel are separate threads.  But I suspect they use some interprocess synchronization mechanism(s) so that each knows the state of the other.

Hi,

 

Thank you for choosing Microsoft Community. I will provide you the steps to create a custom stationary and apply when sending an e-mail.

 

We cannot assign the individual stationary to specific contacts. However we can create multiple stationaries in Outlook and open a new e-mail with a specific stationary and send to the respective contacts.

 

Follow the steps:

 

1. Open a new e-mail and create a personalized e-mail.
2. Click on the Office button -> go to Save As.
3. Save the e-mail as MHT files (*.mht) in the following location:
    C:\Users\<User Name>\Application Data\Microsoft\Stationary.

If you do not see the Application Data folder, then you will need to enable the hidden files and folders. Refer the link:
http://windows.microsoft.com/en-US/windows-vista/Show-hidden-files

4. Reopen Outlook.
5. Click on Action toolbar in the Main Outlook Window -> go to New e-mail message using -> Click on More Stationary and select the stationary created.
6. Now you can insert an e-mail address and send the e-mail.
