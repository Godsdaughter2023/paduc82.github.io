Hi Joy,

You can add title, page numbers, or dates to every page in a document using headers and footers. Though, the headers and footers can be different in each section of a document. Here's how to create it:

  1. Choose Insert and select either Header or Footer.

  2. Select any built in format for your header or footer. After you choose a format, the header and footer area is shown in your document.

  3. Type the text you want in the header or footer.

  4. When you're done, choose Close Header and Footer.

If you want to create a header/footer that will apply to all the pages, please check if the process below will work on your end:

1. Go into the header or footer. This will automatically activate the Design tab of the ribbon.
2. In the Options group, there are check boxes for Different first page and Different odd and even pages.
3. If Different first page is ticked, the first page of a section has its own header/footer that won't be repeated on the next pages. If Different odd and even pages is ticked, even-numbered pages have a different header/footer than odd-numbered pages.

Let us know if this will help resolve your issue.

Thank you.

charles kenyon say:

Sections do not need their own headers/footers. That is controlled by the Link-to-Previous command in each section following the first. By default, headers and footers are linked to the previous section. Note that each section will have three headers and three footers; you will control whether or not those are displayed. Header/Footer Settings Recap

See also the StyleRef Field for the way you can have different content reflecting the body of the text in your headers and footers.

Bottom line, you can have one single header and one footer that display throughout your document regardless of the number of sections. By default, that is how a document is set up. If you change those settings, you have to start handling multiple headers/footers.

suzanne s. barnhill say:

By default, when you insert an endnote, there is a single space between the Endnote Reference Mark and the Endnote Text.

If you want to use a hanging indent, then you should really substitute a tab stop for the space(s) between the reference and the text. As for the style formatting, provided all the endnotes actually use the Endnote Text style, then you should be able to make the changes fairly easily.

First you need to make sure that the Endnote Text style is formatted the way you prefer it.

  1. Right-click in an endnote and choose Style... In the Style dialog, Endnote Text will be selected.
  2. Click Modify, then Format | Paragraph and define your preferred paragraph formatting.
  3.  Click OK to close the Paragraph dialog, OK to close the Modify Style dialog, and Apply to close the Style dialog.

With any luck, your endnotes will have updated automatically. If not, apply the following procedure:

  1. Press Ctrl+F to open the Find dialog.
  2. Click More>> to expand the dialog.
  3. Click Format, then Style, and select the Endnote Text style.
  4. Click Find in and choose Endnotes.
  5. Word will select all the paragraphs that are in the Endnote Text style.
  6. Click the title bar of the document to change the focus from the Find dialog to the document without deselecting the text.
  7. Press Ctrl+Q to reset the paragraph formatting to that of the style.

To add a tab character after the endnote reference mark, you can use Replace.

  1. Press Ctrl+H to open the Replace dialog.
  2. If you have just finished the preceding procedure, click No Formatting to remove the style formatting from the "Find what" box.
  3. In the "Find what" box, type ^e (which is the code for an endnote reference mark.
  4. In the "Replace with" box, type ^&^t (the text you just found, plus a tab character).
  5. Replace All.

If your preference is to have endnotes numbered with ordinary numbers (not superscript), followed by a period and tab character (with hanging indent), see http://word.mvps.org/FAQs/MacrosVBA/UnSuperscptFnotes.htm for a macro.
