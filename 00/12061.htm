Change row height format from inches to points? - Microsoft Community

dcon88 say:

Change row height format from inches to points?

In Excel for mac 2011, how do I change row height format from inches to points, like the windows version.
j.e. mcgimpsey say:
See Excel Help on the topic: Change the unit of measurement
Unfortunately, Bob, this doesn't help much:

Looking up "Change the unit of measurement" in XL2011 Help brings up nothing related to row height (at least for me). (It does in XL2008 Help.)
Second, the original poster wants to set row heights in points. There is NO option to do that directly via the XL user interface (e.g., Format/Row/Height). The options in Preferences/General referred to in the XL2008 Help topic are "inches", "centimeters", and "millimeters".

That said there are at least two options I can think of:
    1) When manually dragging the row height via the row header separator, points show up with the relevant conversion. Set the height to the points desired, note the conversion to inches, etc., then use Format/Row/Height and enter the converted value.

    2) Use VBA: Setting the .RowHeight property of a Row is done in points, e.g.,:
Selection.EntireRow.RowHeight = 12
        will set the rows to 12 points.

This macro may help:

    Public Sub SetRowHeightInPoints()
        Const cdMAXHEIGHT As Double = 409 'maximum row height in points
        Dim wsf As WorksheetFunction
        Dim dPoints As Double
        Dim vResponse As Variant
        If TypeOf Selection Is Range Then
            Set wsf = Application.WorksheetFunction
            With Selection
                dPoints = .Cells(1).EntireRow.RowHeight
                vResponse = Application.InputBox( _
                                Prompt:="Row height in points: ", _
                                Title:="Row Height", _
                                Default:=Format(dPoints, "0.00"), _
                                Type:=1)
                If vResponse <> False Then
                    dPoints = wsf.Round(wsf.Max(0, wsf.Min(cdMAXHEIGHT, CDbl(vResponse))), 2)
                    .EntireRow.RowHeight = dPoints
                End If
            End With
        End If
    End Sub

OK, you have a lot of options - here's one. 

Say that your are using 4 labels, in columns A to H (each label is two merged cells - A&B, C&D, etc). In column I, J, K, and L of the same row, use link formulas to your cells with counts:

='Today''s Order'!I5

='Today''s Order'!I6

etc.
