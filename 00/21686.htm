Let's suppose that the manager has his Contacts "Display As" fields set to list the last name, a comma, a space, and the first name.  Let's also say the Global Address List has then similarly configured.  The first time he uses one of those contacts, the name will resolve to "Lastname, Firstname <address@domain>".  From then on, he'll be able to begin typing either the last name or the address in order to see the autosuggestion.  So, if Mary Smith is in his Contacts and he had the Display As set to "Smith, Mary" and Mary's address is johnswife @ somedomain, the next time, he'll be able to start typing "s" or "j" and Mary will pop up.

Now, suppose that Mary writes to him, but she has her display name set to be "Mary Smith", so that's what shows in the From field when the message is received.  Supposed also that the manager replies to this message.  Now, a new entry will be added to the autocompletion cache, and the manager will be able to start typing "m" as well as "s" or "j" in order for Mary's entry to appear.  For "m" or "j", both entries will appear, but only one will appear when "s" is used.  This is the way it works for me every time.  I suspect that if he examines the behavior, that's what he sees as well.

feedback-hub:?contextid=88&feedbackid=e798c028-cd6b-4002-be90-187eec67654e&form=1&src=2

I don't know why, but with each new build, Windows changes the PhysX setting in the NVIDIA Control Panel to use the CPU instead of either the GPU or the recommended Automatic setting:

NVIDIA Control Panel -> "Configure Surround, PhysX" -> PhysX Settings -> Processor

Over various Windows builds and NVIDIA driver versions, I've confirmed that clean installs of the NVIDIA graphics drivers for my GeForce GTX 970 aren't the problem, and that it's definitely the installation of a new Windows build.  So, if you have an NVIDIA graphics card, you might check this to make sure you're actually using that card for PhysX instead of the CPU.

Microsoft Office Outlook 2007. How do I download my old emails from imap to pop3

Hi all.

I recently changed my account settings from imap to pop3 because it seems more stable in pop3.  The only problem is that my old messages and personal folders don`t appear in the new pop3 account.  How can I get them back?

hal hostetler - mvp-outlook say:
POP accounts know nothing of folder structures and any email those folders may contain, it only knows about the Inbox folder.  So, the simplest way to go about this is to setup the account as an IMAP account, export your entire folder tree to a .PST file, delete the IMAP account and re-create it as a POP account, then open the .PST file you saved and select it as the default storage folder for the POP account.  If you have more than one device, the POP service represents substantially more work to keep Al your devices in sync since it knows nothing about multiple device sync.  This scenario is the reason IMAP was invented, its designed specifically to keep multiple devices in sync.

ron coderre say:

Here are some options:

Even numbers:
=SUMPRODUCT(MOD(A2:A10+(A2:A10<>""),2))

Odd numbers:
=SUMPRODUCT(MOD(A2:A10,2))

(Adjust ranges to suit your situation)

Does that help?
