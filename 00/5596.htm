mike h.. say:

Hi,

 

What you can do is make them veryhidden and if you do this they don't appear in the menu popup to be un-hidden. Here's some code to do that. Change the sheet names in the string s to the names you want to hide but be careful to maintain the exact syntax

 

ALT+F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in. Close VB editor.

 

To run the code you can create a shortcut

 

View tab|Macros and highlight the macro name and click options and enter a letter and OK out, Now when you press CTRL+Letter the code runs. If the sheets are hidden it unhides them and vice versa.

 

 

 

 

Sub veryhidden()
Dim s As String, v As Variant
Dim ws As Worksheet
s = "Sheet1,Sheet3,Sheet5"
v = Split(s, ",")
For Each ws In ThisWorkbook.Worksheets
    If Not IsError(Application.Match(ws.Name, v, 0)) Then
        If ws.Visible = True Then
            ws.Visible = xlVeryHidden
        Else
            ws.Visible = True
        End If
    End If
Next
End Sub
