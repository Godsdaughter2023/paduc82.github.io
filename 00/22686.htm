Hi

Question: Can multiple (2) skype account be linked to a single Microsoft Account?

Answer: A situation wherein two accounts show under the same email address happens when a legacy Skype user (account was created using a Skype Name) signed in to Skype using the associated email address instead of his Skype Name without fully updating the accounts into one, which then resulted to a newly-created account.

Now, the account creation process will only allow a user to only have one account under one email address. It is not possible to create two or more account under the same email address.

Issue: I cant change my Skype ID (maybe have a fee associated to this change like for XBox live??)

Answer: The reason why you cannot change your Skype Name is because Skype never offered this option. We advise users to change their display names instead as a possible workaround. See this link for added details. How do I change my Skype display name? 

Issue: Can't unlink from Microsoft account either...

Answer: Please note that the option to unlink Skype and Microsoft accounts is currently not available. You may refer to this support link for additional details. Can I merge or unlink my Skype and Microsoft accounts?

Example:

12345      This is what is sitting in the cell and the next cell may be  26745

                   I need to have the finished product in the cell look like this:

TRX12345

jeeped say:

If the TRX is static (e.g. always going to be TRX) then you can put the following in an adjacent cell,
="TRX"&A1
... then fill down as far as required. If all of the numbers are not 5 digits and you want to maintain length, you can try,
=TEXT(A1, "TRX00000") which gives results like,
TRX00123
TRX01234
TRX12345 

If you need the new values in the original column, copy the new values and Paste Special, Values into the original column.

hansv mvp say:

As in all previous versions, you have to start Advanced Filter from the target sheet:

  • Activate the target sheet.
  • Create the headers for the 'Copy to' range.
  • Select an empty cell in the target sheet.
  • On the Data tab of the ribbon, in the Sort & Filter group, click Advanced.
  • Activate the source sheet and select the data list including the headers (field names)
  • If desired, click in the 'Criteria range' box, then select the criteria range.
  • Select 'Copy to another location'.
  • Click in the 'Copy to' box, then select the headers of the target range.
  • If desired, tick the check box 'Unique records only'.
  • Click OK.
