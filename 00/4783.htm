If you select the Questions and Answers and then run a macro containing the following code, it will re-arrange them in a random order:

Dim i As Long
Dim rngqna As Range
Dim rngdup As Range
Dim rngq As Range
Dim arrqna As Variant
Dim n As Long, j As Long, k As Long
Set rngqna = Selection.Range
For i = rngqna.Paragraphs.Count - 1 To 1 Step -2
    Set rngq = rngqna.Paragraphs(i).Range
    rngq.Start = rngq.End - 1
    rngq.Text = "_"
Next i
Set rngdup = rngqna.Duplicate
rngdup.End = rngdup.End - 1
arrqna = Split(rngdup.Text, vbCr)
Randomize
For n = LBound(arrqna) To UBound(arrqna)
    j = CLng(((UBound(arrqna) - n) * Rnd) + n)
    If n <> j Then
        temp = arrqna(n)
        arrqna(n) = arrqna(j)
        arrqna(j) = temp
    End If
Next n
On Error Resume Next
For k = LBound(arrqna) To UBound(arrqna)
    rngqna.Paragraphs(k + 1).Range.Text = arrqna(k) & vbCr
Next k
rngqna.MoveEnd wdParagraph, 1
rngqna.Text = Replace(rngqna.Text, "_", vbCr)

per jessen say:

In the macro from HansV there missed an If statement to check if the value in column G is found in column A. If not found skip the next statement:

Sub SortMatch()
  Dim LastRow As Long
  Dim NxtName As Long
  Dim rng As Range
  Application.ScreenUpdating = False
  'Sort Columns A to F then Sort Columns G & H
  Columns("A:F").Sort Key1:=Range("A1"), Order1:=xlAscending, Header:=xlYes
  Columns("G:H").Sort Key1:=Range("G1"), Order1:=xlAscending, Header:=xlYes
  'Determine how long the Long List is
  LastRow = Range("G" & Rows.Count).End(xlUp).Row
  'Loop through the short list
  'Compare the value in G to the value in A
  'If they don't match, insert cells in Columns G & H
  'When they match, stop checking
  For NxtName = LastRow To 1 Step -1
    Set rng = Range("A:A").Find(What:=Range("G" & NxtName).Value, LookAt:=xlWhole)
    If Not rng Is Nothing Then
        Range("G" & NxtName & ":H" & NxtName).Cut Destination:=rng.Offset(0, 6)
    End If
  Next NxtName
  Application.ScreenUpdating = True
End Sub

taylor ehlert say:

Hey Debbie,

Really the only way you can convert a slide deck into a master slide is by copying and pasting the assets from the slide deck into the master slide that would take its place. Hope this helps!

rena yu msft say:

Hi Debbie,

 

Please refer to Create and save a PowerPoint template.

The steps should be:

  1. Open your customized presentation.
  2. Create a new normal presentation.
  3. Go to the master view of the new created presentation via View>Slide Master
  4. Copy and paste contents from the slide deck to the new created presentation slide master to customize the slider master.
  5. Save your new created presentation as a PowerPoint template.

Rena
