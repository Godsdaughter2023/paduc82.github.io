jay freedman say:

I'm not sure I'm seeing the same thing you are, but here's a guess: The default in Visio (at least in the 2010 version, which is what I have) is for shapes to have a light blue shadow along the bottom and right edges. It doesn't show up very well on screen unless you choose a high zoom factor, but it appears noticeably in a printout. If that's what you see, then the solution is to reopen the drawing in Visio, select everything, and set the shading to None.

As for how to insert a Visio drawing in a Word document, the "official" method is to click Insert > Object, choose the From File tab of the dialog, and browse to the file you want to insert. In most cases, dragging the file from Windows Explorer and dropping it into the document does the same thing.

A possibly better method is to use the Save As dialog in Visio to save a copy of the drawing as a graphic file in a format such as PNG (which uses lossless compression). Then use the Insert > Picture button in Word to put the graphic into the document. In between, you could use a graphics editor to resize the drawing, change its resolution, or add other elements as needed -- although Word has some tools for these things, they tend to be less flexible than a program such as IrfanView.
