¡firedog say:

Server Response: -ERR Cannot connect to POP server ...

That is a bogus message! The server cannot respond if you 'cannot connect' to it.

The message is coming from Avast's Mail Shield, which wants to handle the connection instead of allowing Windows Live Mail to do it. Disable the Mail Shield and restart the computer. You don't need mail scanning - especially for outgoing mail - and Avast will keep you safe anyway with its real-time file system protection.

POP server 2001:558:fe21:2a::7

That is the mail server's IPv6 address. Many people have had problems with Comcast's IPv6 configuration, and one way to test whether you have this type of problem is to disable IPv6 in your router's settings. How to go about it depends on which router you have, so look up its documentation. If this leads to no improvement in net connectivity, you can always reverse any changes you made.
