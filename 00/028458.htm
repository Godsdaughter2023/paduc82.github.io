john korchok say:

Locking anything in Word usually reduces the document usability, but here's a technique that might be sufficient. first, you need to display hidden formatting marks, so you can see what you're doing:

  1. In Word, click on the Office button, then on Word Options.
  2. Click on Display, then check Show all formatting marks. OK out.

Now you can set up the picture:

  1. In Word, create a new blank document.
  2. You'll see one paragraph mark in the corner of the page. Click on it and press Return to create another one. Cursor up, so it's blinking beside the first paragraph mark.
  3. Insert your picture. While it is still selected, click on Picture Tools>Format>Position>More Layout options.
  4. Click on the Text Wrapping tab and choose Behind text.
  5. Click on the Picture Position tab. Check Lock anchor.
  6. Assuming the picture will fill the page, set Horizontal to Absolute position 0" to the right of Page.
  7. Set Vertical to Absolute position 0" below Page. OK out.
  8. You should see an icon of an anchor to the left of the first paragraph mark. This is where the picture is anchored.
  9. Now select the second paragraph mark.
  10. Click on Review>Protect Document. The Restrict Formatting and Editing task pane opens on the side of your screen.
  11. Check Allow only this type of editing in the document. Set the dropdown to No changes (Read only).
  12. Under Exceptions, check Everyone.
  13. Click on Yes, Start Enforcing Protection. I do not recommend setting a password.
  14. Save the file.

Click on the photo and try to move it. It previews as if you can move the picture, but as soon as you release the mouse button, it's back where it was.

Click beside the second paragraph. Here you can edit text normally.

Hopefully, this will help.

=IF(B2="","",

    IF(B2>TODAY(), "Certification Invalid",

        IF(B2>TODAY()-90,"Re-certification is Due Soon",
