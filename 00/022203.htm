The Default Shape is not a re-usable preformatted shape as such, it just sets the defaults for Line and Fill formatting for the next shape that you insert.

For example, on the Insert tab click Shapes and click on the Rectangle button.  Draw a rectangle on the slide.  Format the rectangle with Line and Fill colours.  Right-click and Set as Default Shape.  Now insert a second shape, e.g. a Triangle.  It should have the same formatting as you applied to the first shape.

If you insert a Line or Arrow Connector, this will have different formatting.  There are separate default settings for Shapes (which are fillable) and for Lines.

 

Rather than applying a default formatting for all shapes, you can copy and paste formatting from one shape to another using either one of two tools:

1) The Format Painter.  Select a shape, click the Format Painter button on the Home tab, then click on another shape to apply the copied formatting.  If you want to apply the formatting to several shapes, when you copy the formatting from the first shape, double-click the Format Painter and it will stick on.  Click it once more to end.

 

2) Use the Pick Up and Apply buttons.  You have to add these to your Quick Access Toolbar (QAT).  Right-click on the PPT ribbon > Customize the QAT.  Choose Commands From: All Commands, then scroll down the left pane to find "Pick Up Style" and click the Add button in the middle.  Also find and Add the "Apply Style" command.  OK.  You can use these buttons in the same way as the Format Painter, but these tools let you do other things between copying a style from a shape and applying it to others.

rich007a say:
I noted that the shape created as default is only for that specific file and will not remain as the personalized default for other files (which is what I was after)..

What you want is not impossible, but you'd have to use a macro.  Try the following:

 

1) Create a new blank presentation.

2) Press Alt+F11 to open the VB Editor (VBE).

3) Do Insert > Module

4) Paste the following code in the space on the right:

 

 

Sub InsertGreyRightTriangle()
    Dim shp As Shape
    Dim dbWidth As Double, dbHeight As Double, dbTop As Double
   
    dbWidth = 250
    dbHeight = 20
    dbTop = 270
    Set shp = ActiveWindow.View.Slide.Shapes.AddShape(Type:=msoShapeIsoscelesTriangle, _
                                                      Left:=(ActivePresentation.PageSetup.SlideWidth - dbWidth) / 2, _
                                                      Top:=dbTop, _
                                                      Width:=dbWidth, _
                                                      Height:=dbHeight)
   
    With shp
        .Fill.ForeColor.RGB = RGB(150, 150, 150)   'Red, Green, Blue values (0-255)
        .Line.Visible = msoFalse
        .Rotation = 90
    End With
   
    On Error Resume Next
    shp.Select
End Sub

 

5) Close the VBE and return to PowerPoint.

6) Save the file as a Macro Enabled Presentation (*.pptm)

7) With the presentation active, and a slide showing in Normal View, do View tab > Macros > select "InsertGreyRightTriangle" > Run.

8) A grey triangle should appear on the slide.

 

If you want to be able to use that macro on all of your future presentations, you'll need to save the file as a PowerPoint Add-In (*.ppam), and if, having got this far, you're still interested, you need to read and copy the code from here: http://www.pptfaq.com/FAQ00031_Create_an_ADD-IN_with_TOOLBARS_that_run_macros.htm. 

 

Edit that code so that it runs the above insert-shape macro and away you go....

 

Noted that there is no way to modify or remove the Toolbar in Windows Explorer. Quite a shame as it's there for ''all'' the time when a user may ''never'' use it!.

If you remove everything you don't currently use from Windows and Word, PowerPoint & Excel, one day you wont be able to find something that is ...all of a sudden... useful!
