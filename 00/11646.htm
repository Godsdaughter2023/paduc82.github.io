Sub Test()

ronrosenfeld say:

Why not use separate cells for each entry?  I think that would be the simplest solution.

Also, I don't know what your maximum number of entries might be, but there is a limit of 32,767 characters in a cell.

 

Alternative solutions include using a User Defined Function to extract the last line and showing it in either a comment of in an adjacent cell.  (You probably cannot use a worksheet function as, at least in 2007 and probably 2010, they will not handle strings > 255 characters).

 

Here is a UDF that will return the last line of your data.  It assumes that, as in your posted example, the LineFeed character is what separates each entry.  However, in your "real data" it might be something else (e.g a Carriage Return character, of a combination).  If that is the case, you will need to change the "Split" function delimiter from vbLf to whatever you really have in there.

 

The UDF also assumes there are no "line feed" characters AFTER the last printed line.  If there might be, you could either return the next to last element of the variant array, or you might need a more complicated UDF that also processes any terminal LineFeed characters.  Easy to do if necessary.

 

To enter this User Defined Function (UDF), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this User Defined Function (UDF), enter a formula like

 

=LastLine(A1)

 

in some cell.

 

==========================================

Option Explicit
Function LastLine(s As String) As String
    Dim v As Variant
v = Split(s, vbLf)
LastLine = v(UBound(v))
End Function
====================================

 

To add a comment to the cell, you could use this macro, with the same caveats regarding the line break character as above:

 

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the cell or cells to process, then select the macro by name (EnterComment), and <RUN>.

 

 

====================================

Option Explicit
Function LastLine(s As String) As String
    Dim v As Variant
v = Split(s, vbLf)
LastLine = v(UBound(v))
End Function

Sub EnterComment()
    Dim c As Range
    For Each c In Selection
        c.Comment.Delete
        With c.AddComment(LastLine(c.Text))
            .Visible = True
        End With
    Next c
End Sub
