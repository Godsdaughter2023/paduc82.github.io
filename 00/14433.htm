jim gordon mvp say:
As Bob said, the structure of a (.xls) file is completely different from the standard Office Open XML format (.xlsx, .xlsm, ,xlst and .xlsb).  Starting with Excel 2008, the default file format is standard Office Open XML format.

It is highly recommend that you simply save your files in standard format. Otherwise, each time a (.xls) file is opened in either 2008 or 2011 it has to be translated from the old format into current format, worked on in the new format, and then translated back to the old format when it is saved. That involves a lot of complicated translating in and out, and none if it is necessary since both 2008 and 2011 work normally in the standard formats.

Obviously saving in the old format is causing problems, so why do it?
