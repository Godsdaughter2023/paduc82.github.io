You may have a rogue last remembered cell that Excel thinks is the logical extents of your worksheet. This commonly occurs when a value is mistakenly typed into the wrong cell which is far removed from the actual data area. Subsequently clearing the value does not reset the last cell that Excel remembers. You can test for this by tapping Ctrl+End to see what Excel thinks is the last cell. ActiveSheet.UsedRange

jllatham say:

Another thing that might cause this would be invisible "shapes" on the sheet that might be getting copied over and over.  Happened to a client of mine recently - the file grew to over 30MB and we did all of the usual delete extra rows/columns stuff and it remained at 30MB.  I happened to pretty much stumble onto the fact that there were some shapes with no line coloring or fill coloring and no text in them:  over 12,000!! of them.  Once we deleted those, the file size went down to under 90KB.

 

Here is some VBA code that would get rid of any such things on a selected worksheet, but note that it will delete all shapes which might include some things like text boxes you've set up to act like command buttons when clicked, or even some controls, so use this with a COPY OF YOUR WORKBOOK first.

 

Sub DeleteAllShapesOnSelectedSheet()
  Dim anyShape As Shape
 
  'will tell you if there are any on the sheet at all
  'and if there are, how many
  MsgBox "There are " & ActiveSheet.Shapes.Count & " shapes on the sheet.", _
   vbOKOnly + vbInformation, "Will Be Deleted on [OK]"
  For Each anyShape In ActiveSheet.Shapes
    anyShape.Delete
  Next
  MsgBox "Done"
End Sub

garybrazzell say:

This is definitely a disappointment, but here's a fairly efficient work around. Set your picture to layout behind text. Draw a shape designed to wrap your text over the picture. Make sure there is no fill and set the line on the shape to no line. When you set the line on the shape to no line, the text wrapping reverts back to treating your shape as a rectangle. But at this point, you can edit the wrap points of your no line shape, and get the same wrapping precision you formerly got with the picture itself.

Here's another work around I used to use a decade ago in Word. Use Photoshop to knock out or otherwise delete the background of the image in question. Save as a PNG with transparency. Traditionally, Word has recognized the transparent line of a PNG, so when you set that image to wrap tight, the text follows the transparent line of your PNG.
