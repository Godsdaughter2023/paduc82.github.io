             rcwhite  say:
            
            
            
            

﻿Hi, RCP10.


Switching between 32-bit and 64-bit Windows - in either direction - requires a clean install.  :>(  A reformat of the boot volume is not strictly required, but is often recommended.


If you have not yet installed any applications, and have backup for all your data on that partition, simply boot from the 64-bit DVD and install 64-bit Win7 on that volume.  It will wipe out the 32-bit installation - and everything else on that volume -
 in the process.


If you have already installed programs and saved data on that partition, be sure you have original media to re-install, and have your data safely backed up somewhere.  Then install 64-bit Win7 as above, re-install your apps, and restore your data.


Since 32-bit Win7 is already running, you can use WET (Windows Easy Transfer) to help with your migration; WET is included in every Win7 package and will also run on WinXP and Vista.  Use it first on your existing system to save files, data and settings
 to a location that will be accessible from the new system:  USB flash drive, another partition on your hard drive, etc. Then run it again from your new system to retrieve all that information into your 64-bit Win7.




I have not validated my install...




Note that the term is "activate", not "validate".  As you probably noticed, the same Product Key is used to activate either 32-bit or 64-bit Win7.  You should have no problem activating via the Internet; if you do, simply call the toll-free voice number
 that will appear and explain.
