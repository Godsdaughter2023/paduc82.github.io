stefan blom say:

In the Styles pane, only paragraph, linked, and character styles display; this is by design. You can apply a list style via the Apply Styles dialog, as you have noticed. Alternatively, use Home tab | Multilevel List. List styles can be found in the "List Styles" section of the drop down.

What shows in the Styles pane is determined by the settings in the Style Pane Options dialog box. Click the Options link at the bottom of the Styles pane (Ctrl+Alt+Shift+S). "In use" includes style that have been used at some point in the document, even if they are not currently applied to text. Instead of "In use," you can select "Recommended" and choose which styles you want shown or hidden in the pane.
