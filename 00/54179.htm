Which files on c drive are deletable?

how to find which files on c are used by my PC and arernt deletable?
gerry c j cornell say:

Yes you have correct guessed the software I recommend you use to increase the size of your C partition. I would take 25gb from D and add it to C.

 

To increase the size of C you first need to reduce the size of D. This enables you to increase the size of C by taking up the space D has vacated. When you resize partitions it is all about before and after. Thus in the simple operation of D giving up space to C when resizing or reducing the size of D the free space needs to be created before D. This enables you to resize C by expanding into the free space you now have after C. When you have done the above apply the changes.

 

This page from the Easeus web site explains visually.

http://www.partition-tool.com/easeus-partition-manager/help/resizing-and-moving-partition.htm

 

You then need to move your pagefile back to C from D. However you should first clean up. Select Start, All Programs, Accessories, System Tools, Disk CleanUp to empty your Recycle Bin, remove Temporary Internet Files and other selected options. I also recommend you click on the More Options tab, System Restore and remove all but the latest System Restore point.

 

To move the pagefile select Start, Control Panel, System, Advanced System settings, Advanced, Performance Settings, Advanced tab, Virtual memory –Change. Place the cursor on Drive C and check the box before system managed file. And click Set. Next place the cursor on Drive D and check the box before No Paging file. And click Set. Finally click, OK, Apply and OK. Exit and restart your computer.

Puo' provare questa procedura

Il primo passo è accedere alla pagina di configurazione del Router.

Esistono Router che hanno uno specifico software di configurazione, altrimenti aprite il vostro browser web ( es. IExplorer o Firefox ecc..) e, dove inserite gli indirizzi web, digitate l'indirizzo IP del vostro router ( solitamente è 192.168.1.1 oppure 192.168.0.1 ).


Il secondo passo è aprire le porte per l'indirizzo IP della vostra Xbox.

Andate sulla pagina del router dove potete aggiungere le porte, di solito appare sul menu di configurazione del router, potrebbe esserci scritto "virtual server" oppure "NAT" ecc..

Questa ad esempio è la pagina delle porte per il mio router Linksys:



L'elenco delle porte da aprire è questo:

Dal sito ufficiale Xbox Live

80
 TCP
88
 UDP
3074
TCP
3074
 UDP
53
 TCP
53
 UDP


Bisogna specificare quindi:


  1. numero della porta
  2. il protocollo [TCP o UDP]
  3. l'indirizzo IP della vostra XBOX

Scegliete l'indirizzo IP della vostra xbox a piacere. L'importante è che sia uguale all'indirizzo IP del vostro router ad esclusione dell'ultima cifra, ad esempio se il vostro router ha come indirizzo 192.168.1.1 potete usare benissimo 192.168.1.7 o 192.168.1.35 ecc..

Il terzo passo consiste nel settare un indirizzo statico sulla vostra xbox. Andate sul menu di impostazioni di rete - impostazioni IP e scrivete:


  1. Indirizzo IP su cui avete sbloccato le porte nel passo 2
  2. subnet mask: 255.255.255.0
  3. gateway: Indirizzo IP del proprio router (es. 192.168.1.1)
