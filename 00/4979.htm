graham mayor say:
You can save messages as msg format using a simple Outlook macro e.g. as fiollows, which will save selected messages in the folder c:\path\ (change to suit).

The filenaming is the biggest issue. This macro uses a combination of name, subject date and time to avoid as far as possible the problem of duplicate file names. Illegal filename characters are changed for hyphens.

Sub SaveSelectedMessages()
Dim olItem As Outlook.MailItem
Dim fName As String
Dim fPath As String
    fPath = "C:\path\"
    For Each olItem In ActiveExplorer.Selection
        fName = Format(olItem.ReceivedTime, "yyyymmdd") & Chr(32) & _
                Format(olItem.ReceivedTime, "HH.MM") & Chr(32) & _
                olItem.SenderName & " - " & olItem.Subject & ".msg"
        fName = Replace(fName, Chr(58) & Chr(41), "")
        fName = Replace(fName, Chr(58) & Chr(40), "")
        fName = Replace(fName, Chr(34), "-")
        fName = Replace(fName, Chr(42), "-")
        fName = Replace(fName, Chr(47), "-")
        fName = Replace(fName, Chr(58), "-")
        fName = Replace(fName, Chr(60), "-")
        fName = Replace(fName, Chr(62), "-")
        fName = Replace(fName, Chr(63), "-")
        fName = Replace(fName, Chr(124), "-")
        olItem.SaveAs fPath & fName
    Next olItem
    Set olItem = Nothing
End Sub

Hi Alex,
you can do that you want
using a macro.
Your data are in columns A:G
I assume that in row 1 are headings, on sheet1.

Export data in a new sheet
try this..

1) ALT+F11 to open visual basic
2) Insert > Module
and paste the code below to the right

Sub macro1()
Dim ws1 As Worksheet, newWS As Worksheet
Dim r1 As Long, c As Long, i As Integer
Set ws1 = Sheets("Sheet1") '<< change
Application.ScreenUpdating = False
Application.DisplayAlerts = False
For Each ws In Worksheets
If ws.Name = "ConvertData" Then ws.Delete
Next
Application.DisplayAlerts = True
Set newWS = Sheets.Add(before:=Sheets(1))
newWS.Name = "ConvertData"
r1 = ws1.Cells(Rows.Count, 1).End(xlUp).Row
ws1.Range("A1:G" & r1).Copy Destination:=newWS.Range("A1")
For i = r1 To 2 Step -1
If newWS.Cells(i, 1) = "" Then newWS.Rows(i).Delete
Next i
r1 = newWS.Cells(Rows.Count, 1).End(xlUp).Row
For i = 2 To r1
c = newWS.Cells(i, Columns.Count).End(xlToLeft).Column
If c = 7 Or c = 6 Then GoTo myNext
newWS.Range(Cells(i + 1, 1), Cells(i + 7 - c, 1)).Copy
newWS.Cells(i, c + 1).PasteSpecial , Paste:=xlPasteValues, Transpose:=True
newWS.Rows(i + 1 & ":" & i + 7 - c).Delete
myNext:
Next i
newWS.Columns("A:G").AutoFit
newWS.Range("A1").Select
Application.ScreenUpdating = True
End Sub

3) close visual basic
4) ALT+F8 select macro1 from the list and tick the run button.

charles kenyon say:

You are understandably confusing two different features.

Horizontal / vertical scrolling has to do with how you are viewing different pages on a document.

Changing that likely changes it for all documents but does not change how documents print.

Portrait//Landscape determines how text is placed on a page when the document is printed and also changes the appearance of the document on your screen. That is a setting for a single document (or part of a document). Changing it in one document does not change how older documents are viewed or printed.
