william zeng msft say:

Hi Nordmann1087,

Since we can't manage the properties of group site from SharePoint Admin center, however, we can check the properties by using PowerShell.

1) Type the following command into the SharePoint Online Management Shell to connect to the service after input your admin credential:

Connect-SPOService -Url https://organizationdomain-admin.sharepoint.com -credential youradminaccount

2) Type the following command to display the group site properties

Get-SPOSite -Identity https://groupsiteURL | fl

Here is an example from my side.

Get-SPOSite -Identity https://mydomain.sharepoint.com/sites/PlanA | fl

Please refer to the following articles for your reference if you're not familiar with PowerShell.

SharePoint Online Management Shell
