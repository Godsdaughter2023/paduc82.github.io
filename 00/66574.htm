File Explorer Crash when minimizing folder

xsdnide say:

I was able to get a workaround to this issue. My issue was that the explorer window started maximized and then restoring the window, or dragging it risized the window to a very small window with the links in the Navigation Pane kept on blinking.

1. Launch the explorer window - it opens up as a fully maximized window

2. Click on View tab > Panes group > Click on the Navigation Pane drop down and uncheck "Navigation Pane"

3. Resize the window, it will again resize to a very small window but would not blink / freeze

4. Manually resize the window to a size you like

5. Enable the "Navigation Pane" again

Worked for me, so may be others can benefit as well.
