Hi Artah,

 

Generally, when you move public folders to Office 365 cloud, you don’t need to change the HoldForMigration parameter.

 

Based on the error message, if you are doing a migration and the ‘PublicFoldersEnabled’ is set to ‘Remote’ and the LockedForMigration is set to True, you need to start over again.

 

If you don’t do the migration, you need to follow the steps below to create new public folders. Because the issue is related to the HoldForMigration parameter. Here are the steps:

1. Run the following cmdlet to return and delete public folder mailboxes of the Secondary Hierarchy type:

Get-Mailbox -PublicFolder | where {$_.IsRootPublicFolderMailbox -eq $False} | Remove-Mailbox -PublicFolder

2. Run the following cmdlet to return and delete public folder mailbox of the Primary Hierarchy type:

Get-Mailbox -PublicFolder | where {$_.IsRootPublicFolderMailbox -eq $true} | Remove-Mailbox -PublicFolder

3. Create a public folder mailbox without specifying the HoldForMigration parameter by running the following cmdlet:

New-Mailbox -PublicFolder -Database "Mailbox Database Name" -Name "Public Folder Mailbox Name"

 

Note:  If you want to run the commands above, please backup all data in your public folders. All data will be deleted. Please reconfirm the actions before running them.

Show Form by double clicking cell - Microsoft Community

roger on excel say:

Show Form by double clicking cell

Rather than using a command button in my spreadsheet, is there a way to double click or left right click on a spreadsheet cell to make a form appear?

 

The cells to double click on are in the range B20:B30.  The same cell range needs to be active for this event across a number of sheets.

 

Any ideas?

 

Thanks,  Roger

jllatham say:

Since you want this to take place across multiple worksheets, you'll want to use the
Workbook_SheetBeforeDoubleClick
event processor.  When it is triggered (by a double-click in any cell on any sheet in the workbook), it gives you 3 values to work with:
Sh represents the sheet it took place on,
Target represents the cell it took place in, and
Cancel which is a flag you can set to True to cancel the double-click.

This page shows how to put code into the various places it can reside in an Excel VBA Project:
http://www.contextures.com/xlvba01.html For the code below, refer to the Copy Excel VBA Code to a Workbook Module

The code would look something like this, which responds to a double-click in B20:B30 on any sheet by displaying UserForm1:

Private Sub Workbook_SheetBeforeDoubleClick(ByVal Sh As Object, ByVal Target As Range, Cancel As Boolean)
  Dim iSect As Range
  Set iSect = Application.Intersect(Target, Sh.Range("B20:B30"))
  If iSect Is Nothing Then
    'leave Cancel alone, exit sub acting like regular double-click
    Exit Sub
  End If
  Cancel = True
  UserForm1.Show
End Sub

You can pick and choose which sheets to allow it to take place on like this:

Private Sub Workbook_SheetBeforeDoubleClick(ByVal Sh As Object, ByVal Target As Range, Cancel As Boolean)
  Dim iSect As Range
  Set iSect = Application.Intersect(Target, Sh.Range("B20:B30"))
  If iSect Is Nothing Then
    'leave Cancel alone, exit sub acting like regular double-click
    Exit Sub
  End If
  Select Case Sh.Name
    Case "Sheet1", "Sheet2"
      Cancel = True
      MsgBox "UserForm1.Show"
    Case Else
      'again leave Cancel alone, exit acting like regular double-click
      Exit Sub
  End Select
End Sub

 

hansv mvp say:

Right-click the Sheet tab.

Select View Code from the context menu.

Enter or copy the following code into the module that appears:

 

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
  If Not Intersect(Range("B20:B30"), Target) Is Nothing Then
    ' Cancel the default action
    Cancel = True
    ' Display a userform
    UserForm1.Show
  End If
End Sub

 

Instead of Worksheet_BeforeDoubleClick, you can use Worksheet_BeforeRightClick.

If you want to do this for all sheets except a few, it is more efficient to use the Workbook_SheetBeforeDoubleClick (or Workbook_SheetBeforeRightClick) event in the ThisWorkbook module instead of a procedure in each worksheet module:

 

Private Sub Workbook_SheetBeforeDoubleClick(ByVal Sh As Object, ByVal Target As Range, Cancel As Boolean)
  Select Case Sh.Name
    Case "ThisSheet", "ThatSheet", "OtherSheet"
      ' These are the exceptions, let Excel handle it
    Case Else
      ' Check active cell
      If Not Intersect(Sh.Range("B20:B30"), Target) Is Nothing Then
        ' Cancel the default action
        Cancel = True
        ' Display a userform
        UserForm1.Show
      End If
  End Select
End Sub
