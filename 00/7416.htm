Solution to MS Access error: "You must specify the same number of fields when you set the LinkChildFields and LinkMasterFields properties".

I am posting this obscure solution in case it helps somebody else.

Customer had the Microsoft Access error: "You must specify the same number of fields when you set the LinkChildFields and LinkMasterFields properties" when opening several parent-child forms on an MDB database that works perfectly on various versions of full Access and Access Runtime on many other machines.

Resolution: In Windows Regional & Language Settings, Decimal and List Separator were both a full stop.  Hence Access could not separate its field lists properly.  Changed to decimal full stop and List Separator comma.  (Digit Grouping changed from space to comma –probably not necessary).

Note that this was a setup error by the IT company. It had to have been done manually.  Default Windows South African Regional settings are: Comma decimal, space Digit Grouping, and semicolon List Separator. When we set up a machine, we use UK settings, which work better.
