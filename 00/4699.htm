ron coderre say: If you want a function that calculates the number of merged cells that encompass a specific cell,
you'll need to create a User Defined Function (UDF) using Visual Basic for Applications (VBA).

Here's one way:
• ALT+F11...to open the Visual Basic Editor
• Right-click on your workbook's name in the VBA - Projects window
...Select: Insert Module
• Copy the below VBA code and paste it into that module:

Function MergedCellsCount(rRange As Range) As Integer
    Application.Volatile
    MergedCellsCount = rRange.MergeArea.Cells.Count
End Function

Using the below example of the range A1:A17 (A1, A2...etc only describes the cell addresses)

A1 Alpha
A2
A3
A4 Bravo
A5
A6
A7 Charlie
A8
A9
A10
A11 Delta
A12
A13
A14 Echo
A15 Foxtrot
A16 Golf
A17 Hotel

Here's how to use the formula:
C1: Alpha
D1: =MergedCellsCount(INDEX($A$1:$A$17,MATCH(C1,$A$1:$A$17,0)))

In the above example, the formula returns: 3
because the value "Alpha" is part of a 3-cell merged area.

This section of the formula: INDEX($A$1:$A$17,MATCH(C1,$A$1:$A$17,0))
returns a reference to the single cell that contains "Alpha"

If you change C1 to "Golf"...without the quotes
the formula will return: 1

a. user say:

Hi,

Thank you for posting your query on Microsoft Community.

You could either login to https://outlook.com/ and then delete all the messages manually or you could try the steps below which will help you delete all the messages from a device or app.

Login to https://outlook.com/ through the web browser.

If you are upgraded to Outlook.com (preview) interface, then follow the steps.

1. Click on the Gear (settings) icon and select Options.

2. On the left pane, expand Mail > Accounts and select POP and IMAP.

3. Under devices and app that use POP can be set to delete messages from Outlook on the web after download, select the option Let apps and devices delete messages from Outlook on the web.

If you are still in Outlook.com interface, then follow the steps below.

1. Click on the Gear (settings) icon and select Options.

2. Under Managing your account, click on Connect devices and apps with POP.

3. Under deleting downloaded messages, select Do what the device or app says- if its set to delete messages, delete them.

4. Now, click on Save.

However, make sure you manage your settings correctly on the devices they may lead to deleting all your emails from the server if the settings are not set correctly..

Hope this helps. Please respond if you have further related queries.
