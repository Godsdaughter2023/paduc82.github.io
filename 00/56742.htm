It seems like the windows media player database is not getting updated, we can perform the below steps to rebuild the database. Here is how:

a. Close Microsoft Media Player.

c. Type the following into the textbox:

d. Click OK and a folder showing you your library files will appear
e. Erase all the files that end with .wmdb
f. Open Microsoft Media Player
g. Push F3
h. Select the locations for your music files
