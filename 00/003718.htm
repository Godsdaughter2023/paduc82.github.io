stefan blom say:
Set up your numbering format so that each level is associated with a unique paragraph style and then apply styles to text as appropriate. See http://www.shaunakelly.com/word/numbering/numbering20072010.html. Note that Shauna's examples make use of the built-in heading styles, but if you don't want to set up numbered headings, you should associate other styles to your list levels.
