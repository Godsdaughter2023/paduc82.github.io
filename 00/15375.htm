If you are going to put a memory card into N96 it would be good to format it before putting it to N96 specially if you used this card with another phone.

Start from a "clean" memory card. Don't use the same memory card in several phones as installations and other files are incompatible

FORMAT the memory card with the phone. Even if the card comes pre-formatted when New.

non-volatile worksheet function to find last used row (without - Microsoft Community

kergga say:

non-volatile worksheet function to find last used row (without VBA)?

The title says it all; I'm looking for a non-volatile method to find the last used row on a worksheet (Columns A: CZ). Any column might include blanks, so COUNT or COUNTA doesn't help. Each row will have at least one cell with numeric or text value in at least one column.

 

This will be used to build a dynamic named range (A1:CZ??), which will then be the source of hundreds of other calculations. I don't want to change hardcoded ranges each month, and while I'm relatively skilled with VBA, I'd like to avoid VBA for this particular project.

mike h.. say:

Hi,

 

These are ARRAY formula so enter with Ctrl+Shift+Enter.

 

To find the last row use this. Note it currently works up to row 10,000 is that enough? No. increase the number in the formula or preferably make it smaller for speed.

 

=MAX(ROW(1:10000)*(A1:CZ10000<>""))

 

To incorporate that into a formula to find the MAX in that range we could use

 

=MAX(INDIRECT("A1:CZ"&MAX(ROW(1:10000)*(A1:CZ10000<>""))))

 

 

EDIT.... I should have added the formula must be outside of the range being searched or you'll get a circular reference.

 

EDIT Again.... If as you say you have hundreds  of formula that are going to use this dynamic last row then on reflection I wouldn't implement my second formula that way. I'd calculate the last row once and reference that cell in other formula like this

 

=MAX(INDIRECT("A1:CZ"&DB1))

 

Or even more intuative give the cell where you calculate the last row a name such as lastRow and use this

 

=MAX(INDIRECT("A1:CZ"&LastRow))
