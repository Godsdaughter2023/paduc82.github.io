Why can't I get this to work?

doug robbins - mvp office apps & services (word) say:

You will need bookmark the word in B2 and then use the following formula in B5 (assumes that your table is 2 rows by 5 columns as in Word the columns are A, B, C, etc

 

{ IF { REF B2 } = "Refund" { = B3 - B4 } }

 

The above assumes that the bookmark name used is B2.

 

You must use Ctrl+F9 to insert each pair of field delimiters { } and you use Alt+F9 to toggle off their display.

 

You would need to be very careful that if the word in B2 is changed that the bookmark is not destroyed when it is changed

doug robbins - mvp office apps & services (word) say:

Use

 

{ IF { REF B2 } = "Refund" { = B3 - B4 \# "0.00" } }

The article at http://wordfaqs.mvps.org/LetterheadRibbon.htm explains how to set up letterhead from scratch. The section http://wordfaqs.mvps.org/LetterheadRibbon.htm#MultiPage explains how to create letterhead that will automatically be formatted correctly for the second and subsequent pages. When dealing with graphics from an outside company, it's better to know what to ask for to make this process work correctly. If there's a logo involved, I would usually prefer to have just the logo and create the rest of the letterhead (headers and footers) within Word, primarily because this reduces the file size (it does require, however, that you have the font the artist specifies). Insert these images into the header/footer with Square wrapping and position set to Left and Top for the header, Left and Bottom for the footer, both Relative to Page. This will allow the header and footer to remain properly placed even if you change the side margins.

john korchok say:

There is a command called ReviewingPaneHorizontal that will do this. You can access it by creating a keyboard shortcut, or by adding a button to a menu. To add a keyboard shortcut:

  1. In Word, select Tools>Customize Keyboard.
  2. In the Categories list, scroll down to All Commands and select it.
  3. In the Commands list, scroll down to ReviewingPaneHorizontal and select it.
  4. Click in the Press new keyboard shortcut field and press your preferred key combination.
  5. Click on Assign, then OK out.

To add a button to a menu:

  1. In Word, choose View>Toolbars>Customize Toolbars and Menus...
  2. Click on the Commands tab.
  3. In the Categories list, choose All Commands.
  4. In the Commands list, select ReviewingPaneHorizontal.
  5. Drag it to the Quick Access Toolbar or one of the menus in the second listing below the actual menu bar. OK out.
