WinSxS folder contains a complete version history of your system. You can uninstall updates and service packs to any state. WinSxs also has other purposes.

 

The current version of a system file is hardlinked to System32 folder (or whereever the right place is). A hard link is a filename. All files have one hardlink but can have as many as they want. So Svchost.exe has two filenames - c:\...\WinSxs\...\svchost.exe and c:\...\System32\svchost.exe. Both are equal. File data is deleted when the last filename is deleted.
