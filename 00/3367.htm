List all combinations of a list of numbers

I can use the COMBIN function in excel to determine the total possible number of groups for a given number of items.  Has anyone written any code to list all these combinations in excel?
mike h.. say:

Hi,

I have a couple of versions of code which I wrote to do that. Here's one that perms any 3 from x

Put your numbers (or text)  in Col A starting in A1, no gaps; and run the code. The list is output to Col B.

Sub Permutations()
'Perm any 3 from x
'Numbers/Text in Column A
Dim last As Long, i As Long, j As Long, k As Long
Dim l As Long
last = Cells(Rows.Count, "A").End(xlUp).Row
    For i = 1 To last - 2
        For j = i + 1 To last - 1
            For k = j + 1 To last
                Cells(l + 1, 2) = Cells(i, 1) & ", " & Cells(j, 1) & ", " & Cells(k, 1)
                l = l + 1
            Next
        Next
    Next
 End Sub
