  7. Drag your mouse to select the PowerPoint window.
  8. Click on Record.
  9. Run your slideshow. When you're finished, click on the square dot inside a circle on the OS X menu bar to stop QuickTime. Save the video file.

cpitchford say:

winscp will let you "drag and drop" files onto and off your phone, but be very careful with it..

 

There isn't much of a graphical alternative to the command line in Linux/Maemo.. at least not for the off-the-path stuff.. if you need to do something strange, the command line is pretty much the only way to go.. but as for getting files onto and off your device, winscp (which uses sftp or scp) will really help simplify the process..

 

remember though, the permissions inside Unix are a fair bit more critical than windows..

 

There are three permissions you need to worry about

 

read: user can read a file or see the contents of a directory

write: user can write/change a file or add/remove a file from a directory

exec: user can RUN this file or change into a directory

 

Unlike windows where the file extension defines whether a program can be run or not (ie .exe) in unix it is the file permission..

 

blah.sh is a regular file

 

chmod +x blah.sh

activates the eXec permission and now blah.sh is a program that can be run

 

Copying files between windows and maemo (even using winscp) does not take these permissions into account.. so be careful!

 

That might not make much sense now, but the more you use it, the more it will seem relevant :)
