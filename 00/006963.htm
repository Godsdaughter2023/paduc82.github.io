hansv mvp say:

Keep in mind that Excel is NOT a word-processing application, and that autofit for wrapped text is far from perfect: Excel frequently allocates height for an unneeded extra line, and the on-screen display does not always correspond to printed output. So for what it's worth:

You could use the Worksheet_Calculate event in the worksheet module to autofit the rows. For example:

Private Sub Worksheet_Calculate()
    Range("C3:H8").EntireRow.AutoFit
End Sub

where C3:H8 is the 6x6 cells table.

The table has columns: Type, ID, Date, Amount

I am using the Date as the column headings and the Type and ID as row headings.

I need to group by and sub total the amounts when the Type value changes.

Any help is appreciated, Thanks.

qa guy commelec say:

Use a union query ahead of the crosstab query.  You did not say what the datatype of ID.  You will need to use a high number instead of  "zz_SubTotal" AS ID if it is a number.

       YourTableSum --
SELECT YourTable.Type, YourTable.ID, YourTable.Date, YourTable.Amount
FROM YourTable
UNION ALL SELECT YourTable.Type, "zz_SubTotal" AS ID, YourTable.Date, Sum(YourTable.Amount) AS SumOfAmount
FROM YourTable
GROUP BY YourTable.Type, "zz_SubTotal", YourTable.Date;

TRANSFORM Sum(YourTableSum.[Amount]) AS SumOfAmount
SELECT YourTableSum.[Type], YourTableSum.[ID]
FROM YourTableSum
GROUP BY YourTableSum.[Type], YourTableSum.[ID]
PIVOT Format([Date],"Short Date");
