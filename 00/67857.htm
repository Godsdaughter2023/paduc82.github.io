Uninstalling the program won't affect user data, so if messages are missing after re-installing, something else has happened.

On the other hand, removing and then re-adding a mail account will remove all the account's folders and their contents. Is this what your customer did, perhaps?

Which version of the program are we talking about, on which OS? On Windows 7, you may be able to use the Previous versions feature to find an older copy of the store folder:
 

  1. In Windows Live Mail, press Ctrl-Shift-O for Options.
  2. On the Advanced tab, click Maintenance and then Store folder.
  3. Copy the path shown into the start search box and press Enter.
  4. Right-click on the folder and select Restore previous versions. 
  5. If you can locate a previous version of your store folder, Open it and then locate the missing folders.
  6. Copy the folders into a temporary folder on your desktop
  7. Use the File > Import messages function (Alt-F I) in Windows Live Mail to import the temporary folder.

Clean Up System

Left click "start" then "all programs" then "accessories" then "system tools"

Select which drive you want to clean up and click "OK" (after the process is completed, come back and go to the next drive)

De-fragment Drives:
