And of course, it would have been just as simple to fix the spelling in the Template you have :-)

Spelling is controlled by the "Language" specified in the text.  There are 29 "flavours" of English: specify one that your copy of Word doesn't have, and the Spell Checker ceases to work.  There's also a "language" of "No Proofing" that exists purely to prevent the speller from working when you don't want it to.

All you need to do is Open your template, then go to Format>Style>Normal>Modify>Language and set it to the language you want it to use.

For good measure, select all the text in the template (even if there is apparently "none, there is a paragraph mark in there, select it) and apply the Normal style to it.

Then save out and quit Word.  Should work on all new documents created from the template in future.

For existing documents, you will need to repeat that "cure" on each of the documents themselves (although you can get away with simply "Select All" and "Tools>Language..." if you don;t intend to keep the document in edit for years...).

doug robbins - mvp office apps & services (word) say:

Using

 

Dim fd As FileDialog
Dim strfiles As String
Dim arrfiles As Variant
Dim i As Long
Set fd = Application.FileDialog(msoFileDialogFilePicker)
strfiles = ""
With fd
    .Title = "Select the required files."
    .AllowMultiSelect = True
    If .Show = -1 Then
        For i = 1 To .SelectedItems.Count
            strfiles = strfiles & "|" & .SelectedItems(i)
        Next i
    Else
        MsgBox "You did not select a workbook."
        Exit Sub
    End If
End With
strfiles = Mid(strfiles, 2)
arrfiles = Split(strfiles, "|")

arrfiles will contain the selected files
