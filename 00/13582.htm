Conditional Formatting - Microsoft Community

lightbulb2 say:

Conditional Formatting

Is there a way for me to format something so that if a string of text/numbers in a cell found is found in another table (exact match), then it will turn a certain color?  Basically if it is found somewhere in a certain column of the table it will turn the cell green, otherwise it will turn it red?
t. valko say:

Set the default color to red

You can use conditional formatting with a formula of the form

=NOT(ISERROR(MATCH(A1,OtherSheet!$C$1:$C$100,0)))

where A1 is the cell you're trying to format, and OtherSheet!$C$1:$C$100 is the column in which you want to look for the value.


In Excel 2003 you can't directly refer to another sheet in conditional formatting. You can refer to another sheet if you use a named range.

=NOT(ISERROR(MATCH(A1,MyTable,0)))

Also, you can eliminate the NOT(ISERROR functions.

=MATCH(A1,MyTable,0)

john korchok say:

That option is intended to adjust the display of bitmap images, not OLE objects.

When you insert as an object, the entire workbook is included. Double-click on it to open it and scroll to other portions of the sheet or other worksheets. When you insert a file as an object, you cede the display decisions to the program, so you don't get to choose which portion of the workbook displays. 

Instead, open the file in Excel and select the area you want to display in PowerPoint. In PowerPoint, choose Home>Paste Special and paste as an Microsoft Excel Worksheet Object. The area you selected will display and you can still double-click on the embedded workbook to scroll or view other sheets.
