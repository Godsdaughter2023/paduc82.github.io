Skype 8 running on Windows 7

In Application Settings, is it better to have "Launch Skype in the background" turned ON or OFF?
ruwim.b say:

This has nothing to do with receiving or not receiving notifications.

 

When the option "Launch Skype in the background" is turned on, Skype will start when Windows starts, but will be minimized, i.e. will run in the background only without the visible window.

 

When this option is turned off, Skype will start maximized, i.e. with the visible Skype window.
