koby koren say:

Hi Porter,

Please follow these steps to get data from the most recent file in a folder:

  1. In the data ribbon select New Query->From File->From Folder
  2. Browse to the required folder and click OK.
  3. Once the query preview pops up click Edit
  4. Power Query displays a list of files in the folder. To get data from the latest one sort the list by Date Created or Date modified
  5. Click Keep Rows-> Keep Top Rows-> In ‘Number of Rows’ type 1 to keep the latest file only and click OK
  6. On the content column click Binary
  7. Continue analyzing your data

Hope that helps.

    That is not possible, because Excel opens the file exclusive, means it is write protected.

    A batch file can change the attributes, but only when the file is closed.

    The only way is to write an EXE file that does the work for you and assign it with the .XLS file extension within the registry (open the XLS files with your file instead of Excel).

    After the EXE has changed the attribute, it should redirect the file open call to Excel.

    Master Formula, want to create a cell where I can apply a formula that - Microsoft Community

    wix737 say:

    Master Formula, want to create a cell where I can apply a formula that can chnage the rate of euro in all the sheets

    I have 15 sheets with prices. All the prices were in Euros so I created a seperate column next to all price columns in all the sheets and applied the formula for price conversion to GBP. Now I have the furmula for price conversion on every sheet in one cell which I draged down to apply the formula to the whole column. I want to create a cell where I can apply a formula that can chnage the rate of euro in all the sheets. Is that possible?
    mike h.. say:

    Hi,

     

    Pick the cell for your exchange rate and right click, name a range and in the popup give the cell a meaningful name such as EuroBPxchange. Ensure that in the popup the scope of the named range is set to 'Workbook' and then OK out.

     

    Enter the current exchange rate in that cell then on any worksheet you can use the formula

     

    =A1*EuroBPxchange

     

    In that example A1 will then be multiplied by the rate in the named range. The neat thing with this method is that if you change the exchange rate in the cell then all the conversion formula re-calculate.
