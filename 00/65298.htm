Info about Picture Manager
If you resize a photo with Picture Manager and
save it when you see the dialog that says:

"There are changes that have not been saved"

The original photo will be overwritten (replaced)
with the rezized version...IOW the original will
be unrecoverable.

the resized version and left click the Save button.

When you close Picture Manager and see the
dialog:

"There are changes that have not been saved"

Left click the 'Don't Save' button. Now you will
have both versions.

factory reset - Microsoft Community

ianjnicholas say:

factory reset

how do I do a factory reset on my laptop ?

aileen alf say:

Hi Ian,

You may use the recovery partition to factory reset your Windows device. A partition is a portion of the hard drive that is sectioned off. A recovery partition contains all the recovery setup files necessary for a clean refresh of the operating system. Starting a recovery via the recovery partition usually means that you have to press one of the F keys (such as F1-F12), depending on the make and model of your device, the moment after you hit the power button. Doing so tells the BIOS that you’d like to run the recovery setup rather than load your Windows installation. Then, follow the on-screen instructions to complete the recovery.

If your computer doesn't have a recovery partition, but it did come with the operating system pre-installed, it should have come with recovery discs. Recovery disks contain all of the recovery setup data that would normally be stored in a recovery partition. Configure your laptop to boot from the recovery disk, then follow the on-screen instructions to complete the recovery.

Let us know if you need further assistance.
