rich007a say:

So you could use:

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    Dim L As Long
    If ActiveCell.HasFormula Then
        L = Len(ActiveCell.Formula)
    Else
        L = Len(ActiveCell.Value)
    End If
   
    Select Case L
        Case Is < 100: Application.FormulaBarHeight = 1
        Case Is < 200: Application.FormulaBarHeight = 2
        Case Is < 300: Application.FormulaBarHeight = 3
        Case Is < 400: Application.FormulaBarHeight = 4
        Case Is < 500: Application.FormulaBarHeight = 5
        Case Else: Application.FormulaBarHeight = 6
    End Select
End Sub

Note you will need to calibrate the boundaries (100, 200, 300, etc.) depending on your screen size and font size settings, etc.  You'd want to always have your Excel window maximized.

jeeped say:
... need the formula bar to autosize to show all 12 rows of text, then resize to the next cell that might only have 1 row of text

The formula bar does not auto-size according to the value or formula within the current cell and this behavior is by design. If it did, the screen would be constantly flashing if someone was using one of the arrow keys or Tab to move around the worksheet, reducing and expanding the displayed area of the active worksheet while conversely expanding and contracting the number of lines in the formula bar.

You can expand the number of visible lines in the formula bar to the last remembered number of visible lines by clicking the downwards chevron (i.e. guillemet) at the right end of the formula bar. Click again (now a vertical chevron) to return to a single line. If you manually adjust the height of the formula bar, it will be remembered the next time you click the chevron.

You can manually auto-size the formula by double-clicking the formula bar's bottom border once the cursor has changed to a re-sizing cursor. This will expand the formula bar to enough lines to display whatever it contains. The calculation of the number of necessary lines is is performed internally. When recorded as a macro, the resulting code is a set number, not an .AutoSize command, so it does not make sense setting up code to perform this.

Your best option seems to be double-clicking the formula bar's lower border.
