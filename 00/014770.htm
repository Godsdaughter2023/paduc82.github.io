brian tillman [outlook mvp 2007-2019] say:
I don't consider those aliases in the strict sense of what an email alias is.  When two addresses are aliases, it means that messages sent to either address are delivered to the same mailbox.  If you create a send-only or send/receive account in your Outlook.com profile, you are not using the "Create an alias" process.  It is true, however, that if you do create a send-only or send/receive account, they will appear in the Options>More mail settings>Your email accounts, but they won't be listed as aliases there.  They will be listed as aliases, however, if you visit https://account.live.com/names/Manage.  That doesn't really make them aliases, though, regardless of the label on Outlook.com.

Y Axis (auto-scale with macro) - Microsoft Community

chee kein say:

Y Axis (auto-scale with macro)

Hi, I chanced upon the below macro in microsoft discussion group. My question is that if i have a secondary Y axis and I would like the range to be simialar to the primary Y axis, is it possible? If so, can anyone advise me what should be change to reflect it? Thanks.

Sub AutoScaleYAxes()
    Dim ValuesArray(), SeriesValues As Variant
    Dim Ctr As Integer, TotCtr As Integer
    With ActiveChart
        For Each X In .SeriesCollection
            SeriesValues = X.Values
            ReDim Preserve ValuesArray(1 To TotCtr + UBound(SeriesValues))
            For Ctr = 1 To UBound(SeriesValues)
                ValuesArray(Ctr + TotCtr) = SeriesValues(Ctr)
            Next
            TotCtr = TotCtr + UBound(SeriesValues)
        Next
        .Axes(xlValue).MinimumScaleIsAuto = True
        .Axes(xlValue).MaximumScaleIsAuto = True
        .Axes(xlValue).MinimumScale = Application.Min(ValuesArray)
        .Axes(xlValue).MaximumScale = Application.Max(ValuesArray)
    End With
End Sub
