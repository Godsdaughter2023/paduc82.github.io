Sub macro_01()
Dim ws As Worksheet
Set ws = Sheets("Sheet1") '<<< change sht name
Dim r As Long
r = ws.Cells(Rows.Count, "A").End(xlUp).Row
Dim wbName
wbName = ThisWorkbook.Name
Select Case wbName
Case Is = "ABCD.xls"
ws.Range("A1").Resize(, 3).EntireColumn.Insert
ws.Range("C2:C" & r).Value = "xyz; abc12"
GoTo mynext
Case Is = "Daily.xls"

ws.Range("B1").Value = "Type of Revision"

jamcl say: Got it working.  I uninstalled both ActiveSync and the Toshiba Bluetooth stack, restarted my machine and deleted all pairings within Activesync on the phone.  I then reinstalled ActiveSync (v4.5), and installed BlueSoleil (http://www.bluesoleil.com/download/index.asp?topic=bluesoleil_edr_en - great little application!).  In BlueSoleil (with the bluetooth dongle inserted), check My Services>Properties, and note the port number for COM A - it was COM10 on mine.  Make sure connections via this port are enabled in the ActiveSync connection settings.  Now just select Connect Via Bluetooth on the phone's ActiveSync, and a partnership should be set up.  Worked for me!
