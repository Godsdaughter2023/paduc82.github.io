I would like the hyperlink to just take you to the next consecutive slide ie. slide 2 once pasted.

How would I do this?

john sr wilson say:
One way would be to use a macro to locate all links which go to the next slide and change the link to "NEXT" as Shyam suggests. This is totally "top of head" code so use on a COPY. After running the code check in Insert > Action that the links are now NEXT SLIDE

If you do not know what to do with the code see here

Sub switchlinks()
Dim ohl As Hyperlink
Dim raySub() As String
Dim osld As Slide
For Each osld In ActivePresentation.Slides
If osld.SlideIndex < ActivePresentation.Slides.Count Then
For Each ohl In osld.Hyperlinks
raySub = Split(ohl.SubAddress, ",")
If raySub(0) <> "" Then
If CLng(raySub(0)) = ActivePresentation.Slides(osld.SlideIndex + 1).SlideID Then
ohl.Address = ""
ohl.SubAddress = "-1,-1,NEXT"
End If
End If
Next ohl
End If
Next osld
End Sub
