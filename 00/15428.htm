but I think you would be better off using ElseIf throughout you code:
 
Private Sub Worksheet_Change(ByVal Target As Range)
    Set MyPlage = Range("B")    'SEE NOTE BELOW
    For Each Cell In MyPlage
   
        If Cell.Value = "FOI" Then
            Cell.Interior.ColorIndex = 3
        ElseIf Cell.Value = "Fact Check" Then
            Cell.Interior.ColorIndex = 4
        ElseIf Cell.Value = "Issue Note" Then
            Cell.Interior.ColorIndex = 18
        ElseIf Cell.Value = "Media" Then
            Cell.Interior.ColorIndex = 6
        ElseIf Cell.Value = "MPP Liaison Requests" Then
            Cell.Interior.ColorIndex = 1
        ElseIf Cell.Value = "MO Requests" Then
            Cell.Interior.ColorIndex = 2
        ElseIf Cell.Value = "Other" Then
            Cell.Interior.ColorIndex = 5
        ElseIf Cell.Value = "Order Paper Requests" Then
            Cell.Interior.ColorIndex = 7
        ElseIf Cell.Value = "Unsolicited Notes" Then
            Cell.Interior.ColorIndex = 8
        ElseIf Cell.Value = "Ornge" Then
            Cell.Interior.ColorIndex = 9
        Else
            Cell.Interior.ColorIndex = xlNone
        End If
       
    Next Cell
End Sub

 

Note also there's no need for that last long string of "AND"s... you can just use "Else".
 
Actually, Select Case maybe a better method to use:
 
Private Sub Worksheet_Change(ByVal Target As Range)
    Set MyPlage = Range("B")    'SEE NOTE BELOW
    For Each cell In MyPlage
        With cell.Interior
            Select Case cell.Value
                Case "FOI":                  .ColorIndex = 3
                Case "Fact Check":           .ColorIndex = 4
                Case "Issue Note":           .ColorIndex = 18
                Case "Media":                .ColorIndex = 6
                Case "MPP Liaison Requests": .ColorIndex = 1
                Case "MO Requests":          .ColorIndex = 2
                Case "Other":                .ColorIndex = 5
                Case "Order Paper Requests": .ColorIndex = 7
                Case "Unsolicited Notes":    .ColorIndex = 8
                Case "Ornge":                .ColorIndex = 9
                Case Else:                   .ColorIndex = xlNone
            End Select
        End With
    Next cell
End Sub
 

And finally, I think you need to correct the definition of MyPlage range at the top.  I don't think Range("B") is a valid range.  Did you mean the whole column?  In which case use Range("B:B"), but remember that that is a large number of cells (and even more in Excel 2007+).  Something like:
 
    Set MyPlage = Range("B1:B" & Range("B" & Rows.Count).End(xlUp).Row)
 
might be better.  This will only work through from B1 to the last populated cell in column B.

 

Hope that helps.

Hi,

 

This formula will highlight when the current date is 2 months prior to the current date.  You can adjust it back or forth a day by subtracting or adding 1 to the day argument depending on how exactly you want it to work.

 

=TODAY()>=DATE(YEAR(F1),MONTH(F1)-2,DAY(F1))

 

F1 contains the target date.  A little bit spiffer formula would be

 

=TODAY()>=EDATE(F1,-2)

 

Both of these will continue for format the cell after the target date is reached.  If you don't want that then

 

=AND(TODAY()>=EDATE(F1,-2),TODAY()<=F1)

 

.Cheers,

Shane

Many thanks!

stefan blom say:

Look at you capitalization settings at File | Options | Proofing | AutoCorrect Options. Also see if one or more Exceptions have been specified.

suzanne s. barnhill say:

If the sentence ends with a number, the following word will not be capitalized (this is a longstanding problem that was an unintended consequence of preventing automatic capitalization of the beginning of numbered list items).

If that's not the issue, then, as Stefan suggests, the problem is likely an exception. The chief culprit here is the check box for "Automatically add words to list." If that is checked, you'll probably find that a huge number of words have been added without your knowledge. You'll probably want to clear these out.
