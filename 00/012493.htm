Dim xPath As String

damon.bound say:

With a bit of trial and error I have established an answer.

It seems the VBA modules are somehow flagged internally as being pre-2007 format, even though this attribute doesn't show up in the VBA editor.

If you open the workbook in Excel 2010, open the VBA editor and insert new VBA modules into the workbook's VBA project, move all the legacy code across to the new modules, and delete the old modules, then the error goes away, and you can save as ".xlsm" format.

jacob entsch say:

IFS errors

Hi, 

I am comparing two different sets of stock files and am matching on barcode. 

1 stock file has up to 5 entries of barcodes per item and the other only has 1 entry per item.

So I thought IFS would best to use. However the formula isn't jumping from 1 condition to the next condition to return the first TRUE value. It is stopping upon return. 

I imagine you don't need to nest IFS like normal IF formula - e.g IF(Vlookup(A2,B:C,1,0)=A2,"Match",IF(vlookup(A3,B:C,1,0)=A3,"Match",) etc

Instead you can go - IFS(Vlookup(A2,B:C,1,0)=A2,"Match",Vlookup(A3,B:C,1,0)=A3,"Match" etc ??

The normal nested IF function isn't working either - unsure why. 

Any tips would be much appreciated. 

Thanks

teylyn say:

OK, assuming the four barcodes are in cells A1 to D1 on Sheet1 and the list of barcodes is in column A on Sheet2, this formula works for me:

=IF(ISNUMBER(MATCH(A1,Sheet2!A:A,0)),"match",IF(ISNUMBER(MATCH(B1,Sheet2!A:A,0)),"match",IF(ISNUMBER(MATCH(C1,Sheet2!A:A,0)),"match",IF(ISNUMBER(MATCH(D1,Sheet2!A:A,0)),"match","no match"))))

The trick is to catch the error that Match returns if the match value is not found, because any error will turn the whole result into an error. So wrap the Match into an IsNumber() function, which will turn the error into a "False" and the next part of the IF statement is evaluated.

Does that help?

teylyn say:

Of course, you can also use IFS

=IFS(ISNUMBER(MATCH(A1,Sheet2!A:A,0)),"match",ISNUMBER(MATCH(B1,Sheet2!A:A,0)),"match",ISNUMBER(MATCH(C1,Sheet2!A:A,0)),"match",ISNUMBER(MATCH(D1,Sheet2!A:A,0)),"match",TRUE,"no match")
