hansv mvp say:

You could create an event procedure for the Before Update event of the form:

 

Private Sub Form_BeforeUpdate(Cancel As Integer)
    If Me.NewRecord Then
        Me.[Created Date] = Now
    Else
        Me.[Modified Date] = Now
    End If
End Sub

You need Character Recognition software to turn the scanned image (usually a jpeg) into editable text. Many scanners come with a free cut down version, but there are many available that work to different degrees of success. Abbyy Fine Reader seems to be very highly regarded. Generally speaking, the better the quality of the scan, the better the conversion to text will work.
