(filename  :  iview436_setup.exe)
(uncheck the box if you do not want Google Chrome)

(you can multiple select if you hold your Ctrl key)

and the file names of the selected photos should be in the
"Input Images" field.

Now with the two file names in the "Input Images" field...
you can left click the "Create Image" button.

(the left and right positions of these images can be reversed by
selecting a file name and using the "Move Images Up" / "Move
Images Down"...buttons)

Now you should see a display of the combined image.
Go to...File / Save As...

bobmcfly247 say:

Well the problem with Teredo is that it is a network protocol which is blocked by some routers for security reasons. It's totally different to forwarding problems.

Usually teredo problems are caused by the router, not the Xbox. The Xbox only detects them as it is using this protocol to connect to XBL.

If you think the problem may be caused by your Xbox in this specific case and you have checked out the other steps I gave you, you can only try to reset your Xbox to factory settings as a last try. But this also means you have to reinstall all content again.

If you don't like to do a factory reset or even if it doesn't work, I would suggest contact your ISP for assistance.
