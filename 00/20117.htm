OK. I can confirm what you have said in your initial post. I have tested and it worked in xl2010 but not in xl2013 or xl2016.

Interesting thing is that if an option is selected in the Data Validation and only half of the range loses the color then if another option is then selected then the entire range loses the color. (Very strange)

The problem occurs with the Conditional format option "Format only cells that contain" and set the first option cell to "Blanks". I believe that it is a bug in the system.

However, if the option "Use a formula to determine which cells to format" is used and then the following formula then it works fine so you should be able to use this as a work around.

    =ISBLANK(A1)
