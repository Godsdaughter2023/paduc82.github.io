doug robbins - mvp office apps & services (word) say:

Use the following code:

Dim wbsource As Workbook
Dim wbtarget1 As Workbook
Dim wbtarget2 As Workbook
Dim wbtarget3 As Workbook
Dim wbtarget4 As Workbook
Dim cols As Long
Dim rows As Long
Set wbsource = ActiveWorkbook
cols = wbsource.Sheets(1).Range("A1").CurrentRegion.Columns.Count
rows = wbsource.Sheets(1).Range("A1").CurrentRegion.rows.Count
Set wbtarget1 = Workbooks.Add
wbsource.Sheets(1).Range(Cells(1, 1).Address, Cells(250000, cols).Address).Copy _
Destination:=wbtarget1.Sheets(1).Range("A1")
Set wbtarget2 = Workbooks.Add
wbsource.Sheets(1).Range(Cells(250001, 1).Address, Cells(500000, cols).Address).Copy _
Destination:=wbtarget2.Sheets(1).Range("A1")
Set wbtarget3 = Workbooks.Add
wbsource.Sheets(1).Range(Cells(500001, 1).Address, Cells(750000, cols).Address).Copy _
Destination:=wbtarget3.Sheets(1).Range("A1")
Set wbtarget4 = Workbooks.Add
wbsource.Sheets(1).Range(Cells(750001, 1).Address, Cells(rows, cols).Address).Copy _
Destination:=wbtarget4.Sheets(1).Range("A1")

doug robbins - mvp office apps & services (word) say:

With the Workbook that you want to split open, press ALT+F11 to open the Visual Basic Editor and then select that Workbook in Project Explorer pane and then click on the Insert item in the Toolbar and then on Module.  In the Module that is added to your Workbook, type

Sub SplitWorkbook()

and then press Enter.  Then paste the code from my post between the Sub SplitWorkbook() and the End Sub that will have appeared when you pressed Enter.

To run the macro, back in the Excel window, click on the Developer tab and then on Macros. In the Macros dialog that appears, select the SplitWorkbook macro and then click on Run.

doug robbins - mvp office apps & services (word) say:

Use:

Set wbsource = ActiveWorkbook
cols = wbsource.Sheets(1).Range("A1").CurrentRegion.Columns.Count
rows = wbsource.Sheets(1).Range("A1").CurrentRegion.rows.Count
Set wbtarget1 = Workbooks.Add
wbsource.Sheets(1).Range(Cells(1, 1).Address, Cells(250000, cols).Address).Copy _
Destination:=wbtarget1.Sheets(1).Range("A1")
wbtarget1.SaveAs wbsource.Path & "\" & "Rows 1 to 250000.xlsx"
Set wbtarget2 = Workbooks.Add
With wbtarget2.Sheets(1).Range("A1")
    For i = 1 To cols
        .Offset(0, i - 1) = wbsource.Sheets(1).Range("A1").Offset(0, i - 1)
    Next i
End With
wbsource.Sheets(1).Range(Cells(250001, 1).Address, Cells(500000, cols).Address).Copy _
Destination:=wbtarget2.Sheets(1).Range("A2")
wbtarget2.SaveAs wbsource.Path & "\" & "Rows 250001 to 500000.xlsx"
Set wbtarget3 = Workbooks.Add
With wbtarget3.Sheets(1).Range("A1")
    For i = 1 To cols
        .Offset(0, i - 1) = wbsource.Sheets(1).Range("A1").Offset(0, i - 1)
    Next i
End With
wbsource.Sheets(1).Range(Cells(500001, 1).Address, Cells(750000, cols).Address).Copy _
Destination:=wbtarget3.Sheets(1).Range("A2")
wbtarget3.SaveAs wbsource.Path & "\" & "Rows 500001 to 750000.xlsx"
Set wbtarget4 = Workbooks.Add
With wbtarget4.Sheets(1).Range("A1")
    For i = 1 To cols
        .Offset(0, i - 1) = wbsource.Sheets(1).Range("A1").Offset(0, i - 1)
    Next i
End With
wbsource.Sheets(1).Range(Cells(750001, 1).Address, Cells(rows, cols).Address).Copy _
Destination:=wbtarget4.Sheets(1).Range("A2")
wbtarget4.SaveAs wbsource.Path & "\" & "Rows 750001 to Total Rows.xlsx"

graham mayor say:

Type mycompany in a blank document.

Use the ¶ button on the home tab to display the formatting. Select mycompany, but not the following paragraph mark.

From Options -> Proofing -> Autocorrect Options > Autocorrect -  check Formatted text. The same word will be put in the Replace with text box. Save the entry. 

Now when you type mycompany at the start of a new line (assuming mycompany is all one word) then it will stay the same as that replacement, albeit with the format of the paragraph you type into.
