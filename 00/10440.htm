e.j. gun say:

Right-click on My Computer on your desktop.  Select Properties.  Select the "Advanced" tab.  Click on Environment Variables.  There should be a "TEMP" variable in your user profile, and also "TEMP" and "TMP" variables in the System variables section.  These control where various programs create temp files.  Not sure which one works for Office, probably one of the two System ones.  Some temporary files, such as those created by Word, are created in the same directory as the original document.  Others are created in the System or User temp folders.
