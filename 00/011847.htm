Hi, this will require a registry fix . . .

Click your Start Button, type regedit and hit Enter to open the Registry Editor

Navigate to:

HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Word\Reading Locations

Right click Reading Locations and choose Permissions

You need to make this key read-only for everyone so that it can’t be re-written by system

Select each user and make it read only

Click on Advanced then uncheck the option “Replace all child object permission entries with inheritable permission entries from this object'

Click Apply and OK and restart your PC for the change to take efffect

That's It!

max, singapore say:

Try this play using code by Orlando Magalhaes Filho (below)

Steps:
Draw a text box (default named:"Text Box 1") on the sheet first
[Or insert a picture, then rename it as: Text Box 1]


Then install the code by right-clicking on sheet tab > View code
then copy n paste Orlando's code into the code window
[ Adjust the x, y offsets to suit - I set these arbitrarily to 10]


'---
Private Sub Worksheet_SelectionChange(ByVal Target As Excel.Range)
'by Orlando Magalhaes Filho in .programming
x_offset = 10
y_offset = 10
    ActiveSheet.Shapes("Text Box 1").Select
With Cells(ActiveWindow.Panes(1).ScrollRow, _
ActiveWindow.Panes(1).ScrollColumn)
    xpos = .Left + x_offset
    ypos = .Top + y_offset
End With
With Selection
    .Left = xpos
    .Top = ypos
End With
Target.Select
End Sub
'---

rickcp say:
Trojan:DOS/Alureon.A is only 'partially' removed by MSE.

DOS/Alureon is the detection for a variant of the Win32/Alureon malware family of data-stealing trojans that infects the Master Boot Record (MBR). It may also infect and corrupt certain driver files, causing them to become unusable. Recovering from this situation may require measures beyond removing the trojan itself from the computer.

Please note that Microsoft antimalware products don’t automatically repair the MBR when they clean this Trojan, so you would need to take further action manually. You will need to run the following commands using the "bootrec.exe" tool to ensure a complete repair of your computer:
- bootrec /fixmbr
- bootrec /fixboot
- bootrec /rebuildbcd
For more details on these commands, please refer to MS's KB927392, with specific focus to the options: "/fixmbr", "/fixboot" and "/rebuildbcd".

You should also be aware of the fact that it's a known issue for many offline scanners (including WDO) that can not 'automatically' repair the MBR (but only detect and partially remove threats) so this repairing needs to be done 'manually'.
However, there have been reports at this Community [Source], that Anti-Rootkit utility TDSSKiller (from Kaspersky Labs) may repair the MBR automatically (though not guaranteed) when cleaning the system off this type of threats, thus it's most recommended.
See > How to remove malware belonging to the family Rootkit Win32.TDSS (aka Tidserv, TDSServ, Alureon).
You may also find this TDSSKiller User Guide (from bleepingcomputer.com) to be of assistance.
Alternatively, you may also try the new Malwarebytes Anti-Rootkit (Beta) to include its featured 'fixdamage' tool.

Good Luck!
