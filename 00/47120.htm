darloma,
Hardware related means that the problem is not, or can not, be caused by software.  Hardware is the physical components that makeup a computer system - case, motherboard, hard drive, display, battery, etc., and software is the operating system and programs that run ON that hardware.  Sometimes it is difficult to determine if a problem is caused by the hardware failing to function according to manufacturer specifications or if the software is causing an issue becuase it is creating a situation that makes the hardware look defective.  But in the case of power - and specifically a battery - it either works or it does not.  If a battery is not charging it is either a defective battery or a defective charger.  There is not any software involved.  Software can cause a computer to not shut down properly, or wake from sleep properly, but software can not make a battery hold a charge or not hold a charge.  So that is what Anthony was saying.  This issue can not be caused by the operating system therefore you need to contact your hardware manufacturer.

Hi David. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

Programs/apps should be kept on C drive so they're included in its image, since they write keys to the registry on C.

If you want to move anything off of C I would move the User folders following http://www.dummies.com/computers/operating-syst....

This keeps the image smaller so if Windows ever becomes irreparable you can reimage C in 20 minutes and your files will be current and separated in their own partition. If you included them in the image then the files restored will be old.

This is the only reason to separate out anything: to keep the image smaller of C for storage purposes. See here how to use the best free imaging app: https://www.howtogeek.com/howto/7363/macrium-re...

To create a new partition on the hard drive, use Disk management to Shrink C by as much space as you want, then highlight and create a new Partition there:
