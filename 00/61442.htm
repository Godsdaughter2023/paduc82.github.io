Hey HaiLua,

Method 1:

The cause for this issue is that Windows might have renamed the drive to a letter that is already in use. This will happen if you have several card readers, thumb drives or external hard drives attached. It will also happen if you are on a network and have mapped drives.

To find the drive and then rename it, you’ll want to right-click on My Computer and select Manage.

a. From the Computer Management screen, select Disk Management.

b. In this window you should see all of your connected physical drives, their format, if they are healthy, and the drive letter.

c. In this instance you are going to change the drive letter for the External drive. Right-click on the drive in the list, and from the resulting menu select “Change Drive Letters and Paths”.

d. Click on Change so we can change the drive letter.

e. Select a new drive letter from the drop down list, preferably the one that you usually use for this drive.

f. Click Yes on the confirmation screen and you are done.
