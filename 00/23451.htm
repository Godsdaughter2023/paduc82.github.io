Excel custom cell formatting - Microsoft Community

glennhong say:

Excel custom cell formatting

I have a worksheet with wide-ranging numeric values and am mostly happy with the format:

[>0.01]##,###.#0;[<0.01]0.00E+00

However, I would also like to hide zero values.  I tried adapting the commonly recommended 0;-0;;@, i.e.:

[>0.01]##,###.#0;[<0.01]0.00E+00;;@

but the zero does not hide.  Can someone advise?

Thanks

hansv mvp say:

If you specify ranges, this overrides the standard positive;negative;zero;text format.

Moreover, you can specify only two ranges.

So you will have to use something else instead.

If you want to hide ALL zero values on the sheet, select File > Options, click Advanced, scroll down to the section 'Display options for this worksheet', clear the check box 'Show a zero in cells that have zero value', then click OK.

If you only want to hide zeros in a specific range, create a conditional formatting rule:

  • Select the range.
  • On the Home tab of the ribbon, click Conditional Formatting > New Rule...
  • Select 'Format only cells that contain'.
  • Leave the first dropdown set to 'Cell Value'.
  • Select 'equal to' from the second dropdown.
  • Enter 0 in the box next to it.
  • Click Format...
  • Activate the Number tab.
  • Select Custom in the list of Categories.
  • Enter two semi-colons in the Type box.
  • OK your way out.
