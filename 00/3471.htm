    Hello JAB8778:

    Thank you for posting to the forum regarding the issue you are having with accessing roaming options.  The ability to roam is something supported by the Windows Phone and as indicated on the phone is dependent on the mobile operator being used. 

    If the phone is telling you this option is not available that means there is some issue with the SIM, your data plan or possibly the phones firmware.  Please consider the following steps to see if any progress can be made.

    • Try another SIM card in this phone.
    • If you can, try this SIM card in another Windows phone 8.

    Hi

    This a known issue and the fix is in the Insiders Hub. Here it is again.

    You might see a WSClient.dll error dialog after logging in.

    1. Right click the Start button and select Command Prompt (Admin).

    2. In the Command Prompt window, type the following command

    schtasks /delete /TN “\Microsoft\Windows\WS\WSRefreshBannedAppsListTask” /F

    3. Press Enter.

     You should see a "Success" result.
