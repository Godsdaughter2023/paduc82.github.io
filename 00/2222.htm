There is no version of MSE for XP 64 bit. Only XP 32 bit. 64 bit is only supported for Vista and Windows 7.

leong_25 say: @Workaround Walt  
The solution in my case turned out to be as follows:
Open Word>Choose Page Layout on the ribbon>Choose Paper>Print Options>Advanced>Print
Scroll down to "Scale Content for A4 or 8.5x11 inch paper size".

If the box for "Scale Content for A4" is unchecked, check it.  Close Word, then open your document and try printing one page.  If it prints correctly, then your issue is resolved.  If this box is already checked, try unchecking it and test.

Q: Who owns the data posted in the Yammer network?

A: Data posted into a free Yammer network is owned by the individuals posting that data. Those users are the data controllers for their content.
