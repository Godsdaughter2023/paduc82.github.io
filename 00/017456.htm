diane poremsky mvp (slipstick.com) say:

Had to refresh my memory on the steps to use MODI with Office 2007 and verify it worked on Win7 (32 bit - not sure it will work with 64bit windows as my test machines are all 32bit)

If the MODI printer is not installed, go to Add Remove programs (type it in the start search field in Win7) - find office and click Change (at the top) then add or remove features.  Expand Office Tools, select Office Docuement imaging and click on the little arrow, select run all from my computer. You need to install both MODI (the printer) and the OCR pieces (for the viewer) - help is optional.  

Print each page to the office document image writer then open each one in Document Imaging (it's in Start, Programs, office, office tools) - drag from one thumbnail screen to the thumbnails in the other imaging window.  You can print as either MDI or TIF but need to view them in the Document Imaging app to merge them, not the fax viewer - if you print as MDI it will open in the imaging app automatically. (The import command in Image viewer opens a new window, rather than import into the current one)
