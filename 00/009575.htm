jay freedman say:

Unfortunately, Word 2007 is just one version too early to do what you want. In Word 2010 or 2013 there is a Text Effects button in the Font dialog for regular text, in which you can choose the weight and color for the outline; and the font kerning setting on the Character Spacing tab still applies. WordArt in 2010 is still the same as in 2007, but in 2013 the same text-effect functions are available for both WordArt and regular text.

I don't have Publisher 2007 installed to check it, but Publisher 2010 appears to have fewer capabilities than Word 2010 for your task.

jay freedman say:

A test shows that creating the outlined regular text in Word 2010 and saving the document, then opening it in Word 2007, preserves the outline format and the kerning, but not the changed line weight. :(

However, there is a workaround, which should be acceptable for a logo. In Word 2010, after formatting the text as you want it, select it and copy it to the clipboard. Then use Paste Special and choose to paste it as a picture, replacing the selected text. That will survive the transition to Word 2007 without problems.

Sharing of Office 365 on same PC but different user - Microsoft Community

henrikandersen62 say:

Sharing of Office 365 on same PC but different user

Hi I have set up my son on my PC as a seperate user. How can he access the Office apps which I have installed under my user name on the same PC?
mohan suryanarayan say:

Hello Henrik,

Welcome to Microsoft Community.

Office 365/2016 once installed on one user account, will automatically be available for the other user accounts as well.

Login to the other user account and follow these steps:

In the main Windows Start Screen (will all the icons showing) you will be able to locate individual application icons for Office 365 applications. Check in this location and once found, you may pin the icons to the Taskbar or Start Menu by right-clicking on the application icons.

If not found in the main Windows Start Screen then try these steps:

From Windows Explorer navigate to C:\Program Files\Microsoft Office 16\root\Office16 or C:\Program Files (x86)\Microsoft Office 16\root\Office16

Note: If you are on a 64 Bit computer (64 Bit Windows 10) then you will find the Office16 folder in the following locations.

From your Windows Explorer: Browse to the location C:\Program Files\Microsoft Office 16\ClientX86\Root\Office16 or C:\Program Files (x86) \Microsoft Office 16\ ClientX64\Root\Office16.

In the Office16 folder you will find the Microsoft Word application file like this: 

winword or winword.exe

In case you do are able to find the Word file, then right-click on the winword or winword.exe file and select the option: Pin to Start or Pin to Taskbar to add the shortcut to your desktop/start screen. 

Similarly you may add the shortcuts to other Office applications from the following files:

onenote or onenote.exe

excel or excel.exe

powerpnt or powerpnt.exe

Note: If you have an Office 365/Office 2013 installation then you may look for the above mentioned files in the Office15 and Microsoft Office 15 folder/s – the above file locations remain the same as mentioned for Office16 above.

Try the above steps and let us know the result.

Thank you.
