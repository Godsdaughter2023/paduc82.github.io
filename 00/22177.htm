grschinon say:
14-Feb-2008 09:04 AM
hddcrash wrote:
I am Novice in this area, had a doubt if a memory card be possible to use NTFS instead of FAT?? or the 8gig memory in N91??

A memory card behaves exactly like a hard disk. It has a partition table and all. You can format the card how you like with whatever filesystem you want and with whatever partition scheme you want as long as the device you're going to put the card in supports it. I've even used miniSD cards with ext2 partitions in a backup system.

Remember, you're restricted by the device you're putting the card in, not by the card itself. Most devices only know how to use the first primary parition with a FAT or FAT32 filesystem on it.
