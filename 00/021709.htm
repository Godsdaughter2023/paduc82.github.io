jahbur say:

Don't know if this helps, but you can do dice roll on one slide with a few lines of code.

Create a blank slide and add six square shapes (your dice faces)

Number them 1 to 6

Move them all off the side of the slide

Animate them separately so that they move/spin/whatever onto the slide

Add a macro with the following code:

Dim i as long

Randomize

i = (rnd() * 6) + 1

With ActivePresentation.SlideShowWindow.View

     .GotoSlide .Slide.SlideNumber

     .Slide.TimeLine.MainSequence.FindFirstAnimationFor(.Slide.Shapes(i)).MoveTo 1

     .Next

End With

Then create a button and through actionsettings have it run the macro.

Run the slideshow and it should animate one of the six objects randomly whenever you press the button.

jason (office engineer) say:

Hi Jill,

We have a couple features that might be of use to you, but nothing is exactly what you're asking for.

First off, we have a duplicate feature that will allow you to save off copies of your Sway before you rework it each week.  The button for it is on the … menu.

Secondly, we have a template feature that you could use to make it easier to start new Sways from.  Just create a new Sway with the basic info you want in all announcements and choose to save it as a template from the … menu.

If you make use of these features and create a new Sway each week, when it comes time to release the announcement, you could just update the embed code in the courses to point to the new one.  If you don't want to do that, you could decide to make use of one of a variety of free link shortening services online.  They will create an alias that will redirect to whatever URL you give them and then you can embed that alias in the courses.  Some of these services will allow you to come back and update the underlying URL so that you only have to update it in that one place each week.

Let me know what you end up deciding on and how it goes.

Jason

jason (office engineer) say:

If you want to use a link shortener, try these steps:

  1. Create a Sway
  2. Get the embed code for that Sway and paste it somewhere temporarily
  3. From that embed code, get the src URL (i.e. https://sway.com/s/rainier/embed)
  4. Pur that URL into a link shortener service that will allow you to update it later (some of them will just make you a new shortened link each time)
  5. Grab that shortened link and put that into the src of the embed code
  6. Put that embed code into your courses
  7. When you have a new Sway, get the embed URL for it from Sway, go back to the link shortener service, and update the source URL for your link

Let me know if you think that will work for you.

Jason
