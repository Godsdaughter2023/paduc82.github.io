New boot menu in Windows 8 is much prettier than the older one but on the other hand it can slow down the start of the second, or third, operating system as part of Windows 8 is loaded in the background before the boot manager is displayed on the screen.

 

If you have to switch regularly between Windows 8 and other installed operating systems, you may have a preference to use the old boot manager instead as it is displayed right after the start of the system and before any of the operating systems are loaded.

 

You got a way to get back the legacy boot manager with its gray on black interface through some sources. For changing the boot manager from the Windows 8 default manager to the legacy manager known from previous versions of Windows, you need to perform the following steps in Windows 8:

 

a.      Press Windows to go to the start screen and type cmd in there. Right-click on Command Prompt and select run as administrator from the bottom command bar.

b.      Run the following command here and press enter afterwards: bcdedit /set {default} bootmenupolicy legacy

c.       Restart your operating system afterwards. You should now see the legacy boot manager and not the Windows 8 Boot menu.

d.      You can use the following command on the command line to restore the Windows 8 boot menu: bcdedit /set {default} bootmenupolicy standard

e.      You should get a “The operation completed successfully” confirmation after running one of the commands on the command line.

You are talking about RAID 1, mirroring?

RAID 1 (mirroring) is not a backup solution. RAID 1 uses two or more drives, each a duplicate of the others, to provide redundancy, not backup. It's used in situations (almost always within corporations, not in homes) where any downtown can't be tolerated, because the way it works is that if one drive fails the other takes over seamlessly. Although some people thing of RAID 1 as a backup technique, that isnot what it is, since it's subject to simultaneous loss of the original and the mirror to many of the most common dangers threatening your data--severe power glitches, nearby lightning strikes, virus attacks, theft of the computer, etc. Most companies that use RAID 1 also have a strong external backup plan in place.
