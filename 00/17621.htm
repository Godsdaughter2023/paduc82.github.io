doug robbins - mvp office apps & services (word) say:
What do you want to do with the data?
 
The following will create an array containing the
 
Dim xlapp As Object
Dim xlBook As Object
Dim arrData As Variant
On Error Resume Next
Set xlapp = GetObject(, "Excel.Application")
If Err Then
    bstartApp = True
    Set xlapp = CreateObject("Excel.Application")
End If
On Error GoTo 0
Set xlBook = xlapp.Workbooks.Open("C:\mypath\mydata.xlsx")
arrData = xlBook.Names("alldata").RefersToRange.Value
 
To see the data, add the following code
 
Dim i as long, j as long
For i = 1 To UBound(arrData, 1)
    For j = 1 To UBound(arrData, 2)
        MsgBox arrData(i, j)
    Next j
Next i
 
If you want to add it to a combobox or listbox, use
 
[controlname].List = arrData
 
or dispense with the arrData and use
 
[controlname].List = xlBook.Names("alldata").RefersToRange.Value

The above uses late binding so will work regardless of the version of Excel/Office

  • Make sure you include hidden and system files and folders when searching for a pst-file.

  • For Outlook 2007 and previous on Windows Vista and Windows 7 the default location is;

  • For Outlook 2010 on Windows XP the default location is;
