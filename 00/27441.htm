charles kenyon say:

That is not the key sequence in Word 97-2003. A sequence that works is:

Alt+O,F,[tab] [tab] 14 [Enter]

or

Alt+O, F, Alt+S, 14 [Enter]

In Word 2007-2016 the sequence is either of:

Alt+O, F, [tab] [tab] 14 [Enter]

or

Alt, H, F, S, 14 [Enter]

The first sequence uses the Font dialog box. The second uses the font size box in the Home tab of the Ribbon.

"Alt+O" is a combination. "Alt, H" is a sequence. Note that the first sequence for either is identical.

No escape key. No mouse.The Enter key returns you to the document.

What do you mean "how are they used effectively"?  You pick a gradient type to start with then play with it. There is no efficent way other than starting with a preset.  What is effective?

 

 

Resizing ... haven't you heard, 2010 is "improved". AFIK Text size is no longer hard linked to the box size in 2010, you can no longer simply drag the WordArt box to stretch the text. 

There is one way I know of to do it. Save a file in 2003 format, you are now in Compatibility mode. When you insert WordArt, it is still 2003 standard WordArt so you can stretch it the way you wish. You can create the 2003 Wordart object then copy/paste it into the 2010 format file, it will still retain it's "2003 nature".  There are lots of extra things you can do in compatibility mode, once you are aware of it.

By default, there are three slider bars.  A separate color can be assiged to the one at each end by selecting the slider bar and assigning the color to it.  Then the center one can be used for altering the gradient of the fill.  It is also possible to add slider bars to vary the effects.

To change the size of the text, you need to use the format font section of the Home tab.

Internet Explorer won't block pop-ups from websites that are in the Local intranet or Trusted sites content zones. If you want the pop-ups from these websites to be blocked, you have to remove the websites from these zones. Refer to the section Add or Remove a site from security Zone of the following article.

 

Change security and privacy settings in Internet Explorer

http://windows.microsoft.com/en-IN/internet-explorer/ie-security-privacy-settings#ie=ie-10

 

You might have software, such as adware or spyware, on your computer that is opening pop-up windows. To stop these pop-ups, you'll have to identify the software that is opening them and then remove it or change its settings to stop it from opening pop-up windows.
