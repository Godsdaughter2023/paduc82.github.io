If you run a macro containing the following code after you open the document, it will replace the Date field with a CreateDate field so that it will show the date on which the document was created.

 

Dim i As Long
With ActiveDocument
    For i = 1 To .Fields.Count
        With .Fields(i)
            If .Type = wdFieldDate Then
                .Code.Text = Replace(UCase(.Code.Text), "DATE", "CreateDate")
                .Update
            End If
        End With
    Next i
End With

That however may not be the date on which the letter was sent to its original recipient and to change it to the date on which the document was last printed may be better

 

Dim i As Long
With ActiveDocument
    For i = 1 To .Fields.Count
        With .Fields(i)
            If .Type = wdFieldDate Then
                .Code.Text = Replace(UCase(.Code.Text), "DATE", "PrintDate")
                .Update
            End If
        End With
    Next i
