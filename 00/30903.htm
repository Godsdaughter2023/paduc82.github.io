  1. Download the troubleshooter, and select Open or Save in the pop-up window. (If you choose Save, you'll need to open the troubleshooter from the save location.)
  2. Select Next to start the troubleshooter, and follow the steps to identify and fix any problems.
  3. After the troubleshooter finishes, try running Windows Update again. Select the Start button, and then select Control Panel > Windows Update. In the left pane, select Check for updates, and then install any available updates.

Once you're done, update us with the result.
