I can assure you that Windows is only using the file wuauclt.exe now and that it's entirely normal to find a copy of wuauclt1.exe in %windows%\System32 <=this folder.

davem121 say:
Hi IgFe, this is a known issue in the 1803 Update,

It is the recovery drive, you are not meant to see it, it is not meant to have a drive letter, remove the drive letter and everything will return to normal:

Click your Start Button, type cmd, then right click Command Prompt and choose 'Run as Administrator'

Run this command and hit Enter

diskpart

Run this command and hit Enter

list volume

Note down the letter associated with that new drive

Run these commands one at a time and hit Enter (replace X with the correct drive letter)

select volume X

remove letter=X

Close Command Prompt and restart your PC

davem121 say:
Hi Mec-Tech,

If you want to copy everything from your old hard drive, then you will need to clone one hard drive onto the other using 3rd party software like the free version of AOMEI Partition Assistant.

If you want to go from 32bit Windows to 64bit Windows, then this will require a clean installation of Windows 10 onto your New Hard Drive - reinstalling is free, since your licence covers both versions

Honestly, your best method will be:
Backup your data from the old hard drive
Clean install Windows 10 64bit onto the new Hard drive
Reinstall all your apps
Restore your Data

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-gb/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
