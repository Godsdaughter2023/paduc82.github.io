dktech say:

I know this is an old post.  But just in case other people are having the same problem, here are the steps that I've been able to take to fix the problem.

1) Stop the Microsoft Antimalware service

2) Rename the Definition Updates folder to Definition Updates_old. 

If you're a Windows 7 or later user, the location is C:\ProgramData\Microsoft\Microsoft Antimalware.  If you're a Windows XP user, the location is C:\Documents and Settings\All Users\Application Data\Microsoft\Microsoft Antimalware.

3) Start the Microsoft Antimalware service.  You should get a warning that the definitions have not been updated. 

4) Manually run an update.

You should be good to go.
