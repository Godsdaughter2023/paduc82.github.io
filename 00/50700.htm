¡firedog say:

I used the free KLS Mail package to restore all the many emails that I had backed up before the reinstallation.

Did you perform the restore before or after installing Windows Live Mail?

... each of my email account / sub accounts is listed not just the expected once, but two or three times in the drop down list.

If these accounts are set up in the same way, then there should of course only be one of each. It is quite possible to have several copies of the same account, but it takes careful management and can't really be recommended. It sounds as if you set up the accounts in the new installation, then restored the KLS backup which also re-instated the accounts that were there before. You might have to do some detective work to find out which is which and weed out the superfluous ones.

In Windows Live Mail, press Ctrl-Shift-O for Options. On the Advanced tab, click Maintenance and then Store folder. Copy the path shown in the location box and then open an Explorer window at that address. Close down Windows Live Mail.

Each account has two folders, one - the account folder - named with the account's associated email address and the other - the mail folder - named by default with Domain (username), possibly truncated to 11 characters with a suffix of three random characters. Some of these mail folders may have serial numbers appended, like (1).

Check the mail folders for each account. Each will contain a number of sub-folders, but some of them will probably be empty. Is there more than one Inbox folder containing messages for each account? If there's only one Inbox folder with messages in it, we're getting closer.

Each mail folder will also contain an .oeaccount file giving the account settings. If there's more than one of these for each address, you'll see duplicate entries in the From dropdown in a message compose window and in the Accounts dialogue at Alt-F O E. So, for each account you should have

  • a folder named with the account's email address, containing only the configuration file oeconfig.xml
  • a folder containing sub-folders containing messages and a single .oeaccount file

Rename any other .oeaccount files to .old instead, so for example
account{6A01A6BC-A9C1-46E2-80CF-DE2A9DA4C61D}.oeaccount

becomes

account{6A01A6BC-A9C1-46E2-80CF-DE2A9DA4C61D}.old,

and rename the folders they're in to something completely different, like Extra folder 1 ...

Now press Windows key + R, type wlarp into the box and press Enter. In the window that opens, select the Repair ... option. When it's finished, restart the computer, launch Windows Live Mail and test.

You may have to designate one of the accounts as default. This is the account that will be used if the program is invoked by Windows to send a message. Right-click on the name of the primary account in the folder pane and select Set as default account.

Please let us know how you got on.
