john - project say:

RJF61,

It depends on how the Justin Projects subproject was inserted. Try this.

1. Unlink the Justin Projects subproject from the master

2. Save both the master and Justin Projects to insure the link structure change is captured by both files. Failing to do this will leave link fragments.

3. Collapse the master such that none of the subproject tasks are displayed (i.e. only the insertion point summary lines appear)

4. Insert Justin Projects after the last subproject

I'm not sure if you are aware but I always mention this to users when discussing dynamic masters. In Project the link structure of dynamic master is fragile and prone to corruption. None of the files in the structure should ever be renamed, moved, overwritten or saved off to another location. All files should reside in a single directory on a local drive and should not be operated over a network.

Unless you have a specific reason to have a dynamic master, I suggest you consider creating a static master any time you need to view all files together. Do so by unchecking the "link to project" option in the lower right corner of the Insert Project window.

Hope this helps.

Using mail merge to create a directory in Word: Why does whole doc reprint for each entry?

I've set up a mail merge in order to create a directory booklet with a continuous list of the info for each member.  But once I insert a cover page, the document reprints the cover page for every single member.  All I want is a cover page and then the members listing beginning on page 2.

I'm using Office 2007.

Also--in Excel, I have columns formatted as phone numbers, but when they print in the merged doc, the hyphens and parentheses disappear.   Can't figure out why as they're in the same version of Office.

Thanks if you can help.
suzanne s. barnhill say:

That's the way a directory merge works. Any text in the mail merge main document is repeated for each record. You need to merge the document first, then add the cover page and any other elements that should appear only on specified pages.

As for the number formatting, Excel doesn't actually store this with the data; it's just for display purposes. In order to add it to the data in Word, you must add a "numeric picture" switch to the merge field. See http://www.gmayor.com/formatting_word_fields.htm.

andre da costa say:

Your CD drive or DVD drive is missing or is not recognized by Windows or other programs

Fix It for missing CD/DVD drive:
http://support.microsoft.com/kb/982116

Go into BIOS when you boot the computer and change the setting from IDE to AHCI.  This allows Windows 7 to recognize the SATA CD or DVD ROM player you are booting from and it should install.

This probably won't work if your CD/DVD ROM isn't a SATA drive.
