Anyway, I am posting a file here which has the macro. You change the SourcePath and TargetPath.

1. Make a backup of your workbook.

Sub ChangePivotPaths()
    Dim Ws As Worksheet
    Dim Pt As PivotTable
    Dim SourcePath As String, TargetPath As String
    SourcePath = "\temp"
    TargetPath = "\Junk"
    For Each Ws In Worksheets
        For Each Pt In Ws.PivotTables
            Pt.SourceData = Replace(Pt.SourceData, SourcePath, TargetPath)
        Next Pt
    Next Ws
End Sub

Hi,

There is no option to only allow push e-mail via Wi-Fi.  This is not an option with ActiveSync on your device.  There is no option to use Direct Push e-mail via Wi-Fi only.  Wi-Fi uses more power than Cellular so Microsoft decided to restrict direct push to cellular.

As you have experienced, ActiveSync does work over wifi, however there is no ability to tell ActiveSync to use only wifi.  DirectPush is not supported over wifi, so I would suggest that you not set ActiveSync to manual.  When you are connected to wifi and want to sync, just hit Send/Receive in Outlook Mobile.
