andy pope say:

1. The TREND formula is used to return the X position where the line crosses. Nothing to do within chart Trendlines.

2. Returns a column of all the used X values from the original data and the above & below line calculations.

3. Returns corresponding Y values for the line section above the 'Mean' value.

4. Returns corresponding Y values for the line section below the 'Mean' value.

 

As to how many rows the formula in K:M should be copied down well that really depends upon the data.

If you had 72 periods and the points where all above or all below then 72 rows would be enough.

If on the other hand your your data alternated between above and below then you would need (72*2)-1 columns

sandyd49 say:

Can't print book fold with the right margins.

I can't seem to get my document to print with the margins I want.  The document is book fold, landscape, with .5 margins.  It looks fine in print preview.  But when it actually prints, there is a 1" margin on the outside margins (left on the left-facing page and right on the right-facing page).  What am I missing?
sandyd49 say: I have discovered the problem!  It occurs when I use the automatic duplex printing capabilities of my HP Officejet 6000.  It adds to the margin in order to be able to handle the paper turning!  If I print the document with manual duplex, all is fine.  So beware those posts out there that tell you to print book fold with automatic duplex.  You may not like the results.
