You've given up your Windows 7 Licence to Upgrade to Windows 10. So, you can't reuse that License on the same or another PC.

If you have a second unused Windows 7 License there are two options.

2. Use Virtualisation Software.

6. Select a partition to install Windows on.
7. Continue with install.
8. After Windows 7 is installed you can use Windows Easy Transfer (or another backup method you used) to restore your files & settings, and use your program installation disks to reinstall all your programs.
