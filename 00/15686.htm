geoffbarraclough say:

A global variable is alive for the session of code that is utilising it

If you finish the code then start again, the variable will not retain its value

To retain values from prior execution, your best bet is to use a Lookup sheet in excel and write the values to a specific cell so at the end of the routine:

Sheets("Lookup").Range("A1") = MyVariableName

and at the start of the routine:

MyVariableName = Sheets("Lookup").Range("A1")

to make it easier to interpret, I would suggest using named ranges e.g.

Sheets("Lookup").Range("MyVariableName") = MyVariableName
