jean-yves tfelt say:

Hi

The data does not look ok because all data stays in column A as one long string.

This behaviour occurs when the list separator used in the CSV (semi-colon) file is different  from the separator defined in the "Region and Languages" settings (probably a comma).

To get it right, Goto the Control Panel/Region and Language. Click "Additional settings"

Change the List separator to semi-colon ;

That should solve your issue. Re-open the CSV file.

If you do not want to change your list separator, select column A and switch to the Data Tab.

From the Data tool group, select Text-To-Column.

Next dialog, select Delimited and click Next. Select the semi-colon as separator and click next.

Define the data type for each column according to your needs (Text is the safest as not no can be changed, i.e. dates)
