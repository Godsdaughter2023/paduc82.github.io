frederik long say: I have never tried the Acer recovery process but I have had excellent results with Acronis TrueImage. If this was my machine then I would do this:
  1. Connect both disks.
  2. Boot the machine on the old disk.
  3. Partition & format the new disk: 100 MBytes for the boot partition (same as for your existing disk), 60 GBytes for the OS and for applications, the rest for your data.
  4. Install Acronis TrueImage (or some other Windows 7 compatible imaging tool).
  5. Create image files of your existing partitions. Park the files on the data partition of the new disk.
  6. Burn an Acronis Rescue CD.
  7. Disconnect the old disk.
  8. Boot the machine with the Rescue CD.
  9. Restore the image files to their respective partitions.
  10. When finished, update the system image once or twice a year, to be used in case of disaster.
Instead of purchasing TrueImage you could use the native Windows imaging tool. However, I find that it is difficult to use and quite temperamental.
