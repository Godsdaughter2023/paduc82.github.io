jay freedman say:
Assuming you've used Ctrl+F9 to create each pair of field braces, as you mentioned in your original post, then the coding of your fields is OK except for the square bracket [ near the end of the second line (I guess that's just a typo).

The step you haven't mentioned is that, after you make a new selection in the dropdown content control, you need to update the fields to reevaluate the conditions -- that doesn't happen automatically.

There are a couple of ways to do the updating. The quickest manual way is to go to Print Preview and back (in Word 2007/2010, press Ctrl+P and then Esc). To automate it, you'd need some macro code in the template for the documents. For example, this could go in the template's ThisDocument module:

Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, Cancel As Boolean)
    If ActiveDocument.Bookmarks.Exists("Entity") Then
        If CC.Range.InRange(ActiveDocument.Bookmarks("Entity").Range) Then
            ActiveDocument.Fields.Update
        End If
    End If
End Sub

Whenever you click in the Entity bookmark/content control and then click anywhere else, the fields will update.
