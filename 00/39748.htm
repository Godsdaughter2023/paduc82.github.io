The drive should be either MBR or GPT... Both of which are NTFS...

The GPT-FAT32 combination only applies to USB Flash Drives being used for bootable Installation Media...

The only thing I can suggest is to delete anything on the drive and try again...

If your machine is using UEFI Bios this drive must be GPT...

If you are using Legacy Bios it must be MBR...
