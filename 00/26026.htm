Compatibility of Pivot tables in excel 2003 versus excel 2007

We're using Microsoft Excel 2007 application to open and modify a file saved in Excel 2003. The xls (2003 version) file contains pivot table. However, the data is still embedded within the file, as the file size is the same as the original.

How do we contain the filters within the pivot even after saving the file?

 

bill manville say:

The XLS file format supports only limited use of filters (field value selections only) in pivot tables, which were introduced in Excel 2007. 

If you save in XLSX format then you the functionality should be preserved for Excel 2007, and Excel will be able to load it via the file compatibility pack that translates from XLSX to XLS format.  However you should avoid saving the file back to XLSX format in Excel 2003.

In general, it is wise to only use features that are present in the earliest version of Excel that will be used to access the workbook.

brian tillman [outlook mvp 2007-2019] say:

Outlook requires non-mail folders like Calendar, Contacts, Tasks, and Notes. IMAP is a mail-only protocol. It cannot handle non-mail folders like those. So Outlook flags them as "This computer only" so that you know they are local and not on the server. An alternative is to have a separate PST to contain those non-mail folders and then the IMAP folders don't need to contain them.

Deleting an IMAP account in Outlook will not affect any mail on the server. The person who believes they lost mail is, in my opinion, likely misinterpreting what happened. The following formula does what you want, assuming the value of Flash!$B$4 is Mar and not March as you wrote:

=SUMIF('Budget by Entity'!$A:$A,Flash!$F15,INDEX('Budget by Entity'!$B:$F,,MATCH(Flash!$B$4,'Budget by Entity'!$B$1:$F$1,0)))

I won't make any comment re. referencing full columns (i.e. 'Budget by Entity'!$A:$A which means > 1M of rows) while Tables (= dynamic ranges + other benefits) exist since Excel 2007 …

Alternatively a Pivot Table will do what you expect - efficiently!. The only adjusment to make is to add something in A1 so the column has a title/header
