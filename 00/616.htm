steve rindsberg say:

Make sure you're not saving to the older PowerPoint 97-2003 format. 

The older format allows only one set of tabs per text box.  The new version allows one set of tabs per paragraph.  When you save to the older format, the whole text box takes on the tab settings from the first line and you lose all the tab settings on the other lines. 

I suspect that's what's making your tabs go away.

As a test, set the tabs, then click the Office Button, point to Save As and choose PowerPoint Presentation from the flyout menu.  Give the file a new name and save.  See if your tabs persist when you close and reopen this newly saved file.
