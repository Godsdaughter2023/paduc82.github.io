stefan blom say:

To expand just a bit on Doug's reply, when you start a new line (by pressing Shift+Enter), the same paragraph formatting (such as alignment, indentation, tab stops etc.) still applies, because you have just added a new line within the same paragraph. Similarly, if you apply a paragraph style to a text range that contains line breaks, the style characteristics will be applied to the whole paragraph.

For a text range which is smaller than a paragraph, you can only change font formatting (bold, italic, underline, font name, font size, etc.) without also modifying the rest of the paragraph.
