slance310 say:

Hi,

You are not going to have much luck with all of your wishes... Some of them will work, but not all...

To install on the SSD, just do as before... Boot with the ISO either on a DVD or Flash Drive.

If you want to use UEFI BIOS features, the SSD will need to be formatted as a GPT type disk and Windows installed from a flash drive...

I don't think there is any way to save your files, settings, etc., on the new installation.

You can of course just copy your DATA files to the new drive, but the programs will need to be reinstalled. You want them on the SSD anyway if you want the full advantage of the SSD's speed for running programs...

After installation you can relocate your Documents, Pictures, and etc. to one of the other drives.

If desired, you can copy the contents of the ones from your original HDD to the new one...

Be sure to UNHOOK everything but the SSD during the initial install...

One other option if the SSD has enough room, would be to Clone your original to the SSD.....

You will need a cloning\imaging program that can resize the partition though...

I do this with Acronis True Image on a bootable CD... It's installation CD is already bootable...

I can't make any other recommendation for other programs that might work.

You can then manipulate where you want things afterwards....

Don't boot your machine with both OS's active though... It will most likely generate boot errors.
