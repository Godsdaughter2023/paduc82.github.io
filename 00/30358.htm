alden rey say:

Hi Andrew,

Whenever you delete a file or folder, Windows asks for your confirmation before moving it to Recycle Bin but Microsoft has removed this confirmation on some versions (Windows 8). Since you are using Windows 10, you can choose whether to have the file deleted permanently or not.

If you also want to re-enable delete confirmation dialog, kindly the follow the steps below:

  1. Right click on the Recycle Bin.
  2. Click on Properties, untick the option Don't move files to the Recycle Bin. Remove files immediately when deleted.
  3. Click Ok to save the changes.

For any concerns, don't hesitate to get back to us.
