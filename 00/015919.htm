Private Sub Worksheet_Change(ByVal Target As Range)
Dim rngDV As Range
Dim oldVal As String
Dim newVal As String
Dim strVal As String
Dim i As Long
Dim lCount As Long
Dim Ar As Variant
On Error Resume Next

If Target.Count > 1 Then GoTo exitHandler

If Target.Validation.Type = 3 Then
    Application.EnableEvents = False
    newVal = Target.Value
    Application.Undo
    oldVal = Target.Value
    Target.Value = newVal
    If Target.Column = 2 Then
        If oldVal = "" Then
            'do nothing
        Else
            If newVal = "" Then
                'do nothing
            Else
                On Error Resume Next
                Ar = Split(oldVal, ", ")
                strVal = ""
                For i = LBound(Ar) To UBound(Ar)
                    Debug.Print strVal
                    Debug.Print CStr(Ar(i))
                    If newVal = CStr(Ar(i)) Then
                        'do not include this item
                        strVal = strVal
                        lCount = 1
                    Else
                        strVal = strVal & CStr(Ar(i)) & ", "
                    End If
                Next i
                If lCount > 0 Then
                    Target.Value = Left(strVal, Len(strVal) - 2)
                Else
                    Target.Value = strVal & newVal
                End If
            End If
        End If
    End If
End If

exitHandler:
  Application.EnableEvents = True

a. user say:

Hi Terry,

Thank you for posting your query in Microsoft Community.

I understand that you wish to save your Microsoft Edge's favorites in your OneDrive. Perhaps the easiest way is to set the Favorites folder to be saved inside your OneDrive folder.

  1. If you don't already have one there, create a folder named "Favorites" inside your OneDrive folder.
  2. Right-click the Favorites folder, and choose Properties
  3. On the Location tab, click Move, and choose the Favorites folder you created in step 1.
  4. Confirm that you want to move your Favorites folder.
  5. You'll still see a Favorites folder inside your C:\Users\User_name folder, but it'll actually become a redirect, i.e. anything put there will actually be inside OneDrive (both on your hard drive and, of course, synced up to OneDrive cloud).  You can even do this for your Desktop folder and other special shell folders that have a location tab on their properties.

Let us know if the workaround works.

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.

Sub CopyLastDataToActiveRow()
    Dim rng As Range, DesitinationRow As Long, SourceRow As Long
