It's not that simple.

ChangePicture is not exposed in the object model (Unless I just cannot find it!)

You can call it from executeMso but this is just the same as clicking it and won't do what you need. You would have to write code to duplicate it. NOTE this code does not retain animationa as the real code does.

Sub pseudoCP()
On Error GoTo err
Dim opic As Shape
Dim osld As Slide
Dim newPic As Shape
Dim isPic As Boolean
Dim isTall As Boolean
Set opic = ActiveWindow.Selection.ShapeRange(1)
Set osld = opic.Parent
If opic.Type = msoPicture Then isPic = True
If opic.Type = msoPlaceholder Then
If opic.PlaceholderFormat.ContainedType = msoPicture Then isPic = True
End If
If Not isPic Then
err.Raise Number:=vbObjectError + 1000, Description:="Selection is not a picture"
Exit Sub
End If
If opic.Height >= opic.Width Then isTall = True
Set newPic = osld.Shapes.AddPicture("C:\Users\John\Desktop\Pic1.jpg", msoFalse, msoTrue, 0, 0, -1, -1)
newPic.LockAspectRatio = True 'should already be set but worth checking
If newPic.Height >= newPic.Width And isTall = True Then
newPic.Height = opic.Height
newPic.Top = opic.Top
newPic.Left = opic.Left + opic.Width / 2 - newPic.Width / 2
Else
newPic.Width = opic.Width
newPic.Left = opic.Left
newPic.Top = opic.Top + opic.Height / 2 - newPic.Height / 2
End If
opic.Delete
Exit Sub
err:
MsgBox err.Description
End Sub

mrs axe say:

Developer Tab in Word

suzanne s. barnhill say:

Instead of creating another form field (whether legacy Drop-down Form Field or Drop-down List Content Control), you can add another field that references the value selected in the original drop-down.

 

The article at http://gregmaxey.mvps.org/word_tip_pages/repeating_data.html explains how to do both. Section 2 explains how to use a cross-reference (REF) field referencing the built-in bookmark of a legacy form field to reproduce its content. Section 7 explains how to do this with content controls. I have tested the "Simple Linked Content Controls" method, and it works very simply.

jay freedman say:
I think Mrs Axe's intention is to have two separate dropdowns, containing identical lists of choices but able to select different values.

To do that, just select the first dropdown, copy it to the clipboard, and paste a copy in the location for the second dropdown.

If you used the content control dropdown (that is, the third icon on the second row of the Controls group on the Developer tab), selecting the dropdown is done by clicking the little blue rectangle (containing three dots) on the left side, so it changes from light blue to dark blue. Then you can copy it with Ctrl+C. When you paste, use Ctrl+V (or, if you look at the Paste Options button, use Keep Source Formatting).

If you used a legacy dropdown form field, just select the field, copy, and paste. The one caveat is that if you plan to refer to the field in a macro, you need to go to the Properties dialog of each dropdown and insert a valid bookmark name. Because each bookmark in a document is unique, copying and pasting a field that has a bookmark name gives that name to the new field and leaves the old field with a blank name.

a. user say:

Hi jpfairman,

 

The icon that you see is Windows Live Mail. Outlook.com is different from Windows Live Mail. Outlook.com is the new web-based email service of Microsoft (will eventually replace Hotmail), whereas Windows Live Mail is an email client or program which is installed in your computer.

 

Outlook.com is a new web-based email service of Microsoft. Outlook.com was designed with fresh and clean user interface. It gets the clutter out of your way-the header has 60% fewer pixels and there are 30% more messages visible in your inbox than the webmail most people are used to. There are no display ads or large search boxes that take up extra space. For more information about Outlook.com, click here.

 

By the way, Windows Live Mail will function properly if you add your email account on it. Once your email account is in place, the contacts and emails in your email account will be shown in Windows Live Mail. For more information on how to put your email account in Windows Live Mail, click here.

 

To get Outlook.com on your taskbar (it will be pinned under Internet Explorer icon if it is your default browser), you have to create a desktop shortcut of it first. Here's how:

 

1. Go to your desktop and press the right mouse button

2. Select New and click Shortcut

3. On the Type the location of the item field, type http://www.outlook.com

4. Click Next

5. On the Type a name for this shortcut field, type the name you want

6. Click Finish to create the shortcut

 

Once you have created the shortcut, you can now pin it on your taskbar by dragging the icon to it.
