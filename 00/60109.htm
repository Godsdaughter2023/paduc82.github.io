Can I perform a clean install using the Free upgrade?

No, it will require that you are running a previous qualifying version and start the upgrade from within the qualifying version. You can initiate a clean install after completing the Upgrade.

You must first upgrade from the qualifying version of Windows 7 or Windows 8/8.1 (whether you do it through Windows Update or using the .ISO file). Ensure the upgrade is completed successfully and then ensure that it is activated. You can then proceed to do a clean install by using recovery media or using the Reset function in Start > Settings > Update & Security > Recovery > Reset this PC (Get Started).
