hansv mvp say:

Copy the following code into a module in the Visual Basic Editor:

Const pi = 3.14159265358979

  If Abs(ang) <> 1 Then
    ACos = pi / 2 - Atn(ang / Sqr(1 - ang * ang))
  ElseIf ang = -1 Then
    ACos = pi
  End If
End Function

    Radians = ang * pi / 180
End Function

You can now use ACos and Radians in your expressions.

Hi,

here are two macro examples. The first creates checkboxes in the selected cells, the second deletes all checkboxes in the active sheet:

Sub CreateCheckbox()
Dim rngCell As Range
  For Each rngCell In Selection.Cells
    ActiveSheet.CheckBoxes.Add(rngCell.Left, rngCell.Top, rngCell.Width, rngCell.Height).Select
  Next
End Sub

Sub DeleteCheckbox()
Dim shpShape As Shape
  For Each shpShape In ActiveSheet.Shapes
    If shpShape.Name Like "*Check*" Then
      shpShape.Delete
    End If
  Next
End Sub

jay freedman say:
Thanks for all your help, I'll try this.  I'm still rather astonished however that Word doesn't have a simple setting that just says "number pages consecutively throughout the entire document," which would seem like a standard default option.

There is a way to do that, but it requires a macro (and I agree, it should be a button in a dialog somewhere). Read http://www.gmayor.com/installing_macro.htm if needed...

Sub ContinuousPageNumbers()
    Dim secNum As Long
    With ActiveDocument
        For secNum = 2 To .Sections.Count
            .Sections(secNum).Headers(wdHeaderFooterPrimary) _
                 .PageNumbers.RestartNumberingAtSection = False
        Next
    End With
End Sub

If the first section is frontmatter and the numbering really should restart at 1 in Section 2, then change the starting value of secNum in the For statement from 2 to 3.

The simplest way is to remove all the section breaks. Again, headers and footers are creatures of sections. You can replace ^b (section break) with ^p (paragraph mark). This will leave you with one set of headers and footers / page numbers to configure and deal with.

There is a big difference between a manual page break (which should not be used) and next-page section breaks. Instead of manual page breaks, it is preferable to use page break before formatting in a heading style, usually Heading 1 or Heading 2.

A section break is used when you want to (1) change orientations, (2) change column settings, (3) change header and footer configurations, change or restart page numbering.

If you just want to change the content of your header/footer to reflect the document content, you can use a StyleRef field. A subpart of changing headers and footers is to designate a new page (beginning of a chapter) as a first page for purposes of different first page settings.
