macropod say:

If you add the following macro to Word's 'Normal' template and assign a shortcut key to it (e.g. Alt-/), Word will pop up a dialog you can use for inputting of fractions whenever you use that shortcut. The resulting fraction will be given a similar format to the built-in fractions.

Sub EnterFraction()
'Prompt user for the text to format as a fraction

fraction = InputBox("Enter fraction text (ex: ½, 5/32)")
'Find the position of the slash in the text
slashPosition = InStr(fraction, "/")
'Quit if no slash in the text
If slashPosition = 0 Then Exit Sub
preSlash = Left(fraction, slashPosition - 1)
postSlash = Right(fraction, Len(fraction) - slashPosition)
With Selection
  'Type text before the slash as superscript
  .Font.Superscript = True
  .TypeText preSlash
  'Type the slash as normal text
  .Font.Superscript = False
  .TypeText "/"
  'Format text after the slash
  .Font.Subscript = True
  .TypeText postSlash
  'Continue with font back to normal
  .Font.Subscript = False
End With
End Sub

For PC macro installation & usage instructions, see: http://www.gmayor.com/installing_macro.htm
For Mac macro installation & usage instructions, see: https://wordmvp.com/Mac/InstallMacro.html

MS Project- How to assign different resources to a task? - Microsoft Community

efeatikkan say:

MS Project- How to assign different resources to a task?

Hey, 

when more than one resource is  assigned to a task, it uses one of them. However I want to assign 2 (or more) different resources to tasks. For example, I wan to assign Machine1 and Human1 resources (which does not have the same role and can not beused for one another) to tasks. So the task to consumes different pool for each resource.

Thanks!! 

john - project say:

EfeAtikkan,

First, create the resources on the Resource Sheet. Labor and machine resources are "work" type, consumables (brick, mortar, paint, lumber, etc.) are "material" type and expenses (travel, car rental, hotel, etc.) are "cost" type.

Labor resources normally have a time-based cost ($/hr) and machine resource may have a time-based cost or may have a per use cost (leased machinery).

Perhaps the easiest way to assign resources to each task is to use the Gantt Chart view and then open the Assign Resources window (Resources > Assignment group > Assign resources). You can specify how much of each resource should be assigned to the task. Project will spread the assignments linearly over the task duration but you can edit that resource loading in the timescale part (right side) of either the Task or Resource Usage views.

Hope this helps.

John
