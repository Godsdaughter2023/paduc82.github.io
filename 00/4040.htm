Hi Agonzalez1078,

There is a PowerShell script below to generate all the members of the Office 365 groups in your Office 365 tenant and then export the result to a CSV file as below:
$Groups = Get-UnifiedGroup -ResultSize Unlimited

$Groups | ForEach-Object {

$group = $_

Get-UnifiedGroupLinks -Identity $group.Name -LinkType Members | ForEach-Object {

      New-Object -TypeName PSObject -Property @{

       Group = $group.DisplayName

       Member = $_.Name

       EmailAddress = $_.PrimarySMTPAddress

       RecipientType= $_.RecipientType

}}} |

Export-CSV "D:\\Office365GroupMembers1.csv" -NoTypeInformation -Encoding UTF8

And to get the owners, you would change the script to below:
$Groups = Get-UnifiedGroup -ResultSize Unlimited

$Groups | ForEach-Object {

$group = $_

Get-UnifiedGroupLinks -Identity $group.Name -LinkType Owners | ForEach-Object {

      New-Object -TypeName PSObject -Property @{

       Group = $group.DisplayName

       Owner = $_.Name

       EmailAddress = $_.PrimarySMTPAddress

       RecipientType= $_.RecipientType

}}} |

Export-CSV "D:\\Office365GroupOwners1.csv" -NoTypeInformation -Encoding UTF8
