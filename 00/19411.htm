john w. vinson say:

Any multiuser database should (I'd almost say must) be split into a "Backend" containing the tables, and multiple copies of a "Frontend" containing all the forms, reports, queries and code. See http://www.granite.ab.ca/access/splitapp.htm for a thorough discussion. However, such split databases do NOT work well over a wide-area connection such as the internet!

 

What I would recommend instead is that you have the backend on a shared folder; each user would have their own copy of the frontend in a folder on the same central machine (or at least on the same fast, stable Local Area Network) as the backend. They could then use GoToMyPC or some other remote terminal software to connect to the host machine and open their own private frontend. This will allow multiple users to work with the same set of shared data.

 

albert d. kallal say:

I guess what will work will much depend on the resources and what kind of system you have in place now. I think in terms of the least changes, some type of Remote Desktop or Terminal Services is a great choice as suggested here.

The low cost p1 plan starts at $6 per month, and would allow you to install Access (even the runtime) on each desktop, but they would be all sharing the data tables.  There are some limitations with this setup, but this setup could very well work for you.

how to reduce the amount in all cells in a column - Microsoft Community

solkeys say:

how to reduce the amount in all cells in a column

I have Office 2003.In Excel I want to  reduce all the numbers in a column by 20%. How can I do it?
pete_uk say:

Type the value 0.8 in a blank cell somewhere. Select that cell and click <copy>. Then move the cursor so as to select all the cells in the column where you want to reduce the values. Right-click the selection and choose Paste Special. Then check against Values and Multiply, then click OK and then <Esc>.

 

Then you can delete the contents of the original cell.

 

Note that reducing by 20% is the same as multiplying by 80%, and 0.8 is another way of representing 80%.

 

Hope this helps.

 

Pete
