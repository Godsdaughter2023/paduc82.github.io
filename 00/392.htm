fredk2008 say:

Using SUMPRODUCT across 30 spreadsheets

I have installed "MOREFUNC" on Excel Office 2010.  I have 30 tab spreadsheet, and trying to use SUMPRODUCT across all sheets.

This is formula that I am using:

=SUMPRODUCT(--(threed('A1:A20'!$E$4:$E$63)=B6),--(threed('A1:A20'!$B$4:$B$63)=B1),--(threed('A1:A20'!$I$1:$I$63)))

A1:A20 - are the tabs
B6 & B1 are the conditions on the main sheet, where the SUMPRODUCT formula is,
$E$4:$E$63 or so on are the cells on sheet A1 to A20.

Does MOREFUNC work with Excel 2010?  I tested SUMIF across A1:A20, and it works - but SUMPRODUCT does not work!

Any ideas?

t. valko say:

Biff:

I wrote the following SUMPRODUCT for a single sheet:

=SUMPRODUCT(--(B4:B63=AC1),--(E4:E63=AA2),--(I4:I63))

And, it still gave an error message: "#VALUE!".  What is wrong with this equation?

I gave up on idea of running SUMPRODUCT across multiple sheet, why is it giving error message when using it on a single sheet?

Fred


You either already have #VALUE! errors in one or more of the ranges or there are TEXT entries in I4:I63.

The double unary operation on certain text = #VALUE!.

Try it like this:

=SUMPRODUCT(--(B4:B63=AC1),--(E4:E63=AA2),I4:I63)

--
Biff
Microsoft Excel MVP

 

fredk2008 say:

Biff:

Your suggestion worked, there was a text somewhere in the columns, resulting in error message VALUE#!

"ediardp" , now I will try your way by adding up SUMPRODUCT for each sheet.  I will let you know shortly if this works!

IT WORKS, IT WORKS, IT WORKS.  "ediardp" using SUMPRODUCT across multiple sheets works:

HAVE TO WRITE SUMPRODUCT FOR EACH INDIVIDUAL SHEET, i.e.:

=SUMPRODUCT(--(A1!$E$4:$E$63)=B6),--(A1!$B$4:$B$63)=B1),--(A1'!$I$1:$I$63)))+SUMPRODUCT(--(A2!$E$4:$E$63)=B6),--(A2!$B$4:$B$63)=B1),--(A2'!$I$1:$I$63)))+.......... up to A20
