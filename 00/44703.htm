cmdrkeene say:
"I have an instance of wandering where the top icon of an isolated tower of icons located near the right side of the desktop winds up in the "first available" position at the left side of the desktop after I open it by clicking on it and then adding to its contents another entry (all entries are shortcuts to files) and then closing the folder."     

This can happen with some file types where the program creates temporary copies.  For exmaple, if you turn on "show hidden files and folders" in your folder options and then open a Word file, you'll see a new transparent file icon appear -- this is where your changes are being saved.  When you exit Word, the original is deleted and the temp file is saved as the document.  That's why editing a Word file will appear to make the icon move. In reality it didn't move: it was deleted entirely and a new file saved on the desktop. Since the new file never existed, it doesn't have an old location and is placed in the first available lattice spot.
