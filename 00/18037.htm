jay freedman say:
It isn't necessary to use the Replace function at all. Just modify the Footnote Reference style, which is automatically applied to all the footnote indicators both in the text and in the footnotes.

Press Ctrl+Alt+Shift+S to open the Styles pane on the right. If you don't see a lengthy list of style names, click the Options link at the bottom and choose to show all styles. Locate the Footnote Reference style, right-click it, and choose Modify.

In the Modify Style dialog, click the "New documents based on this template" option. Then click the Format button at the bottom left and choose Font.

In the Font dialog, go to the Advanced tab. After choosing Old-style in the Number Forms box, go near the top of the dialog and set the Position boxes to Raised by 2 pt (or 3 pt or whatever looks good). OK both dialogs.

Your numbers should now all be Old-style and raised.

suzanne s. barnhill say:

Footnote Reference is a character style, so if you select a footnote reference mark and clear formatting (Ctrl+Spacebar), it will be reset. But you can easily reapply the Footnote Reference style (after defining it to your preferences).

  1. Ctrl+H to open the Replace dialog.
  2. Click More to expand the dialog.
  3. In the Find what box type ^f for the footnote reference mark (^e for endnote references).
  4. With the insertion point in the (empty) Replace with box, click Format, then Style, then Footnote Reference (or Endnote Reference). Click OK.
  5. Click Replace All.
