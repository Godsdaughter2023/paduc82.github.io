rpbiship say:

I believe I have the final solution.... It is not too terribly different from the original posting.

 

The macro that runs in the add-in is as follows:

Sub auto_open()
If Ucase(ActivePresentation.Name) = upper case file name Then
    Dim osld As Slide
    Dim oshp As Shape
    For Each osld In ActivePresentation.Slides
    For Each oshp In osld.Shapes
        If oshp.Type = msoLinkedOLEObject Then
            oshp.LinkFormat.Update
        End If
        Next oshp
    Next osld

    ActivePresentation.Save
    ActivePresentation.SaveAs (slideshow file name)
    ActivePresentation.Close
End If
End Sub

 

Note that the check for the presentation name is critical because once you establish a add-in, then any presentation you open will attempt to execute this code. Saving the presentation preserves the links update (they have been set to manual) and saves a copy for the slide show.

Next is the bat file that executes each night to automatically refresh the links:

REM This routine simply opens an excel Workbook in order to
REM recalculate the link showing how many days. This is done by executing a
REM macro to force re-calculation now, then saving and closing the application

echo script started: %date% %time% >>logname
echo "Kill Slideshow" >>logname
taskkill /f /im powerpnt.exe
echo "Kill EXCEL" >> logname
taskkill /f /im excel.exe
echo "EXCEL" >>logname
start EXCEL.exe excel filename
ping server1 -n 10 -w 1000
echo "Update Powerpoint Links" >>lta_update.log
start /min powerpnt.exe powerpoint file (.pptx or .pptm)

ping server1 -n 10 -w 1000
echo "Start slidewhow" >> logname
start powerpnt.exe /s powerpoint slide show
echo script stopped: %date% %time% >> logname

 

Note the insertion of ping commands; these are required to allow for the MS applications to completely open before executing the next command.

a. user say:

Hi,

Yes, you can block a specific site in Internet Explorer. I will be happy to answer your query.

Let me ask you;

  • Which operating system is installed in the PC?
  • Are you saying that you do not want to view that specific site in your search result through any search engine?

a. Open Internet Explorer.

b. Go to Tools-> Internet Options.

c. Click on Privacy tab and click Site tab

d. Type the address of the website you wish to block and click Block tab

e. Click OK, Apply and then OK

If you are saying that the website is showing in search results from any search engine, it will show up, however, that site will not be accessible when click on it.

Hope this information is helpful. Please feel free to reply in case you face any other issues with Windows in future.
