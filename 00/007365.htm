The address on an envelope is formatted with the Envelope Address style, which uses a frame to position the text. Microsoft made a mistake, starting several versions ago, in making that frame use a fixed size. suzanne s. barnhill say:

Here's a quick way to fix it:

  1. Start with a blank document.
  2. Open Mailings | Create | Envelopes.
  3. Type anything in the "Delivery address" box.
  4. Click Add to Document. This gives you an envelope attached to the blank document.
  5. Right-click in the address text and choose Style... from the context menu.
  6. This will open the Style dialog with Envelope Address selected. Click Modify...
  7. In the Modify Style dialog, click Format.
  8. From the dropdown menu, select Frame...
  9. In the Frame dialog, change the "Height" setting to Auto.
  10. Change the "Vertical position" setting from Bottom to some reasonable amount; in earlier versions of Word, for a #10 envelope, this was 2.13" Relative to Page.
  11. Click OK to close the Frame dialog.
  12. In the Modify Style dialog, select the radio button for "New documents based on this template."
  13. Click OK to close the Modify Style dialog.
  14. Click Apply or Close to close the Style dialog.
