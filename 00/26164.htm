Hi,

in a query you can use your formula:

Result1: IIF([Salutation]="Dr.", "Dr.",IIF([Gender]="M", "Mr.", "Ms."))

also in a form or in a report control you can use the same formula as control source:

=IIF([Salutation]="Dr.", "Dr.",IIF([Gender]="M", "Mr.", "Ms."))

Ciao Mimmo

scottgem say:

Ok, what you do is create a Combobox for the user to select a degree. The relevant properties of that combo are:

RowSource: SELECT DegreeID, DegreeName, Salutation FROM tblDegrees ORDER BY DegreeName;

BoundColumn: 1

ColumnCount: 3

Columnwidths: 0";2";0"

Then in the After Update event of that combobox you add a line of code:

Me.Salutation = Me.cboDegree.column(2)

If Me.Salutation <> "Dr." Then

     If Me.Gender = "M" Then

          Me.Salutation = "Mr."

     Else

          Me.Salutation = "Ms."

     End If

End If

This will fill in the Dr. if the degree is a doctorate, otherwise it either leave Salutation blank or fill it in with whatever you store with the Degree. If its not Dr. it will check the Gender field and assign the Salutation accordingly.
