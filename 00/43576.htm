Hi Mike,

.asx file is not a media file. It is a file which contains a pointer to some online content. So if you are unable to play .asx file and are able to play other files using Windows Media Player then the best thing is to open the .asx file using a Notepad. When you open the .asx using the notepad you will see a link which starts with http: and so on. Please copy that link and go to Windows Media Player. On the Windows Media Player window press the Alt key on the keyboard and click on File and open url. Paste the link and check if Windows Media Player is able to open the link. If it opens the link then the it is a problem with the formatting of the .asx file and if it does not open the link then it is a problem with the online server of the .asx file.
