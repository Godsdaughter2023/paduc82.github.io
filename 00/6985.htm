If you add that one-line macro, you can assign a shortcut to it with Tools > Customize Keyboard (most Control + Key sequences are available for assignment). Then, this would be the mouseless sequence:

  1. Press the macro shortcut
  2. Press Tab to move to the first field
  3. Define the "Replace" (wrong) sequence
  4. Press Tab to move to the second field
  5. Define the "With" (right) sequence
  6. Press Return twice

If you select/highlight the "right" sequence in the Word document before calling up the macro, then you'll only have to type the "wrong" sequence:

  1. Press the macro shortcut
  2. Press Tab
  3. Type the wrong sequence
  4. Return, Return
