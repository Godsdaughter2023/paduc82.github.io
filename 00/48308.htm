Windows Vista Ultimate, Windows 7 Ultimate (there is no Ultimate edition of Windows 8) - The Ultimate SKU means these are the most complete editions consumers can purchase at retail and be able to use all features and functionality of Windows.

Windows Vista Business, Windows 7 Professional, Windows 8 Pro - This is typically the medium to small business edition of Windows. It is also utilized by Power Users who need to utilize it on power PCs such as gaming desktops with lots of memory or work stations. It is also used in Enterprises where I have seen it deployed. These versions and editions are actually the logical upgrades from Windows XP Professional.

Windows Vista Enterprise, Windows 7 Enterprise and Windows 8/8.1 Enterprise - This is relatively new edition that was introduced in 2006 with the release of Windows Vista. It is aimed at very large businesses that do large roll outs of Windows. Think of businesses that are on an international scale with regional offices around the world, employees who speak different languages, complex requirements and needs for accessing corporate resources on remote computers. Most home users will never need this edition and to even take advantage of its unique features, you need to have a Windows Enterprise/Datacenter server deployed in your organization.

Purchasing it is also another complexity. It is available to volume license customers who are deploying a certain amount of computers with it. Also, activation requirements are different for it depending on how you purchase licenses. While you can buy the other editions in a Store, you can only purchase Windows Enterprise edition through a Value Added Reseller and you even have purchase select type of Volume Licensing Agreement called Software Assurance just to qualify for it.

Other previous editions:

Home Premium (Windows Vista/Windows 7) - for home users

Home Basic (Windows Vista/Windows 7) - for home users in emerging markets such as parts of Asia, Africa, Caribbean.

Starter edition (Windows Vista/Windows 7) - for developing markets where computers are expensive but users only need the basics such as connecting the Internet, email, preparing documents. This edition when available under Windows Vista and XP has restrictions such as 1024 x 768 resolution, only 5 applications could be open at a time.

Since the release of Windows 8, these are the only flagship editions of Windows:

Windows 8 Core - analogous to Windows 7 Home Basic

Windows 8 Pro - analogous to Windows 7 Professional

Windows 8 Enterprise - analogous to Windows 7 Enterprise

Windows RT - new platform based on Windows 8 for devices with the ARM processor, guaranteeing longer battery life and modern applications on mostly mobile devices.
