Delete rows if condition is not met - Microsoft Community

rt_56 say:

Delete rows if condition is not met

Is it possible to delete rows if condition is not met?
On my worksheet, I have a column of dates, if date value is not between two dates then remove row, but leave row, if blank.  Note: Date values will vary from time to time.

This will need to be a macro or lines of code to place in a pre-existing macro.
andreas killer say:
The user will also need to keep the rows where the cell value is blank in addition if the date values are between the two dates the user selects.
Maybe I misunderstood  you?

This presupposes that the dates must be sorted in ascending or descending order.

Andreas.

Sub Test()
  Dim Source As Range, DateCol As Range, All As Range
  Dim S As String
  Dim FromDate As Date, ToDate As Date
  Dim BetweenDates As Boolean

  'Get the dates
  Do
    S = InputBox("FromDate", , S)
    If S = "" Then Exit Sub
  Loop Until IsDate(S)
  FromDate = CDate(S)
  Do
    S = InputBox("ToDate", , S)
    If S = "" Then Exit Sub
  Loop Until IsDate(S)
  ToDate = CDate(S)

  With ActiveSheet
    'Get used range in column A
    Set DateCol = .Range(.Range("A1"), .Range("A" & Rows.Count).End(xlUp))
  End With

  'Visit each cell
  For Each Source In DateCol
    If IsDate(Source) Then
      'Date between our dates?
      BetweenDates = Source >= FromDate And Source <= ToDate
    End If
    'if date value is not between two dates then remove row, but leave row, if blank.
    If Not BetweenDates And Not IsEmpty(Source) Then
      'Remember this row
      If All Is Nothing Then Set All = Source Else Set All = Union(All, Source)
    End If
  Next
  'Delete the rows if any
  If Not All Is Nothing Then All.EntireRow.Delete
End Sub
