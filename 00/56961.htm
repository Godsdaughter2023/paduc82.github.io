p.b.805 say:

After enough playing around on my own, I fixed my issue. I'm sure this will fix yours too Amir, or anyone else who has this problem.

I changed the properties of my microphone inside windows to a setting that shouldn't work right but it does, the fix was instant.

I went to the Control Panel, then selected Hardware and Sound, then selected Manage Audio Devices under the Sound section. From here I selected the Recording tab, then selected my Microphone that I was having trouble with. I clicked on Properties at the bottom of the window, then selected the Advanced tab at the top.

Once I was finally into the microphone settings, I changed the default format setting from "1 channel 16 bit 44100Hz" to "2 channel 16 bit 44100Hz" of course the Hz setting is your choice of quality but I've always used 44100Hz. Anyways, you wouldn't think you would want your microphone set at 2 channel but it worked. Inside my audio software it is still selected as mono and it works great now.

I've seen this out there on other forums, hope this helps everyone with this issue. Good Luck!

42. Restart PC (now up to date)

uselessfactoid say:

Quite by accident I just discovered a better workaround that doesn't require any other software package. All you need is an existing photo with the location tags set.


You'll end up modifying the location on this photo so either duplicate it first or you can erase the new location and reset the old one after the fact. What we do here is modify the location so that a new location gets added to the cache I mentioned earlier. Once its cached you can add it new to any photos using the geo tag field normally.


For the photo with the location (geotag) field already set, right click on the name of the location on the right side bar ("Tag and caption pane"). This is the blue hyperlink text with the name of the city and state. Select the "Rename Location" option from the right click menu that appears. This will display a popup dialog with fields for the city, state, and country (I don't know what the "location" one does, seems like its always blank; I just ignore it). Fill out the fields appropriately for the new location that you want to be able to select for other photos and hit save.


The location on the photo will change to this new location but the old location and newly entered one will both be in the cache now so you can now add either to any other files by starting to type the location name into the geotag field for any photo. If the modified file was a copy you can just delete it now or if it was an original just set the location back to whatever is appropriate.
