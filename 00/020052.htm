MS Word 2007: How to create custom toolbar in ribbon - Microsoft Community

abhishekgupta08 say:

MS Word 2007: How to create custom toolbar in ribbon

Hello,

I want to use few macros in my document. I have already created those and getting these in QAT.

Now I want to create custom toolbar in word 2007. How could I create custom toolbar?

- Abhishek

hansv mvp say:

There is no way any more to create a custom toolbar in the interface; you have to use VBA. Here is a simple example. The toolbar will be stored in the active document, and it will be displayed in the Add-ins tab of the ribbon.

Sub CreateToolbar()
    Const strName = "MyToolbar"
    Dim cbr As CommandBar
    Dim cbb As CommandBarButton
    CustomizationContext = ActiveDocument
    On Error Resume Next
    CommandBars(strName).Delete
    On Error GoTo 0
    Set cbr = CommandBars.Add(strName)
    cbr.Visible = True
    Set cbb = cbr.Controls.Add(msoControlButton)
    With cbb
        .Caption = "Macro 1"
        .TooltipText = "Please click me. Please!"
        .Style = msoButtonCaption
        .OnAction = "MyMacro1"
    End With
    Set cbb = cbr.Controls.Add(msoControlButton)
    With cbb
        .Caption = "Macro 2"
        .Style = msoButtonIconAndCaption
        .FaceId = 49
        .OnAction = "MyMacro2"
    End With
End Sub
 

Sub MyMacro1()
    Beep
End Sub
 

Sub MyMacro2()
    MsgBox "This is macro 2", vbInformation
End Sub

robert aldwinckle on forums say:

Why can't I remove the 'Favorites' bar?  It's redundant and a waste of viewing area.


There are three parts, only one 1 required:  the Favorites Center star button (Alt-c) and it can be minimized to just its icon, e.g. by using Command bar, Customize..   The other parts are:  the Favorites bar (Ctrl-i) which only shows when you request it and the Favorites Bar (aka Links bar) which is optional (same as the Links bar was).  

Note that the Favorites Center star button also doubles/triples as the History button and Feeds button, depending on the last use of the Favorites Center (or whether you pressed Ctrl-h, etc.)   What's really missing IMO is giving the button a more appropriate icon to indicate which mode the Favorites Center is currently in.   People might not be so prone to complain about that button if they realized that it could represent easy access to their History, View By Order Visited Today, for example.
