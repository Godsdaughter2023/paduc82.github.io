ralf_g say:

If a WiFi connection requires a password Windows will prompt you to enter it the first time you try to connect to that site. Windows can remember the password and even has settings to autoconnect to that network if you choose.

There are many modem/router settings that could prevent a computer from connecting to a WiFi network, or even from seeing it at all.

If the home WiFi source (modem?, router? combo?) is set to not broadcast its SSID then the computer won't see the connection unless the SSID has already been manually entered into the computer wifi settings and  that WiFi  is also set to connect whether SSID is broadcast or not.

The WiFi source and computer must be using the same radio frequency or they won't see/connect to each other (2.4Ghz or 5Ghz)

The encryption type and password used in the WiFi source have to be the same on the computer's WiFi adapter.

If the computer WiFi adapter is not using automatic DHCP addressing then it has to be manually set to be on the same local area subnet that the WiFi source is using.

If the WiFi source has MAC filtering enabled then the MAC address of the computer WiFi adapter has to be added to the allowed list or it won't connect.

If the WiFi source has limited the number of DHCP addresses and those are already assigned to other devices then your new computer wouldn't be able to connect.

The WiFi source and computer must be using the same WiFi mode as each other or they can't connect to each other, ie. 802.11a, 802.11b, 802.11g, 802.11n, 802.11ac. Usually either end of the connection is capable of 'falling back' from a faster speed to a slower one but it is also possible to lock either the source WiFi or the computer WiFi into one or another of these modes. If they are not using the same mode or don't have fall-back enabled they won't connect to each other.

Example (wireless modes from a Dlink router) Both 802.11n and 803.11g enabled WiFi devices can connect with the setting that is selected but 802.11b can not.

If your computer WiFi is connecting pretty much every place else other than your home then the problem is most likely the settings in your home modem or router or whatever is providing the WiFi signal. Anyway, all of those things have to be checked at your end. Someone will have to go through the modem/router Wifi and network settings and then match the computer WiFi settings up with them. None of the settings on routers or modems etc. are Windows related, only the computer's WiFi adapter settings are.
