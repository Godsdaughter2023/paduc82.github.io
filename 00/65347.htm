Hi Nicholas,

Thank you for keeping us updated about the issue.

Do you want to remove the password of your Local Administrator Account?

Administrators can change security settings, install software and hardware, and access all files on the computer, if you don’t have an Administrator account

  1. You will not be able to setup a program or install a program that you want to install as it asks for permission to modify your system files. You won’t be able to make changes to other user accounts.

Example: When a program needs to modify your system files or settings, and you're not an administrator, Windows will throw up a "User Account Control" (UAC) window, asking for your permission before it allows the program to continue. If you have set a password for your admin account, you'll have to enter your admin account password before clicking the "Yes" or "OK" button for that dialog box.

   b. You would not have any rights to make changed in the registry files of your computer.

   c. You would not have any control over your security/virus protection software.

   d. No changes can be made in Windows Defender.

If you want to delete your password from the account follow the step below

Remove the password from PC settings

Settings - Change PC Settings - Accounts -  Users - Sign in option - Change (Password) -  Enter the password - next - (Leave the fields blank for new password) click Next and Finish.
