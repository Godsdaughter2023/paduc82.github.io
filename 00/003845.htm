Mac Word does not have the same protection features provided in Word for Windows. It isn't clear who "You" refers to or what options you've been given, but those that pertain to Mac Word probably are complete as stated.

By using Section Breaks you can protect some Sections & leave other Sections unprotected. This prevents users from changing anything in Protected sections but allows full editing of unprotected Sections. There is no equivalent to the Restrict Editing option. Further, if the document was created using Protection features available only in a Windows version the document will be fully protected when opened in Mac Word.

hansv mvp say:

Yes, many developers frown upon the use of SendKeys. SendKeys sends keystrokes to the application as if the user had typed them on the keyboard. But since the user can have many applications open in Windows, there's always a small risk that the keystrokes will be picked up by the wrong application if it happens to become active just as the keystrokes are sent, and this could have unexpected effects.

But since Microsoft has not provided an "official" way to set preferences for the Find dialog, this is the only way.
