You can print any presentation on any size paper your printer will handle.  The result will depend on the presentation setup and the printing options you choose.

If you choose "Scale to fit paper size" in the print dialog box, Paper Handling sectiion | Destination Paper Size, PPT will make your slides as large as possible on the chosen paper size.  Remove the "Scale to fit" checkmark and it'll print at the current slide size, centered on the chosen paper.

Subtract then Divide then Add?!? - Microsoft Community

super95onestop say:

Subtract then Divide then Add?!?

I need the correct formula to subtract one number from another...divide the result by two and then add one of the original numbers.

A5 minus A1 = X divided by 2 plus A1 

I've tried a couple different ways but it's definitely not working.

jllatham say:

The x divided by 2 plus A1 part would be written in a formula as

(x/2)+A1

But the parenthesis should not be needed - normal math precedence should handle it.

 

If you are looking for code to use in a VBA (macro) routine then something like this, and assumes that the value of X has already been set up somewhere before this part of the code runs.

 

Temp1 = Worksheets("Sheet1").Range("A5") - Worksheets("Sheet1").Range("A1")

Temp2 = X / 2 + Worksheets("Sheet1").Range("A1")

IF Temp1 = Temp2 Then

   'do stuff here if they are equal

Else

   'do stuff here if they are not equal

End if

 

You could write it without the Temp1 and Temp2 values like

IF Worksheets("Sheet1").Range("A5") - Worksheets("Sheet1").Range("A1") = _

 X / 2 + Worksheets("Sheet1").Range("A1")Then

   'do stuff here if they are equal

Else

   'do stuff here if they are not equal

End if

 

BEWARE: you may not always get equality when you expect it due to the way the standard for floating point number format in digital systems is written.  You may have some really small value bit or bits set when you think the result is 0, Excel might think one of them is .0000000000001

 

So it would be best to decide how many decimal points of precision you need and include that as part of your formula, like this:

Temp1 = Worksheets("Sheet1").Range("A5") - Worksheets("Sheet1").Range("A1")

Temp2 = X / 2 + Worksheets("Sheet1").Range("A1")

IF Round(Temp1,2) = Round(Temp2,2) Then

   'do stuff here if they are equal to 2 decimal places

Else

   'do stuff here if they are not equal

End if

tasos_k say:

Sub macro_02()
Set ws = Sheets.Add(before:=Worksheets(1))
ws.Name = "ALL_NEW"
For i = 2 To Sheets.Count
Cells(i - 1, 1) = Worksheets(i).Name
Cells(i - 1, 2) = Worksheets(i).[C5].Value

Cells(i - 1, 3) = Worksheets(i).[D17].Value
Next

Cells(Sheets.Count , 2) = "SUM = "
Cells(Sheets.Count , 3) = "=SUM(C1:C" & Sheets.Count - 1 & ")"
Columns("A:C").AutoFit
End Sub
