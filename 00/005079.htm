If you know the process name and location AnyWeBlock is going to run (it should show in Task Manager along with the path when the program runs) or the program's filename or the active file it uses for the program, or know which file is being blocked (not the program - you can't exclude by program, but the file or maybe the folder), then all you need to do is to open MSE and go to Settings and exclude the files/processes/locations (at the smallest level possible) that are being falsely blocked.  That will prevent MSE from scanning whatever you excluded and thus it won't happen again - but keep in mind that everything you exclude is a tiny hole in your protection, so keep it to the absolute minimum needed to get it to work.  Once it works, try removing exclusions to see if they are really needed until you get to the minimum needed to get the job done.
