Tim,

If a threat that Defender has detected has actually been quarantined, then it will be displayed above the 'See full history' title line in the 'Quarantined threats' section where you are currently seeing 'No threats' displayed.  At that point there will also be a 'Remove all' button visible as well as additional 'Restore' and 'Remove' buttons within the individual details listings for each specific malware item listed.

Since the new display is adaptive, it's not possible to see these options when no currently detected items are contained in quarantine.  So if the items detected have instead been automatically removed  or otherwise not quarantined for whatever reason, these items will not be listed nor available to remove or restore within that section.

So what I suspect you may be seeing is the history for an item that wasn't quarantined or might already have been removed from quarantine.

To test this operation and see what I've described above, simply try to download one or more of the Eicar test anti-malware files from the following page.

http://www.eicar.org/85-0-Download.html

These benign text files are named as executable or contained within compressed files to allow you to view how your anti-malware application would operate and display notifications when truly malicious software is detected, without any of the actual danger this might otherwise involve.  I just used them to test Windows 10 Defender myself to provide my descriptions here.

The reason for your issue is that the item detected is contained within an old Volume ShadowCopy backup file, which if it were ever recovered might return the infection to its original location within the Microsoft Edge browser cache where the path indicates it was originally located.

Windows 8 or 10

  1. From the search toolbar, type This PC.
  2. Right click on Local Disk C (C:) and select Properties.
  3. Accept any alerts from Windows UAC.
  4. Click the Disk Cleanup button.
  5. Wait for Disk cleanup to finish calculating.
  6. Select the More Options tab in the new window.
  7. Select the Clean up button for System Restore and Shadow Copies.
  8. Click Delete to confirm.

Since you're not apparently receiving a second detection for this malware within the Edge cache folder itself, I'm assuming it's already been removed from that location, possibly when the initial infection was detected.  If the second detection above is identical to the first, then the instructions above should remove both, but if not, deleting the Edge browser cache from within the browser settings menu should remove that copy as well.

Contacting Chat Support - Microsoft Community

jay_answers say:

Contacting Chat Support

Technical Level : Basic

Summary

There are some issues – account and billing related - that can only be fixed by contacting the official support channels. This article will explain the easiest way to contact chat support.


Details

Community support can fix most problems except for those that are related to the account itself. For instance, you are having trouble making a payment with the card, there is not much the community could do. It is possible that the card is not accepted. Or the payment is stuck in limbo and no amount of adding and removing will fix the problem. Such problems are dealt with by chat support.


Even if a particular problem does not come under the purview of chat support, they do act as a starting point. If they are unable to solve the problem, they usually will manage to set you off in the right direction.

Note - There is a dedicated support department for customers with disabilities. This department also handles queries related to accessibility features on the phone by users with or without disabilities. Find out more about chatting with them or calling them at this link.


Steps to contact chat support.


Step 1. Visit the site by clicking on this link. On the top right corner, you will find a ‘Sign In’. Sign in with same account you use on your phone. Note : The link will take you to the US site. Even if you are from another country, do not worry about it.




Step 2. Select the phone platform – windowsphone 7 or windowsphone 8. This is very important. If you are not sure about your platform, go to Settings > About on your phone. This will you give you the platform details.



Step 3. Under ‘Category’, select Billing + Payment (Store), from the drop down list.



Step 4. Then select ‘Contact Us’



Step 5. You should now be in the page that says ‘Chat’. Press the big blue button to start chatting with a support person. Note - It is possible that the button might show up in red colour. Every time the author of this wiki tried the chat support, the button is always blue.
