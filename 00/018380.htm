You can hide VBA code, but that doesn't make a workbook into an .exe. People will still need Excel to open the workbook.

To hide the VBA code:

  • Activate the Visual Basic Editor.
  • Select Tools | VBAProject Properties... (you may see another name instead of VBAProject).
  • Activate the Protection tab.
  • Tick the check box "Lock project for viewing".
  • Enter a password in the Password box and enter exactly the same password in the Conform password box, then click OK.
  • Make sure that you will be able to remember the password!
  • Save and close the workbook.

Next time you open the workbook, you'll need the password to view VBA code.
