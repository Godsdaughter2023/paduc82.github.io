=DLookUp("Price","Products","Product=" & [Forms]![CM Subform 1]![cboproduct])

I receive the #Name? Error.  Any suggestions?

hansv mvp say:

If Product is a text field, use

=DLookUp("Price","Products","Product=" & Chr(34) & [Forms]![CM Subform 1]![cboproduct] & Chr(34))

Chr(34) is the double quote character ".

I note that your form is named CM Subform 1. If it is really a subform on a main form, you'll have to change [Forms]![CM Subform 1]![cboproduct] to [Forms]![Name of main form]![CM Subform 1]![cboproduct], substituting the name of the main form. Subforms aren't part of the Forms collection.

hansv mvp say:

The value of a calculated control on a form is never saved to the record source.

Strictly speaking, there is no need to do so - the value can be calculated at any moment.

ken sheridan say:

The value of a calculated control on a form is never saved to the record source.

In this case I think it should be, Hans.  It sounds like a row is being inserted into an OrderDetails table or similar.  As the current price of a product can change over time, the prices in an order should not reflect those changes but remain static as the price at the time the order was created.  In relationalspeak the Price column in OrderDetails would be functionally determined by the table's key, so the table is normalized to Third Normal Form and no redundancy exists.

ken sheridan say: Don't use a computed control, but bind the text box to the price control in the OrderDetails (or whatever) table.  In the AfterUpdate event procedure of the cboproduct control assign a value to the text box with:

or you could use:

In the latter the literal quotes characters are each represented by a pair of contiguous quotes characters.

If you are unfamiliar with entering code into a form's or control's event procedures, this is how it's done in form design view:

1.  Select the form or control as appropriate and open its properties sheet if it's not already open.

2.  Select the relevant event property and select the 'build' button (the one on the right with 3 dots).
