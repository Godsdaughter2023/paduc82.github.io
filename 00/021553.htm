rich michaels - mvp (greatcirclelearning.com) say:
Hi Don,

It does work in Word 2011... I do it often between my machines. If there is a website saying that it doesn't work with Word 2011, I'd really like to get that link so I post to it and correct their errors.

When you change the Normal template, make sure Word is not just closed but that it is not running. That means there isn't a little gray dot above it in the dock. In fact, it might be best if you do a full restart and then not start anything until you replace the Normal.dotm template.

The entire process should be done via Finder. Place your Normal.dotm template in a folder on a flash drive, SkyDrive or DropBox. Now open Word and verify the results.

rajesh c. say:

YES - by using a formula like VLOOKUP.

But for that you need to first create a table containing 2 columns.

In the first column you need to list out all the sites and in the corresponding

cell in the second column for the site, key in the region corresponding to that specific site.

Say fir example you have created a List in the Range   A1:B100

Now if you are entering a site in cell D5 and you wish to get the corresponding region of this site in cell E5, then in cell E5 you need to key in the formula

=VLOOKUP(D5,$A$1:$B$100,2,0)
