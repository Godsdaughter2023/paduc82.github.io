ken sheridan say: When assigning an SQL statement to a variable the value of any parameter should be concatenated into the string expression, not referenced.  Note also that the query incorrectly uses a HAVING clause, when a WHERE clause should be used.  A HAVING clause is applied after grouping and is used to restrict the result set on the basis of an aggregated value.  Moreover, a GROUP BY clause is unnecessary here as no values are being aggregated in this query.  Finally the DISTINCTROW predicate is almost certainly unnecessary.  So, taking these points into account

mySql = "SELECT [original amount]-Nz([Sum Of voucher value])+Nz([Sum Of value]) AS Expr1 FROM ([PO list] LEFT JOIN qryTransfersTotals ON [PO list].PO = qryTransfersTotals.PO) LEFT JOIN qryVouchersTotals ON [PO list].PO = qryVouchersTotals.PO WHERE [PO list].PO=" & [Forms]![Transfers]![CmboCompany]

This assumes that the PO column is a number data type.  If it's a text data type the value must be wrapped in literal quotes characters:

mySql = "SELECT [original amount]-Nz([Sum Of voucher value])+Nz([Sum Of value]) AS Expr1 FROM ([PO list] LEFT JOIN qryTransfersTotals ON [PO list].PO = qryTransfersTotals.PO) LEFT JOIN qryVouchersTotals ON [PO list].PO = qryVouchersTotals.PO WHERE [PO list].PO=""" & [Forms]![Transfers]![CmboCompany] & """"
