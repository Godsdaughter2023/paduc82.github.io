You will be able to set default applications by file type and by protocol on Windows 10. To set a default program that is not present, you will have to download and install the application on the PC.

 

If you have already downloaded and installed Foxit phantom PDF application on the PC, follow the steps below to set as default and check if the issue is resolved.

As you can see, I had the same issue on my PC and I finally solved it!!!

The problem (in my case) was not the SSD, but rather the fact that new laptops require USB EFI boot.

EFI boot USB needs to be on FS=FAT32.

Therefore, I did not need to make any changes on my SSD (format quick fs=ntfs is fine) but I did run the following commands on my USB prior to loading the ISO and it fixed the problem:

DISKPART

SELECT DISK (usb)

CLEAN

CREATE PARTITION PRIMARY

SELECT PARTITION 1

ACTIVE

FORMAT FS=FAT32 QUICK

ASSIGN

EXIT

Also, just to be sure, I used the Mount function of Windows 8 to copy the ISO files to the USB to assure that whatever program mounted the ISO did not reformat the USB.
