    1. Open a Finder window, and press Command+Shift+G.
    2. Type the following path:

    3. Press Return. Upon launching Word, this folder will be created if it doesn't exist, and Word will create a temp file inside it. Quitting Word will then cause this temp file to removed, which tells us that the new setting has taken effect.

If you have created a default Blank Document in Word 2010, it is unlikely that you are actually getting double spacing. The default line spacing in Word 2010 is 1.15, with 10 points Spacing After. Unless you are pressing Enter at the end of each line, you should not be seeing anything like double spacing, though the spacing is more than single.

You can achieve single spacing by pressing Ctrl+1, and you can remove the Spacing After in a number of ways. You may want to familiarize yourself with the paragraph and line spacing controls in Word:

  • To get complete control over all aspects of paragraph formatting, use the Paragraph dialog. You can open this by clicking on the dialog launcher (small arrow) in the bottom right corner of the Paragraph group on the Home tab, or by right-clicking in a paragraph and choosing Paragraph... from the shortcut menu.
  • You'll see several options for both line and paragraph spacing if you click the Line and Paragraph Spacing button in the Paragraph group. If you choose Line Spacing Options... on that menu, you'll get to the Paragraph dialog.
  • Click Change Styles in the Styles group on the Home tab and select Paragraph Spacing to get some preset spacing options. After finding the one you want, you can click Change Styles again and then click Set As Default.
  • On the Change Styles | Paragraph Spacing menu, you can also click Custom Paragraph Spacing to open the Manage Styles dialog and set the default line and paragraph spacing for the +Body style, which affects the formatting of the Normal style.

Since it is conventional to allow space between single-spaced paragraphs in letters, you may want to leave the 10 points Spacing After (or even increase it to 12 points) even though you change the line spacing to Single.
