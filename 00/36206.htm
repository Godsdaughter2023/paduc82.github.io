lemp say:

Let's move cautiously here.

Do you have a complete backup of all of your data on some external media?  If not, now would be a good time to make one (and then begin a program of making regular backups).

Windows 7 introduce the concept of "Library" for file organization.  A Library is not the same as a folder.  A folder is simply a container for storing files—nothing more and nothing less. A Library doesn’t contain files. Rather, a Library provides a single aggregated view of multiple folders and their contents.

Windows 7 has four default Libraries: Documents, Music, Pictures, and Videos. Each default Library contains two folder locations: one in the user’s profile folder and the other in the Public profile folder (My Music and Public Music, for example).

A bit confusingly, "My Music" (or "My Documents" or "My Pictures") is not actually the name of the actual folder on your hard drive.  Windows 7 uses the "My" prefix in the GUI file manager (Windows Explorer) to make things more familiar to people who had been users of XP.  The actual name of the default folder for storing music files is C:\Users\<username>\Music

Normally, when you display the top level of one of Windows 7's "Libraries," you'll see a link under the name of the library that indicates how many "locations" are included in the library.

Each of those "locations" is then shown with the number of items (folders and files that are not in a folder) shown in parentheses and path to the "location" shown just below the name.

Let's assume that your username is PMC and that you have 100 files and folders in "My Music"

You should see something like this:

Music Library

Includes:    3 locations

My Music (100)

C:\Users\PMC

folder

folder

folder

file

file

....

Public Music (0)

C:\Users\Public

Does the next item -- My Music (C:) --

  • have the same number in parentheses as the first item (e.g., 100, using the example here)?
  • have the same path as the first item (e.g., C:\Users\PMC)?

If the third item is, in fact, the same physical folder as the first, then all you need do is remove it from the "Library" display.  To do this, click on the link that says "Includes:  3 locations." In the "Change how this library gathers its contents" window, select the item you no longer want to see and click the "Remove" button (or right-click on it for more options).
