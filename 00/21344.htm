I don't think you want to install pre-release software over a working and stable operating system. If you really do want to do this, be sure to keep a back up of all your important files on another PC or external hard drive.


If you really want to do this, follow the following steps, but remember that all your data, installed programs, etc. will be lost once you arrive at Windows 10 Tech Preview.


  1. Find a USB drive of at least 4GB (8GB or more preferably) that does not have important files on it.
  2. Format the USB to FAT32 with default block size and give it a name like "Win10Build9841", you can choose yourself. Keep in mind that all the files on the USB drive will be deleted.
  3. Double click the .iso file you downloaded, it will now mount as a 'fake' DVD in This PC. Open it and select all files.
  4. Copy these files to the USB drive you just formatted and wait for it to copy all the files.
  5. Now shut down your windows (do not hibernate or sleep, shut down).
  6. Boot from the USB drive. For some computers, just booting while having this USB inserted will boot from USB, otherwise you'll have to select that in the boot menu (one of the F-keys while starting the computer).
  7. Once you see the install program, follow the instructions. Be sure to select "Custom Install", remove the old Windows 8.1 partitions and create a new one for Windows 10 Tech Preview.
  8. Now complete the installation, this will take about 10 minutes to an hour, depending on the speed of your USB drive and computer.

I am not responsible for making your computer unusable. You should know the risks that come with installing a new OS without much technical knowledge:

  • Your computer may become unable to boot properly if you do something wrong.
  • You may lose more data than you were anticipating (e.g. by deleting a wrong partition)
  • You may not like the new OS and going back to your old OS means installing the old one again.
  • Getting help from a professional may cost you money.

If you really want to test Windows 10 Tech Preview, but you don't want to give up your current operating system, I'd suggest you download VMware Player (free software):

https://www.vmware.com/go/downloadplayer

You can create a Virtual Machine (a simulated computer running e.g. Windows 10 Tech Preview).

The steps to do this are:

  1. Download and install VMware Player (the free software) from https://www.vmware.com/go/downloadplayer
  2. Open VMware Player and create a new virtual machine.
  3. Select the Windows 10 iso and select 'Windows 8' or 'Windows 8 x64' depending on whether you downloaded the Windows 10 Tech Preview 32-bit or 64-bit version.
  4. Follow the instructions on the screen.

Once you're done, the machine will boot and the Windows 10 Tech Preview install will start. It will not overwrite anything on your computer. You can quickly stop and resume a virtual machine by 'suspending' it. This is sort-of a hibernate, but the operating system will not notice, unless you change any hardware of your own PC.

To change the amount of resources (RAM, CPU, etc.) the virtual machine can use, shut it down completely (do this after the installation is complete!) and select 'Change virtual machine settings' or something similar.

Have fun testing!

Tangent curve - Microsoft Community

vijayakumarchary say:

Tangent curve

 Hi !   :-)    Charts of Sine and Cosine trigonometric functions look good enough because however many data points you add, these curves don't get better. But the chart made with Tan(x) data is a set of spikes that don't mean anything at all. Can this Tangent curve be improved upon in an excel chart ?  Is accuracy the limitation of excel charts that makes Tan(x) other than representative of the trig function ?
hansv mvp say:

It's in the nature of the tangent function.

When x goes from 0 to pi/2, tan(x) increases without limit to positive infinity; at pi/2, the function has a singularity, tan(pi/2) does not exist.

When x goes from 0 to -pi/2, tan(x) decreases without limit to negative infinity; at -pi/2 there is a singularity again.

This pattern repeats itself with intervals of pi.

Excel cannot represent pi exactly, and it cannot represent "infinity" of course, so it's difficult to create a satisfying graph of the tangent function.

It's best to use fractions of PI() in the x-values of the chart and to clear the very large values that Excel computes for the tangent of -pi/2, pi/2 etc.

See this sample workbook on SkyDrive (open it in Excel or download it to your computer first): https://skydrive.live.com/view.aspx?cid=0CAE9FF0A56DD9D4&resid=CAE9FF0A56DD9D4%21147

hansv mvp say:

If you look at column A of the sheet that I uploaded, you'll see that the X values compute multiples of 10, that is multiples of 10 degrees, and converts those to radians using the RADIANS function.

180 degrees correspond to pi radians, where pi=3.141592..., so all the RADIANS function does is multiply the angle in degrees with pi/180.

This ensures that 0 degrees, 90 degrees, 180 degrees etc. will be in the set of X values. Because the tangent function repeats itself every 180 degrees, this will make the chart look (reasonably) good.

You can experiment with the X values, of course, to see how the chart changes.

 

Oh, and the chart type is an XY (scatter) chart where the data points are connected by smooth lines (curves).

The built-in templates are here: /Applications/Microsoft Word.app/Contents/Resources

You will need to right-click and "Show package contents" to see inside.

They are in a folder whose name ends in .lproj and whose first two characters are the abbreviation for the language.  So en.lproj for English.

Before you drag any files anywhere, make a copy of that folder and keep it outside the Applications folder.  Word has a mechanism that enables it to check itself for tampering before launching.  If you make a change it doesn't like, Word will crash a few seconds after launch each time.

Drag one or two .dotx files out of that folder and test by re-launching Word.  If it does not crash when you do a File>New> From Template, you can do the rest.

Obviously: a word to the wise -- make sure you touch ONLY .dotx files :-)  The other files in that folder are quite important.  They do things like "enable Word to run"  :-)

Hope this helps

jessee_968 say:

Hello,

I have RA2 and Yuri Revenge installed on my PC at the moment while running Windows 10. There are a few things you'll have to do. After installing it, go to the folder you installed it to (default is C:\Westwood\RA2 OR C:\Program Files (x86)\Origin Games\Command and Conquer Red Alert II if you're running it through Origin) find the following .exe's

Game.exe and gameMD.exe right click them, select properties, and under the compatibility tab select run this program in compatibility mode for Windows XP SP2, also check off the box that says "run as administrator".

That alone will allow the game to run, but it will be pretty laggy in-game. To help with the lag find the .ini files in the same folder listed above, they are "ra2.ini and ra2md.ini" open them with notepad and under section titled [Video] add the following line"

VideoBackBuffer=no 

optionally you can also change your resolution to a higher one, as the game forces you to play on a lower resolution because its so old, but you can play on 1920x1080 if you monitor supports it. below is an example of my ra2.ini video section:

[Video]
ScreenWidth=1920
ScreenHeight=1080
StretchMovies=no
VideoBackBuffer=no

One thing to note is, you cannot minimize once the game is loaded or it will crash. I'm not sure how to fix that yet.

Lastly, if you want even less lag in-game there is a third party patch to speed up in-game graphics, you can find it on the RA2 Online ladders website (xwis)

http://xwis.net/forums/index.php/topic/173459-graphics-patch-to-speed-up-tiberian-sun-red-alert-2/

The steps to apply that patch are located in that forum post, as well a a download to the patch itself.

Also as a last thing, in windows 10 the MS Xbox DVR app forces VSync to be turned on. I have a 144hz monitor and it was forcing my refresh rate to 60 which made the game feel sluggish.

If you don't use the Xbox app, you can remove it using powershell by opening powershell as an administrator and pasting in this line:

Get-Appxpackage *xbox* | Remove-AppxPackage

This will allow the game to run in whatever your monitors refresh rate is set to. 

Have fun.
