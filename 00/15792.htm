In Microsoft Word 2010 my autotext gallery will not remember my - Microsoft Community

beverlysue say:

In Microsoft Word 2010 my autotext gallery will not remember my entries from day to day

original title: Autotext gallery

In Microsoft Word 2010 my autotext gallery will not remember my entries from day to day.  Is there a solution for this problem yet?

suzanne s. barnhill say:

Since AutoText entries are by default stored in the Normal template, Normal.dotm, this suggests that it is not being saved when you quit Word. Sometimes add-ins interfere with the process. There are several lines of attack:

  1. The best approach is to determine which add-in is preventing Normal.dotm from being saved. If it is an add-in you can do without, you can remove it permanently; if not, you can work around it. You can see a list of add-ins at File | Options | Add-ins. As you select each one in the list, its path will be displayed below. You can move them out of those folders temporarily, one at a time, till you find which one is the culprit.
  2. As a stopgap, you can open Normal.dotm itself, add the AutoText entries, and then save it. Find Normal.dotm in the Templates folder, which you should be able to access from the Places Bar or navigation pane (depending on OS) of Word's Open dialog. You can also find the path to it at File | Options | Advanced: File locations.
  3. If you find that the problem is caused by an add-in you can't do without, there is a shortcut for saving Normal.dotm. In the Customize the Quick Access Toolbar dialog, select All Commands and find Save All. Add this to the QAT. When you click this button, Word will attempt to save all open documents (you can decline to save any specific one) AND Normal.dotm. By forcing a save of Normal.dotm before shutdown, you can save any newly added AutoText entries.
