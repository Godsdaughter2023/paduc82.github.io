shyam pillai say:

I recall a scenario in the past where this would happen if update wasn't called after the data was updated using VBA. Post your code here or double-check on that.

induszing say:                        

Sub UpdateChart(ObjName, Slide, rng, xVal, ws)

Dim Ystr As String
Dim Xstr As String
Dim rng1 As Range
Dim rng2 As Range
Dim ds As Worksheet

    Slide.Shapes(ObjName).Chart.ChartData.Workbook.Sheets(1).Cells.Clear
    ws.Range(rng).Copy
    Slide.Shapes(ObjName).Chart.ChartData.Workbook.Sheets(1).Range("A1").PasteSpecial xlValues
    Slide.Shapes(ObjName).Chart.ChartData.Workbook.Sheets(1).Range("A1").Value = ""

    Set ds = Slide.Shapes(ObjName).Chart.ChartData.Workbook.Sheets(1)
    Set rng1 = ds.Range(ds.Range("B1"), ds.Range("B2").Offset(ws.Range(rng).Rows.Count - 2, ws.Range(rng).Columns.Count - 2))
    Set rng2 = ds.Range(ds.Range("A2"), ds.Range("A2").Offset(ws.Range(rng).Rows.Count - 2, 0))

    Ystr = "='Sheet1'!" & rng1.Address(ReferenceStyle:=xlR1C1)
    Xstr = "='Sheet1'!" & rng2.Address(ReferenceStyle:=xlR1C1)

    Slide.Shapes(ObjName).Chart.SetSourceData Source:=Ystr
    Slide.Shapes(ObjName).Chart.Axes(xlCategory).CategoryNames = Xstr

    For i = 1 To Slide.Shapes(ObjName).Chart.SeriesCollection.Count
        Slide.Shapes(ObjName).Chart.SeriesCollection(i).Name = "='Sheet1'!R1C" & CStr(i + 1)
    Next

    Slide.Shapes(ObjName).Chart.Refresh

End Sub

alexsun_2012 say:

Hi CapDis,

There is no official tool to migrate emails from Hotmail to Office 365 directly. However, a workaround we suggest is you use the Office 365 Connected Accounts feature.

Firstly, you need to add your domain and create the users accounts and mailboxes in Office 365. Then your users can set up the connected accounts in Outlook Web App (OWA). After you provide your Hotmail credentials, Office 365 tries to establish the connection to the Hotmail servers. Once the connection is sucessful, the emails will be downloaded to your Office 365 mailbox from Hotmail every one hour.

Witn connected account, only the emails can be downloaded. To migrate the contacts and calendars, you need to use Outlook .PST Export and Import features as described below.

https://support.office.com/en-us/article/Export-or-backup-email-contacts-and-calendar-to-an-Outlook-pst-file-14252b52-3075-4e9b-be4e-ff9ef1068f91
https://support.office.com/en-us/article/Import-email-contacts-and-calendar-from-an-Outlook-pst-file-431a8e9a-f99f-4d5f-ae48-ded54b3440ac

Thanks for your understanding.

If you're signed into Skype, you can receive calls. You'll see an incoming call notification screen where you can:

Audio call button Select the Call button to answer the call as an audio call.
Video call button Select the Video button to answer with video. For incoming video calls, you can select to answer it as a video call or audio only.
End call button Select the End call button to decline it if you don't want to answer. If you're offline and you have Voice Messaging activated, the caller will be able to leave a message, or you can set up call forwarding to another number.
merge calls button Select the Merge calls button to merge your current and incoming calls.
