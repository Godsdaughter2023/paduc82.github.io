steve rindsberg say:

SharePoint is expensive, complicated, the feature set keeps changing, and it's expensive.  Not really a toy intended for the one-computer user.

You might instead want to experiment with this little trick:

Put your presentation in normal view, add a new blank slide.

Open the presentation you want to insert slides from, put it in Slide Sorter view.

Select the slide you want and rightclick : Copy

Go back to PowerPoint and Paste Special, LINK and choose PowerPoint SLide object if there's any choice.

Enlarge the pasted slide object to fill the blank slide.

You now have a linked slide object filling your slide; it will update when you open the file that holds it.
