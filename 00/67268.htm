PassCODE vs password. - Microsoft Community

judi 74 say:

PassCODE vs password.

I have windows 10. I want to open my computer with a pinn rather than the password. Like I did with Windows 8.1.

***Post moved by the moderator to the appropriate forum category***

a. user say:

Hello Judi,

PIN is a form of security measurement to make a device or a program more secure. It is usually tied to a device and cannot be used to access an account on another device unlike the password. If a password is compromised in some way, the person who has entered the system may have access to all platforms that are linked to that password.

To set up a PIN, simply follow these steps:

  1. Open the Settings app and click on the Accounts icon.
  2. Click on Sign-in options on the left side and click on the Add button under PIN on the right side.
  3. If prompted to verify your account password, enter your local account password and click on OK. If you have a Microsoft account, then enter your Microsoft account password and click on Sign in.
  4. After entering your password to confirm your identity, enter the numbers in a dialog box. The minimum length is four digits (0–9 only; no letters or special characters allowed), but your PIN can be as long as you want. If you need to check what you’ve set at this point then select the icon to the right side of the dialog box. This will briefly reveal the number you’ve entered.
  5. The only criteria for choosing a PIN is that it must be at least four digits long. There are no limitations on maximum length or complexity. Here are some considerations before you choose a PIN:
  • Using more digits will make the PIN harder to guess but make sure you can enter the PIN quickly and accurately, otherwise there’s little benefit over just using a password.
  • Using a simple PIN (0000, 0123, 1111 and so on) will make it very easy to guess; choose random numbers.
  • Reusing PINs from your bank accounts or credit card number should be avoided. Also, avoid using the same PIN on different devices.

Feel free to post back if you have further questions or future concerns.

To Create a System Image in Windows 8 follow these steps:

 

Step 1: Bring the Search window up using the keyboard combination Windows logo button + F. Type in “file recovery" in the search space given and hit the Settings tab. From the given results, click on the option saying windows 7 file recovery.

Step 2: A new window should show up. Click Create a system image in the left panel as indicated in the image.

Step 3: You will be lead to ‘Create a System Image’ window that should search for available backup devices on your Windows 8 system.

For first attempt of making system image, you will be required to specify the location for this system image to be stored in Windows 8.

After selecting a location for your system’s image, click Next.

Step 4: Another window confirming the drives whose files will be backed up and created an image for will show up. Click Start backup.

Step 5: System image making process will be initiated. Wait for a few minutes for its completion.

Step 6: When the process completes and system image is created, a new window will pop up with an option to create system repair disk. If you wish to create it, click Yes. Make sure you have an empty DVD available to create a system repair disk.

Step 7: You can close the window when the system image process completes.

 

To restore the image, follow these steps:

 

a.    Press the Windows + C key on the keyboard.

b.    Click on settings, click Change PC settings and then click General.

c.    Scroll down to advanced startup.

d.    Select System image recovery, follow onscreen instructions.

Not taking ownership, no taking permissions, you even can rename it but not delete. This is an issue that is present since first windows 10 version, as there are other not solved issues since ever.

Update History can be very misleading as it lists all updates which may have been installed (which is why you'll see Successful and Failed updates in the list).

On the other hand, Installed Updates only lists updates which have successfully installed. In other words, if an update's listed as being successfully installed in Update History but it's not listed in Installed Updates, the update has NOT been installed.

  • Of course, there are exceptions to every rule. Some successfully-installed updates will not be listed in Installed Updates (e.g., Flash Player updates; driver updates).  [Yes, it's very confusing!]
