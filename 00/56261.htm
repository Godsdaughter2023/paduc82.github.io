richardeiler say:

Hi Tho,

First, to clarify, BIOS is not a driver per se, it is a motherboard firmware that must be flashed to a dedicated chip on the motherboard that controls the basic input/output controls of the motherboard components, as a whole, before and during the loading of the operating system (regardless of the OS being used, which in this case is Windows). A BIOS update does not/can not be automatically done by an update servicing utility in most, if not all cases, it can only notify you that it is available (and recommended) to be done (in this case the Dell Support Assistant) and it must then be downloaded and performed manually by the user or an agent thereof.

Follow the link provided in the reply post from Sumit (above) then double check that it is indeed applicable to your device and then make sure that you fully understand (before you proceed) and follow all of the instructions to the letter during the process with patience as required (there are typically no second chances if the procedure is performed incorrectly which would/could result in an unusable (bricked) device).

That being said, I've updated the BIOS on many devices/many times over the years without issue, just be careful, understand the given procedure first, and then be precise in the execution of the procedure.

For what it's worth and I hope this helps, let us know how it goes.
