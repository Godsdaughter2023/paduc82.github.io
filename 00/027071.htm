(applicable to all versions of Outlook) 

The following keeps all the resulting DLs (Contact Groups) in one folder

#1 - Highlight the first DL you want to copy

there are 2 ways to make a copy of the same DL

Option#1 - Right-click the DL --> hold Ctrl key down --> drag to the same contact folder where the DL currently exists 

or

Option#2 - Right-click the DL --> Select <Move> --> select <Copy to folder> --> select the same contact folder where the DL currently exists

#2 - Regardless of approach used, the above will result in a duplicate DL name in the contact folder so rename each to reflect something relevant to you for each DL (open the DL and change the name --> Save)

#3 - To bulk delete DL members from a given list --> highlight the members you want to delete, again 2 ways to do that

Option #1 - highlight the first contact in the sequence --> hold down the shift key --> highlight the last member in the sequence --> hit the <delete> key

or

Option#2 - hold down the CTRL key --> highlight the members you want to delete individually --> hit the <delete> key

jay freedman say:

From the symptoms, I'd suspect a keyboard problem.

  • In Word that's operating normally, the F8 key runs the ExtendSelection command. When the key is pressed repeatedly, the first press turns on the extended selection mode; the second press selects the word at the cursor; the third selects the sentence, the fourth selects the paragraph, and the fifth selects the whole document.
  • If extended mode is turned on at some location by a single press, and then you click the mouse elsewhere, everything between the two locations is selected.
  • You can turn off extended selection mode by pressing the Esc key.
  • To see when extended selection mode is turned on, right-click the status bar at the bottom of the Word window and check the Selection Mode item.

If your keyboard has a sticky F8 key, or an electrical problem that the computer interprets as F8 presses, then it could appear as you described.

The simplest way to prove or disprove the theory is to swap keyboards with another computer.
