andreas killer say:
The chart engine was completely rewritten in office 2007, the ShowWindow property of the chart object is now hidden and only available for backward compatibility.

If you set this property in 2003, a window is opened that contains the chart, but in 2007 happens nothing and the ActiveWindow is now the Workbook.

This code is not compatible with Excel 2007 and above and must be rewritten or removed. To clarify: It's not possible anymore to show a chart in a window.
