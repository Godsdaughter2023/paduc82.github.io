Step 1:

By Task Manager, please follow the steps given below.

  2. Click “More details” at the bottom, go to Processes tab.

  3. Locate the explorer.exe and right-click on it.

  4. Select End Process from the menu to kill it.

  5. Repeat the same way for all running explorer.exe processes.

  6. Click File, select Run (or new tasks), input explorer and click OK button.

  7. From your desktop, right click on the time clock, select Properties and set Volume icon to be on.

Step 2:

Modifying registry, please follow the steps given below.

  • Press Windows + R, type regedit and press Enter to run Registry Editor.
  • Within its left pane, navigate to the keys below:

HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\CurrentVersion\TrayNotify

  • In the Details pane, delete both of the IconStreams and PastIconsStream registry entries.
  • Restart your PC to apply all changes.

How to back up and restore the registry in Windows

http://support.microsoft.com/kb/322756

Step 3:

If driver files are corrupted, please follow the steps given below.

  1. Press Win + X key and click Device Manager.

vijay a. verma say:
Type ncpa.cpl in Windows Start Search box (Cortana) > Right click on the network icon for which you want to change IP Address > Properties > Select Internet Protocol Version 4 (TCP/IPv4) > Properties > Select Use the following IP address > Specify your new IP Address, Subnet Mask & default gateway

If you don't know the value of Subnet mask and default gateway, you can get their latest values
Right click on network icon as in previous step > Status > Details > You have current values here

I'm an independent advisor. You can open powershell and do this command:

ipconfig /renew

Scroll to the bottom

Right click
