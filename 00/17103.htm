OK I may understand what you are seeing now.  When you click on the "Show my location" icon (or use the CTRL-Home key combo) the map will move to your detected geo-location.  This icon on the map is an approximation for a PC, since a PC does not have an internal GPS like a Windows phone would have.  This icon is not movable unless your PC moves, so if a laptop connects to a different Wi-Fi router at the coffee shop, Windows Maps would then use that new location, based on whatever it can determine of the Wi-Fi router's location.  My location shows me about .3 miles North-West of where my house actually is.

If you were to share out your phone as a Wi-Fi hotspot and connect from your computer's wireless network adapter through your phone to the internet, maybe it would become much more accurate, due to the phone's actual GPS capability.

So you are right.  There is no way to change or clear "My location", because that is as close as Windows is able to detect to your actual location.

  • There is a Default Location, for use when your actual location cannot be detected.
  • There is also a "Home" Favorite in Maps, which Cortana also uses to establish your Home for offering tips and guidance for commuting, etc.
  • There is a "Work" Favorite which also can be set and used by Cortana.
  • When you have no "Home" Favorite saved, choosing a location may ask if this should be set as your "Home" for Cortana.

After clearing all my Map Favorites (I actually live in Canada) when I right-click 1800 North Beverly Dr., in Beverly Hills, a menu appears with Drop a pin.  I then get this:

If I then choose Save, I can then choose Edit which gives me the choice to save this as my Home or Work location for Cortana's use.

If you select the Star icon in Maps, it will show you if there are already any Favorites set, such as Home or Work.  You can then select these to delete them and then find the correct location for these to save as Home and Work locations in this manner.  There is another one for Car, which could come in handy on a Windows phone when you forget just where you parked in a large lot.

Hopefully this is getting closer to what you are after?
