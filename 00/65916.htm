a. user say:

Hello!

If you use Windows XP Professional to block traffic to ports 80, 443 you can use IPSec.

If you have a domain, you can put the computer in a separate OU and apply to the OU a policy that contains the IPSec settings.

To use IPSec locally to block ports 80, 443:

1. Start->Run->mmc.exe

2.  Go to Console Menu , Add/Remove Snap-in ( or press CTRL+M ), double-click IP Security Policy Management, Local Computer, Finish, Ok

3. Right-click IP Security Policies on Local Computer and from the context menu choose "Create IP Security Policy", Next, set a policy name, Next, leave the box with "Activate the default response" unchecked, Next, tick the box with "Edit Properties", Finish.

4. Under Rules tab, click Add. In the new displayed window called "New Rule Properties", under "IP Filter List" tab click Add, choose a name, untick "Use Add Wizard", click Add, under "Addresses" tab Source: My Computer, Destination: any computer, under Protocol tab choose TCP from any port, to this port: 80. Also Add TCP and 443.

5. In "New Rule Properties" window that is still open, under "Filter action" tab uncheck "Use Wizard", click add, select "Block" then Ok.

6. "In New Rule Properties" under "IP Filter List" make sure that the list created at step 4 is selected and under "Filter action" the filter created at step 5 is selected. Close all windows except the MMC window.

7.  Right-click the new created rule and choose "Assign"

To disable the policy, right-click it and choose "Un-assign".
