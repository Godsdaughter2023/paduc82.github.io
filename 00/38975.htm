First of all, with a SMART alert, I'd backup my files and programs to an external hard drive ASAP! SMART alerts are sometimes 'false positives', but not very often, so I'd take it as a 'Thank you for saving my butt' advisory.

 

As to the process to follow, I'd advise this:

  1. Backup the hard drive to an external drive immediately.
  2. Replacing the hard drive will cost roughly 1/3 that of replacing the notebook, so use that as a guide for deciding whether to replace the hard drive or the laptop.
  3. Regardless of which option you choose, take the System Disk you (hopefully) created from my last post and try to do a Repair Install using it.*
  4. If the Repair Install was successful, thoroughly test the computer, focusing on those tasks which had triggered past problems.
  5. If the Repair Install wasn't successful, e.g., it didn't complete / generated an error message, proceed to doing a full System Install. Once the full System Install is done thoroughly test the computer, focusing on those tasks which had triggered past problems.

To do a Repair Install, follow the directions in my last post. To do a full System Install, follow the same instructions, except, insert the System Disk and then restart your computer. If your CD / DVD isn't the first boot device, make it so by pressing the F12 key on boot-up and selecting your CD / DVD as the 1st boot device (this is a one time setting).

 

* The reason for doing any Install procedure (Repair or full) is to leave you with a working computer until such time as your laptop is either repaired (hard drive) or replaced.
