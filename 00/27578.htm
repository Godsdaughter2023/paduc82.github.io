shane devenshire say:

Excel does not contain frames and you can't us shapes.  However, suppose you sized the range B2:F5 to just the size you want the picture.  Then copy and paste the picture into Excel, anywhere.

1.  Hold down the Alt key and drag the top left corner of the picture to the top left corner of B2.

2.  Hold down the Alt key and drag the bottom right corner of the picture to the bottom right corner of cell F5.

3.  The picture will size with the cells if you right click it and choose Format Picture, Properties...

Of course you can put the picture in a comment:

1.  Don't copy the picture into Excel choose Insert, Comment

2.  With the comment open double-click its "border"

3.  Choose Colors and Lines, open the Fill Color drop down and choose Fill Effects

4.  Choose Picture tab and Select Picture.  Hunt down your picture.

Comments are attached to cells. They can be left displayed, they can be sized as you choose.

If this answer solves your problem, please check Mark as Answered. If this answer helps, please click the Vote as Helpful button. Cheers, Shane Devenshire

karl timmermans [outlook mvp 2012-2018] say:
Do you mean, "can I copy the contents in a PST file to the folders in an Exchange account?"

For the purposes of this response, going to assume the answer to the above is yes and the answer is also yes. For sake of accuracy, Exchange ActiveSync no longer applies to Outlook desktop, it would be configured as an Exchange account.

Two steps to copying data from the PST file to the EX account folder tree

Step 1 - for standard folders (i.e. Inbox, Contacts etc)

1 - Highlight the source folder in the PST file
2 - Press CTRL+A to hightlight all the items in the folder
3 - right-click on the highlighted items --> Select <Move>
4 - you will then have 2 choices - either <Other Folder> or <Copy to Folder>. I'd suggest using the <Copy> option which will not delete anything from the source folder in the off-chance something goes awry during the process. <Other Folder> .is a <Move> operation which will delete items in the source folder.
5 - rinse & repeat for all other standard folders that you want to copy

Step 2 - for custom created folders

6 - drag 'n drop the folder from the PST folder tree to the EX account folder tree

*** I'd also recommend setting Outlook to work in offline mode while your copying things from the PST file to the EX account so that sync'ing doesn't start while the copy process is still going on. When everything has been copied, return Outlook to online mode and let the EX account sync with the server
