Deleting internet browsing history - Microsoft Community

hchrissyg say:

Deleting internet browsing history

Is there a way to set a users controls to disallow them to delete internet browsing history?
halima s say:

Hi hchrissyg,

 

 

You can use Group policy editor to disable “Delete browsing history” in internet explorer.

Group Policy Editor is available only on Windows 7 Professional, Ultimate and Enterprise Editions.

 

Follow below steps to disable “Delete browsing history” in internet explorer using group policy editor.

 

a)   Go to Start type  gpedit.msc (Group policy) in the search box.

b)   Select  Computer Configuration –> Administrative Templates

c)    Select  Administrative Templates–>Internet Explorer

d)   Go to Delete Browsing History select  Turn Off “Delete Browsing History” Functionality in right side.

e)   Now Enable Turn Off “Delete Browsing History” Functionality.

 

 

If you do not have any versions that supports group policy editor then you can use the below method by editing the registry. You will need to perform this task at your own risk as making changes to registry is not recommended.

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base:

How to back up and restore the registry in Windows:

http://windows.microsoft.com/en-US/windows7/Back-up-the-registry

a)   Start - type in Search box -> Regedit

b)   Navigate to the following location: HKEY_CURRENT_USER\Software\Policies\Microsoft\Internet Explorer\Control Panel

c)    In Right Pane - Set a DWORD value named DisableDeleteBrowsingHistory to 1

d)   Close registry editor.

 

Hope this helps!

Halima S   - Microsoft Support.
