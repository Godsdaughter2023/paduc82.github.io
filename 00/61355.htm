Step 11: Clean the ink cartridges

Clean the ink cartridges using the HP software, and then evaluate the results.
To avoid issues that require cleaning, always turn the printer off with the Power button.
  1. Load plain white paper in the input tray.
  2. Search Windows for your printer model, and then click on your printer in the list of results.

Minimum System Requirements for Freedom Fighter game:

If they are grouped in folders, the easiest way I know of is old school.

open a CMD window
In CMD window typing HELP will display list of all available commands
use /? after command to get help on it, ie CD /?
