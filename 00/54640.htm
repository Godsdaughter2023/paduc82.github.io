How do I get a list of jpeg file titles on the computer directory folder to be a text list on Excel or Word

I want to create a library in Excel of all the images we have for given part numbers.

All the images are titled with the part number, for example- 43682 001.

I want to include this list on Excel but I'm not able to get this list of image titles in a text format to add.

Any ideas?

cmdrkeene say:

Here's one way:

  1. Open a Command Window in the Folder
    Hold Shift while you right-click the folder were all the pictures are.  Holding shift provides more options on the right-click menu.  Choose "Open Command Window Here".




  2. Copy the List of File Names With a Command
    At the command window, type this command and press enter:

    dir /b /s |clip

    When you press enter, nothing will appear to happen yet.  In the background, a list of all file names was copied to the clipboard.



  3. Paste the List Into Excel
    Switch to Excel and press Paste (or Ctrl-V).  A list of the file names will appear.



  4. Remove the File Path Info (optional)
    If you want just the names, and to remove the folder path, you can press Ctrl-H and do a replacement:

    Replace:  *\
    Replace With:  (nothing, leave this blank)

    After replacing, you'll have just the file names:

a. user say:

Hi Pat,

The error ‘Media library is corrupt’ occurs when the Windows Media Player library database has become corrupted.

To resolve this problem, clear the Windows Media Player database. To do this, follow these steps:

  • Press Windows key + R, click Run, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.

  • Select all the files in the folder, and then click Delete on the File menu.

Note: You do not have to delete the folders that are in this folder.

Restart Windows Media Player.

If this does not resolve the problem, clear the Windows Media Player database cache files. To do this, follow these steps:

  • Press Windows key + R, click Run, type %LOCALAPPDATA%\Microsoft, and then click OK.

  • Select the Media Player folder, and then click Delete on the File menu.

  • Restart Windows Media Player.

Note: Windows Media Player automatically rebuilds the database.

Hope it helps. Do let us know, if you have any further queries about Windows 10.  We will be happy to assist you.

Thank you.

Macrium Reflect (both free and paid versions) is compatible with Windows 10.
