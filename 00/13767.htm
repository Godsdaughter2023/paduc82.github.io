jeeped say:

Not really a simplistic example at all as you have header labels in the first but not the second and some sort of title (e.g. Range 1 & Range 2) that fouls up retrieving the .CurrentRegion. The subtotal at the bottom of each range's third column complicates things further and requires that a count of the values in column A be used to resize the range to be sorted. Try this,

Sub sort_two_ranges()
    Dim rws As Long
    With ActiveSheet
        With .Cells(5, 1).CurrentRegion.Offset(1, 0)
            rws = Application.CountA(.Columns(1))
            .Resize(rws, 3).Cells.Sort Key1:=.Columns(3), Order1:=xlDescending, _
                Key2:=.Columns(1), Order2:=xlAscending, _
                Orientation:=xlTopToBottom, Header:=xlYes
        End With
        With .Cells(31, 1).CurrentRegion.Offset(1, 0)
            rws = Application.CountA(.Columns(1))
            .Resize(rws, 3).Cells.Sort Key1:=.Columns(3), Order1:=xlDescending, _
                Key2:=.Columns(1), Order2:=xlAscending, _
                Orientation:=xlTopToBottom, Header:=xlNo
        End With
    End With
End Sub

Edit: the full stops (aka periods) in from of .Application... were unnecessary. I'v removed them.

                       FIELD NAME                          FIELD TYPE ORDER

Then By          Task Outline Number           Assignment Ascending

NOTE:  You must leave the "Group assignments, not resources" checkbox to be able to select the Assignment option in the Field Type column.

8.  Click the Save button.
9.  In the More Groups dialog, select the new custom Group (if necessary) and then click the Apply button.

You will now see each resource, grouped by project and then by WBS.  This makes it much easier to see who is working in which projects and where in each project they are working.  Also, because you are using Microsoft Project 2010, your new custom Group is added automatically to your Global.mpt file, which means that your new custom Groups is now available to every current and future project you open.  Hope this helps.

This is usually caused by the lockdown zone setting.

Don't play with the registry unless you are at least sure you can correct any mistakes.

Start with PPT closed

Open Regedit

Go to this key:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings\Lockdown_Zones

You will probably see several folders below the key 0, 1 , 2 etc

Select "0"

In the right pane look for the 1200 value. If it is set to 3 change to 0.

Close and see if the swf now plays.

If it doesn't work you can go back and change to 3

If you do not know how to open Regedit probably best to ask someone to help you.
