rich007a say:

Yes, sometimes it feels like charts were nothing but an inconvenience to the Excel development team/managers... or sometimes it feels like charts in Excel are only designed for people who want to plot 4-bar bar charts showing sales figures... (and so if you are only plotting a series with <10 data points, the auto legend conversion to one entry per point might make sense).

 

Either way, you're not going to be able to change it.

 

Here's what I suggest.  With code, insert a new column to the right of your Y data.  Add a formula that tests for the condition you want for highlighted points.  For example, if your X values are in col A, and your Y values are in col B, headers in row 1, in C2 you could use  =IF(B2>6,B2,NA())    then fill down.

 

This will copy the Y values if that value is greater than 6, otherwise it will put #N/A.

 

You can now plot this new column as a second series.  You can give the series a name (e.g. "Y >6" in C1) and you'll have one legend for the main data and one legend for the highlighted "points".

 

If you choose a marker shape and size such that the second series obscures the same point in the first series (making it the same size and shape should do that too), I think you'll have the chart you want.

 

Alternatively, you could argue that if you only need one entry in your legend then you just don’t need a legend (so delete the whole legend, not one legendEntry at a time).  The chart title should be able to communicate what the chart is showing if there is only one series.

 

And finally, if it's taking time to display legends and delete them... disable screen updating should improve this (I think), but for me it took no time at all for the legend to switch to one entry per point when I changed a single point with VBA.

 

Hope that helps.

Using the COUNTIF function - Microsoft Community

jrhendrix say:

Using the COUNTIF function

I am trying to count the number of hospital rooms on a certain floor using the COUNTIF function.  The problem is that some of the rooms have a dash (ie 451 and 451-2) so I used "4*" and all it counted was the rooms with the dash.  How can I get the formula to include the rooms without the dash
e.j. gun say:

If your room numbers are in column A, starting in cell A2, then a formula like the one below will count only the rooms that start with "4" that do not have a "-" in them.

 

=SUMPRODUCT(--(LEFT(A2:A34,1)="4"),--(NOT(ISNUMBER(SEARCH("-",A2:A34,1)))))

 

If you are trying to count ALL rooms that start with 4, then use this:

=SUMPRODUCT(--(LEFT(A2:A34,1)="4"))

 

HTH,

Eric
