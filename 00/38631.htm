lemp say:

Searching for a program's file name and then manually deleting that file is not the way to uninstall anything other than the very simplest programs.  If that's what you do regularly, you're going to continue to have problems. 

Searching for file names will not find the numerous entries in the Windows Registry that most applications create when they are installed.  Undoubtedly, it is these entries that are causing your difficulty.  In addition, files related to any given program may not have the application's "friendly name."  That is, if you search for "IntelliType" you may not find itype.exe -- which is the name of the executable file for IntelliType.  Using the proper uninstall method should normally deal with both of these issues.

I suspect that the reason that "Add/Remove Programs" (or, as it's called in Windows 7, "Programs and Features") doesn't work is because when you "did a search and deleted the install" you deleted the program's uninstaller.  When you click the "Uninstall/Change" button in Programs and Features, Windows attempts to execute the uninstall routine (or uninstall information) created when the program was installed.  If this has been deleted manually, you can't uninstall the program properly. 

This is why it's common to suggest re-installing a program and then uninstalling.  Unfortunately, some applications -- and from your description it appears that IntelliType may be one such -- will refuse to even install the same version unless all remnants of the existing install are gone.  In this case, you have to search through the registry to remove entries relating to the program and possibly restore default entries. 

The Microsoft "troubleshooter" at https://support.microsoft.com/en-us/help/17588/fix-problems-that-block-programs-from-being-installed-or-removed (which, of course, is not third-party software), attempts to do this.  If this doesn't work, then you either have to fix the registry manually -- which I strongly suggest that you not attempt to do -- or use a third-party tool such as Revo Uninstaller that will attempt to do it for you automatically.

jessen p say:

Hi,

 

A shortcut is a link to an item (such as a file, folder, or program) on your computer. You can create shortcuts and then place them in a convenient location, such as on the desktop or in the Favorites section of the navigation pane (the left pane), so that you can easily access the item that the shortcut links to. Shortcuts can be distinguished from the original file by the arrow that appears on the icon.

 

If the program itself is saved in the desktop then the shortcuts present in the desktop on deletion will remove the program too. You can save the program in documents folder and then create a shortcut on the desktop. Then when you delete the shortcut the program won't get deleted.
