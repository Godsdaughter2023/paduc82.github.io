I recently purchased a 2017 Surface Pro and found the high screen resolution to be a problem. I couldn't use it unless my nose was 3 inches from the screen (I am 64). But all is good after using these two tips I picked up from other forums. You may have this already but I thought it might help anyone else that finds this post.

1. Change scale of text and icons
rt-click desktop / display settings / Scale and layout / 200%

This made my Internet Explorer and Edge readable as well as many other apps like Notepad.

2. Some apps they will run with tiny font by default; sometimes this will make them readable.

Locate the executable for the application.

rt-click the shortcut / open file location or just look in C:\Programs Files...

then

rt-click the app .exe / properties / Compatibility / Settings (about 2/3 down)
Override high DPI scaling behavior Scaling performed by / System (Enhanced)

Photoshop CS5 was unreadable until I did that.
