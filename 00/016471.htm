shane devenshire say:

On each of the sheets put a formula into the same cell which picks up the Subtotal, =R32 or whatever, different reference for each sheet.  Same cell in all sheet, lets say M1.

On the summary sheet, if you want a grand total as one number

=SUM(Sheet2:Sheet9!M1)

If you want to see each subtotal on the summary sheet and then the grand total you could put the sheet names in column A for example and then use INDIRECT to bring back each result.

The best way to do this is to create and use a "sheet sandwich".

Insert a new sheet to the immediate left of the first sheet to be included in the calculation. Name this sheet First.

Insert a new sheet to the immediate right of the last sheet you want to include in the calculation. Name this sheet Last.

Now, when you add new sheets for additional months just place these new sheets between First and Last then use this formula:

=SUM(First:Last!B4)
