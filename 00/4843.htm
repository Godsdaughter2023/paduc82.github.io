I then save and close the document, and open reopening, all of these style changes have been lost: the Heading 1's and 2's revert to the defaults.  Why can't I permanently change these?
suzanne s. barnhill say:

There is only one reason I can think of for this, but it's a long shot.

  1. Go to File | Options | Add-Ins.
  2. Beside Manage, select Templates, and click Go... This will open the Templates and Add-ins dialog.
  3. In the dialog, if the box for "Automatically update document styles" is checked, clear it.

When you modify a style in a document, it is not changed in the attached template unless you have selected that option (New documents based on this template rather than Only in this document). If the described check box is checked, then the next time you open the document, the style definitions in the template overwrite those in the document. Ordinarily you don't want this. Occasionally you would check this box if you're attaching a different template to a document and want to update the styles, but you don't usually leave it checked.
