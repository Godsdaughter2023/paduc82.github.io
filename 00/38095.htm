The "view reliability history" should be there.

If you click "Maintenance" it should be just below, on the far right.

You can also open it this way:

Start > type view reliability history in the "Search program and files" box and press "Enter"

But using the Event Viewer to look at the error is good and shows that the problematic file is this one:

nmspce2.dll_unloaded

The file which is causing the problem is the nmspce2.dll which is associated with Network Magic.

If you uninstall Network Magic the error should resolve.

It may be best to uninstall Network magic but if you want to use the Network magic software make sure you have the most recent version.

If problem persists you could use a program such as ShellExView to disable  extensions related to Network Magic   

There is a download link to ShellExView in the following link which also has some info on bad context menu handlers:

http://windowsxp.mvps.org/slowrightclick.htm

Alternatively you could try unregistering the nmspce2.dll - see "If you'd like to remove the Network Magic links from from the right click menu, follow these steps":

http://www.purenetworks.com/support/faq/48.html

If you use the instructions in the above link make sure you open an "elevated" command prompt:

Start > type cmd in the "Search programs and files" box > then right click cmd.exe and select "Run as administrator"

Then follow the rest of the instructions to unregister the nmspce2.dll.

  • The PC boots into Windows RE (Recovery Environment).

  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

Hi zp11,

2K display is a generic term for display devices having horizontal resolution of approximately 2,000 pixels. When you said 2k, you must be referring to a display resolution of 2048 x 1080 or 1920 x 1080 (can be considered as 2k).
