Hi Dale,

Installing an update for Windows is essential to ensure that you’ll get the upgraded and enhanced features of Windows. These updates also consist of security updates that will help a computer to be more protected. The Windows 10 version 1709 is the recent version Windows that can be installed on your PC.

It could be that having an insufficient storage may cause error when installing an update. We suggest that you try copying some of your personal files to an external storage device and use the Disk cleanup app to free up more space. Please click here for the steps.

Now, if you're still encountering issues with installing this feature update, we suggest that you run the Windows Update Troubleshooter. Then, you may also run the Background Intelligent Transfer Service tool. It can be used if you’ll encounter issues when installing updates and can be found under Control Panel. Here are the steps:

  1. Type troubleshooting in the search box.
  2. Click View all under the Control Panel Home section.
  3. Double-click Background Intelligent Transfer Service or right-click it, then, click Run as administrator.
  4. Restart your computer.
  5. Try to install the update again.

Just in case that you’ll encounter the same result, here are additional troubleshooting steps that you can perform to resolve your concern:

  • Make sure that your computer has enough space for the update. The device requires at least 16 GB of free space to upgrade a 32-bit, or 20 GB for a 64-bit OS. For additional information, see Free up drive space in Windows 10.
  • Run Windows Update a few times. Try to download and install any available updates in Windows Update. You can use the Windows Update Troubleshooter tool if just in case you’ll encounter an error.
  • Unplug extra hardware. If there are external storage devices and drives that are connected, unplug them so it won’t cause any conflicts with the update process.
  • Check Device Manager for driver errors. Type Device Manager in the Search box on the taskbar, then select it from the results. In the window that will pop up, try to look for any device with a yellow exclamation mark beside it (you may have to select each category to switch to the list of devices).   • Restore and repair system files. Run the System File Checker tool. Follow these steps:
  1. Type Command Prompt in the Search box from Start.
  2. Right-click on it, then select Run as administrator.
  3. Enter DISM.exe /Online /Cleanup-image /Restorehealth (note the spaces before each "/").
  4. Enter sfc/scannow (note the space between "sfc" and "/").
  5. Once done, try to check if it resolved the concern.
