Overview

Ambit-Ubee DDW2600 (U10C037)

Login Information:

  • Default Gateway - 192.168.100.1 / 192.168.0.1
  • Username – user / admin / root
  • Password – user / cableroot / root

How to Find SSID (Network Name)

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Click "Wireless" from the menu at the top.
  4. From the menu on the left click on "Basic".
  5. The SSID is on this page labeled Network Name (SSID) towards the top of this page.

 

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Wireless from the menu at the top.
  4. Select Security from the menu on the left.
  5. Look to see what type of passphrase is enabled from the four drop-down boxes at the top. For a WPA key, the passphrase will be listed in the box that says WPA Pre-Shared Key. Verify which key the gateway is using by looking at the drop-down box that says current network key. For a WEP key, look in the box that says Network Key

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Wireless from the top menu.
  4. Select Security from the menu on the left.
  5. Enable the desired security type. (WPA-PSK, WPA2-PSK)
  6. To turn off the security select Disable for all the security types.

MAC Filtering

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Click "Gateway" across the top.
  4. On the left had side click "MAC Filtering".
  5. If the Xbox is on this list put a check in the Clear box and select Apply.

Change Wireless Mode

*There is no option available on this device.*

 

Change Wireless Channel

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Wireless from the menu at the top.
  4. From the menu on the left select Basic.
  5. Select from the channel drop down the desired network channel. (Channels 1,3,9,11)
  6. Click Apply.

DHCP Reservation

(Note: You MUST force release of the IP address on the DHCP page first)

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Gateway from the menu at the top.
  4. Select "Static Lease" on the left.
  5. Enter in the Mac and IP address for the Xbox.
  6. Select the Enable Check box to the right.
  7. Select Apply.

 

UPnP

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Click "Gateway" across the top.
  4. On the left had side click "Options".
  5. There will be a "Enable" box next to UPnP.

DMZ

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Gateway from the menu at the top.
  4. In the section labeled advanced gateway, select DMZ Host.
  5. Enter IP address of the Xbox in the DMZ address box.
  6. Select Apply.

Disable DMZ

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Gateway from the menu at the top.
  4. In the section labeled advanced gateway, select DMZ Host.
  5. Enter a 0 in the DMZ address box. (if 0 is not allowed use 1)
  6. Select Apply.

 

Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Gateway from the menu at the top.
  4. On the left had side click "Forwarding"
  5. Enter in the ports and IP address for the Xbox
  6. (In the "Public Interface IP" section, just leave it at 0.0.0.0.)
  7. Select Apply.

Disable Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Gateway from the menu at the top.
  4. On the left had side click "Forwarding"
  5. Uncheck the Enable box of the ports you want to remove.
  6. Select Apply.

 

Firewall

Disabling IP Flood Detection and Firewall Protection

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Gateway from the menu at the top.
  4. After logging in to the device, click on Firewall at the top.
  5. Select Content Filter on the left.
  6. Uncheck IP Flood Detection and Firewall Protection.
  7. Select Apply.

Restoring Factory Defaults

Through Firmware

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Tools.
  4. Select Factory Defaults.
  5. Select Restore to factory defaults.
  6. Select Apply.

Through Hardware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Locate the reset button on the device.
  3. With paperclip/pen/toothpick hold down the reset button down for 30 seconds.
  4. Wait for lights to turn off then return to normal.

How to Find Firmware Version

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link)
  3. Before logging in the firmware version is located here labeled Software Version.

 

Firmware Update

*Firmware updates are installed automatically by the Internet Service Provider.*

Bridging Guide

*Bridge mode should only be used if you have a router to plug into the device.*

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Tools from the top menu.
  4. Select Operation Mode from the menu on the left.
  5. Select Enable to the right of "Bridge mode:".
  6. Click Apply.

 

 

Multiple Console Support

We have confirmed that this network device is unable to support more than one console with an Open NAT. 

 

Known Issues

This device will give NAT-like issues, random disconnections and in-game errors (such as "Call of Duty Servers Unavailable") while IP Flood Detection and Firewall Protection are enabled.  Try disabling one or both to resolve the issue.

ravinath p say:

Hi hank1111,

 

Welcome to Microsoft Community. It looks like you want to play the audio from the computer and from TV at the same time.

 

As per the design you would not able to hear sound from both speakers and HDMI for a single application at one time. Generally, in a normal circumstance we can only use one device as the output device. Example, the speakers sound is muted when the earphone is plugged to the computer.


This might be possible if you can have two different sound cards or you might have to use a splitter to get sound in both the speakers and HDMI.

 

However you may try this workaround and check if this works.

  1. Open Sound panel from the control panel. 
  2. Select Speakers as the default playback device.
  3. Go to the "Recording" tab.
  4. Right click and enable "Show Disabled Devices" .
  5. A recording device called "Wave Out Mix", "Mono Mix" or "Stereo Mix" .
  6. Right click on the new device and click "Enable".
  7. Right click on the new device and click "Set as Default Device" .
  8. Double click on the new device to open the Properties window.
  9. Go to the "Listen" tab.
  10. Click on the "Listen to this device" checkbox .
  11.  Select your HDMI device from the "Playback through this device" list .

 

And you're done, the drawback is that you won't have a microphone selected as default recording device.
 
Hope this information helps. If you  have furher questions feel free to reply and we will be happy to assist.

There is no need to modify Win10. The vast majority of games that ran on Win7 64bit will run on Win10. 32bit is not an issue.

- First stop with old games (why MS has not made this default on their 'gaming Windows' remains a mystery), is

Programs and Features > Turn Windows Features On or Off  and enable
Legacy Components > Direct Play.

- Some games need fixes/tweaks to configuration files, and occasionally using Compatibility for an older version of Windows can help.

A few pre 2002 games respond to Win95/09 as this forces the game to run on a single cpu core.

The Compatibility setting Override High DPI Behavior is necessary with old games that may not fit your monitor properly

(when the User has created a Custom Scaling for their desktop in  Display Settings )

- Another big issue, which disk based games is ... DRM Blocking.

I have more details on this evil perpetrated by MS in my post at  ---this thread---
