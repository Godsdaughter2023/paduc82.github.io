suzanne s. barnhill say:

To add to what everyone else is saying, what you are seeing is not a "ghostly prompt" or a "suggestion" that you use a header or footer. It is the header/footer area and the Ribbon tab with the appropriate tools for editing the header and footer. These appear only when you activate the header by double-clicking above the top margin. Perhaps it would be easier for you to avoid clicking in this area if you either hid the white space (as has been suggested) or displayed text boundaries (Office Button | Word Options | Advanced: Show document content) which will display your page margins as a dotted line.

To repeat: No one is trying to force you to add header or footer content; what you are seeing is the result of actions you have (albeit perhaps inadvertently) taken to activate the header/footer layer, which makes Word think you want to edit the header or footer.

FWIW, what you describe (a "popup" or "superimposed transparent page") does not describe the normal appearance of the header/footer pane, so I wonder if you are using some add-in that is resulting in a different way of displaying the header and footer. If, however, what you mean is that the text in your document is dimmed, then that is the normal appearance; when you are in the header/footer, document text is dimmed, and when you the document body is active, the header and footer are dimmed.

  1. Download and install Device Security Manager Powertoy for Windows Mobile 5.0 from Microsoft. This is a nifty utility to show what security policy is in effect. If you connect your Blackjack to your PC, you’ll see two tier security policy is in effect.

  2. Download and copy secpolicies.cab to your Blackjack using ActiveSync. I got this from sems.org who got it from Telus support, after following a post on qusers.com.

  3. Run and install it. This allows you to install third party SSL Root Certs.

  4. If you run Device Security Manager Powertoy for Windows Mobile 5.0 now (it is listed in Start - Programs as “Security Configuration Manager”, you’ll see now you are using One Tier Prompt. Select Security Off and click Provision. It will install tool.cab to your device and then provision the settings.

These instructions are based on the ones found for the Motorola Q provided by sem.org.
