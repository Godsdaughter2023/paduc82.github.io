The code below should be inserted into the Workbook module of your Visual Basic Project.  To do that:

 

  • Click on the Develper tab in Excel
  • Click on Visual Basic to open the Visual Basic editor
  • Right-click on the "Workbook" module of your file and select "View code"
  • Paste the code below into the window that appears.
  • Save your file

Every time you open the file, the value of the selected cell will increase by one.  That means that if you open the file seven times in one day, the value will increase by seven.  If that's not what you want, we have to get a little more sophisticated and track the date that marks the last time the value was changed.

 

HTH,

 

Eric

 

Change "Sheet1" to your worksheet name.

Change "$A$1" to the cell you want incremented.

 

'===== BEGIN CODE =====

Option Explicit

Private Sub Workbook_Open()
    Dim i As Long
    Dim theSheet As String
    Dim theRange As String
'
    theSheet = "Sheet1"
    theRange = "$A$1"
'
    ThisWorkbook.Worksheets(theSheet).Range(theRange).Value = _
        ThisWorkbook.Worksheets(theSheet).Range(theRange).Value + 1
'
End Sub
'===== END CODE =====

#1 - Am presuming that the "sidebar" you're referring to is in reference to the Outlook (desktop client) "Navigation Pane".  Just for sake of accuracy, the Nav Pane doesn't reflect all the "email accounts" configured in the current Outlook profile but rather <all> the "data files" that have been configured. In the case where you have one Imap and one POP account configured - there would be at least 2 data files so the Nav Pane should have 2 folder trees - one for each data file

#2 - A POP account would be associated with a PST file and would never reflect the folders for a given "email account" as shown via <webmail> since a POP account is a one-way download of the <Inbox> folder. Anything else must be created manually. No folders in a PST file (in OL'2013/'2016) get sync'd with the server.

#3 - Not sure what you mean by "OL'2013 has no access" - has no access in what way? You've configured your email account as IMAP and it's not sync'ing the email folders?
