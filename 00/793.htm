  2. Do you have an Anti-Virus/Internet Security program installed on the computer?
  3. Are you able to open other apps?
  4. Are you using a proxy server?

Please be informed that for the Windows Store and the Modern Apps to work, you need to have the following settings:
 
• Your screen resolution needs to be at least 1024x768.  Please check this before you proceed.
• If you have turned off User Account Control, your Metro apps will not work.

You can check the screen resolution by Going to the Desktop, right click and select Screen Resolution.
 
You can check if UAC is enabled here:
 

  1. Press Windows key + X.
  2. Click on Control Panel.
  3. Then click on User Accounts.
  4. Change User Account Control Settings.

If the above settings does not fix the issue, you may try the below methods and check if it fixes the issue. 
 
Method 1: Run the App Troubleshooter
 
Click on this link and save the troubleshooter in the computer and then run it. Follow the on-screen instructions to complete the troubleshooting process and check if it helps.
 
http://download.microsoft.com/download/F/2/4/F24D0C03-4181-4E5B-A23B-5C3A6B5974E3/apps.diagcab

Method 2: Clear the Store cache
 
You may try to clear the cache for the Store and check if it helps:
 

  1. Press the “Windows Logo” + “R” on the keyboard.
  2. Type “WSReset.exe” and press “Enter”.
  3. Reboot the computer and try opening the Store.

Method 3

Follow the steps given in the article below and check if it fixes the issue: (Applies to Windows 10 Technical Preview)

http://windows.microsoft.com/en-US/windows-8/what-troubleshoot-problems-app

Method 4: Update the Display driver

I suggest you to try updating the Display Adapter drivers check if it works for you.

Follow the steps to update the Display driver:

  1. Press Windows Key + X key and click Device Manager.
  2. Expand ‘Display Adapters’.
  3. Right-click on the Display Adapter listed and click on ‘Update Driver Software’.
  4. Select the option to automatically search for the updated driver software.
  5. Restart the computer after its updated and verify.

Also try to access the computer manufacturer's website and download the latest display drivers for your computer and install the same.
