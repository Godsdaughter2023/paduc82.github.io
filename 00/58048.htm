    1. Press Windows key + x and select Command Prompt(Admin).

    2. Stop BITS, Cryptographic, MSI Installer, and the Windows Update Services. Type the following command and press Enter after each command:

    net stop wuauserv
    net stop cryptSvc
    net stop bits

    ren C:\Windows\System32\catroot2 Catroot2.old

    net start cryptSvc
    net start bits

    6. After restarting your PC, you may try to install Windows Update.
