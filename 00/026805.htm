bernard liengme say:

Your Subject mentions multiple ranges by the text does not, so I went for the simple case

On worksheet called E I have text in rows 1 to 7

I have a shape; this has the sub below assigned to it, and it is set not to print

When you click the shape an input box pops us asking you to select a range (this can be from any sheet in the workbook - maybe from another workbook but I did not test this)

Whatever range is selected gets copied and pasted to A8 of sheet E

The sheet E is printed.

Hope this helps

best wishes

Sub MyPrint()
    Dim myRange As Range
    Range("A8:AA1000").Delete
 
    Set myRange = Application.InputBox(Prompt:= _
        "Please Select a Range", _
        Title:="InputBox Method", Type:=8)
 
    If myRange Is Nothing Then
        ' Range is blank
    Else
        myRange.Copy
    End If
    Worksheets("E").Activate
    Range("A8").Select
    ActiveSheet.Paste
    Application.CutCopyMode = False
    Range("A1").Select
   
     ActiveWindow.SelectedSheets.PrintOut Copies:=1, Collate:=True, _
        IgnorePrintAreas:=False
End Sub
