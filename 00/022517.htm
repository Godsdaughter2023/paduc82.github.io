hansv mvp say:

Here is such a macro:

Sub HideWhite()
    Dim r As Long
    Dim c As Long
    Dim f As Boolean
    Application.ScreenUpdating = False
    For r = 6 To 145
        f = True
        For c = 3 To 21 ' C to U
            If Cells(r, c).Font.Color <> vbWhite Then
                f = False
                Exit For
            End If
            Cells(r, 1).EntireRow.Hidden = f
        Next c
    Next r
    Application.ScreenUpdating = True
End Sub

One way you can check to see if the application did install properly is to look at the Remove Programs application on your device and see if Microsoft My Phone is listed there.  You can find this application by going to Start Menu > Settings > System > Remove Programs.  Once there, look for Microsoft My Phone to be listed in the "Programs in storage memory".  If it is not listed, Microsoft My Phone was not installed properly on your device.
