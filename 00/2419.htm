ken sheridan say: I suspect you'll have to use VBA domain functions for this rather than a subquery as I think you'll find the latter will not result in an updatable query:

UPDATE [Table] AS T1
SET [Main Category] =
    DLOOKUP("[Main Category]", "[Table]","ID = " &
         DMAX("ID","[Table]", "ID <" & T1.ID & " AND [Main Category] IS NOT NULL"))
WHERE [Main Category] IS NULL;

The name of the Main Category column does make me wonder whether the table is normalized to Third Normal Form (3NF), however.  If the table also includes a column such as SubCategory then it might be that the Main Category column is transitively dependent on the key via SubCategory, and the table would consequently not be normalized to 3NF due to the redundancy arising from the transitive dependency.  

Where there is a tree-structured hierarchy of entity types, only the lowest level in the hierarchy, should be referenced by any referencing table.  A simple example would be a table of locations which includes a CityID foreign key column, but no StateID column as the latter is determined by CityID, not solely by the key of the table, and its inclusion in the table as a foreign key would consequently be redundant, leaving the table open to the risk of update anomalies.

You cannot do this with a single SQL Statement. A SQL statement acts on all records that match the criteria in the WHERE clause. You can't have it change the action based on the changing order of the records.

I would do this with a loop. You loop through each record, and test the value of Main Category. If its not null you assign the value to a variable. If it is null you assign the current value of the variable to the field.

hansv mvp say:

I'm not a financial expert, so for what it's worth:

Let's say you have a start date in A2 and a starting balance in G2 (positive).

A3, A4 etc. contain payment dates, C3, C4 etc. contain yearly interest rates and F3, F4 etc. contain payment amounts (negative).

In B3: =A3-A2 (if Excel formats this as a date, clear the formatting)

In D3: =G2

In E3: =D3*(1+C3)^(B3/365)-D3

In G3: =D3+E3+F3

You can fill the formulas down.
