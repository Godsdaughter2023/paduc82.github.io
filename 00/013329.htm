gary's student say:

O.K.......the SUBSTITUTE() trick looked cool, but its not good enough.

 

Another approach is to split the string based on the position of the last blank in the string.  In C1 enter:

 

=TRIM(MID(A1,1,FIND(CHAR(12),SUBSTITUTE(A1," ",CHAR(12),LEN(A1)-LEN(SUBSTITUTE(A1," ",""))))))

 

Finding the last blank is based on Bob Umlas' post here:

 

https://groups.google.com/forum/?fromgroups#!searchin/microsoft.public.excel.worksheet.functions/last$20substring/microsoft.public.excel.worksheet.functions/N7rhYBV_KBY/hwipEzTBHzQJ

haseeb a say: If the numbers always be after the LAST SPACE, use in C1 to get the number,

=TRIM(RIGHT(SUBSTITUTE(TRIM(A1)," ",REPT(" ",20)),20))+0

If the number always be in 2 decimal format, even for the zeros too, ie: 375.50, 375.00 etc..use TEXT function

In B1 use,

=TRIM(SUBSTITUTE(A1,TEXT(C1,"0.00"),""))

julie fka profjulie say:

You might find Chris De Herrera's Pocket PC Faqs website interesting reading .

ActiveSync is used on XP machines

WMDC is used on Vista and Windows 7 machines.  WMDC is essentially the version of ActiveSync that runs under these operating systems.

If you want to sync contacts, appointments, tasks and/or email, then you must have Outlook installed on your computer.  As Brian mentioned, the only version of Outlook 2010 that can be used to sync is the 32 bit version.

Syncing is two-way.  New or edited data on the windows mobile device is synced over to the computer and new or edited data on the computer is synced over to the windows mobile device.  There is no option to make syncing one way only. If you delete some data on the mobile phone, it will be deleted on the computer and if you delete data on the computer, it will be deleted on the mobile phone when you sync.

There is an option in both ActiveSync and WMDC where you can set a "rule" that ActiveSync/WMDC uses whenever a conflict exists between the computer and the windows mobile device.  The rule will only come into play if you have changed a record on your windows mobile device AND you've changed the same record on your computer.  When you sync, ActiveSync/WMDC will recognize that changes have occurred on a record in both places and proceed in the manner you have selected (always replace the item on the computer; always replace the item on the mobile device).

You can have a syncing partnership for a windows mobile device with up to 2 computers.  You can sync numerous windows mobile devices to a single computer.

ActiveSync/WMDC only sync data between a windows mobile device and a computer.  Activesync/WMDC does not perform a full backup of a mobile device and backup software should be installed to a device and used on a regular basis.

 

BTW, if you have a problem with erroneous notifications and calendar alerts, you might be able to fix the problem without having to resort to a factory reset.  You can review the notification queue on a device and delete these notifications.  See my article entitled "Notifications and Alerts" for more information.

doug robbins - mvp office apps & services (word) say:

I would create a macro in your Normal.dotm template that contains the following code

 

With ActiveDocument.PageSetup
    .LeftMargin = InchesToPoints(0.5)
    .RightMargin = InchesToPoints(0.5)
End With

and then add a button to the Quick Access Toolbar to run it.

 

Or perhaps, use the following macro to print the documents

 

Dim lm As Long
Dim rm As Long
With ActiveDocument
    With .PageSetup
        lm = .LeftMargin
        rm = .RightMargin
        .LeftMargin = InchesToPoints(0.5)
        .RightMargin = InchesToPoints(0.5)
    End With
    .PrintOut Background:=False
    With .PageSetup
        .LeftMargin = lm
        .RightMargin = rm
    End With
End With
