    elrendhel say:

    Greetings wfma!  Welcome to the Windows Mobile Owners Circle Forums!

     

    ActiveSync (AS) is asking you to provide it with some general observations on how you will be using your device.  If you always do all of your work on the Desktop, then set AS to "replace the files on your device" so that you always have the most up-to-date files on the PDA.  But if you tend to use your PDA frequently to edit, compose, or otherwise update items/files, then you should set AS to "replace the item on your computer".

     

    If you want the choice to always be available to ask you which files to update, then set AS to "Leave the item unresolved".  This should cause AS to always ask you which way you would like it to work.  Be warned that if you choose this method and have hundreds of items to synchronize, that you will obviously have to answer the question "OverWrite?" on each and every item...
