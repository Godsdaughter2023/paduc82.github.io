We understand the frustration when things don't go the way they supposed to. Let me assist you to resolve the issue from here.

An NSIS error occurs when you download a program that fails an integrity check during installation. Failed integrity checks usually occur due to corrupt files. Under such circumstances it is advisable to install the program/drivers using a fresh copy. However, if it still does not help, follow the below methods and check if it helps to resolve the issue.

Method 1
