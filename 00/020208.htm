zorvek say:
You are using Excel 2010. Excel 2007 and later versions support a newer version of COUNTIF called COUNTIFS which allows for multiple criteria. This will work in all cases and eliminates the need for column F because it looks at column B and C separately:

=COUNTIFS(B:B,B2,C:C,C2)

If you want to maintain compatibility with Excel 2003 and earlier then use this concatenation formula for column F:

=B2&C2&CHAR(160)

which forces Excel to evaluate the value as a text string versus a numeric value when using your existing COUNTIF formula in column G.
