how do i create a pareto chart in excel

how do i create a pareto chart in excel

j.e. mcgimpsey say: You can find dozens of examples with a quick Google search, but the basic steps are usually some variant of:

1) list your categories and categorical data in columns (or rows). Sort largest to smallest.
2) add a column (row) with cumulative % totals. For instance, if your data is in column B starting at row 2, enter this in C2:

    =SUM(B$2:B2)/SUM(B:B)

    Format as a percentage and copy down as far as necessary.
3) Insert a column or bar chart with both categorical and cumulative % data.
4) Ctrl- or right-click on the cumulative % data series, choose Format Data Series, and click the 
     Secondary radio button in the axis pane.
5) Ctrl or right-click the cumulative data series and choose Change Series Chart Type and change the type to Line.
6)  Format axes as required to be aesthetically pleasing.

Restarting Footnotes in Word 2013 - Microsoft Community

joanne stover say:

Restarting Footnotes in Word 2013

I'm working in a single file that contains the main document and two appendices. Is there any way to start my footnote numbers from 1 for each appendix? The main document is using several sections, so selecting the option to start at 1 for each document section won't work.

guessed say:

If all your footnotes are continuous across the sections, make sure there is a section break in front of the Appendix and position your cursor in the Appendix.

Then go to References > Footnotes Dialog Launcher to open the Footnote and Endnote dialog.

In this dialog, change the Apply changes to = This section

Change the Numbering to Restart each section

Apply

On my machine, this gets the footnotes restarting in the appendix but continuing in earlier sections.

Yes, unfortunately, this is a longstanding weakness of footnotes in Word; restart by section is very restrictive; there is just no way to restart numbering and continue it across several sections. I would strongly recommend that you go to http://word.uservoice.com/, search for a suggestion for improvement in footnote numbering and vote for it. By default, endnotes are located at the end of the document. You can change this to the ends of Sections via References|Footnotes>Location>Endnotes:End of Section.
