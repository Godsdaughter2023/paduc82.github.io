doug robbins - mvp office apps & services (word) say:

I would use three textboxes (named for the following code as txt1, txt2 and txt3 and then use the following code

Private Sub txt1_Change()
With txt1
    If Len(.Text) = 3 And IsNumeric(.Text) Then
        txt2.SetFocus
    ElseIf IsNumeric(.Text) = False Then
        MsgBox "The entry must consist only of numbers"
    End If
End With
End Sub

Private Sub txt2_Change()
With txt2
    If Len(.Text) = 2 And IsNumeric(.Text) Then
        txt3.SetFocus
    ElseIf IsNumeric(.Text) = False Then
        MsgBox "The entry must consist only of numbers"
    End If
End With
End Sub

Private Sub txt3_Change()
With txt3
    If Len(.Text) = 3 And IsNumeric(.Text) Then
        [insert name of next control to receive focus].SetFocus
    ElseIf IsNumeric(.Text) = False Then
        MsgBox "The entry must consist only of numbers"
    End If
End With
End Sub

With the above code, when the user has made a valid entry into a control, the focus automatically moves to the next control.

You can have the hyphens as labels on your form for appearance purposes and use code to combine the entries into a string that includes the hyphens such as

txt1.Text & "-" & txt2.Text & "-" & txt3.Text
