Hi Afaq. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

Try unlocking from the Command Prompt:

Type CMD in Start search, right click Command Prompt result to Run as Admin.

Type the following command including your recovery key:

manage-bde -unlock C: -RecoveryPassword YOUR-BITLOCKER-RECOVERY-KEY-HERE

Once it is unlocked then you can turn bitlocker off with this command:

manage-bde -off C:

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
