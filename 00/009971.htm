Hi,

For reasons best known to Microsoft, this popup commandbar is called 'Cell' to customise it put this code in a general module and run it.

It will add the text MyMacro to the popup and when you click it, it will run a macro called 'MyMacro'

So change the 2 instances of MyMacro to the text you want and the single instance ofMyMacro to the name of your sub.

If you mess up then I've included code to reset everything

Sub CommandBar_Cell_Add_Macro()
Dim NewItem As Object
   Set NewItem = CommandBars("Cell")
   With NewItem
      .Controls.Add(Type:=msoControlButton, before:=1).Caption = "MyMacro"
      .Controls("MyMacro").OnAction = "MyMacro"
   End With
End Sub

 

Sub A_Mess()
CommandBars("cell").Reset
End Sub
