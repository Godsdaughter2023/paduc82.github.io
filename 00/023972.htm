If AH2 holds 34062 then in AI2 you can use the older SUMPRODUCT() style,

=SUMPRODUCT((CS!$C$2:$C$99=AH2)*(CS!$AF$2:$AF$99<>"cancelled")*((SIGN(CS!$K$2:$K$99)*CS!$K$2:$K$99)+(NOT(SIGN(CS!$K$2:$K$99))*CS!$J$2:$J$99)))

=SUMIFS(CS!$J$2:$J$99,CS!$C$2:$C$99,AJ2,CS!$K$2:$K$99,0,CS!$AF$2:$AF$99,"<>Cancelled")+SUMIFS(CS!$K$2:$K$99,CS!$C$2:$C$99,AJ2,CS!$AF$2:$AF$99,"<>Cancelled")

Did you use CTRL+F9 for the brackets around the IF TRUE statement?

{INCLUDEPICTURE {IF TRUE "C:\\temp\\{MERGEFIELD "PICTURE_NA}"} \d}

into the first label then use the macro http://www.gmayor.com/export_field.htm#TextToField to convert the text into fields. There is also another macro on that page to convert fields to text.

Attach the data source from the merge toolbar. Ensure that the document type is label (see http://www.gmayor.com/mail_merge_labels_with_word_xp.htm )

{ INCLUDEPICTURE C:\\temp\\Filename.bmp \d}
