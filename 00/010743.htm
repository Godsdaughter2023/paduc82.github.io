kristine arp say:

Hi,

 

 

Just like what Hector mentioned, the increase in the file size is normal because the font gets embedded when the document is exported to PDF. You can export the file to a minimum size to lessen the file size a little bit. Follow the steps below:

 

  1. Click File.
  2. Choose Export.
  3. Click the Create PDF/XPS button.
  4. On the Optimize for section, choose Minimum Size.
  5. Hit OK.
