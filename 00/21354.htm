One of the drawbacks of using Merge and Split cells is that you're still using the same table structure and your columns/rows aren't independent. Any changes you make to the cells will also affect the columns.

I'm not 100% clear on what's happening based on your description, but I suspect what you need to do is use nested Word tables, like this.

I left the defaults in the screenshot so you can get a better idea of what I'm referring to. In order to make the nested table appear seamless, I go to the Cell properties of the containing cell and set the margins to 0 (zero) inches. For that, with your cursor in the containing cell, on the Table Tools Format tab, click Properties. On the Cell tab, click Options. Deselect Same as whole table and set all of the margins to 0".

I also set the Preferred Width of the nested table to 100% in the Table Properties so it fills the cell. Here's the table again after those changes are made and you can't tell there are two tables. 

You may find you need to use multiple nested tables, which isn't uncommon when working with a complex layout.
