Bubbatron

 

suzanne s. barnhill say:

The only way to have something automatically repeat on every page is to anchor it to the header. Using "Different odd and even" headers, you could have the text box alternate sides. Unfortunately, the content would also repeat on every page; it, too, would be part of the running head.

Another approach would be to create the text box to your requirements and save it to the Text Box Gallery, as follows:

  1. Go to Insert | Text | Text Box and either choose one of the gallery selections or use Draw Text Box to create your text box.
  2. When the text box is selected, you will see the contextual Text Box Tools | Format tab. In the Arrange group, click Position and choose More Layout Options...
  3. This opens the Advanced Layout dialog. On the Picture Position tab, select Book layout: Outside of Margin.
  4. Make any other desired changes in the size, position, and formatting of the text box.
  5. When you have all the settings to your satisfaction, with the text box selected, use Insert | Text | Text Box: Save Selection to Text Box Gallery to add it to the gallery, thus making it easier to insert.

One caveat: Unless the limits have changed, you are limited to 32 linked text boxes, so you will not be able to link the text in all the sidebars throughout the book.
