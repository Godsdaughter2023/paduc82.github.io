Method 2:
Use Advanced sharing option in Windows 7
a.    Right-click a drive or folder, click Share with, and then click ‘Advanced sharing’.
b.    In the dialog box that appears, click Advanced Sharing. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
c.    In the Advanced Sharing dialog box, select the Share this folder check box.
d.    To specify users or change permissions, click Permissions.
e.    Click Add or Remove to add or remove users or groups.
f.     Select each user or group, select the check boxes for the permissions you want to assign for that user or group, and then click OK.
g.    When you've finished, click OK.
