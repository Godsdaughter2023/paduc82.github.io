a. user say:

Many Windows Services are very generic in that there is nothing machine, installation or user specific about them and from one machine to another the Services are all about the same (behind the scenes).

If a standard Windows Service is missing, corrupt or otherwise misbehaving it is usually easy to replace the guts of the Service with those from a system that is not afflicted.

Behind the scenes the Telephony Service is named TapiSrv:

Connect to my OneDrive and download the file called:

TapiSrv.reg

https://onedrive.live.com/?id=EC673DF6F8DFEE84!2368&cid=EC673DF6F8DFEE84&group=0

Save the file to your desktop or someplace you can find it.

Before importing the file into your registry it is wise to backup your registry and consider making a System Restore point so you can undo the adjustments if necessary.  Always have a way to undo any adjustments just in case something goes wrong.

ERUNT is a popular free registry backup tool you can use and you can get it here:

http://www.larshederer.homepage.t-online.de/erunt/

After downloading the TapiSrv.reg file and making necessary backup and restore points, double click the TapiSrv.reg file and respond in the affirmative to this message:

Optionally you can right click the TapiSrv.reg file and choose the Merge option.

Restart the system and check the Services now.

You can delete the TapiSrv.reg file when you are done.
