This is what the Workgroup Templates feature was designed for. In Word, choose File>Options>Advanced. Scroll down to the General section and click on the File Locations... button. Select Workgroup templates, then set a network share to be the template source.

This action sets the network share for Excel and PowerPoint as well. In any of these programs, you can choose File>New. Click on the CUSTOM link, then on the Workgroup Templates folder and select the template you want to open.

To update the templates, just replace the files on the network share.
