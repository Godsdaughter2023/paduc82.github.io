mark ole say:

Hi John,

It is possible that the Master Boot Record or the Boot Configuration Data is corrupt.

You can try to fix this using the USB recovery drive.

  1. Insert the USB recovery drive
  2. Choose to Install Windows from the screen that appears and then select Repair your computer.
  3. From the repair menu, select Troubleshoot > Advanced > Command Prompt.
  4. In the command prompt, type the following command one by one and press Enter.

    bootrec /fixmbr

    bootrec /fixboot

    attrib C:\Boot\BCD -h -r -s ren

    C:\Boot\BCD C:\Boot\BCD.bak

    bootrec /rebuildbcd

  5. Restart your Surface and try to reset the device again.

If this doesn't work, ensure that the partition is active.

  1. Open the command prompt again.
  2. Type DISKPART and press Enter.
  3. Type list disk to list available hard disks.
  4. Type select disk 0.
  5. Type list partition to list available partitions on your hard disk.
  6. Select the partition that has the largest size.
  7. Type Active to mark partition to active.
  8. Type Exit to exit diskpart.
  9. Type Exit again to exit command prompt.
  10. Turn off your Surface and try to reset your device again.

If you're still unable to reset your device, run Check Disk.

  1. Open the command prompt again.
  2. Type chkdsk /r and press Enter.
  3. Once done type Exit and press Enter.
  4. Turn off your Surface and try to reset the device.

Let me know how the issue progresses. Similarly, if you have more questions, please inform me by replying to this post.
