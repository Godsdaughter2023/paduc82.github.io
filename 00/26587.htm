john mcghie say:

Hi Giuseppe:

Yes, there is: you make a set of styles for English and a set of styles for Spanish.

Text formatted with the Spanish styles will be spelled in Spanish, and vice versa.

If you create or modify a style, when you click the Format button, one of the options is "Language".  This enables you to set a language for that style (and any following styles based on it...) to a language different from the document default language.

If you have only one language appearing at a time in your documents, you can create a template for English and one for Spanish.  Set the Language of the Normal style and click the "Default" button, which will set that language into the document defaults.

Whenever you create a document from the Spanish template, the entire document and all its styles will be set to Spanish.

If you want several languages in a document, create a set of styles for each language, and apply the styles to the text appropriately.

I agree with "Other John": Mac Word does not have the "Detect language automatically" feature, and on the PC it is best turned off, it gets things wrong too often.
