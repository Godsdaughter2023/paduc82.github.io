When I select the table of content these extra numbers are not highlighted in grey. I can delete them but they reappear when the table is refreshed. The numbers do not appear in the body of the document.

 

What are these things and how can I get rid of them?

 

Thanks in advance, much appreciated!

 

Greg

suzanne s. barnhill say:

Second-level entries are, basically, any that use the TOC 2 style. If you have used Word's built-in heading styles in your document and inserted a TOC using the default settings, then your Heading 1 paragraphs will appear in the TOC in the TOC 1 style, Heading 2 paragraphs in the TOC 2 style, and so on. TOC styles are used only for TOCs, and they can be formatted entirely different from the heading styles to which they correspond. You can also assign styles from a different level; for example, you can make Word use TOC 3 for the Heading 2 entries. More on this at http://WordFAQs.mvps.org/TOCTips.htm.

The TOC 2 style is not getting numbering from anything in the document; the numbering has almost certainly been added to the style itself. The concept of adding numbering to a TOC style is addressed at http://WordFAQs.mvps.org/TOCTips.htm#UnnumberedHeadings.

suzanne s. barnhill say:
The TOC 2 style is automatically used for second-level entries in a TOC (and nowhere else). It's not a question of disabling it but rather of modifying it if necessary. See http://www.shaunakelly.com/word/styles/ModifyAStyle.html for instructions on modifying styles. You'll need to find out whether numbering has been applied and, if so, remove it.

bthomas say: I just found a solution that worked and I will make it a little easier for those who need it.

Create a batch file with the following and run it from a command prompt.

*******Start File Below********

regsvr32 MSXML3.dll /s
net stop wuauserv
cd /d %windir%\SoftwareDistribution
rd /s /q DataStore
net start wuauserv
regsvr32 softpub.dll
regsvr32 mssip32.dll

******End File Above*******

Go to services and stop the Windows Update service
Go to the Windows directory (default is c:\Windows) and rename the "SoftwareDistribution" folder to "SoftwareDistribution.old"
Go to services and start the Windows Update service
