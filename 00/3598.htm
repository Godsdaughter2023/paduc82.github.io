One way to change the formula would be, if the table body is named Table1 and the Total of Hours is in column D, change the formula to
=SUBTOTAL(109,Table1 D:D)
that is a subtotal of the cells in the intersection of Table1 and column D.  This formula will then copy to the other total cells and adjust appropriately.

Solution:

1 Install the latest service pack
2 Install the latest Cumulative Update for Excel
3 Turn off the Preview Pane, Details Pane and pop ups that show information about file and desktop items
4 Add the OpLocks registry keys
5 Install operating system updates
6 Use MSConfig to stop third party programs and services, including antivirus software
7 Delete the orphaned ~filename.xlsx file from the directory.
