zachd123 say:

I agree with Volunteer J.


The generalized terms "MP4 Video", "Movie Clip", "Video Clip" are the FriendlyTypeName values for the associated file handler for the given extension.  Simplifying things greatly: for foo.AVI we look up HKCR\.avi which on my system finds (HKCR\)WMP11.AssocFile.AVI as the associated type handler.  Then under HKCR\WMP11.AssocFile.AVI we note that the MUI-friendly (localization-friendly) FriendlyTypeName maps to some opaque string #9905 in the file unregmp2.  This happens to map to "Video Clip" as would be inspectable in unregmp2 via resource viewer or as is strongly hinted at (although not definitively identified since the HKCR registry is writable) as being "Video Clip".


So, long story short:

* Your best bet is Volunteer J's option.

* If you really wanted to unwind the typehandler naming scheme you'd need to get rid of the friendlytypenames and change the typehandler defaults to some name you desire.  However since this area is definitively the Windows Media Player "internal" file handler space, your own friendly names can and will be overwritten anytime the file handlers are written to by the system or unregmp2.


I vote Volunteer J's option. :)

ramesh srinivasan say:

As a workaround, I did add the file extension column, but my question remains: why are they different, for the same folder, but viewed on different computers using the same OS version and with same folder settings?

The difference is:

In PC1, Default Programs wasn't used even once.
In PC2, Default Programs was used to set WMP as default.

In the latter case, the ProgID and Hash is modified here:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FileExts\.mpg\UserChoice

In the factory default setting, the ProgID is set to a nonexistent AppID (Appxxxxxxxxxx something), and in that case Windows thinks the file type is unassociated, and then displays the generic type ("MPG file").

You can compare the REG settings yourself, using this tool.

1. Download FileExtInfo.zip from here: 
    http://windowsxp.mvps.org/fileextinfo.htm

2. Unzip the utility and extract the files to Desktop.
3. Double-click "FileExtInfo.exe" to run it.
4. Select .MPG file type from the list box
5. Click "View file association report"
6. Copy the contents of the report and include it in your reply.

Do the same in your another PC, and rename the file as MPG2.txt

Compare MPG1.txt and MPG2.txt you'll know the difference.
