tryx3 say:

The wifi miniport only exists after you have succeeded in setting up a hosted network - it is a virtual device. 

If you can, it would be worth restoring a system image made before all your attempts - just to make sure that there is no debris around to get in the way of your repeated procedure.

The following records have been cut & pasted directly from the command prompt windows I used to successfully set up and use this utility [in Windows 7 & Windows 10].  I mention this because when I tried to find out about doing it I discovered lots of rubbish written by people who had either never tried it & were copying inaccurately from somebody else's notes or they were just giving out from their inaccurate memories.

1  You first need to check that your WiFi card is capable of this job.  Enter this in an elevated command window

netsh wlan show driver

and check that the entry

Hosted network supported : Yes

appears in the results.

2  The WiFi must be turned on before you issue the next command [it does not need to be connected to anything]

netsh wlan start hostednetwork

and you should then get the response

The hosted network started.

[If you do not have the WiFi on you'll get this response - The hosted network couldn't be started.  A device attached to the system is not functioning.]

3  Once you are happy that you are getting the right response then stop the network

netsh wlan stop hostednetwork

4  Then start it for real with an SSID & password

netsh wlan set hostednetwork mode=allow ssid=TestNet key=password

and you'll get the response

The hosted network mode has been set to allow.
The SSID of the hosted network has been successfully changed.
The user key passphrase of the hosted network has been successfully changed.

5  Then you can start it again [as you will have to do with every reboot of the system - you do not need to keep entering the SSID & password as the system will just keep on using the ones you put in during step 4 above].

netsh wlan start hostednetwork

6  You can also get some info on its operation with

netsh wlan show hostednetwork

I just checked these parameters to see what the security was as I could not connect my WinXP [but did connect both a phone's hotspot & a Win10 computer]

7  These are the guides that helped me with my experiment in setting up Windows 7 for a hosted network.  They go further than my notes do, they include setting up ICS.

Previous title - Music and Audio

considerate_guy say:

If you are running the latest version of Movie Maker (version 2012, build 16.4.3505.0912) then there are several choices on the Project tab to decide which audio stream to emphasize.  You want to choose Emphasize Video so that when you have a video clip (with audio) it will sound louder and the music will soften.  It will automatically increase the sound of the music once you go back to just pictures.  For more details, read "What's New in Movie Maker 2012"

 

If you are on an older version of Movie Maker, you can download the latest as part of Windows Essentials 2012.

 

If you are perhaps on Vista or otherwise unable to upgrade beyond Windows Live Movie Maker 2011, you can still do this, but you'll have to do it manually.  Each time you add a video clip, make a pair of splits in the music track (one where the clip starts and one where it ends).  Then you can set the volume on that split music clip from the Music Tools tab by setting the Music Volume.
