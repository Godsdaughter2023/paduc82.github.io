ron coderre say: MS Query will use database rules when reading data.
That means it will assign a data type to each field and ignore any field values that do not match that data type.
In your example, it appears that the first column begins with a number, so MS Query
assigns a numeric data type to that field and ignores any non-numeric data.

I think you might benefit from importing the data instead.

Try this:
• Data.From_Text
• Navigate to your CSV file....Select it.....Click: Import

Display a Criteria Parameter in an Access form - Microsoft Community

pjmaustin say:

Display a Criteria Parameter in an Access form

Is there a way to display a criteria parameter from a query in an Access form?  I know it can be displayed in a report.

What I really want to do is prompt for some data from the user when they open the form.  When the user adds a new record, I would like that paramater to default into one of the fields in the record.

ken sheridan say: You will need to tell the form which field it's being filtered on, which you can do by passing the name of the control as the OpenArgs argument of the OpenForm method into the form with:

DoCmd.OpenForm "YourFormName",  _
    WhereCondition:= "[Teller #] = " & InputBox("Please enter Teller #."), _
    OpenArgs:="[Teller #]"

This assumes that Teller # is a number data type.  If it's a text data type amend the code as follows:

DoCmd.OpenForm "YourFormName",  _
    WhereCondition:= "[Teller #] = """ & InputBox("Please enter Teller #.") & """", _
    OpenArgs:="[Teller #]"

In the form's Open event procedure put:

If Not IsNull(Me.OpenArgs) Then
    Me(Me.OpenArgs).DefaultValue = """" & Me(Me.OpenArgs) & """"
End if

Rather than using an input box, however, a better solution would be to open the form from an unbound dialogue form in which the teller # can be selected from a combo box which lists all teller # values.  You can then reference the combo box in place of calling the InputBox function.

doug robbins - mvp office apps & services (word) say:

To retain the link, in Word, click on the Office button and then on Word Options and then on Advanced and in the Cut, copy, and paste section of the dialog, set the Insert/paste pictures as: to "In line with text"

Thereafter, if you copy the chart from Excel and then use Paste>Special then click on the Paste link button and then select Microsoft Excel Chart Object, the chart will be linked to the source by a { LINK } field.
