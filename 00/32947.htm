¡firedog say:

Is there an easy way of moving all the emails, contacts and account settings from one to the other? 

Two ways to move messages are given by ...winston MVP on this page: Moving Messages from Vista’s Windows Mail to Windows Live Mail. The instructions are old, but they still apply. When you get to importing into Windows Live Mail, please take note of the caveats on this page: Export and import messages

Contacts

Your contacts are saved as files in the folder at C:\Users\{user}\Contacts. Copy the entire contents of that folder from the Vista machine to the same location on the W7 machine ({user} may well be different, of course). Then, in Windows Live Mail, switch to the contacts window (press Ctrl-3 or click the address book icon at the foot of the folder pane), click the Import button and select Address book for current Windows user.

Note that if you're signed in in Windows Live Mail when you conduct the import, the contacts will synchronize with those at contacts.live.com. If you're not signed in, they will be stored as the Home tab of the ribbon will tell you whether you're signed in or not.

Account settings

In Windows Mail, go to Tools > Accounts > {account} and click Export. Repeat for each account, then copy the .iaf files to the W7 machine. In Windows Live Mail, press Alt-F O E to open the Accounts dialogue, then use Import to import the .iaf files. Note that saved passwords will not be included, so these will have to be re-entered.

First let me explain what is Ntldr and bootmgr,

NT Loader (NTLDR) is a software program that helps your computer's operating system start. If your computer says the NTLDR is missing, then your computer is trying to boot up from a non-bootable source, or a drive that does not have the NTLDR software. Ntldr is the boot loader for all releases of Windows NT operating system up to and including Windows XP and Windows Server 2003. NTLDR is typically run from the primary hard disk drive, but it can also run from portable storage devices such as a CD-ROM, USB flash drive, or floppy disk. NTLDR can also load a non NT-based operating system given the appropriate boot sector in a file.

BootMgr is the file name for the Windows Boot Manager, introduced with Windows Vista and included on the Windows 7 operating systems. This file works with your computer's basic input/output system, called BIOS, to tell the computer where to look for your operating system files. The Boot Manager operates each time you power on or restart your Windows 7 computer.

I can understand that on reboot you get an error “missing ntldr press ctrl, alt del, to restart” this issue occurs for various reason.
