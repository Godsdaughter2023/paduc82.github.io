Hi,

 

You have to exclude blank cells, try this and remember to ARRAY enter it

 

=AVERAGE(IF(Summary!A$8:A$500="Phone", IF(Summary!F$8:F$500<8,IF(Summary!$F$8:$F$500<>"",Summary!F$8:F$500))))

 

This is an array formula which must be entered by pressing CTRL+Shift+Enter

joeu2004 say: Mark wrote:

I tried using the following formula:
=AVERAGE(IF(Summary!A$8:A$500="Phone", IF(Summary!F$8:F$500<8,Summary!F$8:F$500)))

 

The average that it returns is different then if I do the average manually on the sheet. I am looking to return the average of values less then 8.00 from a list containing values of 1 through 10 in Column F:F, some cells contain a blank value not to be calculated in the average.    For some reason it is not working.  Any suggestions?


Without a concrete example, we can only guess as to what the problem might be.  (Sigh.)

First, you mention that blank cells should not be included in the average.  If column F might be empty (no constant; no formula), but column A might still contain "phone", your condition Summary!F$8:F$500<8 is not strong enough because an empty F evaluates to zero, which is indeed less than 8.

That problem can be corrected as follows:

=AVERAGE(IF(Summary!A$8:A$500="Phone",
IF(Summary!F$8:F$500<>"",IF(Summary!F$8:F$500<8,Summary!F$8:F$500))))

Second, if column F contains formulas, what appears to be 8.00 might actually be something more.  Consequently, it will be excluded from the average that Excel calculates, but you might include it in the check average that you calculate manually.

That problem can be corrected as follows:

=AVERAGE(IF(Summary!A$8:A$500="Phone",
IF(Summary!F$8:F$500<>"",IF(ROUND(Summary!F$8:F$500,2)<8,Summary!F$8:F$500))))

Third (but necessarily "finally"), all of these formulas must be array-entered.  That is, press ctrl+shift+Enter instead of Enter.  Excel will display an array formula surrounded by curly braces in the Formula Bar, i.e. {=formula}.  You cannot type the curly braces yourself.  If you make a mistake, select the cell, press F2 and edit, then press ctrl+shift+Enter.
