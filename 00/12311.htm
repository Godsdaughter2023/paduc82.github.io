jllatham say:

Remember that you can always call a worksheet function from within code.  So you could do something like this:

IF Application.WorksheetFunction.IsText( thingToTest ) Then

where thingToTest could be a Range() or a value in a cell or variable.

Lets say that cell A2 holds "123 Main Street" and we used the above like this

If Application.WorksheetFunction.IsText( Range("A2") ) Then

      msgbox "It is text"

Else

      msgbox "Not Text"

End If

We would find that we get the "It is text" message.  But if A2 held just 123 then we would get the "Not Text" message (it is numeric).

NOTE:   VBA's IsNumeric() for our cell A2 with "123 Main Street" will return TRUE because it looks to see if the thing being evaluated can be turned into a number.  If we did this in VBA

X = Val("123 Main Street")

we would find that X = 123 (a number) after the operation.

willy denoyette say:

    Standby (S0 Low Power Idle) Network Disconnected
    Hibernate
    Fast Startup

Hibernate = deep sleep.

Close Lid : Sleep (Standby above) for both AC and DC power.
Power button : Sleep (Standby above) for both AC and DC power.
Sleep button: Sleep (Standby above) for both AC and DC power.
Start menu power button: Sleep (Standby above) for both AC and DC power.
So all manual actions result in a Standby (S0 low power idle) transition.

Auto sleep/hibernate transitions:
Auto sleeps after: 11'48"
Hibernates after : 4h13'

That means your system enters Sleep after 11'48" of no user activity and after 4h13' of no user activity following the initial sleep.
That means hibernate after 4h13' + 11'48" of no user activity.

Now the question is: does your system behaves wrongly, as described in your OP, after having been asleep for more than 4h25' ?

stefan blom say:
You can change the text direction in text boxes, shapes, table cells, and frames. If you changed the text orientation (right-to-left versus left-to-right) in Word 2003, I guess you had a right-to-left language installed in that version.
 
On the other hand, if you are referring to page orientation, choose "Landscape" on the Margins tab of the Page Setup dialog box.
 
To display the dialog box, first click the Page Layout tab of the ribbon; then click the dialog box launcher on the bottom right of the Page Setup group.
