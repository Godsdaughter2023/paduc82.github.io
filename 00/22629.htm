jay freedman say:
That's one of Word's current stupidities. When you use the Top of Page or Bottom of Page items, whatever you insert replaces anything already in the header or footer.

First put in the rest of the header. Then position the cursor where you want the page number (if necessary, click at the end of the existing header text and press Tab once or twice to move to the right margin). Click Page Number and choose Current Position (instead of Top of Page). Click Plain Number (or a fancier entry). That will insert the page number without removing the existing header text.

If you want the same header text on the first page as on the second page (but not the page number), you need to copy that text before clicking Different First Page. After making a different first page header, paste the text there.

The thing to understand here is that checking Different First Page displays a completely independent header area that has no connection to the second-page header.
