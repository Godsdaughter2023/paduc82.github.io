rob koch say:

Clearer statement; the firewall is an inbound protection tool, not an outbound one.

If you are using a 'two way' firewall, you are really operating a simplistic IDS/IPS system generally making the user a part of the IPS by asking innane questions.  If that's the kind of security you wish to maintain then feel free, but Microsoft's telemetry has found these to be entirely ineffective in the hands of the general non-technical user population where MSE is targeted.

So neither Windwos Firewall nor MSE are intended to monitor or block 'leaks', since by the time these have occured your system is already compromised and the malware can simply disable your security anyway.  MSE is instead designed to try and block and or kill the actual malware processes and remove them from the system, which is what an anti-malware is supposed to do.

rajithr say:

Hi Ajay,

Thank you for posting your query in the Microsoft Community forums.

Let me try and help you with the query.

It is not advised to cancel a check disk scan. It scans to check the file system and status of the system's hard drives.

To disable check disk, you would need to use the the /x switch on chkntfs command in command prompt. The /x switch will exclude a drive from the default boot-time check.

Follow these steps:

a. Click on Start, type command prompt. Right-click on Command Prompt and choose Run as Administrator.
b. If you have drive C: as your hard drive, then the command to disable check disk from scanning C: is:

chkntfs /x c:


If you have two drives, C and D, you can disable check disk by entering this command"

chkntfs /x c: d:

Press Enter after each command.

Restart the system and check if it helps. If you have more queries, we will be happy to help.
