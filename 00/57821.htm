marilyn o say: Hello JoeHopewell,

See if you have Windows Media Player enabled.
1. Go to Start and in the search type "Turn Windows features On or Off".
2. Click on "Turn Windows features On or Off".
3. Browse to the Media Features and check the mark in front of Windows Media Player.
4. Restart the computer

You can also check to see if you have Windows Media Center enabled as well.

Now go to Start, All Programs. You should have Windows Media Player in the list of programs.

Please let me know if this helps.
