The IF function allows you to make a logical comparison between a value and what you expect by testing for a condition and returning a result if True or False.

=IF(Something is True, then do something, otherwise do something else)

    hansv mvp say:

    Auto recovery files are saved in .xlsb format. This is usually the smallest of all Excel formats; the same workbook takes up less space as .xlsb than as .xlsm or .xlsx.

    In general, the Auto recovery file will be complete, but it won't hurt to perform a quick check.
