Is it a setting somewhere?

 

a. user say:

Hello aland7169,

Using Exchange with WP7, the way syncing works is like this:

  • When you make changes to your Exchange account on your phone (ie...Moving mail to folders, deleting/reading email).
  • Those changes aren't reflected on the Exchange server until the next phone sync. (It does not happen instantly)
  • The phone will next sync next time there is mail to be picked up.
  • When the phone makes a call to the server for the newly arrived mail, the changes you made on the phone should now be reflected and synced on the server side.

This is by design, so that the Exchange client isn't using battery to sync unnecessarily every time you make a change.  If you wish to see instant syncing when making changes, you can manually tap the sync icon at the bottom.  Or you can simply wait for the next passive sync/mail delivery.

hansv mvp say:

Right-click one of the option buttons.

Select "Format Control..." from the context menu.

Click in the "Cell link" box.

Either type the address of a cell, e.g. A1, or point to the cell.

Click OK.

The cell that you assigned should now contain a number that corresponds to the option button that is selected: 1 for the first one, 2 for the second one etc.

If you clear the cell value, the option button should be cleared too.
