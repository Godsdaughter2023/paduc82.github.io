anitalewis say:

Importing contacts from Comcast into Outlook 2013

karl timmermans [outlook mvp 2012-2018] say:

If you're referring to your Comcast address book as the contacts used via the Comcast webmail account - then you would have to export the contacts from Comcast into a CSV file and then import those into Outlook using the Outlook import wizard.

Similarly, if you're referring to your Comcast email account which has been configured in Outlook as either a POP or Imap account - you would need to do the same since "contact" info is not included in POP/Imap accounts

How do I export my contacts from Comcast Webmail?
http://support2.constantcontact.com/articles/FAQ/2035

How do I export my Comcast email contacts to Excel

How to Export Comcast Contacts
http://www.ehow.com/how_5723076_export-comcast-contacts.html

Once you have the data exported - import the CSV file via the Outlook '2013 import wizard

File --> Open & Export --> Import/Export --> follow the appropriate screens and ensure that you click on the "Map Custom Fields" when the button presents itself to ensure that the fields in your CSV file are properly/completely mapped to the correct Outlook field
