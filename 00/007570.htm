Also, the key point in stacking one object in front of another is a matter of Text Wrap. The object you want in front [i.e., on top] must have Text Wrap applied to it. The type of Text Wrap you need to apply depends on whether the underlying object is set to In Line with Text or has some form of Text Wrap applied to it as well.

I can't be sure without seeing it, but it sounds like the underlying object is set to In Line. Try this;

  1. Right-click the Text Box, then select Wrap Text > None.
  2. Right-click the Text Box again & select Arrange> Bring in Front of Text.

Re "Arrange disappears": The Arrange group is on the Format contextual tab for the type of object selected. I think what's happening is that you have a picture selected which produces the Format Picture contextual tab.   1. In Line with Text & treated as though they are an alphanumeric character in the text flow, or
  2. Floating - Having some form of Text Wrap applied, in which case they are anchored to a paragraph.

Managing graphic objects in a Word document can be done, but to be successful it's necessary to understand how the program works. It can be very challenging when dealing with floating objects as well as how multiple objects interact with one another & with the body content.

gord dibben say:

You could use VBA event code in Sheet1 module.

Right-click on Sheet1 tab and "View Code"

Past this code into that module then Alt + q to return to Sheet1

When you put a number in A1 then hit Enter key that number will be copied to next available cell in column A of Sheet2

Private Sub Worksheet_Change(ByVal Target As Excel.Range)
    On Error GoTo stoppit
    Application.EnableEvents = False
    If Target.Address = "$A$1" And Target.Value <> "" Then
        With Sheets("Sheet2").Cells(Rows.Count, "A").End(xlUp)
            .Offset(1, 0).Value = Target.Value
        End With
    End If
stoppit:
    Application.EnableEvents = True
End Sub

How to add a sound to an animation?

mukherjee ashutosh say: http://office.microsoft.com/en-us/powerpoint/HA012329951033.aspx?pid=CH101956281033

Add a sound to an animation

 

You must have previously added an animation effect to the text or object (object: A table, chart, graphic, equation, or other form of information. Objects created in one application, for example spreadsheets, and linked or embedded in another application are OLE objects.) before adding a sound to the animation (animate: To add a special visual or sound effect to text or an object. For example, you can have your text bullet points fly in from the left, one word at a time, or hear the sound of applause when a picture is uncovered.).

  1. Click the slide that contains the animation effect to which you want to add a sound.
  2. On the Animations tab, in the Animations group, click Custom Animation.

     

  3. In the Custom Animation task pane (task pane: A window within an Office application that provides commonly used commands. Its location and small size allow you to use these commands while still working on your files.), click the arrow to the right of the animation effect in the Custom Animation list, and then click Effect Options.
  4. On the Effect tab, under Enhancements, click the arrow in the Sound list, and then do one of the following:
    • To add a sound from the list, click a sound.
    • To add a sound from a file, click Other Sound, and then locate the sound file that you want to use.
