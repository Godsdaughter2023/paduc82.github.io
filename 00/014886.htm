ronrosenfeld say:

Here is a VBA script using a Regex to replace those values.  Note that I am also replacing the space(s) between the end of the name and the "("

VBA's replace function has the same limitations as the Worksheet Find/Replace dialog.  In order to use a true regex, you need to reference the regular expression library, using either late (as I do below) or early binding.

Option Explicit
Sub RemoveParenthesizedNumbers()
    Dim R As Range, C As Range
    Dim RE As Object
Set R = Application.InputBox("Select range to Process", Type:=8)
Set RE = CreateObject("vbscript.regexp")
With RE
    .Global = True
    .Pattern = "\s*\(\d+\)"
    For Each C In R
        C.Offset(0, 1) = .Replace(C.Text, "")
    Next C
End With

By the way, if your data is truly as you show it above, you can accomplish the same result with this "one-liner" in VBA

  SomeVariable = Trim(Left(s, InStr(s, "(") - 1))

Or with this worksheet function:

   =TRIM(LEFT(A1,FIND("(",A1)-1))
