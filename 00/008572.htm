PDf to Word - Microsoft Community

pattisellers say:

PDf to Word

How do I change a PDF document to Microsoft Word document?

jay freedman say:

Because you didn't give any information about what version of Word / Office you have, the answer is "it depends".

If you have Word 2013 or Word 2016 (or Office 365), use the Open command in Word (Ctrl+O) and select the PDF file in the dialog. Word will automatically convert the file to Word format. (Caveat: To create editable text in Word, the PDF file must have been created from editable text. If it was just a scan of a paper document, the result in Word will be just a picture of the scan, and you would have to use Optical Character Recognition software to "read" the text.)

If you have Word 2010 or earlier, it can't open or convert a PDF. There are, however, lots of online services -- some of them free -- that will accept the PDF as an upload and send you the resulting Word document. Just search the web for "PDF to Word".

slaboure say: Auto-Reply.

I was able to make it work... I don't really know how, which is frightening.

I started by modifying some PPP features in the /ppp/gprs-connect-chat file (mostly increasing the timeout values and adding a space before the '=' of cgdcont). But it didn't work. So I went through the wizard again and it worked. Content of my file is now:

TIMEOUT 5
ECHO ON
ABORT '\nBUSY\r'
ABORT '\nERROR\r'

ABORT '\nNO DIALTONE\r'
ABORT '\nRINGING\r\n\rRINGING\r'
'' \rATZ
TIMEOUT 12
OK 'at+cgdcont=1,"IP","gprs.swisscom.ch"'
OK-AT-OK 'ATD*99#'
CONNECT ""
