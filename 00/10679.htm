Problems arise where there are merged or split cells, but for a simple table it is quite a simple process e.g.

    Selection.Tables(1).Columns(2).Select

bernie deitrick say:

Basically, all your code boils down to this - I wasn't sure if you want to count blanks in column I or J: I used J, and I put the blank count into cell B21. And you don't really need the formulas on "Combined" to get the values on Summary - you can calculate those directly to put the values into those cells, or write formulas that reference "Combined" on "Summary".


Sub TestMacro()

    With Sheets("Combined")
        .Range("L2").FormulaR1C1 = "=COUNTIF(C[-2],""*Junior*"")"
        .Range("L3").FormulaR1C1 = "=COUNTIF(C[-2],""*Senior*"")"
        .Range("L4").FormulaR1C1 = "=COUNTIF(C[-2],""*Master*"")-COUNTIF(C[-2],""*Grand Master*"")"
        .Range("L5").FormulaR1C1 = "=COUNTIF(C[-2],""*Grand Master*"") - COUNTIF(C[-2],""*Great Grand Master*"")"
        .Range("L6").FormulaR1C1 = "=COUNTIF(C[-2],""*Great Grand Master*"")"
        .Range("K2").Formula = "=COUNTBLANK(RC[-1]:R[" & .Cells(.Rows.Count, "J").End(xlUp).Row - 2 & "]C[-1])"
    End With

    Sheets("Summary").Range("B16").Value = Sheets("Combined").Range("L2").Value
    Sheets("Summary").Range("B17").Value = Sheets("Combined").Range("L3").Value
    Sheets("Summary").Range("B18").Value = Sheets("Combined").Range("L4").Value
    Sheets("Summary").Range("B19").Value = Sheets("Combined").Range("L5").Value
    Sheets("Summary").Range("B20").Value = Sheets("Combined").Range("L6").Value
    Sheets("Summary").Range("B21").Value = Sheets("Combined").Range("K2").Value
End Sub

john korchok say:

To verify that the issue really is an object anchor, please open Word>Preferences>View and check Object Anchors in the Show in Document section. The anchor will look like this:

You can't delete an anchor. To remove it, you must select and delete the object that is anchored to the text. This object could be off the page, so you can't see it. But if the anchor is visible, the object is selected and pressing the delete key will delete the object and anchor.
