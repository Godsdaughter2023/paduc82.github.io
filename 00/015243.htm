Sub Test()
  Dim Source As Range, Dest As Range
  With Worksheets("Sheet1")
    'Find the last filled cell in column A
    Set Source = .Range("A" & Rows.Count).End(xlUp)
    'Build the range from A2 to this cell
    Set Source = .Range(.Range("A2"), Source)
  End With
  With Worksheets("Sheet2")
    'Find the next cell after the last filled cell in column M
    Set Dest = .Range("M" & Rows.Count).End(xlUp).Offest(1)
  End With
  Source.Copy Dest
End Sub

Try the following:

  • Quit Word. Make sure it is not running any more.
  • Start Word again.
  • Press Alt+F11 to activate the Visual Basic Editor.
  • Do you see Normal in the Project Explorer on the left?

If not, Normal.dotm has not been loaded automatically.

If yes, you should be able to see the macros stored in it.
