eileen gal say:

Hi,

An outdated network adapter driver may cause the WiFi adapter not working or no WiFi available issue in Windows 10.
In this case, you need to update your network adapter driver immediately.

Note: This process needs a stable Internet connection and it is recommended to use the Internet Explorer to access the website. (You may choose to download the driver using a USB on a different computer).

Steps to download updates that include drivers  from the Windows Update Catalog:

Step 1: Get the Hardware IDs.

  1. Type Device Manager in the Search box and press Enter.
  2. Click on Network adapters and double-click on the name of your Network Controller.
  3. On the Details tab, click on the drop-down arrow and select the Hardware Ids option. Copy the last value on the list.
  4. Once you have the ID number, please click on this link to access the Microsoft Catalog website.
  5. On the Microsoft Catalog website, paste the Hardware ID in the Search box. (Just paste the value from the word VEN up to the last number before the & sign. ex. VEN_8086&DEV_15B8.)
  6. Look for the latest update on the catalog and click on Download.

Step 2: Install the updates on your device.

  1. Type Device Manager in the Search box and press Enter.
  2. Click on Network adapters and right-click on the name of your Network Controller.
  3. Select Update driver then click on the Browse my computer for driver software option.
  4. Locate the file for the driver update then click Next.

Let us know how it goes after performing the suggested steps.

bill smithers say:

Hi,

You use file History with an external Hard Drive or USB, not the internal partitions, because if the Hard Drive fails you have lost everything.

And that is why "no useable drives" were found.

Before you start using File History to back up your files, you need to first select where your backups are saved. You can select an externally connected drive, such as a USB drive, or you can save to a drive on a network. There are other choices, but these two provide the best options to help protect your files against a crash or other PC problems. For more info about using File History

Once you get set up, you can follow this Microsoft Method for backing up your Data if you wish to:

"Set up a drive for File History"

http://windows.microsoft.com/en-us/windows-8/set-drive-file-history

"Introducing Windows 8.1: How to Backup Data with File History" (This is a great tutorial on it)

http://www.7tutorials.com/introducing-windows-8-how-backup-data-file-history

"Restore files or folders using File History"

http://windows.microsoft.com/en-us/windows-8/how-use-file-history

Cheers.
