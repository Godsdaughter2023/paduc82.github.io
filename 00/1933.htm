To state what Cyber_Defend_Team already mentioned a bit more clearly, MSE is specifically designed to integrate with the existing Windows firewall on all supported client platforms, since in at least Windows VISTA and 7 it operates over the same Windows Filtering Platform (WFP) that these firewalls were designed to use.
