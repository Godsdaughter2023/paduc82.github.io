What is the formula to add up text lines in a filter in Excel? - Microsoft Community

kimgreenwood say:

What is the formula to add up text lines in a filter in Excel?

Hi

I have a spreasheet that I have put filters in, the filters are columns of text.  Does anyone know what the formula is for adding up the text infomation is? 

I have lots of variants, hence the quickest method to add up in the text information would be to use a formula.

jeeped say:

The SUBTOTAL() can create the equivalent of a COUNTA() on filtered data. If your text values are in column C (and you have a header label in the first row) then use this to get the count of the text and number values values of the filtered data,

=SUBTOTAL(103, C2:C9999)

If you wanted to get the count of numbers in column C then,

=SUBTOTAL(102, C2:C9999)

So if you have both text and numbers in column C and only want the count of the text then this,

=SUBTOTAL(103, C2:C9999)-SUBTOTAL(102, C2:C9999)

Note that SUBTOTAL(103, ...) or COUNTA(...) will count zero length strings like those returned by a formula.

SUBTOTAL function Returns a subtotal in a list or database
COUNT function Counts how many numbers are in the list of arguments
COUNTA function Counts how many values are in the list of arguments
