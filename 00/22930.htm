a. user say:

Hi Karen,


I understand that you would like to send multiple emails to a specific set of people. I would gladly assist you with this.


This can be done by creating a group. Please follow the steps below to create a group.

  1. Sign in at Outlook.com.
  2. Click on the drop down arrow next to Outlook.com.
  3. Click on People. (This should display a list of all your contacts.)
  4. Check all the contacts you would like to send the email to.
  5. Click on Group and then on New Group.
  6. Assign a name of your choice to the new group.

  1. Sign in at outlook.com.
  2. Click on New.
  3. In the To field, type the Group name.
  4. Enter the message subject and body.
  5. This email will be received by all the member of the group.

Unfortunately, at the moment I have bad news for you.

There have been several reports of the Windows 10 upgrade from Windows 7 with Microsoft Security Essentials installed has left MSE on the machine and disabled Defender. The uninstall procedures for MSE do not appear to work.

In another thread, someone reported that they booted to safe mode and removed all registry keys related to Microsoft Security Essentials manually and disabled the startup service from services. stephen boots say:

Microsoft is aware of the problem and are investigating. Supposedly their data shows very, very few machines with this issue as Windows 10 should have recognized MSE as incompatible at installation and removed it automatically. Something apparently prevented that from happening in the cases we've seen here in the forum. I presume that a manual uninstall of MSE may also be problematic in some cases -- either before the Windows 10 upgrade or after a rollback. Note that the uninstall MSE Wiki article (Uninstalling Microsoft Security Essentials ) has been viewed many, many times since publication as it, like other antivirus software, can be balky about an uninstall via normal means.

That said, the latest advice I can offer based on other feedback here in the forum is that you can:

a. do a reset of Windows 10, which will remove all installed programs and data and will leave you with what will in essence be a clean Windows 10 installation with no programs installed.

b. roll back to Windows 7, remove MSE, perform the upgrade to Windows 10 a second time

c.  manually rip out MSE by editing the registry, stopping services while running in Safe Mode, and possibly removing leftover program files manually.

d. not really advice -- wait for Microsoft to provide a fix and risk that your PC is unprotected for a very long time while waiting for a fix that may never appear.

I think that the most palatable solution is b.

Protected enough is a relative term. :-)

Windows Defender on Windows 8 - Introduction and Frequently Asked Questions

No matter what you choose, the following recommendations (I wrote this for older versions of Windows, but most is still applicable today) will assist in protecting the PC from infection:

-Make sure that the Windows Firewall is enabled.

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

-Make sure Internet Explorer is updated with all patches.

-In Internet Explorer use the SmartScreen Filter.

-Make sure that IE Internet Security settings are at least set to medium-high (default).

-Enable the pop-up blocker in IE.

-Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.

-Never open unsolicited email attachments.

-Practice safe web browsing.
