jeeped say:

... For instance: CS12345 would equate to 031912345 since C is the 3 letter, 03 and S is the 19 letter ...


Use Left() and Mid() to parse the characters, Asc() to convert to an ASCII number and Format() to padd with leading zeroes if necessary.

SELECT Format(Asc(Left( [myField] ,1))-64,"00") & Format(Asc(Mid( [myField] ,2,1))-64,"00") & Format(Mid([myField],3,99),"00000")
FROM tblMyTable

This assumes upper case characters. Use UCase() if necessary to convert the parsed characters to upper case before determining the Asc() value. An upper-case A has an ASCII value of 65; e.g,:

... Format(Asc(Left(UCase[myField]),1))-64,"00") ...

sheeloo say:

Adapted from http://www.exceltip.com/st/TipofTheDay/1453.html/

 

To insert a formula field (Calculated Field) as a new column in a PivotTable:

1. Select a cell in the PivotTable report.
2. Select Insert -> Insert Calculated Field in the Cells Group on Home tab.
3. In the Insert Calculated Field dialog box, type the formula name in the Name box.
4. In the Fields list box, select the field name to insert into the Formula text box, and click Insert Field. The field name is copied into the Formula box.
5. Type the formual you want around the field name then click OK.

bill manville say:

If you want the creation of a new workbook to give you something different from normal - e.g. a different standard column width - then you create a Book.xlt or Book.xltx file and save it in your XLSTART folder.

So I suspect someone has done that but not done it correctly - e.g. it's an Excel 97-2003 workbook but it has a 2007+ file extension.

I would look in your XLSTART folders for the file and delete it if you don't need it, or try changing the extension to xlt if you do need it.

Your XLSTART folders will be:

- a sub-folder of the folder from which your Office applications are opened (e.g. C:\Program Files\Microsoft Office\Office12 )

- a sub-folder of your ApplicationData folder, e.g. C:\Users\<username>\AppData\Roaming\Microsoft\Excel\XLSTART

- a folder you have nominated in File > Options > Advanced > General: At startup open all files in...

>> "Cannnot create the folder. Cannot create or navigate to the folder. You cannot create or navigate to subfolders of your inbox when you have a non-empty Root Folder Path"
>>

That means the server wants you to set a root folder. Or you can log into the account in web access and create a folder - then try creating a folder in the same location in outlook.

To set a root folder, Go to File, Account Settings, open the account settings dialog. Double click on the account and click More Settings. Switch to the advanced tab and set a root folder. This should not remove any data from your mailbox but I had one person tell me it deleted folders, so I would try creating the folder online first, especially if you have anything in folders labeled 'this computer only'.
