rohn007 say:

The "registration number" is called a "Product Key" by MS.

MS calls "registration" "activation"

Before Office 2016/365 there was no way for users to de-activate a use of their license.  But there are ways to transfer installations to new computers.

To transfer an installation to the new computer the first thing you have to do is make sure it is no longer usable on the old machine. If the computer is still running, you must uninstall it. If the computer is dead, no longer running, that is also good enough.

Not all Office licenses allow you to transfer the installation to new computers.  So it all depends on exactly how you bought Office 2007.  If you bought it in the big-ugly-yellow-plastic CD holder box you most likely have a "retail" license.  This allows you to transfer installations to new computers and it also allows you to install on 2 or 3 computers at the same time (3 for Home and Student, and 2 for Home and Business or Professional). 

If you bought Office installed and activated at the factory with the computer, you have an "OEM" license, you cannot transfer it.

If you bought Office as a download, no installation disk, then it is most likely a "MLK" license which also cannot be transferred.

So ... assuming you have the retail version with the cd holder box you have everything you need to install on the new computer. Just put the installation disk in the CD drive and run setup.exe. When prompted, the Product Key is on a yellow and black sticker inside of the plastic installation disk holder box.

ashish mathur say:

Hi,

 

You are seeing the #VALUE! error because Feb 27-2012 is not being recognized by MS Excel as a date.   MS Excel does not have a date format of its own - it picks up the Date and Time format from the Windows machine on which it is installed.  Most probably, your Windows Date and Time is set to dd/mm/yyyy.  In this format, Feb 27-2012 is not a recognized date.

 

To test my assumption, rename the tab as 27 Feb-2012 and change your formula to

 

=1*MID(CELL("filename",A1),FIND("]",CELL("filename",A1))+1,256)

 

Format this cell as Date.

 

Does this work?
