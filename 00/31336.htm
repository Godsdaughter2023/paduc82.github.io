You can search in Windows 10 File explorer in exactly the same way as you did in Windows 7 Windows explorer.  The search box in File explorer has not changed its functionality at all.  If you are in, say, the My documents folder then using the search box confines the search to that folder tree.

The question of how to set that keyboard button to open up File explorer is not as straightforward. 

  • That keyboard button was created by your keyboard maker and it will have been supplied with a maker's utility that assigned it the function of opening Windows explorer. 
  • This maker's utility is not part of Windows.
  1. You need to ask the keyboard maker for a Windows 10-compatible version of the utility.
  2. Unless you can find the utility in your Start menu & it turns out to be intuitive to use, you also need to ask the keyboard maker for  instructions about using the utility so that you can assign that keyboard button to open File explorer [almost all these utilities were user customisable].  From your brief description, I think it has defaulted to opening Cortana instead.
  3. If the keyboard maker is no help then see if there are user forums for that brand and search them for advice about Windows 10 versions & customising the key's actions.

I would hazard the guess that the original keyboard utility is already sufficiently Windows 10 compatible.  So the only real difficulty you might have is in getting instructions for customising it but even this might be easy once you find the utility [the last time I had a keyboard with special buttons, its utility had a special section for picking the required action and it was immediately clear how to use it]. 

Best of luck,
