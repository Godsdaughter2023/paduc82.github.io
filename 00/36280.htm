    Hi Annezca,

     

    Welcome to Microsoft Answers.

    Dragging "Microsoft Office" file folders from Windows.old Program Files to OS (C:) Program Files wouldn’t restore the program. You can only restore personal files like Internet Explorer favorites, Desktop folder, My Documents folder and other users' files and folders from windows.old folder. If you need any help restoring these files, you can refer:

    How to restore your personal files after you perform a custom installation of Windows Vista or of Windows 7

    http://support.microsoft.com/kb/932912

     

    In your case, all you can do is reinstall Microsoft Office 2007 on your computer. You can download it from

    http://office.microsoft.com/en-us/downloads/default.aspx

    Uninstall the existing Office components from your computer before reinstalling.

    You can uninstall them from Control Panel.

    1)       Go to Start > Control Panel.

    2)       Click on Add or Remove Programs.

    3)       Highlight Microsoft Office 2007 and click on Remove button.

    4)       Follow the wizard to uninstall.

    To download, simply double click on the installation file and follow the onscreen instruction.
