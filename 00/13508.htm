john sr wilson say:

If you have a lot of pictures then this macro will set slides after the first to greyscale. Not worth the effort unless you have a lot of images though.

Sub GS_Pic()
   Dim L As Long
   Dim oshp As Shape
   For L = 2 To ActivePresentation.Slides.Count
      For Each oshp In ActivePresentation.Slides(L).Shapes
         If isPic(oshp) Then oshp.PictureFormat.ColorType = msoPictureGrayscale
      Next oshp
   Next L
End Sub

Function isPic(oshp As Shape) As Boolean
   If oshp.Type = msoPicture Then
      isPic = True
      Exit Function
   End If
   If oshp.Type = msoPlaceholder Then
      If oshp.PlaceholderFormat.ContainedType = msoPicture Then
         isPic = True
         Exit Function
      End If
   End If
End Function

Are you creating a Section Break at the beginning of the document? If so, that's what's causing the problem :-) The first page by definition is a recto page.

It sounds like you have a pretty good idea of what needs to be done, and you are using some "advanced" features, like macros. Good for you.

Sorry, I'm going to have to go into a few geek level points to help figure out your problem. But hopefully not too deep, or confusing.

Documents can only have one template linked to them at a time.
Documents can have access to some features stored in other templates if they are defined as "global" templates.

Where do you have this custom template saved. If it is in the same folder as Normal.dotm, it is just that a template separate from normal.dotm. If you start a document using that template, then the settings in it like fonts and styles are used and you have access to the macros. If you start a new document using normal.dotm it will use the settings in normal.

But if you have it in the "Startup" folder, then it becomes a "global template" which means those macros are shared with all documents you open.

***************

First thing to setup.
Go to File menu > Options command > Advanced option > Save section.
Make sure "Prompt before saving Normal Template" is turned ON. With this setting turned on, when you (or anyone else, ie installed addins, malware etc) make any changes to the normal.dotm you wlll be prompted before they are saved.

brian tillman [outlook mvp 2007-2019] say:

There is no "Safe Mail Subscriber" list. That list is called "Safe mailing lists".

If you log into http://mail.live.com/, the view you see will have a menu bar along the top. At the left end you'll see either "Outlook Mail" or "Outlook.com". For either platform, at the right end of the menu bar will be a gear icon. Click it and choose Options from the drop-down menu that appears. For Outlook Mail you'll see the options choices down the left hand side. For Outlook.com you'll see a page with the options choices.

To locate the "Exclusive" setting, click "Filters and reporting" in the Outlook Mail list on the left or in the right-hand column of the page for Outlook.com. At the top of the page that appears, you'll see two radio buttons. One says "Standard" and the other "Exclusive". Standard means that the junk filter will be able to catch some of the junk, but some will get through as well. What does get caught will, of course, appear in your Junk Email (for Outlook Mail) or Junk (for Outlook.com) folder. Exclusive means that all mail you receive that isn't from a safe sender or safe mailing list will be delivered to the Junk Email or Junk folder. Safe senders are defined to be your contacts and entries on the Safe Senders and Safe Mailing Lists lists.

So, this means that any messages from addresses that are not in your Contacts or on of of those two lists will be delivered to the Junk or Junk Email folder and not your Inbox. Once consequence of this is that periodically (at least once every ten days) you'll need to scan the senders and subject lines of the messages in the Junk or Junk Email folder looking for entries that you don't consider just and either add the sender to your contacts or the sender or sender's domain to your Safe Senders list in order to have future messages from that sender delivered to Inbox.

Keep in mind that if you have messages arriving in the Junk folder, and they really are junk, that's the correct behavior. Junk messages are supposed to arrive in the Junk folder. You can't keep junk messages from arriving. You can control where they get delivered and the Junk folder is the place for them.
