jeeped say:
It would be easier to have the selection move to A1 when you enter the worksheet rather than selecting A1 as you leave a worksheet. For this you need some code to run on the Workbook_SheetActivate event.

Open the VBE with Alt+F11 and immedaitely right click ThisWorkbook from the Project - VBAProject window in the upper left and select View Code. Paste the following into the new window on the right titled something like Book1 - ThisWorkbook (Code),

Private Sub Workbook_SheetActivate(ByVal Sh As Object)
    Sh.Range("A1").Select
End Sub

Press Alt+Q when you are done to return to Excel workbook. As you enter each worksheet the selected cell will move to A1. The problem with using the Workbook_SheetDeactivate event is that by the time it is triggered, it has already lost the active scope so attempting to manipulate Selection or Activecell is useless.

jllatham say:

This will take you to cell A1 on any sheet in the workbook when you choose (activate) a sheet.  This goes into the ThisWorkbook code module, and I assume since you've been working with macros for a while that you know how to get there.

Private Sub Workbook_SheetActivate(ByVal Sh As Object)
  Application.Goto Sh.Range("A1"), True
End Sub

If you don't want it to happen with every sheet in the workbook, then put this code into every sheet's code module that you do want it to happen with:

Private Sub Worksheet_Activate()
  Application.Goto Range("A1"), True
End Sub

I know you said you wanted it set up on Exit - but this has the same end effect: cell A1 on the sheet is selected the next time you look at the sheet.
