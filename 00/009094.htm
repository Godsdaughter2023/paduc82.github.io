dhookom say: You are definitely not the idiot and this should be simpler. This is how I have built Row Sources in the past. Your exact SQL would change based on object names.

SELECT phyID, phyLastName, IIf(DivID = Forms![YourForm]![YourDivID], " ", "NOT")
