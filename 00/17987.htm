This is a common problem that is easily solved with a lookup table, but can be translated to a macro.
 
If you were doing it with a table, you would set up the table as follows
 
Tiers            Base            Percent
 $-                -               0.60000%
 $25,000,000.00      $150,000.00         0.50000%
 $50,000,000.00      $275,000.00         0.42500%
 $250,000,000.00      $1,125,000.00         0.37500%
 
And then use this formula:
 
=VLOOKUP(B1,Tbl,2)+(B1-VLOOKUP(B1,Tbl,1))*VLOOKUP(B1,Tbl,3)
 
"Base" is calculated by the formula:
 
PrevPercent * (CurrentTier - PreviousTier) + PreviousBase.
 
To do this in a VBA macro, you could either have the table someplace on your worksheet, and use the WorksheetFunction.Vlookup method, or you could hard code arrays within the macro as below:
 
=================================
Option Explicit
Sub CalcFee()
    Dim Tiers, Base, Percent
    Dim Fee As Double
    Dim Gross As Double
    Dim i As Long
Tiers = VBA.Array(0, 25000000, 50000000, 250000000)
Base = VBA.Array(0, 150000, 275000, 1125000)
Percent = VBA.Array(0.006, 0.005, 0.00425, 0.00375)
 
Gross = Range("B1").Value
With WorksheetFunction
Fee = Base(-1 + .Match(Gross, Tiers)) + _
    Percent(-1 + .Match(Gross, Tiers)) * _
    (Gross - Tiers(-1 + .Match(Gross, Tiers)))
End With
 
Debug.Print Fee
 
End Sub
