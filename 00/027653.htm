john korchok say:

Because pattern fills are program-generated, you don't have as much control over them. I tend to stay away from them and use manual fills, as you mention in your second-last paragraph.

The placed graphic doesn't have to have a large file size. You can either create a grid out of drawing objects, which are very small. Or you can create a bitmap grid, then save it as an indexed-color file. A grid only needs 2 colors, one for the background and one for the lines. You'll find that a 2-color PNG is quite small even for a slide-filling grid.

Here's a good reference by Steve Rindsberg on image compression and how to prevent it or turn it off: Blurry PowerPoint Pictures

bob jones aka: cybertaz say:

As you've describe what you want it should be relatively simple - more complex options are available if it needs to go further.

In Outline View:

  1. Select the first statute name by clicking to the left of the small blue symbol
  2. Hold the Command key while you click to the left of each additional statute name
  3. Open the Outline Level list [between the green arrows] & select Level 1

You then can use the Show Level list to select Level 1, which will collapse the entire document to display only the Level 1 items.

To expand any one item, either:

  • click on that line & use the blue ➕, or
  • double-click the symbol @ the left of the statute name

Levels 2-9 (plus Body Text) can be applied to the subtopics within a Level if you wish.

Assigning the Levels also applies the corresponding Heading 1-9 Styles. That makes it possible to use the View> Sidebar - Navigation pane to navigate the document based on those same Levels when in either Outline or Page Layout View. In Page Layout you can collapse the listing in the Navigation pane but cannot collapse the document content, itself.
