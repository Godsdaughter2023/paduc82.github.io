Office Home and Student does not install through the MyAccount page. That is specifically for managing Office 365 licenses.

Office 2013 "new and improved" the license terms so that they only allowed install on one computer at a time. So you will have to uninstall 2013 H&S from your old computer.

To install on the new computer, you will need the 25 character product key and the installation file, which can be online downloaded.  The product key is the problem. If your "card" has a 25 character code on it, you are good, that should be the Product Key. But it is more likely that you have a 27 character "PIN". In that case you are hosed. The PIN is a one time use to retrieve the Product Key. Another "new and improved" change in 2013 is that you can no longer extract the full product key from the computer. So if you don't have a record of the product key, you are S.O.L.  Sorry.

If you have the product key, download and SAVE the installation file:

Download the install files for a one-time purchase of Office 2013 or Office 2016

john - project say:

kind03,

It's a pretty straightforward process for any version. I'd just create a blank file as a carrier for the information.

1. Open your file with the export/import maps, or if they are in your Global, you won't need to open your file

2. Go to the Organizer. I'm not sure exactly where it is on Project 2016 but it should be under File on the ribbon.

3. In the Organizer window select your file (or Global if the maps are there) as the "views available in" at the bottom on either side

4. Select the blank file as the "views available in" at the bottom on the other side

5. Select the "maps" tab

6. Select all the maps in the selection list for your file (or the Global)

7. Hit "Copy" and then "Close"

Now you can sent the blank file to your colleagues and they would use the same process as above to transfer the export/import maps to either their Global or specific files.
