hansv mvp say:

The user could create a macro in his/her Normal.dotm template:

 

Sub PrintCurrentPage()
    ActiveDocument.PrintOut Range:=wdPrintCurrentPage
End Sub

 

and assign this macro to a QAT button: File > Options > Quick Access Toolbar; select Macros from the "Choose commands from" dropdown.

charles kenyon say:
The way I did it back in Word 97 was to record a macro and play with it. That macro can then be attached to a button on the QAT or a ribbon.

The macro is:

Sub PrintCurrentPage()
'
' PrintCurrentPage Macro
' Macro recorded 05/26/00 by Charles Kyle Kenyon
'
    Application.PrintOut FileName:="", Range:=wdPrintCurrentPage, Item:= _
        wdPrintDocumentContent, Copies:=1, Pages:="", PageType:=wdPrintAllPages, _
        Collate:=True, Background:=True, PrintToFile:=False
End Sub

Macro works in Word 2010. It may contain unneeded parameters; I've never parsed it.
