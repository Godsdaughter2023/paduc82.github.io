2.      Right-click the connection, and then click Properties.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

3.      Click the Networking tab, and then click Configure.

4.      Click the Power Management tab, clear “Allow the computer to turn off this device to save power” check box, and then click OK.

Also try booting the computer in Safe Mode with networking and check if it is working, here are the steps:

1.     Restart your computer if it is powered on.
2.     Tap the F8 key after your computer initially powers on.
3.     Once you see the Advanced Boot Options menu you can stop tapping.
4.     Use the up/down arrow keys to highlight your selection.
5.     Select Safe Mode with Networking and press Enter.
6.     You should see drivers loading, and then Please wait.
7.     You should then be at the Welcome Screen.
8.     Logon to your computer using an account with Administrator privileges.

iphone 7 deleting server emails - Microsoft Community

marty steger say:

iphone 7 deleting server emails

Is there a set up feature either on the iphone or Exchange that will eliminate emails being erased from the server when iI delete them from the iphon.  I want to clear my phone, not my laptop!  Please help!

Thanks!

brian tillman [outlook mvp 2007-2019] say:
There are several account types that support email: POP, IMAP, and Exchange. The first one downloads messages only from the Inbox. There are no folders involved with POP. Moreover, POP cannot sync; mail flows one way: from the server to the client (sending is a different process). IMAP and Exchange do handle folders and mail can flow in both directions so that whatever you do on the client gets synched with the server and vice versa. IMAP is mail only and cannot handle calendar and contacts. Exchange is a full-service protocol and syncs calendar, contacts, and email. SO, the bottom line is, if you're using IMAP or Exchange, you have no choice with clearing the phone and keeping things on the server. That's the way those protocols are designed to work. The only way to do what you want is to change the phone's account type of POP, but you will lose any contacts/calendar synching as well as any folders other than Inbox.

Column Width and Row Height differs for Excel in Mac and PC - Microsoft Community

dannylzj say:

Column Width and Row Height differs for Excel in Mac and PC

I apologize if this question has been asked before.

I created a business quotation in Excel for Mac 2011 and adjusted all the columns and rows so that it fits into a page nicely. However, after saving this file and opening it in Excel 2007/2010 for PC, all the columns and rows were of different sizes and the formatting is totally off.

The same thing happens when I try to resize the columns and rows on Excel for PC then opening the file on Excel for Mac. How do I fix this problem?

I have users in the company using Excel on both Macs and PCs.
j.e. mcgimpsey say: Excel was never designed, marketed or sold as a page layout program - one can never be sure that one's layout will appear the same, whether cross-platform or from machine to machine (regardless of operating system). While you can do things that make it more likely that you'll get similar layouts, some are completely beyond your control.

A whole host of things affect how a workbook/worksheet is displayed and printed.

One of the two biggest factors is which default font the user has chosen. Column widths are set in "standard characters" (e.g., the width of a "0" (zero) in the Standard Font at the Standard size. Even though the displayed width may be in inches, cm, or pixels, under the hood it's the character width that determines the column width.

The other biggest factor is the Print Driver that the user has installed/chosen - the driver metrics are used both for printing and for on-screen display. Different print drivers will set different metrics for page size, line widths/heights, etc. Choosing a different printer on the same machine may make a perfect one-page layout blow out onto 2 or 4 pages.

Things that make it more likely that layouts will look similar among users:
  • choose the same standard fonts & widths as the creator of the workbook
  • stick to fonts that Microsoft shipped with Office 2011 (older versions, or versions produced by other manufacturers, even with the same name, may have different metrics).
  • choose common fonts in the worksheet (if the user doesn't have a particular font, XL will choose the "closest", which may not be close enough - and it won't tell you what substitution it made...)
  • Use the same printer/printer drivers
  • Keep your print drivers updated
  • Be careful using scaling: Page Setup/Page/Fit to x page(s) wide by y page(s) tall - different drivers can come up with wildly different settings.
