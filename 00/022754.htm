In Excel 2007 and 2010, you can filter by colour.  Select your data and apply an auto filter via Data tab > Filter.  Click the dropdown at the top of your column of interest and Filter by colour, selecting the colour of the cells you want to delete.  Select that column (or all your data columns) and press Alt+;  (that's Alt key and semi-colon key) to select only the visible cells.  You can now hit the delete key to clear their contents, or you can right-click and delete the entire rows (take care doing this - save a copy first!).

jllatham say:

Michelle,  lets try again together.  As before, make a copy of your workbook to test this all out in.  If it works fine, then you can just continue on using the copy and discarding the original (or better yet - archiving it away for a rainy day).

Start all over - choose all the cells that currently have conditional formatting and then select Conditional Formatting and "Clear Rules" - for selected cells.

Assume that column A contains the outdated list and column B contains the current list.

Starting at the top of column A (the outdated list), select all of the cells in that column.

Choose Conditional Formatting and then choose New Rule and then the "Use a formula to determine which cells to format" option.

Enter this formula, including the first = symbol, for the rule:

=COUNTIF(B:B,A2)=0

Set up the format - I selected a nice green color.

Click [OK]

That will shade/highlight the cells in the outdated column that do not have matches in the current data column.

Click in the header cell for your outdated column (for me that's A1) and just click the "Filter" Icon in the Sort & Filter group on the [Data] tab.

Click the dropdown in cell A1 (top cell of the Outdated column) and you will have options in it to sort or filter by color.

I chose to filter by my nice green color and all that was left displayed were the items that did not have matches in my "current data column". 

At this point you could copy the data displayed, or parts of it, and paste it into another sheet to work with, or just work with it right there in that sheet.

I do hope this helps and gets you to where you need to be.

diane poremsky mvp say:
Hi CVoice,
It depends on which Archive you are talking about. One just moves to a folder called Archive. This was added because people liked to use the Delete key to file mail - then wondered why it was deleted. Now you can use Backspace to file it.

Obviously, that Archive is no better than moving mail to other folders, except it has a one key shortcut - Backspace.

The other archive is now called 'clean up folders' or Autoarchive. This moves older mail to a pst file - and is most useful when the mailbox is "full" and near quota.

Is there a formula for when the text in one cell changes, excel - Microsoft Community

noellehitzelberger say:

Is there a formula for when the text in one cell changes, excel automatically inserts text into a different cell alerting the change to the previous cell?

I have a sheet full of data that many people access.  What I need is, if one person changes the information in one cell, I need the text "RECHECK STATUS" to appear in a separate cell.  That way, the next person who goes into the sheet will see that the information has changed and needs to perform a recheck.
hansv mvp say:

Use this version:

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim cel As Range
    If Not Intersect(Range("D:D"), Target) Is Nothing Then
        Application.EnableEvents = False
        For Each cel In Intersect(Range("D:D"), Target)
            cel.Offset(0, 1).Value = "RECHECK VALUE"
        Next cel
        Application.EnableEvents = True
    End If
End Sub

hansv mvp say:

You could set the font color for the entire column E red - as soon as RECHECK VALUE is entered in a cell, it'll automatically be red. But if you prefer to set it to red in the code:

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim cel As Range
    If Not Intersect(Range("D:D"), Target) Is Nothing Then
        Application.EnableEvents = False
        For Each cel In Intersect(Range("D:D"), Target)
            cel.Offset(0, 1).Value = "RECHECK VALUE"
            cel.Offset(0, 1).Font.Color = vbRed
        Next cel
        Application.EnableEvents = True
    End If
End Sub
