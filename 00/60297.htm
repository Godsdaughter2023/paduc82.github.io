4.      Click the Advanced tab, and then, under Virtual memory, click Change.

5.      Clear the automatically manage paging file size for all drives check box.

6.      Under Drive [Volume Label], click the drive that contains the paging file you want to change.

7.      Click Custom size, type a new size in megabytes in the Initial size (MB) or Maximum size (MB) box, click Set, and then click OK.

Optionally, you can adjust this setting manually by selecting Custom, entering numbers into the two fields under Custom, and then click the Set button:

a.      In the first field, the number should be twice the amount of system memory (in MBs).

b.     In the second field, the number should be four times the amount of system memory (in MBs).

Increases in size usually don't require a restart, but if you decrease the size, you'll need to restart your computer for the changes to take effect. We recommend that you don't disable or delete the paging file.

This suggestion is one that not a commonly accepted propostion.

"Maximum" size should be set to3 times physical memory (that is 2 times the "Initial" size).

The reality is that virtual memory is extra capacity much like the pressure valve on a steam engine boiler  that's there to prevent the boiler exploding if the pressure gets too high. You really do not want a system using over much virtual memory as using the pagefile is much slower than RAM. You add RAM to reduce usage of the pagefile and on the theory expressed above you should increase the size of the pagefile. This is a nonsense.

davem121 say:
Hi MemFizz

This is a very common occurrence on the Internet, that laptop is installed with the volume licence version of Windows 10 Pro, which was illegally activated with a KMS activator . . .

You have a couple of options:

Clean install the Home edition of Windows 10 Home by using bootable Windows 10 installation media, that may auto-activate, or you may be able to use the Windows 7 licence key.

Purchase a licence key for the edition of Windows 10, that is already installed on that laptop from the Microsoft Store . . .

To clean install Windows 10 Home:

Click this link: 
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10 

During the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC'

Make sure you install the same Edition of Windows 10 (Home) you previously had installed to avoid Activation issues
