bob jones aka: cybertaz say:

No version of Word for Mac prior to Office 2016 supports RTL script at all. It is being incrementally implemented in Word 2016 but still isn't complete.

Language is an attribute of the text in Office documents, similar to Bold, italic, font size, etc. You can select text & use Tools> Language to choose the preferred language attribute. However, that does not change the appearance of the text, invoke RTL support, or translate the text from one language to another. It merely determines which Proofing Tools are used to check Spelling & Grammar.

If you go to Tools> Language in Word you can select any of the Languages listed, then click the Default button to set that as the default Language for all Office programs. That only affects new documents you create from that point forward - it does nothing to change the Language used in existing documents.

stefan blom say:

The check marks in the list of languages at Review | Language | Set Proofing Language only indicate which languages have been installed. Most Office installations include proofing tools for more than one language.

What you should do is clear the check mark next to "Detect language automatically" in the Language dialog box, so that Word won't guess the language, and change it, as you type.

The checks just indicate that proofing tools are installed for those languages (each copy of Office comes with proofing tools for three languages; these vary depending on the region in which Office was purchased).
