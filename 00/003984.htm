joeu2004 say:

Arthur wrote:

I have populated two data sets into a scatter plot; the polynomial trendlines through each represents a supply curve, and I would like to add the values on the Y axis to see a combined supply curve.  The data points for each series do not have corresponding values on the X axis. It is unclear to me that what you want to do is numerically correct.  The result might be a case of GIGO -- "garbage in, garbage out".

I am particularly suspicious of your use of a polynomial trendline.  A lot of people (mis)use it because it can be made to fit a small set of data exactly.  But in fact, the interpolation does not represent reality, and that is almost certainly the case for any extrapolation.

In any case, it would help us help you if you provided more specifics.  For example, what degree of polynomial trendline are you using:  6?  And what are your x-axis values:  dates?

One way to generate Y values for a common set of X values is to use the LINEST function.

For example, if one set of data is in X1:X10 and Y1:Y10, you can try to determine the trendline coefficients of a 6-degree polynomial equation by selecting 7 horizontal cells (A1:G1) and array-entering the following formula (press ctrl+shift+Enter instead of just Enter):

=LINEST(Y1:Y10,X1:X10^{1,2,3,4,5,6})

Do something similar for the second set of data, with coefficients in A2:G2.

Then for some set of X values in starting A3, compute the corresponding Y values and their sum using the following normally-entered formulas (just press Enter as usual):

B3:  =SUMPRODUCT(A3^{6,5,4,3,2,1,0},$A$1:$G$1)
C3:  =SUMPRODUCT(A3^{6,5,4,3,2,1,0},$A$2:$G$2)
D3:  =B3+C3

Then you can create a chart with 3 data series.  The X-axis would be the range in column A starting with A3.  (But see the caveat below.)  The Y-axis would be the parallel range in columns B, C and D respectively.

Caveat:  The LINEST formula does not always work well with some sets of X values, particularly dates.  I believe the reason has to do with numerical limitations of Excel computations, not a defect per se.  It would be better if the X values were reasonably small ordinal numbers.  For example, instead of sequential dates, use sequential numbers starting with 1.

In that case, you can either have two different sets of X values, one for the LINEST function and one for the chart; or you can modify the LINEST expression accordingly.

This data represents the water supply for a fire sprinkler system; in spite of the lengthy explanation, the concept is fairly simple. The numbers in cells A2 and A4 are measured empirically; an engineer records the "static pressure" at a fire hydrant, then observes a pressure drop to a plateau "residual pressure" when a second fire hydrant on the system is flowed. The "pitot pressure" at the flowing hydrant is measured, then used to compute "flow", which is entered into cell A6.

 

The Hazen-Williams (H-W) formula is used to calculate available water flows:

QF=observed flow, hr is the drop in pressure from the static pressure to the desired residual baseline and hf is the drop in psi from static pressure to the actual residual pressure that was measured during the test.

 

Cells B11 to B14 list the static and residual pressures (psi), along with two other values used to increase the range of data points. Cells A11 to A14 list corresponding water flows according to H-W. Plotting flows along "X" and pressures along "Y" depicts a pressure loss curve. I've seen a "polynomial" trendline used on other worksheets, but, as you've said, maybe that's not appropriate for this function.

 

Sprinkler system demand, calculated based on required sprinkler flows and anticipated pressure losses, is displayed in cells D3 and E3. If this data point is located above the water supply curve, the water supply is insufficient and a fire booster pump is required.

 

Fire pump performance curves are available from manufacturers, and can be emulated in Excel by entering data points into E11-E14 and F11-F14. In theory, the sum of these curves (combining the pressures at common flows) results in an augmented water supply that can be evaluated against the system demand.

 

This worksheet has generated trendlines for the water supply and fire pump, and I assume there is a way to automatically extract the formulas, plot the data into rows S and T, and output combined values into row U, which could be used to plot the third curve.
