Windows Defender will auto-update itself ONLY if Windows Update (automatic updates) is enabled/turned-on.

Don't worry about the Defender icon in "quick launch overflow area." If something's wrong with Defender (e.g., Definitions haven't been updated in the past 7 days; Defender's been disabled somehow), the icon WILL appear!
