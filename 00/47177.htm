Sorry to hear you are going through this problem. Now when you do run into these System Error codes we are very limited to the troubleshooting we can go through. There is a support article specifically for these error codes titled "I need help with system updates on my Xbox One."

Going through the article you should be able to get to the specific E102 error code, It will have you Restore your console to factory defaults, If this does not work then you will have to go through with the Offline System Update. Unfortunately if still not working then the last solution to this would be a Repair.
