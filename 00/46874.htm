    frederik long say: You could try the VB Script below. It works on my machines but I do not know how reliable it is.
    P.S. Please include the links to your previous posts when referring to them

    '--------------------------------
    'Check the System Restore status.
    '19.9.2017 FNL
    '--------------------------------
    Set oWshShell = CreateObject("WScript.Shell")
    sKey = "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SystemRestore\"
    sActive = "RPSessionInterval"
    sDesc = "RestoreStatusDescription"

    If oWshShell.RegRead(sKey & sActive) = 1 Then
        On Error Resume Next
        sText = vbLf & "Description: " & oWshShell.RegRead(sKey & sDesc)
        If Err.Number <> 0 Then sText = ""
        On Error Goto 0
        sStatus = "active."
        iButton = 64
    Else 
        sText = ""
        sStatus = "not active."
        iButton = 48
    End If

    sMsg = "System Restore is " & sStatus & sText
    MsgBox sMsg, iButton, "System Restore Monitor"

    Thank you for the prompt response.

    If you're still unable to remove the email account, we advise resetting the Mail app through the Windows PowerShell. This will procedure will restore the Mail app to its default settings.

    1. Select the Start button.
    2. Type Windows PowerShell.
    3. Right-click Windows PowerShell and then select Run as administrator.
    4. If prompted, click Yes or enter the administrator password.
    5. Copy and paste the command below and then press Enter to execute it.
      • Get-AppxPackage Microsoft.windowscommunicationsapps | Remove-AppxPackage
    6. Wait for the process to finish.
    7. Restart your PC.

    You can reinstall the Mail app from the Windows Store.

    1. Launch Windows Store.
    2. Search for Mail and Calendar.
    3. Click the app from the result.
    4. Click the Install button.

    Let us know if you need further assistance.
