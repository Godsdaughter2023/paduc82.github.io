DMZ

  1. Login to the router by entering the default gateway IP address.

  3. From the menu on the left select Configuration.
  4. From the sub-menu select Virtual Server.
  5. Select Edit DMZ near the center of the page.
  6. Select enabled and enter the IP address of the xbox in the Internal IP Address line.
  7. Select Apply. 

 

Disable DMZ

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu on the left select Configuration.
  4. From the sub-menu select Virtual Server.
  5. Select Edit DMZ near the center of the page.
  6. Select disabled.
  7. Select Apply.

Hi usb001,

 

·        Do you see any other printer port mentioned?

 

Check to see if the following helps.

 

Step 1: Try unplugging the printer and restart the computer. Plug the printer back and check if it helps.

 

Step 2:  Follow these steps and check if it helps:

 

a.       Go to Control Panel. Select All Control Panel Items.

b.      Open Devices and Printers. At the top of the window, select Add a Printer. Select Add a Local Printer.

Note: The search may take some time.

h.      When the search is complete, go back to the manufacturer section and your printer should now appear there. Select it and click the Next Button.

i.        Your printer will appear in the printer name box, click Next again and the printer will be installed.

gokul t say:

Hi aggie63,

You may enable the default administrator account and check if you can change the settings in the group policy settings.

In Windows® 7, the built-in administrator account is disabled by default. In previous versions of Windows, an Administrator account was automatically created during Out-of-Box-Experience (OOBE) with a blank password.

To enable the default administrator account, follow the steps mentioned below:

1. Open the Microsoft Management Console (MMC) (To open MMC, click Start, type MMC and hit Enter) and select Local Users and Groups.
2. Right-click the Administrator account and select Properties.

The Administrator Properties window appears.

3. On the General tab, clear the Account is Disabled check box.
4. Close the MMC console.

Administrator access is now enabled.

Access the link below for more information:

Enable and Disable the Built-in Administrator Account
