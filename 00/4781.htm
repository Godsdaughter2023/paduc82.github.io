airforce9797 say:

Here's the solution I came up with:

1.    If Outlook2010 is opened, close Outlook2010.

2.    Locate the C:\Users\your-User-Profile\AppData\Roaming\Microsoft\Stationery folder.

3.    Delete the html file for your stationery.

4.    Delete the files folder for your stationery.

This will remove the stationery from the More Stationery dialog. However, it will still appear in the Email Message Using MRU list. The following steps will use RegEdit to clear the Email Message Using MRU. As always, when doing anything in the registry USE EXTREME CARE!!

1.    Open RegEdit.

2.    Locate the following:

HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows Messaging Subsystem\Profiles\\0a0d020000000000c000000000000046

3.    Delete the 101f035e value.

4.    Close RegEdit.
