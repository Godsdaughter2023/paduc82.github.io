doug robbins - mvp office apps & services (word) say:

The way Word seems to work is that if the direct formatting is applied to less than 50% of the text in a paragraph, the direct formatting is retained when a paragraph style is applied.  If the direct formatting is applied to more that 50%, then the direct formatting is removed when the style is applied.

However, you can use a macro containing the following code to find underlined text, apply the style to the paragraph in which it is located and then reapply the underlining to the text that was originally underlined.

Selection.HomeKey wdStory
Selection.Find.Font.Underline = wdUnderlineSingle
With Selection.Find
   Do While .Execute(FindText:="", Forward:=True, _
   MatchWildcards:=False, Wrap:=wdFindStop) = True
       With Selection
           .Paragraphs.Style = "Body Text"
           .Font.Underline = wdUnderlineSingle
