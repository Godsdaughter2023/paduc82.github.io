scaling in windows 10, and the room for improvements. - Microsoft Community

a. user say:

scaling in windows 10, and the room for improvements.

Windows 10 improves on the scaling functionality by offering scaling options for every single display. The sad part is, that the mouse pointer speed is not scaled proportional to the displays scale. Further there are problems with expanded displays if the dpi on the screens is different.

If you have two 24" displays for example, one with a resolution of 3840x2160 (screen 1) and 1920x1080 (screen 2). You would usually expand the desktop from screen 1 to include screen 2, setup a 200% scale for screen 1 and you should be perfectly fine. There are just two problems:

Problem 1 - mouse speed scaling
The mouse speed will not scale with the scaling of the image on screen 1, and there is no possible way in windows to select a mouse speed scale. The mouse will always move double the way on screen 2 as it would on screen 1.

Problem 2 - border resolution
This problem is more complicated, moving a window from screen 1 to screen 2 is only possible in the vertical middle of the monitor (1920px). The second symptom of this problem is that a window that is part wise on screen 1 and 2 will be scaled wrong. If the bigger part is on screen 1 it will be scaled and the part on screen 2 will be larger than desired. Otherwise, it will not be scaled and you will have a hard time using the UI on screen 1

I think users should have the possibility to select if the scaling is effecting the mouse speed and the border size or not.

<!--The following token places all object output inside the <BODY> 

tag.--> 

<!--AccessTemplate_Body--> 

</BODY> 

<BR><BR>

<A HREF = "<!--AccessTemplate_FirstPage­->">First</A> 

<A HREF = "<!--AccessTemplate_PreviousPage-->">Previous</A>

jassymom1 say: I figured out what I needed to do. When setting up my email, I has to go to advanced settings and setup my account as Microsoft Exchange Activesync account. Then:

  1. If there is a Domain field, leave it blank. If there is a Username field, enter your email address.

  2. Enter your password.

  3. For Server name enter s.outlook.com.

  4. Make sure the Use secure connection (SSL)checkbox is checked, then tap Next.

  5. Choose your account options and tap Next.

  6. To finish setup, tap Done.

    I have not had any problems since I did my setup this way.

caption numbering fault

Hello,

My document consists of:

Chapter I

  1.1

  1.2

Chapter II

  2.1

  2.2

I assigned headings to all the title. Everything works well, until I want insert caption to my tables and figures. The caption, shown Table I.1 or Table II.2. How can I change it to Table 1.1 or Table 2.2, without changing the Heading 1.

Thank you

doug robbins - mvp office apps & services (word) say:

To start with, how are you getting the 1.1, 1.2 or 2.1 and 2.2.  If you have the numbering assigned to the Heading 1 style as Roman Numerals, you would get

Chapter I

I.1

I.2

Chapter II

II.1

II.2

To achieve what you are after, you will need to format the Numbering assigned to the Heading 1 styles as 1, 2, 3. and format that Style as Hidden and then you could use the Heading 9 Style for the Chapter I, Chapter 2 etc.  When you start a new chapter, you will need to insert both the hidden Heading 1 and the Heading 9 paragraphs.   With the display and printing of Hidden text turned off, you will not see the Heading 1 styled paragraphs, but their numbering will be available for use in the captions.
