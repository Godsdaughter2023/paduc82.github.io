How to change TimestampMS to date and time - Microsoft Community

raymond dekock say:

How to change TimestampMS to date and time

I exported the chat history but there is a column called TimestampMS which shows a large number. How can I convert that to a readable date and time in Excel?
ruwim.b say:

The TimestampMS value is the timestamp value in the so called UNIX time format.

 

https://en.wikipedia.org/wiki/Unix_time

 

However, to make the life more difficult for Skype users, the value is saved not in seconds, but in milliseconds.

 

In order to convert this UNIX time to a readable date/time value, you should enter the following formula in an Excel cell:

 

=(((FX/60/1000)/60)/24)+DATE(1970,1,1)

 

This assumes that the TimestampMS value is in the F column, while X is the corresponding row number.

 

In any case, if you are trying to export the chat history from the classic Skype for Windows Desktop (the 7.41.0.101 version), it would be much easier to make use of the free SkypeLogView application.

 

https://www.nirsoft.net/utils/skype_log_view.html
