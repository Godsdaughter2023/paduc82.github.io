suzanne s. barnhill say:

There are two issues here:

  1. The paper sizes listed in the Page Setup dialog will never include custom sizes created in Word. The sizes listed come from the printer driver and will vary depending on the printer used. Some printers, however, allow you to set up custom sizes in the driver. Check your printer Properties to see if this is a possibility.
  2. Most printers cannot print "full bleed"--all the way to the edge of the paper. There is an irreducible unprintable area required for paper handling. Your printer may have a "borderless" setting that will permit this; this setting is more often found in printers designed primarily for photo printing (which would likely already include the 5" x 7" size) and sometimes requires a special paper carrier or feed. For printers that cannot print full bleed, your only recourse is to either confine your design area to the printer's printable area or print on larger paper and trim (which is what commercial printers do).

julie sheets say:
Hi PwoodSV,

I'm not sure I understand you completely.  What do you mean by: "If the % complete is 90% or greater of the current date"

 If you want the data as of the current date, I'd use the Status field.
----------------------------------------------
From Help:

Description The Status field indicates the current status of a task, specifying whether the task is Complete, On Schedule, Late, or a Future Task.

How Calculated If the task is 100 percent complete, then Microsoft Office Project sets the Status field to Complete.

If the task start date is greater than the status date, then the Status field contains Future Task.

If timephased cumulative percent complete is spread to at least the day before the status date, then the Status field contains On Schedule.

If the timephased cumulative percent complete does not reach midnight on the day before the status date, then the Status field contains Late.

Best Uses Add the Status field to a task view when you want to display, filter, or group by task status. Use the Status field in conjunction with the Status Indicator field to get a graphical indicator of task status.

Example You have a five-day task that starts on Monday and ends on Friday. Today is Thursday, which is also the status date. The task is 80 percent complete. The timephased Cumulative Percent Complete field shows a value on Thursday, therefore the task is on schedule, and On Schedule shows in the Status field.

If the task were only 40 percent complete, the timephased Cumulative Percent Complete would show a value through Tuesday. The task would be considered late, and Late would be displayed in the Status field.

---------------------------------------------------

If you simply want to return Red, Yellow, or green based upon % complete there is  no complicated formula needed - just a simple reference and the graphic indicators.

I'm not sure whether you are trying to test % work complete or % complete - just adjust the fifth step below as needed.
  1. Add a custom number field to the Entry table.
  2. Right click on the number field and choose Custom Fields to open the Custom Fields window.
  3. Rename the field if you like.
  4. Under custom attributes, select "Formula" to open the Formula for Number(x) window.
  5. Enter [% Complete] if you wish to use % Complete or [% Work Complete] as you wish.
  6. Click OK to close the Formula for Number(x) window and return to the Custom Fields window.
  7. Under Values to displace select Graphical Indicators to open the Graphical Indicators for Number(x) window.
  8. Enter the following:
First row:
Test: Is greater than or equal to:
Value: 90
Image: Green box

Second row:
Test: Is greater than or equal to:
Value: 75
Image: Yellow

Third row
Test: Is less than
Value: 75
Image: Red box
