rick rothstein, mvp - excel say:
Chr(10) or vbLf (if you want to use the built-in constant) is a Line Feed character. Chr(13) or vbCr (if you want to use the built-in constant) is a Carriage Return character. Just so you know, many Windows programs use a combination of Chr(13) & Chr(10) or vbCrLf, which is a predefined character for that combination, for the "newline" character code sequence... Mac's use Carriage Returns by themselves and UNIX/Linux systems use Line Feeds by themselves as the "newline" characters. In Excel, using Alt+Enter produces a Line Feed character, but depending on how, or from where, you receive your text, you could have either or both separating your lines of code from each other, so two separate Replace method calls (one for each character) should solve your problem. I say should because any non-printable character will display a "white box" in their place... are you sure you have a Line Feed and/or Carriage Return character?

Let's see the ASCII codes for any and all non-printable characters in a cell. Go into a Module in the VB editor and copy/paste the code below into the code window, then change my example cell address of E13 to a cell that is giving you trouble. Now run the code directly from the Module and it will print out the ASCII code for any and all non-printable characters in the text within the cell you desinate. Try this with a few of your troublesome cells and compile a list of of ASCII codes in the Immediate window. These are the characters that you have to replace (use the Chr function with the ASCII code as the argument, on Chr call per ASCII code) with the empty string.

Sub ShowNonPrintingCharacterCodes()
  Dim X As Long, Text As String, Char As String, Uniques As String
  Text = Range("E13").Value
  For X = 1 To Len(Text)
    Char = Mid(Text, X, 1)
    If Asc(Char) < 32 Or Asc(Char) > 127 Then
      If InStr(Uniques, Char) = 0 Then
        Uniques = Uniques & Char
        Debug.Print Asc(Char),
      End If
    End If
  Next
End Sub

    Restore contacts

    You can restore your contact list to the state it was in at any point within the past 30 days. This is helpful if you want to do any of the following:

    • Recover deleted contacts
    • Restore contacts after an unsuccessful sync
    • Undo a recent import
    • Undo a recent merge
    Restore your contacts to a previous version
    1. Open Gmail.
    2. Click Gmail in the top left, then choose Contacts.
    3. Click More above your contacts list.
    4. Click Restore contacts....
    5. Choose the time you'd like to revert your contacts list to (e.g. 10 minutes ago, one hour ago, one week ago, etc).

    The 24 hour rule for Chat contacts

    When you delete a contact in Contacts, it will also get deleted from your Chat list. After the contact is deleted, Chat still remembers the invitation status for that contact for another 24 hours. So if you restore a deleted contact within 24 hours, it will be fully restored in Chat. But if you restore a deleted contact after 24 hours, you'll need to re-invite that person to your Chat list.
