ronnie vernon say:

Hi tedz10f

 

This is a typical TDR (Timeot Detection and Recovery) error. The TDR component in Windows was added to prevent damage to the graphics hardware or driver.

 

Before TDR, the system would crash and corruption of the display driver or damage to the hardware was common.

 

Basically a TDR error simply means that the graphics device is being pushed beyond it's available resources.


 

I would recommend that you return the graphics device back to the default setting and test the results.

 

 

You might also want to post this question on the MSI GPU forums and ask if anyone else is experiencing similar problems.

 

MSI Video Cards Forum:
http://forum-en.msi.com/index.php?board=9.0

 

 

Here is a link to more information.

 

TDR changes in Windows 8 (Windows Drivers) Advanced:
http://msdn.microsoft.com/en-us/library/windows/hardware/jj676805(v=vs.85).aspx

 

 

Let us know what you find.
