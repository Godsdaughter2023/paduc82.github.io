Through the magic of a desktop.ini file contained within the folders. Among other things, this file can set custom icons and a display name different than the actual name of the folder. This allows foreign language versions of Windows to display folder names in the user's native language while the OS uses its 'native' English language. This is apparent when you compare the display names of the public folders in Explorer with the names revealed by the 'Dir' command in a Command Prompt:

 

Desktop.inii files have their hidden & system attributes set by default, so to see them, you have to set Explorer to both Show Hidden Files and not Hide Protected Operating System Files:

The files are ordinary text files and can be viewed in Notepad.
