Copy guides in Powerpoint - Microsoft Community

natemulliken say:

Copy guides in Powerpoint

Is it possible to copy guides from an existing slide to a slide master so they can appear on all slides?
steve rindsberg say:

It is, at least with a wee bit of VBA, but only in PPT 2013.  This isn't supported in earlier versions.


Sub MoveGuidesToMaster()
' Run this to copy all of the current presentation guides
' to the slide master
    
    Dim x As Long
    
    With ActivePresentation
        For x = 1 To .Guides.Count
            .SlideMaster.Guides.Add .Guides(x).Orientation, .Guides(x).Position
        Next
    End With
    
End Sub
Sub DeletePresentationGuides()
' Run this to delete the guides in the presentation
' (but not those on master)

    Dim x As Long
    
    With ActivePresentation
        For x = .Guides.Count To 1 Step -1
            .Guides(x).Delete
        Next
    End With
    
End Sub

pologirl12usc say:

Why does EOMONTH for September give the 31st?

Using the fomrula =EOMONTH(MONTH(DATEVALUE(X$5&" 1, "&$C6)),0) where X5 is Sep and C6 is 2010 as part of a larger formula. Alone this gives the 31st. There are only 30 days in September and thus it throws it off the bigger formula.
shane devenshire say:

EOMONTH's first argument is a date.  Month is not a date is it is a number like 1 to 12. 

So use

=DAY(EOMONTH(DATEVALUE((X$5&" 1, "&$C6),0))

to return the day of the end of the month or

=EOMONTH(DATEVALUE((X$5&" 1, "&$C6),0)

To return Sept 30, 2010
