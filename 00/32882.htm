Type disk management in the search bar and hit enter, wait about 20 seconds or so for it to load your drive information. Now maximise it or drag the corner of the box to make it big enough to see. Disk 0, the top one should be the one we are looking for. This disk should contain the 100mb partition (don't touch it, I will explain at the end of the post), and then it should contain the 443GB (note, that it is GB) and then it will probably contain unused space with the remainder of your available disk space. If this is so then put the mouse pointer over the unused disk space portion of the graph bar and right click, select create partiton, you will be asked how large you want it, I advise you just use the default (all remaining space). Now the remaining graph should reflect that it is an unformated partition, right click on it, select format and I advise that you format it NTFS and insure it has a drive letter assigned.

 

 

 

The 100 MB partition contains all of your boot files and other files that must remain uncompressed if you ever decide to use bitlocker. Bitlocker on the boot drive is overkill in my opinion for the average user, I do use it on one of my machines on a second hardrive. Note, your motherboard must support TPM in order to use Bitlocker, something to read up on on rainy day but be careful, you can lose data if you mess it up.

With an Explorer window open that contains these files,

  • Press and release the "Alt" key.  This will bring up an extra menu bar at the top.
  • Select   View -> Details
  • Click on the "Size"  header at the top of the list.  This will sort files by size.
  • Single-left-click on the first file in the list to select it.
  • Using only the "elevator" scroll at the right of the window, scroll down until the last file that you want in the first group is shown.
  • Hold down the Shift key and left-click on this last file.  This should select this one and all above it
  • Drag these selected files to another folder.
