great white north say:

Hi johncarpino:

It is normal to see an "access denied" message when you try to open the CBS.log file from C:\Windows\Logs\CBS\.  This file is protected because your Windows operating system needs to be able to open and write information to this CBS.log file throughout the day.  You can copy and paste the CBS.log file to another location like your Windows desktop and then open it with a text editor like Notepad but this is a very large file (often > 30 MB) and you would have problems scrolling through this file and finding information related to System File Checker (SFC).

Any entry in the CBS.log file related to your SFC scan log is tagged with the label "[SR]".  See the MS support article Use the System File Checker Tool to Repair Missing or Corrupted System Files and scroll down to the section titled "More Information: How to View Details of the System File Checker Process" for instructions on how to run the findstr command in an elevated command prompt to extract your SCF scan log from CBS.log. The command findstr /c:"[SR]" %windir%\Logs\CBS\CBS.log >"%userprofile%\Desktop\sfcdetails.txt" suggested in that support article will extract your recent SFC scan logs and save them to a text file on your desktop called sfcdetails.txt.

To open an elevated command prompt in Vista go to Start | All Programs | Accessories; right click Command Prompt and choose Run As Administrator from the pop-up menu.

I also posted instructions in post # 53 of copimans' VistaForums thread Updates on how to copy and paste long, complex commands like findstr /c:"[SR]" %windir%\Logs\CBS\CBS.log >"%userprofile%\Desktop\sfcdetails.txt" into an elevated command prompt so you don't have to re-type the entire command.
