graham mayor say:

As you have not explained what you mean by 'stuff' it is difficult to ascertain what it is that you want to use Mail Envelope for, but you can certainly add attachments and CCs and override entries that users have made in the mail envelope e.g. as follows.

Sub Macro1()
'It is advisable to ensure that Outlook is running or Mail Envelope is likely to fail
'Use the code from http://www.rondebruin.nl/win/s1/outlook/openclose.htm to ensure that Outlook is running
Dim objRecip As Object
    With Application.ActiveDocument.MailEnvelope
        'Hide the mail envelope
        ActiveWindow.EnvelopeVisible = False
        'Replace any Introduction text with new text (or nothing)
        .INTRODUCTION = "Please read this and send me your comments."
        With .Item
            'clear existing recipients
            For Each objRecip In .Recipients
                objRecip.Delete
            Next objRecip
            'clear existing CC entries
            For Each objRecip In .Recipients
                If objRecip.Type = 2 Then objRecip.Delete
            Next objRecip
            'Add main recipient
            Set objRecip = .Recipients.Add("*** Email address is removed for privacy ***")
            objRecip.Resolve
            'Add CC
            Set objRecip = .Recipients.Add("*** Email address is removed for privacy ***")
            objRecip.Resolve
            If objRecip.Resolved Then
                objRecip.Type = 2
            End If
            'Add an attachment
            .Attachments.Add "C:\Path\Forums\Bookmarks.docx"
            'Replace the subject
            .Subject = "Here is the document."
            'send the message to the outbox
            .Send
        End With
    End With
    MsgBox "Document sent to Outbox"
    Set objRecip = Nothing
End Sub

Export the registry key from another computer

1.   ClickStart, and then click Run.

2.   In theOpen box, type regedit, and then click OK.

3.   Locate the following registry key: HKEY_Local_Machine\Software\Classes\htmlfile\shell\open\command

6.   Note the location of where the file will be saved. The registry file can be saved directly to the hard drive or floppy disk.

7.   Type a unique file name, and then clickSave.

8.   Quit the Registry Editor.

Import the registry key

1.   Copy the Exported registry key to the desktop on the problem computer.

2.   Double-click the.reg file.

You receive the following message:

Are you sure you want to add the information in drive: location file was copied to\file name.reg to the registry?

3.   ClickYes, and then click OK.
