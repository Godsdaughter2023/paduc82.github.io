Hi Ell231,

 

We cannot prevent the specific word from the auto-capitalization at the beginning of the sentence or after a full stop unless we turn off the auto-capitalize.

 

But as a workaround we can prevent it by initially typing the first letter as capital letter.

 

This can be done by the AutoCorrect in the Tools menu.

Hi There,

Let me assist you with the issue.

The Sent Items folder of the non-default email account, cannot be set as default without setting that data file as default.

However, while composing a new email, you can click the respective Data file and then click New Message in order to use that email account to send emails by default.

Note: Which ever data file is selected before composing the New email, that email address will be used to send the email by default.

Sent Items will not have the respective emails sent using that account by default unless that data file is set as default.
