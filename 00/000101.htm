greg maxey say:

I don't think there is any way to alter the wavy line.  If you are just wanting something that will highlight existing spelling errors before printing, saving, etc., you could run a VBA procedure:

 

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oRNg As Word.Range
Dim oErr As Word.Range
Set oRNg = ActiveDocument.Range
oRNg.Font.Animation = wdAnimationNone
For Each oErr In ActiveDocument.Range.SpellingErrors
  oErr.Font.Animation = wdAnimationBlinkingBackground
Next
End Sub

After fixing the errors, then just run the procedure again to ensure the animation is killed.
