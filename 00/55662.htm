john_barnett say:

Hi Serdar Baykara,

Right click on your desktop and, from the menu that appears, click on Personalize. In the Personalize window click the Display Settings link (towards the bottom of the window). When the display settings window opens move the slider to increase or decrease the screen resolution. Make sure you don't increase the display resolution more than your monitor will support otherwise you may have problems. The screen resolution will change and a box will appear asking you if the resolutin is OK. This box has a timeout. If the resolution is OK then click the OK button. If it isn't then click No and try another screen resolution.

cbarnhorst say:

Partitioning was common in Dos days because Dos was not optimized for partitions larger than 32GB.  Partitioning with NTFS, which Windows 10 must use, is unnecessary, even wasteful.  Make one 128GB partition on your SSD.  Period.  Full stop.  

One thing you can do to better utilize the SSD is put your paging file on the HDD.  I do that with SSDs of any size anyway for write considerations.  

In any case, the file system you are using determines any partitioning scheme you should use.  You are using NTFS.  NTFS requires no special partitioning scheme except the one Windows Setup will create for Windows 10 at the time of installation.
