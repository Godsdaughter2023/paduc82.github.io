jane lla say:

Hello Sharron,

The Function Discovery providers programming interface allows a component's discovery provider to be integrated into the Function Discovery system. Clients of Function Discovery normally use the IFunctionDiscovery interface, which delegates queries to the installed provider objects.

Developers use this interface; it is a system file of Windows. Independent Hardware Vendors (IHV) should implement a discovery provider that will enable their devices to be integrated into the Function Discovery system.

You may read about it through our MSDN articles for more information:

Function Discovery Providers

Built-in Providers

About Function Discovery Providers

If you're getting an error message regarding Device Display Object Function Discovery Provider. I recommend running the System File Checker tool to sort any system file corruption.

Also, make sure all drivers are all up to date. Best to manually download the latest driver from the hardware vendor’s website to make sure the one installed on your PC is updated. Outdated or incompatible drivers could cause issues such as this one.
