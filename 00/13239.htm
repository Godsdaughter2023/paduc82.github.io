john sr wilson say:
I think as you say each trigger needs to also close everything. This WILL take a lot of manual programming but I guess you could write a macro that took references to the trigger and all selected shapes and created the animation.

This is just the outline of a possible macro. It creates disappears for all but the first selected item which is used as the trigger. May need some work.

Sub create_animation()
Dim i As Integer
Dim oshp As Shape
Dim oshpR As ShapeRange
Dim oeff As Effect
Dim osld As Slide
Set oshpR = ActiveWindow.Selection.ShapeRange
Set osld = ActiveWindow.Selection.SlideRange(1)
Set oshp = oshpR(1)
For i = 2 To oshpR.Count
Set oeff = osld.TimeLine.InteractiveSequences.Add(1) _
.AddEffect(oshpR(i), msoAnimEffectAppear, , msoAnimTriggerOnShapeClick)
oeff.Timing.TriggerShape = oshp
oeff.Exit = True
oeff.Timing.TriggerType = msoAnimTriggerWithPrevious
Next
End Sub

EMBED is an old Excel 4.0 macro function whose only use is to indicate what type of object is embedded. Its syntax is

 

=EMBED(object_class, item)

where object_class is a string that specifies the type of object as it is listed in HKEY_CLASSES_ROOT in the Windows registry, and item specifies the area selected to copy; for controls it is always "".
