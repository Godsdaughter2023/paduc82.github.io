jay freedman say:

Because you've marked each section to restart page numbering a 1, the first page of each section really is "page 1" and that's what the TOC is showing. There is no provision in Word to show the absolute page number in the TOC and the section-relative page number in the text.

What you might do instead is to restore the "continuous" page numbers to the text, but use formula fields in the headers that subtract the number of the last page of the preceding section from the current page number. That would involve placing a bookmark on the last page of each section, so you can then use a PAGEREF field in the following section.

For example, place a bookmark named "sec2last" on the last page of section 2 (Amendment Record). Then in section 3, wherever you have a PAGE field, change it to

{ = {PAGE} - {PAGEREF sec2last} }

Similarly, put the bookmark "sec3last" on the last page of section 3, and use it in the formula fields in section 4, etc.

Unless the picture has the same aspect ratio as the page, you will not be able to do this without distortion. With the box for "Lock aspect ratio" checked, change either the height or the width (whichever makes the other dimension larger than the page size), then crop the larger dimension down to the page size.

If you drag a picture by a corner, you can change both dimensions at once, then crop as needed.
