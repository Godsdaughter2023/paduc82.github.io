roady [mvp] say:

Note that you can already create a rule based on properties of a currently selected email by right clicking the message and choosing: Rules-> Create Rule...

 

While this doesn't really match your "smart rules" concept is comes close and also prevents you from needing to create a rule from scratch.

 

Also note that it is really not recommended to create lots and lots of rules. Instead, consider using additional sorting methods such as Search Folders, Custom Views and Instant Search.

 

The downside of having lots of rules (especially when being created automatically) is that mails could end up in folders you did not expect when the rules have not been configured or sorted correctly. Also, as most users do not clean up "old rules", automatically creating rules would increase this issue and also makes mail reception slower as all rules will be evaluated when a message is being received.

 

Also see: http://www.msoutlook.info/question/672

Aligning rows of data with other rows based on certain cells matching - Microsoft Community

dimitir say:

Aligning rows of data with other rows based on certain cells matching

I'm trying to align rows of data with other rows based on certain cells matching.

e.g. columns A-D have data. Columns F and G also have data. The data in A should match the data in F (but this is not always the case). I'd like to rearrange the data in F and G to align with the data in A-D based on A and F matching.

Thanks in advance!

hansv mvp say:

Here is a macro you can run:

Sub Rearrange()
    Dim r As Long
    Dim m As Long
    Dim s As Long
    Dim n As Long
    Dim rng As Range
    Dim cel As Range
    Application.ScreenUpdating = False
    Range("H1:I1").EntireColumn.Insert
    Range("F1:G1").Copy Destination:=Range("H1")
    Range("F1:G1").Copy
    Range("H1:I1").PasteSpecial Paste:=xlPasteColumnWidths
    m = Range("A" & Rows.Count).End(xlUp).Row
    Set rng = Range("A2:A" & m)
    n = Range("F" & Rows.Count).End(xlUp).Row
    For s = 2 To n
        Set cel = rng.Find(What:=Range("F" & s).Value, LookAt:=xlWhole, MatchCase:=False)
        If cel Is Nothing Then
            m = m + 1
            r = m
        Else
            r = cel.Row
        End If
        Range("F" & s).Resize(1, 2).Copy Destination:=Range("H" & r)
    Next s
    Range("F1:G1").EntireColumn.Delete
    Application.ScreenUpdating = True
End Sub
