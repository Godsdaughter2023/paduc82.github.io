 What are other possible causes?

If there are no addins in play and no other apps in the background that involve Outlook data then it depends what type of email accounts are involved (i.e. POP, Imap, Outlook.com etc) and what type of data stores are in use (.pst or .ost etc)

For .pst files - run ScanPST to ensure that there are no underlying issues with the file. This is also presuming that any PST file is Unicode based file (right-click on the Root of the folder tree --> Data FIle Properties --> Advanced --> if the description in the Format field contains (97-2002) then it's an ANSI based file and should be converted. For this reply going on the basis that this isn't the case.

For .ost files - might be useful to recreate the files be deleting (or renaming) it at which point Outlook will recreate the .ost file automatically. Just be sure that you have any data backed up that is stored in folders labeled with "This Computer Only" or you'll lose it.

Change the Weather location in Outlook.com

I've seen this question a ton with lots of "solutions" that don't end up solving it.

  1. I switched to Classic mode of the Outlook.com Calendar site.
  2. Click on the Weather icon next to the date on the calendar
  3. Edit Settings....
  4. Add your desired location first
  5. Then delete the incorrect, first location that was listed by default
  6. Save

Go back to Outlook Beta and all was well.


Hope it works for you too!

marypoppers say: I don't know if this works for Office 365. I am using Office 2013 Home and Student on Windows 7. Office 365 may not have an option to "install from disc." However, if it does, this solution should work for you, too.

The workaround I used was to download the "burn to disc" version of office, and either burn it to a disc, or use daemon tools if you don't have a blank CD lying around. Note that you need your product key. I found mine in the original email they sent to me.

SOLUTION DETAILS
I had this problem for a while and non of the troubleshooting worked. To fix it, I went to the office account site (https://office.microsoft.com/MyAccount.aspx). You should see something like this: "Get Started: To get started with Office choose Install..." Instead of clicking on the big green install button, I clicked on the small "Install from disc" located beneath "Account options." Then click on "I want to burn a disc." You will then download a multi-gb .img file. You have to burn this file to a disc to use it, or you can use Daemon Tools Lite (search Google for Daemon Tools). If you haven't heard of it, basically, it loads the file as if it were already burned to a disc. It is a legitimate, free program  (not malware or anything like that).

Using Daemon Tools instead of burning to a disc -->
  1. Change the extension of the file you just downloaded from .img to .iso. For example, I changed mine from "HomeStudentRetail.img" to "HomeStudentRetail.iso"
  2. Download and install daemon tools lite (search google)
  3. Open Daemon tools and click the "Add Image" button. (It looks like a CD on a piece of paper with a green plus next to it). Browse through the file selection windows until you find your "HomeStudentRetail.iso" file. Select this and click OK.
  4. A listing just appeared in the image catalog in Daemon Tools. It should have a red MS Office logo next to it. Double-click the new listing in Daemon Tools. Select OK. A message should appear saying "Mounting image."
  5. A small dialogue box should appear that gives you the option of running Setup.exe. Single-click it.
  6. The installation should start. Strangely, I got the dreaded error message again, "Something went wrong..." However, installation started up anyway behind the error message. I just closed the error message and it was fine. I hope this works for you too.
  7. Use your product key to complete installation. The only place I know to find it is in the original email they send to you.
  8. Note that after installation, you can delete the .iso file. To delete it, you first have to unmount it. Go to windows explorer, click on "computer" in the navigation bar, then look for the mounted image. It should be in the category with any USB flash drives or inserted disks, and it is probably about 2Gb big. Right-click it and select, "eject"

Hope this helps!
