How do I write a formula to remove dashes, letters and numbers from - Microsoft Community

jackierae17 say:

How do I write a formula to remove dashes, letters and numbers from about 1500 cells?

I need to make the following changes "P01-02-C-1" should become "P102C1"

"V1-25-F-2E" should become "125F2E"

 

Please help.  Thanks!

jeeped say:

I would suppose that you wpould start with the SUBSTITUTE function to strip all hyphens, as in:

=SUBSTITUTE(A1,"-","")

... but that leaves you with the following values:

P0102C1
V125F2E

... and I do not see any one set of rule(s) that can govern the text replacement since you want to strip prefixing V's, but leave prefixing P's. The same holds true for numeric values in the second placeholder in other configurations.

If, however, all V's should be deleted and all Pn's should be converted to P's then:

=SUBSTITUTE(UPPER(SUBSTITUTE(UPPER(SUBSTITUTE(A1,"-","")),"V","")),"P0","P")

If you had other specific replacements to be made, they could be stacked into that formula as well. Using the SUBSTITUTE function efficiently really just comes done to a set of logical rules that can be adhered to.
