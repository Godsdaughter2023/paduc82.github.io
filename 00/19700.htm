joeu2004 say: Dean wrote:

The report that I'm needing to work with is laid out in Col (A) like an Outline, the main item is not indented, there are items under that, that are indented by 1 position, then some of those items have children that are indented by 2 positions.  I'm hoping that I can somehow do an IF statement that tests for Indent, for example -

IF((A3)=Indent(0),"A",IF(A3)=Indent(1),"B",IF(A3)=Indent(2),"C","")


Write:

=IF(indent(A3,0),"A",IF(indent(A3,1),"B",IF(indent(A3,2),"C","")))

where indent() is the following user-defined function:

Function indent(r As Range, n As Long) As Boolean
If r.Count = 1 Then
    indent = (r.IndentLevel = n)
Else
    For Each c In r
       If c.IndentLevel <> n Then Exit Function
    Next
    indent = True
End If
End Function

To enter the UDF, in Excel, press alt+F11 to open the VBA window.  In VBA, click on Insert, then Module to open the VBE pane.  Copy the text above and paste into the VBE pane.  You can now close the VBA window.

Note:  The function does more than you require.  For example, indent(A1:A10,1) returns TRUE only if all of A1:A10 are indented by 1.

jay freedman say:
You're correct, the Navigation pane was first introduced in Word 2010. In Word 2007, at the far right end of the Home tab of the ribbon, click Find. Enter the word or phrase you're looking for. Then you can:

  • Click the Find Next button multiple times to go from one occurrence to the next.
  • Click the Find In button to search specific parts of the document -- the main document, headers & footers, or the selected text (if any is selected).
  • Click the Reading Highlight to apply highlighting to all occurrences.


What you won't get is Word 2010's display of a column of occurrences, in context, that you can click to jump to a particular one. If you need something like that, download finder.zip from my site and extract the finder.dot file from it into the folder %appdata%\Microsoft\Word\Startup. Use the shortcut Ctrl+Alt+F or open the Macros dialog (Alt+F8) and run the macro named Finder.
