We might think that one possibility is:  one or more cells contain text, not numbers.  Note that the data type (text or number) is not determined by the cell format.  We can put text into a cell formatted as Number; and we can have a number in a cell formatted as Text.

john korchok say:
that all component documents are in a single folder:

Sub AssembleMasterDocument()
  Dim SubDocFile$, FolderPath$, Template$
  Dim Counter&
  Dim oFolder As FileDialog
  Dim oBookmark As Bookmark
  Dim oTOC As TableOfContents
'Create a dynamic array variable, and then declare its initial size
  Dim DirectoryListArray() As String
  ReDim DirectoryListArray(1000)
  Template$ = ActiveDocument.AttachedTemplate.Path & Application.PathSeparator & ActiveDocument.AttachedTemplate.Name
'Loop through all the files in the directory by using Dir$ function
  Set oFolder = Application.FileDialog(msoFileDialogFolderPicker)
  With oFolder
    .AllowMultiSelect = False
    If .Show <> 0 Then
      FolderPath$ = .SelectedItems(1)
    Else
      GoTo EndSub
    End If
  End With
  Application.ScreenUpdating = False
  SubDocFile$ = Dir$(FolderPath$ & Application.PathSeparator & "*.*")
  Do While SubDocFile$ <> ""
      DirectoryListArray(Counter) = SubDocFile$
      SubDocFile$ = Dir$
      Counter& = Counter& + 1
  Loop

'Reset the size of the array without losing its values by using Redim Preserve
  ReDim Preserve DirectoryListArray(Counter& - 1)
  WordBasic.SortArray DirectoryListArray()
  ActiveWindow.ActivePane.View.Type = wdOutlineView
  ActiveWindow.View = wdMasterView
  Selection.EndKey Unit:=wdStory
  For x = 0 To (Counter& - 1)
    If IsNumeric(Left(DirectoryListArray(x), 1)) Then
      FullName$ = FolderPath$ & Application.PathSeparator & DirectoryListArray(x)
      Documents.Open FileName:=FullName$, ConfirmConversions:=False
      With Documents(FullName$)
        .AttachedTemplate = Template$
        For Each oBookmark In Documents(FullName$).Bookmarks
          oBookmark.Delete
        Next oBookmark
        .Close SaveChanges:=True
      End With
      Selection.Range.Subdocuments.AddFromFile Name:=FullName$, ConfirmConversions:=False
    End If
  Next x
  For Each oTOC In ActiveDocument.TablesOfContents
    oTOC.Update
  Next oTOC
  ActiveWindow.ActivePane.View.Type = wdPrintView
  Application.ScreenUpdating = True
EndSub:
End Sub

3. Under Managing account, click Email forwarding.
4. On the Forwarding page, click Forward your mail to another e-mail account.
5. In the address field, enter the e-mail address where you want to forward your e-mail to, and then click Save. You may also check the box Keep a copy of forwarded messages in your Microsoft account inbox.
6. After you specify an address where you want to forward your e-mail to, you must reply to an e-mail that asks you to validate your account.

 

Here are the steps on how to create a rule to forward to another email address:

 

1. Sign in to https://www.outlook.com/
2. In the upper-right corner of the page, click on the Gear Icon

3. Click More mail settings
4. Under Customizing Hotmail, click Rules for sorting new messages

5. Create a new rule.

6. In step 1 in the first option choose To or Cc line then on the second option contains then on the third option put the email address that you are forwarding email messages from

7. Then on Step 2 click the last option Forward to, then type the email you want to forward it to.

8. Click Save.
