john korchok say:

This feature has been removed from Word 2016 for both Windows and Mac. But you can add very similar numbering back in:

  1. Create a new blank document, then insert a comment.
  2. With the insertion point blinking at the beginning of the comment, choose Format>Style...
  3. The Comment Text style should be highlighted when the dialog open. Click on the Modify button.
  4. If you want all future documents to have numbered comments, check Add to template. Otherwise numbered comments will appear only in the current document.
  5. Click on the Format button and choose Numbering.
  6. Choose a numbering style, then OK out.

    Hey!

    bob jones aka: cybertaz say:
    The default language isn't being changed. Language is a formatting attribute similar to bold, italic, font size, etc. When you copy from another source the language formatting is copied as well so when you simply paste the specified language comes right along with the text.

    Nor does the specified Default Language have any direct impact on pasted content. It only determines what language formatting is in effect when you start a new document.

    For better ways than simply pasting, use Edit> Paste Special - Unformatted Text, Edit> Paste & Match Formatting, or after pasting use the Match Destination Formatting.

    It's also suggested to use Styles for formatting a document & include the appropriate Language specification as a part of the Style's definition. For more information see this page & its numerous links:

    Spelling, Language, Dictionary

    Regards,
    Bob J.

    bob jones aka: cybertaz say:
    You misunderstood... There is only 1 Dictionary but it has access to a variety of Proofing Tools for many languages. The dictionary, itself, does not physically change in any way. The Language formatting of the text tells the Dictionary which set of Proofing Tools should be used to check that body of content. That enables a Word document to be proofed correctly regardless of what mixture of Languages it contains.
