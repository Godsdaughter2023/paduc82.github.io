Spell Check in Visual Basic - Microsoft Community

tiger shoes say:

Spell Check in Visual Basic

Hello,

 I have a Visual Basic sheet and need to have spell check working in the text boxes. Is this possible?

hansv mvp say:

There is no built-in support for this. You could write a primitive spell checker by parsing the text entered by the user into words, and use the Application.CheckSpelling method to test if a word is in one of the dictionaries. But this doesn't provide an interface.

A slightly more sophisticated solution would be to copy the text into a cell (in a hidden sheet, for example), and to call the CheckSpelling method of that cell. This will display the spell check dialog. Afterwards, you can copy the corrected text back into the text box.
