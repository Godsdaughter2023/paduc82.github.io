Saving a CSV to Excel file using VBA

How do I save a CSV file  as a Excel (xlsx)  file using VBA?

Is it also possible to have the user enter a file name using an input box?

jllatham say:

You would have to do this from a different workbook, such as your PERSONAL workbook or a workbook made just for this purpose.  Why?  Because CSV files cannot have macros in them.  They are, in reality, a simple text file such as you might create with a text editing program like Notepad.

Here is what such code could look like (this actually works - it has been tested).

Sub GetCSVSaveAsXLSX()
'allows you to browse for the CSV file
'then it opens it and immediately
'asks for a new name to save it as
'with a suggested name based on the original file's name
'
  Dim srcFilename As String
  Dim csvFile As Workbook
  Dim trialName As String
  Dim newFilename As String
 
  srcFilename = Application.GetOpenFilename("CSV Files *.csv,*.csv", , "Select CSV File", False)
  If srcFilename = "False" Then
    'user cancelled
    Exit Sub
  End If
  Application.ScreenUpdating = False
  Set csvFile = Workbooks.Open(srcFilename)
  trialName = Left(csvFile.Name, InStrRev(csvFile.Name, ".")) & "xlsx"
  ThisWorkbook.Activate
  newFilename = Application.GetSaveAsFilename(trialName, "Excel Files *.xlsx,*.xlsx")
  If newFilename = "False" Then
    MsgBox "The CSV File is still open for use.", _
     vbOKOnly + vbExclamation, "Quitting"
    Set csvFile = Nothing
    Exit Sub
  End If
  Application.DisplayAlerts = False
  csvFile.SaveAs Filename:=newFilename, _
   FileFormat:=xlOpenXMLWorkbook, CreateBackup:=False
  csvFile.Close False
  Application.DisplayAlerts = True
  Set csvFile = Nothing
  MsgBox "File saved as: " & vbCrLf & newFilename
End Sub
