Gutter Spacing for Horizontal Page Orientation

Hello,

Therefore I would like the same margins for both horizontal and vertical page orientation. However when I change the gutter, the header (horizontal orientation) is not affected, which is exactly what I would like it to do. The only possible workaround I can think of is to lower the header in horizontal orientation the same amount I make the gutter in the vertical orientation. Is there really no better way to do this? I can't understand why the gutter won't affect the header.

Thank you very much,

Nico

suzanne s. barnhill say:
You've pretty much hit on the solution. The gutter affects only the primary orientation of the document. To create the appearance of a gutter, you will need to enable Different Odd and Even (note that this is an all-or-nothing setting: if you enable it in one section, it's in effect throughout the document, requiring you to create an Odd Page Header/Footer and Even Page Header/Footer in every section). Then you will need to add some space to the Odd Page Header (adding Spacing After to the header text) and the Even Page Footer (adding Spacing Before to the footer text). You will, of course, have to unlink these headers and footers from the preceding and following sections. This assumes you're duplexing; if you're printing single-sided, then you need only increase the single header in the landscape sections.

nico_2016 say: How can I set different custom margins for vertical and horizontal page orientation. I would like to automatically have custom margins applied when I rotate the page. When I try to do this now, it uses the same custom margins for both vertical and horizontal pages.

suzanne s. barnhill say:
You can't. Margins are a section property, and you have to set them section by section. What you can try is copying the section break at the end of a section with the desired margins and pasting it at the end of another section.
