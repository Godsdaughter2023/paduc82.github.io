How to bullet proof your Windows PC that utilizes Internet Explorer 11 for a browzer.

How to Bullet proof a computer that uses  a Windows operating system and IE 11 (Internet Explorer as a browser Version 11).
‎Monday, ‎March ‎30, ‎2015

Sign on to the internet and go to managed add-ons at the drop down menu on the right hand side at the top (click on the little gear symbol to activate the drop down menu).  This will bring up several lists such as Search Providers, Accelerators, and Tracking Protection. Select  all of these EASY lists and Privacy Choice lists one There are 7 of them)at a time. They are EasyList Standard, EasyPrivacy, Indonesian EasyList, PrivacyChoi ce-All companies,Privacy Choice - Block Companies without NAI oversight and Stop Google Tracking. Check the C:/ Windows/ Program Files and C:/Windows/Program Files (x-86) sub directories for ad ware, or viruses. Look for kooky names like "MyCoup" and check dates installed. Anything new that you did not install?? Discard them.

When all of this is done, create a new set point for the computer to use in the future, if needed.

sam wang msft say: Hi Jim,

Thanks for your updates.

As you don’t have an Exchange Online account, it’s not feasible to directly migrate the contacts and calendar items data from Outlook for Mac to Outlook for Windows. The reasons are as follows:
1. If you would like to configure the Gmail account in Outlook for Windows to sync the data, the connection type is IMAP. However, only emails can be downloaded in an IMAP account, not contacts or calendars.
2. The only export option in Outlook for Mac available at this time is the .olm archive file, while we cannot import an Outlook for Mac .olm file into Outlook for Windows. Also, Microsoft doesn’t have a utility to convert an .olm file to a .pst file(which can be directly imported into Outlook for Windows). Here are some relevant articles for your reference.
Export email messages, contacts, and other items from Outlook 2016 for Mac
You cannot import an Outlook for Mac .olm file into Outlook for Windows

Your understanding will be highly appreciated.

Regards,
Sam
