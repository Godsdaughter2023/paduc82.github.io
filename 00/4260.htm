Just solved the issue myself, allthough I'd like to know why it happens in the first place.

My workaround to get things working goes llike this:

- I opened Regedit with a right cllick as an adminstrator

- I selected the registered plug-in, as shown in the first picture of my original post.

- Right-click the folder in the left tree and pick 'Export

- Name and save the exported registry key in a location that can also be seen by the regular user
(Do NOT pick the desktop, because that is user based)

- Close Outlook

- Double-click the *.reg file just exported (while inside my regular user account)

This type of 'side ways registration' worked.

- After starting Outlook, my add-in was there, doing what it is supposed to do.

Don't know if this has some kind of negative effect on my registry in any way, but for now I am happy.

Making a full backup of the registry would be your safest bett, if you have a similar issue.
