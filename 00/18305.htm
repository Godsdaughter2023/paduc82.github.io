rick rothstein, mvp - excel say:

You can also change the data directly within its own cells using a macro.

 

Sub FixPhoneNumbers()
  Dim LastRow As Long, Addr As String
  Const DataCol As String = "A"
  Const StartRow As Long = 2
  Const AreaCode As Long = 123
  LastRow = Cells(Rows.Count, DataCol).End(xlUp).Row
  Addr = Cells(StartRow, DataCol).Resize(LastRow - StartRow + 1).Address
  Range(Addr) = Evaluate("IF(LEN(" & Addr & ")," & AreaCode & _
                          "&SUBSTITUTE(" & Addr & ",""-"",""""),"""")")
End Sub

 

The only thing you need to do is change the constants (in the Const statements) that are being assigned to match your actual setup.

 

HOW TO INSTALL MACROs
------------------------------------
If you are new to macros, they are easy to install and use. The macro will execute and perform the action(s) you asked for. To make things simpler for you (especially if you will need to call this macro often), when you select the macro from the list, before clicking Run, you can click the Options button and assign a keyboard shortcut to it first and then use that keyboard shortcut the next time you want to run the macro.

pesach shelnitz say:

Hi Sophie,

The changes that you make to a table style should appear automatically in all the tables that have the same style and don't have other conflicting formatting applied to them. I suggest that you place your cursor in each of the other tables and see if the name of your style appears in the Apply Styles dialog box. If it does, click the Reapply button. It it doesn't, you will need to apply the style to the table as I described. Actually, after you have seen the name of your style in the Apply Styles dialog box once, you can just press Shift+Ctrl+S, start typing its name, which should be completed automatically, and press Enter. If you have a large number of tables in your document and you want the same table style applied to all of them, this can be done in one step with a macro, but you should first try to do it manually to a few tables to see what needs to be done.
