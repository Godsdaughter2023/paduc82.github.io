suzanne s. barnhill say:

My guess is that, when you created the font, you created a character for U0027 (ASCII 039), which is the "straight apostrophe," but not for U2019 (ASCII 146), which is the "curly apostrophe." Notepad doesn't have AutoFormat As You Type, so your apostrophe wouldn't be "corrected," while in Word it is. There are (at least) two possible solutions:

  1. Add "curly" single and double quotes to your font; or
  2. When using Scott's Scrawl, disable the "Straight quotes" with "smart quotes" option at File | Options | Proofing | AutoCorrect Options | AutoFormat As You Type.

Record find macro in Word 2010 - Microsoft Community

jonrichco1 say:

Record find macro in Word 2010

I have just wasted an hour trying to record a simple macro in Word 2010 in order to find "zzz" which I use as a temporary bookmark. In case it's of use to anyone else, the secret is to use Advanced Find, not just a simple Ctrl F when searching for the next instance of zzz. If you use Ctrl F, the keystrokes are not recorded. This seems to be an error in Word 2010. To access advanced find, go into Search in the Navigation pane, and use the pull-down arrow next to the Search Document box. This of course refers to any find function in a macro.
rich007a say:

I don't have 2010, but in 2007 recording a macro while I did Ctrl+F and searched for "zzz" yielded this code:

Sub Find_zzz()
'
    Selection.Find.ClearFormatting
    With Selection.Find
        .Text = "zzz"
        .Replacement.Text = ""
        .Forward = True
        .Wrap = wdFindContinue
        .Format = False
        .MatchCase = False
        .MatchWholeWord = False
        .MatchWildcards = False
        .MatchSoundsLike = False
        .MatchAllWordForms = False
    End With
    Selection.Find.Execute
End Sub

 

If you can't get the recorder to record what you want it to (and there are all sorts of things it wont recorde correctly, depending on your version of office and between Word, Excel and PowerPoint), then google is your friend (GIYF)!  Just search for "Word VBA Find" and you will soon find the code you need.

Cheers
Rich

PS. But in fact, I think all you really need is this:

Sub Find_zzz()
'
    With Selection.Find
        .ClearFormatting
        .Wrap = wdFindContinue
        .Execute findText:="zzz"
    End With
End Sub

There are also plenty of examples in the VBA Help files, just type the word Find in VBA and hit F1.
