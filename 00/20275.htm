john visio mvp say:

I would agree with Paul and leave them as Visio drawings within Word.

Word, Excel, PowerPoint and Publisher support Visio like drawings with MSO shapes and smarts. The objects can be connected and moved without breaking the connections. So the experience is Visio like.

You can "convert" Visio drawings to a set of MSOshapes, BUT you will lose all the smarts, connections and the shapes will devolve into basic shapes. A connected pair of boxes will convert into two boxes, a shape for the connection line and two more shapes for the the text block. Not a pretty sight.

The same is true in going in the other direction.

So keep the original drawings in Visio, you can import an image of the drawing into Word to save space, but when it is time to change, edit the Visio drawing.

John... the original Visio MVP (since 1993) Visio.MVPs.org

To round the value of cell A1 to the nearest multiple of 10:

=MROUND(A1,10)

To round the value of cell A1 down to the nearest multiple of 10:
