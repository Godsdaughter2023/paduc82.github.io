How to Remove a next page section break at the end of a document

Using Word 2003, I created a booklet from a Word template.  There is a next page section break at the end of the document that is creating a blank, odd page.  

How can I remove the next page section break without deleting all the formatting in the document?

hansv mvp say:

Place the insertion point above the last section break.

Select File | Page Setup...

Press Enter without changing anything. This will make Word "remember" the page setup.

Place the insertion point below the last section break.

Press F4 to repeat the last action - setting the page setup.

You should now be able to remove the last section break.
