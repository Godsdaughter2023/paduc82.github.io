genevieve cri say:

Hi Mitch,

One of the possible reasons why those applications are stopped working is due to add-ins. We suggest that you open the application in safe mode and disable all the add-ins from there. To do this, follow the steps below:

Note: Close all Office applications before doing these steps.

For Outlook 2016:

  1. Press Windows key and R.
  2. Type in Outlook /safe and click OK.
  3. Click on File and select Options.
  4. Select Add-ins and click the Go button.
  5. Deselect all add-ins and click OK.
  6. Relaunch Outlook.

For Word 2016:

  1. Press the Windows key and R on the keyboard.
  2. In the Run command, type winword /safe (including the space) and click OK.
  3. Go to File and select Options.
  4. In the left corner, select Add-ins and click the Go button.
  5. Deselect all add-ins and click OK.
  6. Relaunch Word.

For Excel 2016:

  1. Press the Windows key and R.
  2. Type in excel /safe (including the space) and click OK.
  3. Click on File and select Options.
  4. In the left corner, select Add-ins.
  5. Change Excel Add-ins to COM Add-ins by clicking the drop down arrow beside it and click Go.
  6. Deselect all add-ins and click OK.
  7. Relaunch Excel.

For PowerPoint 2016:

  1. Press Windows key and R.
  2. Type in Powerpnt /safe (including the space) and click OK.
  3. Go to File and click Options.
  4. In the left corner, select Add-ins and click the Go button.
  5. Deselect all add-ins and click OK.
  6. Relaunch PowerPoint.

Once it's done, check the Office apps one-by-one and see if the issue occurs.

Let us know how the step goes. Otherwise, we’ll be here if you need further assistance.

a. user say:

Hello Hexland,

  • There is no way to manage all of your contact lists from one place, because they're all coming from different sources of origin.
  • The convenience of linking contacts in the People hub is so all of your contact info between services can be seen in one contact card.
  • The linking relationship is kept locally on your device, so linking only affects the contacts on your WP7, and if you do a factory reset of your phone, you'll lose your linked relationships.
