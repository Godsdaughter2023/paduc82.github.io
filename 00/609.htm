I need to do a little rewriting on the article to clarify the sequence at levels past 2 or 3. I think you have it right for level 2. 

Here are the steps for level 3:

  1. In the Level field, click on 3.
  2. From the Link level to style: dropdown, select the style you want to use for this level, Heading 3 in your example.
  3. Delete the contents of the Enter formatting for number: field.
  4. With your cursor still blinking in that field, click on the Include level number from: dropdown and choose Level 1. The level 1 number is added to the Number format field.
  5. Type a period.
  6. Set Include level number from: to Level 2. The level 2 number is added.
  7. Type a period.
  8. Click on the Number style for this level: dropdown and choose  the 1, 2, 3, ... format. The level 3 number is added to the end.

Each additional level includes 2 steps more than the previous. For level 4:

  1. Set Level to 4.
  2. Set Link level to style: to Heading 4, or the next style in sequence.
  3. Delete the contents of Enter formatting for number:.
  4. Set Include level number from: to Level 1.
  5. Type a period.
  6. Set Include level number from: to Level 2.
  7. Type a period.
  8. Set Include level number from: to Level 3.
  9. Type a period.
  10. Set Number style for this level: to 1,2,3, ...

So by the time you get to level 9, you have to set the dropdown and type a period 8 times before you set the final level number!

Let me know if this clarifies things, or whether you're still having issues.
