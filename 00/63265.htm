So to summarize, here are the steps to follow:


1. Download and extract the Windows 7 XPS document writer printer drivers from here Microsoft XPS Document Writer (I suggest you extract to your downloads folder, it's just easier to locate later in step 8.)

2. Open up your Control Panel. (If you open the Windows Explorer, Click on computer, and then Computer at the top, Control Panel is under there)

3. Open Devices and Printers.

4. Delete the Microsoft XPS Document Writer

5. Add a new printer. Click on “A printer that wasn’t listed”

6. Select “add local printer with manual settings” (the bottom one) and hit next


7. There are two options here:

    a.  If you have already set up an XPS Port, then Select “Use a existing Local Port. XPS Port

    b.  If not, then Select Create a New Port

         Select Local Port. then click Next.

         In the Enter a Port Name field, type in XPS Port, then click OK.
         In the Manufacturer list, choose Microsoft.

8. Select “Have disk”, and locate the inf file that your extracted earlier.

9. Finish the Add new printer wizard.

10.  Open Quick Books and create a report and save to pdf just to test.


Victory... You are done!  Now you can save to pdf from any program on your desktop, not just from QuickBooks.
