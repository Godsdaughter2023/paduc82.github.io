  2. If Windows detects an Internet connection, select Activate Windows online now. You might be asked for an admin password or to confirm your choice.
  3. Type your Windows 7 product key when prompted, select Next, and then follow the instructions.

To activate Windows 7 by phone:

  1. Select the Start button, right-click Computer, select Properties, and then select Activate Windows now.

  3. Type your Windows 7 product key, and then select Next.
  4. Select Use the automated phone system. You may be asked for an admin password or to confirm your choice.
  5. Select your location (or the location nearest you) from the drop-down list, and then select Next.
