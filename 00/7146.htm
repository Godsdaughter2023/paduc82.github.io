Auto Select all text that is bold or all text of a certain color in PowerPoint

steve rindsberg say:

That's the spirit!  ;-)

Here's some example code to get you started.  Edit the bits that say Set to reflect the colors you want to change from/to:

        

Sub ChangeTextColors()

    Dim oSl As Slide

    Dim oSh As Shape
    Dim x As Long
    Dim lOldColor As Long
    Dim lNewColor As Long

    ' Set lOldcolor to the RGB value you're looking for
    lOldColor = RGB(100, 200, 100)
    ' Set lNewColor to the RGB value you want to change it to
    lNewColor = RGB(200, 100, 200)

    For Each oSl In ActivePresentation.Slides
        For Each oSh In oSl.Shapes
            If oSh.HasTextFrame Then
                If oSh.TextFrame.HasText Then
                    With oSh.TextFrame.TextRange
                        For x = 1 To .Runs.Count
                            If .Runs(x).Font.Color.RGB = lOldColor Then
                                .Runs(x).Font.Color.RGB = lNewColor
                            End If
                        Next
                    End With
                End If
            End If
        Next
    Next

End Sub

athensproject say:

This may or may not be helpful.

I had Word documents I wanted to save as web sites and have them link to each other and be able to transport the whole directory and maintain function of the hyperlinks. Relative Links were a natural.

This may or may not work outside a Sharepoint environment but perhaps it offers a trail of breadcrumbs for those not using Sharepoint.

  1. Save your Word Document as an MHT file (Single Page Web Archives)
  2. Upload the MHT to Sharepoint with the Add Document link
  3. Select your MHT's check box
  4. Select "Documents" Tab under the "Library" dual-tab.
  5. Click "Edit Document"
  6. When the MHT opens in word, You should be able to create relative hyperlinks with the address: ".\document_name"
  7. Save and close teh document with a new version.
  8. Opening the MHT as a web page allowed me to open another MHT within the same directory.

monique tang msft say:

Hi Business System - Indochine Engineering,

After the latest build of Office 2016 (16.0.6741.2014) is installed, by pressing the Backspace key under the Outlook email, you are using the new feature called “One click Archive” for Outlook 2016 for Windows that the email will be moved to the archive folder.

The first time you use Archive, you'll be asked to let Outlook create an archive folder called Archive, or you can select an existing folder.

Next time, the "Set up One-Click Archive" window will not pop up.  All the emails will be moved to the archive folder automatically when you press the Backspace key.

I’d like to confirm the following information to better understand the situation:

1. The detailed version of your Outlook client. Please make sure it’s the latest version (16.0.6741.2014 or later) on your side by navigating to File> Office Account> Update Options> Update Now.

2. Your OS version.

3. Please navigate to File> Info> Tools> Set Archive Folder to reset Archive as the archive folder and then check if the issue persists.

4. To identify if the issue is related to Add-ins, please start Outlook in safe mode to see if the issue persists. (Click Start > Run> type outlook.exe /safe> click OK.)

If it works, try disabling add-ins to see if it works:

Click File> Options> Add-ins> Select “COM Add-ins” in the Manage drop-down> click Go> Uncheck any add-ins that you are not using> click OK.

Regards,

Monique
