How to change ** default** font of Calibri for charts - Microsoft Community

arbems say:

How to change ** default** font of Calibri for charts to a pdf series such as times new roman . I do not want to have to change font every time I use chart so that it always works in pdf.




CHANGED FROM CALIBIRI THEME TO TIMES NEW ROMAN SO AS TO WORK BETTER WITH PDF FILE PRINTOUTS

bob jones aka: cybertaz say:
The fonts used in Office 2011 Charts are determined by the Font Scheme of the Theme used in the document. Each Font Scheme consists of a Theme Headings font & a Theme Body font -- in some schemes the same font is used for both, in other schemes they're 2 different fonts.

In any Theme you can select a different built-in Font Scheme by going to the Themes group on the Home tab of the Ribbon. Click the Aa button to choose one of the other schemes. Any of them should render well in a PDF.

Unfortunately, there is no provision in Office 2011 to create custom Font Schemes. If you happen to be proficient with XML I believe you can write them & add them to the global or user-specific Theme Fonts folder, but there's no "point 'n' click" user interface such as what's provided in the later Windows versions of Office.

What you can do is format a chart as you wish, then right-click it & choose Save as Template. To create a new chart based on the template use the Other button in the Insert Chart gallery -- Any custom templates will be selectable at the very bottom of the dropdown palette.

bob jones aka: cybertaz say:
This changes only the default font used in worksheets. It has no bearing on the fonts used in Charts. That's determined by the Theme used in the file or by directly formatting the chart element.
