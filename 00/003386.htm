matt mickle say:

This question was answered by Hans Vogelaar previously:

ListBox Answer

If it is a Forms list box on a worksheet, you can't set the font size.

If it is an ActiveX list box on a worksheet: turn on Design Mode, then right-click the list box and select Properties from the context menu.

Click in the Font property, then click the builder dots ... on the right hand side.

You can set the font, font size, font style etc.

In VBA:

Worksheets("Sheet1").ListBox1.Font.Size = 24

If it is a list box on a userform in the Visual Basic Editor, open the form in the Visual Basic Editor.

Select the list box, then work with the Properties pane on the left hand side. The rest is the same as for ActiveX list boxes.

In VBA:

UserForm1.ListBox1.Font.Size = 24

Yes. Text boxes are not linked by default. If you create a new publication and it has a text box that has too much text to display you will see a small box with three dots attached to that text box.
