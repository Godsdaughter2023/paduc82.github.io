Private Sub Workbook_SheetActivate(ByVal Sh As Object)

The proofing language is applied to each character in a document. It is often applied through Style definitions.

suzanne s. barnhill say:
  1. AutoCorrect entries are stored in .acl files by language. The entries you created using English (South Africa) will not be available when the language of the text is English (United Kingdom). You will have to change the language applied to the text. This is most easily done by adding Language to the status bar (right-click on the status bar and click Language on the menu); the status bar then displays the language at the insertion point, and clicking on it opens the Language dialog so you can easily change it. Language is a font property that can be applied (like bold or italic) to as little as a single character. Unless the language matches the AutoCorrect entry, the entry won't be available.
  2. The Proofing Tools are undergoing a complete revision and upgrade for Office 2016, and the Style portion was not complete when the application was released. These tools are being tested internally and should be available in a monthly update soon.
