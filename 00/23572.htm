A hard reset is needed to clean everything out on your phone. In fact, it is best to do a hard reset BEFORE any firmware update. This is because of possible inconsistencies with system files our own usage may have made. Installation of apps also has an effect, as they create their own system files which are set to be compatible to that of older firmware. New files and new system files coming with the newer firmware adding new features everywhere (this one, v50, in fact from my understanding affects display and icon system files due to the kinetic scrolling in icon menu) may have caused incompatibilities with certain apps. Also one must take into consideration orphaned files caused by installation and uninstallation of apps.

 

All of these possible incompatibilities may exist on both phone or memory card which both contains system files. First thing, try to run a disk check on the memory card, if nothing happens, you format. If nothing still happens, you reinstall the fw into the phone. If nothing still happens, you put a formatted card and format the phone and reinstall.

gsm824 say:

is it safe to access email from public wifi

entegy say:
That's not something up to the phone.

Let's do a quick recap on how wireless tech works:

A lot of public places have free WiFi nowadays. If you can connect to the WiFi point without putting in a password for the WiFi point itself (putting in a code in the browser doesn't count), that WiFi point is called an open point. The mere fact that it is open means that all communication over that point is unencrypted and easily intercepted by a third-party on that network. Visiting secure sites (denoted by HTTPS at the beginning of the URL) can still be intercepted, but that data is encrypted.

In comparison, a WiFi point protected by a password will have all its data encrypted while communicating with the access point. It can still be intercepted, and some wireless security (such as WEP) can be cracked.

Your safety at a public WiFi point depends on the trust of others. Checking your email might be okay, but I wouldn't be doing banking at a public WiFi spot.

Now, does Hotmail use secure login? Yes, automatically. Comcast? Dunno. The email service has to have HTTPS setup and most of them do nowadays. In Settings>email & accounts, check each account for an option called something like "This connection requires SSL connection". That is a secure login option, but it may not work with all email providers.. Your Windows Live account does this automatically, so there is no option there.
