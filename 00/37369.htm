Hi,

Error 0x80070002 - system cannot find the file specified occurs because backup and restore is expecting the new folder to exist in C:\Windows\System32\config\systemprofile instead of the actual path.

You may try these steps and check if it fixes the problem. First uninstall or disable any third party backup programs before performing these steps.
1. Launch Backup and Restore
2. Click Change Settings
3. At the “Select where you want to save your backup”, click Next
4. At the “What do you want to back up?”, select Let me choose and click Next
5. Navigate to the new library
6. Deselect the new library
7. Expand Computer
8. Navigate to the new folder, select it, and click Next
9. At the “Review your backup settings” page select Save settings and exit
10. Click back up now.

Error code 0x80070002 when you back up files in Windows 7:

I would suggest you to follow these steps to delete recent pictures from the list in paint:

above step fails follow these steps:

a.       Press Windows + R key on the keyboard, type “regedit” (without quotes), and then click OK.

b.      If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

c.       In the navigation pane, locate and expand the following registry subkey: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Applets\Paint\Recent File List.

d.      On the right pane you will see the list of the files you recently open right on the file and click on delete.

e.      Exit the registry editor and Restart computer and check if the issue persists.

 There is a key on your keyboard called the WINDOWS key.  It's in the lower left corner of the keyboard.  It's between the CTRL and the ALT keys and it has the Windows logo on it.  You press that key down and hold it with your left hand then press the R key with the right hand.  This brings up a little window called RUN.  In the middle of that little window is a white box where you type in the word REGEDIT.

 

 I have to ask though.  Why do you care about clearing this list?  As Vaseem pointed out, fooling around in the registry is a good way to get yourself in trouble, especially when you say you're just starting to learn about Windows.

 

 I hope this helps.  Good luck.
