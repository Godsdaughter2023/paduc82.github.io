The blanks aren't really blank - both in AJ10:AJ17 and in H50:H57, the blank looking cells contain an empty string "", so they match. You can work around this by checking for this in the conditional formatting formula:

 

=AND(AJ10<>"",MATCH(AJ10,$H$50:$H$57,0))

When receiving the Open or Save prompt instead of a Run or Save As prompt in IE these are the steps I took to fix it.

  1. I tried every “-extoff”, "Disable Addons, and “Reset IE” Fix out there – NONE WORKED
  2. Deleted the folder C:\Users\username\AppData\Local\Microsoft\Windows\WebCache – This did not work but may have had an effect on my result

  3. Downloaded Lockhunter (this can be found at lockhunter dot com) and installed it
  4. Referenced the locations of the IE Temporary Folders locations identified in this article > http://www.thewindowsclub.com/temporary-internet-files-folder-location

daniel g. grau say:

You are not recalling correctly!

  • Word allows using Option + Character shortcuts with a very limited set of keys. For instance, Delete/Backspace or Del/Forward Delete.
  • Option + m = µ is a standard Mac combination (it's not user-defined). You can see that and other combinations via the Keyboard Viewer (System Preferences > Keyboard > Show Keyboard..., and then select Show Keyboard Viewer in the menu bar, next to the clock).
  • There are system-wide shortcuts and app-only shortcuts that you can define in System Preferences > Keyboard > Shortcuts, but most people don't use them.
