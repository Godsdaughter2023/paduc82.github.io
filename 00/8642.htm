tom van stiphout say:

Rather than using a calculated field in a table, use one in a query. This expression should work:

DateDiff ("d", [DC_Date], [Admit_Date])

Why do some of the others not work?

DateDiff (d, [DC_Date], [Admit_Date])

The first argument is supposed to be a string expression, and d is not a string. "d" is a string.

DateDiff ("d", [DC_Date] - [Admit_Date])

This does not work because DateDiff takes at least 3 arguments, not 2.

DateDiff [DC_Date], [Admit_Date]

This does not work because DateDiff is a function, and as such its arguments need to be in parentheses. Also the first argument (interval) is missing.

[DC_Date] - [Admit_Date]

This actually does work in a query as well, but I prefer DateDiff because it is more explicit. After all, date1-date2 what does that return? The difference in seconds? in years? DateDiff makes it explicit.

john w. vinson say:

You were close... :-{)

DateDiff("d", [Admit_Date], [DC_Date])

You can select the DateDiff function in the query or VBA editor and press F1 to get the Help page for how to use the function. The first argument is a Text String - "d" for Days, "s" for Seconds, "yyyy" for Years, all sorts of things in between; the second argument is the starting date/time, the third argument the ending date/time.

Do note that this calculation should NOT be done in a Table. For one thing, the "calculated" field cannot call functions such as DateDiff; for another, it's not terribly useful in regular Access. Just store the base data in your Tables and use Queries to do the calculation, e.g. by typing

DaysInProgram: DateDiff("d", [Admit_Date], [DC_Date])

in a vacant Field cell in the query grid; or calculate it on the fly in a Form or Report textbox by setting the text box's Control Source to

=DateDiff("d", [Admit_Date], [DC_Date])

gord dibben say:

I don't know what type of order your photo names and column A names match up.

 

This code will place all selected pictures from a folder into column B.  I'll leave it to you to sort the photos in the folder or just insert them first then add the other data later.

 

Sub InsertPictures()

    Dim Pict() As Variant
    Dim ImgFileFormat As String
    Dim PictCell As Range
    Dim lLoop  As Long
    Dim sShape As Picture

    ActiveSheet.Protect False, False, False, False, False
    ImgFileFormat = "All Picture Files(*.jpg;*.jpeg), *.bmp"

    Pict = Application.GetOpenFilename(ImgFileFormat, MultiSelect:=True)

    If Not IsArray(Pict) Then
        Debug.Print "No files selected."
        Exit Sub
    End If

    Set PictCell = Selection.Cells(, 2)
    For lLoop = LBound(Pict) To UBound(Pict)

        Set sShape = ActiveSheet.Pictures.Insert(Pict(lLoop))

 

        With sShape

            sShape.Top = PictCell.Top
            sShape.Width = PictCell.Width
            sShape.Height = PictCell.Height
            sShape.Left = PictCell.Left
            sShape.Placement = xlMoveAndSize
        End With


        Set PictCell = PictCell.Offset(1)
    Next lLoop

End Sub
