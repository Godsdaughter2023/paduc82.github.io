You cannot directly cause a macro to run from the command line.

You could have a Workbook_Open macro (in the ThisWorkbook module) or an Auto_Open macro in a normal module that would run whenever the workbook was opened.

But I deduce from your question that you would only want to run the macro when the workbook was run from the batch file.

You could write a small loader workbook that loads the target workbook and runs the macro:
The loader workbook would contain something like

Sub Auto_Open()
  Application.OnTime Now, "Continue_Open"  ' sometimes have problems doing too much in Auto_Open
End Sub
Sub Continue_Open()
  Workbooks.Open ThisWorkbook.Path & "\RealApp.xlsm"  ' change the name!
  Application.Run "RealApp.xlsm!MacroToRun"                     ' change the names of the workbook and the macro
  ThisWorkbook.Close False
End Sub
