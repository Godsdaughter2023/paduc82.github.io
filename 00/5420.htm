Start with a master presentation that includes all possible content and is NOT IRM/DRM-protected, or at least not read-only.

Note that some of the information on the Microsoft page is outdated and contains errors, particularly when applied to Outlook 2003, 2007 or 2010. Specifically:
1. Ignore any advice to copy your PST file to Outlook's default location to connect it to your profile. Doing so will often corrupt your Outlook profile. Copy it anywhere BUT the default location and then open it within your Outlook profile.

how to create a desktop shortcut for Outlook 2016?

“sam” say:
Hi
I’m Linnell I'm a fellow Community member here to help.

Create a desktop shortcut for an Office program.
Click Start, point to All Programs, point to Microsoft Office, and then point to the Microsoft Office program for which you want to create a desktop shortcut.
Right-click the name of the program, point to Send To, and then click Desktop

I hope this helps. Please feel free to ask questions and let us know how it goes.
Regards,
Linnell

davem121 say:
Hi Twalk18,

1.

If you are using Outlook online

Open Edge and navigate to OWA

Click the 3 dots at the top right of Edge and choose 'Pin this Page to Start'

Then drag the new Tile from your Start Menu onto your Desktop

Delete the new Tile From your Start Menu


2.

If you are using the Outlook application in Office 2016

Start up Outlook, or any other Office Application

Press CTRL + Shift + Esc to open Task Manager

Right click that Office Application on the processes tab and choose 'Open File Lcation'

When the File Explorer window opens, right click Outlook and choose 'Send to' - Desktop (create shortcut)
