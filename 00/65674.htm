Here are solutions to some common problems with installing updates.

To learn how to perform clean boot, visit <here>.

Once you finish troubleshooting, do not forget to reset the computer to start normally.

------------------------------

Method 1 : Run Built-in Windows Update Troubleshooting Tool

  1. Type troubleshooting in search charm. Then tap or click troubleshooting.
  2. Tap or click Fix problems with Windows Update.
  3. Follow the instrucitons to finish troubleshooting tool.

  4. Once you have done, perform Windows Update to confirm the problems gone.
    If you still have problems, go on to the next method.

  1. Type cmd in search charm. Right click on Command Prompt, then click Run as administrator.
  2. Copy following command.
    DISM.EXE /online /cleanup-image /restorehealth

  3. Paste command to Command Prompt.
    Right click on the blank space, then click Paste.


  4. Hit Enter key to run command. It may take several minutes to finish.
    If you find The operation completed succesfully, click X to close Command Prompt.
  5. Perform Windows Update to confirm the problems gone.
    If you still have problems, go on to the next method.

Method 4 : Install Only Few Updates

  1. If you have large numbers of updates to install, select only 2 or 3 updates.

  2. If the updates are installed successfully, select another 2 or 3 updates.

  3. If the updates are failed to install, skip these updates, and select another 2 or 3 updates.
  4. Repeat Step 2 and Step 3.

  5. If you have no rest updates and some updates still cannot be installed successfully, go on to the next method.

Method 5 : Run System File Checker

  1. Type cmd in search charm. Right click on Command Prompt, then click Run as administrator.

  2. Copy following command.
    SFC /scannow

  3. Paste command to Command Prompt.
    Right click on the blank space, then click Paste.
