danzizme say:

Okay let me know how that goes and what was done to troubleshoot.

You can run this command the same way you ran the dism restore health command

sfc/scannow if that comes back with errors that can't be fixed run the dism command again then you can run the sfc/scannow to confirm the same issues were repaired.

If you you continue to get errors with sfc/scannow after they are fixed chances are the hard drive is having issue. You can follow these steps to scan the drive and mark the sectors as bad so they aren't used. Since there is a possibility that you have a failing disk it is best to back up your data as soon as possible.

Windows key + X > Command Prompt (admin) 

If you have an SSD drive, run:

CHKDSK /f

If you have a standard rotary hard drive, run this command instead:

CHKDSK /f /r

You'll get a message that the tool cannot be run because the disk can't be locked and whether you want to schedule a check on the next reboot. When you have saved your work and are ready to restart type Y and hit enter reboot immediately, or finish what you're doing and then restart.

If bad sectors are found it will mark them so that they can't be used. But if the tool appears to hang for more than a couple of hours at a certain percentage, then that would indicate that your hard drive is defective and will need to be replaced.

lintekpc say:

Your Windows 10 entitlement applies only to the laptop because that's the device you were using when you performed the upgrade. As you mentioned, no activation key was used when you re-installed Windows 10. Microsoft is apparently collecting the hardware device IDs in combination with the activation key associated with the licensed copy of the version of Windows you are running at the time of the in-place upgrade. 

Now, you own the copy of Windows 8.1. You install it to the new desktop and perform the upgrade to Windows 10. Activation will probably fail. I will say you could be able to convince Microsoft that all you want to do is transfer your entitlement to another machine. This might take a call to their activation department.
