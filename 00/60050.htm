"What does the images folder mean?"

I would guess this is your laptops system image partition for restoring Windows. Because they generally

don't come with a Windows installer disk laptops all have a 'restore to factory defaults' partition.

- Better to keep backups of all your Data on an external source. I also keep my games (which are also installed

on separate drives from Windows) backed up to external HDD's .

With close to 200 games I'm not keen on reinstalling them if a HDD fails suddenly.

System Restore is more important for restoring Windows to a recent time, should something go wrong,

so just having it on C:\ should be good enough for most people. 

- Note that long times between restore points means more things that you may want to keep could

get uninstalled in the restore process.

bossdweeb say:

"Do you just drag the folder from the D drive to the external hard drive?"

Yeah ! Drag and Copy. I don't copy the whole folder in one go though. I make a folder of the same name

as my main game folder on the external drive and then copy the games in groups into that folder.

I like to keep it down to around 50+ GB per copy.  This way if something goes wrong with one session

you have less to do over .

Games and software have registry entries in Windows, so as long as you are still running the same

Windows install as you were when you copied the games they will still run if you need to copy them

back to a new HDD.  As long as the Drive Letter/file path matches your shortcuts the games will be good to go

( I use the same name for the external drive as the original, but add it's original Drive Letter in parentheses as

a reminder).

( btw - My Win10 goes back to 2010 via Win7. Most of my disk based games were installed on Win7

in the first few months of '10 ).

update the following drivers:-


Audio driver

Graphic card driver

TV Tuner card driver


You may need to reset WMC if required.


Run Setup Again.

Open WMC > Tasks > Settings > General > Windows Media Center Setup > Run Setup Again

Does it help?

If the above doesn't help then try resetting WMC:-

Open Control Panel > Uninstall a Program > Turn Windows Features on or off > Scroll down to Media Features > Uncheck the box against Windows Media Center > Follow on screen instructions > Reboot > Again open Turn Windows Features on or off under Control Panel > Scroll down to Media Features > Check the box against Windows Media Center.
