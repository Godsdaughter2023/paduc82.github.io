Your Query does not include the specified expression "Customer - Microsoft Community

rehwright say:

Your Query does not include the specified expression "Customer Number" as part of an aggregate function

I have the following query and I am getting an error message stating that the field "Customer Number" is not part of an aggregate function.  I am not trying to group by Customer Number, nor is Customer Number part of an aggregate function.  I am grouping by Order Number which is part of an aggregate function, so I'm not sure what I am doing wrong here.  Any idea what's wrong with my query?

SELECT Customers.[Customer Number], tblOrder.OrderNumber, Customers.[Type of Customer], Customers.[First Name], Customers.[Last Name], Customers.Company, tblOrder.Type, [tblOrder].[SubTotal]-[tblOrder].[Discount] AS Total, tblOrderLineitems.[Item Number], Sum(tblOrderLineitems.Cost) AS SumOfCost
FROM (Customers INNER JOIN tblOrder ON Customers.[Customer Number] = tblOrder.[Sold to Customer]) INNER JOIN tblOrderLineitems ON tblOrder.OrderNumber = tblOrderLineitems.OrderNumber
WHERE (((tblOrder.Type)="Invoice"))
GROUP BY tblOrder.OrderNumber;

scottgem say:

Every Column that is not an aggregate function (or WHERE or Expression) needs to be listed in the Group By List. that's why you are getting the error. 

I would suggest that you need to create at least 3 queries. One to sum the lineitems by Order:

SELECT OrderNumber, Sum(Cost) AS SumOfCost

FROM tblOrder INNER JOIN tblOrderLineitems ON tblOrder.OrderNumber = tblOrderLineitems.OrderNumber

WHERE Type = "Invoice"

GROUP By OrderNumber;

SELECT Customers.[Customer Number], tblOrder.OrderNumber, Customers.[Type of Customer], Customers.[First Name], Customers.[Last Name], Customers.Company, tblOrder.Type, [tblOrder].[SubTotal]-[tblOrder].[Discount] AS Total, tblOrderLineitems.[Item Number]

FROM (Customers INNER JOIN tblOrder ON Customers.[Customer Number] = tblOrder.[Sold to Customer]) INNER JOIN tblOrderLineitems ON tblOrder.OrderNumber = tblOrderLineitems.OrderNumber;

Then join these 2 queries on OrderNumber to get the final result.

rehwright say:

Thanks Scott, You've got me on the right track, I've created the 2 queries but how do I join them?  My SQL's a little rusty not having used it much for several years.

 

scottgem say:
Just use the Query Design mode. Add the 2 queries to the table grid and join then on OrderNumber. OrdernUmber should be in both queries. then add the columns you need to the query.
