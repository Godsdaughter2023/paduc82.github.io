tech-geek say: You cannot upgrade from Windows Xp to Windows 7. You cannot upgrade from any 32bit version of Windows to any 64bit system. If you want to keep both Xp and install Windows 7 (dual boot) you will need to either have a separate hard drive for each or you will need to partition your current hard drive and install Windows 7 to the new partition. There is no need to re-install Xp unless it is just something that you want to do. If you do install Xp for whatever reason, it is best to install Xp first then install Windows 7. Windows Xp has no bearing on the activation of Windows 7.


Second, when you purchase Windows 7 online from Microsoft you will be given the option to download the iso file. Windows 7 will not fit on a cd. You can use WINDOWS 7 USB/DVD DOWNLOAD TOOL to burn the iso file to a DVD. Once you have burned the iso file to a dvd you will need to boot from the disc. If your computer has the ability to boot from removable drives you can install Windows 7 from a usb flash drive. However, even if you intend to install Windows from a usb flas drive you will want to burn the file to a dvd as well as a means of having a backup.
