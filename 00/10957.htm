qa guy commelec say:

Use MID function --

   Mid([YourTable].[YourField], InStr([YourTable].[YourField], "@")+1)

dirk goldgar say:

I am trying to separate an email address into 2 halves. Existing field "email" want to create a new field with the domain or everything after the @ sign e.g. *** Email address is removed for privacy *** becomes "adam","yesdata.com"

I am relatively new at access and am doing this via the query wizard.

The SQL for such a query would be something like:

    SELECT Left([EmailAddress],InStr([EmailAddress],"@")-1) AS MailboxPart, Mid([EmailAddress],InStr([EmailAddress],"@")+1) AS DomainPart

    FROM ...

In query Design View, the calculated fields would look like this:

    MailboxPart: Left([EmailAddress],InStr([EmailAddress],"@")-1)

    DomainPart: Mid([EmailAddress],InStr([EmailAddress],"@")+1)

If the EmailAddress field could be Null, you'll probably need to amend those to forceit to be converted to an empty string, like this:

    MailboxPart: Left([EmailAddress],InStr(Nz([EmailAddress]),"@")-1)

    DomainPart: Mid([EmailAddress],InStr(Nz([EmailAddress]),"@")+1)
