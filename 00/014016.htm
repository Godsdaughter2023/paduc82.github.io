¡firedog say:

… entered my display name, e-ddress and password (generated using the password app from Microsoft). 

I don't know the first thing about Thunderbird, but I can say that you don't generate a password with a password app. I think you're referring to an Authenticator app, which is a different thing altogether.

Once you've set up two-step verification for your Microsoft Account, you have to generate an app password to use in your mail client program instead of the account password. This is because the program has no way to submit the code for the second step of the verification process. When you access the security page of your Microsoft Account, you'll probably have to enter the code from your authenticator.
