andreas killer say:
Okay, do as follows:

Download this file
https://www.dropbox.com/s/700hbz18bsc7nu2/FileS...
Open your Excel file
Open the VBA editor
Press CTRL-M and import the downloaded file
In the menu click Insert \ Module
Paste in the code below
Save your Excel file in the same folder as the PDF files
Run the code

Andreas. 

Option Explicit
Sub Test()
  Dim Path As String
  Dim R As Range
  Dim FS As New FileSearch
  Dim i As Long
  Dim OldName As String, NewName As String
  
  'Get the path from this file
  Path = ThisWorkbook.Path
  If Right(Path, 1) <> "\" Then Path = Path & "\"
  
  With FS
    'Search here
    .LookIn = Path
    'for PDF files
    .FileName = "*.pdf"
    'Do it
    .Execute msoSortByFileName, msoSortOrderAscending
    'For each cell in column A
    For Each R In Range("A1", Range("A" & Rows.Count).End(xlUp))
      i = i + 1
      'Rename the file
      OldName = .FoundFiles(i)
      NewName = Path & R.Value & ".pdf"
      Name OldName As NewName
    Next
  End With
End Sub

Do Publisher documents convert better to Word or Power Point? - Microsoft Community

stacylouch say:

Do Publisher documents convert better to Word or Power Point?

I need to open Publisher files on a computer that does not have Publisher. Can I resave them in a different format? Or will they convert better to one program than another?
davidf2 say:

Publisher files are proprietary. You cannot effectively open them with any other program.

 

Assuming that you have access to your old computer with Publisher installed you can do a Save As a Word file, but only the text will convert. All images will be lost.

 

You can Save As a pdf file using Pub 2007 or 2010 and the text and images are saved. You can select and copy text from those pdf files, and the pictures can be somewhat salvaged using the 'snapshot' feature of a pdf reader and pasting into a image editor for processing. Not a very satisfying solution for images especially but if you just need to view the whole publication a pdf will work.

 

I think a best choice might be to save the file as a filtered web page. Before you do, in Publisher 2007 go to Tools > Options > Web tab and check the option to 'Allow PNG as a graphics format...'. Then File > Save As > Web Page, Filtered and save the files to where you can easily find them such as your desktop. I would suggest you save them to a folder named as per your original Publisher file name. Publisher will produce an index.htm file which is the 'home page' that will render your entire page in any web browser. From there you can select any design element including text. Publisher will also produce an index_files folder that will contain any other pages of your original Pub file as .htm files and copies of all the grahphics and images from your publisher publication and these will be in PNG format. If you had not changed the option as mentioned above you will get a lower resolution GIF and JPG files. Save as a web page and all design elements from your Publisher file will be avialable to you.

 

If you do not have access to your old computer to open and save Publisher files to alternative formats, you can download the free 60 trial copy of Publisher 2010 here:

http://www2.buyoffice.microsoft.com/usa/product.aspx?cache=790915456&culture=en-US&sku=10234422
