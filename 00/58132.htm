windows 7 hang problem

shinmila h say:

Hi Chandresh,

 

Event Viewer as suggested by Palcouk should be able to give you enough clue as in what is causing the computer to freeze up so frequently.

 

In the meantime, you can follow these steps.

 

Oftentimes compatibility issues can cause your computer to freeze up. Check if there’s any such problem with third party application installed.

 

Step 1: Perform Clean Boot

 

Use MSCONFIG to disable all startup programs. Test to see if problem continues to occur. The suggestion to disable all startups using MSCONFIG is to isolate the problem(s) between the baseline OS/Drivers installed and any third party applications.

 

Please refer this article for detailed information on how to do this.

 

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7

http://support.microsoft.com/kb/929135

 

 

Note: Microsoft cannot guarantee that any problems resulting from the use of Third Party Software can be solved. Using Third Party Software is at your own risk.

 

Step 2: Check Device Manager for errors

 

Many at times driver conflict can cause your computer to freeze up.From device manager you can see the devices that are installed and if there’s any problem with the drivers that are installed for those devices. Accordingly you can figure out if any particular device could be causing the problem.

 

To check for error from device manager:

 

1)      Go to start and type device manager.

2)      Select device manager from the control panel list.

3)      From the device list, see if there’s any exclamation mark or error reported. If so, you can remove the driver related to the device and install a new one.

 

Step 3: Update drivers

 

Go to the device manufacturer’s website and download the latest driver for the device that is compatible with Windows 7.

 

 For driver updates, refer this link:

 

Update a driver for hardware that isn't working properly

http://windows.microsoft.com/en-us/windows7/Update-a-driver-for-hardware-that-isnt-working-properly

Regards,

Shinmila H - Microsoft Support

How To change screen resolution - Microsoft Community

hariprasathg27 say:

How To change screen resolution

I have an old HCL (square shaped ) monitor. When i changed the screen resolution from 1240*720 to 400*600 ( recommended as shown in settings) the monitor suddenly turned off (not power off) but the system is still on. What can i do now to change the resolution back...

davem121 say:

You have set a resolution unsupported by your monitor, I imagine it should be 600 X 400 not 400 X 600 . . .

Please Note:, the monitor you are using and the resolution you suggest is not good enough for Windows 10 - minimum suggested resolution 1024 x 768

Start your system in Safe mode and reset your resolution:

Start your System, then just after the Manufacturers logo disappears and Windows 10 tries to boot, press and hold the power button down for 5 - 10 seconds to perform a hard shut down

Do this Twice

On the third boot attempt, Windows will boot into the Windows 10 Repair Environment, from there you can access Startup Recovery, Safe Mode, Command Prompt . . . etc.

Go to Troubleshoot - Advanced Options - Startup Settings, click restart.
Upon restart press 4 to boot into Safe Mode, then you should be able to re-adjust your resolution
