doug robbins - mvp office apps & services (word) say:

If you list the words that you want to find in the first column of an Excel Worksheet and their replacements in the second column, and you run a macro containing the following code when the document in which you want to make the replacements is the active document and you select the Excel Workbook when asked to do so, it will find and replace each of the words listed in the Excel Worksheet.

Dim xlapp As Object

Dim xlbook As Object

Dim xlsheet As Object

Dim myarray As Variant

On Error Resume Next

Set xlapp = GetObject(, "Excel.Application")

If Err Then

    bstartApp = True

    Set xlapp = CreateObject("Excel.Application")

End If

On Error GoTo 0

Set xlbook = xlapp.Workbooks.Open("C:\Users\Doug\Documents\animals.xls")

Set xlsheet = xlbook.Worksheets(1)

myarray = xlsheet.Range("A1").CurrentRegion.Value

If bstartApp = True Then

    xlapp.Quit

End If

Set xlapp = Nothing

Set xlbook = Nothing

Set xlsheet = Nothing

For i = 2 To UBound(myarray)

    Selection.HomeKey wdStory

    Selection.Find.ClearFormatting

    With Selection.Find

        .Text = myarray(i, 1)

        .Replacement.Text = myarray(i, 2)

        .MatchWildcards = False

        .Wrap = wdFindContinue

        .MatchCase = False

    End With

    Selection.Find.Execute Replace:=wdReplaceAll

Next i

How To Import OviMail To Outlook

since nokia stopped its support for ovi mail, is anyone really sure how i can import all the email that i had in that account to my outlook account?

rajithr say:

Hi Stev,

Thank you for the post.

You may follow these steps to Migrate Nokia Mail messages to Outlook.com

  1. Go to outlook.com and sign-in to your existing outlook.com account.
  2. After login, open Settings> Options> Import email accounts>( ). You may also directly access the import panel by opening a long time (sometimes upto a few days).
  3. Forward all incoming mail on your NokiaMail to your outlook account as Outlook will not check for new messages that may arrive in your NokiaMail.

Hope this helps. If you have further queries, we will be happy to help.
