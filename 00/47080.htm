arminatorx say:

The Xbox One X has a thermal sensor that tells the fan when it gets too hot. 

For an entertainment device that is also used as a media player, it's important for the console to be as silent as possible. So even when you "feel" the console gets "very hot", it is not "too hot". 

Should the fan fail, and the console gets too hot, you will get an error message on screen, that the console has to shut down because it ran too hot. 

So unless you get an error message about the console being too hot, rest assured that the temperature of the console is perfectly fine, even if it feels warmer to touch.
