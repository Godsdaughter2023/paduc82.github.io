teylyn say:

Hello,


colour is not data. If you colour code your data, you need to learn with the fact that colour does not present itself easily for analysis and further processing.


As an alternative, use a status column in the table and assign a status code to each row. Then use conditional formatting to colour the row according to the value in the status column. For counting the status values you can use out of the box formulas like Countif() or a little pivot table, constructed with less than ten clicks and no VBA or special formulas.


The benefit of that is that you can quickly change the colour of a row by just changing the status value. Change just one cell and the whole row changes colour.


But maybe that approach is not for you. If you don't want to use VBA, consider arranging your data in an Excel Table. The data must  be contiguous, i.e. no empty rows or empty columns. Then click a cell in the data area and select Insert > Table (or Ctrl-T).


Outside of the table, you can have a Subtotal() function with the Counta parameter, for example


=Subtotal(3,A2:A10)
