Calculated fields in forms - Microsoft Community

philipribs say:

Calculated fields in forms

I Have a Form  with calculations in it.  The calculations are working fine but I cannot  get the Values to show in the  table that  the form is associated with.  Other Information  will show  in  the table  after I inputed it in but  not the calculated fields. 

john w. vinson say:

That's as it should be!

Storing a calculated field in a Table accomplishes three things: wastes disk space (trivial), wastes time (almost any calculation is going to be much faster than a disk fetch), and most importantly risks data validity (if one of the underlying field values changes you'll have data in your table which is wrong with no easy way to detect that it is).

Simply store the underlying data in the table and redo the calculation whenever you need it, as a calculated field in a Form, Report or Query.

John W. Vinson/MVP
