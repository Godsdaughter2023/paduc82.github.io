doug robbins - mvp office apps & services (word) say:

jay freedman say:
Text boxes and frames can't have no anchor at all. They must be anchored to some paragraph of text.

That said, you can set a text box's position with respect to the page rather than with respect to the anchor paragraph. The only gotcha is that if the anchor paragraph moves to a different page, then the text box will also move to that page (but in the same location relative to the page margins). If necessary, you can drag the anchor to a different paragraph to put it back on the original page.

To set this up, insert the text box, right-click its edge, and choose More Layout Options. (You can get to the same dialog box by clicking the little arrow in the bottom right corner of the Size group on the Drawing Tools | Format tab of the ribbon.) On the Position tab, set the horizontal and vertical positions to be Absolute and relative to either Margin or Page.

Frames can be positioned similarly. On the right-click menu, choose Format Frame to get the dialog where you can choose Margin or Page.

How to file actionable feedback to the Windows Camera Team - Microsoft Community

tom a - windows virtual desktop team [msft] say:

How to file actionable feedback to the Windows Camera Team

Hello Insiders,

Tom from the Windows Camera Team here. I just wanted to send out a message to the Insider community to give some instructions on how you can file actionable feedback such that the Windows Camera team can investigate problems you are seeing with your camera on latest Insider builds of Windows. We process feedback on a daily basis and one of the biggest problems we face is feedback isn't actionable. Often cases, feedback is vague ("camera doesn't work") and Insiders are using devices we don't have access too, which makes investigation even more difficult. The best way for you to get your issue investigated is to include three key ingredients: any error message you are facing, screenshot(s), and please use the "Recreate the problem (recommended)" option so we can get logs from your device.

How to give actionable camera feedback:

The camera team wants to be able to take action on your feedback, in order to do so we need you to provide us with the following additional information.

1. Launch the Feedback Hub, type your issue in search box, search, if you found similar problem upvote and add to the existing issue, or file a new issue if a matching issue isn't found.

2. Mark the appropriate feedback type at the top with suggestion or problem radio buttons
3. Select the appropriate category
-      For camera device feedback [For example: Issues like camera doesn't start] select "Hardware, Devices, and Drivers" category and "Device Camera or Webcams" subcategory
-      For camera app issues [For example: Living images doesn't work] select "Apps and Games" category and "Camera" subcategory
-      If you are unsure, select "Hardware, Devices, and Drivers" category and "Device Camera or Webcams" subcategory
4. If issue is a problem (For example you get error message starting camera) provide us with the following additional information:
-      If there's any error, include error message in the title and text of your feedback
-      Include a screenshot of the problem, especially if there's an error
-      Use the "Recreate the problem (recommended)" option to reproduce the problem so we have logs to investigate your issue
-      Once you click "Start Capture", be sure the "Device Camera or Webcams (Default)" option is selected from the drop down.

-      Now, reproduce the problem, after return to the Feedback Hub and click "Stop capture" link, or the red "Stop capture" button

-      Other information that is really helpful is when the problem started occurring, if you just updated to build X, and your camera was working previously, let us know. Also include what app(s) you are experiencing problems with.

One last thing, if you search and find an issue that looks to match yours, you can add to this feedback by clicking on it, and selecting "Add feedback details." From there you can complete the steps mentioned above. Also, if you find existing feedback, be sure to read the Microsoft Engineer response if one's available, and add your own comments. 

Thanks!

Tom
