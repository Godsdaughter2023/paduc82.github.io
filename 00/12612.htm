    Win32/Medfos is a family of trojans that install malicious add-ons or extensions for Internet browsers to redirect its search engine results. It also allows for click-fraud (specifically Trojan:Win32/Medfos.X) to generate profit for a website through unethical means.

    If your computer has Win32/Medfos installed, your Internet searches may be redirected to a predefined website. This means that if you search the web using Google, Bing, or Yahoo, for example, the site returns normal search results; however, if you click on any of the results, you may be redirected to an unexpected/different website you did not intend to visit, instead of going to the correct/true website you're searching for (no need to tell you how dangerous this is).


    Trojan:JS/Medfos.B is MS's anti-malware products detection for a malicious JavaScript file (named "manager.js") packaged within a file named "chromeupdate.crx" that is usually dropped/installed by Trojan:Win32/Medfos.B, in the (in WinXP) "C:\Documents and Settings\<user>\Local Settings\ Application Data" folder. The file is a Google Chrome browser extension package that disguises itself as a legitimate Chrome extension. It may be visible in the browser by the extension name:

    'ChromeUpdateManager 1.0'.

    However, you must be aware that the MMPC Team have observed the "chromeupdate.crx" file also being dropped in computers that do not have Google Chrome installed.


    Further, please note that to enable redirections in Mozilla Firefox, this trojan also installs a Firefox extension named either 'Mozilla Safe Browsing 2.0.14' or 'Translate This! 2.0' or 'Mozilla Framework Assistant 3.0.1'. It is usually installed in the file "%LOCALAPPDATA%\(random CLSID)\chrome\content\browser.xul"; and it's detected by MSE as Trojan:JS/Medfos.A .- Please remove this malicious add-on (and other possibly 'unwanted' add-ons you may find) by following these instructions from Mozilla.


    To help MSE to clean your system completely (avoiding future repeting detections), you need to first try manual removal of above mentioned malicious files from their installation locations. Then do as follows (with all programs and browsers closed):

    - Delete the contents of your Temporary Internet Files Folder.

    - Clear the Java cache, AND Remove ALL VERSIONS of the Java plugin (and DO NOT reinstall it back!).

    - Download and run (Click on the 'Delete' button) AdwCleaner. Please advise us of what this tool removes.

    - Reboot, download and run the Microsoft Malicious Software Removal Tool.

    - Update MSE with the latest virus and spyware definitions and run a full system scan. Please note that as of this post, the latest version is now 1.145.1330.0 released today at 11:31 PM UTC, which shall include (all updates are cumulative) updated definitions for Medfos.B that were released with earlier definition version 1.145.1309.0.


    Additionally, you may also wish to download, install, update its database and run a full system scan of Malwarebytes Anti-Malware (Free-Edition ONLY! -> Not the Trial not the Pro version), and/or HitmanPro3 (as a second opinion malware scanner).
