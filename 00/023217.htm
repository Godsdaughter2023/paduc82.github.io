sororke say:

Hi Dani,

As far as I'm aware, Clear Signed email messages cannot be found using Outlook Search. The way the body of a clear-signed email is packaged through Outlook makes the text inside the email unsearchable.

The best workaround I've found is to set up a Search Folder that filters for encrypted and Clear Signed emails using the custom criteria: Message class is (exactly) IPM.Note.SMIME. Or you can use IPM.Note.SMIME.MultipartSigned to filter for just Clear Signed emails.

Then when you have to search manually for content in an encrypted email, at least you'll have a folder with only encrypted mail in it so there'll be fewer to search through.

Here's a full breakdown of how to set up the Search Folder: https://lookeen.com/blog/how-to-index-and-search-encrypted-files-and-emails

And a breakdown of the message classes you can use: https://msdn.microsoft.com/en-us/library/ee200767%28v=exchg.80%29.aspx?f=255&MSPPError=-2147217396

Hope this helps!

Siobhan
