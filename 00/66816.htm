SOLVED!

From my experience, that problem may arrise if you have 64bit Operating System.

I once had that problem and try everything from every forum out there.

Power options, reset power options, registry fixers, new driver from Pegatron (the issue, is from PHotkey, that is a Pegatron app/driver), Compatibility mode in every .exe of that folder and many others.

Nothing could solve the problem.

Then, from RevoUninstaller, I saw, that the PHotkey program it is a 32bit application. So, it will probabbly install a 32bit driver.

That alone, is enough, to be blocked from the 64bit driver signature service, of the Widnows system.

Every technician knows, that in some cases, the 64bit Driver Signature service, blocks a 32bit driver, without a warning info. I do not know is whitch cases, but it does it!

So I had one last shot.

To try with Driver Signature service to "allowed" mode.

So, I will detaile the steps needed. Just reply, in case it work for you.

###

Uninstall the PHotkey, to start from scratch.

Go to :

Settings -> Update and Security -> Recovery -> Advanced Startup and press Restart Now.

After the restart, the hidden boot options will apear. From there, go to :

Troubleshoot -> Advanced Options -> Startup Settings -> and Restart.

From the next menu, press the coresponding number in the "F" keys, that is near the "Disable Driver Signature Enforcement" line. It is usualy the 7. So press the "F7" key.

After the restart, go to the installation folder of the PHotkey.

Right click the Setup.exe and press Options or Settings. From the Comatibility Tab, select "Windows 7" and the "Run as Administrator" box, just to be sure. If you want, you can press the "All users" button, in the botton of the tab, to change the compatibility options, for all the users.

Start the installation and after the reboot, everything will work fine.

As an extra option, after the installation, I go to the install folder (usually in : Program Files (x86)/PHotkey) and set EVERY   .exe file, in Windows 7 compatibility mode. Just to be sure!

That solution, works for me.

If it works and for you, just reply here or spread the news, to help others.
