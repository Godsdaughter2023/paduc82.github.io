apostolos55 say:

Hi, 2 solutions, best one uses VBA (the 2nd)

1st  

     in a cell of small importance-out of the way, write =cell("address")

     then mark the area you want to monitor and apply conditional Formatting with the rule

     =CELL("address";B5)=$F$5  

     where for you B5 is the default/active cell when you mark your area, and $F$5 is the cell from                     previous step. Keep in mind to retain both $$ at your equivalent of $F$5

     then apply some formatting you like (a simple border outline might work, apply color etc)

     DRAWBACK of this solution is for the step one to work the cell you want must be activated, thing that      is a little tricky (double click, change value ect)

2nd

    In VBA (with Alt+F11) in the desired worksheet (ex: sheet1) paste the code:

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
Static prTarget As Range  ' SEE NOTES BELLOW for this line

If Target.Cells.Count > 1 Then Exit Sub
If Not prTarget Is Nothing Then
prTarget.FormatConditions(1).Delete
End If

Set prTarget = Target

Target.FormatConditions.Add Type:=xlExpression, Formula1:="=TRUE"
Target.FormatConditions(Target.FormatConditions.Count).SetFirstPriority
With Target.FormatConditions(1).Borders(xlLeft)
.LineStyle = xlContinuous
.Weight = xlThin
End With
With Target.FormatConditions(1).Borders(xlRight)
.LineStyle = xlContinuous
.Weight = xlThin
End With
With Target.FormatConditions(1).Borders(xlTop)
.LineStyle = xlContinuous
.Weight = xlThin
End With
With Target.FormatConditions(1).Borders(xlBottom)
.LineStyle = xlContinuous
.Weight = xlThin
End With
Target.FormatConditions(1).StopIfTrue = False

End Sub


Now just click anywhere. Notice that should you use extended conditional formatting there could be some inconvenience

Correction: remove Static prTarget As Range and instead paste it in a module at the top (even an empty module) like Public prTarget As Range as previously changing to other sheets the routine forgot to erase the formatiing...
