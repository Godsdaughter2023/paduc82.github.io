cbarnhorst say:

Windows 10 preview only accepts a Windows 10 product key.  Your Windows 8.1 product key only works with 8.1.

You have not lost your right to use the 8.1 product key to reinstall Windows 8.1 until you install and activate the free Windows 10 RTM.  That is one of the important meanings of "Evaluation copy."  Once you download and install the Windows 10 RTM, "Evaluation Copy" will disappear.

Once you activate the RTM copy of Windows 10 your 8.1 product key will be "consumed" by the Windows 10 license.  You may not reuse the old license (Product Key) after that.

The principle is that an upgrade is cheaper (in this case free) because you are using the retained value of the old license in getting the new one at a discount. 

Only if you are using a "full price" license (not an upgrade) can you also continue to use the old license somewhere else, if the license terms of your old copy allows it.  IF your 8.1 came preinstalled on your computer or is an upgrade license you bought when 8.1 released then it will die when you install the Windows 10 RTM. 

("Upgrade" has nothing to do with how you install Windows but has to do with licensing only.)

      SharePoint File Permissions - Microsoft Community

      ahaynes10 say:

      SharePoint File Permissions

      Is it possible to create a share link that anyone can view and allow specific people to edit? I need to be able to let anyone with the link view the file (but not edit) and allow my colleagues to edit.
      tim liu msft say:

      Hi Ahaynes10,

      It's not feasible to create one link that only can be view for some users and then can be edited with other users. However, it's easy to share the document with different users and let some of them can only view, the others can edit.

      I assume the file is stored in your OneDrive for Business library, select the file>click "...">Share (or click the Share icon when you hover mouse over a file)>now you get a sharing pop-up>click Anyone with the link can edit>uncheck the Allow editing setting options under Other settings section>click Apply>now it changes to Anyone with the link can view>click Copy link>Copy>send the link to others. Who get this link can only view the document.

      To share with your colleagues with edit permission, use the same steps mentioned above, but click Anyone with the link can edit and then select People in your organization, make sure Allow editing check box is selected, click Apply, Copy link to get the link, send the link to users who you want them have edit permissions.

      If the file is stored in SharePoint Online document library, as users may already have access to that library, so you need to set individual permission list for that file.

      1. Select the file, click "...">at the right-top of the sharing pop-up>click "...">manage access>click Advanced in the right panel.

      2. You will access the permission list of that file.

      3. Click Stop Inheriting Permissions.

      4. Select all user/groups in the list after you click the Stop button>click Remove User Permissions.

      5. Now click Grant Permission>enter emails of your colleagues>click Show Options>select Edit>Share

      6. These users will receive invitations for this file. They could access and edit the file.

      7. Use the same sharing method mentioned above to create Anyone can view link for other users who you don't want them to edit the file.

      Best regards,

      Tim
