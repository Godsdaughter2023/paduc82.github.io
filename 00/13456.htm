bill manville say:

To specify a CustomOrder parameter to Sort you need a CustomList number.

CustomLists are a property of the Application object.
You can check the number of a CustomList by providing the list as an array
 Dim iListNo As Integer
 iListNo = Application.GetCustomListNumber(Array("orange","apple","pear"))

If the current instance of Excel does not have that list iListNo will be 0
You can add a new list using
 Application.AddCustomList Array("orange","apple","pear")

Then the previous code will give the custom list number.
 iListNo = Application.GetCustomListNumber(Array("orange","apple","pear"))

You need to add 1 to this to get the customOrder parameter (because customOrder = 1 is used for the normal sort order).
  myRange.Sort ......, orderCustom:=iListNo + 1

When you have finished with a custom list you can delete it using
  Application.DeleteCustomList iListNo
