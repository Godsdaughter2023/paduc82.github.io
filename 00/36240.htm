axelbörjesson say:

I've run into this problem multiple times and I've found two ways you can get around it.

The first one is to find a program that simulates a virtual display (I used zonescreen, but I found this to be a pain in the *** in the long run)
Upside: software solution
Downside: was a pain to change / install / edit

The second is a much better solution (in my opinion) but requires some hardware, what you need to do is to create a dvi dummy plug, since people started bitcoining this got much more widespread use, and so there are plenty of guides (all you need is a vga-dvi adapter and 3 small resistors)
Upside: A simple plug that you can toggle into any dvi contact on any graphic card/motherboard, which I've never had fail. (and it's easy to unplug when necessary)
Downside, requires you to buy some resistors (super cheap thou).

It works as such: The dummy plug with a resistor creates a resistance on the red/green/blue cables, which makes the graphic card think that you have a vga display plugged in, and then you can force all the display settings you want.

Worked well both with windows 7 and now with windows 10.
