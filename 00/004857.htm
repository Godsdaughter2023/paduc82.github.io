suzanne s. barnhill say:

A "horizontal frame" cannot be placed in front of text, so you must have been using some other kind of frame. For your purposes, neither a table cell nor a frame will be appropriate; a text box is going to be necessary. That said, you will need to generate the entry in the List of Tables in another way. For a TOC, you could use a TC field, but for a List of Tables (if it is generated, like a Table of Figures, from caption labels), you will have to try another approach.

One possibility would be to insert the caption somewhere outside the text box (but on the same page) and make it invisible. Unfortunately, you can't do the simple thing and format it as Hidden because then the TOC field can't see it. It does still work (without affecting the formatting of the TOC entry) if you format it as white; you could also make it 1 point so that it would take up very little space.

Since you want the caption label that will actually appear in the TOC to have the right number (without messing up the sequence before and after), make the "invisible" caption the only real one. The one in the textbox should be typed and numbered manually, not using a SEQ field for the number. Alternatively, you could use the \c switch in the SEQ field of whichever caption appears second in the text.
