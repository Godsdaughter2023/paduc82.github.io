Under most conditions, a re-installation of Windows is not necessary to repair this problem. 

 

The file that you are having problems with (SOFTWARE) is one of the files that comprise the System Registry.   By far the easiest way to recover is to [on another machine] create a freeware Hirens Boot Disk:
    Hirens:  < http://www.hirensbootcd.org/download/ >

(Don't be fooled with ad-ware downloads, the correct download link is a ".zip" link in a light-green rectangle near the bottom of the page.)  Burning the CD so that it is bootable requires a special procedure that creates a CD from a .iso file (or use the "Burn Image" feature of your CD burning program).  General instructions for burning the CD are here:

    Hirens Burning: http://www.hirensbootcd.org/burning/


After you create the bootable CD,  boot your machine from this CD choosing the "Mini XP" option at start.  When it comes up, go to the Hirens Boot CD Menu and select "Registry Restore Wizard" somewhere under the registry tools area.  This will restore your registry from one of your restore points.   After that, shutdown, remove CD, and reboot.  All should then be well.

After you recover, be sure to perform a "System Restore" to make sure that everything in your system is synched-up.
  "How to restore Windows XP to a previous state"

  < http://support.microsoft.com/kb/306084 >

 

Since this problem is often caused by file system corruption, a Disk Check (with the "/F" or "/R" option) would also be in order:
"How to perform disk error checking in Windows XP"
  < http://support.microsoft.com/kb/315265 >

Note that, depending on how lucky you are, the Disk Check can in an of itself fix your problem -- although it often takes upward of a couple of hours to run.  (You can run a Disk Check after booting from a Hirens CD)


Should all else fail, you can use the Hirens disk to back up your important files before re-installing Windows.  Simply drag-drop copy the files off to a USB or Networked drive.
