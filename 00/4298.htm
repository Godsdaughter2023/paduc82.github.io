It appears that you are inserting the cross-reference into text that is TNR 12, so, as Stefan says, it should take on this formatting assuming that is the default paragraph font. If that font has been applied to the paragraph as direct formatting, then this might not be the case. If you want TNR to be the default font of your document, you need to change the default font, not just apply that font to the text.

OTOH, if the caption itself has been changed with direct font formatting, then you need to modify the Caption style so that the caption has different formatting from the rest of the text without direct formatting since the direct formatting may be picked up by the cross-reference.
