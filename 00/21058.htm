Sub AAA()
    Dim S1 As String
    Dim S2 As String
    S1 = Workbooks("Book1.xlsm").Worksheets("Sheet1").Range("A1").Text
    S2 = Workbooks("Book2.xlsm").Worksheets("Sheet1").Range("A1").Text
    Select Case StrComp(S1, S2, vbTextCompare)
        Case 0
            MsgBox "Strings are equal"
        Case 1
            MsgBox "String1 is 'greater than' String2"
        Case -1
            MsgBox "String1 is 'less than' String2"
    End Select
End Sub

wandy_a say:

For me, fast startup was a real pain from the first unexpected power on. 

My experience and a possible workaround.

Fast Startup

 

Unexpected Power On

 

Fast startup had been a difficulty through the early involvement project.  A 'hidden' setting defaulted to on was intended to speed up the return from Hibernate or Sleep.  Here  it would power on any keystroke including the double tap of the 'control' key to access the KVM switch.  Unexpected power on was the norm even if all BIOS wakeup settings were disabled and no communications cables were attached. Forgetting to turn this off would eventually end in a reinstall.  The effects were even worse on a dual boot system.

This 'feature'  may have been of some use to someone however here it was a real problem.  Remembering to turn it off after an install was an imperative.

 

Resolution,  Turn off Hibernate.

 

There are some interesting thing things about the ‘real’ version.

One example, my nemesis, Fast Startup isn’t even there!

It was there on insider builds and would “burn the machine to the ground” if I forgot to turn it off.

Good riddance.

 

You should have the fast Startup feature though... all of my machines do...

is it possible that some where along the line you turned off Hibernation with the Powercfg -h off command ?

Powercfg -h on turns it back on...

I can't really think of any other reason it wouldn't be there, but there most certainly might be some...

 

Hibernate

 

Currently all desktops have hibernate turned off.  At the moment there appears to be no downside as sleep functions normally. 

 

  • Saves approximately 6 Gig of disk space.

  • Fast Startup goes away !
