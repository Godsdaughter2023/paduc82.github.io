Original Title: New laptop locked 

rajesh govind say:

Hi,

 

Thank you so much for posting your query on this forum. You have reached the right place and I will try my best to assist you with this issue.

 

 

The reason why you get this error message is because the Microsoft account password stored in the local cache on the computer is still the old one and not updated.

 

In Windows 8/8.1, when you login to the computer for the first time using a Microsoft Account, you need the Internet Connection. At this time it creates a local cache on your computer to store the account login credentials. Thus, after this, it is not necessary to have the Internet Connection to login to the computer using the Microsoft Account as long as you are using the old password.

 

Now, when you change the password of your Microsoft Account from another computer and try to login with the new password on this computer without an Internet connection, then it does not recognize the recent change made on the password and shows that the system is offline.

 

To resolve this issue make sure that your computer is connected to the Internet and that the connection is working properly when you try to login to the computer with the new password. Thus, when the computer finds that the new entered password does not match with the old one stored in the local cache, it connects to the Microsoft Account server to check if there were any changes made to the old password. If yes, then it replaces the old password stored in the local cache on the computer with the new one. Thus, the next time it permits the user to login to the computer with the new password, even if there is not Internet Connection at the time of the login.

 

Hope this is helpful. Please get back to us with the updates.

For anyone with grayed out icons that won't show up, open your Task Manager. You can either Ctrl+Alt+Del and select "Task Manager" or right click on the taskbar and select Task Manager. They open the same window. In the "Processes" tab, scroll all the way down to the Windows processes. Towards the end of these, there is one called "Windows Explorer" that has an icon next to it that looks like a folder. This is the process "explorer.exe". Instead of ending the process and going through the steps to start a new task and bring it back, just right-click on Windows Explorer (explorer.exe) and click "Restart". Fewer steps and your icons should re-appear.

Anyone with input on this subject, feel free to post your solutions that you've found. This is the best solution I have found that's easy and that works.

To uninstall device

a. Click on Start menu, type Device Manager and hit Enter.

b. The Device Manager will then display a list of installed devices in the right pane of the console.

c. Expand the category Network Adapters to uninstall the Wireless Adapter.

d. Right-click the device and select Properties.

e. Select the Driver tab on the device's Properties dialog box.

f. Click the Uninstall Driver button.

A dialog box will pop up asking you to confirm ‘Uninstall’. Click OK to proceed.

How to obtain a new product key for a Microsoft Office product


How to replace a damaged or lost product key for a full version If you cannot find your product key, you can obtain a new one. To obtain a new product key, contact Support. Find the Support telephone number for your area by running the Product Activation Wizard on your computer. For more information about how to run the Product Activation Wizard, click the following article number to view the article in the Microsoft Knowledge Base:
950929  (http://support.microsoft.com/kb/950929/ ) How to contact a Microsoft Product Activation Center
When you contact Microsoft for a new Office product key, please have the following items available for the customer service representative:
  • The product CD-ROM.
  • The computer on which you want to install Office.
  • The readable parts of your original product key (if available).
  • The ProductID information in the registry (for advanced users only).
