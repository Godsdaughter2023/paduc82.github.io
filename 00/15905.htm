Normally, if you do a Copy and then Paste Special/Formats, any Conditional Formatting will be copied along with normal formatting.

You have to be careful about this if the CF is based on formulas - check the CF formulas to make sure that any cells that should have an absolute reference (e.g. $A$2 is absolute, A2 is relative) do, and that cells that should "float" in either row or column directions do.
