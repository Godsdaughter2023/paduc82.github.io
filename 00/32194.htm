brendacambridge say:

how to swap between accounts in windows 8

My husband and I have an account each in Windows 8, how do we switch between account without shutting down? Would be grateful

for any suggestions please

a. user say:

HI BrendaCambridge,

Thank you for posting in Microsoft community!

I will definitely help you with this.

If you have more than one user account on your computer, Fast User Switching is an easy way for another person to log on to the computer without logging you off or closing your programs and files. For instructions on creating user accounts, see Create a user account.

1.       Click the Start button , and then click the arrow next to the Shut Down button .

2.      Click Switch User.

3.      Press Ctrl+Alt+Delete, and then click the user you want to switch to.

Note:

If you are logged on to a remote computer (for instance, using Remote Desktop Connection), you can't use Fast User Switching on that computer.

Disclaimer:

Make sure to save any open files before switching users because Windows doesn't automatically save files that are open. If you switch to a different user account and that user shuts down the computer, any unsaved changes you made to files that are open on your account will be lost.

brian tillman [outlook mvp 2007-2019] on directly say:
You neglected to attach any images.

The symptom you describe about the autocomplete is a sign that your mail profile is corrupt. The thing to do to fix is to create a new mail profile using the Mail applet in Control Panel, add your account to that profile (you do not have to delete the existing profile), then test again as you describe. If the nerw profile works, you can remove the old profile.

What type of account do you have configured? Is it Exchange, IMAP, or POP? If it's POP, you'll be able to reuse your PST in the new profile. When you add the account, there will be an option to reuse the existing PST. If you have an IMAP account, you will not be able to reuse the PST where the mail is delivered, since Outlook will create a new one, but you will be able to reuse the PST containing the non-mail folders. You shouldn't need the mail PST because your mail is kept on the server and Outlook will populate the new PST it creates with the mail on the server. For an Exchange account, you won't need to reuse the OST, but you shoudn't need to because all your data, except for any you keep in a local PST, is on the server. If you do use a local PST for holding local data, you'll be able to reuse it.
