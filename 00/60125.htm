REPAIR THE INSTALLED WINDOWS IMAGE
1. Open the Command Prompt (Admin) or PowerShell (Admin).
• Execute dism.exe /Online /Cleanup-Image /CheckHealth to check whether the image has been flagged as corrupted by a failed process and whether the corruption can be repaired.
• Execute dism.exe /Online /Cleanup-Image /ScanHealth to scan the image for component store corruption.
• Execute dism.exe /Online /Cleanup-Image /RestoreHealth to scan the image for component store corruption, and then perform repair operations automatically.

VIEW & EXPORT INSTALLED DRIVERS

Replace Path with the location where you want to save the drivers.

IMPORT & EXPORT FILE EXTENSION TYPE ASSOCIATIONS

¡firedog say:

Windows Live Mail is a mail client program like many others whose primary function is to download messages delivered to a mailbox hosted by an Email Service Provider (ESP). It can do this in several ways, depending on what the ESP allows.

  • Most mailboxes allow for POP3 access. This is the oldest and simplest way of accessing a mailbox; the client asks the server where the mailbox is hosted whether there are any new messages in the inbox, and if there are, downloads them and displays them in the account inbox within WLMail. You can specify whether a copy of each downloaded message should be kept on the server and give instructions for removing them from the server to keep within the allotted storage space.
  • Some ESPs allow for access via IMAP. This arrangement keeps all messages on the server and downloads incoming messages for viewing in WLMail. It depends on the account settings whether WLMail keeps a copy of incoming messages or just a record of their headers.
  • Hotmail-type accounts can be accessed via DeltaSync, which acts in much the same way as IMAP. It aims to keep the contents of all folders in the mailbox synchronized with those in the client. All messages are saved in WLMail's folders.
  • WLMail has a unique extra feature: photo email. If you want to share large pictures, or many pictures at once, you can use this feature which instead of overburdening the email system uploads pictures to your SkyDrive. The mail message you send contains only thumbnail images of the pictures, so the recipient can view them first and then decide whether to download them or not.

To see what type a mail account is, right-click on its name in the folder pane and select Properties. The type is given in the top box on the Server(s) tab.

The answer to your question, then, is No. Windows Live Mail doesn't back up your messages in the cloud or anywhere else. Messages may be kept on the ESP's server, but this will not be the case for

  • messages in folders other than the inbox for POP3 accounts
  • messages saved in Storage folders but not in an account folder.

There are ways of automating a backup to SkyDrive, but I don't know of any accessible step-by-step instructions for doing so. I might write some...
