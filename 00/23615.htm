What happens if I delete an instance of a sync'd Onenote notebook - Microsoft Community

petergoldstein say:

What happens if I delete an instance of a sync'd Onenote notebook

I'm syncing Onenote between sources.  If I take one of those sources offline, will it effect any of the other sources.  For example, if I delete Onenote or a notebook from a PC version of the app, will this remove that content from the sync'd devices?

Thanks for any insights!

swischner say:

If you delete a page or section of a notebook on one device, that's a change to the notebook that will get synchronised to OneDrive and from there to the other devices. So the page or section will be deleted on the other devices as well.

But. You can't delete a whole notebook from a device because it's simply not stored on that device. The notebook "lives" in the cloud, so on OneDrive, OneDrive for Business or SharePoint Online. To delete a notebook, you'd have to use a web browser and delete it directly at its storage location.

On the devices there is a local copy of all notebooks that are currently opened on the particular device. That copy is called the OneNote cache and is in a special, fragmented format. If you remove OneNote or close a notebook on a device, that cached copy is erased from that device only. The actual notebook data on OneDrive remains untouched. At least this is how OneNote is designed to work.

Now, if you erase a notebook from the cloud storage using a browser but did not close it on a device first, the cached copy on that device remains intact; you can still view and even edit the notebook. But you only see and edit the cached copy. As soon as the OneNote app on that device tries to sync the notebook to the cloud, you will get an error, because the "real" notebook does not exist anymore.

I hope, that explanation is not too complicated (well, the OneNote sync mechanism is).

Some more about this on my blog: 

http://onenote-blog.com/the-onenote-cache-and-offline-notes/
