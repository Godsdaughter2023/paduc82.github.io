Maybe we need a user defined function, UDF. Alt +F11 to open VB editor. Right click ThisWorkbook and insert module and paste the code below in.

Back on the worksheet call the function with

=ccif(A1:A5,C1,B1:B5)
Where C1 is the lookup value

or

=ccif(A1:A5,"USA",B1:B5)
With the lookup value in the formula

 

 

Function ccIF(SrcRng As Range, SrcVal As String, ConCatrng As Range)
SrcVal = UCase(SrcVal)
For Each c In SrcRng
    If UCase(c.Value) = SrcVal Then
    ccIF = ccIF & c.Offset(, 1).Value & ","
    End If
Next
ccIF = Left(ccIF, Len(ccIF) - 1)
End Function

Definitely more information is needed to determine specifically what infection you are dealing with since there are so many variants of crypto malware (file encrypting ransomware).

Are there any obvious file extensions appended to or with your encrypted data files? If so, what is the extension and is it the same for each encrypted file or is it different? Some types of ransomware will completely rename, encrypt or even scramble file names while others do not append any extensions.

These infections are created to alert victims that their data has been encrypted and demand a ransom payment. File encrypting ransomware typically will drop a ransom note in every directory/affected folder where data has been encrypted, although some may drop a note that displays upon computer startup. The ransom notes are often created in multiple file formats (.txt, .html, .png, .bmp, .url) to ensure that the victim can open them and read the ransom demands/instructions. Check your documents folder for an image the malware typically uses for the background note. Check the C:\ProgramData (or C:\Documents and Settings\All Users\Application Data) for a .html, .txt, .png, .bmp, .url file.
