chip pearson say:

I think the source of your problems is that you are not properly using the With statements. As an example, you have

 

    
    With Worksheets("UAT")
        NextRow = MatchRow
        Cells(NextRow, "A").Value = Me.txtTsk.Value
        Cells(NextRow, "B").Value = Me.txtRmdy.Value
        Cells(NextRow, "C").Value = Me.DTPicker1.Value

While this is syntactically correct and compiles cleanly, it is not doing what you think. The Cells function doesn't refer to worksheet UAT. It refers to whatever sheet happens to be active.  This is so because you have omitted the period in front of Cells. It is the period that links the line of code to the object in the With statement. Modifying the example above, you would write

 

    
    With Worksheets("UAT")
        NextRow = MatchRow
        .Cells(NextRow, "A").Value = Me.txtTsk.Value
        .Cells(NextRow, "B").Value = Me.txtRmdy.Value
        .Cells(NextRow, "C").Value = Me.DTPicker1.Value

 

Note the period in front of the Cells function. Examine all your with statements and ensure that you have the periods properly placed.

Avery template downloaded to word 2007 - Microsoft Community

karenseiler say:

Avery template downloaded to word 2007

How do I delete an Avery template that was downloaded to microsoft word 2007?  Now when I try to open a document I get "fatal error".  When I open a new document the avery template is in it and I cannot type a document.

 

jay freedman say:
It only looks like the Avery template is in the new document. What has happened is that the content of the Avery template has been saved into your Normal.dotm template, which is used for creating new "blank" documents. Usually the advice is to open the Normal.dotm template and clean out the unwanted text. The fact that you're getting fatal errors when opening other documents, though, means that more serious damage has been done to Normal.dotm.

The following steps will create a new Normal.dotm with the factory settings:

  • With Word shut down, type the following line into the address bar of Windows Explorer (the "Computer" app):

%appdata%\Microsoft\Templates

      and press Enter.
  • Locate the Normal.dotm file (not NormalEmail.dotm) and rename it to OldNormal.dotm.
  • Restart Word.
  • If you had customized styles or macros in the previous Normal template (now OldNormal.dotm) that you want to keep, click View > Macros and click the Organizer button. One side will show Normal.dotm and the other side will show the current document (which should be Document 1). Click the Close File button under the document; when it changes to Open File, click it again. Go to the %appdata%\Microsoft\Templates folder and select OldNormal.dotm. Select the macros in OldNormal and click the Copy button; repeat this on the styles tab of the dialog.
  • Shut down Word. If you're prompted to save changes to Normal.dotm, say yes. That will create the new Normal.dotm file.
One other thing: In most cases you really don't need to download templates (which are actually documents, not templates) from the Avery site, because the label formats are already in Word's Labels dialog. In those cases where the label's number isn't listed in Word, the Avery site will tell you the numbers of other labels that have the same layout, and you can usually find one of those listed in Word.
