Issues Calling Skype Numbers

Hi all,

Over the last few weeks, many of you have told us you are having issues with Skype Numbers. We are looking into the causes, but i wanted to collect them together in one thread along with some workarounds.

Calls to Skype Numbers from Vodafone UK


this issue has now been resolved, if you have any further issues please post in the community. 

Call to Skype Numbers that ring once then fail with a "Busy" tone

In almost all cases of this we have investigated, this is caused by the server settings for your account not matching what is being shown in some of our clients. We have a workaround for this: please toggle the setting for "allow calls to my Skype number from..".  See this FAQ for details:

Why am I not receiving calls to my Skype Number?

Use this setting on Windows 10:

On Skype for Web:

The new clients on iOS and Android do not currently have a setting for this, so I am afraid you will need to use one of the options above.

Calls going straight to voicemail

This could be for one of the reasons above, so please check your setting using the workaround above. There are other reasons for this to occur, which I will explore below.

If you have been using an ATA device or a Dualphone, then these devices are no longer supported and as a result your calls will be diverted to voicemail or a busy tone will be played. There is no workaround for these devices except to use a supported platform.

Check your settings in the Skype portal for the duration before calls are forwarded to voicemail / call forward settings. Note: settings on the website will override what is displayed in the clients and the settings in the clients will, eventually, be retired.

There has been a temporary problem messages

For the most part these are caused by provider errors and the problems will correct themselves after a short period of time. There is no need to report them to the Community or Skype Support, unless the problem has lasted for a number of hours.

Placing calls to or from the UAE, Egypt and some other countries

Users continue to report problems calling to or from the UAE and Egypt - these problems are being caused by ISPs in these countries attempting to block Skype calls. bernd p. say:

There are two types of notebooks you can create:

Local notebooks

They are stored on your PC, implemented as folders within the Windows file system, containing sections as files of type .one.

Notebooks on OneDrive

They are stored on OneDrive. On the OneDrive Web site you only see the notebook, the sections are hidden. In the OneDrive folder on your Windows you only see a URL as a placeholder for the notebook.

BTW:

The OneDrive software on your PC isn't used by OneNote which has its own sync software.

All other Office files need the OneDrive software for syncing.
