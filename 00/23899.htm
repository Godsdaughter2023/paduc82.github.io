bob jones aka: cybertaz say:

Well, there actually are more factors involved than what you might think :-) Let's eliminate one right off the bat: Dictionary...

Office in general, Word specifically, selects the Proofing Tools (dictionary, thesaurus, grammar, etc.) that need to be used, the user does not 'pick' one. The program uses the appropriate lexicon based on the Language attribute of the text being checked & switches on the fly as necessary. IOW, if the document contains text marked as Italian & other text marked as English, the program will use the correct proofing tools for each body of text. If Office does not have the necessary tools for the specified language that aspect of the text is not checked at all. [What can be considered an exception (but really isn't) is that you also can have Custom Dictionaries to which a language attribute can be ascribed. See this comprehensive article for more information: Spelling, Language, Dictionary

********************************************

To verify your current Default Language, launch Word directly (either using the Dock icon, the program icon in the Applications folder, or close any open documents create a New Blank Document. Then go to Tools> Language -- your Default Language will be highlighted. If it isn't what you prefer, select the correct one, click the Default button & confirm updating your Normal template.

Another fact to keep in mind is that the Default Language specified for Word pertains only to new documents generated from Word's Normal.dotm template. If a document is generated from a template other than Normal.dotm the Language of the new document will be determined by the default language of the template on which the document is based. Further, simply choosing a different Default Language in an existing document does not change the language(s) used in that document. For existing documents you must either select the text & specify the preferred Language or you must modify the Styles used in that document to include the preferred Language.

As for the items offered in the Document Gallery [File> New from Template]:

  • With Office 2016, most of the supplied templates do not use Lorem Ipsum placeholder text. Instead, they are localized based on the System Preferences> Language & Region settings of the Mac from which they're being accessed.
  • If the Mac's Preferred Language or Region is set to Italian/Italy, the text of the templates will be in Italian, regardless of the user's Default Language specification.
  • As indicated above, the Language of the documents generated from those templates is determined by that individual template regardless of what is specified for new documents in your Normal.dotm template.

There's no way to add individual characters to the QAT without a macro. You can add the Insert Symbol button, which quickly opens the dialog. But for this purpose I'd advise using keyboard shortcuts instead. Indeed, if the "Spanish symbols" you refer to are simply accented characters such as á and ñ and ç, there are already built-in shortcuts. These are listed at http://WordFAQs.mvps.org/InsertSpecChars.htm, but you can also see the shortcut for any given symbol by selecting it in the Symbol dialog and looking at the shortcut shown at the bottom next to the Shortcut Key... button. If there is no shortcut assigned (that is, only the Unicode value is given), you can click the button to open the Customize Keyboard dialog and add one.
