jim gordon mvp say:

With regard to this part of your question "make sure Microsoft Word works with this version of Max OSX"...

This message displays when you try to open a version of Word that was made for PowerPC on an Intel Mac that does not have Rosetta running. That would be most Intel Macs these days. It means when you tried to open a Word file by double-clicking it in Finder, Mac OS was set to open an old copy of Word that doesn't run on your hardware in Mac OS anymore. 

The cure is to right-click on the Word document in Finder. Choose Get Info from the pop-up menu, and in the Get Info dialog look for Open With. Change Open With to use Word 2008 or Word 2011, and also click the Change All button.

Then do a Spotlight search for the word old and see if you have old system folders and applications folders you don't need any more. Make archive copies if you want, and then delete old software from your hard drive so that it doesn't take up space.

First, double-click the header area on any page in the section after the section break. On the Header & Footer Tools tab of the ribbon, click the Link to Previous button. Changing contact cell phone number. - Microsoft Community

janetbollin say:

Changing contact cell phone number.

I would appreciate information on how to change the listed contact cell phone number on my MicroSoft account and replace it with the new number I just received.
regile reb say:

Hello Janet,

You may update the phone number associated to your Microsoft account by following the steps below:

  1. Sign in to your Security basics page with your Microsoft account.
  2. Select Update info.
  3. You may be asked to enter a security code to continue. If you don't have access to your alternate email or phone number, choose I don't have any of these and follow the instructions to replace your security info.
  4. Select Add security info then follow the instructions to enter a new email or phone number.
  5. We'll send a security code to that new email or phone number. Enter that security code, and select Next. Your new security info will appear in your list.
  6. Choose Remove for any you want to remove.
  7. If you want to update which security info we'll send alerts to, select Change alert options and deselect the checkbox for the one you no longer want to receive alerts for, unless they're critical.
