hansv mvp say:

Here is a function I use; it was originally written by Arvin Meyer:

Public Function WorkingDays(StartDate As Date, EndDate As Date) As Long
    '....................................................................
    ' Name: WorkingDays
    ' Inputs: StartDate As Date
    ' EndDate As Date
    ' Returns: Long
    ' Author: Arvin Meyer
    ' Date: May 5,2002
    ' Modified slightly
    ' Comment: Accepts two dates and returns the number of weekdays between them
    ' Note that this function has been modified to account for holidays.
    ' It requires a table named tblHolidays with a field named HolidayDate.
    '....................................................................

    Dim lngCount As Long
    Dim dtmCurr As Date
    Dim rst As DAO.Recordset
    Dim dbs As DAO.Database

    On Error GoTo ExitHandler

    Set dbs = CurrentDb
    Set rst = dbs.OpenRecordset("tblHolidays", dbOpenSnapshot)

    ' Use the following line to include StartDate
    ' dtmCurr = StartDate
    ' Use the following line to exclude StartDate
    dtmCurr = StartDate + 1

    lngCount = 0

    Do While dtmCurr <= EndDate
        If Weekday(dtmCurr, vbMonday) < 6 Then
            rst.FindFirst "[HolidayDate] = #" & Format(dtmCurr, "mm/dd/yyyy") & "#"
            If rst.NoMatch Then
                lngCount = lngCount + 1
            End If
        End If
        dtmCurr = dtmCurr + 1
    Loop

    WorkingDays = lngCount

ExitHandler:
    On Error Resume Next
    rst.Close
    Set rst = Nothing
    Set dbs = Nothing
End Function

Use WorkingDays(Date(), Due) to get the number of workdays from today to the due date.

Make sure you are in the Fast Ring, then try this:

- Open the command prompt as administrator and type the following:


net stop wuauserv - (this will temporarily disable Windows Update).


- Now open File Explorer and navigate to C:\Windows;

Delete the folder "SoftwareDistribution".


- Open the command prompt again and type:


net start wuauserv (this will enable Windows Update).


- Wait. The build should show up in a few hours.
