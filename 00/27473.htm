doug robbins - mvp office apps & services (word) say:

Each Section of a document can have:

  • A First Page Header (appears only on the first page)

and

  • A primary header (appears on all pages in the section except for the first page

or

  • An Odd Page Header that appears on the Odd numbered pages except on the first page, if that option is set
  • An Even Page Header that appears on Even numbered pages

If the Section is set to have a First Page Header, the content of the other Headers will not appear UNTIL, those pages are present in the Section.  So you may need to add the necessary number of pages to the Section to be able to see, and hence edit the header.

stat_geek say:

How can I get word to recognize formatted text and apply relevant styles?

Hi,

I am exporting data from another program out to a word document that includes tables and texts.  I can control the text formatting to be the same as the style definitions (ie Arial, 16pt, Bold, Left Aligned) which would be Header 1. However word does not recognize these as styles so I can't use these to create a Table of Contents automatically.

Is there a way to make Word recognize these texts formats as a specific  styles and convert them over?

 

Currently using Office 2003 though info on 2007 and 2010 will be useful as well.

Thanks,

Reeza

jay freedman say:

The thing you're looking for is part of the AutoFormat feature.

In Word 2003, first go to Tools > AutoCorrect Options > AutoFormat. Uncheck everything except "Built-in Heading styles" (and maybe the other settings in the Apply group -- you'll have to experiment to see what these give with your particular document). In Word 2007/2010 the equivalent dialog is at Office button (or File) > Word Options > Proofing > AutoCorrect Options > AutoFormat.

Open a copy of the document (preserve the original in case you have to revert). In Word 2003, click Format > AutoFormat, accept the default "AutoFormat Now", and click OK.

In Word 2007/2010, the AutoFormat command isn't on the ribbon, so you need to add the button for it to the Quick Access Toolbar (or in 2010 you can add it to a custom group on the ribbon). You'll find the AutoFormat and AutoFormat Now commands in the Customize dialog, in the Commands Not In the Ribbon category.
