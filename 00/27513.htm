solemnwishes say:

Unique entries in a range

I would like to find how many unique entries in a predetermined range.

For example

1   2   3   3   3   3   4   5   4   6   3

has 6 unique entries.

steve yandl say:

This user defined function is one option.

Function uniqueVals(myRange As Range)
Set dicUniq = CreateObject("Scripting.Dictionary")
For Each myCell In myRange.Cells
    If Not dicUniq.Exists(myCell.Value) Then
        dicUniq.Add myCell.Value, myCell.Value
    End If
Next myCell
uniqueVals = dicUniq.Count
Set dicUniq = Nothing
End Function

 

Steve Yandl

Yes Steve, I've just done that (see below). Maybe the op has not used UDFs before so:

 

Press ALT + F11, choose Insert, Module and paste the following code into the module.


Function uniqueVals(myRange As Range)
'Steve Yandl
Dim dicUniq, myCell As Range

Set dicUniq = CreateObject("Scripting.Dictionary")
For Each myCell In myRange.Cells
  If Len(myCell) > 0 Then
    If Not dicUniq.Exists(myCell.Value) Then
        dicUniq.Add myCell.Value, myCell.Value
    End If
  End If
Next myCell
uniqueVals = dicUniq.Count
Set dicUniq = Nothing
End Function

 

Now close the Visual Basic Editor (Ctrl + Q)

Enter the formula as normal in your workbook. It is a bit messy to use it in another workbook.

peter schleif say:
The following array formula has no problems with blank cells and is not limited to numbers. Please adjust the last cell A1000 if necessary.

   =SUM(IF(A1:A1000<>"",1/COUNTIF(A1:A1000,A1:A1000)))

NOTE:  Enter formula with <CTRL> <SHIFT> <ENTER>


The VBA function allows you to specify multiple areas and even entire columns or rows:

   =CountUnique(A1:A10)
   =CountUnique(A:A,5:5)
   =CountUnique(B5:C7,D8:E10,F11:G13)

Copy the code below and paste it in a standard module. If you are not sure how to do this, please follow these instructions.
http://office.microsoft.com/en-us/excel-help/create-or-delete-a-macro-HP010014111.aspx#BMcreatemacrovba

Peter


Function CountUnique(ParamArray areas() As Variant)
   Dim r As Long
   Dim c As Long
   Dim a As Variant

   With CreateObject("Scripting.Dictionary")

      For Each a In areas

         If TypeName(a) <> "Range" Then
            CountUnique = CVErr(xlErrRef)
            Exit Function
         End If

         a = a
         For r = LBound(a) To UBound(a)
            For c = LBound(a, 2) To UBound(a, 2)
               If Not IsError(a(r, c)) Then
                  If Trim(a(r, c)) <> "" Then .Item(CStr(a(r, c))) = "foo"
               End If
            Next
         Next

      Next

      CountUnique = CLng(.Count)

   End With
End Function
