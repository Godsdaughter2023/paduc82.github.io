hansv mvp say:

You could use a custom format such as "_ * #_ " without the quotes. That is:

  • an underscore followed by a space
  • an asterisk followed by a space
  • a hash
  • an underscore followed by a space

Since this is a number format, the number will be right-aligned.

 

If you want to center the number, you could use a custom format

"    "General"    "

this time including the quotes. Center the cell and apply standard underlining.

Use as many spaces between the quotes as needed to get the effect that you want.

The downside is that this depends on the column width - if you change the column width, you'll have to change the number of spaces too...
