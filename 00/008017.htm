davecalgary say:

This problem has been resolve.  I waited a few days and have not been able to reproduce the problem so I think it has been resolved (99% confidence).

Steps:

1. Log into system using Administrator account

2. Open Computer Management | Device Manager interface.

3. Expand the Universal Serial controllers tree structure.

4. From the right side, use the More Functions and choose View | Show Hidden Devices.  This will show all the USB devices that have been attached to the unit.  From here, I could not tell which was the problem device so I just plugged in the USB device that caused me the problems.

5. When the USB drive is connected, use the USB disconnect utility (toolbar at the bottom) to disconnect the USB drive.  Leave the drive in the port.  Once the drive is available for removal, the USB port will be identified in the Universal Serial Bus controller interface (it will have an icon indicating that it is connected but cannot be used).

6. My first attempt to resolve this problem was to update the driver.  There is an option to update the driver from the right mouse click menu.  For me, the driver was up to date.  This did nothing for me.

7. Using the right mouse click menu, choose uninstall and remove the USB driver.

8. Reboot the unit.

I hope that this helps others who may encounter this problem.  This solved my "MACHINE CHECK EXCEPTION" issue.

marco bartz say:

Hi,

On the network I manage people all have local administrator privileges so running office as administrator wouldn't change anything.

In my case I managed to resolve this issue by removing the user profile settings for Office.

------------------------------

Before you remove the office settings you may want to backup the Outlook profile

Open regedit - Start and Run..

Type regedit and press enter.

Navigate to the following key:

HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Outlook

Right click on the branch and select "export"

Delete the following registry branch:

HKEY_CURRENT_USER\Software\Microsoft\Office\15.0

Right click on it and select "delete"

Office will treat you like a new user and recreate the registry keys next time it loads. 

Load Word or Excel.

Run through the wizard.

Now you may want to import the outlook key (in regedit menu select File, import.. and the file you exported earlier) if you want your profile information back.

---------------------------------------------

Should all be working normally now.

Hopefully that fixes your issue!
