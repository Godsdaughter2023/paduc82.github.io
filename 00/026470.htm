Private Sub Worksheet_Calculate()
    Dim Rng As Range, rCell As Range

    With Me

        With rCell
            If .HasFormula Then

            End If
        End With
    Next rCell

I have 2 options for you to test.

The first option simply unhides any previously hidden row if the sum of the values in the row not zero.

The second option Hides or Unhides the row depending on the sum of the values in the row. (Sum of values not equal zero then Unhides row;  Sum of values equal zero then Hides row.

To install code:

  1. Right click the worksheet tab name and select View code
  2. Copy the code from one of the options below and paste into the VBA editor
  3. Alt and Q will close the VBA editor
  4. Save the workbook as Macro Enabled
  5. Ensure Options set to enable macros. See help for how to do this. Setting the option to "Disable all with notification" should be OK.

Note: You might need to disable the code while you are developing the worksheet. To do this remove the leading single quote from the Exit Sub line immediately after the Sub name. Restore the single quote to re-enable the sub. (Don't remove the single quote in front of the comment after End Sub.)

Option 1 code: Unhides previously hidden row if sum of values in the row not equal to zero.

Private Sub Worksheet_Calculate()
    'Exit Sub       'Remove the leading quote from this line to disable sub
   
    On Error GoTo ErrHandler
    Application.EnableEvents = False
    Application.ScreenUpdating = False
   
    Dim cel As Range
   
    'In following line Me is the generic object of the worksheet
    'to which this VBA code module belongs
    For Each cel In Me.UsedRange.Columns(1).Cells
        If cel.EntireRow.Hidden Then
            If WorksheetFunction.Sum(cel.EntireRow) <> 0 Then
                cel.EntireRow.Hidden = False
            End If
        End If
    Next cel
   
ErrHandler:
    If Err.Number <> 0 Then
        MsgBox "An error occurred in following:" & vbCrLf _
                & "Worksheet name: " & Me.Name & vbCrLf _
               & "Private Sub Worksheet_Calculate()" & vbCrLf _
                & "Advise Administrator of this application."
    End If
    Application.ScreenUpdating = True
    Application.EnableEvents = True
End Sub

Option 2 Code: Either Hides or Unhides row depending on the value of the sum of values in the row.

You can only have one or the other of the options; NOT both. If you copy both into the editor then the code will error with ambiguous name.

Private Sub Worksheet_Calculate()
    'Exit Sub       'Remove the leading quote from this line to disable sub
   
    On Error GoTo ErrHandler
    Application.EnableEvents = False
    Application.ScreenUpdating = False
  
    Dim cel As Range
   
    For Each cel In Me.UsedRange.Columns(1).Cells
        If WorksheetFunction.Sum(cel.EntireRow) <> 0 Then
            cel.EntireRow.Hidden = False
        Else
            cel.EntireRow.Hidden = True
        End If
    Next cel
   
ErrHandler:
    If Err.Number <> 0 Then
        MsgBox "An error occurred in following:" & vbCrLf _
                & "Worksheet name: " & Me.Name & vbCrLf _
               & "Private Sub Worksheet_Calculate()" & vbCrLf _
                & "Advise Administrator of this application."
    End If
    Application.ScreenUpdating = True
    Application.EnableEvents = True
End Sub
