Apologies for the delay in reply. Here's what you can try to do to achieve the result that you wanted.

  1. From the sample data (Book1) that you've provided you would have to create two additional columns. The first added column should contain Merged address and the second added column should contain Merged name.
  2. Under Merged address, merge Pref street and Pref Cityline using this formula =G2&" "&K2
  3. On the Merged name, merge the First name and the Last name using this formula =D2&" "&C2
  4. To combine everything which you will use for mail merging, open a new worksheet then copy the Merged address column by clicking on Ctrl+C  to the first column of the new work sheet. Instead of clicking Ctrl+V to paste, choose Paste and Paste as Values under the Home tab.
  5. Highlight the column and choose Data from the menu. Select Remove Duplicates.
  6. On the second column label it as Name and insert this formula: =IFERROR(INDEX(Book1!Merged_Name,SMALL(IF(Book1!Merged_Address=$A2,ROW(Book1!Merged_Name)-MIN(ROW(Book1!Merged_Name))+1),COLUMNS($A$2:A2))),"")
  7. Create another column with label as Name Combined them use this formula =IF(COUNTIF(B2:C2,"")=1,B2,B2&" and "&C2)

Note: Book 1 on step # 6 should be replaced with whichever book you have saved your main data.
