suzanne s. barnhill say:

Is there some reason you can't just use a cross-reference? If the paragraphs are autonumbered, you can insert a cross-reference (Reference | Captions | Cross-reference) to Numbered item: Paragraph number. The item will be listed, and you just select it. These cross-references don't update dynamically (as the paragraph numbers themselves do) when you add or remove numbered items from the list, but they do update automatically when fields are updated (such as when printing or switching to Print Preview or manually updating fields with F9).

If the issue is that you need a reference to a paragraph that does not exist in the template, then of course this is more difficult.

stefan blom say:

Using cross-references will be the easiest solution, and the only solution, I believe, if you are using Word's auto numbering.

However, it you make use of SEQ fields for numbering, you can always calculate the previous number.

hansv mvp say:

Here is a macro you can use; you can assign it to a button.

 

Sub SaveAsPDF()
    Dim strFileName As String
    strFileName = Application.GetSaveAsFilename(FileFilter:="PDF files (*.pdf), *.pdf")
    If strFileName <> "False" Then
        ActiveSheet.ExportAsFixedFormat Type:=xlTypePDF, Filename:=strFileName
    End If
End Sub
