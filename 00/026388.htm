doug robbins - mvp office apps & services (word) say:

The use of the term global template would have been in reference to the Normal.dotm template that is considered as a global template.  You can however create other templates that will be treated as "global templates" if you save them in the Word Startup folder, which by default is located at:

C:\Users\[UserName]\AppData\Roaming\Microsoft\Word\STARTUP

When you save a template in that location, it is loaded when Word is started and certain of the features of that template, such as macros and autotext, will be available for use with other documents.

While it is not quite as convenient to create autotext entries and have them saved in such a template, as it is necessary to use File>Open and have that template open when you are creating such entries, there is however no chance of you ever losing those autotext entries as can happen if the Normal.dotm template is overwritten by some process over which you need to be very vigilant to prevent it happening.

***

You do not need to open the normal.dotm template to store AutoText there. It is available for storage of AutoText whenever Word is open. It is the default template for storage of AutoText, but is not the only one available.

Word will ignore the one named normal.dot in that location. The location is the "user templates folder." That is the template from the menu version of Word.

If you want access to AutoText stored in that template (normal.dot), put it in the Word Startup folder. I would rename it to avoid confusion.

Word will ask if you want to save changes to the global template (normal.dotm) whenever any changes are made to that template. Saving AutoText in that template is a change to the template. So is saving a macro, a keyboard shortcut or a style. It should be asking you about this.

I make a backup of my normal template every month even though I try to keep many customizations, including building blocks, in other templates. I generally keep three months of these backups. I do not know that this is necessary, but there are things  that seem to happen to normal.dotm.
