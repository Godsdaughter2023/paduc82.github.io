11. Now click Ok to exist from Properties windows

How to Grant Permissions in Windows 7

1. Locate the file or folder on which you want to take ownership in windows explorer

2. Right click on file or folder and select “Properties” from Context Menu

Lets clear any general corruption :

Windows Key + X - Command Prompt (Admin) or Windows Powershell (Admin) or Start Button - type in CMD then Right Click on it at top and Run as Admin. (Run these one line at a time - note the spaces.)

SFC /scannow

When that completes run these :

DISM /Online /Cleanup-Image /CheckHealth

DISM /online /cleanup-image /scanhealth

DISM /online /cleanup-image /restorehealth

Note if that shows any uncorrected errors - continue regardless.

Then restart the computer.
