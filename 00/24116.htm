First let's clear up what you mean by "I have downloaded MS Office ...."  Downloading is the process of transferring files from an internet file server to a storage device (usually your computer's hard disk drive) on your computer.  Installation is the process of running a setup program that takes program files from a storage location (like a CD or DVD, or a downloaded installation file) and integrates those program files into your system so that you can use the program on your computer.

 

If you downloaded as defined above, the installation files went to a storage location on the computer and are waiting for you to start the Setup or installation process, so there would naturally be no icons for Office programs on your desktop or in the Start Menu.  If you saved the downloaded installation file to the desktop, you should see a new icon for that on the desktop.

 

If you downloaded the installation file and then double-clicked it to run Setup and the installation completed, it would be normal to not see any Office icons on the desktop.  Generally, Office Setup does not sprinkle your desktop with icons for all of its component programs, something that I think we can all be thankful for :-)

 

Did it make a Microsoft Office folder in Start Menu>All Programs?  If so, right click each program's icon that you want on the desktop and drag it there, then choose "create shortcut".
