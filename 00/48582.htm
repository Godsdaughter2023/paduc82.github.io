Go to the Control Panel and select "Regional and Language Options".

Select the "Languages" tab, then, in the "Text services and input languages" pane, select the "Details" button.

This opens the "Add input language" windows.

Click "OK".

You will now have whatever you picked available in the "Default input language" pane.

You now have an active "Key settings" button in the "Preferences" pane. 
You can assign key sequences to select an input language or
use Left Alt+shift to switch between input languages.

If you want, turn on the "Language Bar" in the task bar (or desktop), if it's not already,
by clicking "Language Bar" and then clicking "Show additional Language bar icons in the taskbar"
(or "Show the Language bar on the desktop).
