bernie deitrick say:

If you are extracting the addresses into a 6 cell block, use an array-entered formula (entered using Ctrl-Shift-Enter) like this in another cell, for the 6 cells being A2:A7

=IFERROR(INDEX($A$2:$A$7,SMALL(IF($A$2:$A$7<>"",ROW($A$2:$A$7)-(ROW($A$2)-1)),ROW(A1))),"")

copied down for 5 more cells.

You could also go simpler, with

=IF(COUNTA($A$2:$A$7)>=ROW(A1),IF(A2<>"",A2,$A$7),"")

which will work as long as any blanks are together before cell A7.

ambercomputers say:

I had the same problem, this is what I did to fix it. Hope this helps:

Symptoms

  • All Skype sounds work okay except for incoming voice or the voice on the Echo test.
  • Contacts can hear me without any problems.

Fix

  1. Right-click Speakers (volume) on taskbar.
  2. Select sounds.
  3. Select Playback tab.
  4. You will see Speakers and other Playback devices.
  5. The problem I encountered was that, regardless of which Speakers are selected within Skype, Skype uses the wrong Playback device. If you test Echo in Skype you should see what Playback device Skype is actually trying to use (in sound). In my case is was trying to use Digital Audio.
  6. Disable the appropriate Playback device (if you are not using it for anything else of course).
  7. Go back into Skype Audio Setting and remove any devices not required in Speakers & Ringing if possible.
  8. May need to restart Skype or Reboot PC.

Good luck
