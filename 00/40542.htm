Not a big deal for me, since I leave my PC on, 24/7 and, surprisingly I did not observe such behaviour. What I did find instead, is the very opposite! My desktop PC has now gained a will of its own, and seems it is shutting down automatically after a few hours of inactivity.

Ok, you can say it should be an easy fix if I go and check my power related settings. I did that and, they were not changed after the update, meaning, I had my monitor set to turn off after '5 minutes' and the hibernation set to 'never'!

Ok, but that could an UI bug and the status could have changed regardless. Hi Stephanie,

Thank you for posting your query on Microsoft Community.

I can understand how frustrating it could be when things go unusual. I will certainly help you.

To guide you in right direction we want to know:

1.  What is the make and model of the printer?

2. Do you get error 'no printers installed' while printing from specific application or all?

3. Have you set the printer as default printer?
4. From which application are you trying to print?

 I've seen this issue caused by using an incorrect driver. Try downloading and installing the correct drivers from Printer support site and check.

I suggest you to follow the methods below and check if this helps:

Method 1:

Run the Printer troubleshooter.
Hardware and Devices troubleshooter checks for common problems and makes sure that any new device or hardware connected to your PC is installed correctly.

 a. Type “Troubleshooting” in the search box and hit Enter.

 b. Select “Hardware and Sound” from the Troubleshooter Window.

 c. Select “Hardware and Devices” under Devices option.

 d. Follow the On Screen instructions.

Method 2: Try to uninstall and re-install the Printer drivers.

You may follow these steps to uninstall and reinstall the Printer drivers:

  1. Press “Windows Logo” + “R” keys on the keyboard and type “devmgmt.msc” in the “Run” Command box and press “Enter”.
  2. In the “Device Manager” window, search for the Printer drivers, right click on it and then select “Uninstall”. On the un-installation window, if you have an option: “Delete the driver software for this device”    you may select that and then remove the corrupted drivers from the computer.
  3. Follow the on-screen instructions to complete it and the restart the computer if prompted.

Method 3: You can try installing the drivers in compatibility mode and check if this helps:

  1. Press the ‘Windows key on the keyboard.
  2. Type compatibility in the search box and then press enter.
  3. Click “Run programs made for previous versions of Windows” option from the list appeared in the left side.
  4. Select the drivers and install.

We suggest you to refer the below link to troubleshoot the printer issue.

http://windows.microsoft.com/en-us/windows-10/fix-printer-problems

Hope this information was helpful and do let us know if you need further assistance. We will be glad to assist.

considerate_guy say:

There are a couple ways you can accomplish this... but for a true freeze frame, here's the steps I would do:

  1. Play the video until you get to the point where  you want the freeze frame.
  2. Click 'M' on the keyboard to make a Split (or use the option on the Video Tools | Edit tab)
  3. On the Home tab, click the Snapshot button.  Give the file a name and save it somewhere along with your other project files.
  4. You'll see the snapshot added to the storyboard at the end; drag it back to the point where you made the split.

The result will be video that plays, then the still image is displayed, and then the video continues.  Note: you can adjust the length of the freeze by changing the snapshot's duration.

jmh3143 say:
Generic advice for changing your language in Outlook.com

To change the viewing language on your inbox to your preferred language.
1. Login to your account.
2. Copy and paste the link in the address bar:
   https://home.live.com/options/language?ru=https://home.live.com/options/   (May not work for Outlook)
3. Select the language you want to use.
4. Click the Save button at the bottom of the language choices.

You also need to change the viewing language in your account's Profile. 
1. Login to your account.
2. Copy and paste the link in the address bar:
   https://blu158.mail.live.com/P.mvc#!/mail/options.aspx?subsection=12&n=556230082
3. Select the language you want to use.
4. Click the Save button at the bottom of the language choices.
