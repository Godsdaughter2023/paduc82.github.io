saltgrass say:

Well, if the other end of the cable is plugged into a good port then that would mean the Network Adapter is not active because if it was, there would be some activity to try to sync.

Open the connections panel (type ncpa.cpl in run box) and see if the adapter is listed.  If it is, what is it's status?

If it is not shown, open the Device Manger and see if it is listed there and what status it shows.

If it is not listed in Device Manager, you might need to check your Bios setup to see if it is turned on...

I would check and make sure all connections were secure.  In some cases the connector needs to be inserted just a little more to make a good connection, assuming your cable is not damaged is some manner. 

If it won't work with the Router, try going directly to the Modem to see if you can get some activity.

If that doesn't help, try rebooting everything and wait until the device upstream has established a connection before starting the next one.
