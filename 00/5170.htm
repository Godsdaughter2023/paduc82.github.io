How can I auto-populate several paragraphs based on dropdown - Microsoft Community

warbird36 say:

How can I auto-populate several paragraphs based on dropdown selection?

Hello, everyone! 

I'm attempting to create a template for my employer that should save us some time. We send a lot of letters that contain one of three different versions of "asks" (as in, "please send us $X.XX"). As it currently stands, there are three general types: a straight up listing of different dollar amounts, a request based on their highest previous contribution (HPC), and a request based on their most recent contribution (MRC). 

These are highly formulaic and barely vary--if at all--from letter to letter. That is to say, one MRC ask is going to be like another MRC ask; an HPC ask is going to be almost the exact same from HPC letter to HPC letter.

I've set up a dropdown list at the top of the Word doc that contains three options, each based on the ask type. I'm trying to populate several paragraphs automatically based on which option is selected. This takes care of a good 100-150+ words on a 1200-word letter. There's simply no good reason not to automate this in some fashion, but I'm having a devil of a time figuring out how I need to go about doing so. 

Any suggestions? Thanks!

doug robbins - mvp office apps & services (word) say:

There are a couple of ways in which that could be done.

If the item selected from the dropdown would normally appear in the document, you could use a combobox Content Control for the dropdown and then use the ContentControl on Exit event to populate the relevant part of the document with the information associated with the selection made in the Content Control, which could be done by having the code in the on Exit event set the value of a variable in the document so that that value would be displayed in a corresponding DOCVARIABLE field.

If you do not want the item selected from the dropdown in the actual document, then you could use a userform that contained a combobox control, with the on exit event of that control performing a similar task.

For both of these approaches, you will find information on Greg Maxey's website

http://gregmaxey.mvps.org/Create_and_employ_a_UserForm.htm

http://gregmaxey.mvps.org/Populate_UserForm_ListBox.htm

https://gregmaxey.com/word_tip_pages/content_control_tools.html
