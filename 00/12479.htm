john korchok say:

I took a look. You don't actually have numbering set up for your headings. Instead, you've typed the number 3 in front of the Heading 1 chapter start. So the caption algorithm doesn't know what heading number to use because there isn't any. This is one of the many areas where Word relies on styles to automate features like numbering.

Instead, modify the Heading 1 style to have true numbering:

  1. Right-click on Heading 1 in the Style Gallery and choose Modify.
  2. If you want to add the style to the underlying template, so it's there for subsequent papers, check New documents based on this template.
  3. Click on the Format... dropdown and choose Numbering.
  4. Select a numbering style, then click on OK, then on OK. The style Gallery now previews Heading 1 with a number in front.
  5. If it hasn't updated automatically, select the chapter heading and apply the numbered style.
  6. Insert a picture.
  7. Right-click and choose Insert Caption.
  8. In the Caption dialog, click on Numbering.
  9. Check Include chapter number and select the Heading 1 style. Click on OK, make and adjustments, then click on OK again.

Your caption insertion should go more smoothly now.
