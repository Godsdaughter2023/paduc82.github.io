Help with VBA code - Comment size/shape/position - Microsoft Community

tasha.stacey say:

Help with VBA code - Comment size/shape/position

I've used the below code (found on http://discussions.virtualdr.com/showthread.php?237673-Excel-comments-moving-and-changing-sizes) which has solved the problem I had with the comments in my spreadsheet moving all over the place etc.

 

Dim thisfile As Workbook
Set thisfile = ActiveWorkbook
Dim MyWorkbook As Workbook
Dim MySheet As Worksheet
Dim MyComment As Comment
Dim CommentCount As Long
Dim lArea As Long
Dim fixed As Boolean
fixed = False
On Error GoTo NeedToUnprotect
For Each MyWorkbook In Workbooks
MyWorkbook.Activate
For Each MySheet In MyWorkbook.Sheets
MySheet.Activate
CommentCount = 0
For Each MyComment In MySheet.Comments
With MyComment.Shape
.Placement = xlMoveAndSize
.Top = MyComment.Parent.Top + 5
.Left = MyComment.Parent.Offset(0, 1).Left + 5
.TextFrame.Characters.Font.Name = "Tahoma"
.TextFrame.Characters.Font.Size = 8
.TextFrame.AutoSize = True
CommentCount = CommentCount + 1
End With
If MyComment.Shape.Width > 300 Then
lArea = MyComment.Shape.Width * MyComment.Shape.Height
MyComment.Shape.Width = 200
MyComment.Shape.Height = (lArea / 200) * 1.1
End If
Next MyComment
If CommentCount > 0 Then
MsgBox ("A total of " & CommentCount & " comments in worksheet '" & MySheet.Name & "' of workbook '" & MyWorkbook.Name & "'" & Chr(13) & "were repositioned and resized.")
fixed = True
End If
Next MySheet
Next MyWorkbook
thisfile.Activate
If fixed = False Then
MsgBox ("No comments were detected.")
End If
On Error GoTo 0
Exit Sub

NeedToUnprotect:
MsgBox ("You must unprotect all worksheets before running the macro.")
thisfile.Activate
Exit Sub

End Sub

 

I now have a new problem - when editing the comment(s) if I try to enter a new line in the comment box I can't actually see what I've typed until I've finished editing it. Is there something that needs to be changed in the above?

 

Please help!

gary's student say: That is because of AutoSize.


If AutoSize is False, you get the nice scrolling effect that allows you to see what you are editing, but the box does not re-size.

If AutoSize is True, you get no scrolling effect, but the box re-sizes nicely.

So if you want to go back and edit an existing comment:

  1. switch AutoSize back to False
  2. perform your edits
  3. switch AutoSize back to True to fix the box

bernie deitrick say:

Do you want 2 or 3?  (You are showing 4 of each agent in the second table). Do you want them randomized? Is the Data repeated as shown? What do you want in the second table? You can still use a pivot table - with two extra columns of formulas, you can extract 2 values per agent.

For example, if you have your data in columns A to C, then in D2 enter =RAND() and copy down. Then copy those formulas and paste values over them.  Then in E2, array-enter (enter using Ctrl-Shift-Enter) the formula

=D2>=LARGE(IF($A$2:$A$10000=A2,$D$2:$D$10000),2)

and copy that down (if you have more than 10000 rows increase those values in the formula).  Then use column E as a filter on your pivot table, choosing TRUE.

v_agarwal say:

You may use this formula at C2 and copy it down.  Now filter your data on C Column with True.

=COUNTIFS($A$2:A2,A2,$B$2:B2,B2)<=2
