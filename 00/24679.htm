john korchok say:
The product key is a 25-character code that comes with Microsoft disk-installed products. It's usually found on the software disk case. Some versions of downloaded Microsoft software also use product keys. You normally get those in an email when you download it. You can't activate the product without it.

rohn007 say:

As John said, a MS product key is a 25 character code, typically written in 5 groups of 5 char. You use it to prove to MS that you have a valid license to use the program (like Windows or Office).

You prove to MS that you own the software by providing the Product Key while "Activating" the installed program.

Originally the product key was yours for life. You could use it to activate the product again in the future, depending on the exact license terms.  Then with Office 2013/365 MS introduced a 1-time product key for Office 365.  When you use your product key to activate Office 365, it is stored online 'associated' with your email account.  That is why it is important to be sure to use the right email account.  After you have used the 365 key once it is no longer usable.

Your Windows product key is usually on a sticker somewhere on the computer case.

SharePoint Online: Forms on a page which populate a list? - Microsoft Community

a. user say:

SharePoint Online: Forms on a page which populate a list?

Hey, I notice in SharePoint Online, you can click on the Form Web Parts button on the List Tab inside the ribbon to use a New Item form to input data into a list. I created a SharePoint List which acts as a repository for a suggestion box; and was wondering if I could actually embed that form to a Team Site page, so that people don't need to navigate to the Ribbon to find that button?

The goal I have in mind is for simplicity, and I want the suggestion box to be hidden into its own page, so that suggestions sent into the box can only be viewed by members responsible to reading said suggestions. I would like to create a form that is added as a web part which end users can then type in a suggestion and submit that to the hidden list. Any tips, tricks or solutions would be greatly appreciated.

Thanks!

cliff gu msft say:

Hi Alex,

You can follow the steps below to achieve your requirement.

1. Create a list and click Customize in InfoPath to create a customized form to publish to the list.


2. Set the unique permission on the list and share the list with everyone except external users granting the Edit permission. Now the users can only enter the list via the URL of the list.
3. Navigate to the Team Site Homepage.
4. Click PAGE and then click Edit.
5. Click INSERT and then click Web Part.
6. Select Form to insert an InfoPath Form Web Part.

7. Click Click here to open the tool pane and select the list name you created in the dropdown of List or Library.
8. Save the page and now you can see the form displays on the page.

9. Click PAGE and then click Page Permissions.
10. Set the unique permission on the page and share the page with everyone except external users granting the Edit permission. Now the users can only enter the page via the URL of the page.
11. Return back to the list and create a personal view for yourself via LIST>Create View>Standard View>Select Create a Personal View under View Audience.

 

Now you can only send users the URL of the Team Site Homepage and after they filled in their suggestions, you can navigate to the list to review them.

 

Thanks,

Cliff

a. user say:

Hello TRMR8:

 

Thank you for posting to the forum regarding the issue you are having with notifications on your lock screen.  There are a couple basic things I might suggest to see if we can get this resolved.

 

First, I would do a soft reset of the phone.  This will not effect any of your phone content and will simply remove some temporary files and clear up some things that can cause performance issues.  If you have a removable battery simply remove the battery for about 2 minutes, replace and reboot the phone.  If your battery is not removable you should be able to do the soft reset by pressing and holding the volume keys, power button and camera together for 3-5 seconds.

 

If you notice that notifications are still missing after the reboot the next thing I would suggest you do is manually set the time and date on your phone.  You can do this by going to settings > date + time > set automatic to OFF > manually set the time, date and time zone.

 

The next thing do to if the issue continues is to uninstall the recently added applications.  I would suggest you do this one at a time and restart the phone after each removal.  Monitor the lock screen behavior after each removal and see if you notice notifications arriving again.

 

Please let us know if there are any new details or additional questions/concerns.

 

Mark W.
