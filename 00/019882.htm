Well, here's one possible approach:

The Table need only be 2 columns x 1 row, although you can include a Header Row to label the respective columns if you wish.

Format the cell where the Link Phrase is to be typed using a simple Style (other than Normal or Hyperlink). The cell with the web address will automatically use the built-in Hyperlink style if the entry is a valid address.

In the code line where the URL is to appear;
  1. Insert> Field
  2. Select  StyleRef from the Links & References category
  3. Click to the right of the field name STYLEREF in the editable box
  4. Type Hyperlink
  5. Click OK

Repeat the above steps to insert a StyleRef field into the code line for the Style you used to format the Link Phrase cell.

don cuthbert say:

If your device has a current, activated Digital License for Windows 10 Insider Preview, it is licensed for the public version of Windows 10 also.

The Digital License for Windows 10 is for your device. That license may or may not be linked to a Microsoft Account. But to get Insider builds via Windows Update, you must be logged into Windows using a Microsoft Account, so it is pretty safe to say that the Digital License has been linked to your account. A clean install of Windows 10 public version to unchanged hardware would auto-activate, even if you did not log in with a Microsoft Account. But to activate changed hardware, it has to check your Microsoft Account to find the Digital License which was tied to the previous hardware.

  • If you are keeping your hard disk, you could just install it as-is without a reinstall. However, a clean install is not a bad idea to do now and then.
  • If you do choose a clean install, delete all hard disk partitions and ensure that the new motherboard is in UEFI BIOS mode with Secure Boot enabled. Of course this deletes all software and data from your disk.
  • Install the same Edition (Home or Pro) of Windows 10 that was activated on that PC before.
  • During install, do not enter any Product Key.
  • After install has completed, use the Activation Troubleshooter in Settings. Indicate that you have changed hardware. As long as you have been logging into Windows 10 with your Microsoft Account, your Digital License for that Edition of Windows 10 for that specific device should have been linked to your Microsoft Account. The Troubleshooter should find it and offer to activate your changed hardware with that Digital License.
  • Transferring Windows activation will not work between 2 device types (such as between a desktop PC and a laptop) as they are obviously different devices and not an upgrade of the original device.

The closing End If for  If objShape.HasChart = True Then needs to be moved to the bottom otherwise all font will be applied to the non-chart shapes giving the error.

Sub positioncharts()

Dim objSlide As Slide
Dim objShape As Shape
Dim i As Long
Dim ax As Axis
Dim objcht As Chart


 For Each objSlide In ActivePresentation.Slides
    For Each objShape In objSlide.Shapes
       
           If objShape.HasChart = True Then
                With objShape
                      .Chart.ChartArea.Format.TextFrame2.TextRange.Font.Name = "Segoe UI"
                      .Chart.ChartArea.Format.TextFrame2.TextRange.Font.Size = 9
                      .Width = 703
                      .Height = 454.664184570313
                      .Top = 50.3499221801758
                      .Left = 5
                      .Chart.PlotArea.InsideHeight = 365.92818897678
                      .Chart.PlotArea.InsideTop = 30.0150393700787
                      .Chart.PlotArea.InsideLeft = 50
                      .Chart.PlotArea.InsideWidth = 575.27157480315
                 End With

                Set objcht = objShape.Chart
                objcht.HasAxis(xlCategory) = True
                Set ax = objcht.Axes(xlCategory)
                With ax.TickLabels.Font
                            .Color = RGB(255, 15, 255)
                            .Name = "Segoe UI"
                            .Size = 9
                End With
                objcht.HasAxis(xlValue) = True
                Set ax = objcht.Axes(xlValue)
                With ax.TickLabels.Font
                            .Color = RGB(255, 15, 255)
                            .Name = "Segoe UI"
                            .Size = 9
                End With
                With objcht.Legend
                    For i = 1 To .LegendEntries.Count
                   
                        .LegendEntries(i).Font.Color = RGB(255, 15, 255)
                        .LegendEntries(i).Font.Name = "Segoe UI"
                        .LegendEntries(i).Font.Size = 9
                    Next
                End With
            End If
    Next objShape
  Next objSlide
End Sub

davem121 say:
Hi Here & Now, the general form of referencing a cell on another sheet is:

For a single cell on another sheet:
Sheet_name!Cell_address

For a range of cells on another sheet:
Sheet_name!First_cell:Last_cell

So for example, in your case:

In Cell F4 on Worksheet 2 add this formula:
='Worksheet3'!F4

If you need any other formula, let me know!
