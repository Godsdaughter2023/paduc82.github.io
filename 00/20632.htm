jay freedman say:

Since the only kind of option button you can put into the document body is ActiveX, I'll assume that's what you're dealing with.

Second question first, because it's easier: While the Design Mode button in the Developer ribbon is turned on, select an option button and click the Properties button. Enter some word in the GroupName property. Then add the same word to the GroupName properties of the other option buttons. Only one of the group can be selected at one time, as you want.

While you're looking at the properties, it's best to delete anything you have in the Caption property, and set the Width property to a value just large enough to keep the button itself visible -- about 16 points. Then put the "caption" in the table cell as regular text. That way, there won't be a big white rectangle standing out against the colored cell shading.

Now right-click any of the option buttons and click View Code. That opens the macro editor and starts with an empty macro for the chosen button's Click event. Replace that with the following three macros:

Private Sub OptionButton1_Click()

    With ActiveDocument.Tables(1).Rows(1)

        .Cells.Shading.Texture = wdTextureNone

        .Cells(1).Shading.BackgroundPatternColor = wdColorBrightGreen

        .Cells(2).Shading.BackgroundPatternColor = wdColorAutomatic

        .Cells(3).Shading.BackgroundPatternColor = wdColorAutomatic

    End With

End Sub


Private Sub OptionButton2_Click()

    With ActiveDocument.Tables(1).Rows(1)

        .Cells.Shading.Texture = wdTextureNone

        .Cells(1).Shading.BackgroundPatternColor = wdColorAutomatic

        .Cells(2).Shading.BackgroundPatternColor = wdColorYellow

        .Cells(3).Shading.BackgroundPatternColor = wdColorAutomatic

    End With

End Sub


Private Sub OptionButton3_Click()

    With ActiveDocument.Tables(1).Rows(1)

        .Cells.Shading.Texture = wdTextureNone

The sollution to the above is as below. Thanks to G North.

The only way to acheive this none regular increment whilst grouping is to group by hand. This will only worls with text however. Is the data in the agin column text? If so:

Create the PivotTbale with Aging as columns and Priority as rows.

Then select the headings 0 to 0.2 and right click, then Group.

Repeat this selecting the cells within each band, then grouping. This way you can select and range of values to group and give the impression in your requirement example.

After grouping all the columns replace the text Group 1 with 0-0.2, Group 2 with 0.2-0.4. Click the cell containing the text and Edit it using the formula bar.

Once all the headings have bee edited go to Pivot Table options and turn off Display captions and filter drop downs in the Display tab.

If you don't have your data as text this will not work.

To convert the Aging column to text add another column with the formula =TEXT(A2,"0.0"), copied down.

You can then include this in the source data for the Pivot using Change Data Source and make it the Column field for the Pivot Table.
