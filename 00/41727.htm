Hi,

Windows Media Center might not be able to download Guide data for one of the following reasons:

·         Your computer is not connected to the Internet or there is a problem with your Internet settings

·         The Guide server is not available or is temporarily out of service. If the Guide Setup was previously successfully completed, Windows Media Center will try to download it again daily.

·         The Guide service provider does not have Guide information for your location.

 

1)   Perform Clean boot and check if you are able to download it.

Note: Please ensure that the computer is set to start as usual by following the step 7 from the article.

 

2)   If that does not help, then ensure your localization settings are set correctly to United States

a. Open the “Regional and Language Options” dialog (search for “Regional and Language Options” from the Windows Start Menu search box)

b. On the “Formats” tab, ensure “English (United States)” is selected under “Current format:”

c. On the “Location” tab, ensure “United States” is selected under “C

d. On the “Administrative” tab click on the “Copy to Reserved Accounts” button, in the dialog that pops up, ensure that at least the System `accounts (local system, local service, and network service) check box is selected, and choose OK.

e. Restart the computer

f. Then select “Download now”

 

3)   You can try to run Media Center Update execution file (mcupdate.exe) from the following location on your computer to see if that helps you in downloading guide successfully. Follow the steps below to run mcupdate.exe

 

 

 

a)   Open Windows Explorer by pressing Windows Button + E from the keyboard.

 

b)   Go to the following location: C:\windows\ehome and search for mcupdate.exe

 

c)    Right click on it and choose “Run as administrator”.

 

 Now check if you’re able to download the guide without having any issues.

 

4)   If this did not work, try the following from an administrator command prompt

a. schtasks /delete /tn "microsoft\windows\media center\mcupdate" /f

b. cd /d %WINDIR%\ehome

c. start /wait mcupdate –u

d. Close the command prompt once the command runs successfully.

 

Now check if you’re able to download the guide without having any problems

Stopping "F1 Continue" command during Windows 7 StartUp - Microsoft Community

gilbert k say:

Stopping "F1 Continue" command during Windows 7 StartUp

I want my computer to automatically accept the command "F1 Continue" without me having to press F1 each time during startup.

a. user say:

Hi, Gilbert,

 

Have you added new hardware to the system?

 

1) Open command prompt with elevated privileges

Type secedit /configure /cfg %windir%\inf\defltbase.inf /db defltbase.sdb /verbose

Press Enter

 

Did the message disappear?

 

2) If new hardware has been recently installed into the computer, it is likely that you are receiving the prompt "Press F1 or F2 to enter setup" because CMOS needs to verify that the new detected hardware is properly being setup in CMOS. Enter CMOS setup and save the settings and exit.

 

3) If no new hardware has been added or you continue to get this prompt verify no error or conflictions exist in CMOS. Press the key to enter setup and verify that all settings are correct. e.g. date, time

 

4) Reset CMOS or Bios settings

Look for an option to reset the CMOS values to the default setting or an option to load the fail-safe defaults.

Save and Exit
