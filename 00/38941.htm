Are you playing the game from the 'Redux Collection' disc version? Because earning the achievements on this version of 2033 and Last Light will NOT count towards any game pass quests.

The Redux Collection disc versions of these games are actually recognized as entirely different games from the digital download versions by Xbox Live. Consequently, they each have their own (identical) achievements lists and earning an achievement on one will not unlock the same achievement on the other.

To reiterate: there are two versions of Metro 2033, the digital download that you get from the Microsoft Store and the Redux Collection disc version. These two versions are EXACTLY the same in every way and the achievements are exactly the same, they're just not recognized as the same game by Xbox Live. Same deal with Last Light.

So even if you own and have installed Metro 2033 and Last Light from the Redux Collection disc you'll still need to download the digital versions from Xbox Live and launch those if you want to complete any Quests involving those games.

A very silly error on Microsoft's part, although on the bright side it means you can earn an extra 2000 achievements just for playing these games again.

cmdrkeene say:

Do you use Norton programs?  That symbol you describe sounds just like the "Norton secure symbol" they use to indicate that the file is safe after a scan, and/or backed up using Norton Online Backup.  Of course other synchronization systems use indicators like this, like DropBox, Google Drive, OneDrive, Amazon Cloud Storage, and more.  So another way to help determine is to look at what folder those files are inside. If the symbols only show up on files inside your DropBox folder, for example, you can be pretty sure DropBox is what's showing the status indicator.

As for your other question: the other authors can only update the files if you gave them permission to do so by clicking "Share" and using the "allow users to edit" instead of "view only" option.  The control is up to you.
