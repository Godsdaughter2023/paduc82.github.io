Windows 10: Features for a true multi-size, customizable platform.

I like the direction Win10 is headed but there are quite a few more things I would love to see from a Personalization and Ease of Access point of view in a truly great Windows OS. I have not had the chance to play with the Tech Preview as I do not have a spare capable PC, tried Virtualbox- lousy experience, so gave up on that pretty quickly. So, here's what I would like to see in Windows 10:

1. Starting with the 'Start' menu:

- All ‘Tile Groups’ of Win8.1 Start Menu as cascading main menu items for ease of access to live tiles within that group. This alleviates the need to scroll thru the entire tile directory.

- Ability to create duplicate tile for use in different groups.

2. The all-important 'Taskbar':

- Ability to PIN ‘Tile-groups’ to taskbar with display of live tiles after one/two second delay of mouse-over. i.e. Tile groups can be pinned by right clicking on Start Menu group names. Clicking on Tile group icons would expand a bubble with live tiles, similar to the new start menu. In simple words, multiple customizable mini start menus.

- Ability to create toolbar from Tile-group. The Tile-groups can be checked on/off in the 'Toolbars' tab of the 'Taskbar and Navigation properties' menu.

- I like having a vertical taskbar with small icons. However, for a wide toolbar, only one row of icons fits, as opposed to fitting multiple rows of icons based on the width of the taskbar. This was possible in XP but not anymore. Please enable it again.

- Allow customization option to enable Cascading toolbars on mouse-over. This option currently works only when toolbar partially hidden is accessed by expanding the hidden items. Expand on mouse-over after one second delay, or single click to expand, double click to open in new window.

- Ability to pin cascading 'Control Panel' or any of its sub-items.

- Ability to create shortcuts for frequently used settings- either in the icon tray or in the main taskbar.

- Making Bluetooth control features accessible directly from the taskbar instead of opening a separate window, similar to wi-fi functionality.

3. Main desktop area & Charms bar:

- Allow tiles and tile groups to be ‘pinned’ on the desktop, just like icons. Why should there be a difference between the two? It can be argued either way from a design perspective, but, the user should be able to make that choice. Create new icon themes as an option to replace traditional windows icons. Slowly users will graduate towards tablet like ‘look and feel’ in a traditional windows environment.

- Expand availability of popular widgets like calendar, mail, bookmarks, weather, calculator etc. If there are proprietary rights attached then let them be available for a price from the ‘Store’. Widgets may be installed on the primary desktop or on the secondary desktop. Create a separate category in the ‘store’ for widgets that can be placed directly on the desktop. Will definitely increase interest from developers.

- Include a scalable and scrollable widget for a collection of live tiles/ tile group- similar to the bookmark widget on Android.

- Create Android like notification panel accessible as an expanded charms bar, i.e. instead of a simple 1 inch charms bar, user can enable a 6 inch/half screen notification panel to expand with the charms bar.

- Charms bar to include media controls with the usual suspects, plus the playlist selection and speaker selection (like built-in/Bluetooth, etc). This could be part of the notifications area.

- Another Android/iOS feature: Expandable folders for quick access without opening new windows. Single click to expand, double click to open in new window.

Please excuse any functionality/features that may have already been included in Windows 10 or discussed in another thread (Sorry, I haven't taken the time to read them.)
