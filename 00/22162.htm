However, I want to make sure that is the case, and I am not doing something incorrectly before creating additional shared mailboxes. Does anyone know if I should be able to send from a secondary address on a shared mailbox? 

moranyan say:

Hi Tom,

No, you are not able to send from a secondary address on a shared mailbox because the secondary address is a proxy address of the shared mailbox. It can only be used for receiving emails, not sending emails.

As a workaround, you can create a distribution group whose primary email address is the same as the secondary email address, then grant the Send As permission of the distribution group to you. After that, you can send as the distribution group whose email address is the same as the secondary email address of the shared mailbox.

You can use these two macros (see http://www.gmayor.com/installing_macro.htm if needed) and assign separate Quick Access Toolbar buttons or keyboard shortcuts to them.

Sub CrossRefFigure()
    With Dialogs(wdDialogInsertCrossReference)
        .ReferenceType = "Figure"
        SendKeys "%rO": .Show
    End With
End Sub

Sub CrossRefTable()
    With Dialogs(wdDialogInsertCrossReference)
        .ReferenceType = "Table"
        SendKeys "%rO": .Show
    End With
