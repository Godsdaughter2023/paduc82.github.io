Wireless Security - Microsoft Community

xbr dionysus say:

Wireless Security

Login Information

  • Gateway - 192.168.1.1 or 192.168.0.1
  • Username - admin
  • Password - password

How To Find Wireless Passphrase (WPA/WEP)

WEP

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)
  3. If prompted, enter "admin" for the user name and "password" for the password.
  4. Click Wireless Setup from the menu at the top.
  5. Click Basic Settings, from the menu on the left.
  6. The WEP key will be located in section 5, in the box labeled "Key Code".

 

WPA

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)
  3. If prompted, enter "admin" for the user name and "password" for the password.
  4. Click Wireless Setup from the menu at the top.
  5. Click on Advanced Wireless Setting from the menu on the left.
  6. The WPA passphrase will be located in the box labeled "Pre-shared Key (PSK) for home network.


 

Changing Wireless Security Type/ Disabling Wireless Security

WEP

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)
  3. If prompted, enter "admin" for the user name and "password" for the password.
  4. Click Wireless Setup from the menu at the top.
  5. Select Advanced Settings from the menu on the left.
  6. Select the radio button next to WEP.
  7. To set up a WEP key, delete the text out of the box that has the radio button selected. It will be one of the boxes labeled Key 1- Key 4 depending on which radio button is selected. Type in a new 10 digit WEP key using only numbers 0-9 or letter A-F.
  8. Click Apply at the bottom of the page.

 

WPA

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)
  3. If prompted, enter "admin" for the user name and "password" for the password.
  4. Click Wireless Setup from the menu at the top.
  5. Select Advanced Settings from the menu on the left.
  6. Select the radio button next to WPA.
  7. The passphrase will be located in the box labeled "Pre-shared Key (PSK) for home network, delete everything out of that box.
  8. Type in a new passphrase that is between 8-63 Alphanumeric characters( Any letters A-Z or numbers 0-9).
  9. Click Apply at the bottom of the page.

 

No Security

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)
  3. If prompted, enter "admin" for the user name and "password" for the password.
  4. Click Wireless Setup from the menu at the top.
  5. Click Basic Settings, from the menu on the left.
  6. To disable wireless security, select the radio button in section 4, that says "off".

Hiya Tommy,

We understand the frustration that comes with this type of errors but rest assured we are here to help! We suggest a reset of the console but if it cannot be accessed through the settings then the process is a bit different but there is still a way to do it. To access the troubleshooting screen: 

If you're unable to get to the dashboard at all to do the factory reset you need to access the Troubleshoot screen by using these steps :

  1. Unplug the network cable if you're using a wired network connection.
  2. Power off your console, and then unplug the power cord to ensure that the console is completely powered off.
  3. Wait 30 seconds, and then plug the power cord back in.
  4. Press and hold the BIND button (located on the left side of the console) and the EJECT button (located on the front of the console), and then press the Xbox button on the console.
  5. Continue holding the BIND and EJECT buttons for 10-15 seconds.
  6. Listen for two “power-up” tones a couple of seconds apart. You can release the BIND and EJECT buttons after the second power-up tone.

From there you can reset your console.

  1. Do one of the following, depending on whether you have installation media (such as a DVD or USB flash drive):
