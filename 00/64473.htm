Shinya,

Your reply led me to the fix, ugly as it was:

1. Boot from rescue media and navigate to command prompt (I left out a few clicks here).

2. In the C:\Windows\system32 directory on the system drive rename the drivers folder to drivers.bak.

3. Copy the drivers folder from the "x" drive (rescue drive letter) to the system32 directory on the C drive.

4. Reboot from HDD.

5. Windows starts and and indicates it is repairing the computer, followed by an almost immediate failure.

6. Reboot from rescue media and navigate to command prompt.

7. Delete the recently copied drivers fold from C:\Windows\system32.

8. Rename the drivers.bak folder to drivers.

9. Reboot from HDD. Windows and all of the USB ports work again. All is well once again.

Hi WentzWu, you have 2 options:

1. Manually install version 1803, you will still then, every couple of weeks receive more updates

2. Turn off Windows Updates permanently


1.

1709 is no longer the latest version of Windows 10, 1803 has been released, you should go straight to that version to bring your system up to date and make all previous updates obsolete . . .

With these major upgrades, the most successful method is to upgrade from the Windows 10 ISO - you download the ISO and then perform an Offline Upgrade

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), then choose ISO

When the ISO file completes downloading

Disconnect from the Internet

Uninstall any 3rd party Anti-virus - this is a known issue with 1803

Then, double click it and then run Setup.exe to begin the upgrade . . .


2.
Click your Start Button, type services and hit Enter

Scroll down to find Windows Update Service

If it is Started, double click it and stop that service

CLose the services App

Click your Start Button, type cmd, right click Command Prompt and choose 'Run as Administrator'

Run this command and hit Enter

sc delete wuauserv

Close Command Prompt

Restart your PC


If you also have the Update Assistant installed on your system:

The Update Assistant in Windows 10 cannot be uninstalled, but you can get rid of it

Click your Start Button, type Task Scheduler and hit enter
In the left Pane, navigate to: Windows - Update Orchestrater
Right Pane double click 'Update Assistant'
Click on Triggers Tab
On each of the triggers ( At login . . . etc.) double click and uncheck 'Enabled'

Under 'Update Assistant, there is another event 'Update Assistant CalendarRun'
Do the same for all triggers on that event

Restart your system, that's It!

When finished and after a system restart, open the services app and make sure the Windows Update Service is now gone . . .
