johnlv say: Barb-

I would have to see the entire SQL for the query and know how you're opening it.  Is it the Record Source of a Report?  The Report engine can need multiple passes, so you get prompted on each pass.  Why not use a parameter that points to a value on an open form?  The syntax for a parameter that points to a control on a form is:

[Forms]![Name of form]![Name of Control]

When you do that, the user won't see a prompt.

By the way, your syntax will not necessarily "show all" if any rows contain a Null value.  Your criteria should be:

Like "*" & [Enter part of a segment name or hit "Enter" to show all] & "*" Or ([Enter part of a segment name or hit "Enter" to show all] IS NULL)
