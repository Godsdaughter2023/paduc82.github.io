Here's what worked for me, exactly.  I opened Regedit.  I went to HKEY_LOCAL_MACHINE\SOFTWARE\Policies.  I think Microsoft existed uner that, but I think I had to add "office" (lower case) and then "15.0" below that and "common" (lowercase) below that, and finally "officeupdate" (again, all lower case).

The total path that needs to exist in RegEdit is :

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\office\15.0\common\officeupdate]

Once there, you will create a new dword value for enableautomaticupgrade and set the value to 0.

OR you can save the following lines (everything between the ===) in a file, using Notepad, called "fixthisannoyance.reg" :

===================

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\office\15.0\common\officeupdate]
"enableautomaticupgrade"=dword:00000000

===================

And then you can execute that, which adds that registry entry for you.  I had to restart Office products several times before it took effect - it did NOT appear to work right afterwords, and after I restarted Excel and Outlook.  But later that day (after another restart) it DID work.
