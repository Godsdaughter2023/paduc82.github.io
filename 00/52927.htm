    Hi,

     

     

    A file with the FXR and FXM file extensions is a WinFax Received Document file.

     

    You can open the FXR and FXM file by double clicking on it and let your PC decide which default application should open the file.   If no program opens, the FXR and FXM file, then you probably don't have an application installed that can view and/or edit FXR and FXM files.

    You can use a Notepad or another text editor to open these file.  Many files are text-only files, meaning no matter the file extension, a text editor may be able to properly display the file's contents.  This may or may not be the case with FXR and FXM files, but it's worth a try.

     

    How to an FXR and FXM Files:
    There are two ways to attempt to convert an these file to another file type:

    a.    Open the FXR file in its default program and choose to save the open file as another file format.

    b.    Use a file conversion software from online to convert these file to another file format.
