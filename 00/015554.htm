OneNote 2016 Page Versions - Microsoft Community

aurélie_cg say:

OneNote 2016 Page Versions

How long does OneNote keep versions of your page?

Does it keep all versions from the original on or is there a limit?

And can you change those settings considering the version history?

swischner say:
This is an older answer from a OneNote developer. But from my observations and a few short tests I believe it's still true:

"We create a new version any time a change is made to a page and either a) the author is different from the last change, or b) it’s been an hour since the last version we pinned.
Every time we create a new version, we review the existing versions to see if any should be pruned. The default pruning rules are:
• We keep all of the changes for the first 2 days.
• Older than that, we remove all but the first from any 1-hour period for the next 5 days.
• Older than that, we remove all but the first from any 1-day period, for up to 10 unique days.
• After all of the above, we delete all versions.
All three of those numbers (2, 5 and 10) are all customizable by policy. Furthermore, there are two more settings that can be changed.
• Version pruning can be turned on or off globally.
• A fourth number can be set which is the age (in days) after which any version will be pruned, regardless of the other rules. By default, this is -1 (off).
Regardless of what numbers are plugged in, versions are only pruned when either a new version is being created, or when the containing notebook is being optimized."

ruchi bisht say:

Auto text entries are generally saved in Building block.dotx template because it has capability of handling more objects (e.g. bibliography, citation etc...), however if the Auto text entries are saved in Normal.dotm template and if the template is changed you will not able to access these entries.

 

The normal template usually consists of formatting settings and user customization. The editing options and references (e.g. bibliography, citation,  etc...), are saved in building blocks

How to install the custom fonts in the computer so that you can use in PowerPoint 2007

mukherjee ashutosh say: http://office.microsoft.com/en-us/powerpoint/HA102880841033.aspx

Download and install a custom font on a computer running Vista
  1. Go to Control Panel, and then click Fonts.
  2. Right-click anywhere inside the folder, then click Install New Font.
  3. Do one of the following:
    • To install a font from your computer disk drive, in the Add Fonts dialog box, in the list box containing Drives list box, select the folder that contains the new font files.
    • To install a font from a network drive, click Network, and then specify the drive and folder from which you want to install the font.

       Note   The font files may be in different formats like Type 1, True Type, or Open Type. If the fonts you have selected has font files in these formats, it will be listed under the title List of Fonts.

  4. Click Install, and then click Close.
Download and install a custom font on a computer running Windows XP
  1. On the Start menu, click Control Panel.
  2. In Control Panel, double-click Fonts.
  3. On the File menu, click Install New Font.
  4. Locate the downloaded or unzipped font, and then in the List of Fonts box, click the font that you want to add.
  5. Click OK.

After you have followed the procedures listed above correctly, when you open an Office document, spreadsheet, or presentation, you will see the new font listed in the Fonts list.
