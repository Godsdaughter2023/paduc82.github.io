jeeped say:

The first thing you will need is a VBA User Defined Function (UDF) to reverse the string.

Right click the worksheet tab and select View Code from the Menu.

Use the menus to select Insert, Module then copy and paste the following in:

Option Explicit
 
Public Function ReverseString(Text As String)
    ReverseString = StrReverse(Text)
End Function

Hit the keystroke combination ALT+Q to return to the worksheet.

Now you can apply the following formula (assumes that the data starts in A1):

=LEFT(A1,LEN(A1)-SEARCH("-",ReverseString(A1),SEARCH("-",ReverseString(A1))+1))

I used just the hyphen (e.g. "-") to SEARCH for, but you may want the SPACE+HYPHEN+SPACE (e.g. " - ")

If you are going to resort to using a VBA UDF, then you might as well let it do all the work, especially when it can do so with a single line of code...

Function TBSDFTE(S As String) As String
  TBSDFTE= Trim(StrReverse(Split(StrReverse(S), "-", 3)(2)))
End Function

If you put the above function in a standard Module and then you can simply use this formula on the worksheet...

=TBSDFTE(A1)

Just so everyone knows, TBSDFTE is short for TextBeforeSecondDashFromTheEnd. Of course, you could use any more suitable name that you want.
