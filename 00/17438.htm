vlk l&f say:

TABLE CELL SIZE

I am trying to make the the rows in a table all the same size.  I went to office help (www.microsoft.com/en-us/publisher-help/table-tools-tabs-HA101830292.aspx) and followed the directions to click on table tools then size.  The website says you can adjust the height and width of the CELL,  however, the only options available are the height and width of the TABLE.  I need to set the columns to all be the same size; and the rows to be all the same with the exception of one of a different size.  There is no way to set the size of a cell, or group of cells/rows to a particular size.  Am I missing something or are the instructions wrong; and if they are wrong, how do I accomplish this task?  I do not see any indication of the size of the rows or columns in numerical format.  Any information would be helpful.  Thanks!

davidf2 say:

It appears to me that when you insert your table then all the cells are made of equal size dependent upon the table size. You can change the size of the cells by changing the size of the table. Increase the table height and all the cells will increase in height proportionally. The same is true or increasing or decreasing the width of the table.

 

If you have a 5x5 table and you want cells that are 1 inches wide X .75 inch wide then make your table 5 inches wide and 3.75 inches high.

 

To my knowledge there is no way to change the height or width of a column or row specific cell by typing in a value.

 

If you want one of the 5 rows to be of a different height then you can mouseover the bottom of any cell in that row until you see the cursor turn into a double line with up and down arrow icon, click and drag that line down until it is as big as you want it to be. If it is a row inside the grid it will just push the other rows down but will not resize them. Granted this is a bit imprecise, but you can use the ruler on the side of the page to help guide you.

 

If you want a more precise method then go to View and check Guides (this is also where you check the option to view the ruler). Also go to Page Design > Layout and check Allign to Guides and Objects. Now in the same place insert two horizontal ruler guides and make them the precise distance apart that you want the cell to be. Orient the page such that either the top or the bottom of that row in on one of the guides and then drag the bottom to 'snap to' the other guide.

 

And finally while I am offering workarounds, you could just create as many 1 X 5 cells tables as you want, size the height of the table the way you want and then drag and 'snap to' the tables together to build a custom table with variable row heights.

 

If there is a way to change the size of an individual row or column within a table by typing in a specific value than I hope someone jumps in to share that with us, but at least here are a couple ways you can accomplish your goal.

Hi Jeeped,

you are right. It should be an independent backup file. So I propose your solution as the answer with a little modification:

Sub Workbook_Open()
Dim xlobj As Object
Dim SrceFile
Dim DestFile
    Set xlobj = CreateObject("Scripting.FileSystemObject")
    SrceFile = ThisWorkbook.FullName
    DestFile = Left(ThisWorkbook.FullName, Len(ThisWorkbook.FullName) - 4) & Format(Now, "yyyymmddhhmmss") & ".bak"
    xlobj.CopyFile SrceFile, DestFile, True
End Sub

This event macro should be copied into the codemodule ThisWorkbook . The Auto_Open event is just for compatibility purpose. I added seconds (ss) to the timestamp and variable declarations.
