scarletroseft say:

This error will occur if files, folders, MFT or Master File Table, FAT or File Allocation Table, or the USB drive gets damaged due to certain reasons.

If MFT files were damaged, the system will not be able to access data from NTFS partition. Another reason for this error is if the external hard drive is infected with a virus. Removable drives that are not safely removed from PC can be one of the causes as well.

Formatting it manually is one way of fixing the issue but there's another way how to resolve it. Open cmd or command prompt then type chkdsk /f /the drive letter. Example: chkdsk /f /E
