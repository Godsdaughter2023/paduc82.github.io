Office 2016 Deployment Tool

Configuration.xml

<Configuration>

<ExcludeApp ID="PowerPoint" />

<ExcludeApp ID="Publisher" />

<ExcludeApp ID="Word" />

</Product>

</Add>

<Display Level="None" AcceptEULA="TRUE" />

</Configuration>

Save the file to your desktop. When you save the file, use the file name AddODB.

Note: This file will install the 32-bit English-US version of OneDrive for Business. If you have to have the 64-bit version, open the file in Notepad, and then change the line in the file from OfficeClientEdition="32" to the following:

OfficeClientEdition="64"

Click Start.

cd /d %userprofile%\desktop

setup.exe /configure AddODB.txt

6. When the installation is complete, a new command line appears at the command prompt.

7. Go to Start. In the Start Search box, type OneDrive for Business, and then start OneDrive for Business.

Note: After you set up OneDrive for Business, you may not see the sync icon overlays (that is, the green check marks or red crosses) on files and folders. (You have to restart your computer to enable the sync icon overlays to reappear.)

8. Close the command prompt, and then delete the following files from your desktop:

OfficeDeploymentTool.exe

Setup.exe

Configuration.xml

AddODB.txt
