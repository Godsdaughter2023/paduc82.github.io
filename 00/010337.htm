djsteele, mvp say:

Why not pass the name of the file to the function?

 

Public Function FTP_Data(ParamArray Files())

On Error GoTo Err_Trap
 
     Dim pFile As Long

     Dim intFile As Integer
     Dim strPath As String
     Dim ftpServer As String
     Dim strUserName As String
     Dim strPassword As String
    
     'Path and Name of file to FTP
     strPath = "C:\TempFolder\"
       'FTP Server Settings
     ftpServer = "MyServer"
     strUserName = "MyUserName"
     strPassword = "MyPassword"
    
     'Create text file containing FTP commands
     pFile = FreeFile
     Open strPath & "FTP_cmd.txt" For Output As pFile
     Print #pFile, "user"
     Print #pFile, strUserName
     Print #pFile, strPassword

     For intFile = LBound(Files) To UBound(Files)
         Print #pFile, "Put " & strPath & Files(intFile)

     Next intFile
     Print #pFile, "quit"
     Close pFile
    
     'Create batch file to execute FTP
     pFile = FreeFile
     Open strPath & "FTP_Run.bat" For Output As pFile
     Print #pFile, "ftp -n -s:" & strPath & "FTP_cmd.txt " & ftpServer
     Print #pFile, "Pause"
     Close pFile

    'Execute FTP command
     Shell strPath & "FTP_Run.bat", 1
    
 Err_Trap_Exit:
     Exit Function
    
 Err_Trap:
     MsgBox Err.Number & " - " & Err.Description
     Resume Err_Trap_Exit
 
 End Function

 

You can now pass the file(s) you want when you call the function. To have it only do one file, you'd call it like

 

Call FTP_Data("*ConfirmationOfRepair.pdf")

 

To have it do two files, you'd use

 

Call FTP_Data("*ConfirmationOfRepair.pdf", "*NotificationOfRepair.pdf")

For an underlined tab leader, position the cursor within the first line of the paragraph to be altered. Choose a point on the horizontal ruler you wish to end the underline and click there. You have created a left aligned tab, noted by an L on the ruler. Make a note of where you've placed the tab.
