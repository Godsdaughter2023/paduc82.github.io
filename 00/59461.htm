vijay a. verma say:
Microsoft has made it harder for us to disable Cortana. You used to be able to turn Cortana off in Windows 10, but Microsoft removed that easy toggle switch in the Anniversary Update. But you can still disable Cortana via a registry hack or group policy setting. This transforms the Cortana box into a “Search Windows” tool for local application and file searches.

Home Users / Pro and Enterprise Users: Disable Cortana via the Registry
Windows Key+Registry
Type regedit and Enter
Navigate to the following key in the left sidebar
HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Windows Search
If you don’t see a “Windows Search” key (folder) below the Windows folder, right-click the Windows folder and select New > Key. Name it Windows Search
Right-click the Windows Search key (folder) in the left pane and select New > DWORD (32-bit) Value
Name the value AllowCortana. Double-click it and set the value to 0
Close Registry Editor
Reboot your computer

Pro and Enterprise Users: Disable Cortana via Group Policy
For Windows 10 Professional or Enterprise, the easiest way to disable Cortana is by using the Local Group Policy Editor
Type gpedit.msc and Enter
Navigate to Computer Configuration > Administrative Templates > Windows Components > Search
Locate the “Allow Cortana” setting in the right pane and double-click it
Set the Allow Cortana option to “Disabled” and then click OK
Close the group policy editor
Reboot your computer

Creative Sound Blaster Audigy 2 ZS not recognized under Win 7 Ultimate - 64 bit

Friends,

I do have a situation where I request assistance. It is as follows :

1. The old desktop PC was having a USB based Creative Sound Blaster device - Audigy 2 ZS attached, it was procured in Aug 2008 and was working since then, initially with Win XP (32 bit) - from Aug 2008 till Oct 2012 and then from Oct 2012 (when I upgraded the OS to Win 7 Ultimate 64 bit) till last week when I upgraded the PC hardware. 

2. However, after the upgrade, the PC is not recognizing this device at all - shows a yellow exclamation mark with a label "Unknown device" under Device Manager, when I connect the device on USB port.

3. Last time (in 2012) when I upgraded the OS on the old PC hardware from XP to Win 7, the OS recognized the device even without installing any specialized drivers (i.e. for Audigy 2 ZS from Creative) and it was active (usable) as soon as I connected the same. Though, subsequently I did apply Win 7 64 bit drivers from the support site of Creative. Now even if I applied the drivers for Win 7 64 bit for the device, it is not recognized under Windows.

The config of my old PC was : Intel Core 2 Duo CPU E6540, Intel motherboard, Nvidia GT 220 graphics card, 2 x 250Gb SATA hard disks, 8 Gb RAM.

The config of my new PC was : Intel Core i3 (3rd Gen), Gigabyte motherboard, Nvidia GTX 650 graphics card, 1 x 1Tb & 1 x 250Gb SATA hard disks, 8 Gb RAM.

I do understand that this issue, pertaining to Creative device, may not fall directly under the purview of Microsoft, however, I seek inputs from anyone who may have suggestions or faced identical / similar situations. Thanks for the patience to read this long query.

milind nadkarni say:

BossDweebe,

Thanks for the suggestions. The origin of difficulty was - the PC also has Nvidia GTX 650 graphics card and as a part of the package of drivers for this graphics card, one HD Audio driver (from Nvidia) was also getting installed. This HD audio driver was not allowing Creative Audigy 2 ZS on USB to be recognized as an audio device. Hence, I had to remove the HD audio driver from Nvidia and then install (an updated) driver for the Creative device. That worked.

Thanks for your suggestions.

- Milind
