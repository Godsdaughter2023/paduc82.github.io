How to set Out-of-Office message for Distribution List - Microsoft Community

roegin mangalos say:

How to set Out-of-Office message for Distribution List

Due to holidays, we will have a limited access to our emails. Thus we want to set-up an out of office reply to our distribution lists.

Is that possible? if so, how? Thanks.

rick song msft say:

Hello Roegin,

 

It is not feasible to set up Out-of-office message for a Distribution list, because the Distribution lists don’t have a mailbox. Please let the DL members set up the OOF message by themselves.

 

By default, the value of the following PowerShell command: Set-DistributionGroup -Identity dl@comtoso.com -SendOofMessageToOriginatorEnabled <$true | $false> is set as true. Under this circumstance, the DL members who has set up OOF message will automatically reply to the original sender with the OOF message. So I think it can meet your requirement. If you set the value as false, the members will not send out the OOF message even if they have set it up. Here is an article for your reference: Set-DistributionGroup

 

If it still cannot meet your requirement, please submit the feedback in Office 365 UserVoice forum. Microsoft pays high values on customer’s feedback.

 

Regards,

Rick
