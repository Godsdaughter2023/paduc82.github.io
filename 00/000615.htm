Custom margins in Word - Microsoft Community

laura_lh say:

Custom margins in Word

I created a digital letterhead with custom margins to start below the logo/address header art. The second page only has a small logo header, so I would like a different top margin. When I make this change in the custom margin box the text margin changes correctly, but the header changes from the page two version to the page one version. Is there a different way to achieve this?

john korchok say:

The way to do this is to set Word to have different headers and footers on page 1 (Format>Document, check Different first page).

Then open the header on page one and place a text box with no border. The bottom edge should be where you want the first page top margin to be.

Finally, with the text box still selected, choose the Shape Format tab, click on Wrap Text and choose Top and Bottom. The text on page 1 will start just below the text box, while on subsequent pages it will start at your top margin setting.

  Application.ScreenUpdating = False

  Application.Calculation = xlCalculationAutomatic

When you say “There is no anti-virus software running on this machine” do you mean that there isn’t/hasn’t been an anti-virus installed/running? Or that you temporary disabled it as a troubleshooting step?

 

If you have had an anti-virus program running/installed it is still possible that the anti-virus is causing the problem by being installed (even though currently disabled). As a troubleshooting step, try uninstalling the anti-virus program completely then checking to see if the problem is still present.
