Let's see if your issue is machine or user specific by creating a new user profile and then testing. Follow the steps on the link below to create a new user account:

http://support.microsoft.com/kb/2439218/

Once you've created the new account, log into it and test. Does the same issue occur? If not, this probably means you have corrupt Messenger preferences. Reboot your Mac, ensure Messenger is closed and delete the following files:

  • /Users/username/Library/Preferences/com.microsoft.Messenger.plist
  • /Users/username/Library/Preferences/com.microsoft.Messenger.rtc.plist
  • /Users/username/Library/Preferences/Microsoft/Microsoft Messenger User Cache.plist
  • /Users/username/Documents/Microsoft Messenger.log
    (This file is only present if you turned on the Messenger Logging preference)

Once the files are deleted, open Messenger and test.

Sometimes, Windows or a running application hangs, the user interface abruptly stops responding, and you cannot determine what has caused the problem or how to troubleshoot the issue. This utility tries to detect the software or process that is currently hung, and displays some information that may allow you to sort out and understand what exactly is at the root of such unexpected behavior.
