a. user say:

System Restore was never designed to 'walk the system back prior to when a malfunction or virus hit a system.' That is a common misconception. Only a backup can do that.

A restore point is a collection of essential registry keys, drivers and system files that provide basic functionality to Windows. The idea of a restore point was to hopefully restore just enough functionality for the user to access Windows' own tools and/or their own installed applications in order to solve a problem. In addition, there are no user files (documents, pictures, etc.) in a restore point.

Only an image backup can walk the system back to a prior point in time.

Today backup utilities are easy to find on the web, and people are supposed to be backing up regularly, so System Restore has lost its usefulness.  Another reason why System Restore is less useful is that restore points have a short life. Once a major upgrade has been installed, like the twice-yearly feature upgrades, previously created restore points have to be erased, since you wouldn't want a combination of old and new system files on your computer.

To find System Restore in Windows 10, go to Control Panel > Recovery > Open System Restore.

a.a. ron say:

I finally solved the issue. 

 

After refreshing my computer (which was not needed) then going through all the driver updates, I finally found the solution.

 

On GPT partitioned hardrives (at least mine), when you run Disk Management (windows key+ R, then type diskmgmt.msc then hit enter), EFI System partition is shown as having 100% fee space.

But, if you download a special viewing tool (which I'll provide links to later in this post), you'll be surprised to learn that the drive is full! Mine happened to be full of bitmap images; a poster in another thread indicated that theirs were full of mp3. Either way, you're going to want to delete these files. Unfortunately it's not quite as straight forward as it should be, but it is still fairly simple.

 

The first thing you are going to need to do follow the directions in this article here and run "diskpart".

(my partition is "Volume 4", & I assigned Y: as the pathway, but I suppose you can use whatever letter you want)  

Now that we have assigned a letter to the partition, you are going to need a tool to open and edit the drive. (if you try opening the drive the usual way you look at a drive [file explorer], you will be denied access.

 

Now I used MiniTool Partition Wizard Free Edition, but I was informed by another user in this thread who found a similar solution & tool (which I actually like better). Total Commander is fairly simple to use, and I will explain the rest of the thread using it.

 

After you download and install, right-click the Total Commander desktop icon, and select "run as administrator".

 

In the upper left hand corner, you'll see a drop-down menu containing drive letters. Click on the drop-down menu and select Y: (or whatever letter you temporarily assigned the partition).

 

Now you should see the real contents of this partition, and that it is more than likely full! (but you wouldn't know that according to Disk Management!) 

 

Mine contained probably close to 50 bitmap images, and a [boot] folder & [EFI] folder. I erased the images, but left [boot] & [EFI].

 

Now that this is done, you can go back to Diskpart and un-assign the drive letter to EFI System partition.

 

Now you should be good to upgrade to Windows 10 finally!

andre da costa say:

New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

ronnie vernon say:

Hi

Netbooks, like other computers with pre-installed versions of Windows contain a hidden Recovery Partition on the hard drive. The use of these hidden partitions to recover the operating system are specific to that computer.

Most of these recovery programs will format the system partition when they are used.

You need to read the documentation for that system or contact the computer manufacturers support for the steps you can use to reinstall windows.
