Excel Sumif function

I want to sum different values based on the cell color.  For instance I want to sum all values (in two columns) highlighted yellow.  Is there a way to use the sumif function to achieve this task.  If no, is there another function that will yield the same result.
ossiemac say:

Install the following UDF  (User Defined Function) and it will sum any color. It is not restricted to ColorIndex as per cpearson's example.

However, it does not sum colors applied with Conditional formatting and this cannot be done with a UDF so if conditional formatting then please provide the conditions to set the conditional format and then conditions can be used to create a UDF.

After installing the UDF, you can use the following formula to sum the cells matching a color where $A$2:$A$19 is the range for summing and A2 is any cell with a color to be matched for the Sum (A2 can be within the range to be summed or elsewhere on the worksheet)

=SumByColor($A$2:$A$19,A2)

To install the VBA code:

  1. Alt and F11 to open the VBA editor window
  2. In the VBA editor select menu item Insert -> Module
  3. Copy the VBA code below and paste into the VBA editor.
  4. Close the VBA editor (Cross very top right of VBA editor window)
  5. Save the workbook as Macro enabled.
  6. Ensure macros are enabled. See help for how to do this. (Option to "Disable all macros with notification" should be OK.)

Function SumByColor(rngToSum As Range, rngSample As Range) As Double
    Dim rngCel As Range
   
    For Each rngCel In rngToSum
        If rngCel.Interior.Color = rngSample.Interior.Color Then
            SumByColor = SumByColor + rngCel.Value
        End If
    Next rngCel
   
End Function

    Any code to add a sheet with a name being similar name in counting form? For instance:

    1. Adding a new sheet;

    2. Renaming the sheet as "RESULT_001";

    3. If a sheet with a similar name already exists renaming as "RESULT_002";

    4. Even if only a SINGLE sheet exists in the workbook named as "RESULT_016", renaming the new sheet as "RESULT_017";

     

    Thanx in advance.

    hansv mvp say:

    Try this macro:

     

    Sub AddAndNumber()
        Const strPrefix = "RESULT_"
        Const strFormat = "000"
        Dim wsh As Worksheet
        Dim lngMax As Long
        Dim lngNum As Long
        For Each wsh In Worksheets
            If wsh.Name Like strPrefix & "*" Then
                lngNum = Val(Mid(wsh.Name, Len(strPrefix) + 1))
                If lngNum > lngMax Then
                    lngMax = lngNum
                End If
            End If
        Next wsh
        Set wsh = Worksheets.Add(After:=Worksheets(Worksheets.Count))
        wsh.Name = strPrefix & Format(lngMax + 1, strFormat)
    End Sub

    jeeped say:

    Sub mcrAddResultsSheet()
        iResults = 0

            If Left(Sheets(i).Name, 8) = "RESULTS_" Then
