Windows 7: Can't select to login on local computer account. - Microsoft Community

or what say:

Windows 7: Can't select to login on local computer account.

Windows indicates my network password expired but there is no network account and never has been one.  How do I login to local computer?

 

This is a new computer, it belongs to a workgroup, it has not been setup to login to a network.  It is running Windows 7 and does not have internet access.  The UPS died so it might have spiked the PC.  Its a Dynics computer, out of the box about three weeks ago.  I do not have a Windows 7 DVD but I do have another Dynics computer  

or what say:

Ok, here is the whole story

 

In "User Accounts" I had unchecked "Users must enter a user name and password to use this computer"

 

The account password expired

 

somehow I could not change the password when prompted (either 1. because it was blank, or 2. because it defaulted to a domain account)

 

Josue from Austin told me that to switch from domain to local login to hold the right shift key at bootup

 

This did not work on a USB keyboard so I switched to a PS2 keyboard.

 

Now I am able to select a local user account.

 

So:  A PS2 keyboard and the right shift key at reboot solved the problem after 5 days.

 

Thank you for all the help.  I love this site.
