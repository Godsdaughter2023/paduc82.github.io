a. user say:

> ... Looking to do an Intersect

Hi.   If you want, you "could" use ' Intersect ' because the dates actually fall within the range of Worksheet rows.

I've used this technique in similar situations.   You will have to adjust to what you have.

This is just a general idea...

Sub Demo()
    Dim s1, e1, s2, e2
    Dim Rng1, Rng2, Rng
   
    s1 = DateValue("12/20/15")
    e1 = DateValue("4/17/2016")
    s2 = DateValue("11/7/15")
    e2 = DateValue("2/12/16")
   
    Set Rng1 = Cells(Int(s1), 1).Resize(e1 - s1 + 1)
    Set Rng2 = Cells(Int(s2), 1).Resize(e2 - s2 + 1)
    Set Rng = Intersect(Rng1, Rng2)
    Debug.Print CDate(Rng.Cells(1).Row)
    Debug.Print CDate(Rng.Cells(Rng.Count).Row)
End Sub

Returns:

 2/12/2016

Assuming your references are for endnotes and/or footnotes formatted with the Endnote Reference Style or Footnote Reference Style, as appropriate, you could use:

Sub Demo()
Application.ScreenUpdating = False
Dim i As Long, ArrSty(), Rng As Range
ArrSty() = Array("Endnote Reference", "Footnote Reference")
For i = LBound(ArrSty) To UBound(ArrSty)
  With ActiveDocument.Range

      With Rng
        .Collapse wdCollapseStart
        .MoveStartWhile Cset:=" ", Count:=-1
        .Delete
      End With
      .Collapse wdCollapseEnd
      .Find.Execute
    Loop
  End With
Next
Application.ScreenUpdating = True
End Sub
