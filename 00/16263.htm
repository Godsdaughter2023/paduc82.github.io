Outlook search not working for PST files - Solution

I believe it has to do with the fact that ‘My Documents’ (and my Outlook PST file resides in a directory within that) and other personal folders are on my D: I used the standard Windows method to have my 'personal' user files on my D: and I believe Windows created symbolic links to use to point to these locations. My guess is that these symbolic links reside somewhere on the C: within Appdata.  However when you look at the indexing locations (Control panel/Indexing options) you will likely see that Appdata folders are excluded by default. Therefore those directories do not get indexed and therefore it never actually indexes the PST file.

That is what I surmise but it may be not quite like that, as I do not know the internal workings of Windows. However, when I included Appdata directories in the indexing locations, then forced a rebuild of the index (under Advanced button) rebooted and restarted Outlook, Outlook started reindexing and now my search works again! This took a few hours for the rebuild of the index and likely the same for Outlook to finish. 

 

I don’t think we need to include all of the AppData for indexing as obviously there is a good reason for Windows to exclude it. But something in there (symbolic links is my guess) needs to be included. This may also explain why IMAP search was not affected, because those OST files are on the C: and cannot (at least I could not find a way) be moved.

can I merge multiple worksheets into one excel sheet - Microsoft Community

nanafromnc say:

can I merge multiple worksheets into one excel sheet

I have 20 some worksheets that all have the same number of columns and information in same place.  Is it possible to merge all of them into one worksheet?

Thanks Nana from NC

jeeped say:

... Is it possible to merge all of them into one worksheet?


You could do it with a macro like the following,

Sub mcrGatherAllData()
    Dim s As Integer, c As Integer, r As Integer
    Sheets.Add Before:=Sheets(1)
    Sheets(1).Name = "All Data"
    c = Sheets(2).Cells(1, Columns.Count).End(xlToLeft).Column
    Sheets(1).Range("A1", Sheets(1).Cells(1, c)).Value = Sheets(2).Range("A1", Sheets(2).Cells(1, c)).Value
    For s = 2 To Sheets.Count
        r = Sheets(s).Cells(Rows.Count, c).End(xlUp).Row - 1
        Sheets(1).Range("A" & Cells(Rows.Count, "A").End(xlUp).Row + 1).Resize(r, c).Value = _
          Sheets(s).Range("A2").Resize(r, c).Value
    Next s
End Sub

This macro performs the equivalent of a Copy, Paste Special, Values on all worksheets into a new worksheet called All Data. It assumes that you have a header row in row 1 and your data starts in A2 and extends both right (static number of columns) and down (dynamic number of rows).

Post back with some specifics and/or errors you receive if you are having trouble translating this for your own purposes.
