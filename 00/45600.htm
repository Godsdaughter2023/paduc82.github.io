ronnie vernon say:

Can anyone tell me how to restore desktop short cuts?

Hi

 

If these are shortcuts that you created, you can use the Previous Versions component to do this.

 

 

 

1. Open Windows Explorer and navigate to the following folder.
C:\Users\your name. (Where 'your name' is the name that you use to log on)

 

 

 

2. Right click in that folder and select New/Folder. Name the new folder Desktop Temp.

 

 

 

3. Right click the Your Name\Desktop folder and select the Previous Versions Tab. After the previous versions populates, select a Previous Version of the Desktop folder that has a Date and Time from before you lost the shortcuts you want to recover. Click the Copy Button.

 

 

 

4. In the Copy Dialog that pops up, select the Desktop Temp folder that you created. Click the Copy Button.

 

 

 

This will copy the older version of the Desktop folder to the new  Desktop Temp  folder. You can then Move or Copy the missing shortcuts from there to your current Desktop folder.

Let us know if you have more questions.
