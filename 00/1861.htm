herb tyson say:

If this is the only way that italics are used in the document, then the following search can insert the space for you:

Find what:   leave blank and press Ctrl+I so that Font:Italic appears under the search box.

Replace with: [space]^&   ([space] means for you to press the space bar; so press space, then type caret (^) ampersand (&).

 

What this does is find every occurrence of italics, inserts a space, and then follows the space with whatever matched the italic search (^& = whatever the find search finds).

If you use italics in other ways, then you will need to check each match to ensure that it's what you're looking for. Otherwise, Replace All will do it.
