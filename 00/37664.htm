Hi Velin,

Thank you for posting your query in Microsoft Community. I understand the inconvenience caused to you and will help you in solving the issue.

I would like to inform that if you Upgrade to Windows 10 from Windows 7 the desktop icons in Windows 7 will be carry forwarded to Windows 10. (This applies to upgrade only).

You can add the desktop icons by following the steps below:

  1. Navigate to Start >> Settings >> Personalization.
  2. In the left navigation pane, click on Themes.
  3. In the right pane, click on Desktop icon settings.
  4. Click on checkboxes for the icons that you want back.
  5. Click on Apply and then OK.

In case if you don’t find an icon, you can pin it to the desktop by performing the below steps.

  1. Search for the application which you need to add to the desktop.
  2. Right click on the application >> select open file location.
  3. Right click on the application >> select send to and click on Desktop.

This will add the application which you want it to be on the desktop.
