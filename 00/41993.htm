Windows 10 Compatible Video Cameras - Microsoft Community

kensanders1 say:

Windows 10 Compatible Video Cameras

I keep seeing reviews about cameras not working on windows 10 with Skype. Can you suggest some cameras that will work?

techfreak_ say:
Issues with cameras are generally due to compatibility with Windows 10, not Skype. Older cameras and their drivers may not be compatible with Windows 10 Apps (UWP) but still work with Windows 10 desktop applications. It would be best to locate a camera that fully supports Windows 10 rather than simply supporting an older OS, running in some type of backward compatibility mode. Obtaining a camera that is UVC complaint/compatible should allow that camera to utilize the generic UVC drivers (USB Video Device) that come with an OS like Windows 10. Those drivers are compatible with Windows 10 Apps and Windows 10 desktop applications. If you still run into issues with a UVC complaint camera, make sure you are running the latest build of Windows 10 (currently the April 2018 Update).
