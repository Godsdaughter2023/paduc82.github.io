andreas killer say:
That might be not possible on your PC, because of the limit of your hardware.

As you know, each screen has a certain max. resolution measured in pixels.
BTW, a layout should (as best as possible) always look the same on different computers, the applications use Point as the resolution. One Point is a certain number of pixels.

If you set a row height, you specify the hight in Points. If you try to increase the height from 13.5 to 14.5 by step 0.1 you'll see that not all heights are possible.
This can be reproduced on all computers.

The reason is simple, think of a thin horizontal line drawn at a black & white screen with the width of just one pixel. We can draw the line at e.g. pixel row 547 and at the next pixel row 548.

But it is not possible to draw the line between the rows, because a pixel can be black or white, but can not color itself 50% black and 50% white.

Andreas.
