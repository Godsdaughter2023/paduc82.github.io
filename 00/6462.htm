You can use a VBA procedure to review and apply a "Title" or othe style to short paragraphs:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oPar As Word.Paragraph
For Each oPar In ActiveDocument.Range.Paragraphs
  If Len(oPar.Range.Text) < 15 Then
    oPar.Range.Select
    If MsgBox("Do you want to make this paragraph a title paragraph", vbYesNo, "Make Title") = vbYes Then
      oPar.Range.Style = "Title"
    End If
  End If
Next oPar
End Sub

I encountered the same issue when reviewing changes to the templates in Access 2016. This is not hard but you must know what you are doing when editing the macro.

To fix the error:

Open the Contacts template.

Open the Navigation pane (on the far left)

Find the AutoExec macro in the "Macros" section of the Navigation pane

Right click on the AutoExec macro icon and select "Design View"

Close and save the editor.

Close and re-open the database.

The contact list will load as the default form.
