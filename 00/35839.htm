Liam,

1  It is worth making an additional [spare] password-protected local Admin account before going ahead with what you plan to do.  If your day-to-day Admin user account is ever corrupted [and user profile corruption does happen], having such a spare Admin account ready to use can turn a potential disaster into a petty annoyance.

  • Start
  • Settings,
  • Accounts,
  • Family & other users,
  • [Other users section],
  • Add someone else to this PC,
  • I don't have this person's sign-in information,
  • Add a user without a Microsoft account,

  • Enter a username, password & password hint,
  • Back in the list of other accounts you can now see in Family & other users, [Other users section], 
  • Click on the new account,
  • Click on Change account type,
  • Select the Administrator account type,
  • OK.
  • button, [Just in case you have heard of Password reset disks for local user accounts, note that they achieve nothing more than writing the password down achieves]

2  If, as I assume, you used an elevated command prompt to enable the Built-in Admin then the command can be reversed while you are logged into your day-to-day Admin account by issuing this command in an elevated command prompt

net user Administrator /active:no

Microsoft is aware of an issue that may result in “Hey Cortana” telling the user they must first enable Cortana, even when Cortana is already enabled.  This may occur after installing the Anniversary Update and then doing a device reset, but is only known to occur in a few models (Lumia 930, 1520, Icon) and a few languages (English (US), English (UK), Simplified Chinese).  We are currently investigating a fix for this issue.

To work around this issue and re-enable the “Hey Cortana” functionality, revert your device to Windows Phone 8.1 using the Windows Device Recovery Tool.  Once you are back on Windows Phone 8.1, install all phone updates, then enable Hey Cortana, and then install all Hey Cortana updates from the Store.  Once your Windows 8.1 Phone is fully upgraded and Hey Cortana is fully trained and working, you can use the “Upgrade Advisor” app to upgrade back to Windows 10.  Then when you reinstall the Anniversary Update, Hey Cortana will be available and functional.

Thank you.
