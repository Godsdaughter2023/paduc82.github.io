Press Ctrl + H to start the replace function.

Replace _

^c.

Select the whole document (CTRL+A) then update the fields (F9)

graham mayor say:
If CTRL+F9 is not working, chances are that you have dual function F keys and they are switched to the alternative function.  This is a field simply typing { } will not work.

You could also do it with a macro

    Set oRng = ActiveDocument.Range
    ActiveWindow.View.ShowFieldCodes = True
        With oRng.Find
        Do While .Execute(FindText:="_")
            oRng.Fields.Add oRng, 12, "Num", False
            oRng.MoveStartUntil Chr(21)
            oRng.Start = oRng.Start + 1
            oRng.Text = "."
            oRng.Collapse wdCollapseEnd
        Loop
    End With
    ActiveWindow.View.ShowFieldCodes = False

Hello Jack,

I understand your concern regarding using the Mail App Offline in Windows 10. We will help you.

Mail for Windows 10 doesn’t have a contacts or people folder like other mail clients.

If you are using the Mail App Offline, you will not be able to see the new incoming mails as the system will not be connected Internet. if you looking for the Contacts, you have a separate People App where it stores all the Contacts for you to use in the Mail App.
