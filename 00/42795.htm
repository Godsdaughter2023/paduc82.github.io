hassanb say:

Hello,

You will need to first gain access to the parental account in order for you to be able to successfully agree to the terms and conditions. You won't be able to change the parental e-mail without being able to login to the prior e-mail that's listed.

When you created this account (under the age of 18) you were required to enter in an e-mail before you were able to sign up. You must have put in your mother's email. There's no way that Microsoft would be able to just find your mother's e-mail and add it to your account, it's not how it works. Since you're almost 19 now you should be able to promote your account from a child to an adult after you're able to agree to the updated terms on the parental account. You should be able to do so by following the directions here.

Please remember that this is not something that is done automatically, you need to initiate this process, and it can only be done after agreeing to the latest terms.

The parent must agree to the new terms first and then you can login to your own account and accept them. If you don't have access to the parental account you will need to try to reset the password found here. If that doesn't work try to fill out the Microsoft Account Recovery form found here.

If you're unable to either of those things then you may be out of luck. You can try to contact phone support and see if they are able to assist you in some way.

Further information in regards to the new terms of service agreement update can be found here.

jose j pascual say:

Windows Update service missing 0x80070424 , Windows 10 Pro x64 1803 [Solved] / Error code 0x80070006 or 0x8008005

Hi all.

I've just managed to solve this issue, not being able to find any solution over the Internet after several days of search, test and fail, so I'm sharing it here in case somebody is in the same situation. I hope it is not against any policy doing so.

Version: Windows 10 Home/Pro x64 1803

Issue:    Missing service Windows Update, as a result of deletion  ( sc delete wuauserv ) . Stupid stupid action, I know, but in Home edition, I was out of ideas to avoid automatic updates not having gpo.

             Running services.msc doesnt show the Windows Update service. In Regedit the key[HKLM\SYSTEM\CurrentControlSet\Services\wuauserv] doesn't exist or is empty.

             Windows Store update and download also show errors

error:    0x80070424 when trying to refresh updates manually 

             0x80070006 after a wrong .reg recovery

Windows update troubleshooter, both the one in the windows settings or https://support.microsoft.com/en-us/help/10164 , fail to run. The last troubleshooter also includes running:

DISM.exe /Online /Cleanup-image /Restorehealth  
sfc /scannow

I tried using a .reg available online to recover the service, located at this place: https://www.tenforums.com/tutorials/57567-restore-default-services-windows-10-a.html . The service was back but I have then the error  0x80070006.  It might be due to being a different or early Windows version.

I asked a friend with a Professional 1803 version (we also used different languages for install and run Windows) to export his services\wuauserv registry. I imported it, restarted, and my updates are back. I honestly hope it might help someone out there. Best luck if you need to use it.

Cheers,

Joseps

zigzag3143 (ms -mvp) say:
[quote]
"It's not a true crash, in the sense that the bluescreen was initiated only because the combination of video driver and video hardware was being unresponsive, and not because of any synchronous processing exception".

Since Vista, the "Timeout Detection and Recovery" (TDR) components of the OS video subsystem have been capable of doing some truly impressive things to try to recover from issues which would have caused earlier OSs like XP to crash.

As a last resort, the TDR subsystem sends the video driver a "please restart yourself now!" command and waits a few seconds.

If there's no response, the OS concludes that the video driver/hardware combo has truly collapsed in a heap, and it fires off that stop 0x116 BSOD.

If playing with video driver versions hasn't helped, make sure the box is not overheating.

Try removing a side panel and aiming a big mains fan straight at the motherboard and GPU.

Run it like that for a few hours or days - long enough to ascertain whether cooler temperatures make a difference.

If so, it might be as simple as dust buildup and subsequently inadequate cooling.

I would download cpu-z and gpu-z (both free) and keep an eye on the video temps http://www.cpuid.com/

xbf diego m say:

Hi

Any DLC that you have purchased digitally is yours, whether the game or DLC get's removed from the Store. Once it is removed, you can no longer purchase the DLC, but if you already have it you have nothing to worry about. The main reason that content gets removed is because the developer chooses to stop providing support for games or providing content for those games. The developer has an agreement with Microsoft to maintain the game, but once the developer chooses to stop supporting the game it may get removed. This does not occur with backwards compatible games as Fable's developers are no longer in business, but the game is still available on the Microsoft Store just to provide an example.

The type of games that most commonly experience removal from the Microsoft store are franchise games. These types of games are NBA 2K, WWE 2K, Madden and more. The developer will come out with a new version of the same game almost every year and continue to provide support for some of the previous versions of the game, but will stop supporting much older versions of the game. The reason that Microsoft will remove a game from the Microsoft Store can also be attributed to copyright or licensing issues. This situation does not occur very often, but it is possible.
