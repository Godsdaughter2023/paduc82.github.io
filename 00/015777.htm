greginmich say:

You won’t find that option in the GUI, but it definitely is an option with the CLI. For a scheduled MSE scan that does this (and way more), just follow the instructions here:

http://answers.microsoft.com/en-us/windows/forum/windows_8-performance/windows-defender-windows-8-schedule-option-missing/b6603c6a-cba9-4ba7-a894-37f895fc6c92

 

If you want to shut down the PC after a manual scan, you can use a batch file – just open Notepad and copy and paste these lines:

 

@echo off

powercfg -change -standby-timeout-ac 200

Cd C:\Program Files\Microsoft Security Client

MpCmdRun.exe -Scan -ScanType 2

powercfg -change -standby-timeout-ac 20

shutdown -s -t 300

 

Then name the file something like “Scan and Shutdown”, give it a .bat file extension, and save it to your desktop. Double-click on the icon to start the scan. If you want a “cancel” button, copy “shutdown /a” to a Notepad file, name it “Cancel the Shutdown.bat”, and then save it to your desktop. When the shutdown warning appears, double-click the icon to cancel.
