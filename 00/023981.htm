doug robbins - mvp office apps & services (word) say:

No, the version of Windows has nothing to do with it.

To get the new document to be black, you will need to create the following macro in the Normal.dotm template

Sub AutoNew()
With ActiveDocument
    With .Background.Fill
        .ForeColor.ObjectThemeColor = wdThemeColorText1
        .ForeColor.TintAndShade = 0#
        .Visible = msoTrue
        .Solid
    End With
    .ActiveWindow.View.DisplayBackgrounds = True
End With
End Sub

Or, if you have already gone through the process of opening the Normal.dotm template and changing the Page Color, an AutoNew macro containing the following command will cause new blank documents to have that color

ActiveDocument.ActiveWindow.View.DisplayBackgrounds = True

Having weird scrolling mouse/touchpad issues? Try uninstalling - Microsoft Community

danwdoo say:

Having weird scrolling mouse/touchpad issues? Try uninstalling Logitech's SetPoint

Throughout my testing of Windows 10 I have experienced weird, inconsistent issues with mouse/touchpad scrolling. Some apps seemed to scroll perfectly, while others acted less consistently. The worst in my experience was the Photos app with my touchpad. The slightest touch of the touchpad would send the app scrolling at lightning speed. It made the app completely unusable. I tried every mouse/touchpad setting available in Windows settings as well as my touchpad/mouse driver software, but nothing really seemed to make things better in the long run. One time things would work, but then they always seemed to get wonky again. There always seemed to be an inconsistency between how modern apps scrolled and how desktop apps scrolled. I posted several issues about this in the feedback hub and gave negative feedback on touchpad surveys in new builds because of it.

Then the other day I uninstalled Logitech's SetPoint software. It was like a miracle. All of the sudden everything scrolled perfectly and accurately. To verify this is what really was causing this, I then installed the latest version again and instantly started to see the same bizarre behaviors again. I then Uninstalled it and rebooted again and presto. Everything works perfectly again. I had tried every setting in the software and none of it made a difference. So if you are experience strange mouse/touchpad scrolling issues, this might be worth a try. Just be sure you reboot before testing as some apps (photos) did not start working properly again until I had done so.

I have reported this in the Logitech forums, but thought this might be useful to people here as well.
