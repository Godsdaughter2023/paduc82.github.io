how to share a PDF file via email

I have received a pdf file and it saved in my phone..im unable to view that file from the "FILES" application and I can view that through adobe reader only.my question is how can I attach that file in an email???
julie fka profjulie say:

Since the PDF file is stored within the Adobe Reader app's file system, you won't be able to share it via email, but you could upload it to your OneDrive account and then share a link to the file from there.

A better app to use if you want to share PDF files via email is the OneReader app, which allows you to share PDF files that are stored within its file system via Bluetooth, Email or by uploading to OneDrive.
