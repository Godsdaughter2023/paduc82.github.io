samhrutha g say:

Hi Aging Hippie,

 

You may create a new task and check the results, follow the below steps to create a new task:

Make sure you have correct settings in triggers.

a.     Click on Start, Type task scheduler in the Search box and hit Enter.

b.     On the extreme right hand pane of task scheduler, click on Create Task.

c.      Under the General tab, give a name to the task and select appropriate security settings.

d.     Under the Triggers tab, you can schedule the task to run at what time and when it should stop.

e.     Under the Action tab, choose the program which you would like to start, such as Internet Explorer, media center or Windows media player.

f.       Select Conditions and Settings appropriately.
