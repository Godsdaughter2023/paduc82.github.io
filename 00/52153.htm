How can I get my OneDrive files indexed so they will be found by a (Windows key +F) Search?

cmdrkeene say:

The scope (how much stuff needs to be searched) is much larger from the default search.  The reason opening the folder is so much faster is because it does not need to scan the whole index, it just needs to scan the one single OneDrive section inside the index.

But you can help speed up the default search:

  1. Remove index locations that you don't need indexed (like system folders, entire hard drives, etc).  Only index places that store documents/files with index-able meta-data that you want to search by (file name, author, size, artist, date taken, etc.).  The smaller the index (in terms of locations, not number of files in those locations), the better.

  2. Run the Search Diagnostic (built in from Control Panel / Troubleshooting section) to optimize the search index.
