By default Onenote stores backups locally. These backups are section oriented. The backup date is part of the file names.

Under File, Options, Save you find some parameters to influence the backup behaviour of Onenote:

To use it for recover read this article:

http://www.ilovefreesoftware.com/31/tutorial/restore-onenote-backup.html

The "files" on Onedrive cannot be used for backup.

For recover only a double click on such a .onepkg file is necessary.

First of all, open System Preferences>Keyboard>Input Sources and make sure you have a French Canadian keyboard. Click the plus sign to install one if you don't.
