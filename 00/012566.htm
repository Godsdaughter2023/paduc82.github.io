Internal file sharing/syncing with OneDrive - Microsoft Community

diane banks say:

Internal file sharing/syncing with OneDrive

Is it possible for the files to automatically sync to someone else with the same domain without sharing? I have used Dropbox in the past, and when a file is added, everyone with permission in the organization has access to that file. Is this possible in OneDrive or do you have to share every file?
sheen valentine ali msft say:

Hi Diane,

You can share an entire document library to your organization/specific user and have them sync that library to their desktop. If someone made changes to the library, such as uploading a new file, the sync client will sync this new file to any user that has the library synced to their computer.

You can share a document library by:

1. Going to Library settings

2. Permissions for this document library

3. Stop inheriting permissions (from the site)

4. Grant permissions to the Everyone except External Users group or a specific user

5. Sync SharePoint files with the new OneDrive sync client

Regards,

Sheen
