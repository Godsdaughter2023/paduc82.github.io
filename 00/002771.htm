aaron_sum say:

Hi,

Thank you for posting to Microsoft Community Forums.

In Windows 10, the local music on your system comes with album art and metadata description. The metadata includes song name, artist name and even genre. There are a few ways to edit this metadata, please check the methods below.

Method 1: Editing metadata using File Explorer.

  1. Press the Windows key and E key, this will open the File Explorer for you.

  2. Go to the location where the songs are stored, right-click on the song and select Properties.

  3. This will open a window for you, click on the Details tab. Here you will be able to see the metadata fields inclusive of album name, artist, genre, publisher etc. that can be edited.

     

    Note: The music files with file.  

  4. Edit the required fields and click on Apply and Ok.

Method 2: Edit metadata using Windows Media player.

  1. Open Windows Media Player.

  2. Click on Organize at the top left corner of the Media Player and select Options.

  3. Under the Library tab, check Write media information changes to files.

  4. Click on Apply and then Ok.

  5. If your songs are in the Music library folder within your File Explorer, you will be able to find the songs by clicking on the Music tab on the left pane.

  6. Adjacent to each song, you will be able to find the fields for title, contributing artist, etc.

  7. Right-click on the field you wish to edit, select Edit.

Hope it helps.
