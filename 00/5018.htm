How do I make a table of my crosstab query? - Microsoft Community

ryancross say:

How do I make a table of my crosstab query?

I have a crosstab query from my master table.  Site is the row header and Team is the column header.  How do I do a "make table" of this crosstab query?
hansv mvp say:

Save the crosstab query and close it.

Create a new query in design view.

In the Show Table dialog, activate the Queries tab, select the crosstab query, then click Add.

Close the Show Table dialog.

Add * from the field list to the query grid.

Click Make Table in the Query Type group of the Design tab of the ribbon.

Specify a name for the new table, then click OK.

Click Run in the Results group of the ribbon and confirm that you want to add records.
