The error code (0xc0000005) translates to STATUS_ACCESS_VIOLATION, which can occur for a variety of reasons.  I'm sure you noticed this if you did any searching before you wrote your post. Causes include failing/defective memory (RAM), virus/malware infection, or damaged software (including the operating system).

The steps below assume that you have provided all the information we can possibly get.  It may be helpful to know which Windows XP service pack level you had, as well as the program(s) listed in the title bar of the error messages.  For example:  Systray.exe - Application Error

Did you recently install new memory or other hardware?  If so, removing the new equipment and/or going back to your old memory may fix the problem. 

Run a couple of different memory test programs.  I've personally used Memtest86+ and Microsoft's Windows Memory Diagnostic (info and download here).  These tools boot from a floppy diskette or iso (CD), and run a variety of tests on your computer's memory.  If any problems are found, you should probably replace the affected memory module.

If neither of these resolves the problem, it's probably due to a software problem somewhere.  Windows files may have been corrupted (accidentally or intentionally), you might have a virus/trojan/worm/spyware, or some other program could have a bug or conflict. 

I would just perform a fresh reinstall of Windows, restore my data from a backup (you do back up your system, don't you?), and think about what happened before the problem started.  Get some antivirus protection if you don't already have it- Microsoft Security Essentials is free and seems to work well. Nothing is bulletproof, however, so be sure you don't put your computer at risk from stuff like websites or software from the "wrong side of the tracks".

If your computer came with a System Restore CD, you can use that to reinstall Windows and any preinstalled software.  You would still need to restore documents or other data from backups. You might be able to do a "repair installation" of Windows, or boot from a "live cd" so you can get to your documents.  In my experience, however, the time and effort required far exceed the value of any documents I don't have backed up.
