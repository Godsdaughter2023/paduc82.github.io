Hi Elra,

 

Welcome to Microsoft Community. Your interest in Windows 10 Technical Preview is much appreciated. 

 

Limited Connectivity error means the computer has successfully connected to the router, but the PC wasn't assigned a valid IP address.

 

What is the make and model of the computer?

I would suggest you below methods check if it helps:

Method 1: Change DNS settings

 1.  Open Control Panel

 2.  Click Network and Internet

 3.  Select Network and Sharing Center

 4.  Click Change Adapter Settings.

 5.  Click the connection you want to configure.

 6.  Click change the settings of this connection

 7.  Scroll down to Internet Protocol Version 4 (TCP/IPv4)

 8.  Double click Internet Protocol Version 4 (TCP/IPv4)

 9.  Click Use the following DNS server address

 10. Fill in the DNS server info for your ISP.

 11. Click OK and close all windows.

If the issue still persists, follow next method.

 

Method 2:

STEP 1:

1. First create a system restore point. Then right-click the ‘Start’ button to access ‘Control Panel’ option and select it.

2. Next, chose ‘Device Manager’ option.

3. Next select ‘Network Adapters’ and under it, you will find a Wi-Fi controller listed.

4. Double click on Broadcom 802.11a network Adapter to open the adapter’s Property window and switch to ‘Drivers’ tab.

 (i.e open your wifi N/W adapter properties)

 (Note:- you might have Broadcom 802.11n network Adapter than select that)

5. Click the ‘Disable’ button and hit ‘Ok’ to apply the changes.

See if it helps. If it does not help, re-enable it and perform the below steps:

 

STEP 2:

1. Go back to 4th step and on Driver tab, click on Update driver

2. Select browse my computer for driver

3. Click on let me pic from list of device

4. Uncheck show compatible hardware

5. Now from the manufacturer window select Broadcom and from the Network Adapter list select the driver having name Broadcom 802.11a Network Adapter and re connect your wifi.

Method 3:

You can reset the TCP/IP stack from the Command Prompt. Follow the steps.

  1. Press Windows +X.
  2. Click Command prompt (Admin mode).
  3. Type ‘netsh int ip reset c:\resetlog.txt’ without quotes.
  4. Press Enter.
  5. Reboot the computer.
