juliaanthony say:

I'm not receiving attachments with emails sent to multiple recipients

I am using Windows Live Mail with Windows 7.  When emails are sent (with attachments) from one particular sender to three or four recipients, I don't receive the attachment - the others all do.  When the attachment is sent only to me I receive it.  The attachments in question are both PDF and Word.
¡firedog say:
I don't receive the attachment

Don't receive - or don't see? You can often tell from the size of a message whether it is just a message or it has a file attached, which will add quite a bit to the size.

Could it be that the sender and the other recipients are all using Outlook? If so, these messages are probably being sent to two different contacts - {Julia as Julia} and {Julia as member of group}. These two contacts may have the same email address or different ones. That doesn't matter. What does matter is the message sending format that the sender is using, and it looks as if this might be different in the two cases. Attachments to messages sent using Outlook's proprietary RTF format can only be decoded by Outlook (or Exchange Client, which is more or less the same). No other mail client (Outlook Express, Windows Mail, Windows Live Mail, Thunderbird ...) can understand them.

You'll find an explanation of what's happening and remedies for it on this page: How e-mail message formats affect Internet e-mail messages in Outlook. Note that there's nothing you can do about it; only the sender can change the sending format. RTF should only ever be used within the corporate environment, and only where the sender knows that the addressee is using Outlook.
