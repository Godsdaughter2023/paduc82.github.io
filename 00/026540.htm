This error may occur if the program is unable to communicate with certain system services using Remote Procedure Call. The issue can be fixed by installing the right drivers. Before installing the drivers, I would suggest to uninstall everything related to Realtek audio in device manager, however before you uninstall the drivers, ensure that the following services are running. Open the Run dialog box by pressing the Windows key and the R key together. Type SERVICES.MSC and press the Enter key to open the list of local services. Find the below mentioned services and make sure that they are running.

 

  • Realtek HD Audio Universal Service.

  • DCOM server process launcher.

  • RPC Endpoint manager.

 

If the services are already in running state, proceed with uninstalling the Realtek drivers. Follow the steps given below.

 

  • Right click on Start button (Windows logo) and select Device Manager.

  • Remove all Realtek components under Audio inputs and outputs, Software components and Sound, video and game controllers by right clicking on the individual items one by one and selecting Uninstall device.

  • Reboot the computer and install the latest Realtek audio driver from the system manufacturer’s support site.

  • Download and install the Realtek Audio Control Universal app from Microsoft Store and see if it works.

     

    Realtek Audio Control

 

If the above method doesn’t fix the issue, search for Realtek app on the task bar or the system tray (bottom right corner), right click on it and select App settings. You will the find the Reset option. Click on it to reset the app which should fix the issue.
