If all of you time data was in the form mm:ss in the CSV file, Excel interprets that form as hh:mm.

So at this point, you would need to put the following formula into a parallel column:
=K2/60

You could then copy the parallel column and use paste-special-value to overwrite the data in column K.  Then delete the formulas in the parallel column.

Format the data that you want to keep any way you wish, perhaps Custom hh:mm:ss.

Moving forward, the only way to coerce Excel to correctly interpret min:sec data in a CSV file is for it be written in the form mm:ss.0 or 00:mm:ss.
