Use Command Option Shift V instead of Command V.

If that key combination seems awkward, you can change it or add a new one. To do that:

  • In the main menu bar, select Tools > Customize keyboard
  • In the list to the left, select Edit
  • In the list to the right, locate PasteDestinationFormatting
  • Place the cursor in the Press new keyboard shortcut field
  • Type Command V
  • Word will let you know this is already assigned to the regular Paste commmand (called EditPaste)
  • Click Assign and then OK

There is no need to delete the existing combination, as you can have multiple shortcuts to the same command.

If you later want to undo your shortcut reassignment, repeat the above steps but select EditPaste and assign Command V to it.

john korchok say:

In Excel, when you paste and match destination formatting, Excel does a Paste Special using the VALU option, pasting only the cell values. You can create a custom keyboard shortcut for this command:

  1. In Excel, choose Tools>Customize Keyboard.
  2. In the Categories list, scroll up to All Commands, above File and select it.
  3. In the All Commands list, scroll down to Paste Values and select it.
  4. Select the Press new shortcut key field and press your preferred shortcut key combination.
  5. Click on the Add button. OK out.

This can be solved through Conditional Formatting.

Let's say cell B1 of Tab1 contains the drop down and it has values like Euro, Dollar, Pesa, Yen etc. 

Apply the conditional formatting to a cell by following way -
