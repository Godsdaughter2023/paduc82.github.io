Word 2016 - Tables, How To Prevent Row From Breaking Across Page - Microsoft Community

jgreg7 say:

Word 2016 - Tables, How To Prevent Row From Breaking Across Page

In Word 2016, when I use tables, the first thing I have to do is to go into the settings and disable "Allow row to break across page".  This is tedious since I use a large number of tables, and always have to change each one manually.

Is there a way to change this "Allow row to break across page" setting to default to unchecked at start-up?

jay freedman say:

I don't know of any place to change the default, either in the options or in the registry. However, I can give you a workaround to reduce the job to one click or one keystroke. Install the following macro in your Normal template and put a button on the Quick Access Toolbar to run it, and/or assign a keyboard shortcut to it.

Sub TableRowsNoBreak()
    If Selection.Information(wdWithInTable) Then
        Selection.Tables(1).Rows.AllowBreakAcrossPages = False
    End If
End Sub

When you insert a table, while the cursor is still in the table, run the macro.

An alternative, although not a very good one, is to install the following macro:

Sub TableInsertTable()
    Dim dlg As Dialog
    Dim tbl As Table
    Set dlg = Dialogs(wdDialogTableInsertTable)
    dlg.Display
    Set tbl = ActiveDocument.Tables.Add(Range:=Selection.Range, _
        numRows:=dlg.numRows, numColumns:=dlg.numColumns)
    tbl.Rows.AllowBreakAcrossPages = False

    Set dlg = Nothing

End Sub

This one will run when you click the Insert > Table button and choose the "Insert Table..." entry below the set of blocks (or you can give it a QAT button or shortcut). It displays this built-in dialog box

then creates the table and applies the Do Not Break option. The dialog is a bit less convenient than the boxes for choosing the rows and columns.

The N93 only outputs composite video that has the color signals mixed onto the chroma subcarrier, you see black and white because of the backwards compatibility with old black and white TVs. Anyway withoug getting into more technical details, it not as straight forward as having a connector adapter to change from RCA to the 4 pin S-video cable.
