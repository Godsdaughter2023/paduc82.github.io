Take a USB-drive that's large enough to store all your data. Then just copy the folders/files you want to keep to the USB-drive.

Now, instead of a re-installation I would definately try some other things first. A re-installation is very invasive on the system and very time consuming.

I'd start with extensive malware scanning. Take another computer and download Malwarebytes Anti-Malware, SuperAntiSpyware, GMER and TDSSKiller. Install these on your laptop and run all four of them. In MBAM and SAS do full scans, not quick scans. Remove any and all items found. Reboot your pc after each routine.

Now, when all four of them have finished run an elevated CMD (to open an elevated Command Prompt type: click on Start and type CMD in the search bar. Now press Ctrl+Shift+Enter. An elevated CMD screen will now open. In this screen type: sfc /scannow. Let this run (sfc = system file checker) and when it's done reboot and get back to us with any results.

***

If you do wish to reinstall: make sure that you check your paid for software (if any) on how to reinstall (think license keys etc.) There are some very important things to think of when you do a re-installation and this is one of them. Also check if your COA is well readable/ make a copy of your email database-files/ make a copy of your contacts/ make a copy of your browser's favorites etc/ etc/ etc. Think it over, GOOD.

Good luck and please keep us posted!

Sebastian.

Method 2:

Let’s boot the computer in Safe Mode and check if issue reappears. Safe mode is a diagnostic mode that starts your computer in a limited state. Only the basic files and drivers necessary to run Windows are started. The words Safe Mode appear in the corners of your monitor to identify which Windows mode you’re using.

Step 1: Safe Mode

Windows Startup Settings (including safe mode).

http://windows.microsoft.com/en-gb/windows-8/windows-startup-settings-safe-mode

Step 2: Clean Boot

Putting your system in Clean Boot state helps in identifying if any third party applications or startup items are causing the issue.

Refer: How to perform a clean boot to troubleshoot a problem in Windows 

http://support.microsoft.com/kb/929135

Note: Refer "How to reset the computer to start as usual after troubleshooting with clean boot" under more information to reset the computer to start as normal after troubleshooting.

Method 3:

If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.
