Hello,

Thanks for the support message!

This error code suggests that a child account is having trouble loading family member profiles.  You may see this error code in the following format:

801540A9

Sorry, this profile can't connect to Xbox LIVE on this console. To fix it, download it again.

Solution

Re-associate the child account to the parent account. To do so:

  1. Sign in to your Microsoft account and select the Family tab.
  2. Select Add a child.
  3. Enter the email address of the child account that you want to re-associate with the parent account. This will send an invitation to the child account’s email address.
  4. Sign out of your Microsoft account and sign in to the child account.
    Note For security reasons, you may be asked to verify the account.
  5. Select the button labeled Sign in and accept.
    Note Email can be found by selecting View inbox.

If you keep experiencing the problem, Contact Us.

Change Wireless Channel

Login Information

  • Gateway - 192.168.1.1 or 192.168.254.254
  • Username - admin
  • Password – admin

Change Wireless Channel

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.254.254 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Wireless.
  4. Select Advanced from the submenu on the left.
  5. From the drop down menu labeled Channel select the desired channel. (Channels 1,3,9,11).
  6. Select Apply/Save.
