Slide Titles In Powerpoint 2003 - Microsoft Community

uttt say:

Slide Titles In Powerpoint 2003

Hi,

I am trying to format the slide titles in my presentations. Is there a way to format all the titles in a presentation at one time instead of formatting the color, font etc for each title one at a time. I have powerpoint 2003 version.

Thanks for any tips.

 

steve rindsberg say:

If your presentation has been properly set up, it's very simple. 

First, have a look at Outline view.  Do you see your slide title text next to each of the slide icons in that view?  If so, you're good to go.

Choose View, Master, Slide Master.

For each of the slide masters you see in that view, select the slide title in the slide view on the right and format it however you wish.

When go back to normal view (View, Normal) your slide titles will have assumed the new formatting.  Sometimes you'll need to reaplly the slide layout to a slide if it's been extensively reformatted.

If your title text doesn't appear in outline view, it means that somebody's deleted the title placeholders (which follow the master formatting) and replaced them with plain text boxes (which don't). 

In that case, if you're only going to do this reformatting once, I'd use the format painting brush; apply the formatting you want to one of the titles, pick it up with the format paintbrush, then apply it to the other titles.

If you'll be using this presentation for a while and think you might need more wholesale changes later, let us know.  We'll walk you through getting the placeholders back on your slides and the text into 'em.

 

Steve Rindsberg PowerPoint MVP

jim gordon mvp say:

The message you received has nothing to do with macros.

There are some reserved words in Excel. For example, you are not allowed to use the word History to name a worksheet.

The workbook you are trying to open must be quite old. At one time Print_area was not a reserved word and a named range or object is called Print_area in this workbook. Back in an older version of Excel, when you opened such a workbook having a newly off-limits reserve word Excel would rename the conflicting name for you automatically. Apparently Excel 2016 lost this capability.

Since you can open this workbook in Excel for Windows, find the object or range named Print_area and rename it. If that object or range is used in the macro, change the code to reflect the new name. Then use File > Save As and change the file type from old .xlm to the current .xlsm macro enabled file format.
