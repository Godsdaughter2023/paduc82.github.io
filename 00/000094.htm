ken sheridan say: To apply a parameter add a YEAR([WithdrawalDate]) column in the design grid, select WHERE in the 'Total' row, and enter the parameter in the 'Criteria' row of this column.  Note that in a crosstab query a parameter must be declared as the relevant data type, which you can do through the Parameters dialogue in query design view.
