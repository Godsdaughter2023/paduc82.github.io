How to Refresh all pivot table in excel file - Microsoft Community

couponskiss say:

How to Refresh all pivot table in excel file

How to refresh all pivot table when I open the excel file. Is there any formula or tool available for refresh all my pivot table automatically?

Thanks in advance.

hansv mvp say:

You can set a pivot table to refresh itself automatically when the workbook is opened:

  • Right-click anywhere in the pivot table.
  • Select Pivot Table Options... from the context menu.
  • Activate the Data tab.
  • Tick the check box 'Refresh data when opening the file'.
  • Click OK.

Or, to do it manually:

  • Click in any pivot table.
  • On the Analyze tab of the ribbon, under PivotTable Tools, in the Data group, click the arrow at the bottom of the Refresh button and select Refresh All from the dropdown menu.

Alternatively, you can create code in the ThisWorkbook module of the workbook:

Private Sub Workbook_Open()
    Me.RefreshAll
End Sub

Save the workbook as a macro-enabled workbook (.xlsm) and make sure that you allow macros when you open it.
