net user administrator /active:no [enter]

Exit the command prompt.



You absolutely do not want to have only one user account. Like XP, Vista and all other modern operating systems, Windows 7 is a multi-user operating system with built-in system accounts such as Administrator, Default, and Guest. These accounts should be left alone as they are part of the operating system structure.

You particularly don't want only one user account with administrative privileges in Windows 7 because the built-in Administrator account (normally only used in emergencies) is disabled by default. If you're running as Administrator for your daily work and that account gets corrupted, things will be Difficult.  It isn't impossible to activate the built-in Administrator to rescue things, but it may be more than you want to do. Better not to put yourself into a bad situation to begin with.

The user account that is for your daily work should be a Standard user, with the extra administrative user (call it something like "CompAdmin" or "Tech" or the like) only there for elevation purposes. Running as a Standard user is best practice for security purposes and will help protect your computer from infection. After you create "CompAdmin", log into it and change your regular user account to Standard. Then log back into your regular account.

Enter the correct password for that user account (if there is one) when prompted. Leave it blank if there is no password (null).

mvvksm say:

how to print file details of a music folder

lemp say:
AFAIK, you would have to write a script to retrieve and print this information.  You can see the Duration field in Windows Explorer by clicking View > Choose Details and selecting "Duration" from the list, and then View > Details.

However, again AFAIK, not all music files have these extended attributes (at least as far as seeing them in Windows is concerned).  After setting my Windows Explorer "Details" view to include Duration, mp3 files showed the information, but files downloaded from iTunes as M4P or M4A did not.

A sample Visual Basic Script (to be saved as *.vbs) that will display all of the extended attributes of a file is here: http://technet.microsoft.com/en-us/library/ee176615.aspx

To see how it works, copy the script and paste it into an open Notepad window and save as test.vbs.  Then double click it.  You'll have to edit the file name in the third line from C:\Scripts to a folder that includes mp3 files.  For your first test, pick a folder with only a few files, because as it stands, that script pops up a dialog box (on which you have to click OK) for each attribute of each file.

A list of all of the attributes is here (this script is written in KiXtart, a different scripting language):  http://www.kixtart.org/forums/ubbthreads.php?ubb=showflat&Number=160880&page=1

From this link, you can determine that "Name" is attribute 0 and "Duration" is attribute 21

You could then modify the sample file from the first link by replacing following FOR - NEXT statement

For i = 0 to 34 Wscript.Echo i & vbtab & arrHeaders(i) _
  & ": " & objFolder.GetDetailsOf(strFileName, i)
Next

with two simple commands to only get attributes 0 (name) and 21 (duration) for each file

Wscript.Echo 0 & vbtab & arrHeaders(0) _
 & ": " & objFolder.GetDetailsOf(strFileName, 0)
Wscript.Echo 21 & vbtab & arrHeaders(21) _
 & ": " & objFolder.GetDetailsOf(strFileName, 21)


Also be sure to replace "C:\Scripts" in the third line of the script with the fully qualified file name of your music folder. 

I leave as an exercise for you how to add the necessary loop to deal with the situation of your music folder having subfolders.

Save the file as GetAttribs.vbs (or whatever file name you like, ending with .vbs)

Open a Command Prompt window (Start > Run > cmd > OK)
Paste (click the icon in the upper left corner to get the menu, then select Edit > Paste) the following

cscript //nologo "C:\Documents and Settings\YOURUSERNAME\Desktop\test.vbs" > "C:\Documents and Settings\YOURUSERNAME\Desktop\output.txt"

Press Enter

You can work on making it look prettier if you like.

As always, there are many ways to achieve this.  I'm far from a VBS or scripting expert, and I'm sure that there are more elegant ways to do things.

Microsoft Windows has the ability to set its time and date by syncing with timekeeping server via internet. To fix the issue with your Windows Clock, please follow the steps below:

  • Press Windows key+i to open Settings, then click Time & language.
  • Under Related settings, click Add clocks for different time zones.
  • Under Date and Time, click Internet Time tab.
  • Under Internet Time tab, click Change settings.
  • Check the Synchronize with an internet time server.
  • Select time.windows.com on server.
  • Click Update now.
  • Once you see "The clock was successfully synchronized," click OK.
