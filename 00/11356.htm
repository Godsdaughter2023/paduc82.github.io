pamcaswell say: So, my understanding is that you create the tables in W2010 and apply a table style to them. I'm guessing that you save the tables in a .doc document.  The software combines the tables (with other information?) in a new .doc document that it creates.  When you open that document, the tables do not have the custom table style applied. 

Table styles were introduced with W2002, so if the software uses any older version, it will essentially discard the table style. (I recall helping someone who had round-tripped tables with custom styles from W2007 to W2000, maybe '97, and back. But I don't recall all the particulars.)

I have round-tripped documents with custom table styles between W2007 and W2003 many times. Most of the time the table styles have made the trip.  But a few times, the style has not made it to W2007.  The correct formatting is there, but the table style applied is Table Normal and the custom table style is no longer in the document.

Yet another possibility is that on pasting (or inserting) the 2010 tables into the new  .doc document, Word is keeping  the source formatting but not the style. 

In any case your only recourse may be a macro that adds the table style to the .doc file and applies it to all the tables.
