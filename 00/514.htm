suzanne s. barnhill say:

Okay, let's start again from the beginning (literally, since I had typed several paragraphs and accidentally wiped them out with a stray keypress). In Word 2003 and earlier you could/can use the Insert Page Numbers dialog to easily insert page numbers with a number of options. You could select Inside or Outside, and the page number location would be mirrored on facing pages. You could opt to show or not show a page number on the first page of a document or section. This was accomplished by putting the page numbers in frames and (I suppose) covertly enabling Different First Page.

 

There is nothing comparable to this in Word 2007 and above. You must achieve all of this manually.

First of all, let's be clear that enabling “Mirror margins” in the “Multiple pages” dropdown on the Margins tab of Page Setup is irrelevant, a red herring. It has no effect on the placement or appearance of page numbers; it affects only the location of margins.

 

In order to have mirrored page numbers, you must first open the header and, on the Header & Footer Tools | Design tab, check the box for “Different Odd and Even.” This gives you two different headers and footers: an Odd Page Header/Footer and an Even Page Header/Footer. You must insert page numbers in both of these.

 

Assuming you have no other content in the header or footer in question, you can insert a page number at the left margin of the Even Page Header using Alt+Shift+P. In the Odd Page Header, press Ctrl+R to right-align the paragraph and use Alt+Shift+P to insert a page number at the right margin.

 

If the header or footer contains other content, you can use the built-in tab stops to place text and page number fields. If you’ve used one of the built-in header/footer building blocks, you cannot insert a page number building block without deleting the existing building block. Instead, you must place the insertion point where you want the page number and either use the keyboard shortcut or use Insert | Page Number | Current Position.

 

Unless the headers and footers in every section are linked, you’ll need to repeat this in every section. Note that the “Different Odd and Even” setting affects the entire document; you can’t enable it for selected sections as you can “Different First Page.”

Dim y As Integer 'index

'set x = today's date

With tempWeekBeginningDate

        x = DateAdd("d", 1, x)
