Hello Emma,

I don't know of any way that you can filter based on defined range names.  I'm pretty sure that there isn't a way.  There may be another way to accomplish what you want by using colors for your sections instead of defined range names.

1.  Select/highlight each of the 5 sections individually and their accompanying series.

2.  Select a different Fill color for each section (the background of the cells will be filled with    that color).

3.  Filter your data.

4.  Then select Filter by Color from the Filter drop down menu.

5.  Select the color that represents the section you would like filtered.

Unfortunately, the Filter by Color menu doesn't tell you which color represents which section, therefore, you may have to add a color key/legend to your spreadsheet.  This is how I would do it.  I've tried it on a mock up and it works well.

john_sur say:

Hi Emma,

Actually there is another way to do what you are looking for and that is with a Pivot Table.  In order to do this with a Pivot table you would have to set up your data little differently.  It would look like:

Sections      Bullets

1.1              one

1.1              two

1.1              three

1.2              one

1.2              Two

1.3              One

I'm assuming that you have worked with Pivot Tables before.  Once you have your data formatted this way, then you would insert a Pivot Table using the data. 

In the Pivot Table Field List, drag the fields (in this case Sections and Bullets - Sections being first) from the Choose fields to add to report: section to the Row Labels section.

Remove the row and column totals from the Pivot Table.

right click on any part of the Pivot Table

select the Totals & Filters Tab

uncheck

Show grand totals for rows

Show grand totals for columns

press OK.

You should now be able to select whatever section(s) you want.

Hope this helps,

Hi Frank,

It seems you enable centralized mail in hybrid previously. So, the mail flow is as below.
Internet -> EOP -> Exchange 2010 Edge server -> EOP -> Cloud mailbox
Internet -> EOP -> Exchange 2010 Edge server -> local mailbox

For your current requirement, we can disable centralized mail. For details, you can check the following article.

technet.microsoft.com/.../jj659050(v=exchg.150).aspx
