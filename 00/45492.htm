classicd say:

finding static IP when media disconnected

So, I have 2 NICs

1 @ DHCP - connected

1 @ static - disconnected

ipconfig /all only shows the dhcp settings on 1 NIC but doesnt' show the static IP address on NIC 2. How do i find the static IP settings on command prompt while the media is disconnected.

Thanks

frederik long say:

You would need to extract the information directly from the registry, e.g. by running this batch file:

@echo off
goto Start
----------------------------------------------------------
Determine the IP address of all installed network adapters,
even those that are disconnected or disabled.
10.9.15 FNL
----------------------------------------------------------
:Start
SetLocal EnableDelayedExpansion

set IP=
set Key=
set Desc=
for /F "delims=" %%a in ('reg query "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\NetworkCards" /s') do call :Sub1 %%a
goto :eof

:Sub1
set Line=%*
set Line=%Line:)=]%
set Line=%Line:(=[%

if /i %Line:~0,11%==ServiceName set Key=%Line:~25%
if /i %Line:~0,11%==Description set Desc=%Line:~25%
if not "%Desc%"=="" (
  for /F "tokens=3" %%b in ('reg query HKLM\SYSTEM\CurrentControlSet\services\%Key%\Parameters\Tcpip /v IPAddress') do set IP=%%b
  echo %Desc%
  echo IP=!IP!
  echo.
  set IP=
  set Key=
  set Desc=
)
