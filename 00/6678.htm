jay freedman say:

You can add, delete, and modify almost all shortcuts by clicking File > Options > Customize Ribbon and then clicking the Customize button in the lower left corner. That opens the Customize Keyboard dialog. To use it that way to remove an unwanted shortcut, though, you need to know or browse for the name of the command that is currently attached to that shortcut. There's a better way.

In the body of a document, press the Ctrl and Alt keys and the + key on the number pad. That changes the cursor to a four-leaf-clover shape. Then press the shortcut combination you want to remove. The Customize Keyboard dialog will open with the current command already selected, and with the shortcut shown in the "Current keys" box. Click the Remove button and click Close.

bob jones aka: cybertaz say:

There is no built-in provision for that type of information because there is no such element as a page in the structure of a Word document. Pagination is imposed for print purposes only & all Word tracks in that regard is how many sheets of paper are required to print the document based on volume/nature/formatting of content & layout specs at the time. Microsoft developed the UI to display documents on metaphorical pages because that's what users wanted to see, but it's for viewing purposes only.

Once the document is finalized you can select passages, & observe the word count indicated in the Status Bar or the Word Count dialog window, then record that count manually. Anything other than that would require some sort of VBA procedure... I've seen some attempts at this sort of macro but nothing effective enough to recommend. Either way, keep in mind that any changes to the flow of the document (margins, font size, character formatting, paragraph formatting, paper size, etc.) - not just adding or deleting text - most likely will change that count.

bernard liengme say:

Here is a VBA solution which finds the closest date not the earliest.

Column E in Sheet2 is not needed

In C2 enter =Biz(A2,B2), in D2 =Vert(A2,B2) and drag both down the columns

I have a function for Biz and Vert but a single functions could be coded. However, it would be an array formula and some like to avid these.

best wishes

Function Biz(myDate, myId)
  Biz = "Not found"
  mymin = 9999
  With Worksheets("Sheet2")
   LastRow = .Cells(Rows.Count, "A").End(xlUp).Row
   For j = 2 To LastRow
    If .Cells(j, "B") = myId Then
      test = Abs(myDate - .Cells(j, "A"))
      If test < mymin Then
        mymin = test
        myindex = j
      End If
     End If
   Next j
   Biz = .Cells(myindex, "C")
  End With
End Function

Function Vert(myDate, myId)
  Vert = "Not found"
  mymin = 9999
  With Worksheets("Sheet2")
   LastRow = .Cells(Rows.Count, "A").End(xlUp).Row
   For j = 2 To LastRow
    If .Cells(j, "B") = myId Then
      test = Abs(myDate - .Cells(j, "A"))
      If test < mymin Then
        mymin = test
        myindex = j
      End If
     End If
   Next j
   Vert = .Cells(myindex, "D")
  End With
End Function

=LOOKUP(2,1/((Sheet2!$A$2:$A$18<=Sheet1!A2)*(Sheet2!$B$2:$B$18=Sheet1!B2)),Sheet2!$C$2:$C$18)
