Insert File vs Embed File - Microsoft Community

itsme849 say:

Insert File vs Embed File

What is the difference between inserting a document using MS Word for Mac 2011 and embedding a document?  I have been using Insert > File, then selecting the file I want and Inserting it into a new document, in order to create one large MS Word document that will then be converted to an Adobe PDF file.  I want to be sure that all the pictures, charts, text etc., will actually be embedded into the new document.  Am I doing this correctly?
bob jones aka: cybertaz say:

The simplest distinction:

  • Using Insert> File replicates the content of the source file as editable content.
  • Using Insert> Object inserts a duplicate of the source document as an embedded Object [either an icon or an image] which cannot be edited. The object can be used to open the embedded version of the file when necessary.

Glad to be of help.

The links are just part of information I've collected over time to answer just this sort of question.

OK, your described use it completely "low end", so unless you identify some other need, don't let anyone try to "upsell" you on a "high power" computer. Pretty much any low end computer you see on sale will do for your described needs.

Technology has advanced to the point that using USB "Flash"/"thumb" drives is essentially the same as using external "Hard Drives". The main difference it the amount of storage available and the size of the device and the relative price by GB.  Take a look at the size of the HD in your computer.  I am willing to bet it is in the range of 64-100 GB, maybe even smaller.  Of that probably 20-30GB is used by windows and a few more 5-10 GB for installed applications. The rest is probably open for your personal files.  Today I bought a 32GB thumb drive for $30. I am willing to bet that if all you want to do is backup your personal files on an external drive this will be enough, more than enough, for you.  If you want to be a "responsible computer owner" and do regular system backups then an external HD is the way to go.  Don't "cheap out" and buy the smallest external drive, typically 500GB (it will be trivially cheaper than the next size up), go for a 1-2 TB drive.  They will typically cost $50-125.

Using a USB drive, of either type, is dead simple. Plug and Play. It just takes seconds to become available.  The only thing you have to do is make sure you use the "safely remove" button in the Task bar before unplugging it. Or only unplugging the USB when computer has been powered off normally.  As long as you do that you have nothing to worry about.

Since you primarily do a lot of reading, you may want to consider a "tablet" instead of a laptop or desktop. You may consider the main disadvantage of tablets is that the monitor is smaller. A consideration if you have vision issues.   If you do, you probably want to buy a larger monitor. 21-27" "HD" (1920x1080) LCD monitors typically cost around $150.  Actually, if you check, you will find that many tablets, and pretty well all laptops do also.  If you get a laptop, you can plug in an external monitor with a larger screen when you are sitting at a desk. That is what I am doing right now.

If you are planning on getting a new computer, you may not have to buy an external video camera to Skype.  All tablets and laptops come with cameras and so do many LCD monitors.
