ossiemac say:

It is possible that the text contains additional non-displayable leading and/or trailing characters. You can test by selecting a cell and on the formula bar use the left and right arrow keys to progress through the entry and see if any additional non-displayable characters.

Sometimes you can remove them by inserting an adjacent columns and use the Clean function and/or Trim function. However, this does not always work and required VBA code to remove the additional characters.

If you can't remove the additional characters, if you want a VBA solution the let me know. However, please post a small sample of the data so that I know what type of characters are valid. eg. alpha, numeric, other characters like hyphen, question mark, plus sign, negative sign etc.

ossiemac say:

Some additional information to remove non-displayable characters.

Sometimes you can just use Find and Replace to remove the characters.

If you have non displayable leading characters then in a blank cell insert the following formula (assuming that C2 is one of the cells with the text containing non displayable characters.)

=CODE(C2)

  • The formula will return the ASCII code of the first character.
  • Select Find & Select (Far right of Home ribbon) and then select Replace.
  • Click in the Find what field.
  • Hold Alt key and on the numeric key pad enter the ASCII number but padded with leading zeros to make 4 characters. eg. a space returns 32 so hold Alt and key in 0032. (You may not see anything in the field but the character will be there)
  • Leave the Replace with field blank
  • Click Replace All

If the non displayable characters are trailing characters only then enter the following formula to get the ASCII code.

=CODE(RIGHT(C2,1))

Most times the non displayable characters are all the same but you might need to repeat the process if still cannot centre the text.
