remove letter=X

Close Command Prompt and restart your PC

ethan b. say:
Hey Colton,

This is a recovery partition brought into view in 1803. You cannot remove, but only you must remove its mapping of its respective drive letter, which is D. Once you do this, it brings it back to its intended visibility, being hidden and out of sight. Once you do this, you won't be pestered with low disk space notifications anymore. You will need to remove the drive letter and mount point in DiskPart in order to do this.

Should you want to go ahead and do this, follow these instructions:

Hit Windows key + R together to go into Run. Type in the following: diskpart

Enter the following in the new DiskPart window:

select volume LETTERHERE
remove letter=LETTERHERE

Does not work after a reboot? Try entering this instead in an administrator session of Command Prompt: mountvol LETTERHERE: /D

Of course, LETTERHERE, you replace with your drive letter.

Source: https://www.reddit.com/r/Windows10/comments/8gj...

Note: This is a non-Microsoft website. The page appears to be providing accurate, safe information. Watch out for ads on the site that may advertise products frequently classified as a PUP (Potentially Unwanted Products). Thoroughly research any product advertised on the site before you decide to download and install it.

    There are no settings within WLMail to change the fonts used for most UI elements like the folder list and button labels. However, reading your Additional notes, it seems that you have to increase the size of the text on these pages (at the Microsoft Community site) for ease of reading.     You haven't told us which operating system you're using. If it's Windows 7, you can use the Display settings to increase the size of text and other UI elements to make them easier to see and read. To do this,

    1. Type display into the start search box and press Enter.

    3. Fine-tune the text size by selecting the Set custom text size option in the left-hand pane.
    4. Re-tune your ClearType settings:

    To alter the way text is displayed on web pages (and also for the display of many mail messages in WLMail), use the accessibility options in Internet Options:

    1. Type internet into the start search box and select Internet Options from the results.
    2. On the General tab, click Fonts.
    3. Select a suitable font for the language scripts concerned. You may find Segoe UI Semibold easier on the eye if you have trouble with the clarity of 'thin' fonts like the Segoe UI used at this site.
    4. Click Accessibility.
    5. Select Ignore font styles… to apply the font you selected at (3) instead of the one prescribed by the web designer.
