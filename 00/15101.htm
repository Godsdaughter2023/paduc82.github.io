joão maroco say:

You can add the MathType Tab to MS Word 2016:

  1. Launch Microsoft Word.
  2. From the File tab choose Options.
  3. In the resulting window, from the left-side navigation menu choose Add-ins.
  4. Using the drop-down menu located at the bottom of the window, choose Word Add-ins and click the button labeled, "Go".
  5. In the resulting window, under Global Templates and add-ins you should add the Mathtype plugin
  6. Navigate to the MathType directory in your Programs folder and select (probably will be here C:\Program Files (x86)\MathType\Office Support\32) (I have office x86, if you have office 64 than choose the 64 directory
  7. Select "MathType Commands 6 For Word 2013", ok and close the add in tab in word.

You should see the mathtype tab.

Enjoy
