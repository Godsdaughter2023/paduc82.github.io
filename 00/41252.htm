A .bin file can be created and opened in any number of different programs (some of which require the same program to work).  Here's an article on opening Bin files with some suggested programs and procedures that will do it. http://www.fileextensionguru.com/bin.  Your best bet is to contact whoever sent you the bin file and ask them what program was used to create the file - then you can get the same or a compatable program to open it.  I would not expect Adobe Reader 9.1 to work to open a .bin file (but maybe if you have some ISO burning software program that will help).  Check Start / Control Panel / Default Programs / Associate a file type or protocol with a program to see what program is the current default for opening a .bin file (it's probably Adobe Reader 9.1 in your case) - this may be the wrong program and may need to be changed (using the Change Program button).

superb tech say:

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer

Here create a new DWORD, MultipleInvokePromptMinimum

 

Set the decimal value to 16 to disable the 15 limit.

 

You can set the value lower if you want...

 

0 disabled the right click options

1-15 sets the max number to 1-15

16 removes the limitation.

 

Source: http://support.microsoft.com/default.aspx/kb/2022295/?p=1
