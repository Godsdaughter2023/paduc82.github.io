An after thought.

The following code will position the worksheet with the same top left cell and it will then select the equivalent cell on the worksheet. Install same method but in lieu of previous code.

Dim wsNew As Worksheet
Dim lngTopRow As Long
Dim lngLeftCol As Long
Dim strActiveCell As String

Private Sub Workbook_SheetActivate(ByVal Sh As Object)
    Application.ScreenUpdating = False
    'Following line sets Top Left Cell same as previous worksheet
    Application.Goto Cells(lngTopRow, lngLeftCol), scroll:=True
    Range(strActiveCell).Select
    Application.ScreenUpdating = True
End Sub

Private Sub Workbook_SheetDeactivate(ByVal Sh As Object)
    Application.EnableEvents = False
    Application.ScreenUpdating = False
    Set wsNew = ActiveSheet 'Save new ActiveSheet
    Sh.Activate     'Roll back to previous sheet
   
    'Save top row and left column
    With ActiveWindow.VisibleRange
        lngTopRow = .Row
        lngLeftCol = .Column
    End With
    strActiveCell = ActiveCell.Address
    wsNew.Activate      'ReActivate new sheet
    Application.EnableEvents = True
End Sub

One reason we suggest that you use the built-in Heading-series styles is that they are hard-coded to make their numbering work.  If you create your own series of styles you have to get all the components correct or you get the type of problems you are experiencing.

Explaining how to do all this is several pages worth, so we tend these days to simply say "Use the built-in Heading styles, they work."

As you have discovered, the issue is that the chapter heading STYLE must be defined into the numbering List Template in order to get the level 2 numbering to "reset on higher".

In your case, it would be SO much simpler to have the Level 1 numbering include the word "Chapter" so that you can include numbering in the level 1 style.

If you do not do that, you will be trying to force Word to work against the way it has been designed.  It can be done, but it's ugly and unstable.

You would need to build field-based numbering using ListNum fields.  Then you would need to use TC tags to tag the headings you wanted in the Table of Contents.

Much, much easier to include numbering in your level 1 Chapter heading style.  It doesn't have to be the built-in style, it simply needs to have an Outline Level of 1, and the style name needs to be defined into the numbering list template using Format>Style>Modify>Format>Numbering>Customise...

Hope this helps
