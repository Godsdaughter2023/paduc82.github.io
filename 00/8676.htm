Sub Compare_Wbks()
Const sWB1 As String = "wbook1.xlsx" ' << wb1 name
Const sWB2 As String = "wbook2.xlsx" ' << wb2 name
Dim wb As Workbook, wb1 As Workbook, wb2 As Workbook
Set wb = ThisWorkbook
Dim rr As Range, r As Range
Dim sPath
sPath = "c:\users\username\desktop\"
Dim L As Long
Application.ScreenUpdating = False
Set wb1 = Workbooks.Open(sPath & sWB1)
Set wb2 = Workbooks.Open(sPath & sWB2)
L = wb1.Sheets(1).Cells(Rows.Count, "A").End(xlUp).Row
wb.Sheets.Add before:=wb.Sheets(1)
wb1.Sheets(1).Cells(1, 1).Resize(L, 2).Copy
wb.Sheets(1).Cells(1, 1).PasteSpecial xlValues
Application.CutCopyMode = False
For Each r In wb.Sheets(1).Range("B:B").SpecialCells(xlCellTypeConstants)
Set rr = wb2.Sheets(1).Range("A:A").Find(What:=r.Value, SearchOrder:=xlByRows, SearchDirection:=xlPrevious)
If Not rr Is Nothing Then
wb.Sheets(1).Cells(r.Row, "C").Value = wb2.Sheets(1).Cells(rr.Row, "B").Value
End If
Next
wb1.Close False
wb2.Close False
Sheets(1).Range("C:C").SpecialCells(xlCellTypeBlanks).EntireRow.Delete

Sheets(1).Range("B:B").EntireColumn.Delete

wb.Save
Application.ScreenUpdating = True
End Sub

Hi all,

The disappearing contacts in Outlook 2013 connected to hotmail.com, live.com and outlook.com is a known issue! It affects everybody with the recommended configuration using Exchange ActiveSync.

The issue exists in all versions of Outlook 2013 and therefore has been around for years.

The issue has been acknowledged by Microsoft in this thread:

What is status on error/bug resulting in duplicates or silently vanishing contact/people items in Outlook 2013 using outlook.com?

Creating a new profile will only temporarily make the issue invisible; the disappearing contacts will start again once you start editing a contact created in the new Outlook 2013 profile.

These are guidelines until the issue is fixed:

  • Do not create new contacts in Outlook 2013, use the outlook.com web interface (people.live.com) or another device instead.

  • Do not follow the guide in the first part of the Microsoft article "Import contacts into an Outlook.com account"

  • In some situations it is possible to restore a disappeared contact from https://people.live.com/#vw=deletedContacts, but the last edit is permanently lost.

  • All the contacts you have already created in Outlook 2013 need to be repaired.

    There are several ways to do this, but the safest concept is:

    • Print all the contacts to paper (or export them to a CSV file)

    • Delete the contacts in Outlook 2013

    • Create the contacts in the web interface (or import the CSV file)

Pease check the above thread for the full and latest details; and please vote "Had this question too" to focus Microsofts effort on fixing the issue.
