The easiest way to delete the footers is to go into each one and remove the page number. jay freedman say:

Each page on which the footer changes is the first page of a new section (i.e., there is a section break between that page and the one that precedes it). On each such page, you must go into the footer and, if the Link to Previous button in the Header & Footer Tools ribbon is not shaded, click it to turn it on.

You can use the Next button in the Header & Footer Tools ribbon to go from one section's footer to the next without closing the footer pane.

It's also possible that you have some sections that begin with a Continuous section break, so there could be two, three, or more sections on a single page. To get the footers to continue through all of the sections, you must link all of their footers even if some don't appear on any page. The Next button won't get you there; you'll have to insert a temporary page break (Ctrl+Enter) to make the section's footer visible, fix it, and then remove the page break.

Protecting templates in Sharepoint - Microsoft Community

lisawuertz say:

Protecting templates in Sharepoint

We have templates available for download on our SharePoint.  However, people sometimes modify the template and save over it instead of downloading it first, then modifying.  Is there a way we can "lock" or "protect" templates so they cannot be saved over and must be downloaded?
ray gao msft say:

Hi Lisa,

 

If the templates are saved as files in a SharePoint library, you can change the users’ permissions to Read to protect them. In this way, users can only view and download the templates.

Here are the detailed steps:

  1. Select one of the template file in the library.
  2. Open the details pane and select Manage access.
  3. Scroll down and click Advanced.
  4. Change the permission of the users or groups to Read.

 

If this is not your situation, please provide more information for us.

 

Thanks,

Ran

The Orange Office tile only has access to the folders that are labeled with the type "Documents" on Skydrive, because the Office products on your phone can only open documents.
