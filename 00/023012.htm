daddyman say:

Using the time-tested hit-and-miss method, I came across a way to create a new style set. It's a kluge, but it works - so far. It's not nearly as intuitive as in Word 2003 and 2002, but it works - so far.

1. Create a simple document that includes all the styles you want to have in your style set. (In my case, I already had such a document.)

2. Open your document in Word 2010/2007. On the Home tab, remove all the Quick Styles from the Quick Style Gallery. As far as I know, you have to remove each Quick Style one by one. (Right-click on each Quick Style and select 'Remove from Quick Style Gallery'.)

3. After the Quick Style Gallery is empty, find the small arrow aiming towards the southeast underneath Change Styles and click on it. (Or press Alt+Ctrl+Shift+S.) The Styles window opens and displays all the styles in your document -- all the styles you want to have in your new style set.

4. For each of the styles in the Styles window, right-click on the style and choose 'Add to Quick Style Gallery'. When done, the Quick Style Gallery (on the Home tab of the ribbon) will contain all the styles you want to have in your new style set.

5. Now go to Styles > Change Styles > Style Set > Save as Quick Style Set. Give your new style set a name (it will be given a .dotx extension) and save it.

6. Close Word and save the changes to the Normal template. Ta da.

Now wasn't that obvious (he asks with a certain sarcasm)?

But you're not done yet. There are gotchas:

1. When next you open Word, don't expect to see your style set in the Quick Style Gallery. Word puts its own style set in the Quick Style Gallery. To see your style set, you need to go to Styles > Change Styles > Select Style and select your style (by clicking on it.) I haven't figured out - yet - how to make my style set the default.

2. When your style set is first displayed in the Gallery, don't be surprised if you see some additional styles that were put in there by Word. In my case, for example, Word was very sure that my style set needed to have a Heading 4 style, and several other styles, even though I didn't create them. I couldn't see a way to remove Heading 4 from my style set (and by that time I was tired of fighting with Word) so I did a right-click on Heading 4 and selected 'Remove from Quick Style Gallery'. To make this stick, you have to 'Save as Quick Style Set' again and save over your original .dotx.

Heading 4 (and the other unwanted styles) are still in my style set - I haven't yet figured out how to finally delete them - but they are not Quick Styles, so they don't appear in the Quick Styles Gallery and I can ignore them.

Edit #1: I found the solution to Gotcha #1 on Microsoft's (Word 2007) website. To make your chosen style set appear in the Quick Style Gallery every time you open Word: Display your chosen style set in the Quick Style Gallery and then  simply go to Styles > Change Styles > Set as Default.

Edit #2: To remove extra styles you may find added to your style set: Open the Styles window (click on the southwest-pointing arrow underneath Change Styles or press CTRL+ALT+SHIFT+S) and click on the blue Options link in the lower right corner of the Styles window. Change the current view ("Select styles to show") from Recommended to In Current Document. This will remove Word's 'recommended' styles -- most of them, anyway. I'm still stuck with Heading 4.

For example, you cite having a status for each piece of equipment. If so, what you should have is a lookup table for status like:

tluStatus
StatusID (PK Autonumber)
Status

This would also mean that your main table would have a StatusID foreign key to link back to tluStatus.

You don't indicate whether the form is a single form view or multiple records. If its a single form view, you might want a report by status instead of filtering the form.

You can do with a line of code like:
DoCmd.OpenReport "reportname", acViewPreview,,"StatusID = " & Me.cboSearchStatus
