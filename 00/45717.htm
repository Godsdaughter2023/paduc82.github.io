3. If you're having issues downloading or installing an app, try updating Microsoft Store: On the taskbar, select the Microsoft Store  icon. If repairing the app doesn't fix the problem, select Reset.  If the app still isn't working, try uninstalling and reinstalling the app to see if that fixes it.

In Microsoft Store, select See more  > My Library > Apps. Select the app you want to reinstall, and then select Install. If the app still isn't working, try running the troubleshooter for Microsoft Store apps.

    andre da costa say:
    Press Windows key + R
    Type: control userpasswords2
    Hit Enter
    Select your account

    Click Properties
    Click the Group membership tab
    Select Administrator

    Click Apply then OK
    Click Apply then OK again

    OR

    Press Windows key + R
    Type: control
    Hit Enter

    Open User Accounts
    Click Change account type
    Select Administrator
    Click Change account type

    If that does not work...

    Open Start
    Type: cmd
    Right click CMD
    Click Run as administrator

    At the command prompt, type:
    net user administrator /active:yes

    Hit Enter

    Close command prompt, restart, then try signing into the Administrator account.

    Open Start > Settings > Accounts > Family & other people
    Select your account
    Click Change account type
    Choose Administrator
    Click OK
