    System File Checker will check Windows system files for corruption and try to fix any found.

    bill.p. say: Hi Snugg, I'm Graeme, an independent adviser here to help.

    It's possible there's a script running at Startup from the old Twitch installation, so try this:

    - Right-click the taskbar and choose Task Manager, or press Control+Shift+Escape.
    - Click More Details
    - Under the Startup tab, see if there's any Twitch related startup tasks.
    - Disable it if it's there, then reboot and see if the problem is still there.

    Otherwise, see if Twitch has installed a service that you can disable:

    - Click Start and search for "services.msc"
    - Select the first result, and look in the Services list for anything related to Twitch.
    - Try right-clicking and disabling any Twitch services you find, then reboot and see if the problem persists.

    Hopefully that helps! Please let me know if you need further support. Thanks!

    Boot to the Windows 10 desktop

    Click Device Manager
