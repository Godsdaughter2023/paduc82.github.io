if you put it in an AutoOpen macro, either in the document itself, which would then need to be saved in macro enabled format (*.docm) or in the normal.dotm template or a macro enabled template saved in the STARTUP folder.

Just like NeoBoot, I don't believe that adding another environment increases security, it actually reduces it.  The more complexity added to a system the more potential 'holes' will exist to exploit, not less.

The area you are discussing is specifically browser security, not malware protection which is what MSE does.  If this were added to Internet Explorer 8 it would have some of the same aspects as many of the methods Microsoft is already using, including the link I posted above.  The methods aren't exactly the same, but in general Internet Explorer takes less extreme steps which cause it to 'break' less existing functionality while providing improved protection.

If people choose not to install the current version of the Internet Explorer browser they are losing both functionality and security that no anti-malware product can resolve.  It's far more important to upgrade the browser than install anti-malware, since that's one of the largest entry points into the system in the first place.  If they choose to use a third-party browser then they also choose the additional risks that creates, since vulnerabilities are additive.  In other words, adding Firefox doesn't increase security, it reduces it, since your PC now contains all of the current vulnerabilites in both Internet Explorer and Firefox.

Connecting to car-radio - Microsoft Community

guevara550 say:

Connecting to car-radio

Does anyone know of any product that connects a Nokia N81 up to a car-radio? You know, like the cassette tape things you can get for the iPod? Can't believe there isn't something that does it!

Cheers

Tom

el_loco---01 say: There are not one but many ways:

1) Use a cassette adapter that has 3.55mm connector (there are many brands and very cheap)

2) Get a Car stereo that support A2DP and connect via Bluetooth (of course then you need to change the car stereo but hey this is a good excuse to do it). This is the most comfortable solution as the phone will connect automatically to it when you start the car.

3) Get an FM Transmitter with 3.5mm connector and use the FM radio (I've used the Belkin which I didn't like and also the Kingston which I like you just need to keep the volume down in the phone so that it doesn't distort, Nokia also has an FM transmitter)

4) Get a car Kit or Hands free that connects to a car stereo.

5) Get a car stereo that has Line-in inputs (RCA or 3.5mm). This is by far the best audio quality solution!, and what you should do if you really want to enjoy good sound.

6) I'm not sure but are there uPNP enabled car stereos?
