considerate_guy say:

First, it sounds like you might be on an older version of Windows Movie Maker.

Are you perhaps using one of the following?
Windows Movie Maker 2.1 (from XP)

Windows Movie Maker 2.6 (from Vista)

Windows Movie Maker 6.0 (from Vista)

The latest version is Movie Maker 2012 which is part of Windows Essentials.  It doesn't have a collection area, and you add photos on the right in the project/storyboard area.

In any case, I think I know the problem.  This applies to all versions of Movie Maker.  Movie Maker does not import the actual video or photos into your project.  All it does is remember where those files are on your computer.  If you "import" photos from a removable disc, then it may be that disc is no longer present or is in a different drive.  In that case, Movie Maker can't find the files and shows an error icon to let you know it can't find them.

It is also possible that Movie Maker is trying to read the photos off the disc but, because discs are slower to read than a hard disk, it is showing the same error icon.

The solution, in both cases, is to copy the photos off of the photo disc to a folder on your computer.  Then when you import those photos, be sure to import the copies from your hard drive.  In that way, they will read quickly and be available while you are editing.
