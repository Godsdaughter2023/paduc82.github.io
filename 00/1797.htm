Formula in C6: =SUMIF($B$2:$B$4,$B6,$C$2:$H$4)

Is the SUMIF function only good for summing individual columns (Column C in this case) and not an actual Range (C through H) as it says it will in the formula help window?

hansv mvp say:

The sum range must have the same size as the condition range. Since your condition range is a single column, the sum range must be a single column too.

You can use a SUMPRODUCT formula instead:

=SUMPRODUCT(($B$2:$B$4=$B6)*$C$2:$H$4)

Alternatively, create a helper column with the sum of each row. You can then use SUMIF with that column as sum range.

zorvek say:

Summing multiple columns based on the condition in one column is a little tricky. One technique is to use SUMPRODUCT:

=SUMPRODUCT(($B$2:$B$4=$B6)*($C$2:$H$4))
