Comments- how to alphabetize them - Microsoft Community

john illinois say:

Comments- how to alphabetize them

Comments have been inserted in several cells. In this case the comments are several names. How can those names be alphabetized or sorted from A-Z?

John

 

bill manville say:

You could use a macro like this:

Sub SortCommentLines()
  Dim C As Comment
  Dim vNames
  Dim iName As Integer
  Dim jName As Integer
  Dim stName As String
  For Each C In ActiveSheet.Comments
    vNames = Split(C.Text, delimiter:=Chr(10))  ' assumes the names appear on separate lines
    ' sort
    For iName = LBound(vNames) To UBound(vNames)
      For jName = iName + 1 To UBound(vNames)
        If vNames(jName) < vNames(iName) Then
          stName = vNames(iName)
          vNames(iName) = vNames(jName)
          vNames(jName) = stName
        End If
      Next
    Next
    ' replace the comment.
    C.Shape.TextFrame.Characters.Text = Join(vNames, Chr(10))
  Next
   
End Sub
