Hi Anizzy,

Welcome to Microsoft Community. Your interest in Windows 10 is much appreciated. 

I would suggest you to follow these methods and check.

Method 1: 

Run Network Adapter Trouble shooter

Follow these steps to run Network Adapter Trouble shooter.

  1. Open Control Panel.

  2. Type troubleshooting in the search box and then press enter.

  3. Click Network and Internet and run Network Adapter Troubleshooter.

Method 2:

I would like you to go through the links to find out the cause behind the issue.

Wired and wireless network problems

http://windows.microsoft.com/en-US/windows/network-connection-problem-help#network-problems=windows-8&v1h=win8tab2&v2h=win7tab1&v3h=winvistatab1&v4h=winxptab1

How do I fix network adapter problems?

http://windows.microsoft.com/en-US/windows-8/fix-network-adapter-problems

Method 3:

Uninstall and Reinstall the Network Adapter in previous Operating System Compatibility Mode

Uninstall Network adapter

Follow these steps to uninstall Network Adapter.

  1. Click on Start and select Device Manager.

  2. Search for Network Adapters and expand it.

  3. Right click on the network adapter and select Uninstall.

Reinstall Network adapter in Previous Operating System Compatibility Mode

Follow the steps to install in Compatibility mode.

  1. Download the updated driver from the manufacturerâ€™s website.

  2. Right click the driver installation file and select Properties then tap or click the Compatibility tab.

  3. Place a check in the Run this program in compatibility mode for: box.

  4. Tap or click the drop down box and select a previous operating system, then tap or click OK.

  5. Try installing the driver and check.
