As Bruce said, it will not restore a completely deleted file.  System Restore keeps a list of important operating system files and makes backups of any ones that change so you can restore the system to working condition if a file is damaged.  It is not a backup system for user files -- if you want to have a backup, you need to turn on the backup function yourself (ahead of time).

Got it, DKukski. It appears that your laptop supports wireless display that's why the Galaxy S8 is trying to connect to it. Know that when a wireless display-compatible device tries to connect, it will detect all nearby wireless display devices, and it gets to pick or select which one to connect. In this instance, somebody tried to project to your laptop, which you can ignore.

If you're using a Microsoft Wireless Display Adapter, you can change the pairing mode to Require a Pin code to pair. Here's how to enable it:
