Word Format master template???

How do I create a master document format in Word? The document when you open and create a document the font size is set at 11 and the spacing 1.15 and paragraph spacing...  I want to know how to change these settings so that each time I try to creat a new document the font will be 12, line spacing will be 1, etc.  Please help!  Thanks...
jay freedman say:
You need to change the defaults in the Normal.dotm template, which is the base for new blank documents unless you deliberately choose a different template.

  • Press Ctrl+Alt+Shift+S to open the Styles pane on the right.
  • Click the Manage Styles button, which is the third of the three similar buttons at the bottom of the pane.
  • Click the Set Defaults tab in the Manage Styles dialog.
  • With the "+Body" font selected, choose size 12 and set the line spacing dropdown to Single. If you don't want any additional space between paragraphs, set the Space Before and Space After boxes to 0 pt.
  • Click the option button for "New documents based on this template".
  • If you want, make similar changes for the "+Headings" font.
  • Click OK.
  • On the Home tab of the ribbon, right-click the Normal style and choose Modify. If the definitions in the style dialog are different from the settings you made in the Manage Styles dialog, change the style to match the defaults. Again, choose the option button for "New documents based on this template".
  • When you exit Word, if you're prompted to save changes to the Normal template, say Yes.

In a sort by "Date",  Order of "A to Z" is not an option in Excel 2016 when the data is a Date.    "A to Z" is only available when Excel sees that the date is Text!

Note that as text:

12.1.2016

Will come before

6.1.2016

Because the "Text" is sorted on the first digit.

I would also look at the time values of 00:55 formatted as general.

You may have a value that is greater than 1 (crosses midnight) when you may be expecting values 0-1.
