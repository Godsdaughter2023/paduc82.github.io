Hello BRIANBOOT1,

Try this:

First of all, when you have been faced with Error 1962: No Operating System Found, press and hold the key combination of Ctrl + Alt + Delete. This will reboot your system.
In order to enter the BIOS setup, press the F12 several times in a row.
A little box will appear on your screen and you will see the word Setup written in it. Hit Enter.
At the top of the screen select the Startup tab.
After this, go ahead and select CSM and hit Now select the Enabled option.

Once you’ve finished with the last step, your computer will reboot and load as usual, hopefully without displaying the above error message.

    azeez n say: Hi,

    1. Do you get any error code along with a message?

    Check if the device is getting recognized in disk management.

    To locate Disk management: Click "Start", type in "diskmgmt.msc".

    In the meantime, follow these steps and check if the issue persists.


    Unplug the external Hard Disk Drive
     
    a. Click "Start", type "cmd" in the "Start Search" option.
    b. At the command prompt, type the following lines, pressing ENTER after each line

    set devmgr_show_nonpresent_devices=1
    cd\ %SystemRoot% \System32
    start devmgmt.msc

    c. "Device manager" will open
    d. Click "show hidden devices" under View...
    e. Expand all the nods under device manager
    g. There would be many greyed out entries in device manager uninstall all the greyed out entries under all the nods and then restart the computer and plug the Hard Disk Drive to the computer and check if it works.
     
    Step 2: This issue might be due to incorrectly-configured root hub.
     
    USB controllers require that an IRQ be assigned. The IRQ line is assigned in the computer's BIOS, and usually IRQ 9 is assigned.
     
    Remove and reinstall all USB controllers
     
    To remove and reinstall all USB controllers, follow these steps:
     
    a. Click "Start", click "Run", type "sysdm.cpl" in the Open box, and then click OK.
    b. Click the "Hardware tab".
    c. Click the "Device Manager" button.
    d. Expand "Universal Serial Bus controllers".
    e. Right-click every device under the "Universal Serial Bus controllers" node, and then click Uninstall to remove them one at a time.
    f. Restart the computer, and then reinstall the "USB controllers".
    g. Plug in the removable USB storage device, and then test to make sure that the issue is resolved.
