You probably need to create a new default Outlook Profile, you do that via the mail applet in the Control Panel (32bit)

You manually add the mail accounts (so you will need the mail settings) you can point the accounts to the original data file

 

The type of mail account, pop or imap depends on your mail provider, eg Gmail can be configured as pop or imap

An imap mail account uses its own folder set, whereas a pop mail account uses the OL Personal Folders Inbox/Sent etc

 

Info on OL can be found here http://www.howto-outlook.com/faq/aboutpst.htm

  • Space after
