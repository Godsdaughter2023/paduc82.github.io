dave-marshall say:

Method 1 doesn't work, and step G in method 2 fails. When I try to delete the files in the Catalog folder, nothing happens. If I try to move them to another folder, they vanish for a second, but refreshing the folder brings them back. How do I get rid of them permanently?

You can't delete the catalog files from an explorer window. Start a command prompt as an administrator. Make sure the backup process is really stopped;

(assuming you have already followed the instructions in this thread and entered "net stop srdsvc" at an elevated prompt)

goto services (start->rt click on computer->manage->services and applications->services) and disable the Windows Backup service (rt click on Windows Backup and then click on properties and then change the startup type to disabled).

In the command prompt goto C:\System Volume Information\Windows Backup\Catalogs and delete all the catalog files with the del command. BTW, if you have created image backups you'll need to do this for  C:\System Volume Information\WindowsImageBackup  too.

Then goto services, re-enable Windows backup and restart it (you shouldn't have to enter "net start srdsvc" at this point, but go ahead anyway too make sure it is actually running).

The above worked for me, and I can run backups again.

Guide On Laser Burn

GUIDE ON LASER BURN

Laser Burn is when the laser from your Xbox 360 Console, touches your game disc and leaves a perfect circle scratch on your disc.

Laser Burn is can be caused when one of the following occurs:

  • You bump/move your Xbox 360 Console while the game is playing
  • Playing the game a lot (a long long time)

Unfortunately Laser Burn is very unlikely to be fixed. But it can be prevented.

How to prevent Laser Burn:

  • Keep your Xbox 360 Console in a place where it is not likely to fall over, or tilt.
  • Look after your games, treat them well.

Hi,

The message (This device is currently in use. it by clicking on the Safely Remove Hardware icon in the notification area.

The error can also be caused if your anti-virus is actively monitoring/scanning the USB drive. Normally, when removing a flash drive from a USB port,
Windows should stop all the read-write operations on the drive so that it can be safely removed. If not, the flash drive can be damaged or data may
be lost or corrupted. However, if Windows determines a file is in use, then it may provide a message which indicates the volume cannot be stopped right now.
So, before ejecting or disconnecting any mass storage device, make sure none of the above activities are performed on the mass storage device before ejecting.

Step1:

NOTE: Antivirus software can help protect your computer against viruses and other security threats. In most cases, you should not disable your
antivirus software. If you do have to temporarily disable it to install other software, you should re-enable it as soon as you're done. If you are
connected to the Internet or a network during the time your antivirus software is disabled, your computer is vulnerable to attacks.

Step 2:

6. Click OK.
7. When you are prompted, click Restart.

problem” to determine which program or service maybe causing the issue.
