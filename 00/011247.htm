Hello, Richard Sebire

Skype Customer Service has not had the tools to close accounts for several years as we now manage the account closure process ourselves, be it a Skype account or a Microsoft account. Of note, all Skype accounts were transitioned to be basic Microsoft accounts in September/October 2016. In order to close a basic Microsoft account, that account will need to be updated to a full Microsoft account. 

(Please, let us not dive into a discussion of the merits of this process - my goal here is to guide you and others following in your footsteps to navigate the process.)

1. The good news is the Microsoft Account Team has recently provided a procedure to re-link a Skype account from one Microsoft account to another. Please do be careful about this, though, as the process can be done account, and proceed to close that account.:

Can I move my Skype account to a different Microsoft account if it's already been linked?

2. If you have not already updated your basic Microsoft account, a.k.a. your legacy Skype account, create a new Microsoft account with its own new email address that you can use as a throw-away account. This new account will be used only to link the Skype account that you intend to close. Once this is done, then proceed to close that new Microsoft account with the old Skype account linked with it.

Updating a Skype account to a Microsoft Account: A single sign-in for Skype and your other Microsoft services

j.e. mcgimpsey say: This has been reported several times before, and there have been a number of suggestions. Unfortunately few people have posted back to report on their success. Among them:

1) Make sure you've updated to the latest version (14.1.4, currently).
 2) Try typing CMD-A. It worked for at least one user.
 3) Reset the cursor using VBA: 
  • type Opt-F11 to enter the Visual Basic Editor
  • Choose View/Immediate Window
  • In the Immediate Window type or paste "Application.Cursor = xlDefault" (without quotes, and followed by Return)
  • Type Opt-F11 to return to XL
  • Close XL to save Preferences
4) With all Office apps closed, delete the com.microsoft.Excel.plist file from the ~/Library/Preferences folder (where ~ is your home/user folder). Note that in Lion the Library folder is hidden by default. See http://www.macworld.com/article/1161156/view_library_folder_in_lion.html for ways to unhide it.

Please post back if any of these work for you.
