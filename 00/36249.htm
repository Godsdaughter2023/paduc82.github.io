davem121 say:
  1. Backup all your files, as the following process will wipe your hard drive
  2. Take a note of your Windows 7 product Key - it should be on a sticker attached to your System
  3. Click HERE to download the Windows 7 ISO (make sure to download the correct version, Home, Pro . . .etc)
  4. On that page you will need to enter your Windows 7 product key
  5. Click HERE to download the Windows USB/DVD Tool
  6. Install that tool, with that you can create Windows 7 installation Media on a USB Flash Drive or DVD from the ISO file
  7. Insert the installation Media, turn off your system and turn it back on
  8. Boot your system from the installation Media (you may need to change the boot order in your BIOS
  9. When this window appears, select bottom option (Custom: Install Windows Only)

    10. When this Window appears, select each partition, click disc options and delete all partitions,

          until one Unallocated Partition remains. Select that partition and install onto that . . .

    NOTE: during the installation process, your system will reboot, at that point if you did change the boot order in your BIOS, you may need to go into your BIOS and change the boot order so your system boots from your Hard Drive, otherwise the installation may start all over again - do not remove the installation media at this point
