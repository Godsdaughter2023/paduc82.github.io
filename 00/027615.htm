andreas killer say:
I would read the path & filename in column A in a sheet, write/create the new filename in column B and use a macro to (copy and) rename the files.

Here is a class module how to search files with VBA, an example is included in the comment.
https://dl.dropboxusercontent.com/u/35239054/FileSearch.cls

The rename macro is very simple then:

Sub Test()
  Dim R As Range
  Dim fso As Object 'Scripting.FileSystemObject
  Set fso = CreateObject("Scripting.FileSystemObject")
  For Each R In Range("A1", Range("A" & Rows.Count).End(xlUp))
    fso.CopyFile R, Range("B" & R.Row)
  Next
End Sub
