shawn robert robinson say:

Backing Up My Current Windows 10 system image and data to install on a new hard drive

I am getting a new hard drive on my other laptop. a. user say:

New retail hard drives come with imaging software, and hard drive manufacturers are well experienced in helping users move their stuff from the old to the new hard drive. Fear not. Hard drive manufacturers have copious details about the procedure on their websites and their technical staff are good at explaining. Once you've gone through this you may wonder: why bother with Windows System Image?

shawn robert robinson say: Ok because the new hard drive is going to be in a HP laptop and the image backup that I want to create is on an acer laptop plus I just want to restore the Wndows 10 install from an Acer laptop

a. user say:

That's a different story. If you're moving an image from an Acer laptop to an HP laptop you're going to run into a problem: the Acer image has the Acer's hardware and you want to restore it on HP hardware. Windows may not boot at all, and if it does boot you may crash due to the hardware mismatch.

To restore an image to different hardware you'll need to buy a third-party image program that can do a hardware-independent restore. This is done by substituting generic drivers which you then update after the restore with the proper drivers. That won't be easy because laptops incorporate proprietary hardware for which generic drivers may not work properly (or at all.)

An image is a copy of a partition. You can't restore only a portion of an image, i.e., only Windows.

If you have a corrupted Windows 7 system, you are trying to recover, you want to try a System Repair disk to boot the PC.  It may be more successful in fixing and offer other alternatives.  If you do not have one, go to any other PC that has windows 7 with the same bitness (32/64).  Start globe, All Programs, Maintenance.  You need just a blank CD.  When you boot, ensure that you are in fact booting on that CD and not the hard drive.  Usually pressing F12 will lead you to the option to do so.

This is a common problem for users of W10 CU.  Try using the "magic bullet"  Win+Ctrl+Shift+B.   You may hear a sound when it works.  You may need to do it more than once.   Also, even though you see HDMI No Signal on your monitor we can change that by picking a different display configuration using Win+P.  For example, in my case, after applying the magic bullet I get put back to PC Screen Only and then need to use  Win+P, End, Enter to switch to Second Screen Only.  An additional problem that I have in this scenario is that there is a ridiculously short timed Keep this...? prompt that I need to reply to or it will switch back to the original display configuration.   I do not have enough time to get to that Yes button using my mouse.  So, I just press Tab, Enter (appropriately timed) as if I had been able to reply to the prompt.  (I have no idea where keyboard focus is now; it used to be on the No button so my previous workaround was just CursorLeft, Enter; now fortunately the Tab finds the Yes button from wherever it is.)

I imagine that others could have a monitor which would be very slow at rendering from a No Signal state who might have to take the same action.  In that case, note that you would still have both the magic bullet and Win+P to play with to try again; however, do note that what you would do with Win+P would have to be different than what was already showing.  For example, if you were already in the Second Screen Only state using Win+P, End, Enter would not cause a change.

    tim de baets say:

    To get rid of the M4A files in WMP's library, you will need to reset the library. First note that resetting the library will make you lose all current play counts, and possibly ratings too. Then, follow these steps:

    - Reboot the PC.
    - Go to Start, type  services.msc in the Search field, and press Enter. In Services, find the Windows Media Player Network Sharing Service . If its status is Started , double-click the service and click  Stop .
    - Go to Start, type  %LOCALAPPDATA%\Microsoft in the Search field, and press Enter. This folder will open up in Windows Explorer.
    - In this folder, rename the  Media Player folder to something like Media Player old . If you can't see the folder, then it's probably hidden. Set Explorer to show hidden files and folders, and to show protected system files.
    - Start WMP, your library will have been reset and will be rebuilt.

    afzal taher say:

    Hi,

    You cannot make Windows Media Player exclude .m4a files. You can move your iTunes Media folder to a different location. The default
    location is     \username\My Music\iTunes but it can be moved to a preferred location. When you delete a file from Media Player library, it will
    get deleted from the iTunes library as well because the location of the Media folder is same for both the applications. Moving the media folder
    will solve the problem. You will not have to re associate the songs with the new location. iTunes will retain all the    playlists, ratings, and play history
    even if you move the media folder to a different hard drive.

    Refer this link for help:
