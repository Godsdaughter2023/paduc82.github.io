For example, if a given record has three duplicates, I wish to remove the duplicates, leaving one unique record in the table.

ken sheridan say: You can do it in a single step provided that the table has a suitable column as a candidate key.  If not just add an autonumber column.  The following delete query, for example, would delete all rows bar one from each subset of rows which are duplicated on the FirstName and LastName columns:

DELETE *
FROM Contacts AS C1
WHERE ContactID <>
     (SELECT MAX(ContactID)
      FROM Contacts AS C2
      WHERE C2.FirstName = C1.FirstName
      AND C2.LastName = C1.LastName);

Another very easy way, again using the above as an example, which does not involve adding a key column, would be to create a copy of the table structure, and in the empty new table set the FirstName and LastName columns as the composite primary key.  Then append all rows from the original table to the new table.  All  rows bar one from each subset of duplicate rows will be rejected as a result of the key violations.

Do you have a lot of data in the worksheet coupled with a lot of formulas because if so, it is possibly performing all calculations between each change that is made with the code.

If above is correct then try setting calculation to manual at the start of the code and back to automatic at the end of the code as per the following example.

Private Sub CommandButton1_Click()
    Dim Lr As Long
    Dim mon As Long
    Application.Calculation = xlCalculationManual
    mon = Month(Date)
    Lr = Range("A" & Rows.Count).End(xlUp).Row
    Rows(Lr + 1).Insert Shift:=xlDown

    Cells(Lr + 1, "E") = Day(Date)
    Application.CutCopyMode = False
    Application.Calculation = xlCalculationAutomatic
End Sub

Ensure that you include the square brackets around the workbook name in the Find what field.

Dictate in Word - Microsoft Community

ralphpenney say:

Dictate in Word

How do I activate the dictate feature in Word?
tisky zheng msft say:

Hi Ralph,

 

According to this article Dictate with your voice in Office, the feature is only available to Office 365 subscribers. If you have Office 365 plan, please first make sure your microphone work, then you can find Dictate under Home tab.

 

1. Wait for the icon to turn red.

2. Start talking. As you talk, text appears in your document or slide.

3. Speak clearly and conversationally. Insert punctuation by saying the name of the punctuation mark you want to add.

4. If you make a mistake while dictating, move your cursor to the mistake and fix it with your keyboard. No need to turn off the microphone.

5. When finished, select Dictate again to stop typing.
