create a league table in access - Microsoft Community

danielwood1985 say:

create a league table in access

I have an access database with 1000+ records in it. I want to create something similar to a "league table" based on one of the fields. The table is a list of suggestions by colleagues. I want the table to work out a total number of suggestions raised by each person and then sort it by highest first.

 

Initially the table will just need two rows, "suggestion initiated by" and the total.

hansv mvp say:

Let's say the table has a field named Employee and a field named Suggestion.

Activate the Create tab of the ribbon.

Click Query Design.

Add your table, then click Close in the Show Table dialog.

Add the Employee and Suggestion fields to the query grid.

Click the Totals button on the Design tab of the ribbon.

Leave the Total option for Employee set to Group By, and set that for Suggestion to Count.

Set the Sort option for Suggestion to Descending.

 

The SQL for the query looks more or less like this:

 

SELECT Employee, Count(Suggestion) AS CountOfSuggestion
FROM MyTable
GROUP BY Employee
ORDER BY Count(Suggestion) DESC
