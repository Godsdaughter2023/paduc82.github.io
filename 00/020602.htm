Sub ReplaceAE()
    Dim rngFinds As Range
    Dim rngCel As Range
   
    With Worksheets("Info")
        Set rngFinds = .Range(.Cells(2, "J"), .Cells(.Rows.Count, "J").End(xlUp))
    End With
   
    For Each rngCel In rngFinds
        With Sheets("BrokerData").Columns("E:F")

                   Replacement:=Range("Info!N2").Value, _
                   LookAt:=xlWhole, SearchOrder:=xlByRows, _
                   MatchCase:=False, _
                   SearchFormat:=False, _
                   ReplaceFormat:=False
        End With

End Sub

george222(2) say:

Is this the exact same Definition update its installing every time, or is the definition number increasing with each installation?

The Definition update should increase, it just means you are getting the latest version installed.

For example:

Right click <start>

click control panel

CP search for 'view reliability history'

If you look day by day, you should see that very often one or more versions of the update are successfully applied. If this is happening, then all is well, it just means that your Defender is correctly updating itself.

For example:

on 04/12/2016, KB2267602 (Definition 1.233.1349.0) was applied to my PC.

on 03/12/2016 both KB2267602 ( Definition 1.233.1257.0) AND later KB2267602 (Definition 1.233.1263.0) were applied.

And back it goes. 1.233.1027.0 before that, 1.233.963.0 before that .......

As long as the Definition number is increasing and its successfully being applied, then all is well.

IF its not automatically being successfully applied, i.e. the update is failing or the number is not apparently increasing, then manually download the latest definition for your PC ( W10, Defender, 32 or 64 bit etc ) from here - https://www.microsoft.com/security/portal/definitions/adl.aspx  and manually install it. Then keep and eye on the future few days to see if its starts to correctly automatically install them again.

You can also try forcing a manual update by opening Defender, click the [update] tab, and click the [update definitions] button.
