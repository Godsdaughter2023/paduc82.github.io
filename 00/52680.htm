Hi Lance,

 

This issue could be caused due to some incorrect settings or due to some corrupt system files. I would suggest you to try these steps and check:

 

Method 1: Security scan

 

If you notice performance related issues or if programs takes time to load, there are chances that some security threats got installed in the computer without your knowledge. To make sure that the issue is not caused due to any security threats like virus or malware, I would suggest you to perform a complete security scan. Make sure that the antivirus software is updated with the recent virus signatures before you run a security scan.

 

You can also download and run the Microsoft Security Scanner from here:

 

http://safety.live.com

 

Note: The Microsoft Safety Scanner expires 10 days after being downloaded. The Microsoft Safety Scanner is not a replacement for using an antivirus software program that provides ongoing protection. Any data files that are infected may only be cleaned by deleting the file entirely, which means there is a potential for data loss.

 

Method 2: SFC scan

 

First, I would suggest you to run the System File check scan and check if it resolves the issue. System File checker (SFC) scan is done to check if there are any corrupted system files that could be causing this issue. Refer to these steps to perform a SFC scan:

  1. Press Windows key + X, select Command prompt (Admin) to bring up elevated command prompt
  2. In the command prompt type the following command: sfc/scannow and press enter
  3. Now, restart the computer.

 

Method 3: Clean boot

 

If the issue still persists, I would suggest you to boot the computer in a clean boot state and check. Booting the computer in a clean boot state can reduce software conflicts and help identify the root cause of the issue. You can refer to this article for more information:

 

http://support.microsoft.com/kb/929135

 

Note: After troubleshooting, reset the computer to start as usual. Refer to the step How to set the computer to start as usual after clean boot troubleshooting under more information.

 

For reference, refer to the link to improve computer’s performance:

 

“Ways to improve your PC's performance”
