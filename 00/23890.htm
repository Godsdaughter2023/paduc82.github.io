Nokia 5800 - How to use RealPlayer to stream MP3 radio stations - Microsoft Community

nightowl2 say:

Nokia 5800 - How to use RealPlayer to stream MP3 radio stations

Here is how to use RealPlayer to stream MP3 radio stations:

1) Start RealPlayer

2) Select "Stream links"

3) In "Options" menu, select "New Link"

4) Select "E: Memory card" (You can use either "C: Phone memory" or "E:Memory card", but it is a good idea to conserve the phone's memory)

5) Enter the station name in "Name:" and the http link in "Web address:"

6) If you find the streaming MP3 radio station that you just entered doesn't show up in the "Stream links" list, you will need to exit and then restart RealPlayer.  You will then find the station you entered listed in "Stream links".

Here are a few example of MP3 stations that I like to listen to:

Name: Groove Salad
Web address: http://scfire-mtc-aa06.stream.aol.com:80/stream/1018
This station plays downtempo instrumental music.

Name: Lolliradio
Web address: http://94.76.208.18:8000
This station plays English and Italian Pop and Club music.

Name: Pulse Radio
Web address: http://87.98.137.234:80
This station plays Trance music.

Name: MusicOne
Web address: http://m1live.returning.net/m1live
This station plays Pop and Club music

You can find the http links for MP3 streaming station by going to www.shoutcast.com, click a station's "TUNE IN!".  Then click "Listen in Winamp or Other Player".  Select "Play in your Default Media Player". Then click "Save".  If you are using Winamp, bring up the playlist.  Click "LIST OPTS" and select "SAVE LIST".  

If you selected save the file to the desktop, open the playlist file in notepad and you will see the "http://" link(s).

Enjoy!
