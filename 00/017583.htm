I've also pasted the script below and modified the "oEmailItem.Subject" line to remove the "Emailing: " prefix.  You could change it to whatever you prefer by adding text between the double quotes.
Other benefits are it will attach multiple files, is sent as HTML not plain text and includes your default e-mail signature.
You could customize further by tweaking the code, e.g. add default recipients.
Works great! (I'm running Outlook 2010 on Win7 Pro)

'Mail Recipient.vbs
'Place shortcut in “Send to” folder (i.e. for Windows 7 C:\Users\Nicu\AppData\Roaming\Microsoft\Windows\SendTo) then modify shortcut icon
'http://answers-tips.blogspot.com/2010/05/windows-explorer-send-to-mail-recipient_21.html
Option Explicit
Dim objArgs, OutApp, oNameSpace, oInbox, oEmailItem, olMailItem
Dim a, oAttachments, subjectStr, olFormatHTML
olMailItem = 0
olFormatHTML = 2
'get paths of selected files
Set objArgs = WScript.Arguments
'open Outlook
Set OutApp = CreateObject("Outlook.Application")
'open new email
Set oEmailItem = OutApp.CreateItem(olMailItem)
For a = 0 to objArgs.Count - 1
    Set oAttachments = oEmailItem.Attachments.Add(objArgs(a))
    'recreate the default Subject e.g. Emailing: file1.doc, file2.xls
    subjectStr = subjectStr & Right(objArgs(a),Len(objArgs(a))-(InStrRev(objArgs(a),"\"))) & ", "
Next
If subjectStr = "" then subjectStr = "No Subject "
'default is "Emailing: " in first quotes
oEmailItem.Subject = "" & Left(subjectStr, (Len(subjectStr)-2))
oEmailItem.BodyFormat = olFormatHTML
oEmailItem.Display
