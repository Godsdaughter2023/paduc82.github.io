Visio Timeline - Quarter Display and Labeling - Microsoft Community

mmasonvt say:

Visio Timeline - Quarter Display and Labeling

I've been going crazy trying to figure this out.  I'm building a timeline in Visio - displaying time in quarter increments - but I don't want the month to be on the timeline - I want it to show Q1 Q2 Q3...etc - as I can do with Project or something like SmartDraw.  There must be a way to do this - but I haven't found it.  I even was trying to create another data field but couldn't crack the code - that would have Q1 Q2 - etc.  If my timeline starts - today - then the Start date should be today's date - but then the increments should represent a shortened Q2 (since we're part way through already) - and then Q3, Q4, and repeating until the end date.  Can anyone solve this?
david j parker say:

The problem is that VIsio does not have a Date Format Picture for Quarters, and the Timeline addon creates the Interim markers whenever you change the Time Period Scale in the Configure Timeline dialog.

So the best solution I can come up with is to run a macro to update the format of the interim markers after you have selected the Quarters TimeScale. I have included an option to include the year as a suffix or not.

 

Const IncludeYear As Boolean = False

Public Sub ChangeInterimMarkers()
Dim shp As Shape
Dim shpTimeline As Shape
Dim shpMarker As Shape
Dim dateValue As Long
Dim newFormula As String

For Each shpTimeline In ActivePage.Shapes
    If shpTimeline.CellExists("User.visShapeType", visExistsAnywhere) Then
        If shpTimeline.Cells("User.visShapeType").ResultIU = 10 Then
            For Each shpMarker In shpTimeline.Shapes
                If shpMarker.CellExists("User.visShapeType", visExistsAnywhere) Then
                    If shpMarker.Cells("User.visShapeType").ResultIU = 17 _
                        And Not shpMarker.Cells("Fields.Format").Formula = "FIELDPICTURE(0)" Then
                            shpMarker.Cells("Fields.Format").FormulaU = "=FIELDPICTURE(0)"
                            dateValue = shpMarker.Cells("Fields.Value").ResultIU
                            newFormula = "=""Q""&FLOOR(MONTH(DATEVALUE(" & dateValue & "))/3)"
                            If IncludeYear Then
                                newFormula = newFormula & " & ""/"" & FORMAT(DATEVALUE(" & dateValue & "),""YY"")"
                            End If
                            shpMarker.Cells("Fields.Value").FormulaU = newFormula
                    End If
                End If
            Next shpMarker
        End If
    End If

Next shpTimeline

End Sub

david j parker say:

If you change FLOOR( to CEILING( in newFormula, then you will get the numbers you are looking for.

The Visio addon pushes the date value part (eg 41901) into the shape's text fields

41901 = 19th Sept 2014, as can be returned by the Formula =DATEVALUE(41901)

Setting up an Outlook.com IMAP account
Home / Outlook / Email / Setting up an Outlook.com IMAP account

Last reviewed on September 19, 2013

As of September 12 2013, the IMAP protocol works with Outlook.com accounts. This means you can access your Outlook.com email with any email client that supports IMAP, including Outlook 2011, Thunderbird, and more.

This should make the Outlook 2013 users who want to import email into the Outlook.com servers happy, as you can move mail into the IMAP folders to sync up to the server, something not possible with EAS accounts. In fact, if you are only using Outlook.com for email and don't need to sync Outlook.com calendar and contacts to your desktop, setting the account up as an IMAP account could be a better option.

To set up an Outlook.com IMAP account in Outlook, you'll need to use the manual setup option. Server settings are below; don't forget to set the Outgoing Server tab (in More Settings) to authenticate with the mail server.

Field Value
Incoming IMAP Server: imap-mail.outlook.com
Server port: 993
Encryption: SSL
Outgoing SMTP Server: smtp-mail.outlook.com
Server port: 587
Encryption: TLS

How to setup an IMAP account

You'll need to use the manual setup option when you configure an Outlook.com IMAP account in Outlook.

  1. Select Manual setup or additional server types and then click Next.

  2. Choose POP or IMAP from the next screen. Click Next.

  3. Enter your display name and email address and the Outlook.com IMAP server names. Use your email address for the account name and enter your password.
    Incoming mail server: imap-mail.outlook.com
    Outgoing mail server (SMTP): smtp-mail.outlook.com

  4. Click More Settings.
  5. On Outgoing Server tab, enable outgoing authentication, using the default of same as incoming.

  6. On Advanced tab, select SSL under Incoming Server (IMAP) and TLS for the Outgoing server. Change the Outgoing server to port 587. (The Incoming Server port should update to 993 automatically when you choose SSL).

  7. If you prefer to Purge your mail yourself, and not automatically when you change folders, change the setting at the bottom of the Advanced tab.
  8. Click Ok to return to the Add Account screen
  9. Adjust the Sync Slider if you want to sync more than the last 3 months to Outlook.
  10. Click Next to test the settings and exit the dialog.
