Some possibilities (pretty much anything, unfortunately) ...

- Obviously a corrupt graphics driver install is one possibility.

- You could also verify if the Nvidia gpu is faulting by running on your Intel HD integrated graphics. It should

handle The Sims and Minecraft well enough.

- A conflict with a Service or Starup program. You can troubleshoot this possibility by running in Clean Boot.

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7

- Monitor the temps on the cpu and gpu to verify they are not overheating under load.

- Run memtest to verify there is no problem with the RAM. Running only one stick at a time can also

help determine if one is faulting in the short term, but a long run with memtest is the most revealing.

- The power supply can also cause crashes under load if it is failing to provide a steady 12V current.

Your emails will be present on Yahoo's mail servers, they only downloaded a copy to your computer. You don’t have to move them back to the web mail. You just have to access your mail through your web browser just as you always have.

Warning: This will delete all your settings and mail. You will need to create a new account after performing these steps, so have your mail (POP/IMAP/SMTP) settings handy if you want to continue using Windows Mail.

 

To manually delete the settings do the following:

 

1.       Open Windows Mail, and go to Tools, and Options.

2.       In the Options screen click the Advanced tab and click the Maintenance button.

3.       On the Maintenance screen click the Store Folder button and a window should open with a folder path (it may look like “C:\Users\<username>\AppData\Local\Microsoft\Windows Mail”), select and copy the path in the text box.

4.       Cancel/Close out of the menus and close Windows Mail.

 

Next, go to Start and paste the path you copied earlier into the search box, and press enter to bring up the Windows Mail settings folder. You will need to either move all the files/folders in this folder to another location, or delete them to reset all the settings and mail.
