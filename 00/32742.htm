antoneth ber say:

Hello ANoTest,

Windows Update Catalog is a website that offers updates for computers. Through this, you can download Windows Updates manually. Here are the steps on how to download Update from Window Update Catalog manually:

  1. OpenWindows Update Catalog.
  2. In the search text box, type your search terms. (Example: Windows 10 1607 X64)
  3. Click Search.
  4. Browse the list that is displayed to select the updates that you want to download.
  5. Click Add for each selection to add it to the download basket.
  6. Click view basket under the Search box to view the download basket.
  7. Verify your list of updates, and then click Download.
  8. Note If you are prompted, click Accept to accept the license agreement.
  9. Select the location where you want to save the updates. You can either type the full path of the folder, or you can click Browse to locate the folder.
  10. Click Continue to start the download.
  11. When the download is complete, click Close to close the Download Window.
  12. Close the Windows Update Catalog Window.
  13. Find the location that you specified in step 9. &n Note: If you have downloaded device drivers for installation, go to Installing Drivers.
  14. Double-click each update, and then follow the instructions to install the update. If the updates are intended for another computer, copy the updates to that computer, and then double-click the updates to install them.

Should you need further assistance, feel free to get back to us.
