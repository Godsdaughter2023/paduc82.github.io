Uses the 'Text1' field.  Update 'Text1' in the code below to the field you wish to use.

Credit to Jan De Messemaeker
https://groups.google.com/forum/#!msg/microsoft.public.project/vCC5trIRLhE/JptVNbYbBQAJ


Sub PredecList()

Dim Job As Task
Dim Predec As Task
Dim Lis As String

On Error Resume Next

For Each Job In ActiveProject.Tasks
    If Not Job Is Nothing Then
        Lis = ""
        For Each Predec In Job.PredecessorTasks
            If Not Len(Lis) = 0 Then
                Lis = Lis & ", "
            End If
            Lis = Lis & Predec.Name
        Next Predec
        Job.Text1 = Lis
    End If
Next Job

End Sub

john sr wilson say:
If the hyperlink is to a slide in the same presaentation then the Hyperlink address will be null. You need to look at the subAddress.

Assuming the shape is on slide 1 try
Sub chexlink()
Dim ohl As Hyperlink
Dim oshp As Shape
For Each ohl In ActivePresentation.Slides(1).Hyperlinks
If ohl.Address = "" Then
If TypeName(ohl.Parent.Parent) = "Shape" Then
Set oshp = ohl.Parent.Parent
'subaddress is SlideID,SlideIndex,SlideTitle
MsgBox "Shape: " & oshp.Name & " has link sub address " & ohl.SubAddress
End If
End If
Next ohl
End Sub


Note in the sub address only the ID is totally reliable, the index and title may not be correct if they have been altered since the link was made
