I had this same problem and figured it out by accident. Windows Edge can only import from Internet Explorer, at least that what it showed on mine. So how to get there?

1. Open a webpage, like msn.com and then click on the three dots "..." in the upper right corner of Windows Edge.

2. On that menu you'll see "Open with Internet Explorer." When the page reopens in IE, from the menu bar, click "File", then click "Import and Export" and import the bookmark.htm file you've exported from your original IE favorites that you've saved on this new Windows 10 machine.

3. Close IE after import and open Edge back up.

4. Import Favorites by first clicking on the three lines in the toolbar next to the star and then import from IE and there ya go!

Want to change dual boot to single operating system - Microsoft Community

golddust5364 say:

Want to change dual boot to single operating system

My computer is in dual boot of Vista and Windows 8/8.1.  I'd like to reinstall the upgrades to overwrite Vista, and erase the current install so I can get the rest of my hard drive back for data storage since I no longer use Vista at all and that partition is just taking up space (just over half my hard drive).  My concern is activating it once it's installed.  The upgrades were provided to me by my computer club through TechSoup.org since I'm one of their Windows instructors.  Will I have to call someone to get the activation approved? 

hairyfool say:

You have to be very careful about removing the Vista install as this was the original boot O/S. The modified boot code to give you the dual boot option will be within the Vista drive so you cannot just format the Vista partition. You can remove the dual boot menu by running MSConfig and in the boot tab remove the option to boot Vista, when there is only one option it does not show the menu.


It is cleaning up the redundant files on the Vista drive is what needs to be considered. The files you need to protect will be hidden by default.


Reinstalling the base Windows 8 on a clean system would be the safest option, unfortunately the Techsoup donation from Microsoft is an upgrade program (you should not have both O/S installed anyway). About your only option is to reapply the upgrade but tell it to overwrite the Vista partition this time. You would then have a dual Windows 8/8.1 boot. Upgrade 8 to 8.1 then remove the secondary Windows 8.1 partition and remove the entry from MsConfig. As it is the same system with the same qualifying Vista license it should upgrade however you cannot guarantee that it will not require manual authentication.
