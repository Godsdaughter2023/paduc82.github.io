andrew conforti say:

After pouring over this and other posts about the same issue, I resolved it by changing the following:

  1. Press the 'Window key' and search for "Device Manager"
  2. Expand the node with your computer name (if not already expanded) and scroll down to "Human Interface Devices"
  3. Expand "Human Interface Devices"
  4. Right click and select properties on "Bluetooth Low Energy GATT compliant HID device"
  5. Select the 'Power Management' Tab
  6. Uncheck "Allow the computer to turn off this device to save power"  Screenshot below shows the default setting with it 'Checked.'

Disclaimer: This will probably cause your keyboard to consume battery life more quickly.

Seems the default setting for this keyboard is to go to sleep when the PC does, rendering it unable to wake the PC.  If you wake from a mouse or other device, the keyboard can be slow to wake - leaving you stuck trying to use your finger print to unlock the machine.

I would also recommend keeping "Bluetooth Support Service" running with automatic startup.  I'm unsure if you need to enable "Wake from USB" at the BIOS setting, mine was already enabled.

What didn't work:

  • Command: powercfg -deviceenablewake "BTLE Keyboard Fingerprint ID"
  • Device Manager settings for Bluetooth device "BTLE Keyboard Fingerprint ID"
  • Device Manager settings for Keyboards
  • Device Manager settings for Network Adapters "Bluetooth...."
  • Device Manager settings for any Universal Serial Bus controllers (USB)

My Desktop icons show up as a generic blank icon in Windows 7 - Microsoft Community

bobbymi say:

My Desktop icons show up as a generic blank icon in Windows 7

Shortcut icons on my desktop started showing up as a default icon (generic blank icon), but the programs still work fine. This might be caused when an Icon is a link to another Icon that links to the target file. The following steps worked to recreate the icons.

 

1.       Browser the computer to the actual file location. Example: Excel 2007 is installed in c:\program files\Microsoft office\office12\excel.exe

2.       Right-click the target file, “Excel.exe” and select "Send to", then select "Desktop (create shortcut)".

3.       Click "Yes" when prompting with the warning to replace the old shortcut file.

 

The icon should appear correctly.

 

If the Icon is located on the Taskbar or the Start Menu it might be because of the same reason. Remove the icon, browse to the target file and click “Pin to Start Menu” or “Pin to Taskbar”. The icon should appear correctly.

 

Hope this helps!
