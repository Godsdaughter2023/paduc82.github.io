norman david jones say:

Hi Tobemaster,

If the town names are in column C, to remove initial or trailing spaces, try something like:


'=========>>
Option Explicit

'--------->>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet
    Dim Rng As Range
    Dim Arr As Variant
    Dim LRow As Long, i As Long

    Set WB = ActiveWorkbook
    Set SH = WB.Sheets("Sheet1")                      '<<===== Change

    With SH
        LRow = .Cells(Rows.Count, "C").End(xlUp).Row
        Set Rng = .Range("C1:C" & LRow)
    End With

    Arr = Rng.Value

    For i = 1 To LRow
        Arr(i, 1) = Trim(Arr(i, 1))
    Next i

    Rng.Value = Arr
End Sub
'<<=========

syed n say:

Hi gary125,

 

·        Do you get any message when you try to close it?

 

One possible reason could be an incompatible add-on. Check to see if the following helps.

 

Method 1:

Step 1: Run Internet Explorer in No Add-Ons mode to check for the issue.

 

a.       Click Start, and then type Internet Explorer in the Start Search box.

b.      Click Internet Explorer (No Add-Ons). Internet Explorer opens without add-ons, toolbars, or plug-ins.

c.       Test to verify that it works correctly.

 

Step 2: If it works fine then use the Manage Add-ons tool to determine which add-on is causing the issue

 

a.       Open Internet Explorer. Click Tools, and then click Manage Add-ons.

b.      On the Show drop-down menu, select All add-ons to display all add-ons that are installed on the computer.

c.       For each item in this list, select the add-on, and then click Disable in the Information window.

d.      When you have disabled all the items in this list, click OK.

e.       Exit and then restart Internet Explorer.

f.        If issues do not reoccur, repeat steps 1 through 3. Click Enable for a single add-on.

 

Refer:

Internet Explorer add-ons: frequently asked questions

 

Method 2:You can try optimizing Internet Explorer and check for the issue.
