amit tandon say:

Below is a simple alternative:

=SUM(COUNTIFS(I12,{"PS","PSK","PSS"},E12,">=1"))+--(E12>14)+--(E12>21)

If you also wish to validate if E12 is a numerical, then you may use below:

=SUM(COUNTIFS(I12,{"PS","PSK","PSS"},E12,">=1"))+ISNUMBER(E12)*((E12>14)+(E12>21))

This will return 1 if E12>=1, return 2 if E12>14 & return 3 if E12>21.
