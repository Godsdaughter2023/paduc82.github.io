I'm using Office 2010 for Win 8.

gord dibben say:

This may be near to what you want.

Select row/columns to change then run macro.

Sub ChangeRowsColsInPixels()
  Dim r As Range
  Dim Answer As Double
  Answer = InputBox("How much change?")
         '1.1 increases 10%,   .9 decreases 10%
  For Each r In Selection.Rows
    r.RowHeight = r.RowHeight * Answer
  Next
  For Each r In Selection.Columns
    r.ColumnWidth = r.ColumnWidth * Answer
  Next
End Sub

windows defender has been corrupted, cannot be restarted and shows - Microsoft Community

akatrt say:

windows defender has been corrupted, cannot be restarted and shows error 577. What to to do?

windows defender has been corrupted, cannot be restarted and shows error 577. What to to do?

Windows defender cannot be started, cannot be updated, and when attempting to restart error 577 shows indicating the program has been modified, corrupted, possibly due to an earlier virus. Cannot find how to overcome.

greginmich say:

windows defender has been corrupted, cannot be restarted and shows error 577. What to to do?

Windows defender cannot be started, cannot be updated, and when attempting to restart error 577 shows indicating the program has been modified, corrupted, possibly due to an earlier virus. Cannot find how to overcome.

Error 577 on attempting to start the Windows Defender Service in services.msc is commonly associated with an error state that results when a third-party AV app is installed, but Windows Defender doesn't get properly disabled in its registry key as a result of the installation. The other error codes associated with this error state are Error 0x8007139F when attempting to start the Windows Defender Service in the UI, and Error 0x80070643 for failed Windows Defender definition updates.

If you have a third-party AV app installed, then you can safely ignore the issue. Error 577 is a false alarm – an artifact of this very common error state that generates a lot of confusion, but that is essentially harmless. The Error 577 component of this error condition is something that usually only gets noticed when a Microsoft Support Engineer directs people to services.msc in order to have them start the Windows Defender Service, which of course can’t be done when a third-party AV app is installed; as we see in this thread, for example:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_start/windows-defender-error-code-0x8007139f/93096739-16ce-471b-8716-1776f0c69f89

There are several ways to resolve the issue with the bad entry in the Windows Defender registry key, but it appears that the issue is getting resolved automatically by the Anniversary Update, where the introduction of Limited Periodic Scanning has changed the meaning of the problematic registry entry. So the best thing to do in this scenario is just ignore the issue, and wait for the update.

GreginMich
