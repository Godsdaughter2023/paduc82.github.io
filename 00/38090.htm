a. user say:

Right click on the folders (each one you want to protect) and click on Properties then on the Security Tab (or you can do this on the entire D: Drive if you prefer).  Make sure the Everyone user isn't there - delete it if it is (by clicking Edit).  Then delete any users you don't want having access to the folder until only those who should have access remain (while still in Edit).  Give yourself full rights if you don't already have them.  Check Advanced to make sure there are no special rights being given which override those in the main section (and delete or amend or add them if there are using that screen's edit function).  Make sure you apply the permissions to this folder and all subfolders and files.  That should take care of your problem (in effect, giving D: Drive the same permissions as existed in C:\Users\<username> - which you can review to check your work {though I think what I gave you will work, it's best to be sure by checking what existed before}).
