Wireless Security - Microsoft Community

xb1r athena say:

Wireless Security

Vizio XWR100

Login Information:

To log into your routers configuration page open up a web browser on your computer and in the address bar type in the gateway address below. You will not be prompted for a username and password because it is not required.

  • Gateway - 192.168.1.1 or http://vizio.home
  • Username - N/A (None)
  • Password - N/A (None)

How To Find Wireless Passphrase (WPA/WEP)

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the wireless symbol, (The symbol on the right).
  4. Click Edit button down at the bottom of the page.
  5. The wireless password will be written at the bottom of the page where it says security key.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open web browser.
  2. In the address bar at the top, type in 192.168.1.1 or http://vizio.home.
  3. From the menu at the top, click on the wireless symbol, (The symbol on the right).
  4. Click Edit button down at the bottom of the page.
  5. There will be a drop down box on this screen where you can disable or change the wireless security type.
  6. If WEP is selected, select the WEP Key Length (either 64-bit or 128-bit).
  7. If WPA-Personal is selected, select the WPA Mode (Auto, WPA or WPA2) and the Cipher Type (TKIP, AES or Both).  WPA2 with AES is recommended.
  8. If WEP security is selected the password will need to be 10 charecters in length (if 64-bit) or 26 characters in length (if 128-bit) and can only use 0-9 and A-F.
  9. If WPA or WPA2 was selected the password can be between 8-63 characters, is case sensitive and can use any character on your keyboard.
  10. Enter the Pre-Shared Key or WEP Key.
  11. Once the passphrase has been entered click Save Settings
