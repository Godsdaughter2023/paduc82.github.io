You will want to right-click the name tab of the Limits worksheet and select View Code. Paste the following directly into the right-hand pane titled something like Book1 - Sheet1 (Code),

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Range("O9")) Is Nothing Then
        Application.EnableEvents = False
        Sheets("DataINput_ML").Range("BI7:CR9").Interior.Pattern = xlNone
        'Sheets("Matrix_ML").Range("C7:P44").Interior.Pattern = xlNone
        With Sheets("Matrix_ML")
            .Range("E14:P14,E7:P7,E8:P8,C44:E44,E17:P17,K29:P29").Interior.Pattern = xlNone
            .Range("E15:P15,C43:E43,C40:E40,E9:E9,E14:P14").Interior.Pattern = xlNone
            Select Case Target.Value
                Case 2
                    Sheets("DataINput_ML").Range("BI7:CR9").Interior.Color = 65535  'Yellow
                Case 3
                    .Range("E14:P14").Interior.Color = 65535
                Case 4
                    .Range("E7:P7").Interior.Color = 65535
                Case 5
                    .Range("E8:P8").Interior.Color = 65535
                Case 6
                    .Range("C44:E44,E17:P17,K29:P29").Interior.Color = 65535
                Case 7
                    .Range("E15:P15").Interior.Color = 65535
                Case 8
                    .Range("C43:E43").Interior.Color = 65535
                Case 9
                    .Range("C40:E40,E9:E9,E14:P14").Interior.Color = 65535
            End Select
        End With
        Application.EnableEvents = True
    End If
End Sub

Press Alt+Q when you're done to return to the worksheet. Changing the value in Limits!O9 will clear all of the interior fills in the ranges you've provided and possibly fill on with yellow depending upon what number is provided in Limits!O9.
You will notice a commented line of code (9th one down). This is an alternate method to individually  resetting the ranges in Matrix_ML but only works if there are no other cells with an interior fill within Matrix_ML!C7:P44.
