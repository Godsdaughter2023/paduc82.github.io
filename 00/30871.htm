why do I have one user account (plus guest) and three user - Microsoft Community

whooper cheesie say:

why do I have one user account (plus guest) and three user profiles

After upgrading from Windows XP to Windows 7 I find that I am still confused about user accounts.  I am the only person using this computer so I should have only one account.  In control panel-user accounts and family safety-user accounts I have only one account listed - Charles administrator.  I use no user id or password to log on to this system.  When I click on the "Configure advanced user profile properties I have three profiles stored on this computer - charles-pc\administrator.  charles-pc\charles.  and default profile.  Have I made an error somewhere?  Do I need to delete a profile?
lemp say:

No problems.  Your "computer name" (created when you first setup your computer out of the box) is Charles-pc (or perhaps you attempted to name it Charles' pc and the apostrophe was converted to a dash).

  1. You have created a user account with the name Charles and designated it to have "administrator" privileges.  This is the one you see when you go to Control Panel > User Accounts.
  2. Windows has a built-in administrator account. By default, the built-in administrator account is named Administrator , and is disabled.
  3. Finally, Windows has a default profile, which Windows uses when you create a new user account.

When you look at "Configure advanced user profile properties," the accounts are listed in the form "computername\accountname."

Thus:

account #1 is the built-in (and normally hidden) administrator account: charles-pc\administrator

account #2 is your main user account: charles-pc\charles

account #3 is the default profile for creating new accounts: charles-pc\default
