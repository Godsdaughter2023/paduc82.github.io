To stop sending the Modification Resolution logs to a user, create a registry key:

In Windows click Start > Run and type in regedit.

Navigate to HKEY_CURRENT_USER\ Software\Microsoft\Office\x.0\Outlook\Options
(x.0 corresponds to the Outlook version: 16.0 = Outlook 2016, 15.0 = Outlook 2013, 14.0 = Outlook 2010, 12.0 = Outlook 2007)

Create DWORD value EnableConflictLogging
Modify the value data:
0 = Never save Modification Resolution logs
1 = Always save Modification Resolution logs
2 = Save Modification Resolution logs when a critical conflict occurs.

Disable Mopier mode - this mode is used when the printer has an EIO hard disk installed. If the printer doesn’t have one, it will only print one copy.

  1. Open the (Devices and) Printers Control Panel.

  3. Click the Device Settings tab.

christianpalmroos say:

This worked for me so far:

Start -> settings -> Network & internet -> VPN -> Network & sharing centre -> [on the top(ish) right] Ethernet or wlan -> properties -> configure -> Advanced tab -> In this advanced tab, scroll down and find: Wake On Magic Packet and Wake On Pattern Match. Choose one at a time and on the right, you can set the value to Disabled. Disable both.

I remember having to do this exact same thing with 8.1

Now, I'm not a very tech savvy person and I don'tknow if there's some harm this might cause, but at least the darn thing stays in sleep mode :D
