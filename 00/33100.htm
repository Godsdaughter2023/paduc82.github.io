Hello,

findstr /l /s /m "a string with words" *.*

Then, the command window displays all of the files from the directory and sub-directories. I think I see which file is being searched for the string in real time.

I expected to see only filename/s if the string was found within it.

What am I doing wrong? or, why is this malcfunctionaning?

I got the directives for the findstr command here: https://technet.microsoft.com/en-us/library/bb490907.aspx

Thank you

jjjdavidson say:

I think you want the /c parameter instead of /l.  As written, your search string is interpreted as a list of strings: "a", "string", "with", and "words".  Any file that contains an "a" will return a match.  The /l parameter just keeps your strings from being interpreted as regular expressions.

If you need to match the entire string exactly, type:

findstr /s /m /c:"a string with words" *.*

/c parameter, to search for more than one exact string at once.  For instance:

findstr /s /m /c:"a string" /c:"a word" *.*

will search for either "a string" or "a word" exactly.

mouse pointer hover windows 10 - Microsoft Community

georgechandler say:

mouse pointer hover windows 10

how do i turn off selection by hovering the mouse pointer?

davem121 say:
Hi George,

Open Control Panel - Ease of Access
Click 'Make the Mouse Easier to Use'
Uncheck 'Activate a window by hovering over it with the mouse'
Close Control Panel

If the above does not work:
Click your Start Button, type regedit and hit Enter to open the Registry Editor
Navigate toL
HKEY_CURRENT_USER\Control Panel\Mouse
On the right hand side, double click: MouseHoverTime
Set its value to 4000
Save and close the Registry Editor
