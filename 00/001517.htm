Definition - What does Spinner mean?

A spinner is a graphical GUI widget that allows a user to modify the value in the adjacent text box by either clicking the up or down arrow, or holding the up or down arrow, thereby allowing the value to increase or decrease. If a user holds down the up arrow, then the text box value increases. If a user holds down the down arrow, the text box value decreases. The speed of the value change may increase as the button is held down.

Functionally, spinners are much like scrollbars, sliders, combo boxes or lists, but they are considered beneficial because they do not change the visible display or cover up other values on a screen.

A spinner is also known as a spinbox or spinner control.

How default Word to open with All Outlines Closed - Microsoft Community

nate013 say:

How default Word to open with All Outlines Closed

I've created a document with multiple headings. Word defaults it to open with all the headings open.

Is there a way to open this specific document (or all documents) in an outline view with all the headings closed? 




m.stoni say:

I know this is an old question but I just had the same problem and I found out that there is a setting to make headings which are based on a style default collapsed/expanded:

HOME>Styles>Modify Style>Paragraph>Indents and Spacing>General>Collapsed by default

This option is per style and styles are per document so you would need to change every style in every document where you want your headings to be default collapsed. If you want your headings default collapsed in new documents you would need to change your default template (%appdata%\Microsoft\Templates\Normal.dotm).

Edit: See Rohn007's comment below how to import/export your changed styles to other documents.
