Hi,

You may receive Synchronization Errors including 0x80048820, 0x80070057, 0x80072EFF or 0x80048849 when using Windows Live Mail 2011. Most of the time, these errors go away if try to do the following:

1. Re-configure your account in Windows Live Mail: You can do so by following the instructions on the article: How to configure a Windows Live Hotmail account in Windows Live Mail . Sometimes you may need to remove the account before re-configuring it, make sure you follow the instructions to remove the account given at the bottom of the article, if applicable.

2. Access the account using POP3: If you are on dial up, please try accessing your account using POP3. For Windows Live Hotmail account, you can refer to the article Send and Receive Windows Live Hotmail emails from a Mail Client for instructions on how you can set up your account as POP3.

If the above steps didn’t work for you, these errors may be occurring due to three different kinds of issues: Network Issues, Machine Configuration Issues or Software Configuration Issues. Given below is a list of the issues and you may click on the links below to read articles about how to resolve each of these categories of issues.


Network Issues:

1. You were not connected to the Internet when you tried to synchronize your account.

2. The mail server that you are attempting to send or receive data from may not be available.

3. DNS may have incorrect cached data.

4. The MTU setting on the mail client is larger than the MTU settings configured on network devices between your mail client and the mail servers.


Machine Configuration Issues:

1. Date and Time Settings on the computer are incorrect.

2. SL Security DLL libraries may not be properly registered.

3. Firewall or security software may be blocking Windows Live Mail or may be blocking a required port to send or receive mail.

4. Access is being blocked by your router or Wireless AP.

5. Incorrect entry in the hosts file.

 

Software Configuration Issues:

1. An email is stuck in the Outbox folder of Windows Live Mail.

2. You may not be using the latest version of Windows Live Mail.

3. Your computer needs Microsoft’s latest updates.

4. Your mail account may not be properly configured in Windows Live Mail.

5. Internet Explorer settings may be causing connectivity problems.

 

        A.      SSL settings are incorrect.

        B.      Incorrect Proxy settings.

        C.      Internet Explorer may be set to work offline.

Thanks!

You posted in an XP forum so I'm guessing you are running XP?

Sounds like you do not have a .wmv video file but a .mswmm project
file. The .mswmm project file is only useful for re-editing purposes.

The Red Xs are indicators that the original source files...Movie clips,
Photos, Music files... are no longer available. If you have a folder on

in the project you can right click the Red Xs and browse to the location
of the source files and it may re-establish the path but more than
likely you will have to start over on the new system and recreate the
project.
