Using a Column and Row Index Number with VLookup?

I am looking for a formula that will do the following:

Lookup value from sheet1 cell a1, in sheet 2, and then return the value that is in Column D (so col index num 4) but then the value from the cell 14 cells below this (value of 14 may need to be changeable, but at present this will be set at 14). i.e. If Sheet1 A1 is found in Sheet 2 A7 then return col index 4 +14 down (D21).

I have tried various things and looked at other formulas but can't find what it is I need to make this work.

=OFFSET(INDIRECT("Sheet2!" & ADDRESS(MATCH(A1,Sheet2!A:A,0),1)),14,3,1,1)

Or alternatively

=INDEX(Sheet2!A:D,MATCH(A1,Sheet2!A:A,0)+14,4)
