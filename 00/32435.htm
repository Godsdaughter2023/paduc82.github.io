Restore Factory Defaults

Login Information



  • Gateway: 192.168.0.1
  • Username: admin
  • Password: (blank)





Restore Factory Defaults



Through Firmware



  1. Connect the computer to one of the LAN ports on the router via an Ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  4. When prompted, enter "admin" for the user name and leave the password blank.
  5. Select "Tools" from the menu at the top.
  6. Select "System" from the menu at the left.
  7. Select "Restore Factory Defaults".



Through Hardware　



  1. Connect the computer to one of the LAN ports on the router via an Ethernet cable.
  2. Locate the "Reset" button on the device.
  3. Using a toothpick/pen/paper clip, press and hold the button for 10 seconds.
