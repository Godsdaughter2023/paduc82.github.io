AFAIK you cannot do this in an ActiveX TextBox on a worksheet or with a TextBox on a Userform.

 

You can use a TextBox created on a worksheet from the Insert Ribbon -> TextBox (In the Text block towards right hand end of Insert ribbon.)

 

The following code for a TextBox created on a worksheet from Insert Ribbon.

 

Note the commented out (Optional) If and End If lines. Using the If here will restrict the color to words with a space at the end. eg text and textbox in the following text would have the characters "text" colored red without the If statement but with the If statement only text would be colored red and the text characters of textbox would not.

 

This is a test of changing the text font on  the specific characters in the textbox when the word has been searched and the resulting test.

 

Sub FindTextAndSetFontColor()
    Dim shpTxtBox As Shape
    Dim strToFind As String     'Text to find
    Dim strToSearch As String   'String to search
    Dim lngPos As Long          'Start position of string to find
    Dim lngStart As Long        'Initial start search position
    Dim lngLgth As Long         'Length of text to find
   
    strToFind = "text"
    lngLgth = Len(strToFind)
   
    Set shpTxtBox = Worksheets("Sheet1").Shapes("TextBox 1")
   
    strToSearch = shpTxtBox.TextFrame.Characters.Text
    lngStart = 1
    lngPos = 0
   
    With shpTxtBox.TextFrame
        Do
            lngPos = InStr(lngStart, strToSearch, strToFind)
            If lngPos > 0 Then
           
                'If .Characters(lngPos + lngLgth, 1).Text = " " Then    'Optional
                    .Characters(lngPos, lngLgth).Font.Color = vbRed
                'End If                                                 'Optional
               
                lngStart = lngPos + lngLgth
               
            Else
                Exit Do
            End If
        Loop
    End With
End Sub

charles kenyon say:

I believe Ryan does not understand what you are trying.

I'm not sure it is possible.

Which method to embed are you using?

If Insert Text from File, what you get is just that: text from the file in your new file. This is not a link to the file. You need the password to initially do this, but once it is done, the text is in your new file.

If you use Insert MS Word File as an object, as a link or not, no editing is allowed.

In either case, anyone opening the new, containing, document will have read-access to the content of the original document.

You can insert a hyperlink to the document rather than embedding. That will trigger the password each time it is used, but will not put the protected document inside the unprotected one.
