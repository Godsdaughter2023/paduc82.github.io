As Gina noted you can't do this with the wizard. You will need a VBA function to do it.

Is there only one record in each Text file? 

You would need to use the OPEN and LINE INPUT commands to read in the text file line by line. A search on the commands will get you an explanation of how they work and sample code to use.

If there are more than one record per file, you will need to find a trigger for the first field in each record(Event Type??). The Line Input command will read in each line in the text file one at a time. You will test for the field name (Use the Left and Instr functions to get the text to the left of the :)  Then use the Right and Instr functions to find the data to the right of the :. 

So you will need to first create a table to store the fields you want., then write the data pulled to the correct field in the table.
