daniel g. grau say:
What's happening and how to solve it
When you insert symbols like you did (or by dragging from the palette), you are actually changing the font into Wingdings, which instead of regular letters has all those symbols. And of course, once you do that, you then keep on typing in the Wingdings font.

The solution is to change the font back into what it was before you inserted the symbol. Better yet, first type the surrounding text (before and after the checkmark) and then insert the checkmark.

Another way to do it
There is another way to insert such characters that uses some kind of "wrapper" around the symbol to isolate it from the surrounding text, thus avoiding the font change problem you are seeing. This is done as follows (note that the visual interface is rather obsolete): 

  • In the main menu bar, select Insert > Symbol > Advanced Symbol
  • In the dropdown menu, select Wingdings
  • Locate the checkmarks (bottom right)
  • Click once to see a larger version, twice to insert it

Now, unlike the other method, if you change the font in the whole document, the symbols remain unchanged.

How to automate it
To simplify insertion of such characters (I recommend using the Advanced option, not the palette), you can define an AutoCorrect entry as follows:

  • Select the symbol with your cursor (again, it's better if you've inserted it with the Advanced option)
  • Go to Word > Preferences > AutoCorrect
  • At the top, select the AutoCorrect tab
  • Place a checkmark in "Automatically correct..."
  • In the Replace fill-in field, type a unique sequence, preferably short, like chk\ or /check/. The idea is to have a unique sequence that you would not normally type, so it won't be AutoReplaced by mistake.
  • The "Formatted text" radio button should already be selected.
  • Click on the Add button and you're done.

From now on, typing your unique sequence will produce the selected symbol. Note that this is language-dependent, so if you change to another language, you'll have to redefine it.

You can do that if you use an ActiveX combo box...

smittypro1 say:

You can do it seamlessly with a Change event. The following assumes your validation is in cell A1, and you have a table, called tbl_States. As soon as you change A2, the code will look for its matching value one column to the right with VLOOKUP, and replace it. It will work the same with Employee Name/#, etc.

    '   Code goes in the Worksheet specific module
    Dim rng As Range
    Dim strAbbv As String
    Dim rngStates As Range

        Set rng = Target.Parent.Range("A2")
        Set rngStates = Range("tbl_States")
        '   Only look at single cell changes
            If Target.Count > 1 Then Exit Sub
            '   Only look at that range
            If Intersect(Target, rng) Is Nothing Then Exit Sub
            '   Action if Condition(s) are met (do your thing here...)
            If LenB(Target.Value) > 0 Then
                strAbbv = Application.WorksheetFunction.VLookup(Target.Value, rngStates, 2, False)
               
                Application.EnableEvents = False
                    Target.Value = strAbbv
                Application.EnableEvents = True
            End If
           
End Sub
