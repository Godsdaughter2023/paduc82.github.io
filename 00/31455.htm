@SpreadTrack27: Per favore, la prossima volta crea un thread apposito perché in questo thread si parla di un'altra cosa.

Comunque, in Windows 8 il driver del controller è incluso nel sistema operativo.

Per il Controller Xbox 360 per Windows è necessaria una porta USB alimentata. Se il controller non funziona correttamente, provare a connetterlo a una porta USB diversa. Evitate di utilizzare hub USB o le porte presenti nella parte anteriore del PC. Le porte o gli hub non alimentati potrebbero non fornire energia sufficiente per il controller. Per ottenere risultati ottimali, collegare il controller a un'altra porta USB nella parte posteriore del computer.

Le porte presenti nella parte posteriore di un computer sono in genere porte alimentate, mentre quelle nella parte anteriore potrebbero non esserlo. Un hub USB esterno non dispone di porte alimentate se non è dotato di un adattatore CA o di un alimentatore separato.

Windows 8

In un computer che esegue Windows 8 è possibile utilizzare Dispositivi e stampanti per risolvere i problemi relativi al controller. A tale scopo, attenersi alla procedura indicata di seguito.

  1. Nella schermata Start fare clic su Pannello di controllo.
  2. Fare clic su Visualizza dispositivi e stampanti.
  3. Verificare che il controller sia visualizzato in Dispositivi, quindi fare clic con il pulsante destro del mouse sul controller e scegliere Impostazioni periferica di gioco.
  4. Controllare l'opzione Stato del controller, quindi fare clic su Proprietà.
  5. Nella scheda Test verificare che il controller funzioni correttamente. In caso di problemi fare clic su Taratura nella scheda Impostazioni.
  6. Se i problemi persistono, fare clic con il pulsante destro del mouse sul controller in Dispositivi all'interno di Dispositivi e stampanti e scegliere Risoluzione dei problemi.

All modern processors are 64 bit, so your tablet may have had 32 bit installed due to low RAM or a weak CPU.  Less than 2gb RAM runs 32 bit better.  So there may be a reason it has 32 bit.
