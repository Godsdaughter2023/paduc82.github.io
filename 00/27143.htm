That suggests that the foreign key BuyerID column in BookedViewings is indexed uniquely.  It should be indexed, but not uniquely, i.e. 'duplicates allowed'.

A few general points:

1.  Don't include spaces in table or column names.  Use CamelCase or represent a space with an underscore character like_this.

tammy.m say:

The error code 17897 is an authentication error code.

When this happens there is something wrong with how you are authenticating with the servers.

That is why deleting the profile and recreating it sometimes works.

Here is an article that explains it more fully.

Sending email error "Authentication failed. Error 17897" in Outlook 2011 for Mac

If you search the internet for the error code 17897 you will see that authentication is the problem.

When a problem like this occurs and you did not make any changes then it is likely that the Internet Service Provider (ISP) has made changes and it is usually to the outgoing mail server or aka SMTP server.

The reason these sudden changes occur is because of hackers attempting to send spam email through SMTP servers in order to send you and me spam or phishing email or worse a virus.

Internet Service Providers (ISP's), in order to dissuade this activity must constantly either effect changes on SMTP servers such as requiring secure authentication or setting up a port on the server to require Secure Socket Layers (SSL).

If your ISP is insisting that it is a problem with Outlook then I don't know if we will be able to find a solution because every ISP has different settings for their servers.

I can only suggest at this point that you double and triple check your Incoming and (especially) outgoing email settings for the server.

Word 2016 Text Effects change line spacing - Microsoft Community

g.i. joe_1966 say:

Word 2016 Text Effects change line spacing

Applying Word's text effects (glow, shadow, reflections, etc...) to a line of text containing manual line breaks can profoundly change line spacing.  Also, different combinations of text effects result in different line spacing values.  How does Word calculate the new line spacing value?

jay freedman say:

I don't know that you (or anyone) can come up with a formula -- or more precisely, you could come up with many formulas, one for each combination of effects.

Essentially, the process is the same as when you insert an inline graphic or change the font size of a single character or word in a paragraph that has "At Least" line spacing. The baseline-to-baseline spacing expands to allow room for the largest object in the line. In some cases the additional space is at the top of the line containing the object, and in others (especially Reflection) it's below that line. Glow seems to add much more space than needed, while Outline doesn't seem to add any. With combinations of effects, I think it adds as much space as the effect with the greatest size.

If you want to measure a particular combination, go into the Paragraph dialog and set the line spacing to Exactly 12 pt (or some other starting amount) and then increase it until the paragraph looks the same as it does with At Least spacing -- that will tell you the baseline separation created by the effect.
