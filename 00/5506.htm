You have to take a look at the license terms on the computer. In 2016 I go into one of the Office applications > File menu > About button. This displays the license terms. (2013 may have a slightly different path, MS keeps "improving" things)

Click inside of the license terms then use <CTL><A> to select all. Paste it into a blank document. Search for "transfer".

<snip>
a. Software preinstalled on device. If you acquired the software preinstalled on a device, you may transfer the software directly to another user, only with the licensed device. Before any permitted transfer, the other party must agree that this agreement applies to the transfer and use of the software.
</snip>

In other words, that copy of Office is tied to the computer, even you cannot use the license on a different computer.

gord dibben say:

I edited some of the code I posted earlier.  Looks like you missed some changes.

To create 12 workbooks........one for each month using a template sheet add this revised code to the workbook you are using for the template.  Assumes the template sheet is named Sheet1.

 

Sub Add_Sheets_Months()
    Dim i As Integer
    For i = 1 To 12
    Call AddMonthSheets(i, 2012)
    Next i
End Sub

 

Public Sub AddMonthSheets(ByRef Mnth As Integer, ByRef Yr As Integer)
    Dim wks  As Worksheet
    Dim dte  As Date
    Dim lCounter As Long
    Dim wkbk   As Workbook
    Set wkbk = ThisWorkbook  'workbook with code and template sheet
    Set wks = wkbk.Sheets("Sheet1")    'Sheet to be copied assuming template for others
    Application.ScreenUpdating = False
    Workbooks.Add
    For lCounter = 1 To Day(DateSerial(Year(Now), Mnth + 1, 1) - 1)
        dte = DateSerial(Yr, Mnth, lCounter)
        If Month(dte) = Mnth And (Weekday(dte) <> 1 And _
                                  Weekday(dte) <> 7) Then
            wks.Copy after:=ActiveSheet
            ActiveSheet.Name = Format(DateSerial(Yr, Mnth, lCounter), "ddd Mmm dd ")
        End If
    Next lCounter
    Application.DisplayAlerts = False
    With ActiveWorkbook
        .SaveAs Filename:="C:\Gordstuff" _
                          & "\" & Format(dte, "mmmm") & ".xlsx"
        .Close
    End With
    Application.DisplayAlerts = True
    Application.ScreenUpdating = True
End Sub

always learning... say:

Sending email to all in the address book

always learning... say:

I have resolved this by Googling for an answer. To assist anyone with a similar problem, open address book. You will probably find that the radio box for 'more columns' is checked. Check the other option, the one for 'name only'. Your names will reappear.

 

To send to all on your address book without revealing them to each other, compose your message (you can edit it later before sending). Then click on BCC (blind cabon copy). Go to your address book and click on the first address. Hold down shift and travel to the last address. All addresses will now be highlighted.

 

In the To box put your own address so that you can verify the sending.

 

I do not claim to be an expert but the above worked for me.
