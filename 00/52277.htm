considerate_guy say:

When you add credits, Movie Maker adds two items:

  1. A blank title image (usually black, but you can change this)
  2. The text caption on this blank image.

 

The solution is simple:

  • Select the black image
  • Go to the Video Tools tab
  • Extend the duration of the background image to as long as you need (from the default of 7 seconds).
  • Now go back and extend the duration of the text using the Text Tools tab in a similar fashion.
