gord dibben say:

This worksheet event code will enter a static date/time into a cell in Column B when something is entered in a cell in the range A1:A10

 

Private Sub Worksheet_Change(ByVal Target As Range)
''Col B time will not change if data in Col A is edited
    On Error GoTo ws_exit:
    Application.EnableEvents = False
    If Not Intersect(Target, Me.Range("A1:A10")) Is Nothing Then
        With Target
            If .Value <> "" Then 
                .Offset(0, 1).Value = Format(Now, "dd mmm yyyy hh:mm:ss")
            End If
        End With
    End If

ws_exit:
    Application.EnableEvents = True
End Sub

Right-click on the sheet tab and "View Code".  Copy/paste the code into that module.

 

Edit the A1:A10 range to suit............post back if need help with that...........Alt + q to return to Excel.

 

NOTE: as writen, the date/time in B won't change when A is re-edited.  If you want that altered post back

 

Post edited.............NOTE above is incorrect..........B will change when A is re-edited.

Not quite sure what you're asking for here.

 

Do you want to be able to just select any cell and have the date/time entered in the same cell?

 

You could do that manually using CTRL + ; <space> CTRL + SHIFT + ;

 

Or assign this macro to a button or shortcut key after copying into a General Module

 

Sub NOWDATETIME()
    ActiveCell.Value = Format(Now(), "dd-mm-yyyy hh:mm:ss AM/PM")
End Sub

Or a double-click event in the worksheet module so when you d-click any cell the date/time is entered.

 

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
      Target.Value = Format(Now(), "dd-mm-yyyy hh:mm:ss AM/PM")

       Cancel = True
End Sub
