Should I keep PDF file in OneNote

I am in the process of scanning and copying many home documents into OneNote.

My question is whether if there are any advantages of retaining the PDF files that are copied into OneNote (automatically) given that I have the printed copy displayed in the program. Retaining the PDF files complicates formatting and I assume takes up additional memory If there any reason to retain the PDF files?

Thanks in advance. 

swischner say:
If I am understanding your question right, this is about keeping the embedded PDF file (represented by an icon) along with the viewable printout version, right?

The difference between these two is, that the embedded (icon) version is still a PDF file with all it's properties and characteristics. It's a full featured 1:1 copy of your original file, just "hidden" in the OneNote file system. Yes, you are right, it takes up additional space of course.

The printout (visible) version is not a PDF at all anymore. It's just an image of your original PDF (in fact in PNG format as far as I know). SO it's basically the same as if you would have printed your original PDF on paper, scanned it (or made a photo of it) and put the resulting image onto a OneNote page. While it's searchable (due to Microsoft's OCR-in-images function only), all other properties of the original PDF ("real" text, embedded fonts, watermarks...) are gone.

Knowing this, the answer to your question depends mostly on what you are planning to do with your original PDF. If you want to keep this, you should be able to get rid of the embedded (icon) version in OneNote to save some storage space. But this also means that, if you ever want to open and view the PDF version, you need access to the actual PDF file. With an embedded copy in OneNote you would also be able to open the PDF on another device, as long as that one has OneNote installed and a PDF reader app.
If you are certain that you would never need the original PDF anymore (let's say you scanned a receipt for your taxes and keep the original paper version anyway), it's also safe to remove the PDF version from OneNote.

If you are planning to delete the original PDF but might need a "real" PDF version at some time, you should keep the embedded version next to the printout in OneNote.
