a. user say:

I put several files (mpegs) into a folder on my desktop, came back to work on it the next day half of the files were gone.

Since the introduction of user accounts (in Windows XP, I think,) our personal files are supposed to be stored in our personal folders (Documents, Music, Pictures, etc.) This is where Windows anticipates that users will store their personal files, and for that reason these folders are not touched during updates. 

Of course, there's nothing stopping us from storing our files in other places - and everybody, including me, stores files on the desktop - but files stored outside our personal folders are not protected during updates.

The desktop, in particular, is a tricky place to store files, because what you see on your desktop is a combination of three separate folders: the Desktop folder in your user account; the Desktop folder in the Public account; and the Windows system desktop. Only the first is protected during updates. That's why it's not a good idea to put files on the desktop.

The desktop was intended for shortcuts to personal files, not for files themselves. Every now and then I have to go through my (cluttered) desktop and replace files and folders with shortcuts to the same.

That said, you have presumed,  without any evidence, that Windows deleted some of your mpegs. How do you suppose Windows decided which mpegs to keep?

Finally, it's so easy to backup files and folders everyday, practically effortless, but people still don't take this simple precaution that would alleviate most of their suffering.

What is in Windows.old Folder?__ - Microsoft Community

jrhoot say:

What is in Windows.old Folder?__

I have Win 7 Home Premium and after a restore problem made a second install ( after uninstall). What is the purpose of Windows.old and would I be advised to delete this folder? 

salvation at the cross say:

Hi,

When you perform a custom installation of Windows Vista, files that were used in the earlier version of Windows are stored in the Windows.old folder. The Windows.old folder is generated when you perform a custom installation of Windows Vista.

The Windows.old folder is generated if the following conditions are true:

  • You install Windows Vista on a computer on which Microsoft Windows XP or Microsoft Windows 2000 is installed.
  • You perform a custom installation of Windows Vista instead of an upgrade installation.
  • You install Windows Vista on the drive on which Windows XP or Windows 2000 is installed.

The Windows.old folder contains the following folders:

  • Windows
  • "Documents and Settings"
  • "Program Files"

The Windows folder mainly stores programs and files that are required to run the earlier version of Windows.

The "Documents and Settings" folder mainly stores the "My Documents" folder and the Favorites folder. Additionally, the "Documents and Settings" folder stores the desktop files that were used by the earlier version of Windows.

The "Program Files" folder mainly stores the programs that were installed on the earlier version of Windows. To use these programs in Windows Vista, you must reinstall these programs in Windows Vista

Important Before you remove the Windows.old folder, make sure that there are no important files in it.

For more information about the Windows.old folder, click the following article number to view the article in the Microsoft Knowledge Base:

933209  (http://support.microsoft.com/kb/933209/ ) How to retrieve files from an earlier version of Windows in Windows Vista
