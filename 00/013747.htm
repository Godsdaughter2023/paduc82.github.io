Short term work-around.  Instead of using file save as to create file 2 you can copy paste the file in Windows Explorer (that includes the Word File Open dialog). 

For example, you have your "template", file_one. It is closed. In Word use File Open to find file_one.  Single click to select file_one the use <CTL><C> to copy it to the clipboard and <CTL><V> to paste it back. It will now have the name "file_one copy.docx". You can right click on it and select rename and change the name to whatever, file_two.  Now with the file still selected you can open file 2 and do your editing.


Long term fix is to work through the following trouble shooting procedures to see if you can find where the save is happening.  It sounds like you have some sort of "autosave" macro running.
