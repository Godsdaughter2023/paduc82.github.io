kathleen orland say:

Are you connecting to an Exchange mail server? Everything in your mailbox in that case counts against you because it is all actually held on the server.

 

What you need to do is create Personal Folders. You can then drag and drop your mail into your Personal Folders which removes it from your mailbox on the server. However, once your mail is removed from the server it is no longer being backed up. You will be responsible for doing that.

 

Click on File > New > Outlook Data File. Choose the default Office Outlook Personal Folders File (pst). Click OK. Give it a name (the default is outlook.pst if you don't). Personally I like to use my first initial last name and the date I create it  (example: korland070612). Click OK.

 

To back it up, close Outlook. Locate the file. Right-click on it, scroll to copy, and then paste it in a folder on your network.

 

This all assumes you have an Exchange mailbox however.

Conditional Formatting - If one cell is divisible by 3, then change - Microsoft Community

ethanparker1 say:

Conditional Formatting - If one cell is divisible by 3, then change the color of a different cell

I'm working on a spreadsheet that we are using to monitor a tabletop board game library.

For each game we have a "Number of Times Checked Out" field. I would like to have the row (or even just the game title) in the spreadsheet change color to red whenever this number is divisible by 3.

Obviously the cell needs to be returned to the normal color when the number is not divisible by 3.

hansv mvp say:

Let's say the "Number of Times Checked Out" field is column N, and that the actual data start in row 2.

Select the entire range that you want to format conditionally. I'll assume that the active cell within the selection is in row 2.

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Select 'Use a formula to determine which cells to format'.

Enter the formula

 
=MOD($N2,3)=0

 

Change N to the column letter of the "Number of Times Checked Out" field, and change 2 to the row number of the active cell if needed.

The $ before the column letter is essential.

Click Format...

Activate the Fill tab.

Select red.

Click OK twice.
