You condition on this code line

If (c <> 2 And c Mod 2 = 0) Or c <> Int(c) Or c = 1 Then

is wrong.

The only numbers it will place in AB:AG are the number 1, all even numbers except 2 and all non integers.

The number 9 does not fall into any of these categories.

Try this modified macro and see if it does what you expect:

Sub MarkPrime()
Application.ScreenUpdating = False
Application.Calculation = xlCalculationManual
Dim i As Long, x As Long, c As Range
On Error Resume Next
For Each c In Range("B3:G2550")
   prime_number = True
   If c = 1 Or c Mod 2 = 0 Then prime_number = False
   If c = 2 Then prime_number = True
   For i = 3 To Sqr(c) Step 2
      If c Mod i = 0 Then
         prime_number = False
         Exit For
      End If
   Next
   If prime_number Then
      c.Interior.Color = vbYellow
      With c.Offset(, 13)
         .Value = c.Value
         .Value = .Value
         .Interior.Color = c.Interior.Color
      End With
   Else
      With c.Offset(, 26)
         .Value = c.Value
         .Value = .Value
      End With
   End If
Next
Application.Calculation = xlCalculationAutomatic
Application.ScreenUpdating = True
End Sub
