Font themes are very simple XML files.  Example  ... save this text to a plain text file with an XML extension, edit the bolded bits and you've got a font theme.  If necessary, edit the blanks after a:ea typeface (East Asian font) and a:cs typeface (Complex Script font).

<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<a:fontScheme xmlns:a="http://schemas.openxmlformats.org/drawingml/2006/main" name="NAME OF THEME GOES HERE">
    <a:majorFont>
        <a:latin typeface="Arial"/>
        <a:ea typeface=""/>
        <a:cs typeface=""/>
    </a:majorFont>
    <a:minorFont>
        <a:latin typeface="Georgia"/>
        <a:ea typeface=""/>
        <a:cs typeface=""/>
    </a:minorFont>
</a:fontScheme>

Library:Application Support:Microsoft:Office:User Templates:My Themes:Theme Fonts
