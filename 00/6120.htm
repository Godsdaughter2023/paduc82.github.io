This fixed my RM-875 driver

  1. Go to http://catalog.update.microsoft.com using ONLY internet explorer
  2. Search for "windows phone winusb". You should find "Windows Phone - Other hardware - WinUsb Device".
  3. Tap or click Add in the right-most column.
  4. Tap or click View basket (just below the search box).
  5. On the "Updates in your basket" screen, tap or click the Download button. Tap or click the Browse button and select a convenient location, such as your Desktop or the Downloads folder, to save the file. Tap or clickContinue, and then tap or click Close when downloaded.
  6. Browse to the file you downloaded (which may be contained in a folder called "Windows Phone - Other hardware - WinUsb Device") and extract winrar file inside
  7. There will be 2 files:  winusbcompat.cat and winusbcompat.inf
  8. Copy both of them in C:\Windows\Inf\ 
  9. Open Device Manager, right click RM-875 | Nokia 909, update driver
  10. Tap or click Browse my computer for driver software
  11. Tap or click Browse, locate the C:\WIndows\Inf\ then click OK
  12. and driver will be installed and done ^^

Line chart will simply give a connecting line with the data points given. 

In your eg, 8-Feb - 1; 10-Feb - 2; 18-Feb - 1
Line will present connecting 1 to 2 and then to 1. 

Option1: 

For you to have the peaks and valley correctly, you need to include the data with Zero occurrence also with value of 0 in the occurrence column. 

Apart from the table value given, 

8-Feb - 1, 10-Feb- 2, 18-Feb - 1, 19-Feb - 1

include these data also in the table. 

9-Feb-0. 11-Feb- 0, 12-Feb - 0, 13-Feb-0.......

Now create a line chart. 

Option 2:

Create a table with all dates. from 8-Feb till 19-Feb. Update value only for occurrence dates. Let the other dates be blank in occurrence column. 

Create a Line chart in excel. 

Click on the chart, go to select data under data in Design. Select data source window will open. Click on Hidden and Empty cells, in that select Show Empty cells as "Zero". 

Now your line chart will cover all the dates with proper peak and valley.
