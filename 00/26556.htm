bill manville say:

I got slightly lost in your description, so I will just suggest some principles for working with linked spreadsheets:

1 - have them all in the same folder, or the link sources in sub-folders of the folder containing the master file. [that ensures that the links are held as relative so that if different users see them through different paths they will still work]

2 - never copy files by opening and doing Save as to a different folder (or the links will cease to be relative)

3 - always copy the whole set of files together

4 - name the ranges to which you are linking in the source files rather than using addresses

5 - don't modify a source workbook in a way that will invalidate the links to it,

6 - if making changes to the links, do it with the source files open.

Hope this helps you to avoid problems.
If you still have a problem perhaps you could post the workbooks concerned to a file-sharing site and describe the problem in detail.
