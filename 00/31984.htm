You can rename the folder, but it might cause more confusion.  Windows has a way for folders to have a display name that is different from the actual file system name.  The display name comes from a hidden file within the folder named desktop.ini.  The effect of this is most easily seen in the 'Public folders.  Notice how the display name in Explorer differs from the file system name viewed from a Command Prompt:

So until you're sure of what's pointing where, you might want to hold off on renaming.

In terms of the file system, a user by default has folders for Documents, Pictures, Music, Videos, Downloads, Contacts, Desktop, and OneDrive.  Everything under OneDrive gets synchronized with the cloud.  Looks like Documents, Pictures, and Music are subfolders there by default.  So you essentially have two folders named Documents on your hard drive.

Then you have shortcuts to Documents under This PC and Quick Access.  To determine which folders these refer to you should check the Location tab in the folder's Properties dialog.  Here, This PC Documents points to C:\Users\USerName\Documents while Quick Access Documents points to C:\Users\UserName\OneDrive\Documents. 

This was the result of changing the Auto save settings for OneDrive which are accessed by right-clicking the OneDrive icon in the Notification Area & selecting Settings:

I normally keep them set on This PC only.

So let's make sure everything is saving where you want it to first.

keith a. miller say:

Changing from OneDrive to This PC only changes the target for the 'Documents' and 'Pictures' shortcuts under Quick Access.  It also means that when well-behaved programs 'ask' Windows: "Where are files for this user stored?", they will be pointed to the Documents or Pictures folder in your User folder rather than your OneDrive folder.  As most programs remember the last folder you used for open/save actions, you may only notice this on newly installed programs.

As for the Accounts tab & the Choose folders button, this is for selecting which folders from your Cloud storage will be replicated to your OneDrive folder.  More for those users that want their 'master' file collection in the Cloud on only selected items on their local device due to storage limitations.   I leave them all checked.

Keith

W10 ignoring device installation settings/forcing driver updates

As many of you already know, Windows 10 appears to allow the user the freedom to control how device drivers are handled by Windows Update through the device installation settings panel under advanced system settings.

Forcing OS updates without negotiation can be justified; forcing hardware driver installations cannot (especially when you introduce a setting that should control this behaviour in order to maintain appearances and then proceed to ignore it). Observe the following scenario: A user performs a clean install of Windows 10 Professional 64-bit (w/ ethernet cable disabled so as to install drivers of the user's own choosing and in the order of the user's own choosing). I will make a final observation that just because a company provides a technology that is needed in certain fields (DirectX 12) does not mean that specific market niche is going to be very happy with them if they decide to take a deliberately belligerent stance towards that subset of users.

greg carmack - mvp 2010-2020 say:

The BIOS is not an issue as Windows 10 can run just as well on Legacy BIOS as on the new UEFI FIrmware.  Was there an error or anything else that made you think this is UEFI related? 

More likely the Upgrade is choking on corruption or bloat in the present install which is the biggest cause of failures.  The secondary drive may also be interfering so unplug it and all other peripherals. 

You should have been presented by Get WIndows 10 app with a Compatibility Report informing you of any possible issues before it ran the Upgrade.  If confident your hardware is modern enough to meet  the Windows 10 Specifications  including unsupported feature deprecation, then you can bypass Get Windows 10 app in the System Tray to Install  Windows 10 using the media creation tool - Windows Help.  These steps are working to overcome almost all issues with GWX app in the System Tray,  and/or Upgrade choking on corruption or bloated factory installs:  

Install the tool from an Administrative account, tick the checkbox in first option to automate the process, or select "For another PC" to create bootable flash media or download an ISO to burn to DVD with Windows Image Burner.  Now you have media for reinstalls or needed repairs.  But to run the Upgrade the media must be opened in Windows 7  then right click Setup file to Run as Administrator. 

 If asked do not insert a Product Key at any time, just click Skip or Next. 

If you want to really maximize chances for a successful Upgrade, then choose to Save Nothing so the Upgrade has much less to choke on.  Make sure your files are backed up separately and you have your program installers and any Product Keys for programs like Office

To improve the Upgrade's chances turn off everything at Startup:

In Windows 7, type msconfig in Start Search Box, uncheck everything on Startup and Services tabs, but only after ticking the box in Services to “Hide all MS Services”.  Then click Apply, restart PC when prompted.

Uninstall any anti-virus except Defender.  Bloated AV's like Norton, MacAfee, AVG and Avast need to have special Uninstallers (removal tools) for common antivirus software - ESET Knowledgebase.

Stick with built-in Defender in Windows 10 for best performance and adequate protection.  If you get infected add the inexpensive Real Time protection from the best-in-class free malware scanner Malwarebytes. No one I’ve had do this has ever been infected again.  I'd also periodically scan with free Malwarebytes or AdwCleaner  scanners.

It’s highly recommended to Create a backup System Image of your existing OS before upgrading, which can be recovered using the booted Windows installation media, Repair CD or Recovery flash drivein 20 minutes.   Type Backup in Start Search box, in Backup and Recovery Center choose Create a System Image.  You can also use a more flexible imaging program like free Macrium Imaging which allows recovery to any partition position on the hard drive.

Immediately after install, type Update in Start Search box to Check for Updates, expedite these through requested restart, then check again until no more are offered. 

Next right click Start button to open Device Manager, import any drivers still missing from the PC or Device's Support Downloads webpage.  How to: Install and Update drivers in Windows 10 - Microsoft Community

Feel free to ask back any questions and keep us posted on how it goes.

greg carmack - mvp 2010-2020 say:

The problem is that you have converted your hard drive to Dynamic which is only meant to span a single partition across two or more hard drives. Are there two hard drives installed?

 If you wish to keep the install there is a method to convert the hard drive back to Basic non-destructively.  You will need Partition Wizard 4.2 which is the last version which offers as a free feature to convert a Dynamic Disk to Basic.

Back up your files then follow  the steps in Option One of this illustrated tutorial:

http://www.sevenforums.com/tutorials/26829-convert-dynamic-disk-basic-disk.html?ltr=C


Once the disc is converted, for it to be correct it needs to have the System label on the 1 GB Active partition which is apparently booting the OS now. If it does not show the System label after a reboot then boot into your windows disc or Repair CD to run a few Startup Repairs.

  If you do not have the Windows 7 disk then make now your System Repair disk  in Control Panel>Backup Center as it may need a repair to start after conversion depending upon how much other corruption there is.


Then post back another Disk Management screenshot.
