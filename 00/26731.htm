suzanne s. barnhill say:

You'll note that when you select More Colors under Shading, the Borders button becomes selected, and the black color Word is showing you is the border color. So don't try to do it this way.

 

Instead, select the Home tab, click the arrow beside the Borders button, and select Borders and Shading. On the Shading tab of the dialog, make sure that the selection for "Apply to" is "Cell" rather than "Text" or "Paragraph," and then you can go to click the arrow by the box under "Fill," select "More Colors," and see the RGB numbers on the Custom tab of the Colors dialog.

my .doc and .docx documents go grey - Microsoft Community

pam schultz say:

my .doc and .docx documents go grey

large sections of the word .doc and .docx on my mac goes grey while I am typing.
john mcghie say:
OK, now THAT ios enough information :-)  Sadly, what it tells me is "I do not know what is wrong, I have never heard of this issue before."

Usually, large slabs of grey shading indicate a cross-reference that has gone wrong, sucking in a large section of text including its target (sometimes, several chapters!).

To see if it is that, when it next happens, right-click the shaded area and see if you are offered the option to "toggle field codes".  If you are not, it's not a broken cross-reference.  If it is, the shaded text will vanish!  Don't panic.  It hasn't actually gone anywhere...

Scroll back up the document until you find the cross-reference, which will now read { REF "BookmarkName" }.  That's the code that actually generates the cross reference.

Right-click that code, and choose Toggle Field Codes again.  The text will reappear.

NOW you have a problem :-)  The text that reappears does not exist at that location.  It's being copied in from a different location in the document.  You must now decide which is the correct place for it, and move it to where you want it.

I would use Cut to remove it, and Edit>Paste Special>Unformatted Text to paste it.  That way you will remove any broken field codes that are in the text.  You will need to re-format the pasted text.

Everyone gets broken cross-references when editing documents that contain cross-references, unless they are very careful.  To ve careful, you need to go to Word>Preferences>View and turn "Field Shading" to "Always" so you can see where they are, and thus make sure you do not edit closer than one character to a cross-reference.

There are other things that might cause this indication, although I have never seen them, so these are just wild guesses:

1)  Running out of memory.  Log out of your computer at the end of work each day, and log back in the next day.  This cleans up the memory.  Shut the computer down and re-start it once a week.  This cleans up the OS.  If it's memory problems, this will clear it.

2)  Strange settings in the display of Tracked Changes.  Make a copy of the document and Accept all changes".  If that clears it, the document has too many tracked changes in it, and is in danger of failing completely.  I recommend Compare Documents instead of Track Changes when working on long and complex documents.  It achieves the same thing, but is most unlikely to cause document problems.

Hope this helps

Word 2013 and Greek keyboard - Microsoft Community

merica say:

Word 2013 and Greek keyboard

I have a document that I need to add Greek letters to. Like the Greek letter for alpha.  Does this font come preinstalled in Word 2013? If so, how do I switch to it.  If not can you tell me what I need to do to get the Greek alphabet in Word.  I am a basic Word user, nothing real fancy.
jay freedman say:

There are at least two ways to insert just a few Greek letters using only what you already have.

  • In your regular font, type the English (Roman) letters that correspond to the desired Greek letters, and then change the font to Symbol. The Symbol font is standard in all Windows versions. Here are two lines that were typed identically, but the lower line has Symbol font applied to it:

  • On the Insert tab, click Symbol and then click More Symbols. In the dialog, make sure the "from" box is set to "Unicode", which makes the Subset box visible; then set the Subset box to "Greek and Coptic". Click on the letter you want and then click the Insert button. This works for most fonts, although some don't have a "Greek and Coptic" subset.

    After you use a letter this way, it will show up in the list of recently used characters in the Symbol button's dropdown.

If you have to type whole sentences or paragraphs of Greek text, then you would need to enable a Greek keyboard and switch between languages.
