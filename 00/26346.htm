charles kenyon say:

Before anyone else gets too confused, the Agenda Wizard and other wizards are no longer distributed with Word or available from MS online AFAIK. Like you, Haime, I am a big fan of the Agenda Wizard which I think was introduced with Word 97. The file people should be looking for is "Agenda.wiz" or "Agenda Wizard.wiz." It works fine in Word 2010 and 2013 AFAIK.

That said, a wizard is just a renamed template with a well organized UserForm or userforms and supporting code. It has the filename extension of .wiz. If you want to try modifying it, you will need to find the file and be comfortable with vba.

Copy the file Agenda Wizard.wiz giving the copy a name like AgendaWizard2.doc. That's right, change the file name. Then open it and use Alt+F11 to open the VBA editor and search for the word notetaker in the code. The screenshot below shows one such search. Where it says:

As String = "Note taker" try changing it to

As String = "Recording Secretary"

Then close the vba editor and save the document. Close Word. From Windows, rename it AgendaWizard.wiz. Then double-click on it and see if that works. If it does, great. If not you'll have to search through the code some more.

The thing is, renaming it .doc from .wiz is what lets you open it without running the wizard.

Note, if you do not use the features of the wizard, you could simply use it once to create an Agenda and minute forms, replace some areas with content controls and save the result as a template.

"Writing Word Macros" which is still good reading.

#1 - What version of Outlook is being used?

From the description, sounds like the issue is related to an Outlook data file being configured as the "default data file". For OL versions '2010 and earlier - there are 2 PST data files involved, one for the email folders and the other for contact/calendar data (this one would be set as the default data file).

For OL'2013/'2016 there is one OST file used which if it is also set to be the default data file, you won't be able to remove it if there are other email accounts involved.
