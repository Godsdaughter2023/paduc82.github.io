To uninstall Silverlight 4 and regress to Silverlight 3 do the following:

  1. Uninstall Silverlight 4 by going to Programs and Features in Control Panel.  Right click on the Microsoft Silverlight line item and choose uninstall on the right click menu (only option.)  Restart if required; I don't believe it was but you may want to do it just to be safe.
  2. To install Silverlight 3 I went to the Netflix website and tried to watch an instant play movie.  Netflix indicates you need to install Silverlight, follow the prompts.  Netflix will install Silverlight 3 (at least it did before, maybe they have updated to Silverlight 4.)  If it does install Silverlight 4 uninstall it again; here is a link I found for Silverlight 3 on Green Button http://www.oldapps.com/silverlight.php?old silverlight=3
  3. Then, manually check for Windows updates.  Right click on the Silverlight 4 update and select Hide Update.  This will prevent the update from installing even if you have automatic updates set.
