For UEFI Install the installation media needs to be formatted correctly using Media Creation Tool and booted as a UEFI Device only.  Then clear all partitions in case the previous install was Legacy so it will assure a reset of the formatting to GPT instead of MBR. 

During UEFI install it will automatically set Windows Boot Manager first to boot, not the hard drive.  For a Legacy install, the hard drive is set first to boot. 

To change from one to the other often requires changing settings in the BIOS, or at least booting as a UEFI device, then deleting all partitions which for UEFI must be reset to GPT formatting by the installer, while a Legacy install requires MBR. 

Another reason can be that the correct hard drive is not set to boot first in BIOS because another hard drive is still attached and set first. 

So perhaps best is to reset the BIOS to defaults with only the HD in question attached and set first to boot, Save and restart to see what happens. 

If these are all set correctly then the next thing is to test the hard drive.  If your onboard diagnostics already did this then that's normally sufficient but you can cross check with the maker's HD Diagnostic extended CD scan.
