jay freedman say:
The settings are stored in the registry, but unfortunately in a binary "blob" in the Settings value under the key HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\Word\Data. That blob isn't publicly documented beyond what's in http://www.word.mvps.org/FAQs/Customization/DataKeySettings.htm.

It's safer to put the following macro into a template that you store in each user's Startup folder at %AppData%\Microsoft\Word\STARTUP so it's a global template:

Sub AutoExec()
    With Options
        .AutoFormatAsYouTypeApplyBulletedLists = False
        .AutoFormatAsYouTypeApplyBorders = False
        .AutoFormatAsYouTypeApplyNumberedLists = False
        .AutoFormatAsYouTypeApplyTables = False
       
        .AutoFormatApplyHeadings = False
        .AutoFormatApplyLists = False
        .AutoFormatApplyBulletedLists = False
        .AutoFormatApplyOtherParas = False
       
        .AutoFormatReplaceOrdinals = False
        .AutoFormatReplaceFractions = False
        .AutoFormatReplaceSymbols = False
        .AutoFormatReplacePlainTextEmphasis = False
    End With
End Sub

See http://www.gmayor.com/installing_macro.htm if needed.
