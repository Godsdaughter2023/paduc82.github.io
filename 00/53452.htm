lemp say:

Every computer that comes with Windows pre-installed (as this one undoubtedly did), must include some means to reinstall Windows.  In the old days the manufacturer would include an actual installation CD.

It's much more common these days for computer manufacturers to include a "hidden partition" on the hard drive.  When accessed by the correct key combination, this will restore the computer to the way it was when it left the factory.  For Toshiba computers that came with Windows 7, you do this by pressing and holding down the zero (0) key right after power on.  See

https://crashcloud.com/how-to-reset-a-toshiba-laptop-back-to-factory-settings/

https://support.toshiba.com/support/viewContentDetail?contentId=2737864

This is NOT the same as "repairing" the computer or "restoring" it to some specified restore point (it sounds as if that's what you did).  The drawback to this is that you would have uninstall all of the "bloatware" that originally came with the computer, including any free trial software (which will have expired), and then install all of the necessary Windows updates to get things current.

There are a couple of concerns about this that may apply to your situation.

If you saw something that suggested the computer was going to be reset to a point in 2000 or thereabouts, that's well before Windows 7 was released.  In fact, Windows XP wasn't released until October 2001.  It's possible that the computer originally came with Windows 2000 or even with Windows 98 SE.  In this case, (a) the recovery method described above may not be applicable and (b) you may not want to recover to Windows 2000.

On the other hand, the screenshot you included looks a lot like Windows 7. 

Before you do anything else, please answer the following questions.  Your answers will determine what your options are at this point.

1.  What is the Toshiba model-part number and/or serial number?  See https://support.toshiba.com/sscontent?contentId=4007069

2.  What version of Windows is currently installed?  If the keyboard has a "Windows Logo" key, press that and the "Pause/Break" key simultaneously.  Otherwise, right-click "Computer" (or "My Computer") and select "Properties.  If you have some version of Windows 7, look at the very bottom of the system information screen to see the "Product ID."  This will have the form xxxxx-xxx-xxxxxxx-xxxxx.   Please report the 3 characters in the second group.

3.  Is there a COA sticker on the bottom of the laptop (possibly inside the battery compartment)?  It will look like this:

If you have one of these stickers, what version of Windows is it for?  Do NOT post the "Product Key" here.

 If only a small number of duplicate entries exist, you can delete the invalid ones manually by right-clicking the invalid entry, and then clicking Delete.

 

   If you have a large number of duplicate entries in your library (or if your entire library is duplicated), it might be best to create a new library.

 1.    Exit Windows Media Player.

 2.    For Windows 7: Click Start, click Run, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.

 3.    Select all the files in the folder, and then click Delete on the File menu.

Note: You do not have to delete the folders that are in this folder.

 4.    Restart Windows Media Player.

 Note: Windows Media Player automatically rebuilds the database.

 

 

 Method 2: If this does not resolve the problem, clear the Windows Media Player database cache files. To do this, follow these steps:

 

1.    Exit Windows Media Player.

 2.   If you are running Windows 7, click Start, click Run, type %LOCALAPPDATA%\Microsoft, and then click OK.

 3.    Select the Media Player folder, and then click Delete on the File menu.

 4.    Restart Windows Media Player.

 

Note: Windows Media Player automatically rebuilds the database.

Windows Media Player Library: frequently asked questions

 

 http://windows.microsoft.com/en-us/windows7/Windows-Media-Player-library-frequently-asked-questions

 

Method 3: How can I prevent duplicate or invalid entries from being added to my library when I play music files?

 

When you move digital media files on your computer, the file name and file path information remain unchanged in your library. Then when you select a file to play from its new location, a new entry is created in your library if the option to automatically add it to your library when it is played is selected. As a result of this, your library may quickly contain a large number of duplicate or invalid entries.

 

To prevent music files from being added automatically to your library

 

 

1. In Windows Media Player, on the Tools menu, click Options.

2. On the Player tab, clear the Add music files to library when played check box.

   Now, when you play any music files on your computer or from the Internet, the file will not be added automatically to your library.
