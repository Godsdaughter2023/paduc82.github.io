It is really difficult to see for aging sight.

Thanks

jay freedman say:
If you're referring to the blinking insertion point, and not the I-beam mouse cursor, then unfortunately the answer is that it can't be enlarged. All you can do is set the zoom factor larger -- the insertion point marker will grow along with the text.

If you do mean the mouse cursor, that can be adjusted. In the Start menu, type mouse in the search box and click the item Change how the mouse pointer looks. In the dialog, click the Text select item, click the Browse button, and choose either beam_rl.cur. Click the Open button, and click the OK button in the Mouse Properties dialog.
