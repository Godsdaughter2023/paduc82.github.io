Macro to insert row when cell value changes - Microsoft Community

ilplowboy say:

Macro to insert row when cell value changes

Hi all - I'm looking for a macro that will look at the values in column C and whenever the value changes, it will insert a row above the new value. For example if the values in column C are:

               Col C

Row 1     1

Row 2     1

Row 3     1

Row 4     1

Row 5     2

 

Then the macro will insert a new row between rows 4 and 5.     Thanks in advance for any help you can provide.

jllatham say:

Sub InsertRows()
  Dim lastRow As Long
  Dim rowPtr As Long
 
  lastRow = Range("C" & Rows.Count).End(xlUp).Row
  For rowPtr = lastRow To 2 Step -1
    If Not IsEmpty(Range("C" & rowPtr)) Then
      If Range("C" & rowPtr) <> Range("C" & rowPtr - 1) Then
        Range("C" & rowPtr).EntireRow.Insert
      End If
    End If
  Next
End Sub

 

Be sure you have the sheet with the numbers in column C selected when you run the macro.  Test with a copy of your workbook just in case :).

Siri,

Sure. Project has several extra fields (e.g. Text, Date, Number, etc.). There are 20 extra number fields (Number1-Number20). In order to move all the emails to a single inbox, you will have to configure all the 4 accounts as POP3 account.

Assuming all the accounts are configured as POP3 account, then let us follow the steps mentioned below.

Open Outlook, File---Account settings--- Account Settings dialog; File-> Account Settings-> Account Settings…

Select the POP3 account for which you want to change the delivery location.

Press the “Change Folder” button at the bottom of the dialog.

Select the folder to which you want your new messages delivered for that account.

This can be an already existing folder such as the Inbox of your main mailbox or a separate (sub) folder.

You can for instance create a folder structure such as; Inbox -Account 1,-Account 2,-Account 3
