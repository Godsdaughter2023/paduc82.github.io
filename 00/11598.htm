rohn007 say:
What is advisable / preferable for YOU (or anyone) depends entirely on who you are working and sharing files with.

Like Suzanne, I have 2003,2007,2010 (and 2013 installed).  I split my time almost evenly between 2003 and 2010 (I really dislike the mess they made of the 2013 UI), slowly moving to 2010 (even though I STILL do massively prefer the MENU). 

For simple, short documents I still fall back to 2003 because I'm a manual typewriter trained touch typist. I'm a "keyboard" shortcut fan with the menu commands learned to the level of muscle memory. I suspect a moderators could do a query on the forum database to get a report on the representation of the various versions in questions here <hint hint!> . That would be interesting, although it wouldn't necessarily reflect real world use.

For my own use I prefer to use the "new" x-file formats, even in 2003, for 2 reasons. 

First, the XML files include compression in the file structure.   For Word and Excel files that are mostly text, this can lead to significant file compression on larger files.

Second, since they are XML formatted, they rely much more on text to control the underlying file formatting than the machine only readable coding in the DOC files. The binary DOC formats were too prone to corruption. 

For example the x-file corruption problem "missing end tag". While it is actually an artifact of the conversion to XML it also gives an expert the chance of opening up the file and FIXING it!  When a binary document went corrupt there was little chance of fixing it. Actually, that missing tag problem is the only X-file corruption that I've heard of.

What it is

Bing Ads runs your ads indefinitely unless you set end dates for specific ad groups. Ad Group Settings>Advanced settings>Schedule is for designating an automated start and stop date for your ad groups.

About MSDN or TechNet Product Keys

They are genuine Microsoft product keys for retail licenses, but not for Retail sale. They are intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. The main purpose is for evaluation purposes. It is for one person only to use and no one else. Unfortunately, regardless of the licensing terms, persons still abuse the program, either giveaway product keys or resell it on auction sites. 

What probably happened is, you bought an MSDN licensed key, which carries up to 10 activations unlike full packaged retail licenses which only carry 1 activation. The person who sold it to you probably sold it to 10 other persons. Somewhere along the way, one of those persons might have installed it on a second system, activated it, because it went past 10 activation threshold, Microsoft detected it that it was being abused and blocked the key from further use.
