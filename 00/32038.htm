How can I create a ICS file in Windows Live Calendar?

If you want to create an ICS file containing all the events in your calendar, use the Share option to Get a link. The middle one will generate an ICS file in the format

webcals://sharing.calendar.live.com/calendar/private/{GUID1}/{GUID2}/cid-{CID}/calendar.ics

Changing the protocol prefix from webcals:// to https:// will enable the recipient to download the file via his browser.

I know of no simple way to create an ICS file for an individual event, but it is possible.
