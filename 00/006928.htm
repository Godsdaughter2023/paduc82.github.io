Adding premade sources to Word's citation manager - Microsoft Community

madisonruiz say:

Adding premade sources to Word's citation manager

I am trying to add sources to my references page and need to know if I can just copy and paste a source already into the manager.

My school's database already cites the source for you like this:

Dunnett, R. (2016). How to... be good at meetings. Director, 69(5), 58-59.

I want to add that to Microsoft Word's citation manager so that it can auto update my references page. Is that possible? Or do I need to fill out each section of the manager manually?

macropod say:

Microsoft hasn't provided any tools for Word to import reference data from external sources. As Word stores references in an xml structure, you'd have to convert them to that. As xml code, a typical reference might look like:

<b:Source xmlns:b="http://schemas.openxmlformats.org/officeDocument/2006/bibliography"><b:Tag>The95</b:Tag><b:SourceType>Book</b:SourceType><b:Guid>{A9F88CD0-3296-4707-8F5D-12EAE6681D82}</b:Guid><b:LCID>0</b:LCID><b:Title>The Concise Oxford Dictionary</b:Title><b:Year>1995</b:Year><b:City>Oxford</b:City><b:Publisher>Clarendon Press</b:Publisher><b:RefOrder>1</b:RefOrder><b:Edition>Ninth</b:Edition></b:Source>


and, using APA 6th ed. be displayed as:

The Concise Oxford Dictionary (Ninth ed.). (1995). Oxford: Clarendon Press.

AFAIK, you would have to convert the reference to the required format (the Guid might be a challenge, but I'm not sure whether you have to supply it), which could be done programmatically PROVIDED the raw data use a consistent layout. And therein lies the problem. With the above example, the edition information may or may not be present, so you'd have a lot of programming work to do to handle such variability. Once you've done that, the xml data could be imported using VBA code like:

Dim i As Long
'In case the source is already present
On Error Resume Next
With Selection
  For i = 0 To UBound(Split(.Range.Text, vbCr))
    Application.Bibliography.Sources.Add Split(.Range.Text, vbCr)(i)
    ActiveDocument.Bibliography.Sources.Add Split(.Range.Text, vbCr)(i)
  Next
End With

rap10 say: My ActiveSync does not have the same menu structure as you describe.  But I followed your general lead and did find the sequence that allowed me to change the setting to download and display "All" emails in my Inbox.  The sequence is:  open ActiveSync; Menu; Options; scroll down to Email; Settings; the top settings option is "Download the past:" wherein there are are 6 settings options ranging from "1 day" to "All"; select a new option by scrolling left or right; enter "Done" twice; hit Sync to sync the newly selected option with the server; exit ActiveSync; go to the email Inbox, which should now reflect the new setting.  Thank you for heading me in the right direction.
