http://support.microsoft.com/kb/947215

 

If running the fix it tool does not help, you may try to run the chkdsk utility from command prompt and check if it helps.To run the chkdsk utility follow the steps below:

 

  1. Click on Start.
  2. In the start search box type CMD.
  3. In the list right click on CMD and click on “Run as administrator”
  4. In the command prompt, type the following command: chkdsk /f /r and press enter

 

The check disk will run on the restart. Once you are back to the desktop verify if you can access your account on the computer.

 

Hope this helps. Let us know the results.
