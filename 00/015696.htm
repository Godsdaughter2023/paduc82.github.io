Creating a 'Continued' Header with a Detail Group that spans multiple pages

I have a tabular report with a Group Header and Detail section.  In the Group Header section, I have labels for each of the fields in the Detail section.  I also have a "section title" label above the field labels that says "Status Report".

If the data in the Detail section should print on more than one page, I want the "section title" label on these subsequent pages to say "Status Report (continued)".

To accomplish this, I replaced the "section title" label in the Group Header section with a textbox and set its Control Source to:

    =IIf(IsContinuedSection=True,"Status Report (continued)","Status Report")

In the OnPrint event procedure of the Detail section, I added the following code:

    Private Sub Detail_Print(Cancel As Integer, PrintCount As Integer)
        Dim IsContinuedSection As Boolean
        If Me.Section(acDetail).HasContinued Then
            IsContinuedSection = True
        End If
   End Sub

Unfortunately, Access treats IsContinuedSection as a parameter when I preview the report (which, of course, is not the result I want).

Can someone please advise me on what I need to change in either the Control Source or VBA code to achieve the desired result?  Thanks in advance for any assistance.

hansv mvp say:

You can't refer to VBA variables in expressions on a form, and the HasContinued property is only True if one particular instance of the section continues on the next page, not if a new record is displayed on the next page.

Try the following:

Place a text box in the group header section, with the following properties:

Visible: No
Control Source: =1
Running Sum: Over Group
Name: txtCount

 

Use the following code in the On Format event of the group header:

 

Private Sub GroupHeader0_Format(Cancel As Integer, FormatCount As Integer)
    Static n As Long
    If n = Me.txtCount Then
        Me.lblTitle.Caption = "Status Report (continued)"
    Else
        Me.lblTitle.Caption = "Status Report"
    End If
    n = Me.txtCount
End Sub

 

where GroupHeader0 is the name of the group header section, and lblTitle is the name of the section title label (you can use a label again, instead of a text box).

anna bon say:

Hello,

The Junk Email Filter is designed to keep junk email messages, also known as spam, from cluttering your Inbox. By default, the Junk Email Filter is turned on, and the protection level is set to Low. This is the setting designed to catch the most obvious spam. You can adjust the sensitivity of the Junk Email Filter. Messages detected by the Junk Email Filter are moved to the Junk E-mail folder.

For the instructions on how to disable the junk filter, please follow the steps below:

  1. In Mail, on the Home tab, in the Delete group, click Junk, and then click Junk E-mail Options.

  2. On the Options tab, select No Automatic Filtering. Although this turns off the automatic Junk Email Filter, messages are still evaluated by using the domain names and email addresses in your Blocked Senders List.

You also have an option to delete messages instead of moving them to the junk email folder. Here's how:

  1. In Mail, on the Home tab, in the Delete group, click Junk, and then click Junk E-mail Options.

  2. On the Options tab, select the Permanently delete suspected junk e-mail instead of moving it to the Junk E-mail folder check box.
