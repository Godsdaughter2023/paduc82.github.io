If your computer includes a CD or DVD recorder, you can copy files to a writeable disc. This process is called burning a disc. By default, Windows burns discs in the Live File System format, but you can also choose to burn discs in the Mastered format.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

http://windows.microsoft.com/en-us/windows-vista/Disc-burning-frequently-asked-questions

Why are there two different ways to format a disc?

You can choose between the Live File System and Mastered formats when creating a new recordable disc because each of these formats is used for a different purpose. The default choice is Live File System because it is a more convenient way to create a disc. When you use a Live File System disc, files are copied to the disc right away. When you use the Mastered format, files are stored in a "staging area" until you decide to burn the disc.

When you use the Live File System format with rewriteable discs such as CD-RW and DVD-RW, you can also erase unwanted files from a disc to recover space, which is not possible with Mastered discs. On the other hand, Mastered discs are more compatible with older computers, previous versions of Windows, and other devices such as CD and DVD players. For more information, see Which CD or DVD format should I use?

Microsoft has an arrangement with an organization called Directly, where Directly solicits people who post here and offers them payment in return for answering questions. These are the 'Independent Advisors' you frequently see here. You don't need to have any special qualifications to become an Independent Advisor, you only have to be willing to answer questions. So you are quite right: many of them really don't know much about software, although some certainly do.

As an Independent Advisor, your payment depends on how many questions you answer, with a bonus if the OP (original poster of a question) marks your reply as the correct answer, or if your reply is upvoted. That's why Independent Advisors frequently ask you to upvote their reply.

So the name of the game if you're an Independent Advisor is to say whatever it takes to persuade an OP that your reply is correct, or to get your reply upvoted, whether or not your reply actually is correct or in the OP's best interests.

This leads to ridiculous replies where Independent Advisors post whatever generic troubleshooting they can think of that's even remotely related to an OP's question, hoping that something will work, all in an effort to have their replies upvoted or marked as answers. Worse than that, however, is when an OP wants to do something that's not correct or not in their own best interest, and the Independent Advisor is glad to help, because correcting the OP doesn't earn any points with Directly.
