bob jones aka: cybertaz say:

There is no absolute global answer... Generally: Most Preferences are persistent [such as changes made in Word> Preferences & the display of window elements, including the Ruler]. Settings [made on the Ribbon or in dialogs] are not, unless there is a Default button in the dialog window.

You can determine your update level by going to Word> About Word in the main Menu.

dempsey444 say: Thanks for the quick response.  I haven't been able to make the ruler or other commenting tools persistent.  Is there an easy way to ensure that they're defaulted?

bob jones aka: cybertaz say:

The default behavior of the controls on the Review tab cannot be changed, with Language being the one exception. That one button opens the Language dialog window which has a Default button that allows you to set the default language for new documents.

The Horizontal Ruler should continue to display once it is selected [checked] in the View menu. The Vertical Ruler only displays if it is checked in Word> Preferences - View, as well. [Note, however, that no Rulers display in Outline or Web Layout views.]

       If .UserInitials <> "JGM" Then .UserInitials = "JGM"

       If .UserAddress <> "" Then .UserAddress = ""

       If .RecentFiles.Maximum < 25 Then .RecentFiles.Maximum = 25

       If .DisplayScreenTips <> True Then .DisplayScreenTips = True

        With .View

            If .ShowPicturePlaceHolders <> False Then .ShowPicturePlaceHolders = False

            If .ShowFieldCodes <> False Then .ShowFieldCodes = False

            If .ShowParagraphs <> False Then .ShowParagraphs = False

            If .ShowHyphens <> False Then .ShowHyphens = False

            If .ShowHiddenText <> False Then .ShowHiddenText = False

            If .ShowAll <> True Then .ShowAll = True

            If .ShowDrawings <> True Then .ShowDrawings = True

            If .ShowObjectAnchors <> True Then .ShowObjectAnchors = True

            If .ShowTextBoundaries <> True Then .ShowTextBoundaries = True

            If .ShowHighlight <> True Then .ShowHighlight = True

            If .DisplayPageBoundaries <> True Then .DisplayPageBoundaries = True

       If .AutoFormatAsYouTypeApplyNumberedLists <> False Then .AutoFormatAsYouTypeApplyNumberedLists = False

Nom Daytime

SELECT DISTINCT MergeDataTest.Nom, ConcatRelated("Daytime","MergeDataTest","Nom = " & "Nom") AS All_Daytime

Change the SQL to

To remove MSE, try running the command provided by Jsssssssss in the answer here:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_start/mse-still-installed-in-windows-10/00ae7a4f-00c0-4c92-9847-c65f512c60eb

 

"C:\Program Files\Microsoft Security Client\setup.exe" /x /disableoslimit
