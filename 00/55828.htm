¡firedog say:
Clear text authentication is used (is this Authenticated POP?) 

A bit more detail for you:

... types of authentication to the POP3 server:

In this type of authentication, the logon credentials for the POP3 account are sent as unencrypted clear text to the e-mail server for the POP3 mailbox.

SPA uses encrypted authentication credentials to log on to the e-mail server for the POP3 mailbox each time the POP3 Connector downloads e-mail. Some ISPs require that you use SPA.

APOP uses MD5 password encryption to provide enhanced security during message authentication.

(Extracted from http://technet.microsoft.com/en-us/library/cc546479(v=ws.10).aspx.) The unencrypted clear text is in fact Base 64 encoded, so you can't immediately read the password if it's intercepted.

ken blake - microsoft mvp say:
I am new to windows 7.  I don't know where to go after I scanned the documents into Liibrary>Documents>Scanned.  It needs constantly updated and printed. It is about 13 pages of personal medical history.  I've saved the entire document under the same main title but separated it by page numbers in or der to call out one pagae to update or print. any better ideas? I have never been a clerk or anything lijke that.  Lastly, I would like to add page numbers to the document but I dont know how to do that either.


 

Bear in mind that when you scan a document, you end up with a *picture* of the document, not with a text file containing the text on the original. That means that you can *not* do the editing you want to do on it. That's true in Windows 7 and in any other operating system.

 

You can convert that picture of the document to a text file if you use a program of the Type called OCR (Optical Character Recognition). Many scanners come with starter versions of such software, but I don't know whether yours does or not. If you want to buy such a program, the two most highly-respected are Abbeyy Fine Reader (which I've personally never used) and OmniPage (which I personally use and like a lot).

 

When you use such a program on your scanned document, you end up with a text file that you can edit in Microsoft Word, WordPerfect, or whatever word processor you use.
