halima s say:

Hi PNapa21,

 

 

The prompt to download the HP Office jet 6500 driver at every restart is because you have not installed the right printer driver for the printer that is connected to the computer.

Try the below steps if you want to disable the driver download prompt at every restart.

I would suggest that you check if the driver download prompt is from Windows update by checking if there is any driver available for the printer in the windows update.

If you find an update available in the Windows update the only way to stop it would be to hide that particular update.

a)   Open Windows Update by clicking the  Start button. In the search box, type  Update, and then, in the list of results, click  Windows Update.

b)   Do one of the following:

1.    Click the link that tells you important updates are available if you have important updates to hide.

2.    Click the link that tells you optional updates are available if you have optional updates to hide.

c)    Select any update that you want to hide, right-click it, and then click  Hide update. Do this for each update you want to hide.

d)   Click OK.

 

If there is no update available for printer in the Windows update then I would suggest that you download the printer driver when prompted and check if the printer works properly.

After installing the printer driver if the printer is not working then I would suggest that you perform system restore to the point before the printer driver installation.

System Restore: frequently asked questions

http://windows.microsoft.com/en-US/windows7/System-Restore-frequently-asked-questions

 

Hope this helps!

arjay dom say:

Hi Tina,

Whenever you connect to a new WiFi network, the network will be added and stored in the list of profiles by Windows.

To delete old/previous network connection in Windows 10, follow the steps below:

1. Press Windows + I to open the Settings app.

2. Select Network & Internet.

3. Click on Wi-Fi on the left panel then you'll see a link for Manage Wi-Fi settings.

4. Click on Manage Wi-Fi settings and it will open a new window.

5. Under Manage known networks, select the network that you want to delete or forget then click on Forget.
