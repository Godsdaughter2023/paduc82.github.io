jeeped say:
You should base the conditional formatting rule on a formulas like,

=$X2=123

... and make sure that the Conditional Formatting Rules Manager shows an Applies to: range of something like,

I can easily save the worksheet as a web page and open it in a browser but the auto filter functionality is gone.

ed ferrero say:

A pivot table should work for you if you have a simple list with an autofilter. Try following these steps...

  • Select a cell in the original list you wish to filter
  • Insert -> PivotTable, choose new worksheet
  • Dreag each and every field to the Row Labels area
  • PivotTable Tools -> Options -> Totals & Filters - uncheck row and column grand totals
  • PivotTable Tools -> Options -> Display - check the Classic PivotTable Layout
  • Double-click each column heading in the PivotTable
  • Set Subtotals to None, and, in Layout & Print, check the Repeat Item labels option

You now have a PivotTable that looks just like the initial list (except that blank cells are replaced by the word (blank))

Select a cell in the PivotTable and, in PivotTable Tools, insert a slicer for each field that needs to be filtered.

That's it. You now have a filtered list that will work in a web page.
