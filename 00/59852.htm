2. Click on System > Display.

3. Under 'Change the size of the text, apps, and other items', and 'Resolution', make sure that the recommended options are selected.

Restart the system once  and check.

Method 2: Uninstall the display adapter from Device Manager.

1. Press 'Windows key + R' to open 'Run command'.

2. Type 'Devmgmt.msc' without quotes, and press enter.

3. Expand 'Display adapter'.

4. Right click on the device showing below, and click on 'Uninstall device'.

5. Restart the system once, and check.

If the issue persists, I would suggest you to download the display adapter driver from the system manufacturer site and install it.

SFC does more than fix corrupted Windows system files.

SFC (System File Check) scans the integrity of all protected operating system files and replaces incorrect, corrupted, changed, or damaged versions with the correct versions where possible.  When it cannot repair the damage DISM should be run to fix as many errors as possible.

Because I have used SFC regularly I have found and fixed problems I did not even know were present. I think Windows updates and app updates can cause incorrect, corrupted, changed, or damaged system files.

I would definitely never compare SFC to a "registry cleaner". SFC is a reliable repair tool and "registry cleaners" almost always damage your registry. I would never ever use a registry cleaner. By comparing these two items you could very well frighten an inexperienced computer user. SFC is a very safe tool.
