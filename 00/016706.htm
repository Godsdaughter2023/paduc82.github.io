For what you're doing, ignore the Insert | Text | Signature Line feature in Word 2010. Instead, select your scanned signature and press Alt+F3 to open the Create New Building Block dialog. graham mayor say:

The following macro will do what you want, (as you describe it) i.e.
1. It pastes the text from the clipboard at teh cursor position
2. It inserts an empty paragraph at the end (the equivalent of pressing enter)
3. It replaces the character ¶ in the pasted text with an a paragraph break
4. It formats the resulting pasted text as justified
Note oRng is the inserted text, you can use the macro to perform any valid operation on oRng

Add it to a keyboard shortcut or QAT/Ribbon button

Sub MyPaste()
Dim oRng As Range
Set oRng = Selection.Range
With oRng
    .Paste
    .InsertParagraphAfter
    .Text = Replace(oRng.Text, Chr(182), Chr(13))
    .ParagraphFormat.Alignment = wdAlignParagraphJustify
End With
End Sub

Make a picture to 100% transparency from solid color by VBA - Microsoft Community

kkleung say:

Make a picture to 100% transparency from solid color by VBA code?

How can VBA code make a picture to 100% transparency from solid color?
gary's student say:

You need to place the picture on the sheet in a way that allows transparency to be applied:

 

Sub cutaneous()
Dim FileSpec As String
ActiveSheet.Shapes.AddShape(msoShapeRectangle, 312.75, 176.25, 266.25, 129.75).Select
Selection.Name = "Sargon"
FileSpec = Application.GetOpenFilename()
With Selection.ShapeRange.Line
        .Weight = 0.75
        .DashStyle = msoLineSolid
        .Style = msoLineSingle
        .Transparency = 0#
        .Visible = msoFalse
        .ForeColor.SchemeColor = 64
        .BackColor.RGB = RGB(255, 255, 255)
End With

With Selection.ShapeRange.Fill
    .Visible = msoTrue
    .ForeColor.RGB = RGB(255, 255, 255)
    .BackColor.RGB = RGB(255, 255, 255)
    .UserPicture FileSpec
    .Transparency = 0.5
End With

End Sub

 

You can then modify the transparency using VBA, for example:

 

Sub FadeInFadeOut()
Dim r As Range
Set r = Selection
ActiveSheet.Shapes("Sargon").Select
Selection.ShapeRange.Fill.Transparency = 1
For i = 1 To 100
    Selection.ShapeRange.Fill.Transparency = 1 - i / 100
    DoEvents
Next

For i = 1 To 100
    Selection.ShapeRange.Fill.Transparency = i / 100
    DoEvents
Next
r.Select
End Sub

gary's student say:

This routine will toggle the transparency:

 

Sub ToggleVisibility()
ActiveSheet.Shapes("Rectangle 1").Select
With Selection.ShapeRange.Fill
    If .Transparency < 0.1 Then
        .Transparency = 1
    Else
        .Transparency = 0
    End If
End With
Range("A1").Select
End Sub

Can't i use the Display Name (Department) in the pop-up itself, and once a choice has been made, have it show the Value (contact info)??  As it stands, Display Name and Value keep being the same.

Feels so possible...

Thank you!

macropod say:

If you're using a dropdown content control, and the value to which you refer is the content control's Value for the selected item, you could use a ContentControlOnExit macro coded like:

Private Sub Document_ContentControlOnExit(ByVal CCtrl As ContentControl, Cancel As Boolean)
Dim i As Long, Strval As String
With CCtrl
  If .Title = "MyTitle" Then
    For i = 1 To .DropdownListEntries.Count
      If .Range.Text = .DropdownListEntries(i).Text Then
        Strval = .DropdownListEntries(i).Value
        .Type = wdContentControlText
        .Range.Text = Strval
        .Type = wdContentControlDropdownList
      End If
    Next
  End If
End With
End Sub

Do note that, although the Value defaults to whatever you input for the Display Name, you can change it to something else.
