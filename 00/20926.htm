amit tandon say:

If instead of columns D & E you want to refer to only D2 & E2 then use below formulas.

This formula returns D2-E2 if C2 is EITHER "red" or "black", else the formula returns D2:

=IF(OR(C2="red",C2="black"),D2-E2,D2)

This formula returns D2-E2 if C2 is EITHER "red" or "black", if C2 is blank the formula returns D2, if C2 is not blank and neither red or black then the formula returns "Other".

=IF(OR(C2="red",C2="black"),D2-E2,IF(C2="",D2,"Other"))

You may pick any of the above per your requirement.
