File Name Auto-Populate - Microsoft Community

aom_84 say:

File Name Auto-Populate

How does Word auto-populate the save field?  I am trying to get it so when I click save on a new document (using a dotx form) it auto-populates the file name as follows:  "date - letter to _______".  The date would preferably be formatted like 11-06-16 and the ____________ would be filled in with a {REF }. 

 

Thanks!

suzanne s. barnhill say:

Word takes the first words of the document up to the first punctuation mark or paragraph break unless there is content in the Title field of the Document Properties. In previous versions, you could get Word to present the Document Properties when you first Save As (though that is too late for this purpose); that's not feasible in Word 2007. You can, however, access the Properties before saving. The Title field will be blank unless you have added text to the Title field of the template on which the document is based (in which case it will be repeated).

 

That said, it's just as much trouble to change the Title property as it is to change the suggested filename.

    j0oi say:

    Quick Fix:

    The font seems to be the culprit.

    I changed the font and it fixed the ASCII boxes.

    Try "Yu Gothic". It's quite clean and readable.

    I read that Windows 10 changed it's default Japanese font to "Yu Gothic UI" for various display items.

    Longer Fix:

    My previous font was "MS Gothic" (and "Meiryo" too, I believe). Interesting enough, after applying a different, Windows 10 compatible font to some Japanese text, characters previously written in the MS Gothic font appear, but with odd spacing. (See screenshot)


    Though "MS Gothic" shows as the selected font for these characters, it looks that I cannot apply the "MS Gothic" font to new characters in Windows 10.

    I went to 'Control Panel > Appearance and Personalization > Fonts' and didn't find the MS Gothic font.

    It looks that the ASCII boxes appear because Windows 10 doesn't have the correct font installed - the font that you used for your characters written using say, Windows 8.1, 8, 7, or any other older version.

    If you prefer your older font, you can try installing it (click and drag a downloaded font family item into the fonts window), restart, and see if your characters show!
