be "Disable Antivirus". In which case you should change its DWord value

Right click the folder and select "Export". Select its destination (Desktop)

Right click the Windows Defender folder and select "delete".    No worry!

The Key will reconstruct with only "Default" in it when your PC is restarted.

Back out of the Registry by retracing the steps that you used while entering,

to keep your Registry from getting cluttered.

Once you are at HKLM, select "File" then "Exit".

Restart your PC and check Windows Defender again.

Set wn = Application.Windows(2)

Assuming there are no empty/blank cells in the description range and there is at least some data to work with and the data is text and text only and there are no formula blanks in the range and there are no error values in the range and.....(did I miss anything? <g>)

A2:A20 = Descriptions given the named range Desc

Let's get a count of the unique descriptions. We'll use this count as an error trap.

F1: =SUMPRODUCT(1/COUNTIF(Desc,Desc))

Now let's extract the descriptions from most instances to least instances.

Enter this array formula** in F3:

=INDEX(Desc,MODE(MATCH(Desc,Desc,0)+{0,0}))

Enter this array formula in F4 and copy down until you get blanks:

=IF(ROWS(F$3:F4)>F$1,"",INDEX(Desc,MODE(IF(COUNTIF(F$3:F3,Desc)=0,MATCH(Desc,Desc,0)+{0,0}))))

Or, assuming the descriptions are text strings and using a built-in error trap instead of the counter cell formula...

Array entered** :

=LOOKUP("zzzzz",CHOOSE({1,2},"",INDEX(Desc,MODE(IF(COUNTIF(F$3:F3,Desc)=0,MATCH(Desc,Desc,0)+{0,0})))))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

For descriptions that appear the same number of times as others, the formulas will extract them as they appear in order from the top. For example:
