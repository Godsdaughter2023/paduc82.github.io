Adding data to an existing line graph - Microsoft Community

davidperham say:

Adding data to an existing line graph

Howdy:

I have a  worksheet which has a line graph which has four series defined.

The data for each series is located in a column on the worksheet.

So Series 1 contains data from rows 25-30 of  column C

Series 2 contains data from rows 25-30 of column D and so forth.

Now I have added data in row 31 and need to reset the data source for the line graph to include this row.

So now each series will contain data from rows 25-31.

So far all the examples I've been able to find involve adding a new series, which I do not want to do.  I want to update the existing series.

Thanks for your help (in advance) 

teylyn say:

Here is an example. Consider the following screenshot:

Set up a range name called chtLabels with the formula 

=OFFSET(Sheet1!$B$2,1,0,COUNTA(Sheet1!$B:$B)-1,1)

This range name returns all the x axis labels for the chart, i.e. B3:B5

Now you can create range names for the series by offsetting from that initial range.

chtApples =OFFSET(chtLabels,0,1)
chtBananas =OFFSET(chtLabels,0,2)
chtCitrus =OFFSET(chtLabels,0,3)

Here you can see what the range name chtCitrus refers to.

Now construct a line chart and use the range names in the series and X axis. Important: When entering the range names in the chart dialogs, you need to precede the range name with the sheet name, for example

=Sheet1!chtLabels

Alternatively, you can use the file name before the range name

='My Chart Demo.xlsx'!chtLabels

If the sheet or workbook have spaces in the name, you need to wrap it in single quotes.

The chart will now automatically plot new data when a new value is added in column B.

You can access the sample file used to take the screenshots here.
