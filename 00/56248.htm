slowhand818 say:

Finally managed to fix it.  To help others with this issue this is how I did it. (not sure which step did the trick but one of them did)

1)  I uninstalled the Wacom driver and then deleted the Tablet folder in C:\program files, and any folders called Wacom on the C:\ drive

2) opened Regedit and scanned for all Wacom keys and deleted them (make a backup of the registry before doing this)

3)I unpacked the Wacom installer using winzip onto the desktop. 

4) I noticed only the setup.exe had admin privileges, so added run as admin to each exe file via the compatibility tab.

5)Reinstalled the driver using the setup.exe from the unpacked installer.

6)Rebooted and everything working normally.

I checked what was different and noticed that their was another Wacom service running in parallel with the Wacom Professional Service and I think that is the root of the problem. Without admin I think the services are not installed correctly.
