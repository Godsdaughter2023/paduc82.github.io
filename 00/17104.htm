bob jones aka: cybertaz say:

  1. Go to View> Guides, then select Snap to Grid to clear the check. The arrow keys will nudge in smaller increments.
  2. I believe you're mixing terms here :-) The default spacing between paragraphs is the same as the line spacing [aka, leading], which is based on font size. Space Before/After pertains only to increasing space between paragraphs, it has never accepted values less than zero... IOW, it cannot be used to decrease space between paragraphs. Also, it doesn't change Line Spacing of a paragraph. If dealing with paragraphs consisting of only a single line, changing the line spacing to something less than the default also will tighten up the paragraphs, themselves, i.e., reduce the 'space between' those paragraphs. However, in paragraphs consisting of more than 1 line that reduces the spacing between lines within the paragraph, as well.
  3. I'm not seeing this at all, nor can I reproduce it here. Nothing I paste has a Shadow applied, and there is no setting to make it a default. Without specific details re what's being copied & from where, I can't even hazard a guess. I can only suggest that you confirm that your installation is fully updated... Office should be at v15.18 or higher. If you can provide links to shared images I'll take a closer look. It would be a good idea to post this as a separate question providing as much descriptive info as possible.
  4. The only difference I see between PPT 2016 & 2011 is that the latter increments in 5/100s for Top & Bottom, whereas the former increments in 1/10s. I don't recall any Mac version using increments of 1/100s when using the arrow buttons... IIRC, increments of 1/100 always have had to be typed in, even in Word. [Note: The arrows for Left & Right margins increment in 1/10 in both versions of the program.]

The requirements for using the system are:

  1. The mail merge main document must be of the Letter type, though that does not mean that the output cannot be sent as an e-mail message. 

  2. For the Many To One, Merge with Attachments and Merge to Individual Docs utilities, the data source may be either a table or query in an Access database, or in the form of an Excel worksheet and that worksheet must be the first sheet in the Excel workbook. If the data is on some other sheet, you can easily move that sheet so that it is the first sheet in the workbook by clicking on the sheet tab and dragging it to the left.  For the Chart Merge utility, download the Mail Merging with Charts document that is also on that page of my OneDrive for additional requirements of the data source for use with that utility

  3. For a data source in the form of an Excel worksheet, the field names must be in the first row of the worksheet and there must be a field name in all of the cells in that row that are within the range of columns that contain the data.

  4. For both types of data source, the field names must contain only alphanumeric characters (No @,#,$,%,&,(,), etc) and the field names must not start with a numeric character (0-9).

  5. For a data source in the form of an Excel worksheet, the data must start in the second row of the worksheet and there should be no empty rows within the range of data that is to be processed.

macropod say:

Click on the ¶ symbol on the Ribbon's Home tab to switch on Word's formatting display. Paragraph breaks, for example, should now appear as ¶, whilst Section breaks should appear as a line of = characters. Even so, if a Section break has been inserted at the end of a line, you may not see it. Switching to Draft view may show that you also have Section breaks in the document that you were not aware of.

You can use Find/Replace to delete Section breaks, where:

Find = ^b

Replace = ^12

This will convert the Section breaks to page breaks, so as to preserve the page-break aspect of the layout.

Do be aware that using this with ReplaceAll will impact continuous Section breaks as well. With all the Section breaks replaced, your header should now work as desired. If you have any continuous Section breaks that you need to preserve, you'll need to ensure each of the Sections created by them have the 'same as previous' header/footer attribute applied. You should also be aware that a given Section in a Word document can be configured with a 'different first page' and/or a 'different odd & even' setup. Arbitrarily deleting Section breaks in such a document can result in the wrong header/footer appearing throughout the Sections whose breaks you're deleting.

What you are describing sounds like buggy behavior. I would get around this by creating a style that looks like the black text, but has a different font color. Please follow these steps:

  1. In Word, with your document open, select some of the black text.
  2. Press Alt + Ctrl + Shift + S to open the Styles list.
  3. Click on the Manage Styles button at the bottom. It's the one with a tiny pencil.
  4. On the Edit tab of the Manage Styles dialog, the style name for the black text will be highlighted.

  7. Change the font color to red.

formatting chapter titles for a book using styles Word 2016 Mac

How can I format a chapter title for a book using my header style so that it ends up a third or so down the page without having to use the paragraph space bar or return bar? Isn't there a way to set this up so that with each chapter title the spacing down the page will be uniform?

Thanks,

bob jones aka: cybertaz say:

In your Heading Style include the amount of Space Before in the Format Paragraph> Indents & Spacing dialog. It's intended to be set in points, so if not familiar with the conversion you can safely use the approximation of 72 pt/inch. If your measurement system is set to Inches you can type the value instead but you have to include the in abbreviation... Likewise, use cm for centimeters, etc.

A tip if I may: If you haven't already considered it, it's better to use the Page Break Before attribute in the style rather than chopping the document up with forced [manual] page breaks. It's in the same dialog on the Line & Page Breaks page. That will make sure your chapter titles always start on a new page. Manual page breaks move with the text flow & can wind up in some awkward & unwanted places, even causing blank pages in the document.

There are many ways for you to personalize your Windows Phone, it has a built-in accessibility feature that can make it easier to view emails. You can use the Screen magnifier to help magnifying the screen.

The screen magnifier lets you use gestures to enlarge any screen you're viewing.

  1.  In the App list, tap Settings > Ease of access.
  2.  Turn on Screen magnifier .

  3. On any screen, double-tap with two fingers to magnify it. Once it's magnified, you can try the following:

    • To move around the entire screen (even the parts that aren't showing because you're zoomed in), use two fingers to single-tap and hold anywhere on the screen, then pan in any direction.
    • To change the zoom level while the screen is magnified, use two fingers to double-tap and hold, then pan up or down to increase or decrease magnification.
    • To scroll up or down while the screen is magnified, first pan to the edge of the screen. Once you reach the edge, you can scroll as you normally would by swiping or panning with one finger.
