If you set the default printer for Windows to the printer, instead of leaving it set to a PDF driver, then Word will use the printer every time you start the program. You can choose a different printer driver on Word's Print page, but it will return to the Windows default when you restart Word.

In Windows 8.1, on the Start screen, type printers. When the search results display, select Devices and Printers. (In Windows 7, or in Windows 8.x with one of the third-party Start menu programs, you should find Devices and Printers on the Start menu, or you can choose it from the Control Panel.)

In the Devices and Printers window, right-click your printer and click Set as default printer. The current default printer is the one that displays a check mark in a green circle.

No, your ost-file will grow and shrink as needed throughout the entire year but your cache will only contain the last 12 months at any time as you've set that with the Sync Slider.
