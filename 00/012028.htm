In each of the sheets named T1, T2 & T3, enter the below formula as an array formula (CTRL+SHIFT+ENTER) in cell A2 & copy down:

=IFERROR(INDEX(Sheet1!$A$2:$A$100,SMALL(IF(LEFT(Sheet1!$A$2:$A$100,2)=MID(CELL("filename",$A$1),FIND("]",CELL("filename",$A$1))+1,255),ROW(Sheet1!$A$2:$A$100)-ROW(Sheet1!$A$2)+1),ROWS(Sheet1!$A$2:$A2))),"")

jon - vertex42 say:

I'm guessing that you are entering these values into column A and you may have hundreds of values and you want to split these values out to sheets T1 through T3?

1) select cells A1:A10 in sheet 2

2) enter the following function

=IFERROR(INDEX(Sheet1!$A:$A,SMALL(IFERROR(IF(FIND("T1",Sheet1!$A:$A,1),ROW(INDIRECT("A:A"))),""),ROW(INDIRECT("1:10")))),"-")

3) press CTRL+SHIFT+ENTER to enter this as an array formula

4) repeat for sheets T2 and T3 but change "T1" within the formula to "T2" and "T3"

If you'll have more than 10 values, you can select as many cells as you think you'll have and change "1:10" accordingly.

andre da costa say:

If your computer is UEFI based, these are normally systems that come pre-loaded with Windows 8 or later, you will need to prepare the ISO file for such a configuration or you will receive an error message during setup. The thumbdrive needs to be formatted as FAT32 and use the GPT partitioning scheme. To do this, you need to use Rufus, a small tool you can download for free.

 

After you have installed Rufus:

  • Launch it
  • Select ISO Image
  • Point to the Windows 10 ISO file
  • Check off Create a bootable disk using
  • Select GPT partitioning for EUFI firmware as the Partition scheme
  • Choose FAT32 NOT NTFS as the File system
  • Make sure your USB thumbdrive in the Device list box
  • Click Start
  • Close when complete
