Use the workbook's sheet change event:

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)

and any time the value of the target matches that of a named picture, copy the picture to the cell to the right of the target. Modify the copy / paste part of the code like this

   If Not IsError(Application.Match(Target.Value, Worksheets("Pictures").Range("NameList"),False)) Then

      Selection.Copy

      Sh.Select

      Target.Offset(0, 1).Select
      ActiveSheet.Paste

   End If
