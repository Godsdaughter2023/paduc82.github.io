bob jones aka: cybertaz say:
I think you're referring to the font size of the Content Placeholders of the 4th [Two Content Layout] & 5th [Comparison Layout] Layout Masters not changing when you change the font size of the Office Theme Slide Master's Content Placeholder. If so, that's because the font size of the Content Placeholders on the Layout Masters already differs (by default design) from the font size used on the Slide Master. The font size of the Layout Master Content Placeholders is smaller because the placeholders are smaller. Their font sizes do not reflect the new size used on the Slide Master's Content Placeholder, nor does their font size scale proportionately.

The font size for the Content Placeholders on those Layout Masters has to be changed individually... Changing the font size for 1 doesn't cascade to the others of the same type. If you use copy/paste, click the Paste Options button [little Clipboard] & select the Keep Source Formatting option.
