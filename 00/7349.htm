bobpei say:

Objects disappear in Word

Is there any way to see objects that Word has made disappear? Is there any global view or way to see objects that are located off of visible pages?
john korchok say:

All objects are anchored to text. When you select text that has an object anchor, the object is also selected. Deleting the text deletes the anchored object.

Temporarily deleting a couple of paragraphs on a previous page should move the text with anchor to the previous page. The object is probably positioned relative to that text, so it should appear on the previous page above the text. Delete or move the object, then paste the temporarily deleted text back in.

It's also possible to use VBA to move any off-page objects back onto the page, but it doesn't sound like that's necessary here.

bob jones aka: cybertaz say:

A couple of points for clarification:

  • There is no physical page element in a Word document. The pages displayed in Word are euphemistic constructs designed to show how the document will appear when printed. The program is designed that way because that's what users want to be able to see, therefore...
  • Locking an object anchor does not fix the object at a specific location, it merely locks the anchor to the paragraph to which it currently is attached. IOW, the anchor only cannot be dragged to a different paragraph, the object still can be moved elsewhere on the page, but...
  • An object can display only on the same page as the paragraph to which it is anchored.

With those points in mind, Locking the anchor may well be the cause of the object "disappearing". If the anchor is locked nudging the object off the page will cause it to be removed from view but Word has no stage or workspace beyond the boundaries & the object does not get nudged to the next page if the anchor is locked. IMHO this is a design flaw - If the object is dragged to a different page it will snap back to the top of the page containing the paragraoh it's anchored to but that doesn't happen when nudging.

See if this will help:

Go to the Layout tab, then click the Selection Pane tool to display the pane. You should see that the object is listed in that pane. Click it to select it then use the Position tool on the Layout tab to choose one of the preset positions. The object should then become visible in that position on the same page as the paragraph to which it is anchored.
