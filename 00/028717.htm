hansv mvp say:

Like this:

Sub SetSubscript()
    Application.ScreenUpdating = False
    With ActiveDocument.Content.Find
        .ClearFormatting
        .Replacement.ClearFormatting
        .Replacement.Font.Subscript = True
        .Format = True
        .Text = "(-)(?)"
        .Replacement.Text = "\2"
        .MatchWildcards = True
        .Execute Replace:=wdReplaceAll
    End With
    Application.ScreenUpdating = True
End Sub

In case you still not able to find the icons then check if you find a small down arrow button at the bottom of the main Windows 8.1 Start Screen to reveal the additional Start Menu.   1. If Windows 8.1 and Office 2010 is 64bit, the location of Office exe files is C:\Program Files\Microsoft Office\Office14

  2. If Windows 8.1 is 64 bit and Office 2010 is 32 bit, then the location of Outlook.exe is C:\Program Files(x86)\Microsoft Office\Office14

  3. If Windows 8.1 and Office 2010 is 32bit, the location of Outlook.exe is C:\Program Files\Microsoft Office\Office14

  1. winword or winword.exe for Word 2010

  2. excel or excel.exe for Excel 2010

  3. powerpnt or powerpnt.exe for PowerPoint 2010

  4. onenote or onenote.exe for OneNote 2010 

Right-click on the above Office 2010 application files and click on Send to > Create Shortcut (Desktop). You can also pin these shortcuts to the Start Menu or Taskbar following the steps as mentioned above.
