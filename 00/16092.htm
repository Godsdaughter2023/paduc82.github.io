General info on Exchange....

The only limit is for folders. You should not exceed 2GB per folder. The Inbox in particular needs to be kept as clean as possible. No more than a few hundred messages for best results.

Exchange Basics

The Inbox syncs more frequently than any other and should be kept as clean as possible. (this is the number one cause of problems with sync). Do NOT make subfolders under the Inbox.

Rebuild Basics:

  • Do Not Rebuild Identities with Exchange and IMAP accounts. Delete the accounts (this deletes the cache files in Outlook). Do the rebuild then add back accounts. Better yet start over in a new Identity. If the rebuild does not work then you could sync unwanted changes or deletions to your server and lose data.
  • Make sure you have plenty of free space on your drive. Make sure that you have at least three times as much hard disk space available as the current size of your Outlook identity PLUS 10% free for virtual memory.
  • Rebuilding does not fix all issues. It can make some problems worse. Even though the Database Utility will make a backup, I advise making a copy in the Finder first.
  • You can delete the rebuild version after you are satisfied with the results. This copy will be named like this: Main Identity [Backed up 1-21-2012 13.35]

Outlook for Mac has no size limits. It limited, however, by number of
objects (mail messages, calendar events, contacts, etc.)
 
Last I heard the limit was around 2 million but that may be higher now.
For what it's worth, no average user will ever hit 2 million items. He
would need to receive a new mail message every minute of every hour of
every day for nearly four years.
 
Outlook's performance, however, will be impacted by the number of items
it must maintain. Routinely deleting unwanted messages and emptying the
Deleted Items folder will be just fine. Keeping no more than a few
thousand messages in a folder also helps performance.

If the issue persists, then you'll probably need to reset Microsoft Edge. Please note that resetting Microsoft Edge might cause you to lose your favorites, history, and settings. Before you move forward, it's recommended that you create a system restore point in case something goes wrong, and you'll need to restore your last good known state of Windows 10. You may refer to the steps below:

  1. Close Microsoft Edge.

  2. Open File Explorer and type the following path into the address and hit Enter: C:\Users\UseYourUsername\AppData\Local\Packages(Note: Replace "UseYourUsername" with your user account name.)

  3. Select and delete the Microsoft.MicrosoftEdge_8wekyb3d8bbwe folder. If you get a Folder Access Denied prompt, simply click Continue. There might be some content that you won't be able to delete.   4. You may notice that you can't delete AC folder, but that's fine.

  5. Restart your computer.

    cd C:\Users\UseYourUsername. (Note: Replace "UseYourUsername" with your user account name.)

    Get-AppXPackage -AllUsers -Name Microsoft.MicrosoftEdge | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml" -Verbose}
