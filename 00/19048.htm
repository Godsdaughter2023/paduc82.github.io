Turn off add-ins

If you have Write permissions and the default font setting still doesn’t stick, you might have to turn off Word add-ins and change the default font setting. Here’s how:

Click File > Options > Add-Ins.

In the Add-ins box, find one of the add-ins you want to turn off and note the add-in type listed in the Type column.

For example, click COM Add-ins, and click Go.

Repeat steps 1-4 for other types of add-ins.

After you change the default font, turn on the add-ins.

Click an Add-in type in the Manage list, and click Go.

Check the boxes for the add-ins you want to turn on, and click OK.

How to rename a sheet automatic with cell address?

norman david jones say:


How to rename a sheet automatic with cell address?

Hi Mohammad,

Right-click the worksheet tab
Select  the View Code option
Paste the following code:

'===========>>
Option Explicit

'---------->>
Private Sub Worksheet_Change(ByVal Target As Range)
    Dim Rng As Range
    Const sAddress As String = "A1"           '<<==== Change

    Set Rng = Me.Range(sAddress)

    On Error Resume Next
    If Not Intersect(Target, Rng) Is Nothing Then
        Me.Name = Rng.Value
    End If
    On Error GoTo 0
End Sub
'<<===========

Alt-Q to return to Excel.

Each time that the value of A1 is changed, the name of the worksheet will change to reflect the new value, provided, of course, that this value represents a valid sheet name.

===

Norman

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim Rng As Range
    Dim Rng2 As Range
    Const sAddress As String = "A1"           '<<==== Change

    On Error Resume Next

    On Error GoTo 0
End Sub
