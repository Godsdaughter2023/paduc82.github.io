Hello,

1) If you want to always see the values of column A (including hidden rows), you can try this.

I suppose that you want to see values of column A on 'Sheet1'
Insert three columns before colomn B.

Into B1 of the new column B, enter the integer 1

Into C1 of the new column C, enter the formula:
=SUBTOTAL(103,B1)

Into D1 of the new column D, enter the formula:
="Row: " & NB.SI($C$1:C1;1) & " Val= " & INDEX(A:A;NB.SI($C$1:C1;1))

Copy cells B1:D1 down (twice the length of your data)
If you data end at row 100 then copy the cells B1:D1 down to row 200.

In column D, you should always see all the values from column A even if some rows are hidden.

You can hide column B and C.

2) to hide/unhide a row

If you data are on Sheet1, create with the VBA editor the following event sub into the module of 'Sheet1'

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
If Not (Intersect(Target, Columns(4)) Is Nothing) Then
Rows(Val(Split(Target, "Row:")(1))).Hidden = Not (Rows(Val(Split(Target, "Row:")(1))).Hidden)
End If
Cancel = True
End Sub

Now if you double-click on a cell of column D on your sheet 'Sheet1', the corresponding row wil alternately hide/unhide.

PowerPoint definitely CAN display Chinese text, even on an English-language system.  I frequently use Japanese in PowerPoint and have worked with clients in China who use Chinese text, of course.

The problem you're seeing is probably because you've started with an English-language template or theme.

When you change languages in PowerPoint or just open an English-based file or template in a Chinese language copy of PowerPoint, PowerPoint doesn't translate the text on the slide masters and layouts, so they stay in English.

garyzhu_msft3613 say:

Hi Marco,

No, users can’t access inactive mailboxes directly. The contents of an inactive mailbox are preserved for the duration of the hold that was placed on the mailbox before it was made inactive.

Note: to make a mailbox inactive, it must be assigned an Exchange Online (Plan 2) license so that a Litigation Hold or an In-Place Hold can be placed on the mailbox before it's deleted. If a mailbox is assigned an Exchange Online (Plan 1) license, you would have to assign it a separate Exchange Online Archiving license so that a hold can be placed on the mailbox before it's deleted.
