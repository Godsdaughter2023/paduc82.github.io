Hello Ken,

Welcome to Microsoft Community.

It is not advisable to move an actively used Outlook Data Files to OneDrive or any Cloud Storage, allowing Outlook access to the

You may use the Cloud Storage to save a copy of the PST file for easy access and backup purpose, storing an active PST file on cloud storage may cause issues with the normal functioning of Outlook which is trying to access the PST file. Doing so, may corrupt the PST file.

To resolve the issue, remove the link from Outlook 2016 with the PST file, follow these steps:

  1. Put Outlook in Offline Mode.
  2. Remove the email account that is accessing the PST file.
  3. Remove the PST file from OneDrive and save it locally on your computer or any other location.
  4. Add your email account again in Outlook 2016.
  5. Link your account with the PST file stored locally.
  6. Put Outlook 2016 online.
