fetaa say:

hi sarah,

 

here the answers as below:

(1)the save as location depends on the account you have logged in to office applications. if the account has the service for sharepoint online and onedrive for business, these two will show up in the save as location. by default, office 365 pro plus gives users access to office files that reside on the internet. onedrive for business and sharepoint is the default services for the account. if we want to remove these connected services and only keep the "this pc" location, we need to disable the feature to connect to internet resources. this action will also disable the feature to add a new service. please follow the steps below if you want to remove the full feature:

1. if you use office 2016, from registry editor, browse to:

computer\hkey_current_user\software\microsoft\office\16.0\common\internet\useonlinecontent

2. set the value of useonlinecontent to one of the following (to remove the connected services, set the value as 0. to recover the connected services, set the value as 2):

useonlinecontent value        value type      description

0                                              dword        do not allow user to access office 2016 resources on the internet.

1                                              dword        allow user to opt in to access office 2016 resources on the internet.

2                                              dword        (default) allows the user to access office 2016 resources on the internet.

 

note: if you use office 2013, browse to the following path and change the value:

computer\hkey_current_user\software\microsoft\office\15.0\common\internet\useonlinecontent

 

if you couldn't find the useonlinecontent, you may create it manually as below and change the key name to useonlinecontent and then set the value as 0 .



(2)please note that the office 365 work or school account is created by an organization’s administrator to enable a member of the organization to access all microsoft cloud services such as microsoft azure, windows intune or office 365. an office 365 work or school account can take the form of a user’s organizational email address, such as *** Email address is removed for privacy ***.
