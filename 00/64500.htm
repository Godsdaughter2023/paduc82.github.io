  1. Click on the ‘Windows’ icon at the bottom left corner of your screen.

  3. Click on the ‘Startup’ tab, you will be able to see a list of startup applications.

  4. Look for ‘Mixed Reality’, select it and click on ‘Disable’.

  5. Restart your system. Check if the portal still appears when you are playing your game.

If the issue persists, please follow ‘Method 2’.

Method 2: Uninstall Mixed Reality Portal.

Disclaimer: This method contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs.

You can refer this article on ‘How to back up and restore the registry in Windows’ to create a backup of the registry and restore it if necessary.

  1. Press the ‘Windows’ key and ‘I’ key, this will open the ‘Settings’ app.

  2. If you are able to find ‘Mixed Reality’, please skip to Step 12.

  5. Type ‘regedit’ and press enter, this will open the ‘Registry Editor’.

  6. Go to the following key: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Holographic

automatic out of office reply in microsoft live mail - Microsoft Community

julia schofield say:

automatic out of office reply in microsoft live mail

Is there a way to create an out of office automatic reply in microsoft live mail, or should I start using Outlook? please. Julia

leonard_su say:

Hi Julia,

 

Thank you for posting to Microsoft Community. The option you are inquiring about is currently not available in Windows Live Mail. However, you can achieve the similar feature by following the workaround we provided below:

 

Creating vacation reply in Notepad:

1. Open Notepad from your computer.

2. Type the message you want to appear in the body of your auto response.

3. Select File | Save from the menu.

4. Type Vacation Auto-Reply under "File name:".

5. Make sure Text Documents (*.txt) is selected under "Save as type:".

6. Click Save.

7. Close Notepad.

 

Live Mail steps:

1. Open Windows Live Mail.

2. Click Folders tab.

3. Click Message rules.

4. Under "Select one or more conditions:" enable For all messages option by clicking the check box beside it.

5. Under "Select one or more actions:" enable Reply with message option.

6. Click message beside Reply with under To edit this description, click the underlined word.

7. When the pop up box opened, click the dropdown box beside "File name:" field and select Text (*.txt).

8. Navigate your folders go to where you've stored the vacation reply you've created in Notepad.

9. Click open (after selecting the file) or double click the file name.

10. Click Save rule button.

11. Click OK button.

 

Should you have further questions or clarifications, please don't hesitate to post it here.

 

Regards,

Leonard.
