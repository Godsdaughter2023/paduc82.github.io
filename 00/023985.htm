Having the phone turned on during charging is actually a requirement for second generation Windows Phone devices. You should only allow users to edit data in forms, never directly in the tables themselves.

You can set the Recordset Type property of a form to Snapshot. This will make the records in the form read-only, while users can still filter the records any way they want.
