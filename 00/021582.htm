Display Billable Amount and Cost Amount in MS Project - Microsoft Community

singhinc say:

Display Billable Amount and Cost Amount in MS Project

In project I am trying to show two figures for each resource for each task:

(1) the billable amount (based on a billable rate) and

(2) The internal cost (based on the std. rate costs). 

 

How can I do this?

 

Thanks!

Ravi

john - project say:
singhNC,
I assume you want to see the two cost values on the Resource Usage view. First, for each resource enter the billable rate into one of the other 4 cost rate tables. You can get to the cost rate tables by double clicking on each resource on the Resource Sheet view. Once the two rates are all set up, add the Cost Rate Table field as a column to the Resource Usage view. Select the rate table you used to enter the billable rate. The Cost field will now display the total billable value. Copy those values into a extra cost field (e.g. Cost1). If you then switch back to the cost rate table "A", you will see internal cost in the Cost field and billable cost in the extra cost field.

Unfortunately there is no way to have Project automatically apply both cost rate tables simultaneously in any view. However, you could develop a simple macro that does the switch and copy operation described above.

If you also want to see the difference in total task cost on the Gantt Chart you will need to copy the billable cost from the Resource Usage extra field to an equivalent extra field in the Gantt Chart. Note: Resource Cost1 is NOT the same field as Task Cost1. For more information on that, go to the MVP website at, http://www.mvps.org/project/faqs.htm, and take a look at FAQ 51 - Data types: Task, Resource and Assignment. FAQ 37 - Custom fields in views, shows macros for copying the fields.

Hope this helps.

Update later in the day (after pondering my first answer while I was pounding a bunch of 16d nails);
Av even simpler solution is to set up a customized cost field with a formula. Unless the billing rates for each resource varies all over the place, you canuse a multiplying factor for resource base cost to arrive at the billing cost.
