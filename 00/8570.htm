Hello Larry

Might be useful to include why you're asking the question assuming that it's in relation to another question/issue.

First, the Outlook AddressBook (OAB) is a specialized "view" of the contact folders that comprise the OAB. There is always one contact folder that is automatically a member of the OAB which is the "default contact folder" associated with the first email account configured in the Outlook profile. The default contact folder is stored in the "default data file" for the profile (automatically set to the data file associated with the first email account configured). The default data file can be changed at any time.
To see which data file is the "default", go to Outlook menu, File -->Account Settings --> Data files tab

In addition, additional contact folders cane be manually created but each must manually be set to be a member of the OAB whih is done by right-clicking on the contact folder --> <Properties> --> Outlook Address Book tab --> check the box for "Show this folder as an email address book>. There can be any number of contact folders set to be a member of the OAB.

The above description applies to any version of Outlook desktop - '2010 and beyond.

ian ser say:

Hi Infima,

Antimalware Service Executable is a process used by Windows Defender to protect you real-time for any threats. What I recommend you to do is to change the triggers that starts this process and set it only whenever the device is idle. Follow these steps:

  1. Press Windows key on your keyboard, then type Task Scheduler, right-click the Task Scheduler search result then choose Run as administrator.
  2. On the left-pane, click Windows folder and search for Windows Defender.
  3. Double-click Windows Defender Scheduled Scan to open its setting.
  4. On the window that would appear, click the Conditions tab.
  5. Under Idle, tick Start the task only if the computer is idle for: and Wait for idle for: and set these for around 1 or 2 hours depending on your preference.
  6. Make sure that you also tick Stop if the computer ceases to be idle so that whenever you're actively using the device, the process would stop.
    NOTE: Make sure that only the Idle options are the only one with checks.
