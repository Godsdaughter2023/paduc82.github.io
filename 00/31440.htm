ken blake say:

Although you may see it as a problem, it really isn't. What has occurred is standard.

A higher resolution means that more pixels are put on the screen at once, and in turn that means that each pixel is smaller. That means that a letter which uses a fixed number of pixels is smaller. So all text is smaller.

However, regardless of what resolution you run are running at, in most programs, you can enlarge the text size by holding down the Ctrl key and scrolling the mouse wheel. That's a Windows standard and works in most, but not all, programs.
