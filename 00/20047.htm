There are three places where you can use the Calendar.


1. Calendar.Live.com

2. PC - in this case, your Surface.

3. Your mobile device - your 620.

john mcghie say:
Calming down would be good (yeah, I know, that's no fun...)

The issue is this:

1)  Tracked changes was turned on

2)  Some documents acquired changes

3)  Bob told you how to turn Tracked Changes off for your Normal template.

4)  Existing documents still have tracked changes in them.

5)  For existing documents, you need to "Accept All Changes" and turn Tracked Changes Off.

View: Markup will always appear whenever you open a document that contains any changes.  It is a safety feature that tries to ensure you do not email the next customer all the details from the last customer.

Just let's recap:

Fixing Normal.dotm prevents new documents starting with Tracked Changes turned on.

Opening an existing document that has tracked changes in it will always display the Reviewing Toolbar until all changes have been removed.
