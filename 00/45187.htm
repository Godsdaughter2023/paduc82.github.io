How do I burn a DVD (not CD) in movie maker ? - Microsoft Community

laura_600 say:

How do I burn a DVD (not CD) in movie maker ?

I have a dell laptop.  windows 8 / inspiron.

Movie maker works fine and I can burn to a CD but why can I not burn to a DVD?  Do I need additional software?

considerate_guy say:

Movie Maker will show the 'Burn a DVD' option only when Windows includes a copy of Windows DVD Maker.  That program is only available in Windows Vista and Windows 7, but not Windows 8.  For whatever reason, Windows DVD Maker is not being supported in Windows 8/8.1.

You can't burn a DVD directly, but if you save the movie (either in .mp4 or .wmv format), you can then use other software for burning a DVD.  Remember, if you want a DVD that will play in a standard DVD player, the software must be capable of creating a video DVD.  If you just burn the file onto a DVD directly, you are creating a data DVD and that won't play in most standalone DVD players.

Some possible choices for DVD software are:

  • Freemake Video Converter
  • DVD Flick
  • DVD Styler

All are freeware, so I would make sure you choose the Custom install option, so you can uncheck any add-ons you don't want.

One additional note: video DVDs, by definition, are a standard definition format.  If you are looking at using HD video, you may want to explore other options such as streaming video via your home network, or reading the video off of a USB drive.  Check the capabilities of your TV for more info.
