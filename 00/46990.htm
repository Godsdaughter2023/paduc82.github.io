Windows 10, version 1803 Mouse over problem - Microsoft Community

gjpc say:

Windows 10, version 1803 Mouse over problem

When my system updated to 1803 my windows immediately switch when I moused over them, there was no delay.

I had to go to Control Panel->Ease of Access->Change how your mouse works uncheck "Activate Window by hovering over it with the mouse" select the Apply button then recheck the hover box and select Apply to restore the delay between hover and activate the window.

Also, the name of the ease of access mouse panel in the title bar is "Make the mouse easier to use" not "Change how your mouse works"
