barneja say:

Hi kurobenko -

If your field A is a string then I think you will have a difficult time creating your formula within the pivot table.  I would suggest you go back to your sql database and create a new field that just has a value of .25, .5, or 1.0 that you use in your calculated field.  Then when you create your pivot table you can create your Field C from your updated field.  From my understanding, a pivot tables calculated field will only do a calculation on numeric data types; thus, your text in Field A will not allow it to be included in a calculated field.

Another workaround - what about using Field A as a row in your pivot data - or as a column?  Then you could use your current field B as your data and create multiple calculated fields where one is Field B * .25.  Another is Field B * .5 and a third being Field B * 1.  I don't like that solution very much - but because you use Field A as a column field then the only thing that would show up in each calc'd field would be the portion that would belong to that group (ie, the .25 * Field B would only show up in the column 25% - Almost not qualified").  You could hide the columns with blank data (I don't like hidden columns - which is why I would prefer modifying the sql database).

Hope this at least helps . . .

jeeped say:
I'll assume that you have a Sheet1 with UPC in A1, Image in B1 and Type in C1 with the UPCs going down column A. Start with Sheet2 and put UPC in A1. Put the following array formula in A2,

=IFERROR(INDEX(Sheet1!$A$2:$A$999,MATCH(0,IF(LEN(Sheet1!$A$2:$A$999),COUNTIF(A$1:A1,Sheet1!$A$2:$A$999),1),0)),"")

This requires Ctrl+Shift+Enter to finalize rather than just Enter. Once entered correctly, copy or fill down until you run out of unique entries from Sheet1's column A.

Next, in Sheet2's B1 put in this array formula, 

=IFERROR(INDEX(Sheet1!$C$2:$C$999,MATCH(0,IF(LEN(Sheet1!$C$2:$C$999),COUNTIF($A1:A1,Sheet1!$C$2:$C$999),1),0)),"")

Again use Ctrl+Shift+Enter to finalize. Copy or fill right until you run out of unique entries from Sheet1's column C.

Finally, put this standard formula into Sheet2's B2,

=IFERROR(INDEX(Sheet1!$B$2:$B$999,MIN(INDEX(ROW($1:$998)+(((Sheet1!$A$2:$A$999<>$A2)+(Sheet1!$C$2:$C$999<>B$1))*1E+99),,))),"")

This only requires Enter. once entered correctly, copy or fill both right and down to cover all of the values retrieved by the two previous array formulas.

If you have a unique list of UPCs in column A starting in row 2 and a unique list of TYPEs in row 1 starting in column B then run this macro to transfer the links from Sheet1 to Sheet2.

Sub xfer_UPC_Data()
    Dim lr As Long, rw As Long, vR As Variant, vC As Variant
    Dim app As Application, ws1 As Worksheet, ws2 As Worksheet
    Set app = Application
    app.ScreenUpdating = False
    app.EnableEvents = False
    Set ws1 = Sheets("Sheet1")
    Set ws2 = Sheets("Sheet2")
    vR = ws2.Cells(2, 1).Resize(ws2.Cells(Rows.Count, 1).End(xlUp).Row - 1, 1)
    vC = ws2.Cells(1, 2).Resize(1, ws2.Cells(1, Columns.Count).End(xlToLeft).Column - 1)
    With ws1
        lr = .Cells(Rows.Count, 1).End(xlUp).Row
        For rw = 2 To lr
            ws2.Cells(app.Match(.Cells(rw, 1), vR, 0) + 1, app.Match(.Cells(rw, 3), vC, 0) + 1) = _
              .Cells(rw, 2).Value
        Next rw
    End With
    app.EnableEvents = True
    app.ScreenUpdating = True
    Set ws2 = Nothing
    Set ws1 = Nothing
    Set app = Nothing
End Sub

I would suggest something like:


If necessary, the column labels could be put in some specific order, or sorted (as could the UPC codes.

In any event, this can be done with a macro.  The macro assumes your original data is on Sheet1 and is bounded by at least one empty row and column; and that your results will go onto Sheet2 (and that there is nothing else of value on Sheet2).  These assumptions are easily changed, if you supply specifics.

You will also note that I left the Deleted "Type" blank, as I don't know what, if anything, you want to put in there.

To produce the above result from your data, follow the directions below:

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.

Option Explicit
Option Compare Text
Sub SortUPC()
    Dim vSrc As Variant, vRes() As Variant
    Dim wsRes As Worksheet, rRes As Range
    Dim sTypes() As String, sUPC() As String
    Dim colUPC As Collection, colTypes As Collection
    Dim i As Long, j As Long
    Dim rw As Long, col As Long
   
'Results worksheet and range
Set wsRes = Worksheets("Sheet2")
Set rRes = wsRes.Range("A1")
wsRes.Cells.Clear

'Get Source Data
'assume on Sheet1, but could be anywhere
vSrc = Worksheets("Sheet1").Range("A1").CurrentRegion

'Generate list of unique UPC's
On Error Resume Next
Set colUPC = New Collection
For i = 2 To UBound(vSrc, 1)
    colUPC.Add Item:=CStr(vSrc(i, 1)), Key:=CStr(vSrc(i, 1))
Next i

'Generate list of unique Types
Set colTypes = New Collection
For i = 2 To UBound(vSrc, 1)
    colTypes.Add Item:=CStr(vSrc(i, 3)), Key:=CStr(vSrc(i, 3))
Next i
On Error GoTo 0

'Dimension results array
ReDim vRes(1 To colUPC.Count + 1, 1 To colTypes.Count + 1)
For i = 2 To UBound(vRes, 1)
    vRes(i, 1) = colUPC(i - 1)
Next i
ReDim sUPC(1 To colUPC.Count)
For i = 1 To colUPC.Count
    sUPC(i) = colUPC(i)
Next i

'Types array
ReDim sTypes(1 To colTypes.Count)
For i = 1 To colTypes.Count
    sTypes(i) = colTypes(i)
Next i

'Labels row
vRes(1, 1) = "UPC"
For i = 2 To UBound(vRes, 2)
    vRes(1, i) = sTypes(i - 1)
Next i

'Fill in the array
For i = 2 To UBound(vSrc, 1)
    With WorksheetFunction
        rw = .Match(CStr(vSrc(i, 1)), sUPC, 0) + 1
        col = .Match(CStr(vSrc(i, 3)), sTypes, 0) + 1
    End With
    vRes(rw, col) = IIf(vSrc(i, 3) = "Deleted", "", vSrc(i, 2))
Next i
   
'Write results
Set rRes = rRes.Resize(rowsize:=UBound(vRes, 1), columnsize:=UBound(vRes, 2))
rRes.NumberFormat = "@"
rRes = vRes
rRes.EntireColumn.AutoFit
End Sub
