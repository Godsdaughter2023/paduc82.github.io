Okay,

you will need a macro, it's easy

try this...

 

first,

both workbooks are open

and..

 

step1

Save As the workbook1 with extension .xlsm (macros enabled)

 

step2

1) press ALT+F11 to open Visual Basic

2) Insert > module and paste the code below on the right

 

[Edit]

 

Sub CompareBooksOneSht()
Dim ws1 As Worksheet, ws2 As Worksheet
Dim t As Long
Set ws1 = Workbooks("workbook1.xlsm").Sheets("Sheet1") '<<< change wbks and sheets name
Set ws2 = Workbooks("workbook2.xlsx").Sheets("Sheet1") '<<< also, change names
Sheets.Add
[A1] = "Cells"

[B1] = "wb1"
[C1] = "wb2"
t = 2
For Each r In ws1.UsedRange
If r.Value <> ws2.Range(r.Address).Value Then
Cells(t, 1) = r.Address

Cells(t, 2) = r.Value
Cells(t, 3) = ws2.Range(r.Address).Value
t = t + 1
End If
Next

With Columns("A:C")
.AutoFit
.HorizontalAlignment = xlLeft
End With
End Sub

 

3) Press ALT+Q to Close Visual Basic
