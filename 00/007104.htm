Dim err As String

 If Nz(Field1,"") Then

   End If

scottgem say:

As Scott MD said a Select Case checks a SINGLE item and then does different things based on the value of that item.

However, there is an ElseIF condition that you can use to check a series of different conditions. So in your example you could use:

If Field1 = "" or IsNull (Field1) then

       msgbox ("Field1 is empty") 

ElseIF Field2 = "" or IsNull (Field3) then

      msgbox ("Field2 is empty")

   ...

ElseIf FieldN = "" or IsNull (FieldN) then

      msgbox ("FieldN is empty")

End If

However, I would do it slightly differently:

Dim strMsg As String

If Field1 = "" or IsNull (Field1) then

       strMsg = "Field1 is empty"

ElseIF Field2 = "" or IsNull (Field3) then

      strMsg = "Field2 is empty"

   ...

ElseIf FieldN = "" or IsNull (FieldN) then

      strMsg ="FieldN is empty"

End If

MsgBox strMsg

Or If you have a lot of these I would suggest a loop

Dim ctl As Control

Dim strMsg as String

For each ctl in Me.Controls

      If ctl.Type = "Textbox" Then 

             If ctl.Value = "" OR IsNull(ctl.Value) Then

                   strMsg = ctl.Name & ", "

             End If

      End If

Next ctl

strMsg = strMsg & "controls are empty."

MsgBox strMsg

So the loop prepares a list of empty controls and displays a message box listing all the empty controls.
