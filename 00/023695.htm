phad012 say:

I had this exact problem and solved it, whilst managing to retain my conditional formats. Details below:

Background: I am using a Excel 2016 32-bit, macro-enabled workbook and a lot of conditional formatting. Spreadsheet would crash whenever I clicked 'undo'

Fix: I set all my conditional formatting rules to apply to only one cell. I then set them to apply to the whole data table (around 5000 rows). This seemed to reduce the number of conditional formatting rules and was kind of like starting over on the conditional formatting. I can now use 'undo' without any crashing.

Step-by-step (Excel 2016):

1. Home tab

2. Conditional formatting

3. Manage rules

4. For every rule, where it says 'Applies to' enter =$A$1 (i.e. only applies to cell A1). Delete any duplicate rules

5. OK

6. Go back into conditional formatting (i.e. do steps 1-3 above again)

7. For every rule, where it says 'Applies to' enter the range you want the rules to apply too. E.g. For my 5000 row table this was =$F$23:$AZ$5100

8. OK

That solved it for me. I really hope it works for others (2 hours of my life that I'll never get back...)

This is for straight ranking --

SELECT YourTable.NUMVALUE, (SELECT Count(*) FROM YourTable AS [XX] WHERE [XX].NUMVALUE >= YourTable.NUMVALUE ) AS Rank
FROM YourTable
ORDER BY YourTable.NUMVALUE DESC;

Other variations include ranking in a group and breaking ties.
