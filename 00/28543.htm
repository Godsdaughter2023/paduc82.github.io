How to create a new Identity or switch to a new Identity in Outlook 2011

Export your data as .olm file and import into a new Identity. No need to do setup accounts unless you need this as part of your testing.

If it works in the new Identity, do the complete setup.
If it fails in the new Identity, Testing in a new User will quickly tell you if the problem is system wide or if it's your User's folder that contains the problem.

Drag your Identity to Shared folder.


CREATE A NEW USER

Go to System Preferences --> Create a New User in Accounts. Switch to the New User by logging out/in or use Fast User Switching.
