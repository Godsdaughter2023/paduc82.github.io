Vlookup, Match giving the wrong result - Microsoft Community

rob.callaghan79 say:

Vlookup, Match giving the wrong result

hi, could somebody please help me with this,... 

i'm trying to look up 'orders' in the column and '6' in the row

the formula im using is  =VLOOKUP($C$5,$C$4:$P$11,MATCH($E$4,$E$4:$P$4,0),FALSE)

but instead of returning (E5) its returning (C5) - any ideas ? 

hansv mvp say:

MATCH($E$4,$E$4:$P$4,0) will return 1 since E4 matches E4, the first cell in E4:P4.

Your VLOOKUP formula will use 1 as column index, i.e. return a result from column C. To make it return a result from column E, use MATCH($E$4,$E$4:$P$4,0)+2:

=VLOOKUP($C$5,$C$4:$P$11,MATCH($E$4,$E$4:$P$4,0)+2,FALSE)

amit tandon say:

And to add, it might be a good idea to keep the columns similar (C:P), as below, to avoid confusion:

=VLOOKUP($C$5,$C$4:$P$11,MATCH($E$4,$C$4:$P$4,0),FALSE)
