Open *.tsv file in existing session of Excel - Microsoft Community

ic1234 say:

Open *.tsv file in existing session of Excel

When I try to open a *.tsv file outside of Excel, a new session starts.  Opening within Excel is no problem.

This became an issue when I upgraded to Excel 2007 - no issues in Excel 2003.

(issue continued when upgraded to Excel 2010)

jeeped say:

It sounds like you are trying to open a tab separate value (e.g. TSV) file directly from Explorer by double-clicking it and it opens in a new instance of Excel instead of the currently open Excel that you have in front of you.

This behavior is 'by design'. If you open the file using the Text Import Tool found within Data, Get External Data, From Text then you have a number of choices on where to put the data once it is brought into Excel (e.g. current worksheet, new worksheet, etc). Since you are simply double-clicking it from an Explorer folder pane, Excel has no idea where you want to put the data so it decides for you in the most non-destructive way it can; e.g. a new workbook.

Solution: use the Text Import Tool or open the .TSV in Notepad (or some other TXT editor), copy the data and paste it into the instance of Excel you already have open.
