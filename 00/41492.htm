jeremy m - support engineer say: fatbear

Boot Into Safe Mode
If the problem is being caused by a driver, booting into safe mode will tell us that the issue is with a driver. Safe mode boots into windows utilizing the bare minimum of drivers. To boot into Safe Mode: Reboot, at the manufacturer’s logo, press F8 repeatedly, then choose Safe Mode. See if the problem persists even after booting into safe mode. If the problem goes away while you are in Safe Mode, you should begin by going to device manager and looking for any yellow question or exclamation marks. If none are found in device manager, perform a system restore.

Perform a Clean Boot
A clean boot is a little different and uses a more manual approach. A clean boot will tell you if the problem is coming from any installed application. These are the instructions for performing a clean boot:

Clean Boot
Please note: You are about to disable all applications from starting up at boot time. The bottom of this paragraph will tell you how to reenable these applications.
1. Log on to your computer with an administrative account.
2. Click Start.
3. Type msconfig.exe in the Start Search box, and then press ENTER to start the System Configuration Utility.
4. If you are prompted for an administrator password or for confirmation, type your password, or click Continue.
5. In the General tab, click Selective Startup, and then click to uncheck the Load startup items check box.
6. On the Services tab, click to select the Hide all Microsoft services check box, and then click Disable all. 

7. Click ok, and then reboot

When you boot, do you still have the issue?  If this did not fix your problem, follow the steps below to turn back on all startup programs. If this fixed the problem, you need to figure out which item specifically, caused the conflict. There are basically two ways to do this: The first way is to think of any programs that you have installed around the time that the issue in question began, then go to programs and features and uninstall the applications. The second way involves going back to msconfig and reenabling half the startup programs so that you can recreate the issue again. You will need to repeat this process until you are down to a single program in question. Then, uninstall the program from programs and features in the control panel.

Reenable Startup Services and Applications
1. Click Start
2. Type msconfig.exe in the Start Search box, and then press ENTER.
3. If you are prompted for an administrator password or for confirmation, type your password, or click Continue.
4. On the General tab, click the Normal Startup option, and then click OK.
5. When you are prompted to restart the computer, click Restart.

If this does not help in finding the issue, you will want to try performing a system restore (start menu\ all programs\ accessories\ system tools\ system restore)
Or a repair install, if you have a Vista Disk. If none of these help we will know that the issue is harware, in which case, check the RAM first.

I am using a Toshiba laptop with Windows 7.

When I try to play or rip some music CD's, the sound is distorted/scratchy.

How can I fix this?

Thanks!
techie bear say:

Sound quality is a function of your audio card / drivers. Do you have a separate sound card installed? Do you have an audio chip as part of your graphics display adapter (many do)?

 

What you need to do is check the Windows System Audio properties. Right-click the speaker item near the system clock on the Taskbar, then select Recording devices. What is selected as the default device? Is it your audio card / chip? If its not, and you have an audio processor, switch your default device from whatever it is to your audio device / chip.

 

Then select Properties and make sure the recording is set for CD (or DVD) quality - play around with the various settings until you get the quality you are looking for.

 

For playback go through the same process, but select Playback devices. As a sound comparison on a laptop, you might want to try listening through either external speakers or headphones. Bottom line is that the sound quality will only be as good as the speakers / headphones you are listening to.

 

Hope this helps. Let me know how you fare.
