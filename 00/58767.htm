Restore Factory Defaults - Microsoft Community

xb1r apollo say:

Restore Factory Defaults

Login Information

  • Gateway - 192.168.1.1 or 192.168.254.254
  • Username - admin
  • Password – admin

Restore Defaults

Through firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.254.254 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  4. From the menu on the left, select Management.
  5. Select Settings from the submenu on the left.
  6. Select Restore Defaults from the next submenu.
  7. Select the Restore Default Settings button.

Through Hardware

  1. Locate the reset button on the back of the device.
  2. Using a toothpick or ballpoint pen press and hold the reset button for 30 seconds.
  3. Wait for the device to restart.

zoved say:

Yeah, did some research, the chipset in the wireless board uses USB 2.0 to connect the wireless to the host CPU.  What this means is that the motherboard has USB logic common to both the wireless card, and of course for cabling your controller directly.  Since both of these ways of connecting controllers run into the same problem, it seems to implicate the motherboard logic (that has the USB hub).  "ifixit" has a breakdown of the logic used on the wireless card, one of the chips is "Marvell Avastar 88W8782 WLAN SoC" which has USB2.0 for a host CPU interface.
