Grouping records for an email, can be done, but the process using mailmerge is fairly complex and requires a macro or a 3rd-party addin of some kind.

My Microsoft Word Catalogue/Directory Mailmerge Tutorial, available at:
http://windowssecrets.com/forums/showthread.php/154370-Microsoft-Word-Catalogue-Directory-Mailmerge-Tutorial
or:
http://www.gmayor.com/Zips/Catalogue%20Mailmerge.zip
contains a macro solution that starts of with a catalog merge to create a table of grouped records that then becomes the mailmerge data source for a normal email merge.

Another option would be to use a DATABASE field in a normal ‘letter’ mailmerge main document and a macro to drive the process. An example of this approach can be found at:

http://answers.microsoft.com/en-us/office/forum/office_2010-word/many-to-one-email-merge-using-tables/8bce1798-fbe8-41f9-a121-1996c14dca5d

johnhammon say: The solution in Word 2007 is as follows:

<CTRL> <SHIFT> <S>

This pulls up the "Apply Styles" dialog box.

Type in "Balloon Text" in the box.
If you spell the phrase right, it would un-gray the "Modify" button.
This style is hidden and will not apply on the drop down list.

Click "Modify."

The Modify Style dialog box opens.

Ensure that under properties that "Balloon Text" is the style selected.  Any clicking along the way can change the style being modified.

Chances are, in the formatting line a very small font has been selected.  Not sure how or why.

Standard size usually based off of the "Normal" style and set at 10pt.

Select any font or style you want and click "OK."

In the "Apply Styles" box, click apply.

Again, be careful not to click on anything before you apply the style.  Any random clicking will change the focus and change the wrong style.

I hope this helps.
