graham mayor say:
Do you want to delete items with attachments or permanently delete items with attachments that have already been deleted and are in the deleted items folder? Why not simply empty the deleted items folder from time to time? Storing items that you may want to keep in the Deleted items folder doesn't sound like a very smart plan. It would make more sense to create a folder structure for items that you want to keep.

However if you simply want to delete permanently messages with attachments from the deleted items folder then provided the attachments are not blocked by Outlook the following should work

Sub DeleteMessagesWithAttachments()
Dim olDeletedFolder As Outlook.Folder
Dim olDelItem As Outlook.MailItem
Dim i As Long
Set olDeletedFolder = Session.GetDefaultFolder(olFolderDeletedItems)
For i = olDeletedFolder.Items.Count To 1 Step -1
    If TypeName(olDeletedFolder.Items(i)) = "MailItem" Then
        Set olDelItem = olDeletedFolder.Items(i)
        If olDelItem.Attachments.Count > 0 Then
            olDelItem.Delete
        End If
    End If
Next i
Set olDeletedFolder = Nothing
Set olDelItem = Nothing
End Sub
