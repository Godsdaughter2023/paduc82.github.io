Hi,

This one intrigued me, so I had a play and discovered the following.

 

1) Insert a 32-point Star.

2) Slide the yellow adjustment handle so that the "height" of the star-points is zero, forming a circle or oval.

3) Try to connect a line connector to the shape... only 4 connectors available... :o(

5) Right-click on any one of the star's points and change it from Corner Point to Smooth Point.

6) Try to connect a line connector to the shape... there are now 64 points to choose from!  :o)

davidf2 say:

I am unable to reproduce your results.

I am unclear about what type of merge you are producing. If you are producing mailing labels or postcards where you are printing multiple labels or postcards per sheet of paper then be sure to mark in the print setup that you want multiple 'pages' per sheet vs multiple 'copies' per sheet.

Try another experiment for me. When you have your merge set up and you are looking under the Mailings tab, click the drop down arrow beside Finish and Merge and choose the option to Merge to a New Publication. This should create a new Publisher publication...a new file with the number of pages equal to the number of records in the merge. Before you said you would only get 20 to 30 records. If you Merge to a New Publication do you get all records? If so can you print directly to your printer from this publication? If not then try either printing that file to primopdf or just use the Publisher/Office PDF tool to save that publication as a PDF. Both work for me. If neither of these approaches works for you try the following...

As I said, I don't get the request for a new file name for each sheet but if you have the merge set up correctly and primopdf is asking for a new file name for each individual sheet of paper, then at worst you can choose to append to the first PDF file you made. When you print to primo the first time and the primopdf dialog comes up, click the options button. Check the option to ask whether to overwrite or append as per the following:

Now save the first file to your desktop or someplace you can easily find it and give it a name. Then when primopdf asks for a new file name the original name should be available. When you elect to save as that name primopdf will ask you if you want to overwrite or append that file. Choose append and it will add the additional sheets to the original. Not the best workaround but at least it should get you going.
