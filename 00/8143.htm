  1. Right-click on the Internet Explorer icon.

  4. Click OK to save your changes.

ossiemac say:

I am assuming that date2 is a string created from a date. Is my assumption correct?

You have mixed R1C1 style cell addressing with the alpha/numeric style of cell addressing. ActiveCell.FormulaR1C1 is telling Excel that it is R1C1 style addressing.

The single quotes around the sheet name are missing.

The table array apart from having the incorrect cell addressing type, preferrably should be absolute but there are circumstances where it will work without being absolute if you are not copying the formula to other cells.

Your formula addresses column 6 as the index number when the table array is only one column.

I also like to use the ampersand in lieu of the plus sign to concatenate because it avoids confusion with maths operations.

Try the following. Tested with a dummy formatted date and a worksheet named to match. Note that I have used column 1 as the column index because there is only one column in the table array.

Dim date2 As String

date2 = Format(Date, "yyyy mm dd")

ActiveCell.FormulaR1C1 = "=VLOOKUP(RC[-7],'F_CN01_" & date2 & ".CSV'!R3C3:R8C3,1,FALSE)"
