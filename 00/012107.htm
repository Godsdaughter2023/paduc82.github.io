pamcaswell say:

In order to use chapter numbering in captions (and page numbers), you must use Word's built-in heading styes linked to  multilevel list levels. 

Modify the heading styles to look the way you want.  

Put the cursor in the first heading 1 in your document. 

Click  the  multilevel list icon, and from the list library click the "card" for 1. Heading 1, 1.1 Heading 2, 1.1.1 Heading 3, ... You'll be taken back to the document. 

Without moving the cursor, again click the multilevel list icon, and then near the bottom of the  menu, click Define new multilevel list.

For level 1, change the Number style for this level to I, II, III.

When you choose to include chapter numbering in captions and specify the Chapter starts with style as Heading 2, you first caption label (such as  Figure) will be I.1-1. 

 

If you want to learn how to create multilevel list from scratch, see http://www.shaunakelly.com/word/numbering/outlinenumbering.html . 

 

HTH,

Pam

max_627 say:

Site Hierarchy Best Practices

The current SharePoint Online experience creates a new (hidden) site collection when the user selects "New Site" from the SharePoint landing page.

If you assigned the "Test" as the TItle for the Content Control (in its properties dialog) and then you set up the following code in the ThisDocument object in the Visual Basic Editor for the document

Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, Cancel As Boolean)
For Each CC In ActiveDocument.ContentControls
    If CC.Title = "Test" Then
        If Not IsNumeric(CC.Range.Text) Then
            MsgBox "You must enter a nine digit number."
            CC.Range.Select
        ElseIf Len(CC.Range.Text) <> 9 Then
            MsgBox "The number that you entered does not contain nine digits"
            CC.Range.Select
        End If
    End If
Next CC
End Sub

when the user exits from the Content Control, if the entry is not numeric or if it is numeric, but the number of digits <> 9, an error message will be displayed.
