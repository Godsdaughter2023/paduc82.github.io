a. user say:

Hi Sriraj,

A page file or paging file enables the system to remove infrequently accessed modified pages from storage to let the system use it more efficiently.

For example, when a lot of storage is installed, a page file might not be required to support the system commit charge during peak usage. The available storage alone might be large enough to do this. However, a page file or a dedicated dump file might still be required to support a system crash dump.

We recommend that you enable the page file even if you have a large amount of storage to prevent your device from crashing.

You could check the HD for recoverable errors:

  • Run chkdsk in a Command Prompt at Boot
  • Place the Vista installation DVD into the DVD drive and restart the computer.
  • Boot from the DVD into the System Recovery Options screen.
  • Select Command prompt.In the command prompt, type chkdsk C: /f /r
  • Close the command prompt.
  • Restart the computer to run chkdsk at startup.

The /r swich will cause chkdsk to detect and attempt recovery of bad sectors. The process takes quite a while.
