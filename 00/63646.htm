kreative_joypaul say:

How to Copy 5+Gb wim file in a bootable pen drive .

i configure a client computer , did sysprep and captured image ..  size of the wim file 5+ gb .. i made a 8 gb bootable pendrive but cant copy that .wim file in that pen drive. cause it is in fat32 format . any idea how to copy that 5+ gb file to a fat32 bootable pendrive ? or can we make a ntfs bootable pendrive ? 

nizambakhshi say:

Any file larger than 4GB to your pen drive even when the real capacity of the Pen Drive is higher than that. Although you can copy any size of data when its split into multiple files, but when its a single file, the system will display an error of the form like:

File Too Large

The file’ abc.xyz’ is too large for the destination file system.

As the error says, it’s the file system which can’t carry any file of such big size and there’s nothing wrong with the pen drive. This is because the FAT32 partition is not capable of handling any having size higher than 4GB. And usually a pen drive comes formatted with the FAT32 file system.

How to change the File System format of a Pen Drive and copy larger files

To transfer/copy files larger , all you need to do is to change the file system of pen drive from FAT32 to either NTFS or exFAT. To do that, follow these steps:

Note: All data on the pen drive partition will be lost permanently. Don’t forget to backup valuable data before continuing.

  1. Plug in the pen drive to your PC.
  2. Copy any important data out of  the pen drive to a place from where you can access it later on.
  3. Open Computer/My Computer (or whatever your PC is named) and right click on your pen drive’s name.
  4. Select Format.
  5. Now in the file system drop down menu, change the format from FAT32 to either NTFS or exFAT.
    We’ll recommend exFAT over NTFS, but that might make the drive incompatible with some older systems.
  6. Check on Quick Format.
  7. Finally click on Start.
  8. Your drive will be formatted and you can now copy files of any size (only up to the total capacity of your pen drive, of course) to your pen drive.

Back in the good 'ol XP days Windows added a watchdog timer to monitor the time that threads spend executing in the display driver. You can't change the default font in Wordpad,  but there is a work around.
a. Open Wordpad and change to the font that you want and then save it as a template and exit Wordpad.
b. Go to the file you just saved and make it read only to avoid accidentally writing over it. Make a shortcut to it and use this to run WordPad in the future.

It will always come up with the font you had set but you will have to save it with a new name each time.

Welcome to Microsoft Community. I appreciate your efforts towards resolving the issue. I will certainly help you with the issue that you are facing.

We need to diagnose this issue and check what is causing this on your user account. It might be due to user account corruptions. Try the following methods and check.

You may try to enable the default administrator and check if you have the sufficient privileges. If you are able to do so, create a new user account after you have logged in to the default administrator account.

To enable default administrator account, follow the steps below:

  1. Open Start, and then type cmd.
  2. In the search results list, right-click Command Prompt, and then click Run as Administrator.
  3. When you are prompted by User Account Control, click Continue.
  4. At the command prompt, type net user administrator /active:yes and then press ENTER.
  5. Type exit and then press ENTER.
  6. Log off the current user account.

(Note: If you are prompted to type in the password while opening administrator command prompt, try the password for the old user account)

To disable the default administrator account after you have done the troubleshooting, follow all the steps till 3. Then replace the command to net user administrator /active:no.

Add a new user account

Try to add a new local administrator user account to test if everything is working fine on the new account. By this we can diagnose if this is an issue with the corruptions in user profile.
