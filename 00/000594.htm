I managed to attach the Macro to a Shape (button), which I can place next to my Content Item (instead of a Bookmark). Better yet, by Copying the button I can use it more than once for the same range.


Tom,

If you want a slick way to do this, try:

Sub AllBtnClick()
    Dim rng As Range
    Select Case ActiveSheet.Shapes(Application.Caller).AlternativeText
        Case "Text on btn1": Set rng = Worksheets("Sheet1").Range("A110:B111")
        Case "Text on btn2": Set rng = Worksheets("Sheet2").Range("C110:D111")
        Case "Text on btn3": Set rng = myNamedRange
        Case Else ' do nothing
    End Select
    Application.Goto rng, scroll:=True
End Sub

All buttons call the same macro.  The range selected will depend on the text on the button.

zorvek say:

The only way to cancel an OnTime queue entry is to specify the exact time requested in the first OnTime call.

The technique described below is a safe way to start a process at a specific time or to run a repeating process at a specific interval of time. The OnTime timer has to be restarted each time it triggers if a repeating timer is required. The sample code below illustrates how to do this. In the example, the timer is started by calling StartTimer. It fires every five seconds until StopTimer is called.

Note that if using frequent Application.OnTime events, the Name Box is rendered ineffective.

[Begin Code Segment]

Private mTimerTime As Date

Public Sub StartTimer()

   StopTimer
   DoTimer

End Sub

Public Sub DoTimer()

   ' Do timer work here

   ' Reset the timer
   mTimerTime = Now() + TimeSerial(0, 0, 5)
   Application.OnTime mTimerTime, "DoTimer"

End Sub

Public Sub StopTimer()

   On Error Resume Next
   Application.OnTime mTimerTime , "DoTimer", Schedule:=False
   On Error GoTo 0

End Sub

[End Code Segment]

When setting the next timer, specific intervals at specific times can be specified. Below are some common intervals and times.

Once a minute on the whole minute.

   mTimerTime = Now() + TimeSerial(0, 0, 60 - Second(Now()) Mod 60 + 1)

Once every 5 minutes on the whole 5 minutes:

   mTimerTime = Now() + TimeSerial(0, 5 - Minute(Now()) Mod 5 - 1, 60 - Second(Now()) Mod 60 + 1)

Once every 10 minutes on the whole 10 minutes:

   mTimerTime = Now() + TimeSerial(0, 10 - Minute(Now()) Mod 10 - 1, 60 - Second(Now()) Mod 60 + 1)

Once every 15 minutes on the quarter hour:

   mTimerTime = Now() + TimeSerial(0, 15 - Minute(Now()) Mod 15 - 1, 60 - Second(Now()) Mod 60 + 1)

Once an hour on the whole hour:

   mTimerTime = Now() + TimeSerial(0, 60 - Minute(Now()) Mod 60 - 1, 60 - Second(Now()) Mod 60 + 1)

Once every three hours on the whole three hours:

   mTimerTime = Now() + TimeSerial(3 - Hour(Now()) Mod 3 - 1, 60 - Minute(Now()) Mod 60 - 1, 60 - Second(Now()) Mod 60 + 1)

Once every six hours at midnight, 6 AM, noon, and 6 PM:

   mTimerTime = Now() + TimeSerial(6 - Hour(Now()) Mod 6 - 1, 60 - Minute(Now()) Mod 60 - 1, 60 - Second(Now()) Mod 60 + 1)

Once every 12 hours at noon and midnight:

   mTimerTime = Now() + TimeSerial(12 - Hour(Now()) Mod 12 - 1, 60 - Minute(Now()) Mod 60 - 1, 60 - Second(Now()) Mod 60 + 1)

Once every 24 hours at midnight:

   mTimerTime = Now() + 1

The next day at a specific time (9:45 AM for example):

   mTimerTime = Now() + 1 + TimeSerial(9, 45, 0)

Go to www.malwarebytes.org and download, install, update and run the free version – just follow the prompts.  The prompt  asks if you want to Start trial or Decline.  To get the free version (without real-time scanning or web site blocking features) you need to choose Decline.  You may need to rename the installation file to 123.exe or something similar to prevent the malware from disabling/blocking the installation.  Or you might want to download the installation file to a flash drive or CD and then install Malwarebytes on the infected computer.

 

And/or

Try Superantispyware Portable at: http://www.superantispyware.com/portablescanner.html

SuperAntiSpyware  Portable is "designed" to be downloaded onto a flash drive (or CD) and you could then insert the drive into your computer and run it from there.  If you do that you may need to scan the flash drive for malware after using it on the infected computer.  Superantispyware is a very good scanner but will also identify and remove cookies which are not considered to be malware.

And/or

Try Hitman Pro Trial Version: http://www.surfright.nl/en/hitmanpro   This can be run from a flash drive.

And/or

Try TDSS Killer:  http://www.bleepingcomputer.com/virus-removal/remove-tdss-tdl3-alureon-rootkit-using-tdsskiller  TDSS Killer may be run from a flash drive.

 

And/or

Try the Eset Online Scanner: http://go.eset.com/us/online-scanner

And/or

Try this MS scanner: http://www.microsoft.com/security/scanner/en-us/default.aspx 

 

You may need to use one of the following offline scanners - you'll need to burn a CD using a different computer and change the boot sequence on your computer to run the scan - instructions are included with the links:
