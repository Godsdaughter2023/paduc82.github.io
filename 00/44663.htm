a. user say:

Hi,

Thank you for posting your query on Microsoft Community.

Reset your system means you performed a clean install which erased all the data related to your previous operating system. You cannot recover your deleted data if you did not keep a back of your files elsewhere. This is the reason your roll back to previous operating system has been removed and you can no longer access Windows.old folder.

Temp_SafeToDeleteRefreshFile_Attempt0__File0, this is a temporary file which is created during a restore or reset system process as the name itself suggests it is a safe to delete temporary file. This file should be deleted using a disk cleanup. Disk Cleanup will remove all the unnecessary temporary files from your system which will give your system more memory space.

Please click on the link below and follow the steps to perform a disk clean up.

http://windows.microsoft.com/en-us/windows/delete-files-using-disk-cleanup#delete-files-using-disk-cleanup=windows-8

(also applicable for Windows 10)

Note: You don't have to uninstall files using program features or System Restore. Kindly, perform only Disk Cleanup.
