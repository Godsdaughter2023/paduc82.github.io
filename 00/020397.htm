jay freedman say:

OK, that situation is quite different from the one I expected. I can't think of a practical way to create a condition based on heading text that could be anything. There is a method that works (I've tried it just now), but it's rather more complicated than just IF fields.

The first step is to add the images to the list of AutoText entries in the template used to create the document. This is described at this page in support.office.com. Give each entry a short descriptive name, and keep a list of the names to refer to later.

Also in the template, define a new character style that only sets the font color to white. Name the style something like "image" so you know what it's for.

Now, in the document that you're working on, click at the end of the first heading and insert a QUOTE field in which you include the name of the AutoText entry corresponding to that heading. For example, if the AutoText that corresponds to the first heading is named "desk", then the field should be {QUOTE desk}. Select the field and apply the character style so the result of the field turns white, making it invisible on the white background.

For each of the other headings, insert a QUOTE field naming the corresponding AutoText entry, and apply the character style.

Now open the footer pane. At the place in the footer where you want the image to appear, insert a STYLEREF field containing the name of the character style. For example, if the style is named "image", the field should be {STYLEREF image}. Select the entire field and press Ctrl+F9 to insert another pair of field markers around it. Between the first and second opening markers, insert the word AUTOTEXT. Press F9 to update the field, which should immediately show the image corresponding to the first heading.

If you don't see the remaining images, press Ctrl+P to go to the Print page and then Esc to return to the document. The print preview causes Word to update all the fields.

For the taboo words, see http://WordFAQs.mvps.org/ExcludeWordFromDic.htm. The part you want is the part for Word 2007 and above, specifically the section "Edit the dictionary." If you prepare a list of words to be flagged as errors, you can paste this into the exclusion dictionary when you have it open for editing. It is unfortunate that there is no way to edit this dictionary from within Word as you can other user dictionaries (such as CUSTOM.DIC), but this will work.

The real difficulty will be finding the correct exclusion dictionary for the language in question. Microsoft does have separate LCIDs for Nynorsk (0814) and Bokmål (0414), but if this is an issue of Høgnorsk or Riksmål or something else, then there may be a problem.

Sub AutoExec()
    Dim docReady As Integer
