View files > Date Modified, being applied to all folders, rather - Microsoft Community

raza_1976 say:

View files > Date Modified, being applied to all folders, rather than a specific folder

In Windows 7, in my documents folder, there are various folders having various files. In a specific folder if I chnage the view to sort by date modified, it applies to all the folders in the 'my documents' folder, i just want that sorting in that specific folder, not in all the folders, how to do that?
cmdrkeene say:
To explain this, I need to go a little in depth.  This behavior is because a library is not not actually a folder.  Just like ZIP files look like folders (and are called "compressed folders", the similarities are just skin-deap.  Likewise, Libraries look like folders, but they are merely search results from several locations, grouped up and displayed in a way that you can sort and see them all at once.  No files are ever stored "inside" a library.

For example, my Videos library shows me all videos from my laptop, desktop, roommate's desktop, and two external hard drives.  We can see all our movies in one library view, despite them being actually stored in many places.  Documents and Pictures libraries are exactly the same.

A library itself is nothing more than a definition of which places should be grouped up in the search results, and how they should be displayed (sort order, filters, icon size, layout, etc).  If you are inside a library and make a change, it affects everything that the library displays.

But you want individual folder views?
That's still around -- but you have to open the folder without using the library search.  You loose the cool "arrange by" button at the top-right, but you get to keep your customized per-folder views.  To open the folder directly, either create a shortcut to the folder, or you can click Start > Raza (or your name) > Documents.  This bypasses the library and opens the folder directly.

1. Boot your Windows as Safe-mode

8. Find Atheros AR9002WB-1NG, and click update the driver

9. Use the downloaded driver file, when you update the driver

Please make sure that the automatically installed or advised driver of the network device by Windows 10 is OUTDATED and INCOMPATIBLE with the new OS. You should download the up-to-date one MANUALLY from the manufacturer's website(the driver for Windows 8 also works well for Windows 10).

Access Verification

  • email accompanied by digital signature;
