Change picture of main lock screen when there is more than one account - Microsoft Community

chelsea:) say:

Change picture of main lock screen when there is more than one account

I'd really like to know how to change the picture of the main lock screen? Not the lock screen that goes to the part asking for your password to sign in, but the one that leads to choosing which account to use if you have multiple accounts.

Thanks!

rajesh govind say:

Hi Chelsea,

 

Thank you so much for posting your query on this forum. I’ll try my best to assist you with this issue.

 

I understand that you would like to change the System Lock Screen of your computer. The System Lock Screen is the screen that is displayed when the last logged in user logs out from the computer. The picture displayed on this screen is not tied up to any individual user account on the computer.

 

Unfortunately, there is not a built-in option within the Windows 8.1 interface that allows you to change the image on this lock screen. However, this can be done from the Group Policy Editor for Windows 8 Pro, Enterprise, and Server Editions.

 

For the Windows 8.1 Basic Edition, this can be done only by editing a registry key. If you want to proceed with it, then you may please follow the steps given below:

 

Change the System Lock Screen Picture in Windows 8.1

 

a) Press “Windows Logo” + “R” keys on the keyboard to open the “Run” command box.

b) Type “regedit” in the “Run” command box and press “ENTER”.

c) From the Registry Editor, navigate to the registry key given below:

 

“HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Personalization”

 

d) If the “Personalization” key does not exist under “Windows”, then right-click “Windows”, select the option “New >> Key” from that menu and then name the new key as “Personalization”.

e) In the right-hand pane for the “Personalization” key, check to see if you have a value called “LockScreenImage”.

f) If not, then create it by right-clicking anywhere in the blank space on the right pane for “Personalization” and select the option “New >> String Value”.

g) Enter the name of the string value as “LockScreenImage” and enter the location of the image that you would like to set as the System Lock Screen picture in the “Value data” field.

 

Note: Make sure that you mention the picture file name with its extension at the end of its location in the “Value data” field. For example, if the picture file is in the folder named “Picture” in D drive with the file name “Pic.jpg”, then you should enter the “Value data” as “D:/Picture/Pic.jpg”.

 

h) If the “LockScreenImage” string already existed, then you may edit the ”Value data” field by right clicking the string and selecting the option “Modify…” from that menu and then enter the correct ”Value data” and then click on “OK”.

i) When you are done, close the Windows Registry Editor. The change will take effect immediately; you need not reboot the computer.

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. Follow the steps given here to back up registry.

 

a) Press “Windows Logo” + “R” keys on the keyboard to open the “Run” command box.

b) Type “regedit” in the “Run” command box and press “ENTER”.

c) Locate and click the key or sub-key that you want to back up.

d) Click the “File” menu, and then click on “Export”.

e) In the “Save in” box, select the location where you want to save the backup copy to, and then type a name for the backup file in the File name box.

f) Click on “Save”.

 

Hope this is helpful. If you need any further assistance with Windows Operating System, please post your query on Microsoft Community Forums and we’ll be glad to assist you with it.

Windows Journal Note Writer is a virtual printer that manages the import of document images to Windows Journal notes. When you print documents to the Windows Journal Note Writer, the document is converted by Windows Journal to a document image that is used as the background for a Windows Journal note.

As per the description I understand that after installing Windows Update the Windows Journal Note Writer stopped working. There could be several reasons for causing this issue like driver corruption or corruption in the Windows Journal Note Writer. Let’s try to identify the actual cause and troubleshoot accordingly.

However, before we proceed I would require some more information to assist you better.

1. What are the KB number of the recently installed updates?

2. Check if there is any update failed or pending to install?

3. What exactly happens, when you try to use the Windows Journal Note Writer? Do you receive any error message/code?

Refer the link: See which Windows updates are installed

In the meantime, I would suggest you to repair the Windows Journal Note Writer:

To do so, follow the steps given below:

a) Click on “Start” and type “cmd” (without the quotations) in the start search box.
 
b) When you see the “cmd” icon in the start search list, Right Click on it and select “Run as administrator”.
 
c) Command Prompt Window will be open.

d) In the command prompt, change the directory to ‘Drive:\Program Files\Windows Journal’ (where drive is drive location where windows is installed).

e) By default the directory would be c:\Windows\System 32.

f) To change the directory type “cd..” (without quotes) and press Enter. This will take you to the previous folder.

g) Once you come to the c: drive, type “cd Program Files” (without quotes) and press Enter.

h) Then type “cd Windows Journal” (without quotes) and press Enter.

i) Once you are in the Drive:\Program Files\Windows Journal’ type the following command and then press Enter.

journal.exe /repairnotewriter

The Windows Journal splash screen appears briefly. After the splash screen appears, several seconds of hard disk activity follow. When the repair operation is complete, the Journal.exe process quits. You do not receive a message to notify you when the repair operation completes

j) Type exit and then press Enter to quit Command Prompt.

k) Restart the computer and check if it helps.
