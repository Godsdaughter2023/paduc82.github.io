rifdhan say: Hey ShellyLeBlanc,

As TrekDozer mentioned, the settings you are looking for are found in the router's configuration page. This page can be reached by entering the IP address of your router into an internet browser (such as Internet Explorer). However, the IP address for the router varies by model and manufacturer.

Check the packaging that came with the router to see if it mentions an IP address for the router.

If you cannot find the IP address, try following these steps:
1. Click the Start button (bottom-left), and type 'cmd' into the search box (no quotes).
2. Open the first item that appears. A Command prompt window should appear.
3. Type 'ipconfig' into the window (no quotes, no spaces, no capitals), and press enter.
4. In the list of information that appears, look for 'Default Gateway' (usually is the 5th item on the list).
5. Type the IP address listed for the Default Gateway into an internet browser. The IP address will be something like 192.168.x.x (the 'x' parts will vary from router to router).
6. This IP address should take you to the router's configuration screen. From there, you will have to find the settings you are looking for on your own.

Hope that helps,
Rifdhan

Please do the troubleshooting steps below.

To delete the shared.xml file:
1. On your keyboard, press the Windows key and R at the same time. The Run window opens.
2. Type %appdata%\skype and click OK.
3. Locate and delete the shared.xml file.
4. Open your Skype Name folder and look for config.xml. Delete this file as well.

To Re-name Skype Folder:
1. On your keyboard, press the Windows key and R at the same time. The Run window opens.
2. Type %appdata% and click OK.
