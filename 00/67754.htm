This solution worked for me:

I had what turned out to be a "device disconnect" sound that was repeating every minute (also when trying to play videos/audio).

bossdweeb say: More of a question for DELL support, but if you are inquiring about upgrading your
laptop graphics -
Most laptop graphics cannot be upgraded, with the exception of some high end models.

- If you are inquiring about which graphics processor  drivers to update to -
You need the exact model laptop to determine which graphics processor you laptop uses.

1) Your laptop had two graphics processors. One is an Integrated Graphics processor that
is part to the Intel cpu. This is the HD3000 graphics.
2) Your laptop also has a Dedicated Graphics processor. This would be one of the  models on
 the list such eg - Nvidia  525M (500m series), or  AMD HD 7570M (7000M series).

In both cases you would normally select the driver with the most recent date of highest version number that lists the exact model number of your laptop.
The exception is, in some cases people find there is a problem with a newer driver and a particular
software, so in this case an older driver is often suggested.

It is important that you do not install both AMD and Nvidia graphics drivers.
You must use the one that is appropriate for the make and model Dedicated Graphics processor
that your laptop has.
If you don't know which make and model Dedicated Graphics processor you machine has
you can find which make and series your laptop uses by typing   dxdiag   into the Run command
or Start menu Search Bar.

The same applies to the Wireless driver. If your model is one of the ones mentioned you
should download and install that as well.

NOTE: When upgrading graphics drivers the safest method for a good install is to first
uninstall the current drivers via Programs and Features - reboot.
After the reboot install the new drivers and reboot again
(you should get a prompt to reboot, but this does not always happen).
