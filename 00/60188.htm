paolo mag say:

Hello,

Windows Live Mail are prone to getting lost even if you don’t delete them. In some cases, Windows Live Mail users have reported some or all of their Windows Live Mail folders disappearing either after a Windows update or a Windows Live Mail Update or following the deletion of any one email folder. For us to resolve this issue, we suggest that you follow the methods to restore lost Windows Live Mail email folders listed below:

Method 1: Restore the folders using Compact View

  1. Open Windows Live Mail. Click on View in the taskbar.
  2. Click on Compact View. Doing so will collapse all of the Windows Live Mail folders listed and a green sign in the shape of a plus will appear under them.
  3. Click on the green plus. As soon as you do so, a dialog box that lists ALL of the Windows Live Mail folders on your computer – including the ones that you have lost – will appear.
  4. Simply check each of the lost folders that you want to restore by clicking on the checkbox beside them, and then click on OK.
  5. Once done, click on View and then click on Compact View. As soon as you click on Compact View the second time, all the email folders that you thought you had lost should appear exactly where they should be.

Method 2: Restore the folders that were lost using the import feature from the default directory where the mail’s are stored

  1. The first step is to Hold the Windows Key and Press R.
  2. Then, type %appdata%  and click OK.
  3. Once there, check the following locations:
    AppData\Local\Microsoft\Windows Live Mail
    AppData\Roaming\Microsoft\Windows Live Mail
  4. You will then see your folders in one of those directories, copy everything and create a new folder on Desktop. Name it as Back Up Live Mail or something. This will be the back up folder, the reason it is supposed to be copied is to not corrupt or mess up the existing data.
  5. Now that you've got the back up, you'll need to import the files to your Windows Live Mail.
  6. Open Windows Live Mail.
  7. Open the File menu located in the upper left corner of the window.
  8. Click on Import messages in the File.
  9. Select the file format in which you saved your Windows Live Mail backup. If you’re restoring from the back up in the above method, then use the Windows Live Mail option and click Next. Click on Browse, navigate to the folder in which your backup is stored, click on the backup file (the folder or mailbox) you want to restore to select it and then click on Next. Click on Next. As soon as you receive a message confirming the success of the restoration of the folders, feel free to click on Finish and see that the folders you restored are now visible in the left pane under Storage folders or where they were imported.

Let us know if the resolution provided works.

a. user say:

Hi,

WUDFHost.exe is Windows Driver Foundation- User-mode Driver Framework Host Process. This is related to Windows User Mode drivers. This background process is essential for Windows to communicate with hardware devices and is created automatically to load drivers. Hence wudfhost is needed for the relative program to run properly and so removal is not recommended. 

As for the issue with Windows Media Player increasing your CPU usage, I suggest that you perform the methods stated below and check.

Method 1: Perform full system scan and check if there are virus or malware infection in your PC.

Method 2: Disk Cleanup.

Windows 10 is a service that includes updates, just like Microsoft Office, anything by Google and an increasing number of software applications.

You can physically stop receiving updates e.g., by fiddling with your system settings or by breaking the operating system, but that's really unwise for at least two reasons:

XP automatically restarts on a system failure, so configure your system so it doesn't do that, then you can see the error message.

Configure your system to not automatically restart on system failure and to create a crash dump file when it does crash.

danzizme say:

"You only need one Wi-Fi Direct device to form a group [with another Wi-Fi direct compatible devices]."

The confusion comes from the incorrect universal use of the word Wi-Fi (wireless frequency) to describe a LAN (local area network)  

http://www.wi-fi.org/discover-wi-fi/wi-fi-direct

So basically your PC creates a separate LAN using Wi-Fi direct beside your LAN that is connected through the router or switch. You then have two LAN networks that are completely separate from each other. The network your router is the host and your network in which the PC, in this case, is the host and the two networks don't know about each other.

Try using this http://www.samsung.com/in/support/model/GT-S5620DKAIRD

Toward the bottom is a pc sync. I believe that will help you sync your data. The main reason to use wifi direct is in a situation where there is not host network to use to facilitate a link between the two devices. You may need to right click the setup file choose properties > compatibility tab > choose run in compatibility mode > windows 7 > select run as admin at the bottom of the properties sheet.

If the pc sync from Samsung doesn't work or you need a peer to peer network there is another way to do it using an ad-hoc network, but that feature was removed or hidden well from windows 8, probably because of the wifi direct addition. I have heard people still able to create an ad-hoc network in 8.

https://social.technet.microsoft.com/Forums/windows/en-US/56ff83ff-1f15-4fc1-aa37-6651340d46fa/windows-81-connecting-to-ad-hoc-networks?forum=w8itpronetworking

If this works you can then log in to the ad-hoc network created by the PC using your phones Wi-Fi connection and it would hopefully allow sharing.
