Here's a very simplified example of how you can do this and hopefully with the base you can figure out the rest for your exact situation.

 

Function ExpXL(strPath As String, curVal As Currency)
    Dim objXL As Object
    Dim xlWB As Object
    Dim xlWS As Object
    
    Set objXL = CreateObject("Excel.Application")
    ' you don't need to set this to true but I just did for troubleshooting.

    objXL.Visible = True

    Set xlWB = objXL.Workbooks.Open(strPath)
   
    Set xlWS = xlWB.Worksheets(1)
   
    xlWS.Range("C2").value = curVal


    ' closes and TRUE means save the workbook
    xlWB.Close True
   
    objXL.Quit
    Set objXL = Nothing

End Function

 

 

So in this case, for example, I have a spreadsheet at N:\BobL\ForumStuff\TestForFill.xlsx and my cell F1 has a formula (for calculating off of salary) which refers to C2 (salary) which I am updating.  To call this particular one, it would be something like

 

ExpXL "N:\BobL\ForumStuff\TestForFill.xlsx ", 25000
