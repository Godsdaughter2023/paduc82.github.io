t. valko say:
What does the 1E100 mean?  When inserted, the formula autocorrects to 1E+100.  The formula returns the current date, Or the serial number of the current date.


1E100 is scientific notation for the very large number 1 followed by 100 zeros.

Scientific notation is a "shorthand" method of expressing very large/small numbers.

We (I) use 1E100 to find the last (bottom-most) numeric value in column C.

MATCH(1E100,C:C)

If every numeric value in column C is less than or equal to 1E100 then MATCH will return the LAST (bottom-most) numeric value as the result.

Since we want the LAST (bottom-most) numeric value in the column we just need to use a lookup value that is guaranteed to be greater than any number in the column.

To ensure that the lookup value is greater than any number in column C we (I) use the arbitrary gigantic number 1E100. It's easy to remember and easy to type!

doug robbins - mvp office apps & services (word) say:

Greg,

How about

Private Sub Label1_Click()
Label1.caption = InputBox("Enter new caption")
End Sub
Private Sub UserForm_Deactivate()
SaveSetting("Word", "UserForm4", "Label1Caption") = Label1.caption
End Sub

Private Sub UserForm_Initialize()
Dim strCaption
strCaption = GetSetting("Word", "UserForm4", "Label1Caption")
If strCaption = "" Then
    Label1.caption = "Click to enter caption"
Else
    Label1.caption = strCaption
End If
End Sub

You need to .Hide the form for the Deactivate to fire
