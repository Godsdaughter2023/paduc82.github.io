Although this article claims not to apply to XP, the error message described by it matches the error message that you describe.  Perhaps the solution is the same:

"Windows Server 2008 or Windows Server 2003 machine still honors the Registry Size Limit (RSL) functionality if the RegistrySizeLimit registry key is set"
   < http://support.microsoft.com/kb/2567018 >

Your error message is the last one described.

If this doesn't help, you might try squeezing out the slack space in your registry.  When registry keys are deleted, the space is maintained in the registry and new entries are still added at the end.  There is a freeware program "NTREGOPT" that will squeeze this free space out of your registry, hopefully making it small enough not to trip your warning messages.  Note that this is not a "Registry Cleaning" tool as it will not remove any active entry already in your registry.  A good precaution would be to use ERUNT (same site) to back up your registry before compacting.

NTREGOPT:  < http://www.larshederer.homepage.t-online.de/erunt/ >
