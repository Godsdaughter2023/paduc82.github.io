I ask because when I upgraded to Windows 7 (from XP) I was surprised to find that Windows 7 Professional does not support three monitors. I want to avoid unpleasant surprises.

a_user_here say:

Windows 10 can support 3 and more monitors (probably the same amount as Windows 7), but if you are looking at some of the posts and seeing where people are having such problems, the most likely case is something to do with the drivers.  For instance I have a AMD APU (CPU & GPU combined) that can display on three monitors.  BUT as it turns out the default Microsoft video driver will only display on two of them.  The driver actually meant for this APU will be able to use all three.  So what happens for me is for an initial install only two monitors work.  And then in a short time Windows downloads the correct/non generic one and they all work.

I suspect that having is because 3 displays off of one GPU is pretty unusual.  It sounds like you have a more conventional case where you have at the most 2 displays on one GPU, so it probably will never see this.
