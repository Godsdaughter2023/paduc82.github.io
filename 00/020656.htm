The solution below lets you add as many textboxes as desired.  For the color selections, change the values assigned to BaseColor, SelectedColor, and DoneColor.

Put the code below in a standard module in the VBAProject of the presentation.

Add however many textboxes as you want.

Add one more textbox and set the text in the box to 'Select' (w/o the quotes).

WIth this textbox selected assign a 'on mouse click' action to run the macro RandomSelect.

Run the slideshow and click on the Select textbox.

For a more 'professional' look and feel one would have to use event procedures.  Unfortunately, that is beyond the scope of this forum.

 

Option Explicit
Dim BaseColor As Long, SelectedColor As Long, DoneColor As Long
Private Function NewShapeColl() As Collection
    BaseColor = RGB(0, 0, 255)
    SelectedColor = RGB(255, 0, 0)
    DoneColor = RGB(233, 233, 233)
    Randomize
    Dim Rslt As Collection: Set Rslt = New Collection
    Dim Y As Shape
    For Each Y In ActivePresentation.SlideShowWindow.View.Slide.Shapes
        If Y.Type <> msoTextBox Then
        ElseIf Y.TextFrame.TextRange.Text <> "Select" Then
            Rslt.Add Y, Y.Name
            Y.TextFrame2.TextRange.Font.Fill.ForeColor.RGB = BaseColor
            End If
        Next Y
    Set NewShapeColl = Rslt
    End Function
Sub RandomSelect()
    Static MyColl As Collection
    Static OldShape As Shape
    If MyColl Is Nothing Then
        Set MyColl = NewShapeColl(): Set OldShape = Nothing
    ElseIf MyColl.Count = 0 Then
        Set MyColl = NewShapeColl(): Set OldShape = Nothing
        End If
    If MyColl.Count = 0 Then Exit Sub
    Dim Idx As Integer
    Idx = Fix(MyColl.Count * Rnd()) + 1
    Dim X As Shape: Set X = MyColl.Item(Idx)
    X.TextFrame2.TextRange.Font.Fill.ForeColor.RGB = SelectedColor
    If Not OldShape Is Nothing Then _
        OldShape.TextFrame2.TextRange.Font.Fill.ForeColor.RGB = _
            DoneColor
    Set OldShape = X
    MyColl.Remove Idx
    With ActivePresentation.SlideShowWindow.View
    '.GotoSlide .CurrentShowPosition
        End With
    DoEvents
    End Sub

Hi Jon,

To create the desired sheets you could use the following code:

'---------->>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet
    Dim i As Long, j As Long
    Dim arr As Variant
    Const startDate As Date = #7/27/1984#
   
    arr = VBA.Array("Mid-day", "Evening")
    
    On Error GoTo XIT
    Application.ScreenUpdating = True
    
    Set WB = ActiveWorkbook
    With WB
        For i = 1 To Date - startDate
            For j = LBound(arr) To UBound(arr)
                Set SH = .Sheets.Add(After:=.Sheets(.Sheets.Count))
                SH.Name = Format(startDate + i - 1, "Mmmm dd, yyyy") & " " & arr(j)
            Next j
        Next i
    End With

XIT:
Application.ScreenUpdating = True
End Sub
'<<==========

After doing resrach, I have found that we can add a new Registry key to Windows to archive your goal:

2. Go to: HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Outlook\Preferences
