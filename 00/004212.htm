julie sheets say:
If the working time in your project is Monday through Friday - then tasks can span the weekend but no work will be scheduled.

If you want to count calendar days, create a custom calendar (7Days) and modify the Work Week to make Saturday and Sunday working days.

In a spare duration field create the following formula:

ProjDateDiff([Start],[Finish],"7Day")

This will show the number of calendar days between the task start and finish using all 7 days as working.

Keep the project calendar as M through F to keep the tasks scheduled correctly.

Regarding your specific example - set the duration of the sample in conditioning chamber to 30 ed. Link the conditioning chamber task to the evaluation task the chamber task may end on Saturday - because of the "ed" but the evaluation tasks can only be scheduled Monday through Friday as it is not using elapsed days for duration.
