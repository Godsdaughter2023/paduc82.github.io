It depends on whether or not you want to reduce the size of the page content. If you do, then by far the simplest way is to save the document as a PDF, then use the print settings in Adobe Reader to print it as a booklet.

Alternatively, you can use the "2 pages per sheet" setting in Word's Print dialog, inputting the page numbers in the order required for booklet printing. Note that the total number of pages must be divisible by 4, which means you'll need to add pages if there are just 5 or 10.

athyeh say:

VBA Code that copies rows with dates in column B 30 days older than date in cell B2

Hi Folks,

I am looking for VBA code that will evaluate the date in Cell B2 (11/12/2013) and then moving through the dataset, only copy those rows with dates in Column B that are 30 days older than 11/12/2013 (Row 2 and 4).

Thank you all for your replies and insight.

A B
1 11/12/2013
2 1/1/2013
3 11/11/2013
4 5/1/2013

rich007a say:

Hi,

You want to copy the entire row?

And where do you want to copy it to?

This should get you started:

Sub CopyRows()
    Dim c As Range, rng As Range
    Dim lnLastRow As Long, lnDestRow As Long
    Dim wsSrc As Worksheet, wsDest As Worksheet
   
     'Get references to Source and Destination worksheets:
    Set wsSrc = Worksheets("Source")  '<<-- Update sheet name!
    Set wsDest = Worksheets("Dest")   '<<-- Update sheet name!
    
     'Get range in col B of Source worksheet for cells to check:
    lnLastRow = wsSrc.Range("B2").End(xlDown).Row    'get last row with data
    If lnLastRow = wsSrc.Rows.Count Then Exit Sub 'no data below B2
    Set rng = wsSrc.Range("B3:B" & lnLastRow)
   
     'Now get next empty row in destination sheet col A:
    lnDestRow = wsDest.Cells(wsDest.Rows.Count, "A").End(xlUp).Row + 1
   
     'Loop through col B:
    For Each c In rng
        If c.Value < Range("B2").Value - 30 Then        '(you might want to use   <=   here)
            c.EntireRow.Copy Destination:=wsDest.Range("A" & lnDestRow)
            lnDestRow = lnDestRow + 1
        End If
    Next c
End Sub
