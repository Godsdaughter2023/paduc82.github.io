xircal say:

Are you talking about the WPA-2 key and the SSID? If so, you need to change those on your router.

To access the router setup, do the following:

  1. Right click the Start button and click Control Panel.
  2. Click the link to Network and Sharing Center. (If you have the category view enabled, change it to Small icons first via the link top right first).
  3. Click the link to "Change advanced sharing settings".
  4. Click the radio button called "Turn on network discovery" and then click the Save changes button.
  5. Launch your browser and type the gateway address to your router which will be something like http://192.168.1.1 (it will tell you in the router manual what the correct gateway is).
  6. Type the router login name and password. If you're using the default login and password, it's advisable to change both of those to give yourself additional security.
  7. Navigate to the Wireless setup menu and you should be able to see both the WPA-2 key and the SSID.
  8. After you save changes and exit the router setup, go back to Network and Sharing Center | Change advanced sharing settings and click the radio button to Turn off network discovery. Save changes to exit.

As an added precaution, print a label with the details you changed and stick somewhere on the router in case you forget them.

Oh, and if you do change the login name and password, you usually have to type them both since copy/paste won't work.

You can select the audio files in a folder, then
