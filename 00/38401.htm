The problem is that the RTN online newspaper is displayed as a Flash video, which essentially can not be copied and pasted.  Because you said that you have been copying things for years, I assume that RTN recently changed how they provide the online newspaper.

You have two options.

1.  You can print the pages in which you are interested and keep them in a file or loose-leaf notebook.  Use the "print" icon in the newspaper's display, not your browser's File > Print menu or the browser's print icon.

2.  Download and install free CutePDF Writer and the free Ghostscript GPL converter from CutePDF.  It's probably best to log in as a user with "computer administrator" privileges to do the installation.  See the FAQs.

After you install CutePDF Writer, whenever you print something, one of the choices for which printer to use will be CutePDF Writer. 

So ... print the pages of the RTN online newspaper as in 1 above but use CutePDF Writer rather than your "real" printer.  A dialog box will open asking you to designate a file name and location where the output will be stored.  After a moment, you'll have a pdf document (rather than a Word document) of your selected pages.

c. In the Startup Type list, click Automatic, and then click OK.


d. Close Services.


e. Restart the computer.

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly.
