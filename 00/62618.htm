The pop-ups will be gone once the spyware is erased.


 

Stop System Process
  • 1

    Press "Ctrl," "Shift" and "Esc" keys at the same time to bring up the Windows Task Manager.

  • 2

    Click on the "Processes" tab to view a list of the current programs running in the background.

  • 3

    Find the following entry and click on it: "PC-Cleaner.exe."

  • 4

    Click on the "End Process" button to stop PC Cleaner from running. Then close the Task Manager.

 

Unregister DLL File
  • 1

    Go to the "Start" menu and click on "Run."

  • 2

    Type "cmd" in the search line and click on "OK" to start the command prompt dialogue window.

  • 3

    Type regsvr32/u pcsd.dll" and press "Enter" to unregister the DLL. Close the command line window.

 

Remove Registry Entries
  • 1

    Go to the "Start" menu and click on "Run."

  • 2

    Type "regedit" and click on "OK" to bring up the Windows Registry Editor.

    HKEY_USERS\S-1-5-21-1172441840-534431857-1906119351-500\Software\Microsoft\Windows\CurrentVersion\Run\"PC-Cleaner" = ""C:\Program Files\PC-Cleaner\PC-Cleaner.exe" hide"

rajesh govind say:

 

Hi,

 

I understand that the keyboard cursor stays always highlighting without blinking on all the desktop apps, except the browsers. This issue could be due to the recent changes in the keyboard settings. This issue happens when the blink rate of the cursor is set to the least value or 0. You may refer to these steps to reset this setting.

 

a)       Press “Windows Logo” + “X” keys on the keyboard and choose “Control Panel” from that menu.

b)       In the “Control Panel” window, open the keyboard settings by clicking on the “Keyboard” icon.

c)       Normally, on the “Speed” tab, (the name of the tab may vary), check if you have any settings to increase or decrease the cursor blink rate.

d)       You should be having a slider from the least value (Slow) to the highest value (Fast).

e)        You may select an appropriate blink rate for the cursor by moving the slider from the least value to the highest value.

f)        Click on “Apply” and then “OK”.

g)       Close the “Control Panel” window and then check if that resolves the issue.

 

Hope this is helpful. Please get back to us with the result.
