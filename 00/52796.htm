Hi Mol21,

If you have an issue with temporary profile follow the steps below:

1) Repair a Corrupt User Profile

- Go in safe mode
- Go to Search, type regedit and open Registry Editor.
- When Registry Editor launches, navigate to the following key:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList.
- Click each S-1-5 folder and double-click the ProfileImagePath entry to find out which user account it relates to.
- Once you have determined where is the corrupt user profile(and it doesn’t have a .bak ending), double-click RefCount and change the Value data to 0 and click OK. If this value isn’t available, you might have to create it manually.
- Now double-click on State, make sure the Value data is again 0 and click OK.
- Close Registry Editor and restart your PC.

2) Create a new user profile

- Restart your computer and boot into Safe Mode.
- Open Command Prompt (by right-clicking on the Start Menu button, and choosing Command Prompt (Admin))
- Type in the following command and press Enter:
net user administrator /active:yes
- You should get “The command completed successfully” message. Reboot your - computer

After:
- Go to Settings, and then to Accounts.
- Go to Family and other users. Click on Add someone else to this computer.
- Follow the instructions and finish the creation of new users profile

Next step: transfer your data from the old users account to it. And here’s how to do that:
- Restart your computer, and login to the admin account again (it’s better to use this account, than account you just created)
- Navigate to the corrupt user account (it’s on the disk you have Windows installed on (usually C:), under Users)
- Open your corrupted user profile and copy content form it to your new user profile
- Once everything is done, logout from your admin account, and log back in to the new account, and your settings should be transferred)

3) Perform DISM and SFC scan

- Open Win + X menu by pressing Windows Key + X. Now select Command Prompt (Admin) or PowerShell (Admin) from the list.
- When Command Prompt opens, enter sfc /scannow and press Enter to run it.
- The SFC scan will now start. This process can take up to 15 minutes, so don’t interrupt it.

As well you can use DISM /Online /Cleanup-Image /RestoreHealth. Keep in mind that DISM scan can take up to 20 minutes or more to complete, so be sure not to interrupt it.

4) If issue is related to Windows update

- Start PowerShell as administrator.
- When PowerShell starts, enter Get-AppXPackage -AllUsers |Where-Object {$_.InstallLocation -like “*SystemApps*”} | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”} and press Enter to run it.
- After doing that, the problem with corrupted user profile should be resolved.

Let me know if you need more help

louie del say:

Hi,

The most convenient way of copying contacts is through the use of the CSV file format. You can either import or export from programs that store data in tables. If you’ve already exported the address book in CSV file and import it to Windows 10 Mail and People app, we suggest that you use Outlook.com to sync the contacts. Here are the steps that you can follow:

  1. Launch Microsoft Edge browser or Internet Explorer. Then, go to the Outlook.com website.
  2. Sign in to your Microsoft account.
  3. Click the Settings menu.
  4. Click People.
  5. Click Import contacts.
  6. Click Browse. Look for the CSV file that you’ve saved.
  7. Click Open, then select Upload.

After you’ve successfully imported the contacts, launch the People app and add the same Microsoft account that you’ve used when you signed in to Outlook.com, then contacts will be synced already.
