Right after posting the question I found one work-around. I had used this before but ended up in errors within outlook. What I found that the errors came since I had too many devices (10) marked at OWA for syncing devices (partly due to multiple tries during figuring this all out).

So, if anyone is with similar problem:

1. Check at OWA->Settings->Phone-> that you have less than 10 devices listed. For me, many of the devices were same but likely from different installs or something.

2. Go to Control Panel, Mail -app. Add the profile and then email, manual settings. Put your server details. When clicking Next, it will give you error "Log onto Exchange ActiveSync mail server (EAS): The server cannot be found". Just click Close. Despite the error, the email has already been succesfully created.

3. Start task-editor, locate the Mail-task (might be under Windows host process Rundll32), and end process. This will close the Mail -window.

4. Start Outlook and there you have it, it is again connecting to hosted Exchange server..

A dirty trick maybe, but at least a temporary work-around.

talking moose say:
If I recall correctly, the Make This The Default View option is specific to each folder and to new folders you create. That means new folders you create will inherit the settings from their parent folders. The Inbox, Drafts, Sent Items, etc., have no parent folder.

Create a custom arrangement first, apply that to each folder one at a time and then you can make that view the default so that if you ever change the view for that folder you can quickly reset the view to the default using View menu --> Arrange By --> Custom Arrangements --> Restore to Defaults....
  1. In any folder go to View menu --> Arrange By --> Custom Arrangements --> Edit Custom Arrangements...
  2. Create a new arrangement and set a sort order that you prefer.
  3. Visit each folder. Select View menu --> Arrange By --> Custom Arrangements --> Your New Arrangement.
  4. Select View menu --> Arrange By --> Make This The Default View.
Note that the Preview Pane is not affected by either Custom Arrangements or Restore to Defaults. You'll have to set that separately.
