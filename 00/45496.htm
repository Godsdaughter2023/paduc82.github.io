lemp say:

The AppData folder is hidden (also, there's no space in AppData).  That's why you don't see it.

Outlook contacts are located at C:\Users\<username>\AppData\Local\Microsoft\Windows Live\Contacts

Two ways to get there:

1.  Type the following in the Start > Search box and press Enter

        %USERPROFILE%\AppData\Local\Microsoft\Windows Live\Contacts

2.  Open Windows Explorer

  • Navigate to C:\Users\<username>
  • Click Tools > Folder Options
  • Select the View tab
  • Select the radio button to "Show hidden files, folders, and drives"
  • Click OK
  • You'll now see the AppData folder and can navigate the rest of the way to the Windows Live\Contacts folder described above

I had this problem as well and I fixed it by changing drivers. I confirmed the problem was the driver using the Microsoft debugging tool WinDbg.

Scroll down to where it says "get debugging tools" and follow the instructions to install WinDbg. 
After doing so, run WinDbg as admin (right click - run as admin) 
Click File - Open Crash Dump (Ctrl+D) 

In the pop-up window, browse to C:\Windows and click Open
What happens next, is that the debug tool will process the memory dump from the crash and give you information back.

In my case, it listed 

*** ERROR: Module load completed but symbols could not be loaded for bcmwl63a.sys
Probably caused by : bcmwl63a.sys ( bcmwl63a+e1d1f )

After some googling I found out that it was a Broadcom driver for my Wi-Fi adapter.
I confirmed this by going into Device Manager (Control Panel\ Device Manager) 
I expanded my network adapters tree and right clicked my adapter (ASUS PCE-AC68) and selected properties.
Then I went to the Driver tab, and clicked "Driver Details".
In my case one of the driver files listed was bcmwl63a.sys, which is the same as the problem file listed in the Memory dump I had opened.

So downloaded a different version of the driver and the problem is fixed.

To those with my same adapter, I went from driver version 2.1.1.4 to 2.1.1.5.
I also noticed that beneath the driver version there are region notes.
