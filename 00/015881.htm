dianeoforegon say: You have several options.

Note: If you have IMAP accounts all mail can be downloaded into any client you choose in Mountain Lion.

1)  In your current 10.6.8, have Apple Mail import your data from Entourage 2004. Mountain Lion will auto upgrade Apple Mail for you. Sync Entourage to Address Book and iCal to get that data. Note: in Mountain Lion there is no easy way to go between Apple Mail and Microsoft Entourage or Outlook.

2) Both Entourage 2008 and Office 2011 will import from Entourage 2004 BUT if there is corruption, not all data will import.

a) While you can still boot Entourage 2004 you need to export all data as .rge file. You should get Done at the end. If you get 'one or more items failed to export' this indicates corruption.

If there is corruption, export each item (messages, contacts, events, notes, tasks) individually as .rge file. This will help identify where the corruption is located.

If it's messages, you can use this script to help locate and clean up your data.

Export folders as MBOX files (includes subfolders)

The script will fail on a folder with corruption. If there is no corruption the script will export all folders. Once you find corrupted folder, rename it with z in front of the name so it sorts at the bottom then run the script again. Keep repeating until you have identified all corrupt folders. Once you have identified corruption, you can subdivide folders and keep running script until you get down to 25 messages. Then you drag individual messages to the Desktop. A corrupt message will crash Entourage. After you clean all folders and reduce the size of the Inbox and Sent folders, do the File > Export as .rge file and you should get "Done". How do I find and UNBLOCK a blocked contact

How do I find and UNBLOCK a blocked contact
jomel_g say:

Hi

Depending on the version you're using, below are the steps to unblock a contact:

  • On Skype for Windows 10 version 12.xx:
  1. Click your profile picture and click Settings.
  2. Scroll down and locate Contacts.
  3. Click Manage blocked Contacts and select the contact you wish to unblock.
  4. Click Unblock and select Done.

  • On classic Skype version 7.41:
  1. In the main menu, select Tools > Options.
  2. In the Skype Options box, select Privacy > Blocked Contacts. A list of blocked contacts is displayed.
  3. Select the contact you'd like to unblock and click Unblock this person.

  • On new Skype version 8.xx:
  1. Select your profile picture.
  2. Select Application Settings.
  3. Scroll down to Privacy and select Manage blocked users.
  4. Select the Unblock button next to the contact you wish to unblock.

Let me know if you have further questions.
