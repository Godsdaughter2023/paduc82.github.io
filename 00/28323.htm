This can be done with a VBA Macro.  The Macro assumes your data is
either in the above format, or possibly there are no titles at all
(hence no commas).
 
If you have other formats that need to be handled, you will need to
give examples.
 
The Macro assumes the names to be parsed are in column A (and there is
nothing else in Column A) and begins in A2.  It places the results in
Columns B & C.  These assumptions can be easily changed.
 
To enter this Macro (Sub), <alt-F11> opens the Visual Basic
Editor. Ensure your project is highlighted in the Project
Explorer window. Then, from the top menu, select
Insert/Module and paste the code below into the window that
opens.
 
 
To use this Macro (Sub), <alt-F8> opens the macro dialog
box. Select the macro by name, and <RUN>.
 
=====================================
Option Explicit
Sub ParseName()
    'Last Name + suffixes
    'Last Name is last word before optional comma
    'First Name, Middle Name or Initials
Dim rg As Range, c As Range
Dim re As Object
Const sPat As String = "(^.*?)\s+(\w+(,.*)?$)"
 
'Assume Names in Column A
Set rg = Range("A2", Cells(Cells.Rows.Count, 1).End(xlUp))
Set re = CreateObject("vbscript.regexp")
    re.Pattern = sPat
    re.Global = True
For Each c In rg
    Range(c(1, 2), c(1, 3)).ClearContents
    c(1, 2).Value = re.Replace(c.Text, "$2")
    c(1, 3).Value = re.Replace(c.Text, "$1")
Next c
End Sub
====================================
 

rick rothstein, mvp - excel say:

This should do what you want (just change the StartRow constant in the Const statement to the row number containing the first name and the NameColumn to the letter of the column holding the names)

Sub SplitOutLastNames()
  Dim X As Long, LastRow As Long, SpacePosition As Long, CellValue As String
  Const StartRow As Long = 2
  Const NameColumn As String = "A"
  For X = StartRow To Cells(Rows.Count, NameColumn).End(xlUp).Row
    CellValue = Cells(X, NameColumn).Value
    SpacePosition = InStrRev(CellValue, " ", InStr(CellValue & ",", ",") - 1)
    Cells(X, NameColumn) = Mid(CellValue, SpacePosition + 1)
    Cells(X, NameColumn).Offset(0, 1).Value = Left(CellValue, SpacePosition - 1)
  Next
End Sub

debbie_h say:

You can set Bing Search as your default Search engine in Internet Explorer.

 

To set Bing as your Default Search Engine in Internet Explorer 9, do the following:

 

1.       Launch Internet Explorer 9 and press F4

2.       In the drop down menu, click on Add in the lower right side of the menu which will take you to

          the Internet Explorer Gallery

3.       Select Bing from the list of Search Providers

4.       In the confirmation box that appears, select the ‘Make this as my Default Search Provider’

          check box

5.       Click Upgrade

 

To set Bing as your Default Search Engine in Internet Explorer 10, do the following:

 

1.       Launch Internet Explorer 10

2.       Click the Tools Gear and select Manage add-ons

3.       Select Search Providers in the Add-on Types column to view the current search engine selection

4.       If you don’t see Bing listed, click on the ‘Find more search providers’ link  on the bottom left

          portion of the window which will take you to the Internet Explorer Gallery
