Superfetch was designed to adapt to your PC usage to assist with faster application loading and faster boot times, although, in later versions of Windows it has been found that it causes the disk usage to increase severely.  I just recently ran into this issue with my wife's PC and had to stop superfetch. Once I did that, the disk usage went from 100% down to nearly nothing. If you are still having issues with this, here is how to stop it:

open cmd as admin > net.exe stop superfetch > Enter

With task manager opened in parallel, you will see your disk usage drop almost instantly.  So far I have not seen any downsides to stopping this and if you have read about it online, most claim it will slow your PC down if stopped.

To clarify the different email platform that we have, please see the details below:

  • Windows Live Mail is a free program that offers email capabilities and a calendar. It doesn't have a programming interface and doesn't work with Microsoft Exchange. Unfortunately, we end the support of this platform this year.
  • Outlook Mail for Windows 10 is a new application for Windows 10 that provides a good experience for both touchscreen and desktop users. It works hand in hand with the Calendar app. It's part of the default installed on Windows 10 and if you log in with a Microsoft account it may already be partly configured (especially if you use MS as your e-mail provider).  It provides basic email and calendaring features at home or work, best suited for your Windows mobile devices.
  • Outlook.com is a free web-based email service from Microsoft. Your Outlook.com address connects you to more than just your inbox. Sign in to Skype, find your contacts and Calendar and work in Office Online.
  • Outlook.live.com is an Outlook Web App, a service that provides web clients for popular Microsoft Office Desktop products, like Word, Excel and PowerPoint. Like Outlook Web App, it’s available in an on-premises version, complimentary to SharePoint and a cloud version as part of Office 365 for Business.

If you have other questions, feel free to reach out to us.
