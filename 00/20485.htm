stefan blom say:

If the document includes shading rather than highlighting and the shading is part of one or more styles, it won't go away as you select content and press Ctrl+SpaceBar and Ctrl+Q. Try the following macro:

      

Sub ClearAllShadingFromDoc()
    'Macro created by Stefan Blom, MVP, September 2016
    ActiveDocument.Content.HighlightColorIndex = wdNoHighlight

    With ActiveDocument.Content.ParagraphFormat.Shading
        .Texture = wdTextureNone
        .ForegroundPatternColor = wdColorAutomatic
        .BackgroundPatternColor = wdColorAutomatic
    End With
   
    With ActiveDocument.Content.Font.Shading
        .Texture = wdTextureNone
        .ForegroundPatternColor = wdColorAutomatic
        .BackgroundPatternColor = wdColorAutomatic
    End With
End Sub

If Word crashes, then as Stefan implies, temporary files will be left behind, and, dependent on configuration settings,  the auto-recovery will attempt to recover the crashed files. Start by deleting those orphaned files. See http://www.gmayor.com/what_to_do_when_word_crashes.htm   

The normal template is (by default) stored in the folder  copy
%appdata%\Microsoft\Templates
Copy this to the address window of Windows File Explorer to take you to this normally hidden folder.

The message - "Word automatically saved the changes to the normal template. Do you want to load it?" indicates that a change was made to the normal template that had not been saved and the auto-recovery had made a change, which it now offers you to accept. In the ciurcumstances, answer 'No' to this prompt.

When you leave Word and you get the prompt "Changes have been made to the global template, Normal. Do you want to save those changes?" Save the changes.

Word uses the normal template as a configuration scratch pad. At various times you may make changes while editing a document that are reflected as changes that can be saved in the normal template. Depending on the configuration you will get a prompt to save those changes on exit (the alternmative being to save them automatically whether you require them or not); so this is essentially a good thing and you can choose to save the changes or not at the prompt.
