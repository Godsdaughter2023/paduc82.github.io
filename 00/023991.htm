tzon say:

Office 2013 Greek proofing tools (spell check) point out as incorrect correct words when written in upper case (capital letters)

There seems to be a problem (and I guess it's serious) with the Greek Proofing tools (or spell checker) in Office 2013.

 

When a Greek word is written in upper case it is automatically underlined as misspelled.

 

Greek words get stresses (or "tones"), symbols that indicate where exactly the word is emphasized. But when written in upper case the stresses are omitted. Apparently this causes a huge (and new) confusion in Office 2013. Earlier versions work just fine. For example

 

The word good-morning in Greek is spelled:  καλημέρα (lower case) and KΑΛΗΜΕΡΑ (upper case)

 

but office will recognize as incorrect the correct upper case word and suggest as correct ΚΑΛΗΜΈΡΑ , a word written in upper case containing a stress (on E) which is totally incorrect.

 

Selecting the option "ignore words in uppercase" in MS Word will just ignore the problem but is not an acceptable solution. All previous versions of Office would recognize and spot correctly spelling mistakes both in lower and upper case for the Greek language.

 

Any help appreciated!

The Primary Domain / URL of Yammer us synced with the default domain set in your office 365 admin center.

Starting October 16, 2018, Yammer will no longer support multiple Yammer networks associated with one Office 365 tenant. You can perform a Network Migration to get to a preferred 1:1 configuration.

  • To view a list of the networks associated with your tenant, log in to Yammer as an Office 365 Global Administrator and browse to Network Admin, then Network Migration. Choose Next.
  • If you see multiple networks listed on Step 2 of 3, then you have multiple Yammer networks associated with your O365 tenant.
  • To correct your configuration to a 1:1 configuration, continue using the Network Migration tool.
  • For more information on Network Migration please see Network migration: Consolidate multiple Yammer networks
