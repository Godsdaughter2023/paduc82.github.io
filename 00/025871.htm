Paul has explained that if you change the DATE field to CREATEDATE, it will reflect the date the document was created (assumed to be the date the field was inserted). macropod say:

Word has four kinds of date field:

• DATE - displays today's date

• CREATEDATE - displays the date the file was created, or last saved via Save As

• PRINTDATE - displays the date the file was last printed

• SAVEDATE - displays the date the file was last saved

If you select your updating date and press Shift-F9 you'll see something like { DATE }. If you change 'DATE' to any of the others and press F9 to update the field, you'll see that date instead.

When inserting an auto date, selecting it and pressing Ctrl-Shift-F9 will convert it to plain text so it will never update.

tisky zheng msft say:

Hi Guy,

Could you please provide the screenshot of your Page Break Preview mode?

Besides, please check the Page Break related settings based on the following steps:
1. Remove all page breaks.
   1) Click the worksheet that you want to modify.
   2) On the View tab, in the Workbook Views group, click Page Break Preview.

   3) On the Page Layout tab, in the Page Setup group, click Breaks.
   4) Click Reset All Page Breaks.
2. To return to Normal view, on the View tab, in the Workbook Views group, click Normal.

After working with page breaks in Page Break Preview view, you may still see the page breaks in Normal view because page breaks have been turned on automatically. You can turn them off.

About how to display or hide page breaks in Normal view:
1. Click the File tab.
2. Click Options
3. In the Advanced category, under Display options for this worksheet, select or clear the Show page breaks check box to turn page breaks on or off in Normal view.
