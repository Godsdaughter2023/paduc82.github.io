Hi Edtes2,

Basically, your hunch is correct and here are the focal points in the Delve cards of PowerPoint presentations:

If we use one of the default templates with a Title layout slide (condition), we will get:

1. Title from the Title layout slide.

: Do not use a title slide for a “Thank you” or similar for the last slide. This can overrule smaller titles from slides earlier in the presentation.


2. The Title from the first slide.

3. Text using a large font. Even if you have a Title from Title layout or first slide Title, text in a larger font than what’s used in the styles can be picked up as the title(Priorities).

4.  If there’s no likely title found in the document (condition), the Title property is used. The Title property is usually prefilled with “PowerPoint presentation” unless you change the Title property when you upload the document to SharePoint (condition).

5. If you haven't set the Title property (condition), the file name is used.

Hi All,

 

Thanks for the replys i have now come up with a solution after hunting the world wide web.

 

The solution i used was to use a command button and assign it the Alt + Tab keys to it so when in show it goes to the last viewed window.

 

The code i used is shown below, Hopefully this may help others like me (Basic Knowledge)

 

Private Declare Sub keybd_event Lib "user32" (ByVal bVk As _

    Byte, ByVal bScan As Byte, ByVal dwFlags As Long, ByVal _

    dwExtraInfo As Long)

Private Const KEYEVENTF_KEYUP = &H2

Private Const VK_ALT = &H12

Private Const VK_TAB = &H9

Private Sub CommandButton1_Click()

 

' Grab all of the text in the WebBrowser control.

    ' Press Alt.

    keybd_event VK_ALT, 0, 0, 0

    DoEvents

 

    ' Press Tab.

    keybd_event VK_TAB, 1, 0, 0

    DoEvents

 

    ' Release Alt.

    keybd_event VK_ALT, 0, KEYEVENTF_KEYUP, 0

    DoEvents

End Sub

I prefer it too...

You can't "backup" an installed copy of Office. All you can do is install it on a new machine.

Sounds like you have an "OEM" type license, bought installed and activated on the computer.  They are trivially cheaper, but there are 2 limitations on the license: the install is permanently tied to that computer hardware, and it is only allowed to install on that one computer. 

To figure out if it is OEM, go into one of the Office applications and go into File menu > Help command. Look over to the right, near the bottom is a field called "Product ID". It is displayed 5 digits - 3 digits - 6 digits - 5digits.  Look at the second group. If it displays "OEM" you are out of luck, if it is anything else (most likely numeric) all is good.

Assuming you got lucky and it is not OEM, now you need the 25 character product key.  Look on the outside of the computer for a sticker with the OFFICE Product Key on it.  That means there are 2 stickers with product keys, one for Office the other for Windows.

If you don't find it, or can't read it you can use a tool to extract the key.  If you get to this point I can give you a list of tools to do that with.

louisyan_msft say:

hi matt,

the answer is yes.

you can follow the steps below to achieve the goal.

1. to let the room mailbox sign in to skype for business, it needs a skype for business online license.
go to admin center-> user-> active users-> click the room mailbox-> choose edit under assigned license on the right page.
screenshot below is for your reference:

as a tip, standalone skype for business online licenses can be used in this case. just sign into office 365 portal and download the skype for business software, then follow the steps as well.

2. set a password for the room mailbox.
if a random password is acceptable, you can use the gui of office 365 portal to reset a password:
go to admin center-> user-> active users-> click the room mailbox-> choose reset password under the picture of the mailbox.
if a specific password is required, you can use powershell to set the password you want:
a. connect to exchange online:
technet.microsoft.com/.../jj984289(v=exchg.160).aspx
b. run the command below:
set-mailbox -identity "room mailbox address" -roommailboxpassword (convertto-securestring -string "new password" -asplaintext -force) -enableroommailboxaccount $true

3. attend the meeting using skype for business.
sign in to skype for business with the account and the new password. you can join the arranged meeting by click the "meeting" button as the screenshot below shows:

note: only the meetings which will be held today are available.

if you have any questions when referring to the information above or it cannot address your requirement, please let us know which step is unclear and elaborate your requirements with a detailed example.
