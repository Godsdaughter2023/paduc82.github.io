a. user say:

Greg,
Re:  add a new sheet with custom name

I have assumed the command button is on the sheet to be copied.

   

Therefore the command button is deleted from the new sheet (after copy)
The "Me" reference used in the code refers to the sheet the command button is on.
It is also assumed that the code is placed in the module attached to the worksheet.
So...

Private Sub CommandButton2_Click()
 Dim strName As String
 Dim lngNum As Long

 lngNum = ThisWorkbook.Sheets.Count
 If Len(Me.Range("B33").Value) > 0 Then
   strName = Me.Range("B33").Value
 Else
   MsgBox "Enter new sheet name in cell B33  ", vbExclamation, "NEW SHEET NAME ?"
   Exit Sub
 End If

 Me.Copy After:=ThisWorkbook.Sheets(lngNum)
 lngNum = lngNum + 1
 ThisWorkbook.Sheets(lngNum).Name = strName & lngNum
 ThisWorkbook.Sheets(lngNum).OLEObjects("CommandButton2").Delete
End Sub
'---

[Edit - optional code added]

If you just want a new sheet (not a copy) then ...

Private Sub CommandButton2_Click()
 Dim strName As String
 Dim lngNum As Long

 lngNum = ThisWorkbook.Sheets.Count
 If Len(Me.Range("B33").Value) > 0 Then
   strName = Me.Range("B33").Value
 Else
   MsgBox "Enter new sheet name in cell B33  ", vbExclamation, "NEW SHEET NAME ?"
   Exit Sub
 End If

 ThisWorkbook.Worksheets.Add After:=ThisWorkbook.Sheets(lngNum)
 lngNum = lngNum + 1
 ThisWorkbook.Sheets(lngNum).Name = strName & lngNum
End Sub

Help me convert a long list of addresses from one column of Excel to a - Microsoft Community

hhhava say:

Help me convert a long list of addresses from one column of Excel to a range of rows.

I don't really understand how to use VBA, not very well trained in it. I have a very long list of addresses of varying lengths with varying numbers of rows between addresses all in one column of an excel sheet. How can I easily translate that long column list into a series of rows with about 6 columns separating names, addresses, etc into different columns? 
jeeped say:
... I have a very long list of addresses of varying lengths with varying numbers of rows between addresses all in one column of an excel sheet. How can I easily translate that long column list into a series of rows with about 6 columns separating names, addresses, etc into different columns? 

Tap Alt+F11 and when the VBE opens immediately use the pull-down menus to Insert, Module. Paste the following into the new pane titles something like Book1 - Module1 (Code),

Sub mcrSplitAddresses()
    Range("C1:M20").ClearContents
    Dim a As Range, sTmp As String, lr As Long, bOut As Boolean
    bOut = True
    For Each a In Range("A1", Cells(Rows.Count, "A").End(xlUp))
        If Len(a.Value) > 0 And bOut Then
            Range("C" & Cells(Rows.Count, "C").End(xlUp).Row).Offset(1, 0) = a.Value
            lr = a.End(xlDown).Row - a.Row
            If lr > 2 Then _
                Range("C" & Cells(Rows.Count, "C").End(xlUp).Row).Offset(0, 1) = _
                  a.Offset(1, 0).Value
            Range("C" & Cells(Rows.Count, "C").End(xlUp).Row).Offset(0, 2) = _
              Application.Proper(a.Offset(lr - 1, 0).Value)
            sTmp = Replace(a.Offset(lr, 0).Value, ",", "")
            Range("C" & Cells(Rows.Count, "C").End(xlUp).Row).Offset(0, 3) = _
              Left(sTmp, Len(sTmp) - 9)
            Range("C" & Cells(Rows.Count, "C").End(xlUp).Row).Offset(0, 4) = _
              Left(Right(sTmp, 8), 2)
            Range("C" & Cells(Rows.Count, "C").End(xlUp).Row).Offset(0, 5) = _
              Right(sTmp, 5)
            bOut = False
        End If
        If Len(a.Value) = 0 Then bOut = True
    Next a
End Sub

Change A1 in the third line is your data does not start in row 1. Tap Alt+Q to return to your worksheet. Tap Alt+F8 yo open the Macros dialog and Run the macro.

This appears to work on your supplied sample data.
