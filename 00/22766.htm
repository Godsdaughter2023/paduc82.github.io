  First, please try connecting your phone to multiple USB ports on your computer.  While you are doing this, try another cable as well.  If you are using a USB hub please try connecting directly to the computer.

  If you continue to get the error after trying the above, we would like you to try this update from another computer on another network.  This step is going to be the most sure fire way to determine if your PC is at the heart of the issue or if we should turn our focus to your phone.

e.j. gun say:

If the problem is pervasive, fixing all cells manually might be a chore.  The simple routine below will reset the format of all cells in the workbook with ";;;" to "General".

Eric

Sub FixBadNumberFormat()
    Dim theWS As Worksheet
    Dim theRange As Range
    Dim theCell As Range
'
    For Each theWS In ActiveWorkbook.Worksheets
        Application.StatusBar = "Checking Worksheet " & theWS.Name & "..."
        Set theRange = theWS.UsedRange
        For Each theCell In theRange.Cells
            If (theCell.NumberFormat = ";;;") Then theCell.NumberFormat = "General"
        Next theCell
    Next theWS
    Application.StatusBar = False
End Sub
