    frederik long say:

    You're not getting the option to specify an admin account because you do not have any such account left.

    You fell into a classic trap. You played with your accounts without having a spare admin account that you can fall back on in case of trouble. It's like having a single house key. You can recover from this like so:

    1. Ask a friend with a Windows 10 machine to create a bootable flash disk, using the command recoverydrive.exe.
    2. Use this disk to boot your machine into Windows Repair Mode.
    3. Use System Restore to set your machine back to a point before you did whatever you did with your accounts.
    4. Create, test and document a spare admin account so that you never ever fall into this trap again.

    The above steps require a certain degree of experience at a maintenance level. You might want to ask a friend to help you.

    By the way - the native app snippingtool.exe is a far superior way to create a screen snapshot. I recommend you try it.
