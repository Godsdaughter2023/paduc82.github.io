mouneshwar r say: Hi lacolocho,

Windows Media Player is a component within Windows Vista. In other words, Windows Media Player 11 comes integrated with the Windows Vista Operating System and hence there is no download available to re-install it. It’s not only Windows Media Player; any system components which are integrated within the operating system are not available for downloads.

I would like to know when exactly you get this message “Windows Media Player need to be re-installed”. Is it as soon as you login to the computer or when you try to open Windows Media Player?

Did you recently tried to play any unsupported audio/video formats in Windows Media Player?

Since we came to know that there is no option available to re-install Windows Media Player, we do have an option to repair it.

So, here is your first step for troubleshooting Windows Media Player 11.

1.    Click on Start; in the start search type cmd. In the programs list, right click on cmd and
       run as administrator.
2.    Now execute the following command
       sfc /scannow

Hit Enter after typing the command. The scan would take a while. Once the scan is complete, try opening Windows Media Player 11 and check if you’re able to open it without any error messages.

Check out this link on How to run system file checker .

http://support.microsoft.com/kb/936212

As a second step, you can try to Re-register wmp.dll file.

Type the following command in the administrator mode of the command prompt and then press Enter.

regsvr32 wmp.dll

Hit Enter after typing the command. If you get any message like wmp.dll registered successfully, then click on OK on that message box. Close the command prompt and re-open Windows Media Player to check if you’re able to open it successfully.

If the above steps did not help, I would suggest you to run the Fix it tool as mentioned in the article below.

When I try to use Windows Media Player 11, the program does not start, or some UI elements are blank

http://support.microsoft.com/kb/925704 

Hope this helps.
