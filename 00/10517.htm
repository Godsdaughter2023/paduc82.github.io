    cortig say:

    You don't have the command in the contextual menu, but you should still be able to copy all your event from the list mode.

    1. Click on the search box on the top-right corner of the window to get the Search table of the Ribbon.
    2. From there, click the Advanced button to perform an advance search.
    3. In the drop down menu, select Account > Is > (name of your account)

    Outlook should search for a little while and then list all the events from that account.

    You should then be able to select all events in the list and drag them all to the other calendar of your choice.

    I've had a few issues in the past with Exchange calendars and events that were linked to items. I tricked the whole system by adding the calendars to the Calendar app on my Mac and performing the migration there.
