I want all the choices between 7 AM - 9 AM to be gone. That also goes for the next time they put in. It should start after the end time of the previous task. Is there a way that excel can do that?

ossiemac say:

You have made me think hard on this one.

It is necessary to create a dynamic named range that starts at the value after the start selection and finishes as the end of the list of times.

I created the list of times at 30min intervals on the same sheet as the DropDowns so I can post a snippet and try to explain the procedure. The 2 yellow cells are the DropDowns. Cell C1 contains the Start time DropDown and cell E1 contains the Finish time DropDown.

  • Create a Dynamic Named Range. I called the name Finish. The formula for the named range is as below where:
  • Sheet1!$A$1   is the first cell of the list of times  (Must start at row 1 otherwise additional difficulties will be introduced adjusting the row numbers following Match and for the Offsets)
  • Sheet1!$C$1   is the cell containing the value selected with the Start DropDown
  • Sheet1!$A$1:$A$48   is the list containing the list of times

=OFFSET(Sheet1!$A$1,MATCH(Sheet1!$C$1,Sheet1!$A$1:$A$48,0),0,COUNTA(Sheet1!$A$1:$A$48)-MATCH(Sheet1!$C$1,Sheet1!$A$1:$A$48,0),1)

Accounts become inactive if you don't sign in for more than 270 days VIA WEB BROWSER or within the first 10 days after signing up VIA WEB BROWSER for an account.
