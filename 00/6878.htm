How do I disable all network adaptors?

don cuthbert say:
  • Right-click on Start and choose Device Manager
  • Find Network adapters and expand list
  • Right-click on each item in the list and choose disable
  • When using Windows update you have to wait until the download is complete and the Update is underway, but before the first reboot
  • After the reboot you may want to unplug the Ethernet cable so the install does not try to use it, as it slows it a lot.  If it tries to connect to a wireless network, do not sign in to that either.

rohn007 say:

Please expand on what you want to do.  You can't "import" a spreadsheet into a document. They are fundamentally different file structures. BUT, there are ways to bring parts of a spreadsheet into a Word document.

You can simply copy and paste the data from Excel into Word.  That will create a word table with the data and basic formatting.

Or you can use Paste Special to paste a "mini spreadsheet" into the document.

Or you can use Paste Special with Link to setup an import link between the document and spreadsheet. Double clicking on the spreadsheet in the document would open Excel.

As Paul pointed out, you can use a spreadsheet as a "data source" for a Mail Merge in Word. You create a form letter that imports user specific information from multiple rows in a spreadsheet.

https://www.lifewire.com/inserting-excel-data-microsoft-word-documents-3540362
