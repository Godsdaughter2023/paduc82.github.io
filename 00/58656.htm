it is blank because there is no account set up.  It is not blank as in "hung", but rather blank as in "nothing to show".  Most likely when you first started with Win8 you dismissed the configuration setup for a mail account, for the time being.

 

With the "blank" Mail app open move over to the right charms bar, select settings > accounts and add a new account.
