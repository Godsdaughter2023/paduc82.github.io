If this user is using Office 365 groups, as a workaround you can allow the user to send as or send on behalf of an Office 365 Group.

If this user is using a shared mailbox. By default, messages sent from the shared mailbox aren't saved to the Sent Items folder of the shared mailbox. Instead, they are saved to the Sent Items folder of the person who sent the message.

Here are the steps:

1.) As an admin, modify this setting to copy sent items on the (shared) mailbox sent items folder in the Office 365 admin center. Edit the shared mailbox settings, and choose Sent items > Edit. (this will have both the user mailbox and Shared mailbox a copy of the item.)

mike146 say:

I'm not sure why I was not able to see my undated tasks, but I did find a solution after a bit more research so I will post it here in answer to my own question.

If you want to show BOTH - Dated and Not Date Tasks

1) In the HOME tab>Current View Section click the drop down arrow and select "Manage Views" at the bottom. 

2) Highlight TODAY - Click MODIFY - Click FILTER - ADVANCED tab

3) Remove all the defaults (You can always RESET to defaults if you want them back.

Add the following

FIELD>INFO STATUS FIELDS - STATUS; CONDITION not equal to; VALUE - Completed - Add to list

DATE/TIME FIELDS>START DATE - does not exist - Add to list

DATE/TIME FIELDS>DUE DATE - does not exist - Add to list

DATE/TIME FIELDS>START DATE - on or before today - Add to list

DATE/TIME FIELDS>DUE DATE - on or before today - Add to list

Click OK

You will then be able to see all your tasks, even those without a start date, for tasks which have no specific time to start.  This should have been the default view for Outlook as far as I am concerned.

Before you click out of the Advanced view settings to make things even sweeter, click on COLUMNS, then display the following in this order:

Complete

Priority

Icon

Task Subject

Start Date

Due Date

Categories

Recurring

Flag Status

Click OK

Then go to the GROUP BY button

Group items by Start Date Ascending, then by

Priority Descending

Alternately, instead you might want to SORT instead of GROUP, by Priority and then by Due Date

Click OK Twice

You can apply more or less the same conditions to the ACTIVE TASKS view but change to

FIELD>INFO STATUS FIELDS - STATUS; CONDITION not equal to; VALUE - Completed - Add to list

DATE/TIME FIELDS>START DATE - does not exist - Add to list

DATE/TIME FIELDS>DUE DATE - does not exist - Add to list

DATE/TIME FIELDS>START DATE - anytime - Add to list

DATE/TIME FIELDS>DUE DATE - anytime - Add to list


Click OK

Change  the columns in the same manner as above, as well as the GROUP BY commands

While you are at it, under the VIEW tab, Show the READING PANE on right and TO-Do Bar - Calendar

Now you will have Task Views which actually reflect the views which you selected.  You'll love the way this looks in comparison to the default views which are set by Microsoft when you install the software.

Cheers!
