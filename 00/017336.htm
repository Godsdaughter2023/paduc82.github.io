graham mayor say:
If you apply a template to an existing document, rather than create a new document from the template which is how the template is supposed to be used, then the original document will attempt to retain its original formatting. To remove that original formatting and revert to the named styles, select the document (CTRL+A then CTRL+SPACE and CTRL+Q) or by macro

Sub ResetMyStyles()
Dim oStory As Range
    For Each oStory In ActiveDocument.StoryRanges
        With oStory
            .Font.Reset
            .ParagraphFormat.Reset
        End With
        If oStory.StoryType <> wdMainTextStory Then
            While Not (oStory.NextStoryRange Is Nothing)
                Set oStory = oStory.NextStoryRange
                With oStory
                    .Font.Reset
                    .ParagraphFormat.Reset
                End With
            Wend
        End If
    Next oStory
    Set oStory = Nothing
End Sub

stefan blom say:

Since most styles are based on the Normal style, they will inherit the font (among other settings) from Normal in the document where you use the styles. Explicitly setting the font that you want in the style definitions will take care of the problem.

If you are creating a style set, you can include the Normal style in that set; just make sure it has the formatting that you want.
