    Before giving any suggestions, I would like to make you aware that removing the recovery partition is not recommended as it is there as a backup to get your computer to a working state in case of any mishaps.

    However, if you still want to delete the recovery partition, you can try either of the following.

    Method 1:

    1)    Open the Diskpart application by either typing in "Diskpart" (without the quotation marks) into the search, or type "Diskpart.exe" into Run by pressing the Windows key + R.

    2)    Now with Diskpart open, you must enter the following command lines:

    I. Type "list disk" without the quotation marks and press enter.

    II. Type in "select disk 0" and press enter.

    III. Then type in "list partition" and press enter.

    IV. Then type in "select partition x" where x=the recovery partition you wish to delete. Replace x with the recovery partition number and press enter.

    V. Then finally type in "delete partition override" and press enter.

     

    Method 2:

    1)      Click on “Start” and type “msconfig” in the start search bar.

    2)      Click on “msconfig” icon to open the “System configuration”.

    3)      Under the “Boot” tab, you will find all the partitions present on the computer.

    4)      Choose the partition you want to delete. Then click “delete”.

    Note: This would even remove the entry from the BCD.
