amit tandon say:

It seems you are looking to match each word of a cell in column B with column A, because you have mentioned that "World Disney" also has a match as in "Disney" matches "Disneyland Parks". Refer below image:

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell D2 & copy down:

=IFERROR(INDEX($A$2:$A$10,MIN(IFERROR(MATCH(TRUE,ISNUMBER(SEARCH(TRIM(LEFT($B2,FIND(" ",$B2)-1)),$A$2:$A$10)),0),1E+100),IFERROR(MATCH(TRUE,ISNUMBER(SEARCH(TRIM(RIGHT($B2,LEN($B2)-FIND(" ",$B2))),$A$2:$A$10)),0),1E+100))),"")

The formula looks to match each word from cell B2 with a partial match in column A, and returns the FIRST match found in column A. Please note that the formula is valid ONLY if 2 words are present in each cell of column B separated by space (not non-breaking space). The formula can be updated for more than 2 words in each cell.

You can create an empty workbook set up exactly the way you want.

Save it as an Excel template named Book.xltx in your Excel startup folder - by default, that is

C:\Users\<username>\AppData\Roaming\Microsoft\Excel\XLSTART

(AppData is a hidden folder, so you must show hidden files, folders and drives in order to see it)
