burrwalnut say:

James

You can't execute a .dll file in the same way that you run an .exe or .com program. It has to be called using rundll.

You may want to register or reregister it. If so, open a Run window (Windows Logo key+R), type regsvr32 jetimage.dll (note the space) and press Enter.
