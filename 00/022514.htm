karl timmermans [outlook mvp 2012-2018] say:

III recently had to create a new email account because my sip provider changed to iMac from pop3. ................. How do I merge theses folders to one inbox and sent folder?

Things to know about configuring an IMAP account with OL'2010

#1 - If this was the first account configured in the profile, there will always be 2 folder trees in the Outlook Nav Pane representing the data files associated with the IMAP account

#2 - One data file will be for the email folders and the other will be the default data file for the profile which will contain the default contact/calendar folders. The <inbox>, <Sent> etc folders in the default data file are extraneous but can't be deleted via the Outlook UI.

#3 - If you re-added this account using IMAP while it was still configured in the profile as POP3 - the PST file associated with the POP3 account will be the "default data file" which is actually what you want since that's where your contact/calendar data resides

#4 - If you configured the IMAP account in a new outlook profile, you're going to want to re-add the PST file that was associated for the POP account and then make it the "default" for the profile removing the empty one that was created during the email configuration process. 

From OL'2010 menu --> File --> Account Settings --> Data file tab

(this will show all PST files configured for your current profile - what you need to do next depends on what's listed there)

trip_to_tokyo say:

I want to have a slide show continuing playing (looping) on a monitor that is for display only. 

1 is it possible to have a slide show continuously play?

 

Try this:-

 

1. Slide Show tab.

 

Set Up group.

 

Click on:-

 

Set Up Slide Show

 

In the:-

 

Show options

 

 - part of the pop up window make sure that:-

 

Loop continuously until 'Esc'

 

 - has a tick (check) in it.

 

2. Animations tab.

 

Transition to This Slide group.

 

Place a tick (check) in the:-

 

Automatically After:

 

 - box then set the box to its right to (for example):-

 

00:01

 

 - (meaning 1 second) then, still in the:-

 

Transition to This Slide

 

 - group click:-

 

Apply To All

 

3. Hit the F5 key to run the Slide Show.

 

The Slide Show should now run continuously until you hit ESC to stop.

Thanks.
