Hi there ConnorRobey               

There is no option to configure the history settings of instant messages in Skype for Windows 10, which was introduced on Skype for Windows desktop. Refer to this link on how to manage conversation history in Skype for Windows desktop.  

If you set your history to keep the conversation for forever on Skype for Windows desktop, then the conversation you've had in the past months are still locally stored on your device, but the messages that would show on Skype for Windows 10 are the conversations for the last 30 days that are stored on the cloud-storage which you can access across different devices when signed in to Skype.
