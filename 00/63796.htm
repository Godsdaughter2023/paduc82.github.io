Most, not all, CD and DVD drives use the default Windows drivers. Try the methods here and

REBOOT this will refresh the default driver stack. Even if drive is not shown continue
below.

Then work your way through these - remember the drive could be bad, could have a loose
cable or slight corrosion on the contacts (usually for a laptop) and other issues.

Hardware devices not detected or not working - A Mr Fixit

Another possibility is that the cables are loose. Remove ALL power, then check cables in both
ends. Remove and replace, do not just snug. For laptops you can often clean the power and

device maker to see if there is firmware or drivers for you drive if needed.

===============================

Check with your System Maker and Device maker for both possible firmware updates and

a. user say:

On the internet, websites are actually identified by numbers, not names. But people like you and me find it more convenient to remember names.

The internet came up with the Domain Name System to translate a website's name to its number. When we type the website's URL and press Enter, or click on a link, the URL gets sent to a DNS server that translates it to the corresponding number.

In your case, the DNS server wasn't able to translate the name fictiondb.com to the corresponding number. That could happen for lots of reasons, none of which are your fault. For example, maybe the DNS server couldn't find the number. So much for artificial intelligence, right?

The suggestion was to search for the actual name of the website (FictionDB) with a search engine. Doing that might yield a different URL that the DNS can actually locate.

In any case, it's the internet's problem, not your problem.
