peterbraunston say:

Connect Windows 10 Pro to SBS 2008

I just learned a little trick.  In order to connect Windows 10 Pro to SBS 2008 SP2:

1.  Type Network in Search option

2.  View Network Connections

3.  Select the active network card, right click and select properties

4.  Locate and unselect IPV6

5.  Apply

6.  Open a command prompt and type "ping connect"

7.  You should get a response from your active SBS 2008 Server address

8.  Open Edge and type "http://connect"

9.  It will say that you are not able to connect.  However, it allowed me to run the file from the bottom of the page.

10.  Follow the instructions and restart the machine

Upgrading to Windows 10 and Activation issue - Microsoft Community

amit8sharma say:

Upgrading to Windows 10 and Activation issue

Upgrading to Windows 10 for Windows 7

If you are going to upgrade Windows 10 from Windows 7 then you got a issue related Windows Activation.

And i.e. Product Key Blocked.

Now to activate Windows 10 mostly people use KMS Activation that actually activate Windows 10 but not permanently.

Those people use KMS activation go and check that Windows 10 is activated Permanently or not.

To check: -

1) Go and run CMD as Admin.

2) Then Type this command: -

     slmgr -xpr

     If Windows 10 is not activated then you got this: -

     Windows is in Notification Mode.

     And if your Windows 10 is Activated using KMS Activator the you got this: -

     Expiry Date of your Windows Activation : 06/june/2016

     If you use Genuine Windows Product Key then you got this: -

     Machine is Permanently Activated. 

So, Activating Windows 10 for Permanently is a issue and there is no Activator that Activate Windows 10 Permanently.

In case of Upgrading Windows 10 for Windows 7 and to activate: -  

Buying Genuine Product Key from Windows Store is a only option to Activate Windows 10 Permanently.

Now how to Activate Windows 10 Permanently: -

If you want Permanently Activated Windows 10, then go and install Windows 8.1 and then Upgrade to Windows 10.

Before Upgrading Windows 10 to 8.1 Activate Windows 8.1 using Activator of your Choice (KMS Activator)After Upgrading to Windows 10 go and Activate your Windows 10 online, this will Activate Windows 10 Automatically.

Now go and check Windows Activation detail: -

Open CMD and follow those steps again: -

Type slmgr -xpr

And you got: -

Machine is Permanently Activated...
