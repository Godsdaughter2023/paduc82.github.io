guruaid say:

You can try using the Diskpart utility to clear the read only attributes on the flash drive.

1. Press the 'Windows key + R' to bring up the run box.
2. Type in cmd and hit enter.
3. In the Command prompt, type 'diskpart' (without quotes)
4. Now, type in 'list disk'. This will display the list of disks.
5. Check which one is the flash drive, and type 'select disk=[n]' (n is the disk number of the flash drive)
6. Now type in 'attributes disk clear hidden'
7. Remove the flash drive and plug it back in.
8. Check if you're able to see the files now.
