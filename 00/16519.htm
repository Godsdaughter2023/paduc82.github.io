jp ronse say:

Hi,

Try this UDF:

Function Extract_Words(MyInput As Variant, Extract As Variant)
   Dim RE As Object, MC As Object, c As Range
   Dim strExtract As String, strOutput As String, i As Integer
   
   Set RE = CreateObject("vbscript.regexp")
   
   For Each c In Extract
      strExtract = strExtract & c & "|"
   Next c
   
   strExtract = "(" & Left(strExtract, Len(strExtract) - 1) & ")"
   With RE
      .ignorecase = True
      .Global = True
      .Pattern = strExtract
   End With
   If RE.test(MyInput) Then
      Set MC = RE.Execute(MyInput)
      For i = 0 To MC.Count - 1
         strOutput = strOutput & " " & MC(i) & " "
      Next i
   End If

   Extract_Words = Trim(strOutput)
End Function

rick rothstein, mvp - excel say:

I want to search text for a value and return the value if its there, checking multiple values.

Search column A for a particular value (value list in column E) and return that value in column B

Return multiple values if it has multiples.

Here is another UDF you can consider using. This one works different from JP Ronse's UDF in that it extract the word if and only if that word stands alone and is not part of another longer word. So if your sentence was "I like carpets", my code will not extract anything where as JP Ronse's UDF would extract the word "car". Here is my code...

Function ExtractWords(Sentence As String, WordList As Range) As String
  Dim X As Long, WL As Variant, WordChars As String, Parts() As String
  WL = WordList
  For X = 1 To UBound(WL)
    WordChars = UCase("[!A-Z" & Replace(WL(X, 1), "-", "") & _
                IIf(InStr(WL(X, 1), "-") > 0, "-", "") & "]")
    If " " & UCase(Sentence) & " " Like "*" & WordChars & _
                 UCase(WL(X, 1)) & WordChars & "*" Then
      ExtractWords = ExtractWords & " " & WL(X, 1)
    End If
  Next
  ExtractWords = Trim(ExtractWords)
End Function

To use this UDF, you would put this formula in B1 and copy it down...

=ExtractWords(A1,E$1:E$4)
