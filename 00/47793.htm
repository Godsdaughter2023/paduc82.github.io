Windows 7 Slave Computer Capabilities - Microsoft Community

jim-wa98 say:

Windows 7 Slave Computer Capabilities

How can I attach an earlier windows version machine to a Windows 7 system which is connected to the internet through a router?  The older machine should not be connected to the internet, but does need wireless access to printers and scanners?
hairyfool say:

Your local network will have its own address range normally 192.168.X.Y Of the two values X will be common to all PCs but can depend on how the Router is set up, Y could be anything 1-254.

Normally this address will be assigned by the router along with a value known as the Default gateway. This tells the PC that when it wants to connect to an address outside your network (i.e. the internet) to go to this gateway to connect which is the router.

If you set the default gateway to another address then the PC can no longer connect to the internet but will still be able to see all local devices.

As an example. If the DHCP enabled values are:

IP Address: 192.168.0.32

Subnet Mask: 255.255.255.0

Default Gateway: 192.168.0.1

DNS Server: 192.168.0.1

Manually assign them to:

IP Address: 192.168.0.192

Subnet Mask: 255.255.255.0

Default Gateway: 127.0.0.1

DNS Server: 192.168.0.1

This will allow the PC to still resolve addresses (DNS) so you don't have to wait for timeouts to occur but connect back to itself (127.....) when trying to get to the internet so will fail. Anything else on the local network can still be connected to.
