hairyfool say:

The message is be cause the BIOS is looking to boot from the LAN (not Wireless) before it then boots from the hard disk. The Media test failure is because in order to boot from LAN it tests to see if the wired Ethernet is connected, it's not so it generates the error message.

In order to enter the BIOS press the F2 key after initial power on but before the Windows Logo shows

Integrated peripherals should allow you to define the wired network card settings. Depending on the BIOS it will either "enabled with PXE" or have another option to enable or disable boot ROM.

In the Boot options the LAN card needs to be removed altogether.

This message is ONLY applicable to the BIOS Boot handoff and should have no bearing on the WiFi issues (it is not about WiFi)
