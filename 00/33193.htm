The library is a database that includes links to the digital media files on your computer. Among other reasons, you can't move the library from one computer to another because the links in the database would no longer be correct.

If you want to replicate your current library on another computer, you need to copy your digital media files to that computer and then add those files to the new library.

The library is a database that contains links to the digital media files on your computer. It does not contain your actual digital media files or copies of your digital media files.

ronnie vernon say:

Hi Rick

1. Open Control Panel/Administrative Tools/Task Scheduler.  In the Task Scheduler, click Action / Create Task.

 

2. On the General tab, enter a short name for the task (you’ll use this name to run the command later), and click the Run with highest privileges checkbox. This setting tells Windows to use the administrator token (the one you normally unlock via UAC) when you run this task.

 

3. On the Actions tab, click regedit.exe which will open the Registry Editor. (C:\Windows\regedit.exe)

 

4. On the Settings tab, be sure that Allow task to be run on demand is selected. You’re not actually going to schedule this task but instead are going to run it from a shortcut. Click OK to save.

 

5. Go to the Desktop, right-click an empty spot and select New/Shortcut. In the

 

schtasks /run /tn “taskname“   (Including the Spaces and Quotes)

 

Replace "taskname" with the name of the task you created and click Next.

 

6. Finish.

 

Place the shortcut in a convenient location.

 

You can use this shortcut to run the program with full Administrator privileges and no UAC prompt.

Regards
