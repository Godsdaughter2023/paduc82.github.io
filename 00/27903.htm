julie fka profjulie say: Here are the Bluetooth profiles that are supported in Windows Phone 7:
  • Advanced Audio Distribution Profile (A2DP 1.2)
  • Audio/Video Remote Control Profile (AVRCP 1.0)
  • Hands Free Profile (HFP 1.5)
  • Headset Profile (HSP 1.1)
  • Phone Book Access Profile (PBAP 1.0)
Here are the Bluetooth profiles that are supported in Windows Phone 8:
  • Advanced Audio Distribution Profile (A2DP 1.2)
  • Audio/Video Remote Control Profile (AVRCP 1.4)
  • Hands Free Profile (HFP 1.5)
  • Phone Book Access Profile (PBAP 1.1)
  • Object Push Profile (OPP 1.1)
  • Out of Band (OOB) & Near Field Communications (NFC)
As you can see, the Bluetooth MAP profile was never part of Windows Phone 7, and so it was not removed in Windows Phone 8.

Try:

With owrd.Documents(strSourceFile)
  For x = 1 To .ContentControls.Count
    With .ContentControls(x)
      txtCCcontent = .Range.Text
      txtCCname = .Title

    End With
