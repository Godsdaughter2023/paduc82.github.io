t. valko say:

 

I have a formula in J1:   =SUMPRODUCT(($A:$A="NAME")*($C:$C="Attended")*($D:$D="therapy")*($E:$E))/60 that returns an answer in hours. Column $E:$E is minutes. I have a drop down menu in H1 from a list of names in H26:H46. I would like to select a name from the drop down menu and have the formula in J1 change for that name.  

Just replace ="NAME" with =H1.

Since you're using Excel 2007 use the SUMIFS function. It is much more efficient than SUMPRODUCT especially when referencing the entire columns.

=SUMPRODUCT(($A:$A=H1)*($C:$C="Attended")*($D:$D="therapy")*($E:$E))/60

=SUMIFS(E:E,A:A,H1,C:C,"Attended",D:D,"Therapy")/60

Also, it's better to use cells to hold the criteria:

  • F1 = Attended
  • G1 = Therapy

=SUMIFS(E:E,A:A,H1,C:C,F1,D:D,G1)/60

cristin yan say:

Hi,

 

Here's an example for you:

 

Please paste the following code to a Module (via Alt+F11 >> Insert >> Module):

 

Public dteStart As Date
Public dteEnd As Date
Public i As Integer

'###########################
'Check if the time is over.
'###########################
Sub ReminderProc()
    i = i + 1
    Range("D2").Value = i
    Range("C2").Value = Format$(dteStart, "hh:mm:ss")
    Call EnableReminder
End Sub

 

'#####################################
'Start the reminder once 5 seconds.
'#####################################
Sub EnableReminder()
    dteStart = dteStart + TimeValue("00:00:05")
    Application.OnTime dteStart, "ReminderProc"
End Sub

 

'###################
'Stop the reminder.
'###################
Sub DisableReminder()
    On Error Resume Next
    Application.OnTime dteStart, "ReminderProc", , False
End Sub

 

'#############
'Macro Entry.
'#############
Sub Main()
    i = 0
   
    On Error GoTo errHandler
   
    dteStart = CDate(Range("B2").Value)
    
    If dteStart < CDate(Format$(Now, "hh:mm:ss")) Then
        MsgBox "The Start Time should be later than the current time!", vbExclamation, "Tips"
        Exit Sub
    End If
    
    EnableReminder
   
    Exit Sub
   
errHandler:
    MsgBox "Invalid date, please try again later!", vbExclamation, "Error"
End Sub

Instructions:

1. Both the Start button and the Stop button in E2 are Form Controls;

2. Macro assigned to the Start button is Main;

3. In the above picture, my reminding time was set to every 5 seconds, you can change it from the code line:

    dteStart = dteStart + TimeValue("00:00:05")

    e.g for 25 minutes:

    dteStart = dteStart + TimeValue("00:25:00")

4. Every current reminding time will be displayed in C2;

5. The number "4" in D2 indicates that it's the 4th time for reminding;

6. The reason why I changed the "Stop Reminding" action (you said: UNLESS the 'end time' field has some time or data) to click a button is because that using a button is more convenient.

Hi,

To see where Internet Explorer stores its Cookies in Windows Vista, Windows 7 or Windows 8, open Explorer > Organize > Folder Options > Views > Check ‘Do not show hidden files and folders’ and Uncheck ‘Hide protected OS files’ > Apply > OK.

Now you will be able to see the two real locations of Windows Cookies folders at the following address in Windows 7:

  • C:\Users\username\AppData\Roaming\Microsoft\Windows\Cookies
  • C:\Users\username\AppData\Roaming\Microsoft\Windows\Cookies\Low

In Windows 8 and Windows 8.1, the Cookies are stored in this folder:

  • C:\Users\username\AppData\Local\Microsoft\Windows\INetCookies

The error 400 Bad Request is an HTTP status code which means that the request you sent to the website server (i.e. a request to load a web page) was somehow malformed therefore the server was unable to understand or process the request.

 

 

I would suggest you to try logging in to the website after sometime and check if it helps.

 


 

In the meantime, I would suggest you to follow the methods to troubleshoot the issue:


 

Method 1:


 

I would suggest you to add the address of the website in trusted site and check for the issue.


Follow the steps:

 

 

a) Open Internet Explorer on the desktop.
b) Press Alt key on the keyboard to enable the menu bar if it is disabled.
c) Click on Tools option.
d) Click on Internet Options.
e) Go to Security tab.
f) Click on Trusted Sites.
g) Add the website address and click on Add.
h) Click on Close.

 

 

Method 2:

 

 

I would suggest you to turn off friendly HTTP error messages on Internet Explorer and check if it helps.


Follow the steps:

 

 

Step 1:

 

 

You can follow the steps to disable the settings:


a)      Open Internet Explorer on the desktop.

b)      Click on Tool icon on the top right corner of the browser.

c)       Click on Advance tab and scroll down to browsing section.

d)      Clear the "Show friendly HTTP error messages" check box, and then click OK.

 

Step 2:

 

Uncheck HTTP settings:

 

a) Open Internet Explorer on the desktop.

b) Click on Tools icon on the top right corner of the browser.

c) Click on Advance tab and scroll down to the HTTP 1.1 settings.

 

d. Uncheck "Use HTTP 1.1".

 

e. Click ok to Apply and and then on OK. Restart the browser.

 

Check if the issue persists.

 

Method 3:
