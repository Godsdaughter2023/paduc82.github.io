charlie dude say:

To uninstall Internet Explorer 9, you have to do it through Windows update.

 

Open Windows update and click the link "view update history", in the window that opens click the link that is in "to remove an update see installed updates". In the next screen scroll down to "Windows internet Explorer 9" click it to highlight it and then choose "uninstall" at the top of the page.

 

If you have Internet Explorer 10, uninstall it the same way.

cort andrus say:

I've seen this issues once before on Windows 10. The problem was the preview in the file explorer. To show the preview for PDF file an external program is needed. In this case this should be Soda 8.

To display the preview, the program has to run a service. As long as the service is showing the preview, Windows will interrupt the delete proces, because the file is still opened.

So to delete the files you'll need to turn off the PDF-preview.
