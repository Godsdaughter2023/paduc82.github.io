IF FUNCTION

IF FOR EXAMPLE =IF(A1>A2,"I want this cell left blank?","If there is already a no. in this space and it does not meet the criteria I want the number to stay the same?")

 

What do you put in the questions in the above formula?

hansv mvp say:

The first part is an empty string "". This does not leave the cell really blank, but it looks blank.

About the second part: a formula in a cell can't refer to that cell (except in very specific circumstances) - it would lead to a circular reference that makes it impossible for Excel to calculate the formula.

You could use this formula in A4: =IF(A1>A2, "", A3)

But you can't use that formula in A3 because that would cause a circular reference.

If this is not what you want, you may need VBA code.
