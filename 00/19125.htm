3.Make sure the Startup type is set to Automatic and if the service is not running click on Start.
4.Click Apply followed by OK.
5.Restart your pc

2.Expand Mice and other pointing devices and then right-click on your device and select Update driver.
3.First, select “Search automatically for updated driver software” and wait for it to automatically install latest drivers.
4.If the above fails to fix the issue then again follow the above steps except but on the Update driver screen this time choose “Browse my computer for driver software.“
5.Next, select “Let me pick from a list of available drivers on my computer.“
6.Select the appropriate driver and click Next to install it.
7.Reboot your PC to save changes.

Uninstall Mouse Drivers

2.Expand Mice and other pointing devices and then right-click on your device and select Uninstall.
3.If asked for confirmation select Yes.
4.Reboot your PC to save changes and Windows will automatically install the default drivers.

2.Then select Uninstall a program and find Synaptics (or your mouse software for example in Dell laptops there is Dell Touchpad, not Synaptics).

4.Once the uninstallation is complete reboot your PC to save changes.
5.Now go to your mouse/touchpad manufacturer website and download the latest drivers.
6.Install it and reboot your PC. See if you’re able to Fix Mouse Scroll Not Working On Windows 10.

4. Run SFC Scan & DISM
https://support.microsoft.com/en-us/help/929833...

If you create and Excel workbook in which you store each of the Keys and the notes in that key as shown below

A B C D E F G
1 Key
2 A B C♯ D E F♯ G♯
3 E Flat F G A♭ B♭ C D
4 D
5 etc

and you save that workbook with the filename of KeySubstitutions in your Documents folder, when you run the following macro, it will ask you for the original Key (e.g. A) and then for the key that you want to change to (e.g. E Flat)  and it will then make all of the necessary substitutions.

Dim xlapp As Object
Dim xlbook As Object
Dim xlsheet As Object
Dim arrnotes As Variant
Dim i As Long, j As Long
Dim strkey1 As String, strkey2 As String
Dim bstartapp As Boolean
On Error Resume Next
Set xlapp = GetObject(, "Excel.Application")
If Err Then
    bstartapp = True
    Set xlapp = CreateObject("Excel.Application")
End If
On Error GoTo 0
Set xlbook = xlapp.Workbooks.Open("C:\Users\Steve\Documents\KeySubstitutions.xlsx") 'you may need to modify the path
Set xlsheet = xlbook.Worksheets(1)
arrnotes = xlsheet.Range("A1").CurrentRegion
xlbook.Close xlDonOTSaveChanges
If bstartapp = True Then
    xlapp.Quit
End If
Set xlapp = Nothing
Set xlbook = Nothing
Set xlsheet = Nothing
strkey1 = InputBox("Enter the original key.")
strkey2 = InputBox("Enter the new key.")
For i = 2 To UBound(arrnotes, 1)
    If arrnotes(i, 1) = strkey1 Then
        j = i
        Exit For
    End If
Next
For i = 2 To UBound(arrnotes, 1)
    If arrnotes(i, 1) = strkey2 Then
        k = i
        Exit For
    End If
Next i
For i = 2 To UBound(arrnotes, 2)
     With ActiveDocument.Range
         .Text = Replace(.Text, arrnotes(j, i), arrnotes(k, i))
     End With
Next i
