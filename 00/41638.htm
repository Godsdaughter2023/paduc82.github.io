a. user say:

Hi,

Thank you for posting the query on Microsoft Community.

I am sorry to know that you are getting a black screen after it goes to sleep or hibernation.

If you see the black screen only after sign-in and you can see a mouse cursor:

  1. While at the black screen, try using CTRL+SHIFT+ESC to start Task Manager.
  2. If it becomes visible, try finding LockAppHost.exe on the Details tab and ending the task. If this resolves the black screen, use corruption resolution steps (dism, then sfc).
  3. If this does not work, use File, Run new task in Task Manager to start an Administrator CMD window. Use this to create a new administrator user with the following commands (changing “<password>” to something of the customer’s choosing):
             net user test <password> /add
         net localgroup Administrators test /add

This will give you a test user to try, which will confirm if the issue is user or system-specific.

 

If you see the black screen before sign-in:

1. Try connecting to different video outputs on the PC. Sometimes a change in driver can cause the video to be sent:

  • To a different Adapter: from integrated graphics to a discrete card or vice versa.
  • To a different output: from HDMI to DVI, DisplayPort to VGA, or any other combination.

2. Try to boot into Safe Mode with Networking or Enable low-resolution video.

  • Advanced boot options, Safe Mode with Networking, or Low Resolution Video.

http://windows.microsoft.com/en-us/windows-10/change-startup-settings-in-windows-10#v1h=tab01

  • Check Device Manager and look for Display Adapters.
    1. If you see 2 items under Display Adapters, disable to integrated video card (e.g. Intel HD 4000 or AMD Radeon HD 4200) and restart. (try vice versa if it does not work)
    2. If you see 1 item under Display Adapters, rollback the driver or uninstall the display adapter in Device Manager, and enable the option to delete driver software.

Check the PC or display adapter manufacturer’s site for an updated driver.

Hope this helps in resolving the issue. If the issue persists, do get back to us. We will be happy to assist you.

Hi amsa85
Greetings! I am Vijay, an Independent Advisor. I am here to work with you on this problem.

Recovery drive is used to recover your Windows. It can't recover your files and programs. Hence, size will be small say 4 to 5 GB.

You should take a system image backup which restores everything whatever you have on your computer's primary partition. Hence, it will restore your OS, programs and files. Whatever size is occupied on your hard drive's primary partition, it will be of that size. Hence,if used size is 340GB on your computer, system image would be of 340GB.

In my view, you should take both as you don't require system image every time.

a. user say:

Nothing that Windows 10 has can do what you need to do.

I can explain in a separate post, if you wish. For now it's more important to start backing up. You need to make an image, which is also known as a backup image or a system image. Three terms for the same thing.

An image is an exact copy of a partition (or partitions) compressed into a single container called an archive. When you make an image of your hard disc, you are making an exact copy of everything on your computer, exactly the way it is now.  You can't do this with anything Windows has.

A restore point is not a backup, and it never was intended to be a backup.

To create an image you need a third-party image application. There are many of these, both free and paid. We can talk about brand names in a separate post, or others will make suggestions to you. For now, know that Macrium Reflect is top quality software and has both a free and a paid version. I have the paid version myself because I'm a huge believer in backing up.

It takes time to learn how to use image software. It's not just copying and pasting. The time that you invest in learning how to use the software will be rewarded many times over once you realize that having an image makes you as close to bullet-proof as you can possibly get with a computer. You will never lose data and you will never need to re-install anything, ever again (unless you want to.) I've never used Time Machine but I can't imagine how it can do anything more than this. An image is the ultimate protection.

Making an image with the software takes only minutes. Restoring an image also takes just minutes, and when you restore an image your computer goes back in time to exactly the way it was when the image was made.

Image software lets you schedule images to occur when you want, as often as you want, automatically, in the background so you can continue to use your computer while an image is being made. My images are made automatically, overnight.

You can't learn to make and restore images in a community forum. This is not a "follow these three easy steps" procedure. Pick an image program to start and carefully work your way through the instructions. There is a lot of good information about this on the internet - take advantage of it. You will never regret having spent the time.

I hope disaster never strikes you, but if it does, you can easily and confidently go back in time to when everything was good, in just minutes, while your colleagues here are storming the forum with wild, raving posts blaming everyone else for their problems.
