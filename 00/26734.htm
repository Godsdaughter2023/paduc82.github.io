graham mayor say:

There are four common causes of misalignment

1. You have added a header/footer to the normal template (rename the normal template)

2. The page size set in the printer properties doesn't match the page size of the label sheet.

3. You have a scaling option set in the zoom section of the print dialog

4. Your printer has poor paper handling. Some printers will slip when faced with label stock and thus are incapable of feeding the label correctly. Cleaning the rollers in the print path may help.

'3' is the most common problem, as has already been suggested.

Bing Desktop and built-in theme automatic theme colors with build 9926

Any Bing Desktop users notice that the color theme that goes along with the wallpaper of the day stopped syncing after 9926?  I gave it a couple days just to make sure that it isn't solely due to the same theme color being used for a couple days in a row and sure enough, it's not updating, just sticking to the start menu color carried over from Windows 8.1/Build 9879.  This also happened when choosing any other themes with automatic colors under Control Panel > Appearance and Personalization > Personalization.

I was able to sync the theme color, albeit manually, as follows:

  1. Open up Color and Appearance for that theme
  2. Click on an arbitrary color
  3. Hit cancel.  The color will sync up canceling.
