hansv mvp say:

Indirect's argument should be a reference to a cell which has a cell address. In your data C$1 contains: ActualCY, which is not a cell reference.

That is not correct. If ActualY is a defined name that refers to a fixed range, and if C1 contains the text ActualY, INDIRECT($C1) will work.

In the screenshot below, MyName has been defined as the range A1:A5. Cell D1 contains the name MyName.

The formula =SUM(INDIRECT(D1)) works correctly.

But, as Kirk P. has found, it doesn't work if ActualY is a defined name that refers to a dynamic range. The range being dynamic adds another layer of indirectness and that's too much for Excel.
