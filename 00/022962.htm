joeu2004 say: slyth wrote:
Is it possible to write logic gates within conditional statements (e.g =if)
For example:
A1=0
B1=0
C1  =IF(A1=0&B1=0,1,0)  - returns 0
Does Excel have a constuct for AND, OR, XOR, NAND, NOR, and, NOT?

Not as masking operators.  But as boolean operators:

=AND(A1,B1)
=OR(A1,B1)
=NOT(A1=B1)
=NOT(AND(A1,B1))  ...or...  =OR(A1=0,B1=0)
=NOT(OR(A1,B1))    ...or...  =AND(A1=0,B1=0)
=NOT(A1)

Caveat:  Normally, A1 is considered FALSE if it is zero, otherwise it is TRUE.  But NOT(A1=B1) is XOR only if A1=0 and B1=0 or if A1=x and B1=x (i.e. same x) where x is some non-zero value.
