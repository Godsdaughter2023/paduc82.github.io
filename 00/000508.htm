when external user clicks the first link <file name> in the mail, they have access to view/edit the file using the no sign-in required edit link, which is not the same as we directly share file with a user. therefore, the external user doesn’t show in permission page/sharing column.
if we want to control this kind of access, we need to modify the view/edit link created in get a link tab.

when external user clicks the sign in link in the mail, they are asked to sign in with microsoft account or organizational account. after that, the external user profile is created in sharepoint and the user account is granted view/edit permission to the file. so in onedrive for business library, we are able to see the user account in permission page/sharing column.

in a word, for invite people with required sign-in unchecked condition,
users clicking the <file name> link are given access based on the no sign-in required edit link. no account is created and shown.
users clicking the sign in link are given access by directly granting permission to their external account. accounts can be found.

Hi,

 

Try these.

 

If you want the answer in time format use this and format the cell as time

 

=(B1-A1+(B1<A1))-C1

 

If you want the answer as a decimal number of hours use this formatted as general.

 

=((B1-A1+(B1<A1))-C1)*24

 

Both formula work if the time is in the same day or over midnight
