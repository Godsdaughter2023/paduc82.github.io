teylyn say:

Hello,

Is the service provider in column P or is the formula in column P?

Let's assume it is the service provider in column P. 

You could have a list of service providers in a sheet. Next to each service provider, put the number of columns that this service provider's data is to the right of the column B, i.e. where the first service provider is. So the first one is in B and their number is 0. The second service provider's table starts in column H, which is 6  columns away from B. The third service provider table is 12 columns away from B. And so on. Let's call this "TheOffsetTable".

To work out the offset for a particular service provider, you can use a Vlookup

=Vlookup(P2,TheOffsetTable,2,false)

That returns the number, e.g. 12 for service provider three. Now, add this number to the column match of your index function. You can do it all in one formula:

=INDEX(C10:F20,MATCH(L10,B10:B20,0),MATCH(U10,C9:F9,0)+Vlookup(P10,TheOffsetTable,2,false))

Does that help?

amit tandon say:

Couple of options.

Option 1:

=INDEX(OFFSET(C10:F20,,(MATCH(P10,{"SP1","SP2","SP3","SP4","SP5","SP6","SP7","SP8","SP9","SP10"},0)-1)*5),MATCH(L10,B10:B20,0),MATCH(U10,C9:F9,0))

The formula can be shortened by using a table of values for Service Provider instead of

"(MATCH(P10,{"SP1","SP2","SP3","SP4","SP5","SP6","SP7","SP8","SP9","SP10"},0)-1)*5".

Note: SP1. SP2, ... represent the Service Providers. The above uses the Offset function which is volatile - causes recalculation of the formula in the cell where it resides every time when calculation occurs in the worksheet. This slows your sheet hence it should be used only if your sheet data is small.

Option 2:

This obviates using the Offset function.

=INDEX(C10:AY20,MATCH(L10,B10:B20,0),MATCH(U10,C9:F9,0)+(MATCH(P10,{"SP1","SP2","SP3","SP4","SP5","SP6","SP7","SP8","SP9","SP10"},0)-1)*5)

This adds an Index to the column number using the Match function for the appropriate table (this adds an index of 10 for table number 3, so instead of column E for 1st service provider range in C10:F20, it will select column O for 3rd Service Provider). In this case the Indexed range covers range of all the 10 tables - C10:AY20.

Mms on t mobile with lumia 1020 - Microsoft Community

flybynight1 say:

Mms on t mobile with lumia 1020

I recently switched from att to t mobile. I have a nokia lumia 1020. My mms settings have been confirmed correct by tmobile. However I can no longer text photos. Any suggestions?

gerald mal say:

Hi,

Restart your phone with a SIM card inserted, and retry sending a multimedia message (your phone loads the settings from the SIM card).

  • Tip! Send a multimedia message to your own mobile phone number. This will test the sending and receiving capabilities in one go.
  • Important! Multimedia messaging requires cellular data connectivity.
  • Note! If you have two SIM cards assigned to the same phone number, only the primary SIM card can receive multimedia messages.

If you can connect to the internet but you cannot send multimedia messages, try entering the settings manually

  1. Try searching online to find the multimedia settings for your mobile operator. To add your settings manually, go to Settings > Network & wireless > Cellular & SIM > SIM settings. At the bottom you should see MMS APN. Tap +. 

    Lumia with Windows Phone 8 (depending on your phone model):
    • Option 1 - Go to Settings > access point, and tap +.
    • Option 2 - Go to Settings > cellular+SIM > SIM settings > SIM, and tap add MMS APN.

If you cannot connect to the internet using a mobile data connection

  1. Enable the mobile data connection on your phone. Go to Settings > Network & wireless > Cellular & SIM (Lumia with Windows Phone 8: Settings > cellular+SIM) and switch Data connection to On.
  2. Check that data roaming is activated when you are not in your home network. Go to Settings > Network & wireless > Cellular & SIM (Lumia with Windows Phone 8: Settings > cellular+SIM) and select Data roaming options.
  3. Make sure you have not reached your data limit. Go to Settings > Network & wireless > Data usage.
  4. Ensure background data is not restricted. Go to Settings > Network & wireless > Data usage > Edit. Switch Restrict background data to Never and/or Restrict background data usage when roaming to Off. 
    Lumia with Windows Phone 8: From the Start screen, swipe left to the App list and start Data Sense app. Switch Restrict background data usage and/or Restrict background data usage when roaming to Off.
  5. Confirm with your mobile operator that you have an active mobile data subscription (access to internet) on your SIM card.

Regards.
