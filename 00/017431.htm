Changing the default font is a change to the normal template (normal.dotm).

There are sometimes other programs (called Add-Ins) interfere with saving these changes.

Do the changes last through the same Word session? Try doing a default change and then creating a new document during that same Word session. Are the changes still active? If they are write back and I or someone else will give you some work around and possible long-term solutions.
