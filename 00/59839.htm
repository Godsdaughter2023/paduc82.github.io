don'spc say:

is it possible to straighten a crooked picture in paint?

i know how to crop but can't figure out if there is a way to straighten out a crooked picture and i see no help button in the app either.
cmdrkeene say:

The help button is in the very very top right corner, but I can tell you that no, this is not a function of paint.  It can rotate pictures (or sections of a picture) but only in 90 degree increments.  Paint is pretty limited and old (it was created in 1983 and only had the tiniest of small updates since then).

But, you can do this in the built-in Photos app in Windows 8. Here's how:

  1. Right-click the Picture file and choose "Open with: Photos" (Photos may already be the default app, but this will make sure it's opened in it).
  2. Click the picture once to show the toolbar, and click Edit at the very bottom.
  3. Click Basic Fixes on the left side, then straight on the right side.
  4. When you're done straightening it or making other edits, click once on the picture to show the bottom toolbar again, and choose to either "Save as" a new picture, or "Update original" to modify the original picture file.

can somebody tell me how to open ports 8090, 8091 and 8092 - Microsoft Community

kazt279 say:

can somebody tell me how to open ports 8090, 8091 and 8092

I have windows vista home premium with norton 360 and a motorola sbg900 cable modem
anannya podder say:

Hi kazt279,

Thank you for writing to Microsoft Answers Community Forums.

This port is a somewhat less popular alternative to port 80 for offering web services. (The more popular alternate port is 8080.) Like 8080, its use in a URL requires an explicit "default port override" to request a web browser to connect to port 8080 rather than the http default of port 80.

 

Its current use and its importance today, is due to is adjacency to the super-popular "http" World Wide Web port 80. Sometimes an "off the beaten path" web server will be established to listen not to port 80, but to an alternate related port such as 81, 82, 8080, or 8090. (Related by proximity or appearance.) So port 81 is only noteworthy today because it is a neighbor to port 80 and is sometimes used, as an adjunct or alternative, in the same way.

 

However, as we explained on our page for port 80, port 80 is the "default port" for the web's http protocol. How, then, if we wish to use an "alternate port" like 81 or 8090, do we override this default to specify the port we desire?

 

                                                                      http://www.domain.com:81/

 

If we wish to override the default service port that is implied by the protocol (http), we place a colon (:) after the domain name portion followed by the port we wish to use. The URL shown above would cause the browser to override its normal use of port 80 for http, and instead attempt to connect to a server listening at "www.domain.com" for connections on port 81.

 

If you ever see a URL using this format, with some number following a colon after the URL's domain, you'll know that this URL is asking the browser to connect to a server listening for incoming connections on the specified port rather than its default.

 

Since you give exceptions to these ports in the windows firewall or any other 3 party firewall they go unmonitored which means they are susceptible for hackers. I would suggest not enabling the ports unless it is very important. If you enable them make sure you do not keep any important information on that computer.

 

Hope this helps.
