Hello.  I want to create a drop-down list that shows one thing in the list but then displays something else.  I have about 20 cost centers that I want to spell out in the drop-down list but display the codes we use for each center when a specific cost center is chosen.   Just to use a simple example I'll use names, I want the list to show Bob, Fred and George but when one is chosen I want it to just display the first initial so if you choose Bob it will display a B.  Is this possible in Excel? 

gord dibben say:

It is possible to do this with VBA event code.

Copy this code then right-click on your sheet tab and "View Code"

Paste the code into that module, edit to suit then Alt + q to return to sheet.

Private Sub Worksheet_Change(ByVal Target As Range)
'show just first letter from dv list
Const WS_RANGE As String = "A1,A2,A3"  'edit to suit
Dim cell As Range
Dim oval As String
oval = Target.Value
    On Error GoTo ws_exit:
    Application.EnableEvents = False
    If Not Intersect(Target, Me.Range(WS_RANGE)) Is Nothing Then
    With Target
            If Len(oval) > 1 Then
              .Value = Left(oval, 1)
            End If
    End With
    End If
ws_exit:
    Application.EnableEvents = True
End Sub
