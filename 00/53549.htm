frederik long say:
You can use the recipe below to create several Start Menu entries for the same program. I shall illustrate it for Google Chrome. On my PC, Chrome gets invoked through
c:\Program Files (x86)\Google\Chrome\Application\chrome.exe.

  1. While in the Chrome\Application folder, create as many unique copies of chrome.exe as you need, e.g.
    Startpage.exe
    Yahoo.exe
    You will need elevated privileges to do it.
  2. Using the right mouse button, drag the new files to the desktop to create shortcuts..
  3. Add the desired URLs to the shortcuts, e.g.
    c:\Program Files (x86)\Google\Chrome\Application\Yahoo.exe www.yahoo.com
  4. Copy the shortcuts into this folder:
    c:\ProgramData\Microsoft\Windows\Start Menu\Programs
  5. If you wish to create Start Menu tiles, right-click the desktop shortcuts, then left-click Pin to Start.
  6. Delete the desktop shortcuts.
If this works for you then please remember to mark my reply as "Answer" to your query.
