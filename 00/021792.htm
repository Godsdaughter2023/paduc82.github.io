e.j. gun say:

Ok, with a little help from OssieMac on the TripleState, I have come up with the code below.  It assumes that your checkboxes will be in a column all by themselves, that the column to the right of the checkboxes holds the values you want to see (YES, NO, N/A), and that column "BB" holds the actual value of the checkbox (TRUE, FALSE, #N/A).  Give it a try.

 

The code first deletes all OLE objects on the sheet (not just checkboxes) and then adds a checkbox to each row that has data.  Then the code adds a formula in the cell next to the checkbox to convert the value of the checkbox to "YES", "NO" or "N/A".

Eric

'===== BEGIN CODE =====

Sub AddCheckBoxes()
    Dim i As Long
    Dim nRows As Long
    Dim cbxColumn As Long
    Dim tmpColumn As Long
    Dim objOLE As OLEObject
'
    cbxColumn = 2   ' (Column B)
    tmpColumn = 54  ' (Column BB)
' Find the last row of data on the sheet.  CAUTION:  Assumes just one table of data on the sheet!
    nRows = Cells.Find("*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
'
    With ActiveSheet.OLEObjects
        For i = .Count To 1 Step -1
            .Item(i).Delete
        Next i
    End With
   
    For i = 2 To nRows  ' Assumes row 1 is a header row
        ActiveSheet.Cells(i, cbxColumn).ColumnWidth = 4#
        Set objOLE = ActiveSheet.OLEObjects.Add(ClassType:="Forms.CheckBox.1")
        With objOLE
            .Top = ActiveSheet.Cells(i, cbxColumn).Top + 1
            .Width = 10.5
            .Height = ActiveSheet.Cells(i, cbxColumn).Height - 2
            .Left = ActiveSheet.Cells(i, cbxColumn).Left + _
                    0.5 * ActiveSheet.Cells(i, cbxColumn).Width - _
                    0.5 * .Width
            .Name = "cbx" & i
            .LinkedCell = ActiveSheet.Name & "!" & Cells(i, tmpColumn).Address
            .Object.BackStyle = 0
            .Object.TripleState = True
            .Object.Caption = ""
        End With
        ActiveSheet.Cells(i, cbxColumn + 1).Formula = "=IF(ISERROR(" & _
                ActiveSheet.Cells(i, tmpColumn).Address(RowAbsolute:=False, ColumnAbsolute:=False) & _
                "), ""N/A"", IF(" & _
                ActiveSheet.Cells(i, tmpColumn).Address(RowAbsolute:=False, ColumnAbsolute:=False) & _
                "=TRUE,""YES"",""NO""))"
    Next i
End Sub
'===== END CODE =====

brian tillman [outlook mvp 2007-2019] say:

It is extremely easy for spammers to use any address they choose as the sending address of junk and it doesn't mean in any way that the account associated with that address has been compromised. The spammer harvested your address from somewhere - Facebook, Twitter, the account of someone you who has you for a contact, or any number of other possibilities - and inserted it as the sender address for the junk. The spammer then sent those messages to hundreds of thousands - perhaps millions - of other addresses in his database. Many of those addresses are bogus or otherwise inoperational and so the receiving mail system rejects the messages as undeliverable, sending the non-delivery reports to the address it sees as the sender: yours.

There's very little you can do, since the actual sender can be anywhere in the world. At best, you may be able to create a rule or rules that can redirect the non-delivery reports to the Deleted folder.
