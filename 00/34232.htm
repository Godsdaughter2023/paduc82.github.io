Hello Jeff99999,

 

The system was obviously confused. Windows expects the hiberfil.sys file to be on the C: drive and not on any other drive. Disabling the option from Power options would simply disable hiberfil from you current C: drive as would using the usual powercfg-H Off command at the command prompt.
