tryx3 say:

You are running all three computers using the Built-In Admin account. 

This is risky as any user profile corruption affecting that account cannot be fixed and you will need to reinstall Window 7 to fix problems.

You brother's computer exhibits symptoms of user profile corruption and, as it is the "Administrator" account, the problem cannot be fixed.

There is a chance that repeated attempts to use the corrupted account to create a new account might succeed i.e. it might just manage to create a new account that is not itself corrupted.  But this really is clutching at straws.
