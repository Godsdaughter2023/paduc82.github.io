e.j. gun say:

Here's an example, which has a routine to add IFERROR, and one to remove it again.  The sub "TestIt" shows how to call the other routines.  You can wrap the calls in a loop through all the worksheets in your workbook, or a subset of them.

 

HTH,

Eric

 

Option Explicit

Sub TestIt()
    Call WrapIfError(ThisWorkbook.Worksheets("Sheet1 (2)"))
    Call RemoveIfError(ThisWorkbook.Worksheets("Sheet1 (2)"))
End Sub
'
Sub WrapIfError(theWS As Worksheet)
    Dim frmRange As Range
    Dim theCell As Range
    Dim theForm As String
'
    Set frmRange = theWS.Cells.SpecialCells(xlCellTypeFormulas, 23)
    For Each theCell In frmRange.Cells
        If (Left(theCell.Formula, 1) = "=" And Left(theCell.Formula, 8) <> "=IFERROR") Then ' just to be sure it's and Excel worksheet formula...
            theCell.Formula = "=IFERROR(" & Right(theCell.Formula, Len(theCell.Formula) - 1) & ","""")"
        End If
    Next theCell
End Sub
'
Sub RemoveIfError(theWS As Worksheet)
    Dim frmRange As Range
    Dim theCell As Range
    Dim theForm As String
'
    Set frmRange = theWS.Cells.SpecialCells(xlCellTypeFormulas, 23)
    For Each theCell In frmRange.Cells
        If (Left(theCell.Formula, 8) = "=IFERROR") Then
            theCell.Formula = "=" & Mid(theCell.Formula, 10, InStrRev(theCell.Formula, ",", Len(theCell.Formula), vbBinaryCompare) - 10)
        End If
    Next theCell
End Sub

rhapsodicrose say:

I had this same issue, and was finally able to figure it out. Outlook uses the first account you add as the Primary Account. It seems that "default send account" really means nothing. To use Mail Merge to send from an account that is not your primary account, you need to create a new Mail Profile.

1. Close Outlook 2013
2. Open Control Panel, click on Mail and choose Show Profiles. (Start > Control Panel > Mail > Show Profiles)
3. Add new mail profile and click "Prompt for a profile to be added." Click Apply.
4. Open Outlook 2013, a dialogue box will appear. Choose the mail profile to use for mail merge. Outlook will setup the new account for the first time, so this could take a few minutes.
5. Use Mail Merge in Word and it will send out through the Primary Account in the Outlook that is open.
