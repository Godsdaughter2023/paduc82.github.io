Hi CB,

 

OneNote has an OCR mechanism of its own which is able to find and index text inside image so it can be searched for. In OneNote 2016 for Windows,  this OCR mechanism is in the operating system and fast. In OneNote for Mac, an OCR engine is deployed on the Microsoft servers. scottgem say:

Yes it is an old post that was already resolved so we prefer that you not revivie old threads.

However, to explain this, the Switchboard Manager wizard that Access provides creates a shadowed label for the name of the current switchboard page. it does this by having label1 and label2 slightly offset over each other and using different colors to create the shadowed affect. Therefore when you change Switchboard pages, the code behind the Switchboard changes the Caption property of both labels to display the page name. If you edit the Switchboard form and delete or rename an object (or even rename the form) you will break the code behind it. So you should either leave the objects alone or edit the code referencing the objects you delete or rename.
