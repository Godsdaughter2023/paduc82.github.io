Actually, that's a co-installer and it doesn't have anything to do with coins.

Intel GFX, as you probably know, stands for Intel graphics. Your computer might have an Intel graphics chip whose software used a co-installer as it was being installed.

Method 2:

If you are able to boot your computer in Normal mode, I would suggest you to follow the steps mentioned below.

Step 1:

Follow these steps:

  1. Press Windows key + R
  2. Type services.msc in Run, and then press Enter.
  3. If you are prompted for an administrator password or for a confirmation, type the password or click Continue.
  4. Locate Background Intelligent Transfer Service and check the Status.
  5. If the Status is Stopped, right-click Background Intelligent Transfer Services, and then click Start in the menu.
  6. Locate Windows Update and check the Status.
  7. If the Status is Stopped, right-click Windows Update, and then click Start in the menu.
  8. Locate Workstation and check the Status.
  9. If the Status is Stopped, right-click Workstation, and then click Start in the menu.
  10. Close Services.

Step 2: Run these commands in Command Prompt (Admin).

  1. Click on Start.
  2. Type CMD and right click on it and select Run as administrator.
