stefan blom say:
You would have to do this by anchoring a frame to the header paragraph and manually entering a PAGE field and a number in the frame. To insert the frame, use the Insert Frame button on the Forms toolbar. (To create each "line" in the frame, you would have to press Enter.) Since header contents repeat, you would get the desired numbering on all pages (using that particular header). If you can use a fixed value for line spacing in the document, that would help.
 
For what it's worth, this is how the Legal Pleading template works (except it doesn't include page numbers with the manual line numbers).
