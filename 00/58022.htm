cbarnhorst say:

Yes, virtual machines must have separate licenses.

You can install multiple copies of the same edition of Windows 10 on partitions/drives on the same computer without needing additional licenses but you do need a separate license for a virtual machine whether in Hyper-V, VMWare, VirtualBox or any other.  This was confirmed by Gabe Aul a year ago.  The reason is virtual machines need to be portable and cannot depend on the activation status (or even the OS) on the host in order to remain activated.  Keep in mind that Hyper-V is primarily designed to run virtualized servers in a networking environment where vms can be shifted between hosts for such reasons as load-balancing.

What you can do is make a new partition, clean install a new copy of Windows 10 (same edition), and when you get to the product key request click the link at the bottom that says "I don't have a product key," and on the next page be sure to check the right edition.  Installation will complete and when the computer connects to the internet Windows will activate based on the fact that the computer has already been registered as a Windows 10 device for that edition.

This is a good way to join the insider program without needing to risk your primary installation or buy a new license (product key).
