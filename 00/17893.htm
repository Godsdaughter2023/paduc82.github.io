Unfortunately, the templates for the Avery Tent Cards are not properly setup and there is quite a bit of work to do to produce a satisfactory result.

The steps are as follows:

  1. On the Mailings tab of the ribbon, expand the Start Mail Merge item and select Labels
  2. In the Label Options dialog, select Avery US Letter from the Label vendors drop down and then select 5302 Small Tent Cards from the Product number list and then click on the OK button
  3. At this stage, you can then use the Select Recipients facility to attach the data source to the mail merge main document.

    Here's where all the work starts

  4. Access the Page Layout tab of the ribbon and use the dialog launcher in the Page Setup section to access the Page Setup dialog.
  5. On the Margins tab of that dialog, change the Orientation to Landscape and change the margins as follows:

    Top 0.5"
    Bottom 0"
    Left 1.25"
    Right 1.25"

    and click on the OK button
  6. From the Table Tools>Layout tab of the ribbon, access the Properties dialog and make the following changes:

    First Row Height 3.5"
    Second Row Height 0.5"
    Third Row Height 3.5"

    First Column Width 4"
    Second Column Width 0.5"
    Third Column Width 4"

    Then click on the OK button
  7. Select the first column of the table and use the Split Cells facility to split that cells in 2 columns and 3 rows unchecking the box for Merge cells before split.
  8. Repeat step 7 for the third column
  9. Select the top left cell of the table and use the Text Direction facility in the Alignment section of the Table Tools>Layout tab to change the text direction so that it runs from top to bottom with the text centered in the cell (vertically and horizontally)
  10. Select the second cell in the top row and change the text direction so that it runs from bottom to top with the text centered in the cell (vertically and horizontally,
  11. Repeat step 9 for the fourth cell in the top row and the first and fourth cells in the last row
  12. Repeat step 10 for the fifth cell in the top row and the second and fifth cells in the last row
  13. Insert your merge fields into the first, second, fourth and fifth cells of the first and the last row of the table (It is a good idea to use Alt+F9 to toggle on the display of the field codes to ensure that the { NEXT } field is located in the fourth cel of the first row and in the first and fourth cells of the last row and that the mergefields in those cells are located after the { NEXT } field. Use Alt+F9 again to toggle off the display of the field codes)
  14. You should now be able to execute the merge.

suzanne s. barnhill say:
Yes, Word has a References tab, from which you can insert citations and create a list titled either "Bibliography" or "Works Cited." If you mouse over the latter two in the Bibliography gallery, you will see that the definition for both is the same. I assumed you wanted to know what the actual difference is. A Works Cited list is exactly what it says, a list of the works cited in the document. A bibliography is (or at least certainly can be) something entirely different: a list of works recommended for further reading (even if there have been no actual notes or citations in the work), or a list of all the works related to a topic, or a list of all the works written by a specific author. A References list can be the same thing as what Word calls Bibliography or Works Cited (this is the title stipulated by APA style), but it can also be a numbered, alphabetical list of works cited, allowing the numbers (as cross-references) to be used like endnote reference marks.

scottgem say:
You need to create a multi valued field. This is done in table design mode. Select the field and then select the the Lookup tab under Field Properties. Change the display Control to Comobox, the Row Source type to Table/Query. Enter a RowSource and set the Allow Multiple Values to Yes.

To set this up on your form, you drag the field from the Available fields list unto your form.
