Hello JamesKCambellJr

I found also that the Windows DVD Player does not work automatically.  However, this evening I have just spent a few minutes trying to put this right.

Firstly, you must set the 'Windows DVD Player' as the 'default' player for the .VOB file extension, being the file type for DVD recordings.  To do this you can open the 'System' configuration tools in 'Settings' and on the left click on 'Default apps'.  The DVD Player option may not show, so click on 'Choose default application by file type'.  In the long list presented scroll down to find the .VOB file type (using the wheel in your mouse is best for this).  If the .VOB file type is associated with anything other than the Windows DVD Player it needs to be changed. Having said this, you may find the 'Windows DVD Player' is not available from the given list of options; which is what I found.  So, I changed to the control panel interface for assigning the default player.  On the keyboard hold down Windows Key and X to bring up the access menu and click on Control Panel.  In Control Panel choose Default Programmes, and then choose Associate a file type with or protocol with a specific programme.  Again scroll to find the .VOB extension and when this is selected click the Change Programme button top right. Now you should be able to access the programme in the programmes folder on the C: drive and within the 'Windows Apps' folder find the Windows DVD Player folder, which when opens reveals the .EXE file for the DVD Player App.  Choose this, close back to the Desktop.  Close to your Desktop and simply open the BD-DVD Drive, place in your DVD and it should 'Auto Play'.  It may take a while to boot, but this has worked for me. You may have to right click the drive in File Explorer and choose Auto Play.  Let me know if this works for you. 

Cheers, DaveJB.
