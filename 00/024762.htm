shuntfield---01 say: OK I didnt know this feature existed but I have just tested it with mine and it functions correctly.

Go to Menu --> Settings --> Tools --> General --> Security --> Phone and SIM card.

Go To "Allow Remote Lock" we will assume that the default Lock Code of 12345 is still set first. Switch the "Allow Remote Lock" from No to Yes, it will ask you what you want the message to be, so for example "lockmye71" you have to enter this twice and then it will ask you for your phones lock code which is (default) 12345, the lock will then be set, when you send a message with "lockmye71" on its own to the phone the phone will lock and then reply to the number you sent the message from saying that it has been locked as a confirmation.
