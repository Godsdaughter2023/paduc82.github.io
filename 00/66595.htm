Hey Joshfletch,

 

These steps may help you in playing the game with fewer crash.

 

Step 1: Run the game directly from the source folder.

 

It appears that if you run the game through the launcher, the game may crash after 20 mins or a few hours of gameplay, most notably when entering/exciting areas or battles with many opponents. To prevent this you will need to run the game directly from its source as administrator.

 

1.     Right-click on witcher.exe file in the witcher\system folder.

2.    Select Run as Administrator.

 

Step 2: Reduce the number of game saves.

 

The time required to display the save or load screens apparently increases dramatically as the number of saves increases. Hence, it is recommended that the number of savegames be kept under control to avoid this. The saves are stored in Documents\The Witcher\saves.

 

Step 3: Defragment your C: drive.

 

The save files are placed on drive C: which gets fragmented a lot and the free space is usually fragmented too. Defragment the hard drive or leave a lot of free space so new saves don’t get fragmented as much, making saving and loading faster.

 

Refer this article for help with disk defragmentation.

 

Improve performance by defragmenting your hard disk
