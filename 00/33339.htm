slance310 say:

Hi,

For the long term of things, I think your best bet would be to start over...

Either with a 2TB drive, or switch the 5TB to GPT during the install...

Just as a note... I run all of my machines that allow it, to run as Legacy Bios...

If you have boot problems with a UEFI machine, they can be much more difficult to deal with...

One reason for this is that the Boot Files are on a separate EFI partition, and the Boot Device is "WindowsBootManager"... Not the actual drive\partition...

Another note concerning Windows 7 specifically...

If you install Windows 7 to a unallocated partition, it will also create an extra partition... Similar to the EFI partition, but different...

Regardless, the boot files end up on the extra partition...

If you create the partitions BEFORE installing Windows 10, this does not happen, and the boot files are on the OS partition...

ISSUE:

I was reading an old post where a person was asking about Windows 8 features. After setting a default type of environment (which will be loosely based on the old "types", Home, Pro, Ultimate, etc.) the user will be asked if they want to activate certain features, at which time they will be able to CUSTOMIZE windows to what THEY NEED.
