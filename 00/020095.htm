Feedback: Make Moving Windows 10 installations to SSD or Mechanical hard disk vice versa easier

One of the heah aches or annoying chores of Windows is its ability to grow with the user needs or budget. The Windows Team should ensure this functionality is part of Windows 10. It has been missing in 4 generations of Windows (Vista, Windows 7, Windows 8/8.1 now and thats far too long. System Imaging should make it easier to move a Windows 10 installation whether you are going from a small SSD to a large mechanical hard disk or vice versa. Windows as far back as Vista (editions Business and Ultimate) had Complete PC Backup (System Imaging in Windows 7) that makes it very easy to move a Windows installation from mechanical hard disk to another mechanical hard disk as long as the disk alignment is similar in terms of size. Windows 10 should also fix the issue of the storage device either being equal in size or greater when restoring a system image. As long there is enough space to hold the image being restored, it should not be a problem.
