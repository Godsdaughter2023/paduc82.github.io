I find Avery badge inserts very problematic because printing them correctly is highly dependent on how successful Word is in communicating with your printer. Depending on whether your printer prints narrow paper to the left or centered (which can depend on how your paper guides work), this can be a nightmare, but you need to figure out where the printer expects the strips to be and then feed them there.

Is Microsoft Document Imaging set as the default viewer for .tiff files?

 

By default opening a .tiff file will only display one page at a time in Microsoft Office Document Imaging (MDI).

 

To view the .tiff files in multiple pages, one has to manually insert the file within the document. To do that, do the following:

 

Open Microsoft Office Document Imaging.

Click on File and then click on Insert File.

Select the image and then click on Insert. (In MDI only one image can be selected at the time when using the insert file option).

In the pop up Insert File window, select the radio button At the end of the document and click on OK.

charles kenyon say:
Hi,

We will need much more information to even attempt an answer to this.

How will you identify what is to be imported?

What do you mean by import data from pdf files? Pdf files are not especially good repositories for data. Since Word 2013, Word has had the ability to directly open pdf document. However, some pdf documents (from scanners mostly) are really pictures of text rather than the text itself. Just as you can't look under the hood of a picture of a car, you can't edit a picture of a document.

In that case, you need to convert the picture to text. This is a process known as optical character recognition. This is built into Adobe Acrobat and is also in Office OneNote. Most scanner software comes with an OCR component as well. Once you have text, you can edit it with Word.

If you simply want to write on the document (but not in it) you can add a Text Box floating on top of the document layer, whether or not it has been put through the OCR process.

Word documents that have been saved as PDF will not need the OCR process, they retain their text, although not all of their Word structure and formatting.

Finally, documents converted from pdf (or really any other format) to Word can be tough to edit because the conversion process never has a one-to-one matching of how formatting is done. This means that a converted document will seldom be formatted in Word in a way that makes sense. An example is multiple section breaks to change margins, where in Word you would simply change the paragraph indent.

I hope this information helps. Please let me know if you have any more questions or require further help.

Regards
