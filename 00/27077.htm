dennis wi - msft say:

Hello D,

 

The hyperlink control will only open the form object. However, there are a number of ways you could have your second form open to the same record as your current form.

 

1.     You could create a label field on your current form that looks like a hyperlink field and set the On Click event for that label to use the OpenForm method where you can specify a filter or where clause.

 

DoCmd.OpenForm Method

http://msdn.microsoft.com/en-us/library/bb238021.aspx

 

2.     You could apply a filter to the second form when it’s opened where the filter has a form reference to a field on your current form.

 

My sample uses Form1 and Form2. When I click on the hyperlink field in Form1, I open Form2. In the form On Load event for Form2, I added this code:

 

Private Sub Form_Load()

    Me.Filter = "Field1 = '" & Forms!form1!Field1 & "'"

    Me.FilterOn = True

End Sub

 

This code applies a filter to Field1 in Form2 with the value of Field1 from Form1. Note this sample assumes Field1 is a text datatype field. If you were filtering a number the code would look like this:

 

Private Sub Form_Load()

    Me.Filter = "Field1 = " & Forms!form1!Field1

    Me.FilterOn = True

End Sub

 

Regards,

Dennis
