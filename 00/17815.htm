daniel g. grau say:

When you use Word's existing tab in the Ribbon Document Elements > Word Art, it inserts the "new style" WordArt elements, which Word henceforth considers to be shapes. Right-clicking (or Control-clicking) on them shows either "Format Text Effects" or "Format Shape," but neither of these contextual-menu choices nor my other attempts allowed me to fill up the characters with an image.

However, WordArt objects fetched over from Word 2004 do show "Format WordArt" when right-clicked on and can effectively be filled with pictures in Word 2011. So the functionality is there. I'll describe how to bring it back.

To bring the old functionality back, you need to add a missing command. You can add it to an existing menu or toolbar, or create a new toolbar and add it there. The latter is what I'll describe.

To create a new toolbar, select View > Customize Toolbars and Menus, make sure to select the "Toolbars and menus" tab at the top, then click on the New button. This will produce a small, blank toolbar, which is a bit difficult to spot on the screen.

To add the missing command to that toolbar, click on the second tab, Commands.

  • Make sure All commands is selected in the list to the left
  • In the list to the right, scroll down to the DrawInsertWordArt command that has a slanted "A" icon:

             

  • Drag the command highlighted above to the new toolbar, where it will display as an icon.
  • Click OK to close the dialog box

Clicking on the added icon will produce the old Word Art selection box (although the menu do not quite match the result):

            


Once you insert a Word Art object, the Format tab in the Ribbon takes on a new look!

            


In this new tab, you can click on the bucket tool and choose Fill Effects, shown above. And you can insert a picture as a fill (in this case, Apple's Aurora desktop picture):

            
If you work in a document saved in the old Word 97-2004 format (.doc extension) or if you select an old WordArt element in a .docx, the Ribbon's Document Elements > Word Art button (with the same icon as before) will give you a whole new set of pre-made effects.

All in all, the new button is a way to have both types of effects available without resorting to cutting and pasting between documents.
