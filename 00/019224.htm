This behavior may be caused by Internet Explorer opening as an administrator since this happens during the installation process of programs and no other time. You could test this by right clicking Internet Explorer and then Run as Administrator.

 

If you experience the same thing as what happens when you install a program this is caused by way Windows Vista and 7 handle User Account Control.

stefan blom say:

When you Select All in the main body of the document, Word does not select anything else, that is, headers, footers, text boxes, footnote text, endnote text etc are not included in the selection. This is normal behavior.

However, endnotes (and footnotes) should still be copied along with the main body, because the notes go with their endnote (or footnote) reference marks in the body text. This should definitely work as long as both source and target documents are in Word format.

Some setting, such as suppressed endnotes in a section, may be the underlying cause of trouble here.

Note that my experience is exclusively with Word for Windows. Therefore, I will move your question to the Office for Mac section rather than trying to guess what is wrong...
