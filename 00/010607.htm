dinororg say:

Calculating with numbers based on position

How do I calculate with numbers based on their position in a string? For example, I have account number 2020418037. I need to multiply every other digit by 2, starting with the first digit, then I need to add the resulting numbers together. This means 2*2 + 2*2 + 4*2 + 8*2 + 3*2.

Of course if I only had a few numbers to calculate this way it would be easy, but I have hundreds of numbers. This is only the first in a series of calculations I need to do based on digit location, but if I can get started with help here, I may be able to figure out the rest. Thanks.

ron coderre say:

If you're looking to calculate the "checkdigit"...

This is one approach

 

With

A1: 2020418037

 

This regular formula calculates the check-digit of that number.

B1: =INT(A1/10) - FLOOR(INT(A1/10) - 1, 9)

 

It uses this logic:

1) Sum of 2 times every other digit, beginning with the first (ignoring the last digit)

2) Sum of every other digit, beginning with the second (also ignoring the last digit)

3) Sum of steps 1 and 2 results

4) Sum of the step 3 digits

5) Mod of the Step 4 result for 10

 

In the above example, that formula returns: 2

indicating that the  final digit should be 2....not 7

 

1) = 2*2 + 2*2 + 4*2 + 8*2 + 3*2 = 38

2) = 0 + 0 + 1 +0 = 1

3) =38 + 1 = 39

4) =3 + 9 = 12

5) =Mod(12,10) = 2

 

I hope that helps
