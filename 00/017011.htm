rich michaels - mvp (greatcirclelearning.com) say:

There are a few settings that can cause that to happen but let's just start with a few basic ones and see if they are set correctly.

  1. Establish your selection point in the first row of one of these tables that keeps starting on a new page.
  2. Go to the Home tab and turn on your non-printing characters (‚Åã). This will allow to see if there are any manual page or section breaks preceding the table.
  3. Go to the Layout tab, which is a contextual tab that only appears when the selection point is within a table. On the left side of the Layout tab ribbon is a Properties button, click it.
  4. From the Table Properties dialog make sure the Text Wrapping feature is set to None. This will ensure that the table is "inline" with other paragraphs in your document.
  5. Click the Row button at the top of the dialog. Make sure the Option to Allow Row to Break Across Pages is marked.

Hopefully these suggestion help... but of course post back if it doesn't change the table behavior in your document and we'll have to dig deeper.
