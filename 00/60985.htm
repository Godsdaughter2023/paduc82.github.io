george222(2) say:

You can create a text list of file names by using command prompt.


  • open a command prompt in the folder you wish to get a list of. ( You can open a command prompt and navigate to your required folder, or use explorer and press <space> and right-click your folder at the same time. Then you should see "open a command window here" in the menu - click it.)
  • Enter the command "dir /b > filenames.txt" (no quotes), so you just type dir /b > filenames.txt and press the carriage return/Enter key.
  • Use explorer to navigate to your folder and you should now see a new text file called filenames.txt. Open it and it should contain a list of the files. Then copy and paste the list where you want.

a. user say:

Hi Stephen,

 

1. Are you able to boot to the computer?

2. Did you make any changes to the computer prior to the issue?

 

This issue occurs when the Microsoft Visual C++ Redistributable is missing on the current computer.

 

I would suggest you to perform a hard shut down of the computer and follow the steps given below if the issue persists.

Method 1:

Run System File Checker (SFC) scan to fix any corrupt system files. To do this, follow the steps mentioned in the link below:
How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7

 

Method 2:

Download and install the Microsoft Visual C++ 2010 Redistributable Package, according to the operating systems setup on the computer.

Microsoft Visual C++ 2010 Redistributable Package (x86)
http://www.microsoft.com/en-us/download/details.aspx?id=5555

 

Microsoft Visual C++ 2010 Redistributable Package (x64)
http://www.microsoft.com/en-us/download/details.aspx?id=14632
