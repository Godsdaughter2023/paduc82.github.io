Note: It does sound like you select to quit from the Dock, but to be clear... to quit an app you need to select Quit. Clicking on the red close button does not quit most applications. Powering down before quitting can cause you to loose data.

Outlook stores your data in this location:

 ~/Documents/Microsoft User Data/Office 2011 Identities/your identity

If you move the Microsoft User Data folder or the Identity folder, Outlook will create a new blank one.

1) Check to see if your Identity is still located in the correct location. Check size by doing a Get Info on the Main Identity folder. A blank Identity is 22.8 MB

2) If you only find the blank Identity, do a search for "Microsoft User Data" or "Office 2011 Identities" or "Main Identity" (without the quotes). If you find it move it back. It will overwrite the current Main Identity unless you rename one. Be sure to check the Trash.

3) If the Identity has disappeared, a possible culprit is your anti-virus application or a crash. Some versions identify a virus within the database and puts your database into quarantine. You will need to read the directions on how to release.

4) Do you have a Time Machine or clone backup? If no, then start some kind of backup strategy!

5) Sometimes things go wrong and data disappears. Sometimes the info can be found in an invisible folder
"lost & found". Often, it's just lost forever.

Under the 'go' menu, select 'go to folder'. In that dialog, enter
"/lost+found" (without the quotes).

If there is a lost+found folder it will open, if not, you will get a 'folder
not found' error.

john korchok say:

The most common cause of formatting changing between computers is that you're using fonts on one computer that aren't installed on the other.  When a font is missing, Word still displays the expected font in the font dropdown, leading many users to beloieve it is installed. When a font is missing, Word silently substitutes another and the appearance changes. The solution is to use fonts that are installed on both computers.
