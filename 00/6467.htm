cimjet say:

Hi

Try this macro. It will remove the Caption (Text) beside each Option Box and Group Box

Sub ShapeCaption()
Dim OB As Shape
  For Each OB In ActiveSheet.Shapes
   If OB.FormControlType = xlOptionButton Then
      OB.OLEFormat.Object.Caption = ""
  End If
  If OB.FormControlType = xlGroupBox Then
      OB.OLEFormat.Object.Caption = ""
  End If
  Next
End Sub

While at the same time recording who swiped the badge on a table that i can later pull a report for. The form would look something like what i am attaching. I have been trying but without success to populate the employee information fields.

john w. vinson say:

It sounds like you're starting your database design with the Form. That's backwards! You will need at least three Tables for this:

tblEmployees

tblTraining

tblAttendance

Your Form would have Combo Boxes (probably not textboxes) to select the employee and the course, and would probably be set up as a Form based on tblTraining with a continuous Subform based on tblAttendance.

This is just one sketch of a possible design; you'll certainly want to adapt it to your own needs.

But you MUST start with a logical, normalized set of tables, and build the Form to fit them - not vice versa!
