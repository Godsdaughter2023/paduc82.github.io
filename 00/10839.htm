john_sur say:

Hi,

Try the following instructions.  Mind you that I am using Excel 2010 so what you will select may be different from what I am depending on what version of Excel you are using.

1.  Select a cell anywhere in the pivot table.

2.  Select the Options tab from Pivot Table Tools.

3.  Select Fields, Items, & Sets.

4.  Select Calculated Field...

5.  In the Name: text box give your calculated field a name.

6.  In the Formula:  text box type Total (which is the field name you listed in your post) to the right of the equal sign.

7.  Press the Add button and then the OK button.

8.  Right click anywhere in the calculated field.

9.  Select the Show Values As option from the pull down menu.

10.  Select the Running Total In option from the pull down menu.

11.  In the Base Field dialog box select Completion Date or whatever field name you used for Completion Date..

12.  Press the OK button.

Hope this helps!

John

john_sur say:

Hello Again,

I listed a way in my last post instructing you on how to accomplish with a calculated field what you are requiring to do (running totals in a Pivot table).  Just found out that this is not the correct way.  You don't need a calculated field.  Follow the instructions below and it should give you what you are looking for.

1.  Select your Pivot table.
2.  Drag the Completion Date field to the Values box.
3.  Click on the Count of Completion Date field in the Values box.
4.  Select the Value Field Settings... option in the pull down menu.
5.  Rename the field in the Custom Name:  text box if you wish.
6.  Click on the Show Values As tab.
6.  Click on the Show Values As text box and select Running Total In.
7.  Click the OK button.

Regards,

John

I need to use Word VBA to optionally hide or unhide the Developer Tab.  I tried recording the macro but that didn't work.
greg maxey say:

You can't do is with just a macro.  You need a ribbonX script and some VBA:

The ribbonX:

<customUI xmlns="http://schemas.microsoft.com/office/2009/07/customui"

   onLoad="OnLoad" >

   <ribbon startFromScratch="false">

      <tabs>

         <tab idMso="TabDeveloper" getVisible="showHideTab" />

      </tabs>

   </ribbon>

</customUI>

The VBA:

Option Explicit
Public myRibbon As IRibbonUI
Public bHideDevT As Boolean
Sub Onload(ribbon As IRibbonUI)
'Create a ribbon object
Set myRibbon = ribbon
End Sub
'Callback for getVisible
Sub showHideTab(control As IRibbonControl, ByRef returnedVal)
Select Case control.ID
  Case Is = "TabDeveloper"
    returnedVal = Not bHideDevT
End Select
End Sub
Sub ShowOrHideTheDeveloperTab()
If MsgBox("Do you want to show the Developer tab?", vbYesNo, "TAB SHOW OR HIDE") = vbYes Then
  bHideDevT = False
Else
 bHideDevT = True
End If
myRibbon.Invalidate
End Sub

Appending a series of text files (csv) to one file. - Microsoft Community

rancher dick say:

Appending a series of text files (csv) to one file.

Is the there an easy way to set up a macro that determines the number of files in a specific directory and the appends them one by one to a file?  Thanks.

 

jeeped say:

You didn't specify importing to a single worksheet or a series of worksheets so I don't know if this is going to help you, but you may want to try an old command line trick to consolidate the .CSV's to a single file.

Click Start (or tap the Windows key) and type cmd then hit <enter>. Use standard CD commands to navigate to the folder containing your .CSV text files. Then type,

TYPE *.CSV > ALLCSVS.csv

... then hit enter (and yes, you have to include the word TYPE). The resulting ALLCSVS.csv file should be a collection of all .CSV files within the folder. This should work on anything from a couple to a couple thousand text files. Double-click it to open in Excel.

Note: this does depend upon each .CSV ending in a new line (e.g. the way they are supposed to).
