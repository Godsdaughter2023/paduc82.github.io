IF condition looking the background color of a cell - Microsoft Community

lsuareza say:

IF condition looking the background color of a cell

it is possible to use a IF condition looking the background color of acell, i.e. I want to sum anly cells that are in yellow background color
ossiemac say:

Add the following line Application.Volatile as per the code below and that will force the code to run each time a change is made to the value contents of a cell on the worksheet.

However, when a value is changed on the worksheet the code will update but just changing the color of the cell is not sufficient to force the code to run but if you press F9 then it should update.


Function SumByColor(rngToSum As Range, rngColor As Range)
    Dim cel As Range
    Application.Volatile
    For Each cel In rngToSum
        If cel.Interior.Color = rngColor.Interior.Color Then
            SumByColor = SumByColor + cel.Value
        End If
    Next cel
End Function
