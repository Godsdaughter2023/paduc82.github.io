PC Suite, Bluetooth and Windows 7 - Cannot authenicate phone? - Microsoft Community

chrisdhall---01 say:

PC Suite, Bluetooth and Windows 7 - Cannot authenicate phone?

I had real problems getting PC Suite to connect via bluetooth to my 6300 phone.  I'm using a CSR type dongle which installed via plug and play using the Microsoft drivers.

 

I installed PC suite and followed the instruction to "connect a phone".   The phone was discovered and pairing attempted using the default 123 passkey.  I entered the passkey into the phone.  PC suite sat there for about 5 minutes, then came back with "Cannot authenicate phone".  This was tiresomely repeatable!

 

After much messing about I finally got the PC suite to authenticate the phone and its all working finel.  Here is what I did in case this helps anyone with the same problem:

 

  1. Install PC Suite.
  2. From the Windows Start Menu In "Devices and Printers" the bluetooth device should be visible - Mine shows "CSR Bluetooth Radio"
  3. Check the Bluetooth settings of the device "Allow Bluetooth devices to connect to this computer" should checked
  4. Still in" Devices and Printers", click on "Add Device", if the Bluetooth is working correctly, this should scan bluetooth and discover the phone (make sure the phone is set to visible or discoverable).
    (An alternative method is to open Control Panel, under the "Hardware and Sound, there should be "Add Bluetooth device" option under "Devices and Printers". )
  5. Select the phone and follow the instructions and pair the PC with phone.  This will involve entering the passkey generated by Windows
    If successful the phone should now appear in the Devices list.  I now see "Nokia 6300" as a device.
  6. The next steps are the ones I was missing: right click on the phone device and select Properties and then the "Services" tab
    Windows should access the phone device for the services offered by the bluetooth phone.
  7. Check mark the following services:
    Dial-up Networking [DUN]
    Serial Port (SPP) 'COM1'
    Serial Port (SPP) 'Nokia PC Suite'
    and click ok
  8. Start up PC Suite and go through the "connect phone" procedure again and it should now connect successfully.
      

Hope this works for you

Chris
