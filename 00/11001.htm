Where is the Preservation Hold Library created? - Microsoft Community

kevin alan brown say:

Where is the Preservation Hold Library created?

Split from this thread.

Where does one find the Preservation Hold Library once the library is created?

joanne li msft say:

Hi Kevin,

After a Retention Policy is applied to a site, the Preservation Hold library will not be created immediately. It will be created only when it is triggered, that means when the first items need to be copied to the preservation library.

For example, if I apply a Retention policy to a site and let it keep content for 1 month. When I delete an item from a library/list and from the Recycle Bin, it will create the Preservation Hold library, which can be found from the site's Site Contents.

Note: It will take up to 1 day to apply the retention policy to the locations you chose.
