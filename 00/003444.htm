    Change Copy and Paste Default Settings Office for Mac - Microsoft Community

    babeta say:

    Change Copy and Paste Default Settings Office for Mac

    I have Microsoft Word for Mac 2011 version 14.5.0 I need to change my copy and paste command to automatically keep text only. How do I change it?
    john korchok say:

    You don't have to do anything to the copy command, just paste. You can change this behavior by using a macro. First, you'll have to make the Developer tab visible:

    1. In Word, choose Word>Preferences>Ribbon.
    2. Scroll down until you see Developer and check it. OK out.

    Now you can create the macro:

    1. Click on Developer and choose Macros.
    2. In the Macro name: field, type PasteUnformattedText, then click Create.
    3. The VBA Editor window open with the outline of a macro. Revise it to look like this:
      On Error GoTo ClipboardNotText
        Selection.PasteSpecial DataType:=wdPasteText
        Exit Sub
    ClipboardNotText:
      Selection.Paste
      Err.Clear
    End Sub

    1. In the VBA editor, choose File>Save Normal.dotm. Then choose Word>Close and Return to Microsoft Word.
    2. Back in Word, choose Tools>Customize Keyboard.
    3. In Categories, scroll down until you see Macros and select it.
    4. In the right-hand list, select PasteUnformattedText.
    5. Click inside the Press new keyboard shortcut field and press the keys you want to activate the macro. I don't advise using Command + V, because sometimes you need to do a conventional paste (like if you're pasting tables or fields). However, Control + V is available and easy to remember.
    6. Very important: click on the Assign button. This actually locks it in, I often forget. OK out.

    Test in Word, Ctrl + V should give you Paste Special>As unformatted text.

    suzanne s. barnhill say:

    Whenever you are running more than one version of Word, the newer version will try to wrest control from the older one at every opportunity. In this case, "every opportunity" usually means every time you install updates (though this seems to have gotten better lately). When this happens, you need to reassert Word 2003's dominance. I also use Word 2003 for production work (I also have highly customized toolbars and shortcuts), so I am very familiar with the situation.

    As you've discovered, Open With is not an option when you have multiple versions. Instead, you have to "reregister" the desired version (this is a process that rewrites Registry entries to restore the file associations). Here's how:

    1. Press Windows Key + R to open the Run box.
    2. In the box, type or paste this string:
      "C:\Program Files (x86)\Microsoft Office\OFFICE11\WINWORD.EXE" /r
      and press Enter.
    3. Word will not start, but you will get a message box:
    4. When the "configuration" is complete, you should be back in business.

    The path in the string above is the default one for Word 2003 in Windows 7. If you have installed Word in a different folder, you'll need to modify the path.
