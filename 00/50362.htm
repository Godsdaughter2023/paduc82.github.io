Mail folders on local drive - Microsoft Community

alfje say:

Mail folders on local drive

How do I move mail from my service provider to mail folders on my local drive i.e. I would like to clear the mail on the service providers server?

a. user say:

Hi,

By default, your Windows Live email is stored in the %UserProfile%\AppData\Local\Microsoft\Windows Live Mail location.

STEP 1: Start Windows Live Mail.

STEP 2: Click on the program’s menu located at the upper-left hand corner.

STEP 3: Click Options, then click Mail.

STEP 4: When the Options window appears, click the Advanced tab, then under Maintenance and Troubleshooting, click the Maintenance button.

Windows Live Mail advance options

STEP 5: Click the Store Folder button, which will open a small dialog box that shows the path of the current mail store folder.

STEP 6: Click the Change button to select a new location for the mail store.

Windows Live Mail set store location

STEP 7: When the Browse for Folder screen opens, browse to an existing location where you wish to move your mail to or create a new folder.

STEP 8: Once done, click the OK button.

NOTE: The Store Location dialog box will appear showing you the new path you have selected.

STEP 9: Click OK to have Windows Live Mail use this new folder path.

NOTE: You will now be prompted to restart Windows Live Mail in order for the changes to take effect.

Windows Live Mail change store location

STEP 10: Close Windows Live Mail to start copying all your existing mail and mail folders to the new location.

Windows Live Mail copying folders to new location

STEP 11: Once the copying process is complete, start Windows Live Mail again.

NOTE: Unfortunately, since Windows Live Mail only copied your mails, you will need to delete your old Windows Live Mail folder.

STEP 12: To free up the space on your C: drive, close Windows Live Mail and browse to the old Windows Live Mail folder which is located under the following folder:

Windows 8, Windows7, Windows Vista: C:\Users\<login name>AppData\Local\Microsoft\

Windows XP: C:\Documents and Settings\<login name>\Application Data\Microsoft\

NOTE: When you open that folder, you will see a subfolder named Windows Live Mail, which is your old mail storage.

STEP 13: Rename it to Windows Live Mail.old and start Windows Live Mail again.

NOTE: If you can still access your email, then you’ll know that the program is now using the new folder and you can now safely delete the old one.

STEP 14: Close Windows Live Mail again, then delete the Windows Live Mail.old folder.

STEP 15: Once the folder is completely deleted, proceed in emptying your Recycle Bin

Hi Sarcastic, you can reinstall Windows 10 at any time and it will not cost you anything !

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Click this link:
https://www.microsoft.com/en-gb/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
