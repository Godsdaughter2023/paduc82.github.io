                    cannot access the "X" or "-" in the upper right hand corner. 

There are numerous ways you can deal with that.  The traditional way is to make sure the window has keyboard focus and then press Alt-Space.  Then you have access to the window's window menu even though your mouse may still be unable to reach it.  But if you just want to close it you could press Alt-F4, same as ever.   Those methods would be options for you even if explorer.exe wasn't running.

If you want to resize a problem window you could use the Taskbar Cascade windows command.  Minimize all other windows that you don't want involved in the operation before performing that one.  

Those are all old methods, available for several versions of Windows.  More recently we can also access the same window menu functions via right-click on the window's Taskbar thumbnail.

This is usually heat related, defective hardware, memory or even processor though it is

error implication. You may need a real computer shop (not leeks and peeks at a BestBuy

Check the processor seating, heat sink, and the compound between the processor and
the heat sink which are common causes.

There have been a few instances where Norton/Symantec and other antivirus/security
programs have caused this error.
