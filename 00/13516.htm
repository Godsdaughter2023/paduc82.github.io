Word 2016 How to set up printing from various trays - Microsoft Community

mr j smith say:

Word 2016 How to set up printing from various trays

Hi

I have a printer with 4 trays. They are A4 white paper, colour letterhead, black and white letterhead and A3 white paper.

In word 2016 is it possible to assign a button to each tray somewhere so when clicked what is on screen will print to that tray without having to go into the print menu (control + p).

Next is it possible to set up a button tha would act like above but would print the first page on colour letterhead and all other pages on A4 white paper etc.

jay freedman say:

You can do this with macros, after a little experimentation.

For complex printers such as yours, Word asks the printer driver for a number to assign to each tray, and you have to find out what those numbers are. To do that:

  • Start recording a macro (click View > Macros > Record Macro) and assign a name for the macro, or accept the name suggested in the message box. 
  • On the Layout tab, click the little arrow in the bottom right corner of the Page Setup group to open the Page Setup dialog.
  • On the Paper tab of the dialog, select the first of the trays in the Paper Source lists and click OK.
  • Repeat opening the Page Setup dialog, selecting the next tray, and clicking OK until you've gone through all four trays.
  • Stop recording.

Now click View > Macros > View Macros. In the dialog, select the name you gave to the recording and click the Edit button. There will be four sections in the macro code, each starting with the line With ActiveDocument.PageSetup. In the middle of each section there are lines starting with .FirstPageTray and .OtherPagesTray. Make a note of the number after the equal sign in those lines and which tray each is associated with (A4 white, etc.).

Now you can copy/paste the following code into the macro editor four times:

Sub PrintCurrent1()
    With ActiveDocument.PageSetup
        .FirstPageTray = 0
        .OtherPagesTray = 0
    End With
    ActiveDocument.PrintOut Range:=wdPrintCurrentPage
End Sub

For the second, third, and fourth copies, change the number in the Sub line to 2, 3, and 4.

Next, change the 0s in the .FirstPageTray and .OtherPagesTray lines of the first macro to the numbers you found in the experiment to correspond with the first tray. In the second macro, change the tray numbers to those for the second tray, and similarly for the third and fourth macros.

Finally, use the File > Options > Customize Quick Access Toolbar dialog to assign toolbar buttons to run the macros. You can use the Modify button in that dialog to change the icon and the tooltip text for each button so you can identify them when hovering the mouse over them.

For your second question, you can make copies of the first and fourth macros (presumably you don't want one that prints 'other' pages on either kind of letterhead) and change their names in the Sub lines so they're different from the existing macros. Then make these changes:

  • Change the number in the .FirstPageTray line to the one for the color letterhead. Leave the .OtherPagesTray value as it is -- for A4 paper in one macro and for A3 paper in the other macro.
  • Change the word wdPrintCurrentPage in the next-to-last line to wdPrintAllPages.
  • Assign Quick Access Toolbar buttons to these two macros.
