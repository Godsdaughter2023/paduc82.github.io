Try this

List the failing updates and their failure code. Omitting the one with a different failure code manually install the remainder. Then try installing the last update.

 

To determine failure codes select the keyboard shortcut Win+X, Control Panel, Windows Update, View Update History, double click the failed update and you will see the Failure Code.

 

A lot of problems arise from setting Windows Update to automatically update. You may prefer to change this setting. Select the keyboard shortcut Win+X, Control Panel, Windows Update, Change Settings. Under Important Updates is a box showing the current setting. Click on the Arrow down to the right and change the selection to “Check for Updates but let me choose whether to download and install them” and click on OK.

 

Select the keyboard shortcut Win+X, Control Panel, Windows Update, Check for Updates. If optional updates are available click optional updates and make sure that the box before Windows 10 is unchecked. Check the boxes before the remaining updates you want to install, click on Install and click again on Install in the next window and the process of installing starts. You should let the process proceed without interruption. Multi-tasking at the time updating is in progress is unwise. The progress window will tell you what is going on. You need to be patient.
