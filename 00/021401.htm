colforbin say:

Restore Word style settings to default

At one point, I was working on a document that needed pretty specific formatting. So I modified all of the "heading" styles to fit what I needed there. But now those styles have persisted, and I don't want that formatting anymore. I'd like to go back to the default heading styles. Is there a way to restore these?
charles kenyon say:
  1. The simplest way would be to rename your normal.dotm template, with Word closed.
    How to Find the Normal Template
  2. Open Word and record a macro doing anything. Close Word and save the new normal template which will have been created to save that macro.
  3. Find the new Normal template. It will have the styles you want. Rename that as NormalStyles.dotm.
  4. Rename your old normal.dotm back to normal.dotm.
  5. Open Word.
  6. Open the NormalStyles.dotm file and activate the styles Organizer.
  7. Copy any styles you want from that template to the Normal template.

If you do not have any other customizations in your normal template that you want to save, you can skip steps 2 through 7.
