doug robbins - mvp office apps & services (word) say:

Manipulation of data is almost always done more easily in the data source.

We would really need to know more about the content of the field.

If the data was in Column A and it was really in separate paragraphs, running a macro containing the following code would insert the content of each paragraph in each row in separate cells immediately to the right of the original data

Dim astring As String
Dim arrstring As Variant
Dim i As Long, j As Long, k As Long
With Sheets(1).Range("A1")
    k = .CurrentRegion.Columns.Count
    For i = 1 To .CurrentRegion.Rows.Count - 1
        astring = Sheets(1).Range("A1").Offset(i, 0).Value
        arrstring = Split(astring, Chr(10))
        For j = LBound(arrstring) To UBound(arrstring)
            .Offset(i, k + j).Value = arrstring(j)
       Next j
    Next i
End With

hansv mvp say:

You shouldn't use GetObject here - it tries to open the file. Try this version:

Sub InsertFileProperties()
  Dim fd As FileDialog
  Dim vrtSelectedItem As Variant
  Set fd = Application.FileDialog(msoFileDialogFilePicker)
  With fd
    .Filters.Clear
    .Filters.Add "Various", "*.xls; *.xlsx; *.docx; *.doc; *.pdf"
    If .Show = -1 Then
      For Each vrtSelectedItem In .SelectedItems
        ActiveSheet.Hyperlinks.Add Anchor:=Selection, Address:=vrtSelectedItem, _
          TextToDisplay:=Mid(vrtSelectedItem, InStrRev(vrtSelectedItem, "\") + 1)
        ActiveCell.Offset(1, 0).Select
      Next vrtSelectedItem
    Else
      End If
  End With
  Set fd = Nothing
End Sub

bernd p. say:

Because you talk about .one section files I assume you talk about local notebooks, stored on your computer.

Local notebooks are just folders in the Windows file system.

So the folder that contains your section files IS the notebook. And this folder's name is (surprise, surprise) the name of the notebook in Onenote.

But you don't have to mess around with that folder. These 3 notebooks are separated.

If you don't want to see it in Onenote just close it using the right click menu.

To re-open it the fastest way is using Windows explorer. 

By default your notebooks (their folders) are in

C:\Users\<user>\Documents\OneNote Notebooks

Jump into the folder with your notebook's name and double click any of the .one files in it.

If you just want to delete Hidden text, you can do that with Replace.

  1. Press Ctrl+H to open the Replace dialog.
  2. In the "Find what" box, type ^? and press Ctrl+Shift+H. You'll see "Format: Hidden" below the box.
  3. Leave the "Replace with" box empty.
  4. Click Replace All.

grantthornbury say:

Remove Bullets from bullet library in Word on Mac

In Word on Windows, you can simply right-click on any bullets in the library and click "remove". However, on Mac, there does not seem to be this option. Is there any way to get rid of some of these bullets?
john korchok say:
Possibly you're referring to the Customize Bulleted List dialog, where you see a row of 6 bullets. This operates like the Word 2003 version of bullets, where the 6 options are a rolling selection of your previous bullet customizations. As you create a new custom bullet, that new format is placed at the left end of the 6 and the rightmost one is removed. So there wouldn't be much point to removing bullets here, as they reflect your previous design choices rather than something handed down from Microsoft. Personally, I far prefer this setup than the somewhat loony bullet creation scheme in Word 2007 and up.
