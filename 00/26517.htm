No Video from Web Site Option in PowerPoint 2010

Technical Level : Intermediate

Summary
You may have found that the "Video from Web Site" option is no longer available within PowerPoint 2010 for Windows. What happened to this option, why did it go away, and what can you do to get similar functionality?
Details

When PowerPoint 2010 shipped, there used to be a perfectly usable option that let you add a video from sites such as YouTube within your PowerPoint slide. This Insert Online Videos option was useful, as shown below.

So why did this option disappear from PowerPoint 2010? You may now only see two options rather than the original three, as shown below.

This third option was removed when you may have installed automatic updates, especially an update named KB2553145. You can reinstate this missing Video from Web Site option back once you remove this update. Microsoft explains how to do so on their Description of the PowerPoint 2010 update: September 10, 2013 page.

However, even after you reinstate the option, you may not be able to insert online YouTube videos because of the changes Google made to their algorithms. Indezine's Missing “Video from Web Site” Option in PowerPoint 2010 page has more information on getting this functionality back.

Hi MeredithHammond,

 

The answer is yes, you can allow an external user to book a room calendar. To set it up, connect to Exchange online PowerShell and run following cmdlet to enable the feature:

 

Get-Mailbox <room mailbox displayname> | Set-CalendarProcessing -ProcessExternalMeetingMessages $true

 

After this, run following cmdlet to double check the result.

 

Get-Mailbox <room mailbox displayname>| Get-CalendarProcessing | select *external*

cortig say:

Outlook 2011 and 15.x are completely separate and do not share any of the data. If you launch 2011 now, it will open in the state it was when you last launched it.
It won't be able to import anything from Outlook 15 though. The only thing that can sync across is whatever you have on server based solutions (emails status and folders for IMAP and Exchange, plus contacts and calendars for Exchange).

nikkia---01 say: This works..i was having the same problem but its really simple


From main screen, push "Menu" button, then select "Settings", then select "Tones", then highlight "Incoming Call Alert" and select "Change", then scroll down and select "Off". Now scroll down and highlight "Vibrating Alert" and if not already set to on, select "Change", then select "On". Then to save your changes either scroll down to the end of this menu and select "Save" button or from where you are select "Close" and then "Yes/OK".
