lizette ags say:

We appreciate you for getting back to us with the said information. We understand how frustrating this situation may be for you, that's why we are doing our best to investigate your concern so can provide you with the most appropriate resolution.

Yes, it is possible that the missing 5 GB on your computer was taken up by the Hiberfil.sys file. You can find the Hiberfil.sys hidden system file in the root folder of the drive where your operating system is installed. The Windows Kernel Power Manager reserves the said file when you install Windows. The size of this file is approximately equal to how much random access memory (RAM) that your PC has. In that case, disabling hibernate mode on your PC will allow you to delete this file and reclaim valuable disk space.

Note: When the hybrid sleep setting is turned on, your computer uses the Hiberfil.sys file to store a copy of the system memory on the hard disk . Without this file, your computer cannot hibernate.

If you want to make hibernation unavailable, you can follow the steps below:

  1. Click Start, and then type cmd in the Start Search box.
  2. In the search results list, right-click Command Prompt, and then click Run as Administrator.
  3. When you are prompted by User Account Control, click Continue.
  4. At the command prompt, type powercfg.exe /hibernate off, and then press Enter.
  5. Type exit, and then press Enter to close the Command Prompt window.

To make hibernation available, please refer to the following steps:

  1. Click Start, and then type cmd in the Start Search box.
  2. In the search results list, right-click Command Prompt, and then click Run as Administrator.
  3. When you are prompted by User Account Control, click Continue.
  4. At the command prompt, type powercfg.exe /hibernate on, and then press Enter.
  5. Type exit, and then press Enter to close the Command Prompt window.

foxfifth say: It can backup everything.  Typically (but there are exceptions) it will create both a system image and a separate backup of your data files (so in effect your data files get backed up in two ways).  By default the system image backup only includes the drives required for Windows to run (including all data on those drives) but that can be changed as discussed in:  http://windows.microsoft.com/en-US/windows7/Back-up-your-programs-system-settings-and-files
If it was me, I would be doing a backup that includes both your c and d partitions and the backup destination would be an external drive.  One of the most likely reasons for needing to restore from the backup is a hard drive failure and if that happens both partitions are lost and any backup stored on those partitions would be gone.  I would also be doing a second backup that will be available in the event of a fire or theft of your computer equipment.

There is more help at:
http://windows.microsoft.com/en-US/windows7/products/features/backup-and-restore

http://windows.microsoft.com/en-US/windows7/Back-up-your-files

http://windows.microsoft.com/en-US/windows7/Set-up-or-change-automatic-backup-settings
