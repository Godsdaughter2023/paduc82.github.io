jaymo1978 say:

Really sorry you're having this problem, man. As a general rule, this doesn't sound like it falls into the category of problems that can be fixed with hard resets or troubleshooting steps, and it sounds like you've already done all you can. Even with the best hardware, issues do happen. I heard at one point in the past, I think if I remember correctly, companies expect something like a 2% failure rate with new systems. No matter what, because of manufacturing variances between manufacturing centers and microscopic flaws in the individual hardware components, some devices will fail.

One thing I would suggest checking - this sounds dumb, but are you plugged directly into a wall, or does your Xbox run through a surge protector/power strip. If you're into a power strip, you absolutely want to plug it directly into a wall, because the Xbox power supply already has a built-in surge protector. By plugging it into another surge protector, which has the effect of throttling the flow of power, the device essentially has to work twice as hard to pull the necessary amperage during its startup routine, which results in various components failing not long into the life of the device. (This was one of the biggest causes of the RRoD on the 360.) You might try plugging into a wall and seeing if that makes a difference in its ability to read the BR/Game discs.

Otherwise, sadly, your best bet is going to be going to https://myservice.xbox.com/ and registering your console, and then starting the repair process. :-(

This error occurs when your browser is unable to negotiate with removable media. In your case, the hard drive.

Uninstall and reinstall drivers for your hard drive and check if it helps.

Make sure you have the latest drivers installed for your hard drive for Windows 7.You can download the drivers from the manufacturer’s website.

Always use “safely remove hardware” option while unplugging the hard drive.

Disable any security programs temporarily and check if you still get that error.

Follow these steps to enable DMA mode using the Device Manager

1. Go to "My Computer, ""System Tools," "View System Information," then System Properties, "Hardware," Then Open Device Manager

2. Double-click IDE ATA//ATAPI Controllers to display the list of controllers and channels.

3. Right-click the icon for the channel to which your burner is connected and select Properties. Now click the Advanced Settings tab.

4. In the Current Transfer Mode drop-down box, select DMA if available if the current setting says, "PIO Only."

If the drop-down box already says, "DMA if Available" but the current transfer mode is PIO, then the user must "toggle" the settings. That is, change the selection from "DMA if available" to PIO only, and click "OK".

Then repeat the steps above to change the selection to "DMA if available".
