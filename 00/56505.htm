geethu b say:

Hi,

Thank you for contacting Microsoft Community.

To understand the issue better, I would need more information from your end.

  1. What happens when you run chkdsk and defrag?
  2. What is the knowledge Base article numbers of the Windows Updates that were installed recently?
     

To find Knowledge Base Article number of the Windows Updates, view your Windows Update history:

1.  Right-click on Start and click on Settings.

2.  Open Update and Recovery.  

3.  Click on Windows Update, and then click on Advanced Options.

4.  Click View your update history.     

This issue could have occurred due to conflict between recently installed Windows Update or damaged hard disk.

Method 1:

I see that you have tried performing check disk, however if you have not tried running check disk from WinRE (Windows Recovery Environment) try the same and check if it helps.

a) Insert the Windows 10 installation media and restart the computer.
b) Press F12 key (typically this is F12, but it can differ between computer manufacturers) and choose the drive that you inserted the installation media into.
c) Once the Windows Setup window appears, follow these steps:
d) Click Next and select Repair your computer.
e) You will then see a blue screen and an option to choose.
e) Choose the option Troubleshoot and select Advanced options.
f) Select Command Prompt.
g) In the command prompt, type ‘chkdsk /r’ and press Enter.

Warning: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.


Method 2:

If the issue persists, I suggest you to uninstall the Windows Update that caused the issue and check if the issue occurs or not.

To uninstall the update follow these steps:

a) Right-click on Start and click on Control Panel.

b) Click on Programs and click on Uninstall a Program.

c) Click on View installed updates.

d) Scroll through the list and find the update.

e) Right-click the update and uninstall it.


Hope this information helps. Reply to the post with an updated status of the issue so that we can assist you further.
