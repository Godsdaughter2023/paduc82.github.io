canadian tech say:
In reply to deleted message

You can use Windows Backup.  However, I do not use or recommend that to my clients.  Instead use the drag and drop method you mention to your external hard drive.  I suggest each time you do a backup, you create a folder named "backup user date".  If your external drive is big enough, save 3 generations.  Do NOT leave an external drive running for a long time.  Most all of them do a poor job of dissipating heat, and will cook the drive.  A few hours at most, on time, then put it away in a safe place away from your computer.

Remember, backup only your data.  It is pointless to backup programs or program folders because in the case of a re-installation, you must re-install the programs anyway.

In most cases you will find most all, if not all of your data in your User folder under each users name. C:/Users/....  You can simply drag and drop the whole folder.

 

 

A tip. I do this for all my clients' computers.

 

The default restore point space allotment in Windows 7 is 1 to 3% of drive capacity.  Typically, this does not allow for more than a few restore points.  In almost all cases, the computer is using a very small proportion of the space available. That makes it very practical to raise that 3% to say 8 to 10%.  This strategy will yield far more restore points that will give you much more flexibility in recovering a bad situation.

To change the allotment, type create in the text box above the start globe.  Choose Create a restore point.  Click configure.  Move the slide to the right as much as you see fit.  Setting it at 50 Gigabytes will have no affect if you have a 500 Gigabyte drive and are only using about 100 or so, which is highly typical.

A second tip

After Windows 7, system drivers and all updates are installed and any stable applications like Microsoft Office are installed and updated, and before any (not -Rs) and about an hour.  When you are done you will have a very nice bit of insurance.  Should you ever again need to re-build a corrupted system or replace a hard drive, you will have a precise duplicate of your system as it was at this point.  You can restore that image to a hard drive in about 20 minutes.  Creation of System Image is found in your menu under Maintenance, Backup and Restore.

Another great feature about using the image is that you do not need an install disk or a product key to do the re-install the next time, and you will have saved yourself all the time you put in this time.

I emphasize the need for PLUS R DVD blanks.  Do not use the more common MINUS R DVD blanks.

devlin sco say:

Hello Aprack,

Exporting your current Firewall Rules can be done through Windows Firewall with Advanced Security with administrator rights. Follow the steps below on how to export and import your Firewall Rules.

  1. Select the Start button.
  2. Type Windows Firewall with Advanced Security and then press Enter.
  3. If prompted, click Yes or enter the administrator password.
  4. Click Action and then click Export Policy.
  5. Select the location where you want to save the Policy file (.wfw).
  6. Name the file and then click Save to finish.

For importing the Firewall Rules, follow these steps:

  1. Select the Start button.
  2. Type Windows Firewall with Advanced Security and then press Enter.
  3. If prompted, click Yes or enter the administrator password.
  4. Click Action and then click Import Policy.
  5. Select Yes to continue.
  6. Locate the Policy file (.wfw).
  7. Click Open to complete.

Should you need further help with other concerns in the future, you're always welcome to post back.
