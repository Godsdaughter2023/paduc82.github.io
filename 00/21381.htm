Can this be done in either 2007 or 2010?

 

Thanks,

Shane

deanh say:

The following macro will resize all "InLineWithText" graphics to a WIDTH of your choice in centimeters and maintain ratio with height.

I got this macro from an MVP here years ago and it works wonders in 2003, I have not tested it in 2007/2010.

Beware, if you have any embedded files in the document, these are considered graphics as well.

 

Sub PictSizeWidth()
Dim WidthSize As Integer
Dim oIshp As InlineShape
Dim oShp As shape
WidthSize = InputBox("Enter width (cm)", "Resize Picture", 11)
For Each oIshp In ActiveDocument.InlineShapes
With oIshp
        If oIshp.Count > 0 Then
            For Each inshpPower In .InlineShapes
                With inshpPower
                    sngOldWidth = .Width
                    .Width = CentimetersToPoints(sngNewWidth)
                    .Height = CentimetersToPoints(((.Height * sngNewWidth) / sngOldWidth))
                End With
            Next
        Else
            MsgBox "There are no shapes in this document.", _
               vbExclamation, "Cancelled"
     End With
Next oIshp
End Sub
