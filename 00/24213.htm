If you notice, the first time you expanded the Other projects and commitments tab in the Resource Usage view, you get a loading cursor for a brief moment. This is because Project Professional needed to get the information via Project Online.

After Project Professional has cached the information locally, even if you collapse the tab again, the Resource Usage view below will still show the expanded details. You will only need to trigger Project Professional to load the information again when you close and re-open the project / Project Professional.

If you would like for Project Professional to load this information automatically each time you open a project, or you would like for Project Professional to load this information while viewing it in the split view,

given the situation, we suggest you submit your idea via our Microsoft Project's UserVoice; this is the best way to influence our products and services to make it better for you and others.
