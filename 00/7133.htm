Equation editor font size

I do not want to go with the linear option. How to avoid changing font size automatically? 
jay freedman say:

You cannot avoid the automatic size change if the equation is part of the same paragraph as the surrounding text.

There is a workaround, but it's extremely labor-intensive. You need to insert a text box and put the equation inside it. Then you can format the text box to turn off its borders, size it to fit the equation, change its text wrapping to "in line with text", and position it in the regular text. Because the text box will be taller than the surrounding text, the bottom of the box will align with the regular text's baseline; then you have to select the regular text that's on the same line as the equation, open the Font dialog, go to the Advanced tab, and set the position to Raised by enough to center it on the equation's height.

It would be easier simply to accept the Display format of the equation, putting it in its own paragraph.
