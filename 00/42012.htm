¡firedog say:

This is not a common problem. It will often take some time for an image file to be retrieved and attached to a message, during which time the program may be shown as Not responding in the title bar. This simply means that it's too busy to answer Windows' repeated Are you still there? question. The program will usually respond eventually.

If the program really freezes, there will usually be an Application hang error report in Event Viewer* which will give some clues to what's wrong.

Which version of Internet Explorer do you have installed? Is it fully up to date?

*  Press Windows key  + R, type eventvwr and press Enter to launch Event Viewer.
