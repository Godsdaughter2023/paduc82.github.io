INSERTING CENSOR BLEEP - Microsoft Community

sueswayn say:

INSERTING CENSOR BLEEP

I'm trying (for the first time) to edit some video footage using Movie Maker, I'd like to know how to insert audio "a censor bleep" so that it cuts out the original audio all together for the duration of the censor bleep.  Any help is appreciated. 
considerate_guy say:

This should be pretty straight-forward to accomplish.  The important thing is to isolate the bit of the clip that has the offensive word that you want to "bleep".  You need to make a split right before and another right after that word.  You may want to use the J and L keys to get to the right spot, then M to make the split.  If you are using Movie Maker 2012, the waveform display can be really helpful to "see" where the word starts and stops.  Pause just before the word, make a split.  Pause again just after the word and make another split.

 

Once you have isolated the part of the video, you want to set the Video Volume to zero on that clip.  (If you don't do this, the word will still be audible.)  You select the clip, go to the Video Tools | Edit tab and on the left should be Video Volume which you should set all the way to the left (mute).

 

The last part is to add a bleep sound effect in that place.  You'll need to find a sound file.   I'm sure if you search the web you can find sound clips for a 1 kHz "bleep" tone.  It can be longer than needed and you can trim it.  Anyway, you would add that as "music" (or narration) at the point where you muted the original audio.  Trim the endpoint to the right length.

 

If you did it right, you should hear something like:

"What I just did was (bleep) amazing!"

devin lanei say:

I have successfully installed Photo Story 3 on Windows 8 Enterprise x64, and the Windows 8.1 Enterprise x64 Preview.  I have also discovered exactly what breaks it, and I would guess that this is what broke Photo Story 3 for people on Windows 7: LegitCheckControl.dll

 

If you download Photo Story 3 from the Microsoft website via the alternate method the application will install and work just fine.  If however you choose to install the ActiveX Windows Genuine Advantage version, it installs the LegitCheckControl.dll, a few registry settings, and will result in the following errors.

 

 

Installer error:

"Photo Story 3 for Windows cannot be installed on this version of Windows. Check system requirements for Photo Story 3 for Windows before trying to install the program."

 

 

Application launch error:

"Photo Story 3 for Windows cannot run on this version of Windows.  Check the system requirements for Photo Story 3 for Windows before trying to run it."

 

 

While the problem is going on, you can install Photo Story 3 on systems that are affected by the WGA ActiveX components through the command line with the following example:

 

msiexec.exe /i Pstory.msi /passive SKIPOSCHECK=1 NOLEGITCHECK=1

 

To fix both issues:

As the ActiveX installer does not include an uninstaller, the best solution is to rename the WGA DLL via PowerShell or Command Line as Administrator so that Photo Story can no longer find it, and so you can enable it again in the future without issue.

 

Windows x86:

PS: "Rename-Item $env:windir\System32\LegitCheckControl.DLL LegitCheckControl.Disabled.DLL"

CMD: "rename %windir%\System32\LegitCheckControl.DLL LegitCheckControl.Disabled.DLL"

 

Windows x64:

PS: "Rename-Item $env:windir\SysWOW64\LegitCheckControl.DLL LegitCheckControl.Disabled.DLL"

CMD: "rename %windir%\SysWOW64\LegitCheckControl.DLL LegitCheckControl.Disabled.DLL"
