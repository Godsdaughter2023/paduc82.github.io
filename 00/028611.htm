If you were using an Exchange ActiveSync account in WLM, you won't lose anything (unless you moved the messages to local folders) because that type of account keeps messages on the server.  This is also true for IMAP (but you probably weren't using IMAP, since it's a very recent addition to Outlook.com).  If you were using POP, however, you will lose all messages storeed locally.  If you had deleted them from the server when you downloaded them, you'll be deleting your only copy when you remove the account.
