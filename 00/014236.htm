suzanne s. barnhill say:

You can intersperse custom footnotes with numbered ones. Use the dialog launcher (small arrow in the bottom right corner) in the Footnotes group on the References tab to open the Footnote and Endnote dialog. Select 1, 2, 3 as the numbering style generally ("Whole document" is selected for "Apply to"), with "Continuous" selected for the numbering.

For the custom ones, don't choose *, †, ‡, but instead type the asterisk in the "Custom mark" box, making sure that "Selected text" is selected rather than "Whole document." This will not affect the numbering of "regular" footnotes, which can still be inserted with Ctrl+Alt+F, but  you will have to use the Footnote and Endnote dialog to insert each custom note.
