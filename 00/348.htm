cms3161 say:

I had already followed those instructions exactly as written. After working with Mircosoft's live support, here is the solution we arrived at. Note that this fix will involve a registry change, and should only be performed by those comfortable with doing so, an certain steps may be different or you may need a different version of OneDrive to install at the end.

"A newer version of OneDrive is installed" Fix

1. Make a backup of all OneDrive items, as they may be removed during this process

2 Uninstall all OneDrive applications from computer as listed in Control Panel > Programs and Features

3. While in Programs and Features, select "Change" on the Office suite installation, then select Add or Remove Features, and remove OneDrive from the installation.

4. In Registry Editor, navigate to HKCU\Software\Microsoft\OneDrive, and add a new DWORD called “TeamSiteSyncPreview” with the value of’1 or change the value of the key if it is already there to ’1'

5. Restart the computer

6. Run the Microsoft OneDrive for Business Cache Removal EasyFix tool, from https://support.microsoft.com/en-us/help/3038627/how-to-remove-the-onedrive-for-business-cache-by-using-the-easy-fix-to and restart the computer if successful

7. In a Windows Run dialog, enter "skydrive.exe /reset" and if the command completes successfully, restart the computer.

8. Download OneDrive using the following link and perform the installation and setup as normal: https://go.microsoft.com/fwlink/?LinkID=844652
