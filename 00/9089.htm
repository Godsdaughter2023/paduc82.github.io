It was the Len function.

.lStructSize = LenB(OFN)   ' change it from Len(OFN)

Len Function
Returns the number of characters in a string or the number of bytes required to store a variable.

The LenB function is used with byte data contained in a string. Instead of returning the number of characters in a string, LenB returnsthe number of bytes used to represent that string.
