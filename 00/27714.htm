roger mcdonald say:

How to open files in created in older versions of Microsoft Word

I currently use Microsoft Word for Mac 2011. I need to open a whole lot of files created between 1992 and 2006 but when I try to I get this message: 'You can't open the application"Adobe IllustratorCS2" because Power PC applications are no longer supported'. 

If I drop the file onto TextEdit, it opens some of the text but converts whole sections of important text to gobbledygook.

How can I recover these old MS Word documents in exactly the same format in which I created them?

I'd greatly appreciate some advice.  

john mcghie say:

Chances are the files contain illustrations in Adobe Illustrator format, and Word is calling out to Illustrator to display the pictures.

But it may be that you have a bad entry in your file type database.

First, start Word and use the File Menu: choose Open from the File menu to navigate to and open the file.

If you get no error doing that, then you need to correct your Launch database (see next...).  If the error appears again, the file contains an Adobe Illustrator picture (the gobbledygook that Text Edit is showing you is probably the picture binary). In which case, you would need to install a new version of the Adobe suite and hope that that can handle the pictures.

If you need to correct the database:

  1. Go to the file(s) in question in Finder
  2. Ensure they all have the correct file extension on the file name (probably should be .doc for those files).
  3. Select one and right-click it.
  4. Choose Get Info
  5. In the Open With box, change the application that appears there to the current version of Word.
  6. Click the Change All button.

The file extension is what Apple uses to determine which application to send the file to when you double-click a file in the Finder.  If that's wrong, or missing, you may get strange error messages.

Hope this helps

jim gordon mvp say:

What you are describing is most likely an issue with the chose font. Your Mac probably doesn't have the same version of the font that was used when the message was made, so it is substituting a different font.

Use Font Book(it’s in Applications > Utilities folder) or other font utility to validate ALL your fonts. Delete old versions of duplicate fonts. Delete fonts that don't validate completely. Use Font Book's ADD feature to add your missing font. Restart your Mac and empty the trash to get rid of the unneeded fonts and free up disc space.

 

Use Disk Utility (it’s in Applications > Utilities folder) to verify the startup volume and repair permissions.

jim gordon mvp say:

When fonts are not displaying correctly the first thing to check is whether or not the other machine has the font that was requested to display. If the requested font is not there, font substitution occurs. 

Typically new versions of fonts have more glyphs than older versions of the same font, so it's important that everyone is on the same version of the font.
