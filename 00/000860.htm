ganesh kumar n say:

1.     Unfortunately it is not possible to select more than one item from the drop down from the same cell. You can have only one item per cell from the list.

2.     Regarding the second question it is not possible because you cannot have multiple items on a single cell.

3.     Also it is not possible to sort the items in the alphabetic order.

As a work around, you can create multiple columns to select multiple items, but it is not possible to do it on the same cell.

Also you can create multiple items as one item (like for example item1, item2, item3….) and you can choose multiple items from the drop down.
