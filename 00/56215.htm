matthew_omega say:

Hello Ichigopickle,

 

If we can get into Windows Recovery Environment, we can run chkdsk:

 

1.     Tap F8 during startup to get to the Startup Options menu.

2.     Choose Repair Windows.

3.     After choosing the location and logging on, click on the Command Prompt option.

4.     Type the following command followed by the ENTER key:

CHKDSK C: /F

5.     Let CHKDSK run to completion. It may take some time so be patient and do not restart the computer while it is running.

6.     When CHKDSK has completed, restart and see if we can get into normal and/or safe mode.

If that does not resolve the issue, we can try to swap out the registry hives within recovery console as per the following article:

http://support.microsoft.com/kb/307545  <Please note that this article pertains to Windows XP and we would use the same steps from within WinRE which is accessed as per the above instructions.>

Let us know if you have any questions.
