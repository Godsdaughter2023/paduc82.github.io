2. Instead of printing to One Note, chose to print to XPS file and safe this file (to your desktop)

3. Open One Note 2010 and create an empty new note.

5. Chose the XPS file you've created under step 2.

6. Now a print of this XPS file is beeing made in One Note.

Subtitles are available depending on your video files. To enable/show subtitle in Windows Media Player, follow these steps:

  1. Open Windows Media Player and click “Library.”
  2. Beside “Library” there is a right-arrow key, click it then select “Videos.”
  3. Click the right-arrow key beside “Videos,” then select “All Videos.”
  4. Right-click on “All Videos”, then hover your mouse on “Play.”
  5. A drop-down list of options will show, look for “Lyrics, captions and subtitles,” then select “On if its available.”

 

You can also press Ctrl + Shift + C, while the video is playing to automatically turn on the subtitles if it's available.
