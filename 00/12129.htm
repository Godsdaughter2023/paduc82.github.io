Of course it works. <grin>

You made a sheet visible when clicking a hyperlink.

Now you want to make a sheet hidden when you click a hyperlink.

You don't have to determine the sheet (you are on it)
Each sheet has its own module (a class module). It (the sheet) is referenced in code with the "Me" variable.

'---
Private Sub Worksheet_FollowHyperlink(ByVal Target As Hyperlink)
Me.Visible = xlSheetHidden
End Sub

a. user say:

David,

re:  make sheet visible, so hyperlink to it functions

The code goes in the table of contents worksheet code module. It get the name of the sheet, removes any single apostrophes, makes that sheet visible and the hyperlink does its thing...
'---
Private Sub Worksheet_FollowHyperlink(ByVal Target As Hyperlink)
  On Error GoTo FixThings
  Dim strAddress As String

  strAddress = Application.WorksheetFunction.Substitute(Target.SubAddress, "'", vbNullString)
  ThisWorkbook.Worksheets(VBA.Left$(strAddress, VBA.InStr(1, strAddress, "!") - 1)).Visible = xlSheetVisible
  Application.EnableEvents = False
  Target.Follow
FixThings:
  Application.EnableEvents = True
End Sub

What you're describing is Outlook saving to the "default" calendar set for the profile which is not governed by the default email account but rather by the "data file" set as the default for the profile done by going to

The default email account and default data file set for the Outlook profile are mutually exclusive - setting one does not impact the setting of the other.
