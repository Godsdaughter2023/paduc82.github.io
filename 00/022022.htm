People may not be able to open your file if it is encrypted but you definitely cannot stop someone from moving a file to another system -- not unless you:

1) lock down Windows' ability to copy a file from a hard disk to another device such as a CD or DVD or thumb drive,
2) disable all backup capabilities on the machine,
3) lock out server based and peer-to-peer services Windows Skydrive, FolderShare, LiveMesh, or their competitors, and
4) block someone from disconnecting a USB hard drive from one machine, walking it over to another, and connecting it to that machine.

That excludes even more drastic steps like moving an internal hard drive from one machine to another. {grin}

Also, while I haven't checked this out, my understanding is that there are plenty of ways -- researchable via Google or Live -- to bypass passwords on Office documents.

You also cannot stop them from opening your file with an arbitrary application (say a hex editor).

FWIW, if you encrypt a file with another program (as you appear to be doing), that program has to unencrypt that file and at least temporarily save it on your hard disk so that the Office program can use that file.  Someone sufficiently determined should be able to locate and copy this temporary file...shouldn't they?

Bottom line.  If your file is so important, the best thing to do is not distribute it. :)
