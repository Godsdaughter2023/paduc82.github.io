rich007a say:

Hi.

Actually I think the Presentation_NewSlide event would be better....

 

Here goes...
 
1) Create a new Presentation
 
2) Press Alt+F11 to open the VBE.
 
3) In the VBE do Insert > Class Module.
 
4) In the Properties Pane (press F4 if you can't see it) rename the Class module to:  cEventClass
 
5) Paste the following code into the cEventClass module:
 
 
Option Explicit
 
 'Declare a variable to handle application events:
Public WithEvents PPTEvent As Application
 
Private Const strClName As String = "Solution_Stacked_Chart_Table_ 3_Box"
 
Private Sub PPTEvent_PresentationNewSlide(ByVal sld As Slide)
    Dim cl As CustomLayout
    Dim shpCell As Shape
    Dim tbl As Table
    Dim r As Long, c As Long
       
    Set cl = sld.CustomLayout
    If cl.Name = strClName Then
        Set tbl = sld.Shapes.AddTable(10, 3).Table  '(r, c)
       
        tbl.ApplyStyle "{073A0DAA-6AF3-43AB-8588-CEC1D06C72B9}", True
        tbl.FirstRow = True
        tbl.HorizBanding = True
       
        For r = 1 To tbl.Rows.Count
            For c = 1 To tbl.Columns.Count
                Set shpCell = tbl.Cell(r, c).Shape
                With shpCell.TextFrame
                    .TextRange.Font.Size = 9
                    .MarginLeft = 0.03
                    .MarginRight = 0.03
                    .MarginTop = 0
                    .MarginBottom = 0
                End With
            Next c
        Next r
    End If
End Sub
 
 
6) Still in the VBE, do Insert > Module.
 
7) Paste the code below into the normal code module:
 
 
Option Explicit
 
Public cPPTObject As New cEventClass
 
'THIS WILL ONLY RUN WHEN IN AN ADD-IN AND THE ADD-IN IS LOADED
Sub Auto_Open()
     'Create event object:
    Set cPPTObject.PPTEvent = Application
End Sub

 

 
8) Save the presentation as a Macro Enabled Presentation (*.pptm)
 
9) Now save the presentation as a PowerPoint Add-in (*.ppam)

 

10) Close the pptm presentation (keep this file to be able to edit the code in the future).

 

11) Do: Developer tab > Add-ins > Add new > load your new ppam file > Close.

 

Now you should be able to add a new slide using the "Solution_Stacked_Chart_Table_ 3_Box" Custom Layout and your table will be inserted.

 

Note that it won't work if you Apply the layout to an existing slide, you have to insert a new slide with that layout.

 

Next you'll be telling us that you have several Placeholders in your Custom Layout and the new table is being added to the wrong Placeholder!  Should that be the case, have a read of my posts on page 1 of http://answers.microsoft.com/en-us/office/forum/office_2010-powerpoint/placeholder-master/0d51dcec-f982-4098-b6b6-94785304607a.

 

Hope that helps.
Cheers
