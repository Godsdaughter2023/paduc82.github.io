Hi,

The CMOS battery on the motherboard is likely dead and needs to be replaced. You get the prompt because the system BIOS no longer recieves a trickle of power from this battery to retain custom settings, so by default it prompts you to create them (should any exist) each time you power on. Replacing this nickel-sized battery is fairly easy, and they can be obtained at most stores that sell batteries for watches, garage door openers, etc.

Reinstalling Vista would not help as this problem is with the system hardware at a very basic level, not the operating system.
