Hello KCKay,

 

Here is a real-time monitoring example of the Word application, it’s more convenient than the pop-up dialog box prompts.

 

Macro to refer:

-------------------

Option Explicit

 

Public whenRun As Date

'#############################
'Check if Word instance is running.
'#############################
Sub CheckWordAppProc()
    Dim wdApp As Object
   
    'When an error occurred, goto the label“errHandler”.
    On Error GoTo errHandler
   
    'Get the current Word instance。
    Set wdApp = GetObject(, "Word.Application")
   
    '╱＊Pop up a message box If there are no errors.＊╱
    If Err = 0 Then Application.Caption = "Word application is running"
    'MsgBox "Word application is running", vbExclamation, "Tips"
    Call EnableWordAppChecker
    Exit Sub
   
errHandler:
    Application.Caption = "Word application is not running"
    'MsgBox "Word application is not running", vbExclamation, "Tips"
    Call EnableWordAppChecker
End Sub

'#####################################
'Check Word application once every second.
'#####################################
Sub EnableWordAppChecker()
    whenRun = Now + TimeValue("00:00:01")
    Application.OnTime whenRun, "CheckWordAppProc"
End Sub

'#####################################
'Stop checking if Word application is running.
'#####################################
Sub DisableWordAppChecker()
    On Error Resume Next
    Application.Caption = ""
    Application.OnTime whenRun, "CheckWordAppProc", , False
End Sub

 

Please run the macro ‘EnableWordAppChecker’ to check whether the Word application is running. If you want to stop the real-time monitoring, then run the macro ‘DisableWordAppChecker’.

bernie deitrick say: If your values are in column A with the header in row 10, and your dropdown is a DV list on cell F2, then copy this code, right-click the sheet tab, select "View Code" and paste the code into the window that appears. Change the A and the 10  to the column of values and row of headers, and the F 2 to the actual  cell address. 

Private Sub Worksheet_Change(ByVal Target As Range)
    If Target.Address <> "$F$2" Then Exit Sub
    Application.EnableEvents = False
    With Range("A10", Cells(Rows.Count, "A").End(xlUp))
        .AutoFilter
        .AutoFilter Field:=1, Criteria1:="<>" & Target.Value, Operator:=xlAnd
    End With
    Application.EnableEvents = True
End Sub
