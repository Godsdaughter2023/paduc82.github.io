I don’t know of a setting/process but this should work after you click Shutdown. Click the Windows Orb (Start), type regedit, press Enter and in the left pane navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet and click on the Control folder. Test it first, set to, say, 5 minutes (300000).

Change Wireless Channel - Microsoft Community

xbr dionysus say:

Change Wireless Channel

Login Information

  • Gateway - 192.168.10.1
  • Username – admin
  • Password – admin
 
Change Wireless Channel
  1. Open web browser.
  2. Login to router by entering http://192.168.10.1 into the address bar.
  3. If prompted for a username and password use admin/admin.
  4. Select Wireless on the left.
  5. Select Basic from the sub-menu on the left.
  6. Uncheck the Auto Channel box.
  7. In the Channel drop down select desired channel. (1,3,9,11)
  8. Click Apply.
