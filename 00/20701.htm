tcmnyc say:

This was driving me crazy but I think I found something that works.

Create each month in a different document, and save.

Then, in the document with your first month, select Insert, Object, Microsoft Word Document, and then select the document for your second month.

It will come in unchanged -- that is, it won't reconfigure to the month you already have.

Repeat this process for each additional month and you should have an annual calendar.

daniel g. grau say:
Calendars are created from predefined templates for a specific month and year, and the resulting document includes macro code to update the calendar for a single month.

The only way to have Word repeat this 12 times is to modify the macro code. This would take a longer than doing the calendar creation process 12 times in a row and then pasting the pages together.

Thus, unless you create calendars for a living, doing it manually is the easiest way. However, for combining months from several calendars, you'll need to do this:

  • Command A (i.e., Select All)
  • Command Shift F9 (i.e., unlink selected fields)
  • Copy
  • Switch documents
  • Paste

If you don't unlink the fields before pasting, you'll end up with identical copies of the same month.
