bernie deitrick, excel mvp 2000-2010 say:

in cell A1 of the third tab, use this to combine column A from Sheet1 and column A from sheet2, and copy until you get blanks returned:

=IF(ROW()>COUNTA(Sheet1!A:A)+COUNTA(Sheet2!A:A),"",IF(ROW()<=COUNTA(Sheet1!A:A),INDEX(Sheet1!A:A,ROW()),INDEX(Sheet2!A:A,ROW()-COUNTA(Sheet1!A:A))))

How to make macro enabled sheet (xlsm) independent of - Microsoft Community

hisudhir007 say:

How to make macro enabled sheet (xlsm) independent of workstation?

Hi, I have added openCalender Macro and saved the sheet as xlsm. It works perfect in my workstation.  But when the same xlsm file is transferred to other workstation, this macro is not visible. How to make this macro sheet independent of workstation?

Note that when I open this sheet  in other workstation, I don't get any message asking for enabling/disabling macros. .

paulcordts say: The reason you don't get any message on another workstation is most likely caused by the security setting on that workstation.  Sometimes this is set to disable macros without notification.

Check the settings on the other workstation.

If you are the only one using the file, this should suffice.  If this file is going to be distributed to other and potentially used on many workstation you many need to put an instruction sheet showing users how to properly set macro security to use this file.

If it is really critical that users enable the macro, you can go one step further and hide all the sheets except the macro instruction sheet.  You then have an Auto_Open macro that runs when the file is opened that hides the instruction sheet and unhides all the others.

Thus if the user has macros enabled, they end up seeing the full file.  If macros were not enabled all they can see is the macro instruction sheet.

I have used this before.  The open and close macros look like this

Sub Auto_Open()
WSCount=Application.Worksheets.Count
For S=1 to WSCount
     If Sheets(S).Name <>"Macro Instructions" then 
Sheets(S).Visible=True
     else
Sheets(S).visible=xlSheetVeryHidden
     End If
Next S
End Sub

Sub Auto_Close()
WSCount=Application.Worksheets.Count
For S=1 to WSCount
     If Sheets(S).Name ="Macro Instructions" then 
Sheets(S).Visible=True
     else
Sheets(S).visible=xlSheetVeryHidden
     End If
Next S
End Sub

Note:  xlSheetVeryHidden is a property only available through VBA.  Sheets hidden this way don't even show up on the list of sheets to Unhide.  This way the user has to enable macros to even use the file.

You can also use the regular xlSheetHidden instead, but then the user can unhide these sheets even if macros are not enabled
