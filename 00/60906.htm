ken blake say:

First, note that this has nothing to do with what version of Windows you are running. It has to do with what application programs you have installed.

Second, note that a .wps file is a Microsoft Works text document file, and you apparently don't have Works installed. You can find out what any extension is by going to http://filext.com/

Third, whoever sent you that file foolishly assumes that everybody will be able to open it. He was wrong. Not everyone has the same programs installed that he does.

Fourth, it is usually very unwise to open most e-mail attachments; even ifthe attachment legitimately comes from a friend or relative, it can contain a virus. I'm not suggesting that a friend is likely to send you a virus on purpose, but if the friend is infected without realizing it, any attachment he sends you is likely to also be infected. This is especially dangerous if the attachment comes from someone who doesn't realize that not everyone has Microsoft Works installed.

Personally I never open attachments at all, except from a *very* few trusted sources, and then only when I'm expecting them.

But if you insist on seeing what was sent, write to the sender and ask him to resend it in some format you know you can open: perhaps a text file, a .doc file, a pdf file, etc.
