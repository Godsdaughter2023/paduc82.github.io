Why does my movie maker stop at 37% complete - Microsoft Community

gwandogirl say:

Why does my movie maker stop at 37% complete

I have created a movie file, when trying to burn to DVD only get to 37% and it stops there. Can you tell me why.

considerate_guy say:

Usually the cause of being unable to save your movie is using incompatible file types.  Movie Maker prefers .wmv for video and .wma for audio, so if you are using other file types, consider converting them.

Another common cause is having too "complex" a project.  If your project has lots of elements, then consider breaking it into smaller sub-projects.  You can do this by making multiple copies of your original project.  Then in each copy, delete a portion of the project items.  For example, if you were breaking it into 3 projects, you would keep 1/3 in the first project copy and delete the last 2/3 of the project. Test that it will save as a movie.  In the next one you would delete the part you already saved and again pare down from the end until you have the next 1/3 (or so).  Save that as a separate movie.  Repeat for the last third.

In the end you can always create one final project where you import the movies created from the 3 projects, and make your final movie. Or you can call Windows DVD Maker directly and add the 3 parts.

Hi Gwandogirl,

If you burn a ".wlmp" project file in Windows Live Movie Maker it will save the file as a ".wmv" first, then it will automatically import the ".wmv" file into Windows DVD Maker. • The current 2011 version: 15.4.3555.0308
• Windows Live Movie Maker is not supported on Windows XP

3. Ensure that you have the proper codecs to play your videos and audio files - Audio and Video require codecs to play the file and it’s important to have the right codecs installed or updated in your PC. 
• Try running the Microsoft FixIt tool for codecs: http://support.microsoft.com/mats/video_freezes_or_crashes/

Note: If can't find the codec you need you may convert the video to a supported format (eg. wmv) and then add it to your Windows Live Movie Maker project.

greg carmack - windows mvp 2010-2020 say:
Hi Krong. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

As long as the underlying license you upgraded or that came with WIndows 10 is retail and not factory preinstalled or builder's OEM, then it can transfer since Upgrade inherits the rights and restrictions.

The procedure is here: https://support.microsoft.com/en-us/help/20530/...

It is possible to do yourself if you'd previously signed in with your Microsoft account and it linked to the Digital License as shown in Status at top of the Activation Settings page. Then when you change motherboard, you'd sign in again and run the Troubleshooter. We recently learned it will even make exceptions for OEM if all else is proper. So that's your best bet.

If not you'll need to contact Microsoft Support, asking for Activation help, then keep asking for an agent. https://partner.support.services.microsoft.com/...

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.

robert aldwinckle on forums say:

The only fix I currently know of is connecting my computer to my 1920x1080 HDMI display.

If it is really associated with another monitor you could press Win-Shift-CursorRight.  

If it is associated with the current monitor but offscreen, you can press Alt-Space, M, CursorRight and then drag the window into position.  The window must be in its normal mode for this (otherwise the Move menu item will be inactive).  FWIW I have to do this all the time with my PowerShell window on multiple platforms.

Another alternative would be using right-click on Taskbar for Cascade Windows.  Then be sure to minimize all other window that you do not want involved in that operation.

 

Good luck
