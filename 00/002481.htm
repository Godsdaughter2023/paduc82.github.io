doug robbins - mvp office apps & services (word) say:

If the data that you have is in Sheet1, running a macro in Excel that contains the code shown below will populate Sheet2 with:

QTY PRODUCT_NAME SKU DESCRIPTION LABEL_NUMBER
2 Widget WD001 Plastic Widget 1
2 Widget WD001 Plastic Widget 2
2 Widget WD001 Plastic Widget 3
4 Ball BL002 Blue Ball 1
4 Ball BL002 Blue Ball 2
4 Ball BL002 Blue Ball 3
4 Ball BL002 Blue Ball 4
4 Ball BL002 Blue Ball 5

which you can then use as the data source for your mail merge.

Here's the code:

Dim source As Worksheet
Dim target As Worksheet
Dim i As Long, j As Long, n As Long, m As Long
Set source = Sheets(1)
Set target = Sheets(2)
With source.Range("A1")
    For i = 0 To .CurrentRegion.Columns.Count - 1
        target.Range("A1").Offset(0, i) = .Offset(0, i)
    Next i
    target.Range("A1").Offset(0, .CurrentRegion.Columns.Count) = "LABEL_NUMBER"
    m = 1
    For i = 1 To .CurrentRegion.Rows.Count - 1
        For j = 1 To .Offset(i, 0) + 1
            For n = 0 To .CurrentRegion.Columns.Count - 1
                target.Range("A1").Offset(m, n) = .Offset(i, n)
            Next n
            target.Range("A1").Offset(m, .CurrentRegion.Columns.Count) = j
            m = m + 1
        Next j
    Next i
End With

You can use Internet Explorer's ActiveX Filter feature to prevent videos from automatically starting as you visit websites. The filter can only be set to on or off for a website, so once you allow a site to play ActiveX content, all pages on that site will be able to play ActiveX content.

peter b* say:

'Mark Index Entry’ dialogue box.

The box claims that ‘This dialogue box stays open so that you can mark multiple index entries’, BUT. After marking a term, select another. Click anywhere in the box, and the ‘Main entry’ pigeonhole changes to the new selection, and if you want to edit it you (usually) can. But if you then try to add a subentry, clicking in the ‘Subentry’ pigeonhole makes the main entry revert to the new selection. And if you click in that pigeonhole to re-edit ‘Main entry’, the subentry disappears (etc., etc., forever). So if you’re using subentries, you have to close the box after each mark (one step) and reopen it (two more steps).
If this can’t be fixed, the claim should be revised to ‘This dialogue box stays open so that you can mark multiple main entries (but not subentries)’.
