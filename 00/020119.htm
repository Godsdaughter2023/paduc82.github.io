jay freedman say:

Try going into File menu > Options Command > Advanced Option > scroll down to "Print" section and make sure that "Allow fields containing tracked changes to update before printing" is turned OFF.

I'm not sure that will help if the document doesn't contain tracked changes. However, I don't have any idea why the page numbers are changing if the document outside the TOC isn't changing.

A different approach is to lock the TOC field so it won't update unless you first unlock it. Click anywhere inside the TOC after updating it to the correct page numbers, and press Ctrl+F11. The TOC is now locked.

If you edit the document (or change to a different printer or move the document to a different computer that has a different printer, which can affect the pagination), click in the TOC and press Ctrl+Shift+F11 to unlock it; update it with F9; and lock it again.
