It sounds like your document is opening in Web Layout view. When that happens, click the Print Layout button in the bottom right corner of the Word window, or click the Print Layout button on the View tab of the ribbon.

This is strange, because Word 2013 and 2016 normally force all documents to open in Print Layout view. That's a change from Word 2010 and earlier, which reopened documents in the same view they had when they were closed. I created the SaveViewZoom add-in to restore the old behavior, and it should help you, too. Download the SaveView2013.zip file from http://jay-freedman.info/ and follow the instructions on that page to install it. Then, when you open a document and change it to Print Layout view, make any change in the document and save it, it will automatically open the next time in Print Layout.
