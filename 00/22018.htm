graham mayor say:

By default your macros would have been stored in the normal template, which for Word 2007 was normal.dotm, and which if you have not changed the default location is
located at :-
%appdata%\Microsoft\Templates

Type this address in the address window of Windows Explorer or your Internet Explorer browser and you will be taken to the folder that contains the default template.

Word 2010 also uses this location and a normal template of the same name. When you installed and replaced Word 2007, it should have adapted that template of Word 2007 retaining the macros etc that it contained.

Check whether that folder contains normal.dotm. Check the file date of that file. Do you have a backup of your normal template - maybe even in that folder?

The folder should be configured as a trusted location. Is it?

If you place your 2007 backup of the normal template in that folder and start Word, Word 2010 will use it, but will display Compatibility Mode in the title bar.
To overcome that, open the normal template in Word for editing then save as Normal2.dotm. Close Word (and Outlook) and rename the normal.dotm to oldnormal.dotm and the normal2.dotm to normal.dotm. Restart Word.

For macros in document templates and add-in templates, ensure that the startup folder and template folder are set as trusted locations.
