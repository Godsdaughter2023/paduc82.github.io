    iftikharansari say:

    How can I make the footnote reference font different in body text and footnote text?

    I have just installed Office 2011 for Mac and the default settings for the footnote text style made it 12 point font. When I changed the footnote text style to 10 point font, I found that the reference number remained 12 pt. But then changing the footnote reference style to 10 pt changes the reference number in the actual body of the text as well.

    I want the footnote reference number in the body of the text to match the surrounding text, but I want to set the footnote text - all the text, including the reference number - to be 10 point font. How do I do this?
    john mcghie say:

    You need to set up both the "Footnote Reference" and "Footnote Text" styles.

    Go to Format>Style, set the list to "All Styles", otherwise you won't see the two styles you need.

    Select each, choose "Modify" and set the properties you want.

    Note: If you have been using direct formatting in the documents, you need to remove that, because it will override the style.

    Use Advanced Find/Replace to search for the font properties of the direct formatting (e.g. Superscript and 12 points) then replace with the style "Footnote Reference".

    With the footnote text, it's probably easier to simply convert the footnotes to EndNotes, then open the endnotes, use Control + SpaceBar to remove the direct font formatting, then convert them back to footnotes.

    You may want to do these operations on a COPY of the document just in case of disaster :-)

    Hope this helps 

    rich michaels - mvp (greatcirclelearning.com) say:

    Supporting John's reply, here is a picture of how to open the Styles toolbox from the Home tab so you can modify those styles he mentions.

    1. Click the Styles Toolbox button on the Home tab

    2. The Styles toolbox opens

    3. If you don't see the two styles in the list, change the list setting to All Styles and then scroll the list

    4. Modify the style by clicking on the blue paragraph mark that's to the right of the style name. Make sure you save the changes to the template if you want them to apply to all new documents based on the template.

    peter schleif say:

    Steve Heff wrote on 2010-09-09 23:40 CET:

    Can I do it with SendKeys, or is there a more elegant approach?

    Using the DOM is a more elegant approach. First you have to locate the form in the HTML source.

    • With  getElementById()  if the form has a unique ID.
    • With  getElementsByName()  if the form has at least a name.
    • With  getElementsByTagName()  when the form has neither ID nor name.

    Normally you only need one of these three methods. However, the sample code below contains all three methods in order to show how to use them. Once you have the form, you can loop through all the form fields with foreach. Within the loop, check the name of each element (if available) and write into the designated text.

    Please check out this sample page:  http://vba2htmlform.peterschleif.bplaced.de

    Peter


    Sub Vba2HtmlForm()
       Dim ie      As Object
       Dim frm     As Variant
       Dim element As Variant

       Set ie = CreateObject("InternetExplorer.Application")
       ie.navigate "http://vba2htmlform.peterschleif.bplaced.de"
       While ie.readyState <> 4: DoEvents: Wend

       'try to get form by ID
       Set frm = ie.document.getElementById("myFormID")

       'try to get form by name
       If frm Is Nothing Then Set frm = ie.document.getElementsByName("myFormName").Item(0)

       'try to get form by tag name and index.
       'Item(0) = 1st form
       'Item(1) = 2nd form
       'Item(2) = 3rd form, etc.
       If frm Is Nothing Then Set frm = ie.document.getElementsByTagName("form").Item(0)

       If frm Is Nothing Then
          MsgBox "Form not found"
          ie.Quit
          Set ie = Nothing
       Else
          ie.Visible = True
          For Each element In frm.elements
             On Error Resume Next
             Select Case element.Name
                Case "text1": element.Value = Sheets("Sheet1").[A1]
                Case "text2": element.Value = Sheets("Sheet1").[A2]
                Case "text3": element.Value = Sheets("Sheet1").[A3]
             End Select
          Next
       End If
    End Sub

    doug robbins - mvp office apps & services (word) say:

    There are several types of drop down list (or combobox) controls that can be used in Word

    1. A DropDown type FormField as used in a document that is protected for filling in forms
    2. A Combo Box Content Control (with which the user is not restricted to the items in the list)
    3. A Drop-Down List Content Control
    4. An ActiveX Combo Box
    5. A UserForm Combo Box

    While it depends on the exact purpose for which you want to use the drop down, but possibly the Drop-Down List Content Control is the one that will be most suitable.

    For information on the use of Content Controls, see the following page of Greg Maxey's website

    https://gregmaxey.com/word_tip_pages/content_controls.html

    milagrosm2020 say: Hi, gerlanmituli

    My name is Milagros, Independent Advisor

    1-En Outlook.com, vaya a Configuración Settings > Ver todas las opciones de Outlook > Sincronizar el correo electrónico.

    2-En Cuentas conectadas, seleccione Gmail.

    3-En la página Conectarse a su cuenta de Google, escriba un nombre para mostrar de la cuenta de Gmail. Este es el nombre que verán los destinatarios cuando reciban un mensaje de correo electrónico suyo. Seleccione Aceptar.

    4-Seleccione la cuenta de Gmail que va a usar. Es posible que deba proporcionar un nombre de usuario y una contraseña para la cuenta de Gmail.

    Nota: Si ha activado la verificación en dos pasos para su cuenta conectada, tendrá que crear una contraseña de aplicación para esa cuenta. Vaya a la página de seguridad de la cuenta de Gmail para crear la contraseña de aplicación y, después, vuelva a Outlook.com para introducir la contraseña en la página Conectarse a su cuenta de Google. Obtenga más información sobre la verificación en dos pasos de Google aquí.

    4-Seleccione Permitir.
