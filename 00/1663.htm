robaxelrod8928 say:

Disabling all syncing using OneDrive for Business

What is the best way to disable all syncing for ODB? I was considering using the 

Set-SPOTenantSyncClientRestriction

timliu_msft1261 say:

hi rbaxe22,

it seems that this is the easiest way to achieve your requirement. to learn more, see set-spotenantsyncclientrestriction.

meanwhile, there are some other methods for your reference. we can use the options below to disable the sync feature or disable the personal site:
option 1. for a site
  a. navigate to the site.
  b. click setting gear>site settings>search and offline availability.
  c. chose no for allow items from this site to be downloaded to offline clients in the offline client availability section.
  d. click ok.
    
option 2. for a library.
  a. navigate to the library.
  b. click the library tab>library settings>advanced settings.
  c. choose no for allow items from this site to be downloaded to offline clients.
  d. click ok.
note: every user's personal onedrive for business site has a document library, if we want to use this option to disable the sync feature, we need to navigate to the sharepoint admin center>user profiles>find the user>choose manage site collection owners>add your account as site collection administrators, and then navigate to the user's personal library to disable the sync feature. if you have lots of users, you have to repeat these steps many times.

option 3. disable the personal onedrive for business site.
   a. navigate to the sharepoint admin center.
   b. user profiles> manage user permissions.
   c. choose the existing everyone group (or the similar group includes eveyone) and uncheck the box of create personal site.
   d. click ok.
note: users' existing personal onedrive for business sites will disappear. they will only see the admin's personal site when they click the onedrive tile.

best regards,
tim
