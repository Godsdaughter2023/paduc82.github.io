charles kenyon say:
Never, ever, not even once open a Word document as an email attachment if you intend to do more than read or print it. You know why, now. When you open a Word document that is an email attachment from the email, it downloads and stores itself in a temporary file folder. Then, when you press save, it saves itself to that temporary location.

With luck it it is still there. If you know the name of your document search for it in Windows. Search your computer, not your documents. Turn on the search option to search hidden and system folders and files.

Another possibility is within Word with a blank document on the screen go to your file tab and look in Recent. The number of documents shown here is set in your Advanced Options under the File tab. I don't think making the number larger will recover anything that scrolled off the bottom of the list, though.

If you find it, open it and use Save As to save it someplace you can find it more easily.

It is called a temporary folder (i.e. Temporary Internet Files) because it isnat intended to be for things that you want to keep. You would hope that Word would know better, but it doesn't. It sees that you edited a file that has already been saved, so when you press Save, it follows your orders and saves your changes back to that temporary fiolder.

Hope you can find your file.

Resetting the handset is a little harsh in this case - I would leave that to the last resort.
