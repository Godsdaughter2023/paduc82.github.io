a. user say:

Hello Robert,

we are in the similar situation as well... the below script will help you in retrieving the Distribution group and its members ( works on nested group as well )

==========================================

$saveto = "C:\DistributionGroups.txt"

filter get_member_recurse {

   if($_.RecipientType -eq "MailUniversalDistributionGroup") {

       Get-DistributionGroupMember -ResultSize "Unlimited" $_.Name | get_member_recurse

   } else {

   $output = $_.Name + " (" + $_.PrimarySMTPAddress + ")"

   Write-Output $output

   }

}

$DistributionGroup = Get-DistributionGroup –resultsize unlimited | Sort-Object Name | ForEach-Object {

   "`r`n$($_.DisplayName) ($($_.PrimarySMTPAddress))`r`n=============" | Add-Content $saveto

   $distout = Get-DistributionGroupMember -ResultSize "Unlimited" $_.Name | get_member_recurse

   Write-Output $distout | Sort-Object | Get-Unique  | Add-Content $saveto

}

=====================================

If you're just looking for the name of the distribution groups in csv file, this simple command will help

Get-DistributionGroup -ResultSize unlimited | Select-object Name, DisplayName,GroupType | Export-Csv "C:\Distributiongrouplist.csv"

jay freedman say:

Click File > Options > Customize Ribbon and click the Customize button in the bottom left corner of the dialog. In the Keyboard Shortcuts dialog, set the Categories to Fonts (near the bottom of the list).

Then click on the font you want, click in the "Press new shortcut key" box, and type the shortcut combination for that font; and click the Assign button. Repeat to assign shortcuts to other fonts. Finally, close the dialog.

When you exit Word, if you're prompted to save changes to Normal.dotm, click Save.
