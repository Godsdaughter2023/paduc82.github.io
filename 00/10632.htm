hansv mvp say:

AVERAGEIF and AVERAGEIFS expect contiguous ranges. You can use an array formula (confirmed with Ctrl+Shift+Enter instead of just Enter) like this:

 

=AVERAGE(IF((A1:Z1>37)*ISODD(COLUMN(A1:Z1)),A1:Z1))

 

This will compute the average of the cells in A1:Z1 whose value is >37 and whose column number is odd (i.e. columns A, C, E, ...)
