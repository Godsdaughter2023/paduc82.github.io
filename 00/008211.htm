Greetings-

I have located an approach using older, Windows versions of Excel, however the Excel 2016 for Mac does not provide an "option" button.  http://excelribbon.tips.net/T009043_Replacing_Background_Colors_in_Cells.html

 http://excelribbon.tips.net/T009043_Replacing_Background_Colors_in_Cells.html

Thank you in advance, all suggestions are greatly appreciated. 

bernie deitrick say:

Copy the colored cells and paste formats, to create a column of colored cells.  Enter the number that you want for each color in a second column, next to the color.

Then select the colored cells and use this macro:

Sub EnterColors()
    Dim r As Range
    For Each r In Selection
        r.Value = r.Interior.ColorIndex
    Next r
End Sub

Then name the two columns of values "ColorTable".

Finally, select the cells that you want to convert, and run this macro

Sub ConvertColors()
    Dim r As Range
    For Each r In Selection
        r.Value = Application.Vlookup(r.Interior.ColorIndex,Range("ColorTable"),2,False)
    Next r
End Sub
