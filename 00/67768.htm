    the program is properly installed.

    Create shortcut will put an icon on the desktop; Pin to start will place a tile on the start screen, while Pin to taskbar will place an icon on the taskbar.

Note: The tool might take 15-20 minutes to finish running, so please do not cancel it.

1.Launch PowerShell
2.Set-ExecutionPolicy Unrestricted
3.Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

Restart your device.
