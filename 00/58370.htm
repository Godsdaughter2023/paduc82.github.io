If you get the message, "User Profile Service failed the logon.  User profile cannot be loaded." -- You are loading a TEMPORARY PROFILE.  It does not matter if that PROFILE belongs to an administrative account or other level account.  It does not matter if it belongs to the built-in "administrator" account, guest account or other.  It's still indicating you are loading a TEMPORARY PROFILE because something is 'messed up' with the user's original profile.
 
Easiest thing - if you have another "administrative level" account - reboot and log in as that one, look under the "USERS" folder and find the folder there with the same name as the user account giving you trouble.  Rename that folder to something COMPLETELY different.  Log out and log in as the user giving you the issue - did you get the error message?
