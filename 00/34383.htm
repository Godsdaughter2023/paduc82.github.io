To anyone who is interested. I found a relatively simple solution to the Blitzkrieg 2 run problem, as well as the same problem with many other "older" or "non-supported" games.

It can actually be done two different ways. The key is to set your computer up as two separate systems. It takes a little work, but once done, it cures all of these problems permanently.

NOTES: This involves re-setting up your computer so that it operates on two systems. Don't panick, it's far easier than you would think.  BTW: The commentator who said WinXP Pro doesn't support 3D or gaming is wrong. I've used Pro for years without a problem and it does support 3D, both in on-board chips and on video cards. Just make sure you install the video drivers for your motherboard or video card.

For my installation I used Windows 98 Home edition and Windows 7. I would have used Windows 98 Pro, but I made the mistake of loaning my copy to someone, thinking I wouldn't need it again soon, and never got it back!

Here's what to do:

Method One...
1) Backup your system to an outside drive. (This could be an external or extra hard drive, the cloud etc..)
2) Format your computer's hard drive and create 2 partitions in the format setup. One will be used for your Win7 programs and the other will be your "Games" drive.
4) After formatting is complete, restart your computer. When your computer comes to the start up window, hit DEL and go into the BIOS setup.
5) Go into the "Boot Sequence" settings and set your computer to boot from a CDROM drive first, Hard Drive 2nd. Save and shut down.
6) Boot your computer with the CD for an older version of Windows (Win98, XP, etc.) in the CDROM drive.
7) Install the older version of Windows (Win98, XP, etc.) FIRST on the "Game" partition. (Usually the E: drive).
8) Shut down your computer when installation is complete.
9) Restart computer with the Win7 CD in the CDROM. Install to the other partition (usually the C: drive).
10) When complete, shut computer down and restart. At the start up screen hit DEL, go into BIOS and reset the Boot Sequence to boot first on the hard drive, then on the CDROM. Save, restart.

Re-install your favorite or appropriate software on drive C: with the Windows 7 system and install your games on drive E: with the older version of Windows system.

From now on, when you start your computer, it will give you a screen that provides a choice between which version of Windows you wish to use. If you are going to use Windows 7 applications, then choose Windows 7. If you are going to enjoy one of your "older" games, simply choose "An Older Version Of Windows"!

The alternate method for doing this is simply to purchase a second hard drive and follow the same procedure, but install the older version on the second hard drive.

REMEMBER: Which ever way you do this, you MUST install the OLDER version FIRST. It won't work the other way around.

I now have games such as Blitzkrieg 2, Silent Hunter and many others running without problems when ever I wish, because this eliminates all of the security, administrator and new windows compatibility issues by taking your computer back to an earlier and simpler time when playing a game was all we really wanted to do! I hope this helps end the frustrations of many of my fellow "older" gamers!
