I have looked at this thread several times and without details of the HyperLink and your workbook it is difficult, if not impossible, to test.


However, some advice with code that is in the worksheet's module.


Because you are changing the ActiveSheet (Selecting another worksheet) in the code then this might be your problem because when the ActiveSheet is changed in code from within a module attached to a worksheet, Excel becomes confused and the end results are indeterminable.


If it is necessary to select other worksheets then it is best to place the code in a standard module and call it from the worksheet module and pass the required variables to it like the following example. Note: This code is untested and simply to give you the idea of how to code when changing the ActiveSheet from code within a worksheet module.


'This event code goes in the worksheet's module
Private Sub Worksheet_FollowHyperlink(ByVal Target As Hyperlink)
   
    'Pass the SubAddress to the called sub
    Call TestCode(Target.SubAddress)
        
End Sub



'The following code goes in a standard module


Sub TestCode(strLinkTo As String)
    
     WhereBang = InStr(1, strLinkTo, "!")
     If WhereBang > 0 Then
         priorsampleprojects = Left(strLinkTo, WhereBang - 1)
         Worksheets("priorsampleprojects").Visible = True
         Worksheets("priorsampleprojects").Select
         MyAddr = Mid(strLinkTo, WhereBang + 1)

        Worksheets("priorsampleprojects").Range(MyAddr).Select
     End If
    
     End    'Ends processing without returning to calling sub
    
End Sub


You also need similar coding anywhere that you have code in a worksheets module that changes the ActiveSheet (ie. Selects another worksheet.)

Messaging and People Hub Issues

In order to fix issues with Messaging app or People Hub, it is best to remove and add your Facebook account from your account as well as your phone. This is necessary as there are no advance settings that are available for tinkering on the phone or account, with respect to Facebook.

This consists of four steps.

  • Remove Facebook account from phone.
  • Remove Facebook account from your primary account.
  • Add Facebook account to your primary account.
  • Add Facebook account to your phone.

Remove Facebook account from phone.

  1. On the phone, go to SETTINGS.
  2. Press and hold on the Facebook account.
  3. You will get an option called REMOVE.

Remove Facebook account from your primary account.

  1. Visit outlook.com and login using your primary account. Not sure, which is your primary account? Check this wiki article.
  2. On the top right corner, will be your name and profile picture. Click on it to bring down a menu.
  3. Look for ACCOUNT SETTINGS.
  4. On the left, there would be an option called PERMISSIONS.
  5. Under PERMISSIONS, you would find MANAGE ACCOUNTS.
  6. Under this, you will have FACEBOOK.
  7. Select EDIT, and that will take you to a new page with a lot of options.
  8. At the bottom of this page, there is an option that says, ‘remove this connection completely’. Do this, and Facebook account is removed from your account.
  9. Go ahead and restart the phone.

Add Facebook account to your primary account.

  1. Visit outlook.com and login using your primary account.
  2. As before, go to ACCOUNT SETTINGS > PERMISSIONS.
  3. This time, choose, ADD ACCOUNTS. Facebook should be listed as one of the services.
  4. Select it and connect Facebook to your primary account. Ensure that you put a check mark to all the options.

Add Facebook account to your phone.

  1. On the phone, go to SETTINGS.
  2. Select ADD AN ACCOUNT.
  3. Enter Facebook details.
  4. To force an immediate sync, press and hold the Facebook account, and then tap the option for SYNC.

Facebook Messenger App

If you are simply trying to fix messaging through the messaging app, then it would be easier to use the Facebook Messenger app. It installs and works like a charm. Further, unlike the messaging app, the FB messenger app is updated frequently. So any issues that may present themselves will be ironed out relatively faster.

Link to Facebook Messenger app - Click here

Facebook App – Regular and Beta

Link to Facebook beta app - Click here

Link to Facebook app - Click here

Ideally, it is better that you are using the Facebook App – Regular, although there is no harm in using the beta version of the same app. The beta version has the benefit of getting more features, and more frequent updates than the regular version. The regular version has the advantage of being more stable than the beta version.

If either of the apps should develop an issue, try the following steps.

  • Remove the app from the phone.
  • Restart the phone.
  • Go back to the store, and download the app.
  • If issue is still not fixed, visit facebook.com.
  • Change your facebook password. If possible try to avoid a password with special characters and keep the password short. Many a times, the app simply refuses to work with complicated passwords. I have to add a mandatory disclaimer that a strong password must be as long as possible and have at least one special character.

Last Resort

If all else fails, do a phone reset. You can do this by going to SETTINGS > ABOUT > RESET. Please back up before you try this.
