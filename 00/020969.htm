Lets start and agree on some terminology.

Download - is copying a file from an external source to your local computer. The external source can be the internet or a plug in USB drive, anything not built in to the local computer.

Install - is an automated process that sets up a program to run on your computer. It creates the required folders and puts program files into them.  It also creates any required Windows Registry Keys. Some programs can run without further work, others require "Activation" to give you access to all features.

Activation - is the process of proving to the software owner, like MS, that you have paid for a legal license to use their program. 

There is no stated limit to the number of times you can "download" the Office installation files from MS.

There is no stated limit to the number of times you can move your single 365 activation to new machines.

macropod say:

You can use text representations of field codes in a Find/Replace expression. However, you can use:

Find = ^d ^d

to find two fields separated by a space. There is no Replace expression you can use with this to eliminate the space, so you'd have to do that manually. The alternative is to use a macro, such as:

Sub Demo()
Application.ScreenUpdating = False
Dim i As Long, Rng As Range
With ActiveDocument
  For i = 1 To .Fields.Count - 1
    If .Fields(i).Type = wdFieldIndexEntry Then
      If .Fields(i + 1).Type = wdFieldIndexEntry Then
        Set Rng = .Range(.Fields(i).Code.End + 1, .Fields(i + 1).Code.Start - 1)
        With Rng
          If Trim(.Text) = "" Then .Text = ""
        End With
      End If
    End If
  Next
End With
Application.ScreenUpdating = True
End Sub

julie fka profjulie say:

If you link contacts on the phone, they won't be linked in your Hotmail Contacts web client.  If you combine contacts in the Hotmail Contacts web client, the records get combined into one record.  When you sync the Hotmail contacts to your phone, the combined contact will also be synced. 

With linking - 2 separate records are still two separate records.  The records appear to be a single record because they are linked by a data item that is identifical across both records (i.e. the email address for both records is the same).

With combining records - the 2 separate records are combined into one record.
