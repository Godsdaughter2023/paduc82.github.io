Is there a way to revoke the hardware entitlement from the MS activation server for that specific computer before selling the computer because otherwise if I sell the entire computer without any hardware changes someone else could use the same hardware entitlement and activate Windows 10 for free even without the product key.  The hardware entitlement is created during activation of either the Windows 10 upgrade or purchase.

a. user say:

Hello,

Thank you for posting your concern here in Microsoft Community and welcome to the Windows 10 family. I would be happy to assist you.

From the description provided, I understand that you want to revoke the activation status of your computer before you sell it.

I would suggest you to initially uninstall the product key before passing on the entitlement of the computer to some one else.

Please let us know, if your copy of Windows OEM or Retail. If it is a retail copy, the product key binded to your computer can be uninstalled. Follow the below steps to remove the product key from the current computer leaving Windows not activated.

1. Press Windows key + X, select Command prompt (Admin) to bring up elevated Command prompt.

2. Run the command: slmgr /dlv

Note: Windows will open a message box and tell you information about the license status of your PC, here please take note of your activation ID.

3. To uninstall your product key you need to use the /upk switch, along with your activation ID:

slmgr /upk followed by your activation id.

In case if you want to use the same activation id to activate Windows on a different computer, run the following command:

slmgr /ipk followed by your activation id.

Hope the above information was helpful. If you need further information, feel free to write to us and we would be glad to help you.

Thank You.
