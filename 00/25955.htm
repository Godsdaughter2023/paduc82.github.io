suzanne s. barnhill say:
If you want line breaks in your original heading to be preserved in the TOC, you need to add the \x switch to the TOC field code. See http://word.mvps.org/FAQs/Formatting/TOCSwitches.htm. For other approaches, see http://WordFAQs.mvps.org/TOCTips.htm.

pamcaswell say: The  fast but (later in the TOC) manual way to do this is to replace the return  (enter) after "Chapter" with a line break (Shft+enter).  The TOC feature will ignore the line break when it generates the TOC.  To get  the  chapter and title on two lines in the TOC again add a line break between.  Note that whenever the TOC is updated, you will have to re-insert the  line breaks.

doug robbins - mvp office apps & services (word) say:

Follow the manual method in http://support.microsoft.com/kb/2528748 but as you are using Office 2007, you will have to change the command to:

reg add HKCU\Software\Microsoft\Office\12.0\Word\Options /v NoReReg /t REG_DWORD /d 1

graham mayor say:

The registry change stops the reconfiguration, but it does not address the underlying problem. I would as a first step remove that registry key and run Word once again as administrator (even if you are the administrator) by right clicking the icon in the start menu and selecting Run as Adminsitrator.
