It will be easier if you do not use a table and use a tab stop (a right aligned one would be best) to located the amounts.  Then use the following field construction in the mail merge main document, in place of the last three rows of your table (using CTRL+F9 to insert each pair of field delimiters}

{ IF { MERGEFIELD Discounts } = 0 { IF { MERGEFIELD Fees } = 0 "Ending Balance <tabspace> { MERGEFIELD EndingBalance }" "Fees<tabspace> { MERGEFIELD Fees }¶

Ending Balance <tabspace> { MERGEFIELD EndingBalance } }"}

Press the Enter key where ¶ appears above and press the Tab key where <tabspace> appears.

What happens when you press Alt+F9?  Is there any change to the Table of Contents?


Below the 2 automatic and 1 manual one that you refer to, there is an item "Insert Table of Contents" and if you click on that, the Table of Contents dialog will appear. 




In that dialog, you can change the levels to be shown from the default of 3.


Alternatively, if you then click on the Options button in that dialog, the Table of Contents Options dialog will appear and assuming that you are creating the Table of Contents from the Heading Styles used in your document, if you scroll down through the list of Available styles, you will see that Heading 1, Heading 2, and Heading 3 are assigned by default to TOC levels 1, 2 and 3 respectively.  If you enter a number say 4 into the box alongside Heading 4, TOC level 4 will be assigned to that Heading level.  After clicking on OK in the Options dialog, you may want to click on Modify to  modify the TOC 4 style.   When you finally click on OK, a Table of Contents will be inserted into your document (if there is one there already, you should be asked if you want to replace it).

Inserting Excel into Sway - Microsoft Community

alan-gain it say:

Inserting Excel into Sway

Can insert an Excel file into Sway?
jason (office engineer) say:

Alan,

I've learned that I misspoke earlier and that embedding from OneDrive for Business is supported along with embedding from OneDrive.

If you are using a personal Microsoft account (like @hotmail.com or @live.com), you can embed Office files merely by uploading them into your Sway and choosing embed.  The file will be stored in your OneDrive account and appear in your Sway.  Likewise, you can choose a file from the OneDrive source within Sway and choose to embed.

If you are using an Office 365 account (like from a school or business), the process is slightly different.  You must first upload the file to OneDriver for Business, then open it on the web from within OneDrive, and get the embed code from there.  Then open your Sway, insert an embed card, and paste the embed code in.

Hope this clears things up!

Jason
