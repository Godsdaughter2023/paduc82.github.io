steve rindsberg say:

I think I've found a fix.

There's a registry setting you can add to the Windows registry to make PowerPoint default to no compression.

It seems you can do the same by modifying the prefs file for PPT 2011/Mac.

I went to /Users/[myname]/Library/Preferences and opened com.microsoft.Powerpoint.plist in Property List Editor (there are no doubt other apps that'll work ok for editing plist files; maybe others can chime in with their suggestions).

I selected ROOT then clicked Add Item.

The name for the new item:

14\AutomaticPictureCompressionDefault

Type:  Number

Value: 0

Saved the plist file, fired up PPT and did my usual "Is it compressing?" test, and it's no longer compression.  The test:

Insert a reasonably high resolution image, size it to more or less fill the slide.

Copy the image, size it way down so it's tiny.

Save the file, close the file, reopen the file.

Choose the tiny image and enlarge it to match the large image.  If the little guy is now fuzzy, PPT is compressing.  If not, you've beaten the stupid out of it.  Congratulations.

ronrosenfeld say:

A simple UDF (User Defined Function) can do this:

To enter this User Defined Function (UDF), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this User Defined Function (UDF), enter a formula like 

=UniqueDayCount($A$2:$A$6,$B$2:$B$6)

in some cell.

Option Explicit
Function UniqueDayCount(rStart As Range, rEnd As Range) As Long
    Dim col As Collection
    Dim vStart As Variant, vEnd As Variant
    Dim I As Long, J As Long
   
vStart = rStart
vEnd = rEnd

Set col = New Collection
For I = 1 To UBound(vStart)
    For J = vStart(I, 1) To vEnd(I, 1)
        col.Add Item:=J, Key:=CStr(J)
    Next J
Next I
On Error GoTo 0

   
End Function

dale howard [mvp] say:

Bcossa2001 --

Yes.  Click the View tab to display the View ribbon.  In the Data section of the View ribbon, click the Outline pick list button and select the level of summary tasks you want to collapse.  For example, if you select the Outline Level 1 option, you will see only those summary tasks at the first level of indenture.  To automatically expand all summary tasks to show their subtasks, select the All Subtasks item on the Outline pick list.  Hope this helps.
