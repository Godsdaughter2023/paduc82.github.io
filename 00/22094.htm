Convert binary that is greater than 10 bit to decimal? - Microsoft Community

warren sensata say:

Convert binary that is greater than 10 bit to decimal?

Is there a VB program to convert binary that is greater than 10 bit to decimal?

chip pearson say:

Try the following code. The binary input BinString is assumed to contain only 0's and 1's.  It also assumes that the input is unsigned.

 

Function BinToDec2(BinString As String) As Double
    Dim L As Long
    Dim N As Long
    For N = Len(BinString) To 1 Step -1
        If Mid(BinString, Len(BinString) - N + 1, 1) = "1" Then
              L = L + (2 ^ (N - 1))
        End If
    Next N
    BinToDec2 = L
End Function
