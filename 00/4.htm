The cells probably contain text.  We can enter text into any cell, even if it has a numeric format (General, Number, etc).  Characters that look like numbers can be interpreted as text by Excel under some conditions.

[EDIT] And simply changing the format does not change the original interpretation of the contents.  We must "re-enter" the contents.  But that can be tedious.

To begin with, try selecting the column of cells, click on Text To Columns, Finish.  That might be sufficient to remedy the problem, especially if entering a "some weird formula" like =A1+1 might do the trick.

If TTC does not work, the problem might that the cells contain non-breaking spaces (HTML &nbsp).  Were they copy-and-pasted from a web page?

If that might be the case, the following might work.  In a parallel column, enter the following formula (assume A1 is one of the misbaved cells):

=--TRIM(SUBSTITUTE(A1,CHAR(160),""))

That eliminates all leading and trailing spaces and non-breaking spaces, and causes the remaining numeric text to be intrepreted as a number.

Then copy new column, and use paste-special-values to overwrite the original column (starting with A1).

SELECT CIFM_EMLHP_ADDR_502 AS EmailAddress,
