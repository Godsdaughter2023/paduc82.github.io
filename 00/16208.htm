fl kulchar say:

Picking out the FIRST Two Numbers of a Cell

Suppose in cell A1, I have....15-28

What formula do I use to extract:

                               1. the 15

                               2. the 28

Thank you.

FLKulchar

FLKulchar
hansv mvp say:

For the first number:

=VALUE(LEFT(A1,FIND("-",A1)-1))

For the second number:

=VALUE(MID(A1,FIND("-",A1)+1,100))

The number 100 in the formula is an arbitrary number greater than the expected length of the text in cell A1.

Try these...

A1 = 15-28

=--LEFT(A1,FIND("-",A1)-1)

=--MID(A1,FIND("-",A1)+1,10)

Assumes you want the values extracted as numeric numbers. Note that with this method any leading zeros will be stripped off. For example:

A1 = 01-05

The formula would strip off both leading zeros and return 1 and 5.

If you might have leading zeros and do need to retain them then just remove the double unary minus from the formulas:

=LEFT(A1,FIND("-",A1)-1)

=MID(A1,FIND("-",A1)+1,10)

Those formulas will return the result as TEXT.
