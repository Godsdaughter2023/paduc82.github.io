A value in ANY control on an open form can be referenced using the syntax:

how do use a spreadsheet that will both add and subtract figures - Microsoft Community

robynevans say:

how do use a spreadsheet that will both add and subtract figures

I am trying to make a spreadsheet to keep up with some items and when I use said items. I need to know how to keep a spreadsheet that will automatically add and subtract any figures that i put in
jllatham say:

The easiest way is probably to set up a sheet similar to a checkbook entry sheet; with one column for deposits (adding to your inventory), another for withdrawals (using some items) and a column for the balance (how many items you have left).

Let's use column A for 'deposits', column B for 'withdrawals' and column C for the Balance.

I figure that row 1 will have the labels in it, then row 2 has a kind of kick-start entry - nothing but the starting balance (inventory count) in cell C2.

Then in cell C3, put this formula:

=C2+A3-B3

which will give you the new balance.  Simply fill the formula down the sheet as far as you need to go; the cell addresses will adjust automatically as you fill it down and all you have to do after that is enter quantities of your items added to the inventory into column A and the quantity used into column B.

You could add other columns for dates and descriptions - perhaps using columns A and B for that, and using C for 'deposits', D for 'withdrawals' and E for the balance.

If your needs are more complex than this, such as needing to keep up more than just one item, then let us know.  Of course you could set up tracking for several items by just setting up additional worksheets in the same way for each item.

Contacts deleted from Windows Live Contacts after editing them in - Microsoft Community

josé oliver-didier say:

Contacts deleted from Windows Live Contacts after editing them in MyPhone

I currently have all my contacts and mail sync'ed using the Windows Live Mobile application. I recently started using MyPhone as well. I edited some contacts in MyPhone by adding their picture. To my suprise, all the contacts which I added a Picture where deleted from my Windows Live Contacts, but remained in my phone as a regular Outlook Contact. Is there a way to ensure that they are not deleted from Windows Live? Has anybody else stumbled upon this?
Jose Oliver - http://jose.spaces.live.com

richie weldon - msft say: Hi Jose,

Thank you for using Microsoft My Phone.  What it sounds like you're running into here is a limitation of the My Phone software as it does not allow for contact synchronization to be done both by Windows Live and My Phone as there's no integration between the two services.  If you added a Windows Live account to your mobile device after setting up the My Phone connection, you probably got into this state.

To resolve this issue what I suggest doing is that you allow only Windows Live to sync your contacts.  To do this, you will need to temporarily stop using My Phone, uninstall the application, reinstall it, and get back to using the service.  Don't worry, it's not that difficult to do and hopefully once you follow these steps your contacts will no longer be randomly deleted!

1. Open the My Phone application
2. Select Menu < Account < Stop Using Service
3. Go to the Start menu < Settings < System < Remove Programs
4. Find Microsoft My Phone and remove it.
5. Enter http://myphone.microsoft.com/install on your phone's web browser to download and and install the application.
6. Start using My Phone again!  You might need to enter in your Live ID credentials again as they might not have been saved.

Let me know how this works out for you if you do follow these steps.  Good luck!

Richie Weldon
