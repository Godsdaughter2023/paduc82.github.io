el_loco---01 say: WinAmp PRO

Nokia Music Manager

Both convert to eAAC+ a quick seach on Google may also yield other results.

eAAC+ is a very aggressive compression algorithm. So if you are into better quality audio then you may want to consider other encoding formats.


These are listed in terms of size (larger to smaller) and quality (better to worst) IMHO!

WAV is probably the best since it will not compress the PCM output from a CD. But 1 song in WAV is about 50-60 MB in size, and this is probably not supported by most phones.

Next option is a high bitrate MP3 (MPEG-1 Layer3) something 256 kbps or higher.

Next option is a high bitrate VBR MPEG-4 (Variable Bit Rate) like WMA VBR or MP3 VBR (maybe about 200-300 kbps)

Next option is a high bitrate WMA or MPEG-4 (mp4, m4a, aac, etc.) maybe 192 kbps.

Going higher bitrate in eAAC+ just defeats the purpose and benefits of small file size so you're better off with another format like AAC or M4A or MP3 or WMA

Hi Cubb33,


Data Execution Prevention (DEP) is a security feature that can help prevent damage to your computer from viruses and other security threats. Harmful programs can try to attack Windows by attempting to run (also known as execute) code from your computer's memory reserved for Windows and other authorized programs. These types of attacks can harm your programs and files.

DEP can help protect your computer by monitoring your programs to make sure that they use computer memory safely. If DEP notices a program on your computer using memory incorrectly, it closes the program and notifies you.

Microsoft does not recommend turning off the DEP security settings. But still if you require you will have to be an administrator to turn off the settings for DEP.

To uncheck the option of “Enable memory protection to help mitigate online attacks” you have to login as an administrator. You may also right click the Internet Explorer icon and select the option of “Run as administrator” and then try to uncheck the option.
