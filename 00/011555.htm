If Column A has data starting with IM and column N >2 days , fill the cell in Column N with red colour

hansv mvp say:

Select the cells in column N that you want to format conditionally. I'll assume that N1 is the active cell within the selection.

 

On the Home tab of the ribbon, click Conditional Formatting > Manage Rules...

Select any existing rule, then click Delete Rule...

Click New Rule...

Select 'Use a formula to determine which cells to format'.

Enter the formula

 

=OR(AND(LEFT(A1,2)="IM",N1>2),AND(LEFT(A1,2)="RF",N1>5))

 

If the active cell is in another row, change A1 and N1 accordingly.

Click Format...

Activate the Fill tab.

Select Red.

Click OK until you're back in Excel.

Hi,

Try this: =IFERROR(INDEX('Stock (x)'!$B$2:$AA$32,MATCH('Difference (x-y)'!$A2,'Stock (x)'!$A$2:$A$32,0),MATCH(B$1,'Stock (x)'!$B$1:$AA$1,0))-INDEX('Benchmark (y)'!$B$2:$O$32,MATCH('Difference (x-y)'!$A2,'Benchmark (y)'!$A$2:$A$32,0),MATCH(VLOOKUP(B$1,Sheet1!$A$1:$B$27,2,0),'Benchmark (y)'!$B$1:$O$1,0)),"")

Note: each sheet should have the same number of entries.

Stock (x) goes to AA but Benchmark (y) to O and not all matching names are listed in Sheet1.
