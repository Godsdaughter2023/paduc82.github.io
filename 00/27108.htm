ken sheridan say: The DCount function call as it stands will act independently of the grouping.  I think you'll need to add a further restriction to the DCount function call.  Something along these lines:

"[BL Start Date] < Date() And [Action Owner* Depts] = " & qryActionItemStatus.[Action Owner* Depts]

This assumes that the tblComplianceActionItems table also contains a column named [Action Owner* Depts], and the column is a number data type, i.e. a numeric foreign key.  If the column is of text data type it would be:

"[BL Start Date] < Date() And [Action Owner* Depts] = """ & qryActionItemStatus.[Action Owner* Depts] & """"

If the relevant column in tblComplianceActionItems has another name , then you'd change the above accordingly of course.

'Graham Mayor
