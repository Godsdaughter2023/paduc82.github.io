Highlight Merged Cells Excel 2010 - Microsoft Community

pmbo say:

Highlight Merged Cells Excel 2010

Is there a way to highlight Merged Cells in Excel 2010, Windows 7?  I have a workbook with thousands of lines and I want to find only the cells in a certain range (one column) that are merged.  Thanks.
jllatham say:

Code below should do it - change the reference to column B to the column you are wanting to examine.

 

One 'catch' - it needs something (formula or value) in the last row of the column that you want to examine/mark.  I'd just go to the last used row in the column and if it doesn't already have something in that cell, type in an x or . or your name, just anything to mark the end of the area to test.

 

The code:

Sub FindAndColorMerged()
'this code goes into a regular code module
'see this page for "how to" instructions.
' http://www.contextures.com/xlvba01.html#videoreg
'The sheet you want to work with must be selected
'when you execute this code
'
'you can run this macro from either the [View] or
'[Developer] tab.
'
  Const columnToTest = "B" ' change as required
  Dim testRange As Range
  Dim anyCell As Range
 
  Set testRange = Range(columnToTest & "1:" _
   & Range(columnToTest & Rows.Count).End(xlUp).Address)
  For Each anyCell In testRange
    If anyCell.MergeArea.Cells.Count > 1 Then
      'other color options:
      ' vbRed, vbYellow, vbBlue, vbGreen
      ' vbMagenta, vbCyan and vbWhite
      anyCell.MergeArea.Interior.Color = vbBlue
    End If
  Next
  'release assigned resources
  Set testRange = Nothing
End Sub
