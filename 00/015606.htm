Windows Defender

How do you program Windows Defender to scan automatically every day on Windows 10?
greginmich say:

How do you program Windows Defender to scan automatically every day on Windows 10?

Automatic Maintenance runs a Windows Defender Quick Scan every day when you’re not using your PC (without being scheduled), so there’s generally no need for scheduling a second scan:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_start/windows-10-how-do-i-set-windows-defenders-scan/f399bb87-37c9-4a74-bb35-1d7e3d3c2436

Also, the Microsoft method for scheduling a scan in Windows 10 actually doesn’t work in Windows 10 (I just retested this). To reproduce the problem, follow these steps:

  1. Turn your PC off at the end of the day, and then when you first power up the PC on the next day, open the Windows Defender interface and wait for the Automatic Maintenance scan to finish (this scan will be interrupted by any user activity).  

  2. Then schedule a scan by adding a trigger to the Windows Defender Scheduled Scan task as per the instructions in the Microsoft How-to. Schedule your scan to start in 10 minutes. Then wait for the scheduled scan to start – and when it does, just move your mouse and observe that the scan is immediately terminated – exactly like the Automatic Maintenance scan.

Most people won’t have any use at all for a redundant daily Quick Scan, but anyone who would be inclined to schedule a second Quick Scan would probably want that scan to start at the scheduled time and run with the normal priority – and that's not what you get with the Microsoft method. This thread has some guidelines for scheduling a normal Quick Scan, if that’s what you’re looking for:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/windows-defender-in-windows-10/6979df16-e16e-4ec1-9170-7081f54c22df

GreginMich

greginmich say:

About the only thing I can recommend at this point is a general troubleshooting protocol.

1. Remove any external sources of error:

  • Remove registry/system cleaning utilities. Revert any recent changes made by registry/system cleaning utilities, and then disable/remove those utilities. That would include CCleaner.

  • Remove previously installed third-party antivirus apps completely. Download and run the cleanup tool for every AV app that was ever installed on the PC or that came preinstalled:

http://answers.microsoft.com/en-us/protect/wiki/mse-protect_start/list-of-anti-malware-product-removal-tools/2bcb53f7-7ab4-4ef9-ab3a-6aebfa322f75

  • Remove undetected malware. Run some malware-removal tools to make sure that the system is clean:

Kaspersky Virus Removal Tool:

http://www.kaspersky.com/antivirus-removal-tool?form=1

Emsisoft Emergency Kit:

http://www.emsisoft.com/en/software/eek/

Malwarebytes Anti-Malware (free version only):

https://www.malwarebytes.org/antimalware/

Eset Online Scanner:

http://www.eset.com/us/online-scanner/

  • Repair the operating system. Run both of these system diagnostic/repair tools to confirm the integrity of your Windows 10 installation:

http://answers.microsoft.com/en-us/windows/wiki/windows_10-update/system-file-check-sfc-scan-and-repair-system-files/bc609315-da1f-4775-812c-695b60477a93

2. Repair Windows Defender’s internal sources of error or replace it with a free third-party AV app. Windows Defender does seem to have quite a few error states that can’t be resolved by removing external sources of error. The error messages associated with some of those error conditions are undecipherable; and some of the error conditions themselves appear to be intractable. So when you run into serious issues with scanning, updating, CPU usage, etc., and there isn’t any clear-cut solution in sight, then that might be the perfect time to try out one of the free third-party AV apps:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-to-stop-antimalware-service-executable-from/a3a57d31-4687-43c0-b274-261da7d89245

3. Recovery options: If you’d rather stick with Windows Defender and you just can’t seem to get it working properly, then you might want to consider one of the system-recovery options:

http://windows.microsoft.com/en-us/windows-10/windows-10-recovery-options
