First, make sure that you’re using the correct syntax for the command (with the full path for each folder enclosed in quotation marks, and separated by a comma). In order to avoid errors, you might want to try removing the exclusions one at a time, rather than with a string array.

 

If you’re sure that the Remove-MpPreference method isn’t removing the exclusions, then navigate to this registry location and confirm that the malware entries are still present:

 

Computer\HKEY_LOCAL_MACHINE\Software\Microsoft\Windows Defender\Exclusions\Paths

 

If the entries are there, then you’ll need to manually delete them – but you’ll need to take ownership of the Paths folder in order to do that.
