I've managed to disable 'Skype for Business', so I'm unwilling to start it again to find the version number. 

I found that the key problem is that when you exit Skype for Business, the window closes, but SfB continues to run as an unseen process in background. When you restart Windows, Windows restarts anything running at the time it was shutdown - hence re-initiates the SfB process and that opens the window again. SO the important thing for all those getting fed up with SFB popping up, is that when you turn off the auto-runs make sure the last initiated SfB background process has been closed down before you shut down or restart.
