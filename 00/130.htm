spere_49123 say:

After a number of attempts, I found a solution.

It requires using Mozilla Thunderbird (an excellent product that has a number of advantages over outlook. For the record, I have no legal/professional association with Thunderbird)

Here is the process:

  • Add the problematic IMAP email account to Thunderbird.
  • Create a new folder under inbox (let's call is 'sync_temp')
  • drag a couple test messages into the new folder ('sync_temp')
  • Go back to outlook and manually add the same folder at the same hierarchy position
    • i.e.: if in Thunderbird, you put 'sync_temp' directly under the 'inbox' do the same in outlook.
    • You may be thinking, this is IMAP, the folder should generate with the next server update in Outlook. Yes... Yes, it _Should_.
    • Once you add the folder, sync it using the 'Update Folder' button, under the 'Send/Receive' ribbon bar section (Outlook 2016).
    • If your test messages appear, all is good.
    • If not, make sure you:
      • copied some test messages into the 'sync_temp' folder while in Thunderbird
      • placed the 'sync_temp' folder in the same hierarchy position
      • if possible, check your webmail interface and see if the folder and messages are present as expected.
  • Go back to Thunderbird and drag all the non syncing messages from the affected folder into the new 'sync_temp' folder.
  • Close Thunderbird (likely not required)
  • Go back to Outlook and update the 'sync_temp' folder as before
    • sync it using the 'Update Folder' button, under the 'Send/Receive' ribbon bar section (Outlook 2016).
  • The non syncing messages should now appear in the 'sync_temp' folder. -this may take some time.
  • Once outlook is finished and the progress all shows green checkmarks in the 'show progress' window then move the emails back to the desired folder using drag and drop in Outlook.

At this point the messages should be restored.

If you have multiple folders which are affected, you may be able to drag the entire folder into the sync_temp folder and proceed as above. -- untested.

This may also help with the following:

IMAP not syncing properly

Outlook IMAP folders not updating

Outlook sync issues

Outlook folder disappeared

A couple other things to try:

  • Make sure there are no filters applied (Outlook 2016: 'View' on ribbon bar, 'View Settings', 'Filter', 'clear all', 'ok'
  • Update the folder list: (Outlook 2016: 'Folder' on ribbon bar, 'Update Folder List'
  • Confirm desired folder is in your subscribed folders -- search for "outlook IMAP Folders" or "Choose which IMAP account folders appear in Outlook" for details
  • More Advanced: Install and use OffCAT.

Best of Luck!
