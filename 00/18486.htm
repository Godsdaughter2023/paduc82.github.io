You can open a private or incognito browser using a keyboard shortcut, by right-clicking on the browser's taskbar icon, or using the browser's menu. 

If you use Chrome or a Chromium-based browser, the keyboard shortcut for an incognito or private browser session is Ctrl+Shift+N.

In Edge or FireFox, open a new In-private or private browser session using Ctrl+Shift+P.

jp ronse say:

Hi,

This is the floating point error

The value of 0.24 is after binary conversion 0.239999999999999,9911182158029987476766109466552734375.

So 60*0.23999 is 14.3999999999999. but rounded to 14.4.

If you replace B4-B5 by B4-B5-0 it will also show -1.77E-15.

14.4 is after binary conversion 14.400000000000 0,003552713678800500929355621337890625 but as only the first 15 meaning digits (14.4000000000000) are used the result of the substraction is always 0.
