    karl timmermans (outlook mvp 2012-2018) say:
    >>
    any advice to import my OST file into the Outlook?
    <<

    Hello Pesonna

    Short answer: You can't import an .ost file

    An .ost file is a mirror of what is contained on the server for the email account which created it. If you configure the email account as either IMAP or Exchange (depending on who your email provider is) - then Outlook will re-sync with the account.

    If this is an .ost file for an account that is no longer accessible for whatever reason, you would need to use a 3rd party OST2PST conversion utility and then import the PST file created by that process. Just do a search for "OST to PST conversion" and you will have no shortage of results to choose from. None of these will be free so be sure to download and test whatever program is of interest prior to any purchase to ensure there are no issues reading the OST file.

    Try adjusting this setting:

    1. In Word, click on FILE, then on Options.
    2. Click on Advanced, then scroll down to the Print section.
    3. Uncheck Scale content for A4 or 8.5 x 11" paper sizes. OK out.

    1. In Windows, search for Control Panel, then select it.

    3. Right-click on the printer and choose Printing preferences.
    4. Set the default paper size to A4. OK out.
