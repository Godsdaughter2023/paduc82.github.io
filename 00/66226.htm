mark ole say:

Hi Greg,

If your Microsoft Arc Mouse is connected but stopped working, it's possible that you may need to refresh the wireless connection. To do that, follow these steps:

  1. Select the Start button, then go to Settings > Devices > Bluetooth.
  2. Select Microsoft Arc Mouse.
  3. Select Remove device.
  4. Go to Start  , and then select Settings > Devices > Bluetooth & Other Devices > Add Bluetooth or Other Device > Bluetooth.
  5. Tap or click on “Microsoft Arc Mouse” in the list of Bluetooth devices.
  6. Tap or click on “Pair”. Microsoft Arc Mouse will begin to install.

If the Arc Mouse still doesn't work, force shutdown your Surface.

  1. Press and hold the power button on your Surface for 30 seconds and then release it.

  2. Press and hold the volume-up button and the power button at the same time for at least 15 seconds and then release both. 
    The screen may flash the Surface logo, but continue holding the buttons down for at least 15 seconds.

  3. After you release the buttons, wait 10 seconds.
  4. Press and release the power button to turn your Surface back on. You should see the Surface logo.

Run the Bluetooth troubleshooter if force shutdown didn't work.

  1. Select the Start  button, then select Settings  > Update & security  > Troubleshoot.
  2. Under Find and fix other problems, select Bluetooth > Run the troubleshooter.

Also, here are some things you can try that can help you resolve the issue.

  • Turn airplane mode on and off. On the taskbar, select action center > Airplane mode and leave it on for 10-15 seconds, then turn it off again and pair Microsoft Arc Mouse. If you don’t see Airplane mode when you open action center, select Expand for more options.
  • See if Microsoft Arc Mouse works with another device. This will help you further isolate the issue.
  • Install Surface and Windows updates. For more information on how you can install the latest updates, click here.

Let us know if you need further assistance.

How to connect PlayStation 3 to Windows media sharing on Windows 7 - Microsoft Community

boss man84 say:

How to connect PlayStation 3 to Windows media sharing on Windows 7

Original Title: Streaming

How Do i get windows to detect my ps3 so i can stream movies to it

mouneshwar r say:

Hi Boss Man84,

 

In order to connect Play Station 3 (PS3), you should know your computer MAC (Media Access Control) Address.

 

Step 1: To obtain the MAC address in Windows 7

a.    Click on Start; in the start search type “cmd” (without quotes) and hit Enter.

b.    Type the following command and press Enter to get the MAC address

          getmac

 

Step 2: Connect PS3 to Windows 7

a.    Open up the Network Sharing Center (enter Network Sharing Center into the search field on your Start Menu)

b.    In the sidebar click on the hyperlink “Change advanced sharing settings”:

c.    Turn on Network Discovery and File and Printer Sharing

d.    Then, click on “Choose Media Streaming Options”

e.    In the next page, you’ll find all unknown devices. Your PS3 will be listed as unknown device as well.

f.     Click on all “unknown devices” and look for the MAC address that we found in step (1).

g.    If you find the MAC address of the PS3, select “Allowed” from the dropdown field for that unknown device.

h.    Make sure that Windows 7 firewall is not blocking access to the PS3 device. Sometimes, you also have to forward ports to share files between Windows 7 and your PS3 or if you want to do more than just sharing files.

 

Note: If you’re using any third party firewall that comes with your anti-virus software, you need to change the port settings so that the firewall is not blocking access to the PS3 device. If you do not know how to change the settings, you can disable your security software. Once the connection is established and you’re able to stream movies, you need to re-enable your security software.

Hope this information is helpful and let me know if you need any further assistance.

dm51 say:

Hi,

I have a significant update:

Over the past couple of weeks I have been working with Microsoft Tech Support (who was interfacing with MS Engineering). Today, I spent some time with tech support testing out a possible fix for this issue. I am

The fix is a very simple registry setting. I have been playing with things since making the change and have not seen the issue anymore...

This clearly works with Surface Book 2. I don't know if it will work for other surface devices but if you have this registry entry, you can try making the change. The step to fix is as follows:

1. Open up Regedit (to edit registry)

2. Backup your registry before making the change - Once in Regedit, Right-Click on Computer, then Export to backup file, can put on desktop...

3. In the image below, it shows you where the registry entry is. You need to find this, locate the "FeatureTestControl" entry where the image shows you to find it. The value for FeatureTestControl should show: 0x00009240. Right-Click on this entry and modify binary, change the 40 to 50. Then exit registry.

Note, as indicated on left side, I had to select 0001. If for some reason you don't see FeatureTraceControl there, you could check 0000 just above 0001. Again, I saw it at entry 0000.

4. Once you have exited registry, Reboot system

Again, this worked for me and hopefully for you.

Let me know if this works for you.
