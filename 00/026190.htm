Hi Pastorbarreca,

It is not feasible to use OneDrive as primary storage. Primary storage is the main storage where the operating system resides, as well as the main area in a computer in which data is stored for quick access by the computer’s processor. All of the data you actively use is stored in the primary storage: operating system, applications, application data etc.

The synced folder from OneDrive is resides on the primary storage as well, thus it’s not feasible to set OneDrive as primary storage.

There are two ways to create such an expression in the Insert > Equation editor.

Method 1 is to use the tools on the Equation Tools | Design tab of the ribbon, which appears whenever your cursor is in an equation container. Click the Script button and select the template for an expression with both a superscript and a subscript:

That will create these boxes:

Fill the large box with a lower case sigma. Move the cursor to the upper right box and type the number 2; then move to the lower right box and type the letter X.

Move the cursor to the right end of the equation and continue with the rest of the expressions in the same manner.

Method 2 uses what Microsoft calls the Linear format to type the equation:

The caret (^) indicates that the following 2 will be a superscript, and the underscore indicates that the following X will be a subscript. When you press the space bar at the end of this expression, the Linear format will be converted to the so-called Professional format, which again is

Then continue with the rest of the expressions.

You don't have to use either method exclusively; you can even mix them in the same container.
