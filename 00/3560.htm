sheenali_msft say:

hi bernard,
when you upload the file to sharepoint, you're creating a new file in sharepoint, which is why the creation date is the date the item (document) is uploaded (created) to sharepoint.

for your reference:
get original date created/ document upload.

i understand that you would like to retain the creation date of the document when you upload it to sharepoint. as you said, the metadata still exists, but there is no way to extract it to sharepoint. we do not have an out-of-the-box option in the available columns.

a few workarounds you can try out:
1.    open a sharepoint library via explorer view. in explorer view, the windows os is able to extract the metadata from the document into the date column. as you can see in the screenshot below, the date created column is the date of when the item was created in sharepoint, and the date column is the date of when the document was created.


for your reference:
created date and owner or author in sp upload.

to view a library in file explorer, in the sharepoint online document library, click the library tab on the ribbon, and then click the open with explorer icon in the connect & export group.

if you don’t have the library tab and are using the new look and navigation experience for document libraries, click all documents in the upper-right corner and then click view in file explorer.

2.    another workaround includes the need to work with custom codes, possibly extracting the date created metadata from a document and putting it into a date/time column. if you would like to pursue this area, we’d suggest posting in the msdn sharepoint forum as they are experts in this field.

3.    another workaround: sharepoint search extracts metadata properties from the documents, and with the search you can show to the users the creation date. however, you need to modify the xslt that transforms the resulted xml into html.

if you would like to have this feature built into sharepoint online, we suggest you submit your idea via uservoice, this is the best way to influence our products and services to make it better for you and others.
