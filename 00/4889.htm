OK.  I was just trying to help with the suggested resolution attempt.  Normally I never suggest something without first trying to do some diagnosis.

In your case your problem symptom suggests that you have interference from an add-on.  Edge doesn't have add-ons so has less possibility of a crash from that cause.  Also, your symptom suggests the likelihood of a crash being detected and possibly being recorded in Reliability Monitor. 

To test the first case you can try opening IE this way.   Run... (e.g. press Win-R and enter:)

iexplore.exe  -nohome  -extoff

The objective is to have a window open and stay open.  If you want to try to explore what all you can do with it you will need to use the Stop command to get it out of its NoHome state.   E.g., press Alt-V p

Even if Reliability Monitor is detecting a crash its details are rather scant.  A better diagnostic for a crash is the related Report.WER file which you would have to find by matching timestamp and other path name information.  E.g. you would probably find what you were looking for in a path which contained iexplore

Once you have found the appropriate Report.WER it could contain clues for possible interferers in the list of Loaded Modules.

 

HTH

Robert

For what it's worth, I ran that test myself because it was an interesting question.

I confirm that Excel imported a text file with values as YYYYMMDD as text cells.

And I confimed that Excel imported another text file with values as MM/DD/YYYY as date cells.

May I recommend to the user to add this formula to the next column and roll that formula down?

=DATE(MID(A2,1,4),MID(A2,5,2),MID(A2,7,2))

where A2 is the first cell with the YYYYMMDD value.

To study how this works, please lookup the DATE() and MID() functions.

Your issue is, I suspect, that you're using your browser's Print function to print everything in the browser window, which includes the folder list.  If you want to print just a message, select the message by clicking once on it.  In the menu bar along the top, just to the right of the word "Categories" you'll see an ellipsis (three horizontal dots).  Click them and you'll see the word "Print" at the bottom of the pop-up menu that will appear.  Click it.  That should open a new, small browser window showing just the specific message you want to print and a Print dialogue should open so you can specify the print job's properties.
