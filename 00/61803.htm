    xbr poseidon say:

    Port Forwarding

    SMC8014WG-SHW

     

    Login Information

    Gateway - 192.168.0.1

    Username – cusadmin

    Password - password
    *The login password may be the wireless passphrase which depending on the install tech could be in all capital letters.*

     

    Port Forwarding

    1. Open a web browser.
    2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
    3. When prompted to login enter “cusadmin” as the username and “password” for password.
    4. From the menu on the left select NAT.
    5. From the NAT submenu select Port Forwarding.
    6. Click the Add button under the Customer Defined Service Table.
    7. In the Name field enter XBL or something that indicates Xbox Live.
    8. In the type drop down select TCP/UDP.
    9. In the LAN Server IP enter the IP address of the console.
    10. For the Remote IPs select Any.
    11. From Public IP Ports drop down select Single Port.
    12. In the Start Public Port box enter 3074.
    13. Leave End Public Port blank.
    14. In the Private Ports box enter 3074.
    15. Click the Apply button.

    how do i find my MAC address?


    Open a Run window (Windows Logo key+R), type cmd /k ipconfig /all (note the three spaces) and press Enter.

     

    The Physical Address is the MAC address, which is made up of 6 groups of 2 hexadecimal characters, e.g. 00-21-70-BE-C6-71.

     

    Ensure you use the Physical Address of the correct network adapter as several could be listed, e.g. Ethernet, Wireless and Bluetooth.
