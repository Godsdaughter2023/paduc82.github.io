It will certainly address your current issue, though ifg you want a VBA solution then the following should work for you. Select the folder containing the files from the dialog.,

Dim strPath As String
Dim strFilename As String
Dim strNewName As String
Dim NewName As String
Dim fDialog As FileDialog
    Set fDialog = Application.FileDialog(msoFileDialogFolderPicker)
    With fDialog
        .Title = "Select folder and click OK"
        .AllowMultiSelect = False
        .InitialView = msoFileDialogViewList
        If .Show <> -1 Then
            MsgBox "Cancelled By User", , _
                   "List Folder Contents"
            Exit Sub
        End If
        strPath = fDialog.SelectedItems.Item(1) & "\"
    End With
    ChDir strPath
    strFilename = Dir$(strPath & "*.*")
    While Len(strFilename) <> 0
        strNewName = StrConv(strFilename, vbProperCase)
        Name strFilename As strNewName
        strFilename = Dir$()
    Wend

rohn007 say:

To answer your stated question, you turn off autocorrect/spell checking by going to Review Tab > Spelling drop down > Set Proofing Language command.  In the Language dialog you can turn on "Don't check Spelling".  If you have a block of text selected, the change will only apply to it.

But that doesn't fix your autoformat problem.  It appears there is no user control over that feature.

The only thing I found you can do is use Undo, <CTL><Z>, command after Word tries to read your mind and create a numbered or bullet list for you.  So it goes something like this:

  1. type 1.
  2. spacebar
  3. <CTL><Z> to undo the change
  4. type your point, hit enter
  5. repeat steps 1 to 4

Very slow, but it gets what you want done.

Or you could type your dash followed by 3 or 4 spaces. Select the line minus 1 character. Then you can type your entry, hit enter, paste the copied text, type your next bullet point continue.

Personally, I find the second approach easier.
