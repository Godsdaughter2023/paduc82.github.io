Windows Defender not picking up win erx03 malware - Microsoft Community

dan babic say:

Windows Defender not picking up win erx03 malware

Windows defender not stopping this pop up that is impossible to get out of except with c+alt+del.  Quick scan doesn't find it, full scan does but will reinfect?  Pop up warns that files will be deleted in a number of seconds that count down if don't press update.  Any advice?

rob koch say:

It's not malware that's being found by the full scan, since this item is malvertising and not malware in the traditional sense of a virus.

Though I haven't seen this specific popup malvertisement myself, nearly all of these are simply a script that redirects the browser to the page containing the popup, none of which will typically be detected on-the-fly by anti-malware, since these items are contained in folders that aren't scanned in real-time.

The simple reason for this is that this would slow the operation of browser rendering too much, reducing the display of pages to a crawl.

A quick scan won't find these because the critical operating system locations it scans don't include the browser cache folders where these remnants are usually found.  Running a disk cleanup or similar temporary file cleanup tool will usually clear such leftovers out and allow the browser to start without the popups being present.

A full scan does just that, looking in every folder and file, with the item being detected likely either a script or simply the webpage popup itself found in the cache.  Including the specific detection in your next post would allow us to confirm this or whether there's something unrelated being detected instead, since I hadn't heard that any detection existed for this particular popup.

Rob
