  1. Right-Click on sound icon in the notification area and select ‘Playback devices’.See if the headphone is detected when plugged in.

  3. If it gets detected, select the headphone and select “Set Default”.

john john __ say: It means that SMART thinks that your hard disk is failing and that it needs to be replaced.  If the disk cannot be replaced immediately you should not take chances and backup the disk while you still can, if you don't you risk losing all the files on the disk!

SMART is not always 100% accurate so you should get a disk diagnostic from your hard disk manufacturer and run it to see if it concurs with the SMART utility.  If the disk diagnostic utility reports the disk as failing you should replace the disk in the shortest delay.
