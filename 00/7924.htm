echo s say:

Unfortunately there's not a combination animation-transition setting that tells PPT to move on to the next slide once all the animations have finished.

 

Theoretically, using an automatic transition of 00 seconds should transition the slide immediately following all animation. And it does work. But in my experience, a 00 auto transition tends to make the animations happen really fast!

Give this an appear animation and set it to start "after previous," with "previous" being the video. You'll be able to add a delay to that rectangle's animation so it forces the transition to delay a little if necessary so it doesn't move on quite so abruptly.
