Is there a workaround or maybe a plugin for this?

regards, Markus

yves dhondt say:

Assuming each line of code is its own paragraph, you could use numbered lists. All you would have to do is select the entire code section and hit the numbered list button and all lines should be numbered in one go.

suzanne s. barnhill say:
You can add line numbers to specific paragraphs only, without the use of section breaks. For the paragraphs you don't want numbered, check "Suppress line numbers" on the Line and Page Breaks tab of the Paragraph dialog. To make this easier, incorporate this formatting in all the styles except those used for the code snippets.
