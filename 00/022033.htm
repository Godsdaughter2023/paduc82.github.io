Step 5: Open Windows Explorer and go to the following path: %windir%system32oobe.

Step 6: Create a new folder named info, and open it.

Step 7: Create a new folder within info named backgrounds.

Step 8: Rename your desired wallpaper as backgroundDefault.jpg and place it inside the folder backgrounds (images must be less than 245KB in size).

Every word I type is identified as being incorrectly spelled (i.e. - Microsoft Community

mts4353 say:

Every word I type is identified as being incorrectly spelled (i.e. underlined in red)

I just opened Word for the first time on my Mac (OS X, v 10.8) and every Word I type is identified as being incorrectly spelled (i.e. underlined in red). I've been into the spell check preferences and a custom dictionary is checked. Does anyone know how to fix this problem?
jay freedman say:
It's possible that the text is marked as being a language different from the one you're writing. For example, if you write English text but it's marked as French or Spanish, all or almost all of the words will be marked as misspelled.

Select all the text, click the Review tab of the ribbon, click the Language button, and click the Set Proofing Language item. Click the correct language. It's probably best to uncheck "Detect language automatically", because that feature isn't reliable. Click OK.

If that removes all the red squiggly underlines, then you've found the cause. The permanent cure is to make the same change in your Normal.dotm template. Go back to the same dialog, verify that the settings are what you want, and click the Set As Default button. When you shut down Word, if you're prompted to save changes to the Normal template, say Yes.
