marshall barton say:

You have to use the subform control name to hide or show the data in the form or datasheet displayed in the subform control.   Note that the name of the subform control's name may or may not be the same as the subform control's SourceObject.

The code to hide the subform is simply:

   Me.[subform control name].Visible = False

and to show the subform:

   Me.[subform control name].Visible = True

One thing that bothers me about your question is that subform's with no data will only display the new record.  If you don't want to allow new records, then the subform will display absolutely nothing, which is nearly the same as invisible.  You can set the (sub)form's AllowAdditions property to No or if you want to do it using code in the main form:

   Me.[subform control name].Form.AllowAdditions = False      'or True

charles kenyon say:

Your question has nothing to do with the Normal Template. Documents from your older templates will open in compatibility mode until they are saved. You can open the templates in Word 2016 and use Save As to save them with the same name. This will put them in the latest format.

Do this with copies of your templates. Keep a backup.

To answer your question about the Normal template: Yes. Word 2016's template is a bit different, but it should have nothing to do with your existing .dotm or .dotx templates. I do not know how it is different, but I do know that each version of Word does create a different template.

stefan blom say:

What is stored in your templates, specifically? Macros? Building Blocks? Something else?

I'm asking because, usually, you should be able to work with old templates, and the documents created from those templates, in a new version of Word without issues.

There is an exception to the rule of backward compatibility, namely between Word 2013 and earlier versions. In Word 2013, the layout engine was redesigned which caused certain compatibility issues with older files and templates. See Changes in content layout between Word 2013 and earlier versions of Word at https://support.microsoft.com/sv-se/help/2740483/changes-in-content-layout-between-word-2013-and-earlier-versions-of-wo.
