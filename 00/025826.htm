ziasiddique16 say:

Excel workbook - how to save the individual worksheets as individual pdf files in one go.

Hi

Is there a VBA a code that can be run so that the individual sheets are automatically saved as pdf files in one go.  So if you had 6 worksheets in one workbooks with individual names.  Is there a code that will automatically save 6 pdf files in one go.

Zia

vijay a. verma say:
Use the below code in a Standard Module 

Sub ExportSheetsAsPDF()
    Dim Path As String
    Dim Ws As Worksheet
    Path = ActiveWorkbook.Path
    Application.DisplayAlerts = False
    For Each Ws In Worksheets
        Ws.ExportAsFixedFormat Type:=xlTypePDF, Filename:=Path & "\" & Ws.Name & ".pdf"
    Next Ws
    Application.DisplayAlerts = True
End Sub
