Yep, sorry for the lag.  Also - anyone else who still want's to take the survey is welcome to.

ronnie vernon say:

Hi Patricia

The Trustedinstaller.exe (Windows Module Installer) is a critical system file.

It has full permissions control of most Windows System files in, including Iexplore.exe (Internet Explorer).

Normal Users in Windows still have Read and Execute permissions. on most files, such as Iexplore.exe.

It is located in the C:\Windows\servicing folder.

You should not change the security permissions of the Trustedinstaller.
