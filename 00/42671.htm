tryx3 say:

Nancy,

If you created the system images onto the same external hard drive each time then the results are indeed merged together.  The image restore procedure identifies the most recent version and it initially offers that to you as the image to restore from.  I think you are saying that this is what happens so there is nothing wrong.

Windows has its own impenetrable rules for which bits of older system images it discards to avoid wasting space on your external hard drive and this is all based on the assumption that you will only ever want to restore the very latest one.  There are circumstances in which you can tell the image restore procedure to look for other images and it will show you any suitably intact older versions in addition to the latest one that it expected you to choose.  However, you have no control over what is still intact and the older ones do disappear quite quickly.  So, if you are considering your backup planning, you must always assume that only your latest image is ever going to be available to restore.  If, as you say, you are only ever seeing one then that is perfectly normal.

If, within Windows 10, you are using Control panel, [View by - Small icons, if necessary], Windows backup & restore [Windows 7] to create your system images then you are doing it correctly.  Ignore the comment about that in the first reply posted above as it was based on a misunderstanding of what you are doing.

You have been doing everything correctly.  You are not wasting your time making new system images frequently.  The newer an image is, the less updating you will need to do after it is restored.

There are manual procedures for making and for restoring multiple system images.  It requires manual management of the external hard drive contents [Windows does not support you at all in this] and it really needs a second computer to be available in case the subject computer is too corrupt to allow you to move the images around again ready for restoration.  I do this but I don't think many people do.  I can explain how to do it if you are interested.

I hope this has cleared things up for you.

A system image is an exact copy of a drive. By default, a system image includes the drives required for Windows to run. It also includes Windows and your system settings, programs, and files.
