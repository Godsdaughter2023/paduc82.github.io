The topic title speaks for itself. I tried backing up my desktop to my 3TB Seagate external hard drive disk and was given the following error message:

I've seen dozens of threads pertaining to this very same error code. Nonetheless, I felt it would be beneficial to create my own as it would allow me to go more into detail. If you need any more information, please ask. I'm eager to get this resolved so I can backup my system. 


The reason is that Seagate as well as other manufacturers configure their large (> 2TB) external drives to use 4096k logical sectors. That's not supported by Windows versions prior to Windows 8, as stated in MS KB2510009. Seagate doesn't provide a way to fix it so Windows Image Backup won't work (File Backup alone probably does).

WD on the other hand provides a tool (WD Quick Formatter) to make their large external drives Windows Vista/7 compatible.
