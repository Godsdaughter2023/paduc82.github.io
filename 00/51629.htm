There is no WMM version 5.1; the version you have on your desktop is actually the older Windows Movie Maker 2.1 for Windows XP.

The internal version number of the operating system on your desktop is 5.1 (that means it is Windows XP). If you look lower in the dialog, you'll see that the version of Windows Movie Maker is 2.1.

In any case, you be able to use that version to capture your Sony DV footage to create a DV-AVI file. That file type uses up about 13.5GB for each hour of footage. It will be standard definition 480p video.

I would then transfer it to your laptop for editing within Movie Maker 2012 (yes, I would upgrade). I assume you also have Windows DVD Maker on that computer (Windows 7) so you can make a Windows DVD.

Here's the solution!

1) Boot to Safe Mode with command prompt.

2) Check that short file name are enabled on the volume that contains the

programs.

fsutil 8dot3name set 0


3) Check that Program Files and Program Files (x86) directories have a

short name:

cd \

dir .* /x

4) Set the short name on both folders:

fsutil file setshortname "program files" PROGRA~1

fsutil file setshortname "program files (x86)" PROGRA~2

5) Check that Program Files and Program Files (x86) directories have a

short name:

cd \

dir .* /x

It should look like this:

03/13/2011  10:13 AM    <DIR>          PROGRA~1     Program Files

03/14/2011  10:21 AM    <DIR>          PROGRA~2     Program Files (x86)


6) Restart.
