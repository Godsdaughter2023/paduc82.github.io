bossdweeb say: You don't implement PhysX libraries into games. They either use it, or don't.

While games have been using cpu based physics for a while ( Half Life 2, and 'ragdoll' physics
 is an example) PhysX  is really just an advanced physics system, but there are not many games that make good use of  PhysX - yet.
Apparently Borderlands 2 has had a lot of attention with it's PhysX. Batman AA and AC are
another example of games that makes some use of PhysX.
The difference between normal physics and PhysX  up to this point has been quite small.

I'm not up on the latest regarding ATI as I've been using Nvidia for a long time, but PhysX
processing is linked to the Nvidia Cuda technology.
Some people say you can use your cpu (if it's powerful enough) for PhysX, but it is debatable
if they are actually getting PhysX or normal cpu based physics.

Nvidia bought the rights to PhysX from Agea and they had exclusive rights to it.
Again, I'm not up on the latest, but people were able to use a 'hybrid' system (install an Nvidia card in a machine with an ATI/AMD card) a while back with driver hacks,  but Nvidia driver changes stopped that.  .

  PhysX is most effective when using two Nvidia cards with one as a dedicated PhysX card.
I've not  bothered with this setup  (my motherboard for that machine also had only one PCI-e slot, so I couldn't experiment with it anyway), but a single Nvidia card will still do PhysX.
Batman AC's PhysX  is still quite effective with my GTX560ti.

You should be able to find more details on setting up a dedicated PhysX card if you do a search.

javan clark say:

Hey LBTBear,

 

 Sorry to see this issue pop up for you, here are some simple troubleshooting steps that may assist in solving this issue. Please let me know what your results are so I can proceed or close this thread.

  • 1. Connect and disconnect the Touch Cover a several times to see if it’s just a bad connection.
  • 2. Moisten a lint-free cloth (LCD or Glasses cleaning cloth works great) with a little rubbing alcohol.
  • 3. Rub the cloth over the contact points on the Touch Cover / Type Cover.
  • 4. Let the Touch Cover / Type Cover sit for a few minutes so the rubbing alcohol evaporates.
  • 5. Connect the Surface to the Touch Cover / Type Cover.
