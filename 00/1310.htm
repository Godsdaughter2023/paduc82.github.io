hansv mvp say:

Try this. Modify the file path but keep the trailing backslash \ at the end.

 

Sub ProcessTextFiles()
    ' Path - modify but keep trailing backslash
    Const strPath = "C:\TextFiles\"
    Dim strFile As String
    Dim f As Long
    Dim strLine As String
    Dim r As Long
    ' File number
    f = FreeFile
    ' Get first filename
    strFile = Dir(strPath & "*.txt")
    Do While strFile <> ""
    ' Open file
        Open strPath & strFile For Input As #f
        ' Read three lines, discard first two
        Line Input #f, strLine
        Line Input #f, strLine
        Line Input #f, strLine
        ' Close file
        Close #f
        ' Write info to next row
        r = r + 1
        Range("A" & r) = strFile
        Range("B" & r) = strLine
        ' Get next filename
        strFile = Dir
    Loop
End Sub

Getting iCloud Connection Error in Outlook 365 - Microsoft Community

lindaperry7 say:

Getting iCloud Connection Error in Outlook 365

I thought that I would post a note as to the steps I had to take to fix an iCloud Connection Error in Outlook 365.  My iCloud account was working perfectly in Outlook and then it disappeared...   I think it was around the time that iCloud implemented two-factor authentication.  There are a bunch of posts on the web related to setting up an "Application Password" for iCloud within the Microsoft account...  I did not do that...  mainly, because that step requires the setup of two-factor authentication on the Microsoft side (which I will do soon... but not now).

You need to download the iCloud for Windows app and install it.  I already had it installed...  My steps were:

  1. Make sure Outlook is closed
  2. Type "Settings" into Cortana search box
  3. Select Accounts>Email & App accounts
  4. Be sure to delete the iCloud account in question here if it exists (iCloud will manage the account...  don't do it here)
  5. Click on iCloud icon in the system tray (lower right corner of screen)
  6. Go to "Open iCloud Settings" when the iCloud window pops up
  7. Unselect checkbox next to "Mail, Contacts, Calendars, and Tasks"
  8. Hit Apply Button
  9. REBOOT PC (this is an important step!  Didn't work for me if I didn't)
  10. Re-log into PC
  11. Again, Click on iCloud icon in system tray
  12. SELECT checkbox next to "Mail, Contacts, Calendars, and Tasks"
  13. Hit Apply Button
  14. You can now close iCloud screen
  15. Open Outlook... you should now see your iCloud mailbox

Cheers!
