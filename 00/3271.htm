dale howard [mvp] say:

Genevieve --

If it were me, I would create three different project schedule, one for each scenario your Director wants.  Be sure to use either Fixed Units or Fixed Work to make sure Microsoft Project calculates the Duration based on the Units and Work values you enter for the resource assigned to each task.  Hope this helps.

geneviève_br say:

Hi Dale,

How do I create the three different project schedules?  And how will I be able to revert back to the first Schedule if it's the one decided? I am new to MS Project and self taught myself but I am probably missing a lot of the subtilities of the tool.

dale howard [mvp] say:

Genevieve --

Create the first project and assign all resources at a Units value of 20%.  Save the project with a name that indicates this is the first scenario.

Click File > Save As and save the project with a name that indicates this is the second scenario.  Apply the Task Usage view and then insert the Assignment Units column.  Change the Assignment Units value to 50% for every resource and then click the Save button.

Click File > Save and save the project with a name that indicates this is the third scenario.  In the Task Usage view, change the Assignment Units value to 100% for every resource and then click the Save button.

Show your Director the project schedule for all three projects and let him decide which schedule to use.  Delete the other two and use the schedule he selected for the project.

Hope this helps.
