hairyfool say:

A number of SSD manufacturers give you access to a drive duplicator software package (normally a stripped down Acronis True Image) which will allow you to transfer the running image from the old disk to the new.

However if you have successfully upgraded to Windows 10 you do not the need the key, Upgraded Windows 10 computers do not have one.

Open Start, click on Settings > Update and recovery > Activation. If it refers to having digital entitlement then your system (primarily the motherboard) is registered on the Microsoft servers as having upgraded to Windows 10.

Use this link to create the ISO of Windows 10 (use the install on another computer option), download and save drivers for your computer.

http://go.microsoft.com/fwlink/?LinkId=691209 

Remove the old disk, install the new and install Windows 10 as a clean load on the new disk. After setup is complete and you have connected to the internet Windows 10 should activate with that same digital entitlement, I find it happens fairly quickly but apparently it can take a few days.

Reinstall your programs, copy the user files from the old disk and you have a squeaky clean, super fast installation.

As you have made no alteration whatsoever to the old disk you have the fallback of reinstalling the disk if a problem arises.

When you choose regional and language settings for your computer, such as your default input language or keyboard layout, you can apply those settings to special accounts in Windows called reserved accounts.   1. open Region and Language.

Any user account created on the computer after the settings have been copied to the default user account will have these settings applied. Existing user accounts aren't affected.
