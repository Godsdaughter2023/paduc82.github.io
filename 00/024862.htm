vijay b say:

Hi,

Internet Explorer protected mode is a feature that makes it more difficult for malicious software to be installed on your computer.

In addition to helping protect your computer from malicious software, protected mode allows you to install wanted ActiveX controls or add-ons when you are logged in as an administrator.

Protected mode is turned on by default in the Internet, Local intranet, and Restricted sites zones and an icon appears on the status bar to let you know that it's running.

If you want it to turn on, follow these steps:

a.     Open Internet Explorer.

b.     From the Internet Explorer 8 command bar, choose Tools and then Internet Options.

c.     In the Internet Options window, click on the Security tab.

d.     Below the Security level for this zone area, and directly above the Custom level... and Default level buttons, check the Enable Protected Mode checkbox if it is uncheck.

e.     Click OK on the Internet Options window.

f.      If you're prompted with a Warning! dialog box, advising that The current security settings will put your computer at risk., click the OK button.

Close Internet Explorer and open again and check.

For more Information, refer this linkLApplies to Windows XP)

Hi,

I understand that you want to know how aliases work and I'm here to assist you. To start off, aliases are non existing Microsoft accounts that are associated with existing Microsoft accounts. The alias must not be an existing Microsoft account before you can add to your account. Therefore, if you have two existing accounts, you can't add the other as an alias for another existing Microsoft account.

 

To create an alias in your Microsoft account, follow the steps below:

 

1. Sign in to https://account.live.com.

2. Under Account aliases, click the Add or change aliases.

3. You will be taken to a page where you can see a list of aliases, you will only see your account as your primary alias if you don't have any aliases yet.

4. Now click Add alias. You will then be given two choices, either Create a new email address and add it as an alias or Add an existing email address as a Microsoft account alias.

5. If you choose Create a new email address and add it as an alias, you will create a new email address in your account as an alias but it doesn't make it an existing/independent Microsoft account because it is just an alias for your account.

6. If you choose Add an existing email address as a Microsoft account alias, you can add an account which exist on other 3rd party email providers like Gmail, Yahoo, etc.

 

Feel free to ask if you have more questions.

luke moraga say:
Starting with cell A1, conditional format formula is:

=MOD(SUM(IF(FREQUENCY($A$1:A1,$A$1:A1)>0,1)),2)=1

Pick 1st color of your choice. Second condition:

=MOD(SUM(IF(FREQUENCY($A$1:A1,$A$1:A1)>0,1)),2)=0

Formula will alternate between the two colors. Formula is actually counting unique number values, then the MOD function alternates between even and odd. For other unique value counting formulas, check out the XL help file.
