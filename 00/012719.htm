john korchok say:

Unfortunately, copying and pasting will only get you a low-res version of the photo. To get the original quality, try these steps:

  1. Open the document in Word and choose Office button>Save As>Other Formats.
  2. Create an empty folder.
  3. Change the Save as type: dropdown to Web Page (*.htm:*.html), give the file a name like Photo.htm, then click on Save.
  4. In Windows, open the folder you created, then open the Photo_files folder (the first part of the folder name will be the file name you chose).
  5. Change to Details view, so you can see which graphic file has the larger size. That's your photo.
