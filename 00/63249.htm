A security descriptor identifies the object's owner and can also contain the following access control list:  A discretionary access control list (DACL) that identifies the users and groups allowed or denied access to the object.

WHAT WORKED: System image backup and restore were only successful when the system was booted and run from an Acronis TrueImage recovery CD.  Running the operation from the recovery disk allowed me to backup and restore images of disks and partitions outside of Windows from a recovery boot disk.

SOLUTION 1: Include the proper permissions for "R NT AUTHORITY\SERVICE" in the discretionary access control list (DACL) as required by backup.

SOLUTION 2: Install and use third-party system imaging software that runs from a boot disk, outside of Windows. (recommended) 


SUMMARY:


End users rely on Windows based image backups to secure their data from loss.  Backups and restore can be performed using a third-party application running from a system repair disk outside of Windows.
