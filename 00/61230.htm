Beta testing doesn't entitle you to a free upgrade to Windows 10. Having a valid, activated Windows 7 or Windows 8 license does. That was the policy Microsoft set in place a few weeks before the initial release.

It depends on how Windows 10 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.

There are three separate issues here:

1-Multiple anti-malware applications

Broadly speaking, there are two categories of anti-malware applications. One type of anti-malware application scans your computer more or less continuously for the presence of malware. You should only have one of these.

Windows Defender is an example of an application that scans continuously. That's why Windows disables Defender when you install a third-party application that also does continuous scanning.

The other category performs a one-and-done scan, also known as an 'on demand' scan. Since it only scans your computer when you ask it do so, you can have more than one of these, although you should only run one scan at a time.

2-Removing the message about Cool Hand Luke

CHKDSK is a command we use in Command Prompt to scan, check and repair hard drives in the computers. Check disk disclaimer: While performing chkdsk on the hard drive, if any bad sectors are found on the hard drive then chkdsk tries to repair that sector, if any data available on that might be lost.
