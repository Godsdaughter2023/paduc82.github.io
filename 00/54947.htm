lemp say:

Except for very simple programs, an application installed in Windows is likely to have crucial parts scattered around, both in directories (folders) other than C:\Program Files and in the Registry.  Changing the name of the executable file and/or changing its location is just asking for trouble.

On the other hand, there really is little need for you to do this.  What you probably want is to be able to organize the Start > All Programs menu so that it appears in what you see as a logical order, i.e., an order that makes it easy for you to find what you're looking for.

The Start menu, including the All Programs portion of it, is displayed by merging the contents of two folders.  In XP, these are:

   C:\Documents and Settings\<user name>\Start Menu

   C:\Documents and Settings\All Users\Start Menu

Folders and shortcuts that are in the first of these folders are visible only in the Start menu of the particular user.  Folders and shortcuts in the All Users\Start Menu folder are, not surprisingly, visible in the Start menu of all users.

You can get to the first folder very easily, simply by right-clicking the Start button and selecting "Open."  If you are logged in as a user with "Computer administrator" privileges, when you right-click on the Start button you'll also see "Open All Users" -- otherwise, you'll have to navigate to it in Windows Explorer.

Whichever of the two folders above you open, you'll see at least a folder named Programs.  When you open the Programs folder, you'll see any number of folders and shortcuts.

It is the contents of these two Programs folders that is used to create the menu you see when you click Start > All Programs.

Each folder within a Programs folder represents represents a folder on the Start > All Programs menu.  The shortcuts are, of course, links to the executable files of your application programs.  You can organize these folders, sub-folders, sub-sub-folders, and shortcuts in any way that you like. 

To get things in alphabetical order, when you have any given folder open, click View > Arrange Icons by > Name.
