bob jones aka: cybertaz say:
There is no default Zoom for 'Office' - Office is made up of several programs & each one works differently. Since you posted to the Word forum I'm assuming that is your main concern :-)

For New Blank documents you first need to use File> Open to open the Normal.dotm template located in:
/Users/UserName/Library/Application Support/Microsoft/Office/User Templates

Change the Zoom to the preferred percentage, type a character, delete what you typed (don't use Undo), then Save & Close the template. From then on new documents should open at the Zoom you set.

The Zoom of existing documents won't be affected by that, though. They always open at the Zoom to which they were set when the document was last edited & saved. There is no setting to change the behavior - it would have to be handled by Apple Script or VBA. For my money it's just as effective to use the Shift+Command+Option+= keystroke to Zoom in or assign a keystroke to one of the other ViewZoom commands.

Regards,
Bob J.

What is the major difference between Exchange protocol and IMAP? - Microsoft Community

amytech say:

What is the major difference between Exchange protocol and IMAP?

Original title: Mail Protocol

can anyone tell me a major difference between Exchange protocol and IMAP?

a. user say:

Hello,

Thank you for posting the query on Microsoft Community. I am glad to assist you on this.

IMAP (Internet Message Access Protocol):

This means that you can receive and send messages without downloading them to your local computer. Instead, your messages are saved to the mail server, and you can use your email client and messages from anywhere.


Contactually communicates with your server to gather time stamps, your contact's email addresses, and track your email conversations.

If you are using IMAP and you want to set it up with Contactually, you will need to know:

  • the name of your IMAP server
  • your username
  • and your port settings

Exchange protocol:

This type of email server behaves more or less the same as an IMAP server. The primary difference is that Microsoft manages the mail server, and gives you Microsoft specific features through Outlook.  Same as IMAP, Contactually has no problem connecting with these types of email accounts. 


If you are using Exchange and you want to set it up with Contactually, you will need to know:

  • your Exchange server
  • and username

I hope this information helps. Please do let us know if you need any further assistance.

Inactivity

 

If your email account was inactive within 270 days, you will lose all your email messages due to inactivity even before you renamed the account. Hacked

How to hide row based on a cell value in excel 2010? - Microsoft Community

achillesangi say:

How to hide row based on a cell value in excel 2010?

How do I get Sheet1 to automatically hide row 20 and 21 when in cell C19 the value "No" is selected from a drop down menu?

Thank you

mike h.. say:

Hi,

Right click your sheet tab, view code and paste this code in on the right. Close VB editor and your done.

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Address = "$C$19" Then
    Application.EnableEvents = False
    If UCase(Target.Value) = "NO" Then
        Rows("20:21").Hidden = True
    Else
        Rows("20:21").Hidden = False
    End If
               
    Application.EnableEvents = True
End If
End Sub
