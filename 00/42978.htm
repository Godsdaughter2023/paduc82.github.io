robert aldwinckle on forums say:

Kindle app is still on the Surface and can be used from my wife's account, but I can't access it from my own user account.

I think it would be worthwhile experimenting with PowerShell  Add-AppxPackage.

For example, see what you get from this as an Administrator (i.e. in Administrator: Windows PowerShell)

Get-AppxPackage  -AllUsers  |?  Name  -match  "Kindle"  |%  {Add-AppxPackage  -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml" -WhatIf}

Note the  -Whatif.   You would just be trying to find the  AppxManifest  path.  So then in the actual user's (non-Administrator) PowerShell window you could try using that in an Add-AppxPackage command there.

There is actually an implicit example of this in the DOCS for that command (Winserver2012R2 is as close as we can get to documentation for W8.1 RT)

https://docs.microsoft.com/en-us/powershell/module/appx/add-appxpackage?view=winserver2012r2-ps 

<quote>

Add-AppxPackage -register $manifestpath -DisableDevelopmentMode
</quote>

The difference for you would be that instead of having the  $manifestpath  variable to use (or the previous command used to generate it) you would just replace it with the path that the -whatif found for you.

To check how well this may have worked I would try doing an  LS  command in the non-Administrator  PowerShell both before and after the above command.  Start in the user's directory.  E.g. enter:

cd ~

LS  -Re  -Fi  *Kindle*  -Fo  -EA:SilentlyContinue  |  ft  FullName  -AutoSize  -Wrap

So, before you used the Add-AppxPackage you would expect the above LS command not to find any Kindle files but after it you would.

BTW I tested this idea on W10 with an app which I had only loaded in one account as a loose proof of concept.  A big difference might be that I was connected to the Internet at the time AND the app is still in the Store.  Still I would be hopeful for your actual case.

Good luck

Robert Aldwinckle

WMV/NTSC - Microsoft Community

tompopo(2) say:

WMV/NTSC

Using Movie Maker to create a video in WMV will the video play on television using DVD player?

considerate_guy say:

Unfortunately the answer is "No".

The Windows Media Video format is understood by PCs, but is not understood by the standard DVD player. Video DVDs expect MPEG-2 video inside of .vob files with a specific directory structure and other support files for menu navigation, etc. If you want to put your video onto a video DVD, you'll need DVD authoring software to do that for you.

One caveat -- the video DVD standard was created in the late 1990s, prior to the wide adoption of HDTV. The maximum resolution of a DVD is standard-definition (NTSC) video rather than high-definition (ATSC) video. As a result, anything you send it will be "down-rezzed" to 480p maximum.

To activate your game on Games for Windows LIVE you first need to log in With your Microsoft Account (The Gamertag you signed into Xbox.com with). 

To sign in, load Fallout 3 and press the Home key to access the LIVE interface. Click 'sign in' and log in with your LIVE ID.  If all goes well your account will then be downloaded.  After this is done, you will then be prompted to insert your game serial.
