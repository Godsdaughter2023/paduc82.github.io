marshall barton say:

I don't see why you should use a temp table for the report.  For your example, a query can calculate all that using something like:

SELECT playername, ...,
               {formula for max lift} As MaxLift, ...,
               (SELECT Count(*) FROM tblplayers As X
                WHERE {formula for max lift using X} <= {formula for max lift using tblplayers})
              \ [Enter number of groups] + 1 As GroupLift,
              (SELECT Count(*) FROM tblplayers As X
               WHERE {formula for max squat using X} <= {formula for max squat using tblplayers})
              \ [Enter number of groups] + 1 As GroupSquat,
            . . .
FROM tblplayers

Note that in the above X is the subquery's table name alias so there is no need to change X to something else.

 

Without seeing your code, I can not comment on what you need to do to avoid the parameter error.  OTOH, you should not need to use any code with a query like the above.
