a. user say:

Hi,

 

Which pop up are you referring to?

 

Anytime thumbnails view is enabled or has been enabled in Microsoft Windows the hidden file thumbs.db is automatically created in the same directory as where the thumbnails have been viewed. This file contains the information required by Windows to display the thumbnails for each of the icons and will be placed in every folder thumbnails are viewed. Thumbs.db is much like it's name. It stores graphics, movie, and some document files then generates a preview of the folder contents using a thumbnail cache. This file can be safely deleted from any directory, however, it will be automatically recreated if thumbnails view is still enabled and you view that directory again. So, it is not related to faster thumbnail upload.
