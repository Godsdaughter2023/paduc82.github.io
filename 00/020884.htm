a. user say:

Hi Anto,

I understand how frustrating it is when things don’t work at times. I will be glad to assist you in this issue.

The INACCESSIBLE_BOOT_DEVICE bug occurs because of a boot device failure. During I/O system initialization, the boot device driver might have failed to initialize the boot device (typically a hard disk). This error always occurs while the system is starting.

Note: Remove any recently added hardware, especially hard disk drives or controllers, to see if the error is resolved. If the problematic hardware is a hard disk drive, the disk firmware version might be incompatible with your version of the Windows operating system. Contact the manufacturer for updates. If you removed another piece of hardware and the error is resolved, IRQ or I/O port conflicts likely exist. Reconfigure the new device according to the manufacturer's instructions.

I would suggest you to reinstall all your hardware drivers and keep it up to date.

Also If the issue persists, you may need to run a check disk on the drive and check.

What is chkdsk/r/f?

Chkdsk is an in-build system tool in DOS, OS/2 and Windows. It verifies the file system integrity of a volume and fixes logical file system errors.

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost. Make sure you have a backup of the important data on the drive.

  1. Press Windows key + X on the desktop screen of the computer.
  2.  Select Command Prompt (Admin)
  3. At a command prompt, type the following command, and then press ENTER chkdsk /f /r X:( drive letter)
  4. Press Y when you are prompted to check the disk the next time that the system restarts.

 

Try the above step and let us know if it helps. For further assistance feel free to post your queries in Microsoft forums. We will be glad to assist you.

gary's student say:

This may help.  Say we want to sum "chunks" of column A.  The chunks are between blank cells.

 

In B1 enter:

=SUM(A1:OFFSET(A1,MIN(IF(ISBLANK(A1:A100),ROW(A1:A100),""))-2,0))

In B2 enter:

=IF(A1<>"","",SUM(A2:OFFSET(A2,MIN(IF(ISBLANK(A2:A101),ROW(A2:A101),""))-2,0))) and copy down

Note these are array formulas that must be entered with CNTRL-SHFT-ENTER rather than just the ENTER key.

 

You will see curly braces {}
