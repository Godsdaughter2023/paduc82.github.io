suzanne s. barnhill say:

Exactly what steps are you using to insert a reference? What sort of reference? Are you referring to footnotes? If so, then the problem is probably that you have modified your normal style to be centered. Both the Footnote Text and the Footnote Separator are based on Normal, so they follow the Normal style. Instead of just formatting the footnotes manually, you should modify the Footnote Text style; you can do this as follows:

  1. In a footnote, right-click and choose Style...
  2. The Footnote Text style should be selected in the Style dialog. Click Modify...
  3. In the Modify Style dialog, click Format, then Paragraph (or you can use the alignment, spacing, etc. buttons in the Modify Style dialog itself).
  4. In the Paragraph dialog, select the desired settings.
  5. Click OK to close the Paragraph dialog, OK to close the Modify Style dialog, and Apply to close the Style dialog.

To modify the Footnote Separator, you will have to switch to Draft view, then click Show Notes on the References tab. This will open the footnote pane. In the dropdown (which shows All Footnotes), choose Footnote Separator, correct the alignment (Ctrl+L), and return to Print Layout view. For more on modifying note separators, see http://WordFAQs.mvps.org/NoteSeparators.htm.

legaljoanne say:

Microsoft Word Document from letter size to legal size w/out changing ...

How do I change a Microsoft Word document from Letter size to Legal size with out changing the format or orientation of my current document?

Every time I try to change the paper size it shifts the words and symbols in my document.

charles kenyon say:

You can't.

A legal size page is longer and holds more stuff.

When you change to letter size, the parts at the bottom scroll onto the next page.

If you had it formatted using empty paragraphs or manual page breaks, you will have lots of inconvenient space.

Turn on display of non-printing formatting characters.

  • Showing non-printing formatting marks in Microsoft Word
  • Basic Concepts of Microsoft Word - from Shauna Kelly
  • Basic Formatting in Microsoft Word
