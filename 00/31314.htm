earthlink email

I'm trying to use my Earthlink email address, but Earthlink only uses a POP server for incoming email. Windows RT only allows an IMAP server for incoming email. Is there any way around this? I already contacted Earthlink and they couldn't help. They said to contact Microsoft.

james vinyard say: You can do a work-a-round for that.  I do it with my local POP3 email account.

First thing is to log onto http://www.outlook.com (it probably will automatically) and click on the "gear" icon next to your account name on the upper right corner of the page.

Now, select "More Mail Settings".

Choose "Your Email Accounts".

Select "Add a send-and-recieve account.

Now, type in your information for the Earthlink account.  It will only setup a "receive" account and will not ask for a send mail server.  All emails will be sent out via Outlook.com.

Now, you can select how the emails are received and finish.

When back to the Outlook.com web page, your Earthlink emails should start showing up.  That will mean that it works.

Now, exit the web browser and launch the Mail app.  When asked for email account, select Outlook.com and your emails should flow through.

Do note that it may also NOT work.  POP3 can be quite different by ISP so the best you can do is try it out.

If it does not work, you 'CAN' use Outlook which is part of Windows 8.1 RT (if you upgraded to 8.1).  If you get a failure of that setup with the Mail app when selecting Outlook.com, you will need to upgrade your tablet to Windows 8.1.  You can do that by performing all updates on the tablet and when finished with those, open the Store and the 8.1 upgrade offer should appear there.

Thank you for providing the details. You can change the password of the hidden admin account by following the steps below.

  1. Press Windows key and type "cmd".
  2. Right-click Command Prompt and select Run as administrator.
  3. Type this command: net user "UserName" "NewPassword".
    Type "administrator" in the "Username" and type the password that you want in the "NewPassword".
  4. Retype the password that you've created and close the Command Prompt.

These steps are intended for local accounts. And since the hidden admin account is considered a local account, you can apply these steps.

a. user say:

There's no way to transfer installed programs. Plenty of organizations have tried - and charge for their products - but nothing works as well as reinstalling the software.

Moving most of your user data - Documents, Music, Pictures, Videos - is easy. Just copy the folders to a portable USB drive and paste them into the same locations in your new computer.

Moving other data can be trickier. Moving your email/contacts/calendar varies, depending on what email program you're using. Your bookmarks can usually be exported from your old browser and imported into the new browser. The procedure varies, again, depending on the browser.
