Taking each machine and the router one by one...

The router...

What is the exact model of the 2Wire device?

Make sure the routers DHCP server is enabled and that the number of address leases is adequate, ie. 10 or so for example.

Run the Internet Connectivity Evaluation Tool to see if any issues are reported.

I presume the wireless SSID on your 2Wire is a unique SSID and not the default. If its the default its possible your wireless desktop(s) and laptop(s) are trying to connect to a nearby device with the same SSID. You also should be broadcasting the SSID. If you suspect channel interference run inSSIDer to find an unused channel. Channels 1,6 and 11 are preferred.

Reset the router back to its factory defaults and try to reconnect wirelessly from both machines as a test.

The laptop...

How are the adapters setup, ie. to get a DHCP assigned IP address or did you assign a static IP address? With wired disabled, ie. no cable plugged in, and connected to your wireless router make sure you get a valid IP from the router. Go to Start > Run and type cmd /k ipconfig /all and verifiy you get a valid IP address and that the default gateway and DNS addresses match your routers IP address.

cbarnhorst say:

If you reinstalled the OEM copy of Vista Business on the new hard drive, as Bill says you cannot also use it in a vm.

You can only reuse your OEM copy of Vista Business if it is no longer in use on the computer. What operating system did you install on the new hard drive? Was it an upgrade copy or full license?

If you used an upgrade license for another edition of Windows to replace your copy of Business then the Vista Business license is still in use (to qualify for the upgrade) and you may not reuse it in your vm.

However if you replaced Vista Business with a full license copy of another edition then you are able to use the OEM copy in a vm, but only on the same computer. You need to use the COA product key from the label affixed to the computer. The self-activating product key in the OEM copy cannot find the SLIC table on the host's motherboard. You need to use phone activation from within the vm. Click Start in Vista Business and type "slui 4". Do not respond to the audio prompts until it asks you if you want to talk to an agent. Explain that you have completely replaced the OEM copy with a full license copy of the new Windows and now wish to use the OEM copy in a virtual machine on the same computer. The agent will then give you the activation string. Remember that the vm is not transferrable to any other comptuer.

peterleonard1 say:

Andrew E, your post is slightly misleading, as anything run from the command prompt, even with admin elevation still runs under Windows so files in use will still not be defragged.

You would need a 3rd party app that runs before Windows loads to properly defrag the HDD.

Also Windows defrag will not touch files over a certain size (hence the suggestion to remove the page file), so if your HDD has lots of large files saved in non contiguous sectors defragging will have a minimal effect.

Also even though a disk clean-up is good practice to remove unnecessary files it is not a required step as those files will be defragmented along with any other files defrag can work with.

The main thing the original poster needs to be aware of is that running any defrag app while the Optane acceleration is active could lead to file system errors and possibly break his Windows installation.

The reason being that the Optane driver expects to find the files its caching to be in a particular place on the HDD.

I suggest you read the Optane guides  and information on the Intel website and forums before suggesting the original poster tries to defrag using any method.
