creating a circular text box - Microsoft Community

veronna say:

creating a circular text box

How do you create a circular text box?
daniel g. grau say:
A text box is, by definition, a box. However, Word lets you create other shapes and add text to them.
  • Create a circle (use the Ribbon's Home tab and click on the Oval icon, shown as a blue circle below):
           
  • Press Shift as you drag the mouse to force equal proportions
  • Right-Click (or Control-Click) on the circle and select Add Text (this option will show as Edit Text once text has been added)
  • To modify the shape (for instance, to get rid of the circle's color), Right-Click/Control-Click on the circle and choose Format Shape (or select the circle, then choose Format > Shape in the main menu bar).
  • You can add text effects, also through the Right-Click/Control-Click contextual menu.
