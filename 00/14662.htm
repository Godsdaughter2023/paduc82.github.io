andreas killer say:
Access the visible cells only.

Unfortunately CountIf doesn't support areas, so we have to count in each area.

Andreas.

Option Explicit

Sub Test()
  Dim A_visible As Range
  Dim E_visible As Range
  Dim C As Range, Area As Range

  With Sheets("groupplacements")
    Set A_visible = SpecialCells(.Range("A2", .Range("A" & Rows.Count).End(xlUp)), _
      xlCellTypeVisible)
  End With
  If A_visible Is Nothing Then Exit Sub

  With Sheets("data")
    Set E_visible = SpecialCells(.Range("E2", .Range("E" & Rows.Count).End(xlUp)), _
      xlCellTypeVisible)
  End With
  If E_visible Is Nothing Then Exit Sub

  For Each C In A_visible
    C.Offset(, 1) = 0
    For Each Area In E_visible.Areas
      C.Offset(, 1) = C.Offset(, 1) + WorksheetFunction.CountIf(Area, C)
    Next
  Next
End Sub

Function SpecialCells(ByVal R As Range, ByVal Typ As XlCellType, _
    Optional ByVal Value As XlSpecialCellsValue = &H17) As Range
  'Avoid the SpecialCells-BUG to return all cells from the current region
  On Error Resume Next
  Select Case Typ
    Case xlCellTypeConstants, xlCellTypeFormulas
      Set SpecialCells = Intersect(R, R.SpecialCells(Typ, Value))
    Case Else
      Set SpecialCells = Intersect(R, R.SpecialCells(Typ))
  End Select
End Function

Save to PDF change to order the page

bob jones aka: cybertaz say:

It depends a bit on which method you use to generate the PDF. The key point is that generating a PDF is a print process regardless of method used. You need to change a setting in either [but not both] of 2 places;

  • In Word> Preferences> Print, change the status of the box for Reverse print order. This also can be done in the Print dialog by selecting Copies & Pages> Microsoft Word - Options, or
  • Many printers print from back to front, so in the Print dialog you can go to Copies & Pages> Paper Handling then change the Page Order: as necessary.

If using the File> Save As - PDF method use the first option listed above. If using the File> Print - PDF method you can use either of the 2 options.
