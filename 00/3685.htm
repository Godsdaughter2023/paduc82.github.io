don varnau say:
I have the same problem.  It just started after an automatic update. There has to be an easier fix than reformatting. 

Are you running IE11 on Windows 7?

Work through these procedures. Check after each step.

1. Internet Options> General tab> Browsing History> Delete> uncheck "Preserve Favorites website data" then delete Temporary Internet Files and Download History. No reason, yet, to delete cookies ,history,  passwords, etc.

2. Open IE from Start> Run, type in iexplore.exe -extoff (space after exe) then hit Enter. Try a download or two. If the problem is gone troubleshoot add-ons to find the troublesome one.

(Old article but the procedure hasn't changed)

Troubleshooting and Internet Explorer’s (No Add-ons) Mode:
http://blogs.msdn.com/b/ie/archive/2006/07/25/678113

3. With the warning that a reset will delete AutoComplete data, such as passwords (do you have those backed up?) and you will have to reinstall or re-enable browser add-ons, use the Reset IE Settings button in Internet Options> Advanced.

Windows Vista Help- Reset Internet Explorer settings:
