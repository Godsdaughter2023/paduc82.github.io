It is explainable. BT was always intended as a wire replacement, The wire that goes to a speaker or headphone, the cable that goes to a keyboard or mouse, the serial cable that used to go to a modem. BT uses various Profiles to support the various wires, like A2DP (advanced audio distribution profile) for stereo stuff, HID (human interface device) for keyboards, mice, etc. Most of these allow for a single connection. That is you can attach one speaker or headset, not two. Some things allow multiple instances of a profile for a keyboard and a mouse, for instance.

Controlling the phone audio from the Band likely works over AVRCP (audio video remote control protocol) and likely it supports only one instance. When you connect a BT speaker or headset via A2DP, I bet it automatically snags the AVRCP so you can affect volume, tracks, etc. from that speaker or headset. I think I have an old BT receiver thing that doesn't do AVRCP. I wonder if that would work, indicating it might be possible to separate the profiles. Have to hunt it up and see. Other things that are often stuck together like headset/handsfree and A2DP can be split up if the devices support it, so you could have headset/handsfree go to a car visor BT device and the music be routed to your stereo. It's technically possible.

This is a really good thought though, so maybe get this into the feedback app so that people can support it.

As Dave said, Compatibility Mode simply is an informational message telling you that the file was created in an older version of Word.

Compatibility mode means that the file structure does not support all of the features in your current version of Word.These unsupported features will be disabled in the ribbon, they will appear "greyed out" and you won't be able to use them

You have to "convert" the file structure into the current version. You can do that using File menu > Info command > Convert button. This starts the conversion process. Just follow the prompts.

If it is happening for all new, blank, documents, that means your NORMAL.DOTM file is an older version. You will have to either open it and convert it, or simply rename it and let Word generate a new one.

"Enable Editing" is a separate issue, to do with files marked as being downloaded from the internet.

stefan blom say:
my computer is suddenly double spacing every letter I write to friends and family, how do I reset it to single spacing?

You may have to change not only the line spacing but also the spacing before/after. Both spacing before and spacing after should be set to zero, if you are in the habit of pressing Enter twice to create a new paragraph.

Note, however, that this is not what you should be doing in a word processor. Instead, let Word break the lines manually, and only press Enter (once!) when you do need a new paragraph. See http://shaunakelly.com/word/concepts/rulesfortyping.html.
