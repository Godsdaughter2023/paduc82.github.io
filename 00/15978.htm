gil deg say:

Hi David,

Please be informed that creating PST files does not include folder structure but there is a workaround for that.

Kindly follow the steps below on how to copy current Mailbox folder structure to a new PST file.

STEP 1

The trick to copy the folder structure of your current mailbox is to archive the mailbox with a date so far in the past that none of your emails are actually being archived. This will create an archive file which contains all of your current folders but will all be empty as well.

1. Start a manual archive process;

    File-> section: Info-> button: Tools-> Clean Up Old Items.

    Or

    File-> section: Info-> button: Cleanup Tools-> Archive.

2. Select the root of the mailbox or folder set that you would like to copy. If you would only like to copy the subfolders of the Inbox folder, then you can leave the selection on the Inbox folder.

3. Set the Archive items older than field to a date so far in the past that you are certain no emails will get moved.

4. Set the location of your Archive file to the location where you like to store your PST files. For example; D:\Documents\<your name>\Outlook\mymail.pst

5. Press OK to execute the Archive function. When Archive is done, only empty folders should be created in the new PST file.

STEP 2

Now that you’ve created a new pst-file with the folder structure of your current mailbox, you’ll need to set the newly created pst-file as your new default data file so that all new emails will be delivered to it.

1. Open the Data Files Management dialog.

    File-> section: Info-> button: Account Settings-> Account Settings-> tab: Data Files

2. Select the newly created PST file.

3. Press the button: Set as Default.

4. Restart Outlook when prompted to do so.

Note: To change the display name of the “Archive” PST file, re-open the Data Files Management dialog as indicated above and then double click on the PST file. You can now adjust the “Name” field for the PST file.

Let us know if you need further assistance.

Regards,

Mark

Location for a user's default macros in Word? - Microsoft Community

mmo17 say:

Location for a user's default macros in Word?

Is there a startup or template file in Words that allows one to store "personal macros", i.e. macros that one wants to have always available while editing documents (I am seeking the equivalent to the PERSONAL.XLSB file in Excel, where I can store macros that then are available in all sheets that I open).

hansv mvp say:

Word uses the default template Normal.dotm for this purpose. Macros stored in Normal.dotm will be available in all documents. Apart from that, Normal.dotm also serves as the default document template, and it also stores the user's formatted AutoCorrect entries.

 

Just like Personal.xlsb, Normal.dotm doesn't exist in a clean new installation, but it will be created when needed.

The location for Normal.dotm is

 

C:\Users\<username>\AppData\Roaming\Microsoft\Templates

 

AppData is a hidden folder, so you must display hidden files and folders in order to see it in Windows Explorer.

graham mayor say:
You may also store macros in macro enabled templates stored in the Word startup folder and they too will be available to all documents; and you can store macros specific to certain documents in the template from which those documents are created and the macros will always be available to those documents as long as the template is present and 'visible' to the document.
