If everything works fine after a clean boot, you can infer that some third party services are causing the problem. Proceed with the remaining steps to pin-point out the third party service. After you figure out the problem causing program, you may have to update or install a higher version of the program, if you seldom use that software you can consider uninstalling it.

Chkdsk is a system tool which verifies the file system integrity and fixes system errors. Here is the steps:

  1. Press Windows+X.
  2. Click on Command Prompt (Admin).
  3. Type the command Chkdsk /f/r.
  4. Hit Enter.
