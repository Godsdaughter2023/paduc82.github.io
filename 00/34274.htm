In Microsoft Answers community we try to assist each other and keep the standard up. Personally each time when I try to answer a question, I also improve my knowledge. So, there is no such thing as elementary question, maybe there are other people asking the same questions for answers.

I will try to answer your questions:

1. CreatingSystem Image means you are creating a copy of your current computer known as Mirror. System Images created by Windows Backup are individual copies and subsequent backups are also separate copies withlatest data. If you have lost or deleted old copies, any one of the available images will be able to restore your computer. For example, if you have created 3 images in last 3 months (Jan, Feb, MArch). You want to restore your computer using Feb image, after restoration all you current data will be gone and the computer will be the same as you had back in Feb. System Image is not the same as incremental backup because it does not depend on previous images.

2.File and folder backup is incremental. When you usefile backup (not system image) in your computer with schedule or manually in a location; the each time the backup runs, it creates media file (new instances) with a date and every instance of this backup location will be needed to restore files. So, keep the backup media (HP/USB/DVD) safe and do not delete any data in the backup location. For example, if you have created a backup and ran 3 schedule in last 3 weeks then that particular backup will need all the data in the backup location during restoration.

3. You will need all prior backups (except individual System Image) in order to successfully restore files and folders to the latest.

4. Regardless of mass storage type, you should not delete any backup files created by Windows Backup (except System Image). To delete system image, you just need to delete the image file from the media location. Be sure to create a new one before you delete unwanted image.
