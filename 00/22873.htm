In Word, VBA code is only available

  • from within the document that contains the code, or
  • from within a document based on the template that contains the code, or
  • if the code is in a global template (add-in)

No MS does not recognize that a computer died. How would the activation system know? The dead system can't send a notification ...


It all depends on the type of license you have.  If you have a retail (FPP) license, then you can transfer the installation to a new computer. You will have to do a phone activation. When you phone in, don't reply to the initial prompts (you'll just get the same error). Wait until you are given the option of talking to a live "service" rep. Explain that you are transferring a license from a dead computer. They should allow the activation to complete.

If you have a Product Key Card, PKC, "1-computer" online download, factory installed & activated OEM license, then you are SOL. Those types of licenses are "disposable". They are permanently tied to the first hardware they are installed on. When the computer dies, or is sold / given to someone else the Office license goes with it.

In order to force old Yammer groups to integrated with office 365 groups to have Live stream feature, please follow below steps.

  • Access Yammer from Global admin account > Add the same account as an admin of that Yammer group.
  • Click on the top right gear icon on the Yammer group page > Toggle the group Type from public to private listed or vice versa.
