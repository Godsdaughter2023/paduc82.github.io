Applying a SmartArt Style simultaneously to multiple slides in - Microsoft Community

deford say:

Applying a SmartArt Style simultaneously to multiple slides in PowerPoint 2010

I've created a number of slides, using various SmartArt Graphics or Layouts without applying any particular SmartArt Style to each one. So they all look pretty plain. Now I want to go back and make all these graphics look nice by applying SmartArt Styles like "Polished," "Inset," "Cartoon," "Intense Effect," etc. But I would like to apply the same SmartArt Style consistently to all the slides. It seems that the only way I can do this is to go back to edit each slide and apply the SmartArt Style to each one individually rather than being able to apply the same SmartArt Style to all the slides at once.

Is there some way to set a default SmartArt Style that will be applied to all slides when first creating them? Or some other way to apply a SmartArt Style to multiple slides?

shyam pillai say:

Hi,

I don't know of way to do this automatically thru the UI but you can do this with a macro. Paste this into a code module and edit the code to the desired style name * and then run it. This code works only in PowerPoint 2010

'------------------------------------

Option Explicit
Sub ApplySmartArtStyle()
Dim oSld As Slide
Dim oShp As Shape
Dim StyleIndex As Integer

StyleIndex = GetQuickStyle("Powder") '*
If StyleIndex > 0 Then
    For Each oSld In ActivePresentation.Slides
        For Each oShp In oSld.Shapes
            If oShp.Type = msoPlaceholder Then
                If oShp.PlaceholderFormat.ContainedType = msoSmartArt Then
                    oShp.SmartArt.QuickStyle = Application.SmartArtQuickStyles(StyleIndex)
                End If
            End If
            If oShp.Type = msoSmartArt Then
                oShp.SmartArt.QuickStyle = Application.SmartArtQuickStyles(StyleIndex)
            End If
        Next
    Next
Else
    MsgBox "Could not locate desired style", vbExclamation
End If
End Sub

Function GetQuickStyle(Name As String) As Integer
Dim I As Integer
For I = 1 To Application.SmartArtQuickStyles.Count
    If LCase(Application.SmartArtQuickStyles(I).Name) = LCase(Name) Then
        GetQuickStyle = I
        Exit Function
    End If
Next
End Function
