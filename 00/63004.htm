lemp say:

Service Pack 1 for Windows 7 was released to the public on February 22, 2011 -- more than six years ago.  To characterize as "surprising" your report that you "recently" received a  "warning" to update to Windows 7 SP1 would be a vast understatement.

Timing issues aside, you should have a backup plan that includes saving copies of your data to external media.  Using a separate partition on the same physical hard drive is just playing with fire.

If you are truly considering installing Service Pack 1 to an existing installation of Windows 7 Professional (and not referring to upgrading to Windows 10), then the operation should not mess with your data files (whether or not you move them to your second partition).  Nonetheless, it is always prudent to make a backup -- to external media -- before undertaking any major change to your system.  See https://support.microsoft.com/en-us/help/15090/windows-7-install-service-pack-1-sp1

To answer your specific question, installation of Service Pack 1 for Windows 7 does not have to delete any of your personal data files.

One final thing:  a "restore point" is NOT the same as backing up your system.  System Restore primarily saves the "system" state, which means the Windows Registry and important system files.  System Restore specifically does NOT include any user data files.  See, for example, https://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/ (which covers Windows 7, 8, and 10, despite the name of the link).

lemp say:

Windows Backup in Windows 7 has two options:  you can create a "System Image" or you can back up specific files.  If you elect the second option, you can either let Windows decide what files will be backed up or you can manually select the files.

Windows 7 Backup and Restore isn't designed to use online storage, although there are some tricks that may work:  https://superuser.com/questions/276535/is-there-an-online-storage-service-that-works-with-windows-7-backup-and-restore (I have not tried this).

A system image copies everything on the specified partition or computer.  The catch is that you can't restore individual files or folders from the image; you have to restore all or nothing.

https://www.howtogeek.com/howto/4241/how-to-create-a-system-image-in-windows-7/

The second option will let you back up your personal data files.
