lemp say:

As the name suggests, "svchost.exe" is a "container" or "host" for programs that are written as DLL files rather than EXE files.

It is normal for there to be several instances of svchost.exe running at the same time.

Quite a large number of tasks start up when you first power on your computer, and some of these tasks are hosted by an instance of svchost.exe.  For example, most antivirus programs do some sort of scan of your computer at startup.  Many applications will attempt to connect to the Internet and check for updates when you first start up your computer.

You need not worry about instances of svchost.exe using RAM at startup.  More than likely, the amount of RAM in use will drop after a short time.  And even if they don't go down quickly, unless you start to run out of memory -- highly unlikely with 8 GB of RAM -- there's no need to panic.

If you want to look into this in more detail, open Task Manager and do the following.

  • Go to the "Processes" tab and make sure that there is a column headed "PID"  -- if you don't see this column, click "View" then click "Select Columns" and check the box next to "PID (Process Identifier)"
  • Go to the "Services" tab and click "PID" in the header row to sort the entries by PID
  • Go back to the "Processes" tab, right-click on an instance of svchost.exe that you're interested in and select "Go to service(s)"
  • This will take you to the Services tab with all of the services hosted by that instance of svchost.exe highlighted.

For example, in the first screenshot below, I've select the instance of svchost.exe with PID 1224.  Right-clicking and selecting "Go to service(s)" results in the second screenshot.

Once you know what service(s) are associated with the instance of svchost.exe that you're interested in, you can do some Googling to find out what those services do.

Hi Tina, here is how to revert to Windows 7 . . .

The Windows 7 product key should be o a Sticker on the bottom casing of your PC/Laptop If it is a laptop, or it might be under the battery - you will need it!

Backup all your files

Click this link to download the Windows 7 ISO file
https://www.microsoft.com/en-us/software-downlo...

Click this link to download a small Microsoft utility (Windows USB/DVD Download Tool) to create bootable Installation Media from the ISO
https://www.microsoft.com/en-us/download/detail...

Create the bootable Installation media on a DVD or USB Flash Drive

Insert the bootable media into the PC

Start your PC, on the manufacturers logo screen press the appropriate button to get into BIOS Setup (Usually F2 or Del key)

Press F10 to save settings and restart

Windows 7 will begin installing

When your system reboots, go into BIOS and set your hard drive as top of the Boot Order

Do not remove the Installation media until you reach the Desktop

That's It!
