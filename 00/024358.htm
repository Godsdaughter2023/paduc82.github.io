IF E5 is less then 200, VALUE = 0.00

teylyn say:

The "sliding scale" you describe can be achieved with this formula:

=LOOKUP(E5,{0,200,1000},{0,0.5,1})

You can plug this into your existing formula like this:

=IF(D5="YES",(IF(E5>199,0.25,0))+(IF(F5>199,0.25,0))+(IF(G5>199,0.25,0)),(LOOKUP(E5,{0,200,1000},{0,0.5,1}))+(IF(F5>999,1,0))+(IF(G5>999,1,0)))

Help! TIA!

stefan blom say:

As long as you haven't closed the document, you can always Undo (Ctrl+Z), repeatedly if necessary, until you get rid of the modifications that you don't want to keep.

On the other hand, if you have closed the document, you will have to access an earlier backup.

Note that backups aren't made automatically, unless you save to OneDrive, whose Version History feature can be used. For those of us who choose not to save to OneDrive routinely, File History can be used in Windows 10 (see https://support.microsoft.com/en-us/help/17143/windows-10-back-up-your-files).
