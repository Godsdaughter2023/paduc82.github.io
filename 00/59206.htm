frederik long say:
Now, which is best, restore from system image or reload windows in total?
Restoring from an image will set your machine back to the point when you created the image. The process takes between 30 and 60 minutes.

Reloading Windows will make your machine "as new", forcing you to reload all applications and recreating all settings. This is usually a very time-consuming affair, taking at least half a day plus another week until everything is exactly the way it used to be.

To determine when you should back up your files you need to ask yourself this questions: "How much would it hurt if my disk crashed one hour from now?"

per normann skoglund say:

Mouse is locked to the Scroll bar when moved from it when the mouse button is still pressed in windows10

When I move the cursor away from the scroll bar still pressing down the button - the button is kept active.

This also happens in any text area when the cursor is moved still pressing down the touch button - this is very annoying.

Can I change this?

a. user say:

Hello Per,

 

Thank you for contacting Microsoft Community.

 

I understand the inconvenience you have experienced.

 

Is it a laptop or desktop?

What is the make and model of your device?

Is this happening on any particular Web browser?

 

I guess the problem might be with the drivers or might be with the Mouse setting. It is a touchpad, then do check and ensure that the sensitivity and other options are set appropriately. You to try the below mentioned troubleshooting steps and see if they helps.

 

1.   Open run command by pressing Windows key + R and type “devmgmt.msc” (without quotes) and hit Enter.

2.    Spot and expand the “Mice and other pointing devices” by clicking + sign in front of Mouse and other pointing devices.

3.    Right-click the device driver listed below Mouse and other pointing devices and select “Update Driver Software”. When it prompts to choose an option How do you want to search driver, then go with Search automatically for updated driver software. If you have a driver available on your PC, then select Browse my computer for driver software, then select the appropriate location, where the driver located on your PC for updating it. (Ensure that PC is connected to Internet while performing this step)

4.    If it doesn't fix the issue then right-click the same device driver listed below Mouse and other pointing devices and select Uninstall.

5.    Click OK to confirm removal of the device.

6.    Restart the system. Windows will detect and automatically install the driver.

 

Also, go to Manufacturer website and see if any latest updated mouse driver’s available for your device there. If any try to replace the existing driver with the same to see if it fix the issue. Also check installing the old drivers (I.e. Windows 7 or Windows 8.1) from Manufacturer website to see if they fix the issue.

 

Keep us posted if you require further assistance.
