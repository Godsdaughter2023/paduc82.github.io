lava k say:

Hi MDonaghy,

 

Welcome to Microsoft Answers Community Site.

 

To Disable or Enable ToolTip Popups for Items in Windows Vista, follow the steps below:

 

1. In the Start Search Box, type regedit

2. Scroll down HKEY_CURRENT_USER > Software > Microsoft > Windows > CurrentVersion > Explorer > Advanced

3. You will see a list of entries. Locate FolderContentsInfoTip, StartButtonBalloonTip, ShowInfoTip. Click on modify and set dword value to 00000000 for each of the entry (to undo the same set the value to 00000001)

4. Scroll down HKEY_LOCAL_MACHINE> Software > Microsoft > Windows > CurrentVersion > Explorer > Advanced

5. Repeat Step 3

6. Exit and reboot.

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. frederik long say:

When dealing with file or folder names that have embedded spaces then you MUST surround the names with double quotes. This is standard stuff for all console commands, including robocopy.

'--------------------------------------------------
'Move all files in the source folder to the target 
'folder unless a file exists in the target folder
'with the same name but different size.
'19.7.2017 FNL
'--------------------------------------------------
Set oFSO = CreateObject("Scripting.Filesystemobject")
If WScript.Arguments.Count < 2 Then Error "Please specify a Source and a Destination folder"
sSource = WScript.Arguments(0)
sTarget=WScript.Arguments(1)
If Not oFSO.FolderExists(sSource) Then Error "Cannot find the folder """ & sSource & """"
If Not oFSO.FolderExists(sTarget) Then Error "Cannot find the folder """ & sTarget & """"
If Right(sSource, 1) <> "\" Then sSource = sSource & "\"
If Right(sTarget, 1) <> "\" Then sTarget = sTarget & "\"

For Each oFile In oFSO.GetFolder(sSource).Files
    bCopy = True
    If oFSO.FileExists(sTarget & oFile.Name) Then
        If oFile.Size = oFSO.GetFile(sTarget & oFile.Name).Size _
        Then oFSO.DeleteFile sTarget & oFile.Name _
        Else bCopy = False
    End If
    If bCopy Then oFSO.MoveFile(oFile), sTarget
Next

Sub Error(sMsg)
    MsgBox sMsg
    WScript.Quit 9
End Sub
