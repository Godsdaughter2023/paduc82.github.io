zack barresse say:

Excel Online is not the same thing as desktop Excel. It's difficult to know the difference. This question prompted me to do some online searching, and I basically found nothing describing the differences. Here is the difference between them both.

Excel [desktop]

The de facto standard of Excel. This is for PC only and contains all features. There are different flavors of this version of the app, which is subscription-based, and may dictate certain features not being available in all subscription levels. For example Power Pivot is only available in the ProPlus subscription model. (It is also available with volume licensing or if purchasing standalone Excel from retail.) 

Excel Online

This is available through OneDrive. If you have a Microsoft account, which used to be if you had a Hotmail account, or a Live account, this was your Microsoft account. These days you can create a Microsoft account with any valid email address, and you're not limited to only having a Hotmail account. These are free to get. Once logged in, clicking the button in the top left (a bunch of dots) will bring up your app pane, and you can select Excel. This version of Excel *only* works through your browser, so an internet connection is obviously required. There is a lot of fidelity between this version and the industry standard of [desktop] Excel. It's easier for Microsoft to push updates to Excel Online because the app is installed on their servers, and not downloaded to the client's machine. It's a different way of thinking about software. This enables Microsoft to push updates at a much faster cadence. This version of Excel is free for everyone to use, without exception, so long as you have a Microsoft account, which only takes a valid email address as stated before.

Excel Online can only have ribbon/UI customizations through the new Javascript API's. This means traditional add-ins or ribbon customizations, like those done through VBA or the ribbon/UI, will not persist to Excel Online. The new API's developed by Microsoft persist across most versions of Excel, including desktop & Online.

In summary, customizing the UI across all versions of Excel isn't very easy. If you want to persist to Excel Online you have to create a Javascript Add-In. If you want to take ribbon customizations from one desktop Excel to another, that is as simple as exporting your customizations and importing it on the target Excel. I know this can be confusing, it gets rather complicated when you walk outside of desktop Excel.

If you need additional information, please post back. Have a great day!
