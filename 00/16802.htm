Align by decimal in excel - Microsoft Community

oksana_las say:

Align by decimal in excel

I am using excel for Mac and am trying to align different values in a column by decimal.  Each number has two digits to the right of the decimal, however some values are currency ($) while others are percentages (%).  I have a varying number of digits to the left of the decimal.  Is there a way to align the numbers and not lose the $ and % signs?  Each time I try, excel will align the decimals, but drops the signs showing if the value is currency or percentage.  It should look like this:

  $10.49

      4.67%

    67.01%

$572.39

Thanks for your help!

bob jones aka: cybertaz say:

Excel does not provide Decimal alignment for dealing with mixed formats or for a variable number of decimal places. I'm guessing that you're losing the formatting because you're attempting to force the alignment by typing spaces directly into the cells. That won't work because spaces are not valid characters for numerical values, so they cause the entry to be treated as text - Number formats cna't be applied to text strings.

You can approximate what you want by creating a Custom Number format for the cells containing Currency values. First, select a cell containing a Currency value. Go to Format> Cells> Number, then click on Custom. Select the normal Currency format - $#,##0.00 - then in the Type field click to the right of the formatting characters. Press the space bar 3 times, then click OK.

Depending on the font formatting being used you may need to change the number of spaces in the Custom format. Also, the result still may not provide perfectly sharp alignment on the decimal point due to variable character width of the digits, but it should be close.

jay freedman say:
The dialog that offers an assortment of lines, including artwork, was replaced by the plain-line dialog in Word 2007.

The fact that there haven't been any other questions about the issue since then, I think, pretty much proves that Microsoft guessed right -- almost no one misses the old feature. That also explains why several MVPs had no idea what you were asking about.

There's one other way to get what you want, although with more effort: Go to the Borders & Shading dialog and click the Horizontal Line button in the bottom left corner. Unlike the Horizontal Line command on the menu, this one opens the Insert Picture dialog. If you have previously scoured the Internet and downloaded a collection of pictures of lines or other separators, you can choose one of those.
