trekdozer say:

Reinstall Windows Preinstalled On Computer


OEM computer manufacturers are required to provide at least one way for you to reinstall Windows if you buy a computer with Windows preinstalled - your computer should have included either a restore partition on your hard drive, or Windows install/restore disks. ﻿Most computer manufacturers also have a feature to make your own install disks.

If the restore partition is still there and was not removed, it will reinstall Windows 7. There is usually a prompt when you first start the computer telling you what key to press to start the restore﻿ - check your manual if you're not sure how to. If you have install disks - insert them, restart and boot from them to start the reinstall process. Reinstalling will remove any personal files on the computer, back them up first.


If you don't have the restore partition or install disks:

Contact the OEM computer manufacturer and ask them to send you an install disk for your computer - they may or may not do it. If they agree to send you one they may charge a small fee. Contact information for the manufacturer of your computer can be found here:

http://support.microsoft.com/gp/oemphone/







Product Key - Where To Find Product Key

If you bought a computer with Windows preinstalled there should be a Certificate of Authenticity (COA) sticker somewhere on your computer, on some laptops it’s in the battery bay. It will have a 25 character alphanumeric code on it - that is your product key. If it's not on your computer look in your paperwork or on any CD/DVD cases that were included with the computer.

This would suggest that you have a connection to your switch or router, but there is an issue with the connectivity beyond that.

  1. What happens if you run the Network Troubleshooting?
    (Right click it and then select "Troubleshoot Problems")
  2. Are other devices on the same network able to access internet?
  3. Try to first ping your gateway, and if that works, try to ping one of your DNS servers, for example whatever shows up in ipconfig /all under DNS servers. (If this is the same as your default gateway, try to ping google's DNS service at 8.8.8.8)

If you cannot reach your default gateway, this would suggest some internal network or DHCP problem.

If no devices are able to access the Internet (e.g. your DNS servers) but CAN reach your default gateway, then the problem is with your router or internet connectivity. This might be for example wrong DHCP settings, or your internet connectivity is experiencing problems.

If only this device is affected (e.g. everything else on the network is fine) then it could be a cable or network adapter problem, or something preventing the machine from getting the correct settings for the network adapter.

malkeleah say:

It isn't a change from how previous MS operating systems were provided. Legally if you sell a computer with a Windows operating system preinstalled, you must provide a method to restore the computer to factory condition or to reinstall Windows. This can be done by providing a recovery image on a partition on the hard drive, or with physical recovery disc images, or with the actual operating system installation disc. It has been this way since Windows 95 (although the imaging options weren't used then of course). I don't remember what the rules were for Win 3.x. When we buy or receive donated computers from corporations, the hard drives are nearly always wiped. If they aren't, we wipe them and install a legal copy of Windows.

What did change in Vista and Win7 is KMS and MAK requirements:

http://www.microsoft.com/licensing/existing-customers/product-activation-faq.aspx

I don't know how to tell whether you have a volume license or not from reading the Product Key on the COA sticker on the computers but certainly the MS Volume Licensing support people will be able to answer you.

ralf_g say:

I made some research on the web to find out which software uses this extention, so much came up that it is overwhelming.  It is possible that this can be a  Corel Photo-Paint extension?

 

That is correct. You'll need to convert them to another more suitable format. CPT files can be multi-layered so TIF would be the more similar format and best for editing in anyway. Otherwise BMP, PNG or JPG if a flat format or smaller file sizes are what you want.

I don't know of any picture file viewers which can open .CPT files, unless some current Corel apps will (xnview and irfanview do not). You can of course convert the files with PhotoPaint itself. If you can't find the original Windows version that was used to create them there is a downloadable free version of PhotoPaint which runs in Linux (PhotoPaint 9). You might have to search the web for it though, when Corel abandons a product they do it rather thoroughly.
