a. user say:

There's no 'legacy mode' in a BIOS, however there is a legacy BIOS mode in a UEFI. A better name for it is "legacy - that is, BIOS - mode."

If your computer shipped with UEFI/GPT it's probably compatible with Windows 10, but that's not a guarantee. To know for sure whether your hardware is compatible with Windows 10, you have to make a list of all the hardware in the computer and then find out whether each component is compatible. You do this by checking with each component's manufacturer.

Here's what you will discover:

If the manufacturer supports a hardware component on Windows 10, that means the manufacturer has written a driver for Windows 10 and tested the component on Windows 10.

If the manufacturer does not support the hardware on Windows 10, that means it's not writing a driver or testing the hardware on Windows 10. However, an earlier driver - say a Windows 8 or Windows 7 driver - may still be compatible with Windows 10. That means you can use the hardware on Windows 10 for now, but eventually it will stop working properly or at all.

BTW - the legacy mode in a UEFI doesn't have anything to do with legacy hardware. It means you can boot the computer with an operating system that was written for the older BIOS booting system instead of the newer UEFI.
