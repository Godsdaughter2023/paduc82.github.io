Hello!

Robotic sound usually is produced by a low connection, or maybe you have a good connection but other app is using it, so Skype tries to adapt to the situation.

  • First of all, make sure you are using the latest Skype Version.
  • Make sure no other apps or devices are making a heavy use of your connection. If you have to share files during the call, try to do this after the call is finished. If you can, use a wired connection rather than WiFi.
  • Check Skype Status website.
  • Please try to do a test call, does it work better? Does it only happen with one contact? The problem may be in the other end.
  • Also, try to do a test call using Skype for web.
  • Check the drivers in your computer.
  • Try to make the test call using a different device in the same network to isolate if it's the network, or the device.
  • Reinstall Skype in your computer.

You can find this and many other things to try in this Skype FAQ.

kapil arya [directly] say: Yes, recovery partition is basically present for that purpose.

Here's some other info about recovery partition:

As the name suggests, the partition is solely helpful under recovery scenarios. The data stored on it may contain recovery image tools such as winre.wim. The main benefit of having recovery partition is that the recovery data is on the hard-drive itself rather than on CD/DVD and thus it ensures fast recovery process.
Microsoft suggests to add a separate partition for Windows RE tools and at the end of hard-drive. This will facilitate Windows to automatically and logically expand the recovery partition as per requirement, in future instances.

There are some universal rules which govern recovery partition size:

It should be at-least of 300MB in size

When it is less than 500MB, it have 50MB free space

If it’s size is in between 500MB to 1GB, it should have 320MB free space

For recovery partition sized more than 1GB, it must have at-least 1GB free space

The type ID used by this partition is DE94BBA4-06D1-4D40-A16A-BFD50179D6AC.

lemp say:
Are diacritic (accent) marks available on a standard U.S. QWERTY keyboard?

Yes ... but how you get them depends on the application you're using.

In almost any Windows application, you can use the Character Map accessory (Start > All Programs > Accessories > Character Map).  In addition to allowing you to insert many different characters by copy/paste, this accessory also tells you a method to insert some characters using the numeric key pad (a pain, unless you have a separate numeric keypad) by pressing ALT plus a 4-number code using the numeric key pad.  For example, ® is produced by ALT+0174 and â by ALT+0226.

Some applications have an "insert characters and symbols" function that allows you to insert characters with diacritic marks.

If you are using a Microsoft Office program, e.g., Word, you can use a combination of the CTRL key and another key just before you press the letter key. SeeKeyboard Shortcuts for International Characters .  I know this method works for Word 2003; I assume it works for other versions as well.
