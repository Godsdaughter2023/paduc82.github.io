Split from this thread.

bob jones aka: cybertaz say:
Split from this thread.

Excel 2011 custom toolbars are automatically stored in a special file called Microsoft Excel Toolbars. It is located in the Library folder for your User Account:

~/Users/UserName/Library/Application Support/Microsoft/Office/Preferences/Office 2011

However, that isn't a file that you can open to retrieve a toolbar... Attempting to open the file simply will launch Excel. Any toolbars stored there are loaded when you launch the program. That doesn't mean that they will be displayed. They need to be turned on by going to:

  • View>Toolbars to select them individually, or
  • View> Toolbars> Customize Toolbars & Menus to check the respective boxes

If the toolbars are not listed in those locations it means that there are no custom toolbars stored in the file or that the file is either missing or damaged

I solved the problem:

1. Going to People (menu down)

2. Selecting the contact group

3. Right click and selecting Sharing Permissions

4. Assigning Read and Write permissions

After that, I added the account to the AddressBook app (contacts) and I got all contacts.
