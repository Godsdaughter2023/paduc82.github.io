greg carmack - mvp 2010-2020 say:
Hi Alex. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

You can change the Account name as to how it will appear on your sign-in screen in the online Microsoft Account, however it will not change the User folder name which is a security feature. So you'll be stuck with the first five letters of the MS Account email as I am if you don't establish a Local Account first during install or Setup.

I have elevated this issue for some months now and only heard back recently that they have no plans to change it. If you want to express your opinion on this to Microsoft use the Feedback Hub app in Start Menu where developers are tasked to process consumer feedback. They will not even see it here. This is a tech forums staffed mostly by volunteers trying to help others with your problems.

What you can do is create a new Local Admin Account as shown here: http://www.howtogeek.com/226540/how-to-create-a...

Move your files over, set it up, test all programs run from it or reinstall them, then when ready you can delete the old account in Settings > Accounts > Family and Other Users.

But don't be surprised because while the account will go away the old User folder will still be there until Windows is reinstalled

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
