jeeped say:
As a formula, the DATE() function will construct the date using the YEAR() and DAY() functions to reference the original year and day, then hardcode or use a cell reference to specify the month.

=DATE(YEAR(A1),<type a month or cell containing a number>,DAY(A1))

                

A macro would use DateSerial() for the same process.

Sub new_month()
    Dim rng As Range, m As Long
    m = 12  'new month
    For Each rng In Selection
        rng = DateSerial(Year(rng), m, Day(rng))
    Next rng
End Sub

Select one or more cells with dates and run the macro. Change the month in the second line.

It may be important to note that changing 31-Jan-2013 to month 2 will result in 02-Mar-2013 since you cannot have a 31-Feb-2013.

Whenever I do a clean install of Windows, I always begin by using a Local Windows account only, which produces a matching users folder name.  If I use my Microsoft Account to login with during setup, Windows uses the first 5 characters of my email address as the user folder name, which ends up being longer in my case.
