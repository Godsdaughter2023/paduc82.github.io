The problem here is, there are lots of failures and rollbacks. If you have the SSU installed, reboot and then the other updates, based on my test on few OEM laptops (Traditional HDD), refresh to preinstalled OS without any update, and then update, it takes around 1 hour to 2 hours to finish. SSD will be finished within an hour. Of course, it depends on the network speed, consider there are 90+ hotfixes need to be downloaded, and it may takes additional hours to download these hotfixes.

The expected process is

1. Fresh installed Windows 8

2. Update the Windows update agent, and reboot, take maybe 10 minutes

3. Install the SSU

4. Download 90+ hotfixes.. Download time depends on the network speed. Install time depends on the computer CPU/Disk, should be less than 3 hours in normal situation.

5. Reboot and update again, it's now less than 10 hotfixes. Install and reboot.

If the total process takes longer than 5 hours, check the Windows update, view update history to see if there any failure. Failure means extra time in the rollback and the retry.
