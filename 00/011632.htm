hey

I want to make a distribution group in Office 365. the question is : how could I import the users to the distribution group I want to make from the on-premises security group? is that even possible?

rudy xu msft say:

Hello Ammar,

You can import bulk users to distribution groups in Office 365 via csv files. However, you cannot export csv files from security groups and import it to distribution groups directly. Because the format of the content information is different.

If you want to do it, there is a workaround. Here are the steps:

1. Export members from AD security groups:

PS C:\windows\system32> Get-ADGroupMember -Identity local_sg1 | select name | Export-Csv -Path "c:\11\sg.csv" -NoTypeInformation

Note:  local_sg1 is my security group name and the name is one attribute I want to export. You can edit it based on your situation.

2. You need to edit the csv file. You need to add three attributes information before importing the file to distribution group.

DisplayName, Alias, PrimarySmtpAddress
Erick Chu, Erick, erick@contoso.onmicrosoft.com

 

3. After you edit the csv file, you can run the Exchange Online PowerShell below:

Import-Csv “C:\11\sg.csv” | foreach {Add-DistributionGroupMember -Identity “DGName” -Member $_.PrimarySmtpAddress}

If you have any further questions when doing this, please feel free to let me know. Meantime, you can just create a csv file and import it to distribution group directly.


Regards,

Rudy

Sub Copy_Every_Sheet_To_New_Workbook()
'Working in 97-2010
    Dim FileExtStr As String
    Dim FileFormatNum As Long
    Dim Sourcewb As Workbook
    Dim Destwb As Workbook
    Dim sh As Worksheet
    Dim DateString As String
    Dim FolderName As String

    With Application
        .ScreenUpdating = False
        .EnableEvents = False
        .Calculation = xlCalculationManual
    End With

    'Copy every sheet from the workbook with this macro
    Set Sourcewb = ThisWorkbook

    'Create new folder to save the new files in
    DateString = Format(Now, "yyyy-mm-dd hh-mm-ss")
    FolderName = Sourcewb.Path & "\" & Sourcewb.Name & " " & DateString
    MkDir FolderName

    'Copy every visible sheet to a new workbook
    For Each sh In Sourcewb.Worksheets

        'If the sheet is visible then copy it to a new workbook
        If sh.Visible = -1 Then
            sh.Copy

            'Set Destwb to the new workbook
            Set Destwb = ActiveWorkbook

            'Determine the Excel version and file extension/format
            With Destwb
                If Val(Application.Version) < 12 Then
                    'You use Excel 97-2003
                    FileExtStr = ".xls": FileFormatNum = -4143
                Else
                    'You use Excel 2007-2010
                    If Sourcewb.Name = .Name Then
                        MsgBox "Your answer is NO in the security dialog"
                        GoTo GoToNextSheet
                    Else
                        Select Case Sourcewb.FileFormat
                        Case 51: FileExtStr = ".xlsx": FileFormatNum = 51
                        Case 52:
                            If .HasVBProject Then
                                FileExtStr = ".xlsm": FileFormatNum = 52
                            Else
                                FileExtStr = ".xlsx": FileFormatNum = 51
                            End If
                        Case 56: FileExtStr = ".xls": FileFormatNum = 56
                        Case Else: FileExtStr = ".xlsb": FileFormatNum = 50
                        End Select
                    End If
                End If
            End With

            'Change all cells in the worksheet to values if you want
            If Destwb.Sheets(1).ProtectContents = False Then
                With Destwb.Sheets(1).UsedRange
                    .Cells.Copy
                    .Cells.PasteSpecial xlPasteValues
                    .Cells(1).Select
                End With
                Application.CutCopyMode = False
            End If


            'Save the new workbook and close it
            With Destwb
                .SaveAs FolderName _
                      & "\" & Destwb.Sheets(1).Name & FileExtStr, _
                        FileFormat:=FileFormatNum
                .Close False
            End With

        End If
GoToNextSheet:
    Next sh

    MsgBox "You can find the files in " & FolderName

    With Application
        .ScreenUpdating = True
        .EnableEvents = True
        .Calculation = xlCalculationAutomatic
    End With
End Sub

bobgreenblatt say: You can NOT have a back slash in the file path. Also Excel nodes are separated with a colon. So instead of:
FolderName = Sourcewb.Path & "\" & Sourcewb.Name & " " & DateString

try this:
FolderName = Sourcewb.Path & ":" & Sourcewb.Name & " " & DateString

You have the same problem later in the macro for >saveas foldername _
