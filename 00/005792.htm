Black squares are a type of bullet, and it may well be that they are currently selected as the default bullet on your system. Pictures and Charts are part of the Shapes collection in the object model. You must check the Shape.Type for the enumeration value you want. Here is example Excel VBA that finds shapes on a given worksheet.

Sub findExcelShapes()
    Dim xWB As Excel.Workbook
    Dim xWS As Excel.Worksheet
    Dim shp As Excel.Shape
    
    Set xWB = ActiveWorkbook
    Set xWS = xWB.ActiveSheet
    
    For Each shp In xWS.Shapes
        If shp.Type = msoLinkedPicture Or shp.Type = msoPicture Then
            Debug.Print "Picture"
        ElseIf shp.Type = msoChart Then
            Debug.Print "Chart"
        End If
    Next
End Sub

john w. vinson say:

I'm not aware of any "issues" with date sorting (just so you are sorting a date/time field and not using the Format() function to recast it into text... and yes, some people have innocently fallen into that trap!).

And yes, a Totals query using Min([datefield]) will return the earliest (lowest) value for that date, and Max the latest.

If you need to return the earliest date and the rest of the fields associated with that date, though, you won't want to use a totals query; instead use a criterion

=(SELECT MIN(X.[datefield]) FROM tablename AS X WHERE <criteria>)

as a criterion on the datefield, using criteria which limit the subquery to the desired subset of the records.

If you are having trouble with a specific query, please post the SQL view of the query and indicate what problem you're having.
