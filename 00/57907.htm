a. user say:

If from Control Panel, Sound/Adjust System Volume you mean you are seeing a message like this:

What you should do is determine if the sndvol.exe file is really missing (and it probably is) and if it is missing it needs to be replaced.

The file could have been quarantined (effectively deleted) by Avast and there are a few reports of that on the Internet.

Click the Start orb and in the box enter:

cmd

Right click cmd.exe and then 'Run as administrator' to open a Command Prompt window and enter the following commands:

cd   \windows\system32

dir   sndvol.exe

If you see a message 'File Not Found':

That is the problem so in that case run the System File Checker to see if it will replace the missing file by entering the following command:

sfc   /scannow

Wait for sfc  /scannow to finish (this could take some time so wait for it to finish).

When sfc   /scannow finishes check again to see if the sndvol.exe file has been replaced by entering this command again:
