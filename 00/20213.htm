jeeped say:

The first thing to remember is that your 'numbers' must be input as text since there are 20 digits (with or without a leading zero) and Excel only manages a precision of 15 digits. If they were input as numbers then,

02931504735108219983

  ... becomes,

2931504735108210000

So you need to pre-format the target cells as Text before pasting in the values. This way they will retain the full 20 digits together with any possible leading zeroes.

If you are typing the numbers into c ells that have not been formatted as Text then preceed the number with a single tick (e.g. ' ) like '02931504735108219983 and the number will be converted to a textual representation of a number.

Of course, these textual representations of 20 digit numbers cannot be used mathematically and would lose precision if converted to true numbers and used mathematically.
