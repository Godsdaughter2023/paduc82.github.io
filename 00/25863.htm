steve rindsberg say:

PowerPoint will absolutely make a mess of most presentations when the user changes the slide size to something that's not proportional to the original.

However, there's usually no real NEED to adjust the presentation size for projection.  PowerPoint will make the slide image as big as it can on the available display; so for example on a 4:3 ratio projector/screen, it'll display your 16:9 presentation at the full width of the screen but put black bars top and bottom.

Since black is the absence of light, the projected effect is pretty much the same as if they'd used a 16:9 capable projector on the same screen.

The happy-go-lucky in me says that they're being obsessive in worrying about it.

If they absolutely MUST get rid of the black bands, they'll need to:

- Create two different versions of the presentation, one for each aspect ratio, or ...

- Change the slide size, then spend minutes/hours/days cleaning up the messes, or ...

- Open the presentation in Windows PPT 2013 and use its resize feature, or ...

- Use one of the various resize add-ins on the market (will require using a Windows PC and pretty much any version of Windows PPT; I don't know of any that are available for Mac PowerPoint).  I can recommend a good one if you're interested.  I wrote it. ;-)
