Hi Malcolm,

1. Can I setup O365 with Domain 2 and edit the records on Domain1.

No, since you have set up Office 365 with domain 2, you must set up the DNS records on Domain We cannot edit the records on another domain.

2. Do I need to add domain 2 to Office 365, if I use Domain 1 as login.

If you use Domain 1 as login, generally, there is no need to add domain 2 to Office 365 if you have all the DNS records set up correctly in Domain 1. However, if you want also to use Domain 2 to as login, you must add domain 2 to Office 365 as well and have the Domain verified successfully in Office 365.

3. Would Office 365 process domain 2 without any problems if it is setup with domain 1, using the DNS forwarding.

Before answering this question, I’d like to know what does the “DNS forwarding” you mentioned refer to. If it is referring to Domain forwarding, I’d like to tell you that Office 365 doesn’t support Domain forwarding at this time.
