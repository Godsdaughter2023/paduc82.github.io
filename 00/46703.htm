Hi,

Here is what worked for my Lexmark... It may or may not help with your Dell, but has worked for some... Substitute Dell for Lexmark when appropriate..

The biggest difference from what you have tried is to "Remove" the printer and turn off the "Print Spooler" before uninstalling and reinstalling the software drivers..

1) Turn on and hook up your printer... This needs to be done just this one time to get it to show up in Devices and Printers.... Otherwise, all I get with my machine is the Lexmark Fax, and as a result, you can't "Remove" the printer.

2) Once the printer is visible in Devices and Printers, unhook it..

3) "REMOVE" the Lexmark Printer and Lexmark Fax (if you have one) in Devices and Printers. Just right click on it... Do the FAX first if you have one...

4) Stop the Print Spooler in "Services" .. Control Panel... Administrative Tools... Services

5) Uninstall the software\drivers...

6) At the prompt from the Lexmark uninstaller, Reboot.

7) After rebooting, stop the Print Spooler again.

8) Reinstall software\drivers... I used both XP SP3 and Vista Compatibility Mode on different attempts, and both worked... Hook the printer back up when prompted by the installer. Win 7 may work for your printer depending on it's age..

Before steps 9 + 10, check to see if Bidirectional support is Enabled... It may already be, and if so, skip to Step #11.

9) Unhook the printer again and reboot. Bidirectional Support must be Enabled, and the setting won't stick if the printer is hooked up.. You need to reboot to clear things up before the next step.

10) Enable Bidirectional Support and Reboot. To do this, go to Devices and Printers and select "Printer Properties" and then the "PORTS" tab... The check box is near the bottom.

11) Set the Printer Preferences you use, if any, and it's ready to go.

techfreak_ say:

The new version of Skype (8.x), stores its profile information and setting changes in %appdata%\Microsoft\Skype for Desktop . If your settings are not saving, you either have a corrupt local profile or a program blocking changes to that location.

To resolve this issue you would try the following:

1. Uninstall Skype, reboot, then reinstall Skype.

2. If the above did not work, rename the Skype for Desktop folder in %appdata%\Microsoft to something else like Skype for Desktop1.

3. Start Skype, make changes to Skype, then see if they stay when you restart Skype.

4. If the above did not work, boot into Safe Mode with Networking (this ensures other background apps are not blocking changes).

5. You should have Internet access in Safe Mode allowing you to sign in and make changes.

6. See if those changes stick.

7. If they still do not, perform steps 2 and 3 while in safe mode with networking, renaming the folder to something else.

If the settings save in this Safe Mode you can try in normal mode. If it continues to work in normal mode, your problem should be resolved. If it does not, you may have to add an exclusion to %appdata%\Microsoft\Skype for Desktop and Skype itself, in your antivirus or security software, to allow Skype to function properly.
