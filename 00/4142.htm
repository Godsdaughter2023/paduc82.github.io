mjs1a say:

Msosync is run via a task in Task Manager. To disable it:

Start >> Administrative Tools >> Task Manager >> Highlight "Task Scheduler Library".

You will see a task called "Microsoft Office 15 Sync Maintenance for..."

Right click on it and choose "Disable".

Even if you disable the task, the process for MSOSync will still be running. So, to stop it you either have to reboot, or kill the process using Task Manager.

Hope this stops your high CPU usage.

Sub ExportToYaml()
    Dim r As Long, Name As String, Level As String, Strength As String
    Dim Pathname As String, Filenum As Long, Filename As String
    Pathname = "C:\Users\whippet\Desktop\" '<- replace with desired output destination
    Filename = "yaml.txt" '<- replace with desired output filename
    Filenum = FreeFile
    Open Pathname & Filename For Output As #Filenum
    For r = 2 To 3 '<- Data excluding headers in A2:C3, replace as appropriate
        Name = Range("A" & r).Value
        Print #Filenum, "- Name: " & Name
        Level = Range("B" & r).Value
        Print #Filenum, "  Level: " & Level
        Strength = Range("C" & r).Value
        Print #Filenum, "  Strength: " & Strength
    Next r
    Close #Filenum
End Sub
