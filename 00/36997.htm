Synaptic driver update breaks keyboard special keys - Microsoft Community

tiago.s say:

Synaptic driver update breaks keyboard special keys

Hi all,

I own a Samsung laptop NP350 Series.

On June 15th, I've received an update to the trackpad driver through Windows Update.

The driver has the name "Synaptics - Pointing Drawing - Synaptics PS/2 Port TouchPad" and was marked as an Important update.

After installing and rebooting the machine I started to notice a very strange behavior with some of my keys. Here's the full description of the symptoms:

  • Alt Gr -> No longer works. Instead it is mapped like a normal ALT key
  • Windows Key -> Pressing and holding this key seems to keep calling the start menu. The behavior is like pressing and releasing at a very fast rate
  • Alt -> Behavior changed. Pressing and releasing doesn't call the menu bars on applications. Pressing and releasing the key has the same effect as the normal "pressing and holding".

I spent hours trying to figure out what could be the problem because I refused to believe that the driver for the trackpad would have any effect on the keyboard. Later, I found someone that had the same problem and also had a Samsung laptop PC, so I began to think that the issue might in fact be the trackpad driver.

I uninstalled it, rebooted the machine and now works perfectly.

Just sharing this to others who might be having the same issue. It's pretty bad that a broken driver come through Windows Update, specially because it was marked as important and therefore installed automatically.
