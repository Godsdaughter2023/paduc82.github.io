Outlook 2016 Reminders Window - minimized instead of closed or - Microsoft Community

gordonwalker162 say:

Outlook 2016 Reminders Window - minimized instead of closed or dismissed

In Outlook 2016, if the reminders window (for a meeting or task) is minimized (-) instead of closed (x), no new reminders will appear until the window is restored.

Sequence to reproduce the issue:

  1. Create an appointment with a reminder.
  2. When the Reminder window pops up minimize the window.
  3. Create another appointment with a reminder.
  4. The Reminder window will not pop-up again until the window is restored. 
  5. If the Reminder window is hovered over in the task bar, it will only display the first reminder.  The second reminder will not show up until the window is restored.

If the Reminder window is closed instead of minimized, new reminders will pop-up as intended.

Suggestion:

Don't minimize the Reminder window.  Always close the window if all the reminders are not dismissed.

I know it can be frustrating, but there are some sound features that do not work, if you want to convert your slideshow to a video:
- do not limit the sound to a number of slides
- do not use "Trim"
- do not use "Fade in/out"
- do not use sound with transition, always insert it on the slide
- do not use midi-files
There are two solutions involving third-party programs:
1. Use the free sound edititing program Audacity to trim your sound files or to combine them in one long sound file that will span the whole presentation. Export to mp3 and insert on the slide.

I want the MATCH "Value" to come from the selection of a Combo Box Control.

Is that possible?

hansv mvp say:

You can place two kinds of combo box on a worksheet:

  • From the Form Controls. You can set the cell link of such a combo box in the Format Control dialog. The linked cell will contain the index of the selected item (1 if the user selects the first item, etc.). You can use this index in your formula, either directly or via an extra INDEX function.
  • From the ActiveX Controls. You can set the LinkedCell of such a combo box in its Properties dialog. The linked cell will contain the selected item of the combo box. You can use this in your formula.
