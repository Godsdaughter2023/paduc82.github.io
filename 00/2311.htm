ISEVEN(A1) will yield TRUE when the integer portion of a number is an even number (and 0 is considered even).

=INT(A1)=A1 will yield TRUE if there is no decimal portion.

 

So a formula like

=AND(ISEVEN(A1),INT(A1)=A1)

should do the trick for you - will return TRUE only when the number is an even number with no decimal portion.
