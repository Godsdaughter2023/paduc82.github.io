How do I permanently remove empty rows in excel - Microsoft Community

walt2223 say:

How do I permanently remove empty rows in excel

How do I permanently remove empty rows in excel?  I only need 100 rows...
chip pearson say:

You can do it with code. What does "empty row" really mean? Is a row considered empty if a specific cell in the row is empty but other cells in the column may contain data? Or that the entire row is empty? The code below loops from the bottom up examining the column named by TEST_COLUMN (marked with '<<< in the code; change to suit your needs). IF TEST_COLUMN is empty, the entire row is deleted.

 

Sub DeleteRows()
    Dim LastRow As Long
    Dim WS As Worksheet
    Dim N As Long
    Const TEST_COLUMN = "A" '<<< CHANGE
    Set WS = ActiveSheet
    On Error GoTo ErrHandler:
    With Application
    .ScreenUpdating = False
    .EnableEvents = False
    .Calculation = xlCalculationManual
    End With
    With WS
        LastRow = .Cells(.Rows.Count, TEST_COLUMN).End(xlUp).Row
    End With
    For N = LastRow To 1 Step -1
        If WS.Cells(N, TEST_COLUMN).Value = vbNullString Then
            WS.Rows(N).Delete
        End If
    Next N
ErrHandler:
    With Application
    .ScreenUpdating = True
    .EnableEvents = True
    .Calculation = xlCalculationAutomatic
    End With
End Sub

 

If you want to delete a row if ALL columns are blank, use the following code:

 

Sub DeleteRows()
    Dim LastRow As Long
    Dim LastCol As Long
    Dim WS As Worksheet
    Dim N As Long
    Set WS = ActiveSheet
    On Error GoTo ErrHandler:
    With Application
    .ScreenUpdating = False
    .EnableEvents = False
    .Calculation = xlCalculationManual
    End With
    LastRow = WS.UsedRange.Find(what:="*", LookIn:=xlValues, _
        lookat:=xlWhole, searchdirection:=xlPrevious, MatchCase:=False).Row
    For N = LastRow To 1 Step -1
        With WS
            LastCol = .Cells(N, .Columns.Count).End(xlToLeft).Column
        End With
        If LastCol = 1 And WS.Cells(N, LastCol).Value = vbNullString Then
            WS.Rows(N).Delete
        End If
    Next N
ErrHandler:
    With Application
    .ScreenUpdating = True
    .EnableEvents = True
    .Calculation = xlCalculationAutomatic
    End With
