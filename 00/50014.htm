canadian tech say:

Your Dell came with a PC checkup program.  You should find it under Dell in the Menu.  Run that program.  Run the long version.  It will test all the main hardware components and may give you some good clues, if it is a hardware problem.

You should also run a drive tester program on the hard drive.  First you need to know the brand name of the drive.  If you download and run Speccy (http://www.piriform.com/speccy) It will tell you that.  Now you have to find the hard drive manufacturer's drive tester program, download and run all its component tests.  These tests are used by the drive manufacturers to determine if you have a warranty claim, but still do the job nicely even on an old drive, like yours.  They test the hardware of the drive.

You should run the Windows 7 built in memory tester (Windows Memory Diagnostic).  Just type that in the text box above the start globe.

Random shutdowns usually indicate an-over heat situation, but there are lot of other possibilities.  Speccy will also show you the current operating temperatures of various components.  They will show red if over spec.

Keep in mind that laptops' typical life is usually less than 6 years unless they are very well taken care of.  Most at 6 years are no longer usable.  The main reason they die that young is that repair costs get pretty unreasonable, pretty quickly.  All that said, the Vostro 3700 is a very nice computer.

If after all this, you can rule out a hardware failure, you are likely going to have to re-install the operating system.  If you are going to re-install the OS, you should very seriously consider installing a new hard drive.

It depends on how Windows 10 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.
