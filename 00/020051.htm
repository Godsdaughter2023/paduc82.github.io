jay freedman say:

Unlike the spelling checker, the grammar checker isn't based on a dictionary. There isn't any mechanism for indicating phrases that should be exempted from the grammar rules. (The rules themselves are a work in progress, and some of them create lots of false positives even in nontechnical writing.)

At this time, the best you can do is to define a character style that applies the "do not check spelling or grammar" setting from the Language dialog box, and apply that style to the phrases in question. You can do that in bulk by using the Replace dialog.

WM6 Task Filtering - Microsoft Community

reg37 say:

WM6 Task Filtering

I am new to WM6.  I use tasks extensively and I have tried all the filter/view settings on my WM6 device and have not been able to get it set up like I prefer.

 

I would like to be able to click on my tasks icon and then see only my tasks for that day, sorted from highest priority down to lowest.  I'd then like to be able to view only the next day's tasks and so on.  Additionally, I'd like to be able to view only my tasks without due dates, sorted from highest priority to lowest. 

 

I tried PocketInformant and had major issues.  It seems like it has a ton of bells and whistles that I simply don't need.  

 

Any suggestions?  Hopefully I'm just missing something simple. 

 

Thanks!

REG

julie fka profjulie say: I use Agenda Fusion, which is probably pretty close to PocketInformant in regards to bells & whistles.  I trialed Pocket Informant last year and decided to stick wth Agenda Fusion.  I do not use all the bells & whistles in Agenda Fusion.

By reading the description of what you'd like to accomplish, i think you can do this with Pocket Informant or Agenda Fusion (and probably other PIM software as well).

I don't use the priorities that are built into Outlook (low, normal, high) since they are limited to only 3 priority levels  Agenda Fusion allows me to assign A B C D .... and/or  1 2 3 4 ...., etc priority to tasks.  I decided to prioritize my tasks using numbers (1-2-3-4, etc.).  When I played around with this on Agenda Fusion, I noticed that Agenda Fusion appends the number to the beginning of the task description, so when I sync these tasks back to my computer, I'll see task that look like this:

1: Pick up dry cleaning
2: Make vet appointment
3: Schedule hair appointment

Since I generally use my computer to input most of my tasks, I've just started appending numbers (with the colon) to the task description.  Then when I sync these tasks to my Pocket PC, the priority works exactly as it should in Agenda Fusion.

The key is in how you set up your sorting.  To do what you describe, all you'd need to do is group by Due Date  (this will order your tasks by day) and then sort by task descriptions (this will order all your tasks by priority).  You can group and sort like this both in Agenda Fusion and Outlook on the computer.

Since I don't put due dates on most of my tasks, when I sort them in Agenda Fusion, I do not use a group option, but I sort by task description and then by Due Date.
