rich007a say:

Hi,

You should be able to use some worksheet formulas to do this.

If cell A1 contains "0010 - red car" then  =LEFT(A1,4)  will return the code "0010"

If you have a table somewhere, lets say Sheet2 cells A1:B12, that contains codes in col A and matching names in col B, a simple VLOOKUP formula will return the required name:

=VLOOKUP(LEFT(A1,4),Sheet2!$A$1:$B$12,2,FALSE)

Then join that up with the remains of your data to give the desired outcome:

=VLOOKUP(LEFT(A1,4),Sheet2!$A$1:$B$12,2,FALSE)&MID(A1,5,9999)
