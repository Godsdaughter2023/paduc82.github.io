Word VBA Footer Left Right Alignment - Microsoft Community

dachterberg say:

Word VBA Footer Left Right Alignment

I have been working on a larger macro that will clean up the formatting on potentials hundreds of existing documents. As part of this effort, the existing footers are deleted and new ones created. In the following code the Section Name should be left aligned and the Section Number - Page Number should be right aligned. If anyone could provide any guidance it would be helpful.

Private Sub InsertFooters()
' https://answers.microsoft.com/en-us/msoffice/forum/msoffice_word-mso_other/vba-code-to-enter-first-page-footer/989f89a6-360c-469e-9bae-31bef513b962
    Dim oSection As Section
    Dim oFooter As HeaderFooter
    Dim oRng As Range
    For Each oSection In ActiveDocument.Sections
        For Each oFooter In oSection.Footers
            If oFooter.Exists = True Then
                Set oRng = oFooter.Range
                Select Case oFooter.Index
                    
                    Case Is = wdHeaderFooterPrimary
                        Set oRng = oSection.Footers(wdHeaderFooterPrimary).Range
                        On Error Resume Next
                        oRng.Text = ActiveDocument.CustomDocumentProperties("Section Name")
                        
                        With oRng
                            .Case = wdUpperCase
                            .Font.Name = "Times New Roman"
                            .Font.Size = 10
                            .Font.Color = wdColorAutomatic
                        End With
                        
                        oRng.InsertAlignmentTab wdRight, wdMargin ' sets Section Number - wdFieldPage to right but deletes Section Name
                        oRng.Collapse direction:=wdCollapseEnd
                        oRng.Text = ActiveDocument.CustomDocumentProperties("Section Number") & " - "
                        oRng.Collapse direction:=wdCollapseEnd
                        oRng.Fields.Add Range:=oRng, Type:=wdFieldEmpty, Text:= _
                            "PAGE  \* Arabic ", PreserveFormatting:=True

                    Case Is = wdHeaderFooterFirstPage
                        ' this sets Section Name and Section Number left aligned
                        Set oRng = oSection.Footers(wdHeaderFooterFirstPage).Range
                        On Error Resume Next
                        oRng.Text = ActiveDocument.CustomDocumentProperties("Section Name")

                        With oRng
                            .Case = wdUpperCase
                            .Font.Name = "Times New Roman"
                            .Font.Size = 10
                            .Font.Color = wdColorAutomatic
                        End With
                        
                        oRng.Collapse direction:=wdCollapseEnd
                        oRng.Text = ActiveDocument.CustomDocumentProperties("Section Number") & " - "
                        oRng.Collapse direction:=wdCollapseEnd
                        oRng.Fields.Add Range:=oRng, Type:=wdFieldPage, PreserveFormatting:=True
                        On Error GoTo 0
                        
                    Case Is = wdHeaderFooterEvenPages
                        oRng.Text = "This is the even page footer"
                        
                End Select
                oRng.Fields.Update
            End If
        Next oFooter
    Next oSection
    
End Sub
