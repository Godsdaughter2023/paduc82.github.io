nodrog33 say:

This just means your device does not have the necessary hardware etc for logging on with Hello so the only thing you can do with something like this is to stop the event being logged by disabling the log provider.

In Event Viewer select the Details tab for the event, expand System and note the provider name and/or GUID - in this case it should be 23B8D46B-67DD-40A3-B636-D43E50552C6D.

In Regedit go to this key - Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\WMI\Autologger\EventLog-Application\{23b8d46b-67dd-40a3-b636-d43e50552c6d} and change the value for the DWORD "Enabled" to 0.

The same method can be used to hide other events you don't want/need to see - system event log providers are under the keypath

Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\WMI\Autologger\EventLog-System
