    Hello ilPapi,

    Thanks to Mark L Ferguson, MVP for this information:

    'clip' is the utility you can use to copy any number of clip files to the clipboard.

    C:\Windows\System32\cmd.exe /c type C:\Users\Mark\Desktop\clp.txt | C:\Windows\System32\clip.exe

    There is no reason you might not put hundreds of shortcuts with the correct command in an available location for instant use. Creating those saved files is just as easy.

    clip > c:\readme.txt

    <or, appending the file:>
