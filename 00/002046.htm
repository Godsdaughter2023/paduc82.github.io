jeeped say:

If you absolutely require the use of south asian and north american number formatting, could you use a 'work-around' solution that utilizes a User Defined Function (UDF) for formatting?

Press ALT+F11, then select Insert, Module from the menus. Paste this code directly into the new pane on the right,

Function fcnKACLSDE(Target As Range) As String
    Dim bNeg As Boolean
    Dim nmbr As Double
    Dim ek As String, dus As String, sauasra As String, lac As String, crore As String
    Dim arab As String, kharab As String, strnmbr As String
    If Target.Value < 0 Then
        bNeg = True
    Else
        bNeg = False
    End If
    ek = Right(Format(WorksheetFunction.RoundUp(Target.Value, 0), "#000"), 3)
    nmbr = WorksheetFunction.RoundDown(Abs(Target.Value), 0)
    kharab = Format(WorksheetFunction.RoundDown(nmbr / 10 ^ 11, 0), "00")
    If nmbr >= 10 ^ 11 Then
        nmbr = nmbr Mod 10 ^ 11
    End If
    arab = Format(WorksheetFunction.RoundDown(nmbr / 10 ^ 9, 0), "00")
    If nmbr >= 10 ^ 9 Then
        nmbr = nmbr Mod 10 ^ 9
    End If
    crore = Format(WorksheetFunction.RoundDown(nmbr / 10 ^ 7, 0), "00")
    If nmbr >= 10 ^ 7 Then
        nmbr = nmbr Mod 10 ^ 7
    End If
    lac = Format(WorksheetFunction.RoundDown(nmbr / 10 ^ 5, 0), "00")
    If nmbr >= 10 ^ 5 Then
        nmbr = nmbr Mod 10 ^ 5
    End If
    sauasra = Format(WorksheetFunction.RoundDown(nmbr / 10 ^ 3, 0), "00")
    strnmbr = kharab & Chr(44) & arab & Chr(44) & crore & Chr(44) & lac & Chr(44) & sauasra & Chr(44) & ek
    While Left(strnmbr, 1) = Chr(48) Or Left(strnmbr, 1) = Chr(44)
        strnmbr = Right(strnmbr, Len(strnmbr) - 1)
    Wend
    If bNeg Then
        fcnKACLSDE = "-" & Trim(strnmbr)
    Else
        fcnKACLSDE = Trim(strnmbr)
    End If
End Function

Use it like any other function as in,

=fcnKACLSDE(A1)    <placed in B1>

... and use Custom Number Formatting like "Rs "@ to apply the Rs currency symbol. As it is now a textual representation of a number, you must parse it before using it as a value,

=VALUE(SUBSTITUTE(B1,",",))    <with B1 as above>

The UDF works within a range of 2,14,74,83,648 to -2,14,74,83,648.
