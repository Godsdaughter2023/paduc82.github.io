I've had exactly this same problem, so can tell you how I solved it.

 

In my case, the problem was that I had associate two different email accounts with my nokia messaging account. I don't know that this would work if you wanted to see both accounts on your Nokia at the same time, but I don't want to do that, so it works for me.

Changing height of display area for individual footnotes - Microsoft Community

rp_codie say:

Changing height of display area for individual footnotes

To prevent text pertaining to a single footnote from "running over" onto the next page, I would like to be able to edit & change the amount of page height that is allocated to each footnote. Thus I am not looking to change JUST the height of the footnote separator line (but I would like to know how to do that) but also the total space / number of lines available for each footnote. Any help appreciated. 
 
macropod say:

That is not possible. What you can do, however, is to format the footnote paragraph with the 'keep together' attribute and, if there are multiple paragraphs for the same footnote, apply the 'keep with next' attribute to all but the last.

Alternatively, you could move the footnote reference to a higher or lower position on the page until both the footnote reference and the footnote are on the same page. If that results in the footnote reference being too high/low on the page, you could:

• format the footnote reference as hidden text;

• insert a formatted cross-reference to the footnote in the location where the footnote reference originally appeared.

Carefully done, this will allow you to keep the entire footnote on a single page; the downside is that the desired location for the visible reference to it may no longer be on the same page.
