jiaxing bian msft say: Hi Mary,

You don’t need to share the site with Ringo, if you only want him to access some specific document.

To share a document, you can use either "Invite People" or "Get a Link".

With “Invite People”, the user that’s been invited will receive an invitation email. Once he accepts the invitation by opening the sharing link, the link is no longer available for other users.

Link generated by “Get a Link” is reusable. Anyone with the same link can accept the invitation, and access the document.
