amit tandon say:

The logic for the earlier formula was mentioned as "All values from 1st row where cell value is NOT "N/A" will be returned". This means that where cell values are blank will display the corresponding values from the first row.

In case do not want to display the corresponding values from the first row for cells which have EITHER "N/A" or are BLANK, then use below formula:

=IFERROR(SUBSTITUTE(IF(OR($A2="N/A",$A2=""),"",$A$1&"/")&IF(OR($B2="N/A",$B2=""),"",$B$1&"/")&IF(OR($C2="N/A",$C2=""),"",$C$1&"/")&IF(OR($D2="N/A",$D2=""),"",$D$1&"/"),"/","",COUNTIFS($A2:$D2,"<>N/A",$A2:$D2,"<>")),"")

This formula will NOT display corresponding values from first row where cell values in a row are EITHER "N/A" OR BLANK.

So I can't find the webpage I found the info on how to fix the issue - I printed it out and then took it with me to an open house (I am a realtor) and the instructions worked!  I do not have this error message any longer on either of my laptops!  So I just had to share with you.

You need to run the Outlook Repair Tool

1) Close Microsoft Outlook

2) Open the "Scanpst.exe" file. You can find this fine in Program Files\Microsoft Office\OFFICE12 (on my laptop it was OFFICE14) Click and a window titled "Enter the Name of the File you want to Scan" will open.

3)Enter the name of your PST file or click "Browse" to select it (I went to the search box on the desk top - all I did was type in PST there and the file needed appeared - that was all I needed in order to find it once I clicked the Browse button)

4)Click Start - the scan will begin.  You will be prompted to repair the file if the inbox repair tool finds errors

5) Click "Repair"

6) Open Outlook and log in (for me all worked fine from that point the repair tool told me I only had a couple of errors - meaning only a few things removed)

7) Select the "Go" menu Click "Folder List" The Recovered Personal Folders" may contain a "lost and Found" folder that has the items recovered by the repair

8) Create a new PST file.  On the "File menu click "New" Select "Outlook Data File" Click "Office Outlook Personal Folders File" If you are using Outlook 2003 or 2007 Otherwise select Outlook 97-2002 Personal Folders File Click "OK" Enter a filename and click "OK" Enter a display name for the folder. Enter a password to protect the file if you want Click "OK"

9) Drag the items from the "Lost and Found" folder to the new PST file

I only did the steps 1-6 and I am up and running again.  I have not looked into steps 7-9 yet but right now I am happy not to have that error box any longer.  Good luck.
