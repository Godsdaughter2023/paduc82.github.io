When Movie Maker is saving your movie to DVD, it is calling Windows DVD Maker (available in Vista and Windows 7, but not in Windows 8).  That program creates a video DVD matching the video DVD standard.  It closes the DVD when done burning.  This process isn't designed to make a data DVD and allow you to add more files.

Hi Melissa

My name is Andre Da Costa; an Independent Consultant, Windows Insider MVP and Windows & Devices for IT MVP. I'm here to help you with your problem.

It depends on how Windows 10 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.

gold3n soldier say:

Bonjour,

Il existe un moyen simple pour savoir si une console est banni. Demande à ton vendeur d'occasion d'effectuer ceci devant toi :

1. Débrancher tout vos périphériques(DD,Cartes mémoires)
2. Allumer la console avec le câble Ethernet/Wifi connecté
3. Aller à Configuration Systeme
4. Choisissez réseaux
5. Choisissez « Tester Connexion Xbox Live »
6. Attendre, si ça vous demande d installer une mise a jour METTEZ ANNULER
7. Puis Cliquer sur « Plus informations » puis Y pour avoir le rapport d erreur

Observez la ligne Z.

Z: 8015-190D (cette ligne indique que vous êtes banni)
ou
Z: 8015-1002 (cette ligne indique que vous n’êtes pas banni)

Comme ça été écrit plus haut une console banni, c'est pour la vie! L'utilisateur précédent n'a pas respecter les règles. A savoir qu'un bannissement d'une console est grave, une enquête très longue est mené par une équipe qui après décide selon la faute, si oui ou non il y aura bannissement. Seule les modifications sur la console entraîne un bannissement.
