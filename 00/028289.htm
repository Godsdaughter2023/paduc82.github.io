john - project say:

Shane_1984,

Sounds like you are off to a good start but let's revisit some details, remember, that's where the devil is.

When you create the calendars for each resource group, make sure you select the appropriate shift calendar in the Base Calendar field on the Resource Sheet. Also, since the 2nd shift crosses over midnight, you need to set the From and To periods as follows: (Shift 1 hours do not have that issue)

7:00PM - 12:00AM

12:00AM - 7:00AM

I'm not sure why your start times are at 1:00 PM but it may be an artifact from an attempt to set things up before you posted. At any rate, go to Project/Properties group/Project Information. Be sure to set the time for the Start Date box. For example, if your project starts today, the date should be 4/29/14 7:00 AM. Also make sure you select your modified 24 hour calendar as the project calendar.

With regard to your test task, if you simply assigned the shift 1 resource to work the 90 hr task, Project will automatically assume the assignment level is 100%. You need select the task and then go to Resource/Assignments group/Assign resources. Enter 1000% into the Units column and hit the "Assign" button. Alternately you could simply type "Shift 1 [1000%]" into the Resource Names field for that task on the Gantt Chart view and accomplish the same thing. Either way you should see the duration drop to 0.38 with a start time of 7:00 am and a finish time of 4:00 pm. If you do another example task with the same work content but this time assign the shift 2 resource, you should see a start time of 7:00 pm and a finish time of 4:00 am

To create an effective plan, you really need a good feel for the sequence of tasks. Normally in the construction industry the build must progress in a fairly defined fashion. For example, you need the plumbers in before the slab is poured so they can do the initial rough-in. Similarly you need the electricians, plumbers and HVAC trades to do their rough-ins before the sheet rock guys show up. With a good sequence, you will be able to see where the choke points are with regard to manpower and materials and manage the project effectively. Of course even in new construction, there will be some unknown unknowns that you really can't plan ahead of time. You just have to work them as they pop up. It's called managing the project.

Hope this helps.

John
