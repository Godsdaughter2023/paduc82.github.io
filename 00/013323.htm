https://support.google.com/mail/troubleshooter/1668960

hansv mvp say:

Here is a custom function:

Function SplitData(s As String)
    Dim a
    Dim i As Long
    Dim t As String
    Dim j As Long
    Dim r(1) As String
    a = Split(s, Space(5))
    For i = 0 To 1
        t = Trim(a(i))
        j = 1
        Do While IsNumeric(Left(t, j))
            j = j + 1
        Loop
        r(i) = Left(t, j - 1)
    Next i
    SplitData = r
End Function

Let's say your long string is in A2.

=SplitDate(A2)
