daniel g. grau say:
The language attribute does not have to be the same for the whole document. Just select the different portions and assign them the proper language with Tools > Language. The spell checker will automatically switch between languages.

Pay attention to the setting "Do not check spelling or grammar," as this will let you apply a language in such a way thatWord will completely ignore that text.

If languages alternate very often throughout the text, you should use different styles for each, incorporating the corresponding language to each style definition. For instance, you could have a paragraph style called EnglishBodyText and assign it a keystroke (like Control E), and another called FrenchBodyText (Control F). This would let you easily alternate between languages.

If the languages mix within the same paragraph or sentence, then you should do likewise, except that you would be using character styles instead of paragraph styles.

graham mayor say:

The short answer is no, but you could use a macro. Type the number e.g. 1000 or 1000.50 and then run the macro immediately afterwards. It will format as 1,000 or 1,000.50 respectively. If you have typed the comma(s) manually it will check the format. For ease of use assign the macro to a keyboard shortcut - http://www.gmayor.com/installing_macro.htm    Note there is no period in the second number string.

Sub FormatNumbers()
Dim orng As Range
    Set orng = Selection.Range
    orng.MoveStartWhile "0123456789.,", wdBackward
    orng.MoveEndWhile "0123456789,"
    If InStr(1, orng.Text, ".") > 0 Then
        orng = Format(orng.Text, "###,###,###,0.00")
    Else
        orng = Format(orng.Text, "###,###,###,0")
    End If
    orng.Collapse wdCollapseEnd
    orng.Select
End Sub
