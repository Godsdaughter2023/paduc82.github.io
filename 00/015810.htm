macropod say:

That suggests the software that produces these images saves them in the EPS (Encapsulated PostScript ) format, with the filename instead of at least a display version of the image.

EPS graphics are designed for printing to a PostScript printer.

If an EPS graphics file contains an embedded TIFF or Windows Metafile preview, a representation of the image appears on the screen. The quality of the preview depends on the resolution of the TIFF or Windows Metafile image embedded in the EPS file when it is created. Low, medium, and high resolutions are generally available for creating an EPS file. The higher the resolution of the preview image, the larger the EPS file size will be. Because such previews are intended primarily to be used to position images on the page, resolution is often low. A high-resolution preview is not necessary because it is discarded when the EPS file is printed to a PostScript printer.

If an embedded TIFF or Windows Metafile preview is not included in the EPS graphic you import, the graphic displays a message (eg the filename) instead of a preview of the graphic in the document. The graphic prints correctly to a PostScript printer but, if you print an EPS graphic to a non-PostScript printer, the preview image is printed as it appears on the screen.

The remedy is to save your image files in a different format when you want to use them in Word.

Hi Cheryl

The trick is to make all the numbers the same length by inserting zeros before the house number.  The macro below inserts the address into the next column (that needs to be blank, so insert a column before running the macro).

Sub FormatHouseNumbesNumbers()
Dim c As Range, number As Integer
Dim i As Integer, tmp As String

For Each c In Selection
    number = InStr(1, c, " ") - 1
    tmp = c
    If number < 3 Then
        For i = 1 To 3 - number
            tmp = 0 & tmp
        Next i
        c.Offset(0, 1) = tmp
    End If
Next
End Sub

Hope this helps

bob jones aka: cybertaz say:

The first point is to make sure you have the nonprinting characters [¶] displayed. They are immensely helpful so you can avoid deleting paragraph breaks as well as Section Breaks & other indicators.

Assuming Styles actually were used to format the document: They are stored in the document & can be deleted only if you intentionally choose to do so. Windows 10 OEM Upgrade with SSD

So quite some time ago I had made this idea on IdeaStorm:

http://www.ideastorm.com/idea2ReadIdea?Id=087700000008x0kAAA

Now that Windows 10 is getting closer and this idea requires input from Microsoft and not just the OEM I think its worth mentioning on Microsoft Answers.

Microsoft should allow OEMs to sell SSDs with Windows 10 as a factory image for free for all systems they sold with Windows 7 and Windows 8.1 in the first year of its release.

Whats the incentive?

Many systems shipped with Windows 7 (2009, 6 years old) have no OEM warranty. OEMs will therefore not have much incentive to provide updated system drivers for Windows 10 as they are getting no money supporting older systems, they make their money on selling hardware. The biggest problem with Windows 10 migration will be obtaining up to date third party system drivers. OEMs currently will not make money by supporting 6 year old systems as it stops them from selling newer systems if a user upgrades their old system.

Sales of hard drives and solid state drives from OEMs as upgraded components are generally quite poor as the end user still has to reinstall Windows meaning they usually pay more for less. If the OEM on the other hand was allowed to make a new factory image of Windows 10 on a SSD which only they sell then the OEM will as a consequence provide driver support for Windows 10 for that model prolonging driver support for that model and the number of SSDs sales from an OEM will increase. This will make Windows 10 deployment significantly easier.

As an additional incentive the SSD vastly outperforms the mechanical HDD and increases battery life therefore the end users perception of Windows 10 who purchase it as an upgraded drive from their OEM will be much better as it will significantly outperform their old installation on a mechanical drive significantly helping boost Windows 10 reputation straight after its release.
