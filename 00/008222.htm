daniel g. grau say:

There is a preference in AutoFormat that may be the cause. It's called "Define styles based on your formatting." If it is active, changing the size of a footnote reference in the text would automatically change the size of all of them, given that they all should have the same character style (called Footnote Reference). You'll find this in Preferences > Autocorrect.

If that preference is not active, then try redefining the font size for style Footnote Reference. Access this through Format > Style; highlight Footnote Reference in the list; click on Modify; select Font and change to a larger size.

ankita vaidya msft say:

Hello CraigOSh,

 

Thanks for reply, as this is the first time you are trying to set up your domain in Office 365, so first you need add your custom domain and need to verify the domain in O365.

 

  • Verify your domain. Use the setup wizard to add your custom domain (like www.contoso.com), and verify that you own the domain.
  • Sign in to your Office 365 account, on Admin dashboard click Domains.
  • Choose Add domain to start the setup wizard, which guides you through verifying your domain and other setup steps.

 

You may refer the below articles for Set up your domain

 

  • Set up your domain with Office 365 : https://support.microsoft.com/en-us/help/4028070/office-set-up-your-domain-with-office-365
