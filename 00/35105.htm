    This batch file would do the trick. As it is, it will run in test mode. To activate it you must change do echo ren to do ren towards the end.

    @echo off
    set Folder=C:\Program Files (x86)

    set List="%temp%\Exe.txt"
    dir "%Folder%\*.exe1" /s /b /a-d > %List%
    for /F "delims=" %%a in ('type %List%') do echo ren "%%a" *.exe
    pause
