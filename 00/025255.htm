graham mayor say:
Without a sample of the actual document it is difficult to establish what is being searched for, but I printed your example to PDF and extracted the data with FineReader and the following macro will locate the square bracketed superscripted values. The bold entry processes the found items - here using HighLight, but you can do what you want with oNewRng instead.

Sub Macro1()
Dim oRng As Range, oNewRng As Range
    Set oRng = ActiveDocument.Range

    With oRng.Find
        While .Execute("\[*\]", MatchWildcards:=True)
            If oRng.Characters(2).Font.Superscript = True Then
                Set oNewRng = oRng
                oNewRng.Start = oNewRng.Start + 1
                oNewRng.End = oNewRng.End - 1
                oNewRng.HighlightColorIndex = wdTurquoise
            End If
            oRng.Collapse wdCollapseEnd
        Wend
    End With
End Sub

graham mayor say:
I can only find one reference in the document that fits the parameters you gave and which does not appear to be linked. The following modified version of the macro will find that. Click Cancel to stop in order to edit the hyperlink to what you want. Then run it again to find the next.

Sub Macro1()
Dim oRng As Range, oNewRng As Range
    Set oRng = ActiveDocument.Range
    With oRng.Find
        .Font.Superscript = True
        While .Execute(Forward:=True)
            If oRng.Hyperlinks.Count = 0 Then
                oRng.Select
                If MsgBox("Stop at " & oRng & "?", vbOKCancel) = vbCancel Then Exit Sub
            End If
            oRng.Collapse wdCollapseEnd

        Wend
    End With
End Sub

amit tandon say:

To lookup Monica's marks in Chem:

Option 1: Vlookup function - Enter the column number (4) for Chem so that if the column number changes the formula will NOT work - cell H2 formula:

=VLOOKUP($G$2,$A$1:$E$6,4,TRUE)

Option 2: Vlookup & Match functions - Match the column number for Chem so that if the column number changes the formula will work - cell H3 formula:

=VLOOKUP($G$2,$A$1:$E$6,MATCH($G$3,$A$1:$E$1,0),TRUE)

Option 3: Index& Match functions - Match the column number for Chem so that if the column number changes the formula will work - cell H4 formula:

=INDEX($B$2:$E$6,MATCH($G$2,$A$2:$A$6,0),MATCH($G$3,$B$1:$E$1,0))

The above options illustrate Vlookup, Index & Match functions - Option 2 & 3 are useful when columns or rows are inserted wherein the range is adjusted accordingly.
