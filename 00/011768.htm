hansv mvp say:

Create a form frmParameters with a text box txtFestivalYear.

Change the parameter [Enter Festival Year:] to a reference to the text box:

[Forms]![frmParameters]![txtFestivalYear]

Of course, the form must be open when you run the report. You could place a command button cmdOpenReport on frmParameters to open the report:

 

Private Sub cmdOpenReport_Click()
    If IsNull(Me.txtFestivalYear) Then
        Me.txtFestivalYear.SetFocus
        MsgBox "Please enter a year", vbExclamation
        Exit Sub
    End If
    DoCmd.OpenReport "rptMyReport", acViewPreview
End Sub

 

where rptMyReport is the name of the report.

pete hewitt say:

Hi Jay,

In versions 15.19 and earlier, pasting an HTML signature including images into the signature window would use linked images. In other words it would paste the HTML img tag in as found, pointing at images on the web server. This meant the email was very lightweight and images could be updated on the server and anyone's signature would update (useful for things like instagram's logo update).

In 15.20 the email editor was changed to share code with Word. At this point, pasting in the signature would instead embed a copy of the images into the message. In addition some rather nasty compression was applied to the images.

It means when you send an email to certain mobile clients (such as ios mail) and they reply, your signature images come back as lots of nasty attachments rather than just referring to images on the web.

I am currently on 16.8 (slow) and the issue still occurs. The only way round it was to install 15.19, put on your sig, then run updates to get to the latest version. I believe this is no longer an option as 15.19 won't run on High Sierra.

Hope that clarifies things?

Pete

You can do it with a macro (or a pair of macros) e.g.


To send as e-mail message:


Sub Send_As_HTML_EMail()
Dim bStarted As Boolean
Dim olApp As Object
Dim oItem As Object
Dim oRng As Range
Dim objdoc As Object
Dim objSel As Selection
    On Error Resume Next
    Set oRng = ActiveDocument.Range
    oRng.Copy
    'Get Outlook if it's running
    Set olApp = GetObject(, "Outlook.Application")
    If Err <> 0 Then
        'Outlook wasn't running, start it from code
        Set olApp = CreateObject("Outlook.Application")
        bStarted = True
    End If
    'Create a new mailitem
    Set oItem = olApp.CreateItem(0)
    With oItem
        .BodyFormat = 2
        .Display
        Set objdoc = .GetInspector.WordEditor
        Set objSel = objdoc.Windows(1).Selection
        objSel.Paste
        '.to = "*** Email address is removed for privacy ***"
        '.Subject = "The subject"
        .Display
    End With
    If bStarted Then
        'If we started Outlook from code, then close it
        olApp.Quit
    End If
    'Clean up
    Set oItem = Nothing
    Set olApp = Nothing
lbl_Exit:
    Exit Sub
End Sub


To send as attachment:


Sub Send_As_Mail_Attachment()
' send the document as an attachment _
  in an Outlook Email message
Dim olApp As Object
Dim olInsp As Object
Dim oItem As Object
Dim wdDoc As Object
Dim oRng As Object

    On Error Resume Next

    'Prompt the user to save the document
    ActiveDocument.Save

    'Get Outlook if it's running
    Set olApp = GetObject(, "Outlook.Application")

    'Outlook wasn't running, start it from code
    If Err <> 0 Then
        Set olApp = CreateObject("Outlook.Application")
    End If
    On Error GoTo 0

    'Create a new mailitem
    Set oItem = olApp.CreateItem(0)

    With oItem
        '.to = "*** Email address is removed for privacy ***"
        '.Subject = "This is the subject"
        .Attachments.Add ActiveDocument.FullName
        .BodyFormat = 2        'olFormatHTML
        Set olInsp = .GetInspector
        Set wdDoc = olInsp.WordEditor
        Set oRng = wdDoc.Range(0, 0)
        .Display
        'And insert the message body text
        'oRng.Text = "This is the first line of the message" _
                    & vbCr & "This is the next line of the message"
        'etc
    End With

    Set oItem = Nothing
    Set olApp = Nothing
    Set olInsp = Nothing
    Set wdDoc = Nothing
    Set oRng = Nothing
lbl_Exit:
    Exit Sub
End Sub
