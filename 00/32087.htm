If a program does not show up in the list, you can make the program a default by using Set Associations.

  1. Click the Start button, and then type Default Programs. Click on the Default Programs from the list.

  3. Click the file type or protocol that you want the program to act as the default for.

  4. Click Change program.

  5. Click the program that you want to use as the default for the file type you selected, or click the arrow next to Other Programs to show additional programs. My PC finds my Kingston card reader in the USB port but wont read the - Microsoft Community

zisboombah say:

My PC finds my Kingston card reader in the USB port but wont read the SD card - why?

Multiple USB ports in my PC w/Windows XP all see my thumb drives and can read data but while my Kingston card reader's 4 "type" ports are recognized, it wont read the SD picture card. Why?
lemp say:
You should have installed service pack 3 long ago, but that's not likely the cause of your immediate problem.  Up until the last part of your reply, I would have said that you needed to get a new card reader.  However, if the card reader works on another computer, something else seems to be the matter.

What is the model of your Kingston card reader? It should be on the label. 

With the reader connected to a USB port, open Device Manager (Start > Run > devmgmt.msc > OK).

Normally, you should see a number of entries in the "Disk Drives" category corresponding to the number of different slots in the card reader.  They probably will be named something like, "Kingston USB 2 HS-CF USB Device" and similar.  There also should be an entry in the "Universal Serial Bus Controllers" category that says "USB Mass Storage Device."

Inspect each of the above entries.  If any of them show a warning icon (exclamation), double click on the entry and report what appears in "Device status" and "Device usage."

Right click on each of the entries corresponding to the card reader (should be one for each card reader slot in the Disk Drives category and one USB Mass Storage Device in the USB Controllers category) and select Uninstall.  After all of the entries have been uninstalled, disconnect the card reader from the computer and reboot.  Now reconnect the card reader.  Windows should recognize it and automatically install it. 

NOTE:  If you have a USB-connected hard drive, you need to be careful that you uninstall the correct USB Mass Storage Device.  The safest way to do this is to use the "Safely Remove Hardware" icon to stop your external drive and then disconnect it from the computer.  At that point, its entry will be removed from Device Manager.

If the above procedure doesn't enable the card reader to once again read your card before, I'm out of ideas.
