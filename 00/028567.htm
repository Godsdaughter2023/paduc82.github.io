Dim strPassword As String

With ActiveDocument

If the source range doesn't contain formulas, change

 

        ' Copy over the values from the source to the destination.
        DestRange.Value = SourceRange.Value

 

to

 

        ' Copy over the values from the source to the destination.

        ' Copy over the values from the source to the destination.
