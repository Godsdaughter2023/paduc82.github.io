wwejason say:

Outlook 2010: Once the user has received 99 attachments of the exact same file name, Outlook can no longer open any of those attachments.

After having received 99 file attachments (separate emails) of the exact same name, Outlook can no longer open any of those attachments. In the temp directory where the downloaded attachments are stored, you must delete ALL of the files of the same name and allow Outlook to download them again from Exchange as needed.
