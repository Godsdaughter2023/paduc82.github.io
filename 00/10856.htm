andreas killer say:

You should never create a code in this way.

Beginners write
  Range("C123")= "Whatever"

Experienced beginners ask "how can I change my code if I move the column to E"? Most people think this is the answer:

Sub NeverDoThis()
  Dim zC As String
  Dim r As Long
  zC = "E"
  r = 123
  Range(zC & r) = "whatever"
End Sub

But that is wrong. In Excel we can create Named Cells, so if I name a cell in the headings as MyData I can use that cell (wherever it is) in my code:

Sub DoThis()
  Dim Here As Range
  'Get the top left cell
  Set Here = Range("MyData").Cells(1, 1)
  'Find an empty cell below
  Set Here = Here.Offset(Rows.Count - Here.Row).End(xlUp)
  'Write in the data
  Here.Value = "whatever"
End Sub

In that way you can move the column (the cell) to any location you like, the code works without changes!

Andreas.
