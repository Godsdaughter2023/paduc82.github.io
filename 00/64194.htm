What are my options after the Windows 10 free upgrade offer ends?

Technical Level : Basic

Summary

After the free offer ends you can still upgrade Windows 7 or 8/8.1 using a retail version of Windows 10.

Though not free a boxed or download retail version of Windows 10 cost a lot less than an upgrade version of Windows 7 when it was released to the public.


Details

There is a unique difference between a boxed (or download) version of 10 purchased locally, online and the version purchased at the Microsoft store. We will also explore that difference and OEM / System Builder versus Retail in this article.

Note: List prices may vary based on location and retailer.

==================================================

What are my installation Options?

Option #1: Custom / Clean install

A) Most commonly used by users who just built or purchased a computer that has no prior version of Windows installed.

B) Have no interest in keeping their current /older version of Windows not supported by an upgrade.

Example: Late model Windows XP or Vista based systems.

Precautions: Create an Image Backup your current version of Windows. This is your best and safest means of recovering your older version of Windows should your computer fail to fully support (Example: lack of a full set of device drivers such as LAN/Internet access or Audio).

Reference: Clean Install Windows 10

Option #2: Upgrade your current version of Windows 7 or 8/8.1

A) You missed the free upgrade offer or for whatever reason decided to wait until you were ready to make the switch to Windows 10.

Option #3: Dual Boot

A) The retail version unlike the free upgrade allows you to keep your current version of Windows and install Windows 10 on a separate partition or disk drive. This allows you to keep using some older applications that may not be supported by Windows 10.

Reference: http://www.pagestart.com/win7win10tpdb10121401.html

Where you purchase does make a difference:

A) Retail versions of Windows 10 purchased at a traditional brick and mortar store or on the Internet from Amazon, Newegg, Frys, Etc. you will need your product key to activate Windows 10 if you ever have the need to re-install Windows 10 (Example: disk drive failed) after the initial installation.

B) Retail versions of Windows 10 purchased at the Microsoft store grant you “Digital entitlement”. Re-installing Windows 10 will not require you to input your product key as Microsoft has a “Digital footprint” (unique to every computer) on record and activation is automatic.

Reference: Windows 10 - What type of product key do I have?

Should I purchase a “Boxed” version which includes a USB / Flash drive (write protected so you can’t accidentally erase the drive) to install Windows 10 or a “Digital download”?

A) The boxed version includes the USB installation media and a product key.

B) Download version allows you to create your own installation media. It is a multi-step process where you are sent a key via Email, a starter file which must be downloaded and when you run the starter file it downloads the Windows 10 ISO and a Media Creation Tool used to burn a DVD or create USB installation media. Please note to keep both the product key and the installation media in a safe place.

You can have the Start screen and apps from the Windows Store on only one monitor at a time. When you open the Start screen, charms, or your recent apps on a different monitor, apps from the Windows Store also move to that monitor.

Set up your screens

Once you plug in your screen, you can choose your display options.

To set what's on the second screen

  1. Swipe in from the right edge of the screen, and then tap Devices.
    (If you're using a mouse, point to the upper-right corner of the screen, move the mouse pointer down, and then click Devices.)

  2. Tap or click Second screen and then choose one of these options:

    • Duplicate. You’ll see the same things on both screens.

    • Extend. Your screen is now spread over two monitors, and you can drag and move items between the two.

    • Second screen only. You’ll see everything on the connected screen, and the other one will remain blank.

To control how the cursor behaves when one monitor is a touchscreen and the other isn’t

  1. Open Tablet PC Settings on your touchscreen by swiping in from the right edge of the screen, tapping Search, entering Tablet, choosing Settings, and tapping Tablet PC Settings.

  2. Tap Setup.  You might be asked for an admin password or to verify your choice.

  3. If you’re given a choice, tap Touch Input. Then, follow the on-screen instructions.

This question is better suited for the TechNet forums, but I will try to help you.

Make sure that the computer shows up in your management system, whether it is a Windows Server, or something else. If you are able to login with the user's credentials but not your own, this suggests that the client is not actually contacting the network to download the profile, but it is using the most recent cached user profile.

To check if this is the case, have the user log on to his/her profile on different, known working computer, and make a file on their profile. Then check to see if that same file shows up on the bad computer. 

If it doesn't show up, then the computer is not joined to the network correctly or is having trouble connecting. There could be a multitude of issues.
