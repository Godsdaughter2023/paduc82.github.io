inn0v8 say:

pelagic, thank you for posting to the Microsoft forums.

Restore points are saved until the hard disk space System Restore reserves is filled up. As new restore points are created, old ones are deleted. If you turn off System Protection (the feature that creates restore points) on a disk, all restore points are deleted from that disk. When you turn System Protection back on, new restore points are created.

Depending on how big the hard disk is System Restore would dynamically allocate storage. For example if you have a 60GB hard disk and System Restore is activated, then you would have only a couple restore points, as older ones would be deleted to make room for newer ones. You can however modify the storage space used by Windows Vista for System Restore Space.

-To do this:Click Start
-Type cmd (without quotes)
-Right-click the result at the top of the "Start Menu" and choose Run as Administrator

 This will open a black window, called a DOS prompt, with a flashing cursor.

  • Type vssadmin list shadowstorage and press enter.

This will show you your current settings (used, allocated, and maximum) for each drive. If you choose to specify System Restore Storage space we would then use the following command.

vssadmin Resize ShadowStorage /For=X: /On=Y: /Maxsize=ZGB

  • X = the drive you wish to change
  • Y = the drive you wish to have it stored on
  • Z = the maximum size you wish Vista to use
    • Z can be MB or GB with a minimum of 300MB. I suggest you stay with GB(gigabytes) for ease of use.

For example if we wanted to reduce the space allotted to System Restore to 10GB on drive c: we would use the following command:
vssadmin Resize ShadowStorage /For=C: /On=C: /Maxsize=10GB

An article regarding vssadmin can be found here:
http://technet.microsoft.com/en-us/magazine/dd348398.aspx
 
Naturally, using more System Restore space will decrease usable hard disk space. Please let us know if this helps.

Aaron Barth.

marilyn o say:

Hello akshyap,

Some information for you about allocated space in System Restore points.

By default Vista allocates 15% of your selected hard drive volume to storing System Restore points after installation. If System Restore is turned on much later, it may only allocate a minimum of 30% of the available free space instead. When the allocated space fills up with restore points, System Restore will delete the older restore points to make room for the new ones. System Restore needs at least 300 MB allocated to it to run, and will not run on a hard drive smaller than 1 GB.

These are some reasons why a System Restore point may not be created.
There are some 3rd party programs that may delete System Restore points.
Shadow copies may be lost when you defragment a volume. System Restore points are based on Shadow Copies. 
Before creating a System Restore point, you might want to disable your Antivirus. Some Antivirus products may
interfer with creating a System restore point.

Next, I would look in the System Event logs and see if there are any errors dealing with System restore or shadows copies.

Look at the information here about changing System Restore points space.

bunny_mama say:

Update:

I have been able to successfully reinstall my Dell 966 printer!  I eventually pieced together what I needed to do after a lot of research and even more trial and error.  Finding a link to the driver was extremely helpful and so were the instructions about installing the driver in Vista Service Pack 2 compatibility mode.

Here were the key steps:

1.  Uninstall the printer and the driver from every place imaginable!  You might need to stop the print spooler (in local services within the task manager) in order to delete the printer and driver.  I had multiple copies of partially installed printers--very frustrating!  I used both the Device Manager and the Devices and Printer feature to look for opportunities to delete the device(s) and driver.  The easiest way to delete the drivers was in the File Explorer.

2.  Restart the computer and verify that the printer and driver have been removed.  If not, delete them again, restart the computer, and check again until everything is cleared.  When you finish this step, restart the print spooler if you had stopped it.  Disconnect everything that goes into the printer and disconnect the other ends of those cables, too.  Restart the computer.

3.  Refer to these links for how to download and install the drivers:

http://en.community.dell.com/support-forums/peripherals/f/3528/t/19654131

http://www.dell.com/support/Article/us/en/19/SLN297759/EN?dgc=SM&cid=266887&lid=5563920

http://commweb-ps3.us.dell.com/support-forums/peripherals/f/3528/t/19306947

The directions about using compatibility mode and running the file as Administrator are extremely important!

4.  If the Dell Setup Wizard offers to create the driver directory because it does not already exist, that's a good sign!  Later the Setup Wizard will say that it has completed but that's not the truth.  The next screen will give you an opportunity to do a test print and let Dell monitor your printer usage.  Answer the questions but do not request the test print.  I was never able to get the test print to work at this point.  Next you will see the screen about faxing and scanning.  Close that window and check the Device Manager and Devices and Printers to see the status of the printer.  Make sure that it shows up under Printers and not under Other Devices.  If it says something about installation in progress, that's bad news.  Also, if the setup program gets stuck or you see a Windows application that tells you that your device has not been fully installed, you will need to go back to Step 1.  Another bad omen is a cartridge stall error on the printer.  Otherwise, power down the computer and the printer and do a cold start.

5.  Check again in Devices and Printers and the Device Manager to see that the status of the printer is okay and that it is designated as your default printer.  (Mine reads as Dell Photo AIO Printer 966.)  If so, you'll soon be hearing the familiar sound of your printer cranking out beautiful work for you!

I read that some people used the printer troubleshooter.  I tried it a couple times but the changes it made didn't lead to a successful installation for me.  I saw some posts that said to avoid using the troubleshooter but I also saw a post that said it had worked for two printers for one poster.  For me it seemed that the troubleshooter altered the driver in such a way that my printer wouldn't work, which sent me back to Step 1.

I installed Windows 10 five days ago and getting my printer to communicate with the computer again took up the lion's share of my "spare time."  More than once the inability to use my printer made me regret having upgraded, but as I continued to use Windows 10 and Edge to look for the answers to my problem, I was hoping to find a soluation so that I wouldn't have to uninstall Windows 10.  I hope that what I learned will be of use to other 966 owners and save them some time and frustration!

Disclaimer:  my printer is plugged directly into the USB port, so I am not using a wireless connection.  I also am not using a network printer, so if either of these situations applies to you, you will have to determine how to establish those settings.

Good luck!

Denise
