It's easy enough to do this in code BUT textboxes are NOT placeholders and do not work in the same way. In particular you will need to delete the existing text. The only way to have proper placeholders is have a corresponding layout.

Sub newTB()
Dim osld As Slide
On Error Resume Next
Set osld = ActiveWindow.View.Slide
If Not osld Is Nothing Then
With osld.Shapes.AddTextbox(msoTextOrientationHorizontal, 10, 10, 200, 20)

.Text = "Replace this."
.Font.Size = 12
'more
End With
End With
End If
End Sub
