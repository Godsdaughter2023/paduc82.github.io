Error "The formula in this cell refers to a range that has - Microsoft Community

hughoake say:

Error "The formula in this cell refers to a range that has additional numbers adjacent to it." Formula "=IF(OR(ISBLANK(O13)),"",(SUM(G12:G13)*0.09/365*O13))" what's wrong?

Error "The formula in this cell refers to a range that has additional numbers adjacent to it." Formula "=IF(OR(ISBLANK(O13)),"",(SUM(G12:G13)*0.09/365*O13))" what's wrong?

joeu2004 say:

Perhaps nothing.

I think you are referring to the message that you see when you hover over a colored triangle in a corner of the cell.

Those are warnings, not errors, albeit true that Excel refers to them incorrectly as "error checking".

The message is a warning that the range G12:G13 does not encompass all of the contiguous cells above and/or below the range, which also contain numbers.

So what?!  That might be intentional.  Only you can decide.

If you don't like those warnings (I don't), you can disable them by clicking File (might be something else in Excel 2007) > Options > Formulas, uncheckmark "Enable backround error checking", and click Reset Ignored Errors.

joeu2004 say:

FYI, there is no point to using OR() in your formula.  Simply write IF(ISBLANK(O13),...)

But I think the following is more reliable:

=IF(O13="", "", SUM(G12:G13)*0.09/365 * O13)

ISBLANK() is true only when O13 is empty (no value).  O13="" is true when O13 is empty or its value is the null string (""), like you return in this formula.
