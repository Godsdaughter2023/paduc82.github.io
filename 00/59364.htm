burrwalnut say:

To remove entries for programs/updates that show in Programs and Features (Add or Remove Programs in Windows XP), firstly create a system restore point or backup the registry, just in case you have finger-problems.

 

Now open a Run window (Windows Logo key+R), type regedit, press Enter and in the left pane navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion.

 

Expand the Uninstall key in the left pane and scroll down to find the unwanted entry then right-click and delete it. To make sure you delete the correct entry, check the name in the right pane. DisplayName contains the name that is listed in Programs and Features and UninstallString contains the name of the program (including the full path) that is normally used to do the uninstall.

 

MAKE A NOTE OF THE VALUE of UninstallString so that you can, if you wish, uninstall the program.

I now want to create two logical volumes D and E in that space in DiskMgmt. But when try to create Simple Volume, the system creates a primary partition as D rather than a logical drive. It only creates a logical drive E in the remaining space, but there seems to be no way to create both D and E as logical drives.

I know it is possible because I have 2 logical drives on my other laptop that runs Windows 10, but how?

hairyfool say:

You cannot have two Extended partitions on the other system, you can only have one.

MBR disks can have 3 types of partitions Primary, Extended and Logical but the disk itself can only have four entries in the partition table, only one of which can be an Extended partition.

Extended is a special partition which is in fact just a container and when initially created is empty. You then have to create logical drives within that partition so what you should have is a single extended partition but with two logical drives in it.

Disk Manager is somewhat simplified for general use and a specialist partition manager is often more capable.
