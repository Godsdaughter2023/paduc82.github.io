kosh---01 say:

Hi matandrius,

 

Thanks for your post and feedback.

 

Your E63 will send and receive data as soon as it connect to the internet. If you are not using any internet apps the phone still connects to the internet just to establish the connection. If you do not use the Internet at all you can remove the accesspoint(s) to prevent the phone from setting  up the connection in the first place.

 

To do this, first switch the E63 to offline mode by pressing the power key for a moment and from the popup menu select and activate the offline profile. Then from go to MENU>Tools>Settings>Connection>Access points. Here you will see all available accesspoints. Packet data accesspoints will have a blue icon with two half arrows. Select each and then choose Options>Delete. When all have been removed the E63 will no longer be able to connect to the internet using Packet data. If any Wi-Fi accesspoint have been set up you can still connect using any of these when they are in range.
