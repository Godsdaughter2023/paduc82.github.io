brent-- say: Hi bitosa,

This is more or less by design.  If you never removed the laptop from the docking station, it would remember the settings.  But since you are changing your primary display to the external monitor (not unlike using only an external and closing the lid on the laptop), when you restart your computer out of the dock, it can't find where to display your desktop so it resets to the LCD display on the laptop (versus only getting a blank screen.)

The only real solution that I've found is to use the laptop as the primary display device.  The option to change your primary display is more intended for desktop systems that are never unplugged from the monitor.

Hello,

Thank you for posting your query in Microsoft Community, we appreciate your interest in Windows 10.

I understand the inconvenience with accessing the device after installing Windows 10.

I would like to inform that, these issues are due to un-updated drivers on Windows 10.

I suggest you to follow the below troubleshooting steps by connecting an external mouse and check if it works fine.

If the external mouse is working fine then follow the below:

Method 1:
Step 1:

  1. Press Windows + X keys from the keyboard and click on Control Panel.

  2. Click on Mouse.

  3. On the Device Settings tab of the Mouse Properties screen, click the Disable button to turn off the Touchpad.

  4. Restart the system and after restart enable it.

Step 2: Uninstall Mouse driver.

  1. Press Windows key + X key when you are at desktop.
  2. Select Device Manager.
  3. Expand Mouse.
  4. Right on the Mouse driver and click on Uninstall.

Step 3: Update the driver.

  1. Press Windows key + X key when you are at desktop.
  2. Select Device Manager.
  3. Expand Mouse.
  4. Right on the Mouse driver and click on Update driver software

Also, try to search for the drivers from Manufacturer's website and install them in the compatibility mode.

You can install Windows 8/8.1 drivers in compatibility mode if there are no Windows 10 compatible drivers.

Follow the steps to install in Compatibility mode:

  1. Download the updated driver from the manufacturer’s website.
  2. Right click the driver installation file and select Properties then tap or click the Compatibility tab.
  3. Place a check in the Run this program in compatibility mode for: box.
  4. Tap the drop down box and select a previous operating system, then tap or click OK.
  5. Try installing the driver and check.

Method 2: I suggest you to disable automatic restart on system failure. This prevents Windows from automatically restarting if an error causes Windows to fail. Follow the below steps.

  1. Press Windows + E keys on the keyboard to open File Explorer.

  2. Right click on My Computer/This PC and Select Properties.

  3. Click on Advanced system settings on the left navigation pane. (Enter the admin credentials if prompted)

  4. Click on the Advanced tab.

  5. Under the Startup and Recovery, select Settings.

  6.  Uncheck the option Automatic Restart under System failure.

  7. Click on OK button and close the window.

Try to use the computer now for some time. Check if the computer is restarting now.

Hope it helps. Kindly update the status of the issue, we will be happy to help you.

Failing that, you could always download VueScan which is a universal TWAIN driver that works with Windows 8 and the Canon LIDE 500F.
