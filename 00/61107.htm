The 0x80072EE2 error occurs when the update feature gets prevented by a poor connection, firewall blocking access or system failure. To do so, check the following:

  1. Make sure you have a good Internet connection. A wired connection is preferred so that there's less connection loss.
  2. Turn off Windows Firewall, to isolate that it's not blocking any accesses. To do so, go to Windows Firewall (type Windows Firewall in Cortana search, and press Enter). Click Turn Windows Firewall On or off and click the Public and the Private network settings, then select turn-off.
  3. Run the System File Check (SFC) and Deployment Image Servicing and Management (DISM) in scanning and repairing corrupted files that may have contributed to the concern.

If the issue persists, isolate the error by creating a new account and switch accounts to see if the error will occur.
