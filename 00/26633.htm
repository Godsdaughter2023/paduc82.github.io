vijay a. verma say:

Let's assume that -

1. Sheet1 has data from row 2 to 10000.

2. Your search string is in A1 of Sheet2. (replace $A$1 in formula appropriately, if you have search string somewhere else)

3. Enter the below formula as Array Formula i.e. not by pressing ENTER after entering your formula but by pressing CTRL+SHIFT+ENTER. This will put { } brackets around the formula which you can see in Formula Bar. If you edit again, you will have to do CTRL+SHIFT+ENTER again. Don't put { } manually.

Drag formula down and right.

=TEXT(IFERROR(INDEX(Sheet1!A$2:A$10000,SMALL(IF(IFERROR(SEARCH("*"&$A$1&"*",Sheet1!$A$2:$A$10000),FALSE),ROW($A$2:$A$10000)-ROW($A$2)+1),ROW(1:1))),""),"#")

ken sheridan say:

i modified the query as you suggested.

In your original post you were attempting to filter the report with the WhereCondition argument of a macro.  It appears that you are now trying to restrict the report's underlying query by means of a parameter, in which case:

1.  Remove the WhereCondition from the macro so that it opens the report unfiltered.

2.  Qualify the reference to the control in the query with the form name by entering the following in the criteria row of the [Fecha de Ofrenda] column in query design view:

>= Forms![YourFormNameGoesHere]![ParticularDate]

3.  Declare the parameter in the query as DATETIME.  This can be done in SQL view or via the Parameters dialogue in query design view.

This will translate in SQL to a query along these lines:

PARAMETERS Forms![YourFormNameGoesHere]![ParticularDate] DATETIME;
SELECT *
FROM [YourTableNameGoesHere]
WHERE [Fecha de Ofrenda] >= Forms![YourFormNameGoesHere]![ParticularDate];

shane devenshire say:

1. Highlight all the cells on the rows you want formatted
2. Choose Home, Conditional Formatting, Manage Rules, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following
formula:

=AND(C1=C2,A1="xy")

5. Click the Format button and choose a format.
6. Click OK twice
