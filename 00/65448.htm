Are you saying that you disabled System Restore? If so then this is not a good idea. System Restore would let recover from your current predicament in less than five minutes. The alternative solution is much more painful. It works like so:

  1. Log on under an admin account.
  2. Activate System Restore.
  3. Create a manual restore point.
  4. Run RegEdit.
  5. Click HKey_Local_Machine in the left-hand panel.
  6. Click File / Load Hive
  7. Open this file: C:\Users\{Name of damaged profile}\ntuser.dat
  8. Give the hive a name, e.g. Damaged.
  9. Navigate to HKLM\Damaged\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies
  10. Do your repair.
  11. Right-Click HKLM\Damaged
  12. Click File / Unload Hive
  13. Close regedit.

Thanks in advance to your reply.
lemp say:
NO, you can't do what you are planning to do.

The files available to download called "Windows XP Service Pack 3" are updates to a complete installation of Windows XP.  You may either download an ISO file (or "image") -- which is what your link is -- or an executable file (http://www.microsoft.com/download/en/details.aspx?id=24).

The ISO or image file is used to create a CD by using software capable of doing that, such as (free) ImgBurn.

In either case, what you are getting is the "service pack" and not the complete Windows operating system.

You need a Windows XP installation CD.  These may be "gold" (as initially released) or you may find one that incorporates service pack 1 or service pack 2 or service pack 3.  If you obtain a pre-sp3 install CD, you'll have to update the installation promptly after installing.  You can go from sp 1 directly to sp3 without needing to install sp2, but if you get a "gold" install CD you must first install sp 1 or sp2 before installing sp3.

Because Microsoft stopped selling Windows XP some time ago, genuine installation CDs are getting more difficult to find. 

Although the rep from your computer's manufacturer was correct that you could buy any Windows XP installation CD and install it, the manufacturer was obligated by its license from Microsoft to provide you some method of re-installing Windows when you purchased the computer.  This may have been a hidden restore partition on your hard drive or some CD/DVD supplied with your computer.  I assume that you have lost or damaged whatever was first supplied and the manufacturer is no longer able or willing to replace it with a CD.

In that case you may be seeing a phantom monitor scenario that others have had.  In that case your Ctrl-Alt-Delete prompt would be appearing on the primary monitor (not your screen) and you would have to pretend that you could see it and use only keyboard commands to try to bring it back.   E.g. End and Enter would start it but it might still be on the other screen so then you would have to move it to your screen.  I don't know if Win-Shift-CursorLeft would do it then but that would be the simplest.  Otherwise you would try Alt-Space M and a Cursor key to engage the mouse pointer with the Task Manager Title bar in order to move it back with your mouse or if you were sure of the direction you needed to drag it, continue pressing the Cursor key.  You could get a hint about which direction it might be coming from by moving your pointer all the way to either side in order to try to decide which side the pointer might be passing through.

Alternatively, in the same scenario what should work even before you get a chance to login is to use the Project dialog from the Charms bar, e.g. press Win-P CursorUp, Enter (as if you could see the Project dialog and switch from Extend to Duplicate).  There would be two main consequences of that procedure.  First it would refresh the video system and it would cause the current display to be rendered on your monitor.  In some cases I can "hear" the reset as well as see it, so it would help to be extra observant when trying this.
