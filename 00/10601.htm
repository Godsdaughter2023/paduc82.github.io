Rounding function vs changing number of decimal places - Microsoft Community

michelleaudain say:

Rounding function vs changing number of decimal places

What is the difference between using the Rounding function and changing the number of decimal places?
teylyn say:

Kris Ver, changing the number of decimal places with formatting will NEVER produce a different value of a number. A different value is only created when a different value is entered into the cell.

MichelleAudain, changing the number of decimal places will change the display of the current cell. Rounding will put a new value into a different cell and  as a result of rounding, the rounded number may be different.

In the following screenshot, the value in D1 has been copied to D2. D2 has then been formatted to reduce the decimal points to 2. See the display of D2 and in the formula bar you can see the actual value of D2. The values are identical.

In D3, the Round() function has been used to round the value of D1 to show only 2 decimals. The result in D3 looks the same as D2, but the underlying value is not same as D1 or D2.

This can be very important when using the cells for further calculations. If you calculate based on D2, the calculation will take into account all five decimals, not just the two visible ones. For example, if you calculate 10% tax on D2 and D3, the result will be different.
