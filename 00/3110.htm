andy pope say:

You can use code to report all chart objects in a workbook to the Immediate window.

Sub ReportChartObjects()

    Dim objCht As ChartObject
    Dim objSheet As Object
   
    For Each objSheet In ActiveWorkbook.Sheets
        For Each objCht In objSheet.ChartObjects
            Debug.Print objCht.Name; " on "; objSheet.Name;
            If TypeOf objSheet Is Worksheet Then
                Debug.Print "!" & objCht.TopLeftCell.Address & ":" & _
                                  objCht.BottomRightCell.Address
            Else
                Debug.Print
            End If
        Next
    Next

End Sub

 

Or on a worksheet you can use Goto (CTRL+G) > Special... > Objects. The TAB key will cycle through selected objects.

This Goto/Special will also select objects that are not charts.

Hi,

first you can check how many charts are in the active worksheet:

  • press Alt+F11 to go to the VBA-Editor
  • press Ctrl+G to open the Immediate window
  • enter the following code: ?activesheet.shapes.count
  • press enter while in code line

If you get a return value greater than 0 then:

  • enter the following code in Immediate window: activesheet.shapes(1).select
  • and than: ?selection.topleftcell.address
  • copy the returned address

Now you can switch back to Excel view (Alt+F11) and paste the returned address in the name box. Press Enter.

Repeat this procedure for each worksheet and existing shape.
