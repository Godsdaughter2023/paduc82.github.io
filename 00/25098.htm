    Additionally, the object must always appear entirely on the same 'page' as the line of a paragraph into which it's inserted as

    hansv mvp say:

    The text box part of a combo box displays a single column only - the one indicated by the TextColumn property. The default value of this property is -1, meaning that the first non-hidden column will be displayed. There is no way to make the text box part of a combo box display multiple columns.

     

    You could use concatenation formulas in the list fill range of the combo box to combine the values of two other columns, e.g. =A2&" "&B2, and make the combo box display the column with these formulas.
