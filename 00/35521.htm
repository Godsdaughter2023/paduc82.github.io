canadian tech say:

One common mistake people make is opening windows full screen. 

 

There are three "buttons" on the top right of any window.  From the right:

  1. Close this window --- an X

  2. Toggle this window either as a window or full screen

  3. Minimize the window -- a dash sign

The middle one is the guy you need to concern yourself with.

If it is a single square, it means that this window is displayed as a window and is NOT full screen

If it is a double square, it means this window is displayed full screen -- it fills every bit of your screen.

 

If the button is a single square and the window still fills the entire screen, that means someone expanded the window that way.  You can change the shape of any window by moving your mouse pointer to any edge or corner and holding the button down while dragging the edge to where you want it to be.  This way you can shape any window in any rectangular shape.

 

Now once you have real windows (less than full screen filled) you can move them to any position you want by clicking on the top border of that window and dragging it to where you want it to be.

 

There is a windows setting that will cause a window to change shape in a regulated way if you drag it past the edge of the screen.  That is set that way by default, but that behavior can be changed.

Remove xbox by;

-Go to Start menu,

-Select all apps, 

-Scroll to "Windows PowerShell"

-Click the down arrow

-Scroll to "Windows PowerShell ISE" and RIGHT click on it

-Select "Run as an Administrator".

The Powershell window will open then copy the complete and whole line below and paste it into the PowerShell window.

get-appxpackage *xbox* | remove-appxpackage 

-Press enter

Goodbye  Xbox

Oh and to remove Mail and Calendar its;

get-appxpackage *microsoft.windowscommunicationsapps* | remove-appxpackage

Goodbye Mail
