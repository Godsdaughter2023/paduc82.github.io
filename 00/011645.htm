ken sheridan say: Firstly, data should only be entered via a form, never directly into a table's raw datasheet view.  A form can be in datasheet view if you wish.  In the Clause control's AfterUpdate event procedure you can then assign the value to the Hour control with:

Select Case Me.Clause
    Case "C"
    Me.Hour = 72
    Case "A"
    Me.Hour = 48
    Case Else
    Me.Hour = Null
End Select

If your clause are standardized then why not use a combo box?

Select Case Me.Clause

Don't forget error handling.

mike h.. say:

Hi,

 

If we do it this way we don't need to bother about the colour because it will sort non-coloured rows to the top. This also now sorts blanks in Col C to the bottom. To run it you can create a shortcut

 

Developer tab|Macros|Hightlight the macro|Options and enter a key value (say) M and OK out. Now the macro will run if you click CTRL+M

 

 

Sub Sort_Grey()
Dim LastRow As Long
LastRow = Cells(Cells.Rows.Count, "C").End(xlUp).Row
    ActiveWorkbook.ActiveSheet.Sort.SortFields.Clear
    ActiveWorkbook.ActiveSheet.Sort.SortFields.Add Key:=Range("C6:C" & LastRow) _
        , SortOn:=xlSortOnValues, Order:=xlAscending, DataOption:=xlSortNormal
    ActiveWorkbook.ActiveSheet.Sort.SortFields.Add Key:=Range("C6:C" & LastRow) _
        , SortOn:=xlSortOnCellColor, Order:=xlAscending, DataOption:=xlSortNormal
    With ActiveWorkbook.ActiveSheet.Sort
        .SetRange Range("B5:G" & LastRow)
        .Header = xlYes
        .MatchCase = False
        .Orientation = xlTopToBottom
        .SortMethod = xlPinYin
        .Apply
    End With
End Sub

ashley s. say:

This can happen if the High Contrast Black color scheme is used in Windows XP, Vista or Windows 7 and you have wrap text enabled in your cell properties in Excel 2007 or Excel 2010.

 

To display the printed text you should select a different high contrast color scheme or change the font colors for the text in the formatted cells to a darker color.

2486767 – Previewed or printed Excel 2007 or 2010 worksheets may not display text when the computer is set to use the Microsoft Windows High Contrast Black color scheme
