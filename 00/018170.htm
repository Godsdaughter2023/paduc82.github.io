jim thomlinson say:

Define Smooth. My guess is if you have something that is intended to result in a curve then you want a polynomial regression. You can generate one by adding a trend line to your graph and selecting polynomial. Togenerate the data points that the grph is depicting you can use the trend function. Assuming you have th numbers 1-12 in cells A2:A13 and your data points in cells B2:B13 then this formula should work.

Select Cells C2:C13. Type this formula:

=TREND($B$2:$B$13, $A$2:$A$13^{1,2})

You have to commit the formula with Shift + Ctrl + Enter as it is an arry type formula.

don cuthbert say:

Since your BCD looks right, try running the following from Administrator Command Prompt and see what happens:

bcdboot F:\Windows

bcdboot C:\Windows

Those commands should recreate the System files on your system partition. Running them in the order above will set the F: as default OS, followed by switching the default OS back to C: when you run it again.

I am wondering if you might have some older Windows version System files, such as Windows 7. This should replace them with your current Windows 10 System files.

Because you are running two hard drives, in Legacy BIOS mode, it is probably using whichever one is currently connected as drive 0 for the System files.
