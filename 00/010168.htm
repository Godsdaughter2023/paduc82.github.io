You need 2 things to install Office, your unique 25 character Product Key and the generic installation file/disk for the matching Office bundle.  All of the consumer licenses allow you to re-install Office on the same computer.

1.      Your original package or purchase confirmation email

How to update Internet Explorer to IE 10? - Microsoft Community

a. user say:

How to update Internet Explorer to IE 10?

I recently reinstalled my Windows 7 Ultimate 64 bit. Earlier my version of IE was IE 10. Now it is IE 8. So, how can I update IE 8 to IE 10?

jetta48 say:

Hi

 

You can do it either by using Windows Update (which will also make sure your system is up to date) or by downloading a standalone installer for IE10. Having SP1 for Windows 7 installed is a pre-requisite for IE10.

 

First backup your data in case something goes wrong.

 

To do it using Windows Update:

1. Click Start->All Programs->Windows Update.

2. Click on Change Settings.

3. Make sure that for Important Updates you are set to Install updates automatically (recommended).

4. Click OK.

5. Now check for updates and install. You may have to do this multiple times and restart your computer more than once. Eventually IE10 will get installed.

 

Alternatively, to do it using a standalone installer go to:

http://windows.microsoft.com/en-us/internet-explorer/ie-10-worldwide-languages

to download and install the version you want/need.

 

Further info may be found here:

http://windows.microsoft.com/en-gb/internet-explorer/install-ie#ie=ie-10-win-7
