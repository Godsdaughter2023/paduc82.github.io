marilou ser say:

Hi Bob,

You might be getting this error because more than one computer is using the same IP address, which is resulting in a conflict. If you are setting IP addresses manually using static IP addresses, it is important to check that each network device has a unique IP address.

If your router automatically assigns IP addresses using DHCP, you can try releasing and renewing your computer's IP assignment by following the steps below:

  1. Click Start.
  2. Type cmd in the search box, and then press Enter.
  3. At the command prompt, run the following commands in the listed order:
    • Type ipconfig /release and press Enter.
    • Type ipconfig /renew and press Enter.

We also recommend that you check your router settings for any conflicts in the DHCP settings. Additionally, check the manufacturer's website or support page for an updated driver for your router.

Update us with the outcome and if you need further assistance.
