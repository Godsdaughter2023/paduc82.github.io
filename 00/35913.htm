Creating a New Library for file extensions - Microsoft Community

steve mc209 say:

Creating a New Library for file extensions

I'd like to create a new library for PDF files. I added a new PDF library, but I can't figure out how to have the library show all .pdf extensions.

Is that possible, and if so, how is it done ?

Thanks,

Steve

cmdrkeene say:

Libraries don't do this, at least not directly.  Although they are very similar to the technology that does, which is "search folders" (first introduced in Vista but somewhat modified in Windows 7).

Anyway...

Libraries really just "gather contents" from several locations. So your Documents library by default looks in two places, and shows everything it finds in one combined view for easy access of all documents, regardless of where they really are stored.

Now if you want to have a library that looks inside your Documents folder, but only shows the PDF files, you can certainly do that, but not directly with a library by itself.

---

To do this, first open your Document library, type *.pdf into the search box so that you're seeing all the PDF files in and under the Document library.  Take a moment to set the style you like (group by date modified or author, sort by modified timestamp, whatever you like) and set the icon size you like.  Then click "save search" on the toolbar.  This search will be indexed and act like a library.  As new PDF files show up in the computer, this search will automatically pick them up and show them, like a "live folder" that only shows PDFs.
