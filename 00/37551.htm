Hi,

If you want to set intervals of your choice, you can do it by modifying the registry. Please follow the below steps to set the slideshow intervals:

1. Open Run and type regedit.exe

2. Navigate to:
HKEY_CURRENT_USER\Control Panel\Personalization\Desktop Slideshow

In here, the two important values are the Interval, and Shuffle.

 Double click on Shuffle, and change the value to 1. OK.

Then change the Interval by double clicking on it. You'll first want to change the Base to Decimal, rather than Hexadecimal (otherwise you'll get unexpected interval values). Then change the value to your desired time, in milliseconds (10000ms for 10 seconds).

You can exit regedit now. The final step is to restart Explorer. You can do this through Task Manager if you know how, otherwise simply restart your computer.

[tech support supa foodin] say:

@NormanAcunis

Well, your most in-expensive option would be to simply buy a recover disc that goes to your version of Windows whatever that might be—7, 8, or 8.1. Your activation key should be embedded in the bios of your computer, so that you'll be able to activate Windows offline by phone. You can get these recovery installation discs very cheap from a number of online retailers. They will say, "Windows 8 recovery disc [repair, reinstall, etc]".

From there, re-install that OS, activate it (via phone for example), and then you can use the installation disc you created from my method to jump straight to Windows 10 from there. Just run "setup.exe" from the File Explorer. You don't have to run any Windows updates or anything. It's just recommended that you check your motherboard manufacturer's website for a bios and chipset update, and update those after you reinstall, but before you upgrade to Windows 10.

@UserZQ

Personally, I couldn't tell you because I have no experience with OEM partition recovery and upgrade to Windows 10. However, in theory it should work perfectly fine. You should be able to restore from that partition, and then use the installation disc you created with my method to jump straight into Windows 10. Just make sure Windows is activated first (which it should be by default).

In some heavy cases, the OEM partition of your computer could get corrupted, and if this is the case, you'll have to purge your drive totally by using DISKPART and CLEAN—booting from your Windows installation disc, then going into "Repair your computer", and then navigating the repair menu to find the Command Prompt option.

From there, just run these commands.

diskpart

list volume

select volume #—enter the number listed as your hard drive

clean

You'll need to reinstall the original version of Windows fresh from a recovery disc afterwards. And once it's activated you'll be able to upgrade straight to Windows 10 without doing anything else. So long as Windows is activated, and you're running "setup.exe" from the File Explorer, you should retain your activated status despite whatever your means of recovery you used.
