jessen p say:

Hi,

Sorry for the inconvenience.

If you upgraded to Windows 10 from Windows 8.1 some files used by your previous version of Windows will be left in the Windows.old folder. These files will be automatically removed within 28 days of upgrading.

You can delete the Windows.old folder using the RD Command.

a: Press Windows key + X.

b: Then click Command prompt(admin)

c: Type the following command and hit enter:

RD /S /Q %SystemDrive%\windows.old
