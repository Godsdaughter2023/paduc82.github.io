Windows does not support verifying which program created a task bar icon.  However, you can verify that the MSE UI is running and is a legitmate product by running Task Manager (Right click on task bar, click "Start Task Manager".  Change to the "Processes" tab and click "Show Processes from All Users".  Find the process "msseces.exe", right click, and select "Properties".  In the "Digital Signatures" tab you will find the name of the digital signature.  For MSE it should be "Microsoft Corporation".
