steve winograd say:

Disable and then re-enable Internet Connection Sharing on the Windows 7 computer's Internet connection.  If it asks you to select a private network connection, select the one that connects to the hub.  Then check the LAN connections on each computer:

Windows 7:

  1. Press WindowsKey+R .
  2. Type ncpa.cpl in the box and press Enter .
  3. Right-click the LAN connection and click Status .
  4. Click Details .  It should show these values:
  • IPv4 Address: 192.168.137.1
  • IPv4 Subnet Mask: 255.255.255.0

Windows XP:

  1. Press WindowsKey+R .
  2. Type ncpa.cpl in the box and press Enter .
  3. Right-click the LAN connection and click Status .
  4. Click Support .
  5. Click Details .  It should show these values:
  • IP Address: 192.168.137.x (1<x<255)
  • Subnet Mask: 255.255.255.0
  • Default Gateway: 192.168.137.1
  • DHCP Server: 192.168.137.1
  • DNS Servers: 192.168.137.1

If the Windows 7 values aren't right, Internet Connection Sharing isn't properly configured.

If the Windows XP values aren't right, there might be connection problems between the two computers and the hub.

I figured out the solution.  You have to preconfigure the designated user account for your kiosk before you create the kiosk. The user for the kiosk does not follow the rules of any other accounts so even if you configure the admin account, the user for the kiosk will have different settings.

For example, let's say you create user 1 (a local account):

  1. You log into user 1 and then go to Windows Settings > Devices > Typing
  2. ENABLE "Show the touch keyboard when not in tablet mode and there's no keyboard attached" option.
  3. log out of user 1
  4. go back to a normal account (e.g., the admin account)
  5. create a kiosk and select the user 1 local account
  6. log into user 1 which will go automatically to the designated assigned access app (in my case, Skype)
  7. when you tap a text field, the touch keyboard will show up

thrashzone say:

Click the start button and click on your user name listed near the top,

If the my documents folder is listed here,

Massive resetting of the hidden attributes can be done by right clicking the (my documents) listing on the folder tree, and select properties, general tab section, check the hidden box and click on apply, this will change all of the files to hidden, click yes, then uncheck the hidden box and apply, it will remove the hidden attribute from all of your files, click on yes, then save.

This work for my pictures.......

Also if you right click the main libraries listing on the folders tree, and select restore default libraries.

Look at the top of the folders tree and right click the favorites listing, and select restore default favorites links.
