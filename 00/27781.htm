ossiemac say:
The formulas will need to pickup values from the Data Input sheet relative to the current cell and populate that cell. For example in Sheet2:B23 would be this formula:

My understanding of this is that your formula as above would be in Sheet2 B23 and if the formula was in B24 then it would be like the following where row 11 becomes row 12.

=B$17/(('Data Input'!C12*(1/'Data Input'!D12))+('Data Input'!C12*'Data Input'!E12))

Is this not what you mean or maybe you are not understanding the process I suggested so I will explain some more.

 

The method of recording that I gave you converts the formula into relative addressing and the resultant string can be copied and saved in a constant at the top of a standard module and then used throughtout the worksheet where it addresses the Data Input sheet relative to the cell into which it is copied. Recording is simply to get the syntax for the constant and only has to be done once for each formula.

 

Recording the formula as provided in your original post as it is inserted into cell B23 returns the following line of code.

ActiveCell.FormulaR1C1 = _
    "=R17C/(('Data Input'!R[-12]C[1]*(1/'Data Input'!R[-12]C[2]))+('Data Input'!R[-12]C[1]*'Data Input'!R[-12]C[3]))"

Copy the string enclosed in the double quotes and create a constant as follows. (Unfortunately in this post the line breaks here but just bring it up into one line.)

Const MyFormula_1 = "=R17C/(('Data Input'!R[-12]C[1]*(1/'Data Input'!R[-12]C[2]))+('Data Input'!R[-12]C[1]*'Data Input'!R[-12]C[3]))"

 

Then you can use that constant to insert the relative formula into another cell like the following.

 

Sheets("Sheet2").Range("B24").FormulaR1C1 = MyFormula_1

 

If you then check the cell B24 you will see that it has converted the relative addressing to the following where the row has now become 12 in lieu of 11.

=B$17/(('Data Input'!C12*(1/'Data Input'!D12))+('Data Input'!C12*'Data Input'!E12))

 

If this is not what you are attempting to achieve then I will need more explanation with perhaps some examples.

scorsone say:

I'm building and using spreadsheets on Excel 2010 as that's what my employer has.  Things are well, no complaints.  I'm tasked with updating a Miscellaneous Revenue spreadsheet which has gotten old as I let it go for a few days and then have to remember to go back and up date it.  I already enter the information elsewhere in another spreadsheet, the Tax Spreadsheet - it's become more than just tax related; more a free for all.  I've already linked the two together so that when I now open the Miscellaneous Revenue spreadsheet it'll update from the Tax Spreadsheet and I won't have to spend time with it.  

However I've got seven columns by thirty-one rows, one row for each day of the month, that now contain 0.00 or a numerical value.  Being all full of 0.00's, it can make it hard for the numerical values to stand out even if they are totaled by category and month.  I was hoping to use Conditional Formatting to hide all the 0.00's leaving only the numerical values visible when they occur but can't figure out how as I've not used Conditional Formatting but once or twice and it's changed since last I used it which was considerably long ago...

hansv mvp say:

A number format can contain up to four sections:

format_for_positive;format_for_negative;format_for_zero;format_for_text

It is allowed to omit sections at the end, but if a section is present but empty, the corresponding values will be hidden. In   0.00;-0.00;  the format for positive numbers is 0.00, that for negative numbers is -0.00 and that for zero values is empty since there is a ; but nothing after it, so zero values will be hidden. The format for text is not specified.

If you apply the custom format   ;;;   (three consecutive semicolons) to a range of cells, the value of those cells will be hidden regardless of its type, since all four sections of the format are empty.

hansv mvp say:

Option 1: for an entire worksheet

  • Make sure that the sheet you want to do this for is active.
  • Select File > Options.
  • Select Advanced in the navigation pane on the left hand side.
  • Scroll down to the section 'Display options for this worksheet'.
  • Clear the check box 'Show a zero in cells that have zero value'.
  • Click OK.
  • This will hide all zero values on the entire sheet. The setting will be saved with the workbook.
  • Repeat for other sheets, if desired.

Option 2: for a specific range only

  • Select the range in which you want to hide zeros.
  • Click the little arrow in the lower right corner of the Number group of the Home tab of the ribbon.
  • Alternatively, press Ctrl+1 then activate the Number tab.
  • Select Custom.
  • Click in the Type box, remove what's there and enter (for example) 0.00;-0.00;
  • The second ; is essential, because there is nothing after it, zero values will be hidden.
  • Instead of 0.00 you can use another format, e.g. $#,##0.00
  • Click OK.
