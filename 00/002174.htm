    alan wu msft say:

    Hi Sunny,

    Greetings. This is Alan Wu from Office 365 forum support. Very glad to reply to your case here. 

    Generally, we can refer to the following steps to remove the AAD Connect. 

    1. Download Azure Active Directory PowerShell Module from following location. Complete the installation.
    http://connect.microsoft.com/site1164/Downloads/DownloadDetails.aspx?DownloadID=59185

    2. Click to open the PowerShell using the shortcut created by installation in previous step.

    3. Run the following command to connect to the Azure Active Directory of the tenant for which you are disabling AD Sync.
    Connect-MsolService

    4. Enter the Global Admin credentials for Azure AD/Office 365 tenant.

    5.Run the following command to disable the Directory Sync.
    Set-MsolDirSyncEnabled –EnableDirSync $false

    6. Run the following command to verify if Dir sync is disabled.

    (Get-MSOLCompanyInformation).DirectorySynchronizationEnabled

    7. Go to your AAD connect server and uninstall Microsoft Azure AD Connect using Uninstall Program in Control panel. 

    If you still got a problem, please clarify which step you got stuck and provide a screenshot about the error as Bobdonley said if you got one so that we can help check it.
