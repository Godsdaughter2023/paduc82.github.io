a. user say:

Hi Lyn,

Thank you for posting your query on Microsoft Community.

I understand that you would like to clear the update history list. I will be glad to assist you with your issue.

Steps to erase Windows Update history

a. Right click on the start button and select Command prompt (admin).

b. Copy/paste the below command:

    net stop wuauserv

c. And select OK.

d. Wait until confirmation that the service is stopped.

e. To delete the file, copy the following command in the command prompt:

    del %systemroot%\SoftwareDistribution\DataStore\Logs\edb.log

f. You can also delete it manually by going to the location in windows explorer.

g. you can restart the service with this command:

    net start wuauserv 

Note: Erasing the windows update history will just remove the entries of the updates and not really the updates itself.
