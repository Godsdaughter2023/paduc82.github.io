rick rothstein, mvp - excel say:

The following macro will delete all objects from all worksheets in the active workbook.

If you store the macro in your personal macro workbook Personal.xlsb, you can run it whenever you want.

 

Sub DeleteAllObjects()
    Dim wsh As Worksheet
    Dim i As Long
    Application.ScreenUpdating = False
    For Each wsh In ActiveWorkbook.Worksheets
        For i = wsh.Shapes.Count To 1 Step -1
            wsh.Shapes(i).Delete
        Next i
    Next wsh
    Application.ScreenUpdating = True
End Sub


Instead of iterating backwards through the collection's index numbers, you could also continually delete the first item in the collection until there are no more...

Sub DeleteAllShapes()
  Dim WS As Worksheet
  Application.ScreenUpdating = False
  For Each WS In ActiveWorkbook.Worksheets
    Do While WS.Shapes.Count
      WS.Shapes(1).Delete
    Loop
  Next
  Application.ScreenUpdating = True
End Sub

By the way, I left your ScreenUpdating statements in my code, but I am not all that convinced they are doing anything all that noteworthy for this code.
