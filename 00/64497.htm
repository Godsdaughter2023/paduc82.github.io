bossdweeb say:

"I looked at many posts saying the available memory may get reduced if I add this additional graphics memory. "

This should only happen if your are using a laptop or PC with Integrated Graphics which 'shares'

system RAM with the Graphics accelerator, in which case the maximum limit for your motherboard

may be only 1GB  anyway  ( the setting for graphics 'shared' system RAM  is found in BIOS).

- With the current Nvidia series, the GTX750ti could be regarded as a close replacement for the GTX560ti, with the added  benefit of very low power usage, and the extra 1GB VRAM.

Only 300W minimum power supply is required for the GTX750ti.

- However, for about the same money you could get a GTX660, which needs a minimum 450W power supply, but performs better than the GTX560 ti and GTX750ti.

- Also check the reviews for the AMD R7 265 and R9 270, and compare performance and price

with the Nvidia cards in your region.

Having more than 1GB VRAM can be useful with some newer AAA titles to run max settings @1920X1080, particularly if wanting to use Anti-Aliasing settings higher than 2X. 

Only a handful of games would use much more than about 1.30 GB VRAM 

eg.-  Crysis 2 uses 1.8GB with my set-up.

- As Mike mentioned... 2GB VRAM does not always equate to a faster card. Some low end cards 

come with 2GB VRAM (as a gimick), but they can't process that much data effectively.

Do read some reviews with gaming benchmarks.

- GTA4 can use a bit more than 1GB VRAM, but you really do need 64bit Windows.

While even heavy system RAM use games don't go much over 4GB system RAM, currently you are

not even  getting  the full benefit of the 4GB, due to running Win7 32bit.

btw - GTA4 has bad Aliasing (one of several issues caused by being poorly ported from console to PC),

but this can be fixed by using FXAA in Nvidia control panel (MLAA if using AMD card).

Other Anti-aliasing options have no effect on aliasing in GTA4.

Hi Marc,

This is usually caused by driver incompatibility or a corrupted system file and a malicious software may also have this kind of effect. Let us follow these methods in troubleshooting the said concern.

Method 1: Troubleshooter

Please run the the following diagnostic tools first as a start. It automatically detects and fixes most incompatibilities in your system.

  • Troubleshoot problems with devices and printers
  • Find and fix problems with devices and hardware

  1. Go to Control Panel, on the Search Control Panel bar located on the upper-right corner of the Control Panel window, type Troubleshooting.
  2. Click Troubleshooting on the results pane.
  3. On the left pane, click View all.
  4. Look for Hardware and Devices, right-click, and have it Run as administrator.
  5. Follow the onscreen instructions.

Method 2: SFC

Please run the System File Checker tool as well to be able to detect and fix concerns with your entire system files. You may refer to this article: Use the System File Checker tool to repair missing or corrupted system files.

Method 3: Update Drivers

Now, to make sure everything is synchronized, please update the drivers needed for this process to complete. Here are the steps:

  1. Go to the Cortana search bar, then type Device Manager. Hit Enter.
  2. ExpandKeyboards, look for your keyboard, right-click and then select Properties.
  3. Navigate to Driver tab, then click Update Driver.
  4. Select: Search automatically for updated driver software.
  5. Follow the onscreen instructions.
  6. Check if concern still persists.

You can also get information from this article: Update drivers in Windows 10.

Method 4: Clean boot

  1. Type msconfig on the Cortana search bar beside Start. Press Enter.
  2. A new window should appear, go to Services tab.
  3. On the bottom left corner, put a check on Hide all Microsoft services.
  4. Click Disable all.
  5. Go to Startup tab, then click Open Task Manager.
  6. A new dialog box would appear. Click each item one by one, then click Disable at the bottom for each item.
  7. Close Task Manager. Then on the Startup tab of msconfig, click OK, and then restart the computer.
  8. After the computer restarts, check Scroll down and look for How to determine what is causing the problem after you do a clean boot.

Hope this helps. If you have any other concerns, don't hesitate to come back to us.
