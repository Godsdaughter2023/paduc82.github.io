john korchok say:

I'm not sure why you want to add Format Painter to the QAT. It's already on the Home tab, just to the right of the Paste icon.

The command you're looking can be added to the Ribbon:

  1. Select the Ribbon pane in Preferences>Ribbon & Toolbar.
  2. Change Choose commands from: to All Commands.
  3. Scroll down and select Format.
  4. In the right-hand Customize the Ribbon: list, click on the plus sign at the bottom and either add a new custom group to an existing tab, or a new tab to the Ribbon.
  5. Between the 2 lists, click on the right-pointing arrow button to add the command to the new group or tab.

If your aim is to have the Format Painter visible at all times (the only reason I can think that you might want to add it to the QAT), then add the command to a custom group on every tab of the Ribbon.

jim gordon mvp say:
Word 2011 supports digital signatures in two different ways.

VBA Macro
A macro is a bit of computer code that does stuff. Word documents, templates, and add-ins can contain VBA code. A developer can purchase a digital code signing certificate (fairly expensive) and use it to guarantee the authenticity of the computer code associated with the document.

Email Certificate
An affordable email certificate can be used to guarantee that a document was sent by the person who signed the email message using a digital certificate. This guarantees the authenticity of the sender of the document and the email message that contained the Word document. It's an indirect way of authenticating a document, but the only real practical digital certificate in existence. You save the email so that if you need to prove authenticity, you can.

Something you should never do is to send or accept a picture of a signature within a document as evidence of authenticity. Pictures of signatures can be saved as files with a simple right-click and Save As Picture. Then that signature can be added to ANY document. It's an open invitation to fraud and identity theft.

jan de messemaeker - project mvp say:

Hi,

The only way to do it is through a VBA procedure. It's simple enough but it is necessary.

Supposing the sqm are in Number1 and the sqm/hr in Number2, here's the code:

Sub TilingDuration

dim T as task

for each t in activeproject.tasks

if not t is nothing then

if not t.number2=0 then

t.duration=60*t.number1/t.number2

end if

end if

next t

end sub
