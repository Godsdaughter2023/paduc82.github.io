This clears the memory cache which can fill up after extended periods of use or because of numerous updates. This process fixes slow load times, black screen hangups, etc.

3. Installations aren't just limited by HD speed - when a game installs, it also downloads all updates for the game if you accept when prompted. If you reject the updates, the install shouldn't take long, but if you accept, you're installing from disc and downloading and installing over the internet, which can take considerable time depending on your internet speed. If you reject updates and your game is still taking over an hour to install, there could be a problem with your hard drive that needs service.
