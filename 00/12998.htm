C'mon!

 
john mcghie say:

Couple of points:

1)  Turning change tracking off has no effect on change tracking that is already in the document.

2)  If you use change tracking, you should always "Accept All" before sending the document anywhere in any case. Anything else is not safe :-)

So in this case, you need to go to your template and turn change tracking off, then Accept All.  You then need to save and close the document (changes are not removed from the document until the file has been closed: in case you want to "undo" any).

After that, when you do a File>New>From Template there should be no change tracking active.

However: check the status of Tools>Protect Document.  If the document has been protected for Tracked Changes with a password, you will not be able to turn change tracking off unless you have the password.

Hope this helps
