graham mayor say:

Pressing CTRL+X cuts the selected text to the clipboard. If you then tell the application to clear the clipboard then that's exactly what happens. The text is gone from the source document and deleted from the clipboard.

If when you closed the original document, you also saved it, then that text is gone, unless you have the option to automatically save a backup copy set, when the backup will contain the text.

If the original document has not been closed or saved and is still open, then CTRL+Z should undo the change.

If you cleared the clipboard and saved the document you had cut it from and there is no backup option set, then the text is gone.

If when you closed the original you didn't save it, then re-open the original and the document will be as it was the last time you saved it.

CTRL+X should be used with care CTRL+C (copy) is a much safer option.
