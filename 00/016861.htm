jllatham say:

Here is a User Defined Function that would do the job.  I've set it up so that you pass it the range of cells to be examined along with the 'instance' number you want to find.  Including the range of cells makes it a little more versatile - you could examine other rows or even columns.

Here is how you would write it as a formula in a cell for your current need:

=FindNthPopulated(DW55:FW55,17)

it will actually appear in all lowercase letters in the cell.

Here is the code for it.  You put it into your workbook just like regular code into a regular module, and you can find instructions for doing that here:

http://www.contextures.com/xlvba01.html#videoreg

Text instructions right below that.

Be sure to save the workbook as a macro enabled workbook, type .xlsm or .xlsb

Function FindNthPopulated(whatCells As Range, nCount As Long) As Variant
  Dim foundCount As Long
  Dim anyCell As Range
  If nCount < 1 Or nCount > whatCells.Cells.Count Then
    FindNthPopulated = "#Invalid Count!"
    Exit Function
  End If
  For Each anyCell In whatCells
    If Not IsEmpty(anyCell) Then
      foundCount = foundCount + 1
      If foundCount = nCount Then
        FindNthPopulated = anyCell.Value
        Exit Function
      End If
    End If
  Next
  'if you get here, there weren't enough populated cells
  FindNthPopulated = "#Not Found!" ' you could change this to other text or a number.
End Function
