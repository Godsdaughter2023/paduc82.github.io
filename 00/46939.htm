how to merge two logical partitions - Microsoft Community

dhananjay11 say:

how to merge two logical partitions

how to merge two logical partitions without formatting , I would like to merger my c drive and d drive and make one
burrwalnut say:

First of all your terminology is slightly awry, a logical drive is part of an extended partition. Windows cannot be installed on a logical drive, it has to be a primary partition, so your C drive is a primary drive, not a logical one.

Using the built-in tools you cannot merge C and D, however, what you can do is copy the contents of D to C, delete D then extend C, as follows:

1. Copy the contents of D to C (presumably this is personal data, e.g. videos, music and the like (you should also make an offline copy of this data).
2. Delete the existing D partition, by right-clicking Computer > Manage > Storage > Disk Management, then right-click the graphic of D and select Delete Volume. The partition returns to Unallocated.
3. Right-click the graphic of C and choose Extend Volume.
