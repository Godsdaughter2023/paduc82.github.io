The question is whether or not you saved and restored the .PST file containing your contacts, etc. Your post would indicate that you originally had more than one of these, or that you're using the default .PST file that Outlook 2010 created when you installed it and have not reconnected the one you originally backed up. A General Starting point when Word does something unexpected

As a general rule, when something seems to be going wrong, or at least not the way you want it to go, in Word “menu’ go to Tools and then Options, in Word “ribbon” go to File then Options, and study all of the various options, both turned on and off.

kergga say:

Assuming that your data is in columns A through C:

 

For weekday calls before 8am use:

=SUMPRODUCT((WEEKDAY(A:A)>1)*1,(WEEKDAY(A:A)<7)*1, (C:C="Wish to make a booking")*1,(B:B<0.33)*1)

For weekday calls after 6pm use:

=SUMPRODUCT((WEEKDAY(A:A)>1)*1,(WEEKDAY(A:A)<7)*1, (C:C="Wish to make a booking")*1,(B:B>0.75)*1)

Just add them together if you don't want to use helper columns, e.g.:

=SUMPRODUCT((WEEKDAY(A:A)>1)*1,(WEEKDAY(A:A)<7)*1, (C:C="Query on Existing Booking")*1,(B:B<0.33)*1) + SUMPRODUCT((WEEKDAY(A:A)>1)*1,(WEEKDAY(A:A)<7)*1, (C:C="Query on Existing Booking")*1,(B:B>0.75)*1)

My PC is set up so that Weekday 1 = Sunday, but the results of this formula will vary if it is calculated on a PC that has a different day-of-week assignment. To test, pick a cell and enter =weekday(today()). If you do this today (Wednesday) and the result is 4, then your PC is set up the same as mine.

For your second piece, looking at weekends only-  just change the weekday parameter. In my case, I would set the first one to <2 (to get 1, which is Sunday) and the second one to >6 (to get 7, which is Saturday).

Times in Excel are expressed as a decimal fraction of the day; .5 is noon, .25 is 6am, .75 is 6 pm, and so on. You can enter any target time (7pm and 9am in your second part) and then reformat that cell as a number to see the decimal value.

Note that these formulas are based on exactly what you requested, but I can't help but wonder if you might want to treat things like Monday (weekday) prior to 8am the same as a weekend 'bin'. You can adjust this formula (which days of the week, and which times are being included) based on your actual needs.

HTH,

Keith
