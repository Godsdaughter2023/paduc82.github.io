Bookmark in access - Microsoft Community

hamid-nice say:

Bookmark in access

Hi
Please explain about Bookmark in access and it's capability .
regard
hansv mvp say:

The Bookmark property of a recordset (DAO or ADO) and of a form is a value that uniquely represents the current record in the recordset or form.


Examples of use:


a) You can store the Bookmark in a variable, then move to a different record. To return to the original record, set the Bookmark to the value of the variable.


b) As in another recent thread by you, you can use the RecordsetClone of a form to find the first or next record with a specific value in a field. To move the form itself to the found record, set the Bookmark of the form to the Bookmark of its RecordsetClone.


See Recordset.Bookmark Property (DAO) and Form.Bookmark Property (Access) for more info.

hansv mvp say:

You can't rely on the bookmark still being valid when you close and reopen the form.

The bookmark of a form can only be used for that form, not for other forms - each form has its own bookmark.

toronto glen say:

The problem with resizing graphs once you bring them into PPT is that the Font sizes change.

Best Practice is to ensure your graphs are all the same size in Excel first. To do this accurately I recommend first having the all charts on their own tab (not on a spread sheet page). Then go into Page Setup of the chart tab and to Margins - set your margins the same on each chart tab. This will ensure they are the same size.

If the charts are too large, simply increase the size of the margin in Excel (which will reduce the size of the chart - which by default will be set to a full page).

As for a macro here is a basic one Aligns the image to a top left position (change the number "100" to get your position):

Sub Macro2()
   With ActiveWindow.Selection.ShapeRange
        .Left = 100
        .Top = 100
    End With
End Sub
