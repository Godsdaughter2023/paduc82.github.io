How do I format a cell for a phone number

Phone numbers in the United States are usually formatted as xxx-xxx-xxxx or (xxx) xxx-xxxx

Wikipedia National conventions for writing telephone numbers says, "Twenty-four countries and territories share the North American Numbering Plan (NANP), with a single country code. The formatting convention for phone numbers is NPA-NXX-XXXX, where NPA is the three digit area code and NXX-XXXX is the seven digit subscriber number".

bob jones aka: cybertaz say:

Any format can be applied to any cell, regardless of the cell's content - even if the cell is blank. When appropriate content is entered into the cell the formatting will be imposed.

However, the Sample field in the dialog box populates only if the cell does have content. Even then, the Sample will display the formatting imposed on that content only if it is applicable. When choosing Phone Number for example:

  • If the cell is empty the Sample field will be gray showing no content
  • If the cell contains 1234567890 the Sample field will display (123) 456-7890
  • If the cell contains anything other than a 10 digit number the format in the Sample filed will be partially applied or not applied at all. I.e., if the cell contains:
    • 12 or 1234 the Sample will show as -12 or -1234, respectively
    • abcdefghij the Sample will show those letters without any formatting characters imposed.

What you may be running into is that the dashes have been typed in the cells which display xxx-xxx-xxxx. If so, applying Phone Number formatting can be done but will not cause that content to be reformatted to conform to (xxx) xxx-xxxx because the existing content includes characters which make it inappropriate for that format. Removing the dashes & any other extraneous characters/spaces will permit Phone Number formatting to take effect.

Hi,

The main differences between linked objects and embedded objects are where the data is stored and how you update the data after it's inserted in the Word document.

  • Linked object 
    Information from a object that is linked in Word is stored in the source file and can be updated if the source file is modified. Use linked objects if file size is a consideration.
  • Embedded object 
    Information from an Excel object that you embed in Word doesn't change if you modify the source Excel file. Once embedded, the object is no longer part of the source file and won't reflect any changes made.

To resolve your concern, please follow the steps below:

  1. On the Insert tab, in the Text group, click Object.
  2. In the Object dialog box, on the Create from file tab, choose Browse, and locate the file you want to insert in the Word document.
  3. Choose one of the following:
  • To add the file as a linked object, select the Link to file check box, and then click OK.
  • To add the file as an embedded object, select the Display as icon check box, and then click OK.

doug robbins - mvp office apps & services (word) say:

What is the purpose for which you want to apply the protection?  There are a number of types.

On the Review tab of the ribbon, in the Protect section of the ribbon, there is a Protect Document button with which,

1.    Limit formatting to a selection of styles

2.    Apply the following editing restrictions"
    Track Changes (turns on track changes so that all changes to the document are visible)
   Comments (only allows the addition of comments to the document)
   Filling in Forms (used in conjunction with Legacy FormFields)
   Read Only (prevents changes to the document)

With all of these, a password can be applied and to unprotect that document, it is necessary to supply the correct password.

In addition, by click on the Tools pulldown in the File Save As dialog, and the selecting the General Options item, passwords can be applied that must be supplied to Open the document, or to Modify the document.
