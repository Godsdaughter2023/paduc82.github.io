chatmaker say:

Quick Answer:

"NEED A FORMULA TO MULTIPLY THE UNIT MEASURE ie M2 or M3 by the rate for that item"

=[@Rate]*INDIRECT("tblData[@" & [@UOM] &"]" )

"RATE CELL PRICE DEPENDS ON THE PRODUCT"

=VLOOKUP([@Product],tblProducts,2)

Explanation - Tables are better for this

Create Data Table: To make your data into a table, select the QTY cell (actually any cell in the data will do) and press CTRL-T.  A new tab appears: TABLE TOOLS DESIGN. In the left corner of this tab is the Table Name box with Table1 in it. Change it to tblData. 

Insert Product Column: Right click on the QTY cell and select INSERT > Table Columns To The Left. Change the column header to Product.

Change Column Heading: Unit Of Measure just seems too long. UOM is as well understood as Qty so let's shorten things up a bit -just for style.

Add Amount Formula: =[@Rate]*INDIRECT("tblData[@" & [@UOM] &"]" ) uses a couple of things you are probably unfamiliar with. 

Structured References: [@Rate] is a structured reference.  These come with tables which is why I converted your data to a table.  The @ sign indicates the current row.  [@Rate] means the value in the current row in the Rate column

INDIRECT is an XL function that interprets a string as best it can to an address.  In this case [@UOM] returns M2 (in the first row) so "tblData[@" & [@UOM] &"]" becomes tblData[@M2] which INDIRECT will use to get the value from the current row in the M2 column.

Create Product Table: To make the rate cell depend on the product you need another table, one that has the product ID and its rate.  And while we are at it, let us also get the product Length, Width, and Height.  Make the table using the picture above as a guide.  Use CTRL-T to turn it into a table and name it tblProducts.

Enter formulas in tblData: Now we can get various bits of data from our product master table.

L: =VLOOKUP([@Product],tblProducts,3)

W: =VLOOKUP([@Product],tblProducts,4)

H: =VLOOKUP([@Product],tblProducts,5)

Rate: =VLOOKUP([@Product],tblProducts,2) 

Summary:

  • Use Tables to gain structured references and a whole lot more.

  • Use INDIRECT to use strings to direct where we get data

  • Use VLOOKUP to pull data from other tables into our main table.
