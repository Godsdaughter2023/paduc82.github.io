wruss say:

I will assume that you are using a headset microphone of some sort. If so, and you are using a laptop computer with a built-in microphone, go into the Sound control panel and disable the built-in microphone so that it is not inadvertently picking up your voice as well.

Regarding the "start typing" command, it is meant to allow you to start dictating individual letters like a, b, c, and so on. It is not meant to be used to enter general dictation. Before you attempt to dictate, make sure to turn this mode off by saying "stop typing."
