The OEM license is for use of Office on that one computer, not to one specific user. It "should" not have any impact on a roaming profile. The profile just contains settings and some files, none of the executable files. The roaming profile should be usable on any machine that has a licensed copy of Office installed.

What "new" restrictions on OEM are you talking about. That page you linked to doesn't really go into OFFICE OEM licenses, it mostly talks about WINDOWS OEM licenses.  This page is about Office licenses, but not specifically about OEM:
http://www.microsoft.com/OEM/en/licensing/productlicensing/Pages/office-2013-licensing-packaging.aspx

No matter which type of license you have, you can create multiple Windows User IDs/accounts on it and they all can use the Office installation on that machine, one at a time.

OK, I see the spot you mean:

<snip>

Office Home and Student 2013

  • The latest Office applications—Word 2013, Excel 2013, PowerPoint 2013, OneNote 2013
  • One user, one PC
  • Transferable license*
  • Non-commercial use rights

</snip>

Since the 3rd clause is "transferable", they are talking about the "Retail" / FPP type license (I don't know yet if this term applies to the ESD license also)


That does allow you to only install on one computer (with transfer rights).  But the "1 user" part is a mis-statement of the license terms.  The license says one person at a time may use the license, but any user on the computer it is licensed on may use it.


<snip from MS License Terms  http://office.microsoft.com/en-us/products/microsoft-software-license-agreement-FX103576343.aspx >

we grant you the right to install and run that one copy on one computer (the licensed computer) for use by one person at a time,

</snip>


The "one person at a time" part covers the use by multiple user ID's on the computer.
