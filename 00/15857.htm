ossiemac say:

Cheers, I've got it to work now.

Is there anyway I can get it so it is default?

I need to copy around 200 rows from a table, one at a time, and going to Go To, Special, Visible Cells Only everytime would be a bit tiresome.

I agree with that comment.

  1. I assume that your data has column headers but if not then insert a row above the data and insert column headers. (To do this insert Col1 in the first header and then drag across to AutoFill the remaining headers and they will be Col1 Col2 etc)
  2. Add an extra column and give it a column header.
  3. Go through the data and decide which ones need to be copied and insert a Y in the extra column.
  4. Apply AutoFilter to the table of data. (Home ribbon -> Sort & Filter icon towards far right -> Filter.
  5. Click the drop down on the extra column and set the filter for the extra column to Y. (Only rows with the Y will be displayed)
  6. Now Select the data (excluding the last added column) -> Goto special -> Visible cells only and then Copy and Paste all in one operation.

Hi Duane,

Thank you for posting your query on Microsoft Community.

Certificate errors occur when there's a problem with a certificate or a web server's use of the certificate. Internet Explorer helps keep your information more secure by warning about certificate errors.

Method 1

I would suggest you to add the website to the trusted list and also add the website in compatibility view settings.

Step 1

 A websites’ certificate provides identification of the web server. If the certificate has an error, it might indicate that your connection has been intercepted or that the web server is misrepresenting its identity. If you're absolutely positive of the website's identity, you know that your connection hasn't been compromised, and you understand the risks, you can choose to go to the website. However, we recommend that you don't ignore a certificate warning.

1.  Open your Internet Explorer browser then go to Internet Options.

2.  Click on Security tab.

3.  Click on Trusted sites, then click the Sites button.

4.  On the "Add this website to the zone:" URL bar, type the complete URL address.

5.  Click on Add button.

6.  Once you already added all URL addresses, click the Close button.

7.  Click on Apply or OK button then close your Internet Explorer browser.

Also refer to the link:

http://windows.microsoft.com/en-IN/internet-explorer/certificate-errors-faq#ie=ie-11

If the issue still persist, refer to the below method.

 

Step 2

Try to add the compatibility view for the website.

1. Open the Internet Explorer.

2. Click on Tools button and then click on Compatibility View Settings.

3. Then type “live.com” and click on Add button.

If the issue still persist, refer to the below method.

Method 2

I would suggest you to install the important updates. Refer to the below link for more information.

http://windows.microsoft.com/en-us/windows/turn-automatic-updating-on-off#turn-automatic-updating-on-off=windows-7

I would suggest you check the date and time of the Computer and keep it up to date.

1. Open Date and Time by clicking the Start button, clicking Control Panel, clicking  Clock, Language, and Region, and then clicking Date and Time.

2. Click the Internet Time tab, and then click on Change settings. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

3. Select the check box next to Synchronize with an Internet time server, select a time server, and then click on OK.

Hope this information helps, get back to us if you need further assistance with Windows.

Hello,

I cannot reproduce that behaviour. Any shape from the Insert > Shapes menu that I add to the chart, stays put, even after closing and re-opening Excel. Could you share the problematic file? Save it to your Onedrive public folder, share it, get a link and post that link here. 

Things that you place on the chart with Pivot Chart Tools Ribbon > Insert Shape will stay in their assigned position on the chart.

Things that are placed on the chart while it is NOT selected, using Insert > Shape, will be put these objects on the worksheet where they can be in front of the chart or behind the chart. You may need to re-arrange shapes and the chart. Select the chart and on the Pivotchart Tools > Format ribbon click the Send Backward command drop down and select "Send to Back".
