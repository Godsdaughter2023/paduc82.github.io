deerob900 say:

I hate the stupid white box; crazy that there is not a built-in option to change it in "light mode."

This worked for me:

Open regedit, navigate to: Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Search\Flighting\

Under both the "0" and "1" sub-directories, look for the DWORD value for each registry key that ends in the phrase ". . . InDarkMode."

Change the DWORD value for corresponding regular registry key (i.e., the one that does not end ". . . InDarkMode") to the value of the "InDarkMode" registry key DWORD value. 

This is important: For each registry key changed, open permissions and deny all permissions to the SYSTEM user (this prevents your changes from being undone on restart). 

EXAMPLE:

For the registry key SearchBoxTransparencyHighlightInDarkMode, the default DWORD value is 0x0000001e (30). 

Therefore, change the DWORD value for SearchBoxTransparencyHighlight to 0x0000001e (30),

You will have to do this for 15-20 registry keys (do both the "1" and "0" sub-directories under ~\Flighting).

After doing this and restarting, you should have the dark/grey search box showing when in "light mode."

If you set Hybrid Sleep to “off” then your computer will not sleep. To set it off do the following.
 
1. Open Control Panel\All Control Panel Items\Power Options\Edit Plan Settings

﻿Change Power Settings

How to:

  •  Set your computer to go to sleep after a specified period of inactivity. For more information, see http://windows.microsoft.com/en-US/windows7/Turn-off-a-computer-frequently-asked-questions
  •  Adjust the brightness of your display.
  •  Choose when to turn off the display during periods of inactivity.
  1.  Open Power Options by clicking the Start button, clicking Control Panel, clicking System and Security, and then clicking Power Options.
  2.  Under the plan that you want to change, click Change plan settings.
  3.  On the Change settings for the plan page, choose the display and sleep settings that you want to use when your computer is running on battery (if applicable) and when it's plugged in.
  4.  ﻿For more options, click “Change advanced power settings” and for even more click "Change settings that are currently unavailable" (if available).
