 my microsoft security essenstials has a option in the settings which says limit CPU usage during scan and its set at 50% what is this option for can someone explain this to me in laymans terms as im a computer novice and which setting is best for my laptop to run fast shall i adjust it up to 100% or down to 0%
jetpack_b say:

Hi,

You should adjust the CPU usage during scan depending on your computer speed and what sort of use it will be getting during the scan. If you set the usage to 100%, there's a large risk that your PC will be extremely slow if you attempt to use it during scans.

If you have a fast computer bought or updated recently (especially running Windows 7), you should probably be able to get away with 80% speed and get decent program run-time. If your computer is slow, you can probably get away with around 40% if you want to use other programs (and even then it'll be slow!).  The scan will take longer if you limit its CPU usage. If your computer is average, I'd recommend leaving it at 50% (I have this setting and use a fast computer to optimise response time for other high-usage programs I run).

 

The simple answer: I'd recommend leaving it as it is unless your computer is particularly slow or rather quick. If you are able to leave your PC completely during the scan, you could set it to 100% and the scan would get done faster.

So there is a way to do this via powershell which has been very helpful to me. Someone from a different community assisted me with this. The script is below. It give me each user and the # of folder they have in descending order.

$Mailboxes = Get-Mailbox

$Results = foreach( $Mailbox in $Mailboxes ){
    $Folders = $MailBox |
        Get-MailboxFolderStatistics |
        Measure-Object |
        Select-Object -ExpandProperty Count

    New-Object -TypeName PSCustomObject -Property @{
        Username    = $Mailbox.Alias
        FolderCount = $Folders
        }
    }

$Results |
    Select-Object -Property Username, FolderCount | Sort FolderCount -Descending
