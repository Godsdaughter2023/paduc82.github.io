  3. our adfs.ourdomain.com is in Internet Explorer’s Local Intranet zone (using Computer Policy)

doug robbins - mvp office apps & services (word) say:

Use the commands that you are now using in the ActiveX button click event, and insert them into a macro that you create in a module in the template and name that macro as 

Sub AutoNew()

[commands go here]

End Sub

Then when you use File>New to create a document from the template, the commands that were used by the ActiveX button will be executed so you should get the same result as if that button was used.
