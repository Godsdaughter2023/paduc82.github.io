In general, downloads may be deleted once they've been installed on the computer.  For example, if you downloaded the iTunes Setup file from Apple, once it's installed you no longer need the file you downloaded from Apple.

However, that said ... my standard advice in cases like this is that if you need to go to measures this extreme to free up disk space my best advice is to get a larger hard drive and clone your existing software onto it.  You will gain space measured in megabytes (one million bytes) at a time by this kind of cleaning but your hard drive's space is measured in gigabytes (one thousand million bytes) so you would have to have a LOT of things to clean up to make even a tiny difference.

Normally you can delete setup files, but of course, there would be exceptions. 
It may be you are referring to the installer, I'm not sure, but that can be deleted, as long as
you can easily access another copy should you need to reinstall that software again.
You can backup uninstallers easily on external media.
If the program were to need setup files and you delete them then it's just a matter of uninstalling
and reinstalling the program again.
Another option would be to make a system restore point before deleting the files, so if the
program  stops functioning you can run System Restore.
System Restore, and backups are very useful tools for fixing up things that go wrong when experimenting  .
- A couple of examples of setup files vs an Installer.exe  that can be deleted are drivers for graphics cards and the Direct X Redist installer setup.

-Different PC/laptop mfg's  supply different bloatware, and people have different wants/needs.
Your user manual should have a basic description of the function of software installed by the mfg.
You have to determine for yourself what programs the mfg installed that you won't use.
Then look through your Start Menu and open Programs to further determine if you need them.
Then you can go to Programs and Features and install what you don't want, but there are a number of
things you do need to keep. Many of these don't have an executable or shortcut,  so if you don't know what it is don't  uninstall it .
-Microsoft .NET framework, Microsoft Visual C++, Intel or AMD cpu stuff, and all drivers for component parts  such as Audio, USB, LAN, Graphics, keyboard and mouse, etc. are keepers,
along with things like Adobe Reader.

- For the most part software does not actually take up much HDD space.
Even with uninstalling a lot of software, it's only a matter of time before you will need a larger HDD,
or and external HDD .
If you want to regain some HDD space your Music, and video's are a good place to start. Move them
to an external drive, and then move  on to  photos, and other personal files, if you still need more
space.
