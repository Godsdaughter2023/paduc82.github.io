At some point you must have restarted numbering at 23. Go to the first page 23 and use Insert | Header & Footer | Page Number | Format Page Numbers. Set the page numbering to "Continue from previous section." Then go to the next section (an easy way to do this is to use the Browse arrows at the bottom of the vertical scroll bar, with the browse object set to Browse by Section). You can check if you need to reconnect by doing the following:

1.  Open Outlook

2.  Click File, Account Settings, Account Settings

3.  In the Email Accounts dialog locate your Outlook.com account.

If it shows as Type Exchange ActiveSync then you need to remove it and add it back or create a new profile per instructions in this topic, https://support.office.com/en-us/article/Reconnect-Outlook-2016-or-Outlook-2013-to-Outlook-com-for-uninterrupted-email-access-cda1751d-9503-40bf-bf76-e79454ac5eb3?ui=en-US&rs=en-US&ad=US.
