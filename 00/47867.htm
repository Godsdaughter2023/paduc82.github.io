Installing Windows update would modify the registry entry.

 

A: An update should not remove registry entries populated by other updates, especially when Microsoft advises using those registry entries for detecting if an update is in fact installed. This results in false indication of an update not being applied when in fact it is, and causes a continuous process of thinking it needs to be installed.
