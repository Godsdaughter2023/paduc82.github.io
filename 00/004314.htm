Is there a way to associate a specific color to a region so that each time it shows up in a chart it will be formatted with the same color?

rajesh c. say:

Excel uses dynamic colors while plotting any chart.

Freezing a chart line (or area) color is not possible through any formula.

However, you can write a macro code which can use same color (in the chart) as given in the corresponding cell of the data. You will then have to manually ensure that you use a specific color of your choice to display the chart line/area in a specific color everytime.

Sub ColorChartColumnsbyCellColor()

With Sheets("Sheet1").ChartObjects(1).Chart.SeriesCollection(1)

    Set vAddress = ActiveSheet.Range(Split(Split(.Formula, ",")(1), "!")(1))

    For i = 1 To vAddress.Cells.Count

        .Points(i).Format.Fill.ForeColor.RGB = ThisWorkbook.Colors(vAddress.Cells(i).Interior.ColorIndex)

    Next i

End With

End Sub

The above macro is copied from

(http://www.get-digital-help.com/2013/09/14/format-fill-color-on-a-column-chart-based-on-cell-color/)

The "Relaying not allowed" error means you are not authenticating to the outgoing server and you need to. Not only should you check the box stating that the outgoing server requires authentication, but the one that says to use the same credentials as the incoming server. The bigger issue here, though, is that by using ports 110 and 25 without encryption, you are sending your credentials in clear text over the internet where anyone monitoring your traffic has full access to it.
