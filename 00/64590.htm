chris.jasztrab say:

So I actually managed to answer my own question.  I was able to load the registry hive of the NTUSER.DAT into regedit and make the changes.   This helped so I didn't have to redo the whole default profile.

The steps were as follows:

1. Click "Start" and enter "regedit" into the search field. Point to "regedit.exe" and then press "Ctrl-Shift-Enter" to open Registry Editor.

2. Click "File" and then "Export" to create a backup of the registry prior to editing Ntuser.dat.

3. Enter a name for the backup into the File Name field and then click "Save."

4. Press "Windows-E" to launch Windows Explorer or click "Start" and then "Computer."

5. Click "Tools," "Folder Options" and then "View."

6. Click "Show Hidden Files, Folders, and Drives" and then uncheck "Hide Extensions for Known File Types." Click "OK."

7. Return to Registry Editor. Click "HKEY_USERS." Select "File" and then "Load Hive." Navigate to C:\Users\Default.

8. Select "Ntuser.dat" and then click "Open." Create a name for the hive and then click "OK" to access Ntuser.dat.

9. Click "File" and then "Unload Hive" when you are done making changes to Ntuser.dat.
