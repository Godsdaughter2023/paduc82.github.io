Arranging the order of a combo box based on a value list - Microsoft Community

shane2234 say:

Arranging the order of a combo box based on a value list

I have a combo box control in a form where I want to rearrange the order of items such that the most common selection appears as the default.  The row source is based on a value list.

I have re-arranged the items using the row source property, but the combo box still displays the original default value.  How can I change the order of the value list in the combo box?

ken sheridan say: You can easily add a list of controls to a combo box or list box's list by using the controls' Tag property to indicate which you wish to be listed, and, of those, which you wish to be first in the list, and be selected by default.  The code for the form's Open even procedure would be like this:

Private Sub Form_Open(Cancel As Integer)

    Dim ctrl As Control
    
    With Me.lstControls
        .RowSourceType = "Value List"
        For Each ctrl In Me.Controls
            If ctrl.Tag = "ListMeFirst" Then
                .AddItem ctrl.Name
                .Value = ctrl.Name
            End If
        Next ctrl
        For Each ctrl In Me.Controls
            If ctrl.Tag = "ListMe" Then
                .AddItem ctrl.Name
            End If
        Next ctrl
    End With
  
End Sub

To judge by the RowSource property you have cited, your control appears to have multiple columns, so you'd need to call the AddItem method multiple times within each For….Next construct and set the ColumnCount and ColumnWidths properties of the control accordingly.
