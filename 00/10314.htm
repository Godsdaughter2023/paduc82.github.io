grschinon say: The above answer isn't entirely accurate.

If the phone supports 4GB cards then it is SDHC-compatible because there's no such thing as a 4GB microSD card that isn't HC. Full-size SD cards exist in 4GB non-HC, but not miniSD or microSD.

Since the hardware is there, the limitation is therefore down to firmware. The operating system in your phone doesn't know how to handle filesystems greater than 4GB in size. A firmware update might solve the problem or at least increase the size limit on recognised filesystems but there's no guarantee on an S40 phone such as the 6500s.

Wonder why you are using "(Q4-(Q4-(Q4/1.0825)/1.22))" ... you can use just "(Q4/1.0825)/1.22" instead.

Considering this, use below formula to account for AA4 (if AA4 = "Y" then it is 1.22 else it is 1.23):

=IF(Y4="Y",(Q4/IF(AA4="Y",1.22,1.23)),(Q4/1.0825)/IF(AA4="Y",1.22,1.23))

Blank cells & Conditional formatting - Microsoft Community

a. user say:

Blank cells & Conditional formatting

I'm trying to use conditional formatting to highlight changes in pricing.  My issue is the data I receive only populates a cell when a transaction for a particular product occurs, and I have a number of blanks in the data.  Ideally, I want to highlight price increases with green shading, and price decreases with red shading.

ossiemac say:

Requires a UDF (User Defined Function) to find the previous non blank cell to compare against.

Not sure if you require instructions to install the VBA code but just in case.

  1. Alt and F11 to open the VBA editor window
  2. In the VBA editor select menu item Insert -> Module
  3. Copy the VBA code below and paste into the VBA editor.
  4. Edit the code where indicated with the comments in the code (Displayed as bold italics in the code below but will be green when pasted into the VBA editor)
  5. Close the VBA editor (Cross very top right of VBA editor window)
  6. Save the workbook as Macro enabled.
  7. Ensure macros are enabled. See help for how to do this. (Option to "Disable all macros with notification" should be OK.)

Conditional formatting select the range of data from Feb column; not from Jan column. See the black border around the range to format in the following screen snippet.

In the conditional format use a formula as follows for the first condition and set the color Green for increased price.

=AND(C2<>"",NextLeft(C2)<C2)

For the second condition for decreased price color red use the following formula.

=AND(C2<>"",NextLeft(C2)>C2)

Copy all of the following code into the VBA editor (From Function NextLeft....  to End Function)

Note that the Conditional format starts column C (Feb column). The UDF looks back to column 2 (Jan column). Can't apply conditional format to Jan column because no previous data to compare but Jan column is compared to Feb column.

Function NextLeft(rng As Range) As Currency
    Dim lngCol As Long
    Dim c As Long
   
    lngCol = rng.Column - 1
   
   'Following line the 2 should be column B which is your Jan Column
    'If your data not set up as per snippet above then edit the 2 to the number of the Jan Column
    For c = lngCol To 2 Step -1
        If Cells(rng.Row, c) <> "" Then
            NextLeft = Cells(rng.Row, c).Value
            Exit For
        End If
    Next c
End Function
