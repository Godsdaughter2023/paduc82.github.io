VBA DoEvents - Microsoft Community

a. user say:

VBA DoEvents

Hi, I was wondering what the VBA function DoEvents() actually does.

I often see it in Do While loops, but I've never understood.

Thanks in advance for responses!

- Psyi
hansv mvp say:

DoEvents pauses execution of the VBA code briefly and passes control to Windows, so that it can process 'events' such as mouse clicks and keystrokes. When the events have been processed, execution of the VBA code resumes.

 

This is useful in time-consuming loops. Without DoEvents, the application would become unresponsive to mouse clicks and keystrokes. Among other things, this would make it impossible to "break into" the code using Ctrl+Pause.
