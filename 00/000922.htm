abdullah alhadi say:

Transfer automatically one sheet to another sheet using Excel formula function or other

I have one excel data file like below

 Now how to transfer automatically  one sheet to another sheet using excel formula function or other, like below attach please advise

bernard liengme say:

The subroutine below will do what you ask.

You need to type the T-codes on row 1 of the Table worksheet before running the macro

This site will tell you where to copy the code to

http://www.contextures.com/xlvba01.html

best wishes

Sub tryme()
 Worksheets("Data").Activate
 LastRow = Cells(Rows.Count, "A").End(xlUp).Row
 For k = 1 To LastRow
   mycode = Cells(k, "B")
   myCol = CInt(Right(Cells(k, "C"), 1))
   With Worksheets("Table")
      myRow = .Cells(Rows.Count, myCol).End(xlUp).Row + 1
      .Cells(myRow, myCol) = mycode
   End With
 Next k
End Sub
