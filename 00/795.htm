Word 2003's menus are customizable. As a troubleshooting step, please follow the steps on this page (about 1/2 way down, User-initiated Office Safe Mode) to restart Word in Safe Mode. If the Recent Files item shows, then it's a customization issue. To fix this:

  1. Restart Word normally, then choose Tools>Customize.
  2. Select the Toolbars tab, then select the Menu Bar item.
  3. Click on the Reset button. Word will ask whether you want to Reset changes, click on OK. The menu bar is reset and should now display Recent Items.

If it's not a customization problem, then try repairing Office:

  1. In Windows, search for Control Panel, then open it.
  2. Choose Programs, then Programs and Features.
  3. Right-click on Microsoft Office 2003 and choose Repair.
  4. Allow the process to complete, restart Windows, then see if Word displays Recent Items.

This is not a computer or even a Word setting, it is a document setting (that can be inherited from a template).

As has been pointed out, you cannot prevent this because to change the setting, you need to open the document.

You could make a copy of the document before opening it. Then open it, and change the setting. Then in the open document, try using the Organizer to copy the styles from the unopened copy. (I do not know that this will work but it seems worth a try.) Copy all of the styles three times in case there are cascading styles.

If you try this, please write back so we can see if it works.

Another method, also involving a copy, would be to find the name and location of the template. Close Word and your document without saving, then temporarily change the name of the copy to the name of the template and move that "template" to the proper location. Be careful to not overwrite an existing template with this, especially the normal template! Then open the document again and change the updating switch.

For those who are wondering what the fuss is about, see What happens when I send my document to someone else? Will Word mess up my formatting?
