What does the asterisk in "*.docx" mean? - Microsoft Community

margareemitchell say:

What does the asterisk in "*.docx" mean?

I recently purchased Office 365. When I save a document (*.docx) appears. What does the asterisk mean?

[Original title: Microsoft Word]

jay freedman say:

The asterisk is a "wildcard" character that means "any text". The Word Document (*.docx) in the Save As Type box is just showing you that the document will be saved with the .docx extension, which indicates what kind of file it is.

You can click that box to open its dropdown, where you can choose a different file type. Each of them shows an asterisk followed by the extension for that kind of file.

You have to supply the text that will be used to replace the asterisk in the actual file name. The article http://vlaurie.com/computers2/Articles/filenames.htm describes the rules you have to follow.
