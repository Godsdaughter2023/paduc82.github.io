suzanne s. barnhill say:

It could well be that rotating the diagram would, in the long run, be the easiest approach. The alternative will be a REF field instead of SectionPages. Assuming that you are restarting numbering in each logical section, this will not be too difficult. Restart numbering as usual on the first page of the logical section, then continue it through the landscape section and the portrait section that follows. Insert a bookmark somewhere on the last page of the section (somewhere where it is not likely to be deleted but is certain to remain on the last page). Instead of the SectionPages field use a REF field referencing the page number of the bookmark.

suzanne s. barnhill say:

You can insert the field easily using References | Captions | Cross-reference: Reference type: Bookmark; Insert reference to: Page Number.

 

Insert the bookmark using Insert | Links | Bookmark. The bookmark must be a single word; it can contain numbers but can't start with one. So you can use LastPage but not Last Page, Page2 but not 2ndPage, etc.

Accounts become inactive if you don't sign in for more than 270 days VIA WEB BROWSER or within the first 10 days after signing up VIA WEB BROWSER for an account.

Your account name is still reserved.

How do I delete contacts from Skype? - Microsoft Community

leonspektor say:

How do I delete contacts from Skype?

How to delete contacts from Skype?

Subject/title expanded by Moderator from "Skype."

susan_s. say:

Hello LEONSPEKTOR,

Please see below information for the steps on how to delete a Skype contact depending on the Skype version you're currently on:

The new Skype for Desktop (Windows, Mac and Linux - 8.12)

Select your profile picture > Scroll down and select About

*Delete a contact: Sign in to Skype > Right-click on the contact that needs to be removed and choose Delete contact (You will only see this option if they are not in your synced mobile device address book as well).

Skype for Windows 10 (Skype UWP - 12.10)

Sign in to Skype > Select the ...See more menu at the bottom left of the window > Select About

*Same with the new Skype for Desktop 

Skype for Windows desktop (classic Skype - 7.40)

Sign in to Skype > In the menu bar, click Help > About Skype

 

Delete a contact: Sign in to Skype > In your contact list, find the contact you want to remove. Right-click the contact name and select Remove from Contacts. Then confirm by clicking Remove.

The contact is removed from your contact list but will still appear in the Recent tab with a question mark. Right-click the contact name to see the options: Add to Contacts or Block This Person.

Hope this information helps! Let us know if you need further assistance.
