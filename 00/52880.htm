sysit say:

When booting Linux, look for Windows 10 in Grub and remember at what spot it is.

If it is on spot 4 for example, then remember the number 3 since Linux starts counting with 0.

Go on with booting Linux.

Go into /etc/config/grub

Look for the line GRUB_DEFAULT=0

0 in this case is the system which was in the first spot.

Now you put in the number that you remember for Windows 10 and safe the file.
