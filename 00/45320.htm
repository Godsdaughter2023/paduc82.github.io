This bug check (IRQL NOT LESS OR EQUAL) is issued if paged memory (or invalid memory) is accessed when the IRQL is too high.



Typically, this error occurs when a driver uses an incorrect memory address. Other possible causes of this error include an incompatible device driver, a general hardware problem, or incompatible software


First, I would suggest you to install the latest hardware drivers from the manufacture’s website and check if it helps.




Let’s try the following methods and check if it helps.



Method 1:


I would suggest you to Run Memory diagnostics tool and check if it helps.

http://windows.microsoft.com/en-in/windows7/diagnosing-memory-problems-on-your-computer

Method 2:


I would suggest you to Update BIOS and chipset drivers and check if it helps.


This issue could also be caused due to outdated Basic Input Output system (BIOS) or chipset drivers. I would suggest you to visit the manufacturer’s website to download and install the latest BIOS and chipset drivers.
