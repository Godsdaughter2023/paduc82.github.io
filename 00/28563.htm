Don't use any scaling in the Print dialog nor the booklet option in your printer Properties. Instead, select 11' x 17" on the Paper tab of Page Setup and "Book fold" on the Margins tab. Alternatively, if you are using the booklet option in your printer Properties, consult the printer manual for what other settings are appropriate.

    A common issue and problem is perhaps at one time that database was used for development.

     

    So custom menu bars, and imbedded images in the database are not transferred if you just import the tables in question.

    VBA - User Form - Check Box Do not show this notice again - Microsoft Community

    sglxl say:

    Seasons Greetings to All,

    I have a User Form with  a Check Box and a Cancel and Getting Started Control Buttons. This is displayed when the workbook opens. The user has the choice to click onto a "Getting Started" help manual. Also he has the option not to display the User Form again if he checks the box.

    What VBA code to avoid the display of the User Form if the box has been clicked.

    Many thanks/sglxl
    hansv mvp say:

    Open the workbook.

    Select File > Info.

    On the right hand side, click Properties > Advanced Properties.

    Activate the Custom tab.

    In the Name box, enter DontShow

    Select Yes or No from the Type dropdown.

    Select No as Value.

    Click Add.

    Click OK.

     

    In the Workbook_Open event in the ThisWorkbook module:

     

    Private Sub Workbook_Open()
        If Me.CustomDocumentProperties("DontShow") = False Then
            UserForm1.Show
        End If
    End Sub

     

    where UserForm1 is the name of the userform.

     

    In the On Click event procedure of the check box on the userform:

     

    Private Sub CheckBox1_Click()
        ThisWorkbook.CustomDocumentProperties("DontShow") = Me.CheckBox1
    End Sub

     

    where CheckBox1 is the name of the check box.

    Trigger Macro when cell value changes. - Microsoft Community

    roguebiscuit say:

    Trigger Macro when cell value changes.

    This seems like it should be an easy thing to do but for the life of me I cannot find anything on the internet on how to do it.

    I would like to have a macro triggered whenever the value of a particular cell changes.  Can anyone help me write code that will accomplish this?  Thanks very much for any assistance.

    Andrew

    jeeped say:
    Right-click the name tab of the worksheet containing the particular cell (let's call that A1) and choose View Code. When the VBE opens, paste the following into the pane titled something like Book1 - Sheet1 (Code), 

    Private Sub Worksheet_Change(ByVal Target As Range)
        If Not Intersect(Target, Range("A1")) Is Nothing Then
            Application.EnableEvents = False
            Call mymacro
        End If
        Application.EnableEvents = True
    End Sub

    Edit this and change mymacro to the name of your own macro sub. Tap Alt+Q to return to your worksheet. Any edit change to the value in A1 including pasting a value into A1 or a range of values into cells that include A1 will launch the macro. Note that a change in the result of a formula in A1 will not trigger the event.

    There are a lot of things to consider when using this sort of automation mechanism and your description is a little sparse. Post back with more details and additional error control may be able to be offered.

    Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
