Method1: Try the steps mentioned below:

  1. Open the Task manager.  Here’s a tip:  Press CTRL+Shift+ESC.
  2. Click File > Run New Task
  3. Make sure you have a check mark beside “Create this task with administrative privileges”
  4. Type CMD
  5. Type the following 2 commands at the CMD prompt:

    sfc /scannow

  6. Close the CMD window

Method 2: To make sure user account is not corrupted, I would suggest you to create a new User account and see if the issue still persists or not:

Follow the steps below:

  1. In search type User Account.
  2. Select manage another account.
  3. Select Add a new user in this PC account.
  4. Click on Add someone else to this PC and follow the onscreen instructions.
