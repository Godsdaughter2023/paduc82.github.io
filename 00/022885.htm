Sub SendEmail()

    'This sub uses Late Binding (Does NOT require reference in VBA Tools -> References)
    'This Sub Creates the email and then Sends the Email and returns message of Sent OK or error sending
    Dim objOutlook As Object
    Dim objMailItem As Object
    Dim objAttachments As Object
    Dim strTo As String
   
    strTo = "Email Address"
   
    On Error Resume Next
    Set objOutlook = GetObject(, "Outlook.Application")     'If Outlook already open
    On Error GoTo 0
   
    If objOutlook Is Nothing Then
        Set objOutlook = CreateObject("Outlook.Application")    'Outlook not already open so open
    End If
   
    Set objMailItem = objOutlook.CreateItem(0)      'Start a new email.
   
    With objMailItem
        .To = strTo

        On Error Resume Next        'If email does not send then error occurs

johnwang_msft say:

pillravi,

You cannot categorize multiple messages in smart folders. The reason is because smart folders can contain messages from different accounts, each with different categories.  This is also true if you have accounts grouped (Preferences -> General -> Group similar folders...); you have to navigate down to a particular account (not the unified message folder) to multi-select and categorize.  In short, multi-select categorization is not supported in any view that may contain messages from more than one account.

Yes, Outlook could check selected mails dynamically before determining whether multi-select is possible, but that would have a performance impact.  If you have a 100 emails selected (or more, since you can "Select All"), Outlook would have to look at every single email and see if it's in the same account as the others, and it has to do this before deciding whether to enable or disable the menu item or ribbon button.  The user will likely see a delay, which would be annoying.

Regarding your other suggestion about offering to create a new category for multi-selected messages, Outlook would still have to scan all selected messages to see if categories are associated with those accounts.

How to create shortcut for "character spacing" in MS word 2010 - Microsoft Community

pdar say:

How to create shortcut for "character spacing" in MS word 2010

I try to set up shortcut for that..
suzanne s. barnhill say:

Ctrl+D opens the Font dialog. If you have last used it with the Character Spacing tab selected, it will open that way.

If you want a shortcut for specific character spacing, you can find the Condensed:, Expanded:, Raised:, and Lowered: commands in All Commands. These allow you to select a specific measurement (you can have only one setting for each).

If you want to apply a combination of character spacing settings, you could define a character style and assign a keyboard shortcut to that.
