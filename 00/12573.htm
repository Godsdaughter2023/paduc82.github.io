locate the cell address

Please don't do such things and never use SELECT, SELECTION and ACTIVECELL in your macros. It's slow and error prone.

Use the Range object and the Address property to create formulas:

Sub Test()
  Dim R As Range
  'Find the last used cell in column A
  Set R = Range("A" & Rows.Count).End(xlUp)
  'Create a SUM formula down to that cell
  Range("B1").Formula = "=SUM(" & Range("A1", R).Address(0, 0) & ")"
End Sub

Make a new file, write some value in column A and execute that macro.

allen browne say:

Try:
    =IIf([yesno], Date(), Null)

There were several issues with your expression:

  • .Value is not needed
  • The Value of a yes/no field is True or False, which is not the same as the text "yes".
  • You can't assign a value to some other control (named "date"?) like that.
  • You need to use Null, not a zero-length string as the alternate value. Otherwise Access will treat it as a string and not as a date value.

daniel pineault say:

To correct your IIF statement you have to understand that Checkboxes operate as True/False which also corresponds to -1/0:

=IIf([yesno]=True, Date(), "")

But that said, the proper approach would be to use the afterupdate event of your Checkbox to populate the date control with the current date.  Also, you should not label/name a control date as it is a reserved word and will lead to a problem.  Name it RegistrationDate, FillinDate,... something a little more descriptive.  Then in the afterupdate event of your Checkbox you'd do somthing like

If IsNull(Me.MyDateControlName)=True Then
   Me.MyDateControlName = Date()
End If

josephv_a say:

Hello

You need to have the external numbers as contacts first before you could add them as participants in a group call.

To add a PSTN number as contact, open the dial pad, and call the phone number you want to save. After the call, right-click the number in your chat list, select View profile, and then select Add on Skype.

To make a group call in Skype:

  1. From yourCalls  list select the new call button. 
  2. Select all the participants you want to call. 
  3. Select the Call button. 
    Note: All group calls will always start as audio calls, but you can turn your video on during the call by selecting the video  button. 

Android 4.0.4 - 5.1

  1. From your Calls  list, select Start a new call. 
  2. Select New Group call . 
  3. Select all the participants you want to call. 
  4. Select the audio , or video  button.
