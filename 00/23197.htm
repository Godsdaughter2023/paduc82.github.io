la lawyer say:

Word VBA: Is an Item Selected in a Multi-Select Box (UserForm)

I have a list of items (e.g., dogs, cats, mice) in a multi-select box in a UserForm.

Using Word VBA, how do I determine whether a specific item (e.g., dog) is selected?
greg maxey say: By evaluating the .Selected property:

Private Sub CommandButton1_Click()
Dim i As Long
For i = 0 To ListBox1.ListCount - 1
  If ListBox1.Selected(i) And ListBox1.List(i) = "Dog" Then
      MsgBox "Dog is one of the items you selected."
  End If
Next i
End Sub

jay freedman say:
It depends on whether you want a complete list of all the selected items, or just a yes/no on the selection of a particular item.  In each case, you need to check the value of the listbox's Selected property for one or more indexes in the list, where those indexes run from 0 to listbox.ListCount - 1.

Here are three examples:

Private Sub CommandButton1_Click()
    ' get a list of selected items
    Dim idx As Long
    Dim msg As String
    For idx = 0 To ListBox1.ListCount - 1
        If ListBox1.Selected(idx) Then
            msg = msg & ListBox1.List(idx) & vbCr
        End If
    Next idx
    MsgBox "Selected:" & vbCr & msg
End Sub

Private Sub CommandButton2_Click()
    ' test for selection of specific item
    ' if you don't know where that item sits in the list
    Dim idx As Long
    Dim found As Boolean
    found = False
    For idx = 0 To ListBox1.ListCount - 1
        If ListBox1.Selected(idx) And _
           (ListBox1.List(idx) = "dogs") Then
            found = True
        End If
    Next idx
   
    If found Then
        MsgBox "'dogs' was selected."
    Else
        MsgBox "'dogs' was not selected."
    End If
End Sub

Private Sub CommandButton3_Click()
    ' test for selection of specific item
    ' if you do know where the item sits in the list
    Const catIdx = 1
   
    If ListBox1.Selected(catIdx) Then
        MsgBox "'cats' was selected."
    Else
        MsgBox "'cats' was not selected."
    End If
End Sub
