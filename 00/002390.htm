This isn’t the normal behavior of Internet Explorer 8. First thing to test will be your add-ons. You can do this by:

·         Click Start

·         Click System Tools

·         Click Internet Explorer (No add-ons)

·         Press CTRL + A on your keyboard (this should select all add-ons)

·         Close Internet Explorer (No add-ons)

·         Open Internet Explorer

 

Now all add-ons are disabled in Internet Explorer, test to see if the same problem occurs without add-ons. If the problem is fixed then the next step is to enable the add-ons one by one to find which is causing the problem.

·         Click Tools

·         Click Manage Add-ons

·         Click the first disabled add-on

·         Click Enable

·         Click Close

·         Surf a few pages

·         Close Internet Explorer

·         Open Internet Explorer

 

Again test to see if the problem has returned. If all is well then the add-on you enabled is good to stay. Public Function CreateAppointment(sSubject As String, sBody As String, dtStart As Date, dtEnd As Date, bAllDayEvent As Boolean)
    On Error GoTo Error_Handler
'    'If your are using Early Binding to the Outlook Library then comment out the Late binding declarations and uncomment the following 2 lines
