Hi Gilles,

This issue was already existing when I started using Excel and that is quite some years now. There is nothing wrong with being helpful if it was done correctly, meaning that Excel should interpret the dates in the original format. That format should be included in the file.

An alternative could be that Excel do not convert values that look as a date if not all values can be converted. It becomes even worse when you open a file containing dates with VBA. VBA forces Excel to interpret all dates in M/D/Y format.

Looking to some dates, in a workbook you would see that the first 2 dates are rigth algned, the 2 others left aligned.

Date
04/07/2015
12/07/2015
12/13/2015
07/20/2015

Your formula =DATEVALUE(TEXT(A2;"m/d/yyyy")) will correctly convert it to 07/04/2015 & 07/12/2015 but TEXT can only convert values.

Excel considers the third and fourth date as text strings.

You can use following formula to convert all dates.

=IFERROR(DATEVALUE(TEXT(A2;"MM/dd/YYYY"));DATEVALUE(MID(A2;4;2) & "-" & LEFT(A2;2) &"-"&RIGHT(A2;4)))

The problem of shapes resizing and repositioning has been around for some time. As far as I have been able to determine, it is mostly prevalent when the screen zoom is NOT set to 100%. The work around is to set up all of your shapes and then save the information re Shape Name, Height, Width, Top and Left in another worksheet that can be hidden. Then set up event code for the worksheet activate and run code to resize and position all of the shapes to the saved data. If you need assistance to do this then upload a copy of your workbook to OneDrive and I will help further.

On your other question above re size and align etc , if you select the shapes you will see an additional ribbon called "Drawing tools" above the other standard ribbons. Select this ribbon and then in the "Arrange" block, select the "Align" icon with DropDown.

Guidelines to upload a workbook on OneDrive. (If you already use OneDrive and your process for saving to it is different then you can probably start at step 8 to get the link but please zip the file before uploading.)

  1. Zip your workbooks. Do not just save an unzipped workbook to OneDrive because the workbooks open with On-Line Excel and the limited functionality with the On-Line version causes problems.
  2. To Zip a file: In Windows Explorer Right click on the selected file and select Send to -> Compressed (zipped) folder). By holding the Ctrl key and left click once on each file, you can select multiple workbooks before right clicking over one of the selections to send to a compressed file and they will all be included into the one Zip file.
  3. Do not use 3rd party compression applications because I cannot unzip them. I do not clog up my computer with 3rd party apps when there are perfectly good apps supplied with windows.
  4. Go to this link.  https://onedrive.live.com
  5. Use the same login Id and Password that you use for this forum.
  6. Select Upload in the blue bar across the top and browse to the zipped folder to be uploaded.
  7. Select Open (or just double click). (Be patient and give it time to display the file after initially seeing the popup indicating it is done.)
  8. Right click the file on OneDrive and select Share.
  9. Select "Get a Link" from the popup menu.
  10. Click in the field displaying the link and it should highlight and then Copy and Paste the link into your reply on this forum. (I suggest that you avoid the "Copy" button on the "Get a link" screen because it introduces additional steps that are not required.)
