Remove the Windows update causing this
Control Panel > Programs and Features > Installed updates, find Windows update KB3035583, right click and Uninstall

 

Temporarily stop Windows updates being automatically installed
 Windows update > Change settings > Under Important updates, in the drop-down box choose Download updates but let me choose whether to install them > OK

 

To Prevent that update reinstalling

update > Check for updates, wait until it finishes> right click on KB3035583 and chose Hide, so it won’t not run again.

Windows update > Change settings > Under Important updates, in the drop-down box choose Install updates automatically (recommended)> OK

If you don't have the small hole for the paper clip, try restarting the computer and pushing the eject button as the machine is booting up.

 

If you have another computer that can accommodate that drive, try moving it to the other computer to rule out that the problem is related to the computer and not to the drive (though I find that hard to believe in any case).

 

After that, I'd contact technical support for the computer and/or the drive manufacturer to see if they have any suggestions or special tricks or instructions (they both may have various methods including phone, chat, email, support articles, manuals, articles, knowledgebase information, user guides, forums, and so forth - if still under warranty, clearly choose phone support - otherwise choose based on factors of speed and cost).

 

Finally, take the drive to a reputable computer repair shop (not Geek Squad or anyplace like that but a true professional) that can disassemble the drive to get to the disk.  Then the question becomes was it a fluke, was it something about that particular disk, was it an issue with that patricular type (DVD vs. CD vs CD-R) or brand (some disk brands have problems with some drives) or do you have a more problem with the drive and/or the drive's device drivers.  A skilled technician should be able to test things and give you the story.  Then all is well or you need to decide what to do from that point.

 

I hope this helps.

 

Good luck!
