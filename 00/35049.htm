a. user say:

Hi Joe,

 

I realize the inconvenience you are experiencing with Windows 10. I will certainly help you.

 

The desktop.ini file is a hidden file used to store information about the arrangement of a Windows folder. Essentially, if the layout or settings for a folder are changed, a desktop.ini file is automatically generated to save those changes.

Thumbs.db is a system file that stores previews of the images you save to a folder. It’s automatically generated by Windows Operating System and is supposed to be hidden, but sometimes it can turn up and get in your way. 

 

Please get back to us with an update on issue with Windows 10, we will be happy to help.

davem121 say:

If you have set hidden files to show in Windows Explorer - View Options you will be able to see these files on your desktop.

Nyima explained what these files store and that is the data that will be lost if you delete these files . . .

If you want you can indeed delete them without any adverse effects, but a better solution is to go to Windows Explorer (My PC) and uncheck the 'Hidden Items' on the View Ribbon.

These files were always there, even in previous version of Windows, it is just that Windows 10 makes it much easier to show Hidden Items
