Hi

=IF(AND($D$9="MEDIUM","HIGH"),MAX($F$9:$F9)+1,0)

Thanks for your help

Steve 

hansv mvp say:

The part

AND($D$9="MEDIUM","HIGH")

makes no sense. Each argument of AND must evaluate to TRUE or FALSE. $D$="MEDIUM" is OK, this is either TRUE or FALSE, but "HIGH" cannot be evaluated to TRUE or FALSE.

AND($D$9="MEDIUM",$D$9="HIGH")

would be valid, but it would always be FALSE, since D9 cannot be equal to "MEDIUM" and to "HIGH" at the same time.

Perhaps you want to return the max if D9 is either "MEDIUM" or "HIGH"? If so:

=IF(OR($D$9="MEDIUM",$D$9="HIGH"),MAX($F$9:$F9)+1,0)

or, equivalently,

=IF(OR($D$9={"MEDIUM","HIGH"}),MAX($F$9:$F9)+1,0)
