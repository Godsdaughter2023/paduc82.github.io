dale howard [mvp] say:

Bill --

This is what happens when you create a master project when the subprojects are not connected to a shared resource pool file.  If you have the name Mickey Cobb assigned to tasks in four of the subprojects, the master project thinks there are four resources, each of which is named Mickey Cobb.  The way to eliminate this problem would be to do one of two things:

  • Connect each of the subprojects to a shared resource pool file.  This means you would need to send the master project AND the shared resource pool file to to your subcontractors.  Frankly, I would not recommend that course of action for sharing the master project.
  • Create a static master project instead, for the sole purpose of sending to the subcontractors.  To do this, open a blank project and then click Project > Subproject.  In the Insert Project dialog, select the subprojects and then DESELECT the Link to Projects option near the bottom of the dialog.  Click the Insert button.  Microsoft will create a copy of each of the subprojects in the master project and will combine all of the resources so that you have one copy of each resource name.

For your own use, I would recommend the first option.  For sharing the master project with others, I would recommend the second option.  Hope this helps.

hansv mvp say:

Step 1: declare a public variable in a standard module (not in ThisWorkbook):

Public dtmNextTime As Date

Step 2: change your Timer macro to

Sub Timer()
    On Error Resume Next
    dtmNextTime = Now + TimeValue("0:4")
    Application.OnTime dtmNextTime, "SaveMe"
End Sub

Step 3: create a Workbook_BeforeClose event handler in the ThisWorkbook module:

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    On Error Resume Next
    Application.OnTime dtmNextTime, "SaveMe", , False
End Sub

The False as last argument stops the timer.

Warning: the timer will be stopped even if the user cancels closing the workbook.

andreas killer say:

In addition to Hans:

You can use some code to manage scheduled subs to make the handling easier. See the example code below.

Andreas.

Private Sub Example_Schedule()
  Debug.Print Now, "Example_Schedule"
  'Start 2 scheduled calls
  Schedule "Example_A_Schedule", Now + TimeSerial(0, 0, 1)
  Schedule "Example_B_Schedule", Now + TimeSerial(0, 0, 2)
  'This cancels only subroutine A:
  'Schedule "Example_A_Schedule", Schedule:=False
  'This cancels all scheduled calls:
  'Schedule ""
End Sub
Private Sub Example_A_Schedule()
  Debug.Print Now, "A_Schedule"
End Sub
Private Sub Example_B_Schedule()
  Debug.Print Now, "B_Schedule"
End Sub

Sub Schedule(ByVal Procedure As String, _
    Optional ByVal EarliestTime As Date, Optional ByVal LatestTime As Variant, _
    Optional ByVal Schedule As Boolean = True)
  'Manages scheduled macros
  Static Scheduler As Object
  Dim Items, Keys, i As Integer
  If Scheduler Is Nothing Then Set Scheduler = CreateObject("Scripting.Dictionary")
  If Procedure = "" Then
    Keys = Scheduler.Keys
    Items = Scheduler.Items
    For i = 0 To UBound(Items)
      Application.OnTime Items(i), Keys(i), Schedule:=False
    Next
    Scheduler.RemoveAll
    Exit Sub
  Else
    If Scheduler.Exists(Procedure) Then
      On Error Resume Next
      Application.OnTime Scheduler.Item(Procedure), Procedure, Schedule:=False
      On Error GoTo 0
      Scheduler.Remove Procedure
    End If
  End If
  If Not Schedule Then Exit Sub
  Application.OnTime EarliestTime, Procedure, LatestTime
  Scheduler.Add Procedure, EarliestTime
End Sub

If you wish to add people to your messenger friends, you need to send them an invite via the People page. Here are the steps:

norman david jones say:

Thanks

Hi Yasser,

Only visible sheets can be selected. To select all sheets, hidden or unhidden, it will first be necessary to unhide any hidden sheets.

On this premise, try:

'==========>>
Option Explicit

'---------->>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet

    Set WB = ActiveWorkbook

    For Each SH In WB.Worksheets
        SH.Visible = xlSheetVisible
    Next SH
    WB.Sheets.Select
End Sub
'<<==========

===

Norman
