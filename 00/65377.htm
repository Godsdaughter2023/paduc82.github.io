a. user say:

Hi Morfydd,

Thank you for posting your query on Microsoft Community forums.

I understand that your computer is not performing well due to the stated error message. However, I would like to know some more information to narrow down the issue and provide you with better assistance.

  1. Are you able to use the NFC (Near Field Communications) function or related software that uses this technology?
  2. When exactly do you get the error message?
  3. Does your computer stop responding only after this error or while using other applications as well?

If the error message pops-up at the system startup, then the issue may arise due to a third party software conflict with the driver. As you have updated the latest driver for the NXP NearFieldProximity Provider, I would suggest you to try the following additional methods and check if you get the issue.

Method 1:

To rule out the possibility of a third party software intervention, I would suggest you to boot the system in a selective startup and check if you get the issue. This process is called clean boot. If you do not get the get the issue, then you may restart the system again with a few services enabled to eliminate the specific third party service causing the issue. Refer to the steps provided in the following link to clean boot the system.

How to perform a clean boot in Windows

Note: Refer "How to reset the computer to start normally after clean boot troubleshooting" to reset the computer to start as normal after troubleshooting.

If the issue still exists, try the next method.

Method 2:

Try to run a System File Checker (SFC) scan to check for any system files corruption in the system that might be hindering the Windows updates to get installed. SFC scan will scan for corrupt system files on the computer and replace them.

  1. Press Windows key + X, click Command Prompt (Admin).

  2. In the Command Prompt, type the following command, and then press ENTER:
    sfc /scannow

For more information refer to this link:

http://support.microsoft.com/kb/929833/en-us

In addition to that, please verify if you have updated the driver from the correct link. If not, update the driver from the following link:

HP Consumer Support: Original NXP Near Field Proximity Driver

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.
