how do i find what apps are running - Microsoft Community

andrew hayman say:

how do i find what apps are running

how do i find what apps are running  and can i save my battery by turning them off ?

 

some time  my battery will last 2 days other 14/15 hours ?

lots say it apps running in the back ground  like more info on how I can look at them 

interopbyt say:

Because of the way Windows Phone handles apps that you suspend (tombstone), you don't need to close apps at all to save any battery.


When you open an app, say WPCentral, and press the Windows button on your phone, the app is tombstoned. The only thing the app consumes is an extremely small amount of RAM. It won't consume any CPU power. Unlike Android which lets any app still fully run in the background, Windows Phone just basically "freezes" them and stores it!


For your battery issue, I suggest checking all your Background Tasks. Background Tasks are things that run in the background periodically to update things like Live Tiles. In order to see them listed and disable the ones you don't think you need enabled, open up Settings -> Swipe to the left -> Tap on Background Tasks.


Another thing you can do is turn on "Battery Saver" in settings as well. Since this turns off ALL your background tasks such as automatic email syncing and app tasks, this is not recommended to be turned on at all time.


Final tip! Turn off all your phone radios that you don't need! For example, if you're never going to use NFC on your phone, turn off Tap + Send. If you won't ever use WiFi turn that off. Etc etc... :)

        .PatternColorIndex = xlAutomatic
        .Color = <your color here>
        .TintAndShade = 0
        .PatternTintAndShade = 0
    End With

You can generate the color number using the RGB() function, which takes three arguments ranging from 0 to 255, e.g.:

    .Color = RGB(red, green, blue)

For instance RGB(0, 0, 0) is black, RGB(255, 255, 255) is white, RGB(255, 0, 0) will be pure red, RGB(0, 255, 0) will be pure green, etc.

You can also color a font/cell (say, cell A1)  the way you'd like and type this command in the VB Editor's immediate window to return the color number:

   Debug.Print Range("A1").Font.Color

   Debug.Print Range("A1").Interior.Color
