a. user say:

The numbers actually stored in Excel are 1/100 of the value you actually see in the Excel cell. So you can either create a suitable column in Excel, or in Word, use the nested field

{ =100*{ MERGEFIELD Average_Deferral__with_NPs } \#0.00% }

For the nested field, all the { } pairs need to be the special field code brace pairs that you can enter using ctrl-F9 on Windows Word. You can use Alt-F9 to toggle between |field code view" and "field result view", and select the field codes and use F9 to update the results.
