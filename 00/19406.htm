ken sheridan say: Don't you simply need to open the report filtered to the BrgAuditID and ODType values from the parent form as it's these values which determine the rows returned in the subform?  If the report is opened from a button in the parent form say, and both columns are numeric data types:

Dim strFilter As String

strFilter = "BrgAuditID = " & Me.BrgAuditID & " And ODType = " & Me.ODType
DoCmd.OpenReport "qryInnerDimensions1", View:=acViewPreview, WhereCondition:=strFilter

For a text data type wrap the value in literal quotes characters, e.g.

" And ODType = """ & Me.ODType & """"

If opening the report from outside the form change each Me to a fully qualified reference to the form as a member of the Forms collection.
