Hi

Fir the MIN date

=MIN(IF($A$2:$A$100=$D2,$B$2:$B$100))

and the MAX

=MAX(IF($A$2:$A$100=$D2,$B$2:$B$100))

Both of these are ARRAY formula so see below for how to enter them. ARRAY enter both and drag down.

around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

Note that Excel stores date/time as an integer (number of days) and fraction (hours within that day) so your time conversions are as follows. While you can use these values directly in your formula, it is probably easier to just create a helper range. Put your time values in Z1:Z8 shown as what is in column A in my screenshot. Note that you will have to add the leading zero on the first two for Excel to realize you are entering a time.

Now it becomes easier to write your IF statement in B5 (aircode: check the number of parens)

=IF((B3-B2)<Z5,(B3-B2),IF((B3-B2)<Z6,(B3-B2)-Z1,IF((B3-B2)<Z7,(B3-B2)-Z2,IF((B3-B2)<Z8,(B3-B2)-Z3,

omer atay say: This is a known issue in OneNote 2010 with SkyDrive notebooks. The workaround is to NOT use the Hyperlink dialog (Ctrl + K) but instead just paste the link directly on the page. This will create a link with the page's title then you can right click and choose Edit Link to edit the text. This will correctly link to the page and clicking on the link will take you to the page within OneNote instead of opening the browser.
