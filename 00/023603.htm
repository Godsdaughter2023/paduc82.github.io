The way I did.

Private Sub Workbook_SheetSelectionChange(ByVal sh As Object, ByVal Target As Range)
   
   gvarPrevVal = GetValue(ActiveCell)
   
End Sub

Function GetValue(Optional ByVal Target) As Variant
   Static varOrigVal As Variant
    
   If Not (IsMissing(Target)) Then
      varOrigVal = Target
   End If
   GetValue = varOrigVal
End Function

Private Sub Workbook_SheetChange(ByVal sh As Object, ByVal Target As Range)

You have also have to declare the public variable gvarPrevVal as variant.

Selecting a cell triggers the SheetSelectionChange event and stores the value in the public variant.

The SheetChange event recalls the value stored in the static variable and will make it able to compare with the current value.
