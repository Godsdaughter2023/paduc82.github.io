Create index refering to paragraph number, not page number - Microsoft Community

zoowalk0815 say:

Create index refering to paragraph number, not page number

Hi,

I have a rather long (300 pages) document; every paragraph is automatically numbered (via style template); these paragraph numbers update automatically e.g if I delete a paragraph; I can also use them for cross-referencing.

I am able to create a key word index which provides the page numbers of each selected keyword. However, I would like to have the respective paragraph number, and not the page number. 

Note that I am not talking about a table of contents, but about the key word index.

Any help? Many thanks!

macropod say:

If your numbered paragraphs use a particular Style, you could add a \t switch plus a STYLEREF field to the XE fields to enable the Index to return the paragraph #s. For example, a standard index entry might look like:

{XE "MyEntry"}

If you add the \t switch plus a STYLEREF field to the XE field, thus:

{XE "MyEntry" \t {STYLEREF "MyStyle" \w}}

your index will then return the paragraph #s rather than the page #s.

Note: The field brace pairs (i.e. '{ }') for the above example are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. or you could insert them via Word's Insert|Quick Parts|Field dialogue. You could even use Find/Replace to apply the \t switch and STYLEREF field to multiple XE fields.
