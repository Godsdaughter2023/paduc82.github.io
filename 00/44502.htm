Hello,

There are several things that you can try if your Microsoft Store app is not working as it should be. Follow the recommendations provided in this support article to Fix problems with apps from Microsoft Store.

If the issue still persists, I suggest that you create a new user account with Administrator privileges. Refer to this support article for the steps on how to: Set up accounts.

The corrupted user profile does not function normally such as not launching Start menu, Cortana, Search bar, Store and other Windows applications. In such cases creating a new profile and checking for the same will help in identifying if the issue is specific to a corrupted user profile.

For now, you can change these settings below to change the default location on where to install Windows store apps:

Method 1: Change default storage location for apps

  1. Open Settings.
  2. Click on System.
  3. Click on Storage.
  4. Under Save locations, select the new drive location for "New apps will save to" option.

Also, you can proceed on installing the app on the available drive and then proceed to move it to your preferred drive:

Method 2: Move apps to another drive

  1. Open Settings.
  2. Click on System.
  3. Click on Apps & features.
  4. Select the app you want move, then click the Move button.
  5. Select the destination drive from the drop-down menu.
  6. Click the Move button to transfer the app to another drive.
