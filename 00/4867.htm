chip pearson say:

You can do it without code by using Conditional Formatting. See www.cpearson.com/Excel/Banding.aspx for details. If you want to do it with code, use code like the following:

 

Sub AAA()
    Dim R As Range
    Dim LastRow As Long
    Dim B As Boolean
    Set R = Worksheets("Sheet1").Range("A1") '<<< 1
    With R.Worksheet
        LastRow = .Cells(.Rows.Count, "A").End(xlUp).Row '<<< 2
    End With
    Do Until R.Row > LastRow
        R.Resize(4, 10).Interior.ColorIndex = IIf(B, 43, 6) '<<< 3
        Set R = R(5, 1)
        B = Not B
    Loop
End Sub

Change the line marked with <<< 1 to the sheet and cell in which the coloring should begin. Change "A" in the line marked with <<< 2 to the column that is used to find the last used row of the worksheet. Change the 10 in the line marked wtih <<< 3 to the number of columns that should be colored.  As written, the code will apply colors to A1:J10.  Change the 43 and 6 in the line marked with <<< 3 to the ColorIndex values you want to use. Here, 43 is a light green and 6 is yellow. Change these to the desired colors.
