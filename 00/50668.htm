xbf diego m say:

Hi ThalinneMafra,

Thank you for posting on the Xbox forums. This seems like a difficult issue to solve but I did some research and found that re-installing all of the Windows apps should help with this issue.

Perform the steps mentioned below and see if it helps: 

a: Press Windows key and search for PowerShell

b: Run the program as an Admin

c: Now type the following command and hit enter

Get-AppxPackage *windowsstore* | Remove-AppxPackage

Get-AppxPackage *xboxapp* | Remove-AppxPackage 

After a restart, enter the following command (again in a powershell in admin):

Get-AppxPackage -AllUsers| Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

This will install any missing built-in apps you may have uninstalled and potentially get rid of that pop up. Let us know the results!
