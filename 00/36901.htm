jessew - (ex)msft say:

Hi. Windows doesn't "let" Linux do anything. Once you reboot the computer to a different operating system such as Linux, it has complete control of the whole system. The Linux bootloader(grub) is actually nicer than the Windows bootloader because it can boot to different operating systems. That is until you rip out the supporting files from under it.

 

Anyway, if you want to fix this, you'll need to use the windows recovery tools which are on the windows setup disk. Put the DVD in your computer and reboot it. Select the "repair your computer" option. There should be an option to fix the bootloader. That should remove the rest of grub and replace it with the Windows bootloader.

 

To answer your other question of why Linux developer don't support a clean uninstallation of Grub, its because you can't uninstall a bpotloader -- you can only replace it with another one.
