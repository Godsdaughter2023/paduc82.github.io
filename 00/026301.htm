valley_girl1919, thank you for posting to the Microsoft forums.

By using "Parental Controls" you can setup the PC to regulate times allowed on the PC as well as block sites.

You can access the Parental Control settings through Control Panel:

Classic View: Click Parental Controls.

Category View: Click User Accounts and Family Safety, and then click Set up Parental Controls.

Click Internet Explorer, click Internet Options, click the Content tab, and then click the Parental Controlsbutton.

To set up Parental Controls, you must have an Administrator user account. Parental Controls cannotbe applied to an Administrator user account.

Before you get started, make sure that each customer for whom you want to set up Parental Controls has a standard user account. This is because Parental Controls can only be applied to standard user accounts.

To configure Parental Controls for each desired account, follow these steps:

1.Open the Parental Controls console.
 
2.Click the standard user account for which you want to set up Parental Controls
 
3.Under Parental Controls, click On.
 
4.Adjust the individual settings that you want to control.
 
5.Click On if you want to enable Activity Reporting.
 
When Parental Controls block access to a Web page or game, a notification is displayed that the Web page or program was blocked. The child can then click the Ask an administrator for permissionslink on the notification box. This will allow for the input of an Administrator user account’s name and password. The same notification occurs if a child tries to download a file by using Internet Explorer 7 and Block File Downloadswas enabled within the Web Restrictionsconfiguration.

If Parental Controls are set to enable file downloads, all downloads are logged for later review.

If a customer tries to access a page that has mixed content of data that is contained both on and off the approved list of Web sites, Internet Explorer 7 in Windows Vista displays an Information Bar to request permission from an administrator to approve the download.

When the whole page is blocked, an error page is displayed that also provides a link that lets the child request permission to view the page.

The Help documentation that accompanies Windows Vista provides an overview and how-to information on the Parental Controls features and functionality.
