charles kenyon say:
See What happens when I send my document to someone else? Will Word mess up my formatting?

It is rare for a lengthy Word document to appear the same on two different computers. Use of manual page breaks and blank paragraphs will amplify differences.

rohn007 say:
There are a few reasons.

First of all, those 2 versions use different "rendering engines", the underlying code used to display the document, so there are minor differences how characters are displayed.  This sometimes causes a word at the end of a line to move to the next line.  These extra lines are sometime enough to push the last line on the page to a new page.  As Charles said, if you use manual page breaks, then these "new pages" may only have one or 2 lines.

The only way to ensure that a document is viewed on 2 computers with exactly the same layout is to use a static format like PDF or XPS. Of course, then the file is no longer editable.

The insanity is not part of any master plan, just a happy happenstance.
