It turns out my inability to switch back to the original local account is just Windows 10 (or me) being stupid.  When I had logged into the Windows Store and been automatically switched to a Microsoft account for future logins, Windows kept the user name from the original Local account.  So when I later tried to switch back to a Local account using the original user name Windows got confused and couldn't do it.  In order to get around this, I had to enter a new dummy user name when selecting the option to switch from a Microsoft Account to a Local Account.  This allowed me to finish the process.  After my PC had been reverted to a Local account, I simply had to go into account options and change the account name to the original User Name.

fax and scan find name & network - Microsoft Community

tricia a marra say:

fax and scan find name & network

So easy but you make it so difficult.  Where do you find the name and network address for the fax and scan. I have tried every place and every set of numbers.  Why can't this installation not be so difficult.  Where do I find this information.  this is my computer and why is it like a secret .

sathish s v say:

Hello Tricia,

 

1.       How have you connected the printer to the computer, either through network or locally to the computer?

2.       What is the make and model number of the scanner?

3.       Are you referring to Windows Fax and Scan software?

 

You can find IP address of Fax and Scan using IPconfig command.

Follow these steps:

a.       Press Windows key + R” from the keyboard.

b.      Type “cmd” without quotes in the Run window and hit Enter.

c.       Type ipconfig /all in the command prompt and hit Enter.

d.      It will display IP address of all devices connected to the network.

e.      You can find Fax and Scan IP address there.

 

If you have installed printer or Scanner driver on the computer then it will display the name of the printer in Printer and Devices.
