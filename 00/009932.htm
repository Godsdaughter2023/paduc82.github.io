Thanks

john p spencer say:

I would extract the frames into a separate field.  One place for each type of data.

You can use the InstrRev function to parse out the sections.

Frames: Mid([TimeStamp],InstrRev([TimeStamp],":")+1)

ElapsedTime: Left([TimeStamp],InstrRev([TimeStamp],":")-1)

If you want to store the a point in time you can use TimeValue expression on the expression.  However, you really want to store the duration as a number of seconds.  Assuming no nulls for the value of the time stamp.  The easiest way to get the duration is an expression like the following.

TimeValue(Left(xTimeStamp,InstrRev(xTimeStamp,":")-1))*24*60*60
