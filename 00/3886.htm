Working Days & Time excluding weekly off and Working hours - Microsoft Community

dwarkanathbari say:

Working Days & Time excluding weekly off and Working hours

Dear all,

i need help regarding following issue i have

25/12/2015 11:05:00             27/12/2015 04:45:00

i want to calculate total days for the task excluding weekly off that is any day of week and total hours excluding my 8 working hours. 

Hope u all understand my requirement

chatmaker say:

This solution calculates LOE (Level of Effort) for a task in hours and displays in days and hours. 

NOTE! This solution requires XL 2010 or higher because it uses NETWORKDAYS.INTL and Structured References in XL's Tables.  To create a table simply enter its column headings, select any heading, press CTRL-T and check My table has headers.  To name a table simply select any cell in the table, then in the TABLE TOOLS DESIGN tab, enter the table's name in the Table Name: box on the tab's left.

Setup 1 - Create some meaningful names for our formulas:

  1. Create a table listing your holidays.  Name it tblHolidays
  2. Create a table with three columns: Description, Name, and Value.  Name it tblConstants
  3. In tblConstants' first row enter: Weekday Start, WDS, 9:00
  4. In tblConstants' second row enter: Weekday End, WDE, 17:00
  5. In tblConstants' third row enter: Weekend Type, WET, 1
  6. In tblConstants select Columns Name:Value and all rows (excluding headers)
  7. Use menu option FORMULAS > Create from Selection. In the dialog box check only Left Column

At this point we have some names we can use in our formula


tblConstants tblHolidays

Setup 2 - Create an entry area for calculating "LOE" (Level of Effort):

  1. Create a table with 6 columns: Start, Stop, TWFD, TWLD, Workdays, LOE. 
    TWFD means Time Worked First Day
    TWLD means Time Worked Last Day

  2. Name the table tblTasks

  3. Enter your start and stop dates in tblTasks

  4. In TWFD column enter this formula:  
    =MOD(MIN(INT([@Start])+WDE,[@Stop]),1)-MOD([@Start],1) 
    In English this is: In English this is: The lesser of (Workday's End or Stop time) less Start Time
    This gives us the number of hours worked the first day
    Use format [h]:mm:s

    NOTE: INT(DateTime) provides the date's day portion. MOD(DateTime,1) provides the date's time portion

  5. In TWLD column enter this formula: 
    =IF(INT([@Stop])=INT([@Start]),0,MOD([@Stop],1) - MOD(MAX(INT([@Stop])+WDS,[@Start]),1))
    In English this is: If Start and Stop are same day, 0, else Stop Time minus greater of Workday Start or Start Time
    This gives us the number of hours worked the last day
    Use format [h]:mm:ss

  6. In Workdays column enter this formula:
    =NETWORKDAYS.INTL([@Start],[@Stop],WET,tblHolidays)
    In English this is: Get the number of working days between start and stop excluding weekends and holidays
    See NETWORKDAYS.INT for an explanation of Weekend Types
    This gives us the number of workdays including the start and stop dates.  
    NOTE! This assumes Start and Stop are always during normal working hours.

  7. In the LOE column enter this formula: 
    =[@TWFD]+[@TWLD]+(MAX(0,[@Workdays]-2)*(WDE-WDS))
    In English this is: Add the time worked for first and last days to the number or workdays between start and stop multiplied by the hours between the workday start and workday end.
    Use format [h]:mm:ss

In general, project task LOE is calculated in hours and we want to sum LOEs in hours.  To display LOE in days, add another column with this formula: =INT([@LOE]/(WDE-WDS)) & " days " & TEXT([@LOE]-INT([@LOE]/(WDE-WDS))*(WDE-WDS), "[H]:mm:ss")

Sample: https://dl.dropboxusercontent.com/u/13737137/Projects/Gantt/LOE.xlsx

For more information related to project task scheduling see: Beyond Excel's PapaGantt

              ren C:\Windows\System32\catroot2 catroot2.old

              net start wuauserv
              net start cryptSvc
              net start bits
              net start msiserver

Step 5: Wait for the process to complete.
Step 6: Close the Command Prompt window.
