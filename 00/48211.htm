Hi,

Video issues with Surface Pro docking station is usually caused by loose connections.

Here are some things that you can check to further isolate the issue

  • Check the power light for the docking station is on when your trying to connect a secondary display.
  • Check if the cable is connected properly to the secondary display.
  • Take the Surface out of the docking station then put it back in.
  • Check if the issue only occurs when your device is docked.

You can also try rolling back display adapter driver. To do that, follow these steps:

  1. Open Device Manager.
  2. Expand Display adapters, right-click the driver, and then click Properties.
  3. Click the Drivers tab.
  4. Click Roll Back Driver.
  5. In the Driver Package rollback dialog box, click Yes.

If you're still having the same issue, update the Display adapter manually.

  1. Select the Start  button, type Device Manager, and select it from the list of results.
  2. Expand one of the categories to find the name of your device, then right-click (or tap and hold) it, and select Update Driver. For graphics cards, expand the Display adapters category, right click your graphics card and select Update Driver.

  3. Select Search automatically for updated driver software.

If these steps don't work, try reinstalling the driver: Open Device Manager, right-click (or tap and hold) the name of the device, and select Uninstall. Restart your PC and Windows will attempt to reinstall the driver.

david. f say:

Hello, Jose

Are you located in the US, UK or another region?
Internet TV is not available outside of the US/UK.

Are you using the built in Windows firewall or a different one?
What security software are you using?

If you are not using the built in Windows firewall, make sure that the executables in c:\windows\ehome\ are allowed through.

Additionally, try reinstalling Internet TV.

First, go to Start, Control Panel, Uninstall a program, then check if Internet TV for Windows Media Center is listed. If it is, select it and choose Uninstall.

Next, go to Start, All Programs, Accessories, right click Command Prompt then choose Run as Administrator. In the Command Prompt, run the following command and press enter: start /wait %windir%\ehome\mcupdate -uf -nogc

After that is finished running, open Media Center and check if Internet TV is available or not.
