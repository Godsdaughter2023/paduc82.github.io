charles kenyon say:
Are you opening documents from this web site at work as well? This sounds like a website issue.

It is seldom a good idea to open a web document directly into Word; it is a very good way to lose modifications. The reason for this is when you open a document from the Internet or email directly in Word, it does save a copy on your hard drive - in a temporary files folder! If you make charges and save, it gets saved to that temporary folder. Since this is a hidden system folder it can be hard to find later.

Better practice is to save to your hard drive, in a location you know, and open from that location.
