Column A of the Excel source file goes to column A of the .csv file, B goes to B, D goes to C and F goes to D. The Excel source file isn't always the same length (i.e. number of rows varies), but the data will always reside in the same columns. Is there a macro that can do that automatically?

Thanks for any help you can provide.

JP

bill manville say:

Sub ExportABDFtoCSV()
  Dim wsOld As Worksheet
  Dim wsNew As Worksheet
  Dim vFile
 
  Set wsOld = ActiveSheet
  Set wsNew = Workbooks.Add(xlWorksheet).Sheets(1)
 
  wsOld.Range("A:B").Copy wsNew.Range("A:B")
  wsOld.Range("D:D").Copy wsNew.Range("C:C")
  wsOld.Range("F:F").Copy wsNew.Range("D:D")
 
  ' replace this with setting vFile to a filename of your choosing if preferred
  vFile = Application.GetSaveAsFilename(FileFilter:="CSV files (*.csv),*.csv")
  If TypeName(vFile) = "Boolean" Then Exit Sub ' cancelled
 
  Application.DisplayAlerts = False
  wsNew.SaveAs vFile, FileFormat:=xlCSV
  Application.DisplayAlerts = True
  wsNew.Parent.Close False
End Sub

I'm not able to reproduce this space.

Here are some troubleshooting suggestions:

  • Does the space appear when the page is printed, or only on screen?
  • Does the space disappear when you change the type face, or its point size?
  • Does the space appear/disappear when you change the zoom setting?
  • Does it help to check or uncheck the box in the Options > Advanced dialog, in the Display section, for "Optimize character positioning for layout rather than readability"?
  • Does the space appear if you save the document as a PDF file? And if you print that from Adobe Reader or another PDF program?

jay freedman say:

Well, I did say those were troubleshooting steps, not solutions. :-)

The fact that the space doesn't transfer to PDF tells me that the problem might be in the printer driver that's selected as the Windows default. (Word uses data from the printer driver to do the screen layout.)

If you have a different printer available to the students' computers, try changing the Windows default printer on at least one of them, restart Word, and see whether that cures the space problem. If it does, you can either switch all of the computers, or check the printer manufacturer's web site for an updated driver.
