teylyn say:

Hello,

to find the third largest number in a range of numbers you can use the LARGE() function.

If you want to return the name of the sales person that this number relates to, you can use an Index/Match combination. 

Consider this data. The table starts in cell A1 with the label "Name"

Name value
John 1
Jim 2
Bob 3
Sam 4
Mary 5
Judy 6
Chris 7
Dana 8
Sara 9
Kate 10

 

You can get the third largest value of the numbers in B2:B11 with this formula:

=LARGE(B2:B11,3)

If you want to return the name in column A where the value in column B is the third largest number, you can use

=INDEX(A2:A11,MATCH(LARGE(B2:B11,3),B2:B11,0))
