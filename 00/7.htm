Function DeleteAllRelationships() As String
    Dim db As Database      ' Current DB
    Dim rex As Relations    ' Relations of currentDB.
    Dim rel As Relation     ' Relationship being deleted.
    Dim iKt As Integer      ' Count of relations deleted.
    Dim sMsg As String      ' MsgBox string.

    Dim k As Integer 

    sMsg = "About to delete ALL relationships between tables in the current database." & vbCrLf & "Continue?"
    If MsgBox(sMsg, vbQuestion + vbYesNo + vbDefaultButton2, "Are you sure?") = vbNo Then
        DeleteAllRelationships = "Operation cancelled"
        Exit Function
    End If

    Set db = CurrentDb()
    Set rex = db.Relations
    iKt = rex.Count
    For k = rex.Count - 1 To 0 Step -1

        If Left(rex(k).Table, 4) <> "MSYS" Then
           Debug.Print rex(k).Name
           rex.Delete rex(k).Name

        Else
           iKt = iKt - 1

        End If
    Next k
    DeleteAllRelationships = iKt & " relationship(s) deleted"
End Function

Word (MAC) 2011 - Find and Replace Specific Text within Square - Microsoft Community

codyporche say:

Word (MAC) 2011 - Find and Replace Specific Text within Square Brackets [123] *Help*

I am working on a document and need to find and remove all square brackets WITH a string of numbers between them (the data within the brackets and the square brackets themselves).

For example, I want to delete "[734]" from the following line in MS Word Mac 2011: I have a dog.[734]

Or, in other words, how do you go from

I have a dog.[12412441]
to
I have a dog.

?
Any help is appreciated. I searched for an hour before posting and nothing matched my needs exactly.
rich michaels - mvp (greatcirclelearning.com) say:
Go to Edit > Find > Advanced Find and Replace and in the Find box paste this code:

([\[])([0-9]@)([\]])


Leave the Replace box blank.


Open the full dialog by clicking on the down arrow button that is in the lower left corner of the dialog and in there check the Use Wildcards box. Execute the Find and Replace.


The code reads as follows:


Within the first set of parenthesis it's saying find the left bracket. Since in wildcard searches, brackets have special usage to designate the range to find, we have to repeat the bracket (since we're trying to find it) and place a backslash in front of it to tell Word that we are looking for this special character and not to think that it is designating a range.


Within the second set of parenthesis it is saying find any number 0 - 9 and the @ sign indicates to find as many as there are. So it would find a single number 0 or two numbers 01 or or three numbers 012 or four number such as 9999, etc.


Within the final set of parenthesis we finish the search string with the right bracket we are searching for.


Thus we look for: [ with as many numbers as there are ]


For help with wildcard search strings see Word MVP Graham Mayor's website.

ossiemac say:

From the code example it is not possible to determine for certain which sheets are being referenced.

Near the start of the code you have the following
Set CPsh = Sheet10
CPsh.Select
ActiveSheet.Unprotect

Is Sheet10 the new sheet that was activated, the sheet that was deactivated or another sheet?

It is not a good idea to activate sheets from within code that is in another worksheets module and then refer to the ActiveSheet. Each sheet that you wish to reference should be assigned to a worksheet variable and then use that variable in lieu of ActiveSheet.(It is almost never necessary to Activate a sheet or select a range; simply provide the full reference of worksheet name (or variable to which it is assigned) and the range reference).

I am suspecting the Sheet10 is the worksheet that was deactivated and your code is re-activating it and if so then the following code example below is a guide as to how it should be.

The code in untested so ensure that you have a backup of your workbook before attempting to test.

Private Sub Worksheet_Deactivate()

     Dim LastRow, LastRowA As Long
     Dim CPsh As Worksheet
     Dim wsNew As Worksheet
    
     Set wsNew = ActiveSheet    'This is the new active sheet after Deactivate
    
     Set CPsh = Sheet10
    
     'If Sheet10 is the worksheet to which this module belongs then the following code can be used in lieu of prev line.
     'Set CPsh = Me     'Me is the generic object for the worksheet to which the module belongs
    
     'CPsh.Select        'Don't select the sheet
     'ActiveSheet.Unprotect  'Incorrect. see next line
     CPsh.Unprotect         'Correct
    
     With CPsh     'Using With means you can just insert a dot without repeating the worksheet reference
        LastRowA = .Cells(.Rows.Count, "B").End(xlUp).Row
        LastRow = LastRowA + 25
       
        .Range("A4").Copy
        .Range("A4:A" & LastRow).PasteSpecial Paste:=xlPasteFormulas, Operation:=xlNone, _
         SkipBlanks:=False, Transpose:=False
    
        ' Copy Formula to Count # of services in current month
        .Range("E4").Copy
        .Range("E4:E" & LastRow).PasteSpecial Paste:=xlPasteFormulas, Operation:=xlNone, _
         SkipBlanks:=False, Transpose:=False
        
    
        ' Copy Formula to Count # of rows that need pricing
        .Range("F4").Copy
        .Range("F4:F" & LastRow).PasteSpecial Paste:=xlPasteFormulas, Operation:=xlNone, _
         SkipBlanks:=False, Transpose:=False
                
        ' Remove Duplicate Rows
        .Range("$A5:F" & LastRowA).RemoveDuplicates Columns:=Array(1, 2, 3, 4, 5, 6) _
         , Header:=xlNo
  
        ' Sort the page by Client & Charge Description
    
        .Outline.ShowLevels RowLevels:=0, ColumnLevels:=2
       
    End With
       
    'In following lines of code is "Client Pricing" also Sjheet 10 (or CPsh)?
     'If so then use CPsh in lieu of ActiveWorkbook.Worksheets("Client Pricing")
     ActiveWorkbook.Worksheets("Client Pricing").Sort.SortFields.Add Key:=Range( _
         "B5:B" & LastRow), SortOn:=xlSortOnValues, Order:=xlAscending, DataOption:= _
         xlSortNormal
     ActiveWorkbook.Worksheets("Client Pricing").Sort.SortFields.Add Key:=Range( _
         "C5:C" & LastRow), SortOn:=xlSortOnValues, Order:=xlAscending, DataOption:= _
         xlSortNormal
     ActiveWorkbook.Worksheets("Client Pricing").Sort.SortFields.Add Key:=Range( _
         "D5:D" & LastRow), SortOn:=xlSortOnValues, Order:=xlAscending, DataOption:= _
         xlSortNormal
     With ActiveWorkbook.Worksheets("Client Pricing").Sort
         .SetRange Range("A5:F" & LastRow)
         .Header = xlNo
         .MatchCase = False
         .Orientation = xlTopToBottom
         .SortMethod = xlPinYin
         .Apply
     End With
                
        
 ' Scroll to the top of the page & make sure column A is hidden
     Application.GoTo wsNew.Range("b1"), True   'Specify the worksheet
    
     With wsNew     'this is the new active sheet
        .Rows("4:4").EntireRow.Hidden = True
        .Outline.ShowLevels RowLevels:=0, ColumnLevels:=1
    
    
        Application.CutCopyMode = False
    
        .Protect DrawingObjects:=True, Contents:=True, Scenarios:=True _

    End With

 End Sub
