stefan blom say:

Normal.dotm is called the blank document template because it has no (visible) content. It is supposed to be just a blank page when you open it. You can still modify the styles and/or document defaults that are defined in the template.

Note that you should never try to overwrite Normal or replace it with another file called "Normal.dotm"; instead, you can open it as a document and make changes.

You can of course make any changes that you like to a template (add content, change page setup etc.) but usually it is better to set up a custom template for that purpose and leave Normal.dotm blank. 

The quickest way to get rid of any changes in Normal.dotm and return it to factory defaults is by exiting Word, renaming the template; then restart Word.
