Hi Cintlex,

Something may be causing your hard drive to over work that's why you're getting a 100% disk usage. To work on this problem, please go through the following solutions to determine and address the issue. Make sure to check your disk usage every after performing each of the suggested troubleshooting.

Perform a scan with your antivirus software.

If you have Windows Defender Antivirus installed, please perform a scan to determine whether or not a virus or a malware is causing the issue. Please see Scan an item with Windows Defender Antivirus for the detailed steps.

Disable Windows Search.

Windows Search makes searching files on your computer much faster. However, frequent use make cause slow PC performance as its indexing the searched files. To temporarily disable Windows Search, follow these steps:

  1. Right-click the Windows key and select Command Prompt (Admin).
  2. In the command prompt window, type in net.exe stop "Windows search" and press Enter.

In case the disk usage still the same, then try to permanently disable Windows Search. Here's how:

  1. Press Windows key + R.
  2. In the Run field, type in services.
  3. In the Services (Local) list, locate Windows Search, right-click it, and select Properties.
  4. Set the Startup type to Disabled.
  5. Click Apply, then OK.

Disable SuperFetch.

Superfetch helps decrease boot time and makes programs load efficiently. However, it may cause high disk usage. Follow these steps to disable the service:

  1. Right-click the Windows key and select Command Prompt (Admin).
  2. In the command prompt window, type in net.exe stop superfetch and press Enter.
