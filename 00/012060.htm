    ken sheridan say: I haven't looked at your file, but the way to carry forward selective data from one record to a new record is quite simple.  It's just a matter of assigning the value of each control whose value you wish to carry forward to the DefaultValue property of the control.  In your case you appear to also want to increment the Complaint No value by 1.

    The first thing to do is set the Tag property of each control whose value you wish to carry forward to something suitable, e.g. CarryMeForward. The code would then be like this:

    Dim ctrl As Control

    For Each ctrl In Me.Controls
        If ctrl.Tag = "CarryMeForward" Then
            If ctrl.Name = "Complaint No" Then
                ctrl.DefaultValue = """" & ctrl+1 & """"
            Else
                ctrl.DefaultValue = """" & ctrl & """"
            End If
        End If
    Next ctrl

    DoCmd.GoToRecord acDataForm,Me.Name,acNewRec

    Note that the DefaultValue property is always a string expression, regardless of the data type of the column to which the control is bound, so is always wrapped in literal quotes characters as above.
