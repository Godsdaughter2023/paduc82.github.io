how do i alternate between tabs - Microsoft Community

brendaswasbrick say:

how do i alternate between tabs

how do i alternate (quick key) between tabs on internet explorer 8?
robert aldwinckle on forums say:
how do i alternate (quick key) between tabs on internet explorer 8?


Press Ctrl-Tab and then press Ctrl-Backtab (Ctrl-Shift-Tab)

If you have changed the definition of those keys (in Options Advanced tab) the tabs you would be alternating between would not necessarily be adjacent.   Then another way to alternate would be to identify them by number, between 1 and 9 and press Ctrl and that number.   For example first can be entered by pressing Ctrl-1 and last can be entered by pressing Ctrl-9, regardless of the meaning of Ctrl-Tab and regardless of how many tabs you have opened.   BTW don't hit Ctrl-0 by mistake when you aim for Ctrl-9; that just resets your Zoom setting for the current tab.   ; )

If you're just trying to avoid using your mouse for switching tabs and don't want to go to the bother of toggling that Advanced option one way or the other, you can probably get keyboard focus into the Tab label by pressing various combinations of Alt-d, F6, and Tab or Backtab.   For example,  Alt-d, F6, Backtab works for me.   It probably depends on what all you have enabled in your Toolbars and where your Tabs are.   In any case once you have keyboard focus in a Tab label (signified by seeing dotted lines around it) you can navigate to adjacent tabs by pressing CursorRight or CursorLeft.


IMO that toggle about whether Ctrl-Tab moves between adjacent tabs or in most recently used order should be more accessible, ideally have a shortcut key of its own.   That would actually be possible by creating a script which modified the registry value appropriately, creating a Desktop shortcut (aka .lnk file) for it and assigning a keyboard shortcut to that.  I haven't got around to doing that yet but have been thinking about it.   In the meantime the quickest way I have for doing that switch is pressing  Alt-T,O,Ctrl-Backtab,H,H,CursorUp,CursorUp,CursorUp,Spacebar.   Come to think of it now that is fairly well defined (just not well defined time-wise) so an alternative to my cmd script idea would be to create a keyboard macro based on that sequence,   I could also create a script using SendKeys() with that sequence instead of  needing to know the registry value name for  reg.exe  but if I was going to go as far as writing a script I would use a reg.exe implementation.

 

FYI

Robert Aldwinckle
