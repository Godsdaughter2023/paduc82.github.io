searcher_303 say:

How do I remove a blank page in MS Word 2010?

I have created a table using MS Word 2010, but the table ran over onto a second page , which i do not want.

I have shrunk down the table on the second page but cannot completely delete it, and in any case I want to remove this second page altogether.

I would be grateful for nay help with this. 

jay freedman say:

When a table is the last text in a Word document, there is a paragraph mark below the table that cannot be deleted. If the table goes all the way to the bottom of its page, that paragraph mark is forced to a page that's otherwise empty. You won't see it unless you turn on nonprinting characters with the Show/Hide button on the Home tab.

To get rid of the blank (not really empty) page, you have to fit that paragraph mark on the same page as the table. The way that usually works is to select the paragraph mark and format it as 1 pt font size. For directions, if you need more than that, go to http://wordfaqs.mvps.org/BlankPage.htm.

suzanne s. barnhill say:
If it is a table row, you should be able to remove it by selecting, right-clicking, and choosing Delete Rows or by selecting and pressing Backspace.
