rob koch say:

Since there would be localized versions of the MSRT delivered to different version of Windows or in various countries, it's possible that at some point your system downloaded a version localized for the Arabic language.

Were you ever traveling in a region where Arabic is spoken or might your copy of Windows ever have had the language or region selection changed by mistake, even temporarily to something like the "United Arab Emirates" for example?  Any of these actions could have caused Windows Update to attempt download of a localized version of such an automatic update in the Arabic language.

The download file you reviewed on the Microsoft download site is compressed and the current version found on my own Windows 8.1 x64 based system is displayed as 91.039 KB, though the true size found in the file's Properties, Details tab is 88.9 MB.

I doubt there was ever a real problem here and I'm assuming the "Allow" you found within AVG is related to either a firewall or other networking/download monitor it contains which logged the event?  In this case, if the entry can be deleted I would simply do that so if there is a prompt generated when these occur it will display if this happens again in the future.

In any case I highly doubt that anyone creating malware would attempt to use something so similar to such a well known Microsoft file, since verification that it is a valid version would be quite easy for antimalware to perform via the Details and Digital Signatures information embedded in the file.

jacques raubenheimer say:

You are confusing the operation of the Office Clipboard with the operation of Excel's CutCopyPaste mode.

When you copy/cut cells in Excel, you go into CutCopyPaste mode and must paste what you have copied/cut before you can continue working. If you do start working, you will exit CutCopyPaste mode and not be able to paste any more.

The exception to this is that when you are using the Office clipboard, and you copy/cut some cells, they are copied/cut to the Excel clipboard (same rules apply as mentioned before) as well as to the Office clipboard (separate memory store, different rules). You can paste anything from the office clipboard whenever you want (almost, as you have seen!).

So.

Press Esc. so that you are no longer using the Excel clipboard (dashed lines will disappear). Go to B27. click on the Notepad formula in the Office clipboard to paste it.
