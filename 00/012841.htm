how to express numerical ranges, (e.g. 1-5) within a cell that - Microsoft Community

newtsc say:

how to express numerical ranges, (e.g. 1-5) within a cell that can be used in a formula

Trying to create a lookup formula in Excel which references a numerical range within a cell. For example how can I express a range of 1-5 in a cell so that when that cell is referenced in a lookup formula that is calculating for the number 2 it will deliver a result. If 1-5 oranges cost 50¢ each and 5-10 oranges cost 40¢ each, how do I express the numerical ranges so that if my formula is calculating lookup costs for 2 oranges, it would identify the cost at 50¢ each.
Thanks for any help!
j.e. mcgimpsey say: Your question is ambiguous in at least a couple of ways...

First, and trivially, what should the answer be when looking up 5 oranges? 50 cents? or 40 cents?

Second - does the rate for 10 (40 cents) apply to ALL of the oranges (i.e., 10*0.40=$4.00), or should the 6-10 rate apply only to the last five (i.e., 5*0.5+5*0.4= $4.50)?

If the former, enter a quick table:

    A          B
1   0       0.5
2   6      0.4
3   11     0.38

etc.  

Then, with the quantity in J1, use

   =J1*VLOOKUP(J1,A1:B3,2,TRUE)

The TRUE parameter tells VLOOKUP to make an approximate match - that is lower than or equal to the target value

If it's the latter, there are several different ways to approach it. One way if you have just a few price breaks is to explicitly check each range:

    =MIN(J1,5)*0.5+MIN(5,MAX(J1-5,0))*0.4+MAX(J1-10,0)*0.38

Alternatively, you could use one of the techniques found here

   :http://www.mcgimpsey.com/excel/variablerate.html

e.g., using the above data:

   =SUMPRODUCT(--(J1>{0;5;10}), --(J1-{0;5;10}), {0.5;-0.1;-0.02})

or, after rejiggerig the above table a bit:

     A     B
1  0     0.5
2  5     -0.1
3  10   -0.02

  =SUMPRODUCT(--(J1>A1:A3),--(J1-A1:A3),B1:B3)

a. user say:

Perhaps something like this:

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
    Dim WS As Worksheet
    For Each WS In ThisWorkbook.Worksheets
        WS.Range("A1") = Format(Now(), "dd. mm. yyyy hh:mm:ss")
    Next WS
End Sub

> ...  is there a better way except me inserting the date as soon as i update the file manually each time?

This will fire when a cell on the sheet is updated.

    Debug.Print Target.Address(0, 0) & " was updated"
End Sub

tom ogilvy say:

necrophyte,

If you don't want to loop through the worksheets, then Excel does offer an option to do this without looping and without grouping sheets:

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
With ThisWorkbook
 .Worksheets(1).Range("A1").Value = Now
 .Worksheets(1).Range("A1").NumberFormat = "dd.mm.yyyy hh:mm:ss"
 .Worksheets.FillAcrossSheets Worksheets(1).Range("A1"), xlFillWithAll
End With
End Sub

mary louise herrera say:

Word 2010 adding text next to picture

Once I insert a picture into a Word document, how do I add text immediately to the right/left of the picture and CENTER it?
marysully say:

There are a couple of different methods to adding text to the right or left of a picture you want to center. You can draw a text box to the left or right of the picture, insert text, and then remove the textbox border if needed. Set Wrap style for both picture and textbox to “tight” and you can position them anywhere in the document.

A simpler method is to create a 1 row, 2 column Table. Insert picture into first column, and text into second column. Select table and center on page and remove Table borders if needed.
