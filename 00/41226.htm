You can definitely do that. You'll need disk imaging software.

A disk image is an exact copy of a hard drive (or a partition) including all the metadata. The copy is stored in a compressed file. Once you get a larger hard drive, you restore the image and make it as big as you want.

On the one hand, disk imaging is a common procedure and it's very safe. On the other hand, it's tricky to understand. I would advise you to spend all the time you need to really understand the whole procedure.

Most new hard drives today come with a version of disk imaging software and instructions on how to use it. I would still advise you to do your research and really understand what you're doing before you start. That way, you'll get it done right the first time.

New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash.
