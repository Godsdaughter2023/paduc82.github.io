ronrosenfeld say: You seem to be having problems based on your data set up.  I provide two VBA User Defined Functions, one of which should work.  The first extracts from your original string, the first substring that "looks like" an email address.  That is defined as a bunch of allowed characters, with an "@" in the middle, and with a termination of "dot" followed by 2 to 6 characters.

The second function merely extracts the last "word" in the string, where words are delineated by spaces or their equivalents.

To enter these User Defined Function (UDF), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this User Defined Function (UDF), enter a formula like

=EmailAddr(A1)  or
=LastWord(A1)

in some cell.
======================================
Option Explicit
Function EmailAddr(s As String) As String
    Dim re As Object, mc As Object
Set re = CreateObject("vbscript.regexp")
With re
    .Pattern = "\b[A-Z0-9'._%+-]+@(?:[A-Z0-9-]+\.)+[A-Z]{2,6}\b"
    .ignorecase = True
    .Global = True
End With
If re.test(s) = True Then
    Set mc = re.Execute(s)
    EmailAddr = mc(0)
End If
End Function

Function LastWord(s As String) As String
    Dim re As Object, mc As Object
Set re = CreateObject("vbscript.regexp")
With re
    .Pattern = "[^\s\xA0]+"
    .Global = True
    If .test(s) = True Then
        Set mc = .Execute(s)
        LastWord = mc(mc.Count - 1)
    End If
End With
End Function
