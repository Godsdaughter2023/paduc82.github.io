diane poremsky mvp say:
This is with an exchange account? It's apparently a bug in the December update.

This fixed the missing From field in Search for other affected users:
If you have Outlook.com or Exchange and can edit your registry, see if setting this key fixes the display.
Key: HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Outlook\Search
REG_DWORD: DisableServerAssistedSearch
Value: 1
