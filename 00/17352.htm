john korchok say:

You can get this effect if you change your user name even slightly in Word>Preferences>User Information.

But if you haven't edited your user name, you can still get a color change. The default settings (Word>Preferences>Track Changes) automatically applies a different color to each different author. But since Word is picking the color on the fly, it can choose a different color after you close and reopen the document. To prevent this, open Word>Preferences>Track Changes and change the default By Author choice to a specific color. Then that color will always display on your computer. The color is likely to change when the document is displayed on a different computer, which may have different colors set.

I don't want the title page of my academic paper to be page one. I want to change the page numbers so that the first page of text is page one, not the cover page.

I am typing an academic paper with a cover page.   Word automatically gives the title page the number 1.  I want the first page of text to be number 1 and so on.

How do I fix this?

THANKS!

daniel g. grau say:
For this, Jacqueline, you'll need use sections. Each section is associated to a page size, margins, number of columns, page orientation and a set of headers and footers. By default, a new document only contains one section and thus one set of headers and footers (with or without page numbers).

In order to do what you want:
  1. Place your cursor at the point where you want your page 1 to start.
  2. In the main menu, select Insert > Breaks> Section Break (Next page). You will now have two sections—one for the cover page and another for the rest of the document.
  3. Select View > Header and Footer to go to the header or footer of the second section.
  4. You will notice a purple Headers and footers tab in the Ribbon. Click on it.
  5. Now disable the checkmark next to "Link to previous." This will make the header or footer on the second section independent from the ones in the first section. Do this for the header, the footer or both, depending on whether you need them to repeat or not.
  6. Now place your cursor in the header or footer where you want your page number.
  7. In the main menu bar, select Insert > Page Numbers, click on the Format button and set the number to "Start at 1." This will make section two begin with page #1.
  8. Finally, go the first page of the document (the cover page) and delete whatever material you don't need in the header and/or footer.
This looks complicated, but if you follow the steps in the order in which they are required, it won't be difficult. This is how you would normally proceed:
  • Type the cover page
  • Insert a Section Break
  • Go to the header/footer of the second section
  • Make them independent (purple tab & checkmark)
  • Type the header footer text and insert the page number
  • Reset it to 1

You should have a field construction like:

{IF{MERGEFIELD MyFieldName}= "" "" "output text"}

or:

{IF«MyFieldName»= "" "" "output text"}

Note: The field brace pairs (i.e. '{ }') for the above example are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Likewise, the chevrons (i.e. '« »') are part of the actual mergefields - which you can insert from the 'Insert Merge Field' dropdown (i.e. you can't type or copy & paste them from this message, either). The spaces represented in the field constructions are all required.
