Uninstall Upgrade Assistant In Settings > Apps & Features.

Clear out any Upgrade folders in C: drive root.

Disable "UpdateAssistant" and "UpdateAssistantCalendarRun".

Restart PC, turn back on Windows Update service,

Now you can manually run the Upgrade by installing Media Creation Tool to Upgrade Now: http://windows.microsoft.com/en-us/windows-10/m...
