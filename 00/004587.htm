mike h.. say:

Hi,

 

Try this. Change to column letter where indicated in the code to the one with Yes/No in

 

 

Sub Copy_Data()
 Dim r As Range, LastRow As Long
 Dim LastRow1 As Long, MyColumn As String
 Dim src As Worksheet, dst As Worksheet
 Set src = Sheets("Sheet1")
 'Change to Column with Yes/No in
 MyColumn = "F"
 LastRow = src.Cells(Cells.Rows.Count, MyColumn).End(xlUp).Row
 For Each r In src.Range(MyColumn & "1:" & MyColumn & LastRow)
    Set dst = Nothing
    If UCase(r.Value) = "YES" Then
        Set dst = Sheets("Sheet3")
    ElseIf UCase(r.Value) = "NO" Then
        Set dst = Sheets("Sheet2")
    End If
    If Not dst Is Nothing Then
        LastRow1 = dst.Cells(Cells.Rows.Count, MyColumn).End(xlUp).Row
        src.Rows(r.Row).Copy dst.Cells(LastRow1 + 1, 1)
    End If
 Next r
 End Sub

Hi,

 

This now wont copy rows to either sheet if they're there already. You still need to change Col F to the corect one for Yes/No

 

Sub Copy_Data()
 Dim r As Range, LastRow As Long
 Dim LastRow1 As Long, MyColumn As String
 Dim src As Worksheet, dst As Worksheet
 Dim YesRange As Range, NoRange As Range
 Set YesRange = Sheets("Sheet3").Columns(7)
 Set NoRange = Sheets("Sheet2").Columns(7)
 Set src = Sheets("Sheet1")
 'Change to Column with Yes/No in
 MyColumn = "F"
 LastRow = src.Cells(Cells.Rows.Count, MyColumn).End(xlUp).Row
 For Each r In src.Range(MyColumn & "1:" & MyColumn & LastRow)
    Set dst = Nothing
    If UCase(r.Value) = "YES" Then
        If IsError(Application.Match(src.Cells(r.Row, 7).Value, YesRange, 0)) Then
            Set dst = Sheets("Sheet3")
        End If
    ElseIf UCase(r.Value) = "NO" Then
        If IsError(Application.Match(src.Cells(r.Row, 7), NoRange, 0)) Then
            Set dst = Sheets("Sheet2")
        End If
    End If
    If Not dst Is Nothing Then
        LastRow1 = dst.Cells(Cells.Rows.Count, MyColumn).End(xlUp).Row
        src.Rows(r.Row).Copy dst.Cells(LastRow1 + 1, 1)
    End If
 Next r
 End Sub

kiay say:

Hi Sujith,

To set CSM on Boot option page to UEFI only, 

Click CSM and set to enable. 

Three more options (PXE, Storage, Video) will appear on CSM page. 

Click the button on the right of each option and set it to UEFI only.

Then go back to Advance/CPU configuration page and set Active Processor cores to All.

Save and reboot.  

If you have trouble booting into Windows after the changes. Either revert to the previous settings or try setting the Storage in CSM to Legacy Only if your hard disk was created in MBR not UEFI GPT format and try again.
