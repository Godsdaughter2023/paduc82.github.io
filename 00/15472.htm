This is a many to many relationship.  Owners is a table.  Equipment is a table. My many to many is EventsEquipment table.

hansv mvp say:

Set the Row Source of the second combo box to a query (the following is "air code", you'll have to subsitute the correct names)

 

SELECT EventsEquipment.EquipmentID, Equipment.EquipmentName FROM EventsEquipment INNER JOIN Equipment ON EventsEquipment.EquipmentID = Equipment.EquipmentID WHERE OwnerID = Nz([Forms]![MyForm]![OwnerComboBox], 0)

 

Requery the second combo box in the After Update event of the first one:

 

Private Sub OwnerComboBox_AfterUpdate()
    Me.EquipmentComboBox.Requery
End Sub
