john visio mvp say:

PowerPoint, Excel and Word use SmartArt that looks like and acts like Visio shapes, but they lose this look alike nature if you move them into Visio. So though it may look like you can edit a Visio diagram in PPT/Excel/Word, but they are two different beasts.

So you can import your drawings into PowerPoint and then let them draw over top of your drawing using PPT's SmartArt. It is not a perfect solution, but they could place white borderless shapes over the parts of your drawing they want to "delete" and then SmartArt on top of these shapes. Basically use the bordless shapes as a form of using WhiteOut.

===============================================
Once the PDF is created...it can only print what is on the page.

Printing provides different options than just simply saving.

I use a freeware called PrimoPDF for generating PDF files.

When printing in Publisher 2010...if I go to File / Print and choose...
PrimoPDF as my Printer...it will create a .PDF that looks just like
the Print Preview.

After clicking the Print button in Publisher...the PrimoPDF screen
will appear...left click...Create PDF button and the Save As screen
should appear. Name the file and choose a save location then left
click the Save button.

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

Download and install the freeware PrimoPDF
http://www.primopdf.com/
(the file name to download is  :  InternationalPrimoPDF.exe)

I have not discovered a way to accomplish this result using the
.PDF option in Publisher.

===

Another alternative would be to save your Publication as a Picture...
.png would be a good choice.

Then open a new instance of Publisher...and insert two copies of the
saved .png on a single full sized page. If you save this as a PDF you
will have the result you asked for.
