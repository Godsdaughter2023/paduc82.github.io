geethu b say:

Hi Michael,

Windows sets the initial minimum size of the paging file equal to the amount of random access memory (RAM) installed on your computer, and the maximum size equal to three times the amount of RAM installed on your computer.

However if you want change the Virtual Memory, you may follow these steps.

a. Right-click on This PC and then click Properties.

b. In the left pane, click Advanced system settings.

c. On the Advanced tab, under Performance, click Settings.

d. Click the Advanced tab, and then, under Virtual memory, click Change.

e. Clear the Automatically manage paging file size for all drives check box.

f. Under Drive [Volume Label], click the drive that contains the paging file you want to change.

g. Click Custom size, type a new size in megabytes in the Initial size (MB) or Maximum size (MB) box, click Set, and then click OK.

If you are using 64 bit version of Windows 8.1, you may refer to the article for additional information.

How to determine the appropriate page file size for 64-bit versions of Windows
