hansv mvp say:

Filtering is done by setting the row height of rows not matching the filter criteria to 0, so auto-fitting all rows will undo the filter.

To prevent this, select only the visible rows:

  • Select the entire sheet.
  • Press Alt+; (semi-colon) to select only the visible cells.
  • Double-click a row border.
