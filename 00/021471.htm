a. user say:

Hi Fermat,

 

You can create retention policy in EAC(Exchange Admin Center) and you can only create one policy and apply it to the specific mailboxes you want and according to your condition you don’t need to create a second one. About detailed steps to create and use retention policy, please refer to https://technet.microsoft.com/en-us/library/dd297955(v=exchg.150).aspx and https://technet.microsoft.com/en-us/library/jj150573%28v=exchg.150%29.aspx?f=255&MSPPError=-2147217396  So in a nutshell, firstly you need to create retention tags, secondly you need to create retention policy and add the tags to the specific policy, finally you need to apply the policy to the specific mailboxes.

 


 

For your second question, both E3 and BP plan can use retention policy. But each plan can only use the policy to its own licensed users.

 

For anything unclear, please let me know.
