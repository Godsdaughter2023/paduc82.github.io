gil deg say:

Hi,

Please be informed that you cannot disable Windows Defender Smart Screen for a specific application.

But there are three different Windows SmartScreen filters, and you can configure separate options for each.

Select “Block” to block unrecognized applications, “Warn” to view a warning you can click through, or “Off” to disable Windows SmartScreen entirely. Even if you have “Warn” enabled, SmartScreen will always block known-dangerous content—it’ll just warn you before running unrecognized applications.

If you disable SmartScreen entirely, SmartScreen won’t be able to block known-dangerous files.

SmartScreen settings are now located in the Windows Defender Security Center interface. Launch the “Windows Defender Security Center” shortcut in your Start menu to open it. Click the “App & browser control” icon in Windows Defender’s sidebar to find the settings.
