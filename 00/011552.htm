dirk goldgar say:

I have a date field on a form and I want a simple way to enter a date.  I want to be able to enter either one or two digits for the month and day and 2 digits for the year with a '/' between.  If I use the format mm/dd/yy it wants me to enter two digits for everything.  If I use the Short date format it shows a 4 digit year, I only want a two digit year.

Is there a answer to this?  I'm using Access 2000.


Are you primarily concerned with how the date is *displayed*, or how it is *entered*?  If you specify a format (in the Format property of the text box), you can still enter the date any way you want, in any form that Access recognizes as a date, and when you leave the control the date will be displayed according to the format you specify.  So for example you could specify this in the Format property:

    m/d/yy

and the date will be displayed with 1 or 2 digits for the month (as needed), 1 or 2 digits for the day (as needed), and 2 digits for the year.  This will be true regardless of how you actually enter the date, and no matter how many digits you enter for the month and day.

If you decide to use an input mask to tightly control date entry, things get more complicated.  However, you may not need to concern yourself with that, if all you really care about is how the date is displayed after it is entered.
