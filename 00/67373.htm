technosticrati say:

Windows Driver for Jabra Cruiser2

I recently ran into an issue with the driver on my Jabra Cruiser2 hands free bluetooth device when connecting via USB to my Windows 8 PC. 

Here's the solution: 


1. Download the Jabra Cruiser2 DFU Mode driver at this link:

https://bit.ly/1dEwLwW

(Make sure to download the entire zip folder from the file menu not the individual files)

2. Open up contents of zip file to your desktop or copy and paste them to your desktop once you have the file open.
3. After this is done connect the USB to your PC and plug into the Jabra unit with it turned off.
4. Next hold down the volume up and down buttons at the same time and turn on the unit.
5. Generic bluetooth driver will install automatically. (This step could change depending on if it allows you to install your own driver or automatically installs a generic bluetooth driver like it did to me.)
6. Now go to Control Panel\System and Security\System on your PC. 
7. Click and open Device Manager.
8. Locate Generic Bluetooth Device and right click on this device, which will allow you to go to Properties and intall/replace driver. 
9. Locate new driver file on your desktop and install.

If you already have the Jabra Firmware you can skip step 10 and go to 11.  

10. Download Jabra Firmware to desktop here: http://www.jabra.com/support/jabra-cruiser2_100-47200000-02
11. Open Firmware zip folder and run update. 
12. Success!

    If the first parameter has the same value as the fourth parameter, and the third parameter indicates an execute operation, this bug check was likely caused by a driver that was trying to execute code when the code itself was paged out. Possible causes for the page fault include the following:

    • The function was marked as pageable and was running at an elevated IRQL (which includes obtaining a lock).

    • The function call was made to a function in another driver, and that driver was unloaded.

    • The function was called by using a function pointer that was an invalid pointer.

    Services and Features you can disable on Windows 7 - Microsoft Community

    robin_cox say:

    Services and Features you can disable on Windows 7

    Hey all, this is a very common question people always ask me, "What services do I need running and which of them can be disabled?"

    Well here's a detailed list as to which services are, for the most part, useless and which are essential:

    Please note this is how my services are configured, if you're not sure about it rather leave it alone or set the service to manual instead.

    Go to start, run, type in "services.msc" without the quotes and press ok to launch the services utility program or right click on my computer and select manage, and then navigate to services and applications, and then finally services.

    REMOTE REGISTRY - DISABLE

    Enables remote users to modify registry settings on your computer, provided the remote users have the required permissions.

    Disable this service for added security.

    TABLET PC INPUT SERVICE - DISABLE

    Enables users to use Tablet's pen and ink on their systems.

    If you do not use a tablet then disable this service.

    WINDOWS FIREWALL - DISABLE

    Prevents malicious users or traffic from accessing your system through the internet or network.

    If you run a 3rd party personal firewall this service can be disabled as to prevent conflict.

    REMOTE DESKTOP CONFIGURATION - DISABLE

    Main service responsible for remote desktop sessions.

    As a security add-on, it is recommended you disable this service if you do not wish to have remote sessions.

    REMOTE DESKTOP SERVICES- DISABLE

    Allows users to connect to a remote system

    As a security add-on, it is recommended you disable this service if you do not wish to have remote sessions.

    REMOTE DESKTOP SERVICES USERMODE PORT REDIRECTOR- DISABLE

    Allows the redirection of ports/drives/printers for RDP(Remote Desktop Protocol) connections.

    As a security add-on, it is recommended you disable this service if you do not wish to have remote sessions.

    PARENTAL CONTROLS - DISABLE

    Parental Controls helps you limit how much computer time children have, as well as which programs and games they can use (and perhaps more importantly, when). With the Parental Controls in Windows Media Centre you can also block access to objectionable TV shows and movies.

    If you do not want this service you can disable it.

    PRINT SPOOLER - DISABLE

    Loads files to memory for later printing.

    If you do not use a printer it's best to disable this service as a security add-on as this service may be exploited and allow arbitrary code injection to your system.

    SMART CARD - DISABLE

    Manages access to smart cards read by the computer.

    If you're computer does not have a smart card reader you can disable this service, else set it to manual.

    ROUTING AND REMOTE ACCESS - DISABLE

    The remote access feature provides VPN services so that users can access corporate networks securely over the Internet as if they were directly connected. Remote access also enables remote or mobile workers who use dial-up communication links to access corporate networks.

    RRAS is a software router and an open platform for routing and networking. Its routing services can be used by organizations in local area network (LAN) and wide area network (WAN) environments or over the Internet by using secure VPN connections. Routing is used for multiprotocol LAN-to-LAN, LAN-to-WAN, VPN, and network address translation (NAT) routing services.

    Straight definition from Microsoft, this service can be disabled as a security add-on.

    MICROSOFT iSCSI INITIATOR SERVICE - DISABLE

    Microsoft® Internet iSCSI Initiator enables you to connect a host computer that is running Windows® 7 or Windows Server® 2008 R2 to an external iSCSI-based storage array through an Ethernet network adapter.

    Straight definition from Microsoft, this service can be disabled.

    NETLOGON - DISABLE

    Used to authenticate users with Active Directory

    If you're not on an network with an active directory, one which your credentials are pulled from a domain, this service can be disabled.

    SERVER - DISABLE

    Used for file, print and name-pipe sharing over the network.

    If you are not on a network, this service can be disabled.

    SECONDARY LOGON - DISABLE

    Windows Secondary Logon allows administrators to log on with a non-administrator account to perform administrative tasks without logging off.If you only have one user, you can set this to manual, disabling it may cause some games to not install correctly as stated on us.battle.net

    FAX - DISABLE

    Enables faxing on the local machine or over a network

    If you do not use fax, you can disable this.

    COMPUTER BROWSER - DISABLE

    Maintains a list of computers on the network

    If you're not on a network, you can disable this

    WINDOWS TIME - DISABLE

    Maintains all date and time synchronization to clients on the network

    If you're not on a network you can disable this.

    WORKSTATION - DISABLE

    Creates and maintains client network connections to remote servers over the network using the SMB(Server Message Block) protocol.

    If you're not on a network, you can disable this as a security add-on as the SMB protocol can be exploited and used for arbitrary code injection


    WINDOWS DEFENDER - DISABLE

    Protection against spyware and unwanted software

    If you're using 3rd party antivirus and antimalware you can disable this

    Ok so that's the basic and main services you can disable now let's see which features can be disabled.

    To Enable/Disable Windows features open up control panel by clicking start/control panel and navigating to programs/program and features under category or simply program and features under large/small icons, then, in the left hand pane, click turn Windows features on or off.

    GAMES - UNCHECK

    If you're not a fan a Window's default games you can uncheck the the games checkbox to disable games.


    INDEXING SERVICE - UNCHECK

    If you don't mind waiting a bit longer for search results you can uncheck the Indexing Service checkbox.

    INTERNET EXPLORER 11 - UNCHECK

    If you never use IE like myself, and prefer other 3rd party browsers, you can go ahead and uncheck this checkbox

    INTERNET INFORMATION SERVICES/FTP SERVER - UNCHECK

    If you do not use FTP(File Transfer Protocol) you can uncheck this checkbox

    MEDIA FEATURES - UNCHECK

    You can uncheck this checkbox if you do not use Windows Media Player, Windows Media Centre or Windows DVD Maker.

    TABLET PC COMPONENTS - UNCHECK

    If you do not use a tablet on your computer then you can go ahead and uncheck this checkbox.

    TELNET CLIENT & TELNET SERVER - UNCHECK

    If you do not use telnet or don't know what it is then you should uncheck both these checkboxes for added security.


    WINDOWS GADGET PLATFORM - UNCHECK

    If you don't use Windows gadgets then you can go ahead and uncheck this checkbox.

    WINDOWS VIRTUAL PC - UNCHECK

    If do not run any VM's(Virtual Machines) then you can go ahead and uncheck this checkbox.
