striker117wd say:

I had a similar issue and this is what I did after trying all the other options.

manually restore your registry by booting from CD, get to the recovery tools and launch command prompt. In the command prompt, CD to X:\Windows\System32\config (where X = your primary windows partition that's having issues) then enter each of the following lines (press enter after each one)

  • ren X:\Windows\System32\config\DEFAULT DEFAULT.old
  • ren X:\Windows\System32\config\SAM SAM.old
  • ren X:\Windows\System32\config\SECURITY SECURITY.old
  • ren X:\Windows\System32\config\SOFTWARE SOFTWARE.old
  • ren X:\Windows\System32\config\SYSTEM SYSTEM.old

This renames your core registry hive files to *.old to which windows will not use them.

Now you need to copy in the backups (these will have been made by windows before your problems started)

  • copy X:\Windows\System32\config\RegBack\DEFAULT X:\Windows\System32\config\
  • copy X:\Windows\System32\config\RegBack\SAM X:\Windows\System32\config\
  • copy X:\Windows\System32\config\RegBack\SECURITY X:\Windows\System32\config\
  • copy X:\Windows\System32\config\RegBack\SYSTEM X:\Windows\System32\config\
  • copy X:\Windows\System32\config\RegBack\SOFTWARE X:\Windows\System32\config\

This will move the windows created registry backups into the System32\config folder to replace the old ones. Exit the command prompt and restart. This should get you into ur system.

xylee del say:

Hello,

Ctfmon is the Microsoft process that controls Alternative User Input and the Office Language bar. It’s how you can control the computer via speech or a pen tablet, or using the onscreen keyboard inputs for asian languages. If you are using any of the mentioned tasks, you should leave it enabled. Since you've already tried removing it at the task manager, we suggest that you follow the steps below to disable ctfmon.exe:

  1. Open Settings by clicking the Windows icon.
  2. At the Settings window, type in Administrative tools and select it.
  3. At the Administrative tools window, click Services.
  4. At the services window, look for Touch Keyboard and Handwriting panel service, set this to disabled and stop the service. This will disable the ability to use the onscreen keyboard.
