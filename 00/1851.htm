Typically, you shouldn't have your own address in your Safe Sender list.  The Safe Sender list is for addresses whose messages you want to make sure get delivered to Inbox and not Junk.  The Safe Recipients list is for mailing list addresses.  Sometimes maillist servers send their messages to the list address instead of the actual recipient address.  If that's the case for mailing lists to which you've subscribed, put the address in the Safe Recipients list to make sure it's not filtered to Junk.  The Blocked Sender list is for addresses whose messages you always want deleted.  Addresses should go in only one of the three lists.  If you have an address in your People, you shouldn't put it in any of the Safe or Blocked lists.

PROBLEM: The data transfer add-in for Excel will not work in anything but the first spreadsheet opened in an instance of Excel 2013.

The problem can be circumvented but there are no plans to rewrite the data transfer add-in to conform with the way Excel 2013 is handling windows. Following is a circumvention which can be used instead of simply not opening more than one spreadsheet at a time.

The option is available to launch Excel differently by adding a /X parameter to the start of the program. This will cause additional launches of Excel to start their own instance (their own process, not simply more threads in the same process). You can do this by finding the program Excel.EXE and creating a shortcut for it and adding the /X after the existing shortcut target.

For example, if Excel.exe is found in the path "C:\Program Files\Microsoft Office\Office15\EXCEL.EXE", you can create a shortcut to Excel.exe on your desktop and then modify the properties of the shortcut so that the target is now: "C:\Program Files\Microsoft Office\Office15\EXCEL.EXE" /X
Notice that when launched this way, you will see multiple Excel applications on your toolbar and in the task manager. In order to be able to run a transfer in anything but the first spreadsheet of the first instance of Excel, you will need to launch a new instance of Excel (in other words, simply clicking on File New is not going to get the job done, you have to start a new instance of Excel using the shortcut you created on your desktop).

format figure numbers - Microsoft Community

kelly red say:

format figure numbers

I am adding numbered figures to a large document and am wondering if there's a way to format them in Word 2008 so that they automatically adjust their number when I add a new figure to the middle of the document.

Thanks.
daniel g. grau say:
In Word you can define an arbitrary number of numbering sequences. Each of them can have its own name, so that you can use several sequences, each with it's own format: one with A, B, C... another with i, ii, iii... and of course 1, 2, 3...

These are called SEQ fields (for "sequence"). They will work fine in the main text flow of the document, but if used in floating boxes they require manual updating. So, as long as your numbers are not in floating boxes, it's very easy to use them:

  • In the main menu bar, select Insert > Field
  • Look for the one called SEQ and click
  • In the bottom field, make it look like SEQ AnyNameYouLike
  • Click OK

In itself, that will by default generate a sequence of numbers. If you need another numbering format, then before going back to your document, click on the Options… button, make sure the first tab (General Switches) is selected, choose a numbering sequence and then Add to field.

To facilitate your work, you should put fields in evidence:

  • In the main menu bar, select Word > Preferences > View and enable Field shading: Always. This will add a non-printing gray shade to fields, so you can identify them at a glance.

After all is set, all you need to do is copy a field from somewhere else and paste it as many times as you want, to obtain the sequential numbers. In case your fields don't keep their sequence (like when you add a new SEQ field at the top of the document), do this:

  • Command A (to Select All content in the document)
  • Press F9 (update selected fields)

Also, if you'd like to forego the need to copy and paste fields, you can define an AutoText sequence:

  • Select (highlight) a SEQ field in the document
  • Go to Word > Preferences > AutoCorrect
  • Tick the checkbox at the top (Automatically correct...)
  • Make sure the first tab (AutoCorrect) is selected
  • Check Replace text as you type
  • In the Replace field, type a unique character sequence (like fig\ or /f/ )
  • In the With radio button, select Formatted text
  • Click on Add
  • Click OK

After that, whenever you type your character sequence (fig\, /f/, etc.), Word will automatically replace it by the SEQ field as defined in AutoCorrect.
