jay freedman say:

There are several things wrong in your screen shot:

  • The AllowListEnabled entry that your first screen shot showed in the Common key needs to be in the GraphicsFilterImport key instead.
  • The key under the Security key must be AllowLists instead of AllowList (the final s is missing).
  • You also need entries for the other graphics filters that should be enabled. The support article explains that when you set the AllowListEnabled value to 1, the default list of enabled graphic filters is removed. To re-enable the default graphic filters and add the any graphic filter (EPS in this case), you must specify all the filters in the "Allow List."

Your registry should look like this (allowing for the fact that my installation is English rather than German):

word,
for_home,
