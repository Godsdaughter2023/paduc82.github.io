hansv mvp say:

If a style has been set to update automatically, direct formatting that you apply to some text with that style will be applied to the style, and hence other text with the same style will receive the same formatting.

Right-click the style in the Styles group on the Home tab of the ribbon, or in the Styles task pane, and select Modify... from the context menu.

Clear the check box "Automatically update", then click OK.

 

Remark: the Normal style doesn't have the "Automatically update" check box. If your Normal style exhibits this problem:

Press Alt+F11 to activate the Visual Basic Editor.

Press Ctrl+G to activate the Immediate window.

Enter or copy/paste the following line into the Immediate window:

 

ActiveDocument.Styles(wdStyleNormal).AutomaticallyUpdate = False

 

With the insertion point anywhere in the line, press Enter.

Press Alt+F4 to close the Visual Basic Editor.

Maybe this would get you fairly close:

Sub fixPics()
Dim L As Long
Dim Pos As Long
Dim osld As Slide
On Error GoTo err
Set osld = ActiveWindow.Selection.SlideRange(1)
For L = osld.Shapes.Count To 1 Step -1
If osld.Shapes(L).Type = msoPicture Then
osld.Shapes(L).Cut
With ActivePresentation.Slides.Add(osld.SlideIndex + 1, ppLayoutBlank)
.Shapes.Paste
End With
End If
Next
Exit Sub
err:
MsgBox "Error, did you have a slide selected?"
End Sub
