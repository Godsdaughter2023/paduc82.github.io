Offline DOES NOT mean when the program is closed.  At least not to Outlook.  In Outlook you can configure the program to "Work Offline".  You will find it on the Send/Receive tab of the Ribbon in Outlook 2010.  Click "Work Offline" to put Outlook in Offline mode.  This was traditionally for installations that didn't have always-on Internet (such as dialup) but it can also be used in other scenarios as well.

The alert and envelope are still options (File | Options | Mail...scroll down to Message Arrival) but Outlook MUST be running for it to check e-mail.

cbarnhorst say: No.  SSDs have much longer lives than HDDs.  Just because we know the number of write cycles for an SSD and don't know the same for HDDs doesn't mean that SSDs are less durable than HDDs.  In fact, SSDs are far more durable than HDDs.  Unlike HDDs, the larger an SSD the longer it will last.  If you have a 1TB SSD and you write 25GB of data a week it will take 40 weeks to write once to every memory cell on the SSD (1 write cycle).  A Samsung SSD sustains 50,000 or more write cycles.  You should live so long to write enough data to use one up.  Anyway, your SSD will outlast the devices that use SSDs by a lot.  Will we even be using devices that take SSDs ten years from now?  Twenty?

Create “Pop-ups”in Word

Thanks for the tip Phil.  And thank you everyone for your input.  It looks like the only three options so far are to use existing features within Word.  So far, I have not found a macro.  The three options for creating a roll-over/pop-up box effect so far are:

1.  Comments

2.  Endnotes

3.  Hyperlinks in combination with it's Screen Tips feature

For anyone who is interested in this subject, below is my analysis of the three options along with the advantages and disadvantages of using each option for creating a roll-over/pop-up box effect:

Creating a comment. 

You can create the pop-up box effect by creating a comment.

Advantages:

You can hide any of the symbols and property text associated with comments.  For example, the comment reference mark (e.g., [jb1]) will not display unless the user has the hidden text box checkmarked within MS Word.  You can also eliminate the User Information that displays within the comment pop-up box by deleting the User Information within the Options box prior to creating the comment. 

Disadvantages: 

I don't think you can eliminate the red highlighting from the text within the body of the document to which the comment is referring.  Also, the reader can open up the comment pane, which exposes the source of the pop-up boxes. 

Creating an endnote. 

You can create the pop-up box effect by creating an endnote.

Advantages:

There is no highlighting, and there is no user information associated with the pop-up box.  Also, the user can not reach the endnote pane (similar to the comment pane) without switching from the Print view to the Normal view.

Disadvantages:

There is an endnote reference mark, which appears as a superscript, and you have a corresponding list of endnotes at the end of your document (although you can insert a series of page breaks so that the endnotes do not appear right after the document body).

Creating a hyperlink that includes a screen tip.

You can create the pop-up box effect by creating a hyperlink that includes a screen tip connected to a bookmark.

Advantages:

There is no highlighting, there is no user information, there is no reference mark, and there is no corresponding pane of text like there is with a comment or an endnote.

Disadvantages: 

The number of characters that can be entered into the Screen Tip dialog box is limited and, therefore, the number of words within the resulting pop-up box is limited.
