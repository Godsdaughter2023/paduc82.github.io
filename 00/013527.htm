You're right that it will still go to Deleted folder and you would need to empty it to get the messages permanently deleted. There you'll see the Option to "Delete these messages—don't even put them in my Deleted folder" which you can choose.

This rule will only work with specific messages that you wanted to be included that will match the rule you have created. This is another option you can take aside from the previous one given in this thread.
