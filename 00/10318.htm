a. user say:

Anything that is a recurring appt with occurrences in the future will be displayed on the calendar back as far as the initial start date (I believe).

 

For the sake of keeping memory usage down, the calendar app looks at the 'start date' of the appt and only keeps non-recurring ones back 2 weeks.  If a recurring appt has future occurrences, then it will keep the entire appt (including the 'old' ones) until 2 weeks after the 'last' one (I believe).

hansv mvp say:

Try

 

Sub ListControls()
    Dim myForm As Object
    Dim frm As Object
    Dim frmControl As Object
 

    For Each myForm In AccApp.CurrentProject.AllForms
        AccApp.DoCmd.OpenForm FormName:=myForm.Name, View:=1
        Set frm = AccApp.Forms(myForm.Name)
        For Each frmControl In frm.Controls
            Debug.Print myForm.Name, frmControl.Name
        Next frmControl
        AppAcc.DoCmd.Close 2, myForm.Name, 2
    Next myForm
End Sub

 

This opens each form in design view, lists the controls, then closes the form without saving it.
