internet expert say:

If the hotspot has a web-based authorization process you need to pass through (or a login page) upon connection to the hotspot, the Xbox cannot support that.


However, there is a workaround. You can try connecting to the hotspot with your phone, log into the webpage with your credentials, and then change the Wireless MAC address of your Xbox to the Wifi MAC of your android phone. Disconnect your Android phone from Wifi as soon as you get them connected so you don't get conflicts. 


In the Xbox network settings, go to "Advanced Settings" and select "Alternate MAC Address" to change your Xbox's MAC.


Hope this works for you!

a. user say:

Hello,

Thank you for posting your concern here in Microsoft Community and welcome to the Windows 10 family.

Please be informed that the Windows.old folder contains the following folders:

  • Windows
  • "Documents and Settings"
  • "Program Files"

The Windows folder mainly stores programs and files that are required to run the earlier version of Windows.

The "Documents and Settings" folder mainly stores the "My Documents" folder and the Favorites folder. Additionally, the "Documents and Settings" folder stores the desktop files that were used by the earlier version of Windows.

The "Program Files" folder mainly stores the programs that were installed on the earlier version of Windows.

The Windows.old folder will be removed within the first 30 days of the installation of the Operating System by a scheduled maintenance task.
