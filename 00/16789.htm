graham mayor say:
This error relates to Trados Studio which calls upon an object library MSCOMCT2.OCX in order to display its form correctly. This object library is not present in 64 bit versions of Windows, hence the error message when the add-in calls for it.

Provided you are using the 32 bit version of Office (it is not going to work with the 64 bit version), you need to ensure that MSCOMCT2.OCX is in the folder c:\Windows\SysWOW64 and that it is registered.

You may find the file in the folder C:\Windows\System32 or, if not, in one of the other sub folders of C:\Windows, in which case, copy it to C:\Windows\SysWOW64 and then register it. Instructions for registering it (with illustrations - albeit from Windows 7) can be found at http://www.gmayor.com/popup_calendar.htm   under the sub heading - Date and Time Picker control near the end of the page.

If you are unable to locate the file on your PC, you can download it from http://www.gmayor.com/Zips/MSCOMCT2.zip
