considerate_guy say:

Do you know what software you used to create the slideshow? Is the result a movie file (.wmv, .mp4, etc.)? And do you know the resolution of that video file?

The answer to your question is "yes and no". You can extract stills from a movie using Movie Maker. However, they will not be as high a resolution as the original scanned images. Also, unless the pictures fully fill the aspect ratio of your movie, the saved snapshots will include black bars which would then require further editing in another program to remove. The resolution for printing might be enough for a 4' x 6' but probably not much higher than that.

If you really want to do this, I don't recommend using Windows Live Movie Maker or any of the variants (Movie Maker 2009 to 2012). The reason is the snapshot is only taken from the preview window which will be even smaller than the final video.

Instead, I'd suggest using Windows Movie Maker 6.0. Add the video to the collection and don't even add it to a project. Just play the video from the collection and pause it on each picture that you want to save. Then use Tools > Take Picture from Preview... 

Repeat the process for each photo in the video that you want to save.

The desktop.ini files are backups that windows makes when changes are made to the desktop. If you want to stop these messages, go to Folder Options through Search click settings and you will see Folder Options, click it and the View tab and scroll until you see " Hide protected operation system files (recommended) and put a check in the box.

Removing the file will cause issues with the Operating System and so it is not recommended to remove the files. The desktop.ini file by default has the System, Hidden and Read-only attributes assigned.
