bernd p. say: You wrote:

And as you can see from the red circle on the left, the ESRC postdoc notebook itself still has the icon for a closed notebook, not an open notebook.

This is wrong.

A "closed" notebook on the left bar means: This notebook is open (!) in Onenote but you are not  positioned at it (not actually displayed).

An "open" notebook on the left bar means: This notebook is open (!) in Onenote and you are positioned at it (is actually displayed).

A really closed notebook just doesn't appear at the left bar.

So it seems your config is inconsistent.

You should "reset" your Onenote by closing explicitely ALL notebooks; either by right clicking each notebook (at the left bar) and select Close or by RUN Onenote.exe /safeboot to clear the cache.

Then re-open your notebooks by using File, Open, Browse (navigating to the notebook folder);
not by using Windows explorer.

Bernd

brian tillman [outlook mvp 2007-2019] say:

Outlook 2013 can connect to Outlook.com using POP, IMAP, or Exchange ActiveSync.  The default account type is EAS.  EAS, however, won't allow you to copy messages from a local PST to the Outlook.com folders. It's a drawback in the protocol.  IMAP, however, will allow it.  So you can connect to your Outlook.com mailbox using an IMAP account (configured manually) and then drag/drop or copy/paste the messages from the local data file's folders to the Outlook.com folders.  You can, simultaneously, connect to the mailbox using an EAS account (configured automatically), which will allow you to copy the contacts from the local data file to Outlook.com's contacts folder.  You can do the same with the data file's calendar folder.  Once your data has been moved, you can remove the IMAP account so you don't see the same folders twice.

If iCalendar has a way to export the contacts as a comma-separated values file or as an ICS file and you can then get that file to the PC, you can import the data into the Outlook.com calendar, which will then show up in Office Outlook.  After that, you can set up an EAS account in the iPhone with Settings>Mail, Contacts, and Calendar>Add Account, choosing Hotmail or Outlook.com as the account type.  This will allow the phone to sync to mail, contact, and calendar on Outlook.com so the PC and the phone will see the same data.

After all this, be sure to enable two-step verification, which will protect your account from hackers. You can add the phone to the account, if you haven't already, to display the code you'll need to log in.  This code can ge either SMS from Outlook.com or you can install the Google Authenticator, connect it to your Outlook.com account, and have the Authenticator generate codes.  Designating your PC and your phone as trusted devices will mean you don't have to enter a code every time for those two devices.
