jim gordon mvp say:

In order to use SmartArt your file must be saved in a current Microsoft Word format. 

If your document is in an old Word format (.doc or .dot for example) SmartArt is not available to you, as old file formats and other file formats such as (.rtf) do not have the ability to contain SmartArt objects.

To fix the problem for a specific document, use File > Save As, and choose a current Microsoft Word file format as appropriate. Technically, these are Office Open XML (OOXML) file formats and adhere to open file format standards.

.docx = word document

.docm = macro enabled word document

.dotx = word document template

.dotm = macro enabled word template

Once your document is in a current file format your SmartArt options will become available to you on the SmartArt tab of the Ribbon.

rich michaels - mvp (greatcirclelearning.com) say:

There are probably a number of ways to accomplish this and here is one idea.

Setup two unique Paragraph styles... MyEnglishStyle and MyFrenchStyle. For each assign a Hidden attribute to the Font property and a Page Break Before attribute to the Paragraph property. Now create the document and use both Styles to "pre-establish" the odd and even pages in the document. The single line "hidden" language style paragraph starts each new page.

You complete the the English pages and instruct the translator to complete the corresponding French pages.

Hopefully you are aware that the English pages, based on your description, are going to be the right hand pages and the the French pages will be one the left. You will therfore have to accommodate for this with any upfront pages that start the document's layout and pagination correctly. Also, unless you plan on having each page start as a new section, the header and footer data will most likely have to be in only one of the languages.

Hope this helps

doug robbins - mvp office apps & services (word) say:

You don't really have to worry about the page numbers in the single language documents as when they are combined, the page numbers will fall into place.

The following macro will create a document by interleaving the pages from one document with those from another:

    Dim sourcea As Document, sourceb As Document, target As Document, Pages As Integer, Counter As Integer, targetrange As Range 'targetrange added

    Dim evenpage As Range

    Set sourcea = Documents.Open(FileName:="...")'English Document

    sourcea.Repaginate

    Pages = ActiveDocument.BuiltInDocumentProperties(wdPropertyPages)

    MsgBox Pages

    Set sourceb = Documents.Open(FileName:="...")'French Document

    Set target = Documents.Add

    target.PageSetup.LeftMargin = sourcea.PageSetup.LeftMargin

    target.PageSetup.RightMargin = sourcea.PageSetup.RightMargin

    target.PageSetup.TopMargin = sourcea.PageSetup.TopMargin

    target.PageSetup.BottomMargin = sourcea.PageSetup.BottomMargin

    target.AcceptAllRevisions

    Counter = 0

    While Counter < Pages

        sourcea.Activate

        ActiveDocument.Bookmarks("\page").Range.Copy

        Set targetrange = target.Range

        targetrange.Start = targetrange.End

        targetrange.Paste

        ActiveDocument.Bookmarks("\page").Range.Cut

        sourceb.Activate 'Assumed to be the document containing the even pages

        Selection.EndKey Unit:=wdStory 'Line of code added to start from the end of the document

        ActiveDocument.Bookmarks("\page").Range.Copy

        Set targetrange = target.Range

        targetrange.Start = targetrange.End

        targetrange.Paste

        targetrange.Start = targetrange.End

        targetrange.InsertBreak Type:=wdPageBreak

        Set evenpage = ActiveDocument.Bookmarks("\page").Range

        evenpage.Start = evenpage.Start - 1

        evenpage.Delete

        Counter = Counter + 1

    Wend

    sourcea.Close wdDoNotSaveChanges

    sourceb.Close wdDoNotSaveChanges

djtravtastic say:

It's clunky, but not complicated.  Just figured this out myself.

Drag all the contacts you want out of your Mac address book onto your desktop.  Will create a single .vcf file.

Then, simply right-click on the file, Open With --> Microsoft Outlook.  A new contact window will appear in Outlook.  it will be the first address in your .vcf file.  Just click 'Save'.  Outlook will save it, then begin populating the rest of your contacts.  That's it!
