bernie deitrick say:

No - Excel is by default base 10.

You could convert the number in A2 to your base 2 shortened values using formulas

=A2/(2^(INT(LOG(A2,2)/10)*10))

=CHOOSE(INT(LOG(A2,2)/10)+1,"K","G","T")

or all as one:

=TEXT(A2/(2^(INT(LOG(A2,2)/10)*10)),"#.0")& CHOOSE(INT(LOG(A2,2)/10)+1,"K","G","T")

Make sure that for any other calculations that you use the original values.
