kosh vorlon say:

Hi Auriel,

 

Just to add to what Steve said (and I agree with him), there may be another task you need to perform.

 

If you run backups as you should, these infected files will be in your current backups.  I recommend you delete all current backups and create a new backup of you system now that these infections have been removed.  That way, if you ever do need to use the backup, it will not re-infect your system.

 

Also, FWIW, while CCleaner does a great job for most termporary files, it really doesn't cover ALL temporary files.  The best product (also free) for that is Old Timer TFC http://www.geekstogo.com/forum/files/file/187-tfc-temp-file-cleaner-by-oldtimer/ but because it requires a reboot to finalize, I tend to use it only every month or two just to catch whatever CCleaner might have missed (and it does find a lot even if I run it only an hour after running CCleaner).

 

But the real solution is to do what you can to avoid being infected in the first place.  For that, I recommend the following:

 

I use Secunia PSI http://secunia.com/vulnerability_scanning/personal/ to make sure all my programs are updated and current (since updates often plug security holes in the software and thus can be important as many people never check and/or update their software unless prompted - and this way I don't need to check everything since notifications and sometimes updates are automatic).

 

I always follow the following advice from Stephen Boots - MVP:

 

-Make sure that the Windows Firewall is enabled. No other firewall truly offers better protection or is needed. This one is perfectly fine.

 

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

 

-Make sure Internet Explorer is at version 8 or 9 and updated with all patches.

 

-In Internet Explorer 8 or 9, use the SmartScreen Filter.

 

-Make sure that IE Internet Security settings are at least set to medium-high (default).

 

-Enable the pop-up blocker in IE.

 

-On Vista and Windows 7 make sure that User Account Control (UAC) ON and not running with elevated privileges.

 

-Make sure that Windows Automatic Updates are set to at least notify, but the preferred setting is to download and install automatically. If you update manually, be sure to update as soon as possible after being notified of available updates.

 

-Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.

 

-Never click through links from unknown sources and use caution even if they are from a "trusted" source.

 

-Never open unsolicited email attachments.

 

-Practice safe web browsing.

 

-Always use a standard user account unless you have a specific need for an administrator account.  Then switch, do the administrative task, and return to the standard account.  This greatly limits the access of any type of infection of intrusion into your system since administrative access isn’t in place.

 

I hope this helps.

 

Good luck!

try this approach..

select the desired range and run the code..

Sub Count_MergedSections()
Dim rng As Range, r As Range
Set rng = Selection
Dim n As Long, nCel As Long
Dim sAdr As String
n = 0: nCel = 0: sAdr = ""
For Each r In rng
If r.MergeCells Then
If r.Address = r.MergeArea(1).Address Then
n = n + 1
nCel = r.MergeArea.Rows.Count * r.MergeArea.Columns.Count
sAdr = sAdr & vbCr & r.MergeArea.Address & " (" & nCel & " cells)"
End If
End If
Next r
If nCel > 0 Then
MsgBox "merged sections = " & n & sAdr
Else
MsgBox "merged cells:" & vbCr & "nothing found"
End If
End Sub

david marcovitz say: There are ways to do this with slide libraries. Someone who is good at that will hopefully chime in here. Another way to do this is with custom shows. What you might do is have an opening slide that has two buttons: 5-day and 3-day. The buttons can be hyperlinked to the correct custom show. A custom show is simply a specific set of slides within the presentation in whatever order you choose. Because all the slides are the slides from the show, any change to a slide will change the custom show. It would just be trickier if you decided to add slides.

In fact, you could simplify this with one custom show for the 3-day presentation. The 5-day button could just go to the next slide because you want to show all the slides.

Finally, you could do it without buttons by going to Slide Show > Set Up Show before the presentation and choosing to show all the slides or just the Custom Show.
