charlie dude say:

You can still only install any one given installation and license to only "ONE" computer. You "CAN NOT" install the same license on more than one computer.

If you purchased the cheaper version of Windows 7 (OEM), once it is installed on the very first computer, you can not move it to a different computer. It is forever tied to the first motherboard it was installed to.

Even retail versions of Windows can only be installed on only one computer at a time. Retail versions give you the ability to uninstall it from one computer and install it on a different computer.

You also still do not own Windows 7, you only have usage rights by agreeing to and accepting the terms of the EULA.

If you look at the very small fine print on the packaging it tells you that it can only be installed on one PC.
