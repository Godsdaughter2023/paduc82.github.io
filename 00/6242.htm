It seems that you have changed Autorecovery information save to 1 min in File > Options > Save.

This is not for saving the Excel workbook. This saves a version of your Excel sheet so that you can recover in case of crash. The same file you can find in whatever path is specified in Autorecover file location. If the file is normally closed, this file will disappear.

This is not a substitute for normal save which you do. You can't avoid saving manually.

Word provides an option to save a backup copy of the file which Excel doesn't provide.

However, Excel in Office 365 has incorporated Autosave feature where it saves to Onedrive periodically.
