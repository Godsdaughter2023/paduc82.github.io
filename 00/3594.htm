graham mayor say:

You would probably need something like the following, which will allow you to enter numbers or periods only in the text box and will format the text box when you tab out of it. The message box is superfluous as it will beep as a warning if you enter anything else in the box. The italicised section is probably unnecessary also as you may enter numbers and periods in the text box

Private Sub Validate_Numeric(ByRef TB As MSForms.TextBox, ByRef KeyAscii As MSForms.ReturnInteger)
    With TB
        Select Case KeyAscii
            Case Asc("0") To Asc("9")
            Case Asc(".")
            Case Else
                KeyAscii = 0: Beep
        End Select
    End With
End Sub

Private Sub txtPrice_KeyPress(ByVal KeyAscii As MSForms.ReturnInteger)
Validate_Numeric Me.txtPrice, KeyAscii
End Sub

Private Sub txtPrice_Exit(ByVal Cancel As MSForms.ReturnBoolean)
txtPrice.Text = Format(txtPrice, "0.00")
End Sub

donleverton say:

Yikes ...

If it were me, I would be temped to do this using VBA recordsets.

Option Compare Database

Private Sub Command0_Click()
Dim MyDB As DAO.Database
Dim rst1 As DAO.Recordset
Dim rst2 As DAO.Recordset
Dim fld As Field

Set MyDB = CurrentDb
Set rst1 = MyDB.OpenRecordset("Table1", dbOpenDynaset)
Set rst2 = MyDB.OpenRecordset("Table2", dbOpenDynaset)

Dim NewInfo 'Field Contents
Dim Myfield 'Field Name
With rst2 'Open the recordset of "Table2"
    If .RecordCount < 1 Then Exit Sub 'If there are no records in Table2 ... get outta here.
    .MoveLast 'Populate the Recordset
    .MoveFirst
        Do Until .EOF
       
            For Each fld In rst2.Fields
                Myfield = fld.Name 'Get the Field Name
                If Myfield = "OneID" Then 'We can't do anything with an autonumber PK
                    'do nothing
                Else
                    NewInfo = rst2.Fields(Myfield) 'Read the field contents of the current field
                        With rst1
                               If Myfield = "Field1" Then 'This is the field that we want to increment
                                    .AddNew 'This needs to be a NEW record
                                    !Field1 = DMax("Field1", "Table1") + 1 'Add 1 to the Maximum value
                                    .Update 'Commit (save) the new record
                                Else
                                    .MoveLast 'Make this record the current record
                                    .Edit
                                    rst1.Fields(Myfield) = NewInfo 'Insert the copied data into the corresponding field
                                    .Update ' Commit the change to this record / field
                                End If
                            
                        End With
                End If
            Next fld
        .Delete 'This deletes the record from Table2 after adding it to Table1
        .MoveNext
       
        Loop
   
    .Close

End With

MyDB.TableDefs.Refresh
Set rst2 = Nothing
Set rst1 = Nothing
Set MyDB = Nothing

End Sub

Edit:

You WILL, of course, have to edit a couple of table / field names.

"OneID" is what I used for the AutoNumber-PK field of "Table1"

"Field1" is the field that you want to increment the Max value by 1.

You had mention "importing" Table2, and appending the data from it into Table1.

My code deletes the records from Table2, so that it pretty much impossible to append them twice. Again, I'm assuming that the data in Table2 is both temporary AND expendable.

Can I change to smart quotes in Excel? - Microsoft Community

crdgeval say:

Can I change to smart quotes in Excel?

Hi,

Is there an option to change the quotation marks from straight quotations marks to "smart" quotation marks in Microsoft Office Mac 2011 Excel?

Thanks!
j.e. mcgimpsey say: There's no built-in way. However, I have the macro below stored in an add-in and attached to a custom toolbar. You could put it in an add-in or your Personal Macro Workbook. Select the cells you want to "smarten" and run the macro. Run it again to restore the dumb versions:

    Public Sub ToggleSmartQuotes()
        Dim rCell As Range
        Dim sText As String
        Dim sPattern1 As String
        Dim sPattern2 As String
        
        sPattern1 = "*[" & Chr(34) & Chr(39) & "]*"
        sPattern2 = "*[" & Chr(210) & Chr(211) & Chr(212) & Chr(213) & "]*"
        For Each rCell In Selection
            With rCell
                sText = .Text
                If sText Like sPattern1 Then
                    .Value = SmartQuotes(sText, bSmart:=True)
                ElseIf sText Like sPattern2 Then
                    .Value = SmartQuotes(sText, bSmart:=False)
                End If
            End With
        Next rCell
    End Sub

It requires this function (which you should also put in your add-in or Personal Macro Workbook):

    Private Function SmartQuotes(ByVal sInput As String, _
            Optional ByVal bSmart As Boolean = True) As String
        Const csDumbDQ As String = """"
        Const csDumbSQ As String = "'"
        Const csSpace As String = " "
        Static sOpenSQ As String
        Static sCloseSQ As String
        Static sOpenDQ As String
        Static sCloseDQ As String
        Dim sTemp As String
        
        If sOpenSQ = vbNullString Then
            sOpenSQ = Chr(212)
            sCloseSQ = Chr(213)
            sOpenDQ = Chr(210)
            sCloseDQ = Chr(211)
        End If
        
        sTemp = sInput
        If bSmart Then
            sTemp = Replace(Replace(Replace(Replace(sTemp, _
                    csSpace & csDumbSQ, csSpace & sOpenSQ), csSpace & csDumbDQ, _
                    csSpace & sOpenDQ), csDumbSQ, sCloseSQ), csDumbDQ, sCloseDQ)
        Else
            sTemp = Replace(Replace(Replace(Replace(sTemp, sOpenSQ, csDumbSQ), _
                        sCloseSQ, csDumbSQ), sOpenDQ, csDumbDQ), sCloseDQ, csDumbDQ)
        End If
        SmartQuotes = sTemp
    End Function


pmkm ed say:

To replace existing straight quotes with smart quotes in Excel:

Just in case anyone reads the replies and finds that Replace doesn't work: in Excel for Mac 2011, if you just type straight quotes ' and ' into the Find and Replace fields and hit Replace All, nothing happens. (In that respect, it's different from Word.)

If however you type ' into the Find field and then you copy and paste a smart quote from Word ’, like I've just done into this reply, into the Replace field and then do Replace All, Excel does change them.

  1. Legacy Drop-Down FormField
  2. ActiveX ListBox
  3. Drop-Down List Content Control
