luc sanders say:

RsMile,

If changing the theme does not work this would mean the formatting has been applied directly to the text in the slide (which takes precedence over the master). Or that the text has been typed in textboxes created by the user to which an individual colour (non theme colour) has been applied.

Your best bet would be to go to the slide master and see if you can change the font colour there. View - Master - select the top thumbnail, and select the placeholders, apply the black colour. Then return to your slides if the colour has not changed and it is in the default placeholders, try resetting the slide layout, go to slide sorter view, select all the slides and click the reset button on the Home tab in the slides group.

The background can also be changed in the master. In the slide master ribbon, background group, there is a button to change the background style or format the background yourself.
