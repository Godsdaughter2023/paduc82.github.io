bob jones aka: cybertaz say:

File management on a Mac is handled by the OS. Other than naming new files & creating new folders through the Save dialogs, all file management must be done in a Finder window... That includes renaming, moving, copying & deleting files. None of those actions can be executed from within an application program.

Close the file, then navigate in Finder to the location of the file then do one of the following:

  • Select the file, then press Command+delete, or
  • Right-click the file & choose Move to Trash, or
  • Just drag the file to the Trash.
