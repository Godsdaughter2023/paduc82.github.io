macropod say:

You could use an IF test for each field, coded along the lines of:

{IF{MERGEFIELD MyField} = "" "______" {MERGEFIELD MyField}}
or:
{IF«MyField» = "" "______" «MyField»}

where the underscores in "______" represent one or more underscores or tabs to fill out the output field's width and those underscores/tabs have the highlighting attribute applied.


Note: The field brace pairs (ie '{ }') for the above example are created via Ctrl-F9 - you can't simply type them or copy & paste them from this message. Likewise, you can't type or copy & paste the chevrons (ie '« »') - they're part of the actual mergefields, which you can insert from the mailmerge toolbar.

Try this array formula**.

A1 = the number to look for

For the cell address:

=ADDRESS(MAX((B2:F5=A1)*ROW(B2:F5)),MAX((B2:F5=A1)*COLUMN(B2:F5)),4)

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

Assumes that the lookup value will appear only once in the table.
