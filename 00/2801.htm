anttiah say:

The encryption driver, which lives below the file system driver, keeps the encryption key in the memory as long as the phone is not factory resetted. The same applies for the memory card password. That's why I recommend encrypting phone memory too. Phone memory encryption requires that automatic locking with lock code is turned on and set to 20 minutes or less. Automatic locking on SIM card change and remote locking via SMS should be activated too, IMO. Use a strong lock code. It can be up 255 characters long and contain numbers, uppercase and lowercase alphabets and some symbols.

 

When you have both phone memory and memory card encrypted and automatic locking turned on, you can still answer calls and mute clock or calendar alarms without entering lock code. But if you lose the phone, your data is safe because bypassing lock code requires complete factory reset, which will wipe the encryption key and make your memory card unreadable till it's reformatted. Remember that encryption doesn't prevent your device against thefts or protect it against factory reset, it will only protect your data from falling into wrong hands.

 

I have E75 with all these security features turned on and I have to say it's much more comfortable to keep sensitive or private data in the phone knowing if I ever lose the device, my private data will be safe.
