Sub SaveChkBoxSizeAndLocation()
   
    Dim ws As Worksheet
    Dim wsChkBoxes As Worksheet
    Dim shp As Shape
    Dim rngParams As Range
   
    Set ws = Worksheets("ECN")  'Edit to match the required worksheet
   
    On Error Resume Next    'Next line errors if worksheet does not exist
    Set wsChkBoxes = Worksheets("CheckBoxParams")
    On Error GoTo 0
   
    If wsChkBoxes Is Nothing Then   'Worksheet does not exist so create
        Set wsChkBoxes = Sheets.Add(After:=Sheets(Sheets.Count))
        wsChkBoxes.Name = "CheckBoxParams"
    Else
        wsChkBoxes.Cells.Clear
    End If
   
    With wsChkBoxes
        '.Visible = xlSheetHidden       'Optional
        .Cells(1, "A") = "Ctrl Name"
        .Cells(1, "B") = "TopLeftCell"
        .Cells(1, "C") = "Left"
        .Cells(1, "D") = "Width"
        .Cells(1, "E") = "Row"
        .Range("A1:E1").Font.Bold = True
    End With
   
    ws.Select
   
    For Each shp In ws.Shapes
        If shp.Type = msoFormControl Then
            If shp.FormControlType = xlCheckBox Then
                With wsChkBoxes
                    Set rngParams = .Cells(.Rows.Count, "A").End(xlUp).Offset(1, 0)
                    rngParams.Offset(0, 0) = shp.Name
                    rngParams.Offset(0, 1) = shp.TopLeftCell.Address
                    rngParams.Offset(0, 2) = shp.Left
                    rngParams.Offset(0, 3) = shp.Width
                    rngParams.Offset(0, 4) = shp.TopLeftCell.Row
                End With
            End If
        End If
    Next shp
   
    wsChkBoxes.Columns.AutoFit

End Sub

Private Sub Workbook_Open()
    Dim ws As Worksheet
    Dim rngChkBoxes As Range
    Dim cel As Range
    Dim rngShp As Range
   
    Set ws = Worksheets("ECN")
   
    With Worksheets("CheckBoxParams")
        Set rngChkBoxes = .Range(.Cells(2, "A"), .Cells(.Rows.Count, "A").End(xlUp))
    End With
   
    For Each cel In rngChkBoxes
        With ws.Shapes(cel.Value)
            Set rngShp = ws.Range(cel.Offset(0, 1).Value)
            .TopLeftCell = rngShp
            .Top = rngShp.Top
            .Height = rngShp.Height
            .Width = cel.Offset(0, 3).Value
            .Left = cel.Offset(0, 2).Value
        End With
    Next cel
End Sub

Incrementing A Value - Microsoft Community

gary's student say:

Incrementing A Value

Given the fact that Excel uses a modified IEEE-754 representation for floating point numbers, how can I calculate the smallest floating point number greater than a given number?

For example, suppose A1 contains 1.  what is the smallest number that I can put in A2 such that A2 will be greater than A1?

e.j. gun say:

Let's say in A1 we have:

3.33333333333333000 (that's 15 threes)

Your equation would yield:

3.33333333333336000

but Excel is capable of representing a smaller increment:

3.33333333333334000

so the smallest value greater than the original would be

3.33333333333333000 + 0.000000000000010

or something like

A1 + 10^(-(15-MAX(1,ROUNDUP(LOG10(ABS(A1)),0))))

This equation finds the last decimal location that will have a significant digit, and adds "1" to that place.

At least, I think it does...  :)

Eric

joeu2004 say:

Gary's Student wrote:

Given the fact that Excel uses a modified IEEE-754 representation for floating point numbers, how can I calculate the smallest floating point number greater than a given number?

First, I would not say Excel does uses a "modified" IEEE-754 representation.  I would not even say Excel "does not adhere" to the standard, notwithstanding wording in KB 78113.

Second, that begs the question:  what to do you mean by the next "greater" number?

=A1+2^(INT(LOG(ABS(A1),2))-52) in A2 is the next greater number.  That is, =A2-A1>0 returns TRUE.

However, that difference is not visible when formatted to 15 significant digits, Excel's formatting limitation.

And Excel returns FALSE for the simpler test =A2>A1 because of a dubious heuristic similar to the "close to zero" algorithm poorly described in KB 78113.

PS:  The next smallest number is usually =A1-2^(INT(LOG(ABS(A1),2))-52).  But it is =A1-2^(INT(LOG(ABS(A1),2))-53) if A1 is a power of 2; that is, if =LOG(ABS(A1),2)=INT(LOG(ABS(A1),2)) is TRUE.

PPS:  The smallest magnitude is 2^-1022.  The largest magnitude is (2^1023 - 2^970)*2.

(We cannot write simply 2^1024 - 2^971 because 2^1024 exceeds standard computational limitations.)
