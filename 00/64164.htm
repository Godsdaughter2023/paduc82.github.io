naveen_m say:

Hi Camillus,

Thank you for writing to Microsoft Community Forums.

If hybrid sleep is enabled, the computer writes out all it‚Äôs RAM to the hard drive and enter into a low power state. Windows will put any open documents and programs in memory and on your hard disk and make your computer to enter into a low power state. You can resume your work quickly. Even if there is any power failure, Windows can restore your work from your hard disk.

In sleep mode, if you have any file open and not saved to the hard drive, the file will be lost if there is a power failure. But if hybrid sleep is enabled, Windows will protect the data with a copy on the hard drive even if there is a power failure.

Hybrid sleep will be useful if you are using a desktop, because there are more chances of power failure as they don‚Äôt have a backup battery.

Which edition of Windows 10 are you using?

I would suggest you to uncheck Allow the computer to turn off this device to save power option for Network Adapters in Device Manager to keep internet connection active when in sleep mode.

1.    Press Windows key + R together to open Run.

2.    Type devmgmt.msc and press Enter.

3.    Expand Network adapters.

4.    Right click on a driver and click on Properties.

5.    Go to Power Management tab.

6.    Uncheck Allow this computer to turn off this device to save power option.

7.    Click OK.

8.    Repeat the same steps for all the network adapter driver.
