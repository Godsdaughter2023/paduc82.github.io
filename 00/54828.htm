Method 1:

Run the Power Troubleshooter: - This troubleshooter is an automated Windows 8.1 inbuilt tool which finds and fixes the common issues with the Power on the computer. Perform the below steps to run this tool and check if that helps. 

  1. Press “Windows Logo + W” keys to get the Search option.

  2. Click Yes on the User account control.
  3. Type powercfg /lastwake and press Enter.
  4. Now you will be shown which device or task last woke your computer from sleep.

Note: This will work best if you do this right after your computer wakes up, and before the next time is sleeps. Otherwise you may miss what last woke it previously.

  1. Press Windows Logo + X keys together and select Device Manager from the list.
  2. In the list of hardware categories, find the device that you want to prevent from waking your computer, and then double-click the device name.
  3. Click the Power Management tab and then clear the Allow this device to wake the computer check box. 
  4. Click OK.

Note: If the issue still persist, proceed with Method 3.

Kindly follow the below steps to Prevent programs from waking your computer,

3. On the Change settings for the plan page, click Change advanced power settings.

Reinstall Windows Preinstalled On Computer

OEM computer manufacturers are required to provide at least one way for you to reinstall Windows if you buy a computer with Windows preinstalled - your computer should have included either a restore partition on your hard drive, or Windows install/restore disks. ﻿Most computer manufacturers also have a feature to make your own install disks.

If the restore partition is still there and was not removed, it will reinstall Windows. There is usually a prompt when you first start the computer telling you what key to press to start the restore﻿ - check your manual if you're not sure how to. Or if you have install disks - insert them, restart and boot from them to start the reinstall process.
