hairyfool say:

You don't load the key into the BIOS for Windows 7, this technology was only introduced with Windows 8 and can only be implemented by the system builder.

 

If you have a standard retail Windows 7 upgrade disk during installation it would have asked you to enter the key during setup. If you opted to skip this during setup the system should ask you to authenticate within  a few days of completing setup.

 

If you want to bring up the authentication dialogue manually before then right click on Computer and select Properties, the link to auhentication is at the bottom of the screen.

 

OEM Windows 7 disks from large system manufacturers have a key integrated on the DVD not in the BIOS although they run a check to make sure the disk is being used on the right system so a Dell disk should not work on HP and so on.
