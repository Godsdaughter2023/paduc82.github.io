    Automate PowerPoint 2010 Installer to show Custom Ribbon Addin by - Microsoft Community

    thomasr_cc say:

    Automate PowerPoint 2010 Installer to show Custom Ribbon Addin by default?

    Hi,

    My company are migrating from Office 2003 to Office 2010 in the new year, and I am in the process of taking care of various parts of the Installation process that caters for custom functionality. I have written an extensive VBA/XML toolkit and saved it as .PPAM file - so that it can be found and installed as an addin through the File > Options > Addins facility.

    However, I'd like to make it so that our installation of PowerPoint does this process automatically - so on first open, PowerPoint 2010 displays my custom toolkit on the ribbon without any intervention from the user.

    Is there a way to automate this? Or is there a way that I can write a clever VBA script that would set this feature to the way it should be silently, programmatically?

    Any thoughts much appreciated.
    steve rindsberg say:
    This is actually pretty simple, assuming you have the ability to roll out your PPAM to a directory of your choice and to set a few registry entries:

    HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\PowerPoint\AddIns\YourAddinName

    And under that, create the keys:

    AutoLoad
    REG_DWORD = ffffffff  (hex)

    Path 
    REG_SZ = the full path to your PPAM

    In fact, if you look at that section of the registry after manually installing your add-in, you'll find exactly these values created for you already.

    If you prefer to have the add-in load for all users of the computer and not appear in the Add-ins dialog box (ie, so that users can't easily remove the add-in), put the same registry entries in HKLM instead of HKCU.
