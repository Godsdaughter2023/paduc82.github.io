lemp say:

You don't need to register any version of Windows.

An installation of Windows needs to be "activated" -- i.e., it has to be shown to be licensed -- but it doesn't have to be "registered" -- Microsoft doesn't really care who owns it, although it would like to know that information for marketing purposes.

An installation of Windows that is pre-installed by a computer manufacturer is already licensed and activated.  The computer's manufacturer paid for that license (and incorporated the license fee as part of the cost of the computer).

https://support.microsoft.com/en-us/help/326851/activation-and-registration-information-of-a-microsoft-product

https://www.petri.com/activation_vs_registration

  3. Do Windows 7 or 8(.1) prompt the user at any point prior to downloading the Windows 10 installer if their system is set to perform all updates automatically?

* Windows 8(.1) and 10 automatically set mobile connections to metered when the OS is fully aware the connection is, in fact, a mobile one and should thus be metered; such as connecting directly to a USB port a 3G or similar dongle which is in turn set up by the OS as a 'mobile connection', or connecting to a mobile device's tethered connection via bluetooth.   1. I'm fairly certain I've previously encountered Windows 8.1 automatically choosing to meter a WiFi connection when this was coming from a 'pocket wifi' device (a portable 3G router with wifi connectivity).

    • Would the same be possible for an ADSL modem-router equipped with an automatic 3G fail-over when connected to via wifi?
