steve rindsberg say:

A version or three back, Office stopped supporting Type1/PostScript fonts.

The Aleo family are OTF (OpenType Format) fonts.  OTF fonts can contain either TrueType font data or PostScript font data ... Aleo contains PostScript font data. 

My guess is that the "no go" applies to PostScript in OTF as well as regular PostScript fonts. Unfortunately, I haven't found anything specific one way or the other about this from MS.

In any case, you might want to find a replacement font that's either TrueType or OpenType containing TrueType data.

prashantkumar96 say:
In my Windows 8 system it is also greyed out and it is set to "Automatic (Trigger start)". However I think something like Antivirus should not delay start.

Experimented and found a resolution, guess it will work:

You must disconnect internet before proceeding further, if required take a printout of instructions or open this page in another device but not connect your PC to internet.

1. Turn off Windows Defender.
Open MSE --> Settings tab --> Administrator --> Uncheck "Turn on Windows Defender" --> Click on "Save Changes".


2. Press Win key + R. This will open Run dialog. Type "services.msc" and hit enter.
In the bottom of list of services you will find "Windows Defender Service" with Startup Type "Manual (Delayed Start)" or something like that. Minimize this windows.


3. Press Win key + R. Type "regedit" and hit enter. This will open Registry Editor. Navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WinDefend. First right click on it and select Export. Select desired name and place and save it [It is a backup of that registry key, if something bad happens then you can use this key to revert back]. Now change "DelayedAutoStart" value to "0" OR delete that key.


4. Maximize services.msc, right click on "Windows Defender Service" and click on Refresh. Bravo! You can see now "Startup Type" is "Automatic".


5. Turn Windows Defender back on.
In taskbar you will see Action Center messages. Just click on "Turn on Virus Protection" to get Windows Defender working.

Note: I have tried this method many times with many services and in many computers, first time I have done it with Windows Defender in Windows 8. I tested this in my PC and it is working but I can't say anything about yours! [I think it will work, its just a daunting method of removing "Delayed Start" settings.]
