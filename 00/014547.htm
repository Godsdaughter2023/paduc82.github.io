For a number in a column, I need to find a corresponding value in a - Microsoft Community

fred-a say:

For a number in a column, I need to find a corresponding value in a range in another column and interpolate the corresponding value for that number

Column A,  Pressure Values in PSI:    5837,  5200,   4893

Column B, Psuedo Pressure Values to be determined

 

Column C, Reference Pressure values, PSI:  5873,  5732, 5591, 5451, 5310, 5196, 5028, 4887, 4746

Column D, Corresponding Psuedo Pressure values:  14511, 14160, 13806, 13450, 13091, 12730, 12367, 12002, 11635

 

The function is supposed to look at the first value in Column A (5837) find a pair of numbers that it would fit in Column C and interpolate between the corresponding values in Column D and write the resulting value in Column B

bill manville say:

I will assume:

your column A values are in A2:A4
Column B formulas to be in B2:B4

Column C values in C10:C18 named RefPres

Column D values in D10:D18 named PsuPres
you want linear interpolation

 

Let's build the formula up.


We first need to find the row number within the column C values of the first value that is less than the column A value.

So, in C2 we could put

=MIN(IF(RefPres<$A2,ROW(RefPres)-ROW($C$10)-1,"")
entered as an array formula using Ctrl+Shift+Enter

 

In the case of 5837 this should give 2 since the second value in C is the first which is less than 5837.

 

Now, the interpolated value in that case will be

=14160+(5837-5732)/(5873-5732)*(14511-14160)

 

Using the value in C2 as an index into our table we can express this as a formula in B2:

=INDEX(PsuPres,C2)+(A2-INDEX(RefPres,C2))/(INDEX(RefPres,C2-1)-INDEX(RefPres,C2))*(INDEX(PsuPres,C2-1)-INDEX(PsuPres,C2))

 

We can then copy the formulas in B2 and C2 down to B4:C4

You can do it all in one formula by substituting the C2 formula into the B2 formula wherever C2 appears, but that will make it quite long and even harder to follow.  You could always hide the working cells.
