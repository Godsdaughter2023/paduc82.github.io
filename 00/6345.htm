ken sheridan say:

................I want to be able to see the total weight per Lot at a glance.
To show both in continuous forms view you can use two subforms in an unbound 'container' form.  One subform would show all rows, the other would sum the weight values per lot with a query like the following as its RecordSource property:

SELECT Lot, SUM(Weight) AS TotalWeight
FROM [YourTableName]
GROUP BY Lot;

In the AfterUpdate event procedure of the first subform you should Requery the second subform so that it reflects any changes made to the data in the first.  The syntax would be like this:

    Me.Parent![NameOfSecondSubformControl].Requery

where NameOfSecondSubformControl is the name of the control in the parent form which houses the second subform.  This might or might not be the same as its source form object.

The function you want is INDIRECT and not CONCATENATE. Also I don't understand this underlined bit of your formula:-


(CONCATENATE("tblRes["&B9&"]"),">""",tblRes[Month],C$8))


I think you mean here <>""


Try this:-



Edited formula:-


=(COUNTIFS(INDIRECT("tblRes["&B9&"]"),TRUE,tblres[Month],C$8))/(COUNTIFS(INDIRECT("tblRes["&B9&"]"),"<>",tblres[Month],C$8))
