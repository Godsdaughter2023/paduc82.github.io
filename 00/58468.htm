Hi EaswarS,

Welcome to Microsoft Answers!

First Check if the internal PC Speaker is set to default.

1. Click Start, type Sound into the search field, and then select Sound from the list when it becomes available

2. Verify that the Speakers sound device is set as the default device, and is enabled

NOTE: If the Speakers sound device is not listed, right-click anywhere on the Playback tab of the Sound window, and then select both the Show Disabled Devices and the Show Disconnected Devices options.

3. If the Speakers sound option is Disabled, right-click Speakers, and then select the Enable option.

The sound should now be restored. If the Speakers sound device is still not working properly, then verify whether the sound device is enabled in Device Manager.

 

Step 2: Verify if the sound device is enabled in Device Manager

If there is no sound from the speakers, follow the steps below to determine the state of the sound hardware within the Device Manager.

1. Click Start, type device into the search field, and then select Device Manager from the list.

2. Click on Sound, video and game controllers to expand the list of sound devices installed on the computer.

4. Determine if the sound device in question is enabled or disabled.

·         If a down arrow displays on the volume icon, the sound device is disabled. Go to Updating sound driver.

·          If a down arrow does not display on the volume icon, the sound device is enabled. Go to Verify that the sound device is enabled in Device Manager.

5. Right-click the name of the sound device, and then click Enable. If problems persist, then follow step 6.

6. Right-click the name of the device, select Properties, and then view the message in the Device status field. The Device status field contains either:

A message with more troubleshooting information for your particular device.

Or

A message indicating that the device is working properly.

 

Step 3: Install an updated audio driver

1. Click Start, type device into the search field, and then select Device Manager from the list.

2. Click on Sound, video and game controllers in the Device Manager window.

3. Right-click the name of the audio device listed under Sound, video and game controllers, and then click Update Driver Software.

You may also refer the Manufacturer website and download the latest Vista 64bit compatible drivers.

http://windows.microsoft.com/en-us/windows-vista/Tips-for-fixing-common-sound-problems

timothy davis [ms] say:

Hi Tobore,


There are several reasons that booting can take longer than expected - however most of the time it is related to 3rd party applications behaving poorly during boot.


Windows has great infrastructure for collecting traces to identify the culprit applications, however analyzing the traces usually requires a strong understanding of Windows internals.


One thing to note is that your systems appear to be limited by the speed of the disk.  Just because an application is consuming a lot of your disk bandwidth doesn't mean that it will also be heavily using the CPU.  So task manager may not show you what is going on.


On the bottom of the "Performance" tab in Task Manager, you should see "Resource Monitor".  If you expand the disk section, you can see more information on the active disk activity. 


There are a couple things you can try:


  • Disable/Uninstall your anti-virus to see if that makes a difference.  Some AV products let you specify advanced options, including when to scan.  If AV is causing the perf impact, you might consider trying to prevent it from doing a full scan near boot
  • Make sure you system has enough memory (RAM).  You can have significant perf impact if Windows is having to page out memory to disk while the system is booting.

Hi,

Does Registry Mechanic have an 'undo' function? If so, use it. Otherwise remove this software once you get the system fixed, as like other purported system registry cleaners, it is more dangerous than useful.
