Yes it works. Starting with the November update, Windows 10 (Version 1511) can be activated using some Windows 7, Windows 8, and Windows 8.1 product keys. During the free upgrade, you can use a valid Windows 7, Windows 8, or Windows 8.1 product key to activate Windows 10 (Version 1511 or higher). The following types of product keys are supported:

  • Product keys included in the packaging or the Certificate of Authenticity (COA) attached to your PC that came installed with Windows.

  • Product keys included in the confirmation email you received after buying Windows from an online retailer.

  • Product keys included in the box the Windows DVD came in.

  • Product keys you bought using Windows Anytime Upgrade or Get Genuine Windows.

  • If you bought a Windows 8.1 Pro Pack product key, you can use it to activate Windows 10 Pro.

After Windows 10 is activated, a digital entitlement will be given to your device based on the product key you entered. This means that you’ll no longer need to enter a product key to activate Windows 10 on this device.

On some devices that came pre-installed with Windows 8 or Windows 8.1, the product key is provisioned in the device firmware. On these devices the product key will be verified during setup, and then the appropriate edition of Windows 10 will be installed.

Note:

  • Some Windows 7, Windows 8, and Windows 8.1product keys can't be used to activate Windows 10, such as product keys for volume licensing, including Generic Volume Licensing Keys (GVLK) and Multiple Activation Keys (MAK), and Windows 7 Enterprise, Windows 8 Enterprise, and Windows 8.1 Enterprise.

There are many ways to get there.  You can't do it entirely online.  You need to go to special testing centers to take the individual exams (it can't be done online for obvious reasons - too easy to cheat).  You can use books and online resources to do it without expensive classroom training (I've passed many - not this particular one but others - and never attended a single classroom training session).  While the classroom training might be faster, I don't think it is better.  I passed every exam I ever took on the first try.

It's been a while since my last set of tests so I don't know how they're categorized these days or where the best are located (especially if you want to keep down the cost and/or get the ones that will help you learn the most and pass the quickest).  When I first started, most were free.  The last time I looked, very few free ones offered much of value (but there were exceptions and with the books they were enough).  I will say that unless you have a great deal of experience already, chances of passing with only books (even the Microsoft books or the Exam Cram series) are not very likely.  You really do need more than that (and more/different books isn't the answer - some method of online testing and summarized information is IMHO a {perhaps the} crucial factor).
