sourceWB.Sheets.Copy After:=ThisWorkbook.Worksheets(DestinationSheetCount)

You will also want to remove this line:

  Dim thisWorkBook As String

as ThisWorkbook is a reserved work in VBA and may interfere with the sample code above doing what it is supposed to. The code above would sit in your destination workbook
