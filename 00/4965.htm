Option Explicit

Private Const FLAG_ICC_FORCE_CONNECTION = &H1

#If VBA7 Then
Private Declare PtrSafe Function InternetCheckConnection Lib "wininet.dll" Alias "InternetCheckConnectionA" (ByVal lpszUrl As String, ByVal dwFlags As Long, ByVal dwReserved As Long) As LongPtr

Private Declare Function InternetCheckConnection Lib "wininet.dll" Alias "InternetCheckConnectionA" (ByVal lpszUrl As String, ByVal dwFlags As Long, ByVal dwReserved As Long) As Long
#End If

Sub VerifyURL()
Application.ScreenUpdating = False

Dim myURL As String, cell As Range

For Each cell In Range("B2:B" & Cells(Rows.Count, 2).End(xlUp).Row)
With cell

If Left(.Text, 7) = "http://" Then
myURL = .Text
Else
myURL = "http://" & .Text
End If

If InternetCheckConnection _
(myURL, FLAG_ICC_FORCE_CONNECTION, 0&) = 0 Then
Cells(.Row, 3).Value = "False"
Else

End If


End With
Next cell

Application.ScreenUpdating = True
End Sub

You could create a macro in the template that inserts a number of spaces (4 in this example):

Sub InsertTab()

Sub SetKeyBinding()
    Application.CustomizationContext = ThisDocument

john sr wilson say:
The problem is that it's impossible to have an RGB value for "No Fill" - as you know 0,0,0 is black - there's no RGB value that means Nothing.

Microsoft creates "no fill" by retaining the existing RGB value but setting the visible property to false. I guess your "no fill" shapes were originally white
Check the fill.visible property in your code

Sub WhiteRectangularCover()
Dim sld As Slide
Dim i As Long
On Error Resume Next
For Each sld In ActivePresentation.Slides
For i = 1 To sld.Shapes.Count
If sld.Shapes(i).AutoShapeType = msoShapeRectangle Then
With sld.Shapes(i).Fill
If .ForeColor.RGB = vbWhite And .Visible = True Then _
sld.Shapes(i).Fill.Transparency = 0
End With
End If
Next i
Next sld
End Sub

what's the point of captions? - Microsoft Community

huckleberry99 say:

what's the point of captions?

Captions do not seem to be connected to pictures.  You can insert the caption for a picture, then move the picture and the caption is left somewhere else in the document.  If you put a picture in a text box then you can just type text below the picture and move it all around with the text box.  In what ways is "Insert Caption" useful?

graham mayor say:
If an image is inserted in line (the default setting) it behaves like a large font character and any caption applied is simply formatted text.
If an image is inserted other than in line, the image is in the graphics layer of the document and any caption applied is also in the graphics layer. The image and its caption can then be both selected and moved around the document.
The principle advantages of the caption are that it retains automatic numbering, can be cross referenced, and can be used in a table of figures.
If you insert an image in a text box, you can also insert a caption in the text box.
If you prefer to do it all manually then nothing is stopping you.
