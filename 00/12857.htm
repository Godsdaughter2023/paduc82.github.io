ossiemac say:

Copy the  code below into the Userform module. To populate the Monthview and Textbox the code is placed in the Initialize event code and it will automatically set it to today's date when you show the userform.

Do not rename Private Sub UserForm_Initialize() because it requires the name to automatically run when the Userform is shown.

You might need to edit "CommandButton1" in the Private Sub CommandButton1_Click() if the name of your Enter button is something other than "CommandButton1".

The easiest way to get the correct name for the sub when the button is clicked is while in Design mode, right click the button and select "View code" and if the sub does not exist then it will create the sub name and End Sub but if it does exist then the cursor will be within the sub when the VBA editor opens.

Private Sub UserForm_Initialize()
    With Me   'Me is the generic object for the Userform in the userform VBA code module
        .MonthView1.Value = Date       'Initialize date to today's date
        .txtboxDate.Value = Format(MonthView1, "dd-mmm-yy")
    End With
End Sub

Private Sub CommandButton1_Click()
    'This code for the button to copy the data to worksheet
    Dim lngItem As Long     'Item is reserved word so don't use Item as variable
   
    For lngItem = 0 To ListBox1.ListCount - 1
        If ListBox1.Selected(lngItem) = True Then
            With Sheets("Daily Expences")
                '.Select 'Optional line of code. Not really necessary to select the sheet
                .Cells(.Rows.Count, 6).End(xlUp).Offset(1, 0) = ListBox1.List(lngItem)
            End With
        End If
    Next lngItem
End Sub

ossiemac say:

An after thought which might be interest you.

If you want to also add additional control values to the same row in the worksheet then you can set a range variable to the initial next blank cell and then use Offset to columns to left or right to populate multiple cells on the same row as per the following code example

Private Sub CommandButton1_Click()
    'This code for the button to copy the data to worksheet
    Dim lngItem As Long     'Item is reserved word don't use as variable
    Dim rngDestin As Range
   
    For lngItem = 0 To ListBox1.ListCount - 1
        If ListBox1.Selected(lngItem) = True Then
            With Sheets("Daily Expences")
                '.Select 'Not really necessaary to select the sheet
                Set rngDestin = .Cells(.Rows.Count, 6).End(xlUp).Offset(1, 0)
                rngDestin = ListBox1.List(lngItem)
                rngDestin.Offset(0, -2).Value = DateValue(Me.txtboxDate)    '2 Cols to left
                'rngDestin.Offset(0, 1).Value = DateValue(Me.txtboxDate)     '1 Col to right
            End With
        End If
    Next lngItem
End Sub

Note that you can also use the following line directly to populate the cells on the worksheet without the use of the textbox. The value returned from Monthview is an actual date value and not text so it does not need to be converted back to a date value.

rngDestin.Offset(0, 1).Value = Me.MonthView1.Value
