Are you referring to duplicate music tracks in the Windows Media Player?

This issue may occur if the Windows Media player library is corrupted.

If you are referring to duplicate music tracks in the Windows Media Player, then you may follow the steps given below:

Method 1:

I would suggest you to run the Windows Media Player library troubleshooter and check.
Refer to the following steps to run the troubleshooters.

a. Press the Windows + W key on the keyboard.

b. Type Troubleshooter in the search box and then press enter.

c. Select Troubleshooting. Click on view all.

d. Select “Windows Media Player library” from the list and follow On-screen instructions.

Method 2:

Rebuild the Windows Media Player database and check.

Prior to rebuilding the database, it is required to stop the “Windows Media Player Network Sharing Service”.

Step 1: Follow these steps to stop Windows Media Player Network Sharing Service:

a. Press “Windows key + R”.

b. Type services.msc in the search box and hit “Enter”.

c. Right-click “Windows Media Player Network Sharing Service” and click “Stop” to sop the service.

Note: Restart the service after trying the following steps.

Step 2: Follow these steps to rebuild Windows Media Player database.

a. Exit Windows Media Player.

b. Press “Windows key + R”, type% LOCALAPPDATA%\Microsoft\Media Player in the box and hit Enter.

c. Select all the files in the folder, and then click Delete on the File menu.

Note: You do not have to delete the folders that are in this folder.

d. Restart Windows Media Player.

Step 3: If the above method does not help then you may clear the Windows Media Player database cache files. To do this, follow these steps:

a. Exit Windows Media Player.

b. Press “Windows key + R”, type %LOCALAPPDATA%\Microsoft, in the search box and hit Enter.

c. Select the Media Player
folder, and then click Delete on the File menu.

d. Restart Windows Media Player.

Note: Windows Media Player automatically rebuilds the database.

  3. Now select Programs in the search option.
  4. Under Media Player, select Windows Media Player Settings to run the troubleshooter.

Method 2 : Turn off and turn on Windows Media Player from Windows features.

Turning off Windows Media Player removes it from Windows, but doesn't reduce the amount of hard disk space used by Windows. It will remain stored on your hard disk, so you can turn Windows Media Player back on if you want to.

Step 1: Disable the Windows Media Player

  1. Go to start and in the search type Turn Windows features on or off.
  2. Click on Turn Windows features on or off.
  3. Browse to the Media Features and uncheck the option Windows Media Player.
  4. Restart the computer.


Step 2: Enable the Windows Media Player

  1. Go to start and in the search type Turn Windows features on or off.
  2. Click on Turn Windows features on or off.
  3. Browse to the Media Features and check the option Windows Media Player.
  4. Restart the computer.
