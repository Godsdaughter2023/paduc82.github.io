3. Type devmgmt.msc and hit Enter or right click on Start and select Device Manager.

5. If you are prompted to Confirm then Confirm.
6. Go to the location where you had saved your downloaded setup files of the drivers and right click, then

    select Properties.
7. Select Compatibility Tab.
8. Place a check mark next to Run this program in Compatibility mode and select the operating system

    (Windows 8 or Windows 7) accordingly from the drop down list.
9. Let the driver to install and then check the functionality.
