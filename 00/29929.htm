laurence bothwell say:

Printer Drivers for Windows 7 and HP Deskjet 1280 and HP Deskjet 1220C and Samsung ML-5100A

I've just upgraded from XP Operating System to Windows 7 Home Premium 64 bit Operating System.

 

This has disabled my Samsung ML-5100A Laser Printer

and my wide-format A3 colour printer HP Deskjet 1280 for occasional charts.   

 

 I also have another wide-format A3 colour printer HP Deskjet 1220C (earlier version of 1280, both  taking HP45 and HP78 inks).

 

Are there any reliable printer drivers available to reactivate these 3 printers or are they consigned forever to 'Windows 7 Printer Heaven' ?

 

Laurence Bothwell

London,  England

 

 

a. user say:

Hi Laurence,

 

I see that this is your first post in this forum, welcome to the Microsoft Community forum.

 

When I checked the manufacturer’s website for HP Deskjet 1280 Printer, the latest available driver was for the Windows Vista operating system. In this case, I would suggest you to download this driver and install it in Windows 7 compatibility mode. Refer to this article for more information:

 

http://windows.microsoft.com/en-us/windows-8/older-programs-compatible-version-windows

 

Follow this link to download the driver:

 

http://h20000.www2.hp.com/bizsupport/TechSupport/SoftwareIndex.jsp?lang=en&cc=us&prodNameId=439174&prodTypeId=18972&prodSeriesId=439162&swLang=8&taskId=135&swEnvOID=2102

 

When I checked for HP Deskjet 1220c Printer, it is mentioned that the printer will work with the basic Windows printer driver. For more information, you can refer to these articles:

 

http://h20000.www2.hp.com/bizsupport/TechSupport/SoftwareIndex.jsp?lang=en&cc=se&prodNameId=19142&prodTypeId=18972&prodSeriesId=25521&swLang=8&taskId=135&swEnvOID=4063

 

http://h10025.www1.hp.com/ewfrf/wc/document?cc=us&dlc=en&docname=c01874792&lc=en

 

For Samsung ML-5100A printer, there was no driver available to download. So, if the printer is not working with the basic Windows printer driver, I would suggest you to download and install the Samsung universal printer driver and check. Follow this link to download the driver:

 

http://www.samsung.com/uk/support/model/ML-5100A/XEU

 

Install these drivers and let us know the results. We’d be happy to help if you need any further assistance.
