gerry c j cornell say:

Judy

 

Traditionally the advice offered by experienced computer users is not to allow Windows Update to update motherboard drivers. In the past allowing Microsoft to update these drivers led to many problems. The situation is changing in the direction of letting Windows Update carry out the task but it is still better not to allow it  if you can. Using the update facility in Device Manager frequently fails to find updated drivers when they are available.

 

What is the best way to update drivers does depend on who assembled your computer and the level of computer expertise the user possesses. I have been attempting  to resolve problems where the computer is producing BSOD (Blues Screen of Death) errors and have been doing so for something like 2 years. I have searched many computer manufacturers web sites for drivers. Some sites are easy to navigate and some extremely difficult. Over the last 2 years I have noticed an increasing number of sites having links which can activate tools to scan and offer updated drivers. The quality of information provided about drivers available varies from good to nearly useless.

 

For an inexperienced user I would recommend automatic scanning and detection. As far as I am aware these scanners do not usually offer BIOS updates which can be vital. Many users do not appreciate that when computers are brand new they are frequently sold with out of date drivers. The motherboard DVD containing drivers also frequently has outdated drivers.

 

When you manually update drivers you need to know your computer model, and not just the model series. You then need to know your version of windows and whether it is  32 bit or 64 bit. These are the easy tasks. You then need a list of installed drivers and their versions. You can use Driver View to get these details.

http://www.nirsoft.net/utils/driverview.html

 

You then compare the versions on your computer with the latest versions offered by the computer manufacturer. This is the point where you find out how good the manufacturer's support web site is.


You mentioned third party software which can be used to update motherboard driver. I have seen mixed messages about these sites. Some offer free updates mixed with others you have to pay for. I suspect you can often be asked to pay for something is available elsewhere that is free.  With some updates you have to have detailed instruction to install the driver. Another problem is that you can never be sure that what is offered is the right driver for your computer. Using these sites is fraught with risk.

 

If you post  details of your computer model I will point you in the right direction.

Double-click that and set its startup type to Automatic

Save that and close the Services app
