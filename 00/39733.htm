Hi,


I've just done this with a HP ProBook... I went from a 500GB HDD to a 120GB SSD.....


I used Acronis TrueImage 2011 since I already had it, but the newer versions are on sale all the time...

The retail installation disk is bootable, so no need to install it.


I didn't "Clone" anything... I made a "Backup" Image of Windows only to a external HDD.

I then hooked up the SSD in place of the HDD and "Recovered" the partition to the SSD....

Acronis automatically "Shrinks" it as necessary to make it fit...


If you want to exactly duplicate your entire HDD to the SSD, you can make an image of the entire drive, or individual partitions as desired...


You may also be able to Clone using Acronis, as it is included with a lot of SSD's specifically for this purpose. I prefer making images... just in case.
