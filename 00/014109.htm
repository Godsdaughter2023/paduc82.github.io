         End With
     End If

 End Sub

I have another procedure above this one in the VBA editor which works OK. I presume I'm allowed more than one procedure in the editor.

ronaldoonenil say:

You can't have more than one worksheet Change procedure. The procedure you have above will never be triggered because it is called Worksheet_Change2

You will have to cater for all your code in the one Worksheet_Change procedure.

mike h.. say:

Hi,

No you can't have more than one worksheet change event so you will need to combine them.

Private Sub Worksheet_Change(ByVal Target As Range)
Application.EnableEvents = False
If Not Intersect(Target, Range("B2:B500")) Is Nothing Then
With Target
    If .Offset(0, 1) <> "" Then
    GoTo ws_exit
    End If
        If .Value <> "" Then
    .Offset(0, 1).Value = Format(Now, "dd/mm/yyyy")
         End If
   
End With
End If
ws_exit:
Application.EnableEvents = True
End Sub
