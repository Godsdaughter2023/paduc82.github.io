WIN10: Sound settings input vs. Sound Control Panel - Microsoft Community

nuancetechsupport say:

WIN10: Sound settings input vs. Sound Control Panel

In Win10, there are two different locations to set a default headset for usage:

  1. Selecting the device in the "Choose your input device" drop-down menu under the SOUND SETTINGS > INPUT section.
  2. Setting the device as the Default communication device in the SOUND CONTROL PANEL in the RECORDING tab.

I've recently worked with a customer that does not like how there are two areas to set the default device, especially where changing the default in one menu does NOT change it in the other. This confuses the User since they didn't know which setting would actually set the headset as the default.

After troubleshooting, it appears the 1st method mentioned above is what takes ownership of the device. So for example, if a USB headset is set as default in the SOUND SETTINGS, but a Bluetooth headset is set as default communication device in the SOUND CONTROL PANEL, the USB headset will be the default device when using 3rd-party applications.

However, I cannot guarantee this behavior will occur on ALL 3rd-party applications since I am not the developer of the 3rd-party application.
