Hi Nelson,

The error code can be caused by Windows 7 unable to start the Windows Update service for your computer to connect to the update servers at Microsoft. To prevent the error from happening, you may need to change the Windows Update service settings, and then restart the service. To do this, follow the steps below:

  1. Click on Start and type Admin in the search box.
  2. Press Enter to open the Administrative Tools.
  3. Select Services.
  4. In the right hand pane, locate Windows Update.
  5. Right-click on the Windows Update service and select Properties.
  6. Select the General tab.
  7. Locate the Startup Type option, and set this option to Automatic (Delayed Start).
    Note: If the option is already set to Automatic (Delayed Start), then change the settings to Automatic.
  8. Click OK.
  9. On the Service window, ensure that Windows Update status appears as Started. If not, restart the device to force start the service.
  10. Once you're done, run the update again and check if you encounter the same error.
