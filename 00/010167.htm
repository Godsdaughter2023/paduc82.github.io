mike h.. say:

Hi,

Use this code on the Yes button

Private Sub CommandButton1_Click()
Unload Me
End Sub

For the no button it depends upon whether you want to save any changes that happened when opening, For save changes and close use:-

Private Sub CommandButton2_Click()
ThisWorkbook.Close , True
End Sub

To close without saving

Private Sub CommandButton2_Click()
Application.DisplayAlerts = False
ThisWorkbook.Close , False
Application.DisplayAlerts = True
End Sub
