Hi,

 

If you are trying to install from Store, then, this issue could occur if there are certain compatibility issues with the devices or the drivers installed for them during Out of Box Experience (OOBE) phase of the update installation. Again, when you try to upgrade to Windows 8.1 from Windows 8 using the update from Store, you would not need a product key during the installation phase as it's a free upgrade.

 

1) How are you trying to upgrade to Windows 8.1? Is it using update from Store or using an ISO media?

2) When exactly do you receive the error message?

 

First of all, I would suggest you to disable any other devices apart from mouse and keyboard. You can also disable the any external devices such as video card from BIOS. Refer to the manual provided by the manufacturer of the computer to do so.

Disclaimer: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

 

Secondly, update all the drivers for the devices on the computer. To do so, refer to the manufacturer website to do so.

Step 1: Check the Windows updates for any latest driver updates.

Check in Windows updates if any driver updates are available. If so, install the same and check if it helps.

a)            Press “Windows Logo” + “W” keys from the keyboard.

b)            Type “Updates” in the search bar and click on the icon “Windows Update” from the search result.

c)            On the “Windows Update” window, click on the link: “Check for updates” from the left hand side of the window to receive a fresh list of updates.

d)            If the updates are available, then install all the updates listed there.

 

If the driver updates are not available, then you may try the next step.

 

Step 2: Update Driver Software from the Manufacturer’s website:

Go to the Computer manufacturer’s website and search for the latest drivers available for the all the devices and then install it by following the instructions given in the website.

Now, follow the next steps to start installing Windows 8.1.

 

Step 3:

If you have a third party security software installed on the computer, I would suggest you to temporarily disable it and see if that’s the cause.

Important Note: Security software can help protect your computer against viruses and other security threats. In most cases, you shouldn't disable your antivirus software. If you have to temporarily disable it to install other software, you should re-enable it as soon as you're done. If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

erdeslawe say:

You can't 'clone' an upgrade, because each Windows 10 upgrade has to verify entitlement by reference to the existing functioning Operating System.  What you are currently doing is trying to clone/deploy a clean install of Windows 10, which requires a valid Windows 10 Product Key on each occasion, as it cannot reference the currently installed OS.

As far as I'm aware you can only deploy the OS in the way you are trying to do it in a Windows Enterprise environment where you would have Volume Licensing Agreement with Microsoft.

I understand that the Onedrive program running after startup on Windows 10. I will certainly help you.

By default, OneDrive is set to start when you sign in to Windows to have all files in each folder you choose to sync. In OneDrive on your PC is automatically kept in sync with your online OneDrive.

I suggest you to follow the below steps to disable autostart of OneDrive on Windows 10.

1. Right click on the OneDrive icon on the taskbar notification area and click on Settings.

2. Under Settings tab, uncheck the option Start OneDrive automatically when I sign to Windows.

3. Check to see if the issue is resolved. If not, restart the system.

Hope this helps. If the issue remains unresolved, please get back to us and we would be happy to help.
