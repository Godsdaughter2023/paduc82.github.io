There is a lot of non-helpful information on Security [nuisance] Warnings out there. You can permanently disable the Security Center or the Firewall but you probably don't want to do that. (If you do then you're ahead of my skill curve and may stop reading my post.) The Microsoft Support article referenced below (2288085) is not particularly helpful.

On some machines the Security Warning icon appears in late boot phase and lingers for about a minute or two before going away. This nuisance behavior is harmless and caused because WinXP checks for the operating firewall *before* the firewall has had time to complete booting. (Who let *that* slip by? Bill?) A nice hack (which I do not have for you) would be to add a Registry delay of 120 seconds before WinXP bothered to verify that all security center components were working. I haven't found one yet.

If the security icon remains on for a long time and yet you are sure your security setup is either fully running or otherwise the way you want it, you may either disable the security warning or what it happens to be looking at.

This web reference turns off, for example, just the firewall examination (although you could click other settings):

  http://www.technize.com/turn-off-the-annoying-security-warning-balloon-in-windows-xp/ 

This next one turns off your security center altogether, semi-permanently (unless you specifically go back and restore it so be sure to keep good records of what was done and the program for putting things back the way they were!):

  http://www.mydigitallife.info/disable-or-turn-off-windows-xp-security-center-permanently/ 

This method turns off your WinXP security center but does not turn off or change any of the three security category settings. If you set things right your security is still [pretty] good! Therefore, you could do this *but* you would then have to manually monitor from time to time the fact that Firewall actually was on and running, the same with virus protection and whether Windows was updating either automatically or that you were doing this manually.

Consider the risk that a virus/trojan/generic malware attack TURNED OFF your anti-virus or firewall, or both: check from time to time to make sure your security status hasn't been changed by someone other than yourself. (Hence, the original purpose of the Security Warning icon.)
