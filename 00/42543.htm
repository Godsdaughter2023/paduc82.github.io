syed n say:

Hi bfreemantucker,

 

·        Have you changed any of the file\folder locations on the computer?

 

This issue may occur if Windows Backup skips some files while Windows Backup can't find them. When you perform backup it backs up the Documents, Pictures, Music and Videos through user libraries. So if Windows finds any change in the folder path then the libraries won't recognize them unless you navigate to the corresponding library through Windows Explorer.

 

Check to see if the following helps.

 

a.       Click on "Change backup settings" from troubleshoot dialog

b.      Expand the user node whose libraries are getting reported as skipped.

c.       Deselect the library that is causing the issue.

d.      Expand the "Computer" node in the tree view and select the redirected location of the folders (if any).

e.       Save settings.

 

Refer to the below mentioned thread from TechNet which has a similar issue discussed.
