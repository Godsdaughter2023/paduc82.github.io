vijay a. verma say:
Hi Shinjoy99,
I am Vijay, an Independent Advisor. I am here to work with you on this problem.

1. Windows Key+i > Privacy > Camera > Make sure that Let apps use my camera hardware is ON
Also down below Choose apps that can use your camera is ON for Skype

2. To isolate the issue, can you make call using Web Skype to see if issue is with Skype client

Web Skype - https://web.skype.com/

3. Use below method to uninstall and install Skype.

CTRL+SHIFT+ESC to invoke Task Manager > Kill any instance of Skype.exe

A. If this is UWP (Microsoft Store) Skype app of Windows - Windows Key+i > Apps > Click Skype and Uninstall
Install Skype again from Microsoft Store

OR
Choose to install desktop client of Skype from https://go.skype.com/windows.desktop.download

B. If this is Skype for Desktop client
Copy the below line and paste into Run window and Enter
%appdata%\microsoft\Skype for Desktop

4. If above doesn't work - Share with me your DirectX Diagnostic report
Windows Key+R > Type dxdiag and Enter > Save All information (down button) to a Text file > Upload text file to Onedrive and share the link here

Do let me know if you require any further help on this. Will be glad to help you.
