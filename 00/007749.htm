Do you have the original Access file that the accdr was made from? An accdr is created by renaming either an accdb or accde file to accdr. The purpose of an accdr is to force Access to run the file as if it was the Runtime version. It is used to test running an app that will use the Runtime version when you have the full license installed. It is equivalent to use the /runtime switch in a command line or shortcut. It is possible the file you have has been corrupted or just misnamed. 

The solution is to go back to the original accdb and test that. Than make a new accdr from it.

Storage on Outlook.com is designed to grow at a rate that serves the needs of the vast majority of our customers. Once in a while, a customer's storage needs grow faster than expected. We've provisioned additional storage for your account, but you might want to delete some of your old mail.
