shelluser say:

Just to reply so that others might benefit from it...

 

There is no magic in the "Personal" notebook which, as you said, has its defaults set to "My Documents". If you do happen to wipe the whole thing clear (on purpose or by accident) then the only thing you need to do is logon to your Live account using a browser, and then go to "SkyDrive". Here you simply create a new OneNote notebook using the OneNote web application (click the OneNote icon in the directory of your choice).

Now its easy, do this on your phone:

  • Go to the Office hub and select the 'Locations' tab.
  • Select 'SkyDrive' and find the OneNote notebook.
  • Click on it.

Now you can simply open whatever you need. However, there's more. In order to work properly you should setup a default location for any new notes you make. This basically tells OneNote to store new notes into a certain location:

  • With the OneNote notebook open you'll see several sections.
  • Click on one location untill the menu pops up
  • Select 'Set as default'

Now the section will have a different icon in front of it than the others (instead of a 'tab icon' it'll show a stack of notes). This indicates that any new note you make from the Office hub 'notes screen' will end up here.

 

Its the one thing where I think OneNote mobile is a bit lacking; you can't easily create new notebooks nor sections.

- Power down the computer.

- Wait 5 mins

- Start machine again and Windows 10 setup will restore the system to the previous version of Windows.

- Try installing the update again, do the following before:

1.    Open a Command Prompt (Run as Administrator)
2.    Type: cd %windir% and press the Enter key.
3.    Type: net stop wuauserv and press the Enter key
4.    Type: ren softwaredistribution softwaredistribution.old and press the Enter key.

Proceed with the installation and hopefully it will update.

Yes, the phone is meant to read the settings from the sim, assuming the settings are on the sim. I am using a VF Irl sim in one of my phones now and everthing is working fine. 

 

Before trying anything else, check this setting, Settings>Mobile network>Data connection. It needs to be set to "On" to send MMS's. 

 

Another possibility is an old sim which does not have the settings on it for the phone to read, so if your sim is a few years old, you may want to try a replacement. 

 

Last but not least, you could also try manually setting the apn. Will talk you through this if the above dont solve your problem.

van_m say:

Hi KennethRobinsonxx,

There’s no option for you not to receive the messages when you use Skype on a different device. Since Skype already migrated to cloud-based storage, whenever you sign in to Skype on a different device, the last 30 days of your conversation history will appear across all devices that you will sign in your account.

Also, please be informed that if you delete a conversation/message, it will be deleted from Skype on that specific device, but will still be visible on other devices unless you delete it from there too.

Hope this information helps.
