jason (office engineer) say:

Hi Edwin, in general, text in the same types of cards should be the same size.  If you have text in the caption of an image or video, it will be a little smaller than text in Text cards, which will be smaller than text in Heading and Title cards.  Another way that text could be a little bigger in Text cards is by changing the emphasis on the Text card itself (via the button in the upper right of the card).  If you want to make sure all text is the same size, you will want to make sure it is all in the same type of card.
