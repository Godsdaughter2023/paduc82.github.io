bob jones aka: cybertaz say:

Object Linking & Embedding [OLE] is a Windows technology which never has been fully implemented or supported on Macs. There are numerous discussions on the issue if you search any of the Mac Office Communities.

PDF is one of many unsupported file types... which essentially includes anything other than Word files in Excel or Word & Excel files in Word. PowerPoint for Mac does not support OLE at all. Media file types are a categorical exception.

A PDF can be inserted into Office documents as a Picture, but if there are more than one page only the first will be displayed.
