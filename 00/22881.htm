    how can i sum 5 digit hour in ms excel - Microsoft Community

    avijitguha say:

    how can i sum 5 digit hour in ms excel

    i am unable to sum 5 digit hours in excel example A1 is 12345:55 and B1 is 25648:21 please help
    joeu2004 say: avijitguha wrote:
    i am unable to sum 5 digit hours in excel example A1 is 12345:55 and B1 is 25648:21 please help

    There are two separate issues:  (a) how to input such time values; and (b) once input correctly, how to sum them.

    #B is easy to answer.  Simply:

    =A1+B1

    formatted as Custom [h]:mm.

    The square-bracketed [h] formats time greater than 23 hours.  Excel will format hours up to 71,003,160 -- 2,958,465 days, corresponding to the largest supported date, DATE(9999,12,31).

    #A is harder.  Excel recognizes only up to 9999:59 as time.  (Actually up to 9999:59:59.999.).  Anything larger is interpreted as text.

    I would suggest that you convert A1 and B1 to Excel time values first.  For example:

    C1:  =IF(ISNUMBER(A1),A1,LEFT(A1,FIND(":",A1)-1)/24+MID(A1,FIND(":",A1)+1,2)/1440)
    D1:  =IF(ISNUMBER(B1),B1,LEFT(B1,FIND(":",B1)-1)/24+MID(B1,FIND(":",B1)+1,2)/1440)

    formatted as Custom [h]:mm.

    Then you can copy C1:D1 and paste-special-value-and-format into A1:B1.  Or you can calculate the sum of C1 and  D1, namely:

    =C1+D1

    formatted as Custom [h]:mm.

    If you want to try a manual fix, here is a general process for you to follow:
    Manual Fix

    ·         Make a copy of the file

    ·         Rename the copy from DOCX to ZIP

    ·         Open … .ZIP/word/document.xml in notepad

    ·         Copy the contents of the file to clipboard

    ·         Open Word

    ·         Paste a copy of the copied XML into Word

    ·         (optional) the XML is one long string too hard to read, you can replace some tags, with that tag plus a para mark to break up the text to make it more people readable.

    ·         Open an XML validator, ie this site on the internet: http://www.w3schools.com/xml/xml_validator.asp

    ·         Paste another copy of the XML into the “Syntax Check Your XML” input window

    ·         Click on “validate” button

    ·         Copy the missing tag, ie </mc:Fallback> (yours will be different)

    ·         Return to word Find: mc:Fallback>  (without the </ so you find both open and closing tags).  Repeat find until you hit 2 open tags in a row.  Then you just have to figure out where to put the closing tag between them.  Look for other tags before and after a proper closing tag so you can match the problem area to a good area.
