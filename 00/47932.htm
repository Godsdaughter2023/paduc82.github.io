I feel the reason you can't access Computer B from Computer A is the internet on Computer B might currently be set to a Public Internet connection as its type. This would cause Windows to prevent a lot of connections coming into Computer B, not going out.

Please follow the steps below to change the status of Computer B's internet connection from Public --to-->Private.

Here is how to change the settings:

(All of these steps are to be done on Computer B)

1. Open the Windows 10 "Settings App"
2. Open "Network & Internet"
3. Select "Ethernet"
4. Select "Network" (it's status might be connected or disconnected)
5. Under "Network Profile" , select "Private".
6. Then restart Computer B.
7. Try accessing Computer B again from Computer A.

Please let me know if the above solution leads to the effect you desire.
