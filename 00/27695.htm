teylyn say:

Hello,

this chart type (Insert > Pivot Chart) is different from the pivot chart you get via the traditional method. The charting engine seems to be similar to the new chart types, which also don't allow linking chart titles to cells.

The solution is the same workaround: Hide the built-in chart title, replace it with a text box and link the text box to the cell. Make sure the chart is not selected when you create the text box, otherwise you'll end up with the same issue.

mike h.. say:

Hi,

If the project is locked for viewing then you won't be able to delete it.

ALT+F11 to open VB editor and double click the project,enter the password.

Tools|VBA Project properties and uncheck locked for viewing and remove the password.

Save the workbook and close the application (not just the workbook, you must exit Excel) re-open and the button shouldn't be grayed out.
