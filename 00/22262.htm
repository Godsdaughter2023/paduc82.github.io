chirag. say:

You need to do the following:

1. Setup the monitors in extended display mode.

2. Create your presentation such that it is 3 times the normal width but with the same height. Normal aspect ratio is 4:3 and with 3x monitors side-by-side horizontally, you need the presentation to contain slides for 12:3 aspect ratio. You can change the side dimensions by clicking the "Page Setup" button in the "Design" tab.

3. Use PowerShow to span your slide show across 3 monitors.

brian tillman [outlook mvp 2007-2019] say:
It's simply not a good idea to edit attachments directly from within Outlook, then try to send them on. Save the attachment first, edit, then re-attach. When you open an attachment from within Outlook, you're creating a file within a temporary folder that can come and go. Once that file is closed again, it may or may not stick around and if you're forwarding, then while you may have updated your temporary local copy of the attachment, it may not get repackaged into the message and so you're forwarding the original. Best practice is to save, edit, reattach to guarantee you have your changes included.

suzanne s. barnhill say:

If you have set the default fonts in the document and specified only Arial and Courier for the styles used in the document, you can select the entire document and press Ctrl+Spacebar to clear any direct font formatting. Of course, this will also remove any italics or bold or other properties you have applied intentionally as direct formatting. You can use the Style Inspector (middle button at the bottom left of the Styles pane) to inspect the formatting, but this really doesn't help if you have the entire document selected.

If Calibri is the only suspected interloper, you can search for it specifically.

  1. Press Ctrl+H to open the Replace dialog.
  2. Click More>> to expand the dialog.
  3. With the insertion point in the "Find what" box, click Format and choose Font.
  4. In the Font dialog, select Calibri as the font and leave the rest untouched. Click OK.
  5. With the insertion point in the "Replace with" box, click Format and choose Style.
  6. In the Replace Style list, choose Default Paragraph Font and click OK.
  7. Click Replace All.

This assumes that you have actually modified all the relevant styles to use Arial and Courier, as desired, rather than the default Calibri or Calibri Light.
