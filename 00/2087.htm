john korchok say:

Word does not convert WordArt text to curves or an "object" as you are thinking of it. If it did, it would no longer be editable. For access by other desktop users, sending the font to be installed before viewing the PDF ensures that the WordArt renders correctly. If you are using CreateSpace, you can't use WordArt, since CreateArt doesn't know how to install a font before viewing a PDF.

As a workaround, right-click on the WordArt and choose Save as Picture. Then replace the WordArt with the picture file you just created. Save as PDF. Problem solved.
