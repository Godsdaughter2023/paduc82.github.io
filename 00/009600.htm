vasil l. michev say:

Probably includes the requests as well. You can run it like this:

Search-Mailbox user -SearchQuery 'whatever' -LogLevel Full -LogOnly -TargetMailbox *** Email address is removed for privacy *** -targetfolder blabla

This will generate a detailed log containing all the details for any matching messages and place it in your own mailbox, under newly created "blabla" folder. You can then examine the log file. Or, if you prefer, you can omit the -LogOnly parameter and have it copy the matching messages to your mailbox as well, where you can investigate them :)

To exclude the ones you already deleted, you might want to use the -SearchDumpster:$false parameter.
