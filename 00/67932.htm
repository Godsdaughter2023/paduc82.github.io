alex t jr say:

There is 2 ways.  The first way is to ensure that this option is checked during the install of the program.  If there is no option, then you need to contact the developers to give such an option.  They have strictly coded the installer not to place an icon on your desktop. 

The 2nd way is to navigate to your Program Files folder and find the folder upon which the software was just installed.  Make note of it during the install as it should tell you exactly where it goes.  From there go to that folder, find the executable and right click on it.  There select the option Send To and there is an option for Desktop within there.  Once done you will see it on your desktop.
