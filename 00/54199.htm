Hi,

Thank you for posting the query on Microsoft Community. I understand your concern and will assist you to resolve this issue.

Windows 10: Black Screen after Upgrade or flashing screen.

We have been seeing black screen issues of several types after upgrading to Windows 10. The main causes observed so far are driver issues, output to unexpected port, and possibly also issues with the Lock Screen app. Please use these steps:
If you see the black screen only after sign-in and you can see a mouse cursor:
1. While at the black screen, try using CTRL+SHIFT+ESC to start Task Manager.
2. If it becomes visible, try finding LockAppHost.exe on the Details tab and ending the task.
If this resolves the black screen, use corruption resolution steps:-

• DISM
Try the following steps to run DISM scan to check and restore system file corruption:
•         Open command prompt by pressing Windows key + X and selecting “Command prompt admin”.
•         In command prompt, type the following commands and press Enter after each command:
                DISM.exe /Online /Cleanup-image /Scanhealth
  DISM.exe /Online /Cleanup-image /Restorehealth


• SFC

 Try the SFC scan to resolve the issue. SFC scans the corrupted files and recover it automatically.
To run the SFC/SCANNOW command in Windows 10:
1.       Left click on Start button.
2.       Right click on Command Prompt (Admin) option which will let you run 'Run as administrator’.
3.       Command Prompt Window pops up.
4.       Type:  SFC /SCANNOW and hit Enter.

You can also follow the link below for reference:
http://support.microsoft.com/kb/929833

If this does not work, use File, Run new task in Task Manager to start an Administrator CMD window. Use this to create a new administrator user with the following commands (changing “<password>” to something of the customer’s choosing):
net user test <password> /add
net localgroup Administrators test /add
3. This will give you a test user to try, which will confirm if the issue is user or system-specific.

Check the PC or display adapter manufacturer’s site for an updated driver.

How to update drivers
Steps to update the drivers:-
1. Press Windows +X key.
2. Click Device Manager.
3. Expand Display adapter option, right click on graphics card then select update drivers
4. Choose Browse my computer for driver software. Select the latest driver and update it.

Hope this information helps. Post back with updated issue details for further help.

Thank you

1)       How many language packs have you enabled on your computer?

2)       By this “How can I make this work correctly on a permanent basis?”, are you referring to make a keyboard as the default keyboard layout for your computer?

 

The thing which you have mentioned in your post happens because you have English (UK) language enabled on the computer, apart from the normal one (English (US)). Thus, when you press “Windows Logo” + “Space” keys on the keyboard, it switches the computer keyboard layout from one to the other. If you want to have your normal keyboard layout to be the permanent, then you may remove all the other keyboard layouts which are enabled on the computer. You may follow these steps to do the same.

 

a)       Press “Windows Logo” + “W” keys on the keyboard to display the “Settings” search bar.

b)       Type “Language” in the search bar.

c)       Click on the icon “Language” from the search results.

d)       In the “Language” window, select “English (UK)” and then click on “Remove” button.

e)       This way, you may remove all the unwanted language packs which are listed there.

f)        Now, you may close the “Language” window.

 

If you are trying to make a keyboard as the default keyboard layout, then you may refer to the section: “To make a keyboard your default keyboard” given in the article: “Add a keyboard layout” from this link:

 

http://windows.microsoft.com/en-in/windows-8/change-keyboard-layout
