Ratios of a four gear train. - Microsoft Community

barry buck say:

Ratios of a four gear train.

I have an old coil winding machine with a train of four gears. I have twenty gears to choose from. The ratio will be (A/B)*(C/D). I am trying to set it out in Excel so that I can easily choose the needed gears to fit with wire sizes available. I can do it over many pages with a lot of duplication but maybe somebody has a more realistic solution.

joeu2004 say: Barry wrote:
I have an old coil winding machine with a train of four gears. I have twenty gears to choose from. The ratio will be (A/B)*(C/D). I am trying to set it out in Excel so that I can easily choose the needed gears to fit with wire sizes available. I can do it over many pages with a lot of duplication but maybe somebody has a more realistic solution.

Absent additional constraints, I believe you must look at nearly all 160,000 permutations.  In that case, I think it is best to implement a UDF.  See the VBA function below.

Usage:

  • Select 5 cells in a row, e.g. A1:E1.
  • Enter the following array formula and press ctrl+shift+Enter:
    =gearTrain(R1,G1:G20)
    where R1 is the required ratio and G1:G20 are the gear sizes in ascending order.
  • Then A1:D1 will be A, B, C and D, and E1 will be the actual ratio,
    where A, B, C and D are selected from G1:G20.
    Note that A and C are interchangable, as are B and D.

Alternatively, you can select 5 cells in a column, e.g. A1:A5, enter the following array formula and press ctrl+shift+Enter:

=TRANSPOSE(gearTrain(R1,G1:G20))

Also, the list of gear sizes can be a horizontal row, e.g. G1:Z1.

The UDF returns the set of gears (A, B, C, D) that results in the closest ratio greater than or equal  to the specified ratio.  If you want the closest ratio less than or equal the specified ratio, change the statement

      z = acb / g(d) - dRatio

to

      z = dRatio - acb / g(d)

Also, the UDF assumes that you can use the same gear size multiple times; i.e. you really have 80 gears of 20 gear sizes.  If that is not the case, or if you include multiple gears of the same size in your count of 20 gears, uncomment (remove the apostrophe from) the If statments of the form

      if c = a then goto skipC

There are 3 such statements.

Finally, the UDF treats a difference of -0.000001 between the specified and computed ratios as zero; less than -0.000001 is considered too small.  If you would like a different tolerance, change the statement

      Const e As Double = -0.000001

Note:  The UDF algorithm works best if no gear size is a multiple of another one.  Otherwise, there might be multiple solutions.  In that case, the algorithm makes no effort to select the "best" of those solutions, since you gave no indication of what the "best" criteria might be.

-----

Option Explicit

'usage:
'select A1:E1, enter the following array formula,
'then press ctrl+shift+Enter:
'=gearTrain(R1,G1:G20)
'where R1 is required ratio and G1:G20 are gears in ascending order.
'returns A1=A, B1=B, C1=C, D1=D, E1=(A/B)*(C/D),
'where A, B, C and D are selected from G1:G20.

Function gearTrain(dRatio As Double, rGears As Range)
Const e As Double = -0.000001  'negative diff considered zero
Dim n As Long, a As Long, b As Long, c As Long, d As Long
Dim ac As Double, acb As Double, z As Double
Dim minZ As Double, minA As Long, minB As Long
Dim minC As Long, minD As Long

n = rGears.Count
ReDim g(0 To n) As Double  'note: g(0)=0
If rGears.Rows.Count = 1 Then
   'rGears is horizontal range
   For a = 1 To n
      g(a) = rGears.Cells(1, a)
      If g(a - 1) > g(a) Then GoTo bad  'rGears not ascending order
   Next
ElseIf rGears.Columns.Count = 1 Then
   'rGears is vertical range
   For a = 1 To n
      g(a) = rGears.Cells(a, 1)
      If g(a - 1) > g(a) Then GoTo bad  'rGears not ascending order
   Next
Else
   'error: rGears is not a single row or column
   GoTo bad
End If

'find gear combination that is closest,
'but greater than dRatio.
'assume each gear size can be used multiple times
minZ = 1E+300
For a = 1 To n
   For c = 1 To n
      'if c = a then goto skipC
      ac = g(a) * g(c)
      For b = 1 To n
         'if b = a or b = c then goto skipB
         acb = ac / g(b)
         For d = 1 To n
            'if d = a or d = b or d = c then goto skipD
            z = acb / g(d) - dRatio
            If z <= 0 Then
               If z < e Then Exit For
               minA = a: minB = b: minC = c: minD = d
               GoTo done
            ElseIf z < minZ Then
               minA = a: minB = b: minC = c: minD = d
               minZ = z
            End If
skipD:
         Next
skipB:
      Next
skipC:
   Next
Next

done:
If minA = 0 Then
   'error: no combination >= dRatio
   gearTrain = CVErr(xlErrNum)
Else
   'return row
   'use TRANSPOSE(gearTrain(...)) for column
   gearTrain = Array(g(minA), g(minB), g(minC), g(minD), _
      g(minA) * g(minC) / g(minB) / g(minD))
End If
Exit Function

bad:
gearTrain = CVErr(xlErrValue)
End Function

jeeped say:

If you are referring to the Shannon-Hartley theorem, then the channel capacity would be equal to the bandwidth (in hertz, expressed in Log²) multiplied by signal power (in either watts or volts squared) divided by the interference power (again in either watts or volts squared) plus one (1). Or simply,

C = Blog²(1 + S/N)

... where,

 C is the channel capacity in bits per second
 B is the bandwidth of the channel in hertz (or passband bandwidth in the case of a modulated signal)
 S is the total received signal power over the bandwidth (in case of a modulated signal, often denoted C, i.e. modulated carrier), measured in watt or volt²
 N is the total noise or interference power over the bandwidth, measured in watt or volt²
 ... and S/N is the signal-to-noise ratio (SNR) or the carrier-to-noise ratio (CNR) of the communication signal to the Gaussian noise interference expressed as a linear power ratio (not as logarithmic decibels).

Then put something like this in C1 (assumes you have populated B1, S1 and N1 with appropriate data):

 =B1*LOG(1+(S1/N1)),2)

The brackets around S1/N1 are not required due to mathematical hierarchy, but since I'm doing your homework for you  I thought I'd toss in an extra lesson. If your data (B, S & N) is populating other cells, simply transpose the references in the supplied formula.

If however, you are NOT referring to the Shannon-Hartley theorem, then I'm afraid you have not supplied enough information for me to assist you.

doug robbins - mvp office apps & services (word) say:

The wildcard string that you would need is:


CHAPTER [A-Z]{1,}


However, to apply the Style and insert the Section Break, you will need to use some VBA code:


Dim rng As range
Selection.HomeKey wdStory
With Selection.Find
    Do While .Execute(FindText:="CHAPTER [A-Z]{1,}", Forward:=True, _
    MatchWildcards:=True, Wrap:=wdFindStop) = True
        Set rng = Selection.range
        rng.End = rng.Start
        If rng.Start > ActiveDocument.range.Start Then
            rng.InsertBreak wdSectionBreakNextPage 'wdSectionBreakContinuous
        End If
        Selection.Style = "Heading 1"
        Selection.Collapse wdCollapseEnd
    Loop
End With

Microsoft Windows maintains a list of programs that launch when you turn on your computer. Internet Explorer will start automatically if it appears in this list which is called as Startup items under System Configuration Utility.

A clean boot is performed to start Windows by using a minimal set of drivers and startup programs. This helps eliminate software conflicts that occur when you install a program or an update or when you run a program in Windows 8.1, Windows 8, You may also troubleshoot or determine what conflict is causing the problem by performing a clean boot.
 
How to perform a clean boot in Windows
