xbr dionysus say:

Overview

Actiontec Q1000

Login Information

 

  • Gateway - 192.168.0.1
  • Username - (blank)
  • Password - (blank)

 

How to Find SSID (Network Name)

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Wireless Setup.
  4. Select SSID Setup.
  5. The network name will be located on this page labeled SSID

 

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Wireless Status.
  4. The Wireless Passphrase will be listed where it says security code.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Wireless Setup from the top.
  4. Select Wireless Security from the left.
  5. For step two select the desired Security Type. (To turn the security off select Open for the Security Type)
  6. Select Apply.

 

MAC Filtering

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Wireless Setup.
  4. Select MAC Authentication.
  5. Select the MAC Authentication radio button for Disable.
  6. Select Apply

 

Change Wireless Mode

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Wireless Setup from the menu at the top.
  4. Select 802.11b/g mode from the side menu.
  5. Set the Wireless Mode to 802.11g only.
  6. Select Apply.

 

 

Change Wireless Band (Width)

*This device only supports the 20Mhz Band*

 

Change Wireless Channel

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Wireless Setup from the menu across the top.
  4. Select Channel from the side menu.
  5. In the drop-down box labeled Channel select the desired channel. (Channels 1,3,9,11)

 

DHCP Reservation

*DHCP Reservation is not available on this device.*

 

UPnP

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Advanced Setup.
  4. Check or un-check the option for UPnP.
  5. Click the Save button.

 

DMZ

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Advanced Setup.
  4. Select DMZ Hosting under the security section.
  5. Select the "On" radio button.
  6. Select the Xbox from the drop-down list.
  7. Select apply.

 

Disable DMZ

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Advanced Setup.
  4. Select DMZ Hosting under the security section.
  5. Select the "Off" radio button.
  6. Select apply.

 

Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Advanced Setup.
  4. From the side menu in the security section select Advanced Port Forwarding.
  5. Enter port 3074 for the Port IP Range. (both boxes)
  6. Set the protocol to both.
  7. Enter the IP Address of the console.
  8. Select Add.
  9. Select Apply.

 

Disable Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Advanced Setup.
  4. Select Applications from the left.
  5. Choose the Xbox from the drop down menu in section 1.
  6. Highlight the Applied Rules.
  7. Click Remove.
  8. Select Apply.

 

Firewall

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. From the menu across the top select Advanced Setup.
  4. From the side menu under the security section select Firewall Settings.
  5. Ensure that this option is set to Low or Off depending on preference.
  6. Select Apply.

 

Restore Factory Defaults

Through firmware

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. From the menu across the top select Utilities.
  4. From the side menu select Restore Default Settings.
  5. Select the Restore Default Settings.
  6. Confirm the selection.
  7. Wait for the device to restart.

Through Hardware

  1. Locate the reset button on the back of the device.
  2. Using a toothpick or ballpoint pen press and hold the reset button for 30 seconds.
  3. Wait for the device to restart.

 

How to Find Firmware Version

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Modem Status on the top.
  4. The firmware version will be located on this page labeled Firmware Version:


 

Firmware Update

*Firmware updates are installed automatically by the Internet Service Provider.*


 

Bridging Guide

*Bridge mode should only be used if you have a router to plug into the device.*

*Please have PPPoE information available, if you do not have this, please contact your ISP.*

  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Quick Setup on the top.
  4. Click the link at the bottom for advanced settings.
  5. Select the “RFC 1483 Transparent Bridging” radio button.
  6. Select Apply.
  7. If necessary enter the PPPoE information into the router.

Known Issues

Press WinLogoKey+R, type Regedit in the Run dialog and press Enter/Return.

 

Click Edit > Find, type in itunes6464.msi (or the name of the problem .msi if different), change the Look at boxes so that only Data is ticked, then click Find Next.

 

You should find it somewhere like:

HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Installer\Products\{Long string of letters & numbers}\SourceList

 

Right-click on the immediate parent key with the long string of letters & numbers that contains the SourceList with the reference to the .msi file and click Delete from the context menu. This should stop the Windows Installer from looking for the .msi when you reinstall iTunes.

greg carmack - mvp 2010-2020 say:
Hey Tontos. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

When Windows 10 encounters severe functioning issues, it can sometimes displays an Orange Screen of Death.

How to fix Orange Screen of Death on PC

1. Press and hold your computer’s power button to restart Windows 10. Wait until the PC reboots to see if this simple action solved the problem.

2. If Duet Display is installed on your computer, simply uninstall it. Ddkmd.sys, a component of Duet Display, is known to cause system crashes on Windows.

3. Unplug unnecessary external hardware, such as printers, webcams, additional monitors, mice, headphones and see if this fixes the problem.

4. Remove overclock settings and tools. When you overclock your PC, you change its hardware clock rate, multiplier, or voltage, which generates more heat. As a result, your CPU or GPU may become unstable. Orange Screen of Death errors can also indicate that your GPU is overloaded.

5. Remove softOSD software

Some users have reported that softOSD.exe can cause orange screen issues on Windows 10. Try removing softOSD software from your computer:

Go to Start button > Settings > Apps and Features.

Find softOSD on the list, select it and then click the Uninstall button.
If you’re prompted to remove sds64a.sys, skip that step.

6. Update your graphics card drivers

Go to Start > type Device Manager > expand the Display adapters category.
Right click your graphics card > select Update Driver.
Select “Search automatically for updated driver software”.
If Windows doesn’t find a new driver, go to your device manufacturer’s website and follow their instructions.

If there is no newer driver, try Rolling Back the existing Display driver, or if that isn't available try Uninstall, then restart PC to reinstall.

7. If the Orange Screen of Death appears on boot up and you’ve encrypted your OS using Bitlocker or other encryption tool, type your Bitlocker password, and hit Enter when you see the orange screen. Even if you don’t see a case to type your password, your computer receives the input.

Now, after you log in, launch Command Prompt as an Admin, and type the following command:

BCDEDIT /Set {default} bootmenupolicy legacy

This will turn BitLocker’s pre-boot screen into a “character mode” display, eliminating the orange screen issues.

8. If you’ve encountered the Orange Screen of Death while playing your favorite games, set your game configuration file settings to default. Gamers confirmed that some graphics tweaks occasionally cause OSoD issues.

If the above specfiic steps adapted from https://windowsreport.com/orange-screen-of-deat... don't help then go over this checklist to make sure the install is set up correctly, optimized for best performance, and any needed repairs get done: http://answers.microsoft.com/en-us/windows/wiki...

Start with Step 4 to turn off Startup freeloaders which can conflict and cause issues, then Step 7 to check for infection the most thorough way, then step 10 to check for damaged System Files. Then continue with the other steps to go over your install most thoroughly.
