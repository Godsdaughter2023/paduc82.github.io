heathermassey say:

Lookup a value based on multiple criteria.

I need to find a way to lookup a cell value based on multiple criteria and use that value in a mathmatical equasion.

 

Everytime 001 appears in column C I need some sort of formula that will add column E and F for that row and also go find the value in column E where column B matches the current row and column C is 004. And Populates the result in column G.

 

EX: IF(C2=001, E2+F2+find 004 in column C where the cell in column B matches B2 and return the value in column E, "--")

 

jeeped say:
You sample data doesn't appear to show any match for "+find 004 in column C where the cell in column B matches B2", but a SUMIFS() should return the value of there is a match. In fact, it will return the sum total of all of the values in column E where column B is 004 and columns A matches A2. That might be something to be wary about. The first match (or even the last match) is possible with a more complicated formula, but I'll just offer up a SUMIFS() for now. The formula in G2 would be,

=IF(B2="001", SUM(E2:F2)+SUMIFS(E:E, B:B, B2, C:C, "004"), "- -")

BTW, your numbers in column B & C look like text representation of numbers with zeroes tacked on the left side. You can get more efficient calculation (and possibly less text-to-numbers headaches) if you simply leave the numbers as numbers and use a custom number format of 0000000 and 000.

john korchok say:

The Word interface is somewhat counter-intuitive. Here's how to get this working:

  1. In your form, select the building block content control.
  2. On either the Review or Developer tabs, choose Restrict Editing. The task pane opens.
  3. Under Editing restrictions, check Allow only this type of editing in the document.
  4. Leave the dropdown set to No changes (Read only).
  5. Under Exceptions, check Everyone.
  6. Click on Yes, Start Enforcing Protection.
  7. For each additional control that should remain editable, click on Stop Protection, then select the control and repeat steps 2 to 7.

Your final document will only be editable in the controls (or text areas) that are light yellow.

You could close the workbook instead of saving it. Excel will only prompt you to save the workbook if it has been changed.

Otherwise, you'd need some VBA code (so you'd have to save the workbook as a .xlsm or .xlsb).

Copy the following code into the ThisWorkbook module:

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
    Cancel = Me.Saved
End Sub

There are many facets of a workbook that are retained and have nothing to do with the values and formulas stored within. Printer and page setup immediately come to mind as some of the workbook configuration that are retained within each workbook.

I would think the OP is referring to Disk Cleanup and Defragment and Optimize that are included with all versions of Windows.

To somewhat answer his question, yes you should run these tools at maximum weekly, but seriously once per month is more than efficient.

Every time you start your computer, some files will be written, re-written as per use. With today's SSD drives, it really is not necessary to run these tools as often.
