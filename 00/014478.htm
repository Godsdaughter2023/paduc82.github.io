jeeped say:
You could use VBA to force a text-to-text comparison, leaving 12345A below 123456 but before 234567. Since you've offered no sorting criteria beyond the first 5 characters, it would seem that there is no preferred order beyond that criteria.

Tap Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert, Module (aka Alt+I, M). Paste the following into the new pane titled something like Book1 - Module1 (Code),

Sub mcr_Custom_Sort()
    Dim x As Long, y As Long, lr As Long, lc As Long
    Dim bAsc As Boolean, tmp As Variant
    bAsc = True
    With ActiveSheet.Cells(1, 1).CurrentRegion
        lc = .Columns.Count
        lr = .Rows.Count
        For x = 2 To lr - 1
            For y = x + 1 To lr
                If (Cells(x, 1).Text > Cells(y, 1).Text And bAsc) Or _
                (Cells(x, 1).Text < Cells(y, 1).Text And Not bAsc) Then
                    tmp = .Cells(x, 1).Resize(1, lc)
                    .Cells(x, 1).Resize(1, lc) = .Cells(y, 1).Resize(1, lc).Value
                    .Cells(y, 1).Resize(1, lc) = tmp
                End If
            Next y
        Next x
    End With
End Sub

Modify the third line if you prefer a descending sort (e.g. bAsc = False). Tap Alt+Q to return to your worksheet. Tap Alt+F8 to open the Macros dialog and Run the macro.

A quick pseudo-button to run the macro could be double-clicking on A1. Right-click the worksheet's name tab and chose View Code. When the VBE reopens, paste the following into the pane titled something like Book1 - Sheet1 (Code),

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
    If Target.Address = "$A$1" Then
        Cancel = True
        Application.EnableEvents = False
        Call mcr_Custom_Sort
        Application.EnableEvents = True
    End If
End Sub

Again tap Alt+Q to return to your worksheet. Double-click A1 to run the macro.
