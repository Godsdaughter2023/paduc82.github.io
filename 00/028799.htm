What are the different types of XML document in Word? - Microsoft Community

omar-m say:

What are the different types of XML document in Word?

I want to create Word documents that are XML format

I've noticed that there's 'strict xml' and just xml

Whats the difference?

I want to create a template document that I can use to create many duplicates by reading in a CSV file

I assume strict is the best format?

Thanks

Omar

jay freedman say:

If you're dealing only with Office programs, it doesn't make any difference. There won't be any particular advantage to using either "strict" or regular format.

The difference lies in the fact that Open XML has been codified as an international standard, ECMA-376. Although a lot of that standard is based on work done at Microsoft, the implementation that appears in regular .docx format has proprietary extensions that aren't mentioned in the standard. (That's quite common for computer standards of all kinds.)

If you expect that your document will be edited in some other word processor, one that may adhere to the Open XML standard in some other way or to some other extent, then it's safer to save it as "strict" Open XML. That's the only time it might make any difference.

Much more information about Microsoft's extensions of the standard can be found in the articles linked here.
