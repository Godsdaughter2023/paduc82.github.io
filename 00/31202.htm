a. user say:

Hi,

Thank you for posting your query in Microsoft Community. I understand your concern and I’ll be glad to assist you.

Try these methods:

Method 1: Refer the below link to create new user account:

http://windows.microsoft.com/en-us/windows/create-user-account#create-user-account=windows-8  

Note: Applies to Windows 10 as well.

For Additional information:

http://windows.microsoft.com/en-us/windows/preview-faq#faq=tab0

Method 2:

  1. Log on as a user other than the new user you just created or the user that you want to copy files from.

  2. Press Windows + R

  3. Type . (period)

  4. Double-click Users, double-click the folder with the name of your account, and then double-click My Documents.

  5. Click the Tools menu, and then click Folder Options.If you don't see the Tools menu, press Alt.

  6. Click the View tab, and then click Show hidden files, folders, and drives.

  7. Clear the Hide protected operating system files check box, click yes to confirm, and then click OK.

  8. Locate the C:\Users\Old_Username folder, where C is the drive that Windows is installed on, and Old_Username is the name of the profile you want to copy files from.

  9. Select all of the files and folders in this folder, except the following files  Ntuser.dat Ntuser.dat.log   Ntuser.ini                                                                               

  10. Click the Edit menu, and then click Copy. If you don't see the Edit menu, press Alt.

  11. Locate the C:\Users\New_Username folder, where C is the drive that Windows is installed on, and New_Username is the name of the new user profile you created.

  12. Click the Edit menu, and then click Paste. If you don't see the Edit menu, press Alt.

  13. Log off, and then log back on as the new user.

Keep us posted if you face any issues. We are happy to help.
