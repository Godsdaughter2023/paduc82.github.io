andreas killer say:

macro that will include the cell formatting (border, color fill, text color, bold, italics, etc...) so changes made on the Master sheet will automatically update on subsequent sheets?

In theory yes. But it does not work as you think!

A "referenced cell" means a cell with a formula, means any formula.

We can find formulas like "=Sheet1!A1" and know that the "referenced cell" is Sheet1 cell A1.
But if you have formulas like "=VLOOKUP(A1,Sheet1!A:B,2,0)" we do not know from which cell VLOOKUP pulls the value.

We can catch the moment when the formulas updates, there is a Calculate event. But we do not know which formulas are calculated, means we have to access all cells with all formulas...

At this point, it should be clear that it is theoretically possible, but practically it requires an enormous amount of code. And the runtime is so large that the file is probably unusable afterwards.

An the other hand, if you just want to copy over the formats of many cells e.g. A2:Z100, the runtime of that code should be acceptable.

Andreas.

Private Sub Worksheet_Calculate()
  Dim SaveActiveSheet As Object
  Dim SaveActiveCell As Range
  Dim Source As Range, Dest As Range


  'Refer to all cells to copy in the other sheet
  Set Source = Worksheets("Sheet1").Range("A2:Z100")
  'Refer to the destination cells in this sheet (can be a different location, but same size!)
  Set Dest = Me.Range("A2").Resize(Source.Rows.Count, Source.Columns.Count)
 
  'Prepare
  Application.ScreenUpdating = False
  Application.EnableEvents = False
  'Remember the current sheet
  Set SaveActiveSheet = ActiveSheet
  'Switch to our sheet
  Me.Select
  'Remember the current selection
  Set SaveActiveCell = ActiveCell
  'Apply the formats
  Source.Copy
  Dest.PasteSpecial xlPasteFormats
  Application.CutCopyMode = False
  'Restore the selection
  SaveActiveCell.Select
  SaveActiveSheet.Select
  'Done
  Application.EnableEvents = True
End Sub

Then, format the cell containing that result (see below) using cell format/more number formats/custom, then in the "Type" entry box, put in the exact text you see below in F3 and you should get the format shown below in F4.

In addition to being a little easier than your original approach, it also keeps the data in a date/time format, so that you can calculate other metrics- like average close time, % close time longer than an SLA agreement, or other metrics of interest.

Note that if someone accidently enters a close date earlier than the start date, Excel will return an error (a cell filled with ##### symbols) in the calculated cell, as there is no valid way to show negative duration in Excel.

arvin meyer say:

I've had that trouble and had hundreds of thousands of rows of data to change. I ended up writing a function in Excel and applying it there. I imagine that with just a little more work you could use it from Access with automation. Here's the function:

 

Sub FillFormula()

Dim LastRow As Long

  LastRow = Cells(Cells.Rows.Count, "B").End(xlUp).Row
  Range("G1").Formula = "=A1+B1"
  Range("G1:G" & LastRow).FillDown
  Columns("G:G").Select
  Selection.NumberFormat = "0.00"
 
End Sub

 

The code adds columns A + B in Column G then formats the result. If you only need to format a single columnm try:

 

Sub FormatG()
  Columns("G:G").Select
  Selection.NumberFormat = "0.00"

End Sub
