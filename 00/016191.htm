There are a few reasons why you cant stay logged in.
1-Your browser is set to delete cookies on exit
2-A new security feature has been added
3-A policy has prohibited it on the machine
4-Misc browser issues.


It is possible that your browser option to delete browsing history upon exit is enabled.
Please follow these steps and edit this setting.
1. Open your Internet Explorer.
2. Press and hold Alt key then press "T", you will go to Tools menu of Internet Explorer.
3. Select Internet Options.
4. Under General tab look for Browsing history.
5. Untick Delete browsing history on exit.
6. Click Apply and Ok.
Close your browser and start a new session.
 
It is one of the new security changes.
Part of the change in service include some increased security measures to cut down on the frequency and ease of hacking.

The new log in ensures a more secure way of preventing unauthorized access to your Live ID accounts.
After all if you remain logged in it is an invitation for Hackers to access your account readily to create their mayhem.
No further updates are available on whether or not it will revert back to the previous format.

You could try to click 'keep me signed in' so you will not need to enter your account ID, but you still need to enter your password each time.

Set outlook as the default email client and check if the issue occurs.

 

Follow these steps to set outlook as the default email client:-
Method 1:

Copy and paste the following text in the Open field and then press Enter:

 

Outlook.exe /CheckClient

 

Outlook will prompt if you want to set it up as default email client, click yes.
