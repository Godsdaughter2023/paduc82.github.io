    albert d. kallal say:

    Keep in mind there's two versions of the template now.

    There is the standard longtime classic version, you can find this in 2010 by launching access and then selecting templates.  (if you do not use the search option, you not get the web based version).

    However, if you type in northwind in the search template box, you will get two choices, the 2007 one and the new "web based one". The web based one allows you to publish this application.

    However for the web based version, you cannot use relationships window (web databases allow relationships, but you do NOT use the relationship window to set them up).

    So just keep in mind for access 2010, you can create two types of applications now, and one of those applications allows you to publish your results to the web.

    You should get both templates if you try this url - 

    http://office.microsoft.com/en-us/templates/results.aspx?qu=northwind

     

    The 2007 one is standard is standard Access and allows use of the relationships window.

    The 2010 "web" version is a version that you can run in a web browser if you publish it to office 365 or if you have SharePoint. Once done, you do not need access, but only a web browser to run the web version.

     

    If you not looking to learn about web based development in Access, you likly best use the 2007 version.
