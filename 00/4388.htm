bob jones aka: cybertaz say:

A few meaningful things to check:

  • In File> Page Setup - Page confirm that Print Quality is set to High
  • In File> Page Setup - Sheet confirm that Black & white is not checked. Some printers try to 'formulate' black ink for anything designated as black by combining other colors (rather than using the black ink cartridge) which can result in a murky effect
  • In File> Page Setup - Sheet confirm that Draft quality is not checked
  • In Format> Cells - Font make sure that Normal font is not checked
  • Confirm that Office 2011 is fully updated [14.4.8 as of this writing]
  • Despite your conviction :-) it wouldn't hurt to check the mfr's web site to see whether there is a driver &/or firmware update for your model of printer. Each program uses a different set of parameters, so just because only 1 program's documents don't print correctly doesn't dictate that the print subsystem isn't at fault.

This is the code I use:
Private Sub txtPassword_AfterUpdate()

    'Check that EE is selected
    If IsNull(Me.cboUser) Then
        MsgBox "You need to select a user!", vbCritical
        Me.cboUser.SetFocus
    Else
        'Check for correct password
        If Me.txtPassword = Me.cboUser.Column(2) Then
            'Check if password needs to be reset
            If Me.cboUser.Column(3) = True Or Me.chkChangePW Then
                DoCmd.OpenForm "frmPasswordChange", , , "[LogonID] = '" & Me.cboUser & "'"
                Forms!frmALCLogin.Visible = False
            Else
                DoCmd.OpenForm DLookup("[InitForm]", "tblParameters")
                Forms!frmALCLogin.Visible = False
                Exit Sub
            End If
        Else
            MsgBox "Password does not match, please re-enter! You have " & 3 - intTries & " left.", vboOkOnly
            Me.txtPassword = Null
            intTries = intTries + 1
            If intTries >= 3 Then
                MsgBox "You have exceeded the # of logins, please see administrator!"
                DoCmd.Quit acQuitSaveNone
            End If
            Me.txtPassword.SetFocus
        End If
    End If

End Sub
