5. Start the software installation.

stefan blom say:

Make sure the document whose settings you want to modify is the active document. On the Review tab, click the dialog launcher button (the little arrow on the bottom right) of the Tracking group. In the dialog box that opens, click the Advanced button. In the Advanced Track Changes Options dialog box, clear the check mark next to "Track formatting." Click OK. Note that this change will prevent all format changes from being tracked.

To get rid of existing format tracking, just right-click them and choose to accept the change.
