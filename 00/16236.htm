macropod say:

Close. In my previous reply, I said:

when you manually add a reference, the referenced work is added to both the current document and Word's master source list. That means you can copy from the master list into any other document in which you want to reference the same work.

So, if you want to cite a work for which you have an existing reference in the master list but not in the current document, simply copy it from the master list to the current document. You do that via References>Manage Sources>Copy. You can then cite the work in the current document via References>Insert Citation and, if you insert a bibliography, it will be automatically added to that as well.

zack barresse say: Yup, instead of using the "button" command use the "control" command.  This XML will disable everything but leave the Undo and Redo enabled on the QAT...


<customUI xmlns="http://schemas.microsoft.com/office/2006/01/customui">
   <ribbon startFromScratch="true">
      <qat>
         <sharedControls>
            <control idMso="Undo" />
            <control idMso="Redo" />
         </sharedControls>
      </qat>
   </ribbon>
</customUI>

This is for a 2007 ribbon UI.  2010 uses a different schema...

<customUI xmlns="http://schemas.microsoft.com/office/2009/07/customui">

Assuming that your column headings are on row 1, that the projected plan values are sorted in ascending order with the lowest value in cell F2 and the highest value in cell BO2, and that you want the result for the first, i.e. leftmost, column where the value is greater than the value in cell Sheet2!F2, you can try this formula:

=INDEX(Sheet1!F1:BO1,MATCH(Sheet2!F2,Sheet1!F2:BO2,1))

Publishing to Social Networks from MyPhone - Microsoft Community

steve giscombe msft say:

Publishing to Social Networks from MyPhone

Hello All,

I am a member of the MyPhone team, and I have noticed that there haven't been many post regarding the new publish to Social Network feature of MyPhone, and I wanted to make everyone aware of what the feature is and how to use it.

So the new feature is publishing photos from your Windows Phone or the MyPhone Web Portal to social networks including Windows Live Photos, Facebook, MySpace, and Flickr.

There are 2 ways that you can going about publishing photos to social networks:

- You can publish from your Windows Phone. Here are the steps:
    1. Go to the Microsoft MyPhone application
    2. Click Menu/Share Photos/Social Networks (This will bring up a list of the social networks that you have added to publish)
    3. Add any social network that you would like to publish to
    4. Once you have the social network(s) you want to publish to added, you go to the Pictures and Videos application
    5. Click on the photo you want to publish
    6. Go to menu and click Share
    7. Choose the social network to publish to add a caption and click share

- You can also publish from the MyPhone Web Portal (www.microsoft.com/myphone)
    1. Once you have signed in navigate to Socail networks from the left panel (this will bring up a list of social networks which you have added for publish)
    2. Click 'Add a social networking account'
    3. Choose the social network that you want to add (you will be redirected to that site for authentication)
    4. Onceyou have added the social network(s) you want to publish to, navigate to the Photo page using the left panel
    5. Choose the photo(s) you want to publish and click 'Share Photos' from the top panel
    6. Choose the social network you want to publish to and click the Share button

I hope this helps people to discover one of the new features brought to you by Microsoft MyPhone!

Thanks,
Steve
