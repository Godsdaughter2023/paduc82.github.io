Assumed: Tab 1 is named Sheet1, Tab 2 is named Sheet2, Empolyee IDs are in column A of both sheets.

On Sheet2, in B2 enter

=IF(ISERROR(MATCH(A2,Sheet1!A:A,FALSE)),ROW(),"")

and copy down to match your list.

Then on sheet 1, in a cell where you want the missing IDS to start, use the formula

=IF(ROW(A1)<=COUNT(Sheet2!B:B),INDEX(Sheet2!A:A,SMALL(Sheet2!B:B,ROW(A1))),"")

and copy down until it returns cells that look blank.

How do i publish a Sway presentation externally?

Created a newsletter and want my viewers to view the slide show.  Each time I try and publish, it comes back to the version for editing.  Any tips to how I can share please?

jason (office engineer) say:

Hi Jenny,

There are two types of links you can share for the Sway.  The first one is the one you are using to edit the Sway.  With that link, if the person opening it is signed in as the author of the Sway, they will get the edit page; if the person opening it is not an author of the Sway, they will get the view page.  You can test this by opening the URL after signing out or in and In Private browser window.

If you, as the author, would like a link to the Sway that opens directly into read mode for yourself, add ?play to the end of the URL (note that this is not necessary when sending the link to others).

The other type of link is an edit link and that can be grabbed from the Share menu.  This link should only be used to give author access to your Sway to other users.  When that link is followed, the person opening it will be given the choice to only view the Sway or to edit it and become an author on it.  Only use that if you want to give edit permissions to someone else.

Hope this helps!

Jason
