Windows 10 Corrupted my External Hard Drive - Microsoft Community

qiloff say:

Windows 10 Corrupted my External Hard Drive

If Windows 10 is corrupting your extrenal hard drives, for instance a data drive that is a portable usb drive, the solution is 2 fold. 

First disable "fast Start Up" :

1 settings > 2 System > 3 Power and Sleep > 4 Additional power settings (On right side)  >5 Choose what the power buttons do >6 Choose settings that are currently unavailable >7 turn off fast start up (uncheck). 

I see no difference in the start up speed, but it does corrupt the external drives when on. I am guessing now, but I suspect windows 10 writes to the system drive instructions on what it's state was before shut down, and this includes definition of the external drives. 

Part 2:  The annoying chkdsk at boot up (press any key to stop error checking)

 https://support.microsoft.com/en-us/help/160963/chkntfs-exe-what-you-can-use-it-for

registry hack:

Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager/

 modify the  BootExcute entry on right panel delete the default value,

BootExecute:  REG_MULTI_SZ:  autocheck autochk *    (delete the part that says "autocheck autochk *"

Restart the machine.

This is reversable, so nothing permanent is done. Just run chkntfs.exe /? from the command prompt for info on the command,

or chkntfs.exe /D to go back to disk corruption again.

    To enable Compatibility View for specific Web sites that are not displayed correctly or that are not working correctly, follow these steps:
