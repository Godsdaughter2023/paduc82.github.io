Only way I can think of without using a macro is to have all information in one sheet (let us call it Master)... and in all other sheets have the name call copied over and then use VLOOKUP to populate all the fields from the Master sheet... then whatever you update on Master will reflect in other sheets...

Name id

You can sort the sheets as you want... but make changes only in Master.

doug robbins - mvp office apps & services (word) say:

Here is one way to do it.

Set up the numbering for the Heading 1, Heading 2, Heading 3 styles in the normal way by selecting the type of numbering that you want (Using one of the legal type defaults makes it easy as the numbering levels are then assigned to the Heading Styles.

Then, by clicking on the Define New Multilevel List link in the above dialog, access the Define new Multilevel list dialog and select the level 1 and click on More and change the Follow number with: from the default tab, to a Space.

You will probably also want to format the Heading 1 style so that it has no space before or after the paragraph.

Now define a new style, to which I suggest that you give the name Chapter and base it on the Heading 1 Style and set the Style for the following paragraph to he Heading 1

Make whatever other formatting changes that you want to that style, but I would suggest that you format it so that the paragraphs have no space after them.

Then insert a paragraph with that style into your document and with the selection in that paragraph, access the Define new Multilevel list dialog and change the numbering style for the Level 1 to I, II, III and type the word Chapter followed by a space before the number and also change the Follow number with to a Space.  Then link that level to the Chapter style, all as shown below:

Then, whenever you insert a Chapter style paragraph, it will automatically be followed by a Heading 1 style paragraph which will have an arabic number corresponding to the Roman number of the Chapter Style paragraph.  So that these paragraphs are not visible in your document, format the paragraph of the Heading 1 style so that is has a Negative left indent to causes the number to be located off the edge of the paper.

You can then get your captions numbered in the way that you want

Note:  If you need a Table of Contents in your document, you will need to access the Table of Contents Options dialog and remove the 1 from the TOC level: for Heading 1 and locate your Chapter style in the list of Available styles and insert a 1 in the TOC level: for that style.
