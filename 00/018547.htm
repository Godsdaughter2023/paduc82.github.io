tsarava say:

Hi Selttan

If you want to compare dates for the same city, these two conditional formatting formulas applied to column B should do.

For the case where the date in D is later:

=VLOOKUP($A2,$C$2:$D$7,2,0)>$B2

and format the font green

For the case where the date in D is earlier:

=VLOOKUP($A2,$C$2:$D$7,2,0)<$B2

and format the font red.

Hope this helps

tsarava say:

Hi Selttan

It might be a problem with the dollar signs ($). It's important to use them correctly to make parts of the references absolute. For example $A2 will always refer to column A but the row number may change, whereas $A$2 will always and only refer to cell A2.

In your conditional formatting case you need the range ($C$2:$D$7) to be absolute, but cells $A2 and $B2 need to be able to change to $A3,$B3; $A4,$B4 etc.

jay freedman say:

It sounds like Word is failing to save the Ruler setting in the registry, which it then reads on each startup to determine whether to show the ruler.

Run the registry editor (regedit.exe) and work your way through the tree structure on the left to the key HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\16.0\Word\Options. On the right, scroll down to the entry named Ruler. Its value will be either 0x00000000 (0) if the ruler is not displayed or 0x00000001 (1) if it is displayed.

If the value stays at 0x00000000 (0) after you shut down Word while the ruler is displayed, there's something interfering with Word's setting. Are there other options that won't change properly, or is it just this one?

You can try changing the value in the registry. While you're viewing the key in the registry editor, double-click the Ruler entry's icon. In the small dialog that appears, change the number in the "Value data" box from 0 to 1 and click OK. Then open Word to see whether the ruler appears.

Private Declare Function FindExecutable _
    Lib "shell32.dll" Alias "FindExecutableA" ( _
    ByVal lpFile As String, ByVal lpDirectory As String, _
    ByVal lpResult As String) As Long

Then, use the following procedure:

 

Sub OpenPDF()
    Dim ExeName As String
    Dim FileName As String
    Dim N As Long
   
    ExeName = String(260, 0)
    FileName = "G:\Reference Documents\HP 16.pdf" '<<< CHANGE
    N = FindExecutable(FileName, vbNullString, ExeName)
    ExeName = Left(ExeName, InStr(1, ExeName, vbNullChar) - 1)
    Shell ExeName & " " & Chr(34) & FileName & Chr(34), vbNormalFocus
End Sub
