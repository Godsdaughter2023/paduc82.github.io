Hi Sally,

To enable Windows Defender, please follow the steps below.

  1. Click the Start button.
  2. Type gpedit.msc and press Enter.
  3. Go to Local Computer Policy > Administrative Templates > Windows Components then select Windows Defender.

don cuthbert say:

OK, so Windows 10 is already installed and activated on this same PC.  You want to clean install it again on another disk or partition.  I have done this many times.  I have multi-boot setup on my PC, with 2 builds of Windows 10 Pro, as well as Windows 8.1 Pro with Media Center and Windows 7 Ultimate.

I would stick with USB media, so that you can re-use it.  Plus, a USB flash drive installs much faster than from a DVD.

In your original post, the error was, "The application or operating system couldn't be loaded because a required file is missing or contains error. File: \windows\system32\boot\winload.exe".  Winload.exe is used on Legacy MBR disks on non-UEFI mode PC's.  Does your PC have UEFI BIOS mode available or running?  If so, it should not be trying to use winload.exe.

The install should be quite simple, after booting from the USB.  Just proceed as normal and choose Custom install.  When it comes to asking where to install Windows, select the disk or partition you want to use for the new clean install.  If you want, you can delete any unused partitions from that disk first.  Windows will then create partitions as needed and install there.  It will reboot as needed during install.  Do not interrupt or select anything from the boot menu during the install.  In the end you will have a dual-boot setup with the new install as default.  You can switch between them as needed or choose the original install to be default later on.
