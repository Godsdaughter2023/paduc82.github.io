But what I get is: Microsoft Access cannot find the object 'me.'

I started trying me.requery in different events with no success.

What am I missing?

Thanks,

 

hansv mvp say:

The calculations on the main form will only be updated when the changes in the subform record have been saved.

The keyword Me can be used in event procedures in VBA, not in macros or in expressions.

If you want to recalculate controls on the main form when a value on the subform changes, try this:

 

Open the subform in design view.

Select the Amount control.

Activate the Event tab of the Property Sheet.

Select [Event Procedure] from the dropdown in the After Update event.

Click the Builder button (the ... to the right of the dropdown arrow).

Make the code look like this:

 

Private Sub Amount_AfterUpdate()
    Me.Dirty = False
    Me.Parent.Recalc
End Sub

 

This saves the record and recalculates the parent form, i.e. the main form.

Mail-merge View Source - Microsoft Community

zorabelle say:

Mail-merge View Source

Unable to "View Source" when editing a word list in mail merge.  The button is grayed-out.  Is this feature no longer available?

a. user say:

here, I was able to enable this feature as follows:

  1. click Office Button->Word Options->Customize
  2. in the Choose Commands from dropdown, select All Commands 
  3. locate and select Edit Data Source, and click Add to add it to the Quick Access Toolbar (QAT)

Then click that button in the QAT and the View Data Source button in that dialog box seems to work OK.

However, I suspect the reason that it has been greyed out may be because Word (or perhaps Word under Vista or later)  tends to make a temporary copy of the data source and in that case may not always commit changes made to the temporary data source to the original data source. But maybe the Microsoft people know the reason.

You can also do "View Source" for this type of data source by writing a VBA macro lsuch as

Sub mmviewsource()

ActiveDocument.MailMerge.EditDataSource

End Sub

and assigning it to a keystroke, the QAT etc.
