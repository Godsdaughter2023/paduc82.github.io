zorvek say:

Assuming that the two columns to compare are A and B and the data starts in row 1 and the results are to be put in column C starting in row 1 and the results look like "A10 matches B15" then use this macro:

Public Sub ListMatches()

    Dim Column1 As Range
    Dim Column2 As Range
    Dim Results As Range
    Dim Row1 As Long
    Dim Row2 As Long
    Dim ResultsRow As Long
    
    Set Column1 = ActiveSheet.Range(ActiveSheet.Range("A1"), ActiveSheet.Cells(ActiveSheet.Rows.Count, "A").End(xlUp))
    Set Column2 = ActiveSheet.Range(ActiveSheet.Range("B1"), ActiveSheet.Cells(ActiveSheet.Rows.Count, "B").End(xlUp))
    
    Set Results = ActiveSheet.Range("C:C")
    Results.ClearContents
    
    ResultsRow = 1
    For Row1 = 1 To Column1.Rows.Count
        For Row2 = 1 To Column2.Rows.Count
            If Column1(Row1).Value = Column2(Row2).Value Then
                Results(ResultsRow).Value = Column1(Row1).Address(False, False) & " matches " & Column2(Row2).Address(False, False)
                ResultsRow = ResultsRow + 1
            End If
        Next Row2
    Next Row1

End Sub

suzanne s. barnhill say:

First of all, the default font color (for the Normal style) is not black but Automatic (black on light colors, white on dark colors). I would not recommend changing the default color for all new documents, but if you really want to do it:

  1. Press Alt+Ctrl+Shift+S to open the Styles pane.
  2. In the bottom left corner, click Manage Styles.
  3. Select the Set Defaults tab.
  4. Change the font color to the desired one.
  5. Select "Only in this document" or "New documents based on this template" and click OK.

Two caveats: you'll need to do this for both Body and Headings. Also, font color will depend on the theme applied, so a better approach would be to define a theme with the font color(s) you want.
