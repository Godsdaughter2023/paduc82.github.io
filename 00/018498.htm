Changing the shape of the Excel pointer - Microsoft Community

smithes say:

Changing the shape of the Excel pointer

In Excel 2003,  how can I change the shape of the pointer from a plus sign to an arrow? If VBA must be used, could you direct me to a beginner's site? Thank you.
rick rothstein, mvp - excel say:

First, let's get the code in place. Press ALT+F11 from any worksheet in the workbook you want to be able to do this in... this will take you into the VB editor. Once there, click Insert/Module on its menu bar and then copy/paste this code into the code window that opened up...

Sub ChangeCursor()
  Application.Cursor = xlNorthwestArrow + xlDefault - Application.Cursor
End Sub

Now, go back to your worksheet. The above macro will toggle the cursor between an arrow symbol and a cross symbol everytime it is executed. To execute the macro, press ALT+F8, select ChangeCursor from the list and then click the Run button... you cursor will now be an arrow. Run the macro again, and it will be a cross again. Run it one more time and its back to an arrow... and so on. If you want, you can add a custom button to your toolbar and assign that macro to it... that way you can toggle the cursor shape back and forth by just clicking the button.
