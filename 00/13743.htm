rich007a say:

This should help:

Sub DisableAlerts()
    Application.DisplayAlerts = False
End Sub

Sub EnableAlerts()
    Application.DisplayAlerts = True
End Sub

Sub CloseAll()
' This procedure closes all open visible workbooks except the workbook
' containing this code (ThisWorkbook).
' Un-comment "SaveChanges:=True" to have the workbooks saved on closing.
'
    Dim i As Long
   
    Application.DisplayAlerts = False
    For i = Application.Workbooks.Count To 1 Step -1
        If Application.Workbooks(i).Windows(1).Visible _
            And Not Application.Workbooks(i) Is ThisWorkbook Then
           
            Application.Workbooks(i).Close 'SaveChanges:=True
        End If
    Next i
    Application.DisplayAlerts = True
End Sub

Yes, set the attribute for deleted text to be hidden and for inserted text, have it so that it appears as normal text (none) and make the appropriate setting for the Changed lines item (Keft, Right or Outside)
