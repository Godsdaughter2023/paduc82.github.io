It is possible to connect to Access in such a way that the parameter prompt is shown, but it's a slightly flaky, oldfashioned way, so I wouldn't recommend it. But for what it's worth:

  • In Word, select Tools | Options...
  • Activate the General tab.
  • Tick the check box "Confirm conversion at Open".
  • Click OK.
  • Now connect to the data source for the mail merge again.
  • Word will ask you how to connect to the database.
  • Select "MS Access databases via DDE" and click OK.

I prefer the option to assemble the SQL for the mail merge in code. It isn't necessary to change the SQL of a stored query (although that will work fine), you can specify the SQL on the fly, by automating Word from Access.
