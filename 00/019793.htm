karl timmermans [outlook mvp 2012-2018] say:

You haven't indicated either the type of folders involved or how these were originally created.

The <Move> option is not available on a folder that was <ever> designated as a default folder (in the same root list as an <Inbox> folder. That would include opening a PST file that was originally the default in another installation or other configuration.

In this instance, your only option would be to highlight all the items in the <source> folder and then drag 'n drop those items into the destination folder. Important note <Copying> a folder will not result in the items being copied to the destination folder - that option will merely result in a sub-folder being created underneath the folder you designate as the destination folder.

Sub AllFiles()
    Dim folderPath As String

doug robbins - mvp office apps & services (word) say:

Here's a way that might work:

With the Content Control Check boxes assigned the tags of row1, row2 and row3

and the following code in the ThisDocument object in the Visual Basic Editor:

Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, Cancel As Boolean)

Dim strTag As String

With ActiveDocument

    For Each CC In .ContentControls

        Select Case CC.Tag

            Case "row1"

                If CC.Checked = True Then

                    .Tables(1).Cell(1, 2).Range.Font.Hidden = False

                Else

                    .Tables(1).Cell(1, 2).Range.Font.Hidden = True

                End If

            Case "row2"

                If CC.Checked = True Then

                    .Tables(1).Cell(2, 2).Range.Font.Hidden = False

                Else

                    .Tables(1).Cell(2, 2).Range.Font.Hidden = True

                End If

            Case "row3"

                If CC.Checked = True Then

                    .Tables(1).Cell(3, 2).Range.Font.Hidden = False

                Else

                    .Tables(1).Cell(3, 2).Range.Font.Hidden = True

                End If

        End Select

    Next

End With

End Sub

when a box is checked, the text in the adjacent cell is formatted so that it is not hidden.  Otherwise it is hidden

The display of hidden text must be turned off for this to work.  If the display of hidden text is turned on you would see in which you can probably make out the dotted line under the text in the second and third rows that indicates that it is formatted as hidden.

another option would be to have the code toggle the font colour between White and Automatic.
