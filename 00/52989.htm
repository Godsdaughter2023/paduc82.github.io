Hi David,

Thank you for contacting Microsoft Community. I understand that you are getting a blank screen after upgrading to Windows 8.1. I will surely help you to fix this issue.

If you are getting a blank or black screen after upgrading to Windows 8.1, it could be due to an outdated or incompatible display adapter driver. Updating to the latest display driver should resolve this issue.

 

To understand the issue better, let me know what is the make and model of the computer?

 

I would suggest you to boot the computer to Safe Mode with Networking and install latest display driver for Windows 8.1 from the Computer Manufacturer website.

Step 1:
Boot the computer into safe mode with Networking. In Safe Mode with Networking computer starts with minimal set of drivers and files.

Windows Startup Settings (including safe mode)
http://windows.microsoft.com/en-US/windows-8/windows-startup-settings-including-safe-mode


Step 2:
Now install latest display driver for Windows 8.1 from the Computer Manufacturer website. If display drivers for Windows 8.1 are not available in the Computer Manufacturer website, I would suggest you to install latest display driver in compatibility mode and check if it helps.

 

To install the display driver in compatibility mode, I would suggest you to follow these steps.

 

a. Download the latest display driver for Windows 8 or Windows 7 from the manufacturer’s website and save it on your local disk.

b. Right-click on the setup file of the driver and select “Properties”.

c. Select “Compatibility” Tab.

d. Place a check mark next to “Run this program in Compatibility mode” and select Windows 8 or Windows 7 from the drop down list.

e. Let the driver install and then check the functionality.

burrwalnut say:

1. If Control Panel is not appearing at all, open a Run window (Windows Logo key+R), type regedit, press Enter and in the left pane navigate to HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer.

 

If they are not there it must be a different problem and if you are denied entry to regedit it’s a classic symptom of infection and you should run your ‘anti‘ programs.

 

2. If some items are missing, in a Run window (Windows Logo key+R), type regedit, press Enter and in the left pane navigate to HKEY_CURRENT_USER\Control Panel\don’tload and delete items in the right pane that you want to see in Control Panel.

 

I suggest you create a System Restore point before making registry changes, just in case something goes wrong.
