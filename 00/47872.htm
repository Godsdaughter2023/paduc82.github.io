abdul raqeeb kola say:

Hi,

It would appear that this function is not available in Windows XP.
Hybrid sleep is only supported by Windows Vista and Windows 7.

The main difference between hybrid sleep and sleep is that in hybrid sleep, the RAM is also saved to the hard disk, so that in case of a power failure while the computer is on hybrid sleep, it can still recover.
Hybrid sleep is basically a combination of sleep and hibernation. In XP, you can hibernate (contents of RAM saved to disk) by holding down Shift at the shut down dialog; the Stand By button will turn into a Hibernate button. (However, it will not resume as fast as it does from standby/sleep.

With respect to 4, I assume your question is rhetorical.  It's Apple.  They do what they want. 

The long numbers are Globally Unique IDentifiers (GUIDs).  In this case, they also appear in the "ARPcache" part of the registry.  "ARPcache" is a case of Microsoft doing what it wants ... in this case, misusing terminology. 

The standard use of ARP is in networking (Address Resolution Protocol) and there is an ARP cache that contains Ethernet MAC to IP address links. 

In the Windows XP registry, "ARPcache" refers to "Add or Remove Program cache, and it is where XP keeps the notoriously unreliable info about the size and use of programs that appears in Control Panel > Add or Remove Programs.
