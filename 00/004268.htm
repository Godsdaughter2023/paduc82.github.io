jr de pijper say:

This should not be any problem, just some careful work. To move your notebooks from their present location to a new location, let us say d:\notebooks, do the following:

1. Create a new folder d:\notebooks, the folder that will hold your notebooks.

2. Close all your notebooks in OneNote !!

3. Find the location where your notebooks are currently kept. You can find this in OneNote, in Options => Save & Backup, Default notebook location.

4. Each notebook corresponds to a folder (directory). Use your Windows Explorer to move each notebook folder to your new location. If you had a notebook called Business, you now have a folder d:\notebooks\business, with .one files inside it for each section.

5. Now you can simply open the notebook in OneNote by using File => Open => Open notebook => browse to the folder.

6. Done. If you want new notebooks to be created in the new location automatically, change the default location (see 3).
