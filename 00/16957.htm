If you’re not able to launch Internet Explorer program in Windows 10 using the search box, you can try out the following steps to launch Internet Explorer program.

  1. Open Start menu, click All apps and then navigate to Windows Accessories folder.
  2. Click on Windows Accessories folder. The Internet Explorer icon should appear here. If the Internet Explorer shortcut is showing up, you can resolve the issue by rebuilding the search box. To rebuild, proceed to step 7.
  3. If the Internet Explorer shortcut is not appearing under Windows Accessories folder, navigate to the following folder (Note: C is the drive letter of the drive where Windows 10 is installed.):
    • C:\Program Files (x86)\Internet Explorer (on 64-bit/x64 Windows 10)
    • C:\Program Files\Internet Explorer (on 32-bit/x86 Windows 10)
  4. Once you navigate to Internet Explorer folder, right-click on iexplore (.exe), click Send to, and then click Desktop (create shortcut) option to create a shortcut of iexplore on desktop. You can now click on iexplore shortcut on the desktop to launch Internet Explorer browser.
  5. Copy the newly created iexplore shortcut from desktop, open Run command box by pressing Windows key + R, paste the following path and then hit the Enter key to open Windows Accessories folder. When you see the confirmation dialog, click Continue button.
    • %ProgramData%\Microsoft\Windows\Start Menu\Programs\Accessories
  6. Paste the iexplore shortcut. Rename the shortcut from iexplore to Internet Explorer. The Internet Explorer shortcut should now appear in Start menu Windows Accessories folder.
  7. Manually rebuild the search index by pressing Windows key + R. Type control.exe srchadmin.dll, and click on OK.
  8. Click on the Advanced button.
  9. From the Advanced dialogue box, select Rebuild. Selecting Rebuild will begin the process of deleting, and then rebuilding your PC’s search index.

    hpholm say:

    Pictures on the phone can be uploaded from the Pictures hub (the gallery) to OneDrive through the Share option, where OneDrive is one of several options.
    Either tap and hold (long-press) one file and choose Share ... in the pop-up, or first select multiple (the 'list' icon at the bottom), and then use the '3 arrows in a circle' icon. Videos has to be uploaded one-by-one.

    The Share option may be implemented differently in other apps, but a long-press on a file or looking in the options menu (. . .) is always worth a try.

    Notice that files are only shared with yourself on OneDrive, except those in the pre-defined Public folder, and those you elect to share.

    Next, on the Mac, in Safari or another Internet browser, go to
    https://onedrive.live.com/
    And log on with the same Microsoft user name as used on the phone.
    Files can be downloaded to the Mac from here, and you can upload files from the Mac to OneDrive - which can then be accessed in the OneDrive app on the phone.

    The above is very inconvenient, so I suggest request assistance from an experienced Mac user to install and configure the deprecated USB transfer app and some updated drivers on the Mac. A relevant thread in Apple Discussions:
    https://discussions.apple.com/thread/7278042?tstart=0
