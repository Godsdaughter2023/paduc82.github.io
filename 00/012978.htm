zack barresse say:

Hi there,

There's a few problems you have to correct, but it can work. I'm the original builder of this template (thanks for the link Smitty!). There's a problem with some formulas and some named ranges. Let's start with the named ranges first.

Named Ranges

The date range formulas need adjusting. Since they're partially relative [references], you'll need to select the Dashboard sheet, then select cell D18 before you make these changes. This is important! If you don't, it won't work.

With D18 selected

  • Click on the Formulas ribbon, then Name Manager (or press CTRL+F3).
  • Select 'DtEnd'. In the 'Refers to' box at the bottom, change the reference in the COLUMN formula to Dashboard!A$1. Confirm with ENTER or click the check mark.
  • Select 'DtMiddle'. In the 'Refers to' box at the bottom, change the reference in the COLUMN formula to Dashboard!A$1. Confirm with ENTER or click the check mark.
  • Select 'DtStart'. In the 'Refers to' box at the bottom, change the reference in the COLUMN formula to Dashboard!A$1. Confirm with ENTER or click the check mark.

Formulas

You should still be on the Dashboard sheet, and cell D18 should still be selected. You need to change the formulas in cells D18, D19, D20, and D21. They're all slightly different.

  • D18 formula: =SUMIFS(Income[AMOUNT],Income[DATE],"<="&DtMiddle,Income[DATE],">="&DtStart)
  • D19 formula: =SUMIFS(Income[AMOUNT],Income[DATE],"<="&DtEnd,Income[DATE],">="&DtMiddle+1)
  • D20 formula: =SUMIFS(Expenditures[AMOUNT],Expenditures[DATE],"<="&DtMiddle,Expenditures[DATE],">="&DtStart)
  • D21 formula: =SUMIFS(Expenditures[AMOUNT],Expenditures[DATE],"<="&DtEnd,Expenditures[DATE],">="&DtMiddle+1)

Once you have those in, select all of those cells, D18:D21, and copy them (CTRL+C). Then select E18:O21, or the other cells, February through December. Paste Special Formulas here by clicking the Paste menu button on the Home tab. Not the button itself, but the little arrow underneath, which brings up the menu. Then click the second paste icon, which should be a clipboard with the function symbol on it (hovering over it should give you the pop-up description of "Formulas (F)". The keyboard shortcut for this would be ALT+E, S, F, ENTER. Press ESC when done to cancel copy mode.

That's it, you should be done and it should be working. I downloaded the same copy, made the same changes, and it works as expected for me. If it doesn't, please post back and let us know what is happening. 

The reason for the changes/errors in the templates can vary. Between the builders hands and the users download there are several hands and tests these templates go through. Usually it's in some QA person and they change something because "they have to" and they don't realize the implications, which ends up breaking something down the line. It's not a perfect system. I'll stay in contact with Microsoft to ensure this gets updated on their end so other users don't have this problem, and it's fixed when others download it. Thanks very much for bringing this to the attention of others!

I do remember building this particular spreadsheet. It was one of my favorite templates. I was really pleased with how it turned out. You should've seen the first "rough" build, it was ugly lol!

To empty the cache memory, refer to the steps.

How to Delete the Contents of the Temporary Internet Files Folder

http://support.microsoft.com/kb/260897

(Also applies to Internet Explorer 10)

 

Cache folder is used by Windows Internet Explorer to store webpage content on the computer hard disk for quick viewing. This cache lets Internet Explorer download only the content that has changed since you last viewed a webpage, instead of downloading all the content every time that the page is displayed.

Open Internet Explorer

Click Tools

Click Internet Options

Under the General tab, under the Browsing history heading, check the Delete browsing history on exit box

Click OK

Close Internet Explorer

Open Internet Explorer
