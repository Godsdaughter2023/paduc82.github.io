raju s das say:

Microsoft Office Groove was designed for document collaboration in teams with members who are regularly off-line or who do not share the same network security clearance

 For more information you could refer the link which talks about it

 http://office.microsoft.com/en-us/sharepoint-workspace-help/two-ways-to-use-groove-without-other-people-HA010237426.aspx#BM1

 If you do not want to use Microsoft Office Groove then you may disable it by following the steps mentioned below:

Click on Start> Control Panel> Program > Program and feature> highlight Microsoft Office 2007> click on the drop down next to Groove> select not available> continue> Close.

 And also disable Groove Monitor from startup folder by following the steps mentioned below:

1) Click Start> Type ‘msconfig’ without quotes in search program and files and hit enter.

2) The system Configuration Utility dialog box is displayed.
3) Under the Startup, uncheck the box next to Groove monitor.

4) Click OK.
 This would prompt you to restart the computer, click Restart to restart the computer.

This would disable your Groove folder synchronization.

chris peng msft say:

Hi Dayat,

 

One Office 365 tenant is associated with one portal. It is not feasible to automatically move the mailbox &OneDrive data from one tenant to another.

 

If you’d like to move the data manually, please follow the steps below:

 

To migrate a few mailboxes, we suggest you use PST files to export and import the mailbox data in the Outlook client. For more information, please refer to:
Export or backup email, contacts, and calendar to an Outlook .pst file
Import email, contacts, and calendar from an Outlook .pst file

 

If you have a large number of mailboxes, here is an article for your reference:
https://support.office.com/en-us/article/How-to-migrate-mailboxes-from-one-Office-365-tenant-to-another-65af7d77-3e79-44d4-9173-04fd991358b7?ui=en-US&rs=en-US&ad=US

 

To move the OneDrive Data, please refer to the following methods:

Information about manual migration of SharePoint Online content

Migrating File Shares to SharePoint Online

https://support.office.com/en-us/article/Overview-of-importing-PST-files-and-SharePoint-data-to-Office-365-ba688e0a-0fcb-4bd7-8e57-2b669564ea84?ui=en-US&rs=en-US&ad=US

 

Also we can use Open with Explorer feature or OneDrive for Business next generation sync client to make it:

Get started with the OneDrive for Business Next Generation Sync Client in Windows

How to use the "Open with Explorer" command to troubleshoot issues in SharePoint Online

 

In addition, please note that it’s not supported to share a single domain within multiple tenants.

 

Thanks,

Chris
