barb way say:

With Visio 2010 you do see a slight difference with cascading, but you also have a new option.  When you want to paste something, right-click in an area of the drawing and select Paste, and the objects will appear pasted into that region (approximately centered).  From there, you can drag or nudge the objects into a more precise location.

FYI - if you are pasting objects to a new drawing page, and do not have anything selected yet, the default paste starts at the same location that you copied the shapes from.  This is helpful when creating a new copy of a drawing page, for instance.

I've seen that problem before.  It's not specific to your phone.  It is common amongst a lot of Windows Mobile phones.  The default phone setup assumes that your WiFi connection goes to a private network and not to the internet.  You have to change you settings so that it understands that your WiFi connection *is* connected to the Internet.

 

  1. Click on [Start]
  2. Select [Settings]
  3. Select the [Connections] tab
  4. Select the [Connections] icon
  5. Select tje [Advanced] tab
  6. Select [Select Networks]
  7. Change the drop downs to what ever is associated with your WiFi connection  (probably [My Work Network]).

You'll find the same behaviour on other Windows Mobile phones, so these instructions will work on other phones including the HTC Diamond and HTC Touch Pro
