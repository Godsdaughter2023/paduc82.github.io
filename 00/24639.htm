joeu2004 say:

parabola50 wrote:

I'm probably doing something wrong or interpreting it incorrectly. Can anyone say what the issue is?

Two issues.

First and foremost, it appears that you created a Line chart.  As explained in KB 211967 (click here), trendline does not work with Line charts.  Actually, they do; but they assume the x-axis values are 1, 2, 3 etc.

Instead, you should use an XY chart, optionally with a Line subtype, as demonstrated below.

However, if you use the default precision of the constants in the formula on the left, you probably will not duplicate the trendline very well.

So secondly, you should display the trendline constants with maximum precision.  I prefer Scientific with 14 decimal places.  In this case, the formula becomes:

y = 1.24101325584000E+04x + 3.54956901100000E+09

Alternatively, use LINEST.  For example, select C1:D1 and array-enter the following formula (press ctrl+shift+Enter instead of just Enter):

=LINEST(B1:B4,A1:A4)

C1 is the slope, namely 12410.1325584000.  D1 is the intercept, namely 3549569011.

If you're using Windows XP (SP2 or higher) you might want to look at the solution suggested here:

 

http://www.vinsvision.com/Articles/tabid/66/EntryID/13/Default.aspx 

 

I had very good results after replacing Toshiba Bluetooth stack with the drivers in this patch. (You can have this driver in your laptop even if it's another brand, it's a widely used piece of software.)

 

The method I used was as follows:

 

1) Uninstall all Nokia software from PC

2) Uninstall Toshiba Bluetooth stack (from Add/Remove Programs Control Panel in Windows)

3) Reboot PC

4) At start-up, Windows will find new hardware and suggest installing drivers for Bluetooth adapter; IGNORE THIS and don't search drivers or install anything yet

5) Install the MSI-package from the link above

6) Let Windows install all drivers

7) Pair your Nokia device with PC using Windows Bluetooth software; if this works, all Bluetooth drivers were successfully installed to PC

7) Install latest Nokia PC Suite

8) Start using (hopefully) better working Bluetooth connections

 

For me at least, so far so good! Many sources quote Toshiba BT stack as "incompatible" with PC Suite, so this might be worth a try, if you're stuck and currently use those drivers.
