rohn007 say:
Unless you have a clearly identified need to use ODF, stick with the "XML" fomats.

XML format is the generic name for the new "x" file formats introduced with Office 2007: DOCX, XLSX, PPTX etc.  They are the "native" format for Office 2010. The XML formats are owned by MS, but they also adhere (there is some debate about that, lets say loosely) to open standards. They are a defacto standard just because so many people use them. BUT, the formats can change at any time, ie 2003 DOC is different than 2007 DOCX, which is slightly different than 2010 DOCX, which is also slightly different than

ODF is an 'Open Source" file format that is NOT owned by MS. The advantage with this format is that anyone can build software that uses them to read and write files.  ODF formats also change, but much more slowly since the standard is controlled by a large committee.

XPS is a printing format that is approximately equivalent to PDF.  Don't bother using XPS, no one does.

Yes, you can change file formats at any time using File > Save AS.
