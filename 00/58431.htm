How to VIEW files on internet (e.g. pdf) without saving them

cmdrkeene say:

The ability to view PDF's within the browser is accomplished by a software program known as a Browser Helper Object.  This is installed during the Adobe Reader installation.  Make sure you're running the newest version of Adobe Reader.  If so, check to make sure your browser is set to allow extensions and add-ins.  If you're using another browser besides IE, also make sure you install the plug-in or add-on for PDF files for that browser.
