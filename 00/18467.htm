jay freedman say:
You can hold down the Ctrl key while clicking the edge of each text box. This doesn't work so well if the boxes are spread out over a large document. Another approach is to click Select on the Home ribbon and click Selection Pane. That will show a list of all the objects on the current page, and you can Ctrl+click the text box names in the list; unfortunately, this only works page by page.

If you need to reformat a large number of text boxes throughout a large document, and especially for lots of documents, your best method is a macro. Start with something like this (see http://www.gmayor.com/installing_macro.htm if needed) and set values and additional properties as needed:

Sub FormatTextBoxes()
    Dim Shp As Shape
   
    For Each Shp In ActiveDocument.Shapes
        If Shp.Type = msoTextBox Then
            With Shp
                .Fill.ForeColor = wdColorRose
                .Shadow.Type = msoShadow6
                With .TextFrame.TextRange
                    .Font.Name = "Cambria"
                    .Font.Size = 12
                    .Italic = True
                End With
            End With
        End If
    Next
End Sub

vanderghast say:

In addition to the proposed subquery, you can use a join, which I will bring to the where clause so you could still use the graphical query editor:

 

SELECT a.shipname, a.dateTimeStamp, MAX(b.dateTimeStamp) AS previousDate

FROM ships AS a, ships AS b

WHERE a.shipName = b.shipName AND a.dateTimeStamp > b.dateTimeStamp

GROUP BY a.shipName, a.dateTimeStamp

 

 

Note that the table ships is refered twice, kind of having two fingers, a and b, over the same list, ships, except that once the finger a is stopped over a given shipname, and a given dateTimeStamp value, finger can only 'stops' over a field satisfying the WHERE clause.

 

There is a problem with this query, the first time a given ship is entered into your table, there is NO previous occurence of it, so finger "b" won't find any record and the result will be without that ship mentionned. To correct that, we HAVE TO use a join, instead of the WHERE clause:

 

SELECT a.shipname, a.dateTimeStamp, MAX(b.dateTimeStamp) AS previousDate

FROM ships AS a LEFT JOIN ships AS b

     ON a.shipName = b.shipName AND a.dateTimeStamp > b.dateTimeStamp

GROUP BY a.shipName, a.dateTimeStamp

 

is the only modification required. In the particular case that a.dateTimeStamp is the earliest date a ship appears in the database, MAX(b.dateTimeStamp), which generally return the previous date given the ON clause (which was our previous WHERE clause) will be NULL, but the record will be kept, instead of being disregarded as it was, with the WHERE clause.

doctorpaul say:

How can I copy a formula into the next row that follows the pattern of previous calculations

Hi

I have a workbook where a long table of data in one sheet feeds into a summary table on the next sheet. 16 rows of data are calculated into one answer, and then the next dataset starts. What I am trying to do is set up my summary sheet so that when I drag down the formulae into the next row, it follows the pattern that the calculation occurs for a set of 16.

E.g. C10 contains the referece ='Data'!N133, then C11 contains ='Data'!N149 etc.

If I drag this down the next cell will contain ='Data'!N150, where what I want is ='Data'!N165

I am sure that there must be a way of doing this, but I just can't figure it out, or even how to phrase a web search to find the solution!

Many thanks in advance,

Paul.

shane devenshire say:

Here is a totally different approach suppose you have a formula have =Data!N133 in C10 and you want it to copy down any distance but change to =Data!N149 in the next cell and then to =Data!N165 in C12 and so on. 

1. Highlight the range C10:C25  (this assumes you have nothing else in column C yet. 

2. Drag the fill handle down as far as necessary, in this example let's say to C74.  This will create the desired formulas but with a lot of blank rows between each.

3.  Highlight all the rows C10:C74 and press F5, Special, Blanks, OK

4.  Press Ctrl+- (control minus) and choose Shift cells up.  You will now have the desired formulas in the desired cells.

------------------

This technique generalizes to any spacing you want.  To find out the spacing drag the start by entering the formula in C10 and using the fill handle to drag that one cell down.  Check the formulas to see which row contains the second one you want.  In our case it would be C26.  Then delete all the formulas except the first one. Hightlight that formula and down to the cell directly above the one that held the correct repeat, here C25.  Proceed as above.

If this answer helps, please click the Vote as Helpful button. Cheers, Shane Devenshire
