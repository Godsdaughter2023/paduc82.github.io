jay freedman say:

If the file really is a .txt file (plain text, no formatting), then you can use the File > Open command to open it in either WordPad or Word, and go to File > Save As. Then:

  • In WordPad, choose "Office Open XML Document", which will save a copy of the document as a .docx file. (The Windows 10 version of WordPad doesn't offer the older .doc format. You can later open a .docx file in Word and save it as a .doc file if you really need that.)
  • In Word, click the dropdown that initially says "Plain Text (*.txt)" and choose either "Word Document (*.docx)" or "Word 97-2003 Document (*.doc)".

jay freedman say:

Unfortunately, you can't easily do anything about the field shading option, which is a per-user setting.

The only alternative while continuing to use fields is to include the following macro in your document, which you would have to save as a macro-enabled (*.docm) document; and the recipients would have to store the document in a trusted location or otherwise enable the macro.

Sub AutoOpen()
    ActiveWindow.View.FieldShading = wdFieldShadingAlways
End Sub

Unfortunately, that would leave the option set to "Always" for any other documents they open until they go to the Options dialog and change it. Fixing that would involve more code. You don't want to go there!

One formula you have to give manually, then you have to drag or copy it down, then it will automatically give you corresponding results.

Say you have data in column A & B like this screen shot,  then you have to give this formula

=B3-B2

at C3 and then take you mouse to right/down corner of the cell, it will show a sign of '+', just then you have to press mouse and drag it down. The formula at C3 will be copied down to C4,C5,.......  and you will get corresponding result.

In case you want percentage growth then in the same way you have to give this formula at D3

=B3/B2-1 ( change cell to percentage format)      and then drag it down.
