rich007a say:

First you need to enter your required X values in a column.  e.g. enter 1 in A2, 2 in A3, etc.  (You can drag the 1 in A2 down while holding the Ctrl key).

Then you need to change your formula like this, and enter it in cell B2:

From:      y = -1E-08x6 + 1E-06x5 - 3E-05x4 + 0.0002x3 - 0.0003x2 - 0.0012x + 0.0324

To:

= -1E-08*A2^6 + 1E-06*A2^5 - 3E-05*A2^4 + 0.0002*A2^3 - 0.0003*A2^2 - 0.0012*A2 + 0.0324

i.e. you replace each "x" in your equation with *A2^   ("...times A2 to the power of..."), except for the last x which is to the power of 1, and so we don't need to bother with the ^.

Now drag B2 down as far as your x data.

Cheers
Rich

MS sells "consumer" and "business" class Office 365 bundles.  The business class subscriptions (and licenses) are more expensive, so they have more features and applications than the consumer versions.

Don't worry, you are not confused by MS offerings. This is a common problem.

#1

Office 365 Personal includes the following applications:

    Word

    Microsoft Excel

    Microsoft PowerPoint

    Microsoft Outlook

    Microsoft Access (PC only)

    Microsoft Publisher (PC only)

365 includes 1TB of space on OneDrive. That is up from the 5GB of space available for free with a MS email account.

365 also includes 60 minutes of "Skype" time every month.

 

Using 365 personal you can have active installations on up to 5 computers at the same time. This includes Windows, Macs, smartphones, Android tablets.  The applications on the various operating systems will not support all of the features you see in Windows.  You just have to live with that.  You can control the computers Office is active on using the online "myaccount" page.

MS Flow is a "Business" class application it is not included in your Personal subscription

There is a free version of To Do you can get from the MS Store:

https://www.microsoft.com/en-us/p/microsoft-to-do-list-task-reminder/9nblggh5r558?rtc=1&activetab=pivot:overviewtab

This has fewer features than are available in the version that is included with business class Office 365 subscriptions.

#2

In general, the Online versions of the Office applications are free. Since they are free, you get what you paid for. In this case "dumbed down" versions of the paid for Office applications. They can open and edit documents created in the full versions, but do not support all of the features in the paid for versions.  So in general you use the paid for version. If you happen to be away from a computer with the full version installed, you can use the free version on any computer to view and edit documents. But you won't be able to create using all of the features in the paid versions.

#3

OneNote online vs OneNote UWP (installed for Windows 10) and OneNote 2016 (part of Office 2016/2019)?

Yes that is confusing.   This is typical of the "MS Experience".  MS has a bad habit of using the same name for different applications over time, sometimes at the same time depending on what you have paid for.  Another example is "OneDrive", the online space granted with Office 365.  They have the same name, and many features that look the same, but the underlying technology is very different ... <sigh>

OneNote Online: see comment above, ie dumbed down

A couple of months ago MS "improved" <sarcasm> OneNote out of Office and into Windows. It means that when you install Office now, even though it used to include OneNote, it is no longer installed.  OneNote that came with Office was replaced by the "new & improved" <sarcasm again> OneNote UWP that is now part of Windows 10.

The thing is, the new & improved version of OneNote UWP does not include all the features that the Office version had, specifically it can only save Notebooks on OneDrive. It cannot edit note files created in the old version that are still saved on the local hard drive. There is a way to load those old file into the new version and save the in onedrive.  MS says that it will be adding more features to OneNote UWP, eventually.

We also still have the option of installing a standalone Office version of OneNote 2016. But in your case, I would suggest avoiding that so you don't get used to features that will be going away in a couple of years.

#4 Outlook vs Windows Email

Windows Mail / Email is a very simple <cr>applet

Outlook on Windows has a lot more features than Windows Mail. So in theory it is much better. 

Other than that I'll not comment on Outlook. I don't use it myself, I've never been a fan of it. Instead I just use the online/web user to Outlook.com and Gmail.com.
