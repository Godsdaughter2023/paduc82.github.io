how to indent a single line not the whole document - Microsoft Community

isabellamay say:

how to indent a single line not the whole document

How do i indent a single line on mac word?
if i press tab or indent it does the entire document. GRR!
john mcghie say:
How do i indent a single line on mac word?
if i press tab or indent it does the entire document. GRR!

You need to de-select "Automatically update" in your Style definition.

A "Style" is a collection of formatting: in this case, all of the formatting that applies to a paragraph.  That specific collection of formatting has been set to "Automatically update".

What that means is that if you apply an indent to one of the paragraphs formatted with that style, the style definition updates so ALL paragraphs with that style applied will all change together.

To discover which style, select a paragraph and use Format>Style to open the style dialog and see which style is selected.  That's the name of the style in use.

Now click "Modify" you will see a checkbox at the bottom right named "Automatically update" -- uncheck that, and OK your way out.

Now:  You want a "first line" indent, not a "whole paragraph" indent.  So this time, select the paragraph and use Format>Paragraph to access just the paragraph properties, and just for that paragraph (instead of the style, which applies to the whole document).

To get to the First Line indent, drop-down the list under "Special" and choose "First Line".  Then specify how much of an indent you want in the box to the right of it (usually, 1 cm is fine).  If you wanted all paragraphs of that kind to look like that, you would instead use Format>Style to make the same change in the style rather than to just the one paragraph.

Hope this helps

scottgem say:
If you are using an Access ACCDB file as the back end to your vb.net file you don't need Access at all. You create the connection string to the file within your vb.net program.

The only time you would need the Access runtime is if you are using Access as your front end.

When deploying Access using the runtime, you simply need to install the runtime on the machine. The Packaging wizard can bundle the runtime with your app. But all you need to do is launch your Access file, the runtime will then load.
