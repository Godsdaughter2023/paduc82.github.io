I need to update my new phone but i dont have enough space in my SSD - Microsoft Community

amador-mc say:

I need to update my new phone but i dont have enough space in my SSD that I use for my OS. Is there a way that I can bacup to my many other drives on my computer

Samsung Focus

 

entegy say:

You can use something called a symbolic link. This makes the system think the file is in one location, but it's physically elsewhere. You can read this article to learn about symbolic links.

This has to be done from an administrative Command Prompt. So find "Command Prompt" In your Start>All Programs>Accessories folder and right-click "Command Prompt" and choose Run As Administrator.

 

The backup is stored at C:\Users\<username>\AppData\Local\Microsoft\Windows Phone Update Replace <username> with your Windows username. So move the Windows Phone Update folder to your other location, then run the mklink command

Your end command will look something like this: mklink /J "C:\Users\<username>\AppData\Local\Microsoft\Windows Phone Update" "D:\Windows Phone Update"

Note the quotation marks around the file paths. Those cannot be forgotten!
