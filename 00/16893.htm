If you've written an ActiveX DLL in C++, you should be able to add a reference to it (through Tools | References) and be able to interact with it using VBA.

Insiders on Slow and Fast Ring do not receive updates on any set day.  They only receive new builds when the dev team determines one is stable enough for distribution to us.  Their stated goal is once a week but not if no suitable build is available.  That has been the case most of this month.

You need to check this forum and the Announcements in the FeedBack Hub in order to keep up with the program.  There has been a three week hiatus in new builds due to blocking bugs that the dev team has had to chase down before they reached an acceptable build for flighting this week.
