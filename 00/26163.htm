guessed say:

This macro should get you started

Sub AddATextBox()
  Dim aShp As Shape
  Set aShp = ActiveDocument.Shapes.AddShape(Type:=msoShapeRectangle, Left:=100, Top:=100, _
              Width:=25, Height:=25, Anchor:=Selection.Paragraphs(1).Range)
  With aShp
    .TextFrame.TextRange = "A"
    .TextFrame.TextRange.Font.ColorIndex = wdBlack
    .Fill.Visible = msoFalse
    .RelativeHorizontalPosition = wdRelativeHorizontalPositionMargin
    .RelativeVerticalPosition = wdRelativeVerticalPositionParagraph
    .Left = 0
    .Top = 0
  End With
End Sub

Outlook.pst is a storage file and has nothing to do with a new Outlook profile.  The correct process is:

1. Go to "Control Panel > Mail > Show Profiles"

2. Add a new profile.

3. Double click this new profile and add your email account(s) to it MINUS this Exchange Server account.

4. in the "When starting Outlook, use this profile" section, make sure the "Always use this profile" radio button is selected and make sure the new profile you just added is selected in the drop-down list right below the button.

Is there a way to put a variable into the "LinkedCell" field such that whichever cell I have active it puts the output to that cell?

Hi,

scenario is as follows:

List Range in Active Sheet in cells A1:A30

(add combobox in active sheet)

if you select a cell in column C (in order to add a combobox in that cell)

then linked cell is 2 cells to the right, in column E

try this code...

Sub Add_ActiveX_ComboBox_01()
'Dec 20, 2014

Dim rng As Range
Set rng = [A1:A30]
Set r = Selection
Dim obj As Object
Set obj = ActiveSheet.OLEObjects.Add(ClassType:="Forms.ComboBox.1", Link:=False, _
DisplayAsIcon:=False, Left:=r.Left, Top:=r.Top, Width:=r.Width, Height:=r.Height)
With obj
.Placement = xlMoveAndSize
.LinkedCell = r.Offset(, 2).Address
.ListFillRange = rng.Address
.Object.ListRows = rng.Rows.Count
.Object.Value = rng.Item(1)
End With
End Sub

also,

if already exists a ComboBox, in active sheet

and you want to change or add a Linked Cell

try this,

select a Cell and run this code

Sub Selection_Add_LinkedCell()
'Dec 20, 2014
Dim r As Range
Set r = Selection
Dim rng As Range
Set rng = [A1:A30]
If MsgBox("select ONE Cell ?", vbOKCancel) = vbCancel Then Exit Sub
For Each obj In ActiveSheet.OLEObjects
If obj.Name Like "ComboBox*" Then
With obj
.LinkedCell = Selection.Address
.Object.Value = rng.Item(1)
End With
Exit Sub
End If
Next
End Sub
