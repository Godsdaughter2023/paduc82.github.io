Toggle Checkbox Colors - Microsoft Community

phil hageman say:

Toggle Checkbox Colors

Is there a way (macro) to make the default background color of a checkbox (inside the square itself) to R255, G255, B153 (light yellow), and then have it toggle to R0, G0, B0 (white) when the user clicks the box (check mark appears)?
jay freedman say:

Hi Phil,

Sorry, I see that we've run into a misunderstanding about what kind of checkbox you're using. The fact that you have a CheckBox2_Click() procedure indicates that you've inserted ActiveX checkboxes from the Control Toolbox, not from the Forms toolbar, so nothing dealing with FormField boxes will do anything for you. Only FormFields have exit macros; ActiveX boxes use a completely different section of VBA.

The good news is that the code behind an ActiveX box will let you change the color immediately when the user clicks the box. The not-so-good news is that the color is applied to the entire background of the control, including the caption (if there is one) -- but not to the little square where the checkmark goes! It would probably be best to have no caption as part of the control, and just put plain text as a label to one side or the other.

The code you need (in the ThisDocument module of the document or the template on which the document is based) is like this:

Private Sub CheckBox1_Click()
    CheckboxColor CB:=CheckBox1
End Sub

Private Sub CheckBox2_Click()
    CheckboxColor CB:=CheckBox2
End Sub

' ... and similarly for the other checkboxes in the document

Private Sub CheckboxColor(CB As Object)
    If CB.Value Then
        ' checked, so white background
        CB.BackColor = RGB(255, 255, 255)
    Else
        ' unchecked, so yellow background
        CB.BackColor = RGB(255, 255, 153)
    End If
End Sub

####

For Word, you can make a template containing code, and save it in Word's Startup folder (by default, C:\Documents and Settings\<your name>\Application Data\Microsoft\Word\Startup). Any code in that template will be available in all documents. I think the _Click() procedure for each checkbox still has to be in the document or its base template, though, because they're specific to the controls in the document.
