68GB is rather small for day to day operation of Windows. If C: and D: are 2 partitions of the same hard disk as D: is empty it is possible to grow the size of C:

 

1st use Disk manager to verify that both partitions are on the same drive. While in D: press the Alt key to display the extended menus then select Tools - Folder Options - View and select show hidden files. OK out and check the contents of D: and confirm it is empty. If it is, or if there are only a few files copy them off to an external disk, then you can delete the partition also in DIsk Manager

 

Once deleted right click on the C: partition and chose extend partition. The needs for keeping a C: and D: drive are less of a concern and especially as you have not been using the D: drive you may want to consider extending it to the whole drive as C: but you have to decide.

 

Alternatively if the C: partition is on its own drive (possibly an SSD) it is possible to move some of the folders that make up your user profile on to the D:

 

On the D: drive make a folder Users and in that one for your Username. It is not essential to replicate the folder structure from the C: but it then makes sense.

 

Click on the Start button then your User name at the top. This will show all the folders that make up your User profile. Right click on each one and select properties. Some of the folders will have a Location tab. In that tab it will show the current location (C:\Users\username) with a Move button. Click on that and the dialogue will take you through moving that folder to D:\Users\username. It will move the files and folders and reset all the pointers correctly so that although you now have 2 separate user profile folders, one on C: the other on D: the profile itself will still be a single entity.

 

Once that is done you can consider re-installing your applications and see if they have the option of selecting where they are installed. Create an Applications folder also on the D: drive and install them in there. They cannot just be moved to the D: drive they have to be installed in the right location.

Yes, you can, depending on the make and model of your computer.

RAM is a physical hardware component of your computer, it is not a software feature.

If your computer needs more RAM, you will need to purchase it and install it.

You can use the following to find out what type of RAM in your computer and how much it supports.

Crucial System Scanner | Crucial.com

Windows 10 64 bit comes in handy when you need to address at least 4 GBs or more of RAM. Windows 10 32 bit can utilize up to 3.2 GBs of RAM. Because the memory address space is much larger for 64 bit Windows, that means, you need twice as much memory than 32 bit Windows to accomplish some of the same task, but you are able to do so much more, you can have more applications open, do things like run an Antivirus scan in the background without it affecting your system performance. Windows 10 64 bit is more secure too, malicious code cannot easily infiltrate it, drivers are more reliable since they must be signed before they can work with 64 bit Windows 10.

As for compatibility, you will need 64 bit device drivers for any hardware devices you might have. Also, there is no 16 bit subsystem in Windows 10 64 bit, which means, your applications must be 32 bit only, not 16 bit installer or uninstallers. Also, if you decide to move to Windows 10 64 bit in the future, there is no upgrade path from 32 bit Windows, clean install only. Because you have 4 GBs of RAM in your system, its best you work with 64 bit Windows 10, since you would be underutilizing the system with 32 bit Windows 10.

andre da costa say:
Depending on how much memory is currently installed in your system, you should consider installing in matching pairs for best performace. So if you have 4 GBs of RAM each a single stick of 2 GBs, you should install additional RAM in 2 GBs if there are available banks in on the motherboard. If you don't have enough DIMM slots available, you might have to discard some or all of the installed RAM for larger amounts.

No data will be lost by installing more RAM, just make sure you first turn off your computer and disconnect it from the power source.

RAM is considered as volatile, when you turn off your computer, your data is moved to non-volatile memory (this would be your hard disk). When you start your computer, your data is loaded from the non-volatile memory in main memory.
