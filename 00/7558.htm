Dim lngPidlFound As Long
Dim lngFolderFound As Long
Dim lngPidl As Long
Dim strPath As String

strPath = Space(MAX_PATH)
lngPidlFound = SHGetSpecialFolderLocation(0, lngFolder, lngPidl)

    lngFolderFound = SHGetPathFromIDList(lngPidl, strPath)

    End If
End If
CoTaskMemFree lngPidl
End Function

According to your demand, I suggest you to share  contacts lists via Shared Mailbox (if you have an Exchange server account):

  2. Create a security group and add the people you want to share to: (Recipients->Groups->New+ ->Security group)
  3. Create a new shared mailbox :(Recipients->Shared->Add).
  4. Assign the security group the “Full Access” permission: (Recipients->Shared->Choose the shared mailbox you just added above, click “Edit” icon or double-click->In Mailbox Delegation, select Full Access->Add).

  6. Contact list cannot be accessed via Outlook Web App.
  7. For more details, please check: Sharing calendar and contacts in Office 365

  1. Open Outlook client.
  2. Click “People”->Choose one of your contact folder and right-click it->Click “Properties”.
  3. In “Contacts Properties” dialogue, please select “Permissions” tab.
  4. In “Name”->Click ”Default”.
  5. Under permissions, find “Permission Level”->Choose one permission you want to assign to others.
