bill manville say:

You might mean

A, B, A, C, D, D, C = 18/7 = 2.57

 

If the range of cells containing the grades is R (e.g. A1:G1)  and the values of the letters are always 1,2,3 etc then

=SUM(IF(ISBLANK(R),0,(CODE(R)-CODE("A")+1)))/COUNTA(R)

entered as an array formula using Ctrl+Shift+Enter

 

This allows for blanks in R contributing nothing to the average

ronrosenfeld say: You can assign those numbers with a formula:  =CODE(cell_ref)-64
If the numbers might sometimes be lower case, then use =CODE(UPPER(cell_ref))-64

You can do this for a range, and compensate for blank cells:  e.g:

These formulas must be **array-entered**:


To get the Sum of the grades:
=SUM(IFERROR(CODE(UPPER(A1:G1)),64)-64)

and, assuming the "6" in your above series of seven grades is a typo, you could use

=AVERAGE(IFERROR(CODE(UPPER(A1:G1)),64)-64)

to get the average.

The range you are checking can be any size (or shape) using this method.
If there happen to be more than one letter in a cell, only the first letter will be considered.
