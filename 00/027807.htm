austinapplequist say:

Dimension Accuracy in Visio 2016

I have a user that would like to change the default accuracy of dimensions in Visio 2016. He would like it to default to the whole inches, no fractions necessary. The typical use of Visio in this environment is creating room layouts and cabinet drawings, the woodshop here nine times out of ten only needs measurements in whole inches.

Is there a way to do this? Thanks!

winni liang -msft say:

Hi, 

Please try to resize your drawing line to fit the whole inches instead just change the measure number.

Generally, the Measure tool shape would help to measure the length or scale for the drawing.  Click More Shapes > Visio Extras > Drawing Tool Shapes. Drag Measure Tool to the drawing. Glue one endpoint of the Measure tool shape to one point you want to measure—the endpoint turns green to indicate it is glued. Then drag the other end to the other point. The measure tool shows exactly how far apart its end points are.

Try to use the Measure tool to resize the drawing to the whole inches.

Regards,

Winni
