¡firedog say:

Server Response: -ERR [SYS/TEMP] problem retrieving message.
Windows Live Mail Error ID: 0x800420CD

This is a temporary system error at the server, because there is probably a message in the mailbox at the server that Windows Live Mail can't or won't download for some reason - it's too big, it contains malicious code or it's just plain dodgy.

To clear the blockage, log in to the account in webmail and start by emptying the folders for spam and deleted items. Then create a new folder or folders and move all of the messages out of the inbox and into the new folder. When the inbox is empty, select the account from the dropdown list on the bottom half of the send/receive button in Windows Live Mail. No new email messages should eventually appear in the status bar.

Back in webmail, move half of the messages that haven't yet been downloaded by Windows Live Mail back into the inbox and again try send/receive for this account in Windows Live Mail. If the fault recurs, we now know which half of the messages the bad one is in. Repeat the process, taking half of the messages each time until the dodgy message can be identified and dealt with in webmail.

In future, don't use the webmail inbox as a storage folder. Move any messages for keeping on the server into archive folders, leaving only a few days' worth of messages in the inbox at any one time. Empty the spam and deleted folders regularly.

michelle vil say:

Hello,

If you want to roll back a device driver to a previous version, we recommend following the steps below:

To roll back to a previous version of a driver:

  1. Open Device Manager.

  2. Right-click the device for which you want to roll back to the previous version of the driver, and then click Properties.

  3. Click the Drivers tab.

  4. Click Roll Back Driver.

  5. In the Driver Package rollback dialog box, click Yes.

By the way, the easiest way to prevent Windows 10 from automatically downloading updates is to use a metered Internet connection -- one of the features of limiting your data is that Windows 10 won't force-download OS updates. Some networks, such as cell networks, are metered by default, but you can also meter your Wi-Fi network by going to Settings > Network & Internet > Wi-Fi > Advanced options and turning Set metered connection on.

Note that you can still download updates on a metered connection, you'll just have to do it manually by going to Settings > Windows Update > Download.

If you have any other questions or concerns, feel free to share them with us here. Your input is very much welcome in our Community.
