How to make un-editable sections in a word template - Microsoft Community

lisa_817 say:

How to make un-editable sections in a word template

I'm trying to make a template for proposals and I want certain parts of the document to always stay the same no matter what. Several people will be editing it and formatting tends to change because people copy and paste from different documents. I want the headers and footers to stay the same, certain images and the text font color. How do I do that? 

Thank you!

daniel g. grau say:

Document sections let you redefine a number of attributes: page size, page orientation, number of columns, new headers and footers and probably something else that escapes my memory.

When you insert a section break at the cursor position, all those the attributes are inherited from the current section. For instance, if you document has first page special and a certain header, after inserting a section break you'll have two identical sections, plus their headers and footers will be linked (changing one changes the other).

Those inherited attributes can only be seen if there are enough pages. A one-page section will only show its "different first page" until more pages are added.

If you insert a "Section Break (Continuous)," something similar will happen: it will inherit all its headers and footers from the current section, except that they won't be visible until the continuous section spans more than one page. Then they appear and one wonders "Where did this come from?"

What to do?

  1. Work on a final copy, so that there won't be repagination from added text to cause the above problems from added pages.
  2. I think that all you need to do is insert one "Section Break (Continuous)" at the points where you change protection (either enabling it or disabling it). If you select text before you do that, it will be replaced by the section break, just like it would if you selected it and typed a Carriage Return. So just place the cursor with no selection.
  3. Adding "Section Break (Continuous)" should not alter pagination.
  4. Make a note of the section numbers you need to protect. Just place the cursor somewhere in that section and note it's section number at the bottom of the document's window. If you don't see the section number, make sure the Status bar is enabled in Word > Preferences > View.

What does Windows as a Service entail?

Since the introduction of Windows Update as part of Windows, Microsoft has delivered updates as way to keep Windows updated and secure. This has always been the way the company keeps commercial versions of Windows on the market current.

With Windows 10, Microsoft is making significant changes to this model by delivering new methods of servicing Windows. Taking into account we now live in a mobile, constantly connected world, where threats happen all the time. Part of the solution to this is to keep users current by providing the latest updates as soon available in addition to delivering features as soon as they ready too.

Microsoft will do this using three types of service branches for Windows 10:

  • Security Updates, Features and Fixes are automatically applied.
  • There is no option to delay or customize these updates.
  • Windows 10 Home
  • CBB includes the requirements of the Current Branch, but also provides the option of customizing when and which Security Updates, Features and Fixes are applied, similar to how Windows Update works today in current versions of Windows. 
  • Updates cannot be deferred indefinitely.
  • Windows Updates can be managed using enterprise management tools such as Windows Update for Business and/ WSUS.
  • Windows 10 Pro
  • Windows 10 Enterprise
  • Windows 10 Education
  • Only available to volume license customers running Windows 10 Enterprise.
  • Flexible options for managing Windows Updates.
  • Windows Updates can be customized to only accept Security Fixes.
