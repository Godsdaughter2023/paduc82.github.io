Adding an unnumbered cover sheet. - Microsoft Community

cynthiaevans say:

Adding an unnumbered cover sheet.

Looking for an answer concerning adding a cover sheet.  I have a completed document that contains a numbered cover-page with a running head.  I now need to add a non-numbered coversheet to the top of the document.  How do I suppress page numbering and get Word to recognize the second page as actually the first page of numbering and heading.  Tired to copy and paste the document after the cover sheet but word does not transfer the heading and page numbering to the pasted document.

jay freedman say:
If you use the Insert > Cover Page button to insert one of the pre-built cover pages, it places a Next Page section break between the cover page and the original page 1; the cover page is numbered as page 0, and it has no header or footer.

If you're creating your own cover page, you need these steps to make it behave the same way:
  • If you inserted a page break (Ctrl+Enter) between your cover page and your first page, delete it and replace it with a Next Page section break (Page Layout > Breaks > Next Page).
  • While the cursor is in the first page of the document body (now page 2), double-click in the header. You'll see a tag attached to the dashed line, saying "Header - Section 2". When the Header & Footer Tools tab appears on the ribbon, click the Link to Previous button to turn it off. Then click the Go to Footer button, and again turn off the Link to Previous button.
  • Still on the Header & Footer Tools tab, click the Previous button to go to the footer of the cover page (its tag says "Section 1"). Select any text that's there and delete it.  Then click the Go To Header button, and delete the text from the Section 1 header.
  • Finally, on the Header & Footer Tools tab, click the Page Number button near the left end and click Format Page Numbers. In the dialog that appears, click the Start At option and type 0 (that's a zero) in the box. Click OK. Then close the header.
