When you "formatted the drive", are you referring to the OS partition, and did you leave the system reserved partition in place? If so, you may have left in place a problem that the later upgrade install saw, causing it to fail.

The preferred method of clean install is to delete (rather than format) both the OS partition and the system reserve partition, so that their space is combined into one unallocated space, then, use the "new" item that becomes available to make a new partition for the OS to install into, and agree for the Windows installer to add a system reserved partition.

Hi Jon,

Firstly, I'd like to introduce OneDrive for Business. It's personal site in the Office portal, users can upload files to the library in this personal sites and share the files to others. We can use the OneDrive for Business sync client to sync these cloud files to our local drive. After syncing, the files in the cloud and synced folder are connected, all the changes will synced to each side. OneDrive for Business personal site can be used to save the files for a user himself.

Team site library is normally used for the organization. Members in the tenant who have permission in this library can work together with the files in it. In addition the cloud files in team site library can also be synced to local drive with OneDrive for Business sync client.  
Note: Permission is very important in SharePoint Online and SharePoint Online team site. To learn more, see Understanding permission levels.

For your requirement, you can create a workflow for a list with SharePoint Designer 2013. Every step, you can assign task to specific user and the status for the workflow will display in the list (the task can be approval, review and so on).
