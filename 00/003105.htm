karl timmermans [outlook mvp 2012-2018] say:

In reviewing my original response, I should have better clarified the anwser. You cannot remove user-defined fields at the item level "in bulk". ContactGenie Toolkit provides the facility to remove these user-defined fields (otherwise referred to as "orphan" user-defined fields) in bulk from individual contact items IF those fields exist.

The above presumes you are referring to contacts that are NOT created via a custom form but contacts that are created using the standard message class of <IPM.Contact.>
