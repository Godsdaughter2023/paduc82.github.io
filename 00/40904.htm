Does any one have a fix I could use (and please explain in the simplest of ways for me!)

Thanks a lot.

a. user say:

"Out of range" in this context means that the signal your graphics card or chip is sending to the monitor is beyond the ability of the monitor to display. You typically see this message when you try to use a resolution or frequency that is beyond the monitor's capability.

It's a common error for people like ... uh ... somebody I know, when moving from a CRT to a digital monitor. A digital monitor doesn't need to use a high frequency signal; whatever the graphics card/chip sends (typically 60 Hz) is correct.

lemp say:
I have an old entry in my notes that should cover this.  If it doesn't, post back: it's not difficult to re-create the All Programs menu.

Hidden entries in Start > All Programs

The Start > All Programs menu is a combination of two folders (directories) on your hard drive: 

     C:\Documents and Settings\All Users\Start Menu
     C:\Documents and Settings\<username>\Start Menu

If the subdirectories in either or both of the above is given the "hidden" attribute, you won't see the corresponding entry when you click Start > All Programs.  To check and fix this, right-click on the Start button and select either "Open" or "Open All Users" (you should do both before you're through).  In the Windows Explorer window that opens, click Tools > Folder Options > View.  Select the button to "Show hidden files and folders" and UNcheck the box to "Hide protected operating system files (recommended)."  OK your way out. 

Double-click the Programs folder.  Any folder that's been hidden will appear "faded out."  Right-click each hidden folder and select Properties.  At the bottom of the General tab, UNcheck the "Hidden" box.  Click OK.  Select "Apply changes to this folder, subfolders, and files." Click OK.  Do this for all hidden folders.  When you are done, go back to Tools > Folder Options > View and reset the changes you made earlier.

Repeat the above by right-clicking Start and selecting either Open or Open All Users, whichever one you didn't do first.
