dcayce say:

Help with Wildcard Find & Replace

Hello, Forum;

I've got a tricky situation in Word I don't know how to manage. I've got a list of year dates — 4-digit, like 1995 — that have a tab on the left side and a paragraph break on the right. So each one looks like this: ^t 1995 ^p. I need to insert something before and after the year date, but most are all different from each other, so I need to represent them as a wildcard sequence, yet leave the date the same after I've made the change. What I'm inserting is a set of html tags, so the end result will be something like this:
^t <strong>1995</strong> ^p

Any advice on how to accomplish this would be greatly appreciated.

Cayce

jim gordon mvp say:
There are several ways to approach this problem.

If it were me, I would use the Find and Replace dialog that you get when you click on the page number information at the bottom edge of the document window.

In the Find and Replace dialog choose Replace, and then click the small triangle button to expose the large version of this form. Click the Special button to insert special characters into the search field and replace field and then let Word do the Find and Replace.

daniel g. grau say:
You need to group found terms (using parenthesis) and then refer to each group with a "backslash plus a number" sequence. In addition, finding the paragraph mark with wildcards requires using ^13:

Find: ^t([0-9][0-9][0-9][0-9])^13
Notice the "(group)" of exactly 4 digits, specified four times as the character range [0-9].

Replace:^t<strong>\1</strong>^p
Notice the reference to the first group as \1. In addition, unlike the search field, this field does allow ^p.

You could use this as well:

Find:(^t)([0-9]{4})(^13)
Replace: \1<strong>\2</strong>\3

... where there are now three groups and, instead of writing out the digits four times, 
we specifiy "{exactly 4 times}."

All this should be done through Edit > Find > Advanced Find and Replace and enabling Wildcards.
