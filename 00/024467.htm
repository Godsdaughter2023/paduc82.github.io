jay freedman say:
Rich's reply is for Excel. For Word the equivalent macro is

Sub PasteSpecialEnhancedMetafile()
    On Error Resume Next
    Selection.PasteSpecial DataType:=wdPasteEnhancedMetafile
End Sub

You can add a button for this macro to the Quick Access Toolbar or to a custom group on the ribbon (and in Office 2010, customizing the ribbon
Adding it to a right-click menu is a bit more work, but see http://gregmaxey.mvps.org/Customize_Word2007_SC_Menu_Programatically.htm for the method (it says 2007, but the process is the same for 2010). Also note that there is not just one right-click menu, but many -- essentially one for each part of a document you can right-click on (ordinary text, lists, table cells, etc.).
