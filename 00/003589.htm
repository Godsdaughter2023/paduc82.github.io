2.) Click on the “System” folder in the navigation pane.  

3.) Click on the “View” dropdown menu and then on “Filter”.

4.) Select “Microsoft Antimalware” as the Event source and enter “1117” for the Event ID.

VBA and SQL always use USA date format mm/dd/yyyy regardless of your own Windows settings. So if you enter a literal date in VBA code or in a SQL string, you must use mm/dd/yyyy format. If you refer to a text box on a form, you must convert the value, for example:

Dim strSQL As String
