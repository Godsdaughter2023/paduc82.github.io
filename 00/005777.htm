mike h.. say:

Hi,

Use this ARRAY formula and see below on how to enter it. I recommend you shorten the range if you can and not use full columns

=MAX(IF(A:A=1,IF(B:B=E1,G:G)))

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

 

EDIT... You wanted 6 arguments. Try this but I now strongly recommend you shorten the ranges

=MAX(IF(A:A=1,IF(B:B=E1,IF(C:C="x",IF(D:D="y",IF(J:J="z",IF(K:K=99,G:G)))))))

Once again an array
