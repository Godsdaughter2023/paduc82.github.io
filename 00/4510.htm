greginmich say:

I just upgraded from Win 7 to 10.  All is working well.  However, I don't see an icon for Windows Defender in the System Tray until after I open the Defender program.  I understand this was the default option in Win 8 but not for Win 10.  However, I could be incorrect.

Task Manager displays the service WinDefend service as running.  I just want to be sure everything is OK, and, if possible, how to get the icon to show after doing a reboot.  I uninstalled MSE from my PC prior to doing the upgrade and have never had any other av program on my PC.

The icon doesn’t show up at startup because there isn’t any startup entry for the Windows Defender user interface in the registry. That’s true for both Windows 8.1 and Windows 10. So the only thing that Brink’s mysterious little reg file actually does is add a registry startup line for MSASCui.exe.

You can do the same thing in several different ways without using any downloads or editing the registry. What I did first was go to File Explorer Options in Control Panel and change the (View) option to “Show hidden files, folders, and drives”. Then in File Explorer, I navigated to MSASCui.exe (the executable for the Windows Defender user interface):

Program Files/Windows Defender

I then right-clicked on the file and picked “Create shortcut”. Next, I copied the desktop shortcut to my Startup folder:

Program Data/Microsoft/Windows/Start Menu/Programs/Startup

Then, because I didn’t want the interface itself to load at startup, I just right-clicked on the shortcut, selected “Properties”, and added the “ -hide” parameter (without the quotation marks) at the end of the target path. So this is something that you can definitely do on your own.

Note that the Windows Defender icon is hidden by default. To unhide it, just click on “Show hidden icons”  (the “up arrow” in the notification area) and drag the Windows Defender icon into the notification area.

 [Edit for improved method 11/30/2015]

I’ve just developed a much quicker way to add the Windows Defender icon/startup-task by copying and pasting the specific line for the option you want directly into the shortcut – and then using Quick access to drop the shortcut into the Startup folder:

"C:\Program Files\Windows Defender\MSASCui.exe" -hide

"C:\Program Files\Windows Defender\MSASCui.exe" -QuickScan -hide

"C:\Program Files\Windows Defender\MSASCui.exe" -Update -hide

"C:\Program Files\Windows Defender\MSASCui.exe" -UpdateandQuickScan -hide

1.) Select and copy the run line for the option you want.

2.) Right-click on the desktop and select New > Shortcut and paste the run line as the location (target). Then click Next, name the shortcut, and click Finish.

3.) Open File Explorer: Start Menu > File Explorer; and type (or copy and paste) shell:startup (user Startup folder) or shell:common startup (common Startup folder) into the Quick access box and hit the Enter key to open the Startup folder.

4.) Then drag and drop the shortcut into the Startup folder. All done!

[End edit]

GreginMich
