jrbahr say:

Just an update on this:

Roger's last post worked (the Dlookup with quotes), but it took my query over 20 minutes to run.  So I continued to try and figure out another way.  Eventually, I ended up determining that the code from my original post would work as long as I didn't have the GROUP BY set on my query.  So, for example, my query would have originally looked like this:

SELECT IIf([UPLINE1] In (SELECT UPLINE_CODE FROM tblInvisibleUplines GROUP BY UPLINE_CODE),[AGENT],[UPLINE1]) AS AGENCY_CODE FROM qryUplineGROUP BY IIf([UPLINE1] In (SELECT UPLINE_CODE FROM tblInvisibleUplines GROUP BY UPLINE_CODE),[AGENT],[UPLINE1])

When I took out the underlined part, it worked.  After doing some experimenting, I determined that I could use the GROUP BY, but not with the IIf part.  So I ultimately ended up with this:

SELECT IIf([UPLINE1] In (SELECT UPLINE_CODE FROM tblInvisibleUplines GROUP BY UPLINE_CODE),[AGENT],[UPLINE1]) AS AGENCY_CODE FROM qryUplineGROUP BY UPLINE1

Hi blazingwade,

1. Sign in to your account and go to Calendar.

2. Click the day to create an event.

3. Click View details.

4. Add a title to the event and provide the other needed information according to your preferences.

5. Under How often, click the drop down and select Monthly.

6. Set Repeats every to 1 month(s).

7. Under the On the option, select the second option with the dropdowns.

8. Choose First and Saturday on the dropdowns and click Save.

 

Note: If the event was already created, you can edit that by clicking on the event, click on View details and select Change all then follow Steps 4-8.
