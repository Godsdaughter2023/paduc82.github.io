You can't use Autoruns to disable programs, just control which ones start or don't start at startup (if they were added by the program or by Vista).  I don't know why Software Explorer is not allowing you to enable ptorgrams - it should since it was the program that disabled them.  Perhaps it's an update or some type of permissions problem (or corruption in the program which can't be uninstalled or re-installed on Vista without doing Vista in its entirety.

Let's try this.  Go to  C:\Program Data\ Microsoft\Windows Defender\Software Explorers\ Disabled Startup Folder Items.  Check the list for the item you want to enable (it should be there as disabled).  Try to delete or otherwise remove that item from the folder.  If necessary, try right clicking on the item and then click on Run as Administrator (if that option is available).  If not, right click on the item and click on properties and then on security and give yourself full rights and take ownership (you may also need to do this for the folder itself or maybe even its parent folder as well or as high as necessary to get permissions and ownership).

If that doesn't work, try using one of the following free products to delete the file..  Unlocker at:http://www.softpedia.com/get/System/System-Miscellaneous/Unlocker.shtml or File Assassin http://www.malwarebytes.org/fileassassin.php (with or without Force Delete: http://www.mydigitallife.info/2008/12/27/force-delete-cannot-delete-locked-or-in-used-files-or-folders-with-fileassassin/). These programs often work when normal Vista delete functions fail to operate properly.

If that doesn't work, try uninstalling the program and re-installing it again (if you have the software or can get it from the manufacturer's web site).,  Dell is pretty good about making software downloads available.

If that doesn't work, try the Windows Defender Discussion Group at:http://www.microsoft.com/athome/security/spyware/software/newsgroups/reader/default.mspx?dg=microsoft.private.security.spyware.general&lang=en&cr=US&r=33d293cc-19b6-404b-865b-565a73822c7f where experts in Defender may be able to resolve your problem.

I hope this helps.
