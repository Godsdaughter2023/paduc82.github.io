No, it will not work.  The ARM CPU in the Surface RT and Surface 2 are 32-bit CPUs.  The new version of Windows 10 for ARM is for ARM 64-bit and will only run on 64-bit versions of a special ARM CPU that is designed to run it (Qualcomm's Snapdragon CPU).

a. user say:

Hello Kell,

Thank you for posting your query in Microsoft Community.

I understand that you have issues with dll files in the computer while, accessing some programs. We will certainly help you with the same.

Winsta.dll error messages might appear while using or installing certain programs, when Windows starts or shuts down, or maybe even during a Windows installation. Winsta.dll errors are caused by situations that lead to the removal or corruption of the Winsta dll file.

 You may need to register, or unregister, a .dll file and see if that helps. Follow the steps mentioned below to fix the issue.

  1. Go to Start Screen.
  2. Type CMD, right-click on CMD and Run as Administrator.
  3. Firstly, try to unregister a file, type: regsvr32 -u winsta.dll
  4. Then, try to register (or re-register) a file, type: regsvr32 winsta.dll
  5. Type Exit to exit the Command Prompt Screen.

If that doesn't help, I would suggest you to perform SFC scan. SFC /scannow is a super-useful command you can use in any Windows version. When the SFC (System File Checker) command is used with the /scannow switch, the tool will scan all of the important Windows files on your computer and replace them as necessary. Missing and corrupt operating system files (like many DLL files) are arguably the biggest cause of major Windows issues.

Use the System File Checker tool to repair missing or corrupted system files

http://support.microsoft.com/kb/929833

Hope this helps. Please feel free to write us back for further assistance and we will be glad to help you ahead.
