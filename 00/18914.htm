macropod say:
The following macro makes a backup of all the bibliography sources for the current Word account. The backup is in the form of a standard document containing the relevant xml code for all sources. You can use this to extract the sources from a given PC.

Sub SourcesBackup()
Application.ScreenUpdating = False
Dim oSrc As Source, StrSrc As String
With ActiveDocument
  .Range.Text = vbNullString
  For Each oSrc In Application.Bibliography.Sources
    StrSrc = StrSrc & vbCr & oSrc.XML
  Next
  .Range.InsertAfter StrSrc
  .Paragraphs.First.Range.Delete
  .Save
End With
Application.ScreenUpdating = True
End Sub

The following macro restores all the bibliography sources saved by the above macro to the current Word account.

Sub SourcesRestore()
Dim i As Long
'In case the source is already present
On Error Resume Next
With ActiveDocument
  For i = 0 To UBound(Split(.Range.Text, vbCr))
    Application.Bibliography.Sources.Add Split(.Range.Text, vbCr)(i)
  Next
End With
End Sub

Used together, the two macros can be used to transfer bibliography sources between PCs.

Sub CompareSheets()

'Oct 01, 2015

Dim fg As Boolean

Dim ws1 As Worksheet, ws2 As Worksheet

Dim r1 As Long, r2 As Long, c1 As Long, c2 As Long, x As Long, t As Long
Set ws1 = Sheets("Sheet1") '<< sht1 name, change
Set ws2 = Sheets("Sheet2") '<< sht1 name, change
r1 = ws1.Cells.Find(What:="*", After:=ws1.[A1], LookIn:=xlFormulas, LookAt:= _
xlPart, SearchOrder:=xlByRows, SearchDirection:=xlPrevious, MatchCase:=False).Row
c1 = ws1.Cells.Find(What:="*", After:=ws1.[A1], LookIn:=xlFormulas, LookAt:= _
xlPart, SearchOrder:=xlByColumns, SearchDirection:=xlPrevious, MatchCase:=False).Column
r2 = ws2.Cells.Find(What:="*", After:=ws2.[A1], LookIn:=xlFormulas, LookAt:= _
xlPart, SearchOrder:=xlByRows, SearchDirection:=xlPrevious, MatchCase:=False).Row
c2 = ws2.Cells.Find(What:="*", After:=ws2.[A1], LookIn:=xlFormulas, LookAt:= _
xlPart, SearchOrder:=xlByColumns, SearchDirection:=xlPrevious, MatchCase:=False).Column

'step1: compare number of rows and number of columns in two shts
If r1 <> r2 Or c1 <> c2 Then
MsgBox "not match data in " & ws1.Name & " and in " & ws2.Name
Exit Sub
End If

'step2: compare data per columns, in two shts
ReDim v1(1 To r1)
ReDim v2(1 To r2)
For x = 2 To c2
t = 1
For Each r In ws1.Range(ws1.Cells(1, x), ws1.Cells(r1, x))
v1(t) = r.Value
v2(t) = ws2.Range(r.Address).Value
t = t + 1
Next
If Join(v1, ",") <> Join(v2, ",") Then

fg = True

MsgBox "not match data in column " & Split(CStr(ws1.Cells(1, x).Address), "$")(1)
Else
'MsgBox "match data in column " & Split(CStr(ws1.Cells(1, x).Address), "$")(1)
End If
Next

If fg = False Then MsgBox "match data in " & ws1.Name & " and in " & ws2.Name

End Sub

Il dispositivo Nokia non funziona in modo corretto.


Verifica sempre che nel dispositivo Nokia sia installata l'ultima versione del software. Puoi aggiornare personalmente il software utilizzando Programma di aggiornamento software Nokia, il software per PC gratuito di Nokia. Per saperne di più e scaricare il Programma di aggiornamento software Nokia, visita l'indirizzo http://www.nokia.com/global/support/software-update/other/nokia-software-updates/

 

Se ritieni che il tuo dispositivo Nokia debba essere sottoposto a riparazione, visita l'indirizzo http://www.nokia.com/global/support/locations/. Le località in cui sono presenti centri di assistenza Nokia Care sono riportate nella pagina del supporto del tuo Paese. 

Come posso sapere quando è disponibile un nuovo software per il dispositivo Nokia?
Puoi ricevere notifica di nuove versioni del software per ogni dispositivo Nokia in diversi modi:

1) Tramite Nokia PC Suite
Ogni volta che connetterai il dispositivo all'ultima versione di Nokia PC Suite con un cavo USB compatibile, PC Suite verificherà se è disponibile un aggiornamento per il dispositivo. Questa notifica funziona solo quando utilizzi una connessione via cavo USB.

2) Verifica online
Puoi facilmente controllare l'ultima versione disponibile sul
sito Web del Programma di aggiornamento software Nokia.

Al momento Nokia non pubblica le date di rilascio previste per le nuove versioni del software.


Quali aggiornamenti sono stati apportati in ciascuna versione nuova del software?
I principali aggiornamenti sono pubblicati sul
sito Web del Programma di aggiornamento software Nokia.


Come posso far durare più a lungo la batteria del dispositivo Nokia?
Ecco alcuni dei suggerimenti più comuni. Questi suggerimenti non si applicano a tutti i modelli di dispositivi Nokia:
- attiva il Bluetooth solo quando ne hai realmente bisogno
- imposta la connessione GPRS su "Su richiesta"
- attiva "Modalità standby" se disponibile
