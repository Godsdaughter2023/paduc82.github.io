macropod say:

Hi Suzer9,

Populate a single bookmark and use cross-references to that bookmark. A call to Activedocument.Fields.Update will then update all cross-references in the body of the document. Thus:

Sub ChangeBookmark(BookmarkToCopy As String, BookmarkToPaste As String)
Dim BMRange As Range
With ActiveDocument
  Set BMRange = .Bookmarks(BookmarkToPaste).Range
  BMRange = BookmarkToCopy.Range
  .Bookmarks.Add BookmarkToPaste, BMRange
  .Fields.Update
End With
End Sub

myveryownself say:

When 8,487 and $2.3710 are calculated from a formula, there's a chance of additional non-zero trailing digits that are not appearing. Only a rounded value is appearing. But the extra digits are still there behind the scenes when new calculations use the numbers.

Try this example. Type 1.4 in A1 and 1.4 in B1. Then format the cells to display no decimal digits. Both say 1 now. In C1 put =A1+B1. It says 3, not 2.

To avoid this kind of issue, wrap ROUND(...) around equations, rounding the result to the number of digits you want to keep.
