a. user say:

If that gives you what you need but you just want to count and not click, you could try

Dim totalwords as Long

totalwords = 0 

With Selection
  .HomeKey Unit:=wdStory
  With .find

    .ClearFormatting
    .Font.color = wdColorBlue
    .Wrap = wdFindContinue
    Do While .Execute

      totalwords = totalwords + Selection.Range.ComputeStatistics(wdStatisticWords)
    Loop
  End With
End With

Msgbox "Total word count: " & CStr(totalwords)

a. user say:

With a specific color, you can use the Font.Color instead of the ColorIndex.

e.g. in the case you have, if you only had a red colour with r=200,g=0,b=23 you could use

Sub getRedBlueWordCountFromSelection2()
Dim blueColorValue As Long
Dim redColorValue As Long
Dim countBlue As Long
Dim countRed As Long
blueColorValue = getColorValue(0, 119, 129)
Debug.Print blueColorValue
redColorValue = getColorValue(200, 0, 23)
Debug.Print redColorValue
Dim r As Word.Range
For Each r In Selection.Words
  Select Case r.Font.Color
    Case blueColorValue
      countBlue = countBlue + 1
    Case redColorValue
      countRed = countRed + 1
    Case Else
      Debug.Print r.Font.Color
  End Select
Next
MsgBox _
  "Blue words: " & CStr(countBlue) & vbCrLf & _
  "Red words: " & CStr(countRed)
End Sub
Function getColorValue(red255 As Long, green255 As Long, blue255 As Long) As Long
getColorValue = (blue255 * 256 * 256) + (green255 * 256) + red255
End Function

If the user was deleted and recreated later and now the user gets permission denied errors when visiting OneDrive this is the fix:

  1. Go to portal.office.com
  2. Sign in with global admin credentials
  3. Click on admin tile
  4. Go to SharePoint Admin Center
  5. Click on user profiles
  6. Select Manage user profiles
  7. Enter the name of the user in search and click on find
  8. Click on the dropdown across user's name
  9. Select Manage site collection owner
  10. Make yourself the primary admin and secondary admin as well
  11. Also remove the end user's name from primary admin and add him under secondary admin click on OK
  12. Click on Manage personal site in the dropdown across end user's name
  13. This will navigate you to user's OneDrive for Business site
  14. Go to site settings and select people and group option
  15. Remove everything after the URL ending with  /_layouts/15/people.aspx?MembershipGroupId=0
  16. Select the end user's name by checking box
  17. Under actions option select delete user from site collection
  18. Go back to SharePoint admin center
  19. Click on user profiles
  20. Select Manage user profiles
  21. Enter the name of the user in search and click on find
  22. Click on the dropdown across user's name
  23. Select Manage site collection owner
  24. Check if the user should not be present now
  25. Add the user back again as primary and secondary admin and access with his credentials

With Outlook.com you can still be able to import contacts. Please be informed that contact groups won't get copied as they are onto Outlook.com's contact list. Contacts will only merge with the current Outlook.com's list of contacts.

If you want to copy contacts to Outlook.com, make sure that you have an exported copy saved on your computer. You may refer to your email provider to assist you on how to exprt contacts and make sure that the output format is (.CSV)

4. Click Browse and locate file for the contacts you've exported.
5. Select the file then click Open.
6. Click Import contacts.

Heading is messed up in Microsoft Word 16 - Microsoft Community

aks108 say:

Heading is messed up in Microsoft Word 16

In one of my word document heading 2 is missed up. It shows black box where the number (1.1) should be, see the screenshot below. I tried to reformat it but it doesn't seem to work. How do I fix it? I'm on Mac OS.

doug robbins - mvp office apps & services (word) say:

The problem occurs because the font characteristics in the list template have become corrupted.  It can be fixed by running a macro containing the following code:

Sub Fixit()

Dim templ As ListTemplate

Dim lev As ListLevel

For Each templ In ActiveDocument.ListTemplates

    For Each lev In templ.ListLevels

        lev.Font.Reset

    Next lev

Next templ

End Sub

To see how to make use of the above code, see the following page of Graham Mayor's website:

http://www.gmayor.com/installing_macro.htm

one page document prints 2 pages...how do I delete pg 2? - Microsoft Community

search777 say:

one page document prints 2 pages...how do I delete pg 2?

Hi,

When I go into a 2nd page while creating a document and then condense my content to one page, the document file remains 2 pages. It prints a 2nd blank page, or if I've used a header or footer the 2nd pg will print the header/footer. How do I delete that 2nd page so it just prints the one page?

 

terfar say:

Turn on the ShowAll command (the button on the Home ribbon, Paragraph Group that looks like a reverse upper case P) and look to see what is on the second page and delete it.

But note that the one exception is if the first page has a table at the bottom of the page close to the bottom margin: tables MUST have a following paragraph mark. If that is the case, select the paragraph mark and make sure that the paragraph formatting has all spacings set to zero and set the font size to 1 point.
