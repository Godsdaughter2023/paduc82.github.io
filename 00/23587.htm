john sr wilson say:

See if this works (and if so spend a little time seeing what was wrong before.)

Sub resize()

   Dim w As Single
   Dim h As Single
   Dim l As Single
   Dim t As Single
   Dim Sld As Slide
   Dim selShp As Shape
   Dim Shp As Shape

   'Pick up position
   Set selShp = ActiveWindow.Selection.ShapeRange(1)
   
   With selShp
      w = .Width
      h = .Height
      l = .Left
      t = .Top
   End With
   
   'Loop through slides
   For Each Sld In ActivePresentation.Slides
   'Loop through shapes on slide
      For Each Shp In Sld.Shapes
         If Shp.Name = selShp.Name Then
            With Shp
               .Width = w
               .Height = h
               .Left = l
               .Top = t
            End With
         End If
         Next Shp
         Next Sld
      End Sub

Hi Mandrake,

A lot has changed in IE since some toolbar addons were first developed for IE6 to 8 on XP windows.

Unfortunately, there are no settings in IE that I can recommend that will change the behaviour of third-party toolbar addons in the IE Address bar. Generally, IE won't let you position a toolbar on the same row as the Menu bar, or between the Command bar and Favorites bar. Some toolbar addons have been programmed to position themselves, regardless of your preferences from the View>Toolbars menu in IE.

You could try checking for updates from the vendors websites, but sadly(or gladly), many third-party addons for IE are no longer compatible with the modern (and more secure) versions of IE. After IE6, much of the functionality that was only available as an addon in IE (form filling, popup blocking, web search, site security) was built into the browser instead. With the latest versions of IE there is little need to continue using toolbar addons (and another type of addon called Browser Helper Objects (BHO's))

Common toolbar addons for MSIE browsers can be found at

toolbar.google.com, toolbar.yahoo.com, and toolbar.aol.com

You can examine which addons you have installed in your browser from the Tools>Manage Addons>Show All Addons menu from IE. Addons of type Toolbar or BHO (Browser Helper Objects) are the two types that you are interested in.... examine the list and double click on an item to display its properties dialog... this will show you more details, including (but not always) the publisher of the addon. You will also see in the list usage statistics of all of your addons, including the number of times the addon has been used...

Some addons you can also check for updates from the 'Programs and Features' control panel.

go Start>Control Panel>Programs and Features.... sort the list by Publisher.... then scroll down the list and find any installed addons that you identified from the IE Addons Manager (above)....

When selected either or both the 'Uninstall' and 'Update' buttons will be enabled... the 'Update' button will go to the publishers web site and check to see if there are updates available for their addons.

I would recommend that you choose the 'Uninstall' button for any Addons that you rarely use, Addons that you are not sure about the reputation of the publisher, Addons that you no longer require (like form fillers, popup blockers and web search), because that functionality is already built-in to IE.

The toolbar that 'does nothing' when the [X] button is clicked, probably has been damaged. You can identify its name from the View>Toolbars menu in IE. Currently displayed toolbars will appear checked in the menu list. Click the menu item to unload them from IE. Note their names, then when using the IE Addons Manager it will be easier to identify them.

Also, all windows version come with Windows Defender already installed. Windows Defender scans your computer for known incompatible and malicious IE Addons and when found it can automatically remove the software from your computer or give you online directions for its removal.

Start>Control Panel>Windows Defender

As always, it is wise to have installed an activated and current anti-virus program, and to keep your personal computer up-to-date with the latest software patches and security updates from Microsoft.

From IE select the Tools>Windows Update menu item. Or Start>Control Panel>Windows Update.

Windows Updates are released on the second Tuesday of each month... You can configure your preferences for automatically downloading and installing the updates from the Windows Update control panel. You can keep on working while the updates are being installed, but you may have to at some stage restart your computer for the full update changes to be installed.

Some updates are quite big, and you may need an internet connection that will allow downloads of around 500 mega bytes... Some public access networks like public libraries and public hotspots are unsuitable as they limit the amount of downloads that you are able to make.
