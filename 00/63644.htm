hairyfool say:

For the terms of the license the motherboard and processor is the computer.

Even if you replace like for like activation will fail because it will be detected as a different board, the parameters used to identify such a change will still identify it as being different.

However replacing like for like (or as close as possible to) can be classified as a repair in which case the Activation call centre can allow the transfer.. If an upgrade then you have to buy a new license. In the Activation screen there is an option to enter a new key to the installation on the drive.
