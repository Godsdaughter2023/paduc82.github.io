jdeanwilson say:

This is a serious problem with these 32GB eMMC drive machines. I bought 8 HP Streams with the same hard drive. The problem is primarily installing the Creator's Edition (build 1709), smaller updates may work fine. Nothing you do will get you enough space to install the 1709  update. I know, I spent two days trying.

Here's how I got over the 1709 hump.This only works when setting up NEW computers as you will lose all your data unless you back it up and then reinstate it later.

  1. Create USB flash boot drive here: https://www.microsoft.com/en-us/software-download/windows10
  2. Boot up from USB flash drive.
  3. When you get to the Drives screen, format the C drive to remove everything.
  4. Install Windows. Of the available 27.8GB, this left 14.7GB.
  5. After installing, log in to Windows. Go to Apps control panel and remove all games. This recovers 2GB. Now at 16.6GB. Don't bother with Disk Cleanup Utility, it will get you virtually nothing.
  6. Install Office. Down to 8.6 GB. Ouch. (Or do partial install using this method: https://www.askvg.com/tip-customize-microsoft-office-click-to-run-c2r-setup-to-install-selected-programs-only/)
  7. Install favorite browser, anti-virus and other small things and you're left with about 6GB.

I can't see it getting any better than that. I use these machines at conferences as email-reading and internet-surfing stations, so I can get by with this. For everyday use, this would be usable.

-Jeff

UPDATE: Uninstalling the games does no good. Windows just reinstalls them when you're not looking.
