Hi JohnLattimore1,

Thank you for your response.

This is because both Bluetooth and Wi-Fi technologies use the same radio frequency to communicate. Bluetooth uses 2.402 GHz to 2.480 GHz and Wi-Fi uses 2.412 GHz to 2.472 GHz. They both work in the 2.4 GHz spectrum but Bluetooth is designed to hop to a different frequency automatically when the interference is detected in the channel. It only hops by 1 MHz at a time, so it might still be close to your Wi-Fi frequency.

To resolve this issue, I would suggest you to follow these steps:

  1. Login to the Router and go to 2.4GHZ settings.

  2. Check if the Channel is set to Automatic.

  3. Change it to 11 (Note: If it is already set 11, change to 10)

  4. Save the settings and check if the issue is fixed now.

You may need a different channel to prevent interference with Bluetooth. Experiment with different channels and let us know if it fixes the issue.

Hope it helps.

Windows 8/8.1 automatically manages virtual memory and paging file size by default. It doesn't set virtual memory as per the internal recommendation. The virtual memory set by default is approximately 70% of the recommendation.

 Set Maximum size to twice of your RAM size in MB
 Click on Set
 Click on OK thrice
 Restart your system
