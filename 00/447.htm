Sub CleanMe()
'dimension the only variable in the code
 Dim c As Range
    'set up the loop to loop through each cell
    'that is selected on the worksheet
     For Each c In Selection
     'Make sure there's no formula in the cell because
     'if there is then this code would change it into
     'a value
     If Not c.HasFormula Then
     'using WITH means we can carry out
     'multiple operations on the cells
     With c
        'Replace all instances of ASCII character 160
        'with nothing
         .Value = Replace(.Value, Chr(160), "")
         'get rid of leading, trailing and reduce
         'duplicate middle spaces to a single space.
         'Note here I used WORKSHEETFUNCTION trim.
         'VBA has a TRIM function but it works differently to
         'the worksheet function. Check out help

    'Close the with and if
     End With
     End If
     'do the next cell
     Next
 End Sub

mike h.. say:

Hi,

I suspect the key here is that the data are from HTML and this often means there are characters that look like spaces but aren't, they are actually ASCII character 160.

You can demonstrate this for your self by putting this in a cell

=CODE(Right(a1,1))

and this will return the character code. A space is code 32. Try this code. Select any cells(s) and run the code.

Sub CleanMe()
Dim c As Range
    For Each c In Selection
    If Not c.HasFormula Then
    With c
        .Value = Replace(.Value, Chr(160), "")
        .Value = WorksheetFunction.Trim(.Value)
    End With
    End If
    Next
End Sub

I have a spreadsheet where I've copied and pasted phone numbers into a column (H). However, the numbers are all in different formats. Eg.
(800) 537-7783
1 800 732 9194
800 435 9792
1-800-222-4357

Is there any way I can reformat the cells with a custom format so that they all look uniform? The format I want is: (###) ###-####

Thanks in advance, you Excel geniuses out there.

rohn007 say:
First step will be to get rid of all of the manual formatting. Use find and replace to strip out the spaces, hyphens and brackets.

Excel 2010 has a predefined "phone" format that is ###-###-####. You could also define a custom format that looks like this:

[<=9999999]###-####;(###) ###-####

So you select the column, right click, select Format Cells, Custom Category, then paste the format from the line above into the "type" field, OK out.

jeeped say:
Is there any way I can reformat the cells with a custom format so that they all look uniform? The format I want is: (###) ###-####

Strip the existing phone number values down to numbers only with a series of Find & Replace commands to remove spaces, brackets, hyphens or use the following formula in an adjacent unused column.

=--RIGHT(SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(A1," ",),"-",),"(",),")",),10)

Fill Down to accommodate all phone numbers. Optionally use Copy then Paste Special, Values to replace the phone numbers in the original column.

                    

When you have the phone numbers stripped and behaving like numbers and not text, tap Ctrl+1 then choose a Special number format of Phone Number and select locale.

gabriel bratton say:

There is a new calendar sharing issue we are investigating.  The issue is that users who have permissions are not able to edit calendars.  It might manifest in different ways so work checking.  We are actively working on it and I will post back when we know we have a fix.

Edit to add that the issue is now documented here with workarounds, https://support.office.com/en-us/article/Customers-on-Current-Channel-version-16-0-8326-2062-are-not-able-to-edit-shared-calendars-69d4bd39-1b08-49a0-8e24-331122d20b2c?ui=en-US&rs=en-US&ad=US.

We know the issue is with the latest updates with major version 16.0.8326.2073.  From the cases we have had so far we can workaround it by reverting to the previous major version 16.0.8229.2086.

To revert to the prior build do the steps.

Open a Command Prompt and run these commands:

cd %programfiles%\Common Files\Microsoft Shared\ClickToRun

officec2rclient.exe /update user updatetoversion=16.0.8229.2086

To prevent the update from re-installing click File, Office Account, Update Options, Disable Updates.  I also recommend adding an appointment on your calendar for a couple weeks out to remember to re-enable updates.  

I pulled the older version to install from this document, https://support.office.com/en-us/article/Version-and-build-numbers-of-update-channel-releases-ae942449-1fca-4484-898b-a933ea23def7?ui=en-US&rs=en-US&ad=US.  

I got the revert steps from this KB, https://support.microsoft.com/en-US/help/2770432.
