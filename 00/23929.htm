How would I go about formatting columns B - G so the cells in the same row as the cell being flagged turn red as well?I'm just using the built in "Bad" format.

-uriah
jeeped say:
Is it possible to format columns B - G based on whats happening in column A? Basically, if a cell in column A if being flagged because of a value displayed (in this case equal to another cell), it will turn red ...

You haven't specified what cell column A is referenced to in order to trigger the red fill, but let's say it is column Z in the same row,

First abandon (e.g. delete) your previous rule and then select all of columns A:G with A1 as the active cell. Create a new rule with Use a formula to determine which cells to format and use the following as Format values where this formula is true:,
=AND(ISNUMBER($A1),$A1=$Z1)
... then click Format and select a red fill. Click OK once to save your formatting choices and then OK again to save your new conditional formatting rule.

Note that the formula uses absolute referencing for the column and relative referencing for the row. This is how the cell in one column affect many others in the same row.

    shortcut for paste formulas - Microsoft Community

    jimmytap say:

    shortcut for paste formulas

    Is there a way to create a shortcut for paste formulas in excel 2011 for Mac (or perhaps a work around)? I have already created shortcut keys for paste values and paste formats but I don't see the option to create shortcut keys for paste formulas.
    j.e. mcgimpsey say: One relatively easy way is to type or paste this macro in a regular code module in your Personal Macro Workbook:

        Public Sub PasteFormulae()
            'On Error GoTo ErrHandler
            If TypeOf Selection Is Range Then _
                Selection.PasteSpecial _
                            Paste:=xlPasteFormulas, _
                            Operation:=xlPasteSpecialOperationNone
        ExitSub:
            Exit Sub
        ErrHandler:
            Debug.Print Err.Number, Err.Description
            MsgBox "Can't Paste Formulas!"
            Resume ExitSub
        End Sub

    and assign a keyboard shortcut to it in the Macro/Options dialog

    Note that some browsers replace spaces with non-breaking spaces - if you paste the macro and get syntax errors, delete the white space to the left of each line and replace them with tabs or spaces.

    bob jones aka: cybertaz say:
    Are you looking to paste only the formulas without the formatting? Unfortunately, there is no direct command for every option in the Paste Special dialog -- that's one case in point.

    The easiest alternative is to use the Paste Options button after pasting to select Match Destination Formatting.

    A bit longer is to use Control+Command+V, then select Formulas in the dialog.
