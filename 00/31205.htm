Hi

Check the following policy.

In Windows 7, you can set the Quick Removal policy for an External USB device. What this does is disable the file caching for that device which allows you to bypass the Safely Remove component.

1. Plug the device into the USB port.

2. Open Windows Explorer,
3. Select the device in that list and click the Properties Button and then the Change Settings Button.

4. Select the Policies Tab.

5. Select the Quick Removal option.

6. Click OK/Apply/OK.
