davem121 say:
Hi Gareth

Open Windows File Explorer

On the left click 'This PC'

On the right, right click that card and choose 'Format'

On the resulting dialog, do you get the option of FAT - not FAT 32

If so, accept that and apply a quick format


If you do not get the option of FAT

Note down the drive letter of that card

Close File Explorer

Click your Start Button, type cmd then right click Command Prompt and choose 'Run as Administrator'

Run the following command (replacing E with the appropriate drive letter) and hit Enter

format E: /fs:FAT

Wait for that to complete, then close Command Prompt

That's It!

In case you can’t get to display the subtitles in the movie in Windows media Player or Windows Media Center, then one thing that might be worth knowing is the subtitle format.
Please note that on Windows 7, both Windows Media Player and Media Center make use of the Windows Media Foundation (WMF) framework instead of DirectShow for playing certain file types. As a result, DirectShow subtitle filters like DirectVobSub are not used.
WMF is used for the following file types: MPEG, WMV, MP4, MOV, M2TS. For other file formats, such as AVI and MKV, the Microsoft players are (in most cases) able to display subtitles.
