If you still have the old computer, the simplest approach is to copy the fonts from its Fonts folder to a flash drive and then install them on your new system. Windows 7 makes this really easy: just right-click the font on the flash drive and select Install.

 

If you don't still have access to the old computer, then as Rohn007 says, you'll have to figure out what program Broadway came with. According to http://www.microsoft.com/typography/fonts/font.aspx?FMID=1093, this font is/was supplied with Greetings 99, Home Publishing 99, Office 2000 Premium, Office 2007, Office 2007 Professional Edition, Office 2010, Office 97 Small Business Edition SR2, Office Professional Edition 2003, PhotoDraw 2000, Picture It! 2000, Picture It! 2002, Picture It! 98, Publisher 2000, Publisher 2007, Publisher 98, and Windows Small Business Server 2003. Since Office 2007 is included, it's possible you did not install all the fonts when you installed Office on the new machine; you can do a repair/update install to select all the "extra" fonts (they'll be in the Shared Components) and install them.

To answer your question, included graphics and custom formatting add a tremendous amount of overhead to a Project file. One of the best ways to reduce the bloat is exactly what you have done, save to XML and re-open in Project.

Please consider the following troubleshooting steps:

  1. Make sure that Skype is OK.
  2. Find and delete the shared.xml file. If file extensions are not displayed on your computer, the file will just be called shared.
  3. Open the folder that has your Skype name.
  4. Locate and delete the config or config.xml file.
  5. Press the Windows  and R keys on your keyboard at the same time again.
  6. Type %appdata% into the Run window and click OK.
  7. Locate and rename the Skype folder to Skype_Old.
  8. Uninstall Skype.
  9. Restart your Windows computer.
  10. Download the most up-to-date version.
