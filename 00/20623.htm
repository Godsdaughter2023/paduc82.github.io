Please follow the steps below:

 

  1. In Outlook, on the File menu, click Import and Export.
  2. Click Import from another program or file, and then click Next.
  3. Click Microsoft Excel, and then click Next.
  4. In the File to import box, browse to find the Excel file (.xls) that contains the content you want to import.
  5. Click one of the following:
    • Replace duplicates with items imported
    • Allow duplicates to be created
    • Do not import duplicate items
  6. Click Next.
  7. In the folder list, click the Contacts folder where you want the imported contact information to go, and then click Next.
  8. To add or remove fields to determine the way the contact information is saved in the new imported file, click Map Custom Fields.

Send as PDF - Microsoft Community

peter b. hansen say:

Send as PDF

How can I send a document as PDF attachment, from a macro? The user would then get the email screen where he can write the email address and the accompanying message, and press the Send button.
peter b. hansen say:

This code did it. I actually save a copy beside the document first, and I ask the user for permission to do that, plus timestamping the document. this means, you can find back the copy on the drive.

 

Public Sub FileSendPdf()
    Dim outl As Object
    Dim Mail As Object
    Dim Msg, Style, Title, Help, Ctxt, Response, MyString
    Dim PDFname As String
     
    Msg = "Your document will be timestamped and saved as PDF next to the Word document before mailing it as attachment, please agree?"
        Style = vbOKCancel + vbQuestion + vbDefaultButton2
        Title = "Save and timestamp before printing"
        Ctxt = 1000
        Response = MsgBox(Msg, Style, Title, Help, Ctxt)
        If Response = vbOK Then
            Call Stamp
            ActiveDocument.Save
            PDFname = ActiveDocument.Path & "\" & ActiveDocument.Name & ".pdf"
            ActiveDocument.ExportAsFixedFormat OutputFileName:=PDFname, ExportFormat:=wdExportFormatPDF, OpenAfterExport:=False, OptimizeFor:=wdExportOptimizeForPrint, Range:=wdExportAllDocument, Item:=wdExportDocumentContent, IncludeDocProps:=True, KeepIRM:=True, CreateBookmarks:=wdExportCreateNoBookmarks, DocStructureTags:=True, BitmapMissingFonts:=True, UseISO19005_1:=False
            Set outl = CreateObject("Outlook.Application")
            Set Mail = outl.CreateItem(0)
            Mail.Subject = "PDF copy of Use case " + ActiveDocument.Name
            Mail.Body = ""
            Mail.To = ""
            Mail.Attachments.Add PDFname
            Mail.Display
        Else
            MsgBox "You have chosen not to save + timestamp and your document will not be sent"
            cancel = True
        End If
End Sub
