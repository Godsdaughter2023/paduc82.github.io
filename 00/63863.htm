loobylou61 say:

You'll need to edit the system registry with regedit and change the relavant values to point to where your music is stored.

The registry values you are looking for are in "HKEY_CURRENT_USER\Software\Microsoft\MediaPlayer\Preferences"

In here you will find a set of values named; TrackFoldersDirectories0, TrackFoldersDirectories1, TrackFoldersDirectories2, etc

These wil contain the folders that WMP tracks changes in. When you click on 'Add to Library' in WMP these values will match the list of Monitored Folders.

All you need to do is to change one of the values to the pathname where you music is stored and then when you restart WMP, goto 'Add to Library' and you should see the new path displayed in the list of monitored folders. Once you have the coorect path, WMP should then rescan this folder and add your music to your library.

hairyfool say:

The Location tab tool is a bit unclear as to what is going to happen when you select Move.

Logically if you tell it to move the Documents folder it will do just that, move the folder. What it actually does is move the contents not the folder. So you wanted to move Documents to F:\ but it moved the files to F:\ (the root folder) then set the pointers for the Documents library to the drive itself.

If those folders actually had files then the 1st problem is splitting out the files into two temporary folders.

Then go back to the Location tab and click on Restore Default.

This should reset the pointers and remove the labels on the F:\ drive. Copy the files fixed in step 1 back to the Library folders.

Then you have two options. Either create the destination folders 1st. So create a Profile folder on F:\  and in there create a Documents, Music etc. folder. When you use the Browse function you can then find the folder you created and Move.

Alternatively in the Location tab it shows the current path (C:\Users/my_name/Documents). Edit the entry and replace C: with F: and then Move. The tool will create the path and move the files into that path including creating the Documents folder to hold the files.
