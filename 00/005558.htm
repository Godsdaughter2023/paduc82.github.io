David

 

graham mayor say:
The spell checking should only be turned off for the image itself. If an image is inserted in line then it behaves as text character with spell checking off. Text either side of the image should not be affected unless you have Automatically Update styles checked in the Developer Tab > Document Template dialog (and even then I am not convinced it should make a difference).

You can override the spell check for the image using a macro to intercept the standard paste command (Ctrl + V)

Sub EditPaste()
Dim oRng As Range
Set oRng = Selection.Range
oRng.Paste
oRng.NoProofing = False
End Sub

which will uncheck the no proofing option for ALL pasted items. http://www.gmayor.com/installing_macro.htm
