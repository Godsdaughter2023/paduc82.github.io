Resetting bullets to default - Microsoft Community

a@work say:

Resetting bullets to default

Hi All,

I work with an eLearning software Add-In to PowerPoint that frequently that has erratic behavior when it encounters non-standard bullets. Unfortunately, I also have to adapt a lot of PowerPoints that use these non- standard bullets.

 

Is there a quick way to restore the default bullet format you would find on a blank presentation on an existing non-standard text box? Just turning the bullets on and off doesn't seem to clear the current formatting, and often format paste from a properly formatted text box will not affect all parameters of the bullet style.

 

Either a faster technique or suggestion on a macro would be very helpful!

~A

a@work say:

ok, I have this working. My new code and comments in bold

 

Sub chex_colorvar()

Dim otxtr As TextRange
Dim i As Integer
Dim osld As Slide
Dim oshp As Shape

    For Each osld In ActivePresentation.Slides
    For Each oshp In osld.Shapes
        If oshp.HasTextFrame Then
            If oshp.TextFrame.HasText Then
            Set otxtr = oshp.TextFrame.TextRange
                For i = 1 To otxtr.Paragraphs.Count
                    If otxtr.Paragraphs(i).ParagraphFormat.Bullet.Visible = True Then

' This line below says to run the changes only if the bullet type is not a numbered bullet.
                        If otxtr.Paragraphs(i).ParagraphFormat.Bullet.Type <> ppBulletNumbered Then
                            With otxtr.Paragraphs(i).ParagraphFormat.Bullet
                            .UseTextFont = True
                            Select Case otxtr.Paragraphs(i).IndentLevel
                            Case Is = 1, 3
                            .RelativeSize = 1
                            .Type = ppBulletUnnumbered
                            .Character = 8226

' I also added my typical text color to this version.
                            .UseTextColor = 3366
                            Case Is = 2, 4
                            .RelativeSize = 1
                            .Type = ppBulletUnnumbered
                            .Character = 8211
                            .UseTextColor = 3366
                            Case Is = 5
                            .RelativeSize = 1
                            .Type = ppBulletUnnumbered
                            .Character = 187
                            .UseTextColor = 3366
                            End Select
                            End With
                        End If
                    End If
                Next i
            End If
         End If
    Next oshp
    Next osld
End Sub
