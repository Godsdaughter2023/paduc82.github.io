HDMI vs optical digital audio. - Microsoft Community

liketotallywill say:

HDMI vs optical digital audio.

I have a DENON AVRX3000 with three speakers (but I will get another 4 and a subwoofer in the future though), what is better? Optical or HDMI? And also, I don't know if my gamerpic and/or gamerscore is showing to you,because I can't see it on my screen, why is that?

skeeter quill say:

Hey there,

HDMI has a much higher audio bandwidth than optical. HDMI is capable of transmitting uncompressed 5.1 and 7.1 surround, basically sending the sound for each speaker separately. Optical however can only deliver compressed surround, so basically the console has to compress the audio, then the receiver will have to decompress it the other end.

So if you have the option, use HDMI and set your console to uncompressed 5.1/7.1 rather than Dolby or DTS and Dolby and DTS are both compression methods for surround.

a. user say:

Run the System File Checker (SFC.exe) tool with the sfc /scannow command


The sfc /scannow command scans all protected system files and replaces incorrect versions with correct Microsoft versions.


To run the System File Checker, complete these steps:


1.- Click Start, and then type cmd in the Start Search box.


2.- In the results area, right-click cmd.exe, and then click Run as administrator.

You will be prompted to type the password for an administrator account.

Click Continue if you are the administrator or type the administrator password.

Then, click Continue.


3.- At the command prompt, type Sfc /scannow, and then press ENTER.


The scan may take some time, so be patient.

Windows will repair any corrupted or missing files that it finds.

If information from the installation CD is needed to repair the problem, you may be prompted to insert your Windows Vista or Windows 7 CD.

After you run System File Checker, try to run Windows Update.
