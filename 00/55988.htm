There is nothing within Windows that can fix a physical hardware problem.

If the machine runs happily while connected to AC power, in all probability, the charger and power cord are working as they should, but inside the machine is a small board, maybe a replaceable unit, maybe permanently fixed to the motherboard which splits power between the main machine and the battery as needed. 80010108 RPC_E_DISCONNECTED
The object invoked has disconnected from its clients.

Is the most recent Version of the Windows Update Agent (WUA) installed ?

The error may indicate an issue with the ActiveX component of the WUA,
either it's an older Version that no longer works with the update
servers or security software is interfering with it.

See the " How to obtain the latest Windows Update Agent " section in
this MS KB: http://support.microsoft.com/kb/949104

In the WindowsUpdate.log ( How to read the Windowsupdate.log file
http://support.microsoft.com/kb/902093 )
you'll see an entry like this entry *if* the latest Version of the WUA
is installed:
