You may have to do a one-time change of any formulas that use the google drive path, but here is one approach to making the formulas more universal:

Create a VBA function that resides in the Workbook:

Function GetEnv(whatDoYouWant As String) As String
    GetEnv = Environ(whatDoYouWant)
End Function

Then, instead of:

="c:\users\Fred Bloggs\Google Drive\Data\MyInfo.xlsx"

You can use something like

="c:\users\"&GetEnv("username")&"\Google Drive\Data\MyInfo.xlsx"

In this way, whatever user opens the file will have their username inserted into the path automatically.

Hope this helps,

Eric
