malkeleah say:

I'm not sure what you did in the Recovery Console. What "ran for 36+ minutes"? To fix the "NTLDR is missing" error, boot from the XP CD and take R for the Recovery Console. At the command prompt, type:

COPY D:\i386\ntldr C:\ [enter]
COPY D:\i386\ntdetect.com  C:\ [enter] [a file that may also be missing] then rebuild the boot configuration boot.ini file by typing:
Attrib  -H -R -S C:\boot.ini [enter] [if not found skip the next line]
DEL  C:\boot.ini [enter]
BootCfg  /Rebuild [enter]

Remove the XP CD and hopefully boot into Windows. Consider purchasing an Uninterruptible Power Supply (battery backup). The ones from APC are good and will cost $60-90.

If this doesn't work then Windows was too damaged by the power outage. If the Repair Install didn't work (is that what you mean by "run for 36+ minutes"?), back up your data from outside of Windows and do a clean install. Let me know if you need help with any of that.

a. user say:

If you put in your XP installation CD and boot on it, you get the menu that says:

This portion of the Setup program prepares Microsoft Windows XP to run on your computer:

   To setup Windows XP now, press ENTER.

   To repair a Windows XP installation using Recovery Console, press R.

   To quit Setup without installing Windows XP, press F3.

 

If you press R to enter the Recovery Console, the Recovery Console normally loads and you can proceed with fixing whatever needs fixing and you have to do it by hand.

Pressing R at that point does not not magically repair your XP installation for you or run some repairing process - it just loads the Recovery Console so YOU can repair it using the guidance you have.  The message "To repair a Windows XP installation..." doesn't mean it is going to do it for you - it means you want to do it yourself.

After pressing R, you should get some additional prompting in a fairly short time.

If you press R to use Recovery Console (like the message says) and nothing happens for 36+ minutes, you have another problem (fairly common after power interruptions).
