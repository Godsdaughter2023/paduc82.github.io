End Sub

Basically, this code allows me to type only the character "2" into my field, and have it automatically change to "Alpha222." ken sheridan say: You'll need to do this in the control's NotInList event procedure like this:

    Dim ctrl As Control
    Set ctrl = Me.ActiveControl
   
    If NewData = "2" Then
        Response = acDataErrContinue
        ctrl.Undo
        ctrl = "Alpha222"
        Exit Sub
    End If

This can if necessary be followed by the usual code to insert a new row into a referenced table, subject to user confirmation.

This assumes that the visible column is referenced by the control's BoundColumn property.  If the BoundColumn property references a hidden numeric column, as is common, then you would need to assign the numeric value which determines "Alpha222" to the control's value.

BTW, the LimitToList property is a property of the control, not the column.  It is more likely than an enforced relationship would prevent the insertion of "2".  You certainly should not remove the enforcement of referential integrity.
