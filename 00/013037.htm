Hi,

Welcome to Microsoft Community and thank you for posting the question.

As per the description provided you find the configuration window when you try to open Word or Excel application.

Let me assist you to stop the configuration window when you open Office applications.

 

Which windows operating system do you use (Windows XP, Windows Vista, Windows 7)?

You may follow the steps to register DWORD value and check if it helps:

 

1. Take a back up the registry before proceeding with the next step. Refer the link on How to back up the registry:

 

http://support.microsoft.com/kb/322756 (Windows XP)

 

http://windows.microsoft.com/en-US/windows-vista/Back-up-the-registry (Windows Vista)

 

http://windows.microsoft.com/en-US/windows7/Back-up-the-registry (Windows 7)

 

 

2. Click on Start, Run. (Windows XP and Windows Vista)

    Click on Start, under Search Program and Files (Windows 7)

a. Word 2007 “reg add HKCU\Software\Microsoft\Office\12.0\Word\Options /v NoReReg /t REG_DWORD /d 1”

b. Excel 2007 “reg add HKCU\Software\Microsoft\Office\12.0\excel\Options /v NoReReg /t REG_DWORD /d 1”

 

Restart the computer and check if you are able to open Office applications without the configuration window.

 

I hope the above steps help. If you need further assistance on this particular issue or any other Office related issues let us know and we will be glad to assist you.
