Start button /

Display Settings./

Troubleshoot tab /

Update the graphics driver - Windows 7, Vista

installing it. Also be sure to follow any installation
instructions that may be available at the download site.

driver but the best source would be the computer
manufacturer or the motherboard manufacturer.

Font Size

doug robbins - mvp office apps & services (word) say:

Font sizes are in points of which there are 72 to an inch.  Therefore 3/16" = 13.5 pt

Note however not all letters in a font are on the same size and you may also need to take into account the Leading of the font.

suzanne s. barnhill say:
It will be at least partly a matter of trial and error. As Doug says, there are (nominally) 72 points to an inch. So if you need letters 3/16" high, you would start with 13.5 points. The capital letters should approximate the right size, but the line spacing will be greater, so the baselines of lines of text will be more than 13.5 points apart. This varies from one font to another. Also, for lowercase, the x-height of some fonts is greater than for others, and the distance from the bottom of a descender to the top of an ascender may be more than the nominal font size.

Private Function ProcessCountry(ByVal Country As String) As Boolean

    Dim CountryWorksheet As Worksheet
    Dim MatchColumn As Variant
    
    Set CountryWorksheet = ThisWorkbook.Worksheets("Country")
    MatchColumn = Application.Match(Country, CountryWorksheet.Range("H5:S5"), 0)
    If Not IsError(MatchColumn) Then
        If CountryWorksheet.Range("H4:S4").Columns(MatchColumn).Value = "X" Then
            ProcessCountry = True
        End If
    End If
