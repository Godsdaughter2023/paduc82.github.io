daniel pineault say:

Firstly, I'd highly recommend you use vbscript rather than a batch file.

Secondly, the same principle applies, you have to read the registry key to see what the currently configured version is.  So something like:

FOR /F "skip=2 tokens=2,*" %%A IN ('reg.exe query "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\App Paths\MSACCESS.EXE" /v "Path"') DO set "exePath=%%B"
ECHO %exePath%

The above will return the path then you need to concatenate and add the msaccess.exe, not a big deal, but you can do something like the following to get the whole thing in short notation


set APPEXE=MSACCESS.EXE
set KEY_NAME=HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\App Paths\
set FULL_KEY="%KEY_NAME%%APPEXE%"

FOR /F "skip=2 tokens=2,*" %%A IN ('REG QUERY %FULL_KEY% /ve') DO set "appExe=%%B"
ECHO %appExe%
