Word macro to insert cell - Microsoft Community

niggo108 say:

Word macro to insert cell

I need a word macro (office 2010 and 2013, windows 7 & 10) that inserts a cell above the cell that contains the cursor, and shifts the other cells down. Attempting to create such a macro by going through the steps ends up with a blank macro! Thanks.

< moved from: Office /Word /Windows 10 /Office 2010 >

jay freedman say:

The following macro does what I described (except for hiding borders).

Sub AddCellAboveSelection()
    Dim tbl As Table
    Dim selRow As Long
    Dim selCol As Long
    Dim maxCol As Long
    Dim idxCol As Long
    
    If Selection.Information(wdWithInTable) Then
        Set tbl = Selection.Tables(1)
        maxCol = tbl.Columns.Count
        selRow = Selection.Cells(1).RowIndex
        selCol = Selection.Cells(1).ColumnIndex
        
        ' this adds an entire row above the selection cell
        Selection.Columns(1).Cells.Add BeforeCell:=Selection.Cells(1)
        
        ' remove the extra cell(s) in other columns
        For idxCol = 1 To maxCol
            If idxCol <> selCol Then
                tbl.Cell(selRow, idxCol).Delete ShiftCells:=wdDeleteCellsShiftUp
            End If
        Next idxCol

    End If
End Sub
