frotna say:

With help from another forum i found solution for my problem.

Maybe someone else is going to need this also so i'm posting it.

 

Sub SUM_WBs()

    Dim FileNameXls As Variant, i As Integer, wb As Workbook
   
    'ChDir = "C:\Temp\"      'Default directory
   
    FileNameXls = Application.GetOpenFilename(filefilter:="Excel Files, *.xl*", MultiSelect:=True)
    If Not IsArray(FileNameXls) Then Exit Sub    ' User cancelled
   
    Application.ScreenUpdating = False
   
    For i = LBound(FileNameXls) To UBound(FileNameXls)
   
        Set wb = Workbooks.Open(FileNameXls(i))
        wb.Sheets(1).Range("G9:N20").Copy  'Range of cells that needs to be sum
        ThisWorkbook.Sheets(1).Range("G9:N20").PasteSpecial Paste:=xlPasteValues, Operation:=xlAdd, SkipBlanks:=True, Transpose:=False
        Application.CutCopyMode = False
        wb.Close SaveChanges:=False
       
    Next i
   
    Application.ScreenUpdating = True
   
End Sub

Groove Music to One Drive - Microsoft Community

rogerchamberlain say:

Groove Music to One Drive

How do I upload albums from my Groove music collection to One Drive? I'm using a Surface Pro 3. Have just begun to use Groove Music.
mikem132 say:

To expand slightly on the correct answer already provided and possibly avoid confusion.....

Groove Music is DRM (Digital Rights Management) music. It is protected content and has limited functionality. You can download it and play it on a computer or a phone and that's about it (for the tracks you download). You can stream Groove Music on phones, tablets or computers without downloading, too. In fact, the "4 device limit" of Groove is  ONLY for downloading tracks---streaming music does not have this limitation.

DRM music has two files, the actual .WMA sound file and a required license file. You can move your downloaded (rented DRM) Groove tracks to OneDrive, an MP3 player, etc. and they may appear on that device. However, they will not play. You'll be missing the license file. Sorry if this is too much information, but I've used DRM music from day one and have seen confusion countless times.

mikem132 say:

Roger:

 I'll give you the .50 cent explanation of this.

 Uploading to OneDrive is for music you own (ripped CDs, etc.). I would further say it is for music you own that is also NOT in the Groove catalog (like The Beatles, for example). Uploading your own songs to OneDrive that duplicates songs in the Groove Music catalog is not really necessary. You'd be streaming your stuff from OneDrive (available from any computer, phone or browser) OR streaming Groove stuff from their online servers using any computer, phone or browser. Unless you had some particular reason to upload some super-high audio file or something, it would be needless duplication.

 Albums you download are for offline use. Everything else is online streaming.

 Your "collection" in Groove is a little confusing. You can mark songs or albums as part of your collection, and they appear (while you are online in Groove) as part of your library on that (or any other) device. This is not exactly the same as downloading to your collection---songs can be part of your collection whether downloaded (for offline use) or not.

There are unusual cases where you might want to upload your own songs to OneDrive even if Groove has the same track in the catalog. Say you want to burn a CD of your own stuff to play in a rental car while on vacation someplace. You would not be able to burn tracks using Groove catalog "rental" tracks, but your own stuff is DRM-free and you could. Maybe far-fetched, though.

 Xbox Music used to have a "match" function that looked at their catalog and your own home library and "matched" tracks, creating a "cloud" version of your own music. That apparently is no longer part of Groove Music (there were a lot of complaints this matching thing really messed up libraries). Now the solution seems to be upload everything you own to OneDrive. You can also just go through your own library (stuff you own) and look for it in the Groove catalog and click "add to collection"---this just tags the online track as part of your "cloud" collection in Groove without uploading anything or downloading anything. Confusing enough?

mw613 say:

Excel random sample from rows of data

Column A contains a list of products. There are 500+ rows.

Column B may contain an exclusion character X

Column C contains a formula to randomly generate "Yes"or "No"

I want to randomly sample 50 products for testing, but to exclude any entries in column B.

What is the correct formula for column C?

I assume I have to set Calculation Options to Manual to avoid them recalculating??

mike h.. say:
A quick question before I test this, does your solution work 'top down' in that it will stop as soon as the count reaches 50? In which case it is unlikely to reach the last row.

Hi,

None of the above the code uses random numbers.

The code does this:-

1. Finds the last row in Colum A

2. Generates a random number between 2 and the number of the last row

3. Test if column B in that row has an X in

4. If no X then write Test in column C

5 Repeat steps 1 thru 4 until the word Test appears 50 times in col C

All rows from 2 to n are included. There is however a small bug which this version removes

Sub Testing_Times()
Columns(3).ClearContents
Dim FillRange As Range, LastRow As Long
Dim TheRow As Long
LastRow = Cells(Rows.Count, "A").End(xlUp).Row
Set FillRange = Range("C2:C" & LastRow)

Do
TheRow = Int(((LastRow - 1) * Rnd) + 2)
If UCase(Cells(TheRow, "B")) <> "X" Then
Cells(TheRow, "C") = "Test"
End If
Loop Until WorksheetFunction.CountIf(FillRange, "Test") = 50

End Sub
