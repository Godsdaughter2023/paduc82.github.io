Ignore any warnings and once installation complete, go to settings, themes, general, find the new theme, tap it and it will be applied to the phone.

Microsoft Access - Command button - Hyperlink - Microsoft Community

colinbashbash say:

Microsoft Access - Command button - Hyperlink

Just wanted to post this somewhere...

If you have a command button in Access with a hyperlink, everything will work normally unless the Caption on the button has a "#". So don't make the button text, "View Part #" or "View Account #"  or anything like that. Otherwise it will just tell you it can't get to the specified hyperlink.

Error Message:
Microsoft Access can't follow the hyperlink to <Hyperlink Name>. 

Please verify the destination.
