ronrosenfeld say:

The following VBA routine seems to do what you want, at least on your specimen data set.

It requires that your data start in A1, with no column headers, and that the area of data be bounded by at least one empty row, and one empty column.  The method of selection of the source data can be changed if necessary.

It makes use of Regular Expressions to split out the different segments of the last column, and also uses a Class module to define an object called "DATA".

As written, the macro places the results on Sheet4 starting at cell A1.  This can be easily changed to be placed on a different worksheet, a different range, a new worksheet, etc., depending on your requirements.  A header (label) row could also be included.

You will paste code into two modules as outlined below.

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Class Module.  Select the Class module (it will probably be called Class1), then hit F4 and change Name to DATA.  Then close that Properties window and paste the code below into the window that opens. (That class module should now be named DATA).

==========================================
Option Explicit
Private pFirst7 As Variant
Private pLastCol As String

Public Property Get First7() As Variant
    First7 = pFirst7
End Property
Public Property Let First7(Value As Variant)
    pFirst7 = Value
End Property

Public Property Get LastCol() As String
    LastCol = pLastCol
End Property
Public Property Let LastCol(Value As String)
    pLastCol = Value
End Property
================================

Next, again ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.

======================================
Option Explicit
Sub ParseLast()
    Dim vSrc As Variant
    Dim vRes() As Variant
    Dim rRes As Range
    Dim colData As Collection
    Dim cData As DATA
    Dim I As Long, J As Long, K As Long
    Dim S As String
    Dim V() As Variant
    Dim RE As Object, MC As Object
    
Set colData = New Collection
    
Set RE = CreateObject("vbscript.regexp")
With RE
    .Global = True
    .Pattern = "\d{4}/\d{2}/\d{2}[\s\S]+?(?=(?:\d{4}/\d{2}/\d{2})|$)"
    .MultiLine = True
End With

vSrc = Range("A1").CurrentRegion
For I = 1 To UBound(vSrc)
    ReDim V(1 To UBound(vSrc, 2) - 1)
    For J = 1 To UBound(V)
        V(J) = vSrc(I, J)
    Next J
    S = vSrc(I, UBound(vSrc, 2))
    Set MC = RE.Execute(S)
    For K = 0 To MC.Count - 1
        Set cData = New DATA
        cData.First7 = V
        cData.LastCol = MC(K)
        colData.Add cData
    Next K
Next I

'Results
ReDim vRes(1 To colData.Count, 1 To UBound(vSrc, 2))
For I = 1 To colData.Count
    Set cData = colData(I)
    For J = 1 To UBound(vRes, 2) - 1
        vRes(I, J) = cData.First7(J)
    Next J
        vRes(I, J) = cData.LastCol
Next I

Set rRes = Worksheets("Sheet4").Range("A1").Resize(UBound(vRes, 1), UBound(vRes, 2))
rRes.EntireColumn.Clear
rRes = vRes
rRes.EntireColumn.AutoFit

End Sub

===================================

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.
