marshall barton say:

Page related stuff in subreports is ignored because the main report is "in charge" of pages.

The Report Header section never repeats, so those two options can not do what you want.

A mechanism that is relatively easy to do is for each subreport to have a top level Group Header section with its RepeatAection property set to Yes.  If your subreports top level group header does not contain the info you want to repeat or if there are no group headers, create the top level group based on a constant expression such as =1 or ="MyPageHeader" and select Yes for Group Header.  Then set this new group header's RepeatSection property to Yes and put the controls you want to see on each page in the section.

For your first question, if you are using an email address ( like Gmail , Outlook.com etc.) to send spam emails to office 365 users, the office 365 Exchange Online Protection ( EOP) will detect the received emails with the mail headers which containing the senders email address and its IP address.
