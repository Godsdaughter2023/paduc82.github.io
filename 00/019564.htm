joeu2004 say:

(Critical errata to my previous posting, now deleted.)

Mason wrote:

I have a horribly complex and volatile spreadsheet.  It can take 20+ minutes to open in Excel because it appears that Excel is calculating as it is opening. Is there a way to prevent this?  I wish there was an option akin to "calc before save" that was "recalc when opening".

One way:  open it in Manual calculation mode.

To do that, double-click the Excel program icon (not the file icon) to start Excel with a new worksheet.  Click Formula, Calculation Options, Manual.  Then click File, Open to open the errant file in Manual calculation mode.

Then for future opens, you might put the following into the ThisWorkbook object in VBA:

Private Sub Workbook_Open()
Dim ws As Worksheet
For Each ws In Sheets
    ws.EnableCalculation = False
Next
End Sub

You might want to exclude any worksheet that you want to leave in automatic calculation mode and that is "safe" (not time-consuming) to calculate when the file is opened.

Ideally, you put all the volatile formulas in one or more worksheets, and the non-volatile formulas that depend on them in one or more separate worksheets.

Execute the Worksheet_Open function manually this first time.  Then you can change to Automatic calculation mode.

Also put the following into a new (normal) module:

Sub enableSheet()
ActiveSheet.EnableCalculation = True
End Sub

Sub disableSheet()
ActiveSheet.EnableCalculation = False
End Sub

Sub showSheet()
MsgBox ActiveSheet.Name & "    " & ActiveSheet.EnableCalculation
End Sub

Sub calcSheet()
Dim prev As Boolean
prev = ActiveSheet.EnableCalculation
ActiveSheet.EnableCalculation = False
ActiveSheet.EnableCalculation = True
If Application.Calculation <> xlCalculationAutomatic Then ActiveSheet.Calculate
ActiveSheet.EnableCalculation = prev
End Sub

The active worksheet recalculates in Automatic calculation mode only when transitioning EnableCalculation from False to True.

Save the workbook as xlsm (macro-enabled) or xls (compatibility mode) and exit Excel.

(Be sure the application is in Automatic calculation mode when you save.)

Then when you open the file, you can use alt+f8 to selectively calculate each worksheet, enable them, or show their worksheet calculation status.
