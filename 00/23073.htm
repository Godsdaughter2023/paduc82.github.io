scottgem say:

I see three main tables:

tblEquipment: to record info about the equipment

tluCalibrationType: A lookup table for each type of calibration

tblEquipmentCalibration: This will record the details of each calibration: EquipmentID, CalibrationTypeID, CalibrationDate, Cost, ProviderID

I'm guessing here that each piece of equipment can be calibrated for multiple values.  So you should have a record for each calibration value for each piece of equipment for each year.

qa guy commelec say: If you want to use it for scheduling then you need a field in the tblEquipmentCalibration to indicate if the next calibration is scheduled based only upon the calendar (say one year from last date of scheduled calibration) or one year from last calibration (actual calibration actions can slip).
