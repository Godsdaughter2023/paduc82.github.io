The way freeze pane works is below. Let me know if this is not the case in your case.

1. In View tab < Freeze Panes > There are 3 options, Freeze Panes, Freeze First Row and Freeze First column.
Clicking Freeze first row freezes first row and Freeze first column freezes first column as these say.

2, Choosing Feeze Pane freezes always one column up and one row left. Hence, if you are in Cell C5, it means row 4 is frozen and column B is frozen.

3. Select a cell in column A say A8. There is no column to left, hence no column to be frozen. But row 7 will be frozen.

4. Select a cell in row 1 say D1. There is no row above this, hence no row to be frozen. But column C will be frozen.

5. A1 is a special case. No row above this and no column to the left. It just puts the freeze pane in the middle of your sheet (the portion which is displayed on your computer, middle is calculated as per the display area)

Hi Steve,

Of the two questions you mentioned: 
1. There are some features are not available in Office Online, like macro, but template files like document.dotx should be able to open in Word Online. For Excel files, the Excel template(xltx file) cannot open in Excel Online. 

2. About assign read-only status, for files upload to SharePoint or OneDrive, the read-only is permission assigned to to users instead of files. For example, an Office 365 group has a SharePoint group site, and the group members will have edit permission to the site and the documents stored on the site. You could change their permission to read-only, then they will be able to download the files from the site and edit offline but cannot save any changes to the files online.

For more about Office 365 group: Learn about Office 365 Groups.

For more about permission levels: Understanding permission levels in SharePoint

The best practice to use document templates is set them as templates in the document library in SharePoint:

Click Add template to add the template you want to use in this document library. The Template 1 is a sample template I added to the library. When I create a new document, I could click New > Template 1, the Template 1 will open in Word Online and I can save a document based on the Template 1.

But none of my changes will save to the template. Excel template has a different behavior since Excel template(xltx files) cannot open in Excel Online. If you upload a xltx file as a template, it will create a new xltx file in the library when you click New and select the template. 

You could use a xlsx file as a workaround since it will open in Excel Online and save a workbook based on it.

tylor_955 say:

Excel sheet to automatically change color according to expiration dates

I am trying to manage a excel sheet of vaccination expiration dates, dates are in cells C7 through F161. I'd like the cells to automatically change color according to when they are due. 
I'd like red to represent all dates that are overdue (all vaccinations that are expired on today or before) id like vaccines coming due in 30 days to be yellow  and anything coming due 31 days or later to be green. I need the formulas to enter in.

Thank You!

ossiemac say:
  1. Select the range C7:F161
  2. Select Conditional format - > Manage Rules
  3. Click New rule button
  4. Select Use a formula to determine which cells to format
  5. Enter the formula     =C7<=TODAY()
  6. Click the format button
  7. Select the required tab (Font or fill depending on whether you want font or fill color)
  8. Select the red color
  9. Click OK and OK again and then click Apply
  10. Click New rule and repeat from step 4 to 9 above but this time the formula is

            =AND(C7<=TODAY()+30,C7>TODAY())      and format color is Yellow

 

    11.  Click New rule and repeat from step 4 to 9 above but this time the formula is

          

           =C7>TODAY()+30         and the color is Green

graham mayor say:
This seems very similar to a previous question - and the overriding question remains as to why you are using the deleted items folder as a place to store messages?

However if you only want to remove the attachments from the deleted messages that have them then the code would be

Sub DeleteAttachmentsFromDeletedMessages()
Dim olDeletedFolder As Outlook.Folder
Dim olDelItem As Outlook.MailItem
Dim i As Long, j As Long
Set olDeletedFolder = Session.GetDefaultFolder(olFolderDeletedItems)
For i = olDeletedFolder.Items.Count To 1 Step -1
    If TypeName(olDeletedFolder.Items(i)) = "MailItem" Then
        Set olDelItem = olDeletedFolder.Items(i)
        If olDelItem.Attachments.Count > 0 Then
            For j = olDelItem.Attachments.Count To 1 Step -1
                olDelItem.Attachments.Remove (j)
            Next j
            olDelItem.Save
        End If
    End If
Next i
Set olDeletedFolder = Nothing
Set olDelItem = Nothing
End Sub

As before, messages with blocked attachments will not be affected.
