Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oRng As Word.Range
Set oRng = Selection.Range
  If oRng.Characters.Last = Chr(13) Or oRng.Characters.Last = Chr(11) Then

  oRng.Text = Replace(Replace(oRng.Text, Chr(11), " "), Chr(13), " ")
lbl_Exit:
  Exit Sub

and I'd like for it to read "The value is 1,500,000".

rick rothstein, mvp - excel say:
=CONCATENATE("The value is: ",TEXT(C1,"###,###"))

You can shorten that by 2 characters as you only need the one # sign in front of the comma...

=CONCATENATE("The value is: ",TEXT(C1,"#,###"))

And, of course, you can shorten it even more by using an ampersand instead of the CONCATENATE function...

="The value is: "&TEXT(C1,"###,###")
