That's a fairly silly answer, I would have thought!  You have a customer complaining that the formatting looks bad, so you recommend removing all the formatting so the email looks utterly revolting?  That's progress, I guess, but I am not sure it's 'forward' progress.

LMP:  The problem here is that Apple Mail uses relatively simple HTML that does not support all of the complex formatting Word is capable of.

When pasting into mail, it normally converts "blank lines" into double paragraphs, which does make the format look double-spaced.  Make sure your original has only one return at the end of each paragraph.

Understand that the "paste" is really only going to transfer font formatting: the paragraph formatting does not carry across.

The margin is set by Mail, not Word, so you're stuck with it.

It would be better to use an attachment in this case, I sugest.
