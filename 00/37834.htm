using a wired connection - ethernet won't connect to internet - Microsoft Community

dianajoy say:

using a wired connection - ethernet won't connect to internet

I have a wired LAN 6 cable plugged into my laptop.  My Wi-Fi works well, but I need a little stronger connection for an on-line teaching job, so I'd like to use the cable when I'm teaching.  When I view my internet connections, regardless of whether Wi-Fi is connected, the Ethernet reads "unidentified network" and "no internet."  I have tried reverting to an older driver, updating the driver, turning off the Wi-Fi, and a bunch of other things - but to no avail.  Under Ethernet "status," it says it's sending packets, but not receiving them.  The LAN cable works fine with my husband's laptop, so I know it's not a cable or modem issue.  The IPV4 and 6 connectivity say "no access" (not really sure what that means, but maybe it's important.  When I "diagnose" the problem, it says Ethernet doesn't have a valid IP configuration.  I would greatly appreciate any help! - Diana

a. user say:

Hi Diana,

Disabled IPv4 and IPv6 protocols might cause issues with the Internet. The error that you're getting likely occurs due to an IP address conflict. For us to investigate your concern, we'd like to know if you've made any changes prior to the issue.

In the meantime, to further troubleshoot your Internet issue, we suggest performing these steps:

  1. Uninstall your network driver. To do so, refer to this link. Once done, restart your computer.
  2. Flush the DNS and request a new IP address from your Internet service provider. To do so, launch Command Prompt as an administrator, then run these commands one after another:
    Type netsh winsock reset and press Enter.
    Type netsh int ip reset and press Enter.
    Type ipconfig /release and press Enter.
    Type ipconfig /renew and press Enter.
    Type ipconfig /flushdns and press Enter. 

Let us know if you need further assistance.
