If you can't use conditional formatting, you may try this event macro assuming that the list is in the same sheet as your dropdown.

Put the code in the sheet where you want this function.

Private Sub Worksheet_Change(ByVal Target As Range)
  If Not Intersect(Target, Range("I4")) Is Nothing Then
    If Target.Validation.Type = xlValidateList Then
    Application.EnableEvents = False
    For Each c In Range(Mid(Target.Validation.Formula1, 2, 99))
      If Target.Value = c.Value Then
        c.Copy
        Target.PasteSpecial Paste:=xlPasteFormats
        Application.CutCopyMode = False
        Exit For
      End If
    Next c
    Application.EnableEvents = True
    End If
  End If
End Sub

Change "I4" to the cell with your dropdown.

Whenever the content (in cell I4) is changed the format of the corresponding cell in the list is copied to the cell (I4).
There is however no formatting of the items in the dropdown box itself, just in the cell where the value ends up.

wr8tt5 say:

I discovered what the issue was in my case. After upgrading to Windows 10, the VirtualBox virtual network adapter's network address had changed, however the VirtualBox dhcp server that serves IP addresses out for this virtual network was still configured with the old (pre-windows 10) network address. Therefore, the IP addresses being assigned to my VirtualBox virtual machines were for a network that didn't exist. Updating the dhcp server details with the new network address information now has my VM hostnames being registered and resolvable within Windows 10. This is likely the same issue with my VMWare VMs.

shane devenshire say:

Hi,

It depends on what you are trying to calculate.  Suppose your data is in A2:A7 as shown below under "Values":

Values Bins Freq Cummulative Freq
1 1 1 1
3 3 3 4
5 5 2 6
3
2
5

You manually enter the Bins you want, here they are in C2:C4.

1.  If you want to know the number of items in each bin you can use the formulas in D2:D4:

=FREQUENCY(A$2:A$7,C$2:C$4)

This is an array formula so you need to highlight the range D2:D4 first, then type the formula and press Shift+Ctrl+Enter.  The results are the number of items in each bin.  There is 1 item with a value of 1 or less.  There are 3 items with values greater than 1 and less than or equal to 3....

2. In the range E2:E4 the frequency formula is calculating the frequency of all items in that bin or lower, the formula is:

=FREQUENCY(A$2:A$7,C2)

This is NOT an array formula.  Just enter it into E2 and copy it down.  E2 shows the number of items that are 1 or less.  E3 calculates all the items that are 3 or less (note the difference)....
