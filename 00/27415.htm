phillip m jones say: Two possible ways if you have done neither then no you can't.
  1. You have a separate drive set up as Time machine and set up time machine. (recommended, you can do time machine on the same physical Drive, But the Drive has to be large enough. and you have to Partition when setting up your computer to begin with).
  2. In Word go to Word menu > preferences > save > choose Create (or Make) Backup and turn off auto save.


In method one:

  1. From Finder locate the file you just over-wrote, select.
  2. Open Time Machine
  3. At the right side you will see a series of tick marks start at the current active tick mark and move cursor up and choose a time line about an hour  or two before you overwrote.
  4. Click on that time line The window where the file that was over wrote will move back in time.
  5. Now make sure that file is selected.
  6. now push restore button.
  7. when display come back to a normal Finder display you will be asked to either replace or keep both. In this case choose keep Both. the titles will be rewritten in such away they will be same document but have different titles.

Method two:

Locate where Backup files have been written and pick out the oldest one of that file you think hasn't go any changes saved.

bob jones aka: cybertaz say:

You don't need to use the macOS method of creating keyboard shortcuts in Word. It has its own internal capability for doing so.

I'm not certain which you want, but there are 2 possibilities. First, go to Tools> Customize Keyboard within the program.

If you're looking to launch the Footnotes/Endnotes dialog, in the Insert Category assign the shortcut to the InsertFootnote command. Once the dialog appears select the Endnote option... There is no explicit InsertEndnote command to launch the dialog window.

If you want to immediately insert an endnote based on current settings the built-in keyboard shortcut is Command+Option+E which executes the InsertEndnoteNow command.

An oxymoron I know, but hear me out!

I have a 50 slide PowerPoint file.  6 of them are 'Ask one of us about X' type slides, 4 of them are other ads, 10 of them are people in the office, and 30 of them are products we sell.

Is there a way I can randomize the slides, but still have a specific 'section' come up every few slides?  For example, every 8 slides of products, have 1 of the Question Slides, for every 12 products (counting the 8 we just did) 1 Ad, and for every 20 a Person.  I don't care which person/ad/question or what order the products display - I simply want to control to make sure that we don't go say 15 or 18 products without a face or question or ad.

I've seen VBA code to randomize slides, so is this possible to do?  Thanks for all your help!

Amanda
steve rindsberg say:

First, the important stuff.  I *love* your icon.  Subtly ironic.  Or maybe ironically subtle.  Either way, superb.  ;-)

As far as what you're after, I think it should be possible with a bit of programming.

Let's narrow down what you're looking for:  Do you have some VBA/programming experience and just need help learning where to whack PPT with the VBAstick to make it do what you want?  Or are you hoping to find someone to write the code for you?  If the latter, must it be a freebie or do you want to hire someone? 

And some questions you'll want to answer no matter who does the coding:

Do you want the show to run continuously or just once?  If continuously, do you want it re-randomized each time through the show?

Do you want to pop in a Question slide every 8 slides exactly or do you want to randomize that a bit ... say every [random number between 6 and 10] slides?

Would it be acceptable for one (or more) of the 30 product slides to appear more than once or should a slide, once used, not appear again during the presentation?

I think I'd start by creating lists (collections or arrays in VBAspeak) of the different categories of slides and randomizing each.

Then display one slide at a time from the array of product slides, incrementing a counter for each slide.  When the counter reaches 8, show the first slide from the array of question slides; when the counter reaches 12, show a slide from the ad slides array and so on, then reset the counter and start over.
