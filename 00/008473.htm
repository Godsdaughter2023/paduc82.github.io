Envelopes in Word 2016 - Microsoft Community

Envelopes in Word 2016

doug robbins - mvp office apps & services (word) say:

Start with a Size 6 Envelope and click on the Add to document button and then, with the display of non-printing characters turned on, place the cursor just to the left of the Section Break so that the dialog launcher is enabled in the Page Setup section of the Layout tab of the ribbon

Click on the Dialog Launcher and then go to the Paper tab of the Page Setup dialog and reduce the Width from 6.5" to 5.5" and increase the height from 3.63" to 5.5"

Then, click on the pilcrow in the address area of the Envelope to display the borders of the Frame and then right click on the Border and then on Format Frame...

Then, in the Frame dialog, reduce the Width of the Frame from 5.5" to 5" and, if you want the address to appear higher on the envelope, increase the Height of the Frame 

If you want to get rid of the blank page after the envelope, place the cursor on that page and then change the size of the paper to a custom size of 5.5" x 5.5", and then go to the Layout tab of the Page Setup dialog and change the Section Start from "New Page" to "Continuous".

Then, it's probably a good idea to save the document as a template so that you can just create a new document from it, any time you want to produce an envelope of that size.

I found a solution to revert to an earlier version of Office 2016 Click-to-Run (7766.2060) in which the indexed search work.

  1. Open an elevated Command Prompt window
  2. At the command prompt, run the following command:
    cd %programfiles%\Common Files\Microsoft Shared\ClickToRun 
  3. Run the following command:
    officec2rclient.exe /update user updatetoversion=16.0.7766.2060

It activates the Office update window that will download the old update, close any open Office applications and start the previous update.

After upgrading to 7766.2060, indexing starts automatically.
