Microsoft changed the functionality of the Clip Art site more than a year
ago. If you're interested in the boring details you can search the Forums or
the Microsoft site for more info, but the bottom line is that each image now
has to be downloaded separately. There is no longer a 'Basket' because the
.cil (Clip Art Package) format has been abandoned. Each image is received in
its actual format (i.e., .jpg, .wmf, .png, etc.) into whatever you have set
as the default Downloads folder for the browser you're using.

When you download, choose the option to Save rather than Open. Then use
Finder to navigate to the Downloads folder & drag the images into the
respective Clip Art folders you want them in.

hansv mvp say:

Word does not automatically update cross-references.

 

One way to update them is to select File > Print. Word will display a print preview, and in order to do so update all fields in the document (not just cross references).

To be precise, this depends on a setting that is "on" by default: in File > Options, category Display, section Printing options, the check box "Update fields before printing" should be ticked.

 

Another way is to press Ctrl+A to select the entire document, then press F9 to update all fields within the selection.

chip pearson say:

Try code like the following:

 

    Dim C As String
    Dim R As Range
    Dim N As Long
    Set R = ActiveCell
    For N = 1 To Len(R.Value)
        C = Mid(R.Value, N, 1)
        If C Like "[a-z]" Then
            R.Characters(N, 1).Font.Color = RGB(0, 0, 255)
        ElseIf C Like "[A-Z]" Then
            R.Characters(N, 1).Font.Color = RGB(255, 0, 0)
        ElseIf C Like "[0-9]" Then
            R.Characters(N, 1).Font.Color = RGB(0, 255, 0)
        End If
    Next N

 

 

jim thomlinson say:

This should be close. It formats the active cell.

 

Public Sub ColourText()
    Dim lng As Long
    Dim rng As Range
   
    Set rng = ActiveCell
   
    For lng = 1 To Len(rng.Value)
        With rng.Characters(lng, 1)
        If IsNumeric(.Text) Then
            .Font.ColorIndex = 10
        ElseIf UCase(.Text) = .Text Then
            .Font.ColorIndex = 3
        Else
            .Font.ColorIndex = 5
        End If
        End With
    Next lng
   
End Sub
