considerate_guy say:

The first version of Windows Movie Maker (1.0) was included with Windows ME in 2000. I don't believe that you really want that version, nor is there a separate installer for that old version.

Windows XP released with Windows Movie Maker 1.1, but that was quickly updated to 2.0 and then 2.1.

Windows Vista released with Windows Movie Maker 6.0 (and also a less powerful 2.6 for those with an insufficient GPU to run it.)

That's where Microsoft switched gears. Rather than including Movie Maker with Windows, they decided to create a new version called "Windows Live Movie Maker" which was a separate "Windows Live Essentials" download. There were beta versions in 2009 and 2010. A more usable version came out in 2011.

Then in 2012, Microsoft dropped the "Live" branding and came out with the final version of Movie Maker 2012 which is the most recent version and is part of Windows Essentials 2012.

So when you say you want "Windows Movie Maker 1", what is it you really want? If you want the one one just before the latest update, you'd want Windows Live Movie Maker 2011. I think there are still download links for that older version, but I'm not sure where exactly. Also, contrary to what you stated, it doesn't have *more features*, it obviously has fewer than the latest 2012 version.

Or perhaps you mean the last "classic" version which would be Windows Movie Maker 6.0. If so, you can find an installer for Windows Movie Maker 6.0 here. That version has the more traditional timeline and the ability to overlap audio clips. It also has a 64-bit version where the latest version of Movie Maker 2012 is only 32-bit. It doesn't have the same support for the latest file types (e.g. .mp4) so you'd have to work around that if you used WMM 6.0.
