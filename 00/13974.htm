My Outlook calendar keeps deleting all my old appointments. How do I - Microsoft Community

turbo999 say:

My Outlook calendar keeps deleting all my old appointments. How do I stop this?

Original Title: Outlook calendar

Dear, am facing a problem with my outlook calendar it keeps deleting all my old appointments. For example all my appointment in the last December has been deleted automatically.

brian tillman [outlook mvp 2007-2019] say:

Well, sweetie, try this: right-click the Calendar folder in the Navigation Pane and choose Properties.  Select the AutoArchive tab.  Click the radio button that says "Do not archive itmes in this folder".  Click OK.

If you have archived the older events, they'll be in your Archive PST's Calendar folder.  Open that folder, change the view to a table view like List, select all the events by pressing Ctrl+A, then click and drag the selection to your main Calendar folder.  Release the mouse button and all yoru archived events will move back to the main Calendar.

If you don't see your archive folders in the Navigation Pane, click Tools>Options>Other>AutoArchive.  In that dialogue you can determine whether or not you have configured autoarchive.  If you haven't, of course my first paragraph won't apply; you've simply deleted that old data somehow and no one here can know how you did that.  If it's not in the Deteted Items flder, you're out of luck.  If AutoArchiving is enabled, however, and also enabled for the Calendar, you'll see a check box labeled "Show archive folder in folder list".  Checking this should allow the archive folders to appear in the Navigation Pane where you can try what I first suggested.

I get a visual basic error when launching Word - Microsoft Community

albertsabbas say:

I get a visual basic error when launching Word

Where do I find the normal.dotm file to rename it, as I have read that this may be the cause.
jay freedman say:

In Windows Explorer (the file manager, not the web browser), click in the address bar and type %appdata%\Microsoft\Templates, then press Enter. That folder should contain your Normal.dotm template.

Renaming Normal.dotm may or may not stop the error. If it doesn't help, and if your Word installation has any template-type add-ins, they can also cause the same error. Look in %appdata%\Microsoft\Word\Startup; if you find any files there, move them to a different folder. If that does stop the messages, then you can try moving one file at a time back into the Startup folder, and restarting Word to test the result.
