When you do a clean install of Windows (which is what I assume you mean by "formatted my Dell Vostro), you need to reinstall all of the hardware drivers.

If you used a Dell restore-to-fresh-from-factory method, then drivers for built-in hardware would have been included.

8. This time, turn Family Safety on by ticking the On, enforce current settings button.

9. Switch user from the Parent Windows user account to the child's Windows user account (login to his/her user account). Wait for about 2-3 minutes so that Family Safety will synchronize online.

10. After doing so, logout from the child and then re-log using the Parent account.

11. Repeat steps 1-2.

12. Click Manage settings on the Family Safety website, this will open again to the Family Safety website login screen. Sign in again and verify if changes happen to the child account. Also check if the settings reflects the same with local Family Safety (via Control Panel).

rbs-jos2500 say:

transfer Windows 10 contacts to Windows Live Mail

 Is it possible to transfer Windows 10's People App contacts to Windows Live Mail to create an address book.

***Post moved by the moderator to the appropriate forum category.***

a. user say:

Hi,

You can actually transfer Windows 10 contacts to Windows Live Mail using the steps below:

  1. Go to https://people.live.com.
  2. Under Your Contacts, click on Manage drop-down button.
  3. From the drop-down list, select Export contacts.
  4. Select Microsoft Outlook CSV as format to export.
  5. Import those contacts to your Windows Live Mail.
  • Go to Import and export email, contacts, and calendars with Windows Live Mail > follow the instructions under To import offline contacts into Windows Live Mail.

Let us know how it goes.

Hi Robert,

The options and features of Windows may vary depending on the build and version of the system running on your device. To isolate your concern, we'd like to ask the following:
