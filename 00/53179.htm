Actually pretty quick and easy.  Just open your C:\Users\Jens folder, and right-click the Documents folder.  On the Location tab, you can set the new location for it.  This works the same for Pictures/Videos/SkyDrive, pretty much everything.

'onest_abe say:

Yep, custom toolbars are the equivalent to the old quick-launch taskbar. It's now different because: any program you pin to the normal Windows 7 taskbar, and then right-click shows (using Notepad as an example) a list of the .txt files you have saved with Notepad. It's more interactive with programs, as opposed to the custom toolbars, which just produces the standard list when right-clicking shortcuts (i.e. cut, copy, paste, properties). My guess is, they didn't want to deal with complaints of people pinning a folder like "Windows" on the taskbar, and then having their computer freeze when they right-click it and it tries to produce a list of every file and folder contained within.

While I still recommend making a toolbar, as that is the most efficient; I will explain how easy it is to modify the Favorites, because you expressed interest in that.

 

  • Make a folder directly on your main drive (most likely C). Title it "Mytoolbar" (or as you wish) and put all the programs (shortcuts) you want in it. (You may also use the one you have already created. [Shift+Right-click on a folder adds "Copy as Path", but you have to delete the added quotes"..." after you paste it.] or you can go inside of the folder and press alt+d and copy the path there.)
  • Once you have it activated, right-click on "Favorites" in the start menu and click "Properties" and go to the "Location" tab.
  • Replace C:\Users\Douglas\Favorites with C:\Mytoolbar (<- placing the folder directly on the C: allows you to easily type the path)
  • When it asks you if you want to move the files, select "No"
  • If you want to change it back at any point, simply click "Restore Default" and once again answer "No" to the same question.

Nothing seems to be different when I revert it back to default :)

To change the name: Once you are completely finished, go back to "Properties" where you activated the "Favorites Menu"-- deactivate it press "OK" then go back and reactivate it. Now it should have changed from "Favorites" to whatever your folder is named.
