To change the default time-out setting for Internet Explorer, follow these steps:

1. Start Registry Editor.

2. Locate the following subkey:

HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Internet Settings

3. In this subkey, add a ReceiveTimeout DWORD entry that has a value of (<number of seconds>)*1000. For example, if you want the time-out duration to be 8 minutes, set the value of the ReceiveTimeout entry to 480000 (<480>*1000).

4. Restart the computer.

The service list comes from an outside source and could be in a variety of formats. We typically copy and paste them onto our letters and "fix" them as necessary. 

    I would like to automate this process as much as possible - especially the part of printing the envelopes. I've worked with macros quite a bit - mostly in excel but some in word as well. I'd prefer to append the envelopes to the document if possible (as opposed to just printing them) so that they can be reviewed and edited as necessary before actually printing.

    I have to assume that someone has already written something that will do what I'm looking for or at least something that can be adjusted to work. Any help or advice you can give me on this is greatly appreciate. 

    Thanks

    David

    doug robbins - mvp office apps & services (word) say:

    If you run a macro containing the following code when the address is selected, it will add an envelope with that address to the document.

    Dim strAddress As String
    strAddress = Selection.Text
    ActiveDocument.Envelope.Insert Address:=strAddress

    There are other attributes of the Envelope object that can be set such as the size of the envelope, the ReturnAddress to be displayed, etc. etc.

    Note however, you can only add one envelope to the active document.  To create multiple envelopes you would need to use:

    Dim strAddress As String
    strAddress = Selection.Text
    Documents.Add.Envelope.Insert Address:=strAddress

    Which will create a new document for each envelope (each time that it is run)
