Thank you for posting on Microsoft Community. We regret the inconvenience caused and will assist you in resolving the issue.

 

Before assisting further, did you check if the drive is shown in the File explorer or Device management window?

 

The drivers whose drive letter/name is not allocated will not be shown in the File explorer window. Unallocated drives are those whose drive letter/name is not allocated when partitioned, formatted or installed on the PC. You will be able to view the drive on your PC only if the drive letter/name is allocated.

 

If the drive is not visible in the File explorer window, check and assign a drive letter/name (if not assigned) manually from Disk management by following the steps below:

 

  1. Connect the HDD to the PC, press Windows logo + X keys on the keyboard and select Disk management option from the context menu. Check if the drive is visible in the window.

  3. After allocating a drive letter, right click on the drive and select Open. Check if you are able to open the drive in File Explorer.
  4. Close the Disk management window, open File Explorer (Windows logo + E keys on the keyboard) and check if the changes are effective.

    jasonm_1969 say:

    OK, I persisted with this and found the answer... a faulty printer driver seemed to be the issue. Through patience and a lot of reading, I learned that when you delete a printer and reinstall, Windows will generally reuse the old printer driver unless you remove this after you uninstall the printer.

     

     I followed these steps:

     

    1. Go to start menu >> select devices and printers
    2. Highlight the problem device, right click on the icon and select remove device
    3. Go back to start menu >> select devices and printers
    4. Right click on another device (Microsoft XPS Document Writer) and select printer properties.
    5. Select Ports and highlight the old Standard TCP/IP Port for the printer that you just deleted
    6. Select delete port. If you receive a message advising that the portis in use; reboot your pc and try again

    The next step invloves deleting the printer driver from the registry. To do this follow these steps:

     

    1. Go to start menu and type regedit into the search box field and open the Rgistry Editor
    2. Open the files in the following Order: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Print\Environments\Windows x64\Drivers\Version-3
    3. Locate the sub folder under Version 3 that has the name of the printer that you are deleting. This is the driver for the printer. Delete this file
    4. Close down the registry editor
    5. Reboot your pc

    The next step involves performing a search for the latest driver for your printer. Locate the driver save it and reinstall your printer creating a new TCP/IP Port.

     

    I hope this advice saves someone else the pain that I have gone through to get my printer to work.
