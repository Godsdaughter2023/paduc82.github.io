graham mayor say:
When you create a document from a template (File > New > Select template) it doesn't have any format until you save it.

There is no process by which macros from a template are automatically copied to a document. The macros remain in the template and as long as the document can see the template, the macros are available to the document.

If you want the macros in a macro enabled document, then you will have to copy them to the document from the VBA editor and ensure you save as DOCM.

charles kenyon say:
Reiterating what Graham has said, a new document created based on a template will not have the macros in the template also in the document. Since the template is attached to the document, the macros are available in the document.

The correct (intended) way to use a document template is File >  New or double-clicking on the template. Both of these produce a new document based on the template, without the macros in the document, but available to the document so long as it remains attached to the template.
