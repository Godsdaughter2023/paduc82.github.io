cmdrkeene say:

If UAC is turned off, then everything you run (and every program process, every thing you download, and every site you visit) is automatically running as administrator.  It's a very dangerous thing to do, because even using something simple like Spotify to listen to music could download viruses to your computer through the advertisements.   Normally you're protected because your programs aren't running with full admin access, but yours are since you turned off UAC.

The only ways to stop running everything as admin is to turn UAC back on, or create and use a standard limited account profile instead of an administrator account profile.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate.

So, there is no need to know or get a product key, if you need to reinstall Windows 10, you can use your Windows 7 or Windows 8 product key or use the reset function in Windows 10.

If you are prompted to enter a product key, click 'I don't have a key' and 'Do this later'

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

To fix the DHCP service issue, perform these steps:

Method 1: Boot the computer in safe mode with networking first if you have a wired (DSL/Cable) connection and check if it helps.

a. Restart your computer.
b. When the computer starts you will see your computer's hardware being listed. When you see this information, start to gently tap the F8 key on

http://windows.microsoft.com/en-US/windows7/Start-your-computer-in-safe-mode

Method 2: Check the status of the DHCP client service in services.
a. Click on start and then type services.msc in the start search box and press enter.
b. Look for DHCP client service.
c. If it is stopped, try to start and copy any error.

Method 3: Verify that the Dependency Service / components are running

To verify that the three dependency components are running. Follow the steps below:
a.  Click Start, Run and type DEVMGMT.MSC
b.  In the View menu, click Show hidden devices
c.  Double-click Non-Plug and Play drivers section
d.  Double-click the entry AFD, and click the Driver tab
e.  Set the Startup type to System.
f. Start the service. Note down the error message if any.
g. Similarly start the three other drivers namely:
• Ancilary function driver for winsock
• TCP/IP Protocol Driver
• NetBios over Tcpip
h. Close Device Manager and restart Windows.
