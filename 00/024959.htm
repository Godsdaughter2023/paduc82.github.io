Are you using Microsoft Exchange?  If so, there are lots of third-party tools out there for e-mail archival and searching.  I assume you need multiiple users to search a common store of e-mail, so in that case the default search capabilities of Outlook (restricted to the data stores loaded for a particular user) may not be desirable.  That would require a single archive .pst, which is not designed to be shared by mutliple users.
