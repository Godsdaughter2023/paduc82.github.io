It sounds like you did it by applying local formatting rather than creating the correct formatting in your master. Local formatting is always more work to revise later, which is why I recommend using the master to begin with.

Here are steps to fix up what you've got:

  1. In PowerPoint with your presentation open, choose View>Outline View. The left-hand pane expands to show the outline view of your presentation.
  2. Right-click in the left-hand Outline pane and choose Show Formatting.
  3. Select the bulleted paragraphs in your notes, right-click and choose Paragraph to change the distance of the bullet from the text.
  4. Change both amounts in the Indentation section to the same amount to move the text closer to the bullets.
  5. Move to the next slide and repeat steps 3 and 4.

                f = True
            Case ")"
                f = False
            Case ","
                If Not f Then
                    Mid(s, i, 1) = " "
                End If
        End Select
    Next i
    Modify = s
End Function

The Boolean variable f is true when you're between parentheses, false otherwise.

Modify("logn(x,2),1,100") will return "logn(x,2) 1 100"
