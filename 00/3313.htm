    How can I use the Insert>Object>Insert Text File feature to - Microsoft Community

    giselaineballantyne say:

    How can I use the Insert>Object>Insert Text File feature to insert several files at one time in Word 2010?

    I did it by accident once and can't seem to do it again.  Rather than inserting only one text file at a time, it is possible to use the Insert>Object>Text File feature to insert several files at a time.  I can't remember if they were bookmarked paragraphs or individual document, but I did it a few times one day and now can't remember how I did it. 
    jay freedman say:
    After you click Text From File and the dialog opens, hold down the Ctrl key while you select several files. Their names will all appear in the File Name box, with quotes around each name. When you click the Insert button, all the selected files will be inserted.

    It's very difficult to control the order in which the files get inserted in the document. The best way is this:
    1. In the dialog, instead of clicking the main part of the Insert button, click the arrow at its right end and choose Insert As Link.
    2. In the document, press Alt+F9 to show the field codes of the links so you can see the file names. Each field looks something like { INCLUDETEXT "C:\\Users\\MyName\\Documents\\Doc1.docx" }.
    3. Select a field and drag it to the position you want, then drag another, etc. until they're in the right order.
    4. Select all the fields at once and then press Ctrl+Shift+F9 to unlink the fields, turning them into plain text.

    Remember also that a the a GPS system isn't connected to the same satellites all of the time, even a 12-channel system picking up between six and eight satellites on average will drop out of view of some satellites and into the range of others, so it is constantly recalculating its position from different data. If you are getting signals from less satellites, the 'jumps' become more noticable because the position is being calculated from less data.
