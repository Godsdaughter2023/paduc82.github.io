I have an idea that might help.  I have written some steps below, with the example text "I am fine" to explain my idea:

 

[1]  Start a new blank Word document.

[3]  Select the entire text for "I am fine".

 

[4]  Change the entire text "I am fine" to have condensed spacing, by 1 pt.

 

[5]  Select the letter "f".

 

[6]  Change the spacing on the letter "f" to be expanded by 0.5 pt.

I don't believe you can, at least not so they show up as expected in the animation pane. 

You can use plain old rectangles instead of action buttons; if you put text in the rectangle (select it, start typing) the text will appear in the animation pane.  In fact, you can select an action button and do the same.  Delete the text later.
