gerald vit say:

Hi EllenMoss,

Those icons will appear when the Desktop.ini file exists in one or more of the following folders:

  • C:\Users\<user profile>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
  • C:\Users\<user profile>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs
  • C:\Users\<user profile>\AppData\Roaming\Microsoft\Windows\Start Menu\

The desktop.ini file is a file that determines on how a folder is displayed by the system. You can delete the file but once you make changes to a folder, it will be created again. Rather than deleting it, I would recommend that you hide the file instead. To do this, please follow the steps below:

  1. Type-in folder options in Cortana.
  2. Click on the View tab.
  3. Tick on the check box beside Hide protected operating system files (Recommended).
  4. Click on the radio button beside Don't show hidden files, folders, or drives.
  5. Click OK to exit.

If you have additional questions, or if the issue persists after the troubleshooting steps above, please let me know by replying to this thread. This way, I'll get notified and respond back to you as soon as I can.

My Bluetooth keyboard and/or mouse doesn’t wake the system

I can wake Windows 10 from sleep with a wired mouse or keyboard, but I’m not able to do it with my Bluetooth mouse or keyboard.  Is there a way I can do this?
ariel aba say:

Hi Meliton,

Most laptops (or notebooks or tablets) are not designed to allow you to wake them by a Bluetooth mouse or keyboard.  

Some device manufacturers do allow Bluetooth devices to wake the system.  If the Bluetooth radio’s property page in Device Manager has a Power Management tab, then the Bluetooth radio can support “remote wake”. If no Power Management tab exists, the radio might support wake, but it is unlikely.  You may need to check with your laptop manufacturer to see if your device supports this feature.

If your device does support remote wake, advanced users (or your device manufacturer’s support agent) may need to make changes to the registry to enable this feature.  Advanced users can find details for enabling remote wake at https://msdn.microsoft.com/en-us/library/windows/hardware/dn133844%28v=vs.85%29.aspx.
