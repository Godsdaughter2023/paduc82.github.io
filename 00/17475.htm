john korchok say:

You can create a shared network folder, then set that as the Workgroup Templates folder for all users. Here are the steps:

  1. In Word, choose FILE>Options.
  2. Click on the Advanced tab.
  3. Scroll down to the General section and click on File Locations.
  4. Click on Workgroup Templates, then on the Modify button.
  5. Navigate to the shared network folder and select that location. OK Out.

Choose a folder for Workgroup Templates from Word 2013/365 automatically creates a Document Themes folder inside the network folder. This is where you should store your custom theme. Custom templates go in the parent folder that you set in Word.

Once you have set up each computer with Workgroup Templates, each user will see your custom theme at the top of the list when they click on the DESIGN>Themes button.

ruwim.b say:

All media files, i.e. image files (.jpg, .png), video files (.mp4) are saved on Cloud servers for a period of at least 2 years. This is the same period for which chat history is kept saved on the Cloud.

 

Other files are saved for a period of 30 days only. The same is the case with the video/audio files created using the build-in call recording.

 

You can try this new Chat Export feature.

 

https://support.skype.com/en/faq/FA34894/how-do-i-export-my-skype-files-and-chat-history

 

This feature will allow you to download and save all the chat history and files currently available on the Cloud, i.e. up to 2 years old history. All available files will be downloaded to the \media folder, but with some cryptic names corresponding to their ID.

 

For each downloaded file there will be available an associated .json file. These files are simple text files, and the name of the corresponding downloaded file might be available in this file. Unfortunately there is no other easy way to identify these files.

No Sound When Closing Word - Microsoft Community

bhavfish say:

No Sound When Closing Word

Word 2016 does not make a sound when I close it - can I change this please?

jay freedman say:

The following version of the macro plays the sound whenever you close a document that contains unsaved changes. That includes closing a document by clicking the red X in the title bar, which also closes the window. If there are two or more open documents with unsaved changes, you'll get the sound for each of them.

Sub AutoClose()
    If Not ActiveDocument.Saved Then
        Beep
    End If
End Sub

If you want the sound only when you're closing the last open document, use this instead:

Sub AutoClose()
    If Windows.Count = 1 And Not ActiveDocument.Saved Then
        Beep
    End If
End Sub

Suppose you add a checkbox rather than a button - it might be easier.  Here's how you might do it on the pc:

Add a checkbox and then right click it and choose Format Control, in the Link cell box select or enter the address where you want the results.

joe men say:

Hello jlee77,

The error code 8000000e is usually accompanied by E_ILLEGAL_METHOD_CALL description. This error usually occurs with a device that is unable to get approval from the user. But in your case, you are unable to get permission to uninstall the app in question which is most likely an issue with the phone's system not able to correctly recognize the installation path of the app that you're trying to uninstall.

You can resolve your concern by moving the app to the internal storage or SD memory card of your phone.

If the issue persists after moving the app, reformat your SD card by doing the following:

  1. Connect your memory card to a PC with a USB reader that has a microSD card slot. If your card reader or PC has a regular SD card slot, you can use a microSD card adapter.
  2. Use File Explorer and browse to the removable storage on your computer, right-click the memory card, and select Format.
  3. For 4-32 GB cards, select the file system FAT32 and allocation unit size 32 kilobytes and press Start. For 64-200 GB cards, select the file system exFAT and allocation unit size 128kB.
  4. Press OK to confirm the data loss warning message. When the formatting process is complete, press OK.
  5. Restore only the data that is compatible with your phone (JPEG or MP4 in the photos folder or MP3 in the music folder).
  6. Right-click the memory card, select Eject, and wait until it is safe to remove the card.
  7. Insert the memory card into the memory card slot in the phone until it locks into place.

Important: Before formatting your memory card, make sure you have a backup of the data as it is lost during the formatting.

Let us know what happens after.
