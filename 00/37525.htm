In case of any incompatible drivers or application creating conflicts at the time of System Startup, then system fails to load all basic Widows 10 features. End result, it will throw an error message on to the screen stating, Critical Error. Your Start Menu isn't working. We'll try to fix it the next time you sign in. In this situation, it won't allow you to access your desktop nor any Windows 10 features, except Task Manger.

To fix this critical error, Shutdown your PC completely and then restart it after a while. Another way to fix this issue is by booting your PC into safe mode with the same User Profile. To boot your system in Safe Mode,

  1. Restart your PC. When you reach the sign-in screen, hold the Shift key and select the Power button, and then select Restart.

  2. After your PC restarts, select Troubleshoot > Advanced options >Startup settings > Restart. After your PC restarts, you should see a number of options. Press 5 or F5 for Safe Mode with networking.

  3. When you're on to the Safe Mode, check whether basic features of Windows work fine there. Like Start Menu, Notification bar (I.e. in your system tray) and some other things. If the error message doesn't appear there, then it's an indication that you can login to your profile in normal mode without any issues.

If you see the error message still appearing, then follow the below steps to see if it fix the issue. These steps will repair any missing or corrupted system files, if they causing any issues.

Step 1: Use System File Checker to repair damaged Windows files

Use the System File Checker tool to repair damaged Windows files. To do this, follow these steps:

  1. Press Windows logo key + X, select Command Prompt(Admin).
  2. Run the "sfc /scannow" command by copy/pasting it into the Command prompt(Admin) and hit Enter key.
  3. Once it complete, try to install updates again to see if they work correctly

For more information about the System File Checker tool, see Use the System File Checker tool to repair missing or corrupted system files.

Step 2: Run the DISM tool

If the problem continues, use the Deployment Image Servicing and Management (DISM) tool to fix corrupted system files. To do this, follow these steps:

  1. Press Windows logo key + X, select Command Prompt(Admin).
  2. Run the "DISM.exe /Online /Cleanup-image /Restorehealth" command by copy/pasting it into the Command prompt(Admin) and hit Enter key
  3. Once it complete, try to install updates again to see if they work correctly.
