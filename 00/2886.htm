Okay, I think this macro will do what you want. Note that I modified it to search Column E and Column K only. Now to be clear what my previously posted macro and this macro does is it only makes sure the row number is consistent within the formulas themselves and does not check that the row number the formula is on is the row number for the cells referenced within the formula itself. That is, the formula can be on Row 100, but if all the cell references inside the formula are for Row 2, then the formula is judged as okay. If the row number the formula is on should be the row numbers referenced by the cells within, then let me know and I'll change my code to do this (it's a fairly simple change... I just didn't see the requirement spelled out in your previous posts, although now I think your latest message may be hinting at it). Because of more general method employed by my macro (row the function is on is not coordinated with the row its cells reference), this leads to the possibility of a false negative for Column K formulas. I use the first cell reference within the Column E formula to set the row to be checked against for the other cell references; so, if the first row reference is the incorrect one, the that formula is flagged; however, when looking at the Column K formulas, I have no way of know which row reference in the Column E formula is correct (I only know one or more of them don't match)... so if the first one in the Column E formula is the incorrect row reference, then the Column K formula won't match it and that Column K formula will be flagged. IF you tell me that the row the formula is on should also be the row references for the formulas cell references, this problem goes away... so definitely let me know if this this the case. Anyway, here is the macro as I described it above (with the possibility of it producing a false hit for Column K formulas)...

Sub CheckCellReferences()
  Dim Arrow As Long, Link As Long, RW As Long, Addr As String, Msg As String
  Dim Cell As Range, CurrentCell As Range, OriginalSheet As String, OriginalCell As String
  Application.ScreenUpdating = False
  OriginalSheet = ActiveSheet.Name
  OriginalCell = ActiveCell.Address
  For Each Cell In Intersect(ActiveSheet.UsedRange, Columns("E").Cells)
    Set CurrentCell = Cell
    If CurrentCell.HasFormula Then
      CurrentCell.ShowPrecedents
      Arrow = 1
      Link = 1
      RW = 0
      Do
        CurrentCell.Parent.Activate
        CurrentCell.Activate
        If CurrentCell.NavigateArrow(True, Arrow, Link).Address = CurrentCell.Address Then Exit Do
        If RW = 0 Then RW = CurrentCell.NavigateArrow(True, Arrow, Link).Row
        If RW <> CurrentCell.NavigateArrow(True, Arrow, Link).Row Then
          CurrentCell.ShowPrecedents Remove:=True
          Msg = Msg & CurrentCell.Address(0, 0) & vbLf
          Exit Do
        End If
        Link = Link + 1
        On Error Resume Next
        Addr = CurrentCell.NavigateArrow(True, Arrow, Link).Address
        If Err.Number Then
          Arrow = Arrow + 1
          Link = 1
          Err.Clear
        End If
        On Error GoTo 0
      Loop
      CurrentCell.ShowPrecedents Remove:=True
    End If
    Set CurrentCell = Cell.Offset(0, 6)
    If CurrentCell.HasFormula Then
      CurrentCell.ShowPrecedents
      Arrow = 1
      Link = 1
      Do
        CurrentCell.Parent.Activate
        CurrentCell.Activate
        If RW <> CurrentCell.NavigateArrow(True, Arrow, Link).Row Then
          CurrentCell.ShowPrecedents Remove:=True
          Msg = Msg & CurrentCell.Address(0, 0) & vbLf
          Exit Do
        End If
        Link = Link + 1
        On Error Resume Next
        Addr = CurrentCell.NavigateArrow(True, Arrow, Link).Address
        If Err.Number Then
          Err.Clear
          Exit Do
        End If
        On Error GoTo 0
      Loop
      CurrentCell.ShowPrecedents Remove:=True
    End If
    Worksheets(OriginalSheet).Activate
    Range(OriginalCell).Activate
  Next
  Application.ScreenUpdating = True
  If Len(Msg) Then
    MsgBox "Check these cells..." & vbLf & vbLf & Msg
  Else
    MsgBox "All formulas are fine!"
  End If
End Sub

To print a single section, click in the Pages box on the Print dialog and type the letter s followed by the section number. For example, to print the third section (your Page 3 and Page 4), type s3 in the box.
