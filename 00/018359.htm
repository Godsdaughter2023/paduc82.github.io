How to Extract Letters from an alphanumeric string - Microsoft Community

sam.pat say:

How to Extract Letters from an alphanumeric string

Hi All,

How can I extract alphabets from an alphanumeric string?

For eg:

344-53B01 I want to extract B from this string

A11-088B I want to extract A and B from this string.

Thanks in advance

rick rothstein, mvp - excel say:

Here is function that you can use, either call it from within your own code or use it as a UDF (User Defined Function) directly in a worksheet formula...

Function LettersOnly(S As String) As String
  Dim X As Long
  For X = 1 To Len(S)
    If Mid(S, X, 1) Like "[!A-Za-z]" Then Mid(S, X, 1) = " "
  Next
  LettersOnly = Replace(S, " ", "")
End Function

If you calll it from within your own code, simply pass the text string you want to parse as the argument; if you call it as a UDF, then just pass in the cell reference. For example...

From you own code
------------------------------------
Txt = "A123B-C45D"
MsgBox LettersOnly(Txt)

From a worksheet
------------------------------------
=LettersOnly(A1)

To work as a UDF, the function must be installed in a standard Module (Insert/Module from the VB editor's menu bar)
