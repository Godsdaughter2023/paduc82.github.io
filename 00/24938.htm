Discovered what seems to be the solution, hoorah. For those of you suffering a similar problem I did the following:

1) Deinstall the OneNote App using "get-appxpackage *microsoft.office.onenote* | remove-appxpackage" - this requires an administrative powershell command (started cmd.exe as an administrator and then ran powershell)

3) Reinstalled OneNote from Windows Store

Private Sub Workbook_BeforeClose(Cancel As Boolean)
Application.OnKey "{F9}", ""
End Sub

Private Sub Workbook_Open()
Application.OnKey "{F9}", "CheckVal"
End Sub
