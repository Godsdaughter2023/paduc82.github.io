brian tillman [outlook mvp 2007-2019] say:

Outlook doesn't have a folder named "Spam" or "junkbox".  Outlook uses a folder named "Junk E-mail" for messages that its own junk filter flags.  If you are seeing a folder named "Spam" or "junkbox", then you are probably using an IMAP account and your server is providing that folder and moving messages to it or you are using an antispam add-in which is providing that folder and moving messages to it.  In neither case, then, is this an Outlook issue.

I would recommend that you visit your mailbox using a web browser and look on your mail provider's server for junk mail settings and adjust them.
