If you have applied user-level security to an Access database, the users and groups are stored in a workgroup file (.mdw). If that workgroup file is not available on your new computer, you probably won't have sufficient permissions to use the database. "C:\Program Files\Microsoft Office\Office12\MSAccess.exe" "C:\Databases\MyDatabase.mdb" /wrkgrp "C:\Databases\MyWorkgroup.mdw"
