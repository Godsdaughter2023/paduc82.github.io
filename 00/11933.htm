Rounding numbers in Excel - Microsoft Community

pa311 say:

Rounding numbers in Excel

Hello,

I know two ways to round numbers in Excel, either using =Round(cell,#decimals) or use the Increase/Decrease Decimal buttons on the Home tab ribbon.  Depending upon the actual raw numbers, I can get different values, especially when I need to use the values in formulas.

Note, we work with large values ($$$).

How can I explain to others which is correct?

Thank you.

Paul

hansv mvp say:

Let's say you have a value 12,345.6789 in cell A1.

The result of the formula =ROUND(A1,2) in cell B1 be 12,345.68.

But if you use the Increase/Decrease Decimal buttons on cell A1 to display 2 decimal places, the stored value of cell A1 is still 12,346.6789 even though the displayed value is 12,345.68.

Now enter the formula =100*A1 in A2, and =100*B1 in B2.

The result in A2 will be 1,234,567.89 for even though you didn't see the last two digits in A1 anymore, they were still present.

The result in B2 will be 1,234,568 for the formula in A2 has removed the last two digits.

Which one is correct? That depends on what the 'rules' are, there is no single correct answer. If the business rule says that all intermediate values should be rounded to cents, you should use the ROUND function. But if it says that only the end result should be rounded to cents, you should use ROUND for the final result, but not for the intermediate formulas...
