don varnau say:

Hi,

To have IE open with add-ons enabled...
Check after each step.

1. Creating a new IE icon for the desktop and Start Menu may help.
Internet Explorer 8 starts in "No Add-ons" mode when you use the desktop icon to start Internet Explorer:
http://support.microsoft.com/kb/976915
The Internet Explorer icon is missing from your desktop:
http://support.microsoft.com/kb/945402

2. From Start> Run type in iexplore.exe then hit Enter.

3. Then check this... Internet Options> Advanced> check "Enable third-party browser extensions."

4. From Start> Programs> right-click the Internet Explorer icon> Properties>
Shortcut. If the Target is
"C:\Program Files\Internet Explorer\iexplore.exe" -extoff
edit out the  -extoff  (leave the quotes) then OK your way out of
Properties.

There might be more that one IE icon- one at the top of the Start Menu,
another in Programs, etc. If the issue persists, check all IE icons.

If IE continues to run in no-add-ons mode, work through add-ons troubleshooting to identfy the add-on that's causing the problem.

IEBlog  Troubleshooting and Internet Explorer’s (No Add-ons) Mode:
http://blogs.msdn.com/ie/archive/2006/07/25/678113.aspx

- - - - - - -

Re: Missing updates...
1. Right-click Computer/My Computer and check Properties. Is XP updated to SP3?

2. From Start> Settings> Control Panel> Automatic updates> verify that you've selected one of the options other than "Turn off Automatic Updates."

3. From Internet Explorer> Tools> Windows Update, go to Windows Update, allow the scan and install all Critical or Important updates.
