Hi John,

Considering "a very slow internet connect", please check whether the Office folder has been download successfully. Normally, the Office folder you have downloaded is about 1.10GB. If not, you may need to download it again to the local path on your PC and paste it to the shared folder on the server. And then let the users run the command to install it. To do that:
1. Download Office 2016 Deployment Tool on your PC: https://www.microsoft.com/en-us/download/details.aspx?id=49117

2. Double-click OfficeDeploymentTool.exe, and then extract the following files to the local path on your PC(e.g. C:\ODT ).
 •Setup.exe
 •Configuration.xml

3. Open the Configuration file with Notepad, and then paste the following text to the Configuration file.
 <Configuration>
   <Add SourcePath="C:\ODT" OfficeClientEdition="32" >
     <Product ID="O365BusinessRetail">
       <Language ID="en-us" />
        </Product>
   </Add> 
 </Configuration>

4. Open a command prompt as an administrator on your PC and input the following command. To do this:
 Click Start>In the Start Search box, type cmd> Right-click the Command Prompt, select Run as administrator>Input cd c:\ODT and press Enter> Input setup /download to download the Office package.
 
5. After it is downloaded, check whether it is about 1.10GB. If so, paste it to the shared folder on the server. And then let the users run the following command and check the outcome.
\\server\share\setup.exe /configure \\server\share\Configuration.xml

Click the Tab of the first sheet, then hold Shift while you click the Tab of the last sheet. Once you've selected all sheets click the Select Sheet button, then change the font size.

Keep in mind, though, that doing so may very well cause chaos in the readability & layout of the sheets... Any rows or columns whose height/width have been manually set will not automatically adjust to accommodate the new font size. As a result, text content may not fully display or may extend further than expected & values will appear as #### until you repair all the sheets.
