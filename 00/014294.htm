Find email address within a Word document - Microsoft Community

barb reinhardt say:

Find email address within a Word document

I am trying to find all references within a word document that are email addresses ending with @mycompany.com.   I have a Word Range that can find the @mycompany.com piece, but I need to get the whole email address listed before the @ sign.  Can someone help with the word code to get to this?  

Thanks,

Barb Reinhardt

graham mayor say:

You could use the following which will highlight all the appropriate e-mail addresses. If you remove the italicised line and re-activate the commented out lines, the macro will instead copy the addresses to a new document.

Sub ExtractEMailAddresses()
Dim oDoc As Document
    'Dim oNewDoc As Document
    'Dim oRng As Range
Dim hLink As Hyperlink
    Set oDoc = ActiveDocument
    oDoc.Save

    'Set oNewDoc = Documents.Add
    Options.AutoFormatReplaceHyperlinks = True
    oDoc.Range.AutoFormat
    For Each hLink In oDoc.Hyperlinks
        If InStr(1, hLink.Address, "@mycompany.com") Then
            hLink.Range.HighlightColorIndex = wdYellow
            'oNewDoc.Range.InsertAfter hLink.Range & vbCr
        End If
    Next hLink
    'oNewDoc.Paragraphs.Last.Range.Delete
    'oDoc.Close wdDoNotSaveChanges
    'oNewDoc.Range.Sort
    'oNewDoc.Range.Find.Execute FindText:="(*^13)@", _
     MatchWildcards:=True, _
     ReplaceWith:="\1", _
     Replace:=wdReplaceAll
lbl_Exit:
    Exit Sub
End Sub

macropod say:

You can find the whole email addresses via a wildcard Find, where;

Find = [! ^13^t^l]@\@mycompany.com

No code required.

Note: Graham's code will only find email addresses that are formatted as hyperlinks - mine finds them even when they aren't. The Find expression can, of course, be incorporated into a macro. For example, the following will highlight your email addresses:

Sub Demo()

Options.DefaultHighlightColorIndex = wdYellow

With ActiveDocument.Range.Find
  .ClearFormatting
  .Text = "[! ^13^t^l]@\@mycompany.com"
  .Replacement.Text = "^&"
  .Replacement.ClearFormatting
  .Replacement.Highlight = True
  .Forward = True
  .Wrap = wdFindContinue
  .Format = True
  .MatchCase = False
  .MatchWholeWord = False
  .MatchAllWordForms = False
  .MatchSoundsLike = False
  .MatchWildcards = True
  .Execute Replace:=wdReplaceAll
End With
End Sub
