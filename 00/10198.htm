This problem has actually been in media player all along but it may have been fixed or worked around in the WM 5.0 SKU3 that was on the Dash etc.  This never worked right on the Motorola Q as far as I know, or any other landscape phone.

 

My guess is that the media player got some work done for WM 5.0 SKU3 but it introduced some other problems so they reverted to the older version of it, forgetting that there was this issue on landscape phones.

 

There's a possible solution, you can try adding HKLM\System\GDI\Rotation\LandscapeMode=0 to the registry.  Doesn't seem to be working for me in the emulator but it may work for you. 

 

I don't think this is likely to be fixed before the next SKU of Windows Mobile.
