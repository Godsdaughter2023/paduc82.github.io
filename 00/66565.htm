DMZ
  1. Open web browser.

  5. Check the box next to Default DMZ Server.
  6. Enter the Xbox’s IP address into the boxes.
  7. Click Apply.

 

Disable DMZ

  2. Login to router by entering http://192.168.0.1 into the address bar

This message reports one bug and makes one feature request for managing 3D Stereo Pictures in Windows 7.  I've been taking 3D photos for many years, and was excited to see the new features in Windows 7 for digital pictures.  Great viewer, explorer, manager, etc.  However, there is one tormenting bug that most people won't consider important unless they're using Windows 7 for 3D stereo photos. 

 

THE BUG:

The problem is that from time to time, Windows Explorer adds extra white space above a picture.  As I scroll through my pictures, using 'Extra Large Icons', Explorer will suddenly begin adding extra white space, sometimes, but not other times.

The configuration I use is to turn off the navigation pane, set view options to 'Extra Large Icons', and narrow the window enough to place two Explorer windows side-by-side, with a single column of pictures in each window.

Here's why it's a problem in 3D.  I use two Explorer windows side-by-side in order to view my pictures in 3D.  This is important when unloading my two cameras and managing the pictures.  So long as each window presents their pictures like each other, everything is fine.  But when just one window starts adding white space, now I have to manually fudge with icon positions and it significantly slows down the process.  It makes managing 3D on Windows 7 as tedious as before Windows 7 existed.

 

WORK AROUND:

There are two ways to solve it for a single picture.  By simply sliding the viewer options for 'Extra Large Icons' down and back up, the white space is removed.  Also by highlighting each of the two pictures, scrolling them up off the screen, and then back down into view, they once more become synchronized.  However, once Windows 7 begins displaying this bug, I have to make the manual adjustments on nearly every photo.  It's terribly, unacceptably tedious.

 

BUG FIX:

The real solution is for Microsoft to fix the bug which causes Windows Explorer to unnecessarily display extra white space.  The bug is inconsistent, and I haven't been able to discover the circumstance that causes it to begin.  Usually it happens after viewing more than a dozen pictures.

 

FEATURE REQUEST:

In addition to the bug fix, Windows needs a new feature.  Windows Explorer needs side-by-side, synchronized windows like Excel.
