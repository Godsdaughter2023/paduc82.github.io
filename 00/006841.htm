Your computer is permanently licensed for the edition of Windows 10 that you had. Insider builds expire, but your computer's license for Windows 10 does not. That means you can always reinstall Windows 10 and it should auto-activate, as long as you install the edition that matches your license (Home, Pro, etc.).

Your pre-upgrade image may be of use for restoring Windows to the previous version, if it is too late to use the rollback option. You would need to ensure that any recently created data is saved.

Insider builds are temporary and are meant to allow users to test unreleased Windows 10 builds, in order to help identify issues and bugs. They are expected to be unstable. This may mean that from time to time you need to completely reinstall Windows to resolve problems that are introduced by using Insider builds. They are not intended for daily use, as your ability to use the computer may be at risk or data might be lost.

Eventually, some Insider build will become the next public release. Public releases are scheduled every 6 months around March and September. In between those dates, Insider builds are often very unstable and full of issues.

Personally, I only use Insider builds as a dual-boot option or in a Virtual Machine. That way I can use my public version of Windows 10 without interruption each day and test Insider builds as desired. I can boot up an Insider build when I want to try it.

Animated credits in PowerPoint

I have a long list of names I need to scroll.  I've created 7 slides using text boxes with a set of names in each box.  I set the animation to credit.

I need for each slide to follow the previous one until all the names have been displayed.  Is there a better way to create and scroll a long list of names?  Using MAC, Microsoft office 2016.

Help is greatly appreciated.

william zeng msft say:

Hi Jwphill,

1) Put all the names in 7 text boxes in a single slide as E for Excel suggested.

2) Set the first name as Start on click and the rest of the names as Start with Previous.

3) Go to Timing and add the appropriate Delay time for the names in the rest of the 6 text boxes to let them scroll smoothly one by one.

For example, if the first text box delay is 0 second.

The names in the 2nd text box delay 5 seconds.

The names in the 3rd text box delay 10 seconds.

The names in the 4th text box delay 15 seconds.

The names in the 5th text box delay 20 seconds.

The names in the 6th text box delay 25 seconds.

The names in the 7th text box delay 30 seconds.

As PowerPoint 2016 for Mac doesn't like PowerPoint 2016 for Windows which has the time line that you can adjust them easily, you need to adjust them manually to meet your requirement.

Regards,

William

steve rindsberg say:

I ran this in your presentation and got correct results:

Sub TestTheLinks

Dim oSl as Slide

Dim oHl as Hyperlink

For each oSl in activepresentation.slides

for each oHl in oSl.Hyperlinks

Debug.Print oSl.SlideIndex & VbTab & oHl.Address

Next

next

End Sub

This is in PPT 2007 SP2.  There were a few bugs in the OM having to do with hyperlinks in the first release, but none that'd account for the problem you're seeing, at least not that I recall.  If your hyperlinks are in SmartArt shapes, that might account for what you're seeing.
