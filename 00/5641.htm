Manual Registry Hack for Last Active Window

Open up regedit.exe through the start menu search or run box, and then head down to the following registry key:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

Once you’re there, create a new 32-bit DWORD value on the right hand side, give it the name LastActiveClick, and set the value to 1.

jim gordon mvp say:

Right-click or control-click the arrow buttons in the lower left corner to display a list of worksheets in your workbook. If you have more sheets than fit in the pop-up you will see More Sheets at the bottom of the list. 

This macro displays the More Sheets dialog when you open the workbook:

Sub Auto_Open()
 'Based on code posted by Jim Rech
 On Error Resume Next
 Dim intHowMany As Integer
 Let intHowMany = Application.CommandBars("Workbook Tabs").Controls.Count
 If ActiveWorkbook.Sheets.Count <= 16 Then
 Application.CommandBars("Workbook Tabs").ShowPopup 500, 225
 Else
        With Application.CommandBars("Workbook Tabs")
            .Controls(intHowMany).Execute
        End With
 End If
On Error GoTo 0
End Sub

john korchok say:

Please give this a try:

  1. Open the template in PowerPoint.
  2. Choose View>Slide Master, then select the larger slide at the top of the left-hand slide window.
  3. Select the first line of the text placeholder that reads Click to edit Master text styles.
  4. Choose Home>Paragraph>Bullets and add a bullet.
  5. Set the color to black (or whatever color you're after).
  6. Save the template.
  7. Remove the bullet.
  8. Save the template.
  9. Test it to see if a local-formatted (not style-based) bullet is black.

john korchok say:

Adding a bullet from Home>Paragraph>Bullets is local formatting, so there is only a limited connection with your template styling. Getting it back to the default shape is the best you can do.

BTW, it's not a good idea to remove the text placeholder from a Master slide. The master is the parent to the slide layouts and it's the one place you can set presentation wide text formatting that automatically ripples through the layouts.
