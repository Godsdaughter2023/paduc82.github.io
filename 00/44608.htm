Unable to Add/Remove Windows Components - Microsoft Community

burnout421 say:

Unable to Add/Remove Windows Components

When attempting to access the above, I get a message: Setup was unable to open information file netoc.inf.  It also gives the error code 0x2 at line 0.  Can anyone help with this please?

a. user say:


Trying to use Add/Remove Programs to Add/Remove Windows Components you see a message like this:

Setup was unable to open information file netoc.inf.
Contact you system administrator.  The specific error code is 0x2 at line 0.
The application could not be initialized.

Error code 2 is "file not found" - that means the netoc.inf file is missing for some reason and needs to be replaced.

The netoc.inf file is supposed to be in the c:\windows\inf folder.  If the file is missing, you will see that message.

The netoc.inf file is not a Windows protected file so it is unlikely you will find a copy in the c:\windows\system32\dllcache folder and it will not automatically be replaced by Windows File Protection.

Running sfc /scannow will not help you.

You may find a copy of the netoc.inf file on your system in some other folder, such as:

C:\WINDOWS\ServicePackFiles\i386

You can also use the Windows Search function to look on your entire system for a replacement copy of the netoc.inf file and if you find one (in some other folder), just copy and paste the netoc.inf file into the c:\windows\inf folder where it is supposed to be.

If you have your genuine Windows XP installation CD, you can expand the netoc.inf file from the installation CD into your c:\windows\inf folder.

If you cannot find the netoc.inf file on your system in some other place and you do not have a genuine bootable XP installation CD, you can copy the netoc.inf file from another working system of the exact same XP version and service pack as the afflicted system (replace the missing file) - or have someone email you their file.

Hi,  a friend has a Windows 8 pro computer he bought on an auction site that is licensed with a volume_mak style license.  The windows 8.1 update doesn't show in the store - I believe it's because of this style of license.  What are his paths to upgrade to Windows 8.1?  Thanks
andre da costa say:

That computer is not authorized to use it, you need to purchase a genuine Windows 8 Pro product key and install it.


Volume License distributions of Windows 8 Pro are only available to large customers doing large deployments of Windows.


Volume License distributions of Windows 8 such as Windows 8 Pro and Enterprise utilizing KMS and MAK licenses should be updated by downloading the installation files for Windows 8.1 from the volume license servicing center, or TechNet/MSDN if you have a subscription or install based on your employers parameters.


How to purchase additional keys:


Press Windows key + X


Click System


Click View details in Windows Activation


Click the link: Buy Windows for another computer

http://www.windows.com/buy



Yes, you might be prompted to go through the install routine with just the 5 mb web installer, but you don't actually have to do the download, just make the acquisition and the product key will be presented to you in the Upgrade Assistant (you will also have a copy sent to your email address).



When you get the key, do the following:


Press Windows key + R

Type: slui.exe 3

Hit Enter

Enter the new key and click Next to activate over the Internet.


You can then upgrade to Windows 8.1 from the store.
