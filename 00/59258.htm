anne ros say:

Hi Thomas,

This error can occur if there is a driver attempting to be installed on the system that was not signed by Microsoft digitally. Driver digital signatures help verify the integrity of the driver packages. To avoid this error, you need to disable the driver signature enforcement. Follow the steps below:

  1. Press and hold the Shift key then click Restart button.
  2. Choose Troubleshoot, then select Advanced options.
  3. Select Startup Settings and click Restart.
  4. Once the computer restarts, you'll see a list of options. Press F7 to select Disable driver signature.
  5. Computer will now restart. Once done, check if the same error will occur.

Let us know the outcome so we can provide further assistance.

That məans that installing 64-bit Windows--even though it may do very little for you at present--puts you into a better position for the future.
