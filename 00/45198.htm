    Hi Jorgeej,

    Which Brand of Windows Mobile do you own? Is it a Touch Screen or Non-Touch Screen?

    Follow these Steps to Create a Playlist for both Touch Screen and Non-Touch Screen.

    Create a Playlist for a Touch Screen Phone:

    You create playlists from the media files that you add to the Now Playing screen.

    Step 1 == Tap Start > Programs > Windows Media.

    Step 2 == If you do not see the Library screen, tap Menu > Library.

    Step 3 == Navigate to a media file, and then tap Menu > Queue Up.
    Repeat this step for each media file that you want to add to the playlist.

    Step 4 == After queuing up files, tap Menu > Now Playing.

    Step 5 == Tap Menu > Save Playlist, and then enter a name for the playlist.

    Step 6 == Tap Done. Your playlist will appear on the My Playlists screen in the Library.

    Refer to this link for details:

    http://www.microsoft.com/windowsmobile/en-us/help/v6-0/create-playlist-touch.aspx

    Create a Playlist for a Non-Touch Screen Phone:

    You create playlists from the media files that you add to the Now Playing screen.

    Step 1 == On the Home screen, click Start > Windows Media.

    Step 2 == If you are not on the Library screen, click Menu > Library. If it is necessary, click the BACK button until you see the word Library near the top of the screen.

    Step 3 == Locate a media file, and then click Menu > Queue Up.
    Repeat this step for each media file that you want to add to the playlist.

    Step 4 == After queuing up files, click Menu > Now Playing.

    Step 5 == Click Menu > Save Playlist, and then enter a name for the playlist.

    Step 6 == Click Done. Your playlist will appear on the My Playlists screen in the Library.
