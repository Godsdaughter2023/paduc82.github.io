The problem is that when Windows Creates a recovery USB stick it uses the FAT32 file system.  In recent versions of windows and windows partitioning tools the maximum size Windows allows to create and format for a FAT32 drive is 32 Gigabytes.

Recent versions of Windows, using the partitioning and formatting tools that come with the operating system, have this limit hard coded.

This is easily solved by modifying the recovery media creator in Windows to format the recovery USB sticks to NTFS as Windows can boot from this file system.

If you are a user, contact the developer and request a new version.

An error that begins with 0xC1900101 is usually a driver error. Here are some steps that you can try first to resolve your concern:

  1. Remove all external storage devices and drives, docks, and other hardware you might have plugged into your device that isn’t needed for basic functionality.
  2. Check all hard drives for errors and attempt repairs. To automatically repair hard drives, open an elevated command prompt, switch to the drive you wish to repair, and type the following command. You will be required to reboot the computer if the hard drive being repaired is also the system drive.
    • chkdsk /F
  3. Attempt to restore and repair system files by typing the following commands at an elevated command prompt. It may take several minutes for the command operations to be completed. For more information, see Repair a Windows Image.
    • DISM.exe /Online /Cleanup-image /Restorehealth
    • sfc /scannow
  4. Uninstall non-Microsoft antivirus software.
    • Use Windows Defender for protection during the upgrade.
    • Verify compatibility information and re-install antivirus applications after the upgrade.
  5. Uninstall all nonessential software.
  6. Perform a clean boot. It will start Windows by using a minimal set of drivers and startup programs that can eliminate software conflicts. It can also identify if any third party programs or startup items are causing the issue. On the article "How to perform a clean boot in Windows", you will learn the step-by-step process on how to perform a clean boot depending on the Windows version that you're currently using.Note: Please go through the section: "How to reset the computer to start normally after clean boot troubleshooting" to reset the computer in normal startup after troubleshooting.
  7. Update firmware and drivers. You can find third-party drivers and installation instructions for any hardware you’ve added to your device on the manufacturer’s website.
  8. Verify at least 16 GB of free space is available to upgrade a 32-bit OS, or 20 GB for a 64-bit OS.
