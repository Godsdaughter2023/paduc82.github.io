This is a known issue and as a work around, you must create an alias and use that for the mean time.

-Log in to http://www.account.live.com using your account, go to "Your info" tab and click on "Manage how you sign in to Microsoft".

-Enter your password and last 4 digit number of your phone or alternate mail, enter the code once received.

-After that, you should be able to see the page where you can add an alias, either create new or use a 3rd party email. Set the new one as primary.

-Log out of the page, restart your browser and go to http://www.outlook.com and log in using the newly created alias. Use the same password.

*Note: This is just a workaround for this type of issue. Once the issue is fixed, you can set your old account as primary again.
