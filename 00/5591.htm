ronnie vernon say:

Hi Jonard

InstantGO is a hardware feature that is installed and implemented by the Computer Manufacturer.

You can check if InstantGO is available on your computer.

1. Right Click the Start button and select Command Prompt (Admin).

2. Type the following command.

powercfg /a

Press Enter.

If the results show that Standby <Connected> is available, then you do have InstantGO on the computer.

If it shows:

Standby <Connected>

The system firmware does not support this standby state.

Then InstantGO is not supported on the computer.

pranavmishra5582 say:

Hi,

Please write to us with the following detail.

What is make and model of your computer?

You could face this issue if the audio driver is corrupt on outdated.

Follow the methods given below and check if it helps you.

Method 1: Run Playing Audio Troubleshooter

Please follow the steps given below to run the Playing Audio troubleshooter.

            a) Type Troubleshooting in the search box present on the Taskbar.

            c) Select the Troubleshooting option obtained after the search.

            d) Select View All option from the left panel click on Playing Audio option.

            f) Click on Next to perform the Playing Audio troubleshooter and follow the steps.

Move to next method if the issue persist. 

Method 2: Uninstall the Sound Video and Game Controller and install from manufacturer’s website.

Step 1: Follow these steps to Uninstall. 

  1. Press Windows key + X and select Device Manager.

  2. Search for Sound Video and Game Controller and expand it.

  3. Right click on the Sound Video and Game Controller and select Uninstall.

Step 2: Go to the manufacturer’s website and then install the latest Sound Video and Game Controller.
