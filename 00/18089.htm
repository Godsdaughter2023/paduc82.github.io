Whether you can actually install and activate on the new computer depends entirely on how you bought Office 2007.

If you bought the big-ugly-ecologically-insensitive-yellow-plastic-box, you probably have a retail license that allows you to install on 2 or 3 machines and to transfer those installations to new machines.

david gore say:

Can't Insert Page Numbers in Word for Office365 Pro Plus

Hi.

I'm submitting this post because I had the same frustration as many others (so I have found), where I was trying to insert page numbers into word documents in Word 2016 and when I went to the insert menu the template options did not show (greyed out) and it said to Insert more page number options from Office.com but the link there was also greyed out.

Yes, there is a short term option for adding page numbers Alt+Shift+P, but this isn't resolving the problem.  I should be able to insert page numbers using the menu function, that's what it is there for and what I pay my subscription fee to be able to use.

So after trawling for answers on umpteen different forums I finally turned to the source of all great answers... YouTube.  Have a look at the link I've inserted below (some cool accompanying music with it), it fixed the problem within twenty seconds.  Ahh, I feel so much better.

https://www.youtube.com/watch?v=YJm8Ty5ZxL0

If you prefer the text option see below.

1. Close out of Word.  Then in File Explorer navigate to C: (or whatever your local drive is)\Users\UserName\AppData\Roaming\Microsoft\Document Building Blocks\1033\latest sub folder (for me this is 16 in the screen shot below)

2. In the sub-folder you should find at least one, but perhaps more files (as in the screen shot below).  For Word 2016, the appropriate file is "Built-In Building Blocks.dotx".  I presume for earlier versions is will be simply called "Building Blocks.dotx" or similar.

3. Right click and go to properties

4. On the General tab of the "Built-In Building Blocks.dotx Properties" window (see below), Untick the "Hidden" attributes check box (by implication this makes the item visible now).  Click "Apply" and then click "OK".

5. Open Word > go to the Insert menu > Click on the Page Number option and voila Page Number options now work
