On Wed, 23 Feb 2011 22:34:25 +0000, Darth_Raythe wrote:

How do I set up multiple computers in a workgroup to all display the exact same time?

Write a batch file containing the command
NET TIME \\computer  /SET /YES     (replace computer with the name of the computer you want to synch to).

Put that batch file (or a shortcut to it) in the startup folder of each of the other computers on the network.
