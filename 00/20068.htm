doug robbins - mvp office apps & services (word) say:

I haven't looked through all of the previous responses to this thread to see if something similar has been offered previously, but if a macro is run with the following code, it will insert "Comment # - " before the text of the comment, where # is a sequential number assigned to each comment in the order in which they appear in the document:

Sub NumberComments()

Dim i as Long

Dim rngComment as Range

With ActiveDocument

    For i = 1 to .Comments.Count

        Set rngComment = .Comments(i).Range
        With rngComment
            If Left(.Text, 7) = "Comment" Then
                .Text = "Comment " & i & " " & Mid(.Text, Instr(.Text, "-"))

            Else

                .Text = "Comment " & i & " - " & .Text

            End If

        End With

    Next i

End With
End Sub

The macro can be run multiple times so that if additional comments are added, they will be renumbered in the order in which they appear in the document.

Basically - assuming that the text file is structured enough like a database table - you can use File... Get External Data... Import, and select "Text files" from the "files of type" dropdown to import the data into a local Access table. You can use File... Get External Data... Link to create a link to the file without importing it - this won't be updateable and may be quite slow if the file is large, but it will let you see changes in the data if someone else is updating the text file.

There's an Import Wizard that will help you map out the file into fields, name those fields, etc.
