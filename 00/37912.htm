Windows 10

How to transfer files, data, settings etc. between pcs which are running Windows 10? Thank you.

a. user say:

Transferring files is easy. Nothing else can be transferred.

There are lots of ways to transfer files; here are two:

1- Backup the files from computer A and restore them on computer B.

2- Copy the files from computer A to an external USB drive, then attach the external drive to computer B and copy the files to computer B.

You can also make a copy of your entire hard drive and restore the copy to another hard drive. A copy of an entire hard drive is called a disk image. You'll need to install disk image software on computer A. However, there are some significant details to be aware of:

1- Unless computer B's hardware is identical to computer A, you'll have driver problems that may prevent computer B from working properly. Your disk image software may allow you to create an image with generic drivers that will get computer B up and running well enough for you to install correct drivers.

2- When you restore an image to computer B, everything on computer B's hard disk is erased.

jharris1993 say:

Create *GLOBAL* keyboard shortcut, (hotkey), to insert a particular Unicode character?

Issue:

I would like to be able to create a global keyboard shortcut/hotkey combination to insert a specific Unicode character.

I would like the solution to meet the following requirements:

  1. If at all possible, I want to avoid third-party software add-ins.
    1. Since I periodically rebuild my system from scratch to remove cruft, the fewer dependencies on third party software the better I am.
    2. Many third party applications, especially system-level add-ins, can have bizarre side-effects depending on what other software/drivers might be installed on your system.  I want to avoid this if at all possible.
  2. I want it to work the same way any other global hotkey works, regardless of individual software product running.
    1. I know that it is possible to create a keyboard shortcut that works within a specific program.
    2. I want to create a keyboard shortcut that works within any program that can accept keyboard shortcuts.
  3. If at all possible, I would like to save the change made, (perhaps it's stored in the Registry somewhere?), so that I can "re-install" it at some subsequent date if/when I rebuild my current system or build a new one.

-------------------------------------------------------

My specific problem:

I frequently communicate with people in Russia, and it is not uncommon for us to be discussing money or prices for items, sometimes in U.S. dollars, sometimes in Russian Rubles.

There exists within the Windows keyboards, both the dollar sign ($) and the ruble sign (₽).  In order to get the ruble sign, I have to switch keyboard languages to Russian, and then hit Right-Alt-8 to get the Russian ruble sign.  Note that I have both US-English and Russia-Russian keyboards installed.

Since many of my conversations are in English, it is manifestly irritating to have to stop typing, switch languages, type in one character, switch languages, and then continue typing.

For the sake of argument, assume that the "4" key has no hotkey assigned to it.  Since in the US-English keyboard the shifted "4" key is already the currency symbol for the dollar, it would be convenient to have a hotkey (say CTL-4) become the currency symbol for the ruble.

Question:

Given that there is now an internationally accepted Unicode character (\u20bd), for the Russian Ruble currency symbol, is there a way to assign that particular Unicode character to a keyboard combination like CTL-4?

Thanks!

Jim "JR"

cmdrkeene say:

Well I'm sorry to say from the start that there's no way to do this with the requirements or at least preferences you've outlined.  AutoHotKey is the industry standard for such tasks, or you can memorize the Alt+shortcut method that's built-into Windows since Windows 2.0 back in the 1980s.

For example, if you want to type the registered trademark symbol, you can hold down alt while you type 0174 on the 10-key number pad, and then when you release the Alt key, the ® symbol appears.
