The problem undoubtedly relates to a defective macro either in the normal template, the document template or an add-in template. Rename the normal template to normal.dotm.old, and remove any templates from the Word startup folder - If you have not changed the preferred startup folder it can be located (in English language versions of Windows) by typing

to the address window of Windows Explorer and you will be taken to the folder that contains the default template normal.dotm. Note that this folder is normally hidden.

Hi MartenM,

 

As I mentioned above, you need to click the pop out button first. Here are the detailed steps:

  1. Navigate to the group conversation and click to add a message.
  2. Click the pop out button on the right side.
