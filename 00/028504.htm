With the insertion point within the second section, this should have been set to Start at 1. However, the field representing the total number of pages seems to be using the { NUMPAGES } field instead of the { SECTIONPAGES } field.

Go into the header or footer that you've placed the Page x of xx and tap Alt+F9. This toggles the view between Show field results and Show field codes.
