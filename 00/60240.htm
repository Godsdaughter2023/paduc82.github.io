Hello Roger,

Thank you for posting your concern in Microsoft Community and welcome to the Windows 10 Family.

From the description provided, I understand that you are unable to rip media files onto a CD.

Please be informed that when attempting to rip an audio CD, you may receive such errors if you try to rip the CD to a low quality.

This may occur when a user switches from ripping their songs to WMA format to MP3 format and does not increase the quality of rip. Follow the below recommendations and ensure you've got an improved audio quality recording.

Follow the below steps and ensure you've got an improved audio quality recording.

Method 1:

a)   Open Windows Media Player.

b)   In Windows Media Player click Tools from the File menu. If you do not see the file menu press the Alt key.

c)    Under Tools click Options.

d)   Within the Options Window click the Rip Music tab and drag the Audio quality slider to the right where it is either the best quality or the second to highest quality.


Method 2: Enable error correction.

Windows Media Player can play some CDs that contain errors if the error correction option is turned on.

a)   Open Windows Media Player.

b)   Go to Tools -> Options -> Devices.

c)    Select CD/DVD device and choose Advanced.

d)   Place a checkmark on “Use error correction”.

e)   Once you’ve done that, try ripping the CD.

Hope the above information was helpful. If you need further information, feel free to write to us and we would be happy to help you.

Thank You.
