rick rothstein, mvp - excel say:

The Resize property change the size of the range reference it is attached to. I think things may be clearer if you see it in action.

 

MsgBox Range("C3").Address(0, 0)

 

will display C3, obviously. Now try this...

 

MsgBox Range("C3").Resize(4, 9).Address(0, 0)

 

This will display C3:K6... the original cell is expanded to a range of 4 rows by 9 columns. If you use a multi-cell range as the "base", for example,

 

MsgBox Range("C3:F5").Address(0, 0)

 

which displays C3:F5 and modify it with the same Resize property...

 

MsgBox Range("C3:F5").Resize(4, 9).Address(0, 0)

 

you still get C3:K6 displayed. The Resize property uses the cell in the upper left corner of the original range only and builds a new range off of it using the number of rows and columns specified in its arguments. Given that, you can use Resize to shrink a large range if need be. Consider this...

 

MsgBox Range("C3:Z1000").Address(0, 0)

 

and modify it using the same Resize as earlier...

 

MsgBox Range("C3:Z1000").Resize(4, 9).Address(0, 0)

 

And again, you get C3:K5. The only cell in the original range that matters to Resize is the first one. Now, you can omit arguments in the Resize property... if you do, the new range retains the original size of the original range for that omitted argument. For example,

 

MsgBox Range("C3:F5").Resize(4).Address(0, 0)

 

displays C3:F6... only the number of rows were change. Likewise...

 

MsgBox Range("C3:F5").Resize(, 9).Address(0, 0)

 

displays C3:K5 (notice there is a comma after the opening parentheses for the Resize property, meaning the row number defaults to whatever it currently is and only the number of columns  changes).

 

Hopefully that answers your question.

  • The print preview should now display all map detail including shadows

    How to enable TRIM Command in Windows 7 with a Solid State Drive

    To enable or disable TRIM Command, you will need to open an Elevated Command Prompt window.

    How To Open an Elevated Command Prompt window: Click on Start Orb > Type "CMD.exe" in Search box > Right click on "CMD" and select "Run as Administrator" (If you receive a prompt confirmation, click YES)

    How to Enable TRIM Command

    In the Elevated command Prompt windows, type the following:

    fsutil behavior set disabledeletenotify 0

    How to Disable TRIM Command

    In the Elevated command Prompt windows, type the following:

    fsutil behavior set disabledeletenotify 1

    How do I know if TRIM is working in Windows 7?

    In the Elevated command Prompt windows, type the following:

    fsutil behavior query disabledeletenotify
    Results explained below:
    DisableDeleteNotify = 1 (Windows TRIM commands are disabled)
    DisableDeleteNotify = 0 (Windows TRIM commands are enabled)

    niton_ say: Private Sub LimitRecipients()

     

    Dim currItem As mailItem

    Dim currRecipients As Recipients

    Dim currRecipCount  As Long

     

    Dim copyItem As mailItem

    Dim copyRecip As recipient

     

    Dim i As Long

    Dim recipLimit As Long

     

    ' Set this as needed

    recipLimit = 3

     

    ' Currently open item

    Set currItem = ActiveInspector.currentItem

    Set currRecipients = currItem.Recipients

     

    currRecipCount = currItem.Recipients.count

     

    While currRecipCount > recipLimit

     

        Set copyItem = currItem.copy

        copyItem.Display

           

        copyItem.To = ""

        copyItem.CC = ""

        copyItem.BCC = ""

          

        For i = currRecipCount To (currRecipCount - recipLimit + 1) Step -1

              

            ' Copy recipients from the end to the beginning

            Set copyRecip = copyItem.Recipients.add(currRecipients(i))

            copyRecip.Type = currRecipients(i).Type

           

            ' Removing from the end so the i index is not corrupted

            currRecipients.Remove (i)

            currRecipCount = currRecipCount - 1

           

        Next i

       

        copyItem.Save

       

    Wend

     

    ExitRoutine:

        Set currItem = Nothing

        Set copyItem = Nothing

        Set copyRecip = Nothing

       

    End Sub
