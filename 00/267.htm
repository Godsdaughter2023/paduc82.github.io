Fancy a macro?

Alt+F11 to open VB editor. Right click 'ThisWorkbook' and insert module and paste this code in.

Sub LoopThrough()
Dim rCell As Range

    For Each rCell In Selection.Cells
        rCell.Copy Destination:=DstSheet.Cells(rCell.Row, rCell.Column)
    Next rCell
End Sub

tushar mehta say:

Mike's code has the advantage that one can select any arbitrary range before using it.  The modifications below automatically select all the contants in the current region of the current selection or the current region for cell B3.

Sub LoopThrough2()
    If Not TypeOf Selection Is Range Then Exit Sub
    If Selection.Areas.Count > 1 Then Exit Sub
    Dim rCell As Range
    Dim DstSheet As Worksheet
    Set DstSheet = Sheets("Sheet2")
    For Each rCell In Selection.CurrentRegion.SpecialCells( _
            xlCellTypeConstants, _
            xlErrors + xlLogical + xlNumbers + xlTextValues).Cells
        rCell.Copy Destination:=DstSheet.Cells(rCell.Row, rCell.Column)
        Next rCell
    End Sub


Sub LoopThrough3()
    Dim rCell As Range
    Dim DstSheet As Worksheet
    Set DstSheet = Sheets("Sheet2")
    For Each rCell In Range("B3").CurrentRegion.SpecialCells( _
            xlCellTypeConstants, _
            xlErrors + xlLogical + xlNumbers + xlTextValues).Cells
        rCell.Copy Destination:=DstSheet.Cells(rCell.Row, rCell.Column)
        Next rCell
    End Sub

Where I can find the formulas to explain =S7/$T$3 - Microsoft Community

peter aoh say:

Where I can find the formulas to explain =S7/$T$3

Original Title: "Excel question - PLEASE HELP"

Can someone please direct me to where I can find the formulas to explain =S7/$T$3 on my excel speadsheet.
jeeped say:
There are two things about that formula.

First, it is simply a formula that returns the quotient of the value in S7 (dividend or numerator) divided by the value in T3 (divisor or denominator). There is no error control in the event that T3 is blank or zero so it is possible that it may return a #DIV/0! error (division by zero).

Second, S7 is a cell reference that uses relative row and column descriptors. This means that it will change if copied/filled dragged to a new location. The T3 cell reference uses absolute for both row and column descriptors. This means that it will not change if copied/filled/dragged to a new cell. For example, if the cell with this formula is copied and then pasted two rows down and one column to the right then the formula would change to =T9/$T$3.
