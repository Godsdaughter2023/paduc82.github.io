jay freedman say:

If the indenting isn't coming from autoformat, it must be part of the definitions of whatever styles are applied to the indented paragraphs. Styles are far more fundamental to the way Word works than is autoformat, although the results can look the same.

To find out what's causing the indents, put the cursor in an indented paragraph and press Ctrl+Shift+S to open the Apply Styles dialog box.

The name of the style in that paragraph appears in the dialog.

If the problem is that the wrong style has been applied to the paragraph, just open the dropdown and click the name of the correct style.

If it's the correct style but it shouldn't be indented, modify the style's definition (but be aware that this will remove the indentation from all paragraphs that have that style, which may not be what you want). To do this:

  • Click the Modify button in the Apply Styles dialog box to open the Modify Style dialog box.
  • Click the Format button at the bottom of the dialog, and click Paragraph in the menu that pops up.
  • In the Indentation section of the Paragraph dialog, set the Left and Right boxes to 0 or whatever indent you want. Click OK to return to the Modify Style dialog.
  • The default is for changes to affect only the current document. If you want to change the style in all future documents based on the same style, click the "New documents based on this template" option. (This will not affect any existing documents.)
  • Click OK.

Private Sub Tif2PDF(filename As String)

Dim AcroApp As CAcroApp
Dim AVDoc As CAcroAVDoc
Dim PDDoc As CAcroPDDoc
Dim IsSuccess As Boolean
        
    AcroApp = CreateObject("AcroExch.App")
    Set AVDoc = CreateObject("AcroExch.AVDoc")
    Call AVDoc.Open(filename, "")
    
    Set AVDoc = AcroApp.GetActiveDoc
    
    If AVDoc.IsValid Then
    
        Set PDDoc = AVDoc.GetPDDoc
                
        ' Fill in pdf properties.
        PDDoc.SetInfo "Title", "My Title"
        PDDoc.SetInfo "Author", "The Author"
        PDDoc.SetInfo "Subject", "The Subject"
        PDDoc.SetInfo "Keywords", "Keywords"
                
        If PDDoc.Save(1 Or 4 Or 32, _

        PDDoc.Close

    AVDoc.Close True
    AcroApp.Exit
