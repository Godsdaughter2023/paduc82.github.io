Need to reset to ID auto number. - Microsoft Community

brent doncliff say:

Need to reset to ID auto number.

I have started a new Access database, but for some reason the autonumber ID always want to start at 3, I want to it to start at 1 - how do I change this?

john w. vinson say:

Why do you care?

An Autonumber has one and only one purpose: to provide a unique ID for a record. This is often used as a Primary Key (which must be unique) but that's not essential - you can have an Autonumber that's not a PK or vice versa. Autonumbers will always have gaps; if you start to add a record and then cancel the transaction, or if you delete a record from the table, that ID will be used up forever. You should NOT treat it as a "record number" much less as a record count.

That said, you should be able to empty (or delete) your Table and Compact the database to reset the autonumber seed.

You can perform the soft reset by having your phone turned on. Then, press and hold the volume down and         Power buttons until your phone vibrates and restarts. Note that you should not touch the device screen during this process (not even when the 'slide down to power off' prompt appears).

Start by downloading and installing Windows Device Recovery Tool (for Windows 10, 8.x and 7) from here:

The procedure to restore your phone using Windows Device Recovery Tool is as follows:

  1. Connect the phone to your PC (using the USB cable that came with the phone).
  2. After a few seconds the PC should find the phone and you'll see the Phone Selection screen. Click your phone in this screen to select it.
  3. After a few seconds more the latest available firmware version for your device will be shown. Click the 'Install software'/'Reinstall software' button to flash your phone

Windows Device Recovery Tool will then download and install the latest firmware + OS available for your phone.

Note! Your phone will be reset to factory status when using this tool.
