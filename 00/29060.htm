Even after following instructions at many other Microsoft sites to fix these issues such as https://support.microsoft.com/en-us/help/943996/some-services-do-not-start-in-windows-vista-and-windows-7 I was still not able to get DPS running.

Per the suggestion of a user over on https://social.technet.microsoft.com/Forums/windowsserver/en-US/492217c2-b92f-4e24-a12f-cccdd986a43f/solution-for-diagnostic-policy-service-access-denied-2008-r2?forum=winservergen

I did not take notes as I tried the fixes so here's what I did:

Process Explorer captures A LOT of data so you need to setup a filter where "RESULT" contains "denied"

Have an elevated Command Prompt ready with "net start DPS" which attempts to start the DPS service

Start Process Explorer capturing

I would try to start the DPS Service by going to the CMD prompt and hitting enter

As soon I saw the Error 5 Starting DPS "Access Denied" I would stop Process Explorer

It tells you what registry key DPS is trying to access but can't - in my case it was HKLM\System\CurrentControlSet\Control\WDI\Config

Navigate to the key in regedit

Grant those permissions to "NT SERVICE\DPS" (it helps to have a functional machine with the same version of Windows to copy from)

After figuring this out I was able to apply the same tactic to Windows Firewall aka "mpssvc" and get it running as well.

An account suspension completely blocks the affected account from being able to sign in to the Xbox Live service for the duration of the suspension. Account suspensions are issued for a specified duration, often of 1, 7, or 14 days, though egregious or repeated violations can result in a permanent suspension.

A common misconception is that a certain number of reports will result in an automatic suspension. According to the myth, even if you haven’t done anything wrong someone can get you suspended by encouraging players to file many complaints. This is completely untrue.

XBLPET investigates complaints and takes each report into consideration when making an enforcement decision, but a certain volume or frequency of complaints doesn’t guarantee a suspension. We take an evidence-based approach to enforcement and take numerous factors into account when considering issuing a suspension. The context around the content or activity is particularly essential and drives our decision-making process. Simply put, there is no magic formula that instantly results in a suspension.

Despite what some may claim, XBLPET will never ever ban anyone “for no reason”. Suspensions alwaysoccur as a result of a specific Code of Conduct violation.
