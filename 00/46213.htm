hairyfool say:

Of the 3 options an SSD will give the most noticeable improvement.

 

Memory increases have diminishing returns and quite significantly so even a hike to 12GB would not be as good as the increase from 2GB (64 bit minimum) to 6GB.

 

Readyboost I have never seen any significant benefit and is mostly aimed at a system with minimum spec.

 

Use MSConfig to look at what is preloading in the startup tab and get rid of the non essentials. There are lots of complaints about the big antivirus suites slowing systems down, if you have one look at the modules that load and see which ones are really not needed.

 

It takes a little interpreting but instead of Resource Monitor get Process Explorer from SysInternals to get a better idea of what is slowing the system down although from recollection you may have already seen it with the offline files option. Make sure the network store only has what you are actually going to want in offline mode to minimise the replication process.

DISM command is used to fix corrupt files found by System File check or Scanhealth tool.

DISM is just an extension of SFC that allows you to repair corrupt files from Installation media where SFC cannot repair corrupt files.

  2. In the right side, select Create Task. A new window will open. Here under the General tab, fill in the Name and Description. Also check, Run with highest privileges.

  3. Under the Triggers tab, click New. Another window will open. Here. Select one time (or Daily if you want it repeated every day at a particular time). Set the Date and Time when you want your PC to wake up from Sleep.

  4. Next under the Actions tab, you have to mention a task. Click New. You can run a simple task, like say, first launching & then closing the command prompt window. This thus will wake your machine at the pre-set time to carry out the task.

  5. Select Action as: Start a program. To schedule a task that executes the command cmd.exe with command line arguments, copy-paste under Program/script /c “exit”

  6. Under Conditions tab, check on the Wake the computer to run this task check-box. This is important!

  7. Click OK and Exit Task Scheduler.

  8. Your computer will wake up from Sleep at the particular time.
