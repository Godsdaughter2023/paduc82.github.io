Here's the solution :

Since the release of Windows 10, Microsoft has added a new feature to Windows Explorer or This PC. Whenever you attach a removable drive such as USB drive or external hard disk, a separate icon of the drive is shown in the left-side Navigation Pane. Also the same drive icon is shown under This PC section, so you get double entries of the same removable drive in Navigation Pane. It was done to provide quick access to removable drives but having two duplicate drive icons in Navigation Pane becomes confusing sometimes.

But that's not the problem which we are going to talk about. The problem is, many Windows 10 users are getting double entries of fixed hard disk drives in Navigation Pane of This PC. For many users, Windows 10 is showing many drives twice in Navigation Pane just like it does for removable drives. But the problem is those duplicate drives are not removable. The drives are fixed hard disk drives.

Following are 2 different screenshots which were shared by our readers. You can see the issue in the screenshots:

If you are also facing this duplicate drive entries problem in Windows 10, this tutorial will help you in fixing the issue and removing the double drive entries from Navigation Pane of This PC.

To fix this annoying issue, you just need to remove a key from Registry Editor which is responsible for showing removable drive icons separately in This PC's Navigation Pane. Since somehow Windows 10 detects the hard disk drives as removable drives and shows their icons twice in Navigation Pane, we can disable the built-in feature to show removable drive icons separately in Navigation Pane to fix this bug or issue.

We have already shared the Registry tweak to remove removable drive icons from Navigation Pane in This PC in following exclusive tutorial:

[Windows 10 Tip] Remove Icons from Navigation Pane of This PC

The trick to remove removable drive duplicate entry from Navigation Pane has been given in the Part 5 of the above mentioned tutorial.

For your convenience, we are going to provide the same solution again here:

1. Press WIN+R keys together to open RUN dialog box. You can also open it from WIN+X menu. Now type regedit in RUN dialog box and press Enter. It'll open Registry Editor.

2. Now go to following key:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace\DelegateFolders

3. Under "DelegateFolders" key, you'll find {F5FB2C77-0E2F-4A16-A381-3E560C68BC83} key.

4. Now delete the {F5FB2C77-0E2F-4A16-A381-3E560C68BC83} key and it'll immediately remove the duplicate drive icons from This PC's Navigation Pane.

Now you'll no longer see any drive icon twice in the Navigation Pane of This PC.

NOTE: If you are using 64-bit edition of Windows 10, you may notice that the drive icon is still present in "Browse" dialog box such as Save, Save as or Open in some 3rd party 32-bit programs. To remove the duplicate drive icon from there, you'll need to perform the same steps in following key as well:

HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace\DelegateFolders

Under "DelegateFolders" key, delete {F5FB2C77-0E2F-4A16-A381-3E560C68BC83} key.

tryx3 say:

Willum,

If Windows 7 was preinstalled by the computer maker then Belarc will find a "default" product key.  Upgrading to Windows 10 does not change that behaviour.  Belarc would only find a real product key if Windows 7 had subsequently been reinstalled using the product key shown on the Certificate of authenticity label on the outside of your computer [or in its battery compartment].

The "default" product key is of no use whatsoever.  It cannot be used for activation.

Several other key finder utilities suffer from the same limitation.

As Andre explained, an activated Windows 10 has its activation recorded on MS servers so that you can reactivate automatically upon reinstallation.  So there would seem to be no point in finding out if the key returned by productkey.vbs is "correct" or not - you will never be called upon to use it.

Incidentally, I recently re-installed Windows 10 from scratch on a couple of computers.  I went online then immediately opened Settings, Update & security, Activation to check up on it but it had already completed by the time I reached that dialog [so, about 10 seconds at most].

When you upgrade to the free upgrade, there is no product key to keep.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already ownto reactivate if necessary. or use the reset function in Windows 10.

 

The only product key value is really your original Windows 7 key.

How to Find SSID (Network Name)

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless Settings from the menu on the left.
  4. Your network name is listed next to Name(SSID).

How To Find Wireless Passphrase (WPA/WEP)

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless Settings from the menu on the left.
  4. Your Wireless Passphrase can be found in the lower section next to Passphrase.

Changing Wireless Security Type/ Disabling Wireless Security

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless Settings from the menu on the left.
  4. From the Security Options section Select the desired security type, None if needed.
  5. Click the Apply button if any changes were made.

  4. From the drop down menu labeled Control Channel select the desired channel. (Channels 1,3,9,11)

  3. Enter the username “admin” and the password “password”.
  4. Select Wireless from the menu on the left.
  5. From the 802.11 Mode drop down select “g only”.
  6. Click the apply button at the bottom of the screen.

DHCP Reservation

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Lan IP from the menu under the Advanced section on the left side.
  4. Enter the Desired IP, the MAC Address of your console in the provided fields.
  5. Click the Add button under the Address Reservation section. 

UPnP

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select UPnP from under the Advanced section on the left side.
  4. Toggle UPnP on or off by checking or unchecking the Turn UPnP On box.
  5. Click the Apply button at the bottom of the screen.

DMZ

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select DMZ Host from under the Advanced section on the left side.
  4. Check the Default DMZ Server box and enter the IP Address of your Xbox console.
  5. Click the Apply button at the bottom of the page.

Disable DMZ

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select DMZ Host from under the Advanced section on the left side.
  4. Uncheck the Default DMZ Server box.
  5. Click the Apply button at the bottom of the page.

Port Forwarding

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Port Forwarding under the Advanced section.
  4. Type a Service Name that has something to do with Xbox.
  5. Set the Protocol to Both.
  6. Set both the Start Port and End Port to 3074.
  7. Enter the IP Address of your Xbox console in the Server IP Address space.
  8. Click the Add button at the bottom of the page.

Disable Port Forwarding

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Port Forwarding under the Advanced section.
  4. Select the rule that pertains to port 3074.
  5. Click the Delete button at the bottom of the screen.

Firewall

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Services under Content Filtering.
  4. Uncheck Firewall features and IP Flood Detection
  5. Click the Apply button at the bottom of the screen.

  2. Enter the username “admin” and the password “password”.
  3. Select Router Status from under the Maintenance section.
  4. Make a note of the Firmware Version.
