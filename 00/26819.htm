thanks, C.

jay freedman say:

The macro below will work if there are exactly two open documents at the time you run it. All it does is call the command equivalent to the View Side by Side button, and then turn off the synchronous scrolling. You can add a button to your Quick Access Toolbar to run the macro.

Sub SideBySide_NoScroll()
    Dim n As Integer

    If Documents.Count <> 2 Then
        MsgBox "This macro requires exactly two open documents."
        Exit Sub
    End If

    If ActiveDocument.Name = Documents(1).Name Then
        n = 2
    Else
        n = 1
    End If

    Windows.CompareSideBySideWith Documents(n)
    Windows.SyncScrollingSideBySide = False
End Sub

For some reason the Microsoft developers neglected to provide a way for a macro to invoke the dialog that appears when you click the View Side by Side button in the ribbon and there are three or more open documents, allowing you to choose which other document you want next to the active document. It would be possible to simulate that with a userform containing a list box, but I didn't want to get that fancy here -- if you need it, ask.
