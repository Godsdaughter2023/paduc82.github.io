stefan blom say:

The simplest approach is to type in the text that you want in your header, and then tab to the next tab stop, where you can insert the page number. You can quickly add a page number field by pressing Alt+Shift+P.

If you added a predefined header building block, be sure to place the insertion point in a "free" area, that is, make sure not to select anything that's already in your header, prior to inserting the page number.

suzanne s. barnhill say:

The secret here is that the page numbers in the Page Number gallery are building blocks, and so are the headers in the Header gallery. If you insert a page number from the gallery, you replace the header building block you have already inserted.

There are several ways around this:

  1. If you want to use a fancy header building block, you can choose one that already includes the page number.
  2. If you select one that doesn't include the page number, click in the header (somewhere outside the content controls) and use Page Number | Current Position. An alternative way to insert the PAGE field anywhere in a document is to press Alt+Shift+P.
  3. You can use the Blank or Blank Three Columns header building block. All of these have content controls that are overwritten when you type in them, so again you can use Page Number | Current Position or Alt+Shift+P to insert the PAGE field.
  4. You don't have to use a building block at all. You can activate the header by double-clicking in it and just type in the text that you want, inserting a PAGE field (by either of the methods described above) at any location. By default, there are two tab stops in the header: a center-aligned one in the center and a right-aligned one at the right margin. If you have changed your margins, you may need to drag those tab stops on the ruler so they are in the right place for your new margins. These allow you to place text left, center, and right.
