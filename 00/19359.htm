irst, you need to download the latest drivers from Intel’s website at http://downloadcenter.intel.com/

Download and extract the zip file, and navigate to Graphics folder, and find the inf file (in this case, it’s igdlh64.inf but it might be different in your case)

Open the file and scroll down to section that says “Windows 8.1 Install” and then change “NTamd64.6.3″ to “NTamd64.6.4“. This will enable drivers to be installed to Windows 10 “Threshold”.

Delete the igdlh.cat file in the same directory.

Now that you altered this file, this broke the signature of the drivers, so it can no longer be installed as is, since drivers need to be digitally signed to be installed.

To bypass this check, you need to boot with the integrity check disabled. Go to Settings > Update & Recovery > Advanced Startup and click restart now. Follow the instructions there to change startup behavior and disable integrity check (F7).

This will restart the computer with integrity check disabled once, so you don’t need to change this back. Once it’s back up, go to Device Manager, and select Intel HD graphics and open it’s properties. Select Driver tab, and click on Update Driver > Let me pick from a list of device drivers on my computer > Have Disk and select the inf file you modified before.

      

This will start installing the driver, make sure to select install this driver anyway when prompted.

stephanie a. say:

Autocaption

I want to insert a graphic and  have the caption automatically inserted. I can do it with tables, but not with any graphics, even bmp. Any ideas how it can be done? The graphics to be inserted will be bmp, jpg, png, and others.

john korchok say:
  1. In Word, choose FILE, then Options.
  2. Click on Customize Ribbon.
  3. In the right-hand list of tabs, check Developer. OK out.
  1. Click on DEVELOPER, then on Macros.
  2. In the Macro name: field, type InsertPictureWithCaption, then click on Create.
  3. The VBA editor window opens with the starting point for your macro:
'
' InsertPicture Macro
' 
'

End Sub

Delete the lines that start with apostrophes. (Apostrophes turn any text after it into a comment, so if you accidentally paste a line of code after an apostrophe, that line won't run) Edit the macro to look like this:

Sub InsertPictureWithCaption()
  Dim sRange As Range
  Set sRange = Selection.Range
  Set IPDlg = Dialogs(wdDialogInsertPicture)
  With IPDlg
    If .Show = -1 Then
      With sRange
        .SetRange Start:=sRange.Start, End:=sRange.End + 1
        .Select
      End With
      With Dialogs(wdDialogInsertCaption)
        .Label = "Figure"
        .Position = wdCaptionPositionBelow
        dlgResult = .Show
      End With
    End If
  End With
End Sub

  1. In the VBA Editor, choose File>Save Normal.
  2. Close the VBA editor.

To make it easy to access this macro, assign a keyboard shortcut to it:

  1. In Word, click on FILE, then on Options.
  2. Click on Customize Ribbon, then on the Customize button beside Keyboard shortcuts.
  3. In the Categories list, scroll down to Macros and select it.
  4. In the right-hand list, select InsertPictureWithCaption.
  5. Click in the Press new shortcut key: field.
  6. Press a key combination like Alt + I. Word will tell you if your chosen combination is already assigned to some other command.
  7. Click on the Assign button, then on Close.

Now you can insert a picture and have the caption dialog come up automatically by pressing your key combination. The macro above assumes you are inserting pictures inline with text. If you prefer floating pictures, let me know and I can modify the macro.

Try feeding this function with your named range:

Function ConcAllNonBlankCellsInRange(rng As Range) As String
    Dim c As Range
    Dim strComma As String, strOut As String
   
    strComma = ""
    For Each c In rng
        If c.Value <> "" Then
            strOut = strOut & strComma & c.Value
            strComma = ", "
        End If
    Next c
    ConcAllNonBlankCellsInRange = strOut
End Function

 

Sub Example()

End Sub
