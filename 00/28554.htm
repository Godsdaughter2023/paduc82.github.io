Assuming California is in B2

=INDEX($A$1:$H$1,MATCH(MAX(OFFSET($A$1,MATCH(B2,$A$2:$A$9,0),0,1,7)),OFFSET($A$1,MATCH(B2,$A$2:$A$9,0),0,1,7))) &" " & TEXT(MAX(OFFSET($A$1,MATCH(B2,$A$2:$A$9,0),0,1,7)),"$ 0.00")

OFFSET($A$1,MATCH(B2,$A$2:$A$9,0),0,1,8) establishes a range 7 columns wide in the row matching B2

MAX finds the max number in this range 153,271.00 in your example

MATCH(MAX(OFFSET($A$1,MATCH(B2,$A$2:$A$9,0),0,1,8)),OFFSET($A$1,MATCH(B2,$A$2:$A$9,0),0,1,8))

finds the position of the max number

MATCH(153,271.00,OFFSET($A$1,MATCH(B2,$A$2:$A$9,0),0,1,8)) which equals 3

Index ($A$1:$H$1,3) returns the month (February)

&" " & just adds a space

TEXT(MAX(OFFSET($A$1,MATCH(B2,$A$2:$A$9,0),0,1,7)),"$ 0.00") is the actual value (displayed as text)

shane devenshire say:

Here is the help on the SUBTOTAL function: You would use the numbers 1,2 or 3 and 7 or 8 although the help screen doesn't exactly make that clear in my opinion.

SUBTOTAL(function_num, ref1, ref2, ...)

Function_num   is the number 1 to 11 (includes hidden values) or 101 to 111 (ignores hidden values) that specifies which function to use in calculating subtotals within a list.

Function_num
(includes hidden values)		 Function_num
(ignores hidden values)		 Function
1 101 AVERAGE
2 102 COUNT
3 103 COUNTA
4 104 MAX
5 105 MIN
6 106 PRODUCT
7 107 STDEV
8 108 STDEVP
9 109 SUM
10 110 VAR
11 111 VARP

Ref1, ref2   are 1 to 254 ranges or references for which you want the subtotal.

Remarks

  • If there are other subtotals within ref1, ref2,… (or nested subtotals), these nested subtotals are ignored to avoid double counting.
  • For the function_num constants from 1 to 11, the SUBTOTAL function includes the values of rows hidden by theHide Rows command under the Hide & Unhide submenu of theFormat command in the Cells group on the Home tab. Use these constants when you want to subtotal hidden and nonhidden numbers in a list. For the function_Num constants from 101 to 111, the SUBTOTAL function ignores values of rows hidden by theHide Rows command. Use these constants when you want to subtotal only nonhidden numbers in a list.
  • The SUBTOTAL function ignores any rows that are not included in the result of a filter, no matter which function_num value you use.
  • The SUBTOTAL function is designed for columns of data, or vertical ranges. It is not designed for rows of data, or horizontal ranges. For example, when you subtotal a horizontal range using a function_num of 101 or greater, such as SUBTOTAL(109,B2:G2), hiding a column does not affect the subtotal. But, hiding a row in a subtotal of a vertical range does affect the subtotal.
  • If any of the references are 3-D references, SUBTOTAL returns the #VALUE! error value.

Example

scoobyman---01 say: Hi turbo

Firstly the "two flashing arrow type font" means that A-GPS=Assisted GPS is enabled and using an active network connection. "Red Cross" means that satellite lock-on has not yet been achieved.

Go outside awy from high buildings and trees with view of open sky, hold phone with keypad slider open at 45 degrees, don't cover the "#" key as GPS antenna is below this.

Go to Menu > Tools > Settings > GPS data > Position > Open > Options > Satellite status > Open:

You need a minimum of three solid black bars, but usually get four for lock-on position.

If empty rectangles haven't turned to solid bars by 20 mins your GPS isn't working!

Now that you've determined GPS is working Open Maps application - at first there is RED CROSS on satellite icon at bottom right - this disappears when GPS locked-on

Close application and you see "saving data" above globe on screen. This creates folder structure on memory card that Maploader requires.

sheen valentine ali msft say:

Hi Avian,

I would first like to explain that for the data connection to work, you need to have the file locally. If you would like to store your Excel files in OneDrive for Business, you would need to download the sync client and move the files to your sync folder. This way, your files are available locally, but are also synced to OneDrive for Business.

However, if you open the main workbook and refresh the data connections from another computer, you will get an error that the linked notebooks can't be found; this is because the path of the workbooks are from your other computer, where the files are stored locally (for example: C:\Users\Sheen\OneDrive sync folder).

With that out of the way, here are the answers to your questions:
1. OneDrive for Business is a cloud storage service. Macro will still be run in the context of Excel. Macro functionality only works with the Excel client. Excel Online does not have macro functionality.

2. When you refresh the data connection, Excel will pull the data from the local Excel file that is synced with OneDrive for Business. If you do not have the file locally, this will not work; which means that Excel can't pull the data from OneDrive for Business. After you refresh the data and save, this newer version of the file will be uploaded to OneDrive for Business by the sync client.

3. Yes, you will need to run the macro manually. As mentioned above, Macro will still be run in the context of Excel and not OneDrive for Business. Excel files stored in OneDrive for Business can have macros embedded.

Sheen
