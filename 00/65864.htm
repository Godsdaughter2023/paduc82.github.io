Media Transfer Protocol(MTP) allows media files to be transferred automatically to and from portable devices. Devices using MTP have a USB connection that allows users to connect the device directly to a computer. To transfer the files we require a specific driver, if the driver is corrupted, we come across these issues.

I suggest you to follow the troubleshooting steps mentioned below:

Method 1: Install Media feature pack.

I would suggest you to install Media Feature Pack. Select the edition as 1803 and follow the on screen instructions. Once the installation is done, go back to C:\Windows\INF folder and search for "wpdmtp.inf" file.  

Method 2: Install the MTP driver.

Step 1:

  1. Press Windows and R key together on your Keyboard.
  2. Type devmgmt.msc and hit enter.
  3. Select the driver, right click and select update driver.
  4. Click on “Browse my computer for driver software”.
  5. Now select "Let me pick from a list of device drivers on my computer".
  6. You will have a list, select "MTP USB Device" and click on next.

Step 2:  If MTP USB device in not listed in compatible hardware, then manually install the MTP USB driver.

  1. Click on Action > select Add legacy hardware.
  2. Select install the hardware that I manually select from the list (Advanced).
  3. Select "Portable Device" > Now, select MTP USB device and start the installation.
  4. Now MTP device will be visible under compatible hardware mentioned in the step 1.

Method 3: Modify registry.

Note: Serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. To backup and restore registry key, refer Microsoft article on How to back up and restore the registry in Windows.

  1. Press Windows and R key together on your Keyboard.

  3. Navigate to the following key: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class

  5. Type Portable Devices into the box and click Find Next.
  6. Check for UpperFilters from the right pane and right click on it > Select Delete.

Hi,

Thanks for posting your query to Microsoft forum. I will certainly help you with this.

I would like to know;

  1. Did you make any changes to your computer prior to the issue?
  2. Do you have any pending updates?
  3. Have you tried to check the same in other device?
  4. Is it a SD card slot in the computer or an external card reader?

This issue usually occur if the SD card is locked or if there is any corruption in the drivers. This can be a hardware issue aw well. You can try the methods below and check.

Method 1: Use the hardware troubleshooter to fix the issue.

The Hardware Troubleshooter is an automated tool which will check the hardware connected to the computer for any known issues with it and provides the details and on how to fix them. Follow these steps and run the troubleshooter.

  1. Connect the devices to the computer.

  2. Press Windows + W key on the keyboard.

  3. Type troubleshooting in the search tab and press Enter.

  4. Click View all on left panel.

  5. Click Hardware and devices, new window will popup.

  6. Follow the on screen instructions.

If the issue persists, follow method 2.

Method 2: Try reinstalling USB controllers.

Uninstalling and reinstalling USB controllers will not affect the USB port, as when you restart the computer/laptop it gets reinstalled automatically.

  1. Press Windows Key + X on the keyboard.
  2. Go to Device Manager.
  3. Expand the USB Controllers, click on the drivers.
  4. Click on properties, click on uninstall tab.
  5. Click OK.

Also check the basics:

Method 3: Unlock the memory card.
There is a Lock switch on the left side of the SD card. Make sure the Lock switch is slide up (unlock position). You will not be able to modify or delete the contents on the memory card if it is locked.


If the issue persists, follow method 2.


Method 4: Removing the write protection depends on the source of the problem.

The SD card has a write-protection tab on the side that has slid down. This can happen when the card is being inserted into the computer. Try removing the card, push the tab back into place and try inserting it into the computer again. Read the manual that came with the SD card for help.

Please post back with the results and the information asked, We will be glad to assist you further.
