suzanne s. barnhill say:

The general rule is that index entries will have no special formatting provided that the XE fields are in the Default Paragraph Font. That is, if you put an XE field in a heading that is 90-point bold italic, and the XE field is 90-point bold italic, then the index entry will be unformatted, but if you remove that formatting from the XE field, then, perversely, the index entry will likely acquire it (at least some portion of it). To make sure that the XE fields are formatted the same as the surrounding text, you can select them and press Ctrl+Spacebar (ResetChar).

If the italic formatting persists, make sure that the XE field does not contain an \i switch, which formats the entry as italic.
