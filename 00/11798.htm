RANK - Rank IF > Rank with criteria - Microsoft Community

lz. say:

RANK - Rank IF > Rank with criteria

Intro

All examples are presented in Tables (vs. Ranges). The same things can be achieved with ranges.
Amongst others Tables offer the following benefits: Dynamic range adjustment as rows/columns are added/removed; No need to copy down formula; Formulas easier to understand with structured references (i.e. [Product] vs. A2:A10); Style can be applied; etc…

How To: Format as Table

Assumption

You understand how COUNTIFS function works

Why RANK(C4, IF(…), 1) can't be used

If you try something like RANK(C4, IF(A4:A11=A4, C4:C11, ""), 1) you may get an expected rank. And if you try to array enter (Ctrl+Shift+Enter) the formula you will get a #VALUE! error

  1. IF(A4:A11=A4, C4:C11, "") generates/results in an array of values, i.e. {150,300,200,250,"","","",""}
  2. RANK syntax: RANK(number, ref, order) ==> RANK expects a reference (i.e. C4:C11), not an array
    RANK(C4, {150,300,200,250,"","","",""}, 1) won't work either as {…} is again an array (of constants)

Available options with formula

If you search the Web you will find 2 common approaches

SUMPRODUCT
More difficult to understand than COUNTIFS. Less efficient than the latter on large dataset
NB: Only option if you need to nest another function (i.e. YEAR, MONTH…)

COUNTIFS
Preferred option whenever possible
NB: Can't be used if another function needs to be nested

Overall logic with COUNTIFS

Same approach with SUMPRODUCT but different syntax

1 criteria: [Points]

First comparison operator is =; Second comparison operator is > (or < for descending order); Then add 1

in D2: =COUNTIFS([Group],A2, [Points],">"&C2) +1

2 criterias: [Points], then # [Won]

Same as 1 criteria
+ a 2nd COUNTIFS where the 2nd operator (> or + a 2nd COUNTIFS where the 2nd operator (> or <) becomes = and another comparison criteria is added with a > or < operator (still +1 at the end)

in E14: =COUNTIFS([Group],A14, [Points],">"&C14)
           + COUNTIFS([Group],A14, [Points],C14, [Won],">"&D14) +1

3 criterias: [Points], then # [Won], then [Bonus]

Same as 2 criterias
+ a 3rd COUNTIFS where the same logic (replace > / < with = and add another criteria with operator > or <) is applied (still +1 at the end)

in F26: =COUNTIFS([Group],A26, [Points],">"&C26)
           + COUNTIFS([Group],A26, [Points],C26, [Won],">"&D26)
           + COUNTIFS([Group],A26, [Points],C26, [Won],D26, [Bonus],">"&E26) +1

With 4, 5... criterias just expand the above logic

SUMPRODUCT with nested YEAR function

2 criterias: [Amount], then [Date] Ascending

in C23: =SUMPRODUCT(--(YEAR([Date])=YEAR(A23)), --([Amount]>B23))
            + SUMPRODUCT(--(YEAR([Date])=YEAR(A23)), --([Amount]=B23), --([Date]<A23)) +1

More examples in this workbook

  • RANK - Dense rank ignore blanks and errors
