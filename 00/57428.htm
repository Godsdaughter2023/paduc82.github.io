christopher miller say:
If you have replaced the memory, I would look next at video card drivers and the power supply.  I had a similar problem with my last PC.  When I replaced the power supply with a newer and higher capacity supply, the problem went away.

christopher miller say:
Your video card is probably drawing more power that the SATA drives.  If the power supply is starting to fail, you will see lockups or other failures with the PC just sitting there.

ulricnijs say:

So this error code is definitely something to do with power? 

christopher miller say:

I don't know what would cause that specific error code.  But from personal experience, random freezes when the PC is idle has a few suspects to look at.  One is failing memory (and you ruled that out already), another is buggy video card drivers, and the next one on the list is the power supply.

If you want to rule out software as the issue, try running with a fresh copy of Windows.  Get another hard drive and either clone the existing boot drive to it and then reset Windows, or install Windows fresh on it.  Then grab some tool does benchmarking and let it stress the machine.  If that works fine, then it's a buggy driver.
