lemp say:
  1. First of all, the fear you expressed in your initial post is well-founded.  If you take the drastic step (very likely unneeded, IMO) of doing a clean install, you WILL have to reinstall applications (programs) from original installation media (CD/DVD/download).  If you don't have media or if the download is no longer available, you won't get the program back.
  2. You don't NEED any of the .NET Frameworks.  These are resource collections used by application developers.  When you install an application that requires a particular "version" of the .NET Framework (yes, you may need more than one "version"), the application installer usually installs the .NET Framework along with the app.  If you uninstall a .NET Framework "version," any app that requires it will no longer run and will throw an error message complaining about it.


I don't have step by step instructions for you to follow, but read through the following collection of articles and you should be able to -- at the very least -- uninstall all of the .NET Frameworks from your system.
