I don't see a screen shot of your actual list and due to the limited space in the formula field for data validation, I can't see all of the formula to determine if it is correct.

Therefore I will provide some guidelines for setting up a dynamic range for Data Validation and see if it helps. Feel free to get back to me if still encountering any problems.

When creating a dynamic data validation list, create a Defined Name with the formula and use the Defined Name in the Data Validation. See the following screen shot of the dialog to create the Defined Name. I have used the column header for the Defined Name and the offset formula starts on cell $A$1 (Column header) then offset down one row off the column header to start the dynamic range.

Formula for the Defined name in "Refers to" field:    =OFFSET(Sheet1!$A$1,1,0,COUNTA(Sheet1!$A:$A)-1,1)

Note that all references are absolute with the $ signs. The COUNTA result has 1 subtracted from it because the column header is not included in the Height count.
