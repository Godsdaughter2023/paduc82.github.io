kristine arp say:

Hi,

 

 

OneNote 2010 is a part of Office 2010 suite. That being said, there is no option to uninstall it without uninstalling the whole Office suite. It is possible that since you have two different versions of OneNote installed on your computer, the system is unable to identify Send to OneNote 2016 printer driver. You can try the following methods.

 

Method 1: Set OneNote 2010 as unavailable.

 

  1. Go to the Control Panel.
  2. Choose Programs.
  3. Select Program and Features.
  4. Right-click on Microsoft Office 2010.
  5. Select Change.
  6. Choose Add or Remove features.
  7. Click the drop-down beside OneNote and choose Not Available.
  8. Hit OK.

 

Method 2: Set the Send to OneNote 2016 as the Default Printer.

 

  1. Type Control Panel in Search.
  2. Choose Hardware and Sound.
  3. Select Devices and Printers.
  4. Right-click on Send to OneNote 2016.
  5. Select Set as default printer.

 

Method 3: Update the Send to OneNote 2016 driver.

 

  1. Type Control Panel in Search.
  2. Choose Hardware and Sound.
  3. Select Devices and Printers.
  4. Right-click on Send to OneNote 2010.
  5. Select Remove device.
  6. Click Yes.
  7. Right-click on the blank area on the Devices and Printers window.
  8. Select Add devices and printers.
  9. Choose The printer that I want isn’t listed.
  10. Select Add a local printer or network printer with manual settings.
  11. Click Next.
  12. Under the Use an existing port, select a port.as the Manufacturer.
  13. Select Microsoft Send to OneNote 2016 Driver.
  14. Choose Replace the current driver.
  15. Hit Next.

 

If the methods above didn’t work, you might want to consider upgrading the whole Office suite to the 2016 version. Check this link for instructions on how to upgrade to Office 2016.

 

If you prefer to use the OneNote 2016 standalone with Office 2010, download and run the Office Configuration Analyzer Tool. This would help us identify known Office configurations that might have been causing the issue.

 

Let us know if you need additional assistance.

You should not be modifying the OfficeUI file to modify the Ribbon.

Modifying the Ribbon in Microsoft Word

The ribbon has been something that could be customized in individual documents or templates since Word 2007.

seancashmere say: Oh, I see what you're saying now...

The Nokia 6300 automatically adds those folders to your memory card.
