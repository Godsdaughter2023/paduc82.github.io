How to fix Access Rights problem

Sometimes the problem has been that Windows has had problems granting the right access to files, folders and the registry. The following tips provide some workarounds.

Unfortunately, the system here has rendered your upload without color and without checkbox labels displayed.  But I think I have an idea of what you want, so I will try to explain:

 

Checkboxes can be given a Linked Cell to store either True or False in depending on whether or not the check box is checked.

 

So you could test those and in effect adjust the formula.  You just need a short IF statement for each of the checkbox link cells.  Lets say those are all over in column Z, from Z1 on for each of your checkboxes.  You can always hide that column if it is an annoyance, and set your PrintArea up to ignore it for printouts also.

 

Your formula could look something like this:

 

=SUM(IF(Z1,A1,0), IF(Z2,A2,0), IF(Z3,A3,0))

So if the checkbox associated with cell Z1 is checked, then it will add in the value from cell A1, same for checkbox associated with Z2 (add in value from A2), etc.  for as far as you care to go.

 

Since the values in Z1:Zn  are TRUE or FALSE, the IF() statements work properly as I've shown here.
