Hi Arsalan,

This error in MS Word is commonly faced by many users when they try to update changes made to a Word file by clicking on save. The causes behind this error message can be:

  • If the document is previously saved as ‘read only’ file / Template. This means that the file contents cannot be modified.
  • If user try to modify content of a file that is saved in a location, which user cannot access i.e. not having admin permission.
  • Trying to modify a file from a network shared folder.
  • When Windows Search service is enabled.
  • When multiple antivirus programs are running on the system.

When this error is displayed, modified content of the Word file cannot be saved. But, there are certain solutions to successfully save the file even when this error message is getting displayed. They are:

  • Click on Save As and save it in a new location.
  • Before editing the file, check whether the file is saved as Read-Only. If so then change the preferences.
  • If you do not have sufficient user permissions to modify any file, login as administrator.
  • Disable Windows Search service. Go to Start > Click on Run > Type services.msc > locate Windows.
  • Search and double click on it > Then click startup type > select Automatic (Disabled) > apply.
  • If the system has many antivirus programs running, disable all except any one.
  • Last but not the least, make use of inbuilt repair utility i.e. ‘Open and Repair’ option to repair the Word file.

Let us know how it goes after performing the aforementioned solutions. We'd be glad to further assist you.

john korchok say:

You do this by inserting an Equation field. Here's a page that gives you the steps: Superimposing Characters

Although this is written for Word for Windows, follow the Word 2003 steps in 2011 and it will work. Equations allow more flexible positioning of characters.

Later edit: I also think you can find many useful characters in Unicode fonts. As an example. U+2611 is a checkbox with a check in it. Here's a page that has more details, plus a link to show which fonts contain that character: Ballot Box with Check Character

To make it simpler to access Unicode characters, open System Preferences>Keyboard>Input Sources, click the plus sign, scroll to the bottom and choose Others. In the right panel, select Unicode Hex Input and click on Add. Then, in the finder, click on the flag icon in the OS X menu bar to switch from your usual input language to Unicode Hex Input. Finally, in your Word document, select a Unicode font , hold down the Option key and type 2611 to enter a checked checkbox.
