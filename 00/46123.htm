rajithr say:

Hello,

To send or receive a file using Bluetooth, the receiving device must select “Receive a File” to wait for a connection.  On the receiving device, click on the Bluetooth icon in the notification area (the bottom corner of the taskbar), and select Receive a File.

Then from the sending device, you can right-click the Bluetooth icon and select Send a file, and follow the instructions on the screen.

If you don’t see the Bluetooth icon, you can enable the icon by going to Settings > Devices > Bluetooth > More Bluetooth options.  In the Options tab, make sure the Show the Bluetooth icon in the notification area is enabled.

Also, make sure both devices have all options enabled in the Options tab under Settings > Devices > Bluetooth > More Bluetooth options.  And then check the Share tab, and make sure Allow remote devices to browse, send, and receive pictures, music, and other files is enabled.
