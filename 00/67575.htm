lemp say:
I recently got a new computer and was using the default administrator account which came with it.

That's a bad idea.  You should save the built-in account named Administrator to help you recover when (not if) your day-to-day computer administrator account gets corrupted.

To answer your question, Windows XP encourages the behavior described in the previous paragraph by removing the built-in Administrator account from the Welcome screen as soon as you create a computer administrator account.

You should be able to access everyone's files from your new computer administrator account, even those of Administrator.  However, if necessary, you can access Administrator either by starting in Safe Mode or while at the Welcome screen with no other account logged in, press CTRL+ALT+DEL twice to get the "classic" logon window.  Enter Administrator as the user and whatever password you assigned.  If you didn't create a password for Administrator, the default is blank (i.e., no password).

If despite the warning you still want to use Administrator, you can download TweakUI and use that to display Administrator on the Welcome screen, For links to TweakUI, directions, and another method to accomplish this, see http://windowsxp.mvps.org/admins.htm

The “Change only the text size” option was turned off in Windows 10 Creators Update. Unfortunately, the setting was not supported throughout the entire system. For example, it didn’t work with the Start menu, Cortana, Edge and some applications. 

For an alternative method of changing your text size, you can use the Scale and layout options within Settings > Display. Use the dropdown under “Change the size of text, apps, and other items” to make things bigger or smaller.

If you’d prefer your text, apps, and other items to be even bigger or smaller, click the Custom scaling feature to go outside the ranges that Windows recommends for your hardware’s screen size and resolution.

If the text looks fuzzy or blurry when you use custom scaling, this might happen if the custom size is not set to a 25% increment. For example, if your custom scaling is set to 157%, try changing it to 150% or 175%.

C:\Windows\System32\

 

and rename the Catroot2 folder to Catroot2.old
