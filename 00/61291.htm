How do I change the Font size on Windows Live mail 2012? - Microsoft Community

michael edwards14 say:

How do I change the Font size on Windows Live mail 2012?

How do I change the font size on Windows Live Mail 2012?
¡firedog say:

Font size of what?

  • Items in the folder pane, on the command ribbon, in the message list, in the calendar window, in the contacts list and RSS feeds: increase the screen DPI. How to do this depends on your version of Windows. This will affect all text.
  • Received plain text messages in the reading pane or in a message window:
    • In Internet Explorer, on the View menu, click Text size and select a larger size (Alt-V X)
    • In Internet Explorer, increase the zoom (Ctrl +, that's Control-Plus)
    • In Windows Live Mail, when viewing the message list, press Ctrl-Shift-O for Options. On the Read tab, click Fonts, then select a font and font size for each encoding you're likely to need. If you don't know what these are, select fonts for Western European and Unicode.
  • Received HTML messages in the reading pane or in a message window: some of the above methods may affect the size of text in these messages. It depends on how the sender formatted the message.
  • Messages you compose: when viewing the message list, press Ctrl-Shift-O for Options. On the Compose tab, click Font Settings and select the size you'd like.
