vipin a s say:

Word document with chapters

I need to format my doc with chapters. The heading should be like this

Chapter 1

Title of Chapter

suzanne s. barnhill say:

The simple answer is to use a line break (Shift+Enter), but before you take this easy road, consider whether (a) you will be adding a table of contents and (b) whether you will be including the chapter number in caption labels (Figure 1.1, for example). In either of those cases, you may want to consider making the chapter number and chapter title separate paragraphs in different styles, and (even if you don't plan a TOC or including the chapter number in captions labels), you may want to use autonumbering for the chapter numbers just because it makes numbering the chapters easy and prevents you from using the same number twice (I typeset books, and you wouldn't believe how often this happens).

It is easy to set up autonumbering to generate "Chapter 1," "Chapter 2," etc., automatically: just apply the list template shown in the Multilevel List dropdown that already shows "Chapter 1" for the first level. By default the numbering is linked to Heading 1, but you can change that if necessary (see below).

You'll need to open the Define New Multilevel List dialog to make a few changes (including the linked style if desired). Click More>> to expand the dialog. Even though the chapter number paragraph will be centered, leave the number alignment set to Left (this is counterintuitive but necessary), but set "Follow number with" to "Nothing." This will ensure that the paragraph is truly centered.

For the chapter title, you'll use a different style. This allows you to have the chapter number and chapter title in different fonts, if desired, and/or at different font sizes, and to fine-tune the spacing between them using Spacing Before/After. This can result in a much more professional appearance.

With regard to the TOC, please see http://wordfaqs.mvps.org/TOCTips.htm#UnnumberedHeadings. The gist of this discussion is that, if you use the line break (as first suggested), then what you are going to get in the TOC is "Chapter 1 Title of Chapter" instead of, say, "1. Title of Chapter." By using two different styles for the chapter number and chapter title, you can set up the TOC to use the formatting you choose.

suzanne s. barnhill say:

I assume by "index" you mean a table of contents. The TOC knows what the page number is because every page has a number whether or not that number is displayed or printed. If you want the number to be printed, you have to insert a PAGE field in the header or footer of the page. The easiest way to do this is to access the header/footer, place the insertion point where you want the page number, and press Alt+Shift+P.

In documents with multiple sections, headers and footers are usually linked by default, so inserting it in any section will insert it in all. If you need to omit display of the page number in some sections, you'll need to unlink the header/footer of such sections from those preceding and following. Additionally, by enabling Different First Page and/or Different Odd and Even,  you can have up to three separate headers and footers in a given section. For more, see http://wordfaqs.ssbarnhill.com/HeaderFooterRibbon.htm.

Remove Shortcut Virus IEXPLORE.VBS From Flash Drive And Computer - Microsoft Community

0125 say:

Remove Shortcut Virus IEXPLORE.VBS From Flash Drive And Computer

I got infected by this virus through my USB drive yesterday. It took me three hours to figure out how to remove it. So I wanted to share my steps.

Symptoms: The worm creates shortcuts for any files and folders placed onto the USB drive and makes the original files invisible.

The created shortcuts point to the following path:

C:\Windows\system32\cmd.exe /c cls&cls&cls&cls&cls&cls&cls&start weiche.JPG&cls&cls&cls&cls&cls&cls&start iexplore.vbs&cls&cls&cls&cls&cls&cls&exit

Anti-Malware tried: I ran a scan with the following tools, none of them detected the worm:

Microsoft Security Essentials

Malwarebytes

Hitman Pro

ADWCleaner

TDSSKiller

Norton Power Eraser

Microsoft Safety Scanner

Solution:

  1. Kill wscript.exe in task manager
  2. Remove iexplore.vbs from startup (it created three different entries)
  3. Make files on USB drive visible: Open cmd, type attrib -h -r -s /s i:\*.* (with i: being your drive letter)
  4. Delete those files on USB drive and format just to be sure
  5. Go to C:\Users\{user}\AppData\Roaming\Internet Explorer\ and delete it. The folder seems to be empty but inside sits iexplore.vbs

Good luck.
