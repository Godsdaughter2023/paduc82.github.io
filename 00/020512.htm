romel ram say:
Hi Karen,

We understand that you'd like to set your custom template as default template on Outlook 2016.

You can follow the steps below as your reference:

1. In the Mail view, click Home > New Email to create a new email. Compose the new email, and then click File > Save As.

2. In the opening Save As dialog box, please do as follows:

(1) Open the stationery folder with pasting the address of %appdata%\microsoft\stationery into to the Address box and press the Enter key; (2) Type a name for the new stationery in the File name box; (3) Select HTML from the Save as type drop down list; (4) Click the Save button.

You have now created a custom stationery in Outlook. Close the new email.

3. Click File > Options to open the Outlook Options dialog box.

4. In the Outlook Options dialog box, click Mail in the left bar, and then click the Stationery and Fonts button under the Compose messages section.

5. Now, the Signatures and Stationery dialog box comes out, click the Theme button.

6. In the popping out Theme or Stationery dialog box, click to select the new created stationery in the Choose a Theme list box, and click the OK buttons successively to close all dialog boxes.

From now on, when you click Home > New Email to create a new email in Outlook, the specified stationery will be applied to the new email automatically.

jeff ford_68 say:

Hi Karen,

I have been struggling with this same exact issue, but I was recently able to figure out how to update the default shape settings on the default email template.  Here are the steps that I used:

  1. Close out of Outlook (need to make sure the default email template isn't locked).
  2. Navigate to C:\Users\user name\AppData\Roaming\Microsoft\Templates where user name is your user name.
  3. Open the NormalEmail template (NormalEmail.dotm)
  4. When the template opens in Word, click Insert and insert the shape into the document.
  5. Set the desired fill, line color, and weight for the shape.
  6. Right click on the shape and click Set as Default Shape.
  7. Delete the shape so that the file is blank (or however you want your default email to look).
  8. On the File tab, click Save.
  9. Restart Outlook.

Your default shape should now be how your custom settings.

steve rindsberg say:

Would it make more sense to skip the color change animation and just use VBA to change the colors in the first place?

This might make things simpler.

For example:

 

Sub ChangeColor(oSh as Shape)

oSh.Fill.ForeColor.RGB = RGB(255,0,0)

End Sub

 

If you assign a Run Macro action setting to a shape and choose ChangeColor as the macro, then during a slide show, when the user clicks on the shape, it'll run the macro and change the shape's fill color to red.

By substituting variables for the values passed to RGB(x,y,z) above, you could customize the fill color.

 

john sr wilson say:
If you are clicking the shape that needs to change colour and vba is OK in the presentation then Steve's code is a good solution.

If you need to change the animation colours though then you can do this with vba as well. This code assumes they are triggers not simple page clicks.

Sub col_swap()
Dim oeff As Effect
Dim osld As Slide
Dim i As Integer
For Each osld In ActivePresentation.Slides
For i = 1 To osld.TimeLine.InteractiveSequences.Count
For Each oeff In osld.TimeLine.InteractiveSequences(i)
If oeff.EffectType = msoAnimEffectChangeFillColor Then
'Change RGBs to suit
If oeff.EffectParameters.Color2.RGB = RGB(0, 255, 0) Then _
oeff.EffectParameters.Color2.RGB = RGB(255, 255, 0)
End If
Next oeff
Next i
Next osld
End Sub

If this is how you would fill out the table (with only the appropriate time period filled in, with the others blank), then the formula for the EXTENDED cells could be this in F2, with this table in A1:F3, then copied down.

=A2*B2*SUM(C2:E2)

Quantity Duration Time frame DAY WEEK MONTH EXTENDED

The formula - now in G2 - would be

=A2*B2*INDEX(D2:F2,MATCH(C2,$D$1:$F$1,FALSE))
