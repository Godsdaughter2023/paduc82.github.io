jllatham say:

Warning #1: Test all of this out on a copy of your real workbook since it is a 'data destructive' process.

What this does: it copies all of the worksheets in the workbook that the code is in into a new workbook.  Then it goes to each sheet in that new workbook and turns the formulas in cells into values, and it also removes all hyperlinks from those sheets.  If all goes well, you won't even have to any resetting of anything in your regular stock pricing workbook.

It then returns to your source workbook and tells you the job is done.  The other workbook will be hidden behind the original, you can use View --> Switch Windows to bring it into view and save it with a new name and distribute it as needed.

The code goes into a regular code module, see this page for instructions on how to put it into one:

http://www.contextures.com/xlvba01.html#Regular

The code:

Sub MakeCopyForDistribution()
  Dim newWB As Workbook
  Dim anyWS As Worksheet
 
  Worksheets.Copy ' copy all from this workbook into a new workbook
  Set newWB = ActiveWorkbook
  For Each anyWS In newWB.Worksheets
    'converts all formulas to their values
    anyWS.UsedRange.Formula = anyWS.UsedRange.Value
    'deletes any hyperlinks on the sheets
    anyWS.Hyperlinks.Delete
  Next
  ThisWorkbook.Activate
  MsgBox "Values only copied into workbook: " & newWB.Name
  Set anyWS = Nothing
  Set newWB = Nothing
End Sub

paulcordts say: The easiest way would be to save a copy under a different name and then just break the links before sending.

In Excel 2007
Click the Office Button (The round thing in the upper left corner)
Select Prepare, then select Edit Links to files.
You can select all files by using (Shift + mouse click) then click the Break Link button.

If you really want a macro this will do the same thing

Sub Linkbreaker()
Dim WBlinks As Variant, A As Integer

WBlinks = ActiveWorkbook.LinkSources(Type:=xlLinkTypeExcelLinks)

If Not IsEmpty(WBlinks) Then
    For A = UBound(WBlinks) To 1 Step -1
        ActiveWorkbook.BreakLink Name:=WBlinks(A), Type:=xlLinkTypeExcelLinks
    Next A
End If

End Sub

scottgem say:

If you want to schedule a report to run what I do is create a separate front end with either an Autoexec macro or use the On Open event of the default form. This front end is then run using Windows Task Scheduler at the selected time. 

If you want this report to run whenever the app is open, then again use an Autoexec macro or the On Open event.
