·        ipconfig /release in the command prompt and press ENTER

·        ipconfig /renew in the command prompt and press ENTER

Both the SPs, if downloaded and installed correctly, should not present problems in running FSX.   They have been proven over these many years.  Your downloaded file/installer should look like this:  fsx_sp2_ENU.msi, if you have got a genuine copy and not a crack.

The usual way to correct any issue of this type, would be to first uninstall SP2 from your Uninstall program list (in Control Panel > Programs > Programs & Features).   You will see it as FSX SP2.  This only uninstalls the SP, not FSX. Once this SP is removed, then insert Disk 1 into the Drive and select the Repair Option.   This will replace any missing files in your FSX installation, but be aware, it may also remove any third party add ons you may have installed and also alter some settings (e.g. keyboard/joystick mapping, FSX configuration file, etc.).
