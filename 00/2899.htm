graham mayor say:
You can do it with a script run from a rule. Set the rule to identify the incoming messages by address then run the following script from the rule to move the message to a sub folder of Inbox called Foldername :

Sub MovebyTime(oItem As Outlook.MailItem)
    With oItem
        If Format(oItem.ReceivedTime, "h") >= 3 And _
           Format(oItem.ReceivedTime, "h") < 4 Then
            .Move Session.GetDefaultFolder(olFolderInbox).folders("Foldername")
        End If
    End With
End Sub

danbuie say:

Expiry time URL to Skype chat

How long does the URL for a Skype chat last? Can I rejoin the conversation weeks after it was created? I will be joining from iOS app and need videoconferencing capabilities.
susan_s. say:

Hello DanBuie,

Thank you for your reply.

As long as the Group profile is active, all people whom you sent the link will have the opportunity to join the group and there is no expiration for this.

To change the settings, please go through these steps: (You need to sign in to the Skype app to perform this action)

1. Click on the Group profile picture or click Settings (gear icon). A new window will pop-up.

2. Under Administrator settings, you will be given two options: Make chat history available to everyone and Make a link for anyone to join.

Make chat history available to everyone - means old chat will be available even to new member of the group.
Make a link for anyone to join - allows you to share the URL for new contacts to join the group.

For more information, kindly refer to this link below:

https://support.skype.com/en/faq/FA34572/how-do-i-join-a-skype-conversation-using-the-link-i-received
