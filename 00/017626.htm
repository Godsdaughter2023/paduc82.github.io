First of all, do you really need the Pro version

http://answers.microsoft.com/en-us/insider/wiki/insider_wintp-insider_install/what-are-the-editions-of-windows-10-available/195a4038-c061-4902-bc2b-35922c9d41bc

You need a backup strategy (up to you which software you use). Whether you want to do a disc image and file backup at each stage is again up to you.

(1) Back up existing 8.1 system partition

(2) Upgrade 8.1 to W10 and back that up (especially if this is the one you want to use)

(3) Do a clean install of W10, make such configuration changes as required, and back up as a baseline disc image.

(4) Either reinstall (2) or continue to install programs and files to (3). Back up when in working order.

(5) periodically (day/week) do an automatic incremental file backup. Every month do a working incremental disc image. These are in the event of a disc failure or if something serious (eg virus) goes wrong.

(6) Every 6 months reinstall the base line disc image and allow all newer updates to install. Back this up and reinstall working version.

Installing a disc image only takes a few minutes. Hopefully, what ever happens, it will not take much longer than that to be up and running.

Hi LuRe,

The Full Access permission is required to access other’s mailbox.
