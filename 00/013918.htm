The name "theme body" refers to "The font that is specified as body text by the current document theme."

I don't use themes at all, because they create too much confusion for me.  If you set the document theme to be "Office" (normally the theme in the top left corner) then "Theme Body" should be "Calibri" font, and "Theme Headings" should be "Cambria" font.  Of course, if you have edited your themes, they can be anything.  If you use a different theme, the specific font will be different.

The idea of themes is that if you change theme, ALL of the styles in the document will change to the fonts and colours set in the theme you changed to.  So if you change to Office theme, all the body text styles will change to Calibri font, and all the Heading styles to Cambria.

To say that another way:  "Theme Body" is not a font, it is a pointer to the place where Word stores the theme. In the Theme, there is a slot named "Body", and in there is a pointer to whichever font the theme has defined as its body font.  When you change the theme, you change the pointer in the Body slot to point to a different font.

The PowerPoint professionals use Themes to set the overall look-and-feel of their slides.  But in Word, I have never found any use for themes, they just confuse users, who can't find any proper explanation of them.  What we WANTED was "Cascading Style Sheets". A theme behaves a little like a cascading style sheet.  But it doesn't contain most of the commands, and it doesn't cascade.  Pretty useless, if you ask me...

To avoid this nonsense potentially messing up all my styles, I set my styles to the font I want. For example, I set the font to "Calibri" not "Calibri (theme body)". That breaks the link between the theme and my styles (provided you do the same change for all the styles you are using).  Which means some idiot can't come along and wreck havoc with my formatting by changing theme.  They can change theme all they want, but the content in my document will continue to ignore the change until they get sick of it and go away :-)

So my suggestion is to set "Office" theme.  All your body text fonts (including Normal) should t hen snap back to Calibri font.  You can then set the fonts you want in the individual styles, avoiding any font that describes itself as a "Theme" font.

I'm not sure what you found online nor will I be so presumptuous as to pick it apart :-) but unless this Tutorial by MVP Suzanne Barnhill is what you're attempting to follow it's not surprising that your result isn't working out:

“How to set up a letter template”—how to create a letterhead, have different margins on the first page, set up letter styles, etc.

In more than 20 years these are the only effective & definitive instructions I've seen for creating 2 page Letterhead templates as well as other documents of similar ilk.

Just a couple of additional points:
  • What Phillip intimated about documents behaving differently on different computers is true, even if not for the specific reasons he suggested. However, it isn't the "Page and Margins", it's the flow of the content within those boundaries. Differences in Fonts & printers are the most common reasons.
