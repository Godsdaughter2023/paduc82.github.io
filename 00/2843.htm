mohan suryanarayan say:

Hello,

Welcome to Microsoft Community.

 

To resolve the issue I suggest you to try uninstalling and reinstalling Outlook 2013 only. Follow these steps:

 

Note: Back up the registry key before you modify or delete any key:

 

  1. Press the Windows key + R.

  2. In the Run box that opens type Regedit and press Enter.

  3. The User Account Control window opens. Click Continue.

  4. Click on File > Export.

  5. In the File name box, type a name that you can remember, such as Registry Backup.

  6. Select a location to save the Registry entries (.reg) file/s.

  7. Click Save.

 

Now delete the following Registry Keys:

 

  1. Click on Start > Control Panel > Programs and/or Programs and Features.

  2. Right-click on Microsoft Office 2013 and click Change.

  3. Click Continue for Add/Remove features.

  4. Click on the little arrow attached to Microsoft Outlook and select Not available > Continue.

  5. Click on Start > Run (Windows key + R) and type regedit, click OK.

  6. Locate the key HKEY_CURRENT_USER/Software/Microsoft/Office.

  7. Delete the Outlook key under Office 8.0/9.0/11.0/12.0/14.0

  8. Go back to the top.

  9. Locate the key HKEY_LOCAL_MACHINE/Software/Microsoft/Office.

  10. Delete Outlook key under Office 8.0/9.0/11.0/12.0/14.0

  11. Close Registry Editor.

 

You may now repair Office 2013 following the steps from the article below:

http://support.microsoft.com/kb/2769349

 

Check if the above step helps to resolve your issue.

 

Thank you.
