Let's say your businesses are listed in A2:A99 and the amounts (sales revenues) are in D2:D99. If you create a drop down data validation list of the businesses in Z1 then you can use the selection as criteria in a SUMIF() function.

=SUMIF($A$2:$A$99, Z1, $D$2:$D$99)

Multiple criteria can be accommodated with SUMIFS(). Note that there are some syntax differences between the two functions.

hansv mvp say:

If the files are in the same folder as the workbook, the hyperlink address consists of the file name only, without the path.

After moving the workbook, the address must be changed to include the path (but not file:\\\):

Sub FixHyperlinks()
    Dim wks As Worksheet
    Dim hl As Hyperlink
    Dim sNew As String
 
    Set wks = ActiveSheet
    sNew = "G:\Folder1\"
    For Each hl In wks.Hyperlinks
        hl.Address = sNew & hl.Address
    Next hl
End Sub
