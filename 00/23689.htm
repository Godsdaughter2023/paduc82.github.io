terryatscholars say:

When reading a text file, need to append multiple lines

I am trying to write a VBA routine that will read lines from a text file.  In this particular text file, lines longer than 75 characters are wrapped to the next line, and that line always begins with a <space> character.  I can, of course, check that the line begins with a <space> character, but how do I then say "go back to the last line and append this one to it"?  There could be more than one wrapped lines that start with a <space>.  Do I need to do something with the pointer and the line count?

macropod say:

You could add each line to a variable that you clear and reset every time a line that doesn't start is a space is encountered. That way, you never need to go back. For example:

Sub ProcessTextFile()
Dim strLn As String, strTmp As String, x As Long
Open "C:\Test.txt" For Input As #1
Do Until EOF(1)
  Input #1, strLn
  If Left(strLn, 1) = " " Then
    strTmp = strTmp & strLn
  Else
    'Do something with strTmp, followed by:
    strTmp = strLn
  End If
  x = x + 1
Loop
Close #1
End Sub

andreas killer say:

Alternatively you can read the file in one step and split the the lines into an array and process that.

Andreas.

Sub Test()
  Dim fso As Object 'Scripting.FileSystemObject
  Dim ts As Object 'Scripting.TextStream
  Dim FName As String, Contents As String
  Dim Lines As Variant
  Dim i As Long, k As Long
 
  FName = "C:\test.txt"
 
  'Read the whole file in one step
  Set fso = CreateObject("Scripting.FileSystemObject")
  Set ts = fso.OpenTextFile(FName, Format:=-2)
  Contents = ts.ReadAll
  ts.Close
 
  'Split lines
  Lines = Split(Contents, vbCrLf)
  k = LBound(Lines)
  For i = LBound(Lines) + 1 To UBound(Lines)
    If Left(Lines(i), 1) = " " Then
      'Concat to previous line
      Lines(k) = Lines(k) & Lines(i)
    Else
      'Next line
      k = k + 1
      If i > k Then Lines(k) = Lines(i)
    End If
  Next
  'Remove unused slots
  ReDim Preserve Lines(0 To k)
 
  'Write into the sheet?
  'Range("A1").Resize(UBound(Lines) + 1).Value = WorksheetFunction.Transpose(Lines)
End Sub
