Comctl32.ocx is a Visual Basic 6.0 runtime file that's not included in Windows 10. The golf program which was presumably developed using VB 6.0 needs to supply the required runtime libraries.

Support Statement for Visual Basic 6.0 on Windows Vista, Windows Server 2008, Windows 7, Windows 8 and Windows 8.1, Windows Server 2012, and Windows 10

Download VB 6.0 Common Controls

However, you may try the Visual Basic 6.0 Common Controls (includes comctl32.ocx) from the following Microsoft Download Center link: https://www.microsoft.com/en-us/download/details.aspx?id=10019

Don't run the setup. Instead, open the exe file using 7-Zip or any other archiving utility and extract the file Comctl32.OCX file to the following folder:

Copying the OCX file:

For Windows 64-bit systems, extract the OCX file to: C:\Windows\SysWOW64

For Windows 32-bit systems, extract the OCX file to: C:\Windows\System32

Register the OCX file:

Right-click Start, click Command Prompt (Admin)

If you're using Windows 32-bit, type the following command and press ENTER:

regsvr32 comdlg32.ocx

If you're using Windows 64-bit, type the following command and press ENTER:

C:\Windows\SysWOW64\regsvr32   C:\Windows\SysWOW64\comdlg32.ocx

Try launching the golf program now.

The controller is the chip on the MB that makes the ports work. Unfortunately, it's not user replaceable.

You need to press a key before you get to the F8 options. Usually it's Delete or F2 or F10. It should be listed in your manual how to get into the BIOS.
There is a jumper on the motherboard that will reset the BIOS back to defaults. You have to remove the battery, then jump the 2 pins with a jumper for 30 seconds or so, then remove the jumper and put the battery back in. The PC has to be unplugged from the wall before doing this.

"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

Here are the possible reasons why the battery won't charge:

I do not know how to change the hard drive or what one looks like, so would welcome some help with this. Would it be better to have an external hard drive, & if so, how do I install it.  Thank you.

hairyfool say:

You cannot use an external drive as a replacement for the internal drive.

To replace the existing drive may require extensive disassembly depending on the model of laptop. If you don't know how then it may be better to trust it to a repair workshop. Apart from the physical act of replacing the drive you would also need to know how to transfer Windows to the new drive.

To get an idea of what is involved do a web search for videos of replacing laptop drives for your model of computer. It could be as simple as removing two screws and it slides out but it could equally be 20 screws and unclipping the entire case.
