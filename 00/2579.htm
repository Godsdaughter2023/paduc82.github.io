rich007a say: Hi,
You could use the Workbook Open event to clear the autofilters.
Press Alt+F11 then double-click on the ThisWorkbook module of your file in the top-left pane in the VBE.
Paste this code on the right.
You will then need to save the file as macro enabled .xlsm and ensure users enable macros when the file is opened.
Here's the code:
 
Private Sub Workbook_Open()
    Dim ws As Worksheet
    For Each ws In Worksheets
        If ws.AutoFilterMode Then ws.AutoFilter.ShowAllData
    Next ws
End Sub
