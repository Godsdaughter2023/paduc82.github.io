j w stuart say:

Lastly then, if I do buy another retail licence, is it transferable should I wish to use it elsewhere eventually? 

A "Full" retail version is transferable and requires that "NO" prior version of Windows is pre-installed and activated.

An "Upgrade" version is transferable but "requires" that a previous version of Windows is installed.

An "OEM" version is "Not" transferable (its tied to the first computer / motherboard it is installed on and can only be used to perform a "Custom" / Clean install, meaning you can't use it to upgrade from a prior version of Windows.

Note: Due to the scarcity of "Retail" version the cost $$$ is no longer practical in most cases as you could purchase several OEM versions for the cost of a single "Retail" version of Windows 7 Pro.
