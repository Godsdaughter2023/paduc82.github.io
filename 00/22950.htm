Hi

 

If focused Inbox has come out, it means the clutter has been replaced by it. Clutter will not work as expectedly after you turned off focused inbox. I suggest you can try to do the following:

  • Right-click the Clutter folder and the select Manage Clutter.
  • Sign in to Outlook on the web .

  • Select Separate items identified as clutter > Save

 

If you're not currently in OWA:

  • At the top of the page, select Settings, and under Your app settings, select Mail.
  • Under Automatic processing, select Clutter.
  • Select Separate items identified as clutter > Save

 

After doing that, you can move some emails to the clutter and wait for some time to check whether the clutter works.

anula d'souza say:

Hi Jevenson,

Thank you for posting in Microsoft Community.

 

As per the issue description when you click on Reply in any email you receive an error message. Let me assist you with the issue.

 

You usually receive this error message is the issue is with the profile. Create a new profile; configure the email account in new profile and check. Refer to the link mentioned which explains how to create new profile:

http://support.microsoft.com/kb/829918

 

You may also refer to the knowledge based article mentioned as reference which discusses about similar error message:

http://support.microsoft.com/kb/312354

 

Regarding Outlook created a separate inbox;

Do you see only the Inbox folder or you see the complete PST file which includes this Inbox folder?

Are you viewing the inbox folder under Favorites?

 

If you have selected the option to create a new PST file while configuring the email account then all the emails received in that email account on web-mail will be delivered to the appropriate Inbox folder in Outlook.

If you want the emails to be delivered to appropriate PST file then while configuring the email account you have to select the option Existing Data file and browse to the location where the PST file is saved.

OR

If you have already configured the email account and want to deliver the emails to a particular Inbox folder the;

Click on File< Account Settings< Email tab< highlight the email account, at the bottom of the window click on Change folder, select the appropriate Inbox folder and click Ok.

 

Note: The above steps are applicable only if you have configured POP account. If you have an IMAP or MAPI you won’t see the option Change Folder or get an option Existing data file while configuring the email account.

 

I hope the above suggestions help to resolve the issue. If you have any other Office query let us know and we will be glad to assist you.
