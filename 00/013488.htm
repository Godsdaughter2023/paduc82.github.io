Another alternative is to define a new paragraph style in the document, setting at least the font color in the style. Assign a keyboard shortcut to the style.

When you type a response, while the cursor is still anywhere in the same paragraph, press the shortcut to apply the style. The color will be changed to whatever is defined in the style.

If your responses are sometimes more than one paragraph, be sure to set the "Style for following paragraph" in the style definition to the same custom style. Then apply the style while you're entering the first paragraph of a response, and the same style will be automatically applied to the new paragraph when you press Enter.

germain navas say:
Hello !

A customized Office instalation is possible following these steps :

First, download the ODT tool from the page:
https://www.microsoft.com/en-us/download/confir...
We execute the downloaded file. This generated two files called "Setup" and "Configuration.xml"
Both files are placed in a new folder that we named "ODT" on disk "C"
Then we modified the file "Configuration.xml" with the editing tool http://officedev.github.io/Office-IT-Pro-Deploy.... The desired version is Office proplus 32 bits, in English just Word and Outlook. The file is as follows (this is just an example, you must generate your own file )

<Configuration>
<Add OfficeClientEdition="32" Channel="Broad">
<Product ID="O365ProPlusRetail">
<Language ID="en-us" />
<ExcludeApp ID="PowerPoint" />
<ExcludeApp ID="OneDrive" />
<ExcludeApp ID="OneNote" />
<ExcludeApp ID="Lync" />
<ExcludeApp ID="Groove" />
<ExcludeApp ID="Excel" />
<ExcludeApp ID="Access" />
<ExcludeApp ID="Publisher" />
</Product>
</Add>
<Updates Channel="Broad" Enabled="TRUE" />
<Display Level="Full" AcceptEULA="TRUE" />
</Configuration>

We already have the instructions of the version of Office that we want. Now we download Office with this command from the Command Promt as administrator:
c: \ odt> setup.exe / download c: \ odt \ configuration.xml

Finally, we install Office with this command:
c: \ odt> Setup.exe / configure c: \ odt \ configuration.xml

Finally, this is the tool to uninstall Office in case you had a previous installation on a computer:
https://support.office.com/es-es/article/Desins...

Hope it hepls !!
