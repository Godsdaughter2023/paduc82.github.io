neilsmith1984 say:

As Stuart said, these games never had digital versions on 360. When Xbox 360 launched, disc-based games never had digital copies in the store. It wasn't until a couple of years later that they started doing 'games on demand' which meant AAA studios could release their games digitally, rather than just small XBLA titles.

leopardclaw,
Is the drive spinning continuously or is it stopped?  If it is spinning continuously it is not going to allow the disk to be ejected.  What software were you using to watch the movie?  Have you tried loading up the software again to see if the movie will play?  Then you can tell it to stop and eject the disk.  If the drive is not spinning, go into device manager and see if it sees the drive and if it has any exclamation point over it.  Try to reinstall the driver to see if that gets the drive working again.  If it is not spinning and does not show up in device manager, it is possible that the drive went bad and you may have to get a computer tech to look at it.
