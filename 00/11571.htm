brian tillman [outlook mvp 2007-2019] say:

> Can someone guide me step by step to upload the PST file without using drive

> shipping or network upload methods?

Obviously, you can't put anything "in the cloud" without network upload. The "cloud" IS a network - the Internet - and an Office 365 mailbox requires networking in order to access it. So, if you want to make the contents of a PST available in the Office 365 mailbox, you'll need to upload that contents.

A PST itself can't be store in the cloud, but you can certainly move its contents there for storage. Connect Outlook 2016 to your Office 365 mailbox using an Exchange account, connect the PST to Outlook 2016, and then copy the data from within the PST to the folders i the cloud mailbox. When you're done copying, the data will then be available to any client program you also connect to that mailbox and it will also be available if you access the mailbox using a web browser.

  • The camera won't open
  • The device starts using too much battery

mike middleton say:
What am I doing wrong? 
Several things:

(1) I think you're using a Line chart type instead of the XY Scatter chart type. With a Line chart, you could use 6,18,30,... for the x-axis labels, but the numerical values 1,2,3,... are being used for the trendline calculations.

(2) Those coefficients of the 5th order polynomial actually have 15 significant digits. You're using only a few digits in your calculations. I suggest using the array-entered worksheet function LINEST to obtain the coefficients for your calculations.

(3) Generally, a 5th order polynomial is an example of the statistical problem called "overfitting." (You can use Bing or Google to search for more info about "overfitting.") However, the 5th order polynomial might be an expedient way for you to estimate values for the gaps.

For more discussion of these issues, see "Exel 2007 displays inacurate trendline for high order polynomials" at

http://answers.microsoft.com/en-us/office/forum/office_2007-excel/exel-2007-displays-inacurate-trendline-for-high/1300c027-4470-e011-8dfc-68b599b31bf5
