Safe mode has helped similar issue. So, let’s start Outlook in safe mode following steps mentioned below and verify the result.

* Copy and paste, or type the following command in the Open box, and then press Enter.
Outlook /safe Note: There is a space between Outlook and '/'

a) Click on File menu, click Options, Add-ins, Go button besides Manage: Com-in Add.

c) Close the Office program and restart it.
d) Disable each add-in one at a time, restart the Outlook, and repeat the above procedure.

If the issue reappears again, it will help you to determine which add-in is causing this issue.
