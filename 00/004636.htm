Thanks in Advance

vijay a. verma say:

If your cell is formatted as date / time so that value appears as 02:50:09 or 00:10:26 so you can multiply that cell with 86400.

Hence, if your value is in cell A1, the formula would be

=A1*86400

jgkzdl say:

If 2h 50m 9s is a string in cell A1, then you can use the following formula to convert that string into seconds:

=TIMEVALUE((SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(IF(ISERROR(SEARCH("h",A1)),"00h" & IF(ISERROR(SEARCH("m",A1)),"00m" & A1,A1),A1),"h",":"),"m",":"),"s","")))*86400

The formula works even for strings like 3h 7m or 7m 3s, or 8s, but not for 8h 5s. So if only the minute value is missing the formule will not work.

As Palcouk noted, Office 2007 is a Windows PC-only program.  Like all other Windows PC-only programs, it will not install or run on a Mac natively.

In order for it to run on your Mac, you would have to install a Windows Operating System emulator program, or a Windows Virtual Operating System program, onto you Mac, and then install Office 2007 into that emulator/virtual OS.  Search on VMWare Fusion and Parallels Desktop to see more about this.

You could also start your Mac into a Windows Operating System using Bootcamp, but then you'd have to buy a licensed copy of Windows to do so, so it might not be cost-effective---all depends upon your situation.

Microsoft does sell Office versions for Mac, too.  Office:Mac 2011 is the most recent traditionally installed version of Office for Mac.  The Office 365 Home Premium product works with PCs, Macs, and to a lesser extent mobile devices like tablets and phones.  One way to do it would be to uninstall and sell your Office 2007 (assuming it's a "full retail/FPP" license) and then use that money toward either Office:Mac 2011 or Office 365 Home Premium.

Yet another option is to just use the free Apple apps you are entitled to download when you buy a new Mac (the suite that some of these apps came in used to be called iWork).  The programs mimic the functions of the three major various Office programs:  Word/Pages, Excel/Numbers, and PowerPoint/Keynote.  If you don't need a lot of interoperability with MS Office users and are just doing things for yourself, using the free Apple apps might be all you need.
