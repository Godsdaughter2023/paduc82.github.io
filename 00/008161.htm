ossiemac say:

I have made an assumption that if first row not blank then the first group is from row 1 to the first blank. I believe that the only way to achieve the desired result is with VBA code. The code does not invoke the Conditional Format functionality; it simply formats using the code.

 

To install the code:

Right click the worksheet tab name and select View code

Copy the code below into the VBA editor

Close the VBA editor (cross with red background top right)

Save as Macro Enabled workbook

Ensure that Macros are enabled. See help for how to do this. (The Option to "Disable all macros with notification" should be satisfactory.)

 

Now when ever a change is made in column F the formatting will be updated. Make a dummy change initially by just entering the same number again in any cell in column F.

 

I have used a dynamic method of finding the last row used in the column. See the comments in the code where you can use a fixed number of rows if you like and comment out the line with the dynamic method.

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim lngLastRow As Long
    Dim rngCel As Range
    Dim cel As Range
    Dim rngIndiv As Range
    Dim lngStartRow As Long
    Dim rngLastRow As Long
    Dim i As Long
   
    If Not Intersect(Target, Columns("F:F")) Is Nothing Then
        Columns("F:F").Font.ColorIndex = xlAutomatic    'Reset color of entire column
   
        'lngLastRow = 240      'Use for fixed number of rows. Ensure 240 includes a blank at bottom
        'Alternative dynamic method for last row
        lngLastRow = Cells(Rows.Count, "F").End(xlUp).Row + 1   '1 row added for last blank
       
        If Range("F1") <> "" Then lngStartRow = 1   'Test if first cell not blank
        For Each rngCel In Range("F2:F" & lngLastRow)
            If rngCel = "" Then
                If lngStartRow = 0 Then
                    lngStartRow = rngCel.Row
                Else
                    lngLastRow = rngCel.Row
                    Set rngIndiv = Range(Cells(lngStartRow, "F"), Cells(lngLastRow, "F"))
                    For Each cel In rngIndiv
                        For i = 1 To 4
                            If cel.Value = WorksheetFunction.Large(rngIndiv, i) Then
                                cel.Font.Color = vbRed
                                Exit For
                            End If
                        Next i
                    Next cel
                    lngStartRow = lngLastRow
                    lngLastRow = 0
                End If
            End If
        Next rngCel
    End If
End Sub

If the file is already in 97-2003 format, it will open it ‘compatibility mode’ when you open the file in Excel 2010. You can edit this file in Excel 2010, save it, and work on the same file in Excel 97. Working in compatibility mode will ensure that the format of the file (97-2003) is retained.
