Saving Drop Down Lists - Word 2013 - Microsoft Community

mirjschoenf say:

Saving Drop Down Lists - Word 2013

Hi,

I am creating email templates in Word and would like to be able to save and use multiple drop down lists. I would prefer not to remake a drop down list every time, if it's going to be used 20 times in the document.

Is there a way to do this?

Thanks

jay freedman say:

OK, this will be fairly simple.

Select one of the dropdown list content controls that you want to repeat, by clicking on the "handle" with the three dots so the handle and the down arrow area change from gray to blue:

  or    

Press Alt+F3 to open the dialog where you can define a building block. Enter a name for this list, and set the Gallery to "Custom Quick Parts" (scroll down the list of gallery types to find this). In the Save In box, select the template for which you're defining this list -- the default of Normal.dotm probably isn't appropriate.

Repeat the preceding steps for each of the other repeatable list boxes, choosing a different building block name for each one. If there are lots of list boxes and you have some reasonable scheme for classifying them, you can set up various categories (in the Category box, choose the Create New Category item) and group the building blocks that way.

To use these building blocks, go to File > Options > Quick Access Toolbar. Set the top left dropdown to All Commands, scroll down the left list and click on Custom Quick Parts, and click the Add button to send the button to the list on the right. You'll now have a button on the Quick Access Toolbar that opens a gallery of your list content controls. They'll probably all look alike, so you'll have to choose the one you want by its name alone.

You could also add a custom group to the Insert tab of the ribbon and put a button for the Custom Quick Parts gallery in that group. That would be less convenient than the Quick Access Toolbar button, though.
