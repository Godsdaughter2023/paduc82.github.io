andre da costa say:

Sounds like your recovery partition is damaged which is used to reinstall Windows 8. You can't use retail media to reinstall the preinstalled Windows 8, it will not be activated.

With Windows 8, Microsoft has changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Eliminating the product key sticker also removes any worry that the sticker might get damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

If the user has to reinstall the operating system on a machine that came with Windows 8, the installation process automatically grabs the software product key from the motherboard BIOS with no input from the user. This means that those familiar Windows product key stickers will no longer appear on the Windows 8 computers.

This means you will need to request recovery media from the manufacturer of your computer in order to reinstall Windows 8 properly:

Hi Toby,

Thank you for posting your query on Microsoft Community.

I would suggest you to try the following steps and check if it helps.

Method 1: Run a complete system scan.

Check if your system isn't infected with any virus by running a complete system scan.

Method 2: Run DISM Tool to correct the files which SFC couldn't restore.

  1.  Press Windows key + X and click onCommand Prompt (Admin). Kindly provide the administrative password, if prompt appears.
  2. Type Dism /Online /Cleanup-Image /ScanHealth and click Enter.
  3. Type Dism /Online /Cleanup-Image /CheckHealth and click Enter.
  4. Type Dism /Online /Cleanup-Image /RestoreHealth and click Enter.

For more information about DISM click on the link below: https://support.microsoft.com/en-us/kb/947821

Method 3: Try Clean boot

To disable all non-Microsoft services refer the article called Clean Boot.

Follow: https://support.microsoft.com/en-us/kb/929135

 

Note: After troubleshooting, refer to this section "How to reset the computer to start normally after clean boot troubleshooting"

(Methods for Windows 8.1 are also applicable for Windows 10 for the above provided links.)

 

Hope this helps. Please respond if you have further related queries.
