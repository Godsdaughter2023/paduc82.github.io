mike h.. say:

Hi,

Here's a macro to do it

Sub Remove_DupesInString()
Dim str As String
Dim fval As String
Dim strArr() As String
Dim x As Long
Dim k As Long
Set sht = Sheets("Sheet1")
Lastrow = sht.Cells(Cells.Rows.Count, "C").End(xlUp).Row
For Each cell In sht.Range("B2:C" & Lastrow)
    Erase strArr
    fval = ""
    str = cell.Value
    strArr = Split(str, "/")
For rw = 0 To UBound(strArr)
    For k = rw + 1 To UBound(strArr)
        If Trim(strArr(k)) = Trim(strArr(rw)) Then
            strArr(k) = ""  'if duplicate clear array value
        End If
    Next k
Next rw
     For x = 0 To UBound(strArr)
        If strArr(x) <> "" Then
            fval = fval & Trim(strArr(x)) & "/"
        End If
    Next x
fval = Left(fval, Len(fval) - 1)
cell.Value = fval
Next cell
End Sub

chip pearson say:

Here's a function that will return the distinct words (separated by the Sep parameter) in a string:

Function NoDupWords(InS As String, Sep As String) As String
Dim SS() As String
Dim N As Long
Dim M As Long
Dim B As Boolean
Dim ResultS As String
SS = Split(InS, Sep)
For N = LBound(SS) To UBound(SS)
    B = False
    For M = N + 1 To UBound(SS)
        If StrComp(SS(N), SS(M), vbTextCompare) = 0 Then
            B = True
        End If
    Next M
    If B = False Then
        ResultS = ResultS & " " & SS(N)
    End If
Next N
NoDupWords = Trim(ResultS)
End Function

You can then call this from a loop that contains all the cells you want to fix:

Sub AAA()
    Dim R As Range
    Application.EnableEvents = False
    For Each R In Range("A1:A10").SpecialCells( _
            xlCellTypeConstants, xlTextValues)
        R.Value = NoDupWords(R.Text, "/")
    Next R
    Application.EnableEvents = True
End Sub

Hi, CamillaFahmie,

Can I make calls from my skype telephone number to another country without paying any charges

Actually, no.  Here is why: A Skype Number only receives calls as it is a fixed line number that people can use to call you, and you answer those incoming calls on Skype.

Is the telephone for calls coming in or for calls going out? Only for calls coming in.

Can I change my number?

Yes, and no: The only way to change the number is to cancel the subscription for the number you no longer need or wish to use.  This cancellation stops the subscription from renewing.  You can still use the number until its subscription expires.  There is no facility to change a number on an existing subscription; you would need to start a new Skype Number subscription to obtain a new number.

Can I use only the chosen area code that is the same as the first one?

No; you may choose any number from the list of numbers available.
