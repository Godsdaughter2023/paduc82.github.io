    You are seeing it is only 117MB instead of 128MB as this is a differing of what manufacturers say a MB is, compared to what it actually is, NOT because of the way the file system works. This is the same for Hard Drives.

    A MegaByte (MB) is actually 1024 KiloBytes (KB) but manufacturers class them as only being 1000KB, therefore memory always looks different. This works for GigaBytes, in that they are actually 1024MB but manufacturers say they are 1000MB, therefore it appears to be smaller.

    - select the <Email Account> --> click on <Account Properties

    Option #3

    - open the Control Panel <Mail> app
    - select <Email Accounts>
    - select the desired <email account> --> select <Change>

    Now, each time Word starts, the "Print on Both Sides" option should appear on the Print page.
