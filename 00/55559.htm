shawn eary say:

If I you running Windows 7 Pro (or higher), you can install Win XP Compatibility mode.  http://www.microsoft.com/windows/virtual-pc/download.aspx

Once you get that working, you should be able to use Windows 7 to start the XP Virtual Machine.  The Win7 hosted XP Virtual Machine should then *awesomely* have the majority of your Win 7 drives mounted and you can use the XP Search to find the files that are on your Win 7 machine.

Alternatively, you can start a PowerShell session (should come with Win 7) and then change to the drive of your choice and so something like:

ls -rec -Filter *.pdf 

(you get the idea, to find all of your PDF files on the desired drive)

  1. "ls" is an alias for the PowerShell Get-ChildItem command
  2. -rec is a switch that tells PowerShell to perform a recursive search
  3. -Filter is a switch that allows you to user wildcard patterns, but I don't know if the filter has FULL PERL/grep regex compatibility
or you could

ls -rec -Filter bob*docx 

to search a drive recursively for a MS Word document that you titles Bob_Joes.docx

I have had better luck searching for files using either PowerShell or XP Compatibility Mode than simply using the Win 7 search box.  It appears that the Windows 7 Search box is his/miss on some of the rarely used files I have on my computer.

Best, 

 

Shawn
