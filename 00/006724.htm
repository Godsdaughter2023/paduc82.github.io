rob koch say:

I'll just add a note that the Microsoft scan cache feature that determines whether to rescan files that have been scanned previously is always reset when the operating system is shut down.

This is because there's no way for Windows Defender to know whether the file or its modification timestamp may have been manipulated while Windows wasn't operating, so the cache is automatically cleared when the system restarts.

Most files that had been scanned since the system last restarted should be skipped during additional scans, but a smaller number are always scanned regardless.

Superscripts in a Drop-Down List - Microsoft Community

j_vita say:

Superscripts in a Drop-Down List

Hi. How do I get superscripts in a drop down list content control in Word 2010? I want to have ordinal numbers as my values for a drop down list in a fill-in form. Thanks.
macropod say:

Basically, you can't. However, you can use a macro to reformat the displayed content when the content control is exited:

 

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
Application.ScreenUpdating = False
Dim i As Long, Rng As Range
With ContentControl
  If .Title = "Dropdown" Then
    If .Range.Text = .PlaceholderText Then Exit Sub
    .Type = wdContentControlRichText
    Set Rng = .Range
      If IsNumeric(.Range.Text) Then
          With Rng
            i = CLng(.Text)
            .Start = .End
            .InsertAfter Ordinal(i)
            .Font.Superscript = True
          End With
      End If
    .Type = wdContentControlDropdownList
  End If
End With
Application.ScreenUpdating = True
End Sub

 

Function Ordinal(Val As Long) As String
Dim strOrd As String
If (Val Mod 100) < 11 Or (Val Mod 100) > 13 Then strOrd = Choose(Val Mod 10, "st", "nd", "rd") & ""
Ordinal = IIf(strOrd = "", "th", strOrd)
End Function

 

The above macro assumes your content control's title is "Dropdown".

jay freedman say:
I don't think there's any way to do that. I've taken a look through the big Unicode fonts, Arial Unicode MS and Lucida Sans Unicode, and there are no superscript characters for st, nd, and th or for the individual letters that make them up. That would be the only way you could get an entry in the list that contains both superscript and normal-size characters.

In the control's Properties dialog you can assign a style to be used to display the control's results, and you could pick a superscript style such as Footnote Reference, but then the entire result would be superscript.

I thought it might be possible by running a macro that runs after you make a choice and exit the control, selects the ordinal, and makes it superscript. But, just like doing the same thing manually, changing just two characters in the controls makes the entire content of the control superscript.

I think you're going to have to stick with non-superscript ordinals like 1st, 2nd, 3rd, 4th.

Sub AddImage()
    Dim thePix As Shape
    Dim imageURL As String
    Dim wks As Worksheet
    Dim pasteBelowCell As Range
    Set wks = ThisWorkbook.Sheets(1)
    Set pasteBelowCell = wks.Range("D8")
    imageURL = "http://www.gap.com/resources/productImage/v1/770868002/P01"
    For Each thePix In wks.Shapes
        thePix.Delete
