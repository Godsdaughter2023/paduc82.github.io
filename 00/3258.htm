If you want to view which email addresses are linked to your Hotmail account, you can do this via Account Information page. Please follow these simple steps:

  1. Log in to your Hotmail account.
  2. Click Options in the top right corner of the screen.

mike sullivan [msft] say:

Hi Bob,

Thanks for the report.  I wanted to clarify one thing.  The prompt that appears in step #3 isn't a confirmation prompt.  It's asking you if you want to re-link the selected tables to tables with different names in the new data source.

If you select "No" here, you will successfully re-link at the data source level, without the prompts asking you to re-link each table with a new name.  Selecting "Yes" tells Access that you want to change the link information at the table level, i.e. to change the name of the source table.

I agree that the wording here isn't exactly clear, and I will see if we can change the message to make it clearer that this is about changing the names of the tables rather than re-linking the data source itself from one database to another.

Mike
