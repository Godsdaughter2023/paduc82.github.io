Hi Jason,

Windows 10 has a feature that allows end-users to reset the operating system in the event that they want to start over 100% back to factory defaults or, just reinstall the operating system yet keep all your files. The feature is called Reset this PC and it’s extremely handy if you’re experiencing stability or performance issue. Resetting your device won’t uninstall your current build. This is because of resetting, as a function, aims to reinstall the default state of your PC’s current version.

To assist you with your concern, we recommend that you try to reset your PC using the Windows environment option or the WINRE (Windows Recovery Environment) mode. To do this, follow the steps below:

  1. In the Search box, type Settings and press Enter.
  2. Click on Update & Security.
  3. In the left pane, select Recovery.
  4. Now in the right pane, under Reset this PC, click on Get Started.
  5. Follow the on-screen instructions carefully.

If the “Reset” function doesn’t work in the operating system. You can try the WINRE (Windows Recovery Environment) mode. To do this, follow the steps below:

  1. Click “Recovery” then select “Advanced startup”.
  2. Click “Restart now”, the operating system will reboot into WINRE (Windows Recovery Environment) mode.



  3. Select “Troubleshoot”.



  4. Select "Reset this PC".

    jedi prankster say:

    Your problem is that the drop-down menu isn't labeled right.  It is opposite of what makes sense.  There is a drop-down box, and below that it says 'map to', and below that is another drop-down box.  Select the paddle (or other button you want to use) in the top box, and then select the button whose function you want on that paddle in the bottom box.  

    For example:  For a shooter game like Halo where the 'a' button is jump by default, if you want to make the lower right paddle the jump button, select 'lower right paddle' in the top drop-down box, and then select 'a' in the bottom drop-down box.  It is not the most intuitively labeled menu.  

    It looks like you are taking a button and mapping its function to another button, but you are actually taking the button and mapping another button's function to it.  

    I hope that helps.

    lemp say:

    I can't give you a definitive answer because I can't find any specifications for the WSTER WS-2517BT speaker, but many newer Bluetooth devices use Bluetooth 4.x, also called Bluetooth LE (low energy) and Windows 7 does not natively support this version of Bluetooth.  This may (or may not) be your problem.

    Open Device Manager on your computer (click Start, type Device Manager in the Search box, and press Enter)

    Double-click the Bluetooth category to expand it

    Right-click your Bluetooth adapter (receiver, transceiver, etc.) and select Properties

    Select the "Advanced" tab

    Check the LMP version

    • LMP 0.x - Bluetooth 1.0b
    • LMP 1.x - Bluetooth 1.1
    • LMP 2.x - Bluetooth 1.2
    • LMP 3.x - Bluetooth 2.0 + EDR
    • LMP 4.x - Bluetooth 2.1 + EDR
    • LMP 5.x - Bluetooth 3.0 + HS
    • LMP 6.x - Bluetooth 4.0
    • LMP 7.x - Bluetooth 4.1
    • LMP ​8.x - ​Bluetooth 4.2
    • LMP 9.x - Bluetooth 5.0

    If you don't have LMP 6.x or better and the speaker uses Bluetooth 4, that's your problem.

    Can you pair the speaker with your smartphone?  Most smartphones do support Bluetooth 4.

    If this is, in fact, your issue, you can buy an inexpensive Bluetooth USB transceiver that will work with Windows 7.  For example: https://www.newegg.com/Product/ProductList.aspx?Submit=ENE&N=100158097%20600315449&IsNodeId=1&Description=bluetooth&bop=And&PageSize=96&order=RATING

    If you do get a Bluetooth transceiver, disable the onboard Bluetooth in your computer.
