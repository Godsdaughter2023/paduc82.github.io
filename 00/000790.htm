Like in the example below, let's say i designate the certain text to be the word "car", when i type the word "car" into A1, conditional formatting will highlight not just A1, but all the cells from A1 to A5000, regardless of whether those cells have any text in them or not.

ossiemac say:

In Conditional formatting select "Use a formula to determine which cells to format" and use the following formula.

=$A$1="Car"

Note the absolute referencing ($ signs).

Ensure that the "Applies to" field in the Conditional formatting dialog is =$A$1:$A$5000  If you select this range before selecting the Conditional formatting then it should default to that range but you can edit the range in the Conditional formatting dialog.

Sub macro_02()

Dim r As Long, c As Long, t As Long, i As Long
Application.ScreenUpdating = False
Set ws = Sheets("data") '<<<< change sheet name
c = ws.Cells(1, Columns.Count).End(xlToLeft).Column
Sheets.Add
t = 1
For i = 1 To c - 1 Step 2
r = ws.Cells(Rows.Count, i).End(xlUp).Row

The registry edits we made can stay in place, no need to do anything further. Most of the changes we made were inside the registry for your profile, but since you'll be using a new profile, its like starting with a clean slate.

To bring over your documents/data:
  • Click the Start Orb
  • Click Computer
  • Right click the partition you have you Vista installed on which is typically C:
  • Click Open (this will open a new window)
  • Go back to the Computer window
  • Double click on the same partition you opened 2 steps ago
  • Open Users in both windows
  • In one window open the old profile, in the other the new profile
  • Click Continue when alerted that You don't currently have permission to access this folder
  • Click Continue on the User Account Control
  • Open the folder you want to copy the information from
  • Select the file/files you want to copy
  • Right click any of them
  • Click Copy
  • Go to the new profile's matching folder
  • Click Paste
  • Click Continue when asked for permission
  • Click Continue on the User Account Control
  • Repeat the copy/paste process for all files you want to switch over
There is no rush to delete the old profile, the corruption won't spread, so give it a couple weeks and make sure you get everything over before you go back into User Accounts in the Control Panel to delete the old profile.
