john korchok say:

By default, Windows does not display file endings. So if you save a file, then look at it in a file window, the file ending is not shown. If it displays the stand Word icon, it has the correct file ending. BTW, turning on the display of file endings makes your system most secure, since a favorite trick of virus distributors is to send out files that appear to have one file ending when actually they have another.

You can make the file endings visible with these steps:

  1. Open a file window in Windows.
  2. On the file window menu, choose Tools>Folder options.
  3. Click on the View tab.
  4. Uncheck Hide extensions for known file types. OK out.

The spaces between words are likely the result of you using a font that the people on the other end don't have on their computer. Word remembers the metrics of the font on your computer (with width between letters and words), but uses a different font to display on the other computer. The result has poor spacing.

The solution is to use a font you know they'll have, like Arial or Calibri, or to embed the font in your document, if it will let you.

You want to convert the actual number to a hyperlink (not put the hyperlink into a separate cell), correct? If so, give the following macro a try. Select all the cells you want to convert to hyperlinks and then run this macro...

Sub MakeHyperlinks()
  Dim Cell As Range
  For Each Cell In Selection
    Cell.Hyperlinks.Add Cell, "http://sharepoint.auth.wellpoint.com/sites/SEGov/Lists/" & _
                              "Small%20Enhancement%20Request/DispForm.aspx?ID=" & Cell.Value & _
                              "&Source=http%3A%2F%2Fsharepoint%2Eauth%2Ewellpoint%2Ecom%" & _
                              "2Fsites%2FSEGov%2FLists%2FSmall%2520Enhancement%2520Request%" & _
                              "2FPersonalViews%2Easpx%3FPageView%3DPersonal%26ShowWebPart%" & _
                              "3D%7B748A5860%2D322A%2D4607%2DB385%2D036D33672DB0%7D"
  Next

There are several different versions of the fake Adobe Flash Player update that you might have encountered, depending on your web browser. The one that’s currently being detected on MSN pages is detected as Trojan:JS/Flafisi.D, which is specifically a detection for the FlashPlayer.hta file associated with the KovCoreG malvertising group. The fake update is delivered via a compromised advertising domain that's connected to the host site – so the detection will recur frequently upon returning to the host site, without indicting the presence of any persistent malware:

https://www.bleepingcomputer.com/news/security/malvertising-group-spreading-kovter-malware-via-fake-browser-updates/

For Microsoft Edge, the best defense is to install uBlock Origin, or another ad-blocker extension, directly from the Edge Settings menu:

Settings and more > Extensions > Get extensions from the Store

Information for installing uBlock Origin on other browsers, and the general documentation, can be found here:

https://github.com/gorhill/uBlock/wiki
