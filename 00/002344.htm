Every single application and service will have handles to various Kernel system resources such as semaphores, communication ports (TCP ports, Windows Sockets etc.) and files that are accessible to all threads in the process. ronrosenfeld say: Jeeped's solution will work so long as you have no <spaces> within your data values.

If that is not the case, you need another solution.

You could use a series of IF statements:

=MID(CONCATENATE(IF(A1="","","-"&A1),IF(B1="","","-"&B1),IF(C1="","","-"&C1),IF(D1="","","-"&D1), ... ),2,999)

For the ellipsis, substitute as many IF's as required.  Ensure the "999" is larger than the length of the longest possible string.

Or, more simply if VBA is available in MAC 2011, you could use a User Defined Function.

To enter this User Defined Function (UDF), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this User Defined Function (UDF), enter a formula like

=ConcatRange(a1:G1)

 in some cell.

=================================
Option Explicit
Function ConcatRange(rg As Range, Optional Separator As String = "-") As String
    Dim c As Range
    Dim v() As String
    Dim i As Long
ReDim v(0 To WorksheetFunction.CountA(rg) - 1)
i = 0
For Each c In rg
    If c <> "" Then
        v(i) = c.Text
        i = i + 1
    End If
Next c
ConcatRange = Join(v, Separator)
End Function
============================



jeeped say:
You could take advantage of the TRIM() function's ability to reduce multiple spaces to a single space, truncating prefixing and suffixing spaces at the same time.

             

The formula in A7 is
=CONCATENATE(A1,"-",B1,"-",C1,"-",D1,"-",E1,"-",F1,"-",G1)
... and E7 contains
=SUBSTITUTE(TRIM(SUBSTITUTE(A7,"-"," "))," ","-")

Put them together in a nested function as
=SUBSTITUTE(TRIM(SUBSTITUTE(CONCATENATE(A1,"-",B1,"-",C1,"-",D1,"-",E1,"-",F1,"-",G1),"-"," "))," ","-")
... or more concisely as,
=SUBSTITUTE(TRIM(CONCATENATE(A1," ",B1," ",C1," ",D1," ",E1," ",F1," ",G1))," ","-")

macropod say:

A selection_change macro would entail a fair bit of coding and risks unlinking when you don't want it to. Much simpler is to run a macro on demand. For example:

 

Sub SectionUnlink()
Dim HdFt As HeaderFooter
With Selection.Sections.First
  For Each HdFt In .Headers
    HdFt.LinkToPrevious = False
  Next
  For Each HdFt In .Footers
    HdFt.LinkToPrevious = False
  Next
End With
End Sub

 

For installation & usage instructions, see: http://www.gmayor.com/installing_macro.htm
