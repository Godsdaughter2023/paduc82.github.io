quietman7 say:

Randomly named alpha/numeric folders are commonly created and used temporarily when updating Windows components. They are also used by some software programs (i.e. Windows, Microsoft Office, Microsoft Visual Studio, etc) during update or installation to hold setup files (.inf, .cat, .gpd, .ppd and .dlls) and other information. These files and folders are usually automatically removed as part of the update process. However, its not uncommon for them not to be cleaned up and left behind after the update has been applied. When that occurs they usually can be manually deleted at any time.

The creates a temporary folder with randomly named alpha/numeric characters (i.e. 79f142e5e9e574d23954) on the root drive (C:\) of the computer which contains several files to include mrt.exe, mrtstub.exe and a file named $shtdwn$.req. Since external drives can be a hiding place for malicious files, MSRT will scan them too and you may find a left over randomly named alpha/numeric folder in that location as well. Usually after performing a scan and you click finish or cancel, the folder will automatically be removed right away or after the next restart of the computer. If not, Microsoft says the folder and its contents can be manually deleted without an adverse effect on the computer.

Microsoft Windows Malicious Software Removal Tool Summary (KB890830) provides a summary of the tool and frequently asked questions which includes one for a randomly named directory. MSRT can be accessed and manually run from the %windir%\system32\ folder by double-clicking on the mrt.exe file.

patc---01 say: This is usually because you have duplicate phone numbers in your Contact list.

Basically, if a number appears more than once, the phone doesn't know which name to display so it defaults to displaying the number. Which so far as I can see is the correct thing for it to do.

To test this, first ensure you have the latest version of Nokia Search installed on your phone. Start Nokia Search and select:

Options .. About

The version should be v3.2 or later. If it's an earlier version or Nokia Search is not installed, download and install the latest version (a search of this board will give you a link).

Once you have made sure you have the latest version of Nokia Search installed, take a note of a number that you know is giving this problem.

- Start Nokia Search

- Select "My Content" (this option is not available in earlier versions of Nokia Search)

- Search for a part of the number - part because it may appear in different formats in your contacts, eg: 0207 123 4567 and 02071234567

- goto Contacts and view the results

The number will appear in more than one contact

In theory, WM should have other significant advantages, such as in video playback, with support for more codecs (via third-party apps) than the iPhone has, but there too, it's mostly useless as HTC's hardware (Qualcomm MSM7200 chipset) is not powerful enough to play back what even the original iPhone from 2007 can play.  When connected to my LAN on the TP2, I can use Remote Desktop (you can do the same via VLC on jailbroken iPhone, but 800x480 res on the TP2 makes it much more usable).  Similarly, there's Pocket TI, an emulator for TI graphing calculators (for those who need advanced calculations), and other quirky apps specific to WM (mostly all ancient apps that happen to still work).  But all of these features don't mean much if I can't get my email working...

HTC's TouchFlo3D and other UI enhancements or stock app replacements (SMS, keyboard, contacts, calendar, etc.) make WM mostly usable, but Outlook Mobile looks like a major remaining weak point for business use.  ActiveSync/WMDC is another one-- I don't understand why iTunes does a better job of syncing with Outlook than WMDC does.  In two years with the iPhone, I never encountered any significant data sync issues (besides obscure contact fields in Outlook like "spouse" not being present on the iPhone), but in two weeks with the TP2, I already have a whole series of duplicated calendar events going back several years.

Due to the broken IMAP client, I'm forced to use webmail via Opera Mobile on the TP2, which is obviously a huge productivity killer vs. the iPhone, where I was able to quickly shoot off emails.  The iPhone email client also has lots of useful features like being able to load older messages with just a tap, mailbox-wide (including server-side!) search via the phone's Spotlight engine, and more.  What's keeping me on the TP2 right now is solely that it's (unfortunately) the only way I can get 3G on T-Mobile at the moment-- might still have to revert to the iPhone if I can't find a solution for the IMAP issue...
