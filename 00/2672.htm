amit tandon say:

Enter below code in the worksheet module ie. in the code module of the worksheet which has data.

Private Sub Worksheet_Change(ByVal Target As Range)

On Error GoTo ErrorHandler
Application.ScreenUpdating = False
Application.EnableEvents = False

Dim rng As Range, strNewValue As String
Set rng = Range("A1:A1000")

If Not Application.Intersect(Target, rng) Is Nothing Then
    strNewValue = Target.Value
    Application.Undo
    If Target.Value = "" Then
        Target.Offset(0, 4).Value = Format(Now, "mm/dd/yyyy hh:mm:ss")
    End If
    Target.Value = strNewValue
End If

ErrorHandler:
    Application.ScreenUpdating = True
    Application.EnableEvents = True

End Sub

The date is entered in column E only if cell in corresponding row column A was previously blank and now has data.
