Hi,

This issue can be caused by an incompatible or problematic display driver.

As an initial troubleshooting, boot your computer in safe mode and uninstall your graphics driver.

Since you're unable to boot your computer into normal mode, we recommend booting into safe mode through the Advanced Startup Options:

  1. Turn off your computer, then turn it back on.
  2. As soon as you see the Windows logo with the spinning dots, press and hold the power button (4-5 seconds) until your computer turns off. Do this process twice.    
  3. Turn on your computer for the third time, and let it load.    
  4. It should give you the message: Preparing Automatic Repair.    
  5. On the next screen, choose Advanced Options.    
  6. Select Troubleshoot.    
  7. Go to Advanced Options.    
  8. Click on Startup Settings.    
  9. Select Restart.
  10. Press the 4 or the F4 key on your keyboard to boot into Safe Mode, press 5 or F5 to boot into Safe Mode with Networking.

Once you were able to boot into safe mode, follow the steps below to reinstall your graphics driver:

  1. Press Windows key + R to open the Run dialog box.
  2. Type devmgmt.msc and hit Enter to open Device Manager.
  3. Expand Display adapters.
  4. Right-click your display adapter and select Uninstall.
  5. Choose Delete the driver software for this device.
  6. Click OK.
  7. Once done, open the Run dialog box again (by pressing Windows key + R).
  8. Type msconfig and press Enter.
  9. Click the Boot tab.
  10. Uncheck Safe boot.
  11. Select OK.
  12. Windows will automatically reinstall the driver upon restart.

don varnau say:

I've seen so recent few posts on this well-known problem with PDFs and 64-bit Internet Explorer that I assumed that it had been resolved by updates from either Microsoft or Adobe, but this registry edit was the fix a few months ago...

This registry edit has fixed this problem for many 64-bit IE10/IE11 users.

Work slowly and carefully and create the registry backup that I mention.

Go to Start> Run, type in regedit and hit Enter. 

In the left pane of the regedit window, select Computer, right-click it and choose Export. Save that REG file to the desktop. That's your backup. By double-clicking or right-clicking> Merging that REG file you can restore the registry and undo changes, if necessary.

You navigate the registry just like you would drill down through a folder structure to get to a specific file (key.) Just click the + to open the next level.

Start> Run> type in REGEDIT and hit Enter.

By clicking in the [+} drill down to

HKEY_CURRENT_USER

  Software

     Microsoft

       Internet Explorer

         Main

With Main selected in the left pane, right-click in the right pane and choose New> DWORD (32-bit)

Name the new DWORD Value TabProcGrowth then right-click it and set the value to 1.

If TabProcGrowth already exists (you didn't have to create it) just change the value from 0 to 1.
