Shifting Task Due Dates in Bulk - Microsoft Community

mbonge say:

Shifting Task Due Dates in Bulk

I am very reliant on my Outlook tasks.  Sometime I do not get all of my tasks completed on time (OK most of the time).  Is there a way that I can shift the due date for a task forward by one day, two days?  Ideally I could select several tasks and cause their due dates to shift foward by one day in a single step.  For instance, I have 8 tasks due on 5/12 that I know will not be completed.  How can I select these 8 tasks and move the due date to 5/13 without having to adjust each one independently?
graham mayor say:
You would have to do each one independently, but you can do so with a macro e.g. the following will prompt you to change all overdue tasks in the default tasks folder, to be due two days hence. You can adjust the date comparisons and/or remove the prompt if you wish

Sub Change_Task_Date()
Dim olNS As Outlook.NameSpace
Dim Fldr As MAPIFolder
Dim olTask As Outlook.TaskItem
Dim sChange As String

Set olNS = GetNamespace("MAPI")
Set Fldr = olNS.GetDefaultFolder(olFolderTasks)
For Each olTask In Fldr.Items
    If Format(olTask.DueDate, "YYYYMMDD") < Format(Date, "YYYYMMdd") _
       And olTask.Complete = False Then
        sChange = MsgBox("Change date of " & olTask.Subject & "?", vbYesNoCancel, "Task Overdue")
        Select Case sChange
            Case Is = vbYes
                olTask.DueDate = Format(Date + 2, "Short Date")
                olTask.Save
            Case Is = vbCancel
                Exit For
                GoTo Cleanup
            Case Else
        End Select
    End If
Next olTask
Cleanup:
Set olTask = Nothing
Set Fldr = Nothing
Set olNS = Nothing
End Sub
