Hi ijioma,


There could be a number of factors as to why you are experiencing sound problems in Windows 8.1. This could be caused by an outdated/incompatible device driver or because Windows Audio Service is not running.

Meanwhile, if you are using an external speaker, kindly make sure that external speaker is properly connected or plugged into the correct connection on the sound card.

If the issue persists, follow the steps below to troubleshoot sound issues in Windows 8.1:

Method 1: Run the Playing Audio and Hardware and Devices troubleshooter.

  1. Press Windows key + Q on your keyboard.
  2. Type Troubleshooting, and then hit Enter.
  3. On the left pane, select View all.
  4. Select and run the Playing Audio and Hardware and Devices troubleshooter.

Method 2: Roll back to the previous Audio driver.

  1. Go to Device Manager by pressing Windows key + Q, type devmgmt.msc, and then hit Enter.
  2. Expand Audio inputs and outputs.
  3. Right-click the audio adapter driver, and then click Properties.
  4. Choose Driver tab.
  5. Click Roll back driver.
  6. Restart your computer.

Note: The Roll Back Driver button is only available if a previous version of the driver was installed. If the current driver for the device is the only one that was ever installed on this computer, then the Roll Back Driver button is not enabled.

Method 3: Enable Windows Audio Service.

  1. Press Windows key + Q on your keyboard.
  2. Type services.msc, and then hit Enter.
  3. Double-click Windows Audio on the list of Services (Local).
  4. Go to the General tab. Under Startup type, select Automatic from the drop-down list.
  5. Click Start.
  6. Click Apply and then hit OK.

A couple of registry keys to check that might be causing this:

Open Regedit (Press Windows Key, type regedit and hit Enter)

Check these 2 registry keys for correct values:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\

Find the Userinit key in the right hand pane and replace the value with the following:
C:\Windows\system32\userinit.exe,

In the same place that you have Userinit above, there is another key called Shell

Make sure that is set to only explorer.exe with no other switches . . .
