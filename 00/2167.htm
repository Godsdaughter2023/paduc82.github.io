daniel g. grau say:
Before deleting the row, make sure the table has "Fixed column width" (despite the singular form, it affects all columns).

daniel g. grau say:

You need to apply "Fixed Column Width" to every table. Place the cursor anywhere in the table and select the option. No indication is shown. However, if you check the table's properties, you'll see each column now has a "Preferred width" and a numerical value.

If you have lots of tables, just run this macro to apply the "Fixed Column Width" setting to all tables in the document:

Sub FixedColumnWidth()
For Each CurrentTable In ActiveDocument.Tables
    CurrentTable.Select
    Selection.Tables(1).AutoFitBehavior (wdAutoFitFixed)
Next CurrentTable
End Sub

Hi,

Right to Transfer the License
If you have used up your number of allowed concurrent installations, the "retail" license ALSO gives you the right to "transfer" the license from one computer to another. For pretty much all other license types, the license is permanently tied to the hardware. If the computer dies, so does the license. This is a real issue for 2010 "Product Key Card" licenses. Many people were caught by surprise when they had to buy a new copy of Office when their new computer died and the hardware was replaced under warranty. The PKC license died with the hardware.
The Office 2013 Retail license was initially "improved" to remove the transfer right. Due to consumer pressure this change was retroactively revoked to allow the installation to be transferred to new computers. As well, the 2013 PKC license was amended to allow transfers when the Motherboard dies and is replaced under warranty. See the links at the bottom of this entry for the terms specific to these 2 Office 2013 license term modifications.

el_loco---01 say: Don't use PC suite for music and videos. Use either WMP11 for music or just drag and drop using File Transfer (mass storage).

Anyway the fact that you still had the contents of your contacts and pictures in your phone means that you didn't format the Phone memory and you didn't format the Mass memory.

To format the phone memory:

Type: *#7370# then enter 12345 and select Yes. The phone should reboot and ask for your Country and City.

To format the Mass memory:

You have 2 options: 1) Go to Tools-> Memory and then select Options->Format. 2) Connect to a PC with USB cable and select File Tranfer (mass storage) and then on the PC open the File Explorer and right click on the external drive that represents your phone (e:\ or g:\ or whatever it got assigned) and then select Format.

If problem persists take it to a Care center.
