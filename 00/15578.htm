kim.me say:

I finally found the answer after many important emails were deleted.

Turns out there multiple settings for this, but changing the ones we are familiar with from previous versions still allows emails to be deleted.

There is a new "global" setting that defaults to permanently delete after 14 days.

TO FIX THIS! 

Select the File menu / Options.  Select "Advanced"  Half way down is an "AutoArchive" button.  You will see that it has the default settings of "14 days".  This is what you need to disable.  You will also note that the defaults here will delete email after 14 days until you disable the option.

I have found no way to recover the emails I lost before I found this additional archive settings.

I hope this helps others having this same problem.

=INDIRECT(F2)
