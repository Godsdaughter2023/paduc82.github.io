To change your display name on the Skype website,

Note: To distinguish the Skype name and the display name, your Skype name will be after the word Accounts while your display name will be in a larger font size.

The KERNEL_SECURITY_CHECK_FAILURE error indicates the kernel has detected the corruption of a critical data structure o your PC.

 

You might be facing the above issue due to the compatibility of the hardware or due to the non-updated USB ports on your PC.

 

Method 1: I suggest you to update the USB port drivers from Device manager manually by following the steps below and check if the issue is resolved.

 

  1. Press Windows logo + X keys on the keyboard and select Device manager option.
  2. Expand Universal Serial Bus Controller option and check if the device is detected without showing any yellow exclamation on the device driver icon.
  3. If there is a yellow exclamation shown on the device driver, right click and select Update. Update the drivers without exclamation anyways.
  4. Follow the on-screen instructions.
  5. Close the window after the update is complete and check if the issue is resolved.

 

Method 2: If the issue persists, I suggest you to run the USB driver in compatibility mode by following the steps below and check if the issue is resolved.

 

Step 1: Compatibility mode runs the program in the current version of Windows using the settings from previous version of Windows.

 

  1. Download the latest driver setup file from the manufacturer’s website or locate the downloaded driver setup file of the previous operating system in your PC.
  2. Right-click on the driver setup file and click on Properties.
  3. Click on the Compatibility tab, check the box Run this program in compatibility mode for, select Windows 10 operating system from the drop down menu and proceed with the installation.
  4. Once the driver is installed in compatibility mode, restart the PC for the effective changes to take place.
  5. Check if the issue is resolved.

 

Step 2: You can also follow the steps below and check if the issue is resolved.

 

  1. Press Windows logo key on the keyboard and type Run programs made for previous versions of Windows.

  3. Check if the issue is resolved.
