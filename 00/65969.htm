rubencalvo say:
I had the same problem recently.

A temporary solution is to change the screen resolution, then reset it to the original resolution.

The permanent solution is to change a System Performance Setting: 
Control Panel -> System and Security -> System -> Advanced System Settings -> Performance Settings... -> UNCHECK the option to "Fade out menu items after clicking."

I found this solution at the following webpage:
http://www.thedr1ver.com/2011/06/menu-selection-stays-on-screen-after.html
