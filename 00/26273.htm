Hard to say what will be supported in the future, but Microsoft is definitely moving further away from the .mdb format. As it stands right now, if you have a file created in Access 97 (still the mdb format), you can't convert it to the new .accdb format. You can't really blame MSFT for not supporting a format that is 10+ years old, especially when they've been quite vocal in their calls to upgrade those to newer formats.

If you sell this application then I'd certainly look at upgrading, if for no other reason that a "modernization" of the app. That's your call, of course, but it would certainly hedge your bets in regards to your question.

paulcordts say: The weirdness comes from using the tab key, looks good when typing, but does not post right.

I think I understand what you are trying to do.

One additional question, is there a formula that determine what group a given number is in.  If so this can be used to make what you are going a lot easier.

If not then you have this table of cell addresses with a corresponding group, which you can still work with, but it is a little more tricky to work with A1 style references.

this formula in A1 would give you A1 as a result
=ADDRESS(Row(A1),Column(A1),4)

a simple lookup would give you the group  
=VLOOKUP("A1",$E$1:$F$8,2,FALSE)
This would return D

combine the 2 to get
=VLOOKUP(ADDRESS(Row(A1),Column(A1),4),$E$1:$F$8,2,FALSE)
and this is the formula we need for the conditional formatting

=VLOOKUP(ADDRESS(Row(A1),Column(A1),4),$E$1:$F$8,2,FALSE)="D"

You need to write 2 conditions one for yellow and one for green

Select A1 and then select Conditional Formatting:New Rule
Use the rule type Use a formula to determine which cells to format. and enter this for the formula
=VLOOKUP(ADDRESS(Row(A1),Column(A1),4),$E$1:$F$8,2,FALSE)="D"
Then select a yellow shading for the format and click OK

Do this again for the next condition
Select A1 and then select Conditional Formatting:New Rule
Use the rule type Use a formula to determine which cells to format. and enter this for the formula
=VLOOKUP(ADDRESS(Row(A1),Column(A1),4),$E$1:$F$8,2,FALSE)="F"
Then select a green shading for the format and click OK

The use the format painter or copy:paste special formats to copy the format from A1 to the rest of your data.

stefan blom say:

Note that after you have enabled "Different odd and even," each section contains two types of headers (and two types of footers): even and odd ones. If you also enable "Different first page" for a section, it has a third header (and a third footer): a first page one.

   

To clarify, each section can have up to three different headers (and three different footers), which is the case if you have selected both  "Different odd and even" and "Different first page" (in the Page Setup dialog box or on the Header & Footer Tools Design tab on the ribbon).

When unlinking headers (or footers), each type of header (footer) must be separately unlinked from its "neighbor" in the preceding section.
