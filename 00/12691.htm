Vlookup from text or Access file - Microsoft Community

x mike say:

Vlookup from text or Access file

I am using Excel 2003.

Is there a way to perform a Vlookup or other type of lookup in Excel that will lookup up a value in either a TXT or microsoft access file?

 

hansv mvp say:

Here is a function you could use, but beware - it may cause unacceptable overhead:


Function DatabaseLookup(MyValue, MyDatabase As String, MyTable As String, _
    LookInField As String, LookInType As String, ReturnField As String)
  Dim dbs As DAO.Database
  Dim rst As DAO.Recordset
  Dim strWhere As String
  On Error GoTo ErrHandler
  DatabaseLookup = CVErr(xlErrNA)
  Set dbs = DBEngine.OpenDatabase(MyDatabase)
  Set rst = dbs.OpenRecordset(MyTable, dbOpenDynaset)
  Select Case LookInType
    Case "String"
      strWhere = Chr(34) & MyValue & Chr(34)
    Case "Date"
      strWhere = "#" & MyValue & "#"
    Case Else
      strWhere = MyValue
  End Select
  strWhere = "[" & LookInField & "]=" & strWhere
  rst.FindFirst strWhere
  If Not rst.NoMatch Then
    DatabaseLookup = rst.Fields(ReturnField)
  End If

ExitHandler:
  On Error Resume Next
  rst.Close
  Set rst = Nothing
  dbs.Close
  Set dbs = Nothing

ErrHandler:
  Resume ExitHandler
End Function

Use like this:

=DatabaseLookup(1000, "C:\Access\MyDatabase.mdb", "tblPersons", "PersonID", "Number", "LastName")

to return the last name of someone with PersonID = 1000 in the table tblPersons in C:\Access\MyDatabase.mdb.

doug robbins - mvp office apps & services (word) say:
I have an Excel file that gets populated with data & then saved as another name.  I want to create a Word document that has fields(?) that would be variables, that get populated from one of the Excel documents.  I am very good w/ Excel & know next to nothing about Word.  All of the Excel files are formatted the same way, so the particular piece of data will alway be in cell F4 (for example), but which Excel file will be a variable.  How do I even begin to go about this?  Are there tutorials?


The following code will allow you to navigate to and select the desired workbook and then populate a variable ("xlData") in the activedocument with the data contained in cell F4 of the first worksheet in the selected workbook:

Dim strFileName As String
Dim fd As FileDialog
Dim xlApp As Object
Dim xlbook As Object
Dim xlsheet As Object
Set fd = Application.FileDialog(msoFileDialogFilePicker)
With fd
    .Title = "Select the file that you wish to use."
    .Filters.Add "Excel Workbooks", "*.xls*"
    .AllowMultiSelect = False
    If .Show = -1 Then
        strFileName = .SelectedItems(1)
    Else
        MsgBox "You cancelled the operation"
        Exit Sub
    End If
End With
Set xlApp = CreateObject("Excel.Application")
With xlApp
    Set xlbook = .Workbooks.Open(strFileName)
    Set xlsheet = xlbook.Worksheets(1)
End With
With ActiveDocument
   .Variables("xlData").Value = xlsheet.Range("F4").Value
   .Range.Fields.Update
End With
xlbook.Close SaveChanges:=False
Set xlbook = Nothing
Set xlsheet = Nothing
xlApp.Quit
Set xlApp = Nothing
Set fd = Nothing

You can get the data displayed in the document by inserting a { DOCVARIABLE xlData } field in the document where you want the data displayed.

Force Sync
