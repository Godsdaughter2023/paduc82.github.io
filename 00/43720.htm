Hi Charles,

Apologies for the inconvenience and I will be happy to answer your query. I appreciate the effort made by you to fix the issue.

SearchIndexer.exe is the Windows service that handles indexing of your files for Windows Search, which fuels the file search engine built into Windows that powers everything from the Start Menu search box to Windows Explorer, and even the Libraries feature.

You can try the following methods and check the status.

Method 1: Place the computer in clean boot state

To help troubleshoot error messages and other issues, you can start Windows by using a minimal set of drivers and startup programs. This kind of startup is known as a "clean boot." A clean boot helps eliminate software conflicts.

Placing your system in Clean Boot state helps in identifying if any third party applications or startup items are causing the issue. You need to follow the steps from the article mentioned below to perform a Clean Boot.

How to perform a clean boot in Windows        

Note: Refer "How to reset the computer to start normally after clean boot troubleshooting" to reset the computer to start as normal after troubleshooting.

Method 2: Make SearchIndexer use less RAM or CPU

The best option is to cut down on the amount of data that you’re indexing—there’s usually no reason to index every single file on your drive. You’ll need to open up the Indexing Options through the Control Panel or Start Menu search box to make the changes.

The first thing you should notice is the Pause button on this window, which can pause indexing for up to 15 minutes—useful if you’re trying to do something and Windows Search happens to be in overdrive mode, though it really shouldn’t since it only runs while your PC is idle.

You’ll want to click the Modify button and then trim down the list of locations to just the ones you really need indexed—this can improve the performance of Start Menu search box as well. 

Method 3: Run System File Checker tool

System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files.

Press Windows key + X and click Command Prompt (Admin)

Type the following command and press Enter

sfc /scannow

Note: Do not close this Command Prompt window until the verification is 100% complete. The scan results will be shown after this process is finished.

 Also, refer to: Windows 10 - Accelerate File Operations with the Search Indexer

They never de-activate anything from an old computer, you have to erase it yourself.  You only call them to tell them to reset the activation counter back down to 0 (zero) so that you can install it on another computer.  If you ever have it on more than one computer and their activation count goes up past 1, your computer will stop working until you do it again.  That's why it's your job to erase it from the first computer.

How to increase font size - Microsoft Community

josephpaulsen say:

How to increase font size

How to increase Font size , both on sending and receiving ?

[Original subject: windows Live Mail-2012]

¡firedog say:

You don't have much control over the font used in messages you receive. Here are some suggestions for mitigation:
 

  • You can specify the font size for reading plain text messages:
    When viewing your message list, press Ctrl-Shift-O for Options.
    On the     Read tab, click Fonts. Specify fonts and font sizes for each encoding you're likely to come across. It will in most cases be enough to select options for Unicode, User defined and Western European.
  • If you find that this is easier to read, you could on the same Read tab select Read all messages in plain text. Then, to see a message in HTML, press Alt-Shift-H.
  • The Text size setting in Internet Explorer will affect the display of text in Windows Live Mail. You'll find the option on the View menu. Any change may not take effect until the next time you start Windows Live Mail. 
  • If you find that text is in general too small on your screen, it may be better to adjust the display parameters to make text and other items bigger on the screen. You will then be able to adjust downwards in most programs where text size isn't a problem at the moment.
  • In HTML messages, the font size will normally be specified in the HTML by the sender. You may find that you can change the size for reading by using Ctrl-Scroll on your mouse, or (with an English keyboard layout) Ctrl + (Control-Plus) and Ctrl - (Control-Minus). Ctrl-0 (Control-Zero) will revert to the default 100% size.

As regards messages you compose: when viewing the message list, select File > Options > Mail. On the Compose tab, click Font Settings and select the size you'd like.
