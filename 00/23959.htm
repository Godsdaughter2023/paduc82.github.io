rob koch say:

The following Excerpt is from the Microsoft Security Essentials Privacy Statement under the section titled Microsoft Active Protection Service (MAPS).

 

"If you join MAPS with a basic or an advanced membership, Microsoft might request a sample submission report. This report contains specific files from your PC that Microsoft suspects might be potentially unwanted software. The report is used for further analysis. You will be asked each time if you want to send this sample submission report to Microsoft."

 

Since by default MAPS is selected for Basic membership in Settings, then yes, MSE will prompt for such information if something new and unknown is detected on your PC, so you might be prompted to optionally upload the item(s) to Microsoft via MAPS.  Whether you do this is completely up to you and should be determined based on whether the files are of a truly personal nature or simply program or related files.

Uploading these not only helps Microsoft potentially detect new malware faster, but also may help protect you better immediately, since if the item is known by MAPS a new definition that has yet to be released widely via a definitions update can be downloaded and used by MSE to remove the item.  Think of this as an extension of the local definitions on your own PC to a much larger and more current definitions database at Microsoft.

 

As to how to tell whether such notifications (messages) are real, this can be difficult.  However, most fake notifications are simply browser pop-up windows, so confirming that these are actually local dialog boxes with active components that you can highlight and review, rather than simply 'pictures' of something is one way to help tell the difference.
