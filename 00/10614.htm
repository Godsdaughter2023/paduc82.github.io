Replace upper-case acronyms with small caps in MS Word - Microsoft Community

italosalves say:

Replace upper-case acronyms with small caps in MS Word

I have a 200-page MS Word file with several acronyms spread all over the text, and they are all upper-case. I want to replace every acronym with four or more characters with a small caps version of it.

Example:

  • NATO should become ɴᴀᴛo,

but

  • NHS should remain NHS.

Is there any way to do that (semi-)automatically in MS Word 2010?

doug robbins - mvp office apps & services (word) say:

Run a macro containing the following code

Selection.HomeKey wdStory
Selection.Find.ClearFormatting
With Selection.Find
    Do While .Execute(FindText:="[A-Z]{4,}", MatchWildcards:=True, Forward:=True, Wrap:=wdFindStop) = True
        With Selection
            .Text = LCase(.Text)
            .Font.SmallCaps = True
            .Collapse wdCollapseEnd
        End With
    Loop
End With

You cannot do it with the Find and Replace facility because, in addition to applying the Small caps attribute, it is also necessary to change the text from Uppercase to Lowercase.

jay freedman say:

This macro will find each sequence of four or more consecutive capital letters, change its case to lower case, and apply small caps formatting to it.

Sub SmallCapAcronyms()
    Dim rg As Range
    Set rg = ActiveDocument.Range
    With rg.Find
        .MatchWildcards = True
        .Text = "[A-Z]{4,}"
        While .Execute
            rg.Case = wdLowerCase
            rg.Font.SmallCaps = True
            rg.Collapse wdCollapseEnd
        Wend
    End With
End Sub

Office 2019 installation process. - Microsoft Community

vivek rawal say:

Office 2019 installation process.

Follow below steps to install Office 2019.

1) Download the Office Deployment Tool (ODT)

2) Create a folder on desktop and extract the ODT file

3) Then Open Configuration-Office2019Enterprise.xml

4) Replaced with below text and save it

<Configuration>

<Add OfficeClientEdition="64" Channel="PerpetualVL2019">

<Product ID="ProPlus2019Volume">

<Language ID="en-us" />

</Product>

</Add>

<!--  <RemoveMSI All="True" /> -->

<!--  <Display Level="None" AcceptEULA="TRUE" />  -->

<!--  <Property Name="AUTOACTIVATE" Value="1" />  -->

</Configuration>

Note:-  If you want to download Standard 2019 Office then changes the Product id on above line to "Standard2019Volume"

Refer this Product ID article :-https://support.microsoft.com/en-in/help/2842297/product-ids-that-are-supported-by-the-office-deployment-tool-for-click

5) Open CMD run as Administrator

6) Navigate to ODT folder which we have saved on desktop

7) Then run this command :- Setup.exe /download configuration-Office2019Enterprise.xml

8) New folder will be created called "Office". Monitor the size of the folder it near 3.27 GB

9) Once the download completed the run this command to start the installation :- setup.exe /configure configuration-Office2019Enterprise.xml

If you have any question please feel free to share on this post.

Regards,

Vivek Rawal
