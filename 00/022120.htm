What does all of this mean? 

You can force that behavior with Office 2016 itself.

Make a new file, write something into cell A1 and save the file as "Excel 97-2003 Workbook (*.xls)".

Close Excel and reopen the file.

If you e.g. write something in A2 and save the file = > no issue.

    Deleting this folder will not affect your Windows 10, but it will remove your ability to rollback to Windows 7 if the upgrade has problems.  Windows 10 should remove it automatically after 30 days.  But if you want to remove it manually, the best way is to use Disk Cleanup:

  1. Click Start and type Disk Cleanup

  3. Select your C: drive and click [OK].
