Hi Noel,

Yes, it’s possible by using PowerShell but we don’t have the -replace cmdlet.

Here is an example which you can use:
Get-Mailbox -Filter {RecipientTypeDetails -eq "UserMailbox"} |ForEach-Object {Set-Mailbox -Identity $_.identity -CustomAttribute5 $_.displayname}

The filter ensures that the operation will only affect user mailboxes since system mailboxes or shared mailboxes normally don’t need signature. You can adjust the command based on your needs.
