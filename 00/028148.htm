andreas killer say:
I would use a Label instead of a TextBox, but that doesn't matter. For the example below, create a Userform with a Label and paste the code below into a normal module.

To center the text use the TextAlign property of the Label.

Andreas.

Option Explicit

Private Declare Sub Sleep Lib "kernel32" (ByVal dwMilliseconds As Long)

Sub Test()
  Dim Delay As Long
  'Load the form
  Load UserForm1
  With UserForm1
    'Store the values into the controls
    .Label1 = "Hello World"
    'Show it and return immediately
    .Show vbModeless
    'Wait 0.5 seconds
    Delay = 500
    'Install an errorhandler, maybe the user closes the form
    On Error GoTo ExitPoint
    Do While .Visible And Delay > 0
      'Wait a short time
      Sleep 50
      'Count down
      Delay = Delay - 50
      'Give the user a chance to act
      DoEvents
    Loop
  End With
  'Close the form
  Unload UserForm1
ExitPoint:
End Sub
