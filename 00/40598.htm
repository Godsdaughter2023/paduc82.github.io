Printer Spooler message

Hi, I have a Lexmark X5495 All In One Printer. I have had it for 3 years & hooked to this same computer. I printed a shipping label from my computer 5 days ago. I checked the cables & plugged it in to another AC outlet. When I restart the computer, I get a message that a new device is detected. Why is my printer not working now with windows XP ?

lemp say:
Yes:

del /Q "%windir%\system32\spool\PRINTERS\*.*"  {Enter}

sc config spooler depend= RPCSS   {Enter}

exit    {Enter}


The first command deletes all the files in C:\Windows\system32\spool\PRINTERS, which is where the Print Spooler stores the print jobs as they are being sent to the printer.  I used %windir% just in case Windows is installed in a different directory, such as C:\WINNT.  If there are any corrupt print jobs (which can cause problems with the spooler) this will get rid of them.  The print spooler has to be stopped in order to delete files in this directory.

The second command fixes the dependency of the Print Spooler service so that it ONLY depends on the RPC service.  Some Lexmark printers install a service named LexBCE (or something similar).  If that service gets removed or corrupted, the Print Spooler can't run ... because it is waiting for the Lexmark service to start.  The RPC service is so central to Windows that if it isn't running, you'd have a lot more problems than just the Print Spooler not working.

The third command closes the Command Prompt window.  Strictly, this isn't necessary.  You could just click the X in the upper right corner of the window.

Good luck.
