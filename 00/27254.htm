Unfortunately in the world of the computer, facts will always win over personal preference.

If you'd actually read those Microsoft articles regarding the lack of security within the Windows Sidebar, they indicate that the problem is far deeper than the gadgets themselves, which means that the issues can't be fixed without the complete removal of this archaic system.

The following paragraphs from the Windows Development Center provide a partial description of how the Windows Sidebar system worked.

"A gadget is an HTML and script-based application designed to present the user with a limited set of information or functionality obtained from other applications, controls, or websites and services. Gadgets, although hosted by the Sidebar, are not confined to the Sidebar area; the user can undock and move them onto the desktop as desired."

"A gadget is distributed as a .gadget file—a renamed .zip archive—consisting of a collection of XML, HTML, Microsoft JScript, and Cascading Style Sheets (CSS) files."

Why such a horribly insecure design was ever allowed by Microsoft at a time when malicious actors were already ravaging the inherently insecure Windows XP operating system is a good question, but not truly important any longer since Microsoft has already admitted the mistake and long ago removed the Windows Sidebar from the systems of those who allowed it.

joeu2004 say: Janyjan wrote:

If you want help, it would behoove you to provide some examples.

If you want to compute percentage by A1/B1, but it results in an error when B1 is zero, you might try the following:

=IF(B1=0,SIGN(A1),A1/B1)

formatted as Percentage.  Alternatively, if you are using Excel 2007 or later:

=IFERROR(A1/B1,SIGN(A1))

Those formulas display -100%, 0% and 100% for A1<0, A1=0 and A1>0 respectively when B1=0.

The choice of 100% when B1=0 is arbitrary.  There really is no right answer.  You might prefer no result at all, for example:

=IFERROR(A1/B1,"")

How to convert several hundred footnotes in a word document into a - Microsoft Community

jørnriise say:

How to convert several hundred footnotes in a word document into a bibliography/reference list

Hi,

I received a reply to my previous question concerning this issue, but it did not seem to relate to my problem. The answer dealt with citations in the text, but I have several hundred footnotes which I want to convert into a bibliography/reference list all together; not one by one. I do not want to change the footnotes/references; just to make a comprehensive list of all of them at the end of the document.

Regards,

Jørn Riise

john mcghie say:

Hi Jørn:

Technically, this cannot be done.  A footnote is not a citation (it has no data fields in it, it's just text).  Word's citation manager can't work with footnotes.  So in theory you can't do this.

However, what you could do is convert them all into endnotes.  Then convert the endnotes into normal text.  And then convert the text into your reference list by manually inserting the commas etc.

  1. On the menu bar, choose Insert>Footnote (can't do this from the Ribbon)
  2. Click Convert
  3. Choose footnotes to endnotes

When you do this, the numbering format will change to lower-case Roman.  If that is important to you, use Insert>Footnote again and choose the number format you want. Click Apply and they will all change.

Now you need to convert the whole of the endnotes text into plain typed text, so it won't change while you are working on it.

  1. To do this, select all of the endnotes and Copy.
  2. Reposition your insertion point above the endnotes
  3. Then from the Edit menu choose Paste Special and click Unformatted Text.

That converts the numbers into ordinary text.  You can now delete the endnotes.

You then have ordinary text, you can edit this into your list of citations manually.

To obtain a proper Bibliography, you must first have inserted your citations as citations, not footnotes.  To insert a citation, you must first define it as a Source in your Citation Manager, then insert it into the document.

For details on how to do this, read the Word Help topic "Add or change sources, citations, and bibliographies".

Hope this helps
