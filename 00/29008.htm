Can I delete Windows Live Mail now that I'm using Windows 10 - Microsoft Community

mkj307 say:

Can I delete Windows Live Mail now that I'm using Windows 10 mail?

I am now using Windows 10 mail & need room on my hard drive. Can I delete Windows Live Essentials or any component of it like the email or do I need the files for other programs? I have deleted other programs I was told I didn't need on the HP forum & even after a restore am still having problems. Can anyone tell me for sure if I can do this?
¡firedog say:

Uninstalling Windows Essentials will gain you probably something less than 100MB of space. The big space hogs are the store folders, which contain all the mail folders and messages, account settings and calendars. On my HP machine, a recent fresh install of Windows 7 with five mail accounts in Windows Live Mail, the store folder for my Windows user account is almost 2GB. So, if you can do without all saved messages, you can uninstall Windows Essentials and delete the store folders, by default at
 

C:\Users\{user}\AppData\Local\Microsoft\Windows Live Mail

 
There will be a store folder for each Windows {user} account on the machine that has used Windows Live Mail. 

Contacts can also take up a lot of space, so you should also delete the folder where these are stored: C:\Users\{user}\AppData\Local\Microsoft\Windows Live. Again, there will be one for each Windows user who has used Windows Essentials. 

If you do need to keep some of the messages, you can export them before uninstalling the program. You'll have to find some other program to be able to read them, though.

PS: removing Windows Essentials and its user data will have no effect on any other program.

ron sommer mvp 2009-2017 say:

When you upgrade to Windows 10, your old files are saved in windows.old .

If you have a windows.old file, one way to get more hard drive space is to put windows.old on a CD or other storage and then delete it from your hard drive.

Windows Live Mail has no concept of 'spam' and doesn't tag messages.
