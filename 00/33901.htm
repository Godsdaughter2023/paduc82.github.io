Defaultuser0 is temporarily created and used while system is installed. It does not belong to any users and disabled in default. There’s no password for defaultuser0. We’d suggest you to follow steps mentioned below to remove Defaultuser0.

  1. Once you are on the sign screen with defaultuser0, press and hold Shift key on the keyboard and click the Power button on the right bottom of the login screen. Then click Restart.

  2. Then select Troubleshoot > Advanced options > Startup Settings.

  3. Click on Restart on the screen of Startup settings.

  4. After restarting the computer, press F6 or 6 to enable safe mode with command prompt. If not F6 or 6, select the one which enables safe mode with command prompt.

  5. In the Command Prompt Windows type the following command, net user administrator /active:yes and then press Enter.

  6. Now restart the computer and log in to Administrator account.

  7. Press Windows key + R on the keyboard and type netplwiz, then press Enter.

  8. Select Defaultuser0 (user account) under User Name and click on Remove.

  9. Now

  10. Sign in with new administrator account which you just created.

  11. Search for Command Prompt in search bar.

  12. Right click on Command Prompt and select Run as administrator.

  13. Type the following command to disable inbuilt administrator account, net user administrator /active:no and press Enter.

Hope it helps.

    waseemulla shariff say:
    Hi,
     
    This usually happens if an operation on a socket could not be performed because the system lacked sufficient buffer space or because a queue was full.
    Try the following steps and check the result.
    Step 1: Renew IP (Internet Protocol) address.
    1. Click Start, type cmd, right click cmd and click run as administrator.
    2. Type the following command and hit enter: ipconfig /renew 3. Check if the issue persists.
     
    Step 2: Reset Winsock
    1. Click Start, type cmd, right click cmd and click run as administrator.
    2. Type the following command and hit enter: netsh winsock reset
    3. Check if the issue persists.
     
    Step 3: Reset TCP/IP
    1. Click Start, type cmd, right click cmd and click run as administrator.
    2. Type the following command and hit enter: netsh int ip reset c:\resetlog.txt
    Note: If you do not want to specify a directory path for the log file, use the following command: netsh int ip reset resetlog.txt
    3. Reboot the computer.
    4. Check if the issue persists.
     
    For more information on "How to reset Internet Protocol (TCP/IP)" check this article: http://support.microsoft.com/kb/299357
     
    Hope this helps.
