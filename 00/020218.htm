Open PowerPoint, Click on the Office button > check if the file is listed in the recent items list.

  4. Open PowerPoint, and then click Open on the File menu.
  5. Click the arrow next to the Files of Type box at the bottom of that dialog box, and then click All Files.
  6. Navigate to the folder where the TMP file is located.
  7. Click the TMP file, and then click Open.

jay freedman say:

If you're using "legacy" form field check boxes, you must protect the document for filling forms in order to make them work. When the document isn't protected, single-clicking a form field simply selects it, and double-clicking it opens its Properties dialog (where you can choose Checked or Not Checked, but that's just supposed to be the starting value, not the way you're supposed to use the boxes).

If you're using content control check boxes, they work with a single click as long as the Design Mode button is not activated. If design mode is turned on, the check boxes don't work at all.

A third but less likely possibility is ActiveX check boxes. If the Design Mode button is turned off, they work with a single click. If Design Mode is turned on, they don't operate, but double-clicking them opens the macro editor.
