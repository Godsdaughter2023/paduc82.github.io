ken sheridan say: To sort the form by last name then first name you simply need two rows of code, which you can follow with code to go to the first record matching the last name entered in the input box:

    Const MESSAGE_TEXT = "No matching record found."
    Dim stWhatSearch As String
    
    Me.OrderBy = "LastName,FirstName"
    Me.OrderByOn = True
    
    stWhatSearch = InputBox("Enter Claimant's Last Name!")
    
    With Me.RecordsetClone
         .FindFirst "LastName = """ & stWhatSearch & """"
           If Not .NoMatch Then
               Me.Bookmark = .Bookmark
           Else
             MsgBox MESSAGE_TEXT, vbInformation, "Warning"
         End If
    End With

Put the above in the button's Click event procedure.
