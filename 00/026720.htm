daniel g. grau say:
If the previous answers did help you, please indicate so in the previous post so other users will benefit from your experience.

So, Melina, you have an A3 document that won't fit on an A4 page. OK.

When printing, the actual size of the page does not matter, because you can print in any size of paper. In the Print dialog box, click on the third dropdown menu and choose Paper Handing. Then enable "Scale to fit paper size." In this case, the printed size will be reduced.

In addition, you have a manual scale control, at the bottom. If you enter 50%, all should be fine, but you may need to change the page orientation.

Note that if you had it the other way around (an A4 document printing on A3 paper), you would not need to do anything, unless you wanted it to print at a larger (or smaller) size.

Requery the textboxes in the AfterUpdate event of the combo box:

 

Private Sub Combo2_AfterUpdate()

Forms!Form2!Text0.Requery

Forms!Form3!Text0.Requery

Hello,

I would like to help you by providing basic information related to Sending reported error 0x800ccc0f. This error message appears in Outlook due the following reasons:

Reasons:

  • Connectivity issues in Network.
  • Damaged or Corrupted PST file.
  • Enabled Antivirus/ Windows Firewall have blocked the message.
  • Incorrect server settings.
  • Incorrect configured account settings.

You can try the given below solutions for above problems:

Solutions:

  • Make sure that you are connected to the Internet.
  • Repair Damaged or Corrupt PST file by performing scanpst.exe.
  • Disable Antivirus or Firewall program.
  • Remove Third Party Plugins.
