ken blake say:

You are running 32-bit Windows.

Nothing is wrong and there's nothing to fix.
All 32-bit client versions of Windows (not just XP/Vista/7/8/10) have a
4GB address space (64-bit versions can use much more). That's the
theoretical upper limit beyond which you can not go.

But you can't use the entire  address space. Even though you have a  4GB address space, most people can only use *around* 3.1GB of RAM. That's because some of that space is used by hardware and is not available to the operating system and applications. The amount you can
use varies, depending on what hardware you have installed, but can range from as little as 2GB to as much as 3.5GB. It's usually around 3.1GB.

Note that the hardware is using the address *space*, not the actual RAM itself. If you have a greater amount of RAM, the rest of the RAM goes unused because there is no address space to map it to.

¡firedog say:

With no programs running, press Windows key + R to open a Run box, type wlarp and press Enter. This should start the Windows Live Add/Remove Programs applet, where you can elect to uninstall all of the programs in the suite. When it's finished, reboot the computer.

This will not necessarily free up a lot of space on your machine. On mine, the program files for Windows Live Mail, Photo Gallery, Movie Maker and Writer together amount to 145MB. If you're sure that it's a shortage of space on your C drive that's adversely affecting the computer's performance, you may find that removing your store folder - where all your messages and calendars are stored - will help even more. If you're quite sure that you don't need any of these files (have you saved important emails somewhere else?), you can delete the folder, which by default is

C:\Users\{user}\AppData\Local\Microsoft\Windows Live Mail

where {user} is the name of the Windows user account. There will be one of these folders for each Windows Live Mail user on the machine.
