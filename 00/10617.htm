I don't think you can do what you want with Publisher. However you can try the 'text autoflow' feature which allows text to flow from one text box to another much as the text does in Word from page to page. The problem is you have to give up ability to precisely position the images because you would have to insert them 'inline' in the text so that they would flow from text box to text box with the text.


Reference my detailed response in this thread for how to use the text autoflow feature: How to insert text and photos in Publisher 2013: http://answers.microsoft.com/en-us/office/forum/office_2013_release-publish/how-to-insert-text-and-photos-in-publisher-2013/ea6b1755-7f23-435f-bcb3-a34de4f45d1d

Also reference: http://office.microsoft.com/en-us/publisher-help/create-link-is-under-text-box-tools-linking-HT102769754.aspx?CTT=1

Inline vs. Exact positioning - after you drag an image onto a text box filled with text you can choose to have the text wrap around the image and maintain the exact position on the page with the text flowing around the image or your can position the image inline such that it flows with the text. Right click the image > layout tab and choose inline vs exact to get your image to flow from text box to text box or to set the text wrap settings.

I guess it depends upon your particular page layout whether 'text autoflow' and 'inline positioning' will do what you want.
