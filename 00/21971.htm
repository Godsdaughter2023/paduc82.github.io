e.j. gun say:

Something like this will work.  It is not efficient within VBA to use Select, Copy and Paste.  Instead, you should learn to set cell values directly from other cell values.  The example below assumes you have sheets numbered in order, which may not be the case, but you'll get the general idea.

 

HTH,

 

Eric

 

 

Option Explicit

Sub Macro2()
    Dim i As Long
    Dim curSheet As Worksheet
    Dim theSheet As String
'
    Set curSheet = ActiveSheet
    theSheet = "sheet"
'
    For i = 2 To 101
        Sheets(theSheet & i).Range("B2") = curSheet.Range("A1")
    Next i
End Sub

   

ossiemac say:

You need to build the array of required sheets to be selected.

Below are 2 methods:

First method simply builds the array for all sheets between the first and last in the loop. However, please understand that the sheet indexes are numbered from left to right and if  a user shifts another sheet into this area then it gets included. Need to use a separate variable for building the array because the sheet numbers will not necessarily match the array element numbers. (Not a good method but demonstrates the method of identifying the sheets in the "from to" as per your question.)

Second method builds the array by testingall sheets and if the name of the sheet meets a criteria. Firstly tests that the first 5 characters match and then if the numeric suffix is within the required range. This would be my preferred method because it overcomes the problem of a user shifting the sheets.

 

'First Example:

Sub SelectMultipleSheets()
   
    Dim arrShts()
    Dim i As Long
    Dim j As Long
   
    For i = Sheets("Feuil2").Index To Sheets("Feuil10").Index
            j = j + 1
            ReDim Preserve arrShts(1 To j)
            arrShts(j) = Sheets(i).Name
    Next i
   
    Range("A1").Select
    Selection.Copy
    Sheets(arrShts()).Select
    Range("B2").Select
    ActiveSheet.Paste
    
End Sub

 

 

'Second Example:

Sub SelectMultipleSheets_2()
    Dim ws As Worksheet
    Dim arrShts()
    Dim j As Long
   
    For Each ws In Worksheets
        If LCase(ws.Name) Like "feuil*" Then
            If Mid(ws.Name, 6) >= 2 And Mid(ws.Name, 6) <= 10 Then
                j = j + 1
                ReDim Preserve arrShts(1 To j)
                arrShts(j) = ws.Name
            End If
        End If
    Next ws
   
    Range("A1").Select
    Selection.Copy
    Sheets(arrShts()).Select
    Range("B2").Select
    ActiveSheet.Paste
End Sub

Example with 5 names

  • Setup a table like the one in I2:J6 making sure the "Index" starts at 0 and inc. by 1
  • Formula in B3:
    =VLOOKUP(MOD(ROW()-1,5),$I$2:$J$6,2)

If you have 10 names change the above bolded 5 with 10 and adjust the VLOOKUP range - should be $I$2:$I$11...
