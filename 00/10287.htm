jim gordon mvp say:

This is not an error, as Phillip assertted.

The dialog is informational and part of the extreme reaction Microsoft has whenever someone shouts out that Office is insecure somehow. Because you are obtaining data from Outside the Word "sandbox" Microsoft feels compelled to let you know an additional application will be launched to allow you to import the data from the source you chose.

We're stuck with this dialog because some idiot somewhere might choose a data source that has malware or can attack their Mac and then Microsoft can say, "We told you this could happen."
