vasil l. michev say:

"Never delete" is a retention tag, it control automatic deletion of mail, and it will *not* prevent you from deleting mail. Basically it means that a message stamped with this tag will remain in the mailbox *until* the user takes any action on it (move it to archive, delete it, etc).

If you want to prevent users from deleting messages or you need to meet some compliance requirements, use Litigation hold/In-place hold.
