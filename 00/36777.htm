a. user say:

Hi,

Thank you for contacting Microsoft Community.

I would suggest you to follow the steps below to which will help you configure an Automatic Shutdown after a user defined amount of idle time

  1. Open Task Scheduler and go to 'Create Task'.
  2. In the Create Task window under the General tab, set a name and description for the task then dot "Run whether user is logged on or not" then put a check/tick at "Run with highest privileges", be sure to set the "Configure for:" for your flavor of Windows
  3. Now at the Actions tab at the lower left click "New", in the "New Action" dialog box leave "Action" set at "Start a program" at "Program/script:" type %windir%\system32\shutdown.exe then at "Add arguments" type /s /f then click OK.
  4. Now at the Triggers tab at the lower left click "New" in the "New Trigger" dialog box set the parameters that you want for the task.
  5. Next head to the Conditions tab, and make sure to check the box to only start the task if the computer is idle for more than x minutes. You can also choose to stop the task if the PC isn’t idle.
  6. Lastly, head to the Settings tab, and you can choose to restart the task every 30 minutes if the task fails—that way, if you are on the PC for a while past the normal time, the job will start up later on and shut down the PC.

Hope this information is helpful. Please do let us know if you need further assistance, we’ll be glad to assist you.
