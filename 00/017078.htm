The following steps explain how to change computer language to English in Windows XP.

1. Open Control Panel and double click on the Regional and Language Options icon.

2. In the first tab, Regional Options, select English (US), English (UK), or any other English (there are many versions of English pre-loaded) as per your wish. The option is available in the first drop down list under Standards and Formats . You may also click the Customize button next to the drop down to customize the format of date, time, currency etc.

3. Click on the next tab, Languages and then click the button labeled Details. In the Text Services and Input Languages dialog box, you can see all the installed languages. Select English in the list of available languages under Default Language Settings .

4. To create a keyboard shortcut for switching among installed languages, use the Key Settings under Preferences . Please note that you can also OK to close the dialog box.

5. To set English as the default language for the particular user account, click on the Advanced tab of Regional and Language Options dialog box. Make sure English is being displayed in the drop down box and check the last option in the dialog box .

6. Click OK to close the dialog box and logoff for the settings to take effect.

Though this is the most common method used to change the computer language to English in Windows XP, it only changes the GUI and sometimes creates problems with few applications such as Internet Explorer and Outlook Express. If you are facing such problems, you can download MUI patches from MS download center.

NOTE: The above method is often considered as a temporary solution when you have to use different languages. To change the computer language to English in Windows XP so that it has a system wide affect, it is recommended that you perform a fresh install of Windows XP and select English when prompted for language during installation. Later, you may install other languages as required.

When you double-click an attachment straight from a message in Outlook, the file gets temporarily stored in a temporary folder like /private/var/folders/z0/jtxtweflj2q7bw713ccvtzgm800343gn/T/com.microsoft.Outlook/Outlook Temp/ and the file is locked indeed to prevent any alteration.

Outlook won't let you modify it because 1) it's a temporary location that will get wiped out eventually and 2) altering the file would mean altering the content of the original email.

So in essence, you have to either Save as or Duplicate if you proceed like that (and no amount of preference deletion and reset withal change anything to that :-> ).

The easiest with attachments is to directly save them to the destination of your choice right from the email (and you can use drag and drop to drop them to the desktop or the folder of your choice for instance). When you say "translation", do you mean translating a section of a document from one language to another, or are you talking about spelling and grammar (proofing tools)?

For translation, Word 2007 uses a web-based service (WorldLingo), not something that installs on your computer. Select the text, go to the Review tab, and click Translate to open the Research pane. For spelling and grammar, you must buy a separate language pack, which you install after installing Office. If you had the pack, I assume you'd know about it.

    Incoming Server - imap.mail.yahoo.com
    Incoming Port - 993 (requires SSL)
  
    Outgoing Server - smtp.mail.yahoo.com
    Outgoing Port - 465 (requires SSL/TLS)
  
User name: full email address (for example, bill @ yahoo.com* or bill @ rocketmail.com*)
