daymondevader say:

Edge uses the Windows 10 Credential Manager to store names/passwords. Credential Manager can be found in the Control Panel.

The link below gives an overview of using Credential Manager.

http://edgetalk.net/how-microsoft-edges-password-manager-works/

Credential Manager in turn uses Credential Locker to store credentials.

The link below talks about Credential Locker for Windows 8.1 so some of the info is not completely up to date.

https://technet.microsoft.com/en-us/library/jj554668.aspx?f=255&MSPPError=-2147217396

Credential Locker is part of the operating system that securely stores credentials for windows.

If you are not domain joined (not a work computer), and you use a Microsoft account to log into windows, then your credentials will roam to other trusted devices that you have.

Backing up contents of an IMAP account - Microsoft Community

dz7 say:

Backing up contents of an IMAP account

What's the appropriate way to back up the contents of an IMAP account?

I read on the Office help website that the PST file for an IMAP account is not meant to be moved around (unlike other PST files). Is there another way to back it up such that I could move it to another instance of Outlook in the event of a problem?

a. user say:

What's the appropriate way to back up the contents of an IMAP account?

I read on the Office help website that the PST file for an IMAP account is not meant to be moved around (unlike other PST files). Is there another way to back it up such that I could move it to another instance of Outlook in the event of a problem?


No need to back up an IMAP email account at all.

The IMAP email account mirrors what is on the Mail server, so if your Outlook goes belly up, just re-create the account and all your subscribed folders (with their content) will magically re-appear...

If you find my response helpful, please click on the "Vote as Helpful" button!Thank you!

roady [mvp] say:
No need to back up an IMAP email account at all.

The IMAP email account mirrors what is on the Mail server, so if your Outlook goes belly up, just re-create the account and all your subscribed folders (with their content) will magically re-appear...

True, but that would only be the case if you can trust that the IMAP server will be backed up regularly. If somehow all the data would be lost on the IMAP server, Outlook would be empty after a sync as well; a local sync does not count as a backup, it's like having a RAID1 system.

If you want to make a backup of your IMAP account, you can simply copy the pst-file to your backup location. If you would need to restore data after a server failure, you can reconnect your pst-file via;
File-> Open-> Open Outlook Data File

You can then copy the missing content back to the server.

Note that you cannot use this backed up pst-file to skip the initial synching process with the IMAP server in the case of needing to reconfigure your IMAP account in Outlook.
