If you wish to highlight cells F104:K104 where cell value meets C104. Then follow below steps:

- Select the cells range F104:K104

- Type =C104

If you wish to highlight cells F104:K104 where cell value NOT meets C104. Then follow below steps:

- Select the cells range F104:K104

- Select Cell Values from format only cells with, Not euqal to, =C104

- Choose your cell formatting by clicking on Format button and then click on Ok.

dstottle say:

Fixed it!!!!!

I did find the first part Doug referenced in Word.  checking the box "confirm file format conversion to open", but that didn't make a difference.  (or maybe that combined with the next thing I found was the trick). 

When you open the word doc and say yes to perform the SQL command and merge the file the next box is text encoding.  I played around with these options and under other options found that by default it was picking Western Europe (Windows)  Changed it to Western Europe (Mac).
