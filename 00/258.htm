Contents and formatting should be preserved when copying if you make sure to select full rows, that is, if you include the end of row marks, before copying.

Most webpages contain information that tells the web browser what language and character set to use when displaying the page. This type of information is referred to as language encoding. 1.  Open Internet Explorer.

1.  Open Internet Explorer

If whatever is occuring is not the result of a sync or import, then with all due respect, more details are going to have to be provided. That said, some background information may be in order when using the "Add to Contacts" option from an email:

The FileAs is derived from the construct of the FullName and the FullName is set as per your global settings. The one key thing that must be kept in mind is that if you get an email that has <FirstName LastName (emailaddress)> and you have the FullName global setting set to <Last, First> - Outlook will populate the FullName with the value of <LastName> in the FirstName field and vice versa. That can be easily verified by clicking on the FullName button to see the exact values of how the contact name was populated particularly since the detail contact window opens when you add a new contact. It is from this FullName field that the FileAs field gets constructed.

A side comment, I would not consider this a <bug> that needs fixing since there is no way to know what value to expect in the email, the sender can just as easily have their info be <LastName FirstName (emailaddress)> or any variation in between so no matter what action Outlook takes in this instance - someone is not going to be happy with the results based on their established settings and there is no way to determine what the inbound value is "meant" to be.

Assigning Macros to Shapes - Microsoft Community

ironsiren say:

Assigning Macros to Shapes

I want to assign a macro to a shape that I created in excel.  I want the macro to export the chart I have in excel into a powerpoint file in particular powerpoint template.  Can someone please tell me how to do this Thanks so Much!
jllatham say:

Job 1: write (or record) the macro that does what you need to be done.

Next - select your shape on the sheet.  Move the mouse pointer over towards an edge until it changes to a small 4-headed arrow thing (like a + symbol with an arrow head on all 4 legs).  Right-Click on the shape at that point and choose "Assign Macro..." from the list that appears.  Choose the appropriate macro in the list and click the [OK] button.

When you click the shape, the macro will run.  I kind of like using the TextBox for this kind of thing since you can format them up very nicely and put almost unlimited text in it.
