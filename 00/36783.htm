bossdweeb say: In addition to what Mike H said -
12GB is the wrong memory configuration for that cpu/motherboard.
Your system has a Dual Channel memory controller, so  you should have either two or four sticks  of equal quantity  RAM with a   total of 2, 4, 8, or 16GB. 
That is to say you should not use 2x4GB and 2x2GB =12GB, or the RAM will not
perform optimally.
The RAM has to all be identical for the system to effectively run in Dual Channel mode.
While a kit of either 2 or 4 matched RAM DIMM's is best practice, two individual kits of matched DIMM's will generally be OK, but they must be the same quantity ( eg- 4GB), model and revision number.

lemp say:

First of all, not every computer (particularly laptops) that is sold with Windows 10 pre-installed will have Windows 7 drivers for all of its hardware.  The computer manufacturers may not want to do that.

In the case of the Acer ES1-512, Acer only supplies drivers for the 64-bit versions of Windows 8.1 and Windows 10.  See https://www.acer.com/ac/en/US/content/support-product/5611;-;Aspire%20ES1-512

There are, however, Windows 7 drivers for the Qualcomm Atheros AR956x Wireless Network Adapter.  For example, the Microsoft Update Catalog has them: 

Windows 7 x64: 20601467_31951e47b6f7a55c0c25e2e3de4b37d3056e8b67.cab

Windows 7 x86: 20601468_d2a8a3e5147c2709daca9dfd0afc299322ae18a1.cab

On the other hand, if you don't see an entry for your wireless network adapter in Device Manager, it's very likely that it's simply turned off.  AFAIK, all modern laptop computers have a means for disabling the wireless adapter.  For your ES1-512, this is an "airplane mode" that Acer describes as follows:

Acer notebook computers feature an Airplane mode hotkey that turns
the network connection on or off. You can use the network
management options to turn your wireless network on/off or control
what is shared over the network. From the Start screen, start typing
’wireless’ then select Connect to a network or Change Wi-Fi
settings. Otherwise, open Internet Explorer and follow the
instructions.

The "Airplane mode hotkey" is the Fn key and the F3 key pressed simultaneously.  What happens when you do this?  Unlike some laptops, the ES1-512 does not have a light that indicates whether the wifi is on.

If you don't see an entry for a wireless network adapter in Device Manager, how do you know that it's a Qualcomm Atheros AR956x ?  In fact, the Windows 10 wireless network adapter drivers for your model include 2 different Broadcom drivers plus an Atheros driver.
