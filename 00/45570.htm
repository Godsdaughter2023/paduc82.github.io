Please note, you need to have a qualifying license for Windows 10 if you plan on using the free upgrade.

Windows XP or Vista will not work as a qualifying license, you need a minimum Windows 7 or Windows 8 installed or have a product key for Windows 7 or Windows 8/8.1

Also, make sure your computer does support 64 bit Windows 10, some older processors even when 64 bit compatible will not run a 64 bit version of Windows 10, you might have to switch to the 32 bit version instead.

As noted, if you are able to boot from a flash drive, it will be sufficient for performing the installation.

How to: Perform a clean install of Windows 10

Prepare Windows 10 ISO file

After you have downloaded the Windows 10 ISO, you need prepare it, this is especially important for Windows 7 users..

After obtaining the .iso file you can use the built in Disc Image burning tool or Microsoft .iso to USB/DVD tool to create a bootable DVD or USB (requires a blank DVD or USB flash stick of at least 4 GB).

After downloading the .ISO file, right click it and click Burn disc image

Insert a blank DVD

Click Burn

Windows 8/8.1 users can mount the .ISO file and start the upgrade automatically. Learn more here

Important:

If you are creating a bootable USB thumb drive, make sure the thumb drive is first formatted as FAT32.

Connect the thumb drive

Open Computer

Right click thumbdrive

Click Format

Make sure FAT32 is selected as the file system.

For UEFI based systems

If your computer is UEFI based, these are normally systems that come pre-loaded with Windows 8 or later, you will need to prepare the ISO file for such a configuration or you will receive an error message during setup. The thumbdrive needs to be formatted as FAT32 and use the GPT partitioning scheme. To do this, you need to use Rufus, a small tool you can download for free.

 

After you have installed Rufus:

  • Launch it
  • Select ISO Image
  • Point to the Windows 10 ISO file
  • Check off Create a bootable disk using
  • Select GPT partitioning for EUFI firmware as the Partition scheme
  • Choose FAT32 NOT NTFS as the File system
  • Make sure your USB thumbdrive in the Device list box
  • Click Start
  • Close when complete

Hi BrokenHalo,

 

Firstly, check if your TV is capable of receiving S-video signals. If the TV is not compatible to receive S-video signals, then that is when it shows black and white image.

 

Also, the reason you get a black and white signal is because the Super Video (s-video) output sends two color groups to your TV, 1 for color information, the other for brightness and lighting information. If your TV is not designed to receive S-video source it may not pick up the color information hence the black and white image. 

 In this case, you can change the graphics card output mode to composite (RCA). If you’re not sure how to do it, then you can try using converters that makes your graphics change automatically to a composite output and give you full color.
