greginmich say:

As you might have already realized, this alert isn’t actually coming from Windows Defender, it’s just a Tech Support Scam popup – and the fake Zeus Virus alert is one of the oldest themes for this scam.

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/zeus-virus-pop-up/9f2681d1-2112-4757-b1c4-988ef4bec473 

Any kind of alert that just pops up in your web browser has got to be a fake – because legitimate alerts will either pop up in your PC’s notification area, or appear in an independent program/system window outside of your browser. The fake alert is just there to lure you into calling a fake tech support number where you’ll be scammed. These Tech Support Scam pop-up messages (dialogs) can be virus detections, hacker alerts, hardware error reports, system error reports, password requests, or what have you – but these warnings are all fakes, and you just have to ignore the content and close the page.

Of course the problem is that most of these Tech Support Scam websites will try to coerce you into calling their fake tech support number by preventing you from closing their page. They do that with a dialog loop, which is just a script that reloads the fake pop-up message (dialog) every time you try to close it. So in order to escape from one of these malicious pages, you first have to try to close the alert. That allows the browser to detect the loop, and then add a Dialog Loop Protection checkbox (e.g., “Don’t let this page create more messages”) to the fake alert popup – and checking this option should allow you to close the page.

On browsers that don’t support Dialog Loop Protection, you still have to close the browser window with Task Manager (right-click on the taskbar; start Task Manager; and then end the browser process) and then go directly to your home page when you open the browser again:

https://answers.microsoft.com/en-us/protect/forum/protect_other-protect_scanning/1-888-2361-scam/0990122e-61a9-431b-947e-a904c202fa7c?msgId=8d26f1a8-041a-4351-9c24-a222fb305cbd

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/microsoft-virus/5335f80f-4cfc-49e0-bbcf-8f575730779a
