Unable to add to Reading list because it doesn’t appear on share - Microsoft Community

analy [msft] say:

Unable to add to Reading list because it doesn’t appear on share

I want to add something to my Reading list but I don’t see it as an option when I open Share. I have the app installed and it opens up fine.

analy [msft] say:

a) The app doesn’t support sharing links


Most likely the app that you’re trying to share from doesn’t support sharing out links. Share will display apps based on what you’re trying to share. For example, if you’re trying to share an image you may get different options than if you’re trying to share text. Reading List will only show up if you’re sharing out links.


b) You’re in a desktop app


If you’re in a desktop app you will notice that on top of the Share it says “Screenshot”, this is because sharing is not supported from the desktop and the only option available is taking a screenshot and sharing it as an image. So you’ll only see apps that support sharing images when you share from the desktop and Reading List does not support it.


c) You’re in a desktop web browser


Sharing from Internet Explorer should work fine with Reading List as long as you’re inside the Internet Explorer app* and not the desktop version. The Internet Explorer app will open when you launch it from the tile on your Start screen but you’ll only get the Internet Explorer app if Internet Explorer is set as your default browser.


The easiest way to know if you’re in the desktop version is if it has minimize, maximize and close buttons on the top corner.


Here’s some information on how to Make Internet Explorer your default browser.


*Make sure you don't have any text selected on the website, if you have a selection you will be sharing the text instead of a link to the webpage.


Note: Other browsers may also support an app version that should work with Reading List. For instance the latest version of Chrome enables an app version if you set it as your default browser so Reading List should work from there too.


d) You disabled the app from being a share option


You can change which apps are listed in the Share charm so check to ensure Reading List is enabled.


To do so:


1. Swipe in from the right edge of the screen, tap Settings, and then tap Change PC settings.
(If you're using a mouse, point to the lower-right corner of the screen, move the mouse pointer up, click Settings, and then click Change PC settings.)
2. Tap or click Search and apps, and then tap or click Share.
3. Change the settings you want.


You can find this and more information on Search, share, print, and more


Hope this information is helpful.


You can find more help on Reading List in the Reading List FAQ

considerate_guy say:

A .vob file (video object file) contains MPEG-2 video -- the format used on DVDs. This format requires a licensed MPEG-2 codec to decode the video for playback (or editing).

In previous versions of Windows (Windows Vista and Windows 7 in particular), this licensed codec was included to allow you to play back DVDs and to import/edit the contents of non-copy protected DVDs into a program like Movie Maker.

Subsequent versions of Windows (Windows 8, 8.1 and 10) have not included this codec as part of the shipping version of Windows. My guess this was partly a cost-saving measure (since each copy of Windows would require a licensing fee be paid to the MPEG committee) and also keeping pace with technology. As you are probably aware, the .vob format on a DVD can't contain HD video. The video DVD format was created in the late '90s before the wide adoption of consumer HDTVs. It has a maximum resolution of 480p (NTSC). Since most consumer video from digital camcorders and smartphones is HD video quality, it makes less sense to continue working with this legacy SD format.

So even if you reinstall either Windows Movie Maker 6.0 or Movie Maker 2012, they won't be able to read the .vob file. This is also true of the Photos App on Windows 10. Unless you have paid for a licensed codec for decoding/editing the MPEG-2 video format.

One workaround I've seen is to find a conversion program that can convert from .vob format to something more modern like .wmv or .mp4. That should import smoothly into either Movie Maker or the Photos App. Do realize though that the quality of the video will be no better than 480p even if you change the output resolution to HD.

Windows 10 Digital License
When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already own to reactivate if necessary. or use the reset function in Windows 10.
