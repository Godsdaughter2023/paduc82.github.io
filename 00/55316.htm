frederik long say:

Since I'm not an expert I was wondering if there is a way to install windows 10 on my SSD and be able to boot from there while keeping my HDD as it is?

You can do it like this:

  1. Disconnect your existing disk, connect your SSD.
  2. Install Windows 10.
  3. When Windows is up and running, reconnect your existing disk.
  4. Set the SSD as the primary boot device in the BIOS.

This will work perfectly.

ken blake say:

See JimWynn's reply and let me say much the same thing, in slightly different words.

Neither Windows 10, nor any other version of Windows (except Windows 8 RT), has ever included Excel, Word, PowerPoint, Access, nor any other significant application software. Such programs have to be bought, either by themselves or as part of Microsoft Office.

If your previous computer, running an older version of Windows, came with one or more of these, it was because the vendor who sold it bundled it as part of the package he sold you, not because that version of Windows came with it. Some, but not all, vendors do the same with Windows 10 computers (but if they do, it is very likely just a trial version).

If you want to buy Word, or Microsoft Office, you don't have to buy the 2016 version. Word 2013, 2010, and 2007 will all run under Windows 10. The older versions, of course, if you can find them, will be less expensive than 2016.

Instead of Word you might want to consider the free OpenOffice or LibreOffice.Neither is perfectly compatible with Word, but both are close.
