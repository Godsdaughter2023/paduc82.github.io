I realize the inconvenience you are experiencing while installing Windows update on Windows 10. I will certainly help you.

This problem may occur if one or more of the following conditions are true:

  • The Windows Update database is corrupted.
  • Antivirus software is configured to scan the %Windir%\SoftwareDistribution folder.
  • The Windows Update service stops during the installation process.
  • A required system file is registered incorrectly.

I suggest you to restart the Windows Installer service and check if it resolves the issue. Windows Installer service detects and enables the download, installation of the Windows Update. Restarting this service can resolve these type of issues. Here's how:

1. Right click on Start, select Run and type services.msc, press Enter.

2. The Services window opens, search for Windows Installer service.

3. Double click on the service, click on General tab, under Startup type, select Automatic.

4. Then click on Start > Apply >OK to save the changes.

5. Check if the issue is resolved, if not restart the system.

If issue persists, perform a Clean Boot to check if any third party application is causing the issue, as Clean Boot helps in eliminating software conflicts that occur when you install a program or an update or when you run a program in Windows.

https://support.microsoft.com/en-us/kb/929135
