brian tillman [outlook mvp 2007-2019] say:
OK, then, double-click an account, then click More Settings, then select the Advanced tab. There you will find a check box that is labeled "Leave copies of messages on the server". Make sure that's unchecked. This will cause Outlook to send a delete command to the mail server after it downloads the messages. Now, if your mail service honors the delete command, the messages will be removed from the server. By default, some servers, like Outlook.com and gmail, do not honor the delete request, but there are server-side settings that can be changed so the commands are obeyed.

suzanne s. barnhill say:

In the section where the page number is a constant 109, the likelihood is that the page number has been typed instead of inserted with a PAGE field. Access the footer in that section, delete the page number, and press Alt+Shift+P to insert a PAGE field.

As for the TOC, you'll find help in http://wordfaqs.mvps.org/TOCTips.htm, but the general principle is that you need to modify the TOC styles at each level to be flush left while retaining the graduated hanging indents. You can also apply bold or any other desired font formatting.

Also, if you want your page numbers to be flush right (without the hyphens), then go to each section that has the page numbers formatted that way and go to Insert | Header & Footer | Page Numbers | Format Page Numbers.... In the Page Number Format dialog, select 1, 2, 3 numbering instead of - 1 -,- 2 -, - 3 -. If you want the hyphens, you can add them manually to the page field in the footer; they won't be picked up by the TOC.
