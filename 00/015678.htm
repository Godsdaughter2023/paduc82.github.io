      jr de pijper say:

      Unfortunately, in OneNote there is no way to achieve what you want.

      Because as a heavy user of both OneNote and Outlook for my planning stuff I wanted the same kind of thing that you do, I wrote a little utility for myself - still pretty much beta stuff- that generates an Outlook task and uses the notebook and section names to assign an Outlook category to the task. So the task's subject would still be "Send release to purchasing", but it would have a category of "AA.HMF". This way, you can get some pretty useful task views, since in Outlook you can group, order and filter on these categories.

      The utility as it is now also generates a OneNote tag, so that you can also use the tag summary feature in OneNote.

      I have this implemented as a standalone utility, not as a OneNote addin. Instead of hitting a button in the OneNote ribbon, I hit Win-R (run) and type "task". This generates an Outlook task and OneNote tag for the current paragraph in OneNote. The utility also has a number of switches to fine-tune the command. So I can give commands like "task /today", "task /due 25-12", "task /complete" and "task /high" (for high priority tasks).

      If you are interested, I am willing to make it available to you and since I am still in the middle of developing it, you can even tell me how you want it. For instance, if you prefer to have the notebook/section reference in the task subject rather than as a category, this is no problem. Maybe you can think of other things that I have not thought of.

      Let me know if you are interested.
