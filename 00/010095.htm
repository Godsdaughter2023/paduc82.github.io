jeeped say:

Lets say your column of values is in column A starting in the second row and continuing to row 999. Select A2:A999 (with A2 as the active cell) and create a conditional formatting rule based upon a formula. use one of these for the Format values where this formula is true:

To indicate all of the second, third, etc. duplicates, (not including the first occurrence) use,

=COUNTIF($A$2:$A2, $A2)>1

To indicate all of the duplicate values, including the first occurrence use,

=COUNTIF($A$2:$A$999, $A2)>1

Click the Format button and select a yellow Fill. Click OK to accept the formatting and then OK again to create the new CF rule.

A representative was unable to set a default. Short of changing the default within the program I am searching for another way. I have .jpeg of my preference. Thank to all who have suggestions. jay freedman say:

Probably the easiest to use is not a Quick Access Toolbar button but an AutoCorrect entry. Set it up this way:

  • Insert your jpeg image into any document.
  • While the image is selected, go to File > Options > Proofing > AutoCorrect Options.
  • In the "Replace" box, type a name for the AutoCorrect entry. It should be something that's very unlikely to be typed by mistake, such as \hbar. The "With" box will be empty because Word can't display images there, but the "Formatted" option button above it should be activated automatically.
  • Click the Add button and then click the OK button.

To use this to insert the image in a document, just type the name you assigned, followed by a space or pressing Enter.

There are other possibilities, such as using an AutoText entry or a macro to do the same job. An AutoText gallery can be added to the Quick Access Toolbar, as can a button to run a macro. I think the AutoCorrect entry is simpler to use than the others.
