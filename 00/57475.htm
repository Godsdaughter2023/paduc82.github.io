steve durbin say:

I found this was due to permissions in the registry - Windows 10 upgrade seems to cause an issue here. The relevant key is HKEY_CURRENT_USER\Control Panel\International.

The same issue also stops WiFi from working - symptom is that you can't see WiFi networks in range in explorer, but can see them using NETSH from the command line. You can't connect to any.

To fix - Registry changes, so please backup first!

Open Regedit.exe

Navigate to HKEY_CURRENT_USER\Control Panel\International. Try opening it - you should see at lease a subkey of Geo, and possibly two others User Profile and User Profile System Backup. If you can't see them, the permissions error is probably present.

Right click on International, select Permissions. Select Advanced. Click on "Change" next to owner if it's not already set to (machinename)\Administrators, and set it to "Administrators". Click "Add", select "Users" and tick Full Control, click OK. Repeat for "Administrators". Tick "Replace all child object permission entries with inheritable permission entries from this object". Click OK.

You may get an error about some keys not being changed. Check you can see permissions for users on the "Geo" subkey; this MUST be visible. The "User Profile" subkey should also be visible, the "User Profile System Backup" one doesn't seem to matter.

Reboot the machine and check to see if you can now change region / use WiFi.

Hope this helps!

Esto es lo que hace la resolución 1080i:

  • Muestra las imágenes con una resolución máxima de 1.920 x 1.080 píxeles.
  • Ofrece un área de visionado de alta definición (HD) de 16:9.

La "i" de 1080i es la abreviatura en inglés de "barrido entrelazado". Esto significa que el televisor de alta definición "redibuja" la mitad de las líneas que componen las imágenes de la pantalla 60 veces por segundo.
