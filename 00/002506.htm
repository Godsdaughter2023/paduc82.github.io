___chris___ say:

Hi Cal,

 

I was able to fix my Bing Desktop install using to following steps. Though, it's kind of hacky and you have to dig in the registry a little bit. So...

 

Here's the steps I followed (all without the double quotes).

 

1: Find the BingDesktop installer file in C:\Windows\Installer. On my pc, this is the file I found:

        06/27/2013  12:25 PM         9,793,536 4ffcd2.msi

    I found an actual file called BingDesktop.msi, noted the size, and looked for one of the same size in C:\Windows\Installer. I new it was the right one because I got the same error when I tried running it.

 

2: Go into the registry editor. To start it press Windows+R, (Or click "Start" then "Run") and type "regedt32" in the run box, and click okay. you will probably have to grant permissions when asked.

 

3: In the registry editor, navigate to:

  Computer -> HKEY_LOCAL_MACHINE -> SOFTWARE -> Microsoft -> Windows -> Current Version -> Installer -> UserData

 

4: In the "Edit" menu click "Find" (or press Ctrl+f), and search for "BingDesktop". Keep searching (F3 will do "Find Next") until you find an entry that has a folder called "InstallProperties" and a bunch of stuff listed in that folder. For me, it was in:

  S-1-5-18 -> Products -> 554590D7179DC4D4E9DFA96F6A85F4A3

 

5: One of the lines in "InstallProperties" will be "LocalPackage". Double-click on that and change the file name to the one that you found in C:\Windows\Installer. On my machine, I changed the LocalPackage to read "C:\Windows\Installer\4ffcd2.msi"

 

6: Go to Add/Remove Programs (Or type "appwiz.cpl" in the run box that you used before). When it finishes loading, click on "Bing Desktop" and click "Repair" at the top.

 

The repair should run, ask you to restart your computer, and you should be good to go. :)

 

I hope that this helps!
