daniel g. grau say:
  • Easiest: Save As > Plain Text (extension .txt) using the default options (Mac OS).
  • A bit more work: Select All (Command A), Edit > Paste Special > Unformatted text. Then Save As with another name.
The difference:
  • Plain Text files contain just characters.
  • Unformatted text stored in a Word document is an oxymoron, as it still contains formatting information, except that it's not evident, since everything takes on the looks of the Normal style (which of course has its own font, size, color, etc.).
Text editors are just primitive word processors. Some offer a certain degree of on-the-fly formatting (for instance, TextWrangler can visually structure HTML text), but in general they only produce Plain Text files.

ossiemac say:

The window that comes up is the Text Import Wizard. You go through this and select the delimiter or fixed width. If a delimiter then select the type of delimiter etc. If fixed width then in next dialog you can drag the column widths or remove a width line by dragging off the box.

Further through the dialog boxes you get to select the type of data in each column. Mostly this is not required but if you have numerics like phone numbers then you can select the particular column and make it text so that leading zeros are retained. Also has a date selection where you can select the particular column and the date format of a column. Note that the date format selected is the format that is in the text file; not necessarily the date format of your region.

Towards the end you need to select the first cell of the destination.
