suzanne s. barnhill say:

If what you want is actually the usual 1 followed by 1.1 and 1.2, 2 followed by 2.1 and 2.2 (discarding the red herring of Level 3), then the problem is that the Level 1 and Level 2 numbering have gotten reversed in Level 2. The way to solve this is to start from scratch:

  1. With the insertion point in the first Level 1 paragraph, open the Define New Multilevel List dialog.
  2. Select Level 2.
  3. Clear the text from the "Enter formatting for number" box.
  4. Under "Include level number from," select "Level 1. You will see "1" in the "Enter formatting for number" box.
  5. Type a period after the number.
  6. Clearing the box will have set the "Number style for this level" to "(none)." Change it back to "1, 2, 3." You'll see "1.1" in the box above.
  7. If necessary, perform the same procedure for lower levels, including each subsequent heading level.

In straightening out level numbering, it can be helpful to temporarily reset the starting number for each level, setting the starting number for Level 2 to 2, Level 3 to 3 and so on. This makes it very clear which level is which in the "Enter formatting for number" box. If you see 1.2.3.4, you know it's right. If you see 1.2.4.3, you know it's gotten mixed up.

 

If you just want standard 1, 1.1, 1.1.1 numbering (or some variation of it, such as 1, 1.1, 1.1.a), the easiest thing to do is to start over with the multilevel list that is already formatted that way and linked to the heading styles. It's easy to change the number style and indents without affecting the levels and numbers themselves.

jason (office engineer) say:

This is more of a YouTube question than a Sway question.  While you can't only embed a portion of a video, you can embed it so that it starts at a particular spot.  To do so, do the following:

  1. Go to Youtube.com and open the video you want to use
  2. Click Share, click Embed, click Show More
  3. Copy the embed code
  4. In your Sway, insert an Embed card
  5. Paste the embed code into the card
  6. Within the embed code, at the end of the URL, inside the ", add ?start=48 (where 48 represents the number of seconds into your video you want it to start)
  7. Click out of the card to commit it
  8. Press the Undo button (or press Ctrl+z) to keep the card as a generic Embed card instead of letting it convert to a Video card

Hope this helps!

Jason
