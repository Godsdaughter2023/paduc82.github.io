yann... say:

How to get Excel 2010 to display a csv properly?

Hey all,

 

I have a user running Windows XP and Excel 2010 that is not able to have csv files displaying in a user friendly page layout just like a regular xslsx file.

Is there a way to fix that without playing with regional settings and taking a chance to messup other programs?

 

Thank you.

 

Yann

joeu2004 say: Yann wrote:
I have a user running Windows XP and Excel 2010 that is not able to have csv files displaying in a user friendly page layout just like a regular xslsx file.  Is there a way to fix that without playing with regional settings and taking a chance to messup other programs?

Perhaps not.  It depends on exactly what you mean by "user friendly page layout like a regular xlsx file".

The best that you can expect is what you would see in a "regular file" when you initially enter data before modifying any formatting.  In other words, opening a CSV file in Excel is like entering data into cells formatted as General.

Making matters worse, Excel does not always write text to CSV files as quoted strings, and it does not always read quoted strings as text.  Consequently, some "numeric text" becomes numbers when the file is reopened, which can result in inappropriate changes, even errors.  For example, account IDs with more than 15 digits are changed.

Sometimes you work around these problems by selecting a format for an entire column using the Text Import wizard.  But that is severely limited; and it does remedy all such problems.

And of course, we can fix anything by reading the file using VBA.  SMOP! ;-)

Bottom line:  IMHO, CSV files should be used only for passing raw data from one application to another.  If you want to re-access the data with Excel, use an Excel file format (xls, xlsx, xlsm, etc).

subhash_uvn say: Hi,
You can do it, but its lengthy & crude method. It can be done using PC suite.

Install PC suite.
Keep your music files under different folders e.g. English Rock, English Classics etc. Put these folders under Music folder on Flash drive.
Now when you open music manager in PC suite, you see all song present on Flash drive under My Collection -> Nokia 5310 XPressMusic title.
Now go to View option of music manager & enable path option in it.
Now you can see file names & their locations on screen. Now sort the file according to Path.(By clicking on 'Path' tab on upper right hand side)
Now Select files by one-by-one clicking in any perticular path i.e.Folder and right click on those selected files. Their you can see option to add to playlist.Add these all files from perticular location to this playlist.

Thus all files from one location are added to playlist.
