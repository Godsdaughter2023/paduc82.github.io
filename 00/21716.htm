column B = A * 1.0417  (100/96)

You can use the Paste special function to multiply all of the values by 1.0417.

  1. Copy column a into column b
  2. type 1.0417 into any cell
  3. Select that cell
  4. <CTL><C> to copy 1.04017 into the clipboard
  5. Select all of the values in column be
  6. open the paste special dialog: Home tab > Clipboard group > Paste drop down > Paste Special command. 
  7. Click on the "Multiply" radio button
  8. click OK

This will multiply all of the values selected in B column by the value in the clipboard
