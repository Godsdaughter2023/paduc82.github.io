argmarovski say:

Hi,

1. Build 10240 is Escrow, not RTM. It is officially stated that the build watermark will be cleared for this build.

2. Note that Enterprise Editions do not qualify for a free upgrade.

3. When you install Windows 10 Preview build on top of a previous OS it will save your previous activation data. If you perform a clean install, it will ask for a key and in order to receive the free RTM upgrade on that machine you will need to reinstall your previous OS. Now Windows 8/8.1 stores activation data differently than Windows 7, so there is a possibility that the actual activation data of Windows 8/8.1 is still preserved. If this is a laptop or OEM OS version, activation keys are usually stored in Bios/UEFI. You can verify that using a small utility that would pull out your older Windows OS key in case it is present. No such scenario if your previous OS was Windows 7 & you performed a clean install on a custom built PC.

4. Activation of newly installed builds is currently suspended until 7/29. You cannot install new builds with the upgrade program & in case you have old ISO, it will not activate. Latest ISOs are removed.

5. In regards to your daughter's computer I have two suggestions: first verify in Bios that Execute Disable Bit & CPU Virtualization options are enabled & second, you need to use a PS/2 connected mouse & keyboard to install Windows 10 on an older machine. USB connected ones will fail the install.

pc-learner say:

Unless you have saved an image from a previous (but not too old) Insider build which was the Pro version and was marked activated, I'm afraid you are out of luck, since the versions preceding 101240 cannot be activated any more, and upgrading to 10240 requires using a Microsoft ID on an activated build.  If you have such a version, you should reimage with that and then let the system upgrade to 10240, which should preserve the Pro status.

In general, the Insider program should have installed the corresponding version to the one you upgraded from, but there were a lot of upgrades, and the rules seemed to be a little different when some (or any) of the ISOs were used for either an upgrade or clean install.  Unfortunately, it may be too late to correct your situation without contacting a real Microsoft support person after July 29.

As far as your daughter's computer is concerned, since it looks like that one has never been in the Insider program, it's too late now and you should wait until after July 29th to try it.  You should be able to use the "Get Windows 10" program on her computer now to make a reservation.  That final version may work better, or you may want to wait until the official ISOs are available after the 29th.  Give the upgrade a try then before falling back to a clean install.  Also make sure she has the latest video drivers and network drivers for her OS version already installed before starting the upgrade.

If the Show AutoCorrect Options are enabled, it'll cause the auto-correct lightning bolt in Outlook to show up. To resolve your concern, we suggest that you follow the steps below:
