Let me help you understand things better.

Windows 7 was designed in such a way that only the product key stays on the Hard Disk and not on to any physical hardware. If it an OEM Edition, then these tools will help you retrieve the OEM keys from the Registry, which i accept. But if you are doing a reinstallation of Windows, you have already lost the key the moment you wipe out the Harddisk. And the only way to retrieve it back would be from the sticker of the Device. After installation, if you use these tools and retrieve product key off the installation, what you will get is called a Default Key - that is used to skip installation.

Having a single admin account is as risky as having a single house key. Sooner or later you lock yourself out. Recovering from this predicament is not easy. You could have the machine serviced or you could go through a rather complex process. Here is Phase 1:

  1. Set your BIOS so that it uses the DVD drive as its primary boot device.
  2. Boot the machine with your System Repair CD. If you do not have one, ask a friend to burn one for you via the Control Panel / Backup and Restore. It takes less than five minutes.
  3. Press a key when prompted to boot from the CD.
  4. Select "Repair PC" from the menu.
  5. Select "Repair" when prompted.
  6. Select the Command Prompt from the menu.
  7. Identify the correct drive letter for your Windows installation. It could be a drive other than C: but it won't be X:. You do it by typing the commands
    dir C:\Win*
    dir D:\Win*
    dir E:\Win*
    until you find the Windows folder.

The maximum graphics color bit depth is 32 bits. It's a completely separate thing from the computer hardware or OS being 64bits.

The first thing to try IMO would be use the Windows7 System Restore (NOT a SONY system recovery) to  reset the computer to where it was before that VAIO update started.  If there are any restore points from when you first powered up the laptop that is.

Sounds like a corrupted video driver is causing your issue. Was it actually a Viao update from Sony that ran or was it just an MS update naming some Viao hardware or other? Makes a difference. If it was an MS update, after the System Restore you can temporarily turn off automatic updaties in Windows before the problem update starts again and then stop that particular update from installing automatically again by "hiding" it in the MS updates window. MS Update displays a list of available updates and when you click on each one for its detailed description you get a "hide this update" checkbox below the text. Once hidden the update will be ignored until/unless you unhide it again. MS drivers and 3rd party drivers for the same piece of hardware don't often play nice together so stick with Sony's original drivers or download Sony's updates for your computer when you have the choice.
