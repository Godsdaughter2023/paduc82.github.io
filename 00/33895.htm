stefan maerz say:

Generic driver for an old multimedia video controller

Original Title: Multimedia Driver

I have a Dell Optiplex GX280 with Windows Xp Pro. I have an old Multimedia video controller that has 4 connectors (I think are BNC connectors). It is old and has no label, so I don't know where to hunt down a manufacturer for a driver. I am wondering if there is a generic driver available.

Any suggestions on how to proceed would be appreciated. Thanks!

vinod n say:

Hi,

Every device has a Vendor and Device id associated with it. If you can find this ID, you can find the manufacturer. In Windows it's easy to find the vendor and device id.

a)    Open Device Manager (Control Panel>System>Hardware>Device Manager)

b)    The hardware whose drivers are missing will appear as Unknown device, so it's easier to locate the device.

c)    Right click on the unknown device and click on Properties.

d)    Under the Properties window click on Details tab and select Device Instance Id from the drop down box.

e)    You should see a code similar to this

PCI\VEN_8086&DEV_27DC&SUBSYS_30868086
&REV_01\4&1E46F438&0&40F0

 

f)     The portion of the code highlighted in RED is the Vendor ID and the portion highlighted in GREEN is the Device ID. In this example:

Vendor ID = 8086
Device ID = 27DC

 

Now you may download and install the drivers from the manufacturer’s website.

For your reference: Information about hardware device drivers for Windows XP. http://support.microsoft.com/kb/319908

 

Hope this information is helpful.
