ken sheridan say:

You need an input mask, not a validation rule.  The former controls the format, the latter the allowable data values.  Try:

 

00\->L<LL\-0000\ 00:00:00\ >LL;0;_

 

You should also Format the control as:

 

dd-mmm-yyyy hh:nn:ss AM/PM

scottgem say:

It doesn't matter how you enter date/time data. A date/time datatype stores data as a double precision number where the integer portion represent the number of days since 12/30/1899 and the decimal portion a fraction of a day (i.e. .25 = 6 AM),

How you display the value is dictated by formatting. You can format a date/time datatype to display the value in any valid date/time format.

john w. vinson say:

You can check the value for a nonzero time portion in the control's BeforeUpdate event. 

However, bear in mind that midnight (the zero default if no time is entered) is itself a valid time (one that's coming up here in, hmmm, 79 minutes or so); it might not be correct to forbid the entry of a midnight value! That's up to your business rules of course!

Try something like

Private Sub txtMyDateTime_BeforeUpdate(Cancel as Integer)

Dim iAns As Integer

If IsDate(Me!txtMyDateTime) Then

  If TimeValue(Me!txtMyDateTime) = #00:00# Then

      iAns = MsgBox("Do you want to enter midnight as the time?", vbYesNoCancel)

      Select Case iAns

          Case Is vbYes

               ' do nothing, just leave the zero time

          Case Is vbNo

                 Cancel = True

          Case Is vbCancel

                 Cancel = True

                  Me!txtMyDateTime.Undo

       End Select

  End If

Else

   MsgBox "Please enter a valid date and time", vbOKOnly

   Cancel = True

   Me!txtMyDateTime.Undo

End If

End Sub

amit tandon say:

A solution using Excel formulas:

Enter below formula as an array formula (Ctrl+Shift+Enter) in cell D2 & copy down:

=IFERROR(INDEX(A$2:A$100, MATCH(0, IF(IFERROR(A$2:A$20-A$1:A$19,0)<>1,0,1)+COUNTIF(D$1:D1, A$2:A$100)+(A$2:A$100=""), 0)), "")

Enter below formula as an array formula (Ctrl+Shift+Enter) in cell E2 & copy down:

=IF(COUNTIF(A$2:A$100,D2+1),MIN(IF((ROW(A$2:A$20)-ROW(A$2)+1>MATCH(D2,A$2:A$20,0))*((A$3:A$21-A$2:A$20<>1))*(A$2:A$20<>""),A$2:A$20,"")),"")

Presuming the numbers in column A are in ascending order. If not the formula can be modified accordingly.
