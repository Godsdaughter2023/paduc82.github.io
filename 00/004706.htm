The conditional format I desire is to change the cell color based on a value in another cell.  (The value in the other cell is based on additional logic, and the cell value is either I, M, or T.  If the value of the other cell is I, I want the cell with conditional format to be shaded red.  For M, I want the cell background to be yellow, and for T, I want green.

Here's some of the code I've tried.  The variable i is the row number.  For row 4, I want this code to make cell E4 background color 255 if cell J4 contains the letter I.

Sheets("Scenarios").Range("E" & i).FormatConditions.Add Type:=xlExpression,Formula1:="J" & i & "=""I"""
Sheets("Scenarios").Range("E" & i).FormatConditions(1).Interior.Color = 255

If there is a far easier path, I'm open to suggestions for that.  However, I'd still like to get a better understanding of quotes and variables within formula expressions.

hansv mvp say:

Rule 1: You cannot use variables inside quoted strings. You must concatenate the fixed and variable parts with the & operator.

Example: You want to assign the formula =SUM(A1:An) to the active cell, where n is a variable.

Dim n As Long
n = ...
' This will not work:
ActiveCell.Formula = "=SUM(A1:An)"
' This is the way to do it:
ActiveCell.Formula = "=SUM(A1:A" & n & ")"

Rule 2: To include a double quote " within a quoted string, you must double the double quote to "".

Example: You want to assign the formula =SUMIF(A1:A100,"North") to the active cell.

' This will confuse Visual Basic:
ActiveCell.Formula = "=SUMIF(A1:A100,"North")"
' This is the way to do it:
ActiveCell.Formula = "=SUMIF(A1:A100,""North"")"

hansv mvp say:

You should use the formula =J4="I" interactively (note the = before J4). When you do this in code, you should use

..., Formula1:="=J" & i & "=""I"""

Note: if you set the same type of conditional formatting rule for a range of cells, you don't need to loop through the cells, you can use code like this:

Sheets("Scenarios").Range("E4:E100").FormatConditions.Add Type:=xlExpression, Formula1:="=J4=""I"""

Excel will interpret the condition for the first cell and automatically adjust it for the range.

Prevent losing email in Outlook.com when you've set up POP3

Outlook.com has a feature that prevents losing email messages if you accidentally set up POP3 to delete them after reading them on the server. Here’s how it works:

    The first time that Outlook.com receives a POP3 instruction to delete email messages after reading from an email app, Outlook.com won't delete the messages.
