tim liu msft say:

Hi HeatherL24,

 

To achieve requirement in SharePoint Online, we need to user Calendars Overlay feature. So, different event can display different color in the same calendar. Below are the steps of an easy example:

  1. Create a SharePoint Online Calendar via adding an app. I call it C2.
  2. Click CALENDAR tab>List Settings. In the Columns section, find and click Category column. Enter Payroll and Tax, select Yes for Require that this column contains information.

     3. Return to the calendar, click CALENDAR>choose the default Calendar view>Modify View>in theFilter section, configure the settings as the screenshot shows below>OK.

     4. Return to calendar>click CALENDAR>Create View (in the Manage Views section)>enter view name Payroll>in the Filter section, configure settings as below.

      5. The same method to create a view for Tax.

      6. Return to calendar>click CALENDAR>Calendars Overlay>New Calendar>enter Calendar Name Payroll, choose a color, click Resolve> after resolving, you can choose C2 in the List section, and then choose Payroll (the view we create in the former steps), select Always show>OK.

      7. Click New Calendar, use the same method to create an overlay for Tax view. Select another color for this overlay.

    8. Now, return to C2, and make sure it displays the default Calendar view. When you add events, you need to choose a category. The different categories will display different colors on the default view.

This is an easy example. You can change the settings according to your requirement.

 

Best regards,

Tim
