Table border: dashed white border prints solid

I am trying (so far without success) to apply a dashed white border to a single table cell. Try the following
  1. create a 3x3 table
  2. remove borders from table
  3. apply shading of RGB 89,89,89 to table
  4. apply 1pt dashed border (4 option) to the top and bottom of the central cell only

Is the border solid?

I have tried this on 3 separate machines just to rule out a broken installation and get the same result each time. (I have also tried this in Word 2013 with the same result.)

Have I discovered an undocumented feature?


italophile say:

Actually the answer is simpler than that. After giving this some further thought I realised that the border appears solid because it has white dashes with white spaces in between. This is because instead of placing the border over the shading Word actually replaces the shading with the border.

So to get a dashed white border on a shaded table you need to think in reverse and create a dashed border of the same colour as the shading. That way the space between the dashes becomes the border.
