doug robbins - mvp office apps & services (word) say:

If the line spacing of the Body Text style has not previously been modified, then a change in the line spacing of the Normal style should be applied automatically to text that is formatted with the Body Text style.

If however, the line spacing in the Body Text style had previously been changed from Single to 1.5 lines, and then the line spacing in the Normal Style was changed from Single to Double, that change will no longer be reflected in the definition of the Body Text style or in text to which that style has been applied.

If however, you modify the formatting of the Body Text style itself, then such modification should automatically be applied to text that is formatted with that style UNLESS that text has been modified by the use of direct formatting.  If that has been done, you can get such text to be formatted with the Body Text style as modified by selecting it and then pressing CTRL+Q, which has the effect of removing any direct formatting so that the text returns to the format defined in the style.

The basics

Using Excel to perform a mail merge requires:

  • A range that contains the data to be merged. In the example, this is on the sheet named Data.
  • A range that contains the text. In the example, this is on the sheet named Form.
  • A cell that contains the row number of the "current record." In this example, that cell is C3.

...long time insiders remember Microsoft put builds on hold for new insiders on hold for a short time shortly before major updates before a public release. This occurred when the original Windows 10 was released back on July 15 2015...

I believe that 2015 "hold" for new Insiders was simply to do with not being able to activate an Insider Build, freshly installed on a new device that had never been on Insider before. It may also have been briefly not possible to upgrade from Windows 7 or 8.1 to Windows 10 Insider Preview (prior to build 10240 release) without install media.

Prior to that, most Insider Builds could be activated by Product Key, even on devices which had never been licensed for Windows 7 or 8. Once they switched to "Digital Entitlement" no one could activate Windows 10 Insider Preview on a new device.  There was a lot of complaining, as some Insiders lost their Digital Entitlement, for reasons unknown, when they did a clean install.

Some users eventually discovered that you could use an old 10074 Insider Pro Product Key to activate Windows 10 (if you used that Pro Key on build 10130 or earlier) and thereby gain Digital Entitlement/License for Windows 10 Pro on an otherwise unlicensed device. Apparently, this actually worked all the way through to August 2016 (if activated on some very old expired builds, with the device's date rolled back) even though those old builds had expired almost a year earlier.

So technically, I think that it was never impossible to join the Insider Program (unless Microsoft had at some point disabled the sign-up website?) as I think you could join right from a newly installed Windows 10 device. Plus the device did not need to have Windows 10 activated to join.
