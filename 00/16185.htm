 do not want to create other problems by deleting a previous Microsoft Windows 10 update

A problem is that now that we have cumulative updates you don't just lose the functionality changes you also lose the security patches and they document exploits that are already in use, so removing the cumulative update leaves you vulnerable to all the exploits and even more vulnerable because now they are more widely known so more criminals may try using them to go after the people who are avoiding the patches because they don't like the functionality changes that came packaged with them.  This problem of separation of functionality changes (e.g. hotfixes, etc.) and security patches used to be handled by the GDR/LDR paradigm but the problem with that is eventually MS needed to create and manage "rollups" and "service packs", so now we just get the equivalent of those every month.
