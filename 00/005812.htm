rich007a say:

Three simple macros will do this for you.

In Word, press Alt+F11 to open the VBE.  In the left pane, click on Normal, then do Insert > Module.

Paste the code below on the right.

Sub CommentTense()
    Selection.Comments.Add Range:=Selection.Range, Text:="You need to change the tense here."
End Sub

Sub CommentWordOrder()
    Selection.Comments.Add Range:=Selection.Range, Text:="You need to reconsider your word order here."
End Sub

Sub CommentReadTheQuestion()
    Selection.Comments.Add Range:=Selection.Range, Text:="You have not answered the question here. Read the question carefully."
End Sub

In the VBE, click the Save button to save the changes to your Normal template.

Close the VBE to return to Word.

In Word, right-click on the ribbon at the top > Customize Quick Access Toolbar

Choose commands from: Macros

Add a button for each of the three macros to your QAT.  OK.

Now you can click either of those buttons to insert the required comment at the current selection.

Cheers
Rich

graham mayor say:

Embroidering Rich's suggestion, especially useful if you have a larger selection of standard comments, I would suggest the use of a userform with a couple of command buttons and a list box. Call the userform frmComment - you can leave the rest of the controls with their default names.

Create a macro to call the userform

Option Explicit
Sub MyComment()
Dim oFrm As New frmComment
    If Documents.Count = 0 Then
        MsgBox "Open a document first!"
        GoTo lbl_Exit
    End If
    With oFrm
        With .ListBox1
            .AddItem "You need to change the tense here."
            .AddItem "You need to reconsider your word order here."
            .AddItem "You have not answered the question here. Read the question carefully."
            'etc - insert as many comments as you wish.
            .ListIndex = -1
        End With
        .Show
        If .Tag = 1 Then
            Selection.Comments.Add Range:=Selection.Range, Text:=.ListBox1.Text
        End If
    End With
lbl_Exit:
    Unload oFrm
    Exit Sub
End Sub

and in the userform itself insert the following code:

Option Explicit
Private Sub CommandButton1_Click()
    With Me
        If .ListBox1.ListIndex = -1 Then
            MsgBox "Select a comment!"
            .ListBox1.SetFocus
            Exit Sub
        End If
        .Tag = 1
        .Hide
    End With
End Sub

Private Sub CommandButton2_Click()
    Me.Hide
    Me.Tag = 0
End Sub

Private Sub UserForm_QueryClose(Cancel As Integer, CloseMode As Integer)
    If Not CloseMode = 1 Then
        CommandButton2_Click
        Cancel = True
    End If
End Sub

The result is as follows. Select an item and click CommandButton1 labelled "Insert Comment'
