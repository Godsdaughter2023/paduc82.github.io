The problem usually arises because a backup has written to the System Reserved or EFI System (on UEFI Systems - same error) clogging it.  If the files are not examined to remove the offending program, it will do so again and can cause even more serious problems.

When this error pertains to EFI System it is not as easy since errant files must be deleted using a boot disk while the OS is inert, and the hidden MSR adjacent partition will block extending EFI System part until it is deleted.  So it's best there too to clean them out of intruding files.
