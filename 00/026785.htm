keratl say:

Can you tell us a little more about your workbook, and what you need to accomplish?

I'm not sure there is a suitable mouseover (no-click) event, but what about a macro that allows you to click an image to enlarge it for a few seconds, then it returns to normal size?

I'd start by writing a macro that cycles all images on the sheet and apply the same macro to all of them;

something like (aircode)

ActiveSheet.Shapes.Range(Array("Picture 1")).OnAction = "Macro1"

so they all have the same macro (Sub Macro1) assigned. each picture should have a unique name (if you aren't positive of this, you could also loop the shapes names into a dictionary and check for duplicates, and throw a warning)

Then in Macro1, grab the picture's name with something like

TargetPicture= Selection

(one of the few times I'd use selection, and other forum members may suggest a better approach)

Then

TargetPictureOldheight= TargetPicture.height

TargetPictureOldwidth= TargetPicture.width

Then change just the dimensions, not location

TargetPicture.height=TargetPicture.Height * 2

 TargetPicture.width=TargetPicture.width * 2

put in a few seconds of a delay loop (I'd suggest using an approach that stops Excel's other processes, so the user doesn't do something unexpected that might break the code while it is running, like deleting the picture before it gets resized back to normal)

Then reset back to normal:

TargetPicture.height=TargetPictureOldheight

 TargetPicture.width=TargetPictureOldwidth

Note that if you click a picture that is too far to the right or bottom of the screen, I suspect it will just expand offscreen. You could also use the top and left properties to make sure it is always fully visible, but that would require more complicated code.

Hope that is enough to get you started!
