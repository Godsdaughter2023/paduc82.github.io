I think there's a bit of a misconception here.

There's absolutely no point in backing up program files the way you mean, because that won't backup the Registry entries for each program that the program depends on to run. Those entries are only added to the Registry when the program is installed.

The only way to be able to restore working programs is to use a System Imaging application.

Otherwise if, for example you re-format he HDD, you have to re-install your programs in order to add the relevant entries to the Registry.
