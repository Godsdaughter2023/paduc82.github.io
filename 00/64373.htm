How do I prevent unintended folder deletions from the navigation pane - Microsoft Community

sbrumbyd say:

How do I prevent unintended folder deletions from the navigation pane (focus on two places at one time}?

When using  Windows 7 Explorer for file management, the blue highlight indicating system focus may appear in both the Navigation pane and the Details pane. If this happens and the user wants to delete the item with focus in the Details pane, the system will delete that object but will also delete the parent folder showing focus in the Navigation pane. :Regardless of the setting for the Delete Confirmation message, the user will not have the opportunity to confirm or cancel the deletion. If the folder in the Navigation pane contains many objects, the 'Preparing to delete' process may take a long time and provide a chance to cancel if the user suspects a  problem because the process is taking a long time.

In summary, there are two problems:

1. No delete confirmation message is provided for folder deletions from the navigation pane.

2. Focus locus shows in the Navigation pane and the Details pane at the same time but is really only in the Navigation pane.

Some third-party Explorer replacements do not allow deletions from the Navigation pane. I did not experience these problems in earlier Windows Explorer versions, so this is another example of something being fixed that was not broken ….

lemp say:
The reason that your network adapter has been assigned an IP address of the form 169.254.x.y is that it is set to receive its IP address automatically (the default, and generally a good idea) but it is unable to communicate with the DHCP server that is supposed to supply that IP address. That essentially means that there is no communication between your computer and your router.

The main reason for the lack of such communication is a wireless network connection where the wireless security (usually the encryption password) has not been correctly configured.  This rarely happens with a wired connection, but if that is the case, there usually is a hardware problem somewhere.

Unfortunately, your use of a registry cleaner may have rendered your problem not easily fixable. Does the registry cleaner you used have an "undo" function?  If so, please undo all changes that it made.

6.  When you have finished composing the e‑mail message, click Send.

Notes
• It's a good idea to optimize your pictures for e‑mail by making them smaller before you send them because very large files can be stopped by some e‑mail systems. Many e‑mail servers impose a limit of 1–2 megabytes (MB) per message. Optimizing your pictures for e‑mail does not affect the size or quality of the original pictures on your computer; it simply resizes the e‑mail versions.
 
• You can send videos using this same technique, but they will not be resized. Video files are generally large, so it's important to be aware of the 1–2 MB size limit.
 
• If you prefer, you can drag picture files directly into an open e‑mail message. If you use this method, though, you will not have the option to automatically make the picture files smaller.
