This should get you started (Note, this function does not lock unbound controls)...

***** Copy in a module window but do not name module the same as the Function...

Public Function fncLockUnlockControls(frm As Form, LockIt As Boolean)
'Lock or unlock all data-bound controls on form,
'depending on the value of <LockIt>: True = lock; False = unlock.
'Dirk Goldgar 2005

On Error GoTo Err_fncLockUnlockControls
Const conERR_NO_PROPERTY = 438

Dim ctl As Control

For Each ctl In frm.Controls
With ctl
If Left(.ControlSource & "=", 1) <> "=" Then
.Locked = LockIt
.Enabled = True
End If
End With
Skip_Control: 'come here from error if no .ControlSource property
Next ctl

Exit_fncLockUnlockControls:
Exit Function

Err_fncLockUnlockControls:
If Err.Number = conERR_NO_PROPERTY Then
Resume Skip_Control
Else
MsgBox "Error " & Err.Number & ": " & Err.Description
Resume Exit_fncLockUnlockControls
End If

End Function

a. user say:

It is located in the hidden folder
C:\Users\<your windows user name>\AppData\Local\lxss\rootfs
But you should not try to copy data in windows to this location as it will not be seen in linux.
You should be in the bash shell and mount c: windows drive and copy it in linux.

Example if you have a file c:\users\<your windows name>\Documents\sample.txt

Start bash and at the bash prompt linuxname@PCNAME:~$ type
cd /mnt/c/users/<your windows user name>/Documents
ls (should see sample.txt)
cp sample.txt /home/<your linux username>
cd /home/<your linux username>
ls (should see sample.txt)
vi sample.txt
