rob koch say:

There's nothing truly different when talking about Microsoft Windows Updates between the client systems that consumers use or the servers used in business, since the base updates for most of these are almost exactly the same.

For this reason this question is as valid here as any regarding the recent ransomware attacks like WannaCry, since they leverage precisely the same vulnerabilities found in the similar Windows platforms from the same era.

However, the question you should be asking isn't what patches protect from WannaCry or any other specific malware, since that's not how updates work.  Instead, you need to determine which updates are available to fix any unpatched Windows vulnerabilities for your specific Windows versions, regardless of whether any current malware may be attempting to leverage these specific vulnerability flaws or not.

As already mentioned, this means you should always install any critical security updates provided by Microsoft as soon as they re made available, since in most cases there will be at least some malware attacking these vulnerabilities within days of their being published.

As of the June 13th "Black Tuesday" update cycle, there were several new vulnerabilities patched related to the same SMBv1 and also SMBv2 that could be leveraged for use in a worm.  Though I haven't confirmed these were used in the more recent attack, they easily could be, so these should of course be installed as soon as possible.

The following short Microsoft security team blog article discusses the June 2017 update release, including the fact that some of these have again been provided for certain outdated platforms including Windows XP, Vista and Server 2003 [R2].

Microsoft releases additional updates for older platforms to protect against potential nation-state activity

For the currently supported Windows 7, 8.1 or 10 systems, these can all of course be installed using the normal Windows Update process and so likely have been already for most systems set to automatically update.

However, as with the earlier patches provided after the original WannaCry attack, those for any outdated versions of Windows must be downloaded and installed manually.  See the section titled Further Resources at the end of the article above for links to the knowledgebase article providing guidance for these.

Rob
