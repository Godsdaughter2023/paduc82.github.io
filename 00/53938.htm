special k_ say:

I had the same problem when I deleted the C:\ProgramData\Microsoft\MapData folder. You can't simply copy this folder from a file backup or another computer, because it has a special security permission user called MapsBroker. It is impossible to add this user from the Folder Security Edit function, as this special user isn't available from the list. After much frustration I found a technique which works.

First, you'll need another Windows 10 computer. Use the Windows Backup utility (Search/Backup/Windows 7 backup). Backup the C:\ProgramData\Microsoft\MapData folder to a removable media, such as an external drive or a USB flash drive. Make sure you have unchecked the box that says "Include a system image".

After the backup completes, take the external medium where you made the backup and attach it to the computer where Maps is broken. Run the Windows 7 backup utility again, navigate to where your backup is, and select Restore. Depending on the location you designated for restore, the MapData folder will either be under C:\ or C:\ProgramData\Microsoft\C\. Cut and paste the MapData folder so that it is under C:\ProgramData\Microsoft\MapData. Verify that the folder permissions include the MapsBroker user. At this point you can launch Maps and it will work correctly.

The Windows 7 Backup/Restore tool is not the easiest to use. You can probably use some other backup utility to accomplish the MapData backup/restore as long as it backs up and restores not only the files, but more importantly the file permissions on the MapData folder.

Select Settings, then Change PC Settings.

Select Devices.

You might see a note after each device saying that it had not been updated because the computer is on a metered connection.

Set the item Download Over Metered Connection to ON.

Wireless Security

Login Information

  • Gateway – 192.168.1.1
  • Username – admin
  • Password – admin
  How To Find Wireless Passphrase (WPA/WEP)
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Wireless from the menu on the left.
  5. Select Security from the sub menu.
  6. The passphrase is on this page labeled PRE-Shared Key.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Wireless from the menu on the left.
  5. Select Security from the sub menu.
  6. Select desired security type from the Encryption drop down menu. (To turn off the security select None)
  7. Select Apply Changes.
