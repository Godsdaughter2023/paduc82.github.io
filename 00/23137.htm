macropod say:

FWIW, when you specify, say, bold as a find attribute, by definition whatever occurs immediately before and after the found range is not bold. Granted, that doesn't tell you what other formatting might be there, but at least you know it's not bold.

So, if all you're concerned to do is to insert a paragraph break before the bold text, all it takes is:

Find = (Font: bold)

Replace = ^p^&

No macros required. It's only when you need to know something about the attributes of the non-bold text that you might need a macro to do it all in one step. Even then, there are manual methods. Suppose you wanted to insert paragraph breaks only where the bold text follows some italicized text. To do that manually that you might use:

Find = (Font: bold)

Replace = §^&

Find = (Font: italic)

Replace = ^&§

Find = §§ (Format: no formatting)

Replace = ^p 

Find = § (Format: no formatting)

Replace = (nothing)

where the § symbol is ASCII 167 (or any other character not otherwise used in the document).

doug robbins - mvp office apps & services (word) say:

You cannot use that sort of mix of formatting in the Find utility.

If the start of the bold text is always the start of a new sentence, then you could use a wild card search in VBA for

. [A-Z]{1}

and then used code to determine if the first letter after the period and the space is formatted as bold and if so, insert a carriage return in front of it.  The following code would do that:

Dim rng As range
Selection.HomeKey wdStory
Selection.Find.ClearFormatting
With Selection.Find
    Do While .Execute(FindText:=". [A-Z]{1}", Forward:=True, _
    MatchWildcards:=True, Wrap:=wdFindStop) = True
        Set rng = Selection.range
        rng.Start = rng.Start + 2
        If rng.Font.Bold Then
            rng.InsertBefore vbCr
        End If
        Selection.Collapse wdCollapseEnd
    Loop
End With

andreas killer say:
I Want to do Same in EXCEL , What i want is that whenever i make a Entry of Credit in A ledger it should ask whom i want to debit that amount 
That means you have to write a macro that spreads the credit. Below is an example, copy the code into the code module of a sheet.

The code ensures that the sum in column A and B is always equal to the sum in C, D and E.

If you have no experience with macros, here is a online tutorial for beginners:
http://www.wiseowl.co.uk/blog/s161/online-excel-vba-training.htm

Andreas.

Option Explicit
Option Compare Text

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim SumIn As Double, SumOut As Double, Diff As Double
  Dim i As Integer
  Dim Cols As String
  Dim R As Range

  'We take care on column A..E
  If Target.Column > 5 Then Exit Sub

  'Get the amounts
  With WorksheetFunction
    For i = 1 To 2
      SumIn = SumIn + .Sum(Columns(i))
    Next
    For i = 3 To 5
      SumOut = SumOut + .Sum(Columns(i))
    Next
  End With
  'Did we have a difference?
  If SumIn = SumOut Then Exit Sub
  Cols = InputBox("Debit account of " & SumIn - SumOut & " to?" & _
    vbCrLf & "(Enter up to 3 column names of C,D,E):")
  'Abort?
  If Cols = "" Then Exit Sub
  'Remove bad input chars
  For i = 1 To Len(Cols)
    Select Case Mid(Cols, 1, 1)
      Case "C", "D", "E"
        'Okay
      Case Else
        Mid(Cols, 1, 1) = vbNullChar
    End Select
  Next
  Cols = Replace(Cols, vbNullChar, "")
  'Spread the difference
  If Len(Cols) > 0 Then
    'Events off, otherwise we call ourself
    Application.EnableEvents = False
    Diff = (SumIn - SumOut) / Len(Cols)
    For i = 1 To Len(Cols)
      Range(Mid(Cols, i, 1) & Rows.Count).End(xlUp).Offset(1) = Diff
    Next
    Application.EnableEvents = True
  End If
End Sub
