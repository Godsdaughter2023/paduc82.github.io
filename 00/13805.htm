scenario is as follows:

1) convert a csv file to xlsx file (append data)

2) both files (csv, xlsx) are closed / also, data in xlsx file is in sheet1 tab

3) write the vba macro in a new file (in a module)  and save as xlsm (macros enabled)

[edit..]

Sub Convert_ONE_CSV_to_XLSX()
'append data

'Aug 25, 2015

Const sD As String = ","  '<< select comma or semicolon

Dim sPathCSV As String, sPathXLSX As String
sPathCSV = "C:\Users\Username\folder CSV\" '<< CSV file path, change as needed
sPathXLSX = "C:\Users\Username\folder XLSX\" '<< XLSX file path, change

Dim fNameCSV As String, fNameXLSX As String
fNameCSV = "fileCSV.csv"  '<< CSV filename, change
fNameXLSX = "fileXLSX.xlsx"  '<< XLSX filename, change
       
Dim wbCSV As Workbook, wbXLSX As Workbook
Application.ScreenUpdating = False
Set wbCSV = Workbooks.Open(sPathCSV & fNameCSV)
Set wbXLSX = Workbooks.Open(sPathXLSX & fNameXLSX)

Dim r1 As Long, r2 As Long
r1 = wbCSV.Sheets(1).Cells(Rows.Count, "A").End(xlUp).Row
r2 = wbXLSX.Sheets(1).Cells(Rows.Count, "A").End(xlUp).Row + 1

wbCSV.Sheets(1).UsedRange.Copy wbXLSX.Sheets(1).Cells(r2, "A")

Dim v As Variant
Dim x As Long, i As Long
For i = r2 To r2 + r1
v = Split(wbXLSX.Sheets(1).Cells(i, 1), sD)
For x = 0 To UBound(v)
wbXLSX.Sheets(1).Cells(i, x + 1) = v(x)
Next x
Next i

wbCSV.Close False
wbXLSX.Save
wbXLSX.Close False
Application.ScreenUpdating = True
MsgBox "done"
End Sub

You may create an email rule on your account. Follow the steps below on how to create specific email rule to block a Sender by his/her name and check if this helps:

Blocking emails using keywords that might appear in the sender's name:

  1. Open your inbox.
  2. At the upper right part of the page, click on the gear icon beside your Profile name then choose More mail settings.
  3. Under Customizing Outlook, click on Rules for sorting new messages.
  4. Click New.
  5. Under Step 1; on the first box, choose Sender's name; on the second box, choose contains; then on the third box, type in the keyword that you want to block.
  6. Under Step 2, choose the radio button Delete these messages—don't even put them in my Deleted folder.
  7. Click Save.
