greginmich say:

First of all, it is possible to turn off Windows Defender’s Automatic Remediation feature, and regain full control of the actions that Defender takes when a threat is detected – but this is something that I would only recommend for experienced Windows Defender users:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning-windows_10/windows-10-pro-defender-stop-deleting-threats/418e68e0-4753-4de1-918c-67a8d1e3af5a

 

In any case, permanently disabling Windows Defender is never the solution for issues like this, and shouldn’t be recommended as such. All Internet-connected PCs require continuous real-time AV/Antimalware protection in order to keep the user (and the OS) safe from online threats – and that’s why Windows Defender is installed and enabled by default in Windows 10. If you’re not happy with Windows Defender for any reason whatsoever, then all you have to do is install another free AV app – and Windows Defender will automatically be disabled by the OS as soon as the new AV app registers itself with the system as an antivirus provider. Then if you later uninstall the third-party AV app, Windows Defender will automatically be enabled again so that no lapse in your AV/Antimalware protection ever occurs.

 

So if you’re not happy with Defender for any reason at all, then just follow the steps in this thread to replace it with another free AV app:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-to-stop-antimalware-service-executable-from/a3a57d31-4687-43c0-b274-261da7d89245

 

Once you’ve installed another AV app, you’ll have the option to turn on Limited Periodic Scanning – and that will allow Windows Defender to run right alongside your new AV app with its real-time protection turned off – so that it doesn’t interfere with the new app. Limited Periodic Scanning runs an automatic Defender Quick Scan every now and then; and also allows you to run any other type of on-demand scan. And it actually enables most of Windows Defender's other features, with the notable exception of Controlled Folder Access. You can’t enable Controlled Folder Access when Defender is running in its Limited Periodic Scanning mode:

https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/limited-periodic-scanning-windows-defender-antivirus

 

Here’s another great compatibility feature that’s built into Windows 10:

 

Automatic disabled mode is enabled so that if the protection offered by a third-party antivirus product goes out of date, is not updated, or stops providing real-time protection from viruses, malware, and other threats, Windows Defender AV will automatically enable itself to ensure antivirus protection is maintained on the endpoint.

https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-compatibility

 

When this feature is working correctly, Windows Defender will automatically be activated as soon as you disable the real-time protection of your third-party AV app; and then it will be automatically disabled again as soon as you turn the third-party app’s real-time protection back on. So the lapse in protection will be on the order of seconds.

 

For more information, see this thread:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning-windows_10/how-do-i-turn-windows-defender-off/944322a3-9c1e-470b-875a-844677f0af96
