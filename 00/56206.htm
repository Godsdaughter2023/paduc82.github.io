david witteried say:

Same problem for me. However, I did find the file at "C:\Windows\SysWOW64\comctl32.ocx". I changed my command prompt to the SysWOW64 folder and reran the suggested command and it worked.

  1. Your command prompt probably opens in "C:\Windows\System32", mine did.
  2. From your command prompt type CD.. and press the Enter Key, this will move you to C:\Windows (i.e. up one level in the directory structure).
  3. Then type CD SysWOW64 and press the Enter Key. This moves you into the SysWOW64 folder. You command prompt should say C:\Windows\SysWOW64> if you got the commands right.
  4. Now enter the command regsvr32 comctl32.ocx and you should get confirmation that the file was registered.

David
