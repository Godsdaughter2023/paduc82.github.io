bossdweeb say:

What brand and model graphics card did you buy ?

I see no fans in that case. You will need to add as many fans as possible or the graphics card is likely to run hot

(or leave the side panel off the case to help compensate for poor air flow). The heat from the graphics card will also reduce the

cooling capacity of the cpu cooler if not case fans are added.

The fan on the power supply is not intended to replace the case exhaust fan, and the additional heat created by the graphics card

and extra load on the psu could cause the psu to overheat if that fan draws hot air created by the graphics card and cpu over the

psu components.This is of less an issue if the graphics card you bought exhausts the heat out the back, but even then the card will

create more heat inside the case.

With the exception of low end graphics cards, modern graphics cards require a PCIe power connector (black), or two.

Some are six pin and some are eight pin, and depending on the graphics card there may be two power inputs on the card.

Your power supply is a generic 350W psu, with only 216W (18Amps) total on two +12v rails, so unless you have bought a low end

graphics card it may not be able to provide enough current for the graphics card under load.

Yes, you already have the answer that is you can perform clean install Windows 10 using Media Creation tool. Since Windows 10 activation on the concept of digital entitlement so you will not face the problem which you faced while installing Windows 8.1 via installer on a Windows 8 machine. Windows 8.1 product keys are different from Windows 8 product keys. You can’t enter a Windows 8 product key into the Windows 8.1 installer, just as you can’t enter a Windows 7 product key into the Windows 8 installer. You also can’t install the original version of Windows 8 with a Windows 8.1 product key.

The Windows 8.1 installer refuses to accept the Windows 8 product key and won’t allow us to install Windows 8.1 with it. However, Windows 8.1 will accept the Windows 8 product key if you enter it on the desktop after installing Windows 8.1
