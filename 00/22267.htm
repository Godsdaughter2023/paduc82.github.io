How to setup reminders in Access 2013? - Microsoft Community

jcombs30 say:

How to setup reminders in Access 2013?

Hello,

So I am looking to track progress of "projects" in Access until they're finished.  I don't know if setting up reminders with some sort of timelines would work, or having some sort of tracker in Access that shows me where at in the que the project is at any given time.  

From the time these projects are received in "received in date", we have 60 days to complete all specified tasks and have the completion date entered in. It would be nice to have something remind any person who opens the database that project x, y, and z are at 30 days and then past 60 days if the completion date has not been entered in.  If the completion date has been entered then the project will not pop up when opening the db.

Any help would be much appreciated!

Thank you.

scottgem say:
There are two factors here. The first factor is determine the time lines and the second factor is reminding the users.

The first factor is handled with a query. Something like:
SELECT ProjectID, etc. FROM tblProjects WHERE CompleteDate is Null;
You can expand on that query by adding a column that shows elapsed time. something like:
Elapsed: DateDiff("d",ReceivedDate,Date())
Set the Criteria for that column to >=30 to get only those that are 30 or more days elapsed.
The next thing I would do is build a report based on that query. Use conditional formatting to highlight projects that have an elapsed value of more than 60 days.

The second factor would be solved by running that report when the app opens. This can be done with an AutoExec macro (just name the macro Autoexec and use it to open the report. Or, if you have a form that opens with the database, you can use the On Load event of the form to run this line of code:
DoCmd.OpenReport "reportname", acPreview

So whenever the app opens a report pops open that lists all projects more than 30 days since receipt and highlights those past 60 days.

Managing Citations:

1. To manage your citations, click on the Manage button.

2. In the ribbon and click the + button in the lower left-hand corner of the window.  This will bring up the Create New Source dialog box. Fill all the relevant information.

3. To edit the page range of the citation, click on the arrow to the right of the citation, choose edit this Citation and enter the page number.
