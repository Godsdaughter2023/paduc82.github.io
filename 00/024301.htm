If you're making a new "incoming message" then you're not modifying the existing message. If you want to modify parts of a message then no need to modify the source. Just call the parts of the mail message such as "subject", "content", etc.

tell application "Microsoft Outlook"

set newContent to "This is a test." & return & return & oldContent

set content of theMessage to newContent

end tell
