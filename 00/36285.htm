How to uninstall the preinstalled apps in Windows 10? - Microsoft Community

sccabrian say:

How to uninstall the preinstalled apps in Windows 10?

Original title: Apps don't fully uninstall?

There are a few preinstalled apps that I uninstalled when I initially got my surface; Sports, News, 3D Builder, etc. Every few days though, they reinstall themselves. After uninstalling them, they no longer appear in the "apps" section of the settings menu, but if I search for them in the store, it says they are already installed. Does anyone else have this issue? It's not a real "problem" per se, but it is definitely a bit irritating to have the apps reinstall themselves every now and then.

a. user say:

Hi,

Welcome to Microsoft Community.

Sorry to know that the apps gets reinstalled after uninstall on Windows 10. I will certainly help you.

I suggest you to run the Powershell commands to uninstall the apps and check if it resolves the issue.

1. Press Windows Key + Q, and in the search box, type Powershell.

2. From results, right click on Windows PowerShell and select Run as administrator option.

3. Type Get-AppxPackage and press Enter.

(This command will output a long list of all installed apps containing lots of information about each app).

4. Type Get-AppxPackage | Select Name, PackageFullName and press Enter.

(This command will show only the name of the app and its PackageFullName information).

5. Select the PackageFullName information of the desired app and copy it.

6. Type Get-AppxPackage PackageFullName | Remove-AppxPackage and press Enter.

(This command is to remove the app from the system, replace PackageFullName with the copied information with Step 5).

For example, if you want to remove 3DBuilder app, the command to remove it will be as following:

Get-AppxPackage *3dbuilder* | Remove-AppxPackage

OR

Get-AppxPackage *3d* | Remove-AppxPackage

7. Wait till completion of the command, and check if the app is uninstalled, if not restart the system and check.

Note:

If you want to uninstall the app from all user accounts, use the following command format:

Get-AppxPackage -allusers PackageFullName | Remove-AppxPackage

If you want to remove the app from any particular user account, use following command:

Get-AppxPackage -user username PackageFullName | Remove-AppxPackage

(Replace username in the above command with the correct user account name).

Hope this helps. If the issue remains unresolved, please get back to us and we would be happy to help.

Thank you.

You are absolutely correct.  Windows XP users that upgrade online do not have the option to create media through Upgrade Assistant.  XP simply didn't ever have the capability. 

 

Here are your options, depending on why you need media.   

  1. If you are interested in a "clean" install, you can select Keep Nothing on the migration choice screen, and then you can just use the Install Now option on the installation choice screen.  This is a clean install of Windows 8.

If you want to reformat your hard drive as part of your install, follow #2 above, but please don't reformat your drive BEFORE you start setup.  You'll want to start in XP, reboot to the media, and then choose "custom install" to enable the formatting through setup.  We have content on that as well: http://windows.microsoft.com/en-US/windows-8/clean-install for more detail.

Win 8.1 & Microsoft Cloud - Microsoft Community

bobselff say:

Win 8.1 & Microsoft Cloud

Good Morning,

         Could someone please explain to me what the "Cloud" is in Win 8.1 and what it is used for? Can I restore from it ? Where exactly is my data stored? Who has access to my data? I currently backup to my Sentinel Servers, is this no longer necessary?

Thanks,

Bob

cmdrkeene say:

Windows 8.1 has SkyDrive built in.  Personal online file storage, just like DropBox or Google Docs, but the Microsoft version.

They give you 15 GB of free space, and you can buy more if you want.  The largest single file you can store online there is 2 GB.

It also synchronizes your settings like your favorite colors and your internet favorite bookmarks.  If you get a new computer, all those settings will be restored upon it when you login.  And if you have more than one Windows 8.1 computer, the settings will stay synchronized.

It is not a backup or archival service: it's more like an actual drive.  If you delete a file from your computer from the OneDrive folder, it will be deleted from the cloud, and all other computers too.  In that way, you can see it's not like a real backup.

Learn more here:  What is Microsoft Account?
