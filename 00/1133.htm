julie fka profjulie say: There is no File Explorer app for Windows Phone.

If you received some files via Bluetooth and can't find them in the Office hub (as is the case with PDF files), then the files were placed in temporary storage and will eventually be removed from the device.  You will never be able to access them.  The only way to receive and work with PDF files is by downloading them from SkyDrive or some other online storage service or by receiving them via email.  When you open them up, they are stored in the PDF app that was used to open them.  So when you want to open them again, you can select the PDF app (from your programs list) and then you should find the PDF files there.  If all you do is download a PDF file and never open it, the file doesn't get stored in a place that can be accessed at a later date.
