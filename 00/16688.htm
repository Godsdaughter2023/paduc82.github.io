There is no settings tab in Windows Defender on Windows 10. Settings for Windows Defender are accessed via the gear in the upper right hand corner.

However, there is no setting for the scheduled scan, either. That's handled as part of maintenance and is coded to be a quick scan as a scheduled task.

You can create a new scheduled task yourself in the task scheduler to perform a full scheduled scan.

------------------

Courtesy of GreginMich :

There’s no option for a scheduled Full Scan in the new Windows Defender. That’s “by design” – because the Windows Defender Scheduled Scan task is now run as a component of the Automatic Maintenance package – and Automatic Maintenance tasks are only allowed to run in limited time windows when the system is idle. There is a popular hack for adding a Full Scan by adding the Full Scan parameter to the command line for the Windows Defender Scheduled Scan task; but that hack inadvertently forces Automatic Maintenance to run a daily Full Scan, and that could easily prove to be an impossible chore given the limitations imposed on Automatic Maintenance. So the only legitimate alternatives would be running your Full Scans manually or running them with an independent scheduled task.

What works well for most users is just running an occasional manual Full Scan with Windows Defender in conjunction with occasional scans with some of the reputable third-party malware-removal tools. Truth be told, the malware-removal tools are probably more likely to find malware than a Full Scan with Windows Defender.

Nevertheless, it’s not all that difficult to schedule a Full Scan with the Create Basic Task Wizard, and I’ve provided some links in this thread that can walk you through the setup:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_start/windows-10-how-do-i-set-windows-defenders-scan/f399bb87-37c9-4a74-bb35-1d7e3d3c2436?page=2

As long as real time protection is enabled, a full scan on a frequent basis is usually overkill. A quick scan looks for malware in the most common locations. A full scan has to touch every file. Conditional Header - Microsoft Community

steve.weaver say:

Conditional Header

Is it possible to have a conditional header based on the value in cell B2? For example, if the value in cell B2 is:

 

02  then Carroll & Associates to appear in the header,

03  then Hydraulic Components,

05  then Casting Sales, etc.

 

Please share your ideas.

rich007a say:

You could do either:

Public Sub Workbook_BeforePrint(Cancel As Boolean)
' This will load the header for each sheet according to the value of A2 on each sheet.
'
    Dim i As Long
   
    For i = 1 To ThisWorkbook.Worksheets.Count
        With ThisWorkbook.Worksheets(i).PageSetup
            Select Case LCase(ThisWorkbook.Worksheets(i).Range("A2").Text)
                Case "02"
                    .LeftHeader = "H. G. Carroll & Associates"
                Case "03"
                    .LeftHeader = "International Hydraulic Components"
                Case "05"
                    .LeftHeader = "Casting Sales"
                ' continue with other cases
                Case Else
                    .LeftHeader = vbNullString
            End Select
        End With
    Next i
End Sub

 

 

or:

 

Public Sub Workbook_BeforePrint(Cancel As Boolean)
' This will load the header for each sheet according to the value of A2 on "Sheet1".
'
    Dim i As Long
    Dim strLHeader As String
   
    Select Case LCase(ThisWorkbook.Worksheets("Sheet1").Range("A2").Text)
        Case "02"
            strLHeader = "H. G. Carroll & Associates"
        Case "03"
            strLHeader = "International Hydraulic Components"
        Case "05"
            strLHeader = "Casting Sales"
        ' continue with other cases
        Case Else
            strLHeader = vbNullString
    End Select
   
    For i = 1 To ThisWorkbook.Worksheets.Count
        ThisWorkbook.Worksheets(i).PageSetup.LeftHeader = strLHeader
    Next i
End Sub

 

Hope that helps.
