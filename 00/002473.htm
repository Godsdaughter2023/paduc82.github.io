Anyway, you can use this function to count colored cells which I quickly put together for you

1. Make a backup of your workbook.
2. Open your workbook and ALT+F11
3. Locate your Workbook name in Project Explorer Window

=CountColors(A1:F8) 

Function CountColors(Rng As Range) As Long
    Dim Cell As Range
    Application.Volatile
    For Each Cell In Rng
        If Cell.Interior.ColorIndex > 0 Then
            CountColors = CountColors + 1
        End If
    Next Cell
End Function

I told Microsoft to correct or pull that article 3 years ago because it contains an error. You should never copy a PST file to the default folder because there is a good chance you will overwrite another PST file and corrupt the Outlook profile.

You will need to do what I told you using your last good copy of the PST file. Creating an Outlook profile is easy:http://support.microsoft.com/default.aspx?scid=kb;en-us;829918&Product=out2003. You should recreate your mail accounts from scratch in the new profile. But be aware that Outlook will create its own blank, new PST file. So to reuse your previous PST file, you must first open it in Outlook and set it as your default. Then restart Outlook and close the new empty PST file the new profile created.
