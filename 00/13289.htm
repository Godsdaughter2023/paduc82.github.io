bob jones aka: cybertaz say:
Just for background... Themes in the Windows version of Office are made up of 3 components:
  1. Theme Colors: A coordinated color scheme
  2. Theme Fonts: A coordinated set of fonts [which may be different for Theme Headings & Theme Body, or the same font may be used for both]
  3. Theme Effects: Appearance formatting for Charts, SmartArt, WordArt, etc.
In PPT -- unlike the other Office apps -- the Themes actually are included in the Design Template of the same name, such as Composite.

Of the 3, Theme Fonts most readily can be 'swapped out' or substituted because almost all Themes for Mac or PC use the same combinations. Even though the Theme name may be different the fonts used are the same.

If you find that you do need the complete Composite Theme, the easiest option is to create a new file using PPT 2010. Apply that Theme & save the file. Bring that file to your Mac to continue your work. You can also save that file's Theme using the command in the Themes gallery menu so you can reuse it. The colors & effects sometimes don't translate very well, but the majority of the specified fonts do.
