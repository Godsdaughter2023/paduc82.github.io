jllatham say:

Hopefully I've interpreted your excellent description properly:

Two sheets involved: [Master Listing] with entries from row 13 down through row 4715 in columns A and B.  From this list you must choose 46 entries at random and copy them, values and formats over to the second sheet [Serial Number Selections].

Now, I have assumed that you do this at one sitting rather than picking 5 one week, 5 more on some other week, etc until you have 46 chosen for the quarter.

If that's true, then I think you'll find this macro will do the job very well for you.  You'll notice there are a group ofConst values at the beginning of the Sub, and those are values you can change to adapt the code to any changing requirements for this process in the future.  I think you'll see the relevance of each of them to what you're doing at this time.

This process uses worksheet objects and range objects (references to cells) to get the job done without you having to do anything other than run the macro.  No more having to copy and paste from L# into the Edit --> Goto box.  I think we could have taken care of the .PasteSpecial with one line of code, but I played it safe and went with 2 lines.  Give it a try and let me know if I've helped or just confused you.

Sub Select46AtRandom()
  'change these sheet names if you need to
  Const mstrSheetName = "Master Listing"
  Const snsSheetName = "Serial Number Selections"
  Const firstColToCopy = "A"
  Const lastColToCopy = "B"
  Const firstDataRow = 13
  Const lastDataRow = 4715
  Const PickMax = 46

  Dim masterSheet As Worksheet
  Dim serialSheet As Worksheet
  Dim rowsUsed() As Long
  Dim LC As Integer ' to loop through rowsUsed()
  Dim alreadyUsedFlag As Boolean
  Dim dataToCopy As Range
  Dim nextToCopy As Long
  Dim whereToPaste As Range
  Dim PLC As Long ' pick loop counter

  'some preparation
  Set masterSheet = ThisWorkbook.Worksheets(mstrSheetName)
  Set serialSheet = ThisWorkbook.Worksheets(snsSheetName)
  ReDim rowsUsed(1 To PickMax)
  'help assure random series
  Randomize
  'to improve performance speed
  Application.ScreenUpdating = False
  'clear any old information on the [Serial Number Selections] sheet
  serialSheet.Cells.Clear
  For PLC = 1 To PickMax ' loop 46 times
    'choose a row number not yet used
    alreadyUsedFlag = False
    'this will pick a random number and make sure
    'that it hasn't been used already during this "pick 46" sequence
    Do While Not alreadyUsedFlag
      '                     same as RANDBETWEEN(13,4715)
      nextToCopy = Int((lastDataRow - firstDataRow + 1) * Rnd + firstDataRow)
      'see if this number has already been used
      For LC = 1 To PLC
        If rowsUsed(LC) = nextToCopy Then
          alreadyUsedFlag = True
          Exit For
        End If
      Next
      If Not alreadyUsedFlag Then
        rowsUsed(PLC) = nextToCopy
      End If
    Loop
    'get the data to be copied: cells A & B on the random number row
    Set dataToCopy = masterSheet.Range(firstColToCopy & nextToCopy & ":" & _
     lastColToCopy & nextToCopy)
    dataToCopy.Copy ' place onto the clipboard
    'copy the data over to the [Serial Number Selections] sheet
    'PLC will point to the row we need to paste it into
    Set whereToPaste = serialSheet.Range(firstColToCopy & PLC)
    whereToPaste.PasteSpecial Paste:=xlPasteValues, Operation:=xlNone, SkipBlanks:= _
        False, Transpose:=False
    whereToPaste.PasteSpecial Paste:=xlPasteFormats, Operation:=xlNone, SkipBlanks:= _
        False, Transpose:=False
  Next ' end of PLC loop
  'housekeeping and cleanup
  Set dataToCopy = Nothing
  Set whereToPaste = Nothing
  Set masterSheet = Nothing
  Set serialSheet = Nothing
  'inform user work is finished
  MsgBox PickMax & " Random entries have been selected and copied.", vbOKOnly, "Task Complete"
End Sub

Hello,

the error message tells you what's wrong. The range name refers to a two-dimensional range with two rows and ten columns. The series value can only be one row with many columns, or one column with many rows, or one cell but not a two-dimensional range.
