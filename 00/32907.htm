a. user say:

Hi Arvid,

Code 43 is means that one of the drivers controlling the device notified Windows that the device failed in some manner, in this case, your graphics card driver have failed. As a first troubleshooting step, we suggest that you run the Hardware and Device troubleshooter and see if it resolves the problem or give additional info about the problem. You may download and run the troubleshooter by clicking this link.

If the troubleshooter does not resolve the issue, you may attempt to update the driver through Windows Update. you may update your driver through Windows Update by following the steps below:

  1. Select the Start button, then select Settings > Update & security > Windows Update.
  2. From there, select Check for updates.
  3. If there are updates waiting to be installed, kindly do so and restart your device.
  4. After restarting, check Device Manager if it still shows the error code.

However, if Windows Update did not find an update for your device, you may update it by going to the device manufacturer's website, and download the latest driver from them, and follow their instructions on how to install the driver.

Note: Make sure that the device drivers that are being installed are compatible with your current Windows version and platform.

If the issue persists, feel free to post back to us.

There is no actual unlinking method available, the digital license linked to your Microsoft Account is actually an in addition to option, which makes it convenient to reactivate; especially when using the Activation Troubleshooter or transferring a retail license to another computer.

The license itself is registered to the hardware.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

 

You can always switch the Microsoft Account to a Local Account from within Start > Settings > Accounts > Your info
