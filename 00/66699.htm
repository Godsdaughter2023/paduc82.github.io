Here's how:

  1. In the search box on the taskbar, type command prompt, press and hold (or right-click) Command Prompt, and then select Run as administrator > Yes.

mailto: is a Windows protocol. When Windows sees this prefix followed by an email address, it goes looking in the registry for an installed program that can create a mail message to send to the address given. Gmail isn't a program - it's a service that you can access through your browser.[1] On modern computers, the only way to make mailto: work is to have a mail client program installed on the computer.

You now have Windows Live Mail 2012 installed and you have presumably set up your Hotmail account to send and receive mail. [1]    Once upon a time, the Google Toolbar included a function that trapped mailto: and diverted it to a web page where a mail message could be composed.   • Play the file in Media Player. While the file is playing, right-click the file in the library, and then click Properties. On the  File tab, look at the Video codec section.
  • Use a non-Microsoft codec identification tool. Several are available on the Internet.

Once you locate the FourCC code, searching for it should identify exactly what it is and (hopefully) be able to download and install.

ken blake say:
Although by default the task bar appears on the bottom of the screen, it can be on any of the sides you prefer it on. To move it from any side to another, simply click on an unused part of it and drag it where you want it. Be sure to click *within* it, not on the edge; clicking on the edge and dragging will resize it, not move it.

If it won't move, it may be locked. In that case, right click on it and uncheck "Lock the taskbar," then try again.

By the way, with today's wide-screen monitors, I prefer the task bar on the left or right side of the screen. I think that makes a better use of screen real estate than having it at the bottom.
