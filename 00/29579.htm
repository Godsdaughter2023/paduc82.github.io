You should not try to change permissions on a Junction. That My Documents junction does not contain anything and cannot be used like a normal folder.

It's only purpose is to redirect older, legacy applications to the real My Documents folder for compatibility purposes.

Changing permissions on a junction can cause some programs to quit working, prevent some legacy programs from installing properly, as well as other unexpected behaviors in the file system.

If you need to place something in My Documents, use the one without the shortcut arrow.

Regards,

Ronnie Vernon MVP – Windows Desktop Experience

There are two items called My Documents in the C:\Users\{your user name} folder:

  2. A hidden, non-user-accessible junction point file, with a shortcut arrow on its icon, that points to the actual My Documents folder.
