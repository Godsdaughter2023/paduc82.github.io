Just to clarify something - an Outlook Address Book is a specialized "view" of all the contact folders that have been set to be a member of the Outlook AB which will be a minimum of one to as many contact folders exist in the Outlook profile. Unless I'm mistaken, what you are referring to as "Address Books" are in fact contact folders.

Outlook will only have one "default" contact folder set for the profile which is stored in the "default data file" configured for the profile (to determine which that is go to File --> Account Settings --> Data Files tab). The default data file setting is separate and distinct from the email account set as the default for the profile.

So.............how many contact folders you have is based on a number of things

#1 - Type of email account. Each Outlook.com/Exchange account will have its own contact folder which can't be changed however there is no requirement to create contacts in those folders

#2 - If you've added/used any PST files from another Outlook setup that was configured as the default data file

#3 - If you've created any of your own contact folders

Other then what was mentioned above, you would only have one contact folder no matter how many POP3/Imap accounts configured in the profile with the first account being the one where the default contact/calendar folders would reside.

First, let me say that MS is renowned for using the same term for multiple things - kind of like almost naming anything related to email as "Outlook".

For the sake of simplicity, am going to use the terms as follows repeating my original reply

A) Outlook AddressBook - Outlook only has one and is comprised of "Contact Folders" (which may or may not contain contact items" that have been set to be a member of the Outlook AB

B) Contact Folder - contains contact items (including Contact Groups)

C) Exchange/Outlook.com accounts configured in Outlook will each have its own "contact folder" in the account folder tree in the Outlook Nav Pane.

D) Imap accounts - only the first account will have a contact folder in the IMAP folder tree and have the added words "This Computer Only". The ONLY way that you would have a contact folder in their respective folder tree is if one of the two conditions apply

#1 - The data file for the IMAP account was set as the default data file at some point (or it was configured as the first email account in the Outlook profile)
#2 - A contact folder was manually created

E) There will only be ONE default data file configured in an Outlook profile at any single point in time. That data file will have the "default contact folder" for the profile.

<IF> you're asking "how do I only end up with one contact folder in your profile?" Answer is "you don't". As stated, each Exch/Outlook.com account will have it's own contact folder but there is absolutely no requirement for any contacts to be created in that folder. You cannot alter the behavior of the creation of contact folders for Exchange/Outlook.com accounts OR for the first email account configured should it be either a POP3 or IMAP account.

As previously stated, there is only "default contact folder" for your profile regardless of the number of email accounts configured. There is no "one size fits all answer that can tell you which one that would be without knowing the sequence in which the email accounts were configured.

If you want to know which data file is set as the default data file which contains your "default contact/calendar" folders - go to Outlook menu -->
File --> Account Settings --> Data files tab
