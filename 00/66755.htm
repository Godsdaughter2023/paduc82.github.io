nakul kashyap say:

can i connect my laptop to two wifi networks at the same time? (in windows 8)

is it possible

a. user say:

Hi Nakul,

Windows always accesses the Internet using the network connection that defines a default gateway. In the presence of more than one default gateway (as in your case), it will use the one w/ the smallest metric. If they are the same metric, it will use the network connection w/ the highest priority.

Windows is only going to use one of the network connections for Internet access and ignore the other for those purposes. That’s just the way it works.

Hi,

 

1. What are the names of the updates (along with KB numbers) that you are trying to install?

2. Are you aware of any changes made to the computer, prior to the issue?

 

The definition of the error code you’ve received 0x80070641 can be found below:

 

ERROR_INSTALL_SERVICE_FAILURE - The Windows Installer Service could not be accessed.

This can occur if the Windows Installer is not correctly installed.

If you can successfully install applications that rely on the Windows Installer then the service itself is most likely functional, and was simply in use at the time of the update. Thus, you will want to make sure that the Windows Installer service is not in use prior to attempting updates.

 

Method 1:

 

Refer to the following link.

Troubleshoot problems with installing updates

http://windows.microsoft.com/en-us/windows-vista/Troubleshoot-problems-with-installing-updates

 

Method 2:

 

Stop the Windows Installer service and then try to install the update. The Windows Installer service will automatically be triggered to start when you are going to install the updates.

 

Step 1: Stop the Windows Installer Service.

 

a. Click Start, click All Programs, click Accessories, right-click Command Prompt, and select Run as Administrator.

 

b. At the Command Prompt window type the following, and press Enter:

 

net stop msiserver

 

Note: This will effectively stop the Windows Installer service. This service will load-on-demand so manually stopping it does not adversely affect the computer.

 

Step 2: Check for updates.

 

Launch Windows Update and install the available updates.

andre da costa say:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

trekdozer say:

If the OEM disk from Microsoft is the same version as the product key you already have - yes you can use the other install disk to reinstall Windows (example: both the disk and key need to be Windows 7 Home Premium).

 

Note 1: You can only use the OEM key on one computer, OEM can not be moved to another computer. You must use the key from your HP computer on that computer.


Note 2: You can not use an OEM recovery disk that came with a different computer to reinstall Windows - OEM recovery disks are only valid with the computer it came with.

If online activation fails:

"Automatically Activate When Online" then click OK/Next to finish the installation.

1. Begin by clicking the Start menu, then type: slui.exe 4

2. Press Enter on your Keyboard

3. Select your Country.

4. Select the Phone Activation option, call the number, do not select any options and wait for a person to pick up, then explain what happened.
