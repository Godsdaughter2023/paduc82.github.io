Thanks,
Curt
jim gordon mvp say:

Windows 7 is not PowerPoint. You could have had a number of different PowerPoint versions installed, so I can only give you a generalized answer.


Toolbars and menus are portable between the PC and the Mac if they were made using the Toolbars collection of VBA, and if the toolbar commands themselves are in the PowerPoint 2011 object model. Some commands were deprecated and are no longer available. Some commands are version-specific. Some commands are platform-specific. 


Ribbon customization is not available on the Mac. Even though the Ribbon is a toolbar, it is not part of the toolbar collection (a bit of laziness on Microsoft's part), so the Ribbon can't be customized on the Mac. So if your add-in does stuff to the Ribbon, it won't work on Mac.


Addins in .ppa and .ppam format .ppam format are cross-platform. However, there are Mac only features and Windows only features. If the add-in's author was not cognizant of these differences the add-in will fail. If the author was on the ball, the add-in will work.

This modification should do,

Sub mcrCopy2Sheet2Ovr()
    r = 1
    Sheets("Sheet1").Range("A10:Z10").Copy
    If Sheets("Sheet1").Range("A10") = Sheets("Sheet2").Range("A65536").End(xlUp) _
        Then r = 0
    Sheets("Sheet2").Range("A65536").End(xlUp).Offset(r, 0).PasteSpecial _
        Paste:=xlPasteValues
    Application.CutCopyMode = False
End Sub

Install instructions from a zip file to WM 6.1 device - Microsoft Community

brian strick say:

Install instructions from a zip file to WM 6.1 device

Where can I find instructions on how to take zip files and install them onto a WM 6.1 device?

johan van mierlo say:

Extract the zip file on your desktop.

 

You can get two possible file extensions.

 

1 a .exe file

 

this file you run on your desktop with your device connected to this computer. through either active sync or Windows Mobile Device Center. This will install the program on your device.

 

2 a .cab file is a self install file that needs to be placed on your device using copy and paste. Connect your device through either active sync or Windows Mobile Device Center. Use the explore function in these control centers to copy and paste the file onto your device.

Now disconnect the device and use the explorer on your device to find this .cab file. Select the file and the program will be installed onto your device.
