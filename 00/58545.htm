Let's try this a 2nd time... since the first time just deleted my post without posting it...

Log in to linksys

Go to status

Find the Default Gateway there and put it in your address bar

Once on the AT&T site (this is your modem), click on Advanced and then Connection Configuration

Select "Yes, use a public IP Address" and save the changes.

Power cycle your network properly (unplug modem and router for 30 seconds, plug modem in first and wait about 30 seconds before plugging the router in).

If you have properly port forwarded, you will have an open NAT.  

To make sure the modem is configured correctly, log back in to the linksys and go back to status.  Make sure the IP and Gateway start with something other than 192.168.*

2.Select Storage.

3.Select All Devices.

4.Select Gamer Profiles.

6.Select Delete.

2.Go to Settings and select System Settings.

3.Select Storage or Memory.
