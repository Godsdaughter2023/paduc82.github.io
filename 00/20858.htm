All of the vents and heatsinks are clean and properly connected with a silver thermal compound. The issue isn't the hardware.


Yes it is the hardware.  No properly designed PC, especially a laptop which should be using something like the Intel Speed-Step technologies to throttle processor performance and reduce power consumption, should ever overheat.

If the problems aren't resolveable by either cleaning the venting, proper heat-sinking or processor power configuration, then it's a design problem with that particular laptop.  Usually these issues only happen after the fact if someone replaces the processor themselves without the proper heat-sinking kits that are required with modern PC systems, laptop or otherwise.

 

There may be workarounds such as some third-party power control applications, but I've never required these for any major name laptop I've owned or supported.  All of my laptops have always throttled themselves using power controls so that even theough the processor can appear to be constantly running at 100%, it's actually reducing the processor speed to compensate for potential overheat.

Manual scans aren’t throttled; but with the default settings, you can run the Microsoft Antimalware Scheduled Scan (complete with CPU throttling) on demand, either directly in the Task Scheduler GUI, or by creating a desktop shortcut with this target (in Windows 7):

 

C:\Windows\System32\schtasks.exe /Run /I /TN "\Microsoft\Microsoft Antimalware\Microsoft Antimalware Scheduled Scan"

 

In the shortcut’s Properties, click on “Advanced” and check “Run as administrator”.
