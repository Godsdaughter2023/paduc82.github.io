robert aldwinckle on forums say:

 stuck between "Preparing Automatic Repair" and "Diagnosing Your PC" .

This is where a boot menu would be useful.  You could press F8 in the boot menu and interrupt the loop.  Alternatively, keystrokes which might interrupt the loop are Shift-F10 or F8 pressed as soon as possible after powering off and on.  Supposedly if you power off and on enough it will eventually allow you to get into the Advanced Boot Options dialog.  Apparently we need a real keyboard to make this happen though; e.g. the screen cover doesn't work and there can be no On-Screen keyboard in that state.
