All 32-bit client versions of Windows (not just XP/Vista/7) have a 4GB
address space (64-bit versions can use much more). That's the
theoretical upper limit beyond which you can not go.
But you can't use the entire  address space. Even though you have a
4GB address space, you can only use around 3.1GB of RAM. That's
because some of that space is used by hardware and is not available to
the operating system and applications. The amount you can use varies, depending on what hardware you have installed, but can
range from as little as 2GB to as much as 3.5GB. It's usually around
3.1GB.
Note that the hardware is using the address space, not the actual
RAM itself. If you have a greater amount of RAM, the rest of the RAM
goes unused because there is no address space to map it to.

Replacing the ^ key on my keyboard with the pound sign.

shinmila h say:

Hey 3no,

 

There is a way to display the pound sign (£). However remapping of keys can be done only via third party remapping tool. If you’re interested in getting the pound sign (£) to display in any application, here’s how.

 

 To display the pound sign (£) on a non UK keyboard you can use a simple shortcut which uses the number pad on the right of your keyboard.

This works in any writing program, you press and hold the Alt button and press 0 then 1 then 6 then the 3, when you let go of the Alt button you should get the £ sign wherever the cursor is flashing.

 

 NOTE: This works only on a desktop keyboard and laptop keyboards that have a dedicated number pad- many older Laptop keyboards do not have number pads.

 

If your laptop keyboard does not have a separate number pad on the side then on many (but not all) laptops you need to press the Function key (FN) before you press and hold the Alt button and then press 0 then 1 then 6 then the 3, when you let go of the Alt button you should get the £ sign wherever the cursor is flashing. You need to then press the Function button again or the next time you type a letter where the number pad is you will get just the number!
