    james.burgess say: You should be able to boot from the Windows DVD and use the Automated system repair to scan the HDD for Windows installs and re-write the affected MBR. (that's the easy but time consuming way)

    Alternatively you can use the Bootrec tool (Also on the Windows DVD)
    http://support.microsoft.com/kb/927392

    I would suggest something like 
    bootrec /RebuildBcd

    I believe this should scan for any OS's on the drive, and rebuild the boot record accordingly. (sorry it's been a while since i've done it myself)
    you may also need to use /fixboot


    or you can try using the bootsect tool from the Win7 DVD. (under tools)
    http://technet.microsoft.com/en-us/library/dd744577(v=ws.10).aspx

    Just to clarify, I believe bootrec fixes which OS's are listed in the Boot Record (if you've got a dualboot or whatever)
    whereas Bootsect actually writes a new boot sector to the HDD. (/fixboot may invoke bootsect.exe, can anyone confirm)

    I'd probably recommend taking an image of the drive using some drive imaging software before you try anything just so you don't cop any data loss

    Hope this helps.
