rohn007 say:
First, there is no "2003-2007" document. Those 2 versions of Word have significantly different default file structures, there is Word 97-2003 DOC, 2007 DOCX and 2010 DOCX (which is a minor variation on 2007 DOCX).  2010 can SAVE AS to 2003 format, but not as 2007.

WordUser, you can use a variation of Stuart's suggestion. In Word 2010, create a dummy "working" document that you SAVE AS in 97-2003 DOC format (or open an existing 2003 DOC file).  Create your Wordart with the image in it in this 2003 DOC format. Then COPY and Paste it into your 2010 DOCX. 

The "old" 2003 AND 2007 functionality is built into Word 2010, you just have to trigger word into the right compatibilty mode by opening a file created in the right version. That also means that if you want a 2007 specific formatting feature (I can't think of a specific example, but I'm sure there are some) you can use 2010 simply open a file created in 2007, create your effect then paste it into 2010.

Using this trick you can really extend the functionality of your application. I just takes "thinking outside of the box".  Unfortunately, this does not extend to the color palette so you can't use this technique to "trick" 2010 Excel or Word into giving us back the old 16 color scheme so many people wish for.

Hi,

 

I see that you want to delete a folder. Let me assist you on this.

 

To delete a folder, right click on the said folder, and then select Delete.

 

To delete all the emails in a certain folder, right click, and then select Empty folder.

 

To delete certain emails in a folder, left click the folder, and then select the email/s to be trashed. Click Delete on the toolbar. You may select multiple messages by checking the box on the left side of the email.

 

I hope these help. Should you have further questions, feel free to reply to this thread.
