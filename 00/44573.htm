¡firedog say:

Server Error: 0x800CCC90 


That means that you couldn't be logged on to the POP3 server. The usual reason is wrong username and/or password, but other error states can throw the same error. That's the situation you're in:

Server Response: -ERR [IN-USE] account is locked by another session or for maintenance, try again. 


You couldn't be logged on to your mailbox because another user is already logged on. There can only be one connection at once to a mailbox, and presumably your phone is hogging that connection. If it's configured to receive messages as soon as they arrive at the server, try turning push mail off in the phone. If you now receive mail in WLMail, you have the culprit. Set up the phone to use POP3 instead, polling at different intervals from WLMail to avoid collisions.
