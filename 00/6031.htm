suzanne s. barnhill say:

See http://wordfaqs.ssbarnhill.com/LetterheadRibbon.htm for general principles.

Using a full-page background (although it can be done) will cause every document to be gigantic. And using a full-color background will be disappointing because your printer not only will use prodigious amounts of ink/toner but also will very likely not print clear to the edge.

Instead, if you need art in the header/footer, create it there, anchored to the header/footer paragraph and formatted as Behind Text or Square (depending on how you want the document text to interact with it). If your header/footer art is full page width, with wrapping set to other than In Line With Text, you can set the position as Top/Bottom Relative to Page and Left Relative to Page, and it will stay put regardless of changes in margins.
