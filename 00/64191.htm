a. user say:

There are two types of user account:

  • a local account can be used only on the computer on which it was originally created
  • a cloud account can be used on any computer connected to the internet.

A Microsoft account is simply Microsoft's version of a cloud account.

A user account should always be protected by a password. If you're signing in to a local account, you'll use a local account password, which is also known as a user account password or a Windows password. If you're signing in to a Microsoft account, you'll need a Microsoft account password.

Clear as mud? Wait ... it gets worse.

Microsoft wants everyone to use only a Microsoft account, because follow the money. For that reason, Microsoft tries to blur the distinction between a local account and a cloud account to make it seem that a Microsoft account is the only account you and I should need. That's where your confusion comes from.
