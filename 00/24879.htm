(On 14955) This policy file has no language resources, so gpedit.msc cannot display it.

Create a text file "C:\Windows\PolicyDefinitions\zh-CN\GameDVR.adml" (change zh-CN to your language), and write the following text. You can change the translation if you like.

<?xml version="1.0" encoding="utf-8"?>
<!--  (c) 2006 Microsoft Corporation  -->
<policyDefinitionResources xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" revision="1.0" schemaVersion="1.0" xmlns="http://schemas.microsoft.com/GroupPolicy/2006/07/PolicyDefinitions">
  <displayName></displayName>
  <description></description>
  <resources>
    <stringTable>
      <string id="GAMEDVR">GAMEDVR</string>
      <string id="GAMEDVRHELP">GAMEDVRHELP</string>
      <string id="GameDVRMode">GameDVRMode</string>
      <string id="GameDVRMode_Help">GameDVRMode_Help</string>
    </stringTable>
  </resources>
</policyDefinitionResources>

C.Remove the stored Credentials in the Credential Manager:
