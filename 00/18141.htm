Word doesn't have a way to make this automatic.

What you can do is to modify the Footnote Text style to create a tab stop at the location you want the text to start, such as 0.25". (Remember that you have to allow enough space for two-digit footnote numbers, or three if you're a heavy user of footnotes.) Then each time you insert a footnote, press the Tab key before you start typing the text. When modifying the style, be sure to click the option for "New documents based on this template" at the bottom of the Modify Style dialog to save the revised style in the template.

doug robbins - mvp office apps & services (word) say:

In addition to setting a tab stop as Jay suggests, you may also want to format the Footnote Text style so that it has a corresponding Hanging indent so that the text for a long footnote will wrap to a position under the beginning of the text, rather than to a position under the footnote reference number.

Then, if you have the option set for using the Tab and backspace key to set left- and first-indent, you will need to use CTRL+TAB to insert the tab space.

If you first modify the Footnote Text style to have the appropriate tab stop and (if desired) hanging indent, then you can use Replace to add the tab character. graham mayor say:

You can't do it with a rule, but the following macro should mark the selected message as complete and categorize with the user's name. Although aimed at Word, Outlook VBA is virtually identical so see http://www.gmayor.com/installing_macro.htm

Sub FlagCompleted()
Dim olMsg As MailItem
    On Error Resume Next
    Set olMsg = ActiveExplorer.Selection.Item(1)
    With olMsg
        .TaskCompletedDate = Date
        .Categories = Environ$("UserName")
    End With
lbl_Exit:
Set olMsg = Nothing
    Exit Sub
End Sub
