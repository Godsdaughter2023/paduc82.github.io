9. Disconnect from that network.
10. Reconnect to that network and if it goes as planned the ISP should force a Dynamic IP. Check http://www.whatismyip.com/ if it has changed. If you cannot browse go to Step 6 and select "Obtain an IP address automatically" then Disconnect -> Reconnect the network.

thomas ferguson say:

You ask, "Can I install an operating system on my ram?"

Microsoft does not provide a way, nor does it support any way, to install any of its operating systems to any volatile storage device. That includes RAM in any form or configuration.

For a moment, let us suppose you could do it: As soon as power was removed from the device, all contents are lost. So, the installation would have to be repeated the next time you wanted to use the computer.

Consider an outline of what the installer does instead. The Windows DVD is put into the DVD drive and the installer copies a basic OS image to the HD and readies it for use. Control passes to the newly established system and a detection phase starts that determines the hardware the computer uses and assembles on the HD an image for the particular machine. The system restarts and uses the custom image. (The actual process might require several reboots and downloading files from the Internet, if a connection is available). At the end of the setup, there exists on the HD an image of the OS operating set customised for that computer. Each time the computer starts, it loads its core operating set. How much is loaded depends on the amount of RAM available. The Windows memory manager must take account of/respond to the memory requirements of programs that are started and run as well as background tasks like networking, file transfers for system and application use, &c.

Perhaps you would find it useful to consider the use of SSD or hybrid devices rather than or in addition to regular HDs.

Tom Ferguson
