phamnd say:

When opening a text document, got MS Word message. Word cannot start the converter mswrd632.wpc. Then got File Conversion message to select the encoding that makes your document readable

I deleted the registry key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Shared Tools\Text Converters\Import\MSWord6.wpc and now got the File Conversion message to select the encoding that makes your document readable.  I selected Windows (Default) and the document opens.  I found http://support.microsoft.com/kb/290981 which relates to Microsoft Word 2000 version. However, we have MS Word 2007.  To resolve the File Conversion window from prompting for MS Word 2007, I updated the HKEY_CURRENT_USER\Software\Microsoft\Office\12.0\Word\Options key instead of the HKEY_CURRENT_USER\Software\Microsoft\Office\10.0\Word\Options key as mention in article 290981.

1.     Quit all Microsoft Office applications.

2.     Click Start, and then click Run.

3.     In the Open box, type regedit, and then click OK.

4.    Locate, and then click to select the following registry key:

HKEY_CURRENT_USER\Software\Microsoft\Office\12.0\Word\Options

5.     With the Options key selected, on the Edit menu, point to New, and then click DWORD.

6.     Type DefaultCPG and then press Enter.

7.     Right-click DefaultCPG, and then click Modify.

8.     In the Edit DWORD Value box, select Decimal as the base.

9.     In the Value data box, type 1252 and then click OK.

10.  On the File menu, click Exit to quit Registry Editor.
