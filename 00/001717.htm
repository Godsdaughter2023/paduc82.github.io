jllatham say:

Have you tried right-clicking to "capture" the object and then selecting "Cut" from the pop-up menu?  I suspect you have some type of embedded Active X control in the cell.

You may want to expand the size of the cell by increasing row height/column width long enough to give this a try.

[Edit] If the object is an ActiveX control, you'll probably have to go to the [Developer] ribbon and click the "Design" icon in order to capture it with a right click.

If this fails, you can try this VBA code which will remove ALL "objects" of type Shape which includes controls.  Hopefully you don't have anything else like textboxes and such on the sheet that it will also remove:

Sub RemoveObjects()
  Dim anyObject As Shape
  For Each anyObject In ActiveSheet.Shapes
    anyObject.Delete
  Next
End Sub

To use the code: open the workbook, select the problem sheet.  Press [Alt]+[F11] to open the VB Editor and then choose Insert --> Module and copy the code and paste it into the code module.  Place your cursor anywhere within the code and press the [F5] key.  After that you can delete code (and even the module).  If the workbook doesn't have any other macros at all in it, you can later save the file as type .xlsx to remove all traces of the code/code module.

How to have text - that when you click on it - expands into an image - Microsoft Community

vitiran say:

How to have text - that when you click on it - expands into an image

I have a document that has lots of images in it.

The document is too hard to read if all of the images are visible.

So I would like to have them all collapsed by default.

So I would like a way, that as the person is reading through the document - when they come across a place where they would like to see the image - they can click on some text (that says like "Image of Cylinders") and it expands that part of the document to display the image.

Can anyone direct me in the right direction?

Thanks.

Vit

jay freedman say:

To make the following method work, all the pictures in your document must have "In line with text" wrapping, the document must be saved as a macro-enabled (.docm) file, and the recipient must store the document file in a trusted location such as their My Documents folder.

To set up each picture, insert it in a separate paragraph. In the same paragraph, press Ctrl+F9 to insert a pair of field braces { }. (Note: you can't type the braces on the keyboard because that won't make a field.) Between the braces, type the words Macrobutton TogglePic followed by the text that you want to display (in your example, "Image of Cylinders" but without the quotes). Press F9 to update the field; it will now display only the text.

Repeat for each other picture, changing only the display text in the field.

Open the macro editor (Alt+F11). Click on the name of your document in the Project pane on the left, and click Insert > Module. In the big empty pane on the right, paste this code:

Sub TogglePic()
    With Selection.Paragraphs(1).Range.InlineShapes
        If .Count > 0 Then
            .Item(1).Range.Font.Hidden = Not .Item(1).Range.Font.Hidden
        End If
    End With
End Sub

Sub AutoOpen()
    Options.ButtonFieldClicks = 1
End Sub

Sub HideAllPics()
    Dim pic As InlineShape
    For Each pic In ActiveDocument.InlineShapes
        pic.Range.Font.Hidden = True
    Next pic
End Sub

Put the cursor in the HideAllPics macro and press F5 to run it. That will make all the pictures in the document hidden, leaving only the fields next to them visible in those paragraphs.

Save the document.
