1) Click Start, and in the Search box type:   slui.exe  4

2) Next press the 'ENTER' key

3) Select your 'Country' from the list.

4) Choose the 'Phone Activation' option.

5) Stay on the phone (do not respond to any of the automated prompts) and wait for a person to help you with activation.

6) Explain your issue clearly to the support person.


if that fails:


To properly analyze and solve problems with Activation and Validation, we need to see a full copy of the diagnostic report produced by the MGADiag tool (download and save to desktop - http://go.microsoft.com/fwlink/?linkid=52012 )

Once downloaded, run the tool.

Click on the Continue button, after a short time, the Continue button will change to a Copy button.

Click on the Copy button in the tool (ignore any error messages at this point), and then paste (using either r-click/Paste, or Ctrl+V ) into your post.

Please state the version and edition of windows from your COA. do not state the product key.

george1009 say:
Notification Reason: 0xC004F00F

Error Code:  0xC004F00F

The Software Licensing Server reported that the hardware ID binding is beyond level of tolerance. 

The hardware has changed or the drivers were updated on the system.

(in this case a motherboard change)


Because you have an OEM license the motherboard change would invalidate the license. An OEM license is tied to the original motherboard it was installed on. Having said that try phone activation and hope the activation rep will activate it for you.( you may get lucky)

n05033 say:

hi’

would like to update on new development on my above query. Two days ago I have to make clean install again and I noticed that taskbar search box was working and it was working even after updates. But as soon as I made changes in “internet options” it stopped working.

So after few experimentations I noticed that when in (internet options >> general >> browsing history >> settings >> caches and databases >> allow website caches and databases) is selected the search box begins to work; and if it is unchecked (not selected) then it again stops working after few minutes.

If you are unable to check or uncheck this option then at “command prompt (admin)” type “gpedit”. And then in group policy editor select under “Computer Configuration >> Administrative Templates >> Windows Components\Internet Explorer\Internet Control Panel\General Page\Browsing History and then make sure that options in undermentioned both settings are either selected “Not Configured” or selected “Enabled”:

Allow websites to store application caches on client computers

Allow websites to store indexed databases on client computers

then go back to (internet options >> general >> browsing history >> settings >> caches and databases >> allow website caches and databases); and if you in gpedit as explained above have selected “not configured” then you will be able to check and uncheck it’ and if you have selected “Enabled” then it will automatically be checked and you won’t be able to uncheck it.

So; if it is checked; the taskbar search box works and if it is unchecked the search box won’t work (if it was already working it will stop working after some few minutes on unchecking it.)

It is strange that Taskbar Search box working has link to “Internet Options”.

However if it is unchecked and search box is not working; then on connecting to internet it starts working (as explained in my original query above.)

So; there are two ways of making search box work; either check the explained box in internet option; if not; then go online it will work even if “allow website caches and databases” is unchecked.
