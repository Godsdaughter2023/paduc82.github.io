You can create a password hint by creating or changing your password from the Control Panel's User Account option or from the Settings.

These steps are from the Control Panel:

  1. Type Control Panel from the Cortana search field, then click Control Panel from the results.
  2. Select Change account type under User Accounts.
  3. Choose the user you want to set the password with.
  4. Create or change the password, then type the password hint just below the password fields.
  5. Click the Create password button if you're creating the password for the first time, or the Change password button if you're only changing your password.

  2. Select Accounts, then Sign-in options from the left pane.
  3. Click the Add button if you still don't have a password, or the Change button under the Change your account password pane.
  4. Input your password or new password in the fields, then the password hint below.

lemp say:

You may not need to "rebuild" the system, but if you want to do so, see the following.

1.  No.  Windows Update will not work until sp3 is installed.


2.  A long time.


3.  You say that you have CDs for SP1 and SP2.  If your actual installation CD is the original no-service pack Windows XP, you do not need to install both SP1 and SP2.  SP3 can be installed on a system that has EITHER SP1 or SP2.  Be sure to disable or preferably uninstall any antivirus program before installing service packs.  I suggest that you install SP2, then reboot twice.  Then proceed as follows.

Windows XP Service Pack 3 --> http://www.microsoft.com/en-us/download/details.aspx?id=24

Ignore the comments on the download page about "intended for IT professionals."  That was written when these service packs were available through Windows Update, which is no longer the case.

Before installing sp3, read this --> http://msmvps.com/blogs/harrywaldron/archive/2008/05/08/windows-xp-sp3-read-all-prerequisites-for-a-successful-installation.aspx

To install service packs, log in as a "computer administrator" and disable (or preferably don't install yet) any antivirus software.  Reboot twice.  Run "Resolution Method 2" from here --> http://support.microsoft.com/kb/943144

Go to Windows Update and install the many post-sp3 updates.  You might want to do this in small batches, rebooting as needed. 

As an alternative to downloading all post-sp3 updates by visiting Windows Update, consider slipstreaming service pack 3 together with all of the post-sp3 updates.  See --> http://xdot.tk/

Don't forget to install an antivirus program.
