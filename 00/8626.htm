Right click on sheet1 tab, select View Code and paste in.....

[new Edit..]

Private Sub Worksheet_Change(ByVal Target As Range)

'in row 1 are headers
Dim r As Long
Dim rr As Range
r = Cells(Rows.Count, "A").End(xlUp).Row
If Not Intersect(Target, Range("A2:A" & r)) Is Nothing Then
Target.NumberFormat = "h:mm;@"
If IsEmpty(Target) Then Exit Sub
If Target.Value >= 1 Or Not IsNumeric(Target.Value) Then
MsgBox "wrong"
Target.Value = ""
Exit Sub
End If
For Each rr In Range("A2:A" & r)
If rr.Address <> Target.Address And _
Hour(rr.Value) = Hour(Target.Value) And _
Minute(rr.Value) = Minute(Target.Value) Then
MsgBox "wrong"
rr.Select
Target.Value = ""
Exit Sub
End If
Next
End If
End Sub

jannine_p say:

Hello

Welcome to Skype Community forum!

To change the email address where you can receive the verification for your account, you must update your account's security information. Security information is an alternate email address or phone number where you can get security codes.

To update it, please follow these steps:

1. Go to Security settings and sign in with your Microsoft account.

Note: After merging the accounts, your Skype account is now a Microsoft account. These accounts will now have one single sign in.

2. Under Security info helps keep your account secure choose Add security info.

3. Follow the instructions to enter your email or phone number, then press Next.

4. You'll get a security code at the email or phone number you just added. Enter that security code, and press Next again. Your new security info will appear in your list.

Let us know if you have any further questions or clarifications.
