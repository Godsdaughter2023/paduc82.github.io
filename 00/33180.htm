Error message when you try to restore from a full backup of a Windows Vista-based computer: "The system cannot find the file specified. (0x80070002)”:
http://support.microsoft.com/kb/951002

 You may also check all the drives for errors. To do this, try these steps.

1. Open Computer by clicking the Start button, and then clicking Computer.
2. Right-click every drive to check, and then click Properties.
3. Click the Tools tab, and then, under Error-checking, click Check now. If you're prompted for an administrator password or confirmation, type the password
or provide confirmation.

To perform a thorough check, select Scan for and attempt recovery of bad sectors. This scan attempts to find and repair physical errors on the drive itself, and
it can take much longer to complete.

To check for both file errors and physical errors, select both Automatically fix file system errors and Scan for and attempt recovery of bad sectors.
4. Click Start.
5. Once the drives error check is complete, try to restore backup and check the result.
