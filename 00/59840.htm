edbott live say:

I think you might be looking at the wrong partition.

There are four partitions on the default storage. The first is a 350MB Recovery Partition, the second is a 200MB EFI Partition, the third is a 53.88GB Primary Partition (System drive), and the fourth is the 5.08GB Recovery Partition containing the system image used for restore operations.

On my Surface Pro 3 with 64 GB of storage, I have installed the full version of Office and a number of other desktop programs and I still have more than 20 GB of space remaining.

One possible cause of storage vanishing is if you have a large amount of files in OneDrive and you try to sync the entire OneDrive to your device.

edbott live say:
This assumes you have the latest Windows 10 release. Right-click OneDrive folder in File Explorer, click "Choose OneDrive folders to sync." Select only folders you want synced, clear check box for folders you want cloud-only.
