This issue could be due to the bad sectors created on the hard drive due to the frequent unexpected shut down of the computer or it could be due to the faulty hard drive. If it is faulty, then you may have to contact the computer manufacturer to change the hard disk.

 

Since, you are unable to boot the computer; you need to have a Windows 8 installation DVD to boot into it and then access the command prompt. If you do not have a Windows 8 installation DVD, then please create it by following the given instructions:

 

Please refer to this article:

 

Create installation media for Windows 8.1

 

Follow the steps given in the above article to create an installation media for Windows 8. Instead of using the Windows 8.1 setup program, you can use the Windows 8 Upgrade Assistant from this link:

 

Upgrade Assistant: FAQ

 

After creating a Windows 8 installation DVD, run the check disk from the command prompt by following the instructions given below:

 

Run Check Disk from Command Prompt by booting the computer to the Windows 8 DVD

 

Check Disk (Chkdsk.exe) is a command-line tool that checks volumes for problems. The tool tries to repair any bad sectors, lost clusters, cross-linked files, and directory errors that it finds on the hard disk.

 

a)       Contact the computer manufacturer to change the boot device priority to CD/DVD drive and then boot from the Windows 8 installation disk.

b)       In the “Windows Setup” dialog box that appears, set the Time and currency format, Keyboard or input method and the language to install and then click “Next”.

c)       Click on “Repair your computer” from the lower left corner.

d)       From the “Choose an option screen”, click on “Troubleshoot”.

e)       Click “Advanced Options” in the “Troubleshoot” screen.

f)        Choose the option “Command Prompt”.

g)       On the "Command Prompt” type the following command, and then press “ENTER”

 

chkdsk /r

 

h)       Follow the on-screen instructions and let’s know the result.

 

Important: While performing Error-checking or ChkDsk on the hard drive, if any bad sectors are found and when attempting to repair that sector, the data available on that sector might be lost.
