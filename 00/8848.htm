suzanne s. barnhill say:

First enable both Different First Page and Different Odd and Even. While you still have only a single section, set up your Odd Page Header, Even Page Header, Odd Page Footer, and Even Page Footer. The First Page Header and First Page Footer will be blank.

You will then insert an Odd Page section break before each Day page and before each chapter-opening page. This will put Day One on a recto page followed by a completely blank verso page. Chapter One will also be a recto page, and the chapter will continue with the odd and even headers and footers. To restate: Each part title (day page) will be a separate section, and each chapter will be a separate section.

You can leave all the headers and footers linked except the First Page Footer. You'll have to unlink all of those in order to have a drop folio on the chapter-opening pages and no number on the day pages.

You'll probably also have some front matter, too, so have a look at http://wordfaqs.mvps.org/FrontMatterToolbar.htm (Word 2003 and earlier) or
http://wordfaqs.mvps.org/FrontMatterRibbon.htm (Word 2007 and later), as appropriate.

For the paragraph formatting, create or modify styles for your part titles (day pages) and chapter titles. I would advise using Heading 1 for part titles and Heading 2 for chapter titles if both will be included in the TOC. If only chapter titles are included, you can use Heading 1 for them and define a Part Title style for the day titles. Add enough Spacing Before to put the titles where you want them (I don't advise using vertical centering, even if it weren't too much trouble for alternate sections, because it always looks too low--optical centering is preferable). For the chapter titles, also add Spacing After. Spacing Before/After is stated in terms of points rather than lines, but you can also enter it in inches  (in or ") or centimeters (cm), and Word will make the conversion.

The only way to get Word to force you to show facing pages is to enable either "Mirror margins" or "Different Odd and Even" (which you will have done) and view two pages in Print Preview Edit Mode. If you're using Word 2007 or earlier, that is the Print Preview you get; if you have Word 2010  or later, you will have to add Print Preview Edit Mode to the Quick Access Toolbar.

For more on creating a book with Word, see http://daiya.mvps.org/bookword.htm.

tom van stiphout say:

You are confusing storage with presentation.

Double or Single is the correct data type for floating point values.

Decimal should be avoided as there is little support for it in VBA.

After changing the data type you can run a few update queries if you like, such as to round all values to 2 decimals.

10 will still be 10.

Then when it comes time to present the data, either in a form or a report, you can apply the correct formatting using the Format property of the control.
