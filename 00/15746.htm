    Me.Left = Application.Left + Application.Width - Me.Width - 25

Hi ByteBowser,

 

The message “To help protect your security, Internet Explorer blocked this site from downloading files to your computer. Click here for options” in information bar allows you to decide whether the blocked file is needed.

 

To disable this notification, configure the following setting in Internet Explorer.

 

1.    Open Internet Explorer.

2.    Go to Tools > Internet Options.

3.    Click on Security Tab.

4.    Click on Custom Level.

5.    Scroll down to Downloads section.

6.    Under file download, click on Enable.

7.   Click OK to confirm the setting.

 

The "enable" setting turns off the notification by Information Bar and will allow web sites to directly prompt you when downloading files.

 

That should be it.
