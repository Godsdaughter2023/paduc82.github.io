Sumproduct is slower than Sumifs, but more powerful. Sumifs cannot extract the month from a date and compare it with a value. If you want to use Sumifs, you need to create two helper columns to extract the month number from the date cells.

In this screenshot, cell D2 has the formula

=MONTH(B2)

copied down and across to column E. 




The formula in H1 calculates the unpaid items for May with

=SUMIFS(A2:A10,D2:D10,5,E2:E10,"")

The formula in H2 calculates the paid items for May with 

=SUMIFS(A2:A10,D2:D10,5,E2:E10,5)

teylyn say:
Hello,

to see a sum of values due in May (the 5th month) but not paid, use

=SUMPRODUCT(--(MONTH(B2:B10)=5),--(C2:C10=""),A2:A10)

To see a sum of values due in May and paid in May, use

=SUMPRODUCT(--(MONTH(B2:B10)=5),--(MONTH(C2:C10)=5),A2:A10)

Instead of entering the 5 for May as a constant, you an also refer to a cell that contains a month number or a value from which the month number can be deduced. For example, if cell D1 has a date in May, then you can use 

=SUMPRODUCT(--(MONTH(B2:B10)=month(D1)),--(C2:C10=""),A2:A10)

That way, if you want the formula to calculate the values for June, you just need to change the date in cell D1.

sven johannsen say:

What limitations? There are a few more supported profiles on WP8, than on WP7.

Here are the WP7 profiles.

Here are the WP8 ones.

 

What is supported, works.

 

The notable difference is in Object Push Profile (OPP 1.1). This allows sharing via BT. It does not set up a network so you can explore your phone from a PC, or vice-versa. With no File Manager or user accessible file system, this makes sense.

 

Yes you can send files, pictures, media, to and from a WP8 via BT. Pairing 'appears' to not work, but the devices, PC/Phone do understand each other. It says not connected, because there isn't anything you can do, until you initiate a transfer. You just need to start a transfer from one side or the other, via the Share option if from the phone. Select the file to share, then BT via the share menu.  If the devices have been previously paired, the file will transfer, if not pairing will be prompted.

 

Typically stuff sent to the phone will just wind up where it needs to be, as long as the phone understands what is being sent. Music will wind up in the music list, documents will wind up in office.

 

Transferring to a PC, the PC receive process will typically ask where you want the file saved.

 

And yes, the Lumia 920 runs WP8, and that WP8 set of profiles.
