russ valentine say:

Several things you needed to do differently:

1. Never use a sync with a PDA to "restore" Outlook data. It will never do so correctly and you'll usually have problems with the data a sync creates.

2. Never move or alter an Outlook data file until you have disconnected it from your Outlook profile. Doing so will corrupt your Outlook profile.

Since you created new mail accounts, those accounts are now using a new (empty) data files, so of course you won't see your existing data. You now need to open your previous data file (which Outlook cannot find now that you've moved it). Use the File > Open > Outlook data file command to open that file. You can then copy the data you need from it into your new data file(s).

Another wrinkle: Outlook 2010 creates a separate data file for each mail account you create. Most users don't want separate data files, but Outlook does not care what you want. You will need to decide whether you want separate files for each account or not. If you do, then you will need to copy the data you want from your old file into all of your new files.

cbarnhorst say:

Please always give the Insider ring and Insider build your device is presently on when asking a question like this.  The answer varies according to what you have.

For Fast Ring, Active Development, Slow Ring, and Release Preview Ring go to the Windows Insider Program page in Settings/Update and Security.  At the top of the page, click on the button that says Stop Receiving Insider Previews.  Choose an option from the list that appears (below). 

I recommend the fourth option (circled).  That is the same as CrazyKat's first recommendation except you don't have to wait for the next version of Windows to release before choosing to opt out.  You will continue to receive any preview builds released for your ring until the final Redstone 4 build is released to the public (next month).  Then you will receive that and your device will be automatically removed from the Insider program completely.  This must be done for each device you wish to opt out.  There is no "Insider" choice that opts you out of all Insider devices at once.  Any other choice may require re-installing Windows.

For Fast Ring, Skip Ahead, you must re-install the current public release (presently Fall Creators Update) to opt out of the program on your device.  There is no other way to leave Skip Ahead.
