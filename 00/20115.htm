rob koch say:

The reason antimalware programs can't find it is that it's not really malware, it's a leftover side effect of a possible malware file having been removed, but the startup item that was misused by the possible malware still remains.

So what is popping up is the Windows startup items failed attempt to execute the now missing file.  TO stop this you must find and remove this startup item.  Microsoft has the Autoruns for Windows application by Mark Russinovich available from its website in order to display, disable or delete these items.

https://docs.microsoft.com/en-us/sysinternals/downloads/autoruns

The instructions are also displayed on that page and it should be easy to find and disable/delete this leftover item using that app and page.
