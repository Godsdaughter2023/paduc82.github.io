rob koch say:

We recently had a report by someone currently located in South Africa that they were experiencing errors attempting to manually download current copies of the Microsoft Malicious Software Removal Tool (MSRT).  Two of us helping in these forums were able to duplicate these download issues at least on occasion, so there appear to be random flaky issues with downloads for at least the Microsoft Security applications.

As already mentioned, these definitions update packages occasionally contain new engines (detection program coding) which typically release around this time of the month and will also require an entirely new definitions database to match any changes made to the engine design.

However, since this engine update doesn't seem to have occurred yet, that means there are more definitions changes since the last engine release, meaning that if the much smaller daily delta definitions package download should fail, the full definitions download package that MSE then attempts to download will be much larger.

This is likely the reason for the recent growth you've seen in these downloads, so if you've got a slow network that's easily aggravated, then the possibility of a download failure increases with the size of these downloads as well.

I just looked at the manual MSE download file for a 64-bit system which contains both engines and definitions and this has currently grown to 139MB in size, while the page containing these various downloads is now termed "Security intelligence updates".

https://www.microsoft.com/en-us/wdsi/definitions

Can't say when the next engine release will occur, but when it does these package sizes should all decrease significantly, though over time they obviously tend to grow due to the increasing number of malware being detected.

  8. Save your presentation.
