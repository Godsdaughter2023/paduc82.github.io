I fixed the problem by redirecting the location where Word looks for the templates.  1 way is inside Word 2010 is to open it, do to File Tab, Link "Options" (in left side column),  click link "Advanced," scroll down.  Click button <File Locations...>, then modify the paths for the templates.   [[I have a folder in the root named Personal, and all my subfolders in there. It includes folders for Downloads and Templates.   In <File Locations...>, I navigated to select my C:\Personal\Templates folder.]]  I haven't had problems since.

 

2nd way. Go to C:\Users [[or \Documents and Settings ]], go to your username\roaming\app data\microsoft\ [[either Windows or Office is next\ then highlight, to select, folder Templates.  Right click on foldername Templates, select Properties.  Go to Location Tab.   Click <Move...>, and navigate to your personal folder named Templates, and click <OK>.  When prompted, yes, let it MOVE everything from the default folder.  THEN, close the pop-up, re-open the properties, select to MOVE it back to 'default' location, but do NOT move any files back.   NEXT, do the 1st method above, where you 'hard code' the selected Template Location in Word's Options/Advanced/File Locations.

 

Strong suggestion:  ALWAYS make a second copy (maybe even a 3rd) of your normal template, and let it be a copy that is automatically names copy (2) and copy (3). That way, you can recapture your template if any errors mess up the file's contents.

 

FYI...Office 2007, 2010, and 2013 have so many default locations you can reset that I also recommend you open YOUR template, and take 15-20 minutes to go through EVERY link/page in Options and setup your locations, default selections, personal quick toolbar, printing (e.g. 'reverse pages'), spacing, how cursor selects/moves, same for Insert key, and much more.  Definitely try using AUTOCORRECT (which you can set so you type an abbreviation and it prints several words or entire sentences for you, with no spelling errors.  Example:  I type "110" and mine automatically corrects to my full home street address.  Type "fm" and it goes to "from," and "mtc" goes to "maintenance."  You get far more typed in less time, and as long as you verified correct spelling when you created the Autocorrect entry, you won't have nearly as many potential spelling errors to worry about.   (This Autocorrect file (extension is .ACL) can be moved to another computer and/or new/different Word application, too.

 

Try redirecting the template location and see how that works for you.

  • Boot to a Windows 8.1 install USB or DVD
  • Choose Repair your computer
  • go to a Command Prompt
  • change the date to today instead of 2030 by typing "date 05-28-15"

jay freedman say:

Using the instructions in http://www.gmayor.com/installing_macro.htm, paste the following macro into a module in Normal.dotm. Replace the "Fred" with the actual name shown in the comments (including capitals and spaces as they are now), and replace the "Wilma" with the reviewer name that you want for those comments. Keep the quote marks around both names. Then run the macro.

Sub x()
  Dim cm As Comment
  
  For Each cm In ActiveDocument.Comments
    If cm.Author = "Fred" Then
      cm.Author = "Wilma"
    End If
  Next
End Sub
