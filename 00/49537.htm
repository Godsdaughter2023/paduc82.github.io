CPU setting C1E C2E C4E

CPU setting, should I have all three C1 C2 C4 state Enable

vijay b say:

Hi jazpaisley,
No, You need not have all the three C1,C2,C3 States enabled, the following are some of the functions.
1. Stop Grant Disconnect Clock Divisor - this is the same as for Halt Disconnect Clock Divisor with the difference that this one sets frequency divisor for C2/Stop Grant power saving mode. The effect of power requirements reduction is the same as in case of C1/Halt mode, but there are few differences. First - chipset has to be set to disconnect FSB upon receiving Stop Grant request, not Halt request. On many chipsets, there are separate bits for Halt and for Stop Grant, so be sure to set the right one.
2. Full Ramp - determines a max possible delay when ramping up/down the internal CPU clock. No information is currently available for this setting, so it's best to leave it default.
3. Ramp Delays (Clock Change Timings) - these settings affect time between clock changes inside the CPU while entering to/from C1/Halt or C2/Stop Grant states, provided that the Halt/Stop Grant Bus Disconnect is enabled in the chipset settings. Faster settings mean lower latencies and therefore higher performance and better resistance to problems with sound cards, HDD performance etc. But on the other hand, faster settings impose higher requirements on internal clock generator. If you experience problems with stability, use the slower settings (e.g., 8 or 16 for all fields).
4. Enable Stop Grant APIC Bus Disconnect - controls whether the CPU should reject any APIC Bus messages while being in C2/Stop Grant state. If you have problems with this low-power mode, try disabling this option.

NOTE: APIC Bus Disconnect has nothing to do with the AMD S2K Bus Disconnect feature that is enabled separately in the chipset configuration registers settings and is beyond the scope of RMClock utility.
5. ACPI State to Show/Modify - sets the ACPI power management state to manipulate with. These states correspond to PMM0-PMM7 System Management Action Field (SMAF) codes and are as follows: PMM0 = C2 (Stop Grant); PMM1 = C3 (Sleep); PMM2 = C0, FID/VID Change; PMM3 = S1 (POS); PMM4 = S3 (STR); PMM5 = C0, Throttling; PMM6 = S4/S5 (STD/SOFF); PMM7 = C1 (Halt).

Thanks for the reply.  I eventually found a remarkably simple solution that works on W10

https://pricklytech.wordpress.com/2014/02/04/windows-8-1-enabling-ahci-after-installing-windows/

Basically

1. From command prompt (Admin) enter the following:

bcdedit /set {current} safeboot minimal

2. Reboot into the BIOS and set the drive controller to AHCI

3. Reboot and it'll go into safe mode

4. Again at the command prompt enter the following:

bcdedit /deletevalue {current} safeboot

5. Reboot

6. JOB DONE

Worked a treat on my Windows 10 installation.

How to Change Windows 7 Product Key
http://www.sizzledcore.com/2009/10/26/how-to-change-windows-7-product-key/

Also: http://www.blogsdna.com/2233/how-to-change-windows-7-product-key.htm

Step by Step:
Click on the Windows 7 Start Button
Go to All Programs/Accessories
Next right on Command Prompt and select “Run as Administrator” from the menu.
Now type following at the command prompt:  slmgr.vbs -ipk  <insert the new product key here>
Finally activate windows after changing the key, at the command prompt:  slmgr.vbs -ato

Be aware the OEM product keys are unique and different from a retail key. Also why would you want to use up a perfectly good retail key on an already activated OEM product?

Hi Wahsah

While it is nice to hear that he is trying to be honest, there could be some problems with changing that key. The only real advantage would be that changing the OEM key to a retail key would give him the right to get free support from Microsoft for 90 days after the install date of the operating system.

However, this could also have the effect of nullifying any warranty that the Laptop manufacturer included with that system. The manufacturers warranty would be for 2 to 3 years and include any hardware replacements.

The student offer does not have any restrictions about reselling the key, as far as I know, since it is a normal retail product.
