Word's Label mechanism doesn't really provide for printing in a columnar manner. It works with the basic printer directionality which is Left-to-Right, Top-to-Bottom, or Across-then-Down. There isn't a simple means of changing that to Down-then-Across.

What you're looking to do, though, might better be served by using the Mail Merge Document feature called Catalog rather than Labels. doug robbins - mvp office apps & services (word) say:

To have a second set of numbering in the document that displays the page number of each Section, starting at 1 for each Section, insert a bookmark at the end of each Section such as S1 for Section 1, S2 for Section 2, etc. and then beginning with the second Section of the document, where you want the SectionPage of SectionPages to appear, insert the following fields

Page { = { PAGE } - {PAGEREF S# } } of { SECTIONPAGES }

where the # in S# is the number of the previous Section (if you use bookmark names as suggested above)

You must use CTRL+F9 to insert each pair of field delimiters { } and ALT+F9 to toggle off their display.
