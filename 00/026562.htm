    Excel - numeric format(number) - Microsoft Community

    jaykandwal say:

    Excel - numeric format(number)

    I am finding difficult in getting numbers if it more than 15(i.g. 123456789123456789) in excel cell. Either it will show scientific format or give wrong data. I wanted to show all numbers in the cell particularly if it is more than 15 character long.

    Thanks

    pete_uk say:

    15 digits is the limit of precision in Excel. If you want to have more then you have to treat the numbers as text - either you can pre-format the cell as Text before you type the number in, or you can type an apostrophe as the first character of the number, like:

    '123456789123456789

    The apostrophe will not show in the cell - it just tells Excel to treat the value as text.

    Hope this helps.

    Pete
