feelew say:
This worked for me. If you need to find out what number to attribute to the constant "myColor", use the second sub with the appropriate RGB values into it.

I limited the effect on the selection. If you want to apply it to a whole sheet (I haven't tested it), change Selection for Me.Cells.  It may take a long time to run through all the cells, though.

Sub EraseColor()
  Dim c As Range, i As Long
  Const myColor = 255
  For Each c In Selection
    If IsNumeric(c.Value) = True Then
      If c.Font.Color = myColor Then c.ClearContents
    Else
      For i = Len(c.Text) To 1 Step -1
        If c.Characters(i, 1).Font.Color = myColor Then
          c.Characters(i, 1).Delete
        End If
      Next i
    End If
  Next c
End Sub

Sub ColorFinder()
  MsgBox RGB(255, 0, 0)
End Sub

Hope that can help.
