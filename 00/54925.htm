a. user say:

Hi Jeff,

 

I would like to help you burn your video using Movie Maker. Here's how:

 

If you have a newly created video

1. Upon finishing the movie you created, save the project first for back up and renovation purposes.

2. After saving do not close Movie Maker. Press ALT + F.

3. Click Save movie.

4. Select Burn a DVD.

5. You will be asked to save a movie (Windows Media Video File format). Save it.

6. After saving, Windows DVD Maker will appear. Click Options on the lower right portion.

7. Kindly check the options there and change it on the desired configuration. (Note: If you want to save it on high-definition, you may need to choose 16:9 under DVD aspect ratio.)

8. Click Ok if you are done. Then select Next.

9. Next will be menu styles. Kindly configure it as you want. Then click Burn when ready.

 

If you have an existing project file

1. Open the project file on your Movie Maker.

2. Press ALT + F.

3. Follow the steps 3-9 above.

 

If you have only the video file

1. Open the video file on your Movie Maker.

2. Press ALT + F.

3. Follow the steps 3-9 above. (You may need to save the movie again upon doing step number 4.)

 

Thanks!
