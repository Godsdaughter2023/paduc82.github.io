vinay, india say: A normal VLOOKUP only retrieves the value of a cell. It does not retrieve the Hyperlink of the cell.

To lookup and reestablish the hyperlink, you may use a VBA function and INDEX and match functions.

OPEN VBA using ALT+F11, Right click on the VBAProject in the Project Explorer and INSERT a new module. Paste the following function in the code area on the right side.

Function getlink(HyperlinkCell As Range)
    If HyperlinkCell.Hyperlinks.Count > 0 Then
        getlink = HyperlinkCell.Hyperlinks(1).Address
   Else
        getlink = ""
   End If
End Function

Now on the sheet try using the following formula instead of a normal VLOOKUP.

Suppose the data is on Sheet 1, cells A1:B3 as follows
G          Google
Y           Yahoo
D          D_Drive

The second column values are respectively hyperlinked to:

http://www.google.com
http://www.yahoo.com
D:\

A6: D
A7: =HYPERLINK(getlink(INDEX($B$1:$B$3,MATCH(A6,$A$1:$A$3,0))),VLOOKUP(A6,$A$1:$B$3,2,0))


Try changing A6 to G or Y and see the hyperlink changing.

Hope this gives you the clue to incorporate similar logic in your working sheet.

chip pearson say:

Are you using Insert Hyperlink from the Insert menu or are you using the =HYPERLINK worksheet function? Post your formula. I created a simple sheet with values in column A and hyperlinks in columns B and C. A match successfully returned the cell's text, not the actual hyperlink address. You might want to post your VLOOKUP function.

 

If you need to get the actual http link, you'll  have to use VBA rather than a formula. Try code like

 

Function GetHyperlink(Value As Variant, _
    LookupRange As Range, ColumnOffset As Long) As Variant
    Dim R As Range
  
    For Each R In LookupRange
        If StrComp(R.Value, Value, vbTextCompare) = 0 Then
            If R.Hyperlinks.Count > 0 Then
                GetHyperlink = R(1, ColumnOffset).Hyperlinks(1).Address
                Exit Function
            Else
                GetHyperlink = CVErr(xlErrRef)
                Exit Function
            End If
        End If
    Next
    GetHyperlink = CVErr(xlErrNA)
End Function

 

This will return the hyperlink address (e.g., http://yourdoman.com), or #N/A if Value is not found in LookupRange or #REF if the found cell has no hyperlinks.

chip pearson say: Another thing crosses my mind. Ensure that the fourth and final parameter to VLOOKUP is FALSE, not TRUE or omitted. If it is TRUE or omitted, Excel requires that the lookup range be in ascending sorted order and it will return the largest value less than or equal to the lookup value. If the last parameter is FALSE, the data need not be in any sorted order and Excel will look for an exact match.

Longer version:

2. With the release of Windows 8, Microsoft created an additional app called "OneNote" (veterans may remember the radial menu), that should offer tablet users a very stripped down lightweight program to access OneNote notebooks that are stored to OneDrive only. The reason for this very likely was the introduction of the Surface tablets and the need for a "killer app" for them.

3. In 2014 Satya Nadella took over and declared the companies priority to be "cloud first mobile first". So Microsoft released additional OneNote versions (for MacOS, iOS, Android, Windows 8 Phone, Webbrowsers).
