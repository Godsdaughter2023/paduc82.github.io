ossiemac say:

I don't want to use the conditional formatting as I only want to control the background of the cell and not the font type border etc.

If you don't include the font type and border type etc when setting the required conditional format then they will not be affected with conditional format. Guidelines to use Conditional format to set interior color based on cell value.

  1. Select range C7:C200
  2. Select Conditional formatting
  3. Select "Manage rules"
  4. Select "New rule"
  5. Select "Format only cells that contain"
  6. First DropDown field select "Cell value" (Probably default)
  7. Second DropDown field select "Equal to"
  8. Third field enter 1
  9. Click "Format" button
  10. Select "Fill" tab (at top of dialog box)
  11. Click "More colors" button
  12. Select "Custom" tab (at top of dialog box)
  13. Enter the R,G,B color codes as 255, 255, 204
  14. Click OK -> OK -> OK and then click Apply.
  15. Repeat from step 4 (New rule) for each of the remaining values 2 to 5 and refer to the following table for the RGB values to use for each of the values.

Third field 2 RGB values 204, 255, 204

Third field 3 RGB values 255, 255, 153

Third field 4 RGB values 255, 255, 0

Third field 5 RGB values 153, 204, 255

I have given you the RGB values to use based on the color index that you used in your code. Of course instead of using the RGB values, you can simply select a color from the palette in the Conditional formatting.
