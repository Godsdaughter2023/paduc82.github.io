SHORTCUTS WITH CERTAIN WORDS IN THE NAME OF THE SHORTCUT CANNOT BE PINNED TO THE TASKBAR.

THE LIST OF WORDS THAT ARE FORBIDDEN ARE IN THE REGISTRY KEY:

 

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FileAssociation\AddRemoveNames

 

You can edit the list where necessary if you want to pin a shortcut which contains any of those words.

If there are no Administrator accounts on the system, you can boot to Safe Mode to access the Built-in Administrator account.  From this fail-safe account, you can force-reset your account's password.

  1. Press +I to display the Settings charm menu.
  2. Click the Power Options icon, then hold down the Shift Key while you click the Restart option. Keep holding shift until the screen says it is restarting.
  3. On the first screen that appears, select Troubleshoot, then select Advanced Options. Finally pick Startup Settings and click the Restart Button.
  4. On the next screen, press the Number 4 Key to choose Safe Mode.

Now that you are in safe mode, you can use the built-in backup Administrator account to perform you necessary recovery. The built in Administrator account should not be used for any other purpose, or it won't be available in situations like this when you need it.

  1. At the login screen, click the Arrow Icon and select the Administrator account.
  2. After Safe Mode starts, press +W and open Control Panel / User Accounts.
  3. Select Manage another account and select the account you wish to change to an Administrator account. You can also use the Change Password option here to reset the other account's password.

When you're done, restart the computer normally.

lemp say:

As I said, your cable modem is not a fax modem.  You do, however, have a fax modem -- incorporated within the hardware of your multi-function device.  Apparently, you have connected that device to one of the RJ-11 telephony ports on your cable modem and can successfully fax.

Although many manufacturers of such devices provide proprietary software that is capable of communicating with the fax modem from within Windows (as contrasted with being able to fax only from the machine itself), many of these interfaces are not designed to work with the fax software that is part of Windows, i.e., Windows Fax & Scan for Windows 7.

Your fax device apparently can be recognized through the printing interface,  but not through Windows Fax & Scan.  Windows has never had a very good fax interface, and Fax & Scan is no different in this regard.  It requires a fax modem interface, not a printer interface. 

Older versions of Microsoft Office incorporated interfaces with the fax software built into Windows.  Thus, for example, you could select File > Send to > Fax recipient and that would invoke the "Fax Wizard" where you would enter the necessary details and send the fax.  Newer versions of Office have a similar -- but not identical -- feature.  File > Save & Send > Send as Internet fax.  This feature recognizes that most people these days do not have a fax modem directly connected to their computer.  For those instances in which using fax is necessary, Microsoft apparently believes these people will rely on an Internet fax service. 

I have no idea why (or when) Microsoft elected to remove the capability of invoking the Fax Wizard from within Office applications.  This means that even though I have an internal fax modem I can't "Send to fax recipient" directly from an Office application.

On the other hand, you always could -- and still can -- print to a fax device.  You can do this either after you have saved your document and exited your Office application or directly from within the application.  Just go to File > Print, select the Fax printer, and click Print.  If the "Fax printer" has been created by Windows Fax & Scan (Windows Fax Services in earlier versions of Windows), selecting the Fax printer will cause the "Send Fax" dialog to open so that you can fill out the necessary data and then send the fax.

Given that you can, in fact, "print" to a fax, here's what I suggest:  install a second instance of your printer, give it a different name, and configure it to act as a fax.  Here's how (from http://superuser.com/questions/250417/how-can-i-make-a-copy-of-a-printer-in-windows-7):

  1. Go to [Control Panel] -> [Devices and Printers] and select [Add a Printer]
  2. Now select [Add a local printer]
  3. Select the set port of your printer in [Choose a printer port], commonly USB001 etc.
  4. In [Install the printer driver] select your printers Manufacturer and Model
  5. [Next]
  6. In [Which version of the driver do you want to use?] select [Use the driver that is currently installed]
  7. Enter a name for this new printer reference in [Type a printer name]

So, what we have now is a new printer reference, a reference you can see and select in the [Print] window printer selection [Name]. What we have to do now is to setup this printers individual settings

  1. Go back to [Control Panel] - [Devices and Printers] and select your printer (clicking)
  2. Select [Customize your printer]
  3. In [Select a printer] select your printer the same way as you would in the [Print] menu
  4. The [Print property] window pops up and you can set your individual printer settings here

The above directions require a bit of clarification for the second part of the process.

In step 1, when you go back to Devices and Printers, you will see only one instance of your printer.  Don't worry.

In step 2, there is no "customize your printer" that I know of; what you need to do here is to right-click on the icon for your printer and select "Printer properties."  This will open a sub-menu with the two names (you did remember to give your second copy a different name).  This is what step 3 means.

["Customize your printer" actually appears if you double-click the printer's icon (or right-click > Open).  When you click this link you'll get a drop-down list to select which copy of the printer you want to "customize."]

Select the second instance of the printer, configure it to act as a fax (however you do that now) and exit.  See the screen shot below:

You can then select the one configured to fax from within Word or other program.

Has anyone else had this problem or know of a solution?

I had this same problem with my Surface 3. The steps below worked for me.

First like others have already said make sure you have all of the windows updates installed. Even optional ones. If you are still having the problem look at this registry key

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Audio

"DisableProtectedAudioDG"=dword:00000000

Make sure it is set to 0.

In my case I had to create it and then reboot then videos worked fine after than.

Info provided as is

Always backup registry or make a restore point before making registry changes. Just in case :)
