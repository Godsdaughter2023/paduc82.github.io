Give it time...

A clean install of Windows will always invoke a lot of background processes... disk indexing, Windows Update, file system checks... open Task Manager and don't expect things to be "normal" until CPU activity gets below 15%.

Trying to open the start menus on good old Windows 8.1 is just not worth trying for a few minutes after a clean install; Windows 10 can be much worse if you are using a Microsoft Account - Win 10 preview will attempt to install the latest versions of your apps from the Store... that will cause app tiles on the start menu a few long pauses as the app itself updates... or waits to be called upon.

2. Please look into C:\Users\User_Name\AppData\Roaming\Microsoft\Excel (replace user name with your user name, you can directly reach here by copy and pasting following without quotes in Explorer address bar - "%appdata%\Microsoft\Excel") and see if is contains few files / folders. To disable add-in > File > Options > Add-ins > Down below you have Manage: Com Add-in : Press Go > Untick the Add-ins to disable and tick to enable them
