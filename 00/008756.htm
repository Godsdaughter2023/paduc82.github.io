suzanne s. barnhill say:
In the mail merge main document, open the Page Setup dialog. On the Layout tab, make sure that "Section start" is set to "New page" and not "Odd page." In a letter merge, each "letter" is a new section, starting with page 1. If each letter is only one page and the section break is an Odd Page break, you'll get a blank even page between pages of text. The same would be true of envelopes or sheets of "labels" (including cards).
