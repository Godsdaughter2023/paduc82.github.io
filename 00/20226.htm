You need to make sure to use the name of the subform CONTROL (the control which houses the subform on the main form) and not the subform name itself unless they share the exact same name.

Also, if the subform and subform control don't share the exact same name you need to include the .Form. part to let Access know you want to refer to something on the subform instead of the subform control:

 

Forms!MainFormNameHere.SubformControlNameHere.Form.ControlName = strMRN

so if your subform control is named frmRoundsSubfrm and the subform itself is not you need:

Forms!frmConsolidated.frmRoundsSubfrm.Form.RndsPatMRN
