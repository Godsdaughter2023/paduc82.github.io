Macro to copy highlighted word & open File Property Tags - Microsoft Community

sparks t say:

Macro to copy highlighted word & open File Property Tags

Word Windows 10 Office 365

I have been trying to figure out a way to easily add tags to a file as I am creating it.  

I want to be able to highlight a word from my document and then start a macro that will copy the word and open the File Property Tag field so I can paste that word into my tags.  I can manually add the tags but that involves moving my hands off the keyboard and over to the mouse which is very disruptive and slow.

Anybody know of a command I can use to open the File Properties?

jay freedman say:

Versions of Word from 2010 onward have little to no VBA access to the "Backstage" (the pages reached by clicking the File tab). If that's where you're trying to reach the Tags field, you need a different approach.

The VBA statement

Dialogs(wdDialogFileSummaryInfo).Show

opens this legacy dialog:

Anything typed in the Keywords field will appear in the Tags field on the Backstage > Info page and in the Details tab of File Explorer’s Properties dialog.

To use this dialog in a macro, you don’t have to open the dialog with the .Show method; instead you can set the content of the .Keywords property and use the .Execute method.

Try this macro:

Sub CopyPasteTag()

   Dim dlg As Dialog

   With ActiveDocument

      If Selection.Type = wdSelectionNormal Then

         Set dlg = Dialogs(wdDialogFileSummaryInfo)

         If dlg.Keywords <> "" Then

            dlg.Keywords = dlg.Keywords & ";"

         End If

         dlg.Keywords = dlg.Keywords & Selection.Text

         dlg.Execute

         .Save

        Set dlg = Nothing

      End If

   End With

End Sub

(You may have to close and reopen the document to see the Tag field’s new content in the Backstage.)
