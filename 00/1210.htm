How to use sumif with criteria that says "not equal to"? - Microsoft Community

jakethesnakez say:

How to use sumif with criteria that says "not equal to"?

Hi everyone,

This is what I have:

(SUMIFS('MM'!H:H,'MM'!AO:AO,"Acknowledged",'MMi'!AM:AM,NOT("201403")))

Basically, I'm trying to sum the total of all values that are NOT equal to "201403".

Currently, I have the values in column B, and I want that to be summed based on what each of their criteria is in column AB.

Thanks in advance.

jeeped say:

···The above formula gives me the #VALUE error.

That is likely because you are using full column references; something to be avoided with SUMPRODUCT(). The text you are using as a column label in H1 is fouling up the mathematical calculation. Try,

=SUMPRODUCT(('MM'!H2:H9999)*('MM'!AO2:AO9999="Acknowledged")*('MM'!AM2:AM9999<>"201403"))

If it isn't text in H1, then there must be some text somewhere else in column H. If you absolutely must use full column references, try,

=SUMPRODUCT(('MM'!AO:AO="Acknowledged")*('MM'!AM:AM<>"201403"), 'MM'!H:H)


Isolating H:H from the criteria seems to help this by not always including it in the calculation.
