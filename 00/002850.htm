macropod say:

The APA standard for citations is-

One author: Write only the author’s surname in-text and in brackets.
Two authors: Cite both surnames every time the reference occurs in text.  Use ‘and’ between authors, but use an ‘&’ when in brackets.
Three to Five authors: Name all authors in your first in-text reference, but use ‘et al.’ after the first author's surname in subsequent references. Use ‘and’ between the last two authors in your sentence preceded by a comma, but use an ‘&’ instead of ‘and’ when in brackets.
Six or more authors: Cite only the first author’s surname, followed by ‘et al.’ in-text and in brackets.

Microsoft's implementation does not conform to APA - and never has. Instead Word inserts all as per the specs for the first reference only - in brackets. Word does not do non-bracketed references.

As Paul noted, the way Word handles bibliographies has never been up to published standards. And it always takes MS several years to add updates to the various standards when they are updated by the standard owners.

The various citation formats used by Word are defined in XML files.

As I see it you have a few options:

  1. use 3rd party tools to do the bibliography work for you
  2. download XML style files from some other source
