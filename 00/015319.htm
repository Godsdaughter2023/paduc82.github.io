keeping font consistent in new footnotes/pasted text - Microsoft Community

emmaspary say:

keeping font consistent in new footnotes/pasted text

Hi, I've tried searching for the answer to this to no avail. I am working on a document using a particular font. Whenever I paste in some text from another file, or create a new footnote, the program uses a different font--in the former case, the font of the source file; in the latter case, the default font.

I don't want to change to default font globally--is there another way to get my pasted text excerpts and my new footnotes to keep the font of the document I'm working on? Even one of these would be helpful.

Thanks!

stefan blom say:

You can change the default font for the document you are composing which would be the easiest approach. In the Styles pane (Ctrl+Alt+Shift+S), click the Manage Styles icon and click the Set Defaults tab. Specify the font that you want. Since you just want to make this change for the current document, ensure that you select "Only in this document" before you click OK.

Alternatively, you can change the font of the footnotes, specifically. Click inside a footnote and then press Ctrl+Shift+S. In the Apply Styles pane, verify that you see Footnote Text as the style name. Click the Modify button and change the font. When you are done, click OK. The text of new footnotes (in the active document) will now use the font that you specified.

Note that the above suggestions define the default font for the document or for a particular paragraph style. At any time when you bring in formatting from another document, formatting may change. What you can do is click the Keep Text Only option found on the Paste Options button; this ensures that Word uses the font of the target text when pasting.
