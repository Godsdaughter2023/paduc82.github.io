***Post moved by the moderator to the appropriate forum category***

jay freedman say:

You can do this without a macro by using the Replace dialog. Click the More button in the dialog and check the box for "Use wildcards". In the Find What box, enter the expression

(^13{2})^13@

In the Replace With box, enter

\1

Then click the Replace All button.

If you still want a macro, for instance as part of a larger operation, the equivalent in code is

Sub ZapExtraCrs()
    Dim rg As Range
    Set rg = ActiveDocument.Range
    With rg.Find
        .MatchWildcards = True
        .Text = "(^13{2})^13@"
        .Replacement.Text = "\1"
        .Execute Replace:=wdReplaceAll
    End With
End Sub

The expression for the Find What says "find an instance of two vbCr characters (ASCII 13) followed by one or more additional vbCr characters". The \1 in the replacement refers to the first part of the expression surrounded by parentheses. Thus, three (or more) paragraph marks will be replaced with only two paragraph marks.

macropod say:

I want to force the entire file into 2 forced paragraphs (that is, programmatically delete the extra vbCr IF and only if there are 3 vbCr's in a row).  I do not want to change the normally separated paragraphs (i.e., hitting the enter key once).

Even that is not an optimal outcome. What you should be doing to ensure consistency is using a single paragraph break to separate all paragraphs and setting the paragraph before/after spacing to a suitable value, preferably at the paragraph Style level - using different Styles with different before/after spacing settings, according to how much inter-paragraph spacing you want.
