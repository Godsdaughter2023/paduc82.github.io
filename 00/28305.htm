phillip m jones say: Two ways:

First method:
  1. Open blank document. In Word Preferences > Save
    Click check box next to: prompt to save normal template.
  2. Next select desired font.
  3. Type something.
  4. Now save to desktop as test.docx.
  5. Quit word.
  6. You then will be asked to whether you want to save changes to normal, click okay to Change.
Second method:
  1. Open a blank document change font to desired font.
  2. Next go to Format Menu > Style
  3. Make sure Normal is chosen
  4. Choose your desired Font, and size.
  5. Choose Modify
  6. In Modify window change to desired font (this sets default font).
  7. Click in the add to Template check box
  8. Then choose okay.
Continue with you document. when you save your document you should see a notice about saving changes to normal.
