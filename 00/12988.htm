You may be able to leverage the Baseline functionality that is available in MS Project to track the original dates that the vendor created.

Under the "Project" Tab, select the "Set Baseline" button to have all of the current Start & Finish date values of the project's tasks copied to the static Baseline Start and Baseline Finish fields.  These baseline fields will capture the initial dates provided by the vendor and you can use them to check against any Start/Finish dates that my change in the future.
