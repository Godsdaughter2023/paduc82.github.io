deano152 say:

Increase video memory

How do I increase my video memory on my windows 10 laptop?

***Title modified from: video memory***

junaid_a say:

Hi Deano,

Thank you for writing to Microsoft Community forums.

  • What is the make and model of the machine?

  • Do you have dedicated graphics card installed on the machine?

By default, a particular fixed amount of video memory is allocated for every system with integrated graphics card. However, if you have a dedicated GPU that has more memory built-in, installed on the system, you may increase the dedicated video memory.

If you do not have a dedicated GPU installed on the machine, it is not recommended to manually expand the integrated video memory. Increasing video memory manually can have detrimental effects than helpful as it won’t increase performance. In integrated graphics devices, all graphics memory is system memory. The graphics driver allocates memory from the operating system as it needs it – dynamically.

Dynamic Video Memory Technology (DVMT) dynamically allocates system memory for use as graphics memory. It balances 2-D/3-D graphics and system performance. Graphics memory is allocated based on system requirements and application demands (up to the configured maximum amount). The dynamically allocated portion returns to the operating system for other uses, when an application no longer needs memory.

In most cases, the computer manufacturers configure the computer to use the maximum amount of graphics memory possible. For more information on the same, refer your device’s specs.
