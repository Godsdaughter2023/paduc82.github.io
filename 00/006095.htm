Do you know what you used to create these fractions?

If you click Alt-F9, instead of the "Error!" message, do you see field code braces like this

{  }

but with something written inside?

If what you see is 

{ EQ something }

then a possible problem is that EQ field expressions often have lists separated by the list separator character that is defined in Windows COntrol Panel (typically it is either "," or ";"). If your new notebook has a different list separator set up in Windows control panel, the EQ fields will not work properly - you would need either to change the character in Windows Control Panel (Regional Options) to be the same as it was on your old PC, or change all the relevant list separator characters inall your EQ fields to use the same character that is set up in Windows Control Panel (not very easy IMO!).

Otherwise, the chances are that you used the old equation editor, which inserted an Equation object. That may not work in Word 2016 (I would have to check). These days, Word uses a completely different and incompatible equation editing feature.
