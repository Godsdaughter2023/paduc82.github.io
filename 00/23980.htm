To start up in Safe Mode, press 4 or F4. To start up in Safe Mode with networking, press 5 or F5. To start up in Safe Mode with the Command Prompt, press 6 or F6. hi jc9375,

thanks for your clarification.

by default, email messages that are sent from exchange online in office 365 use the transport neutral encapsulation format (tnef). messaging systems that aren't based on microsoft exchange may be unable to interpret messages that use this rich text format. if the recipient’s messaging system can’t process this format, a file attachment that's called winmail.dat is added to the message.

as an office 365 admin, you can use windows powershell to change the message format to prevent the winmail.dat attachment from being sent to external recipients.please follow these steps:
1. connect to exchange online using remote powershell
2. 
    a. if you just want to change the message format for an external contact that was added to exchange online, please refer to these commands:
        1. set-mailcontact <externalemailaddress or guid> -usemapirichtextformat never
        2. set-mailcontact -identity <externalemailaddress or guid> -useprefermessageformat $true
 
    b. if you want to change the message format for all messages that are sent to a specific domain, please use these commands:
        1. new-remotedomain -name <name of external domain> -domainname
        2. set-remotedomain -identity <name of domain> -tnefenabled $false

note: 1.  you can copy commands then right-click in the powershell console.
          2.  when inputting a command like “usemapirichtextformat”, you can enter usem and hold down the key“tab” to match.
          3.  if yours is not a global administrator account, you need permission to run these commands.

for detailed information, you can go to the following microsoft websites:
• set-mailcontact: http://technet.microsoft.com/en-us/library/aa995950.aspx  
• new-remotedomain: http://technet.microsoft.com/en-us/library/aa997168.aspx
• set-remotedomain: http://technet.microsoft.com/en-us/library/aa997857.aspx
