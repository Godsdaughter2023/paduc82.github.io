Drive D: is full and I was going to try and compress it but it tells me that access is denied to do so. Is compressing it the best option and how do I get access to do it?
steve winograd say: What's on drive D:?

Some computer makers, including Dell and HP, put a recovery partition on the hard disk of new computers.  By pressing a special key combination when the computer starts, you can use the recovery partition to restore the computer to the way it was when it was new.

If that's what's on Drive D:, there's nothing wrong, and you don't have to do anything.  Windows has assigned a drive letter to the recovery partition, but nothing in the system will access it, and it doesn't matter that it's almost full.

In that case, I recommend that you remove the drive letter from the recovery partition:

1. Click the Start button.
2. Type diskmgmt.msc in the Search box and press Enter .
3. In the Disk Management window, right-click the recovery partition.
4. Click Change Drive Letter and Paths .
5. Click Remove .

john_barnett say:

Hi Tamazyn,

Compressing is not the best option, for one it slows down file access. Did you create the D: drive/partition or was it there when you purchased your PC. Normally PC manufacturers put imaged copies of the operating system on the D: drive as part of their recovery options. If this is the case you shouldn't be storing files (other than the manufacturer's image) on the D drive. The manufacturers recovery partition (D: drive) is also usually on the small side so if you are putting files there then it will soon fill up.

Removable USB drives are not all that expensive and are ideal for backing up your documents and other files.

davem121 say:
Hi Titus

Did you create Bootable Installation Media using the media Creation Tool?

If not, see the not at the bottom . .

If you have done that:
Insert the bootable installation media into your system
Boot into BIOS and click Restore Defaults, so you know you are starting from a known point . . .
Then change the boot order so that installation media is top of the boot order list, so your system is booting from the Installation Media
Save that and exit BIOS, your system will restart and if the installation media is created correctly it should boot from that and being the Windows 10 installation


To create Bootable Installation Media:
Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool


If you have problems setting that drive as the boot device:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
