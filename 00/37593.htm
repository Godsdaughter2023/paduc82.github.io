How to Find SSID (Network Name) - Microsoft Community

xb1r hydros say:

How to Find SSID (Network Name)

Login Information

  • Gateway: 192.168.1.1
  • Username: 1234
  • Password: 1234

 

How to Find SSID (Network Name)

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the gateway address: 192.168.1.1 into the web address bar, select apply.
  4. This will take you to a login page, type 1234 for the username and type 1234 for the password.
  5. Select Network on the main page.
  6. Select Wireless LAN.
  7. Your current SSID will be shown on this page. You can also change the SSID from here as well.

a. user say:

Hi,

 

Welcome to Microsoft Community.

 

I understand that you want to know about "Conversations on" and you want to sort by date but not conversation, let me assist you with this.

 

Conversation threading is grouping email messages from a contact as if you and the contact are in a conversation. You will see your replies alternating each other resulting in more convenient way to read messages with a contact.

 

If you only want to sort emails by date, please follow the steps below to disable the Conversation On:

 

1. Sign in your account to Windows Live Mail

2. Above the list of tab, click View and look for the button Conversations and click Off

 

Now go back to your list of emails and click the Sort by then choose Date
