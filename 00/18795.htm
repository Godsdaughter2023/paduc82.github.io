If the user account does have the license and user mailbox but it doesn't appear in Exchange admin center, you can run PowerShell command to convert the user mailbox to a shared mailbox.

Set-Mailbox <user account> -Type Shared

We can only guess at this point.  One of two possible mistakes; perhaps both.

First, what type of chart are you using?

I suspect it is a Line chart.  Trendline assumes that the x values are 1, 2, 3, etc.

You should use an XY Scatter chart, with a line subtype, if you wish.

Second, exactly what formula are you using to apply the trendline equation?

I suspect you are using a formula of the form:  =0.00014*X1 + 0.56390.

The problem is (might be):  the coefficients do not have sufficient precision to generate estimated y accurately.

Right-click the trendline formula in the chart, and click Format Trendline Label > Number to choose an appropriate format with a sufficient number of decimal places.

I prefer Scientific with 14 decimal places.  Although that might be overkill, it works for all magnitudes of coefficients, and it displays the maximum number of significant digits that Excel formats (15).

Copy-and-paste the coefficients into your worksheet.

Even better:  consider using LINEST in the worksheet.  For a linear trendline, select to horizontal cells (e.g. A2:B2), type =LINEST(ydata,xdata), then press ctrl+shift+Enter instead of just Enter.  A2 will be the slope; B2 will be the intercept.  Your formula becomes:  =$A$2*X1 + $B$2.
