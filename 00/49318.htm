wolfgangwohlkinger say:

Ok. Solved.

 

Solution

 

After installing the HP Laserjet 1018 driver from Windows Update, you have to:

 

1. Remove the driver completely (use Print Management tool - find it via Start | Search)

2. I had to remove the printer from my Devices and Printers so that it was not in use.

3. Download the latest driver from HP - make sure you have the correct 32bit or 64bit version.

4. Reinstall with printer disconnected and connect when asked to do so.

 

That fixed it for me.

 

Just don't install the driver from Windows Update again.

If your console was online, your savegame worlds are safely stored in the cloud.

After that, you will be able to play Minecraft again.
