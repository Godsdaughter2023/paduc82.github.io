bob jones aka: cybertaz say:

It sounds like you've adjusted the settings for the individual Animation Effects but have not adjusted the overall Transition Timing for the slide, itself. The Effects do not have a "Delay After" attribute. As a result, the slide is automatically advancing as soon as the last animation is complete.

Go into Slide Sorter View & use the Rehearse Slide Show button or select Rehearse Timings in the Slide Show menu to set the overall duration of the slides. You can record the Timings as you advance manually at the appropriate point.

Alternatively, select the slide, then go to Slide Show> Slide Transition, then check the box for Automatically after & enter the preferred number of seconds. The Timing of each slide can be adjusted in that same dialog at any time.

mike smith mct say:

SharePoint Views can't directly sum a Calculated column. The solution is to create a SharePoint Designer workflow that copies the value of the Calculated column into a simple Number column. The Number column can be totaled.

The basic steps:

  • Add a simple Number column to your list.
  • Open SharePoint Designer and connect to your site.
  • In the left column click Workflows.
  • From the ribbon click List Workflow.
  • Pick your list.
  • Name the workflow and select the "2010" style of workflow. (The 2013 style will work, but has a few extra steps.)
  • Add a Set Field in Current Item action to the workflow.
  • Click "field" and select your simple Number column.
  • Click "value", click fx, in the Field from Source dropdown select your calculated column. Click OK.
  • In the ribbon click Workflow Settings.
  • Set the workflow to start on Created and Changed.
  • Publish the workflow and test.
  • Modify your View and add the Number column and configure the View to sum that column. Uncheck the original calculated column.

Note that only new and edited items will be updated by the workflow.

Hi

Remember that the Windows 10 Technical Preview is pre-release software and there is no assumption that any third party hardware will work on this version.

Try the following.

1. download a fresh copy of the latest driver for the device and place it on your Desktop.


2. Right click the driver installation file and select Properties / Compatibility Tab.


3. Place a check mark in the Run this program in compatibility mode for.


4. In the drop down menu, select Windows XP.


5. Click Apply /OK.


6. Right click the installation file and select the Run As Administrator option to install.

Test to determine if this process was successful.

If this does not work, contact support for the device manufacturer and ask if they plan to release drivers for Windows 10.
