This issue might occur due to display settings on the computer. Sometimes, updating the display driver is also recommended to avoid such scenarios.


Did you install any other third party software to enable Windows XP style view?


If that's the case, I would suggest you to uninstall it and check if the Charms bar option shows blank when using Alt+Tab option to switch applications. 


For further troubleshooting, follow the steps to check if disabling the peek effect fixes the issue:

  • Press Windows+X keys and select System from the context menu.
  • Click on Advance System Settings.  Choose Settings for performance menu under Advance tab.
  • Uncheck the option that says “Enable Peek”.
  • Click on OK.

If this does not work, I would suggest you to refer to the manufacturer website to download and install the latest display driver for Windows 8.1. Refer to the link to know how to update display card driver:

http://windows.microsoft.com/en-us/windows-8/find-install-display-drivers
