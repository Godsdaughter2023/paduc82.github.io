balaji kundalam say:

Fast Startup is a feature available from Windows 8. Quick Boot is a feature present in BIOS. Since Quick Boot is hardware dependent, it reduces time by skipping some POST (Power-On Self Test) tests. The settings can be modified in the BIOS settings page. Refer your PC manual for this.

In Windows 7, Fast Startup feature can't be implemented. But, Quick Boot can be enabled in the PC hardware, but boot time will not have much impact as Windows boot time remains the same, whether Quick Boot is enabled or not, as it's purely hardware based.

You can try the following link about making Windows 7 boot faster:

http://www.wikihow.com/Make-a-Windows-Computer-Start-up-Faster

In Windows 7, Fast Startup feature can't be implemented. But, Quick Boot can be enabled in the PC hardware, but boot time will not have much impact as Windows boot time remains the same, whether Quick Boot is enabled or not, as it's purely hardware based.
