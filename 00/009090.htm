How can I paste this without disabling merged cells, copying and then merging again?

Thank you very much!!
teylyn say:
Hello, 

that's the problem with merged cells. They screw up in places where you don't expect it. 

Rather than merging the cells B2 to D2, use "Center across selection" if you want the text to be displayed like that. If you don't want it centered, then just leave the text in cell B2 alone. Any other appearance can be done with formatting. 

Merging cells does not help. It only introduces problems. 

Don't. Use. Merged. Cells. 

So, select your merged cell B2, and unmerge it by clicking the "merge and center" icon in the ribbon. Then select the cells B2 to D2, right-click and select "Format Cells". On the "Alignment" tab, select "Center across selection" in the drop-down for "horizontal" The text in cell B2 will now be centered across columns B to D. (If you want the text to be left-justified in cell B2, just enter the text and don't perform any formatting). You can now select cell B2 and copy it. If you paste it into cell C4, only cell C4 will be changed and cells D4 and E4 will not be affected. 

Merged cells create more headaches than there is aspirin in a store that sells pills. Don't use them. They're not worth it.
