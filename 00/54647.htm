tagesmann say:

You don't need to install a language pack or change display settings.

Try this easy solution that I got from CShark on superuser.com

 

Create a text file called 火.txt and save it to your desktop. And then reboot.

This works because of font-caching.

There are two main parts in Windows that create the cache. One is the Windows Explorer, the other one the DirectWrite part of DX. The problem is that DXW fails with Unicode. So when a program starts up using unicode and the dxw api, the font-cache is build by dxw and does not contain unicode because of the earlier fail.

However, if you place the 火.txt file on your desktop, explorer fills the cache first. And explorer supports unicode, so the cache is initialized correctly.
