    Hi Terry,

    manolo007 say:

    I think I sorted this out:

    When creating new users windows uses the Users/Default as a template and generates the new user directory, if it fails in copying this template it will fail generating the new user.

    Went to the Event Viewer and found a warning regarding two files that couldn't be copied from the users/default directory to the directory destined for the new user.:

    C:\Users\Default\AppData\Local\Microsoft\Windows Live\SqmApi\SqmData720896_00.sqm

    C:\Users\Default\AppData\Local\Microsoft\Windows Live\Bici\Bici1_00.sqm

    I just deleted the Windows Live directory under users/default/appdata/local/microsoft, and 'voilá!' sorted
