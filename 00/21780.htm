kosh vorlon say:

Hi Jellybean,

 

Why do you feel you need to update manually when MSE really does (or should) update automatically?  For further information, please see http://answers.microsoft.com/thread/74e507b8-f6da-4eca-8ce7-d1aca7d3f1ba.

 

By manually, do you mean you need to open MSE, go to the Update tab, and click on Update Now or do you mean you need to download the big file from the MMPC and install it because even the tab doesn't work?  If the former, does it download but fail to install or does it do nothing at all?  If you mean the latter (regardless of the type of problem with the button), please see the following thread where this is explained in greater detail and includes options and other links to examine for forther information: http://answers.microsoft.com/en-us/protect/forum/mse-protect_updating/automatic-update-downloads-but-fails-to-install/f0a60677-a8bf-4760-a702-335dd8c9b23e.  Pay specific attention to the post from Stephen Boots.

 

Are Windows Updates working properly?  Are you getting any error codes with them or with the MSE updates (like 0x80070424)?  If so, the problem may be with updates in general or possibly with your MSE installation.  Please see: http://answers.microsoft.com/en-us/protect/forum/mse-protect_updating/my-windows-automatic-update-is-not-turn-onalso-i/d503ef0d-7ef2-4d0f-b1ec-cfb28f6ab23c if you have this error code or other problems with automatic Windows Updates.

 

How current are you trying to keep your MSE definition updates?  They are released approximately 3x/day as optional updates, but it is absolutely not necessary to keep up with them that frequently.  Once per day (which is what MSE does at an uncontrolled time - and as Chris said that can't be turned off or on because it's hard coded) is quite sufficient (and even if you went an extra day it would not be a big deal).  

 

If you do not manually update at all (whatever you really mean by that), what happens?   How long does it take for MSE  to automatically update or does it go without updates until your icon turns orange telling you that your system is potentially at risk due to not being updated (which is about 7 days)?

 

If you are expecting MSE to update immediately upon starting the computer (as part of the process of starting MSE) or shortly thereafter, then that doesn't occur (it might sometimes and might not most of the time and cannot be made to occur in any event). It is not an available option - and it is not necessary. Just be patient and it will eventually automatically update at some point. There's no need to update manually because it does not automatically update at startup or within a few minutes of startup. This is true even if you check and see that more current updates are available provided it does eventually automatically update.  That is perfectly normal and we all face that situation.

 

To increase the frequency or probability of updating (and make sure you are current when doing scheduled scans), I generally recommend you open MSE, go to Settings, go to Scheduled Scans, and check the box to automatically check for updates before performing the scan.  This is not a substitute for automatic updates (though if doing daily scans, it can possibly accomplish the same objective (if it works any better) and even if it works and especially if you don't scan that often, then properly functioning automatic updates should be fixed as noted above.

 

I hope this helps.  If not, please explain exactly what you are doing and when and how frequently and what makes you think automatic updates are not working.

 

Good luck!
