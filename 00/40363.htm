What is WLMailSearchSentinel.eml?


When Windows Live Mail is installed, a folder called Sentinel is created in the store folder. This folder contains a single file, a text file called WLMailSearchSentinel.eml and containing a simple text string, easily reproduced on a keyboard but not likely to appear in any other file on the computer.

Conducting a search for this string, for example by typing it into the start search box, will indicate whether the computer's settings enable Windows Search to find mail messages in the store folder.

Hi,

Sleep Mode feature can malfunction in Windows 10 for different kinds of reasons. It can be due to suspected malware infection, an application that you're trying to use that prevents your Windows 10 device from using the sleep mode properly or it can also be due to a driver from a hardware component.

Since you have done the powercfg - lastwake command to check what causes this issue, we can try to explore wake up events using the Event Viewer instead. Here's how:

  1. To open Event Viewer, type event in the Search filed at the taskbar and then select Event Viewer.
  2. In the left-hand pane, drill down to Event Viewer (Local) > Windows Logs > System. We’re going to filter the information to the stuff we need to look at. Right-click the System log, click Open and choose Filter Current Log at the right panel of the window.
  3. Click the Filter Current Log. A new window will pop-up, click the Event sources drop-down menu, select the Power-Troubleshooter option and then click OK.
  4. Back in the main Event Viewer window, you’ll see when the computer is waking up from a low-power state. In the new filtered view at the lower part under General or Details, you can scroll through every instance where your computer has woken over the duration of the log.

From there, we will be able to check if there any difference from what you saw in the powercfg - lastwake command. If there are any changes, let us know so we can proceed with the troubleshooting steps.

jennifer bri say:

Apologies for the delayed response. Since the Event Viewer is also not showing you any results, we can adjust the settings of your device to prevent them from waking up your PC. Here's how:

  1. Open Device Manager by pressing the Windows key, typing Device Manager, and then pressing Enter.
  2. In the Device Manager window, locate the device you want to prevent from waking your computer such as your keyboard, mouse, network adapters or power buttons. Right-click the device and select Properties from the context menu.
  3. On the Power Management tab of the device’s properties window, disable the Allow this device to wake the computer option and then click OK.
  4. While you’ve got Device Manager open, go ahead and disallow any other devices you don’t want waking up your computer. When you’re done, you can exit Device Manager.

Do let us know if you need additional assistance.

eileen gal say:

We appreciate the information that you've provided, Jakob. We recommend that you enable Allow the computer to turn off this device to save power using Registry Editor and Windows will turn off the device when not in use so that power will not be wasted. Please follow the steps below:

  1.  Go to Device Manager, expand the Mice and other pointing devices tree, right-click on the device and select Properties for which you’re having the issue.
  2. In the Properties window, select Details tab, select the Driver Key from Property dropdown list. Copy the key as shown below. The last portion after \ of the Value so copied is device number which is 0000 in our case – but if the option is grayed out you may see a value of 24.



  3. Type regedit in the Search box, then press Enter to open the Registry Editor. 
  4. Navigate to the location path below:

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\Driver Key

    (Replace the word Driver Key obtained in Step 2.)

  5. In the right pane of this location, find the DWORD named PnPCapabilities. If the DWORD doesn’t exist, you can generate it using right click -> New -> DWORD Value. Double click on the same DWORD to modify its Value data.
  6. You have to input the Value data to 0 in order to allow the computer to turn off the device while it is idle. You may now exit the Registry Editor and restart your computer to make changes effective.


Note: This solution will modify the registry. Serious problems may occur if you modify the registry incorrectly. Prior to attempting these steps, you should back up the registry for restoration in case problems will occur.
