Changing tab order for controls based on conditions in previous - Microsoft Community

andrewbartell say:

Changing tab order for controls based on conditions in previous controls

I am trying to change the tab order for controls within a word document based on responses to the initial control.  The document is being filled out by different entities within my company. The first control is a text form field asking for the name of the entity.  Certain entities will not need to complete every control in the document so I want the tab order to change based on the entity's name.  Is there a way to change the tab order for controls based on the response to a previous control?  Any push in the right direction would be appreciated.

doug robbins - mvp office apps & services (word) say:
What type of controls are you using?  If they are formfields as used in a document that is protected for filling in forms, you can use a macro that runs on exit from the formfield to move the focus to the required field depending upon the .Result of the formfield.

andrewbartell say:

The first control is a legacy text formfield.  What would be the syntax for a macro that directs a user to control "x", then to control "y" and so on if they fill out "z" in the initial formfield?  I have assigned a bookmark to each control.  The tab order throughout the document will be different based on responses to the initial text formfield.

Thank you very much for responding.

doug robbins - mvp office apps & services (word) say:

Set the following macro to be run on exit from the Initial FormField:

Sub ExitInitialFormField()

If ActiveDocument.FormFields("TheInitialFormField").Result = "A" Then

                Selection.GoTo What:=wdGoToBookmark, Name:="A1"

Else

                Selection.GoTo What:=wdGoToBookmark, Name:="B1"

End If

End Sub

Then, set the following macro to be run on exit from each of the other formfields

Sub Taborder()

Dim strCurFFld As String

If Selection.FormFields.Count = 1 Then

                'FormField is not a textbox but a check- or listbox

                strCurFFld = Selection.FormFields(1).Name

ElseIf Selection.FormFields.Count = 0 And Selection.Bookmarks.Count > 0 Then

                'FormField is a Textbox

                strCurFFld = Selection.Bookmarks(Selection.Bookmarks.Count).Name

End If

Select Case strCurFFld

                Case "A1"

                                Selection.GoTo What:=wdGoToBookmark, Name:="A2"

                Case "A2"

                                Selection.GoTo What:=wdGoToBookmark, Name:="A3"

                Case "A3"

                                Selection.GoTo What:=wdGoToBookmark, Name:="A4"

              'etc.

                Case "B1"

                                Selection.GoTo What:=wdGoToBookmark, Name:="B2"

                Case "B2"

                                Selection.GoTo What:=wdGoToBookmark, Name:="B3"

                Case "B3"

                                Selection.GoTo What:=wdGoToBookmark, Name:="B4"

                'etc.

End Select

End Sub
