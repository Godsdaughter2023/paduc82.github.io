oreveredrist say:

I'll just stick this one thing that leaped out at me while I look your post over in more detail:  your USB flash drive has to use GPT instead of MBR.  So, just after "clean", you should have "convert gpt".

EDIT 1:  based on some research I did several months ago, in general, the automated tools to make flash drives bootable tend not to handle GPT very well.  So, your best bet is the manual list of steps (plus "convert gpt") you listed.

EDIT 2:

- You don't need to force an mbr onto the flash drive.  Once you use the "convert gpt" command, that should be all you'll need.  At that point, you should change back to UEFI Only (and Secure Boot if your system will handle it).  When you boot the the flash drive, you'll have to pick the UEFI version of the device.

- I don't know why you can't boot from a USB 3.0 port.   

- As to why the Windows install wouldn't actually install to the drive it partitioned/formatted, I don't know.  Hopefully, it will work with a GPT flash drive instead of an MBR one (though, that's a pretty weak statement).  Once you're booted into the GPT flash drive and start the install, clear off all the partitions on the hard drive and tell the install to use that space.  It should create all the necessary partitions on its own.  Once you're up and running, go to Disk Management (right click on the Start Button to see it) and right click on the drive you installed to.  Select Properties | Volumes and it should say "GUID Partition Table (GPT)" in the Partition Style area.

bernie deitrick say:

All numbers are divisible by 3 - so I will assume you mean numbers that are multiples of 3.

Select cell C2, and use Conditional Formatting with the formula option, using the formula

=MOD(C2,3)<>0

Select Red as the fill.

Then copy C2, and paste special /  formats over the rest of column C.

You could also use a column of formulas like

=IF(MOD(C2,3)<>0, "Error","Not error")

Or use Data Validation   Custom, with the same formula - that will prevent any number that is not evenly divisible by 3 from being entered into column C.
