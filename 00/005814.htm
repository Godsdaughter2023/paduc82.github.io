Thank you for posting to the forum regarding the issue you have run into when attempting to update your phone.  We understand your excitement to get the most recent update and hope that we can get past the error you are receiving.

 

Issues with updates can be caused by multiple things from the computer your are using, to the Zune software you have installed or the phone itself.  We will do our best to try and narrow down where your particular issues lies.

 

The first thing we suggest you check is the date and time on your computer.  Click the time in the bottom right corner and make sure date, time and time zone are accurate.  If they are off at all, please correct the information and move to the next step.

 

Next, we would suggest you check your PC for any available updates.  Go to the following link and be sure to click on the "check online for updates from Microsoft update" option,  http://update.microsoft.com/microsoftupdate.  Run and install any and all updates found.  Once this is complete, restart your computer and try the update again.

 

If you still have this update issue, uninstall and reinstall the Zune software to your computer.  Use this article to uninstall the software, http://support.xbox.com/en-US/zune/software/remove-and-reinstall-zune.  Once the software has been removed, install from the following location, http://www.microsoft.com/en-us/download/details.aspx?id=27163.

The final thing we can suggest you try on this computer is to try a clean boot, http://support.microsoft.com/kb/929135.  This should run your computer using the minimal number of drivers and will often times allow us to determine if the issue is some software program running on the computer.

 

If you cannot make any update progress with the above steps, try connecting to another computer.  If the update fails with the same error on the alternate PC, it is likely the issues is with the phone and not the computer.

 

Please let us know if you have any other questions or concerns.
