milagrosm2020 say: Hi. FulgarOdinvolve

I welcome you. In this case, try to uninstall the network adapter driver, and then restart the computer and have Windows automatically install the latest driver. Use this procedure if the network connection has stopped working correctly after a recent update.

Before uninstalling it, make sure you have a backup of the drivers available. Visit the manufacturer's website of your PC and download from there the latest driver for the network adapter. If the computer cannot connect to the Internet, you will have to download a driver from another computer and save it to a USB flash drive to install it on your PC.

You will need to know who is the manufacturer of the PC and the name or number of the model.

1. In the search box of the taskbar, type Device Manager and then select Device Manager in the list of results.

2. Expand Network Adapters and search for the device's network adapter.

3. Press and hold (or right-click) on the network adapter and then select Uninstall device> Remove the driver software from this device> Uninstall box.

4. After uninstalling the driver, select the Start button> Start / Shut Down> Restart.

When the computer restarts, Windows will automatically search for and install the network adapter driver. Check if the connection problem has been solved. If Windows does not automatically install a driver, try installing the backup drive that you saved before uninstalling it.
