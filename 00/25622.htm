john w. vinson say:

I think your best solution is to programmatically assign a "row number" to each record as you import it. This could be done by linking to the text file and running an Append query into a table which includes an Autonumber field, or (probably safer) using VBA code to open the external file and use a loop to increment a numeric ID and insert the record into your table.

Then you would have a "handle" to unambiguously identify "the previous record"; a Self Join query would do it easily:

SELECT PrevRow.*
FROM yourtable AS CurrRow INNER JOIN yourtable.PrevRow
ON PrevRow.ID = CurrRow.ID  - 1
WHERE CurrRow.Field3 = "Permission Denied";

Without the numeric ID, though, there is - again - nothing in an Access table nor in a linked text file to indicate which "row" is which; all you can use is the actual contents of the data.
