Server Response: -ERR [SYS/TEMP] Server error - Please try again later.

This is a temporary system error at the server, so no amount of repairing, uninstalling or re-installing the program will have any effect on it.  Since the server repeatedly asks you to enter the password, there seems to have been a problem with authentication. If it doesn't come right in an hour or two, this can often be resolved by changing the password, then entering the new password in Windows Live Mail.
