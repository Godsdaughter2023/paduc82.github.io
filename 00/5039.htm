ken sheridan say: You can do it with a UNION ALL operation in a subquery, and then Sum the column returned by the subquery:

SELECT SUM(TotalAmount) AS GrandTotal
FROM
    (SELECT TotalLarge AS TotalAmount
     FROM qryDirectoryOrderTotal
     UNION ALL
     SELECT TotalMemberHomes
     FROM qryTotalMemberHomes
     UNION ALL
     SELECT TotalNonmemberHomes
     FROM qryTotalNonMemberHomes
     UNION ALL
     SELECT DoNotSendHomeCopy * -1
     FROM qryDirectoryOrdersHomeReduction
     UNION ALL
     SELECT TotalLeaders
     FROM qryDirectoryOrderTotalsLeaders);

Note the multiplication by -1 to include a negative value in the summation.
