toddallcock say:

There are two types of WiFi connections, "infrastructure", for connecting to network devices like routers, and "ad-hoc" for connecting to other devices, like phones and computers.

 

Windows Phones only connect to infrastructure connections, not ad-hoc, like Joiku creates.

 

Some newer Android devices can connect to ad-hoc, but most, still only connect to infrastructure connections.

 

Most modern phones' internet sharing apps create infrastructure connections instead of ad-hoc these days, for greater compatibility with other devices.
