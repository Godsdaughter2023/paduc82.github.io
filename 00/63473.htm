miekiemoes say:
Hi,

The AdvrCntr6.dll isn't a microsoft file, but comes installed with nero. It should be located in the following directory: C:\Program Files\Common Files\Nero\AdvrCntr6
If that file is indeed there and you still get that error, then it might be an incompatibility issue with other programs. For example, it has been reported that Airfoil For Windows could be the culprit:
http://forum.my.nero.com/index.php?showtopic=14822
You can find instructions there how to solve this.

If you don't have Airfoil for Windows (so it won't be the culprit) - I suggest you uninstall Nero again, reboot, then run the Nero General CleanTool Utility: http://www.nero.com/eng/support/service/nero-clean-tool.php
And then reinstall the latest Nero Media software from the manufactures website instead:
http://www.nero.com/eng/downloads/
