3. Expand and delete all credentials related to Microsoft Office.

suzanne s. barnhill say:

Make sure that the template on which your document is based does not have "Automatically update document styles" enabled (Developer | Templates | Document Template); with this setting, every time you open the document, the formatting of every paragraph will be reset to the formatting of the style in the template. This means that style changes you make in the document cannot be preserved unless you also save the changes to the template.

At the same time, make sure that the style definition itself is not set to update automatically because that causes every paragraph in the style to be changed if you change the formatting in any one paragraph in that style. This can be helpful when you're setting up a template and experimenting with formatting but is not sustainable when you start using the template/document because inevitably there will be some paragraphs to which you will want to make one-off modifications without affecting the rest.

dminnc say:

Thank you, Suzanne. I appreciate you taking the time to respond. That is good info, and I will verify both of these settings. I haven't experienced the behaviors you describe, though, so I don't think I have anything set to update automatically.
