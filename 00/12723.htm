macropod say:

If you open a document then go to Developer|Document Template>Templates and check the 'Automatically update document Styles' option, Word will reset all that document's Styles to those of its template when the document is next opened. Naturally, this won't affect any Styles that exist in the document but not the template; neither will it correct all hard-formatting that's been applied to content in one Style to make it look like something else.

If you close Word, then delete or rename Word's Normal.dotm file, usually found in:
C:\Users\%Username%\AppData\Roaming\Microsoft\Templates\

Word will create a new one with all the factory defaults next time Word starts. All new documents will be created with the default Styles.

By combining both approaches, that'll take a given document back to as close to the factory defaults as you'll get it without a macro that also cleans up any residual hard-formatting.
