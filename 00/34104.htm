diana d say:

Hi,

Windows Meeting Space replaces Microsoft NetMeeting. It is available only to users of the Windows Vista operating system, since it's built on Vista's peer-to-peer networking technology and uses Vista features, such as WS-Discovery.

 

Microsoft recommends using newer applications such as Windows Meeting Space, Remote Desktop Connection, Remote Assistance, Windows Live Messenger and Microsoft Office Live Meeting in place of NetMeeting.

 

However in Windows7, you can use Windows XP Mode (a virtual XP computer running within Windows 7) to regain all NetMeeting functionality.

    portions of a video to be removed frame by frame.
    This is a very complicated and time consuming task.
    and it requires expensive advanced software.

    It may be possible to accomplish this with Movie
    Maker if time is no object. Basically you would have
    to save each frame of the video as a still photo...

    Then the photos would have to be imported into
    advanced image editing software to remove the
    background one at a time. When all the frames
    are prepared you would reimport them into Movie
    Maker and create a video.

    The following article shows what is involved in the
    Rotoscoping process.

    Rotoscoping in Photoshop
    http://www.youtube.com/watch?v=fpEqWKJj7Yc



    A YouTube search for Rotoscoping will find more
    tutorials if you are interested.
