Change the size of virtual memory


1. Open System by clicking the Start button, clicking Control Panel, clicking System and Maintenance, and then clicking System.
2. In the left pane, click Advanced system settings. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
3. On the Advanced tab, under Performance, click Settings.

6. Under Drive [Volume Label], click the drive that contains the paging file you want to change.
7. Click Custom size, type a new size in megabytes in the Initial size (MB) or Maximum size (MB) box, click Set, and then click OK.


Increases in size usually don't require a restart, but if you decrease the size, you'll need to restart your computer for the changes to take effect. We recommend that you don't disable or delete the paging file.

  • The PC and Xbox are both connected via ethernet cable to the same router.
  • The XBox and XBox 360 are both showing as active network objects in the Network tool on the PC.
  • The XBox network test for connections to both XBox Live and the PC say that the connection is good.
  • The Extender can be installed/uninstalled from the network tool on the PC without issue.
  • Windows Media Center, regardless, will not recognize the installed extender or install it when uninstalled from network
  • These factors disallow the use of Windows Media Center from the XBox360.
  • Also I have added windows media center extenders to my list of approved objects on windows firewall.
I solved this, but not entirely sure how/why it worked this time over the last time I tried the same.  Shortly after posting this I manually installed the extender using the network tool on windows.  Then I told the XBox to connect to Windows Media Center.  While the XBox attempted to connect, I went into settings>extender, brought up the extender icon, then attempted to configure from there.  Before I did this I attempted to configure the extender from a few other parts, like tasks>extender and settings>extender but not on the specific extender.  So if anyone else has a similar issue, install it manually, then configure it rather than trying to have WMC install it.
