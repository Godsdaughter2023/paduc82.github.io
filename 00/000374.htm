Hello

Welcome to Skype Community forum!

First, make sure you and the person you're calling are using the latest version of Skype. Ensure that both of you are using a strong Internet connection, if Skype detects a weak connection it will reduce the quality of the call. As a workaround, you may try to update your audio driver. Here's how:

  • Select the Windows logo/Start button, type Device Manager, and select it from the list of results.
  • Expand one of the categories to find the name of your device, then right-click (or tap and hold) it, and select Update Driver.
  • Select Search automatically for updated driver software.

If these steps don't work, try reinstalling the driver:

  • Open Device Manager, right-click (or tap and hold) the name of the device, and select Uninstall.
  • Restart your PC and Windows will attempt to reinstall the driver.

If the above information does not work, please do not hesitate to reply to this post. Thank you!

Text box rotation option in Word 2007 - Microsoft Community

dawnc138 say:

Text box rotation option in Word 2007

Hi!

Is there any way to rotate a text box in Word 2007? I can find the rotation option, but it's grayed out ... like there is some setting or something I'm not seeing. I can rotate a picture, but not this darn text box. I remember being able to do so. What am I missing?

jay freedman say:

No, Word 2007 can't rotate a text box. That capability first appeared in Word 2010.

You can insert a rectangle from the Insert > Shapes gallery and add text to it. The rectangle can be rotated, but the text will remain horizontal. That also changed in Word 2010.

doug robbins - mvp office apps & services (word) say:
You could use a one cell Table as even as far back as Word 2003, it was possible to rotate the text in a table by plus or minus ninety degrees.
