candaceruiz say:

Macro for Word 2013 Export to Excel

I would like to export comments made on a Word document into Excel. I need the code for exporting Word 2013 comments into an Excel spreadsheet.  I do not have sections numbers, but would like the Comment #, Author, comment and date.


Thanks, Candace

doug robbins - mvp office apps & services (word) say:

Use the following code in a macro in Word:

Dim xlapp As Object
Dim xlbook As Object
Dim xlsheet As Object
Dim i As Long
Set xlapp = GetObject(, "Excel.Application")
If Err Then
    Set xlapp = CreateObject("Excel.Application")
End If
On Error GoTo 0
Set xlbook = xlapp.Workbooks.Add
Set xlsheet = xlbook.Worksheets(1)
With xlsheet.Range("A1")
    .Offset(0, 0) = "Comment #"
    .Offset(0, 1) = "Author"
    .Offset(0, 2) = "Comment"
    .Offset(0, 3) = "Date"
End With
With ActiveDocument
    For i = 1 To .Comments.count
        xlsheet.Range("A1").Offset(i, 0) = i
        xlsheet.Range("A1").Offset(i, 0) = .Comments(i).Author
        xlsheet.Range("A1").Offset(i, 0) = .Comments(i).Range.Text
        xlsheet.Range("A1").Offset(i, 0) = .Comments(i).Date
    Next i
End With
