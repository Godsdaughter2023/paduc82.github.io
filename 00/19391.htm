todtown say: Okay. Yes, that helps. You can't do this with a formula. You can do it with a macro. If you're still interested, here's how it works.

You'll need a table somewhere to put the text you want to display for each value the user clicks on. Something like this:

ValueA   ValueB
Dog       Canine. 
Cat         Fur Ball
Mouse    Was eaten by the Cat

Now you are going to put some code in the Worksheet SelectionChange Event. Here's the code:

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Target.Row > 1 And Target.Row < 5 Then
        If Target.Column = 1 Then
            ActiveSheet.Range("C1").Value = Evaluate("=VLOOKUP(""" & Target.Value & """,$I$2:$J$4,2,FALSE)")
        End If
    End If
End Sub

In the code, $I$2:$J$4 is the range where your table is located. The target.row is greater than 1 and less than 5 and the target.column is Column A (column 1). So you'll need to adjust these to fit your actual scenario.

Now you need to put this code in the worksheet module. With Excel open, click Alt+F11 to bring up the VB Editor. In the top-left side of the screen there is a window that says Project-VBAProject and has a list of your open workbooks. Find the workbook and the worksheet where you want this to happen and double click on the worksheet name to bring up the module. Copy and past the above code to that   module. Now close the VB Editor by clicking the X in the top-right corner of the screen. 

Now when the user clicks the value in Cell A2, A3 or A4, the corresponding value from the table will appear in Cell C1.
