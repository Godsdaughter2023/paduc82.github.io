Windows Live Mail contact list - Microsoft Community

lionelhead say:

Windows Live Mail contact list

How do I copy the Contact List in Windows Live Mail to a USB stick?
monica cam say:

Hi Lionel,

To export Windows Live Mail contacts to your USB disk, kindly follow the instructions below:

  1. Open Windows Live Mail.
  2. Click Contacts folder, click Export button and select comma separated values (.CSV).
  3. In the CSV Export window, click Browse and select the Desktop folder.
  4. Type WLMContacts in the file name and then click the Save button.
  5. Click Next and select all the fields that you want to export.
  6. Click Finish button and wait for the export process to be completed.
  7. Close Windows Live Mail.
  8. From the File Explorer, click Desktop folder.
  9. On the Desktop folder, look for the WLMContacts file.
  10. Copy the file to the USB disk.

Let us know how it goes.

ralf_g say:

Apart from what Sangeeta already suggested there are 2 other things you can check for.

First check that Windows is not creating System Restore points on your recovery drive. It is unnecessary for that drive and Restore points will continuously eat up space. You only really need to have System protection enabled on your system drive (C:). Go into Control Panel and click on System then select System protection.

Follow the highlighted steps below to disable System Restore on your Recovery(Q:) partition only. I don't have a recovery partition listed in mine so just use this as a guide.

The second thing to check is that there is no virtual memory set for that drive. This is also unnecessary. Starting from the same place as above except select Advanced system settings. Follow the highlighted steps below and check that the setting for your Recovery(Q:) drive is set to None.

That should take care of it.
