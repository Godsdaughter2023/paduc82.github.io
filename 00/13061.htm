john korchok say:
  1. Select the slide where you want to show a comparative list.
  2. On the Home tab, choose Layout and select either the Two Content or Comparative layouts (Comparative has a subhead over each column, Two Content doesn't).
  3. Format both text boxes with the same bullet style and type your list.
For more exact comparative lists, you often want to have the opposing points beside each other. In this case, use a Title and Content layout, but insert a 2-column table, with separate bullet points in each row. This way, the contrasting points remain aligned with one another.
