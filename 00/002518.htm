If you open a form in dialog mode; e.g.,

    DoCmd.OpenForm "ProgressBar, WindowMode:=acDialog

... then code in the calling form stops until the dialog form is closed or hidden.  This is the chief characteristic of dialog mode.

To me, it doesn't make sense to open your progress bar form in dialog mode, because a progress bar usually is updated to reflect some process that is going on behind the scenes.  Opening the progress bar form in dialog mode would keep that process from, well, progressing until the form is closed.

chip pearson say:

I think it would be easier to examine the result of the VLOOKUP to see if the item was found. You can call VLOOKUP (and other worksheet functions) from VBA in one of two ways. The first is to go through the WorksheetFunction object, as you do in your code:

 

 

Dim V As Variant
On Error Resume Next

Err.Clear
V = Application.WorksheetFunction.VLookup("aa", Range("A1:B5"), 2, False)
If Err.Number = 0 Then
    Debug.Print "success"
Else
    Debug.Print "failure"
End If

 

In the code above, if the value "aa" is not found, an error 1004 will be raised. You must handle the error trapping with On Error statements and the Err object.

 

The other way is to omit the WorksheetFunction class:

 

 

Dim V As Variant
V = Application.VLookup("aa", Range("A1:B5"), 2, False)
If IsError(V) Then
    Debug.Print "failure"
Else
    Debug.Print "success"
End If

 

Here, if the value "aa" is not found, no error is raised but the variable V gets an error value (which is why you must declare it as a Variant) that can be tested with IsError or IsNA:

 

Dim V As Variant
V = Application.VLookup("aa", Range("A1:B5"), 2, False)
If Application.IsNA(V) Then
    Debug.Print "failure"
Else
    Debug.Print "success"
End If

 

Coding in this manner takes the error handling out of the formula and places it in the VBA code, which, I think, is easier to understand in this context. 

 

To adapt this to your code, try something like

 

    Dim V As Variant
    V = Application.VLookup(c, tblRange, 5, False)
    If IsError(V) Then
        '''''''''''''''''''''''''''
        ' code for value not found
        '''''''''''''''''''''''''''
    Else
        .Range(exstPOCol & c.Row) = V
    End If
