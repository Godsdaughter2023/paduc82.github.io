joeu2004 say:

Mark wrote:

I want to list all of the possible combinations of set of numbers using the rules of Pick Five numbers listed 1-10 & Pick One number listed 1-5.

One way....

Enter 1, 2, 3, 4, 5 and 1 into A6:F6.  Enter 10 into B1 and 5 into B2.

Enter the following formulas:

B3:  =B2*COMBIN(B1,5)

A7:  =IF(AND($F6=$B$2, B6=$B$1-3, C6=$B$1-2, D6=$B$1-1, E6=$B$1), A6+1, A6)

B7:  =IF(AND($F6=$B$2, C6=$B$1-2, D6=$B$1-1, E6=$B$1), IF(B6=$B$1-3, A7+1, B6+1), B6)

C7:  =IF(AND($F6=$B$2, D6=$B$1-1, E6=$B$1), IF(C6=$B$1-2, B7+1, C6+1), C6)

D7:  =IF(AND($F6=$B$2, E6=$B$1), IF(D6=$B$1-1, C7+1, D6+1), D6)

E7:  =IF($F6=$B$2, IF(E6=$B$1, D7+1, E6+1), E6)

F7:  =MOD(F6,$B$2)+1

Copy A7:F7 down through A1265:F1265 (1260 rows, based on B3).

Since the number of columns and rows is specific to pick 5 of 10 and pick 1 of 5, the formulas can be simplified, namely:

1. Replace $B$2 with 5.

2. Replace $B$1 with 10, $B$1-1 with 9, $B$1-2 with 8, and $B$1-3 with 7.

I would use a VBA procedure to generate a more general solution, allowing B1 and B2 to be variables (not always 10 and 5).

Note that if you are using an earlier version of Access you might find that the colour of some form objects such as buttons shows incorrectly and you will need to  amend the form design accordingly.

This little demo file illustrates the use of a number of age functions.  To return a person's current age you'd use the following:

Public Function GetAge(varDoB As Variant, Optional varAgeAt As Variant) As Variant

    If IsMissing(varAgeAt) Then varAgeAt = VBA.Date
   
    GetAge = DateDiff("yyyy", varDoB, varAgeAt) - _
        IIf(Format(varAgeAt, "mmdd") < Format(varDoB, "mmdd"), 1, 0)

    ' adjust for leap year if necessary
    If IsLeapDate(varDoB) And Month(varAgeAt) = 2 And Day(varAgeAt) = 28 And Not IsLeapDate(varAgeAt + 1) Then
        GetAge = GetAge + 1
    End If
   
End Function

which calls the following to cater for dates of birth on 29 February in leap years:

Public Function IsLeapDate(ByVal dtmDate As Date) As Boolean

    If Month(dtmDate) = 2 And Day(dtmDate) = 29 Then
        IsLeapDate = True
    End If
   
End Function
