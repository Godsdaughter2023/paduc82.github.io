mohammadheydari say:

How to separate letters from numbers?

How to separate letters from numbers?

exampel : 09121089048 ( dsdfsf ) 09194994490 ( eretrtyhg) =   0912108904809194994490

jeeped say:
With 09121089048 ( dsdfsf ) 09194994490 ( eretrtyhg) in A1 you can get hte firs number with,

=TRIM(LEFT(SUBSTITUTE(A1,"(",REPT(" ",99)),99))

... and the second number with,

=TRIM(RIGHT(SUBSTITUTE(TRIM(SUBSTITUTE(SUBSTITUTE(A1,TRIM(RIGHT(SUBSTITUTE(A1,"(",REPT(" ",99)),99)),""),"(",""))," ", REPT(" ",99)),99))

So both of them together would be,

=TRIM(LEFT(SUBSTITUTE(A1,"(",REPT(" ",99)),99))&TRIM(RIGHT(SUBSTITUTE(TRIM(SUBSTITUTE(SUBSTITUTE(A1,TRIM(RIGHT(SUBSTITUTE(A1,"(",REPT(" ",99)),99)),""),"(",""))," ", REPT(" ",99)),99))

Function GetNumbers(rng As Range) As String
Dim x As Long
For x = 1 To Len(rng)
    If Mid(rng, x, 1) Like ["#"] Then
    GetNumbers = GetNumbers & Mid(rng, x, 1)
    End If
Next
End Function
