toddallcock say:

I'm not aware of any freeware to backup a non-touchscreen Windows Mobile device. Having said that, you could use the cloud to backup the device.

 

Create a throwaway Hotmail or Live address, go into ActiveSync on the device, and add a new server source. Use m.hotmail.com as the server, and use the email address (with the "@Hotmail.com") and password as the username and password.  If the Q requires a domain name, use a backslash ("\") followed by your email address.

 

Now sync your device by pressing sync in ActiveSync and wait until the Contacts and Calendar are uploaded.  Now (here's the vitally important part!) in ActiveSync options, set the sync schedule to "Manual". (Because, if some WMDC sync catastrophe with your PC wipes out the data on the device, the next cloud sync would also remove the data from the cloud as well, eliminating your backup. We'll cover what to do if the PC sync goes bad in a moment.)

 

Now, if the sync with the PC works as planned (you select the option to replace data on the PC with data on the device and it all syncs), you can delete the server source from the device, wipe the data from (and close) that Hotmail account, and happily sync with the PC from this point on.

 

If the sync DOESN'T go as planned, and the data on the device is wiped out, follow these instructions carefully: first, remove the account from the device WITHOUT SYNCING FIRST!  You might get a message saying all of the synced data will be removed from the device.  Select "ok" (it was already erased during the PC sync anyway.)

 

Now, re-enter the same Hotmail/Live account you just deleted and sync.  Since this will now be the "first" sync with the server, the cloud data will resync down to the device.  Now, (again making sure we're still set to sync manually) retry the PC sync. The PC will see the synced cloud data as "new" data added since the last sync, and sync it to the PC.

 

I hope that was clear enough to follow.  Feel free to ask any questions that will help clear it up before risking any of your data, if anything isn't clear.
