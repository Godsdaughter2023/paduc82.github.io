suzanne s. barnhill say:

There is a way, but it is a little tricky.

  1. Create a text box using Insert | Text | Text Box | Draw Text Box.
  2. Drag the text box to the desired position at the bottom of the page and size it as desired; add or remove border, set fill color, etc.
  3. When the text box is selected, the contextual Drawing Tools | Format tab will be displayed. On that tab, select Arrange | Position | More Layout Options.
  4. On the Position tab of the Layout dialog, select the desired position Relative to Page or Margins (not Paragraph because you won't know what paragraph it will be anchored to).
  5. On the Wrapping tab, set the wrapping to Square.
  6. Finally, when you get the text box just the way you want it, select it and on the Insert tab, select Text Box and choose Save Selection to Text Box Gallery.
  7. On any given page, to insert your quote text box, just select it from the gallery, then type the desired content.

Would a solution that uses VBA be ok?  If so, this should do it:

Option Explicit

Sub PasteToEachSlide()
' Assumes you've added the text box or other shape you want
' on Slide 1, formatted it to taste and that it's selected

    Dim oSh As Shape
    Dim x As Long
    
    Set oSh = ActiveWindow.Selection.ShapeRange(1)
   
    For x = 2 To ActivePresentation.Slides.Count
        oSh.Copy
        ActivePresentation.Slides(x).Shapes.Paste
    Next
    
End Sub
