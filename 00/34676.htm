andre for directly say:
Hi BilalAnsari47

My name is Andre Da Costa; an Independent Consultant, Windows Insider MVP and Windows & Devices for IT MVP. I'm here to help you with your problem.

On Windows 10 and Windows 7, both are virtual memory, where they use your local hard disk as SSD as a substitute for physical (volatile) - which stores programs and files when you start the computer.

The difference is that Windows 10 and Windows 7 allocates it differently, Windows 7 uses more while Windows 10 uses less. This is due to changes in efficiency over the past 10 years.

When Windows 7 came out, systems didn't have a lot of RAM and so, the operating system assumed at some point or another, large pagefile would be needed when you ran out of main memory.

If you want to increase it, follow the steps in the following article:

How to increase Page File size or Virtual Memory ... - The Windows Club
https://www.thewindowsclub.com/increase-page-fi...

Information in the above link is sourced from a trusted Microsoft MVP blog.

Available memory is what the computer uses to operate.

Available memory refers to how much RAM is not already being used by the computer.
