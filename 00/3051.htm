m h g say:

IOW you have extra text in the 1st column but want to find entries in the 2nd column in the 1st column, and indicate when an entry in the 2nd column lacks a match in the 1st column.

C2:  =MATCH(1,COUNTIF(A2,INDEX(B$2:B$1000&" *"),0),0)

D2:  =MATCH(B2&" *",A$2:A$1000,0)

Fill C2 down into as many rows as there are entries in column A. Fill D2 down into as many rows as there are entries in column B. The resulting column C values are numbers when the column A entry has a match in column B, and #N/A when the column A value doesn't have a match in column B. The resulting column D values are numbers when the column B value has a match in column A, and #N/A when the column B value doesn't have a match in column A.
