I'm sorry, you will have to program the VBA procedure achieving this. Supposing it is the Text9 field you want filled in automatically, and the new task is inserted above the selected line, the code could be as follows:

Sub Auto_Text9

set Nutask=activeproject.tasks.add(before:=activeselection.tasks(1).ID)

nutask.text9=nutask.outlineparent.text9

end sub

I hope this will inspire you to roll your own.

On masamiki.com you can find instructions on how to find the code modules.

How to apply condition formatting to one cell depending on the value - Microsoft Community

jmb_2712 say:

How to apply condition formatting to one cell depending on the value in another cell - Eg color code cell A1 Red, Green or Yellow depending on whether the value in cell A2 is 1, 2 or 3 respectively

For example, I need to color code cell A2 Red, Green or Yellow depending on whether the value in cell A1 is 1, 2 or 3 respectively.  This was a piece of cake in Excel 2003 but I can't find a way to do it in the 2007 version.

 

If the above is possible, would the solution also work if the values in cell A1 were "Yes", "No" or "N/A". 

 

Thank you in advance for your help.

 

Jean-Marie

hansv mvp say:

Select A2.

 

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Click "Use a formula to determine which cells to format".

Enter the formula =A1=1 or =A1="Yes" depending on which version you want.

Click Format...

Specify red text color or fill color, then click OK twice.

 

Repeat the above steps, with =A1=2 or =A1="No" and green.

Also repeat them with =A1=3 or =A1="N/A" and yellow.

roady [mvp] say:

You can make the Reading Pane smaller ;-)

When the email format is Plain Text, then it should wrap at 76 character. If the message was sent in the HMTL format, then it will take a much space as the text container allows which is usually the width of the window that it is in.
