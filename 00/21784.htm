andreepin say:

Hi David,

If print options are set to Black and white or Draft quality — either on purpose, or because the workbook contains large or complex worksheets and charts that caused draft mode to be turned on automatically — cell shading cannot print in color.

  1. On the Page Layout tab, in the Page Setup group, click the Page Setup dialog box launcher.

  2. On the Sheet tab, under Print, make sure that the Black and white and Draft quality check boxes are cleared.

Note: If you do not see colors in the worksheet, it may be that you are working in high contrast mode. If you do not see colors when you preview before you print, it may be that you do not have a color printer selected.

Keep us updated if anything comes up so we can further assist you.

recover deleted emails - Microsoft Community

a. user say:

recover deleted emails

Is it possible to recover deleted items (more than 30 days) for a specific mail-box?

We have E3 subscription.

Also, where do I check what is our retention policy?

Thanks,

Ashok

poppy yang msft say:

Hi Ashok,

Q1: Is it possible to recover deleted items (more than 30 days) for a specific mail-box?

A1: The short answer is yes.

For data in Exchange mailboxes, generally, after some emails are deleted, they are moved to Deleted Items folder.

The duration that deleted items are retained in this folder is based on the deleted item retention period configured for the mailbox. Emails will be moved to the “recover deleted items” folder after 30 days by default, during this period, end users can still restore their emails by themselves. Then emails will go to “purge folder” after 14 days, after that, only administrator can restore the messages. And after another 14 days, emails will be permanently deleted and cannot be restored. 
We can enable the In-Place hold s service to prevent data loss. If we have enabled the In-Place hold before deleting those emails, we can use the In-Place eDiscovery  to recover them no matter how long it has been since they were deleted.


Only admins can restore these deleted emails via creating an ediscovery search in office 365 eac  (exchange admin center) to find these emails and export them into a pst file.

create an in-place ediscovery search.

export ediscovery search results to a pst file.

If admins cannot search out deleted emails via ediscovery, this means these emails are permanently deleted. we cannot restore them.

If anything is unclear or you encounter any issue when performing the steps mentioned above, please feel free to post back.

 

Q2: where do I check what is our retention policy?

A2:
1. Log into Navigate to exchange admin center, navigate to Recipients > Mailboxes.

2. In the list view, select the mailbox, and then click Edit .

3. In User Mailbox, click Mailbox features.

4. In the Retention policy list, it is your retention policy.

Regards,
poppy

prevent word from hyphenating - Microsoft Community

9bill say:

prevent word from hyphenating

With hyphenate this story checked, how do I prevent a specific word from being hyphenated?

--Bill

rohn007 say:
The standard approach is in this tip:
Turning Off Paragraph Hyphenation

http://word.tips.net/T000916_Turning_Off_Paragraph_Hyphenation.html

Summary: Need to make sure that a particular paragraph never has any hyphenated words in it? You can make sure that Word won't automatically add any hyphens by following the easy steps in this tip. (This tip works with Microsoft Word 97 | 2000 | 2002 | 2003 | 2007.)




I also found this old tip. 

THE ZERO WIDTH NON-BREAKING SPACE

While it’s easy to tell Word that you want to put a hyphen in a particular place, using the Ctrl- “soft hyphen”, it’s deucedly difficult to tell Word that you don’t want to permit a hyphen in a particular place. Yes, you can (at least in theory) go through the whole document using the “Manual” button mentioned in the preceding section, but the minute you change something in the document, you have to scan it all over again. Blech.


Jack Lyon at The Editorium came up with a great hack. It uses a Unicode character called the Zero Width Non-Breaking Space, or “ZWNBSP” to cognoscenti.  (I talk about Unicode in WOW 7.53, http://www.woodyswatch.com/office/archtemplate.asp?v7-n53 .) If you put a ZWNBSP where Word is sticking hyphens, the hyphens go away.


If you have the “This is unbelievable.” document hanging around that I mentioned in the preceding section, try this. Hit enter a few times, then type


FEFF


(that’s the Unicode value for a ZWNBSP). Immediately hit Alt+X. That converts FEFF into a character, which you can see by clicking Tools | Options | View and checking the box marked All.


Select that now-visible character (which looks like a box within a box), copy it, and put it between the “un” and the “believable” in the first “unbelievable” that Word automatically hyphenated. The hyphen disappears, eh?


Play with it a bit and I think you’ll appreciate how very cool this trick is.


{EQ} WORKS, TOO!


I have a different way for preventing hyphenation. It works quite differently from Jack’s method, and there are pro’s and con’s to both. If you find yourself in a Herculean battle with the Word Hyphenation Gods, this one’s worth trying, too:


  • Select the word you don’t want Word to hyphenate - double-click on it, say.
  • Cut the word (Ctrl+X or Edit | Cut)
  • Click Insert | Field, pick Eq. (In Word 2002, you then need to click the button marked Field Codes.)
  • Click after the EQ, and paste the word (Ctrl+V). Then click OK.

You’ll end up with a field in your document that produces whatever word you selected. If you are trying to keep Word from hyphenating “unbelievable”, for example, you’ll have the field:


{EQ unbelievable }


and it will show up in the doc as, uh, unbelievable.


I think it’s easier to work with the {EQ} field than with the ZWNBSP - you don’t get a red squiggly underline, and it’s easier to select the whole word (which you probably want to do) - but you can judge for yourself.

mike a. porter say:

copy list to another site

I would like to copy a list from one Sharepoint site to another but none of the videos or pages I've seen look like the Sharepoint I am using. We just started using Sharepoint so I think I am using Sharepoint online.

neha singh msft say:

Hi Mike,

When you add a list to a SharePoint site, you're using a list template. Saving list templates let you reuse customized lists. A template saves fields, formulas, and formatting, and optionally, data as well. Templates can be downloaded for archiving or uploaded to another site collection.

If you are using the site connected to group i.e. Office 365 group site, then these sites do not currently support list templates.

For other SharePoint Online sites and if you are the Global admin, first enable scripting to use list templates. For more info about how to enable scripting, see Turn scripting capabilities on or off.

Once you’ve enabled the custom script, please follow the steps in this article under the section Online,2016,2013 to know how to Create a list template, Download a template and Upload a template to the list template gallery in SharePoint Online.

After following the last step from the article i.e. Upload a template to the list template gallery in SharePoint Online, go to the site, click Settings>Site Content> New>App.

 

In the search box, type the name of the list template which you saved and from the search result click on the list template. It’ll ask you to provide a new name. After giving the name, you can use this list.

 

Please note, if you are using the Office 365 group site, then this feature is currently not available.

Reference: Manage list templates

 


Let me know if you have any doubt.

Thanks,

Neha
