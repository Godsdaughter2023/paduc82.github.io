sami_ say: I went through the list but nothing helped. But I did manage to fix the problem in the end.

After I did not find the solution in the list, I had a go with my favorite power debugging tool, the SysInternals Process Monitor. I set the filter to WWAHost.exe which is the process that crashes. Just before it crashes, it is trying to access the HKCR registry key root which fails with Access Denied.

I went to Regedit and gave the "ALL APPLICATION PACKAGES" user read access to the HKEY_CLASSES_ROOT key. After that everything started working, no more crashes.

So it seems that the Windows 8.1 upgrade did not set the registry permissions correctly, and consequently all modern apps were crashing at startup.

arminatorx say:

... if I connected the joystick with a serial to USB adapter to the Xbox Adaptive Controller, would it work? If so, what type of adapter would I need? 

In order for the Adaptive Controller to recognize an USB joystick, the USB adapter would present itself as a HID-compliant (Human Interface Device) Joystick. 

Most USB to Serial adapters however advertise themselves as a serial COM Port, so normal USB to 9-pin converters won't work. 

You could try to contact Curtiss Wright, the maker of the Compact Joystick, if they know of an USB adapter that makes this joystick available as an actual USB Joystick device. 

When talking to Curtiss Wright, make absolutely sure that they are aware of the HID Joystick requirement. A serial port USB adapter will not do the trick!

Assuming that you are using the new Story Remix interface on the Photos app, the feature to add multiple songs on a single project is not available as of yet.

It's a great feature though, and it would be a neat idea to carry over this option to the new Story Remix to bring you the best experience that you used to have from the Windows Movie Maker. Hence, I would like to suggest that you bring this to our developer's attention by sharing this to our Feedback Hub.

To submit a new feedback, please follow these steps:

  1.     On your PC, search for Feedback Hub in Cortana search, then launch the app.
  2.     In the left menu, navigate to Feedback > Add a new feedback.
  3.     Select the Problem, share any details you think are relevant, and choose an appropriate category and subcategory.
  4.     Important: If possible, reproduce the problem(s) after clicking Begin monitoring (or Start capture) near the end of the form. Click Stop monitoring when you're done.
  5.     Click Submit once you've completed the form and monitoring.
  6.     Click Continue using Feedback Hub.
  7.     Click My feedback at the top, then find the item that you submitted.
  8.     Open it, then click Share.
  9.     Click Copy link, then paste the link here in your response (it will look like https://aka.ms/<unique link>).

For now, what I can suggest as a workaround is to use a third party app from Windows Store that you can use to edit and join two songs into one, so you can use it with the Photos app. We have tons of apps, free and paid, that you can use to edit songs, so check out our store if you have the time.
