Hi,

try this code..

 

Sub RangeName()
Dim shName As String
Const ListName As String = "DATA" ' <<< range name
Const SearchName As String = "Age Bucket" ' <<< search header
shName = ActiveSheet.Name
Dim rng As Range, r As Long
On Error Resume Next
Set rng = ActiveSheet.UsedRange.Find(What:=SearchName, SearchOrder:=xlByColumns, SearchDirection:=xlNext)
r = ActiveSheet.Cells(Rows.Count, rng.Column).End(xlUp).Row
ActiveWorkbook.Names.Add Name:=ListName, RefersToR1C1:="=" & shName & "!R" & rng.Row + 1 & "C1:R" & r & "C1"
On Error GoTo 0
End Sub

Background:

When you change indents or fonts on text or at your insertion point, that is called "direct formatting" as contrasted with style-based formatting. Such direct formatting continues as you type and will continue to new paragraphs started by pressing Enter.

Pasted text usually will not continue that formatting. Nor will text created using the awful "Click and Type" feature.

Style-based formatting is both predictable and easily applied and modified.

"Current Microsoft wireless keyboards encrypt keystroke data using 128-bit AES encryption. The prior generation of Microsoft wireless keyboards used XOR encryption which was shown to be insecure.
USB dongles from both the AES and XOR encrypted generations of Microsoft wireless keyboards accept unencrypted keystroke packets transmitted to the RF address of a wireless mouse. This applies to standalone wireless mice, as well as mice sold as part of a keyboard and mouse set."

I don't think I'd personally expect a completely secure solution for older devices using the XOR encryption and would instead evaluate my true potential risk from this technique.  Since unlike a WiFi or other 2-way communications hack this wireless keyboard dongle vulnerability only allows an attacker access to the keyboard and mouse input. it has limited value.

Though in theory in might be used to gain minimal control of a PC, turning this into something useful would be more trouble than it is worth for most, since only someone located within 100 yards of the PC could actually use it.  This limits the attacker to someone either located within a nearby living space or sitting outside your location in a vehicle.  So unless you feel a concern that your neighbors or someone else with knowledge of your wireless keyboard use have both the technical ability as well as the motivation to acquire the tools necessary to perform this type of attack, I wouldn't typically consider it a significant risk.

If you are concerned, then simply replace these relatively cheap items with a more current wireless device or even a much more secure and standardized Bluetooth device, which often works over larger distances by design.  Since these devices typically wear out physically over time, for most the relatively small cost to entirely replace them is worth it for the often improved design and better operation of a new set.
