First of all, I apologize for not sharing this solution for a long time since I'm busy working two jobs.

If the PC you're working on is on a domain environment, you need to bypass WSUS by running this on a elevated command prompt:

REG ADD “HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU” /v UseWUServer /t REG_DWORD /d 0 /f

Restart Windows update service, go to Region and Language settings, pick the language > Options and download language pack, handwriting, speech, etc. and it should start working again.

After you're done, just delete the registry key.
