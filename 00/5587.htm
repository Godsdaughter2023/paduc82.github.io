alba1982---01 say:

Okay okay okay- I found a work-around on my own:

 

My spb.file was a backup-file created with the Samsung Kies software and as far as I learnt this kind of files can only be managed by that software.

 

So, first of all I had to convert the .spb file into a more common file format as .vcf. To do so, you basically will have to open the Samsung Kies software on the PC, open the last saved .spb file and save the file again as .vsf manually. For more detailed information have a look at this link:

 

http://www.file-extensions.org/spb-file-extension

 

Then you import the new created. vsf-files to Outlook, or gmail or to another phone. Afterwards you use the "Transfer my Data"-app for file-transfer via Bluetooth with the other phone or sync your Outlook, gmail, etc. account with your Lumia phone.

 

P.S.
"What is SPB file extension?

 

File type specification:

 

Backup file type icon Backup file type

 

The SPB file extension is mainly related to Samsung Kies, a software used for communication between latest Samsung devices, such as Samsung Galaxy or Samsung Omnia mobile phones.

 

A *.spb file contains list of contacts from Samsung devices saved in Samsung Kies program. There is currently no way how to convert this file.

 

However, users who would like to have their contacts saved in VCF file format, can try these few steps:

 

    Open your Samsung Kies software

    Open your last saved *.spb file

    Save the file again in VCF format in the file name area

    Go to the default folder where *.spb files are saved (C:\Users\ _\Documents\samsung\Kies\PIMS\)

    Locate the *.vcf file.

    Select all contacts and copy them in a directory (contacts)

    Connect your Samsung device to PC and copy the directory (contacts) in to the SD card

    Disconnect your Samsung device

    Select the contact icon

    Press left key and select Import Contacts From SD"

 

Source: http://www.file-extensions.org/spb-file-extension
