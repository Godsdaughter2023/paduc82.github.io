Hi,

 

To disable the In-private setting in Internet Explorer, it’s as simple as making a change to the registry key.

 

a)      Press Windows key + R, type regedit and press enter.

b)      Locate the following key HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Internet Explorer\Privacy\EnableInPrivateMode.

c)      When enabled (the default), the dword value is set to 00000001. To disable InPrivate mode, just change that last “1” to a “0” as in 00000000.
