john - project say:

Adrienne XPO,

It depends on what you are defining as "over-arching work streams". You can filter the Resource Usage if you set an assignment flag for those assignments that are on your defined work streams, but there is no special view or settings to isolate what you want.

Keep in mind that identifying the work streams with a flag field on a Task view will not directly transfer over to the assignment rows of the Resource Usage view. For example, Task Flag1 is not the same field as Assignment Flag1. You will have to either manually copy the task flag field to the assignment flag field or use VBA to do so automatically.

You could also try the Task Usage view. With that you WILL be able to set a work stream task flag and then filter the Task Usage view. It will only show desired work stream tasks and the resources assigned to them. For example, if you consider the critical path to be the "over-arching work stream" then you can simply filter on the Critical field in the Task Usage view.

Hope this helps.

=VLOOKUP(B3,Sheet2!$A$1:$M$131,1,FALSE)

=IF(COUNT(MATCH(B3,Sheet2!A:A,0)),B3,NA())
