daniel g. grau say:
That's a system-wide effect and can be adjusted globally for all applications or individually per application. You can't turn it off, but you can control the pop-up delay until the yellow tooltip appears. This involves adding an extra parameter to Word's preference file, com.microsoft.Word.plist.


1. Quit Word. Go to the Finder, select Go > Go to Folder and type /Library/Preferences. Locate preference file com.microsoft.Word.plist  and make a backup copy by Option-dragging it to the Desktop (this is just a safety measure).


2. Open the Terminal application, in Applications > Utilities. You'll see an old-style terminal interface with a prompt like this:


     123-456-789-000:~ YourUserName$


3. Type the following Unix command (or copy from here and paste), then press Return:


     defaults write ~/Library/Preferences/com.microsoft.Word NSInitialToolTipDelay -int 5000


    The numerical parameter above is the number of milliseconds until the tooltip shows. The normal value is about "1000." Thus, using 5000 ms will give you a 5-second lead time. This should be adequate. As long as you don't hover on a tool for more than 5 seconds, you'll never see the tip, but it will still be available.


4. After pressing Return, don't quit Terminal yet. Launch Word and check the delay. If you don't like what you see, re-issue the command with another number. You need to quit and reopen Word each time, so it will read its preferences file (com.microsoft.Word.plist).


5. To restore the original delay, use "1000."


6. If you have any problem, quit Word and restore the original preferences file you copied to the Desktop.

graham mayor say:

Actually yes. If you add the text <SECTIONSTART> to the start of a paragraph where you want a section break and <FOOTER> to the start of a paragraph you want to be the Footer, then the following macro will do that, when you use it to import the named RTF file into the active document; however it might need some fine tuning to get the results you require, particularly with regard to footers in multi-section documents.

Sub ImportRTF()
Dim oSource As Document
Dim oTarget As Document
Dim oPara As Paragraph
Dim strText As String
Dim oRng As Range
    Set oTarget = ActiveDocument
    Set oSource = Documents.Open(Filename:="C:\Path\SourceDoc.rtf", Visible:=False)
    For Each oPara In oSource.Paragraphs
        Set oRng = oTarget.Range
        oRng.Collapse wdCollapseEnd
        Select Case True
            Case InStr(1, oPara.Range.Text, "<SECTIONSTART>") > 0
