ron coderre say:

When attempting to save an excel file that has any square brackets
in the path or file name you should receive an error message something like this:

The file could not be accessed. Try one of the following:
•Make sure the specified folder exists
•Make sure the folder that contains the file in not read-only
•Make sure the file name does not contain any of the following characters: <>?[]:| or *
•Make sure the file/path name does not contain more that 218 characters.

Consequently, since that is a limitation...you'd do better to work *with* Excel,
instead of *against* it, and adjust your folder/file names to avoid those characters.

As Jim Rech (of your referenced post) mentioned, square brackets are used in formulas
that refer to other workbooks. Allowing them in file names or paths would cause problems.

I hope that helps.

Ron Coderre
Microsoft MVP - Excel (2006 - 2010)

how do I continue numbering or bulleting in sequence from one page to the next

davidf2 say:

I assume you are talking about Publisher 2010 and numbering...bullets are not sequential.

On the second page and the following text box select the column of text that you want to apply numbering to, right click and in the text formatting tool bar that pops up click the downward pointing chevron on the 'bullets icon' > Bullets and Numbering > Numbering tab > choose the format of the numbers you want to apply > 'Start At' the number you want.

Alternatively you can find the 'numbered list' option by itself on the main menu under Home tab and in the Paragraph grouping of options.
