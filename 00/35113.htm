arminatorx say:

[quote user="KingKotstulle"]sie wollen einem zuerst glaubhaft machen dass ... es die Technik der Zukunft ist und es ÜBERHAUPT nicht möglich ist euch wieder eine IPv4-Adresse zu geben[/quote]

In der Aussage steckt eine gewisse Wahrheit, die UnityMedia aber zu ihren Gunsten biegt, weil sie es vorher vermasselt haben.

Wahr ist, dass IPv6 die Zukunft ist, und dass es im Grunde genommen eine sehr gute Sache ist.

Dass es UnityMedia nicht möglich ist Euch eine IPv4 Adresse zu geben, liegt daran, dass sie "zu spät" IPv4 Adressen eingekauft haben. Während die "ersten" Internetprovider und große Datencenter sehr früh riesige IPv4 Blöcke gekauft haben, und so genug Adressen für Kunden in der Zukunft haben.

UnityMedia hat halt zuwenig IPv4 Adressen bekommen, und hat nun mehr Kunden als IPv4 Adressen. Daher bekommen Neukunden eben eine IPv6 Adresse und "doppelte" IPv4 Adressen, die vorher bei UnityMedia geNATed werden, so dass ältere Geräte "passiv" ins Internet verbinden können.

You can use dism to repair windows files. 

From http://windows.microsoft.com/en-us/windows-8/windows-update-error-0x800f081f

In the Administrator: Command Prompt window, type the following commands. Press the Enter key after each command:

dism.exe /online /cleanup-image /scanhealth

dism.exe /online /cleanup-image /restorehealth

If it can't find the source files then your Windows update is also likely corrupt and you will need to download the source files again with the media creation tool.
