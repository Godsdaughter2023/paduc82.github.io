jay freedman say:

Create a separate template for each set of margins you want to save, and base each new document on the template that has the desired margins.

Because margins are a property of a section, and an otherwise blank template usually has only one section, you can't save multiple margin settings in one template.

To simplify choosing the correct template to create a new document, name the templates to identify their margin settings. You can use the NewDocUserform add-in from my web page at https://jay-freedman.info/ to select the template from a dropdown list quickly.
