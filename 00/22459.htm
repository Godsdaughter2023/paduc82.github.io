If things work fine in Safe Boot but fail in a regular boot the first place to look for the source of the problem is your fonts. Safe Boot opens with a minimal set of fonts and empties the font cache.

Use a font utility and verify ALL your fonts. Delete old versions of duplicate fonts and delete any font that doesn't completely validate. Simply turning them off is not enough. Restart your Mac, empty the trash and then try again.
