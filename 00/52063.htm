ronnie vernon say:

Hi Justin1583

 

 

Windows has nothing to do with the battery charging components.

 

Windows can only display the information that is reported by the processor from the charging components.

 

Here is a graphic that shows the typical hardware charging components.

 

laptop battery charging diagram

 

One thing that you do not want to do is intentionally drain the battery. This can significantly reduce the charging capacity in a modern Li-ion battery. The typical lifetime of a Li-ion battery is 1 to 3 years.

 

You can create a detailed report of the battery status to double check the condition of the battery.

 

1. Go to Start / All Programs / Accessories. Right click the Command Prompt and select Run As Administrator.

 

2. In the command prompt window, copy/paste the following command.

 

powercfg -energy -output %userprofile%\desktop\energy-report.html

 

Press ENTER.

 

Wait until the command completes (usually about 2 minutes).

 

This will create a file named energy-report.html and place it on your Desktop.

 

3. Go to the Desktop and double click the file to open.

 

Scroll down to the Battery: Battery Information section.

 

You will see:

Design Capacity xxxxx

Last Full Charge xxxxx

(Where xxxxx is a number)

 

Copy this section and paste it in a reply here, for analysis.

 


Regards

cesar_e. say:

Hello Leonard,

Welcome to Microsoft Community Forum. I understand that you want to arrange your photos manually using Photo Gallery. We would like to inform you that there are no options to do a manual arrangement on photos. But, you can choose on the arrangements of your images by going to View. Afterwards, you can choose to arrange them by Date, Geotag, Person, Name, Tag, Published, Folder, Media type and rating.

If you want to suggest the manual arrangement on Photo Gallery, we suggest that you create a separate thread on this link. Microsoft will be reviewing your feedback for future reference.

In case you have other inquiries regarding Photo Gallery, please let us know. We'll be here to assist you.

Thank you.
