    Word 2013 merged time field displaying incorrectly - Microsoft Community

    four cs say:

    Word 2013 merged time field displaying incorrectly

    When I merge in Word pulling data from Access (or Excel ... doesn't seem to make a difference), instead of the time displaying as hh:mm in my merged document, it displays as hh:mm:ss.

    Is there a way to correct this?

    hansv mvp say:

    Right-click the merge field in the merge document.

    Select 'Toggle Field Codes' from the context menu.

    You'll see something like

    { MERGEFIELD TimeField }

    or

    { MERGEFIELD TimeField \* MERGEFORMAT }

    where TimeField is the name of the field. Make it look like this:

    { MERGEFIELD TimeField \@ "HH:mm" }

    for 24-hour time format (e.g. 21:35), or

    { MERGEFIELD TimeField \@ "h:mm AM/PM" }

    for 12-hour time format (e.g. 9:35 PM).

    Press F9 to hide the field code and to update the result.
