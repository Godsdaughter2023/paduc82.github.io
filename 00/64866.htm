andre da costa say:

Yes, you can do that.

How to: Perform a clean install of Windows 10

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate.

So, there is no need to know or get a product key, if you need to reinstall Windows 10, you can use your Windows 7 or Windows 8 product key or use the reset function in Windows 10.

If you are prompted to enter a product key, click 'I don't have a key' and 'Do this later'

hairyfool say:

The only reason primarily is disk space so is often considered with small SSD drives.

Don't use the physical location to view the profile folders (C:\Users\username)

Instead select Libraries from the Taskbar.

In the left pane select Desktop

In the right pane look for the folder labelled with your User Name and double click.

It will still highlight Desktop in the left pane but in the right will be all the folders that make up the User Profile. This will still be the case even when these folders are in different physical locations.

Right click on each one and those that can be relocated will have the Location tab and in there will the physical path and the option to move the folder.

Be careful because the move location utility has (what I consider) a bug in that although it says it is going to move the folder it actually moves just the contents.

i.e. You select Music Properties and tell it to move Music to the D drive. What you end up with is all your Artist directories in the root of D: when you expected it to be in D:\Music. The mess is even made worse if you don't realise and move Documents as well, you then end up with Documents mixed up with Music.

The safest way is to edit the path directly in the Location tab, replace C:\Users\ with D:\ (or whatever drive letter you are using) then you end up with a folder path D:\username\Music.

The important bit is that only folders that can be moved will have the Location tab and by using this method to open the User Profile store they will all be listed even when in different locations. It saves confusion because you can still have a physical folder C:\Users\username\Music even if that is not your Music Library location.
