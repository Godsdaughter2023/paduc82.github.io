a. user say:

Hello,

 

Thank you for posting. In addition to the post of Bruce, there are 2 ways to print in Windows Live Mail. You can follow either one of these:

 

A. Through .csv file

  1. Go to Contacts in Windows Live Mail.
  2. Click Export under Tools pane and choose Comma separated values (.CSV).
  3. Click Browse then locate where you want to save the file and fill in the desired name.
  4. Click Next then please check the information you want to be included then select Finish.
  5. Locate the .csv file you saved and edit it with Microsoft Excel. After that, kindly print it through Microsoft Excel.

B. Directly from Windows Live Mail

  1. Go to Contacts in Windows Live Mail.
  2. Press CTRL + P.
  3. A pop-up window will appear. Kindly select the printer you want to use.
  4. Under Print range section select All Contacts and Memo for Print style section.
  5. Click Print.

sher amo say:

Let’s have a different approach on resetting the permission for your external hard drive. We suggest that you perform an atttrib.exe command line to display, set and remove the read-only, archive, system, and hidden attributes assigned to files or directories. To do this, follow the steps below:

  1. Use the Windows key + X keyboard shortcut to open the Power User menu and select Command Prompt (Admin).
  2. Type D: and press Enter. (If your external hard drive has a different drive letter, please check on the File Explorer and use the specific assigned drive letter instead).
  3. Type attrib -r -s -h -a D:\*.* /s and press Enter. Please note that it is successful if you do not receive a message confirmation after entering the command.

You can't uninstall a printer if you have files in your print queue. Either cancel printing, or wait until Windows has finished printing them. Once the queue is clear, Windows will remove the printer.

1.     1. Open Devices and Printers by clicking the Start button, and then, on the Start menu, clicking Devices and Printers.

2.     2. Click your printer's icon, and then click See what's printing on the command bar.

Hi Dr PaulAshe,

This issue normally occurs when there is any proxy enabled on the computer. It could also occur if the antivirus program is blocking it.

 

Method 1: Check the Date & Time Zone is correct:

 

Follow these steps to change the time.

 

a. Double Click on the current time on the bottom right hand corner of your screen.

b. Then click Change date and time settings, then click the Change time zone button

c. Select your time zone and tick automatically adjust clock for daylight saving time and press OK

d. Then press Apply, and OK.

 

Method 2: Network troubleshooter

Run the Network Troubleshooter to check for any Internet connectivity issues.

 

a. Press the ‘Windows + W’ key on the keyboard; this will bring the search interface.

b. Click ‘Troubleshooting’ in the search result.

c. Under ‘Troubleshoot computer problem’ click on ‘Network and Internet’.

d. Click on ‘Network Adapter’ to start the troubleshooter.

 

Method 3: Check the proxy settings in Internet Explorer.

 

Follow the steps to disable proxy:

 

a. Click the Gearbox at the top right corner in internet explorer.

b. Click the Tools button, and then click Internet Options.

c. Click the Connections tab, and then click LAN settings.

d. Uncheck the box next to “proxy server for your LAN”.

e. Click OK to save and close the window.

 

Hope this information helps. Reply with the status so that we could help you further.
