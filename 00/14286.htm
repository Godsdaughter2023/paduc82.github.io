I'd use the On DblClick event of a control on the form, for example the ID field:

 

Private Sub ID_DblClick(Cancel As Integer)
    If IsNull(Me.ID) Then
        Beep
        Exit Sub
    End If
    If Me.Dirty Then Dirty = False
    DoCmd.OpenForm FormName:="OtherForm", WhereCondition:="ID=" & Me.ID
End Sub

 

The above assumes that the name of the ID field is simply ID, and that it is a number field. If it is a text field, use

 

        ..., WhereCondition:="ID=" & Chr(34) & Me.ID & Chr(34)

 

Chr(34) is the double quote character ".

For this you need a form, in single form view bound to the Lots table, and within it subforms for the other tables.  Whether a form can contain 25 subforms is another matter; I've never had occasion to use more than half a dozen at the most.  Each subform should be linked to the parent form by setting the LinkMasterFields and LinkChildFields properties to LotID, or whatever the primary key of Lots and the corresponding foreign keys are called.  Each subform will then show only the procedures relating to the current lot in the parent form, and when a new record is inserted in a subform the foreign key LotID column will automatically receive the correct value.

lars-Åke aspelin say:

Dear all,

I have a table with different range of cells set up with different border format (i.e. thickness and style) and I would like to change the border color of the entire table.

I found that I am only able to select another border color and reapply to border again to the cells; while this method would need me to re-set the thickness/ style of the boarder and re-apply to each range of cells.

Is there any way in Excel 2013 that allow user to change the border color of the existing border instantly?

Many thanks in advance!

You can do this with a macro. Select the range (table) that you want to apply a new color to (without changing the style or widths of the individual borders) and run this macro:

Sub recolor_borders()
    my_color = RGB(0, 0, 255)  'blue
    For Each cell In Selection
        With cell.Borders(xlDiagonalDown)
            If .LineStyle <> xlNone Then .color = my_color
        End With
        With cell.Borders(xlDiagonalUp)
            If .LineStyle <> xlNone Then .color = my_color
        End With
        With cell.Borders(xlEdgeLeft)
            If .LineStyle <> xlNone Then .color = my_color
        End With
        With cell.Borders(xlEdgeTop)
            If .LineStyle <> xlNone Then .color = my_color
        End With
        With cell.Borders(xlEdgeBottom)
            If .LineStyle <> xlNone Then .color = my_color
        End With
        With cell.Borders(xlEdgeRight)
            If .LineStyle <> xlNone Then .color = my_color
        End With
    Next
End Sub

Change the row with "my_color = RGB(0, 0, 255)" to the color of your choice.
