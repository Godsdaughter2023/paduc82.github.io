Choosing Significant Figures as an excel format

joeu2004 say: Nix wrote:
I often have to do a lot of manual reformatting in excel because you can only select the number of decimal places you want - you cannot select how many significant figures you want. Microsoft excel needs a 'Significant Figures' format option!

Not exactly what you want, but:

=--TEXT(A1,"0." & REPT("0",C1-1) & "E+0")

where A1 is the value and C1 is the number of significant digits.

Remember that Excel will format only up to 15 significant digits, even though the internal representation might be more precise.

This is usually not a problem, especially if all of your notebooks are on your local hard drive (My Documents\OneNote Notebooks).  Your only concern is if you have notebooks on UNC or SharePoint servers and you made changes to them while offline.  Otherwise, just click Okay to let OneNote rebuild the cache - once it's done all of your notebooks should still be open.  You'll still need to upgrade any notebooks you created with the Beta (there should be a bar at the top of the page asking you upgrade the notebook - click the bar to do so).

Just to be safe, it wouldn't be a bad idea to backup all of your old notebooks beforehand.  By default you'll find them in My Documents, in the OneNote Notebooks folder: just copy this entire folder someplace else before doing the upgrade.

In the unfortunate event you do have out-of-sync changes, your best chance to recover them is to open the backup files OneNote periodically creates.  Go to File-->Info-->Open Backups, and select the file that is out-of-date.  You can then copy the pages you need into your newly upgraded notebooks (you'll still need to upgrade these backup files first).

sheeloo say:

It assumes that each cell in col A contains text like below where each line is created using ALT-ENTER which is ASCII character 10

Features: Made out of copper
Keeps wine cold
Word wine embossed
7.5" H x 15" W

you may replace debug.print with statements which write to a cell or a text file or just copy and paste from 'immediate' window (CTRL-G to toggle)

Sub createHTML()
j = 1
x = Cells(j, 1)
Do While x <> ""
    y = Split(x, Chr(10))
        Debug.Print "<strong>" & y(0) & ":</strong> "
        If UBound(y) > 0 Then
             Debug.Print "<ul>"
            For i = 1 To UBound(y)
                Debug.Print "<li>" & y(i) & "</li>"
            Next i
            Debug.Print "</ul>"
        End If
    j = j + 1
    x = Cells(j, 1)
Loop
End Sub
