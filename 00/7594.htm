wildrosend say:

Macro to change format of all cells formatted as currency

I have a large Excel workbook and need a macro to find all cells in the workbook that are formatted as number/currency, and then change the currency symbol from the selection "none" to $.

All help you may provide would be greatly appreciated.

Tom
mr andrew say:

Sure, it's easy.  Just copy/paste the lines and change each time for the different formats.  Applying across all the sheets in the workbook is not so tricky but it isn't something you can directly record.  Try this:

 

Sub replaceFormats()

    Dim theSheet As Worksheet   'Not strictly necessary but it's good programming practice.
   
    For Each theSheet In ActiveWorkbook.Sheets  'This begins a loop through all the sheets in the current workbook
       
        'The first search-and-replace.
        'It uses theSheet.Cells to refer to all cells on the sheet.  Otherwise it is the same
       
        Application.FindFormat.NumberFormat = "\$#,##0.00"
        Application.ReplaceFormat.NumberFormat = "#,##0.00"
        theSheet.Cells.Replace What:="", Replacement:="", LookAt:=xlPart, SearchOrder _
            :=xlByRows, MatchCase:=False, SearchFormat:=True, ReplaceFormat:=True
       
        'Copy the above lines and specify another number format:
       
        Application.FindFormat.NumberFormat = "[$€-2] #,##0.00"
        Application.ReplaceFormat.NumberFormat = "#,##0.00"
        theSheet.Cells.Replace What:="", Replacement:="", LookAt:=xlPart, SearchOrder _
            :=xlByRows, MatchCase:=False, SearchFormat:=True, ReplaceFormat:=True
       
        'Repeat as necessary here
       
        Next   'This goes back to the beginning of the loop if there are any more sheets to process.

End Sub
