    a. user say:

    I think you meant to say that you did a Reset.

    Pop Quiz: What's the difference between a

    a) Restore

    b) Reset

    c) Refresh

    d) Reformat

    Answers:

    a) A restore point is a set of key system files and registry entries intended to restore the user interface so you can troubleshoot a problem.

    b) A reset is a clean install of Windows; your hard disk is erased and Windows is re-installed.

    c) A refresh is also a clean install, but first Windows backs up your default user folders (Documents, Music, etc.) your user accounts and your Windows Store apps. These are restored after Windows is installed.

    d) A reformat removes the partitions of your hard disk, in effect erasing your hard disk.
