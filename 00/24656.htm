Please ignore the first two replies.

The truth is that you can change a notebook's name only in one place: on Onedrive

(https://onedrive.live.com) using your browser. It seems possible in Windows, this is an illusion. This change changes only the local display name, not the real name. That's the reason why you don't see the change in Android.

The reason behind is that a name change changes the path to the notebook and so kills your connection from a device to Onedrive.

A safe procedure for a name change of a notebook is:

  1. Close the notebook on ALL devices (including Windows)
  2. Change the name on Onedrive with your browser
  3. Re-open the notebook on all your devices

This toggling code routine is one I use to resize images with a click.

To be entered into a General Module in your workbook.

Assign the macro to each object.

Sub PictureBigSmall()
    Dim shp As Shape
    Dim big As Single, small As Single
    Dim shpH As Double, shpOriH As Double

    big = 1   'Set large scale, 2.5 = 250%
    small = 0.2    'Set small scale, 0.5 = 50%

    Set shp = ActiveSheet.Shapes(Application.Caller)

    shpH = shp.Height
    shp.ScaleHeight 1, msoTrue, msoScaleFromTopLeft
    shpOriH = shp.Height

    If Round(shpH / shpOriH, 2) = big Then
        shp.ScaleHeight small, msoTrue, msoScaleFromTopLeft
        shp.ScaleWidth small, msoTrue, msoScaleFromTopLeft
        shp.ZOrder msoSendToBack
    Else
        shp.ScaleHeight big, msoTrue, msoScaleFromTopLeft
        shp.ScaleWidth big, msoTrue, msoScaleFromTopLeft
        shp.ZOrder msoBringToFront
    End If
End Sub

Dim FileName As Variant

What kind of controls?

 

You would basically create building block entries for each possible combination of cat/action and then after the user selects the cat/action insert the appropriate buildingblock.

 

You could do this via userform or the exit event from the document controls.

 

It is going to take some VBA code, so if you don't know how or want to learn then you might have to hire it out.

 

The gist:

 

Select Case "Cat"

   Case 1

      Select Case "Action"

         Case 1

            'Do this

         Case 2

            'Do that

         Case 3

 

         Case 4

 

     End Select

   Case 2

 

   Case 3

 

   Case 4

 

End Select
