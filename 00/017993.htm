It seems that once the cursor passes over a chart, the most recent tooltip remains attached to the cursor no matter where it goes in the document. Since either "Chart Area" or "Plot Area" is the tooltip for the outermost part of the chart, that's generally the one you get. I don't see a way to prevent this -- unchecking the display option "Show document tooltips on hover" doesn't suppress the chart tooltips -- but there is a way to stop it once it starts. Click in the text anywhere outside a chart, and then as much as possible avoid passing over/through a chart.

greginmich say:

Most of these Tech Support Scam pages just use JavaScript tricks to lock up your browser – and they consequently won’t be detected by any AV or Antimalware app. You can sometimes escape their clutches by just clicking on the “OK” button in the popup dialog; putting a check in the “Don’t let this page create more messages” box; and then closing the page. But with some of the more sophisticated scam pages, you’ll have to right-click on the taskbar (or press Ctrl + Alt + Delete) to start Task Manager, and then end the browser process. And Microsoft Edge still isn’t handling these scam pages properly, because if you kill the Edge browser process with Task Manager, then you’ll have to restart Edge indirectly with a web search, and then right-click on the search page tab and select “Close other tabs” in order to remove the scam page.

 

If you’re able to escape from the Tech Support Scam site by using one of the above methods, then it was just locking up your browser with JavaScript, and didn’t actually install any malware. But these scam pages will naturally tend to pop up again whenever you revisit a site that’s hosting a compromised advertising domain, just because the compromised domain will get another chance to deliver a malware-site redirect every time it gets reconnected to the host, and not because there’s some kind of persistent malware involved – so the best way to deal with this malvertising issue is to just install an ad-blocker extension like uBlock Origin, which will selectively prevent any suspect domains from connecting to the host site. Although uBlock Origin includes some granular script-blocking features, you’ll most likely be able to manage the issue with just the default domain-blocking functionality. Ad-blocker extensions for Edge are available directly from the Edge Settings menu:

 

Settings and more > Extensions > Get Extensions from the Store

 

https://github.com/gorhill/uBlock
