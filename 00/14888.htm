Office 365 Email Data Collection into Excel

I have around 20 emails arriving each day from the same recipient with the same subject heading however the content is different:

Client: XXX
Location: XXXX
Job: XXX XXX
Reference: xxxxxx
Start Date: xxxxxxx

I am spending hours each week adding this data into Excel so I can then analyse the data and find out what the demand and supply is at the end of each week. Can this process be automated so the data is extracted from the emails and populated into Excel automatically by just running a script from Office 365 Business Outlook email?

rick xu msft support say:

Hi ChoiceMC,

 

I’m wondering if you want to do this in OWA (Outlook for Web App) or Outlook client.  We cannot do that in OWA. But you can use macro to achieve it in Outlook client.

choicemc say:

Hi Rick,

I usually use Office 365 through IE however for this task we can use the Outlook Client on a desktop.

Never used a Macro before but if you can let me know what I need to do I can follow the steps? Please note I am using Office 365 Business.

rick xu msft support say:

Hi ChoiceMC,

 

You can create a specified folder in Outlook client and then using an inbox rule to move all messages which match the subject to this folder.  Next, export this date to a CSV file (This file can be opened with Excel). If want to filter data by multiple criteria in Excel, please post a new thread in our Office forum for further support.  

Export data to a CSV file:

  1. Click File > Open & Export >Import/Export
  2. Select Export to a file >Comma Separated Values
  3. Choose the specified folder and Save exported file as
  4. Tick the Export “Export ‘Email messages’ from folder” option

 

Regards,

Rick
