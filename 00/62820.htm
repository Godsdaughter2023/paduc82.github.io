wunders say: The "MARGIE" in "C:\Documents and Settings\MARGIE\... is not a "Computer Name", it is a user "Profile name".  Once a user is created and its associated profile created, this name cannot be changed directly.  The only practical way to change this is to create a new user, copy the user profile data from MARGIE to this new user, and then delete the MARGIE profile.  To do this, pretend that the "MARGIE" user profile is corrupted and then follow the instructions in the following KB article:

"How to recover the damaged user profile in Windows XP"
  < http://support.microsoft.com/kb/555473 >

Was the package sealed when you got it also the product ID that you see on the package has to do with what is generated when the packaging is created.  The product ID that you are seeing in your system information screen is the ID that Windows generated when it installed.  This activation has been around since XP.  The product key is entered then a random installation ID is generated based on certain aspects of your PC's hardware I don't remember what it checks for off-hand.  When you click Activate Windows your pdouct key along with that ID is transmitted to the servers at Microsoft.  Once there the product key is checked to see if it has any associations with other installation ID's If the number of activations allowed has been exceeded it sends a message back to you to let you know that your key is no longer valid.  You have 30 days to activate Windows after installation.  if I remember right if you run out of time it turns your background black, causes certain settings to not work, and displays the message "This copy of Windows may not be genuine!"  i know in XP it locked you out of your desktop altogether.  I would try to get the seller to refund your money and return it to him or her.  If that doesn't work you can use Paypal's resolution center but as for getting it to work you have the option to buy another key but Microsoft will charge you the full retail price for whichever version of Vista you are ordering.  You can then use that key and activate the copy you installed thus saving you the time and headaches of reinstallation.

pcr just pcr say:

You were supposed to have that Lfsvc key. And deleting the "3" subkey was supposed to be a cure.

(1) Have you run an aggressive registry cleaner or spyware remover?

     I can think Geolocation Service could be taken for a spy. Then, check whether the cleaner has an undo for it.

(2) Is Cortana running? If you ask Cortana "Where am I", do you get an answer?

     If Cortana is running, you may decide you don't need Geolocation.

(3) R-Clk START, & select Device Manager.

     Any ?/!/X on any icon?

(4) R-Clk START, select Run, type "MSInfo32", & hit ENTER.

     Open "Components", & click "Problem Devices". Any showing?

(5) It could be worth running DISM & SFC.

(a) R-Clk START, & select "Command Prompt (Admin)".

(b) Type "DISM /Online /Cleanup-image /ScanHealth", & hit ENTER.

That took 13 minutes to finish. It sat on 100% for 2 minutes before completing.

(c) If yours reported corruption, then type ""DISM /Online /Cleanup-image /RestoreHealth", & hit ENTER.

     This time it will repair files.

(d) Type "SFC /ScanNow", & hit ENTER.

That took 11 minutes to complete.

(e) If either DISM or SFC repaired/recovered a file/driver, reboot for the file to run & hopefully create the missing registry keys.
