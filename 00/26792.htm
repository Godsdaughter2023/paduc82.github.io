Get-Mailbox -ResultSize unlimited|%{$mbs=Get-MailboxStatistics -Identity $_.DistinguishedName;$mbs|Add-Member NoteProperty UserPrincipalName $_.UserPrincipalName;$mbs}| select UserPrincipalName,LastLogonTime

larry yu msft say:

Hi RakeshGR,

 

As I know, we can run the following command in Exchange Online PowerShell to get all mailboxes’ last logon details at once:

Get-Mailbox -ResultSize unlimited | Get-MailboxStatistics | select-object identity,lastlogontime,DisplayName

Please check if it meets your requirement. Feel free to let us know if anything is misunderstood.
