    considerate_guy say:

    The project file is NOT a movie. It's essentially the save state of your editing session which remembers the names of the files you imported, the trims you made, etc. The project file contains no media within it and isn't a playable movie. So you can't import a project into another project.

    You can *open* that project (go to File > Open project) and it will reopen in the prior save state. It will look for the original media files that you added to your storyboard. Note: If you didn't also movie the media files to your laptop, the project file is useless. It's equivalent to having a blueprint for building a house but forgetting to buy the building materials. Or having a recipe for a cake, but not having the ingredients.
