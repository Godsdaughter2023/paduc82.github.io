charles kenyon say:
A StyleRef field captures everything within a given style beginning its search with the top of the page by default. Understanding Styles in Microsoft Word. Check to see what the style of the text at the top of the anomalous page is.

If the style is continued onto the current page from the previous page, that will be the first use of the style on the page.

If the style is not used on the page at all, the StyleRef field will capture the previous use.

If a style is used only once on a page, that will be the first and the last use of the style. It will be picked up by a StyleRef field set to get the first use on the page and one set for the last use on the page.

See also: Reference for the StyleRef field http://office.microsoft.com/en-us/word-help/field-codes-styleref-field-HP005186193.aspx?CTT=5&origin=HA102110133

Private Sub Form_Timer()
    Dim frm As Form
    Dim frmCaptionStr As String  

Do While Me.OpenFormsListbox.ListCount > 0
        Me.OpenFormsListbox.RemoveItem (0)
Loop
     
For Each frm In Forms 
        frmCaptionStr = frm.Caption
        Me.OpenFormsListbox.AddItem (frmCaptionStr)
Next
End Sub

Private Sub OpenFormsListbox_Click()
    Dim frm As Form
    Dim frmCaptionStr As String
   
    frmCaptionStr = OpenFormsListbox.Value
   
For Each frm In Forms
    If frmCaptionStr = frm.Caption Then
    frm.SetFocus
    End If
Next
End Sub

pc-learner say:

Assuming that Disk 1 is a 500GB SSD you are using in an external USB drive, it clearly hasn't been partitioned or formatted yet, and that's why Windows Explorer doesn't see it - it's not a usable drive yet.

Right click in the Unallocated area displayed for Disk 1.  It should then turn crosshatched and give you a drop down list.  Click on "New Simple Volume".  That starts a wizard where you can choose how much space to use (probably the maximum shown), what drive letter to assign (probably the next unused one, D, and how you want to format it.  In the Format menu, choose File system NTFS if the drive will only be used with Windows computers, or FAT32 if you want to use it with non-Windows devices too.  Leave the Allocation unit at Default, and give it whatever Volume label will be meaningful to you. Be sure to check the box for Perform a quick format.

When you've made the choices, the wizard will display them and ask you to click Finish or Cancel.  If you think they're correct, click Finish and in a minute or so you'll have a usable Windows drive.
