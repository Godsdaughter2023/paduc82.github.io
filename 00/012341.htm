jay freedman say:

Your proposal A is closer to correct, but still not right.

The bookmark is one that you apply through the Insert > Bookmark dialog; it doesn't exist "inside" the SEQ field as you tried in proposal B. The intention is that you apply the bookmark to some other SEQ field elsewhere in the document. The example described in the documentation involves bookmarking the SEQ field in the caption of a picture. The SEQ field that contains the bookmark's name then repeats the number displayed inside the bookmark.

This kind of field has no hyperlinking capability. If that's what you want, then use the References > Cross-reference dialog. Set the "Reference type" dropdown to "Bookmark", leave the "Insert reference to" dropdown set to "Bookmark text", and select the name of the bookmark that you applied to the target of the cross-reference. By default, the "Insert as hyperlink" box is checked.
