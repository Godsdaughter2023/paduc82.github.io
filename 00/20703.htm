Sub macro_01()
'Dec 25, 2015
Dim ws As Worksheet
Set ws = Sheets("Sheet1") '<< source name, change name as needed
Dim r As Long, c As Long, L As Long, i As Long, j As Long, t As Long
r = ws.[A1].CurrentRegion.Rows.Count
c = ws.[A1].CurrentRegion.Columns.Count
L = ws.Cells(Rows.Count, 1).End(xlUp).Row

.ClearContents
.WrapText = False
.AutoFit

t = r + 2

For i = 2 To r

v(j) = ws.Cells(i, j)

t = t + 1

L = ws.Cells(Rows.Count, 1).End(xlUp).Row

The logic for handling this kind of delete could probably be 'generically'/pseudo-coded something like this

Get all available data, each column into a uniquely identifiable variable.

With all possible data received, then

Test for presence or absence of any given data group, i.e. AA, BB ... YY, ZZ.

If the group is absent, find that group's location in the table by examining the labels on the worksheet to find a match and delete the entire column.  Check for next Group.

If the group is present, find that group's location in the table by examining the labels on the worksheet to find a match and process it. Check for next Group.

Repeat from "Test for presence or absence..." as required for all potential groups.
