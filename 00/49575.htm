 Once the account exists or if you want to hide an existing account do the following:

 

a.       Click Start and click on Run, type regedit and hit enter.

 

b.      Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList

 

c.       If SpecialAccounts and UserList are not found, create the new sub-key and name them those names.

f.       Then double click on the key you just created.

 

g.       Keep the value at zero to hide the account or change it to 1 if you want it to be seen on the welcome screen.

 

h.      Exit the registry.

 

i.        Restart the computer for the changes to take effect

 

How to Logon to a Hidden User Account:  

 

Press Ctrl+Alt+Del to bring up the windows login dialog box and then type in the user name and password.

Yes. You're actually right. When activating the new computer, you may encounter an activation error that says "The Product Key is in use" so you have to do phone activation in order to move the license from the previous installation to another. At the end of the phone activation after you enter the installation ID, you would be asked a question about it. But you have to be sure that you have completely formatted the previous hard drive first before activating it on a different computer.

 

To format the computer's hard drive and remove windows installation:

1. Enter the Windows 7 installation cd, restart the computer, and boot from the cd.

2. Select the Language, and then Next. Click Install Now and then accept the license agreement.

3. Choose Custom Installation. On the next window, choose and highlight the partition where windows is installed.

4. Click Advanced to show the Format option. Click Format and wait for it to finish. Once done, turn off the computer.

 

To do Phone Activation:

1. Go to Start Menu and type slui 4 in the Search box and press Enter.

2. Enter the closest country and then follow the instructions on the next window.
