OT:
When using "Justified" in Word 2010 and earlier there is an extra step you should take. The default justification Word uses is "dumb", it just inserts full n-space character equivalents BETWEEN words. That is not the intent of justification.  There should be partial "spaces" between both letters and words.  WordPerfect has been doing that way since the 1980's. MS copied that feature, but did a sloppy implementation (it has FINALLY been fixed in 2013).

In 2010, go to File menu > Options command > Advanced option. Scroll down to "Compatibility Options" section, click to expand "Layout Options", scroll down to "Do full justification like WordPerfect ...", turn it on.  You'll like the results of justification MUCH better
