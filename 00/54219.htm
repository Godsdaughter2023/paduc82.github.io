Hello Lisa,

Many parts of Windows 7, as well as individual programs, maintain activity logs of the type you describe. One way to search is to search the C:\ drive with the wildcard "*" (without the quote marks), like this, *.log; that will return all search results with the file extension "log". All will be viewable with the Notepad application.

 

The other place to look is the Windows Event Viewer, the place where Windows keeps track of all your errors and warnings. This tends to be more technical in nature, so I don't know how helpful it'll be to you, but.... Go to Control Panel, click on Administrative Tools, then click on Event Viewer when the new window opens.

MsMpEng is the Security Essentials AV engine, so it's not surprising that you'll see activity like this when you start yor PC. Also, svchost.exe is an essential part of Windows. It provides the interface between service programs and their dynamically-linked libraries. You can't stop it. Seeing a lot of svchost activity at startup is also normal. It may be possible to reduce the number of server host processes by not having as many services start automatically, but unfortunately, it's pretty difficult to determine which are necessary and which are not.
