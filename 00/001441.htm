scottgem say:
A column that includes both letters and numbers will be sorted by their ASCII values, character by character. So it first sorts by B, then by 1s, then by the 3rd character etc.

If you want to sort by letters first, then by numbers you need to split the column. If the pattern of your values is a B then an number, you can use:

=Right(A1,Len(A1)-1)

That will show the numeric value in a column. Use:

=Left(A1,1) to split the B out separately. Then sort by those 2 columns.

jay freedman say:
The only reasonable way to do this is with a macro. The following code (see http://www.gmayor.com/installing_macro.htm if needed) goes through the document one paragraph at a time. Wherever it finds a paragraph that isn't already in Normal style, it saves the character formatting and the paragraph formatting into variables, changes the style to Normal, and then applies the paragraph's original (style) formatting as direct formatting.

Sub DirectFormat()
    Dim para As Paragraph
    Dim fnt As Font
    Dim pfmt As ParagraphFormat
    For Each para In ActiveDocument.Paragraphs
        With para
            If .Style <> ActiveDocument.Styles("Normal") Then
                Set fnt = .Style.Font
                Set pfmt = .Style.ParagraphFormat
                .Style = ActiveDocument.Styles("Normal")
                .Range.Font = fnt
                .Range.ParagraphFormat = pfmt
            End If
        End With
    Next
End Sub

If there was any direct formatting in these paragraphs at the start, that will be discarded (keeping it would be a lot more work). For your illustrative purpose, I don't think that will be a problem.
