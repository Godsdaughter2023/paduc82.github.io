I just upgraded my version of Outlook from 2013 to 2016 and also had problems with AutoComplete not working.

I just recently switched from using POP3 to IMAP on my Comcast email account. I was using Outlook 2013. The configuration I ended up with (which is what I wanted) retained my old emails and all my folders that were stored locally in a pst file. 

The new IMAP account uses an .ost file.

In account settings under the email tab I only have my new IMAP account listed. On the data Files tab I have entries for both the IMAP account ost file and the old POP3 account's pst file.

I have been using this setup for a few days and the only thing that I couldn't get working was AutoComplete. No matter what I tried it would not remember the addresses. I finally fixed it today. 

What had happened was that my old pst file was set as the default under data files rather than the new IMAP account. Switching the defaults got AutoComplete working. 

However in the process I lost all of the entries in the address book. Luckily I had exported them to a .csv file before doing the upgrade to 2016 so I was able to recover them by Importing the .csv file into the Contacts folder.

To populate the now working AutoComplete file with all of the addresses from the imported address book, I used the trick I found elsewhere. \:

1. Start Outlook and put it into Work Offline Mode.

2. Create a new empty email and press the To button so the address book pops up.

3. With the top name in the address book highlighted scroll down to the bottom entry and hold the shift key and select the bottom entry. This way all names in the address book are selected and put into the TO field.

4. Give the message any subject (e.g. test) and write anything in the message body of the email.

5. Press send and the message will go into the Outbox (note if you forgot to set Work Offline you have just sent this message to all of your contacts!!).

6. Select the Outbox and delete the message you just created.

7. Now put Outlook back online and you should be all set.

Bottom line here - for AutoComplete to work make sure your default Data File (.ost) in account settings is associated with the default account in the email settings.

Hope this helps.

Mike

brian tillman [outlook mvp 2007-2019] say:

> However in the process I lost all of the entries in the address book.

Your address book doesn't contain anything. It is a specialized view of your Contacts folder. When you switched the data file default, Outlook 2013 created a new Contacts folder in the IMAP OST and made it the default because the default Contacts, Calendar, etc. folders are always in the default data file. outlook.com preview (the new look in 2016), how to disable - Microsoft Community

why- say:

outlook.com preview (the new look in 2016), how to disable "group by conversation"

hi all,

it took me a while to figure this out. so in case you need to know how to do this, here it is.

to disable the group by conversation, you need to click on "inbox", and then on the right hand side (above the time-stamp of the emails) there is the word "all", and then if you click this, there will be a pull-down menu, and at the bottom, there are two options:

"messages"   = disables   group by conversation

"conversations"  = activates   group by conversation

you then click on the one that you want.

ie. each of these two options are not things that you check/uncheck.  

ie. if "conversations" is checked (ie. there is a tick next to it), and then you click on it, it will remain checked. the only way to uncheck it (ie. have no tick next to it), is to click on "messages".

 

hope this helps.
