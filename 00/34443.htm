andre da costa say:
Open Start, type: advanced settings
Hit Enter

Click the System Protection tab
Click System Restore

On the initial page of the System Restore wizard, click Next.
On the next page, click to select your preferred restore point from the available list.
Click Next to continue.
On the Confirm your restore point page, click Finish.

On the warning box that appears click OK to confirm your action.

Wait until Windows 10 restores to the selected restore point and restarts automatically.
Start using the operating system normally.

Truthfully - the least expensive way, for many, to get a Windows 7 machine is to find a nice refurbished (factory recertified) machine (off-lease ones are a good idea) that include Windows 7 as the operating system installed.  You get a computer that should work well with the operating system, you get to keep your old system as you get used to the newer system and you can transfer your files/folders over to the new system at your leisure.  If you could set aside 50 to 100 USD/month, in 4-8 months you could have MORE than enough for a very good refurbished computer with Windows 7 already installed.

However, unless you have a pretty good reason to move to Windows 7 - I am unsure what the rush might be.  Yes - in April 2014, Windows XP will no longer be receiving updates from Microsoft.  Also, there is some newer software that already will not run on anything before Windows Vista as well as hardware - and that trend will, no doubt, continue.  Given the question as phrased - not so sure that would be a problem for you at this time or even in the near future.  Would I be incorrect in my assumption?

As for how much time you have to save up - if you have no pressing reason to move to a newer system/operating system... you have 23 months.  Even if you could only save 10 USD/month - you could get a decent Windows 7 system by then.
