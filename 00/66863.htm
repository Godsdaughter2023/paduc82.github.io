Hi Aravindh,

If you use automatic updating, Windows Update will check for updates and install them when they're ready.

If you don't use automatic updating, you should check for updates yourself at least once every week. We usually release important updates once a month. However, updates could be released at any time.

No matter what your settings are, some updates aren't installed automatically. This includes optional updates and updates that require you to accept new terms of use. When these updates are available, Windows Update will let you know that they're ready to be installed.

You can also set Windows Update to install recommended updates automatically or let you know that they're available. Disable data usage over metered connection:

There are two metered connection settings that can further reduce data usage. The first is downloading device software. Go to Settings > Change PC Settings > PC & Devices > Devices and make sure that "Download over metered connections" is off.

The second setting is syncing settings. Go to Settings > Change PC Settings > Skydrive> Metered connections> make sure that both "Sync settings over metered connections" and "Sync settings over metered connections even when I'm roaming" are off.

pa bear - ms mvp say:

...for security reasons should not the Windows Defender download its definitions first?

No, NIMHO - especially this month.

1. Among the non-security updates this month was KB3025417, an antimalware platform update for Windows Defender (updates Windows Defender services, drivers, and User Interface). As such, you'd want KB3025417 installed before updating Defs.

2. Assuming Automatic Updates is set to the default Automatic configuration, [1] Defender will auto-update Definitions daily (usually, but at least three-times-a-week by default).  If your computer's left running & connected to the internet 24/7, Defs will auto-update once-a-day, usually at the same time every day. Translation => Defender's Definitions are already up-to-date. [2] Of course, you can update Defender's Defs manually anytime you wish via the Update button on the Update tab.

=================================================

[1] In Win8(.1), Defender will not auto-update unless Automatic Updates is set to the default Automatic configuration.

[2] Defender considers itself sufficiently up-to-date as long as its Defs are no more than 72-hours-old.
