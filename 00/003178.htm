If there are only the two reviewers, maybe the Track Changes option is set to use only one color for all reviewers. To check that, open the Advanced Track Changes Options dialog as shown below, and choose "By author" as the color for insertions and deletions.

If that's already the selected value, then how many reviewers have tracked changes in the document? If there are more than 16, then Word is forced to reuse at least one of the colors -- the palette is limited to the ones in that dropdown. Also, several of the colors there are hard to tell apart, and "Auto" is usually the same as Black.

Another approach is to hide the tracked changes from some of the other reviewers. Click the Show Markup button, select the Specific People item, and click a reviewer's name to turn off the check mark and hide that person's changes. (It just hides them, but doesn't remove them.) Setting Up an Active X Control in Access - Microsoft Community

scott j. schrader say:

Setting Up an Active X Control in Access

I would like to use the Calander 12.0 Active X Control on a Main Menu form to "Open" a Planner Detail form with all the appointments for the date selected or events for that date. How do I do this ?

hansv mvp say:

Place a calendar control on the form (click ActiveX Controls in the Controls section of the Design tab of the ribbon). Name it for example ctlCalendar.

Also place a command button with caption OK or Open or whatever you prefer. Name it for example cmdOpen.

Create an event procedure for the On Click event of the command button:

Private Sub Command1_Click()
    If IsNull(Me.ctlCalendar) Then
        Me.ctlCalendar.SetFocus
        MsgBox "Please select a date.", vbExclamation
    End If
    DoCmd.OpenForm FormName:="frmPlannerDetail", _
        WhereCondition:="dtmAppointment = #" & _
        Format(Me.ctlCalendar, "mm/dd/yyyy") & "#"
End Sub

where frmPlannerDetail is the name of the planner detail form, and dtmAppointment is the name of the date/time field on which you want to filter.
