You suffered the typical symptoms of a Corrupted Profile.  Had you followed the instructions in the following article, you would be just fine now:

"How to recover the damaged user profile in Windows XP"
  < http://support.microsoft.com/kb/555473 >

Sometimes a simple Disk Check (chkdsk c: /f) will restore the old profile as well.

However, when Windows detects a corrupted profile, it then on-the-fly creates a new profile that you end up logging in to.  This explains why your screen look like it was the first time you had logged in.  This profile is temporary.  In other words, when you log out, the profile is deleted and anything in it is deleted as well.  So because you copied all of your files to this temporary profile, they all disappeared when you logged off.
