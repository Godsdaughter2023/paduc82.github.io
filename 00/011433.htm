ken sheridan say: In the AfterUpdate event procedure of each of the unbound controls on the parent form put code along these lines:

    Me.YourSubFormControl.Form.ControlOnParentForm.DefaultValue = """" & Me.ActiveControl & """"

NB:  YourSubFormControl is the name of the control in the parent form's Controls collection which houses the subform; ControlOnParentForm is the name of the control in the subform whose default value you want to set.  Note that setting the DefaultValue property does not initiate a new record in the subform, that only happens when the user either changes one of the values or inserts a value into another control.

a. user say:

Hi,

 

We apologize if we are not able to respond in a timely manner. As I understand your concern, you are trying to send an email to multiple recipients using the email addresses from your excel file. At this point, please be informed that once you copy and paste the email addresses from excel to your Outlook.com, the first email address will only be copied to the “To” field.

 

As a workaround, follow the steps below.

 

1. Open your excel file and then press Ctrl + F.

2. Click on the Replace tab.

3. On Find what field, type in .com

4. On the Replace with field, type in .com;

5. If other email addresses ends with .net, just replace it with .net;

6. Once done, copy all the email addresses and then paste it in your Outlook.com.
