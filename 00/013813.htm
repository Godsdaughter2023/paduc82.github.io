When you buy, it is my personal preference to insist on the "FPP" or Retail license for home and small business. In the Retail version you get the Installation disk built into exactly the same price you would pay for software only in the online download. If your brand new computer "melts down" in the warantee period, it will be replaced, but you will have to BUY another PKC license!

Hi Jaskaran,

Thank you for your interest in using Windows 10 Technical Preview.

Performance is not the issue with running Windows 10 on your HP Stream. Since the HP Stream comes preloaded with Windows 8.1 64 bit and Windows 10 uses the same system requirements as Windows 8.1

The problem is Storage (disk space).

If you want to run Windows 10 on your HP Stream 7, your only option is a clean install.

Please note, Windows 10 is in preview, you should not replace a production Windows 8.1 64 bit install with Windows 10 Technical Preview unless you have a backup copy.

Also, recent builds of Windows 10 Technical Preview requires double the amount of disk space because of a bug with driver duplication occurring during setup. So in addition to the minimum 20 GBs, you need an additional 20 GBs or more to support the installation.

If you really want to evaluate Windows 10 Technical Preview, Create a complete System Image Backup with Windows 8.1.
