lemp say:

From what I've seen, ESRV.exe is said to be either the Intel Driver Update Utility OR Sony VAIO Care.

Do you have a Sony computer?

Did you actually uninstall the Intel Drive Update Utility or Sony VAIO Care (if you have a Sony computer)?

If you uninstalled whatever application uses esrv.exe, you should no longer get an esrv.exe "application error," although you may well still get another error message to the effect that esrv.exe can't be found -- or something similar.

If you still get an esrv.exe application error, then you haven't uninstalled the correct application.

On my 64-bit Windows 7 Pro system, esrv.exe is described as the "Intel(R) System Usage Report" and it is located at

         C:\Program Files\Intel\SUR\QUEENCREEK\x64

If you have a 32-bit system, you may find it in C:\Program Files\Intel\SUR\QUEENCREEK\x86

Perhaps of some interest is the fact that although I have a whole slew of Intel applications in "Programs and Features," the "Intel Driver Update Utility" is not one of them.  Thus, despite what "the Internet" says, esrv.exe is not (or at least not necessarily) part of the Intel Driver Update Utility.  If I had to guess, I'd guess it's part of the "Intel Computing Improvement Program" -- which is another not-really-needed application.

Here's what I suggest:

1.  Open Windows Explorer (the file manager, not Internet Explorer) and type esrv.exe in the search box at the upper right corner.  If you have a large, slow hard drive, the search may take a while (If you see a message that "Searches might be slow in non-indexed locations: C:\.  Click to add to index ..." I suggest that you NOT click there). 

  • If a file is found, in what folder is it located?
  • If no file is found, please mention that fact.

2.  Download Autoruns from https://docs.microsoft.com/en-us/sysinternals/downloads/autoruns  Extract the contents of the zip file.  Right-click on either Autoruns.exe or Autoruns64.exe (depending on your system) and select "Run as administrator."  Respond as needed to the UAC prompt.  When the scan completes, select the "Everything" tab.  Click the "Binoculars" icon or press CTRL+F and search for esrv.exe.

  • If nothing is found, please mention that fact.
  • If an instance of esrv.exe is found, do the following two things:
    1. Clear the check box at the left end of the line in which esrv.exe appears
    2. Carefully scroll up in the Autoruns search results using the up arrow key until you get to a purple line.  This line will either show a folder icon or a regedit icon in the left margin.   If it is a folder icon, the line is most likely to say "Task Scheduler."  Otherwise, the line should begin with HKLM (or possibly HKCU).  Right-click on the line, select "Copy" and Paste the info into your reply.

If you found an instance of esrv.exe in Autoruns and cleared the check box, when you restart the computer, you should no longer get an error message.  Please confirm this.  Your responses to the other questions will help to determine which Intel utility you might want to uninstall.

How to repair and reboot to Windows 7:

a) Insert the Windows 7 System Repair DVD or Installation DVD and reboot.

b) Select the correct keyboard input method and click ‘Next’.

c) The System Repair tool will search for a valid operating system and will report that “Windows found problems with your computer’s startup options. Do you want to apply repairs and restart your computer?”

d) Click ‘Repair and restart’ and reboot again to the Windows 7 System Repair environment.

e) Click ‘Next’ at the keyboard selection prompt and when the ‘System Recovery Options’ screen appears verify that the “Use recovery tools that can help fix problems starting Windows” is selected.
Click the ‘Next’ button.

f) Now from the menu list select the ‘Command Prompt’ option.

g) At the DOS prompt type the following three commands:

bootrec.exe  /fixmbr

bootrec.exe  /fixboot (may return an 'Eliment not found' message)

bootrec.exe  /RebuildBcd

h) Close the command prompt window and click the ‘Startup Repair’ option.

i) The repair process may take some time, so wait until the process completes at which point you will see two messages “Windows cannot repair this computer automatically” and “Startup Repair cannot repair this computer automatically”.

This time the repair process may only that a few seconds and when prompted “Startup Repair could not detect a problem”, click ‘Finish’  and run the ‘Startup Repair’ option one more time, click ‘Finish’, and then remove the System Repair DVD and click ‘Restart’.  Reboot the computer.
