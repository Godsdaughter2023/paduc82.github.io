How do I create a button that clears designated cells content (but - Microsoft Community

b.s.t. say:

How do I create a button that clears designated cells content (but not formulars)?

An excel spreed sheet is reused every week for poduction but manually all the previous values need to be cleared before the following weeks values can begin to be entered. Could I get detailed directions on how to create a button (beleive it would be a macro button) that clears designated cells content (but not formulars) please? It would make peoples working lives so much easier (quicker and less errors to find). thank you in advance B.S.T.
jllatham say:

The basic VBA command to clear the contents of a cell or range of cells is:

Range("A5").ClearContents ' single cell on currently active sheet

Range("A5:B10").ClearContents ' cells A5, A6, A7, A8, A9, A10, B5, B6,B7,B8,B9 & B10 on currently active sheet

Now, if you do a .ClearContents on a cell that does have an actual formula in it, that formula is removed.

Notice also that I mentioned that the format of the command I showed works on the cells on the current active worksheet, and that may not be the sheet you really want to clear the cells on.  So you can be very specific and clear the contents on any sheet by telling the code which sheet, like this:

Sub ClearUserEnteredData()
  Application.ScreenUpdating = False ' makes things much faster, no flickering
  With ThisWorkbook.Worksheets("TheSheetNameAsShownOnItsNameTab")
        .Range("A1").ClearContents
        .Range("A5:B10").ClearContents
        .Range("G4:J12").ClearContents
   End With
End Sub

notice in this, that there is a period/dot in front of the word Range inside of the "With ... End With" block.

And the code above is exactly the kind you need for what you want to do!!  Just add it to a regular code module and change the name of the worksheet referenced in the With ... statement to the name as it appears on the sheet's name tab, and change the "A1", "A5:B10", "G4:J12" references to what ever cells you want to clear.  You can add more to the list as needed.  If you need to clear more cells on other sheets, just add another With ... End With block like this:

Sub ClearUserEnteredData()
    Application.ScreenUpdating = False ' makes things much faster, no flickering

    With ThisWorkbook.Worksheets("SomeSheet")
      .Range("A1").ClearContents
      .Range("A5:B10").ClearContents
      .Range("G4:J12").ClearContents
     End With
    With ThisWorkbook.Worksheets("SomeOtherSheet")
      .Range("R1").ClearContents
      .Range("X5:Y10").ClearContents
      .Range("B4:C12").ClearContents ' they don't have to be in any particular order!
     End With
End Sub

See this page on how to put code into a regular code module:

http://www.contextures.com/xlvba01.html#Regular

Now, to add a button to a sheet to make this work.  I personally kind of like to work with a TextBox instead of an acrtual button.  Add a text box to the worksheet and set its text to something useful such as "clear for new entries" and format it up to look pretty.  Then move your mouse cursor toward the edge of the textbox (or button if you chose to use one) until it turns into a narrow cross with an arrow-head at the end of each part of the cross - kind of a 4-pointed arrow pointing North, South, West and East all at once.  When you get that, RIGHT-click and choose [Assign Macro...] from the popup list.  Highlight the appropriate entry in the list that appears (ClearUserEnteredData if you use my example) and click the [OK] button.  That's it.  Now when you click on that "button" that code will run.
