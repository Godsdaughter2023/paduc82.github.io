arjay dom say:

Hi,

If you're getting certificate errors, it means that the website is having certificate problems.

Installing the certificates can help you get rid of the error on Internet Explorer:

  1. Once the error message appears, click Continue to this website (not recommended).
  2. Click on the Certificate Error button to open the information window.
  3. Select View Certificates, then click Install Certificate.
  4. Follow the on-screen instructions.
  5. On the warning message that appears, click Yes to install the certificate.

Note: Only install certificates from websites that you trust. If you think that the website is malicious, do not install the certificate.
