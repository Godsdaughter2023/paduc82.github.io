john korchok say:

The commands available for the QAT have always been a small subset of available commands. To illustrate this, click on the down-pointing arrowhead in the QAT and choose More Commands. The dialog opens to the QAT pane. Change the Choose commands from... dropdown to All Commands. 125 commands are displayed. Now click on the Ribbon tab and again display All Commands. About 1025 are displayed (I lost count a few times, but that's close). The QAT in Word 2016 for Mac has never come close to displaying all available commands, but until now, they've been the commands you're looking for.

In general, you can't substitute a single QAT command for an action that takes more than 1 selection action. With highlighting, you choose the highlighting dropdown to invoke the highlighting command, then you choose the actual color as the second action. If MS created QAT commands for these, they would have to have 1 for every highlight color.

However, you can create a macro to do this job. Here's the VBA code to highlight text in yellow:

  Selection.Range.HighlightColorIndex = wdYellow
End Sub

Once you create the macro, then you can add it to the QAT and have your own custom QAT buttons. Please post again if you need help creating a macro or making a QAT shortcut for it.
