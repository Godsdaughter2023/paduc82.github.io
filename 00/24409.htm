joeu2004 say: Michael wrote:
How do you stop Excel 2013 from showing percentages in 1000's.?For example, format the cell to percentage, type in 50 and it shows 5000% instead of 50%.

If you want to type 50 and have it interpreted as 50% (0.5), you might set the option to "Automatically insert decimal point".  In Excel 2010 (I don't have XL2013), click on Files, Options, Advanced, select the option and set (Decimal) Places to 2.  You also have to format the cell as Percentage.

However, that affects all numeric data entry thereafter.

I think it would be better to simply type "%" after "50".  Of course, that is hard to do if you want to use only the numeric keypad for data entry.

Otherwise, remember to deselect the option when you are done with your percentage data entry.
