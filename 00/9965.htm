florence c. say:

Hi Adrian,

 

Hotmail calendar can be exported in .ics format but cannot be opened using Word or Excel. You can import this information into a different calendar and use it.

 

To import your Hotmail calendar into different calendar, please follow the steps below:

  1. Log in to Hotmail Calendar
  2. Click "Share” in the top calendar menu, then select the calendar you wish to export.
  3. Select “Share this calendar”, then “Send people a view-only link to your calendar”.
  4. Click “Get your calendar links”.
  5. Click “ICS: Import into another calendar application” under the first section (“Links that show event details”) to get a popup containing the link to the ICS file generated from your Calendar.
  6. From the "ICS Links" message box, copy the URL and click "OK" close the dialog box.
  7. Paste the link into another browser window. Then replace "webcal" at the beginning of the address with "http" and hit "Enter".
  8. The browser will now download your calendar’s ICS as a .ics file

Once the export is complete, you can uncheck the "Send people a view-only link to your calendar" checkbox so that your calendar is no longer shared via a private URL.

shenan stanley say:
Put in your Microsoft Office XP installation CD.
 
1.     Start Word.
 
2.     On the Help menu, click Detect and Repair.
 
3.     In the Detect and Repair dialog box, do any of the following:
 
Click to select the Restore my shortcuts while repairing check box. By default, this check box is selected. If you select the Restore my shortcuts while repairing option, Detect and Repair will keep your custom Office shortcuts. If you do not select this option, Detect and Repair will remove your custom Office shortcuts.
Click to select the Discard my customized settings and restore default settings check box. If you select the Discard my customized settings and restore default settings option, Detect and Repair will set the following back to the state that they appeared in when you first installed Office:
  • The Assistant character selection.
  • Most-recently-used entries on the File menu will be removed.
  • The size of the program window for all programs.
  • Menu and toolbar position and any customizations.
  • The security level for each program.
  • View settings in the program, such as the Calendar view in Outlook.
  • You must re-enter your User Name and Initials when you restart your Office programs.
4.     Click Start.
 
5.     Click Ignore if the Close Office Programs dialog box displays the following message (where program is any Office program that is currently running):
            In order to correctly pick up or restore your settings, the following programs must be closed:
            Microsoft Word
 
6.     After the repair process has completed, click OK to the message:
            Microsoft Office XP (2002) Setup completed successfully.
