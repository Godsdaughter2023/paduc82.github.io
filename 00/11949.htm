What does WAD mean?

kyuta say: If you're comfortable with editing your registry, I found a fix here but I'll break it down for you.

First you'll need a registry editor for your WM device you can search Google for one.

Run the registry editor and browse to HKEY_CURRENT_USER\Control Panel\Sounds

you should then see 4 editable keys though there's only 2 you'll be editing, AllSounds and FileSizeLimit

I set my FileSizeLimit to 999999999 granted I doubt I'll ever use a file that big

Double click AllSounds and scroll to the end of the list and add \My Documents\My Ringtones\*.*

Now any file you have in \My Documents\My Ringtones\ or \SimCard\My Ringtones\ should show up.
