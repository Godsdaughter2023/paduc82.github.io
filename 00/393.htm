how to change the default format for comments

j.e. mcgimpsey say: AFAIK, there's no way to do this directly in MacXL (in Windows versions you can't do it within XL, but you can change the operating system tooltip format which is then used for XL comments).

You can use a macro to insert a formatted comment. Here's one that I have in my startup add-in - I replace the Insert Comment commands in the built-in menu and toolbars with ones that point to this:

    Public Sub NewInsertComment()
        Dim oComment As Comment
        With ActiveCell

jason vaizard kouros say:

How can you insert a picture within a cell?

Hi, I'd like to know how exactly could I insert an image within a cell and just make it...one and the same 'item' let's say. I actually want to know if I could even use the cell normally thereafter and even type on top of that picture as if it's nothing but a background (cell per cell situation answer ONLY! I know how to change the overall background, thank you).

peter thornton say:

A cell cannot have background image as a sheet can. However You can insert an picture "over" a cell, sized to same size as the cell and set to "Move and size" with the cell (eg cut/copy, move, resize row/column containing the cell). The cell doesn't know anything about the image though the picture knows what cell(s) it is over from its top-left and bottom-right cell properties.

The cell will continue to work normally and can contain values and calculate etc. The picture could have a transparent background or be opaque to see through to the cell under it. However when you click on the cell with the picture you'll probably select the picture, but you could move the cursor into the cell to select or maybe size the picture to say half the width of the cell.
