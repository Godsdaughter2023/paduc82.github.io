lydiajn say:

Hi,LIRY

You can also use the VBA. 

Try this code:

Sub FindAndHighlightAllCaps()
  Dim objRange As Range
 
  With Selection
    .HomeKey Unit:=wdStory
 
    With Selection.Find
      .ClearFormatting
      .Text = "[A-Z]{2,}"
      .Replacement.Text = ""
      .Forward = True
      .Wrap = wdFindContinue
      .Format = False
      .MatchWildcards = True
      .Execute
    End With
    Do While .Find.Found
      Set objRange = Selection.Range
      objRange.HighlightColorIndex = wdPink
      .Collapse wdCollapseEnd
      .Find.Execute
    Loop
  End With
End Sub

Thomas,

There is a recommended (non-critical, recommended) update that contains some improvements to Windows Update Client in Windows 7 Service Pack 1 (SP1).  This update was originally included in the

Since Microsoft Security Essentials uses the background services provided by portions of Windows Update to perform its own updates, some of these would likely be replaced as part of this recommended update.  If you don't have the Give me recommended updates that same way I receive important updates checkbox selected in Windows Update, these will not be installed unless you select them manually.

I can't say with 100% assurance that this is your problem, but it often does create such issues as this line in the About this update section for the original Windows Update Client for Windows 7 KB article mentions.

"An optimization that addresses long scan time for updates that's reported on some computers."

So in your case looking for the July Rollup within the Optional Updates to determine whether this has been installed might be worth the effort.

For the other Vista operating system user Shawn who jumped into this thread, this particular update has not been made available for Vista, even though we've been hearing of many similar issues relating to both MSE and critical Windows Updates with Vista as well.  The reason for this hasn't been officially stated, but possibly the fact that End-of-Life Support for Windows Vista occurs on April 11th, 2017 may have something to do with it.  Since Google has already officially ended Chrome update support for Widows Vista (IE 8 or IE 9) as of April 2016, there's really little reason to hold onto this operating system any longer than absolutely necessary and your primary objective should be to replace it entirely as soon as possible.

Another person often posting here who still has older systems with Windows Vista recommends disabling Windows Automatic Update and performing these manually, though I believe he still receives MSE updates when running this manually, so your problem may be different and related to the other recent problem you mentioned instead.
