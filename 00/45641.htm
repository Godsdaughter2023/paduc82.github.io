a. user say:

Hi,

Thank you for posting in Microsoft Community and being a part of Windows 10.

I understand your concern about accessing the 24 hours clock settings on lock screen. We will help you with this issue.

Try the below steps and check if you are able change the clock settings.

  1. Sign-in as an administrator to your computer.
  2. Press the +R keys from the keyboard to open the Run dialog, type regedit, and click on OK.
  3. In Registry Editor, navigate to the location to: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\SystemProtectedUserData
  4. Expand open SystemProtectedUserData, and expand open the long S-1-5-21... subkey for the user account that you want to change their lock screen clock time format.
  5. Under the long S-1-5-21... subkey for the user in the left pane, expand open AnyoneRead, and select LocaleInfo.
  6. In the right pane of LocaleInfo, double click on the TimeFormat string value to modify it.
  7. Type in the time format from the table below you want to display on the lock screen clock for the user, and click/tap on OK.
  8. When finished, close Registry Editor, and sign out of the Administrator account.
  9. To test, sign in to the user account that you changed their lock screen clock time format, and lock (+ L) the computer to see the lock screen with the clock's new time format.

net start bits

net start msiserver
