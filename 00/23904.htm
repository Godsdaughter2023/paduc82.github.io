I have recently switched to Mac, but before I was using a normal pc and i had no problems with outlook on this. 
talking moose say:
The Send & Receive schedule only applies to POP and iMAP accounts. It does not affect Exchange account.

Outlook for Mac automatically checks for mail once per minute for Exchange accounts. This time cannot be adjusted. You will either need to quit Outlook completely to stop receiving messages or use File menu --> Work Offline to set Outlook to completely stop receiving messages and then manually set Outlook to Work Online when you're ready to receive messages.

If all you really want is to not receive notifications when messages are received then select  Outlook menu -->  Preferences... --> Notifications & Sounds and disable all notifications and sounds.

If that doesn't meet your needs and you're not afraid to do a little bit of script work then you can automate Outlook checking for messages by using a schedule.

First, open the AppleScript Editor application found in /Applications/Utilities and copy/paste the following five lines of text into it.

tell application "Microsoft Outlook"

set working offline to false

delay 300 -- number of seconds to receive messages

set working offline to true

end tell


Select File menu --> Save As... and save this script as something like "Timed Schedule.scpt" to a permanent location on your computer.

In Outlook, select  Outlook menu -->  Preferences... --> Schedules and click the  +  (plus) button to create a new schedule with the following details:

When: Timed Schedule
Action: Run AppleScript
Script: select your newly created script

Edit the Timed Schedule options to run the schedule when you want. Save this schedule with a name of your choosing. Close these windows and when everything is ready select File menu --> Work Offline to stop receiving email until the schedule fires the script.

The only downside to this script is that while it is running Outlook will be unresponsive to you. You will not be able to scroll, click or edit any messages. Set the delay in the script to the smallest number possible (maybe 60 seconds) if you feel that you will receive all new messages from your server within one minute. You might experiment with 30 seconds or less.

Finally, please be sure to let Microsoft know you'd like to see this as a permanent feature in future versions by using the Help --> Send Feedback mechanism in any Office application.

Hope this helps!
