Important:The backup operation fails because Automated System Recovery (ASR) unnecessarily returns an error for noncritical disks. The expected behavior is that ASR returns an error when critical disks are not available. ASR then logs a Warning event for noncritical disks. However, ASR returns errors when both critical disks and noncritical disks are not available.

Note: Critical disks are disks that are required by the operating system, such as the disks that save the system volume and the boot volume. Whereas, noncritical disks are disks that are used to store user data or application data.
