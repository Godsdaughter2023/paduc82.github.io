With Outlook.com, you can recover email that might have been accidently deleted from your inbox. This is also helpful if your account has been hacked, because hackers often delete all the messages in an account. 

To recover deleted email

In the left pane of the Outlook.com window, click the Deleted folder. At the bottom of the window, click recover deleted messages.            

Outlook.com will recover as many messages as possible and put them in your Deleted folder. If you don't see the deleted messages that you wanted, it means they are permanently lost.            

When the recovered messages are back in your Deleted folder, you should move the important ones back to your inbox or personal folders because the Deleted folder gets cleaned out periodically.

Note

This feature doesn’t work for child accounts. This way, parents can be confident that when they delete messages from their children’s accounts, they stay deleted.

rodney zhang msft say:

Hi Carl,

 

Thank you for sharing the experience.

 

In your case, you may try the steps below to see if you can insert a chart web part:

 

At the beginning of the steps, we need to enablecustom script. If we enable custom script in SharePoint Admin center, it will take about 24 hours for the change to take effect.

If you want to take effect immediately, after selecting to allow, we can try the steps below:

  1. Download and install SharePoint Online Management Shell.
  2. Run as an administrator and type the following commands.

Connect-Sposervice, then input your admin site URL.

Set-SPOsite <the Group SiteURL> -DenyAddAndCustomizePages 0

 

After trying the steps above, here is the primary steps to insert a chart web part:

1.       Go to: Settings > Site Settings > Web Designer Galleries > Web parts.

2.    Click the FILES tab(in the classic experience).

3.   Click New Document.

4.       Scroll down the list and look for: Microsoft.Office.Server.WebControls.ChartWebPart.

5.    Check this item.

6.    Scroll back up to the top, and then click the Populate Gallery button.

7.    Wait for it to refresh.

8.    Then the next time you open a web page in edit mode and you'll see the Chart web part there.

Meanwhile, if you want to use pie chart, you can edit the web part after inserting it.

Rodney
