doug robbins - mvp office apps & services (word) say:
Change the mail merge main document type to Directory.

doug robbins - mvp office apps & services (word) say:

An Access Report can be sent to Word where it will end up as an .rtf file, but the formatting is often less that desired.

Using your present approach, if each record produces a single page in the document created by the mail merge, you could use a { SECTION } field in the mail merge main document and as there will be one page to each section, the pages will be consecutively numbered.

If the output results in more than one page for each record, Use Ctrl+A to select the whole of the document created by the mail merge and then on the Insert Tab of the Ribbon, use the Page Number pulldown in the Header and Footer section of the Ribbon and select the Format Page Number dialog and check the Radio Button for the numbering to continue from the previous section.
