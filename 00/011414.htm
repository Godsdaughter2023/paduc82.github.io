hansv mvp say:

You have posted your question in the English language forums, so I'll reply in English first.

I assume that you meant conditional formatting instead of unconditional formatting.

I will also assume that A2 should be formatted with color y if A2 is between F2 and G2, otherwise the two conditions aren't mutually exclusive.

 

Select cell A2, or a range within column A starting at A2 if you wish to format multiple cells.

 

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Select 'Use a formula to determine which cells to format'.

Enter the formula

 

=AND(B2="", C2="", A2>E2, A2<F2)

 

Click Format...

Specify the desired color x in the Font or Fill tab.

Click OK twice.

 

Click Conditional Formatting > New Rule... again.

Select 'Use a formula to determine which cells to format'.

Enter the formula

 

=AND(B2="", C2="", A2>F2, A2<G2)

 

Click Format...

Specify the desired color y in the Font or Fill tab.

Click OK twice.

bill manville say:

You could have one block which will populate with the details of a row from Master selected by a number in a cell.  So, for example if that number is in E5, change the formulas from e.g. =Master!G2 to =OFFSET(Master!$G$1,E5,0).

Then by changing the number in E5 you will get the data for a different row.
Note that 1 corresponds to the first data row in the Master worksheet, ie row 2.

If you need multiple sets of data to be visible simultaneously then copy the whole block to 11 rows lower and change E16 to be =E5+1.

That block can then be copied down to make as many instances as you need (N) and you will get all N updated when you change the number in E5.

bill manville say:

You wanted to be able to copy and paste a block of cells and have the new block reference a different row in the source table.

One way to do that would be to select the cells in the new block and Home > Find & Select > Replace: M with N where M is the row number the formulas are currently referencing and N is the row number you want them to reference.

A neater way to do this is to have a row number stored within each block of cells and have the formulas use that row number to access the data from the source table.  I suggested E5 as a cell you might use for that purpose in the first block (and then E16, E27 etc in subsequent blocks).  You can choose to use a different cell by making appropriate changes to the formulas I suggest.  For now, put 1 in E5

Now we need to change the formulas to use the row number.

For example where you have =Master!G2, change this to =OFFSET(Master!$G$1,E5,0).

This formula says starting from G1 in Master go down the number of rows in E5 (and stay in the same column) and give me the value in that cell - so when E5=1 you will get the value from G2.

Once you have changed all the formulas in that first block in this way, changing the value in E5 will show the data from a different row - note that 1 in E5 gives you the first data row (row 2); 3 gives you row 4 etc.

Now you can make copies of this block if you wish and, to get the data you want you just have to set the appropriate row number in the cell equivalent to E5.

So, if you want to have 5 blocks each one 11 rows below the previous one, showing the data from the first 5 rows in Master you can copy and paste the top block and then set E16 to 2, E27 to 3 etc.

If you had 100 such blocks it would be tedious to set the row number values individually.  So having made the first copy you could make E16 contain =E5+1.  Then copy and paste the second block to make the remaining blocks and they will automatically reference successive rows in the Master sheet.  And then when you change E5 to 101 you will automatically get the data from rows 101 to 200.
