a. user say:

Hello Megz,

Thank you for posting your question on Microsoft Community Forums.

I understand that you wish to make your flash drive password protected for the security of your data. However, Microsoft does not provide any software for password protection of any portable storage media. But, here's what we can do:

Encrypting File System (EFS) is a feature of Windows that you can use to store information on your hard disk in an encrypted format. If you backup your data in a flash drive, you may keep the data in the encrypted format which is user account dependent. If you access the data on the same computer and on the same user account, you need not to decrypt the data. However, if you try to access the data in some other user account or other computer, you need to have the EFS certificate backed up to decrypt the data. This is described below in the following links.

Encrypt or decrypt a folder or file

http://windows.microsoft.com/en-us/windows/encrypt-decrypt-folder-file#1TC=windows-7


Now, as you encrypt data on your computer, you need a way to recover that data in case something happens to the encryption key. Please refer to the following article on how to backup the EFS certificate:

Back up Encrypting File System (EFS) certificate

http://windows.microsoft.com/en-us/windows/back-up-efs-certificate#1TC=windows-7

On the other hand, Microsoft provides Bit locker to password protect the drives but that cannot be installed on Windows 7 Pro so if you want you might search online using your favorite search engine and download a third party software for password protection of your flash drive.

Disclaimer: Using Third Party Software, including hardware drivers can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the use of Third Party Software can be solved. Using Third Party Software is at your own risk.

Hope this information is helpful. If the issue still persists, please post back for further assistance, we'll be glad to assist you.

Defender - Microsoft Community

bairn_966 say:

Defender

Can you use defender to empty the recycle bin

sarahkong say:
No but you can create a task to empty your recycle bin on a schedule if that is what you are looking for.

Go to your search and search for Task Scheduler and open it.
Right click the Task Scheduler Library and select New Folder
Name the folder what you want. For example mine is called My Tasks for all the task that i manually create.
Highlight the folder and then click Create Task.
On the General task enter the name Empty Recycle bin.
On the Triggers click New
Select to begin the task on a schedule. (or you can do it when you log off your machine, this is what I do)
On the Actions tab, click New.
Under Settings, on Programs/script enter cmd.exe.
Under Settings, on Add arguments enter the following argument and click OK:

/c "echo Y|PowerShell.exe -NoProfile -Command Clear-RecycleBin"

Click OK to complete the task. And that's it.

You can test if you want by right clicking the task and selecting Run.

Offline Files/Folders

arya s asok say:

Hi HKHaggis,

 

With Offline Files in Windows Vista and Windows 7, you can specify a limit for the total size of the local cache. This limit includes automatically cached files and manually cached files. There is also a limit (within the total local cache size limit) for the amount of disk space used by automatically cached files.

 

For example, you can specify that the total cache size-limit for storing offline files on your computer is 5 gigabytes (GB), and within that limit, only 2 GB can be used to store files that are cached automatically. With this limit, files that were cached automatically are removed on a least-recently used basis as the disk space used by this type of files approaches the limit you specify.

 

Files that were cached manually are never removed from the local cache. When the total local cache size limit is reached and all files that were cached automatically have already been removed, you can not make files available offline until you specify a new limit or delete files from the local cache by using the Offline Files control panel item

 

These two local cache size limits can be specified in theDisk Usage tab of the Offline Files control panel item.

To change the Local Cache size limit, follow the steps below:

 

a. Click on Start, click on Control Panel.

b. In the search box type Offline files or click on Sync center.

c. Click on Manage offline files and click on disk usage tab.

d. Click on Change limits button.

e. Reduce the maximum amount of space for temporary files can use.

f. Click on Ok.

g. You can also click on to delete all the temporary files from there.

 

 

For more information about working with offline files access the link below:

http://technet.microsoft.com/en-us/library/cc749449(WS.10).aspx

 

Note the above link is for Windows Vista and the same holds good for Windows 7.

Hope this helps. Let us know the result.

david. f say:

Hello, HKHaggis

If the program needs to create temp files and it chooses the same place as the file, it will create the files there. If you are referring to a Microsoft Office program, like Word or Excel, it does create temp files in the same location as the original file for easy recovery should there be an unexpected outage.

Folders that are set to be available offline will have a local copy of the entire folder for use when there is no connection, this behavior is to be expected. If you are concerned with the amount of space this takes, you will need to adjust the settings using the instructions that Srinivas gave.
