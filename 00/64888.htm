a. user say:

Hi Phaneesh,

 

When the computer enters sleep or hibernation mode, some of the devices are turned off completely to save power. When you wake the computer, sometimes devices like network adapters face issues in turning ON. To prevent this you can turn off the deep sleep by following these steps:

 

a)      If the computer has recently waken from sleep mode, restart the computer to make sure that the network adapter is turned ON.

b)      Open Device Manager by clicking the Start button, clicking Control Panel, clicking System and Security, and then, under System, clicking Device Manager.‌ If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

c)       Expand Network adapter section, locate your network adapter, right click and select properties.

d)      In the power management tab, uncheck the box against “Allow the computer to turn off this device to save power”.

e)      Restart the computer and then try to install the driver and check.

 

Try these steps and let us know the results. We’d be happy to help if you need any further assistance.
