ossiemac say:

Yes it is a real pain when you copy a cell with conditional format and paste to an adjacent cell and it creates a new set of rules.

 

When you copy a cell, when selecting the paste area, if you include the cell being copied then the Applies to field of the Conditional Format is expanded to include the new cells without creating additional format rules.

 

If you place the cursor on the little black square bottom right of a cell to be copied and drag to AutoFill then the Conditional format Applies to range is expanded without creating additional conditions.

 

To fix your workbook that now has the additional rules:

  1. Select all cells with the Conditional format except the first one.
  2. Select the Clear icon in the Editing block at the far right of the Home ribbon and from there select Clear formats. (Do NOT use Right click and Cear Contents)
  3. Select the first cell that still has the Conditional Format and Right click and Copy.
  4. Select all of the cells to apply the conditional format including the copied cell.
  5. Right click and Paste special -> Formats.
