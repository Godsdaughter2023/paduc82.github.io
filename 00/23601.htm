don cuthbert say:

Digital Entitlement to run Windows 10 is recorded by Microsoft per device. Once each device has activated a public release of Windows 10, it gains permanent Digital Entitlement to Activate that edition (eg. Home or Pro, etc.) of Windows 10.

So, no matter what OS you install in future on the device, it does not remove the ability to Activate any non-expired build of Windows 10 that you install.

This means that as long as you install a non-expired Windows 10 or Insider Preview build, it has Digital Entitlement to activate Windows.

The only concern you might have is if you have never activated a Windows 10 public build on a device after July 29, 2015. If you do not do so prior to July 29, 2016, you will miss out on the free upgrade offer.

You should use the Windows license from your tablet on the tablet, so as not to waste another device's license on it.  I don't think you need to enter that BIOS-stored product key during install at all, as long as you install the matching Windows 10 edition to the Windows 8 that it shipped with. If it came with Windows 8 Single Language or N or Home, do not attempt to install Windows 10 Pro or any edition that does not match the original Windows 8.
