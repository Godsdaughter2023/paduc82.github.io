This normally works for me when I have update problems.

Resetting Windows Update Components will fix those corrupted Components. To reset Windows Update Components, follow these steps:  

  1. Run Windows Powershell (Admin). You may have to repeat Step 2 during the process as the listed service sometimes restart automatically.

  2. Stop BITS, Cryptographic, MSI Installer and Windows Update Services. To do this, type the following commands:  

    1. Net stop wuauserv  

    2. Net stop cryptSvc  

    3. Net stop bits  

    4. Net stop msiserver  

  3. Rename SoftwareDistribution and Catroot2 folder. To do this, type the following commands:  

    1. Ren C:\Windows\SoftwareDistribution SoftwareDistribution.old  

    2. Ren C:\Windows\System32\catroot2 Catroot2.old  

  4. Restart BITS, Cryptographic, MSI Installer and Windows Update Services. To do this, type the following commands:  

    1. Net start wuauserv  

    2. Net start cryptSvc  

    3. Net start bits  

    4. Net start msiserver  

    5. Restart your computer and try to install Windows Update.

G1 contains the total accrued, formula: A1*C1%/360*F1 (ie, 564.44)


Did you mean 569.44?

     

You formula in G1 should be,
=IF(E1<B1,0,A1*C1/360*F1)
... or,
=IF(TODAY()<B1,0,A1*C1/360*(E1-D1-1))
... to get a zero value. For blanks that would be,
=IF(E1<B1,"",A1*C1/360*F1)
... or,
=IF(TODAY()<B1,"",A1*C1/360*(E1-D1-1))
