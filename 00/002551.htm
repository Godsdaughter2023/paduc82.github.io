Use an If statement to check if the drive is found and then save there if found or to C: if not found.

This is very basic, you'll need to expand on it.

If fn_validate_drive ("J:") = False Then

' drive J: not found, save to C:

ActiveWorkbook.SaveAs Filename:="C:\" & ActiveWorkbook.Name

Else

' drive J: was found, save to J:

ActiveWorkbook.SaveAs Filename:="J:\" & ActiveWorkbook.Name

End If
