Sequential Numbering

lr-jl say: For a single user database, in the default value of the control, use the DMAX function + 1

=DMAX("[Control Number]","yourtablename","")+1

marshall barton say:

There is a potentially major time lag between applying the DefaultValue (when the first character is entered) until the record is saved.  That leaves the door wide open for two or more users to get the same number.

 

Much safer to wait until the record is about to be saved and use VBA code in the form's BeforeUpdate event procedure:

 

If Me.Newrecord And Not IsNull(Me.[Control Number]) Then

    Me.[revision field] = Nz(DMax("[revision field]", "your table", _

                                    "[Control Number]=" & Me.[Control Number]), 0) + 1

End If

 

Note that this has nothing to do with how the control number is entered, but if you also want them calculated automatically, then use similar code in a "New Control" command button :

 

If IsNull(Me.[Control Number]) Then
    Me.[Control Number] = Nz(DMax("[Control Number]", "your table") , 999) + 1

End If

ken sheridan say: You need two tables for this, one for control numbers in which the ControlNumber column is the primary key, another for the Revisions in which a ControlNumber column is a foreign key and which also a  Revision column.  The primary key of the Revisions table is a composite one of ControlNumber and Revision.

For data entry you can use a form based on Revisions in which an existing control number can be selected in a bound combo box, cboControlNumber say, whose RowSource property would be:

SELECT ControlNumber FROM ControlNumbers ORDER BY ControlNumber;

To enter a new control number automatically you can use a button in the form to insert a row into the ControlNumbers table and then requery the combo box on the form with code like this:

Dim intNewControlNumber As Integer
Dim strSQL As String

intNewControlNumber = Nz(DMax("ControlNumber","ControlNumbers"),999)+1

stSQL = "INSERT INTO ContrlNumbers(ControlNumber) " & _
     "VALUES(" & intNewControlNumber & ")"

CurrentDb.Execute strSQL, dbFailOnError

Me.cboControlNumber.Requery
Me.cboControlNumber = intNewControlNumber
cboControlNumber_AfterUpdate

The next revision number can then be automatically computed using the method illustrated in the  'Sequential Numbering by Group' option in the demo file CustmNumber.zip in my public databases folder at:

https://skydrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

If you have difficulty opening the link copy its text (NB, not the link location) and paste it into your browser's address bar.

Code in the cboControlNumber control's AfterUpdate event procedure would then call the function to get the next revision number.  This code would be similar to that in the AfterUpdate event procedure of the cboGender control in the frmContactsByGender form in my demo.

Based on a US, AT&T, 920 in Settings>Speech is an option to read aloud text messages, and you can select, always, BT, Headset, or BT and headset.
