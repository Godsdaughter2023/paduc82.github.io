a. user say:

Hi,

In Outlook, you can only request read receipts and delivery notifications to all messages that you send or to a single message. However, you can achieve selective automatic read receipt requests by creating an outgoing message rule. Here's how:

  1. Click the File menu, and then select Manage Rules & Alerts to open Rules Wizard.
  2. Click New Rule.
  3. In the Start from a blank rule section, choose Apply rule on messages I send.
  4. In the next screen, select sent to people or public group.
  5. In the bottom screen, click people or distribution list or people or public group to select the contacts or addresses from your Address Book. Outlook will request read receipts when you send emails to these contacts.
  6. Click Next to go to the Actions screen, and select notify me when it is read.
  7. Click Next to advance to the next screen to set any exceptions when needed.
  8. Click Next to give the rule a name. For Instance: Request Read Receipt.
  9. Click Finish and then OK to close the open windows.
