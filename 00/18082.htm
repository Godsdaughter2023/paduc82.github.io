ignaciochavezlt say:

Limit number of characters in a text box using Design Mode

Original Title: "Design Mode"

 

In Design Mode I am trying to limit the number of characters that can be typed into a text box. How do you format that?

stefan blom say:
With a text form field, you can specify the number of characters in the Properties dialog box, which you can display by double-clicking the field (in design mode). The limit will be in effect after you have enforced protection (in the Restrict Formatting and Editing task pane) on the document.

Note, however, that you can't prevent people from entering excessive returns/paragraph marks by pressing the Enter key. To work around this, place the field in a table cell whose dimensions are fixed (hide or display borders as desired).
