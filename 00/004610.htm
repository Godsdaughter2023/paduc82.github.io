bob jones aka: cybertaz say:

What Daniel doesn't mention & may not have been clear is the statement in my reply  that the Shape must be a floating object... IOW, it must have Text Wrap applied to it. The ones you're inserting are In Line with Text rather than floating.

IIRC, the default in Word 2011 is In Line. To change the default go to Word> Preferences> Edit then select Square from the list of Insert/paste pictures as: choices.

Rather than changing the Default you can apply/change the Text Wrap property of the objects individually by either;

  • Right-click the Shape then use Text Wrap in the contextual menu, or
  • Select the Shape & use the Wrap Text button on the Format tab of the Ribbon, or
  • Select the Shape then go to Format> Shape - Layout

The property of any Shape can be changed at any time.
