Error message "A recently installed program has disabled the Welcome screen and Fast User Switching" while turning on the Welcome Screen Updated: June 26, 2005
Updated: June 26, 2005
Send your feedback Symptom

When you attempt to turn on Fast User Switching or Welcome Screen, you may receive a message stating that a program used CSGina.dll ( or any other DLL ginamsi.dll etc ) to disable the Fast User Switching and the Windows Welcome screen features. The exact error message follows:

A recently installed program has disabled the Welcome screen and Fast User Switching. To restore these features, you must uninstall the program. The following file name might help you identify the program that made the change: <DLL filename varies >

Cause

This issue occurs if a third-party GinaDLL value is set in the registry, thus disabing the Microsoft Graphical Identification and Authentication file (Msgina.dll) automatically. CSGINA.DLL is a filter driver of Cisco Systems VPN Dialer program. The filename may vary depending upon the software that you've installed.

Resolution

To resolve the problem and restore the Welcome Screen and Fast User Switching, you need to reset the GinaDLL entry using these steps:

  • Click Start, Run and type REGEDIT
  • Navigate to the following key:

HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows NT \ CurrentVersion \ Winlogon

  • Backup the key by exporting to a file.
  • In the right pane, right-click GinaDLL , and then click Delete. (You need to remove the whole value, not just its Value Data )
  • Close Registry Editor

slance310 say:

Hi,

Sorry... I should have mentioned that...

If you open the CD or the installation file\folder, you should see something like "SETUP" or "Install" which launches the installation...

You want to right click on it, select Properties, and then the Compatibility Tab...

Part way down you will see a box for Compatibility Mode...

Check the box and then the menu. You will see options for choosing a previous version of Windows that your printer worked on.

You could try Windows 8, but it might be better if you select Win 7...

I don't know which version you started with..

I have a Lexmark X5340 that uses drivers for XP SP3, and this works for it...

I have encountered issues in which uninstalling and reinstalling did not work in both Win 8 and Win 10...

What resolved the problem was to go to Devices and Printers, right click on the printer, and select "Remove"...

If the printer also has a FAX, Remove it first... be sure not to remove the Microsoft FAX if it is there...

After Removing it, physically disconnect it from the computer..

Go to Control Panel... Administrative Tools... Services, and find "Print Spooler"

Right click on it and select "STOP"

Then uninstall the software\drivers and reboot...

Again stop the Print Spooler, then reinstall the software\drivers after setting the Compatibility...

Depending on how much spare time you have, you could just try Uninstall\reinstall in Compatibility Mode first... If that works then the other stuff won't be necessary...

The bad part is that if it doesn't work, you will need to do it again...

BTW... For some reason it is very common for the printers to not work after upgrading to Win 8, 8.1, and 10.

I have been running Win 10 Preview builds for a year now, and everytime there is a new build, I have to reinstall my printer..
