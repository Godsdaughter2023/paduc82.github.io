It means it should be an issue with the installation on the Mac. Here are some suggestions for you

1. Sign Out from the OneDrive app menu bar. Then sign-in back again.

If sign out and sign in did not work, try a clean OneDrive installation:

  1. Unlink OneDrive by click OneDrive icon, choose Preferences, then About, click Unlink.
  2. Remove the app by following these steps:
    1. Open the Applications folder.
    2. Drag the blue OneDrive cloud to the trash.
    3. Enter your MAC credentials.
    4. Click OK.
  3. Remove the additional OneDrive files.
    1. Press Apple (COMMAND) + SHIFT+ G in finder, or use Terminal.
    2. Go to /Users/<name>/Library/Containers
    3. Find the 2 folders
      • com.microsoft.skydrive-mac
      • com.microsoft.SkyDriveLauncher
    4. Drag them to the trash.
  4. Remove the locally stored OneDrive folder.
  5. If you installed with the default settings, there will be the OneDrive folder under your name.
  6. Drag it to the trash.
  7. Now, you can re-install OneDrive.

*None of these steps actually delete your files from OneDrive but you should always have a back-up of all your files, just in case you ever lose access to your OneDrive. However, all data that was synced before will need to re-sync after following these steps. Files that have not yet been synced won't be on your OneDrive yet, so those won't be recovered if you follow these steps.

talking moose say:
Two things to try:
  1. Create a dummy POP or IMAP account. The information doesn't need to be correct and you don't need to connect to it. Set this as your default account. Restart Outlook. Now, set your Exchange account as default and see if this corrects the issue. You should be able to delete the dummy account afterward.
  2. Your database has some corruption causing this problem because this is not a preference you can set. Meetings always go to the default account's Calendar. I suggest you begin a new identity by launching Outlook with the Option key held down. This will launch the Database Utility application. Click the " + " button to create a new identity and give it a name. Set this as the default identity. Quit and launch Outlook normally. You'll need to configure your Exchange account, rules, preferences, etc., but this should take care of the issue.
Hope this helps!
