I was wondering if it is harmful for my HDD if I run disk defragmentation frequently, say once in a week or a month. Actually in my computer I have turned on scheduled defragmentation, as recommended by the OS, but one of my friends told me that it might damage my HDD. Is it true or simply bogus? If true, how frequent should I perform the operation?

Thank you. 
cmdrkeene say:

It really comes down to how you use your computer.  But my personal recommendation would be to set it monthly, then just ignore it.  With modern PCs and modern file systems like NTFS on Windows 7, the impact of file fragmentation is almost impossible to see a benefit.


In fact, Windows defrag will ignore large file chunks during defrag because there's almost no benefit to arranging them. Monthly schedule lets Windows keep things tidy.


Defrag is harmful to an SSD hard drive (solid state drive) and Windows will not let you defrag them.  But for other hard drives, the stress of defrag isn't considerably more than normal usage and wouldn't noticeably affect the lifespan of your hard drive (which averages about 5 years life for most hard drives).

andre da costa say:

With the release of Windows 10 November Update (1511), and later versions such as the Windows 10 Anniversary Update (1607), you can switch directly to a 64 bit version of Windows 10 with or without a product key.

Before switching, read the following:

Windows 10 32 or 64 bit – Which is the Right Architecture for You?

  • Switching to Windows 10 64 bit is free, you don't have to pay for it. As long as you have a qualifying Windows 7 or 8 or Windows 10 license, you are entitled to use it.
  • Switching requires that you perform a new installation, that means files, applications and settings will be deleted.
  • You will have to reinstall applications and native 64 bit drivers.
  • Backup before migrating: How to: Create a system image in Windows 10

Step 1: Download Windows 10 64 bit then create a bootable copy:

How to download official Windows 10 ISO files

Step 2: Boot from the 64 bit Windows 10 install media, then perform a custom or clean install:

  • How to: Perform a custom installation of Windows - Choose this option if you want to recover files from Windows.old
  • How to: Perform a clean install of Windows 10 - Choose this option if you want to erase the drive, files and applications will be deleted.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers.
