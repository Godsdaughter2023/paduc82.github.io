Excel concatenate columns based on priority of the row - Microsoft Community

peterdarragh say:

Excel concatenate columns based on priority of the row

4 15 21 2 12 5
           
R S R F L N
C O T H I A
O S S      
           

I need to concatenate the following table based on the proirty of the column order listed vertically , the proirty of the column may change so the formula has to account for this by looking up the number first and then listing the letters. In the example above I need to end up with the following letters FHRCONALISOSRTS

Any suggestions would be great

hansv mvp say:

Press Alt+F11 to activate the Visual Basic Editor.

Select Insert > Module.

Copy the following code into a module:

Function ConcatenateWithPriority(DataRange As Range, PriorityRange As Range) As String
    Dim r As Long
    Dim m As Long
    Dim c As Long
    Dim c2 As Long
    Dim n As Long
    Dim cols() As Long
    Dim s As String
    m = DataRange.Rows.Count
    n = DataRange.Columns.Count
    If PriorityRange.Columns.Count <> n Then
        ConcatenateWithPriority = CVErr(xlErrRef)
        Exit Function
    End If
    ReDim cols(1 To n, 1 To 2)
    For c = 1 To n
        cols(c, 1) = PriorityRange.Cells(1, c).Value
        cols(c, 2) = c
    Next c
    BubbleSort cols
    For c = 1 To n
        c2 = cols(c, 2)
        For r = 1 To m
            s = s & DataRange.Cells(r, c2).Value
        Next r
    Next c
    ConcatenateWithPriority = s
End Function
 
Sub BubbleSort(cols)
    Dim i As Long
    Dim j As Long
    Dim n As Long
    Dim tmp As Long
    n = UBound(cols)
    For i = 1 To n - 1
        For j = i + 1 To n
            If cols(i, 1) > cols(j, 1) Then
                tmp = cols(i, 1)
                cols(i, 1) = cols(j, 1)
                cols(j, 1) = tmp
                tmp = cols(i, 2)
                cols(i, 2) = cols(j, 2)
                cols(j, 2) = tmp
            End If
        Next j
    Next i
End Sub

Switch back to Excel.

Use a formula like this to concatenate the values:

=ConcatenateWithPriority(A3:F5,A1:F1)

where A3:F5 is the data range, and A1:F1 the priority range.

Don't forget to save the workbook in a macro-enabled format (.xlsm, .xlsb or .xls, but not .xlsx), and to enable macros when you open the workbook.
