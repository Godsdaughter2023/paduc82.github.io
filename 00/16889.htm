Form auto fill based on a different field value.

Lookup Values of Dept, Code and Locations will be predefined.

jay freedman say:
Sorry, I thought that both your primary and secondary fields were dropdowns. If the secondary fields are just text form fields, the technique is similar but actually a bit simpler.

Where Greg's code uses Select Case statements to add multiple entries to the secondary dropdown field, you can replace that with simple assignments. For the example you gave, it would look something like this:

Sub OnExitDeptList()
    With ActiveDocument
        Select Case .FormFields("Dept").Result
            Case "HR"
                .FormFields("Code").Result = "001"
                .FormFields("Location").Result = "Atlanta"
            Case "Finance"
                .FormFields("Code").Result = "002"
                .FormFields("Location").Result = "Chicago"
            Case Else
                .FormFields("Code").Result = ""
                .FormFields("Location").Result = ""
        End Select
    End With
End Sub

You would insert a Case (before the Case Else) for each department listed in the Dept dropdown. Tip: The items in the Case statements are "case-sensitive" -- they must match exactly the entries in the dropdown.
