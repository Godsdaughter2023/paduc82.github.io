Put the following code into the Document_Open event or into an AutoOpen macro.

With ActiveDocument
    If .ProtectionType = wdNoProtection Then

If MsgBox("Are there any problems with the document?", vbYesNo + vbQuestion) = vbNo Then

ossiemac say:

This looks to be a bug with Excel that needs to be reported, however is there a workaround?

I don't think it is so much a bug as something that was never intended to work that way. Are you selecting more than one item in the listboxes (ie. multiselect)?

If you are using single selection then if you dispense with the frame and use the following Click event code then it aligns the options list in the alternate listbox with the one where the selection has been made in the first listbox.

I am assuming that this is what you are attempting to do because the second selection is most likely to be found around the same position as the first selection. If not what you are intending then I don't understand the use of the frame at all.

Private Sub lbA_Click()
    lbB.TopIndex = lbA.TopIndex
End Sub

Private Sub lbB_Click()
    lbA.TopIndex = lbB.TopIndex
End Sub
