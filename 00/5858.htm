Actual answer depends on what you actually purchased and from where.i.e. a single use key card from a 3rd party vendor versus a subscription

It depends ...

Since you bought Office separate from the laptop it is likely you have a retail license which does allow transfers.

You either need your original 25 character Office Product Key or you originally took advantage of the offer to "Register" your product key. Try logging in to the MyAccount site with your normal email address and see if the license is associated with your email account: HTTPS://OFFICE.COM/MYACCOUNT

. uninstall Offfice from the old computer.

https://products.office.com/en-us/download-back...
. after you download it, run the setup file
. you will be prompted to enter your product key.

Manage - Stuff you can Do in MyAccount: Install / Deactivate / Offline Installer / Downgrade / Transfer - WIKI
https://answers.microsoft.com/en-us/msoffice/wi...

If you want to “transfer” an install to a new computer:
. • log in to the MyAccount site http://office.com/myaccount
. • follow instructions in the wiki to “Deactivate” the old install
. • follow instructions in the wiki to Install on the new computer

    kosh vorlon say:

    Hi Subham.

    You say "a malicious software" - how do you know this occurred, do you have any details like the Full name of it (or absolutely any other details you know about it), any idea how and where you got it, what you used/tried and how you got rid of it (if you did or think you did and believe this is just a remaining side effect), and when this all occurred?

    Before starting, make sure you're fully backed-up, have made a backup copy of the registry, and have created a system restore point (you'll be happy you did if things go sideways).

    To proceed, you'll need to make sure you're logged in as an Administrator and when opening the specific programs, right click and Open as Administrator.

    "Some settings are managed by your organization" can be due to Group Policy changes which can be caused by some infections.  Are you certain Windows Defender itself is functioning?  Please see the Answer here: http://answers.microsoft.com/en-us/windows/forum/windows_10-security/windows-10-some-settings-are-being-managed-by-your/46094760-f198-46bd-a50e-111da80967c6 which applies to Real time protection as a whole.  If that specific screen is fine, then try the same parent location of Windows Defender and check the options there and for the various tabs to see if a similar problem exists for these exclusions which can be corrected in a similar fashion.

    I see that you tried regedit and got "unable to delete all specified values".  Did you attempt to delete all of them from HKEY_LOCAL_MACHINE > SOFTWARE > Policies > Microsoft > Windows Defender > Exclusions > Paths or did you try to delete them individually.  I ask because if you try to delete all of them, it will fail since it will not allow you to delete the default.  If that's what you did, try doing them individually instead.

    I notice you had lots of programs open when you did those things you tried.  Sometimes this can affect what you can and can't do. I recommend starting with a clean boot (see http://www.tenforums.com/tutorials/41804-clean-boot-perform-windows-10-troubleshoot-software-conflicts.html ) and then see if the exclusions will allow you to remove them directly in settings.  If not, then see if you can do the registry changes you tried (but try using the method described in my prior paragraph if that's not how you did it).  If neither work then this is not going to help and I recommend you return the computer to starting in normal mode (as described in the link in this paragraph).

    At this point, if nothing above has worked, then please post back with the results of what you tried (along with responses to the questions in the first paragraph above), which I'd rather do than proceed now with a shotgun of options, including extreme ones, for a bazillion possibilities, and perhaps your feedback will provide us with a clue as to what's best to do next.  It may come to an extreme solution, but I'm not convinced we're there yet.

    Best wishes,

    Kosh
