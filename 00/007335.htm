In every version of FrontPage the navigation view boxes are in 4 colours
1) Yellow - page is included in navigation bars
2) blue - page is selected in navigation view, and is included in navigation bars
3) grey - page is not included in navigation bars
4) dark grey - page is selected and not included in navigation bars.

The wording on the right click menu (in FP2003) is "Include in navigation bars", with an icon that is dimmed when the page is excluded. (Long time since I used that menu).

The yellow and grey are shown in the image at http://www.rxs-enterprises.org/images/fp/998566610_navigation2.jpg

The message "edit the properties of the navigatiion bar to display hyperlinks here." indicates that the navigation bar used is not suitable for the page.
For example, a nav bar depicting "Child Pages under Home" will be empty on the Home page, unless Home is also ticked in the nav bar properties.

The possible navigation bar choices are listed at
http://www.rxs-enterprises.org/fp/articles/navigation/fp-navbars.aspx

jimmy rishe say:

Yes, this is possible.  In the Submit Options dialog, select the radio button that says "Perform custom action using Rules."

Then in the rules panel that appears, create two Submit data rule actions - one for the SharePoint submit, and one for the E-mail submit.  You'll need to add data connections for both of these before doing this.
