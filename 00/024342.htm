Hi,

 

A formula can't change any cell other than the one it's in so you can't do that with a formula. You can do it with a macro.

 

Select all the cell with micrograms in and run this code and it will convert them to milligrams.

 

Sub Milli_To_Micro()
Dim c As Range
For Each c In Selection
    c.Value = c.Value / 1000
Next
End Sub
