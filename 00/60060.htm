      There are differences in migwiz interface in Windows 7 and we will not be able to complete saving offline files to Savedata.mig file and move them to d:\csc folder.

      You may try these steps and check:

      a. Set this registry key HKLM\System\CurrentControlSet\Services\CSC\Parameters\MigrationParamters, REG_DWORD, 1
      b. Run the Windows 7 migration wizard.
      c. After it has scanned the data you will see a “Customize” link; click that & make sure “Windows Settings” is checked for all the items displayed in the list.
      d. After you have collected the data, set the registry to specify the new cache location.
      HKLM\System\CurrentControlSet\Services\CSC\Parameters\CacheLocatio, String, \??\<Drive>:\Folder; example \??\D:\CSCCache
      e. Reboot the computer.
      f. Run the Windows 7 migration wizard & apply the gathered files.
      g. Reboot the computer.
