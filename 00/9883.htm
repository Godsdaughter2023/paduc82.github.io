Hi Rai,

To understand the issue more clearly, can you please let us know how these 3 accounts (with their mailboxes) were created in the first place?

Usually, the user and mailbox creation should be done in the on-premises in a hybrid deployment. when the mailbox is moved to Office 365, a proxy address which ends up with the domain.mail.onmicrosoft.com will be added to the Office 365 mailbox so that the Autodiscover process can be redirected to Office 365 correctly using the mail.onmicrosoft.com domain.

If the user is created in Office 365, we can create a corresponding on-premises user using the same UPN and email address, then use the SMTP matching to match the two users using DirSync. For more details, please refer to: https://support.microsoft.com/en-us/kb/2641663.

Note if the mailbox is enabled in Office 365 before hybrid is deployed, the default mail.onmicrosoft.com email address is not added in the mailbox automatically. To solve the issue, I suggest you back up the data in Office 365 mailbox, then deactivate DirSync and delete the Office 365 user permanently by referring to: https://msdn.microsoft.com/en-us/library/azure/dn194132.aspx. After that, re-activate DirSync to sync the user to Office 365, and re-enable the mailbox in Office 365 by running the cmdlet Enable-RemoteMailbox.

jay freedman say:

When you have the ribbon collapsed, and you single-click one of the tabs, the ribbon expands and covers the Quick Access Toolbar, the horizontal ruler (if that's displayed), and the top few lines of the document page. If you click the tab again or any of the buttons on the ribbon, then the ribbon collapses and you can see the other items.

To push the other items down so the ribbon doesn't cover them, double-click the tab instead of single-clicking it. The ribbon will no longer be "collapsed but temporarily expanded". To collapse it again, double-click the tab again. The keyboard shortcut Ctrl+F1 also toggles the full display.

There's also a button at the top of the window, to the left of the Minimize button, that opens a menu of ribbon display options. Besides the "fully expanded" and "tabs only" options, there's an AutoHide option that collapses the ribbon and the tabs -- giving maximum space for the document -- and leaves just a thin bar with three buttons.
