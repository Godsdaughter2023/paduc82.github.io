sky pei msft say:

Hi IMK,

Yes, we can connect SharePoint Online Access web app to PowerApps.

As data displaying in Access web app is stored in Azure SQL server, we need to create a connection using SQL server other than the URL. Here are the steps:

1.Open the Access Web App in the Access client

2.Click the File tab > Info > click Manage within the Connections group

3.Select From Any Location or From My Location to allow connections. Note: If you are on an IPv6 network, you will only have the option to connect From Any Location.

4.Select either Enable Read-Only Connection or Enable Read-Write Connection

5.Depending on the previous selection, select either View Read-Only Connection Information or View Read-Write Connection Information

6.Copy and paste the Server, Database, UserName and Password values into Notepad or Word, or leave the Connection Information window open so you can copy and paste the information from it later.

7.Open PowerApps via the browser ->connections ->new connection

8.On the new connection page, under Standard ->select SQL server -> on the pop-up SQL server table -> select connect directly and type Server, Database, UserName and Password got from step 6 -> add a connection.  Then the connection will be created like blow.

However, PowerApps related questions/issues are handled directly by our PowerApps forum. About creating a mobile app using the connection above, we would suggest you post to our PowerApps forum via the link below, which is dedicated in handling this kind query.

https://powerusers.microsoft.com/t5/PowerApps-Community/ct-p/PowerApps1

john korchok say:

To print to the edge of a page, you need a photo printer that can do borderless printing. Do you have such a printer? Most printers have an unprintable area around the edge.

If you have a printer that can do borderless printing, you can resize your presentation to fill an 8.5x11" sheet. Choose DESIGN>Slide Size>Custom slide size and set the Slides sized for: dropdown to Letter Paper (8.5x11 in). There will be some distortion that results from this resizing. Then print.
