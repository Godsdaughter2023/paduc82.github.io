arodi chellaya say:

SetFocus to a field

I have a statement form.

The user has to select the client's clientID or client's full name.

If the user gotfocus on any other fields than clientID or FullName, the user is setfocus to FullName and a msg text(FullNameTest) is visible below.

For eg, The form is displayed as new(blank) that is statementID=(new) ,if i gotfocus to statement date:

If Me.Client_ID.Value = Null Or Me.FullName.Value = "" Then
    Me.FullNameTest.Visible = True
    Me.FullNameTest.Value = "Please choose a ClientID or a Client FullName"
    Me.FullName.SetFocus

End If

I received the error msg: You entered an expression that has no value.

I think this happen because the form has not been loaded, that is pass to a new record value(statementID).

Then how would to figure this out ???

scottgem say:

First, you don't have fields on a form, you have controls that may or may not be bound to a field in a table. This is a subtle but important distinctions. 

Second, It appears to me that you are making this more complex then it needs to be. You don't indicate where you are running this code which is a key issue and should have been mentioned. 

What I would do is use the On Load and/or On Current event to set focus to the ClientID control. 

Me.ClientID.SetFocus

then use the Lost Focus event to test it:

If IsNull(Me.ClientID) then

     MsgBox "ClientID must be entered!"

     Me.ChildID.SetFocus

End If

This way you can't leave the control empty. If you do, focus will be placed right back in the control. Set the Tab Order, so the name control gets focus next and do the same thing in the Lost Focus event.
