graham mayor say:
Before suggesting a method to do this, it would help if you explained exactly what it is that you are trying to achieve.

hansv mvp say:

You already asked this in How do I create a group of sequentially numberd files in bulk ?

Here is a step-by-step guide:

Start Word or Excel.

Press Alt+F8 to activate the Visual Basic Editor.

Select insert | Module to create an empty code module.

Copy/paste the following code into the module:

 

Sub CreateFolders()
    Dim i As Long
    For i = 1000 To 1750
        MkDir "C:\Documents\E " & i
    Next i
End Sub

 

Modify the path where the folders will be created to suit your purpose. In the above example, the folders will be created in C:\Documents. If you want to create folders in C:\Users\Jeff\Documents\Test instead, change

 

        MkDir "C:\Documents\E " & i

 

to

 

        MkDir "C:\Users\Jeff\Documents\Test\E " & i

 

Position the insertion point anywhere in the code.

Press F5 to run it.

Now open Windows Explorer to view the folders.

If you need to run the code (with modifications) again later on, save the document or workbook that you created. Choose a macro-enabled Word document (.docm) or macro-enabled Excel workbook (.xlsm) as file type.

On the other hand, if this was a one-off operation, you can discard the document or workbook.

karl timmermans [outlook mvp 2012-2018] say:

Going to assume that you are referring to exporting "all contacts" assigned to a single category.

By itself - there is no selection option available in the contact folder to CSV file export process.

There are some less direct alternatives available

Option #1 - PST 2 PST export

#1 - Export the contacts you want to a new PST file and use a filter for the category you want (filter button is at the bottom of the window where you select the folder to be exported)

#2 - Open the newly created PST file

#3 - Export the contacts contained in the newly created PST file to a CSV file

Something that may be of interest

Understanding How Outlook PST2PST Import / Export Differs
http://www.contactgenie.info/understanding-pst2pst-import-export

Option #2 - Cut 'n paste

*** am not a fan of this approach since it's wieldly for large contact folders and/or when a lot of fields are going to be included - adding it here just for information

#1 - Create a custom view selecting contacts for the desired category

#2 - Highlight all contact items and the "copy"

#3 - Open Excel, paste the copied contact items to a new worksheet and save the worksheet as a CSV file

Option #3 - Filter output file

#1 - Export all the contacts including the category field to Excel

#2 - Filter the worksheet using a <Contains> option on the category field and copy the desired rows to a clean worksheet and then save that worksheet to a CSV file

Option #4 - 3rd party solution

If you are going to be doing this on a regular on-going basis, then ContactGenie Exporter may be of interest which has an export filter option

http://www.contactgenie.com/export_4_outlook.htm

Export Overview

http://www.contactgenie.com/export_overview.htm
