This issue may occur due to browser add-ons or due to any recent changes on the computer that might have changed the Internet Explorer settings.

Follow the below methods to fix the issue.

Method 1: Run the Internet Explorer performance troubleshooter.

Performance troubleshooter helps to automatically identify and fix IE performance.

  1. Press the ‘Windows’ key on the keyboard.
  2. Type Troubleshooter in the Search box and then press Enter.
  3. Select Troubleshooting. Click on View All and select Internet Explorer Performance.
  4. Click Next and follow the on-screen instructions.

Method 2: No Add-ons Mode

This method helps us to identify if the issue occurred because of corrupt add-ons.

Click Start. Type in "iexplorer.exe -extoff" and press enter.

This will open IE in No Add-ons mode. Check if the issue persists.

If there are no issues when using this mode, you can use the add-on manager to disable all unwanted add-ons.

To do this, follow these steps:

  1. Open Internet Explorer.
  2. Click the tools button (wheel button at the right hand side corner)
  3. Click Manage Add-ons.
  4. On the Show drop-down menu, select all add-ons to display all add-ons that are installed on the computer.
  5. Look for each item in the list, select the add-on, and then click Disable in the Information window.
  6. When you have disabled all the items in this list, click OK.
  7. Exit and then restart Internet Explorer.

Method 3: Reset Internet Explorer Settings.

Resetting Internet Explorer settings, helps to return Internet Explorer to the state it was in when it was first installed on your computer. This is useful for troubleshooting problems that might be caused by settings that were changed after installation.

Follow the steps in the below Microsoft article to Reset Internet Explorer Settings

http://windows.microsoft.com/en-us/internet-explorer/reset-ie-settings#ie=ie-11

doug robbins - mvp office apps & services (word) say:

It's purpose is to allow you to set tab stops, select the way in which the text is aligned at those tab stops, and also to select a leader for the tab space that is created by moving the selection to a particular tab stop.  All but the last of these actions can also be performed by clicking on the tab stop button at the top left of the document window to change the type and then clicking on the ruler at the position where you need the tab stop.

The dialog can also be used to clear individual tab stops by selecting them from the list and the clicking on the Clear button, or to clear all of them by the use of the Clear All button.  You can also remove an individual tab stop by clicking on it in the ribbon and dragging it off the ribbon.

When you set a tab stop, the tab stops at the default spacing that would have been to the left of the tab stop that you set are cleared.  If you clear all of the tab stops, the default tab stops will be reinstated.  The default tab stops are all left aligned and their positions can be seen by the little tick marks below the ruler.
