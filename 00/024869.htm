ken sheridan say: Also, this code is redundant:

        ' ensure New Document Type has been added
        If Not IsNull(DLookup("typeID", "tblType", "typeDocType = """ & _
            NewData & """")) Then
        Response = acDataErrAdded
        Else
            strMessage = NewData & " was not added as a new Document Type."
            MsgBox strMessage, vbInformation, "Warning"
            Response = acDataErrContinue
            ctrl.Undo
        End If

You merely need:

    Response = acDataErrAdded

If for any reason the row was not inserted into the table an error will be raised on the execution of the line:

    DBEngine(0)(0).Execute strTmp, dbFailOnError

so the above current code will not be reached.  You only need to check in this way for the row having been inserted if the code is opening a form in dialogue mode to insert the new row, which will include other data inserted by the user into the dialogue form as well as the NewData value, which would be passed to the dialogue form via the OpenArgs mechanism.  You'll find examples of the use of the NotInList event procedure in a variety of contexts as NotInList.zip in my public databases folder at:

Repeat Table CAPTION (not headers) on next page(s) - Microsoft Community

kerry paul say:

Repeat Table CAPTION (not headers) on next page(s)

How can I repeat the table caption (the table number and title, e.g., Table 1: Earthquake Monitoring Network, Oklahoma) for a table that is+1 page in length.  I know how to repeat the header rows which is great but I must have the table number on top too.  Please help. 

KP

jay freedman say:

First you'll have to split the table so that each page contains a separate table; add the caption only to the first one. (There is no way to insert captions into the middle of a table that breaks across pages.) Place the splits so that there's enough room to add the caption to each page without making the table spill over to the next page.

Put a bookmark around the caption at the beginning of the table. Then put a cross-reference to the bookmark at the top of each additional page.

doug robbins - mvp office apps & services (word) say:

Insert the Table Caption above the table and then select it and use Cut and Paste to paste it into a row, with the cells merged into one, at the top of your table (before the header row(s) if you have them).  If desired, remove the top and side borders of that row.
