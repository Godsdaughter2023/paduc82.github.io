amit tandon say:

Try below formula:

=SUBSTITUTE(ADDRESS(1,A1,4),"1","")

Where column number is in cell A1.

This will return the column letter for the column number.

Or if you just enter below formula in any cell, it will return the column letter of that cell:

=SUBSTITUTE(ADDRESS(1,COLUMN(),4),"1","")
