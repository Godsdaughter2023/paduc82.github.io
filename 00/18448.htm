Using VBA to copy text of custom document properties from one document - Microsoft Community

jac.chr say:

Using VBA to copy text of custom document properties from one document to another

Hi Experts

I am trying to create a macro where I can copy the text of custom document properties from one document to another. However I am a bit unsure how the basic VBA syntax is for referencing a procedure on a specific document. The following is just a dummy example:

Public Sub CopyDocProps()

   Dim oOldDoc As Document

   Dim oNewDoc As Document

   Set oOldDoc = ActiveDocument

   Set oNewDoc = "C:\MyDocument.docx"

   oNewDoc.SetDocProp("CustProp1", oOldDoc.GetDocProp("OldCustProp1"))

End Sub

I already have already created the SetDocProp to set a custom document property with a name and text. Additionally the GetDocProp will return the text of a custom document property with the name specfied.

But the question is if it is even possible to mix the procedures between two documents?

jay freedman say:

Your general idea is ok, but your proposed syntax won't work. The dot operator in the expression oNewDoc.SetDocProp implies that SetDocProp is a method of the Document object, but there is no such method defined for any Document object (and the same is true for GetDocProp). Therefore, if you try to use this syntax, you'll always get a compile error "No such property or method".

Instead, you can pass a Document object as an argument of each of the two procedures. The first lines of the procedures will be something like these:

Function GetDocProp(oSourceDoc As Document, strSourcePropName As String) As String

Sub SetDocProp(oDestDoc As Document, strDestPropName, strPropValue As String)

Then you can call the procedures in the statement

SetPropDoc(oNewDoc, "CustProp1", GetDocProp(oOldDoc, "OldCustProp1"))
