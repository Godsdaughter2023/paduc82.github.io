thahaseena m say:

Hi,

 

Most advrcntr2.dll "not found" errors are due to issues with the Nero CD and DVD burning software program. The advrcntr2.dll DLL file is a file that must exist in the correct folder for Nero to burn CDs or DVDs.

 

To fix this problem, you will have to rename a folder and reinstall Nero. Follow the steps mentioned below.

a.       Delete the following folder: C:\Documents and Settings \<Username>\Local Settings\Temp

b.      Rename the folder "AdvrCntr4" in folder: C:\Program Files\Common Files\Nero into "AdvrCntr4_2".

c.       When reinstalling Nero, the folder "AdvrCntr4" will be created again. As soon as the folder "AdvrCntr4" is once again created in the directory

C:\ Program Files\Common files\Nero the directory "AdvrCntr4_2" can be erased.

d.      Please make sure to deactivate your antivirus software before following the steps below, otherwise you may encounter problems and Nero cannot be removed/reinstalled properly.

e.      Use the Nero Cleantoolhttp://www.nero.com/redir.php?id=4558 to delete Nero.

f.        And install the latest Nero 9 update:http://www.nero.com/eng/downloads-nero9-update.php

g.       The folder "AdvrCntr4" was created again in the directory C:\ \Program Files\Common Files\Nero

h.      Folder "AdvrCntr4_2" can now be erased if you haven’t already done so.

 

If the issue still persists then, I would suggest you to contact the Nero support. Check the link mentioned below.

http://forum.my.nero.com/index.php?setlanguage=1&langid=en&showforum=2
