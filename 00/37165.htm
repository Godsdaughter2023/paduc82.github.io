techfreak_ say:

Traditional use of the term mirrored (horizontal flip) is different than upside-down (vertical flip). In the newest versions of Skype the local feeds are mirrored in the video settings preview window and the local live video feed during a call. The classic versions (7.x) were not mirrored in the video settings preview window but were still mirrored in the local live video feed during a call. Except for that change, both versions still display the feed non-mirrored on the remote end.  If your feed is actually upside down, I suggest you update your camera drivers or use the generic USB Video Device drivers that come with Windows 10 to see if that resolves that condition. 

If Skype is not behaving as I described, the issue may be related to specific versions of Skype, hardware on your end or either issue on the remote end. Try Skype for web on one end or both ends to see if anything changes.

Hey SavannahMick,  

This behavior occurs because Windows automatically closes any registry handle to a user profile that is left open by an application. Windows Vista does this when Windows Vista tries to close a user profile.

In versions of the Windows operating system that are earlier than Windows Vista or Windows 7, you must install the User Profile Hive Cleanup Service (UPHClean) utility to have the same functionality. However, the UPHClean utility is incompatible with Windows Vista and Windows 7. Additionally, the UPHClean utility is not needed because this functionality is built into Windows Vista.

http://support.microsoft.com/kb/947238

You may try performing aClean Boot to check if any security software is not letting windows to close the registry key.

To help troubleshoot error messages and other issues, you can start Windows 7 by using a minimal set of drivers and startup programs. This kind of startup is known as a "clean boot." A clean boot helps eliminate software conflicts.

How to troubleshoot a problem by performing a clean boot in Windows 7:

k_cat say:

Changing desktop background in Windows 8

I want to use a photo of my own as the background picture on my desktop in Windows 8, but that option doesn't seem to be available...? The "old" way of opening a picture, right klick and choose "set as computer background" doesn't work: the only options I am presented with are lock screen, app tile, app background. Advice, anybody?

hosch250 say:

Go to the Control Panel/Personalize/Change Theme.  Click the Change Desktop Background link that is at the bottom of the page.  Select Pictures Library (if that is where the picture is located) for the Picture Location.  Click the picture and click Save Changes.  When I right-click my pictures, I get the option to Set as Desktop Background near the top of the right-click pop-up window.
