sijoyc j say:

how create an emi calculator in excel

pls help me to create an excel loan emi calculator
joeu2004 say: sijoyc wrote:
pls help me to create an excel loan emi calculator

See the PMT function, assuming EMI stands for "equal monthly installments".

The keys to using PMT correctly are:  (a) using opposite signs for the loan amount and the final balance ("balloon payment"), if not zero; (b) use a monthly interest rate, not the annual rate; (c) likewise, use the number of payments (months), even if the loan term is expressed in year; and (c) explicitly round the PMT result if you want real-world results.

The problem with #b is:  there are several different ways to convert an annual interest rate to a monthly rate, based on how the annual rate is reported in the country.  The following example uses US rules.

Consider a 15-year loan of $100,000 at 4% interest with a final balance of $20,000  Then the EMI is:

=ROUND(PMT(4%/12, 15*12, -100000, 20000), 2)

Note that I use -100,000 for the loan amount.  Using a negative number for the loan amount will yield a positive result for PMT.
