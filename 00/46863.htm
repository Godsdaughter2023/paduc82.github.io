mel77084 say:

When win 7 came out, it was the first OS so large that it had to be distributed on a DVD rather than a CD. The CD/DVD players that were manufactured prior to that time came with a CD driver (not a DVD driver) included in their firmware, so one could easily run the XP installation CD for a clean install; however, to run the win 7 install DVD, one would have to hit a key to pause the installation, remove the win 7 DVD, insert a CD containing a DVD driver to install it, then eject the CD, re-insert the win 7 install DVD, and the newly installed DVD driver would allow the installation of win 7 to proceed.

The same procedure was required if win 7 was to be installed on a scuzzy drive.

DVD players manufactured after win 7 came out included the DVD driver in their firmware, so the extra step to install a DVD driver was no longer necessary in order to do a clean install of win 7 and later OSes.

I wouldn't imagine that you have a CD/DVD player that's old enough to require installation of a DVD driver, though.

Perhaps you have a GPT drive that you will need to convert to MBR in order to install win 7 on it, but that wouldn't cause the error code that you mentioned.

I fully agree with the other comments about installing win 7, unless it's for a computer museum project of some sort.

Hello,

Thank you for posting your concern on our Community forum page. With regard to your concern, the predictive text is an option that is only available for a mobile device. However, the new Surface Pro and other Surface Pro devices functions both  as a tablet and a laptop. The only defining factor is the keyboard or as we all know it, Type Cover.
