julie sheets say:
Hi Jeanne,

You may create resources with maximum units above 100% indicating that you have multiple of those resources.  For example: if you had 3 developers you can create a resource with 300% max units.

The downside:
  • All the resources represented by the greater max units share the same calendar.  So, if one resource is gone for 2 weeks, you cannot put the 2 week time in the resource calendar as non-working without placing all 3 resources as non working.
  • All resources need to have the same standard rate. For most organizations, this isn't much of an issue as they use blended rates for resource costing.
  • It may be more of a challenge from a responsibility  and assignment tracking.  If I have 200% of the 300% resource assigned to a task - you have to be very clear about which 2 of the 3 developers are responsible for the task.

I hope this helps.


Julie
