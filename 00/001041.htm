Mailing List Manager (MLM)
Junk Mail Filter
Rules (custom rules you create)

richard xu msft say:
Hi RH,

Welcome to community, and I understand that you want to enable message encryption in office 365.

Office 365 Message Encryption requires the purchase of Microsoft Azure Rights Management and Office 365 Enterprise E3 and E4 users will get Office 365 Message Encryption at no extra cost. If you are not, maybe you need to purchase of Microsoft Azure Rights Management. You can view more details: https://products.office.com/en-us/business/microsoft-azure-rights-management?legRedir=true&CorrelationId=04a0a13f-c00e-4194-a3ab-da0ef59f87b5

If it is available for you to setup and enable office 365 message encryption, there are three main steps that you need to followed:

1. Activate Azure Rights Management

2. Setup Azure Rights Management for Exchange Online

3. Setup transport rules to enforce message encryption in Exchange Online

For Activate Azure Rights Management, you need go to Right Management and active it. 
1. Login to Microsoft online Portal (https://portal.microsoftonline.com/ )
2. Go to Rights Management in admin
3. Active it

For setup Azure Rights Management for Exchange Online, you need to connect to exchange online with PowerShell.
1. Connect to exchange online
2. Verify IRM configuration
3. Configure RMS with the online key-sharing location
4. Import the Trusted Publishing Domain
5. Verify successful setup of IRM in Exchange Online
6. Disable IRM templates in OWA and Outlook
7. Enable IRM for Office 365 Message Encryption
8. View the IRM Configuration

For Setup transport rules to enforce message encryption in Exchange Online, you need go to office 365 admin portal.
1. Open the Office 365 Admin Portal (https://portal.microsoftonline.com)
2. Open Exchange Admin Center and go to mail flow
3. Create your transport rule
4. Test your transport rule

There is a reference link for you. And you can check detailed information form it.

Thanks,
Richard
