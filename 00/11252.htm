Create a Default Theme in PowerPoint 2013 for Windows - Microsoft Community

geetesh bajaj say:

Create a Default Theme in PowerPoint 2013 for Windows

Technical Level : Basic

Summary

Learn how you can change the default Theme in PowerPoint 2013 for the new presentations you create.  This can be very helpful indeed when you want to use your corporate, organization or company Theme as the default.


Details

You can use a Theme other than the blank white design that PowerPoint 2013 provides by default. Even though this default design is a great starting point, there are situations when you need to use particular Theme for all your presentations. If that's the situation, follow these steps to create a new default Theme:

  1. Open a new presentation. Access the Design tab of the Ribbon.

  2. Within the Themes group, click the More button to bring up the Themes drop-down gallery.
  3. Bring up a context menu by right-clicking the Theme to be saved as default. Within this menu select the Set as Default Theme option.

Hereafter, whenever you choose the File | New | Blank option, or even just launch PowerPoint, a presentation will be created based on the Theme you chose to be the default.

Note: What happens if you also have a default template in addition to a default Theme? To learn more about what's different between a default template and a default Theme -- and to learn how you can create a default template, look at the Change the Default Template or Theme in PowerPoint 2013 page (external link).

Hello Stephen,

 

MAPI account in Outlook is associated with the .OST file and not with the PST file.

 

PST file is associated when you setup POP3 email account type in Outlook.

 

Once you delete the MAPI account from Outlook and setup the same as POP3 type, all those emails present on the server would be received to the PST file associated.

 

However, in POP3 email account type only the emails which are on INBOX will be downloaded into Outlook and the  folders created on the server are not synced and it’s also the case with Contacts and Calendar items.

 

In order to remove an email account: Open Outlook > click on ‘File’ > Account Settings > Under ‘Email’ tab > highlight the account to be removed > click on ‘Remove’ button on top.

 

Check if you can configure the Hotmail account as POP3 email account type as suggested in the article below:

move data from one column to adjacent blank cells in adjacent column - Microsoft Community

beth m. johns say:

move data from one column to adjacent blank cells in adjacent column

I'm a fairly inexperienced Excel user, but have a situation where I need some help. This is a worksheet that has about 100,000 rows of data and if I or my assistant needs to copy and paste (which is easy and can be done), we are going to lose our minds! 

I have two columns of data. Both columns contain blank cells (this is expected). 

I want to move data from the first column into the adjacent cell in the next column. 

Example:

Column D, Row 1 has the number 1234-5678

Column E, Row 1 is blank

I need to move the number from Column D to Column E. 

This happens in "chunks" -- there are many, many instances of this. We'd like a method where Excel will handle this with a formula or macro (I know nothing about macros! -- But will learn.). 

As an aside, once the data is moved into the appropriate blank cells in Column E, Column D will be deleted. Basically, we end up with one column of data. 

I've tried various ways to make this simpler with sorting and auto filters but I seem to run into problems when I do this. 

Here is a screen shot to further explain: 

Those "chunks" go on and on for 100,000 rows. 

If I need to clarify this further, please ask. Thank you. 


ossiemac say:

Following procedure should help.

  1. In Column E set AutoFilter to blanks.
  2. In first visible cell of column E insert = and then click the adjacent cell in column D and press Enter. (You should have a formula like the following but D5 will be whatever was adjacent cell:    =D5)
  3. Copy the cell with the formula.
  4. Select all of the blank cells below the formula in column E to bottom of your data.
  5. On Home ribbon select "Find & Select" icon (far right).
  6. Select "Go To Special" from the DropDown.
  7. Select "Visible cells only" from the dialog and then click OK.
  8. Right Click over one of the selected cells and Paste.
  9. Clear the Filter in column E (Select all).
  10. Select entire column E and Right click and then Copy and then Paste Special Values.
