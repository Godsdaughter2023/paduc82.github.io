tom ogilvy say:

Under the Options for Ctrl+F are  Look in Formula and Look in Values.  You would look in Values if you want the results of a formula.

A pivot table produces constant values - no formulas (even for totals and subtotals) so Ctrl+F should have no problem finding information in Pivot Tables regardless of the select of where to look.

for example, I put your table above on sheet 3 and then on Sheet2 I copied the Emp_ID and next it I used vlookup to pull back the names.  I selected the entire sheet and did Ctrl+F and typed in Hanley and it failed.  I looked at options and it was set to look in formulas.  I changed that to values and was successful in finding Hanley.    Later I did Ctrl+F and looked for Romano and was immediately successful because the change to Values was retained (it is a persistent setting).

Public Function getHTML(strURL As String) As String
 'Returns the HTML code underlying a given URL
Dim oXH As Object
Set oXH = CreateObject("msxml2.xmlhttp")
With oXH
    .Open "get", strURL, False
    .send
    getHTML = .responseText
End With
Set oXH = Nothing
End Function

Public Function parseGoog(strSearch As String, strHTML As String) As String
    strSearch = "," & strSearch & ":'"
    If InStr(1, strHTML, strSearch) = 0 Then

        parseGoog = Mid(strHTML, InStr(1, strHTML, strSearch) + Len(strSearch))
        parseGoog = Mid(parseGoog, 1, InStr(1, parseGoog, "'") - 1)
