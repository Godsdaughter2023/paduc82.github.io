Hi,

 

1.      “.NET” is an integral part of many applications running on Windows and provides common functionality for those applications to run.

 

2.      “.NET Framework” is needed for the games and applications, which are created for the Windows platform to function properly. Without .NET Framework even you cannot install Windows updates. Even for developers .NET Framework is needed to install the applications. Updates for .NET Framework is required to fix some stability, reliability, and performance issues in the Microsoft .NET Framework.

.NET Framework version numbers 2.0, 3.0, 3.5 and any updates are all part of ONE full framework (2.0 is the base, and 3.0 and 3.5 are extensions of 2.0). Therefore you cannot, for example, remove .NET Framework 2.0 and keep .NET Framework 3.5.

dominicp say:

Libraries are not located anywhere. Libraries are a collection of references to folders. The folders referenced in the libraries have locations.

 

You can delete any of the entries in the Library itself: this only deletes the entry in the library, not the folder referred to. But be careful: once you are working with an item referenced in the library your actions are on that item, not on a library reference.

 

Go to C:\Users\<your username> and right click on any of the folders below that; select Properties then the Location tab and change the location of the folders you want to D. Windows will offer to move all the files in the folders to the new location.

 

You can then include that location in the relevant (or a new, named as you choose) Library and delete the reference to the old location.

 

Saving and opening items will start with the default location specified for the application you are using. This must be set for each application used.
