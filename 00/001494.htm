mackles24 say:

The reason the references remain correct when the property "Don't Move and Size" is selected is because whether the checkbox moves with the cells, or stays in the same spot, the cell reference remains the same.

So, for example, I have a checkbox in cell B3, linked to cell A3.  When I sort the list and the data that was in cell A3 is now in A12, if the property is set to "Move and Don't size", the checkbox will move to cell B12, however the cell reference remains as A3.  If the same thing happens with the "Don't move and size" property selected, the checkbox will stay in cell B3 with cell reference A3.
