1)  Open the registry editor by typing regedit on the start menu search field and then clicking on regedit.

4)  Click the "Advanced" button to add a new user.
5)  Type in "Everyone" as the new user and click OK.
6)  Configure the permissions for user "Everyone" to "Full Control".
