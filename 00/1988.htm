morg. say: The problem is caused by the office2003 compatibility layer of office 2007, which behaves very badly with native office2k3 files. 

There is no fix afaik, except using the xml (docx) format. 

The broken compat api will create a new 2k3 native document on save and rename the old 2k3 document to *.tmp::hidden, only to delete it when you close word or excel. 

Because it creates yet another 2k3 native document, the next time you save, the exact same thing happens. 

And thus of course, as it's a newly created file on your desktop, it ends up in the first available slot (or the second one if you have set "show hidden files and folders"). 

It would be really easy to fix if we had the office source code, but it's in microsoft hands and they're probably happy with people moving to xml faster anyway (and you should, the old format is **** in comparison).

Desktop Sharing

Steps needed to get this set up
susan_s. say:

Hello

Steps on how to share your screen in Skype are as follows:

  1. Sign in to the Skype app.
  2. Start a voice or video call with a single person or a group. 
  3. During the call, tap or click the plus icon* in the call bar, then choose Share screen. 
  4. If you're using more than one screen, select the screen you want to share. 
  5. Use the Share your screen drop-down to choose what to share - either your full screen or a specific window, then choose Start. 
  6. Select Stop sharing when you're finished. 

*plus icon will only be visible if the person or group you're calling already answered your voice or video call.
