I get that Powerpoint wasn't designed for this, but it could easily have been made to do this by simply not scaling the default spacings/sizes according to the page size.  I see many postings asking about how to retain "normal" spacings/sizings in the context of pasting into a different  sized destination page, which isn't quite the same problem, but it shows that people don't want content to automatically scale according to the page size.  In fact, if I ever have to increase my canvas beyond Tabloid/Ledger for a schematic that is already in progress, I'll be up against a problem for that very reason.

So understand the proposed explanation that the current behaviour of scaling spacings/sizes to match the canvass is to emulate wholesale rescaling in order to take existing presentation content and display it on different format media.  Or for presentations created from scratch, to emulate the drawing experience of a un-supersized canvass.  But it doesn't make sense to me.  It's simple enough to fit content to different destination pages in the printing phase.

I agree that the solution is to go to Visio.  But this was suppose to be a quick-and-dirty schematic, and in principle, the barriers to accomplishing it do not have to be.  And, I argue, is not the most sensible design behaviour.  It's only my opinion, but it's substantiated by the many inquiries online about how to avoid this unwanted dynamic rescaling of spacing/sizes, and the ready solution to actual wholesale rescaling that is more sensible.  As well, in a locked-down corporate environment, moving to another app is anything but quick and trivial.  We're very much at the mercy of the microsoft's design decisions.

kristine arp say:

Hi Bernard,

 

 

Activation error is one of the possible reasons why you are getting a message that your subscription expiration date is incorrect. Start with checking if your subscription expiration date is correct in your account. Log in to Office.com/myaccount using the Microsoft account associate with your Office 365 subscription and go to the Services & Subscriptions tab.

 

If the expiration date of your subscription is correct in the Services & subscriptions page, download and run the Activation Troubleshooter. Follow the steps below.

 

If you’re using Edge:

 

  1. Select Save.
  2. Select Open.
  3. Select Next and follow the prompts.

 

If you’re using Internet Explorer:

 

  1. Select Open. If the file doesn’t open right away, select Save and Open Folder.
  2. Double-click the file to run the tool.
  3. Select Next and follow the prompts.

 

If you’re using Chrome:

 

  1. Select Show in folder.
  2. Double-click the file to run the tool.
  3. Select Next and follow the prompts.

 

If you’re using Firefox:

 

  1. Select Save File and hit OK.
  2. Select Show all downloads from the upper-right corner of the browser window.
  3. Select Downloads > officeact.diagcab.
  4. Select the folder icon.
  5. Double-click the officeact.diagcab.
  6. Click Next and follow the prompts.

 

Keep in mind that you don’t need to be online all the time to use Office. However, it is important to be connected to the Internet from time to time. This way, the system can verify that your Office 365 subscription is still active. If the system wouldn’t be able to check the status of your subscription for at least 30 days, you’ll see a Couldn’t Verify Subscription message and Office will go into reduced functionality.

 

Let us know if you need additional assistance.
