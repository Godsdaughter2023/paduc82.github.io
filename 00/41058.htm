a. user say:

Hi,

 

This issue occurs when the drivers for the devices are not getting installed properly. Sometimes, you should check to see if Windows can now find a driver for your device when the hardware device does not function properly.

 

Have you tried reinstalling all the device driver for the USB ports?

 

If not, I would suggest you to uninstall and reinstall all the USB port driver and check if it resolves the issue. Follow the methods:

Method 1:

Refer to the link and check if the steps helps you to resolve the issue:

http://windows.microsoft.com/en-in/windows7/what-to-do-when-a-device-isnt-installed-properly

 

If this does not work, check with the next method.

 

Method 2:

Let’s check if USB devices are powering down. Follow the steps:

a)            Type power plan in the search box on the Start and click Choose a power plan.

b)            Next to your currently selected plan, click Change Plan Settings.

c)            Click Change advanced power settings.

d)            Click the box to expand USB Settings, then expand USB selective suspend settings.

e)            Click Plugged in, then click the drop down menu and select disabled.

f)             If you're using a laptop, click Battery, then click the drop down menu and select disabled.

g)            Click Apply, then click OK.

 

Check if you still find the same issue when you connect the camera. If yes, move to method 3.

 

Method 3:

Uninstall and reinstall the USB controllers drivers and check if it fixes the issue.

Follow the steps:

a)            Press Windows and R key together. Type “devmgmt.msc” and hit Enter.

b)            Locate the Universal Serial Bus controller and expand the driver icon.

c)            Right Click on all Universal Serial Bus controller driver icon and select Uninstall.

d)            Restart the computer. After restarting the computer, USB driver will automatically be installed.

 

Check if the same issue occurs again. If yes, you may refer to the manufacturer website to check if there are latest chipset driver available. If they are, download and install them to try and fix the issue further.

 

Please let us know about the results.

Hi,

 

Method 1:- Please follow the below steps to reset Proxy settings

 

a.     Please click on "Start" Button, click "All programs", and click "Accessories".

b.    Right click on Command Prompt and select Run as Administrator.

c.     Type "netsh winsock reset" (without quotes) at hit enter.

d.    Then type “netsh winhttp reset proxy”at the command prompt, and press Enter.

Type the below command one after the other and hit enter.

Ipconfig /flushdns

Ipconfig /release

Ipconfig /renew 

e.    Once the above command is completed please restart the computer and check if the issue is resolved

 

Method 2: Disable proxy server

 

  1. Launch Internet Explorer, access the "Tools" menu and click on "Internet Options" from the dropdown menu.
  2. Select the "Connections" tab.
  3. Click on "LAN Settings" (if you use broadband connection) or click on the "Settings" button within the "Dial-Up and Virtual Private Network settings" (if you use dial-up connection).
  4. Disable (uncheck) the check box that corresponds to "Use a proxy server" option, then click "OK" to confirm.

 

Method 3:  Release and Renew IP address

 

  1. Press the "Windows" key and the "R" key on your keyboard to open the Run dialog box.
  2. Type "cmd" in the text box and press "Enter" to open the command prompt.
  3. Type "ipconfig /release" (without quotation marks) and hit "Enter" to release the IP address.
  4. Type "ipconfig /renew" (without quotation marks) and press "Enter" to renew the IP address.

 

 

Method 4: Obtain IP Address automatically:

 

  1. From the Start menu, select Control Panel, or Settings and then Control Panel.
  2. Double-click Network and Sharing Center.
  3. From the list of tasks on the left, click Change adapter settings (in Windows 7), or Manage network connections(in Vista).
  4. For a wired network connection, right-click Local Area Connection, and select Properties.

    For a wireless network connection, right-click     Wireless Network Connection, and select Properties.
  5. From the Networking tab, click Internet Protocol Version 4 (TCP/IPv4), make sure it is checked, and then clickProperties.
  6. Click Obtain an IP Address Automatically, and then click OK.

 

Hope it helps.

randyau93 say:
When you buy a game, you're buying a license to use that software. In the case of disc-based Xbox One games, the disc is the license - the "key" - that allows the software to run. So even though it can be installed on multiple consoles, it will only run in the single console that has the disc in its drive. In the case of digital Xbox One games, you get two licenses - one for whatever console that you set as your account's Home Xbox, and one for your use when you sign into another console.

So, if you would like for someone to use a game at the same time as you, you would need to (1) buy the game digitally then (2) go to your cousin's house and set that console as your account's Home Xbox. This can cause problems (for example, if the Internet goes out at your house, you can't play your own games), but it's still your choice as to what console you set as your Home Xbox.

As was stated earlier, be sure to keep in mind that any transfer of account details (that is, letting someone else know your account name and password) is strictly prohibited by the Xbox Live Code of Conduct and Microsoft Services Agreement.
