imran chand say:

Hi tfarrebs,

Try the below troubleshooting steps and get back to us with results:

Step 1:
Start the Windows Installer service:

1. Click Start, type Services in the search box.

2. On the Services window, look for the Windows Installer service.

3. Double click on the Windows Installer service and set the Startup type of Windows Installer to Manual.

4. Click Start to start the service. Note down the error message if any.

5. Click OK.

Step2:
Reregister the Windows Installer

1. Click Start, and then type cmd in the search box,

2. At the command prompt, type msiexec.exe /unregister and then press ENTER.

3. Type msiexec /regserver, and then press ENTER

Step 3:
1. You may try downloading the Windows Installer Cleanup utility and run that to remove any leftover ITunes installer files that may be causing issues with your current installation.

2. You may refer the following article to download the Windows Installer Cleanup Utility:

http://support.microsoft.com/kb/290301  

3. Click on Start > All Programs and then click Windows Install Cleanup; the Windows Installer Clean Up Utility window appears, listing software that is currently installed on your computer.

4. Select iTunes from the list and click Remove. If you have multiple iTunes files, remove all of them.

5. After running the Windows Installer Cleanup utility check whether you can download the ITunes and install on your computer.

Step 4:

You may also refer the Apple Discussion support link for more support.

http://discussions.apple.com/index.jspa

Hope this helps you.
