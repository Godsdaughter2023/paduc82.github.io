Applying a Text Wrapping Default to All Pictures in a Document - Microsoft Community

jamesriley1776 say:

Applying a Text Wrapping Default to All Pictures in a Document

I have some long, multi-page documents with hundreds of images and I would like to apply the same text wrapping option to all of them.  I've used File-Options-Advanced to set the paste default option to "square" and then I've pasted the document in to a new word document to no avail.  I've examined the different paste options and none seem to deliver.

Any suggestions?

jay freedman say:

The paste options apply only when you're pasting an individual image into a document, not when you're pasting an entire document. Instead, you'll need a macro that sets the wrapping for each image in the open document.

There are two complications that must be considered while creating the macro:

  1. Do all of the existing images already have some wrapping that is not "In line with text"? The reason is that Word considers inline objects to belong to one group, and all other objects (with "floating" wrapping) to belong to a different group. If some of the documents may contain members of both groups, the macro has to account for that.
  2. When an object has a non-inline wrapping other than square, and you (or the macro) change the wrapping to square, the layout of the surrounding text may change, and the position of the image itself may change. What, if anything, do you want to do about that? Is there a particular location (left or right margin, top or bottom of the associated paragraph, etc.) that you want to enforce?
