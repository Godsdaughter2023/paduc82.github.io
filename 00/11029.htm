Word2013 Watermark problem

I'm running Office Professional Plus 2013 on a Windows 8.1 (64-bit) desktop computer.


I have a document consisting of the following 3 sections:

Section 2 = Table of Contents page.

Section 3 = rest of the document (4 pages).


I want a "Draft" water mark for the entire document. I can place a watermark in one section. However, when I place it in a second section, the watermark in the first section is removed.


How do I get a watermark for the entire document?

jay freedman say:
It's an unfortunate (a.k.a. "stupid") design in the Word code that inserting a watermark in any section first removes all watermarks from all sections. There's also no built-in way of applying the same watermark to all sections when some or all sections have had Link to Previous turned off. Here's the workaround.

After placing the watermark in the first section:
  • Double-click the header pane of the first section to open it.
  • Click the watermark to select it, and copy it to the clipboard.
  • In the Navigation group on the Header & Footer Tools tab, click the Next button to go to the Section 2 header. (If the button is grayed out, click in the header pane again to enable it.)
  • Paste from the clipboard.
  • Click Next again to go to the Section 3 header.
  • Paste from the clipboard.

In future documents, it will be quicker to insert the watermark in the first section's header before you turn off Link to Previous in the later sections. That way, the watermark will automatically appear in all the sections, and it will remain there after you unlink the headers.

jpg or png for powerpoint?

I have always used .jpg images for powerpoint slides. But I am wondering if .png would be better in terms of image quality. I know the file size is larger. Is there any downside to using .png? Any real advantage? Most of the images would be color photos, medical, surgical.

I suppose I could just make a sample pptx presentation with both formats and see if I can see the difference when projected, but I will have a presentation for a large audience and large screen in a couple of months, and I have no way of pre-testing in that environment.

Jack

john korchok say:

You can get an acceptable photo image at a smaller file size by using JPEG. PNG doesn't offer any significant advantage for photographic images. 

Line art, like a logo, is a different story. Since these images feature large areas of uniform color, you will get a much better image with a smaller file size by using a PNG file and indexing the color. A simple way to achieve this is copy and paste an image from a PDF. Acrobat automatically indexes the colors for you. I've written a couple of articles about this that have more detail: JPEG Logos? Fail! Logos - Production Secrets
