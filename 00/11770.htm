Problems with managing headers and footers for different sections - Microsoft Community

rockheart say:

Problems with managing headers and footers for different sections

I need to use different header for each of my sections in a long document. Thus, I uncheck 'Same as Previous' option in the Headers & Footers Ribbon to do so, but it factually doesn't work! I can't get it why!
Could anybody explain me why the things are so complicated?
john mcghie say:
Hi Rocky:

I had to set up a set of headers and footers for a book to print double-sided the other day, and I was horrified at how out-of-practice I have gotten :-)  It is a little tricky.

However, it's no more "complicated" than the document you are making, and that's the key to it.

You need to know three facts:

1)  Headers and footers are stored in the section break FOLLOWING the text to which they apply.

2)  There are potentially six headers and footers in each Section break: First Page, Even Page and Odd Page headers, and First, Even, and Odd footers.

3)  Only ONE of these headers and ONE footer exists by default: you have to turn the others on if you decide to use them.

In Format>Document>Layout, you must first decide which "side" this section will start on.  For double-sided printing in English, this will almost always be set to "Odd Page".

Next, decide whether you want alternating headers and footers for left and right pages: if you do, check "Different Odd and Even".  Instead of one header and one footer, the section now has four: left and right header, and left and right footer.

Normally, it looks silly to have a running header above the Chapter title, and by convention in English all Chapters start on a right page.  To give the first page of each section special treatment, check "Different First Page".  You now have six headers and footers in the section: headers for First Page (which is also a right page), Even Page, and Odd Pages (other than the first).  And footers for the same set of pages.

NOW "Same as Previous" will make sense.  If you happen to be in an Even Page header and you click "Same as Previous", AND there is a Section in the text BEFORE the current one, "same as previous" will copy the preceding EVEN page HEADER into the current header.  If there is no prior section break, the option won't be offered.

Similarly, if you happen to be in the Odd Page footer, same as previous will copy the odd page footer from the previous section.

Now, a couple of gotchas: one of which deeply embarrassed me last week :-)

*  If the previous section does not have the required header or footer, same as previous will copy forward from the previous section that does.  In my case, I wanted to put two sideways pages into a portrait section.  So I added two section breaks to encapsulate the landscape pages.  All good so far.

Then I switched off "Same as previous" so I did not trash my portrait headers and footers.  All good.

However, the problem set in in the FOLLOWING section.  I had forgotten when I changed the Header and Footer in the landscape section that I had changed the FIRST page and EVEN page headers and footers.  So in the following section, "same as previous" copied the LANDSCAPE headers and footers forward, not the PORTRAIT ones I wanted.

Another gotcha is that the Header and footer are typically blank in the front matter of a book.  If you are not alert, when you set headers and footers in the first Chapter, all of a sudden your headers and footers will pop up in the Front Matter and make you (or in this case, "me") look silly.

You have to keep your mind on the job!

To get different headers in a long document, it's often easier to use only one section break and a "StyleRef" field.  The StyleRef field copies text of the style you nominate into the header or footer.  If you choose "Heading 1" you conveniently get the Chapter Heading in the header: use this for your Left headers.  If you choose "Heading 2" in a StyleRef field, you copy the subsection header: use that your your right-hand headers.  Perfect dynamic headers all the way through the book, with only one section break added.

Hope this helps
