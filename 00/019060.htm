The quick and easy way is with a document property content control, which you can name and tag to be anything you want. If you rename it, you will want to copy and paste rather than insert again as a document property. It will work re-inserting, but you lose the name. The link is still there.

I got the idea for using the Document Property Content Controls from Greg's page on Repeating Data (Populating Multiple Like Fields). It is one of the many methods he discusses and can be the easiest one in the appropriate situation. It is limited in the number of controls available as well as in the types of controls you can do it with.

Windows 10 really wants to install the new drivers for the intel display. The new drivers will work if you do not care of the touch screen is upside down when in mirror mode.
