kathleen orland say:

If Outlook opens normally in safe mode, there is a pretty high chance that an Add In is preventing normal function.

 

First, make sure that when you close Outlook it doesn't remain running in the task manager > processes tab. If it does, end the task.

 

Open Outlook in safe mode and disable all Add Ins. Then open Outlook normally (no safe mode). It should open. Then add one Add In back and test. Repeat until you find the offender.

 

It will often be your antivirus integration (not required, your real time scanner will catch everything that it's updated to find), synch software (for use with a handheld), FAX software, etc.
