karl timmermans [outlook mvp 2012-2018] say:

One thing that can lead to confusion is "the terminology used" - in this case am specifically referring to "............and those in my <filed e-mail folders>" - not at all clear what you're referring to in terms of "filed e-mail folders".

Here are some simple rules to go by

#1 - Outlook never deletes any PST files so the only way anything would go AWOL is if you deleted it yourself

#2 - The PST file for email folders associated with an IMAP account in OL'2010 (and earlier) automatically gets recreated and sync'd with the server if the file is found to be missing or inaccessible

In short - really hard to loose data contained in PST files unless you explicitly delete the file yourself. The quick (and absolute best) way to know both the filename & location for any PST file configured in your Outlook profile is to go to <Account Settings> --> <Data Files> tab - the one thing you want to take special note of is the file designated as the "default" for the profile which is where you're default contact/calendar folders are stored.

For POP accounts configured, go to the <Email> tab, highlight each POP account individually and look down to the <Change Folder> button to see which PST file is associated with that particular POP3 account.

jeeped say:
Whenever an unwanted automated change is made through AutoCorrect, you can simply tap Ctrl+Z (i.e. Undo) immediately after the change to revert to your original typing. This applies to formatting changes as you've described as well as spelling corrections, hyperlinking, etc.

If you never want an initial asterisk to cause a formatting change to a bulleted list, go into File, Options (Word 2010) or Office Button, Word Options (Word 2007) and select Proofing from the .list down the left side. Click the AutoCorrect Options button and select the AutoFormat tab. Clear the Automatic bulleted lists checkbox. Click OK all the way out.
