Modify Default bar styles - Microsoft Community

baptm say:

Modify Default bar styles

Hello,

I'm wondering if there is a way to change the 'default' gantt chart bar styles in project ?

So that when i open a new project, i don't have to redo the formatting.

Do you know of such a thing ?

I can go around by copying a 'standard' empty formatted project, but it's not optimal.

Thanks.

john - project say:

BaptM,

Yes. Since bar styles are part of a view and a view is resident in the Global, all you need to do is to create a custom view, or views, each with your desired bar styles. Note, if your desired default view is based on an exiting view, (e.g. Gantt Chart), I recommend you make of copy of the existing view instead of directly modifying the existing view directly, in case you want to revert back to the default view. Then go to File > Options > General group and set the "Default view" to your customized view.

Once your new custom view is created, go to File > Organizer and transfer your custom view and the associated custom view table, (and filter and group if applicable), to your Global. Exit Project to save the Global.

The next time you start Project it will open to your custom view as the default.

Hope this helps.
