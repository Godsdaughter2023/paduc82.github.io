woodykiddy say:

I managed to resolved all these issue by myself after a few days of trail and error. The whole process wasn't easy but was worth it. A pretty good learning experience on how to do Win8 installation.

 

The missing HDD issue that I mentioned above was due to BIOS SATA settings. As ISRT is enabled by default on my laptop, I must load Intel Mobile Express Chipset SATA RAID Controller (iaStor.inf) driver to ensure that my hard drive would be detectable by the OS. Then I could pick a partition in the list to start installing the Win 8.

 

Alternatively, we could swtich ISRT to AHCI in BIOS at boot up, and all the available harddrives should be automatically picked up by the OS during installation. However, to be able to enjoy the performance boost from SSD caching, I personally still opted for "ISRT with manual loading drive" approach.

 

The rest was all straightforward. But just don't forget to turn out system protection when OS is installed sucessfuly and to create a restore point. Having a backup plan is always a good idea.

 

So summary of my approach:

1. Check SATA settings in BIOS, (use either ISRT / AHCI)

2. Boot up from installation DVD

3. Start re-installation

4. Load driver (iaStor.inf) if ISRT enabled

5. Pick a partition from the list and proceed with OS installation

6. Hopefully everything works great

considerate_guy say:

There is no option in the latest versions of Movie Maker to put a slideshow directly on to CD.  CDs are generally used for storing data or audio, not video.  If you want to put video onto a disk, your choices are to directly burn the .wmv or .mp4 (video) file onto the CD as a data file or to create a DVD. 

 

The first option (CD) is done outside of Movie Maker, just as you would for any files you want to put onto a disc.  This disc will not be playable in a standard DVD player, however, if that is what you are looking for.  It should be playable in another computer however.  Remember CDs can only accommodate about 680MB, so be sure your video is short.

 

The other option is to create a video DVD.  In most versions of Windows 7, there should be an option to Burn a DVD inside of Movie Maker.  What that actually does is call Windows DVD Maker which is included with most versions of Windows 7.  If you don't see that option, you may not have Windows DVD Maker.  In any case, if you are able to make a DVD this way, it should be playable in most standalone DVD players.

Prepare the USB Stick:

Notes

To reset your Surface from a USB recovery drive:
Step 1: Make sure your Surface is turned off and plugged in.
Step 2: Insert the USB recovery drive into the USB port on your Surface.
Step 3: Press and hold the volume-down button while you press and release the power button on your Surface.
Step 4: When the Surface logo appears, release the volume button.
Step 5: When prompted, select the language and keyboard layout you want.
Step 6: Select Troubleshoot > Reset your PC.

Step 8: Select Yes, repartition the drives, and on the next page, select Next.
