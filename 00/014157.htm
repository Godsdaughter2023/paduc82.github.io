Sub AutoF_CopyPaste_ListObj_02()

Const rTo$ = "A5" '<< top left cell in target sheet

Remove spaces at the end of text string - Microsoft Community

kourpa say:

Remove spaces at the end of text string

Can you please help me write a function that will remove spaces only at theend of a text string.
ronrosenfeld say:
On Mon, 19 Jul 2010 09:38:30 +0000, kourpa wrote:
 
>Can you please help me write a function that will remove spaces only at theend of a text string.
 
 
You can do this with a worksheet function.
 
This formula must be **array-entered**:
 
=LEFT(A1,MATCH(TRUE,LEN(TRIM(MID(A1,ROW(INDIRECT("1:255")),255)))=0,0)-1)
----------------------------------------
 
To **array-enter** a formula, after entering
the formula into the cell or formula bar, hold down
<ctrl><shift> while hitting <enter>.  If you did this
correctly, Excel will place braces {...} around the formula.
 
To make the function non-volatile, you could define a name such as
"seq"
 
seq  refers to: 
 
=ROW(INDEX(Sheet1!$1:$65536,1,1):INDEX(Sheet1!$1:$65536,255,1))
 
and then use the formula (still array-entered):
 
=LEFT(A1,MATCH(TRUE,LEN(TRIM(MID(A1,seq,255)))=0,0)-1)
 
If your string might be longer than 255, change all the 255's above to
a larger number.

hansv mvp say:

You could run a macro. The following macro will resize both inline and floating pictures (note the inconsistent way these are treated in Visual Basic <grin>):

 

Sub ResizeEm()
    Dim shp As Shape
    Dim ish As InlineShape
    For Each shp In ActiveDocument.Shapes
        Select Case shp.Type
            Case msoPicture, msoLinkedPicture
                shp.LockAspectRatio = True
                shp.ScaleHeight 0.5, True
        End Select
    Next shp
    For Each ish In ActiveDocument.InlineShapes
        Select Case ish.Type
            Case wdInlineShapePicture, wdInlineShapeLinkedPicture
                ish.LockAspectRatio = True
                ish.ScaleHeight = 50
        End Select
    Next ish
End Sub
