a. user say:

Hi Mike,

 

If Windows doesn't detect a network after it returns from standby mode, it could be because the wireless router or access point is turned off or isn't working properly.

 

What is the make and model of the PC?

Follow these methods and check if it helps:
Method 1:
You can use Device Manager to change the Power Management settings for a network adapter.
To disable this setting in Device Manager, follow these steps:
a. Press Windows logo key + X key together and choose Device Manager. 

b. Expand Network Adapters, right-click the adapter, click Properties.

c. Click the Power Management tab, and then clear the Allow the computer to turn off this device to save power check box.

Method 2:
If that does not help, update the wireless network driver and check if it helps:

a.  Press Windows logo key + X, choose Device Manager.
b. Expand Network Adapters.
c. Right-click on wireless device and click on Update driver and follow the on-screen instructions.


Let us know if it helps. If you have further queries, we will be happy to help.
