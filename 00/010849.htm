Unless this has changed very much, the Windows default is not changed (I think you'll find that if you print from your browser, for example, the laser printer will still be the default). But the active printer in Word is changed until you change it back or quit Word and start a new Word session. Note that the ScreenTip for the Print or Quick Print button will display the currently active printer.

I am in a similar situation, with a monochrome laser set as the Windows default and a color inkjet as a second printer that I use infrequently. What I find helpful is to have a toolbar button that allows me to print the current document from that printer.
