No you can't do that but you could use VB code to display dynamic content.  This assumes we already have a comment in cell A1. Right click the worksheet tab, view code and paste the code below in. Now if you change the cell A1, whatever you put in there is also added as the comment text,

 

 

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Cells.Count > 1 Then Exit Sub
If Target.Address = "$A$1" Then
    Application.EnableEvents = False
    Range("A1").Comment.Text Text:=Range("A1").Text
    Application.EnableEvents = True
    End If
End Sub

If you want a formula solution...

Data on Sheet1 in the range A2:B6.

Assumes no empty cells in the customer range B2:B6.

In the formula:

Name refers to Sheet1$A$2:$A$6

Customer refers to Sheet1!$B$2:$B$6

On your summary sheet, A2 = Joe, A3 = Sue

Enter this array formula in B2 and copy down to B3:

=SUM(IF(FREQUENCY(IF(Name=A2,MATCH(Customer,Customer,0)),ROW(Customer)-MIN(ROW(Customer))+1),1))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

Here are the steps to copy a shortcut to another folder:

  1. Make sure you've installed OneDrive on your computer. Installing creates a local folder that automatically syncs with your OneDrive cloud storage.
  2. Open up 2 windows, one pointing to your User folder (i.e. C:\Users\YourUserName) and the other to C:\Users\YourUserName\AppData\Roaming\Microsoft\Windows\Network Shortcuts.
  3. In the window pointing to C:\Users\YourUserName, right-click on the OneDrive folder (has a cloud icon). Choose Create shortcut. A shortcut icon is created in the window.
  4. Drag the shortcut icon to C:\Users\YourUserName\AppData\Roaming\Microsoft\Windows\Network Shortcuts.
  5. Before you close all the windows, test in Publisher by choosing Save As and choosing OneDrive as the Place. The dialog will still open to Network Shortcuts, but now you will see the OneDrive icon.
  6. Double-dick on the OneDrive icon and save your file. It is immediately synced with your OneDrive account.

There is not, really. There has never been such a feature in Word.

Word does not think in terms of pages but of units like paragraphs and sections.

If you use the built-in heading styles for the heading for a page (perhaps formatted for page-break before) you can move logical sections, which may be pages, around easily in either Outline view or using the Navigation Pane. I.e., you can move headings and associated text easily, but not "pages."

Why use Microsoft Word’s built-in heading styles? by Shauna Kelly

This method requires that you:

  1. Use a built-in heading style (probably H1) at the top of each page (Ctrl+Alt+1)
  2. Again, I would suggest that you modify the style to have paragraph formatting of page break before. 
  3. Restrict your typing so that text does not flow into a new page.

Powerpoint thinks in terms of slides. Acrobat thinks in terms of pages. In neither program does text flow from page to page as it does in Word. 

If you have Publisher, it would also allow moving of pages.
