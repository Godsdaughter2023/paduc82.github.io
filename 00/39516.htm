gerry c j cornell say:

The main difference is size. Memory dumps are larger and contain a little more information.

 

In most situations it is preferable to configure the computer to produce a kernel memory dump when system failure occurs. A complete memory dump is a very large file to handle so it is more practical to work with Kernel memory dumps when trying to resolve blue screen errors.

 

I suggest you change your dump files settings to produce a Kernel memory dump. Select the keyboard short cut Win+X , Control Panel, System, Advanced system settings, Advanced tab, and click the Startup Recovery Settings button. Under System failure check the boxes before Write an event to the system log and Automatically restart. Change the Write debugging information option using the scroll to the right to Kernel memory dump. Click OK and, in the next window click Apply and OK. Restart your computer to apply the changes.

Steps to perform SFC scan.

  4. A questo punto vi verrà chiesto se siete già in possesso di un Windows Live ID. Per poter modificare l’indirizzo email, infatti, avrete bisogno di un account di tipo Hotmail o MSN, Xbox.com oppure Passport Network. Se ne avete uno tra questi, selezionate Si ed andate avanti.

  5. Inserite la password del vostro account (che successivamente verrà sostituito con quello nuovo) e selezionate “Accedi“.

  6. Adesso inserite il nuovo Windows Live ID che verrà associato al vostro account XboxLive: scrivete l’indirizzo email e la password di accesso e selezionate nuovamente “Accedi“.

  7. Se i dati inseriti sono corretti, confermate le vostre modifiche selezionando “Si, modifica.“.

  8. Perfetto. D’ora in poi potrete utilizzare il vostro nuovo indirizzo email per accedere al vostro gamertag su XboxLive.

Naturalmente questa mini-guida è valida solo se avete la possibilità di accedere al vostro account XboxLive con le credenziali del vostro indirizzo email.
