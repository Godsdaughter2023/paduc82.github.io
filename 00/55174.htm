how do i turn on my wifi adapter on using cmd?

how do i turn on my wifi adapter on using cmd?

a. user say:

Hello Siphomangena,

Thank you for posting your query in Microsoft Community Forum.

If you want to enable the Wi-Fi adapter, you may try the following steps. However, it does not enable or disable the network driver installed in your computer.

  1. Go to Start, click All Programs and open Accessories.
  2. Right click on Command Prompt in the Program list and then select Run as administrator. If you are prompted for an administrator password or for confirmation, type your password, or click OK, or else click Yes.
  3. In the Command Prompt, type the following command, and then press ENTER:
    netsh interface set interface name="Local Area Connection" enable

    Note:     Replace Local Area Connection with the name of your adapter, e.g. Wireless Network Connection.
  4. This enables you Wi-Fi adapter.

Corollary: Replace "enable" with "disable" in the command in Step 3 in order to disable the Wi-Fi adapter.

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.

sigmathedj say:

Hi,

Here's what I did to sort this mess out.

1. Use this tool to download an ISO of Windows 10: -

https://www.microsoft.com/en-us/software-download/windows10

Make sure you select the correct version, so if you have Windows 8.1 Pro, you need to select Windows 10 Pro, for example.

2. Once the ISO has downloaded you have a couple of options: -

You can mount the ISO in Windows (right-click on it and choose "Mount"), then browse to setup.exe and run it.

You can burn a DVD of the ISO, insert it into your computer's optical drive, then browse to setup.exe and run it.

This will allow you to upgrade your current Windows installation to Windows 10.

Once the upgrade is complete, you will have a fully licensed version of Windows 10. If you choose the option to keep nothing when upgrading, you can run Disk Cleanup once the upgrade is complete to remove the leftovers of your previous installation which will have been dumped into a Windows.old folder, although doing that will mean that you can't go back to your previous version of Windows should you wish to.

Once you have upgraded and made sure that Windows 10 is fully licensed, if you burned a DVD of the ISO, you can use that to do a clean install in the future if you wish. If you do a clean install, when it asks for a key simply hit "Skip". Once the installation gets to the end it will ask you to sign into your Microsoft Account, so do that, then when it boots to the desktop you will have a totally clean, fully activated Windows 10 install.

A word of warning - Don't download the ISO and immediately try and do a fresh install using a burned DVD or a bootable USB stick. You MUST upgrade your existing Windows installation to Windows 10, then once you have done that, you'll be able to do clean instals in the future.

Hope that helps!

Figured out a solution after following some advice here (which i'll explain next):

http://www.tomshardware.com/answers/id-3271714/power-options-settings-managed-system-administrator.html

The steps I took were:

  1. Open Group Policy Editor (Win+R, then type "gpedit.msc")
  2. Navigate to [Computer Configuration]->[Administrative Templates]->[System]->[Power Management]
  3. Double click the Specify a custom active power plan policy setting
  4. Set to Disabled
  5. Click Apply then OK


I was then able to change my power settings. Not sure how this got out of whack but this solved it for me.

Running "powercfg -restoredefaultschemes" in cmd as admin
