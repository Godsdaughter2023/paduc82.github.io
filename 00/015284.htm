knighterrant66 say: I fixed my problem. Apparently there is a size limitation for the data consolidate routine in Excel and when you exceed the size limit Excel does not report an error or warning; it processes the data until the limitation is reached and then returns nothing - no consolidated data and no error. The size limit appears to be more that 44848 rows by 50 columns, as the first worksheet in which it worked was that size, but the second worksheet that had 66592 rows by 50 columns did not. When I sliced the second worksheet into two equal sized pieces I was able to get consolidate to work.
