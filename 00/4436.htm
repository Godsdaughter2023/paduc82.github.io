suzanne s. barnhill say:

Any filename that starts with ~$ indicates a temporary file (see “Description of how Word creates temporary files” at http://support.microsoft.com/?kbid=211632). These are usually deleted at shutdown, but this one apparently wasn't. And if the AZWzrd10.dot template were present, Word wouldn't be fooling around trying to open this temp file.

You can certainly get rid of the message by just deleting this file (which is useless) from Word's or the Office Startup folder. Then, if you really need the Avery template wizard, you can reinstall it, but most people find that Word's built-in Labels dialog (Tools | Letters and Mailings | Envelopes and Labels) pretty much has all the Avery label definitions needed.
