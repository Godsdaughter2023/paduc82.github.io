brian tillman [outlook mvp 2007-2019] say:

There's no such thing as a POP calendar.  The POP protocol is a mail-only protocol.  It knows nothing about calendar or contacts.  Moreover, it is one-way - from the server to the client.  Nothing can pass back up the link to the Inbox of the server.

 

Exchange ActiveSync accounts do synchronize, but since they must at all times reflect the contents of the server, they will always have their own folder sets, including calendar and contacts.  If you want to designate the EAS account as the default account and the EAS account's data file as the default data file, you'll be as close to what you want as you can get.

jllatham say:

OK!  We can do that with simple formulas.  In A10 on Room 01 sheet put this formula:

=OFFSET('All Rolls'!$A$1,ROW()-10,0)

The -10 would be the same as the first row number in any seating arrangement.

In B10 we need to account for seats already used, so we modify the formula a little:

=OFFSET('All Rolls'!$A$1,ROW()-10+10,0)

The +10 accounts for all the seats to the left of this column that would be filled.

In C10 this formula:

=OFFSET('All Rolls'!$A$1,ROW()-10+20,0)

All we had to change was +10 to +20

Now for the other formulas

cell E10  formula =OFFSET('All Rolls'!$A$1,ROW()-10+30,0)

cell F10  formula =OFFSET('All Rolls'!$A$1,ROW()-10+40,0)

cell G10  formula =OFFSET('All Rolls'!$A$1,ROW()-10+50,0)

cell I10  formula =OFFSET('All Rolls'!$A$1,ROW()-10+60,0)

cell J10  formula =OFFSET('All Rolls'!$A$1,ROW()-10+70,0)

cell K10  formula =OFFSET('All Rolls'!$A$1,ROW()-10+80,0)

At this point you can select cells A10:K10 and fill down to row 19 and your seating should populate automatically.  Unused seats near the end will show up as zero in the chart.

 

You can adapt that formula for your 2-seat benches easily also.

After an account becomes inactive, all messages, folders, and contacts are deleted.
Incoming messages will be sent back to the sender as undeliverable.
Your account name is still reserved.

If you don't use your Windows Live ID for 365 days, your Windows Live ID will be permanently deleted. 

Once the account is marked as in-active all data is purged so even if you could get the email account back the data would be gone.
