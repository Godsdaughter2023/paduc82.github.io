Ok, let's try to establish the exact method you are using.

 If you use the Media Creation Tool and you wish to install W10, you are requested to use a working PC and state that you want to install on another PC.  This routine will give you a "Bootable" USB.  Contrary to some on here, you do not need to use Rufus to make a "Bootable" USB since you already have a Bootable USB.

You now seem to have a "dead" PC likely due to the non-authentic W7.  This changes the mix for you.  You now have multiple issues. 

Just never saw a "dead" PC due to a bad O/S that I could have have back up and running in the time it takes to install using BIOS.  Heck, if I built the PC in my garage, it too has no O/S.

Here is what I do and I am assuming that you have a dead PC.  I also assume that you are happy with a Home Edition since you bought the key already -

1   Go into BIOS and change the load order to USB and close down

2   Insert the USB and Power ON and you should see USB activity.  This should confirm a load from USB

3   Go default on partitions

4   At some point you are asked to remove the USB

5   At this point I go back into BIOS and change load order to Drive C.  In real terms, if no USB present, it should default to Drive C but you need to change in any event

6   Re-start the PC.  You should be asked for a key and the Home key should load Home

7   Enjoy W10 Home.

A non-authentic W7 should not be an issue here.  You are actually installing Windows onto a PC with no prior O/S.

The price difference is such that there is no need to return your Windows 10 Home license.  If you are doing a clean install (strongly, strongly advised) all you need to do is install Windows 10 Home and then pay the difference for Pro at the online Microsoft Store.  You will see a link called "Go to Store" on the Windows 10 Home activation page.  The additional cost is $100.  

You can save $20 by buying a Pro license ($200) up front but only if you bought your Windows 10 Home from a source that will allow a return.  The Windows Store allows the return of a Windows license within 30 days (they block the product key so it can't be used).  

You can do an in-place upgrade from Windows 7 Ultimate to Windows 10 Pro with a Windows 10 Pro license but Home will require a clean install followed by an upgrade to Pro if you choose to do that as described in my first paragraph.  The advice in the forums is very heavily in favor of not doing that due to issues people have run into following such an upgrade.  A clean installation is by far the favored path so I would simply clean install Home and upgrade from the Store.
