teylyn say:

Hello,

here is an approach that might work. I load a table with two columns, "Column1" and "Column2". Then I create a custom column with an if statement that places text into the row where the trigger text appears and puts a null value in all other rows. Then use "Fill down" on that helper column, so only the text will be filled down to the last row. Now you can remove the null values by filtering them out. Finally, you can remove the helper column. Here is the full M code:

let
    Source = Excel.CurrentWorkbook(){[Name="Table1"]}[Content],
    #"Changed Type" = Table.TransformColumnTypes(Source,{{"Column1", type text}, {"Column2", type text}}),
    #"Added Custom" = Table.AddColumn(#"Changed Type", "Custom3", each if [Column1] = "Date Holiday" then [Column1] else null),
    #"Filled Down" = Table.FillDown(#"Added Custom",{"Custom3"}),
    #"Filtered Rows" = Table.SelectRows(#"Filled Down", each ([Custom3] = "Date Holiday")),
    #"Removed Columns" = Table.RemoveColumns(#"Filtered Rows",{"Custom3"})
in
    #"Removed Columns"
