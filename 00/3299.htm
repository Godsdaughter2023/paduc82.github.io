GPT implies a UEFI firmware enabled system. UEFI/GPT systems have 4 partitions by default, with a maximum of 128 partitions:

1) System (aka. EFI) partition, must be FAT32

2) MSR Microsoft Reserved partition.

3) Windows (aka. Boot) partition, must be NTFS.

4) Recovery partition.

Check MSDN for more details about GPT partitioning.

Mail Merge Printing Problem - Microsoft Community

pmpizzano say:

Mail Merge Printing Problem

I have created a 3 page letter and completed a mail merge.  I am trying to print this letter using double sided printing.  The problem is that the 2nd letter starts on the back page of the first letter - mail merge does not know where to start the next letter.  Short of inserting a blank page, does anyone know of any way to flag the start of a new letter?
suzanne s. barnhill say:
You're going to have to change the New Page section breaks between letters to Odd Page. I don't know of any way to do this in the mail merge main document, so it's going to have to be done in the merged document, which could be rather tedious if there are many letters, but here are a couple of ideas:
 
  1. A macro, which someone who can "do" VBA will probably come along and supply.
  2. Brute force: Use the Find dialog to search for section breaks (^b). Find the first, change the type, then use the Browse button (Find Next) to go to the next, F4 to repeat, Find Next, F4, etc. to the end.
  3. A refinement of #2: Change the section start type for Section 2, select the Next Page section break and copy to the Clipboard. Then use Replace to search for section breaks (^b) and replace with the Clipboard contents (^c).
 

doug robbins - mvp office apps & services (word) say:

Execute the merge to a new document and then run a macro containing the following code to send that document to the printer.  It will send each letter to the printer as a separate print job so that each letter will start on a new sheet of paper:

Dim i As Long
With ActiveDocument
    For i = 1 To .Sections.Count
        .PrintOut Range:=wdPrintFromTo, From:="s" & i, To:="s" & i
    Next i
End With
