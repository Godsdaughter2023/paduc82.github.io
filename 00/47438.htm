Hi Preston_C,

This issue occurs when all the following conditions are all true:

• You have a removable drive configured as drive letter C.
• You do not have a removable disk in the removable disk drive.

To resolve the issue, use one of the following methods:

Change the drive letter assignment for the removable drive to a letter other than C.

To change the drive letter assignments in Windows 7, follow these steps:

1. Log on to the computer as Administrator.
2. Insert a disk in the removable drive.
3. Click Start, right-click on Computer, and then click Manage.
4. Click Disk Management.
5. Right-click the partition, logical drive, or volume that you want to change, and then click Change Drive Letter and Paths.
6. Click the removable drive, click it, click Change, click the drive letter that you want to use, and then click OK.

That should be it.

Change, add, or remove a drive letter

I didn't bother with the recovery disk but got the issue fixed using the command stated in the article. Yes there are a lot of steps listed below but thats because I tried to be detailed. In all it takes like 5 minutes to do all the steps.

1. Insert the bootable Windows 8 or Windows 8.1 DVD install disk.
2. Reboot PC
3. Press F9 to change the boot order to boot from the CD/DVD drive (the install disk)
4. When prompted, press any key to boot from the CD or DVD...
5. Wait 3-5 min for the install program to load
6. On the first Windows Setup screen, click Next
7. On the next Windows Setup screen, click the 'Repair your computer' link in the lower left corner
8. On the next screen, choose the 2nd box ('Troubleshoot Refresh or reset your PC, or use advanced tools')
9. Choose the third box 'Advanced options'
10. Choose 'Command Prompt'
11. You should see a command prompt window
12. Type the following, pressing return after each line
c:
bootrec /fixmbr
13. Note the message, 'The operation completed successfully.'
14. Type the following, pressing the return after the line
exit
15. Eject installation DVD
16. Choose the last box 'Turn off your PC'
17. Power up after the PC turns off
18. Done

My experience and the steps above indicate the issue was with the master boot record. The upgrade from 8 to 8.1 must have buggered it up in some cases. I hope this helps!

UnInstall Bluetooth drivers and re-install Bluetooth drivers meant for Windows 7 or Windows 8.0 in "Windows 7 Compatibility Mode".

I had tried all suggestions and troubleshooting steps including disabling power saving options, doing a factory reset and clean install of Windows 8.0/8.1 but I continued to have problems with Microsoft Bluetooth mice (w/o USB dongles) randomly disconnecting. The ONLY solution that worked for me was un-installing existing Bluetooth drivers from Control Panel as well as Device Manager and re-installing Bluetooth drivers meant for Windows 7 or Windows 8.0 in "Windows 7 Compatibility Mode". The key here is installing Bluetooth drivers in "Windows 7 Compatibility Mode" by right-clicking the Bluetooth EXE file > Properties & selecting "Windows 7 Compatibility Mode". Avoid Bluetooth drivers meant for Windows 8.1. Luckily, my touchscreen laptop manufacturer (Acer) had both Windows 7 & Window 8.0 Atheros Bluetooth drivers for my model on their support website. Additional side-benefits I noticed after doing this:

  1. All three models of built-in Bluetooth Microsoft mice (Microsoft Scuplt Touch, Sculpt Comfort & Bluetooth Notebook Mouse 5000) started working perfectly without any sudden disconnects or freezing;
  2. The overall mouse operations also became lot smoother and predictable;
  3. Windows Power Management started working perfectly with the laptop going into sleep / standby nicely and waking up instantly without any lockups or WiFi issues
  4. CPU / Fan noise is much quieter-- it seems the Bluetooth drivers installed as I suggest are more "in harmony" with the rest of the system.
  5. Everything including the Bluetooth mouse is working great for almost a week since I made these changes. I promptly updated my complete system image backup with the new setup.

Hi there,

The Game DVR for Windows 10 is great when it works, but it has some issues.  It looks Microsoft cares because they are working on it (Updates are planned for the Anniversary update).

General support for the DVR can be found here:

Windows Game DVR System Requirements:

http://support.xbox.com/en-US/xbox-on-windows/system-requirements/pc-hardware-requirements-for-game-dvr

AMD:

  • Radeon HD 7000 series
  • Radeon HD 7000M series
  • Radeon HD 8000 series
  • Radeon HD 8000M series
  • Radeon R7 series
  • Radeon R9 series

NVIDIA:

  • GeForce 600 Series or later
  • GeFOrce 800M series or later
  • Quadro Kxxx series or later

Intel:

  • HD 4000 or later
  • Iris 5100 or later

Troubleshooting Steps:

Trouble shooter guides can be found at:

http://support.xbox.com/en-US/xbox-on-windows/social/troubleshoot-game-dvr-windows-10

Try Full-Screen Mode:

Sometimes you'll get an error "There's nothing to record" when running a game in windowed mode.  It'll refuse to run unless the game is in full-screen mode.  You'll have to change the settings in your game to get it to fill screen.  Another option is to press Alt-Enter and that may switch it to full screen.

You won't see the game bar in full screen so you'll need to use your shortcuts to start/stop recording.

http://support.xbox.com/en-US/xbox-on-windows/social/record-game-clips-game-dvr-windows-10

According to that article you need to press these keys in sequence to start recording.  Other keyboard shortcuts won't work.

  1. Windows + Alt + G
  2. Windows + Alt + R

Restart GAME DVR Task:

  1. Open up the Task Manager (Right click on the taskbar and go to Task Manager OR Press Windows-X and then press T)
  2. Find the DVR Server task (bcastdvr.exe).  Click it and go to End Task.

Delete Temporary Files:

  1. Open Settings App and go to System
  2. Go to Storage and choose This PC
  3. Scroll down and find Temporary files
  4. Click on it and then click on Delete temporary files.

This process may take a while to complete - it all depends on how many you have.  Wait until it's complete and then reboot your PC and try again.

Change the Shortcut Keys:

Sometimes it's possible that the shortcut keys are blocked by another program - you can't record by pressing Windows + Alt + R for example.

https://support.microsoft.com/en-us/instantanswers/7ed48b7a-438b-6748-067f-eebbd3328da8/fix-problems-with-game-bar

You can change these or assign a secondary shortcut by:

  1. Start menu
  2. Xbox
  3. Settings
  4. Game DVR
  5. Keyboard shortcuts

Reinstall or Upgrade Your Graphics Card Drivers:

Visit your graphics card manufacturer's website, locate your model and download the latest drivers.  Install/reinstall them, reboot and try the DVR again.
