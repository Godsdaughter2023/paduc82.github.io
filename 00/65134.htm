a. user say:

The simplest way to delete the contents of the temp folder is to use Disk Cleanup:  http://www.vistax64.com/tutorials/76073-disk-cleanup.html..  When you come to the options box, just deselect everything except the temporary folders and run the program (or you can do a better cleanup if you want and have the time).

If that doesn't work, try Unlocker at: http://www.softpedia.com/get/System/System-Miscellaneous/Unlocker.shtml which sometimes works when Vista's delete program doesn't.

The folder could be open for a number of reasons - programs currently running actively or in the background or something that didn't close properly.  Try rebooting and see if you can delete it before you open any programs other than those in the start menu.

If that doesn't work, try booting into safe mode (repeatedly clicking on F8 while booting) and try deleting the temp folder from there.

If that doesn't work, do a clean boot http://support.microsoft.com/default.aspx/kb/929135?p=1 and see if you can delete the folder now.  Regardless, remember to reset the system back to normal operation when you are done (this is included in the linked procedures above).

I hope this helps.

Good luck!
