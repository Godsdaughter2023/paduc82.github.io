  1. In the Lock screen, press/hold the shift key and click on power icon>restart.

mala_s say:

Hello RCarelli1,  

Thank you for writing to Microsoft Community forum.

The above error message seems to be caused by File Stream due to hard disk problem or corrupt system file. Kindly follow the below mentioned steps to resolve the issue.

Step 1: Stop ADSM service.

Go to search window and type services.msc and press Enter. Scroll down to ADSM (ASUS data security management) service, right click on the service and select Stop.

Step 2: Run System File Checker (SFC) command.

System File Checker is a utility in Windows 10 that checks for system file corruption. You may follow the article on

Step 3: Run CHKDSK /F /R command.

  • In the search box on the taskbar, enter Command Prompt. Press and hold (or right-click) Command Prompt (Desktop app) from the search results and select Run as administrator.

  • Type CHKDSK /F /R command and press Enter.

Wait until the command completes 100% and later you may try to save the files.

Check disk disclaimer: While performing chkdsk on the hard drive, if any bad sectors are found on the hard drive then chkdsk tries to repair that sector, if any data available on that might be lost.

Kindly get back to us if you face any further issues. We are here to help you.

Regards,
