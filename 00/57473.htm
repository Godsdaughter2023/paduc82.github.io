frederik long say: You can't add a user account/password to a backup drive but you can certainly add a user account/password to a Windows installation. If you find the GUI method confusing then here is a two-command method.
1. Get onto the machine that you use as a file storage medium.
2. Click the Start button.
3. Type the three letters cmd
4. Press Ctrl+Shift+Enter
5. Launch the Console session as an Administrator.
6. Type this command:
    net user backup xxyyzz /add{Enter}
    This command will create an account called "backup" with a password of xxyyzz.
7. Type this command:
    net localgroup administrators backup /add{Enter}
    This command will promote the "backup" account to Administrator level.

When creating the backup task, use "backup" as an account name and "xxyyzz" as a password.

Depending on the actual backup program you use, you may have to go through Steps 1 to 7 on the other machines too.
