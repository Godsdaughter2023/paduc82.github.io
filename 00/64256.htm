Do the various sound quality options on Windows 7 make any - Microsoft Community

darklaw say:

Do the various sound quality options on Windows 7 make any difference?

I have a Windows 7 64bit, and Logitech Z623 2.1 speakers (it's a 2.1 setup).

I have changed in the Windows playback devices, under those speakers device, under the "Default Format" option, the sampling rate from the default 16bit (lowest) to  "24bit 192000 Hz (Studio Quality)" option - which is the best.  "24bit 192000 Hz (Studio Quality)" option - which is the best.

I want to know - does it have any effect on the sound quality that I hear? I don't necessarily feel any difference - but theoretically – what's necessary to feel the difference - On the speakers (and do my qualify), on the music files (what bit rate do they have to be in order to feel the difference - all my music is 320 bit).

Thanks

balaji kundalam say:

24-bit present in the setting is called the bit depth, which is the number of bits of information in each sample (sample is short clip of speech or music extracted from a longer piece of audio, in the case of digital world). 192000 Hz or 192 kHz is the sample rate, which is the number of samples of audio carried per second. There will not be much difference when we change the sample rate from 44100Hz to 48000Hz to 92000Hz to 192000Hz, as far as the sound quality is concerned as the 2.1 speaker has it's own amplifier (a device that removes noise and is used to control the volume level of audio) to drop any unwanted noise. When it comes to headphones/speakers without buit-in amplifier, little difference can be noticed. When the sample rate is reduced below 44100Hz, we can notice a drop in the audio quality.

Kindly note that 320 bit is not bit depth, it's the bit rate of the file, defined as 320kbps (kilo bits per second). Bit rate is the number of bits (8 bits = 1 Byte) that are processed per unit of time. Higher the bit rate, higher the sound quality of the file as well as the file size. We can notice significance difference in the audio quality if the bit rate is adjusted. Bit rate is specific to a file and differs with each file. It doesn't have any impact on the audio settings/sound quality options Windows (or any operating system) offers.

Right click the network icon in System Tray at bottom right end of task bar to Troubleshoot Problems.

Check that the wireless button on a laptop is toggled on. It may be a Function (Fn) key or it may be an actual switch on the side.

Try powering off modem, router and PC and then restarting in that order with 1 minute in between each.

Try updating the Network adapter driver from the PC's Support Downloads webpage. http://answers.microsoft.com/en-us/windows/wiki...

If that doesn't help try Rolling back and then Uninstalling the Network adapter driver in Device Manager. Right click Start to open Device Manager, choose the Network adapter you're using, on Driver tab choose Roll back, restart PC. If that doesn't help ry Uninstall, restart to reinstall.

Go over this checklist to make sure the install is set up correctly, optimized for best performance, and any needed repairs get done: http://answers.microsoft.com/en-us/windows/wiki...

Start with Step 4 to turn off Startup freeloaders which can conflict and cause issues, then Step 7 to check for infection the most thorough way, then step 10 to check for damaged System Files. Then continue with the other steps to go over your install most thoroughly.
