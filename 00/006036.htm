graham mayor say:
ASD files are not supported. Nor do they necessarily contain a copy of the document. They are recovery files, which if valid will recreate the document when Word is restarted following a crash. They are not foolproof, but an aid to recovery.

If the file was a genuine copy of the document, if you rename it to have a ZIP extension, it should open with Windows Explorer to reveal the internal file structure. Or give it a DOCX extension and attempt to open it. It will almost certainly fail.
