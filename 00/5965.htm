doug robbins - mvp office apps & services (word) say:

The only customization of a page number that will be picked up in the Table of Contents is the ability to include the Chapter Number (based on a Heading # style).

While you can define a numbering system that uses A, B, C numbering and starts at F, with the F followed by O so that when the Heading style to which that number is linked is used in the document, the FO will appear, when the page numbering is formatted to include that Heading Numbering, it is only the F that will appear in the page number.

I think that the only way to achieve what you are after would be after the document is complete, unlink the Table of Contents by selecting it and then Pressing CTRL+SHIFT+F9 and then manually edit the page numbering.

Thanks for any help!

suzanne s. barnhill say:

Are you referring to the navigation pane that opens on the left when you press Ctrl+F or the actual Find/Replace/GoTo dialog that opens when you press Ctrl+H?

If the former, then yes, you can mouse over the right-hand side of it till the mouse pointer changes to a double-headed arrow, then drag to the left to reduce the pane size. There's a certain point at which it can't be reduced further, however.

If the latter, then no, there's no way to change the size of the dialog except by clicking More to expand it or Less to condense it. But if you are just using the Find command (rather than Replace), you can, after the first search, close the dialog and use the browse arrows at the bottom of the vertical scroll bar, which have the function Find Previous and Find Next until you use Select Browse Object (the button between the arrows) to change the "browse object" back to Page or to some other object.

You can turn off AutoCorrect. In the AutoCorrect options dialog, clear the check box for "Replace text as you type".

However, that isn't necessary. Type HSI in a document and let it change to HIS. Then move the mouse over the word; a small blue rectangle will appear below the H. Click the rectangle to open a menu, and click the item 'Stop Automatically Correcting "his"'.


You should be able to do the same thing by adding HSI (or hsi) to the RoamingCustom.dic file through the Custom Dictionaries dialog. At least, that's where the word appeared when I used the'Stop Automatically Correcting' menu item.
