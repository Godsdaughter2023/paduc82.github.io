balaji kundalam say:
You can configure Windows XP Mode to use Shared Networking (NAT) and there is no need to setup additional WiFi network. In this setting, the network to which the host (Windows 7 Ultimate) is connected, is shared with Windows XP Mode.

Once Windows XP Mode is started (menu not available in fullscreen), go to Tools->click Settings. Click on Networking on the left pane.
