a. user say:

No - not even close!

Almost any third-party backup software is easier to use and has more features than Windows' own backup utilities.

Like other Windows utilities - Scan, Reader, Windows Defender, etc. - File History provides a basic level of functionality for people who don't know how or are too afraid to install software.

If you are comfortable with installing software, you will find many great choices in backup software. Acronis True Image is one example. (I used it myself for several years.) And there's no reason why you can't use one application for making disk images and a different one for making file and folder backups.

Most important is to schedule your backups so they happen automatically, every day. That way, you're never more 24 hours out from your most recent backup.
