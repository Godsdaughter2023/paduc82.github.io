per jessen say:

This should get you started:

Sub CopySheetToOtherWbk()
Dim CopyFromBook As Workbook
Dim CopyToWbk As Workbook
Dim ShToCopy As Worksheet

Set CopyFromWbk = Workbooks("NameOfBookToCopyFrom.xlsx")
Set ShToCopy = CopyFromBook.Worksheets("Sheet1")
Set CopyToWbk = Workbooks("NameOfOtherWorkbook.xlsx")

ShToCopy.Copy After:=CopyToWbk.Sheets(CopyToWbk.Sheets.Count)
End Sub

joeu2004 say:

Steven wrote:

I have been working on a STATS page for a spreadsheet at work. i have just realised that work has Office 2003 and i am working on 2007.  How do i convert the following formulas to 2003.

=COUNTIFS(SQUAD!Q$2:Q$109, "<=" & EDATE(TODAY(), 6), SQUAD!D$2:D$109, A3)
and
=COUNTIF(SQUAD!H95:H101,"P*")


No need to change COUNTIF.  That is compatible with Excel 2003.

Instead of COUNTIFS, the following works in all Excel versions:

=SUMPRODUCT((SQUAD!Q$2:Q$109<=EDATE(TODAY(),6))*(SQUAD!D$2:D$109=A3))

The multiplication (*) serves two purposes.  It functions like AND, which we cannot use in this context with the intended purpose.  And it converts TRUE and FALSE to 1 and 0, as needed.

There is a difference between the way that COUNTIFS and SUMPRODUCT interpret numeric text in cells.  COUNTIFS will interpret it as numeric; SUMPRODUCT will interpret it as text.

So try the following instead:

=SUMPRODUCT((--SQUAD!Q$2:Q$109<=EDATE(TODAY(),6))
*(--SQUAD!D$2:D$109=A3))

The double negative (--) has the effect of converting numeric text to numbers.  It will work with numbers, too.

=SUMPRODUCT(

It sounds as though you're confusing the slide's .Name property with the title text that happens to appear on the slide, if any.  They're not in any way related.  To find the name of the slide currently in the main window, Normal view,  go to the immediate window and do this:

? activewindow.Selection.SlideRange(1).Name

With ActivePresentation.Slides("My Slide Name")

End With

 

If you need to get a reference to a slide with a specific title, that's another matter.

Sub RangeToPresentation()
' Set a VBE reference to Microsoft PowerPoint Object Library

Dim PPApp As PowerPoint.Application
Dim PPPres As PowerPoint.Presentation
Dim PPSlide As PowerPoint.slide

' Make sure a range is selected
If Not TypeName(Selection) = "Range" Then
    MsgBox "Please select a worksheet range and try again.", vbExclamation, _
        "No Range Selected"
Else
    ' Reference existing instance of PowerPoint
    Set PPApp = GetObject(, "Powerpoint.Application")
    ' Reference active presentation
    Set PPPres = PPApp.ActivePresentation
    PPApp.ActiveWindow.ViewType = ppViewSlide
    ' Reference active slide
    Set PPSlide = PPPres.Slides(PPApp.ActiveWindow.Selection.SlideRange.SlideIndex)

    ' Copy the range as a piicture
    Selection.CopyPicture Appearance:=xlScreen, _
        Format:=xlPicture

    ' Paste the range
    PPSlide.Shapes.Paste.Select
   
    ' Align the pasted range
    PPApp.ActiveWindow.Selection.ShapeRange.Align msoAlignCenters, True
    PPApp.ActiveWindow.Selection.ShapeRange.Align msoAlignMiddles, True

    ' Clean up
    Set PPSlide = Nothing
    Set PPPres = Nothing
    Set PPApp = Nothing
End If

MsgBox "Copied cells " & Selection.Address

End Sub
