graham mayor say:

What sort of 'dropdown menu'? Can we assume content controls? You can do it with a macro. In the following code replace the bold words with the words that you want to find i.e. "Name" and replace i.e. "Item". The macro collects all the list entries from each dropdown into a text string in which you can search and replace the text (this is case sensitive). It clears the entries from the lists then puts back the changed entries.

Dim oCC As ContentControl
Dim oEntry As Object
Dim strEntryList As String
Dim vEntry As Variant
Dim i As Long
    For Each oCC In ActiveDocument.Content.ContentControls
        If oCC.Type = wdContentControlDropdownList Then
            strEntryList = ""
            For Each oEntry In oCC.DropdownListEntries
                strEntryList = strEntryList & oEntry & "|"
            Next oEntry
            oCC.DropdownListEntries.Clear
            strEntryList = Left(strEntryList, Len(strEntryList) - 1)
            strEntryList = Replace(strEntryList, "Name", "Item")
            vEntry = Split(strEntryList, "|")
            For i = 0 To UBound(vEntry)
                oCC.DropdownListEntries.Add vEntry(i)
            Next i
        End If
    Next oCC

If you only want to change one entry from a variety of similar entries leaving the searched text intact in the other entries, then that is possible by moving the replacement to after the split and before the entry is written back to the list, so you would need the following instead

Dim oCC As ContentControl
Dim oEntry As Object
Dim strEntryList As String
Dim vEntry As Variant
Dim i As Long
    For Each oCC In ActiveDocument.Content.ContentControls
        If oCC.Type = wdContentControlDropdownList Then
            strEntryList = ""
            For Each oEntry In oCC.DropdownListEntries
                strEntryList = strEntryList & oEntry & "|"
            Next oEntry
            oCC.DropdownListEntries.Clear
            strEntryList = Left(strEntryList, Len(strEntryList) - 1)
            vEntry = Split(strEntryList, "|")
            For i = 0 To UBound(vEntry)
                If i = 1 Then 'The lists are numbered from 0
                    oCC.DropdownListEntries.Add Replace(vEntry(i), "Item", "Name")
                Else
                    oCC.DropdownListEntries.Add vEntry(i)
                End If
            Next i
        End If
    Next oCC

Acronis True Image does not support Windows Insider Program builds (beta builds of Windows). Although, Acronis True Image usually works on beta builds, you might experience issues when using our software with these versions of Windows.

As stated in the Windows Insider Program Agreement, Windows Insider program participants are volunteers who receive experimental and early pre-release software. This means that these new builds of Windows that you receive through this program are still in development and testing and are not production-ready yet. These builds are not as stable as the official release builds are and may have issues that affect the software installed on these Windows versions. During the test period, Microsoft fixes many of the issues within the operating system and thus the software issues may be resolved as well.

We do test our software on beta builds internally, however, thorough tests are made on the late beta builds and release candidate. For Acronis True Image, we can state the official support of a new Windows OS build only after its official release.
