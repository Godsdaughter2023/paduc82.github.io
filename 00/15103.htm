Just as experiment to identify the problem, try executing the following macro some time before doing the copy-and-paste:

Sub doit

I am not suggesting that you set that mode permanently; it might cause significant delay to occur with each recalculation.  Change True to False and re-execute the macro after the experiment.

Shortcut open website in private

IE10 cannot be set to default of In Private, but a shortcut will work. How can I create a shortcut to open a specific website in private?

don varnau say:

Go to [C:\Program Files\Internet Explorer] right-click iexplore.exe and send to Desktop as a shortcut.

Right-click that shortcut> Properties> Target. Change the Target to (for example)

"C:\Program Files\Internet Explorer\iexplore.exe" -private www.bing.com

Note the spaces

"C:\Program Files\Internet Explorer\iexplore.exe"<space>-private<space>www.bing.com
