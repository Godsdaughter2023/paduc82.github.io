end repeat

Apart from Outlook, you can run multiple Office versions on the same platform. Here for example I have elements of 2003, 2007, 2010 & 2013 all co-existing quite happily, with 2010 as my default.

bob jones aka: cybertaz say:

Your concern was addressed several times earlier in the conversation... Any type of placeholder always will overprint any user content placed on the Master. Placeholders are programmatically designed to do so because users are intended to be able to access Placeholders but not intended to be able to access Slide or Layout Master content.

Think of inserting a slide as a 2 phase process: In the 1st phase, the foundation of the slide is generated based on the content designed into the master. That master content is not accessible on the slide. As the 2nd phase of the generation process the program adds the Placeholders onto the slide as slide content based on the size & position specified by the master.

I understand that the explanation isn't what you would prefer it to be, but there is nothing that can be done at the user level to change that fact. stefan blom say:

F9 only updates fields in the selection, which means that fields in text boxes or in headers and footers won't update by Ctrl+A followed by F9.

However, you can always display the Print screen (File | Print) and then return to Print Layout view (for example by pressing ESC), which would update most fields.

stefan blom say:

The "rules" for updating fields in Word have always been confusing (or even inconsistent). Going to the print preview and then back to Print Layout view has always worked, though, and it is not necessary to select the "Allow fields containing tracked changes to update before printing" option. If you are tracking changes in the document, perhaps you want to select the option, but you don't have to.

You may want to select "Update fields before printing" at File | Options | Display, but (for most fields) it isn't required. The SEQ fields used by the Caption feature will be updated without that option being selected.
