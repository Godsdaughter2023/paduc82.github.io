There could be a number of factors as to why you are experiencing Night Light issue. One possible caused could be due to an outdated graphics driver. You can refer to the steps below on how to troubleshoot Night Light issue is Windows 10:

Method 1: Update the graphics card driver.

  1. Press Windows key + R on your keyboard.
  2. On the run command box, type devmgmt.msc, and then hit Enter.
  3. Expand Display adapters.
  4. Right-click the graphics card driver and then select Update Driver Software.


Method 2: Uninstall and reinstall the graphics card driver.

  1. Press Windows key + R on your keyboard.
  2. On the run command box, type devmgmt.msc, and then hit Enter.
  3. Expand Display adapters.
  4. Right-click the graphics card driver and then select Uninstall.
  5. Restart your computer. The Display Adapter Driver will automatically reinstall after the Restart.


Method 3: Reset Night Light settings manually.

This steps will require you to modify some settings in Registry. This is an advanced troubleshooting so I suggest that you create a System Restore point so you can go back to this state in case that you will encounter any issues along the process. To create a system restore point, refer to this article.

Once you're done creating a system restore point, you may then proceed to modify the registry settings. To do this, refer to the steps below:

  1. Press Windows key + R on your keyboard.
  2. On the run command box, type regedit and then hit Enter.
  3. Navigate to this path:

    HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\CloudStore\Store\Cache\DefaultAccount

  4. Inside the “DefaultAccount” (folder) key, right-click and delete the following folders:

    $$windows.data.bluelightreduction.bluelightreductionstate
    $$windows.data.bluelightreduction.settings

  5. Restart your computer.

CCleaner is a generally good program and I recommend it. However the
one area where it is not good and I strongly recommend against using
it, is its registry cleaning functionality.
Registry cleaning programs are all snake oil. Cleaning of the
registry isn't needed and is dangerous. Leave the registry alone and
don't use any registry cleaner. Despite what many people think, and
what vendors of registry cleaning software try to convince you of,
having unused registry entries doesn't really hurt you.
The risk of a serious problem caused by a registry cleaner erroneously
removing an entry you need is far greater than any potential benefit
it may have.
Read http://www.edbott.com/weblog/archives/000643.html

and http://aumha.net/viewtopic.php?t=28099

and also
http://blogs.technet.com/markrussinovich/archive/2005/10/02/registry-junk-a-windows-fact-of-life.aspx

Let me point out that neither I nor anyone else who warns against the
use of registry cleaners has ever said that they always cause
problems. If they always caused problems, they would disappear from
the market almost immediately. Many people have used a registry
cleaner and never had a problem with it.

Rather, the problem with a registry cleaner is that it carries with it
the substantial risk of having a problem. And since there is no
benefit to using a registry cleaner, running that risk is a very bad
bargain.

basith m say:

Hi,

First of all, I would like to apologize for the delay in responding.

I would suggest you to try the following steps and check if it works for you.

Follow the steps to change the default location of Windows Explorer:

a. Right-click on Windows Explorer (or its shortcut) and then click Properties.

b. In Windows Explorer Properties, click on Shortcut tab if you’re not already there.

c. The text box after “Target” by default will shown the value as below:

%SystemRoot%\explorer.exe

d. Change the "Target" parameter in the text box to:

%SystemRoot%\explorer.exe /n, /e, <New Path>

Note: Here New Path represents the full path to the new different folder that you want Windows Explorer to open as default folder on launch. Here the spaces should not be avoided.

For Example "%SystemRoot%\explorer.exe /n, /e, D:\"

e. Click on Ok to save changes.
