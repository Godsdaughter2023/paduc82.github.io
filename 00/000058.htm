Hi Scott,

Answers:
Q1. What will happen to the users that currently have Premium licenses but will no longer be covered?
A1. 10 users with expired Office 365 Business Premium licenses will be in an unlicensed state, which will affect their usage of related Office 365 service.

Q2. Question about renewing 55 licenses of 65.
A2. The 55 licenses will be assigned randomly. As a workaround, you can remove other 10 premium licenses from the uses that you don’t want to renew now. Then renew the 55 Premium licenses, the 55 Premium license will be assigned to the 55 users that have licenses assigned. After that, he can reassign the 10 licenses back to the users and let them expire.

Q3. Will the E3 licenses just be automatically assigned to random people to cover the Premium licenses that will be gone?
A3. For those 10 people, Office 365 E3 license won’t be automatically assigned to them. Admins need to manually assign Office 365 E3 license to the users.

EXCEL - one minute to 15 minute data - Microsoft Community

katierassmn say:

EXCEL - one minute to 15 minute data

I have 1 minute soil moisture data in excel, paired with the date/time for that date.  I would like to average these values out to every 15 minutes (and maintain the time stamp at 15 min intervals).  How can I do this?

 

Column A has the date/time in the following format:  mm/yy/dd hh:mm

Column B has the soil moisture value:  0.00

gary's student say: In C1 enter:

=ROUND(A1*96,0)/96  and copy down - this rounds to 15 minutes

In D1 enter:  1 and in D2 enter:

=IF(C2=C1,D1,D1+1)  and copy down - this assigns a data group number for averaging

Finally enter the array formula in E1:

=AVERAGE(IF(D1:D100=ROW(),B1:B100))  and copy down.

An array formula must be entered with CNTRL-SHFT-ENTER rather than just the ENTER key.

bernie deitrick say:

Here is a better way - no keys, works on a single worksheet of a single workbook.  This is event code (a macro) that will save the file every time a row is completed.

Let's say that when a row is completed, there are the same number of entries in that row as in the header row (row 1) - I.E., what the formula

=COUNTA(1:1) 

would return.

1) Copy this code.
2) Right-Click the sheet tab of interest.
3) Select "View Code"
4) Paste the code into the window that appears.
5) Save the file as a macro-enabled .xlsm file.
6) Make changes to the header range  - "1:1"  as needed

Private Sub Worksheet_Change(ByVal Target As Range)
    Application.EnableEvents = False
    If Application.CountA(Target.EntireRow) = Application.CountA(Range("1:1")) Then ThisWorkbook.Save  
    Application.EnableEvents = True
End Sub
