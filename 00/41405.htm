How to delete extra Vista Operating System. I am getting two Vista - Microsoft Community

sklein say:

How to delete extra Vista Operating System. I am getting two Vista Boot Entries on startup

Original Title: Deleting Vista Operating System

I accidentally installed a second Vista operating System on my computer. how can I delete the new one?

mouneshwar r say: Hi sklein,

Welcome to Microsoft Answers!

According to you, you’re getting two Windows Vista Boot Entries when you switch on the computer. This means you’ve installed Windows Vista on two different partitions on the computer.

Firstly, boot to your older version of windows vista and check if everything is working fine.

Then, follow the steps to format the partition that contains the new windows vista.

   1. Click on Start; Right click on Computer and click on Manage.
   2. Click on Disk Management and format the partition where you’ve your New windows Vista.

Note: In order to check which partition has the new operating system installed, you can open Windows Explorer by pressing Windows + E button and check the size of the partition.

The newly installed operating system partition will have three basic windows files such as Program Files, Users and Windows and nothing else in it. Moreover you can figure it out easily by checking the size of both the partitions. Make sure that you know the free size and the total size of the partition that contain the newly installed vista so that you can format it through Disk Management.

   3. Now, you can format the partition where you have the New Windows Vista operating system. To format, right click on the partition and click on format.
   4. Now the partition formatted will turn out to be unallocated.
   5. Right click the partition and click on New Volume and follow the instructions to create a healthy partition.
   6. Close the Computer Management window.

Now restart the computer and you’ll still find two Windows Vista boot entries and the reason is the boot entries for both the operating system reside in the root partition which is your default system partition.
 
Among the two entries, the first entry would take you to the desktop and the second entry will be non-executable operating system.

So, in order to delete the non-executable operating system, follow the steps below.

   1. Click on Start; In the Start Search type msconfig and hit Enter.
   2. Click on the Boot tab. Select the operating system which says (no-execute).
   3. Delete it and click on apply and save.

Restart the computer and you’ll have only one operating system entry and one operating system installed.

Regards,
