Stacks are reserved memory that programs use to process hardware events. A stack overflow occurs when there is not enough space in memory to run the hardware interrupt routines.

 

When Windows delivers an error message that is related to an internal stack overflow, there is not enough space in memory either set aside or available to handle the calls that are made to the computer hardware.

Outlook allows you to export your contacts to the default Contacts app on Android. This allows you to easily see the name of a contact when you receive a call or text message from them, and view all of their contact information directly in the default Contacts app.

Two-way contact export is supported for O365, Outlook.com and OPCC accounts. Two-way support ensures that any new contacts you add or existing contacts you edit will be synced between your device and Outlook accounts.

Ok.. here's what you'll need to flash under linux..

 

  1. 2gig USB stick (ideally larger, I used an 8 gig one.. not sure if it matters)
  2. Ubuntu 9.10 desktop ISO
    Torrent: http://releases.ubuntu.com/9.10/ubuntu-9.10-desktop-i386.iso.torrent
    Direct download: http://releases.ubuntu.com/9.10/ubuntu-9.10-desktop-i386.iso
  3. USB Ubuntu installer for Windows: http://www.pendrivelinux.com/downloads/u910/USB-Installer-for-Ubuntu-v0.2.exe
  4. Flasher debian package from Nokia:
    http://tablets-dev.nokia.com/maemo-dev-env-downloads.php
    The file you need is called: maemo_flasher-3.5_2.5.2.2_i386.deb
  5. The eMMC Image for the N900:
    http://tablets-dev.nokia.com/nokia_N900.php
    The file you need is: RX-51_2009SE_1.2009.41-1.VANILLA_PR_EMMC_MR0_ARM.bin
  6. The OS Image for the N900:
    http://tablets-dev.nokia.com/nokia_N900.php
    Make sure you download the correct file for your region: USA, Middle East/North Africa, Global or UK
  7. The Flasher script:
    http://chris.intrepid.cx/N900/flasher.zip


On you windows machine:

 

Run the Ubuntu installer (downloaded in step 3) and install Ubuntu onto your USB drive. It will ask you where your Ubuntu ISO is, this it the file downloaded in step 2. It will take some time to finish

 

Once it has finished, open up your USB drive and extract the zip file onto the USB drive

This will create a directory called "flasher". Inside that directory you will see 1 file (flasher.sh) and 3 directories.

 

Copy the maemo_flasher-3.5_2.5.2.2_i386.deb file downloaded in step 4 into the deb directory

Copy the eMMC image downloaded in step 5 into the emmc directory

Copy the OS image downlaoded in step 6 into the os directory

 

This completes the preparation..

 

Now reboot your system and boot from the USB device.. It will take a little while.. Once you see the desktop, do the following:

 

Click on Applicatons at the top of the screen, then click "Accessories" then click Terminal

 

A white box will appear

 

in this box type these commands:

 

sudo -s

cd /cdrom/flasher

bash flasher.sh

 

You will be asked to install the Maemo flasher.. a blue box may appear, use the cursor keys to select OK to accept the license and proceed.

 

After that is finished you should see a menu with 4 options:

 

1) R&D Mode

2) Flash OS

3) Flash eMMC

4) Exit

 

First try to flash the eMMC image:

press 3 then enter

 

The next menu will give you 4 options

press 1 to select the eMMC image you want to flash

select 1 again to select  emmc/RX-51_2009SE_1.2009.41-1.VANILLA_PR_EMMC_MR0_ARM.bin

If it tells you no images are found, or you can't see this item in the list, STOP!!!!

Now select 3 again to start flashing..

 

Remove the battery on your phone, replace the battery, press and hold down 'u' and insert the USB cable.. 

 

The flasher should start..

 

Let me know how you got on..
