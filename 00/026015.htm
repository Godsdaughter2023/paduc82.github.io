hi alex,

we can set separate permissions for a list with the steps below:
1.navigate to the list.
2.click the list tab>list settings>permission for this list.

3.click stop inheriting permissions.
4.now, we can delete the groups or users that we don't want them to access this list. grant permissions to the groups/users who we want them to access.

moreover, if we click setting gear>site settings>site permission, it's the permission setting page for the site. for list/library, we need access list/library settings page with the steps mentioned above. for an individual file, we can click "…">share>shared with>advanced to access the permission setting page for that file.

Sub SendEmailUsingGmail()

Dim NewMail As CDO.Message

Set NewMail = New CDO.Message

'Enable SSL Authentication
NewMail.Configuration.Fields.Item _
("http://schemas.microsoft.com/cdo/configuration/...;) = True

'Make SMTP authentication Enabled=true (1)

NewMail.Configuration.Fields.Item _
("http://schemas.microsoft.com/cdo/configuration/...;) = 1

'Set the SMTP server and port Details
'To get these details you can get on Settings Page of your Gmail Account

NewMail.Configuration.Fields.Item _
("http://schemas.microsoft.com/cdo/configuration/...;) = "smtp.gmail.com"

NewMail.Configuration.Fields.Item _
("http://schemas.microsoft.com/cdo/configuration/...;) = 25

NewMail.Configuration.Fields.Item _
("http://schemas.microsoft.com/cdo/configuration/...;) = 2

'Set your credentials of your Gmail Account

NewMail.Configuration.Fields.Item _
("http://schemas.microsoft.com/cdo/configuration/...;) = "*** Email address is removed for privacy ***"

NewMail.Configuration.Fields.Item _
("http://schemas.microsoft.com/cdo/configuration/...;) = "*********"

'Update the configuration fields
NewMail.Configuration.Fields.Update

'Set All Email Properties

With NewMail
.Subject = "Test Mail from LearnExcelMacro.com"
.From = "*** Email address is removed for privacy ***"
.To = "*** Email address is removed for privacy ***;*** Email address is removed for privacy ***"
.CC = "*** Email address is removed for privacy ***"
.BCC = ""
.HTMLBody = "Write your complete HTML Page"

' For multiple Attachment you can add below lines as many times

.AddAttachment "C:\ExcelMacro-help.xls"
.AddAttachment "C:\ExcelMacro-help2.xls"
End With

NewMail.Send
MsgBox ("Mail has been Sent")

'Set the NewMail Variable to Nothing
Set NewMail = Nothing

End Sub
