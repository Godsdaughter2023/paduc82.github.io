Formulas will be easier to suggest if we can see the column letters and row numbers. Assuming the Expense Value column is column M and the headers are in row 1, then you can use this Sumif() formula in cell N18, which is the yellow cell. 

=sumif($N$2:$N$16,"Reimbursement",$M$2:$M$16)

captain jesse say:

How do I add contacts from my contact list to an existing group?

How do I add contacts from my contact list to an existing group?

a. user say:

Here is a complete set of How to Manage Groups:

 

How to go to People window in Outlook.com:
Open outlook.com > top left next to the logo Outlook, there is a down arrow > click the arrow > click People.

 
How to create a group :
Go to People window > toolbar at top, click Manage > click Manage groups > click the plus (+) sign, a new box will appear, enter a name for the new group > click Apply > a new group is created.

How to access a group and see the contacts within that group:
In the People window, near top left, click at the word " All v " > a drop down box will appear with a list of your groups > click the one you want > all the contacts within that selected group are now on the left side > scroll to see the whole list.

How to add contacts to the group you have created :

Still in People window > click a contact you want to be added to the group > if you have multiple contacts to add, click the box at top left of the contact > continue clicking the top left box of each contact you want to add to the group until you are done> at the toolbar, click Groups > check the box in front of the group name > click Apply > all the contacts you have selected are now in the group.

Note : make sure you click squarely at the check box. If you click outside the box, all previous boxes you have checked will be nullified, and you’ll have to start all over.

How to delete a contact from a group but keep it in the main contact list:

Go to People > click the contact you want to remove from a group > toolbar, click Groups > un-check the box in front of the group in question > click Apply when done.

How to Edit contacts within a group :

On the People window, top left, click the “ All v “ > you group is on the drop down list > click the group name > now you have all the contacts within that group listed > click one to view > to edit, click at the contact’s name ( not its check box ) > click Edit at the toolbar to do editing......
Note :
1. Edit option will not appear until you have selected a contact.
2. Edit option will not appear if you click at the contact's check box instead of the contact's name

check all that apply - Microsoft Community

tammyscheffers say:

check all that apply

I have a document library in Access 2007 and would like to apply a topic or multiple topics to the documents--some will cover more than one topic.  How can I create a "check all that apply" option?  Thank you.

boblarson say:

With that you normally use a junction table which is set up like this:

 

DocumentID - Long Integer (FK)

TopicID - Long Integer (FK)

 

and then you can use the two as a composite key.  Or you can add an autonumber to the table and use that as the key.  But then you can use a subform to assign the topics to the document.  As long as the main form (for the documents) and the subform (using this junction table) are linked with the Master / Child links set to the DocumentID then you can just have a combo on the subform to select a topic (the DocumentID would be added automatically by Access for each document/topic record in the subform).

 

To check all that apply, you could use a list box set up as multi-select but that would require some code to implement.  It would still use this same junction table but it would require some code to add the records to the table and read them and set the listbox when you navigate to the document record.
