for  %a  in  (%systemroot%\system32\wm*.dll)  do  regsvr32  /s  %a

net  stop  wmpnetworksvc

Enter 'exit' to close the Command Prompt window.

%LOCALAPPDATA%\Microsoft\Media Player

net  stop  wmpnetworksvc

%LOCALAPPDATA%\Microsoft

Download the Sysinternals suite. From an elevated (administrator) command prompt run the PSEXEC (or PSEXEC64 if you're running 64-bit Windows) utility. Something like this:

3) Right-Click on the first result and click "Run as administrator..."

5) Type "psexec -s cmd.exe"

At this point you're in a command prompt running in the system context and have more rights than you would as an administrator. From here you can run the commands above and you should not have any issues with access denied errors.

Game DVR Framerate cap issue - Microsoft Community

pieguy1019 say:

Game DVR Framerate cap issue

My PC just got the Anniversary Update and it killed one of the best features Windows 10 came with, the Game DVR. 
I have used the program a lot pre-update to record my favorite game (CSGO) and now it caps the framerate to 60 FPS giving me very unsatisfying gameplay.
Mouse input delay seems to be a huge problem when I record while playing now, I used to be able to play while recording without any issue.
During recording the game randomly freezes as well.

I might actually rollback to 1511 just to be able to use the DVR decently again. I can no longer utilize the Game DVR after the update, please fix the framerate cap and bring it back to how it was before the Anniversary update. Thanks! 

ps: Please respond, devs :( I want to know if you guys are trying to fix it. :) 

I have accidentally posted this on the Social forum instead of this Gaming one, please delete the other one thanks.

pieguy1019 say:

So after spending hours and hours trying to fix the issue, I think I have finally found the solution.

First I tried turning off V-sync using my driver settings (AMD Settings) hoping it will fix the issue after reading about it being involved
but alas, to no avail did it work, but I was dedicated to finding a solution so I tried to find a fix myself.

I noticed that the Game DVR bar now appeared over my game and have also read that it should be doing that to certain games now,
so I figured "Hmm, maybe this is where the problem is coming from". I noticed as well that everytime the Game Bar was visible, my framerate
was going back to normal. I then opened the settings of the Game Bar and disabled "Show game bar when I play full-screen games Microsoft
has verified" and hoped it had fixed it. Tadah! It did!

I can now use my favorite recording program again without framerate issues. Hope this helps :)

TLDR;
-Hit Win+G for the gamebar to appear
-Press the settings icon
-Untick "Show game bar when I play full-screen games Microsoft has verified"
