Hi

 

Press the WinKey+X to open the System Menu and select Control Panel.

 

Set the View to Icons and select Recovery (you can also use the Search to locate Recovery).

 

When the Recovery component starts, click the Next button.

 

The default is to create the recovery drive using a USB Memory stick. The USB device must be able to hold at least 256MB and cannot contain any other files. Insert the disc and when it is recognized, click the Next button and follow the instructions.

 

Note:

You will also see an option to create a repair disc using a CD or DVD.

 

When you start this procedure, if your computer contains a manufacturers Recovery Partition, you will see an option to copy that partition to the recovery disk. This will require more space on the USB memory device.

a. user say:

Hi Mark,

Thank you for posting your concern on Microsoft Community.

As you have mentioned, you want to locate the service which Windows Firewall has blocked. I’ll certainly assist you with that.

  1. Open Windows Firewall by clicking the Startbutton, and then clicking Control Panel. In the search box, type firewall, and then click Windows Firewall.
  2. In the left pane, click Advanced settings.
  3. In the left pane, click Outbound Rules and check in the list for the services blocked.

You may also refer to the following link for further reference:

http://windows.microsoft.com/en-us/windows/understanding-firewall-settings#1TC=windows-7&section_3
