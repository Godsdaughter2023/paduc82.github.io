john korchok say:

In PowerPoint, the language settings applied to your slide master (View>Master>Slide Master) can influence the language settings of new slides. A conflict with the OS X language settings could also create a problem. When you start a new presentation in a particular language, I'dd suggest the following steps:

  1. In the Finder, set your input language from the flag dropdown at the top right of your Finder menu. If that's not visible, open System Preferences>Language and Region and set your preferred language.
  2. Open PowerPoint and use Tools>Languages and Set As Default to the same preferred language.
  3. Use View>Master>Slide Master. Select all text placeholders on the master (the larger slide at the top of the list of layouts) and use Tools>Language to set them to the same preferred language. Repeat for the slide layouts.

Now the language should stay constant while you are editing.

For existing presentations, set the language for all existing slides by using View>Outline View, then selecting all text in the left-hand pane and applying your preferred language with Tools>Language.
