The "Arrange by" option for libraries is hidden in Windows 8. - Microsoft Community

tzon say:

The "Arrange by" option for libraries is hidden in Windows 8. Make it again easy to use!

File libraries was a very helpful feature introduced by Windows 7 in 2009. With this feature everybody could easily create a virtual folder containing other folders with similar content. 

 

But even more useful than creating a virtual "mega-folder" was the "Arrange by" option within libraries, that worked along with the indexing service and was able to short and group files in virtual sub-folders by selected attribute, such as date, author etc.  

 

For example, someone who has thousands of photos in many subfolders under Pictures could select Arrange by month, and then, all of the picture files would be viewed in virtual folders named by each month and year.

 

Unfortunately, this feature is a little bit harder to find in Windows 8, maybe to promote the modern interface Apps or maybe by mistake: There is no "arrange by" option in the File Explorer ribbon, neither in the "View" tab nor in the "Library tools". The only way to do this is by right clicking and selecting the appropriate context menu, which could be a little bit tricky for users if they a) don't know already about the feature or b) get confused with the option "group by" that sounds similar.

 

Here is a link within the Community of other users having the same trouble with locating the tool.
