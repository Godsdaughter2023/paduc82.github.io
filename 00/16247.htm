tom ogilvy say:

If you actually are going to add an I or an O to the end of you date and time, then the entry will change from a number/date and time to a string

Function CountCcolorDates(range_data As Range, criteria As Range) As Long
Dim datax As Range
Dim xcolor As Long
xcolor = criteria.Interior.ColorIndex
For Each datax In range_data
    If datax.Interior.ColorIndex = xcolor And _
       ((Instr(1,datax.Text,"i", vbTextcompare) > 0) Or (Instr(1,datax.Text, "o", vbTextcompare) > 0)) Then
            CountCcolorDates = CountCcolorDates + 1
    End If
Next datax
End Function

So this would check if there is an I or an O in the cell.
