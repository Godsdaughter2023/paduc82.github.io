paulgailiunas say:

Hi

If you link a linked spread-sheet to a table and the table fields are key fields or have a unique index - then the query is updateable.

I suspect that your zip code field in not a key so the query will not update.

you should link the table and XLS ONLY on the customer ID, assuming it is a KEY field.

 

 

Access is VERY fussy about allowing updates on any linked tables. One side of the link must be a key or Unique indexed field.

 

if - in you access table - the customer number  and zipcode is defined as a combination key then it should work - but I suspect that is not the case.
