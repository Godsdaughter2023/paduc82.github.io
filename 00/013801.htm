jllatham say:

Exactly what do you mean by as a comment?  That when the file is opened later the information on the row would somehow become comments in some cells?

Pretty much by definition a CSV file contains no formatting information at all.

 

As far as I know there are only 2 types of rows in a CSV file, and one of those only appears once.

The first row in a CSV file can either be data like the rest of the rows of information, or it can be 'header/label' information for the following rows of data.

All rows after the first row are going to be considered as data.

As always, someone may show me to be wrong, but that's been my experience with .csv files over the years.
