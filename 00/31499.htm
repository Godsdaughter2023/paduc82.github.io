jannine_p say:

Hello

May I know the Skype version on your computer? Open Skype application > Help > About Skype.

May I know if the sounds is working on a different application? To test if the audio is working, you may use a recorder.

If the audio and sounds is working on different applications, please follow the troubleshooting steps below:

  • Make sure that Skype is no longer running. (Or you may simply press Ctrl Alt Esc, click Skype and then click End Task)
  • Select the Windows logo/Start button, type Device Manager, and select it from the list of results. 
  • Expand one of the categories to find the name of your device, then right-click (or tap and hold) it, and select Update Driver. 
  • Select Search automatically for updated driver software. 
  • If these steps don't work, try reinstalling the driver:  Open Device Manager, right-click (or tap and hold) the name of the device, and select Uninstall.
  • Open and sign in to the Skype application and initiate a call.

If the above information work, please do not hesitate to reply to this post.
