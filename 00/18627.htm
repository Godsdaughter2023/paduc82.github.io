How do I type an umlaut over a letter - Microsoft Community

peterweil say:

How do I type an umlaut over a letter

How do I type an umlaut over a letter
jay freedman say:
On a US keyboard, hold down Ctrl and Shift while you press : (the colon key), release all of them, and then type the letter. This works for the vowels ä, ë, ï, ö, ü, and ÿ  (and their upper case versions) because those characters are present in Unicode fonts.

If you need to put an umlaut over any other letter, type the letter followed immediately by the number 0308, then press Alt+X. The Unicode character (hexadecimal) 0308 is the "combining dieresis", meaning it's automatically positioned over whatever character is immediately to its left.

Also, I have a large number of radio buttons throughout the presentation and these had a white box around them which was a different shade to the background, I have made them transparent in the properties window and this works when viewing the presentation in the edit mode, but when it is in full screen presentation mode it still shows the white box. Why would this be happening? Is there a way to solve it?

david marcovitz say: Is this all text boxes on all slides or just specific text boxes? As long as there is a way for the code to recognize which textboxes to play with, it is very easy to change the size.

john sr wilson say:

If you are using ActivX TextBoxes this code would fix it

Sub fixActivXText()
Dim osld As Slide
Dim oshp As Shape
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
If oshp.Type = msoOLEControlObject Then
If oshp.OLEFormat.ProgID = "Forms.TextBox.1" Then
oshp.OLEFormat.Object.Font.Size = 10
End If
End If
Next oshp
Next osld
End Sub

There's no way to get transparent activX shapes if you use them directly on a slide in show mode
