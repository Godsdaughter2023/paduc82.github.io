georgeogden say:

This sounds like you're possibly another victim of the failed 16288.1 update and is in need of a manual rest of the Windows Update process..... in my case the "NewOS" directory wasn't being created in the update directory C:\$WINDOWS.~BT which may have also corrupted the Windows Update process. 

If you've already tried running the Windows 10 Update Troubleshooter and that hasn't fixed the problem (and it quite often doesn't) then you'll need to take more drastic action and attempt to reset the Windows Update process manually.

A simple method is to copy the text between the ++++ lines and paste it into Notepad

++++++++++++++++++++++++++++++++++++++++++++++++

net stop wuauserv

net stop cryptSvc

net stop bits

net stop msiserver

ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

ren C:\Windows\System32\catroot2 Catroot2.old

net start wuauserv

net start cryptSvc

net start bits

net start msiserver

+++++++++++++++++++++++++++++++++++++++++++++++++

Save as UpdateReset.BAT (note the .BAT and NOT .TXT) to your desktop and close it.

Go to the desktop and right click on the .BAT file and select "Run as Administrator"

That should flush out any corruption in the Windows update files and enable you to move on to download the latest build 16296.0. If you get a message saying that wuauserv can't be stopped then run the Update Troubleshooter and then try the batch file again.

Once done, go back and check for updates and also check to see if the C:\Windows\SoftwareDistribution.old and C:\Windows\System32\Catroot2.old directories were created and, if they exist, delete them in case you need to re-run the batch file again.

Finally, before trying to update again make sure you detach any non-essential peripherals, USB devices and completely remove any 3rd party anti-virus software as they may interfere with the update process,

Hope that helps and good luck.

Set-DistributionGroup -Identity *** Email address is removed for privacy *** –GrantSendOnBehalfTo *** Email address is removed for privacy ***
