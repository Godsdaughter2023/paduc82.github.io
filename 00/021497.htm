bob jones aka: cybertaz say:
For the 1st issue:
Re footnote going to another page is most likely due to 1 or more Section Breaks with the 1st footnote preceding / the 2nd footnote following the Section Break [even though the Section Break may be 'Continuous']. This has historically been a shortcoming of the program regardless of whether you're using a Mac version or a Windows version. The suggested "fix" -- which applies only to single column sections -- is described here, although I've never found it to be particularly effective:

Section break causes an unexpected page break in Word

MVP Suzanne Barnhill offers another workaround which she readily admits is awkward, but (usually) it does work for both single & multiple column sections:


I have a footnote reference in a column, and I want the footnote to span both columns. Is this possible?


For the 2nd issue:
I'm not sure what you're actually wanting to increase/decrease/adjust, but for more space between the Separator & the 1st footnote;
  • You first need to go to Draft View, then select Footnotes from the View menu
  • Select Footnote Separator from the drop-down in the Footnotes Bar
  • Use Format> Paragraph> Space After

Alternatively, you can modify the Footnote Text style if you want to modify space between footnotes, but that can get a little messy.

ken sheridan say: In reply to deleted message
Mea culpa!  I'd used the MAX operator where I should have used the MIN operator.  To return the time difference formatted in hours:minutes first add my TimeElapsed function to your database, and use this:

SELECT T1.*,
TIMEELAPSED(NZ(
               (SELECT  MIN(Report_Time)
                FROM WorkSchedule AS T2
                WHERE T2.ID = T1.ID
                AND T2.Report_Time > T1.Report_Time),Release_Time) - Release_Time,"nn") AS TimeDifference
FROM WorkSchedule AS T1;

To return the time difference in minutes, formatted as in your example with two zeros as decimal places, use this:

SELECT T1.*, FORMAT(NZ(
DATEDIFF("n",T1.Release_Time,
               (SELECT  MIN(Report_Time)
                FROM WorkSchedule AS T2
                WHERE T2.ID = T1.ID
                AND T2.Report_Time > T1.Report_Time)),0),"0.00") AS TimeDifference
FROM WorkSchedule AS T1;
