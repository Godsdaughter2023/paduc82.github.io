ranmo_msft say:

hi vikas,

dirsync server does not support this scenario. this can be deployed with the new synchronization tool azure ad connect. basically, what you need to do is set up two different azure ad connect servers which connect the on-premises ad and two office 365 tenants (azure ad directory)
 
however, please note that each azure ad connect (aadcoonect) server must be configured for filtering part of the ad objects so that each server has a mutually exclusive set of objects to operate on, for example by scoping each server to a particular domain or ou. for how to set filtering in aadconnect, please refer to following article.
https://azure.microsoft.com/en-us/documentation/articles/active-directory-aadconnectsync-configure-filtering/

and since a dns domain can only be registered in one azure ad directory so the upns of the users in the on-premises ad must use separate namespaces as well. for example, in the picture above three separate upn suffixes are registered in the on-premises ad: contoso.com, fabrikam.com, and wingtiptoys.com. the users in each on-premises ad domain use a different namespace.

also, please note that, there will be no gal sync between each office 365 tenants so the address book in exchange online will only show users in the same directory.

rich007a say:

If your "date" in cell P1 is a text string and you want to check that it is valid, e.g.:

'30 June 2012     is a valid date

'31 June 2012     is NOT a valid date

you can use:

Try:

=NOT(ISERR(DATEVALUE(P1)))

DATEVALUE returns a date serial number from a text date.  If it can't it returns an error which is detected by ISERR.  The NOT function then reverses the result so your function returns TRUE if the string CAN be converted to a date.

This can all be shortened by using:

=ISNUMBER(DATEVALUE(P1))

Hope that helps.

Cheers
Rich

mike h.. say:

Hi,

 

You Can't protect/unprotect multiple sheets at the same time and you can't protect a single sheet in a group so you need to capture the selected sheets names, ungroup then and then do the protection.

 

Sub LocKProtectHideSheet()
Dim Pwd As String
Pwd = "password"
Dim wsh As Worksheet
Dim MyString As String, v As Variant
For Each wsh In ActiveWindow.SelectedSheets
       MyString = MyString & "," & wsh.Name
     Next wsh
v = Split(MyString, ",")
ActiveWindow.SelectedSheets(1).Select
For Each wsh In ThisWorkbook.Worksheets
    If Not IsError(Application.Match(wsh.Name, v, 0)) Then
        wsh.Cells.Locked = True
         wsh.Protect Password:=Pwd
         wsh.Visible = False
    End If
Next
 End Sub
