The command syntax should take the form chkdsk c: /f. Also, when the command prompt comes up and chkdsk is invoked, it will default to the disk from which the command is being run, so it's only necessary to specify the volume if it's something other than C:. In other words, if the C: drive is the target, chkdsk /f will suffice.

canadian tech say:

The first question you must get the answer to is, is your computer capable of running Windows 7.

https://www.microsoft.com/en-ca/download/details.aspx?id=20

That will also tell you which drivers you need and which version you can run -- 32 or 64 bit.

Next, you need to take a look at memory, it is likely you will want to increase it.
