Resynchronize your device with the receiver
To resynchronize the mouse or the keyboard with the receiver, follow these steps.
Note When you synchronize the mouse or the keyboard with the receiver, put the device near the receiver and put any other wireless keyboards and mouse at least 6 feet away.

Press the button on the receiver, and then verify that the light on the receiver starts to blink.

Note Some newer Microsoft devices are permanently bound to their receiver and cannot be reset. If your device does not have a button, skip to Method 2.
If a dialog box appears on the screen in several seconds, follow the instructions that appear on the screen. Otherwise, go to the next step.
Press and release the Connect Channel button on the bottom of the mouse or the keyboard.
Check the green light on the receiver. A steady green light means the synchronization was successful. If the problem persists, try steps 1 through 4 again.
For the Wireless Notebook mouse: This issue can occur if the micro-switch on the bottom of the mouse is not released. Use a pin or a pen to try to release the button if it is depressed.

bossdweeb say: +1 - First get rid of McAfee and Norton. Both are notorious for blocking things silently, and bogging machines down with their heavy resource use and constant scanning of this and that.

Then ditch PC Cleaner Pro.  A nasty bit of work that many have trouble getting out of their machine.  If you want to use a 'cleaner' program try  If PC Antivirus come from the same people I would dump that as well.
PC Cleaner Pro removal thread -

Avoid letting any of these utilities do registry  'cleans' though.  Registry cleaning is a throwback to slow machines and small HDD's and RAM, and can do more harm than good.
The main things that need cleaning can be done from Windows Disk Cleanup and with occasional defragmenting (after installing software or games is a good time) of the Hard Drive (SSD's don't need defragmenting).
You can also manually delete the files in the Temp folder in your -

Also go to System Configuration - type  msconfig  into the Start menu search bar or into
the 'Run'  command - and go to the  Startup tab.  Clear the tickbox next to all programs
that don't need to be loaded when you boot into Windows.

You could drop the other 'bloatwares' as well - Glary Utilities and PC Tools Utilities They can be handy but mostly they just do tweaks that you can do manually  from Control Panel apps or places like your Task Bar and Start menu 'Properties'.

AVG , is OK, but MS Security Essentials has been gaining popularity ( I still use Avast), as
is Malwarbytes (Often as a cross reference/backup and not running all the time).
The new wave threat is rootkits . Rootkits are difficult (impossible ? ) for AV's to detect, so
you need something like BitDefender Rescue Disk  for rootkit detection. This is run from a
DVD/CD or bootable flash drive.
Sophos Anti-rootkit will run from Windows, but is not as effective as BitDefender.
Note that there are legitimate programs that run in the same way as rootkits (eg. -SecureRom ). 

I currently have Avast running in resident, and have Spybot Search and Destroy and Malwarebytes installed to use for cross reference scans. I've had no issues with these three installed at the same time.
