laura von harten say:

Complete your profile

Your profile includes your geographic location. The location you enter determines your time zone. This is how it worked for me on a laptop running Windows 10:

  • Open a web browser.
  • Log on to the Skype website (not the Skype app).
  • Look in upper left corner. It should say something like "Your profile is 50% complete."
  • If the number is less than 90% it means you need to enter geographic information.
  • Go to your profile page. Click edit.
  • Complete the fields for city, state and country.
  • Click save.

I haven't reached 100% completeness, probably because I haven't entered another phone number yet. But at least I can tell what time a call or message came through.

msob say:

Formating a Bullet List - How can I add more than one line between bullets?

When formatting a bullet list I would like to format the information so that the line spacing for each bulleted entry is set at one (1) and the spacing between bulleted entries is set at one and a half (1 1/2).

 

Can I do that?  Know How?  Like to help?

 

Thanks!
marysully say:

At the first bullet and while on the Home tab click the little box next to the right of Paragraph to open the Indents and Spacing window. The Line Spacing should be set to Single. Under Spacing, adjust the Spacing After: (to approx. 6?). Uncheck the box under Spacing that reads "Don't add space between paragraphs of the same style." Your bullets should now be single spaced with 1.5 spaces between bullets.

yves dhondt say:

Select the paragraphs in the bulleted list. Then in the "Home" tab, click the little arrow next to the "Paragraph" group name. Set "spacing after" to a value you want and ensure that "Don't add space between paragraphs of the same style" is NOT checked.
