Paste is in different font size - Microsoft Community

erselking say:

Paste is in different font size

When I copy from text document A, which is in 14 font, and paste into text document B, which is also in 14 font, the pasted text always appears as 12 font. Both documents are in the same font type style. 
john mcghie say:
When you paste the same-named style into another document, the pasted text "should" adopt the formatting of the target document.

So if you have the style Body Text set to 12 points in the target, and 14 points in the source, you will get 12.

If you have changed the font size, but not the style, in the target document, the incoming text will be set to the formatting of the style, not the surrounding text.

To avoid these issues, do all your formatting with styles.  Set up the styles in your document to be as you would like them (and check the "Add to template" box in "Format>Style..." so your changes are preserved for future documents.

That way, you will get no more surprises.

Is it possible Visio shape to show more detail as you zoom in - Microsoft Community

cee.hyacinth say:

Is it possible Visio shape to show more detail as you zoom in

Is it possible in Visio shape to show more detail as you zoom in. Eg terminal details.  Is it possible in Visio shape to show more detail as you zoom in. Eg terminal details.  It is an AutoCAD function that I would love to see in Visio.

 Thanks

david j parker say:
  1. The Zoom property is a member of Window in Visio.
  2. A Window has a ViewChanged event when the Scroll or Zoom changes.
  3. A Window has a Page property
  4. Shapes on a Page can be programmed using the ShapeSheet to display different geometry/text depending on a value in a Page.PageSheet

 

So, my conclusion is that it can be done provided that you have an add-in responding to the ViewChanged event so that it updates a cell in the PageSheet, say User.ZoomLevel. Then you would need to have formulas in your shapes that show/hide geometry/text in accordance with the Page!User.ZoomLevel value.

 

(In fact we did do this a few years ago with Visio for a wargaming solution)
