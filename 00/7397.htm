BTW I am using Word 2010 and Windows 7.

doug robbins - mvp office apps & services (word) say:

If you have all of the checkboxes in a table (so that you can refer to the checkboxes in the .Range of the table), if you set the following macro to run on exit from each of the checkboxes, it will display a message if more that three of the checkboxes are selected:

 

Dim i As Long, j As Long
j = 0
With Selection.Tables(1).Range
    For i = 1 To .FormFields.Count
        If .FormFields(i).CheckBox.Value = True Then
            j = j + 1
        End If
    Next i
End With
If j > 3 Then
    MsgBox "You have exceeded the maximum number(3) of allowable number of items.  If you wish to select the current item, please cancel one of your previous selections."
    Selection.FormFields(1).CheckBox.Value = False
End If

With appropriate modification fo the code, you could use the .Range of some other object if it contained all of the checkboxes.

 

It will work best if you tab out of each checkbox field rather that using the mouse which would cause the next (5th) checkbox to be checked, though the code will have unchecked the fourth one.

Dynamic cell address

I want to sum data in column A.  For example sum(a1:a3).  However the row numbers (1 to 3) are dynamic and entered into other cells.  Again, let's assume cell F1 contains the starting row and cell F2 contains the ending row.  How do I resolve the contents of cell F1 (e.g., 1) and F2 (e.g., 3) into the sum function?

mike h.. say:

What is the format if A is on another sheet?  'sheet name'! is not working inside indirect.

Thanks,

Hi,

It should work just fine, try this

=SUM(INDIRECT("Sheet1!A" & $F$1 & ":A" & $F$2))

Or if the sheet name is also in a cells try this with the sheet name in F3

=SUM(INDIRECT("'" & F3 &"'!A" & F1 &":A" & F2))

jllatham say:

In addition to what Mike H. offered, if the sheet name contains spaces and you want to just type it in, then do it like this:

=SUM(INDIRECT("'My Data Sheet'!A" & $F$1 & ":A" & $F$2))

Notice that there is a ' right in front of My and just before the ! - not needed if the sheet's name doesn't have spaces, but doesn't hurt to put them in even then, Excel will just ignore them.
