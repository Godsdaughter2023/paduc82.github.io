bill manville say:

As Tushar and I have both said, the Plot Area can have only one interior fill setting - usually one colour.

You could have a 2-colour fill pattern going from red at 0 to green at 100, but it would look rather nasty and would not have boundaries where you want them.

You could add 3 series to the chart as a stacked column chart group with no gap between the bars and no borders on the series, coloured red, yellow and green.  But they would obscure the gridlines.

Finally, you could place coloured semi-transparent rectangles over the chart, through which you would be able to see the gridlines.  I think the last is probably the most practical solution.

If you want help here on any of the above, say which. 

If you want to email me via the contact page of my website ( www.manville.org.uk ), referencing this post and saying which you want I could send you an example.

bill manville say:

 It is not clear to me if you intended to have different colours in different areas, depending on the values on the line.
The plot area can only be one colour.
It can't be set conditionally in the way that a cell background can.
I guess you could make it transparent and carefully position it over cells that you use conditional formatting on.
But simpler would be a macro that would set it, let's say the colour depends on the last value on the first series and you want to do it to each chart in the workbook.

Sub ColourCharts()
  Dim WS As Worksheet
  Dim CO As ChartObject
  For Each WS In ActiveWorkbook.Worksheets
    For Each CO In WS.ChartObjects
      With CO.Chart
        Select Case .SeriesCollection(1).Values()(.SeriesCollection(1).Points.Count)
          Case Is >= 0.8: .PlotArea.Interior.ColorIndex = 4
          Case Is >= 0.65: .PlotArea.Interior.ColorIndex = 45
          Case Else: .PlotArea.Interior.ColorIndex = 3
        End Select
      End With
    Next
  Next
End Sub

You could call that procedure from an appropriate event handler - clicking a button, calculating the worksheet or entering data into certain cells on the worksheet.  Come back if you need more help with that and we can explain how to do it
