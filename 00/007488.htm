You need to set odd and even page headers in page setup > layout. Then manually insert the appropriate watermarks in the headers of the odd and even pages as required. don varnau say:

See if these 2 procedures help:

1. Clear the cache.

Internet Options> General tab> Browsing History> Delete> uncheck "Preserve Favorites website data" then delete Temporary Internet Files. No reason, yet, to delete cookies, history,  passwords, etc.

2. Clear the DNS Cache.

For Windows 7...
From Start> All programs> Accessories, right-click Command Prompt and choose "Run as administrator". At the flashing cursor type in ipconfig /flushdns (note the space between config and / ) hit [Enter] then Restart.
