The term backend refers to a database that is split. All the tables are stored in a back end that is stored on a shared network drive. Everything else is in a front end stored on each local user's PC. This is an essential configuration for a multi-user database. If you do not have such a configuration then you are inviting corruption. 

If the most recent copy is 3 months old, then your back up system needs to be changed. You should be backing up the back end on a regular basis and have copies of at least the current and previous versions of the front end. Anything less is courting disaster.
