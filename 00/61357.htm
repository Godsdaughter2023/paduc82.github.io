slance310 say:

Hi,

A couple of things to check...

Is the SSD "Initialized" properly ?

If you are running UEFI Bios on your machine, the SSD needs to be "Initialized" as a GPT Type drive...

If you are running the older Legacy Bios it needs to be "Initialized" as a MBR type drive..

This is something that needs to be done with nothing on the drive... There is a notice that pops up in Disk Management when a blank drive is hooked up to make this choice... This is something that needs to be done before it is "Formatted"..

If you are using UEFI Bios, you will not select the SSD as a Boot Device like you would in "Legacy" Bios... UEFI Bios requires the Boot Device to boot into windows is "WindowsBootManager"...

This could account for why your cloning didn't work as well..

Step 1: Follow the steps provided below to uninstall the video card drivers:
