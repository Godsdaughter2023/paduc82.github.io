My sister's computer ended up figuring itself out after a few reboots. For those still having problems, try the solution below as suggested by Tahir. It worked on my Lenovo Yoga. 

  1. Click on the Start button.
    • If you're using Windows 8, click on the down arrow icon in the lower left corner to go on to the next page.
  2. Select Control Panel.
  3. Click Sound.
    • If you do not see Sound, first click Hardware and Sound, then click Sound.
  4. On the Playback tab, click the Default Device (indicated by a green circle with a check mark inside).
  5. After selecting the Default Device, click on the Properties button.
  6. In the new window that appears, click on the Enhancements tab.
  7. Make sure that the box for Disable all enhancements or Disable all sound effects is checked.

    Let’s try these steps to cancel or delete a Stuck Print Job in the Print Queue.

     

    1. Press Windows Key + R to open the Run prompt.

    2. Type services.msc and hit enter.

    3. Find Print Spooler in the list, and click the Stop button.

    4. Now open up C:\drive, click on Windows folder> System32>spool>PRINTERS

    Note: Click ‘Yes’ if it prompts for permission.

    5. Now you’ll see a bunch of files in the folder, delete all of them. Don’t delete the PRINTERS folder itself, just the files.

    6. Now go back to Services and click the Start button to start up the service again.

    7. At this point your stuck print job should be gone.
