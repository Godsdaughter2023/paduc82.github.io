ken blake say:
Although by default the task bar appears on the bottom of the screen, it can be on any of the sides you prefer it on. To move it from any side to another, simply click on an unused part of it and drag it where you want it. Be sure to click *within* it, not on the edge; clicking on the edge and dragging will resize it, not move it.

If it won't move, it may be locked. In that case, right click on it and uncheck "Lock the taskbar," then try again.

By the way, with today's wide-screen monitors, I prefer the task bar on the left or right side of the screen. I think that makes a better use of screen real estate than having it at the bottom.

ambition5253 say:

Windows Defrag Showing HDD as SSD

I checked the Windows Defrag application today and noticed the following issues:

1. My SSD (OS drive) is coming up as an SSD yet shows that it is being optimized - I was under the assumption that windows would not optimize/defragment an SSD.

2. My Data drive (512GB HDD Toshiba) is also showing up as an SSD but says optimization is not available - This is definitely not an SSD and I would imagine defragmenting should be allowed.

I did some reading and the two main possible fixes I found were to set the "Optimize Drives" service from manual to automatic (delayed start), or to run winstat formal from an elevated command prompt.  I modified the optimize drives service to start automatically to no avail and have yet to try the winstat command.

Does anyone have any ideas why this is happening or if something has been corrupted to cause this?

Thank you!

ambition5253 say:

Fixed!

Like I said in my original post, I set the "Optimize Drives" service to start automatic (delayed start) from manual.  Although that did not fix the problem, I do believe this should be done anyway so that the service can run the optimization when the time comes.  Otherwise, it seems it will not run unless you open the defragment tool yourself.

What I believe ultimately fixed the problem was opening an elevated (admin) command prompt and running "winsat formal".  This re runs the tests that windows uses to determine system performance hence the name "System Assessment Tool".  Anyway once I ran that and restarted my system, the defragment tool showed my HDD as an HDD and optimization was once again available. 

Hope this helps someone else!
