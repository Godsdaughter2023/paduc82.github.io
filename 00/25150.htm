There is a big difference between the two symptoms.   One may only mean that the site is being slow, so you could try being patient with it or somehow try giving it less to do.  The other is a euphemism for "has crashed", so you could try diagnosing that by looking at such things as crash location, error code, stack back trace (or failing that, list of associated modules which were loaded at the time of the crash.)  All are details which used to be provided to us automatically via drwtsn32.log but now must be extracted from various other diagnostics such as (principally) a dump.

what damage can a malware infected word document do to a system? - Microsoft Community

steve richter say:

what damage can a malware infected word document do to a system?

You do a scratch install on a PC to get rid of a malware. Then you install microsoft security essentials to protect against malware ever getting on your system again. Last step is to reload files and documents from the system which had been hacked. Would code embedded in word documents by the evil hacker be able to hack the PC again?  Can malware embedded in word and excel documents be detected?

thanks,

 

yves dhondt say:
You do a scratch install on a PC to get rid of a malware.

Unless the virus got somehow embedded on your bios (unlikely but possible). In that case, an install from scratch won't help much.

Would code embedded in word documents by the evil hacker be able to hack the PC again?

If you execute macros in documents you got from others, there is always a risk that those macros will do something damaging to your system. Only run macros if you know their origin, or if you checked their source yourself.

Can malware embedded in word and excel documents be detected?

Yes ... and no. Most anti-malware and anti-virus software can detect a lot of macro viruses. Throwing in heuristic search patterns, a lot of 'future' viruses will also be detected, but some will always slip through. After all, it is not possible to defend yourself against everything which is not known yet. It's the same with medical vaccines. We have pretty good flu vaccines, but every once in a while, a totally 'new' strain like H1N1 shows up, and new vaccines for such a virus can only be made after the virus has been spotted somewhere.

On top of the above, there can always be security issues in Word or Excel which can cause specially crafted documents to be executed automatically. Again, patches will only become available after the issue has been spotted (in the wild).

Personally, as long as you don't run macros from unknown sources and keep your anti-virus software up-to-date, I don't think you have to worry about the subject.

Can't download a current Windows 10 update - Microsoft Community

donprince say:

Can't download a current Windows 10 update

I am a former Windows 10 tester and after my hard drive with Windows 10 on it went bad I have been unable to download the latest Windows 10 to install on my new drive as a dual boot  with Windows 7 on my other computer drive. Am I out of luck as far as getting another W10 update to burn to a DVD so I can load it as a dual boot with my Windows 7?

don cuthbert say:

No.  You can create official Windows 10 install media and install that.  Was the last Windows 10 version you had activated?  Which version was it?  If you had had activated (e.g. Pro or Home).

Shrink your C: drive by 40GB to make some room for Windows 10.  Boot with the official ISO on install media, such as a USB flash drive and use Custom to install it to the 40GB un-partitioned space.  Skip past any screens asking for a product key.  If your device is still registered as a licensed Windows 10 device, Windows 10 may auto-activate.

If it does not auto-activate, you can install another copy of Windows 7 to that new partition instead and activate it with your Windows 7 product key.  Then use the matching Edition of the Windows 10 ISO to upgrade that partition by running setup.exe, choosing to keep Windows settings and files and not entering any product key during the upgrade.  It will auto-activate after this upgrade.  Now you are back to a dual boot setup and can re-join the Insider Program from within Windows Update on the Windows 10 partition if you wish.

http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install

donprince say: I had the latest Windows 10 Pro 32 bit activated but now since my drive is clean I want to activate the same version but 64 bit.  I tried to do so downloading it yesterday but when I ran the DVD I made it didn't like my Windows 7 key or the key that originally came with the test versions of Windows 10. Does anyone have any idea what I can do now? Thanks for any ideas any of you may have. I really want to try Windows 10 on my new solid state hard drive.

don cuthbert say:

You can't use your Windows 7 or 10 product keys to activate Windows 10.  The 7 key should activate Windows 7 64-bit or 32-bit.  The generic Windows 10 key will not activate anything, because the free Windows 10 upgrade never uses any product key for activation.

Because you have said that you had the latest Windows 10 32-bit activated, you should be able to clean install Windows 10 64-bit to your new SSD. 

If your computer is UEFI-capable it would not have had full UEFI settings configured with Windows 7, but you ought to enable this with Windows 10 (if you can).  Windows 7 uses some compatibility settings if UEFI is enabled, which Windows 10 does not need.  Secure boot requires UEFI without these compatibility settings.

So if UEFI, fully enable that before clean installing Windows 10.  The SSD will be initialized as a GPT disk, which is more robust than the Legacy MBR initialization required by Windows 7.
