Can I use RegEx on word? - Microsoft Community

yamaping say:

Can I use RegEx on word?

as title, please teach me how to use, thank you. 
jay freedman say:
Although the "use wildcards" option in Word's Find dialog does a lot of what RegEx does (see http://www.gmayor.com/replace_using_wildcards.htm for details), it is not the same as RegEx.

The real RegEx is built into the VBScript engine, which is available in all recent Windows versions. To use it in a Word macro, you open the macro editor, click Tools > References, and check the box for "Microsoft VBScript Regular Expressions 5.5" or whatever is the latest version listed. Then you can declare a variable in a macro as an object of type RegExp, and use its properties (.Pattern, .IgnoreCase, etc.) and methods (.Execute, .Replace, etc.).

For an introduction, read http://bytecomb.com/regular-expressions-in-vba.

steve rindsberg say:

As John's mentioned, you need to click on something outside the ActiveX text box to return to the usual slideshow focus.

This won't be obvious to the casual user, so it'll help if you give them a cue of some sort.

For example, you can add a shape next to the ActiveX text box, have it say "OK" and give it a Hyperlink to Slide action setting (hyperlinking to the same slide as the shape and ActiveX textbox are on).

Result: user types some text, clicks OK and the focus returns to the slide show but doesn't automatically advance (because the OK box's action setting "absorbed" the click)
