Actually the Command button wizard will do this partially for you. Use the wizard and create a button to print a report. This button will print all reports via a macro. Then make a copy og the button and in the Macro editor add a WHERE clause to the macro that would be like this:

"[primarykey] = " & Forms!formname!controlname

where primary key is the name of the PK field in your table, formname is the name of the form and controlname the name of the control bound to the PK field.
