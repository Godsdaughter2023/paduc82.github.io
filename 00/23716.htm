infernet calls

aspergerguy say:

@Tpanchal

 

You need SIP VoIP Settings for Symbian S^3 devices from:http://www.developer.nokia.com/info/sw.nokia.com/id/d476061e-90ca-42e9-b3ea-1a852f3808ec/SIP_VoIP_Settings.html

 

1) Install the above

2) Create a SIP profile by going to Menu > Settings > Connectivity > Admin Settings > SIP Settings
     Option > New SIP Profile

3) Create the VoIP service using that SIP profile
Admin Settings > Net settings > Advance VoIP Settings > Create new service

You should then see a net call icon at the top of the screen when you open your contacts.
