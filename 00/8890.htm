rick rothstein, mvp - excel say:

You show two spaces between the day number and the year number, but I am assuming that is a typo. If that assumption is correct, then you can use this nice and short formula to get your real Excel date...

=--REPLACE(LEFT(A1,11),7,1,", ")

If those two spaces are in fact correct, then you can use this formula for that situation...

=--REPLACE(LEFT(A1,12),7,1,",")

These formulas return a real Excel date which you can format to look anyway you want.

shane devenshire say:

What program are you in?  Assume Excel, assume Actuals in column A, Budget in column B

In 2003:
1. Select the cells you want to format say A1:A10
2. Choose Format, Conditional Formatting
3. Choose Formula is from the first drop down
4. In the second box enter the formula:

=$A1>.9*$B1

5. Click the Format button
6. Choose a color on the Patterns tab (or any available option)
7. Click OK twice.

In 2007:
1. Highlight all the cells on the rows you want formatted
2. Choose Home, Conditional Formatting, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following formula:

=$A1>.9*$B1

5. Click the Format button and choose a format.
6. Click OK twice
