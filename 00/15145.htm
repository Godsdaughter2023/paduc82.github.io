I think you need a UDF. Open the visual basic editor with <ALT><F11>.

Insert module and copy/paste this code:

 

Function MyLarge(MyRange, k)
    Dim cll As Range, col As New Collection, rng() As Variant, i As Integer
    On Error Resume Next
    With col
        For Each cll In MyRange
            .Add Item:=cll, Key:=CStr(cll)
        Next cll
    End With
    On Error GoTo 0
    ReDim rng(1 To col.Count)
    For i = 1 To col.Count
        rng(i) = col.Item(i)
    Next i
    MyLarge = Application.WorksheetFunction.Large(rng, k)
End Function

What is drag-lock and how do I turn it off?


An option in Mouse Properties which typically becomes active after dragging for a set amount of time.  Uncheck it if it's on and you don't want to have it.  I have also had it defined as a non-toggling button on my Microsoft wireless mouse.  In either case it can be annoying until you get used to dealing with it. E.g. it is cancelled by using another click which seems unnatural.  (Sort of like trying to get rid of sticky tape by trying to shake your hand and then having to stop to remove it.)  If you use Mousekeys it is activated by pressing 0/Ins and inactivated by pressing ./Del.  See Mousekeys help.   Actually for some reason I always get those two backwards.  Maybe by having to write it down and explaining I will be able to remember the next time.  Otherwise I'll just continue to experiment with one or the other until the desired effect happens.  The good news is that it can be used there to cancel a Drag-lock caused by other means, such as the other two that I mentioned.


http://windows.microsoft.com/en-US/windows7/Use-Mouse-Keys-to-move-the-mouse-pointer

How would I synchronize "multiple pointer handlers"?


I use Mousekeys.  Press and hold Alt+Shift+NumLock to see it.  After it's enabled just toggle NumLock appropriately.  E.g. if it looked as if something was holding the Cursor down I would just press the 8 key to reverse the direction.  Note that that is the Numeric keypad 8 and unfortunately we cannot use the one in any Touch keyboard.  However, most laptops have a mode in which a set of standard keyboard keys gets mapped to be used as a substitute for the Numeric keypad.  See your User guide to check out that possibility (if you don't have a real Numeric keypad.)
