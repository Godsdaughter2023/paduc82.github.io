Hidden files are hidden for good reason:  you should not delete them unless you know what the files are.

    C:\Users\AppData\Roaming\Apple Computer\MobileSync\Backup\

When you run TreeSize Free, make sure to run it "as Administrator"

  1. Open iTunes
  2. Click Edit > Preferences
  3. Select "Devices"
  4. Select the backup(s) you want to delete and press the "Delete" button

jessen p say:

Hi Bonniep,

1) Who is the manufacturer and what is the model of the printer?

The printer displays offline when the printer is not connected to the network or is turned off or has gone offline.

Verify that the printer is running and that all cables are secure. There might be a button to push to bring the printer online.

http://technet.microsoft.com/en-us/library/cc938175.aspx

For more information visit http://windows.microsoft.com/en-us/windows7/Open-the-Printer-troubleshooter

Set your printer as the default printer and see if it resets it to online status.

Method 1: To select a default printer

1. Open Devices and Printers by clicking the Start button.

2. Right-click the printer you want to use, and then click Set as default printer. (You'll see a check mark on the printer's icon signifying that it's now your default printer.)

For more information visit the following link:

http://windows.microsoft.com/en-us/windows7/Change-your-default-printer

Next thing to check is if your printer is set to sleep, if the printer is set to sleep, I would appear offline.

Method 2:

a. Open Device Manager by clicking the Start button, clicking Control Panel, clicking System and Maintenance, and then clicking Device Manager.‌ If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

b. In the list of device categories, expand Ports.

c. Right-click the entry for the printer, and then click Properties.

d. In the printer Properties dialog box, click the Power Management tab, Uncheck the Allow this computer to turn off this device to save power check box, and then click OK.

Method 3: If the above method does not help, check for the following setting:

1. Open the Start menu and click on “Devices and Printers”. This will open a window with a list of the printers currently set up on your computer. Now click on the printer that you have under the "Printers and Faxes" icon.

2. Double-click on the icon of the printer you want to change to online. A pop-up window detailing all current print jobs will open.

3. Go to "Printer" in the menu bar of the pop-up window and uncheck "Use Printer Offline". This action will change the printer from offline to online.

There are different methods to access safe mode options. Details are mentioned below.

•      If hitting F8 doesn’t work, and you have media for Windows 8.1 (such as a DVD or USB), you can boot from media using the instructions here. Further instructions about Safe Mode can be found here.

•      Press ‘Windows Key + R’

•      In the run window type ‘rstrui.exe’ and click on OK

•      Click on next in the System Restore windows

•      Now choose the restore point of your choice, and try to restore your computer back to earlier state.


System Restore disclaimer: When you use System Restore to restore the computer to a previous state, programs and updates that you installed are removed.
