nunof32 say: Thanks for the suggestions.

Actually, I have been able to figure out how to work around the malfunction :

When you do Insert>Hyperlink>Document>Select and you pick a document from the selection dialog box, Word inserts the name of the document in the "Link To" box.

• sometimes (?!) Word inserts the full path of the document : file://localhost/Users/ ... / <nameOfDocument>;
• sometimes (?!) Word inserts just <nameOfDocument>.

(As I am always selecting a document that resides on my working directory, this behaviour really seems erratic)

When Word inserts just <nameOfDocument> all works fine (i.e. the hyperlink works and you can even specify a bookmark within the target document) PROVIDED <nameOfDocument> DOES NOT CONTAIN SPECIAL CHARACTERS PRECEDED BY A SPACE.

If <nameOfDocument> contains special characters preceded by a space, you have to specify the full path to the document. Sometimes (?!) Word does it; but if it has been lazy and didn't do it you have to manually add "file://localhost/Users/..." before <nameOfDocument>. And then all is ok.

My suggestion if you encounter this problem : make sure that Word has inserted the full path to the target document; if it has not, add manually the full path : file://localhost/Users/...

I do not fully understand all the implications of this workaround, so be carefull. For me, it has solved my issue.

    andreepin say:

    Hi Lewis,

    You are seeing two green balls if you added a custom group or command when you customized the ribbon but did not assigned an icon to represent that custom group of command. We suggest that you add an icon to the other custom group or command that you created. Kindly follow the steps provided below:

    Add an icon to represent the custom group or command

    1. Click the File tab.

    2. Under Help, click Options.

    3. Click Customize Ribbon.

    4. In the Customize the Ribbon window, under the Customize the Ribbon list, click the custom group or command that you added.

    5. Click Rename, and then in the Symbol list, click an icon.

    6. In the Rename dialog box, click OK.

    7. To see and save your customizations, click OK.

    For more information about how to customize the ribbon, see Customize the ribbon or Video: Customize the ribbon.

    I hope this information is helpful. If you have additional questions, feel free to post on this thread.
