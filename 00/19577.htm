I'm running v16.8 and icons still not available for the reject and accept change functions.

So we ended up going with a variation on option 1 ie. used VBA code with custom icons (but called builtin macros and so didn't need to use CommandBars). Here's the VBA:

Sub DO_Accept_Change(control As IRibbonControl)

    On Error Resume Next
    Application.Run ("ToolsRevisionMarksAccept")
    On Error GoTo 0

End Sub

Sub DO_Reject_Change(control As IRibbonControl)

    On Error Resume Next
    Application.Run ("ToolsRevisionMarksReject")
    On Error GoTo 0

End Sub
