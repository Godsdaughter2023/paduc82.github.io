setting message flags? - Microsoft Community

new to windows live email on windows 7 say:

setting message flags?

What does the message "setting message flags" mean? It comes on my email screen after I delete messages.
¡firedog say:

Whenever you perform any action on a message - e.g. read it, reply to it, forward it, delete it - the message database sets a flag against its record of the message to register what you've done to it. This is normally instantaneous, but if there are many messages involved, it might take more than a few seconds and you see the progress pop-up to tell you. For an IMAP account (like Gmail, for example), it will always take longer because those flags also have to be set at the server. xb1r athena say:

Restore Factory Defaults

Login Information

  • Default Gateway - 192.168.0.1
  • Login Username - Select admin in the dropdown
  • Login Password - leave blank by default

Restore Factory Defaults

Through firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser and in the address bar enter 192.168.0.1 and enter admin for the username and leave the password blank when prompted.
  3. Select Tools from the top menu.
  4. Select System from the menu on the left.
  5. Click the Restore Factory Defaults button.

Through Hardware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Locate the reset button on the device.
  3. With paperclip/pen/toothpick hold down the reset button down for 30 seconds.

a. user say:

This will not help you remove the ThinkPoint affliction, but should get your Task manger running:

Some malicious software is aware of the methods you are most likely to use to find and remove it (Task Manager, System Restore, Command Prompt, the Internet, Google, etc.)

The malicious software does not want you to use things like Task Manager to find and remove it, so it prevents them from becoming a running process on the system and the process will not launch.

The malicious software is happiest when it fools you into giving up on fixing your system and making you think you have to use System Restore, a Repair Install or just reinstall your XP from scratch to fix your system when none of those things are necessary.

You can outsmart the malicious software (then fix it).

Using Explorer, navigate to the c:\windows\system32 folder and locate taskmgr.exe.

Copy the taskmgr.exe file and rename the copy something that the malicious software will not recognize.  I like to call the copy:

c:\windows\system32\jose.exe

Double click jose.exe and see if the Windows Task Manager window will open.

The problem is the malicious software will not let the taskmgr.exe be a running process, but it does not know about jose.exe as a running process (it might someday though), so you just need to outsmart it.
