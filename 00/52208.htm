shafeeq_khan say:

Hi Tim,

 

Thank you for writing to Microsoft Community Forums.

 

Unfortunately, there are no metadata facilities for WAV files in Windows and it is by design that you cannot use Windows 10 nor Windows Media Player to edit WAV metadata.  Other tools will do this for you, but generally editing WAV metadata is prone to failure and thus the current Microsoft tools avoid doing that.

 

However, you can edit metadata of WAV files in Windows Media Players library, but none of the changes will be written to the file. This in turn can lead to some strange issues, so using WAV is generally not recommended.

 

If you're going to rip to WAV using Windows Media Player, be sure to set the metadata correctly before ripping.

 

Hope it helps.

james-l83 say:

Hi all. Just replying to my post haha :-)

I found the solution myself: Force the PC to power off (hold the power button down),

Have your system image disc to hand. Boot up the PC and insert the disc.

In the recovery options, use the arrow keys and navigate to 'boot in safe mode'.

The OS will roll back any applied updates and restore defaults to their previous state.

Once in Windows 7 (in safe mode) click 'start', then right click on 'computer', in the left pane click 'Advanced system settings'. The Advanced tab should appear first, click 'settings' under the performance section. In the performance options dialogue box click on the 'Advanced' tab and under Virtual memory click 'change'. See what the recommended value is, if you cannot see it, uncheck the box at the top (Automatically manage paging file size for all drives). Look down the bottom under - Total paging file size for all drives - and see what it recommends. Mine was 2111 MB but it recommended 3016. Therefore, I clicked 'custom size' and set the initial size to 3016 MB and the maximum size to 20000 MB. Click 'OK' then click 'Apply'. Now reboot and eject the boot disc. Once the OS boots into Win 7, click 'start' 'all programs' and click 'Windows update'. The system should carry out the update now, which may require several restarts.

Regards.
