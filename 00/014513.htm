jay freedman say:

The nonbreaking hyphen is never represented in the .Text property of a range. I'm not sure why this is, but it just isn't there. That makes copy/paste of the string unusable.

Instead of using copy/paste, assign the .FormattedText property of the source range to the .FormattedText property of the destination range. This doesn't use the clipboard at all. There is also no reason to "select" anything; use Range objects as shown here:

Sub Sample()
  Dim rngSource As Range, rngDest As Range
  Dim docSource As Document, docDest As Document
  Dim tblDest As Table
  
  Set docSource = ActiveDocument
  Set docDest = Documents.Add
  Set tblDest = docDest.Tables.Add(Range:=docDest.Range, _
    numrows:=2, numcolumns:=2)
  ' set up the repeating heading row
  With tblDest.Rows(1)
    .HeadingFormat = True
    .Cells(1).Range.Text = "Text"
    .Cells(2).Range.Text = "Page"
  End With
  
  Set rngSource = docSource.Range
  With rngSource.Find
    .Format = True
    .Style = docSource.Styles("Heading 3")
    .Wrap = wdFindStop
    While .Execute
      ' when .Execute returns True, rngSource has been
      ' reset to cover the found text
      Set rngDest = tblDest.Rows.Last.Cells(1).Range
      rngDest.FormattedText = rngSource.FormattedText
      tblDest.Rows.Last.Cells(2).Range.Text = _
        rngSource.Information(wdActiveEndPageNumber)
      tblDest.Rows.Add
      ' move rngSource to the end of the current location
      ' to continue searching to the end of the document
      rngSource.Collapse wdCollapseEnd
    Wend
    ' remove the empty last row
    tblDest.Rows.Last.Delete
  End With
End Sub

jay freedman say:

It also occurs to me that you may be able to do the job without VBA. Instead, make a table of contents that includes only the single style you were searching for. 

To do that:

  • Click References > Table of Contents > Custom Table of Contents.
  • In the dialog, click the Options button.
  • By default, Heading 1, Heading 2, and Heading 3 have check marks and entries in the TOC level column. Delete all the entries in the TOC level column, and those check marks will disappear.
  • Scroll the list to find the style you used for your information, and type the number 1 in the TOC level box for that style.
  • Click OK in each dialog.

There are other tricks you can play with a table of contents -- see http://www.word.mvps.org/FAQs/Formatting/TOCSwitches.htm.
