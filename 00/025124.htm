The following is a similar macro for InlineShape objects:

Sub ExampleResizeInlineShapes()
 'Example macro created by Stefan Blom, MVP, July 2017
 Dim s As InlineShape
 For Each s In ActiveDocument.InlineShapes
 s.LockAspectRatio = True
 s.Width = InchesToPoints(2)  'specify the value you want
 Next s
End Sub

stefan blom say:

You can use the following macro:

Sub ExampleResizeMacro()
'Example macro created by Stefan Blom, MVP, July 2017
Dim s As Shape
For Each s In ActiveDocument.Shapes
s.LockAspectRatio = True
s.Width = InchesToPoints(2)  'specify the value you want
Next s
End Sub

The macro changes all Shapes (objects whose Text Wrapping is NOT "In line with text") so that the width is 2 inches. The aspect ratio is preserved.
