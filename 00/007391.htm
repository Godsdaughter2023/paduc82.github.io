robert aldwinckle on forums say:
My Question: Is it actually possible to completely remove Internet Explorer from my machine


No but you are asking the wrong question, thinking that Internet Explorer is just a program.   IE as a stub program iexplore.exe which can be used as an Internet browser.  It and its shortcuts can be hidden--by unchecking Internet Explorer in Windows Features.   Underneath that is the OS browsing functionality at the level that is shown for the Internet Explorer feature.   That cannot be removed and would continue to be updated, including updated to a new level, unless you deliberately inhibit the relevant OS updates.

 

In Vista your base browsing functionality once may have been the level known as IE7.   Consequently for a time you may have seen an update which is called Internet Explorer 8 in your list of installed updates.   However, as soon as you install a service pack, previous updates to your OS are subsumed by that update.  Consequently you would then no longer see an update which would appear to be IE8 to be removed.   In other words, if you really want to see IE8 be "uninstalled" what you are really saying and have to do is uninstall all OS maintenance back to the level that it could be seen as a separate update and then uninstall it, with all the reduced security that such action would imply.

 

So, instead of thinking that there is a product called IE9 that you don't want to install and apparently are having trouble installing, you should be thinking that there is an update to your OS that you cannot install for a feature that you do not normally use.   You can hide the feature but you should not try to inhibit its functionality because it is used by the rest of the OS, not just the stub browser program, iexplore.exe.

 

 
FYI

 
Robert Aldwinckle
