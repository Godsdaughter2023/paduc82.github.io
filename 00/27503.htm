Sub  FindSymbols(FindChar As String, FindFont As String)

Application.ScreenUpdating = False

 

'set range to return to in case symbol not found

Set OriginalRange = Selection.Range

 

strFound = False

With Selection.Find

    .ClearFormatting

    .Text = FindChar 

    .Replacement.Text = ""

    .Forward = True

    .Wrap = wdFindStop

    .Format = False

    .MatchCase = False

    .MatchWholeWord = False

    .MatchWildcards = False

    .MatchSoundsLike = False

    .MatchAllWordForms = False

   

    Do While .Execute    'Keep going until nothing found

        If Dialogs(wdDialogInsertSymbol).Font = FindFont Then

           'If the correct character was found, exit loop

           strFound = True

           Exit Do

        Else

           'Otherwise search again

           Selection.Collapse wdCollapseEnd

        End If

    Loop

 

    If Not strFound Then

        'if nothing found, search from the beginning of the document

        ActiveDocument.Range(0, 0).Select

        Do While .Execute

            If Dialogs(wdDialogInsertSymbol).Font = FindFont Then

                strFound = True

            Else

                Selection.Collapse wdCollapseEnd

            End If

    End If

    OriginalRange.Select

End If

Saving to USB is part of your problem.  It is actually a Windows issue.  When you "save" a file to USB, due to physical limitations of the USB flash memory, Windows writes to a temporary "Cache" file.  When there is a "block" of data it gets moved to the USB drive.

You don't have to do anything special. Just install and activate normally.  The main thing you have to do is BEFORE you uninstall Office make sure you have the 25 character Office Product Key. If you don't have it from your original purchase, you can extract it from the hard drive.
