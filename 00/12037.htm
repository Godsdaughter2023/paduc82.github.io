bob jones aka: cybertaz say:

Word is based on the use of Styles for formatting. Styles control the formatting of text anywhere the Style is applied. One of the attributes of a Style is Automatically Update, which causes the Style to be modified any time direct formatting changes are made to text which has that Style applied to it. In turn, the same change is made to any other text which has that same Style applied to it.

You need to identify which Style that is & modify the Style to remove the Automatically Update attribute.
