petermatthews5629 say:

Share ALL Calendars within an Organization Automatically.

I was wondering if there is a way to automatically share all users calendars in an office 365 organization using security groups and distribution lists?

yangzheng_msft6300 say:

Hi PeterMatthews5629,

Before giving the exact answer to your question, I’d like to know if the calendars are the SharePoint Online calendars or the calendars in your Exchange Online mailboxes?

If you mean the calendar in users’ mailboxes, based on my experience, there are no ways to automatically share all users’ calendars in an Office 365 organization. However, you can add all the users to a security group and then let all the users manually share their calendars to the security group so that they can all access each other’s calendar.

Moreover, by default the users in the same Office 365 tenant will have the default permission to access the free/busy information of each other’s’ calendar. So if you just want all users to see all the event details in the calendars, you can use PowerShell to change it to Reviewer so that they can see the detailed calendar items in each other’s calendar. Here are the steps:
1. Connect to Exchange Online using PowerShell with an admin account.
2. Run the command below.
Get-Mailbox | foreach{ Set-MailboxFolderPermission $($_.UserPrincipalName+":\calendar") -User default -AccessRights Reviewer}
After the steps above, when the users want to access others’ calendars, they can directly add the users’ calendars in their Outlook client or Office 365 OWA (Outlook Web App).

If my understanding is not correct, please provide more details about your requirement.

Other Browsers and Windows Defender - Microsoft Community

williamgaul say:

Other Browsers and Windows Defender

Thinking of dumping Norton and going with the default Windows Defender on my Windows 10 laptop - my question: Does it protect using other browsers besides Edge (which isn't protected by Norton, so I don't use it anyway) Chrome covered? android phone? Blue Tooth?
rob koch say:

Apparently Microsoft has created an extension specifically for Google Chrome which adds the website blocking capabilities of SmartScreen Filter to that 3rd-party browser.

Windows Defender Extension Enhances Chrome's Security

Windows Defender Browser Protection - Chrome Web Store

So combined with Windows Defender for malware protection, this should approximate the same level of protection for Chrome which Microsoft already includes in Windows 10 for the Edge browser.

Protection for Android would need to be done on that device itself, so that's not an issue for Microsoft, any more than Apple should try to protect Windows or Microsoft to protect an iPhone.

Bluetooth security is something that each vendor must design into its own products as well, since this is typically a version or configuration issue on that specific device or revision of the operating system in question.

Having multiple security systems trying to protect the same device or technology is always a potential source of conflict, typically causing more problems than it solves.
