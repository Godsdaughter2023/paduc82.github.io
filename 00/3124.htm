ken sheridan say:

If you want an unbroken sequence which follows the, not necessarily unbroken, sequence of an autonumber column you can easily do so in a query.  The most efficient method is by means of a JOIN e.g. with the following simple example using a Transactions table:

SELECT COUNT(*) AS RowCounter, T1.TransactionDate, T1.TransactionAmount
FROM Transactions AS T1 INNER JOIN Transactions AS T2
ON T2.TransactionID<=T1.TransactionID
GROUP BY T1.TransactionID, T1.TransactionDate, T1.TransactionAmount;

The sequence will of course remain unbroken if rows are deleted from the table, so is not a fixed value per row.

You can also easily do so in a report by means of the RunningSum property of an unbound control.
