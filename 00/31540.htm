Most likely the driver name you are using on XP to share the printer has not matching 64bit driver name.  The driver names need to match in order to add the printer as a connection to the 64bit machine using the normal RPC protocol.  You can setup the printer on the 64bit Windows 7 as a local printer that will basically copy the printer formatted data to share on the XP machine using SMB protocols.
