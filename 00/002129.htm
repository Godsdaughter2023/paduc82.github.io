daniel g. grau say:
There used to be naming problems with older Word versions, which did not admit a slew of characters, neither in the file name nor in the containing folder, or the folder containing that one, an so on, up the chain and all the way to the disk name (a.k.a. file path).

Since Word's update to 14.1.3 (check your version in the main menu bar's About Word), I only see the "invalid characters" message when the file names themselves contain forward slashes (/), or when there are leading or trailing backward slashes (\) in file names or folder names. (There may be other sequences that I haven't identified so far.)

In my iMac, Word 2011 displays no errors otherwise, not even if the name of the containing folder includes a slash. However, if the file's path contains a leading or trailing backslash, an error occurs. Thus, in my experience, three characters are prohibited:

  • A colon in files or folders (that's a Finder prohibition)
  • Any slash in Word document names
  • A leading or trailing backslash in file names or path names

If I delete an email on one computer, will it delete the email on my - Microsoft Community

sweetcaroline_11 say:

If I delete an email on one computer, will it delete the email on my other computer?

Original Title: "Deleting e-mail"
I have Outlook on two different computers and it is accessed to the same e-mail account. I was wondering while deleting e-mail from one computer, will it delete from the other one as well?
talking moose say:
This is possible only with IMAP and Exchange accounts. If you are connecting using POP then deleting messages from one computer will not delete them from the other.

The nature of IMAP and Exchange accounts is to synchronize what's on your computer with what's on the server and vice versa. You can go to Outlook menu -->  Preferences... --> Accounts  and select your account to see how you're connecting. If you see IMAP Account or Exchange Account under the account name on the right then your messages and folders should be synchronized between all computers.

If you see POP account then you'll need to contact your email service provider and ask if they offer IMAP  or Exchange support.

Hope this helps!
