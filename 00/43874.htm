kevin hau say:

Hello Everyone,

Usually games freezing can be attributed to many different things.  I will try and recap some below:

1. Ensure your hardware has all the lates drivers.  Video, Sound, BIOS, Intel, Mouse, Keyboard, Joystick, etc all need to be up to date Windows 7 drivers, if no Windows 7 drivers are available you can try to install the latest Vista drivers but there can be no guarantee that this will fix the freezing.

2. Check for updates / patches to the game itself. 

3. Make sure you have proper ventilation.  Freezing and Blue Screens can often be caused by overheating.  When playing games the system is taxed much more than normal use (internet, word documents, videos, etc) so overheating is always a concern.  Make sure there's adequate ventilation in front/side/behind the computer and make sure it's not super dusty around the power supply and exhaust fan areas.

4. Make sure that your CPU settings are not modified from the manufacturer defaults (overclocking).  This can cause issues with #3 above, and cause freezing and crashing.  Also make sure that your GPU (video card) is not overclocked as well.

5. Try to unplug any non-required hardware (External HD, Card Reader, USB drive, Webcam, etc) in case it is your power supply that is underpowered and not able to keep up with the power demands of the system.

6.  Try playing in a Clean Boot by disabling all NON MICROSOFT services and applications from START / RUN / MSCONFIG and testing the game in that environment.

I hope this helps.
