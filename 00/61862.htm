What is the syntax for installing a new font via a bat file?

We have four true type fonts that have to be in place for our app to display correctly.  To make sure they are installed I want to exicute a bat file to ensure they are installed.

maverick.69 say:

What is the syntax for installing a new font via a bat file?

We have four true type fonts that have to be in place for our app to display correctly.  To make sure they are installed I want to exicute a bat file to ensure they are installed.

Seriously, I don't know why you would need a batch file to install fonts. Simply copying the *.ttf files (using Windows Explorer) directly into the fonts folder will work.C:\Windows\Fonts .  Nevertheless, if it will make you happy - I'll provide a simple solution for you.

Copy & paste the following into Notepad.

=================================================

@echo
cd /
copy X:\{directory}\name_of_font.ttf C:\%windir%\Fonts

=================================================

make sure you replace X:\{directory}  with the actual location of where your fonts are located.  Also make sure to wrap the entire path in quotes if any of your folder names or font names contain spaces.

When you save in Notepad, it will default to the *.txt file extension.  Go ahead and give it whatever filename you wish.

Afterwards you can change the file extension to *.bat in Windows Explorer.

How to view file extensions in Windows Explorer
