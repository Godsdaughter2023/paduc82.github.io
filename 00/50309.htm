 Power ISO should not be used to make bootable media.  The prompt for drivers  almost always means the media is bad since all modern SATA drivers are in the installation media. 

I would create the flash media using Rufus for UEFI/GPT, then boot it using only the BIOS Boot Menu Key.  Choose the UEFI version of the flash stick, then Custom install, delete all partitions down to Unallocated Space to click Next to auto-format and begin. 

If you changed any UEFI settings in BIOS setup, set them back to UEFI first, turn off Secure Boot, CSM disabled, Windows Boot Manager set first to boot.  If in doubt post back pictures and we'll help you sort the settings correctly.  It might be good to also update the UEFI firmware as this can matter.
