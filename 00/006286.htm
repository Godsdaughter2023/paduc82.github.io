plorent say:

Add Exchange account using another Exchange account

In Outlook 2013 Desktop you can add an Exchange account by using another Exchange account to validate the access. I'm assuming (which one should never do...) that this is not possible on my phone? Or... is it?

martin crosbie say:

No, you can't do this on the phone.

This works on the PC because the 2nd mailbox is set with permissions that allow your other account to access the mailbox, in much the same way that people share their calendars etc. You're accessing the 2nd account through the first, not separately.

the phone cannot use the same method to authenticate through to shared mailboxes (or folders such as calendars) via the phone calendar or email apps, otherwise it could end up synching everything you have access to - potentially the entire server.

HOWEVER

there is a work around - you can access such things via outlook web access. while not ideal on the phone, it is at least possible. you can bookmark specific folders easily.

you could sync with the other mailbox if your admin could give you the password, but I can understand why he might not want that. In theory you wouldn't have any different access, but the admin then wouldn't be able to tell who made the changes, since others would use the same logon credentials. Your admin has set it up the way it should be - securely.

rules could send messages to a specific folder in your 1st account, so you could see them (even on a separate tile) giving your the ability to read them, but this a) duplicates messages (avoiding which was half the point of the 2nd account) and b) wouldn't let you reply to them FROM the other account

Sorry this is a why not, not a how to answer, but it's an answer. and a feature I've been asking for myself.
