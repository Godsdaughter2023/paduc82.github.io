joeu2004 say: JoZaZa wrote:
Eventually I want to add a custom format to both cells that allign them on the right with space the width of a parentheses separating the value from the celll edge, i.e., @_)

The Custom format 0;;;@_) or 0;-0;0;@_) with Right horizontal format seems to work.  We can still enter the formula =A1, and it is interpreted as a formula, not text.

I can understand why simply the Custom format @_) does not work.  Note that the Text format is simply the Custom format @.  So @_) effectively says:  Text format indented from the right the width of one parenthesis.  If a cell is formatted as Text, all data entry thereafter will be interpreted as text.

In contrast, the Custom format 0;;;@_) and 0;-0;0;@_) specify the formats for positive, negative, zero and text values.  That implies that numeric data entry will be treated as numbers and formulas as formulas before they are treated as text.

PS....  The x;x;x;@_) format is very fickle.  While the two forms above work just fine, General;-General;General;@_) does not.  Go figure!
