Thanks for any help.

graham mayor say:

Unlike Word Perfect, Word is a style driven application so even manual formatting takes the form of adding character styles to the underlying text, so what you propose may not be straightforward in Word. For what you are attempting you can set the output to Word to send formatted text, which as you say adds a raft of styles. What you could then do is apply your preferred style to the resulting document and run the following macro, which establishes whether any of the paragraphs contains bold or italic text or small capitals  (I am not convinced that the OCR process can distinguish between small capitals and regular capitals) and then if not, removes the character styling from the paragraphs. You should then have a more limited number of character styles to address.

I agree about Finereader. I have used it since version 4 and currently have version 9, but like all OCR software it is not perfect.

Dim oPara As Paragraph
Dim oWord
Dim bFormat As Boolean
With ActiveDocument
    .Range.Style = "Normal"
    For Each oPara In .Paragraphs
        bFormat = False
        For Each oWord In oPara.Range.Words
            If oWord.Font.Bold = True Or _
            oWord.Font.Italic = True Or _
            oWord.Font.SmallCaps = True Then
                bFormat = True
                Exit For
            End If
        Next oWord
        If bFormat = False Then
            oPara.Range.Select
            Selection.ClearFormatting
        End If
    Next oPara
End With
