john korchok say:

Cloud services don't have the same file structure as your computer. To make relative paths work, you'll have to place all files in the same folder, put that folder in a .zip archive, then have your users download the zip and expand it to access the Word file and its PDFs.

If your Word file must be online, then you'll get better results by uploading all files, getting Share links for each PDF and placing those in the Word document as hyperlinks. These are not relative paths, but they work.

Or you might do both, so there are online and downloadable versions.
