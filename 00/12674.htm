Sub BracketFootnoteNumbers()
    Dim rg As Range
    Set rg = ActiveDocument.Range
    With rg.Find
        .Format = True
        .Style = ActiveDocument.Styles(wdStyleFootnoteReference)
        .Wrap = wdFindStop
        While .Execute
            If rg.Characters(1) <> "(" Then
                rg.InsertBefore "("
                rg.InsertAfter ")"
                rg.Style = ActiveDocument.Styles(wdStyleFootnoteReference)
            End If
            rg.Collapse wdCollapseEnd
        Wend
    End With
    Set rg = Nothing
    Set rg = ActiveDocument.StoryRanges(wdFootnotesStory)
    With rg.Find
        .Format = True
        .Style = ActiveDocument.Styles(wdStyleFootnoteReference)
        .Wrap = wdFindStop
        While .Execute
            If rg.Characters(1) <> "[" Then
                rg.InsertBefore "["
                rg.InsertAfter "]"
                rg.Style = ActiveDocument.Styles(wdStyleFootnoteReference)
            End If
            rg.Collapse wdCollapseEnd
        Wend
    End With
    Set rg = Nothing
End Sub

hansv mvp say:

To display 0.00000122 as 12.2 you need to "scale up" by 10^7, a rather unusual factor. You can do this with a custom number format if you turn on Wrap Text but leave the cell one line high.

The number format contains a line feed; this has to be entered by pressing Ctrl+J.

Once you have entered this custom number format it's difficult to edit, so if you make a mistake, it's best to clear the Type box and start from scratch.

The number format is

 

0.0,<Ctrl+J>%%%%%

 

where <Ctrl+J> represents pressing Ctrl+J to insert a line feed.

If you want 1.22, you can use

 

0.0<Ctrl+J>%%%
