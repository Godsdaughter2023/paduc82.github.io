Windows Spotlight may not work as expected if there are corrupted system files. Since you were able to update to the latest Insider Build, we suggest that you follow these steps to reset and re-register Windows Spotlight in Windows 10:

  1. In Settings, click the Personalization icon then change your Lock Screen to either Picture or Slideshow.
  2. Open Notepad then paste this command: :: Reset Windows Spotlight
    DEL /F /S /Q /A "%USERPROFILE%/AppData\Local\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\LocalState\Assets"
    DEL /F /S /Q /A "%USERPROFILE%/AppData\Local\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\Settings"

    :: Re-register Windows Spotlight
    PowerShell -ExecutionPolicy Unrestricted -Command "& {$manifest = (Get-AppxPackage *ContentDeliveryManager*).InstallLocation + '\AppxManifest.xml' ; Add-AppxPackage -DisableDevelopmentMode -Register $manifest}"
  3. Save the file as Spotlight.bat on your desktop.
  4. Run the file and wait for it to finish.
  5. Restart your PC.
  6. Change your Lock Screen to Windows Spotlight.
  7. Lock your computer to see if Windows Spotlight is working properly.

Let us know if this worked for you.

how do I undo a sort - Microsoft Community

rainy0467 say:

how do I undo a sort

I sorted a column and I want to revert to the original order I entered the data in.  How do I undo the sort?
bob jones aka: cybertaz say:
There is no 'default order' for records in a Excel list. To restore a previous order you'd have to Undo everything done since sorting the records... Usually that isn't preferable :-) If there is no combination of fields inherent in the data which you can use to sort again to return the records to the order you prefer, you need to anticipate this when creating the list.

Include an additional field (column) in the data range so you can assign a sequential 'record number' to each record as it's added to the list. That provides each record with a unique identifier which can later be used as a sort key to return the records to that order.
