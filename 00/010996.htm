hansv mvp say:

You cannot use VLOOKUP to perform a lookup across multiple sheets.

If the number of sheets is small, you can use IFERROR in combination with VLOOKUP, e.g.

 

=IFERROR(VLOOKUP(A1, Sheet1!A:G, 7, FALSE), IFERROR(VLOOKUP(A1, Sheet2!A:G, 7, FALSE), IFERROR(VLOOKUP(A1, Sheet3!A:G, 7, FALSE), "Not Found")))

 

but this becomes very cumbersome with a larger number of sheets. I think you'd be better off with a custom VBA function that loops through the sheets until it finds a match.

I don't know what condition you are setting but will try to explain the principle.

 

If cell A2 is to have red fill if value is less than zero. 

Use the option "Use a formula to determine which cells to format"

Enter the following formula to color the cell if the value is less than zero:

=A2<0

 

However, if you copy the format to cells B2 and C2 then Excel thinks you want to test the value of B2 and C2 but if you want B2 and C2 to be colored based on the value in A2 then the above formula needs to have a $ sign in front of the A so that the column to be compared is absolute like the following.

 

=$A2<0

ossiemac say:

My apologies for not getting back to you sooner on this but I have been busy with other things and I have had to give quite a bit of thought to the problem. I cannot directly add the conditional format to the adjacent cells. However, I have achieved it by use of VBA code if that will be satisfactory.

 

The code will run each time that the worksheet calculates. Don't know if you have expertise in VBA so some instructions to install the code.

 

Right click the worksheet tab name and select View Code.

Copy the code below and paste into the white area of the VBA editor.

Close the VBA editor (cross with red background top right.)

Save the workbook as Excel Macro-Enabled workbook (.xlsm)

 

Ensure that macros are enabled. To do this Select File -> Options -> Trust Center ->Trust Center Settings. (The last one is towards middle of right side of dialog box).

 

Select Macro settings and set the option "Disable all macros with notification". (A confusing option but it means to disable the macros until the user approves the notification to allow them.)

 

The following is the code to copy.

 

 

Private Sub Worksheet_Calculate()

    Dim rngConditFormat As Range
    Dim cel As Range
   
    With Me     'Me is the generic variable for the worksheet to which this code is attached
        'Following line assigns the used range in column A to a range variable.
        Set rngConditFormat = .Range(.Cells(2, "A"), .Cells(.Rows.Count, "A").End(xlUp))
       
        'Following iterates through the used range of column A and sets the interior color
        'of the 2 cells to the right to same interior color as conditionally formatted cell.
        For Each cel In rngConditFormat
            .Range(cel.Offset(0, 1), cel.Offset(0, 2)).Interior.Color = cel.DisplayFormat.Interior.Color
        Next cel
    End With


End Sub

Because in your initial post you had column headers Col 1, Col 2 and Col 3 then I assumed that you were working with columns A, B, C and that the columns would have headers on row 1. Therefore conditional format applied to rows 2 to bottom of data of column A.

 

The following line assigns the range from A2: to bottom of data to the range variable rngConditFormat.

 

 Set rngConditFormat = .Range(.Cells(2, "A"), .Cells(.Rows.Count, "A").End(xlUp))

 

If this is not the range with the conditional format then it needs to be edited.  The part .Cells(2, "A") references row 2 of column "A" therefore change the row number and column to the first cell of the conditional format. The next part Cells(.Rows.Count, "A").End(xlUp)) references the last used cell of column "A". Only need to change the "A" to the required column (same column as first part).
