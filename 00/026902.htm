VBA to copy 1st sheet of all workbooks with formats and values, but no formulas into new workbooks

Sub MoveHedgeSheets()
     Dim strPath As String
     Dim strFile As String
     Dim strName As String
     Dim lngPos As Long
     Dim wbkS As Workbook
     Dim wbkT As Workbook
     Dim wshS As Worksheet
     Dim wshT As Worksheet
     Dim colFiles As New Collection
     Dim varFile As Variant
     ' Prompt user for folder path
     With Application.FileDialog(4) ' msoFileDialogFolderPicker
         If .Show Then
             strPath = .SelectedItems(1)
         Else
             MsgBox "No folder selected!", vbExclamation
             Exit Sub
         End If
     End With
     ' Make sure path ends in \
     If Right(strPath, 1) <> "\" Then
         strPath = strPath & "\"
     End If
     ' Reduce screen flicker
     Application.ScreenUpdating = False
     Application.DisplayAlerts = False
     ' Create target workbook with one sheet
     Set wbkT = Workbooks.Add(Template:=xlWBATWorksheet)
     Set wshT = wbkT.Worksheets(1)
     ' Turn on error handling
     On Error GoTo ErrHandler
     ' Loop through the files
     strFile = Dir(strPath & "*.xls*")
     Do While strFile <> ""
         ' Add filename to collection
         colFiles.Add strFile
         ' Get the next filename
         strFile = Dir
     Loop
     For Each varFile In colFiles
         ' Open workbook
         Set wbkS = Workbooks.Open(Filename:=strPath & varFile, ReadOnly:=True)
         ' Reference to Hedge sheet
         Set wshS = wbkS.Worksheets("Hedge")
         ' Copy the sheet to the new workbook
         wshS.Copy After:=wshT
         ' Reference to new sheet
         Set wshT = wshT.Next
         ' Get position of "." in filename
         lngPos = InStrRev(varFile, ".")
         ' Get part before the "."
         strName = Left(varFile, lngPos - 1)
         strName = Replace(strName, "[", "")
         strName = Replace(strName, "]", "")
         strName = Left(strName, 20)
         strName = strName & " - Hedge"
         ' Name the new sheet
         wshT.Name = strName
         ' Also copy the sheet to a separate workbook
         wshS.Copy
         ' Save the separate workbook
         ActiveWorkbook.SaveAs Filename:=strPath & strName & ".xlsx", _
             FileFormat:=xlOpenXMLWorkbook
         ' And close it
         ActiveWorkbook.Close
         ' Close the workbook
         wbkS.Close
     Next varFile
     ' Delete the first (empty) sheet of the new workbook
     wbkT.Worksheets(1).Delete
ExitHandler:
     Application.DisplayAlerts = True
     Application.ScreenUpdating = True
     Exit Sub
ErrHandler:
     MsgBox Err.Description, vbExclamation
     Resume ExitHandler
End Sub

Assuming that this new template contains styles with the same names as the styles in the documents you're reformatting... This is not a style set, it's just the definitions of the styles in the formatting you want.

Create a macro in the template with Doug's code in it -- you need a Sub statement at the beginning and an End Sub statement at the end, like this:

Sub FixMargins()
  With ActiveDocument.PageSetup
      .LeftMargin = 72
      .RightMargin = 72
      .TopMargin = 72
      .BottomMargin = 72
  End With
End Sub

Make a button for it on the Quick Access Toolbar. See http://www.gmayor.com/installing_macro.htm if needed.

When you get a document from the software, go to the Developer tab, click the Document Template button, check the "Automatically update document styles" box, click the Attach button, and select your template. Click OK. Then immediately go back to the same dialog and uncheck the Automatically update box. Click the button to run the FixMargins macro.

That should do it, unless we've missed something.

How to use Avery Business Card Template with Word 2010? - Microsoft Community

a. user say:

How to use Avery Business Card Template with Word 2010?

Hello,

I am a Microsoft Online Customer Service, who contacting on behalf of my customer. He is unable to use Avery Avery Business Card Template with Word 2010. And he couldn't find it on the Tool option of Word 2010.

jay freedman say:
If your customer is trying to use one of the "templates" that can be downloaded from www.avery.com (which are .doc document files and not .dot template files), that isn't necessary. Everything needed to create business cards on Avery stock is built into Word.

Also, Word 2010 doesn't have any "Tool option", but the use of term indicates that the customer was probably reading instructions for Word 2003 or earlier.

The key is that business cards are handled as labels. In Word 2010, click the Mailings tab of the ribbon and then click the Labels button near the left end. Click the Options button in the Labels dialog. Open the "Label vendors" list box and choose Avery US Letter. In the "Product number" list, choose any of the Clean Edge Business Card items from #8869 through #8878 -- they're all the same layout.

Simply filling in the text in the Address box of the Labels dialog is sufficient for all-text cards. If the customer wants to include graphics, he or she should read this page: http://www.gmayor.com/graphics_on_labels.htm

Since you are running both Office 2016 and 2003 on the same machine, and you want Office 2003 to be the default, clear the option (in Word 2013) at File | Options | General that says "Tell me if Microsoft isn't the default application for viewing and editing documents." When you setup an Outlook.com account, you provide profile information.  It would be simpler to create a new account and have the correct profile information than to rename learner1 and change the profile information.

How to configure Outlook to connect to an outlook.com email account
