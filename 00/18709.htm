How to lock in a font on a document! - Microsoft Community

anthonypavloff say:

How to lock in a font on a document!

How can you lock in the font on a document that is being reviewed by a panel of 5?  Track changes works but when you accept changes the pasted edition remains in the font it was typed in and does not automatically convert to the original document font.  Is this possible without using the paste options and merging fonts which creates an additional step.
jay freedman say:
You can't change the behavior of the built-in Accept and Reject commands. They simply can't deal with the situation you're facing. When you need to change Word's behavior, you need a macro to apply the extra logic.

While the reviewers' changes are still marked as Tracked Changes, a macro can look at each change and remove all the "direct" formatting (that is, anything that isn't part of the style applied to the text).

There are two things to be aware of: (1) It assumes that the fonts you don't want were applied directly and not as part of a style that might have been defined in the document that was the source of the pasted text. (2) The macro below removes or you're stuck with the manual method.

For installation instructions, see http://www.gmayor.com/installing_macro.htm. Here's the macro:

Sub ResetRevisionFont()
    Dim rev As Revision
    ActiveDocument.TrackRevisions = False
    For Each rev In ActiveDocument.Revisions
        rev.Range.Font.Reset
    Next
End Sub

Make a button on the ribbon or the Quick Access Toolbar to run the macro. Open the reviewed document and run the macro before you start accepting/rejecting the changes.
