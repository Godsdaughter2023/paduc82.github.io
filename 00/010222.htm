km678 say:

IE: Delete Browsing History vs. deleting files in AppData Temporary Internet Files

When I visit "MyLocalWebsite" using HTTPS and a self-signed cert, and then delete the cert from the PC and try to visit it again, IE will display a certificate error page that does not allow you to "Continue to website". If I delete the files in C:\Users\<me>\AppData\Local\Microsoft\Windows\Temporary Internet Files, the error page remains the same. However, if I go to Internet Options, Delete Browsing History, and only check "Temporary Internet file and website files", IE will now completely forget I ever visited "MyLocalWebsite" and will display a certificate error page that DOES allow you to "Continue to website". 

What is the difference between manually deleting the files in C:\Users\<me>\AppData\Local\Microsoft\Windows\Temporary Internet Files, and deleting only "Temporary Internet file and website files" through the Delete Browsing History page?

The reason I've given the background about the certificates is in case someone knows something specific about certificates that is affected by the "Temporary Internet file and website files". I do have my Explorer settings set to show hidden files and folders so I am reasonably sure that I manage to delete everything in the folder when I do it manually. 

Thank you for your help.

shafeeq_khan say:

Hi,

 

Thank you for writing to Microsoft Community Forums.

 

The Temporary Internet Files folder is used by Internet Explorer to store files such as images, HTML pages, executable, script files, and webpage content on the computer’s hard disk for quick viewing. This folder lets Internet Explorer download only the content that has changed since you last viewed a webpage, instead of downloading all the content every time that the page is displayed.

 

Whereas, the Temporary Internet files and website files option in Internet Options will include the website data and that will include the certificates of the websites. So, if you clear Temporary Internet files and website files, then the certificates of that website will also get cleared, whereas the Temporary Internet Files folder will only delete the temporary internet files.

 

Hope it helps.
