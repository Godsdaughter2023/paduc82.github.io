Install from ISO should be default option

The Technical Preview of Windows 10 should have been offered as an ISO as the default option. The reason I suggest this is that an ISO install requires some level of knowledge to accomplish, at which point people would generally know what they are letting themselves in for.

I cannot help but feel that a number of unnecessary installs overwriting the existing Windows installs could have been avoided. The level of technical expertise required to handle this sort of preview release should be greater than clicking an update button. I understand the need to reach people of all levels of competency during the trial period, but now is too soon. Replacing a permanent OS with a temporary one is not an exercise for beginners.
