bob jones aka: cybertaz say:
In general it's recommended to avoid the Master Document feature while creating the document. Unless you're properly experienced  & meticulously careful there can be some serious issues... And even then things can get dicey. They're significantly more stable in the .docx format than in the former .doc but not to be taken lightly :-)

FWIW, my preference is to first create a Template in which all Styles, margin settings, etc. are defined. Generate each new chapter from that template. The completed work can later be assembled using the Insert> File command or by inserting subdocuments into a Master Document at that time. That's when the TOC, Index, etc. can be compiled.

When the time comes. you must start in Outline View... the Master Document button is located on the Outlining toolbar which displays only when in that view. That button activates the specialized Master Document toolbar which is where you'll find the tools for working with Master Documents.

Regards,
Bob J.

john mcghie say:
Yes, follow Bob's advice exactly and you will have very little trouble.  The secret is to create all your chapters from the same template, so you have matching styles etc in every subdocument.

The other hint I will give you is "Don't put any text in the MASTER document".  Treat the Master Document as a throw-away item.  Add the subdocuments to it for things like printing and indexing.  If it blows up, throw it away -- it only takes a few months to create a new master document, provided it does not contain any valuable text.

You could even do this pre-emptively:

1)  Create all your Front Matter, Back Matter and Chapter files.  They don;t need any text at this stage, just the level 1 heading.

2)  Add all the subdocuments into the Master Document.

3)  Close Word, and make a copy of the Master Document (not the subdocuments, just the Master).

4)  Place the copy somewhere safe.  If the master blows up, re-save the copy over the top of the dead master file, and away you go again!

Hope this helps
