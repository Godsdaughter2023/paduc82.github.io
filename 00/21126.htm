Hi Christian Stig Andersen,

How many actual Hard Drives are installed? In order to properly install the Technical Preview, first you need to boot to the Hard Drive, as if the USB Drive, wasn't in the Boot Order and make sure that you can't boot up the computer.

Then make sure when you insert the USB Drive, the computer will bypass it and go to the Hard Drive again and fail to boot up.

Now, do not go back into Computer Bios, but go to the Boot Menu and select the USB Drive to boot to. Now when you install the Technical Preview, the process will put what is needed to boot up the Hard Drive, on the Hard Drive, not on the USB Drive.

When finished and the computer reboots, the Technical Preview will start and finish installing.
