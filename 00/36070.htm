I had some problems with the WU reporting.  So, instead of depending on that I think that I would look at the KB article's File Information for definitive clues about your update state.

Oh.  There is only one module to consider?  Let's use Powershell to do that, although if you prefer File Explorer (Win-E) could be used too.

Hmm... this is going to be trickier than I thought.  I don't have an installed copy of Tiworker.exe any more, just cached versions.  (WTH?)  Anyway, this will show you all of them:

PS C:\Windows> dir -Re -Fi "Tiworker.exe" | ft Length, FullName -AutoSize

FWIW the version that I have is two updates beyond the one mentioned in that article, so if you had that too you would not worry about that patch level.  I only use the Full path name and the length now for clues because I have found that other details which could have been easier to use such as FileVersion and DateWritten are often unreliable and hence potentially misleading.

You can also use Powershell to check on the update itself

PS C:\windows> Get-HotFix KB2919442

Here's another method you can try in a cmd window.  I'm not sure if it uses exactly the same data or not.  We have to navigate to WBEM because the RT doesn't have any Path variable!

C:\Windows\System32\wbem>wmic.exe qfe where (HotFixID="KB2919442") get InstalledOn, Status

It probably works in interactive mode too but I haven't really tested that.

One big problem with all these updates is correlating all the possible diagnostics and then trying to infer what each message may mean.  On real Windows 8 we can use ProcMon to assist with that but on the RT there is no such assist.   The best that I can think of for finding what clues the update may have left behind is to use Windows Search in File Explorer for  DateModified:Today  and then sort by that field.   Note that you need to avoid using the index and search everywhere with full permissions to be sure that you aren't missing something.  So, depending on the size of your drive(s) you can assume that that procedure will be SLOW.
