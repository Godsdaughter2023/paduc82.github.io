graham mayor say:
The active template can be identified from the document template button on the developer tab. the developer tab is not displayed by default, so you will first need to add that from the customize ribbon tools. If the original template is no longer available that template will be the normal template.

If you made appropriate use of styles, you can modify the styles applied to the text and universally change the text. If you used manual formatting CTRL+Q and CTRL+Space will reset the text to the underlying styles and remove all manual formatting. You can then apply the styles you require to format the text.

Changes to styles affect only the current document. You can check the 'New documents based on this template' box in the format style dialog to apply your style changes to the attached template, but this will only affect new documents that use those styles. You would need to re-attach the template (template button on the developer tab again) and check the button to automatically update styles to apply changes to the document (having first removed manual formatting in that document), and then uncheck the button to avoid unwanted future changes.
