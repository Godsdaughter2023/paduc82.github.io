[Original subject: Windows Live Mail]

¡firedog say:

Is Windows Live Mail still supported by Microsoft?                

That depends on what you mean by supported. There is no program team working on it, so there's no expectation of any improvements or other changes in the program. If you mean is there help to be had, then you've come to the right place for it.

... is it secure?                 

I'm not sure what you mean by this. Email is not a secure means of communication unless you encrypt the messages you send and receive. It's up to the Email Service Providers you use whether messages you send and receive could be intercepted and read by outsiders. The program and user data on your computer are as secure as all the other data - documents, pictures etc.

... should I switch to Outlook?                 

Outlook is a different program with advanced capabilities in some areas, but it lacks some of the features that Windows Live Mail has. It depends what your needs are.

... would all saved emails in Windows Live be moved to Outlook?                

No. The two programs store messages in completely different ways.

If you're having problems with Windows Live Mail, please use the Ask the community option to start a new thread about each one. Be sure to mention your version of the program and your version of Windows whenever you ask a question.

The file or directory is corrupted and unreadable error message may occur when the location is not available or accessible, and/ or some system files are lost and corrupted.                        chkdsk /f h:

The h is the drive which you have a problem. You might want to check these articles as an additional reference:

  • Run Check Disk from a Command Line to Check for and fix Disk Errors
  • Repair a Windows Image

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.

a. user say:

none

If you have one of the following AV, anti-virus, programs installed, it will shut down your Windows Defender. Reason being those AVs have their own defenders. 2 defenders running in the same machine will create conflicts.
Bottom line : it's ok. It's normal.
List of AVs that will shut down Windows Defender :

Norton
McAfee
AVG
Microsoft Security Essentials

If you do NOT have one of the above installed, here is how to re-start Windows Defender :

Start button > in the search box, typeServices > press Enter key > scroll to find Windows Defender, click it > ....
under Status heading, it should say ......... Started
under Startup Type, it should say ............ Automatic
If it does not say that, Right click on Windows Defender > click Properties > on the new window, make the necessary changes as mentioned above > click OK when done.
