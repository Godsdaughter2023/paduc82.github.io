cpitchford say:

Yeah, I've seen some funky behaviour like that..

 

Most obvious and relevant is this forum.. entering text into the post field (ie where I'm actually typing at the moment) is fine using the physical keyboard, but using the virtual keyboard is a bit hit and miss..

 

For starters, tapping in the text box to open the keyboard does not show any previously entered text

 

I believe this is an artifact of the way the virtual keyboard works.. it seems to be a bit like this (on a pc)

 

open note pad == open virtual keyboard

display current text == copy current text from browser into notepad

finish editing == copy text from notepad into browser

 

ie when you finish editing, the new text is dumped in one go into the text box.. like paste dumps text in one go..

 

This means it doesn't gel with active webpages that react to key presses and do things depending on text in text boxes (like google displays suggestions)

 

It seems a very difficult thing to solve, if the keyboard is not "real" time.. as the n900 virtual keyboard isn't..
