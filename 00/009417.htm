Formula to average a range of values if cells are certain color - Microsoft Community

linusng say:

Formula to average a range of values if cells are certain color

Hi, I've entered a column of values, and color coded them without any explicit criteria.  Only 2 colors involved.  Is there anyway to average by each of the colors into 2 different cells at the bottom?

hansv mvp say:
You could combine the SumColor and CountColor functions from http://www.cpearson.com/excel/colors.aspx to calculate the average value of cells having a specific color.

ossiemac say:

Method of averaging contents based on cell Fill color. Note that it is essential to create the Defined Name for the formula. The formula does not work directly in a cell on the worksheet.

1. Create/insert a helper column to the right of the column containing the values to be averaged. Can be any number of columns away from the column to be averaged (perhaps to the right of all other data).

2. Count and note the number of columns from the helper column back to the column to be averaged. (Click in the helper column and it is the number of times you need to press the left arrow to get to the column to be averaged)

3. Select Formulas ribbon

4. Select Define name

5. Enter FillCol in the Name field. (The name FillCol is optional and can be any name of choice)

6. Enter the following formula in the Refers to field

    =GET.CELL(63,OFFSET(INDIRECT("RC",FALSE),0,-3,1,1))

7. Edit the above formula by first pressing F2 while the cursor is in the Refers to field and then change -3 to match the number you noted in step 2. (If you don't press F2 first to place the field in edit mode then if you use the arrows to move within the formula then Excel thinks you are selecting cells on the worksheet)

8. Press OK to finish.

9. In the helper column and in the first cell opposite the first value in the column to be averaged, enter the following formula. (If you did not use FillCol as the defined name then use the name you chose).

    =FillCol

10. Copy the formula down for the length of the data and you will see a number representing the Index color of the cell fill color in the column with the data for averaging. Note the numbers for the colors required for the Average.

11. Now in the cells where the average is required (cell A23 in the screen shot below) enter the following formula but edit D2:D21 to the range of your helper column, 6 to the number representing the first color required and A2:A21 to the range containing the range of values to extract the average.

=AVERAGEIF(D2:D21,6,A2:A21)

12. Repeat last step 11 for your second color (Cell A24 in the screen shot below) but edit the 6 to the value representing the second color for which you require the average.

13. The same procedure can be used if you want to average on the font color but in the Defined name formula edit the 63 to 24, However, note that it only looks at the color of the first character in the cell if there are multiple colors used for the font in the cell.
