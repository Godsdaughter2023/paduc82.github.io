Easiest way that I have found is with a registry edit. It brings up the password box because just pulling up the dll with the command line still never let me turn the actual feature back on...it just let me add approved sites to the list. Here is what I did:

Click on your Start button
Select Run

Type in "regedit" without the quotes
Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\Ratings\.Default

Double click it and change the Value data from 0 to 1
Hit OK and close the registry editor

Hi, William,

 

Pop-up Blocker is turned on in Internet Explorer by default. To turn it off or to turn it on again if you've already turned it off, follow these steps:

  1. Open Internet Explorer by clicking the Start button , and then clicking Internet Explorer.

  2. Click the Tools button, and then click Pop-up Blocker.

  3. Do one of the following:

    • To turn off Pop-up Blocker, click Turn Off Pop-up Blocker.

    • To turn on Pop-up Blocker, click Turn On Pop-up Blocker.

    •  

If you want to allow a specific website to display automatic pop-ups, follow these steps:

  1. Open Internet Explorer by clicking the Start button , and then clicking Internet Explorer.

  2. Click the Tools button, click Pop-up Blocker, and then click Pop-up Blocker Settings.

  3. In the Address of website to allow box, type the address (or URL) of the website you want to see pop-ups from, and then click Add.

  4. Repeat step 3 for every website you want to allow pop-ups from. When you are finished adding websites, click Close.
