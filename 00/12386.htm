This seems to do the job

I have demo'ed this with just 3 rows but it works with however many you have

best wishes

Sub tryme()
  Worksheets("Sheet1").Activate
  myrow = 2
  newrow = 2
  Do
    mycount = 0
    For k = 3 To 16
      If Cells(myrow, k) = "" Then
          mycount = mycount + 1
      Else
          Worksheets("Sheet2").Cells(newrow, "C") = Cells(myrow, k)
          newrow = newrow + 1
      End If
    Next k
    If mycount = 14 Then Exit Do
    myrow = myrow + 1
  Loop
End Sub

”*” - Wildcard, log all information except for the v and the x option. To include the v and the x option, specify “/l*vx”.
