joeu2004 say: Bullhead John wrote:
I've got the formula under control to calculate the amout of monthly payments for a loan amount - now I'm looking for a formula to tell me what a loan principal amount would be using a fixed payment.

Basically looking for the "reverse" of
=-PMT(Interest_Rate/Num_Pmt_Per_Year,Loan_Years*Num_Pmt_Per_Year,Loan_Amount)

Use the PV function.  If Payment is fixed amount, then:

=PV(Interest_Rate/Num_Pmt_Per_Year,Loan_Years*Num_Pmt_Per_Year,-Payment)

Note the use of -Payment in the function.

Caveat:  The PV() result might not be exactly what you expect if you round the PMT() result.

bill manville say:

Part of the problem is that not all Unicode characters are included in every font.

Here's a macro you can run to create a table of characters.

You can then copy the character you like the look of from there and paste it where you need it.

If you select the entire table and change the font you may find some different characters.

In particular, Windings, Webdings, Symbol.

Sub BuildTable()
  Dim lRow As Long
  Dim iCol As Integer
  Dim lChar As Long
  Dim stRow As String
  Const Width = 16
  Workbooks.Add xlWorksheet
  For iCol = 1 To Width
    Cells(1, iCol + 1) = Hex(iCol - 1)
  Next
  lChar = 0
  lRow = 2
  Columns(1).NumberFormat = "@"
  Do While lChar < 65536
    stRow = Hex((lChar \ Width) * Width)
    Cells(lRow, 1) = String(4 - Len(stRow), "0") & stRow
    For iCol = 1 To Width
      Cells(lRow, iCol + 1) = ChrW(lChar)
      lChar = lChar + 1
      If lChar = 65536 Then Exit For
    Next
    lRow = lRow + 1
  Loop
  ActiveSheet.UsedRange.Columns.AutoFit
  Range("B2").Select
  ActiveWindow.FreezePanes = True
End Sub
