Basic LCD Calibration Guide - Microsoft Community

nielo tm say:

Basic LCD Calibration Guide

Basic LCD Calibration Guide

 

 

Do not use this guide to calibrate PC Monitor or Plasma

 



Before proceeding, apply the following settings:

 

Xbox 360:

  • Output Resolution: Set it to your display’s native resolution (720p/1080p).
  • Reference Level (Only affects RGB Color Space when using HDMI): Standard.
  • HDMI Color Space: YCbCr709, which is considered safe since the user doesn’t have to configure the reference level. But if you’re familiar this matter, choose RGB and set the reference level accordingly.

 

Display:

  • Set picture mode to Movie (or Game if your TV has high input lag)
  • Set the color temperature to Warm 2. If Warm 2 is not available, select Warm
  • Disable all image enhancing features such as dynamic contrast, edge enhancement etc.  inc light sensor.

 

 


Download the Zip filed labelled (Test Patterns) and extract the images on to an optical media (in DATA format) or onto a USB flash memory. To view the images, go to Apps > My Apps and Picture Viewer




Backlight (If Available):

It is recommended that the backlight should be set to less than half the total value. So if the backlight value ranges from 0 – 10, it should be set to 5 or below.

 

 

Brightness (Black Level):

Darken the room as much possible and display the image labelled Black. Lower the brightness value on the TV to 25%, and then slowly increase the value until there's a change in black level. Once you’ve found it, double check to ensure it is accurate. For an example, if the change occurs at 51, lower it to 50.

 

Side Note: If you have the latest (2010-2012) Panasonic LCD, keep the brightness at default.

 

 

Color (Saturation of the Primary Colors):

View the image labelled Color Chart. Set the color value on the TV to maximum and rapidly decrease while focusing on the color red. Make a mental note at the point where it begins to de-saturate. Increase the color level to that point and fine tune it. For example, if the change occurred at 45, increase the color level to 55 and slowly lower it until you see the change..

 

 

Contrast (White Level):

Side Note: If you have Panasonic or early LCD models with no backlight option, ignore this section and set the contrast value to 30-50%.

 

Before proceeding, view this image on your PC monitor. As you can see, setting color and contrast too high can destroy vast amount of detail. Display image labelled Black and While Level and focus on the second row of boxes (particularly shades 249 and 252 on the far right). Set contrast to maximum and slowly lower it until those boxes become barely visible.

 

 

Sharpness (Edge Enhancement):

Ideally, the sharpness should be set to 0 or half the total value. But that’s not always the case. If the images appears too soft when set to 0, you’ll have to find the sweet spot. Load the image Black and While Level and focus on the numbers on black and grey shades. As you increase the sharpness, you should notice halos forming around them. That’s what sharpness does; it doesn’t actually sharpen the image. It simply creates the illusion of sharpness. Most people set it too high, which causes all sorts of problems. Slide the value from 100 to 0 and stop when the numbers are clearly of any haloing. Then fine tune it.

 

If you have any questions, feel free to ask.

Hi Adam, if you want to permanently stop all Upsates:

Click your Start Button, type services and hit Enter

Scroll down to find Windows Update Service

If it is Started, double click it and stop that service

CLose the services App

Click your Start Button, type cmd, right click Command Prompt and choose 'Run as Administrator'

Run this command and hit Enter

sc delete wuauserv

Close Command Prompt

Restart your PC


If you also have the Update Assistant installed on your system:

The Update Assistant in Windows 10 cannot be uninstalled, but you can get rid of it

Click your Start Button, type Task Scheduler and hit enter
In the left Pane, navigate to: Windows - Update Orchestrater
Right Pane double click 'Update Assistant'
Click on Triggers Tab
On each of the triggers ( At login . . . etc.) double click and uncheck 'Enabled'

Under 'Update Assistant, there is another event 'Update Assistant CalendarRun'
Do the same for all triggers on that event

Restart your system, that's It!

When finished and after a system restart, open the services app and make sure the Windows Update Service is now gone . . .

CD /d %userprofile%\AppData\Local

DEL IconCache.db /a

EXIT
