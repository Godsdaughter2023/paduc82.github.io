jim gordon mvp say:

There are two different ways a .docx file could become a .txt file.

One is to simply change the file extension

The other is to open the document then use File > Save As and change the file format to text

Do a little test. In Finder, change the extension of one of the affected documents (choose one that has something more than just text in it) from .txt to .docx and then open the file in Word. If all you see is text and the non-text content has been removed, then the file was opened and saved using Save As. If all the content is still there, then just the file extension was changed from .docx to .txt.

If you can reply with the test results we can take it from there.
