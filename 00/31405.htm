Hi,

There is no easy answer... But windows needs to be installed in the same manner as it was before...

The message you are getting indicates that your windows was installed on a older machine, or one that can run Win 10 with Legacy Bios settings... This type of installation uses a HDD that is Initialized as "MBR"...

Newer machines run Win 10 in UEFI Bios mode with the boot files on a small EFI partition instead of internally on drive C:\ like Win 7 type machines, for instance...

They use a HDD that is Initialized as "GPT" or "GUID"..

This is something that needs to be done before the HDD is ever formatted...

If you have not made any changes to your machine, then the problem is the way the USB Flash Drive was created...

If the Flash Drive is formatted as "FAT32" the machine will recognize this as a attempt to install windows on a GPT HDD with UEFI Bios...

If the USB Flash Drive is formatted as "NTFS" the machine will recognize this as an attempt to install windows as "Legacy"... the old way... with a MBR type HDD...

Based on what you have posted, I would suggest you remake the flash drive with the Win 7 USB\DVD tool... It will create a flash drive that is compatible with the way you machine seems to be set up...

Does this happen when you boot into Safe Mode?

You can get to Safe Mode by restarting your computer and repeatedly pressing the F8 key while it is booting up and selecting Safe Mode from the Advanced Boot menu.
