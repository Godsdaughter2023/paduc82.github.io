Method:

A print spooler is software that temporarily stores print jobs on the computer hard disk or in memory until the printer is ready to print them.

 

If you've been waiting an unusually long time for a document to print, or if you see an error message about the print spooler, you might need to restart the print spooler on your computer.

 

The simplest way to restart the spooler is to save your work and restart the computer. You can also try these manual steps:

 

Check for the dependencies of Print Spooler service and verify whether they’re running.

 

1. Click Start and type services.msc in the Start search box and hit Enter.

2. Locate Print Spooler.

3. Right-click and select Properties.

4. Click the Dependencies tab.

 

Look for the “Dependencies” under “Print Spooler” and make sure all the dependency services are running and set to “Manual” or “Automatic”.

gerry c j cornell say:

Click on the Start button and select Control Panel, Administrative Tools, Services. Locate Print Spooler in the list. Check that it has started and that the StartUp type is Automatic.

I suspect that it has stopped and that just starting it will not solve the problem.

You may need to show hidden files. Type Folder Options in the search box above the Start button and select View, Advanced Settings and verify that the box before "Show hidden files and folders" is checked and "Hide protected operating system files" is unchecked. You need to scroll down to see the second item. You should also make certain that the box before "Hide extensions for known file types" is not checked.

Now access Windows Explorer and select c:\windows\system32\spool\printers. Normally this folder should be empty but there is a problem when files with “spl” and “shd” extensions stick there. Delete the files and restart your computer. This may resolve your immediate problem.

Assuming you have checked your Recycle Bin and gone through all those programs to recover the deleted folder, the two things I would do are:

1.   if you remember the folder or file name, try a search for it using the Search box in your File Explorer.

2.   do a System Restore to an earlier time to when you accidentally deleted the folder.   Go to Control Panel > System & Security > System > Advanced System Settings > System Protection > System Restore.   If you haven't got enough restore points for an earlier date, click on the box to Show more Restore Points.   Using this method, you will lose any updates you have made subsequent to the restore point that you will use so you will need to run them again.
