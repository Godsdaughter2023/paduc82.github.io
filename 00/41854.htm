canadian tech say:

I have been using this process to solve this problem with success many times over.  I recommend you follow it exactly as below.  I have done this on many computers. 

The biggest issue most people have is lack of patience.  This process no longer taking minutes but many, many hours, even days.  Effectively, Win7 WU is getting the lowest possible priority on WU servers. 

Patience is the best tool.  On most of these things, make sure power is supplied, shut down hibernate/sleep functions, and leave the machine for many hours.  Over night would be a good starter.

Windows Update not working

This is a very common problem lately.  The most important tool you need right now is a lot of patience.  Windows Update resources at Microsoft are prioritizing Windows 7 updates at a very low service level and have been since early Summer 2015.  It is not uncommon for Windows Update to take as long as an over-night.

In many cases, the following rather simple and quick process works.

http://answers.microsoft.com/en-us/windows/forum/windows8_1-update/windows-update-problem-error-code-80080008/de8114b5-1487-4b31-9b28-f11974718df1?auth=1  This thread essentially lists a series of commands that I have found usually fixes WU.

Start, All Programs, Accessories, Right-click on Command prompt, Choose Run as administrator, OK.  Type the following in the black box:

  1. net stop wuauserv
  2. net stop cryptSvc
  3. net stop bits
  4. net stop msiserver
  5. ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
  6. ren C:\Windows\System32\catroot2 catroot2.old
  7. net start wuauserv
  8. net start cryptSvc
  9. net start bits
  10. net start msiserver
  11. pause
  12. Exit

Type services in the text box above the start globe.  Find each of the following and ensure they are set to automatic: Background Intelligent Transfer services, Cryptographic, Windows Installer, Windows Update.  Double click on each item and choose automatic.

If you still have a problem, try this.  It will take a long time, but in almost all cases gets Windows Update back on track.

  1. Download and run the correct version for your system of the System Update Readiness tool KB947821. 

    Now re-start Windows Update.  Do NOT restart Windows.   Be very, very patient, it will likely take a long time.
