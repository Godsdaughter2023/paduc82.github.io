ken blake say:

On Sun, 18 Jul 2010 11:37:44 +0000, Hutch4 wrote:

I am about to format and reinstall windows 7 on my laptop and wish to know how to "Save all settings prior to format and reinstall"....Thanks Hutch

You can not do this. Formatting removes everything on the drive. All
you can do is back up what's important to you on external media.

However why do you want to reformat and reinstall? In my view, it's
usually a mistake. With a modicum of care, it should never be
necessary to reinstall Windows (XP or any other version). I've run
Windows 3.0, 3.1, WFWG 3.11, Windows 95, Windows 98, Windows 2000,
Windows XP, Windows Vista, and now Windows 7, each for the period of
time before the next version came out, and each on two or more
machines here. I never reinstalled any of them, and I have never had
anything more than an occasional minor problem.

It's my belief that this mistaken notion stems from the technical
support people at many of the larger OEMs. Their solution to almost
any problem they don't quickly know the answer to is "reformat and
reinstall." That's the perfect solution for them. It gets you off the
phone quickly, it almost always works, and it doesn't require them to
do any real troubleshooting (a skill that most of them obviously don't
possess in any great degree).

But it leaves you with all the work and all the problems. You have to
restore all your data backups, you have to reinstall all your
programs, you have to reinstall all the Windows and application
updates, you have to locate and install all the needed drivers for
your system, you have to recustomize Windows and all your apps to work
the way you're comfortable with.

Besides all those things being time-consuming and troublesome, you may
have trouble with some of them: can you find all your application CDs?
Can you find all the needed installation codes? Do you have data
backups to restore? Do you even remember all the customizations and
tweaks you may have installed to make everything work the way you

How to correct spelling of admin name - Microsoft Community

david cockburn say:

How to correct spelling of admin name

How do I change my admin name?  A friend restored my laptop but typed Davud instead of David for my admin name, and I would like to correct it.
frederik long say:
Is there really nothing I can do about the Davud?

David

There is but creating a new account called David is much, much easier. "Windows Easy Transfer" would be the tool to change the account name - see here:
http://windows.microsoft.com/en-us/windows7/Transfer-files-and-settings-from-another-computer.

Here are the brief steps to use this tool to clone the Davud account to David:
  1. Click Start.
  2. In the Search box, type the words Windows Easy Transfer.
  3. Open Windows Easy Transfer.
  4. Click Next.
  5. Click "An external hard disk".
  6. Click "This is my old computer". Wait until all accounts have been scanned.
  7. Tick the account you wish to clone. Untick the others, then click Next.
  8. Enter a password if desired.
  9. Type a file name, e.g. D:\StandardAcc.mig, then click Save.
  10. Wait. This can take an hour or more.
  11. When finished, close the Windows Easy Transfer screen.
  12. Double-click the file D:\StandardAcc.mig.
  13. Tick the account name you wish to clone (Davud).
  14. Click Advanced Options, then type the name for the cloned account (David). To do so you need to open the drop box to the right.
  15. Click Save, then Transfer.
  16. Wait. This can take an hour or more.
When the process is complete then you will have a cloned account on your machine.

I had the exact same issue as OP. Trending banner did not update within the opened app, nor did it update on the live tile.

I was able to resolve this issue by gaining access to the WindowsApps directory. For whatever reason, when installed, the Bing Search app has two folder located here. After uninstalling Bing Search from the computer, one folder remains. Its that remaining folder that causes the issue and needs to be deleted before Reinstalling the Bing Search App.

WARNING> Permissions of the WindowsApps directory must be changed, proceed only if knowledgeable and confident.

All Windows Store apps that we install through the Store app are physically stored in the folder %ProgramFiles%\WindowsApps. In order to get access to the folder %ProgramFiles%\WindowsApps, we need to follow these steps:

1. Assign the ownership of the folder to the current logged user (and since a folder can only have one “owner“, the “TrustedInstaller” is no longer the “owner” of the folder);
2. Assign permissions to access the folder to the current logged user; 
3. Assign the ownership of the folder back to the “TrustedInstaller” user, so that apps can continue to be installed/uninstalled without problems.

Here is the complete write up and instructions for changing the ownership and then changing it back:
http://caioproiete.net/en/get-access-to-the-programfileswindowsapps-folder-and-view-apps-source-code/

Uninstall Bing Search, gain access to the WindowsApps directory with the above steps, delete the left over Bing folder, restore "TrustedInstaller" ownership to the WindowsApps directory, reinstall Bing Search app.
