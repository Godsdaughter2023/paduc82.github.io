Script

Function expire-WBitems {

param([string]$SiteName,[[System.Reflection.Assembly]::LoadWithPartialName(”Microsoft.SharePoint”)

[DateTime]$today = (get-date).date

[Bool]$updateList = $False

$spSite = new-object Microsoft.SharePoint.SPSite($siteurl)

$spWeb = $spSite.OpenWeb($SiteName)

$spAnn = $spWeb.lists[$ListName]

[Array]$splist = $spAnn.Items

foreach ($item in $splist) {

[bool]$skip = $false

$exptest = $item.GetFormattedValue("Expires")

if ($exptest -eq "") {$item.Recycle();$updateList = $True; $skip = $True}

While ($skip -eq $False) {

[DateTime]$itemExp = [DateTime]::Parse($item.GetFormattedValue("Expires"))If ($itemExp -le $today) {$item.Recycle();$updateList = $True}

$skip = $True

}

}

IF ($updateList -eq $True) {

$spAnn.Update()

}

}

Spreadsheet 1 is my product list downloaded from my GoDaddy web store.
Spreadsheet 2 is my updated inventory with part # & inv. qty.

I sort both spreadsheets by pat number but spreadsheet #1 has additional parts #'s not in spreadsheet #2 so they don't line up. I need to match on part #, which is in both spreadsheets. Take the value of the inventory cell in spreadsheet #2 and put that in the inventory cell in spreadsheet #1.

This is something I have to do every day.

Please advise!!!!!!!

Thanks in advance so very, very, very much.
Chris
sheeloo say:

Assuming you have part numbers & inventory values in columns A & B in both sheets starting in row 2

enter this formula in B2 of Sheet1

=VLOOKUP(A2,Sheet2!A:B,2,FALSE)

and copy it down...

This will find the first row with part no A2 in Col A of sheet2 and return the inventory value from the adjacent cell in col B...

 

You may change the B in Sheet2!A:B to the column letter with your inventroy values and adjust 2 accordingly... 2 is the relative position from Col A to the column containing inventory no. So if inventory is in col E you should use

=VLOOKUP(A2,Sheet2!A:E,5,FALSE)

 

If your part nos are in col c and inventory in col f then use

=VLOOKUP(A2,Sheet2!C:F,4,FALSE)

 

Let us know how it goes...
