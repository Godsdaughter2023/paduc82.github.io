If you use a Microsoft Account log-in to Windows the new Windows 10 Skype Version will start as a background process and there is no way to turn it off - which is a disturbing omission. However if you change the Windows log-in for a local account, Skype does not load as a background task and if you use the new version, rather than the traditional version, then you will have to log into it whenever you want to use it. Unfortunately it remains running as a process when you close the traditional Skype or log out of the Windows 10 version, but if that is a problem you can always kill the running process from Windows Task Manager.

Make sure that you turn off the People options on the task bar also as they hold it open.

Using a local account does not automatically load you into things that a Microsoft account would, but that's probably not a bad thing. I guess it depends how you use the PC and how much intrusion you are prepared to put up with.
