  3. You will be asked to supply a Password, use the same one used to install software.
  4. After Computer finishing loading in Safe-Mode*, do a regular restart.

rick rothstein, mvp - excel say:

As a follow up, the following code will execute much faster than the loop you are using (notably so if there are lots of cells in rngRankings)...

With Worksheets("SheetX").Range("rngRankings")
    .Replace "1", "#N/A", xlWhole
    intNumWinners = .SpecialCells(xlCellTypeConstants, xlErrors).Cells.Count
    .Replace "#N/A", "1", xlWhole
End With

If you retype the above code instead of copy/paste it, pay particular attention to the leading "dots" (one in each of the 3 main statements)

Try your code this way...

For Each C In Worksheets("SheetX").Range("rngRankings")

Next

where you would change my example worksheet name of SheetX to your worksheet's actual name.
