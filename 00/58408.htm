rdrewd2 say:

Hey loyall,

Found a new trick that may help.  See if this works for you: right-click any open space on your desktop, click new/ shortcut, browse to the location of the file you want to pin, and click OK.  That will insert the file path into the location box in the shortcut creation window.  Then in that same box, go to the very beginning of the file path and type explorer with a space after it and click next.  Then on the next screen, name the shortcut with the original file name and click Finish.  Now you should have a desktop shortcut with an explorer icon and the correct file name on your desktop (if you want to change the icon, right-click it, click Properties / Change Icon and pick a different one).  Now if you right-click that icon, you should have Pin to Start and Pin to taskbar as options, so you can put it wherever is most convenient and then delete the desktop shortcut.  By adding "explorer(space)" to the front of the shortcut location, it allows you to pin the actual file to the start menu or taskbar, and not just the program that created it.  Hope that way works better for you.
