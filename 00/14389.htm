The simplest way to do this would be to export your Outlook contacts to a tab-delimited file and then import that into Address Book.
  1. In Outlook select File menu --> Export... and select Contacts to a list (tab-delimited text).
  2. Save the new file to your Desktop.
  3. In Apple's Address Book select File menu --> Import... and select your file.
  4. Use the next window to match the field names in Address Book with those in your export file.
Hope this helps!

My name is Parmi, I am an Independent Advisor. Thank you for posting your question. I'll be happy to help.

1. Collect information about when/why the absence is required --- This can be done in Forms. featured topics--- https://support.office.com/en-us/forms
2. Automatically send an email to the relevant line manager to approve --- This is Not available by default within Forms, you will need to use Microsoft Flows to integrate and send an approval flow(workflow)

4. Automatically email this information to the admin team who arrange cover ----Again this can be done if, you can integrate it with MS Flow
5. Have an automated email sent to the original requester to confirm whether or not their request has been approved ---- Again this can be done if, you can integrate it with MS Flow
5. Have everything collected in a spreadsheet --- In Forms, you can have the responses stored in a single Excel sheet, but not in individual Excel sheet. Analytics export of excel is available by default in Forms.
