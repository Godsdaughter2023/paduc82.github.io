1) Open your Terminal Server Configuration program ON THE SERVER found in the admin tools folder, click "Connections" on the left hand list in the window when the window for the program opens.

2) Then in the list on the right, right click on "RDC-Tcp" and select Properties...  

3) Then on the properties dialog that pops up select the "Client Settings" tab and if you are limiting the color depth (about half-way down on the tab) to anything less than 16-bit then your Windows 10 RDC client will appear garbled.  Now either uncheck the "limit maximum color depth" or set it to 16 bit.  15 bit won't work...  It has to be 16 bit and then it will display just fine using the MS Windows 10 RDC.  Sorry about you folks (like me) where doing this will KILL performance of an RDC based application.  In my case menu construction now takes upwards of 30 seconds to appear from the ether as opposed to 3-5 seconds when it was set to 8 bit.  Oh well...  The message from MS is apparently "Enjoy your break, you've got nowhere else to go today!"  That or you need to buy new hardware and software - now...

    pvtjoker say:
    amuana wrote:
    My Nokia N73 does support s60 3rd edition applications e.g nokia browser and ovi browser,nokia store and ovi store.Can someone help me?Please.

    The N73 is an S60 3rd Edition (S60 3.0) phone, so support for modern apps is quite limited. The Nokia/Ovi browser is for Series 40 phones only, anyways. The S60/Symbian web browser is built-in the phone, but the version is quite old. You'll have to upgrade the firmware to get the most recent one available for the N73, but it's still very old.

     

    Nokia Store can be accessed with the built-in web browser at http://store.ovi.mobi .There is some content there, but not much for such an old phone. For web browsing I suggest you install Opera Mini, which can be downloaded for free from http://m.opera.com .  The Nokia Suite for Windows also supports the N73 and can be used for firmware upgrades and installing free apps (including Opera Mini) through a Windows PC.
