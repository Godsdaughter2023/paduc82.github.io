periwinkle_28 say:

Hi, QaiserQewzd.

 

Welcome to the forum. :)

 

The Other category consists mostly of various operating system caches (temporary files), Maps and 3rd party application data. Also email, calendar, contacts and Office application data counts towards this figure, but are typically smaller contributors.

 

The figure shows how much content, downloaded through normal use of the phone, remains available for re-use. For example, when the phone is used to browse the Internet,  or install applications and application updates, the caches related to these functions grow in size correspondingly. Another contributor to growth of this category is Maps data, whether it is downloaded on-demand when browsing around in Nokia Maps or when you download the maps of entire countries or other regions for off-line use. Furthermore, Photos, Music and Video create an index that grows with number the files located in Phone memory and in any inserted micro-SD card.

 

To manage the phone's memory, try these:

 

1. Remove unwanted stuff from your Lumia Windows 8 Phone by using the Storage Check application. Go to Settings > Storage check, swipe left, the screen will show you the amount of storage used by Apps, Downloaded Maps and Temporary files:

 

  • Select details under Apps to find out what apps you have on your phone, consider un-installing some of them to increase the space in your phone memory.
  • Select maps under Maps, the screen will show you the maps you have downloaded to your phone. Long press an individual map to get an option to remove it, or delete all maps.
  • Select clear under Temporary files to clear the files that are safe to remove.

2. Delete browsing history, temporary files, cookies and saved passwords in Internet Explorer (Settings > applications > Internet Explorer > delete history).

 

3. Delete previous map searches and temporary files from your phone (Settings > applications > maps > delete history).

 

4. Delete Bing search history, including Bing Vision and Bing Music searches (Settings > applications > search > delete history).

 

5. Remove old Office documents from your phone: Press Start, swipe left, and select Office > places (locations) > phone. Long press any document and then select Delete.

 

6. Remove offline mixes from MixRadio: Press Start, swipe left, and select Nokia MixRadio > offline mixes. Long press any mix and then select Delete.

 

7. If you have a PC, connect your phone to a computer with a compatible USB cable. On your computer, use Windows Explorer and browse to your phone to see a list of the folders: Documents, Music, Pictures, Ringtones, and Videos. Right-click the file you want to move, select Cut, right-click a location on your PC, and then select Paste.

I want to limit the character count in these cells to prevent paragraphs (and thus cells) from wrapping to the next page. In other words, I want to lock the format so that cells stay on the page where I originally put them. When cells wrap it screws up the entire rest of the form.

jay freedman say:

It's possible to write code to check the character count on exiting a content control (https://msdn.microsoft.com/en-us/library/office/ff191963.aspx) or, with more code (http://gregmaxey.com/word_tip_pages/content_control_custom_events.html), to check it as the user types. However, that requires each user to allow macros to run, which is not within your control as developer.

Instead, you can set the row heights of some or all table rows to "Exactly" desired values in the Table Properties > Rows dialog. That will prevent cell contents of any kind from changing the form's layout. If users type more than will fit, the extra text disappears below the cell's bottom border. Most users will figure out that they shouldn't do that.
