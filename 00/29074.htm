a. user say:

Hi,

Thank you for keeping us updated.

Installing certificates in the user profile could be due to the previous user profile being corrupted. Hence, you may continue using the new one and transfer your data from previous user profile to the new one.

Here's how:

a. Press Windows key + E to open File Explorer

b. Double-tap or double-click the drive that Windows is installed on (typically, the C: drive).

c. Double-tap or double-click the Users folder.

d. Double-tap or double-click previous user name (click new user name in the other file explorer)

e. Open the folders that contain the files you want to retrieve.

f. Copy data from previous user and paste it in the new user profile's library.

Important: Please ensure you transfer all the data to new user account before your delete the old user account.

Steps to delete old user account:

a. Press the Windows key + X and then select Computer Management.

b. Now, select Local users and groups and then Users.

c. Now, right click on the account that you want to delete and click Delete.

Hope this helps. Please feel free to reply in case you face any other issues with Windows in future.

Click Device Manager
Expand CD/DVD

Click Update driver then follow the on screen instructions.

reg.exe add "HKLM\System\CurrentControlSet\Services\atapi\Controller0" /f /v EnumDevice1 /t REG_DWORD /d 0x00000001

3) Restart the computer.
