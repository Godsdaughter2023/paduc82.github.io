In previous versions of PowerPoint I have been able to use the Shift + Arrow key to move an object a greater distance, (what I am calling a large nudge), than simply using the arrow key alone. In Office 365 for Mac, Shift + Arrow key resizes the object. Is there still a keyboard shortcut for a large nudge, or large move, that is roughly 10x larger than simply using the arrow key?

Thanks!

Matt

bob jones aka: cybertaz say:

In order to have the larger nudge you need to enable Snap to Grid [View> Guides> Snap to Grid]. The arrow keys then will provide the larger nudge. To nudge in smaller increments use Control+Command+Arrow.

Just in case you aren't aware, if your need is to move the object a fair distance horizontally or vertically without affecting its position in the other plane, hold Shift while dragging rather than nudging. The Shift key constrains movement to the direction you start dragging.

As you well may know, the suggestion to "press the command key" is meaningless. None of the modifier keys have an effect on nudging with the arrows unless Snap to Grid is enabled.
