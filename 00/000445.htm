As Karl asks... what are you clicking? Or how (specifically, step by step) are you trying to open the database?

As Gina asks, is the database "Split" into a frontend containing the forms, reports, queries and code and a backend containing the tables? If so, you're seeing the cost of using a mapped drive name (G:/ or Q:/) - the same network folder could have different letter names on different computers. If this is the issue, open the frontend holding down the SHIFT key to bypass the startup; then use External Data... Linked Table Manager. Select all the tables, and navigate to the new location of the backend; but do so using "My Network Places" (or the Windows 10 equivalent), rather than using the (changable) mapped drive letter. This will change the link from "Q:/mydata_be.accdb" to something like "\\servername\path\mydata_be.accdb", which is going to be the same for all users and be more likely to be stable.

If you are clicking on a desktop shortcut, you may need to change the shortcut's Target to the full path and filename of your database.

In a For-Next loop where variables (not explicitly declared) are used to  designate the start and end limits, the program runs correctly on a Mac Platform. However, execution stops with the "Division by Zero" if another application is accessed simultaneously.

'
' Macro1 Macro
'
Dim LastLine As Integer
Dim Counter As Long

LastLine = Application.Workbooks("Workbook2.xlsm").Worksheets("Sheet1").Cells(1, 1)

Counter = 0

For i = 1 To LastLine
    
    For j = 1 To LastLine
        Counter = Counter + 1
        Application.Workbooks("Workbook2.xlsm").Worksheets("Sheet1").Cells(2, 1) = Counter

'

roady [mvp] say:

The Advanced Editor for the Signature Editor has nothing to do with the email editor option in Outlook. All the Advanced Editor button does it opening the selected signature in your default HTML editor. You can configure your default HTML editor via Control Panel-> Internet Options-> tab Programs

Note that you do not need an external HTML editor just to insert an image in your signature. Right click in the Signature Editor to reveal the Insert Image... option.

rh100 say:

How to best use multiple Office 365 accounts at the same time? Default browser per account?

I have Office 365 accounts with multiple organizations that I need to be signed into at the same time.  I know I can do this by using multiple browsers or private browsing mode, but there is a problem with this in that clicking links to shared resources like a OneDrive for Business document will open in your default browser.  I can copy the link to the clipboard and paste it into the right browser, but that is a bit cumbersome and prone to a lot of accidental clicks opening up things in the wrong browser.  Is there a better way to handle this?

neha singh msft say:

Hi RH100,

AFAIK, there’s no other way. This happens because the browser has saved your current login credentials.

If you have logged in to Office 365 portal with your work or school account on one browser, then it won’t’ allow you to access another Office 365 account on the same browser. You should either use a different browser or InPrivate mode.

 

Regards,

Neha
