Sub UpdateCoNames()
  Dim C As Range

jay freedman say:

There's no way to have the AutoFit Window be applied automatically when a column is removed. The user must take some action after removing the column.

Besides clicking the AutoFit button and selecting from the menu, there are two other ways that are more efficient. Each one needs a little setup:

  • Right-click the AutoFit Window entry in the menu and click Add to Quick Access Toolbar. This only has to be done once, and from then on the button will be available with one click. It will be grayed out if the cursor isn't in a table.
  • In the File > Options dialog, go to the Customize Ribbon page and click the Customize button in the bottom left corner. In the Customize Keyboard dialog, click Table Tools | Layout Tab in the left column and then click AutoFitWindow in the right column. Click in the "Press new shortcut key" box and press a key combination such as Alt+Shift+Ctrl+A.  Click the Assign button and then Close.

From then on the assigned keyboard shortcut will apply the AutoFit Window to the table containing the cursor. It won't do anything if the cursor isn't in a table.

jay freedman say:
While the cursor is in the remaining column of the table, so the Table Tools tab is visible, click the Layout tab under that. Then click the AutoFit button and choose AutoFit Window in the menu.
