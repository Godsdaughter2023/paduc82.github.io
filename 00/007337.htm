doug robbins - mvp office apps & services (word) say:
In a long document, is it true that pressing ENTER more than once is a bad idea?  (WORD 2003) I want to position an image low on a page, below an area of white space.  I also want to double-space between some, but not all, paragraphs.  Must I use page breaks to do so?  Thanks!


It is bad form to use the Enter key more than once to achieve spacing between paragraphs in any document, not just a long one.   You should make use of appropriately defined Styles with the spacing between paragraphs controlled by appropriate settings of the Before or After space in the Format Paragraph dialog. 

Hard page breaks should also be avoided as subsequent editing in other parts of a document can cause the break to occur in other than the position in which it was intended.  Once again, it is better to use the appropriate paragraph formatting of Page Break Before or Keep with next.

srinivasu guntimadugu say:

Either the *.msg file is corrupted or the Outlook data file is corrupted.

What is the type of e-mail account (POP, MAPI, IMAP or Exchange) setup in Outlook 2007?

Check if you are able to open that e-mail message (On the desktop) on another computer?

Try the following steps:
1. Open the e-mail which is on the desktop.
2. Copy the Contents to a new e-mail message and save it on a different location as *.msg.
3. Now try to open the e-mail message from the desktop and check if you are able access and use it as template.
