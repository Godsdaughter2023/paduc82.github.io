teylyn say:
Hello,

Is the space always after four characters or can it be in different positions? If it is always after the fourth character, then use the method suggested by Jaeson, but if the space can be in different positions, you need another approach.

Can you edit the other department's workbook? If so, you could create a column with a Substitute() function and then use that column for the Vlookup.

If that is not possible, you could try this formula. Have both workbooks open and replace the file name with the name of your other workbook.

=INDEX([otherbook.xlsx]Sheet1!$B$1:$B$10,MATCH( A1,SUBSTITUTE([otherbook.xlsx]Sheet1!$A$1:$A$10," ",""),0))

This is an Array formula and must be confirmed with CTRL-SHIFT-ENTER.

In this example, the formula returns the value from column B where the department is a match in column A, just like a Vlookup(A1,[otherbook.xlsx]Sheet1!$A$1:$B$10,2,False) would do.

Adjust to your situation. 

When you close the other workbook, the formula will show the complete file path of the other workbook, but it will continue to work with the closed file.
