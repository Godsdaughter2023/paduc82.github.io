amit tandon say:

Array formula (Ctrl+Shift+Enter) in cell F4:

=VLOOKUP($A$11,IF($A$2:$A$10=F$1,$B$2:$D$10),3,FALSE)

Non-Array / Normal formula in cell F3:

=INDEX($D$2:$D$10,MATCH(F$1&$A$11,INDEX($A$2:$A$10&$B$2:$B$10,),0))

The above 2 formulas return the first matching value only (for Store A and Code 12345). To return SUM of ALL matching values use below normal formula in cell F2:

=SUMIFS($D$2:$D$10,$A$2:$A$10,F$1,$B$2:$B$10,$A$11)
