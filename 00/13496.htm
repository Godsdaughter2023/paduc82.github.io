VBA to Hyperlink Inside Multilevel List - Microsoft Community

icybricks say:

VBA to Hyperlink Inside Multilevel List

I have a long multilevel list outline that often has repeated steps of previous outline bullets. What I hope to accomplish with VBA is for Word to scan through the paragraphs and find any identical paragraphs and have it hyperlink back to the first instance of the paragraph. Every paragraph is a heading style of some number. For example:

  • Item A
    • Do this
    • This
    • And this
  • Item B
    • Do this
    • That
    • Then this
  • Item C
    • Don't do this
    • That
    • Or this

Hyperlinks created:

Item B's "Do this" is linked back to Item A's "Do this"

Item C's "That" is linked back to Item B's "That"

Why? The outline is a list of steps of electrical repairs for filming. Some steps like "open access panel" are going to reappear often. In those cases the step will simply be filmed once and edited into multiple spots (less work, more consistency).

andreas killer say:
Option Explicit

Sub Test()
  Dim Dict As Object 'Scripting.Dictionary
  Dim P As Paragraph
  Dim R As Range, OldR As Range
  Dim B As Bookmark
  Dim H As Hyperlink
  Dim Key As String, BName As String
  Dim i As Long

  Set Dict = CreateObject("Scripting.Dictionary")
  Dict.CompareMode = vbTextCompare

  With ActiveDocument
#If Develop Then
    .Bookmarks.ShowHidden = True
    Do
      For Each B In .Bookmarks
        B.Delete
      Next
      For Each H In .Hyperlinks
        H.Delete
      Next
    Loop Until .Bookmarks.Count = 0 And .Hyperlinks.Count = 0
#End If
    For Each P In .Paragraphs
      'Build a key from the indent level and the text
      Key = P.Range.ListFormat.ListLevelNumber & ":" & P.Range.Text
      'Found before?
      If Not Dict.Exists(Key) Then
        'Get the range without the EOP char
        Set R = P.Range
        R.MoveEnd wdCharacter, -1
        'Save it with an associated number
        Dict.Add Key, Array(R, i)
        i = i + 1
      Else
        'Get the first occurence
        Set OldR = Dict(Key)(0)
        'And the associated number back
        BName = "_Link" & Dict(Key)(1)
        'Create a bookmark if not exists
        If Not .Bookmarks.Exists(BName) Then
          Set B = .Bookmarks.Add(BName, OldR)
        End If
        'Create the hyperlink
        Set H = .Hyperlinks.Add(P.Range, SubAddress:=BName)
      End If
    Next
  End With
End Sub
