the bizdoctor say:

Here’s the actual steps needed to get your external USB drive assigned a letter in Windows XP if it does not show up in Windows Explorer:

 

1)     Right-click on “My Computer”

2)     Click “Manage”

3)     Double-click “Disk Management”

4)     You will probably see Disk 0 and Disk 1 for your primary hard drive. Go to lower right portion of box and scroll down until you see Disk 2. It is probably your USB drive without a letter.  It should show the size of the drive.

5)     Right-click in the left portion of that drive box (Disk 2)

6)     Click “Initialize” and “OK” in the next dialog box. This takes just a few seconds and it should now say “Online” in the “Disk 2” box.

7)     Right-click in the box to the right side of the “Disk 2” box (where it shows your disk drive size and says “Unallocated”.

8)     Click “New Partition” and a “New Partition Wizard” will open up.

9)     Click “Next”, then select “Primary Partition” and click “Next”.

10) Verify partition size (use entire disk size available unless you are creating multiple drives) and Click “Next”.

11) Assign drive letter (“F” is typical if CD-Rom is E) and click “Next”.

12) Format this partition with “NTFS”, “Default”, “New Volume” and click “Next”.

13) Then click “Finish” to exit the wizard and start formatting.

14) Be patient… this can take awhile (it took just over an hour for a 320GB drive).

15) When done, you should see an icon for your drive in Windows Explorer!

 

Hope this helps!!

Blue screen errors can have a multitude of causes, so I'll provide some general troubleshooting advice.

Firstly, if you're concerned about the security of your PC, run a Malwarebytes security scan:

Otherwise, try running the System File Checker in case it can repair any corruptions:

- Right-click Command Prompt and run it as administrator.
- Type sfc /scannow
- Press Enter.

You may also want to run a checkdisk to check your drive's integrity:

- Click Start and type ""cmd"".
- Right-click Command Prompt and run it as administrator.
- Type ""D:"" (without quotes) and press Enter. If your external drive isn't D, change it to the appropriate letter.
- Type ""chkdsk /x /f /r"" (without quotes) and press Enter.

It'll ask you to reboot so it can check the disk. Press Y and your computer will reboot, check your disk and repair any errors.
