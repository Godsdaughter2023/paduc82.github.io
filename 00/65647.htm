jim mcnugget say:

It is not sufficient to just check the Remote Server Administration Tools option as this will not select any of the sub-options such as GPMC. In the Windows Features dialog again, expand the Remote Server Administration Tools section with the +, followed by Feature Administration Tools and then select Group Policy Management Tools.

Once this has installed, Group Policy Management should appear inside the Administrative Tools section of Control Panel.

If you are looking to just manage the policy for the local machine, a quicker way is just to run gpedit.msc. If, on the other hand, you're working with a domain then installing the admin tools as we have discussed is the right way to go.
