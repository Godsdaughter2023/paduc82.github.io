yves dhondt say:

If all you wanted to do is change the text with the hyperlink, you could run the following macro:

===================================

Sub ChangeHyperlinkDisplay()
    Dim h As Hyperlink
   
    For Each h In ActiveDocument.Hyperlinks
        h.TextToDisplay = h.Address
    Next
End Sub

===================================

Of course you could go more complex and add the website between brackets after the text or anything else you can imagine:

===================================

Sub ChangeHyperlinkDisplay()
    Dim h As Hyperlink
   
    For Each h In ActiveDocument.Hyperlinks
        h.TextToDisplay = h.TextToDisplay & " (" & h.Address & ")"
    Next
End Sub

===================================

To install macros from listings, see  http://www.gmayor.com/installing_macro.htm

NOKIA has a factory in China (as well as ten or so other countries, too), so a genuine NOKIA phone can very well be made in China (from the same parts and using the same manufacturing process as if it had been made in some other Nokia factory in some other country).

No fake NOKIA phones run Symbian, either (they're usually based on Windows Mobile or some Linux variant).

 

If you're phone says NOKIA on the phone, on the labels, on the sales package, etc., and also works fine with Nokia PC software, then it is a NOKIA phone.

macropod say:

There is no "Manual hard return" Find option. What Word has is a Find option for a "Manual line break".

The "Manual line break" option does not purport to insert the code for a paragraph break (^p) but only for a manual line break (^l).

If you want to specify a paragraph break using 'Special', you'd select ... "Paragraph break" - which inserts a ^p.

Finally, files that have been constructed in other environments may contain characters that look like ¶ but are not, in fact, paragraph breaks recognised by Word - they might just be ASCII 182, or they could be ASCII 13. To Find either of these, you could insert ^0182 or ^13 into the Find dialogue.

robert19 say:

If you have "Use Wildcards" ON, and want to search for non-printing “paragraph marks” (aka “paragraph end marks”), you need to search for “^13” (without the quote marks).
If you have "Use Wildcards" OFF, and want to search for non-printing “paragraph marks” (aka “paragraph end marks”), you need to search for “^p” (without the quote marks).

If you want to search for printing PARAGRAPH CHARACTERS (aka “PILCROW SIGNS” or “PARAGRAPH SIGNS”), you need to search for “^u182” (without the quote marks and with the "Use Wildcards" option OFF). Note that a pilcrow sign gets inserted by pressing “Alt+0, 182” (without the quote marks).

If you want to search for “Manual Line Breaks” which are inserted with “Shift+Enter”, you need to search for “^l”  (with lowercase “l” and without the quote marks). Alternatively, you can search for “^11” (without the quote marks). It does not matter if the "Use Wildcards" option is OFF or ON. Also you might want to have a look at the following thread:

What is a manual line break, and how do I insert one?

Here is an excerpt:

"Often when you paste from the web, and also from some other applications, characters come in that display as paragraph marks but don't behave like “proper” paragraph breaks should—they behave like manual line breaks! So you might find that when you center a “paragraph,” several other adjacent paragraphs also get centered. To cure this, use Replace: in the “Find what” box type ^013; in the “Replace with” box, type ^p and click Replace All."

HTH.
