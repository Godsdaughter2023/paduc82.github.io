The last line is normally left justified. If it isn't, then your last line ends with a soft return (shift enter) rather than a hard return (normal enter).

"Distributed" is NOT the same as "Justified" -- the difference being, as you found,
whether the last line of the paragraph is forced to the right margin or not.
 
Select the paragraph(s) in question and press Ctrl+J to apply Justified formatting,
instead of Ctrl+Shift+J for Distributed formatting. (The Paragraph dialog's Alignment
dropdown includes the Justified choice but not the Distributed choice.)

I have to disagree with you there -- WP justification is  *not* "just like Distributed

   formatting than they are in the Distributed formatting, because (like WP) the Justified

SUMIF for multi-columns - Microsoft Community

steve mc209 say:

SUMIF for multi-columns

I haven't had to work with spreadsheets in a couple of years, and I'm a bit rusty.

I have this formula     =SUMIF($C$7:$C$327,C7,$E$7:$E$327)

which works for the E column, however, I need to also include data in every other column G, I, K, M, etc. 

What formula would accommodate that data in other columns ?

Thanks,

Steve

hansv mvp say:

Adjust the ranges as needed in the following formula:

=SUMPRODUCT($E$7:$Y$327,($C$7:$C$327=$C7)*(MOD(COLUMN($E$7:$Y$327),2)=1))

steve mc209 say:

Perfect.  Thank you very much.

One other question. I don't recall using Mod before. What does that do ?  and the ,2)=1)). Am I correct in assuming the 2 means every 2nd column starting at E ?  And the 1 ?

Thanks again,

Steve

hansv mvp say:

MOD(a,b) returns the remainder of dividing a by b as whole numbers.

For example MOD(23,7) returns 2 because 7 goes 3 times into 23, with 2 left over.

And MOD(28,7) returns 0 because 7 goes exactly 4 times into 28, with no remainder.

MOD(n,2) returns 0 if n is an even number, and 1 if n is an odd number.

MOD(COLUMN($E$7:$Y$327),2) is 1 for odd-numbered columns in E7:Y327 (i.e. columns E, G, I, ...) and 0 for even-numbered columns in this range (i.e. F, H, J, ...)

So MOD(COLUMN($E$7:$Y$327),2)=1 checks for odd-numbered columns.
