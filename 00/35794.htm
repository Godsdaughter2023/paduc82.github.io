One suggested fix is:

* In start menu find services.msc

* Right click on it and select “Run as Admin”

* In the displayed list, find the Windows Modules Installer service, “TrustedInstaller”

* Right click on it and select “Stop” (Do NOT close the services dialog yet)

* In File Explorer, go to C:\Windows\Logs\CBS\

* Delete or move or Rename any ALL of the CBS.LOG or Cbspersist_XX.log files

* In file Explorer, go to C:\Windows\Temp

* Delete all CAB_xxxx_x files

* Go back to the Services dialog

* Right click on the TrustedInstaller service again, Select “Start”

* Close the services dialog.

* Restart the computer.

This should stop the broken process that was generating all of these files to fill you drive.

An INF file is a text file that contains all the information necessary to install a device in Windows O.S where INF stands for information.
