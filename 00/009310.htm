a. user say:

I was able to get this done using the following:

$myO365Sku = New-MsolLicenseOptions -AccountSkuId contoso:ENTERPRISEPACK -DisabledPlans MCOSTANDARD

Get-MsolUser –All | Where { $_.IsLicensed –eq $true -and $_.AccountSkuId -eq $contoso:ENTERPRISEPACK } | Set-MsolUserLicense -LicenseOptions $myO365Sku
