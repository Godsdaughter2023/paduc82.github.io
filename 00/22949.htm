peterfine---01 say: Your phone has a GPS chip which is perfectly able to calculate the location data from the info provided by the satellites. Also, network towers do not have GPS recievers. They don't move, so there'd be no point.

AGPS and Network based methods both send the ID of your network tower to nokia, using a data connection. Nokia looks up the location of the tower using its database. For network methods, they send back this location - its a rough estimate because it's the tower's location, rather than yours. It doesn't require any GPS reciever though - any phone can use it.

For the AGPS method, however, they send back information about which GPS satellites are likely to be found at that location - your phone can then use this data to know what to look for, speeding up the GPS fix process from ~2 mins to ~30 seconds, compared to non-assisted-gps.

Hi:

 

Try this small macro:

 

Sub Unconditional()
Dim N As Long, L As Long, r As Range
N = Cells(Rows.Count, "A").End(xlUp).Row
For L = 1 To N
    If Cells(L, "A").Value = "Header" Then
        Set r = Range("A" & L & ":O" & L)
        With r
        .ClearFormats
        .Font.ColorIndex = 2
        .Interior.ColorIndex = 1
        End With
End If
Next
End Sub

john korchok say:

So you want a hyperlink in Word that takes you to the top of the document (I think)? These are the steps to do that:

  1. Create a bookmark at the top of the document or whatever place you want to return to. One possibility is to select the document title, then use Insert>Bookmark, give the bookmark a name like TheTitle and click on Add.
  2. Go to the first spot in your document where you want a link to take you back to the top.
  3. Choose Insert>Hyperlink.
  4. In the left-hand Link to: column, select Place in This Document.
  5. In the main panel Select a place in this document:, look under the Bookmarks heading and choose TheTitle. In the Text to display: field, type back to top. OK out. A hyperlink is created.
  6. Copy and paste this hyperlink through your document.
