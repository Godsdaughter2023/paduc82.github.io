I understand that you are unable to use your Lumia 640 as a WiFi hotspot. Have you tried to perform a soft reset? If not please do so.

You can perform the soft reset by having your phone turned on. Then, press and hold the volume down and         Power button until your phone vibrates and restarts. steve rindsberg say:

PowerPoint has never had the ability to run anything from startup folders; it will only run code automatically if it's in an Auto_Open subroutine in an add-in.  Then it will run the code when the add-in loads.

Mac PowerPoint has never had the ability to install/modify the ribbon, though it's rumored that this capability may be added eventually to Mac PPT 2016.  Currently, PPT 2016 doesn't support ribbon customization OR command bars, meaning that if you need to create any sort of user interface at all for Mac add-ins, you need to run it in PowerPoint 2011/Mac.  

People who rely on add-ins will NOT want to move to Mac Office 2016 just yet.
