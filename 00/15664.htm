It isn't clear but apparently some values have no letter at the end?

I believe you can create a small public function that returns just the alpha characters on the right with beginning spaces providing some padding so that "C" comes befor "AA". These values will be converted to "    C" and "   AA". If you will EVER have more than 5 letters at the end, it won't hurt to change the 5 to 10. 

Public Function FindEndingText(varText As Variant) As Variant
    Dim intChar As Integer
    Dim intFirstText As Integer
    If Not IsNull(varText) Then
        For intChar = 1 To Len(varText)
            If Not IsNumeric(Mid(varText, intChar, 1)) Then
                Exit For
            End If
        Next
    End If
    FindEndingText = Right("     " & Mid(varText, intChar), 5)
End Function

Your query can then use the expressions in columns for sorting:

SortAlpha: FindEndingText(DWGNUMBER)

hansv mvp say:

You cannot enter a date consisting of a day and month only.

You can do the following, however.

Let's say that the dates are in E2:E100 (with a header in E1).

In a free column, for example in column F, enter the following formula in row 2:

 

=DATE(2000,MONTH(E2),DAY(E2))

 

Fill down to row 100.

Optionally, enter a header in row 1 in this column, e.g. DayMonth.

Sort the entire range on the new column. You can then hide this new column if you prefer.
