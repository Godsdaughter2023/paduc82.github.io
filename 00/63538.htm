We appreciate you getting back to us with an update, Greg.

Cortana failed to be enabled while in lock screen has already been identified as a bug in the Fall Creators Update, but is now resolved. You may follow these steps to resolve yours too:

  1. Type regedit from the Cortana search bar
  2. Select regedit Run command from the result.
  3. Follow these path to reach Windows Search:
    HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Windows Search
  4. Change the value data to 1.
  5. Navigate to HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Search.
  6. Change the BingSearchEnabled value from 1 to 0. Make sure that all of the Cortana flags are set to 0 instead of 1.

Note: This method to fix the issue will need Registry modification. Before you modify the Registry, make sure to back your files up and understand how to restore the registry if a problem occurs. Please refer to this link on how to back up and restore the registry in Windows.

If there is a missing Registry path, you may safely add the missing folder. Here's how:

  1. Click the Registry key under which you would like to add a new key. On our case, it should be under the CurrentVersion.
  2. Click New on the Edit menu and then click Key.
  3. Type a name for the new key and then press the Enter key from the keyboard. The name should be Search.
  4. Click and unfold the created key (Search), and right-click on the right pane.
  5. Choose New > DWORD 32bit Value > Name it BingSearchEnabled and change the Value data from 1 to 0.
  6. Reboot your PC.

Let us know the result.


Regards.
