If you have failed to connect the hotspot connectivity using connectify, you may have to contact connectify support since creating a Wi-Fi hotspot feature is not available in Windows 8.

c. Instead of typing nameofnetwork in the above step, type any name that you want to give to your network. Instead of typingnetworkpassword in step above, type secure hard-to-guess password that you want for your network so that anyone trying to connect to your network must know the password.

e. If you want to remove or stop the newly created ad-hoc connection, type netsh wlan stop hostednetwork in command prompt (admin).
