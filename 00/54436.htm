wizard fait say:

I know it's a little late, but I stumbled upon this issue myself, and found the solution (at least for my own version of this problem).

When I experienced the same issue as you on my E: drive, I searched the Registry for the name (mine being "V100W" made that a lot easier a task than "Games" would have been) and stumbled upon the following key:

Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\DriveIcons\E\DefaultLabel

The (Default) value of that key was set to "V100W".

I simply deleted the E folder, as there were no other drive letters there, and this was the only one exhibiting the symptoms. I then plugged in a device I had claiming to be "V100W" not even 5 minutes prior, and it came up as Removable Drive E:.

So in your instance, you most likely had a Registry key in the same place, but within a G instead of an E, and set to "Games".

Hopefully this helps someone in the future who has this issue! :)
