drtarr1962 say:

I had the same troubles with file explorer too after updating to 10 from 7.  Everything worked fine from safe boot, and also when I made a local login for testing.  Didn't work during a clean boot.  I found out that the trouble I had was caused by an item in the context menu (the little menu that pops up when right clicking) that one of the programs I installed in 7 had added to the context menu.

You can disable the items in the context menu either by editing the registry, which can be a pain in the posterior region, and shouldn't be done unless you know what your doing.  Or by using a program like Shell Extension Manager for Windows which is by nirsoft and is freeware.  If you decide to try Shell Extension Manager here's the instructions.

1.) Run the program as administer.

2.) From the Options menu and click on Filter by Extension Type and pick Context Menu.

3.) From the Options menu again click on Hide all Microsoft Extensions.

4.) Hold down the CTRL button and click on the Third part extensions. Which will be highlighted in pink.

5.) Click on the Red button at the left, top of the program to disable the extensions

6.) Log out, and re-log back in, or restart the computer to reload the registry, and test to see if that worked.

If it worked then you can re-enable the extensions one at a time to find out which one is causing the problems.  Just don't forget to log out, re-log back in between each one you re-enable.
