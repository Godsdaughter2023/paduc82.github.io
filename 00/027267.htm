bob jones aka: cybertaz say:

It has nothing to do with Headers/Footers or Page Breaks :-)

Margins are a property of Sections in a Word document. Apparently, your document consists of only one (default) Section, so when you change the margin you're changing it for the entire document. To get the result you want;

  1. First, display the nonprinting characters [¶] so you can better see what you're dealing with
  2. Next, remove the manual Page Break if it's still there [it's always best to avoid them in the first place]
  3. Place the insertion point at the last ¶ on the first page
  4. Go to Insert> Break> Section Break (Next Page)
  5. Place the insertion point anywhere on the second page
  6. Go to Format> Document - Margins to set the values you want
  7. Confirm that Apply to: is specified as This Section
  8. Click OK

If that presents any problems with the Header/Footer, Page Numbers, etc., don't panic :-) Post a reply here describing the issue in order to get help making the necessary adjustments.
