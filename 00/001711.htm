You should consider using a UserForm.

However if you create autotext entries of the paragraphs that you want to insert, here is some text that will insert an autotext entry that is named with the item selected in a dropdown formfield.
 

' Macro created 15-11-97 by Doug Robbins to add the address corresponding to a drop down name

'

Set myDrop = ActiveDocument.FormFields("Dropdown1").DropDown

Company = myDrop.ListEntries(myDrop.Value).Name

Address = ActiveDocument.AttachedTemplate.AutoTextEntries(Company).Value

ActiveDocument.FormFields("Text1").Result = Address
