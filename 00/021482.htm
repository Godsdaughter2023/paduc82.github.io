jay freedman say:

That can be a display problem, especially with the default 1/2 point border thickness and with zoom factors of 100% or less. Word may need to make the border only one pixel wide or high, and that can fall between the locations of the screen pixels.

Things you can try:

  • Set the zoom factor slightly larger, such as 105% or 110%.
  • In File > Options > Advanced, in the Display section, change the options "Optimize character positioning..." and "Disable hardware graphics acceleration" from unchecked to checked or vice versa.
  • Change the screen resolution setting in Windows, if possible (Settings > System > Display).

Use the group header section's ForceNewPage property to get the group to start on a new page.  Then add a page break control (named pgBreak) at the very top of the group header section.  Add code to the group header section's Format event:
