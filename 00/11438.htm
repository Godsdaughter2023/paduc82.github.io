 Application.ShowMenuFloaties = True

Sub ShowFloaties()   ' True = False, 2nd line toggles to True

As is most all formatting, the Language formatting of Comments is controlled by a Style. In fact, there are 3 Comment Styles [Comment Reference, Comment Text] but the primary one is Comment Text.

One option is to modify the style to include the preferred Language attribute. However, that style is based on the Normal style will cause the Language attribute to be inherited by Comment Text as well as by any other styles based on Normal. IOW, modifying the Normal paragraph style to include the Language attribute may well be all that is necessary.to update the Language formatting f the entire document, including the Comments. It just depends on how the styles have been defined & implemented in the document.

suzanne s. barnhill say:

If you look at File | Options | Advanced: File locations, the path for "User templates" is the one you want. The default folder for user templates in Word 2013 is C:\Users\Your Name\AppData\Roaming\Microsoft\Templates, but this folder may be hidden on your computer; you'll need to tell Windows Explorer to display hidden and system files and folders.

Note also that you won't see a Normal template at all unless you have done something to modify it. Till then it exists only as a definition in the Word executable; the file is created when you customize something.
