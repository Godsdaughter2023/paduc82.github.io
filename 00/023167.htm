Possible solution to copied placeholder losing bullet formatting - Microsoft Community

rich007a say:

Possible solution to copied placeholder losing bullet formatting (2 of 2)

I would like to present the second of two potential work-arounds to the problem of copied text placeholders losing the ability to properly handle multi-level bullets in PowerPoint 2007.

In this second of two options, the code attempts to reverse engineer a custom layout to match the current positioning and formatting of the actual placeholders on a slide.  It then adds an additional placeholder to the new custom layout, in the same position as the offending copied placeholder (now a textbox). The new custom layout is then applied to the slide and the contents of the selected textbox are copied and pasted into the new placeholder. The textbox is then deleted.

This is a more permanent solution (compared to the first option I presented) in that any change made to the indent levels of paragraphs in the textbox after it has been "converted" to a placeholder will now use the correct bullet formatting. The risk is that we have made changes to the rest of the slide (the formatting of contents of the other placeholders) that may not be desirable.

I present this here, firstly to share the idea and code with anyone who has been similarly frustrated by this issue (and frustrated by other users who can't learn to apply a layout with the right number of placeholders when constructing their slides, which is of course the real, if clunky, solution to this) and also to seek input from the community to help improve it - What did I miss? What are it's limitations? What improvements could be made to the code?

Please see my first potential solution here: Possible solution to copied placeholder losing bullet formatting (1 of 2).

Cheers
Rich

Here is the code:

Sub ConvertCopiedPlaceholderToPlaceholder()
' Code to "convert" a textbox (that was created by copying and pasting a
' placeholder and in doing so lost the formatting of multi-level bullets)
' into a placeholder.
'
' YOU MUST HAVE THE OFFENDING TEXTBOX SELECTED BEFORE
' RUNNING THIS CODE!
'
    Dim sld As Slide
    Dim cl As CustomLayout
    Dim shp As Shape, tb As Shape, ph As Shape
    Dim i As Long, lnPHCount  As Long
   
     'Check a textbox is selected:
    If ActiveWindow.Selection Is Nothing Then
        GoTo SelTB
    ElseIf ActiveWindow.Selection.Type = ppSelectionNone Then
        GoTo SelTB
    ElseIf ActiveWindow.Selection.Type = ppSelectionSlides Then
        GoTo SelTB
    ElseIf ActiveWindow.Selection.ShapeRange.Count > 1 Then
        GoTo SelTB
    ElseIf ActiveWindow.Selection.ShapeRange(1).Type <> msoTextBox Then
        GoTo SelTB
    End If
   
     'If all the above are passed, a single textbox is selected so OK to continue.
    Set tb = ActiveWindow.Selection.ShapeRange(1)
   
     'Put text in any empty placeholders:
    Set sld = ActiveWindow.View.Slide
    Set cl = sld.CustomLayout
    For Each shp In sld.Shapes.Placeholders
        If shp.HasTextFrame Then
            If shp.TextFrame.TextRange.Text = "" Then
                shp.TextFrame.TextRange.Text = shp.Name
            End If
        End If
    Next shp
   
     'Duplicate the custom layout:
    Set cl = cl.Duplicate
   
     'Delete all placeholders on the new layout:
    For i = cl.Shapes.Placeholders.Count To 1 Step -1
        cl.Shapes.Placeholders(i).Delete
    Next i
   
     'Add a placeholder on the new custom layout, one per placeholder
     'in the slide, in the same position and of the same placeholder type
     'as on the slide and copy formatting from the slide:
    For Each shp In sld.Shapes.Placeholders
        Set ph = cl.Shapes.AddPlaceholder(Type:=shp.PlaceholderFormat.Type, Left:=shp.Left, _
            Top:=shp.Top, Width:=shp.Width, Height:=shp.Height)
       
        If shp.HasTextFrame And (shp.PlaceholderFormat.ContainedType = msoTextBox _
            Or shp.PlaceholderFormat.ContainedType = msoAutoShape) Then
           
            ph.TextFrame.TextRange.Font.Size = shp.TextFrame.TextRange.Font.Size
            ph.TextFrame.TextRange.Font.Bold = shp.TextFrame.TextRange.Font.Bold
            ph.TextFrame.TextRange.Font.Color.RGB = shp.TextFrame.TextRange.Font.Color.RGB
            ph.TextFrame.TextRange.ParagraphFormat.Bullet.Visible = shp.TextFrame.TextRange.ParagraphFormat.Bullet.Visible
           
            For i = 1 To 5
                ph.TextFrame.Ruler.Levels(i).FirstMargin = shp.TextFrame.Ruler.Levels(i).FirstMargin
                ph.TextFrame.Ruler.Levels(i).LeftMargin = shp.TextFrame.Ruler.Levels(i).LeftMargin
            Next i
           
            ph.TextFrame.MarginBottom = shp.TextFrame.MarginBottom
            ph.TextFrame.MarginTop = shp.TextFrame.MarginTop
            ph.TextFrame.MarginLeft = shp.TextFrame.MarginLeft
            ph.TextFrame.MarginRight = shp.TextFrame.MarginRight            
            
        End If
    Next shp
   
     'Add a placeholder for the selected textbox:
    Set ph = cl.Shapes.AddPlaceholder(Type:=ppPlaceholderObject, Left:=tb.Left, _
        Top:=tb.Top, Width:=tb.Width, Height:=tb.Height)
    ph.TextFrame.TextRange.Font.Size = tb.TextFrame.TextRange.Font.Size
   
     'Apply the new custom layout to the slide:
    sld.CustomLayout = cl
   
     'Copy the text from the textbox to the new placeholder
    tb.TextFrame.TextRange.Copy
