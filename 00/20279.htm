Try this:

Sub SearchAllHeadersAndFooters()
    Dim rng As Word.Range, s As Word.Section, hf As Word.HeaderFooter
    For Each s In ActiveDocument.Sections
        'headers and footers are part of each section in a document
        'all sections must be searched
        For Each hf In s.Headers
            'there are 3 headers that will be searched (primary, even, & first page)
            Set rng = hf.Range
            If InStr(1, rng.Text, "your string") = True Then
                'perform your custom action
            End If
        Next hf
        For Each hf In s.Footers
            'there are 3 footers that will be searched (primary, even, & first page)
            Set rng = hf.Range
            If InStr(1, rng.Text, "your string") = True Then
                'perform your custom action
            End If
        Next hf
    Next s
End Sub

van dinh say:

I'm looking to get the true height of the form if it were opened properly instead of resized with scroll bars.

Can this be done?


It sounds to me that you want ot get the total height of the Form, whether it is visible in the Form "window" or not?

If so, you can simple add the height of each Section of the Form, e.g. for Normal Form View:

Dim lngFormHeight As long

With Forms![YourForm]

  lngFormHeight = .Section("secFormHeader").Height + _
    .Section("secDetail").Height + _
    .Section("secFormFooter").Height

End Width

 

(You will need to check the names of the Sections on your Form since the Section names above are not automatically assigned by Access - I tend to rename everything with prefixes).

 

If your Form is in Continuous Form View, then your code need to work out the number of records and then multiply the Detail Section height by the number of records.

mel77084 say:

My experience has been that updated drivers are best obtained through Windows Update, rather than directly from provider, because new drivers, at least in some cases, are provided to Microsoft before they're made available to the public.

Therefore, I would go into Device Manager and uninstall the graphics adapter and delete the driver for it, and then reboot. During the boot, the adapter will reinstall (perhaps using a driver that is already present, but wasn't being used in the previous installation). If there isn't such, the adapter will still work at a lower resolution than you normally would want, meaning everything will be larger than you want. Then, check for an updated driver through Windows Updates.

Even before getting a new driver, you may be able to increase the resolution some in Settings, Display, Advanced Display Settings. Once you get a new driver, you certainly will be able to do so.

Your current problem might be caused by the new driver that you got from the manufacturer not correctly installing due to the old one not getting properly uninstalled.

Display drivers can take several minutes to install, so a little patience might be needed. A reboot will be required, but sometimes an extra one seems to help settle things down.
