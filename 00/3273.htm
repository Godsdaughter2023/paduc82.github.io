migrating folders to outlook from verizon - Microsoft Community

terrencek say:

migrating folders to outlook from verizon

How do I migrate folders and groups to outlook from verizon

brian tillman [outlook mvp 2007-2019] say:

Outlook questions belong in the forum dedicated to your version of Office.
http://answers.microsoft.com/en-us/office/forum/office_2003-outlook
http://answers.microsoft.com/en-us/office/forum/office_2007-outlook
http://answers.microsoft.com/en-us/office/forum/office_2010-outlook

What type of mail service does Verizon support?  If it supports IMAP, create an IMAP account in Outlook and you'll be able to access your Verizon data in the mail server.  Outlook will see all mail folders on the server and you'll be done.

If it supports only POP, then create that type of account in Outlook and download all your Inbox data.  If you want to keep your data online as well instead of having Outlook delete it after downloading, make sure you specify the option in the account settings to leave copies of messages on the server.  Next. create a temporary folder in Outlook and move the contents of Inbox to that folder.  Now, log onto the Verizon's mail server via a web browser.  If you left copies on the server, create a temporary folder on the server and move to it the contents of the server-side Inbox.  Now, copy (not move) the contents of another of your server-side folders to the server-side Inbox and then in Outlook perform a send/receive.  Outlook will now download that data to its Inbox.  Create a folder in Outlook whose name matches the server-side folder that held the messages and move to it the messages from the Outlook inbox.  Delete the contents of the server-side Inbox.

Follow this same process for each of the other server-side folders you have.  Finally, move the contents of the temporary folder you first created on the server back to the server-side Inbox.  Outlook may re-download that data again when you do another send/receive.  If so, then just delete the temporary folder you first created in Outlook because it should contain nothing more than copies of the data in Outlook's Inbox at this point.  Otherwise, you can move the data from the temporary folder you first created in Outlook back to the Inbox and then delete the folder.  Outlook should now match in content what's on the Verizon server.
