Hello maweiss,

Determine the GUID of the installed Office by using the guide in the mS KB: http://support.microsoft.com/kb/2186281

The important value is the Release Type.  If the installed Office does not show as being from the Volume Licensing channel, then chances are it is authentic.

A1.  Yes.  Pirates buy software licenses under the Microsoft Developer Network (MSDN) program, which allows developers access to a wide range of Microsoft titles, to use the software for development or testing.  They then sell the one product key they get for each title numerous times.  The more often the PK is sold the more likely it is that MS activation servers will detect and block the abused PK.

A2.  Yes.  One of the main goals of software pirates who prepare and sell counterfeit product is to embed malicious programs into the counterfeit discs in order to spread malicious code to as many computers as possibe.  Do an internet search for "botnet."

A3.  Yes.  A large percentage of software on warezz sites and on file share networks have malware embedded into the files.
