I have a workbook that is intensely macro driven. I'm trying to find a way to determine if a macro was called by another macro. For example:

Sub Clear_Range()

    Sheets("Sheet1").Select

    Range("B1:D5").Select

 Sample1' Macro I want to call

    Range("A1").Select

End Sub

Sub Sample1()

'  I need code here that determines if this sub was called by the other one, and exit so the rest of this sub is ignored

    Selection.Clearcontents
End Sub

jasles say:

Add an optional parameter to Sample1, and then use the IsMissing function.

Sub Sample1(Optional wasCalled As Variant)

    If IsMissing(wasCalled) = False Then Exit Sub

    Selection.Clearcontents

End Sub

Then change the calling procedure to this by adding to the line that calls the Sample1 procedure:

Sub Clear_Range()

    Sheets("Sheet1").Select

    Range("B1:D5").Select

    Sample1("called")

    Range("A1").Select

End Sub
