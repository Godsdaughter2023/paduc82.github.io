    If issue persist then the issue could be due to hard drive errors. To check if your hard drive is causing the issue you can try running chkdsk command.

    You should periodically use the Check Disk tool to check the integrity of disks. Check Disk examines disks and can correct many types of common errors on FAT16, FAT32, and NTFS drives. One of the ways Check Disk locates errors is by comparing the volume bitmap with the disk sectors assigned to files in the file system. Check Disk can’t repair corrupted data within files that appear to be structurally intact,

    Note: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive, it tries to repair that sector, if any data available on that might be lost. 

    Steps to run CHKDSK command for Windows 8.

    1. Press Window key + X.
    2. Now select Command Prompt (Admin).
    3. Press OK the UAC prompt if shown. That will open an elevated command prompt. (It will show Administrator in the caption.)
    4. This will open command prompt tool.
    5. You'll see X:\Sources
    6. Type "chkdsk c:\" at the flashing prompt to begin the Chkdsk utility. Replace "c:\" with the letter of the drive for which you want to run the Chkdsk utility. If the hard drive is "d:\", then type that instead.
    7. After the "chkdsk c:\" command insert "\f" to fix errors on the disc volume or "\r" to recover bad sectors and then press Enter.
    8. Restart the computer once the chkdsk is completed.
