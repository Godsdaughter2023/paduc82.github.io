Do you have a Windows 10 Disc?

If so, follow these steps:

Backup your Data

Insert the Installation Media

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS)

If you have problems booting from a boot disc, you may have UEFI BIOS, go into your BIOS and make the following changes:
1. Insert the Bootable Media
2. Disable Secure Boot
3. Enable Legacy Boot
4. If Available enable CSM
5. If Required enable USB Boot
6. Move the device with the bootable disc to the top of the boot order
7. Save BIOS changes and restart your System

NOTE: during the installation process, your system will reboot, at that point, you may need to go into your BIOS and change the boot order so your system boots from your Hard Drive, otherwise the installation may start all over again - do not remove the installation media at this point

If you do not have Windows 10 Installation Media, click this link:
https://www.microsoft.com/en-gb/software-downlo...

If you previously had Windows 10 Home installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

If you pop open the Settings app and choose Privacy, you can see every program that is allowed to use your microphone, and you can turn the switch off for Cortana.

Thank you for the quick response. Below are the steps that you can follow to continue troubleshooting:

Step 1: Reset Windows Store cache. This will help clear the cache without changing your personal settings.

  1. In the Windows Search bar, type WSReset.
  2. Right-click the WSReset from the search result and select Run as administrator.
  3. A command window will open. Wait until you get a message that the process has been completed.
  4. Wait for a few minutes before launching the Windows Store.

Step 2: Disable the proxy server from the Internet Options.

  1. In the Windows Search bar, type Internet Options.
  2. Go to Connections tab.
  3. Click the Lan settings button and under Proxy server, uncheck the Use a proxy server for your LAN.
  4. Click OK > Apply then OK again to save changes.
  5. Launch the Windows Store.

After performing the steps above, you can hit us back and tell us the result.
