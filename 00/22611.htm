After you select SAVE you should get another dialog box asking where to save the file.  At the top of the box, it will show the folder where Windows intends to save the file and at the bottom is shows the filename that Windows intends to use  This is labeled "Save As".  Windows gets that filename from the server where you are downloading the file, but you can change it.  Windows gets the "Save In" folder from the last place it was told to save any downloaded file.  All you need to do is change the Save In folder to wherever you want it to save the file.  Once you do that, Windows will normally use that same folder for any future downloads until you change it.

drag window without maximizing it - Microsoft Community

winisue say:

drag window without maximizing it

Just started using Office 2010 on Windows 7.  When I have a document open in less than maximized size and try to drag it on my desktop, it always jumps to maximized size.  How do I prevent that from happening, so I can see more than one window at a time?

a. user say:

Hi,

There's a new feature in Windows 7 that resizes windows when your mouse arrow touches the sides of the screen. 

For instance, if you drag your document to the left side until your mouse arrow touches the screen's edge, your document's size will change to take up half your screen space. You can open another document and do the same thing on the right side. Now you'll have two documents side-by-side, each taking up half the screen, and you can view them both.

You'll notice, though, that if you drag your document to the top of the screen until your mouse touches the edge, the window will maximize completely. If you don't want to change your documents' sizes at all, make sure your mouse arrow doesn't touch those edges!

Hope that helps!
Kara
