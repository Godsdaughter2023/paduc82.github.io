Hi,

A slightly different VBA approach using a function. Call the function with this in a cell

=First42(A1)

It will return a maximum of 42 characters or less if the last word would be split.

Function First42(rng As Range) As String
Dim FindSpace As Long
If Mid(rng.Text, 43, 1) = Chr(32) Then
    First42 = Left(rng.Text, 42)
    Exit Function
End If
FindSpace = InStrRev(rng.Text, " ", 42)
First42 = Trim(Left(rng.Text, FindSpace))
End Function
