  1. Type Services.msc on the Cortana search bar.
  2. Scroll down to Windows Update, right click and select Stop.
  3. Launch the File Explorer.
  4. Go to This PC > Local Disc (C:) > Windows > SoftwareDistribution.

  6. Restart your PC.

This problem occurs if the computer has the following registry value configured when you set the printer to the Offline status:


Key: HKEY_CURRENT_CONFIG\System\CurrentControlSet\Control\Print\Printers\<print queue name>

To resolve this issue, right-click the new print queue in the Printers and Faxes tool, and then clear the Use Printer Offline setting. This resets the registry key to the correct value.
