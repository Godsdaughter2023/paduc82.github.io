If you trust the workbook, you can then enable editing. XLA and XLAM files are not part of this change, they will continue to not open in Excel.

 

How do I get the update? It depends on which version of Officeyou have.

  • Office 365 subscription (Click-to-Run)—install the latest updates

The Excel team has made a change in the behavior of certain file types to increase security. This change came in the security updates KB3115262, KB3170008, and KB3115322. How to insert text and photos in Publisher 2013 - Microsoft Community

geoffoto say:

How to insert text and photos in Publisher 2013

Is there a way to insert text or photos into an existing page such that existing text flows automatically to the next page instead of off the bottom margin?  Every time I add a photo, etc. into a page I have to manually cut the overflow and paste it onto the next page.  Very clunky.  I have to believe Publisher provides a simpler way.  
davidf2 say:

Publisher does not work the same as Word where overflow text is automatically moved to a new page. Publisher uses text boxes to allow for better page design and you can use the 'autoflow text' feature.

1. Insert a second page by Insert > Page  and probably a blank page.

2. Insert/draw a text box on the second page.

3. Return to the original text box on the first page that has the 'overflow' of text and click in the text box.

4. Click the square icon with three dots on the lower right side of the text box. If you mouseover that icon you will see 'Text in Overflow'. This will result in your cursor turning into a pitcher icon.

5. Go to the second page, click in the new text box that you created and the overflow text will 'flow' into the second text box.

You can click the arrow icons on the side of those linked text boxes to move between them. You can also break that link or link multiple empty text boxes prior to importing or inserting your text. After selecting a text box go to Text box tools tab and the Linking group.

If you insert or import a large text file such as a Word file into a text box Publisher will either offer to create additional text boxes and pages as necessary and autoflow that text, or in some versions will just do that without asking. That auto feature was changed in later versions of Publisher. If you want to try it just create a blank text box in a new publication, go to Insert > Insert file > browse to and select a Word file.

Scroll down this page to read about linking text boxes in Pub 2013: https://support.office.com/en-us/article/Basic-tasks-in-Publisher-2013-0e5ed249-1927-433f-a35c-63beb8216fcf?ui=en-US&rs=en-US&ad=US

Also reference: http://office.microsoft.com/en-us/publisher-help/create-link-is-under-text-box-tools-linking-HT102769754.aspx?CTT=1

phillip m jones say:

Was The image jpg? If so that the Problem. On PC Jpg is saved in a Compressed state and used in a compressed State.

On Mac jpg is saved compressed, but must be in order to used decompressed  That just nature on a UNIX based system. However if do the following to Jpeg Images as soon as you get them to use in Office. You'll end up with images That look just as good and are often up to 25% as large a File size.

  1. Open your image in any Graphics program capable of converting to another file type.

  2. First Sharpen Focus, and Save as jpeg (this will be iamge you work with.)

  3. Now Locate Menu where you can adjust size and DPI.

  4. The first thing to do reduce actual size to size of image you want in the Document.
    Say they sent you an 10 by 14" picture but you only need 5 by 7.
    Change size to 5 by 7 and save as another version of File.
    Now save this time, in PNG format.

  5. Next Locate adjustment for DPI or PPI
    save using theses numbers depending up on use:

    Printed Report meant to impress 200-300 dpi (ppi).
    To be show in computer screen as work or excel document, or PowerPoint Presentation Computer Only no more than 75 dpi/ppi tops.
    Presentation to be feed into a projector 150 to no more than 200 dpi/ppi

  6. Save these Files as .png. files.
    (Both Office Mac and PC can use jpg and png equally well.)

  7. Now save final size under a different name and use the images instead of the .jpeg images

Png has the advantage that it is a lossless format. There is no degradation in quality whether going up or down in size as long as kept proportional, quality is affected on non proportional only because your stretching or squeezing the shape of the pixels, they are not lost. JPEG is a Lossy format that each time its compressed or decompressed pixels are thrown away and therefore the image is degraded with each use.

PNG is based on GIF format but is open source (a royalty free format). If you convert JPEG to PNG and change in the right order, you can get as high quality image, that looks as good as original images.

On a JPEG image if after you have opened four or 5 times and zoom in at 1000 percent, you'll notice around the edges an effect that look like 35mm film that has been overheated because the fan in the projector is not working right. It will be smeared and look like some kind of liquid is being poured around the edges of images.
