considerate_guy say: If you are going to do the conversion, do it before starting the project.  Convert to .wmv since that is what Movie Maker prefers.  That way you eliminate the need for additional codec packs.  I personally feel it is bad to load a bunch of codec packs.  Apart from loading your system with possibly buggy codecs, they have also been used by virus makers to introduce viruses into your system. For more details read this article from LifeHacker:
http://lifehacker.com/5877208/what-are-codec-packs-and-should-i-use-them

You can read the project files from WMM 6.0 into WLMM.  But remember a project file contains no video/audio files within it.  A project file just contains a list of files used in your project along with the instructions to order the clips, apply effects, add transitions and titles, etc.  It's like a recipe for creating your final movie.  So whether you use WMM 6.0 or you use WLMM, there's no advantage, really, in bring a project from one to the other.  (By the way, you can't go backwards from a WLMM project back to WMM 6.0).
