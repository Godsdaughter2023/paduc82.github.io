martin pratt say:

Solved: Bulk move deleted items back to Inbox

In common with many others I accidentally deleted 4000+ emails that were then stuck in my deleted items folder with no apparent way of moving them back in bulk (only a few could be selected at a time). The main reason for this seems to be calendar items that are mixed in with the emails, when you try and move in bulk it wants to create a new item.

There's an easy fix.

1) Go to your "Deleted items" box in the desktop version of Outlook

2) Select all items in there

3) Sort by "Type" as follows:

4) Now find and select all calendar entries (you'll see a calendar icon or similar next to them)

5) Right click on the selection and choose "Delete"

6) Now select all the remaining items in your "Deleted Items" folder > right click on your mouse > select "Move" > select "Inbox"

7) Problem solved!

If this doesn't work immediately (as for me) you have calendar items remaining. Just select these and delete and go back to (6).
