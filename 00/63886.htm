Freezing can be so many issues. If you're sure it's clear of viruses, have the latest drivers, and the disk is healthy then it could be:

Other hardware failure

Other processes running on the machine - try opening Task Manager and keep an eye on CPU and Memory usage. Do you notice if the laptop heats up or the fans become noisy when this happens?

Major OS problem

It might be worth looking in Event Viewer but I'm not sure you'll find much. It could be that you are getting silent bluescreens which should show up some errors in Event Viewer.

shaunmoncrieff say:

You can upgrade Windows 8.1 with Bing devices to Windows 10.

The issues you may encounter are as follows:

-Limited or no access to Manufacturers Drivers or applications (such as screen rotation apps) for Windows 8.1 with Bing or for Windows 10 (makes it difficult to do a clean install of Windows 10)

-limit on hard drive space available (for 16GB Devices) to do an in-place upgrade

-availability of all Windows 10 drivers for your specific device 

If you still want to upgrade to Windows 10 then please follow the steps below:

Firstly make sure your Windows 8.1 with Bing device has been activated with Microsoft and that you have created recovery media to allow you to re-install Windows 8.1 with Bing to your device if something goes wrong or Windows 10 does not work correctly with your device. Instructions to create a recovery media can be found here:

http://windows.microsoft.com/en-US/windows-8/create-usb-recovery-drive

Then download and create your own USB installation media for the Windows 10 Home edition 32-bit version as per instructions found here:

https://www.microsoft.com/en-us/software-download/windows10

Once you have created the USB install media for windows 10 Home 32-bit run the setup.exe located on the USB from within Window 8.1 with Bing. Do not boot to the USB media as this will do a clean installation and you will not be able to activate Windows 10. You need to do at least one in-place upgrade in order to activate Windows 10 using your activated install of Windows 8.1 with Bing.

Make sure you choose to do an Upgrade as this will keep all drivers and applications installed as part of your Windows 8.1 with Bing installation and use the activation for Windows 8.1 with Bing to allow activation for Windows 10. 

If you receive a warning during the Upgrade that there is not enough free space on the device to complete the upgrade then select the USB drive (that has the Windows 10 installation files on it) as the drive to use for temporary installation and recovery files.

After you have successfully upgrade to Windows 10 then you need to connect to the internet and ensure that the Windows 10 installation has been activated with Microsoft. (Once Windows 10 has been activated with Microsoft them you can re-install Windows 10 at any time and it will re-activate).

You should also run Microsoft Update from the Windows 10 Settings to get the latest Windows 10 Updates and any Windows 10 Drivers available for your device.

Hopefully this helps.
