d0pp13r say: awright, i finally managed to solve this together with ross barkman. for my setup seems to be decently stable, it works like this:

1) NOKIA UMTS CID 1 script
2) IPv6, compression, error corection switched off
3) echo either on or off (preferably off)
4) DNS servers ans serach domains pre-filled.
5) no telephone/user/pass information in network preferences, only in Internet Connect (see below)


the resulting connection seems to be both stable and fast, according to TPTEST (http://tptest.iis.se/lista_artiklar.asp?lngCategoryId=1307) conected to tptest.telenor.se in stockholm, the TCP/UDP speed is 213/399 kbps, which is a tad more than expected. translated to everyday langueage, this equals about 40k/s for downloads and about half for uploads. the DL speed starts at 10k/s and then increases until the maximium level within one minute.

the trick in order to make my setup work was to download connect2net from nova media and check the internet connect log using verbose logging after a successful connection. in the internet connect log, you can clearly see what string that is used for connecting. in my case it was /// CCLWrite : AT+CGDCONT=1,"IP","internet.vodafone.net","0.0.0.0",0,0\13 /// the information "internet.vodafone.net" is very relevant, since this is the so-called APN. the APN is what you use in the field "Telephone Number" in the application Internet Connect. user/pass is then set to "guest/guest", alternatively left out blank entirely.

d0pp13r say: there is also an alternative way to connect for those who do not have ross barkmans NOKIA UMTS CID 1 script. you can use the nokia infrared and specify the APN in the data connection preferences of the cellphone instead. in the "Telephone Number" field of Internet Connect, you should now instead use the default "*99#" or similar. i havent tried this myself, but it should be a reliable way to connect as well. i assume error correction and compression should be switched off.

good luck!

jay freedman say:
Without a macro (and I do like macros, but they aren't always needed):

  • Open the Styles pane, by clicking the little dialog launcher arrow in the bottom right corner of the Styles group on the Home tab or by pressing Ctrl+Alt+Shift+S.
  • Right-click the name of the Caption style (or other style if it was applied to the captions) and look at the Select All item in the dropdown menu.
  • If it says (No Data), which it will the first time you do this after opening a document, then click it and right-click it again.

If you want a count of the number of occurrences in all the documents in a folder, that will require a macro, which would be an extension of the one Andrea showed. Essentially, it would open each document in turn, get the count for that document, and add it to a running total.
