a. user say:

Hi,

It could be an outdated driver of your Microphone. Driver updates for Windows 10 and many devices (such as network adapters, monitors, printers, video cards, etc.) are automatically downloaded and installed through Windows Update. While it's likely you already have the most recent driver, if you're having trouble with a specific device (like your Mic), you can try fixing it by updating the driver or reinstalling it. Here's how:

  1. Select the Start Windows logo, type Device Manager, and select it from the list of results.
  2. Expand one of the categories to find the name of your device, then right-click (or tap and hold) it, and select Update Driver.
  3. Select Search automatically for updated driver software.
  4. If Windows doesn't find a new driver, you can try looking for one on the device manufacturer's website and follow their instructions.
  5. If these steps don't work, try reinstalling the driver: Open Device Manager, right-click (or tap and hold) the name of the device, and select Uninstall. Restart your PC and Windows will attempt to reinstall the driver.

You can also check within the Skype app if your Mic is working properly by making a test call. Locate Echo / Sound Test Service contact. It will prompt you to record a message, and then play it back for you – so you’ll know right away if your sound is working. If you can't hear your voice - your Mic is not working.

If updating your Mic's driver won't work, I suggest to contact your device manufacturer to check this for you.

To set up shared access of the QuickBooks company files, perform the following steps:


   1. On the computer that will act as the QuickBooks server, open Windows Explorer and right-click on the folder into which you installed QuickBooks Pro.

   2. Choose Sharing from the shortcut menu to open the Folder Properties dialog box.

   3. Select Shared As. The folder name is automatically entered as the Share Name, and that is probably a good choice. However, you can change the name of the share if you wish.

   4. Set permissions for Full Control.

   5. Click OK.

If you're using Windows 98 or Windows ME, the permissions are configured on the Sharing tab of the Properties dialog box. If you're using Windows NT, Windows 2000, or Windows XP, click the Permissions button on the Sharing tab to use the Permissions dialog box. It's a good idea to map the shared folder as a drive on the other users' computers. It's not necessary, but it speeds up access to the company files:

   1. On each user’s computer, open Network Neighborhood and select the remote computer that's acting as the server for QuickBooks Pro.

   2. Right-click the shared resource for QuickBooks.

   3. Choose Map Network Drive from the shortcut menu.

   4. In the Map Network Drive dialog box, select a drive letter for this resource.

   5. Select the Reconnect at logon option.

   6. Click OK.
