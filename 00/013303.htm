It took some time, but here is my final report to the issue: The "Transfer my Data"-App from Nokia is indeed compatible with the N9 and is able to transfer SMS. But there's a catch: It is not possible to transfer more than 2048 messages using that app.

This workaround takes a while, but who cares: you only have to do it once and it allowed me to transfer thousands of text messages.

joeu2004 say: RT wrote:
I was wondering if someone could help me with an IF(OR formula
[....]
The criteria to return a "No" result
A2= "Varies" or "Sales"      OR      A2 =B2

The criteria to return a "Yes" result
A2 <>B2     OR       B2="#N/A"

I presume that "#N/A" refers to the #N/A Excel error.  You must test for that first and separately; otherwise, the IF() function will return a #N/A error itself.

Ostensibly, you might write:

=IF(ISNA(B2),"Yes",IF(OR(A2={"Varies","Sales"},A2=B2),"No","Yes"))

Alternatively, in XL2007 and later, you could write:

=IFERROR(IF(OR(A2={"Varies","Sales"},A2=B2),"No","Yes"),"Yes")
