uk_michael say:

I managed to get file History to backup my SD card by first formatting it in NTFS and then adding Authenticated User to the permissions.

Right click the NTFS formatted SD card and select Properties then the Security Tab, under Group or User name click the Edit button then Add and enter Authenticated User followed by Check Names, OK etc.

Then create a folder, such as SD-Documents, and right click this folder and select Include in library > Documents or what ever library you want the folder added to.
