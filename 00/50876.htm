Hi,

You could face this issue when the audio driver is not installed or if it is out dated. You could also face this issue when the Microphone is not set as default device. Please write to us with the following detail.

1) What is the make and model of your Laptop?

2) Did you make any changes to Microphone settings prior to this issue?

As to have mentioned that you started to face this issue after you installed Windows 8.1 Pro, make sure that you have all the Windows Updates installed on your computer. For this

  1. Windows Key + C Key together.
  2. Settings, Click on Change PC settings.     
  3. Update and recovery and click Check now.

  4. Method 1: Run Recording Audio Troubleshooter

     

    Please follow the steps given below to run the Recording Audio troubleshooter.

     

                a) Press Windows key + W key on your desktop.

                b) Write Troubleshooting in the search box on the upper right hand corner.

                c) Select the Troubleshooting option obtained after the search.

                d) Select View All option from the left panel.

                e) Select Recording Audio option.

                f) Click on Next to perform the Recording Audio troubleshooter and follow the steps.

    Check the issue and move to next method if the issue persists.

    Method 2: Make sure that the Microphone is not disabled and is set as default.

    Make sure that the microphone that you are using is not disabled and is set as default in the computer. Follow these steps to check this:

    a) Right click on the volume icon and select “Recording devices”.

    b) Now, right click on an empty space and select, “Show disconnected devices” and “Show disabled devices”.

    c) Select “Microphone” and click on “Properties” and make sure that the microphone is enabled.

    d) You may also check if the microphone that you are using is set as default.

     

    Let us know the status of the issue, we will assist you further.
