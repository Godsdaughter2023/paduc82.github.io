A tip when using a monitor and TV/monitor dual monitor setup with - Microsoft Community

cbarnhorst say:

A tip when using a monitor and TV/monitor dual monitor setup with Windows

I use a dual monitor setup where the second monitor is also a TV.  Every time I install a new version of Windows (every six months, natch) I lose the sound on my main monitor whenever I watch TV.  Here is what is happening.

The main monitor is attached to my GTX 1050ti card via displayport and the TV/monitor is attached via HDMI.  I learned a long time ago when I had a similar setup using a DVI and VGA connector that Windows Setup defaults to the oldest video protocol available during Stage 1 of installation up to when it installs a video driver and stops using the default Windows driver.  Then it switches display to the newer protocol.  So I would see Stage 1 until the first restart and for a minute or so into Stage 2 and then Setup.exe would switch to the other monitor.  This is by design to ensure the largest number of computers display Setup correctly before a third-party video driver can be installed.  The assumption being that the oldest protocol is the likeliest to be successful on the largest number of devices.

The same thing still happens.  HDMI is older than displayport so HDMI get first usage.  Then when the third-party driver for the video card is installed the display switches.  This effect is also present when restarting the system.  The main monitor is black until the boot sequence loads Windows and any boot display is visible first on the second ("older") monitor. 

As it turns out, it doesn't also switch the default sound device to the displayport connector at Stage 2 or after.  There may be other features affected but I haven't noticed that.

The solution is to disconnect the TV/monitor before starting Setup or change the default sound device after.  The later is easier because the user doesn't have to set up for dual monitor after installation.

This tip is to alert you to what is going on if you are puzzled by this change to your sound settings after an installation.
