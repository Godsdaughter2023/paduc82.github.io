Inserting spreadsheet into a word document - Microsoft Community

rraybould say:

Inserting spreadsheet into a word document

Where you want to insert a spreadsheet into a word doc  and it is longer than one page length of the document, is there any way of automatically dividing it into page length pieces including new headings for each page?

Thanks

 

marysully say: Instead of inserting the spreadsheet as an object you can always copy and paste the spreadsheet into a Word document where it will automatically break into pages as a table. If you want the same heading on each page you can create a table heading which will appear on each page. Just select the row you want as the heading, right click and choose "Table Properties" from the menu, click on the "Row" tab and under "Options" check box for "Repeat as header row at top of each page."

doug robbins - mvp office apps & services (word) say:

No. The spreadsheet is inserted as an Object and Objects cannot span more than one page.

You would need to insert the two sections (selected areas) of the spreadsheet as separate items.
