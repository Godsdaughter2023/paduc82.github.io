hansv mvp say:

Here is a variation on code from Rick Rothstein (see Excel Number Format: Indian Style Comma Separation):

 

Sub FormatIndianWhole()
    IndianNumberFormat 0
End Sub
 
Sub FormatIndian2Decimals()
    IndianNumberFormat 2
End Sub
 
Sub IndianNumberFormat(n As Long)
    Dim C As Range
    Dim S As String
    For Each c In Selection
        S = Trim(Replace(Format(String(Len(Int(C.Value)) - 1, "#"), _
            " @@\\,@@\\,@@\\,@@\\,@@\\,@@\\,@@0"), " \,", ""))
        If n > 0 Then
            S = S & "." & String(n, "0")
        End If
        C.NumberFormat = S
    Next C
End Sub

 

Select the cells you want to format. If you want to format them as whole numbers, run FormatIndianWhole, or if you want to format them with 2 decimal places, run FormatIndian2Decimals.

 

The format will work correctly for numbers up to and including 999999999999999.

a@work say:

How do I create a macro to delete unused master layouts?

Hi All,

Like the thread title says, how do I create a macro to delete unused master layouts?

 

Logically, I would want to detect each slide master, then each layout within the master. Then check to see the layout has been used on any slides in the active presentation. If it was not, then delete it. then move on to the next master. I can't seem to figure out how to find which layouts are unused.

 

Thoughts?

 

Thanks!

shyam pillai say:

You can use this:

Sub CleanupDesigns()
    Dim I As Integer
    Dim J As Integer
    Dim oPres As Presentation
    Set oPres = ActivePresentation
    On Error Resume Next
    With oPres
        For I = 1 To .Designs.Count
            For J = .Designs(I).SlideMaster.CustomLayouts.Count To 1 Step -1
                .Designs(I).SlideMaster.CustomLayouts(J).Delete
            Next
        Next I
    End With
End Sub

 

Alternately, you can can enum thru all the slides in the presentation and create a collection of layouts used and then delete the rest too.
