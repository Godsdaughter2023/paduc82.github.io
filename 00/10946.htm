    Conditional formating only if a value is placed in a cell - Microsoft Community

    damoncartmell say:

    Conditional formating only if a value is placed in a cell

    Hello,

    I am trying to build a spreadsheet to make a task easier for my operators. I am using conditional formatting with two separate rules. The first highlights the cell if a numerical value is between x and y. The second highlights the same cell with a different format if the value is not between x and y. My problem is that when I copy the conditional formatting to other cells they are being highlighted when the cell is blank. I would like the cell to stay white until a value is input and then have the formatting apply once a value is entered.


    Is this possible?

    In case it helps what we are doing is checking the incoming weight of a product against an expected tolerance for that weight. If the weight is within an acceptable +/- range I want the cell to be green. Conversely if a value is input that is above or below the tolerance I want the cell to be red.

    Thanks in advance for any help.  

    ashish mathur say:

    Hi,

    For green colouring, use this formula in the conditional formatting dialog box

    =AND(A2>=x,A2<=y)

    For red colouring, use this formula in the conditional formatting dialog box

    =AND(A2<>"",OR(A2<=x,A2>=y))

    x is the cell with the lower limit number

    y is the cell with the upper limit number

    Hope this helps.
