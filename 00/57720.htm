a. user say:

Hello Ling,

Thank you for contacting Microsoft Community.

From the screenshot, I guess you're using Lenovo product. Let me clarify what all these partitions are and chances of removing them.

EFI system Partition (I.e. Extensible Firmware Interface): EFI System partition is used by computers adhering to the Unified Extensible Firmware Interface. When system powered on, UEFI firmware loads files stored on the EFI System Partition to start operating systems from the place where it is installed and other various utilities. This will be removed when you perform new install.

OEM Partition: OEM partition is most likely your device Manufacturers Utility Partition. It essentially contains diagnostic tools that you can access via BIOS. This partition won’t require for recovery or a factory restore. I guess you can also download the same exact utilities from Manufacture’s website and put them on a CD if you need them. If you want to remove this, please take help of your device Manufacturer for doing the same.

Recovery Partition: You run into problems with your PC, a recovery partition can help you restore your PC to factory state. Removing this will no longer allow you to restore your PC to Factory state.
