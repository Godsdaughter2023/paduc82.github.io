Function MonthLength()
    Select Case Forms!frmMonthlyRosterMenu!Month
        Case 1, 3, 5, 7, 8, 10, 12 ' Months with 31 Days
            MonthLength = 31
        Case 4, 6, 9, 11 ' Months with 30 Days
            MonthLength = 30
        Case Else
            If DatePart("m", DateAdd("d", 28, DateSerial(Forms!frmMonthlyRosterMenu!Year, Forms!frmMonthlyRosterMenu!Month, 1))) = 2 Then
                MonthLength = 29
               Else: MonthLength = 28
            End If
    End Select
End Function

Unless it's a typo, you may have modified the wrong template. Word 2008's source file is Normal.dotm, not Normal.dot, so what you modified may be an old template left over from a previous version of Word.

Another possibility is that you navigated to the template in Finder & double-clicked it. That results in creating a new document, it doesn't open the template file. To open Normal.dotm you must use the Open command from within Word's File menu.

However, there is no actual need to open the Normal.dotm template to make such a change. Instead;

  1. Launch Word then go to Format> Style
  2. Select Normal in the Styles list & click Modify
  3. Click the Format button & select Paragraph
  4. Change the necessary specifications then click OK
  5. In the Modify Style dialog make sure the box is checked for Close
  6. Quit Word - If prompted to save changes to the normal template click Yes
