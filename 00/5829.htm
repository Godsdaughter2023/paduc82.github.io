jim gordon mvp say:

One of the images should be a copy of the image that was inserted into the document as it was inserted.

The other image is how the image appears in the document after you have applied changes such as formatting, shadow, rotation, etc.

In a Word document saved in Word format, only the original picture is saved (just once) and word generates a display picture with your Word applied modifications on-screen and for printing on the fly. That way if you copy and paste a picture within a Word document, Word doesn't make another copy of the picture. In a Word document, repeated representations of the same picture use the same reference picture. This scheme reduces file size when pictures are re-used within a document. PowerPoint is like that, too. But HTML doesn't allow that sort of economy, so the original picture is saved for reference purposes if you should decide to re-open the HTML file in Word, and Word is forced to generate another picture to display within the web page so that it matches what it looks like in Word.

There's both a 64-bit and 32-bit version of the WMDC. But I think the automatic install should address your concerns on identifying the needed version.

b) what procedure I use to set up the partnership on the new PC, to ensure that I don't lose the data from the phone, and that I can synchronise that back to the old pst file (on the new machine).

The first thing you'll want to do is to get your outlook PST restored to your new machine. Depending on which version of outlook you have installed there may be some slight variation in the wording on the screens you see but you can restore the PST contents by doing the following:

Importing old PST information into Outlook

  1. Open Outlook
  2. Open the "File" menu and select "Import/Export"
  3. Select "Import from another program or File" and select "Next"
  4. Select" Personal Folder File (.pst)" and select "Next"
  5. Click on "Browse" to select the location of your PST.
  6. You'll need to select whther the items in the PST your are importing have priority over the ones you may have recreated on your computer (if so select "Replace duplicates with items imported"). Otherwise you have the option of preserving both the items in the PST you are importing and the ones you've created on your PC ("Allow duplicates to be created") or you can give items you've already created on the PC priority ("Do not import duplicates")
  7. Select the items you would like to import. If you are only importing Contacts then select the Contact folder. If you want to import everything (Contacts, old e-mails, calendar items, so on) then select the very first item in the list of folders to import and make sure the "Include Subfolders" checkbox is checked. 
  8. Select "Finish" and wait while your items import. 

Creating new Partnership

  1. Connect your phone to your computer
  2. Wait while the WMDC installs
  3. You may be prompted to accept a license while this occurring. If so accept the license
  4. After the installation finishes the Partnership Setup begins. If your device has two partnerships on it you may be prompted to erase one (erase the one from the computer you no longer use). 
  5. If you are prompted on how to handle duplicates for this first sync choose the option to over write items on your PC with items on your phone. Since you've kept the phone up to date this will preserve those items. After the sync is done if you want the edits on the PC to have priority this setting can be changed. 

    (side note, since you have a backup of the items on the PC in that backup PST if you later decide you needed information from the PC version of a contact you can retrieve it from your original PST)
  6. Let the sync finish. After it's done you'll have all of your information on your PC and Windows Phone. 

BTW: Have you ever used the Microsoft  MyPhone service? If you have an unlimited data subscription you may want to check it out. If you use this service (it's free) it will do nightly backups of your contact data, text messages, and other data on your phone.

I'm not sure that is the correct interpretation.

But definition, a system of circular references has no beginning or end.  But Excel must stop the iteration with some cell.  For some systems of CRs, that might result in some inconsistencies -- calculations based on a previous iteration's results.

The "change" between Excel versions might simply be a difference in where Excel chose to stop the iteration.  That might be due to subtle differences in the way the dependency graph is constructed or processed.  Or it might simply be "dumb luck".

Without seeing your system of CRs, I cannot say for sure that that explains what you see.  But it has explained CR inconsistencies that I helped people with in the past.

ken sheridan say:
I do not want a new record automatically created on the subform



It won't be.  Once a new row is inserted into a form whose AllowAdditions property is True the form will show an empty row ready for another row to be inserted.  No new row is inserted into the table, however, until data is inserted into the empty row in the form.

 

If you don't want to see the empty row then you should set the subform's AllowAdditions property to False (No), and in the code behind the 'Create a new Assessment' button first set the AllowAdditions property to True before navigating to a new row.  Then in the subform's AfterInsert event procedure set the property back to False.

a. user say:

Try a System Restore to a point in timeBEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesn’t work. The recovery disk works a bit different from the above procedures but if you follow the prompts from the System Restore menu option with the above information you should be able to restore with no problems.  I've never tried to recover add-ons before so I don't know if it will work, but it might and is worth a try.

Otherwise, I think you're pretty much out of luck short of downloading and re-installing them all again (and you may need to uninstall and reinstall IE if that is possible to get the ones that come with the program though a reset may accomplish the same thing - I've never tried to do it for this purpose before so I never checked how reset functioned in this capacity).  There's no real way to re-install them even if we were able to actually recover the files (or to know which one goes with which add-on).  They make changes throughout the system and in the registry and you wouldn't have the faintest clue what those changes were, where those files should be located, or how to install the add-0n without an installation program.  Let's hope the System Restore works.

I hope this helps.

Good luck!
