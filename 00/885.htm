Typing in Columns - Microsoft Community

buzzls say:

Typing in Columns

I am typing a report in a landscape orientation, with 3 columns of text.   On some pages, the first column fills text vertically to the bottom of the page before starting the next column.  However, on others, when I enter several lines of text, the text starts to appear on the next column.   How can I keep the text flowing to the bottom of the page before starting the next column?  Thanks in advance!


suzanne s. barnhill say:

This will happen if you have a section break at the end of the multicolumn section. Word's default action is to "balance" snaking columns. If you don't need the section break (which could have been inserted for the purpose of having content in a single column spanning all the columns), you can delete it (note that you'll then have to reselect the three-column setting for your remaining section).

If you need to keep the section break, go to Office Button | Word Options | Advanced and scroll all the way to the bottom, where it says Compatibility Options and click on the + for Layout Options. Check the box for "Don't balance columns for Continuous section starts."
