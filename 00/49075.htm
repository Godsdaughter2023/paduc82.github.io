Kinguin is notorious for selling non-genuine licenses that belong to MSDN subscriptions.\

Open Start, type: CMD

Hit Enter

https://answers.microsoft.com/en-us/windows/wik...

About MSDN licenses:

They are genuine Microsoft product keys, they are actually retail licenses, but it is intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. The main purpose is for evaluation purposes. The great thing about them, unlike trial Microsoft software, MSDN or TechNet keys don't expire. Because the agreement under which the subscription is provided is a single license, none of the software should be distributed outside of it. Even though its $50,000 worth of licenses, it is for one person only to use and no one else. Unfortunately, regardless of the licensing terms, persons still abuse the program, either giveaway product keys or resell it on auction sites. Microsoft licenses it in good faith that customers won't do so, but I guess human nature wins out.

In your case, what probably happened is, you bought an MSDN licensed key, which carries up to 10 activations unlike full packaged retail licenses which only carry 1 activation. The person who sold it to you probably sold it to 10 other persons. Somewhere along the way, one of those persons might have installed it on a second system, activated it, because it went past 10 activation threshold, Microsoft detected it that it was being abused and blocked the key from further use.

gerry c j cornell say:

Nicki

 

If you now do not have a System Restore point that means there were none to remove when you ran Disk CleanUp. Neither Disk CleanUp nor cCleaner remove all restore points, both programmes always leave one restore point.

 

Some good news is that you do have a 25gb recovery partition after all. That means you can restore to factory settings if you need to. I would still favour doing an in place upgrade, if possible, as it leaves you with less work to restore your computer to the way it was before your problems started. It also leaves you with the ability to restore to factory settings using the copy of Windows 7 in the recovery partition. It is, however a more involved procedure and it does not always work. It worked for me the one time I have needed to it.

 

The method of doing an In Place Upgrade described in KB2255099 works well if you have an installation DVD, which is a complete copy of Windows 7, not an upgrade disk, and it is a disk that has the SP1 update included. If you do not have a disk that has a complete copy of Windows 7 with the SP1 update, you can still do an In Place Upgrade by using a copy of Windows 7 downloaded from the internet. If you think you need to download a copy of Windows 7 please do what follows.

 

What you need is an ISO copy of Windows 7 Home Premium 64 bit with SP1 included. This is freely downloadable from http://www.heidoc.net/joomla/technology-science/microsoft/14-windows-7-direct-download-links

 

This is the one you need -Windows 7 Home Premium x64 SP1 (media refresh) (English language version –(other languages are available)

http://msft.digitalrivercontent.net/win/X17-58997.iso

 

You need to down the copy of Windows 7 to your D partition (not your C partition). After downloading make sure the copy you have downloaded is not zipped and extract if it is.

 

The instructions to follow are at 4. Repair Installed Windows 7 SP1 using Windows 7 with SP1 ISO

http://www.sevenforums.com/tutorials/3413-repair-install.html

 

Once you have completed the instructions in Step 4 you move to Step 7 and then work down the list. The process takes several hours so if you choose to try it make sure you set aside sufficient time to complete the Upgrade. When it has completed you will need to run Windows Update to bring the system completely up to date. The updates are all those post the SP1 update so that is a lot of updates. To safeguard against something going wrong I recommend you back up your data files using Windows BackUp and Restore to external media before you start.

 

To activate use the product key you found earlier.
