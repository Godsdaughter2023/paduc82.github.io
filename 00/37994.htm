a. user say:

Hello Emca,

I would be happy to assist you.

If you already used an Microsoft Account to sign in to your activated Windows 10 Home or Pro PC, your MSA will be automatically linked. You can use this MSA linked digital license to re-activate your Genuine Windows 10 device by running the Activation troubleshooter, if you run into Activation issues caused by hardware changes.

Using the same Microsoft Account on multiple devices does not imply that the same digital license will work across all the Windows devices with same Microsoft Account. Because, digital licenses are unique based on the hardware signatures of the PC you are using.

The advantage of using a single Microsoft Account on multiple devices (upto 5), is that all the Microsoft services and the user settings will be synced across devices.

Hope the above information holds good.
