Update Cross References in WORD 2011 (MAC) - Microsoft Community

tsod say:

Update Cross References in WORD 2011 (MAC)

Hello everyone,

I am suing Word for MAC 2011.
When updating my document, all cross references (table, figure, etc.) are printed bold. Where can I change the format? I want the cross references to be formated in the standard text format. 

Thank you very much in advance


john mcghie say:
This was annoying me intensely the other day. Word "passes through" any direct font formatting from the target of the cross-reference.

For example: if I have a Caption style that sets a bold font, and I cross reference to Table 1, all is OK.  If I change the label (Table 1) to be "Not Bold", Word will pass through the command "Toggle Bold".

"Toggle" means "If it's on, turn it off, if it's off, turn it on."

So your cross reference will appear in text in a normal font, and the "toggle bold" command will turn the cross reference bold.  This will happen only if you manually override the formatitng of your caption.  For example, my current style guide requires the "Label" of a caption to be normal face and the 'text' of the caption to be bold.  I override the style on the Caption to set the label to normal text.  That causes the problem.

To control this, you need to add a switch to the cross-reference field to override the formatting.

The switch you need is " \*MERGEFORMAT".  If you set your document to View Field Codes, you will find that each cross-reference looks like this: { REF _Ref198307014 }

The curly braces are not actually "characters", they are "Field bounding delimiters" that are generated by Word, you can't type them, but you can create them if you need to.

The active ingredient is "REF" which indicates that a little piece of computer code is being called at that position.  The code searches the document for a bookmark named "_Ref198307014".  (The underscore simply means that the bookmark is "hidden").

You need to insert the switch after the bookmark name.  
{ REF _Ref198307014  \*MERGEFORMAT }

Then you remove the character formating from the cross-reference (select it and Control + SpaceBar) and update the field.  Whatever formatting you have now set for the cross-reference will be held whenever the field updates.

Hope this helps
