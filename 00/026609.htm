If you have Multi-Select turned on, the listbox itself doesn't have a value. You have to loop through the ItemsSelected collection. Here is an example of such code: 

Dim frm As Form, ctl As Control

    Set frm = Form!formname
    Set ctl = frm!listboxname
    For Each varItem In ctl.ItemsSelected
        place code here to populate table
    Next varItem

So you need to use VBA code to do this. Maybe, if you explain what you are trying to do in detail we can offer alternatives.

scottgem say:
Ok, So I assume those three values are in the listbox. So your code would look like this: 

Dim frm As Form, ctl As Control
Dim varItem As Variant
Dim strSQL As String
    Set frm = Form!formname
    Set ctl = frm!listboxname
    For Each varItem In ctl.ItemsSelected
        strSQL = "INSERT INTO FaultReport (Activity, PluginsUsed, SoftwareRunning) " & _
                      "VALUES('" & varitem.Column(x) & "', '" & varitem.Column(y) & "', '" & _
                       varitem.Column(z) & "');"
        CurrentDB.Execute strSQL
    Next varItem

Where x, y and z, are the respective column indexs from the listbox (Note the column index starts with so the first column is Column(0).
