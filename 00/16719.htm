hansv mvp say:

The keyboard shortcuts don't set spacing to a specific number of points, but to a number of lines.

Here is a macro for (at least) 5 points line spacing:

Sub Space5pt()
    With Selection.ParagraphFormat
        .LineSpacingRule = wdLineSpaceAtLeast
        .LineSpacing = 5
    End With
End Sub

and one for setting the spacing to 5 lines (which is a lot!):

Sub Space5lines()
    With Selection.ParagraphFormat
        .LineSpacingRule = wdLineSpaceMultiple
        .LineSpacing = LinesToPoints(5)
    End With
End Sub

You can copy either of these (or both) to a module in your default template Normal.dotm and assign a keyboard shortcut to them.

If you have not added the accounts yet, I suggest you to turn off/on the phone (and remove the battery), try in wifi and 3g, try an hard reset: Settings > about > hard reset. (it deletes any personal data, but you can save music and pictures first into the PC, in this link you can find more info: http://www.windowsphone.com/en-US/how-to/wp8/music/sync-music-ringtones-and-podcasts-using-my-computer).

 

If instead, the accounts have already been added onto the phone and give you the error message "attention required", it should also say "tap here". If you tap, please check the error code it will give you.

 

I suggest the following:

 

- from a PC go to http://account.live.com and log in with the Microsoft account of the phone. Check if you can sign in or not. If you cannot sign in from the PC, the issue if related to the account itself, and you can try to contact the specific support for the accounts here or https://support.live.com/default.aspx or in the section of this forum specified for the account issues: http://answers.microsoft.com/en-us/windowslive/.

 

If you can log into from the Pc:

- try with wifi and 3G and also with other wifi connections.

- verify that you don't have the two steps authentication active (from http://account.live.com). If yes, switch it off.

- change the password from http://account.live.com to a new one with less than 14 characters. Then change it on the phone: Settings > email+accounts > tap the Microsoft account > enter the new password.

- check that date and time are correct. (you can try both automatic and manual settings, from Settings > date and time)

- it can be a termporary server issue.

- try to reset the phone (settings > about > reset the phone). (it deletes any personal data, but you can first save music and pictures on the Pc, here you see hot to do that: http://www.windowsphone.com/en-US/how-to/wp8/music/sync-music-ringtones-and-podcasts-using-my-computer). After the reset, try to re-add the same Microsoft account, so that contacts and will be synced and you can re-download your apps. You can also try to do a back up: setting back up > app list > back up now, and then re-add the Microsoft account during the initial setup of the phone straight after the reset.
