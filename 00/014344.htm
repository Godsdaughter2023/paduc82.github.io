Creating a Macro in Excel 2013 to uncheck a checkbox when a new one is checked.

I need to create a macro to uncheck a checkbox when another checkbox is mark. The worksheet will be broken up in section and a selection has to be made in each section of the worksheet. From what I have read already on this forum that I may need to move the check boxes to a separate column. Any help will be appreciated,

 

ossiemac say:

To Trina D,

This is a follow up with what I believe to be a better solution than my previous post.  It still involves using Forms Control Check Boxes.

  1. Open VBA editor. (Hold Alt key and press F11 to open the VBA editor)
  2. Insert a standard module. (Select menu item Insert -> Module to create a module.)
  3. Copy the code below and paste into the standard VBA module.
  4. Close the VBA editor (Hold Alt key and press Q)
  5. Save the workbook as macro enabled. 
  6. Create all of the Forms Control Check Boxes (if not already created).
  7. Right click each checkbox in turn and select Assign Macro and assign the common macro name (ProcessChkBoxes) to the control.  (If you get a popup menu options to move here, copy here, cancel then try again because you need the popup with the option to select Assign macro.)
  8. Repeat above step for all the checkboxes.
  9. Right click the first checkbox of a group.
  10. Hold the Ctrl key and right click the remaining checkboxes in the required group so all boxes in the group are selected.
  11. Release the Ctrl key and Right click in one of the selected checkboxes in the group.  (If you get a popup menu options to move here, copy here, cancel then try again because you need the popup with the option to select Groups)
  12. From the popup menu Select Group  and then Group again to the right.
  13. Click somewhere else on the worksheet to unselect the group.
  14. Repeat for the remaining groups of check boxes.

Because the code below is assigned to all of the checkboxes, it runs irrespective of which checkbox is clicked. If the checkbox ix checked (xlOn) then all of the checkboxes in the SAME group are turned off and then the clicked checkbox is turned back on.

The names of the checkboxes and groups are irrelevant just so long as the macro is assigned to the checkbox. You can also add more groups at a later time and assign the same macro to them.

The big advantage of this is that you do not need to know the names of the checkboxes and you do not need to alter the VBA code to match checkbox names.

Sub ProcessChkBoxes()
    'Processes Forms Control CheckBoxes based on Groups
   
    Dim shpChk As Shape
    Dim shpGrp As Shape
    Dim shp As Shape
    Dim i As Long
   
    Set shpChk = ActiveSheet.Shapes(Application.Caller)
   
    If shpChk.ControlFormat.Value = xlOn Then   'xlOn is Checked. (Calling CheckBox)
       
        Set shpGrp = shpChk.ParentGroup
       
        For i = 1 To shpGrp.GroupItems.Count
            Set shp = shpGrp.GroupItems(i)
            shp.ControlFormat.Value = xlOff     'xlOff is Checked. turns off all in the Group
        Next i
       
        shpChk.ControlFormat.Value = xlOn       'This is the calling CheckBox. Gets turned back on
   
    End If
End Sub

ossiemac say:

Hello again Trina D,

I decided to place this in a separate post. it is an automated utility for creating the Forms Control CheckBoxes and Grouping them.

You might like to copy the code into a blank workbook first and test to see how it works.

Simply copy the code and paste into a standard module.

Also copy the code that is assigned to the checkboxes into the same module (See my previous post for that code.)

You can run the code from the VBA editor by clicking anywhere within the sub and press F5.

Follow the instructions in the InputBox.

When finished, Excel will take you back to the VBA editor. Hold Alt key and press F11 to get back to the worksheet.

Sub CreateCheckBoxes()
    'Use this code to create checkboxes with groups
    'At InputBox select first range and then hold Ctrl Key to select remaining ranges for the CheckBox groups.
    Dim strPrompt As String
    Dim ws As Worksheet
    Dim rngUnion As Range
    Dim rngArea As Range
    Dim cel As Range
    Dim arrChkBoxes()
    Dim i As Long
    'Dim j As Long  'Only required for re-naming controls.
    Dim shp As Object
    Dim shpGrp As Shape
    Dim dblLeft As Double
    Dim dblTop As Double
    Dim dblWidth As Double
    Dim dblHeight As Double
       
    strPrompt = "1. Select the first range for the first group of CheckBoxes." & vbCrLf _
                & "2. Hold the Ctrl Key and select the range for the second group." & vbCrLf _
                & "3. Repeat until all ranges are selected for the groups of CheckBoxes." & vbCrLf _
                & "4. When all required ranges are selected, click OK."
   
    On Error Resume Next    'Cannot assign the Cancel to an object variable
    Set rngUnion = Application.InputBox(Prompt:=strPrompt, Type:=8)
    On Error GoTo 0
   
    If rngUnion Is Nothing Then
        MsgBox "User cancelled. Processing terminated."
        Exit Sub
    End If
   
    Set ws = rngUnion.Parent
    Application.Goto ws.Cells(1, 1)    'Activate worksheet with the selected ranges
   
    For Each rngArea In rngUnion.Areas
        i = 1
        ReDim arrChkBoxes(1 To i)   'Reset to one element. (Can't redim preserve without at least one element)
        For Each cel In rngArea
            dblLeft = cel.Left + cel.Width - cel.Height  'All CheckBoxes positioned at right of cell
            dblWidth = cel.Height               'All CheckBoxes square (Height and Width same)
            dblHeight = cel.Height              'Set Height to height of row
            dblTop = cel.Top                    'Set Top to top of row
            Set shp = ActiveSheet.CheckBoxes.Add(dblLeft, dblTop, dblWidth, dblHeight)
            'j = j + 1
            'shp.Name = "Check Box  " & j       'Errors if name duplicated. Best to not rename
            ReDim Preserve arrChkBoxes(1 To i)
            arrChkBoxes(i) = shp.Name   'Array is used to create the Group
            i = i + 1
            shp.OnAction = "ProcessChkBoxes"    'Edit this line to reflect required sub to be called
            shp.Characters.Text = ""
        Next cel
        If UBound(arrChkBoxes) > 1 Then     'Must have more than one checkbox to group
            Set shpGrp = ActiveSheet.Shapes.Range(arrChkBoxes()).Group
        End If
    Next rngArea
  
End Sub
