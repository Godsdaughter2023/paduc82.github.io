cliff gu msft say:

Hi Kentosdad99,

 

From your description, you created some teams in Office 365 Teams. So the site quotas of the sites you want to change are group sites and the group sites will not appear in the SharePoint admin center. You need to set the quotas with the SharePoint Online Management Shell.

 

It seems that you want to manually set all group sites’ quotas. If yes, please follow the steps below to see and let us know the result.

1. Upload a full screenshot showing whether you have set the option of Site Collection Storage Management to Manual. If no, the script running in the SharePoint Online Management Shell will not take effect.

2. Download the SharePoint Online Management Shell from https://www.microsoft.com/en-sg/download/details.aspx?id=35588.

3. Save the following script to a .ps1 file.

 

#Step1: Get values for input parameters

$domainname=Read-Host "Enter the Domain Name(DomainName.com):"
$o365DomainName=($domainname -split "\.")[0]
$Quota=Read-Host "Enter the Storage Quota value in MB:"
$WarningQuota=Read-Host "Enter the Warning Storage Quota value in MB:"
Import-Module Microsoft.Online.SharePoint.Powershell -Verbose

#Step2: Connect to SharePoint Online using “Connect-SPOService” cmdlet
$SPOAdminCenterUrl="https://"+ $o365DomainName +"-admin.sharepoint.com/" 
$credential = get-credential 
Connect-SPOService -Url $SPOAdminCenterUrl -Credential $credential 
#Step3: To get all the Office 365 groups, we use “Get-UnifiedGroup” cmdlet, which depends on Exchange Online
$Session = New-PSSession -ConfigurationName Microsoft.Exchange -ConnectionUri https://ps.outlook.com/powershell -Credential $credential -Authentication Basic -AllowRedirection
Import-PSSession $Session
$Groups=Get-UnifiedGroup
$Groups | Foreach-Object{

$Group = $_
$GName=$Group.Alias
$SiteURL=$Group.SharePointSiteUrl
If($SiteURL -ne $null){
Set-SPOSite -Identity $SiteURL -StorageQuota $Quota -StorageQuotaWarningLevel $WarningQuota
$site=Get-SPOSite -Identity $SiteURL -Detailed
      New-Object -TypeName PSObject -Property @{
     
      GroupName=$GName
      CurrentStorage=$site.StorageUsageCurrent
      StorageQuota=$site.StorageQuota
      StorageQuotaWarningLevel=$site.StorageQuotaWarningLevel
}}}|select GroupName, CurrentStorage, StorageQuota, StorageQuotaWarningLevel

4. Open the SharePoint Online Management Shell with the administrator privilege and connect it to SharePoint Online via the steps in https://docs.microsoft.com/en-us/powershell/sharepoint/sharepoint-online/connect-sharepoint-online?view=sharepoint-ps.

5. Type cd <the location where the .ps1 file is> and run ./<file name>.ps1.

 

Thanks,

Cliff
