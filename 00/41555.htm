Skype on Windows, Mac, Linux, and Skype for Windows 10 (version 14)
Click your profile picture.
Click Settings.
Click General.
Click Language.
Scroll through the list to select from the available languages.

Skype for Windows 10 (version 12)
Skype for Windows 10 (version 12) follows the language set as default in your operating system's Language Preferences. Please note that when you go through these steps, other applications may be affected as well.

To use Skype for Windows 10 (version 12) in a different language:

Select the Start button, then the Settings gear.
Select Time & Language.
Select Region & Language settings, select the desired language and click on Set as default.

Language picker
It will appear as the first item in the list of languages:

Language picked
Restart Skype for Windows 10 (version 12).

If the desired language is not already available in the list, you may add it by downloading a language pack:

In Region & Language settings, select Add a language.

Add language
Navigate to the desired language and follow instructions to download the language pack.

Find language to add
Learn more about the language packs available for Windows 10.
Note: After a new language pack is installed for Windows, it may not display until Skype for Windows 10 (version 12) updates again.
