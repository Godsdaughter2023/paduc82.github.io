This message occurs when the registry key pointing to the uninstaller for iTunes has become corrupted (or the iTunes uninstaller has been moved or deleted).  The simplest thing to do is just clean things out and try again.

Microsoft used to have a tool just for this, called the Windows Installer Cleanup Utility.  Apparently, however, when this tool was used (I think with MS Office), it sometimes removed more than it was supposed to, and therefore Microsoft removed the tool from its download servers.

The Windows Installer Cleanup Utility is still available from http://majorgeeks.com/Windows_Installer_CleanUp_Utility_d4459.html  Or you can use free Revo Uninstaller.  Before you use either of these tools, create a System Restore point, backup the registry using ERUNT, or (if you have the means) create a full disk image or clone -- just in case.

Here are the usual directions to perform a "clean installation" of iTunes, based on Apple Support article HT1926.


Go into Control Panel > Add/Remove Programs
Remove these applications:
1. Apple Software Update
2. Apple Mobile Device Support
3. Bonjour
4. iPod for Windows (this may or may not exist, either way, it's not needed so we can remove it)
5. iTunes
6. QuickTime

Open My Computer > C: > Program Files
Delete this folders (if present)
1. iPod
2. iTunes
3. QuickTime

Empty your temp directory:
Open My Computer > C: > Documents & Settings > your user name > Local Settings and delete the contents of the folder called "Temp"

If you can't see Local Settings, turn on hidden files:
Tools > Folder Options > View tab
Select Show Hidden Files & Folder and click OK

After that folder is emptied, restart the computer

[see note above about Cleanup Utility]
Download & Install the Windows Installer Cleanup Utility:
http://support.microsoft.com/kb/290301/
Launch the Utility: Start > Programs > Windows Install Clean Up
Select any of the above mentioned applications, and click Remove

Restart the computer again
Download & install iTunes again from here: http://www.apple.com/itunes/download/

Computers, which are built by large manufactures that come with Windows Pre-Installed, come with two (2) Product Keys:

 

 OEM SLP: This key comes pre-installed in Windows, when it comes from the Factory. This key is geared to work with the special instructions found only on that Manufacturer's computer hardware.
