Hi Remo,

Please do not take this personally but what you did is NOT a Clean Install. If you initiate a Install/Repair process from within a running OS al manner of info is passed on from the old, broken, install into the new one. Info regarding your account, drivers, etc.

FYI (and indeed all Insiders):

1) A Clean Install must be initiated by booting from installable Media.

2) A Clean Install can only be done unto a blank partition (formatting the partition during the install is OK)

The difference is crucial and, in your case, quite likely the cause of you not being able to fix the trouble.

Again, don't take this as anything personal as many Insiders make the same mistake, so I thought it high time somebody clarified this.

So, back to your problem, if you had 'digital entitlement' then you should be ok.
