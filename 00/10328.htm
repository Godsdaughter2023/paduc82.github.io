john w. vinson say:

I would suggest that rather than using the inefficient and limited DLookUp, that you use a "Non Equi Join" query. Create a new query joining your table to this Fees table with a query like

SELECT yourtable.thisfield, yourtable.thatfield, Fees.Fee

FROM yourtable INNER JOIN Fees

ON yourtable.Value >= Fees.MinValue

AND yourtable.Value <= Fees.MaxValue;

The DLookup syntax for looking up the fee using the value entered on a Form named MyForm in a textbox named txtValue would be:

DLookUp("[Fees]", "[RegFees]", [Forms]![MyForm]![txtValue] >= " & [MinValue] & " AND
