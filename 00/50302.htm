Hi,

 

Windows 7 has native touch screen support no doubt in that. Unfortunately touch gesture is not possible on remote desktop connection.

 

When a remote desktop session is established in between two computers (one with touch capability and other with no touch capability) as in your case. There might not be a chance to access a touch capable computer with a non touch capable desktop computer, though Windows 7 is touch wiz compatible.

 

In order to use this touch features or programs, both the computers should be installed with touch capable hardware. For example a Touch screen monitor, Touch pad, etc.

Without a touch compatible hardware on both the computers, touch capable remote desktop connection in between two computers might not possible.

 

The only possible way to create a touch capable remote desktop connection is by installing touch interface hardware on your desktop computer.
