The all users Startup folder is a hidden system folder located at:
C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup

a)      Go to Start

b)      Type Folder Options and hit Enter.

c)       Then click on the "View" tab in the Folder Options window

a)      Go to Start, type the following command at the search bar and hit Enter.

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup

b)      Right click on the shortcut in the Startup folder and click on Delete.

c)       Close the Startup folder when done.
