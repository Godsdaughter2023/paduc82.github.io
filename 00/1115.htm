Hi,

 

Does your Skype account has a unique or personalized Skype Name? If yes, I strongly advise you to enter the correct Skype Name on this link and follow the password reset steps.

 

If the account is created using the Microsoft account, this means the account is a Microsoft Skype account. Technically it is a Microsoft account that has been signed in for Skype use. To change the password of this account, you need to change the Microsoft account password.

 

A merged Skype account is a Skype account that has a unique or personalized Skype Name that has been merged to a Microsoft account. elrendhel say:

Greetings PaulG187!   Welcome to the Windows Mobile Owner's Circle forums!

 

If you ever performed a Sync with a host computer (your desktop or laptop, etc), then your PIM (personal information manager) data should be backed-up there, so you would be safe proceeding with a Hard Reset.  Then re-establish an ActiveSync partnership on the 8925 then sync your old data back over.

Alternately, if you ever backed-up your device with a backup program like Sprite Software's aptly named "Backup" program, then once again, your PIM data will be available for you to restore from the backup file image.  This data will only be as up-to-date as the original data was when you originally made the backup.

 

Lastly, some devices have an automatic safe-mode built in that gets tripped whenever you perform at least two or more soft resets within a specific time-frame.  For some Pocket PC's like my HP iPAQ h5555, it is tripped when the user performs two soft resets within a three minute period. 

 

I cannot say whether or not this feature is present in your 8925, but I'll wish you the best of luck in your endeavors!
