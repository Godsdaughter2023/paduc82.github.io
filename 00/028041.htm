a. user say:

Hi Bev,

 

Thank you for posting your query in Windows forums.

 

From the issue description, I understand that you need to save the additional desktops even after shutting down the computer.

I would like to inform you that Multiple desktop thread/process gets reset after every Windows login, that is the reason previous session is lost.

Reason: Say you have 5 different applications running on each desktop that you have created. Now for a computer to remember are:

  • Last state of the running application.
