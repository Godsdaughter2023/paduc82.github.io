Open the database and select the Modules pane of the navigation pane, and create a new Module. Copy and paste these three lines

Declare Function clt_OpenClipboard Lib "user32" Alias "OpenClipboard" (ByVal hWnd As Long) As Long

Call ClearClipboard

=ClearClipboard()

in the Close property of the form's properties' Events tab.
