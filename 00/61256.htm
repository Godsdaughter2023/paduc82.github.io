a. user say:

System Restore is not a time machine and if you think your system is infected, you need to fix it a different way (not with SR).  

First, you need to get your system running.

Microsoft has an article you can read and see if they relate to your symptoms of a black screen on reboot, etc.

http://support.microsoft.com/?scid=kb;en-us;314503

Hopefully, your situation is not Cause 1.

If it is not Cause 1, it is Cause 2.

Parts of the resolution for Cause 2 are probably not necessary for you or just do not apply to most modern systems, but parts of the resolution for Cause 2 make good sense and are worth investigating.

Having resolved this problem many times and inflicted it upon myself on purpose many more times,, I have some knowledge of which parts of the resolution for Cause 2 are most like to resolve your issue.

You will not get anywhere in the resolution for Cause 2 process (or fixing your system) if you can't boot on something and you can do everything you need to do from the resolution of Cause 2 process using the XP Recovery Console.  

Even if you think the symptoms in the MS article do not fit your situation exactly, you still need to boot on something in order to fix your system.

You can make a CD that will boot your system into the XP Recovery Console by following the directions below:

 

Boot into the Windows Recovery Console using a bootable XP installation CD, or create on a bootable XP Recovery Console CD. 

This is not the same as any recovery disks that might have come a store bought system.  If you are not sure what kind of bootable CD you have, make a bootable XP Recovery Console CD and  be sure.

You can create a bootable XP Recovery Console CD when no XP media is available by following the directions in this link:

http://www.bleepingcomputer.com/forums/topic276527.html

After booting into the Recovery Console, you should be in this folder:

C:\WINDOWS

For each of your hard disk partitions, you should then run:

chkdsk /r

For example, from the Recovery Console prompt, enter:

chkdsk c: /r

Let chkdsk finish and correct any problems it might find.  It may take a long time to complete or appear to be 'stuck'.  Be patient.  If the HDD light is still flashing, it is doing something.  Keep an eye on the percentage amount to be sure it is still making progress.  It may even appear to go backwards sometimes.

The chkdsk /r needs to complete without any errors, so you may need to run it multiple times until it does.

 

Remove the CD and type 'exit' to leave the RC and restart the computer.

If your computer still does not start, you will at least have some maneuvering room to get started on the steps (only the necessary steps) listed in the resolution for Cause 2.  

If you are not sure which steps apply, post back when you have booted the XP Recovery Console for additional assistance.

When you post back, please do provide your system make and model number if you can make that determination.
