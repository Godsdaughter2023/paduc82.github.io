Sub FormatAllComments()
'www.contextures.com/xlcomments03.html
  Dim ws As Worksheet
  Dim cmt As Comment
  For Each ws In ActiveWorkbook.Worksheets
    For Each cmt In ws.Comments
      With cmt.Shape.TextFrame.Characters.Font
        .Name = "Times New Roman"
        .Size = 12
      End With
    Next cmt
  Next ws
End Sub

The dialog that you can use applies the changes to the active selection, so this chooses a single cell and captures what you choose (font name, size, and style) in the dialog.

Sub ChangeCommentFont()
    Dim strFName1 As String
    Dim strFStyle1 As String
    Dim iFSize1 As Integer
    Dim strFName2 As String
    Dim strFStyle2 As String
    Dim iFSize2 As Integer
    Dim r As Range
    Dim ws As Worksheet
    Dim cmt As Comment

    Set r = Range("A1")

    r.Select

    With r.Font
        strFName1 = .Name
        strFStyle1 = .FontStyle
        iFSize1 = .Size

        Application.Dialogs(xlDialogFormatFont).Show

        'Capture the new settings
        strFName2 = .Name
        strFStyle2 = .FontStyle
        iFSize2 = .Size

        'Restore the cell
        .Name = strFName1
        .FontStyle = strFStyle1
        .Size = iFSize1
    End With

    For Each ws In ActiveWorkbook.Worksheets
        For Each cmt In ws.Comments
            With cmt.Shape.TextFrame.Characters.Font
                .Name = strFName2
                .FontStyle = strFStyle2
                .Size = iFSize2
            End With
        Next cmt
    Next ws

brian tillman [outlook mvp 2007-2019] say:

What type of account are you using - POP, IMAP, Exchange, or Hotmail/LIve?  Do you see the sent messages in your Sent Items folder, either in Outlook or on the server's Sent folder if you log into the mailbox using a web browser?  There's a good chance that no one is using your account to send mail, but rather simply using your address as the sending address.  If this is the case, then it will stop soon because spammers change the sending address frequently.

However, if you log into the mailbox and see the outgoing messages in the Sent folder on the server or you see them in Sent Items in Outlook, then there's  a real problem, particularly if you see the messages in Outlook and you're using a POP or Exchange account.  In either case, but particularly POP, your PC is likely compromised.  If IMAP or Hotmail/Live, then it's likely your mailbox has been compromised.  In that case, if you can still log in, do so, change your password online to a very strong password with mixed case and non-alphameric characters and change any security question answers (be sure to choose answers that no one can guess because they know you or can research you on the Internet) and that don't appear in the dictionary.  If you have the ability to specify a cell phone that will be notified of a change, specify it.  If you ever access your mailbox via a web browser in a public access spot, always use https to make that connection and if the mail service has the option of enabling https for all connections, enable it.
