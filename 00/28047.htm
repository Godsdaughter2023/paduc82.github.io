It's not always the db that is at fault though because with CD use diminishing, a lot of different versions of the same disc (maybe originally intended for export or something) find their way into stores. If you are sure your settings are correct (the key one is the top box in the Privacy tab - that must be checked....) then try searching manually by inserting the disc, clicking on the art square and selecting 'Find Album Info' - that will take you into the database where you can enter artist/title etc. and search manually. If you still don't find it you can save yourself some typing by copy/pasting track titles from a site like Discogs - one at a time unfortunately but it does save some time! - Cheers, Ric.
