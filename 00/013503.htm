jeeped say:
I think the only problem is that you are trying to assign the returned Vlookup() to Me.txtMSIScore1.Value instead of just Me.txtMSIScore1. As mentioned in a previous thread, the Val() is completely unnecessary and will throw an error when trying to convert a text value to a double ( the returned value fo Val()). Both of the following code lines should work regardless of whether txtMSIScore1 is a text box or a text label.

    Me.txtMSIScore1 = Application.WorksheetFunction.VLookup(txtFabric1, Range("MSI_Affiliates!A1:B19"), 2, False)
    Me.txtMSIScore1 = Application.WorksheetFunction.VLookup(txtFabric1, Sheets("MSI_Affiliates").Range("A1:B19"), 2, False)

This assumes that a conventional worksheet =VLOOKUP(...) works. There should be no leading or trailing spaces in the Material ID column and any inner spaces should be regular spaces (e.g. ASCII character 32) and not non-breaking spaces (e.g. ASCII character 160).
