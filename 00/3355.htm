Hi John Wilson,

 

The INET_E_DOWNLOAD_FAILURE or error 0x800C0008 means there’s an interruption when you are downloading from Internet Explorer browser. To resolve this, you will need to repair your Internet connection first.

 

Here’s what you can do:

 

Step #1: Uninstall and reinstall Internet Explorer.

uninstall it.

it again.

 

Step #2: Reset Internet Explorer from the app option itself. Visit this link: Change or reset Internet Explorer settings and follow all the instructions there.

 

Step #3: Run the Network troubleshooter. To diagnose your network configuration, perform the steps below:

 

1. Open Start menu, then click Settings icon.

2. Click Network & Internet.

3. On the left pane, click Status.

4. Click Network troubleshooter to diagnose and fix network problems.

 

Step #4: Perform an Internet connection reset via Command Prompt. To do this, follow the steps below:

 

1. Press Windows + R to open the Run window.

2. Type cmd, then press Enter.

3. Type the following commands, then press Enter after typing each commands below:

netsh int ip reset resettcpip.txt

netsh winhttp reset proxy

netsh int ip reset

ipconfig /release

ipconfig /renew

ipconfig /flushdns

netsh winsock reset

4. Restart your PC to complete the process.

Step #5: We also suggest that you restart your router and wait for 15-30 seconds then turn it on again. Visit this link: Ethernet adapter driver is also suggested there. Make sure that you disable all other anti-virus programs and enable Windows Defender only.
