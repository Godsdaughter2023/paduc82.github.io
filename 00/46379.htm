frederik long say: Here you go:
1. Keep tapping F8 during the early boot phase.
2. Select Repair from the menu.
3. Select an admin account, then enter its password.
4. Select System Restore when prompted.
5. Set Windows back to a point before you hacked the registry.

Note also that what you call the Command Prompt is actually the Command Prompt in Repair Mode. Using a console command to modify the registry would have no effect at all on the normal mode registry. It could be done with regedit.exe but the process is complex. Much easier to do a System Restore.

tech jason say:

Here is a quicker/less invasive way:

1. Keep tapping F8 during the early boot phase.
2. Select Repair from the menu.

3. Select an admin account, then enter its password.

4) Select Command prompt.

5) Type "regedit" and press enter.

6) Click on one of the roots then click File.

7) Click Load Hive.

8) Navigate to Computer - "Local Disk" - Windows - System32 - Config.

9) Select SYSTEM.

10) Name it something easy to remember.

11) Click the + next to the select root.

12) Select your loaded hive and drill down to ControlSet001\services\rdyboost.

13) Change the Start REG_DWORD to 0.

14) Do the same for ControlSet002\services\rdyboost.

15) Click File then Unload Hive.

16) Exit Regedit and the command prompt.

17) Click Restart.

andre da costa say:
What is the exact make and model of your Dell computer?

Press Windows key + X
Click Device Manager
Expand Cameras
Right click the camera, then click Uninstall

Note: If you don't see it under cameras, look under Imaging devices.

Exit device manager then restart.

Windows 10 should redetect it then reinstall it.

Check if the camera is working again.

If you have a third party Antivirus utility installed, please remove it.
Also check the manufacturers website for updated driver.

Option 1. Press Ctrl + Shift + Esc or right-click empty space on the toolbar.
Open “Task Manager” and go to the “Processes” tab.
Carefully analyze what programs are currently running in the background of the system.
If you found an app that uses Camera, for example, Skype running in the background, click it, and select “End Task.”

Option 2. Try a Clean Boot
Click “Windows key” and type msconfig.
Click “System Configuration” and open “Services” tab.
Mark “Hide all Microsoft services” box and select “Disable all.”
Open “Startup” tab and click “Open Task Manager.”
Click on each startup item and select “Disable.”
Close Task Manager and click “OK” on the System Configuration window.
Reboot the system.

If Clean Boot helped to fix error, we would recommend you to enable apps one by one. After each enabled app, check if the Camera is working. This way, you will isolate the problematic application.
