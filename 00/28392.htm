Sub DeleteAllBookmarks()
    Dim i As Long

Turn the {PAGE} - 3 part of the code into an equation field:

{ IF  { PAGE } < 4 "" { = { PAGE } - 3 } }

Use Ctrl+F9 to create each pair of field markers. The left brace before the = sign matches the right brace that immediately follows the 3.
