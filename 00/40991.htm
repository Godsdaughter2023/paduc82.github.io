a. user say:

Hi Tino,

 

To troubleshoot your issues with the Windows 10 Photo Viewer, follow the methods below:

 

Method 1: Run the troubleshooter for Windows app on this link.

 

Method 2: Update graphics card driver.

 

  1. Press Windows key + S to open Search.
  2. Type Device Manager and select it from the results.
  3. Expand Display Adapters.
  4. Right-click your graphics card, then select Update Driver Software.
  5. Select Search automatically for updated software.
  6. If Windows doesn’t find a new driver, you can try looking for one on the device manufacturer’s website and follow their instructions.

 

Method 3: Reinstall graphics card.

 

  1. Open Device Manager.
  2. Right-click your graphics card, and select Uninstall.
  3. Restart your PC and Windows will attempt to reinstall the driver.
