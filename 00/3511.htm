dhookom say:

Are you actually attempting to update any information or just display? If all you want to do is display, you can create the union query and add ",Null" to pad out for extra fields like:

SELECT FldA, FldB, FldC, Null as FldD, Null as FldE

FROM table1

UNION ALL

SELECT FldA, FldB, Null, FldD, FldE

FROM table2;

Then you can create a totals query from the union and return the Max() of some columns.
