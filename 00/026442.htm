@ Minz,

 

Sorry to hear the Store is still not working. Check if there are any updates available for your device. If yes, update, if no, try the following actions.

 

  • Make a backup with Nokia Suite and reinstall the phone software. The process is the same as when updating with Nokia Suite, but at the end select "Reinstall".
  • You can also do a soft reset. Type in *#7780# followed by lock code (default is 12345). No data should be lost. The settings will be restored to factory state.
  • Type in *#7370#. This is hard reset, phone memory will be wiped. Restore back up.
  • Take the device to Nokia Care point.

Hope you get the Store working again!
