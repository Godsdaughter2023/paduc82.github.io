john - project say:
Pam,
No problem. The revised code follows:
Sub userID()
Dim t As Task
i = 1000
For Each t In ActiveProject.Tasks
    If Not t Is Nothing Then
        t.Text30 = i
        i = i + 10
    End If
Next t
End Sub

With regard to setting up and running the macro, it depends on which version of Project you are using. Use the following steps:
1. copy the above code to your clipboard
2a. Project 2010 - select Developer/Code group/Visual Basic
2b. Previous versions - Select Tools/Macro/VB Visual Basic Editor
3. In the VBA code window that opens paste the macro from the clipboard. The macro is now attached to the file. You can either run the macro by going to Run/Run Sub/Userform or close the VB editor window and follow these steps:
4a. Project 2010 - Developer/Code Group/View Macros
4b. Previous versions - Select Tools/Macro/Macros
5a. Project 2010 - find filename!UserID in the list and select it
5b. Previous versions - find UserID in the list and select it
6. Hit Run

Correction: In my earlier post I said this would populate the Text1 field but the actual macro code uses the Text30 field. If the macro code is used verbatim be sure to add the Text30 field as a column in the view.
