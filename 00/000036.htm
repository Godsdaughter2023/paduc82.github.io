shane devenshire say:

Are these arrays ranges or VBA arrays.  What do the arrays contain - numbers, text, dates, some combination?

You can dump the arrays from VBA in one step to a range.

Then you can create a range of formulas that compare the two ranges:

=1/(A1=AA1)

Will return Div/0 or 1.  Then you can use Goto Special to select all the Div/0 errors (formulas with errors) and do whatever it is you need to do.

You can also add a column which sums each row, therefore returning Div/0 on any row with a Div/0 anywhere on it.  Then Filter on that column for either errors and copy and paste.
