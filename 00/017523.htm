bob jones aka: cybertaz say:
First of all, never change a file name extension -- the extension doesn't dictate or determine the file type, it is merely an indicator of what the file format is. In most cases, editing the extension complicates issues by wrongly indicating the file type which can cause permanent damage to the file.

To resolve your issue, Quit all programs then select any .xlsx file in Finder. Use Get Info to confirm that Excel is the app set for Open With. If it's already selected, select it again then click the Change All button.

Next,  launch Disk Utility to repair disk permissions & restart your Mac.
