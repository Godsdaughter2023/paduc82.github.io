andre for directly say:
If Office 365 came preinstalled on your new computer, it is likely a 1 month trial.

Since Office 2016, applications in the Office 365 suite are bundled together and cannot be removed. Its Word with everything. So, you have to keep around the applications you want to use.

If you are running Skype 8 or later, you should be able to uninstall it in Windows 10:

Click Start > Settings > Apps > Apps & features > scroll down
Select Skype, click uninstall.

If you are not running Skype 8, you might be running Skype 12, use this method to uninstall it:

Press Windows Key+X
Click Windows Powershell (Admin)

Enter the following command then hit Enter

Get-AppxPackage -allusers Microsoft.SkypeApp | Remove-AppxPackage

OneDrive can also be uninstalled from Start > Settings > Apps > Apps & features > scroll down
Select OneDrive, click uninstall.

bill.p. say: Hi Suppressed, thanks for your question. I'm Graeme, a Microsoft Community expert and fellow user.

Office 365 installs all of its apps together, and if you uninstall Office 365, it'll remove all of the apps together including Word.

It's possible that you have two versions of Office installed on your computer. You can check this by clicking Start and typing "apps". Click on Apps & Features, and see if you can find multiple versions of Word in the list of apps that shows up.

If you've got Office 365 and another version such as Office 2016 or earlier, you can safely uninstall Office 365 and continue using the older version of Word. It won't remove any of your documents, just the apps.

Hopefully this helps! Please let me know if you have any other questions! Thanks!

gord dibben say:

Alternative to typing in sheetname.

Select from a pop-up list.

Bob Phillips' Browse_Sheets code.

Sub Browse_Sheets()
Const nPerColumn  As Long = 20          'number of items per column
Const nWidth As Long = 8  '5                 'width between each column of names
Const nHeight As Long = 18 '16             'height of each row
Const sID As String = "___SheetGoto"    'name of dialog sheet
Const kCaption As String = " Select sheet to goto"    'dialog caption
                                        
Dim i As Long
Dim TopPos As Long
Dim iBooks As Long
Dim cCols As Long
Dim cLetters As Long
Dim cMaxLetters As Long
Dim cLeft As Long
Dim thisDlg As DialogSheet
Dim CurrentSheet As Worksheet
Dim cb As OptionButton
    Application.ScreenUpdating = False
    If ActiveWorkbook.ProtectStructure Then
        MsgBox "Workbook is protected.", vbCritical
        Exit Sub
    End If
    On Error Resume Next
        Application.DisplayAlerts = False
        ActiveWorkbook.DialogSheets(sID).Delete
        Application.DisplayAlerts = True
    On Error GoTo 0
    Set CurrentSheet = ActiveSheet
    Set thisDlg = ActiveWorkbook.DialogSheets.Add
    With thisDlg
        .Name = sID
        .Visible = xlSheetHidden
        'sets variables for positioning on dialog
        iBooks = 0
        cCols = 0
        cMaxLetters = 0
        cLeft = 78
        TopPos = 40
        
        For i = 1 To ActiveWorkbook.Worksheets.Count
            If i Mod nPerColumn = 1 Then
                cCols = cCols + 1
                TopPos = 40
                cLeft = cLeft + (cMaxLetters * nWidth)
                cMaxLetters = 0
            End If
            Set CurrentSheet = ActiveWorkbook.Worksheets(i)
            cLetters = Len(CurrentSheet.Name)
            If cLetters > cMaxLetters Then
                cMaxLetters = cLetters
            End If
            iBooks = iBooks + 1
            .OptionButtons.Add cLeft, TopPos, cLetters * nWidth, 16.5
            .OptionButtons(iBooks).text = _
                ActiveWorkbook.Worksheets(iBooks).Name
            TopPos = TopPos + 17  '13 vertical distance between
        Next i
        .Buttons.Left = cLeft + (cMaxLetters * nWidth) + 24
        CurrentSheet.Activate
        With .DialogFrame
            .Height = Application.Max(64, _
                Application.Min(iBooks, nPerColumn) * nHeight + 15)  '12
            .Width = cLeft + (cMaxLetters * nWidth) + 24
            .Caption = kCaption
        End With
        .Buttons("Button 2").BringToFront
        .Buttons("Button 3").BringToFront
        Application.ScreenUpdating = True
        If .Show Then
            For Each cb In thisDlg.OptionButtons
                If cb.Value = xlOn Then
                    ActiveWorkbook.Worksheets(cb.Caption).Select
                    Exit For
                End If
            Next cb
        Else
            MsgBox "Nothing selected"
        End If
        Application.DisplayAlerts = False
        .Delete
    End With
End Sub
