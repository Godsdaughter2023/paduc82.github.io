Method 1: Make sure that Xbox services aren’t disabled


1. In the Search bar type Services. Select the Services from the list of results.

2. In the Services window sort the services by their name.

3. Scroll to the bottom and find the following services: Xbox Live Auth Manager, Xbox Live Game Save, Xbox Live Networking Service.

4. You’ll have to repeat this process for each of the services listed above. Right click the service and select Properties.

5. Find Startup type section and make sure it’s set to Automatic. If the service isn’t started, click Start to start the services.

6. Save changes.


Method 2: Clear the local Xbox identity data


1. Open File Explorer.
2. In the address bar paste and go to this location:
    %userprofile%\AppData\Local\Package\Microsoft.XboxIdentityProvider_cw5n1h2txyewy\AC\TokenBroker
3. If there are Accounts and Cache folders available make sure that you delete them.
4. Now try starting Xbox app again.

An alternative to Anannya's suggestion is to turn the live tile off completely so that it doesn't display anything.

To reverse this, follow the instructions above but at step 3 click the Turn Live Tile On option.
