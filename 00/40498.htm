I think you're confusing "download" and "install."  I assume that you successfully downloaded a file from the Canon website but then had a problem when you tried to run the file.  Is that correct?

Which file did you download?

  • win-ts9100-1_1-n_mcd.exe ("Full Driver & Software Package")
  • mpnp-win-ts9100-1_01-ea34_2.exe ("MP Drivers Ver.1.01")

I suspect that you downloaded and ran the first of these two files.  Based on the sizes of the files (the first one is about 18 MB and the second is about 92 MB), as well as your description of step 12 of 14, it seems to me that the first file is a "stub" (albeit a bit larger than usual) that examines your system, determines what files you need, and then downloads the necessary files from a Canon server in Japan.
