tasos_k say:

Hi,

try this code to solve your problem....

target cell is B2

Sub Resize_Cell_from_Picture_Size()
Const maxH = 409.5 ' << max row height = 409.5/by default
Const picName As String = "pic-01" '<< picture name
Dim cel As Range
Set cel = [B2] ' << target cell in active sheet
Dim nH
Dim nW
Dim nPix
Dim x
On Error Resume Next
Dim fPath As String
With Application.FileDialog(msoFileDialogFilePicker)
If .Show = True Then
ActiveSheet.Pictures(picName).Delete
fPath = .SelectedItems(1)
Set p = ActiveSheet.Shapes.AddPicture(fPath, False, True, cel.Left, cel.Top, -1, -1)
With p
.Name = picName
nH = .Height
nW = .Width
End With
If nH > maxH Then
MsgBox "wrong, picture's height is > " & maxH
p.Delete
Exit Sub
Else
cel.RowHeight = nH
nPix = nW / 0.75
x = (nPix - 12) / 7
nW = Round(x + 1, 2)
cel.ColumnWidth = nW
p.Placement = xlMoveAndSize
cel.Select
End If
Else
MsgBox "cancel"
End If
End With
On Error GoTo 0
End Sub

You can either display the document as Final without markup, or go through and accept the changes.

Dim arev As Revision

How to Stream Zune on Windows Phone - Microsoft Community

d1zoogie say:

How to Stream Zune on Windows Phone

I just purchased a windows 7 phone.  I have the 1 month Zune Pass and I want to stream my music to my phone, but I cannot see how.  My software on my PC and my phone is up to date.  I am not using my Zune to listen to music at the moment.  How do I sign in to Zune on my phone to stream music? 
julie fka profjulie say:

You need to make sure the windows live id that you used when you first set up the phone is the same windows live id for your Zune Pass or else you'll never be able to use the Zune Pass on the phone.

After that you can go to Settings > applications > music & video > make sure the "Connect with Zune" option is turned on.

While on this screen you can tap the link, "Zune Account Settings"  This will open up the Internet Explorer browser on the phone and take you to the Zune.net website where you can enter your windows live id and password.
