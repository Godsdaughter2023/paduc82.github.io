Disclaimer: The Reset Internet Explorer Settings feature might reset security settings or privacy settings that you added to the list of Trusted Sites. Method 2:

3. Remove the check mark from the address option.

4. If the Address bar appears in the upper right corner and not available, Right Click Address and click Lock Toolbar to remove the check mark.

5. Finally click Address and drag the address bar to where you want it.

Method 3:

1. Click on Start.

2. Type run in the Start Search box and press Enter.

3. The Run dialog box appears.


4. In the Run dialogue box type: 'REGSVR32 /i BROWSEUI.DLL' (without the quotes) and click on Ok.

5. Restart your computer.

6. The address bar should now be back on the main toolbar.
