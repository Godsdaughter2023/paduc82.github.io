hansv mvp say:

You should start with xrow = 2, for in row 1, you can't compare with the row above.

 

You don't need to select a cell to get its row, and it might be a good idea to allow for more than 65000 rows. xRow and xLastRow should be of type Long.

 

Finally, you can use a For ... Next loop instead of Do Until ... Loop:

 

Sub MyMacro()
    Dim xLastRow As Long
    Dim xRow As Long
    xLastRow = Range("C" & Rows.Count).End(xlUp).Select.Row
    For xRow = 2 To xLastRow
        If Left(Cells(xRow, 5), 4) = Left(Cells(xRow - 1, 5), 4) Then
            ' compares the 4 digits of the customer ID from the line without no name
            ' with the 4 digits of the customer ID of the line right on top of it that has a customer name
            Cells(xRow, 3).Value = Cells(xRow - 1, 3).Value
        End If
    Next xRow
End Sub
