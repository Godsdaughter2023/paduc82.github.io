    Slick,

    I use Task Scheduler to allow a standard user to avoid the Admin prompt for some specific jobs that normally require it.  I have not had any problems.

    1     Shortcut  I put a shortcut to "C:\Program Files (x86)\Utilities\ControlTSTasks\Run TS task - RobvdWTimeSync.bat" on the standard user's Desktop.  [For the avoidance of doubt, this shortcut does property set.]

    2     Batch file  The batch file contains the single line SCHTASKS /Run /TN "RobvdWTimeSync"

    3     Task scheduler  I opened TS as Admin [right-click on the TS shortcut and select Run as Admin] in order to set the task up and save the Admin password within it.  The TS task RobvdWTimeSync contains these settings.

    3.1     General: Author - Admin; use the account - Admin; Run whether user is logged on or not; Run with highest privileges.

    3.2     Action: Start a program - "C:\Program Files (x86)\Utilities\RobvdWTimeSync\TimeSyncWeb.vbs" [a vbs that needs Admin permission to run]

    3.3     Settings: Allow task to be run on demand

    4     Notes

    4.1     TimeSyncWeb.vbs does not need any user interaction; it has no user interface.  That is a necessary condition for TS to run anything "whether user is logged on or not".

    4.2     Both the batch file and the vbs are within the Program files folders so that their content is protected from change [Admin permission is required to edit them].

    4.3     I studied the Vista link you gave and the only difference I could see was that in Windows 7 you must deliberately open TS as Admin to set up a task such as this even if you are already logged in to an admin account.

    5     Fault-finding  If this setup does not work for you first time then I suggest that you fault find by first [5.1] proving the TS part of the setup and then [5.2] proving the content of the job you are actually trying to do. 

    5.1     Replace the TS task Action with %windir%\system32\defrag.exe, argument C:  This will prove that steps 1,2,3 are working correctly for a job that normally needs Admin permission.  Defrag can then be stopped by running Disk defragmenter C:\Windows\System32\dfrgui.exe and telling it to stop.

    5.2     Type cmd in the Start menu search box then right click on the cmd shortcut and select Run as Admin.  Once you have got through the Admin prompt, enter your main batch file's path & name [in my example, "C:\Program Files (x86)\Utilities\RobvdWTimeSync\TimeSyncWeb.vbs"].  This will prove that the code for the main job is running correctly without any further interaction. 

    6     HTC issues  HTC support were very helpful on the single occasion that I called them [ http://www.htc.com/uk/contact/phone/  - this also shows other options such as email].  There might be useful guidance at http://www.htcforums.com/  I check for updated HTC Sync Mgr versions every so often so that I can install any new drivers even though I do not use Sync Mgr itself - I only mention this because the memory problem you refer to could easily be a driver issue.   I use my HTC tethered as the internet connection for my PC but you are doing it the other way around so I cannot help with that.
