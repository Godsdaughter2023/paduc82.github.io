harry stevens say:

Hi,

Since Windows 10 is a relatively new OS, it automatically downloads and installs newest updates to make sure that your Windows is up-to-date and working properly. For now, there's not an option to turn these features off. But Microsoft released a tool that allows you to manage updates that will be automatically installed on your system. You can download this tool here.

For automatically downloading updates, what you can do is to set your connection to a metered connection. This will stop downloads from happening, unless you set a specific time to download them. But this is not an option that you can use if you are using a Wired Connection.

If you're using a wired connection, your alternative would be to stopping the Windows Update Service from running. But this can be risky since you'll be missing out a lot of important updates to make your Windows stable.
