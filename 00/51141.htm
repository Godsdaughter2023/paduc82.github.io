Port Forwarding - Microsoft Community

xbr dionysus say:

Port Forwarding

Login Information

  • Gateway - 192.168.0.1
  • Username – admin
  • Password – admin

Port Forwarding

Enable Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of an internet browser.
  3. When prompted, leave the username blank and enter "admin" for the password.
  4. Select Security from the menu on the left.
  5. Select the Persistent Port Forwarding link.
  6. Enter Xbox1 for the Decription
  7. Enter 3074 for the Inbound and Private ports.
  8. Select TCP for the Type.
  9. Enter the IP Address of the Xbox for the Private IP Address.
  10. Click the Add button.
  11. Repeat steps with Xbox2 as the Description and TCP for the Type.

 

 Disable Port Forwarding

  1. Open web browser.
  2. Login to router by entering http://192.168.0.1 into the address bar.
  3. If prompted for a username and password use admin/admin.
  4. Select Virtual Server on the left.
  5. Select Port Range Forwarding from the sub-menu on the left.
  6. Un-Check the Enable box for all ports you wish to disable.
  7. Click Apply.

aakkam22 say:

These problems are likely due to corrupt networking services. There are many of them to check.

First, open the services console (Start > Type services.msc) and check that all network related services are set to Automatic and are running. These services include:

  • Network Connections
  • Network List Service
  • Network Location Awareness
  • Network Store Interface Service

Next, see this article and follow the instructions in it to reset the cache for the Network Connections service.

It may be a good idea to check Device Manager as well to see if your network adapter is giving trouble.
