ken sheridan say: The following should return all purchase orders in the current calendar year where a part has been ordered more than once in the year, and the unit price of the product has changed at least once over the course of the year:

SELECT *
FROM [YourTableNameGoesHere] AS T1
WHERE YEAR([Order Date]) = YEAR(DATE())
AND EXISTS
    (SELECT *
     FROM [YourTableNameGoesHere] AS T2
     WHERE YEAR([Order Date]) = YEAR(DATE())
     AND T2.[Part Number] = T1.[Part Number]
     AND T2.[Unit Price] <> T1.[Unit Price]);

PARAMETERS [Enter start date:] DATETIME,
[Enter end date:] DATETIME;
SELECT *
FROM [YourTableNameGoesHere] AS T1
WHERE [Order Date] >= [Enter start date:]
AND [Order Date] < NZ([Enter end date:],DATE())+1

Indeed the PST file contains Contacts and there is no separate file for Contacts anywhere on the hard drive. They are right where the address book dialog told you they are: in the Contacts Folder in your Personal Folders file.

We post information on how to backup and transfer Outlook data daily here: The file you need is your Personal Folders file (*.pst) It's where all the mail, calendar, contacts etc are stored.
