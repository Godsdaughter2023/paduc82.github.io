susanclose say:

mayank gupta10 say:

Hi Susan,

Thank you for posting your query on Microsoft Community.

This error message indicates that another computer on the network is using the same IP address that your computer is using. Each computer on a network must have a unique IP address.

This issue seem to bit unclear. However, in order to guide you in a correct direction we will need the following information as well. Please help us in answering these questions.

  1. Are you trying to connect to internet and you are unable to and getting an error message to contact the computer administrator?
  2. What is the make and model of your computer?
  3. Is there any error code?

Method 1:

I suggest you to follow the link and check if it helps.

http://windows.microsoft.com/en-us/windows/get-help-ip-address-conflict#1TC=windows-7

Method 2:

You may try to release and renew the IP address on both the computer which has same IP address.

To do this, follow these steps:

1. Click on Start search box, type ‘cmd’ in the box. Right-click on ‘cmd.exe’ and choose ’Run as administrator’.

2. In the C:\ prompt, type ipconfig and press Enter.

3. Type ‘ipconfig /release’ and press Enter.

The IP configuration information will refresh. You should see your IP address change to 0.0.0.0.

4. Type ‘ipconfig /renew’ and press Enter.

5. Your computer will attempt to obtain a new IP address from the network. This may take a few minutes.

If this process is successful, you will see a new IP address listed when the IP configuration information refreshes.

6. Restart the computers and check if you have similar issue.

 If it does not help, you may also manually assign the IP address e.g.   192.168.1.2.

Hope this helps. Please let us know the results. We will be happy to assist you further.

malkeleah say:

Answer D. is ridiculous and completely wrong. The way it is written (or which you have quoted it as written), the question is about the desirability of having the requirement to press Ctrl-Alt-Del at logon instead of using the Welcome Screen or automatically logging into one particular user account. If this is what the question is really about (and it is ambiguous and therefore poorly written), then all of the answers are wrong.

The reason for requiring Ctrl-Alt-Del to be pressed at logon (usually done only in corporate environments or in environments where security is an issue) is because in order to log on that way a username and password needs to be known and typed in. Of course this only provides a small level of security because even without knowing a username/password one can get into a computer, but it is still a good thing to do - like locking your car door. It won't keep out someone who knows what s/he is doing and who is determined but it will keep out casual passers-by who might try to log into the computer.
