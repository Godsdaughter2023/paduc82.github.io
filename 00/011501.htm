a. user say:

Hello,
Yes, the Internet Explorer 11 (IE 11) does use the appropriate compatibility and security. You do not have to change anything in the settings. However, there is a significant difference between running Internet Explorer 11 on Windows a Windows 7-32 bit and Windows 7-64-bit as you asked earlier.
The Enhanced Protected Mode (EPM) in the new IE 11 adds additional security to Protected Mode and includes AppContainer and 64-bit tabs. With AppContainer turned on, IE11 checks the add-on registration for an AppContainer compatible flag. If it's present, the Browser Helper Object (BHO) or toolbar loads and runs. If it isn't present, the BHO or toolbar is blocked, and the user is notified of the EPM incompatibility.


Now, tabs running in Enhanced Protected Mode on Windows 8 run inside an AppContainer. On Windows 7, AppContainer does not exist, so EPM only enables 64bit tabs on a 64bit OS. (That also means that enabling EPM on a 32bit Windows 7 system doesn’t do anything, because a 32bit Windows 7 system supports neither 64bit nor AppContainer).



[Refer to the blog: http://blogs.msdn.com/b/ieinternals/archive/2012/03/23/understanding-ie10-enhanced-protected-mode-network-security-addons-cookies-metro-desktop.aspx]
