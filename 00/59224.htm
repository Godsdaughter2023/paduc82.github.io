firyre say:

I found a thing that worked for me. It's not great, but it does the job.

-------------------------Step One---------------------------

Try this to create a .txt file list:

https://support.microsoft.com/en-us/kb/196158

-------------------------Step Two---------------------------

Copy the data from this file and insert it in Excel. Use the Text Import Wizard to get nice little columns. If you are smart, you are already done here.

-------------------------Step Three---------------------------

If you are stupid as I am and used spaces in your file names (thusly spreading your file names into several columns) you might want to give this one a shot (tl;dr follows):

https://support.office.com/en-us/article/Combine-text-from-two-or-more-cells-into-one-cell-81ba0946-ce78-42ed-b3c3-21340eb164a6

Tl;dr: With A1 being the cell to contain the beginning of your file name use the formula =(A1&" "&B1&" "&C1&" "&D1) to combine the text from cells A1 through to D1. Variations are available, check link above for more info.

-------------------------Step Four---------------------------

If you want to go even further and be able to filter for file extensions, take a look here (tl;dr follows):

http://stackoverflow.com/questions/16063463/triming-extension-from-filename-in-excel

Tl;dr: With A1 being the cell in whitch your clean and proper file name is located, use the formula

=LEFT(A1,LOOKUP(2^15,FIND(".",A1,ROW(INDIRECT("1:"&LEN(A1)))))-1)

to output the file name only, and/or use 

=RIGHT(A1,LEN(A1)-FIND("|",SUBSTITUTE(A1,".","|",LEN(A1)-LEN(SUBSTITUTE(A1,".","")))))

to output the extension only.

-------------------------THANKS TO---------------------------

Step Three comes from an official MS site, the formulas from Step Four are copied from stackoverflow.com users barry houdini and Adarsh. Thanks, guys :)

Disable your security softwares and perform the backup to determine if it’s interferring with the  backup process. Still no go? You may follow the steps mentioned below for further troubleshooting. Ensure that you enable the security softwares once you’re done with the troubleshooting.

This problem could also be due to one or more profiles under HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows
NT\CurrentVersion\ProfileList with missing ProfileImagePath.
 
To check whether you have missing profiles:

Caution: The registry contains critical settings that are necessary for your system to function properly. Take extra caution while making changes. . For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article below:

Back up the registry
http://windows.microsoft.com/en-US/windows7/Back-up-the-registry

1.       Open regedit, navigate to the above registry key. (HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList). Expand the list
 
2.       Click on each of the profiles listed. The first 3 profiles should have ProfileImagePath value of
%SystemRoot%\System32\Config\SystemProfile,
%SystemRoot%\ServiceProfiles\LocalService, and
%SystemRoot%\ServiceProfiles\NetworkService
respectively.
 
3.       Starting from the 4th profile, the ProfileImagePath should contain path to the user profiles on your machine, such as C:\users\Christine
 
4.       If one or more of the profile has no profile image, then you have missing profiles.
 
To work around this, delete the profile in question:
1.       First, export the ProfileList key for safekeeping. (Right click on the key, choose “Export”, and save it to the desktop.)
2.       Right click on the profile in question, choose delete.
3.       Try backup again.

andre da costa say:
Well, if it was backed up on a Mac, thats a major contributing factor why its not working in Windows 10. macOS uses a completely different file system called (HFS+ OR APFS if you are running the latest version 10.13). Both are incompatible with Windows 10 which uses a different file system called NTFS. You will have reconnect it to the Mac in order to access your files. If you want share an external hard disk between Mac and Windows, you need to have the external formatted as a exFAT.
