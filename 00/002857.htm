Function ConvertToLetter(iCol As Integer) As String
   Dim iAlpha As Integer
   Dim iRemainder As Integer
   iAlpha = Int(iCol / 27)
   iRemainder = iCol - (iAlpha * 26)
   If iAlpha > 0 Then
      ConvertToLetter = Chr(iAlpha + 64)
   End If
   If iRemainder > 0 Then
      ConvertToLetter = ConvertToLetter & Chr(iRemainder + 64)
   End If
End Function

john korchok say:

There are actually quite a few ways to do this, but here's a simple one (thanks to Greg Maxey). The first step is to make the Developer tab visible in Word:

  1. In Word, click on File, then on Options.
  2. Click on Customize ribbon.
  3. In the right-hand column, check Developer. OK out.

The Developer tab is added to the ribbon. Now you can add a Content Control:

  1. Position your insertion point in the document where you want to type the company name.
  2. Click on the Developer tab.
  3. In the Controls group, select the icon that look like a black and white capital and lower-case Aa. When you hover your mouse over it, you'll see a tooltip Plain Text Content Control. Click on it.
  4. A content control is inserted in your document. While it is still selected, click on Developer>Properties.
  5. Give the control a title.
  6. Check Content control cannot be deleted. OK out.
  7. Copy the content control.
  8. Click in the document where you want the duplicate company name to appear.
  9. On the Home tab, click on the bottom half of the Paste button and choose Paste Special....
  10. Change As: to Unformatted Text.
  11. Click on Paste link. OK out.
  12. Repeat steps 8 to 11 for each place in the document where the company name is to appear.

After it's set up, you can type the company name into the content control. When you tab out of the control, all the company names are updated.

jay freedman say:
You can't do this with the Columns button. Instead, use a 2-column table. At each place where an image has to line up opposite a particular piece of text, start a new row (by putting the cursor in the right-hand cell and pressing the Tab key).

More at http://www.word.mvps.org/FAQs/Formatting/UsingColumns.htm and http://word.mvps.org/FAQs/TblsFldsFms/TableBasics.htm.

doug robbins - mvp office apps & services (word) say:
I want to keep typing in the first column of a document with two columns even when I reach the end of the page.   In the second column I then want to insert images that go with the print in the first column.  How do I make this happen without the first column automatically going into the second column at the end of the page?


One way to do this is to use a two column table.  You should however start a new row every now and then as having a cell span too many pages can cause problems.  A benefit of using a new row is that it will facilitate aligning the images with the corresponding text.

Using columns (of the Page Layout type), you can insert column breaks to control where the text is located, but this method has the disadvantage of the breaks moving if the text is edited.

rajesh sinha. say:

LD:

You can use this VBA code as Standard Module to Hide/Unhide Rows across Sheets.

Private Sub CheckBox1_Click()

Dim sh As Excel.Worksheet

If CheckBox1.Value = True Then

   For Each sh In Sheets(Array("Sheet1", "Sheet2"))

    sh.Rows("1:25").EntireRow.Hidden = True

    Next

End If

    

If CheckBox1.Value = False Then

   For Each sh In Sheets(Array("Sheet1", "Sheet2"))

   sh.Rows("1:25").EntireRow.Hidden = False

   Next

End If

End Sub

N.B. 

  • Remember this Code will work when you want to hide/unhide identical Rows across Sheets.
  • In case if you want to hide/unhide different Rows then use below shown code.

Dim MyRange As Range

Set MyRange = Range("$A$2")

If MyRange = False Then

    Sheets("Sheet1").Rows("15:25").EntireRow.Hidden = True

    Sheets("Sheet2").Rows("55:75").EntireRow.Hidden = True

ElseIf MyRange = True Then

    Sheets("Sheet1").Rows("15:25").EntireRow.Hidden = False

    Sheets("Sheet2").Rows("55:75").EntireRow.Hidden = False

End If

N.B. 

  • Cell A2 has to Linked with, Check box.
  • Sheet Name's & Row references are editable.
