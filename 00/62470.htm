    I would suggest you try the following steps and see if it helps.

    Method 1: Run Windows Update Troubleshooter.

    1. Press Windows key + X and click on Control Panel.

    2. Change View by from Category to Large Icons/Small Icons.

    3. Click on Troubleshooting and then select View all.

    4. Select Windows Update and follow the on-screen instructions.

    Method 2: Run DISM Tool.

    1. Press Windows key + X, and click on Command Prompt (Admin). If prompted for an administrative password, kindly provide the password.
    2. Type Dism /Online /Cleanup-Image /ScanHealth and click Enter.
    3. Type Dism /Online /Cleanup-Image /CheckHealth and click Enter. 
    4. Type Dism /Online /Cleanup-Image /RestoreHealth and click Enter.

    For more information about DISM click on the link below: https://support.microsoft.com/en-us/kb/947821

    (Steps for Windows 8.1 are also applicable for Windows 10)

    To download drivers from Manufacturer's Website, click on the link below and later follow the given instructions:

    http://www.realtek.com/downloads/

    1. Once you've been redirected to the site, click on High Definition Audio Codecs (Software) and then after reading the mentioned text, check the box for I accept the above and then click Next. 

    2. According to your Operating system type, choose either 32bits Vista, Windows7, Windows8, Windows8.1, Windows10 Driver only (Executable file) or 64bits Vista, Windows7, Windows8, Windows8.1, Windows10 Driver only (Executable file).

    3. Click on Global next to the respective compatible driver.

    4. Once the download is complete, run the setup file and install the software.

    5. Follow on-screen instructions.

    6. Restart your system

    Disclaimer: Using Third Party Software, including hardware drivers can cause serious problems that may prevent your computer from booting properly.
