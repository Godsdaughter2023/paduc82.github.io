.rozaly say:

According to my experience, for older programs stability to run in Windows 8 64bit, including Flight Simulator X ,

it is best to install directly to Drive C:\ do not install into c:\Program Files or c:\Program Files (x86). This

includes my stocks trading software it worked well now in Windows 8 64bit.


Set the .exe file properties to C:\Microsoft Games\Flight Simulator x\fsx.exe\Property\compatibility\Run this

program in compatibility mode for\windows 7.


For stability make changes to fsx.CFG, back-up before you do, set:

TEXTURE_BANDWIDTH_MULT=n (n=30-40 for dual core, 70-90 for quad core)


The default is 250 that makes Flight Simulator X crash so easily, so in fsx.CFG change according to your video

card memory, for example if your video card memory is 1 GB set to 1024 acordingly:

TEXTURE_MAX_LOAD=4096 (for 4096×4096 textures)
TEXTURE_MAX_LOAD=2048 (for 2048×2048 textures)
TEXTURE_MAX_LOAD=1024 (default value when global texture resolution is set to
very high)


Experiment with what suits you.
Note: Flight Simulator Service Pack 2 works with newer video cards otherwise there will be issues.
