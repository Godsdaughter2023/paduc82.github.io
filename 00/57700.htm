'rickt say:

My question is: How I can use all my memory without page file like Linux ?

Linux uses a paging file - unlike Windows you can't actually turn it off; the difference is that they have a parameter for how "swappy" the OS will act. 

Obviously, with 32GB RAM, you have no need for a swapfile if you're not concerned about crash recovery but, just as obviously, the games think you should have one.  So, like the advice says, set a minimum size of some kind. 

Of course, you could always make a ramdisk and set the swapfile in there - plenty of google hits for that.  Utterly silly of course, but it should fool your games.
