macropod say:

A significant impediment to hyperlinking Indexes to the content they reference is that a given page reference in the Index may relate to multiple items on the same page. Similarly, The Index entry's name could reference content spanning dozens or hundreds of pages. For TOCs, OTOH, there is only ever one heading in the body and one page # per entry. Continuing the TOC analogy, for it's hyperlinks to work, Word applies a hidden bookmark to each entry and hyperlinks to that entry. That approach wouldn't work for an Index whose title refers to content on multiple pages and whose page #s might reference multiple entries on the same page - the hyperlink would have to go to only one of each - and the hyperlinks in the INDEX field would each have to reference a particular bookmark. Although Word could apply bookmarks to every INDEX field, that wouldn't overcome the problem that a given page reference in the Index itself can only point to one such bookmark. Doubtless, Word could be re-programmed to apply bookmarks to every INDEX field and, every time the Index is refreshed, to hyperlink each page # in the Index to point to the first index entry on each page, but I doubt MS would regard the investment as worthwhile given the low level of demand for such functionality. It would also break the backwards-compatibility of the INDEX field, since no existing Word version would support an Index with page # hyperlinks like those embedded in a TOC.

The only option at this stage is to create the Index in the normal manner, then unlink it and create such hyperlinks as a required between the entries and the content on the pages concerned.

steve rindsberg say:

This'll fix most text on the slides, though it won't touch text in graphics, tables, charts, smart art, groups, etc:

Sub De_Pilcrow_Me()
    Dim oSl As Slide
    Dim oSh As Shape
    For Each oSl In ActivePresentation.Slides
        For Each oSh In oSl.Shapes
            If oSh.HasTextFrame Then
                If oSh.TextFrame.HasText Then
                    oSh.TextFrame.TextRange.Text = _

                   Replace(oSh.TextFrame.TextRange.Text, _

                    Chr$(182), vbCrLf)
                End If
            End If
        Next
    Next
End Sub
