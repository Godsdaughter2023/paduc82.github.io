If a xy-scatter appears to be plotting the x values as sequential values 1 to number of points it usually means that at least 1 of the x values is not a true number.

Anything in the x value range of cells that is being treated as text by excel will cause this. It could be real text or numbers being treated as text.

Assuming your data are values and not formula try putting a 1 in a spare cell, copy it, select the x value range and paste special >  multiple.
