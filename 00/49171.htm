Hi,

Is there an Out of the Box feature in Windows 10 to extract and compress files ?

Thanks

***Post moved by the moderator to the appropriate forum category.*** 

a. user say:

Hello,

Thank you for posting your query in Microsoft Community.

You can zip (compress) a file so that it takes up less space. This feature is similar to one in previous versions of Windows.

To zip (compress) a file or folder

  1. Click on File Explorer icon on the taskbar or press Windows key + E together, to open file explorer.
  2. Select the file or folder, tap or click Share, and then tap or click Zip in the Ribbon toolbar.
    (System might take some time to compress the folder)

To unzip (extract) files or folders from a compressed folder

  1. Click on File Explorer icon on the taskbar or press Windows key + E together, to open file explorer.
  2. Double click on the zip file. A new window opens.
  3. Click on Extract to and select the destination folder.
  4. Follow the onscreen procedure to complete the process.

pa bear - ms mvp say:
I want to disable Automatic Updates because my computer, which is a notebook, turns on at 3 AM and it wakes me up.

I'd be remiss to entirely disable Automatic Updates (AU) for that reason, my friend. Instead, you could...

    • Configure AU to install updates at a more convenient time (e.g., 7:00 PM).

    • Configure your Power Options such that the computer enters Hibernate (versus Sleep or Hybrid-Sleep) mode overnight. See http://windows.microsoft.com/en-US/windows-vista/Turn-off-a-computer-frequently-asked-questions

    • Change your AU configuration to the "Download updates but let me choose... option." After doing so, updates will not be installed without your explicit approval. See http://windows.microsoft.com/en-us/windows7/Change-how-Windows-installs-or-notifies-you-about-updates [good for Vista, too]

The #1 reason it's not advisable to disable AU is so that you won't miss any new security updates, some of which are released out-of-band (i.e., not on the second or fourth Tuesday of the month).

~~~~~~~~~~~~~~~~~

Tip: For maximum battery longevity, always connect the computer to an external power source when it's available and remove the battery from its bay & store it in a cool, clean location. For long-term storage, the battery should be charged to 40%, not 100%.  My current battery's still alive & kicking after five (5) years doing this!
