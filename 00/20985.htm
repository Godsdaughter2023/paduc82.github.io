cortig say:

Outlook stores all the data it is using in an Identity folder located in ~/Documents/Microsoft User Data/Office 2011 Identities
The active identity is where all e-mails get downloaded etc. It's also where Outlook will go looking for data like contact information, etc.

If the file in question is in the active identity, then a simple double-click will open it up in Outlook.
If it is in a different folder, then Outlook will look for it in the active identity, it will fail to find it and won't display anything for that record.
If you have more than one folder in ~/Documents/Microsoft User Data/Office 2011 Identities, you can launch Outlook pressing the Option key down to select which identity (in ~/Documents/Microsoft User Data/Office 2011 Identities and only there) will be used by Outlook.

graham mayor say:
As Suzanne has indicated, this is not an easy task ... even for VBA. I think that the following macro will help, but there may be a few characters missed

Dim orng As Range
Const strList As String = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    On Error Resume Next
    Set orng = ActiveDocument.Range
    With orng.Find
        Do While .Execute(FindText:="([A-Z]{2,})", MatchWildcards:=True)
            orng.Underline = wdUnderlineSingle
            orng.Collapse wdCollapseEnd
        Loop
    End With
    Set orng = ActiveDocument.Range
    With orng.Find
        Do While .Execute(FindText:=Chr(32))
            If InStr(1, strList, orng.Previous.Characters(1)) > 0 Then
                If InStr(1, strList, orng.Next.Characters(1)) > 0 Then
                    orng.Underline = wdUnderlineSingle
                    orng.Collapse wdCollapseEnd
                End If
            End If
        Loop
    End With
    Set orng = ActiveDocument.Range
    With orng.Find
        On Error Resume Next
        Do While .Execute(FindText:="I")
            If orng.Next.Characters(1).Underline = wdUnderlineSingle Then
                orng.Underline = wdUnderlineSingle
                orng.Collapse wdCollapseEnd
            End If
        Loop
    End With
    Set orng = ActiveDocument.Range
    With orng.Find
        On Error Resume Next
        Do While .Execute(FindText:="A")
            If orng.Next.Characters(1).Underline = wdUnderlineSingle Then
                orng.Underline = wdUnderlineSingle
                orng.Collapse wdCollapseEnd
            End If
        Loop
    End With
    Set orng = ActiveDocument.Range
    With orng.Find
        Do While .Execute(FindText:=Chr(32))
            If orng.Underline = wdUnderlineSingle Then
                If orng.Next.Characters(1).Underline = wdUnderlineNone Then
                    orng.Underline = wdUnderlineNone
                End If
                orng.Collapse wdCollapseEnd
            End If
        Loop
    End With
