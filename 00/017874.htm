If you're unable to insert a PDF file into the Word document, we recommend inserting a link to the PDF file instead. You can insert a link to the PDF file into your document.

  1. Save the PDF you want to use to the same location as your document.
  2. On the page where you'd like to put a link to the PDF, select the text, shape, or photo you'd like to be hyperlinked to the PDF file.

  4. Click Select and browse to the location where your PDF file is saved.

Junk Mail Protection isn't available if you have an Exchange account as your org's Ironport (or whatever message security service they're using) & SPF should be taking care of that. In which case you shouldn't be seeing a lot of junk/spam if it's been configured properly.

If you are getting a lot of spam however your message security service may have a web login user configuration page that allows you to increase your detection rate from normal to high (Postini is one such service that provides users access to their Junk mail configuration settings). 
That takes care of the spam at the gateway before it even hits your Inbox (you can also set it up to block specific e-mail addresses etc) so there's no point having access to your Junk Mail Protection settings in Outlook in that case.
If the message security service doesn't have user configurable settings you'll need to speak to your admin or your ISP to get the detection rate increased if spam is a problem for you.

But if you still want to push on and get access to Junk Mail protection you can have POP or IMAP enabled on your Exchange account and setup a new server connection using POP/IMAP - you'll then be able to play with your Junk Mail Protection settings

If POP or IMAP's not enabled your Exchange Admin should be able to switch it on in AD or your ISP may make this option available to you - all you should need to ask for are the connection settings
