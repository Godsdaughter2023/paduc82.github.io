Jump lists is an extra generation feature for navigation to files, and applications in Windows 10 like in the earlier versions on Windows. The Start on the right pane showcases four option. Read the last one, Show recently opened items in Jump Lists on Start or Taskbar.

On Start Menu, click on the arrow next to the program and its displays you the Jump Lists.

Execute a right click on a program on Taskbar and you can no more see the Jump Lists items.

macropod say:

I am trying to convert an ASCII ( .txt ) file to Word without changing any of the formatting.  

When I convert it, I have to changing the spacing and margins and still can't get it to show 25 lines per page.

ASCII files don't contain any formatting, so there's nothing to 'change'. When you open such a file in Word, though, Word attaches its 'Normal' template to the file and applies whatever page layout and 'Normal' paragraph & font attributes are set as Word's defaults. You can't change that behaviour and changing the default page layout, paragraph & font attributes in the 'Normal' template will affect all new documents based on that template (so you probably wouldn't want to do that).

What you're necessarily left with is reformatting the text file, bearing mind that, if you re-save it as an ASCII text file, all that formatting will be lost and you'll need to do it all over next time you open it.

john korchok say:

Protected View is normally invoked when documents are emailed to you or when you download them from a web site, not when you put files on a flash drive.

Documents on Flash drives get marked as read-only because editing a document while on a flash drive is the quickest known method to corrupt a document. To prevent this, copy the file to your user folder (C:\Users\YourUserName\My Documents), edit it there, then copy it back to the flash drive to move to the other computer.

You could use the Master Document feature, which combines several documents (called "subdocuments") into the master without actually doing so, since al documents remain separate.

Some experienced users here recommend abstaining from using this recourse, but I've never had any problem with them. Test it, see how it fares… and keep backups of your individual documents.

  • Create a new document (the master)
  • Select View > Master Document (a new toolbar will appear and Word will switch to Outline view)
  • Use the toolbar to insert new documents
  • Go back to Draft or Print Layout Views
