thierryschluep say:

Hide/unhide rows based on cell value of different row

Hi everyone,

I saw different threads for this question but it didn't quite have the answer that I was looking for.

The question is as following:

In Cell C19 I have a dropdown for 0, 1, and 2.

If a person chooses 0, rows 20-23 shold be hidden.

If a person chooses 1, rows 21 and 22 should be hidden.

If a person chooses 2, all rows should be visible.

How can you do that with VBA so that this works automatically (VBA)?

Thank you so much in advance

amit tandon say:

Enter below code in the Sheet module ie. the module of the sheet wherein you wish to hide rows:

Private Sub Worksheet_Change(ByVal Target As Range)

On Error GoTo ErrHandler
Application.EnableEvents = False
Application.ScreenUpdating = False

Dim rCell As Range, lRow As Long
Set rCell = Range("C19")
lRow = rCell.Row

If Not Application.Intersect(Target, rCell) Is Nothing Then
    Rows(lRow).Offset(1, 0).Resize(4).EntireRow.Hidden = False
    'If a person chooses 0, rows 20-23 shold be hidden
    If rCell.Value = 0 Then
        Rows(lRow).Offset(1, 0).Resize(4).EntireRow.Hidden = True
    'If a person chooses 1, rows 21 and 22 should be hidden.
    ElseIf rCell.Value = 1 Then
        Rows(lRow).Offset(2, 0).Resize(2).EntireRow.Hidden = True
    'If a person chooses 2, all rows should be visible (below 2 code lines can be ignored)
    ElseIf rCell.Value = 2 Then
        GoTo ErrHandler
    End If
End If

ErrHandler:
    Application.ScreenUpdating = True
    Application.EnableEvents = True

End Sub

As mentioned above, because C19 is a drop down list with only 3 possible values, below 2 code lines in the above code can optionally be ignored
    ElseIf rCell.Value = 2 Then
        GoTo ErrHandler
