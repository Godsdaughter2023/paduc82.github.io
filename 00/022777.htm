johnshields_7044 say:

Setting up shifts

I have been trying to set up two 10 hour shifts per day.  There are a couple of issues here.  One, is that the night shift runs from 9:00p of one day to 7:30am of the next day.  The second issue is the day shift starts at 7:00am which is a 1/2 hour overlap with the night shift.  So far I have not been able to set this up.  Is there a way to do it?

julie sheets say:
Hello John,

You will need two calendars - one for the day shift and one for the night shift.  You can modify the standard calendar to the day shift time and then create the new calendar for the night shift.

To modify the Standard calendar:
  1. Project tab > Change working time.
  2. Click the Work Weeks tab.
  3. Select the "Default" entry and click Details.
  4. In the Details for [Default] click and drag from Monday through Friday to highlight all working days.
  5. Click the option "Set Days to these specific working times."
  6. Edit the 8:00 am to 7:00 am
  7. Edit the lunch break as needed.
  8. Edit the end time as needed.
  9. Click OK to close the Details for [Default] and return to the Change working time window.
  10. If you are going to use that calendar as the project calendar, click the Options button and adjust the default start time and default end time as needed.
  11. Edit the Hours per day to 10 and the hours per week appropriately.
  12. Click OK to close the Options window if needed and return to the Change Working Time window.
  13.  Close the window to save the changes.

To create the night calendar - you might start with the Night Shift calendar that is included with each Project.

  1. Return to Change Working Time.
  2. At the top of the Change Working Time window, click the For Calendar: drop down and select Night Shift.
(You'll notice the night shift calendar does not span over midnight.  If you select any Monday you'll see the working time is 11:00 pm to 12:00 am.  Click any Tuesday and you'll see it starts at 12:00 am until lunch break, lunch break to end of the work day in the morning, and then starts back up at 11:00 pm until midnight the next entry.

Try using that as a guide and edit the Default Entry.  Let us know if you have any further questions.

To use the calendars, assign them to your resources on the Resource Sheet view.  Select the Base calendar for each shift.  As you assign the resources, the times of the tasks will shift to the resource's working time.
