Hi,

Your CD/DVD player might not be able to play the CD for several reasons, including the following:

a.       Your CD/DVD player may not support playback of data CD or DVD discs. Refer to the CD player's documentation for details.

b.      The CD/DVD might be burned onto media, such as CD-RW, that is not compatible with the CD player. Refer to the CD player's documentation for details.

c.       Your CD/DVD player may not support playback of the type of music files burned on the disc (WMA or MP3). Refer to the CD player's documentation for details.

d.      The songs might be protected with media usage rights. Other computers may be able to acquire usage rights and play the songs if they have a connection to the Internet, but devices such as portable CD players and car stereos will not be able to acquire those rights, so protected content will be unplayable.

e.      The CD might not have been closed when it was burned. Usually you can resolve this problem by adding more songs to the burn list so you are close to the maximum capacity of the CD. Then, try burning the list again to a new CD.

f.        It could also be an issue with the CD/DVD burner.

gene_918 say:

It took a lot of experimentation, but here is the fix, at least on my machine which was upgraded from Windows 7. Windows 10 seems to be confused about the Libraries.

1. Click the Start button.

2. Click File Explorer.

3. Scroll down and click Libraries. You don't have to open the subdirectories, just the main Library folder.

4. Close the folder.

5. The backup error messages no longer appear.

This has been a real aggravation and I'm very surprised Microsoft hasn't figured it out, unless they did and just didn't tell anybody.

UPDATE, September 28 ----

I feel I can now confirm that opening the Libraries folder has solved the problem. I can speculate that backup was still using options left over from Windows 7 that weren't  updated to reflect changes by Windows 10 to the Libraries.
