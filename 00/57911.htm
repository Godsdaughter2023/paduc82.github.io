kevin d souza say:

Hi Paula,

 

 

We wouldn't recommend you to extend the C drive where Windows is installed because extending the C drive might damage the boot files and this might lead to a no boot situation.

 

However, if you would like to do it, then there are some third party software's (partition tools) which will help you extend the C drive.

kevin d souza say:

Paula,

 

Try to extend the partition using the Disk management wizard.

 

- Press Windows key and R to open the Run prompt.

- Type diskmgmt.msc and press enter.

- Right click on D drive, select Extend volume and then follow the on screen instruction and assign a volume to be extended.

 

Restart the computer once this is done so that the changes made are applied.
