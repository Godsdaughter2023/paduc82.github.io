drumz0rz say:

Found this answer on another question. It worked for me, so I'm copying it here for anyone else who has this problem:

First go here to download PsTools. Extract it to a folder on your desktop.

Open Start and type cmd. Right-click the program and choose to run as administrator.

Navigate to the folder where you extracted the tools by typing "cd C:\Users\Username\Desktop\Pstools", replace Username with your username and Pstools with the folder where you extracted the tools.

When you have navigated to the correct folder, copy the following command without the quotes: "psexec.exe -i -s %windir%\system32\mmc.exe /s taskschd.msc".

Open the command window and right click, choose paste. Press enter.

Navigate to Task Scheduler Library -> Microsoft -> Windows -> UpdateOrchestrator.

Right click the task called Reboot and click on Disable.

The task should now be disabled.

Problems with downgrading Windows 8.1 to Windows 8.

I purchased an ASUS M11BB Essentio Series desktop computer from my local Best Buy around December of 2013. I had upgraded Windows 8.0 to Windows 8.1 last month (January 2014) and I immediately recognized the changes.

Windows 8.1 wasn't compatible with a game that I played often while I had Windows 8.0. The game is known as Need for Speed Hot Pursuit 2010 (Criterion Version). This game simply won't run with Windows 8.1, even with the full range of compatibility modes, patches, Windows 8.1 updates, and administrative privileges. I had purchased this software with the purchase of my desktop because I knew that my new system could handle this game on medium-high settings with its integrated chipset graphics. (AMD Radeon HD 8670D)


After a few weeks searching for the solution, I came across perhaps my only resort to get this game working. I attempted to downgrade my Windows 8.1 installation to Windows 8.0 using the recovery option for Windows 8.1. (Since Best Buy didn't want to make a copy of the Windows 8.0 recovery disk)



The option that I specifically chosen was the option to reset using the 8GB recovery partition that should recover the Windows 8.0 installation without removing any of my currently installed documents and files. This seemed like an excellent option to me at first, until I got a list of programs that I quote "...need the discs or files to reinstall these apps after refreshing your PC."

The list went through just about every single program that I have on my system on both my 160GB Windows 8.1 C: partition, and my 720GB Program storage D: partition. (All partitions are on a single 1TB SATA HDD including the recovery partition)


This is ludicrous, why is this worded in such a way that gives me the false hope that I could just recover Windows 8.0 and maintain my Windows 8.1 registry and Windows.old files! Why does this not work similarly to any other version of Windows OS that had the function of installing the new over the old and maintaining the files from the old system entirely! (Let alone backing up the registry and the important system files can save a lot of trouble too)

The way this was worded seemed logical, but this recovery just removes the Windows 8.1 system files and the registry keys and replaces it with the Windows 8.0 system files and registry keys, which renders all of my installed software useless if they relied on important system files and registry keys to reference serial keys and user/system authentication!

I think that there should be a more logical method to recovering and reverting from Windows 8.1 to Windows 8.0. Especially since the most common method is to upgrade through the Windows Store App.

Please don't implement a similar recovery structure in the next release of Windows. Please think of the desktop users and not towards the tablet and touch-screen market, there's a reason why such implementations "cost more and make less."

:EDIT:

Even if you bought the Windows 8.0 installation disk (mine was from BestBuy, the same store that I purchased my system from) there is still no probable method to downgrade without having to completely remove all of your files.

I have no way to back up ~ 373.8GB of programs, documents, and archive backups by any means, and I only have roughly 1 4GB flashdrive, 1 4GB MicroSD card (Camera), 1 4GB MP3 player (Storage), 1 DVD-R (4.7GB), and 1 CD-RW (700MB). I am still short of ~357GB in terms of storage in order to backup my files.

This is atrocious!

A better method would have been implemented with the Windows Store ability to downgrade the OS by making a server-side restore point, or to revert back to the capability to actually select which partition to format and clear. (The old method that was used in almost every version of Windows since Windows 7)

This is not optimal in any sense, and it leaves too much to be desired for any future versions of Windows. If Microsoft doesn't get this fixed by their next release, then they will lose money and lose trust from their community.
