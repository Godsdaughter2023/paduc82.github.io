Re-install Windows 7 II
  1. Fully backup your DATA.
  2. Use Windows Easy Transfer (built-in) to create a transfer file.  Works almost always and reproduces all your settings, backgrounds and most data.
  3. Use your hard drive manufacturer’s drive tester software to ensure you are working with a good hard drive.  If yours is a laptop with a hard drive 5 years or more, just replace the hard drive.
  4. This is likely to take an elapsed time of around 12 hours.
  5. First go to your computer’s OEM support site and find and download the drivers for your computer and store them on a USB stick.
  6. Do not do any formatting or partitioning.
  7. If you made a set of disks for recovery or an image copy at the time your computer was new.  This is the time you need them.
  8. Start your computer on the first of the disks as instructed and in an hour or so, your computer will look exactly as it did then.
  9. If you do not have that set of disks. You will need a legal reusable Microsoft Product Key.  You will need a Windows 7 install disk. The win7 disk must match the edition of your product key, and its bitness (32 or 64).  If you do not have the original Windows 7 install disk, borrow one from a friend.  Hopefully, the disk you use will be labeled SP1 (Service Pack 1), because that will save you an additional 4 hours or so.
  10. Place the Windows 7 install disk in the disk reader and start your computer.
  11. Once the install process is started, choose CUSTOM.  Ignore the check box about drivers, unless you can not proceed further.
  12. When you get to the place where it asks you where to put the installation, click Advanced and delete all partitions.
  13. When the installer asks about Windows update, choose Ask me later
  14. Do NOT install any programs, none
  15. Hopefully, you started with a Win7 install disk that was SP1 (Service Pack 1). If not, run Windows Update until you get to the point that SP1 has installed itself as part of this procedure.  It entails multiple Windows Update passes and restarts which you keep updating until you get there.
  16. Once Windows 7 SP1 is installed, install the following:
  17. KB3020369 Win-7 32 https://download.microsoft.com/download/C/0/8/C0823F43-BFE9-4147-9B0A-35769CBBE6B0/Windows6.1-KB3020369-x86.msu
  18. Win-7 64 bit https://download.microsoft.com/download/5/D/0/5D0821EB-A92D-4CA2-9020-EC41D56B074F/Windows6.1-KB3020369-x64.msu
  19. KB3138612 Win-7 32 bit, https://www.microsoft.com/en-us/download/details.aspx?id=5120
  20. Win-7 64 bit, https://www.microsoft.com/en-us/download/details.aspx?id=51212
  21. Open Windows Update, change Windows Update setting to Never check for updates
  22. Do not install anything else at this point – NOTHING
  23. Start the update process.  It is likely to include about 200. It will take some time.
  24. Once you have a list of updates, you need to prevent certain specific updates from being installed to prevent Windows 10 from taking over.
  25. Click once on each Update that is NOT labeled SECURITY and check the date of issue on the right.  If that date is after January 1, 2015, Right-click on it and hide.
  26. Click install updates and wait for it to finish.  Restart when asked to do so.
  27. After re-start is complete and you see a desktop, start Task manager – Right-click on task bar. Look at the % at the bottom.  Do NOT attempt to use the computer for any purpose until you see that % fall to and stay at 10% or less.  Windows Update is still working and has a lot of work to do.
  28. Keep running Windows Update again and again till it offers no new ones.
  29. Start Internet Explorer, click the gear (upper right) in IE11 and select Compatibility settings and enter Microsoft.com in the list
  30. Start Windows Update and click the link to include updates for other Microsoft software.
  31. Run Windows Update again and again until you are satisfied you have all the updates you want.
  32. After the install is complete, check Device Manager. Type device in the text box above the start globe when you click it and choose Device manager from the list to find out if Win7 was able to supply the drivers you need.  You should get drivers only from the maker of your computer or Intel.  Its best to get all the drivers at this point.  Win7 install may have installed all of them or nearly. Do NOT use any of those driver download sites.  They are all bogus, have bad drivers, and install malware.
  33. Install your Microsoft Office software and then run Windows Update again and again till no more are proposed.
  34. Defragment your drive.  Type defragment in the text box above the start globe when you click it once.  Choose the Defragmentation link.  Wait till it completes all passes..
  35. Next, you will need to take steps to prevent Windows 10 from taking over.  First step is to change the Windows Update setting to NEVER.
  36. Re-install your programs
  37. Use Windows Easy Transfer and fill in missing data from your backup

New install tip:

After Windows 7, system drivers and all updates are installed and any stable applications like Microsoft Office are installed and updated, and before any (not -Rs) and about an hour.  When you are done you will have a very nice bit of insurance.  Should you ever again need to re-build a corrupted system or replace a hard drive, you will have a precise duplicate of your system as it was at this point.  You can restore that image to a hard drive in about 20 minutes.  Creation of System Image is found in your menu under Maintenance, Backup and Restore.

Another great feature about using the image is that you do not need an install disk or a product key to do the re-install the next time, and you will have saved yourself all the time you put in this time.

I emphasize the need for PLUS R DVD blanks.  Do not use the more common MINUS R DVD blanks.

gza say:

The first thing I would suggest is making sure UPnP is enabled on the Cisco.  See pp. 50-51 of the manual here:

www.cisco.com/.../4028630.pdf

If UPnP is enabled and if it is a good working implementation of UPnP (a big if), you should not need to forward ports manually to get open NAT (test on the Xbox again).  Otherwise, when forwarding ports manually, the local IP address is the IP address of the device (e.g. your Xbox) to which you want to forward the port.  You can find the current IP address in the Xbox network settings.  Because that DHCP IP address can change over time, you really need to assign a static IP to the Xbox and forward to that.  However, some have experienced problems setting the Xbox One to a static IP...

gza say:

Your "modem" appears to be a gateway (modem+router), which if so creates a problematic double NAT situation.  To verify double NAT is the issue, please log in to the web interface of your Netgear router and note the WAN/Internet IP it reports (WAN IP, not the LAN IP!).  If that IP differs from the one reported by http://www.tracemyip.org, then your Xbox is behind multiple layers of NAT (double NAT) and getting open NAT will require special steps.

DLC not working when offline means the console license for the DLC is not bound to the console you're using.

 

Go to the License Transfer page and start the transfer process. It will show you your console IDs and how many licenses each has bound to it. To fix the problem, you'll have to perform a transfer to your current console.
