The code creates the list in column A of the Active Sheet.

If you don't have a column header in column A then the first cell will be empty.

Sub Test()
    Dim Alpha
    Dim Alpha2
    Dim Numb
   
   
    For Alpha = 65 To 90
        For Alpha2 = 65 To 90

            Cells(Rows.Count, "A").End(xlUp).Offset(1, 0) = Numb & Chr(Alpha)

    Next Numb

cbarnhorst say:

Did you not make a backup of your system before starting on the Insider Fast Ring?  This is the advice MSFT and all the veterans here always give.  If you don't have a backup that will get you where you want to go, then simply back up your data and install the Fall Creators Update.  You will need to reinstall your apps.

The easiest way is to go to the WIP page and click on Stop Insider Preview Builds and choose the method you want to use to return to the FCU.  

You will then be presented with some choices.  The immediate way is to choose "Roll me back to the last Windows release."  That is the same as using the Media Creation tool and takes far fewer steps.  As always, make sure any files you want are saved just in case something goes wrong.

You may also choose the slow opt-out with the last option.  That will not require any re-installation but will keep you in the program until next Spring.

ianrichardson2 say:

Opposite of Concatenate

I want to separate a text string into its individual characters, one in each column.  For example, if "Test 1" is in A1, then I want a formula to translate this into T in B1, e in C1, s in D1, t in E1, space in F1, and 1 in G1.  Is this possible, please ?
ronaldoonenil say:

Enter the below formula in B1 and then drag to the right for a least as many columns as you require to be populated with a single character

=MID($A1,COLUMN()-1,1)

vijay a. verma say:
Put following formula and drag it to right

=MID($A$1,COLUMN(A1),1)

To make it more robust so that you don't worry about to drag till what column
=IF(LEN($A$1)>=COLUMN(A1),MID($A$1,COLUMN(A1),1),"")
