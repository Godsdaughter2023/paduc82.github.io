jllatham say:

Try this, or more accurately, edit your code to use the ThisWorkbook object and its own objects.

When using the

With

End With

construct you must preface the members of the "With" object with a . to let Excel know that you are referring to a property or method for that object.  Consider these examples:

With ThisWorkbook.ActiveSheet

    .Cells(1,1).Interior.Color = vbRed

End With

Note the .Cells reference, which is in effect the same as writing

ThisWorkbook.ActiveSheet.Cells(1,1)....

But if we did it this way:

With ThisWorkbook.ActiveSheet

    Cells(1,1).Interior.Color = vbRed

End With

Because there is no . in front of Cells, Excel would actually perform the cell shading in cell A1 on the current active sheet of the currently selected (active) workbook!

Without testing, it looks like the section of code probably needs to be edited to appear something like this:

    With ThisWorkbook

Thanks for explaining that to K.  I'm snowed under at work at the moment.

But in your (like you said, "untested") code here, I suspect that the second line:

    Rows(ActiveWindow.ActiveCell.Row).Select

could fall over.  Since Rows in unqualified, Excel will select a row in the active sheet of the active workbook, not necessarily ThisWorkbook!

I suspect that using ActiveWindow.ActiveCell.EntireRow.Select would work better....

K,

Try this instead:

    Dim i As Long
   
    ThisWorkbook.Activate
   
    ActiveCell.EntireRow.Copy
    ActiveCell.EntireRow.Insert
    Application.CutCopyMode = False
   
    For i = 1 To Cells(ActiveCell.Row, Columns.Count).End(xlToLeft).Column
        With Cells(ActiveCell.Row, i)
            If Not .HasFormula Then .ClearContents
        End With
    Next i

By activating ThisWorkbook in your code, you can then be free to use unqualified:

Cells

ActiveCell

ActiveSheet

etc.

Cheers
Rich

Hi,

OneNote for Windows 10 is a built-in application in Windows 10. You can view, edit, and create notes via your Windows 10 computer.

OneNote 2016, commonly referred to as the OneNote desktop app comes with Office 2016 or Office 365. Aside from viewing, editing, and creating notes, OneNote 2016 works together with all the other Office applications.

OneNote for Windows 10 and OneNote 2016 is the same in functionality. However, there are features that are available in OneNote for Windows 10 that are not present in OneNote 2016, and vice versa. You can use the two applications side by side or you can choose one depending on the features that you need.

This is quite normal, because you don't have local notebooks, they are all on OneDrive.

So for notebooks on OneDrive you see

  1. nothing in the Default Notebook Location
  2. shortcuts in your local (!) OneDrive folder
  3. your real notebooks on https://onedrive.live.com 
  4. your backups on your hard disk
