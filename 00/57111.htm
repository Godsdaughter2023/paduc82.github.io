So you have double NAT. I'm not sure why you are using TWO routers.



Here is what you need to try:



start by writing down all the current settings in the DLink router for SSID
name, security passphrase for wifi and the admin password.




1. unplug the cable modem. power down all computers.



2. unplug the USR8004 and remove it from your network



3. unplug the DLink router and then plug the WAN port of this DLink into the
cable modem. plug any wired computers into the LAN ports on the DLink



4. Plug in the DLink router and do a factory reset the DLink router. I believe
there is a reset hole on the back where the ports are, use an unfolded paper
clip.



5. when the router comes back up (you'll know by the lights), power up a wired
computer and log in and setup the wireless exactly as you had it before and
change the admin password back to what you had it.



6. the router should reboot. when it does, plug in the cable modem



7. with the wired computer that is already up, verify connectivity



8. power on the touchsmart and anything else



this should solve the double NAT issue.
