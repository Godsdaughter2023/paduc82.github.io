ken sheridan say: I think you'd need to engineer this by determining the ordinal position of the current group in the set of groups.  The following is a simple example which returns a list of transactions where each date can be in multiple rows.  Each group with the same date is numbered from 1 upwards.  By using this number with the MOD operator the TransactionAmount is multiplied by 1 or -1 for each alternate group.  By ordering a report first by TransactionDate (the group) and then by AmountSigned, both in ascending order, the TransactionAmount values in each alternate group will be returned in ascending and descending order respectively.

SELECT COUNT(*) AS GroupCounter,
T1.TransactionDate, T1.TransactionAmount,
T1.TransactionAmount * IIF(COUNT(*) MOD 2 =0, -1, 1) AS AmountSigned
FROM Transactions AS T1 INNER JOIN
    (SELECT DISTINCT TransactionDate
     FROM Transactions)  AS T2
ON T2.TransactionDate<=T1.TransactionDate
GROUP BY T1.TransactionDate, T1.TransactionAmount, T1.TransactionID;

Note that the primary key, or any candidate key, must be included in the GROUP BY Clause to ensure that any rows with duplicate non-key values are returned twice or more where necessary.  By using instances of your report's current query in an outer query and a subquery you would be able to return the equivalent of AmountSigned column in the report's recordset in this way, and order your report in ascending order on this.

Let's start with differentiating these terms: 'contract', 'no contract' and 'unlocked phone'. When we say Contract phones, these are the phones that you can get by availing your provider's services within a certain period of time, depending on the contract that you've agreed upon. These phones are usually locked and their software has been customized by your operator. Usually, you can see your operator's logo or themes on the phone. 

No contract phones are the prepaid phones that you usually purchase from your operator. The phone can either be locked or unlocked. Unlocked phones are the generic phones. They are not factory locked and can be used with another SIM card from another operator.
