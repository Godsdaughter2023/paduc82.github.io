Solved: Outlook 2010 "you cannot delete this outlook data - Microsoft Community

fredayl say:

Solved: Outlook 2010 "you cannot delete this outlook data file"

All of the answers to this sticky situation involve creating a new profile and selecting a new data file.  I did not want to recreate my profile because I had a number of different emails to reconfigure, each with complex setup.

There is no need to create a new profile.  Simply exit Outlook, then rename the offending data file, e.g. Outlook_2018 to Outlook_2018_renamed.  Then start Outlook, it will display a message that it cannot find "Outlook_2018", simply redirect it to your new data file, e.g. Outlook_2019, and Outlook will continue to start up.  You may see two instances of "Outlook_2019" in the mailbox, that's OK.  Shut down Outlook and restart it again.  It can now find "Outlook_2019" and will only open one instance of it in the mailbox.  Problem solved.

Omitting the page number on page 1 without eliminating the footnote - Microsoft Community

srdiamond say:

Omitting the page number on page 1 without eliminating the footnote

I'm trying to eliminate the page number on the first page of a legal line-numbered pleading page. I created a section break at the page's end, but if I choose different first page footer, it deletes not only the page number at the bottom of the first page but the line numbers, which are part of the footer (apparently). How do I omit the page number on the first page without omitting what I want to retain?

yves dhondt say:

Wrap your page code inside an if field to not display it.

Double click your footer to activate. Then where you want to insert your page code, insert the following:

  { IF { PAGE } > 1 { PAGE } }

Note that the brackets { and } are not typed but produced by CTRL+F9. After you typed everything, press ALT+F9 to change from field codes to what will actually be displayed.

So you would type CTRL+F9, after which you would type IF between the brackets followed by CTRL+F9 again. You would then write PAGE between the inner brackets. Then > 1 after the inner brackets, followed by CTRL+F9 again for the second set of inner brackets in which you type PAGE again.
