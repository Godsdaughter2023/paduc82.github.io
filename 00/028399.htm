bill manville say:

Sorry you didn't understand.  Let me try again.

If you have, say, 5 keys that you want to sort by, and you want the whole table sorted by A, and for each value of A you want the rows sorted by B and for each combination of A and B you want the rows sorted by C, and for each combination of A, B and C you want the rows sorted by D, and for each combination of A, B, C and D you want the rows sorted by E then....

you first sort the data using Data > Sort and supplying 2 keys: D first and E second.

you then sort the data using Data > Sort and supplying 3 keys A first, B second and C third.

since you had previously sorted by D and then E the records for a combination of A, B and C are kept in the order that they were after the first sort, ie in the order D then E.

Try it and see.

Bill Manville. Excel MVP, Oxford, England. www.manville.org.uk

glen95 say: If you can turn on an AutoFilter for your range, you can sort on many columns!

(BUT: don't FILTER and sort at the same time... Excel can mess things up!)

If you need to sort a filtered range, I've had good success in copying the visible cells to a blank sheet, and sorting there.

Example:

dim tempSheet as WorkSheet
set tempSheet = ActiveBook.WorkSheets.Add

dim r as Range
set r = ActiveBook.ActiveSheet.Range("A1:AA5000") ' whatever range you are sorting
r.AutoFilter field:=1, Criteria1:="=MySearch"   ' whatever filter you need

tempSheet.UsedRange.Clear ' if used multiple times
r.SpecialCells(xlCellTypeVisible).Copy tempSheet.Range("A1")
tempSheet.UsedRange.AutoFilter
With tempSheet.AutoFilter.Sort.SortFields
    .Add Key:=Range("C:C"), SortOn:=xlSortOnValues, order:=xlAscending, DataOption:=xlSortNormal
    .Add Key:=Range("J:J"), SortOn:=xlSortOnValues, order:=xlDescending, DataOption:=xlSortNormal
    .Add Key:=Range("F:F"), SortOn:=xlSortOnValues, order:=xlAscending, DataOption:=xlSortNormal
    .Add Key:=Range("I:I"), SortOn:=xlSortOnValues, order:=xlDescending, DataOption:=xlSortNormal
    .Add Key:=Range("A:A"), SortOn:=xlSortOnValues, order:=xlAscending, DataOption:=xlSortNormal
End With
With tempSheet.AutoFilter.Sort
    .Header = xlYes
    .MatchCase = False
    .Orientation = xlTopToBottom
    .SortMethod = xlPinYin
    .Apply
End With

' do something with the sorted values in the temp sheet

pinfo say:

frozenjim say:
Outlook will not start in safe mode either.  Tried repairing office.  Can't even delete a profile as that hangs as well.

I've seen this problem several times and it's not tough to fix.

Rather than deleting a profile or taking any other destructive approach, I've found that you can simply create a new one with no risk.

Step One: Make sure Outlook is not running:
  • CTRL-ALT-DEL to open the Task Manager.
  • Choose the "Processes" tab to show all running processes.
  • Right Click on Outlook.exe (if it is running) and select "End Process Tree".

Step Two: Back up your profile:
  • Go to Control Panel/Mail and choose "Show Profiles..." to open the profiles dialog.
  • Rename your broken profile to "old profile"
  • Select "Prompt for a profile to be used".
Step Three: Start Outlook - problem solved:
  • Now start Outlook normally.
  • When it asks which profile to use, click on the "New..." button. 

Outlook will now simply create a completely new profile which will, hopefully, solve all of your problems.

Once you have finished this troubleshooting/repair, tell Outlook to always use your new profile (See Step Two for instructions).
