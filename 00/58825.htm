rgharper911 say:

Windows does not have any limit to the number of devices it can handle as far as I know of, this is most likely a hardware/motherboard limitation.  Not all motherboards can provide full connectivity to all expansion slots in a PCIe environment, they cheat and divide them up in such a way that not all slots can be populated or if they all are, functionality is limited.  For example, some inexpensive motherboards provide two PCIe x16 slots but if both are used each gets only x8 instead of x16 for a single slot.  Some provide three x16 slots and if only two are used they both get x16 but if three are in use the resources are split x8-x8-x8.

Check with your motherboard manufacturer to see if this is the case.
