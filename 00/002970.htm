=SUBTOTAL(9,FI$11:FI12)

When copied across to the next column in row 12, the formula will become

=SUBTOTAL(9,FJ$11:FJ11)

and copied down again it will become

=SUBTOTAL(9,FJ$11:FJ12)

OK. Don't create a script (although a script would be useful for this). One way to do this is to create a slide for each image.
