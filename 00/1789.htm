How is the text wrapping set for the text box?

stefan blom say:

There is no "lock on the page" option in Word. What you can do is lock the text box anchor to prevent it from being moved via drag and drop (the option is found in the Layout dialog box).

As you have noticed, you can't anchor a text or (or any other object, for that matter) to the actual page in Word. The object is always anchored to a text paragraph on the page. You can position the text box relative to the page edges; however, if the anchor is moved to the next page (for example as a result of editing), the text box will move with it.
