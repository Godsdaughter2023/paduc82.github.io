g.      Empty the Recycle Bin.

h.      Restart the computer.

i.       When you go back, you will notice the Size of the IconCache.db file is smaller, and the Date Modified is now the current date.

BitLocker and Other Users - Microsoft Community

dani_f1 say:

BitLocker and Other Users

Hello,

I encrypted a volume with BitLocker. I have three other users in my computer.

Now, Can other three users access to this encrypted drive?

If I decrypted this drive with my password, Do other three users do not need password for access to this drive? (Automatically unlock for other users in this PC?) 

Thanks

srimadhwa_b say:

Hi PI_Daniel,

Thank you for posting your question in the Microsoft Community Forums.

Good to know your interest in BitLocker and its features. I see that you have asked a nice question and let me explain it to you in detail.

Firstly, if there is a drive encryption on the operating system, then no users will be able to login unless they know the password as BitLocker will request for password when the computer boots. The same applies to the other partitions also. They will not be able to access it if they are not aware of the BitLocker password or recovery key.

When you unlock the drive using the Bitlocker password, and if the system is not rebooted, then other users will be able to access the drive and they need not type the Bitlocker password.

Here is the interesting part depending on your choice. If you do not want other users to access other partitions, then it is not recommended to select the option "Turn on Auto-unlock". When you set the drive to auto-unlock, it will decrypt automatically when you enter the Bitlocker password while booting.

You can set a different Bitlocker password for different partitions. Once you have logged in to your account and if you unlock the drive, the other users can access the volume if the computer is not restarted.

Only users with admin rights can modify the system settings including Bitlocker settings.

Let me know if you have any further questions related to Windows, I will be happy to answer them.

Regards,

Srimadhwa B
