bernie deitrick say:

Sub TestMacro2()


    Dim lngR As Long

    'Find the last row

    lngR = Cells(Rows.Count, 1).End(xlUp).Row

    

    'Enter the formulas

    Range("N2:N" & lngR).Formula = "=A2"

    Range("O2:O" & lngR).Formula = "=IFNA(INDEX($B$1:$M$1,MATCH(P2,B2:M2,0)),B$1)"

    Range("P2:P" & lngR).Formula = "=IFERROR(1/(1/MAX(B2:M2)),"""")"

    Range("Q2:Q" & lngR).Formula = "=A2"

    Range("R2:R" & lngR).Formula = "=IFNA(INDEX($B$1:$M$1,MATCH(S2,B2:M2,0)),B$1)"

    Range("S2:S" & lngR).Formula = "=IFERROR(1/(1/MIN(B2:M2)),"""")"

    

    'Convert formulas to values

    Range("N2:S" & lngR).Value = Range("N2:S" & lngR).Value


End Sub

jason-h. say:

How to count strikethrough cells?

Gods and mods,

 

I'm in dire need of a formula or vb script that will allow me to count cells that contain the strikethrough font.  I've done some digging, but nothing is jumping off the page at me.  Any help is appreciated.

 

Jason

hansv mvp say:

Copy this function into a module:

 

Function CountStrikeThrough(rng As Range) As Long
    Dim cel As Range
    For Each cel In rng
        CountStrikeThrough = CountStrikeThrough - cel.Font.Strikethrough
    Next cel
End Function

 

Use like this in a cell formula:

 

=CountStrikeThrough(B3:D12)

 

Remark: the formula will be updated when the sheet is recalculated. Merely turning strikethrough on or off will not cause the formula result to be updated.
