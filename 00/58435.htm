a. user say:

Licensing and activation is different in Windows 10. It's not what we've been used to in the past.

If a computer is currently running Windows 7 SP1 or Windows 8.1, no license key is needed to upgrade to Windows 10, which is why none is supplied. The upgrade is free. During the upgrade, the installer calculates an activation code based on your computer's hardware, which identifies your computer to Microsoft. Your activation code is stored in the cloud.

Once Windows 10 has activated, you can reinstall or clean install it on the same computer as many times as you like until the end of time without ever needing a license key, because your activation code for that computer is stored in the cloud.

For retail licenses of 7 and 8: Those licenses let you move to a new computer, but once you install Windows 10 on a computer it is bound to that computer. So if you want to move Windows 10 to a new computer, you have to install 7 or 8 on that computer and upgrade to SP1 or 8.1, respectively, then upgrade to 10.

But how often would you want to move Windows 10 to a new computer? Most computers by now are running Win 7 or 8 anyway, which don't need a license to upgrade to 10.  Maybe if you're building, in which case you'd have to install 7 or 8 first then upgrade to 10. Not that terrible.

a. user say:

If you want to be sure of getting Windows 10 free for life, upgrade your computer any time before 7/29/2016. The very next minute you can restore an image of your previous OS.

Once a computer has been activated, it is activated forever. That means you - or whomever else gets your computer - can reinstall or clean install Windows 10 on that same computer as many times as they like without ever needing to enter a license key.

Your right to use Windows 10 is based on activation, not license. That's the difference between 10 and earlier versions. A Windows 10 license merely gives you the right to activate Windows 10 on that computer. If your computer is running 7 or 8 you have until July 29, 2016 to activate without a license - that's Microsoft's "special offer."

Activation means that a hash calculated from your computer's hardware - the activation code - is stored online. At any time Windows 10 can recalculate the hash and compare to the hash stored online. That's how Windows 10 knows it's running on the same computer on which it was originally activated.

That means your copy of Windows 10 can only be used on the computer on which it was originally activated - it can't be moved to another computer. because another computer would have a different activation code.

"...if my PC broke and i need to use the same license into a new machine..."

The point is moot. In the first place, if you upgraded from 7 or 8 you don't have a Windows 10 license because you didn't need one. And if you bought a Windows 10 license (because you built a PC or upgraded a computer running something other than 7 or 8,) once your license has been used to activate Windows 10, the license is worthless.

So if you have a retail license for 7 or 8 which, under the old system, can be moved to a new computer, you have to move 7 or 8 to the new computer and then upgrade to 10. As long as you upgrade before 7/29/16 you can upgrade without a license.

What hasn't been settled, I don't think, is if you have 10 Home and buy an upgrade license to 10 Pro. Can you move 10 Pro to another computer running 10 Home?

    c. Right click on the desired driver, and select Uninstall.
    d. Follow the onscreen instructions to uninstall the driver.
    e. Also in the device manager windows select view and then select the show hidden devices.
    f. Again follow step b. and right click the hidden device and uninstall it.
    e. Restart the computer.
