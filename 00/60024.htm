The simple answer is: in Windows 10 there is a bug that means you cannot rename a user folder

In any other version of Windows, when you changed a user name the folder also was renamed, this is not so in Windows 10. The only solution is to create a new user account with the new name and copy over your files and folders to that new account and delete the old account . . .

hairyfool say:

The basic answer is No. The User Folder name is derived from the original User logon name unless you use a Microsoft Account where it comes from the Live ID Username (not the actual email address used to log on). It will NOT change if you change your Account user name. The folder will be pointed to at various keys in the registry, two of which are critical but others can have unforeseen results. It is not a bug but by design, why is a moot point.

You will find various threads detailing how to change it for different versions of Windows mostly by directly editing the Registry but the results are unpredictable and security settings for the folder make it difficult to change anyway.

Therefore "concrete and safe" is not possible. Having found this I now install Windows using a local account with the required user name and then convert to the Microsoft Account afterwards

Even when you create a new user as suggested you still don't get to choose the folder setting, it comes from that user name. This would be the SAFE option.

jack mvp say:

Hi

There are two aspects to this issue.

1. The laptops have physical switch (or combination Alt - Fkey) to switch the Wireless card On.
This has nothing to do with the OS or drivers that is a switch that switch the electrical circuit of the Wireless off to conserve battery.
If the switch is Off, it is like you do not have a wireless card.
Read the manual of the laptop and make sure that the switch is On.

2. Make sure that you have the latest drivers from the laptops support page.

Then Delete whatever is in the Device Manger that relate to the Wireless card, Reboot and install the Vendor's Drivers.

If Windows does not detect the card during boot then either it is Off or it is a busted card.

In general, OEM Wireless cards can be found for less than $20, and most support page provide an install manual that explains how to replace a card.
