jansn say:

How to fix disk errors

My computer has disc errors which are not fixed when I reboot. I have tried a registry fix from Symantec but it will not work because of the errrors. Also, tried System Restore and it will not work for the same reason. My computer is running ok but slow.

 

mike hall... always say:

Continual disk errors are a symptom of a drive which is physically failing. If your computer has to repair them at each reboot, it is already too late to clone the contents over to a new drive.

Your next steps should be:

  1. Order a recovery disk set from the manufacturer
  2. Buy a new hard drive
  3. Continually back up (copy) important stuff to external media until such time as you have both the above items
  4. Insert the new drive into the machine
  5. Use the recovery disks to get your computer up and running again, ensuring that you run ALL updates EXCEPT for hardware, and install a good anti-virus solution (Microsoft Security Essentials is good) before you do anything else
  6. Restore your backed up important stuff to the default folders  
  7. Continue to back up (copy) important stuff to external media in case it happens again

frederik long say: A registry "fix" cannot possibly fix a damaged disk, same as a session with your shrink won't fix a broken leg (other than perhaps letting you ignore the problem . . .).

You should do this:
  1. Back up your important files to an external device while you can.
  2. Launch the Disk Manager (diskmgmt.msc).
  3. Perform a Disk Check. Tick both boxes before starting.
