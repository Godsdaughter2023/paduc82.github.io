jack schaller say:

Creating a formula based on cell font format

I am trying to build what I think is a very simple formula, but I can't seem to find the proper syntax to complete it. I want to sum a column of numbers, IF the number does NOT have the "strikethrough" feature applied to the cell's font. Thus when I "strikethrough" a value in the column, it will be excluded from the calculated total at the bottom of the column. Seems like it should be simple, but I can't figure out what the value is to evaluate the IF statement.
rick rothstein, mvp - excel say:

You are going to need to use a UDF (user defined function) to do that. Press Alt+F11 from your worksheet (which will take you into the VB editor), click Insert/Module on the VB editor's menu bar, then copy/paste the following code into the code window that opened up...

Function CountStrikethroughs(ParamArray Rng())
  Dim R As Range, A As Variant
  For Each A In Rng
    For Each R In A
      If R.Font.Strikethrough Then CountStrikethroughs = CountStrikethroughs + 1
    Next
  Next
End Function

Now, go back to your worksheet and enter this formula...

=CountStrikethroughs(B3:F7,I6:K11)

where you would change the argument to the CountStrikethroughs function to the range (or ranges; comma separate the ranges if more than one) that you want to count the strikethroughs in.

Edit
=====
I just noticed you said to add the values if the "number does NOT have the 'strikethrough' feature"... I gave you the code for if it did have that. Here is the corrected code.

Function CountStrikethroughs(ParamArray Rng())
  Dim R As Range, A As Variant
  For Each A In Rng
    For Each R In A
      If Not R.Font.Strikethrough Then CountStrikethroughs = CountStrikethroughs + 1
    Next
  Next
End Function

terry-c say:

Saving each page of a multipage document individually

I am trying to save each page of a 100 page word 2010 document individually as html. Is there an easy way to do this?
I do not want to save the entire document as 1 html file but rather save each page and retain the formatting. The document contains themes, is it likely to retain this?
doug robbins - mvp office apps & services (word) say:
On 1/06/2010 4:41 AM, Terry-c wrote:
> I am trying to save each page of a 100 page word 2010 document
> individually as html. Is there an easy way to do this?
> I do not want to save the entire document as 1 html file but rather
> save each page and retain the formatting. The document contains
> themes, is it likely to retain this?
Use a macro containing the following code:
 
Dim Counter As Long, Source As Document, Target As Document
Set Source = ActiveDocument
Selection.HomeKey Unit:=wdStory
Pages = Source.BuiltInDocumentProperties(wdPropertyPages)
Counter = 0
While Counter < Pages
Counter = Counter + 1
DocName = "Page" & Format(Counter) & ".htm"
Source.Bookmarks("\Page").Range.Cut
Set Target = Documents.Add
Target.Range.Paste
Target.SaveAs FileName:=DocName, FileFormat:=wdFormatHTML
Target.Close
Wend

sheeloo say:

You can use conditional formatting to highlight cells in Col F which have a corresponding entry in Col G

Select Col F and in conditional formatting choose FORMULA IS option and use the following formula

=COUNTIF(G:G,F1)>0

Caution: This will highlight all entries in Col F for the case where (for example) you have 100 in multiple cells in Col F but only one entry of 100 in Col G

You can add Col H and use the formula in H1 and copy down

=COUNTIF(G:G,F1)

It will give you the number of matching entries in Col G for the entry in Col F

kosh vorlon say:

Hi Hetty,

 

First, uninstall Norton and then use the removal tool from this list List of anti-malware program cleanup/uninstall tools to get rid of the remnants that will be left behind and completely remove it from your system.

 

To install or re-install MSE, check out this article and follow ALL the procedures:http://answers.microsoft.com/en-us/protect/forum/protect_start/check-list-for-installing-microsoft-security/bf757e6a-e320-4a67-92bc-767e6acb26c4.  Think of this more as a checklist because it covers a lot but everything you need is below.

 

To install, go to http://www.microsoft.com/en-us/security_essentials/default.aspx and click on more languages and versions and then choose your language and the appropriate version for your operating system. When you download, save it to the desktop (rather than running it). When finished, uninstall and remove any remaining AV program you retained for protections during the download and then reboot.Then right click on the installation icon and click run as Administrator (even if you’re logged in as an administrator) as I’ve found this sometimes helps. 

 

Installing MSE will automatically disable Windows Defender if it was running and should enable Windows Firewall if that was disabled by Norton.  If either don't happen, you'll get an error message and you can post here with the word-for word copy of the message and we'll help you set things right. 

 

If it didn't do so automatically, restart the computer.

 

Then you need to configure your settings, set a scheduled scan, enable real-time protection, update MSE, and then run a quick scan (or a full scan if you have time).  It's really just a matter of opening MSE and checking out all the tabs, starting with the Settings tab which I'll cover a bit below.  Then in Settings, click on Real Time Protection and make sure the box is checked and save it.  Then click on Scheduled scan and choose to do a quick scan once a week when the PC will be on.  Then go to the Update tab and update MSE.  Then go to the Home Tab and run a Quick scan and follow that this one time with a Full Scan.

 

I also recommend the following setup for MSE (though the decision is yours - I've just found this helpful and the most effective and best way to set it up and is how I have mine configured):

 

Make sure that ALL threat levels are sent to Quarantine where you can then decide whether or not to allow them or leave them there, or remove them. It is the safest solution for having the most options for dealing with the situation. Quarantine is as safe as removed, but in some cases you may want to remove them to be more certain - and with this one I would remove it (when the time comes). To do this, open MSE, click on the Settings Tab, and for each of the threat levels, set the action to be Quarantine and save.

 

Go to the History tab and click on allowed and unless you specifically placed something in allowed, remove everything else from the allowed category. Only put things there that you ABSOLUTELY know are not problems (are "false positives") and triple-check before you do it as doing this could allow a real threat to bypass security and infect you and it won't be detected or rather won't be reported or acted upon by MSE.

 

Go to the Settings Tab and click on MAPS and click on Advanced and save. This is not at all dangerous to you and no information sent will be used by Microsoft to identify you even if some minor identification information is sent in terms of defining the infection, but it can sometimes help with new threats by initiating an immediate update to deal with the problem if it is in the MAPS database (and it helps Microsoft update definitions to provide you and everyone else better protection).

 

Go to the Settings Tab and click on Advanced and check the box for save Quarantine items and change it from one month to three months so you have more time to decide if you want to restore something.

 

If the scan picks up anything, please make a note of what it is exactly and place it in a reply so we can deal with any required malware-removal (and even if it removes what it finds, it's a good idea to double-check to be on the safe side - but we'll cross that bridge if we need to).

 

*************************************************************

 

If malware was found and removed, then you need to again uninstall and re-install MSE as it will never work properly if installed on an infected system (which we didn’t know at the time).

 

If you have any troubles other than malware (in which case post back here), then post back here or open a support case describing the problem starting at: https://support.microsoftsecurityessentials.com/.

 

I don't think I've ever explained this any simpler or shorter than this.  Feel free to post with any questions about anything at any time and for any reason at any point in the process.

 

I hope this helps.

 

Good luck!

 

Kosh
