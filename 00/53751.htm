Host Embedded Controller Interface (HECI) driver is a software interface that is used to communicate to the AMT subsystem (Management Engine) to access AMT capabilities. Recovery Drive D is almost full in Windows 7

foxfifth say: Nothing new should be going to that drive.  If you have been making backups and not paying attention to where the backup is going, that may be the problem because sometimes a backup will default to that drive.  You should try to identify what new things have gone to the drive and remove them.  Recovery drives (partitions) normally don't need more space than how they were sized by the hardware manufacturer as their intention is that nothing more than the recovery information that was originally put there by the hardware manufacturer should ever be written to that drive.  You may just need to look at what else has been going there and move it.  One common issue is for backup software to write to that partition and that is not a good idea as any hardware failure on the drive is likely to result in the loss of that partition as well.

Hi John,

Thank you for posting in Microsoft Community.

Microsoft doesn't provide any indicator, only the toggle keys. Windows itself has never had such a feature.  You must have been using some other program on your old computer, perhaps a program that you downloaded from the Internet or a program that your computer manufacturer pre-installed for you. As a basic troubleshooting step, I recommend that you repair the email app by following the instructions below:

 

1. Click Start (a circle button with a Windows logo in it).

2. Tick Control Panel.

3. Click the icon beside the word Programs.

4. Tick Programs and Features.

5. Search for Windows Live Essentials then double click it.

6. Select Repair all Windows Live programs.

 

Before doing so, I suggest that you import and export email, contacts, and calendars with Live Mail by following the steps on this link to prevent losing your data.

 

If the issue persists, please perform clean uninstall then reinstall the program as follows:

 

1. Press Windows + R, input AppWiz.cpl, and click OK.

2. Select Windows Essentials 2012/Windows Live Essentials, click Uninstall/Change.

3. Click Remove one or more Windows Essentials programs-or-Remove.

4. Tick Mail.

5. Click Uninstall.

6. Download Mail and install.

 

Let us know how things turn out.
