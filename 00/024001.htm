It seems you talk about local notebooks, stored on your computer.

Some facts about local notebooks:

A local notebook is nothing more than a folder within the Windows file system. Its name is the notebook's name . It contains one or more section files, the .one files. These files contains the real stuff, the pages.

These notebooks/folders are located by default under 

C:\Users\<user>\Documents\OneNote-Notebooks

These folders have to be copied/moved to the new computer.

If you only copy a .one file and open it in Onenote using File, Open, Onenote will not recognize it as a notebook. This has the result that this file will always be closed by closing Onenote.

To open a notebook you either have to open its folder by File, Open or by just double click one of its .one files.

If you've done it the right way, the notebook will stay open even after closing Onenote ....

Validation rules for dates? - Microsoft Community

jacie say:

Validation rules for dates?

How do i write a validation rule for a date field?

I am working on a database involving project records.

I have a start date and an end date in a form(and table, but i dont want the uses seeing the table). I want to make sure that the end date is at least one day more than the start date(aka, if it takes less than 24h, it should get entered in the database).

Im hoping there is some simple way to do this. thanks :)

allen browne say:

Open the table in design view.

Open the Properties box.

Beside the Validation Rule in the Properties box, enter an expression like this:

([start date] Is Null) OR ([end date] Is Null) OR ([end date] >= [start date]+1)

Be sure to use the validation rule in the Properties box (the rule for the table), not the one in the lower pane of table design (which is the rule for a field, and so inadequate for comparing 2 fields as you don't know which one the user will enter first.)

More info on validation rules:

http://allenbrowne.com/ValidationRule.html
