cbarnhorst say:

Hello ,

How often Microsoft releasse windows 10 Patches or updates ?

If you are not an Insider you will receive patches on the second Tuesday of each month, called "Patch Tuesday."  There is a second Patch Tuesday on the fourth Tuesday of each month on which you may also receive patches.  In cases of "urgent" OS vulnerabilities you may receive urgent patches at any time.

You will receive Defender malware definintion patches automatically several times a day but will not be notified.  

If you are an Insider in the Release Preview ring you will receive the same patches as non-Insiders and at the same times.  At one time Release Preview Insiders received these patches a few days in advance of public release in order to help Microsoft prevent bad patches from reaching the public.  That practice has been abandoned by Microsoft, no reason stated.  

If you are an Insider in the Slow Ring you will receive complete new builds of the version of Windows 10 currently under development about once a month.

If you are an Insider in the Fast Ring you will receive complete new builds of the version of Windows 10 currently under development about once a week.

There is no guaranteed schedule for the Insider Rings, but this is the current practice.

You can be in different Rings on different devices.

dcadon1962 say:

I have solved this myself

from the main screen - swipe left and look for 'Outlook Mail'

click on Hamburger sign (top left corner - 3 bars)

select mail that was unpinned..... and hold down.... Unpin from start (again) from here

Select the same mail program, and then select 'Pin to Start'

Locate the 'new' icon on the start menu, and hold down long.... 

you will then be able to move it to the previous location that is now blank

All Done.
