It sounds like someone manually changed the original file extension from when the file was saved. Are you sure it is even an Excel file?

Your idea of fiddling with a copy is a good one, as you don't want to make changes to the original while troubleshooting.

The default file format for Excel 2008 was .xlsx. Changing the file extension tells the OS what app to use to open the file and also tells the application what kind of file it is. When you change the extension from whatever it was originally, you are lying to both the OS and the App about what to expect. Only if you happen guess the actual kind of file it is will it open. Choosing Excel File Formats in Office 2011 for Mac
