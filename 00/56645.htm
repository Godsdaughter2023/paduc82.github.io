Hello Chiho,

Thank you for posting your query in Microsoft Community.

It happens with improper selection of Time zone. Try the below steps and check if they helps.

  1. Click Start, type Date and Time into the search space and select the same from the list.
  2. Click Change time zone and select the appropriate time zone there and click Ok.

don cuthbert say:

Is this on Dell's recovery partition for Factory reinstall or on the Microsoft Recovery partition? Actually, no matter which, the files are rarely used. The Dell partition almost never, unless you want a factory restore, and Microsoft's has so few files that it seems unlikely to be fragmented. If it is the Microsoft partition, you could probably move the file off and back on to defragment it.

I doubt this condition will ever have any negative impact, in any event. Sometimes there are unmoveable files, such as swap files, that prevent full defragmentation. I have at times in the past deleted these to defrag a disk, but now I ignore fragmentation and let Windows do it automatically.
