karl timmermans [outlook mvp 2012-2018] say:

Just a couple of things for sake of accuracy

#1 - Removing a POP3 email account doesn't remove the associated PST file for the account which is what you're seeing in the Outlook Navigation Pane

#2 - No real point maintaining the PST file associated with an IMAP account for the Email folders since that file is a copy of what's contained on the server

With that said

#3 - As Palcouk mentioned, maintaining a PST file on network drive in an active Outlook profile is not supported and can lead to data corruption. If you've been working in an environment where that has been the case (using a network drive) and no problems were encountered, simply means that you've been lucky.

#4 - What's missing from the description for the newly configured IMAP account is that the PST file that was associated with the POP3 account was also configured as the default data file for the profile which means that's where the default contact/calendar folders are located. It also means that you cannot delete the file from the profile until another data file is designated as the default which cannot be the PST file containing the email folders for the IMAP account

palcouk say:

The use of a pst on a network drive is not supported by MS as it will lead to pst corruption (you can backup to a network provided the pst is not connected to MS Outlook)

You cannot backup an IMAP data file, an IMAP data file on the local PC is a cached copy of what is on the server

Your Contacts/Tasks/Calendar will still be located on the pst as you have Office 2010 (If outlook 2016 this is held within the IMAP data file, on folders marked 'This PC only' resetting the mail account will cause these to be lost)

Michael JamesDoyle -

Thank you for posting on the Windows Phone Community.

Unless some of those emails have very large attachments, it is rather odd to have messaging take up that much space.  The one possibility I can think of, is if you have a large amount of emails in other folders that you aren't aware of.

To check this, open your email inbox.  Tap on the ... and then select Folders.  Then tap on Show All Folders.  This will show folders other than just your inbox or sent items, including Junk and Trash.  You can go through those folders to make sure you don't have anything taking up a large amount of space.
