Excel for Mac uses nearly identical ODBC functionality as Excel on the PC.

I have a detailed tutorial on how to set up everything on the Mac here:

A key factor is choosing the right ODBC driver (they're inexpensive). For a file based data source (one you can navigate to using Finder) use the Actual Technologies driver. If you're going through a network connection use the Open Link driver.
