teylyn say:

Hello,

you cannot use structured references in conditional formatting formulas.

You can apply a conditional format that uses regular cell references to a table column and that format will be applied to new table rows.

It gets VERY messy when new rows are inserted and deleted in the middle of the table. Then Excel creates individual rules for inserted rows, and the list of rules will grow and grow.

Overall, basic conditional formatting and Excel Tables work well, but more complex scenarios with formulas don't work so well.

marcine lohman say:

I have found a solution that seems to work for me but it is only for Windows 7.  Our problem was with the Windows 7 Credential Vault.

 

Windows 7:
 1. Close Outlook.
 2. Click Start.
 3. In the search box enter "Credential". You will then see Credential Manager.
 4. Select Credential Manager.
 5. Under Generic Credentials, you will most likely see an entry that has Outlook in the name.
 6. Delete it. In fact, you can delete all the entries in Generic Credentials if you like.
 7. Close the Credential Manager window and open Microsoft Outlook.
 8. If it asks for a password, enter the password and check for it to remember the password.
 9. A new entry will be created in the credential vault with your latest login and password.
 10. Problem fixed.

 

The problem for us has been that the saved password in the credential vault file doesn't appear to work (especially if we've changed the password recently) so by deleting it, another credential will be created the next time you log into Outlook with your latest password.

 

I hope this helps.
