Guys, i've FINALLY found an answer to how to get this to work again, (courtesy of andy562)!  Andrés this may work for you too. 

The following finally fixed all issues for me (Store and all apps broken with black X, can't access PC Settings):

A.) Fixing the app store:

  1. Open registry editor typing regedit.exe from a command line.
  2. Browse to the registry key at HKEY_CURRENT_USER\Software\Classes\Local Settings\software\microsoft\windows\currentversion\appmodel\repository\packages.
  3. Right click on the “packages” key and bring up the “Permissions” tab.
  4. Click the “Advanced” button located at the bottom right corner.
  5. Check to see the account name that shows up as the “Owner” (this is the first line of text on the “advanced security settings” dialog for the “packages” key). It should say “SYSTEM”.  (For me this showed the built-in local Administrators group)
  6. If it is any other account, click on the “Change” button next to it.
  7. Type SYSTEM in the “select user” dialog. If “Multiple Names Found” window comes up, select the row for “SYSTEM”.
  8. Click OK to the dialogs and close them all.
  9. Run from admin command prompt:     Powershell -ExecutionPolicy Unrestricted Add-AppxPackage -DisableDevelopmentMode -Register $Env:SystemRoot\WinStore\AppxManifest.xml

B.) Fixing all modern apps:

  1. Run from admin PowerShell:

((Get-ChildItem "HKLM:SOFTWARE\Microsoft\Windows\CurrentVersion\Appx\AppxAllUserStore\InboxApplications") | Get-ItemProperty).Path | Add-AppxPackage -Register -DisableDevelopmentMode

    * When I ran this the black X persisted until after the next step and reboot.  But trying to start an app gives me a new message and a link to the store.

C.) Fixing 3rd party apps:

  1. Delete the regkey:HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Appx\AppxAllUserStore\[UserSID] 

          You can find your SID by going to an elevated command prompt and typing

                        wmic useraccount get name,sid

D.) Restart.  After a reboot the apps still showed the black X for a minute or so, but it soon cleared up on its own and all apps seemed to be working again.  I was also able to update the apps from the app store once signing in with a Microsoft account.  Finally, I am again able to get into PC Settings.

That implies that you do see the right-click menu.  (Control Panel is a bad example because it isn't there any more.)

Previously your examples (with no other contradictory details) would suggest that you were looking at a secondary screen.  E.g. previously left-click on the Start button on a secondary Taskbar would show on the primary screen; so if the primary was a phantom the user would never see it.  Similarly I would suspect that your attempt to start PowerShell would imply that the UAC prompt appears only on the primary screen and nothing can happen until it is answered or cancelled.  You may be able to get some assistance with what you can't see by starting Narrator--press Win+Ctrl+Enter.

hairyfool say:

In desktop mode right click on the Start Icon and select Control Panel.

Double click the Keyboard utility and switch to the Speed tab.

There are two value sliders here. The repeat delay sets the time taken for repeat to start i.e. having displayed the 1st character then how long until the 2nd (repeated) character appears.

The Repeat rate sets the delay between the 2nd character appearing and then the 3rd, the 3rd to 4th and so on.

So a long delay will probably stop most of your second characters but not stop them altogether. As I rarely repeat characters deliberately I have this set to Long but it can affect other programs that use the repeat function to detect that you are indeed holding down the key although this is not that common.

To turn off the autorepeat altogether you need to go to the Ease of Access Centre also in Control Panel. Look for the option Make the keyboard easier to use and in there Set up Filter Keys. However some of the options are a little obtuse. For example the opposite of Turn on Repeat Keys and Slow Keys should logically be Turn off... but in Microsoft Speak it is Turn on Bounce Keys (?) You can try various options and there is a test box for you to try the settings. Clicking on the Cancel button will undo the changes and removing the tick from Turn on Filter Keys will revert back to the Keyboard Utility settings.

Gordon is correct.  Someone who knows what to do can stand behind you and guide you through once or twice and then you'll have got the hang of it.  If you have any visitors who know how to use computers, they'll probably get you up & running inside ten minutes.

Windows 7 has "Wordpad", a "Word processor" and it is this that you want to use.  There are other word processors, such as Microsoft Word [part of Microsoft Office] that are much more capable but WordPad is pretty good anyway [WordPad is much more capable than the first word processor I had to use in 1984].

The Windows help system has a guide for you.  Just open Help & type in using Wordpad.  The same page is available online at Using WordPad.

You can experiment on your own and you might find that you take to it easily.  Just to get you started, here's a little exercise.

1a  Start WordPad - method 1 Click on the Windows 7 Start button in the bottom left corner of your screen.  At the bottom of the window that pops up, there is a search box with some grey lettering "Search programs & files".  In that box type wordpad and you'll see the shortcut to WordPad appear at the top of the window, click on it and WordPad will start.

1b Start WordPad - method 2 The next time you want to start Wordpad try this procedure instead - all methods are as good as each other.  Click on the Windows 7 Start button in the bottom left corner of your screen.  Because you have already used Wordpad you should see the shortcut to WordPad appear in the list of that Start menu window, click on it and WordPad will start.

1c  Start WordPad - method 3 The next time you want to start Wordpad try this procedure instead - all methods are as good as each other. Click on the Windows 7 Start button in the bottom left corner of your screen.  Near the bottom of the Start menu window you'll see the entry "All programs", click on it.  Some other shortcuts will appear and many will have the same icon [a folder icon].  The first of the folder icons is called Accessories, click on it.  You'll see the WordPad shortcut appear in the list of accessories, click on it and WordPad will start.

2 You can now see the Wordpad window.  There is a toolbar at the top and the big blank area represents a sheet of paper.  Type any old rubbish and you will see the letters appear.

3 You need to "Save" what you are typing to be able to look at it again later on.  It is a good habit to Save whilst you are working but you must save before trying to close Wordpad.  Near the top left corner of the window, there's an icon that looks like a clipboard labelled paste & just above that there's a much smaller icon with a small downward arrow next to it.  Click on the icon or its arrow & a "dropdown" list appears - Save is an entry in this list - click on it.  The Save dialog box appears.  Now, I don't know how you have organised your computer but one of the items that appears in the left hand side of the dialog is "Desktop" with a small picture to its left that is supposed to look like your Wiondows 7 computer screen.  Click on Desktop.  Next, click lower down to the right of the entry "File name" - it is probably saying Document.rtf - and type in any name you want [Exercise 1, if you like].  Then click near the bottom right  where there is a "button" with the word save on it. Your work has now been written onto your hard disk and can be recovered later on.

4 You can continue working [typing something useful this time] and can save as often as you can be bothered.  When you have finished the letter / grand opus, save it again.  If you try to save later on using the same document name then the PC will ask if you are sure you want to overwrite the previous document - in this case you'd say yes because you've just been updating the same piece of work.

5 You said you wanted to print - the first thing to do is set the paper layout up.  Click again on the small icon above Paste and then click on Page setup.  In the dialog window that appears, check the paper size makes sense to you [I'm in England so mine says "A4" size but I think, if you are American, it will say Letter].  You can change the paper size to suit the size you use in your country [& hence what you have put in your printer].  You don't need to worry about any of the other settings for this exercise but you can, for example, change the margins to suit your preference.  Then click on okay.  This takes you back to the main Wordpad window.  If you changed the paper size or other settings then things may have shifted around a bit.

6 You said you wanted to print - so here goes.  Click again on the small icon above Paste and then click on Print.  The print dialog opens up.  Your installed printer should be shown at the top.  Near the bottom is a button labelled Print - click on it.  The PC will pause while it & the printer talk to each other and then printing will start.

7 Now you have finished what you set out to do so you can close WordPad.  Click again on the small icon above Paste and then click on Exit [there are other ways of doing this, by the way].  As it closes, it might ask you if you want to save again, just say yes.

8 Minimise all your open windows by, for example, clicking on the small rectangular box at the very right-hand end of the Taskbar [probably therefore in the very bottom right-hand corner of your screen].  You will see an icon on your desktop called Exercise 1 or whatever you called your file.  Double-click on it [double-click just means click twice very quickly] and your saved file will open in WordPad.  You can continue working on it whenever you want using this method.  For now, just close Wordpad again.

9 For your next exercise, you can save a document with a different name and you'll be able to see both of them on your desktop.  If you try to save later on using the same document name then the PC will ask if you are sure you want to overwrite the previous document - normally you would not unless you'd just been updating the same piece of work.

10 Now go to the help system & look for using Wordpad [or use the online version I gave you the link to].  You will be taken through explanations of the clever things you can do in Wordpad that would be difficult or impossible if you were typing something straight onto paper.
