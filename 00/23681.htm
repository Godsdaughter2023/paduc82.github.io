suzanne s. barnhill say:
There is an even more complex relationship of styles in Word 2007 and above. When you modify defaults (using the Default... or Set As Default button in the Font or Paragraph dialog), you're actually changing the document defaults, which are reflected both in the Normal style and in "(no style)." You can change the Normal style without affecting the document defaults, but if you change the document defaults you change not only Normal (if it hasn't been modified) and all the styles based on it but also all styles based on "(no style)."

for those that still need to keep their files in legacy .doc/.xls/.ppt '03 format, the actual problem, for my company anyway, is that there are missing visual basic references that need to be unlinked.

right-click the "file" ribbon title, and click "customize the ribbon..."  in the main tabs area, check the "developer" box.  close this menu.

the "developer" tab should appear in your ribbon bar.  click this tab, then click the "visual basic" button.  in the visual basic interface, click the "tools" menu, and click on the "references..." option.  uncheck any references that start with "MISSING"

close the visual basic interface and you should be able to save your document without the above error message.

    Exclude hidden cells in a subtotal 103 (counta) not working - Microsoft Community

    sean in mi say:

    Exclude hidden cells in a subtotal 103 (counta) not working

    =SUBTOTAL(103,F5:BRL5) is my current formula. I have tried 3 & 103, but it counts hidden cells either way.
    When i add value (text or #'s) to the range, I want it counted. But when I hide it, I want that count removed.
    hansv mvp say:

    SUBTOTAL(103, ...) omits hidden rows from the count, but not hidden columns.

    You'll probably need a custom VBA function to do what you want:

     

    Function CountAVisible(rng As Range) As Long
        Dim cel As Range
        For Each cel In rng
            If cel.EntireColumn.Hidden = False And cel.Value <> "" Then
                CountAVisible = CountAVisible + 1
            End If
        Next cel
    End Function

     

    Use like this:

     

    =CountAVisible(F5:BRL5)

     

    This function is not dynamic, you'll have to recalculate it when necessary.
