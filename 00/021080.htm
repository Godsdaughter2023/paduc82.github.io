andreas killer say:
I'm sorry, but I said I can't use "for each" or any loop because not all worksheets need a graph.

Of course, you can!

If you want to add a chart to most sheets, then you can easy skip some sheets.

Use the LIKE-Operator and/or a SELECT-statemant to identify this sheets.

And/or maybe we can use another trick to identy a sheet wich should get a chart, for Example, we can look at Range("K22:M26") and if there are numbers in it, then this sheet get's a chart.

See an example of both below.

Andreas.

Sub AddAllCharts()
  Dim WS As Worksheet, C As Chart
  Dim AddIt As Boolean, R As Range
 
  For Each WS In Sheets
    AddIt = True
   
    'Check name of sheet
    If WS.Name Like "Data*" Then
      'Dont't add
      AddIt = False
    Else
      Select Case WS.Name
        Case "Setup", "Temp"
          'Dont add
          AddIt = False
      End Select
    End If
   
    'Check Data of sheet
    For Each R In WS.Range("K22:M26")
      If Not IsNumeric(R) Then
        AddIt = False
        Exit For
      End If
    Next
   
    If AddIt Then
      WS.Select
      Set C = Charts.Add
      With C
        .SetSourceData Source:=WS.Range("K22:M26"), PlotBy:=xlColumns
        .ChartType = xlColumnClustered
        .HasTitle = False
        .Axes(xlCategory, xlPrimary).HasTitle = False
        .Axes(xlValue, xlPrimary).HasTitle = False
        .Location Where:=xlLocationAsObject, Name:=WS.Name
      End With
    End If
  Next
End Sub
