Sub MkVideo()
If ActivePresentation.CreateVideoStatus <> ppMediaTaskStatusInProgress Then
ActivePresentation.CreateVideo FileName:=Environ("USERPROFILE") & "\Desktop\test.wmv", _

    Using COUNTIF function with RANGE stored in a separate cell - Microsoft Community

    radarservices say:

    Using COUNTIF function with RANGE stored in a separate cell

    I'm using the COUNTIF function within a formula which refers to a RANGE in another spreadsheet. A new spreadsheet is created each month so I'm trying to find a way to update the formula each month without having to type out the whole thing again (mainly because I want other people to do this for me who won't know how to recreate the formula.

    Here is the formula in full:

    =IF(ISERROR(COUNTIF('[QS EBD February 2016 v1 Excpetion.xlsb]Data'!$S$7:$S$1929,CONCATENATE(B5," ",C5,"1"))/COUNTIF('[QS EBD February 2016 v1 Excpetion.xlsb]Data'!$R$7:$R$1929,CONCATENATE(B5," ",C5,"1"))),"",COUNTIF('[QS EBD February 2016 v1 Excpetion.xlsb]Data'!$S$7:$S$1929,CONCATENATE(B5," ",C5,"1"))/COUNTIF('[QS EBD February 2016 v1 Excpetion.xlsb]Data'!$R$7:$R$1929,CONCATENATE(B5," ",C5,"1")))

    If you ignore the "ISERROR" this is the short version:

    =COUNTIF('[QS EBD February 2016 v1 Excpetion.xlsb]Data'!$S$7:$S$1929,CONCATENATE(B5," ",C5,"1"))/COUNTIF('[QS EBD February 2016 v1 Excpetion.xlsb]Data'!$R$7:$R$1929,CONCATENATE(B5," ",C5,"1"))

    I have both spreadsheets open at the same time when I input this formula so haven't put the full filepath. And I known "exception" is spelled incorrectly but that's how it was named...also I appreciate the various CONCATENATES could be refined but again this is linked to another user spreadsheet...

    So instead of putting '[QS EBD February 2016 v1 Excpetion.xlsb]Data' is there a way to reference another cell in the same spreadsheet which contains a hyperlink to the external spreadsheet?

    rory archibald say:

    You could use INDIRECT.

    Assuming that A1 contains: [QS EBD February 2016 v1 Excpetion.xlsb]Data

    Your simplified formula would become:

    =COUNTIF(INDIRECT("'"&A1&"'!$S$7:$S$1929"),CONCATENATE(B5," ",C5,"1"))/COUNTIF(INDIRECT("'"&A1&"'!$R$7:$R$1929),CONCATENATE(B5," ",C5,"1"))

    The major downside to this is that INDIRECT is volatile so your formula will recalculate whenever the workbook does, even if none of its inputs have changed. (It also won't work with a closed workbook, but nor will COUNTIF anyway.)

    If you only have one external link in the workbook, it might be easier to show the users how to update that link using the Edit Links dialog, or use a simple Find/Replace on the workhook/sheet name.

    ashish mathur say:

    Hi,

    Both COUNTIF() and INDIRECT() functions will stop working when the source Excel workbook is closed.  To extract data from a closed Excel workbook, you will have to use the SUMPRODUCT() and INDIRECT.EXT() functions.  The INDIRECT.EXT() function is available in the Morefunc add-in.  For further details, you may refer to my solution at this link.
