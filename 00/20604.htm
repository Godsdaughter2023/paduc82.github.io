VBA code to export Table data to excel (many table but one xls - Microsoft Community

irshadalam say:

VBA code to export Table data to excel (many table but one xls file)

I am looking for a vba code to export each table data to a new sheet but in one excel file.

For example :

TblEmployee

TblQuotation

TblCustomer

To be exported to one excel file but in different sheet.

Please advice.

hansv mvp say:

1) If you want to export ALL tables to Excel:

Public Sub ExportAll2Excel()
    Dim dbs As DAO.Database
    Dim rst As DAO.Recordset
    Dim strFile As String
    ' Modify path and filename as needed
    strFile = CurrentProject.Path & "\Export.xls"
    Set dbs = CurrentDb
    Set rst = dbs.OpenRecordset("QTblNames4Expo", dbOpenForwardOnly)
    Do While Not rst.EOF
        DoCmd.TransferSpreadsheet TransferType:=acExport, _
            SpreadsheetType:=acSpreadsheetTypeExcel9, _
            TableName:=rst.Fields("Name"), _
            FileName:=strFile, _
            HasFieldNames:=True
        rst.MoveNext
    Loop
    rst.Close
End Sub

hansv mvp say:

Specify the same filename each time in DoCmd.TransferSpreadsheet. For example:

Sub Export2Excel()
    Dim strFile As String
    Dim varTable As Variant
    ' Modify path and filename as needed
    strFile = CurrentProject.Path & "\Export.xlsx"
    For Each varTable In Array("TblEmployee", "TblQuotation", "TblCustomer")
        DoCmd.TransferSpreadsheet TransferType:=acExport, _
            SpreadsheetType:=acSpreadsheetTypeExcel12Xml, _
            TableName:=varTable, _
            FileName:=strFile, _
            HasFieldNames:=True
    Next varTable
End Sub
