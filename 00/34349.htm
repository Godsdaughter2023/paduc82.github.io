a. user say:

Hi,

If a drive is not formatted using NTFS or FAT32 file system, then it would displayed as RAW format. Windows won’t be able to read/write in this format. So you need to format the drive using a file system. The recommended file system NTFS file system. Refer to these steps to format the drive using NTFS file system:

  1. Press Windows key +X, select “Disk management’.
  2. Right click on the external hard disk and choose format.
  3. Select the file system as NTFS and format the drive.
  4. You can also right click on the drive in File explorer, select Format and do the same.
