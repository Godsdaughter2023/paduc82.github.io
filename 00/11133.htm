Between (IIf(IsNull([MinDate]),#1/1/1900#,[MinDate])) And (IIf(IsNull([MaxDate]),#12/31/2100#,[MaxDate])) Or ([MinDate] And [MaxDate]) Is Null

1. If [MinDate] and [MaxDate] are null, all records are displayed. -This is good.

2. If [MinDate] is null, records with the date less than [MaxDate] are shown and all records with blank date fields are displayed. -This is acceptable

3. If [MaxDate] is null, records with the date less than [MinDate] are shown and all records with blank date fields are displayed. -This is acceptable

4. If [MinDate] and [MaxDate] are not null, then records with the date between [MinDate] and [MaxDate] are displayed, but also all records with blank dates are displayed. -This is bad.

I'm open to any idea, including an expression that does not show blank date records for results 2 and 3, or adding another parameter that asks the user if they want to show records with blank date fields which would turn 'on' everthing after the 'Or'.

franky chen msft say: Hi Belette2B,

Yes, you can use the cmdlet Set-CASMailbox to enable or disable IMAP for mailboxes in Office 365.
Please refer to:
https://support.microsoft.com/en-sg/kb/2416434
https://technet.microsoft.com/en-us/library/bb125264(v=exchg.160).aspx

Since you need to disable IMAP for all mailboxes except one, you can run the cmdlet Get-Mailbox | Set-CASMailbox -ImapEnabled $false to disable IMAP for all mailboxes, and then run the cmdlet Set-CASMailbox -identity *** Email address is removed for privacy *** -ImapEnabled $true to enable IMAP for the specific mailbox.

murpthesurf say:

change fonts on Appointment input screen, (Office 365, Outlook 2016)

How do I change the fonts on New Appointment (input screen), (Office 365, Outlook 2016)

[Moved from:Office/Outlook/Windows 8/Office 2013]

ariel acu say:

Hi,

When creating an appointment in Outlook, you can change the font styles and size used in the body text by going to the Format Text tab in the ribbon. However, this doesn't change the font used in the subject of your new appointment. If you want to change the font used in the subject when it appears in your calendar, you can change it by performing the following steps:

  1. In your Outlook calendar, go to the View tab.
  2. Select View Settings.
  3. Click Other Settings.
  4. On the pop-up window, click Font for you to change the fonts used. (Note: After you change the font, this will be applied to all calendar events.)

Let us know if that works for you.
