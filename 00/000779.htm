To make a window "always on top" you may use an AutoHotkey script:

1. Paste the following line into Notepad and save the file with .ahk extension ( in "Save as" dialog, Save as type: all files; File Name: script.ahk )

^SPACE::  Winset, Alwaysontop, , A

For the scenario you described, I suggest you use the following steps:

  5. Then move the corresponding mailbox to Office 365.
