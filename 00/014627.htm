Up to 2016 even if you saved an excel document would always open to position to top left corner and about 1/3 of the screen filled, whether a blank document or a saved document. Clicking on the green stoplight button would open the saved document to the initially saved size.

There is one way that works but requires the use of created template. 

  1. Open Excel and set up the page, so it fills the page the way you want it.
  2. Customize as to Font you want to use and Font size, and orientation, but leave the rest uncustomized.
  3. Save the document as a ".xlst" file. You can name it "Sheet.xlst" 
  4. Now Remove the extension by click on name and change to "Sheet."
  5. Quit Excel.
  6. Go to Applications Folder.
  7. Locate Office2011 Main folder open.
  8. Find a folder called start or startup.
  9. Open the folder and drop that template you just created in that folder.
  10. Quit out of everything.
  11. If properly done, next time you open Excel the document should open to fully open.

Now if you add additional sheets you will have to customize each other sheets.

To take care of that repeat the above except name document Worksheet or Workbook and place in the same start folder the each additional sheet will open pre-formated

Oh in 2016 once you save an excel document it will keep its settings, in Excel 2011 and lower,  it wouldn't.
