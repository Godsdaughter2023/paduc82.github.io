a. user say:

I am the author of CallStation. As usual, the Microsoft support engineers here will never believe or admit that there is anything wrong with the Windows operating system, and will always blame some third-party application or driver. As Cheryl said, this is NOT a problem with the CallStation software or the way that it is installed. Also, this is NOT a problem with the modem drivers. This is most definitely an operating system problem.

Just to clear things up, TAPI and Unimodem are part of the Windows operating system, and have been every since Windows 95 came out. Unimodem is the "telephony service provider" used by TAPI to communicate with standard modems. A Unimodem Audio Device (often referred to as the modem wave driver) allows applications like CallStation to play/record audio through the modem. The modem wave driver is NOT something supplied by the modem manufacturer, it is a part of the Windows operating system. Now in Windows 10 the modem wave driver no longer works.

Normally, when a TAPI compliant voice modem is installed, the modem wave driver can be easily found in the Device Manager under "Sound, video and game controllers".  It is shown as a "Unimodem Audio Device" (half-duplex or full-duplex), and if you look at the properties it is tied to your modem.  After the Windows update, this driver just disappeared.

Instead, clicking the promotion simply brought me to the store to install it, and I could install it because I previously acquired the license.
