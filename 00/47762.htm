The "Support Engineer" has misunderstood your problem.

Open an Elevated Command Prompt (click Start Orb > type cmd.exe > right-click on the link to cmd.exe and select "Run as administrator -- respond to UAC prompt as necessary)

Type the following command and press Enter

   net localgroup HomeUsers

You should get a result like this:

C:\Windows\system32>net localgroup HomeUsers
Alias name     HomeUsers
Comment        HomeUsers Security Group

Members

---------------------------------------------------
Administrator
HomeGroupUser$
<user 1>
<user 2>
<user 3>
The command completed successfully.

where the <user n> are the names of your user accounts.  If all 4 of your user accounts appear in this list, stop here, because you have a different problem than the one I'm thinking of.

Assuming that the 4th user account is NOT listed, type the following command in the Elevated Command Prompt window and press Enter

     net localgroup HomeUsers <user 4> /add

where <user 4> is the account that hasn't had access to the HomeGroup.

I don't think you have to reboot, but do so anyway.

If your router is dual band, you will need to access its Graphical User Interface (GUI) and set the wireless frequency to 5GHz. You need to open a browser and type-in the router IP address. Usually it's 192.168.0.1 or 192.168.0.1, but it depends on the manufacturer.

If you are not well-versed in the router settings, I suggest that you reach-out to the manufacturer of the router so that they can safely guide you on how to set the wireless frequency of your (if it is a dual band) router.
