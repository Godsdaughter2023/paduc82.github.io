    obsessive power say:

    The Xbox 360 gives the same error when failing to read a game running off the hard drive as it does when reading from a DVD game disc.  Since you've also stated that you are receiving disc read error messages from your installed Mass Effect game, it sounds to me that the hard drive could be the fault.

    Does Mass Effect have a disc read error at the beginning when launching the game, or does it happen sometime later?  If it happens when the game is launching, then the issue is with the disc.  If it happens much later on, then the problem is likely with the hard drive.  Installed Xbox games only access the disc at the beginning, they don't access it again during play.

    There are a couple of things you can try:

    1. Delete one of games you are having problems with.  Download a large demo, or some videos from the marketplace, then download the game again.  This will place the game you purchased to a different area of the hard disk.  See if you get an error now.

    2.  Try downloading the game to USB storage.  To do this you will need to add an item from the marketplace first and then add your purchased game second.  This will place the second item as 'pending'. Press A on the pending download, select 'change storage device'.  Change the download location to your USB storage.  You can now cancel the first item if you wish.  Your game should then be downloaded to external storage.

    Once the game has finished downloading to the external storage, try running it again to see if the issue has gone.
