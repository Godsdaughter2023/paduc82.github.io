Temp files are normally stored in the user’s temp folder. Typically that would be:

C:>Users>UserName>AppData>Local>Temp

With all applications closed, that folder should be empty. So after a fresh boot, navigate to the folder and delete all its abandoned files (though there may be a couple that the AV creates freshly after a startup that cannot be deleted).

Similarly, when all apps are closed, there shouldn’t be any temp files in the documents folder either, so delete all those too.
