t. valko say:

Hello,

What is the syntax for returning a value in a color other than black? We want the values greater than zero to be green and less than zero to be red.

Thank you


You can do this by using conditional formatting.

Let's assume you want to apply this formatting to the range A1:A5

Select the *entire* range A1:A5 starting from cell A1. Cell A1 will be the
active cell. The active cell is the one cell in the selected range that is
not shaded. The formula will be relative to the active cell.

Goto the menu Format>Conditional Formatting
Select the Formula Is option
Enter this formula in the box on the right:
=A1<0
Click the Format button
Select the Font tab
From the Color drop down select a shade of RED
OK

Click the Add button

Select the Formula Is option
Enter this formula in the box on the right:
=AND(COUNT(A1),A1>0)
Click the Format button
Select the Font tab
From the Color drop down select a shade of GREEN
OK out
