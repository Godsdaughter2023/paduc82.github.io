Sub GetWordDocContents()
  Dim oWord As Object
  Dim vFiles
  Dim iFile As Integer
  Dim R As Range
  Dim strName as String
  vFiles = Application.GetOpenFilename("Word files (*.doc*),*.doc*",Title:="Please select the files you want to copy from", MultiSelect:=True)
  If TypeName(vFiles)="Boolean" Then Exit Sub   ' Cancelled
  Set oWord = CreateObject("Word.Application")
  oWord.Visible = True
  Set R = Worksheets.Add.Range("A1")
  For iFile = LBound(vFIles) To UBound(vFiles)  
    oWord.Documents.Open vFiles(iFile)
    strName = oWord.ActiveDocument.Name
    oWord.ActiveDocument.Tables(1).Select
    oWord.Selection.Copy
    ActiveSheet.Range("A1") = strName
    ActiveSheet.Range("A2")
    Activesheet.Paste R
    Set R = Cells(ActiveSheet.UsedRange.Rows.Count+1,1)
    oWord.ActiveDocument.Close False
  Next
  oWord.Quit
  Set oWord = Nothing
  ActiveSheet.Columns.Autofit
End Sub

Sorting data numerically when some numbers are followed by letters and - Microsoft Community

alexkohl say:

Sorting data numerically when some numbers are followed by letters and symbols in Excel 2007

I have a lot of data in one column that I need sorted in numerical order, however some of these numbers are followed by slashes or letters. For example: 4728/P/T or 4727A. I can't seem to find a way to sort these in numerical order and get excel to disregard the letters/symbols after some of the numbers. It always shifts anything with a letter down to the bottom of the list and I spend hours just trying to numerically organize 500 cells. If anyone has a solution to this that I'm not seeing, please let me know. Thanks!!

ronrosenfeld say:

Extract the numerical part into a "helper" column.  Then sort on the helper column.

Since your numbers are at the beginning of the string, you can use this formula to extract the numeric portion:

=1/LOOKUP(2,1/MID(A1,1, ROW(INDIRECT("1:15"))))

The 15 should be at least as long as the longest number to be extracted.   I chose 15 because that is the limit of significant digits in Excel.

amit tandon say:

Will a macro solution work? For Excel:


Assuming data in A1:A500, create a helper column B by entering this formula (as an array formula - CTRL+SHIFT+ENTER) in cell B1 & copy down:

=--LEFT(A1,COUNT(--MID(A1,ROW(INDIRECT("1:"&LEN(A1))),1)))

This extracts the numerical portion & then select both column A & B, do a custom sort on column B > this will sort the original values (numbers followed by letters/symbols) in column A.

v_agarwal say:
