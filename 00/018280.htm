MS tells you to look at Add-Ins, while in Outlook Safe Mode. If it closes, you can't look at the Add-ins.

I think I have the solution.  I am presuming that you have several e-mail accounts being "controlled" by the Outlook client. Look at them in Windows Control Panel >User Accounts>Mail>Outlook. When the pop-up dialog box appears, select "Show Profiles." If you have any MS mail accounts (Hotmail, Livemail, Outlook, etc) that are using "ActiveSynch" server, remove them from the Profile. If it doesn't allow you to just remove the offending e-mail account and rebuild it, you will need to create a new Outlook Profile. The default name is "Outlook." Create the new Profile with a different name, such as Outlook2, New Outlook, etc.
