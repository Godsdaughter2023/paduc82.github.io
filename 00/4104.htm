bob jones aka: cybertaz say:
What you're looking for is a Decimal Tab Stop. Although you can get to the Tabs dialog from the Format> Paragraph dialog in Word, you can't do the same in PowerPoint.

In the Table cell use the instructions you'll find in PPT Help: Set or clear a tab stop

Note: In order to use a Tab Stop in a PPT table cell you must use the CTRL+Tab keystroke because the Tab key alone takes you to the next table cell.
