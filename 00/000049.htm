  1. Turn on volume for your computer.

  2. Start Excel.

  3. Press ALT+F11 to open VBA editor.

  4. Go to the Immediate pane/window (if you don’t have it, press CTRL+G)

  5. Paste this line:  print CommandBars.FindControls( ,23).Count

  6. Press Enter.

mike h.. say:

Hi,

 

I don't have Excel 2013 so haven't used IFNA but the concatenation methods will be the same as earlier versions.

 

E18 will display whatever it's formatted to display and that may be different to the underlying value. When you concatenate you get the underlying value so you must format the concatenated value like this

 

&TEXT(E18,"0.00")&

 

This would display 2 decimal places, if you wanted none then you could use.

 

 

&TEXT(E18,"0")&
