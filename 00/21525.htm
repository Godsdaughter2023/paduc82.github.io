jim liu msft say:
Hi Avian,

OneDrive for Business uses the two-way synchronization and always sync the last changes, no matter where you made it (whether in the cloud or in the local synced folder. It won't create duplicate files if we copy the file to the two locations at the same time.

Specifically:
1. For the local synced folder, when you put the file into it, it will replace the downloaded one.
If you put the file before OneDrive for Business finish downloading it from the cloud, it will start syncing the current one and won't download a duplicate file.

2. If the file has been uploaded to the cloud, when you try to put the same file in the cloud, it will block this operation and shows an error.
