doug robbins - mvp office apps & services (word) say:

If the SAVEDATE field is in the body of the document, use CTRL+a to select everything and then use CTRL+F11 to Llock the fields in the document.  If you later want to Unlock the fields use CTRL+SHIFT+F11

If the SAVEDATE field is in the header, you will need to access the header to lock the fields.

Alternatively, you could use a macro with the following code:

With ActiveDocument
    For i = 1 To .Fields.Count
        .Fields(i).Locked = True
    Next i
    For j = 1 To .Sections.Count
        With .Sections(j)
            For n = 1 To .Headers.Count
                With .Headers(n).Range
                    For i = 1 To .Fields.Count
                        .Fields(i).Locked = True
                    Next i
                End With
            Next n
             For n = 1 To .Footers.Count
                With .Footers(n).Range
                    For i = 1 To .Fields.Count
                        .Fields(i).Locked = True
                    Next i
                End With
            Next n
        End With
    Next j
End With
