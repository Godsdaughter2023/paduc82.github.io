# Old fix from Sept 2014 Green Button at

# 65.55.5.170 data.tvdownload.microsoft.com

# changed IP addresses from old fix to the following (may need to remove if updated in future):



65.55.186.113  data.tvdownload.microsoft.com

72.246.56.69  cdn.epg.tvdownload.microsoft.com

3. Save the file, then at the command prompt run:

ipconfig /flushdns

4. Select 'Get Latest Guide Listing' in MCE/Tasks/TV/Guide to update guide.

I'm sure that technicality is useful, but I'm not sure what it means with respect to my question.

Not wishing to be rude, it means that you have a certain level of computer knowledge but there may be some quite basic things that you don't know how to do.  When you click on a folder and accidentally move the mouse at the same time it's easy to accidentally drag and drop the folder into another folder (using the technique you were unaware of).  That folder then vanishes from sight.  When people tell me that their files have vanished it almost turns out that they have been accidentally moved but they don't have the computer skills to conduct a thorough search.

If your pictures are somewhere on your C: drive then we need to search for them.  All the pictures taken on my camera have file names DSCN followed by a number (then a .jpg file extension).  Take a picture on your phone and connect it to the computer so you can view the format.  Suppose it is DSCN777.jpg .  Now we can guess that the missing pictures might include DSCN750.jpg so we search for that.  Open File Explorer and navigate to your pictures folder.  In the search box at the top right enter DSCN75.  You'll probably find that the search works so fast that it already shows you all pictures with names that begin DSCN75 which might be DSCN75.jpg from along time ago and DSCN750.jpg to DSCN759.jpg.

If the search finds a missing picture then right click it and choose Open file location (down at the bottom of the menu list).  The address bar will then show you the location of the folder where that file resides and you will see all the pictures in that folder.

If that does not work, widen the search to the C:\Users folder or even the entire C: drive.
