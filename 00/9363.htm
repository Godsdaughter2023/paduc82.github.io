robert aldwinckle on forums say:

keys I press on the keyboard (on both my PC using Windows 7 64 bit and my Surface RT 1) do not always register in the text  box


Sounds like a performance problem with scripting.  E.g. even in here there are scripts driven by each keypress to identify Ctrl- sequences.  So, if it was possible to toggle off scripting when all you are doing is typing you would be depending only on the performance of your keyboard, not how fast a script can be driven and respond for each keyboard action.   In fact, some forum software does have a mode which is better than that, e.g. when you switch to its HTML source mode you are less likely to be impeded by such filtering.  However, the only time that I have been aware of characters actually being lost in such a scenario is when there was an interfering incompatible driver also in the picture; a fingerprint reader is an example of that IIRC.  Otherwise all you have to do is believe your typing worked no matter how slow your typeahead is.  It's when you start second guessing your keyboard that you get into trouble, e.g. typing the same character more than once or backspacing (or using a cursor movement) in the same unacknowledged input, can result in unintentional overstrikes and garbled input.

andreas killer say:

Hi ShyButterFly,

the reason is that the FORMAT function looks into the systems settings and tries to guess the date when you pass a string as argument. Furthermore the returned formatted string depends also on the systems settings, e.g.:

Sub Test()
  Debug.Print Format(Now, "dd/mm/yyyy")
End Sub

Outputs
  07.04.2016
on my German system.

So you should:

a) Get the real date and store it into a date variable
dim D as Date

D = CalendarForm.GetDate

b) Pass the date to the Format function if you want to reflect the systems settings or

c) Use your own Format function as shown in the code below.

Andreas.

Sub Test()
  Debug.Print MyFormat(Now, "dd/mm/yyyy")
End Sub

Function MyFormat(ByVal D As Date, ByVal Mask As String) As String
  Mask = Replace(Mask, "yyyy", Year(D))
  Mask = Replace(Mask, "mm", Right("0" & Month(D), 2))
  Mask = Replace(Mask, "dd", Right("0" & Day(D), 2))
  MyFormat = Mask
End Function
