I'm not saying that you have to manually create bookmarks for cross-references. I'm saying that Word always creates bookmarks (hidden ones) when you choose to create a cross-reference.

The problem with bookmarks is that they may expand to include excessive content and then the cross-references will be incorrect. To avoid the problem, always add content at the beginning of paragraphs, not at the end.

Once the problem has occurred, the easiest fix is to recreate the cross-reference.

As long as you are careful when dealing with documents containing cross-references, updating does work. For example, you can use Ctrl+F2 followed by ESC (which switches to Print Preview and then back to your previous view).

j.e. mcgimpsey say:
Is there a way to ensure that the column widths remain fixed no matter who opens the doc?

In a word, "no". You've taken several of the steps that make it more likely that workbooks will open the same way on two machines, but XL is not a layout application, and there are several other factors that affect column width.

  • Your user may be using a different Standard Font - column width is set in units of "numeric characters" (i.e., 0-9, which are almost always uniformly wide to allow vertical alignment). Even if XL displays the width in inches, changing the default font or font size will cause the columns to space differently.
  • Fonts of the same name do not always have the same metrics. Different versions or manufacturers can cause small but significant changes. Using fonts that came with Office makes it more likely that things will look the same.
  • Print Drivers cause a wide variety of variability in on-screen and printed layout, including margins, kerning, and font spacing.
  • Fit to... settings can cause havoc with spacing.

just_a_user_6545 say:

VBA for goal seek using a range of cells, with a range of goals?

I have very-little-to-no knowledge of VBA and I'm trying to construct a macro using goal seek. 

What I'm looking for is:

I have a range of cells, say O9 to O150 which are the cells that I'm trying to change using Goal Seek (the ''by changing cell'').

With a range of values that are my goal values from B9 to B150 (the ''target value'')

With a range of target values from AY9 to AY150 (the ''set cell'')

For example, I want to change cell O9 (the variable cell) with the value found on B9 (the set value) to match cell AY9 (set cell),

and so on with O10, B10 and AY10, etc.

How can I create a macro to automatically Goal seek all of the above cells?

Thanks

Note: The range AY contains formulae, the other two contain just values.

andreas killer say: 
Sub Test()
  Dim B As Range, O As Range, AY As Range
  'In each cell in column B from B9 down till the last used
  For Each B In Range("B9", Range("B" & Rows.Count).End(xlUp))
    'Refer to the cells in the other column
    Set O = Range("O" & B.Row)
    Set AY = Range("AY" & B.Row)
    'Apply the goal seek
    AY.GoalSeek Goal:=O.Value, ChangingCell:=B
  Next
End Sub

Sub Test()
  Dim B As Range, O As Range, AY As Range
  'In each cell in column B from B9 down till the last used
  For Each B In Range("B9", Range("B" & Rows.Count).End(xlUp))
    If IsError(B) Then GoTo Skip
    'Refer to the cells in the other column
    Set O = Range("O" & B.Row)
    Set AY = Range("AY" & B.Row)
    'Apply the goal seek
    AY.GoalSeek Goal:=O.Value, ChangingCell:=B
Skip:
  Next
End Sub
