doug robbins - mvp office apps & services (word) say:

Hi Jim,

Maybe it was not your intent, but you sort of implied that the only way to have a picture in a text box was to use it as the fill for the textbox.

It may not be the same on a Mac, but in a Windows version of Word, it is possible to insert a picture into a text box (in the regular manner), but by default, it is inserted so that it is "in-line-with-text" and as a result, you cannot get the text to wrap around the picture and even inserting an Advance up field before the text, will only allow you to get the first line moved up

However, if the reason for using the text box is so that you can get other text to wrap around the textbox, you could instead make use of a table that is set to have the text wrap around it (assuming that option is available on a Mac) and then you can have the picture in the table with text set to wrap around the picture and text in the body of the document set to wrap around the table

Another way is to insert the picture into an blank document and then use a screen capture program such as SnagIt from www.techsmith.com to capture the image plus a suitable white border around it so that when you use that screen capture as the fill for the textbox, the picture is positioned where you want it and you can then, by setting a suitable tab stop in the text entered into the text box, using the tab key where required, get the text to appear "wrapped" around the image.

As far as I can tell, a picture can only be inserted into a frame with the text wrapping set to in-line-with-text, so it is pretty much the same situation as with a text box, except that you do not then have the picture fill option that would allow you to use the last method detailed above.

Sorry, I assumed you knew how to create new bar styles.

1. Go to Format/Bar Styles group. Click on the Format icon and select Bar Styles

2. In the Bar Styles window scroll to the bottom of the list

3. Add as many new bar styles as you want for your custom summary bar colors. From your list, it looks like you will need 5 new bar styles

5. In the Show For...Tasks, enter "Summary,Flag1" for the first new bar, "Summary,Flag2" for the second new bar and so forth
