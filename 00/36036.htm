chris r0kn1c say:

My 5GHZ wifi network is no longer appearing (revisited)

I came across a thread by Everclarity dated Nov '17 reporting that they were no longer seeing their 5G network on their local wifi. I saw several responses and some people were helped by these responses. I had the same problem but mine persisted despite any corrective actions I took based off the given responses. I have since figured out the problem and corrected it. However, the old thread discussion is now locked so I cannot add my solution to it.

For those still experiencing a problem locating their 5G wifi, these steps worked for me. I hope they work for you too. (This is for windows 10)

1) Find the link to 'This PC' or 'My PC'. I open a folder and it was in the left hand bar.

2) Right click on 'This PC' and select 'Manage'

3) Find and click on 'Device Manager'. (I'm sure there are other ways to get to this point but this was the one I found)

4) Click on the dropdown arrow next to 'Network Adapters'

5) Find the network adapter that corresponds to the one you have, right click, and select uninstall. (If in doubt, uninstall all of them that look promising)

6) Now restart your PC. That's all. Your 5G network should appear back under your available networks.

Good luck

I would check the status of the Print Spooler Service.

Click on the Start button and select Control Panel, Administrative Tools, Services. Locate Print Spooler in the list. Check that it has started and that the StartUp type is Automatic.

I suspect that it has stopped and that just starting it will not solve the problem.

You may need to show hidden files. Type Folder Options in the search box above the Start button and select View, Advanced Settings and verify that the box before "Show hidden files and folders" is checked and "Hide protected operating system files" is unchecked. You need to scroll down to see the second item. You should also make certain that the box before "Hide extensions for known file types" is not checked.
