hairyfool say:

The normal definition of xml is Extensible Markup Language and is most used in web development. There is no logical reason why scans would contain xml within them.

All files have an extension which is used by Windows to identify which program to use to open the file. Word uses .doc or .docx, Excel .xls or .xlsx, Adobe uses .pdf etc.

Right click on one of the files and select properties, in the general tab it will show the type of file and its extension.

You refer to "initials pvt" but pvt is not a common file extension and is used by a couple of applications but neither have anything to do with scans or graphics.

Until you have worked out what is actually happening make a copy of the files (copy the directory as a whole) and leave the originals untouched.

Although Word will work with graphics files that is not normally the native program to use, can you explain the steps you are using to open the files.
