doug robbins - mvp office apps & services (word) say:

The issue with the document was that there were cross references where the referenced text included a Section Break that was at the end of a landscape oriented section of the document.  As a result when such cross reference fields were updated, the text in them included the Section Break which resulted in the part of the Section, before where the cross reference was located, taking on the orientation of the Section that contained the referenced text.

 

The issue is not unique to Word 2013 and the same behavior occurs in earlier versions.

suzanne s. barnhill say:
If you are applying only a single format (Bold, Italic, Underline, All Caps), I would use the keyboard shortcut for each (you can turn any one of them off using Ctrl+Spacebar). If the formatting you have to apply combines two or more of these, then it's probably worthwhile to create a character style and assign a keyboard shortcut to it; note that the keyboard shortcut for the character style will not be a toggle (as the other formats are), but it, too, can be removed with Ctrl+Spacebar.

davem121 say:
Hi Craig,

Type some text, then select that text, right click and choose Font. Set that to Bold, All Caps and underline. Click OK

Select that text and click Styles - Create a Style

Give the style a meaningful name

Both that style and normal will appear beside each other in the Styles section of the Home Ribbon (Tab)

Right click each style and choose 'Modify'

At the bottom left of the resulting dialog, click Format - Shortcut Key

Assign a keyboard shortcut to each style
