' Saves all other workbooks in the same folder
' with this file as comma separated values (.csv) files
'NOTE: In multi-page workbooks the page selected/active
'when the other workbook(s) was/were closed will be the sheet
'exported/saved to the .csv file.
'
  Dim any_xls_File As String
  Dim rootPath As String
  Dim myName As String
  Dim newFileName As String
  Dim separatorPosition As Integer
 
  'get name of this workbook to prevent converting it
  myName = ThisWorkbook.Name
  'get location/folder to process
  rootPath = Left(ThisWorkbook.FullName, _
   InStrRev(ThisWorkbook.FullName, "\"))

  'going on with opens/closes of the files
  Application.ScreenUpdating = False
  'begin looping through possible files
  Do While any_xls_File <> ""
    If any_xls_File <> myName Then
      'change filename to .csv
      'and create well formed path for it
      separatorPosition = Len(any_xls_File) - InStrRev(any_xls_File, ".")
      newFileName = rootPath & _

    'get next possible filename
    any_xls_File = Dir$()
  Loop
'work is completed

On the random computer that is saving a blank CSV this is because the CSVs can only have one sheet. The vba is saving some other sheet than the one you want to save to csv, hence it is blank. palcouk say:

A pop mail account will not sync with other devices

The default setting for a pop mail account is to download all mail on the device on which the account is opened.

If you have several devices with pop accounts set up you need to set the option on each device, in accounts setup, to keep a copy on the server.

However sent mail will only appear on the device on which it has been sent

An IMAP account will sync mail only, on other devices
