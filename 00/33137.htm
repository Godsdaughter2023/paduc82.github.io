                   c. Click Default and then click OK.

                   d. On the File menu, click Save As, and save the bitmap under a new name.

                   e. Exit and restart Paint.

Hi Shivam,

when you get a new build, windows will create a windows.old folder containing the previous version of windows sometimes files are erroneously moved into that folder as well - check there for any user files.

if it is not there then I doubt you will be able to recover the file, if windows needed the space it has probably used the space freed by deleting the file and therefore has overwritten and thing that might have been recoverable.
you can try using something like recuva from https://www.easeus.com/ad/recover-hard-drive.ht...

bare in mind though that anything you write to the disk is likely to overwrite anything that may have been recoverable, and reduce your chances of getting anything back.
it is better to remove the disk and use another pc to attempt recovery.

The windows.old folder also takes a few GBs of space on the disk and is used if you need to roll back to the previous build.
you can remove it using Disk cleanup, if you don't want to roll back.

After you have recovered your lost data, download and install Treesize from https://www.jam-software.com/treesize_free/
this will tell you which folder is using the space allowing you to investigate.

running beta versions of any software can cause data loss, if you want to run Insider versions of windows it is essential that you have backups, so that you can easily get back to a working state when things go wrong. (or don't work as expected)

windows backup is being deprecated and Microsoft recommend using third party software for backups.
