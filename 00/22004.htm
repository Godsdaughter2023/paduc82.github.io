Sub RenameName()
  Const Title = "Rename Name"
  Static OldName As String, NewName As String, Temp As String
  Dim N As Name
  Do
    OldName = InputBox("Existing name:", Title, OldName)
    If OldName = "" Then Exit Sub
    On Error Resume Next
    Set N = ActiveWorkbook.Names(OldName)
    If N Is Nothing Then
      MsgBox "Name '" & OldName & "' doesn't exists. Please try again", vbExclamation
    End If
  Loop Until Not N Is Nothing
  Do
    NewName = InputBox("New name", Title, NewName)
    If NewName = "" Then Exit Sub
    N.Name = NewName
    If N.Name = OldName Then
      MsgBox "Can't rename '" & OldName & "' to '" & NewName & "'. Please try again"
    End If
  Loop Until N.Name = NewName
  Temp = OldName
  OldName = NewName
  NewName = Temp
End Sub

You have to cheat to fool them. You can't leave the surname field blank, and you can't just put an ordinary space there - or even a non-breaking one. But you can put a 'special' space instead. To get one,

  1. Type charmap into the start search box and press Enter.
  2. Select Segoe UI for the font
  3. Click Advanced view
  4. Select Unicode for the character set and Group by > All
  5. Type space into the search box and click Search.
  6. Click in the (highlighted) first box in the grid and then press the right arrow on your keyboard a couple of times. You'll see the names of various spaces at the bottom of the box.
  7. Click Select and then Copy (you won't see the character you're copying - that's the idea).
  8. At outlook.com, click on your display name and select Edit Profile.
  9. Click Edit at the top of the page.
  10. Paste (Ctrl-V) into the Surname box and click Save. 

You should now have a blank last name for your account.

jay freedman say:

The command you want is the Combine command, which you'll find by clicking the Compare button near the right end of the Review ribbon.

You can only combine two documents at a time. In the dialog, select your original document and one of the edited documents (and select the types of tracked changes and comments you want), and click OK. On the next pass, select the result of the first pass as the 'original document' and another edited document. Keep repeating this, adding the changes in the next edited document to the accumulated changes from the previous pass.
