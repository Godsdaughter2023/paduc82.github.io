Pivot table works depend on which data you have selected. By the way if you wish you can go by this way to sum of your stock field:

- In Pivot table move the "Field name" to the "Values" box which you want to do SUM.

- Then "click on the "Field Name" in "Values" box, and select "Value Field Settings"

- While a dialog box appear select "Summarize values by" tab and again select "SUM".

- Click on "OK" and you have done.

or

Assume that, your Stock field range is G2:G100 cell without field name, then outside the pivot table, use this function:

=SUMPRODUCT(($G$2:$G$100)*($G$2:$G$100>=0))

This will sum all positive numbers.

Take the following formula.

Adjust the ranges as needed.

Enter it in a cell and confirm it with Ctrl+Shift+Enter to make it an array formula.

=IFERROR(INDEX(Sheet2!$D$2:$D$100, MATCH(1, (Sheet2!$A$2:$A$100='Drinks Data'!A2)*(Sheet2!$B$2:$B$100='Drinks Data'!B2)*(Sheet2!$C$2:$C$100='Drinks Data'!C2), 0)), 0)
