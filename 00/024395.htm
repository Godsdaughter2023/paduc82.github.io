bill manville say:

Sub DoBorders()
  Dim rFirst As Range
  Dim iRows As Integer
  With Range("A1").CurrentRegion
    Set rFirst = .Range("F2")
    Do Until IsEmpty(rFirst.Value)
      iRows = 1
      Do While rFirst.Offset(iRows) = rFirst
        iRows = iRows + 1
      Loop
      .Rows(rFirst.Row).Resize(iRows).BorderAround xlContinuous, xlMedium
      Set rFirst = rFirst.Offset(iRows)
    Loop
  End With
End Sub

An alternative would be to use conditional formatting to get Excel to do the work for you.
Select the entire table, with A1 active.
Home > Conditional Formatting > New Rule > Use a formula
Formula is:=$F2<>$F1 > Format > Borders > bottom border

(You could set the left and right borders manually)

Sheet or Sheet tab names - Microsoft Community

anthonydlh say:

Sheet or Sheet tab names

I cant find a "formula" that returns the worksheet name (or tab) in excell. ie, if I want to insert the worksheet name in a cell I woiuld assume something like "=sheetname(nn)" would do. Is there no function for this?
tushar mehta say:
I cant find a "formula" that returns the worksheet name (or tab) in excell. ie, if I want to insert the worksheet name in a cell I woiuld assume something like "=sheetname(nn)" would do. Is there no function for this?


Once you save a workbook, you can use the following formula (entered in F2):

=MID(CELL("filename",F2),FIND("]",CELL("filename",F2))+1,255)

If you enter it in some other cell adjust the two references to F2 to that cell.
