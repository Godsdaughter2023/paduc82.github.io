Lookup and return multiple rows to New Worksheet - Microsoft Community

collenaguttmann say:

Lookup and return multiple rows to New Worksheet

I have a worksheet that I would like to lookup a value in column A and if true return the entire row into a new sheet inside a workbook in consecutive rows.  For instance, if column A is "Names" and there are 10 "Lisa" in that column I want to return all rows corresponding to "Lisa" to a new worksheet labeled "Lisa" how do I do this?
gary's student say:

See:

 

http://office.microsoft.com/en-us/excel-help/how-to-look-up-a-value-in-a-list-and-return-multiple-corresponding-values-HA001226038.aspx

nbvc say: Using formulas, here is a method.

Assuming Lisa isn't the only sheet to extract rows for...

In the Database sheet, add a helper column with formula like:

=A2&"_"&COUNTIF(A$2:A2,A2)

copied down.  This concatenates the name with the consecutive count of that name as you copy down.

In the "Lisa" sheet, enter "Lisa" into say A1 and in adjacent cell, say B1, enter formula: =COUNTIF(Sheet1!A:A,A1) this counts total matches in the database, Sheet1.

Then in A2 or A3 begin extracting with formula:

=IF(ROWS($A$1:$A1)>$B$1,"",INDEX(Sheet1!A:A,MATCH($A$1&"_"&ROWS($A$1:$A1),Sheet1!$C:$C,0)))

copied down and across as far as necessary to get all data.

Note:  That formula assumes you put the helper column in Sheet1, column C.  Adjust as necessary.

have a look at the following codes;

 

Factory settings reset. Type *#7780# and the phone's lock code (Nokia default lock code is 12345). This reset restores the phone's original settings.

Soft reset(reformat). Type *#7370# and the phone's lock code.

Hard reset. Switch the phone off. Press and hold three keys together, the green key, the * key and the number '3'. While you are holding these keys, switch the phone on. Don't release these three keys before you see the NOKIA logo.
Warning! Hard reset (as soft reset) wipes the C (phone memory) drive.

Hard reset for N97; Press shift + space + delete and turn the phone on at the same time. Don't let the three keys before you see the NOKIA logo.

Hard reset for other S60v5 phones; press green + red + camera key and turn the phone on at the same time.
