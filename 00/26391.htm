cbarnhorst say:

If you upgraded to Windows 10 between July 29, 2015 and July 28, 2016 then your device was permanently registered as a Windows 10 device and all you have to do is re-install the same edition of Windows 10.  Earlier versions of Windows no longer matter and earlier version product keys have no further use.

Even a clean installation will be considered a "re-installation" and you do not need a Windows 10 product key to do it.  Windows 10 product keys do not do the same thing as the product keys for earlier versions of Windows did.  With Windows 10 the device (meaning the motherboard) is what matters, not the software.

Download from here, 

https://www.microsoft.com/en-us/software-download/windows10

create media, start the computer with the media inserted, and Windows 10 should install and automatically activate.

dirtymoose say: This didn't work. Well it did but it's telling me my Windows isn't genuine. Or it days or needs to be activated. But I know I had the technical preview but when I replaced my screen and had to reinstall could I have erased whatever would have allowed Windows 10 to activate properly?

cbarnhorst say:

There has been nothing on the hard drive to erase since the changeover to the current activation method in July, 2015.  In Windows 10 the software is now activated by comparing the unique hardware IDs (think motherboard serial number, etc.) to that device's hardware IDs recorded at Microsoft (digital entitlement).  If the device is so registered with MSFT as a Windows 10 Device, then any copy of that edition of Windows 10 will automatically activate.  All the product key is for now is to register the device.  Once registered the device is always registered for the life of the device (a motherboard retail upgrade) will introduce new hardware IDs and deactivate Windows).  Nothing on the hard drive is involved at all anymore.  This is an anti-piracy measure.  No files to hack, no more hacking.

If your original copy of Windows 10 was installed using the special preview product keys prior to July 2015 and you did not go through the required procedure to permanently register the device after that date then your original beta activation would not survive a clean reinstall.  The rule given us in July, 2015 was that if you came into Windows 10 beta by clean installing a beta preview using the special product keys at the time and did not later upgrade from Windows 7 or Windows 8.1 then you could continue in the Insider program as long as you kept up with upgrades and did not try to clean reinstall the software.  The free upgrade was allowed during the first year after public release of Windows 10 and expired in July, 2016.

If that is your scenario you need to now register the device by purchasing Windows and using the product key for the one-time registration.  This was previously performed by entering a Windows 7 or 8.1 product key but that procedure expired in July, 2016.

Can i set a rule that only one person can edit the document at one time?

thanks.

sheen valentine ali msft say:

HI Alison,

Yes, you can. Planner actually uses a SharePoint Document Library to house the documents. a Document Library has a feature called Check-out, that you can enforce so that only one person can edit it at a time.

Here's the link to learn more:

Check out, check in, or discard changes to files in a library

You can directly access the Document Library by clicking on an attachment, then on the top-left, click on the Shared Documents to access the Document Library.

Then, go to the Library settings via the Cog button on the top-right of the library > Versioning settings > and select Require documents to be checked out before they can be edited? to Yes.
