reinstall xp, lose data? - Microsoft Community

sierralightfoot say:

reinstall xp, lose data?

if i reinstall xp pro will i lose data file and outlook pst files. i have a hal.dll error i can't overcome
malkeleah say: Yes. However:

1. Since you didn't say why you can't overcome this error, you may not have tried the usual fix - which I will give you below.

2. If the usual hal.dll fix doesn't work, then perhaps a Repair Install will. A Repair Install will leave all your data and programs intact although you will need to reinstall all Windows Updates. It is worth a try anyway.

3. If #1 and #2 don't work (or you have already tried them), you can retrieve your data even if you can't boot into Windows. Boot the computer with a Linux Live CD such as Knoppix and copy the data files to an external hard drive.

Doing the work requires a certain level of computer skills. You know yourself best and know whether you should instead take the machine to a competent local computer tech. This will not be a BigComputerStore/GeekSquad type of place.

1. To fix the hal.dll error - Set the BIOS to boot CD before Hard Disk.  Boot the XP CD and, instead of Setup, take the immediate R for Repair.  Assume any password requested is blank, and TAB over. At the command line type:

Attrib -H -R -S C:\boot.ini [enter]
DEL C:\boot.ini [enter]
    [to delete the bad one]
BootCfg  /Rebuild [enter]
    [to search for Windows installations and make a new one]

Exit from the Recovery Console and try to boot into Windows.

2. http://www.michaelstevenstech.com/XPrepairinstall.htm - Repair Install How-To
http://michaelstevenstech.com/cleanxpinstall.html - Clean Install How-To
http://www.elephantboycomputers.com/page2.html#Reinstalling_Windows

3. Using Knoppix for data retrieval:

You will need a computer with two cd drives, one of which is a cd/dvd-rw OR a usb thumb drive with enough capacity to hold your data OR an external usb/firewire hard drive formatted FAT32 (not NTFS)*. To get Knoppix, you need a computer with a fast Internet connection and third-party burning software. Download the Knoppix .iso and create your bootable cd. You need to burn as an image, not as data, using third-party burning software. Then boot with the CD you made and Knoppix will be able to see the Windows files. If you are using the usb thumb drive or the external hard drive, right-click on its icon (on the Desktop) to get its properties and uncheck the box that says "Read Only". Then click on it to open it. Note that the default mouse action in the window manager used by Knoppix (KDE) is a single click to open instead of the traditional MS Windows' double-click. If you want to burn CD/DVDs, use the K3b program.

*My understanding is that you can now write to an NTFS partition from Linux. If you wish to do this, Google for instructions.
