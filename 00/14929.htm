rich007a say:

If the 3 columns you want to import are always in the same column, and you know how many columns there are in a file, you can make use of the FieldInfo parameter of the Workbooks.OpenText method.  Best thing to do is to open your file and run the Text Import Wizzard while recording a macro.  On the last page, you can click on each column and set whether or not it is imported.
 
As an example, if there are 6 columns of data, the following will import columns 2, 4 and 5:
 
    Workbooks.OpenText Filename:=.SelectedItems(i), _
        DataType:=xlDelimited, _
        TextQualifier:=xlDoubleQuote, _
        ConsecutiveDelimiter:=True, _
        Tab:=True, Semicolon:=False, Comma:=False, Space:=False, Other:=False, _
        FieldInfo:=Array(Array(1, 9), Array(2, 1), Array(3, 9), Array(4, 1), Array(5, 1), Array(6, 9))
 
You can see that FieldInfo is made up of an Array of 2-element arrays.  The first element is the column number (1-based), and the second element is one of the XlColumnDataType constants specifying how the column is parsed.

9 means the column will not be exported, 1 means the column will be imported with the General format.
 
See Workbooks.OpenText Method for more details.
 
To remove specific strings from your imported data, you can use the Range.Replace mehtod.
 
    Worksheets("Sheet1").Columns("B").Replace _
        What:="_SRK______P/F Metal Trim_", _
        Replacement:="", _
        LookAt:=xlPart, SearchOrder:=xlByColumns, MatchCase:=False
       
    Worksheets("Sheet1").Columns("B").Replace _
        What:="___1/32_Square_", _
        Replacement:="", _
        LookAt:=xlPart, SearchOrder:=xlByColumns, MatchCase:=False
       
    Worksheets("Sheet1").Columns("B").Replace _
        What:="_50_PC_CT_", _
        Replacement:="", _
        LookAt:=xlPart, SearchOrder:=xlByColumns, MatchCase:=False
       
    Worksheets("Sheet1").Columns("C").Replace _
        What:="SHEETROCK Brand Paper-Faced Metal Trim,", _
        Replacement:="", _
        LookAt:=xlPart, SearchOrder:=xlByColumns, MatchCase:=False 
 
etc. (check the replace text carefully in that last one).
 
Finally, you can do:
 
    Columns("A:C").EntireColumn.AutoFit
 
to adjust the columns automatically, or:
 
    Columns("A:C").ColumnWidth = 10
 
to set spcific widths of column(s).
 
Hope that helps.

Not sure what's to not understand here...

In VBA, you define a string by enclosing it in quotation marks " ".  If there is a quotation mark IN the string, you double it.  So if you want to replace


(2-1/16") Variable


you would use

    What:="(2-1/16"") Variable"

...unless you mean Variable is a variable, i.e. a string object containing some other text???  In which case you would use:

    Dim strMyString As String
    strMyString = " some text"
    ......
        What:="(2-1/16"")" & strMyString
    ......
 
notice how we use the & to concatenate the two strings.

 

I also had another question, It seems there is more text than I thought to get rid of. Is there some way I can use something like:

(Some text), and ,(Some Text), and,(Some text), and ,(Some Text) ect to delete in one string?

Best thing to do here is probably to make an array or variant-array and loop through each element:

    Dim vArray As Variant, v As Variant
   
     'First column:
    vArray = Array( _
        "text A", _
        "text B", _
        "text C")
       
    For Each v In vArray
        Worksheets("Sheet1").Columns("A").Replace _
            What:=v, _
            Replacement:="", _
            LookAt:=xlPart, SearchOrder:=xlByColumns, MatchCase:=False
    Next v
   
     'Next column:
    vArray = Array( _
        "text D", _
        "text E", _
        "text F")
       
    For Each v In vArray
        Worksheets("Sheet1").Columns("B").Replace _
            What:=v, _
            Replacement:="", _
            LookAt:=xlPart, SearchOrder:=xlByColumns, MatchCase:=False
    Next v

If you have a really long list of strings to delete, you could list them in a Worksheet and loop through each cell in a range:


    Dim c As Range, rng As Range
    Set rng = Worksheets("Sheet1").Range("A1:A100")
    For Each c In rng
        Worksheets("Sheet1").Columns("A").Replace _
            What:=c.Value, _
            Replacement:="", _
            LookAt:=xlPart, SearchOrder:=xlByColumns, MatchCase:=False
    Next c
 

Hope that helps.

ronrosenfeld say:
  • Open or import the file into Excel,  with data starting in A1
  • With the Text to Columns wizard, split the lines using the colon ":" as the delimiter
  • Select A1:A3
  • Copy
  • Select E1 (or some other cell)
  • Paste Special ► Transpose
  • E2 formula:

=INDEX($B:$B,MOD(COLUMNS($A:A)-1,4)+INT(ROWS($1:1)-1)*4+1)

  • Fill right to G2
  • Then select E2:G2 and fill down as far as needed.
  • When you have converted them all, you will start to see zero's
  • Then select the results
    • Copy ► Past Special ► Values  (to remove the formulas)
