1.) Create a simple Excel file without ANY DATA WHATSOEVER in it. 

2.) If it is Synchronized properly while your other files are not, then you are suffering due to the incompatibility between Excel for the PPC and Excel (for the PC).

jay freedman say:
There is a setting you can use to change what you see on your computer, but there is no way to control what others will see when they open the same document -- and that is as it should be.

To change what you see, click Office button > Word Options > Trust Center > Trust Center Settings > Privacy Options. On the right, uncheck "Make hidden markup visible when opening or saving". (The inclusion of this item under the heading "Document-specific settings" is a mistake; this is specific to your computer, but is applied to all documents and not just to one.) Again, this affects only what you see; if you send your document to another user, what they see depends on whether this option is checked or unchecked on their computer.

The reason this option is in the Trust Center is that if you send a document without realizing that it contains tracked changes (that is, both the before and after versions are present in the file), you can cause yourself a lot of trouble. Read http://www.shaunakelly.com/word/sharing/publicexamplesoftrackchanges.html for some examples, and http://www.shaunakelly.com/word/sharing/howtrackchangesworks.html for more advice.

If you need to send a document that shows only the final view, while you keep the tracked changes for your own use, then use Save As to make a copy of the document and accept all the changes in the copy. Or make the copy as a PDF file, which will make it less likely that the recipient will edit the document without your cooperation.
