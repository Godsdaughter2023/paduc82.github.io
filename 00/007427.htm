Using countif with 2 criteria using time range in excel - Microsoft Community

kianalee say:

Using countif with 2 criteria using time range in excel

I am trying to count occurrences between a time range...eg. 0:01:00 and 14:00...I know it is countif in excel but don't know how to count between the time range.
jeeped say:
I am trying to count occurrences between a time range...eg. 0:01:00 and 14:00...

With Excel 2007/2010 you can use COUNTIFS(). I prefer to define time constants with the TIME() functio like this,
=COUNTIFS(A:A,">="&TIME(1,0,0),A:A,"<="&TIME(14,0,0))
... to count times in column A between 01:00:00 and 14:00:00 (i.e. 1:00am and 2:00pm). I think that is what you meant by your example.

For backwards compatibility you can use SUMPRODUCT() the same way,
=SUMPRODUCT((A:A>=TIME(1,0,0))*(A:A<=TIME(14,0,0)))

Edit: if you meant from one minute past midnight till two in the afternoon, this might be better,
=COUNTIFS(A:A,">0",A:A,"<="&TIME(14,0,0))
... or,
=SUMPRODUCT((A:A>0)*(HOUR(A:A)<=14))

- changed the Language via the Review -> Language -> Set Proofing Language to "English (United States)"

shovon11 say:

Hi James,

Thanks for your comment.

The reply from from Ganesh Kumar N was a solution for my problem.
Which is:

Try the following steps:-

Step 1:- By using Macro we can change the language in powerpoint for all slides.

Create a new macro:
1. Go to Tools, Macro, and Visual Basic Editor.
2. Insert a new empty module by selecting Insert, Module.
3. Paste this code on the right panel and save the macro:
Option Explicit  
Public Sub ChangeSpellCheckingLanguage()  
    Dim j As Integer, k As Integer, scount As Integer, fcount As Integer
    scount = ActivePresentation.Slides.Count
    For j = 1 To scount
        fcount = ActivePresentation.Slides(j).Shapes.Count
        For k = 1 To fcount
            If ActivePresentation.Slides(j).Shapes(k).HasTextFrame Then
                ActivePresentation.Slides(j).Shapes(k) _
                .TextFrame.TextRange.LanguageID = msoLanguageIDEnglishAUS
            End If
        Next k
    Next j
End Sub

Step 2:- This "msoLanguageIDEnglishAUS" which is used in the above macro can be replaced by any desired language. The full list of languages can be found in this article http://msdn.microsoft.com/en-us/library/aa432635.aspx

Step 3:- Execute the macro (by pressing F5 within the editor, or by selecting Tools, Macro, Macros, ChangeSpellCheckingLanguage, and clicking Run).
After that all text elements within the presentation will have the new spelling language.

Sorry for the delay... I've been doing some tests, but I cannot hide/disable Outlook...
However, I been testing somethin more related to your first request. I found that is possible to install customized Office. A little bit technical but I'm able to help out. This procedure is designed to massive deployments, but is valid for one machine as well...

Deploy Office 365 ProPlus from a local source:
https://docs.microsoft.com/en-us/deployoffice/d...

First, download the ODT tool from the page:
https://www.microsoft.com/en-us/download/confir...
execute the downloaded file. This generated two files called "Setup" and "Configuration.xml"
Both files must be placed in a new folder that we named "ODT" on disk "C"
Then modifiy the file "Configuration.xml" with the editing tool http://officedev.github.io/Office-IT-Pro-Deploy....
The desired version is Office proplus 32 bits, in English just Word and Outlook. The file is as follows (this is just an example, you must generate your own file )

<Configuration>
<Add OfficeClientEdition="32" Channel="Broad">
<Product ID="O365ProPlusRetail">
<Language ID="en-us" />
<ExcludeApp ID="PowerPoint" />
<ExcludeApp ID="OneDrive" />
<ExcludeApp ID="OneNote" />
<ExcludeApp ID="Lync" />
<ExcludeApp ID="Groove" />
<ExcludeApp ID="Excel" />
<ExcludeApp ID="Access" />
<ExcludeApp ID="Publisher" />
</Product>
</Add>
<Updates Channel="Broad" Enabled="TRUE" />
<Display Level="Full" AcceptEULA="TRUE" />
</Configuration>

We already have the instructions of the version of Office that we want. Now we download Office with this command from the Command Promt as administrator:
c: \ odt> setup.exe / download c: \ odt \ configuration.xml

Finally, we install Office with this command:


c: \ odt> Setup.exe / configure c: \ odt \ configuration.xml




Finally, this is the tool to uninstall Office in case you had a previous installation on a computer:

https://support.office.com/es-es/article/Desins...

Hope it hepls !!
