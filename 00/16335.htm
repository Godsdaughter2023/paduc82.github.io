Goal Seek - VBA. Average 1 cell by changing 10000+ cells using - Microsoft Community

jme_200 say:

Goal Seek - VBA. Average 1 cell by changing 10000+ cells using positive numbers & integers only

I am trying to figure out a Goal Seek VBA code for Microsoft Excel 2013.

By changing data in cells B4:B10582, I would like to receive an average in cell B1 of approximately 1.25 (does not have to be exact). The cells in range B4:B10582 have to remain integers, cannot be negative or zero, and have to range from values of 1-5. How would I code this in VBA? Thanks!

mike h.. say:

Hi,

This is intriguing. We have a range of 10579 cells and need an average of ~1.25. you haven't provided a tolerance.

Assuming these are randomly distribute we can expect; but not assume, the average will be greater than you require as the number 1 is the only number that can drag the average down. So the answer to this isn't goal seek but IMO brute force like this

1. Average the range.

2. If it is > 1.25 then randomly replace a number >1 with a 1

3. If it < 1.25 then randomly replace a number <1 with a random 2 thru 5

3. Average again and repeat steps 2 and 3 until we hit 1.25 average

This code does that. set your own tolerance wher indicated.

Sub somesub11()
Application.ScreenUpdating = False
Application.Calculation = xlCalculationManual

Dim Tolerance As Double
Tolerance = 0.01
Dim Myrange As Range, Ave As Double
Set Myrange = Range("B4:B10582")
Do
Ave = WorksheetFunction.Average(Myrange)
If Abs(Ave - 1.25) <= Tolerance Then
    Range("C1") = WorksheetFunction.Average(Myrange)
    Exit Do
ElseIf Ave - 1.25 > 0 Then
    Cells(WorksheetFunction.RandBetween(4, 10582), "B") = 1
ElseIf Ave - 1.25 < 0 Then
    Cells(WorksheetFunction.RandBetween(4, 10582), "B") = WorksheetFunction.RandBetween(2, 5)
End If
Loop
Application.ScreenUpdating = True
Application.Calculation = xlCalculationAutomatic
End Sub

Sub PTWSlicerResetFUNDSEGMENT()

     'resets slicers named below - I have 3 slicers with FUND SEGMENT in the name.

    Dim slcrC As SlicerCache

    Dim slcr As Slicer

    Application.ScreenUpdating = False

    For Each slcrC In ActiveWorkbook.SlicerCaches

        For Each slcr In slcrC.Slicers

            If slcr.Shape.Parent Is ActiveSheet Then

                 If InStr(slcr.Name, "FUND SEGMENT") Then

                                    If slcrC.FilterCleared = False Then slcrC.ClearManualFilter                      

                    End If

                End If            

        Next slcr

    Next slcrC

    Application.ScreenUpdating = True    

End Sub

And this one lists all filtered slicers in a message box (quick and dirty, but does the job. I use it for myself, not the user, but it could be if your dashboard is not too complex)

Sub List_FILTERED_SLICERS()

     'lists all the slicers that are filtered on activesheet

    Dim slcrC As SlicerCache

    Dim slcr As Slicer

    Dim SLICERFILTERED As String

    Application.ScreenUpdating = False

    SLICERFILTERED = ""

    For Each slcrC In ActiveWorkbook.SlicerCaches

        For Each slcr In slcrC.Slicers

            If slcr.Shape.Parent Is ActiveSheet Then

 

                If slcrC.FilterCleared = False Then SLICERFILTERED = SLICERFILTERED & slcr.Name & vbCrLf

                       

            End If

        Next slcr

    Next slcrC

    

    Application.ScreenUpdating = True

     MsgBox "These slicers are filtered:" & vbCrLf & SLICERFILTERED

End Sub

suzanne s. barnhill say:

The alternative approach is to open Normal.dotm for editing and create the AutoText entries directly. The easiest way to do this is as follows:

In Word, press Alt+F11; the Visual Basic Editor opens. Press Ctrl+G to activate the Immediate window. In that window, type

     NormalTemplate.OpenAsDocument

and press Enter. Normal opens, but note that it may not be on top of all windows. You can close the Visual Basic Editor. Make the desired changes to the Normal file, and then save and close it. If the changes you have made have not been recognized as changes (the Undo button is not activated), press Spacebar, then Backspace to "dirty" the file, thus ensuring that the template will actually be saved.
