Windows home or professional?

Stopping updates will also stop windows defender definitions updates and will potentially leave you at risk of nasty's.

if you are going to force them to stop, allow it to run once a day.

Assuming you are running windows home.

click Start and type services, right click on Services and select Run as Administrator

Scroll down the list of services to Windows Update

Right click on the Windows Update service and select Properties.

in Start up Type choose Manual. Click Ok and close services.

use task scheduler to turn the service on and off.

Open Task Scheduler and from the right hand pane, select Create Task.

Give it a name (that means something) and description, Run with Highest privileges and configure for Windows 10.

Click on the Triggers Tab and select New

this is the Start the Service schedule so make sure to select a time that you want it to get Updates.

Select Daily (or weekly if you only want to do it once a week), select a time that you are happy to start the Windows Update service and click Ok

Click on the Actions tab and click New.

type Net into the program/script box and Start wuauserv into the Add arguments box.

Click Ok

I will let you decide on Conditions and Settings - they will only affect this task if you change anything.

Click OK

it will ask for User Password - give it the details of an administrator account.

Create another task in the same way for the Stopping of the service (replacing Start with Stop)

Add two triggers for the stopping of the service 1. at the time you want to turn it off) and 2. when the computer starts - just to make sure that if it were to restart, that the service will not continue to run.

a. user say:

Step 1 is to tell us which email program (e.g., Outlook, Thunderbird, etc.) or email service (e.g., Outlook.com, Google, Yahoo, etc.) you use, because each one works differently.

In general: Email programs and services are designed to store email inside the program's or service's own folders. If you want to store your email in document folders you're going to have to manually copy each email individually. While that may be okay for a few individual emails, it will get real tedious real quick if your plan is to do that for every email.

Plus: Storing your email outside the program or service means you can't easily use the program's or service's tools to work with your email.

You can do that if you want, but it's like carrying around your family's and friends' contact information in a little black book. Yes, that works, but why make it hard on yourself? (For the record: I got rid of my own little black book after I got married, but that's different.)

To help troubleshoot error messages and other issues, you can start Windows by using a minimal set of drivers and startup programs. This kind of startup is known as a "clean boot." A clean boot helps eliminate software conflicts.
