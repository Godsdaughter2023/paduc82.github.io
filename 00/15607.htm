After sorting you can put this formula in C2:

 

=IF(COUNTIF(A$2:A2,A2)=COUNTIF(A:A,A2),A2,"")

 

and copy it down. It will give you the item number only on the last row for each item, like this:

 

Item
 Date Inspected check

You can then apply an autofilter on column C with the condition Non-blanks to display only the latest date for each item.

Sub abc()
Dim ws As Worksheet
Dim r As Long
Set ws = Sheets("Sheet1")
Application.ScreenUpdating = False
r = ws.Cells(Rows.Count, "A").End(xlUp).Row
ws.Range("$A$1:$B$" & r).Select

ws.Range("$A$1:$B$" & r).Select
Selection.Sort Key1:=ws.Range("$A$2"), Header:=xlYes
For i = r To 3 Step -1
If ws.Cells(i, 1) = ws.Cells(i - 1, 1) Then ws.Rows(i - 1).Hidden = True
Next i
Application.ScreenUpdating = True
End Sub
