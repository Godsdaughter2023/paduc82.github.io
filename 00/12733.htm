¡firedog say:

If you have a document on OneDrive (the new name for SkyDrive), then you don't need to download it or attach it to an email to deliver it to the addressee.

Select the file in OneDrive and click Share on the command bar. In the panel that opens, click Invite people, then type the addressee's email address in the To box. If the address is in your contacts list, it will be completed for you. Add a message if you want, then click Recipients can edit. To maintain the document intact, select Recipients can only view and it's most likely to work properly if you select Recipients don't need a Microsoft account.

If this is an Office document, it will open in the online application when the recipient clicks on the link in the email he receives. If he needs to have a copy of it, he clicks File > Save as to download a copy.

This method saves everybody's bandwidth and doesn't clog up the email system with unnecessary file attachments.

If the file you want to send is a picture, you can certainly copy from OneDrive and paste into an email. Right-click on the image file in OneDrive and select View original. This will open the image file in its own page; you can then right-click on the picture and select Copy (or Select all and then Copy in some browsers), then paste into your mail message.

Brock,

Interesting that your boss wants it to start on 1/2/15 (Friday) instead of 1/5/15 (first Monday of the new year). Since New Year's Day is on a Thursday, normally that Friday is a "dead day".

The duration in working time, (standard calendar with no holidays, only weekends as non-working), over that period is indeed 65 days. And the duration in calendar days, which includes weekends, is 90 days - not sure why you believe it is 91 days.

Text1=DateDiff("d",[Start],[Finish]) & " Edays"

Don't forget to check the option to use the formula for calculation of group and summary rows.
