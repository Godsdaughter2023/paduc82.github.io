how to find the normal.dot (x) in Office 2010/Windows XP - Microsoft Community

k_nelson say:

how to find the normal.dot (x) in Office 2010/Windows XP

Just need to find the file so I can modify it.   thanks

jay freedman say:

The quickest way -- without needing to know what folder the file is in:

  • Press Alt+F11 to open the macro editor.
  • Press Ctrl+G to open the Immediate window at the bottom right.
  • Copy/paste this into the Immediate window and press Enter:

NormalTemplate.OpenAsDocument

  • Close the macro editor.

The template (actual name is Normal.dotm, indicating that it's a macro-enabled template) will be open in the main Word window, where you can make your changes and save it.

If your installation of Office wasn't customized, the Normal.dotm file will be in the Templates folder. That folder is a hidden system folder, not usually visible in Windows Explorer or Word's File > Open dialog. The easiest way to get there is to paste this into the address box and press Enter: %appdata%\Microsoft\Templates

The %appdata% is a Windows environment variable that expands to C:\Users\<your login name>\AppData\Roaming, although you don't need to know that to make it useful.
