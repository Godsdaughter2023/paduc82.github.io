GETPIVOTDATA is an excel function inserted by Excel when we try to select any range from a Pivot Table by using the computer mouse. We, as a best practice, try to avoid using it - instead we try to directly refer the pivot table range using the column-row naming convention as we use regularly.

We will have to make use of INDIRECT function, we may have to make use of dynamic named ranges, etc.

Pardon me for bumping in.  If you want to display writing report as a single bar - create a summary task and create the parts of writing the report as subtasks.  You can then have Task 1 as the predecessor to report 1 and task 2 as the predecessor to task two.  Collapse the subtasks and there is your single bar.
