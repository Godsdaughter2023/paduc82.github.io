vincentwu_msft say:

Hi Jim,

When read-only permission are granted on the subfolder, limited access permission will be automatically assigned to the parents(folder, DOCS library and subsite) as accessing the subfolder requires permission(limited access) to cross the subsite > DOCS library > folder.

It is expected that,
a: the subfolder can be seen in the folder whereas
b: the folder can't be seen in DOCS library and
c: access denied is got when accessing the subsite.

The reasons are:
1. Read-only permission level include the permission for viewing the subfolder.
2. Limited access can only ensure crossing the folder but it is not enough for viewing the folder.
3. When accessing the subsite, it will default to the home page, which the group does not have permission for viewing because of default permission inheritance(subsite > site pages library > home page).

In order to access DOCS and the susbite, you can grant the group view only permission level for the DOCS library and the same to the subite.

If you have any questions, or anything is misunderstood, please feel free to let us know.

Is "blue" your highlight color? Then it's likely that the documents have been protected. If you try to select a word in a protected document, it hightlights momentarily, then deselects.
