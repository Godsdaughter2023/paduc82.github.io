Sounds like a candidate for some simple code.

Example to resize on pic to 3 x 2 inch and position 2 inch from top and 1.5 inch from left

Sub fixPic()
Dim opic As Shape
Const inch2point As Long = 72
On Error GoTo err
Set opic = ActiveWindow.Selection.ShapeRange(1)
With opic
.LockAspectRatio = False
.Width = 3 * inch2point
.Height = 2 * inch2point
.Top = 2 * inch2point
.Left = 1.5 * inch2point
End With
Exit Sub
err:
MsgBox "Is a shape selected?!"
End Sub

How to use code

You could also automate the whole process with code or use Photo Album and then code to resize all pics at once.

tom ogilvy say:

One way it to not hide the sheet and put a filter on the the FULL LIST, but you apparently do not want to expose the Full List.  Another approach would be to create a pivot table from the FULL LIST on the visible sheet and use the Client as a page  field in that pivot table.

Pulling multiple rows with formulas generally require array formulas which can be calculation intensive.   The formulas check all rows and determine which rows match the requested client.  Then the small worksheet function can be used against this list of rows and sequentially extract the nth row to use.  This is an interior part of the formula.  The index function is used with this row to retrieve the data.

A less calculation intensive approach would use simple formulas next to the full list to establish which rows to put back be produce numbers from 1 to N. 

then you could use Index and match to pull the data back.
