After re-creating your Office global admin account, you can’t access some specific SharePoint collections.

How to remove users from SharePoint Online

john mcghie say:
Mmmmm...  :-)

1)  Pictures

2)  Fonts

3)  Tracked changes  :-)

4)  Compatibility Mode

If you paste a high-res picture into Word, then crop it to size (or drag-re-size it), the unneeded pixels are not removed unless you use File>Reduce File Size.

Word has (or had...) a bug where if you pasted text in a font that was not installed, Word would save a complete copy of the font in the document for each paste, or some equally strange problem.  Make sure you have applied the 14.1.2 update to cure that.

If you have inadvertently left Tracked Changes ON, but hidden them, every change you have made stores a new copy of the text.  The old copy remains in the document.  Very embarrassing if this happens: your recipient will be able to read all the "deleted" material.  Turn Tracked Changes off, then use "Accept all changes in document" to remove it.

Compatibility mode (the obsolete binary format) is a bear for doing this.  When you force a document to be stored in the obsolete format, it stores two  copies of everything, one in the new format, one in the old.  For pictures, it stores a native format for screen display and the original format for printing: in compatibility format, it stores two more, in the old format :-)

Try saving the file as .docx, close it, then re-open it.  That should clean it out.  .DOCX is about a quarter the size of the old format.  If that DOESN'T clean it out (and, if the document gets damaged, sometimes it won't) then you need to Maggie it:

The Maggie:


1. Create a new blank document in .docx format

2. Carefully select all of the text in the bad document EXCEPT the last

paragraph mark

3. Copy it.

4. Paste in the new document.

5. Save under a new file name and close all, then re-open.


This technique for de-corrupting is known as "Doing a 'Maggie'", after

Margaret Secara from the TECHWR-L mailing list, who first publicised the

technique.


Anyone working with Indesign is probably a professional, so I have been fairly cryptic.  Come back if you need a fuller explanation of any part.
