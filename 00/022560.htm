If you combine the accounts, the only way to be able to select which address to use for sending is to access the @outlook.com account using a browser.

To add another Microsoft account, you must forward the emails to the @outlook.com account and add the Microsoft account as a Send only account.

To reply to a forwarded email with the account that originally received the email, you would have to login using a browser and you would have to select the To address.  Clicking Reply would just send the email back to the account that first received the email.

Other disadvantages are only one display name and one signature will be used.

In my opinion, it is much simpler to setup the different accounts in an app on each device.

melerix say:

if the Skype Metro/Modern version, try this...

open a PowerShell Window with admin rights and write the following command:

Get-AppxPackage

it will display a list of all the Metro/Modern apps that are installed in your PC.

now check if in the list exist Skype, and check what is the PackageFullName of it.

then write the following command:

Remove-AppxPackage -package PackageFullName

that will uninstall Skype completely.

OneNote for Mac FAQ

Where can I get OneNote for Mac?

OneNote for Mac is available in the Mac App Store. You can also get all the OneNote apps at OneNote.com.

 

What countries/regions are OneNote for Mac available in, and in what languages?

OneNote for Mac will be available in all markets where the Mac store is available. OneNote for Mac supports 15 languages including:

  • Chinese (Simplified) 
  • Chinese (Traditional) 
  • Danish 
  • Dutch 
  • English 
  • Finnish 
  • French 
  • German 
  • Italian 
  • Japanese 
  • Norwegian (Bokmål) 
  • Polish 
  • Russian 
  • Spanish 
  • Swedish

 

What are the system requirements for OneNote for Mac?

OS X 10.9

 

How does OneNote for Mac sync with OneDrive?

OneNote for Mac users simply sign in using an existing or new Microsoft Account.

 

Does OneNote for Mac work with Office 365?

OneNote for Mac is available for free as an individual download from the Mac App Store and doesn’t come as part of an Office 365 subscription/Office for Mac 2011.
