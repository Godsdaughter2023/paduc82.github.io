steve winograd say: To prevent certain types of malware infections, Windows 7 doesn't allow users to store files in the Program Files and Program Files (x86) folders.  For that reason, taking ownership of those folders is not recommended.

Applications that are compatible with Windows 7 should store their data in the user's data folder, not in the program folders.  Some programs let you specify that choice during installation.

To edit a file in the program folder, right-click Notepad and run it as an administrator.

How to Change Wireless Mode - Microsoft Community

xb1r hades say:

How to Change Wireless Mode

Login Information

  • Default Gateway - 192.168.0.1
  • Username - user
  • Password - user

Older Firmware

How to Change Wireless Mode

  1. Open a web browser.
  2. Enter http://192.168.0.1 in the address bar.
  3. Select login from the side menu.
  4. Enter the username and password. (user/user)
  5. Select Wireless from the menu at the top.
  6. Select Radio from the menu on the left.
  7. Select G-Only mode from 802.11 n-Mode drop down menu.
  8. Select Apply.

 

Newer Firmware

 How to Change Wireless Mode

  1. Open a web browser.
  2. Enter http://192.168.100.1 in the address bar.
  3. When prompted enter user for the username and user for the password.
  4. Select Wireless from the menu at the top.
  5. Select Radio from the menu on the left.
  6. Select G-Only mode from 802.11 n-Mode drop down menu.
  7. Select Apply.

¡firedog say:
When I try to delete them, they come right back.
...
I cannot open them and I cannot move them to storage unless I open them.


This indicates that you haven't given the program time to process the changes after manipulating a large number of messages.

When you want to manipulate a lot of messages at once, it's always best to click Work offline before doing so. This eliminates interference from scheduled send/receive operations while you're clearing up.

Once you've selected messages for deletion and then used Delete, give the program time to settle down before doing anything else. It will take several minutes for the program to record the deletion of so many files; if you interrupt the process by continuing to use the program, you will end up confusing the message database.

Once the program has settled down, close down WLMail and wait ten minutes before re-starting it and using it. You can see how busy the program is by pressing Ctrl-Shift-Escape to start Task Manager. Watch the activity of wlmail.exe on the Processes tab.
