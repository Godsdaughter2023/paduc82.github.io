The file will be in .zip format and is probably located in My Computer/Downloads unless you switch your browser default download directory to one of your preference. NB. The file takes quite some time to download, you can't open it unless it has been fully downloaded

6. When finished downloading, there'll be a Documents.zip file in your Downloads folder. Extract it. It'll show your whole list of notebooks and notes. The notes will be in .one format which can be opened with Desktop OneNote 2013 (I'm using 2010).

graham mayor say:

You can do it with some macro code - http://www.outlookcode.com/article.aspx?id=72

If you want to narrow it down to your work domain then modify as follows.

Option Explicit

Private Sub Application_ItemSend(ByVal Item As Object, _
                                 Cancel As Boolean)
Dim objRecip As Recipient
Dim strMsg As String
Dim res As Integer
Dim strBcc As String
Dim strDomain As String
Dim bAddBCC As Boolean

    On Error Resume Next

    ' #### USER OPTIONS ####
    ' address for Bcc -- must be SMTP address or resolvable
    ' to a name in the address book
    strDomain = "myCompanyDomain.com" 'The company domain
    strBcc = "*** Email address is removed for privacy ***" 'Your address
    For Each objRecip In Item.Recipients
        If InStr(1, objRecip.Address, strDomain) > 0 Then bAddBCC = True
        Exit For
    Next objRecip
    If bAddBCC Then
        Set objRecip = Item.Recipients.Add(strBcc)
        objRecip.Type = olBCC
        If Not objRecip.Resolve Then
            strMsg = "Could not resolve the Bcc recipient. " & _
                     "Do you want still to send the message?"
            res = MsgBox(strMsg, vbYesNo + vbDefaultButton1, _
                         "Could Not Resolve Bcc Recipient")
            If res = vbNo Then
                Cancel = True
            End If
        End If
    End If
lbl_Exit:
    Set objRecip = Nothing
    Exit Sub
End Sub

Where you want the random numbers to appear in your document, insert the following field

{ DOCVARIABLE varRandomNumber }

using Ctrl+F9 to insert the field delimiters and Alt+F9 to toggle off their display.

Then in the VBA Editor, paste the following into a module

Sub PrintRandomNumbers()
Dim i As Long
For i = 1 To 200
    With ActiveDocument
        .Variables("varRandomNumber") = RandomNumbers(100000, 999999)
        .Range.Fields.Update
        .PrintOut
    End With
Next i
End Sub
Public Function RandomNumbers(Lowest As Long, Highest As Long, _
Optional Decimals As Integer)
   If IsMissing(Decimals) Or Decimals = 0 Then
      Randomize
      RandomNumbers = Int((Highest + 1 - Lowest) * Rnd + Lowest)
   Else
      Randomize
      RandomNumbers = Round((Highest - Lowest) * Rnd + Lowest, Decimals)
   End If
End Function

Then when you run the macro PrintRandomNumbers, it will cause 200 copies of your document to be printer each with a random number between 100000 and 999999

If you don't want to print all of the copies at once, you could delete the For i = 1 to 200 and the Next i lines of code and then run the macro to print the document.  Each time you do that, a new random number will appear in the document that is printed.

You might however be better off to just use sequential numbers for each client.  For that, see the following website:

http://word.mvps.org/faqs/MacrosVBA/NumberDocs.htm

I assume that "2001" above should actually be "2010".

It's not the repair of the Internet connection that the error message is recommending, it is the repair of Office.  Open Control Panel, choose Add or Remove Programs, Select Microsoft Office 2010, and click Change.  Select the "Repair" radio button and click Continue.  Answer any additional prompts.
