You can try these methods to troubleshoot your Windows 10 update issue:  

Method 1: Run SFC Scan.  

To run an SFC scan, follow these steps:  

  1. Run Command Prompt as Administrator.  

  2. Type sfc /scannow and hit Enter.  

This will check and run an auto-repair on the issue.  

Method 2: Run DISM Tool.  

After running the SFC scan and the issue still persist, run the DISM to check the system health and to try restoring the files. To run DISM tool, follow these steps:  

  1. Run Command Prompt as Administrator.  

  2. Type the following commands:  

    1. DISM.exe /Online /Cleanup-image /scanhealth  

    2. DISM.exe /Online /Cleanup-image /Restorehealth  

NOTE: This might a while to finish scanning. Do not cancel.  

Method 3:  

After doing Method 1 and Method 2 but the issue still persist, you can manually reset the Windows Update Components and Run Windows Update process again.  

Resetting Windows Update Components will fix those corrupted Components and will help the Update installation quickly. To reset Windows Update Components, follow these steps:  

  1. Run Command Prompt as Administrator  

  2. Stop BITS, Cryptographic, MSI Installer and Windows Update Services. To do this, type the following commands:  

    1. Net stop wuauserv  

    2. Net stop cryptSvc  

    3. Net stop bits  

    4. Net stop msiserver  

  3. Rename SoftwareDistribution and Catroot2 folder. To do this, type the following commands:  

    1. Ren C:\Windows\SoftwareDistribution SoftwareDistribution.old  

    2. Ren C:\Windows\System32\catroot2 Catroot2.old  

  4. Restart BITS, Cryptographic, MSI Installer and Windows Update Services. To do this, type the following commands:  

    1. Net start wuauserv  

    2. Net start cryptSvc  

    3. Net start bits  

    4. Net start msiserver  

    5. Restart your computer and try to install Windows Update.

Enter the following User Defined Function:

 

Public Function FindValues(r As Range, M1 As Long, M2 As Long) As Variant
Dim rr As Range, v As Variant
FindValues = 0
For Each rr In r
    v = rr.Value
    If v >= M1 And v <= M2 Then
        FindValues = v
        Exit Function
    End If
Next
End Function

So in A1, enter:

 

=findvalues(D16:K21,409,610)

Resizing Photos - Microsoft Community

debra1007 say:

Resizing Photos

Is there a way in Sway to resize a photo?   When I add a photo as an image card, Sway automatically makes it a certain size.   I know that on the image card I can select the emphasis to be subtle, moderate or intense.  But even subtle is just way too big.  Any help is greatly appreciated!

jason (office engineer) say:

Hi Debra,

I would suggest shrinking the image before you insert it into Sway.  To do so, just open it in Paint and click the Resize button; then choose a percentage for the height and width that is the same to keep the dimensions the same.  Try uploading that image into Sway and keeping it subtle.

Another thing you may want to try out is the Focus Points feature.  It allows you to specify the important regions of your image to focus in on.  This may end up having the effect of making the image bigger though unless you choose the entire image is important option.

Hope this helps!

Jason
