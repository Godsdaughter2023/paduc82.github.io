Hi tondsmith,

Was Family Tree Maker compatible with Windows XP? (You don't actually say whether it worked on Vista but no longer works since installing SP1). If it worked on XP then try installing in compatibility mode.

1/ Locate the 'setup' file for Family Tree Maker and right click on it.

2/  From the drop down menu, click Properties.

3/  In the properties window, click the Compatibility tab.

4/  In the compatibility window you will see a section (at the top) marked 'compatibility mode'

5/  In this section you will see a box with the words 'run this program in compatibility mode for' to the right of it.

6/ Click in the box so that a 'tick' appears.

7/ In the dialogue box below you should see 'Windows XP (Service Pack 3).'

8/ If you don't see this, click the arrow to the right of the box and select Windows XP (Service Pack 3) from the drop down list.

9/ Finally, click OK.

10/ Your Family Tree Maker software should now start to install in compatibility mode for Windows XP.

If that doesn't work (it usually does work) then you may have to upgrade to a version of Family Tree maker that does support Windows Vista.

Clear ARP table

Use the NetShell (netsh) command to clear, delete or refresh the Address Resolution Protocol (ARP) cache by following the following steps.
