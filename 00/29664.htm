    tryx3 say:

    I take it that you are referring to passwords for admin accounts that you have on each computer to do your work. 

    You can just use a single external drive such as a USB stick for the whole collection. 

    1  Create the first "password reset disk" then create a folder with a name that identifies the user account, PC & OS [and possibly customer] then move the file userkey.psw from the root of the drive into that folder.  You can then repeat the process on the same external drive for each user account, PC & OS that you need.

    2  If you ever need to reset your admin password on a PC, just move the relevant userkey.psw file back into the root of the external drive.  The PC will then detect that one and that one only.
