dan v [msft] say:

Problems opening password protected Excel files in Office Mobile apps

Hi all,

Some of you may have problems opening a password protected Excel file in Excel for Android or Excel Mobile for Windows.

This can happen if the file was created and saved as an .XLS file in Excel 2010, 2007 or 2003 or if the file was saved in a format currently not supported on Android or Windows Mobile, such as .XLT or .XLTX.

To avoid this problem, open the password protected file in Excel on a Windows desktop PC or on a Mac, and then save it as a different Excel file type that works on Android or Windows Mobile, such as .XLSX. Once you've saved the file in the new file format you should be able to open it without a problem in Excel for Android or Excel Mobile for Windows.

For more information, go to Error when opening password-protected files on Android or Windows Mobile.

john korchok say:

There are 3 ways to get audio into a presentation.

  1. Record audio separately i.e. in Sound Recorder or some other audio program, then insert those files into the presentation using the Insert>Audio>Audio on my PC command. You'll see an icon of a speaker viewed from the side with sound rays. With this process, you would have the original recordings on your computer somewhere.
  2. Record audio in PowerPoint using Insert>Audio>Record Audio command. This also results in the grey speaker icon on your slide, but you don't have separate original files, they are embedded in the deck.
  3. Record a narration while the show runs. This makes one audio file per slide. They are also embedded in the presentation, but the icons look like a 3-D picture of a speaker with a blue area inside.

Before starting, check whether any of your sound files have been trimmed in PowerPoint. Select each sound file, then click on Audio Tools>Playback>Trim Audio. If the green marker is at the far left and the red marker is on the far right, the audio has not been trimmed in PowerPoint. If you don't have any trimmed audio files, quit now, your problem will not be solved by the following steps. Instead, please start a new Question describing your issue.

If you do have trimmed audio, proceed. With #2 and 3 above, the audio files are embedded in your presentation. Here are the steps:

  1. Make a copy of your presentation to a separate folder.
  2. In this copy, check every sound file and remove the PowerPoint trimming.
  3. Edit the file ending to .zip and click on OK when Windows asks if you want to do that.
  4. Expand the zip file
  5. Open the ppt folder, then inside that open the media folder. Your audio files are in there named media1.m4a, etc. in the order you created them. You'll probably have to play them to figure out which is which, but you.
  6. Download and install Audacity (free download from here).
  7. Download and install the FFmpeg converter installer (free download from here) needed to edit m4a files in Audacity.
  8. In Audacity, open Edit>Preferences>Libraries and click on the Locate... button beside the FFmeg library heading and find the FFmpeg library to connect it to Audacity.
  9. Open one of the audio files. Click and drag to select a portion. Adjust the end points to match the trim of the same file in the original presentation (the time units are identical in both programs).
  10. Save the file.
  11. When you're done, re-zip the set of folders plus the [Content_Types].xml file and change the file ending back to .pptx.
  12. Open the presentation and test, then try a video export.

If you have any difficulties, please post back here.

STEP 1: Perform a clean boot

1.01  Log on to the computer by using an account that has administrator rights.
1.02  Click the "Start" menu.
1.03  In the "Start Search" box, type "msconfig.exe" without the quotation marks.
1.04  On the list of results under Programs, right-click msconfig.exe and click "Run as administrator". This will start the System Configuration Utility. If you are prompted for an administrator password or for confirmation, type your password or click "Continue".
1.05  Go to "General" tab.
1.06  Highlight the radio button beside "Selective Startup".
1.07  Remove the checkmark beside "Load startup items".
1.08  Go to "Services" tab.
1.09  Check the box beside "Hide all Microsoft services".
1.10  Click "Disable all".

Note: Following this step lets Microsoft services continue to run. These services include Networking, Plug and Play, Event Logging, Error Reporting, and other services. If you disable these services, you may permanently delete all restore points. Do not do this if you want to use the System Restore utility together with existing restore points.

1.11  Click "Apply", and then click "OK".
1.12  When you are prompted, click "Restart". This will put Windows into a clean boot state. If you receive a dialog box upon startup, select "Don't show this message". Click "OK".
When the computer has restarted, please follow the troubleshooting steps below.

STEP 2: Uninstall Office 2010

2.01  Log on to the computer to an account with an administrator rights.
2.01  Go to the URL provided below.

http://go.microsoft.com/?linkid=9737366

2.03  When the File Download dialog box appears, click "Run".
2.04  Follow the instructions shown on the setup wizard.
2.05  Wait for the process to complete, and then restart the computer.

STEP 3: Enable hidden files and folders

3.01  Log on to the computer to an account with an administrator rights.
3.02  Click the "Start" menu.
3.03  On the right side of the pane, click "Computer" to open the Windows Explorer browser.
3.04  Click the dropdown arrow beside "Organize".
3.05  Select "Folder and Search Options".
3.06  Go to "View" tab.
