Good find TasosK.

And if you want to find the installed path for AcroRd32.exe, try the following:

 

1) From http://www.cpearson.com/excel/registry.htm, download: http://www.cpearson.com/Zips/modRegistry.zip  and  http://www.cpearson.com/Zips/modFormatMessage.zip

 

2) Excract the .bas file from each and import into your VBA project.

 

3) Add this code to a code module:

 

Sub OpenPDF_page()
    Dim pageNum As String, PdfPath As String
    Dim part1 As String, part2 As String, part3 As String
   
    PdfPath = "C:\FilesPDF\myFile.pdf"
    PdfPath = "C:\Users\rk\Desktop\Economic Assessment Vehicles - FINAL.pdf"
    pageNum = InputBox("please, select a page", , "1")
    If pageNum = "" Then Exit Sub
    If Not IsNumeric(pageNum) Then Exit Sub
   
    part1 = """" & Get_AcroRd32_ExePath & """"
    part2 = "/A ""page=" & pageNum & """"
    part3 = """" & PdfPath & """"
   
    Shell part1 & " " & part2 & " " & part3, vbNormalFocus
End Sub

 

Function Get_AcroRd32_ExePath() As String
    Dim vKeyVal As Variant
    Dim strKeyName As String, strValueName As String
   
    strKeyName = "SOFTWARE\Microsoft\Windows\CurrentVersion\App Paths\AcroRd32.exe"
    strValueName = "Path"
    vKeyVal = RegistryGetValue(HKLM, strKeyName, strValueName)
    If vKeyVal <> False Then
        Get_AcroRd32_ExePath = vKeyVal & "AcroRd32.exe"
    End If
End Function

a. user say:

Hey austin1_001,

I am working off of the error you posted in your description and not the title, they are two different errors.
The error " root certificate, not secure connection" comes from Entourage not trusting the SSL's connection. For entourage to trust the connection, it has to verify each step in the certificate chain. When one step in that process leads to an unknown or unrecognized certificate, Entourage considers the certificate as a whole untrustworthy. So between Entourage and your server there is a certificate that is being seen as untrustworthy. 

To fix this you need to obtain the root certificate your server is sending you, and add it to your Mac's list of trusted root certificates. You should check your providers support web site, or if that proves difficult, email them asking them for it. 

Once you have it, you need to import it to the keychain "/System/Library/Keychains/X509Anchors"

Hope this was helpful.
