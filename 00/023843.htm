jllatham say:

The easiest way might be to use a 'helper' column to combine the contents of each pair of cells in columns A and B and use that for the comparison.  The helper column could be anywhere on the sheet, but I'll use column C for this example

In cell C2 you'd put this formula and fill it to the end of your list

=A2 & B2

Now you can select columns A and B and use this for the conditional formatting rule:

=AND($C1<>"",COUNTIF($C:$C,$C1)>1)

all rows that have same pairings would be highlighted until you deleted enough of them to reduce that to a single unique pairing - then it would no longer be highlighted.  You can do the deleting from the bottom up or the top down, doesn't matter.  I have another idea that would require deleting from the bottom up, but I need to test it and make sure it works (no helper column involved).

jllatham say:

Ok, here is my "no helper column needed" solution:

Select cells in columns A and B beginning at row 2 on down to the end of your list (and beyond if you want to have room for growth of the lists)

With them selected enter this as the conditional format formula:

=IFERROR(MATCH($A2,$A$1:$A1,0)=MATCH($B2,$B$1:$B1,0),0)

only the 2nd and subsequent duplicate entries will be highlighted.  So you could delete them from the "top down" - any pair that is highlighted has a matching pair somewhere above it in the list.  You could also delete from the bottom up without any problems also; again any highlighted pair has one or more duplicate pair somewhere above it on the sheet.
