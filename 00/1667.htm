macropod say:

Hi David,

Word doesn't have the same kinds of date calculation functions that Excel has. Word is, after all, a word processor.

Word can do, a wide range of calculations, however. In this case, you could use a field coded as:
{QUOTE
{ASK StartDate "What is the starting date, in dd/mm/yyyy format, please?" }
{SET sm{=MOD({StartDate \@ M}+9,12) \# 0}}
{SET sy{={StartDate \@ yyyy}+5+({StartDate \@ M}>8)*1}}
{SET sd{StartDate \@ d}}
{SET ey{Date \@ yyyy}}
{SET em{Date \@ M}}
{SET ed{Date \@ d}}
{SET md{=IF((em=2),28+(em=2)*((MOD(ey,4)=0)+(MOD(ey,400)=0)-(MOD(ey,100)=0)),31-((em=4)+(em=6)+(em=9)+(em=11)))}}
{Set Yr{=ey-sy-(em<sm)-(em=sm)*(ed<sd)}}
"Year {Yr}."}
 
Note: The field brace pairs (ie '{ }') for the above example are created via Ctrl-F9 - you can't simply type them or copy & paste them from this message.

richard xu msft say:

Hi Jason,

 

As we know, Outlook stores the Master Category list in the mailbox or pst file itself so if you backup your pst file, your Categories will be backed up as well. This of course doesn’t allow you to easily share your Categories or use it in another mail profile. To allow for this you can use the following method:

 

1. Backup Categories from another profile which didn’t delete the categories:

  • Create a new Note item (keyboard shortcut: CTRL+SHIFT+N).
  • Write “Categories” in it to give it a name.
  • Close the Note item.
  • Switch to your Notes folder (keyboard shortcut: CTRL+5).
  • Right click on it and choose Categorize-> All Categories…
  • Select all the Categories which you want to backup.
  • Save the Note outside of Outlook as an msg-file Outlook via File-> Save as… or by dragging it out of Outlook and into an Explorer folder or onto your Desktop. By default it will be called Categories.msg

 

2. Restore Categories:

 

  • Open your Notes folder (keyboard shortcut: Ctrl+5).
  • Drag & drop the Categories.msg file from Explorer into the Notes folder.
  • Switch back to the Mail Navigation (keyboard shortcut: CTRL+1).
  • Right click on the top of your mailbox folder set and choose Properties.
  • Click the button “Upgrade to Color Categories…”.
  • Outlook will scan your mailbox for items with Categories which aren’t in the Master Category List yet and will add them.

Richard
