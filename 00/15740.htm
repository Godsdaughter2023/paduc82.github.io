Hi DelIT,

 

For your concerns, as far as I know, when you generate an audit log report, you can only view the site collections you have access to. The question “Are Audit Logs in Office 365 Security and Compliance center security trimmed”, the short answer is yes.

 

Please note that If users are assigned the View-Only role on the Permissions page in the Security & Compliance Center, they won't be able to search the Office 365 audit log. Admin has to assign the permissions in Exchange Online. This is because the underlying cmdlet used to search the audit log is an Exchange Online cmdlet.

doug robbins - mvp office apps & services (word) say:

The TimesNewRomanPSMT font appeared in the Properties of the pdf file as a result of the use of Manual Page Breaks in the Word document.  By running a macro containing the following code to remove the manual page breaks and format the paragraphs that had followed those page breaks to have a page break before them, when the resulting document was then saved as a pdf file, the TimesNewRomanPSMT font was no longer present in the pdf.

Selection.HomeKey wdStory
With Selection.Find
    Do While .Execute(FindText:="^m", Forward:=True, _
    MatchWildcards:=False, Wrap:=wdFindStop, MatchCase:=True) = True
        Selection.Delete
        Selection.ParagraphFormat.PageBreakBefore = True
    Loop
End With
