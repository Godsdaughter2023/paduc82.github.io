a. user say:

Hi Tony,

As per the description, I understand that you are facing issue with sound. As you have run the Sound troubleshooter, I suggest you to follow the methods provided below and check if it helps.

Method 1: Update the Sound drivers.

  1. Press Windows + X keys and select Device Manager.
  2. Expand sound, video and game controllers.
  3. Right click on the driver and select Update driver software.

Method 2: Uninstall the Sound drivers.

  1. Press Windows + X keys and select Device Manager.
  2. Expand sound, video and game controllers.
  3. Right click on the driver and select Uninstall.
  4. Restart the system after the driver is uninstalled.

Hope this helps. Keep us posted on the status for further assistance.
