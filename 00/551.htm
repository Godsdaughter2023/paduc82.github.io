don cuthbert say:

In your boot menu photo, as you have noticed, that all it is offering is UEFI network boot, using IPV4 or IPV6.  That means that your PC is not detecting any other UEFI-capable boot device, such as a USB  flash drive or hard disk.

In order to boot a hard disk or USB thumb drive, the hard disk or thumb drive requires a System partition initialized as GPT and formatted as FAT32.  For the thumb drive, a utility called Rufus can initialize and format it properly.  As a removable drive it can only have a single partition created.  For the hard disk, Windows requires the disk to be empty of any partitions to convert it to a GPT disk.  It is possible to convert it using GPARTED running on Ubuntu Live boot media, but is a risky operation if you are trying to keep your data.  Cleaning all partitions off the hard disk is simpler, but requires a backup and restore.

So if you use Rufus to prepare the USB flash drive as FAT32, it should show up in the boot menu, if your PC is using Secure Boot and UEFI BIOS mode.  Secure boot can be used, as long as your hard disk and USB media are initialized and formatted right.

When using Windows boot manager to try to boot Ubuntu on a UEFI PC, I believe Windows looks for bootx64.efi, so one way that has been used is to rename the Ubuntu boot file to bootx64.efi.  Windows will then boot the renamed Linux file instead.  Windows Vista or newer uses a BCD instead of the old autoexec.bat, boot.ini, etc.  So the BCD has to be edited on a USB or hard disk for multi-boot setups.

I think the process here should work  http://www.dedoimedo.com/computers/dual-boot-windows-8-ubuntu.html.

I don't use Ubuntu much, so I can get by running it from a live disk or in Hyper-V.

gordon setter say:

As soon as you insert a PDF in OneNote as a printout it is not a PDF anymore but just an image (picture file).

You cannot save this as a PDF anymore (except for converting the whole note page into a new PDF which in no way matches the original one)

If you draw on a PDF in OneNote you just place lines, markers and text on top of that image, no form of combining or inserting as done.

OneNote is a note taking app, for "real" annotating of PDF files and saving those edits back in a PDF you need a PDF editor software like Foxit or similar tools.

brian tillman [outlook mvp 2007-2019] say:

"Contact Us" links are typically MailTo URLs and web-based email systems, particularly for Windows 7 and later, don't work.  You can't very well have a MailTo URL start a web browser with the necessary values for opening a new message window.  The target needs to be a mail client.  It doesn't have anything to do with Outlook.com vs. Hotmail, it has to do with how Windows 7 and later handle MailTo URL targets.

The reason you don't see a Send button is because Outlook 2007 is the mail client you have installed, but you don't have a mail profile containing a mail account defined.  The MailTo URL launches Outlook 2007, but Outlook 2007 has no idea what to do without a mail profile.

If anyone want's to play around with trying to define the target of a MailTo so that it creates a new message window for an Outlook.com address, the string for creating a new message window is

https://mail.live.com/?rru=compose

webbrewers3 say:

Paul,

I'd take a look at the url query filter web part. It will let you filter a list on the fly based on the value in a field. You add the web part to a page with your list on, connect it to the list/field and then provide some event to trigger the filter. As a simple example, let's say you have a look up column called "Field" and you want to filter the list to only items with "Value" in that field.  Add a link formatted as <site/page.aspx?Field=Value > to the page and it will filter the connected list and return only those items. 

You may have to tweak it a bit in your scenario but this should sidestep the issue of having to get values from another list.

a. user say:

Hi Siva,

Here's some tips in case you can't send or receive text messages:

  1. Restart your phone.
    The date and time may be incorrect after restarting your phone. To set the time correctly, go to Settings > Time & language > Date & time (Lumia with WP 8: Settings > date+time).
  2. Check the signal strength on the phone screen, and move to an area with better reception.
  3. Send a text message to your own mobile phone number. This will test your sending and receiving capabilities in one go. 
  4. Check the SMS center number (Messaging > ... > settings > SMS center number). If the number is missing or you think it is incorrect, confirm the number with your network operator.
  5. If you have Multi-SIM (two SIM cards with the same mobile phone number), note that some text messages can only be received on your assigned primary SIM card.

Note: If the previous steps do not help, insert a different operator SIM card into your phone to verify that the phone is working properly.
