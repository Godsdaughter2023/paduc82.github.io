Word 2003 compatible text box positioning settings

I operate Word 2007 but often have to save documents as .doc because of a customer who uses Word 2003. However, when I save the document as a Word 97-2003 document I get a message to say that text box positioning will change, and it does - in fact, the page numbers disappear off the page. What positioning criteria can I use that Word 2003 will cope with?
suzanne s. barnhill say:

I would not have thought that the settings for text box positioning had been changed between Word 2003 and 2007, but comparing the Advanced Layout dialogs in the two versions, it would appear that "Relative position" (Horizontal and Vertical) has been added in Word 2007. You can use Alignment, which allows you to set Left, Centered, or Right relative to Margin, Page, Column, etc., but if you specify an exact measurement using "Relative position," then presumably that setting will be lost in backward conversion to Word 97-2003 format. I would think you could achieve what you want either with the Alignment setting or with the "Absolute position" setting (relative to Page).
