If no CD or DVD  is recognized, then it is pretty likely the CD drive itself.  This drive is a common failure point in PCs.  In laptops, they fail more frequently than desktops.  Although the physical replacement is quite easy in most cases, the drive itself is a proprietary part from the OEM of you laptop.  That makes it pretty expensive, although a contact  with them may prove differently.

The only time the ICS PC can access other network resources is when the home network comes up as a PRIVATE connection, but that is very rarely and appears to be random.

This issue was 'fixable' in previous windows 10 feature releases. The resolution (though not easy to look up in online docs etc) was to run the home group troubleshooter. Just to clarify - I never used home groups on my network - but the troubleshooter resolved the problem.

The removable of the HOME GROUP facility in the 1803 release has removed the troubleshooter. Now there is no way to switch a second network to PRIVATE.

The 1803 release needs an urgent bug fix so that when windows asks if this is a private or public connection (which in my case refers to the USB smart phone internet connection), then it switches all connected networks to the option selected (which would include the Ethernet connected network to the router with all devices connected).
