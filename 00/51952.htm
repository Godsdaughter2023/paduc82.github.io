notaspeckofcereal say:

Feature Request: Random change of lock screen image from folder

Windows 10

On Settings, Lock Screen, Background:

Add: "Picture (randomly change)"

Config:

  • When "Picture (randomly change)" is selected, add "Choose Album for your random picture selection." (much like the current 'Slideshow' option Album selection).
  • Another option could be how often to change:
    • Daily
    • Every x hours
    • Every Lock Screen (new image for each lock screen)

Currently: When a user tires of the same Lock Screen image, they manually change it.

Request: For "Picture (randomly change)", select a different image from the selected / configured Album. Each time the user is presented with a Lock Screen, they see a different image from the Album. The system would select the image from the album randomly. 

Notes: This is not a request to replace the Slideshow option. For this new option, when the Lock Screen is viewed, the image would remain static (not change) until the next Lock Screen (after logging on to a Windows session and subsequently coming back to the Lock Screen by one of several mechanisms, whether it be for the home user or the corporate energy saving policy member.

Also, this would be a much more energy efficient option than the Slideshow option, since it would just select a single image and the monitor would still power down per the Power settings just like the standard "Picture" option. It merely displays a different image each time (or on a scheduled basis) the Lock Screen is invoked.

So this allows a user to display their own images, their favorite images, etc. over time without a) using a Lock Screen slideshow, and b) also having a desktop Screen Saver configured.  On that last point: Windows 10 currently does not allow there to be a Lock Screen Slideshow configured simultaneously with a Desktop Screen Saver configured. This new option would appease to the users that are unhappy with that restriction.

debashishdas_systemtest say:

How to add or subtract 2 times, calculation with windows 7 calculator ?

For example:

I want to add 2 different times as follows. Following time represents with Hour:Minutes:Seconds format.

  H MM SS

   1: 50: 31

+2: 20: 30

-------------------

  4: 11: 01

 

How to perform this time calculation with Windows 7 Calculator or any other calculator in Windows 7.

 

Reply.

jetta48 say:

In your example:

 

First add the hours, mins & secs separately

Hours = 1 +2 = 3

Mins = 50 +20 = 70

Secs = 31 +30 = 61

 

Now, calculate whole minutes due to seconds column:

61 seconds/60 = 1.016 mins (approx.)

 

Ignore the bit after the decimal point and add the whole mins to the minutes column:

Mins = 70 + 1 =71

 

Calculate remaining seconds:

61 - 60 = 1 second.

 

Now, calculate whole hours due to minutes column:

71minutes/60 =  1.183 hours (approx.)

 

Ignore the bit after the decimal point and add the whole hours to the hours column:

Hours = 3 + 1 = 4

 

Calculate remaining minutes:

71 - 60 = 11 mins.

 

Hence total is:

4 Hours, 11 minutes, 1 second.

 

The calculator is not a huge help in doing these sorts of calculations because it works in decimal which means it can't directly calculate/convert seconds/minutes/hours that are in multiple of 60s. If you had a spreadsheet program like Excel you could set it up (with a bit of work) to do this sort of thing,

How to Solve the Data Error (Cyclic Redundancy Check) Issue?

1. Using CHKDSK Utility

To solve CRC error, you must repair the hard drive by running CHKDSK utility via command prompt. CHKDSK checks the integrity of your hard drive and fixes logical file system errors, bad sectors & file system metadata. You can run CHKDSK by following these steps:

  • Connect the storage device to your PC.
  • Press ‘Windows+Q’ and type CMD in the search bar.
  • Right-click on the ‘Command Prompt’ and select ‘Run as Administrator’.
  • A command prompt window appears. Type chkdsk X: /f where X is the hard drive or any other media storage device letter connected to your system. Alternatively, you can type chkdsk X: /F /X /R for the rigorous scanning and repairing.
  • Hit the ‘Enter’ button and wait until ‘chkdsk’ utility repairs the disk errors.

After CHKDSK completes error checking, re-connect storage media to the system and then try to access it from File Explorer. If you are able to access the data without any error; immediately copy data from it to a safe location on your system and format the hard drive or any other storage media via Disk Management tool or DiskPart.

2. Run SFC Scan

SFC stands for System File Checker, which is used to repair Windows system files including Windows DLL files, registry files, or other protected files. Sometimes due to corrupt file system or registry files, as discussed above, CRC error occurs.

The SFC scan finds the file issue and replaces the corrupt or damaged file with a new one. The entire process can take around 5 to 20 minutes depending upon the files size and volume. To run SFC scan, follow these steps:

  • Connect the storage drive
  • Click the Windows icon to open the search bar
  • Type cmd. Right-click on ‘Command Prompt’ and select ‘Run as Administrator’ from the options
  • Type sfc /scannow in the cmd window and hit the ‘Enter’ button to run the SFC scan
