    Unfortunately, Microsoft has failed to program the CELL function correctly for international use.

    Function SheetName(Optional ref As Range) As String
        If ref Is Nothing Then
            Set ref = Application.Caller
        End If
        SheetName = ref.Parent.Name
    End Function

    =SheetName(A1)
