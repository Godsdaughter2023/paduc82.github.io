31.74 is incorrect.  The correct answer is 30.0126, computed as follows:  25000*0.0012 (30.000) plus 1000*0.001026 (1.026).

And your formula would return the correct result if you set up the table limits (first column) correctly.  The table should be set up as follows:



Note the limits in column J.  They conform to the instructions in McGimpsey's article (click here), if you read closely.  (Of course, you do not need column K.  I provide it to match your column references.)

However, you will want to round the formula so that the result is dollars-and-cents.  I would write:

=ROUND(SUMPRODUCT(--(A1>$J$2:$J$6), A1-$J$2:$J$6, $M$2:$M$6), 2)
