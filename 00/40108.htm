Windows Easy Transfer vs disk imaging/cloning - Microsoft Community

j_mu say:

Windows Easy Transfer vs disk imaging/cloning

Hi everyone, please I would like to know the difference between using the windows easy transfer utility to migrate files, settings, data...  from an old computer to a new one and just cloning the complete system using either windows or third party utilities. thanks

palcouk say:

Its in the name; a clone is a copy of the entire sys, or drive you have cloned.

If a new PC generally you can only restore data and some setting from a clone image

gts-nj say: Cloning is way of copying the entire system including Operating System and all data to another hard drive.  Cloning software sets up the paritions and master boot record so that the sytem is bootable.  We ususally do this to replace a failing hard drive or to upgrade to a larget drive.  It is generally not usable to move to a new computer.

EFT transfers user accounts, certain data and certain settings.  See here for detail
http://windows.microsoft.com/en-US/windows7/how-does-windows-easy-transfer-make-setting-up-a-computer-easier

j_mu say:

Thanks GTS-NJ, its a lot clearer now, but I thought you could use cloning to install an OS on multiple PC's, kind of still unclear here.Thanks


gts-nj say: You're welcome.

   There are ways of managing a system image for installation to multiple PC's that are sometimes used in a business environment.  This is not as simple as just duplicating an image.   There are issues that need to be addressed like the Hardware Abstraction Layer (if non identical hardware), licensing and activation, etc.  Microsoft has provided tools for this including Sysprep and the newer Windows Automated Installation Kit.   These tools are mainly be used in large scale corporate IT operations.

   There are some consumer tools that can clone an image (or restore a backup) to different hardware, but there are often license and other technical issues that require some knowledge to address.

I hope that clarifies it for you.

    a. user say:

    Hi Dave,

    Thank you for posting in Microsoft Community, we appreciate your interest in Windows 10.

    I understand your concern about computer restart to install the updates on your device.

    I would like to inform that to install the downloaded updates on your device you will have to restart the computer.

    As you have initiated the time to restart the device, your computer will be restarted at that particular time, if you are not using the device.

    Note that even if you manually schedule a restart time, Windows won’t force reboot when it turns out you’re busy using your computer at the selected time. It will offer to delay the restart by what it estimates to be the best time. You will receive the pop-up Reschedule/ Dismiss.

    You can manually set your device to restart after downloading the updates.

    Go to: Settings > Update & Security > Windows Update > Advanced options > under Choose how updates are installed> select Notify to schedule restart.

    You can either let Windows schedule a restart during a time you usually don’t use your device or select a restart time yourself.

    You can schedule the reboot up to 6 days in the future. Of course you can also click Restart now to be done with it immediately.
