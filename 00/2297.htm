suzanne s. barnhill say:

FWIW, there is no built-in style named Hyperlink Followed. The style is FollowedHyperlink, and it is automatically applied to a hyperlink that has been followed. Your custom Hyperlink Followed style would not be used for this purpose.

Moreover, there has never been a way to add a bottom border to selected text (only to a paragraph), so a bottom border could never have been made part of a character style such as Hyperlink or FollowedHyperlink. A border applied to selected text is always a box border. See http://wordfaqs.mvps.org/Overbar.htm#UpperBorders.
