As for terms: Leading is a typographical term which is the height of a line, measured in points like type. An 8 point type with 10 point leading would have 2 points between each line. oksüübaron say:

Theme Fonts in Office for Mac

I am using Word and PowerPoint for Mac; Both versions 16.20, Office 365; MacOS Mojave 10.14.2.

Until recently it was possible to create custom theme fonts (font sets) for Office by adding custom .XML files into the folder ../User/Library/Group Containers/UBF8T346G9.Office/User Content/Themes/Theme Fonts. This seems to have been the only way to add custom font pairs to Theme Fonts selection in Word (under Design > Theme Fonts) and PowerPont (under Slide Master > Fonts) in MacOS. So in addition to the default pairs (eg Arial/Arial, Calibri/Times etc) you got to choose from your own custom sets of fonts, including non-default Office fonts.

The theme fonts selection is important when creating charts and tables in documents and presentations – whatever has been chosen to be the theme fonts, are used by default. Regardless of what you have set up as styles in Word.

So, the XML-thing worked until recently, but does no more and I am stuck with a fixed set of default Office font pairs.

Any new methods for creating your own Theme Font sets for Word/PowerPoint on a Mac?

john korchok say:

This is starting to sound like a bug. Just to rule out an XML bug, could you post one of the non-functioning font themes to OneDrive or other cloud storage, then post a Share link here? I'll examine and test it here. 

As a temporary workaround, you can copy font themes to the folder that stores the Microsoft font themes:

  1. Open your Applications folder.
  2. Right-click on Microsoft Word or Microsoft PowerPoint and choose Show Package Contents.
  3. Open Contents>Resources>Office Themes>Theme Fonts.
  4. Copy your font theme, then restart Word. The theme will not show in the Custom section of the Font Theme dropdown, but alphabetically in the Office section.

This will last until you apply an Office update, at which time the custom themes will be deleted. Keep a copy!
