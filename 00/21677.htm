Font Roulette: the amazing world of MS font transformation

For the last 25 years, I have been migrating up the versions of MS Word and other Office products hoping that the word "up" might indicate something functionally better than the previous version. If all you want to do is write text, save and print, I can say that nothing has changed over that time - the bugs are still the same.

bob jones aka: cybertaz say:
That's because the feature you're using employs a SEQ field which -- by its very definition -- generates a Sequential Number :-) That's so the Table [of Figures, of Tables, of Equations, etc.] can keep proper track of the items.

The actual caption is nothing more than a text string (if the figure is In Line) or a Text Box (if the figure has Text Wrap applied) also generated by the Insert> Caption feature.

In a nutshell, if you don't want the Figure labeling or the sequential numbering & don't intend to generate a Insert> Caption feature :-) Type your own captions as paragraphs above/below/adjacent to In Line figures or draw a Text Box (or other Shape) to label those with Text Wrap.

An alternative, of course, is to continue using the feature delete the field codes from each caption... but I'm not sure that offers any advantage & may over-complicate the underlying document structure.

andreas killer say:

In Excel you have to type the number in the cell and then format the cell with the appropriate number format.

There is a predefined number format in Excel for the English Philippine Peso:

[$₱-3409]#,##0.00

Or maybe you mean "Filipino"

[$₱-464]#,##0.00

And you can have (almost) any currency format used around the world, the syntax for currency formats of cells is

"[$" & Currency Symbol & "-" & Language ID in hex & "]" & usual number format

Sub Test()
  Dim LanguageID As Long
  Dim CurrencyASCII As Long
  CurrencyASCII = 8369 'The sign in ASCII code
  LanguageID = 13321 'The language id (3409 in hex)
  ActiveCell.NumberFormat = _
    "[$" & ChrW(CurrencyASCII) & "-" & Hex(LanguageID) & "]#,##0.00"
End Sub

Execute that and look into the cell format.

You can also apply that with a formula:
=TEXT(1,"[$₱-3409]#.##0,00")

pc-learner say:

There is no Windows 10 Ultimate.  Windows 7 Ultimate will upgrade to Windows 10 Pro.

The keys you can see on Windows 10 upgrades are essentially temporary ones used for installation only.  It doesn't matter that they're all the same, because they can't be used to activate a computer or other device.

Free upgrade computers/devices are activated using their registration in the MS database, done at installation time.

The only way you can get a unique key for Windows 10 is to buy one, presumably for a computer that isn't eligible for the free upgrade.
