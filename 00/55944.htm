bruce hagen ~ mvp say:

The menu bar is hidden by default in Windows Explorer 7. Menus such as: File, Edit, and View are no longer visible by default.

To display these options follow these steps:

  • Go to the Start menu and select My computer
  • If you press F10 or Alt, the task bar will appear, but it will disappear if you press any key
  • For this bar to remain permanently in Explorer, do the following:
    • Go to Control Panel > Folder Options > Views
  • Tick " Always show menus"
  • Click Apply and then OK
