miksu---01 say: Hi,

"My Bluetooth Places" comes with Widcomm driver and it is working only when your BT hardware is using Widcomm driver.

So when you uninstalled Widcomm SW (what you didn't need because you are using MS BT driver), "My Bluetooth Places" uninstalled also.

willidh say: So ideally I should be using the WIDCOMM driver not the MS one included in XP SP2. Is this technically possible? If so, how?

miksu---01 say: Yes, it is possible;

So, if you want to use Widcomm driver:
1. Install Widcomm SW (if it is not already installed)
2. Go to Control Panel, System, Hardware and Device Manager.
3. Open Bluetooth Devices, select Microsoft Bluetooth Enumerator, click Action, Uninstall and OK.
4. Select that BT device, click Action and Update Driver.
5. Select "Install from a list or special location" and press Next. Then select "Don't search. I will choose the driver to install" and press Next.
6. Select the second driver (which is not digitally signed) and press Next.

And If you want to try to use MS BT driver:
1. Go to Control Panel, System, Hardware and Device Manager.
2. Open Bluetooth Devices, select that device, click Action, Uninstall and OK.
3. Click Action and Scan for hardware changes.
(windows should automatic update install MSBT drivers)

HUP PRODUCT KEY's - Microsoft Community

sjmckee say:

HUP PRODUCT KEY's

I purchased a new laptop and it has MS Office 2010 installed, when trying to activate it, it won't except my product key which is

a HUP key from software I purchased. I still have the HUP software, but would prefer to use the version modified for Windows 8

Anyway I can accomplish this.?

dan at it associates say:

That is expected behavior.

 

The Office 2010 Single Image Installation (SII) pre-loaded on the new laptop will accept full retail, product key card, and OEM product keys for the three Office 2010 consumer editions:  Home and Student 2010, Home and Business 2010, and Professional 2010.  The "flavor" of the product key determines which edition and license type gets deployed.

 

The Office 2010 you get from HUP is Office Professional Plus 2010, which has additional components that are not pre-loaded with the SII installation, so that's why it won't accept your Office 2010 Pro Plus HUP product key.

 

Uninstall Office 2010 from the Control Panel>Programs applet, restart twice, then install and activate Pro Plus HUP using the HUP disc and HUP product key.
