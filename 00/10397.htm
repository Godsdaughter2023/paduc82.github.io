Inserting Rows then Sum the previous Rows

Would like to have the ability (macro) that will insert a row after every two rows, then in the new inserted rows, sum the previous two rows in multiple cells but not all cells: The spreadsheet has approx. 5000 Rows.

bill manville say:

If each pair of rows has a specific value in one column, different from the following pair, you can use Excel's Data > Subtotal command.

If that is not the case, you could create an additional column containing such values (assuming the first row is a header and the data starts at row 2) by entering this formula:

=INT(ROW()/2), and filling down,

Then copy the column and paste special values [because otherwise the formula will recalculate while subtotal rows are being added!].

Then you could use Data > Subtotal. 

You could then clear the additional column if you wish.

A macro could be written that would do this for you, but I would encourage the use of Excel's feature first.
