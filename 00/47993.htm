john_barnett say:

Hello vineetthakur,

 

What error messages are you receiving from AutoCad?

 

I would imagine that earlier versions of AutoCAD would have problems with a newer operating system such as Windows 8. However, usually, if an application works on Windows 7 it will 'usually' work on Windows 8. After all Windows 8 is still based on the Windows 7 kernel.

 

Have you tried compatibility mode?

 

1. Locate the AutoCAD install/setup.exe file and right click on it. (You will need to view the files in File explorer rather than auto run the CD.)

2. From the drop down menu, click on the Properties option.

3. In the properties Window, click on the Compatibility tab.

4. When the Compatibility tab opens look for the section marked 'compatibility mode.'

5. You will see an option marked 'Run this application in compatibility mode for.'

6. To the left of this option is a box. Click on this box to insert a check mark (tick).

7. The default compatibility mode is Windows XP SP3. To change this, click on the down arrow to the right of the words Windows XP SP3.

8. A list of operating systems will now appear.

9. From the list, select Windows 7. You could also try an earlier versions such as Vista if you like.

10. Finally press the OK button.

11. Now click on the install/setup.exe file and it should install in compatibility mode.

 

If it still refuses to install in compatibility mode you may have to consider buying the latest version of AutoCad.

 

Another alternative is using Virtual Machine software. However, as AutoCad is a memory intensive program you may find VM software to be slow.

Hi there,


if you own a game is determined by the license, you own the game on disc, so the license is present on the Disc, which is why you will need to insert the disc into the console.

When a game is purchased digitally the license will be available on the account so that account wont require a disc to play the game.


The fact that a game is already fully installed doesn't change anything, since your account doesn't have the license on it it will still require the disc to play the game, since the disc contains the license.

j w stuart say:

No it does not format the hard drive, What is does is place your existing version of Windows into a folder named "windows.old", this in effect clears out the old version files and folders, leaving free space on the existing "NTFS" partition (formerly ocupied by your previous version of windows) to which Windows 8 is installed.

 

And "NTFS" is the only file system Windows 8 supports.
