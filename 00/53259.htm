gerry c j cornell say:

Ken

Your next step is to reconcile the total for C of 31,736mb (say 31 gb) to the 51 gb you stated earlier. The formula for reconciling these figures is:
C partition + System Volume Information folder + Free space + gain from file compression = Disk or partition capacity

The free space figure can be found in the bottom left corner of the Treesize window. What is the current figure.

To ascertain the size of the System Volume Information folder select Start, control Panel, System, System Protection, click on Configure under Protection Settings. What is the Current Usage figure?

You say "Ive been running Ccleaner for a few years". You should update to the latest version. In the latest version when you open the programme you will see a list of options. Click on Tools and then on System Restore. You will gain free disk space by removing the older schedulecheck points and all but the latest one labelled Windows Update.

A caution regarding cCleaner. Do not use the Registry Cleaner.
