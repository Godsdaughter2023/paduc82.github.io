jeeped say:

... what would be the code to find the last cell in column B and get the number in that rows column BA ...


If there are numbers in column B then,
=INDEX(BA:BA,MATCH(1E+100,B:B,TRUE))
... but if there are text values in column B then
=INDEX(BA:BA,MATCH("ž",B:B,TRUE))
... or combined as,
=IFERROR(INDEX(BA:BA,MATCH(1E+100,B:B,TRUE)),INDEX(BA:BA,MATCH("ž",B:B,TRUE)))
... or,
=INDEX(BA:BA,IFERROR(MATCH(1E+100,B:B,TRUE),MATCH("ž",B:B,TRUE)))

 1E+100 is a very large number (i.e. googol) and ž is Alt+0158 and is considered last alphabetically in an Excel sort. The idea being that you want to search for something that is numerically larger or alphabetically after any possible value and return the last row of any value that didn't match.

ossiemac say:

Answer by Jeeped is correct but I thought that you might be interested in a suite of  3  UDF’s (User Defined Functions)  to return the last cell Address, Row number or Value.

 

The UDF’s can be called from other VBA code or from the Worksheet. See examples.

 

It does not matter if the Column is hidden. However, if Rows are hidden, the last visible row is referenced by the code.

 

Suggest that you test them in a new workbook. Insert some data in column B of Sheet1 and  Sheet2 so that each sheet has a different number of rows of data. Run each test sub separately by placing the cursor somewhere within the required sub and press F5. There is a stop inserted in the code. When the code stops and displays the code, hover the cursor over the variables to see their values before Resetting. (To Reset select Run - > Reset or click the toolbar Reset button.)

 

Copy the code into a Standard Module and it can then be called from any sub in any other module.

 

 

Public Function LastCellAddr(rngCol As Range)

    With rngCol

        LastCellAddr = .Cells(.Rows.Count).End(xlUp).Address(0, 0)

    End With

End Function

 

Public Function LastCellRow(rngCol As Range)

    With rngCol

        LastCellRow = .Cells(.Rows.Count).End(xlUp).Row

    End With

End Function

 

Public Function LastCellVal(rngCol As Range)

    With rngCol

        LastCellVal = .Cells(.Rows.Count).End(xlUp).Value

    End With

End Function

 

 

 

'************************************************************

'The 2 subs between the 2 asterisk lines are examples

'of using the UDF's in VBA

Sub UsingTheFunctionsInVBA_1()

    'Example for returning data from the ActiveSheet only

    Dim strLastAddr As String

    Dim lngLastRow As Long

    Dim varLastValue As Variant

   

    'Code for ActiveSheet (Worksheet not specified.)

    strLastAddr = LastCellAddr(Columns("B:B"))

    lngLastRow = LastCellRow(Columns("B:B"))

    varLastValue = LastCellVal(Columns("B:B"))

   

    'When code stops, hover cursor over variables to see their value

    Stop

End Sub

 

Sub UsingTheFunctionsInVBA_2()

    'Example for returning data from a specified worksheet.

    Dim wSht As Worksheet

    Dim strLastAddr As String

    Dim lngLastRow As Long

    Dim varLastValue As Variant

   

    'Code specified worksheet. (Note: Specified sheet can be the ActiveSheet.)

    Set wSht = Sheets("Sheet2")

    strLastAddr = LastCellAddr(wSht.Columns("B:B"))

    lngLastRow = LastCellRow(wSht.Columns("B:B"))

    varLastValue = LastCellVal(wSht.Columns("B:B"))

   

    'When code stops, hover cursor over variables to see their value

    Stop

End Sub

'*************************************************************

 

 

The following are examples of entering the functions in a worksheet cell to return the desired values for the ActiveSheet.

 

=lastcelladdr(B:B)

 

=LastCellVal(B:B)

Mac Word does not insert Citations as footnotes, it inserts only as in-text citations.

What may work for you is that you can create a footnote as usual, then insert the citation into the footnote text. Note, however, that the citation will be inserted as static text rather than as fields... Any citations inserted as footnotes won't be recognized/updated by the built-in  mechanism.
