greginmich say:

How remove windows defender on windows 10

The Windows Security Center should automatically disable Windows Defender when any third-party AV app is installed. In many cases, however, that hasn’t been happening lately due to compatibility issues or corruption in the Security Center registrations. So some of the people who want to “remove Windows Defender” actually just want to disable it in order to keep it from interfering with their new AV app. In that case, you should visit the vendor’s website and check the compatibility of the version that you’ve installed, and then follow any recommendations for upgrading/reinstalling. If that doesn’t work, then you can disable Windows Defender via Group Policy:

http://www.tenforums.com/tutorials/5918-windows-defender-turn-off-windows-10-a.html

Note that disabling Windows Defender in this way is a stopgap measure – so when you have this issue with a purportedly compatible AV app, you might want to first try running the cleanup tools for any previously installed or preinstalled AV apps – and maybe stopping and restarting the Security Center service.

Some people might want to remove Windows Defender because they’re experiencing persistent problems with it crashing, using too much CPU, failing to update, etc., and haven’t found a fix. Other people might find it ineffective against malware – or just not see the sense in settling for “adequate protection” when more effective protection can be had at no cost. I’ve posted a simplified guide for salvaging Windows Defender or switching to another AV app here:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-do-i-delete-windows-defender-10/89eb5c93-24dc-4f7f-84b9-4268368fc211#LastReply
