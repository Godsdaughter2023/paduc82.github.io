suzanne s. barnhill say:
I'm curious as to why you have set the starting page number at 2 on page 2
when "Continue from previous section" would give you the same result.
Unfortunately, this decision has the effect you have seen, that each new
section you create restarts numbering at 2. To correct this, you must do the
following:
 
1. Place the insertion point in the multi-column section.
 
2. On the Insert menu, choose Page Numbers...
 
3. In the Page Numbers dialog, click on Format.
 
4. In the Page Number Format dialog, select the radio button for "Continue
from previous section."
 
5. Click OK to close the Page Number Format dialog.
 
6. Click Close (*not* OK) to close the Page Numbers dialog (if you click OK,
you'll get an additional PAGE field in the footer).
 
7. Repeat for any other section that exhibits the problem. If the section
has its own header/footer, you can use the Format Page Number button on the
Header and Footer toolbar to access the Page Number Format dialog.
