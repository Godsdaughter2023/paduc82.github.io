The document must contain at least some page breaks to create the pages. Please try this to reveal more information about your documents:

If the above doesn't work, try clicking and dragging areas in your document. Is there text visible in the selection? If so, your text may be colored white. Edit>Select All, Format>Font and change the text to Black or Automatic and it should become visible.

It sounds like you may have a problem with some of the Access wizard files on your machine. To try and recreate these wizard files, you could try the following:

 

+>First you will need to locate these files on your machine. By default, they would be in the following location:

 

32 bit Windows machine - C:\Program Files\Microsoft Office\Office14\ACCWIZ

64 bit Windows machine - C:\Program Files (x86)\Microsoft Office\Office14\ACCWIZ

32 bit Windows machine - C:\Program Files\Microsoft Office\Office14

64 bit Windows machine - C:\Program Files (x86)\Microsoft Office\Office14

+>Go to your Start Menu and launch Access from the All Program list. This should then force Access to reconfigure, which will then recreate all of the above files.

+>After Access reconfigures, test your import process again.

 

I hope this helps.
