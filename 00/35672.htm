a. user say:

It's easy to transfer your user data to a new user account (Documents, Pictures, Music, Video, Desktop, Downloads.) Simply copy them to the corresponding folders in the Public user, then move them from the Public user to the new user account. Not only is this easy, if tedious, it helps you avoid situations where you have to reconfigure folder access controls. That's a task you would like to avoid.

Most applications get installed to every user account. Applications that are installed to only one user account can't be moved to another user account - they have to be re-installed in the new user account.

Configuration settings that are only applied to one user account can't be moved to another user account. They have to be re-configured in the new user account.

As you go about setting up your new user account, be mindful to only make configuration choices that are supported in Windows. Unsupported configurations may work, for a while, but they will ultimately create big problems. Don't ask me how I know this, okay?
