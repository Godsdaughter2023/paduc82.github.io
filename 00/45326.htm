    I tried to ring the activation centre but they require my product ID. My product ID is "not available" when I view 'system information'.

    What can I do now?

    hairyfool say:

    You have to be a bit clear about the terms used.

    The license key is a 5x5 alphanumeric key that is used to authenticate Windows when installed, a label carrying such a key used to be attached to computers (known as the COA label) as proof that the computer is properly licensed.

    You are quite correct that no such label is applied to Windows 8 PCs or included in the box. This is because Windows 8 incorporates licensing into the system BIOS itself which locks the installed software to the specific PC by the manufacturer.

    Computers sold with Windows already loaded are pre-authenticated and in normal circumstances do not have to go on line for this purpose. Microsoft's authentication servers know this type of key should not authenticate and any attempt to do so with this key will be rejected as you have found. Therefore the key itself is of no specific value to know what it is as it cannot be used for any purpose.

    The Product ID is effectively the part number for the software installation. Run services.msc and check the startup setting for the Software Protection service is set to Automatic(Delayed Start), if not set it and reboot. 

    Others on here may have alternative fixes that might resolve your problem but as you may have to perform a reset the 1st thing to do would be to back up all your personal data as a precaution.
