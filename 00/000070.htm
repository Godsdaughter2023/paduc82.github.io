robert aldwinckle on forums say:

Is there any way to get this option installed please, or can I access the facility by some other route? I don't want to have to keep going into IE and turning off all add-ons manually every time I want to run a test.

Create your own No Add-ons mode shortcut.  All it is: another standard IE Desktop shortcut (aka .lnk file) with the switch  -extoff   added to the shortcut's Target command line.

However, another way of doing it is just to  Run... (e.g. press Win-R) and enter:

    iexplore.exe  -extoff

FWIW I also like to add  -nohome  to that so I can test IE before it tries to open anything.   The No Home effect occurs before the  No Add-ons  effect so I list them in that order.  Also, when you use  -nohome  you may have to use the  Stop  command  (e.g. press Alt-V p)  to get out of the   No Home  state.

iexplore.exe  -nohome  -extoff

Then depending on how often you use the Run... dialog it would probably still be there for you to use again the next time, (e.g. as long as it stays in Run...'s MRU list.)

BTW "turning off all add-ons manually" is not the same thing as using No Add-ons mode.

in powerpoint 2013 how to direct Insert pictures to correct folder - Microsoft Community

nancy3243 say:

in powerpoint 2013 how to direct Insert pictures to correct folder

When I select insert picture it opens a library contained in Windows Explorer which does not include all pictures
john korchok say:

By default, PowerPoint looks for pictures in the folder that is designated My Pictures. It is possible to change the location of this folder by editing the registry. Here are instructions on how to do this: Change My Pictures Folder

However, what is easier and saves you having to edit the Registry is to simply create shortcuts in the default folder that point to the folder(s) where you actually keep your pictures. To do this, follow these steps:

  1. Open 2 folder windows.
  2. Point one to the folder that normally opens when you use Insert>Picture in PowerPoint.
  3. Point the other window at the folder that holds the folder where you prefer to save pictures. In other words, you're one level up from your preferred folder location.
  4. Right-click on the folder icon that you prefer and drag it to the default My Pictures folder. When you release the mouse button, you'll see a pop-up with 4 choices. Choose Create Shortcuts Here. A shortcut will be created.
  5. If you have more than one folder where you like to store pictures, repeat steps 3 and 4 until you have created shortcuts for all your picture folders.

Now when you use Insert>Picture in PowerPoint, you'll still start at the default folder. But there will be clickable shortcuts there that will instantly take you to your preferred locations.

How to set up Thunderbird to sync your Outlook.com account

I can’t get Thunderbird to sync with my Outlook.com account. How can I correct this? I can’t get Thunderbird to sync with my Outlook.com account. How can I correct this?

a. user say:

To set up Thunderbird to sync to your Outlook.com account’s email:

 

1.       Open Thunderbird.

2.       From the Application menu, choose Add Account…

3.       Input your account information, choose Options… then Account Settings…

4.       Click the Account Actions button, then choose Add Mail Account…

5.       Input your account information.

6.       Click Continue.

7.       Click Manual config.

8.       Set the Incoming settings as follows:

          a.       Server hostname: imap-mail.outlook.com.

          b.      Port: 993.

          c.       SSL: SSL/TLS.

          d.      Authentication: Normal password.

 

9.       Set the Outgoing settings as follows:  

          a.       Server hostname: smtp-mail.outlook.com.

          b.      Port: 587.

          c.       SSL: STARTTLS.

          d.      Authentication: Normal password.

 

10.   Click Done.

 

For more information on connecting your app to Outlook.com, visit this link: http://windows.microsoft.com/en-us/windows/outlook/send-receive-from-app.
