Are you sure you have Office 2008 fully updated? I realize that the installation process should have captured everything, but check Excel> About Excel just to be sure -- It should indicate 12.3.4 as the Version. Custom Error Bars require at least 12.2.0 but that was some time ago :-)

As per the Excel Help topic: Add error bars to a chart

  1. In the chart, select the data series that you want to add error bars to. For example, in a line chart, click one of the lines in the chart, and all the data markers of that data series become selected.
  2. On the Format menu, click Data Series.
  3. In the navigation pane, click Error Bars.
  4. Under Display, select an option for displaying error values as Plus, Minus, or Both.
  5. Tip To delete the error bars from the data series, click None.
  6. Under Error amount, click Custom, and then click Specify Value. The Custom Error Bars dialog box appears. In the Positive Error Value and Negative Error Value boxes, type the values that you want for each data point, separated by commas (for example, 0.4, 0.3, 0.8), and then click OK. Note: You can also define error values as a range of cells from the same Excel workbook. To select the range of cells, in the Custom Error Bars dialog box, clear the contents of the Positive Error Value or Negative Error Value box, and then select the range of cells that you want to use.
  7. In the Format Data Series dialog box, click OK.

If you still have a problem, Quit all programs, run Disk Utility to repair disk permissions, then restart your Mac to see if things improve. BTW: It's a good idea to follow this procedure after applying each required update rather than applying them in succession. Otherwise you may not get them all.
