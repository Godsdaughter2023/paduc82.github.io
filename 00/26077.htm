    doug robbins - mvp office apps & services (word) say:

    If the macros are really in the document, they will not be deleted by saving the file as a macro enabled template (dotm).  If the document contained a custom toolbar, when saved as a macro enabled template, and then used in Word 2007 or later, the buttons from that toolbar will appear on the Add-ins tab of the ribbon.


    If that is not the case, then the macros and toolbar are not in the document.


    If you think they are, if you send me a copy of the document in its original format, I will investigate the issue.

    guessed say: Your terminology is not quite right. An add-in is a separate file which is automatically loaded when Word starts up. A template does not 'contain' an add-in but a template could 'be' an add-in. It is possible to turn add-ins off but saving a document or template does not make add-ins disappear.

    It sounds like what is happening is this. You have a document with macros and a customised ribbon saved in a TEMPLATE which is attached to the document. When you then save the document as a dotm template, the attached template is dropped off. This is because you can't attach a template to a template.

    Check the original file where you can see the macros and ribbon. On the developer tab, have a look at the name of the attached template (second last button of that ribbon). This is likely to be the location of the macros and ribbon customisations. You need to copy those across the the dotm if you want them also to appear in the template.

    charles kenyon say:
    If your macros are truly in your document (as in you recorded them and told Word to save them in the document) and you use Save As and make the file type a .dotm file, they will be there. If you just give the name "myDocument.dotm" it is quite likely that Word is saving as myDocument.dotm.docx. If macros are being stripped out, Word will give a warning before saving.

    As was already pointed out, Add-Ins are not something that are in your document. Any toolbars under the Add-Ins tab come from other files ordinarily. (It is possible to create them in your document with a program, but if that were what was happening, I don't think you would be having the problem you are asking about.) The toolbars under the Add-Ins tab are almost certainly in a different template, perhaps the attached template to your document.

    If the toolbars and your macros are in the template attached to your document, when you save your document as a template, you do break the link to those. They are not saved with the new template because they never were in your document.

    How to Change Where IE11 looks for Favorites - Microsoft Community

    aakunz say:

    How to Change Where IE11 looks for Favorites

    I ran a bunch of Windows Updates (using Win 7 with IE11) and now when I run IE, it shows no Favorites.  I do have my Favorites backed up but I need to know how to get IE11 to look for them where they are located.
    don varnau say:

    Don't understand why IE can't see the Favorites right there in C:\Users\Andrea or how I can manually make it find this location

    -

    You can edit the registry to make IE find that Favorites folder.

    Go to Start> Run, type in regedit and hit Enter.  

    In the left pane of the regedit window, select Computer, right-click it and choose Export. Save that REG file to the desktop. That's your backup. By double-clicking or right-clicking> Merging that REG file you can restore the registry and undo changes, if necessary.

    You navigate the registry just like you would drill down through a folder structure to get to a specific file (key.) Just click the + to open the next level.

    You need to get to the User Shell Folders key.

    Follow this path:

    HKEY_CURRENT_USER

       Software

          Microsoft

             Windows

                 CurrentVersion

                     Explorer

                        User Shell Folders

    Select (left-click) the User Shell Folders key.

    In the right pane right-click Favorites and choose Modify.

    Type this into the Value data field  "%USERPROFILE%\Favorites"  (without the quotes- and no spaces)

    Then click OK and close regedit.

    Let me know if you need additional clarification of any of this procedure.

    Don
