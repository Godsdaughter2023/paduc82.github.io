-Espen-

madcat_sj say:

If sim is removed, phone should not try to connect to GSM network or transmit any signal. If you remove sim while phone is turned off, it will open in "offline" mode.

 

Basically, using full power broadcast towards antenna will kill the battery quite fast. Devices always try to conserve power.

When sim is present, but you are in a place without signal, device will not transmit towards antenna. When a signal presents itself, device will try to register GSM network, by using a full power broadcast.

GSM signal from towers (even from multiple towers, cells) is already being received by device. Device keeps doing calculations for cell power, and tries to change cell when there is another cell with more powerful signal. To change the cell, device will again use a full power signal.
