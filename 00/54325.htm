Clean Install = Erasing hard drive and starting totally fresh.

Repair Install = (quite tricky and involved and should only be used in very limited situations) Keeping programs and data intact and laying down a new install of Windows, with hopes that everything works out right.

Parallel Install = Hard drive is not erased but a new (parallel, and thus potentially very messy) installation of Windows is laid down.

This is a foolproof way to perform a Clean Install:
