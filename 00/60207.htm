lemp say:
The message probably says that there is no default viewer for TIF files (JIF is peanut butter).

Although you can configure any file viewer capable of viewing TIF files as the default, the usual default is Windows Picture and Fax Viewer. 

Easy way:

Find any file that has a TIF extension.
Right click and select "Open with"
Select Choose Program
If Windows Picture and Fax Viewer is listed, select it and check the box to "Always use this program to open this type of file"

If Windows Picture and Fax viewer is not listed, then ...

In Windows Explorer:

  •     On the Tools menu, click Folder Options...
  •     Click on the File Types tab.
  •     Scroll down to "TIF" (if it isn't there, click "New" and add it)
  •     Click the Restore button, if it exists.
  •     Click the Advanced button.
  •     Click on the Open action.
  •     Click on Edit...
  •     In the Application used to perform action: enter

    C:\WINDOWS\system32\rundll32.exe C:\WINDOWS\System32\shimgvw.dll,ImageView_Fullscreen %1

    Where "C:\WINDOWS" is the directory containing your Windows installation.
  •     Press OK

peter smith from redmond say:

I've just been trying this out for you, and have information and good news, and bad news. [edit: I got Juno email to work perfectly; you just have to go to the Juno web site and turn on POP email]

The information is that all the port numbers are all correct in the Mail app and you don't need to change them.  The port numbers that Juno carefully tells you to enter are just \the default port numbers for the SSL-enabled versions of the email protocols.  It's been many years since the major email programs didn't use those ports by default.

The good news is that I managed to get a Juno account set up.  The steps I took are:

  1. Go to Settings-->Manage Accounts-->Add account
  2. In the Add an account dialog, pick Advanced Setup (you might have to scroll the dialog box to see this)
  3. In advanced setup, select Internet Mail
  4. In the internet email account dialog, type in these values
    1. email address is *** Email address is removed for privacy ***
    2. User Name is youraddress without the @juno.com
    3. Password is your password
    4. Account name is youraddress without the @juno.com
    5. Send your messages with this name is your regular name
    6. Incoming email server is pop.juno.com
    7. Account type is POP3
    8. Outgoing (SMTP) email server is smtp.juno.com
    9. Keep all the checkmarks checked
    10. Tap "Sign in"
  5. On Juno.com, change your mail preferences.  If you don't do this step, the Mail app will always just say that you don't have new email.
    1. In a web browser, go to www.juno.com and sign in using your JUNO account. 
    2. Click options-->General to bring up the general options.  In the POP Mail section, select ON - POP my mail to my desktop / smartphone.  Then click Save.
  6. After a few minutes, the Mail app should be able to get your JUNO mail.
