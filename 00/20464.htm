Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Cells.Count > 1 Or IsEmpty(Target) Then Exit Sub
If Target.Address = "$A$1" Then
    Application.EnableEvents = False
    Select Case UCase(Target.Value)

            Call teachers_Salary
        Case Is = "STUDENT"

    End Select
               
    Application.EnableEvents = True
End If
End Sub

charles kenyon say:

I suspect that the pdf was of a scanned image, rather than from a word processing program. If so, you would need to run it through OCR. OCR comes with most scanners. I believe if you open the pdf in OneNote, you'll be able to get text, but it won't be a pdf or Word document.

rohn007 say:

What you want to do is called "OCR", Optical Character Recognition.  This is when the program looks at an image and "recognizes" characters.

Word 2013 does NOT do this!  Starting in Word 2013 you could open PDF files, and if they were formatted correctly, with an "image layer" and a "text layer", then Word can extract the text for editing.

OneNote 2013 can do some OCR.

Open the PDF in your PDF reader application.  Then Print it. Select the "Send to OneNote" printer option.  A dialog will pop up asking where to put the printout.  Once the printout is there, right click on the image. Select the appropriate "Copy text from ... page(s)" option. Then paste, into the OneNote or Word as you wish.

You may have to edit the text to correct characters it could not read properly.

I have not heard of any OCR built into Windows 10.
