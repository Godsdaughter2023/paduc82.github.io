bobbymi say:

Hello LD,

 

This error can happen for any of the following reasons:

·         Windows has not been activated

·         Installing a 3 party application may have triggered the error

·         Windows files may have become corrupt

 

To fix this issue you will need to activate Windows. To do this, follow these steps:

 

1.       If you installed a program and the problem started after that installation, uninstall that program.

2.       Restart your computer

3.       Re-enter the product key by running the SLUI.exe command:

  -   Click Start and type Slui.exe 3 in the Search window and press ENTER

  -   Type your product key and click Next

  -   Follow the instructions to activate Windows

4.       If this is not successful, Activate Windows over the phone, to do this follow these steps:

  -   Click Start and type Slui.exe 4 in the Search window and press ENTER

  -   Select your nearest location and click Next

  -  Follow the instructions to call and activate Windows

5.       Once you have activated, visit this link and Validate your PC:

 http://www.microsoft.com/genuine/validate

 

Also, visit the site below if you have other questions on Genuine Windows:
