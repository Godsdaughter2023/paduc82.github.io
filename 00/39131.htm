In a default installation, Windows creates the page file in the root folder on the same drive that holds the Windows system files. The size of the page file is determined by the amount of RAM in your system. By default, the minimum size is 1.5 times the amount of physical RAM, and the maximum size is three times the amount of RAM (twice the minimum). You can see the page file in a Windows Explorer window if you configure Windows to show hidden and system files; look for Pagefile.sys in the root of your system drive. a. Press Windows and X keys together.

b. Click Control Panel, click on System and then click on Advanced System Settings.

c. Click on Settings, under the Performance section.

e. Click on Change under Virtual Memory Section.

f. Ensure that Automatically manage paging file size for all drives is selected.

g. Click on apply and OK to save the changes
