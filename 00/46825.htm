miguelchiappori say: Hello,

This problem may occur when one or more of the .dll files that the Microsoft Management Console (MMC) uses are missing or damaged. Some of these files are:
Msxml.dll
Msxml2.dll
Msxml3.dll
To resolve the problem, follow these steps:
Click Start and Run, type cmd and click OK.
At the command prompt, type cd% windir% \ system32, and then press ENTER.

Note: the% Windir% folder is the folder where you installed Microsoft Windows.
Write the following commands. Press ENTER after each command.
Regsvr32 Msxml.dll
Regsvr32 Msxml2.dll
Regsvr32 Msxml3.dll
Type exit and press ENTER to close the command prompt window.
Restart the computer.
