rob koch say:

Seems like you already have a fairly good idea of the answer, but here's my take on this.

Yes, Windows Defender included with Windows 10 (or 8.x) incudes not only antispyware, but also the antivirus component that used to be installed separately within Microsoft Security Essentials (MSE) for Windows 7 or earlier versions.

The arguments are for the same reasons they always occur, because some individuals feel strongly that their own favorite security application is better or have had issues in the past while using Windows Defender that they feel make it somehow inferior at protecting them.

Though in some cases the attempt to "sell" an alternate program may be involved, the more common reason is typically the personal belief that their own choice for security is better or that Windows Defender is inferior.  In some of these cases the individual has experienced problems themselves when using Defender, but if you could get them to fully disclose, you'd often find they've also had similar issues while using other 3rd-party security products in the past.

In my own experience, Windows Defender (on Windows 8, though it's improved on 10) and MSE before it have done exactly as advertised, though as with any such signature based program there is always the risk that something may be missed, either due to the malware being very new or otherwise unable to be properly detected.  However, in my own case I've never personally experienced this while using either of them.

As for why you might wish to use an alternate program in place of Defender, in some cases you may desire a program that is more aggressive at protection, such as the need to protect children or other less capable users of the PC while browsing.  Or if these users may download or install questionable 3rd-party files or applications with little regard for safety or review of the messages these may display indicating that additional software will also be downloaded and installed as well.

A common alternative that many here prefer is to retain the use of Windows Defender itself, but also add an alternative on demand antimalware application such as the popular MalwareBytes AntiMalware (MBAM) Free product, taking care not to select the "free Trial" during installation since this installs the Premium paid version which also includes a real-time component that in some cases has interacted badly with Defender.

The reason for this is that MBAM does well as both a sanity check for common malware that Defender normally detects, while also adding more aggressive detection for things like adware and other questionable software often called PUPs (Possibly Unwanted Software) that Defender is not always designed to detect.  This is another reason that some feel the Defender is somehow doing less, when in truth this is the result of a clear set of decisions by Microsoft and often other major security programs as well, to concentrate on the truly malicious software rather than these items which are typically of less concern.

Rob

Try these one by one. Most probably, step 1 should solve your issue.

1. Sometimes, add-ins can cause problems. To determine, if add-in is a problem start your application in safe mode and see if problem has gone away.

To start in Safe Mode -

Hold CTRL key and click on application icon and don't release the CTRL key till it asks you for Safe mode confirmation.

If yes, then start application normally and disable add-ins one by one and start application again every time you disable an add-in to determine the culprit add-in.

2. Sometimes, this is caused due to Word not being able to access the default printer. Close all office applications and change your default printer through Start > Devices and Printers to another printer. It may be some physical one or One Note or XPS or PDF or anything else. If this is the problem, then you will need to update your printer driver for default printer.

4. Login to Windows with a new user profile and see if problem is resolved.

5. Run Microsoft Office Configuration Analyzer Tool 2.2 to list down problem issues
