Please try the following:

  1. Hold down the Alt key and press JP, then SZ. The Format Picture task pane opens.
  2. Press the tab key to rotate through the Height and Width fields.
  3. Press the tab key to get down to Position. When it's highlighted, press Enter to open the position fields, then tab into them.

I realize this is a lot of tabbing and not very elegant. I also have noticed the underlined letters in the task pane. I would expect them to work when the focus is on the task pane (when the tab keys are functional in that area), but they don't. I've never been able to activate them either.

ossiemac say:

Hello Kisnah,

Try the following code. Note the comments about the Width and Height of the picture. You need the top and left to position the picture.

The Top, Left, Height and Width can be changed in the With shpPic code.

If LockAspectRatio is True then the picture size is determined by the last command of width or height. eg if you set the height and then the width on the next line then the height is ignored and vice versa.

However if LockAspectRatio is False then need both the width and height and they may distort the picture.

The code in the With shp Pic is optional. If you don't want to change anything about the picture then it is not required.

Sub Macro1()
    Dim shpPic As Shape
    Dim strPathAndFile As String
    Dim lngLeft As Long
    Dim lngTop As Long
   
    strPathAndFile = "C:\Users\User\Documents\Excel\Test Macros\Toys\Pic2.jpg"
    lngLeft = Range("C3").Left
    lngTop = Range("C3").Top
   
    'Setting Width and Height to -1 retains picture current dimensions
    'and locks the aspect ratio.
    Set shpPic = ActiveSheet.Shapes.AddPicture(Filename:=strPathAndFile, _
                                LinkToFile:=msoFalse, _
                                SaveWithDocument:=msoCTrue, _
                                Left:=lngLeft, _
                                Top:=lngTop, _
                                Width:=-1, _
                                Height:=-1)
                               
    With shpPic
        .LockAspectRatio = True 'Can be True or false
        .Width = 50
    End With

End Sub
