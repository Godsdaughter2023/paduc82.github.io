suzanne s. barnhill say:

Formatting of index entries comes from two sources:

  1. The formatting of the index styles for the various levels (Index 1, Index 2, etc., up to [shudder] Index 9).
  2. Direct font formatting of the XE field. Note that if you insert an XE field in a heading that is, say, 16-point Arial Bold, then the XE field will also have that formatting, and that's okay because that's the Default Paragraph Font. If you remove that formatting, you may find that you get the reverse effect, with the entry formatted incorrectly in the index.

When you insert an index entry, you have the option of making the page number (not the entry) bold. If you check the box for "Bold" in the Mark Index Entry dialog, a \b switch is added to the XE field.

You can search for highlighted text, but you can't specify the color.
