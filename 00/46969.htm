change icon font on desktop

how do I change my desktop icon font & spacing
cmdrkeene say:

There's not a way specifically just the desktop font.  If you change it, it will also change in a great many other places: folders, menus, title bars, other apps that read in your user variables to determine what font to use and such.

Also since screens aren't as universal as they were in the 90's (with varying icon density and resolutions; versus the old days when 1024x768 was just about the standard), setting different options can cause really unpredictable results.

If you do want to get into the deep settings that control it, it's all in the registry. I wouldn't really recommend it, and make sure you make a full backup before you begin.  It's all under here:

Computer\HKEY_CURRENT_USER\Control Panel\Desktop\WindowMetrics

Or, use a program like WindowBlinds by StarDock company (or programs like it). It makes it easy.
