I can't paste in email addresses from excel into Outlook.com to - Microsoft Community

frankbramwell say:

I can't paste in email addresses from excel into Outlook.com to send one email to many recipients.

This is perfectly possible in Hotmail but in Oulook.co. only the last name appears in the send box.
jamesmarwick say: You also do not state if your email addresses are in a column or row. I am assuming the are in a column and are contiguous (cells are next to each other).

 

If my assumptions are correct then the problem is that excel puts a Line Feed (LF) character after each email address. The LF character causes the To: field in outlook.com to ignore everything after the first line feed.

 

While there are many ways to solve this problem, one of the simpler methods might be:

  1. Ensure each email address ends in a semi-colon (;)
  2. Select the contiguous email address cells in the column
  3. Press Ctrl-C or right-click Copy
  4. Click on an empty cell anywhere on the spreadsheet or create a new temporary spreadsheet
  5. Right-click and select "Paste special"
  6. Select "All" and select "Transpose" then click "OK"
  7. The email addresses are now all in that cell reading across (the LF characters are gone)
  8. With the new cell still selected press Ctrl-C or right-click Copy
  9. Go to outlook.com and press Ctrl-V or right-click Paste.

The To: field should now have all your email addresses.

 

I hope this provides some help or answers your question.

Document Inspector "Delete Hidden Rows and Columns" option is missing

I was recently using the Document inspector in Office 2010 and when it reached the Remove Hidden Rows and Columns task Excel crashed. This is a valuable tool for me in cleaning up documents so would like to fix this. Please let me know if anyone has a fix for this.

 

Thanks!

 

chaimlubin say:

Ok, so I resolved my own issue.

 

It seems that since excel crashed when the "Delete Hidden Rows and Columns" portion of document inspector was running that Excel automatically disabled this component. Therefore all that was needed was to go back and enable this function. The different options within document inspector are each their own add-in and can be enable or disabled accordingly.

 

In order to bring back this functionality, once in Excel, I had to go to File -> Options -> Add-Ins

 

Then under the "Manage" drop down at the bottom of this screen I selected "Disabled Items", there I saw the Hidden Rows and Columns Add-In had been disabled. I simply enabled the Add-In, restarted excel and the functionality has returned.

 

I hope this helps more than just me.
