scottgem say:

It makes no difference what the extension of the file is, whether accdb, accde or accdr. The location of the linked tables is stored within the FE. So if the FE says the table is in S:\AccessBackEnds\MyApp\Myapp.accbd, then that's where Access will look and pull the data from. 

So yes an ACCDR front end should have no problem linking to an ACCDB back end.
