rohn007 say:
Quickie review. At it's simplest the goal of normalization is to ensure you type a piece of information only one time in the DB.  As soon as you denormalize, you introduce the chance of "misplacing" data. 

Say you denormalize publication name and you type "Redbook" and "Reddbook" and "Reedbook". When you search for "Redbook" you will find only one entry.

On the flip side, the choice to denormalize should be just that. A thought out, well reasoned decision to skip the overhead of an extra table if it really isn't needed. The tricky part is making a good decision early on in the life of the project.

Think of it this way. It would take a lot of time to cleanup errors in thousands or 10's of thousands of inacurate denormalized data. But, writing the sql and running it to populate denormalized fields in a new table would be trivial.  So, if you invest the time up front to normalize the DB, denormalizing it later is not very significant.  And taking the extra time to think through all of the data to normalize your design will prevent problems in the future.

Normalization is goal, it is not an absolute requirement.

mike h.. say:

Hi,

 

Right click your sheet tab, view code and paste this in. Whenever A1=99 the checbox changes colour. Change the address and value to the correct ones

 

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Address = "$A$1" And Target.Value = 99 Then
    With ActiveSheet.CheckBoxes("Check Box 1")
        .ShapeRange.Fill.ForeColor.SchemeColor = 43
    End With
Else
    With ActiveSheet.CheckBoxes("Check Box 1")
        .ShapeRange.Fill.ForeColor.SchemeColor = 65
    End With
End If
End Sub

 

Or.

 

Right click the checkbox and assign this macro and now ehen it's clicked it changes colour

 

Sub CheckBox1_Click()
If ActiveSheet.CheckBoxes("Check Box 1").Value > 0 Then
With ActiveSheet.CheckBoxes("Check Box 1")
    .ShapeRange.Fill.ForeColor.SchemeColor = 43
End With
Else
With ActiveSheet.CheckBoxes("Check Box 1")
    .ShapeRange.Fill.ForeColor.SchemeColor = 65
End With
End If
End Sub
