graham mayor say:
There are two issues involved in inserting a full page image into Word:

1. Insert an image into Word and it will shrink to the current margins. Select the image, right click and select Wrap text. Change the wrap to anything but 'In line with text'. You will then be able to drag the sides of the image to fill the page.

2. Most personal printers will not print to the edges of the page. They will instead leave an unprintable white border, as you will observe in Print Preview. As the document is to be converted to PDF, which will allow a full page image on screen, you need to set the printer driver to a PDF driver. PrimoPDF and PDFCreator are both free to download and either will do the job. You can then use the driver to create the PDF.

The PDF will still not print to the edge of the paper on a personal printer.

stefan blom say:

You can force an update of cross-references by switching to Print Preview and then return to Print Layout view. You can use Ctrl+P followed by ESC.

If the fields refuse to update, ensure that they are not locked. For example, right-click a cross-reference field and see if Update Field is greyed out on the shortcut menu. If the command is greyed out, fields are locked, and you should select them and press Ctrl+Shift+F11 which will unlock them.
