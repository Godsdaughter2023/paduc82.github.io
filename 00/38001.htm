There are several ways to resolve printer spooler error, probably because there are many variations of this error.

The error caused by faulty software can be resolved by deleting all the information found in the directory.

Thereafter, navigate to the Control Panel, click on Administrative Tools and then services icon.

Find the service labeled Print Spooler, highlight it and restart the process. If that doesn’t work, the program might be corrupted.

Click on start, then Run. Search for the document labeled services.msc, and this will trigger the opening of another window.

On this window, scroll down and click “Print Spooler”.

Check its setting and turn on automatic mode.

Click Okay, and restart the machine.

If you’re lucky, this might be all you have to do. Still doesn’t work?

Navigate to the Add/Remove Programs screen of the Control Panel.

Find the all the software responsible for making the printer run including the drivers, and uninstall them. Be sure to be thorough about it, or the problem may not be solved.

Download the latest copy of your printer’s software from the manufacturer and reinstall it.

Hi Marcia, the homegroup function was removed in version 1803 of Windows 10, please try this:


On each PC, create a new folder. right click that folder and choose properties

Click on the Sharing Tab of the resulting dialog and click 'Share'

Then give that folder a Unique Name on your Network



If you have a problem seeing each PC in the Network Section in File Explorer:


Click your Start Button, type Services and hit Enter

When the Services app opens, scroll down to find the 'Function Discovery Resource Publication' service

Double-click that and set its startup type to Automatic

Then Start that service

Save that and close the Services app

That's It!



If you get a Network Credentials popup, you should be able to turn off that requirement . . .

Open Control Panel

Go to Network and Sharing Center

Select Change advanced sharing settings in the left pane

Expand the network you want to disable password protected sharing on by clicking the down arrow on the right of the profile

Select 'Turn off password protected sharing' and then click Save changes.

churchmice say:

I had this same problem. I have an older receiver (Harmon/Kardon AVR147) that only takes optical in for digital audio, and before I got my Xbox One I had bought an HDMI switch with optical out (about $40 on Amazon) to strip the audio out of an HDMI input to go into the receiver. Here's the one I got:

www.amazon.com/.../B00H7KG0VY

When I got the Xbox, I didn't need this anymore since everything was going to run through it. I connected the Xbox directly to the receiver via the optical out, and HDMI to my tv. Under settings, I had "Optical audio" set to "Bitstream." With that setup, I had the exact problem you're having with Plex movies in stereo - my receiver was only recognizing the Dolby Digital stream and wouldn't use Dolby Pro Logic II (or anything else) to convert to 5.1, giving me just stereo output using the right and left speakers only.

With the switch I linked above, that problem is gone. I now have the Xbox connected via HDMI only to the switch, and the switch has HDMI running to the tv and optical out running to the receiver. Under settings, I have "HDMI audio" set to "5.1 uncompressed", and "Optical audio" set to "Off" (since there's no optical cable running from the Xbox anymore, only HDMI). With this setup, my receiver is once again able to use Dolby Pro Logic II to convert Plex movies with stereo tracks to 5.1. I haven't tested it with running just an HDMI to the receiver from the switch, but my guess is it would work that way too.

Hope this helps you solve your problem.
