john john __ say:

Make sure that "Automatic restart on system failure" is not enabled, the system may be giving you a bugcheck error message (BSOD) but you may not be seeing it if automatic restart is enabled.  Look in the Event Log to see if any relevant errors are being logged.  If you are certain that the problem is caused by one to the 20 updates in the list here is a useful method to narrow them down and find the culprit:

  1. Keep an accurate record of the list of updates
  2. Install half (10) of the updates and use the computer as you normally do for 5 days and see if the problem reoccurs.
  3. If the problem does not reoccur the problem is caused by one of the 10 remaining updates, install half (5) of the remaining updates and wait again for 5 days to see if the problem reoccurs.... and so on.
  4. If the problem does occur after you installed the first ten updates then uninstall half (5) of the installed updates and wait for 5 days to see if the problem reoccurs, ...and so on.

Using this 50% elimination method can help you quickly sort through a list of problem updates (or startup entries) when other methods are exhausted.
