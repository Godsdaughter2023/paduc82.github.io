jay freedman say:
Replace the Date field with a SaveDate field.

You can either use the Insert > Quick Parts > Field dialog and choose SaveDate in the list, or you can right-click the existing Date field, choose Toggle Field Codes, and change the word DATE to SAVEDATE; then press F9 to update the field.

Of course, the first time you save the document after changing the field, it will show the current date, because you just saved it. However, it won't update again until you make some other change and save again.

charles kenyon say:
You could use the SaveDate field instead, but that only changes after the document has been saved (not simply modified but unsaved) and with the field being updated. If it is in a header or footer, it will be updated when you print.

Note that updating the field is itself a modification and will prompt you as to whether you want to save changes.

See Using Date Fields in Microsoft Word.
