rick rothstein, mvp - excel say:

You can do this in VBA quite easily. On the sheet with your hyperlinks, press ALT+F11 to go to the VBA editor and copy/paste the following line into the Immediate Window (press CTRL+G if it is not already showing) and execute it (press ENTER when the text cursor is anywhere on the code line)...

ActiveSheet.Hyperlinks.Delete

You can nowgo back to your worksheet... the hyperlinks are gone. If you will need to do this more than once, then you can make a macro for this. From the worksheet, press ALT+F11 to go to the VB editor, click Insert/Module from its menu bar and copy/paste the following code into the code window that opens up...

Sub RemoveHyperlinks()
  ActiveSheet.Hyperlinks.Delete
End Sub

That's it. You can now go back to the worksheet with the hyperlinks and press ALT+F8 to bring up the macro dialog box... choose RemoveHyperlinks from the list and click the Run button... you hyperlinks are gone.

AutoText and AutoCorrect don't (and never did) operate the same way.

AutoCorrect does (and always did) complete the entry after you enter the complete "replace" expression and press the spacebar or a punctuation character.
