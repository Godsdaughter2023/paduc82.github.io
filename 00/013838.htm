When you attach a file to an email, the attachment has to be encoded as text.  This adds 33 1/3 % to the size of the attachment.

Microsoft already has a solution of uploading a file to SkyDrive and sending a link.

This eliminates the need to encode the file and also eliminates the email size restrictions.  So there is no need to find out what size of a file an email recipient can receive.

Bandwidth is saved when you upload the file and when the recipient downloads the file.
