stefan blom say:
If you chose one of the built-in formats that includes heading styles, what you are seeing is to be expected, since the headings by default have the "Style for following paragraph" set to "Normal" in the Modify Style dialog box. You should change that, if you plan to use headings with no intervening text. In addition, you should turn off the "Keep with next" setting which is also enabled for the headings.
