suzanne s. barnhill say:

Even if the footer margin is set to 0cm, you will still see some footer area if you access the footer because Word must allow space for the default empty paragraph. If you also set the bottom margin to 0, then the footer will appear to be (and would be if you added one) above the bottom margin. The footer margin determines where the bottom of the footer is; if the footer content exceeds the space between that margin and the bottom margin, then the footer will begin to push the bottom margin of the document/page up.

 

I'm not sure what you mean by "When I slide the ruler that says top margin when I hover over it." That setting affects the top margin of the page, not of the footer.
