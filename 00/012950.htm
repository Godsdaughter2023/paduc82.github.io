chip pearson say:

On the Windows Start menu, choose Run and enter the following:

 

C:\Program Files (x86)\Microsoft Office\Office14\Excel.exe  /regserver

 

Of course, your path to Excel 2010 may be different. To determine the appropriate path, open Excel 2010, then open the VBA editior (ALT F11),  then open the Immediate window (CTRL g) and then enter the following in the Immediate window and then press ENTER:

 

?Application.Path

 

This will return the name of the folder in which Excel 2010 is installed. Use that folder in the Run command above.

The /regserver switch causes Excel to start up and reset and reregister all its registry keys and settings, among other things. /regserver can cure many problems.

emilyseymour say:

Using a Mail Merge Recipient list on Another Computer

Hello, I have a recipient list saved onto my computer. But I'd like the same list to be on another computer without having to manually enter them in again (it's a large list), how can you do this? Can you just copy and paste the 'My Data Sources' folder onto a USB and than paste that folder into 'My Documents' on the other computer? 

Help would be appreciated, thank you.

doug robbins - mvp office apps & services (word) say:
Use the recipient list with a Directory type mail merge main document in which you insert a one row table into the cells of which you insert the mergefields.  Then when you execute that merge to a new document, that document will contain a table with a row of data for each record in the data source.  You can then add a row to the top of the table into which you insert names for the fields and then save that document and put a copy of it on the other computer for use as a data source.

emilyseymour say: Thank you for your time but I'll be honest, I'm not sure what any of that means. May you please dumb it down for me? I am in a Business Admin traineeship so I am still learning all the terminology. I just created a new Mail Merge recipient list for my boss on my computer but she needs it on her computer, Thank you

doug robbins - mvp office apps & services (word) say:

With a new blank document, from the Start Mail Merge drop down in the Start Mail Merge section of the Mailings tab of the ribbon, select Directory.  Then use the Select Recipients facility to select your recipient list.  Then, insert a one row table in the document with as many columns as you have fields in the recipient list and into the cells of that table, insert the merge fields via the Insert Merge Field drop down in the Write & Insert Fields section of the ribbon, one mergefield to a cell.

 

Then via the Finish & Merge item in the Finish section of the ribbon, select the inappropriately named "Edit individual documents" item.

marysully say:

Along with Doug’s suggestions, you can also just copy and paste the .mdb file (as you asked) onto a usb drive and use it on another pc to create a new mail merge.

 

The mail merge recipient list will be located in the “Data Source” folder on your pc. Find the list/file in that folder. This may take some searching if you don’t remember the name of the file.  It will have an .mdb extension as it is an Access table.
