joeu2004 say: JP Ronse wrote:
Is there a way to update the Statusbar when ScreenUpdating = FALSE?
I will "hide" the screenupdates that the macro performs but in the same sub, show the progress in the statusbar.

Seems to work without doing "anything" special.  See the experiment below.

However, I find that it is prudent to call DoEvents each time after changing Application.StatusBar.  Otherwise, the Excel thread might not have a chance to display the change, depending on how cpu-intensive the VBA code is.

Option Explicit

Sub testit()
Dim t As Date, i As Long
Range("a1").Clear
Application.ScreenUpdating = False
For i = 1 To 10
    t = Now
    Range("a1") = t
    Application.StatusBar = t
    DoEvents
    Do: Loop Until Now <> t
Next
End Sub

[EDIT] Execute the macro by clicking on an empty worksheet tab, then pressing alt+F8 and running "testit".
