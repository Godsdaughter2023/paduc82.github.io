dell-vishal s say:

Hi Stefania,

 

My name is Vishal and I work for the Social Media and Community Team at Dell.

'Spooler Subsystem App has encountered a problem and needs to close' is not a virus, and it’s just that there might be some error with the Spooler file, which is a printer related file. Check the following steps and the Microsoft Fix it link to fix the issue:-

 

  • Open a Run window (Windows Logo key+ R), type CMD and press Enter. Now type these commands, which are in capitals for clarity:
  • NET STOP SPOOLER and press Enter
  • DEL %SYSTEMROOT%\SYSTEM32\SPOOL\PRINTERS\*.* and press Enter
  • NET START SPOOLER and press Enter
  • EXIT and press Enter


  • Microsoft Fix it Link: - http://go.microsoft.com/?linkid=9740813

 

In case both of the above steps don’t work, please go through the following steps in the link below which will help you fix the issue.

 

http://support.microsoft.com/kb/810894

 

Hope this helps you to resolve the issue!!

 

Thanks,

 

Dell-Vishal S
