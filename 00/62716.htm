austin_gao say:
您好，我是微软社区独立顾问（Microsoft Community Independent Advisor）Gao ling，很高兴为您服务！

关于 “Windows 安全中心威胁记录无法清除” 的问题，请您尝试重启 Windows 安全中心，看一下能否解决此问题。

（注意：此操作要对注册表进行更改，请务必进行备份。）

按 “Windows 徽标键+X”，启动 “Windows PowerShell（管理员）”，输入以下命令：

reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender" /v "DisableAntiSpyware" /d 1 /t REG_DWORD /f

回车后重启设备，然后再次启动 Windows PowerShell（管理员），输入以下命令：

reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender" /v "DisableAntiSpyware" /d 0 /t REG_DWORD /f

回车后再次重启设备，查看问题是否解决。

xbf javier r say:

Hello there fellow gamer,

Considering that this is occurring on all 3 consoles, a suggestion here would be to roll back the system updates for one of the consoles to test possible corruption in regards to the OS.

This can be done by following solution 7 within our troubleshoot problems with an update to your Xbox 360 console support article:

Solution 7: Reinstall the latest Xbox 360 system update

To reinstall the latest Xbox 360 update, follow these steps:

  1. Clear the system cache. To do this, see the "Clear the system cache" section above.
  2. Roll back and then reinstall system updates. To do this:
    1. Press the Guide button  on your controller.
    2. Go to settings and select System.
    3. Select Storage.
    4. Enter the following button sequence: X, LB, RB, X, LB, RB, X.
    5. Select the option to roll back all system updates that are installed. The console will restart automatically and then download the updates again.
  3. Reset your network hardware. Here's how:
    1. Unplug the power cable from the back of the router, modem, or gateway. If you have a router and a modem, unplug the power cable from both devices.
    2. Turn off your Xbox 360 console by pressing the power button on the front of the console.
    3. Wait 5 minutes.
    4. If you're using a router, plug in the router and wait for all the lights to return to their normal state.
    5. Plug in the rest of your hardware and try to connect to Xbox Live.

Try this for us and let us know the results.
