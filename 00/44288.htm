Method 1: Run Chkdsk to Check the file system and physical error

Disclaimer: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.

  1. Start computer in "Safe Mode with Command Prompt".
  2. In Command Prompt, type in "Chkdsk /f/r" and press enter.
  3. If prompted any message press "Y" key on the keyboard and press enter to run
    scan on next reboot.

Note: If used without parameters, Chkdsk displays only the status of the volume and does not fix any errors.

Note: /f Fixes errors on the disk and /r Locates bad sectors and recovers readable information.

Important: Interrupting chkdsk is not recommended. However, canceling or interrupting chkdsk should not leave the volume any more corrupt than it was before chkdsk was run. Rerunning chkdsk checks and repairs any remaining corruption on the volume.
