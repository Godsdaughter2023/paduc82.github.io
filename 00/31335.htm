andre da costa say:

Starting with Windows 8, Microsoft had changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Eliminating the product key sticker also removes any worry that the sticker might get damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

If the user has to reinstall the operating system on a machine that came with Windows 10, the installation process automatically grabs the software product key from the motherboard BIOS with no input from the user. This means that those familiar Windows product key stickers will no longer appear on the Windows 10 computers.

You should be able to find the product key using the following instructions:

Windows 10 came preinstalled on my computer, how do I find the product key?

Press Windows key + X

Click Command Prompt (admin)

Enter the following command:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Hit Enter

The product key will be revealed, copy the product key then enter it

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

If you are unable to activate Windows 10 after all the above and you are certain you have the correct edition installed, then please attempt phone activation:

Activate Your Windows 10 License via Microsoft Chat Support

Hi,

Normally, there is no option to rename the user folder name without encountering an error. Once you've created a user account on the computer, its user folder name will be permanent. You can rename the user folder of that account using a different administrator account however, the user account will become a temporary user profile. As a workaround, we recommend creating a new local administrator account so that you can set up a new user folder with the name that you want.

If you really want to change the user folder name without creating another user account, we recommend posting your concern in our Windows 10 TechNet forum because this will involve editing the registry. When you create a new user, it will ask you to input the username of the new account. The username that you will input will become the name of the user folder of that account. If you will change the name of the user folder of an account using a different administrator user, that account will automatically change to temporary profile because Windows will not be able get all of the information of that account.
