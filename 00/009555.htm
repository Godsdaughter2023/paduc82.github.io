Office 365 business offline - Microsoft Community

n0495923 say:

Office 365 business offline

hi ,

Is there a way to install office 365 for business offline ?

frank fei msft say:

Hi N0495923,

Yes, you can use Office Deployment Tool to download the source and then install offline.

1. Download Office Deployment Tool.

2. Setup the configuration.xml file replacing the SourcePath as appropriate in your case.

<Configuration>
	<Add SourcePath="PATH_TO_YOUR_DOWNLOAD_FOLDER, e.g. E:\Downloads" OfficeClientEdition="32" >
		<Product ID="O365ProPlusRetailO365BusinessRetail">
			<Language ID="en-us" />
		</Product>
		<Product ID="VisioProRetail">
			<Language ID="en-us" />
		</Product>
	</Add>



3. Finally run the following command to download the source.


 setup.exe /download configuration.xml




4. Wait for the download to finish. This can take some time depending on your internet speed.


Once you have the installer downloaded you can easily install it offline using


setup.exe /configure configuration.xml




For your reference:

Download Click-to-Run for Office 365 products by using the Office Deployment Tool



Deploy Click-to-Run for Office 365 products by using the Office Deployment Tool

What I suspect is that you are using a "hide messages marked for deletion" view in the Inbox.  When the rule moves the message, it marks the original for deletion, but doesn't actually delete it.  Now, Outlook 2010 can be figured to put deleted messages

You're moving a message by rule.  A copy is placed in the destination folder (just like when you're deleting) and the original is marked for deletion, with the view hiding it.  I suspect that if you change your view to one that doesn't hide marked messages,
 you'll see the original copy in the Inbox with a line through its entry.

    After clicking on a hyperlink in Word 2008, how do I return to the - Microsoft Community

 belkin3  say:


            After clicking on a hyperlink in Word 2008, how do I return to the previous place in the text?

                    For example, if I click on a page from the Table of Contents, how can I return to the Table of Contents without scrolling all the way back up? I'm working with a long document, so it's amounting to a considerable amount of time.






Other versions of Word have hyperlink menus with back buttons, but I can't find anything similar in Word 2008.







Thank you for any suggestions!

                



            



            



            
        

             john mcghie  say:

    
    

            
            
            
            You can't -- the function does not work correctly in Word 2008.




If you click in the text, then click a hyperlink, you can get back to the previous cursor position with Shift + F5.





However, this often doesn't work in Word 2008.  Especially, it does not work in the TOC because you are not clicking a hyperlink.  Word Mac does not have Hyperlinks in the TOC, they left that feature out of the TOC Generator.  It's actually a GOTO bookmark,
 and if you click it, it does not update the cursor position.  Bugs, bugs, bugs...





Try Outline View and the Document Map.





Document Map appears in the Word Help.  Outline View is here: http://office.microsoft.com/en-us/word-help/CH001026780.aspx  Sorry:  Microsoft thought Mac users were too stupid to
 use Outline View so they left it out of the Help.  We put them right in Word 2011...





The Document Map displays a collapsed view of the document.  It's good for skipping around in the document, but not much good for editing.





Outline View is a full document structure editing view.  Very powerful, and it's what the professionals use for long-document work.  It also enables you to skip around the document, but it also enables you to hurl chunks of text around.





Both relay on Styles.  Don't use Document Map unless you have already styled the headings in the document, otherwise it will try to "guess", slapping styles "anywhere" with frightening results.





Hope this helps
