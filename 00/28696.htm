hansv mvp say:
It depends on whether the landscape page ends up being a left-hand (even-numbered) or right-hand (odd-numbered) page. She can use an even section break or odd section break to force the landscape section to begin on an even or odd page.

suzanne s. barnhill say:

As Hans suggests, publishing convention is to put the top of a landscape table at the left side of the (portrait) page. This means that the top is on the inside of recto (odd) pages but on the outside of verso (even) pages. That is to facilitate reading tables that continue from one page to the next, requiring rotation in only one direction.

I don't think that's what you're trying to change, though. The problem comes when you send a document to a duplex printer. Some printers rotate landscape pages the wrong direction in relation to portrait ones. There really isn't much you can do about this except print those pages separately, manually rotating the page as required to get the proper orientation.

PdF will not convert to Word - Microsoft Community

gd_daze12 say:

PdF will not convert to Word

PDF will not convert to Word so I can edit. Using Windows 7

rohn007 say:

Not all PDFs will allow you to edit the text.  The PDF has to be in a special format that has text in a separate "layer" from the images.  If you are not getting text from the PDF, you will need to use an OCR, Optical Character Recognition, tool to extract the text.

PDF editing in Word 2013

http://office-watch.com/t/n.aspx?a=1730

Any ‘open’ PDF file (that is one that permits changes or extraction of data) can be opened in Word 2013 via the File | Open dialog or ‘drag and drop’ just like any other document.

Word 2013 can only convert to editable text content that’s available as embedded text in the PDF. Not all things that appear as text in a PDF are actually stored with a ‘plain text’ version as well. If the PDF has an image (even if that image is of a page full of text) then Word can only display that image.

To edit the imported PDF you need to go one more step – to View | Edit Document

When you try to save the edited PDF back to the original PDF – you can’t! Word 2013 will present a Save As dialog box defaulting to a docx format.

You can change the ‘Save As’ type to PDF but you can’t save it over the original PDF. Word 2013 considers that an open / read-only document which isn’t accessible.

Find/Replace the current "text highlight colour" to a - Microsoft Community

davidfutures say:

Find/Replace the current "text highlight colour" to a different colour

How does one go about searching an entire text word document and replace the current "text highlight colour" to a different value such as increasing the font size or underlining the words and removing the highlighted colour? looking forward to a comprehensive answer. cheers dave

suzanne s. barnhill say:

To remove the highlight and format the text as underlined:

  1. On the Home tab, in the Font group, click the arrow beside the Highlight button and choose No Color.
  2. Press Ctrl+H to open the Replace dialog.
  3. Click More>> to expand the dialog.
  4. With the insertion point in the "Find what" box, click Format and select Highlight. You will see Format: Highlight under the box. This will find currently highlighted text.
  5. With the insertion point in the "Replace with" box, click Format and select Highlight. This will replace with the currently selected Highlight color (which is No Color).
  6. With the insertion point in the "Replace with" box, press Ctrl+U. You will see Format: Font: Underline, Highlight.
  7. Click Replace All.

If you want to change the font size instead of (or in addition to) underlining the highlighted text, with the insertion point in the "Replace with" box, click Format, then Font, and select the formatting you want. You can also use Ctrl+I and/or Ctrl+B in the box to add italic and/or bold.
