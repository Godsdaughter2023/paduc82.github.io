bossdweeb say:

Looks like a fairly lean Win10 install (my System is currently at 24.5GB - 27.4 including 'Reserved') .

System is of course Windows itself. Why MS says - "These system files help Windows run properly" is one

of the mysteries of how MS does things. These files are Windows .

If you open C:\ and R/Click on the Windows folder you will get a number that matches, or very nearly matches,

the total of System and Reserved as shown in Settings > Storage.

System Reserved is, or used to be, a small amount of space allocated for Boot configuration, among other things

when Windows gets installed. You cannot safely remove it after Windows is installed, and doing so is not recommended. I'm not sure if they are using the term in the same way on Win10.

It looks to be more about space allocated to the Page File, Hibernation and System Restore.

- The default Virtual Memory ( Page File ) is dynamic, so it can move up and down, but if you have 8GB or

more RAM you can normally get by with 1 or 2GB  for both min and max. 

Add another 1GB for a decent size System Restore. Not enabling Hibernation can save more space.

- If you are running short on HDD space and 2GB is 'kind of a  lot of space' your HDD/SSD is too small.Particularly if you have less than 15% free space as Windows needs this much free space for defragmenting

a HDD, and on SSD to facilitate quicker write speeds and as replacements for 'dead' memory cells .

You either need to move some data to an external drive or buy a larger drive.
