teylyn say:

Hello,

when you select the data and use the command Data > Subtotal, then the subtotals are automatically formatted bold. 

If you want to bold the actual numbers, then you can use conditional formatting that checks the first column for the word "total" with this formula:

=SEARCH("Total",$A1)>0

Adjust the formula to your column letter, but make sure that the column letter is preceded by a $ sign if you want to apply the format to several number columns.

Use the CurrentRegion property to get the data cells, then you can use any formatting property of the Range object as you like (and the macro recorder shows, but WITHOUT Select, Selection).

Andreas.

Sub Test()
  Dim R As Range

  R.BorderAround

  R.Borders(xlInsideVertical).LineStyle = xlContinuous
End Sub

bob jones aka: cybertaz say:

IIRC, that's a part of the MUD content that was relocated several years ago. The locally installed Help files never have been updated :-\

The current location is:

/Users/ username/Library/Application Support/ Microsoft/Office/Office 2011 AutoRecovery

Just in case you aren't aware :-): Please keep in mind that AutoRecovery is not a 100% guaranteed Golden Parachute. It's a feature that attempts to recover unsaved work in the event of a critical failure. In some circumstances that simply cannot be done. Under no circumstances does it modify your actual document file & should never be thought of as either a backup or as an automatic save utility.
