I had an account on my Office 365 Tenant with this issue.  Here are the steps I used to correct the problem.

To reset the name of Archive1 to Archive, follow these steps to change the language in OWA to something else and then change it back.

 

  1. First make sure there is no folder present named “Archive”
    1. If there was one created by you or anything else, delete it first and make sure Outlook has completed a full Send/Receive
  2. Close Outlook
    1. It is recommended that Outlook be closed during this language change in OWA
  3. Login to the user’s Office 365 mailbox via OWA (http://portal.office.com)
  4. Once in email/Outlook Web App, click the Settings cog, then choose the Mail setting/options link
    1. Notice where the Mail setting link is in this menu, you have to navigate back to it later in a different language
  5. Choose General->Region and time zone and then change the Language to something other than English.
    1. Choose something easy to translate in case you can’t find your way back to these settings.  I used Española (Mexico).
  6. Once you Save the new language change, go back into email and notice the folders are all in the new language. 
    1. If you choose Española (Mexico), your Archive1 folder will now show as Archivo.
  7. Now, return to the Settings Cog and Mail setting/options link again
    1. Change the General->Regions and time zone-> Language back to English (United States)
  8. Notice now the Archive1 folder has been reset to the name “Archive”

jess can say:

Hi,

There is a way to view this information. Here is how to get the IP address of an email sender in Microsoft Office Outlook:

1.      Start Microsoft Outlook

2.      Double click on message header in order to open it in a window

3.      Click File -> Properties

4.      Under the Delivery options in the Internet Headers box you can see the mail header information.

Important! You will not be able to find the real IP address if the sender uses an anonymous proxy server.

Lastly, if you are using a web-based email service like Yahoo, Gmail or Outlook.com - using a browser- this option is not available as they only display the email address where they were sent from.

Should you have any more questions, feel free to contact us.
