The code for the list box must loop through the selections. I have separated your code for each selection to a sub of its own and the required sub is called for each selection in the ListBox.

Note that a sub name cannot have spaces so after the selection is assigned to a variable in the code I have replaced the space with an underscore and the called sub names then contain an underscore instead of the space.

The first loop through the selected options goes in the Userform module. All of the other code goes in a standard module.

For convenience I have used constants at the top of the Standard module for the worksheet name variables so they do not need to be reassigned in each sub.

Note that the code cannot use "Call" for the required sub when a variable is used for the sub name. Must use Application.Run with the variable.

Feel free to get back to me if you have problems with the code examples.

Sub CommandButton1_Click()
    Dim i As Long
    Dim strLstValue As String
   
    With Me.ListBox1
        For i = 0 To .ListCount - 1
            If .Selected(i) Then
                strLstValue = .List(i)
               
                'Can't have spaces in Sub name so replace space with underscore
                'Name subs with Underscore included
                strLstValue = Replace(strLstValue, " ", "_")
               
                Application.Run strLstValue 'Calls the appropriate sub
            End If
        Next i
    End With
End Sub

hansv mvp say:

Before creating the macro, activate the External Data tab of the ribbon.

Perform all the steps for exporting the query but don't click Finish in the Export Text Wizard.

Instead, click Advanced..., then Save As... to save the specification for the export. You can accept the name proposed by Access (probably "cube query Export Specification") or enter a custom name.

You can now cancel the export.

 

Make sure that your database is stored in a Trusted Location (see Create, remove, or change a trusted location for your files).

 

On the Create tab of the ribbon, click Macro.

In the Show/Hide group of the Design tab of the ribbon, click to turn on "Show All Actions".

 

Select the SetWarnings action.

Set Warnings On to No.

 

Select the TransferText action.

Fill in the arguments:

Transfer Type: Export Delimited.

Specification Name: the name you used above.

Table Name: cube query

File Name: C:\My Documents\Access\cube_query.csv

Has Field Names: set to Yes if you want to export the field names to the first row.

 

Select the SetWarnings action.

Set Warnings On to Yes.

 

Save the macro.
