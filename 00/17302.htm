The black screen issues usually occur due to display adapter drivers.

I would recommend you to try the below suggestions and see if it helps this time.

Method 1

Try the Windows device drivers update for your PC to see if it resolves the issue.

Method 2

If you can boot in normal mode, you may disable the graphic card drivers from Device Manager and check. Follow the below steps.

A. Press Windows key + X on the desktop Screen and select Device Manager.

B. Expand Display adapter and right-click on driver and select ‘disable’.

C. Restart the computer and check.
