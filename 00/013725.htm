ken sheridan say:

Can you describe the contexts of the forms, reports etc in more detail.

 

With ordering/invoicing databases it is important to understand that the UnitPrice of a product recorded in a Products table must not be looked up in a computed column in a query or a computed control in an OrderDetails form when inserting a new row into an OrderDetails table for instance.  The price in a Products table will change over time, that in an OrderDetails table should remain static at the value when the order was created.  Consequently a UnitPrice column is also needed in OrderDetails and the current UnitPrice value of a product is inserted into a control bound to that column when the row is inserted into OrderDetails.

 

Northwind (in Access 2007 at least) does this in the AfterUpdate event procedure of the ProductID control in the Order Subform for Order Details form.  The Northwind code is a little long winded, but the basic process of looking up and assigning the value can be done quite simply with:

 

    Dim strCriteria As String

 

    strCriteria = "ID = " & Me.ProductID

 

    If Not IsNull(Me.ProductID) Then

        Me.UnitPrice = DLookup("UnitPrice", "Products", strCriteria)

    Else

       Me.UnitPrice = 0

    End If

The 'change' from unit price to list price is merely a question of labelling.  the caption property of a label can be anything you wish.  When you add a bound control to a form or report its label's caption property is the name of the column (field) to which it is bound by default, but you can change it, and often will want to, to whatever you wish.

kenellmaker say:

Viewsonic VG2732m-LED monitor not detected

After installing Technical Preview my monitor is set to Generic non pnp monitor. My actual monitor is a pnp Viewsonic VG2732m-LED. Is there a way to correct this? The install did find my Nvidia Geforce GTX 275 display adapter.  
ucanfixit say:
After installing Technical Preview my monitor is set to Generic non pnp monitor. My actual monitor is a pnp Viewsonic VG2732m-LED. Is there a way to correct this? The install did find my Nvidia Geforce GTX 275 display adapter.  

Your monitor was detected; however, the installation could not find the appropriate manufacturer's driver (in its driver database). This has nothing to do with your graphics card. You will need to try and install the latest driver from Viewsonic to fix this issue; however, if your display is working properly, I wouldn't even bother because most manufacturers simply issue the same driver as you already have (eg, it has the exact same color profile as the win generic driver) so it is really only a benefit in identifying the specific monitor (by manufacturer name) in device manager. You can download the latest driver here:

http://www.viewsonic.com/us/vg2732m-led.html

blakew. say:

RichH67,

To check what exactly is using you're storage on your phone follow the steps bellow:

  1. Navigate to "Settings"
  2. Scroll down to "Storage Check" (Almost at the bottom)
  3. Give it a minute or two and then the storage should be loaded. Make sure that the total is 64GB because if it isn't that means your microSD Card is not being loaded
  4. Swipe to right to the "Details" page.
  5. From there you will be able to see what is using most of your storage.

Another possible solution is that the microSD Card that you had purchased is faulty or it is not seated in the phone correctly. If the above total space is not 64GB then try to take your microSD Card out of your phone and put it back in. Usually they make them so that it will only go in one way, but just be sure that you're putting it in the correct way.

Hope that helps,

Blake
