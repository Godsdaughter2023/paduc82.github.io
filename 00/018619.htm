marshall barton say:

That is correct analysis of why you can't do it that way, but it is not what was happening.  The group header events can see the first detail record and that should be what you were using to determine the label's visbility.  As you deduced, that's not very useful in your situation.

Instead, add an invisible text box (named txtNotNull) to the group header section and set its expression to:
    =Count(productionper8hrs)
where productionper8hrs is presumably the name of the record source field, not a text box control, that has the label you want to hide.  Then you can use this line of code to hide the label when all the details in the group are Null:
   Me.Label21.Visible = (txtNotNull > 0)

The only trick here is to remember that all the aggregate functions (Count,Sum,Max,etc) ignore Null values so the count will only be 0 if the field is Null in all of the detail records.
