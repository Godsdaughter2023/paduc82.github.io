rick rothstein, mvp - excel say:

Okay, maybe I see what you want. Give this macro a try...

Sub AlternateColors()
  Dim X As Long, LastRow As Long, LastColumn As Long, Col As Long, Color As Long
  Const FirstColor As Long = 34  'Blue
  Const SecondColor As Long = 38 'Pink
  Const ColorSum As Long = FirstColor + SecondColor
  '  Find first blank in Column A
  LastRow = ActiveCell.End(xlDown).Row
  '  Calculate last column in use
  LastColumn = ActiveCell.End(xlToRight).Column
  '  Store ActiveCell column number
  Col = ActiveCell.Column
  '  Clear existing colors
  ActiveCell.Resize(LastRow - ActiveCell.Row + 1, LastColumn - Col + 1).Interior.ColorIndex = 0
  '  Set color to the one you want first (assumed to be blue)
  Color = FirstColor
  '  Fill first row
  ActiveCell.Resize(, LastColumn - Col + 1).Interior.ColorIndex = Color
  '  Loop through cells after first cell
  For X = ActiveCell.Row + 1 To LastRow
    '  If current cell differs from previous cell, then swap colors
    If Cells(X, Col).Value <> Cells(X - 1, Col).Value Then Color = ColorSum - Color
    '  Color the cell with the currently selected color
    Cells(X, Col).Resize(, LastColumn - Col + 1).Interior.ColorIndex = Color
  Next
End Sub

how do I find a Word document that has been accidentally deleted - Microsoft Community

lmfo in ak say:

how do I find a Word document that has been accidentally deleted

 I had just finished a Word document I have been working on, and was in the final stages of editing it I highlighted the whole document to change the font and it disappeared. what can I do to get it back.
john mcghie say:
If you have not quit Word or closed the document, you can Command + Z to undo whatever you just did.

If you have quit Word, look in the folder the document was in.  If you have Word>Preferences>Save>Always create backup switched on, you will find the version from the most recent manual save in the same folder, labelled "Backup of..."

Otherwise, you will need to restore the document from your backup.

If you haven't switched "Always create backup" on, there is only one copy of the file; if you delete the text, it's gone.  If Time Machine is running, the version from an hour ago is on there (provided you have saved it).

If you never saved it to a file name, it actually never actually existed in physical form and now it's gone forever.

    I misread the question initially. 
    Is what you want is the average of the last 3 values in the row, or the average of 3 adjacent cells up to and including the last one in the row that contains a value?  [there is a difference in the case that there is no value in the month before last]?

    The second is easier (the average of the 3 months leading up to the last value)
    You can use an array formula, entered with Ctrl+Shift+Enter.
    =AVERAGE(OFFSET($A2,0,MAX(IF(ISNUMBER($B2:$IV2),COLUMN($B2:$IV2),""))-3,1,3))

    Error: 0xC004F074 The Software Licensing Service reported that the computer could not be activated. No Key Management Service (KMS) could be contacted. Please see the Application Event Log for additional information.

    Please first check if your system clock is set to the correct time.

    As experienced Windows users known, when activating Windows or other Microsoft products, need to compare the time and set your computer's time to the correct time.
    If your computer's time was incorrect, the activation server would not activated your product, so 0xC004F074 error also would be occurring.

    I was not particularly clear the allowed incorrect time range for activation. Generally if it was shy of two or three minutes, namely just within a few minutes, would be allowed; otherwise if the incorrect time reached over about ten minutes, it certainly would not be passed.
    So, before running the activation command, first be sure to compare the time and set your computer's clock to the correct time.

    You could manually adjust and set the time to correct, also could run the following command to resync the time on the computer:

    w32tm /resync
