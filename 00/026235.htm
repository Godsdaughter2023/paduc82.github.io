Having different page numbers in header and footer - Microsoft Community

benzaman benzaman say:

Having different page numbers in header and footer

Everyone, 

As a novice user of MS word, I was using section breaks and wanted to edit and header and footer in a way that the footer would show the total page number and the header would show the section wise page number. 

For example - a 50 page document I have created 5 sections, each with 10 pages. What I am looking for is that the footer would show the total page but the header will show section wise page number (for example section 1 - page 3 out of 10 or section 4 page 5 out of 10). And this pattern will continue throughout the document. 

What is a smart way of doing this so that if I make a template, if wont become repetitive?

Please let me know

 

macropod say:

Word doesn't really support that and its implementation is a bit convoluted:

1. In your footer, insert a set of fields for 'Page x of y' numbering, coded as:

Page {PAGE} of {NUMPAGES}

2. Ensure each Section's headers are unlinked from the previous Section.

3. In the first Section's header, insert a field coded as:

{SET Sctn{SECTION} {SECTIONPAGES}}

then copy it to all subsequent Section headers preceding the last one in which you want the Section numbering to appear. This will create a separate bookmark for each Section named Sctn#, where # is the Section No, with a value equal to the page count for that Section.

4. In the first Section's header, insert a field coded as:

Page {PAGE} of {SECTIONPAGES}

5. In each Section's headers after the first, insert fields coded as:

Page {={PAGE}-Sctn1} of {SECTIONPAGES} - in Section 2

Page {={PAGE}-Sctn1-Sctn2} of {SECTIONPAGES} - in Section 3

Page {={PAGE}-Sctn1-Sctn2-Sctn3} of {SECTIONPAGES} - in Section 4

and so on.

If you have continuous Section breaks for Sections that don't span more than a single page break, you can omit those Sections from the {SET Sctn{SECTION} {SECTIONPAGES}} field. If they do span more than a single page break, you'll need to use:

{SET Sctn{SECTION} {={SECTIONPAGES}-1}}
or:
{SET Sctn{SECTION} {={SECTIONPAGES}-2}}

depending on whether there is a preceding or succeeding Section on the same page as the continuous Section starts or ends.

Note: The field brace pairs (i.e. '{ }') for the above examples are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. It isn't practical to add the nested fields via any of the standard Word dialogues. The spaces represented in the field constructions are all required.
