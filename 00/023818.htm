Formula in column E, when the week begins with Sunday is this:

=SUMPRODUCT((INT((A$2:A$100-DATE(YEAR(A$2:A$100),1,1)-WEEKDAY(A$2:A$100))/7)+2=C2)*(B$2:B$100))

I have a list of Salesmen.

Column A = static number that doesn't change. It is the 'Target' amount for a Salesmen's monthly goal.

Column B= the Salesmen's current month-to-date sales.

How do I now show the percentage of the 'Target' based on the month-to-date figure? I've literally tried everything from calculated fields to calculated columns to using the 'Show value as....% of' command. None of these options work. I wind up with either 100% across the board or null values.

lz. say:

Hi Joe,

Assuming the following data:

  1. Create your Pivot Table based on $A$1:$C$4
  2. Drop SalesMen in the Row Labels
  3. Click Fields, Items, & Sets in the Excel ribbon then Calculated Field
  4. Name the Calculated Field i.e. MTDvsTARGET
  5. In the Formula box enter: =IF( NOT(ISERROR('MTD Sales'/Target)), 'MTD Sales'/Target, 0)
  6. Click OK
  7. Format this new Value as Percentage.
