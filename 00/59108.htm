Step 1: Go to settings by typing Settings in the search bar next to Start button, and press Enter.

Step 2: Click on System option from the window, and select Default apps from the left side panel of the screen.

Step 3: Under Choose Default apps, click on Music player option, and chose a default app from Choose an App window.

Step 4: Close the window, and check if the issue is resolved.

 

Method 3: If the issue persist, follow the steps below, and check if the issue is resolved.

 

Step 1: Go to settings by typing Settings in the search bar next to Start button, and press Enter.

Step 2: Click on System option from the window, and select Default apps from the left side panel of the screen.

Step 4: Scroll the page down, and click on Choose default apps by file type.

Step 5: Click on the file types individually, and select a program to make it as default from Choose an App window.

Step 6: Close the window, and check if the changes are effective.
