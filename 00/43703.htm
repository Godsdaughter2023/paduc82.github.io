Port Forwarding - Microsoft Community

xb1r hades say:

Port Forwarding

Login Information

*There is none.  This device requires proprietary software to configure.*

Port Forwarding

  1. Open AirPort Utility (located in the Utilities folder in the Applications folder on a computer using Mac OS X, and in Start > All Programs > AirPort on a computer using Windows), select your base station, and then click Continue.
  2. Select the internet button at the top.
  3. Select the NAT tab.
  4. Check the Enable NAT Port Mapping Protocol box.
  5. Click the Configure Port Mappings.
  6. Click Add
  7. Fill out the following lines as needed (Listed below in bolded text)

  8. Click Continue.

  9. Enter a description for the service you’ve just created. Example: Xbox Live.

  10. Click update

 

Service: This line can be left blank

Public UDP port: enter 3074

Public TCP port: enter 3074

Private IP Address: Enter the xbox’s IP address

Private UDP port: enter 3074

Private TCP port: enter 3074

 

Disable Port Forwarding

  1.  Open the AirPort Admin Utility. (It is installed in the Utilities folder in the Applications folder on a Macintosh computer, and in Start > All Programs > AirPort on computers using Windows.)
  2. Select your wireless device.
  3. Choose Base Station.
  4. Manual Setup, or double-click the device icon to open its configuration in a separate window. Enter the password if necessary.
  5. Click the Advanced button, and then click Port Mapping.
  6. Delete any port forwarding rules.
  7. Click Update.
