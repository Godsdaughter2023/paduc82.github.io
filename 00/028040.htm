Let's do a simple test to verify Spotlight is working.

com.microsoft.SyncservicesAgent.plist

Launch the Terminal application found in /Applications/Utilities.

Enter this command and then hit the return key:

sudo mdutil -E "/Users/<your user name>/Documents/Microsoft User Data/Office 2011 Identities"

Replace <your user name> with your own home folder name.

This will force Spotlight to reindex your mail. Give it some time to complete.
