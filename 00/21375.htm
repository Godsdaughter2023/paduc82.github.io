steve rindsberg say:

First, please indicate *exactly* which version, edition and "bitness" of PowerPoint you're using and how recently updates/service packs have been applied.  The way PPT handles links to Excel seems to vary from time to time.  

Meanwhile, what I'd suggest trying is this:

Create the files on your local machine.  Before creating any links, put the XL file in the same folder as the PPT file, after having saved the PPT at least once.

To create the link, select the chart in XL, rightclick and choose Copy.

joeu2004 say:

There is no need to upload an example file.  The explanation is obvious.

The "problem" is with your understanding of the value in column D.  It is not 1.4.  Instead, it is 1.44341801385681.

If you want it to be 1.4, change the formula to:

ROUND(1/(C6*0.866), 1)

Then the values in column F will be as you expect.

Additional notes:

1. There is no point in using SUM around 1/(C6*0.866).

2. Although ROUND in column D works for your example, only you can decide if you would prefer ROUNDDOWN or ROUNDUP.

3. 0.866 looks like an approximation of some calculation.  It might be better to use the calculation instead of the approximation.  Similarly for the values in column E.
