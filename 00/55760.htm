"It doesn't start until I increase the volume to 100%."

That would not come as a surprise. The volume should generally need to be @100%.

Sounds like you have a mic that has had the input volume pushed to the max which, in conjunction with max speaker

volume, is causing feedback.

Or maybe you don't have the speaker volume set high enough in Windows. Go to...

Control Panel > Sound. Click on Speakers > Properties

Open the Levels tab and adjust the Speaker volumes to 100%. Having them at 100% here should provide

enough gain to allow you to use the volume control on desktop (or via keyboard) at a lower level.

If you don't use the mic, set it at 0. If you do, adjust it so you don't get feedback when the speakers are cranked up,

or moderate you speaker volume when using the mic.

j w stuart say:

Take the side cover off and check that all the connections (Power supply to motherboard, CPU cooling fan, Graphics card, Etc.) are not loose and fully inserted, While your at it blow out any dust bunnies from the CPU fan and power supply. See if that helps.

Then run Memtest86+:

This runs from a boot disk or CD and should eliminate or confirm if one or more of your memory

sticks are bad or the SPD values in the BIOS are correct.

Let it run for as long as you can: 2,4,6,8 or more hours (at least 3 full passes), if no errors by then your ram is OK.

http://www.memtest.org/

Memtest86+ Guide/How To's (use the .ISO to create a bootable CD)

http://www.overclockers.com/forums/showthread.php?t=409152

Install Core Temp:

Monitor the temperature of each core of your processor.

Note: For non overclockers using the stock Intel/AMD heatsink and cooling fan you can expect

a temperature range of 35 to 40C at idle and from 60C to 65C max when running Prime95.

http://www.alcpu.com/CoreTemp/

Prime 95:

http://www.mersenne.org/freesoft/

It's a stand alone .exe file contained in a .zip archive.

Just choose to run the "stress test" option for 8 hours or more.

If your PC can pass this test both your memory and CPU

are fine (close the case cover to maintain proper ventilation)
