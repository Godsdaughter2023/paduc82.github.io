hansv mvp say:

If you clear all data in a bound form, you're actually clearing the data in a record, not just undoing changes.

Users can press the Escape key to undo changes in the current field, and press Escape twice to undo all changes in the current record.

If you want to make this easier for them, you could place an Undo command button on the form. The On Click event procedure for such a button could look like this:

 

Private Sub cmdUndo_Click()
    If MsgBox("Are you sure you want to undo all changes to the current record?", _
            vbYesNo + vbQuestion) = vbYes Then
        Me.Undo
    End If
End Sub
