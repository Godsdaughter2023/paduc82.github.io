Hi,

the statement If Cells(row_no, 17) > 1 And Cells(row_no, 17) < -1 Then will never become True. There is no number that is both greater than 1 AND less than minus 1.

I guess you mean: If Cells(row_no, 17) > 1 Or Cells(row_no, 17) < -1 Then

 

It's not necessary in a For..Next loop to increment the loop count (here: row_no). The loop will do it automatically.

Your code should now read:

Sub goal_seek()
Dim row_no As Integer
