srinivasu guntimadugu say:

What type of email account is used in Outlook? (POP or IMAP or Exchange)

Sent Items folder is the default store location for all outgoing items in Outlook. Sent Messages folder will never appear in Outlook by default until it’s created manually or if it’s available in the mail server depending upon the account type.

If a POP account is used in Outlook then you may move all items from Sent messages folder to the Sent Items folder and then delete the Sent messages folder. Ensure no rule is created in Outlook to move emails to the Sent messages folder. Right-click on the Sent messages folder and select delete to remove it, delete option for Sent Items folder will not be available since it’s a default folder.

Hi Maxime,

Office 2016 for Mac does not have an AutoSave feature, so we suggest always saving your file after you have made some progress.

Regarding AutoRecover, it can help Word recover a file that you were editing after a power failure or other problem that interrupts Word while you are editing the file.

However, using AutoRecover does not replace saving your files at regular intervals or when you are done editing the file.

As a result, even if you have AutoRecover enabled, you should save a file frequently while you are working on it to avoid losing data due to an unexpected power failure or some other problem.

For your reference:

Save a file in Word 2016 for Mac.

Regarding your document, to make sure you have checked the correct folder, the AutoRecovery folder is located under ~/Library/Containers/com.microsoft.Word/Data/Library/Preferences/AutoRecovery/.

If the folder is empty, you can try opening Word again and see if it once again prompts you to reopen and recover your files.

If it doesn't, then unfortunately the file may be permanently lost.

I would also like to explain that based on your description, the issue seems to be Word freezing each time you want to save your file, and not AutoRecover failure. AutoRecover managed to recover your file the first time, but when it comes to save the file again, Word freezes. This is when you force quit the application, which may result in the loss of the recovery file.

If you wish, we would like to try and troubleshoot the freezing issue. Are you still having freezes when saving a new document?

We understand your frustration regarding data loss and would like to help ensure this doesn't happen again.
