The MEMORY_MANAGEMENT bug check has a value of 0x0000001A. This indicates that
a severe memory management error occurred.

Unfortunately memory tests do not catch everything such as mismatched memory, when slower
memory is placed in front of faster memory, and other issues.

You can try Powering Off all power and removing and reseating the memory. One or two stick(s)
at a time if possible to see if the system bluescreens and if so on which sticks.

Other possibilities are BIOS and antivirus/antispyware/security programs and even
motherboard problems.

Drivers are possible however that would be unusual - maybe chipset drivers.

Look in the Event Viewer to see if anything is reported about those.
