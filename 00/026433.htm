Two Running Page Numbering Systems

How would I go about having in one footer page numbering throughout the document of both linked and unlinked numbering? I can do one or the other but I do not know if I can set them to count independently of one another.
john mcghie say:
AFAIK there is no option available in Word 2011. However it may be possible with a VBA code.

This is a built-in function of all versions of Word since Word 5.1.

You need to insert the field codes you want manually, instead of relying on the Insert>Page Numbers function.

There are four fields involved.  {PAGE} contains the number of the current page in the document.  {NUMPAGES} contains the number of pages in the document.  {SECTIONPAGES} contains the number of pages in the current section. And {= } enables us to calculate page numbers.

So on the Left we have "Page {PAGE} of {NUMPAGES}.

On the right, we need "Page {={Page}-{REF}} of {SECTIONPAGES}

To explain: that line computes the current page number minus the page number of a bookmark at the start of the section.  SECTIONPAGES returns the number of pages in the section.

If you code it correctly, it will look like this:

Section Page { = { PAGE} - {PAGEREF Chapt03 } + 1} of { SECTIONPAGES }


Instead of using Insert>Page Number, you use Insert>Field.  Insert the fields shown there.
{ PAGEREF Chapt03 } is Insert>Bookmark and choose the bookmark which begins as the FIRST page of the section.  In this case, Chapt03 is a bookmark that contains the whole of Chapter 3.  That bookmark is used for the individual Tables of Content, one per chapter, in the book I copied the example from.


Let me know if you need any more detail.


Hope this helps

john korchok say:

This behavior is by design. When a previous user has set up tracked changes, it's because the review process is important, at least to that person. So Word ensures that you can't accidentally bypass this feature. If the document has tracked changes, and that document is part of a review process that doesn't include you, accepting all changes sabotages that group's efforts to collaborate as they prefer. The best practice is to check with the sender to verify that the version you receive is the final before accepting all changes.

You can create a macro that turns off change tracking on your computer:

Then you can assign a keyboard shortcut to run it and/or you can add it to the Quick Access Toolbar to make it easy to find. You could even call it Sub AutoOpen() instead of Sub NoTracking(), then it will automatically run for every document you open.

You can set a background colour for a Word document, but it is only visible in Web Layout. The background will appear white in Print Layout. Use the View tab's Document Views group, Web Layout command to see it one has been set.
