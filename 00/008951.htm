bcs1010 say:

Customize right click menus Access 2010

cristin yan say:

Hello,

 

I have found several examples that you can refert to page "http://msdn.microsoft.com/en-us/library/ff194247.aspx".

 

The 3rd part VBA code on the above page is:

The following example creates a shortcut menu named cmdReportRightClick that contains commands that are useful to use with a report. This example shows how to change the Caption property of each control as they're added to the shortcut menu.

 

Sub CreateReportShortcutMenu()
    Dim cmbRightClick As Office.CommandBar
    Dim cmbControl As Office.CommandBarControl

   ' Create the shortcut menu.
    Set cmbRightClick = CommandBars.Add("cmdReportRightClick", msoBarPopup, False, True)

    With cmbRightClick
        
        ' Add the Print command.
        Set cmbControl = .Controls.Add(msoControlButton, 2521, , , True)
        ' Change the caption displayed for the control.
        cmbControl.Caption = "Quick Print"
        
        ' Add the Print command.
        Set cmbControl = .Controls.Add(msoControlButton, 15948, , , True)
        ' Change the caption displayed for the control.
        cmbControl.Caption = "Select Pages"
        
        ' Add the Page Setup... command.
        Set cmbControl = .Controls.Add(msoControlButton, 247, , , True)
        ' Change the caption displayed for the control.
        cmbControl.Caption = "Page Setup"
        
        ' Add the Mail Recipient (as Attachment)... command.
        Set cmbControl = .Controls.Add(msoControlButton, 2188, , , True)
        ' Start a new group.
        cmbControl.BeginGroup = True
        ' Change the caption displayed for the control.
        cmbControl.Caption = "Email Report as an Attachment"
        
        ' Add the PDF or XPS command.
        Set cmbControl = .Controls.Add(msoControlButton, 12499, , , True)
        ' Change the caption displayed for the control.
        cmbControl.Caption = "Save as PDF/XPS"
        
        ' Add the Close command.
        Set cmbControl = .Controls.Add(msoControlButton, 923, , , True)
        ' Start a new group.
        cmbControl.BeginGroup = True
        ' Change the caption displayed for the control.
        cmbControl.Caption = "Close Report"
    End With
    
    Set cmbControl = Nothing
    Set cmbRightClick = Nothing
End Sub

 

Hope that this can provide some help for you.

 

Cheers,

Cristin Yan

ossiemac say:

The following solution is not attached to a button. It creates a Hyperlink in a cell so that you just click the cell which is basically the same as clicking a button.

I am assuming the following:

  • You want to open an email in Office Outlook
  • The following cells contain the email addresses, subject and body of the email.

To:               E9 & F1

cc:                F2

Subject:       F3

Body:           F4

Enter the following formula in a cell:

   =HYPERLINK("mailto:"&E9&";"&F1&"?cc="&F2&"&subject="&F3&"&body="&F4,"Send Email")

Note carefully where the double quotes open and close in the formula to concatenate the various parts of the string inside the brackets. In particular note &subject and &body. The leading ampersands are part of the field identifier and not used in concatenation of the string.

The last parameter is for the text to be displayed in the cell. If you leave out Send Email (and the preceding comma) then you will see the entire created string in the cell with all of the email addresses and Subject and Body text.

You can use Workbook_SheetChange in the code module ThisWorkbook to have one routine instead of one routine in each sheet.

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
  'Which sheet?
  Select Case Sh.Name
    Case "Sheet1", "Sheet2"
      'etc.
  End Select
End Sub

And if the code does not have to be permanent, you can write one file (maybe AddIn) and take over that events for all files and all worksheets.

Dim WithEvents App As Application

Private Sub App_SheetChange(ByVal Sh As Object, ByVal Target As Range)
  'Which file?

      Select Case Sh.Name
