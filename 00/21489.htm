In order to remove users in bulk I would request you to please follow the below steps

  1. Do an Export User to get the email aliases of the user, by Clicking on the gear icon on top left > Network Admin > Export Users > All Users then open the export file and copy the email address of the users that are not present in the AD but showing up in your Yammer network

 

  1. Again go to Network Admin > Data Retention > select (Hard delete)

 

  1. Now, You'll need to create a CSV (comma separated value) file with the user account information.

 

  1. Your CSV file should be formatted as a table and must include a header, or first line, that defines the fields in your table.

Note: If you use Office 365 or Yammer's Single Sign On Integration, your users' passwords are no longer managed by Yammer. Please do not include passwords in your CSV.

Example:

 

Example:

Action

Email Address

New Email

Update

tgraves@contoso.com

 

In Microsoft Excel, you can save a spreadsheet as a CSV file by going to File > Save As and choosing Comma Separated Values (.csv) from the Format dropdown.

 

  1. Now Login as a verified admin in Yammer

 

  1. Click on gear icon on bottom left

 

  1. Go to Network Admin and go to Bulk Update Users

 

  1. Upload the CSV file and click on Bulk Update Button
