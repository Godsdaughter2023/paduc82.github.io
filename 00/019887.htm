dirk goldgar say:

If you are not building your own user interface, as most developers do, but instead are relying on the navigation pane and other features that Access makes available by default, then it is possible to put the forms and other objects into any sequence you want, and have it stick.  To do that, right-click on the nav pane caption or one of its subordinate group captions, and choose Sort By -> Remove Automatic Sorts from the popup menu.  Then you can drag the objects in the nav pane into the order you want.

If you're using POP, it's NOT webmail.  Webmail by definition means you're using a web browser to access the mailbox directly.

If you are seeing the messages in the Unread Mail search folder, then Outlook IS receiving the messages.  Unread Mail is not a real folder.  It is a search context.  The "In Folder" field will tell you what real folder contains the messages.  If it says Inbox, then they must be in the Inbox and if you can't see them there, it's because you have a view setting that is hiding them.  Reset your view to its default values.  Make sure you're using a view like Compact or Single.  Make sure you have no filters enabled by examining the view settings.  Finally, make sure the date is set correctly in Windows.
