When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate.

Reset takes the computer back to its original state and removes everything not on the system when sold. It is probably the safest option.

2nd is to create a new account on the PC. From your description it would appear that you have just changed the login for your fathers account from his Microsoft Account to yours. Log in as that user and you should get a completely new desktop with default settings. If there are any files in the other account you will need to transfer them to the required directories in your own new folders. Then use the Accounts tool to delete your fathers account. Finally if you want the features offered by logging in using the Microsoft Account then again with the Account tool switch your local account to use a Microsoft Account. Until you do you will operate the account in classic mode, the same as it does on Windows XP/Vista/7
