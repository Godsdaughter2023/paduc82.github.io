A General Starting point when Word does something unexpected

As a general rule, when something seems to be going wrong, or at least not the way you want it to go, in Word “menu’ go to Tools and then Options, in Word “ribbon” go to File then Options, and study all of the various options, both turned on and off.

When I was teaching Word and other Microsoft Office programs at a private business college, I had three rules:

  2. When in doubt, right-click on something
  3. If you’re wondering how to set up a special kind of material in a document, consider whether Word tables are suitable. The simplest fix is shutting down your computer normally and restarting it. Often subtle “corruption” happens to create unusual effects. Shutting down and restarting the computer resets all of the loaded software, often clearing up strange problems.

Windows 10 and AMD Chipsets - Thermal Considerations - Microsoft Community

model299 say:

Windows 10 and AMD Chipsets - Thermal Considerations

Hello,

I'm running Windows 10 on an MSI 970A-G46 Mainboard with an Athlon FX 6-Core processor. It has 16GB of RAM, runs on a RAID 0 Array with 4TB using the ATI RAID system and is configured for UEFI Windows 8 boot into Windows 10. 

I had problems initially when I was using Crossfire X with two DirectX 10 Radeon HD 6950 cards. I was able to resolve that issue by installing an MSI AMD R9 380 4GB Gaming video card, which seemed to improve compatibility, but not this other issue.

The biggest problem here is the support for the AMD chipset, so I also installed the latest AMD Video drivers as well as the AMD chipset drivers for the 9xx series chipsets.

I've found that most of the problems with this chipset are not related to the drivers or the Windows 10 software. The issue is that Windows 10 places a higher demand on the system's northbridge, which is an AMD970 and it uses this in combination with the SB950 southbridge.

The problems with this chipset are thermal. The higher demands created by the Windows 10 operating system are not allowing adequate cooling of the northbridge. If there were a way to use the DirectX 12 system without so much demand on the northbridge the system may be more stable. I am going to install an additional heatsink and fan to see if I can keep the system temperatures lower. 

This may solve the problem on AMD chipsets for a lot of gamers. However, I have to ask, is there better support under development for AMD chipsets upstream? This issue would be easier to solve if there were lower overhead on the local bus system through the northbridge. Some hardware just wasn't designed to meet these thermal demands.
