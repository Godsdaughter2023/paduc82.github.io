a. user say:

If you are using an actual UserForm, with a text box in which the product code is entered and a command button, when clicked, that adds the textbox value to a worksheet then...

Enter theVBA code in the command button click event.
It should look something like...
'---
Private Sub CommandButton1_Click()
  Dim Num As Long
  Dim lngRow As Long
  Dim lngEntry As Long
 
   lngEntry = Val(Me.TextBox1.Value)
  'Assumes product code are 6 digits or more
  If lngEntry < 100000 Then
     MsgBox "Please check your entry"
     Exit Sub
  End If
  'Assumes product codes are in column A
  'Assumes worksheet is named Sheet1
  On Error Resume Next
  Num = Application.WorksheetFunction.Match(lngEntry, _
        ThisWorkbook.Worksheets("Sheet1").Range("A2:A1000"), 0)
  On Error GoTo 0
  If Num > 0 Then
    MsgBox "Duplicate"
    Exit Sub
  Else
  'Code to add the program number to the worksheet...
   With ThisWorkbook.Worksheets("Sheet1")
     lngRow = .Cells(.Rows.Count, 1).End(xlUp).Row + 1
    .Cells(lngRow, 1).Value = lngEntry
  End With
  End If
End Sub
'---

Excell sheet names automatically update - Microsoft Community

ladiebeth say:

Excell sheet names automatically update

Is there a way to set a formula in the sheet names of excell spreadsheets, so that if I fill in the date on the first one it will automatically fill in the next date on the next 15 or so?
jeeped say:

The modifications you need are very minor. As previously mentioned, after the macro removes all of the extra sheets but one, it renames the first sheet according to today's date. If you are willing to rename the first sheet yourself, simple delete this line,

Sheets(1).Name = Format(Now(), "dd-mmm-yy")

... or comment it by places a single quote (') at the beginning. Since the macro depends upon a data as the name of the first sheet, it now becomes imperative that the first sheet be renamed in a date format before running the macro.

Next you need to make a minor change to the amount of time added to each new sheet so change,

Sheets(i).Name = Format(DateAdd("m", 1, CDate(Sheets(i - 1).Name)), "dd-mmm-yy")

... to,

Sheets(i).Name = Format(DateAdd("ww", 2, CDate(Sheets(i - 1).Name)), "dd-mmm-yy")

... and everything should work according to plan.

Good luck and thanks for the feedback!

jeeped say:

I would use a macro here rather than VBA code on an auto- trigger. Try this,

Sub AutoSheetName()

' AutoSheetName Macro

Dim n As Integer
Dim i As Integer

n = 15   'number of sheets including original

While Worksheets.Count > 1
    Application.DisplayAlerts = False
    Worksheets(2).Delete
    Application.DisplayAlerts = True
Wend

Sheets(1).Name = Format(Now(), "dd-mmm-yy")

For i = 2 To n
   Sheets.Add After:=Sheets(Sheets.Count)
   Sheets(i).Name = Format(DateAdd("m", 1, CDate(Sheets(i - 1).Name)), "dd-mmm-yy")
Next i

End Sub

The sheets are named in sequential months beginning with today. If you manually name the firs, that line can be commented out..

Warning: this macro deletes all worksheets but the first one in order to gain a finite reference point.

Hyperlink from PowerPoint to a specific location in another file

Suppose that you want to link to a specific location in a Word or Excel file? You can do this as well.

Note: In Word, to create a bookmark, you choose Insert (tab)> Bookmark. Enter the bookmark name (no spaces; it can’t start with a number), and click Add. In Excel, select a cell or range of cells. In the Name box (on the left, above the top-left cell), enter a name and press Enter.

Follow these steps:

1.   Create the bookmark or named range and remember the name.

2.   Select the object on the slide that you want to use for the hyperlink.

3.   Choose Insert > Hyperlink (2007: Insert tab> Links group> Hyperlink) or press Ctrl-K.

4.   In the Insert Hyperlink dialog box, choose Existing File or Web Page from the Link To bar at the left.

5. Click the Bookmark button. Well, this is what you’re supposed to do, but you’ll get an error message! Here you see it after I’ve clicked the Show Help button.

an error message

6. Click OK to get out of that! You’re back in the Insert Hyperlink dialog box and you can see your document in the Address text box. Here’s the trick: at the end of the address enter # (the pound sign) and the name of the bookmark or named range, as you see here.

in the Insert Hyperlink dialog box
