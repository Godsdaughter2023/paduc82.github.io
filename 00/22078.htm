I do not have an administrator. this is a home computer and I am the sole user. How in single step-by-step language do I open the files in word 2007 without disrupting files created in 2007? 
jay freedman say:
First go to the web page at http://support.microsoft.com/kb/938810.

Scroll down to the heading "Method 1: Use a downloadable update to automatically re-enable all file types".

Under Step 1, after the paragraph that starts "To re-enable Word files types", click the link to http://download.microsoft.com/download/2/8/B/28B644CE-9A30-4D99-AFEF-830A4FB5B839/UnblockWord.reg. You should be asked whether to save or run the file (whether you see this depends on your browser's settings). If you're given the choice, run it and answer yes/OK to the prompt about changing the registry. If you aren't given that choice, then save the file to your desktop or some other folder you know about.

If you ran the file, you should now be able to open old Word documents in Word 2007. If you saved the file, go to it and double-click it, and answer yes/OK to the prompt about changing the registry.

None of this will have any effect on documents created in Word 2007.

graham mayor say:
My understanding of that page is that it is for Word 2003. For Word 2007, open the reg file in Notepad and change the 11.0 in the line below to 12.0 then run the registry patch again.

[HKEY_CURRENT_USER\Software\Microsoft\Office\11.0\Word\Security\FileOpenBlock]
"FilesBeforeVersion"=dword:00000000

Then ediit the line as follows (note the - sign at the start)

[-HKEY_CURRENT_USER\Software\Microsoft\Office\11.0\Word\Security\FileOpenBlock]

and run the patch again to remove the earlier entry.
