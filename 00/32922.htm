gerry c j cornell say:

Maggie

 

So far as I am aware you cannot automatically delete messages in your Inbox. The quickest way to clear the contents of the inbox is to hold down the Shift key, place the cursor on the first message then click the last message. This highlights all messages. You release the Shift key and use the Del key to place the messages in the Deleted items folder or the Shift+Del key to bypass the Deleted Items folder.

 

You can select individual messages by holding down the Ctrl key and clicking individual messages. This causes the selected messages to be highlighted. If you then release the Ctrl key and press the Del key you delete the selected messages, leaving the other messages in the folder.

 

You can automatically empty the Deleted items folder. Select Files, Options, Mail, Advanced, Maintenance and check the box before "Empty messages from the deleted items folder on exit. Close the window and click on Apply to complete the change.

 

Most users wish to keep incoming messages or some of them. This is likely to be why Windows Live Mail does have an automatic delete function to work with the Inbox.

 

You can use Message Rules to divert incoming messages from the Inbox to the Deleted Items or other user created Local folder. The mechanics can be quite complicated.
