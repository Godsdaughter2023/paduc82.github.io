jay freedman say:

If you're using the Insert > Footer button (not the References > Insert Footnote button -- footers and footnotes are completely different) and the Insert > Page Number button on the ribbon, insert the footer first. Put the cursor to the right of the footer, and press the Tab key if necessary to get to the position where you want the page number. Then click the Page Number button and click Current Position before choosing the number.  DO NOT choose Bottom of Page, because that will wipe out the existing footer.

Instead of using the Page Number button at all, you can insert a page number by pressing Alt+Shift+P (all three keys simultaneously).

ossiemac say:

Alternative code for the Save sub that you might find easier, particularly if you have a lot of columns. 

The code directly references the column Id's rather than trying to count the Offsets.

Firstly it identifies the row of the found cell and assigns it to the variable "r" and then you can use the row reference in conjunction with the column Id to reference the destination.

A similar method could be used when the record is found for populating the Userform controls.

See my comment in the code that the Worksheet should be referenced in conjunction with this method. (With range variables, the workbook name, sheet name and the actual range is saved within the variable and therefore the worksheet does not need to be referenced separately).

Private Sub cmdSave_Click()
    'This sub copies the Userform control values to the worksheet
   
    Dim wsData As Worksheet 'Worksheet with Database
    Dim rngEmpNos As Range  'Range containing employee numbers
    Dim rngToFind As Range  'The Employee Number to find
    Dim r As Long           'Row number of found cell (rngToFind)
   
    Set wsData = Worksheets("Data")
   
    With wsData
        'Assign the Employee Numbers to a range variable
        Set rngEmpNos = .Range(.Cells(2, "A"), .Cells(.Rows.Count, "A").End(xlUp))
    End With
   
    Set rngToFind = rngEmpId(rngEmpNos, Me.cboEmpNo)    'Call UDF to find the record
                       
    If Not rngToFind Is Nothing Then
        'If Employee found then Save the record
        r = rngToFind.Row   'Assign the row number to a variable to be used in the cell references
       
        'Note that the method of referencing specific cells by row and column Id's
        'then the With statement to identify the worksheet should be used
        With wsData
            .Cells(r, "B") = Me.txtFamName
            .Cells(r, "C") = Me.txtGivName
            .Cells(r, "D") = Me.txtLocat
            .Cells(r, "E") = Me.txtPosition
            'Following line will convert string to date based on regional locale
            .Cells(r, "F") = CDate(Me.txtComDate)
        End With
           
        Call cmdClrForm_Click   'Note this calls the click Event for the Clear form Command Button
    End If
   
End Sub
