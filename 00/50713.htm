How to uninstall the updates?
1- In Cortana write> "Updates" and select the entry "View installed updates".

2- The "Installed Updates" window will open, where you can see each and every installed update

3- Here, the next thing you should do is look for the latest installed updates and click on each of these with the right button. In the context menu we will see the option "Uninstall".

4- Click on it and, after a few seconds, the update will be uninstalled from the system.

5- Restart the computer and, when restarting, the problems should have disappeared.

When detecting that the patch is not installed, Windows will try to download and install it again, so we must pay attention to prevent this from happening, at least until Microsoft solves the problems, something that may take a few days.

To prevent Windows from installing updates again it is recommended to disable automatic updates.
