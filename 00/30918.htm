Setting up twp separate email accounts - Microsoft Community

jim2491 say:

Setting up twp separate email accounts

I am upgrading from XP to Windows 7.  To separate the email accounts of myself and my wife in XP, I set her up on Outlook Express and me on Microsoft outlook.  I have just learned that Windows 7 does not support Outlook Express.  How then can I set up two separate email accounts with two separate icons?
tryx3 say:

You can each be set up as separate users on the Win7 PC.  You both run applications independently then.  She can use her email address in her "user account" using Outlook and you can use your email address in your "user account" using Outlook.  The user account regime in Win7 is quite different to that you've seen in XP.  You can switch between users quite easily ["Start menu", "Shutdown menu", "Switch users" / "Logoff" then "Login" to the other user account - Switching users is very convenient as it means that everything you are doing remains loaded so you can switch back in a second or two]

You have to create a user when you install Win7.  That user account automatically has administrator privileges so name the user at installation OurWin7Admin1 or somesuch.  When you have completed Windows & application setting up you then create a user account for you and a user account for her, both of which should be set as standard users rather than administrators as a system protection measure. 

I also recommend [to everyone] creating a OurWin7Admin2 administrator account as sometimes account profiles can get corrupted and if its an admin account that corrupted then you have a big problem. 

When users do something that needs higher level admin approval [anything that changes the system setup, anything that might damage the system] then a dialog pops up demanding the admin password.  You put it in then the operation can proceed.  If you routinely use one admin password & she the other one then you would, between you, be testing the pair of admin accounts so if one was ever corrupted the chances are that the other would still be intact so you could get together & sort out the admin accounts.  If either of you ever find your user account to be corrupted then either admin acccount could be used to sort that out.

I also find it useful to set different desktop backgrounds for each account so I cannot forget which I'm in at any time.  I use a horribly bright colour as a background for the admin accounts just as a reminder that I'm workign at the higher privilege level.

To create accounts, you need to be logged in as an Administrator & you go to Control panel, User accounts.  The procedure is easy.  After creating any account you go to the same place to create a "Password reset disk" for each account which can be used in the event of having to rescue a corrupted account.  These disks are used if system corruption prevents you logging in to an admin account - they are as important as a piece of paper with the account passwords written on it so do need to be put away safely.

Of course, if you don't want to do this you could look for a different email application to replace Outlook express but none of the free applications have the power of Outlook so one of you would be missing out.

Please provide a copy of your System Information file. To access your System Information file select the keyboard shortcut Win+R, type msinfo32 and click OK . Place the cursor on System Summary. Select File, Export and give the file a name noting where it is located. Click Save. for the file to be fully populated before exporting a copy. Please upload the file to your OneDrive, share with everyone and post a link here. Reports in normal mode preferred. Please say if the report has been obtained in safe mode. Place the cursor on System, select Action from the Menu and Save All Events as (the default evtx file type) and give the file a name. Do not provide filtered files.
