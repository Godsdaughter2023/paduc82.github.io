ralf_g say:

Try removing the phone from Devices and Printers so that Windows bluetooth can discover it again. Remove the PC entry from the phone's preferred bluetooth devices list as well. Set both into bluetooth discovery mode again. Either one or both devices should be asking you for the pairing code automatically when you tell either of them to connect to the other. That's after the devices have newly discovered each other.  Can't be too specific in the how-to's because the instructions for Brand-X phone might not be the same as for Brand-Y.  

The pairing code for phones is often preset to 0000 or 1234 - the owners manual should tell you what it is for yours. Windows can generate a random pairing code for the computer automatically or you can enter one manually but the phone may be limited as to the number of digits you can enter for the code. On the ones I've paired the phone's code cannot be changed from the 4 digit factory default.  It might be easiest to initiate the pairing from the PC rather than doing it from the phone. Right-click Add a Device on the Bluetooth icon in the system tray. When the phone appears in the bluetooth device list click on it and follow the prompts. When the prompt for the pairing code appears enter the default code for the phone in place of any Windows generated code (to keep it simple). Enter the same pairing code in the phone after it detects the incoming connection from the computer. That should do it.

The individueal bluetooth functions of your phone might require being connected separately when accessing it from Windows.. ie. you might have to disconnect media streaming before you can use the phone as a modem or internet connection, or browse files in its storage memory. Right click "Control" on the phone's icon in Devices and Printers to access its bluetooth functions.

After you get a successful pairing you can set options to allow automatic connection between the two and do any configuration for the various bluetooth features that might be available for the phone (ie. modem, COM ports, streaming audio/video etc.).   Windows might also add bluetooth COM ports and modem to Device Manager. You can increase the default port speeds there as well.
