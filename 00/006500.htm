jean-yves tfelt say:

Hi,

 

In your test : If Range("C3") = "Reactive"

Range("C3") is a single range of cells. Excel will translate that into :

If Range("C3").value = "Reactive"

 

Then in : If Range("C3:C27") = "Reactive"

Range("C3:C27") become an array of cells that you cannot compare to a single string.
