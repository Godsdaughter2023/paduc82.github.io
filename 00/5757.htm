Hi Petra,

Based on your description, you would like to find and recover lost Publisher file. I would like to suggest you check the following:
    1. Open Run in the startup bar, type in %temp%. Find a recent file with the pub prefix, change the .tmp extension to .pub, try opening the file.
    2. If you have enabled autosave in Publisher, you can go to this location and find the file: C:\Users\CurrentUser\AppData\Roaming\Microsoft\Publisher.

If this is not your scenario, please provide the information below, so we can better understand your situation and help you.
