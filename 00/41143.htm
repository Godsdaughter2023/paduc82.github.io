    7srj say:

    Hey all, I was having this problem too and finally figured it out. This fix could apply to anyone with a Gigabyte Thunderbolt 3 certified motherboard, it will solve "Billboard Device" errors including "error 10":

    On the Gigabyte website we are instructed to download and install a firmware update to upgrade to Thunderbolt 3 capability (http://www.gigabyte.com/support-downloads/Utility.aspx#de). The instructions say:

    "Please follow the steps instructed after executing Thunderbolt FW Update Tool, remove the system power cord for at least 30 seconds after FW update. Thunderbolt function will be available after system powers on."

    The problem is the wording in these instructions, what it SHOULD say is this:

    1) Download the zip file and decompress it.
    2) Run the setup file and install the program (this will take 2-3 minutes and will have several steps)

    Note: it appears a lot of people (including me) stopped here and assumed the update was complete, however, this is not the case!

    3) After the program has finished installing go to C:\program files (x86)\GIGABYTE\flashTBT and run the file "flashTBT_100.exe". This program will flash the BIOS and upgrade it to Thunderbolt 3 capability.

    So yeah, that's it! Everything is working great now. It's a really simple fix, but until Gigabyte updates the wording in their instructions I suspect people are going to continue to have this problem.
