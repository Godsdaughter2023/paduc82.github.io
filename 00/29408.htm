considerate_guy say:

Quick question, have you ever been able to connect this laptop to a monitor via HDMI?  Obviously there are several things in succession that need to be working for a signal to appear on your TV.


1) Laptop needs to be sending video to the HDMI port.  When you are on your desktop, right-click and choose Preferences.  Does it show one display or two?  Have you tried just switching to the TV as the primary monitor without doing the duplication?  Or perhaps extending the desktop to the second monitor?  Does that help make it work?

2) TV needs to be set to the correct HDMI input.  I assume you have already tried this, but sometimes the sequence matters.  Try connecting the HDMI cable while the TV is off, have the laptop already on, then turn the TV on and select the HDMI input.  Any different?


You didn't mention the specific model of Toshiba laptop, but you might want to also ask on the Toshiba forums; if you haven't already.

Method 1: Run SFC Scan

SFC /scannow is a super-useful command you can use in any Windows version. When the SFC (System File Checker) command is used with the /scannow switch, the tool will scan all of the important Windows files on your computer and replace them as necessary.

Method 2:  I suggest that you check the status of the Event log service. This service should be started and set to ‘Automatic’.

c. Right click on service and click on properties.
d. From the dropdown, select the startup type to ‘Automatic’
 
Also try to run Disk Defragment on your computer.
 
a. Open Disk Defragmenter by clicking the Start button, clicking All Programs, clicking Accessories, clicking System Tools, and then clicking Disk Defragmenter.  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
