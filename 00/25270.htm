Excel VBA - Code to determine if there is an image in the activecell - Microsoft Community

headlydotcom say:

Excel VBA - Code to determine if there is an image in the activecell

Is there any way to determine if there is a picture in the activecell? TIA
mike h.. say:

Hi,

Images float over cells and aren't contained within then but we can check if the top left and bottom right of an image are over the active cell like this. Note I used a message box in the code but this isn't necessary you can do what you want with the Boolean variable HasImage  which will be TRUE if the image is over the cell.

Sub ImageThere()
Dim HasImage As Boolean, shp As Shape
HasImage = False
For Each shp In ActiveSheet.Shapes
If shp.TopLeftCell.Address = ActiveCell.Address _
    And shp.BottomRightCell.Address = ActiveCell.Address Then
    HasImage = True
    GoTo getmeout
End If
Next
getmeout:
If HasImage Then
    MsgBox ActiveCell.Address & " has an image"
Else
    MsgBox ActiveCell.Address & " doesn't gave an image"
End If
End Sub

    daniel g. grau say:
    I just realized that you mentioned Word 2010. This is the Mac forum, so perhaps you'll be able to find an answer in the Office 2010 forum.

    I would just like to clarify that Language and Spell-Check/Do-Not-Spell-Check are not a setting or preference, but an attribute of the text, just like boldface or font color. Some users think it's a setting and that "enabling" the proper language is enough, when in fact it should be applied to the text just like any other character attribute. Thus, unless they've selected all content and applied those two attributes to it, spell-checking may ignore some portions.
