A Copy&Paste did not exist for userforms at runtime, but you can add controls and customize them as needed. Run the code below in an empty userform.

 

And if you want to add button's that should execute some code you must add an event handler.

 

Andreas.

 

Dim WithEvents MyButton As MSForms.CommandButton

Private Sub MyButton_Click()
  MsgBox "Hello World"
End Sub

Private Sub UserForm_Initialize()
  Dim C As Control
  Set C = Me.Controls.Add("Forms.CommandButton.1")
  C.Top = 5
  C.Left = 5
  C.Caption = "Hello World"
  Set MyButton = C
End Sub

andreas killer say:
There are 2 problems within your code:

a.) Set frm(i) = Controls.Add("Forms.Frame.1", "frm1")

You must parameterize the name for the frame, because you can not add 2 or more frames with the same name. But I guess that you store the controls in a global array? If so you can remove the name part and let Excel assign the name. Same is with the option button.

b.)   With opt16(i)            .Top = 6 + (i - 1) * 48

You have added the option button to the frame correctly. But the TOP property of the option button now refers to the frame, not to the whole form. So the option button is added, but you moved it out of the screen. :-) Try ".Top = 6 " only.

But makes you FOR loop sense? If I assume that UBound(byte_index) is 10, your codes adds 10 Frames with just one option button in each frame.
