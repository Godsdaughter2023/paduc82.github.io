harvey a. say:

Hi all,

i'm currently developing a VSTO addin fo powerpoint(2010).I need a thumbnail preview of the slides in custom pane or in a xaml usercontrol. i'm now opening a presenatation in background and gets the list of slides and showing in a checkboxlist like(slide1,slide2...). i need to make that it into a thumbail view of slides becoz it would be more user friendly. can anyone help me please.

thanks in advance.

Assuming on your userform you have an Image control named image1. 

This is a 2 step process, first you need to create a temp folder of the slide images

ActivePresentation.SaveAs "TempFolder", ppSaveAsGIF

The above code will create a new subfolder in the same folder that the file resides with a gif picture of all the slides named Slide1,Slide2,....  

  Note : Slide naming is based on slide numebrs so if your slides start from 2 then the naming of those pictures will start from 2, so you need to make sure you remove all the gifs from this subfolder because if you used it for a presenttaion that has slides starting from 0 and the second time you run it for a presentation that slides start from 1 the slide 0 from previous run won't be overwritten.

Second step is to load  those gif pictures (I simply loaded slide1 picture , but you can  add some code for selecting any of those pictures based on some logic

Image1.Picture = LoadPicture(ActivePresentation.Path & "\TempFolder\" & "Slide1.gif")
