Display beats per minute in library in Windows Media Player

We manually record the beats-per-minute for our rock&roll songs on Windows Media Player using Advanced Tag Editor.  We've used the Choose Columns function to personalise the library view with all the other required fields. 

How can we display the beats-per-minute column for songs in the library screen?  We need to sort tracks from slowes to fastest for dancing. 

Thanks for any advice,
Wren19
brian-- say: Hi Wren19,


Unfortunately there isn’t a way to do this in Windows Media Player 11. You may be able to find a third party application that will give you this feature.

I can however offer you an option that will give you a very similar result. You can create a custom column and assign values to each track under that column, like adding the beats per minute (although it won’t actually say beats per minute). To do this, do the following: 

1.       Open Windows Media Player 11

2.       Go to your Library

3.       Right-click the empty column between “Artist” and “Title”, and click Choose Columns…

4.       Scroll down until you get to Custom 1 and check the box next to it, then click OK

5.       Now there will be a new column entitled “Custom 1”

6.       Right-click on a track, but do it under where the column for Custom 1 is, then click Edit

7.       When the box under Custom 1 appears enter a value, in your case you can put what the beats per minute would be, then click somewhere else or press Enter

8.       After doing this to all of your songs you will be able to sort them from higher to lower, or lower to higher values.  

I know this isn’t exactly the option that you were looking for, but I hope that you find it suitable in sorting your tracks by beats per minute. If you have any further questions, please don’t hesitate to ask.
