jay freedman say:
The cell may contain many paragraphs, where the last one ends with a cell marker and each one except the last ends with a paragraph mark. When you use the Len() function, the cell marker counts as 2 characters but the paragraph mark only counts as 1 character. If you use a Range object and its MoveEnd method, moving by 1 character works in both cases to exclude the unwanted character. This will work better:

For Each thisPara In thisCell.Range.Paragraphs
    Set thisRg = thisPara.Range
    thisRg.MoveEnd wdCharacter, -1
    thisRg.Text = Trim(thisRg.Text)
Next

 thisLineText = thisPara.Range.Text
