Hi Paul,

Thank you for updating us. It must be unfortunate that you are still facing the same issue. Let me help you.

If you are facing troubles using Windows Photo Gallery, I suggest you to perform a Windows Essential Repair and check if it helps.

Method 1: Repair Windows Essential Repair

  1. Press Windows + x key to open a task list from the bottom left corner of the screen.

  2. Select Control Panel.

  3. Select Programs and Features.

  4. Right click on Windows Essentials.

To fix the program, select Repair or, if that’s not available, Change.

Check if it helps.

As a work around, I suggest you to download and install the latest graphics drivers from your computer manufacturer website and check if it helps.

You can also install the older programs in compatibility mode and check if it helps.

Install the drivers for Windows 7/8/8.1 from the manufacturer's website in compatibility mode and check if that works.

Make older programs compatible with this version of Windows

Check if it helps.

I hope the information helps. Please keep us posted on the issue. We will be happy to assist you accordingly.
