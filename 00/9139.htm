Macro to compare sheets and highlight differences - Microsoft Community

raderm say:

Macro to compare sheets and highlight differences

I have two sheets that I would like to compare.  Each sheet has a unique identifier in column A.  So I would like to perform a vlookup to compare data that is in various columns in sheet2 against sheet1.  For any differences I would like to highlight the differences on sheet2.  How can I do this via a macro?

For example:

Sheet 1

Employee Number      Department   Manager

1                                  Toys               Smith

2                                  Books             Johnson

Sheet 2

Employee Number      Department   Manager

1                                  Books             Smith

2                                  Toys               Johnson

3                                  Books             Smith

After running the macro the following would be highlighted on Sheet 2: Books, Toys, 3, Books, Smith

andreas killer say:
Option Explicit

Sub Test()
  Dim DataAR As Range, DataBR As Range
  Dim DataA, DataB
  Dim i As Long, j As Long, k As Long
 
  'Refer to all data cells
  Set DataAR = Sheets("Sheet1").Range("A1").CurrentRegion
  Set DataBR = Sheets("Sheet2").Range("A1").CurrentRegion
  'Read in all values
  DataA = DataAR.Value
  DataB = DataBR.Value
 
  'Remove all colors
  DataBR.Interior.ColorIndex = xlColorIndexNone
 
  'For each row except the header
  For j = 2 To UBound(DataB)
    For i = 2 To UBound(DataA)
      'Search for "Employee Number"
      If DataA(i, 1) = DataB(j, 1) Then
        'If found, compare the cells in this row
        For k = 2 To UBound(DataA, 2)
          'Color if not equal
          If DataA(i, k) <> DataB(j, k) Then
            DataBR.Cells(j, k).Interior.ColorIndex = 3
          End If
        Next
        'Done
        Exit For
      End If
    Next
    'Color the "Employee Number" if not found
    If i > UBound(DataA) Then
      DataBR.Cells(j, 1).Interior.ColorIndex = 3
    End If
  Next
End Sub

mohan suryanarayan say:

Hi Mottya,

Welcome to Microsoft Communities.

This issue could happen due to many reasons, but mainly due to the presence of a previous version of Office or remnants of another version of Office on the computer.

First check what type of internet connection you use (Wired or Wireless)?

If you are using a wireless connection then you may try to hardwire the connection. Once this is done continue with these steps as mentioned below.

Having checked the above, login to your Microsoft Account with your Microsoft Account Credentials and check if your Office 365 edition reflects on the My Account page. The URL for the Microsoft Account webpage is: http://office.com/myaccount. Microsoft account credentials would be your Outlook.com or Hotmail.com or Live.com email account which you have used for the Office 365 purchase registration.

If your My Account page reflects your version of Office 365 - then follow these steps to remove the current remnants of Office and install Office 365. 

  • In order to uninstall Office remnants from your computer, you will have to install and run the Office 365 FIXIT tool.
  • Download / Install and run the Office 365 FIXIT tool from the following link:

    Once you have completed running the FIXIT follow these steps to delete the Office 15 folders.

    • From Windows Explorer please navigate to the following location: C:\Propgram Files\ or C:\Program Files (x86)\
    • Make sure to delete the Microsoft Office 15 folder. In case you are able to delete the folder then right-click on it and click on Rename. Rename the folder as Microsoft Office 15 old. Ignore these steps in case you are not able to find the Microsoft Office 15 folder in the above mentioned location or if you are not able to delete or rename the Microsoft Office 15 folder.
    • Once you have completed running the FIXIT and deleted/renamed the Office 15 folder be sure to restart your computer.
    • Now try logging into the My Account page and install Office 365. 

    Check if you were able to download and install Office 365 successfully. Once this is completed check if Office 365 prompts you for activation. If yes, proceed with the steps on screen and activate Office 365. 

    In case the above steps do not work then please follow the steps from this link to perform general troubleshooting for Office 2013 / 365.

    http://support.microsoft.com/kb/2822317

    Please try the above steps and let us know if you need further assistance.

    Thank you.
