when using multilevel list continually changes format - Microsoft Community

roberthodson say:

when using multilevel list continually changes format

When I use the multilevel list, the formatting changes to one level every time I save the document. I use the Ctrl-Z command to restore the correct formatting. Any Idea why the formatting always changes?
john mcghie say:
Yes, we know why, but it's pages of technical detail.

You don't need to know "why", you need to know "what to do about it".

You need to create a List Style, then apply it to your text.  The following represents the basics: get back to us if you need more detail:

Here is how to create a List Style from the beginning.  Just for the purposes of demonstration, do it EXACTLY this way (later, you can skip back and forth or use different access routes, but until you get the hang of what you are looking for, do it exactly this way): 


1)  Create a new blank document 


2)  Create your List Style by using Format>Style>New... 


3)  Give it a name, and choose a Type of List 


4)  Now click Format>Numbering>... 


5)  Choose any one of the samples.  It's best to avoid the ones with "Heading" in their samples unless you are making a Heading Numbered List.  If you want to use 1, a, i numbering, choose that sample. 


6)  Click Customise... 


7)  At the bottom of that Customise Outline Numbered List dialog, click the blue disclosure triangle at the extreme bottom left to reveal the Link Level to Style field. 


8)  In the Link Level to Style field, choose "List Number" as the style.  Ignore the other settings for a moment. 


9)  In the top left of the Customise Outline Numbered List dialog, set the Level to "2" 


10)  In the Link Level to Style field, choose "List Number 2" as the style. 


Repeat these last two steps until you have attached as many styles as you want levels in your list.  Even though a List Style can potentially have nine levels, only paragraph styles List Number through List Number 5 will appear in the document.  If you need more, come back and I will tell you how to get 6 to 9 to appear. 


When you have set up levels 1 to 5, OK your way all the way out and save the document (so if your changes blow up later, you have a foundation to go back to). 


11)  Now go to Format>Style... and choose the name of the style you just created.  Hit Modify>Format>Numbering>Customise...  Notice that several options are greyed out, because you are working on a List Style, which does not have the other options. 


12)  NOW you can go from Level 1 down to Level 5 and set your indents and tab positions.  All of the other settings will probably be correct, so leave them alone. 


13) When you have set the tabs and indents for all five levels, OK your way back to the Modify Style dialog and check the "Add to Template" box. 


14)  OK your way out now, hold down the Shift key, and choose "Save All", which will appear on the File Menu when you hold down Shift. 


Notice that we do not change the paragraph styles?  We do not need to, they List Style will reflect our indent and tab changes into them automatically. 


Notice that we have not yet "applied" the list style to anything?  We never need to: it will be applied automatically when we apply the paragraph styles. 


You can now close that document and create a new one.  If everything worked, our new List Style will be in it, but not applied to any text. 


Now put some text in there. 


Select the paragraphs you want numbered, and apply the paragraph style List Number.  Behind the scenes, Word will automatically attach the List Style we just made. 


To adjust the level of numbering in each paragraph, place your cursor at the beginning of each paragraph and hit Tab or Shift Tab.  When you do that, the List Style automatically applies the correct paragraph style for you, which automatically calls in the correct level of numbering as defined by the List Style. 


The list style acts as a "holder".  It is actually a tiny list structure like an Excel table that contains a line for each level of the list, and a row of data for each level that indicates which paragraph style, character style, or List Template contains the indents, fonts, or numbering level, respectively. 


Yes, you are right: it's quite complex.  There are other ways of setting it up (suitable for lawyers...).  I strongly recommend that you do not use those ways, because it leads to the problem that bought us in here. 


Done this way, the list style and all of its associations will be stored in your template, will copy into each new document automatically available for us, and will even paste into another document if you copy a paragraph formatted with it. 


It works very well, and its stable.  Despite the lawyers... 


Hope this helps

deepestblue---01 say:

dvdsik, I would suggest that you try a hard reset. First backup your phone memory as a hard reset overrides the UDP (User data protection) and clears up everything in the phone memory. Then, turn off the phone, press and hold down the Call, '* (Star)' and '3' buttons, turn on the phone without leaving them. Release them only when you get to the region selection screen.

 

Before you proceed with restoring the backup, it is a good idea to see if everything works fine. And next, restore only one item at a time, from the backup. So that, if at all there's a problem, you can reset the phone again and when you restore the backup you can skip the item that you'd restored last.

 

Hope this helps :smileyhappy:
