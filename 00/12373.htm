andreas killer say:

A better way is to create a MailItem from the saved MSG file with the OpenSharedItem method. And it's possible to save the body from the mail only.

Andreas.


#Const SaveBodyOnly = False

Sub SaveAsTXT(MyMail As Outlook.MailItem)
  Dim Path As String, FName As String
  Dim mAttachment As Attachment
  Dim mItem As MailItem
#If SaveBodyOnly Then
  Dim ff As Integer
#End If

  Path = "C:\Documents and Settings\jskelton\Desktop\Email Attachments\"

  'Visit each attachment
  For Each mAttachment In MyMail.Attachments
    'Mail attached?
    If mAttachment.Type = olEmbeddeditem Then
      'Save the attached MSG as file
      mAttachment.SaveAsFile Path & mAttachment.Filename
      'Create a MailItem from saved MSG file
      Set mItem = Session.OpenSharedItem(Path & mAttachment.Filename)
      'Create a file name for the text file
      FName = Path & Replace(mAttachment.Filename, ".msg", ".txt", Compare:=vbTextCompare)
#If SaveBodyOnly Then
      'Save the body of the mail as plain text
      ff = FreeFile
      'Create the TXT file
      Open FName For Output As #ff
      'Write the body, remove double line feeds
      Print #ff, Replace(mItem.Body, vbCrLf & vbCrLf, vbCrLf)
      'Close the file
      Close #ff
#Else
      'Save the mail as plain text
      mItem.SaveAs FName, olTXT
#End If
      'Close the MailItem (otherwise we can't erase the file)
      Set mItem = Nothing
      'Erase the MSG file
      Kill Path & mAttachment.Filename
    End If
  Next
End Sub

aaronzhu_msft say:

Hi James,

The recipient limit in Exchange Online Protection is 500. If you send an email to 2100 recipients, the email will only be delivered to the first 500 recipients. Here is an article for your reference: Exchange Online Protection Limits.

Feel free to post back if anything is unclear.

Best Regards,
Aaron

Hi James,

Here is a workaround: you can create a DG (distribution group) which contains all the recipients, and send emails to this DG instead of those separate recipients.
When sending email to a DG, the DG will be considered as one recipient. Regarding the limitation of a DG, please refer to this link: https://technet.microsoft.com/en-us/library/exchange-online-limits.aspx#DistributionGroupLimits  .
