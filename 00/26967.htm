=AND(LEN(A2)=14,ISNUMBER(-MID(A2,ROW(INDIRECT("3:6")),1)),FIND(MID(A2,ROW(INDIRECT("1:2")),1),"ABCDEFGHIJKLMNOPQRSTUVWXYZ"),FIND(MID(A2,ROW(INDIRECT("7:8")),1),"ABCDEFGHIJKLMNOPQRSTUVWXYZ"),ISNUMBER(-MID(A2,ROW(INDIRECT("9:14")),1)))

Upper/Lower case letters

Why is it when I type in my email  address using 1 upper case letter in my last name, Microsoft/Live sends it back to me with a lower case letter. I don't spell my name nor would I ever spell anyone's last name with a lower case letter. Please explain why that happens it really bothers me.

brian tillman [outlook mvp 2007-2019] say:

That's a good question because if you read the standard that describes the format of email addresses, you see that while the domain part (after the @ sign) is case insensitive, the username part (before the @) is not and the receiving mail server is free to treat "John.Jones" differently than "john.jones".  That also means, however, that the receiving mail server is just as free to ignore case and that's what the Microsoft server does.

jay freedman say:

In the File > Page Setup dialog, if you set the margins you want for all new documents and then click the Default button at the bottom of the dialog, that should update your Normal.dot template to store those values.

If you've done that and the values don't stick, then something is preventing the saving of the Normal.dot template. It might be a badly behaved add-in, or a permissions problem with the template itself. To work around that, open the Normal.dot template "as a document" -- that is, not as the base for a new document. Make the changes in the opened template and save it, then close it and try making a new document.

Because the folder that contains the template is normally hidden, it can be difficult to find. The simplest way to open it as a document is to follow these steps:

  • Press Alt+F11 to open the macro editor.
  • Press Ctrl+G to open the Immediate window in the editor.
  • In the Immediate window, type or paste the following line and press Enter:

NormalTemplate.OpenAsDocument

since you are not an administrator, please involve the global administrator of your organization into this thread to change your primary email address by referring to the following steps:
1. login to the exchange admin center (outlook.office365.com/.../) with the admin account.
2. click recipients > mailboxes.
3. select the mailbox and edit it.
4. click email address, select the email address which you want to edit and click the pencil icon.
5. check make this reply address and save.

then this email alias will be used as the primary email address for this mailbox.
