    How to change pathway names using Windows 10 - Microsoft Community

    conniemclaughlin say:

    How to change pathway names using Windows 10

    My computer name ACER the pathway user name associated with most of my files is Sally Smith Jones so it looks like this: ACER\Sally Smith Jones.  I want to take the Jones part off of the pathway.  How do I do that?  I've already changed my user name but it didn't change all of the file names.
    hairyfool say:

    The original user profile folder would have been generated based on the Username variable and itself cannot be changed. Although we tend to think of the user as being based on the Username technically it is not, instead when created each user is given a Security ID. It does not matter how often you change the name the SID remains constant and the profile will always remain as initially set up. You may find guides on how to change it but be aware that as an unsupported change it is not certain to be reliable.

    Externally the Acer\Sally Smith Jones is actually the Share name of that folder, by default it is the same as the folder name but you can reset the Sharename different to the folder name.

    However if you really want to have just Sally Smith as the user and everything based around that then the best thing is to create a completely new user.

    Rename the existing User to different name altogether, create a new user as you want it to be, make sure it is assigned as Administrator and log in at least once as that new user to create the profile, copy the User files from the old account to the new one.

    Once you are happy with the new profile is as you want it then you can delete the old user account including its profile.

    carolynsnider say:

    Carolyn @ *** Email address is removed for privacy ***

    Thanks so much.

    a. user say:

    Hello Carolyn,

    Thank you for contacting Microsoft Community.

    We understand your requirement in this regard.

    You can use Windows Media Player to copy songs from audio CDs to your computer. This process is called ripping. After that you can move those files to USB stick. During the ripping process, the Player makes a copy of each song, stores it on your hard disk as a Windows Media Audio (WMA), MP3, or WAV file, and then adds it to your Player Library.

    1. For best results, make sure that you are connected to the Internet before you begin.
    2. When your computer is connected to the Internet, the Player attempts to retrieve media information about the tracks being ripped from an online database. The Player then adds the information to the files during ripping.
    3. Note that you can rip a CD when your computer isn't connected to the Internet. However, the Player won't be able to identify the name of the CD, the artist who created it, or the names of the songs until you go online.

    4. Insert an audio CD into the CD drive.

    5. Open Windows Media Player. To do this press Start, type Windows Media Player and select same from list.

    6. (Optional) If you want to select a different format (for example, MP3 instead of WMA) or bit rate (for example, 192 Kbps instead of 128 Kbps) for the files created during ripping, in the Player Library, click the Rip settings menu, and then click Format or Audio Quality.

    7. (Optional) If you don't want to rip every song, in the Player Library, clear the check boxes next to the songs you don't want to rip.

    8. Click the Rip CD button in Now Playing mode or Rip CD in the Player Library to begin ripping.

      After the songs have been ripped, you can find and play them in the Player Library.

    9. The Player stores ripped songs in the Music library on your computer. If you delete your Music library or remove all locations from it, then you won't be able to rip music from a CD.

    10. Now press Windows logo key + E, then open Music folder. There you can find all songs which are ripped before. Insert USB Flash Drive and then copy all those song to your USB Flash Drive.

    Alternatively, directly go to ThisPC, right-click on CD/DVD drive and select Open. Select and copy all files you want and then copy them to USB Flash Drive. You're done.

    Keep us posted if you require further assistance.
