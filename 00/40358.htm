andrew_whs say:

I realise this is an old thread, but in case someone else finds it, then this may help.

I went the "local user" way when a client had the same problem.

FIRST MAKE SURE YOU HAVE ALL PASSWORDS TESTED AND WRITTEN DOWN!

  1. Log in to the pc using the current account and go to "Manage Your Account"
  2. Select "Sign in with a local account instead" (NOT "Manage my Microsoft account")
  3. Follow the prompts to create a new local account.
  4. When complete, sign out and back in with the local account
  5. Return to "Manage Your Account" and select "Sign in with a Microsoft Account"
  6. Again, follow the prompts to associate the desired Microsoft account.
  7. Log out and back in, this time with the new MS account.

That should do it. In my case, it retained all profile settings and programs.
