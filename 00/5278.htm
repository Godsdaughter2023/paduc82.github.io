a. user say:

If you have not already added Hebrew in Windows Control Panel->Regional and Language Options->Languages->Details, do that. In that option, make sure that when you do that, you install a Hebrew keyboard - this is a "logical" keyboard, which can type Hebrew characters regardless of what physical type of keyboard you have. That's what really lets you type Hebrew characters in Word.

Then use Microsoft Office 2007 Language Settings to add Hebrew as an editing language, and restart all your Office programs.

In Word, to type Hebrew, go to the icon in the Windows taskbar that shows the current language (e.g. it may show "EN" in a small blue box. Select "HE". If you have more than one logical keyboard for Hebrew, you should see a separate keyboard icon pop up. Select the Hebrew one.

(If you don't see the language bar, it can also be enabled in Windows Control Panel->Regional and Language Options->Languages->Details )

If a border has been applied as direct paragraph formatting (not part of the
style definition), then Ctrl+Q (ResetPara), a command that is as old as

this will not work. And if it has been applied as direct *font* formatting,
then Ctrl+Spacebar would work. The OP says, "this borders were not applied
as a style but with direct formating," which suggests that Ctrl+Q should
work.
