Check Settings/Update and Security/Recovery and see if you have the option to return to the previous version (which should be 15063).  If not you can reinstall 16053 or you can wait for a more stable build and then stop receiving insider previews.  You will stay on whatever build you are on at that time until the current cycle ends and Redstone 3 is released to the public.  You will automatically receive the new version the same as if you had been getting new preview builds all along.  Another approach is to switch to Slow Ring and wait for the first Slow Ring build to flight and see if the problems have been fixed.

I always make an image backup of the new public releases for just this sort of scenario.  I use the tool in the Control Panel/System and Security/Backup and Restore (Windows 7).  The Create a System Image link is in the upper left corner of that page.  (Ignore the reference to Windows 7.  The page is very poorly named.  Only one tool on the page applies to Windows 7.)
