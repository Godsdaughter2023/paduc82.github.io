An archive mailbox is designed to be under the control of the owner, so all clients are designed with that principle in mind and therefore you don't have the ability to disable delete access to the archive. 1) created some work resources with simple hourly costs, groups and codes in the resource view
2) created some tasks and task headers in the task view
3) assigned work resources and duration to the tasks in the task view

Now Project calculates and shows me the sum of cost of all resources per task and task header.

I want to see the sum of cost of the different resource groups or codes per task and especially task header. As an example: The cost of "external" and "internal" work resources applied to a task header.

julie sheets say:
Hello SebastianVad,

If you are open to using the Task Usage view and grouping, you can get what you seek.  Apply the Task Usage view, add the cost field to the table (left side) and create the following group:

Group by:
Field Name: Outline Number
Field Type: Task
Order: Ascending

then by:
Field Name: Name
Field Type: Task
Order: Ascending

Then by:
Field Name: Resource Group
Field Type: Assignment
Order Ascending
