Correct, the top one is the one the mail app sees as the Junk mail folder. The one named Junk Email is the one the server uses for junk mail. The folder name on the server (and in outlook desktop software) is Junk Email - Apple displays the name just as 'Junk' - normally, it uses the same folder as the server and just changes how the name is displayed. outlook.exe /resetfolders

This will reset the folder assignments to the default folders for the account and recreate any missing folders.

scott mcdaniel say:

You can use the following to fill your listbox:

Function ShowDriveLetters()
  Dim fso As Object
  Dim drv As Object 'Drive

  Set fso = CreateObject("Scripting.FileSystemObject")

  For Each drv In fso.Drives
      Me.List0.AddItem drv.DriveLetter
  Next
End Function

From there, it can get fairly difficult to recreate an explorer-like interface, although you might be able to kludge one together with other components of the FileSystemObject.

FWIW I had need to do this on a project, and ended up using the ActiveX version of this product:
