      Option Explicit

      Public Sub DeleteRange()
          Dim WB As Workbook
          Dim SH As Worksheet
          Dim Rng As Range, rCell As Range, delRng As Range
          Dim LRow As Long
          Dim CalcMode As Long

          Set WB = ThisWorkbook
          Set SH = WB.Sheets("Sheet1")

          With SH
           LRow = .Cells(.Rows.Count, "T").End(xlUp).Row
           Set Rng = .Range("T2:T" & LRow)
          End With

          With Application
              CalcMode = .Calculation
              .Calculation = xlCalculationManual
              .ScreenUpdating = False
          End With

          For Each rCell In Rng.Cells
              With rCell
              If IsNumeric(.Value) And .Value < 0 Then
                  If delRng Is Nothing Then
                      Set delRng = rCell
                  Else
                      Set delRng = Union(rCell, delRng)
                  End If
              End If
              End With
          Next rCell

           If Not delRng Is Nothing Then
              delRng.EntireRow.Delete
           End If

          With Application
              .Calculation = CalcMode
              .ScreenUpdating = True
          End With
      End Sub

      sher amo say:

      Hi John,

      In short, skip ahead is a new option for Insiders in the Fast ring that allows users to jump ahead, past the current Fall Creators Update preview builds, and onto the development branch where the update coming after the Fall Creators Update is being developed. It puts Insiders on the Redstone 4 channel of development builds, meaning Insiders who choose to skip ahead won't be given the chance to test the Fall Creators Update RTM.

      With the new skip ahead option, Microsoft can begin testing the next version of Windows 10 much sooner than in the past, even before the active Windows 10 development cycle is complete. It gives Insiders a choice between testing the upcoming release or testing the release that's coming after the upcoming one.

      In line with your query about functionalities, you won't lose anything since your device is running the skip ahead option meaning you are using one of the advanced builds.

      First of all, do not use Insider previews on your new laptop if it is your primary computer.  I cannot emphasize that too strongly.

      Second, if Windows Pro came preinstalled on your old computer you cannot transfer the license.  You can only transfer a license if you bought that copy of Pro at retail and then used it to upgrade the old computer from Home to Pro.  If you have a retail (stand-alone) license for Windows Pro, contact Microsoft Support for help transferring it by clicking on the Get Help icon on your Start Menu.

      Third, your new laptop's manufacturer will not support Windows under the warranty while the laptop is running pre-release software and the manufacturer will require you to restore the laptop to factory condition before giving help.  Hardware will probably still be covered but your experience with the manufacturer's support personnel is unpredictable when you have a preview installed.  Most support personnel have no idea how to help unless the original OS is running.
