cmdrkeene say:

That's a good question.  Typically the longest portion of it is the "user data migration" bit.   You see, these updates might be advertised as just updates, but they are really entirely new versions of Windows -- just like upgrading from XP to 7 (for example).  The system first moves all user files to a temporary place, updates the OS, then moves the files back.  If the drive uses a spinning drive instead of solid state, it can take forever, especially if you have dense data like movies (I have more than 13,000 pictures and about 600 GB of personal files, so I feel that pain too).

Microsoft made some major changes to how the updates work to speed it up, but unfortunately most of those fixes are currently still on the insiders-only track.  Once it makes it to the public, you should see some dramatic improvements also.

Hi,

The frequency of the back up process depends on how you set it up. You can schedule a time on a daily basis where your system can automatically backup your files. To set up automatic back up, you may follow the steps below.

  1. Go to Control Panel and select System and Security.
  2. Select Backup and Restore (Windows 7), and then Set up backup.
  3. Select the external drive that you want to use and click Next.
  4. Select all the items that you want to back up/protect.
  5. Check the Include a system of drives: System Reserved, (C:) option and click Next.
  6. On the Schedule section, click the Change schedule link.
  7. Select the day, time and frequency of backing up. (It is advisable to select the back up schedule when you don't usually use your computer)
  8. Click OK, and then click the Save settings and exit button for the configuration to take effect.
