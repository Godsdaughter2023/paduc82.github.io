word count problems - Microsoft Community

sean o fitzgerald say:

word count problems

I have a 50,000 word thesis excluding references, is there a way of separating out the references so I can get a differential word count e.g. my text is 45,000 and all references is 15,000, that is without highlighting or deleting which is laborious?
macropod say:

Hi Sean,

Did you use a particular paragraph Style for the indented blocks of references that you didn't use elsewhere?

For in-line quotes conforming to the description you gave (and provided nothing else fits that description), you can use the following macro:

Sub RefWordCount()
Application.ScreenUpdating = False
Dim i As Integer
With ActiveDocument.Range
  With .Find
    .ClearFormatting
    .Text = "[" & Chr(34) & ChrW(8220) & "]*[" & Chr(34) & ChrW(8221) & "]*\(*[0-9]{4}:[!\)]{1,}\)"
    .Replacement.Text = ""
    .Forward = True
    .Wrap = wdFindStop
    .Format = True
    .MatchCase = False
    .MatchWholeWord = False
    .MatchWildcards = True
    .MatchSoundsLike = False
    .MatchAllWordForms = False
    .Execute
  End With
  Do While .Find.Found
    i = i + .ComputeStatistics(wdStatisticWords)
    .Collapse wdCollapseEnd
    .Find.Execute
  Loop
End With
MsgBox i & " words found in inline quotes."
Application.ScreenUpdating = True
End Sub

The Software License Terms say that there are Transfer rights.  In the case of a purchased license, there is a product key for Windows 10.  That key can be transferred like earlier Windows keys (e.g. slui 4).

The more problematic technically is transferring a stand-alone 7 or 8 key that has been upgraded to 10.  Assuming that you can still transfer the 7 or 8 product key to a new device, how does Microsoft prevent a clean install and activation with no product key of Windows 10 on the previous licensed device which the license was removed from?  Perhaps they tie the Windows 10 device to the previous OS product key and can remove it from auto-activation if that previous OS key is transferred.

Another issue is if a stand-alone license is upgraded to Windows 10, it gains transfer rights from the previous license.  If that license is transferred after July of 2016, it should remain a transferrable Windows 10 license.  If the only way, technically, is to transfer the previous OS product key and upgrade, then the upgrade to 10 has to remain free of charge even after July 2016 (for devices upgraded from stand-alone software during year one) in order to maintain the Transfer rights for Windows 10 stand-alone, even though there is no unique product key for the upgraded Windows 10.
