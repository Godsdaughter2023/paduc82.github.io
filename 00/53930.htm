4) Restart your computer with the USB plugged in

Let the Workgroup name be the same for both computers.

If you want sharing between the two computers, they can set up a Homegroup.

Method 1: You need to follow the steps mentioned below to change the network to a home network.

a) Click on “Start” and type “Local security policy” (without the quotations) and hit “Enter”.
b) Now select “Network list manager policies”.
c) In the right pane double click “Unidentified networks”.
d) Under “Location type”, place a dot next to “Private” (work) and click on “ok”.
e) Now restart the computer and check.

Note: Local policy editor works only in the Professional, Enterprise and Ultimate editions of Windows 7.
If the steps mentioned in Method 1 does not help then go to Method 2.

Method 2: You need to follow the steps mentioned below –

a) Click on “Start” and right click on “Computer” and select “Manage”.
b) Now select Device Manager and expand “Network adapters”.
c) Right Click on all the network adapters on by one and select “Disable”.
d) Now enable the adapter that is showing up in the Unchangeable public network.
e) Go to “Network and sharing center” and select “Choose home group and sharing options”.
f) Click on “What is this location?”
g) Select “Work” and re-enable all the other adapters.

Choosing a network location
