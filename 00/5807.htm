Hello Alex Baggott,

Generally, a Shared mailbox is not listed in Account Settings. This is because a shared mailbox doesn’t have a license, which is not an “account” in Outlook. When a user is added as a member of a shared mailbox, then it will automatically appear in Outlook client by default (which is called automapping feature).

If you want to remove the shared mailbox from Outlook client, you can click File > Account Settings, select your account and then click change > More settings. Under Advance tab, choose the shared mailbox listed there, and then click Remove.

Formula in cell C2:

=SUM(IF(FREQUENCY(A2:A100,A2:A100)>0,1))

=SUMPRODUCT(1/COUNTIF($A$2:$A$20,$A$2:$A$20))
