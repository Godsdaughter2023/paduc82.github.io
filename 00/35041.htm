a. user say:

Hi Ray,

Thank you for posting your query on Microsoft Community forums.

Have you installed any other printer after the Brother printer previously installed?

Sometimes, traces of an old printer are still stored in the registry entries even if you have uninstalled them. Thus, if you have not installed any other printer currently, then I would suggest you to try the following steps and check if the issue persists.

Step 1: Create a restore point

  1. Press the WinKey+X to display the system menu and click System.
  2. On the left side menu, click System Protection.
  3. In the Protection Settings section, click the C: (system) drive.
  4. Click the Create button.
  5. Type a name for the System Restore file (The Date and Time will be added automatically).

Step 2: Boot your computer in the safe mode.

Step 3: Browse to C:\WINDOWS\SYSTEM32\SPOOL\DRIVERS\w32x86 & delete the contents referring to the old Brother's printer.

Step 4: Browse to C:\WINDOWS\SYSTEM32\SPOOL\PRINTERS & delete the contents referring to the old Brother's printer.

Reboot your computer in the normal mode and check for the issue.

Hope this information is helpful. If the issue still persists, please post back for further assistance, we'll be glad to assist you.
