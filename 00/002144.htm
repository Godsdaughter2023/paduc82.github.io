jay freedman say:

Probably the easiest way to script this is to supply a .reg file that creates or modifies the SharedTemplates entry in the registry key HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Common\General. The entry is a REG_EXPAND_SZ value that should contain the path to the Workgroup Templates folder on the network server.

Because the value of the entry is stored as a hex array rather than as a simple string, it's best to create the value in the registry editor on a single workstation, export the General key to a .reg file, and edit that file in Notepad to remove all the other entries except the SharedTemplates one.

For computers whose Office suite has already been updated to Office 2016, the path to the registry key should contain 16.0 instead of 15.0. I'm not sure whether the Office 2016 installer will carry the setting forward from Office 2013; if not, then the updated .reg file would have to be used.

Depending on the way security is enforced in the network, it may be necessary to execute the .reg file with Administrator credentials on each computer.

Help with pre-printed letterhead paper - Microsoft Community

josephpear say:

Help with pre-printed letterhead paper

Dear Community members, I need help with pre-printed letterhead paper.

 

I have a letterhead paper, which needs a 2 ½-inch top margin and 1 ½-bottom margin.  For the second and so forth pages, I need a 1-inch margin all over.

 

How can I set that in Word 2013?

 

Your help is very much appreciated.

 

Joseph P.

jay freedman say:
Click the Page Layout tab on the ribbon.

At the left end, click the Margins button. If the Normal item (1" for all four margins) isn't selected, click it. This is the value that will be used for all pages except the first one.

Next, double-click in the header area, which opens the Header & Footer Tools tab on the ribbon. On that tab, check the box for Different First Page. Notice that the tag at the bottom of the header area now says "First Page Header". Leave the cursor in the header paragraph.

Again click the Page Layout tab. In the Paragraph group, under Spacing, change the After value to 132 pt.  To arrive at this number, I started with the desired 2.5", subtracted the 0.5" default distance from the top edge of the paper to the top of the header text, multiplied by 72 pt per inch, and subtracted the 12 pt depth of the header text paragraph:
   ((2.5 - 0.5) * 72) - 12 = 132

Move the cursor to the First Page Footer. On the Page Layout tab, in the Paragraph group, under Spacing, change the Before value to 60 pt. (The math is the same, but starting with 1.5".)

Finally, double-click in the document body to close the header and footer areas, and save.

For example if you want to use the worksheet change event to copy and paste the entire row where the user has made a change then a simple test of how many rows selected and if only one row then use the entirerow command and copy like the following: (You can copy and paste multiple entire rows but following is just an example of testing how many rows in the target.)

 

Private Sub Worksheet_Change(ByVal Target As Range)

    If Target.Rows.Count > 1 Then Exit Sub
   
    On Error GoTo ReEnableEvents
    Application.EnableEvents = False
    Dim rngDestin As Range
   
    With Sheets("Sheet2")
        'Only require one cell for destination
        Set rngDestin = .Cells(.Rows.Count, "A").End(xlUp).Offset(1, 0) 'Next empty row
    End With
   
    Target.EntireRow.Copy Destination:=rngDestin   'This will cancel any previous Copy.
   
ReEnableEvents:
    Application.EnableEvents = True
End Sub
