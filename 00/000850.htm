burrwalnut say:

Frank

Is it possible that you have a dual-band router, i.e. one network at 2.4 GHz and the other at 5GHz?

Check you router documentation or look at the router details via your browser. If your router appears in the Network folder, right-click or press and hold it and choose View device webpage.

If it doesn’t appear, type 192.168.x.y into a browser’s Address bar. The values of x and y depend upon the make of router. If you don’t know the values, open a Run window (Windows Logo key+R), type cmd /k ipconfig (note the two spaces), press Enter, make a note of the Default Gateway address and type it into the browser’s Address bar.

If asked for the User Name/Password, type the default values, which are often admin and admin. If they are not known, look in the router’s user manual.
