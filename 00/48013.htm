munkyxtc say:

Jay340

If you are using the Instant On feature of the Xbox One with an external hard drive you'll want to make sure your settings are properly configured.

Following the instructions here:

support.xbox.com/.../change-power-settings

You'll need to make sure the "Automatically download updates and purchases" option is checked.

Additionally, you'll want to make sure the "When Xbox is off, turn off storage" option is unchecked.  Some external drives do not operate in a low power mode and as such even if the xbox tries to download in the background the external drive will not be operational which would prevent the automatic updates from occurring.

  • It uses a USB 3.0 connection. If you’re using a USB hub, the hub must also be USB 3.0 or later.
