When you print a Microsoft Word document, the printer settings are ignored. These printer settings include the paper size, the page orientation, and the page margins.

.harbinger say:

Using OLE with Office apps across OneDrive

Is it possible to link a remote (via OLE) that is stored on OneDrive?  For example, a user wants to have an Excel spreadsheet on OneDrive that they update regularly but also wants the PowerPoint presentation (also on OneDrive) to automatically update certain slides where the user has embedded the Excel spreadsheet via OLE.  I tried it and it didn't work...no error, but not working either.  

If this isn't an option, what could work in this instance? 

ryan lv msft say:

Hi .Harbinger,

I could understand the reason that you prefer using OLE to embed an Excel file into PowerPoint, as other users can directly check/edit on the sheets within PowerPoint.

To mention that embed file via OLE is more like linking a static copy of original file at the point you perform the action, but not the original one. Therefore, if the original file stored in OneDrive gets updated, embedded copy in PowerPoint won't update correspondingly. This results in two different versions of the file. 

Per my further test, although this can be done via mapped drive, you may encounter some unexpected issues when collaborating on the files. For instance, if you use real-time coauthoring feature with Office 365, the embedded file may not get updated correctly.

Therefore, my suggestion is to add hyperlink in your PPT to let users access the online location of your Excel files based on your scenario. Users can later work in Office Online for quick editing or pin the file to their list for later use.

Meanwhile, I would also share the idea here about embed file via OLE in case it may be helpful to you.
1. Follow the steps in How to configure and to troubleshoot mapped network drives that connect to SharePoint Online sites in Office 365
(Authenticate to Office 365 & Add your SharePoint Online sites to trusted sites
2. Follow the steps in Mapping a drive to OneDrive for Business to map OneDrive to local drive.
3. Open PowerPoint to embed Excel by finding the file in the mapped drive you just created.

Note:
a. When configuring the mapped drive, use IE to do the preparation.
b. The mapped drive will need to be recreate after certain period due to the limitation of this method.

Regards,
Ryan
