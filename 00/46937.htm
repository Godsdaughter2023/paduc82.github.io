  • Point to the Windows 10 ISO file

  • Make sure your USB thumbdrive in the Device list box
  • Click Start
  • Close when complete

piansa say:

How can I redirect my screenshots to be saved to my computer rather than to OneDrive directory?

Because I just discovered that my screenshots are saved only in my OneDrive directory, so choosing to uncheck the option 'to automatically save to my OneDrive' would mean that no screenshots would be saved?
keith a. miller say:

If you only  press PrtSc, you copy a screenshot to the Clipboard, but it's up to you to open a program such as Paint to paste & then save the screenshot.

Try viewing the Properties of the Screenshots folder under OneDrive.  On the 'Location' tab, try Restore Default or Move, that should restore the default path.

If the system balks with 'Access Denied' or similar, copy & paste the following line into a Command Prompt:

Reg Delete "HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders" /v {B7BEDE81-DF94-4682-A7D8-57A52620B86F} /f

Then, the next time you use Winkey+PrtSc, it  will create the Screenshots folder in your Pictures folder.
