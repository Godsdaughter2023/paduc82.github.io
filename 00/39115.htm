As has been pointed out many times on here, it can't be done. The Product ID that a key is valid for is integrated into the key. Windows 8 has a different Product ID to Windows 8.1 so a key can only be used for the original operating system.


In order to use the key you have to install Windows 8 then upgrade to 8.1, the alternative is to make a system image as the system stands at the moment and then restore that image when needed.


There are some workarounds in order to get the Windows 8.1 ISO which might be useful when doing a refresh of the existing installation. f. Enter the required details.
g. Then login to the local account and check how it works.

 

Step 2: Switch back to Microsoft account

 

Follow these steps to create a “Microsoft account”.
a. Press “Windows key +W”.
b. Type “users” without quotes in the search box.
c. Then select “Users” from the left panel.
d. Select “Switch to Microsoft account” right below your user account name.
e. Enter the required details (Primary Email ID).
f. Restart the computer and then try to login using new Primary account.
