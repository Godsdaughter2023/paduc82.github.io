a. user say:

I pressed a combination of keyboard keys that changed the orientation of the desktop.  The start bottom and all icons are upside down.


2 ways ( actually 3 )

1. press Ctrl and Atl keys together and HOLD > use the Arrow keys to manipulate the screen back to right side up.

2. right click on an open space of the desktop > Graphics Options > Rotation > try 180 degree.

3. do a system restore. Pick a good date when your screen was right side up as your restore point.
Start button > search box, type system restore >press Enter key > uac prompt > click Choose a different restore point > Next > select the date as your restore point, click it > Next > Finish
Sit back and wait. The machine will re-start when it is done.
For the benefits of others looking for answers, please mark the suggestion as Answered if it solves your problem.

Start your PC in the usual way then try the following.

Hold down the Windows key and press i then release the Windows key. This should open Settings.

Hit the Tab key once. This selects System in Settings.

Hit the Enter key to open System.

Hit the Tab key 6 times. This selects the scale slider.

Tap the left arrow a few times and it should change the scaling.
