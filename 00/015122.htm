E for Excel wrote:

You need to use below Array formula to do the trick -

=AVERAGE(--TEXT(A1:A30,"00\:00"))

To restore the value of the average to the original "military time" form of hhmm, array-enter the following (press ctrl+shift+Enter instead of just Enter):

=--TEXT(AVERAGE(--TEXT(A1:A31,"00\:00")),"hhmm")

formatted as Custom 0000 to display leading zeros.

The double negate (--) converts "numeric" text to a numeric value.

Note that the average value expressed in "military time" is truncated (not rounded) to the minute.  If you want the average array-enter the following (press ctrl+shift+Enter instead of just Enter):

=--TEXT(AVERAGE(--TEXT(A1:A31,"00\:00")),"hhmm\.ss")

formatted as Custom 0000.00 .

Caveat:  The formats above assume that midnight is written 0000, not 2400.  If the original data are elapsed times, not times of the day, it is unclear how you would want to represent averages greater than 2359.  Perhaps changing "hh" to "[hh]" would work in that case.

hansv mvp say:

Refreshing a form should not change the position of records in a form - Refresh doesn't remove deleted records or add newly created records, it merely updates existing records.

Requery reads the entire record source of the form from scratch, removing deleted records and adding new records. After a Requery, the first record will be the active record.

But you can use SelTop to "remember" where you were:

 

    presentposition = Me.SelTop
    ' do something with records
    ...
    Me.SelTop = presentposition
