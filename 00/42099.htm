Hello. So I put up the question and have a working solution but the other threads on this are all locked and so I had to create a new one.

Just solved this problem on a Win10 Home laptop. The problem was not with the CTFMON.EXE file but with the InputLocaleManager.DLL file.


On the laptop that was not working:

InputLocaleManager.DLL 10/21/18 135K

On another Windows 10 Pro that was normal:

InputLocaleManager.DLL 09/15/18 136K

Running SFC /scannow did not fix the problem. I looked up the locale code for the US here:

https://www.science.co.il/language/Locale-codes.php

United States = 409

So checking in the system registry:

HKEY_LOCAL_HARDWARE\SOFTWARE\MICROSOFT\Input\Locales

I verified that there was the appropriate record:

loc_409

for the English-US keyboard. With everything looking ok but Cortana, Edge browser, or any other Windows input field not working I decided to rename the 135K file InputLocaleManager.DLL to InputLocaleManager.DLL.OFF and copy the 136K file of the same name I got off the other Windows 10 to replace it. Windows 10 wouldn't directly allow me to rename this file as I wasn't a "trusted installer" but right clicking on the file name, going into properties, then in the security tab I added my user name on the list and checked all the allow boxes for full control of the file, and hit apply. The rename was then allowed and I could copy the slightly larger InputLocaleManager.DLL into the System32 directory no problem.

I fully expected to have to restart Windows but to my surprise Cortana was taking keyboard input, and so did the Edge browser. The owner of this laptop wanted to create a new user and going into User Accounts and adding a new user the input fields (username, password, etc.) were all accepting keyboard input as normal. Problem solved.

There are many other files in this Win10 System32 folder with a 10/21/18 date but by using one that was dated 9/15/18 and was 1K larger this Windows 10 Home input problem was fixed. A truly bizarre problem probably caused by one of the many Windows updates gone awry.
