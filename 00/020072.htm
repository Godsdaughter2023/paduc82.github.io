hansv mvp say:

Macros don't work that way. You could do the following:

Step 1: define cells that can be edited.

  • Select the cells that the user should be able to fill in.
  • Press Ctrl+1 to activate the Format Cells dialog.
  • Activate the Protection tab.
  • Clear the Locked check box.
  • Click OK.

Step 2: protect the sheet.

  • Activate the Review tab of the ribbon.
  • Click Protect Sheet.
  • Click OK. You can specify a password if you like.
  • Press Tab in the protected sheet to move to the next unlocked cell.
  • If you clear the check box "Select locked cells" in the Protect Sheet dialog, you won't be able to move to locked cells at all.

andreas killer say:

Am 31.12.2010 15:11, schrieb CarmenMorales:

Need a macro that will go only to a specific are in the spreadsheet wait for me to enter information and when i hit enter go to the next place i need it to go to.  This will help me to be more productive and faster in my filling in the forms.

You can use the change event to switch to the next cell after you filled a cell.

Customize the line
  Set VisitCells = Range("A1,B2:C3")
in the code below to all cells that you want to visit and put it in the code module of your worksheet.

Andreas.

Option Explicit

Private Sub Worksheet_Change(ByVal Target As Range)
  'Durchläuft die Zellen in einer bestimmten Reihenfolge
  Dim VisitCells As Range, R As Range, GetNextCell As Boolean
   'Alle zu durchlaufenden Zellen vereinigen
  Set VisitCells = Range("A1,B2:C3")
   'Ereignisse aus, sonst rufen wir uns selber auf
  Application.EnableEvents = False
   'Haben wir eine unserer Zellen geändert?
  If Not Intersect(VisitCells, Target) Is Nothing Then
    'Auswahl auf unsere Zellen begrenzen
    Set Target = Intersect(VisitCells, Target)
    'Nichts machen wenn die Zellen leer sind
    'If IsEmpty(Target) Then GoTo ExitPoint
    'Ja, suche nach der nächsten Zelle
    For Each R In VisitCells
      'Gefunden?
      If GetNextCell Then
        'Ja, dann diese (nächste) Zelle anwählen
        R.Select
        'Fertig, nix mehr zu tun
        GoTo ExitPoint
      End If
      'War dies die letzte Zelle?
      If R.Address = Target(1, 1).Address Then
        'Ja, im nächsten Durchgang die nächste Zelle wählen _
          lassen
        GetNextCell = True
      End If
    Next
    'Dies trifft nur zu wenn es die letzte Zelle war.
    If GetNextCell Then VisitCells.Cells(1, 1).Select
  End If
ExitPoint:
  Application.EnableEvents = True
End Sub

steve rindsberg say:

If your presentation is set up to a slide/page size that's proportional to 16:9

AND

If you display the presentation on a 16:9 format monitor/screen/projector

List of Formatting Shortcuts

Formatting is a mystery in Sway (except for Emphasize). It appears that there is some kind of formatting syntax going on in the text for things like bold and bulleted lists (like Markdown), but I haven't seen a list of the shortcuts. Bullets 

* Item 1

* Item 2

* Item 3

Will show as:

  • Item 1
  • Item 2
  • Item 3

Note: Every space you put in front of an asterisk will indent the bulleted list. Like this

* Item 1

 * Item 2

will show as:

  • Item 1
    • Item 2 

Numbered List

#. Item 1

#. Item 2

#. Item 3

Will show as:

  1. Item 1
  2. Item 2
  3. Item 3
