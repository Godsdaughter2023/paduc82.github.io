Obviously, not everybody considers the issue as annoying as I do ;-), but I am definitely not the only one!

Basically, it has to do with the changes in the files' icons in Windows Explorer that lots of applications (One Drive, Dropbox, Tortoise SVN, etc.) use to indicate the status of the file in that application (like synchronized, new file, modified locally, etc.). funkspunk say:

Quick steps to resolve the issue.

1. Press Win + R Key 

2. Type "regedit" in the run dialogue. 

3. Navigate to 

HKEY_CURRENT_USER\SOFTWARE\Policies\Microsoft\Windows\CurrentVersion\PushNotifications

4. Change the dword value of "NoToastApplicationNotification" to 0 (zero)
5. Close everything and log out and login (or restart system)
6. Open Settings > System > Notifications & actions. "Show app notifications" is no longer grayed out and is active.

Alternately, you can also verify whether Group Policy has disabled notification:

1. Press Win + R shortcut keys together on the keyboard to open the Run dialog.
2 .Type the following in the Run box: gpedit.msc
3. Go to the following path in the left pane of Group Policy Editor:
4. User Configuration \ Administrative Templates \ Start Menu and Taskbar
5. Locate and modify the following setting: Turn off all balloon notifications.
6. Set it to Disabled
