ram kumar p say:

Hello,

Thank you for posting your query in Microsoft Office Community.

If you select text and change the style, the style for that text changes to match the selected style. This is by design. 

However, this will not change the manual formatting that you have applied (Bold, italics etc)

If you select a text and double click on any style, then the manually added formatting will also change to match the selected style.

If you only want to change the spacing in the document, then change it by following the steps below.

1. Select the document (CTRL + A)

2. Open Paragraph settings

3. Change the spacing for Before and After to 0 Pt.

4. Change the Line Spacing to Single.

Hope the above steps helps. Reply with the result and I will be happy to help you further.

Note that when you apply a paragraph style, it affects the current paragraph (= the paragraph containing the insertion point) or the selected paragraph(s).
