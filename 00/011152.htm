tasos_k say:

Hi,

try this

export data in a new wb named: new file

Sub Export_data_in_new_WB_multi_shts()
'May 02, 2017
Dim wb1 As Workbook, wb2 As Workbook
Set wb1 = ThisWorkbook
Dim sPath
sPath = wb1.Path & "\"
Dim vSht As Variant
vSht = Array("WIP", "Bank Detail")
Dim x As Integer
Application.ScreenUpdating = False
Set wb2 = Workbooks.Add(xlWorksheet)
For x = 0 To UBound(vSht)
wb2.Sheets.Add after:=wb2.Sheets(wb2.Sheets.Count)
ActiveSheet.Name = vSht(x)
wb1.Sheets(vSht(x)).Range("A1").CurrentRegion.SpecialCells(xlCellTypeVisible).Copy
Range("A1").PasteSpecial xlValues
Range("A1").PasteSpecial xlFormats
ActiveSheet.UsedRange.EntireColumn.AutoFit
Next
Application.CutCopyMode = False
Application.DisplayAlerts = False
With wb2
.Sheets(1).Delete
.SaveAs Filename:=sPath & "new file.xlsx"
.Close False
End With
Application.DisplayAlerts = True
Application.ScreenUpdating = True
MsgBox "done"
End Sub

How do I format a header and page number without losing the first when I insert the other?

I need step by step instructions for formatting a header with page numbers for an APA paper.  When I insert my header and then try to insert page numbers, I lose the header.  If I try the reverse, the header disappears. 
suzanne s. barnhill say:

If you are inserting the header and page number as building blocks, then each will replace the other. There are several alternatives. The first and simplest is just to double-click in the header area and type/insert whatever text you require. You can insert a PAGE field by pressing Alt+Shift+P.

If you want to insert a page number in a header building block that doesn't include one, place the insertion point where you want the page number and press Alt+Shift+P to insert a simple number or, if you want one of the fancy ones, use the Current Position setting for Insert Page Number and then choose from the gallery.

Code Snippet

To disable and turn off the SMS threading or the text messages conversation mode in Windows Mobile 6.1 device, simply install a Registry Editor (such as CoRegEditor or PHM Registry Editor, see also how to install PHM RegEdit in newer Windows Mobile system), and then navigate to the following registry key:

HKEY_CURRENT_USER\Software\Microsoft\Inbox\Settings\OEM\

Under the registry key, create a new DWORD value with value name as SMSInboxThreadingDisabled and set it value data to 1.

Alternatively, download and apply the following DisableSMSThreading.cab file (unpack from the ZIP archive) on the PPC Phone device to automatically install the required registry key. Note that the .CAB file also install the registry key to force enable the Sent SMS notification bubble.

Yes, if you sign up with Skype using your Microsoft account, your friends can find you using your display name, email or your Skype ID (live:xxxx). The steps to find your Skype ID (live: xxxx) are as follows:

  1. Sign in to the Skype website.

Once you're contacts on Skype, you can start making and receiving calls with them. You don't need a Skype Number to receive calls from your Skype contacts. You will only need a Skype Number if you intend to receive calls from phone numbers.
