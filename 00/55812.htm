Inaccessible boot drive occurs when your PC is unable to read the operating system installed on your hard drive. It is possible that some Windows system file became damaged or corrupted, leading to that error. To troubleshoot this issue using a USB bootable media, kindly follow these steps:

  1. Boot your computer using the bootable USB media.

    dism /image:C:\ /get-packages
