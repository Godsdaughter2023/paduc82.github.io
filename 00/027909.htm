hi josephpotthast,

it is not feasible to disable the quarantine feature in office 365.
however, based on your description, i understand that you want to avoid messages from some users are blocked in quarantine. 

if you want to add these recipients to your organization’s allow list, you can follow the steps below with an administrator account (if you are not an administrator, please involve your administrator to follow the steps below):
1. log into  exchange admin center, go to protection> spam filter.
2. on the general page, double-click the default policy in order to start editing it.
3. navigate to allow lists> sender allow lists. click add, and then in the selection dialog box, add the recipients you wish to allow. click ok to return to the allow lists page.
4. as the default spam filter policy is applied to every user in your organization, the goal is achieved after this.

also, if you change the spam and bulk actions to move message to junk email folder in exchange admin center > protection > spam filter, the message won't be moved to the quarantine.

please feel free to post back with the error message if you encounter any problems during the process or if anything is unclear. we are always here to help.
