Public Function ClipBoard_SetText(strCopyString As String) As Boolean
    Dim hGlobalMemory As Long
    Dim lpGlobalMemory As Long
    Dim hClipMemory As Long
      
    hGlobalMemory = GlobalAlloc(GHND, Len(strCopyString) + 1)
   
    lpGlobalMemory = GlobalLock(hGlobalMemory)
   
    lpGlobalMemory = lstrcpy(lpGlobalMemory, strCopyString)
   
    If GlobalUnlock(hGlobalMemory) = 0 Then
        If OpenClipboard(0&) <> 0 Then
            Call EmptyClipboard
            hClipMemory = SetClipboardData(CF_TEXT, hGlobalMemory)
            ClipBoard_SetText = CBool(CloseClipboard)
        End If
    End If
End Function

Opening a PST file will preserve all of these. That is why we do not advise

I have created a Project Timeline in Project 2013 that I need to save to a shared drive so the rest of the project team can view it. However, I do not want them to be able to edit it. Is there a way to save it as a "read-only" format so that they cannot edit it?

john - project say:

Alison,

Save the file with a write protection password and recommend as read only. Users will still be able to modify the file with a little nefarious manipulation but at least a warning will be prominently displayed upon file open.

How do you save a file with a write protection password? Go to File > Save As. Click on the "Tools" quasi-button at the bottom and select "General Options". That will bring up the Save Options sub-window as shown below. Set your options as desired.

The best protection is to insure you have the original file stored on your local hard drive. That way, if the shared file is "tweaked" by users, you can always use your backup.

Hope this helps.
