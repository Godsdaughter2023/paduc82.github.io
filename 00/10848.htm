First, just for the sake of accuracy - Office 2016 is a perpetual license and Office 365 is a subscription based product. The following answer applies regardless of which was installed/needs to be re-installed.

What you don't mention is whether or not the computer hard drive was re-formatted etc so for the sake of this answer, assumption is that you simply uninstalled and need to re-install Office.

You aren't going to lose anything. Information regarding your email accounts and data file is stored in the Outlook profile which persists any uninstall/reinstall. The exception to this is if you uninstall Office using the Microsoft <Easy Fix> uninstall tool which also removes all Outlook profiles. In that case, you will need to re-configure all your email accounts.

One last point about your AOL account. If the folder tree for this account contains folders with the words "This Computer Only" in the folder description (contact/calendar) AND those folders contain data that you want to keep then it's highly advisable to backup (export) those folders to avoid the possibility of losing that info. If Outlook recreates the OST file for any IMAP based email account, all data in "This Computer Only" folders will be lost.
