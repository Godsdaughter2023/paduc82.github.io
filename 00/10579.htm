Hi Aaron,

Do you mean this issue occurs when you send emails from Office 365 to external recipients?

By default, email messages that are sent from Exchange Online in Office 365 use the Transport Neutral Encapsulation Format (TNEF) . Messaging systems that aren't based on Microsoft Exchange may be unable to interpret messages that use this rich text format. If the recipient mail system can’t process this format, a file attachment that's called Winmail.dat is added to the message.

Therefore, this behavior depends on whether the recipient mail system can process TNEF.

On the Office 365 side, we can change the message format to prevent Winmail.dat attachments via PowerShell commands. For the detailed steps, see: support.microsoft.com/.../en-us.

I suggest you let the administrator try the commands. If this issue persists, we will help you do further analysis.

       The Fermata eye like symbol ( the password reveal button ) have been introduced in Windows 8/W7 IE10 . I believe this feature is mainly for the users who are using the Windows Tablets . It is quite difficult to retype to whole password if he made a mistake . So that I believe it should be mainly for the Windows 8 tablet users .  Using the feature a user wouldn't have to delete the entire password and start all over again if he believes that he has made a mistake 

      However, some desktop and laptop users might feel that this feature can come in the way of their security and in that case disabling it altogether would be the best option. So lets see how you can disable the reveal password button throughout Windows easily.

To remove the Password reveal option in Windows 7 . You must have to add the key in the Windows Registry to do this , you should follow my instructions.
Step 1 : Open Notepad .
Step 2 :Paste the contents in the notepad
                  
Windows Registry Editor Version 5.00

   [HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Internet Explorer\Main]
       "DisablePasswordReveal"=dword:00000001

Step 3 : Save it as DisablePasswordReveal.reg 

Step 4 : Double click or open the file , Accept it and restart the PC and check .
