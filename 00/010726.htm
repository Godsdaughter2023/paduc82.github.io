jay freedman say:
It appears that your workaround involves starting with the first structure under "Brackets with Separators" in the Bracket gallery, and building two separate matrices in the placeholders. The closest I could get to the desired appearance with that setup was to right-click an entry in the rightmost column, choose Matrix Spacing, and set the vertical spacing to Exactly 2.3 lines. That doesn't align the 1 and 0s with the centers of the fractions on the left, but it's somewhat better than the initial attempt.

Another way to try this is to start with a 5x4 matrix without the bar and enter all the numbers. That gets the alignment correct. Then use Insert > Shapes and use the line tool to draw the bar (hold down the Shift key to ensure that the bar is vertical). That looks better, but it has the problem that the line isn't part of the equation and can get pushed out of its position within the matrix.

There is a solution that really works, but it's labor-intensive. Start with the 5x4 matrix with the numbers inserted. Use the Windows 7 snipping tool, which should be on your Start menu, to grab a clip of the matrix, and copy that to the clipboard (it isn't necessary to save it as a file). Go to a new paragraph in the document and paste the picture of the matrix into the document; if it's in-line with text, change its text wrapping to Square. Use the line tool from Insert > Shapes to draw the bar over the matrix picture. Ctrl+click the picture of the matrix, so it and the bar are both selected; right-click and choose Group > Group. Change the text wrapping of the combined picture to in-line with text. Now you can delete the original equation and replace it with the picture.

macropod say:

I'm not familiar with the Visio object model, so I can't tell you how to go about updating the content. That said, the following code updates an embedded in-line Excel worksheet. Be warned: there is nothing 'quiet' about the process.

 

Sub OLEUpdate()
Application.ScreenUpdating = False
Dim objOLE As Word.OLEFormat
Dim objXL As Object
Set objOLE = ActiveDocument.InlineShapes(1).OLEFormat
objOLE.Activate
Set objXL = objOLE.Object
With objXL.ActiveSheet
  .Cells(1, 1).Value = 5
  .Cells(1, 2).Value = 4
  .Cells(1, 3).Formula = "= A1 * B1"
End With
' Deactivate object
objXL.Application.Quit
SendKeys "{ESC}"
Set objXL = Nothing: Set objXL = Nothing
Application.ScreenUpdating = True
End Sub
