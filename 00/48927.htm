I would check the status of Services. You are interested in Remote Procedure Call (RPC) and Windows Management Instrumentation (WMI) which should both be set to Automatic and have started. These are the services the Security Center needs to be able to start.

tryx3 say:

Patrick,

Check the Number lock setting in the BIOS. 

  1. Start to computer & while the Dell logo is displayed [before the Windows logo appears], press F2 to get into the BIOS.
  2. Look down through the BIOS entries - BIOSes vary but NumLock enable is probably in the group labelled POST behaviour. 
  3. Setting NumLock enable on in the BIOS should force it to be on at every start.

Denis

patrick423 say: We did this and the numb lock status was already set to "enabled". Still have to turn it on with login or restart. 

tryx3 say:

Patrick,

I can only suggest things you have probably already tried.

1  Use Safe mode to check whether it is Windows causing the problem or third-party software [Start your PC in safe mode in Windows 10 - Windows Help].  Then see if the NumLock is on or off. 

2  If it is Windows that is causing the problem -

2.1  Check that it is not your Mouse settings - Settings, Ease of access, Mouse, Mouse buttons.  Turn all three entries off [you'll need to turn the top one on for a moment to enable the other two so you can set them off].  This fixed the problem for at least one user - Windows 10 Number Lock Keypad Problem Fix.  These Mouse buttons settings might be the corresponding UI entries for the Registry key that you referred to before but not all Registry entries are what they seem to be so it is always worth checking the UI settings directly as well.

2.2  I think you could be wasting your time using this forum's search facility because there might not be an Answer there - number lock in windows 10.  If you do try searching then focus on threads with only a few replies as those with more than a dozen or so tend, in practice, to be less useful.  There are also links to some suggested threads near the top of this current webpage under the heading Related content. 

2.3  You could try asking Dell support but I have never found them to be any use.

3  Other than that, I can only suggest a workaround to automatically toggle the NumLock setting as you boot.  You might regard this as being more work than just remembering to press the button after booting though. 

3.1  Create a new file [text file] and copy in the following single line command

WScript.CreateObject("WScript.Shell").SendKeys "{NUMLOCK}"

3.2  Rename the file with a .vbs extension [such as ToggleNumLock.vbs]

3.3  Put the file away somewhere out of your way.

3.4  Create a shortcut to it.

3.5  Copy the shortcut to C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup

Best of luck,

Denis

A network bridge is software or hardware that connects two or more networks so that they can communicate. You can create only one network bridge on a computer, but one bridge can handle any number of network connections.

   
 

1.

Open Network Connections by clicking the Start button , clicking Control Panel, clicking Network and Internet, clicking Network and Sharing Center, and then clicking Manage network connections.

2.

Hold down CTRL and select each network connection that you want to add to the bridge.

3.

Right-click one of the selected network connections, and then click Bridge Connections.  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

You shouldn't create a bridge between an Internet connection and a network connection because it creates an unprotected link between your network and the Internet, which makes your network accessible to anyone on the Internet.
