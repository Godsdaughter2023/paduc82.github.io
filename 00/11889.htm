¡firedog say:

One or more folders in your mailbox have name conflicts with others or system reserved names

Technical level: Basic

Description

You are repeatedly receiving email messages from Outlook.com reading:

  
One or more folders in your mailbox have name conflicts with others or system reserved names.
Folders with these names can't be downloaded by your IMAP e-mail program.

[Names of the folders in conflict, most commonly Junk and Sent}

Please connect to your mailbox using your Web browser and rename those folders. Once renamed, those folders will be displayed in your IMAP e-mail application correctly.

  

Even if you follow the instructions to visit This occurs because the application you are using to access your Outlook.com account¹ is automatically creating folders, which are then uploaded to Outlook.com when the account is synchronized using the IMAP protocol. The Outlook.com account already has an IMAP folder for junk mail (Junk) and sent messages (Sent), so the conflict occurs. 


To complicate matters still further, Outlook.com may use a different name for the folders when you view the account using a web browser. This might be in the language you have specified for the Outlook.com interface, so for English language users, they're called Junk Email and Sent Items respectively, while Spanish language users will see Correo no deseado and Elementos enviados.

To prevent this from happening involves changing some of the advanced settings for the account in the application you're using. How to do this will be different for each application and for each version of the application, which may be one of these:

  • Apple Mail
  • Mac Mail
  • Gmail app on a tablet or smart phone
  • Manufacturers' apps on Android devices, e.g. Samsung Email

You will have to consult the help documentation for the one you're using to find out how to match (or 'map') the Outlook.com folders to the ones in the application. There are too many variants for it to be possible to give them here. You may be able to get the correct instructions by asking

 
How do I map the Sent folder at Outlook.com to the sent items folder in [insert the name and version of your email application]?
