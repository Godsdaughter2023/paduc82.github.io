Find out if your scanner uses TWAIN technology. TWAIN is an application programming interface (API) that is used to import images from scanners and digital cameras to your computer. Windows Fax and Scan use a different technology, called the Windows Image Acquisition (WIA) API, and do not work with TWAIN scanners. If your scanner uses TWAIN technology, you might need to use the program that came with your scanner instead of using Windows Fax and Scan.
