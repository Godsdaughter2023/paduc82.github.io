hansv mvp say:

Select rows 2 to 100 or however far down you want. I will assume that the active cell within the selection is in row 2.

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Select 'Use a formula to determine which cells to format'.

Enter the formula    =$F2="PAID"

Make sure that there is a $ before the column letter F but not before the row number 2. This is essential.

Click Format...

In the Font tab of the Format Cells dialog, select green as text colour.

Click OK, then click OK again.

mike h.. say:

Hi,

the formula

 =DATE(YEAR(B2),MONTH(B2),DAY(B2)+90)

adds 90 days and not 91.

It's easier to explain if we take a number smaller than 90 because we can then do the math in ours heads.

B2 = 1 Jan 2010

=DATE(YEAR(B2),MONTH(B2),DAY(B2)+10) adds 10 days to that and return 11 jan 2010 i.e 10+1=11

If this post answers your question, please mark it as the Answer.

hansv mvp say:

If you want to add B1 days to the date in cell A1, you can simply use =A1+B1.

By the way, =DATE(YEAR(B2),MONTH(B2),DAY(B2)+90) does add 90 days to B2, not 91 - you must have made a mistake while counting.

To add B1 months instead of days to A1, use =DATE(YEAR(A1),MONTH(A1)+B1,DAY(A1))
