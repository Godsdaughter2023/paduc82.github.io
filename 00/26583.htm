mike h.. say:

Hi,

Right click your sheet tab, view code and paste this in and try entering a duplicate code in column B

Private Sub Worksheet_Change(ByVal Target As Range)
Dim Num As Long
If Not Intersect(Target, Range("B:B")) Is Nothing Then
    Application.EnableEvents = False
    Num = WorksheetFunction.CountIf(Range("B:B"), Target.Value)
    If Num > 1 Then
    MsgBox Target.Value & " already used, select a different code."
     End If
    Application.EnableEvents = True
End If
End Sub

byhamc say:

You can actually achieve this by means of standard Excel Data Validation - without the need for macros.

Basically, select your entire column B. Then go to the 'Data' ribbon and select Data Validation.

On the Settings tab in the Data Validation pop-up, select 'Custom' from the Allow dropdown. In the Formula box, type =COUNTIF(B:B,B1)=1 (assuming you've selected the entire column B, cell B1 will actually be highlighted).

The Input Message tab can be used to display a message to the user when a cell is selected in column B - ignore the tab if you don't need it.

On the Error Alert tab, select 'Stop' in the Style box, then add a Title and Error Message (i.e "This code has already been used...."), and hit OK.

Now, every time you enter a value in column B, Excel checks the column for the number of repetitions of that value (using the COUNTIF function) and if that number is not equal to 1, your error message is displayed.

Whilst Macros are extremely useful, and I have more than my fair share of them in use on a daily basis, it's usually best to try and find a solution within Excel before hitting the VBA.

Hope this is useful.

You can refer to the following table.

Search Core Results Query string parameters

K Search keywords. This should be the URL-encoded value that the user typed in, including the full-text query and any managed properties.
S The scope to query. This is the scope from the scopes drop-down list. To specify multiple scopes to query, you can separate them with a comma (which when URL-encoded become %2c).

V The default ordering (or view). The valid options are relevance and date.
