a. user say:

Hi,

 

 

On the login screen below the password box click on Sign-in Options. You can view two option, one for password and one for Pin. Click on PIN and enter the pin. When you sign out and sign into the computer, by default Windows will remember the previous option which you have chosen to login to Windows (PIN).

 

But if you restart the computer then you have to manually choose the sign in options and there is no way to set sign using PIN as the default option. The reason behind this behavior is that picture password and PIN didn't work in Safe Mode. Thus there are some limitations in making picture password and the PIN as the default sign-in option.

Will I notice any differance between Home Premium 32 bit or 64 bit?

Interface wise, nothing, but under the hood they are a little different when it comes to capabilities. Well, its performance efficient on Netbook form factors compared to Windows Vista. Improvements such as smaller disk foot print, memory foot print.
