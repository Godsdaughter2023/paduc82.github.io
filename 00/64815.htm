There are many ways to do the same basic task in Windows 8 we have been doing for many years. Creating a user account is one of them and today, we take a quick look at how to create a user account.

On your keyboard, press Windows Key + C

Click Change PC Settings

Under PC settings, click Users

Under Other Users, click Add a user

The type of user you will want to create will be dependent on if you will want to have seamless integration with online services such as Skydrive, Hotmail and other services. Syncing is another component of user accounts in Windows 8, with a Microsoft Account, you will be able to do this across all your Windows 8 PC‚Äôs, this includes syncing themes, app settings, and some passwords. A Microsoft Account also lets you download apps from the Windows Store. If you are not planning on using these services, then its best you use a Local Account. As the name suggest a local account stores data only on the PC where it is created. A local account works just like a traditional Windows account we have been using in Windows 7 and prior versions of Windows. It does not integrate with online services and cannot be synchronized across PC‚Äôs.

For the purposes of this review, I will only be creating a traditional Windows Account or what is known as a Local Account.

Click the link Sign in without a Microsoft account

Click Local account

Enter the User name, account, confirm it and create a hint just in case you forget. Click Next after completing these steps.

If you would like to enable Parental controls on the account you just create, check the box, otherwise, click Finish.

Thats it!

 

If you want to switch it to a Microsoft Account, do the following:

1) Just move your mouse to the bottom-right corner of the screen till the side menu pops up, and click on Settings (gear icon).
2) From there, click on the "More PC settings" link on the bottom-right.
3) Now you should be on the PC settings full-screen menu. On the left, go to the "Users" section.
4) On the right side, it will show your current account with a button that says "Switch to a local account". Clicking on this button and following the steps will enable you to be able to log into Windows as you do with all other versions of Windows like Windows 7, Vista, XP, etc.

You can also change your password via the corresponding button from the same Users screen.

"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Windows 8 will be reinstalled, and you'll need to change any settings and reinstall any apps that you want to get back after resetting. If you know if any files that you don't want to lose, then you should back those files up before starting the reset"

Windows Movie maker

throughout. I transferred it to Windows Movie Maker and am confused. How do I remove a section, i.e. .55-1.09, and piece it together without it.

Please advise.

considerate_guy say:

Set your playback marker to 0:55 and make a split. You can either use the menus Video Tools | Edit > Split or you can use the shortcut of pressing M on the keyboard.

Repeat the process but with your marker at 1:09 and make another split.

You should now have 3 clips showing in your storyboard area. You want to remove the middle clip, so select it and then press Delete.

Preview the revised version of your movie and then if you want to save a new movie with these edits, you'd use File > Save movie. Give it a new name and Movie Maker will create a new .mp4 (or .wmv) file for you.

The program worked in earlier versions of windows but won't install or run now.
6 In the Which version of Windows did this program work on before? window select Windows 7.
7 When you click Test the program, it installs and works perfectly.
