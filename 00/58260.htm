    ¡firedog say:

    I use a Gmail account through windows live mail.

    I am unable to make folders in windows live mail. 


    Gmail doesn't use folders. It uses labels instead, so any message can have a number of labels. The default setup for Gmail accounts in Windows Live Mail is IMAP, so what you see in folders in the client is a mirror image of what you see under labels in webmail. You can choose which Gmail labels to be available in WLMail.

    I want messages to come into Gmail and windows live mail.
    I wish to delete in either account without losing the email in both accounts.


    All messages sent to your Gmail address will arrive at Gmail. Using IMAP, they will also all arrive in Windows Live Mail. Because IMAP synchronizes Gmail labels with WLMail folders, a message deleted in the client will be deleted in webmail and vice versa. If you want to retain a message in the client but remove it from the server, move it to a folder under Storage folders. If you want to retain a message in Gmail but remove it from WLMail, apply a label that is not synchronized.

    In windows live mail when I delete in my inbox my emails go in duplicate to unread mail.


    I'm not quite sure what this means.

    Sent mail goes into all mail as well.


    All mail is included in the All mail label. It's not usually a good idea to synchronize the All mail label, because you will see duplicates. For example, any message in the inbox will also appear in All mail.

    I also want messages to come into my desk top computer as well as my laptop computer.


    Use IMAP in both places to see the same on both machines.
