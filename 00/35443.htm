andre da costa say:

KMS activation is only available to volume license customers who purchase Windows in bulk for large deployments of Windows in organizations. This requires that a company set up a KMS host which handles all the activation request for machines in the organization. KMS activation requires that the organization meets a certain thresh hold to have additional machines start activating, such as 25 client machines minimum.

 

Basically, this is not for the average user. If you are a single user, the off the shelf single license is less intimate. Simply install it and the minute you connect to the Internet, it will activate automatically.

 

There are nefarious individuals who setup illegal KMS host, but they can put your computer at risk and Microsoft is shutting them constantly.
