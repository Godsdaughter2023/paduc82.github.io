Hi Matthew,

I’d like to firstly explain the following things for you:

1.There are two versions of OneDrive for Business sync clients.

One is the legacy OneDrive for Business sync client which is normally in the Office suite. It also has stand-alone Click-to-Run version. It’s Groove.exe.

The other one is the new sync client of OneDrive for Business which is OneDrive.exe. It has more good features. We suggest you use the new sync client.

Note: The new sync client can only be used to sync personal OneDrive for Business libraries. To sync SharePoint Online team site libraries, we need to use its preview build.

2.To identify these two sync clients, see Which version of OneDrive am I using? The sync client which asks for URL is legacy OneDrive for Business. On the other hand, the sync client which asks for accounts is the new sync client.

4.There are two kinds of installations for Office suite. One is MSI installation and the other is Click-to-Run installation. Some scenarios of them cannot be installed on the same computer. That is the reason when you want to install OneDrive for Business, you get the error message “cannot install OneDrive For Business 2016 because the 2013 version is installed”.

4.On the OneDrive screen, under Setting, click Prevent the usage of OneDrive for file storage.

5.In the Prevent the usage of OneDrive for file storage box, select Disabled, then click OK. (Make sure here Disable is selected.)
