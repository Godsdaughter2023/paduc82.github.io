a. user say:

Happy Fourth of July! For me this is the 21st Independence Day since I became a citizen.

Doing something just because that's the way you always did it is not always the best advice, and that's particularly true for technology, which changes so much and so quickly.

The original idea behind the recovery drive was to provide a means to fix a computer when it won't boot into Windows. If a computer can boot into Windows, you can take advantage of Windows' own tools to (hopefully) fix problems. If you can't boot into Windows, you can't take advantage of those tools, obviously.

A Windows recovery drive has two things: 1) a bare bones version of Windows that has just enough to provide the most basic elements of the Windows operating system; and 2) a set of tools that will function in that bare bones version of Windows which might make it possible to boot into Windows again or, if that doesn't succeed, will make it possible to perform some basic repairs or, as a last resort, to re-install Windows.

A Windows recovery drive is far from the only type of recovery drive. There must be thousands of different types of recovery drive which have different types of tools for different types of problems. And because you're booting the computer from the recovery drive, they can use any operating system. Linux is popular.

Back to the topic: A recovery drive only gives you the opportunity to use the recovery tools on the drive. Those tools may or may not help. And if the problem that's plaguing your computer isn't addressed by those recovery tools, they're completely irrelevant. That's why the ultimate recovery tool in a recovery drive - the last resort - is a tool that reinstalls Windows.

There's no guarantee that reinstalling Windows will help - it may make things worse - and you have to make sure to backup your personal files before you start (if you know how) so you won't have to worry that they will be lost by the reinstall, which admittedly is not likely but possible. And if the reinstall doesn't work you'll have to do a clean install, which erases your hard drive and installs a fresh copy of Windows, meaning you'll have to setup your computer from scratch all over again.

You'll be spending a lot of time with no guarantee of success, and if you're not totally sure of what you're doing, you could make matters much worse.

Compare that to the simplicity and elegance of a third-party image backup tool. (I use Macrium Reflect but there are lots of other good choices.) These programs have their own recovery drives. To restore an image, I boot the computer from Reflect's recovery drive, navigate to the backup I want to restore, and start the procedure. The whole adventure is over in about 15 minutes and my computer goes back in time to when the image was made, and life is good (for my computer).

I'm not exactly proud to say this, but I've had plenty of occasions to restore backups, thanks to my 'experiments' with my computer, so I know it works.

For the record, one of the recovery tools on a Windows recovery drive is a tool that lets you restore a backup image made with Windows 10's own System Image Builder. But Microsoft deprecated (= abandoned) SIB with the Fall Creators Update, which means it's not being developed any more and might be removed in the future. Microsoft recommends instead that people use a third-party image backup program.

Change Wireless Mode - Microsoft Community

xb1r apollo say:

Change Wireless Mode

Login Information

 

  • Gateway: 192.168.1.1
  • Username: root
  • Password: 12345

 

 

Change Wireless Mode

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Advanced" from the sub menu.
  6. Select the desired wireless mode from the "54g Mode" drop down menu.
  7. Select Save/Apply.
