graham mayor say:

It shouldn't be necessary to resize the image, as long as the aspect ratio of the image is correct, as the table will have fixed width cells and the image will adapt to that width, so you just need to ensure that you don't put images in the spacer cells e.g.

Option Explicit

Sub Macro1()
Dim oDoc As Document
Dim oTable As Table
Dim strPic As String
Dim oCell As Cell
Dim oRng As Range
    strPic = BrowseForFile("Select the image to insert")
    If strPic = "" Then
        MsgBox "Cancelled"
        GoTo lbl_Exit
    End If
    Set oDoc = Application.MailingLabel.CreateNewDocumentByID(LabelID:="1359804843")
    Set oTable = oDoc.Tables(1)
    For Each oCell In oTable.Range.Cells
        If oCell.Width > CentimetersToPoints(0.5) Then
            Set oRng = oCell.Range
            oRng.End = oRng.End - 1
            oRng.InlineShapes.AddPicture strPic, False, True
        End If
    Next oCell
lbl_Exit:
    Exit Sub
End Sub

Private Function BrowseForFile(Optional strTitle As String) As String
'Graham Mayor - http://www.gmayor.com - Last updated - 16/04/2017
'strTitle is the title of the dialog box
Dim fDialog As FileDialog
    On Error GoTo Err_Handler
    Set fDialog = Application.FileDialog(msoFileDialogFilePicker)
    With fDialog
        .Title = strTitle
        .AllowMultiSelect = False
        .Filters.Clear
        .Filters.Add "Graphics Files", "*.jpg,*.png,*.gif, *.tif, *.bmp"
        .InitialFileName = Environ("USERPROFILE") & Chr(92) & "Pictures"
        .InitialView = msoFileDialogViewList
        If .Show <> -1 Then GoTo Err_Handler:
        BrowseForFile = fDialog.SelectedItems.Item(1)
    End With
lbl_Exit:
    Exit Function
Err_Handler:
    BrowseForFile = vbNullString
    Resume lbl_Exit
End Function

patc---01 say: One thing: you're probably going to have similar issues everytime you upgrade PC Suite or NSU (Update Phone Software).

Cleaning up the registry is optional and isn't usually necessary. The following process usually works:

- uninstall PC Suite (via Control Panel Add or Remove Programs)
- uninstall Nokia cable drivers (via Control Panel Add or Remove Programs)
- uninstall NSU (via Control Panel Add or Remove Programs)
- (optionally) run the clean up tool
- (optionally) remove any Nokia-related directories
- (optionally) clean up the registry
- reboot the PC
- download and install the latest versions of PC Suite and NSU
- after connecting the phone wait for ALL drivers to install
- reboot the PC

Everything should then work. The two reboots do seem to be necessary, but you should only need a warm reboot, ie: Restart from the shutdown menu.
