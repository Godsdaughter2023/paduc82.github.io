Outlook attachment file name. - Microsoft Community

babenewyear say:

Outlook attachment file name.

How do you configure outlook to save the file name of an attachment without actually resending the attachment.  So basically, in the email chain you can see what file was previously attached. How do you configure outlook to save the file name of an attachment without actually resending the attachment.  So basically, in the email chain you can see what file was previously attached.
graham mayor say:

There is no built-in function that would allow this, but you could arrange something with an Outlook macro e.g use the following to create the reply:

Sub ReplyListingAttachments()
Dim objItem As Outlook.MailItem
Dim oMail As Outlook.MailItem
Dim olInsp As Outlook.Inspector
Dim olAtt As Outlook.Attachment
Dim strAttachment As String
Dim wdDoc As Object
Dim oRng As Object

Dim i As Long
    Set objItem = ActiveExplorer.Selection.Item(1)
    strAttachment = vbNullString
    For Each olAtt In objItem.Attachments
        If Not olAtt.DisplayName Like "image*.*" Then
            strAttachment = strAttachment & "'" & olAtt.DisplayName & "', "
        End If
    Next olAtt
    If Len(strAttachment) > 2 Then
        strAttachment = Left(strAttachment, Len(strAttachment) - 2)
    End If
    Set oMail = objItem.Reply
    With oMail
        .BodyFormat = objItem.BodyFormat
        Set olInsp = .GetInspector
        Set wdDoc = olInsp.WordEditor
        Set oRng = wdDoc.Range(0, 0)
        oRng.Text = "Hi " & _
                    Left(objItem.SenderName, _
                         InStr(1, objItem.SenderName, Chr(32))) & vbCr
        oRng.collapse 0
        If Len(strAttachment) > 2 Then
            If InStr(1, strAttachment, Chr(44)) > 0 Then
                oRng.Text = "With reference to your attachments: " & strAttachment & vbCr
            Else
                oRng.Text = "With reference to your attachment: " & strAttachment & vbCr
            End If
        End If
        oRng.collapse 0
        oRng.Select
        .Display
    End With
    Set objItem = Nothing
    Set oMail = Nothing
End Sub
