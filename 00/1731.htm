How do I Insert a long division symbol in Word 2003? - Microsoft Community

jl 60 say:

How do I Insert a long division symbol in Word 2003?

I'm a teacher and I am struggling to do this for a worksheet.

Thanks

suzanne s. barnhill say:

You can do this with the Equation Editor.

  1. To open the Equation Editor, use Insert | Object. On the Create New tab, select Microsoft Equation 3.0.
  2. This will open a box for the Equation Editor object and display the Equation Editor toolbar, which has a selection of different "templates."
  3. The second button on the bottom row is "Fraction and radical templates." Click on that.
  4. The bottom two buttons are for long division with and without a solution. Click the appropriate one.
  5. This will place the division symbol in your document with boxes for you to enter numbers for dividend and (if desired) quotient. (I'm not sure why there's not a box for the divisor; I guess you just place that to the left of the Equation Editor object.)
  6. When you have entered the desired numbers, click outside the box to return to your document.

If you will be doing this often, you'll want to add an Equation Editor button to one of your toolbars. You can find it in Tools | Customize | Commands in the Insert category. Scroll down to Equation Editor, select it, and drag it to a toolbar. Then just click on that button when you want to insert an equation.

don cuthbert say:

When I have used gatherosstate.exe, I just copied this EXE from the Windows 10 ISO to a folder on the desktop and ran it there.  It generates the XML file in the directory it is run from, so it needs to be located somewhere you can write to.  A USB flash drive containing the Windows 10 install media would be ideal, as you could just copy gatherosstate.exe to a folder on the USB and run it there, generating the XML file from your running OS (Windows 7 or 8) and making it available right on your Windows 10 install media.  After a clean install of Windows 10 you could then copy that XML file to the C:\ProgramData\Microsoft\Windows\ClipSVC\GenuineTicket\ folder to activate the clean install of Windows 10. 

When I tried this activation method, I copied the file there and restarted and found the file vanished, but Windows 10 was not activated yet.  I then copied the file there again and restarted and this time Windows 10 was activated.  Afterwards, I did a normal clean install of Windows 10 by first wiping the hard disk of all partitions and using no product key or file to prove that the computer was truly now recognized by the activation servers and would auto-activate.

Not sure if the XML file always needs a couple of tries, but it is a good file to have available.  You could even create one now, for use if you ever lose activation in future on that computer.
