Hi Calvin. I'm Greg, an installation specialist and 9 year Windows MVP here to help you.

There are possibly two things going on here. First is that there's a bug causing Pro Upgrades to report Enterprise instead. So far it's resolved only by contacting Microsoft Support by one of these methods: https://partner.support.services.microsoft.com/...

Then there is the issue of moving a Pro license from one OEM PC to another. Is this a retail full or Upgrade pro license which can always move as long as it's only on one PC at a time? If so then you're good.

But if it's an OEM PC that came with Pro from the factory, you'd need to use the Activation troubleshooter and try to move it using the procedure to reactivate after changing hardware here: https://support.microsoft.com/en-us/help/20530/....

Over the past few years and for the first time ever they made an exception for OEM allowing these licenses to move, too, using the troubleshooter or using Support. But there are recent reports that this has been cut off. So you'll need to find out if that is your case.

So I would contact Support and please report back what they tell you.
