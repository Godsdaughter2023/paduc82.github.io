wunders say:

These files are being encrypted by the Encrypting File System (EFS).  This would indicate that you have an XP Pro and not XP Home.  When it comes to EFS, administrators have little to no power to decrypt these files.  Even though you are administrator, you will not be able to access these files.

Looking forward...  One way you can gain access to encrypted files is to create a "EFS Recovery Agent".  You can then use this recovery agent to access any file encrypted after the agent was installed.  You will find that only files encrypted after the recovery agent is in place can be accessed.  Files created before then will not be accessible.

"How to add an EFS recovery agent in Windows XP Professional"
  < http://support.microsoft.com/kb/887414 >
"How to remove file or folder encryption in Windows XP"
  < http://support.microsoft.com/kb/308993 >
... and of course, the Mandatory reading for anyone using EFS:
"Best Practices for the Encrypting File System"
  < http://support.microsoft.com/kb/223316 >

You can also disable EFS.  You can do this with Group Policy in a domain environment, but at home, a registry value can be changed.  Use with caution.  If you don't understand the following or how to edit the registry, don't do it.

< Quote >
"Note: Group Policy sets a registry key which is checked by EFS during user operations. The key is:
HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows NT\CurrentVersion\EFS\EfsConfiguration
In the case of local computers that are not members of a domain, local policy is not available for disabling EFS.
However, a different registry key may be set to disable EFS. If the key is set to a DWORD value of 0x01, EFS will be disabled.
Registry key:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\EFS\EfsConfiguration "
< /Quote >

As found in:
"Encrypting File System in Windows XP and Windows Server 2003"
  < http://technet.microsoft.com/en-us/library/bb457065.aspx >

If it is of any consolation, EFS is known as the "Delayed Recycle Bin" because if a user does not back up his certificate (as most do not), any crash of Windows requiring re-installation of Windows will permanently remove access to these files to everybody.

I hope this helps,
   JW

frederik long say: Let's try and keep this simple:
  • The command chkdsk /? will tell you what options there are for the command.
  • The command chkdsk c: /F will check and fix the file system on drive C:.
  • The command chkdsk c: /R will check and fix the file system on drive C:. It will also perform a physical disk check by locating bad sectors.
  • If your installation is damaged then you can run chkdsk like so:
    1. Keep tapping F8 at boot time.
    2. Boot into Repair Mode.
    3. Open a Command Prompt.
    4. Run your chkdsk command.
  • If this still does not work then you can run chkdsk like so:
    1. Burn yourself a Windows Repair CD via the Control Panel / Backup and Restore.
    2. Use this CD to boot the machine into Repair Mode.
    3. Run chkdsk as above.
  • A destructive Factory Restore will destroy all your data and remove all applications.
  • Performing a destructive Factory Restore may or may not work with a damaged disk.

The only thing that needs to happen during reactivation is you read out the confirmation ID to the call agent when you use phone activation. Since OEM System Builder Windows 8 carries transfer rights, that should not be a problem.


How To Activate Windows 8

Windows Product Activation (WPA) sends an installation ID number to Microsoft before Windows is verified and then activated.

One is an Installation ID, generated by entering a Windows 8 product key, that the computer owner submits to Microsoft, either over the Internet or by phone.

The second is a Confirmation ID, assigned by Microsoft, which is used to activate the operating system’s software.

Activate Windows 8 Using Internet

The easiest way to activate your Windows 8 is to activate it over the Internet. To start the Product Activation Wizard, follow these steps:

1. At the Start screen, swipe in from the right side, or hover your mouse in the upper-right corner of the screen to display the charms menu.

2. Tap or click the Settings charm.

3. Tap or click Change PC Settings.

4. Select Activate Windows (If your Windows is activated successfully, you will not see Activate Windows in the PC Settings)

5. Tap or Click Activate.

6. Enter Product Key.

The product key should be printed on the box that the Windows 8 DVD came in or included in an email you received after buying Windows.

7. Windows will now automatically detects whether the product key is valid. If your product key works, click Activate .

8. If Windows Activation is successful, click Close.

If activation was not successful, check the key and try entering it again.

Activate Windows 8 Using Phone

If your are unable to activate Windows 8 over the Internet then you can contact Microsoft by phone to activate the product. You do not need Internet connection to activate Windows 8 over phone.

To start the Product Activation Wizard, follow these steps:

1. At the Start screen, swipe in from the right side, or hover your mouse in the upper-right corner of the screen to display the charms menu.

2. Tap or click the Settings charm.

3. Tap or click Change PC Settings.

4. Select Activate Windows

5. Tap or click Activate by phone.

6. Select the country/region from the drop down menu, and then click Next .

7. Call the toll free number provided.

8. Follow the instructions from the telephone system and write down the confirmation ID.

9. Enter the confirmation ID numbers into fields A through H, in the same sequence as the telephone system provided them. When done, click Activate .

If product activation was successful, a success screen is shown and you are done.
