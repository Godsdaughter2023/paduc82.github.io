Try removing any external devices connected to the computer and check if you can boot to the computer.

  1. Boot into the Windows 8 disc. ( If you did not make one, you can find the ISO file download to create a disc at the Create installation media for Windows 8.1 page. You will have to use another computer to do so. Make sure you create the appropriate version disc, x86 or x64, depending on your current OS build)
