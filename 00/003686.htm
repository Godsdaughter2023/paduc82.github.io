john p spencer say:

First is simple.  Add a textbox control to the report in the PartNum footer and set its source to

   =Avg(DateDiff("d",[OrdrDate],[RecvdDate]))

The second is more complex since you are working between records.  It might be easiest to get the prior date in the source query. 

SELECT A.PartNum, A.SupplierNam, A.OrdrDate, A.AmtOrdrd, A.RecvdDate, A.AmtRcvd
,  Max(B.OrdrDate) as PriorDate
FROM tblOrders as A LEFT JOIN tblOrders as B
ON A.PartNum = B.PartNum
AND A.OrdrDate < B.OrdrDate
WHERE A.PartNum = [Enter Part Number]
GROUP BY A.PartNum, A.SupplierNam, A.OrdrDate, A.AmtOrdrd, A.RecvdDate, A.AmtRcvd

With that modified query, you could then use
    =Avg(DateDiff("d",[PriorDate],[OrdrDate]))

You could also try the following
    = Avg(DateDiff("d",DMax("OrdrDate","tblOrders","PartNum='" & [Partnum] &"' AND OrdrDate<#" & [OrdrDate] & "#"),[OrdrDate]))

How do I enable pop-ups for a particular site? - Microsoft Community

carol's pups say:

How do I enable pop-ups for a particular site?

How do I enable pop-ups for a particular site?

a. user say:

Hi,

To better assist you, we would like to know which browser you are using. If you are using Internet Explorer, we suggest you to follow the steps and check if that helps.

  1. Click the gears icon at the top-right of the window and choose "Internet options".
  2.  When the "Internet Options" dialog box opens, click the "Privacy" tab.
  3.  Make sure "Turn on Pop-up Blocker" is checked, then click the "Settings" button. The "Pop-up Blocker Settings" dialog box will appear.

Type in a website address underneath "Address of website to allow". Only enter the host name, not the full path, i.e.microsoft.com. Click the "Add" button.

Also refer: http://windows.microsoft.com/en-IN/internet-explorer/ie-security-privacy-settings#ie=ie-11

Let us know if the above solution helps. Give us an update.
