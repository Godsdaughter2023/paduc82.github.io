cliff gu msft say:

Dear Juan,

 

Thanks for your updates.

 

After investigation, I’d like to confirm whether you create the list directly in SharePoint Online and fill values in the list by users or export the list to SharePoint Online in Excel 2016 via clicking on Table Tools>Design>Export>Export Table to SharePoint list.

 

If the list is exported from Excel 2016, the filter search box may be not available. If you want the option in the Text column, I suggest you export a .csv file to SharePoint Online by running the following script in SharePoint Online Management Shell (SharePoint Server 2013 Client Components SDK).

#Load SharePoint CSOM Assemblies

Add-Type -Path "C:\Program Files\Common Files\Microsoft Shared\Web Server Extensions\15\ISAPI\Microsoft.SharePoint.Client.dll"

Add-Type -Path "C:\Program Files\Common Files\Microsoft Shared\Web Server Extensions\15\ISAPI\Microsoft.SharePoint.Client.Runtime.dll"

 

##Variables for Processing

$SiteUrl = "xx"

$ListName="xx"

$ImportFile ="xx"

$UserName="xx"

$Password ="xx"

 

#Setup Credentials to connect

$Credentials = New-Object Microsoft.SharePoint.Client.SharePointOnlineCredentials($UserName,(ConvertTo-SecureString $Password -AsPlainText -Force))

 

#Set up the context

$Context = New-Object Microsoft.SharePoint.Client.ClientContext($SiteUrl)

$Context.Credentials = $credentials

 

#Get the List

$List = $Context.web.Lists.GetByTitle($ListName)

 

#Get the Data from CSV and Add to SharePoint List

$data = Import-Csv $ImportFile

Foreach ($row in $data) {

    

    #add item to List

    $ListItemInfo = New-Object Microsoft.SharePoint.Client.ListItemCreationInformation

    $Item = $List.AddItem($ListItemInfo)

    $Item["Column1(replace it with your list column name"] = $row.Column1

    $Item["Column2"] = $row.Column2

    $Item["Column3"] = $row.Column3

    $Item["Column4"] = $row.Column4

    $Item.Update()

    $Context.ExecuteQuery()

   

}

Write-host "CSV data Imported to SharePoint List Successfully!"

Please  check the information above and welcome to share any updates at your convenience.

 

Best Regards,

Cliff
