oldpcguy_db say:

I have solved the issue, though it was not at all obvious.

This morning I found the following: Office 2016 Error Opening File.   It pointed to the DisplayLink driver as the culprit.

On my system, I use a Toshiba Dynadock.  DisplayLink software is used for this docking station.  The DisplayLink Core Software driver was at version 7.9 but the DisplayLink Graphics driver was at version 7.6.

I went to the DisplayLink Downloads page, downloaded and installed the current Windows 10 DisplayLink driver ( Core Software: 7.9.1488.0  Graphics:  7.9.1505.0)

The protected view issue is now fixed.  I can open Office files in protected view and I can preview the files within Outlook
