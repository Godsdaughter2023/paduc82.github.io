The size limit of the Unicode.pst (Outlook 2003 and later) has been increased up to 20GB compared to the older format of outlook data file ANSI .pst (Outlook 2002 and earlier).

http://support.microsoft.com/kb/291602

You do not have to change any registry value to increase the size of the pst file because by default Outlook 2003 and outlook 2007 can store data up to 20GB and for Outlook 2010 it is 50GB.

damouk24 say:

Windows Insider

What's the benefits of joining Windows Insider, and can it damage your computer??
dan-at_work say:

You get access to Insider Preview builds of Windows, and a cute little Insider badge under your name on the forums. Being an Insider means that you are expected to provide feedback for those builds. Suggestions to make it better, or what broke when you did an upgrade.

Depending on how you configure your Windows Updates, you'll be in a perpetual state of beta testing.

Can it damage your computer? It all depends on what you mean by that. It shouldn't hurt the hardware, but I've read some interesting stuff here. Some builds have failed to install drivers, or installed the wrong drivers for various accessories - sound cards, video cards, etc.

Before you install an Insider Preview version, you'll want to make sure you have a good backup on your system - documents, favorites, etc. I've had times where my documents and favorites went missing after an upgrade. Full disclosure - that was during the testing phase (Oct 2014 - Jul 2015).

Also, before you upgrade to another build, or install any updates, make sure you have a good backup.

Other than that, there should be nothing to worry about. You'll always have the latest version of Windows.

I only run Windows 10 Insider Preview on a test PC. I use the Enterprise LTSB version for other work.

neftali ramirez say: Lets try to completely uninstall OneDrive:
1. Open Command Prompt in Administrator mode: Right-click on the Windows icon in the taskbar and select Command Prompt (Admin).
2. Type in taskkill /f /im OneDrive.exe to terminate any OneDrive processes and hit Enter.
Then type in either %SystemRoot%\System32\OneDriveSetup.exe /uninstall if you’re using 32-bit Windows 10 or %SystemRoot%\SysWOW64\OneDriveSetup.exe /uninstall if you’re using 64-bit Windows 10 and hit Enter.


After that Go to RegEdit on this path: HKEY_CURRENT_USER\Software\Microsoft\OneDrive
You can remove all folders that include One Drive version
Account folders also can be removed
After that restart the pc and install One Drive again
