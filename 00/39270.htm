Unknown Device on Network - In This Case, Bonjour

So, there are numerous threads on this subject and I don't believe it can answered in a way that fits every possible scenario.  There is no cure-all answer because there are so many variables. 

Recently, I found an answer to a strange device that is presently on my network, but didn't list a MAC address in my Check Point 730 NGTP's network object table. 

I'm not an Apple person, but I did give up on Android in favor of the iPhone.  Hence, I'm taking hard lessons on how Apple intrudes into everyone's space. Also, because I recently deployed a home automation system running on SmartThings, I've been a bit sloppy and have everything on the same subnet.  I will be changing this very soon.

I was able to identify this unknown network device by installing the Apple Store app, Network Analyzer (how original...), on my iPhone and the object was shown to have Bonjour services. 

Within iOS, Bonjour has created it's own network object and is running a base of 3 services:

  • _sftp-ssh._tcp
    • Secure File Transfer Protocol over SSH (via port 22)
  • _ssh._tcp
    • SSH Remote Login Protocol (via port 22)
  • _workstation._tcp
    • Workgroup Manager (via port 9) - WTH; the audacity!


Furthermore, because I have WD MyCloud and WDTV devices on my network there were 2 more services running.

(I killed my WD NASes due to Western Digital being unwilling or unable to keep the backdoor shut in their code - which is well-documented on the web).

  • _wd_tv_webremote
  • _wd_2go

I found my iPhone IP connecting to both WD devices every few minutes, as well as attempting to reach an external url that my Check Point filtering rules have blocked as malicious.  My hunch is that Bonjour was hacked by an offending iPhone app to access the existing hack in the WD NAS devices, and possibly I haven't run that app since I reset the iPhone's network settings.  I really don't have the time or expertise to work through it, so I'm just going to buy a non-WD NAS.

All that said, I still have an active network device created by Bonjour.  I'm getting ready to block it's IP address on internal traffic and see what turns up in my Check Point security log.
