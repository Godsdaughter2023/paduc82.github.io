jenicel dri say:
Hi Marcus,


Assuming you can print from other Office apps like Word, a plain text document from Notepad prints okay, and you can print a picture/graphic without issue, here are some things that you can check:

1. When you do a print preview you are seeing the whole worksheet as expected as what was shown in the screenshot, then perhaps you have a faulty printer cartridge or printer cable.

2. Make sure you have a default printer selected.  Excel has some problems rendering a worksheet if there isn't a default printer selected.

3. Sometimes when something doesn't print, it can be the printer driver is corrupted or even the video driver. We suggest you delete the old printer and reinstall the printer driver. Make sure you are also using the latest video drivers.  Video problems can prevent a document from printing since the video layer is used to render the print job before it is sent to the printer.

4. Check that your print area is set as you expect.  Go to the View tab, and in the Workbook Views section, select Page Break Preview.  You'll see the worksheet broken down into pages.  You can adjust what prints on each page by adjusting the page breaks.  Also, check Print Layout to verify your page layout is what you are expecting.

Note:  If you try to go into Page Break Preview and you don't have the default printer setup, then you'll get an error and will need to select a printer.  Make sure your default printer is the printer you want to print to and not an old printer that is no longer available.

Hope one of those ideas help you out.

glenkilgore say:

Go to C:\Users\USERNAME\AppData\Roaming\Microsoft\Spelling\en-US\custom.dic

Note: Your user name is different than mine because we have different names.

Right click on custom.dic and open custom.dic with Notepad.

You will see a list of words you added.  Edit that list and save and close the custom.dic file.

Note: 1033 is a Microsoft label for the English language.  Don't change that, anything below that label is your list of custom words.

Scroll to the bottom you may see a second list of 1033 words and those are from OneNote, the first list is from Office. 

Edit the list as you see fit.

You are now good to go!
