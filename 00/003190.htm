Hi Dhiraj,

Yes. You can use them for your team site.

If you set up Site Collection Storage Management to Automatic, you don't have to worry about the storage management of your team site. Site collections use what they need when they need it, up to a maximum of 25 terabyte (TB) per site collection.

ken sheridan say:
Does this mean that if I add a field to the Customer table, then I cannot import the data for that new field?

You can do it, but not by overwriting the table, or at least not easily.  As John points out, importing the data en bloc won't work as that necessarily deletes the existing data before replacing it, which the relationship prevents.  If, having created a link to the Excel worksheet or having imported it into a separate temporary file, you update the existing table with an update query the existing data is preserved, with the only the new column's values being updated by being filled with the new data in the corresponding column in the Excel worksheet.

It might be possible to do it by importing the data en bloc, but it would involve first deleting the relationship, then undertaking the import, and then re-creating the relationship, but this assumes that the primary key values of the imported table precisely match those of the existing table before the import.  If they don't then it would not be possible to re-create the relationship because of mismatches between the referenced and referencing tables.  It's also a laborious way of going about it compared with using an update query.

The first thing we have to do is make sure your installation is up to date on both Windows and Mac.

In Windows, make sure Check for Updates is set so that it also updates Microsoft products. That's done in Windows Settings (aka Control Panel). I don't have Office 2019 for Windows handy right now, so I don't know the current version information that you should expect to see.

INSTALL UPDATES

Make sure your Mac has a working internet connection before checking for updates. Install Mac OS X updates followed by Office updates.

Install Mac OS X updates first:

To get Mac OS updates, go to the App store on your dock, and then click the Updates button to reveal available updates.

Then Install Office Updates

To get the update:

1. Open any Office application

2. Go to the Help menu and choose Check for Updates

3. Let AutoUpdate check for and install updates

4. Restart your Mac

If you have any add-ins such as WebEx, Mendelay, EndNote, Zotero, TypeIt4me, etc it is essential that you install the latest updates for your add-ins. This includes font management add-ins, such as Suitcase. If you use Suitcase, you should have at least version 9.

Your copy of AutoUpdate may be too old to run in Mojave. If you were told no updates are available, you will have to install the current version of AutoUpdate. Install the latest version of AutoUpdate and when it runs, let it do its thing.
