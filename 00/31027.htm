Please do the following:
1. Restart the PC.
2. After BIOS press (and hold down) the F8 keyboard key.
3. Select Safe Mode with Networking.
4. Log on using the default built-in administrator.

Note: The password can be blank.

5. Open command prompt and run is as administrator.

 Security note: If UAC is enabled, then you’ll get a UAC prompt on screen. Please specify
credentials or permission to pass the UAC elevation.

6. Type the following commands:

net localgroup Administrators <your username> /add

net localgroup Users <your username> /delete

The first command adds that user to the Administrators group and
the second command removes the user from the Users group which leaves
him in the Administrators group.
