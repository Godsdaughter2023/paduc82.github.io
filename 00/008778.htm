The header margin is 0.5" by default, so that part should be okay. j.e. mcgimpsey say: I assume you're not grouping by quarter, since that should be done automatically when you subtotal.

Both the Group and Subtotal commands insert an Outline. You can only have one outline per sheet, so if you've already got an outline inserted by Group, then the Subtotal command will replace the outline.

However, you can  insert SUBTOTAL() formulas at the Quarter breaks (either via the Subtotal command, or manually), then remove any existing outline. You should then be able to group.

craigbrody say:

Hi,

Exchange Server 2013 online archive option?

each user on the network use AutoArchive  to backup to their hard drive PST?

Export to PDF functionality?

solution from others such as Barracuda?

michael bauer (vboffice.net) say:

What's best for you depends on your needs. For instance,

- A local PST archive is not searchable by other users.

- PDF might be limited in searchng as well. If you need the ability to get archived messages back to an Outlook folder, PDF is ruled out. PDF cannot save a message with its attachments.

For sure there are a lot more pros & cons for the different options.

roady [mvp] say:

In addition to Michael Bauer's reply:

It also depends on your business needs and why you want to archive the data. If it is still needed for business purposes, then you should really keep things centralized and manageable by corporate policy. Leaving it on the Exchange server or implementing a storage solution with lifecycle management capabilities is then the way to go.

I'd only recommend using a pst-file for archiving when it is for personal needs.

Storing it as pdf-files would only really make sense when the archive is going to be part of a knowledge base or a document management system for which you standardized to that format.
