  1. Changing the "Lay out this document as if created in" to Word 2010 will clear the "Split apart" check box, but it will also clear about a dozen other check boxes at the same time. The result is difficult to predict because it depends on what's in the document. For example, clearing "Don't use proportional width for Korean characters" won't affect you if you don't use those characters, but clearing "Use Word 2003 kerning pair rules" or "Use Word 2003 table autofit rules" might change the appearance of the document, sometimes substantially.
  2. If you had Word 2013, but the same document that was created in Word 2003, then yes, you would see the same problem. jay freedman say:

> is there's a way to keep the Split apart option box unchecked (permanently) for that specific document?

I think what you need to do (in word 2010) is click File > Info and click the Convert button, then save (preferably Save As to *.docx format). That will remove the information about Word 2003 from the document's file, change the "Lay out this document" setting to Word 2010, and clear all the check boxes. They'll stay unchecked unless you manually check them or use Save As to make it a *.doc file again.
