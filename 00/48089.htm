Login Information

  • Gateway - 192.168.254.254
  • Username - admin
  • Password – admin

DMZ

  1. Open a web browser.
  2. Enter http://192.168.254.254 into the address bar. When prompted enter admin for the user name and admin for the password.
  3. Under the section titled Setup, select Firewall from the menu on the left.
  4. Select DMZ.
  5. Select “enable DMZ with this host IP address:”
  6. Enter in the IP address of the Xbox in the box.
  7. Select Apply.

Disable DMZ

  1. Open a web browser.
  2. Enter http://192.168.254.254 into the address bar. When prompted enter admin for the user name and admin for the password.

  4. Select DMZ.
  5. Select Disable DMZ.
  6. Select Apply.
