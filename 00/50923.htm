a. user say:

Hi,

I will be glad to help you with the issue you have with the computer.

What is the make and model of your computer?

While working with Microsoft Windows 8.1/10, when you automatically get disconnected from the Internet, and you run the troubleshooter, you might get this error saying “Default Gateway Not Available” from time to time.

The reason because of which Windows 8.1/10 displays this error message is that the network adapter that is used to connect to the network or Internet is by default configured to get turned off automatically to reduce the power consumption. This is mostly the case when you are using a laptop computer and it is running on batteries.

Solution to this issue is very simple. You must follow the steps given below in order to resolve this problem:

  1. Log on to the Windows 8/Windows 8.1 computer using the administrator account.
  2. From the Start screen that appears, click the Desktop tile.
  3. Once on the desktop screen, click the File Explorer icon from the taskbar.
  4. On the opened This PC window, right-click This PC from the navigation bar in the left.
  5. From the displayed context menu, click Manage
  6. On the opened Computer Management sap-in, from the left pane, click to select Device Manager.
  7. From the right pane, expand the Network Adapters tree.
