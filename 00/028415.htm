See http://word.mvps.org/FAQs/Formatting/LandscapeSection.htm

If you have a single landscape page and you want to remove it put the cursor in the landscape page and run a macro containing the line

Selection.Sections(1).Range.Delete

Word is not a page layout application. Page is a vague and volatile concept in Word that can be made up of a variety of elements and depending on what is on the 'page' it may not be possible to simply delete the 'page' without affecting the surrounding text. Much depends on how the document is configured.

In a simple document a macro containing the lines

On Error Resume Next
ActiveDocument.Bookmarks("\page").Range.Delete

is about the best you can achieve

karl timmermans (outlook mvp 2012-2018) say:
>>
Can someone please tell me how I can change my email settings from IMAP to POP?
<<

Hi Margaret

#1 - You can't "change" settings from "one protocol - i.e. IMAP" to "another protocol - i.e. POP".

#2 - You cannot configure the same email account in Outlook if it's already configured. Or to put this another way, you can't add the same account as a POP account if it already exists as an IMAP account.

That leaves you with two options

#3 - Remove the IMAP account and then re-add it as a POP account. You do that by clicking on <Advanced Options> after entering the email address and then check the box for "Let me setup the email account manually".

OR

#4 -

a) With Outlook closed, open the Control Panel <Mail> app
b) create a new profile (don't need to delete the existing one)
c) configure the email account(s)
d) make sure to set the newly created profile as the default for the next time Outlook is opened.

If you have any questions - ask away.

julie sheets say:
You will need to add a custom date field with the following formula:
DateAdd("d",5,[Current Date])

That will calculate a date 5 days from the current date.  You can then use that Date field in your filter.

*please note, this uses 5 calendar days - so it does not omit non-working time.  If you want to count only working days use:
ProjDateAdd([Current Date],5*[Minutes Per Day],"standard")

in a custom date field.
