vijay a. verma say:

I think entire confusion is arising from the fact whether if A1=10-11-2015 and B1 = 10-11-2015 and then A1-B1 = 0 or 1. It entirely depends upon you how do you want to treat this. If you want that this should be 1, then you will have to add 1 i.e. your formula has to be =A1-B1+1. If you want it to be treated as 0, then your formula should be =A1-B1 only.

Now, if you put simply =A1-B1 for 3-1-2016 and 3-31-2016, it will be 30 only. If you want to treat this as 31, your formula has to be =A1-B1+1.

Leap year has no role to play in this and Excel takes care of date nuances like Leap Year etc on its own.

In case, you are using DATEDIF, same logic would apply.

=DATEDIF(A1,B1,"d")+1

And

=DATEDIF(A1,B1,"d")

rob koch say:

From the file path you listed this is clearly one of the installation files for the deployment of Microsoft Office 2013 to a Windows 8.1 system, likely for use via a USB per the WIN8USB portion.

Since the license for an Office 2013 installation no longer allows the use of this version in an Office 365 business situation, it may not be possible to recover a new copy of this file if it were needed to reinstall.  However, since a current license for Office 365 would typically now deliver the newer version of Office 2016, that may not matter.

So I don't see any reason that you'd need to keep a copy of this file unless you still have this older system and no supported way to get this from Microsoft.  Since you indicated in your selections above that the operating system you're using is Windows 10, then if that system was upgraded from Windows 8.1 it's unlikely such an old installation file would have any use at this point.

If the other files are found within the same Office 2013 installation folder with the same date, they are probably related to that same installation package and can be deleted as well.

Emails on the server download to Office Mac via IMAP.

However, local emails copied to the IMAP inbox, with the expectation that it will upload/sync with the IMAP server are not being synchronized.

Any suggestions?

Thanks.


talking moose say:

You can right-click or Control-click any IMAP folder and select Repair Folder. This will clear the folder's contents and synchronize them again from the server.

Since you have moved messages into the folder you may want to create a copy of the folder before repairing it. You can move the messages into that folder again when complete. This should start the synchronization.

Hope this helps!

talking moose say:

Sorry, no. Outlook for Mac only logs activity with Exchange accounts.

The only other suggestion I can make is to remove your IMAP account altogether, restart Outlook for Mac and then add your account again. FYI, this will delete not only the account but any messages from your Mac.

If that doesn't resolve the issue then I suspect something may be happening with your IMAP server that's preventing synchronization.

Hope this helps!

how do i down load photos from my phone to my laptop - Microsoft Community

alanbrownye say:

how do i down load photos from my phone to my laptop

how  do I down load photos from my Microsoft phone to my lap top

lester_c say:

Hi alanbrownye, 

Since you are using a Windows Phone, you can save files on OneDrive and log-in with the same Microsoft Account on your Laptop, or you can do this:

  • To transfer photos and videos from your phone to PC, connect your phone to the PC with a USB cable. Make sure the phone is on and unlocked, and that you're using a working cable, then:
    • On your PC, select the Start button and then select Photos to open the Photos app.
    • Select Import > From a USB device, then follow the instructions. You can pick the items you want to import and select where to save them.
  • If you're having trouble with an Android device or iPhone, see the below notes:
    • On Android devices: You might need to change the USB setting to allow you to transfer media or photos and not just charge the phone. Go to Notification Center and choose Transfer files or File transfer. If your photos or videos are stored in the cloud, you won’t be able to download them, since they then exist in the cloud and not on your device. If you want to import your photos or videos to your PC, download them from Google Photos in your browser or use the Your Phone app from Microsoft Store.
    • On iPhones: Make sure your phone is unlocked, and when you see the Trust this device notification, select Accept. If iCloud is enabled on your iOS device, you won’t be able to download your photos or videos, since they then exist on iCloud and not on your device. If you want to import your photos or videos to your PC, do so via the iCloud desktop app.
  • Having trouble connecting your phone? Get help with USB problems.

If you still have problem about importing photos, you can contact manufacturer of your Laptop regarding with your concern.

Hope this helps you.
