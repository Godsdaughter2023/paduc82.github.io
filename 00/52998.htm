It doesn't but you can improve the usability by using the keyboard.  E.g.  Alt+Tab will show you thumbnails of all your open windows on your primary monitor.  Win+Tab will show you which ones are on which visible monitors, so the windows which don't show via Task View need to be moved (if you want to see them somewhere else).  You can do that by establishing keyboard focus on the window that you want to move and then using either  Win+Shift+Cursor (move directly to a different monitor)  or  Win+Cursor  (Snap a window in a monitor and when possible Snap through that one to a different monitor, although that procedure is apparently less predictable with a command shell window).  The reason you may want to use Snap instead of a direct move is that sometimes windows are rendered off screen or get moved off screen and then it may be difficult finding them again (without resorting to the old standby of Alt-Space Move Cursor and dragging it into view).  Being able to use Snap in this scenario is a recent and welcome improvement.

It doesn't but you can improve the usability by using the keyboard.  E.g.  Alt+Tab will show you thumbnails of all your open windows on your primary monitor.  Win+Tab will show you which ones are on which visible monitors, so the windows which don't show via Task View need to be moved (if you want to see them somewhere else).  You can do that by establishing keyboard focus on the window that you want to move and then using either  Win+Shift+Cursor (move directly to a different monitor)  or  Win+Cursor  (Snap a window in a monitor and when possible Snap through that one to a different monitor, although that procedure is apparently less predictable with a command shell window).  The reason you may want to use Snap instead of a direct move is that sometimes windows are rendered off screen or get moved off screen and then it may be difficult finding them again (without resorting to the old standby of Alt-Space Move Cursor and dragging it into view).  Being able to use Snap in this scenario is a recent and welcome improvement.
