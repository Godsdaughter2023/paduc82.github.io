There is no difference between a directory merge using a doc format document or a docx format document, though the Word 2010 document converter may miss the document type if created in an earlier version. There are probably a dozen ways you could do it.  The MOST important thing is EVERY TIME, BEFORE you unplug any USB storage device from you computer you must use the "Safely Remove Hardware" applet to properly shut down the device.   This is MANDATORY because of the technical limitation of USB drives. The flash memory has a limited number of writes so to "save" them files are actually saved to a "cache" on the HD, and only written to the flash device when there is a 'full block' of data.  If a file does not "fill a block" the tail end of the file sits in the cache until it is "flushed" by the "safely remove hardware" tool. pamcaswell say: Since Word is working as designed, I only have two suggestions.
  1. Lock the TOC field so it is not updated. Select the TOC and the empty paragraph mark after it, and then press Ctrl+F11.  When you need to update the TOC, unlock it by pressing Ctrl+Shft+F11.

One question please. In the app what is the difference between "Sent Items" and "Sent" under the "More" heading.

You have folders on the server.

An account setup using Pop3 in a program or app will only download emails from the Inbox folder on the server.

Accounts that are setup using IMAP or EAS will sync all folders.

You have a program or app that wants to use Sent Items and the server uses Sent.

As long as a sent email is being saved, do not worry about it.
