a. user say:

Hi Brettos,

If you are facing issues with Journal writer in Windows 8.1, then you can try repairing the Journal writer by following these steps:

  1. Press Windows key +X, select ‘Command prompt (Admin)’.
  2. In the command prompt, change the directory to ‘Drive:\Program Files\Windows Journal’ (where drive is drive location where windows is installed).
  3. By default the directory would be c:\Windows\System 32. To change the directory type “cd..” and press enter. This will take you to the previous folder. Once you come to the c: drive, type “cd Program Files” and then “cd Windows Journal”.
  4. Once you are in the Drive:\Program Files\Windows Journal’ type the following command :

journal.exe /repairnotewriter

Try these steps and let us know the results.
