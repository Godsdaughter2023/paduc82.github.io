hansv mvp say:

Do you want to unhide ALL hidden sheets? If so, you could assign the following macro to a command button, or run the macro directly from the Macros dialog (Alt+F8):

Sub UnhideSheets()
    Dim sh As Object
    For Each sh In Sheets
        sh.Visible = xlSheetVisible
    Next sh
End Sub

Is your workbook protected for structure? If so, you'd have to unprotect the workbook before you can unhide sheets. In a macro:

Sub UnhideSheets()
    Dim sh As Object
    ActiveWorkbook.Unprotect Password:="secret"
    For Each sh In Sheets
        sh.Visible = xlSheetVisible
    Next sh
    ActiveWorkbook.Protect Password:="secret", Structure:=True
End Sub

Use =IF(ISBLANK(ref),NA(),ref)

The NA() will place #N/A in the cell. The chart engine ignores this

If you dislike the appearance of #N/A in the cell, use conditional formating to hide it by making the font colour the same as the background colour.

Which Formula to Return sheets name? - Microsoft Community

thaiphan say:

Which Formula to Return sheets name?

Hi guys. So I have many sheets on my Excel file. It goes like: Sheet1 has a list of name, and Sheet2, Sheet3, ... are a sign up list of people. For example, Siz is a name on Sheet1, and sign up on Sheet4 & Sheet5.

My question is: what formula(s) can I use to do things like these in Sheet1: Find(Siz) on all the sheets, if (Siz) appear on Sheet4 then return "Sheet4" to the cell. I expect the result goes like "Sheet4", "Sheet5" in the cell where I have the formula.

Please help me guys. Thank you so much.

hansv mvp say:

Press Alt+F11 to activate the Visual Basic Editor.

Select Insert > Module.

Copy the following code into the module:

Function ListSheets(strName As String) As String
    Dim wsh As Worksheet
    Dim rng As Range
    Dim strList As String
    Application.Volatile
    For Each wsh In Worksheets
        If wsh.Name <> Application.Caller.Parent.Name Then
            Set rng = wsh.Cells.Find(What:=strName, LookAt:=xlWhole)
            If Not rng Is Nothing Then
                strList = strList & ", " & wsh.Name
            End If
        End If
    Next wsh
    If strList <> "" Then
        ListSheets = Mid(strList, 3)
    End If
End Function

Switch back to Excel.

Let's say you have a name in A2. Enter the following formula in B2:

=ListSheets(A2)

This formula can be filled down if you have other names in A3 and below.

Remark: you must save the workbook as a macro-enabled workbook (.xlsm)
