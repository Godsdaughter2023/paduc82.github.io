ericyang_msft6286 say:

Hi Hallie,

Using the OneDrive for Business sync client, we can’t sync a single file and we need to sync the container of the file. The container can be either a folder or a library. However, the Shared with me section is just a view that lists all items shared with you but not an actual SharePoint library. Therefore, we can’t sync the Shared with me section down as it is not a container. When someone else shares an item with, there are two scenarios:

1. If someone else shares an entire folder with you, you can enter the folder and then directly sync all items in it down.
2. If someone else shares a single file with you, you need to go to the original location of the file and then sync it down. Here are the steps for your reference:
1) Open your OneDrive for Business library in a browser, then click Return to classic OneDrive on the lower-left corner.
2) Under the classic interface, click the ellipsis next to the file name, then choose View Library.
3) You can see the Sync button on the ribbon to sync the library down with Edit permission.

Let us know if you can sync the file down successfully.

Thanks,
Eric Yang

If you are using mapped network drives, it doesn’t actually sync the files. It just map the OneDrive for Business library’s online path to your computer. You can enter the files from your computer.
