jay freedman say:

Assuming that Charles is correct about the structure being a table (there are other ways to get the same appearance, but they're much less likely), the solution involves two steps:

  1. Remove the shading from the table cell on the left, by using the Borders & Shading dialog's Shading tab and selecting No Fill.
  2. Insert a rectangle from the Insert > Shapes gallery into the cell. Set the text wrapping of the rectangle to "In line with text". Set its fill to grey and its border to No Line. Drag its height and width to fill the part of the table cell on the first page.

The difficulty with this workaround is that the portion of the table row that appears on the first page may change if anything before the table or in that first row is edited; or if the document is opened while a different printer is selected; or if the margins or page size change; etc. If the document will be distributed to other users, save its final format as a PDF file and distribute that.

gerald mal say:

Hi Kirang,

We suggest that you use the Windows Device Recovery Tool. Start by downloading and installing Windows Device Recovery Tool (for Windows 10, 8.x and 7).

When the installation has finished, start the application and the main screen will appear:

The procedure to restore your phone using Windows Device Recovery Tool is as follows:

  1. Connect the phone to your PC (using the USB cable that came with the phone).
  2. After a few seconds, the PC should find the phone and you'll see the Phone Selection screen. Click your phone in this screen to select it.
  3. After a few seconds more, the latest available firmware version for your device will be shown. Click the 'Install software'/'Reinstall software' button to flash your phone

Windows Device Recovery Tool will then download and install the latest firmware + OS available for your phone

Note! Your phone will be reset to factory status when using this tool.
