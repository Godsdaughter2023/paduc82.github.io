First, make sure you have installed a program that is capable of scanning inside PDF files (this is known as an iFilter).  Adobe Reader, for example.  Not all PDF readers have this ability (most are just viewers, not capable of scanning inside a file during a search).

3 - Click anything in the list, then PDF to jump to PDF files.
4 - Select the PDF file and make sure it's set to index files and contents.
