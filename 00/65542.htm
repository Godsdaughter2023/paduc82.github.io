stevepayne1994 say:

Create a logon message FOR A SPECIFIC USER!

I already know how to create a Logon Message for everyone to see before they choose they're account, but can I create a message to appear AFTER they logon on to their account?

Thanks All!

frederik long say:

Here you go:

  1. Click Start.
  2. Type the word notepad into the Search box.
  3. Press Ctrl+Shift+Enter
  4. Click "Run as Administrator".
  5. Copy, then paste the code below into this file. Do not retype it!
  6. Save the file as
    C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup\Message.vbs
    then close it.
  7. Create a list of qualifying users in c:\Windows\temp\Users.txt, one account name per line.

Users whose account name is included in Users.txt will now get a pop-up message after they have logged on. To accelerate the testing process you can double-click the file C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup\Message.vbs via Windows Explorer.


'-------------------------------------------------

'Utility to pop up a message if the current user's
'account name is included in the user list.
'14.11.2010 FNL
'-------------------------------------------------
sMessage = "Hello world"
sList = "c:\Windows\Temp\Users.txt"
Set oWshShell = CreateObject("WScript.Shell")
Set oFSO = CreateObject("Scripting.FileSystemObject")
If Not oFSO.FileExists(sList) Then
    MsgBox "Unable to locate """ & sList & """", 0, "Startup Message"
    WScript.Quit
End If
sUserName = LCase(oWshShell.ExpandEnvironmentStrings("%UserName%"))
aUsers = Split(oFSO.OpenTextFile(sList).ReadAll, vbNewLine)
For Each sUser In (aUsers)
    If LCase(Trim(sUser)) = sUserName then MsgBox sMessage, 0, "Startup Message"
Next

whyjoe say:

Windows doesn't come with the ability to show thumbnails for PSD (Photoshop) or many other Adobe file types. Worse, Adobe no longer includes a plugin allowing Windows to do this (they used to, years ago.)

For now, the only solution is to use one of a couple different third-party plugins. Most are paid, between $15-$25, some are free.

My personal favorite is Ardfry Imaging PSD Codec, $25, which also includes support for Illustrator, InDesign, and other Adobe thumbnails. In addition to providing thumbnails, it ALSO lets Windows index the metadata associated with the files, which is handy for searching.

Other options are:

Mystic Thumbs (I can't vouch for this package, but it seems iffy. Free to $29 depending.)

Sagethumbs (Free)

One thing I like about Ardfry over the others is that it reads in the embedded thumbnail that's already included in AI, EPS, and INDD files.  (The other suites use GhostScript to reinterpret those files and generate a new thumbnail, which is unnecessary and I imagine slower.)

hula_baloo say:

 The best network protection is to have a router which gives a hardware firewall and turn on the Windows firewall even if you only have one computer.  Most folks have either cable or DSL connections and with these your computer is ALWAYS connected to the Internet.  This makes it easy for hackers to get into your computer if you don't have good firewall protection.  The Windows software firewall is free and you can buy routers these days for as little as $15.  If you're using a wireless connection (like with a laptop), you also want to be sure that your connection is encrypted with something like WPA-2.  If you do all these things, you'll be in good shape.  If not, you're taking unnecessary risks.
