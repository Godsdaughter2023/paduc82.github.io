Hi,

 

Try this

 

1. Data in sheet 1 is arranged as follows: - project names from cell A2 downwards (say in A2:A4), months (Jan, Feb, Mar etc) are in B1:M1.  Figures are in B2:M4

2. On sheet, project names are mentioned in A2:A4

3. In cell B2, enter this formula and copy down to compute Revenue to date

 

=SUM(INDEX(Sheet1!$B$2:$B$4,MATCH(A2,Sheet1!$A$2:$A$4,0),1):INDEX(Sheet1!$B$2:$M$4,MATCH(A2,Sheet1!$A$2:$A$4,0),MATCH(TEXT(TODAY(),"mmm"),Sheet1!$B$1:$M$1,0)))

 

4. In cell C2, enter this formula and copy down to compute Revenue to completion

 

=SUMPRODUCT((Sheet1!$A$2:$A$4=A$2)*(Sheet1!$B$2:$M$4))-B2

I don't use Excel2013, but this is how I'd do it on previous versions.

Data Validation/Custom/Formula

A1 is your cell with 0 or some other value

B1 is your data validation cell

Formula for DV   

=A1<>0

From then on, B1 is only editable when A1 is not zero

Disadvantage: if A1 reverts to zero later, the value in B1 is not editable. This may be desirable in some situations, but if you want B1 to return to blank when A1 returns to zero, this won't get you there.

madison knight say:

I realize this is an old post, but it's one of the first results in Google and the answers provided so far miss an important option.

The best solution I found for making the Quick Access Toolbar icons larger is to increase the font size for Menu in the Windows Color Scheme area. To do this in Windows 7, right-click on the desktop and click on "Personalize" and then select "Windows Color". Once there select "Menu" from the drop down list ("Desktop" is selected by default). Then where it says font, pick a size that is larger than what you currently have. The font on one of my computers was set to 12 and I changed it to 16. Once you've done that, hit "Ok" and you should see the change put into effect. You can go back and adjust it as necessary to make it larger or smaller depending on how big you want to end up making the menu area.

The possible downside of this solution is that it will make the menu bigger in all programs, but if the QAT icons were hard on your eyes, then making the menu larger everywhere will quite possibly be desirable.

Unable to Launch office 2011 applications - Fonts Issue - Microsoft Community

a. user say:

Unable to Launch office 2011 applications - Fonts Issue

When fonts go bad or corrupt on your Mac they can cause all sorts of problems like shutting down your graphics program when you select that particular font.

In Office 2011, applications will not open if you have font issues. You will notice this as soon as you launch the application and you see the splash screen where you will see the status: Updating font menu (..%) This status will not continue and the application will crash.

All Macs come equipped with an application called Font Book. To find out which fonts are good and which ones are bad we can run the validation option available in Font Book.

To find Font Book, open up your Applications folder, find Font Book and double click to open it.

The Font Book lists all the fonts that are installed on your Mac.

To validate one or more fonts:

1. Select the fonts you want to validate from the list

2. Right click on your chosen fonts and choose "Validate Fonts" from the File menu.

3. A "Font Validation" window opens and shows the results of validation. A green check mark indicates that font passed validation. A red x beside a font tells you there is a problem.

If you do find a problem you should delete these fonts:

How To Delete Fonts

Check the box beside the corrupt font in the "Font Validation" window.

Click on the "Remove Checked" button at the bottom of the Font Validation window. An alert appears. Click to confirm, and the fonts are deleted.

To Resolve Duplicate Fonts:

4. Repeat Step 1

5. Resolve Duplicate Fonts: Right click on your chosen fonts and choose "Resolve Duplicates" from the File menu.

or click on Edit from the Menu on top and select "Resolve Duplicates"
