Hi Martin Drabble,

Thank you for contacting us. As I understand it, you want to open the old mailbox which was converted to a shared mailbox in your account. And according to the error message, it’s probably related with mailbox permissions of access. You can follow the steps below to add the Full Access permission of the shared mailbox to your account.
1. Login to Admin portal, and click Admin icon to Admin center.
2. Under Admin centers tab, click Exchange to Exchange admin center.
3. In the new page, navigate to recipients > mailboxes, and then Edit the shared mailbox.
4. In the pop up, select mailbox delegation > Full Access, if your account doesn’t exist in the list of Full Access, please add it in.
5. Please wait for few minutes to make the setting works, and then you can try to click the option open another mailbox. 

And if the issue persists, please run Windows PowerShell to verify if you have the permission to access the shared mailbox.
1. If you have never run PowerShell commands in your computer before, please right-click Start and run Windows PowerShell (Admin) and run the following command in the first.
Set-ExecutionPolicy RemoteSigned

2. Right-click Start and run Windows PowerShell commands to connect the Exchange Online server. In this process, you need use your admin account to sign in.
$UserCredential = Get-Credential

$Session = New-PSSession -ConfigurationName Microsoft.Exchange -ConnectionUri https://outlook.office365.com/powershell-liveid/ -Credential $UserCredential -Authentication  Basic –AllowRedirection

Import-PSSession $Session

3. Run the following command to verify if your account have the access permission.
Get-MailboxPermission -Identity <the shared mailbox email address> -User “<Your account ID>”
(e.g. Get-MailboxPermission -Identity contoso.com -User “Kevin Kelly”)

4. If there is no access permission in your account, please run the following command to add it.
Add-MailboxPermission -Identity “<the shared mailbox ID>” -User “<Your account ID>” -AccessRights FullAccess -InheritanceType All 
(e.g. Add-MailboxPermission -Identity "Terry Adams" -User "Kevin Kelly" -AccessRights FullAccess -InheritanceType All)


5. Before you close the Windows PowerShell, please disconnect to Exchange Online server by running the command below
Remove-PSSession $Session

a. user say:

(1) As Suzanne says. But the Display language is just the language that is used in menus, toolbars, buttons, options etc., and AFAIK Microsoft has chosen a specific locale for each "language group). e.g. for English it is English (US). I assume that this is because most English speakers can reasonably be expected to understand "Center" rather than "Centre" and so on, and this choice does not affect their ability to write text in their own locale's version of the language.

(2) The dictionaries etc. are associated with the Editing language. You have already worked out how to get them I think.

(3). The taskbar option selects a locale+a keyboard layout+(possibly) an "Input method". So if for example, you have a physical QWERTY keyboard but you want to work in French with an AZERTY keyboard, you should be able to set up the French+AZERTY combination as an option in the taskbar list. When you select that, your QWERTY keyboard will behave as if it had an AZERTY layout. (Or, alternatively, you can have an extra AZERTY keyboard connected to your PC and use that instead).

Selecting the language/keyboard in the taskbar has a different effect from applying a proofing language (e.g. via Word->Review->Language->Set Proofing Language, because the latter does not affect the behaviour of your keyboard in any way. e.g. if you set the Proofing language to Hebrew, your keyboard does not suddenly start typing right-to-left in the Hebrew script. (Typically, though, setting the language/keyboard in the taskbar *will* change the proofing language for new text.

[NB, an "Input method" is a piece of software that helps you to type in a script such as Chinese when using (say) a QWERTY keyboard). e.g. it may let you type some letters and then present a list of characters to choose from)]
