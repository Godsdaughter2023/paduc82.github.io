DEP is a security feature that can help prevent damage to your computer from viruses and other security threats. DEP monitors programs to make sure they use system memory, safely. According to this article, if you have libraries with Checkout, Required columns or metadata, or when Draft Item Security is set to either Only users who can edit or Only users who can approve items in Version Settings of the library, these items will be synchronized as read-only for SharePoint online.
