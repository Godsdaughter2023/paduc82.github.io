Always greatful WT
ron coderre say:

There is no built-in function that will display the currently logged-in user.

You could use one of these VBA solutions, though:

Using Regular Excel VBA
Function NetworkUserName() As String
   NetworkUserName = Environ("Username")
End Function

Using the windows API
Declare Function apiGetUserName _
Lib "advapi32.dll" Alias _
"GetUserNameA" (ByVal lpBuffer As String, _
nSize As Long) As Long

Then use this API function call in your own function
Function GetUserName() As String
    Application.Volatile
    Dim sBuff As String * 25
    Dim lBuffLen As Long
    
    lBuffLen = 25
    apiGetUserName sBuff, lBuffLen
    GetUserName = Left(sBuff, lBuffLen - 1)
End Function

Copy/Paste either approach into a general VBA module.
Then,  one of these functions in your worksheet:

=NetworkUserName() ...if you used the first approach
or
=GetUserName() ...if you used the second approach.

Does that help?

Using defined named ranges:

  • Data refers to =Sheet1!$B:$B

On Sheet2:

A2 = some date

Enter this formula in B2. This will return the count of records that meet the criteria. This value will also be used in the "lookup" formula for an error trap.

=COUNTIF(Dates,A2)

=IF(ROWS(C$2:C2)>B$2,"",INDEX(Data,SMALL(IF(Dates=A$2,ROW(Dates)),ROWS(C$2:C2))))

el_loco---01 say: If you haven't sent it in you may want to try something.

Mind you, only try this if you've left the phone without use for a while and then it did the problem.

In some rare ocasions, Hard Drives are known to get "stuck" if left for a while without usage. The problem is sometimes referred to as "stiction" or "static friction".

When this happens the recommended action is to give the hard drive some "gentle persuasion", i.e. hit it gently.

So you could try giving your N91 a few "taps" with your hand. Not too strong but not too soft either, do this while the device is booting up. Try tapping it 2-3 times with your hand. A good way is to hold it with your right hand and then hit it with your open left hand by the camera lens.

Again don't do this if you're afraid to damage it further or void the warranty.

At any rate, the service centre should be able to replace your HDD with a new one under warranty. But you will lose all the data that was in the old hard drive.
