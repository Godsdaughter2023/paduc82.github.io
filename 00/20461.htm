macropod say:

Unless you're using formfields, there's probably no reason you couldn't use the "No Changes (read only)" editing restrictions for the whole document, with exceptions marked as required - and you'd only need the content controls for non-text inputs (e.g. dropdowns, date selections).

Method 2: Disabling JavaScript.

Step 1: Open Internet Explorer > Click the Tools button > Click Internet Options > Go to Security tab.

Step 2: Click the Custom level button located in the Security level for this zone section to display the Internet Zone security settings. Scroll down until you locate the Scripting section.

Step 3: To disable JavaScript, locate the Active scripting subheading and click on the accompanying Disable radio button. If you would rather be prompted each time a website attempts to launch any scripting code, select the Prompt radio button.

Method 3: Add To Trusted Sites.

Step 1: Open Internet Explorer > Click the Tools button > Click Internet Options > Go to Security tab > choose Trusted sites.
