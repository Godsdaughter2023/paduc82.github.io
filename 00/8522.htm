doug robbins - mvp office apps & services (word) say:

It is not that likely that all of the Next Page Breaks would have changed to Odd Page Section breaks, but in the Page Layout tab, click the dialog launcher at the right hand end of the Page Setup bar and go to the Layout tab and check the setting for Section start when you have the cursor in a page after one of the Section Breaks that you inserted.

 

Also check the Paragraph formatting to make sure that the first paragraph on a page is not formatted to have a page break before it, or that all of the paragraphs are not formatted to be kept with next.

suzanne s. barnhill say:
In addition to what Doug said, note that Breaks | Next Page is not the same thing as a page break (which is just "Page" at the top of the menu). You can insert a page break without having to resort to the Ribbon; the keyboard shortcut is Ctrl+Enter. If you need only a page break, then a section break is overkill.

suzanne s. barnhill say:

I'm not entirely clear what you're trying to do with the page numbers, but if you want page numbers on some pages and not on others, then, yes, you will need to insert section breaks. But if you are starting the numbering at 1, then Word will regard that as an odd page (since it is) and force a Next Page break to be Odd Page.

 

If you're just trying to omit numbering (or use different numbering) on front matter, see http://www.word.mvps.org/FAQs/Formatting/front_matter_2007.htm (which applies to Word 2007 and 2010 and should be pretty accurate for 2013 as well). The most important tip is that you can restart numbering or change the number format without unlinking sections, but if you want to completely omit page numbers from a given section, you do need to unlink the relevant header/footer from the preceding and following ones.

 

You don't have to worry about headers/footers/page numbers on the blank pages that Word inserts, as these will be truly blank. You can also omit a header or footer on the first page of a section by enabling Different First Page and leaving the First Page Header/Footer empty; see http://WordFAQs.mvps.org/HeaderFooter.htm.
