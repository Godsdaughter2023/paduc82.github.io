This may be hard to believe. But, to be honest, the above files are actually folders. This also happens with individual files like .txt files, even the font files don't display their thumbnails. I rebuilt the icon cache with the following .bat file.

:: Created by: Shawn Brink
:: http://www.sevenforums.com
:: Tutorial:  http://www.sevenforums.com/tutorials/49819-icon-cache-rebuild.html

@echo off
set iconcache=%localappdata%\IconCache.db

echo The Explorer process must be killed to delete the Icon DB.
echo.
echo Please SAVE ALL OPEN WORK before continuing.
echo.
pause
echo.
If exist "%iconcache%" goto delID
echo.
echo Icon DB has already been deleted.
echo.
pause
exit /B

:delID
echo Attempting to delete Icon DB...
echo.
ie4uinit.exe -ClearIconCache
taskkill /IM explorer.exe /F
del "%iconcache%" /A
echo.
echo Icon DB has been successfully deleted. Please "restart your PC" now to rebuild your icon cache.
echo.
start explorer.exe
pause
exit /B

¡firedog say:

Is this "verify your email to finish setting up your account" message, apparently from microsoft, a scam?

No. It's the mechanism that stops someone from registering someone else's email address as a Microsoft Account, whether by accident or design. You should click the cancel link each time you get one of these requests for verification. The instigator will get the message sooner or later.

Alternatively, you could register the Gmail address as a Microsoft Account yourself. You would then get a legitimate verification request; once complied with, you'd never get any more unless you allowed the Microsoft Account to lapse.
