andreas killer say:
Perfect, now I and other experts can look at the file and make suggestions.

Before we start with your rules, a technical note:
Whatever you calculate in Excel are numbers, no times, no dates, no percentage, no currency, and so on. E.g. one hour =TIME(1,0,0) is a value of 0.041666667

What the cell shows depends only on the cell format!

Therefore
B6: =IF(A35="WORKING", E47, TIME(0,0,0))
is oversized, this is the same
B6: =IF(A35="WORKING",E47,0)

You do not need to put everything in parentheses, even leading + signs are not necessary. Excel follows the mathematical rules.
If you use TIME(1,0,0) in a formula Excel calculates that every time, but as the expression never changes it is enough to calculate it once.
We can do that if we use a name instead. Open the name manager add a new name OneHour and use the formula =TIME(1,0,0) instead of a cell reference.

Therefore
G7: =(COUNTIF(A35:A109,"SICK FULL"))*(8*TIME(1,0,0))
becomes
G7: =COUNTIF(A35:A109,"SICK FULL")*8*OneHour

Now to your travel rule, I say it in other words:

The travel time is the depart time of the previous row minus the arrival time of the current row.
But if the location of the previous row is "Home" or the location of the current row is "Home", we have to subtract one hour.
(That is the same as:
They don't get paid for up to the first hour of travel when starting from home
When they travel home, we do not pay for up to the last hour)
If we calculate that always it is possible that the result is negative, so we have to set a limit at zero. (The same as "but anything over we subtract the 1st hour.")
They also do not get paid while taking a lunch.

F38: =IF(OR(A38="",A38="Lunch"),"",MAX(C38-D37-IF(OR(A37="Home",A38="Home"),OneHour,0),0))
and drag down.

I can not find your "overtime formula" nor can I see any pound symbol, but as you know now, you just have to change the cell format.

Any questions?
