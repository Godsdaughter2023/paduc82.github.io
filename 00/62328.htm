felixwong_ say:

I had a somewhat similar problem with my keyboard deactivating.

In my case, FILTER KEYS was accidentally turned on as result of holding down the SHIFT key too long (apparently, holding down the shift key for 8 seconds is a shortcut to filter keys).  Filter keys won't allow a keystroke to register unless you hold each key down for at least a couple seconds, hence giving the impression the keyboard is not working.

To resolve, go to Control Panel (Desktop Control Panel for Windows 8.x)->Ease of Access->Change How Your Keyboard Works->and unclick "Turn on Filter Keys."

I would also recommend then clicking on "Set up Filter Keys" and uncheck the "Turn on Filter Keys when right SHIFT is pressed for 8 seconds.

I would further recommend doing the same for Sticky Keys.
