I use DAO code like this FAQ on Tek-tips to change the SQL property of saved pass-through queries.

Function fChangeSQL(pstrQueryName As String, strSQL As String) As String
' basQueryDefs.ChangeSQL
'-------------------------------------------------------------
' Purpose : update the SQL property of a saved query
' Copyright: Duane Hookom
' Author : Duane Hookom
' Notes :
'-------------------------------------------------------------
' Parameters
'-----------
' pstrQueryName (String) Name of saved query
' strSQL (String)        SQL Statement
'-------------------------------------------------------------

'-------------------------------------------------------------
' Revision History
'-------------------------------------------------------------
' 07-09-2001 DKH:

    Dim db As DAO.Database
    Dim qd As DAO.QueryDef

    qd.SQL = strSQL
    Set qd = Nothing
    Set db = Nothing
End Function

newt_othis say: Hi Daniel,

Just thought I'd let you know that we have resolved our issue. It was the CRM add-in, writing millions of times a minute to the local cache database file (.sdf), located C:\Users\username\AppData\Local\Microsoft\MSCRM\Client

We also discovered that using Outlook /safe *does not* stop the CRM add-in from running and accessing this file.

So be aware that Outlook /safe is not a guarantee that no add-ins are loaded.

The fix, for us, was to run the CRM Configuration Wizard, remove all settings, close it, re-run the wizard and add the settings back. This recreated the (corrupt) .sdf file and Outlook was then much happier.

                                                      Page 1                                                  

                                              TITLE OF DOCUMENT

Except that there would be no spacing between the line and the title of the document and I don't know how to correct that in this post.  Any advice?

stefan blom say:

Here's one way to do what you want: Activate the header/footer view, for example by double-clicking the header. Place the insertion point in the footer. Press Ctrl+E and then type "Page." Press Alt+Shift+P to insert a page number field. Press Enter and type the title of the document. Select the first paragraph. To add a bottom border, click the arrow on the Border button (in the Paragraph group on the Home tab of the ribbon) and choose Bottom Border. To exit the header/footer view, double-click in the main body of the document.
