puigchild say:

Hi Guys,

 

We understand this must be most inconvenient for Mac users. However, there is a workaround to the backup issue, though it may sound a bit gimmicky. 

 

First back up your phone: Menu > Settings > Sync & backup > Create backup. This will create a backup file to "Backup files" folder on your memory card.  

 

After this, remove the memory card and connect it into your computer using an adapter (usually supplied with the card). Finally, copy the backup file on your Mac for safekeeping. 

 

If you need to restore your phone, just copy the backup file back on the memory card and select Menu > Settings > Sync & backup > Restore backup.

 

There is a workaround to the calendar and contact sync issue as well. When Mail for Exchange becomes available for your device, first sync your iCal/contacts with GMail/Hotmail then use MfE to sync it with your phone. 

 

Let us know if this helps,

 

Puigchild
