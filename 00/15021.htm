steve rindsberg say:

Interesting.  If you want to have a go at it, you can save a theme then modify it.

Save the theme as a THMX file, tack on a .ZIP extension, and crack it open.

Poke around inside and find the file named theme1.xml

Edit that in a decent text editor and it's pretty easy to see where the Headings and Body text fonts are specified.  Change them (one place for each) to whatever font you like, save and put the file back into the ZIP where it came from , save the zip, remove the .ZIP extension and voila.  You have a theme with your choice of fonts.

Which, I'll happily agree, doesn't go very far in answering the original question, "Why do they make this so hard????"

But at least it can be done.  If you're geek enough. ;-)
