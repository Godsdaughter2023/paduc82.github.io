Windows Product Key: *****-*****-F36R3-7HM2B-RXTCX

Does this match the product key shown on the COA sticker?  If not, you probably have a legitimate Windows 7 Home OEM license, which explains why trying to activate with that key doesn't work.  If you want to keep using Windows 7 Pro you'll need a legitimate license.  If you want to use the Home version, you might be able to do a factory reset which will reinstall the original version of the OS, while wiping out everything else including programs and personal data.
