lemp say:
The following is based on my experience with XP; I haven't done a lot of work/research on Win 7 yet so I can't tell you absolutely that this will work, but it's worth trying.

Typically, printer drivers are installed using an INF file.  However, the first time an installation is attempted, Windows "compiles" the relevant INF file into a companion PNF file with the same name.  For subsequent installs, Windows uses the PNF file.

If you suspect a corrupted driver and download a new one, unless you have totally removed the old driver, including its PNF file, Windows often will just use the old files.  You think you're installing a new driver but you're really not.

In Windows XP, the solution to this was to go to "Server Properties" on the File menu of the Printers and Faxes window and delete the suspect driver from there.  In Win 7, it's a bit more complicated:

Control Panel > System and Security > Administrative Tools
Print Management
In the left pane, expand Print Servers > <local computer name> > Drivers
Select the suspect driver in the right pane and Delete (I don't know if you have to remove the complete "Driver Package" or not; that option wasn't in XP).
Now reinstall the driver.

If the remote user is not using a "computer administrator" account, I suspect that you may have to right click "Print Management" and select "Run as administrator."

P.S.:  An alternative procedure in XP was to use cleanspl.exe from the Windows 2003 Resource Kit (available here --> http://www.microsoft.com/en-us/download/details.aspx?id=17657).  According to this Technet thread, the same tool will work in Win 7.

What is the difference between creating a 'Full Backup' and creating a 'System Image' of your computer in case you have to restore your computer? In Windows 7 for example.

I'm creating a "Full Backup" and creating a "System Image" of my computer on some DVD-Rs. I was just wondering what's the difference between the two as far as restoring your system. Thanks.
steve winograd say: A System Image is an exact copy of everything on the computer's hard disk.  You can use it to restore the system after a catastrophic crash, hard disk failure, etc.

A Full Backup contains data files, but not everything on the disk, and you can't use it to restore the system.
