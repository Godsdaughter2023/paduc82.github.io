brijesh h say:

Using a FUNCTION to display a Message Box.

Need help in finding a function that would display a Message Box on successful Validation.

Eg: If value of Cell A1="X", then Message Box should display "The Value is X".

I know this can be done using Macros, but I don't want to use them.
jeeped say:

Fair enough, but since there is no 'function' built-into Excel that can be used to generate a message box in the manner you requested, you would be stuck with a VBA UDF or User Defined Function in any event. I realize you don't want to go the VBA route, but I'll post the solution here to anyone else interested.

With Excel open to your worksheet, press ALT+F11 to open the VB editor. Use the menu to Insert, Module. In the new Book1 - Module1 (Code) window, paste the following:

Function fcnWhatValueIsWhere(ByVal Target As Range)
     MsgBox "The value of " & Target.Address & " is " & Target.Value
End Function

... and press ALT+Q to close the editor and return to Excel.

Usage: in any out-of-the-way cell put

=fcnWhatValueIsWhere(<cell address>)

... where <cell address> is the cell you wish to display the value of. Note that upon initially entering this formula into a cell a dialog will popup referencing the cell used as the parameter. The reason for putting this into an out-of-the-way cell is that it serves absolutely no purpose beyond acting as a 'trigger' when the cell it references recalculates.

Search and highlight text between quotation marks

I'm trying to write a macro to search for defined terms, and highlight them so that I can make them bold in with a macro rather that doing this over and over.

greg maxey say:

Do you want to highlight them or make them bold?  If bold then:

Sub FindAndBoldTextInQuotes()
Dim oRng As Range
Set oRng = ActiveDocument.Content
With oRng.Find
 .ClearFormatting
 .Text = "[^0147^01486^34]<*>[^0147^01486^34]"
 .Forward = True
 .Wrap = wdFindStop
 .MatchWildcards = True
 Do While .Execute
   With oRng
     .MoveEnd Unit:=wdCharacter, Count:=-1
     .MoveStart Unit:=wdCharacter, Count:=1
     .Font.Bold = True
     .Collapse wdCollapseEnd
   End With
 Loop
End With
End Sub

Delete the MoveEnd and MoveStart lines if you want the quotation marks bolded as well.

I suspect the OP is after something like:

Sub DefinedTermsMarkUp()
Application.ScreenUpdating = False
Dim RngStory As Range, arrWords, i As Long
'Insert the defined terms into the following array (add/subtract terms as necessary)
arrWords = Array("term 1", "term 2", "term 3", "term 4", "term 5")
For Each RngStory In ActiveDocument.StoryRanges
  With RngStory.Find
    .ClearFormatting
    .Replacement.Text = ""
    .Forward = True
    .Wrap = wdFindStop
    .Format = False
    .MatchCase = False
    .MatchWholeWord = False
    .MatchWildcards = True
    .MatchSoundsLike = False
    .MatchAllWordForms = False
    For i = 0 To UBound(arrWords)
      .Text = "[^34^0147]" & arrWords(i) & "[^34^0148]"
      Do While .Execute
        With RngStory.Duplicate
          .Start = .Start + 1
          .End = .End - 1
          .Font.Bold = True
          .Collapse (wdCollapseEnd)
        End With
      Loop
    Next i
  End With
Next RngStory
Application.ScreenUpdating = True
End Sub
