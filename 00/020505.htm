Private Sub ButtonName_DblClick(Cancel As Integer)
DoCmd.OpenForm "FormName", acNormal, "", "", , acDialog
End Sub

Or 
set the focus to your form either by clicking it or by add to to code above - in which case you don't really need to open in Dialog mode -
       Private Sub ButtonName_DblClick(Cancel As Integer) 
       DoCmd.OpenForm "FormName", acNormal, "", "", , acDialog 
       Forms!FormName.SetFocus
       End Sub

Or
create a public modual like this (allon one line)
       Public Declare Function LoadOnTop Lib "user32" (ByVal hwnd As Long) As Long
and on the form's OnLoad event add this
       Private Sub Form_Load()
       Call LoadOnTop(Me.hwnd)
       End Sub

Lots of ways to do what you want
