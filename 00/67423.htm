marc cai say:

Hi,

In order to get rid of the Error: 577 on Windows Defender, disabling or uninstalling third party antivirus programs will actually get the job done. In addition to your troubleshooting, follow the steps below:

Note: The method to fix the issue will have Registry modification. Before you modify the Registry, make sure to back it up and understand how to restore the registry if a problem occurs. Please refer to this link:

1. On the Start menu, start typing regedit.exe and press Enter.

2. Go to this key address: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Defender

3. Look for DisableAntiSpyware and DisableAntiVirus.

4. Double click each file and change their value data from '0' to '1'.

5. Once done, restart the PC and run Windows Defender.

In case that you could not locate the files in the registry key address provided above, you can also check this: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender.
