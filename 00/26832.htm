amit tandon say:

Presume Table is in Sheet A, range A1:J10, where any no of columns are drop-down lists (drop-down list can also be of a named list) - say column J is a drop-down list. You are probably doing a vlookup (say from Sheet B) where the corresponding cell(s) in column J sheet A is blank and a value is not selected from the drop-down list in that cell(s), and in this case vlookup will return 0. Else, vlookup will work fine irrespective of the table having columns containing drop-down lists.

Example:-  use below formula to do a vlookup from sheet B, for a lookup_value in cell A1 in sheet B, and do a vlookup in table array A1:J10 in sheet A: =VLOOKUP(A1,A!$A$1:$J$10,10,FALSE). If corresponding cell in column J (column 10) is blank, the formula will return 0.

Note, that you can also do a vlookup for a named range: =VLOOKUP(A1,Table,10,FALSE), where Table is the named range A1:J10 in sheet B.

jllatham say:

I'm going to try to lead you step by step through this formula build up so you will understand the rather long formula that we would end up with.  Or you can just jump to the 'last page of the book and see if the butler did it or not' by going to the end of this post and using the formula that we come up with at that point:

First, MATCH()  Match returns the relative position in a list where a match is found.  So if we were to put a formula like =MATCH(AK2,Services!$A:$A,0) it will return the row number of the first match found in that column.  Let us say it found the match at row 9.

Next ADDRESS()  Address returns the cell address for which you supply a row and column reference, so if we put a formula like this on a sheet:

=ADDRESS(9,1)  it would return $A$9  -- because 1 is the number for column A.

Substitute the Match() formula for the row number and we get

=ADDRESS(MATCH(AK2,Services!$A:$A,0),1)  and we still get $A$9

So now we have a formula that will give you the cell address of the first match.  But that is not enough for a hyperlink, we also need to know the name of the workbook and the name of the sheet. 

We will assume the name of the sheet is always going to be Services.  And we will 'hard code' that into the formula.  This still leaves us needing to know the name of the workbook.  I'm not going to assume that it will always be the same name - perhaps other people will rename their copy of it, or you might have several copies of the same type of workbook for different purposes.

This formula will give you the name of the workbook (after the workbook has been saved to disk - doesn't work in a brand new workbook).  Well, first let's see what one piece of this looks like and what it gives us:

=CELL("filename")  this gives us the full path and filename (and the name of the sheet the formula is on) of the workbook, like:

C:\Users\Angelique\Desktop\[HLINKS.xlsm]Sheet2

But we only need this part of it right now "[HLINKS.xlsm]"

So this gets us that - and this is were it starts looking rather complex and ugly:

=MID(CELL("filename"),FIND("[",CELL("filename")),FIND("]",CELL("filename"))-FIND("[",CELL("filename"))+1)

That will give us simply [HLINKS.xlsm]

NOW we can tie it all up into a formula that will give us a hyperlink address to the first cell where a match was found:

=MID(CELL("filename"),FIND("[",CELL("filename")),FIND("]",CELL("filename"))-FIND("[",CELL("filename"))+1)& "'Services'!" & ADDRESS(MATCH(AK2,Services!A:A,0),1)

To show it in a cell as a hyperlink, we would use the HYPERLINK() formula like this:

=HYPERLINK( MID(CELL("filename"),FIND("[",CELL("filename")),FIND("]",CELL("filename"))-FIND("[",CELL("filename"))+1)& "'Services'!" & ADDRESS(MATCH(AK2,Services!A:A,0),1), "Link to First of Multiple Matches")

Finally, lets return to one of the formulas I gave you earlier and put this into the section where we handle multiple matches and see what it all looks like.  I'm going to write this as 3 separate sections, be sure you just make it one long formula in your workbook:

1st line is just a test to see if there are multiple matches.

Starting with HYPERLINK( is the section that sets up the hyperlink when there are multiple matches

Starting with VLOOKUP( is the section that simply returns the value when there is only 1 match in the table.

=IF(COUNTIF(Services!$A:$A,AK5)>1,

HYPERLINK( MID(CELL("filename"),FIND("[",CELL("filename")),FIND("]",CELL("filename"))-FIND("[",CELL("filename"))+1)& "'Services'!" & ADDRESS(MATCH(AK2,Services!$A:$A,0),1), "Link to First of Multiple Matches"),

VLOOKUP(AK5,Services!$A:$I,2,FALSE))
