Project 2010 filter "Changed Cells" - Microsoft Community

shellfish say:

Project 2010 filter "Changed Cells"

When using the Changed cells highlight feature can a filter be used to gather the changed cells (tasks) so they are easy to see and can the changed cells be left on or visible as you continue to edit other tasks in the project plan? I will have novice users trying to see what has changed.

Thank you for your help.

john - project say:

shellfish,

As far as I know, no, change highlighting applies only to the latest edit that impacts schedule fields (e.g. Duration, Start, Finish, etc.). You can get a partial cumulative change highlight by turning off automatic calculation. Edits to each task will immediately show changes for that task but will not display changes to follow-on tasks that may be linked to that task. For example, let's say you have two linked tasks A and B and calculation is off. An edit to the duration of task A will show a change highlight for task A's finish date but it will not show that the dates of  the linked successor, task B, has changed. You can then continue to edit other tasks in a similar fashion. When you are done editing and turn calculation back on, you will see change highlighting to all successor tasks in the network chain but you will no longer see change highlighting for the actual tasks that were edited.

Here's a suggestion. Designate a flag field for keeping track of changed tasks. After each edit, set the flag for those tasks that show change highlighting. When all edits are done, apply a filter for the flag field. The changes won't be highlighted but you will have a filtered set of all changed tasks. Granted, this is a tedious labor intensive approach so it is far from ideal.

It should be possible to automate a "change tracker" with VBA but that is a more complex approach.

The best bet with "novice users" is thorough training. For example, Baseline fields exist to allow the user to compare the original plan with the current plan (i.e. built-in functionality for seeing changes).

Hope this helps.

John

To stop Word from tracking format changes in the current document, do the following: On the Review tab, click the arrow on the Track Changes button, and then click Change Tracking Options. In the Track Changes Options dialog box, clear the check mark next to "Track formatting" and click OK. To remove all formatting markup, do the following: Click the Show Markup drop down and deselect Comments, Ink, and Insertions and Deletions. Leave Formatting selected. Then click Accept | Accept All Changes Shown. Save the file.

suzanne s. barnhill say:

Two approaches:

  1. Prevention: In the Track Changes Options, clear the check box for "Track formatting." If you don't mark formatting changes in the first place, there won't be an issue.
