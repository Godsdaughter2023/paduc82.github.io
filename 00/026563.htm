Ok, in that formula above, if the COUNTIF = 0 then the formula returns a char 32 standard space character. Instead of the space character, have the formula return what it known as a formula blank.

=IF(COUNTIF(ActivityPins!F129,"C")>0,"A","")

Notice the difference:

  • " "
  • ""

Not like this: =IF(COUNTIF(ActivityPins!F129,"C")>0,"A"," ")
Like this: =IF(COUNTIF(ActivityPins!F129,"C")>0,"A","")

Now, in your other formula replace:

COUNTIF(I6:I13,"*")

With:

COUNTIF(I6:I13,"?*")

a. user say:

Hi,

 

Thank you for posting your question here in Microsoft Community. We appreciate the detailed description of the issue that you have provided.

 

We understand that you did not open the affected files on the two computers at the same time but this behavior would only occur in the event of a conflict between multiple versions of the same file. It is how the system prevents you from losing data.

 

What's happening is that the files are being modified on multiple computers before the file has had a chance to synchronize everywhere. Whenever the system detects that there is a newer version on the server, but the local version has been modified, it will rename one of them with the computer name to alert you that there was a conflict.

To avoid this, make sure that you are only modifying files on one machine at a time and that you wait for your changes to be synced everywhere before modifying the files elsewhere. Or, edit the file that’s on SkyDrive and not the synced file from the client.

 

It is a known issue, and a fix is being actively tested. We do not know when this will be released (probably soon), but until then avoid rapid saves of Office files as much as possible.

rich michaels - mvp (greatcirclelearning.com) say:

Hi Viola,

An important concept to understand for this action you are thinking about performing is, Clearing Formatting takes the Style of the selected paragraphs to the Style Formatting of the "Normal" Style of the template attached to the document.

So, to clear the formatting on everything in a document, all you do is Command + A to Select All and then with the Styles Pane open (accessed from the Home tab), click Clear Formatting. All paragraph style formatting will revert to the settings of the "Normal" Style of the attached template.

If that attached template is Word's default "Normal Template" then that will be the style that gets applied to everything you selected. If the attached template is a special template you setup and where you altered the "Normal" style of that template to your unique style setting, then that is how your document's styles will get applied and be called "Normal".
