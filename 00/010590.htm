Putting multiple files (book chapters) into one file - Microsoft Community

hatman99 say:

Putting multiple files (book chapters) into one file

Have written book with 20 chapters.  Publisher asks for one large file with all chapters included.  How do you link Word files of the 20 chapters together into one file???
bob jones aka: cybertaz say:

I may have thrown you off a bit - Was apparently using Word 2011 without remembering that the Insert> Footnote dialog had been redesigned. That's why you couldn't find the Apply features, although it seems you were making the appropriate adjustment :-) Instead, you have these 2 dialogs:

This is what works here using Word 2008;

  1. In a new blank document, insert the first chapter
  2. Use Insert> Break - Section Break (Continuous)
    • Repeat steps 1 & 2 for each chapter (but no Section break after the last chapter)
    • Don't be concerned about the endnotes accumulating at the end of the document
  3. Once all chapters are inserted;
  4. Go to Insert> Footnote
  5. Select Endnote if necessary
  6. Click the Options button
  7. Choose End of Section from the Place at: list
  8. Choose other options as preferred
  9. Click the OK button
  10. Click the Close button

bob jones aka: cybertaz say:

Please understand that this process is virtually seamless if properly planned for from the beginning :-) When attempting to guide someone blindly without ever even having seen the documents involved is highly prone to not take every possibility into account...That's why I provided the 2 answers.

Your original question made no mention of endnotes, for example, which most definitely must be considered. The endnotes, themselves, are not what is "stopping the insertion.", it's a matter of how they are formatted & possibly the method you're using. Essentially, it's [almost] impossible to add content after Endnotes if they are specified as End of document -- Word takes that attribute quite literally :-)

First of all, there should be no need "select all" because there is no copy/paste involved... The chapter documents should not even be open. You're supposed to be starting in a new blank document. You will use the Insert> File method to insert one chapter after another.

  • When you go to Insert> File you will be presented with a file browser window -- similar to an Open or Save As dialog. Use it in exactly the same way;
  1. Navigate to the folder containing your chapter documents,
  2. select the first one to be included in the new document,
  3. click the Insert button. [I suggest that you not use the Link to File option since the combining process was not planned for to begin with.]
  • What is required to adjust the endnotes depends on their current formatting. In all probability this should suffice;
  1. Once you've inserted a document go to Insert> Footnote
  2. Reset the Endnotes location to End of Section
  3. Confirm the preferred Number Format
  4. Specify Numbering: Restart each section rather than Continuous if you wish
  5. From Apply changes to select Whole document
  6. Click the Apply button (not Insert)
  • Next, go to Insert> Break to select Section Break (Next Page)
  • Repeat the process for each of the remaining chapters
