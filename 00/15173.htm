First, why are you using an Update statement like that. Second do it this way:

Dim strSQL As String

strSQL = "UPDATE client " & _

  " SET id=" & Me.txtID & ",  clientname='" & Me.txtName & "'" & _

  ", gender='" & Me.cboGender & "', phone='" & Me.txtPhone & "'" & _

The _ is a continuation character. it tells VBA (not VB) that the code continues on the next line. 

The syntax for an UPDATE query is UPDATE tablename SET fieldname = value. You can add multiple field/value combinations by separating them with commas. 

Using Debug.Print prints the generated statement to the Immediate window where it can be copied and pasted into a query for testing.

To ensure it works, just check the record before and after you run the query.

Try the steps in the following order:
