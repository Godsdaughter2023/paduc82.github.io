doug robbins - mvp office apps & services (word) say:

The following code will display the name of the first bookmark in the selected text and if there is none in the selected text, it will display the name of the first bookmark in the document after the start of the selected text if there is one.  Otherwise it will display a message advising that there are no bookmarks in the document after the start of the selected text:

Dim rng As Range
Set rng = Selection.Range
If rng.Bookmarks.count > 0 Then
    MsgBox rng.Bookmarks(1).Name
Else
    rng.End = ActiveDocument.Range.End
    If rng.Bookmarks.count > 0 Then
        MsgBox rng.Bookmarks(1).Name
    Else
        MsgBox "There are no bookmarks in the document after the start of the selected text."
    End If
End If

jeeped say:
The example of a VLOOKUP() formula you provided was,

=VLOOKUP(101,'Residents'!A1:Z500,2,FALSE)

This would be changed to,

=VLOOKUP(A1,'Residents'!A1:Z500,2,FALSE)
=VLOOKUP(A1,'Residents'!A1:Z500,3,FALSE)
=VLOOKUP(A1,'Residents'!A1:Z500,4,FALSE)

Note that the 2, 3, 4, etc will change depending upon what column of information you want returned from the 'Residents'!A1:Z500 data matrix. With these in place, you should be able to select A1, tap Ctrl+A then Data, Sort.

You probably will not want the automatic sort to occur every time a change is made on the Residents worksheet. If you waited until the Residents is left another worksheet is selected then you wouldn't have to wait for multiple sheets to sort each time an edit is made; just once when the Residents worksheet is exited or deactivated.

You haven't disclosed the names of hte other worksheets, nor what columns are meant to be sorted, but perhaps you can transcribe the following for your own purposes.

Right-click the Residents worksheet's name tab and chose View Code. When the VBE opens, paste the following into the pane titled something like Book1 - Sheet1 (Code), 

Private Sub Worksheet_Deactivate()
    Application.ScreenUpdating = False
    Application.Calculation = xlCalculationManual
    Application.EnableEvents = False
    On Error GoTo FallThrough
    With Sheets("Sheet2").Range("A1").CurrentRegion
        .Cells.Sort Key1:=.Columns(2), Order1:=xlAscending, _
                    Key2:=.Columns(4), Order2:=xlAscending, _
                Orientation:=xlTopToBottom, Header:=xlYes
    End With
    With Sheets("Sheet3").Range("A1").CurrentRegion
        .Cells.Sort Key1:=.Columns(2), Order1:=xlAscending, _
                    Key2:=.Columns(4), Order2:=xlAscending, _
                Orientation:=xlTopToBottom, Header:=xlYes
    End With
    With Sheets("Sheet4").Range("A1").CurrentRegion
        .Cells.Sort Key1:=.Columns(2), Order1:=xlAscending, _
                    Key2:=.Columns(4), Order2:=xlAscending, _
                Orientation:=xlTopToBottom, Header:=xlYes
    End With
FallThrough:
    Application.EnableEvents = True
    Application.Calculation = xlCalculationAutomatic
    Application.ScreenUpdating = True
End Sub

Edit the Sheet2, Sheet3 and Sheet4 to more closely resemble the sheets you actually want to sort. Edit the Key1 and Key2 if need be; it is currently sorting on column B as the primary sort, then column D. Change the order from xlAscending to xlDescending if you like. Add more With Sheet([...]) is you require more worksheet sorting.

Tap Alt+Q to return to your workbook.
