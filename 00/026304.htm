desireek850 say:

So, as soon as I wrote this, i gave google one last try, and I found it!! I have already tested it out, and it works beautifully!! Just change all of the CAP words to your own files! 

Dim objOutlook As Outlook.Application
Dim objEmail As Outlook.MailItem
Dim strAttach1 As String
Dim strAttach2 As String
Dim strAttach3 As String
Dim strAttach4 As String
Dim strAttach5 As String

Set objOutlook = CreateObject("Outlook.application")
Set objEmail = objOutlook.CreateItem(olMailItem)

'Output Reports
Rem <!-- change "Reportx" to match the report names you wish to export. & _
IMPORTANT: Make sure the location you select to save your reports to exists, Access will & _
not create the folders for you. -->
DoCmd.OutputTo acOutputReport, "REPORTNAME", acFormatPDF, "SAVE REPORT LOCATION", False
DoCmd.OutputTo acOutputReport, "REPORTNAME2", acFormatPDF, "SAVE REPORT2 LOCATION", False


'Set Attachments
Rem <!-- make sure to correlate the attachments to each of the reports you wish to send -->
strAttach1 = "SAVE REPORT LOCATION"
strAttach2 = "SAVE REPORT2 LOCATION"


'Generate email
With objEmail
.To = "EMAIL1;EMAIL2"
.Subject = "SUBJECTLINE"
.Body = "BODY TEXT"
.Display
.Attachments.Add strAttach1
.Attachments.Add strAttach2

End With

'Remove attachments from drive
Kill strAttach1
Kill strAttach2

Kill "F:\MyDatabases\MyPDFFiles\Temp\*.*"

mccbleue say:

The Lumia 1520 is a global model. Lumia phones sold in the US and China may have differences specific to those particular markets (and therefore different RM-numbers), generally speaking models sold in all other countries are pretty-much the same.

 

In terms of 4G, Lumia phones may be sold in a particular market as either 4G-Ready or 4G-Enabled. The latter means that the phone can connect to 4G services, the former means that the hardware is not enabled in the software, usually because there is not sufficient coverage in that country at the time of launch. This decision is taken by Microsoft's people in India for the Indian market, and should the network situation change 4G can be enabled in a software update.

 

I would hazard a guess that your phone is 4G-Ready only, it would also explain why the Microsoft web site for India does not show 4G frequencies in the specs.

Default Open Folder In Quick Access Menu - Microsoft Community

robertweiner say:

Default Open Folder In Quick Access Menu

I had been using Windows 10 Pro and Office 365 for over 8 months on a Dell XPS8700. It seems like something has changed with an update.

 

It is only a minor irritation, one among many.

 

When I use an Office program I always set up the Quick Access Menu to my liking. In Excel I added the OPEN icon to the Quick Access Menu. In the past when I clicked on the OPEN icon in the Quick Access Menu I would be taken to my MS Excel Docs folder in my Document Folder. Now when I click on the OPEN icon in the Quick Access Menu I am taken to the File Menu where I must click on BROWSE to get to my desired folder.

 

This is not a major problem, but it is an added inconvenience.

 

Is there any way to setup a default folder for OPEN icon in the Quick Access Menu?

 

Thanks,

 

Bob

ossiemac say:
Select menu item File -> Options -> Save and see if the "Default file location" is correctly set and if not then edit it.

robertweiner say:

Thank you OssieMac,

 

Setting up the default file location is one the first things I do on a new install of Office programs, and have done that for every version since the first version of Office I had in the early 90s.

 

While searching for an answer to my question I checked to see if the same thing was happening in Word. I do not use word very often. Word exhibited the same behavior. In both Excel and Word the default file location is correct.

 

However, in Word, I noticed an option "Don’t show the backstage when opening or closing files". I hadn't noticed it in Excel. It was unchecked. Not knowing what that really meant I checked the box to find out, and lo and behold both Excel and Word now go to the default file location when I click on the Quick Access Menu OPEN icon. When I checked the option in Excel, it was checked.

 

I know Office programs are interrelated, but this is the first time I have seen changing an option in one of the programs effected another.

 

Thanks for taking the time to respond. Hope this helps others with similar problems.

 

Have a great day.

 

Bob

ossiemac say:

Great that you have found that option. See the following tip at the link below where it indicates that it applies to all office applications.

Tip: If you want to bypass File > Open and always use the Open dialog box, do this: Select File > Options > Save, and then select the Don’t show the Backstage when opening or saving files check box. Then, close and reopen any open Office applications.

On the flip side, if I check the "Use Cached Exchange Mode" box, mail takes ages to come (sometime 30 minutes after I get the notification on my phone), but switching from one email to another is almost instant.

How do I configure outlook so it will receive mail fast, and switch between emails quickly too? 

Running office 365 on Windows 10. 

barry xie msft say:

Hi Cgfrank1,

 

If your network environment is not well, we recommend that you turn on Cached Exchange Mode feature on the Outlook client. Cache Exchange Mode enables a better experience when you use an Office 365 account. In this mode, a copy of your mailbox is saved on your mailbox is saved on your computer. This copy provides quick access to your data, and it’s frequently updated with the server.

 

For a better user experience, use Cached Exchange Mode in a poor network environment and turn off this feature when the network environment is good. You can decide by yourself.
