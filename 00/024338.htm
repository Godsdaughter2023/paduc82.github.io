Macro to overcome the problem with propagation of mail merge field fields with a label type mail merge when using a Tablet PC and some other Computers that have Tablet PC Functionality available:

Sub MailMergePropagateLabel()

Dim atable As Table

Dim i As Long, j As Long

Dim source As Cell, target As Cell

Dim myrange As Range

Set atable = ActiveDocument.Tables(1)

Set source = atable.Cell(1, 1)

Set myrange = source.Range

myrange.Collapse wdCollapseStart

ActiveDocument.Fields.Add Range:=myrange, Text:="NEXT", _

PreserveFormatting:=False

source.Range.Copy

For j = 2 To atable.Columns.Count

    Set target = atable.Cell(1, j)

    If target.Range.Fields.Count > 0 Then

        target.Range.Paste

    End If

Next j

For i = 2 To atable.Rows.Count

    For j = 1 To atable.Columns.Count

         Set target = atable.Cell(i, j)

         If target.Range.Fields.Count > 0 Then

              target.Range.Paste

         End If

    Next j

Next i

atable.Cell(1, 1).Range.Fields(1).Delete

End Sub

 

It is suggested that you put this code into a template that you save in the Word Startup folder so that it becomes an add-in and will be available whenever you need it.  As the macro has the same name as the Word Command that is supposed to do the job, this macro will run when the Update Labels button is used when setting up the mail merge label main document.

4. Select Advanced Setup

6. Select Internet Mail.

13. Make sure the two boxes (Require SSL for incoming email and Require SSL For Outgoing Email) are checked.

14. Scroll back up to the Outgoing (SMTP) Email Server line.  In that line, type: smtp.aol.com

sven johannsen say:

Brian,  you indicated Win7. Aren't you using Windows Mobile Device Center? In any case, the way it has worked in the past is that music (or any media) syncing was taken care of by Windows Media Player. If you have your device hooked up to the PC, open Media Player and create a sync partnership/profile/whatever with your device. You should be able to designate either the Device, or a memory card in the device as the destination for that sync setup. They should show up individually on the left side tree. Typically it is better to set that up as automatic. What I have done is created some "Portable Playlist" and made that the object of my automatic sync. That way I can manage that playlist on the PC and changes are reflected on the device.

What used to happen with ActiveSync/WMDC was that checking music made the sync established via Media Player happen, even if Media Player wasn't open. That is it triggered the Media Player sync. If you didn't check music, you would need to start Media Player while the device was connected to kick off the sync.

Hope it helps. The key is that flash cards are seen as independent items with regard to Media Player syncing. You should be able to do what you want. I have. Haven't done it much though, because I prefer a dedicated media player, Zune. Don't like to risk running the battery down on the phone.
