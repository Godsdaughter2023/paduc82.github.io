Problems with Windows Update? - Microsoft Community

georgeogden say:

Problems with Windows Update?

OK, we've all (probably) had some issues with Windows Updates or our PCs over the various builds due to failures to install for one reason or another. Some are simply down to corrupt or incomplete downloads. Whatever the reason, the idea of a batch file to clean up the detritus and reset Windows Update took hold and below is the result of my efforts to make it simple.

I've tested it, several times, and this is the final version ..... unless of course  you have any suggestions for improvements :) .... Also, I suspect I may have missed a couple of entries in the section which reregisters the BITS and the Windows Update files so, if you know what I've missed, please, let me know so I can update it and include them.

To create your own batch file simply copy the text between the ***s, open Notepad and paste it into a new text file. Once you've saved it, rename it as "Reset Windows Update.bat" or something similar BUT make sure you use the .BAT ending. Run it by right clicking and "Run as Administrator".

Once you've rebooted, try going to settings and checking for updates again

Obvious health warning - it works for me on build 10162 but it should also work on any earlier (and future?) builds just as well. That said, if you have any doubts whatsoever DO NOT use this if you do not have a current backup of all your data.

One final comment... I've noticed that when copying and pasting the text below a number of spurious blank lines appear in the text document. These can either be ignored, as it does not impact the working of the batch file, or deleted if you, like me, want to tidy it up. :)

*********************************************************************************

@ECHO OFF
ECHO.
ECHO *** WARNING ***

ECHO.

ECHO This batch file will reset the Windows 10 update components.
ECHO For it to work it MUST be run at the Administrator level.
ECHO To run as Administator right click on the batch file and select "RUN as administrator". 
ECHO.
ECHO Please confirm that you are running this as Administrator and wish to proceed.
ECHO.
CHOICE /N /C:YN /M "ARE YOU SURE? Y or N"%1
IF ERRORLEVEL ==2 GOTO END
IF ERRORLEVEL ==1 GOTO RESET_IS_YES
GOTO END
:RESET_IS_YES
ECHO.
ECHO Stopping BITS, Windows Update and Cryptographic services.
   net stop bits
   net stop wuauserv
   net stop appidsvc

ECHO.
   net stop cryptsvc

ECHO.

PAUSE
CLS

ECHO.
ECHO If you are prompted for the administrator password or

ECHO.
PAUSE

zack barresse say:

Ok, so you want to return a value on this other sheet? If so, you can do that with a couple tweaks...

Public Function AdjacentSheetLink( _
       ByVal LinkCell As Range, _
       Optional ByVal SheetRight As Boolean = True _
       ) As Variant

    Dim Sheet As Worksheet
    Dim Index As Long

    On Error GoTo AdjacentSheetName_Error

    Set Sheet = Application.Caller.Parent

    If SheetRight Then Index = WorksheetFunction.Min(Sheet.Index + 1, Sheet.Parent.Worksheets.Count)
    If Not SheetRight Then Index = WorksheetFunction.Max(Sheet.Index - 1, 1)

    AdjacentSheetLink = Sheet.Parent.Worksheets(Index).Range(LinkCell.Address).Value
    Exit Function

AdjacentSheetName_Error:
    AdjacentSheetLink = CVErr(xlErrNA)

End Function

To call this function you'll now need to reference the cell itself, but it must be on the calling sheet. This means, if you're on 'Sheet1', and you want to reference cell A1 on 'Sheet2', then in 'Sheet1', type in the formula:

    =AdjacentSheetLink(A1,TRUE)

Parameters:

  1. Cell to link. Range. Required. Specify the range of the cell to reference in the indicated worksheet. Relative to address only.
  2. Look right. Boolean. Optional. True by default. Change to False to look left one sheet instead of right (by index).

So in the above example, you could have omitted the "TRUE" portion, as that is the default.

Here is an example of something you could use:

Public Function AdjacentSheetName( _
       Optional ByVal SheetRight As Boolean = True _
       ) As Variant

    Dim Sheet As Worksheet
    Dim Index As Long

    On Error GoTo AdjacentSheetName_Error

    Set Sheet = Application.Caller.Parent

    If SheetRight Then Index = WorksheetFunction.Min(Sheet.Index + 1, Sheet.Parent.Worksheets.Count)
    If Not SheetRight Then Index = WorksheetFunction.Max(Sheet.Index - 1, 1)

    AdjacentSheetName = Sheet.Parent.Worksheets(Index).Name
    Exit Function

AdjacentSheetName_Error:
    AdjacentSheetName = CVErr(xlErrNA)

End Function
