Is the device you're playing the game on using Intel integrated graphics? If so, it's the driver that's the issue.

My mum had this exact same issue on her Surface. The game would boot up OK, but as soon as she chose a puzzle and it was loading up, the screen would go black, then the game would crash and disappear. An error was logged in Event Viewer each time, listing a file related to the Intel graphics driver as the cause.

As you suspected, it's loading an ad when it crashes. A full screen video ad.

The solution is to update the Intel driver to the latest version, which you can get from Intel's website. Once you've done that, the ad will play and you can get beyond it and play the game.

The only problem you might have is with Windows Update deciding that you're not running the "correct" driver, which leads it to download and install the driver you were using previously with no option to stop it, then of course the issue returns. What I did in that instance, was to allow the bad driver to be reinstalled, then I went into Device Manager and rolled back the graphics driver, which put me back on the latest driver again, and when it asked why I was rolling back I chose the option that said the driver wasn't working properly. That seemed to stop Windows Update from reinstalling the dodgy driver over and over again, however, some people have had to use other means to stop it, but yeah - see how you get on.

Windows 10 Update Fails Error 0x800705b4 -Solution- - Microsoft Community

terrymuench say:

Windows 10 Update Fails Error 0x800705b4 -Solution-

=> THE PROBLEM

I have Windows 10 on all 3 of my home machines; two are on the original release (v1511) and I was forced to update the third to the Anniv. Update (v1607). Windows Update (WUP) was working on all machines until about a week ago. On the two v1511 machines, "Check for updates" would stall and after 30m report Error 0x800705b4. However on the v1611 machine it was working fine. Hmmmm. Sounds like Microsoft might be doing this intentionally to "encourage" users to upgrade to the Anniversary Update. 

It took a lot of time and research and fix this. I hope what I learned will help others who have this problem. 

=> THE SOLUTION

1. Download Microsoft Hide-Show-Updates Tool if you haven't already

This tool from Microsoft will let you block (hide) updates you don't want ... for example the 'Feature Update' v1607 that has caused many many problems. 

2. Download KB_Extractor (here) and extract the EXE  to your Desktop

This is a great little utility that extracts the contents of a CAB file. I have never been able to do this using the built-in DISM -- it always says "File not found". 

3. Run the Hide-Show-Updates Tool and see what updates are pending

This may take a l-o-n-g time but should eventually complete. Select [Hide updates] and make a list of all other-than-driver updates -- OS, MSO, Flash, etc. -- along with their KB#'s. I had six MSO updates, one OS (KB3205386) update, one Flash update plus MSRT (Malicious Software Removal Tool). 

4. Download each update from the Microsoft Catalog http://www.catalog.update.microsoft.com/Home.aspx

Here you can search by KB# and download each installer; make sure you download the correct Windows 10 version for x86 or x64. 

I downloaded six MSO (CAB) files and one OS (MSU) file. I skipped Flash and MSRT. 

In case you're not familiar with these file types a CAB file is a compressed Cabinet file and a MSU file is a Standalone Microsoft Installer File. More on this later. 

5. Install updates from each CAB or MSU file in the order listed by the Hide-Show-Updates Tool

CAB: simply drag the file from File Explorer and drop it on KB_Extractor

a. the utility will extract the contents to a TEMP folder and open that folder
b. the folder should contain a MSP file and a XML file
c. right-click on the MSP file and choose Apply

MSU: right-click on the file and select Open

a. this will bring up the Windows Update Standalone Installer
b. if the update is not already installed, click [Yes] to install

6. When all updates have been installed goto Settings > Updates & security > Check for updates
If the Moon is in the Seventh House and Jupiter is aligned with Mars (look it up), this should complete and you will see a list of updates not installed ... in my case Flash and MSRT. 
Install these updates and Restart if necessary. 

7. Run the Hide-Show Updates Tool and select [Hide updates]

Only Driver updates should be listed!

WSR, Windows Speech Recognition, is listening for sounds of words. However, extraneous sounds in the environment are often interpreted as words if they enter the audio stream between your mouth and the microphone. The best answer is to eliminate these sounds from the audio stream. If you are using a far field microphone like those built into the notebook computer/tablet, try saying Stop Listening when you are not dictating and then Start Listening when you wish to dictate or issue commands. Alternatively you can hold the Windows Key down and tap the Control Key to act as an On/Off toggle.

For best accuracy you should use a good noise canceling close talk microphone. The simplest are ones found on a headset with the microphone at the end of a wire boom. If you are already using a good noise canceling headset, place the microphone element at the end of the boom off the corner of your mouth. This will avoid breath sounds from entering the audio stream. (Removed by FM)
