First, I’d like to explain that each plan in Office 365 Planner will create an Office 365 Group and related files is actually stored in Office 365 Group file library.

Back to your questions: the answer is no, there aren’t any out-of-box solutions to move files from SharePoint document libraries to “Planner” files (Office 365 Group files).

Instead, we can share files in SharePoint document libraries with Office 365 Group users to let users view or edit these files. Moving files from SharePoint document libraries to Office 365 Group files library is unnecessary. To do so:
1.Login to the SharePoint document library.
2.Select the file and click Share.
3.Enter the group name (or the name of the plane in Planner). In my scenario, the plan is called Public Group.
4.Click Share.

5.Login to Planner and find the plan.
6.Click ellipsis> Files>  and you will see those shared files.


If you still want to move files from SharePoint to Office 365 Group files, here is another workaround for you (Note: the metadata and versions will lose in this method):
1.Login to the SharePoint document library using Internet Explorer.
2.If you see the document library looks like image below, please click Settings gear> View in File Explorer.


3.If you see the library similar below:

4.Click Library tab and then click Open with Explorer. This action will open the document library in Windows Explore.

5.Leave the window there and go to Office 365 Portal. Then click OneDrive.
6.Click the Office 365 Group name under Groups. Here, I still use the Public Group.

7.Go back to the Windows Explore window. Then drag and drop files into the group files window in Internet Explorer.
