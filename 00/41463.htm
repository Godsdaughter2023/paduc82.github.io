Display Driver nvlddmkm.sys stopped respoding - Microsoft Community

yrral25 say:

Display Driver nvlddmkm.sys stopped respoding

Just this week my HP Pavillion DV2700 got problem when windows updated my Display Adapters Nvidia GeForce 8400M GS.
Screen slowly distorted till i got blue screen message said "Display Driver nvlddmkm.sys stopped respoding. System Halted".

I just noticed that any uninstall and reinstall of nvidia drivers will not remove the old nvlddmkm.sys file and newer version didn't overwrite the old nvlddmkm.sys file.

I found the solution and fixed the problem today. What happen during the installation of the most current drivers, an OLD file nvlddkm.sys is copied into windows/system32/drivers and not the current one in the install. Result of this, new drivers are attempting to access a file dated 6/25/2008 instead of 11/21/2009 which is the newest WHQL driver for vista 32.

Solution: Go to windows/system32/drivers and rename nvlddmkm.sys to nvlddmkm.sys.old. Go to the nvidia directory and find the file nvlddmkm.sy_  and copy it to windows/system32. Using command window (DOS BOX) type EXPAND.EXE nvlddmkm.sy_ nvlddmkm.sys. When expansion is complete, go to windows/system32, look for the new nvlddmkm.sys and copy it to windows/system32/drivers and restart the computer. Computer should now work properly.

Question: Windows Updated for newer version of my nvidia driver which caused my display distorted and malfunctioned. Worst thing... computer keeps on restarting over and over. I don't know why this happens? Do  you have a better solution for this matter?

Thanks 

irfan h say: Hello yrral25, If the optional Hardware update for Nvidia has caused the problem, I suggest that you don’t install this update if this is offered at Windows update.

First thing when you install the Nvidia drivers on the computer it would also install associated program. I suggest that you remove the drivers and uninstall the programs from the Programs and features.
Remove the display driver
========================

1.      Click Start. In the Start Search box type "devmgmt.msc" and press Enter.
2. Expand "Display Adapters", double-click on the entry of the display adapters.
3. Click the tab "Driver", click "Uninstall".
4. Click Start. In the Start Search box type "appwiz.cpl" and press Enter.
5. Remove the entries related to the display adapter.

Once you clean up all the drivers, then download the drivers from the Nvidia Website and try to install it once more. Here is a link to download the latest drivers from the Nvidia website: http://www.nvidia.com/Download/index.aspx?lang=en-us Note: Please make sure that you don’t install the Nvidia update if this is causing the problem on your computer.
