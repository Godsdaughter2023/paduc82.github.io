hansv mvp say:

After inserting a Next Page section break before and after the page, activate the header. In the Design tab of the ribbon, the Link to Previous button is highlighted by default, both on the blank page and on the page after it. Click it to "unlink" the header from the previous section. Similar for the footer.

suzanne s. barnhill say:

If you have inserted a section break (Next Page) before page 6, then page 6 will be the first page of this new section and will get the First Page Header/Footer, which you leave empty. Page 7 will have a page number (7), and numbering will be continuous from that point.

If you want to restart numbering in this section, then you may not be able to start at 1 without getting a blank page 6 in the previous section because Word will sometimes not start numbering at 1 without forcing an Odd Page break (this usually results from having "Different Odd and Even" or "Mirror Margins" enabled because both choices indicate to Word that you intend to duplex the document). So the sixth page you actually see will be page 1, your seventh page (second page in Section 2) will be page 2, etc., but they will actually be the seventh and eighth pages in the printed document.

Using the "Set as Default Line" or "Set as Default Shape" commands in a document works only within the current session, but it's forgotten as soon as you close Word. To set a true default:

Use File > Open (or Ctrl+O), navigate to your Templates folder (usually %appdata%\Microsoft\Templates), and open the Normal.dotm template as a document.

Insert a line shape and change its color to black. You can also set the line weight if you wish.

Right-click the line and click Set as Default Line.

Delete the line.

Repeat for a shape (e.g., rectangle) if you want to change the default for shapes.

Save and close Normal.dotm.

a. user say:

Hi,

Since Windows Defender is a malware protection software built into Windows 10, we'd like to know more why you're trying to disable it. In the meantime, you can follow these steps to disable Windows Defender:

  1. On the Start menu, type in Local Group Policy Editor then press Enter.
  2. In the left pane of Local Group Policy Editor, look for this path  Computer Configuration\Administrative Templates\Windows Components\Windows Defender Antivirus and click it.
  3. Look for Turn off Windows Defender Antivirus, double click/tap on the Turn off Windows Defender Antivirus policy to edit it.
  4. Choose Disabled and click OK.
  5. Once done, you can close the Local Group Policy Editor.
