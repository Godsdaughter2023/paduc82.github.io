imran chand say:

Hi,

Which application are you using to burn CD?

Assuming you are using Media Player, The files will have two dates, Date Created, and Date Modified. The Date Modified is the more important one because that's the date that indicates the last time the file was updated or changed.

Date Created is a file system entry that says "this copy of the file was created on date X". That means that any time you make a duplicate of a file, and then the file system applies current date and time that to the Date Created field of the new copy. The Date Modified remains the same as the original source file.

Windows XP needs a third party add-on to maintain the Date Modified on the files you are putting on CDs and DVDS.
