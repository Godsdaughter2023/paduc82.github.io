If one mouses over the presentation part of the Presenter View screen, the Main display gets overlaid with the Presentation, covering the "Swap Displays" button, requiring one to exit presentation mode entirely in order to continue.

The workaround is to move your mouse to the "Swap Displays" button without the mouse going over the Presentation image (navigate the mouse around the edges of the view - this took me a month to figure out), hit "Swap Displays", which makes the Presentaiton show properly on the Primary Display and the Presenter View on the 2nd Display.

Then hit "Swap Displays" again (no need for mouse maneuvering) and you'll be in the proper display mode, with Presenter View on the Primary Display and the Presentation on the 2nd Display.

Another workaround is to disable Preferences > Slide Show > "Always use Presenter View for 2 Displays".

Yet another workaround is to set Sldie Show [Menu] > Show Options... > "Browserd by an individual (Window) & scale up the resulting window on the 2nd display.

Neither of these workarounds allows for the use of the excellent Presentation View tools.

bob jones aka: cybertaz say:
Yes, you can copy the fonts from another Mac. Assuming the system is set up 'normally' there should be a Microsoft folder in the Library/Fonts folder. Copy the Microsoft folder to a USB stick or other shared location, then copy that folder into the ~/Library/Fonts folder on your Mac [or use Font Book to install the fonts if you prefer]. Note that it's best that no programs be running when installing fonts.

I believe you could also use another individual's Office 2011 install media to do a Custom installation of fonts only.

Before launching any of your programs:
  • Make sure to run Font Book to Resolve Duplicates. Doing as above most likely will cause some duplication... a number of fonts [e.g., Arial, Times New Roman] are installed both by Office & OS X
  • Go to ~/Users/UserName/Library/Application Support/Microsoft/Office/Office 2011. Delete the Office Font Cache file & empty the Trash
  • Also -- as a precaution -- repair disk permissions & restart your Mac
Word will rebuild the Office Font Cache file when you launch the program, so it will take a moment longer to launch. After that Office apps should launch more quickly.
