don cuthbert say:

That is like moving a hard disk to a different computer and expecting Windows to remain activated.  The other device was never licensed for Windows 10, so will not be activated.  Windows is licensed per device, whether physical or virtual.  Each VM on the same host PC needs it's own license for Windows and activation will be based on having upgraded from a qualifying OS, such as an activated Windows 10 Insider Preview, on each VM.

So once you have an activated VM with Windows 10, you can replace the VHDX file, or wipe all partitions from it.  Clean installs of Windows 10 or newer Insider Preview Builds will activate, but you can't replace the VM itself, if you want it to remain activated.  The VM has become a licensed Windows 10 device, with the ability to clean install Windows 10 with no product key.  Changing the VHDX file is not enough of a change to deactivate the VM device as a licensed Windows 10 device.
