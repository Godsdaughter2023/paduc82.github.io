Assuming that your WD external drive has nothing you need on it at this time, you can format it as NTFS.

If you open File Explorer and can see the WD drive, right-click on it and a menu should appear that includes Format.  Choose Format.  First choose Restore Device defaults and then, if necessary, under File system, choose NTFS.  Ensure that Quick Format is checked, or it will take a very long time.  You can give it a label if you wish.

netsh lan>sh in

marc cai say:

Thank you for posting, Kamron.

This issue, frame rate capped in 60fps, has been known even from the previous Windows 10 versions. Many of the users reported that the culprit is the Xbox app feature called GameDVR. Try disabling it:

  1. Open Xbox App.
  2. Sign in (this should be automatic if you've signed in before)
  3. Click on the Settings (gear icon) button
  4. Go to GameDVR and turn it OFF.

If you do not have a Microsoft account, you may try the known registry tweak:

Note: It is helpful to back up your Registry settings before proceeding: https://support.microsoft.com/en-us/kb/322756

  1. Open Registry Editor (Run > regedit)
  2. Go to HKEY_CURRENT_USER\System\GameConfigStore
  3. Set the value of DWORD "GameDVR_Enabled" to 0
  4. Go to HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\
  5. Create key "GameDVR"
  6. Create DWORD 32bit called "AllowgameDVR" and set to 0

Feel free to chime back about any results.
