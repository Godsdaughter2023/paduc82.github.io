Similarly, Font Weeding is very important.  There is a core set of Apple and Microsoft Fonts that must be available, and no duplicates, or Word will be glue-slow.

You may also wish to Maggie that document:

The Maggie:


1. Create a new blank document in .docx format

2. Carefully select all of the text in the bad document EXCEPT the last

paragraph mark

3. Copy it.

4. Paste in the new document.

5. Save under a new file name and close all, then re-open.


This technique for de-corrupting is known as "Doing a 'Maggie'", after

Margaret Secara from the TECHWR-L mailing list, who first publicised the

technique.


However, if this is an important document, you may wish to re-construct it by pasting the text into a new document as "Unformatted Text" and re-formatting.  Once a document slows down due to corruption, you can't really trust it in the future.


Hope this helps
