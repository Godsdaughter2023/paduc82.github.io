srfreeman say:

No, you cannot have more than one instance of PC settings.

It is true that this preview provides several ways to access the configuration settings (PC settings, zPC settings, Control panel, etc...) necessary to properly personalize the user experience. In later builds this will become more consolidated based on user feedback, so now is the time to provide detailed explanation of how you want these things to work and/or be accessed, via the Feedback app.

The current Start menu, severely, restricts usage of the search system, opting more for the old 'scroll through the menu until you find it' method of accessing things. The current Start menu is just a prototype that was designed to showcase the possibilities back in Windows 8.1. Either the prototype will be fleshed out in later builds or it will be left in as a placeholder to be used by the developer community to enhance or it will go away altogether.

Without the Start menu in the way, the Search experience makes it very easy to find and access the various personalization settings for the new OS (there are considerably more options than existed previously) with the new 'Windows settings' filter - accessible on the PC via the drop down list above the Search field or the Windows Logo Key + W keyboard short cut. For example; Typing Windows Logo key + W network will provide a list of all available settings concerning the network.  Typing Windows Logo key + W account will provide a list of all available settings concerning accounts. The Search fly out will come in from the right side of the screen, without affecting any other apps you have currently running.

The Search experience is controlled by the new indexing capabilities that were introduced in Windows 8.1, if you are coming from Windows 7 or below to this new OS it could seem foreign to you. A bit of research on this subject may be in order.

Many users find it strange that Internet based resources now show up as results in what they consider to be a 'desktop' search. While it just represents the more 'connected' life of the PC, these 'Internet' features can be easily turned off.

The largest amount of 'Internet' resources appearing in Search results come from addition of the Bing index to the local indexing processes. This can be turned off through zPC settings > System > Search > Use Bing to search online. Other Internet resources may appear in Search results due to the default settings that provide indexes of the Internet Explorer history files and the user's Favorites files, these can be turned off via the Indexing Options settings.

don cuthbert say:

I have a 2007 laptop that also was upgraded from Vista to Windows 7 to Windows 10. It is Legacy BIOS only, so none of the UEFI info applies.

Windows 10 by default, on a Legacy BIOS PC, creates a 500MB NTFS combined System and Recovery partition at the beginning of the disk, which also needs to be marked "Active" to be able to boot. The rest of the disk can be a single NTFS partition. But if you let Windows setup configure this, after booting with install media on a USB stick or DVD, that is the disk layout that you will get automatically. I think that some of the recent Windows 10 install media will not fit on a normal DVD, which is another reason to use a USB stick instead.

My old laptop's Wi-Fi adapter is getting to be not compatible with the latest Windows 10 and causes blue-screen crashes, so I recently bought a tiny external TP-Link Wi-Fi adapter. It fits in one of the USB ports and is barely even noticeable. They are only about $10. There are USB adapters like that too, which can do the newer USB 4.0 if you need that.
