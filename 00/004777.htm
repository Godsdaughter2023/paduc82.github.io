Right-click in one of the heading paragraphs and choose Paragraph... Select the Line and Page Breaks tab of the Paragraph dialog. Clear the check box for "Page break before" and click OK. The easiest way is to select the text, and to click the Bold button on the Home tab of the ribbon until none of the text is bold (this usually takes two clicks).

 

If you prefer to use the Replace dialog:

  • Press Ctrl+H to open the Replace dialog.
  • Click More >> to display the extra options.

  • Click Format > Font...
  • Select Bold in the Font Style list.
  • Click OK.

  • Select Not Bold in the Font Style list.
  • Click OK.

sum if match - Microsoft Community

cody6173 say:

sum if match

I'm looking for a formula that will sum a column if two other columns match.

 

For example:

 

if column R of worksheet 1 contains the same value as column L of worksheet 2 then I would like the sum of the corresponding values in column J from worksheet 1

hansv mvp say:

Let's say you want to sum across rows 2 through 100:

 

=SUMPRODUCT(('Worksheet 1'!R2:R100='Worksheet 2'!L2:L100)*'Worksheet 1'!J2:J100)

 

Change the names of the worksheets to the ones you're using (the single quotes around the names are only required if the worksheet names contain spaces or punctuation).

otávio alves ribeiro say:
