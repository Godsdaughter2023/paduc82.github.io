General Recommendations For Setting Up Users In Vista:

You absolutely do not want to have only one user account. Like XP and all other modern operating systems, Vista is a multi-user operating system with built-in system accounts such as Administrator, Default, All Users, and Guest. These accounts should be left alone as they are part of the operating system structure.

You particularly don't want only one user account with administrative privileges on Vista because the built-in Administrator account (normally only used in emergencies) is disabled by default. If you're running as Administrator for your daily work and that account gets corrupted, things will be Difficult. It isn't impossible to activate the built-in Administrator to rescue things, but it will require third-party tools and working outside the operating system.

The user account that is for your daily work should be a Standard user, with the extra administrative user (call it something like "CompAdmin" or "Tech" or the like) only there for elevation purposes. After you create "CompAdmin", log into it and change your regular user account to Standard. Then log back into your regular account.

You see the following error code and message when you try to connect to Xbox Live on a different console. Sorry, Xbox Live profiles can't be downloaded right now. Try again later.

8015190E

This may mean that there's a problem with the Xbox Live service or that your Xbox Live account is blocked.

Solutions

To resolve the problem, try the following solutions.

Solution 1: Clear the system cache of the Xbox 360 console that you're using. Here's how:

Press the Guide button on your controller, go to Settings, and select System Settings.

Select Storage or Memory.

picture

Highlight any storage device, and then press Y on your controller. (It doesn't matter which storage device you select; the cache will be cleared for all storage devices.)

Select Clear System Cache.

picture

When prompted to confirm storage device maintenance, select Yes.

If you still experience the problem, try the next solution.

Solution 2: Unblock your account. Here’s how:

Go to the Microsoft account (formerly known as "Windows Live ID") sign-in page.

Enter the email and password that you use for your Xbox Live account.

If you see an email telling you that your Xbox Live account is blocked, follow the steps in the email to unblock it.

Note You may be required to access the original email account that you used to set up your Xbox Live account, and also to reset the password for your Microsoft account.

If you still experience the problem, try the next solution. Sign-in

Solution 3: Wait 2 to 4 hours. There may be a problem with the Xbox Live servers. Wait 2 to 4 hours to log on to your Xbox Live account, and then try again.

If you still experience the problem, try the next solution.

Solution 4: Contact Xbox Support

If you are still experiencing the problem, contact Xbox Support

al jarvi say:
Original Title:  my wirless connection is attempting to anothe network connection which is not required to me
my required network connection which is fixed in my computer is not properly connecting when i on the computer but it is automatically niegbouer's connection try to picup in reult both connection countered and my explorer given connection error what is the perminent soloution for this problem ?

Are you using a unique wireless network SSID with your wireless router? If not that could cause a problem like this. Make sure you change from the default. See this for other wireless network security guidelines...

http://theillustratednetwork.mvps.org/LAN/SoHoWirelessSecurity.html

Otherwise...

Open up the Network and Sharing Center then click on Manage wireless networks on the left. Select your preferred network, right click and go into the properties of the network you want to automatically stay connected to and make sure the Connect automatically when this network is in range check box is checked. On any other networks make sure that box is cleared. You can also delete any nearby unwanted networks that you may have inadvertently connected to in the past.

http://cid-25ab668da65c8fbe.photos.live.com/self.aspx/Windows%20images/Autoconnect.png

You might also use merge/delete to delete any old or unwanted networks.

http://cid-25ab668da65c8fbe.photos.live.com/self.aspx/Windows%20images/MergeorDelete.png

Lastly if you want to completely block access to a specific network use a netsh command (as an administrator) to block access to the problematic network...

netsh wlan add filter permission=block ssid=<ssid_to_be_blocked> networktype=infrastructure

See the first part of this page for details...

http://theillustratednetwork.mvps.org/Netsh/Block_wireless_networks.html

If you want to block access to all nearby networks except your own run these commands as an administrator...

netsh wlan add filter permission=denyall networktype=infrastructure
netsh wlan set blockednetworks display=hide
netsh wlan add filter permission=allow ssid=Your_SSID networktype=infrastructure

All nearby wireless networks will be blocked and hidden while you will still be able to connect to your private network, ie. the only one listed in the available networks list.
