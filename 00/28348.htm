Unlicensed user account

I know that if an user account is deleted, Microsoft will only retain the mailbox for about 30 days before it will be permanently purged.  So, I wonder for unlicensed user accounts, will the mailbox and mail content of the associated account be retained? If so, for how long?

Thanks!

richard xu msft say:

Hi FredPoon,

As we know, when a user’s license is removed, all data that is associated with that user account is held for 30 days. After the 30 day grace period, the data is deleted and can’t be recovered, except for documents that are saved on SharePoint Online. If you want to keep the user's email, BEFORE you delete the account, move the email to a different location. If you've already deleted the account: if it's been less than 30 days you can restore it, then move the email data, then delete the account.

About the deleted a user who is unlicensed, the data retention time also is 30 days. Meanwhile, we strongle recommend you should save the contents of a employee's mailbox before you delete it:

  1. Add the this email address to your version of Outlook 2013 or 2016, and then export the data to a .pst file. You can import the data to another email account as needed. To learn how to do this, see Get access to and back up a former user's data.
  2. Place a Litigation Hold or In-Place Hold on the mailbox before the deleting the user account. This is much more complicated than the first option but worth doing if: your Enterprise plan includes archiving and legal hold, litigation is a possibility, and you have a technically strong IT department. Once you convert the mailbox to an “inactive mailbox,” administrators, compliance officers, or records managers can use In-Place eDiscovery tools in Exchange Online to access and search the contents. Inactive mailboxes can't receive email and aren't displayed in your organization's shared address book or other lists.

Thanks,

Richard

amit zaveri say: Hi Bill,

Excel offers two types of drop down boxes
- Form Controls
- ActiveX Controls

Form controls have restriction on font size and like HansV mentioned you can control font size only by adjusting zoom size.

ActiveX controls on other hand has options to give font size and you can use them in similar manner as Form control but with greater flexibility.

Like in Form Controls you can set "Input Range" and "Cell link" using Format control; for ActiveX control you can set "ListFillRange" and "LinkedCell" using Properties (you have to type the range).

Please note to edit properties of activex control, "Design mode" should be on. And to use combobox, "Design mode" should be off.

http://office.microsoft.com/en-us/excel-help/add-a-list-box-or-combo-box-to-a-worksheet-HP010236681.aspx#BMadd_or_edit_a_combo_box_%28control_tool

hansv mvp say:

The size of the text in the validation dropdown list can't be changed except by setting a higher zoom percentage.

The size of the value entered in the cell is simply the font size of the cell, you can set that to anything you like. But if the cell has been set to "Shrink to fit" in the Alignment tab of Format | Cells, Excel will decrease the font size if the value doesn't fit in the cell at the selected font size.
