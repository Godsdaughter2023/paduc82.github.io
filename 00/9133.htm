a. user say:

Hi Jakub,

Thank you for your interest in Windows 10 Technical preview.

Method 1

If you are facing any issues with accessing this app, that might be because you may be using a different Microsoft Account for login than the one you used for Windows Insider Preview registration.  Try logging in with an account linked to your Windows Insider Preview registered Microsoft Account and it should work.

Method 2

I suggest you to check if KB3036140 Update is installed. Refer to the steps below:

  1. Press Windows key + W.
  2. Type Windows Updates.
  3. Click on Windows Updates.
  4. Click on Advanced Options.
  5. Click on View your Update history.
  6. Check if KB3036140 is installed.

If KB3036140 is not installed, I suggest you to install the update which resolves many issues including Start Menu not launching.

Reference: https://support.microsoft.com/kb/3036140?wa=wsignin1.0

Method 3

If the update does not help, you may convert the Microsoft account to a Local account and back to Microsoft account and check if you are then able to run the apps:

To convert the account, you can do the following:

  1. Sign in to your Microsoft account on Windows 10 Technical Preview.
  2. Click Start and then PC settings.
  3. Click Users and accounts and under Your Profile click Disconnect on the right of the screen.
  4. Enter your Microsoft account's password, and click on Next.
  5. Type in a Username, Password, and Password Hint that you want set for your Local account and click Next. Click on Sign out and Finish button.

In order to switch back, follow these steps:

  1. On the newly converted Local account, click Start and then PC Settings.
  2. Click Users and accounts and under Your Profile click Connect to a Microsoft account.
  3. Enter the Microsoft account ID and Password and click Next.
  4. Choose if you want OneDrive as your default file saving location, click Next.
  5. Click Switch.

Hope this information is helpful. Please do let us know if you need further assistance, we’ll be glad to assist you.

e75_finn_user say:

Hi,

This solution comes maybe a bit late for some people, but at least I found it useful instead of removing future recurring events one by one.

Might be that this helps other Nokia phone users as well?

 

The reason for this clean up might not be that unsual :
I changed the corporation, but was allowed to keep my old phone.
For at least temporary before getting new phone, I would like to connect the phone to new company's Exchange, but found it irritating having old company's calender events still on the phone giving alerts etc.

 

Try this (applies while using Nokia's Ovi account):

 

1) If not yet available, create for example Ovi account (I bet some other free public calendar service might do if having connectivity to mobile)
2) Set Ovi calendar to connect/sync with your Nokia phone (send Ovi.com sync settings to your phone from Ovi calendar app)
3) Accept Ovi calendar sync settings on your phone
4) Select Synchronization -> options and either 'switch current profile' or Create new profile and browse for previously saved Ovi.com -sync settings.
5) Select Synchronization -> options and 'Modify profile'

-> browse downwards until you find 'way of synchronizing' setting
and change the option as 'Towards server'
6) Synchronize by selecting Synchronization -> options and 'Sync'

After this you should have all the phone calendar events on Ovi calendar

7) go to Ovi.com, open calendar and from calendar you can open each recurring meeting by editing series, then delete
(unfortunately I didn't yet find an option to clear the whole Ovi calendar, but this is still way more userfriendly)

I hope you figure out where this is leading?

Once Ovi web calendar is erased

8) Select Synchronization -> options and 'Modify profile'

-> browse downwards until you find 'way of synchronizing' setting
and change the option as 'Towards phone'
6) Synchronize by selecting Synchronization -> options and 'Sync'

After this little exercise the phone calendar should be emptied based on emptied Ovi web calender.

Easy? Well... at least I found this fancier than browsing calendar one by one or resetting the whole phone or something similar.
I don't really know how is this topic covered in other Nokia phone models, but I hope there would be an option to empty the whole calender if user wants to perform such action.
I though do admit that making it difficult (or letting it be as is...) serves the other purpose which is ensuring accidental emptying of your phone/work calendar is rather difficult.
Because, as always -> even if accidental loss of any data is caused by false usage, the device is always claimed!
:-)

Anyway, I hope this helps someone and keep posting replies if you found this any useful.
