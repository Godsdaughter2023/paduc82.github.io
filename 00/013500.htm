peter atherton say:

I do not think a function will achieve what you need. A macro, however, can look up a value in in one page and then add the total to to the same items previous totals. You will need to tweek this macro:

 

'======================================================

Sub UpDateSales()
  Dim rng As Range
  Dim rng2 As Range
  Dim c As Range, sold As Integer, batchQty As Integer
  Dim FindRow As Long
  'Make sure we start on the correct sheet _
   the input sheet
  Sheets("Sheet1").Activate
  Set rng = Range("Items")
  Set rng2 = Range("ItemsSold")
 
  For Each c In rng2
    If Len(c) > 0 Then
      'the offset column might need to be changed
      batchQty = c.Offset(0, 1).Value
      FindRow = WorksheetFunction.Match(c, rng, 0) + 1
        With rng
          'the column offset may need to be changed
          sold = .Cells(FindRow, 2)
          sold = sold + batchQty
          .Cells(FindRow, 2) = sold
        End With
    End If
  Next
 
  Range("InputArea").ClearContents
End Sub

 

'=================================

 

I would make a new workbook to test this. One sheet will contain all the item names and current qtys sold. Name the first column 'Items'. The other sheet is your new sales sheet. I applied Data validate to the names, select the Names input area and then choose:= Data>Validation>allow List>Source =Items 

I gave this list the name 'ItemsSold'

 

My workbook contained just two columns on each sheet, Items and Qty so the offset column is 1 and the sold cell is 2.

 

Finally I cerated another range on the input sheet that combined the ItemsSold and the Qty range and used this to clear the input qty.

 

How to use: Copy the code into a module enter some data and pres ALT + F8 and run the macro. Once you have tweeked it add a form Command button from the Insert Controls toolbox in the developers tab and assign the macro.

 

HTH

Peter

File, Options, Advanced and under Image Size and Quality, check the option ‘Do not compress images in file’ and make sure to select All documents in the box above it.

In this particular document, you may need to crop the ‘line’ around the edges or reinsert the image depending on which is most convenient for you.
