This issue could occur due to corrupted system files or it has something to do with the permission settings. We suggest you to perform a Clean Booton your PC, and once you’re done in performing a Clean Boot, follow the steps below, and check if you will still have the same issue.

  1. Right click on the file/folder/registry key with which you’re having the issue and select Properties.
  2. Go to Security tab and click Advanced.
  3. On Advanced Security Settings window, click Change beside Owner. Take ownership of the item, even if you’re already the owner.
  4. After taking ownership, check Replace owner on subcontainers and objects below owner’s name.
  5. Check as well Replace all child object permissions entries with inheritable permission entries from this object on the bottom part of the window.
  6. Click Apply.
  7. Select Yes on Windows Security pop up.
  8. Another pop up for Windows Security will appear, hit Yes again, and then OK.
  9. Try to check if you will still get the same issue.

mohdrafi say:


Please perform the below troubleshooting steps to check the status of Volume shadow copy service

1. Click Start > All programs > Accessories > Run

Type the following text in Run window and click ok:

 Services.msc

You will get Services window.

2. Please check Volume shadow copy service is "Started" and set to "Automatic".

If the above services are not set to "Started" and "Automatic", please right click on that particular service and select "properties".
On properties window, please change the Start-up type to Automatic and use the Start button to start the service.

Click Apply, click ok.

Close the Services window

Please restart the computer and check if the issue re-occurs.

If problem still occurs , start your computer in safe mode and try to do a system restore again . Disable your anti-virus protection temporarily

http://windows.microsoft.com/en-us/windows-vista/Start-your-computer-in-safe-mode
