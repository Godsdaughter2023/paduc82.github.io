About saving / backing up the OneNote "files" directly from OneDrive: That's not possible. guessed say:

Diane

The colour of the text is dependent on which style is currently being used and what the settings are for that style. There are well over 200 styles in the simplest document so this is not as easy as it might appear. However, the majority of text is created using the so-called 'Normal' style. Additionally, this style is commonly the basis of many other styles so changing its colour has a ripple-on effect and changes a bunch of other styles.

Sometimes the colour of text is defined precisely but more often, the colour is picked from a theme set. When colour is defined using a theme colour, changing the theme will cause any content in the document using theme colours to change.

If you haven't yet learnt the value of styles, do a search on Youtube and watch some videos on how you use and modify styles. Once you understand how that works, changing the default colour of your text becomes much easier.

charles kenyon say:

As Andrew said, the colors are set in styles. The default color for most styles is "Automatic" which means black on a white background and white on a dark background.

The heading styles have default colors as well.

You should not, though be applying heading styles automatically unless you have AutoFormat As You Type set up to apply heading styles to sentence fragments or one-sentence paragraphs.

If you want to continue with the AutoFormat As You Type conversions, then check which style is being applied and change the color of that style in your normal template (or other document templates if you are using others).

As a workaround, Skype for iPhone can be used on an iPad with iOS 9.
