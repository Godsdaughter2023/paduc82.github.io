Windows unloads each user's profile and user's section of the registry when the user logs off. Whether your OS is 32 bit or 64 bit makes no difference to how the drive is formatted and shouldn't effect the old drive fitted as a second drive. 32 or 64bit refers to the OS and not how the drive is formatted. If you boot from the 64bit OS then that should be able to read the second drive.
