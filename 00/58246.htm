a. user say:

Some files and folders are hidden.  To see them, go to Start / Control Panel / Folder Options / View and uncheck the box to hide system files and check the button to show hidden files and folders.  That may account for some of the difference.

Also, Vista doesn't do a very good job of reporting used disk space.  This is a common problem with no real solution within Vista.  Some files and folders just aren't included (for some reason) in the summary or remain hidden despite the above changes.  I believe this applies to system restore and shadow copy information and probably other files/folders as well.  You can change the maximum space used by these programs which may clear up some space (with the trade-off that you have fewer restore points).  Here's how to do that: http://www.vistax64.com/tutorials/76227-system-restore-disk-space.html.

I recommend http://www.hdgraph.com/ to scan your system for how the space is being used.  It's not perfect either, but it is a bit more accurate and provides more data and structures the information in a way that makes it easier to read and understand.  I'm not sure if it captures System Restore and Shadow Copy space either, but it might (and is probably worth a try just to see what it does show).

I hope this helps.

Good luck!
