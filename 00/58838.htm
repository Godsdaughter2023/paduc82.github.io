a. user say:

Hello Sai,

Thank you for posting your query in Microsoft Community.

As per the problem description, I understand that your computer fails to boot with “No bootable device” message.

If you’re getting this error message while trying to boot your computer after upgrade to Windows 8.1, possibly some important system files responsible for successful booting of Windows 8.1 operating system have gone corrupt or got deleted from the computer.

In this scenario, you need to boot the computer from Windows 8 reinstallation disc or Recovery USB drive and then try rebuilding or repairing the Master Boot Record, Boot Sector and the Boot Configuration Data and check if that resolves the issue.

To boot the computer from the Windows 8 Recovery USB Drive, change the boot device priority to USB drive in the BIOS.

To access the BIOS, you need to either contact the Gateway Technical Support or refer to the user manual that came with your computer. Changing any settings in the BIOS is very risky and is always recommended to be done by an expert. So, I would suggest you to get in touch with the Gateway Technical Support for further assistance on this.

Important: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

Once after changing the boot device priority to USB drive, boot the computer from the Windows 8 Recovery USB disk and then follow these steps:

  1. Boot the computer from the Windows 8 Recovery USB disk

  2. In the “Windows Setup” dialog box that appears, set the Time and currency format, Keyboard or input method and the language to install and then click “Next”.

  3. Click on “Repair your computer”.

  4. Click on “Troubleshoot” and then select “Advanced Options”.

  5. Click on “Command Prompt” and type the following commands and press “Enter” after each command:

    BOOTREC /SCANOS
    BOOTREC /FIXMBR
    BOOTREC /FIXBOOT
    BOOTREC /REBUILDBCD

  6. Restart the computer and check if the issue still persists.

You may also, refer to suggestions by the Community Member “phanlon7” on October 12th, 2013 from the thread link given below:

Windows crash - Refresh - Hard drive locked, unlock hard drive

Hope this helps. Please let us know the results. Feel free to write us back for any further assistance, we’ll be glad to assist you.

davem121 say:
Hi Rednibycc, that thread you linked through is not coming through, so I have no idea what you have already tried to fix this issue . . .

If you have already gone through these steps, please ignore them . . .

Before you do anything or move any files around
Restart your PC 3 or 4 times, this will fix this issue a lot of the time . ..

Otherwise:

Please open Windows File Explorer
Navigate to C:\Users
Do you see two user folders that look like yours?
Open those folders and check inside Pictures, Documents . . . etc. and see if your files are there

If your files are there:

Create a new user profile with Admin privileges
Log into the new user profile
Copy all your data from the old profile to the new profile (Documents Library, Pictures Library . . . etc.)

If you still have not found your files, look for a Windows.old folder on your C Drive, expand that to open the users folder contained inside - check there for your files, icons . . .etc.

If you do find your files . . . etc. there, copy them out as soon as possible as Windows will remove the Windows.old folder in time

NOTE VERY IMPORTANT: Do Not copy your files into the Temporary profile you are currently in, when the situation is corrected, your files will be deleted with the Temporary Profile and will not be recoverable !!
