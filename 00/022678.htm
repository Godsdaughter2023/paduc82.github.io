bernard liengme say:

I have written a VBA subroutine to do this and I have tested it on 500 numbers - the code is shown below

You can get my file from my OneDrive: https://1drv.ms/x/s!AkhpKJf_GSEWgbhP0tl533y9Na-heA

From the screenshot below (do an internet search with Excel 2010 specifications) you will note that you will be able to see on 256 characters in the cell and the maximum number of characters is 32,767. Hope your numbers are not too big. If you bang into the limitation you will need to write a subroutine to directly send data to a TXT file rather than to a cell. Do an internet search with Excel data to text file vba.

best wishes

Sub Concat()
  LastRow = Cells(Rows.Count, "A").End(xlUp).Row
  mystuff = Range("A1")
  For j = 2 To LastRow
    mystuff = mystuff & ";" & Cells(j, "A")
  Next j
  Range("C1") = mystuff
End Sub

e.j. gun say:

Formula For First Visible Item in a Filtered List

Let's say I have a filtered list with several columns.  The range of the list, including headers, is $A$5:$K$200.  Column K is labelled "COMB".  Now say I filter that column for one or more items.  What I want to be able to do is capture the first visible item in that column in Cell K2.  Is that possible without using VBA?  I could easily write a UDF, but I'm looking for a plain old Excel worksheet formula approach.

 

Thanks,

 

Eric

mike h.. say:

Hi.

 

You can use a helper column which you can hide and here I've used column L next to the table. Put this formula in L2 and drag down to the length of your unfiltered data.

 

=SUBTOTAL(103,K$2:$K2)

 

Now a simple INDEX match to return the first visible item in Col K

 

=INDEX($K$2:$K$200,MATCH(1,$L$2:$L$200,FALSE),1)
