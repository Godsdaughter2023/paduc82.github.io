Yes, we can't use Office Online without OneDrive for Business.

Currently there is no Office 365 subscription which includes Office Online but has no OneDrive for Business. And we assign only Office Online license to a user, SharePoint Online license is needed.
Actually, when we create a document in Office Online, it has to be saved and kept in somewhere and it's OneDrive for Business library by default.


If you don't want to enable OneDrive for Business for users, we can consider using these methods:
1. For newly created users who have not signed in to Office 365, global admin can go to SharePoint admin center > user profile > Manage User Permissions > if you see Everyone except external users listed in the box, please remove it. After doing the above, the new user's personal site will not be created and he will not be able to access OneDrive for Business library. Admin can also go to Admin center (new UI) > Users > Active users > select the user and click OneDrive Settings, he will see a message saying "OneDrive is not provisioned for this user".

2. For users who have already used OneDrive for Business, the above steps will not work. We can disable the sync button for the user's OneDrive for Business library:

Go to Admin center > Users > Active users > click OneDrive Settings > Access files > click the link to go to the user's OneDrive for Business library > click Return to classic OneDrive in the lower left corner > Settings icon > turn Ribbon on > Library tab > Library Settings > Advanced settings > select No in Offline Client Availability.

In your Excel pivot table on row 4 (or thereabouts) you should see, Year, Quarter, Week and Resources.
