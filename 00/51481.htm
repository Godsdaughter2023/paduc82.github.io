Windows Vista uses the Local and LocalLow folders for application data that does not roam with the user. Usually this data is either machine specific or too large to roam.

Appdata\Roaming Folder

Windows uses the Roaming folder for application specific data, such as custom dictionaries, which are machine independent and should roam with the user profile
