techfreak_ say:

How to Export Skype 7.x history:

Using Skype 7.x = Tools - Options - IM & SMS - IM settings - Show advanced options - Export Chat History
This exports data from your local main.db into a CSV file you can view with a text editor or a spreadsheet program like Excel. This output is text-based.

Using Skype 8.x = Press the Alt key to bring up the hidden menu - Tools - Export Chat History from Skype 7.x...
This exports data from your local main.db into a folder with an HTML file you can view with your web browser. This output contains text and some graphics.

Be aware that only data found in the main.db file will be exported. This will be any history up to the last time you signed in with classic Skype 7.x on that machine (assuming no file corruption or deletion), plus possibly any recent history in the cloud (30 days). If you are missing any older history you can try to slowly scroll through your history in Skype 8.x and it will attempt to download history older then 30 days up to 2 years. Most users can view a little over a  year. Since no export utility currently exists for Skype 8.x, any missing data found, would have to be manually copy and pasted into a document.
