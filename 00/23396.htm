japanzaman say:

 

Did you want the name of the presentation or the name of the file itself? Are we adding this title to a placeholder (Title box) or generating another text box?

Here is a macro you can use to add the name of the presentation (file name) to the first slide into the title box.  This will trim off the file extension as well

 

 

Sub addfilename()

Dim shp As Shape

 

projectname = ActivePresentation.Name  'get your project name

 

For Each shp In ActivePresentation.Slides(1).Shapes  'go through all shapes on slide1

 

   If shp.Type = msoPlaceholder Then  'if the shape is a placeholder then add the projectname

      shp.TextFrame.TextRange.Text = projectname

      For I = shp.TextFrame.TextRange.Characters.Count To 1 Step -1 'delete this loop if you want to keep the file extension

         If shp.TextFrame.TextRange.Characters(I) = "." 

            shp.TextFrame.TextRange.Characters(I).Delete

            Exit For

         End If

         shp.TextFrame.TextRange.Characters(I).Delete

      Next

      Exit Sub

   End If

 

Next shp

 

 

End Sub
