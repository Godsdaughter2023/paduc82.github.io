>>and don't understand what you mean by "For IMAP email accounts - for email data there is nothing to copy. "

IMAP email data is stored on the server and sync'd with an .OST data file (offline storage) located on your hard drive. In short, the ost file is a "mirror" of what's stored on the server. Hence, there's nothing to copy when you configure the email account on another device whether computer, tablet or mobile.

What that means is that <IF> you will need to copy the email data you've saved in the PST file associated with the POP account to the corresponding folder in the IMAP folder tree on the new computer.

This option requires a UDF (User Function).

Your example of the initial string shows semi colons between the separate strings and these are a requirement of the UDF code.

The results need to be on the rows immediately following the initial string.

An error is returned when there are no more occurrences of the string. (ie. if there are 3 occurrences of "apple" found then 3 rows will have results and the 4th row will return an error).

To install the UDF.

  1. Alt and F11 to open the VBA editor.
  2. In the VBA editor select menu item Insert -> Module.
  3. Copy the code below and paste into the VBA editor.
  4. Close the VBA editor (Cross with red background top right of window)
  5. Save the workbook as Macro enabled.
  6. Ensure that options are set to allow Macros. See help for how to do this. (Set the macro option to "Disable all with notification".)

In the first cell under the initial string insert the following formula. (Example assumes the initial string is in cell A1 otherwise edit $A$1 to the cell where the initial string is located; noting the absolute addressing with the $ signs. )

=findinstr($A$1,"apple")

'*******************************************************************************

Function FindInStr(rngString As Range, strFind As String)
    Dim rngCaller As Range
    Dim arrSplit As Variant
    Dim arrFind()
    Dim lngOccur As Long    'The occurrence of the string to be found
    Dim i As Long
    Dim k As Long
   
    lngOccur = Range(Application.Caller.Address).Row - rngString.Row
   
    arrSplit = Split(rngString.Value, ";")
   
    k = 1
   
    ReDim arrFind(1 To k)
   
    For i = LBound(arrSplit) To UBound(arrSplit)
        If InStr(1, UCase(arrSplit(i)), UCase(strFind)) Then
            ReDim Preserve arrFind(1 To k)
            arrFind(k) = arrSplit(i)
            k = k + 1
        End If
    Next i
       
    FindInStr = arrFind(lngOccur)

End Function

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
    Dim MyTime
    MyTime = #10:00:00 AM#
    If Intersect(Target, ActiveSheet.Range("B4")) Is Nothing Then Exit Sub
