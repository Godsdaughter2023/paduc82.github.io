andre da costa say:

The Windows.old folder stores a copy of your old Windows installation (in this case Windows 7). It can be used to recover data that you might not have backed up in addition to that it can be used to reinstall Windows 7.

Delete if you are sure all your data and applications have migrated safely.

Windows~BT is used for diagnostics purposes, I suggest you do not delete it.

See instructions about how to delete the Windows.old folder:

http://windows.microsoft.com/en-US/windows7/How-do-I-remove-the-Windows-old-folder
