how to delete a long running script in SIMPLE terms - Microsoft Community

darthawk say:

how to delete a long running script in SIMPLE terms

My page freezes a pop up says to stop script. How do I find it and stop it? I am NOT computer savvy. Thanks

don varnau say:

If this is happening only on one (or a few) website(s) there really is (by Internet Explorer standards) a long running script on the page and IE is doing you a favor by providing the "Stop script" option. Click on the Stop script button.

Again, if only a site or two are affected...

It might break a desirable feature on the site, but placing a site in Restricted Sites will disable JavaScript and the script messages will stop.

Internet Options. Security> Restricted Sites...

If you're seeing the message on all or nearly all sites, a browser add-on might be causing it.

Note: The Google Toolbar has caused script problems for some users.

Start Internet Explorer without add-ons from Start> Programs> Accessories> System tools> Internet Explorer (no add-ons)
or

from Start> Run (or Windows logo key + r) type in iexplore.exe -extoff

(iexplore.exe<space>-extoff) then hit Enter.

If the problem goes away, an add-on is causing it.

To determine which add-on is causing the problem see this old (but still relevant) article:

IEBlog  Troubleshooting and Internet Explorer’s (No Add-ons) Mode:
http://blogs.msdn.com/ie/archive/2006/07/25/678113.aspx

Another option: Other browsers like Edge, Chrome and Firefox are more forgiving of these script problems. Consider using another browser on affected sites.

scottgem say:

There are two ways to filter the records viewed in a form. One way is to use a query as the Recordsource of the form and set criteria within the query, the other is to use the Filter property of the form. There are pros and cons to either method and one method may be better than another depending on circumstances. 

In the situation you describe I would be more inclined to use a Filter. So the Code behind your Toggle button would look like:

If Me.TogFilter Then

     Me.Filter = "datefield BETWEEN DateAdd("d",-1,Date()) AND Date()"

     Me.TogFilter.Caption = " 1 Day old"

Else

     Me.Filter = "datefield BETWEEN DateAdd("d",-10,Date()) AND Date()"

    Me.TogFilter.Caption = " 10 Days old"

End If

DoCmd.RunCommand acCmdApplyFilterSort

Setting the Caption property will show you what the current filter is. 

But I would do this a different way I would add an unbound TextBox, labeled Days Old with a default value of -1. Then when you want to filter enter a new value in the text box. The code in the After Update event of the text box would be:

Me.Filter = "datefield BETWEEN DateAdd("d",Me.DaysOld,Date()) AND Date()"

DoCmd.RunCommand acCmdApplyFilterSort

This gives you maximum flexibility to set the date range to any value you want. And the number of days will be a visual indicator of the filter.

How do I set all references within a column of cells to absolute references?

I have multiple columns within a spreadsheet that contain relative references to other cells within the same worksheet.

I need to change each relative reference to an absolute reference.

I know that I can select each cell individually and then press F4 to create an absolute reference, but is there a way I can select the cells in a column and then use a command to make all of them into absolute references?

Thanks

rick rothstein, mvp - excel say:

How about a macro solution...

Sub ConvertAllReferencesToAbsolute()
  Dim S As Range, C As Range, A As Variant, Addresses() As String
  For Each S In Selection
    If S.HasFormula Then
      S.Formula = Replace(S.Formula, "$", "")
      Addresses = Split(S.Precedents.Address, ",")
      For Each A In Addresses
        For Each C In Range(A)
          S.Formula = Replace(S.Formula, C.Address(False, _
          False), C.Address(True, True))
        Next
      Next
    End If
  Next
End Sub

Note 1
-------------------
If any of your formulas have text constants in them that contain $ signs, those $ signs will be removed; so, you cannot use the above on cells with text constants containing $ signs. There is a way around this problem, but it makes the macro take longer to execute do to the overhead associated with the needed additional code.

Note 2
-------------------
If you are new to macros, they are easy to install and use. To install it, simply press ALT+F11 to go into the VB editor and, once there, click Insert/Module on its menu bar, then copy/paste the above code into the code window that just opened up. That's it.... you are done. To use the macro, go back to the worksheet with your data on it and press ALT+F8, select the macro name (ConvertAllReferencesToAbsolute) from the list that appears and click the Run button. The macro will execute and perform the action(s) you asked for.
