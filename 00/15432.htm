Sub Test()
  Dim Source As Range, Dest As Range

  With Worksheets("AMP_My_Business")

macropod say:

All of the paragraph attributes applied to your Recipe Name Style can be applied to the Heading 1 Style. The following macro should expedite the process:

Sub ReformatHeading1Style()
Dim Stl As Style
With ActiveDocument
  Stl = .Styles("Recipe Name")
  With .Styles(wdStyleHeading1)
    .Borders = Stl.Borders
    .ParagraphFormat = Stl.ParagraphFormat
    .Font = Stl.Font
    .Shading = Stl.Shading
    .NextParagraphStyle = Stl.NextParagraphStyle
  End With
End With
End Sub

For PC macro installation & usage instructions, see: http://www.gmayor.com/installing_macro.htm
For Mac macro installation & usage instructions, see: http://word.mvps.org/Mac/InstallMacro.html

suzanne s. barnhill say:

If what you're looking to do is create a TOC, you don't have to change the style; you can just create the TOC to include the Recipe Name style; see http://wordfaqs.mvps.org/TOCTips.htm

But it is equally easy to modify the Heading 1 style to match your current Recipe Name style. Open the Styles pane and make sure that Heading 1 is visible. Select a paragraph in the Recipe Name style. Right-click Heading 1 in the Styles pane and choose Update to Match Selection. You can then use Replace to replace the Recipe Name style with the Heading 1 style.

jay freedman say:

Remove all of the text your typed into the Find What box and the Replace With box, leaving both boxes empty. All you need is the settings in the Format: lines below the boxes.

The reason the Replace doesn't work with this is that you've told it to look for the literal text "A Recipe Name" in the document, but that text doesn't occur anywhere in the document. By leaving the boxes empty, you'll be saying to find any text with the specified formatting, and change only the formatting.
