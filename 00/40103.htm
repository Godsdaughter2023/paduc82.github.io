Q. If my customer asks me to upgrade a PC with new hardware components, when is a new operating system needed? When would the PC be considered "new"?

A. Generally, an end user can upgrade or replace all of the hardware components on a computer—except the motherboard—and still retain the licence for the original Microsoft OEM operating system software. rey g. santos say:

To insert an ASCII character, press and hold down ALT while typing the character code. For example, to insert the degree (º) symbol, press and hold down ALT while typing 0176 on the numeric keypad.

Note:   You must use the numeric keypad to type the numbers and not the keyboard. Make sure that the NUM LOCK key is on if your keyboard requires it to type numbers on the numeric keypad. If you are using a laptop you must press the "fn" + "numlk" (numlk light will lightup), then you will use the keys that doubles as numbers (numeric keys) M(0), J(1), K(2), L(3), U(4), I(5), O(6),  P(.),  7(7), 8(8) and 9(9). Don't forget to deactivate the "numlk" (press fn + numlk) or those keys will always type numbers.

See this for other symbols and their keys:
Insert ASCII or Unicode Latin-based symbols and characters
http://office.microsoft.com/en-us/access-help/insert-ascii-or-unicode-latin-based-symbols-and-characters-HA010167539.aspx
