This is not a regular Word function. The closest is Autocomplete for AutoText entries. Then it would be happening if you had AutoText entries with names that are the same as the beginnings of the words you are typing.

Automated Boilerplate Using Microsoft Word

Take a look at which word(s) are giving you this problem. Try starting Word in safe mode and seeing if the same thing is happening. If it is AutoText causing the problem, you'll need to rename your AutoText entries. You want your entries to have names that are not the same as ordinary text. I often start the names of entries for my pleadings with the letters pl. The default in Word is to give the entry a name which is the same as the beginning of the text. This makes them easier to remember but can lead to the problem you are seeing.

Greg Maxey's Building Blocks Tools might help with renaming your entries. Building Blocks & AutoText You would want to use the building blocks loader feature. That gives you a Word document where you can edit the names. You would have to insert your entries into column 3 and rename them. Another set of utilities that might help are from Jay Freedman. They can be found on his Macro Zone. Although they are designed to work in non-ribbon versions of Word, they will give you results for entries stored in .dotm files. Where can Building Blocks be stored?

john_tesla say:

Dear mianahmad,

 

A warm welcome to the Nokia Support Discussions.

 

We are sorry you are having issues with the latest Belle update, Unfortunately it is not possible to roll-back the software once Belle is installed.

 

However, reinstalling Belle properly will resolve any errors in the software.

The current software might have been installed incorrectly which can cause these kind of errors. 

 

First you'll need to perform a hard reset in order to delete by bugs in the software caused by faulty installation of Belle.

 

Back up your data with Nokia Suite, but don't tick the Settings box as this will backup any bugs in the software.

 

Now enter *#7370# into your phone, the default security code is 12345.

 

Now Connect your device to Nokia Suite via USB and click on the green arrow to reinstall Belle. 

 

After you've done this, place the backup back onto your phone and this should fix all errors in the device.

 

Let me know how you get on so that we can troubleshoot further, if necessary.
