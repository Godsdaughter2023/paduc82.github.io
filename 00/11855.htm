roberto oca say:

Hi Jomac,

Message body disappearing in Outlook is a known issue of two commonly used antivirus software: AVG and NOD32. It happens like this:

• A message arrives in the Exchange mailbox and synchronizes with Outlook.
• The AVG or NOD32 add-in, which is added to Outlook automatically if you have the software installed, deletes the message body.
• The message is then synchronized with the server, so it is also synchronized with OWA and with wireless devices.

To prevent this from happening, you need to remove the AVG or NOD32 add-in in Outlook.

Note: This solution will not restore message texts that have already been deleted. The add-ins mentioned here both delete message bodies permanently.
