macropod say:

This really isn't making much sense. If the documents are just "derived from scans OR the web", where do the REF fields come from? A document created that way won't have them. That said, try the following macro:

Sub PromptReplace()
Dim VResp, Rng As Range
Set Rng = Selection.Range
With ActiveDocument.Range
  With .Find
    .ClearFormatting
    .Text = "\([!\(\)]{4,}\)"
    .Replacement.Text = ""
    .MatchWildcards = True
    .Wrap = wdFindStop
    .Execute
  End With
  While .Find.Found
    .Start = .Start + 1
    .End = .End - 1
    .Select
    VResp = MsgBox("Make this one Bold?", vbYesNoCancel, "Change Text")
    If VResp = vbCancel Then GoTo Finished
    If VResp = vbYes Then
      .Font.Bold = True
    End If
    .Collapse wdCollapseEnd
    .Find.Execute
  Wend
Finished:
End With
Rng.Select
Set Rng = Nothing
End Sub

The '4' in .Text = "\([!\(\)]{4,}\)" specifies that the string must be at least 4 characters long. You can change it to something else if you prefer.
