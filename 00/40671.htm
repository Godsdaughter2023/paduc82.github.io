cbarnhorst say:

I don't know why you are bothering with Windows 7 in the first place.  All you ever had to do was create a Windows 10 installation disk and clean install Windows 10.  There is absolutely no need for Windows 7 at all.  

The first time Windows 10 was installed on the device the licensing software registered it as a Windows 10 device with Microsoft's servers.  The upgrade-from-an-activated-copy-of-Windows-7-or-8-nonsense was just to get the initial registration done.  The user doesn't keep repeating that process every time he needs to reinstall Windows 10.

Once Windows 10 has been installed and activated on a computer it can be re-installed at any time without any product key.  A product key page appears in case you are installing a different edition, but you simply click the link at the bottom of that page that says "I don't have a product key."  That is because you are doing a re-installation.

There is nothing recorded on a registered Windows 10 device about activation or licenses.  Activation of Windows 10 on a registered Windows 10 device is handled by the licensing software submitting the device's unique hardware identifiers to the Microsoft servers.  Microsoft looks up the identifiers and confirms that the device is already registered.  The licensing software then activates Windows.  As long as the motherboard hasn't changed it always works.  

Forget what you knew about Windows 7 activation.  That was done in software and because software can be hacked Microsoft got rid of it.  Everything is different now.
