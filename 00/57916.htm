Hi,

Recovery partitions to reinstall the Operating System back to new condition are normally between 5-19GBs.

The System Reserved partition you do not touch;

http://www.howtogeek.com/192772/what-is-the-system-reserved-partition-and-can-you-delete-it/

This partition contains two important things:

  • The Boot Manager and Boot Configuration Data : When your computer boots, the Windows Boot Manager starts up and reads the boot data from the Boot Configuration Data (BCD) Store. Your computer boots the boot loader off the System Reserved partition, and it boots Windows from your system drive.
  • The startup files used for BitLocker Drive Encryption: If you ever decide to decrypt the main encrypted drive and boot the encrypted Windows system.
