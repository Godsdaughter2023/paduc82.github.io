Index entries - Microsoft Community

amritpradhan say:

Index entries

How do I delete all the Index entries in my Word 2016 file at one-go?
suzanne s. barnhill say:

If you are really talking about an index and not a table of contents (users sometimes confuse them), then this is the method:

  1. Since index entries are formatted as Hidden, press Ctrl+* (or click Show/Hide ¶ in the Paragraph group on the Home tab) to display nonprinting characters, including Hidden text.
  2. Press Ctrl+H to open the Replace dialog.
  3. In the Find what box, type: ^19 XE
  4. Leaving the Replace with box empty, click Replace All.

^19 is the code for an opening field brace, and XE is the field code for an index entry field. When Word finds this combination, the entire field is selected and will be deleted by being replaced with nothing.
