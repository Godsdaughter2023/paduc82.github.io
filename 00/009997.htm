gord dibben say:

Have you tried the Text to Columns Wizard to split the column into several or many columns then delete the columns you don't want?

Or would you prefer to cut all but the first 20 characters?

You could use VBA for that.

Sub truncate_text()
    Dim Cell As Range
    Dim myrng As Range
    'select top cell in desired column
    Set myrng = Range(ActiveCell, Cells(Rows.Count, _
    ActiveCell.Column).End(xlUp))
    For Each Cell In myrng
        If Len(Cell) > 20 Then
            Cell.Value = Left(Cell.Value, 20)
        End If
    Next
End Sub
