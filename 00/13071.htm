No, that policy only applies when the user has Archive mailbox provisioned. If not, the Retention tag will act instead, and as the default Retention tag is set to 'never delete', nothing will happen.

For the permissions, Full access will always have precedence. It doesn't depend on any folder level permissions, it applies on the mailbox level.
