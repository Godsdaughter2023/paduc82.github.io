How do I find the 'All Commands' group in Word 2010 in order - Microsoft Community

still learnin' after all these years say:

How do I find the 'All Commands' group in Word 2010 in order to make the print preview margins 'draggable'?

I just started using Word 2010 after having the 2002 version for 10 years, so getting used to the ribbons and where things are located is a little challenging.  I read an old post by someone who had the same problem as I currently do:  is there a way to get the margins shown in print preview so that you can drag them to where you want them to be.  One of the replies mentioned there is a 'show margins' icon in the 'All Commands' group and that you could go to it and activate this function.  However, I can't figure out how to find the 'All Commands' group and I'm tired of looking.  Any suggestions, folks?
jay freedman say:
You're going to add a button to the Quick Access Toolbar or a custom group on the ribbon, or both. Clicking the button will open a print preview that isn't otherwise accessible. It's called "Print Preview Edit Mode", not "Show Margins".

Click File > Options. To add a button to the Quick Access Toolbar, click that tab in the options dialog. Open the "Choose commands from" dropdown and choose "All Commands" (that answers the question in your subject line). In the list of commands, scroll down to "Print Preview Edit Mode" and click on it. Click the Add button in the center of the dialog, which places the command in the list on the right. You can use the up and down arrows on the far right to change the order of buttons on the toolbar.

To add a button to the ribbon, go to the Options dialog and click the Customize Ribbon tab. You can't change anything in the built-in groups, so select a tab in the right-hand list to add a custom group to, and click the New Group button at the bottom. Click the Rename button to change the name of the group if you want. With that custom group selected in the right-hand list, go to the left and set the dropdown to "All Commands", scroll down to "Print Preview Edit Mode" and click on it, and click the Add button in the center of the dialog.

When you click the "Print Preview Edit Mode" button in either place, it opens with the Magnifier active. To be able to drag the paragraph indents (not the margins), you must uncheck the Magnifier box in the ribbon. If you want to affect the margins instead of the indents, use the Margins button, which is the same one as on the ordinary Page Layout tab of the ribbon.

john mcghie say:
Hi Mary:

If you can, create a new document, save it with a new name, then copy the text from the old document to the new one.

If you cannot save the new document, create a file in TextEdit and copy to that instead so you save your changes.

You should always save a new document before you put any text in it, so you know in advance if it is going to give problems.

Now, which disk are you using?  What kind is it?

That error message is very specific: it may be that what it says is true!

It may also be that you have named your computer something like "Mary's Mac".  If you have, that apostrophe gets added to the front of every file name, and prevents Word from saving to anywhere on the disk.  You can simply change the name of the Hard Disk to remove the apostrophe.  The Apple Help will show you how.

It may be that you have never updated Microsoft Office since you installed it.  Re-start the computer then go to Help>Check for Updates in Word.  There were several updates that cured issues of this nature.

If all else fails, there may indeed be something wrong with that disk, in which case you need to go back to the Apple Store and tell 'em to fix it properly this time.

Hope this helps
