Port Forwarding - Microsoft Community

xb1r athena say:

Port Forwarding

Login Info

 

  1. Gateway Address: "192.168.2.1" or try "172.16.0.1" or try "http://gateway.2wire.net/management"***
  2. Username: Try "admin" otherwise there is no default username.
  3. Password: Try "admin" or try “2wire” otherwise there is no default password.

 

***Note: The Management and Diagnostic Console (MDC) settings are dependent on the 2Wire gateway software release. The MDC are for 2Wire gateways running software release 4.21.x If your gateway is running a software release earlier than 4.21.x, some of these settings may not be available.

 

Port Forwarding

 

  1. Open a web browser.
  2. Type the gateway address: "192.168.2.1" into the web address.
  3. When prompted for a password, type in "admin" for both the username and password. If not, check the label on the gateway or ISP documentation for a username and password if you are prompted for a login.
  4. Once logged into the gateway, click on "Firewall" on the top.
  5. Click on the "Firewall settings" subcategory.
  6. On the main page, for “Step 1” select the device you would like to port forward.
  7. For “Step 2”, select the applications you would like to add in "Allow individual “Applications" list and click "add".
  8. Click the "Done" button on the bottom of the page.

  

 

Disable Port Forwarding

 

  1. Open a web browser.
  2. Type the gateway address: "192.168.2.1" into the web address.
  3. When prompted for a password, type in "admin" for both the username and password. If not, check the label on the gateway or ISP documentation for a username and password if you are prompted for a login.
  4. Once logged into the gateway, click on "Firewall" on the top.
  5. Click on the "Firewall settings" subcategory.
  6. On the main page, for step two select "Maximum Protection".
  7. Click on "Done" at the bottom of the page.
