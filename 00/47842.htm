How do I get my password to WiFi

 


If you have another wireless Windows 7 computer on your network you can open the Network and Sharing Center, click on Manage wireless networks at the left, right click on your home wireless network, select Properties, go to the Security tab and check the Show characters checkbox and display the wireless encryption key/passphrase/password.

Secondly if you have another Windows 7/Vista/XP wireless computer you can also use Windows Connect Now [WCN] to save off your wireless network profile to a flash drive. Simply plug in the flash drive in the new Windows 7 computer to load the profile.

http://theillustratednetwork.mvps.org/LAN/SoHoWirelessSecurity.html#Windows_Connect_Now

You also might try logging into the wireless router from another computer, wired or wireless, and see if the wireless key is displayed in plain text. If so you can copy-n-paste the key to a file then transfer that file to the Win 7 computer and copy-n-paste the key to the profile. Note that not all routers display the wireless key in plain text.

Otherwise you may need to simply redo the wireless key/passphrase/password in the router and all of your wireless clients.
