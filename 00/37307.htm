The RS4 build is the current Public Release of Windows 10.  It is Version 1803.

The RS5 builds are preview builds that are being tested by a large group of users called Insiders.  They find bugs and errors and report them to the Microsoft engineers through Feedback.  In the fall these builds with the corrections will be released to the public as a new version.

This is an ongoing process that is scheduled to be continuous.

It is possible to run both at the same time if you are an Insiders but the RS5 builds are limited to the Insiders.  The general public does not have access to these builds.

Please note that the RS5 builds can crash a computer if there is a bug or error.  These are usually corrected and a new build is issued to the Insiders.

andre da costa say:

There is no need to perform new install in order to create partitions.

Press Windows key + X then click Disk Management

Select the system partition where Windows 10 is currently installed. Right click it then click Shrink.

Enter the amount of space you would like to allocate then click Shrink. The more, the better.

An unallocated partition will now be available. Right click it then click New Simple Volume

This will start the New Simple Volume wizard, which will let you format, assign a drive letter and label the volume.

If you want to create more partitions, right click the new partition you created, click Shrink then follow the instructions as described. So, make sure the first partition you create is large enough to create additional partitions from.

Source: https://answers.microsoft.com/en-us/windows/wiki/windows_10-windows_install/how-to-dual-boot-windows-7-or-windows-8-with/9ac7acc1-2152-4907-a9de-bd507273a57d
