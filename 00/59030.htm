xbr poseidon say:

How to Find SSID (Network Name)

Westell 7550

Login Information

Gateway - 192.168.1.254

Username - N/A

Password - Device Access Code printed on the device

 

ATT/WESTELL Firmware (Blue/White)

 

How to Find SSID (Network Name)

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.254
  4. Select Wireless from the menu on the left.
  5. When prompted, enter the Device Access Code printed on the device.
  6. The Network Name (SSID) can be found next to Network Name (SSID). If a new name is desired type it here.
  7. Select Save.

 

Verizon Firmware (Red/Black)

 

How to Find SSID (Network Name)

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless Settings from the tabs at the top.
  6. From the menu on the left choose Wireless Status.

Andrew E's suggestions are valid, but if you have a semi-recent OEM computer, you may not have an install CD to boot from (most OEM computers now come with a recovery partition).  If this is your case, download and create a free "Hiren's Boot CD".

    Burning Hirens CD < http://www.hirensbootcd.org/burning/ >
You can then boot from this CD, select "Mini Windows XP", and it will start up with a mini version of XP that you can use to directly run a "chkdsk" against you hard drive.

You're asking a Windows 7 question in an XP forum, but since the answer is the same regardless of Windows version, I'll answer it.

You should have a recovery partition on your hard drive and the instructions that came with your computer will tell you how to use it.
