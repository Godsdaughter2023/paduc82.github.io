The "Total" is a calculated field in the "Supervisor" Footer "=Sum([# Emps])

 

How do I sort on this field first, (Descending), then by Supervisor (ascending).

marshall barton say:

No, you can not sort the report by something calculated in the body of a report.  Instead, you need to calculate the totals in the report's record source query.  Maybe something vaguely like:


SELECT table.*, S.Total

FROM table INNER JOIN (SELECT X.Supervisor, Sum(X.[# Emps]) As Total FROM table As X) As S

     ON table.Supervisor = S.Supervisor


This way, the Total is available in the report as a record source field and you can easily sort on it.


Note:  You can not use the query designer with a subquery like that so stay in SQL view.  If you can not deal with working in SQL view, then create a separate query supervisor and Total fields to act as a virtual table and use that instead of the sub query.
