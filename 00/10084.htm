I understand Macros is an option, but I don't have a lot of experience on th

jasles say:

You can use conditional formatting for only the cell color change - it won't work for changing partial text color. However, you can use VBA to do both.

To use VBA to do it, open the VBA editor by pressing Alt+F11, and then double-click the worksheet in the explorer on the left. Then paste this code in the editor to the right:

Private Sub Worksheet_Change(ByVal Target As Range)

Application.EnableEvents = False
On Error GoTo ErrHandler

Select Case Right(Target.Cells.Value, 1)

Case "b"
    Target.Cells.Interior.Color = vbBlue
    Target.Cells.Characters(Start:=Len(Target.Cells.Value), Length:=1).Font.Color = vbBlue
Case "r"
    Target.Cells.Interior.Color = vbRed
    Target.Cells.Characters(Start:=Len(Target.Cells.Value), Length:=1).Font.Color = vbRed
Case "g"
    Target.Cells.Interior.Color = vbGreen
    Target.Cells.Characters(Start:=Len(Target.Cells.Value), Length:=1).Font.Color = vbGreen
Case "y"
    Target.Cells.Interior.Color = vbYellow
    Target.Cells.Characters(Start:=Len(Target.Cells.Value), Length:=1).Font.Color = vbYellow
End Select

Application.EnableEvents = True

Exit Sub

ErrHandler:

Application.EnableEvents = True

End Sub

This will accomplish it for every cell in that worksheet that you enter the single character color code in lowercase at the end of the entry.

teylyn say:
Hello,

A line chart treats the x axis as a category (= text) axis, unless it determines that the values are dates.

As said above, use an XY scatter chart. It can look exactly like a line chart, but you have much greater control over the x axis, which by default will be a numerical axis.
