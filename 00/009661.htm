but when I want to do the same for a range on my if statement with countif, it returns '#values'

 

This doesn't work...

=IF((IF(IF(D2:D212=1, 1, 0)&IF(F2:F212=0, 0, 0)&IF(H2:H212=0, 0, 0)="100","Y","N"))="Y",COUNTIF(K2:K212,"personal check"),"0")

jeeped say:
Actually, your formula will work if you use it as an array formula. Select the cell with the formula and press F2. Now complete the formula by pressing Ctrl+Shift+Enter together. If this is performed correctrly, Excel will wrap the formula in braces like this:

{ =IF((IF(IF(D2:D212=1, 1, 0)&IF(F2:F212=0, 0, 0)&IF(H2:H212=0, 0, 0)="100","Y","N"))="Y",COUNTIF(K2:K212,"personal check"),"0") }

With that said, the formula shows some poor design in sections. The statements IF(F2:F212=0, 0, 0) and  IF(H2:H212=0, 0, 0) are useless since no matter what is in F2 and H2, a zero will be returned. The outer IF() function also seems to be misplaced since you don't have to discern a Y or N. Perhaps something like this would be better:

=COUNTIFS(D2:D212,1,F2:F212,0,H2:H212,0,K2:K212,"personal check")

Note that COUNTIFS() is a function introduced in Excel 2007. If you need the formula to work in earlier versions of Excel, post back for a solution.

Sub AUM_Report()
'
' AUM_Report Macro
'

'
    Range("A1").CurrentRegion.Select
    ActiveSheet.ListObjects.Add(xlSrcRange, Range("$A$1:$D$1635"), , xlYes).Name = _
        "Table1"
