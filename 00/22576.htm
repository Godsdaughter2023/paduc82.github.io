Using WORKDAY Function in Multiple Column Cells - Microsoft Community

lisalhl say:

Using WORKDAY Function in Multiple Column Cells

I have many rows needing the WORKDAY function to determine due date as a business day. I have the holiday data in one set of fields. How do I get the holiday fields referenced to remain the same in every field as I copy the formula down throughout the column? 

hansv mvp say:

Use absolute referencing. For example, if the holidays are specified on Sheet2 in A2:A50, use a formula such as

=WORKDAY(D1, 180, Sheet2!$A$2:$A$50)

where D1 contains a starting date and 180 is the duration in days. This formula can be filled down. The $ iin $A$2:$A$50 ensure that this reference won't change.

Even easier: select the range with holiday dates (A2:A50 on Sheet2 in our example). Then click in the address box on the left hand side of the formula bar, type the name Holidays and press Enter. You can now refer to the range by its name:

=WORKDAY(D1, 180, Holidays)

How to Draw (Free form) a Text Box in Word? - Microsoft Community

tylerhorton say:

How to Draw (Free form) a Text Box in Word?

I am attempting to create a text box in the shape of Africa so my text (a poem) will adhere to the shape of Africa. I go to edit points and I have made the points in a shape of Africa, but my text is still just a square box.
suzanne s. barnhill say:

Text in a shape (including a text box) is always in a rectangle. There are two ways to achieve the effect you want (more or less):

  1. Insert the desired shape (of Africa) and place a text box (No Line, No Fill) over it. The text box should extend beyond the sides of the shape. Then type in the text box, using line or paragraph breaks, indents, centering, etc., to place the text within the shape.
  2. Create the text-filled shape in an illustration program, which is set up to do this sort of thing.

There is actually a third way, but it is even more involved: Create two shapes with sides that make up two halves of the shape of Africa. Use these shapes (with Tight wrapping) to enclose normal paragraph text between them.
