Running something in a sandbox is terminology describing how to isolate a suspicious or unknown application and limiting what it can do to the host PC.  This is not something that MSE does, although real-time protection could be considered very similar  (the difference being in one case allowing something to run but isolating it, vs. monitoring in real time to block suspicious activity.)  Some browsers have a limited capability to do this (chrome...) but it is not user configurable as far as I know.
