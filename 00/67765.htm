Problem with paging file.

milagrosm2020 say: Hi. Steven
My name is Ben
Independent Advisor
I hope to help.

To do this, minimize the paging file, the Windows paging file, or pagefile, support system crashes and allow the system to use physical RAM more efficiently by writing some content from the files on the disk if the main memory is close to your computer. Capacity.

On most Windows 10 systems with 8 GB of RAM or more, the operating system manages the size of the paging file very well. 2.5 GB is typical for 16 GB systems and 5 GB for 32 GB systems.

However, the more memory installed, the smaller the paging file you need, because it is unlikely that the computer will use all of its memory at once. This means that you can reduce the size of the paging file.

To minimize the paging file, follow these steps:

1. Open System Properties. (Right click on the Start menu or Start select System or System and then click on Change Settings or Change Settings, or use the Control Panel or Control Panel).

2. In the System Properties dialog box, on the Advanced tab, click the Settings button in the Performance section.

3. In the Performance Options dialog box, on the Advanced tab, click the Change button in the Virtual Memory section.

4. In the Virtual Memory dialog box, note the minimum, recommended, and currently allocated size of the paging files in the last section. Next, uncheck the box that says "Automatically manage paging file size for all drives" (Automatically manage paging file size for all drives). If more than one unit appears in the list of available drives, click on the system unit (the one where Windows is installed, which is usually C :).

5. Click on Custom Size, and then type the values in the Initial Size (MB) and Maximum Size (MB) boxes, making the maximum size 1 GB (1,000 MB) or up to 4 GB (4,000 MB), depending on the amount of RAM installed.

6. Click Set (Set), and then OK (OK).

After these steps, close all open dialog boxes and restart the computer.

Note: If you have multiple physical drives (not just multiple partitions of a single drive), you can increase the speed of the system by dividing the paging file by two drives. Assign about 300 MB of paging file for the system unit, select the second unit in the Virtual Memory dialog box, set the initial size and maximum size values, and click Set.
