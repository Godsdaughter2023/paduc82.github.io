You will not get Excel to drag or copy and skip reference row like you want it to do.  You can however use a formula with references that will copy down.

INDEX($C$2:$C$21,MATCH(E2,$A$2:$A$21,0)) will return the corresponding batch total.  

If you are unfamiliar with these function MATCH will find the position of E2 in the range $A$2:$A$21.  Then index will use this number to give you the corresponding number in the range $C$2:$C$21

Use this in F2 and you have a formula you can drag or copy 
=0.25*INDEX($C$2:$C$21,MATCH(E2,$A$2:$A$21,0))


Note your expected result for batch 6 is incorrect .25 * 15 = 3.75 not 2.75
