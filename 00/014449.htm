john guin say:

There is no way to lock only a page.  OneNote gets it permissions from the folder that holds the section files.  If you have a set of pages that should not be changed, you can move them to a section in a read only folder.  Everyone will be able to see them without making changes.

OneNote 2010 has Versions, though, and they may help you.  The Share tab has a Page Version command that will show recent edits folks have made to the page.  You can restore versions from past edits if needed.  Give it a try.

Hi Michelle,

try with this queries:

Query1

SELECT A.ItemNo, A.UnitPrice, A.ReceivedQty, Table2.OnHandQty, A.CreatedDate, [OnHandQty]-Nz((Select Sum(B.ReceivedQty) From Table1 As B Where B.ItemNo = A.ItemNo And B.CreatedDate > A.CreatedDate),0) AS Qty, IIf([Qty]<[receivedqty],[qty],[receivedqty])*[UnitPrice] AS TotalCost
FROM Table1 AS A INNER JOIN Table2 ON A.ItemNo = Table2.ItemNo;

Query2:

SELECT Query1.ItemNo, Last(Query1.OnHandQty) AS UltimoDiOnHandQty, Sum(Query1.TotalCost) AS SommaDiTotalCost, Sum([TotalCost]/[OnHandQty]) AS WeightedAverageUnitCost
FROM Query1
GROUP BY Query1.ItemNo
ORDER BY Query1.ItemNo DESC;
