scottgem say:
No code is necessary. On your form use the Combobox wizard to create a combobox. If you follow the defaults for the wizard, it will create a combo bound to the field. You select the Supplier Table to lookup to and select the Supplier name field. The wizard will automatically include the SupplierID field but hide it. So the user sees and selects from the Supplier name. But the SupplierID is what is stored.

On a separate note, your Deliverables table should look like this:

tblDeliverables
DeliverableID (PK autonumber)
TaskID (Foreign Key)
SupplierID (FK)
SupplierPriority

You need a TaskID as an foreign key to identify which task the deliverable is for. You should not have 1st and 2nd supplier fields, but a SupplierID FK and a field that identifies the priority of the supplier (1, 3, etc.)

This way you can more easily report on supplier and the tasks they are assigned to.

In ref to (1), insert a placeholder on the Layout using the Insert Placeholder button on the Slide Master tab (when in Slide Master View).

(2) When you paste a slide you'll see the Paste Options button appear next to the bottom right corner of the slide.  From here you can choose Keep Source Formatting - doing this will copy the  Slide Master and all its Custom Layouts from the source presentation into the destination presentation.  Choosing Use Destination Theme (the default) the slide will inherrit the Slide Master of the preceeding slide, a new Layout is created.  You can change the layout of a pasted slide to one you want by using the Layout button on the Home tab.  Once a layout is used by no slides you can delete it.

(3) I meant copying a slide or slides.

(4) Go to Master View.  Add a new layout.  Switch off background graphics for that layout.  Draw new graphics on the layout.  Done.

Remember that "uneditable" graphics can appear either in a Slide Master or on a Custom Layout.  If in a Layout you'd have to copy them on to all Layouts where you want them.  If on the Slide Master, they will appear in all layouts, unless you turn off background graphics.
