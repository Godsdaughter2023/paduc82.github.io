Error code xxxx-xxxx-xx8x-xxxx-8007-xxxx occurs when you’re trying to download an update to your Xbox 360 console

Overview

If you see error code xxxx-xxxx-xx8x-xxxx-8007-xxxx (the first, second, fourth, and sixth 4-digit groups in the sequence vary according to factors such as region) when trying to download a console software update to your Xbox 360 console, it means that the update failed to download.

Note This error code is sometimes mistakenly searched for as "80070000000000.”

Read this page to learn more about xxxx-xxxx-xx8x-xxxx-8007-xxxx and what you should do about it.

On this page

Topics

Error code and message

Problem

Solutions

Error code and message

Error code xxxx-xxxx-xx8x-xxxx-8007-xxxx is preceded by the following message:

Cannot process update.

Status code: xxxx-xxxx-xx8x-xxxx-8007-xxxx

Problem

If you’re seeing this message and error code, it means that the update failed to download to your Xbox 360 console.

Solutions

To resolve this problem, try the following solutions:

Solution 1: Download the update again

Restart your console.

Press the Guide button on your controller (center X button on your controller).

Go to Settings, and then select System Settings.

Select Network Settings.

Select Wired Network or your wireless network name (if you're prompted to do so).

Select Test Xbox LIVE Connection.

Select Yes if you are prompted to update your console software.

If you still experience the problem, try the next solution.

Solution 2: Check your Xbox LIVE connection

If you're having problems connecting or staying connected to Xbox LIVE, see Xbox LIVE connection problems.

If you still experience the problem, try the next solution.

Solution 3: Make sure you have enough storage space

The latest console software update requires at least 190 megabytes (MB) of storage space (or 256 MB if you live in Japan). If you do not have enough free space, you need to move or delete some items. See Copy, move, or delete Xbox 360 content.

If you still experience the problem, try the next solution.

Solution 4: Clear the system cache

Removing the temporary files on your console might solve the problem. Follow the instructions at Clear your Xbox 360 system cache, then try downloading the update again.

If you still experience the problem, try the next solution.
