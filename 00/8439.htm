suzanne s. barnhill say:
That indicates that the document is being "scrubbed" using the Document Inspector (File | Info | Inspect Document. There's no way at this point to retrieve the original reviewer information, but to prevent the problem in future, go to File | Options | Trust Center | Trust Center Settings | Privacy Options and clear the check box for "Remove personal information from file properties on save." If the original author is using the Document Inspector (which results in this box being checked), you should make this change before you start editing (each time you receive the document back from the author). You might also want to caution the author about using the Document Inspector wholesale.

rick song msft say:

Hi Palcouk,

 

From the NDR you posted, the issue should be related to your domain’s SPF record. Your IP address wasn’t included in the domain’s SPF record. So when you send an email, your email address will be considered to be not able to send emails on behalf of the domain. For example, the SPF record is v=spf1 ip4:192.168.0.1/16 –all, then it only allows email address whose IP address is included between 192.168.0.1 and 192.168.255.255 while your IP address is not include.

 

For the detailed information ,you need to contact your admins and DNS provider to check whether they changed the SPF record. To fix the issue, the SPF records need to be corrected. Meanwhile, you can refer to this link for more information about SPF records: https://technet.microsoft.com/en-us/library/mt712724(v=exchg.150).aspx

 

If the issue persists, please provide the NDR via private message. Please click on the link below to access your Private Message : https://answers.microsoft.com/en-us/privatemessage/inbox

To create a new folder in Outlook 2010, right-click any folder in the Navigation Pane and choose "New Folder" from the pop-up context menu to create a subfolder of that folder.  To create a top-level folderm right-click your mailbox name (the root of the folder tree) in the Navigation Pane.  The process is similar for any Outlook version.  You can also click the Folder tab above the ribbon and choose the "New Folder" button on the ribbon.

If you are using a POP account, no folder on your client ever appears on the server.  However, if you are using an IMAP account, folders you create in the IMAP account folder tree should appear on the server as well.
