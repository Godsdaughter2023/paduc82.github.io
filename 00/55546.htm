Backup to Disk - Microsoft Community

frankduer say:

Backup to Disk

Hi, I would like to know how I can back-up my photo's to a disk in case my computer somehow crashes and I lose my photo's.

josé oliver-didier say:

Hello Frankduer,

Windows Photo Gallery does not have a built in backup functionality. However, there are a number of backup options available, and when it comes to precious memories like photos having a good backup strategy with more than one backup option is always better.

You could try Windows Backup:

You can check out the following article on Windows Backup

http://windows.microsoft.com/en-us/windows7/products/features/backup-and-restore


Now personally, for my backup strategy- I use OneDrive to keep my most recent photos continuously synced to the cloud and accessible on my smartphone. Onedrive works well with Windows Photo Gallery. For ALL my photos I periodically perform backups to an external drive using Acronis True Image.

Acronis also has a cloud backup solution. Additionally, there are other cloud backup services you can try such as Mozy or Carbonite. These services are a good option, if catastrophe strikes like the loss of your PC and the external backup device, but may take time to backup or sync if you have a lot of photos and a slow internet connection.

Now, when it comes to Windows Photo Gallery, make sure you are backing up all the folders which are included WPG.

Also, bear in mind that Windows Photo Gallery has a folder in which the original images of the ones which have been modified in the app are stored. Within Windows Photo Gallery you can configure to retain the original photos forever or have them deleted over a time. If you wish to retain the unmodified photos you should include the "originals" folder your backup strategy. The location of the folder may vary but check out this article which describes the location: https://support.microsoft.com/kb/944370/en-us

Hope this helps,

- J Oliver
