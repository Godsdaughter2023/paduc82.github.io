Select the cells you want to convert and run this macro:

 

Sub formulate()
For Each r In Selection
    r.Formula = "=" & r.Value
Next
End Sub

 

  Macros are very easy to install and use:   1. ALT-F11  brings up the VBE window 2. ALT-I     ALT-M opens a fresh module 3. paste the stuff in and close the VBE window   If you save the workbook, the macro will be saved with it.   To remove the macro:   1. bring up the VBE window as above 2. clear the code out 3. close the VBE window   To use the macro from Excel:   1.    ALT-F8 2.    Select the macro 3.    Touch RUN   To learn more about macros in general, see:   http://www.mvps.org/dmcritchie/excel/getstarted.htm
