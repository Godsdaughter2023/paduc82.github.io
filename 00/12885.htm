corrinnecareens say:

Copying and pasting to new document changes footnote style

Hi all,

I know how to ensure that "copy and paste"  portions of my text ("Document A) conforms to the style that I have in another document ("Document B"). Great for me! However, I notice that the footnotes from  "Document A" do not conform to the style of the "Document B".

How do I control this?

Using Office for Mac

Thanks in advance.

daniel g. grau say:

Let me state up front that you can solve many copy-and-paste inconsistencies if you copy styles between documents using Tools > Macro > Macros > Organizer and select the "Styles" tab at the top. 

When you have two documents, each with its own styles plus like-named styles, the following happens when you copy text from one and paste in the other:

  • If you paste a complete paragraph (including the ending ¶ character) that has a certain paragraph style applied and
    • that same paragraph style exists in the second document
      • the pasted paragraph adopts the character format of the style as defined in the second document;
      • any additional format (not defined in the styles) is preserved.
      • If the style was "Based on" another style that exists in the second document (even Normal), the pasted text takes on all the attributes from that based-on style, but keeps any manual/additional changes. If the source's "based on" style does not exist in the target, the pasted style is redefined with all the required attributes.
    • the style name does not exist
      • the style is created in the second document, preserving any formatting dependencies specified in "Based on."
  • If you paste part of the text into a paragraph (not including the ending ¶ character)
    • the text takes the formatting as defined in the style of the target (i.e., takes on as much of the style as it can, character attribute by character attribute);
    • any change additional to the source's paragraph style definition is preserved.

Word strives for consistency, not originality. "OK, he's pasting a complete paragraph with style INDENT but I already have that style here, so let's respect it." Word will preserve extra formatting only where it cannot be consistent.

When pasting between two documents, the above criterion will work correctly as long as you use styles just about everywhere and if most of them have identical names. If you need a title in boldface, define a style for titles in both documents. If you'd like a style to change automatically when altering another style, then make the second style "Based on" the first one, and in both documents.

Then, even if the documents look a lot different, pasting from one into the other will always be a "paste and auto-match" behavior.

Most of what I said above in reference to "character formatting" (bold, size, underline style, color, font…) also applies to "paragraph formatting" (line spacing, indents, space before, keep with next…).

greginmich say:

If Windows Defender is actually turned off via Group Policy, as suggested in the thread cited above by Jsssssssss, then you should be able to fix things with PowerShell.

 

Right-click on the Start button and select Windows PowerShell (Admin), and then copy, paste, and enter this command:

 

REG DELETE "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware

 

On older versions of Windows 10, this would immediately update the status of Windows Defender, but on the newer versions that include the Windows Defender Security Center app, you’ll need to use the Restart power option in order to have the setting take effect.
