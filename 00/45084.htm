steve winograd say:

A network bridge combines two or more physical networks into one logical network.  For example, if you had separate wired and wireless networks, you could bridge them to allow the computers on each network to access the computers on the other network.

Your computer is probably trying to access the Internet through router 1.  To prevent that, manually assign the IPv4 properties to the network connection that uses router 1.  Assign an IP address and subnet mask, but don't assign a default gateway.

llamalad say:

That stuff is stored in the Windows Registry.  From my notes:

 

How to disable Search history and Search box suggestions:

When you enter search terms into a Windows 7 Search box, they are stored in the Windows Registry at HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\WordWheelQuery and appear as suggestions for later searches.  To remove the stored terms, delete that entire part of the Registry.  To prevent the storage of search terms, go to HKEY_CURRENT_USER\Software\Policies\Microsoft\Windows\Explorer, rightclick on Explorer, mouse to New and click DWORD (32-bit) Value, type DisableSearchBoxSuggestions, and press Enter.  Doubleclick DisableSearchBoxSuggestion, type 1 for its value, and click the OK button.

Never mind! I found it and cleared it. Thank you

 

Open Registry editor.
1. Type regedit in the start search field and press enter
2. Browse to the following registry entry:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\WordWheelQuery

3. Delete MRUListEx on the right pane by right-clicking on it and selecting delete.
4. Close registry editor.

Open Windows explorer, test and see if the search results are cleared.

You can opt to permanently disable Windows Most Recently Used lists (MRU) if Group Policy is enabled in your edition of Windows. Group policy editor is available only in Professional, Ultimate and Enterprise edition of Windows 7. You will not have the Local Group Policy Editor available in the Windows 7 Starter and Home Premium editions.

Open Group Policy Editor:

1. Click start and in the search area type gpedit.msc
2. Now the group policy editor will open. In it go to User Configuration > Administrative Templates>Windows Components>Windows Explorer
3. In the settings panel under Windows Explorer look for Turn off display of recent search entries in the Windows Explorer search box
4. Right click it and select edit.
5. Use not configured or disabled to keep search history enabled.
6. Use Enable to disable the search history.
7. Close the group policy editor.
