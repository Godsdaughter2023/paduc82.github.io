Hi Julian,


There could be a number of factors as to why you are getting "Error opening the file" pop-up box. This could be caused by corrupted system files or due to a software conflict.

I would like to know more about your concern. Kindly verify the changes made to the computer prior to the issue.

Meanwhile, we suggest that you run the System File Checker and DISM tool. The System File Checker (SFC) is a utility in Windows that scans for corrupted Windows system files. If it finds a problem, it will try to replace the problematic system files from a cached version that’s on your PC. If SFC can’t replace the files, the Deployment Image Service and Management Tool (DISM) can check the corrupted files then download and replace the files with Windows Update.

  1. On the Start menu, type Command Prompt. Right-click the result and then click Run as administrator.      
  2. Copy or type each command followed by hitting Enter:

Three points:

1. These days, when a 1TB drive is about the smallest anyone buys, and
they cost around $50 USD, 1GB is about a nickel's worth. It's hardly
huge if you look at it in terms of dollar value.

2. If it's a tmp file, it almost certainly can be deleted. Tmp stands
for "temporary" and its life is normally over when the computer is
next booted.

3. To be on the safe side, you can change the file's name or move it
to a different folder rather than deleting it. Wait a week or two, and
if nothing complains that it can't find the file, delete it then.
