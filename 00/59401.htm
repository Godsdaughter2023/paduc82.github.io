Okay - so I just wen through a full power cycle from the tech support people (not just turning off and on) and it worked!

Here's the steps I was given;

1.) Go to "Settings" >> Disc & Blu-ray >> Persistent storage >> Clear persistent storage

2.) Press and hold the power button on the Xbox for 10 full seconds

3.) Unplug all cables from the Xbox

4.) Unplug power supply from the wall/surge protector and wait for the light on the "brick" to shut off

5.) While waiting for step 4, press and hold the power button on the Xbox for 2-3 seconds, repeating another 2 - 3 times; this will remove any residual power from the Xbox

6.) Once the light goes off the on "brick", reconnect the plugs to the Xbox, then the power supply back to the wall/surge protector

7.) Start your Xbox normally and try to start the campaign.

If it works, great!

If not, I would contact support, like I did.

Using F1 - F12 Keys on Universal Folding Keyboard - Microsoft Community

t. bishop say:

Using F1 - F12 Keys on Universal Folding Keyboard

OS:  Windows 10 (all updates and patches run as of 5:00PM EDT 9/3/2015)

Device:  Microsoft Universal Foldable Keyboard

I almost didn't buy the MUFK because it lacked F1 - F12 markings on any of the keys, and I could find no documentation regarding the functionality of the Fn key.

Based on trial and error, I am please to report MUFK DOES support F1 - F12 in Windows mode.

A couple of pointers. First, when pairing MUFK to a Windows device, make sure the Windows OS is selected (follow the diagram inside the box lid). It can be paired to a Windows device using one of the other OS settings, but you won't get F key functionality.

Second, the F keys map to the numeral keys, with three exceptions. 1 is F1 with Fn key pressed; same is true through F9. 0 is F10. - (hyphen) is F11, and = is F12.

I'm going to add a suggestion to the Feedback area that this be explained in some official Microsoft documentation or support page.
