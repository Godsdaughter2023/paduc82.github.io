Obligatory nag about Windows XP: 

Windows XP has not been supported since April 8, 2014.  It has become increasingly unsafe to use a Windows XP computer connected to the Internet.  Many third-party programs have stopped working with Windows XP and an increasingly large number of secure web sites (e.g., banks) will no longer work with Internet Explorer 8, which is the last version of IE available for Windows XP.

You may get that message if your wireless network adapter is being managed by a utility provided by the maker of the adapter or the maker of the computer.  You usually can determine this by clicking or right-clicking the icon for the wireless network adapter in the System Notification area (next to the clock display).

Once the Services applet opens, scroll down to "Wireless Zero Configuration" (it will be very near the bottom of the list) and double-click on that entry.

On the "Properties" page, select "Automatic" from the drop-down list next to "Startup type."

If the "Service status" is not already "Started," click the "Start" button.

Click OK and then close the Services applet

Hi Ted,

 

Just to share the difference between Windows Hello and Windows Hello for business.

 

Windows Hello just use fingerprint or face recognition uniquely identifies and authenticates that user to access Windows on that device.


Windows Hello for Business is Microsoft Passport technology. It uses "Windows Hello" to release a stored credential that is used as the second authentication factor by Microsoft Passport.

Maybe this helps.

From ASUS site (Laptop FAQ Battery)
Why doesn’t the battery charge when power level is above 95% with AC adapter connected?
 
This is a normal condition for battery protection, and there is no problem in either battery itself or charging function.
When battery is charged up to 100% , the notebook will automatically stop charging. At this time, battery will self-discharge very slowly owing to the chemical characteristics of Li-on battery even though the notebook doesn’t consume any battery power.
The current design of ASUS Notebook comes with built-in RSOC (Reconfigurable SOC) and the purpose is to prolong the lifespan of the overall battery.  Below description explain this phenomenon:
1. When the battery capacity is below 95% with AC adapter connected, the battery would be charged to 100% then stop charging.
2. When the battery capacity drops from 100% down to 99%~95% with AC adapter connected, the battery would not charge until power level drops below 95%.
3. When the battery capacity remained around 94% to 97%, battery would not charge with AC adapter connected. You can unplug and re-connect the adapter for several times to resume charging to 100%.
