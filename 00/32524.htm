The Something went wrong error shows up when Windows 10’s attempt to communicate with Microsoft servers fail. The following are some of the most effective methods you can try to resolve this issue:

    3. To make the newly created account an administrator, type the following command and press Enter: net localgroup administrators (name of newly created account) /add
