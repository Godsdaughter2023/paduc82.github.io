Rich is correct about this being a merged row. Note how the blue highlight continues across each cell until it moves up outside the right end of the row. mike h.. say:
Is it error prone?


Hi,

 

Yes it is.

 

Try this. On a brand new worksheet fill (say) a1:a20 with some data. Select a1 and run your code and it should correctly identify A20 as the last row,

 

 

Manually enter something in A30 and then delete it and run you code again and it will probably identify A30 as the last row.

 

Save the workbook and run the code again and it will have forgotten a30 and correctly identify a20 as the last row.

 

My favourites

 

Last in a worksheet

 

lastrow = Cells.Find(What:="*", After:=[A1], _
SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Row

 

Last in a column

 

lastrow = Cells(Rows.Count, "A").End(xlUp).Row

Sub Demo()
    [B2] = "Junk"
    [D4] = "Junk"
   
    'Correct - D4
    Debug.Print Cells.SpecialCells(xlCellTypeLastCell).Address
   
    [D4].Clear
    'Not Correct - D4
    Debug.Print Cells.SpecialCells(xlCellTypeLastCell).Address
   
    'First, Do a Reset
    ActiveSheet.UsedRange
    'Now, Correct  B2
    Debug.Print Cells.SpecialCells(xlCellTypeLastCell).Address
   
End Sub

2013 Publisher --4 Panel brochure only one size option? - Microsoft Community

ann farrow say:

2013 Publisher --4 Panel brochure only one size option?

I am looking to make a four-panel brochure (8 pages in total) but I am looking to make the pages each approximately 8.5 x 11. I would like to have some graphics that flow from one panel to another. I can't seem to find a way to do this currently with Publisher unless I manually guestimate the inches and build my custom template, but I fear that it won't be suitable for sending out to have them printed and folded. Any help would be greatly appreciated.
davidf2 say:

I can't imagine a 4-panel brochure on a page just 11 inches wide as there wouldn't be enough room to do anything on each panel. I think that is why you don't see that as an option in the templates. Most people that are going to use letter size paper will do a 3-panel.

Never the less you can select one of 4 panel brochure templates > resize the page to 11 inches (Page Design > Page setup group > down arrow below Size and change to letter from legal) and then resize the four panels and content to fit on 2.75 inches of space instead of 3.5 inches.

Make sure you go to View and check Boundaries, Guides and Rulers. Also go to Page Design > Layout and check Align to Objects and Guides.

After you have resized the page the vertical guide lines may dissapear. If so go to Layout > Guides > down arrow and choose the 4-panel of 3 vertical guides. leave the center group of guides where it is and move/drag the other two double lines to 3.5 inches, or a 16th inch on each side of that quarter page as per the Ruler at the top of the page.

Now since you can see the Boundaries of your text boxes and you have the 'snap to' function activated you can drag the boudaries of the text boxes to snap to the guides that form each quadrant. You can also drag the picture border across multiple panels. The text boxes are going to be very skinny, not much more than 2 inches which is not very workable in my view, but you have your 4 panel brochure on letter size paper.

Good luck.

DavidF

doug robbins - mvp office apps & services (word) say:

Use Alt+F9 to toggle off the display of field codes.

suzanne s. barnhill say:
Those are not RTF tags. The one you cite is a TC field, which generates a TOC entry. TC and XE fields are formatted as Hidden text. Hide Hidden text and you will not see them. Removing them may have unexpected and unwanted consequences.
 
If, however, you are sure you want to delete them, then you can (with the fields displayed) search for ^19 TC and replace with nothing. That search string will result in selecting the entire field when it is found, so the entire field will be deleted.
