There is a procedure you can follow to sort your problem:

1. Connect the phone to the PC via USB and select "Data Transfer Mode"
2. Run route66 sync on the PC.
3. Select the map files, etc in the right hand pane within route66 and press backup.
4. When complete you should see the backup file in the left hand pane of route66.

7. Start route66 sync again and restore the backup to the mobile.

System Compression in Disk Cleanup utility is a feature that compresses files that Windows hasn't accessed for a specified period of time.  It compresses the old files and it doesn't delete any files from the drive. The files are still available, but there will be a slight increase in access times because the files will be decompressed the next time they are accessed.
