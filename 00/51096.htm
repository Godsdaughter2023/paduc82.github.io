Huge Unknown .Cab File taking up my entire hard drive. - Microsoft Community

jewels207 say:

Huge Unknown .Cab File taking up my entire hard drive.

Hi.  I'm running Windows 7 on my lenovo tablet, and I'm almost out of disk space on my C: drive.  I turned on all hidden files and system files, selected them, and looked at their properties.  The total space used by these files, 129 GB, is nowhere near the 252.5 GB that supposedly makes up the sum of used space.  My C: drive is only 287GB, so you can see that nearly half of it is unaccounted for.

I ran WinDirStat to figure out where the missing data was, and it turns out that there's a 116.7 GB unknown .cab file on the drive.  It's address is C: ___.cab (there's no name before the .cab).  Do I need this .cab file?  What's it encompass.  There's no way for me to access it and explore it.  Is this a shadow copy of all of my files?  How can I pare it down or eliminate it so that my computer runs faster?

Thanks,
J
jewels207 say: I figured it out.

First of all, that wasn't a .cab file.  It was actually just the way that windirsat tree represented unknown space, which they gave an arbitrary .cab extension.

Secondly, the problem stemmed from Lenovo's automatic backup program.

There was a system file called RRBackup that I was able to explore using the MiniTool Partition Wizard Home Edition (free download version).  I couldn't access RRBackup's content on my normal computer's directory, and to even see it, I had to go into Folder Options and deselect "Hide Protected System Operating Files (recommended)."

I went into the RRBackup file on the MiniTool to look around, and sure enough, there were my missing 100+GB of disk space.  No good. 

So, to delete Lenovo's backups, I followed the advice on this post:
http://www.howtogeek.com/forum/topic/thinkpad-rrbackups-and-disk-space
Then I backed up my computer somewhere else.


In a nutshell, here's what I did:
  1. Opened Lenovo Rescue and Recovery (aka, Lenovo’s Enhanced Backup and Restore)
  2. Launched advanced Rescue and Recovery
  3. On file menu, opened Advanced, then clicked Delete Backups
  4. Deleted the backup of my C: Drive and setup the backups to be stored on an external hard drive.
  5. Backed up my computer on a terabyte-sized external hard drive. 


FYI, my tablet is a Lenovo Thinkpad, model x201 2985eyu, 64 bit, running Windows 7 Pro. 


I'm nowhere near an expert, and anything I say should probably be taken with a grain of salt, but this seemed to have worked very nicely for me.
