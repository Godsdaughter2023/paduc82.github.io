Why is the default paste option "keep source formatting" - Microsoft Community

avielshapira say:

Why is the default paste option "keep source formatting" instead of "match destination formatting"

It seldom makes any sense to use source formatting instead of destination formatting or keeping text only.  Using source format is unsightly, goes against the rules of good writing (see the Chicago Manual of Style) and reduces readability. It also increases the size of the document because fonts are embedded.  

It does not make any sense when pasting text from one Powerpoint presentation to another or even a whole slide. One should always use the destination theme. Presentations with mixed formats are unsightly and distract the audience. Pasting from PP to Word or vice versa is equally bad. 

I suppose that in a few situations keeping the source' format is desirable, but these must be very uncommon in everyday writing. 

In Windows applications, there is at least an option to select the default (which should have been "match destination" in the first place). Not so on Word for Mac 2016.
