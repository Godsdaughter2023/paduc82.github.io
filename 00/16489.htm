hansv mvp say:

I assume that you haven't protected the sheets yet. First, run this macro:

 

Sub UnlockAndProtect()
    Dim i As Long
    Dim wsh As Worksheet
    Dim rng As Range
    For i = 1 To 4
        Set wsh = Worksheets(i)
        Set rng = wsh.Range("A12:IU" & wsh.Rows.Count)
        rng.Locked = False
        wsh.Protect 'Password:="secret"
    Next i
End Sub

 

If you want to set a password, change the word "secret" to the password that you want to use and remove the apostrophe before the word Password.

 

In the Visual Basic Editor, double-click ThisWorkbook, then copy the following code into the module that appears:

 

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
    Dim i As Long
    Dim wsh As Worksheet
    Dim rng As Range
    For i = 1 To 4
        Set wsh = Me.Worksheets(i)
        wsh.Unprotect 'Password:="secret"
        Set rng = wsh.Range("A12:IU" & wsh.Rows.Count)
        On Error Resume Next
        rng.SpecialCells(xlCellTypeConstants).Locked = True
        On Error GoTo 0
        wsh.Protect 'Password:="secret"
    Next i
End Sub

 

The same comment as above about the password.

 

Switch back to Excel and save the workbook in a format that supports macros (.xlsm, .xlsb or .xls, but NOT .xlsx).

 

Users will have to enable macros, otherwise the code won't run.
