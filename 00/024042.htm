jim gordon mvp say:

The message you are getting lets you know that when you save in Microsoft Word file format (.docx) your macro code will be stripped out and removed. The default file format does not ever contain VBA code.

You must use Save As and choose a Macro enabled format (.dotm for template or .docm for document). These formats can contain VBA macros.

Whilethis article was written for Word 2011, the same formats apply in Word 2016.
