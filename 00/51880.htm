A File conversion Utility is a specialized application that opens multiple types of files and allows the user to to convert the file to another type.

A good example on a Macintosh computer is Graphics Converter  that will open any type of Graphic file jpg, jpg200, png, gif , pdf and converter it to a different versions.

Say you have a drawing in gif,jpg, or png and you want convert to say PDF to merge into an existing PDF.   You would open that file in its original format. Then on Mac you would use SaveAs and choose PDF the click on okay. A copy of the file will then become a PDF file. 

I am explaining this in Macintosh terms because that is what I use. I am sure that there are equal converter Utilities on UNIX,Linux, or Widows PC and the wording of commands will be different. But the Principle works the same.

Most application such as Database, Spreadsheet, Word Processing. Graphics program all have some limited abilities to convert to different formats. But a Converter or conversion Utility has much Richer abilities.
