Seems as if the network is somewhat sluggish. I would use a loop and wait until the folder is ready.

  Dim Path As String
  Dim fso As Object 'Scripting.FileSystemObject
  Path = ShellGetFolder("Z:\", "Select a folder", BIF_BrowseFolder)
  'Aborted?
  If Path = "" Then Exit Sub
  Set fso = CreateObject("Scripting.FileSystemObject")
  Do While Not fso.FolderExists(Path)
    DoEvents
  Loop

Maybe use a counter (and a wait time) and/or an Errorhandler inside the loop to avoid endless loops.
