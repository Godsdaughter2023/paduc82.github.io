jay freedman say:

The advice you've gotten is correct -- the only way to recover an overwritten file is to have another copy of it somewhere else as a backup.

Other than a manually created backup such as the one you have from October, there are two possible ways you could have a backup file that you weren't aware of. Both of them depend on having the mechanism set up before the overwriting occurred. If neither of them was turned on when you last saved the file, then there is no going back.

  • In Word 2007, go to Word Options > Advanced, scroll down to the Save section, and look at the setting for "Always create backup copy". If that box is checked, then there should be a file in the same folder with the document, named "Backup of <document name>.wbk". That will be the file as it existed just prior to the last Save. If there is such a file, you can rename it to something like "Recovery of <document name>.docx" and open it in Word.
  • In Windows 10, the File History feature can back up selected folders at regular intervals, which would give you multiple versions of the document to recover. The feature isn't turned on by default, and it does require an external drive (possibly a USB drive or network share) or a second drive inside the computer. See https://support.microsoft.com/en-us/help/17143/windows-10-back-up-your-files for setup instructions.

rmhunt16 say:

Conditional Formatting with Dates

I have a spreadsheet with expiration dates and I want the cell to change colors when it comes within 30 days of the expiration date and change to a different color if it passes the expiration date.  How can I do this?
hansv mvp say:

Let's say the expiration dates are in D2:D100.

Select this range, then click Conditional Formatting > Manage Rules... on the Home tab of the ribbon.

 

Click New Rule...

Select "Format only cells that contain".

Leave the first dropdown set to "Cell Value".

Select "less than" from the second dropdown.

Enter the formula =TODAY()+30 in the box next to it.

Click Format...

Specify the desired formatting, then click OK twice.

 

Click New Rule...

Select "Format only cells that contain".

Leave the first dropdown set to "Cell Value".

Select "less than" from the second dropdown.

Enter the formula =TODAY() in the box next to it.

Click Format...

Specify the desired formatting, then click OK twice.

 

Finally, click OK to close the Conditional Formatting Rules Manager.

e.j. gun say:

If you data is in Column A, starting at A1, and expiration date in $B$1, use two rules:

 

For dates prior to the expiration, but within 30 days of it:

=AND($A1<$B$1,$B$1 - $A1<=30)

 

For dates beyond the expiration:

=($B$1<$A1)

 

Adjust the ranges to suit your data set.
