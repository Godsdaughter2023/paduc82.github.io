Insert the code below into ThisWorkbook module. It will run and insert the pagebreaks every time the worksheet print Previewed or Printed. Not sure of what your expertise is here so following steps are required.

Alt  F11 to open VBA editor

Double click ThisWorkbook in project explorer on left of screen

Copy the code below and paste into the VBA Editor

Close VBA editor (X with red background top right)

Save workbook as macro enabled workbook.

If not already set then need to enable macros.

Click MS button (Large colored button top left of screen)

Select Excel Options (towards bottom right of dialog box)

Select Trust center (left column)

Select Trust Center Settings (towards centre right side of dialog box)

Select macro Settings (Left column)

Select Disable all macros with notification.

click OK and OK again to close the dialog box.

 

 

 

Private Sub Workbook_BeforePrint(Cancel As Boolean)
  Dim lngLastRow As Long
  Dim i As Long
 
  With ActiveSheet
    lngLastRow = .Cells(.Rows.Count, "A").End(xlUp).Row
   
    .Cells.PageBreak = xlPageBreakNone
   
    'Following assumes first row of column headers repeated each page
    'changing Step 40 will alter number of rows per page
    For i = 41 To lngLastRow Step 40
      .Rows(i).PageBreak = xlPageBreakManual
    Next i
   
    With .PageSetup
      .PrintArea = "$A$1:$P$" & lngLastRow
      .PrintTitleRows = "$1:$1"
      .CenterHorizontally = True
      .FitToPagesWide = 1
      .FitToPagesTall = False
      .PrintGridlines = True
    End With
   
  End With
 
End Sub

 

The code makes the first row the column headers on every sheet and makes the printout 1 page wide by how ever many pages long is required. it also centers horizontally on the page.

You can record code to perform other page setup functions. However, the recorded code will be in a standard module and you will need to copy the relevant parts and paste into the above code. If you have problems doing this then let me know what else you require in the setup and I will edit the code for you.
