COUNTIF function, is designed to count the number of cells in a range meeting a particular criteria.

I am sure you would be aware if this.

Let us take an example.

=COUNTIF($A$1:$A$25,"Tom")   will count the number of cells in the range A1:A25 where the content of cell is "Tom"

Suppose say, in the above range, the name "Tom" appears 4 times, then the above formula would

return an answer of 4.
