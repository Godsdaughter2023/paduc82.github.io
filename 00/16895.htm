scottgem say:
The SalesPersonID in the Customer table should be a Number>Long Integer datatype. This is the underlying type for an Autonumber. A Foreign Key field needs to be the same data type as the Primary Key it links to. So what you will be storing in your Customer table is the ID of the Sales person. When you create the combobox to select the salesperson you will set it so it shows the name, not the ID, but stores the ID (wizard will do this for you). When you need to do reports you will create a query joining the two tables and pulling the name values from the Salesperson table.
