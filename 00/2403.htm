andy indo say:

Why isnumber result is false for a number within a cell

Dear Expertise,

I have 32 x 4 within cell N7.  

               

4      =FIND("x",N7)                                            

32   =LEFT(N7,FIND("x",N7)-1)              

  =LEFT(N7,FIND("x",N7)-1)              

FALSE       =ISNUMBER(LEFT(N7,FIND("x",N7)-1)) ……….formula (3)

 

Request:

What’s wrong with formula (3).

How to revise this formula so result should be “TRUE”

 

Best regards,

Andy Huang

hansv mvp say:

The result of LEFT is always a text value, even if it looks like a number. You can use -- to convert it to a number, if possible:

=LEFT(N7,FIND("x",N7)-1) returns the text value "32".

=--LEFT(N7,FIND("x",N7)-1) returns the number 32.

=ISNUMBER(--LEFT(N7,FIND("x",N7)-1)) returns TRUE.
