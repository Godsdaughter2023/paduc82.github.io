3) Advanced

4) Find Now

6) check the box "Replace Owner On Subcontainers and Objects"

1) select "Change Permissions" near the bottom

2) select EDIT if the new owner is there...

ADD if not and steps 3-5 above

3) Give the new user\owner full permissions

4) check the box "Include Inheritable Permissions From This Object's Parent""

charlie klausen say:
Hi AnniPanwar,

Click start, and type “services.msc” and open it up.

Find “Windows Update” and double click it. Windows Update Properties will open, and you can change it to “Disable” - click apply, and close out of services. I’d reboot, and then check services again to make sure it’s still disabled, but this should prevent your laptop from trying to grab updates moving forward.


You should also be able to “Set as a metered connection” - which will give you the option to download/install updates.

https://support.microsoft.com/en-us/help/402845...

Select Start > Settings > Network & Internet > Wi-Fi > Manage known networks.
Select the Wi-Fi network > Properties > turn on Set as metered connection.

Hope this helps!
