lemp say:
Both a stand-alone installation of Windows XP and a virtual instance of Windows XP running as "XP Mode" on Windows 7 (or in another virtual machine) will "run" after April 8, 2014. 

After that date there will be no more security patches (or other updates) and thus any such instance of XP connected to the Internet will become increasingly vulnerable to attack.

If your software package doesn't need Internet connectivity, you can run it on XP for as long as you like.

I haven't seen anything lately, but a year or so ago there were rumors that after Microsoft stopped extended support for Windows XP that it would also remove the activation requirement.  If that happens, you'll be able to do a reinstallation of XP without worrying if your product key will be found to be invalid and thus keep your system from running normally.
