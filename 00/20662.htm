gary's student say:

If you have values including blanks from Q4 thru Q98, first enter the following UDF:

 

Public Function NoBlanks(rIn As Range, L As Long) As Variant
Dim K As Long
NoBlanks = ""
K = rIn.Count
If L > K Then Exit Function
K = 0
For Each rr In rIn
    If rr.Value <> "" Then
        K = K + 1
        If K = L Then
            NoBlanks = rr.Value
            Exit Function
        End If
    End If
Next
End Function

 

Then in the spreadsheet, pick a cell and enter:

 

=NoBlanks(Q$4:Q$98,ROWS($1:1)) and copy down as far as you need.

   3. close the VBE window
