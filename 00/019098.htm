- - -

Does "Long running requests" really mean long running script?

If you are asking this as a developer, as someone who is maintaining this webpage, I can only suggest that you go carefully through all of the JavaScript on the page to see if you can find the script that's giving IE trouble.

You may get some help with this at MSDN, the IE Web Development forum:

https://social.msdn.microsoft.com/Forums/ie/en-US/home?category=iedevelopment

- - - - - - - - - -   - - - - - - - - - -

Here's my advice to users who are encountering this "long running script" problem...

If this is happening only on one (or a few) website(s) there really is (by Internet Explorer standards) a long running script on the page and IE is doing you a favor by providing the "Stop script" option. Click on the Stop script button.

- -

Again, if only a site or two are affected...

It might break a desirable feature on the site, but placing a site in Restricted Sites will disable JavaScript and the script messages will stop.

Internet Options> Security> Restricted Sites...

Internet Options> Advanced, unchecking Use smooth scrolling might help.

Running an ad blocker like Adblock Plus may help on some sites.

https://adblockplus.org/

- -

If you're seeing the message on all or nearly all sites, a browser add-on might be causing it.

Note: The Google Toolbar has caused script problems for some users. See if disabling that toolbar stops the messages.

Start Internet Explorer without add-ons from Start> Programs> Accessories> System tools> Internet Explorer (no add-ons)
or

from Start> Run (or Windows logo key + r) type in iexplore.exe -extoff

(iexplore.exe<space>-extoff) then hit Enter.

If the problem goes away, an add-on is causing it.

To determine which add-on is causing the problem see this old (but still relevant) article:

IEBlog  Troubleshooting and Internet Explorer’s (No Add-ons) Mode:
http://blogs.msdn.com/ie/archive/2006/07/25/678113.aspx

- -

A preventive measure: Internet Explorer will allow memory usage to increase to unreasonable levels Closing all iexplore processes from Task Manager may be inconvenient but will resolve this.

- -

Another option: Other browsers like Edge, Chrome, Opera and Firefox are more forgiving of these script problems. Consider using another browser on affected sites.

Don

Top 10 things that MUST be fixed before Windows 10 is released - Microsoft Community

big bad bombastic bob say:

Top 10 things that MUST be fixed before Windows 10 is released

I thought I would add a new topic where we list the top 10 issues that we think need fixing before Microsoft releases Windows 10 for manufacturing.

I certainly don't want any major irritations stopping Windows 10 from being successful.  If this were to happen, PC makers and software developers would be negatively impacted.

So here's my list, in order of most important to least important

1.  Greatly improve APPLICATION LOAD TIMES!  Slow-loading applications give the perception of SLOW, especially for 'spinny' (non SSD) drives, which people DO still have.  So PLEASE fix the slowness and inefficiency of EVERYTHING, from the 'Shell' interface (like 'ShellExecute()') to the registry and any 'paranoid' disk cache behavior.

2.  Restore ALL configuration options found in 7 (or even XP), particularly DESKTOP APPEARANCE (window borders, titles, fonts, colors, icon spacing, etc.).

3.  Have an available theme that looks like 7 or XP, so that we can IGNORE 'the metro' if we don't want it.  This would include window decorations, window border thickness, icons, and so on.

4.  'Classic Shell' as an option [built-in or Microsoft equivalent].

5.  Microsoft Login needs to be an OPTION, without being scared or coerced into using one, nor "not recommended" being so prominent. Local Login should not be "pre-empted" by a Microsoft Login.  In other words, I want to log in as "me" not "*** Email address is removed for privacy ***", and have a user name and picture that is DIFFERENT for the Microsoft login that does NOT show up in the login screen.  Or not, as a choice.

6.  Remove ANY and ALL requirements for 'Secure Boot' by equipment makers.  'Dual Boot' to LINUX should be possible without, for example, without Linux having to have a pre-loaded key.  The computer belongs to the USER, not Microsoft.  Leveraging things like 'Secure Boot' is bad policy.

7.  Remove ALL ADWARE from the distribution media.  That includes all of the metro apps in builds 9926 and 10041 that serve up ads as part of the 'Metro' advertisement API.  We are *NOT* "Advertisement Receptacles" nor "Marketing Units".  We are people.

8.  ONE CONTROL PANEL, please, preferably a CONTROL PANEL, not a metro 'Settings' app.  Yuck.  Make "Metro Settings" a CONTROL PANEL APPLET if necessary.  And I like the traditional 'small icon view', so please don't take that away.

9.  Eliminate the alphabetized "All Apps" and give us our hierarchical Start Menu back BY DEFAULT, WITHOUT the BILLBOARD area, and if you MUST pre-install that, make ALL of the tiles SMALL and give us an option to EMPTY IT with a single command.

10.  Uninstalling (and possibly, installing) metro apps is PAINFULLY SLOW.  Try it some time.  It takes around a MINUTE for the icon to go away after I choose 'uninstall'.  What's really funny is if I uninstall several of them in succession, then wait SEVERAL MINUTES for it all to 'take'.  Sometimes it doesn't "take" at all.  But I really don't want all of that "CRUFT" pre-installed anyway, so maybe you should have us check a list as to which ones we REALLY want during the install process?  OR, give us a SINGLE OPTION to WIPE THEM ALL AWAY.  And DO NOT make them re-appear when I create a new user!!!  And as for "seting up your applications" which took 12 minutes PER USER in build 10041?  UNACCEPTABLY SLOW!

Cookies or Cookies and website data
History

For other web browsers:

Chrome

In the browser bar, enter:

chrome://settings/clearBrowserData

Select the following:
Browsing history
Download history
Cookies and other site and plug-in data
Cached images and files

From the Obliterate the following items from: drop-down menu, you can choose the period of time for which you want to clear cached information. To clear your entire cache, select the beginning of time.

Exit/quit all browser windows and re-open the browser.

Firefox

From the History menu, select Clear Recent History.

If the menu bar is hidden, press Alt to make it visible.
From the Time range to clear: drop-down menu, select the desired range; to clear your entire cache, select Everything.
Next to "Details", click the down arrow to choose which elements of the history to clear; to clear your entire cache, select all items.
Click Clear Now.
Exit/quit all browser windows and re-open the browser.

Microsoft Edge

In the top right, click the Hub icon (looks like three horizontal lines).
Click the History icon, and then select Clear all history.
Select Browsing history, then Cookies and saved website data, and then Cached data and files. Click Clear.
After the "All Clear!" message appears, exit/quit all browser windows and re-open the browser.

If you define your holiday test vector like this

={DATEVALUE("2010-01-01");DATEVALUE("2010-01-18");DATEVALUE("2010-02-15");DATEVALUE("2010-04-02");DATEVALUE("2010-05-31");DATEVALUE("2010-07-05");DATEVALUE("2010-09-06");DATEVALUE("2010-11-25");DATEVALUE("2010-12-24");DATEVALUE("2011-01-01")}

Try the following formula:

=WORKDAY(TODAY()-1,1,A1:A10)=TODAY()

This formula will return TRUE if today is not a holiday and FALSE if today is a holiday.

A1:A10 is a range of holidays that you can omit if you just want to consider Saturdays and Sundays.
