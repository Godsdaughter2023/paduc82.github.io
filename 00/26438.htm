    If you have multiple accounts in Outlook, when you compose a new email message, Outlook picks the default account as the From account. However, if you want to Reply, Reply All, or Forward an email, Outlook will send your email with the same account the original mail was sent to. 

    For example, Sender 1 has two accounts. The default account is first.account@email.com. The second account is second.account@email.com. If Sender 1 receives an email message from Sender 2 at her second.account@email.com account, when he replies to Sender 2's message, his reply will be sent using second.account@email.com. If he creates a new message to Sender 2 that's not a reply or a forward, the message will be sent using her first.account@email.com account.

    If you want to change the default email account, you can change it by using the following steps:

    1. Select File > Account Settings > Account Settings.
    2. From the list of accounts on the Email tab, select the account you want to use as the default account.
    3. Select Set as Default > Close.
