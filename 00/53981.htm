m3tal daz3 say:

Uninstall and reinstall the Teredo adapter

To uninstall the Teredo adapter:

  1. Right-click the Start icon, and then select Command Prompt (Admin).
  2. Run the following command from the Admin command prompt:

    netsh interface Teredo set state disable

  3. Right-click the Start icon, and then select Device Manager.
  4. Select View > Show Hidden Devices.
  5. Select Network Adapters.
  6. Identify any adapters that contain "Teredo," such as Teredo Tunneling Pseudo-Interface or Microsoft Teredo Tunneling Adapter. Right-click the adapter name, and then select Uninstall.
  7. Restart your PC.

To re-enable the Teredo adapter:

  1. Right-click the Start icon, and then select Command Prompt (Admin).
  2. Run the following command from the Admin command prompt:

    netsh interface Teredo set state type=default

vandergrif say: Weirdest problem I've had in a while - but I fixed it. Anyway, if anyone in the future ever happens to have that same issue - in regedit go to HKEY_CURRENT_USER then Software, then Microsoft, and finally click on the Command Processor header. In that I found an entry specified AutoRun, and in the value for it (along with a bunch of other stuff) happened to be "20,5" which I changed to "85,35". The command window now opens at a functional size. So it would appear that was what was causing the problem.
