EDIT : Fixed the issue.  If you have this issue, and your windows search service is not starting, go here and download the  reg file for your version of windows. 

http://www.winhelponline.com/blog/windows-search-service-error-1053-windows-7/

The update required windows search service to be started and working. Strangely, my search functions worked so i had no clue that windows search service was not started and whenever i started the service, I got an error stating "Error 1053: The service did not respond to the start or control request in a timely fashion"

I did a lot of searching and finally found that site. Decided to give the reg file a try and it worked. I couldn't believe it. Windows search finally started and then i tried to install update  KB4074588 again and it worked!
