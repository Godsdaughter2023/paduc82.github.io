Hi Shalini,

The ADFS name space (ADFS service name) should be refer to the public domain: sts.frankloft.com. This name should be resolved over Internet. If this name is an internal domain “xxx.local”, ADFS/SSO will not work in external network. Therefore, please use sts.frankloft.com as the ADFS name space.

Moreover, in the public DNS hosting provider, please add a DNS A record for sts.frankloft.com, which points to your public IP.

Anything unclear, please let us know.
