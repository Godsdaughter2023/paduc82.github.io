There is no tab named Tools. The "Tool Tabs" are special tab sections that are only displayed when a specific item is selected.

For example, if you select a chart, you'll see a Chart Tools section on the ribbon, with tabs Design and Format:

These tools sections are not displayed if your selection is not a chart or a cell in a pivot table.

How to set default font for New Workbook - Microsoft Community

randybw57 say:

How to set default font for New Workbook

Is there a way to change the default font for New Workbooks for iOS Excel?

So anytime I create a New Workbook the new font is used.

rogerwaltes98 say: Hi randybw57

My name is Roger W. Im an independent advisor

By default, Excel for Mac uses the body font in font size 12, which is displayed as a Calibri font in font size 12 when you type data in a spreadsheet, header or footer, or text box . However, you can change the default font and font size of all the new books that you create.

On the Excel menu, click Preferences.

In Creation, click on General.

In the Default font and Font size boxes, select another type and another font size.

Click on OK

Note: This will not change the source of any existing book.

Close and restart Excel.

The new default font will be applied when creating spreadsheets or blank books.

Source https://support.office.com/en-us/article/change...
