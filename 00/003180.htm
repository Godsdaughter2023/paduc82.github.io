If the Outlook.com account is configured in Outlook 2013, first check the junk email settings. Junk Email options is on the Home ribbon. Is uit set higher than No filtering or Low filtering?

outlook.exe /cleanserverrules

This will clear any rules on the server - I've seen a few cases where the person did not knowingly set a rule and it wasn't showing in Outlook desktop but had messages going into RSS or the mailbox root folder.
