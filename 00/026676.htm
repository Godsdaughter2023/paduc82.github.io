Matching Destination formatting is done as follows and the Destination format will remain as is:

PasteSpecial -> Values

or

Reason for having separate PasteSpecial -> Formats is to allow PasteSpecial -> Values and then immediately PasteSpecial -> Formats to that both the Values and Formats can be pasted without the formulas.
