Hello Andre:

Thank you for posting to the forum regarding the issues you are having syncing your account with your Windows Phone.  This sync error indicates the phone is not properly connecting with the Microsoft account servers.  The following have been found to resolve errors like this.

  • Try a soft reset of the phone.  First, go to settings > application > Internet Explorer > delete history.  Once this is done press and hold the volume keys, power button and camera together for 3-5 seconds. 
  • On the phone go settings > time + date > set automatically to OFF > set the time, date and time zone.  Once you do this go to settings > emails + accounts > tap and hold the Microsoft account > select sync. 
  • If the account still will not sync the next thing to do is go to the following site and sign in with the same Windows Live account that is on the phone, http://account.live.com .  Once signed in here, go to password & security.  Look at the section related to "two step verification".  If you see the option to disable this, you will need to create an app password on this page or disable two step verification.  If you create the app password, that password would be used as your sign in password on the phone.  If you see the option to ENABLE two step verification, you will want to reset your account password.  Make sure the password is 14 characters or less. 
  • The final thing you can try is to change the server name that is listed on the phone.  You can do this by going to settings > emails + accounts > tapping the account > tapping past the error.  Go to the bottom of the account setting page and you will see a server name field.  Here enter each of the following and they try and sync the account after each one; m.hotmail.com, bay-m.hotmail.com, blu-m.hotmail.com and dub-m.hotmail.com.
