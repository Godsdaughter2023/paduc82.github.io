electricbee1 say:

Where do I find a Recorded PowerPoint Presentation in Office 2010

I recorded a PowerPoint Presentation using the Built-In Software. But, I don't know where to find that recorded file. Where would it be on my Computer or my OneDrive?
steve rindsberg say:

PowerPoint now embeds rather than links sounds by default, and this would apply to sounds recorded using the record narration feature.

Try this:

Make a copy of your presentation file.

Rename the copy to put a .ZIP extension at the end of the file name (you'll have to make sure that Windows is set to show you file extensions first; if your file name doesn't already end in .PPTX, then Windows is hiding stuff from you.  

Once the file's got a ZIP extension, Windows will treat it like a ZIP file and will open the ZIP archive so you can view the contents.  Look for a MEDIA folder within the ZIP and there you'll find your sound files.
