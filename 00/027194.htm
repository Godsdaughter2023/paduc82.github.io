tom hoke say:

Hi,

   You may use the desktop version of Excel to create a workbook where you hide all the rows & columns that you don't want to be visible ( the scrollbars will then not show because there is no rows/columns to scroll to ). Additionally you can set the red cells as "unlocked" and then protect the worksheet - this will make all the other cells "locked". The user will be able to select any of the cells they see but only be able to enter data into the ones you unlocked ( e.g. the red cells ).

I hope that helps,
