This error means that something may be wrong with a device driver, your PC's memory, or antivirus software on your PC. Here are some things to try:

  1. Check that your drivers are up-to-date by going to the search box and entering drivers. Select Update device drivers. In Device Manager, select Action > Scan for hardware changes. Restart your PC.

Troubleshoot blue screen errors

considerate_guy say:

A few clarifications:

  1. You can't use a CD in place of a DVD, so be sure you have a DVD-R disc, not a CD.
  2. You can't just burn an .mp4 or a project file (.wlmp) onto a disc and expect it to play. The disc must be encoded in the MPEG-2 DVD standard so that DVD players understand it.
  3. Windows DVD Maker needs the file in .wmv (Windows Media Video format). Try a test save to your computer, but instead of the H.264/MPEG-4 format (.mp4), pick Windows Media Video (.wmv) format instead.
  4. You should then be able to add that file to Windows DVD Maker, pick a menu, etc. Make sure everything previews as expected and then burn a DVD. That should play in a standard DVD player.

Hi,

To resolve your 100% disk usage issue, please follow the steps below:

  • Press Windows key+i to open Settings, then select Privacy.
  • Under Privacy, select Feedback & diagnostics.
  • Under Diagnostic and usage data, change Send your device data to Microsoft to Enhanced or Basic.
  • Close the Settings.

Let us know how it goes.
