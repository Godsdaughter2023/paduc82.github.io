Restarting steam service is really just a workaround and does nothing to help whatever is causing steam to crash.

After you upgrade to windows 10 you can check the health of windows system files by running "sfc /scannow" in command prompt.  Upgrade process can have issues and problems. Try searching steam support pages for your issue AFTER making sure your windows 10 install is healthy.  No point in trying to run games if windows is not healthy.  ;)
