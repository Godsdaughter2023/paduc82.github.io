Alias vs Account - Microsoft Community

liby kim say:

Alias vs Account

If you had a personal email address and a work email address, would you group them up into just one account using the aliaa feature, or would you use separate accounts for each?

What would be your reasons?

0 people found this helpful

ron sommer mvp 2009-2017 say:

An account will have only one display name and signature.  An alias would use the same display name and signature.

A separate account would have its own set of contacts.

If you use a device to access your account when you send an email, the default "From" address will be address that the recipient sees.  This means that you would only send from a device using one address.

With two separate accounts, each account can be the alternate email address for the other account.

With one account, if the account was hacked, then both addresses would be hacked.

kukurukuku say: If you have a main Microsoft account and you added your work email as an alias, they both can be used as usernames to access the main email with the same password. 

Take note that existing Microsoft accounts cannot be added as an alias unless you deleted it, you will need to wait for 60 days before it can be added as an alias.

Much better not to add it as an alias, since if your work email have problems ( domain issues ), you may issue with verification. 

Learn more about aliases here ( http://windows.microsoft.com/en-us/windows/outlook/add-alias-account )
