jay freedman say:
The bottom-line answer is that the RSIDs can't be used for the purpose you have in mind. The relevant sentences in the blog article are in the last paragraph: "First is that the RSID tells us nothing about the time or order things were done. They are completely random, and are only used for seeing where things match. So they aren't of much use unless you are merging with another document that also has RSIDs."

Although the article doesn't say so explicitly, RSIDs are inserted in the document file only when Track Changes is turned on, or when you use the Compare Documents feature (which marks differences as if they were tracked changes). It seems unlikely that someone engaging in plagiarism will conveniently track changes and leave those fingerprints in the final document (although it could happen, as a lot of users know nothing about Tracked Changes, and turn it on by accident). If you get lucky that way, then the tracked changes themselves have date stamps -- but that has nothing to do with RSIDs.

john mcghie say:
No.  That stands for "Inherit underlying properties of the Paragraph style".

A Character style will override the Paragraph style, and direct formatting will override both.

However, if you APPLY a character style over character formatting, the character style will override the existing character formatting (either style-based or direct).  To override that, you need to apply the direct formatting last.

One other unpublicised characteristic ("feature" or "bug" depending on how much it annoys you...) is that if you apply a Paragraph style to less than half a paragraph, it will be internally changed into a "Linked Style" which has both Paragraph style and a Character style components.  If you select less than half, it will be applied as a Character style.  If you select more than half, it will be applied as the equivalent Paragraph style.

stefan blom say:

You can't get access to styles in a template by adding the template to the Startup folder. In Word, styles reside in templates and documents; there are no such things as globally available styles. If you create a new document from your template and then import existing content, you can easily reformat text. (Or you can copy styles manually between files.)

See if you find style sets useful. The built-in style sets are available via Home tab | Change Styles | Style Set, and you can create custom sets if you wish. However, creating separate templates for separate types of documents is far more convenient than depending on style sets.

Greetings paupav!  Welcome back to the Windows Phone forums!

 

The error message "Windows Marketplace not available at the moment" can be caused by many different things.

I presume that you are experiencing the problem beyond the realm of the occasional issue, as most people just try again in 5 or 10 minutes.

Presuming this to be the case, I would suggest making sure that your network connections are good:

1.) The WiFi bars at the top of the display are not signal indicators, just indicators of whether or not you are connected via WiFi.   The WiFi connection's real signal strength indicator can be found at Settings > Wi-Fi next to each SSID name.

2.) If you are using your cellular/wireless dataplan to connect, you can gauge your relative signal strength from your cellular/wireless signal strength indicator in the upper left-hand corner of your display.  It has an auto-hide feature, so if it's not visible, then you can force it to appear by sliding your fingertip in a downward direction from a point above the screen.  This will drag the interface with your signal strength and other indicators for approximately 10 seconds before disappearing again.

3.) If both connections are good, then your problem could be a flakey OS session, in which case I would recommend a soft reset by pressing and holding the power button.   Repeat the process to turn your Windows Phone back on.

Let us know if you continue to have problems and we'll go from there!

Please look into C:\Users\User_Name\AppData\Roaming\Microsoft\Excel (replace user name with your user name, you can directly reach here by copy and pasting following without quotes in Explorer address bar - "%appdata%\Microsoft\Excel") and see if is contains few files / folders. Move those files to some other location (i.e create a backup of those files / folders and delete all files / folders from here). Hence, make Excel folder blank.

Note - If the above path is not applicable for your installation, you need to find XLSTART path. Excel folder contains XLSTART folder in itself (In case, your XLSTART path is not in Appdata)
1. Open Excel
2. ALT+F11
3. If Immediate Window is not visible, type CTRL+G. Same can be also be accessed through View > Immediate Window
4. Type "? application.StartupPath" without quotes and press enter
5. You will have your path below.

3. Sometimes, add-ins can cause problems. To determine, if add-in is a problem start your application in safe mode and see if problem has gone away.

To start in Safe Mode -

Hold CTRL key and click on application icon and don't release the CTRL key till it asks you for Safe mode confirmation.

If yes, then start application normally and disable add-ins one by one and start application again every time you disable an add-in to determine the culprit add-in.
To disable add-in > File > Options > Add-ins > Down below you have Manage: Com Add-in : Press Go > Untick the Add-ins to disable and tick to enable them

4. Try to login through another Windows profile. If there is some corruption in profile which is causing this problem, this would correct.

5. Sometimes, this is caused due to excel not being able to access the default printer. Close all office applications and change your default printer through Start > Devices and Printers to another printer. It may be some physical one or One Note or XPS or PDF or anything else. Close them one by one and see whether it helps. Start your computer in Safe mode with networking and see whether the problem is solved. If yes, then some program is causing the problem.
https://support.microsoft.com/en-us/help/12376/...

7. You may try to disable the Hardware Graphic Acceleration and verify the result:

Open Excel.
Click on file, Options.
Go to the Advanced tab.
Under the Display section, check the box for 'Disable hardware graphics acceleration'.
Click ok and restart Excel.
