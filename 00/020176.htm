hansv mvp say:

You cannot add tables directly to the Linked Table Manager.

Instead, activate the External Data tab of the ribbon.

In the Import & Link group, click Access.

Click "Link to the data source by creating a linked table".

Click Browse...

Locate and select the database that you want to link to, then click Open.

Back in the Get External Data dialog, click OK.

Select the tables that you want to link to (simply click to select a table, click it again to unselect it).

Finally, click OK.

The tables should automatically be added to the Linked Table Manager.

ken sheridan say: Like most developers I don't use macros, but as I understand it the ImportAttachText argument of the RunCommand action merely opens the standard user interface for importing/linking to an external table or file.  If you use the TransferText action instead this allows you to specify the name of the linked table and automate the whole operation.  In VBA the following line for example links to a fixed width text file named Parts.txt in the current folder, and names the linked table NewParts.  It uses the Parts Link Specification which was set up previously, but you should already have a specification created by your earlier operations:

DoCmd.TransferText acLinkFixed,"Parts Link Specification","NewParts","Parts.txt"

The TransferText  macro action has identical arguments to the VBA method of the DoCmd object.

Microsoft Defender USB Scan - Microsoft Community

Microsoft Defender USB Scan

Does Microsoft Defender have the ability to automatically scan a USB drive as soon as its plugged in via Real-Time scanning?
sarahkong say:
Hi ZWebb

My name is Sarah Kong and I am an independent adviser that is here to try and help you with your issue.

There is a registry setting that you can turn the automatic scanning.
Before you do this create a restore point.
To do that search create a restore point and then create one.
Then go to search and type regedit and open it.

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender\Scan

Go to the key DisableRemovableDriveScanning
Set it to 0 to enable scan.
