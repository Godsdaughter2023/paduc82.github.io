Okay, so another redditor found a solution. What fixed it for us was a registry trick, so don't go ahead with this if you don't know what you're doing.

The idea is finding "zombie" users in the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList key. I had a key containing a value that pointed to the home folder of a user named UpdatusUser, which is used for NVIDIA driver updates. So I backed up that key, deleted it and the update went fine. I restored the key after I was done, to make sure there are no problems with my NVIDIA updates in the future. I'll try this again if I ever have problems with Windows Update, but it's a PITA. a_felix say:

How to know a profile is left over profile..??

Plz help..!!!

Here's some stuff that should be kept:

* %systemroot%\system32\config\systemprofile

* C:\WINDOWS\ServiceProfiles\LocalService

* C:\WINDOWS\ServiceProfiles\NetworkService

* C:\Users\DefaultAppPool

* C:\Users\<insert your username here>, for all "normal" users on the computer

These are all folders that exist and are necessary.

If you've deleted a usual user profile (through normal means) before and it's still in ProfileList, delete it. If you have NVIDIA drivers and you see a profile with the ProfileImagePath C:\Users\UpdatusUser, delete it if the folder doesn't actually exist (but back up the key anyway).

If there are other profiles with inexistent home folders, try using Google to find out what they are for and if they're safe to delete. After all, the update will fail as long as there are still home folders that don't exist.

Hope that helped.

andre da costa say:

Which versions of Microsoft Office work on Windows 10?

The following versions of Microsoft Office with the latest Service Packs have been confirmed to be compatible with Windows 10:

  • Office 2016 Preview
  • Office 2013/Office 365
  • Office 2010 with exception Office Starter 2010
  • Office 2007

Older versions of Office such as Office 2000, Office 2003 and Office XP are not certified compatible with Windows 10 but might work using compatibility mode.

andre da costa say:
Many users are reporting issues with this update, I recommend you block it until further notice:

Is there an option or work around to block updates or hardware drivers that might cause problems?
Yes, Microsoft has released a KB update (KB3073930) that will let users block or hide Windows or driver updates.

You can download it at the following link:
https://support.microsoft.com/en-us/kb/3073930

After downloading, launch it, click Next, wait while it detects pending Windows Updates.

Click the Hide Updates menu.

Scroll through the list then select then select the update. Check the box next to the update then click Next to confirm changes.
