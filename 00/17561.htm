In cell C1, enter the following array formula (Ctrl+Shift+Enter) and copy down

 

=INDEX($C$51:$C$62,SMALL(IF(ISTEXT($C$51:$C$62),ROW($C$51:$C$62)-ROW($C$50)),ROW(1:1)),1)

twyla naythias fox say:

A fair number of people have reported corrupted/incomplete downloads of the ISO(s), so this is the first thing I would check.  The downloads page lists both the correct size and the hash/checksum for each ISO - if either doesn't match, download a fresh copy using a different filename.  (Some have argued that the browser you use has an effect, but this is not true - corrupted downloads can happen regardless of browser.)

Once you're certain the ISO is correct and intact, try burning another disc.  No media is foolproof and write errors can occur even on the best systems.

Another possibility is that this system's optical drive may need cleaning or may otherwise be defective.  Try using Explorer (or a third-party utility such as Rufus) to create a bootable USB.  (Most of the ISOs will fit on a 4GB USB drive.)

Two other things to note:

One, double-check that your system is sufficient to run Windows 10:

  • 1 GHz (or faster) CPU
  • 1 GB RAM (for 32-bit) / 2 GB RAM (for 64-bit)
  • 16 GB (or more) of space on system drive
  • Graphics system capable of using DirectX 9
  • Display resolution of 1024x768 (or greater)

Two, even if your system meets all the above requirements, some older CPUs and hardware aren't supported by the Windows 10 Technical Preview.

If none of these factors addresses your issue, it is likely that the Technical Preview doesn't support this particular system.

tomk 15 say:

I found one cause and an easy fix for Office PowerPoint 2013.

PPT keeps a running list of the last 10 files opened.  If any of those files was on a removable drive (like a USB) that is now unoccupied, this error message will pop up every time you start PPT.

To stop the messages...

1) open PPT, get message, note the drive letter and close the error message

2) Right-Click on each filename on the short-list that starts with that drive: letter

3) Select "Remove From List"

When they are all gone from the Recent list, close PPT, reopen it and enjoy your error message free experience.
