MSE scheduling using the task scheduler. - Microsoft Community

dpjohn44 say:

MSE scheduling using the task scheduler.

I'd like to be able to schedule scans so the PC will awaken from hibernate.  The built-in scheduler does not have that option.
greginmich say:

It is possible to finagle the MP Scheduled Scan task (that Security Essentials sets up in Task Scheduler) so that it wakes the computer to do a scan, as described by bunklung in this thread: http://social.answers.microsoft.com/Forums/en-US/msescan/thread/d3f29cba-c0bc-4992-bb0b-c48dc3711a3b

 

But when you go back into the main interface and modify the scheduled scan settings, the MP Scheduled Scan “Wake the computer to run this task” setting is often automatically reset to the default (disabled); so it’s better to set up your own scan task if you want to use the “wake” option.

 

When you run the Scheduled Task Wizard, and browse to MpCmdRun.exe in the Security Essentials folder, the Task Scheduler paths for your new scan task should look like this for a Full Scan:

 

Run: "C:\Program Files\Microsoft Security Essentials\MpCmdRun.exe" -Scan -ScanType 2

 

Start in: "C:\Program Files\Microsoft Security Essentials"

 

For a Quick Scan you would substitute “-Scan -ScanType 1”.

 

The signatures will be updated automatically if the “Check for the latest virus & spyware definitions before running a scheduled scan” setting is enabled in the main interface (that’s the default setting).

 

 

GreginMich

In order to retrieve the emails that were deleted more than 30 days you can use Email Recovery App https://appsource.microsoft.com/en-us/product/o... First insert the Microsoft Account from which you want to retrieve emails (These instructions better to perform on PC in browser)

1. Login to https://outlook.live.com/
2. Click the gear icon at the upper right side of the screens then click view full setting.
3. Select Mail > Customize actions, then click Get Add-ins and hit save.

7. In the search result, under the Email Recovery, click Add button.
8. Go back to Customize actions page, click Email Recovery and click save.

10. Recover both Recoverable Items and Purges.
11. Please do not do anything while the recovery is in progress.
12. Once it’s finished, a folder will appear on the left pane which contains the emails that you recovered.

Hi

Unfortunately, the craze to make real applications act like web browser windows makes your request impossible to fulfill. The formula bar is integrated to the workbook window and can't be added to custom toolbars so that it can be moved and re-sized.

In order to get the formula bar locked into the window, a lot of re-engineering had to be done. It now has the ability to take up multiple lines (click the little triangle at the right-hand end of the formula bar to see this). Toolbars don't have this ability, so the formula bar is now forever locked to each window. Getting it all figured out how to do this in a way that makes sense was not easy, and for some of us it makes no sense at all.

If you want Microsoft to make the formula bar the way it used to be, use Excel's help menu and send feedback and make your request. I wouldn't hold my breath, though. This web-style fashion is like saggy pants, and it's not likely to go away soon.
