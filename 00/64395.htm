Thanks for any help!

azam k say:

Hi,

The WLX file type is primarily associated with 'Live Mesh' by Microsoft Corporation. Live Mesh is a way of storing changing files on a Web service yet accessing those files from one or more computers with automatic updates to and from the local and remote computers. The .WLX file indicates synchronization is taking place but is not yet complete. Often you will see a double extension with the actual file's extension coming before the .WLX extension.
