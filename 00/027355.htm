jeeped say:

I'm guessing that you have a header row in row 1 for column labels. AutoFilter really relies on this and you should include one if you don't have one. This is the way I go about it.

Sub tryDeletingOneMoreTime()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets("Discrepancies")
    With ws.Cells(1, 1).CurrentRegion
        .AutoFilter
        .AutoFilter Field:=2, Criteria1:="2"
        If Application.Subtotal(103, .Columns(2)) > 1 Then
            .Offset(1, 0).EntireRow.Delete
        End If
        .AutoFilter
    End With

    Set ws = Nothing

End Sub

.Cells(1,1).CurrentRegion is the equivalent of going to A1 and tapping Ctrl+A once. You should always check to see if there are rows to delete before deleting them. Checking visible rows with Application.Subtotal(103, ...) means you do not have to use On Error Resume Next to avoid crashes when there is nothing.

BTW, if you were filtering on a single columns (e.g. Range("B1:B" & lastrow).AutoFilter), you would be filtering Field:=1 for column B since it is the only one referenced, not Field:=2.

Edit: forgot Set ws = Nothing. Corrected above.
