bob jones aka: cybertaz say:

As I understand it, BitLocker & BitLocker to Go are Windows-only encryption tools. They are not available for systems running other than specific editions of the Windows OS. It is not supported on Macs, so portable devices running OS X cannot be accessed.

If this is an absolute necessity, AFAIK, you would have to either;

  • Create a specially constructed Boot Camp partition to install an appropriate version of the Windows OS, which is a complex technical process & has consequences to be considered beforehand (see BitLocker and Mac OSX | Apple Support Communities)

, or

  • Use virtualization software [such as Parallels Desktop or VMware Fusion] to create a Windows Virtual Machine on which to install an appropriate version of the Windows OS. Confirm support of the encryption software with the developer before investing in either.

I've also heard of a utility called Dislocker which works with Linux, but attempts to get any conclusive information re OS X have proved futile.
