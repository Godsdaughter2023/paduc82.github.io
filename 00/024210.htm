aaronmooney say:

Concantenate data from different sources

From the table below

Dt En De Bu Ge Dr Cc Ch Mu Ar Group/Gd
B     B B           Dt B_Bu B_Ge B

I need the final cell headed Group/Gd to contain the Heading i/e Dt and the the Grade B below and do this for all entries in this row so that I end up with Dt B_Bu B_Ge B. This denotes the Subject heading and all the respective grades for that pupil. Each piece of data needs to separated with a space or underscore.

How can this be done?

Aaron

tom ogilvy say:

Here is the basic formula you can use.   It is written for the data you show starting in A1.  (Headings in row 1).

=IF(A2<>"",$A$1&" "&A2&"_","")&IF(B2<>"",$B$1&" "&B2&"_","")&IF(C2<>"",$C$1&" "&C2&"_","")&IF(D2<>"",$D$1&" "&D2&"_","")&IF(E2<>"",$E$1&" "&E2&"_","")&IF(F2<>"",$F$1&" "&F2&"_","")&IF(G2<>"",$G$1&" "&G2&"_","")&IF(H2<>"",$H$1&" "&H2&"_","")&IF(I2<>"",I1&" "&I2&"_","")&IF(J2<>"",$J$1&" "&J2&"_","")

this will leave a hanging _ on the last entry.   You can eliminate that with

=IFERROR(SUBSTITUTE(IF(A2<>"",$A$1&" "&A2&"_","")&IF(B2<>"",$B$1&" "&B2&"_","")&IF(C2<>"",$C$1&" "&C2&"_","")&IF(D2<>"",$D$1&" "&D2&"_","")&IF(E2<>"",$E$1&" "&E2&"_","")&IF(F2<>"",$F$1&" "&F2&"_","")&IF(G2<>"",$G$1&" "&G2&"_","")&IF(H2<>"",$H$1&" "&H2&"_","")&IF(I2<>"",I1&" "&I2&"_","")&IF(J2<>"",$J$1&" "&J2&"_",""),"_","",COUNTA(A2:J2)),"")

You would then copy either of the formulas down the column.

Microsoft Security Essentials: "potentially unprotected".

Mircrosoft  security essentials says my computer status says it is potentially unprotected.

Yikes, now what!  I can't seem to click on the right thing to make the 'green' come back on for my computer status. 

stephen boots say:
hello I have c cleaner I think it took over my Microsoft sec settings how do I fix this thanks

Courtesy of forum users le Boule and ahaap:

 

Are you using CCleaner?  Piriform has programmed it for some reason to remove part of the MSE History files and it's deleting the files MSE checks to determine the last time a scan was conducted. 

Open CCleaner Applications Tab, Under Windows uncheck MS Antimalware...that should prevent CCleaner from deleting the MS Antimalware scanning history files.  The actions of CCleaner  should not have any effect on the antimalware protection provided by MSE.

Per suggestion of MSE User MartinHampton you may also need to uncheck MS Management Console.

You may need to run a "Quick Scan" and possibly re-boot the computer to re-establish a scan history. 

 

jimr1 say:

MSE will show "potentially unprotected" if a scan is overdue, if real time protection is turned off, or if definitions are out of date. If you double click the MSE icon in the notification area MSE will open and you will see the action to take to resolve the "potentially unprotected" status.
