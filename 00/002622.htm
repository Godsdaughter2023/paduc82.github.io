hansv mvp say:

The Number Format of a cell only changes the way the cell is displayed on screen and in printouts, it does not change the underlying value.

If you want to convert a date to a formatted value, you can do the following:

Let's say you have dates in column A, starting in cell A1.

In B1, enter the formula =TEXT(A1,"yyyymm")

Fill down as far as needed.

Select the cells with formulas in column B.

Copy, then paste specialwith the Values option.

You can now delete the original date values.

jim gordon mvp say:

Here are some diagnostic troubleshooting steps to try.


First, save a copy if the presentation to work with so you don't introduce further corruption or lose content.


First thing to try - save in a different format. Use File > Save As and from the Format pop-up choose Macro Enabled Presentation (.pptm).


Is it fixed? Good!  Then use File > Save As and change it back to PowerPoint format (.pptx) or just leave it in (.pptm) format.


Not fixed? Bad! There's probably a corrupt object on one of the slides. Use a process of elimination. Open the presentation and delete half the slides.  Save with a new name. Fixed? Then the problem slide is in the other half. Not fixed? The problem slide is in this half. Keep deleting half the slides in the problem half till you narrow it down as to which slide is the bad one. Then go back to the original presentation, delete the bad slide and things should be all better.
