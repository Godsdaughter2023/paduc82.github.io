If the sole reason for uninstalling/reinstalling Outlook is because of issues you're having with your BT account, then that would likely be waste of time serving no purpose.

Information/settings for any email account is stored in the Outlook profile which does not get deleted when you uninstall Outlook/Office which means that when you you go thru the uninstall/reinstall process, the same Outlook profile would be used when Outlook is opened.

To reset any potential Office applicable issues, would suggest doing a full online repair of Office 365 (go to Control Panel --> Programs --> Change -- > <Full Online Repair>

In terms of starting over with configuring the email account to eliminate the possibility of the Outlook profile being corrupt, that would be done by creating a <New> Outlook profile by opening the Control Panel <Mail> app (with Outlook closed)

--> <Show Profiles>
--> <Add> --> Give the profile a name
--> Configure the email account(s)
--> when done, be sure to set the newly created Outlook profile as the default for the next time Outlook is opened otherwise the same existing profile will be used
