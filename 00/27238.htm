adrianmhsmith say:

You should not be using the Bluetooth GPS as this is for devices without integrated GPS. Although it should be possible to connect a separate GPS receiving device and connect it with Bluetooth to your N96, unnecessary.

I do not think any of the positioning methods cost you any money. The GPS is independent of the phone networks and you can even get a GPS position when there is no phone signal. However with Assisted GPS, the GPRS network transmitters are used to give an approximate position before the device is able to get an accurate GPS fix.

What costs you money is the data to download the maps as you use them. If you have a N96 it is passable to download and install the European Map (UK, France, Germany, Poland, Latvia, Italy, Spain) to your handset. It is about 1.2 Gb and takes about 1 hour to install. But it is worth it if you move around Europe a lot and wish to use Nokia Maps without incurring expensive data charges.

You can try below mentioned steps:

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base: 322756  How to back up and restore the registry in Windows.

1.       Exit Excel

2.       Click Start-> run-> type REGEDIT.

3.       Locate the registry key HKEY _CURRENT_USER\SOFTWARE\MICROSOFT\OFFICE\14.0\EXCEL\ Security\FileOpenBlock           

 

4.       If the FileOpenBlock subkey does not exist, you must create it.

5.       Select the Security subkey.

6.       On the Edit menu, point to New, and then click Key.

7.       Type FileOpenBlock, and then press ENTER.

8.       After you select the subkey that is specified in step 3, locate the DWORD value.

 

Note :If this value does not exist, you must create it. To do this, follow these steps:

 

1.       On the Edit menu, point to New, and then click DWORD Value.

When you change the value of "FilesBeforeVersion" from 1 to 0, you can open files that were saved in earlier file formats.

On the File menu, click Exit to exit Registry Editor.
