rick rothstein, mvp - excel say:

If you are interested in a VB macro solution (change the two Const assignment statements to reflect your actual column and start row for the area you want to fill in)...

Sub FillInTheBlanks()
  Dim Area As Range, LastRow As Long, CellRange As Range
  Const ColLetter As String = "A:B"
  Const DataStartRow As Long = 2
  LastRow = Cells.Find(What:="*", SearchOrder:=xlRows, SearchDirection:=xlPrevious, LookIn:=xlFormulas).Row
  Set CellRange = Intersect(Rows(DataStartRow & ":" & LastRow), Columns(ColLetter))
  On Error Resume Next
  For Each Area In CellRange.SpecialCells(xlCellTypeBlanks).Areas
    Area.Value = Area.Rows(1).Offset(-1).Value
  Next
End Sub

To install the macro, press Alt+F11 to go into the VB editor, click Insert/Module from its menu bar and then copy/paste the above code into the code window that opened up. That's it. You can invoke this macro from the worksheet by press Alt+F8, selecting FillInTheBlanks from the list and click the Run button.
