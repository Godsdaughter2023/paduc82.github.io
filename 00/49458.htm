rohit siddegowda say:

Hi Isaac,

 

Which Microsoft Account do you usually use to login to the computer?

 

This issue might occur due to two Microsoft Accounts with the same ID on the computer.

 

First, I would suggest you to check the how many accounts are present in the User Accounts:

  1. Go to the Start screen.
  2. Type “User accounts” without quotes and select it from the list of results.
  3. Click on Manage another account.
  4. Check the listed accounts.

 

If you are not using one Microsoft Account of them, you may try to take a backup of the data and remove the account. Refer to the section “Can I make changes to someone else's user account?” in the following article to remove an account:

User accounts: frequently asked questions

http://windows.microsoft.com/en-au/windows/user-accounts-faq#1TC=windows-8
