hansv mvp say:

Assuming that column T is not used, I'd enter a column heading in T1, e.g. No Appointments, then enter the following formula in T2:

 

=COUNTA(F2:S2)=0

 

and fill down to the end of the used rows. You can now apply AutoFilter to the range (or to column T only), and easily filter the rows that have TRUE in column T.

For conditional formatting:

Select the range that you want to format. I'll assume that it starts in row 2.

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Click 'Use a formula to determine which cells to format'.

Enter the formula

 

=COUNTA($F2:$S2)=0

 

Click Format... and specify the desired formatting.

Click OK twice.

b) select <Tools> --> Options
c) set the desired contact folder to open by default

jay freedman say:
The syntax that works is

{ IF { =B8 }>{ =B7 } “pass” “fail” }

The two expressions in the condition are both fields, created by selecting the =B8 and pressing Ctrl+F9 and then repeating for the =B7.
