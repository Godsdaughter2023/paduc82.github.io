Display Date field as the number of week - Microsoft Community

golda lorrell bishop say:

Display Date field as the number of week

I need to create a report that is grouped upon what week of the year it is. For example, January 1st -7th = Week 1, and so on for all 52 weeks. I know you can specify how data displays by using custom formatting in your tables, but I cannot find a preexisting format that does this. Please help!

ken sheridan say: You can return the week number in a computed column in a query on which the report is based by means of the format function, e.g. Format(#2017-06-13#,"ww") will return 24.  For more details see:

https://msdn.microsoft.com/en-us/library/office/gg251755.aspx

As you'll see the function allows you to specify which day is treated as the first day of the week, the default being Sunday, and the basis on which the first week of the year is determined, the default being the week in which January 1 occurs.

You will, therefore, only be able to use January 1 as the first day of week 1 if the dates are all within the same calendar year, and the function call will of course then be specific to that year.

tom van stiphout say:

> January 1st -7th = Week 1

This is HIGHLY unusual and will require you to write your own logic.

The 3 common ones are:

1. Week 1 is the week with Jan-1

2. Week 1 is the first week of the year with at least 4 days in the new year

3. Week 1 is the first week with 7 days in the new year.

These 3 are supported in VBA: https://support.office.com/en-US/article/Format-Function-6F29D87B-8761-408D-81D3-63B9CD842530

ken sheridan say: If you really want to partition the year into sets of 7 days of which the first begins January 1, then you should be able to use a combination of the Format function to return the day-of-the-year, and integer division by 7:

Format(TheDate-IIf(Month(TheDate)=1 And Day(TheDate) = 1,0,1),"y")\7+1

The IIF function is called so that 1 is not subtracted from 1 January, as this would return 31 December of the previous year of course, and consequently a week number of 52 or 53.  

The final week of the year will be one of less than 7 days of course.

You can see how it would work if a few date literals are used:

TheDate = #2018-01-07#
? Format(TheDate-IIf(Month(TheDate)=1 And Day(TheDate) = 1,0,1),"y")\7+1
 1

whereas:

TheDate = #2018-01-08#
? Format(TheDate-IIf(Month(TheDate)=1 And Day(TheDate) = 1,0,1),"y")\7+1
 2

But as Tom says, this sort of week numbering would be very unusual.
