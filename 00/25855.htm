Hello mythilik,

When the contents are not properly saved in the NormalEmail.dotm template before Outlook closes, this concern usually happens. So, the practice to prevent this issue from happening is to let Outlook shut down properly before shutting the whole operating system (Windows). To help solve this issue, we suggest following the steps below:

  1. Close all of MS Office.
  2. Use Task Manager to be sure that the Outlook process is closed.
  3. In the folder C:\Users\%username%\AppData\Roaming\Microsoft\Templates 
    Rename NormalEmail.dotm to NormalEmail.OLDdotm
    Rename Normal.dotm to Normal.OLDdotm
  4. Copy NormalEmail.dotm and Normal.dotm from a known good PC; paste into that folder
  5. Start MS Office.

If you were able to restore the Quick Parts, you can back up your files, by save it to your desired location.

robert19 say:

Try the following wildcard Find/Replace:

Find What:

([0-9]{1,2})([^034“”])

Replace With:

\1

Note that the curly quotation marks can be copy/pasted into the Find What box. But more easily, you can place the insertion cursor within the Find What box and press “Alt+8220”, then “Alt+8221”. “Alt+8220” and “Alt+8221” will insert true curly quotation marks.
