The real problem is that Ovi music is incompatible with Nokia phones (can't connect via Bluetooth, won't recognize N900 via USB etc.)

  • Open Settings > Settings... & select the <phone name> tab (far right)

andrejgieci say: What do you mean by resetting App? uninstall and install? or any other way?

ruwim.b say:

You should see 3 files in the asyncdb folder:

 

cache_db.db

cache_db.db-shm

cache_db.db-wal

 

The cache_db.db is the actual SQLite database file keeping the history of all emoticons and mojis used by Skype.

 

The .db-shm and .db-wal files are temporary SQLite files, which should only exist when the main database file (cache_db.db) is in use, i.e when the list of emoticons and mojis is updated or new emoticons/mojis are opened in Skype.

 

Under normal conditions, the size of the . cache_db.db-wal should not be larger than few MB. The size (13.9 GB) you are seeing in your Skype folder is absolutely abnormal. It should be safe to delete this file.

 

Sign out of Skype. This will close the Skype application. Delete both the cache_db.db-shm and cache_db.db-wal files.

 

Restart Skype application. Watch now again the size of the cache_db.db-wal file.
