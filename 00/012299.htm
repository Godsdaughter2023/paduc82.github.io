Leapcheang

*** Email address is removed for privacy ***

jay freedman say:

The Combine and Compare functions are very similar. If you have only two versions of a document, the results of both functions are identical.

The difference is that the Combine function expects you to have revisions in two or more separate copies of the document, as you would if you sent each reviewer a separate copy of the original instead of passing one copy from reviewer to reviewer. By running the Combine function multiple times, each time adding another review copy to the result of the previous run, you build up a single file with all the revisions.

Also note that the Combine Documents dialog box lets you specify a name to be attached to "unlabeled" changes on each side, whereas the Compare Documents dialog box allows only a name to be attached to all changes in the revised document.

hansv mvp say:

Try

Sub Quadruple()
  Dim r As Long
  Dim m As Long
  m = ActiveSheet.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
  For r = m To 1 Step -1
    Range(r & ":" & r).Copy Destination:=Range((4 * r - 3) & ":" & 4 * r)
  Next r
End Sub

If you have formulas, you may want to check whether they work as intended after the macro.

rick rothstein, mvp - excel say:

Okay, here is one more macro for you to consider...

Sub QuadrupleRows()
  Dim X As Long
  For X = Cells.Find(What:="*", SearchOrder:=xlRows, SearchDirection:=xlPrevious, LookIn:=xlFormulas).Row To 2 Step -1
    Rows(X).Resize(3).Insert
    Rows(X).Offset(3).Copy Rows(X).Resize(4)
  Next
End Sub

This code assumes the first row to be quadrupled is Row 2; that is what the 2 at the end of the For statement is for, but you can change it if you need to.

There is a simple non-macro way of doing this. First of all, add a simple sequence (1, 2, 3, etc) to the next blank column - put 1 in row 1 and 2 in row 2, select both those cells and then drag down the fill handle to row 500 and release the mouse button. This will have filled 1 to 500 down that column.

Then highlight rows 1 to 500 and click <copy>, then <end> followed by <down-arrow> will take you to the next empty cell and <paste> the data starting in row 501. Repeat this twice more so that you have data copied into row 1001 onwards and 1501 onwards (assuming you have exactly 500 rows).

Now you can highlight all that data and sort using the helper column as key field, and you should end up with what you wanted. Delete the helper column when you've finished.

Hope this helps.
