jllatham say:

What is your goal over in Word?  Do you just want a 'picture' of the part of the spreadsheet in the Excel document?  Or are you trying to get it into Word as a table?

 

A simple "copy" in Excel with a "paste" into Word normally ends up with a table in the Word document.  You're going to have to play around some with Word's page size and layout to be able to see and work with wide tables created this way.  You can go up to 8.5" x 22" and set it up in landscape to get a pretty wide sheet of paper (good luck on finding a printer that will handle it).

 

You can also copy the cells from Excel and then go over to the [Home] tab in Word and choose Paste and then Paste Special.  Several options are available, such as just pasting a link to the file (when clicked, if the file location is accessible, it should open in Excel) or you can paste as a picture that you can then resize.  I find the best appearance is to paste as an Enhanced Metafile.  Those look pretty good - but of course it's just a graphic, not an actual table.

Blocking access to OneDrive (MySite) for a user - Microsoft Community

manuela rusu-hack say:

Blocking access to OneDrive (MySite) for a user

Technical Level: Intermediate

Description:

In this article, you can find information, how to prevent the access to OneDrive for users. This could be useful, if an employee is leaving the company and his access to the data should be blocked. Or if you just want to prevent users of using the OneDrive Site. To achieve this, you will need to install and use the SharePoint Online Management Shell and run some commands, as described in this article.

1. Download and Install Sharepoint Management Shell

 https://www.microsoft.com/en-us/download/details.aspx?id=35588

2. Then run

Connect-sposervice -url https://<Domain name>-admin.sharepoint.com -credential  *** Email address is removed for privacy ***

Set-SPOSite -Identity https://<domain>-my.sharepoint.com/personal/user_domain_onmicrosoft_com -LockState NoAccess

If you want to activate access again then run:

Set-SPOSite -Identity https://[contoso]-my.sharepoint.com/personal/[user]_[domain]_[extension] -LockState UnLock

For additional information on the cmdlets please read  Set-SPOSite article.

john korchok say:

The first 2 slides may use a different layout. Select one of the first 2 slides, then choose View>Slide Master again. It will take you to the slide layout responsible for the selected slide. Adjust the footer on the slide layout.

You hide the footer using Insert>Header & Footer. Uncheck Footer. To turn off the footer on just the selected slide, click on Apply. To turn off the footer for all slides, click on Apply yo All.

If you are trying to include extra information in a printout, but not the show, take a look at Print>Settings and change Full Page Slides to Notes Page. Then your printouts will include Notes you can type in the pane below the slide in the main edit window:
