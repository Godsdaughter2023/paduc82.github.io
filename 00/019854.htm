bob jones aka: cybertaz say:

First, please understand that the AutoRecovery feature does not create backup copies of your files... Nor does it save changes into the working file itself [automatically or otherwise]. AutoRecovery is a 'safety net' that attempts to help as best it can if you encounter a critical failure while a file is in use. Once you Save & Close the file its AutoRecovery Data file is instantly expunged.

If you do not have a backup copy [Do you use Time Machine?] there is no way to recover content that has been deleted once the file has been closed.

Automatically update footer when file path and name change - Microsoft Community

rtpftguy say:

Automatically update footer when file path and name change

Is there a way to construct a header or footer so the path and/or filename change when Save As is used and they change?
jay freedman say:

The {Filename \p} field does not automatically update. The article http://support.microsoft.com/?kbid=832897 suggests a macro that updates the field (and any others) whenever you reopen the document, but it doesn't do anything when you do a Save As and then continue working without closing the document.

If you use the following two macros instead, the fields will be updated immediately upon saving the document to a new name or location. In addition, you can assign a Quick Access Toolbar button and/or a keyboard shortcut to the UpdateAllFields macro to update at any time, not necessarily while doing a Save As.

Sub UpdateAllFields()
    Dim currentView As WdViewType
    currentView = ActiveWindow.View
    Options.UpdateFieldsAtPrint = True
    ActiveWindow.View = wdPrintPreview
    ActiveWindow.View = currentView
End Sub

Sub FileSaveAs()
    Dim dlg As Dialog
    Set dlg = Dialogs(wdDialogFileSaveAs)
    dlg.Show
    UpdateAllFields
End Sub

If you use Antidote, Webex, Silverlight, EndNote, Zotero, Mendeley, Suitcase, 3rd party font management tools or other add-ins there is a potential for crashes and unwanted misbehavior unless you also update your add-ins to the latest update. Install updates for these programs if you have them.
