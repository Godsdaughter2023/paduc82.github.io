bernie deitrick say:

When code resides in a specific sheet's codemodule, any non-fully-qualified range defaults to the parent sheet.

So, when you use

Worksheets("Ledger2285").Range( _
            Cells(lastLedgerRow, colLedgerDate), _
            Cells(lastLedgerRow, colLedgerDate))

and the other sheet is active, the code cannot create a range on Ledger2285 based on cells on sheet BankData.

You should change all those references to just cell objects on the specific sheets

Worksheets("Ledger2285").Cells(lastLedgerRow, colLedgerDate)

So your code should be:

dataBankDate = DateSerial( _
      Year(Worksheets("BankData").Cells(lastBankRow, colBankDate).Value), _
      Month(Worksheets("BankData").Cells(lastBankRow, colBankDate).Value), _
      Day(Worksheets("BankData").Cells(lastBankRow, colBankDate).Value))
      Year(Worksheets("Ledger2285").Cells(lastLedgerRow, colLedgerDate).Value), _
      Month(Worksheets("Ledger2285").Cells(lastLedgerRow, colLedgerDate).Value), _
      Day(Worksheets("Ledger2285").Cells(lastLedgerRow, colLedgerDate).Value))
