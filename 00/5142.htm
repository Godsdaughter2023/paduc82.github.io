Hi John,

To setup a registered destination, you can follow the steps below:

  1. Click the gear on the upper right corner of the site and select Site settings.
  2. Click Manage site features under Site Actions.
  3. Activate the Content Organizer feature. You will get the "Drop off Library" as well as 2 new options in the Site Settings, Content Organizer Rules and Content Organizer Settings.
  4. Select Content Organizer Settings in the Site Administration section.
  5. Scroll down to the bottom and copy the Web service URL.
  6. Go to your SharePoint admin center and select records management.
  7. In the Connection Settings section, type the Display name and paste the Web service URL to the Send to URL.
  8. Select the Send to action and add the Explanation.
  9. Click Add Connection then the send to another location function should work.

    devlin sco say:

    Hello,

    We understand that you need more work space to perform your tasks effectively. Most of your inquiry can be done by configuring the View Settings.

    Limit the message list entries to a single line.

    1. Go to View tab.
    2. Click View Settings > Other Settings.
    3. Clear the Use compact layout in widths smaller than 125 characters check box.
    4. Click OK and then OK again to save the changes.
    5. Go back to View tab.
    6. Set the value for Message Preview to Off.
    7. Select whether you like to set it for all mailboxes or a single folder.

    Disable contact photographs in emails.

    1. Click File > Options > People.
    2. Clear the Show user photographs when available (requires restarting Outlook) check box under Online status and photographs.
    3. Click OK to save the changes.
    4. Restart Outlook.

    Apparently, there is no option to reduce the space between folders in the folder list. We would like to clarify what you meant by your last question. What exactly do you mean by larger font size without more white space?
