doug robbins - mvp office apps & services (word) say:
For the page numbering in Section 2, use { page \* roman } and format the page numbering to Start at 1 rather than continue from the previous section
 
For the page numbering in the Third Section and the other Sections forming the body of the document, first, insert a bookmark with the name of FirstPage at the beginning of the third section and insert a bookmark with the name of LastPage on the last page of the body of the document and then use the following field construction
 
{ page } of { = { pageref LastPage } - { pageref FirstPage } + 1 }
 
You will of course need to unlink the footer in the third section from that in the first and second sections and also unlink the footer in the section containing the appendices from that in the previous section.
 
You must use Ctrl+F9 to insert each pair of field delimiters { } and you use Alt+F9 to toggle off their display.

karl timmermans [outlook mvp 2012-2018] say:

The things you're running into (at least from what's described) have to do with having 2 Outlook versions installed "side-by-side" - something that can be done using any prior version of Outlook and the "C2R" (click-to-run) version of Outlook '2013.

 

In this case, the error is specific to when 2 versions are installed and moving back and forth between the two.

 

As for "only being able to have one Outlook program at a time" - could easily be caused by trying to open one version before the other has fully completed it's shutdown process.

 

Re: should you uninstall anything to do with O'2013?

 

Something that only you can decide on in terms of what you want to work on an on-going basis. The problem that was introduced ("Not implemented") certainly does make moving between two different versions a pain until a fix is released for it. "Not using" two versions of Outlook would in essence be the same as uninstalling with the difference being that in the case of '2013 - you can't pick and choose what you want to uninstall - it's an "all or nothing" process.
