·         Click Run as Administrator

·         Click Tools

·         Click Internet Options

·         Click the Security tab

    john korchok say:

    It's not uncommon for margins to shift slightly between those versions, but it's usually a difference of only about 1/100".

    A more noticeable difference is visible in documents that include tables. In 2007-2010, the left table cell margin is subtracted from the margin, so that text inside a table is left aligned with text outside the table. In 2013-2016,  the left table cell margin is not subtracted, so the left border of the tables is aligned with text outside the table. This shift may make it appear that the margin has shifted to the left, when it is actually the tables that have moved to the right.

    If you determine that this is what you're seeing, the workaround is to set the left cell margin of all tables to 0, then use indented paragraph formatting to make the space to the left of the table text. This displays the same in all versions of Word.
