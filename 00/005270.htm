The effect is of a hidden page break at the bottom of each page.  But - I used to be able to get Word to show all the markup symbols, like page breaks, paragraph breaks, etc. but that disappeared some ago, maybe the victim of an update?

What's going on?

jay freedman say:

In a letter-type merge, each record ends with a section break. Normally the break is of the Next Page type, and each section restarts page numbering at 1.

Your symptoms imply that the section breaks have been converted to the Odd Page or Even Page type. That would generate a blank page at the end of each section, which Word won't display but will send to the printer.

To try to fix it, click the dialog launcher arrow at the bottom right corner of the Page Setup group on the Page Layout tab of the ribbon. Go to the Layout page of the dialog, change the Section Start dropdown to New Page, and change the Apply To dropdown to Whole Document. Click OK.

That may still print a single blank page at the end of the last section; if so,  you can click in the blank page, go to the Layout dialog, and choose the Continuous start type and apply to This Section.

ossiemac say:

What event are you using to run the code? The following works with the change event.

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Range("D31")) Is Nothing Then
       Rows("32:32").Hidden = Target <> "LESS THAN 1 YEAR"
    End If

    If Not Intersect(Target, Range("D38")) Is Nothing Then
        Rows("39:39").Hidden = Target = "NONE" Or Target = "DEALER"
    End If
End Sub

Note that equals and not equal are case sensitive. If the Target values are not all upper case then the above comparisons will not return True. If this can occur then use the following to convert target to upper case for the purpose of comparison. Note that the actual value in the worksheet does not get converted; just converted within the code for purpose of comparison.

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Range("D31")) Is Nothing Then
       Rows("32:32").Hidden = UCase(Target) <> "LESS THAN 1 YEAR"
    End If

    If Not Intersect(Target, Range("D38")) Is Nothing Then
        Rows("39:39").Hidden = UCase(Target) = "NONE" Or UCase(Target) = "DEALER"
    End If
End Sub
