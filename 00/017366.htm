Hi John:

"No" :-)  Sorry, Office 2016 uses the system recent files list, maintained automatically by Mac OS.  So you can't jam entries into it: it's dynamically generated by the system.

Nor can you add documents to the Quick Access Toolbar, like you used to be able to; which is equally annoying.

What you CAN do is use the File menu.  Choose the "Open Recent" command.  If you see nothing listed, choose "More" to force it to rebuild the list.

You can also place aliases (shortcuts) to your favourite documents in the Dock, where you can get 'em with a single click whether Word is running or not. For more advanced use, look up the Apple Help about Smart Folders. A "Smart Folder" is actually a stored 'search', and the mechanism is very powerful. Apple has already created one for you, named "All my files", which will appear in your Finder sidebar (and you can add it to the Dock).  This will show you a date-sorted list of every file on that computer, most-recently opened at the top.

So: While I am as grumpy as you are about the 'loss' of features we used to have, there is really no point in Microsoft replicating functions that are already built-in to Apple OS.

Hope this helps

jay freedman say:
The table is the right solution for what you're doing.

To align the number in the right cell vertically with the center of the equation, click in the number's cell, go to the Alignment group of the Table Tools | Layout tab of the ribbon, and choose one of the pictures in the middle row of the nine-box display. (This display is also available on the right-click menu when you point to Cell Alignment.)

If you have a lot of equations to write this way, make a table that contains an empty equation box in the left cell and a vertically centered number in the right cell. Select the whole table and press Alt+F3 to make an AutoText entry containing the table; give it a name and store it in the Normal.dotm template. After that, you can insert the table at any place in the document by typing its name and pressing Enter when prompted (or pressing F3 if there is no prompt). Then all you need to do is type the equation and update the number.
