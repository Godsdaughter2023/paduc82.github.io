wunders say: The usual way to verify a publisher is by using a digital signature.  Your system may not be able to verify this digital signature because it can't trace this signature through a path to a "Trusted Source".  Successfully tracing signatures up the tree will end up at one of your trusted Root Certificates installed on your machine.  Unlike later versions of Windows, Windows XP does not update its root certificates very well.  Your application may be using a newer signature path that you do not have installed on your machine.  There is no guarantee that this will solve your problem but you seem desperate and it can never hurt to update your Root Certificates on your copy of Windows.

To update your Root Certificates, go to the following KB article:
   "Windows Root Certificate Program Members"
  < http://support.microsoft.com/kb/931125 >
 
Skip down to the paragraph titled "Root Update Package (Intended for Windows XP Only)".  Click on the "Update for Root Certificates..." link (do not use the Update Catalog option).  This will take you to a page where you can click to download an update package.  Download this package and double-click on it to install.

Reboot, and if you're lucky, this may solve your problem.

Q. If my customer asks me to upgrade a PC with new hardware components, when is a new operating system needed? When would the PC be considered "new"?

A. Generally, an end user can upgrade or replace all of the hardware components on a computer—except the motherboard—and still retain the licence for the original Microsoft OEM operating system software. If the motherboard is upgraded or replaced for reasons other than a defect, then a new computer has been created. Microsoft OEM operating system software cannot be transferred to the new computer, and the licence of new operating system software is required.

If the motherboard is replaced because it is defective, you do not need to acquire a new operating system license for the PC as long as the replacement motherboard is the same make/model or the same manufacturer's replacement/equivalent, as defined by the manufacturer's warranty.
