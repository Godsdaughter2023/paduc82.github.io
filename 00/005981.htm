You will have to decide which VBA you are going to write - Project VBA or Excel VBA: they are significantly different

The Datedifference function is Project VBA; but Project does not at all know things like sheets nor Cells: typically Excel objects. On the other hand, Ihave written something like 20000 lines of code in Project VBA but never seen Networkdays, so I could not react to your original question. What are you trying to achieve?

Project VBA however knows Hoursperday

nbrjs=nbrminutes/(60*Hoursperday)
