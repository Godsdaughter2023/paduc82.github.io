burrwalnut say:

Very few keys in HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall  have subkeys in the left pane. If you remove the key in the left pane, the item will certainly disappear from ‘Add or Remove’ (as you call it), but you will have to make a note of the value of UninstallString in the right pane. This is the full path to the program that you will need to run should you wish to uninstall it, i.e. as the entry doesn’t appear in Control Panel > All Control Panel Items > Programs and Features (Add or Remove) you will have to navigate to it and double-click the program name.

 

An example of this would be an entry for Foxit Reader in the left pane and UninstallString in the right pane containing C:\Program Files\Foxit Software\Foxit Reader\Uninstall.exe.
