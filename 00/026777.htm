Solved:For anyone else that has this issue:

When you go into the Address Book in Outlook Desktop there is a drop down that asks what Address book you want to use. If this is not set to Contacts you may get another list. To change this default list use the 'Tools" option.

I had thought the contact list would autocomplete as I typed the contact email in but it doesn't.

The folder initially set as the default folder to open would be the contact folder associated with the first email account configured in Outlook which in turn would be the default contact folder for the profile.

Adding other accounts which have their own contact folders (i.e. Exchange) would require that either (a) the OAB be changed to open a different contact folder by default or (b) you need to select whatever contact folder you want to select contacts from.
