    time lady say:
    Every Windows 10 update requires a certain amount of storage space on your device to help ensure the installation goes smoothly and that it runs well.

    Before downloading the update, Windows runs a scan to make sure there’s enough space. If there isn’t, you’ll see a message asking you to free up some.
    External or internal?
    If your device doesn’t have enough space for the update, Windows will ask you to connect an external drive to extend the space on your hard drive temporarily while the update is being installed. Or, you can free up space on your device by deleting nonessential files if you don’t have an external storage device or don’t want to use one.

    The More space needed for update window only shows one option at a time for freeing up space, but you can select either I want to use external storage instead or I don’t have an external storage at the lower-left side of the window.
    Updating with external storage
    If your device has less than 50 GB total capacity on its hard drive, updating is easiest using an external storage device.

    Step 1: Free up space

    Although you’re using an external storage device, your hard drive still requires space to install the update. Here’s how to make some:

    In More space needed for update, select Free up space.

    Clear space with an external storage device.
    This opens Storage in Settings, which shows how much space is available on your device. Select your C: drive to see what’s using the most space and to clear it by deleting nonessential files. (For tips, see Free up drive space in Windows 10).
    Return to More space needed for update and select the refresh icon next to the Free up space button. If you deleted enough files, a green check mark will appear next to the hard drive Internal hard drive icon icon.
    Step 2: Connect an external storage device

    Connect your external storage device and wait until you see it in the drop-down list under Step 2 (you might need to select the refresh ﻿ icon). If the external storage device has enough free space, a green check mark will appear next to the storage drive icon.
    Select Continue to finish installing the update.
    Updating with your hard drive
    If you don’t want to use an external storage device, you’ll need to free up more space on your hard drive.

    In More space needed for update, select Free up space.

    Clear space from your hard drive
    This opens Storage under Settings, which shows how much space is available on your device. Select your C: drive to see what’s using the most space and to delete nonessential files. (For tips, see Free up drive space in Windows 10.)
    Return to More space needed for update and select the refresh icon next to the Free up space button. If you deleted enough files, a green check mark will appear next to the hard drive Internal hard drive icon icon.
    Select Continue to finish installing the update.

    Hi Keith,

    It's possible that the Recycle Bin is already corrupted. To troubleshoot your concern, kindly follow these steps:

    Method 1: Perform a clean boot. It helps eliminate software conflicts. Here's the link.

    Method 2: Use the System File Checker and the inbox Deployment Image Servicing and Management (DISM) tool. This will help you scan for corruptions in Windows system files and restore corrupted files. To run it:

    • Select Start Windows logo Start button, and in the search box, enter Command Prompt. Press and hold (or right-click) Command Prompt (Desktop app) from the search results and select Run as administrator.
    • Enter sfc /scannow (note the space between "sfc" and "/").
    • Enter DISM.exe /Online /Cleanup-image /Restorehealth (note the spaces before each "/").

    rd /s /q C:\$Recycle.bin

    After that, please restart your computer. This command will reset the $Recycle.bin folder which is present on the C drive. You will have to do this for every partition on your hard disk, by replacing C with the drive letter/s.

    Hi Oli1411,

    Greetings from Skype!

    There is one way you can recover your old Skype account. Old desktop versions of skype (7xx and below) logs the user name/s that you have used. If you are still using the same computer, try to do the steps below:

    For Windows OS.

    1. Quit Skype.

    2. Press windows key + R

    3. Type %appdata%/Skype

    4. Inside your Skype folder. Check the user name/s that you used to signed in to Skype.

    For MAC:

    1. Quit Skype.

    2. Click on GO (Located on your apple menu bar)

    3. Click on Go to folder.

    4. Type ~/library/application support/Skype

    5. Inside your Skype folder. Check the user name/s that you used to signed in to Skype.

    Should you have questions, please don't hesitate to reply!
