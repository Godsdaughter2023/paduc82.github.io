hansv mvp say:

ColorIndex is one of the 56 colors of the "old" Excel color palette. It may not work if you work with cells that have a color that is not in that palette. You could try

Function SumByColor(CellColor As Range, rRange As Range)
    Dim cSum As Double
    Dim Col As Long
    Dim cl As Range
    Col = CellColor.Interior.Color
    For Each cl In rRange
        If cl.Interior.Color = Col Then
            cSum = WorksheetFunction.Sum(cl, cSum)
        End If
    Next cl
    SumByColor = cSum
End Function

a. user say:

Hello,

If you are using the same Microsoft account from your old phone, the text messages should sync when you sign in to your new phone. Since you mentioned that you created a backup, we suggest that you check if you turn on the sync message on your phone by going to Messaging > More ... > Settings > Turn on Sync messages between devices. Make sure you are connected to a stable connection when syncing it.

With regard to contacts+message backup app, you need to install the app first on the old phone and then create a backup of the text messages (in XML format) that you will restore to the new phone.

You can only manually sync the text messages, email, calendar and contacts on your phone. If you are referring to an option that will sync it all, then it is currently unavailable. We encourage sending a suggestion to the Feedback Hub app so we can consider it on the next update. To send a feedback on your phone, you can follow the steps below:

  1. Go to All apps, search for the Feedback Hub app.
  2. Open the Menu icon at the top left side.
  3. Select Feedback, then tap the Add button (+) at the top right side.
  4. Select Suggestion and provide the details needed in the box.
  5. Select the right category and then submit.

Let us know if you have any questions.
