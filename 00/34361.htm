andre da costa say:

Unfortunately, clean install is your only option, you can't downgrade from Pro to Home. Changing the key will not work. You could try the following work around then test your luck:

  1. Open Registry Editor(WIN + R, type regedit, hit Enter)

  2. Browse to key HKEY_Local Machine > Software > Microsoft > Windows NT > CurrentVersion

  3. Change EditionID to Home (double click EditionID, change value, click OK). In your case it should at the moment show Pro

  4. Change ProductName to Windows 10 Home. In your case it should at the moment show Windows 10 Professional

  5. Browse to key HKEY_Local Machine > Software > Wow6432Node > Microsoft > Windows NT > CurrentVersion, change the same two values as in steps 3 and 4 above

  6. Close the Registry Editor

  7. Insert the Windows 10 Homeinstallation media, do an in-place upgrade by launching the setup from desktop (not booting with Windows 10 Home install media)

  8. When asked, enter your validWindows 10 Home product key and select what to keep(settings, personal files and apps, only personal files or nothing)

  9. Windows 10 Home in-place upgrade will be done now.

alanphillips2610 say:

How do I transfer email from Windows Mail to Outlook Express?

My office computer runs Windows Mail, my home computer runs Outlook Express. How do I transfer email from Windows Mail to Outlook Express?

burrwalnut say:

1. To copy from Vista’s Windows Mail to XP’s Outlook Express, navigate to the normal location of the Windows Mail boxes. If you haven’t moved it to a different location, it is the hidden folder C:\Users\username\AppData\Local\Microsoft\Windows Mail\Local Folders then Inbox, Sent Items, etc.

Copy the .eml files (not other files such as winmail.fol) to an external device (or use a network).

2. Create a folder in Outlook Express (File > New > Folder named, say, Old Inbox), which will initially receive the copied files. Close Outlook Express. Now open both Outlook Express and the folder containing the .eml files.

Select (highlight) all the .eml files and drag them (strangely, Copy & Paste will not work) to Outlook Express, either to the new folder name in the left pane or to the folder contents in the right pane. Now open Old Inbox and drag the contents to your usual Inbox.
