red hoffman say:

hansv mvp say:

Select Insert | Symbol...

Select "(Normal Text)" from the "Font" dropdown.

Select "Unicode (hex)" from the "from" dropdown.

Select "Number Forms" from the Subset dropdown.

You should see some additional fractions. If not, select "Arial Unicode MS" from the "Font" dropdown.

Select one of them, e.g. ⅓ and click AutoCorrect...

Type 1/3 and click Add.

Repeat for the others.

yves dhondt say:

The fractions you are looking for are located in the unicode range 2150-215E (seehttp://unicode.org/charts/PDF/U2150.pdf on the second page).

You can use the Symbol dialog to look them up in Word. It is probably easiest if you copy them from there and then paste them in your autocorrect list in the same way autocorrect entries for the other fractions are made.

Additional suggestions:


1. Open Microsoft Word. Use File > Open and change the format pop-up to Recover Text from Any File. Then open the file. Any text that remains will be recovered.


2. Try opening the file in TextEdit. If you can get your file open in TextEdit use File > Save As to save a new version that might work OK.


3. Try opening the file in SkyDrive.  If your file opens in SkyDrive save it with a new file name.
