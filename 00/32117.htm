Windows 10 - Disk Management stalls with message "Connecting to - Microsoft Community

irongamer say:

Windows 10 - Disk Management stalls with message "Connecting to virtual disk service"

Disk Management never loads and stalls at "Connecting to virtual disk service."

This may be an issue if you are running SSD and have a SSD manager that suggests turning off a number of services to improve SSD longevity.  Or if you have disabled services using a tweak site like http://www.blackviper.com/.  Not sure which was the case for my computer but I'm pretty sure it was one of those two. =)

I've seen two solutions outside of the Microsoft Answers so I'm going to post both of them in a single post.  One worked for me but others have had the other issue of the two.

Possible Solution 1

Virtual Disk is disabled.  To enable:

  1. Type services.msc in search or run box and press enter
  2. Scroll down to Virtual Disk Service.  Is it disabled?
  3. Right click > Properties > Startup type > Manual or Automatic
  4. You can also press the start button to start the service.
  5. Check Disk Management to see if your drives show up.

Possible Solution 2 (this worked for me)

Volume Shadow Copy is disabled.  To enable:

  1. Type services.msc in search or run box and press enter
  2. Scroll down to Volume Shadow Copy Service.  Is it disabled?
  3. Right click > Properties > Startup type > Manual or Automatic
  4. You can also press the start button to start the service.  (I set it to manual and then pressed the start button)
  5. Check Disk Management to see if your drives show up.  (Disk Management showed the drives instantly)

Hope this helps someone.

    When you look at newer Operating System products, you'll find that they usually incorporate new hardware drivers, fixes for problems that may have been seen previously and other stuff that improves with time.

    If these are 'orphaned' songs (that are already deleted from hard drive), then resetting the media library is probably the only way to get rid of them. First note that resetting the library will make you lose all current play counts, and possibly ratings too. Then, follow these steps:

    a.       Reboot the PC.

    b.      Go to Windows key+R, type services.msc in the field, and press Enter. In Services, find the Windows Media Player Network Sharing Service. If its status is started, double-click the service and click Stop.

    c.       Go to Start, type %LOCALAPPDATA%\Microsoft in the Search field, and press Enter. This folder will open up in Windows Explorer.

    d.      In this folder, rename the Media Player folder to something like Media Player old. If you can't see the folder, then it's probably hidden. Set Explorer to show hidden files and folders, and to show protected system files.

    e.      Start WMP (Windows Media Player), your library will have been reset and will be rebuilt.
