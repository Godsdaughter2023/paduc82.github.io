This issue isn't really related to TransferText and acExportDelim; it has to do with your use of the FileDialog object to choose the output file.

I believe the FilePicker dialog, which you're selecting by specifying msoFileDialogFilePicker, only allows you to pick an existing file.  If you enter a file name that doesn't exist in the displayed folder, the dialog will give you that message.  I think the FileOpen dialog, selected by specifying msoFileDialogOpen, behaves the same way.

You could specify the Save As dialog by specifying msoFileDialogSaveAs.  However, when you specify that dialog, for some reason you can't change the .Filters property; you would have to remove your line:

    .Filters.Add "CSV", "*.CSV"

because it will raise an error.  You always get "All files", "*.*" with this dialog.

If that's good enough for you, great.  If not, there are ways to call the Windows "Save As" dialog directly, rather than using the Office FileDialog object.

I don't know whether your enclosing If statement, "If DCount("*", "TableName") > 0 Then", is preventing the code from getting any further, but I do know that there is no constant, "msoFileDialogFileSaveAs".  The constant for the "save as" dialog is "msoFileDialogSaveAs".

Rearranging data in an excel cell - Microsoft Community

ronaldwalkersr say:

Rearranging data in an excel cell

How do I create a function or macro that rearranges data in a cell by moving the text to the front of the string?  I have a spread sheet with 380 lines of data and growing. The first column has a character string that needs to be rearrange so I can conduct a search function later.  For example:  2031CF01 needs to be converted to CF203101.

mike h.. say:

Hi,

ALT+F11 to open VB editor, right click this workbook and insert module and paste the code below in on the right.

Back on the worksheet call with this:-

=reorder(A2)

Where A2 contains your string. You can drag the function down.

Function ReOrder(TheString As String)
    Dim Character As String
    Dim x As Long
    Dim Tmp As String
    Dim Tmp1 As String
    Tmp = ""
    For x = 1 To Len(TheString)
        Character = Mid(TheString, x, 1)
        If Asc(Character) >= 48 And _
          Asc(Character) <= 57 Then
            Tmp = Tmp & Character
        Else
            Tmp1 = Tmp1 & Character
        End If
    Next
    ReOrder = Tmp1 & Tmp
End Function
