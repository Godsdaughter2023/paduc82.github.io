hansv mvp say:

The default aggregate function for a field in the Values area of a pivot table is SUM if all values of the field are numbers, and COUNT otherwise. If there is even one non-numeric value - and an empty cell counts as non-numeric - Excel will use COUNT. There is no way to change that.

As far as I know, you cannot change the default name of value fields (Count of ..., Product of ...) either. Sorry!
