To rectify the problem now, display Hidden text so that your XE fields are displayed. Then run a Replace as follows:

  1. Press Ctrl+H to open the Replace dialog.
  2. Click More>> to expand the dialog.
  3. In the "Find what" box, type: ^d XE
  4. In the "Replace with" box, type: ^&
  5. With the insertion point in the "Replace with" box, click Format | Style and choose "Default Paragraph Font." Click OK.
  6. Click Replace All.

Need Calculate Gross from Net - Microsoft Community

mahakumaran say:

Need Calculate Gross from Net

Hi,,

I need formula for found Gross from Net after get Tax also..

    0      - 3000  = 0%

joeu2004 say:

MahaKumaran wrote:

My Net was 25,000 now i wand find my Gross including Tax & Pension Deduction, Tax Pension Deduction = 5%

    0      - 3000  = 0%

3000.1 - 3800  = 25%

3800.1 - 5900  =30%

Gross salary - Pension - Tax = Net

Refer to the image below.  You can also look at the "deductible pension" worksheet in "gross up.xls" (click here) [1].

The inputs that you provide are in the yellow highlighted cells.  The other cells are calcuated.

The formula in E2 is:

=ROUND((B1 + LOOKUP(B1, $H$11:$K$11, $J$3:$J$6)) / (1 - $B$2) / (1 - LOOKUP(B1, $H$11:$K$11, $H$3:$H$6)),2)

That depends on a some of assumptions that you neglected to mention.

1. Is the pension contribution tax-deductible (pre-tax)?  I assume it is.  In that case, the taxable amount is gross minus pension contribution (5% of gross).  Otherwise, the taxable amount is gross.  See the "nondeductible pension" worksheet.

2. Does the tax table reflect flat tax or "marginal" tax brackets?  I assume they are "marginal" tax brackets.  Thus, for example, total tax for a taxable amount over 5900 is 25% of the first 3000 to 3800, plus 30% of the over 5900.  In contrast, a flat tax would be 35% of the entire taxable amount.

However, in a random sampling of net incomes up to 999,999.99, the formula in E2 is off by +/-0.01 due about 6.6% of the time due to rounding.

If that is okay, you only need the calculations in E2:E6.

Otherwise, you also need the calculations in E9:E13, and E9 is the correct gross amount.  The formula in E9 is:

=ROUND(E2-(E6-B1) ,2)

We cannot simply change ROUND to ROUNDUP or ROUNDDOWN in E2 because the off-by-1-cent goes either way half the time.

(In theory, the correction might be more than +/-0.01, and it should be calculated.  But as I noted, the formula in E9 seems to suffice for net amounts up to 999,999.99.)

-----

The basic theory of operation is....

deductible pension:

    net = gross * (1-p) * (1-t)

    gross = net / (1-p) / (1-t)

nondeductible pension:

    net = gross * (1-p-t)

    gross = net / (1-p-t)

where p is the pension deduction rate (5% in B2), and t is the average tax rate.

Those formulas would suffice if the tax table reflects flat tax brackets.

(Let us know if you need help with the Excel formulas in that case.)

But in either case, the problem is:  t depends on gross, which is the amount we want to calculate.

For marginal tax brackets, we can accommodate that variability by calculating the following for each tax bracket:

deductible pension:

    net = gross*(1-p) - baseTax - (gross*(1-p) - minBracket)*bracketRate

    gross = (net + baseTax - minBracket*bracketRate) / (1-p) / (1-bracketRate)

    Example:  gross = (25000 + 830 - 5900*35%) / (1 - 5%) / (1 - 35%)

nondeductible pension:

    net = gross*(1-p) - baseTax - (gross - minBracket)*bracketRate

    gross = (net + baseTax - minBracket*bracketRate) / (1 - p - bracketRate)

    Example:  gross = (25000 + 830 - 5900*35%) / (1 - 5% - 35%)

where baseTax is the sum of the max tax for each previous tax bracket.

I combine baseTax - minBracket*bracketRate and call it "tax offset", which is calculated in J3:J6.  For example (J4):

=ROUND(SUMPRODUCT((G4-$G$3:G3) * $I$3:I3) - G4*H4, 2)

We use the net amount to select taxOffset and bracketRate.  The min net amount for each tax bracket is calculated in H11:K11.  See the "net limits" table in G9:K11.

In all cases (E5, E12 and H10:K10), that tax amount is calculated with a formula of the form (E5, for example):

=-ROUND(SUMPRODUCT((E4>$G$3:$G$6) * (E4-$G$3:$G$6) * $I$3:$I$6),2)

That relies on the min tax brackets in G3:G6 and the differential tax rates in I3:I6, which are simply the difference between adjacent tax bracket rates.  For example (I4):

=ROUND(E2+E3,2)

Excel Drop Down List Automatically Name To Code - Microsoft Community

brittanie spivey-perry say:

Excel Drop Down List Automatically Name To Code

How do I create a drop-down list that will automatically change the name to a code when selected? I am trying to create a drop down list of different facilities; I want the user to be able to view the full name when selecting from the drop down list but I want the selected name to display a code instead of the full name.

bernie deitrick say:

Make a 2-column list of your Names and codes, with names in the first column. Then use Data Validation on your entry cell with the list option, using the values from the first column. Then:

1) Copy this code.
2) Right-Click the sheet tab of interest.
3) Select "View Code"
4) Paste the code into the window that appears.
5) Save the file as a macro-enabled .xlsm file.
6) Make changes the the cells addresses as needed - see comments

Private Sub Worksheet_Change(ByVal Target As Range)

    If Target.Address <> "$A$2" Then Exit Sub   'Address of the cell with the dropdown

    'Turn off events to keep out of loops
    Application.EnableEvents = False

     'Change the "C1:D10" in the next line to the address of your list
    Target.Value = Application.VLookup(Target.Value, Range("C1:D10"), 2, False) 

    'Turn events back on to get ready for the next change
    Application.EnableEvents = True
End Sub

If you have multiple cells to apply this to, it is easy to modify, so let me know....

The address C1:D10 is for the list of values - it can be anywhere on the same sheet, and as large as you need - it just needs to be two columns of values.  If it is on another sheet, then it needs to include the sheet name:

    Target.Value = Application.VLookup(Target.Value, Worksheets("SheetName").Range("C1:D10"), 2, False)

Private Sub Worksheet_Change(ByVal Target As Range)

    If Intersect(Target, Range("A2:A30")) Is Nothing Then Exit Sub   'Addresses of the cells with the dropdown

    Target.Value = Application.VLookup(Target.Value, Range("C1:D10"), 2, False) 

    'Turn events back on to get ready for the next change
    Application.EnableEvents = True
End Sub
