jeanette cunningham say:

Reading back over this thread, I am going to suggest that you stick with a single form - probably what you had at the start of this thread.

Change the record source of the form so that both tables are joined into a query - that allows fields from both tables on the form to be mixed together over the form.

You can make some of the textboxes locked so that users can't change the data in them - that would be your blue controls. The pink controls stay unlocked so the user can edit them.

If you have a strong wish to separate the blue controls to one part of the screen and the pink on another part of the screen - that is easy to do.

The main time to consider using 2 forms side-by-side - blue controls on one form and pink on the other - is when you can't join the 2 tables into a query that allows edits.
