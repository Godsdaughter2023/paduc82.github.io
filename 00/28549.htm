jay freedman say:

You need two fields, one inside the other. The IF field is the "outer" field, and the REF field is the "inner" field. To make this, press Alt+F9 again to expand the field code; select the REF__
part, and press Ctrl+F9. The result should look like this:

{ IF { REF __ } = "male" "He" "She" }

Then press F9 to update the fields.

jim gordon mvp say:

About the only thing that can cause formatting differences between one computer and another (mac vs PC is irrelevant here) is fonts.

Here's what's almost certainly happening. Someone is building a presentation on their computer and it looks just right. They've applied fonts and font formatting options. They save the presentation and pass the PowerPoint file on to someone else.

The second person opens the presentation on a different computer and it looks different because one or more of the fonts that was used originally doesn't exist on the different computer and PowerPoint substitutes with some other font that is available. The substitute font might even have the same name, but could be a different version of the font from what was used to make the presentation file originally.

The presentation will display as originally designed when every computer running the slide show has the same version of the fonts that were used when the presentation was made.
