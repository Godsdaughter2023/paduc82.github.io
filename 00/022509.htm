1. Extend the AD Schema using Exchange 2013 setup command line

2. Install Exchange 2013 and configure as normal (Accepted domains / Virtual Directories / mail flow etc...)

3. Set users UPN in AD to their email address login i.e. *** Email address is removed for privacy ***

   Set-ADUser cloud.only –Replace @{msExchRecipientDisplayType = “-2147483642”}

  Set-ADUser cloud.only –Replace @{msExchRecipientTypeDetails = “2147483648”}

  Set-ADUser cloud.only –Replace @{msExchRemoteRecipientType = “4”}

these set the mailbox to a migrated, remote mailbox. You will see the recipient in Exchange on prem as "Office 365" mailbox.

4. Install ADD Connect and sync users. They will be soft matched according to UPN to the O365 login.

5. Run HCW.

As I said this is a reverse scenario and not like to the move to O365 methods as expected but more matching on premise AD users to O365 plus install Exchange Hybrid.

john korchok say:

You'll have better luck using Windows Movie Maker to add the audio to your movie. It's probably already installed on your computer, but if not, there's a free download here: Download Windows Movie Maker

In WMM, import both your video and audio using the Import Media button. Near the bottom of the screen, change the Storyboard dropdown to Timeline, so you can see the video and audio timelines. Then drag the video to the Video timeline and audio to the Audio/Music timeline. 

The preview window allows to to see if your audio and video are aligned correctly. When you're done, choose Publish to This Computer for best quality.
