Had similar problems on a RAID1 system, W7x64U. Symptom: Periodic inconsistencies in RAID are 0-10%. After several tests (changed hdd, cables, power, testing all, reinstalling...+google) I came to the conclusion that:

 

Enabling Hybernation on a W7 RAID Desktop System will cause RAID desync or inconsistency first area of mirrored hyberfil.sys volume, either if you use hybrid sleep or hybernation. Disabling hybernation (which is default on desktop systems) will make this problem dissapear when you are still using hybrid sleep (better and faster). Voila!
