russ valentine say:

Those instructions do not provide the safest method for moving a PST file, as your troubles illustrate. It is far safer to disconnect a file from a profile before you move it (by closing it), move it, then open the file from its new location the next time you start Outlook. That's what the developers always told us to do, but they don't write the Help files.

If you're lucky, moving the file before you disconnect it will provide you with the "Browse" dialog Aravinda mentions, but it doesn't always, and many people miss that dialog and end up with a profile they can no longer use. If you've lost that dialog but can still get Outlook to open, then just open the file in Outlook using the File > Open > Outlook data file... After that you can set that file as your default and continue to use it.

If you can no longer open Outlook, then you will have to create a new Outlook profile from scratch following these instructions: http://support.microsoft.com/default.aspx?scid=kb;en-us;829918&Product=out2003

When you do, use the manual configuration method so you can tell the new profile to use an existing data file and direct it to the current location of your file.

FWIW, I've found very few of Microsoft's Help files regarding PST files to be completely accurate, which is one of the reasons so many people are having trouble with them.
