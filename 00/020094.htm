Hiding 0 totals in Word table

I set up a table in Word with the intention of data being typed in or hand-written in.

If the data are typed, the total row will sum the data.

When the cells are empty, the total row reads "0.00."

If the data are hand-written, I want the enterer to calculate the total and hand-write it in.

Is there a way to hide the "0.00" unless data are typed in cells?

macropod say:

That can be done by applying a numeric picture switch to the field code that's doing your summation. For example, if you press Alt-F9 to display the field codes and you see:

{=SUM(ABOVE)}

you might change that to:

{=SUM(ABOVE) \# ",0.00;-,0.00;"}

When you're done, press F9 to update the field, then Alt-F9 again to hide the field codes.

Add character before end of cell marker in word 2013 - Microsoft Community

mikerothschildxa say:

Add character before end of cell marker in word 2013

I want to add a character before the end of table cell marker in WORD 2013 but I don't see a way to put the end-of-cell marker in the "FIND" part of the Replace dialog.
jay freedman say:

You're correct that there is no way to specify a table cell marker in the Replace dialog. You'll need a macro for this job.

If you want to add the character to only the cell that contains the cursor, use this (see http://www.gmayor.com/installing_macro.htm if needed):

    

Sub AddToSelectedCell()
    Dim cl As Cell

    If Selection.Information(wdWithInTable) Then
        Set cl = Selection.Cells(1)
        cl.Range.Text = _
            Left(cl.Range.Text, Len(cl.Range.Text) - 2) _
            & "a"  ' <= the character you want to add
    Else
        MsgBox "Put the cursor in the cell and rerun this macro."
    End If
End Sub

If you want to add the same character to the end of every cell in the table that contains the cursor, use this instead:

Sub AddToEveryCell()
    Dim tbl As Table
    Dim rw As Row
    Dim cl As Cell
    
    If Selection.Information(wdWithInTable) Then
        Set tbl = Selection.Tables(1)
        For Each rw In tbl.Rows
            For Each cl In rw.Cells
                cl.Range.Text = _
                    Left(cl.Range.Text, Len(cl.Range.Text) - 2) _
                    & "a"  ' <= the character you want to add
            Next cl
        Next rw
    Else
        MsgBox "Put the cursor in the table and rerun this macro."
    End If
End Sub

why lg e900 window phone cannot power off - Microsoft Community

williamfoo say:

why lg e900 window phone cannot power off

My window phone cannot be power off. After I power off, it automatically power on again. Can someone advise how to resolve it? Is it something wrong with the hardware or the software?
jack cook - aka help_line say:

I would Hard Reset the device.  You can do it either using the included software method (see below) or you can do it using the hardware keys.  After the hard reset, set it up with your Live ID again and then check it to make sure it powers off (I'm guessing that it will).

If by some chance that it repeats the same issue then you have a hardware problem so you want to bring it in for repair/replacement.

First I would try this to make sure:

Try to restart your Windows Phone by pressing and holding the Power button until you see Slide down to power off, then flick down. After it’s off, wait to see if it powers on again by itself BEFORE you press the power button to turn on your phone again. Make sure that it DOES NOT have the charger cable attached!

So if it automatically restarts after you tried the above, if it were mine I would hard reset the device.

With the device on,

  1. Tap Startand flick left to the App list.
  2. Tap Settings, and then tap About.
  3. Tap Reset your phone, located at the bottom of your screen.
  4. Tap Yes to confirm your reset.

 

You can also do a hard reset using the hardware keys.

Press and hold the Camera, Volume Down, and Power buttons simultaneously.   

  1. Release Power when the phone vibrates, but continue to hold down Camera and Volume Down. 
  2. Release Camera and Volume Down when the Format prompt is displayed on screen. 
  3. Press the Windows button  
  4. Press the Windows button again to confirm the hard reset.
