Security Certificate Expiration on HTC 8X (PM23300)

On Monday, October 19, 2015, a security certificate on the Microsoft authentication servers was updated. This certificate change causes customers using AT&T HTC 8X devices to be unable to log into and use a variety of Microsoft services. This severely impacts the device experience.

 

How does that affect you?

  • The HTC 8X device uses a security certificate named SHA-1 (pronounced “Shaw One”) to authenticate you on Microsoft services, such as Outlook, OneDrive, Xbox, Windows Store, etc.
  • This security certificate has known vulnerabilities and is being replaced across the industry.
  • Microsoft began replacing this certificate in their authentication servers on Monday, October 19, 2015.
  • After this certificate is replaced, the HTC 8X device can no longer use Microsoft services, such as Outlook, OneDrive, Xbox, Skype, etc.
  • Because of the age of the device operating system, the only way to change the HTC8X security certificate is to change the device firmware using the Microsoft Device Recovery Tool.
    • It cannot be changed by an Over the Air (OTA) update.
  • This process should take about 45 minutes to complete.

AT&T HTC 8X devices with Windows Phone 8.0 can’t use features that need a Microsoft Account

 

Symptoms

Starting on October 20, 2015, AT&T HTC 8X devices running Windows Phone 8.0 will not be able to authenticate to the Microsoft Account service due to a problem updating an SSL certificate.   

When using an app or feature that requires Microsoft account authentication, you may see some of the following errors such as:

  • "We can't connect right now. Please wait a few moments and try again."
  • "Can't connect. We're having a problem signing in. Make sure you have good reception. If you do, the problem may be on the other end. Wait a little while and try again."
  • The Windows Store will present the following message when attempting to download, purchase, or update apps:  "Microsoft account service is unavailable at the moment try again later"

 

 

Resolution

Back up your files to a PC and use the Windows Device Recovery Tool (WDRT) to re-flash the device.

  1. Transfer any files (such as photos, videos, and downloaded music) you would like to keep to an alternate location.  To do this, connect the device to a PC and use File Explorer to manually move the files from the phone to the PC.   Instructions for using File Explorer to transfer files can be found here:
  2. Use the WDRT to re-flash your device with a replacement ROM image.  You can find instructions and the download location for the
  3. When the recovery tool is completed, your device will be re-flashed to the original factory state with an updated OS image.  You’ll then go through the setup process.
  4. If your device was set to back up to the cloud, you can sign in with the same Microsoft account you originally used, and you will see a “Restore” option during setup.  This will restore many files and settings and help you download your previously acquired applications.
  5. Once setup is complete, you can transfer any files you copied to your PC back to the device.   To do this, connect the device to a PC and use File Explorer to manually move the files from the PC to the phone.   Instructions for using File Explorer to transfer files can be found here:
