Can this be done and if so, how?

anna ma msft say:

Hello KMRL84,

Thanks for posting in our forum.

May I confirm whether your concern is: you have a public folder enabled Mail settings and it have a SMTP address publicfolder@contoso.com , you want to remove this SMTP address from Public folder and assign it to a user account?

If you are in pure Office 365 online environment, yes, it can be done. If you move public folder smtp address to this user, further emails send to this address will be routed to this user mailbox. The existing emails stored in Public folder will not be affected (if you don’t delete this folder).  Please make sure the address should be unique, which means, you can’t assign the same address to both public folder and user.

You can navigate to Exchange admin center ->Public folder->Edit to remove the smtp address (for example publicfolder@contoso.com).

If it is the default SMTP address for this folder, you need to add an additional smtp address first (like publicfolder01@contoso.com), then set additional smtp address as default via Exchange Online PowerShell:

1. Connect to Exchange Online PowerShell.

2. Run the command: Get-MailPublicFolder -Identity " Public Folder name " | FL to check EmailAddresses and PrimarySmtpAddress

3. Set-MailPublicFolder -Identity "Public Folder name" -EmailAddressPolicyEnabled $false to temporarily disable email address policy.

4. Set-MailPublicFolder -Identity  "Public Folder name " -PrimarySmtpAddress <full address you added in exchange admin center before: publicfolder01@contoso.com >

5. Set-MailPublicFolder -Identity "Public Folder name" -EmailAddressPolicyEnabled $true to enable email address policy.

6. Login Exchange Admin center and delete publicfolder@contoso.com address

Then you can add address (publicfolder@contoso.com) removed from public folder to this user account.

Please feel free to let me know if you have different scenario. meanwhile, we suggest you back up public folder items before performing above steps.
