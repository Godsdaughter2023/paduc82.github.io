The solution involves changing the registry.  I've spent time on this problem on and off for over a year.  I finally narrowed the likely cause to a problem with the Firewall Policy registries.  I could not determine the exact registry that was the problem.  So, I replaced all of the registries.  Beware: changing registry entries improperly can prevent your pc from operating.

HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlSet/services/SharedAccess/Parameters/FirewallPolicy


On the working pc, access the Registry Editor by clicking the Start button and entering "regedit" in the "Search programs and files" window.

On the problem pc, first set a system restore point. 


Follow the same process above to find the "FirewallPolicy" registry folder and EXPORT that folder FIRST to your removable drive.  Now you've saved your old settings that you can import back should this solution not work.


With the "FirewallPolicy" folder on the problem pc still selected, go to File and Import the policies from the working pc.  The action is instant.


Restart the problem pc, open Folders and wait for the pc to process the new registries.  When I did this, after a few minutes both the HomeGroup heading and Network heading correctly populated with the other pc's and devices on my network.
