Can't open website url from mail app message - Microsoft Community

john wickham say:

Can't open website url from mail app message

I sent someone an email including a website url from a website forum online webform with a copy to myself which I opened in Mail app on RT but the website link is plain text and I can't highlight it to use CTRL+C and CTRL+V to open it in a browser (when I open the email in Outlook on another pc the link is clickable).

I've looked at the Microsoft website for keyboard combinations but Shift + arrow or Ctrl+ arrow don't highlight the text of the url.

How do I highlight text in a Mail app message. (If I double click the trackpad it highlights one word, but I can't extend that to cover the whole url).

ANSWER: Double click the trackpad mouse to highlight the first word, then move the mouse to the end of the last word and use Shift+click to highlight all the words (ie the whole url), then use CTRL+C and then CTRL+V into a browser address bar window.

HOWEVER: How do you do it using only the touch screen and popup keyboard if keyboard combinations don't work without a mouse?

ANSWER: Touch the first word of a sequence and it will show two circles at the sides of the word. Drag one of the circles with your finger to the side to highlight more words (ie a whole url), then use ctrl+c and ctrl+v using the popup keyboard to open in a browser.

 

Why is it that we have to learn these tricks by trial and error? The Microsoft website for keyboard combinations http://windows.microsoft.com/is-IS/windows-8/keyboard-shortcuts doesn't include either of the methods above and doesn't include Windows key+S (it lists Windows key+R, then Windows key+T) which is a very useful snipping tool to use in OneNote after copying something there, like an image, as the RT keyboards don't have a Print Screen button. I think the snipping tool works for most applications, certainly IE10 and Messaging.
