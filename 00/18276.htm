worksheet.visible is a property for being visible, hidden, or veryhidden. I think you may have intended to use worksheet.activate.

Test case: make a blank workbook with Sheet1 and Sheet2. Select Sheet1. in code (or in your immediate window), run

Sheet2.visible=true

debug.print activesheet.name

I'd expect it to tell you that Sheet1 is still selected. In your case, the activesheet when you run your code is presumably blank, and therefore you get the error that there is nothing to print until you manually select the target sheet.

So, replace ws.visible=true with ws.activate, see if that helps
