djsteele, mvp say:

You cannot use IIf statements to change criteria like that.

Use a WHERE clause like:

WHERE (MyTable.MyField = [Forms]![frmMainMenu]![ReportingYearSelected] AND [Forms]![frmMainMenu]![ReportingYearSelected] <> 99)
OR (MyTable.MyField NOT IN (9, 11) AND [Forms]![frmMainMenu]![ReportingYearSelected] = 99)

How do i convert a VLOOKUP function result as a link to that result - Microsoft Community

abymoni say:

How do i convert a VLOOKUP function result as a link to that result

Dear sir,

Is there any way to get the result of vlookup function as a link to that result


Suggestions are appreciated in advance
jeeped say:
Yes. You can use VLOOKUP() to return the friendly_name in a HYPERLINK() function and ADDRESS() and MATCH() to return an address that can be coupled with the # symbol for the link_location. Consider the following data,

                

The formula in H4 is,

=HYPERLINK("#"&ADDRESS(MATCH(G4,$A$1:$A$10,0),4,4,1,"Sheet1"),VLOOKUP(G4,$A$1:$E$10,4,0))

Clicking on H4 will take the selection to D6. Note that I've included the name of the worksheet (e.g. Sheet1) in the ADDRESS() function's optional sheet_text parameter. This isn't necessary if the table and the HYPERLINK() are on the same sheet.
