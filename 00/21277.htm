hansv mvp say:

The easiest solution would be to create a linked table in the active database that links to Table1 in C:\Test2.mdb. You can then treat it as if it is a local table.

If you prefer not to do that, you could use code like this:

 

Private Sub Label2_Click()
    Dim db As DAO.Database
    Dim rs As DAO.Recordset
    Dim lngMax As Long
    Set db = OpenDatabase("C:\Test2.mdb")
    ' Use totals query to get the maximum value
    Set rs = db.OpenRecordset("SELECT Max(SNO) AS FROM Table1")
    ' Store maximum in variable
    lngMax = rs.Fields(0)
    ' Close this recordset
    rs.Close
    ' Open recordset on the table
    Set rs = db.OpenRecordset("Table1")
    ' Add a new record
    rs.AddNew
    rs!SNo = lngMax + 1
    rs!NameMe = "5Alam"
    rs.Update
    rs.Close
    db.Close
    MsgBox "Data posted in the other mdb"
End Sub

You can use built-in functions to get the sheet name:

=MID(CELL("filename",A1),FIND("]",CELL("filename",A1))+1,255)

Note the file must have been saved and given a name for the above to work.

Tools>Options>Calculation tab>set to Automatic>OK
