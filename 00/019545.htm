FA:

Let me explain the reason that, why Excel allows "="& as Prefix with the Cell Reference.

But first the use of & symbol:

  • Basically & is known as an ampersand symbol used to Connect Text, Cell value and Numbers too.

=A1&A2,  return value from Cell A1 and A2.

Suppose Cell has Values like 10 and 20 then will be 1020.

=A1&", "&A2 will return 10, 20

Or 

="Text1"&"Text2" will return Text1Text2

Or 

=100&200 will show 100200

Now about the "="& :

Check this Formula to Count A:


=COUNTIF(A2:A8,"=A")  

or  

=COUNTIF(A2:A8,"A")

Or even,

=COUNTIF(A2:A8,"="&A1)

Here Excel reads the Formula,, 

=COUNTIF(A2:A8, Equals to value in Cell A1)

When instead of Value, the Cell reference is been used then the Syntax(Command's Grammar) changes, and with the Relational or Comparison Operators =, <>, >, <, >=, <=  the Connector or popularly knows as Connection Operator,  &  comes into the picture.


The basic reason to use the Cell reference in lieu of Value,  is to make the Formula Dynamic, which enhances the scope of the Calculation.
