deanh say:
I've created a document in Word and have several tables created in the document. As I edit and make changes in the document, the table moves, sometimes even moving to the end of the document. How can I anchor it in the place I want it to be in the document? Word 2003. Thank you.


Go to Table Properties, Table tab, ensure that the Text Wrapping is None. This will anchor the table InLineWithText.

Be careful anytime you select and adjust the table with the mouse, this will revert the table back to floating (or the Around wrap).

If you are describing that as you insert new text the tables move "down" the document and if you remove content the table moves "up", this is normal behaviour for a table in Word. If what you want is that the table remains on a specific "page" regardless of what you do before it in the document - this is not possible in Word - automatically that is. You can always move the table to where you want it after you have finished editing the text :-)

Come back with more detail of what you are doing, what you want to happen and what exactly is happening, if none of the above helps.
