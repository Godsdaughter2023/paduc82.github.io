How are you "re-using" the table of contents? If you make use of copy and paste, you must always update it manually after you have added it to another document.

To more easily re-use a TOC, do the following instead: Select a TOC that has the desired formatting. On the References tab, click Table of Contents, and then click Save Selection to Table of Contents Gallery. In the dialog box, type a name for the entry. Pick a location for "Save in." By default, Word suggests the Building Blocks template, but you could change that to Normal.dotm if you wish. When you are done, click OK.

You can now insert this particular TOC format via References | Table of Contents. The preview may not be accurate, but once you have inserted the TOC, you should see the correct appearance without any updates. (Of course, if you later add or remove headings in your document, you have to update the TOC.)
