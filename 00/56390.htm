Usually, if Windows Explorer crashes, it is because of a non Microsoft third party extension that has been added.  Extensions help make up the right click menu in Explorer and sometimes when you add new software to your system, it will add a new Explorer extension.

Since your afflicted system crashes intermittently, it may be hard to track down.  You can compare the afflicted system Explorer extensions to one that work properly and see what might be different.

http://www.nirsoft.net/utils/shexview.html

ShellExView doesn't install anything on your computer, it just runs and displays.

Sort the display by clicking the Microsoft column so all the "No"s are at the top and easy to see.

The "No"s would be things you have added (non Microsoft).
