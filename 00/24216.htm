Separate comma-delimited field into separate fields in MS Access? - Microsoft Community

pat dools say:

Separate comma-delimited field into separate fields in MS Access?

Hello,

Is there a way within an MS Access query to separate the values present in a comma-delimited field into separate fields on the same record?  I have some slick ways to do this in Excel (in addition to the Text-to-Columns functionality) and would love to know how to do the same within the Access environment.

Thank you!

daniel pineault say:

Pat,

It would seem you can't use the Split function in a query (although I am 100% sure I have done so in the past!), so it would seem that Mimmo's solution is the proper approach.  Create a user defined function that invokes the Split function and then use it in your query.

Building on Mimmo's code

Function mySplit(sMyText As String, sDelim As String, lIndx As Long) As String
    On Error GoTo Error_Handler

    mySplit = Split(sMyText, sDelim)(lIndx)

Error_Handler_Exit:
    On Error Resume Next
    Exit Function

Error_Handler:
    If Err.Number = 9 Then
        mySplit = ""
    Else
        MsgBox "The following error has occured" & vbCrLf & vbCrLf & _
               "Error Number: " & Err.Number & vbCrLf & _
               "Error Source: mySplit" & vbCrLf & _
               "Error Description: " & Err.Description & _
               Switch(Erl = 0, "", Erl <> 0, vbCrLf & "Line No: " & Erl) _
               , vbOKOnly + vbCritical, "An Error has Occured!"
    End If
    Resume Error_Handler_Exit
End Function

Then you'd build a query similar to

SELECT mySplit([GROUPDN],",",0) AS Name, mySplit([GROUPDN],",",1) AS City, mySplit([GROUPDN],",",2) AS State, mySplit([GROUPDN],",",3) AS Country, mySplit([GROUPDN],",",4) AS Continent
FROM [YourTableName];
