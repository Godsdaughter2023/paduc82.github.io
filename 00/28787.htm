You can also install a "router" between your cable modem and computer.  This is a very simple form of protection.

How Does a Router Protect Me?

http://www.terryscomputertips.com/computers/how-does-a-router-protect-me/




Of course, it is just another small computer, so you have to 'harden" it also.

Even consumer / home routers have some simple steps you MUST take to protect yourself.

1. Change the Default administrator password. 
Default admin userids and passwords for consumer and commercial routers are easily found on the internet.

1a. Change the default admin userid.
Some consumer devices allow you to define a new admin userid. Do so if possible

2.  Define a non-specific SSID.
Use a random SSID or one that cannot be easily associated with you. For example, "Bob Hakmeplease home network" is not a good idea.

2a. Don't bother with hiding SSID.
That is a trivial security feature that provides minimal security.

3. Disable HTTP configuration for the router, if possible
Do no allow admin access to the device from the internet. Turn on the option that requires a wired connection for admin access

4. Maintain physical security.
Don't place the router where anyone can easily access it and use the reset button to disable all of your security configurations

5. Control Wifi signal leakage.
If you can, don't place the router near external walls and windows, where signal will leak allowing your neighbours and drive-by / "wardriving" hackers access to the signal.  If possible, use directional antenas and turn down the signal strength to minimum required in your home network area

5a. Disable WiFi
Although it is getting rare, if you are not using WiFi for wireless connection between your devices and the internet, turn it off on each device: router, computer(s), printer, TV, etc

6. Enable highest level of encryption - WPA2
Turn on the best encryption on the device. Many have encryption and either do not turn it on or use the minimum.  From best to worst: WPA2, WPA, WEP.
