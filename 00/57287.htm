considerate_guy say:

Windows DVD Maker was created specifically for the Vista version of Movie Maker (e.g. Windows Movie Maker 6.0).  The only output formats supported in that version are DV-AVI and WMV, so it makes sense that Windows DVD Maker would accept those two file types as inputs.


Later when Windows Live Movie Maker was initially released, it's only supported output was WMV.  So linking it to the older Windows DVD Maker was relatively easy.  No updates were really needed.


Now several years have gone by. Movie Maker has been updated to support MP4 output, but Windows DVD Maker is untouched.  In fact, it isn't even included in Windows 8.


So basically, you are looking at a product (Windows DVD Maker) that isn't being developed any further. If you are using Windows Vista or Windows 7, you can use it but only with the file types it was designed to support when it was first created. Other than that, you can convert to those file types or look for a different program for creating video DVDs.  Video DVDs are from the era of analog TVs with a maximum resolution of 480p (NTSC) or 576p (PAL) anyway and the video DVD standard isn't designed to handle today's modern HD video formats anyway.


A better option is to look into creating discs that will store HD video and will playback in Blu-ray players, or to forego using a disc at all and just stream video to your HDTV directly from a PC or server.  With DLNA streaming over your home network, for example, you can play your HD videos and not even need a shiny disc.  You can also store copies of your videos on sharing sites like YouTube, Vimeo and SkyDrive in HD quality for sharing.


As much as I would love to see Microsoft support older standards for longer than they currently do, it seems inevitable that they will drop support for them at some point.  Windows DVD Maker is one that is not being supported and isn't even included in Windows 8 or beyond.


But for now, if you have Windows 7 and Windows Movie Maker and Windows DVD Maker, you can edit your video in Movie Maker and then create a DVD.  If you do it in two steps, just be sure to choose WMV as the output format in Movie Maker since it can't handle MP4 directly.  If you do it directly by choosing File > Save movie > Burn a DVD, it will be handle automatically for you.

Most laptops and many OEM (HP, Sony, etc.) desktop computers come with a factory restore image on a special partition on the hard drive. If you did not create the physical recovery discs and/or damaged the recovery image, contact the computer mftr.'s tech support to order a recovery disc set. Retrieving data when you can't get into Windows:

1. Pull the drive and put it in a USB drive enclosure or use a USB drive adapter. Attach this to a computer running a working install of XP/Vista/Windows7. Use the working Windows Explorer to copy the data to the rescue system's hard drive and then burn the data to cd or dvd or copy it to an external hard drive. I prefer not to do this if I know the drive is infected because there is a chance of infecting the host system. In these cases, I use #2 below.

2. You can boot the target computer with a Bart's PE (XP only) or a Linux Live CD such as Knoppix or Ubuntu and retrieve the data that way. This has the additional advantage of telling you whether the problems you're having are hardware or software-related because if the computer behaves perfectly under Linux you know that Windows (software) is the culprit. Conversely, if the computer misbehaves under Linux you will know that there is a hardware problem. Here is general information on using Knoppix for data retrieval when you can't get into Windows:

You will need a computer with two cd drives, one of which is a CD/DVD burner OR a USB thumb drive with enough capacity to hold your data OR an external USB hard drive. Download the Knoppix .iso and create your bootable CD*. If you are doing this in an older operating system (XP or Vista), you'll need third-party burning software like Nero, Roxio, or the free ImgBurn (Windows 7 can burn .isos natively). Burn as an image, not as data. Then boot with the CD you created and Knoppix will be able to see the Windows files. If you are using the USB thumb drive or the external hard drive, right-click on its icon (on the Desktop) to get its properties and uncheck the box that says "Read Only". Then click on it to open it. Note that the default mouse action in the window manager used by Knoppix (KDE) is a single click to open instead of the traditional MS Windows' double-click. If you want to burn CD/DVDs, use the K3b program.

This depends upon the size of the document that you are trying to print although this issue is quite familiar with Canon printers.
Printer only recognizes that size of paper for which print job is requested. So that depends upon the settings of paper size.
 
However you can set the printer not to detect this setting by disabling the bidirectional printing.
Here is how:
1. Click on Start and go to "Devices and Printers".
2. Right click on the Printer and select properties.
3. Under details tab click "Spool Settings and down the botton you can choose not to support bidirectional printing.
