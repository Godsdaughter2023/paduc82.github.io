jay freedman say:

The feature of expanding and collapsing text below headings is part of the Word program, and is not something that's saved into any file (whether Word's .doc or .docx format, .pdf, or any other that Word can create). In order to have that functionality in a PDF reader, the reader would have to implement it separately from Word. I'm not aware of any reader that does so.

If the "interactivity" you want in the table of contents is collapse/expand like the headings in the text, that isn't available even in Word. If you mean the hyperlinking of the TOC entries to go to the headings, that should work in PDF. If your reader doesn't do it, you can tell Word to save bookmarks on all the headings: When you Save or Save As and choose PDF, click the Options button in the Save As dialog box and click the option for bookmarks based on headings.

Presumably your PDF reader can display a list of bookmarks and let you jump to them.

doug robbins - mvp office apps & services (word) say:

I have reported the issue to the Word Product Team.

As a work-around, use a macro containing the following code

Dim rng As Range
Selection.HomeKey wdStory
Selection.Find.ClearFormatting
With Selection.Find
    Do While .Execute(FindText:="\<I\>(*)\<I\>", Forward:=True, MatchWildcards:=True, Wrap:=wdFindStop) = True
        Set rng = Selection.Range
        rng.Text = Mid(rng.Text, 4)
        rng.Text = Left(rng.Text, Len(rng.Text) - 3)
        rng.Font.Italic = True
    Loop
End With
