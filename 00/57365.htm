"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Hello,

Noting the details that you have provided.

The most common reason for the problem with the icons is the corruption of the Windows icon cache or could be that some Windows files had gone bad. We'd like to confirm few things about your concern so we can come up with an effective resolution.

Are you seeing this icon from File Explorer or from the desktop?

Prior to this issue, did you make any changes to your settings?

Did you recently install some software or updates?

In the meantime, let's try clearing some cache to refresh Windows icons and see if it would do the trick. Here is how:

In Windows 10, the icon cache file is located in:

C:\Users\<your username>\AppData\Local\Microsoft\Windows\Explorer

(Replace <your username> with the actual login name for your Windows account.) In this folder, you will find a number of icon cache files:
•    iconcache_16.db
•    iconcache_32.db
•    iconcache_48.db
•    iconcache_96.db
•    iconcache_256.db
•    iconcache_768.db
•    iconcache_1280.db
•    iconcache_1920.db
•    iconcache_2560.db
•    iconcache_custom_stream.db
•    iconcache_exif.db
•    iconcache_idx.db
•    iconcache_sr.db
•    iconcache_wide.db
•    iconcache_wide_alternate.db

  1. To rebuild the icon cache, you have to delete all the icon cache files that appear in this folder. It’s not as simple as clicking on them and pressing Delete, though: those files are still in use by Explorer, so you can’t just delete them normally.
  2. Close and save anything that you are working on before proceeding. Open File Explorer and go to the following folder:
    • C:\Users\<your username>\AppData\Local\Microsoft\Windows\Explorer
  3. (Replace <your username> with the actual login name for your Windows account.)
  4. Press and hold the “Shift” key and right-click on the Explorer folder. Select “Open command window here.”
  5. A command prompt window will open at that path. To make sure the command prompt is in the correct folder, type the dir command. You should see the iconcache and thumbcache files we discussed earlier appear.
  6. Right-click on the Windows taskbar and choose “Task Manager” from the shortcut menu.
  7. Right-click on “Windows Explorer” in the list and choose “End task” from the shortcut menu. The Explorer and desktop will disappear. Exit Task Manager and make sure that no other application is running except for the command prompt window.
  8. In the command prompt window type the following command:.
    • del iconcache*
  9. Press Enter. The asterisk after iconcache is necessary to make sure all files with names that start with iconcache will be included in the delete operation. That should delete all the icon cache files.
  10. Run the dir command to check the list of remaining files. If one or more iconcache files are still listed, it means some applications are still running in the background. Close them and repeat the procedure again, if necessary.
  11. Now press the Ctrl+Alt+Del keys simultaneously, and choose “Sign off.” Sign back in, and any out-of-date or missing icons should hopefully get repaired or re-created.

Restart your device and check if it changes anything.

Let us know how things go on your end.
