Hi,

Andreas has kindly given you some very nice code for colouring every point in every series... and that is how I would approach this (using Objects for the Chart, Series, etc) but when getting started with VBA, learning how to re-write what the macro recorder has given you is a valuable skill.  To take the code you posted, you can change it like this:

   

    With ActiveChart.FullSeriesCollection(5).Points(2).Format.Fill
        .Visible = msoTrue
        .Transparency = 0
        .Solid
        .ForeColor.RGB = RGB(8, 8, 8)
    End With
 
Basically, where ever you see .Select at the end of one line and Selection. at the start of the next, you can delete the
    ___.Select 
    Selection.___
to make one line of code that will do the same thing, but wont actually select anything.
 
One thing to note: when I tested your code in Excel 2013, I found that the .Solid method undid any changes done by the preceding .ForeColor line.  Presumably this is an error in the way the macro recorder works.  Hence in my code above, I moved .ForeColor.RGB... to the end of the With block.
 
So the above code is where you end up if you hack about with the recorded macro, but if I were doing it from scratch, I'd do it in a similar way to how Andreas did:
 
Sub FormatSeriesPoint()
    Dim cht As Chart
    Dim ser As Series
    Dim pnt As Point
   
    Set cht = ActiveChart            'could then use Andreas' code to check that cht is not Nothing...
    Set ser = cht.SeriesCollection(5)
    Set pnt = ser.Points(2)
   
    With pnt.Format.Fill
        .Visible = msoTrue
        .Transparency = 0
        .Solid
        .ForeColor.RGB = RGB(8, 8, 8)
    End With
End Sub
 
One final tip, you don't have to rely on the ActiveChart.  You could reference the chart directly, if you know what it is called / where it is. 
 
For a chart on a Chart sheet you can use:
    Set cht = Charts("Chart1")
 
but for a chart on a Worksheet you need:
    Set cht = Worksheets("Sheet1").ChartObjects("Chart 1").Chart
 
where "Chart 1" is the name of the embedded chart which, in Excel 2013, is displayed in the Name Box to the left of the Formula Bar.
 
You can also reference the series by name:
    Set ser = cht.SeriesCollection("My Series")
 
but there is nothing to stop two series being named the same!
 
Hope that helps.

You don't have to sign in if you save your documents to the local hard drive. In Word,  select "Save to Computer by default" and clear "Show additional places for saving, even if sign-in may be required"; both options are found at File | Options | Save.

Also, you may want to set up a local user account on your Windows 10 computer, as opposed to using a Microsoft account.
