You are running a Windows 7 Enterprise volume license client using KMS licensing.

KMS licensing requires that you activate through a Host KMS. This is normally configured by an IT department at a business to handle client activations over a network. This must be done every 6 months (180 days).

If you are no longer enrolled at the college, you are no longer authorized to use the license and must remove it from your machine.

This means, you should reinstall the original license that came with the machine, which could likely be Windows 7 Professional Windows 8.1 Pro.

Look at the bottom of the machine for a COA sticker or a Windows 8 sticker to determine which is version and edition of Windows came preinstalled on the machine.

  6. select Administrators in 'Change owner to', check 'replace owner on subcontainers and objects'..this might take a while

  8. type 'cacls l:\messedUpFolder /T /e /g Administrators:f' ...this might take a while
  9. rmdir /S l:\messedUpFolder
