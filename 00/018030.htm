hansv mvp say:

On the Create tab of the ribbon, click Query Design.

In the Show Table dialog, add both tables, then click Close.

Drag the Name field from the first table and drop it onto the corresponding field in the second table to create a join.

Double-click the join line and select the option to return ALL records from the first table, then click OK.

Do the same for the Surname field.

Add the * from the first table to the query grid.

Add the Name (or Surname) from the second table to the query grid.

Clear the Show check box for this field.

Enter   Is Null   in the Criteria row for this field.

Switch to Datasheet view to inspect the result.
