VBA to Insert Red, Transparent, Circle Shape - Microsoft Community

z_a_s say:

VBA to Insert Red, Transparent, Circle Shape

I am having trouble creating VBA code to that will create a red, transparent, circle into my current word document. I also want to be able to adjust line thickness and set to "square w/ text wrapping". So far, all I have is this:

Sub Box()

Dim BBB As Shape

    Set BBB = ActiveDocument.Shapes.AddShape.msoShapeOval

    With BBB

        .PictureFormat.TransparentBackground = True

        .Line.ForeColor.RGB = RGB(255, 0, 0)

    End With

End Sub

jay freedman say:

The problem here is that the AddShape method isn't the correct syntax. Instead of following it with ".msoShapeOval", you need to have a pair of parentheses containing a series of parameters, like this:

Sub Box()

Dim BBB As Shape

    Set BBB = ActiveDocument.Shapes.AddShape(Type:=msoShapeOval, _

        Left:=100, Top:=0, Width:=50, Height:=50, _

        Anchor:=Selection.Range)

    With BBB

        .PictureFormat.TransparentBackground = True

        .Line.ForeColor.RGB = RGB(255, 0, 0)

        .WrapFormat.Type = wdWrapSquare

    End With

End Sub

(Note: the space and underscore at the end of each of those lines is a "statement continuation" indicator, so all three lines are part of one statement.)

You can adjust the values of the Left, Top, Width, and Height parameters to change the location and size of the oval. If the Width and Height have the same values, you'll get a circle. The Anchor indicates the place in the text that the shape is attached to, in this case the location of the cursor at the time the macro runs. The Top value says how far below the anchor the top of the shape will be (or above it if the value is negative).

The kind of text wrapping is set by the WrapFormat.Type value, which I've added in the With group.
