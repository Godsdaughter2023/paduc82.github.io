Just changed my Windows 7 password, and now I can't get back in to - Microsoft Community

the_broad say:

Just changed my Windows 7 password, and now I can't get back in to my Admin account. Please help.

I didn't create a recovery, because the password was easy, now, it's not working.

cmdrkeene say:

If the computer has another account with administrator-level permissions, you can go into that one and reset the original account's password from it.

If there are no Administrator accounts on the system, you can boot to Safe Mode to access the Built-in Administrator account.  From this fail-safe account, you can force-reset your account's password.

  1. Shut down the computer. Turn it back on and tap the F8 key as it boots up (keep tapping it until options appear).
  2. When the boot options appear, choose Safe Mode.

Now that you are in safe mode, you can use the built-in backup Administrator account to perform you necessary recovery. The built in Administrator account should not be used for any other purpose, or it won't be available in situations like this when you need it.

  1. At the login screen, you should see the "Administrator" account that is normally hidden from view.
  2. Open Control Panel / User Accounts.
  3. Select Manage another account and select the account you wish to change to an Administrator account. You can also use the Change Password option here to reset the other account's password.

When you're done, restart the computer normally.

If Shawn's method does not solve your problem then you can use this method:

  1. Set your BIOS so that it uses the DVD drive as its primary boot device.
  2. Boot the machine with your Windows 7 Repair CD.
  3. Press a key when prompted to boot from the DVD.
  4. Select "Repair PC" from the menu.
  5. Select "Repair" when prompted.
  6. Select System Restore when prompted.
  7. Set Windows back to a point before you changed the password.
  8. Plan ahead a little and create, test and document a spare admin account, same as you have a spare house key for emergencies. This will save you an enormous amount of trouble.
