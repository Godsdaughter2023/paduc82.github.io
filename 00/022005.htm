tom_bpc say:

Lucy,

Your summary task probably finishes 1 hour after start of workday (according to Project Calendar) on 3Jan'19.  This suggests one of the following scenarios:

  1. At least one resource is applied whose standard workday starts 1 hour later than that of the Project Calendar.
  2. The Default start time (in Project Options/Schedule/Calendar Options) is (or has been) one hour later than that of the Project Calendar.  This leads to one of the following two issues:
    • When entering a start constraint, the 1-hour delayed constraint start time cascades through the constrained task and its successors.
    • When entering an actual start date, the 1-hour delayed start cascades through the task and its successors.
  3. (Unlikely) There is a task calendar applied whose standard workday starts 1 hour later than that of the Project Calendar.

To fix:

  1. In General Project Options, select a Date format that shows the time of day.
  2. Starting with the last task (1.8) and working backwards, find the first task with a finish that is early in the day.  Then find the cause.

Good luck, tom

count occurrences of a character in a selection - Microsoft Community

hannayaldo say:

count occurrences of a character in a selection

Hello,

 

Is there a Word macro that counts occurrences of specific characters in a selection. Selection.Characters.Count return all the characters. I only need a count for a specific characters.

 

Thanks,

 

~ Hanna

jay freedman say:
The macro below deletes all the characters in the selection that are not in the list named SpecChars, counts the number of characters that are left, and then undoes the deletion.

Sub CountSpecCharsInSelection()
    Dim rg As Range
    Dim iCount As Long
    Dim SpecChars As String
    SpecChars = "aeiou" ' change this as needed, or use InputBox to get it
    
    Application.ScreenUpdating = False
    
    Set rg = Selection.Range
    With rg.Find
        .MatchWildcards = True
        .Text = "[!" & SpecChars & "]"   ' wildcard expression for "any character not in SpecChars"
        .Replacement.Text = ""
        .Execute Replace:=wdReplaceAll
        iCount = rg.Characters.Count
    End With
    
    ActiveDocument.Undo
    Application.ScreenUpdating = True
    
    MsgBox "Found " & iCount & " characters of " & SpecChars
End Sub
