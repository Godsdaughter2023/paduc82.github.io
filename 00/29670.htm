a. user say:

Hi,

You need to have a RETAIL licence, to transfer your Windows OS from a motherboard to another, if that's what you want to do. To know which version you are on, you need to:

1. Digit "cmd" in windows search bar
2. Right click on "Command Prompt"
3. Select "Run as Administrator"
4. Digit "slmgr /dlv" without quote and press ENTER

Read the MAIUSC word on "Description" line. That's your licence kind.
If it is RETAIL, you can install the old hard drive on the new machine. Once booted, you will need to activate Windows 10. To do so:

1. Press WINDOWS + I
2. Click on "Update and Security"
3. Click on "Activation"
4. Click on "Troubleshooter"
5. Click on "I recently change hardware"
6. Follow on screen instructions.

P.s: Please, if the answer resolve your issue, click on Yes. Thanks.

¡firedog say:

It sounds as if the printer is using different fonts in the two cases. When graphic elements are present, it's likely that fonts will also be transmitted to the printer as bitmaps. With plain text, they won't. If the character set being transmitted as bitmaps doesn't contain the special characters you need, you won't get them on the printout. You'll have to contact the printer manufacturer for help with this, unless you can find settings in the printer control panel to change this behaviour.

Try using a different printer, e.g. the XPS Document Writer.
