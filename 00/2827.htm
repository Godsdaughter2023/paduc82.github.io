mediumbrowntoast say:

 ... BTW I'll assume that windows 10 supports legacy bios mode?

It does yes. Thanks for replying :) 

I finally managed to get Win 10 installed and fully working. Although OSX is not installed. I destroyed the GPT info and started from scratch. The key to installing in legacy BIOS mode was to boot from a DVD I had burnt from the .iso. As long as I booted from the DVD, the Apple EFI boot launcher would hide all EFI capabilities from Windows 10 (which is what's needed to get successfully install in legacy BIOS mode). Using a USB drive to boot won't work because you'll only have the option to boot a EFI install.

If you start the install and you're not sure if you've booted into an EFI or non-EFI install, there is one giveaway I noticed...if the resolution looks native and the text is crisp then you've booted into an EFI install. If the resolution is low and the text is crappy, you've booted into a legacy BIOS install.

Windows 10 (10074) has been running great for the last few days. All hardware is working (including sound).

In theory, it should be possible for me to successfully use BootCamp as long as I use a DVD to do the install. BootCamp should correctly tell the Apple boot loader to load with the CSM enabled and to also hide the GPT from the Windows installer. If a USB drive is used instead, the UEFI installer will run and it will be aware of the EFI capabilities and GPT. During install it'll complain about trying to install on an MBR partition and after install the sound won't work because the CSM won't be instructed to load.

TL;DR - Boot the Windows 10 install from a DVD.

Howdy The Cool Dude,

Thank you for posting on the Microsoft Support forums.

There are a couple of steps that can be performed to resolve this issue.

The first is to reset the save locations in the Music library:

  1. Open a File Explorer window. (Simply open a folder)
  2. On the left, right click the Music library under the Libraries item in the tree view.
  3. Select Properties.
  4. Select Restore Defaults then select OK.

The next step is to make sure the folders Xbox Music is looking in exist and aren't an external source (Web drives, Dropbox, OneDrive, etc):

  1. Open the Xbox Music app.
  2. Swipe in from the right or press Windows Key + C.
  3. Select Settings then select Preferences.
  4. At the top, select Choose where we look for music on this PC.
  5. Add your main Music folder to the list and remove all the others.

As soon as those two steps are done, try to download a song again. If you're still having issues downloading music then contacting Xbox Music live chat support at the link Nathan Roberts provided will be ideal.
