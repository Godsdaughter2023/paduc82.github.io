cmdrkeene say:

Geek Squad could certainly do it, as easy as it is for them to do I'm sure they love those kinds of jobs.   But you can totally do it for free without a ton of work.

As typical there's several ways it can be done.  Two of the easiest ways are:

A)  In the Settings app, under Updates & Security, choose Recovery and then click the "Reset this PC" section.

or

B)  Inside the Windows Defender Security Center app, click the "Fresh start" option.

or

C)  Download the Windows 10 Media Creation Tool and run that tool to create your own USB install stick (or burn a disc).  Then you can boot up from that drive and clean-install the system directly.

C is usually the option I use, but I don't have a good reason why it's just what I've always done.  When you boot up from your install drive, you choose "Install Windows" and then when it asks where to install, I usually delete all listed partitions.  Then when the only hard drive area left is "unallocated space", I click next and let it install.

---

You are right about making a backup of your important files first, since this would make the computer "just like new".  OneDrive and DropBox are great, but if you have a lot of data they might not be free choices.  You can get a USB thumb drive for less than $10 at Walmart and copy files to that too for safe keeping.

muralidhar_g say:

Hi,

Thank you for writing to Microsoft Community Forums.

This might be caused by a compatibility problem between application and version of Windows you are using.

The application developer might have not updated the app to continue using it on latest Windows version. This is the root cause of the issue.

Follow the below steps if you are referring to built-in application.

Run the application in compatibility mode.

1. Right-click on an app and select Properties. You can do this from the app's shortcut or by navigating in File Browser to the EXE file.

2. Select the Compatibility tab, then check the box next to "Run this program in compatibility mode for:"

3. Select the version of Windows to use for your app's settings in the dropdown box.

4. Hit Apply, then run your app and see if this fixed your issues.

 Follow the below steps if you are referring to online application from browser. 

Add a site to the Compatibility View list

1. Open Internet Explorer, select the Tools button, and then select Compatibility View settings.

2. Under Add this website, enter the URL of the site you want to add to the list, and then select Add.

Note: If you add a site to the compatibility View list and the page looks worse, the problem might not be compatibility and you should remove the site from the list

Once you turn on Compatibility View, Internet Explorer will automatically show that site in Compatibility View each time you visit. You can turn it off by removing the site from your compatibility list.

Hope it helps.
