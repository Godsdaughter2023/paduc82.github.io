how do i uninstall photo app from win 10? - Microsoft Community

nathancoop say:

how do i uninstall photo app from win 10?

Having trouble importing photos using Photo app on Win 10. Was working but not anymore. Would like to uninstall and reinstall the app. Help!!

a. user say:

Hi Nathan,

Welcome to Microsoft Community.

Sorry to know that you are facing issues with Photo app on Windows 10. I will certainly help you.

I suggest you to follow the below troubleshooting methods and check if it resolves the issue.

Method 1: Run the Apps Troubleshooter

Run the App troubleshooter to fix issues that may prevent app from working properly.

Click here to download and run the Apps troubleshooter.

Method 2: Reinstall the Photo app

If issue persists, I suggest you to uninstall and reinstall the Photo app. Here's how:

1. Press Windows Key + Q, and in the search box, type Powershell.

2. From results, right click on Windows PowerShell and select Run as administrator option.

3. Type Get-AppxPackage and press Enter.

(This command will output a long list of all installed apps containing lots of information about each app).

4. Type Get-AppxPackage | Select Name, PackageFullName and press Enter.

(This command will show only the name of the app and its PackageFullName information).

5. Select the PackageFullName information of the desired app and copy it.

6. Type Get-AppxPackage PackageFullName | Remove-AppxPackage and press Enter.

(This command is to remove the app from the system, replace PackageFullName with the copied information with Step 5).

For example, if you want to remove Photo app, the command to remove it will be as following:

Get-AppxPackage *photos* | Remove-AppxPackage

7. Wait till completion of the command, and check if the app is uninstalled, if not restart the system and

check.

Note:

If you want to uninstall the app from all user accounts, use the following command format:

Get-AppxPackage -allusers PackageFullName | Remove-AppxPackage

If you want to remove the app from any particular user account, use following command:

Get-AppxPackage -user username PackageFullName | Remove-AppxPackage

(Replace username in the above command with the correct user account name).

8. Launch the Windows Store and reinstall the Photo app.

Hope this helps. If the issue remains unresolved, please get back to us and we would be happy to help.

Thank you.

We also suggest that you re-register Windows Spotlight by following the steps below:

  1. Open Settings.
  2. Click Personalization.
  3. Select Lock screen.
  4. Under Background, select Picture or Slideshow.
  5. Once done, press Windows key + S and type PowerShell, right click on Windows PowerShell, and choose Run as administrator.
  6. Enter the administrator credentials once prompted.
  7. In Windows PowerShell window, type or copy and paste this command:
    Get-AppxPackage -allusers *ContentDeliveryManager* | foreach {Add-AppxPackage "$($_.InstallLocation)\appxmanifest.xml" -DisableDevelopmentMode -register }
  8. Press Enter.
  9. Go back to Settings and set Windows Spotlight as the background.

Try the following steps below which can also resolve Windows Spotlight issue:
