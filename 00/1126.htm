jay freedman say:

Because the legacy dropdown formfield has a maximum entry length of 50 characters, it won't be usable for full sentences. You'll have to use a dropdown content control instead.

Although you can paste text into the list item box in the control's Properties > Add dialog, it would still be quite tedious to do it manually because you must close the Properties dialog after adding each item in order to be able to select and copy the next sentence, and then reopen the dialog and click the Add button before you can paste. That argues in favor of a macro solution.  :-)

The following macro assumes that each sentence that you plan to copy ends with a paragraph mark, making each "sentence" a separate paragraph. (Word VBA has an inconvenient definition of "sentence", so dealing with paragraphs is more robust.) If that isn't true of the document now, you can use the Replace dialog to replace each period-and-space occurrence with period-and-paragraph mark (using the code ^p to represent the paragraph mark).

The macro further assumes (and checks) that you have inserted a bookmark around each group of sentences that will form the entries for one dropdown content control. If you have twenty bookmarks, the macro will create twenty content controls. Each bookmark can contain any number of sentences.

The content controls will be inserted at the end of the document that contains the sentences; from there you can copy/paste them to anywhere else.

Sub LoadDropdown()
    Dim bk As Bookmark
    Dim par As Paragraph
    Dim rg As Range
    Dim CC As ContentControl
    
    If ActiveDocument.Bookmarks.Count = 0 Then
        MsgBox "Insert a bookmark around the sentences" & vbCr & _
            "to add to each dropdown and then rerun this macro."
        Exit Sub
    End If
    
    For Each bk In ActiveDocument.Bookmarks
        Set rg = ActiveDocument.Range
        rg.Collapse wdCollapseEnd
        rg.InsertParagraphAfter
        
        Set CC = ActiveDocument.ContentControls.Add( _
            Type:=wdContentControlDropdownList, Range:=rg)
        
        For Each par In bk.Range.Paragraphs
            Set rg = par.Range
            rg.MoveEnd unit:=wdCharacter, Count:=-1 ' don't include paragraph mark
            CC.DropdownListEntries.Add rg.Text
        Next par
    Next bk
End Sub
