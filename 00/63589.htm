Most people just use the Windows 10 built-in synchronization feature to effectively "transfer" their favorites from one Win10 device to another.  There are a few "gotcha's" to this system.... you must log into your computer using a Microsoft Account (not a local account), and both devices must be running Win10.  The sync is a low priority system, so it could take several minutes to occur.

There is however, another way... You can use a 3rd-party application called EdgeManage to make a text-based bookmark.htm file (like you may have done with Internet Explorer).  Then you can transfer this "export file" to the new system and use EdgeManage to import it.
