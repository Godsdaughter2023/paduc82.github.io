I had a similar issue today after upgrading to Windows 10. I had a Hotmail account with an old email that I had associated with the account.  It seems that it picks up the first in alphabetical order amongst associated and alias emails. If you open up regedit (from the search box) and search the reg editor Edit > Find for the offending email, you should come to a registry entry called HKEY_CURRENT_USER\SOFTWARE\Microsoft\ActiveSync\Partners\{GUID}.  In here there are 2 entries Email and EmailAddress.  I changed the Email to my desired account and removed the email I did not want from the list of emails in the EmailAddress entry.

The email also occurred in another location (keep finding next in the reg editor)HKEY_USERS\S-1-5-21-GUID\SOFTWARE\Microsoft\ActiveSync\Partners\{GUID}

I updated in the HKEY_USERS section and this updated the entry in the HKEY_CURRENT_USER section.

Not sure if this will help, but at least you will be able to find where the details are stored.

considerate_guy say:

If this is a standard "full screen" TV signal, that's a 4:3 aspect ratio (closer to square).  It sounds like your Movie Maker settings are also 4:3 which makes sense.

The issue is that YouTube has standardized on a 16:9 (widescreen) display in your browser to handle both widescreen and fullscreen.  And if you upload full screen, the unused space is black.

There isn't much you can do except start working with 16:9 widescreen source material.  You can't make a (nearly) square frame fit into a rectangle if you want to keep the correct proportions.  The other option would be to find a way to either stretch the video (make everything look short and fat) or crop the video (losing the tops and bottoms of things).  Neither of these stretching/cropping choices are available in Movie Maker anyway, so I'd just live with it.  YouTube is actually maintaining the correct ratio.

    More...

    Transferring data from Outlook Express (WinXP) to Windows Live Mail:

    For Messages:
    Copy the *ENTIRE* OE message store folder to a flash drive. (Folders.dbx must be included). Place this on the Desktop or other location on the machine using WLMail => Alt+F | Import Messages | Microsoft Outlook Express 6 and point to where you saved it.

    OE Message Store Location:
    In OE: Tools | Options | Maintenance | Store Folder will reveal the location of your Outlook Express files. Write the location down and navigate to it in Windows Explorer or, copy and paste it into Start | Run.
     
    In WinXP the OE user files (DBX and WAB) are by default marked as hidden. To view these files in Windows Explorer, you must enable Show Hidden Files and Folders.
     
    For Addresses (Contacts):
    Open the Address Book in OE and File | Export | Address Book (wab) and save it to the Desktop. Copy to a flash drive. Place this on the desktop or other location on the machine using WLMail.
     
    For WLMail 2012: Switch to the contacts window (click the address book icon at the foot of the folder pane, or press Ctrl-3). Click the Import button, select Windows Address Book (wab) and point to where you saved it.
     
    Note: If you use a CD or DVD instead of a flash drive, after placing on the new machine you must remove the Read Only attribute in Properties before you import.

    For Account Settings:
    In OE: Tools | Accounts, select the account and export it to the desktop. This will be an IAF file. Copy it to the new machine's desktop.
     
    For WLMail 2012: Click on/open Alt+F | Options | E-Mail Accounts and import the settings from the location you saved them.

    andre da costa say:

    The fact is, Macs don't come preinstalled with Windows, a full Windows license must be bought by the owner of the Mac.


    OEM Windows is much cheaper than its retail counterpart, but it carries certain restrictions. OEMs will bundle certain applications and utilities to reduce the overall cost of the machine by forming partnerships with software vendors.


    Unfortunately, your options are limited. Your best option would be to customize the system the way you like by removing all the bundled applications and utilities you don't want then create a recovery image point. Trying to using retail media or other OEM media will render your system inoperable requiring that you contact the manufacturer for recovery media to reinstall the OS.


    Press Windows key + X


    Click Programs and Features


    Select the application/s you don't want and click Change/Uninstall on the command bar


    Follow the on screen instructions to remove it.


    After you have customized the system the way you like, create a recovery image point which you can use in the future to reinstall Windows 8 without having to customize it again.

    Create a recovery image point:



    From the Start screen, type cmd, and then right-click the Command Prompt app and click "Run as administrator." Click through the UAC prompt. and at the command prompt type recimg /createimage C:\Recovery. The computer will churn for awhile as it writes the image to disk—it's not unusual for this process to take several hours, so give it time.

    moving the restore image:



    If in the future you'd like to move the C:\Recovery directory elsewhere to save space on your main drive, you can; all you have to do after you move it is fire up another administrative command prompt window and type recimg /setcurrent [New directory name] to set a new path for the recovery image.


    Also, note that while a refreshed PC will include all desktop applications you had installed at the time you made the image with recimg, it will uninstall all desktop applications you've put on there since. If you'd like to update the image with new drivers or applications, just run recimg with the /createimage switch again.
