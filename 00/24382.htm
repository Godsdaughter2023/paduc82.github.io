With this approach you are going to open the report for each owner in the employee details?  Why not create a single report and start each employee on a new page, then you need not perform this loop.

As for your direct question, I'm assuming that owner is a text field and as such the value need to be surrounded by single quote, such as:

DoCmd.OpenReport "Owner test", acViewReport, , "[owner] ='" & rs![owner] & "'"

Also, name is a reserved word and you should not use it in any of your code or database objects to avoid potential conflicts.

DoCmd.OpenReport "Owner test", acViewReport, , "[owner] =""" & rs!owner & """ "

Calculate a percentage in Excel column when filtered - Microsoft Community

crtate say:

Calculate a percentage in Excel column when filtered

Hello

I'm trying to calculate the percentage of figures in a column which are over 2.0. I'm using this formula to calculate it:

=COUNTIF($A$1:$A$10,">=2")/COUNTA($A$1:$A$10)

I want it to calculate the percentage when the data in the table is filtered. Is this possible and how?!

Thank you.

ashish mathur say:

Hi,

You are welcome.  The SUBTOTAL function only runs the arithmetic on the visible cells of a filtered range.  The ROW(A1:A10)-ROW(A1) portion of the formula causes the SUBTOTAL function to iterate over every row of the range starting from cell A2 and if the row is visible, then perform the arithmetic (add numbers).  The result of the SUBTOTAL(2,OFFSET(A2,ROW(A1:A10)-ROW(A1),)) portion will be 1 or 0 (1 if the row is visible and 0 if it is hidden).  When these numbers are multiplied with the actual numbers in range A2:A10, one gets a 0 or the actual number.  We then run the condition of >2 and simple count all those rows.

Hope this clarifies.

If my reply helped, please mark it as Answer.

andreas killer say:
You can replace the references with any name you like, e.g.

=SUMPRODUCT(SUBTOTAL(3,INDIRECT("A"&ROW(Table[Nr])))*(Table[Nr]>=2))/SUBTOTAL(103,Table[Nr])

rohn007 say:
Your approach to headers works at the "section" level.
Margins also work at the section level. So you can only have one set of margins defined for a section.

If you really do need different margins on the second page you will have to insert a section break. If the content on the first page is limited, it could be a next page break. Or if the text flows from page one on to page 2 you could try placing a continuous section break. Exactly where it needs to be you'll have to figure out for yourself.

Sections, Section Breaks, Page Numbering, Columns, Headers and Footers, and Watermarks in Microsoft Word - http://www.addbalance.com/usersguide/sections20...
Sections are the Word feature that controls page number formatting, Headers and footers, orientation (portrait/landscape), margins and columns. If you are having problems or questions with one of these, you need to know more about Sections in Word.

Changing Page Margins http://wordribbon.tips.net/T009501_Changing_Pag...
Part of determining page layout is to specify the size of the margins that surround the text on a page. Word allows you to easily modify the margins used on your pages.
