Restarting the Service

Step 1
Open the Start menu. Type "services.msc" into the search field. Click the link that appears -- it will have a small gear-shaped icon next to it. This will open the "Services" window.

Step 2
Scroll down to find the "Security Center" listing. Check the "Status" column. It will either read "Started" or be blank. A "Started" status indicates that the service is running but that the system is having trouble accessing it. A blank status indicates that the service is turned off.

Step 3
Check the "Startup Type" column. If the column reads "Manual" or "Disabled," right-click on the listing and click "Properties." This will open a small window titled "Application Information Properties." By default, this window will open to the "General" tab. Halfway down the window is a drop-down menu labeled "Startup type." Select "Automatic." Click "Apply," then "Ok."

Step 4
Highlight the "Security Center" listing within the "Services" window. A portion of descriptive text explaining the service will appear in the left sidebar. At the top of this text will be a link. If the Status column reads "Started," the link will read "Restart this service". If the Status column is blank, the link will read "Start this service." If the service starts or restarts normally after the link has been clicked, then the problem is solved.


Fixing Security Problems

Step 1
Uninstall any redundant antivirus or security software programs. Whether you decide to use a Windows security product or a third-party application is up to you.

Step 2
Perform a clean reinstall of your chosen security programs. Though removing extra programs should clear up the problem, it is best to start from scratch in order to get Security Center back on its feet.

Step 3
Run a full scan of your computer using the malware scanning tool of your choice. Remove any Trojans, worms or other malware that you find per the instructions within your security software.

Skip entering your Windows 7 product key and complete the installation.

Once you reach the desktop, click Start, right click Computer

Click Properties

Scroll down to Windows Activation

Click the link x days until activation.

Click the link that says 'Show me other ways to activate'

Enter product key

Click Next

Select Phone Activation

Click Next when Enter your key page displayed

Select your country

Click Next, call the number listed

Remember to explain to the call agent your situation.

Provide the generated installation ID when requested by the call agent

They will give you a confirmation ID in return, enter it

Click Next to complete the activation.

The requirements for upgrade media though is that you have a previously qualifying operating system such as Windows XP or Vista installed to use it. From the Windows 7 End User License Agreement.

15. UPGRADES. To use upgrade software, you must first be licensed for the software that is eligible for the upgrade. Upon upgrade, this agreement takes the place of the agreement for the software you upgraded from. After you upgrade, you may no longer use the software you upgraded from.

So, if you are still denied, you will have to simply reinstall Windows XP or Vista and let it do the eligibility check.

 How to activate Windows 7 manually (activate by phone)

To turn the function keys on for Windows 10, you can try the following steps:
