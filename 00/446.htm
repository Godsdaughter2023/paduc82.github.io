suzanne s. barnhill say:

Provided you have set up heading numbering as described at http://shaunakelly.com/word/numbering/numbering20072010.html, here's what you do:

  1. On the References tab, in the Captions group, click Insert Caption.
  2. In the Caption dialog, click Numbering...
  3. In the Caption Numbering dialog, check the box for "Include chapter number" and select the heading style that includes the numbering you want to include. To be able to use this feature, you must use Word's built-in heading styles (at least for the level whose number you want to include).

roady [mvp] say:

No, connecting to an Exchange account from Outlook via the Exchange ActiveSync protocol isn't supported. This protocol is only intended to be used for mobile apps.

The preferred protocol for Outlook to Exchange connectivity is MAPI-over-HTTP with a fall-back to RPC-over-HTTP. In Outlook, this is just exposed as "Exchange".

For this, there is no manual configuration as it completely relies on your Autodiscover service of Exchange. For this, you need to use Auto Account Setup (either from within Outlook or from the Mail applet in Control Panel) and fill out your name, email address and password. If Outlook can't configure itself, it is likely you have a problem with the configuration of this service.

You can use the Remote Connectivity Analyzer from Microsoft to perform Outlook Connectivity and Outlook Autodiscover tests;

https://testconnectivity.microsoft.com/

Additionally see: Autodiscover: Some quick methods to get it working
