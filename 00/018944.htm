chip pearson say:

This can be done only with code. There is no point-and-click way to accomplish this.

 

The Visible property of a worksheet has three possible values: xlSheetVisible, xlSheetHidden, and xlSheetVeryHidden. If a sheet's Visibile property is xlSheetHidden, it can be made visible by the user through the Excel user interface. If a sheet's Visible property is xlVeryHidden, it will not appear in the list of hidden sheets and can be made visible only through VBA code. Using only the Excel User Interface, it is impossible to determine whether the sheet even exists, let alone make it visible. The code syntax is something like

 

Dim WS As Worksheet

Set WS = Worksheets("Sheet2")

WS.Visible = xlSheetVeryHidden

 

to restore the sheet to visible status, use

 

WS.Visible = xlSheetVisible

greginmich say:

The CPU usage of the scan engine (MsMpEng.exe) was so atrocious with OneCare that a few of us were pleading for auto-throttling, and I was personally recommending ThreadMaster as an interim solution. The CPU-throttling feature in Security Essentials is very effective, but it’s only applied to the scheduled scan, and it’s not clear that the scheduled scan is actually what’s being tested here.

 

To test the CPU-throttling feature; first make sure that the “Start the scheduled scan only when my PC is on but not in use” option is checked, so the scan will be run with a Task Scheduler task. Then open Task Scheduler and right-click and “run” the Microsoft Antimalware Scheduled Scan. Alternatively, right-click on the desktop and select a new shortcut. Then copy and paste this location (for Windows 7):

 

C:\Windows\System32\schtasks.exe /Run /I /TN "\Microsoft\Microsoft Antimalware\Microsoft Antimalware Scheduled Scan"

john korchok say:

As tempting as it might seem to try to rework the existing presentation, it's best to start over.

Create a new template and start with a color theme and a font theme. Then format the Slide Master with as much formatting as possible that is common to most of your slides.

Next, create slide layouts that leverage the master where possible. Often your cover or title slide and your section divider slide will use different formatting, so for them remove the background graphics inherited from the master. Each slide layout can handle the formatting for different types of slides, if you analyse the layout.

Finally, create your slides and paste in the content from the slide layouts of your old presentation. This sounds like a lot of work, but in the long run you'll have a presentation that will be easy to use and update over the years.
