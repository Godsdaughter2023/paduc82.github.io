Hi RichieP40,

Based on your description, the affected user is using the Outlook client to move emails to other’s mailbox. 

Please perform the following steps to check the affected user’s permission:

1. Connect to Exchange Online PowerShell.

2. Run this commend to check the affected user’s permission in the shared other user's mailbox:

Get-MailboxPermission -Identity user@contoso.com -User "user1"

3. Run the following commend to remove the affected user’s permission:

Remove-MailboxPermission -Identity user@contoso.com -User "user1” -AccessRights FullAccess -InheritanceType All

4. Run the following command to assign permission to the affected user:

Add-MailboxFolderPermission usera@contoso.com:\Calendar -User userb@contoso.com -AccessRights Author

About the AccessRights level, please refer to this article: https://technet.microsoft.com/en-us/library/dd298062(v=exchg.160).aspx

For detailed steps, please refer to these articles:

Get-MailboxPermission

Remove-MailboxPermission
