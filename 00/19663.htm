stefan blom say:

If this is an OLE (automation) link, that is, an inserted object, then what you are seeing is by design. Word only displays the first page. What you can do is choose the option to "Display as icon"; the icon can then be clicked and the linked document will open in its own window.

A better option may be to insert the file via the Insert File dialog box. Here's how to do it: Click the Insert tab, click the arrow on the Object button, and then choose Text from File. In the dialog box, locate the file you want to insert, click it, click the arrow on the Insert button, and select Insert as Link. This adds an INCLUDETEXT field, which doesn't involve OLE.
