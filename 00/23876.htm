niranjan i k say:

Hi,
                you are getting this error while saving or opening the file because the path (including file name) exceeds 218 characters. Excel limits the path (including file name) to 218 characters; this limitation includes three characters representing the drive, the characters in folder names, the backslash character between folders, and the characters in the file name.

The limitation is based on the following:

  1. Up to 31 characters in a sheet name.
  2. Apostrophes and brackets used to denote the workbook name.
  3. An exclamation point.
  4. A cell reference.

To resolve this issue make sure that the path to the file contains fewer than 219 characters. To do this, use one of the following methods:

  1. Rename the file so that it has a shorter name.
  2. Rename one or more folders that contain the file so that they have shorter names.
  3. Move the file to a folder with a shorter path name.

For more information on the error "Filename is not valid" click on the link below.
