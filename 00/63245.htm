  1. Press the Windows Key + X.
  2. Select on Command Prompt (Admin).
  3. At the command prompt, type net start, and then press ENTER to display a list of currently running services. If Task Scheduler is not displayed in the list, type net start "task scheduler", and then press ENTER.

Note that you may have to change the parameters for your situation.
