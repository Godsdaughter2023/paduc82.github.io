karl timmermans [outlook mvp 2012-2018] say:

Both Outlooks are POP.  I would like to import all of my contacts, email, etc.   

There's absolutely no reason to import anything - you can easily start off on the new computer exactly from where you left off on the old computer - just by paying attention to one item when you configure the email account on the new computer (as per screenshot below)

Do the above and your new computer will start from the same place with the following very important note to keep in mind which is:

If you plan on keeping the old computer in motion - make sure that you change the POP account setting to "LEave Messages on Server" - otherwise if you access the account on the old computer before accessing things on the new computer - you won't get anything on the new computer. It will all come down to the sequence in which you connect to your accounts. If you're going to access the account from both computers, you may want to set the second computer to "Leave Messages On Server" for "X" number of days at which point they will automatically delete to make sure that you have accessed the account from the other computer
