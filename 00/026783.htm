amit tandon say:

Not clear about your specific example, but to return 1 for one Team, to return 0.5 for 2 Teams, to return 0.33 for 3 Teams, and so on, you may use as below.

Presuming the teams are mentioned in range A2:A10 viz. Team 1 (in cell A2), Team 2 (in cell A3), and other cells are blank, you may use either of below formulas:

=1/COUNTA(A2:A10)

COUNTA(A2:A10) returns 2 which counts all non-banks in the range

1/2 will return 0.5

Alternatively, you may also use below formula:

=1/COUNTIF(A2:A10,"Team *")

COUNTIF(A2:A10,"Team *") returns 2 which is the count of strings starting with "Team "

1/2 will return 0.5

If Team 3 is mentioned in cell A4, the above formulas will return 0.33.
