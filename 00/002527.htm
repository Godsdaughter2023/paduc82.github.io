A* = 12

A = 10

B = 8

C = 6

D = 4 

E = 2 

So in the above example of one student's set of GCSE exam results, using the above values the numerical value of the students' GCSE exam results would be 78.

amit tandon say:

Refer below image:

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell E2 & copy down:

=SUM(IFERROR(CHOOSE(IFERROR(MATCH(IF(NOT(ISNA(MATCH(MID($D2,ROW(INDIRECT("1:"&LEN($D2))),1),{"A*","A","B","C","D","E"},0))),(MID($D2,ROW(INDIRECT("1:"&LEN(D2))),1))),{"A*","A","B","C","D","E"},0),0),12,10,8,6,4,2),0))-(LEN($D2)-LEN(SUBSTITUTE($D2,"A*","")))/2*10

Pls note that columns A & B have not been referenced in the formula (their values have been hard coded within the formula itself).
