access2007techmi say: I ran into a similar issue.  Here's how I fixed it.

On the login form, I filled in a textbox based on a ReturnVars value:

SetProperty
     Control name    txtUserID
     Property              Value
     Value                =[ReturnVars]![rvarUserID]

Next, I stored a TempVar, using the value from my text field

SetTempVar
     Name                 tvarUserID
     Expression=     [txtuserID]

In the OnLoad property of the form, I used the set property to enter the UserID

SetProperty
     ControlName     UserID
     Property               Value
     Value                   =[TempVars]![tvarUserID]

It took a few attempts to get the syntax right, but this worked for me.
