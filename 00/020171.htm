Gradient line for an arrow connector - Microsoft Community

mattherington say:

Gradient line for an arrow connector

Hi there. This is possibly a really simple question, but how do I set a gradient line for an arrow connector, with a gradient from the beginning of the arrow to the end, regardless of the position on the page?

Many thanks,
Matt

john goldsmith say:

Hello Matt,

Well you can fix this, but only by making a change in the ShapeSheet as far as I can see.  Basically, the Dynamic Connector, which I think you're using here, is a special shape and Visio writes to various cells in the shape, one of which is the Angle cell.  In the Dynamic Connector this is fixed to 0 degrees.  The gradient angle uses this value and, because it's guarded, the gradient angle doesn't change either.

If you want to change the behaviour you'd need to open up the master in the document stencil and set the LineGradientAngle cell to the following formula:

=-ATAN2(EndY-BeginY,EndX-BeginX)

This sets the gradient angle to be an offset of the angle of the shape itself.

If that sounds a bit over complicated, then a simpler alternative would be to use a different connector.  The other connectors don't have a protected Angle cell and so work as expected.  Have a look under More Shapes / Visio Extras / Connectors.

Hope that helps.

Best regards

John
