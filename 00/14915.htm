Scott is absolutely right. I never distribute ACCDB files. And I disable the default shortcut menus as well as the full menus. The code I use to set the database properties to limit what the users can do is as follows:


Private Sub ChangeProperty(db As Database, strPropertyName As String, _
        varPropertyType As Variant, varPropertyValue As Variant)

    On Error GoTo err_PROC

    db.Properties(strPropertyName) = varPropertyValue

exit_PROC:
    Exit Sub

err_PROC:
    'If property not found, create it.
    If Err = 3270 Then
        Dim prpNew As DAO.Property
        Set prpNew = db.CreateProperty(strPropertyName, varPropertyType, _
                varPropertyValue, True)
        db.Properties.Append prpNew
        Set prpNew = Nothing
    Else
        MsgBox Err.Description, vbCritical
    End If

End Sub
