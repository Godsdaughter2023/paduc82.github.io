ruel mayoya msft say:

Hi Raechel,

If a RPT or a Personal Tag is applied to a special folder, it will always have the option of the “Use Parent Folder Policy” on the Folder Properties, Policy tab.


A DPT will be applied to items that do not have a retention tag applied. That is why the Folder properties have the Policy Tab, but cannot modify the Folder Policy.


Therefore for the folders that were applied with retention policy, it is not possible to remove default “Use Parent Folder Policy” option. This is an expected behavior.

Thus we cannot prevent users from changing the policy, to know if the policy is changed or to change it back, your admin can use Powershell cmdlet Get-RetentionPolicyTag and Set-RetentionPolicyTag. 

Your Admin has the option to remove the Retention Tags on the Default MRM policy, to prevent the users from changing policies between folders.

Difference between a pptx presentation and potx template - Microsoft Community

izzie7id say:

Difference between a pptx presentation and potx template

What is the technical difference between a pptx presentation and potx template.

I know that a template is without content and used to start a presentation, but what changes when you save a presentation as a template?

hansv mvp say:

The difference is most noticeable from File Explorer.

The default action when you double-click a presentation is to open the saved presentation so that you can edit it.

The default action when you double-click a template is to create a new, unsaved presentation based on the template. If you want to edit the template itself, you have to right-click the template and select Open from the context menu. (Or open it from within PowerPoint, of course)

Note: A template is not necessarily empty. If you want to reuse a fixed set of slides in many different presentations, you could save that set as a template. Any new presentation created from the template would already contain those slides.

If you really need the Expand/Collapse markers to go away, from group policy or the Office Customization Tool (OCT), set the Set the minimum zoom level for expand/collapse on-object UI to 110 and Set the maximum zoom level for expand/collapse on-object UI to 90. The little markers to the left of the heading go away. If you right-click, you can still collapse the heading. Hi Keith,

 

These log files also known as Temporary Files. It’s created by the desktop application on the computer to handle necessary user editing. Because a desktop application can’t have multiple documents open at once, it must immediately create a temporary file that allows you to undo any editing.

A program that’s installed on a computer is referred to as a desktop application.
