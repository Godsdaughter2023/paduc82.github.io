xb1r apollo say:

Port Forwarding

 

Login Information

  • Gateway - 192.168.1.1 or 192.168.0.1
  • Username - admin
  • Password - password

Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)

  3. If prompted, enter "admin" for the user name and "password" for the password.

  4. Select Firewall Settings from the menu at the top.

  5. Click on Port Forwarding from the menu on the left.
  6. Click where it says"Add" in red, in the bottom left hand corner. 
  7. For step 1,  select Xbox Live from the dropdown box that says "Select A Service/ Rule"
  8. For step 2, make sure the radio button next to Host is selected.
  9. For step 3, select the MAC Address of the Xbox in the dropdown menu next to where it says "Select A Discovered LAN Device".
  10. Click Apply at the bottom of the page.

Disable Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)

  3. If prompted, enter "admin" for the user name and "password" for the password.

  4. Select Firewall Settings from the menu at the top.
  5. Click on Port Forwarding from the menu on the left.
  6. Find the Xbox in the table that list the applied Port Forwarding rules.
  7. Click on the delete symbol to the right of it. (The delete symbol looks a box with a red "X" in front of it.

hairyfool say:

This is only in the case where hard drives or graphics cards have been added since the system was built.


The recovery process depends on the computer being exactly the same hardware as originally supplied so the instructions are telling you to remove anything that was not part of the computer supplied by HP. If a graphics card or drive has not been added you can't remove them. This is a general guide video and is not limited just to your model of computer.


In practice an all-in-one such as yours only has limited possibilities for extra devices. Before you start just make sure you have the system with a keyboard and mouse plugged in, nothing else. Remove any USB keys, printers or external drives. If you have a special gaming keyboard or mouse I would be replacing them with basic standard ones for the recovery.


Also be certain that ALL files that you want to keep have been saved to another device before you start. This process will delete EVERYTHING with no option to undo the changes.


As Windows has been upgraded you may have to use the second option to press the F11 key during boot to recover the computer. Watch the video again and follow through the steps carefully.
