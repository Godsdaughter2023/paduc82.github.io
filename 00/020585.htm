How do I start the next page? - Microsoft Community

petermayers say:

How do I start the next page?

I have finished Page 1.  How do I move on to the next page?
bob jones aka: cybertaz say:
The simple answer is "Just continue typing." :-) The 'page' will be generated automatically once the volume of content exceeds the capacity of the current page.

Most definitely avoid the 'keep banging on the return key' method. If you do not want to completely fill the current page before going on there are a few better ways to force a new page;

  • Insert> Break - Page Break [Shift+enter / fn+Shift+return on a laptop]
  • Insert> Break - Section Break (Next Page)
  • Format the next page Heading using Format> Paragraph> Line & Page Breaks - Page Break Before (preferably using a Style)

It just depends on a number of variables as to which is the better option. Generally, the 1st option should be avoided but can be used in a 'one-&-done' document such as a short report or letter.

The 2nd & 3rd options are more professional . They provide more flexibility, consistency & flow control, especially if the document will be revised/reissued.

doctorx_256 say:

Excel will not evaluate a cell correctly

Hi,
I am attempting to evaluate value in 3 cells.  If the value in any of the three cells is greater than zero, that value is placed in the cell in which the formula is written.  The problem is that some of them work and some don't, even though the formula is the same but the cell is different.  Here is an example:

=IFERROR( IF(G13 > 0, G13, IF(G15 > 0, G15, IF(G17 > 0, G17,"x"))),"-")

The first two work correctly they being:

IF(G13 > 0, G13, IF(G15 > 0, G15,

It is the last one that doesn't: 

IF(G17 > 0, G17,"x"))


When I rewrite the equation to =IFERROR(IF(G17 > 0, G17,"x") it works correctly

Yet this one works completely:

=IFERROR( IF(L21 > 0, L21, IF(L23 > 0, L23, IF(L25 > 0, L25,"x"))),"-")

Your ideas will be greatly appreciated.

doctorx_256 say: Found the problem.

In using a formula like this =IFERROR((H15/I15),"-0") a TEXT value of whatever is between the quotes is placed in the cell.  When the formula =IFERROR( IF(G13 > 0, G13, IF(G15 > 0, G15, IF(G17 > 0, G17,"x"))),"-") reads that cell, it will evaluate the text to be greater than zero.  So I changed all formulsa like this =IFERROR((H15/I15),"-0") to this =IFERROR((H15/I15),0) and they all work.

Thanks for looking into this for me though.
