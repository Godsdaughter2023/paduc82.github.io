You are probably only removing the "Display Text" and the underlying { HYPERLINK } field is probably still in the document.  You can check that by using ALT+F9 to toggle on the display of the field codes in the document and see if you see such fields.

You could either then select and delete each one.  Or, if you want to delete all of them, you could use a macro containing the following code:

Dim i As Long
With ActiveDocument
    For i = .Fields.Count To 1 Step -1
        If .Fields(i).Type = wdFieldHyperlink Then
            .Fields(i).Delete
        End If
    Next i
End With

A Table of Contents (TOC) is a field inserted with switches to identify which items in the text should be displayed. By default it displays things that are in Heading Styles.
