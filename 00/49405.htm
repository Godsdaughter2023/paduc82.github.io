There is a wealth of settings for a Gmail IMAP account. For a start, in the Gmail interface you can specify which labels to show or hide in IMAP. Then, in WLMail, you can specify which of the available IMAP folders to view or hide. This is done by clicking on the Gmail account name in the folder pane and then clicking Alt-S F.  Refresh the folder list to reflect any changes you may have made at google.com.

You presumably have Save a copy of sent messages in the 'Sent items' folder selected at Options > Send (press Ctrl-Shift-O for Options). Gmail automatically saves a copy of sent messages anyway. The copy that you ask WLMail to save is put in the Sent items folder under storage folders if the one under the account is being used by Gmail.

You can specify which of the WLMail special folders to store on the IMAP server by right-clicking on the account name and selecting Properties. The mapping of WLMail folder names to Gmail labels will be available if you elect to store these folders on the server.

Lastly, you can right-click on a Gmail folder in the folder pane to set its synchronization options. You may want only to see headers for some of them rather than the whole message.
