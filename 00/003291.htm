vincentchoy say:

A domain user having been sync to Azure AD simply means he has an account on Microsoft's "Windows Server" on Azure.

This is the similar to creating a user on your On-Premise AD. If you didn't buy any other software license the user simply can only access On-Premise AD.

In the case of Office365, it is a family of Microsoft Online Products. Since you have Pro-Plus, the possible Online Products your company subscribed to might be either Pro-Plus Standalone, E3,E4 or E5.

If you do not assign him a specific license, then he will not be able to use that particular service.

For example, a company has 300 staff. This company signed up 200xE3, 100XE1. Now E1 does not come with Pro-Plus. How would the system know which of the 200 users will get to use Pro-Plus? Therefore you actually do need to assign the correct license to each individual user before they can use the service.

If you subscribed to a bundle (e.g. E3, E4, E5) apart from assigning a full license to a user, you can also turn on or off a particular product within the bundle. For example you may choose allow a particular user to only access Exchange Online and Pro Plus. When you do so, he will not be able to access sharepoint or Skype for Business.

Hope that clarifies.

sarahkong say:
Hi Christine ... if you embed one file into another then you only have one file to manage. But then if you lose that one file you lose both.
They way I look at it is if a file that you are thinking of embedding will also need to have it's own changes and you want to track those changes separately (versioning) then don't embed.
You can always give links to the other files rather then embed when using SharePoint.
Hope that helps a little.

v.sutharsan say:
Hello Christine,

Based on my personal experience with PowerPoint, I advise you against embedding your file into a Microsoft file as your sole copy. Since this is no right or wrong, it is just my opinion:

Firstly, it may degrade the original quality (like photos/videos). But even if it does not degrade, it may be impossible for you to extract the files as how it was original before it was embedded into.

Secondly, it has a tendency to corrupt if you move between harddrives, particularly thumbdrive. Once it is corrupted, both files are gone.

Thirdly, it is heavier. If the computer has weak hardware and the file it is opening is too big to be processed, you will not be able to even get the first slide working. The process will be terminated, or restarted and forever will not open unless it is fully processed.

So if you put them as separate copies and link them, in summary,

1. They are more liquid (you can easily move around)
2. If one corrupts, you still have access to the others
3. If the overall file is too big, at least your PowerPoint slides still get processed and can continue with your presentation. Then maybe close your presentation before opening video

Hope it helps!

Sutharsan
