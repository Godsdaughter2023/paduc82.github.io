Text-to-speech is the ability of the operating system to play back text in a spoken voice. It's available in English, French, Spanish, German, Japanese, Simplified Chinese, and Traditional Chinese. However, please note that using French as the default system language is only applicable to Windows 7 Ultimate or higher. You may refer to this article. It explains about the Windows version and edition requirements, how to install the language packs, and how to change the default system language.

Since you've mentioned that you already tried downloading its language pack, we would like to know the following so we can investigate the issue:

  • Which edition of Windows 7 are you using? Is it Windows 7 Home & Premium, Windows 7 Professional, Windows Ultimate, or higher? To check it, press the Windows key + R key, type winver and press Enter.
  • If you're using Windows 7 Ultimate or higher, have you set French as the default system language? It is required that you set the computer's default Region & Language to French (France) for the Text-to-Speech functionality to work properly. You may refer to the article provided above on how to set it up accordingly.
