By default, (left) Alt + Shift is a shortcut to change between languages. The action of switching language occurs at the time those 2 buttons are pressedtogether. Because you turn on stickey keys, Windows "locks" your (left) Alt key, then without knowing anything, you press the shift key. Obviously, the language bar will switch languages because Windows already locked your (left) Alt key AND the Shift key. Even without turning on stickey keys, I found that (left) Alt + Shift combination is annoying especially while playing certain games, so I disabled the (left) Alt+Shift sequence.

You can disable or change that key sequence if you like, just type "change keyboard or other input methods" in control panel's search box then open the first item found. Navigate to "advanced key settings" tab, then change the sequence.

For example, I use 3 languages and these are my shortcut key sequences:

  • Switch between input languages: (none) --as I said, [Left Alt + Shift] alone is annoying
  • To English: Left Alt + Shift + 1 
  • To Japanese: Left Alt + Shift + 2
  • To Korean: Left Alt + Shift + 3

Unlike Windows XP or Vista, Windows 7 does not provide such an "extend IME services to all input" checkbox in its keyboard language settings. I assumed that this checkbox is active by default in windows 7. I tested on my wordpad and Korean input works fine.

morrigandisapproves say: I realize this is an old thread, but in case anyone else has this issue I may have a very simple solution for people who has an issue similar to the OP's and also have both Korean and US English keyboard services installed. 

Solution: Uninstall the U.S. English keyboard, make Korean IME default keyboard

How to do this: 
  1. Open 'region and language' in control panel 
  2. Click on 'Keyboards and languages' tab 
  3. Click on 'change keyboards' button 
  4. In the new dialogue box (text services and input languages), select 'EN English (United States), then click 'Remove' 
  5. Select 'Microsoft IME' under 'KO Korean (Korea)'
  6. Click 'move up' if it is not the first service listed (my wife's PC had both 'Keyboard / Korea' and 'Microsoft IME' installed)
Why this works: The Korean keyboard already includes English letter input, so having the US keyboard installed is unnecessary at best and can cause issues at worst (like the OP's problem with the shift key changing input language). 

Another issue caused by multiple keyboards is the han/english button on my wife's korean USB keyboard will not work if the US keyboard is installed in 'text services and input languages.' If you haven't seen a korean language keyboard before, they have extra buttons on both sides of the space key to switch quickly between korean and english text input. It beats the heck out of alt shifting constantly. After removing the US keyboard and moving the Korean IME to the default keyboard layout, the han/english key works fine. 

This 'solution' is very easy to undo in case it's not right for you- just add the US English keyboard back into services.

Hope this helps.
