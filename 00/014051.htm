Hi,

you could use AverageIFS with multiple conditions. If Date is in column A, try this for averaging all 2016 values

=AVERAGEIFS(H:H,A:A,">=1/1/2016",A:A,"<1/1/2017")

and this to average only 2017 values

=AVERAGEIFS(H:H,A:A,">=1/1/2017")

johann eva say:

Hello Glenn,

Internet Explorer requires a periodic reset to restore the browser to its clean state. you need to uninstall and reinstall it. To disable it, follow the steps below:

  1. Press Windows + X.
  2. Click Control Panel.
  3. Click Programs.
  4. Click Programs and Features.
  5. Click Turn Windows Features on or off.
  6. In the Windows features window, uncheck the Internet Explorer program to uninstall it.
  7. Restart your PC to save changes.
  8. Repeat step 1 to step 5.
  9. In the Windows features window, check the Internet Explorer program to install it again.
  10. Wait for the installation to finish.
  11. Restart your PC again.

If you can launch Internet Explorer, you can reset it from the app option itself. Visit this link to change or reset Internet Explorer settings.

Let us know if the solution above works for you.
