ISO File for Windows 8 Pro - Microsoft Community

ajchan say:

ISO File for Windows 8 Pro

Installed Windows 8 Pro by installing directly without saving a copy of the ISO file. How can I get an ISO file to create a USB drive for subsequent re-install?

dpm2009 say:

Today, I went through the process successfully with help of this community. I summarize these steps for you (I may have forgotten some steps, but you got the gist).

 

Download the Setup-.exe from:

http://web.esd.microsoft.com/W8DL/WSEC5B1D8A9DFDFD92DFB736C5B1D8956B5B1D8/Windows8-Setup.exe

 

Click the Setup.exe. The Windows 8 Assistant will start. Enter your Windows 8 key and press Next.

Click Install Now and let the download complete.

The installation files will be stored by default in the folder C:\ESD\Windows.

 

When the "Install Windows 8" message appears, select "Install by creating media" to create a bootable USB flash drive or an .iso (a disk image) file. Chose a folder to store the image e.g. "Windows.iso".

 

Wait for the Assistant to create the bootable media or .iso file. The flash drive must be have 3GB or more of free space available.

 

Burn the .iso file, if that's the path you chose, to a blank DVD.
