You can only setup your integrated graphics to dedicate a limited amount of system RAM to graphics.

The amount varies between machines/motherboards. Some machines can dedicate 2GB of system RAM to 

graphics but others are limited to 1GB or less. I believe most recent laptops allow for 2GB.

The amount of system RAM that can be dedicated to graphics is usually set from BIOS when using a desktop PC,

but I don't know if laptops have another option from within Windows.

The rest of the available system RAM is allocated to games by demand.

Games can still make use of the remaining unused system RAM, as they would if you were using a  dedicated

graphics processor with built in dedicated VRAM, but this is not taken into consideration by games when they

look for dedicated video RAM.

Some games do not run on integrated graphics at all. I believe this is limited to games that only accept the

faster VRAM as a viable RAM for the game.

This game also has a minimum requirement of 8GB System RAM on top of the minimum 2GB VRAM, which makes

it one of the most, if not the most,  RAM intensive games going atm. Which is not really a surprise as this game

is supposed to be MS's poster boy for DX12.

Aside from the RAM issue, the graphics processor also has to be capable of running the game in question.

The minimum requirements for this game specifies 2GB VRAM ( 4GB VRAM recommended).

The AMD R7 250 is the lowest level dedicated graphics processors to meet the games minimum requirements.
