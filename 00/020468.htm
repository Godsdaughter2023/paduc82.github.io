It sounds as if your figure captions include a chapter number. If you press Alt+F9 in the original document to examine the figure number, you'll see something like this:

Figure { STYLEREF 1 \s }‑{ SEQ Figure \* ARABIC \s 1 }

The STYLEREF field is picking up the numbering of the referenced field, in this case Heading 1. If you don't have any Heading 1 paragraphs with autonumbering linked to the style in the document, then you'll get a syntax error for the part that represents the heading number.

Is it possible to highlight all duplicate text in Word, with a filter - Microsoft Community

antonbreed say:

Is it possible to highlight all duplicate text in Word, with a filter for larger words?

I have a large document that includes the names of chemicals (which are quite large: eg, N-ethyl-N,4-dimethyl-3-(4-morpholinophthalazin-6-yl)benzamide); I would like to be able to highlight all duplicate chemical names in the text, so I was wondering if there was a way to perform a wildcard search (such as the one pasted in below) that could also exclude shorter words (ie, those with less than 30 characters); this way, I would not have to manually look through common words like "and" and "chromatography".  Searching each individual chemical name using the Find and Replace function would take too much time. 

Another possible way to search would be to find any word that was only repeated less than five-ish times.

Thanks in advance,

Tony

To find consecutive duplicate words, you could use a wildcard Find, where Find = (<*>) \1


Similarly, for duplicate word pairs, you could use a wildcard Find, where Find = (<*>) (<*>) \1 \2



There is no general Find expression that will locate unspecified duplicate phrases that might be separated by other unspecified phrases.

Cheers,
Paul Edstein
[MS MVP - Word]

jay freedman say:

Try running this macro (see http://www.gmayor.com/installing_macro.htm if needed).

    

Sub HighlightLongWords()
    Dim rg As Range
    Dim rg2 As Range
    Set rg = ActiveDocument.Range

    With rg.Find
        .MatchWildcards = True
        .Text = "[! ^13]{30,}"  ' change the 30 to your minimum
        .Wrap = wdFindStop
        While .Execute

            If rg.HighlightColorIndex <> wdYellow _
              And rg.HighlightColorIndex <> wdUndefined Then

                Set rg2 = ActiveDocument.Range
                rg2.Start = rg.End + 1
                With rg2.Find
                    .MatchWildcards = False
                    .Text = rg.Text
                    .Wrap = wdFindStop
                    While .Execute
                        rg.HighlightColorIndex = wdYellow
                        rg2.HighlightColorIndex = wdYellow
                    Wend
                End With
            End If
        Wend
    End With
End Sub
