Excel :: Find/Search list & Copy Row To Another - Microsoft Community

dinesh bhuwad say:

Excel :: Find/Search list & Copy Row To Another Worksheet

I need help to create a macro for a task. I need to compare the values from Column A in Sheet1 to all of the values in Column B on Sheet2. If a match is found on Sheet2, I need to copy that row and paste in Sheet3. If a match is not found, it should copy corresponding cell value from Column A in Sheet1 to corresponding cell value of Column B in Sheet2. The sequence needs to maintain in Sheet3 as specified in Sheet1.
bernie deitrick say:

Insert a column of formulas into sheet1 - Put a header on the column, and in row 2 enter the formula

=ISERROR(MATCH(A2,Sheet2!B:B,False))

and then copy down to match column A.

This will return TRUE for the values that are not matched, and FALSE for the ones that are.

Apply a data filter to that column, and filter first for FALSE - copy the visible cells and paste at the bottom of Sheet3. Then filter for TRUE and copy the visible cells from column A to the bottom of column B sheet2.

 

You can record a macro when performing all these steps = just make sure that you record entering the column of formulas properly so that it is sized to match your data amd that you use current area to capture all the visible cells.
