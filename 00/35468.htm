The System File Checker tool To determine whether the issue that you are experiencing is caused by one or more system files that are used by Windows Vista, run the System File Checker tool. The System File Checker tool scans system files and replaces incorrect versions of the system files by using the correct versions.

To run the System File Checker tool, follow these steps:
  1. Click Start
    Collapse this imageExpand this image

When the scan is complete, test to see whether the issue that you are experiencing is resolved. The System Restore tool Other troubleshooting methods, such as using the System File Checker tool, may not resolve the issue. If they do not resolve the issue, and the issue that you are experiencing started occurring recently, you can use the System Restore tool. By using this tool, you can restore the computer to an earlier point in time.

Using the System Restore tool may not necessarily help you determine the issue. The System Restore tool uses restore points to return system files and settings to an earlier point in time. You can use it to restore the operating system to a point in time in which you did not experience the issue. When you use System Restore to restore the computer to a previous state, programs and updates that you installed are removed.

To restore the operating system to an earlier point in time, follow these steps:
  1. Click Start

  3. In the list of restore points, click a restore point that was created before you began to experience the issue, and then click Next.
  4. Click Finish.

    The computer restarts, and the system files and settings are returned to the state that they were in at the time that the restore point was created.

lemp say:

StyleRush.dll appears to be related to a Potentially Unwanted Program (PUP) and therefore has undoubtedly been removed by your antimalware software.  See http://www.herdprotect.com/stylerush.dll-68bdfb3887a1b19e19659f668e79ecbaa8975b27.aspx

Assuming that you're getting a "not found" error regarding this file when your computer starts, the reason is that  your antimalware software did not remove the mechanism that was intended to invoke this DLL file.

Based on the webpage linked above, click the Start Orb and type "Task Scheduler" (without quotes) in the Search box and press Enter.  Find the task that runs StyleRush.dll.  You can either click "Task Scheduler (Local)," expand the "Active Tasks" list and scroll through it or try to find it by expanding the "Task Scheduler Library" tree.  See screenshot below:

If you find the task in the list of Active Tasks, double-click on it to open the task.  If you find it by looking through the "library," it should already be open.  Right-click on the task and "disable" or "delete" it (disabling will let you change you mind if you turn out to have selected the wrong thing).

If you can't find the task in Task Scheduler that starts StyleRush.dll, download and run Autoruns. 

Click the "Logon" tab and look in the "Image Path" column for the StyleRush.dll.  It probably isn't in the Logon section, so either click the "Everything" tab and use the File > Search tool or go to the "Scheduled Tasks" section directly.  An idiosyncrasy of the Autoruns search tool is that you have to move the selection line below the "found" item before you "Find Next."


Once you find the line with the offending file, either "Delete" it or (if you want to be safer) uncheck it.  The latter prevents the line from being executed but you can reactivate it later by replacing the check mark.
