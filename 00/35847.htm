taidara say:

Hey there Vault!

I'm not sure if you already tried this, but whenever you speedtest, or ping, for basic IP connectivity, through your PC machine, then please keep in mind that these tests are based on best ping. This means that when you run the application, it'll send out a series of ping requests against a series of pre-configured test servers. The attribute for this process is quite simple, "find me a destination with the shortest route" (Round-Trip Time, RTT).

So, if you haven't already, go to your PC machine and run the application you're using to test for basic IP connectivity (pingtest.net etc.). Now, instead of using the applications default settings, change the test server to a server located in Seattle, WA. This will not give you a 100% accurate test as if you would to ping through your Xbox One, because routing paths between different destinations can be a bit more compliacated. But, it'll give you some bearing, in comparison to pingtests that are based on best ping.

All Skype calls are international calls, and dialled phone numbers must be preceded by the called country's Country Code.

 

The Country Code for US is +1, so to call NY 917 area code, the user should dial +1917.

The Country Code for Tunisia is +216, so dialling 216 results in a call to Tunisia.
