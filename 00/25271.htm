a. user say:

Hi Solution_Pro,

 

There are three ways to upgrade from previous version to the latest: Automatic, in-place or swing migration.

 

The easiest way is to use Automatic upgrade that it will auto update every time we release a new version. Run PowerShell cmdlet Get-ADSyncAutoUpgrade to get the status of it. If it says Suspended it may be blocked by system, to enable it, run https://docs.microsoft.com/en-us/azure/active-directory/connect/active-directory-aadconnect-feature-automatic-upgrade

 

The in-place upgrade is recommend for the tenants who has a single server and less that about 100,000 objects. It's recommended to perform it during the weekend to decrease the potential impact. Please check the details in the following article.

 

For the ones who have a complex deployment or many objects, it's recommended to perform a swing migration (require two servers at least). For example, you have two severs, regard one as active server and another as a staging server. While active server is responsible for the "real work", the staging server is preparing for the new release or configuration. With this alternatively working, impact can be decreased for substantial changes. Please check the details in the following article.

 

Three methods to upgrade AAD connect: https://docs.microsoft.com/en-us/azure/active-directory/connect/active-directory-aadconnect-upgrade-previous-version

 

Download link: https://docs.microsoft.com/en-us/azure/active-directory/connect/active-directory-aadconnect-version-history#117500

scott a. helmers say:

You shouldn't need the converter because Visio 2013 includes an option to save in the older format. Consequently, you can create the .vsd files you need on your own laptop. With a diagram open in Visio 2013, click File>Save As and then select Visio 2003-2010 Drawing (.vsd) from the Save as type drop down.

Note that you might lose some diagram attributes when you downgrade the files. For example, Visio 2013 themes are different than the ones in 2010, and 2013 supports gradients but 2010 does not, so colors and shading might not look quite the same. However, most things should be OK.
