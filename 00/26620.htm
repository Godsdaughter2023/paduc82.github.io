As I said before, you can't have different font settings in the result of a formula, so having some of the text underlined is not an option if using a formula to create the text.

=A1 & " - " & A2

but depending on how the date in A2 is held you might need

=A1 & " - " & TEXT(A2,"d-mmm-yyyy")
or some other number format of your choice

To add the bullet
="• " & A1 & " - " & TEXT(A2,"d-mmm-yyyy")

Bullet characters can be found via Insert > Symbol, but be sure to select the same font as will be used for the rest of the text; scroll down quite a long way and you should find the bullet character.
