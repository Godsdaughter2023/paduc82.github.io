john w. vinson say:

Well... no, you probably DON'T want to do that. Storing the country redundantly is a very bad idea; it would lead to anomalies such as the vendor being in Austria in the main table and in Australia in the child table if someone edits one of the table values.

If you want to keep track of the country as an attribute of the Vendor, and - separately - as an attribute of the part, you can do it, but typically that field would be in only one table.

doug robbins - mvp office apps & services (word) say:

That is an idiotic requirement as the only way that it could be done is when you have finalised, finished, completed your thesis, you will need to modify each of the TOC styles to insert a tab stop at the position that you want the dots to start and then after doing that, which will cause the page number to be aligned at that new tab stop, then place the selection before each page number and insert a new tab space.
