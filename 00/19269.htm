mattbrockway say:

Hi,

I believe that the underlying service plans for 'Office 365 Education Plus for Students' are:

YAMMER_EDU

OFFICESUBSCRIPTION

SHAREPOINTWAC_EDU

SHAREPOINTSTANDARD_EDU

EXCHANGE_S_STANDARD

MCOSTANDARD

So there is no ONEDRIVESTANDARD service plan for you to disable.

word 2013 command line switches - Microsoft Community

p.w.e say:

word 2013 command line switches

win 10 pro

word 2013

I have searched for documentation on word 2013 command line switches and come up with every version but 2013 :(

I want to start word with a custom template and have a VBA 'macro' automatically execute. Word opens with a new blank document, but the code does not run. If I go to the VBA module I can manually execute the routine. Here is the command line

"C:\Program Files\Microsoft Office\Office15\WINWORD.EXE" /a /mmergeDocs /tC:\path to the template\mergeDocuments.dotm

from other versions of Word:

/a should suppress the loading of normal.dotm, in my case I see normal.dotm in the project explorer.

/t should load the template, in my case it does

/m should execute the routine mergeDocs, in my case it does not

jay freedman say:

The documentation is poor, but the fact is that the /m switch can run only macros that are stored in the Normal.dotm template. Because your macros is in the mergeDocuments.dotm template instead, the command line can't run it.

To make this scheme work, either rename the mergeDocs macro as AutoNew, or create an AutoNew macro and insert a statement in it to call the mergeDocs macro. (The latter might be useful if you want to call mergeDocs from several different macros.) Word will automatically execute any macro named AutoNew when a new document is created from the template. More information is at http://www.word.mvps.org/FAQs/MacrosVBA/DocumentEvents.htm.

An alternative to renaming the macro as AutoNew is to move it into the ThisDocument module and name it Document_New.

After making either of these changes, remove both the /a switch and the /m switch from the command line, leaving only the /t switch.
