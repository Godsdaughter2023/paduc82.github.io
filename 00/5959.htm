mike h.. say:
See http://www.cpearson.com/excel/ordinal.aspx
I should have been a little more specific maybe. What I'm trying to accomplish is to have Excel display the ordinal number of the day in a date entered in a cell, but not in another cell. Meaning I want the date entered to display the ordinal numbered day in the same cell it was entered in, not in a seperate cell that contains the formula given at the other website. Hopefully that makes more sense, though I'm still not sure if it's possible.


Hi,

 

You can't do it with a number format but this bit of code will change the date in the same cell to ordinal. Right click the sheet tab, view code and paste this in. Close VB editor. This currently works on A1:a100 which you can change.

 

Private Sub Worksheet_Change(ByVal Target As Range)
Dim MyRange As Range, c As Range
Dim Suffix As String
Dim MyDate As Date
'Change to suit
Set MyRange = Range("A1:A1000")
If Not Intersect(Target, MyRange) Is Nothing Then
    For Each c In MyRange
        If IsDate(c) Then
        MyDate = c.Value
    Select Case Day(MyDate)
        Case Is = 1, 21, 31
        Suffix = "st"
    Case Is = 2, 22
        Suffix = "nd"
    Case Is = 3, 23
        Suffix = "rd"
    Case Else
        Suffix = "th"
    End Select
c.NumberFormat = "dddd" & " d""" & Suffix & """ mmm yyyy"
End If
Next c
End If
End Sub

Sub GetNum()
Dim filenam As String
Dim FileNum As Integer
Dim nSeqNumber As Long
FileNum = FreeFile
filenam = "C:\MyNum.txt"
If Dir(filenam) <> "" Then
Open filenam For Input As FileNum
    Input #FileNum, nSeqNumber
Close FileNum
Sheets("Sheet1").Range("A1") = nSeqNumber
Else
Open filenam For Output As FileNum
    nSeqNumber = 1
    Print #FileNum, nSeqNumber
Close FileNum
Sheets("Sheet1").Range("A1") = nSeqNumber
End If

Open filenam For Output As FileNum    
    Print #FileNum, nSeqNumber + 1
Close FileNum

sheylapérez nazco say:

How do I recover lost work on mac, I saved it but the work is now gone?

Yesterday night I started working on my document in Word, and added some stuff to it, I saved it various time while working on it, when I finished, I saved it and went to sleep. Today when I woke up, I opened the file and some of the work is gone. Not everything that I worked on yesterday, but part of it

john mcghie say:
Sheyla:

There is nothing you can do about THIS instance, unless you had a couple of things turned on before the problem occurred.  But you can ensure it does not happen again:

1)  In Word>Preferences>Save, turn on "Always create backup...".  When that is on, every time you save a document, the previous version is stored in the same folder labelled "Backup of...".  If you have a problem, open the backup, save it as the original, and carry on working.

2)  Enable Time Machine on your system.  Time Machine runs a backup once an hour, from which you can get the document as it existed then.

Hope this helps
