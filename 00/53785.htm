From your issue description I understand that you want to know why the archive attribute get resets between backups in Windows 8

Normal/full backups: All files that have been selected are backed up, regardless of the setting of the archive attribute. When a file is backed up, the archive attribute is cleared. If the file is later modified, this attribute is set, which indicates that the file needs to be backed up.

 

 

Incremental backups: Designed to create backups of files that have changed since the most recent normal or incremental backup. The presence of the archive attribute indicates that the file has been modified and only files with this attribute are backed up. When a file is backed up, the archive attribute is cleared. If the file is later modified, this attribute is set, which indicates that the file needs to be backed up.

Fade Out: Adds a timed fade effect at the end of the audio clip. Within the Fade Out box, directly enter the exact number of seconds you want the clip to fade out.

Hello Bill,

Two possibilities come to mind. The first is that the Archiver program changed the icons, in which case you'd have to reinstall the program, figure out how the icons were changed, reverse the process, and then uninstall the program again. Phew. Its a lot of work, but I had a program do just that very thing to my drive icons and that's the process I had to follow to undo its 'handy work'!

 

The other possibility is that its related to it being an external drive, which don't always play nicely with Windows. Lets start with this possibility first. First off, try this Fix It from Microsoft:  http://support.microsoft.com/mats/Windows_USB_diagnostics . Not fixed? Try this second Fix It:  http://support.microsoft.com/mats/hardware_device_problems/ . Still not fixed? Is the drive being recognized by Windows - right drive size / format type ?

Now that the drive is recognized by Windows, has the icon problem "gone away"? If the answer is still No, then I think you need to look at the Archiver program as a real possibility. The only way to know of course is to install the program a second time and look for a setting (try Configuration) where the program is changing the program icons.
