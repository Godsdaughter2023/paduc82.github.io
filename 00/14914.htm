omer atay say:

So as you can see, the png images + the powerpoint is already 1775KB (not sure if you insert the powerpoint into OneNote as attachment as well). OneNote also stores an XPS document generated from the powerpoint, so with that and other bookkeeping information about the images, it is normal that the OneNote file becomes 2021KB.

 

To reduce this size you can try the following:

- Open the Powerpoint and File > Save as,

- Choose file type PNG and when prompted choose All slides

 

This will save each slide as a PNG file. The files will be slightly smaller in dimensions than the ones in OneNote and may be much more smaller in file size. You can try and see. Then,

- Select all the images from where you saved, and drag into OneNote.

 

This will reduce the overall size because:

- Image files exported by powerpoint are smaller

- There is no XPS file stored in OneNote.

 

However you will loose certain functionality:

- XPS file is vector format and it allows OneNote to render the slide with full fidelity, even if you zoom in a lot. With a PNG file exported from powerpoint, you will see that it is blurry when you zoom in.

- XPS file also contains all the text in the slide in text format, which gives 100% accuracy when you search those text in OneNote. With PNG file exported from powerpoint, OneNote has to depend on OCR, which may not be as accurate, expecially if the text has fancy formatting or overlapping with shapes or background images on the slide.

 

Try these and let me know how much size reduction you get.
