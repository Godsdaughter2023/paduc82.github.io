karl timmermans [outlook mvp 2012-2018] say:

Thank you for looking into this. The pst file format says Outlook Data File (97-2002).

That indicates that your PST file is in the old ANSI format and needs to be converted to UNICODE.

The quickest way to convert this PST file is to perform PST2PST export ensuring the you select the <Root> folder of the PST file as the source and ensure checking the box to include all sub-folders.

Once done,

1) add the newly exported PST file to the Outlook profile (File --> Open & Export --> Open Outlook Data File) 

2) check all the folders in the newly added PST file to ensure it contains everything expected (should be a duplicate of the original)

3) go to File --> Account Settings --> Email Tab --> highlight your POP account -->  click on "Change Folder" at the bottom of the window to point to the <Inbox> in the newly added PST file you just exported

4) Ensure that the new PST file is also designated as the "default" for the profile if the original was so designated (Account Settings --> Data Files tab) 

5) Remove the old PST file via Account Settings --> Data Files tab 

You should now be good to go

This will be lengthy but first, reboot the tablet and get to the desktop.  You will need a micro-SD card or a USB pendrive for data storage.

Next, open the File Manager and go to "This PC" and double-click on "Drive C:".

Next, double-click on "Users" folder and look for your account name.

Next, double-click on your account name and copy all of those folders to that storage.  You can also use an external hard disk if you have one that works with the tablet and copy those folders in a folder called "backup" so that you will know what the files it contains are.

When finished, slide open the Charms Bar menu and click on "settings", then "Change PC settings", and then "Update and recovery".  Go to "Recovery" and go to the second option "Remove everything and reinstall Windows".

The only way to get rid of malware/viruses is to wipe out the tablet and start all over.  However, you just backed up all your data which is still good and that includes your web favorites.
