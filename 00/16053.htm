How to add watermark for multiple files? - Microsoft Community

alanstrachan_546 say:

How to add watermark for multiple files?

I have created multiple word docs (1700 of them) and recently have been asked to add a watermark to all of them .  They are in a folder with multiple sub folders.  Is there a way to do this other than manually?

 

Thanks,

graham mayor say:
You could do this with a batch processing macro, but try it on some copies of the documents first! The macro 'BatchProcess' opens each document in turn. It then opens each header of each section and calls a function to insert the watermark building block - here "CONFIDENTIAL 1" - then saves the document and logs the document name in another document (useful if there is a problem).

The function expects to find the building block in BuildingBlocks.dotx, which is where the built-in building blocks are stored. If yours is stored elsewhere, move it to this template using the building blocks editor, and replace "CONFIDENTIAL 1" where indicated in bold type, with your building block name. Note this is case sensitive!

The macro works on ALL documents in the selected folder (so you might want to create a test folder to try it with a few documents).

The macro will crash if the documents are protected! If you need to work with protected documents, you will need extra code, but we will cross that bridge if we come to it.

See http://www.gmayor.com/installing_macro.htm

Option Explicit
Sub BatchProcess()
Dim strFileName As String
Dim strPath As String
Dim oDoc As Document
Dim oLog As Document
Dim oRng As Range
Dim oHeader As HeaderFooter
Dim oSection As Section
Dim fDialog As FileDialog
Set fDialog = Application.FileDialog(msoFileDialogFolderPicker)

With fDialog
    .Title = "Select folder and click OK"
    .AllowMultiSelect = False
    .InitialView = msoFileDialogViewList
    If .Show <> -1 Then
        MsgBox "Cancelled By User", , _
               "List Folder Contents"
        Exit Sub
    End If
    strPath = fDialog.SelectedItems.Item(1)
    If Right(strPath, 1) <> "\" _
       Then strPath = strPath + "\"
End With

If Documents.Count > 0 Then
    Documents.Close savechanges:=wdPromptToSaveChanges
End If
Set oLog = Documents.Add
If Left(strPath, 1) = Chr(34) Then
    strPath = Mid(strPath, 2, Len(strPath) - 2)
End If
strFileName = Dir$(strPath & "*.doc?")

While Len(strFileName) <> 0
    WordBasic.DisableAutoMacros 1
    Set oDoc = Documents.Open(strPath & strFileName)
    '
    'Do what you want with oDoc here

    For Each oSection In oDoc.Sections
        For Each oHeader In oSection.Headers
            If oHeader.Exists Then
                Set oRng = oHeader.Range
                oRng.Collapse wdCollapseStart
                InsertMyBuildingBlock "CONFIDENTIAL 1", oRng
            End If
        Next oHeader
    Next oSection
    'record the name of the document processed
    oLog.Range.InsertAfter oDoc.FullName & vbCr
    '
    oDoc.Close savechanges:=wdSaveChanges
    WordBasic.DisableAutoMacros 0
    strFileName = Dir$()
Wend
End Sub

Function InsertMyBuildingBlock(BuildingBlockName As String, HeaderRange As Range)
Dim oTemplate As Template
Dim oAddin As AddIn
Dim bFound As Boolean
Dim i As Long

bFound = False
Templates.LoadBuildingBlocks
For Each oTemplate In Templates
    If InStr(1, oTemplate.Name, "Building Blocks") > 0 Then Exit For
Next
For i = 1 To Templates(oTemplate.FullName).BuildingBlockEntries.Count
    If Templates(oTemplate.FullName).BuildingBlockEntries(i).Name = BuildingBlockName Then
        Templates(oTemplate.FullName).BuildingBlockEntries(BuildingBlockName).Insert _
                Where:=HeaderRange, RichText:=True
        'set the found flag to true
        bFound = True
        'Clean up and stop looking
        Set oTemplate = Nothing
        Exit Function
    End If
Next i

If bFound = False Then        'so tell the user.
    MsgBox "Entry not found", vbInformation, "Building Block " _
                                             & Chr(145) & BuildingBlockName & Chr(146)
End If
End Function

Hi Pat,

 

Thank You for choosing Microsoft Community. I will assist you by providing the steps to enable Import/Export option which is missing in Outlook 2007.

 

Is the Import/Export option disabled (Greyed Out) or is it missing from the File option?

 

Follow the steps:

 

Method 1:

Open Outlook in safe mode and check if you can see the Import/Export option. Follow the steps:

Try opening Outlook in safe mode and check. Follow the steps:
1. Click on Start -> All Programs.
2. Go to Accessories -> Click on RUN.
3. Type Outlook /safe (leave a space after Outlook) -> Click OK.

 

If Outlook safe works, then follow the steps:
Important: Ensure to back up the registry before changing the registry keys:
http://windows.microsoft.com/en-us/windows7/Back-up-the-registry

 

In the registry editor window browse to the locations mentioned below and delete the Add-ins keys.

 

HKEY_CURRENT_USER\Software\Microsoft\Office\Outlook\Addins
HKEY_CURRENT_USER\Software\Microsoft\Office\12.0\Outlook\Addins
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\12.0\Outlook\Addins
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\ Outlook\Addins

 

Once the keys you have deleted the add-ins keys, Open Outlook and check if it works fine.

 

Method 2:

 

If the above step does not work, then repair Office 2007 suite, restart the computer and check. Follow the steps provided in Method 2 (Detect and Repair feature) in the following article to repair Office 2007 suite.
http://support.microsoft.com/kb/924611

 

I hope any of the above methods fixes the issue. If not, then reply and I will be happy to help you.
