You're able to use the same product key.

These are the rules:

1. If yours is a retail copy, you can install it and reactivate it as
many times and as often as you want to or need to. You can change
hardware and even change it to a completely new machine, as long as
it's on only machine at a time.

2. If yours is an OEM copy, you can also install it and reactivate it
as many times and as often as you want to or need to. You can change
hardware (such as a disk drive), but you can not move it to a
completely new computer. Its license ties it permanently to the first
computer it's installed on and it may never legally be moved to
another.
