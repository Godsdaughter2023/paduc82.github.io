antonio rodriguez olmos say: Of course. I used this:

1) Go to Regedit (Windows-->Run or Ctrl+R --> regedit.exe)
2) Then go to: 
User Key:[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\ Internet Settings]
3) Create:  
Value Name: DisablePasswordCaching
Data Type: REG_DWORD (DWORD Value)
Value Data: (0 = default, 1 = disable password cache)
4) Reboot

I tried the other fixes like Protected Storage (deprecated, non-existant Service in Windows 8) or IE settings but couldn't finish them, so I'm 99% sure that's what solved it.
