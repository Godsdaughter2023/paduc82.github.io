2- Look for a Windows 10 driver for the device on the device manufacturer's website.

3- If the device in question is a USB port on the motherboard you may need to re-install the motherboard drivers, if they are compatible with Windows 10. Your computer manufacturer will have these drivers and instructions for installing them.

robertjwin say:

Type shell:sendto in File Explorer's address bar and press Enter. Anyway you don't even need to waste time placing shortcuts in SendTo because Winamp itself can add context menu items to folders.

As per the help page here, go to Winamp's Preferences > General Preferences > File Types > Explorer Settings and enable Show Winamp in folder context menus in Windows Explorer (Official description: Select this if you want "Play in Winamp" etc. added as an option in the Windows Explorer right-click context menu for folders.)

If you have the Jump To File Extra plugin (gen_jumpex.dll) installed you can also look at Shell Options > Windows Shell File Association Options > Windows Explorer Menu Control to further customize the folder context menu items.
