erdeslawe say:

Microsoft is not alone in implementing 'User Account Controls' to prevent unwary users, or websites you may visit, from installing random software.  Apple does the same thing.

The fact is that that Windows is advising you that it cannot identify either the source of the software or the authenticity of the publisher's certificate and therefore will not allow the installation without an Administrator's intervention.  If you are the Administrator, you need to verify the authenticity of that piece of software and can over-ride the block by adjusting the UAC settings (preferably, temporarily).

It's for your own safety/system security that Windows carries out such checks and it would be totally irresponsible of Microsoft if such protection was not built in to the operating system.  It's exactly the same as running Firewall and Anti-Virus software which block users from downloading from web sites that have been flagged as untrustworthy.

Or, perhaps you would prefer your System to be totally unprotected. In which case simply disable UAC and  remove all Anti-Virus, Firewall and other security software; and simply take your chances.
