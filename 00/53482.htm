folder location - Microsoft Community

eart say:

folder location

how come my document folders don't stay in their "open folder location"? when I want to customize a fold with a different icon I always have to click properties and "open folder location" to get the tab "customize". How can I get the "open folder location" stay as my default and not preversions? 
meghmala say:

Hi Eart,

 

While accessing the folder in the documents you are actually in a library.

Libraries are where you go to manage your documents, music, pictures, and other files. You can browse your files the same way you would in a folder or you can view your files arranged by properties like date, type, and author.

In some ways, a library is similar to a folder. For example, when you open a library, you'll see one or more files. However, unlike a folder, a library gathers files that are stored in several locations. This is a subtle, but important, difference. Libraries don't actually store your items. They monitor folders that contain your items, and let you access and arrange the items in different ways. For instance, if you have document files in folders on your hard disk and on an external drive, you can access all of your document files at once using the Document library.

So if you need to customize a folder you will have to go to the exact location and then change the icon there. You cannot do it in a library as the actual folder is not there. You cannot even set the open folder location as default. You will have to follow the actual path to customize the folder as this is by design.

Hope this helps. Let us know the results.
