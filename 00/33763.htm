If C: and D: are on separate physical disks  (Disk 0 and Disk 1) they cannot be merged.

If you are technically inclined you can swap the disks. Otherwise, time to buy a larger hard drive for C:

To swap the disks:

1- Backup the C: drive and the D: drive to an external hard drive. You'll need third-party image backup software to do this.

2- Format both disks. You'll need to boot from some type of repair media to do this.

3- Swap the disks.

4- Restore the backup of your C: drive to the larger disk. Restore the backup of your D: drive to the smaller disk.

a. user say:

A restore point is not created before a major update (e.g., Anniversary Update, Creators Update). In fact, all existing restore points are erased by the system after a major update. You would not want to restore an old restore point after a major update.

Note: A restore point is not a backup. Restore points don't include your personal data.  Only a backup is a backup.

"why don't they advise you will loose your data before upgrading?"

... because upgrades don't cause loss of data. If you lost data following an upgrade that's a sign that there was something wrong with your computer.

Your personal data (documents, pictures, etc.) should be stored in the default folders of your user profile. That's how Windows was designed. Those folders aren't touched by Windows Update. If you store personal data anywhere else, e.g., the desktop, Public folders, built-in Administrator, etc., it can get erased during an update. The same can happen if your user folder has been damaged to the point where it is unusable.

Even when you do everything right, a power blackout or virus attack, among other reasons, can cause you to lose data. Backing up is quick and easy. This is why people are supposed to backup every day.
