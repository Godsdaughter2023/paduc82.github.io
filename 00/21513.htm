ffoa say:

Which formula limits the number of characters in a cell?

Which formula limits the number of characters in a cell? I have a column of data that I want to limit to only the first 5 characters, but can't remember or find the formula to make that happen. Thanks CG
j.e. mcgimpsey say: Formulas can't modify the contents of other cells (or their own, really) - they simply return a value.

If you want the first 5 characters of cells in column A to be displayed in Column B, then you can use

B1:     =LEFT(A1, 5)

and copy down as far as needed. That won't change the values in column A, though.

If you want to limit the user's entry to 5 characters, you CAN use Data Validation:

Select the cells you want to limit. Choose Validation from the Data menu. In the Data Validation dialog, choose "Text Length" from the "Allow" dropdown. In the "Data" dropdown, choose "less than or equal to", and in the "Maximum" inputbox enter 5.

Note, however, that Data Validation only works on keyboard entries - it doesn't survive a Paste. To get around that requires an Event Macro...
