Items are not marked as Read automatically when Unread filter is used - Microsoft Community

alex rizeanu msft say:

Items are not marked as Read automatically when Unread filter is used in Outlook Web App

Technical Level : Intermediate

Summary

PROBLEM:

 

When a Microsoft Office365 user filters messages using the "Unread" filter in Outlook Web App, after reading a message, the message is not automatically marked as read.


Details

CAUSE:

 

This is by design. The functionality to mark an Unread message as Read was never implemented when using the Unread filter as otherwise it would remove the message from the list view.

 

 

SOLUTION:

 

Use either right click to "Mark as read" a message or the "Mark as read" shortcut on the top right corner of the message.
