what is quarnite file in microsoft security essentials? - Microsoft Community

dvspatel say:

what is quarnite file in microsoft security essentials?

what it is and why it should be clear on periodically ?
kosh vorlon say:

Hi dvspatel,

 

I'm going to assume you mean a quarantined file.

 

It is a file that was detected as a threat by MSE during a scan and the default action for that threat was to send it into quarantine.  This is a place where the file cannot harm your system and you have time to check to make sure it is really a threat and not a "false positive".  The default is to automatically remove items from quarantine and from your system in 1 month.  In the meantime, you can let it sit there doing no harm, or restore it if it is a "false positive", or remove it if you prefer to get rid of it rather than let it sit in quarantine.  I allow them to remain in quarantine until they are automatically removed just in case it turns out it was a "false positive" - then it's easier to recover than if it is removed (if it can be recovered after being removed).

 

There's no need to clear it periodically as it does so on its own automatically.

 

I hope this helps.

 

Good luck!

3. Scroll down the settings list and tap on language. If you are unable to read the text, language is the 31st item counting down from the top, or the 18th item counting up from the bottom.
4. Any installed languages will show in screen. Tap + add languages.
5. Select the languages you wish to add by ticking the respective boxes, then tap the tick at the bottom to proceed.
6. Some languages require additional information to be downloaded. Tap yes to proceed (the left hand white box).
7. You must now restart the phone to complete the change. Tap the white box below the + symbol to restart phone.
8. When the phone restarts, it will now be in your chosen language.
