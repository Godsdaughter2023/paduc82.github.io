So the answer I gave almost a year ago still stands: if you sign in in WLMail, then all you have to do on the new computer is sign in with the same account.

If you don't sign in, you'll find the registry key holding the blocked senders list at

HKCU\Software\Microsoft\Windows Live Mail\PerPassportSettings\0\Junk Mail\Block Senders List

You can export the key and import it on another machine.
