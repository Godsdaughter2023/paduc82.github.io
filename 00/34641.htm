A licença é o direito de poder usar um determinado conteúdo na Xbox Live. Tudo o que você comprar virá com duas licenças: a local e a do sistema.

Qual a diferença entre a licença local e a licença do sistema?

A licença local está relacionada ao Xbox 360 ao qual você baixou algum conteúdo pela primeira vez. Simplificando, ela dá direito a todos os perfis desse Xbox 360 usarem o conteúdo baixado.

Diferentemente da local, a licença do sistema está relacionada à conta que efetuou a compra de algum conteúdo. Resumindo, ela te dá direito a usar o conteúdo que você comprou em sua conta, em qualquer lugar. Isso pode ser feito baixando o seu perfil em outro Xbox 360.

Então o conceito dos tipos de licenças pode ser esquematizado dessa forma:

Licença Local -> Pertence ao primeiro Xbox 360 a baixar algum conteúdo.

Licença do Sistema -> Pertence ao perfil que compra algum conteúdo.

assocom say:

Microsoft Loopback Adapter not available in Windows XP Mode under Settings

I have installed a Microsoft Loopback adapter on my Windows 7 Host machine. When running my Windows XP mode machine the Loopback Adapter is not available as a option under my network settings for the virtual machine
azeez n say:
Hi assocom,
 
You probably do not have loopback adapter installed in Windows XP.
 
Follow the steps to install loopback adapter in Windows XP.
 
1. Click Start – Search for cmd, right-click cmd and select “Run as Administrator”
 
2. From the command prompt, run “hdwwiz.exe“. This should launch the “Add Hardware Wizard“. Click Next.
 
3. Select “Install the hardware that I manually select from a list (Advanced)” and click Next.
 
4. Now from the list select Network Adapters and click Next.
 
5. Under Manufacturers, select Microsoft and select “Microsoft Loopback Adapter” under Network Adapter and click Next.
 
This should start the installation. Click Finish when the installation completes.
 
To confirm the loopback adapter installed, right-click Computer, select properties. Click Device Manager and expand Network Adapters and you can see the Loopback adapter
 

Additional Information
After you've installed loopback adapter, you may try this :
 
1) Create a loopback adapter in the *host* and set it to a fixed IP address.
 
2) In the VPC settings for guest set the networking to point to
"Local Only". This makes the virtual network private.
 
3) Inside each guest set the IP address of the Ethernet NIC (*not* the loopback adapter)
 
Now you should be able to ping between the guest and also to the host and back.
