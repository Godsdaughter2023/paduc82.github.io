Formula in cell H4 - returns the minimum value:

=MIN($B$2:$F$10)

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell I4 to return the store:

=INDEX($A$1:$F$1,SMALL(IF(($B$2:$F$10)=$H$4,COLUMN($B$2:$F$10)-COLUMN($B$2)+1),1)+1)

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell J4 to return the product:

=INDEX($A$2:$A$10,MATCH($H$4,INDEX($A$2:$F$10,,SMALL(IF(($B$2:$F$10)=$H$4,COLUMN($B$2:$F$10)-COLUMN($B$2)+1),1)+1),0))

=INDEX($A$1:$F$1,SMALL(IF(($B$2:$F$10)=MIN($B$2:$F$10),COLUMN($B$2:$F$10)-COLUMN($B$2)+1),1)+1)&" "&INDEX($A$2:$A$10,MATCH(MIN($B$2:$F$10),INDEX($A$2:$F$10,,SMALL(IF(($B$2:$F$10)=MIN($B$2:$F$10),COLUMN($B$2:$F$10)-COLUMN($B$2)+1),1)+1),0))

For second lowest value, enter below formula in cell J3 & copy down:

=IFERROR(SMALL(IF($B$2:$F$16<>0,IF($A$2:$A$16=$I3,$B$2:$F$16)),2),"NA")

=IFERROR(INDEX($B$1:$F$1,MATCH($J3,OFFSET($B$1:$F$1,MATCH($I3,$A$2:$A$16,0),),0)),"NA")
