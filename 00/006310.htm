jay freedman say:
The code I posted was in response to your post about inserting the built-in Blank (3 Columns) building block. It's not at all the same as the macro you would need for inserting 'Sample Text1' and 'Sample Text2' with different fonts and sizes.

As I said before, you're very unlikely to get anything useful by recording a macro for this kind of work. You must either learn enough VBA to write it from scratch, or get someone with experience to write it for you.

The macro below creates the header and footer you requested in your first post. The footer is the same as the one Suzanne described. Making nested fields in VBA is complex, and can't be done directly in the footer, so the macro constructs the field in the main text and then cuts and pastes it to the footer.

Sub InsertSpecialHeaderAndFooter()
    ' Center aligned header in first page should read
    ' "Sample Text 1" with font size 12 CALIBRI
    ' and the right aligned header should read
    ' "Sample Text 2" with font size 16 CALIBRI.
   
    Dim rg As Range
   
    With ActiveDocument.Sections(1)
        .PageSetup.DifferentFirstPageHeaderFooter = True
        Set rg = .Headers(wdHeaderFooterFirstPage).Range
        With rg
            .Font.Name = "Calibri"
            .Font.Size = 12
            .MoveEnd wdCharacter, -1 'exclude paragraph mark
            .Text = vbTab & "Sample Text 1" & vbTab
           
            .Collapse wdCollapseEnd
            .Text = "Sample Text 2"
            .Font.Size = 16
        End With
    End With
   
    ' Center aligned footer in last page should be
    ' "Sample Footer Text1" with font size 8 CALIBRI
    ' and the right aligned footer should read
    ' "Sample Footer Text2" with font size 8.
   
    'Insert dummy para at end of document
    ActiveDocument.Range.InsertAfter vbCr
    Set rg = ActiveDocument.Range
    rg.Collapse wdCollapseEnd
    rg.Select
   
    ' Insert nested field (see http://www.word.mvps.org/FAQs/MacrosVBA/NestedFieldsWithVBA.htm)
    With Selection
        .TypeText "IF "
        .Fields.Add Range:=Selection.Range, Type:=wdFieldEmpty, Text:="PAGE ", _
            PreserveFormatting:=False
        .MoveRight Unit:=wdCharacter, Count:=1, Extend:=wdExtend
        .MoveRight Unit:=wdCharacter, Count:=1
        .TypeText Text:=" = "
        .Fields.Add Range:=Selection.Range, Type:=wdFieldEmpty, Text:="NUMPAGES ", _
            PreserveFormatting:=False
        .MoveRight Unit:=wdCharacter, Count:=1, Extend:=wdExtend
        .MoveRight Unit:=wdCharacter, Count:=1
        .TypeText """" & vbTab & "Sample Footer Text 1" & vbTab & "Sample Footer Text 2" & """"
       
        .HomeKey Unit:=wdLine, Extend:=wdExtend
        .Fields.Add Range:=Selection.Range, Type:=wdFieldEmpty, PreserveFormatting:=False
        .Fields.Update
        .EndKey Unit:=wdLine, Extend:=wdExtend
        .MoveEnd Unit:=wdCharacter, Count:=-1
        .Cut
        ActiveWindow.View.ShowFieldCodes = False
        ActiveDocument.Paragraphs.Last.Range.Delete
    End With

    With ActiveDocument.Sections(1)
        Set rg = .Footers(wdHeaderFooterPrimary).Range
        With rg
            .Font.Name = "Calibri"
            .Font.Size = 8
            .Paste
        End With
    End With
End Sub

larry diver say:

Access 2003 random test generator

Greetings, novice Access 2003 user, but wish to generate a simple randomized 50 question test from a data base of say 100 questions.  Thanks.  Larry
hansv mvp say:

Click Query Design in the Create tab of the ribbon.

Add your table.

Add * from the field list to the query grid.

In the next column, create a calculated field:

 

Rand: Rnd([FieldName])

 

where FieldName is the name of a number field in your table (it doesn't matter which, but an AutoNumber field is ideal for this purpose).

Clear the Show check box for this column.

Click in an empty part of the top pane of the query design window.

If necessary, activate the Property Sheet.

Set the Top Values property to 50 (this value is not in the dropdown list, but you can type it yourself).

Each time you "run" the query, it will select 50 records at random.
