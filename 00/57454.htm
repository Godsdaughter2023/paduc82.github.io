On a Windows 10 PC, all user accounts in the same group have the same permissions. Thus, all Administrators accounts have the same permissions and all Users accounts have the same permissions. With respect to permissions, it makes no difference whether a user account is a local account or a Microsoft account.

The distinction between a local account and a Microsoft account is that a local account is only accessible from the computer on which it was created, whereas a Microsoft account - which is simply Microsoft's branded version of a cloud account - is accessible from any device that can connect to the internet.
