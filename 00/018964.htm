andreas killer say:
You can do that with a VBA code. As the name states Power Query is a Query Object, but not in the sheet, the Query is inside the table resp. the ListObject.

So if we assume that a sheet has a Table/ListObject which is created by Power Query, you can access it from within the codemodule of the sheet with a code like this: 

Sub Test()
  Dim Q As QueryTable
  Dim L As ListObject
  Set L = Me.ListObjects(1)
  Set Q = L.QueryTable
  'Note:
  '  If you set BackgroundQuery argument to False the next line is executed after the refresh is done.
  '  Otherwise the code continues and the query updates the data in the background (multi tasking)
  Q.Refresh False
End Sub


The rest is up to you at which point you want to update the query, that depends on the layout of the file and how it should work.

You can use the Worksheet_Change event to get noticed that the source table has changed and update the query immediately.
Or (if the query is in a different sheet) use the Worksheet_Activate event if you switch to that sheet.

Andreas.
