andre da costa say:

Windows To Go has certain technical requirements, both hardware and software.

1. You need to be running Windows 10 Technical Preview for Enterprise.

Windows 10 Technical Preview for Enterprise | TechNet ...

Because its a preview, I recommend you setup it in a dual boot configuration and not upgrade a production installation of Windows such as Windows 8/8.1 or Windows 7/Vista with it.

Learn more:

http://answers.microsoft.com/en-us/windows/forum/windows_tp-windows_install/windows-10-technical-preview-issue/e578e8f6-c317-4e00-87e5-11fd0baed81a

2. You need a USB 3.0 capable thumbdrive that supports Windows To Go.

3. You need to use it on a system specifically designed for Windows 7 or Windows 8/8.1

4. Review the following article for instructions how to create a Windows To Go setup:

Windows To Go: Screenshots | ZDNet

Other wise, if you do not meet these requirements, then the answer is no.

reza ameri say:

There are two scenarios:

1) Someone wants to boot Windows from USB instead of CD/DVD and install Windows using a USB drive and install on a system like Laptop or Desktop.

or

2) Someone want to create a Windows on USB (Windows To Go) and whenever insert the USB in any PC (which meet minimum requirements to run Windows), then be able to boot Windows from USB.

It doesn't work in way that you copy Windows files on a USB but if you have already installed version of Windows (e.g. on your PC or Laptop), you should insert a USB and run Windows To Go and follow steps to create it for you.

gleam2 say:

Excel is a good choice.

You can modify the first one "Employee Shift Schedule" to suit your needs:

Delete B5 "Monday" (Ignore " marks in all enter data instructions)

Type in C5 "Monday", in D5 "Tuesday" and so on up to I5 "Sunday"

Select J5 to M5 and then Delete/Delete Sheet Columns

Delete rows 12 down

It may help to have the date shown:

In row 6 delete B6 "Kelly F"

in C6 enter the date for the first Monday eg (27/12/2010 or 27/12/2010 if you are in the US)

In D6 enter "=C6+1"

Move the cursor to the bottom right of D6 until it changes to a small plus sign. Press the left mouse down and hold it down and drag across to I6. The cells will

now be filled with the dates.

Enter your list of names starting at B7 replacing "Tom Y"

Select cells B5 to the cell in Column I in the last row. Right click and select Copy. (If the last row with with a name is 20 then select B5:I20)

Leave one blank row and click in column B to select the cell.(B22 in the example) Right click and select paste.

The dates will be wrong: select the first new Monday (C23 in the example) and change this to "=C6+7" The dates will then be correct for the following week.

If you copy and paste the second week of data you should find that it gives the correct dates for the third week.

If the number of people changes, or you delete the first week of data, you may find that you need to revise the formulae, but hopefully it will

be reasonably clear what to do.
