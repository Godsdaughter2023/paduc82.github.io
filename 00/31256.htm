If you arrange it so that you can see these columns:

Name, Nickname, First name, Last name, Company, email addresses

then the Name field will be constructed from data in the remaining columns. It will consist of the first match found when scanning those fields in this order:

  1. Nickname
  2. First name + Last name (in the order specified at Sort by in the Contacts window)
  3. First name or Last name
  4. Company
  5. E-mail address

When you see a contact picker list (eg. after clicking To... or typing in the To... box in a message compose window), you will see a list of Names in alphabetical order.

anne ros say:

Hi Shailesh,

Reducing the resolution will also reduce the size of the image displayed on the screen. Since your video card's maximum resolution is 1980 x 1080, which is a 4K display, if it is reduced to 1280 x 720, which is high-definition, then the displayed screen will really shrink to fit the set resolution. You can have full screen by stretching the image, though it will not be as clear as it was when not in full screen using a lower resolution. Check from the settings if the background is set to Stretch to have a full screen view. To check, go to Settings > Personalization > Background > Choose a fit > Stretch.
