Private Function IsOpen(ByVal FileName As String) As Boolean
  'True if FileName is open
  Dim slot As Integer
  If Dir(FileName) <> vbNullString Then
    On Error Resume Next
    slot = FreeFile

jose maya say: Follow these steps that can help you

1. With Excel closed: Open Windows Explorer >> Folder options
2. Enable View Hidden Files >> Un-enable Hide protected operating system files.
3. In the Windows Explorer, use the search option to find the file named "excel14.xlb" (you will usually find it in the path C: \ Users \ user \ AppData \ Roaming \ Microsoft \ Excel).
4. Once the file has been found in the browser of Windows Explorer, go to the destination of the file and change the name excel14.xlb to excel14.xlb.old
5. You can re-set the folder settings to how they were initially (Do not show hidden files and Hide files protected by the OS).
6. Open Excel and enjoy again the correct operation of Right Click


Good luck
