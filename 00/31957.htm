considerate_guy say:

There are two things that Movie Maker will save:

1) A project file - when you go to File > Save, what you are saving is the project file.  The best analogy is to think of your project file as a recipe.  It lists all the ingredients used (videos, photos, music) along with all the instructions on how to mix them together (trimming, splitting, order, effects applied, transitions applied, captions added, etc.)  The project file lets you go back and make edits to the content, the order, captions, etc.  You should always save your project file.

2) A movie file - when you go to File > Save Movie, what you are saving is the movie file.  Using our prior analogy, this is like baking the cake.  Movie Maker pulls together all the ingredients from your hard disk, mixes them together as instructed and creates the finished movie.  As you have noticed, this process is time-intensive.  It takes awhile for Movie Maker to encode all the video into a new .wmv file.

Okay, now that I got that out of the way, let me answer your questions:

a) Can a "movie" be re-edited?  Yes, if you have your original project file you can go back and modify the recipe.  You can change the order of clips, add new ones, change transitions and captions, etc.  When you like the new settings, you can either save the project with a new name (so you have both versions) or just keep the latest version if you no longer need the first.  Similarly, you can then save a movie in your desired format.

b) Can a movie be saved in a different format?  Yes, you can go back and choose to save your movie with different settings.  However, as you noticed, this always takes awhile because Movie Maker has to essentially start again with baking your cake.  Think of it as wanting two cakes and each one has to bake separately.

c) Can you save a movie again after a bit of tweaking?  Yes, if you are changing things in your project, you can always decide at any point to create a movie.  It takes time, but if after baking your cake, you can always go back, change the recipe a little and then bake a different version of your cake.  As aways, this step of saving your movie does take time.

d) Do I have to save my movie again each time to get different formats, extra copies?  In general, the answer to this is "yes".  If you want different versions of your final movie (some as a DVD, some as a low resolution movie on a flash drive, etc.) you would need to save it in each of these different formats.  However, what you might try to do is to standardize on one format (say a good resolution .wmv file to your computer).  You could then upload that movie to YouTube and just give people the link.  Or you can use that same file and put it on a USB drive that people could view on their PC.  Or you could use Windows DVD Maker to take that .wmv file and make it into a video DVD that could be watched via a standard DVD player.

You have a big heart to be doing this for others.  I've done a lot of similar work with filming school activities and everyone loves to get a copy for their own purposes.  But I don't think they always realize how much work it is to create say 30 cakes, but slightly different for each person.

eatingboat68 say:

StalkerStang50, sorry to hear about your issue.  Here is my suggestion for you to try.  Since routers have a firewall to protect you from outsiders you need to set certain ports to listening mode. That means you need to open ports that Xbox Live Server requires. The ports are 53, 80, 88, and 3074.  You can't use Port Range Forwarding since it will only set those ports into listening mode to the IP address you set it. So if you use Port Range Forwarding it will only be open/available to one console(the one using the IP address). Use Port Range Triggering. Disable Port Range Forwarding and the DMZ. Make sure UPnP is enabled. To solve the MTU problem set your MTU size to 1364. This settings will work even if you have one or multiple consoles running behind the router. Just remember that you need to make sure first that you have a good connection from your ISP (Internet Service Provider). Online gaming consumes a lot of bandwidth. So just couple things to do. Trigger the ports at the Port Range Triggering, set MTU to 1364 and make sure UPnP is enabled disable your DMZ.  Try this and let me know if it works for you :)  Thanks
