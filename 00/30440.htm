System Thread Exception Not Handled error in mostly arises due to incompatible or old graphics card drivers. System Thread Exception Not Handled error is often displayed along with the Blue Screen of Death (BSOD).

After the Blue Screen of Death is displayed with System Thread Exception Not Handled error, even if the computer is powered off and then powered back on, it keeps getting restarted after every few seconds.

In order to fix this issue, the computer must be restarted in safe mode, and the problematic graphics driver must be uninstalled.
