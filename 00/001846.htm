charles kenyon say:

Confusion reigns!

See The Distinction between the User Templates Folder and the Default Personal Templates Folder Word - 2013 and later. Here is a discussion on this Answers site about it.

The user templates folder and the custom templates folder can be different folders. By default, they are different. It is the custom templates folder that shows up under Personal in File > New.

The user templates folder holds the normal template and was the default location for storing new templates up until Word 2007.

You can set them to be the same. The setting for the Custom templates is under File > Options > Save.

I set them to be the same folder in my file options.

jay freedman say:

If you do change the "default personal templates location" box in the Save options to point to the same folder as the User Templates folder, then you should also move all the templates from the Custom Office Templates folder to the User Templates folder. This arrangement will have the following consequences:

  • The File > New page will default to the "Custom" display, which will contain all the templates in the User Templates folder (including any you save there or paste in from elsewhere).
  • The "Featured" display on the File > New page will show only templates that are offered for download from Office.com.
  • The Custom Office Templates folder will be empty and should remain that way. The only downside to this is that your custom templates won't be included in the "roaming" data put in the cloud with your Microsoft Account (this is why MS split off the Custom Templates and put it under the Documents folder). If that's a problem, you can add the User Templates folder to the list of folders automatically copied to your OneDrive account, or put individual templates there.

charles kenyon say:
  1. If the Custom Document Templates folder is designated as the current "User Templates Folder" if that is in the default location, it is in a roaming data area.

  3. The Classic FileNew dialog will contain all templates in the User Templates Folder combined with the Workgroup Templates Folder. With the Custom Templates stored in the User Templates folder, this includes the custom templates which would otherwise not be shown.
