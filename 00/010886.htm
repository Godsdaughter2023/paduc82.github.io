bob jones aka: cybertaz say:
You can thank Apple for this :-)

In 10.7 they removed a feature that was present in OS X 10.5 & 10.6 which enabled Minimizing the Toolbar Area -- an oblong button at the right-end of the document window's title bar. It disappeared just as mysteriously as it had been introduced with no fanfare or documentation. What seems to be happening is that for those who upgraded an existing installation to Lion while the Toolbar Area was minimized the setting is still in effect but there's no longer a [simple] way to turn it off.

See this thread for options to edit the com.microsoft.Excel.plist file.

If you're not comfortable doing that I believe you can remedy the situation by simply deleting the .plist file. [In Lion, I believe the User/Library folder is Hidden -- To access it you need to be in Finder then select it from the Go menu.] Excel will build a new one when next launched & should use the default of displaying the Toolbar Area. The 'downside' is that you may have to reset some of your Excel Preferences.

Whichever method you use, make sure Excel isn't running at the time.

Regards,
Bob J.
