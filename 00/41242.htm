I don't use a network location to store system images.  The speed of a backup or recovery is much slower than a local external or internal drive.

The image needs to be on the root of a drive for local storage.  The network location does not have that requirement.

The main problem I see is connecting to the NAS when booting to the recovery media.  Of course you can test that and see if the system finds the image.  You will probably need the network path and possibly password for the NAS.  Wireless may not have a driver but Ethernet should work outside of Windows.
