We appreciate that you bring this to the Microsoft Community. We hope that everyone can help you at their best in this matter.

The error code: 0x807800C5 could mean that the UI for the Windows Backup is not running, or working differently. Please check under Services.msc (or "View local services" in Start/Settings search tab), if the Windows Backup service is running when you try to generate a backup.

Next, please double check if the target location has old backup files. The system might get confused if the old backup files are still there so please opt to move them to another location. These are the common files that has to be removed from the target location:

- Datafile - MediaID.bin

- Folder - Windowsimagebackup

- *ComputerName*-PC (the file name itself)

  3. Enter sfc /scannow (note the space between "sfc" and "/").

Feel free to keep us updated.
