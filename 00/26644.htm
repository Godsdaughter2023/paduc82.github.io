Hi Pat

If you show the property sheet in form design view, and click on the checkbox in the control toolbox, you should see the properties for "Default Check Box".  Several of these properties are to do with the automatic creation of a label when you add a checkbox. 

The LabelX and LabelY properties specify the default position of the label relative to the checkbox.  If you set this to a negative value then the label will be to the left.

Take a look at the properties for "Default Text Box" for a good value to start with.
