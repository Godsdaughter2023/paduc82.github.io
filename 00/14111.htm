Gray out checkboxes when another checkbox is selected in Forms for - Microsoft Community

michelles_424 say:

Gray out checkboxes when another checkbox is selected in Forms for Word 2010

Created a form in Word 2010 with checkboxes (i.e., Yes or No checkboxes). If the user puts a check in the Yes box the No checkbox will then become grayed out.
suzanne s. barnhill say:

A check box is intended to be a dual-state field, so instead of having two check boxes, one for Yes and one for No, you should state your yes/no question in such a way that a checked box indicates Yes and an unchecked one indicates No. For example, instead of asking, "Are you over 21?" you can have the check box labeled, "I am over 21" or "My age is over 21."

The bottom line is that check boxes are not intended for mutually exclusive options; for those you use radio buttons (and you can have any number of alternatives, only one of which can be selected). Radio buttons, however, are not available as content controls or legacy form fields.

formula help - Microsoft Community

twice2x2 say:

formula help

Hello,

I have a spreadsheet with two columns:

Advisor Name , Model

How can I write a formula that counts the number of distinct Advisor Names in each Model?

Thanks,

Tim

mike h.. say:

Hi,

In this formula I've assumed that Col A is Advisor and Col B is the model. C1 contains the model we are looking for. This is an ARRAY formula, see below for how to enter it

=COUNT(1/FREQUENCY(IF($B$1:$B$10=C1,IF($A$1:$A$10<>"",MATCH($A$1:$A$10,$A$1:$A$10,0))),ROW($A$1:$A$10)-ROW($A$1)+1))

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

bernie deitrick say:

One approach is to use a helper column of formulas - say that you have your data in A:B, with headers in row 1.  In C1, Enter the header "Unique" and in C2, enter the formula

=(COUNTIFS($A$1:A2,A2,$B$1:B2,B2)=1)*1

Then copy C2 down column C to match your values in A:B.  

To finish, insert a pivot table, and use Model as the Row Field, and Unique as the data field, set to sum, and you will get a nice summary table, with a list of the unique Models, and a count of the Unique Advisor Names for each.

ashish mathur say:

Hi,

Try this non formula approach

1. Select the dataset (including the header row) and go to Insert > Pivot Table > Add this data to the Data Model > OK

2. Add Model to the Row labels in the Pivot Table

3. Add Advisor Name to the Value area section of the Pivot Table

4. Right click on any number in the Value area section and under Summarise Values By, click on More

5. Scroll to the bottom and select Distinct Count > OK

Hope this helps.

amit tandon say:

Refer below image:

Enter below formula in cell D2 & copy down:

=IFERROR(INDEX($B$2:$B$100,MATCH(0,INDEX(COUNTIF(D$1:D1,$B$2:$B$100),,),0))&"","")

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell E2 & copy down:

=IF($D2="","",SUM(IF(FREQUENCY(IF($B$2:$B$100=$D2,IF($A$2:$A$100<>"",MATCH("~"&$A$2:$A$100,$A$2:$A$100&"",0))),ROW($A$2:$A$100)-ROW($A$2)+1),1)))

Column D returns the unique Models. Column E returns the count of distinct Advisor Names in each Model.

Excel Formula for first Business day after the 14th of each month

I need a formula that will return the first workday after the 14th of each month.  If the 15th is on a Monday then the formula would return the 15th.   If the 15th is a Saturday then the formula would return the 17th.
jp ronse say:

Hi Michael,

Try this one:

=WORKDAY(A1,(WEEKDAY(A1,2)>=5)*1)

EDIT: above formula returns the 14th when next day is a working day. The following should return the 15th:

=WORKDAY(A2+1,(WEEKDAY(A2,2)>=5)*1)

ronrosenfeld say:

With any date in A1, this formula will return the first workday after the 14th of the month in A1:

=WORKDAY(DATE(YEAR(A1),MONTH(A1),14),1)

And the WORKDAY function has an optional Holidays argument that you can use if required.  You need to list the holiday dates in some range, then reference that range.  See HELP for more details.
