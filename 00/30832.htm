¡firedog say:

I have used Windows Live Mail 2011 for years ...

That version has been obsolete on all versions of Windows later than Vista since 2012. Unless you're still on Vista, you should consider upgrading to lost in the upgrade.

... this error message: There was an error when attempting to connect to the Windows Live Calendar service.

Since this suddenly started happening, it's most likely that the account you sign in with has been upgraded*. The new version doesn't support the DeltaSync protocol that Windows Live Mail uses to synchronize mail, contacts and calendar with their online counterparts at mail.live.com, contacts.live.com and calendar.live.com respectively.

You can prevent this error from happening by clicking the right-most button on the Home tab of the ribbon and selecting Sign out of Windows Live Mail. When you do this, the

If you have the same account as the one you sign in with set up as a mail account so that it shows in the folder pane, it will also stop working if it's set up to use HTTP. You'll find instructions for setting up the account to use IMAP instead in this article: Windows Live Mail and Outlook Mail

If Microsoft is doing away with Windows Live Mail ... 

The only change that has taken place is that free Microsoft accounts are all being upgraded to a new version that Windows Live Mail isn't equipped to deal with. The only program functions that are affected are the synchronization of HTTP mail accounts, contacts and calendars. It continues to work as before in all other respects: POP3 and IMAP mail accounts, news accounts, RSS feeds, integrated contacts and calendar.

Note: Steps given for Windows 8.1 applies to Windows 10 as well.

Place the computer in Clean Boot.

Disclaimer: After you have finished troubleshooting, follow these steps from section “How to reset the computer to start as usual after clean boot troubleshooting” to reset the computer to start as usual.

Method 2: Run System File Checker scan.

System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files.

Run the SFC (System File Checker) scan on the computer to fix the system file errors on the computer.

To run SFC scan, follow the steps:

Open an elevated command prompt - Swipe in from the right edge of the screen, and then tap Search. Or, if you are using a mouse, point to the lower-right corner of the screen, and then click Search. Type Command Prompt in the Search box, right-click Command Prompt, and then click Run as administrator. Next follow the below steps to perform sfc /scannow:

  1. At the command prompt, type the following command, and then press ENTER: sfc /scannow.

The sfc /scannow command will scan all protected system files, and replace corrupted files with a cached copy that is located in a compressed folder at %WinDir%\System32\dllcache.

The %WinDir% placeholder represents the Windows operating system folder. For example, C:\Windows.

To know more about SFC scan you can refer to the link below:

Nonpaged Pool 2.2gb (Win8.1) - Microsoft Community

adl a say:

Nonpaged Pool 2.2gb (Win8.1)

Hello

I have started getting errors indicating low RAM while only browsing the internet. I have 8GB RAM installed, so i thought i might look into what's eating it all (aside from the browser).
 I found out that my Nonpaged pool size is about 2.2GB, which i later found out it is an anomaly; i should have been well below 512mb.

I booted into Safe Mode and saw no change in Nonpaged Pool size.

I couldn't find out which driver caused the memory leak with Process Explorer, so i ran the Driver Verifier which told me that the leak was from wdf01000.sys which occupied almost 2GB in the nonpaged pool.

I updated all my drivers and scanned my system for malware with a number of software. There were no malware detected.

I tried manually setting the size of the Nonpaged pool in the registry, but that somehow got ignored on reboot.

So now I'm stuck with 2.2GB used RAM in Nonpaged pool. What else should I try to fix this?

Thank you.

zigzag3143 (ms -mvp) say:
In order to diagnose your problem we need to run Windows performance toolkit the instructions for which can be found in this wiki

If you have any questions feel free to ask

Please run the trace when you are experiencing the problem

adl a say:

Thanks ZigZag!

I ran the wprui.exe and found out from the .etl file that the program using all that nonpaged memory had the tag KaMi. Using the findstr command in the system32\drivers folder i found that the driver was THAccel.sys - which is a toshiba driver for HDD Acceleration. It reserves a portion of the memory as a cache. I turned it off from the toshiba system settings and voila the nonpaged pool dropped by 2gb to its normal level. 

note: i tried disabling the driver with Autoruns and that broke windows - bsod on startup. Which required a system restore....

Here’s what you can do:

 

If she has upgraded from Windows 7 to Windows 8 Pro the language packs will be absent from Windows 8 Pro. There is a compatibility message shown “You’ll have to add your languages back in Windows 8”. If you haven’t installed the required language pack then you might not see them in “Language Settings”. So please go ahead and installed the required language pack in Windows 8 Pro and then try to add them in “Language Settings”.

 

 

Please follow the steps mentioned below to choose your required language in the “Handwriting Option”.

 

 

1. Please hold the (Windows key + C) to go to the Charms bar.

 

2. On the Charms bar please click on Search.

 

3. Please select Settings under Search option and type “Language”.

 

4. Now please click on “Language” and click on “Input method and Handwriting"

 

5. You can add your required language now.

 

6. You can personalize the Handwriting option by clicking on “Personalize hand writing recognition”.

Here's a solution for using Old vision of IME Pad in Win 8,

To able to have the handwriting feature (IME Pad)for Chinese or whatever countries in Window 8, you DO NOT have to download anything.

 

Go to control panel, languages,

On the left hand side bar, press ' Advanced Setting'

Scroll down and you will see 'Switching input method'

Tick the box ' Use the desktop language bar when it's avaliable'

And it asks you to log off,

After the bar shows up, it looks the same as the old window,

Choose Chinese (Taiwan), Tool menu, IME Pad (Don't forget to switch it to Chinese from English before choosing the IME Pad)

 

Thats it.
