It most certainly can be done in Word... in several ways.

You can also use the Accent tools of the Equation Editor if writing formulas. On the Insert tab click the Equation button [ 𝛑 ] or go to Insert> Equation, then open the Accent palette. To insert it as a text character in a cell or a Placeholder, type the character then click Symbol on the Insert tab. In the Characters pane select Unicode from the list of categories, then locate & double-click the Combining Overline [UC0305] in the Combining Diacritical Marks group.

jörn bosse say:

Hey,

 

never thought about that without vba, but try this:

Create a macro named Autokeys. Here you create a submacro with the sendkey syntax for the keys and on action you choose to go to an other record. This is not tested, but i think that could work. But i also guess that there will be errors if no form is open or no record is selected. In 2007 you can´t add an error trap to a macro like in 2010 and i think there will be an error when your already on that first record and wanna go back one more step. This is pretty unsafe at all.

 

btw:

turn key preview on and copy that code into your formmodule:

 

Private Sub Form_KeyDown(KeyCode As Integer, Shift As Integer)
On Error GoTo fehler

If KeyCode = 37 Then
    KeyCode = 0
    DoCmd.GoToRecord , , acPrevious
ElseIf KeyCode = 39 Then
     KeyCode = 0
     DoCmd.GoToRecord , , acNext
End If
Exit Sub

fehler:
    If Err.Number = 2105 Then
'don´t do anything
Else
    MsgBox Err.Number & " " & Err.Description
End If
End Sub

 

That´s it. You won´t have any problems with that code. I guess you should use this instead of macros.

hansv mvp say:

To change the colors automatically when the user enters/edits data:

Right-click the sheet tab and select View Code from the context menu.

Copy the following code into the worksheet module:

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim cel As Range
    If Not Intersect(Range("A3:M23"), Target) Is Nothing Then
        Application.ScreenUpdating = False
        Application.EnableEvents = False
        For Each cel In Intersect(Range("A3:M23"), Target)
            Select Case LCase(Left(cel.Value, 1))
                Case "s"
                    cel.Interior.Color = vbRed
                Case "a"
                    cel.Interior.Color = vbBlue
                Case "h"
                    cel.Interior.Color = vbYellow
                Case "v"
                    cel.Interior.Color = vbGreen
                Case "l"
                    cel.Interior.Color = RGB(255, 192, 255)
                Case Else
                    cel.Interior.ColorIndex = xlColorIndexNone
            End Select
        Next cel
        Application.EnableEvents = True
        Application.ScreenUpdating = True
    End If
End Sub

 

To run the code once for the entire range, create a macro in an ordinary module:

 

Sub ColorCells()
    Dim cel As Range
    Application.ScreenUpdating = False
    Application.EnableEvents = False
    For Each cel In Range("A3:M23")
        Select Case LCase(Left(cel.Value, 1))

                cel.Interior.Color = vbRed
            Case "a"
                cel.Interior.Color = vbBlue
            Case "h"
                cel.Interior.Color = vbYellow
