·        Right click the Taskbar or  Start button

·        Click OK

·        Right click the Taskbar or  Start button

·        Click the Start Menu tab (if not already selected)

·        Under Privacy, check both boxes (Store and display recently opened programs in the Start menu and Store and Display recently opened items in the Start menu and the taskbar)

·        Click Apply

·        Click OK

Thanks in advance

don cuthbert say:

See if you have a hidden folder at C:\$Windows.~BT.  If there is a file there called C:\$Windows.~BT\Sources\Install.esd you can convert that file into an ISO file which you can mount within Windows to install Windows 10 Build 10130 or use to create a bootable USB flash drive or DVD to do the install with.

You need a tool called ESD Decrypter to convert the file, but it saves downloading it all over again.  Also, having the ISO gives you more install options, such as trying a dual boot with a clean install of this latest Build.
