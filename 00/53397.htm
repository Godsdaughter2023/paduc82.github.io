Keyboard if 3rd party with their own software and drivers and any other
major device drivers you have.

installing actually shows up. This is because some drivers rollback before the

Photo Gallery archives unedited copies of edited
photos in a hidden file just for the purpose of
reverting.

If you deleted those original copies...reversion
would not be possible.

The path to the folder on a Vista system is:

C:\Users\<user name>\Appdata\Local\Microsoft\
Windows Photo Gallery\Original Images

This would be a hidden file so you may need to
change the following setting:

Go to...Control Panel / Folder Options / View tab.
