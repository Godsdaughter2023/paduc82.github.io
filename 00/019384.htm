If you want to display the dates that way, while keeping the underlying data as real dates, simply apply the custom number format yyyymmdd to the cells with the dates.

If you would like a value such as 20121231 as a number, you can use formulas adjacent to the cells with the dates.

Let's say you have dates in A2:A100.

In B2, enter the formula

=10000*YEAR(A2)+100*MONTH(A2)+DAY(A2)

Format B2 as General.

I have downloaded data in excel. My date fields are in the format dd.mm.yyyy format i.e. 31.12.2012. I would like to convert this date 31.12.2012 into 20121231 i.e. yyyymmdd format.

  1. If you have a column of text in the form dd.mm.yyyy or d.m.yyyy, you can use the Text To Column feature.
     
    Select the data, click on Data, Text To Columns, select Delimited, click on Next, select Tab, click on Next, select Date and DMY from the pull-down menu, change Destination if you do not want to overwrite the original data, then click on Finish.
     
    Finally, format the Text To Columns result as Custom yyyymmdd.
     
  2. If the original data are Excel dates -- that is, ISNUMBER(A1) is TRUE -- simply change the format to Custom yyyymmdd.
     
  3. If the original data are text in the form dd.mm.yyyy (e.g. 01.02.2012 with leading zeros), but the original data are in a row -- you can convert the original data with the following formula in a parallel row:
    =DATE(RIGHT(A31,4),MID(A1,4,2),LEFT(A1,2)
    formatted as Custom yyyymmdd.
     
  4. If the original data are text in the form d.m.yyyy (e.g. 1.2.2012 without leading zeros), but the original data are in a row, you can convert the original data with the following formula in a parallel row:
    =DATE(RIGHT(A1,4), MID(A1,FIND(".",A1)+1,2),LEFT(A1,2))
    formatted as Custom yyyymmdd
