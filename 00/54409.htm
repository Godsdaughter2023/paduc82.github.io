Under most circumstances hibernating is perfectly safe.  You should reboot at least once a week just to clear the memory

cmdrkeene say:

Yes it shouldn't hurt anything.  Plus, if you are using Windows 8 or newer, you are already using hibernate without knowing it.  If your "fast startup" setting is turned on, you're already using hibernate every time you click shutdown.  Instead of shutting down, it just "logs off the user session then hibernates what's left".  So even if you think you're using shutdown, you're actually using hibernate anyway.

This has implications for troubleshooting as well, because it means that a "restart" is a more complete reboot than "shutdown" is.

Do I need Office if I want to open WordPad text documents? - Microsoft Community

petersxe say:

Do I need Office if I want to open WordPad text documents?

My WordPad text documents are automatically incorporated into Office07.  Can I access these through WordPad or do I have to purchase Office to use WordPad?
jeeped say:
WordPad files have the RTF (Rich Text Format) file extension and may have become associated with Microsoft Office's Word program if you had a trial Office installation at one time. If you do have an expired Office trial on the computer that you have no intention of ever re-activating through purchase, you should uninstall the expired installation first through the Control Panel's Programs and Features applet. If you have a working installation of Office 2007 but wish to have RTF files open in WordPad, just leave the Office 2007 installation alone and proceed.

To re-associate your WordPad RTF files with WordPad, locate one within Explorer and right-click, then select Open with... then Choose default program from the flyout.
If WordPad is within the Recommended Programs section, first make sure that Always use the selected program to open this kind of file is checked in the lower left and double-click WordPad. The RTF document will open in WordPad and the default file association will be re-instated.
If WordPad is NOT within the Recommended Programs section, use the Browse button to locate Wordpad.exe, usually found within C:\Program Files\Windows NT\Accessories on a Vista or Windows 7 system. Once located and you are back at the Open With dialog, make sure that Always use the selected program to open this kind of file is checked in the lower left and double-click WordPad.
