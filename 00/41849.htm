Yes, you can use same Hard Drive to store System Image and use it as a drive for File History.

A system image is an exact copy of a drive. By default, a system image includes the drives required for Windows to run. It also includes Windows and your system settings, programs, and files. arminatorx say:

As the video data is a digital data stream transmitted via HDMI, errors in the transmission result in flickering pixels or lines across the screen. 

These errors are usually caused by faulty HDMI cables or faulty connections (e.g. the HDMI cable not being properly plugged in, or corrosion starting to happen). Please unplug the cable on your TV and the console and re-seat the cable properly. Often this pulling and plugging is enough to scrape the corrosion off the contacts, and you can confirm that the cable is properly seated in the plug again. 

If the flickering is still there, try a different cable. Or try the Xbox on a different TV or monitor. 

If the flickering is still visible on a different TV with different cables, it might be that the HDMI plug on the console is defective itself, in which case you need to send in the console for repair. 

If it's within the one year warranty, the repair will be free. 

Simply go to https://devicesupport.microsoft.com to start the repair process. Register your console if not already done so, and sign it up for repair.

It sounds like you could have a virus or other malware. Get your antivirus program up to date and boot into Safe Mode. Note that some viruses can hide themselves from your antivirus program in normal mode, so you really need to scan in Safe Mode. To get into Safe Mode when you first power up, hit F8 about once a second until you get the menu and select Safe Mode. Then run a full system scan.

Microsoft has suggestions and offerings at

http://windows.microsoft.com/en-US/windows7/how-do-I-remove-a-computer-virus

You can check for corrupted system files. Open an administrator command prompt and run SFC if the above doesn't help. Click START, then type sfc in the search box, right-click SFC.EXE and click Run as administrator. Then from the command prompt type sfc /scannow.

Finally if all else fails, you can check the rather cryptic system event log. To do so click Start -> Control Panel -> Administrative Tools -> Event Viewer. Once in Event Viewer click on the system log and scroll through the entries looking for those flagged "error" to see if you can find hints as to where the problem could be.
