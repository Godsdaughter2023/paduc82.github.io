How to join two tables? - Microsoft Community

ricardolgomez say:

How to join two tables?

I have two tables that contain the same data,  collected at two different points in time...I'd like to combine those 2 tables into one. How can I achieve this? thanks as always!

ricardo

hansv mvp say:

You could append all records from one table to the other.

 

Manually:

  • Open one table.
  • Press Ctrl+A to select all records.
  • Press Ctrl+C to copy the records.
  • Open the other table.
  • Select the blank new record at the end.
  • Press Ctrl+V to paste the records from the first table.

Using a query:

  • On the Create tab of the ribbon, click Query Design.
  • Select the first table, then close the Show Table dialog.
  • Drag * from the field list to the query grid below.
  • On the Design tab of the ribbon, click Append.
  • Select the second table as target, then click OK.
  • Click the Run button (the exclamation mark) on the Design tab of the ribbon.
  • Confirm that you want to run an append query and to append the records.

Second, I have many cases where I have multiple versions of a single file. I use Save As a new name and usually add a date or a single digit and that way I keep a backup of my original file. Personally I find it a safer work flow as this way if something happens to my most current file, and it somehow becomes corrupt I can always go back to the most recent version which I have in backup. Just sayn' that there are worst things then having multiple copies of the same publication.
