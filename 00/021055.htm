tontcha say:

Thanks for the reply, but I figured it out myself. After hours of reading about similar issues (but with other versions of browsers, mostly IE9 on W7), I found somebody mentioning that IE 11 on Windows 8.1 can crash with Protected Mode on. So I disabled it - and voila, problem solved.


Actually there are 2 ways of getting rid of this issue:

1. Go to Internet Options, Security tab and uncheck "Enable Protected Mode" (see the screenshot below). Restart IE and issue will be gone, but you'll get that pop-up message that IE is not in Protected Mode any more, which may make you feel not so comfortable.

2. Or you can go to Internet Options, Advanced tab and in Security section check "enable 64-bit processes for Enhanced Protected Mode" (see the screenshot below). This will require a restart of your PC, as you can see there. Reboot and issue will be gone.


This workaround is for 64-bit only, Windows 8.1 x64 and IE11. Similar issue on other versions of Windows and other versions of IE may be resolved by methods that I tried in my first post.
