a. user say:

Hi,

Thank you for being part of the Windows 10 family.

Sorry to know that you are facing issues with Windows 10.

Refer to the below methods and check if it helps.

Method 1: I would suggest you to run the printer troubleshooter and check if it helps.

http://windows.microsoft.com/en-us/windows-10/fix-printer-problems

If the issue still persist, refer to the below method.

Method 2: I suggest you to install the latest driver for printer driver from the device manager.

1. Press Windows Key + R Key.

2. Then type “devmgmt.msc” without the quotes and hit on Enter Key.

3. Expand Printer Queues.

4. Right click on the driver and click on Update driver software.

If the issue still persist, refer to the below method.

Method 3: I suggest you to uninstall and install the latest driver for printer driver from the manufacture of the Computer.

Step 1: Try to uninstall the printer driver. 

1. Press Windows Key + R Key.

2. Then type “devmgmt.msc” without the quotes and hit on Enter Key.

3. Expand Printer Queues.

4. Right click on the driver and click on Uninstall.

Step 2: I would suggest you to run the Program Compatibility mode troubleshooter and then try to install the driver. Check if it helps.

1. Tap on Windows Key and click on search bar.

2. Type “Troubleshooting” without the quotes and Click on Troubleshooting.

3. Click on View All the left panel of the screen and then click on Program Compatibility mode

4. Then click on Next and follow the instructions.

Get back to us if you need any further assistance on Windows related issues and we will be glad to help.
