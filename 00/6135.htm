You would need something like:

 

Sub AdjustWindow()
Application.WindowState = xlNormal
ActiveWindow.WindowState = xlNormal
With Application
    .Top = 40
    .Left = 40
    .Height = 400
    .Width = 400
End With
End Sub

 

You will need to "play" with the values to get your desired result with whatever display and display settings you are using.

That is the way that Word works.  The Table of Figures is constructed from the Captions that you have inserted below each figure, so going to the Caption should be near enough.  If however the Caption is on the next page, you should insert the figure with the text wrapping set so that it is "in-line with text", which will result in it being located in a paragraph, which you should then format so that it is kept together with the next paragraph.
