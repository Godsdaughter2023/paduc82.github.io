vanessa yar say:

Hi SupineFlunky838,

The ability to automatically delete older backups to free some space for the new backups is not possible as backups are created depending on the available disk space. However, you can also change how Windows retains older system image. Instead of letting Windows manage the space used for backup history, you can choose Keep only the latest system image and minimize space used by backup. Doing so will delete your earlier system images and keep only the latest System Image.

For more information, you may read the blog, Managing backup disk space.
