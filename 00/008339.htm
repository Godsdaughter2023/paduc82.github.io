Publisher 365 and Fonts - Microsoft Community

Publisher 365 and Fonts

I want to use the fonts that are available in my Windows 7 font folder with Publisher 365. Is it possible to access these fonts?

davidf2 say:

Sure, but if it were me I would first identify the specific fonts used in your Publisher publication and change out those only. I would go into the Win10 fonts folder and add or change the extension of those fonts to '*.old'. Example: Arial.TTF to Arial.old This way if your experiment goes bad you can easily go back to the Win10 fonts by removing the .old extension. Then after renaming the fonts files, you can copy the individual Win7 fonts and paste them into the Win10 font folder. The next time you open your Pub file it will use the Win7 font files.

It sounds like you are trying to resolve the rendering issues of Pub files created on Win7 machines and older, that started as of Win8. Users found that if they opened old Pub files on new Win8 or Win10 machines the layout was messed up. MS had made changes to the fonts that resulted in text rendering at a slightly different size which through off the text box layouts. Switching to Win7 fonts fixed that problem. I would suggest that though this workaround did work that I would just tweak and and adjust the layout and start living with the new fonts. MS is not going back to the 'old style' font, nor should you expect them to fix this issue, so you might as well get used to it. I would also imagine that at some point in time some forced MS update will automatically replace those Win7 fonts with 'updated' Win10 fonts.
