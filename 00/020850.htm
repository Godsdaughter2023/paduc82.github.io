johnlv say: Ed-

Each of the forms on a Navigation form is actually a subform of your Navigation form.  To Requery any of them, the syntax in VBA is:

Forms![<name of Nav form>]![<name of subform>].Requery

If the code is running on one of the subforms on the Nav form, you can also do:

Me.Parent![<name of other subform>].Requery

Forms![<name of Navigation form>]![Navigation Subform]![<subform on the current form>].Requery

marshall barton say:

IF the other nav forms are open, try using:

    Forms![some other nav form].NavigationSubform.Form.Requery

But, if only some of the other nav forms might be open along with an ocassional regular form you'll need a way to idenify which open forms are nav forms that might need to be requeried.  One way would be to set the nav forms' Tag property to NAV.  Then you can loop through the Forms collection and requery the tagged forms:

Dim frm As Form
   For Each frm In Forms
      If frm.Tag = "NAV" Then
         frm.NavigationSubform.Form.Requery
      End If
   Next frm

If that doesn't take care of requerying any subform within another nav form's subforms (i.e. a subsubform), we'll have to loop through each nav form's active subform looking for 2nd level subform controls and requery them as well.

Refer to the link mentioned which contains the fonts installed with Microsoft Office 2013:

http://support.microsoft.com/kb/2800393

 

Browse to the location mentioned below and check the fonts that are missing are listed in this location:

C:\Windows\Fonts

 

If the fonts are listed under Fonts folder but are not displayed in Office 2013 program then change the default printer and check:

http://windows.microsoft.com/en-US/windows-8/set-change-your-default-printer

 

If fonts are not displayed under Fonts folder then repair Office 2013 installation. Refer to the link mentioned which explains how to repair Office 2013 installation:

http://office.microsoft.com/en-in/project-help/repair-or-remove-office-HA010357402.aspx

 

If repair does not fix the issue then copy the fonts from and other system and install it on the present system
