A file system is the underlying structure a computer uses to organize data on a hard disk. If you are installing a new hard disk, you need to partition and format it using a file system before you can begin storing data or programs.

NTFS allows you to gain the maximum benefits for the needs of today’s enterprise business environments from Windows Server, such as increased security, more robust and reliable performance, as well as a design for greater storage growth, features not found in FAT.

NTFS is the preferred file system for this version of Windows.

Please refer:
Comparing NTFS and FAT file systems
http://windows.microsoft.com/en-US/windows-vista/Comparing-NTFS-and-FAT-file-systems
(Also holds good for Windows 7)

Additional Information:
https://technet.microsoft.com/en-us/library/cc778410(v=ws.10).aspx

Note: Converting a hard disk partition or volume from NTFS to FAT32 requires that you reformat the partition, which deletes any data on it. Be sure to back up any data you want to keep before you begin.

The recovery image is stored on a dedicated recovery partition on your PC, and is typically 3-6 GB in size.

So, you may require at least 3-6 GB storage disk size. Normally, using a DVD(4.7 GB), you may create a system repair disk.

For system image, you may require multiple DVDs.

We recommend saving system image on the hard drive. However, the files can be saved on a USB flash drive, CDs and DVDs.
