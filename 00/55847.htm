docdaneeka1969 say:

There are two possible issues that stop people playing Minecraft Splitscreen Multiplayer on the Xbox.

1. The Xbox must be connected to the TV via an HDMI cable AND the Xbox System>Display setting must be 720p or 1080i.

2. If you are trying to create a local split screen game where one or more of the players does not have Gold membership then the game settings must have the online game check box UNCHECKED. This check box is found by selecting the "More Options" button on the Minecraft create game dialog box (I.e the place where you can select creative or survival modes). This selection is persistent. Once it is set to on-line game no non Live Gold players will be able to join in this and future games

So just to recap. To play a game of split screen multiplayer on Xbox 360 Minecraft locally on the Xbox when some players don't have a Gold Membership. Start the game. Select Play Game. Choose to Create New World or select a saved world. In the dialog box select creative or survival mode and then select More Options and ensure the on-line game check box is unchecked. Start game. The second player can now join by pressing the start button.

This and other forums explain the first problem well but it took me and my kids ages to work out the second. We came across many frustrated posters who were clearly having this problem but hadn't been able to solve it. Hopefully this will save others some time.

nusaram say:

Ok, I have solved the problem...finally!!!

This is what I did:

  1. Go to Start | Control Panel | Networking and Sharing Center
  2. Click on "Change adapter settings" on the left hand side
  3. Right-click on your network connection and select "Properties"
  4. In the "This connection uses the following items" box, select "File and Printer Sharing for Microsoft Networks"

On my second PC the item #5 was enabled/selected, but when I looked at my main PC, I noticed that the checkbox was empy! So I enabled the option and--bam!--it worked!!! Woohoo! I ws now able to connect from my second PC to my main!

I stumbled onto this when I saw another post somewhere about checking out the TCP/IP configuration and I remembered whow to do that in Vista. But I didn't know the equivalent procedures in Windows 7, so while hunting around, I luckily stumbled onto the above!!!
