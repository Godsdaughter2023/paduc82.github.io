resize the comments, near to cells (visible=true)

Sub ResizeComments()
On Error GoTo errh
Dim r As Range
For Each r In ActiveSheet.UsedRange.SpecialCells(xlCellTypeComments)
With r.Comment
.Shape.TextFrame.AutoSize = True
.Visible = True '<<< or False
.Shape.Top = .Parent.Top - 10
.Shape.Left = .Parent.Offset(0, 1).Left + 10
End With
Next
Exit Sub
errh:
MsgBox "sht: " & ActiveSheet.Name & vbCr & "no comments found"
End Sub

jam1000 say:

I have been having the same issue.  In addition to the OneNote files you mentioned, I was also having the issue with two .pst files that Windows and/or Office saved to a folder in Documents.  

I believe I have found at least a workaround.  After you have auto-saved your desktop, cut the problematic OneNote files (and any other problem files) and paste them to your desktop.  Then let the Auto Save process run.  Once the Auto Save process has completed, you can move the OneNote and, if applicable, other files back to where they were.  I noticed that when I did this, the OneNote and .pst files auto saved in the Desktop folder on OneDrive, then (a) disappeared from the Desktop folder in OneDrive when I moved them back from the desktop to their original folders in Documents and (b) appeared in the appropriate sub-folder in my OneDrive Documents folder.

Hope this helps.

try this approach, without autofilter

Sub Hide_Visible_Colors()
'Mar. 03, 2017
Dim rng As Range, r As Range
Set rng = [A2:A20]
rng.EntireRow.Hidden = False
For Each r In rng
If IsEmpty(r) Or r.Interior.Color = RGB(192, 192, 192) Then
r.EntireRow.Hidden = True
End If
Next
End Sub
