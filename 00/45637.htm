Windows 10 Upgrade and WinBook TW100 Windows 8.1 tablet... three things to watch out for...

I just upgraded my WinBook TW100 tablet from Windows 8.1 to Windows 10 and had three issues that

anyone using this tablet should be aware of BEFORE they upgrade:

1) There is a file name touchscreen.gt  in c:\windows\inf.  The upgrade installer did not copy this file into

the new Windows 10 installation and it rendered the touchscreen unusable.  Make a backup of this file

and drop it into c:\windows\inf and reboot the tablet after you upgrade if your touchscreen is not working

right.

I tried to copy the file from the windows.old but kept getting an error saying the file could not be copied.

Don't know whether this was because of insufficient privileges or because the file was somehow corrupted

or compressed, etc.

2) The Intel HD Video driver installed by Windows 10 was set to a resolution of 1024 x 768, would not

allow me to change it to the native resolution of the tablet (1280 x 800) and insisted that the LCD panel

in the tablet was a flat panel display (external).  i had to go to Intel's website and download the Intel HD

Graphics driver for their 4000 series chipset.  This will allow you to set correct screen size, and the new

driver install correctly recognized the internal display as default.

This was an older version of the driver than what Windows 10 installed, so I am hoping that it does

not try to do a Windows Update on the driver again.... ugghhh...

3) The upgrade installer DID make a backup of Windows 8.1 in c:\windows.old, but when I tried to roll back

to Windows 8.1, there was no option under restore applet to roll back to Windows 8.1, so make a complete

backup of your tablet BEFORE running the upgrade installer else you will be stuck with Windows 10 and

unable to go back.

4) This is trivial, but the wallpapers Windows 10 installed are not the correct size for the screen... I used

Google Image search and got a different size for the default wallpaper with the blue window (I really like

this one).

Hope this info helps anyone else with this Winbook tablet....
