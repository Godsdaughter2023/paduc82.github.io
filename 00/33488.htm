You would have to experiment with using the Bell server as the outgoing mail server for the Cogeco account. To test this,

  1. Right-click on the Cogeco account name in the folder pane and select Properties.

    Enter the Bell username and password you use to access the Bell account in webmail.
  3. On the Servers tab, select This server requires a secure connection (SSL) for the outgoing mail server.
    Ensure that the outgoing mail port is 587.
  4. Close down Windows Live Mail and wait a few minutes before re-launching.
  5. Compose a test message and send it using this account to the same address.
