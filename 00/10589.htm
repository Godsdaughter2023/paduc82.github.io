Hello,

Each cell in the range is equal to a name or the text "N/A".

What I would like to create a macro that will look at every cell in "Names_to_Export", and for every cell that does not contain "N/A", copy that row and write it into the text file as a tab delimited text, with each new row as it's own new line.

I am thinking that this would need a FreeFile set up with a loop, but I'm not sure how to do that for a situation quite like this.

Any help is greatly appreciated!

Thanks in advance!
chip pearson say:

Try code like the following:

 

Sub ExportRange()

Dim FName As Variant
Dim FNum As Integer
Dim R As Range
Dim C As Range
Dim S As String

FName = Application.GetSaveAsFilename( _
    filefilter:="Text Files (*.txt),*.txt")
If FName = False Then
    ' user cancelled
    Exit Sub
End If
FNum = FreeFile
Open FName For Output Access Write As #FNum
For Each R In Range("NamesToExport")
    DoEvents
    If StrComp(R.Text, "n/a", vbTextCompare) <> 0 Then
        Set C = R
        S = vbNullString
        Do Until C.Column = 25 ' column Y
            ' create one line for export
            DoEvents
            S = S & C.Text & Chr(9) ' build each line
            Set C = C(1, 2) ' move right
        Loop
        S = Left(S, Len(S) - 1) 'get rid of trailing tab
        Print #FNum, S ' print text to file
    End If
Next R
Close #FNum
End Sub

scott zimmerman say:

To fix this issue I found this the Apple support site and it worked for me... :)

Problem solved?

 

My solution was found to be to simply refresh (reload) the USB driver for c:\program files\common files\apple\mobile device support\drivers.

 

Chris Lim explains the method using Windows 8.1, and it worked exactly the same for me in Windows 10Technical Preview build 10122..   see

 

Procedure

  • Plug in your iPad or iPhone
  • select Device Manager
  • select Portable devices
  • select iPhone or iPad
  • select Update Driver software
  • RIGHT click and select  'Browse computer for driver software'
  • Browse to c:\program files\common files\apple\mobile device support\drivers (assuming 64bit) or c:\program files (x86)common files\apple\mobile device support\drivers for 32bit systems.
  • Click on OK
  • Click on Next
  • You should see a message saying..   'Windows has successfully updated your driver software'   This should now mean that the Apple Mobile Device USB Device has now been renewed/refreshed/updated
  • Click on Close

 

I hope this works for you too!
