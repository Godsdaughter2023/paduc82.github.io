stefan blom say:

Try using a LISTNUM field. For example:

1 Heading 1

1.1 Heading 2

(IUO) { LISTNUM \L 2 } This is your "another subsection" paragraph.

You can insert the LISTNUM field via Insert tab | Quick Parts | Field. Alternatively, you can add the field code directly to text as follows: Press Ctrl+F9 to add the field delimiters, { }. Type in the code as shown. Press F9 to update. Press Alt+F9 to show/hide field code display.
