bernie deitrick, excel mvp 2000-2010 say:

I use a macro to insert, scale, and center the picture: Select the range where you want the picture, then run this

Sub test()

Dim myR As Range
Set myR = Selection
'Insert the picture
ActiveSheet.Pictures.Insert( _
   Application.GetOpenFilename( _
   "JPG picture files (*.jpg),*.jpg", , "Select the picture")).Select
  'scale the picture to the width or height of the selection
myScale = Application.Min(myR.Width / Selection.ShapeRange.Width, _
         myR.Height / Selection.ShapeRange.Height)
Selection.ShapeRange.ScaleWidth myScale, msoFalse, msoScaleFromTopLeft
Selection.ShapeRange.ScaleHeight myScale, msoFalse, msoScaleFromTopLeft
Selection.ShapeRange.Left = myR.Left + (myR.Width - Selection.ShapeRange.Width) / 2
Selection.ShapeRange.Top = myR.Top + (myR.Height - Selection.ShapeRange.Height) / 2
End Sub

allen browne say:

The code to do this in Access (VBA) would involve the following steps:

  1. OpenRecordset()
  2. Loop through the records
  3.     Open a file for output
  4.     Print # each line of HTML text, including the appropriate tags (e.g. <P>.)
  5.     Close the file.
  6. Next record until the loop completes.
  7. Close the Recordset.

Here's an example of some VBA code that creates an HTML page out of Access:
    http://allenbrowne.com/AppOutputHtmlCode.html
It doesn't create a page for each record, but it shows how to create one HTML page. The logic above of looping through the records will do it for you.

If you want the page automatically generated each time a record is added/edited in Access, you would use the AfterUpdate event procedure of the form where the users input the data. You may also want to use the form's Delete + AfterDelConfirm events to delete the web pages as record(s) are deleted from the database.

ynnhoj gnahz say:

Hi Catherine,

 

In Office 365, we can forward emails from one address to multiple email addresses. Here are the steps:

1. Sign into Outlook Web App (https://outlook.office.com/owa) with your Office 365 account.

2. Click Settings (the gear icon in the upper-right corner of the page) – Options – Mail –Automatic processing-- Inbox and sweep rules. Click “+” to create a new rule. Set “When the message arrives, and it matches all of these conditions” as “Apply to all messages”, set “Do all of the following” to forward to the email accounts you want. Click Save at last.
