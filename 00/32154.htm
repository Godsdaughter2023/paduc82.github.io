a. user say:

Hi,

If the screen turns off after 1 minute and turns ON when use the mouse or keyboard, then the screen is going to sleep mode. This could occur due to some incorrect power settings. I would suggest you to try these steps and check:

Method 1:

The built-in power troubleshooter can automatically detect and fix most of the incorrect settings. Refer to these steps to run this troubleshooter:

  1. Press Windows key +F to bring up the search Charms bar.
  2. Select settings in the drop down menu below search and type Troubleshooting.
  3. Click on Troubleshooting and select the view all option on the left pane.
  4. Click on Power troubleshooter and follow the prompts.

 

Method 2:

You have mentioned that you have already disabled the sleep settings. To make sure that you have turned off all sleep settings including the Hybrid sleep, check out these steps:

  1. Press Windows key + X, click on ‘Power options’.
  2. Click on ‘Change plan settings’ against the power plan selected.
  3. Click on ‘Change advanced power settings’.
  4. In the power options window, expand the Sleep option.
  5. Set the options ‘Sleep after’ and ‘Hibernate after’ to never.
  6. Set the option ‘Hybrid sleep’ to off.
  7. Expand the ‘Display’ option and set the ‘turn off display after’ to never.

Method 3:

If the issue still persists, I would suggest you to boot the computer in Safe mode and check if the issue persists. Refer to this article:

http://windows.microsoft.com/en-US/windows-8/windows-startup-settings-including-safe-mode
