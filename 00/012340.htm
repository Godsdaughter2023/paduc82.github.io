todtown say: What you are asking is not possible. I could code you a solution, but it would be convoluted and awkward. Not as seamless as you are envisioning. 

However, I created a procedure for somebody not too long ago that allows you to change the frozen panes by right clicking them. It's not exactly what you're asking, but you can try it out:

Private Sub Worksheet_BeforeRightClick(ByVal Target As Range, Cancel As Boolean)
    dim ans as integer
   
   If Target.Row = 1 Then Exit Sub
    Cancel = True
    ans = MsgBox("Freeze Pane?", vbYesNoCancel, "Freeze Pane")
    If ans = 2 Then
        ActiveWindow.FreezePanes = False
    ElseIf ans = 7 Then
        GoTo NormalRightClick
    Else
        ActiveWindow.FreezePanes = False
        Rows(Target.Row).Select
        ActiveWindow.FreezePanes = True
        Target.Select
    End If
    Exit Sub
NormalRightClick:
    Cancel = False
End Sub

Put this code in the module for the sheet where you want this to happen. Then all you have to do is right click on a row (any cell in that row). The message box will ask if you want to freeze the pane? Click Yes to freeze the pane on the row above where you clicked.  The pane will freeze above the row you click. Click No and you will get the normal right click functionality. Click Cancel and it will unfreeze panes. 

If you need help with this just let me know.

john korchok say:

To create a custom page size:

  1. In Word, choose File>Page Setup.
  2. Click on the Paper Size dropdown and choose Manage Custom Sizes.
  3. Click on the plus + sign.
  4. Double-click on Untitled and give it a name like 7x10.
  5. Set the page size and margins,, then click on OK.
  6. Using File>Page Setup, set your document page size to the 7x10 custom page size, then save.

Now you can choose the 7x10 page size in the  Page Setup dialog. To set this as a default page size:

  1. Open System Preferences>Printers & Scanners.
  2. Select 7x10 from the Default paper size dropdown.
