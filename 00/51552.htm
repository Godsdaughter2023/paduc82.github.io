monica cam say:
Hi RonKropp,

There could be a number of factors as to why you are experiencing issues with USB devices upon the startup of your computer. This could be caused by an outdated driver or due to incorrect device settings causing conflict with the USB devices.

You can follow the steps below on how to troubleshoot USB devices issues in Windows 10:

Method 1: Run the Hardware and Devices troubleshooter.

  1. On the Start menu, type Troubleshoot and then hit Enter.
  2. Under Find and fix other problems, select Hardware and Devices. Click Run the troubleshooter.

Method 2: Disable the Allow the computer to turn off this device to save power option.

  1. Press Windows key + R on your keyboard.
  2. On the run command box, type devmgmt.msc and then hit Enter.
  3. Expand Universal Serial Bus controllers.
  4. Right-click the USB Root Hub and then select Properties.
  5. Go to Power Management tab.
  6. Uncheck the Allow the computer to turn off this device to save power checkbox.
  7. Click OK.
  8. Restart your computer to apply the changes.

Method 3: Modify the registry.
This is an advanced troubleshooting so I suggest that you create a System Restore point so you can go back to this state in case that you will encounter any issues along the process. To create a system restore point, refer to this article: Create a system restore point.

Once you're done creating a system restore point, you may then proceed with modifying the registry. (Note: You have to be very careful when modifying the registry as it will cause a serious issue when modified incorrectly.) Follow the steps below on how to modify the registry settings:

  1. Press Windows key + R on your keyboard.
  2. Type regedit on the run command box.
  3. Navigate to this path:
    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Executive.
  4. Right-click AdditionalCriticalWorkerThreads and then select Modify.
  5. Set the Value data to 2 and then click OK.
  6. Restart your computer to apply the changes.

Let us know how it goes and we'll assist you further if needed.

Follow the steps listed below to disable the touchpad.

davem121 say:
Hi JEP2014

Start your PC, just as Windows attempts to load (spinning dots), press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down

Do this twice

On the third start Windows will boot into the Recovery Environment and from there you can access System Repair, Safe Mode, Command Prompt . . . etc.

Please try each option one at a time, then see of Windows will boot . . .

Go to Troubleshoot - Advanced Option - System Restore
Check to see if you have a restore point you can go back to

If you do not have a Restore Point, go to Troubleshoot - Advanced Option - Startup Settings and click Restart
Upon restart, press 4 to enter Safe Mode
Once in Safe Mode, backup your data if you do not already have a backup
Open Device Manager and check to make sure all your Drivers are up to date
Reboot to see if Windows will start Normally

If the above fails, go to Troubleshoot - Advanced Options - Reset this PC
Choose to keep your files and apps, just your files or to keep nothing
Then start the Reset

vijay a. verma say:
A. Try to boot into safe mode

Power ON your computer and as Windows / manufacturer's Logo appears, Power OFF - Repeat 3 times
Now your computer will go into Advanced Recovery Environment
Advanced Options > Troubleshoot > Advanced Options > Startup Settings > Restart
Press 4 or F4 to enable Safe Mode (5 or F5 to enable Safe Mode with Networking)
Log in once Windows starts in safe mode

B. Another option when you are in Advanced Options in previous step is to choose (Note - You won't lose any files in this step)
1. System Restore - This will restore your system to a point when things were working fine.
OR
2. Startup Repair - Startup Repair upgrade fixes all Windows errors and retains all files, applications and settings.

C. Final option could be Reset this PC.
In Step A > Troubleshoot > Reset this PC - Keep my files and Remove everything. Choose this carefully as most likely you would like to keep your files.
All apps will be removed and windows will be reinstalled.
