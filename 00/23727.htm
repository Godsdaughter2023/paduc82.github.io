You could only do this with code


Sub toGreen()
    Dim osld As Slide
    Dim oshp As Shape
    Dim L As Long
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
If oshp.HasTextFrame Then
If oshp.TextFrame.HasText Then
With oshp.TextFrame.TextRange
For L = .Runs.Count To 1 Step -1
If .Runs(L).Font.Bold Then
.Runs(L).Font.Color.RGB = RGB(20, 85, 43)
.Runs(L).Font.Bold = False
End If
Next L
End With
End If
End If
Next oshp
Next osld
End Sub

First of all, open Applications > Utilities > Disk Utility, select your hard disk and repair permissions. This process will make sure that Apple-created folders and other infrastructure folders have the read/wrote/delete protections properly set for the system, the users and the applications.

After repairing permissions, restart your computer and test Word.

If Word still does not work, check the location of the fonts. There are three places where fonts reside:

1) Macintosh HD > Library > Fonts  (fonts for all users of your computer). This folder should contain a "Microsoft" folder that groups all Office fonts. Except for that one folder, it should not contain any other folder (you can create them for classification purposes, but I don't know what could happen if they end up having the wrong access permissions). The fonts here appear in Font Book's "Computer" category.

2) Macintosh HD > System > Library > Fonts (in general, reserved for the system). It should not contain any folders and you should refrain from adding or deleting fonts here. These fonts

3) Macintosh HD > Users > UserName > Library > Fonts (fonts available only to user UserName). To access this folder, go to the Finder, press Option and select Go in the Finder's main menu bar. As above, you should not have any unplanned folders in there. The fonts here appear under Font Book's "User" category.

______________________________________

When you disable a font through Font Book, it is moved to a folder called "Fonts Disabled" next to the regular "Fonts" folder, where the font becomes inactive.

Take a look at those three (or up to six) folders to make sure the disabled fonts appear in the "Fonts Disabled" folders and there are no stray copies in "Fonts." You can move them manually between those two folders, of course, instead of using Font Book, or move them between the other folders (be careful here!). You can also add and delete fonts (be even more careful!). In some instances you will be prompted for the user password.

I already said (twice now) that a comparison of prices makes no sense in this context.  I won't comment on that further, in order to avoid sounding like a broken record.

Using CORREL with the set of percentage changes is a better measure of correlation, since it factors in magnitude of change as well as direction of change.

However, there might be some methods of analysis where you only want to consider correlation of changes in direction or "runs" (number of consecutive trading periods before a change in direction).

You can still use the set of percentage changes for that, since as a positive change in price is a positive percentage change, and a negative change in price is a negative percentage change.

That does not make any particular approach "correct" or "not correct".  The world of financial and economic analysis is not black and white.  Each methodology provides a different perspective.

Changing default font size for email addresses typed in Excel - Microsoft Community

cathystone say:

Changing default font size for email addresses typed in Excel

Is there a way to change the default font size when an email address is typed into an Excel cell?

Call me lazy but it drives me crazy to have to go back and change the font to match the rest of my workbook.

mjcorkery (mct regional lead) say:

Hi Cathy,

 

For an individual workbook you can change the style for a hyperlink. To do so, go to the Home tab, Styles group. Locate the hyperlink style in the gallery (you may need to click on the arrows to scroll through the list of styles.) Once you find the hyperlink style, right-click it and choose Modify.  Make your changes and click OK to close the dialog. Any hyperlinks in the workbook should update to the new style.

 

Unfortuntely, Excel does not have the concept of global styles link Microsoft Word does.  The style you just modified is only valid for that particular workbook.

 

One approach to making this easier is to record a macro of yourself modifying the style. If you assign a keyboard shortcut to the macro you will be able to quickly run the macro whenever you need it.

 

If you modify the style in a workbook and save the workbook as a template, all new workbooks based upon that template will have the new hyperlink style.

 

Please let me know if you have more questions.

Mike
