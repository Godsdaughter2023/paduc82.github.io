What happens if you tap the same key instead?   I'm thinking those keys on your keyboard may be operating the same way that special keys are overloaded on the "chicklet" keyboard in the TIP.   E.g. instead of tapping on some of the letter keys you can tap and hold to make them bring up an array of loosely related keys from other languages.*   A related analogy would be "texting" on a telephone.  E.g. multiple taps on the same key would cycle you through the set of letters that were related to that key and switching to another key would leave the last one you left showing as the one that you "typed".

* An early preview gives a good description of the feature

http://blogs.msdn.com/b/b8/archive/2012/07/17/designing-the-windows-8-touch-keyboard.aspx

" interesting counter example in press-and-hold behavior"

Another thing to do is open the On-Screen Keyboard, e.g. Run... osk.exe  since doing that may show you some clues in their potentially more relevant keycaps than the potentially "incorrect" physical keycaps.  Note that unfortunately it is not possible to have both the OSK and the TIP open at the same time, so while the OSK is running you would not be able to start the TIP.

Hi,

The issue might occur due to the following reasons.

  1. The volume is online and it has outstanding changes.
  2. Changes were made to the volume and the computer was shut down before the changes were committed to the disk.
  3. Corruption was detected on the volume.

I suggest you to perform a chkdsk by following the steps provided below and check if it helps. chkdsk runs to verify the file system integrity and to attempt to fix any issues with the volume.

  1. Press Windows + X keys and select Command Prompt (Admin).
  2. Type the command line below and press Enter:

CHKDSK C: /F /R (Replace the letter "C" with the drive letter of the SD card).

auggy say:

It sounds like the issue is with the Samsung Drive Manager software and not the drive itself.

In the following topic the poster received information from Seagate Support saying the Samsung AutoBackup software is not compatible with Windows 10:

http://answers.microsoft.com/en-us/windows/forum/windows_10-update/samsung-m3-portable-hard-drive-doesnt-like-windows/0732aefa-1fcd-471c-907e-d2c51f669b5b

Windows 10 is not listed as a supported operating system in the download link:

http://www.seagate.com/ca/en/support/downloads/item/samsung-drive-manager-master-dl/

You could try uninstalling Samsung Drive Manager and then reinstall the latest version of Samsung Drive Manager using "Compatibility Mode" and choose a previous operating system.

However the software itself may have to be updated for Windows 10.
