ossiemac say:

Try the code below.

Install the code as follows (Just in case you need the guidelines.)

  1. Alt and F11 to open the VBA editor.
  2. Double click "ThisWorkbook" in Project Explorer (left column)
  3. Copy the code below and paste into the VBA editor
  4. Edit the code for the Worksheet names and cell addresses for the Date Validation.
  5. Do not alter the sub name or it will not work
  6. Save the workbook as "Macro Enabled"
  7. Ensure macros are enabled. See Help for how to do this. (Option to "Disable all macros with notification" should be OK)

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
    Dim strSht As String
    Dim strShtA As String
    Dim strShtB As String
    Dim strValAddrA As String
    Dim strValAddrB As String
   
    'Edit next 2 lines "Sheet2" and "Sheet3"to sheet names with validation
    strShtA = "Sheet2"
    strShtB = "Sheet3"
   
    'Edit address next 2 lines to the validation addresses for each sheet
    strValAddrA = "$B$4"    'Validation Address on sheet assigned to strShtA variable
    strValAddrB = "$A$2"    'Validation Address on sheet assigned to strShtB variable
   
    On Error GoTo ReEnableEvents
    Application.EnableEvents = False
    Select Case Sh.Name
        Case strShtA
            If Target.Cells.Count <> 1 Then Exit Sub
            If Target.Address <> strValAddrA Then Exit Sub
            Worksheets(strShtB).Range(strValAddrB) = Target.Value
           
        Case strShtB
            If Target.Cells.Count <> 1 Then Exit Sub
            If Target.Address <> strValAddrB Then Exit Sub
            Worksheets(strShtA).Range(strValAddrA) = Target.Value
    End Select
           
ReEnableEvents:
    If Err.Number <> 0 Then
        MsgBox "An error occurred in ThisWorkbook Sub Workbook_SheetChange"
    End If
   
    Application.EnableEvents = True
   
End Sub

The hard reset will bring your phone back to factory settings, the same state as when the device was first turned on. All apps, messages, media, personal settings, etc will be removed from the phone. If backup is turned on under Settings, you can restore your apps and settings after performing a hard reset (as long as you initialize the phone with the same Microsoft Account).

Note: Resetting your device will not remove any OS or firmware updates. You will remain on the same version of Windows Phone after the reset.

Follow these steps on how to perform a hard reset:

  1. In the App list, tap Settings .

  2. Tap About, and then tap Reset your phone.

  3. You'll receive two warnings. If you're absolutely sure you want to restore your phone to its factory settings, tap Yes, and then tap Yes again.

    It might take a little while for the process to complete.

To reset an unresponsive phone
  1. Press and hold the volume down and Power buttons at the same time until you feel a vibration (about 10–15 seconds).

  2. When you feel the vibration, release the buttons, and then immediately press and hold the volume down button until you see a large exclamation mark.

  3. Once the exclamation mark appears, press the following four buttons in this order: volume up , volume down , Power , volume down . Your phone should now reset and restart itself. (It might take a while for the reset to finish).

NOTE: Resetting your phone will erase all content stored on your phone, including apps and games (along with app data and game progress), text messages, call history, music, photos, and more.
