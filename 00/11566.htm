yongjoyce say:

Help needed to undo "reduce file size" in Powerpoint for Mac 2011

Hi, I wanted to email a powerpoint file and had used the Reduce File Size feature in PowerPoint Mac for one of my powerpoint files. It affected all my other powerpoint files when I opened them. How do I turn off that Reduce File Size to restore the original file size?

john korchok say:

Reduce file size downsamples the images in your presentation. Unless you undo this at the time, it's a permanent operation. Next, time, experiment on a copy.

If you have a backup copy, you can open that. If you use Time Machine, you should have earlier versions of your presentation stored in it. Otherwise, you'll have to replace all bitmaps files with new copies at a higher resolution.

yongjoyce say:

Does this feature affect all my other powerpoint files? I notice when I redo the presentation slides, and saved the new presentation file, it is in “reduced size” even though I didn’t click on that feature. It somehow becomes a permanent feature for subsequent new files saved.

john korchok say:

Images are not compressed automatically by PowerPoint unless you tell it to do so. 

I think you're trying to refer to the Reduce File Size dialog. The dropdown is set by default to Best for Printing (220 ppi). This is not a reflection of the current resolution of your graphics. The images are not downsampled until you click on the OK button in that dialog. If this doesn't answer your questions, please explain using the actual commands and dialog names in PowerPoint.
