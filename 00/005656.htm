How to Display Instruction if Legacy Option Button is Selected - Microsoft Community

lauren shipton say:

How to Display Instruction if Legacy Option Button is Selected

I am creating a form in Word 2007 and would like to display a message only if a certain legacy control option button is selected.

For example, I write a question, "Have you traveled outside the continental U.S. during the last twelve months?" and this question is then followed by two legacy control option buttons "Yes" and "No".

I want to somehow display a message only if the "No" button is selected.

For example, if the person selects "No", then the following message is displayed next to the button, "Please skip to questions 5".

If the person selects "Yes", then the message is not displayed.

jay freedman say:

Here's one way (there are others).

First, type the message in the document where you want it, and surround it with a bookmark. Let's say the bookmark for this particular question is named skipto5.

I assume you've already entered the same GroupName value in the Properties pane for both of these option buttons, to make them operate as a group -- when one is turned on, the other one is automatically turned off. Also, you should change the names of the option buttons from the default like OptionButton1 to something like Q2Y (for "question 2 Yes") and Q2N.

On the Developer tab, turn on Design Mode, and then right-click one of the option buttons. In the menu, click View Code.

In the code window, remove the empty code procedure that Word inserts automatically, and replace it with this:

Private Sub Q2Y_Click()
    If Q2Y.Value = True Then
        ActiveDocument.Bookmarks("skipto5").Range.Font.Hidden = True
    End If

End Sub

Private Sub Q2N_Click()
    If Q2N.Value = True Then
        ActiveDocument.Bookmarks("skipto5").Range.Font.Hidden = False
    End If
End Sub

Of course, if you've used different names for the buttons and the bookmark, change the code to match them.

Close the code window, turn off Design mode, and try clicking each option button in turn. The bookmarked text should appear and disappear appropriately. Note: If the text doesn't disappear, but it has a dotted line under it, you need to go to File > Options > Display and uncheck the box for "Hidden text".

Another method that would work is to use the code to change the .Caption property of the No option button, so it includes the message or not.

For question 2, the user needs to have Edit permission to be have to use "Pin to the Filters pane' option. You need to grant the user Edit permission if you want him to use the feature.
