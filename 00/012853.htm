t. valko say:
Shane Devenshire wrote:

I realize that you may not consider {,,,} an array but it is as far as I am concerned. 


There's a difference between an array constant and entering a formula as an array.

This version needs to be array entered** :

=SUM(SUMIF(A:A,D1:D3,B:B))

Where D1:D3 = x, y, z

Note that without the SUM function (in this example) the SUMIF function will only calculate based on the first criteria cell D1 even if the formula is array entered. So:

 

Shane Devenshire wrote:

SUMIF does not support multiple criteria without entering it as an array

Is not accurate.

This version does not need to be array entered:

=SUM(SUMIF(A:A,{"x","y","z"},B:B))

You could use this normally entered version that handles the criteria as a range array:

=SUMPRODUCT(SUMIF(A:A,D1:D3,B:B))

Also note that all 3 versions of the formula I've posted here have virtually identical calculation times so there is no performance advantage of one over the other. There is a "design" advantage in using a range array, however.

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.
