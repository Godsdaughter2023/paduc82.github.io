jay freedman say:

The file mentioned in the message is the "owner file" for the template that belongs to the Avery label wizard add-in. Besides the fact that the Avery wizard doesn't do much you can't do with Word's built-in label feature, the macro code in earlier versions of the wizard template often caused a compile error -- see my Answer at https://answers.microsoft.com/en-us/office/forum/office_2010-word/compile-error-in-hidden-module-az-wizard-modul.

Usually Word won't even try to open an owner file, and I'm not sure why it's happening to you. What you can do, though, is to go to the folder %appdata%\Microsoft\Word\STARTUP and delete the file ~SWiz14s.dotm (and AVWiz14s.dotm if it exists).
