Can anyone help me adjust the following code from Ron Debruin's - Microsoft Community

twice2x2 say:

Can anyone help me adjust the following code from Ron Debruin's excel site

Can anyone help me adjust the following code Ron Debruin's excel site.  I'm using this code to automatically create outlook emails with attached spreadsheets and  I'm running into problems with the body of the email.  This code simply populates the email body with "Hi there" but I need to insert an entire letter.  Is there a way to copy the contents of a word document and then insert the data into the body of the email? 

 

Any help would be greatly appreciated.

 

Thanks,

 

Tim

 

Sub Send_Row_Or_Rows_Attachment_1()
'Working in 2000-2010
    Dim OutApp As Object
    Dim OutMail As Object
    Dim rng As Range
    Dim Ash As Worksheet
    Dim Cws As Worksheet
    Dim Rcount As Long
    Dim Rnum As Long
    Dim FilterRange As Range
    Dim FieldNum As Integer
    Dim mailAddress As String
    Dim NewWB As Workbook
    Dim TempFilePath As String
    Dim TempFileName As String
    Dim FileExtStr As String
    Dim FileFormatNum As Long

    On Error GoTo cleanup
    Set OutApp = CreateObject("Outlook.Application")

    With Application
        .EnableEvents = False
        .ScreenUpdating = False
    End With

    'Set filter sheet, you can also use Sheets("MySheet")
    Set Ash = ActiveSheet

    'Set filter range and filter column (column with names)
    Set FilterRange = Ash.Range("A17:C" & Ash.Rows.Count)
    FieldNum = 1    'Filter column = A because the filter range start in column A

    'Add a worksheet for the unique list and copy the unique list in A1
    Set Cws = Worksheets.Add
    FilterRange.Columns(FieldNum).AdvancedFilter _
            Action:=xlFilterCopy, _
            CopyToRange:=Cws.Range("A1"), _
            CriteriaRange:="", Unique:=True

    'Count of the unique values + the header cell
    Rcount = Application.WorksheetFunction.CountA(Cws.Columns(1))

    'If there are unique values start the loop
    If Rcount >= 2 Then
        For Rnum = 2 To Rcount

            'Look for the mail address in the MailInfo worksheet
            mailAddress = ""
            On Error Resume Next
            mailAddress = Application.WorksheetFunction. _
                          VLookup(Cws.Cells(Rnum, 1).Value, _
                                  Worksheets("Mailinfo").Range("A1:B" & _
                                                               Worksheets("Mailinfo").Rows.Count), 2, False)
            On Error GoTo 0

            If mailAddress <> "" Then

                'Filter the FilterRange on the FieldNum column
                FilterRange.AutoFilter Field:=FieldNum, _
                                       Criteria1:=Cws.Cells(Rnum, 1).Value

                'Copy the visible data in a new workbook
                With Ash.AutoFilter.Range
                    On Error Resume Next
                    Set rng = .SpecialCells(xlCellTypeVisible)
                    On Error GoTo 0
                End With

                Set NewWB = Workbooks.Add(xlWBATWorksheet)

                rng.Copy
                With NewWB.Sheets(1)
                    .Cells(1).PasteSpecial Paste:=8
                    .Cells(1).PasteSpecial Paste:=xlPasteValues
                    .Cells(1).PasteSpecial Paste:=xlPasteFormats
                    .Cells(1).Select
                    Application.CutCopyMode = False
                End With

                'Create a file name
                TempFilePath = Environ$("temp") & "\"
                TempFileName = "Your data of " & Ash.Parent.Name _
                             & " " & Format(Now, "dd-mmm-yy h-mm-ss")

                If Val(Application.Version) < 12 Then
                    'You use Excel 2000-2003
                    FileExtStr = ".xls": FileFormatNum = -4143
                Else
                    'You use Excel 2007-2010
                    FileExtStr = ".xlsx": FileFormatNum = 51
                End If

                'Save, Mail, Close and Delete the file
                Set OutMail = OutApp.CreateItem(0)

Want No Header on First Page But DO Want Footer - Microsoft Community

jackibar2 say:

Want No Header on First Page But DO Want Footer

I'm trying to set up a book that will include a page number in the footer of ALL pages (even the first page of each chapter) - but I do NOT want the Header to appear on the first page of each chapter.  I can't figure out how to do this!!!!  Any help GREATLY appreciated!!

daniel g. grau say:
You have to check "Different first page" to get a separate set. However, it is possible that your Word is misbehaving, so try this:

  • Create a blank document
  • Insert some text and/or page breaks to obtain a couple of pages
  • Select View > Header and footer
  • The blue tabs should say Header | Close ⓧ and Footer | Close ⓧ
  • Type something in the header and footer (like "General header" & "General footer")
  • Select Document > Layout  > Different first page (this should create a blank header/footer only for page 1)
  • Go to the first page and do View > Header and footer
  • The blue tabs should say First Page Header | Close ⓧ and First Page Footer | Close ⓧ
  • Type "1st Header" and "1st Footer" and check that the other header/footer remains untouched
  • You can navigate them with the "Previous Section" and "Next Section" buttons in the "Header and Footer" purple tab in the Ribbon.

If you are unable to obtain independent headers/footers with the preceding instructions, there is something wrong with your installation or document.

jackibar2 say: Thanks, SO much for all the help, Daniel!  This was very helpful in leading me to figuring out the big mess I had!  Seeing that it WOULD work on the blank document, and following the steps you provided - especially really looking at those "blue tabs" for the headers and footers so I could SEE what WORD thought I was doing(!) - is what kicked my brain into gear finally...

I think - no, I KNOW - the reason my situation was so complex is due to the fact I had ALL the various options enabled - Different First Page, Different Odd and Even Pages, and Sections...

Here were the steps that finally worked:

1.  Click somewhere on the first page of each section (chapter).

2.  Format > Document > Layout:  Click Different Odd and Even and Different First Page.  (NOTE:  I did NOT realize I needed to do this for every section!).

3.  View > Headers and Footers.

4.  Add the page number in the footer by clicking on Page Number button (instead of using the "Insert Page Number" feature).

5.  Check to make sure the header is erased on Page 1 of that section.

6.  Close the header/footer editor.

7.  Repeat for each section.
