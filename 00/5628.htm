In addition to the Document_Close document event, there is a DocumentBeforeClose application event (read http://msdn.microsoft.com/en-us/library/aa140279%28office.10%29.aspx to see how to set one

stefan blom say:

Just to be clear, you are saying that there should be a "split" in the table of contents? If you just want the TOC to start on a right-hand page, that is a lot easier: just insert an Odd Page section break at the end of the cover page.

If you must break the TOC into two pieces, you can do what I suggested, but putting the TOC field inside a linked text box will be easier. Insert a text box on the first page for the TOC and insert another text box on the second page where the TOC should continue. Click in the first text box and, on the Drawing Tools Format ribbon tab, click Create Link, and finally click in the second text box. Insert the TOC in the first text box.

Note that if the TOC spans more than two pages, you'll need a third text box, and so on.

Try these basic troubleshooting steps first and check after each step and please update me. (Please note your data is stored on the cloud, hence you won't be losing data)

2. Uninstall and reinstall Skype app (Note - You will not lose any Skype data because of this)
Uninstall
Windows Key+X > Windows Powershell (Admin) > Copy the below, paste into Powershell > Enter

Get-AppxPackage -allusers Microsoft.SkypeApp | Remove-AppxPackage

Reinstall

Get-AppxPackage -allusers Microsoft.SkypeApp | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}
