yingruhan say:

Hi all,

jp ronse say:

Hi,

Try this way:

=SUMPRODUCT((LEFT(A2:A12,3)="ABC")*SIGN((ISNUMBER(SEARCH("DE",A2:A12))+ISNUMBER(SEARCH("F",A2:A12))+ISNUMBER(SEARCH("FA",A2:A12))+ISNUMBER(SEARCH(2,A2:A12))))*(B2:B12))

The explanation.

The string must start with ABC and contain also DE or F or FA or 2.

LEFT(A2:A12,3)="ABC" is clear, string must start with ABC

ISNUMBER(SEARCH("DE",A2:A12)) and the other conditions can be tested with this construction, SEARCH returns a value or an error but the error can be catched by ISNUMBER (TRUE/FALSE). Because it is an OR check, all checks must be summed. The pitfall is the sum, the check on FA will return TRUE if FA is in the string but the check on F will also return TRUE and 1+1 =2 (AFAIK)

To catch this, SIGN can help because it returns 1,0,-1 for positive, 0 and negative values. Finally we have to multiply the results by the sales figures.

business owner la say:

After much frustration and research I have found a solution that has worked for both of my gmail addresses:

In Outlook click on File

Click on Office Account

Under Account Privacy click on Manage Settings

In the Privacy Settings - the last one All Connected Experiences - you need to tick the box Enable connected experiences.

Click OK

You should now be able to download your emails. It was suggested when I found this solution that once all emails were downloaded you could go back and un-tick that box and it would still work fine. This was not the case for me. I need to have the box ticked to get my emails to work. Having this box ticked for me does concern me re privacy but I have little choice if I want to see my gmails in Outlook.

Hope that helps
