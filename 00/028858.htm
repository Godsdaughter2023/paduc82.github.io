jllatham say:

Get rid of the .Select in statements like this

Set C = .Find(whatname, LookIn:=xlValues, LookAt:=xlPart).Select

it should simply be

Set C = .Find(whatname, LookIn:=xlValues, LookAt:=xlPart)

It is the .Select combined with Set C that is generating the error.  'C' is a range in this case, and .Select is an action (method).

You seem to have added (or had left over) the .Select part.  The code Gord provided would have worked - it's actually from a 'classic' example provided as part of the Help for Microsoft Excel 2003, and I use it often as the model for such actions myself.  Got a copy of the original tucked away for the day it totally disappears from searches - it's kind of hard to find even now.

Below is a slight modification of your own modification.  Note that we do not have to actually .Select anything!  Saves us some time, can be done from any sheet in the workbook without leaving it, and doesn't mess up the display with a big selected area.

When you try to Set something, if it fails/can't be done/found then the something ends up being Nothing (rather Zen-like), but if it works, then it is Not Nothing, and you can test to see if you found it or not with a statement like

  If Not IsNothing myObject Then

          'you found it, work with it

  End IF

Or alternatively if you are primarily interested when it is NOT found, do it like it's done in the code below; with the

If myObject Is Nothing Then

  'wasn't found - do what you need to do

End if

Sub FindTest2()
  Dim InvRecIDtoFind As String
  Dim FindID As Range

    InvRecIDtoFind = InputBox("Enter Inv. RecID", "Find")

'    ActiveSheet.Range("A:A").Select ' again not required: messes up display and time consuming

'we make reference to the 'what to look in for the .Find' as part of this statement

    If FindID Is Nothing Then
      MsgBox ("Inv. RecID not found")
    'optional else section
    Else
      MsgBox "Inv. RecID is in the list."
    End If

End Sub

If you had multiple instances found and wanted them all selected use this code.

Sub Select_Stuff()

'Bob Flanagan code

Dim C As Range, tempR As Range
Set myrng = Range("A:A")
    'check each cell in the selection
    InvRecIDtoFind = InputBox("Enter Inv. RecID", "Find")
    For Each C In myrng
    If C.Value = InvRecIDtoFind Then
            If tempR Is Nothing Then
    'initialize tempR with the first qualifying cell
                Set tempR = C
            Else
    'add additional cells to tempR
                Set tempR = Union(tempR, C)
            End If
        End If
    Next C
    'display message and stop if no cells found
    If tempR Is Nothing Then
        MsgBox "There are no Rows " & _
                "that meet the criteria."
        End
    End If
    'select all qualifying cells
    tempR.Select
End Sub
