Glyph variations are not separate characters, but different representations of the same character. That's why you won't find Glyph 408 in Character Viewer—it's an E.

Curiously, pasting Glyph 408 from Font Book into Text Edit and searching for "e" does not find the character. But there is another way to enter that glyph that allows Text Edit to find it as "e":

  • Type "e" in Text Edit
  • Highlight the character
  • Select Format > Fonts > View Fonts
  • Apply Apple Chancery (if not applied)
  • Click on the gear icon at the bottom left
  • Open "Typography"
  • Open "Glyph variants" at the bottom and select a character

Once you enter a character that way, it does transfer to Word as a regular "e" instead of the Replacement Character (i.e., the white-on-black question mark).

vijay a. verma say:
"," means that it is a character comma and formula should not treat this as a separator.

bernie deitrick say:

It represent a string composed on the single ASCII character 44, which is a comma.

The formula itself is meaningless, since it always evaluates to ","

In the IF function, 1000 is the conditional, which Excel evaluates as TRUE (since it is non-zero). So the function will return the value given as the second parameter - the third parameter is returned when the conditional evaluates as FALSE, which 1000 won't.

sudo mdutil -E

Does PowerPoint 2010 support hanging punctuation? - Microsoft Community

kmckay5 say:

Does PowerPoint 2010 support hanging punctuation?

I've added a large quote to a slide, and I want all the text left-justified with a hanging quotation mark. Is this supported in PowerPoint 2010? The only workaround I've found is to put the quotation in a separate text box and place it to the left of the first line. 
chirag. say:

To get hanging punctuation in PowerPoint 2010, do the following:

  1. Insert the quotation mark at the beginning of your quote. Stay in edit mode for editing the text in your text box.
  2. Turn ON Rulers: Check the "Ruler" box in "View" tab.
  3. On the ruler, the top arrow marker is for the start of the first line. The bottom arrow marker are for subsequent lines. Adjust the bottom marker to align with the start of text of the first line to get hanging punctuation.

Alternatively, you can use a quotation mark as your bullet since the above ruler settings are preset for you in the case of bullets.

- Chirag
