It has to be something in the OneDrive folder or its subfolders. Open File Explorer to OneDrive, & look there. But I doubt it's a real Desktop.ini. (1) Click the OneDrive icon, then the Open Folder button.

(2) Click File Explorer's View tab, then the Options button (or ask Cortana for Folder Options)...

(3) Type "Desktop.ini" into File Explorer's search box...

There they are, but OneDrive is oblivious to them. Right-click the OneDrive icon in the Tray, & select "View Online". I can find no equivalent of Folder Options there to make them show up.

How much space needed for full backup with Windows 7 - Microsoft Community

robertquarteroni say:

How much space needed for full backup with Windows 7

Hi. I've tried a complete backup with a 16GB flash drive and it said that wasn't enough. Before I waste endless hours again and have me tell me there's  not enough space, can someone tell me how much I need?  Thanks.

canadian tech say:

Most, if not all your data is usually found in your User folder.

  1. Computer
  2. Double-click C:
  3. Double-click Users
  4. Right-click on your User account, choose Properties

You will then see an estimate.

If you have other users on your system, you will need to repeat the procedure to get the size of their storage.  Add them together.

USB thumb drives are not a great place to put a backup.  They are much less reliable and slower than using an external USB connected hard drive.  After you have an estimate of the space you need, you can buy yourself the appropriate size.  Best advice is to buy much bigger than you currently need because your need is very likely to grow.
