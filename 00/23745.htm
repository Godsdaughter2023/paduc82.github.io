Delete Duplicates / triplicates from Excel Single Cell

Dear Experts,

I imported data from outlook and all my contact were duplicated and triplicated. I managed to sort out with the exception of the data in Notes Cell. I'm attaching an image of my problem.

Is there any way to solve this issue as the same applies for more than 700 cells.

ronrosenfeld say:

You can use a VBA macro to remove the duplicated lines from each cell.  The macro below assumes your source data is on Sheet1 in column A, and it will write the results of the filtering to Sheet2 columnA, so make sure there is nothing of value there.  If you examine the code, it should be obvious how and where to make changes if need be.

The macro uses "Regular Expressions" to define the pattern to be accepted which, in brief, is any line or series of lines which is not subsequently repeated within the cell.  So, in effect, it is returning the last item of the duplicated items.

If there is a problem, post back with as much detail as possible.

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.

========================================

Sub RemoveDupPhrases()
    Dim RE As Object, MC As Object, M As Object
    Const sPat As String = "(^.+$)(?!(?:[\s\S]+)\1)"
    Dim wsSrc As Worksheet, wsRes As Worksheet
    Dim vSrc As Variant, vRes() As Variant
    Dim rRes As Range
    Dim I As Long, S As String
    
Set wsSrc = Worksheets("Sheet1") 'Original Data
With wsSrc
    vSrc = .Range("A1", .Cells(.Rows.Count, "A").End(xlUp))
End With

Set wsRes = Worksheets("Sheet2") 'Results go here
Set rRes = wsRes.Range("A1")

'Initialize Regex
Set RE = CreateObject("vbscript.regexp")
With RE
    .Global = True
    .MultiLine = True
    .Pattern = sPat
End With

'Size Results array
'Should be one entry per row of source data
ReDim vRes(1 To UBound(vSrc), 1 To 1)

'Filter duplicates
For I = 1 To UBound(vSrc)
    S = vSrc(I, 1)
    If RE.test(S) = True Then
        Set MC = RE.Execute(S)
        For Each M In MC
            vRes(I, 1) = vRes(I, 1) & vbLf & M
        Next M
        vRes(I, 1) = Mid(vRes(I, 1), 2)
    End If
Next I

'write results
Set rRes = rRes.Resize(rowsize:=UBound(vRes))
With rRes
    .EntireColumn.Clear
    .Value = vRes
    .ColumnWidth = 255
    .EntireColumn.AutoFit
End With
End Sub

You can't attach more than one paragraph style to each numbering level in an outline-numbered list, as you have noticed.

What you can do is base a style on a style which is part of an outline list (in the Modify Style dialog box for a style not part of a list, make use of the "Based on" box). ·         Click OK

·         Press Enter on your keyboard

·         Uncheck Load startup items

·         Click the Services tab

·         Check the Hide all Microsoft services box

·         Click Disable all

·         Click OK

·         Click Restart Now

Once your computer has finished restarting, test to see if you still get the same effect when typing 6. If not then you just have to identify which service/startup item was causing the problem.

·         Click Start

·         Click Run…

·         Type msconfig into the Open: field

·         Click OK

·         Press Enter on your keyboard

·         Click the Services tab

·         Check the Hide all Microsoft services box

·         Check (Enable) the top half of the service boxes

·         Click OK

·         Click Restart Now

Once your computer has restarted, again test to see if the problem is still gone. If so, then the services you enabled are good to stay. Repeat the enabling process for half of the remaining disabled services and continue this process until the problem returns or all services are enabled. If the problem returns then you will need to test the last batch of services you enabled one by one to find which was causing the problem to occur.

 

If you enable all services and the problem still hasn’t returned:

·         Click Start

·         Click Run…

·         Type msconfig into the Open: field

·         Click OK

·         Press Enter on your keyboard

·         Click the Startup tab

·         Check (Enable) the top half of the service boxes

·         Click OK

Follow the same process as before, with testing after each restart.
