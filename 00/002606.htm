Mandatory Cell assignment - Possible? - Microsoft Community

girishkumarnair say:

Mandatory Cell assignment - Possible?

Dear all,

I have an excel form which is intended to be filled in by a lot of end-users.  There is one cell which is key to all other entries in the form (for example let us call this 'Staff No.'.  I would like this to be made a mandatory entry.  Meaning, if an user does not enter the Staff No, then no other entry can be made in the sheet.  If the end user tries to enter any other information, then excel should prompt him/her to enter the Staff no first.

Is this possible in Excel, please?  Highly appreciate, if someone can guide me on this, please.

Thanks and best regards,

Girish Kumar Nair.

ossiemac say:

This can be done with Data Validation. Note that the validation is applied to all of the cells that you want to disallow entry until Staff No is populated. Validation is NOT applied to Staff No cell.

Assume that cell A2 is the mandatory cell for this explanation. Edit to you cell that contains the Staff No.

  • Select any one of the other cells that requires entry only after cell Staff No is populated
  • Select Data ribbon
  • Select Data Validation -> Data validation
  • Select "Settings" Tab (should be the default tab)
  • Click the "Allow:" field dropdown and select "Custom" from the list 
  • Uncheck the "Ignore blank" box. (This is essential)
  • In the "Formula:" field enter the following formula  (Edit $A$2 in the formula to your Staff No cell.)

=NOT(ISBLANK($A$2))           

  • Select "Error alert" tab
  • Enter a title in the "Title:" field (Suggest "Error!")
  • In "Error message:" enter something like  "Enter Staff No first in cell A2"   (Edit A2 to your Staff No cell)
  • Click OK to close the dialog.
  • Copy the cell with the Data validation and then select other cells to which it is to apply. You can select the first cell of the ones to which it is to apply and then hold the Ctrl key while you select others.
  • When required cells selected then right click over one of the selections and then select Paste Special -> Validation and the validation will be applied to all of those cells.
  • If you have a lot of cells for the Data validation then I suggest that you copy and paste a few at a time because it is not so easy to select a large number of cells while using Ctrl key without making an error. However, if you do past to a cell that should not be there then select the individual cell and then on the Home ribbon, select "Clear" towards far right of ribbon and from the DropDown select "Clear all" and the Data validation will be cleared from that cell.
