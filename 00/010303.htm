2. Restore original phone settings: It will be under phone settings or type *#7780# in your phone's standby mode. No data will be lost, but you will reset all your phone's settings back to the default.

3. Soft reset: Type *#7370# in your phone's standby mode. It will format your phone and wipe all your data. If you haven't changed your security code, it will be 12345 (default). Backup your data before doing this.

 

4. Hard reset: AKA the 3 finger reset. With the phone off, you can try switching the phone on by pressing and holding the following buttons -- '3', '*' and the green "call" -- all pressed at the same time. Phone will be formatted and data gone.

 

(This applies to most s60 edition phones, but some s60 versions don't have this. Like my 5800 XM [not applicable for version 11. 5. If that still doesn't change anything, try checking for a software update using Nokia Software Updater. :)

 

NOTE: Steps 3, 4 & 5 will remove all personal content and data added after the original default state. So you might want to do a backup using Nokia PC Suite's Backup application and/or backup to your memory card before you do any of those steps.

j.e. mcgimpsey say: First, reinstalling an application on a Mac (the preferred abbreviation of Macintosh - "MAC" in all caps usually refers to "Media Access Control" addresses) is nearly always a waste of time. Applications, which are read-only files, rarely if ever become corrupt. Unlike in Windows, where installations usually reset the registry, Mac installations don't usually touch the Preferences, where corruption usually occurs.

You don't say what "won't open" really means - do you get an error? a crash? a system dialog? nothing?

I'd start with some basic troubleshooting:

First, make sure you've updated Office to the most recent version.
Second, try starting Word with the Shift-key down to bypass preferences and the saved Normal template.
Third, try these in approximate order:
   -  Run the Disk Utility application to repair disk permissions. Restart your machine.
   -  Run the Font Book application to resolve corrupt and duplicate fonts.  See http://word.mvps.org/Mac/fontweeding.html.  Again restart.
   - If not fixed, with ALL Office applications closed, delete (or rename) the following file
             ~/Library/Preferences/com.microsoft.word.plist
        and this folder
             ~/Library/Preferences/Microsoft
        where ~ is your home directory. Note if you're using OS X Lion, your Library folder is hidden.
    - If none of these fix the problem, test whether another application or system extension
       is interfering with Word by creating a new user account and seeing if you get the same problem
