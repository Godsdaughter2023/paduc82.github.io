Try the following:

 

Download the latest LightScribe System Software (1.18.26.7) from the following link;

 

http://www.lightscribe.com/downloadsection/Windows/index.aspx?id=810

 

Click on the .exe to run the LS update and when the window pops up "Welcome to the Installation Wizard for LightScribe System Software" do not click "Next" but leave the window open.

 

Then do the following:

 

Windows Key + R > type %temp% in the Open box > OK

 

This will open your Temp folder.

 

In the Temp folder locate the folder that holds the extracted files from the open LightScribe installation.

 

The folder will be named {5BFB22E2-F51C-467C-AB75-97EF31590120} or something like that.

 

In that folder locate the LS_HSI.msi.

 

Right-click the LS_HSI.msi > Properties > Compatibility and then, under "Compatibility mode" put a checkmark in front of  "Run this program in compatibility mode" and select "Previous version of Windows" > OK

 

Then go back to the open LightScribe installation window and click "Next" and proceed with the installation.

Method 1: Run the Search and Indexing Troubleshooter.
 
Follow the below steps:

  • 1)Press Windows+X key.
    2)Click on Control Panel and select ‘view by’ on the top right corner.
    3)Select Troubleshooting and click on View all
    4)Find Indexing Troubleshooter.
    5)Run the Troubleshooter.

Method 2: Rebuild the Search Index and check. Refer to these steps.

  • Press Windows+X Key.
  • Click on Control Panel.
  • Select Small icons or Large icons on the View by menu in the upper-right corner of the Control Panel window.
  • Click Indexing Options.
  • Click the Advanced button at the bottom of the Indexing Options window. This displays a new window titled Advanced Options.
  • Click the Index Settings tab if it isn’t selected already.
  • Click the Rebuild button to delete and rebuild the file index.
  • Click OK to confirm.
