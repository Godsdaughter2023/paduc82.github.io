dianeoforegon say:
  1. Open Entourage Preferences under Entourage in the Menu bar (Command ,)
  2. Select Sync Services
  3. Select to merge Entourage with Apple Address Book.
  4. Give it some time to sync. Often it can be stubborn to start sync.
  5. Apple Address Book has a remove duplicates command under Card in the Apple Address Book Menu bar.
  6. The results will sync back to Entourage....eventually.
  7. Turn off Sync Services in Entourage if you no longer wish to sync.

Tip: Before syncing I suggest you export your Entourage data as .rge file under File > Export as .olm file and only check contacts. In Apple Mail I suggest you also export File > Export > AddressBook archive. Sometimes sync can erase all. If you have a backup it's easy to restore.

slance310 say:

Hi,

That is most likely caused by a driver not playing nicely with the Fast Startup feature...

Try turning off Fast Startup and see if that helps...

This feature only works when you do a SHUTDOWN and then Boot.

It doesn't effect a RESTART.

To turn it off or on...

Go to Control panel...Power Options, and select Choose What the Power Buttons Do on the left.

Then select Change Settings That Are Currently Unavailable near the top center of screen...

Lower down on the window, uncheck Fast Startup.
