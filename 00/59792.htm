Hi,

We appreciate you for posting back the status of your concern. This issue was actually related to the network adapter and dependency services to run it. To resolve this issue, we suggest that you follow the steps below:

Method 1:

  1. Press Windows key + R and put services.msc in Run dialog box. Click OK to open Services.
  2. In the Services window, scroll down and look for the Network Location Awareness service. Since you’re facing this issue, this service may not be running. Double click on same service.
  3.  In the Properties window for Network Location Awareness service, at General tab, set the Startup type to Automatic. If the service status is Stopped in your case, click Start button there. Click Apply followed by OK.
  4. You can now close the Services window and boot the system into Safe Mode with Networking.
  5. In this mode, once the issue is resolved, you can now restart computer in normal mode and Store must be opened without any error.

Method 2: Check Date & Time

  1. You will not be able to connect to the store if your system time is not accurate. Press Windows key + S  and then type in Date and Time.
  2. Now you can click Change date and time to set the system to the correct time. Also, make sure that you set the correct Time Zone.
  3. Now try to open the Windows Store app and see if you are able to connect.

Method 3: Proxy Server

  1. You may also check if your system is going through some proxy server. If it is, you can try to disable the proxy server and see if that lets you connect to the store. You can do this by clicking on the gear icon at the top right, then clicking on Internet Options.
  2. Then click on the Connections tab and click on the LAN Settings button.
  3. Uncheck the Use a proxy server for your LAN box.
