john sr wilson say:

 Question is perfectly clear so I don't know why you ask for a screen shot (except it's a quick way to "answer" maybe)

It would be possible but not simple to write code to do this

This eg would add a superscript letter to all data labels in a column chart. It is only meant to be a "top of head" start. I doubt it will do what you want out of the box but might be a pointer.

Sub dataLbl()
Dim ocht As Chart
Dim pt As Point
Dim charCnt As Long
Dim L As Long
Dim S As Long
Dim raylabls() As String
On Error Resume Next
raylabls = Split("A,B,C,D,E,F", ",")
Set ocht = ActiveWindow.Selection.ShapeRange(1).Chart
If Not ocht Is Nothing Then
For S = 1 To ocht.SeriesCollection.Count
With ocht.SeriesCollection(S)
For L = 1 To .Points.Count
Set pt = .Points(L)
If pt.HasDataLabel Then
pt.DataLabel.Text = pt.DataLabel.Text & raylabls(S - 1)
charCnt = pt.DataLabel.Characters.Count
pt.DataLabel.Characters(charCnt, 1).Font.Superscript = True
End If
Next L
End With
Next S
End If ' no chart
End Sub
