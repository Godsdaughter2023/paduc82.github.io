frederik long say:

I would use both.

The router provides a basic barrier between your PC and the Internet. It is very difficult to get through it from the outside (unless you already have some agent on the inside). Is effective even when the software firewall is temporarily turned off (which you have to do once every so often while testing things).

The software firewall gives you additional flexibility so that you can enable or disable certain functions, e.g. pinging or port access.

I recommend you use both. To me having a router plus the native Windows firewall is adequate. Many people prefer a more versatile third-party software firewall instead. Note that having two software firewalls is counter-productive.

bruce chambers say:

Ackman wrote:

I am presently using ZAP with Windows 7 Ulitmate 32bit.

My new router, DIR-655 (once I get it working) has a firewall built in.

      If you use a router with SPI/NAT, it's still a very good idea to use a software firewall that can monitor out-going traffic. SPI/NAT-capable routers do nothing to protect the user from him/herself (or any "curious," over-confident teenagers in the home).  Again -- and I cannot emphasize this enough -- almost all spyware and many Trojans and worms are downloaded and installed deliberately (albeit unknowingly) by the user.  So a software firewall, such as MSE, Comodo, Sygate or ZoneAlarm, that can detect and warn the user of unauthorized out-going traffic is an important element of protecting one's privacy and security, by alerting you to an unwanted malware application's activity. (Remember: Most antivirus applications do not even scan for or protect you from adware/spyware, because, after all, you've installed them yourself, so you must want them there, right?)
     When I ran WinXP, I used both a router with NAT and Sygate Personal Firewall, even though I generally know better than to install scumware.   Now that I'm using Windows 7, I still use a software firewall (MSE) in addition to my router.  When it comes to computer security and protecting my privacy, I prefer the old "belt and suspenders" approach.  In the professional IT community, this is also known as a "layered defense."  Basically, it comes down to never, ever "putting all of your eggs in one basket."
     Having said that, it's important to remember that firewalls and anti-virus applications, which should always be used and should always be running, while important components of "safe hex," cannot, and should not be expected to, protect the computer user from him/herself. Ultimately, it is incumbent upon each and every computer user to learn how to secure his/her own computer.

I find the easiest way to turn a text document or anything on your monitor into a jpg/jpeg is to screenshot it.

You can do this by: 
1) Open whatever you want to capture an image of, in this case the text document in notepad.
2) Look at your keyboard, there should be a button called "Print Screen" or PrtScrn.
3) Sometimes if you press this button alone it will capture the screen, often you have to press a function button at the same time as you press print screen because it's a secondary function of the button in question. By function button I mean the left shift button or "Fn" button. Press that and the print screen button at the same time.
4) This will save a jpg of the screen to your temporary clipboard.
5) Open your start menu and type into the search box "Paint" and open the paint program.
6) Click on the paste icon (or press the Ctrl and V buttons at the same time). 

7) This should have pasted the image into paint.
8) Click the save icon, select where to save the file and an appropriate file name.
9) From the drop down menu called "save as type", select JPG or JPEG and click save.
10) The file should then be saved as a JPG.

I hope this solved your problem :)

- After you execute all commands close Command Prompt and check if the problem is fixed.
