bob jones aka: cybertaz say:

If the original content was Numbers you could use a custom Format, but since you specified "text strings" that won't work. There is no built-in feature for doing this, but the process isn't difficult using the =CONCATENATE() function. Assuming the current content starts in A1 & includes something such as F21RT0005 to which you want to add a prefix of ID #: including the spaces;

  1. In an available column enter the formula =CONCATENATE("ID #: ",A1)
  2. Copy the formula down as far as necessary, leaving the cells selected
  3. Copy the resulting column
  4. Select the original cells beginning with A1
  5. Go to Edit> Paste Special - Values, OK to replace the original content with the new result
  6. You can then Clear the column of formulas you created

tomhathaway say:

I'm assuming that you have already set up under your C:\Users\xxxxx\Documents\Excel directory (where "xxxxx" is your user name) several additional sub-folders in which to store various Excel Files.

Assuming that you already have this file structure in place, once you have opened Excel, click on the tab at the far left which is labeled "FILE".   This should take you to a new screen with an arrow in the upper left hand corner and a list of commands down the left hand side.  (The arrow will take you back to your new blank worksheet which should have opened when you opened Excel).  Click on the bottom command which should be "Options".  This should open a windows dialog box entitled Windows Options and have a list of commands down the left side.

Now click on the command "Save". This should open a new dialog box.  In the middle of the page you should see a line entitled "Default File Location:" and in the window it should show the path which corresponds to the location where Excel is CURRENTLY going when you click on the "Open File" Command in Excel.  In this window you will need to type the path of the location that you DO want the program to open in (assuming that it's different than where it CURRENTLY opens).  Unfortunately you will have to TYPE it in as there is no BROWSE feature to navigate to it. 

When you've finished, click "OK" and then close the Excel.  When you re-open it, and go to the Open File command it should open the dialog box where you specified.

If for some reason you cannot get Excel to accept the command you type in, or it won't save, then you have some sort of "security" issue where your user name is not being recognized or accepted by the program.

I had that problem and am embarrassed to admit that while I was able to find a fix, I've forgotten what I did.  But I was able to locate something that allowed me to resolve that issue.

Good Luck!

Tom
