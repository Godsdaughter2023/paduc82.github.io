justinecallen say:

Disable Pinch Zoom Permanently

I have an HP Pavilion with Windows 10. I have turned off my pinch zoom by going to settings, devices, mouse and touchpad, additional mouse options, synaptics settings and unclicking the pinch zoom box. While this works as a temporary solution, every time I restart my computer the pinch zoom is enabled again. I would like to disable the pinch zoom permanently so I do not have to go through the settings each time I turn on my computer. Is there a way to make the setting change permanent?

thalazycoder say:

Hi there,

There seems to be an issue with the latest Synaptics driver resetting user settings on reboot. There's a workaround till Synaptics updates and fixes the issue:

  1. Hit Win + R > type "regedit" > hit OK. This will open Registry Editor.
  2. Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Synaptics\SynTP\Install.
  3. In the Install folder, you will find a key called DeleteUserSettingsOnUpgrade. Double-click on this key.
  4. Set the value to 0. Click OK.
  5. Go to Synaptics settings and set everything to the way you want. Reboot the computer.
