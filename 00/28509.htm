Windows Defender won't turn on after Windows 10 install from Windows 7

UPDATE

After trying the one reply I did receive (Thanks for the reply!) that did not correct the problem I tried various other options I found across the user boards etc.

Here are the steps I tried that eventually correct my problem. Windows Defender now turns on and it running:

Removed existing antivirus and antispyware software

  • Although I had an old version of Trend Micro installed it was not running. I downloaded a toolkit to uninstall the Trend Micro software in case this was the problem.
  • To make sure the Microsoft Security Essentials was completely removed with the Windows 10 installation I ran a command from the command prompt “MsiExec.exe /X{75812722-F85F-4E5B-BEAF-3B7DA97A40D5}” .  It responded back that it couldn’t remove something that wasn’t there. So the Windows 10 installation did remove this correctly.
  • These steps did not correct my problem.

 

SFC scan

  • I ran the System File Checker (SFC) tool repairs corruption in system files. This tool is used to verify whether Windows Defender is corrupted or not. Follow this KB article: https://support.microsoft.com/en-us/kb/929833
  • These steps did not correct my problem.

 

Restart Security Center Service

  • To restart Security Center service, Ifollowed these steps:
    1. Press Windows key + R. This will open Run. Alternatively, you can go to Start and search for 'Run'.
    2. In Run dialog box, type 'services.msc' and hit enter.
    3. In Services, search for 'Security Center'.
    4. Right click on 'Security Center' and click on 'Restart'.
  • These steps did not correct my problem.

Delete conflicting Registry Entry

  • Some malwares adds malicious entries in registry that blocks real anti-viruses from running. To remove these entries, follow these steps:
    1. Press Windows key + R. This will open Run. Alternatively, you can go to Start and search for 'Run'.
    2. In Run dialog box, type 'regedit' and hit enter. This will open Registry Editor.
    3. In Registry Editor, navigate to HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\CurrentVersion\Image File Execution Options.
    4. In this key, if you find any entry for MSASCui.exe, MpCmdRun.exe or MsMpEng.exe then right click on it and click on Delete. If you do not find any of these entries then it is normal and you don't need to do anything.

 

Confirm Whether Windows Defender is Not Disabled by a Registry Key

  • Press Windows key + R. This will open Run. Alternatively, you can go to Start and search for 'Run'.
    1. Once the Registry Editor window arrives, travel this way – HKEY_LOCAL_MACHINE => SOFTWARE =>

      Policies => Microsoft => Windows Defender

2.  On the right pane look for the DWORD DisableAntiSpyware.

3.  Put its value as 0 only
