mouran yan msft support say:

Hi Nadsurf,

 

No, the DKIM put in place by Microsoft will not continue to work as before after you created a custom DKIM record for your domain.

 

According to the “Default behavior for DKIM and Office 365” part of this article, If you do not enable DKIM, Office 365 automatically creates a 1024-bit DKIM public key for your custom domain and the associated private key which we store internally in our datacenter. By default, Office 365 uses a default signing configuration for domains that do not have a policy in place. This means that if you do not set up DKIM yourself, Office 365 will use its default policy and keys it creates in order to enable DKIM for your domain. So, when setting up DKIM yourself, it will override the Office 365 default policy and keys.

Use the UDFs just like regular built-in Excel functions:

1st UDF format example

=SumNumerators(A1:A10,5)

2nd UDF format example:

=SumTextNumerators(B1:RQ400,"Yes",5)

Note that the range could include columns that don't even have "yes" as possible entries in them, they would just be ignored and result in a count of 0 for those columns.

The code:

Function SumNumerators(cellRange As Range, maxValue As Long) As Long
'INPUTS:  cellRange = reference to cells to be examined
'         maxValue is limiter for value of numerators found.
'OUTPUT:  total of all numerators given maxValue limitation
'
'ASSUMPTIONS:  The contents of the cells is in the form of n/d
'              where the division symbol / displayed in it, as 1/3 or 6/8, etc.
'
  Const divSymbol = "/"
  Dim anyCell As Range
  Dim oneValue As Long
  Dim tempTotal As Long
 
  For Each anyCell In cellRange
    'cell must have / in it
    If InStr(anyCell, divSymbol) > 0 Then
      oneValue = Val(anyCell)
      If oneValue > maxValue Then
        oneValue = maxValue
      End If
      tempTotal = tempTotal + oneValue
    End If
  Next ' end anyCell loop
  SumNumerators = tempTotal
End Function

Function SumTextNumerators(cellRange As Range, conditionText As String, maxValue As Long) As Long
'INPUTS:  cellRange = reference to cells to be examined
'           cellRange can include cells that have other types of data in them
'            - they should be ignored since they would not meet the =conditionText test.
'         conditionText is text to test for in the cells.
'         maxValue is limiter for value of numerators found.
'OUTPUT:  total of all matching conditionText given maxValue limitation
'
'ASSUMPTIONS:  The contents of the cells is text
'
  Dim anyCell As Range
  Dim oneValue As Long
  Dim colTotal As Long
  Dim tempTotal As Long
  Dim firstRow As Long
  Dim lastRow As Long
  Dim firstColumn As Long
  Dim columnCount As Long
  Dim rowPtr As Long
  Dim colPtr As Long
  Dim theCondition As String
 
  theCondition = UCase(Trim(conditionText))
  firstRow = cellRange.Cells(1, 1).Row
  lastRow = cellRange.Cells(cellRange.Cells.Count).Row
  firstColumn = cellRange.Cells(1, 1).Column
  columnCount = cellRange.Columns.Count
  For colPtr = 0 To columnCount - 1
    colTotal = 0 ' reset
    For rowPtr = firstRow To lastRow
      If UCase(Trim(Cells(rowPtr, colPtr + firstColumn))) = theCondition Then
        colTotal = colTotal + 1
      End If
    Next ' end rowPtr loop
    If colTotal > maxValue Then
      colTotal = maxValue
    End If
    tempTotal = tempTotal + colTotal
  Next ' end colPtr loop
  SumTextNumerators = tempTotal
End Function
