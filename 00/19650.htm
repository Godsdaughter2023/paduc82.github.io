uk troll say: Another thing to check is the Default Application set to open .doc/.docx files.

If you didn't have everything shut down properly when you were updating to Office 2011, or had faults in your Permissions files (as mentioned in the previous post) that hadn't been fixed by the time you did the install, then your Mac may be getting confused as to which Application should be opening these files - i.e. it may still think they're meant to be opened by an older version of Word, bits of which it can no longer find.

To check that these types of file are set to open with Word 2011 as the Default Application:

- In Finder View, select one of your .doc files and choose ' Get Info ' or ' Cmd - I '.

- Near the middle of the Info Window, you'll see ' Open with: ' (you may need to expand the section).

- Check that the Application shown is the latest version of Microsoft Word, by clicking on the Application Name in the pop-down menu bar.

- This should read: ' Microsoft Word (default) (14.1.2) ' if your computer is set to use the latest version of Word (as at 28Jun11) as the default Application.

- If not, then using the pop-down menu, either select the correct version of Word from the list, or navigate your way to the correct version using the ' other... ' option at the bottom.

- Then click the ' Change All... ' button, just below, to make the correct version of Word the Default Application to open all .doc files in the future.

- Go through the same process for one of your .docx files, to set the correct version of Word as the Default Application for .docx files as well.

This may also help your problem.


- As an aside, this feature can be useful on a temporary basis, for example if you want to force certain picture files such as .jpg to open with Preview instead of, say, Photoshop; which takes a long time to fire up if you just want to check images quickly.

Hope this is of use...
