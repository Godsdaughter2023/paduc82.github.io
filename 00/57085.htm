spiritx say:

Hi,

ntoskrnl.exe is a Windows component which means something else drove it into Fault and it

probably is hardware related.

If the system has on-board video you can remove the Nvidia video card to see if that helps. If

not you would need a replacement card, hopefully that is known to be good, to try in its place.
And there the issue may not be the video card though it is suspect.


Definitely check with MSI and Nvidia Support and forums (to check for known issues). Sometimes
you have to revert back to an older more stable driver (again the issue may not be the video driver).

Also check with Intel and run the Intel Driver Update Utility.

Rob - SpiritX


spiritx say:

Also don't run Driver Verifier all the time as it can cause instability when there would otherwise

be none. Only run it to check drivers after you have a BSOD when suspect drivers are involved.

Rob - SpiritX
