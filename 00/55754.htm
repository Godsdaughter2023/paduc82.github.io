    You can place a Symbolic Link in your OneDrive folder.  This can point to your music files without duplicating them.

    You need to run a Command prompt as Admin to create a symbolic link.

    C:\Users\keith\OneDrive\Music>mklink /d Abba C:\users\keith\music\abba

    So sorry you had to go through hours of "shell games'.  I guess since you indicated you were a father, I assumed you would be comfortable with a single command from a command prompt (surely you've used "dir" at some point, haven't you?).
