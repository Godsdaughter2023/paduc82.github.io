umesh p say: Hi Red Top Mahoney,

Thank you for using Microsoft Answers forums.

Verify that the Windows Media Player Network Sharing Service or Windows Media Connect network service is not running by doing the following:

1. Click Start, and then click Run
2. In the Run box, type services.msc
3. In Services, find Windows Media Player Network Sharing Service or Windows Media Connect (WMC), and then verify that the Status column indicates Started
4. Double click the service and click the Stop button, change the Startup type to Disabled
5. Restart the computer.

To reactivate the service repeat the process above, clicking the Start button instead of Stop, and changing the Startup type to Automatic.

Now try deleting the libraries.

For more information visit

http://windows.microsoft.com/en-US/windows7/Remove-items-from-the-Windows-Media-Player-Library
