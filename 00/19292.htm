set option button to blank - Microsoft Community

caminowalker say:

set option button to blank

i have an expense report form.  this can be used for any type of expense.  to quickly populate the account cell, you can choose one of three  option buttons - Cash Advance, Inventory, or Consultancy.  however, not all expenses will fall into one of these three categories so i want the option buttons to be unchecked when the form opens. 

 

also i am confused about whether i should use a form control or an Active X control.

 

thanks in advance for your help.

chip pearson say:

By "Form" do you mean just an organized worksheet, not a VBA UserForm? I would use the ActiveX controls.  The code to clear all the option buttons on the worksheet, use code like the following.

 

Sub ClearOptionButtons()
    Dim Obj As OLEObject
    For Each Obj In Worksheets("Sheet1").OLEObjects
        If TypeOf Obj.Object Is MSForms.OptionButton Then
            Obj.Object.Value = False
        End If
    Next Obj
End Sub


 

To clear one option button, use

 

Worksheets("Sheet1").OLEObjects("OptionButton1").Object.Value = False

 

To clear all the option buttons in a specific GroupName, use code like:

 

Sub ClearGroup()
    Dim GroupName As String
    Dim Obj As OLEObject
    GroupName = "Group1" '<<< CHANGE TO SPECIFIC GROUP
    For Each Obj In Worksheets("Sheet1").OLEObjects
        If TypeOf Obj.Object Is MSForms.OptionButton Then
            If StrComp(Obj.Object.GroupName, GroupName, vbTextCompare) = 0 Then
                Obj.Object.Value = False
            End If
        End If
    Next Obj
End Sub

 

This will clear all the option buttons whose GroupName property is "Group1".
