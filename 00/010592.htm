Hi Sharief,

Currently, there is no way to fully switch an install of Windows 10 Technical Preview from one language to another.  For example, if you installed an English (US) version of Build 9926, then updated to Build 10041 and changed your settings to one of the newly enabled Cortana languages, Cortana won’t be available.  In order for Cortana to be enabled, the initial install of Tech Preview must be done in the Cortana-supported language you want to be in.

To resolve your issue, please go to Windows 10 Tech Preview ISO to install the localized builds in English (United Kingdom), Chinese (Simplified), German, French, Italian, or Spanish.  After it is successfully installed, please set your Windows Update to the Fast ring to get the update for Build 10041 and install it.  You may have to wait for the update to be pushed to you.  Once Build 10041 is released to the Slow ring, the ISO files for Build 10041 will be available, and you can directly install the ISO for the new Cortana languages without having to perform the update.

If you choose to keep your personal profile through the upgrade to Build 10041, please check the following settings and may sure they are all aligned to the same languages and/or region:

  1. Go to Settings > Time & Language > Region & Language
    • Check the Country or region setting
    • Check the Windows display language (set as default) setting
  2. Then go to Settings > System > Speech
    • Check the Spoken language setting

To summarize, make sure the ISO you installed and your settings match as follows:

ISO Country or region Windows display language Spoken language

Abridging a Word Doc - Microsoft Community

zyzzyva57 say:

Abridging a Word Doc

I like to read conference calls, etc, which are often super long with a lot of fluff, as for example the following:

BECKY QUICK: So does that signify that you've stopped buying in Apple because of where prices have —

 

WARREN BUFFETT: Well-- maybe, maybe not.

 

BECKY QUICK: Okay.

What I would like to do, say in this doc, is cut-out all BECKY QUICK so as to have just WARREN BUFFETT's COMMENTS

Thank you

jay freedman say:

You can use a wildcard find/replace to locate paragraphs that start with a particular text (such as your example WARREN BUFFETT) and mark them in some way, such as changing the font or the color. Then a second find/replace can remove everything that isn't marked in that way.

Using your example:

  • Open the Replace dialog and click the More button.
  • Check the box for "Use wildcards".
  • In the Find What box, enter the expression (WARREN BUFFETT:[!^13]@^13)
  • In the Replace With box, enter the expression \1.
  • With the cursor still in the Replace With box, click the Format button, click Font in the menu, and set the font color to red (the second box in the Standard Colors group).

The Replace dialog should now look like this:

Click the Replace All button. Now every paragraph that starts with WARREN BUFFETT will be red, and the rest of the document will have the Automatic color (usually black).

Now to remove all of the Automatic text:

  • In the Replace dialog, clear the "Use wildcards" box and click the No Formatting button , and erase the contents of the Replace With box so it's blank.
  • Erase the contents of the Find What box. While the cursor is still in that box, click the Format button, click Font in the menu, and set the font color to Automatic (at the top of the color dropdown).

Now the Replace dialog should look like this:

Click the Replace All button. Everything that isn't red will be deleted.

Finally, you can press Ctrl+A to select everything, and press Ctrl+spacebar to get rid of the red color, leaving ordinary black text.
