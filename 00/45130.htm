Hi Patricio,

Changing the name of a USB can be done without formatting the USB. Follow the steps below:

     1. Open Notepad > Type the following code:

[autorun]

label=MY NEW USB NAME

Note: Change MY NEW USB NAME to you're desired name.

     2. Save the file as autorun.inf in your USB.

     3. Safely remove the USB then put it back.

     4. Hide autorun.inf by right-clicking the file > Select Properties > Check Hidden > Hit Apply then OK.

Keep us posted if you're able to replace the name of your USB.

I would like to inform you that USB 3.0 ports are either marked with the SuperSpeed USB Logo or the port is typically blue. Newer PCs have both USB 3.0 and USB 2.0 ports. If you want your SuperSpeed USB device to perform at top speed, find a USB 3.0 port and connect the device to that port. A SuperSpeed device that is connected a USB 2.0 port, operates at high speed.

 

In Windows 8, first, make sure that you have a USB 3.0 port and an xHCI host controller. If your SuperSpeed USB device is connected to the xHCI host controller, Windows 8 shows a "Connected to USB 3.0" message in specific portions of the Windows 8 UI. If the device is connected to an EHCI controller instead of your XHCI controller, the messages will instead read, "Device can perform faster when connected to USB 3.0".
