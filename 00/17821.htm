In Visio Professional (and with 2010, also in Premium), there are two separate solutions for doing Database schema diagrams and UML model diagrams.  However, they are not able to work together in the way that you describe.  If you want to create a comprehensive Database diagram using the Reverse Engineering capability of the "Database Model Diagram", you could possibly create your own shapes to represent the classes and essentially annotate the schema diagram.  But this would not be easy, and neither the DB model solution or the UML model solution have public APIs, so you can't automate either one.

There are likely other modeling tools that will get you much closer than Visio will out of the box.  Just by searching on "Model database and C# classes", I found several sites that you can scan for more information :http://www.bing.com/search?q=model+database+and+C%23+classes&form=QBRE&qs=n&sk=

diane poremsky mvp (slipstick.com) say:
Don't store an active pst file in OneDrive. It can lead to corruption and lost mail. You can store a backup in it (if you have space) but should copy it to a folder outside of onedrive to use it.

Do you have a godaddy account that supports IMAP? That would allow you to have the same email on all devices, including sent messages. It won't sync calendar and contacts though.

Better would be an office 365 exchange mailbox - godaddy is an office 365 reseller or you can buy directly from Microsoft for as little as $4/month for email-only ($5 for additional business services). This will sync calendar, contacts, tasks, mail to all devices. (IMHO, I think going directly through microsoft is a better deal.)

Another option is an Outlook.com account but you can't use your custom domain with outlook.com.

Finally, there are sync utilities to sync the desktop and laptop. See https://www.slipstick.com/outlook/sync-outlook/... for a list.

ronbaysden say: I have both devices set up as IMAP accounts on Godaddy.  How do I get outlook to sync the send and other folders?

karl timmermans [outlook mvp 2012-2018] say:
From the description, you have the email account(s) configured as a POP3 account which does not sync data across devices. POP3 only downloads <Inbox> data to the local device where the PST is stored.

To sync all your data (email, contacts, calendar etc) you would need to either use an Outlook.com or hosted Exchange account (i.e. Office 365 email account).

Some background info on email account protocols

Understanding Email Protocols (POP, Imap, MAPI, EAS, Exchange)
https://www.contactgenie.info/understanding-ema...

As a sidenote, storing PST files on OneDrive that are actively being using by Outlook on a device is not supported and can lead to file corruption.

The sent items folder should sync - it did when i tested it with a godaddy account. (I logged into webmail to see if it synced up).
