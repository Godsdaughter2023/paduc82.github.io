d pauldalton say:

How to change the defaults for tables in Word 2013?

Every time I create a table in Word 2013, the first thing I find myself doing is going to Table Properties, clicking on 'Options', and then changing the left and right cell margins from 0.08" to 0.01".  In a document where I have a lot of tables, it's a real annoyance to have to do so many manual changes to have the tables formatted as I prefer.  

For columns, I have a similar concern, in that - when I do put text into two columns - I greatly prefer to have a line between the columns and for the spacing to be set at 0.2".   But even when I set up columns the way I like and select "Set as Default" in Page Setup, those choices I made don't "stick".  

I would LOVE to know how to change the settings in Word 2013 so that:

1.  Whenever I create a table, the default left/right cell margin in that Table will always be 0.01".  

     and

2.  Whenever I select "Two" from the "Columns" tab, the two columns will be spaced 0.2" apart with a line between them.  

Is there a way to accomplish these changes?  

Thanks,

Paul


doug robbins - mvp office apps & services (word) say:

For the table setup, insert a table and then on the Table Tools Design tab of the ribbon, click on the dropdown arrow in the Table Styles Gallery

Then click on Modify Table Style

And then in the Modify Style dialog, click on Format and then on Table Properties and make the desired changes.  Before clicking on OK in the Modify Style dialog, select the radio button for "New documents based on this template"

For the Text Columns requirement use a macro containing the following code:

With Selection.Sections(1).PageSetup.TextColumns
    .SetCount NumColumns:=2
    .Spacing = InchesToPoints(0.2)
    .LineBetween = True
End With
