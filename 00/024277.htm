jay freedman say:

The majority of clip art files supplied by Office 2007 are in Windows MetaFile (.wmf) format, which by default is associated with Microsoft Paint. It appears that the .wmf extension has become associated with Windows Media Player instead.

If that is the case, try to fix it by opening the Control Panel and selecting Default Programs. Click "Associate a file type or protocol with a program". Scroll down and click on the .wmf extension in the Name column. If the entry in that row and in the Current Default column isn't Paint, then click the Change Program button at the top right and choose Paint from the dialog box. Close the Control Panel and try viewing clip art in Word.
