How can I keep the numbers in the main text the same size as they are now, while reducing their corresponding numbers in the footnote itself?

I have Word 2007.

Thanks!

suzanne s. barnhill say:

Make sure that the Footnote Reference style does not  have a defined font or font size. It should be defined as Default Paragraph Font + Superscript, period (in Word 2010, this is expressed as "Superscript, Style: Hide until used, Priority: 100  Based on: Default Paragraph Font"). If the style has acquired any font formatting, it may not be possible to remove that formatting directly, but you can try.

  1. With a footnote reference selected, right-click and choose Style... In the Style list, Footnote Reference should be selected.
  2. Click Modify...
  3. In the Modify style dialog, select the font and press delete in hopes that this will clear the font box.
  4. Repeat for the font size.

If this doesn't work, your only recourse may be to reset the Footnote Reference style to its default formatting. The way to do this is to select the style in the Style list and click Delete. This will reset the style to the default, but it will also remove it from all your footnote reference marks, and you'll have to go through the document and reapply it to all the footnotes, so this is a pretty drastic solution.

Assuming that there is never more than one cross-reference to a given figure, it is possible (but a lot more work than I'd consider reasonable).

After inserting the cross-reference, insert a bookmark on (or next to) the cross-reference. Then move to the figure, and insert next to it a cross-reference to the bookmark.
