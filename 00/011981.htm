How can I output a summary of a checklist that only includes the "no"s?

I have a workbook with multiple sheets; they are checklists, and answers are either Yes, No, or Not Applicable.  It's coded so that the responses are represented by numbers in a column, No being 2.

I want to summarize the checklist on a final page, and output the rows that have a "no" answer.

What's the best way for me to do this?

Thank you!!!
gary's student say:

Here is the macro:

 

Sub GatherData()
Dim ws As Worksheet
col = "C"
k = 1
For Each ws In Worksheets
    If ws.Name <> "summary" Then
        ws.Activate
        n = Cells(Rows.Count, col).End(xlUp).Row
        For i = 1 To n
            If Cells(i, col).Value = 2 Then
                Cells(i, 1).EntireRow.Copy Sheets("summary").Cells(k, 1)
                k = k + 1
            End If
        Next
    End If
Next
End Sub

 

It assumes that the summary is in a sheet named "summary".  It also assumes that the column containing the answer is column "C".  You can change these.

Windows OS Info from registry

The Windows API GetVersionEx() typically returns the OS major / minor and build version information
But quite interestingly  this might not exactly match the actual system versions as I have observed with the latest update to windows server 2012  : the 2012 R2
This API typically presents the following 2 implementation caveats :
  1. Applications have to maintain the mapping between the productName and the version for each operating system, which breaks when the application runs on a newer version of the OS.
  1. Determining the OS flavor - professional, standard etc  would require one to get the GetVersionInfoEx followed by GetProductInfo to collect the full version information.
An alternative option is to collect the version details from the windows registry  key:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion
