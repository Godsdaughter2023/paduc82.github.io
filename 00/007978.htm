robert long [msft] say: Hi Cobus -

Thanks for sending me your workbook and working with me on this offline.

So we investigated this and this does indeed seem to be a bug.  The bug is that in the web app we strip everything in a hyperlink URL after (and including) a ‘#’.  When I click on the hyperlink in the client it works correctly, the URL is intact and it takes you to the “test” anchor on the anchortest.htm page.  When I click on it in the web app, it strips away the #test and it just takes you to the top of the anchortest.htm page.  The reason you are seeing a problem is that some of your URLs contain a # and we strip that and everything after it away.    We’ve logged a bug for this, although I can’t say at this point when it would be fixed. 
 
More info:
The URLs in your workbook look like this. Notice the “#” (I replaced parts of the URL with x's): 
https://skydrive.live.com/?cid=xxxxxxxxxx&id=xxxxxxxxxx%211786#cid=xxxxxxxxxx&id=xxxxxxxxxx%211813
 
When you click that URL in Excel desktop app, it works OK and in the browser you will see a URL like this (notice there is no “#” anymore):
https://skydrive.live.com/?cid=xxxxxxxxxx&id=xxxxxxxxxx%211813
 
So a workaround here is you could replace the current hyperlinks in your workbook with the ones that no longer contain “#”s.  These should work fine from Excel Web App.  

Sorry for the inconvenience and thanks for reporting this, and thanks for using Excel Web App.
 
Rob

tasos_k say:

Okay,

scenario is as follows:

merge all csv files in one csv file.

Expected results:

In column A are unique values/names and in colums B-C-D,.. are numbers

in a file named: MERGE ALL CSV.csv

and in path c:\tmp\

(change path and name as your needs)

XXXXXXXXXXXXXXXXXXX

you can do that you want using a vba macro

step1

take a look here...

https://www.youtube.com/watch?v=vtTa8y6i_rM

step2

Save As, your Workbook with extension .xlsm (macros enabled)

step3

1) press ALT+F11 to open Visual Basic

2) from the ribbon:

select, Insert > Module and paste the code below on the right

Sub Macro_01()
'Dec 23, 2014
Const SourcePath = "D:\trial\"     '<< source path
Const TargetPath = "C:\tmp\"    '<< target path
Const TargetName As String = "MERGE ALL CSV" '<< merge csv file name
Dim wb As Workbook, wb1 As Workbook
Application.ScreenUpdating = False
Set wb1 = Workbooks.Add
Dim sFile
sFile = Dir(SourcePath & "*.csv")
Dim t As Long, r As Long
t = 0
Do While sFile <> ""
Set wb = Workbooks.Open(SourcePath & sFile, xlDelimited, local:=True)
r = wb.Sheets(1).Cells(Rows.Count, "A").End(xlUp).Row
If t = 0 Then
Range("A1").Resize(r, 2).Copy wb1.Sheets(1).Range("A1")
Else
Range("B1:B" & r).Copy wb1.Sheets(1).Cells(1, t + 2)
End If
t = t + 1
wb.Close False
sFile = Dir()
Loop
Application.DisplayAlerts = False
With wb1
.SaveAs TargetPath & TargetName, FileFormat:=xlCSV, local:=True
.Close False
End With
Application.DisplayAlerts = True
Application.ScreenUpdating = True
MsgBox "done"
End Sub

3) Press ALT+Q to Close Visual Basic

4)To run the macro, press ALT+F8, 

select <Macro_01> from the list and click the run button.
