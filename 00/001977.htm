Private Sub ButtonName_Click()

End Sub

Private Sub ButtonName_Click()
If MsgBox("Are you sure you want to print the report?", _

--------------------------------------------------------------------------------------

setlocal

:End
Endlocal

--------------------------------------------------------------------------------------
Run the script on the computer.
Start an Office application (such as Excel), and then select File > Account.
