rs.Open cmd, , adOpenStatic, adLockOptimistic

dr. strangelove say: By installing avg, you now have more than one real-time anti-malware product installed, which you cannot have.  Doing so will cause performance problems and system instability.

Remove AVG, and any other real-time anti-malware product, leaving only MSE.

However, if you insist on removing MSE:

You should be able to uninstall from the “Add/Remove Programs”, or “Programs and Features”, depending on your OS.  If that fails, you can use this procedure provided by forum contributor Aiscer.

1.     Download the MSE installer file from the website and save the file on the Desktop.  This will allow you to find it easily after the download is done.

2.     Now press the Windows key (or Start key) and R both at the same time. This will open the RUN dialog box.

3.     Next, delete/clear any entries inside RUN, then drag the MSE installer file into the RUN dialog box.

4.     Depending on your Operating System, after dragging the file into RUN, it should appear as one of the three below:

·         C:\Documents and Settings\[YOUR USERNAME]\Desktop\mssefullinstall-x86fre-en-us-xp.exe

·         C:\Users\[YOUR USERNAME]\Desktop\mssefullinstall-x86fre-en-us-vista-win7.exe

·         C:\Users\[YOUR USERNAME]\Desktop\mssefullinstall-amd64fre-en-us-vista-win7.exe

5.     Now put a SPACE at the end of the line, followed by a  /U  as seen below:

·         C:\Documents and Settings\[YOUR USERNAME]\Desktop\mssefullinstall-x86fre-en-us-xp.exe /U

·         C:\Users\[YOUR USERNAME]\Desktop\mssefullinstall-x86fre-en-us-vista-win7 /U

·         C:\Users\[YOUR USERNAME]\Desktop\mssefullinstall-amd64fre-en-us-vista-win7.exe /U

6.     Click OK.

7.     Restart your system after the uninstall.

shane devenshire say:

Is this a Data Validation drop down?  And when you say you want nothing to appear when they pick No Selection do you mean in the drop down list?

You can do this if you are using a data validation list - it isn't the best solution but it may be satisifactory:

In 2007:
1. Highlight all the cells on the rows you want formatted (the cell or cells where the data validation list is, lets say A1)
2. Choose Home, Conditional Formatting, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following formula:

=A1="No Selection"

5. Click the Format button and choose Font tab, and pick White for the font.
6. Click OK twice
