ossiemac say:

They need to be printed as separate print jobs one after the other. Because you are changing zoom then the worksheets need to be selected and the zoom set between each print job with the VBA code. Reason for this is that zoom applies to the Active Window.

If you are page numbering then you need a variable that you increment with the code to retain the page number plus use a variable (probably a count from the array) for the total number of pages. The variables are used in the page setup like the following example in lieu of the default code that recording the page Setup produces.

    Dim lngPgeNo As Long
    Dim lngNoPges As Long
   
    lngPgeNo = 11       'For testing only. (Will need incrementing in the code)
    lngNoPges = 20      'For testing. Obtain value from the array.
   
    With ActiveSheet.PageSetup
        .RightFooter = "Page " & lngPgeNo & " of " & lngNoPges
    End With

Note that if you have already set the  Page Setup then by restricting the code to only the setup changes required then the remaining existing page setup parameters remain unchanged.

gaprogrammer say:

I realized the solution to my printing problem and am able to have different scale, orientation, etc AND send it out in one print job.   The problem was that once I executed my print function, the PrintOut method was actually selecting all the worksheets in my array of sheet names.  I couldn't tell they were selected, and so when I made an adjustment to any of the sheets, it made it to all the selected sheets!    I added code to save and restore the active worksheet and have accomplished what I needed.  Below is a code snippet in case anyone needs help with the same issue.

And thank you OssieMac for your replies.

Public Sub PrintReport()

    Dim ws As Worksheet
    Dim nRow As Integer
    Dim Response
    Dim nSheets As Long
    Dim nArrayPos As Integer
    Dim aSheets()
    Dim sActiveSheet As String
    
   ' Save currently active worksheet
    sActiveSheet = ActiveSheet.Name
    
    ' Make sure user is ready to print
    Response = MsgBox("Ready to Print the Package?", vbYesNo, "Print All?")
    If Response <> vbYes Then
        Exit Sub
    End If
    
  '    ActiveSheet.PageSetup.LeftFooter = Range("Version").Text
  '    ActiveSheet.PageSetup.CenterFooter = ""
    
    ' Using Overview Sheet as driver because when add new adjustment periods, we need to print those sheets too, and we already have added
    ' the name to the overview sheet.   In this way, Staff can manage which sheets get printed.
    
    ' This is a function I wrote to get rid of automatic page breaks on certain worksheets, and
    ' to  insert some between data tables on the same page so get a new page for the next
    'table
    AdjustPageBreaks
    
    ' Master worksheet for workbook that lists all sheets. Has a column that indicates whether a sheet should be printed
    Set ws = Sheets("Overview")
        
    With ws
        
        ' Tried several methods to find the first non-blank row but none consistently worked or either got overflow error.
        ' Just decided to start at known row with data and go from there
        
        
        ' This first Do Loop is merely counting then number of sheets to be printed so I can reset the array dimension
        nSheets = 0
        nRow = 2
        
        ' Column A contains the sheet name
        Do While .Cells(nRow, "A").Value <> ""
            nRow = nRow + 1
            
            ' Column F contains a Y if the sheet should be printed as part of the package
            If .Cells(nRow, "F").Value = "Y" Then
               nSheets = nSheets + 1
            End If
        Loop
        
        ' Create array of worksheets that should be included in the print job based on number calculated above
        ReDim Preserve aSheets(1 To nSheets)
        nArrayPos = 0
        
        nRow = 2
        
        ' Column A contains the sheet name
        Do While .Cells(nRow, "A").Value <> ""
            nRow = nRow + 1
            
            ' Column F contains a Y if the sheet should be printed as part of the package
            If .Cells(nRow, "F").Value = "Y" Then
               nArrayPos = nArrayPos + 1
               aSheets(nArrayPos) = .Cells(nRow, "B").Value
            End If
        Loop
       
    
    End With
    
    ' used on Error statement because got error message if user Previewed, then hit Print within preview, and subsequently cancelled the print
    On Error Resume Next
    
    ' Using PrintOut Method causes all sheets to be printed in one print job, so other user's pages are not mixed in
    Sheets(aSheets).PrintOut Copies:=1, Preview:=True ', IgnorePrintAreas:=True    ', PrintTofile:=True ', PrToFileName:=SavePath
                 
    ' Deactivates all the worksheets that are apparently activated/selected by using the PrintOut method with an array of names.  Select the one
    ' that was active when Print function invoked
    Sheets(sActiveSheet).Select
   
    
End Sub
