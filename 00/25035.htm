There is no way to do it with a field code, which has no way to know where the clicked hyperlink was.

But Word does have a way to behave exactly like a browser: Press Alt+left arrow (the "Back" shortcut in IE, Firefox, etc.), or customize the Quick Access Toolbar to add the Back button (which is in the All Commands category).

pesach shelnitz say: There is a key that takes you back to the previous location in Word 2007 and other versions of Word. It's Shift+F5. Of course, if you don't like that key combination, you can assign different key combination to the GoBack command.

jay freedman say:

The GoBack (Shift+F5) command isn't quite the same as the Back (Alt+left) command.

GoBack will take you to the most recent point where you made an edit (typing or changing formatting); two more presses take you to the two recent edit points before that; and the fourth press starts cycling through the edit points again. Clicking a hyperlink doesn't count as an edit point, so GoBack won't take you to the hyperlink unless you also made some edit there within the last three edits.

The Back command ignores edit points, but does return to the hyperlink you clicked.

don cuthbert say:

That sometimes means that you are trying to boot a GPT disk in a Legacy BIOS mode device. In Legacy BIOS mode, the disk has to be marked Active to boot.  Check your BIOS settings and see if UEFI is available.

If this is an older Dell and only has Legacy BIOS, you can try booting Windows install media on USB.  Then get a Command Prompt to run Diskpart to see if the disk is GPT or MBR and if the System partition is marked Active.

paulewills say: When you 1st bootup, there should be a notice showing for a short bit of time telling you which key to hit to get into your BIOS, usually F1, F2, or, DEL. If you can get into your BIOS, check to make sure your system drive is listed first in the boot order.
