jay freedman say:

The fact that the form opens in read-only mode (technically known as Protected View) isn't related to the forms protection. It has to do with how the file is transmitted from one computer to another and in what folder it's stored. There are also options in the Trust Center in Word to disable Protected View (http://blog.laptopmag.com/disable-protected-view-microsoft-wor), although doing so is analogous to pulling the batteries out of your smoke detector.

It's better to train your users to work properly:

  • Never open a Word file directly from an email attachment and edit it. The edits will probably be lost when the file is closed, because the changes are saved into a temporary file that will immediately be deleted and eventually overwritten. See http://www.gmayor.com/outlook_attachments.htm for the kinds of hoops you'd have to jump through to recover the changes.
  • Save the attachment (or a file received any other way) in the Documents folder or another trusted location (as indicated in the Trust Center's page about Trusted Locations).
  • If, after the previous two steps, a document contains macros and the macros won't run, right-click the document file in Windows Explorer, click Properties, and click the Unblock button if there is one there.
