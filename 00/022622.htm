A space plus an underscore at the end of a line will create a break in an otherwise single line of code. However, it cannot be directly implemented in the middle of text that is enclosed in double quotes.

If you have long text that is enclosed in double quotes then you need to close off the double quotes and add an ampersand before the space and underscore then start the next line with the double quotes again like the following example.

Change the following line:

MsgBox "I want this text to wrap around into 2 lines because it is too long"

to the following:

MsgBox "I want this text to wrap around" & _

        "into 2 lines because it is too long"

Sub MyPrint(ParamArray Args())

What about outputting the string to a text file and then open with Notepad where you can set the Format to Word wrap. Notepad will open over the VBA editor window and even remembers its size and location from the previous setting so you can set it up over the Immediate window area.

Example code:

Sub Test()
    Dim strDummy As String
   
    'Following creates a long string without line feeds for testing.
    strDummy = "Yes, the underscore is great when I manually create the line. " & _
                    "But some of my code prints diagnostic messages out to the Immediate Window, " & _
                    "and the text being printed is of variable length.  Some of it is quite long. " & _
                    "And the width of Immediate window varies according to the user, " & _
                    "depending on the prevailing needs of the moment. " & _
                    "So it would be nice if stuff printed to the Immediate window " & _
                    "automatically wrapped according to the window width."
   
    Call PrintToTextFile(strDummy)

End Sub


Sub PrintToTextFile(strToOutput As String)
    Dim intFileNum As Integer
    Dim strPathAndFile As String
    Dim varMsgResponse As Variant
    Dim varOutput
   
    intFileNum = FreeFile 'The next free filenumber
   
    strPathAndFile = ThisWorkbook.Path & "\" & "TextFile.txt"
   
    If Dir(strPathAndFile) <> "" Then
        varMsgResponse = MsgBox(strPathAndFile & " already exists." & vbCrLf & _
                "Yes to delete and create new file." & vbCrLf & _
                "No to keep and append data to it." & vbCrLf & _
                "Cancel to exit processing.", vbYesNoCancel)
               
        Select Case varMsgResponse
            Case vbYes
                Kill (strPathAndFile)
            Case vbNo
                'Do nothing
            Case vbCancel
                MsgBox "User cancelled. Output not sent to file."
                Exit Sub
        End Select
    End If
       
    Open strPathAndFile For Append As #intFileNum
   
    Print #intFileNum, strToOutput
    Close #intFileNum
   
    varOutput = Shell("Notepad.exe " & strPathAndFile, vbNormalFocus)

End Sub
