pc-learner say:

Unless the wireless adapter in the computer or laptop is dual band, it will not see the 5 GHz signal at all.  Many older and not that old devices do not have dual band adapters, though Apple has been pretty good about this. You need to check the specs of each device to make sure it either supports the ABGN or AC wireless standards.  Even if the adapter supports them, the 5 GHz signal drops off much faster than the 2.4 GHz, so the adapter is likely to see the 2.4 GHz signal is stronger and choose to use that band.  Also, in Windows Device Manager, Device Properties, Advanced Tab, many dual band adapters offer settings which can control which bands the adapter will use, and whether to prefer to use 5 GHz if it is available.

Many of us have been using dual band wireless adapters with dual band wireless routers successfully at 5 GHz throughout the preview.

I am presuming you have given the 2.4 GHz and 5 GHz bands different SSIDs on the wireless router. It's OK to give them the same name, but that makes it difficult to tell whether both bands are visible and which band a given device is using.

talking moose say:
Assigning a default signature to an account isn't about specifying which signature you'll use when sending to someone. It's about specifying which signature you'll use when sending from one of your accounts.

Let's clarify:
  1. You've created three signatures "Exchange sig", "POP sig 1" and "Pop sig 2".
  2. You've set "Exchange sig" as the default signature for your Exchange account.
  3. You've set "POP sig 1" as the default signature for "POP account 1".
  4. You've set "POP sig 2" as the default signature for "POP account 2".
If you're currently viewing anything within your Exchange account then Outlook's behavior is to create new messages (and replies and forwards) from that account. If you create a new message now you should see your "Exchange sig" and the From menu at the top of the window will display your Exchange account.

If you're currently viewing anything under On My Computer then Outlook's behavior is to create new messages (and replies and forwards) from the email account you've set as default under Outlook menu -->  Preferences... --> Accounts. If your default account is the Exchange account then it will be the default sending account. If your default is "POP account1" then that will be the default sending account. "POP sig 1" will appear in your new messages and "POP account 1" will appear in the From menu at the top of the new message window.

Changing the From menu to another account in a new mail message will not change the signature. You must do that manually using the Signatures menu in the ribbon.

Hope this helps!

don varnau say:

Hi,

 

1. On the affected sites, if the Compatibility View icon (torn page next to the Address Bar) is visible, click it. Or, from Tools> Compatibility View settings, add the website to the list.

 

2. Internet Options> Security, try adding the site to Trusted Sites.

 

3. From Internet Options> General> Browsing History> Delete, uncheck "Preserve Favorites website data" and clear the Temporary Internet Files and Cookies.

 

4. From Start> All programs. Accessories, right-click Command Prompt and choose "Run as administrator". At the flashing cursor type in ipconfig /flushdns (note the space between config and / ) hit [Enter] then Restart.
