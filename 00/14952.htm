Crashes often leave file fragments behind that make it easier to crash next time. Please try the following:

  1. Open Applications>Utilities and start Disk Utility.
  2. Select your hard disk in the left pane, then click on the Verify Disk button.
  3. If disk verification reports errors, restart from your installation DVD or other startup disk, re-run Disk Utility, select your hard disk, but choose Repair Disk.
  4. Restart from your regular disk and see if PowerPoint is more stable when printing.

If you still get crashes, a font used in your presentations may be at fault. You can check them using these steps:

  1. Open Applications>Font Book.
  2. Press Command + A to select all fonts.
  3. Choose File>Validate Fonts.
  4. If you haven't run the utility before, you'll probably see a lot of duplicate fonts, which are flagged as minor problems. Remove the duplicates and anything with major problems from your system, then retry PowerPoint.
