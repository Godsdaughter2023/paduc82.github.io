considerate_guy say:

The list of Recent Projects is dynamic.  Movie Maker is remembering the last several projects that you have opened.  Eventually, as the list grows beyond a fixed number of entries, the older ones will drop off the list.  If you have a certain project that you want to stay in the list, you can click the pin icon to keep it in the list until it is unpinned.  Remember this list doesn't have any correlation with what might be happening with the project files on the hard disk.  A project that hasn't been opened in awhile may drop off the list, but still be on the computer.  Similarly, you may delete a file from the computer and it will still show in the list.

The list is stored in the registry and unless you are comfortable with editing the registry, I wouldn't suggest fiddling with the Recent Projects list.  Unfortunately, there isn't a way to manage it within Movie Maker except to pin/unpin items.

As for removing music, I assume you are looking at a project where the music is listed in a separate rectangle.  To remove any item in a project, you just right-click on the rectangle and choose Remove.  If you are talking about the audio/music that is part of an actual video, that is done by muting the audio.  Click on the video then go to the Video Tools | Edit tab.  On the left you should see a control for Video Volume and you can set it to the far left (mute).

How to delete a shortcut with command line ? - Microsoft Community

idinpc say:

How to delete a shortcut with command line ?

How to delete a shortcut with command line ?

How to backup local users and groups whit command line ?

wunders say: You delete a shortcut just as you would delete any other file.  A shortcut will end with the extension ".lnk".  So assuming that you are trying to delete the shortcut named "Shortcut.lnk" that appears on the user "Bob"s desktop, you would use the delete (del)  or "erase" command specifying the entire path to the file.  (Be sure to put filename in quotation marks if there is a space anywhere in the path or name) as in:

    del  "C:\documents and settings\Bob\Desktop\Shortcut.lnk"

What kind of a "local users and groups" backup do you want to make?  If it is a backup of the users and groups as entered in the registry, a good registry backup program such as ERUNT will do this from the command line.  If you're trying to back up all users and their data as well, then you are better off with a backup program.
