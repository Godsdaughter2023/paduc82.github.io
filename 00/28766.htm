Unlike previous version of Office, the new Office uses the Click-to-Run technology to install. Click-to-Run is a streaming and virtualization technology and Streaming allows users to use a Click-to-Run product before the complete product is downloaded.

 

By default Office installs on the respective drive where the Operating System is installed and cannot be customized.

If you already have a sparkline, you can copy the cell that contains the sparkline as a picture and paste it to a different location.

blacklab22 say: People page won't open... People page just "hangs" forever... can't access my Contacts list... address box does not populate when typing a new e-mail... etc.  This all seems to be the same very common current problem (bug?) with the failure of MS OUTLOOK's contact database to load correctly on the PC site.  Most browsers seem to be pretty equally affected (perhaps Chrome less so - if you have it - try it).

The forum moderators variously suggest clearing your browser's Cache and Cookies, Updating to the latest version and/or Resetting your browser, etc.  All reasonable ideas and if you look through recent forum topics you will find these suggestions laid out in full.  Sadly not that many successful outcomes - but then you might just be lucky?

These are 4 Workarounds being suggested, one or more of them may work for you:

1.  You may be able to at least access your Contacts list by using OUTLOOK's Mobile site login page (https://profile.live.com/home/contacts?bfv=um).  Best to try this when you are fully logged out of OUTLOOK.  Sadly this is of limited use as the Mobile page "footer" link back to the PC site People page will just hung forever as well.  However it will prove your Contacts list still exists and in extremis you could copy/paste or write out e-mail addresses!

2.  Try disabling the Adobe Flash plugin in whatever browser you are using and then just use the reload/refresh button and hopefully the "hung forever" PEOPLE page will then complete loading normally.  It's a pain to have to keep disabling and re-enabling Flash but this Workaround may allow you to use your Contacts list almost normally again.

If not sure how to disable/re-enable the Flash plugin a search engine will quickly find the instructions.

(More information in forum topics: People page hangs on PC site [WORKAROUND FOUND] & Address Book inaccessible in OUTLOOK.COM Hotmail topics.  Thanks to CWH803 for finding this Workaround.)

3.  Try a different browser - Chrome seems to be the least affected, others have had success trying Firefox, and some have succeeded by resetting whatever browser they were currently using.

4.  If you have access to other machines try them... especially if you have an older machine still running XP/SP3 - several users report that their old XP PC is unaffected by this problem.

Hope the above might help - you are not alone - there's a shed load of similar topics, information and other frustrated OUTLOOK users on the forum.  All in all it's a right pigs ear and hardly MS' finest hour?

Create summary page from other sheets totaling by categories from a - Microsoft Community

ebrett say:

Create summary page from other sheets totaling by categories from a drop down list

I need to create a summary page that totals amounts from other sheets. The other sheets have drop down boxes (for categories) in column A and then an amount in column C. I need the summary sheet to take each of the categories (ie Taxes) and total all the amounts where column A = taxes. Is there a way to do this? Also, the number of sheets that the summary sheet will be totaling is unknown, so I think I would have to do something where the summary sheet totals all amounts (in column C) that have a particular text in column A in between two hidden blank sheets.
max, singapore say: Think a simple formulas solution which might suffice could be something along these lines, using SUMIF and within it using INDIRECT (to read sheetnames and ranges) so that we can easily copy the SUMIF across/fill down

In your summary sheet
List the source sheetnames involved in B1 across, eg:   Sheet1, Sheet2, etc
List the various categories to be summed in A2 down, eg:   Taxes, etc 
Put this in B2:
=IF(OR($A2="",B$1=""),"",SUMIF(INDIRECT("'"&B$1&"'!A:A"),$A2,INDIRECT("'"&B$1&"'!C:C")))
Copy across/fill down to populate in one swoop
And if needed, just use a simple SUM in a col to the right to get the row totals for each category
