  • Type “chkdsk” and press Enter.

  • If the hard disk is SCSI, check for problems between the SCSI controller and the disk.

Disclaimer: When you run the check disk, there are chances of losing data so you can take a backup before running check disk.

  • Press Windows key + X, click Command Prompt (Admin).
  • In the Command Prompt, type the following command, and then press ENTER:
