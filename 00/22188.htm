stefan blom say:

It sounds as if your keyboard has "hijacked" some shortcuts. There is no setting in Word that can start Windows search. See if holding the Fn (function) key makes these shortcuts "behave."

charles kenyon say:
I just switched to Office 2013 and Windows 8.1 from Office 2010 and Win 7 Pro. When working on a merge document in Word, pressing Alt-F9 brings up the Windows search panel instead of toggling the field code. Ctrl-F9 also brings up the Window search instead of adding a blank field code. Is there a setting that is causing this?

Laptop?

The default BIOS setting on my laptop gives special meanings to the function keys which can be disregarded by pressing the fn key. Every time the bios gets updated, it resets to this setting. A real pain. I suspect that is what is going on for you.

Take a look at the function keys. Chances are they have icons indicating some special functions for your laptop.

Likely, this is not an Office problem but rather one that is global for your computer. What tipped me off was that on my computer the F5 key, that I normally use to refresh the browser window, is assigned to the backlighting on the keyboard. Like you, I had problems with the F9 key in Word.

If so, you need to check with your computer manufacturer on what to do to make global change to give you access to the F-keys without using the fn key. In the meantime, using the fn key should give you access to the software-driven functions of these keys.
