a. user say:

Hello Geo,

We realize the inconvenience you have experienced.

Is this issue specific to this software?

From the description, I suspect that the problem might be with the application compatibility on Windows 10. Suggest you to check by uninstalling and reinstalling the software to see if it works fine. To uninstall the software, right-click on Start button and select Programs and Features. Locate the software and then click Uninstall/Change. Follow the on-screen instructions to remove the software from your computer. Reboot your computer and then try to reinstall the software to see if it works fine.

If problem still persist, try installing the software in compatibility mode. Compatibility mode runs the program using settings from a previous version of Windows. Follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the software setup file (.exe file) from the developer's website.

b. Uninstall the existing one from your PC through Control Panel.

c. Right-click on the setup file and click on ‘Properties’.

d. Click on the ‘Compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 8/8.1 or 7 operating system from the drop down menu and proceed with the installation. Once this is done, restart the computer and check if the issue persists.

If this issue is specific to this software or the problem still appears, then try to get in touch with Sony Support team for more assistance.

Let us know how it went.
