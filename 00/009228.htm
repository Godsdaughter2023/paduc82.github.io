suzanne s. barnhill say:
Numbered lists behave differently as a result of user demand. Unfortunately, the setting applies to numbers in running text as well as numbers in lists, so you can write, "I sent the letter on March 5. he replied on March 6," and Word will not correct the capitalization.

jay freedman say:
BUT even if the initial letter is not auto-corrected, it is still signaled by a blue underline as "wrong". I would like to remove also that warning which is really annoying. Any way to do it?

Go to File > Options > Proofing. From there you have two choices.

  • You can suppress only the warning underline for lower case where Word thinks it should be upper case. Click the Settings button next to the Writing Style item; in the next dialog, clear the check box for "Capitalization".
  • You can suppress all blue underlines. In the Options > Proofing page, clear the check box for "Mark grammar errors as you type". Infractions of the grammar rules will still appear in the manual spelling check (F7) if you leave the check mark on "Check grammar with spelling".
