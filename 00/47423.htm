a. user say:

Read the bit about Password Protected Settings. You turn if off if you do not want to be asked for username, etc.:

http://technet.microsoft.com/en-us/library/bb727037.aspx

 

Have a read of the above link re Vista File and Printer Sharing.

 

Permissions/Share info is there as well.

 

If using Norton, McAfee, Trend Micro, Kaspersky I.S., make sure file and printer sharing is enabled in THEIR firewall (or LAN allowed, depending on how their Exceptions are worded in their Firewall)

 

1 thing to do is make sure that the Workgroup Name of ALL the computers is the SAME.

 

In Vista Network and Sharing:

 

Network Discovery: ON (So it can see the other computers)

 

Network set to Private (Public is for hotspots, airports, etc)

 

File Sharing: ON

 

Public Folder Sharing: ON (Vista’s Public Folder is the same as XP’s Shared Docs)

 

Password Protected: OFF (unless you want to set up identical usernames and passwords on ALL computers in your Network) If you have it ON, and no identical usernames/passwords set up, you will be asked for a username and password when you try to access a Vista computer from an XP computer, or a Vista computer.

xb1r apollo say:

Wireless Security

Login Information



  • Gateway: 192.168.0.1
  • Username: admin
  • Password: admin





How to Find Wireless Passphrase (WPA/WEP)



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Wireless Settings" from the menu on the left.
  5. Select "Wireless Security Settings" from the sub menu.
  6. If the security mode is WEP, the passphrase will be dicated by the "Default Key".(By default, the key will be "WEP Key 1".)
  7. If the security mode is WPA or WPA2, the passphrase will be labeled as "Pass Phrase".



Changing Wireless Security Type/ Disabling Wireless Security



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Wireless Settings" from the menu on the left.
  5. Select "Wireless Security Settings" from the sub menu.
  6. Select the desired wireless security type from the "Security Mode" drop down menu. (Select "Disable" from the drop down menu.)
  7. Select Save.
