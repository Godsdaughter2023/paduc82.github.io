    For i = 1 To .Paragraphs.count

armen stein say:

Gina is right, if you're thinking of multiple simultaneous users.  OneDrive uses a sync model - you have a local copy and a cloud copy.  When the local copy is changed, it updates the cloud version.  So it won't work for simultaneous usage.

However, and please understand I'm not recommending this, it would be possible for one user at a time to make changes to the BE file, which would be updated in OneDrive, then another user could make further changes.  If they tried to do things at the same time, bad things would happen, like duplicate copies of the BE data with each person's changes in different files.

This is similar to Dropbox, which some people *carefully* use to share an Access BE.
