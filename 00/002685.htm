kkkwj say:

Setting emphasis style properties in semaphore sidebar

My problem is when I  assigned the normal emphasis style to some black text inside the sidebar, the emphasized text turns white,  and loses all contrast with the gradient light blue. Outside of the sidebar on my normal page, the emphasis style simply applies italics, and does not change the colour.  The font colour is set to automatic.

So I think the sidebar Textbox must somehow be overriding the font colour. is there some way I can stop it from doing that, given that the sidebar  has such a light blue gradient on it that  white font colour does not work? I am perfectly happy if the emphasized style keeps the font black. Thank you

jay freedman say:

The white text seems to be an incorrect interaction of the Automatic font color with the background color of the sidebar. Automatic is supposed to be black against light backgrounds and white against dark backgrounds, but the algorithm it uses to determine light vs. dark is, um, unreliable...

If you put the cursor in the unmodified text of the sidebar, and click the Font Color button on the Home ribbon, you can see that the font color is not black but one of the Theme colors, "Dark Blue, Text 2, Darker 50%". The solution is to apply that color along with the italic format.

Although you could modify the Emphasis style to do that, it would also affect any use of that style in regular text. It would be preferable to define a new custom style just for the sidebar. To do that:

  • In the sidebar, directly apply italic format to some text (Ctrl+I). Leave that text selected.
  • Click the bottom down-arrow on the Styles gallery, and click Create a Style in the menu.
  • In the small dialog, click the Modify button. (DO NOT click OK or press Enter, because that will create a paragraph style instead of the character style that you want.)
  • In the Create New Style from Formatting dialog, change the name to something like "Semaphore Emphasis" and change the style type to Character. Because of the selection's formatting, the color and italics will already be included.
  • To be able to use this style in other documents based on the Normal.dotm template, click the "New documents based on this template" option.
  • Click OK.
  • Apply the new style to everything in the sidebar that you want to format as italic.
  • If you're prompted to save the Normal template when you exit Word, click Save.
