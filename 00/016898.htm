stefan blom say:

You can hide individual styles in the Style Gallery on the Home tab by right-clicking them and choosing Remove from Style Gallery.

The Styles pane is tricky, though, and I don't think you can get consistent behavior there. In a Word document, a style A is considered "In use" after it has been applied to text, even if you later apply some other style B, so that no "occurrence" of A is present in the text.

The settings on the Recommend tab of the Manage Styles dialog box only apply when the view for styles is set to "Recommend." Those settings won't necessarily work with the "In use" option in the Style Pane Options dialog box.
