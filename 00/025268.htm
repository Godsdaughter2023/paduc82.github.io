Count the number of cells with alphebets in a row - Microsoft Community

nebinnepolian say:

Count the number of cells with alphebets in a row

Dear Sir,

Kindly help me out is their a formula to count the number of a specific alphabet in a row to a particular cell or the number of Zeros in a particular row to a cell.

vijay a. verma say:
let's assume you want to count alphabet "a" or "A" (in below formula, you need to specify lower case letter only)

If your string is in a particular cell as say

=LEN(A2)-LEN(SUBSTITUTE(LOWER(A2),"a",""))

For a range of cells

=SUMPRODUCT(LEN(A2:F2)-LEN(SUBSTITUTE(LOWER(A2:F2),"a","")))

For 0s, simply replace "a" with "0"

davem121 say:
Hi ebinnepolian, you can use the formulas below:

To count the number of 0's in a row (you may need to change A1:Z1 to suit the row you are using)

=COUNTIF(A1:H100,0)

To count the number of a certain text in a row, (you may need to change A1:Z1 to suit the row you are using)

=COUNTIF(A1:Z1,"aText")

Or if you want to count how many times text from a cell appears in the row, (you may need to change A1:Z1 to suit the row you are using)

=COUNTIF(A1:Z1,B1)
