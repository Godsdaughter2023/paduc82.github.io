If it came preinstalled on your computer.

New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

 

This is how the recovery partition is accessed for the most popular makes

For Dell, press CTRL + F11 directly after powering up the unit

For HP, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

For Acer, press and hold ALT + F10 as soon as you see the logo

For Samsung: Tap the F4 Key at the Samsung logo when you first power on.

For Asus, press F9 as soon as you see the Asus logo.

For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

Post subject: HOW TO do a clean install of WinXP if your PC is infected
Posted: Sun 10/17/10 11:59 am


If (a) you do NOT have an anti-virus application installed and/or (b) your subscription expired, however briefly and/or (c) you didn't have a valid, fully-functional anti-virus application installed when your computer got infected and/or (d) you've neglected to keep the machine fully patched at Windows Update, see...

    http://technet.microsoft.com/en-us/library/cc700813.aspx

Back-up any personal data (e.g., online banking & credit-card passwords, none of which should be considered 100% trustworthy at this point) then format the hard-drive (typically the C: drive) & do a clean install of Windows. Please note that a Repair Install (AKA in-place upgrade) will NOT fix this!

    • How to back up or transfer your data on a Windows-based computer
    http://support.microsoft.com/kb/971759


NOTE: If your computer didn't come with a set of disks, there will be a hidden Recovery partition (not to be confused with System Restore) you would use to do the clean install (AKA a "destructive recovery").

After the clean install, you will have the equivalent of a "new computer" so take care of everything in the list below, in order, before otherwise connecting the machine to the internet or a local network (i.e., other computers) and before connecting a flash drive, SDCard, or any other external drive to the computer:

  1. Make sure the Windows Firewall is enabled.
  2. Uninstall any now-invalid Norton, McAfee, etc., free-trialware which was also reinstalled, run the appropriate removal tool (see below), then install an anti-virus application of your choice (e.g., Microsoft Security Essentials - free).
  3. Check in at Windows Update & get the computer fully-patched.
  4. If you do not install Microsoft Security Essentials or your chosen anti-virus application does not include an anti-spyware component, you can install Windows Defender or another anti-spyware application which includes realtime protection.

TIP: Use another, uninfected computer to download the installers for, e.g., SP2 and/or SP3, the installer for your new anti-virus application and either of the removal tools cited below. Transfer these files to CD or DVD (not a flash drive) then transfer the files from the CD or DVD to the desktop of the computer you're working on.
