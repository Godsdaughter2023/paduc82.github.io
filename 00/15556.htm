GKB,

Yes, that is correct, only one user can have write access to the file. But then that is actually true of a linked structure also although users are not "locked out".

There are different methods for viewing linked tasks in Project. One is to use the Relationship Diagram. Another is to use the Task Inspector found under Task > Task group.
