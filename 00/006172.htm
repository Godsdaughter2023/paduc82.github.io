Here's a simple, non-Excel way to get rid of all the line breaks in your text:

Paste the data into TextEdit, put your cursor at the end of any line, then hold shift and press the right arrow once. This will select the invisible new line character. Now hit command + c to copy this invisible character, and then got to

Now you shouldn't have any more line breaks in your text. :)

365 SB - Outlook - Microsoft Community

palcouk say:

365 SB - Outlook

I am trialling 365 SB edition and installed it on my none production PC.

I'm unsure of the implications, if any, in resetting the DNS settings on the Domain I own in order to use Outlook, presumably an Exchange connection, and how this might impact on the Outlook 2010 mail settings which I run on my production PC (pop mail) 

Thanks 

roady [mvp] say:

Hi Palcouk,


It depends a bit on how you want to approach it. For running a trial, while you currently have a POP3 service for your domain, the following approach is probably the safest way to go and wouldn't disrupt your current POP3 service:

  1. Set all the required DNS entries at your current provider except for the MX records.
  2. The SPF record can often be safely omitted during testing, but if you want to set them, make sure you allow both the outlook.com servers as the servers from your ISP.
  3. Associate your own address (or your testing address) with the Exchange Online account and set it as the default for the mailbox
  4. In the admin portal of your current provider, set a forwarder on your mailbox to the @accountname.onmicrosoft.com address. Make sure that it will also still deliver to your mailbox on that ISP.

You can basically run like this indefinitely. Even when you want to stop using your POP3 account and want to take the Exchange Online account into production. You can opt to stop delivering it to your POP3 mailbox and only forward it to the onmicrosoft.com address or to keep delivering it to both and clean out the POP3 account every now and then.


This configuration also allows you to only host a couple of mailboxes for your domain on Office 365 while others remain POP3. In this setup, you must however set your domain as an Internal relay domain on Office 365. You can do this in the Exchange admin center-> Mail flow-> Accepted domains

Copy/Paste:

Windows Defender in Windows 10 and Windows 8.1, and Microsoft Security Essentials in Windows 7 and Windows Vista help protect your PC from malware and other threats in exactly the same way. You can't use Microsoft Security Essentials with Windows 10 or Windows 8.1.

Windows Defender in Windows 10 and Windows 8.1 is built into Windows and ready to work as soon as you turn your PC on.

Microsoft Security Essentials works on Windows 7 and Windows Vista. You can download it from the Microsoft Security Essentials website.

Before the release of Windows 8, Windows Defender was a separate tool for antispyware. This tool is no longer supported, as it is now fully integrated into Microsoft Security Essentials (in Windows 7 and Windows Vista) and Windows Defender (in Windows 10 and Windows 8.1).

In other words, you don't need to have both Microsoft Security Essentials and Windows Defender if you're using Windows 7 and Windows Vista.
