stefan blom say:
Unfortunately, Word only supports a document grid.

markbarsamian say:

Thanks, Stefan. Definitely not the answer that I wanted to hear. Word 2007 has a floating grid for the drawing canvas. Bummer that they dropped that in Word 2010. On the other hand, I suppose that there is a work-around that is similar to what one often has to do in Word 2007 anyway. Namely, it is helpful to create a drawing in a way that its objects are aligned to gridlines and such that it contains simple geometric objects that make the corners of the drawing easy to determine. Then, when editing an old drawing, one can enlarge the drawing canvas, select all of the objects of the drawing, and move them around until the corners of the drawing align with gridlines. At this point, all of the objects of the drawing will align with gridlines, and it is straightforward to edit the drawing.

Thanks again,

-Mark 

stefan blom say:

How did you create a grid for a drawing canvas in Word 2007? As far as I know, only the document grid existed in that version too.

If you set the Text Wrap for the drawing canvas to "Square" (or anything besides "In line with text") you can align the canvas itself, which may be helpful.

markbarsamian say:

In Word 2007, suppose that you have a page of mostly text, but with a drawing canvas floating in the middle of the page.

 

If the grid is turned on and the cursor is located in a region that NOT in a drawing canvas, you see a grid that covers the entire page and whose gridlines extend through any drawing canvas on the page.

But if you click on the drawing canvas, you see a grid only inside the drawing canvas, and the origin of the grid is in the upper left corner of the drawing canvas. (At least I think that's the origin. What is certain is that horizontal & vertical gridlines cross at exactly the upper left corner.) If the drawing canvas is moved to a different spot on the page, this grid moves with the drawing canvas (while the grid for the page stays fixed with the page). This way, if any objects in the drawing canvas were aligned with the drawing canvas gridlines, they will still be aligned with the drawing canvas gridlines after you move the canvas.)

 

Does that explanation make sense?  Sorry if it doesn't.

 

-Mark
