How to Find Wireless Password (WPA/WEP)

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. In the address bar at the top, type in 192.168.2.1
  4. Click Login in the upper right.
  5. Input the password and click Login (by default, the password should be left blank).
  6. Select Security from the menu on the left under the Wireless section.
  7. Uncheck the box labeled "Obscure PSK" to make the Wireless Password visible.

Changing Wireless Security Type/Disabling

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. In the address bar at the top, type in 192.168.2.1
  4. Click Login in the upper right.
  5. Input the password and click Login (by default, the password should be left blank).
  6. Select Security from the menu on the left under the Wireless section.
  7. To change or disable the Wireless Security, select the desired Security Mode from the dropdown (Disabled, WPA/WPA2-Personal(PSK), 128-bit WEP or 64-bit WEP).
  8. If WPA/WPA2-Personal(PSK) is selected, choose the Authentication (WPA-PSK, WPA2-PSK or WPA-PSK + WPA2-PSK) and Encryption Technique (TKIP or AES).  WPA2-PSK with AES is recommended.
  9. Enter the Pre-Shared Key or WEP Key.
  10. If WEP security is selected the password will need to be 10 charecters in length (if 64-bit) or 26 characters in length (if 128-bit) and can only use 0-9 and A-F.
  11. If WPA or WPA2 was selected the password can be between 8-63 characters, is case sensative and can use any character on your keyboard.
  12. Once the passphrase has been entered select Apply Changes at the bottom of the page.

even with good cooling there isn't really much can be done.

Replacing thermal paste (and pads on gpu VRAM) is a popular suggestion, but that generally only helps if the overheating is recent,

and is in fact due to the thermal paste loosing conductivity.

- With desktop PC's the desired temps are generally about 10C lower than laptops, which due to poor cooling tend to run within

the upper limits of what is considered non damaging temps. The cooler the better is always best for stability and long life.

- The cpu should not be anywhere near

around the 60-65C (my last AMD was 2010), and about 75C is desirable max load temp for Intel.

My Intel i7 4790, which is known for running a bit hotter than most Intel cpu's at idle peaks a bit over 70C on hot days under full load.

The max Idle temp is about 50C on a hot day. My earlier Intel i5 3570 runs about 5C cooler than the i7 4790.

Note- These temps are with an aftermarket cpu cooler.

Aftermarket cpu coolers are an important feature on a gaming PC due in part to them keeping the cpu cooler than the stock cooler

during the extended stress of long gaming sessions, but also because that extra efficiency means the fans can run a lower quieter speeds,

so the fan speed does not cycle up and down from fairly quiet to jet engine volumes as the stock coolers tend to do.

for about 70-75C as max when ambient air temps are  over 30C.
