3. Copy and paste the following script to a notepad and save it as a .ps1 file in a local place.

#Variables for processing

$AdminURL = "https://[tenant domain]-admin.sharepoint.com/"

$AdminName = "your account"

Connect-SPOService -url $AdminURL -credential $Credential

 

#Add Site collection Admin

Set-SPOUser -site $SiteCollURL -LoginName $SiteCollectionAdmin -IsSiteCollectionAdmin $True

For it to be totally automatic some VBA code would be needed

In A2 (or the cell holding No. 1) enter =ROW(A1) to get a 1

Drag this down the column

Each time you insert or delete a row you will need to repeat the dragging

ashish mathur say:

Hi,

Try this

1. In cell A1, type 1

2. In cell A2, type =COUNTA($B$2:$B3) and copy till cell A5

3. Select A1:C5 and press Ctrl+T

4. Right click on cell A4 and go to Insert > Table Rows Above

5. When you now type something in cell B4, the numbers in column A will update

6. When you add data in the first available blank row, the numbers will update
