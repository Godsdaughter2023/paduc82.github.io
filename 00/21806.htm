How do I merge two texts into a two column document in Word 2010? - Microsoft Community

ulrichie say:

How do I merge two texts into a two column document in Word 2010?

I have two texts in two different languages, and I want to be able to put those two texts next to each other in two columns to be able to point out the differeces in translationstyles by refering to a specific numbered line. I know two columns is possible, but is it possible to get two different texts in the two different columns?
pamcaswell say: In addition to what Graham suggests, if you want paragraphs to align, each paragraph of each document should be in a separate row.  That's more work but you may be able to reduce that work if you have the same number of paragraphs in each document.

  • Select the contents of document 1 for the first column and convert it to a table with the Separate text at set to Paragraphs.  
  • Do the same for document 2.
  • Copy and paste the table from document 1 into a new document.  Click the ¶ on the Home tab to show formatting marks.  Adjust the table width to about half the page width.
  • Copy  the table for document 2. Place the cursor in the new document's table at the end of the first row between the grid line (or border) and the end of cell marker and paste. 

You should get a two-column table with aligned paragraphs.  This has worked for me with simple, all text documents, although I was not working with other languages.

Make sure that you have no files named something ~$blablabla.bla in yours 'sync pending' directory. Files named like that is:

  1. Temporary Microsoft Office files
  2. Hidden - you can see them only if you checked 'hidden items' checkbox in Explorer

joe's answer say:

 

 

Try Different SMTP Ports

Typically, SSL and port 465 work most of the time. If you can receive email but can't send any messages in Outlook, double-check that you're using the correct port number.

Sometimes ISPs or network providers block specific ports for security reasons. If the default settings don't work, you can try some alternate ports.

To Change Your Outlook SMTP Port Settings

  1. In Microsoft Outlook 2010, click File, and then click Account Settings.
  2. Select the account you want to change and click Change.
  3. Click More Settings... and select the Advanced tab.
  4. Next to Use the following type of encrypted connection, select SSL.
  5. Next to Outgoing Server (SMTP), type 465. Click OK and click Next.
  6. If those settings don't work,repeat steps 1-3 and select None for Use the following type of encrypted connection. Try these other ports for Outgoing server (SMTP): 80, 3535, or 25.
  7. Click OK and Next to see if the other ports will work
