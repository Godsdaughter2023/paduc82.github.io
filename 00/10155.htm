I have a very large list of 11-digit NDC codes that I need to convert to 10 digits with hyphens.  The problem is that for any code that starts with "0", the code has to go from 09999999999 to 9999-9999-99.  Code 99999099999 has to become 99999-999-99.  And Code 99999999909 has to become 99999-9999-9.  Can anyone help with a formula that can do that for any of these situations?

kimjager say:

HOLY MOLY!!  I figured it out!!!  You got me started in the right direction, JP Ronse!  Here's the formula I came up with that is working:

=IF(LEFT(C30,1)="0",TEXT(MID(C30,2,10),"0###-####-##"),(IF(MID(C30,6,1)="0",TEXT(MID(C30,1,5)&MID(C30,7,5),"#####-###-##"), (IF(MID(C30,10,1)="0",TEXT(MID(C30, 1,9) & MID(C30,11,1), "#####-####-#"),0)))))
