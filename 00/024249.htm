Moving Pages in OneNote (Desktop, Web App & Mobile)

If you are wanting to move/copy a page or tab to another notebook, here is what is available:

Desktop:
  1. In the Notebooks pane, right-click a section.

  2. On the shortcut menu that appears, click Move Or Copy.

  3. In the Move Or Copy Section dialog box, select a notebook or a section in the notebook.

  4. At the bottom of the dialog box, click either Move or Copy.

Hi,

Before jumping into a full Index rebuild though, to solve the Outlook 2010 search problems we should also make sure your Outlook data files aren’t corrupted. Data file corruption can occur over time, and there's a possibility that the Outlook 2010 Search function stops working because of this corruption.

You can use the Inbox Repair Tool to fix any corrupted files, and it’s one of the easier ways to fix the Outlook search function. Search for Scanpst.exe with your desktop search tool, and then run it. If after this process you find that your Outlook 2010 search function is still not working, then try to rebuild again the Index in Outlook 2010.

6) Restart Excel.
