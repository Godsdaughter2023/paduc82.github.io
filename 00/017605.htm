jeeped say:
In your Month Total worksheet, try this in B2,

=SUM('Week 1'!B2,'Week 2'!B2,'Week 3'!B2,'Week 4'!B2,'Week 5'!B2)

You cannot use Excel's ability to sum through the worksheets if the worksheets are non-adjacent.

ashish mathur say:

Hi,

 

I have answered a similar question here - http://www.ashishmathur.com/summarise-data-from-different-cells-of-multiple-worksheets/

 

The answer to your specific question is this formula in cell B2 od the "Month Total" worksheet

 

=SUMPRODUCT(SUM(INDIRECT("'"&$F$1:$F$5&"'!B2")))

 

F1:F5 has Week 1, Week 2, Week 3, Week 4 and Week 5

 

Just ensure that the spellings in range F1:F5 match with the spellings of the tab names.

cfleenor1125 say:

ToC wrong when page x of y excludes title page

Hello,

In page numbering x of y, I know how to advance page numbering to exclude the title page and ToC page in the total page count. So, in a 30 page document the first page is numbered 1 of 28 (rather than 3 of 30).

However, the table of contents is showing it as page 3. Why isn't it taking the page number from the footer? How can I fix this?

Connie

john mcghie say:

Hi Connie:

Page x of y contains two fields: the { PAGE} field and the { NUMPAGES } field.

When setting up front matter, you need to end the front matter with a section break.  That enables you to use Format>Page Numbers to set the page numbering to Start at 1 at the beginning of that section.

You then need to change the second field in Page X of Y to { SECTIONPAGES } rather than {NUMPAGES}.  The former tell you how many pages in the section, the later tells how many pages in the whole document.

Generally, I use more than one section break in the body of a book, so SECTIONPAGES won't work either. What I do then is to add a bookmark to the end of the document, and replace the second field in Page X of Y with a cross-reference to the page number of the bookmark.

That way, you get an automatically-maintained Page X of X.  You need only remember to update fields before you print.  If you set the option in the Print preferences, Word will do that for you.

Hope this helps

per jessen say:

Try this:

Private Sub ListBox2_Click()
    Select Case ListBox2.Value
    Case "18-34"
        For Each cell In Output.Range("J7:J10").Cells
            ListBox3.AddItem cell.value
        Next
    End Select
End Sub

or better use the RowSource Property, assuming that the sheet name is 'Output':

Private Sub ListBox2_Click()
    Select Case ListBox2.Value
    Case "18-34"
       Cells
            ListBox3.RowSource="Output!J7:J10"
    End Select
End Sub
