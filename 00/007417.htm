john w. vinson say:

Not one symbol, but one operator: you need to use OR logic (to find records where Courses = C1 OR where Courses = C4, rather than the impossible situation where the Courses field = C1 and also at the same time = C4):

If Me.cboCourseCode.ItemsSelected.Count > 0 Then

        If strWhere <> "" Then strWhere = strWhere & "("

        For Each i In Me.cboCourseCode.ItemsSelected

            If Criteria <> "" Then

               Criteria = Criteria & " OR "

            End If

            Criteria = Criteria & "Courses = "'" & Me.cboCourseCode.ItemData(i) & "'"

        Next i

        strWhere = strWhere & Criteria & IIf(InStr(strWhere, " OR ") > 0, ")", "") & " AND "

    End If

Put a break point in the code after the END IF and check the value of strWhere to see if it makes sense - the above is a bit of a guess and may not be correct in the context of your application.

adam.hirshfield say: To the OP and tech support - I have found the cause and resolution for this widely reported issue:

Error code 771 on MS Office 365 install occurs when the MS Licensing server is unable to validate whether your device is entitled to a free 365 subscription.

If the MS server checks your system time against it's own time and they do not match within a certain range, it is unable to validate your device eligibility and this error is presented.

The cause is likely that your system clock is simply incorrect! Please ensure your local time is set correctly and retry the installation.

Hope this helps and you enjoy your Linx and MS Office sub.

All the best,
Adam

Reporting PowerPoint Quiz Results - Microsoft Community

rick_kc say:

Reporting PowerPoint Quiz Results

I am using PP 2010 as its what we have available at work and building quizzes in presentations.  I'm using the code below and it is working fine, but I also need to capture which questions are answered wrong so we can go back over the material with the student.  I would want this information to be added to the Excel spreadsheet. 

Thanks in advance for any suggestions, direction or (especially) code to accomplish this. 

Code in use came from David Marcovitz's site at www.loyola.edu/

=============================================================

Dim numCorrect As Integer
Dim numIncorrect As Integer
Dim userName As String
Dim qAnswered As Boolean

Sub SaveToExcel() 'ADDED
    Dim oXLApp As Object
    Dim oWb As Object
    Dim row As Long
    
    Set oXLApp = CreateObject("Excel.Application")
    'On a Mac change \ to : in the following line
    Set oWb = oXLApp.Workbooks.Open(ActivePresentation.Path & "\" & "Results.xlsx")
    If oWb.Worksheets(1).Range("A1") = "" Then
        oWb.Worksheets(1).Range("A1") = "Name"
        oWb.Worksheets(1).Range("B1") = "Number Correct"
        oWb.Worksheets(1).Range("C1") = "Number Incorrect"
        oWb.Worksheets(1).Range("D1") = "Percentage"
    End If
    row = 2
    While oWb.Worksheets(1).Range("A" & row) <> ""
        row = row + 1
    Wend
    oWb.Worksheets(1).Range("A" & row) = userName
    oWb.Worksheets(1).Range("B" & row) = numCorrect
    oWb.Worksheets(1).Range("C" & row) = numIncorrect
    oWb.Worksheets(1).Range("D" & row) = 100 * (numCorrect / (numCorrect + numIncorrect))
    
    oWb.Save
    oWb.Close
End Sub

Sub GetStarted()
    Initialize
    YourName
    ActivePresentation.SlideShowWindow.View.Next
End Sub

Sub Initialize()
    numCorrect = 0
    numIncorrect = 0
    qAnswered = False
End Sub

Sub YourName()
    userName = InputBox("Type your name")
End Sub

Sub RightAnswer()
    If qAnswered = False Then
        numCorrect = numCorrect + 1
    End If
    qAnswered = False
    MsgBox "You are doing well, " & userName
    ActivePresentation.SlideShowWindow.View.Next
End Sub

Sub WrongAnswer()
    If qAnswered = False Then
        numIncorrect = numIncorrect + 1
    End If
    qAnswered = True
    MsgBox "Try to do better next time, " & userName
End Sub

Sub Question3()
    Dim answer As String
    
    answer = InputBox(Prompt:="What is the capital of Maryland?", _
        Title:="Question 3")
    If answer = "Annapolis" Then
        RightAnswer
    Else
        WrongAnswer
    End If
End Sub

Sub Feedback()
    MsgBox "You got " & numCorrect & " out of " _
        & numCorrect + numIncorrect & ", " & userName
    SaveToExcel 'ADDED
End Sub
