shane devenshire say:

Chee is correct, this can't be done without code.

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim isect As Range
    Application.EnableEvents = False
    Set isect = Application.Intersect(Target, Range("A1:C10"))
    If Not isect Is Nothing Then
        If Target = "" Then Target = "None"
    End If
    Application.EnableEvents = True
End Sub

Change the range A1:C10 to suit the area where you have your cells for the user to enter into

Text editor support only printable characters between ASCII 32 (which is blank) and ASCII 126 (which is ~)... hard return is ASCII 13 (new line is ASCII 10) which notepad does not recognize. Any rich text editor (like Word or this editor in the forum) will handle that easily.
