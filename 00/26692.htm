daniel g. grau say:
For customizing the looks of the text, you need to modify the currently-applied style:

  • Select the text to modify
  • Go to Format > Style
  • Modify the highlighted style name

The style of footnote numbers is defined through the Footnote Reference character style. Note I've defined it as green:

                            


The style of footnote texts is defined through the Footnote Text paragraph style. Note I've defined it as blue:

                            


Bear in mind that one style can be based on another (you'll see this setting when you press the Modify button). If such is the case, changing the base style's attributes will propagate to the styles based on it. For instance, if style Footnote Text is based on Normal, any change to Normal will reflect in that style.

In case of a contradiction (for instance, Normal being red and Footnote Text being blue), the "outermost" style will prevail (in this case, Footnote Text will still be blue).

I can completely understand how you feel at the moment and I appreciate the steps you have tried. This issue is usually caused by BIOS/Firmware update delivered through Windows Update or other OEM Channel. The' Secure Boot' settings may be toggled to 'Disabled', which may be causing this issue.

I would suggest you to follow the steps given below:

  1. Power the device 'Off' and then back 'On'.
  2. You should be able to find a message on the boot screen just before the manufacturer logo appears. You may need to press the ‘F1’, ‘F2’, or ‘Delete’ button, this is usually indicated on the boot screen.
  3. Enter the BIOS settings, look for a tab called 'BOOT' and select that page.
  4. Depending on the BIOS manufacturer, this page could be 'BOOT', ’ADVANCED’, 'STARTUP', etc.
  5. Look for a setting for the 'UEFI Secure Boot'. Each BIOS could have a different name for this. These are the keywords to look for: 'UEFI', 'Secure Boot', and ‘Legacy Boot'.
  6. Toggle the 'Secure Boot' setting to 'Enabled' and attempt to boot the machine.

You can find the BitLocker key by referring the Microsoft article on Find my BitLocker recovery key, search in the Microsoft account and enter the key to unlock the machine.  If you would like me to provide additional advice, simply respond to this thread.
