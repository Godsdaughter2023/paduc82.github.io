PC Mover used to be a good program. It isn't any longer. You should never try to move programs from one system to another -- especially 32 bit to 64 bit.

Instead, you should use the Windows Easy Transfer utility on the Windows 7 setup disk to move your data and settings and install your programs fresh.

Persoanlly, I'd start over and format the new PC, restore to factory, whatever. Then use WET to move data and settings and install your programs, including MSE fresh.

If you insist on attempting to continue on the current path, you might try the information here to rip MSE out of the system so that you can install MSE fresh -- the correct version.

http://answers.microsoft.com/en-us/protect/forum/protect_start/uninstalling-mse/a63b8c4b-58ed-437e-8086-fa08d80725a4
