spiritx say:
Hi,

Just so you know IF you were to upgrade Windows 7 SP1 or Windows 8.1 the
resulting Windows 10 carries the same exact capabilities and rights that the
qualifying OS did. Of course you can't use both the original OS and the Windows
10 upgrade at the same time on the same or different computers.

Windows 10 carries the same licensing rights that Windows 7/8/8.1 did and
is available in two flavors to the consumer.

1. Retail license - can install to any machine or hardware as long as only one
machine at a time per license.

2. Home OEM builder - tied to whatever computer it is first installed on and
can't be installed on another computer even if removed from the original
one. Some hardware changes may require you to call the activation center.
This license is supposed to be for someone building a computer they intend
to sell and not for themselves.

OEM Licensing - FAQ
https://www.microsoft.com/oem/en/licensing/sblicensing/pages/licensing_faq.aspx

¡firedog say:

Hi again Jim.

The Blocked senders feature of WLMail is a bit limited, and judging by the complaints about it here in the forums, it doesn't work very well. You can ostensibly block individual sender addresses, you can block some domains (e.g. spam.com, junk.org) but not others (e.g. hotmail.ca, gmail.com) and you can block country-specific Top-Level Domains (e.g. .cn, .ru) but not other TLDs (e.g. .com, .info).

For a POP3 account, you could use a message rule to, say, not download messages where the sender's address contains ".info". The Message rules dialogue is accessed via a button on the Folders tab.

You could apply a similar rule in webmail for HTTP or IMAP accounts, where the server allows rules like this.

I find that the Safe list only option works best. Then only mail from safe senders (and contacts, with that option selected) gets to the inbox and everything else is consigned to junk, where it's quickly dealt with.

Good luck!
