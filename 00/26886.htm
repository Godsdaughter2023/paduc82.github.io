rick rothstein, mvp - excel say:

You could iterate the tables on the ActiveSheet looking and intersection between those table's ranges and the ActiveCell; something like this maybe...

Dim LO As ListObject, FoundOne As Boolean
For Each LO In ActiveSheet.ListObjects
  If Not Intersect(LO.Range, ActiveCell) Is Nothing Then
    FoundOne = True
    Exit For
  End If
Next
If FoundOne Then
  MsgBox "Cell " & ActiveCell.Address(0, 0) & " is inside the " & LO.Name & " table."
Else
  MsgBox "Cell " & ActiveCell.Address(0, 0) & " is NOT inside any tables."
End If
End Sub
