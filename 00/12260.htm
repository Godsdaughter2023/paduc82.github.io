stevedb1 say:

I use the personal.xlsb to store all of my non-xlam (ribbon) macros.

The xlsm is pretty much the replacement for the old xls formatted files, in that you can store macros within the xlsm to go with each individual file- so that the macro is specific to that file, as it was with the xls files (i.e., you will not be able to use that macro in any other file when it's store in an xlsm file).

As the xlsx does not allow macros to be stored within the individual files, the xlsb is great, because it's still a globally (globally, meaning any and all xl* files can access the macro of interest) accessible file that all of your macros can be accessed from.

It took a while getting use to, but now I swear by the xlsb and the xlam files for storing, and pulling my macros from to do vba operations.

As to removing your xlsb file. Excel automatically creates a new xlsb each time it opens, and does not find it. I use to wipe out my old xlb files in 2000, and began noticing after a while a new one would appear "out of the blue." Thus, if you do not want to make use of the global nature of the xlsb, do not save your macros to it-- I believe you're given a choice when you record the macro. If not, then open the xlsb file, select the recorded code, and move that code over to an xlsm file.

I'm not sure if you can go into the Excel options (big office button at upper left corner of excel, click options, at bottom right window that comes up) and change where to store your macros so they are always stored there by default. My understanding is that the default location for storing a recorded macro is the xlsb file, and it makes up a new module for each time you make a new recording.

Hope that helps.

Private Sub cmdInsert_Click()

    Dim SetNo As String
        
    SetNo = txtSetNo
    
'To stop screen flicker
        Application.ScreenUpdating = False
        
'Select first blank cell in column A to enter set at that point
    Cells(Rows.Count, "A").End(xlUp).Offset(1).Select
    
'Open Special Sets Workbook
    Workbooks.Open Filename:= _
        "S:\Price_Lists\Special_Sets.xlsm"
    Workbooks("Special_Sets.xlsm").Activate
        
    With Workbooks("Special_Sets.xlsm")
    .Sheets("Sets").Activate
     
'Autofilter to select set
With ActiveSheet
    .AutoFilterMode = False
        With .Range("A1:D1")
            .AutoFilter
            .AutoFilter Field:=1, Criteria1:=SetNo
        End With
    Application.DisplayAlerts = False

        .AutoFilter.Range.Copy
    
    End With

'Return to Quote File
    ThisWorkBook.Activate
'Paste selected set
    Selection.PasteSpecial Paste:=xlPasteAllUsingSourceTheme, Operation:=xlNone _
        , SkipBlanks:=False, Transpose:=False

    Workbooks("Special_Sets.xlsm").Close False
'   Turn alert messages back on
    Application.DisplayAlerts = True

    End With

    karen_[01] say:

    Hello

    Please be informed that you cannot link two Skype accounts, it's only possible if you will link the Skype account to a Microsoft account.

    However, if the second account was created when you sign in to Skype using your email address, you can link the email address to your existing Skype account. This way, you can log in to Skype using the email address without creating a new account, instead, it will load the existing Skype account.

    Follow these steps to link your accounts:

    1. Go to https://account.microsoft.com, if you're already signed in, sign out.
    2. Enter your Skype name and password when prompted and then select Sign in. 
    3. If your Skype account is not updated already, you will be notified to update your account.
    4. Select Next to continue.
    5. You will be asked to add an email to your account, and verify that email. 
    6. Congrats, your Skype account is also a Microsoft account.
