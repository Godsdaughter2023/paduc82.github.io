steve rindsberg say:

Sounds like the default formatting for shapes has been set to include text with blue underlining.

Add a new shape (a rectangle will do nicely).  Format it the way you'd like all new shapes to look (fill, outline etc).  Add some text to it and format the text to taste also.  Obviously, remove any underlining if it appears.

Once it's all the way you'd like it, rightclick the shape and choose Set As Default Shape from the popup menu.

Each presentation can have its own set of defaults for shapes, textboxes etc, so if you've got a couple of misbehaving presentations, you'll need to do this for each.  You may be able to save time by copy/pasting the shape you created/formatted for the first presentation into any other presentations you need to fix.  From there it's just right-click, Set As Default Shape for each presentation.
