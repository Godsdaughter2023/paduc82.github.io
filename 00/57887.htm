When Windows has a problem, it  will try to repair itself.  I have no idea why, but often when it displays this message at the conclusion of its attempt to do its repair, it in fact has repaired itself.  In other words, sometimes, this message itself is in error.  Try simply restarting to see if it is indeed not repaired.

Quicken Program missing - Microsoft Community

annewalker say:

Quicken Program missing

Original title: Quicken

We've just installed Windows 10 from XP and I now can't find my Quicken programme.  Can anyone help?
taurarian say:

Do you have backups of your Quicken data files?

Quicken normally installs under "programs" on the boot drive normally "C".

Browse to the Quicken folder (Default location is C:\Program Files\Quicken).

  1. Double-click the Qw.exe file to start the program.
  2. Right-click the Qw.exe file and select Send to Desktop to create a new shortcut.

adapted from: http://quicken.intuit.com/support/help/account-transaction-issues/clicking-the-desktop-icon-fails-to-start-quicken/GEN82029.html

 

If Quicken isn't installed and you don't have backups of your Quicken data, you may have to wait until the IT Guy returns and find out exactly what he has done etc.

I wish I could help you further, but the most important part is the data files, so hopefully you still have them.
