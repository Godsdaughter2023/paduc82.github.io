One other item to keep in mind in addition to Guruprasad's response is that when you have both O'2010 and O'2013 installed on the same system......

 

Any 3rd party programs that invoke Outlook (i.e. send mail to/via Outlook) will end up using O'2013 and there is no way to get around that. 1. Go to this link: https://profile.live.com

3. Click Edit.

4. Then click Remove this connection completely.
