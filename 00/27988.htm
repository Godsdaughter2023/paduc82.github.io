Microsoft Word Indexing - Microsoft Community

marty b say:

Microsoft Word Indexing

There seems to be a bug in Microsoft Word indexing.

In order to accurately build an index in Microsoft Word, it is NECESSARY to turn off "Show All Formatting Remarks" in FILE>OPTIONS>DISPLAY before inserting or updating an index otherwise Word can list the location of some index entries as the wrong page number.

So what a normal person might do is to first turn off Show All Formatting Remarks, then build an index by marking the words to be included in the index and then clicking INSERT INDEX (OR UPDATE INDEX).  But that could introduce an incorrect index.  The reason is that every time a user marks a new word to appear in the index, Word TURNS ON Show All Formatting Remarks.

The way WORD SHOULD work is, whenever a user inserts or updates an index, Word should FIRST automatically turn off Show All Formatting Remarks  in order to insure an error-free index.

This appears to be a bug or an oversight in the programming for indexing.
