david m. connell say:

Unfortunately, Windows 8 64-bit doesn't support 16-bit programs, so there's no way to install or run it directly. 32 bit Windows has backward compatibility for 16 bit programs built-in, but 64 does not.

You might be able to do some workarounds: You could set up a virtual machine using VMWare or Virtualbox. It's not too difficult to set up - I recommend a little research, but the Virtualbox setup is pretty straightforward for a Windows virtual machine.

The problem is, it'll require you to have an unused Windows license, which you'll probably have to buy.

Another option is to install the 32 bit version of Windows 8 on that machine. The problem with that is 32 bit Windows can't use more than around 3.5 GB of memory. So if your computer has 4 or more GB of memory, it'll go unused.

But if that program is critical to your business, it might be worth the trade-off.

andre da costa say:

Only if it comes with Windows 10 Pro OEM preinstalled will you have downgrade rights.

Understanding downgrade rights

Consumer versions of Windows 10 Pro that come preinstalled on PCs have downgrade rights to a previous version of Windows (example: Windows 8, Windows 7, Windows Vista).

If your copy came pre-installed on your system, you may downgrade your copy to Windows 8.

If not you need purchase a copy of Windows 8.1 and install the same on your computer.

• Downgrade rights are only available for licenses obtained through OEM or Volume Licensing (VL) and not apply to Full version (FPP).

• OEM downgrade rights apply to Windows 10 Pro and allows for downgrades up to two prior version

• Software Assurance through VL (Volume licensing) provides the greatest flexibility, allowing for downgrades to all prior versions and editions besides Windows 7 Ultimate and Vista Ultimate editions.Without Software Assurance coverage, VL has downgrade rights to prior versions of Windows with the exception of Enterprise.

• It is not possible for a Windows 10 copy with multi-language function to downgrade to Windows 8 Pro or Windows XP Pro as multi language function is exclusive to Windows 10 Enterprise and Pro edition.

They both relate to email.

Webmail just means that the email is accessed using a web browser such as Internet Explorer or Firefox rather than a dedicated email client app such as Microsoft Office Outlook, Windows Live Mail or Thunderbird.

Email client apps can be the default client app which allows, for example, other apps or right-click menus (Send to->Mail recipient) to open them on demand. Web mail usually cannot.

ajay_sattee say:

Acer Aspire ES1-512 Touchpad/Mouse Not working in windows 10

Came across this issue and seeing soooo many folks have reported this issue, wanted to share my successful fix....

Windows 10 Home failed to recognise the Mouse (touchpad).

Tried the following:

Based on my similar experience with some ASUS and HP stream laptops, many had pointed to the IO driver. Most folks tend to try the synaptics or Elan driver, but it is often the IO driver that initialises some components on the motherboard. Despite that, in this instance, it failed to fix the issue.

A couple of cautions for those considering this.

1. If available, use the S/N or SNID to find the exact drivers/BIOS page on Acer Website. Wrong BIOS may destroy your motherboard! The BIOS utility is an EXE file that can be run from within windows. Just to add, there is no mention on Acer page whether it resolves the touchpad issue....thanks Acer!

2. If you already had windows 10 installed, upon restart, you may encounter no boot device found (I did). Re-enter BIOS and change boot mode from UEFI to Legacy and windows booted fine.
