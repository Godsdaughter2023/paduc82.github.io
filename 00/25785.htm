All of my conversions have one thing in common - I have properly removed all existing realtime antimalware and used the manufacturer's cleanup tool.  I have updated all machines to the latest service pack and updates.  And, I have used disk cleanup and disk defrag on all machines.

Verify that you have removed all other  real-time antimalware products that were ever installed on your PC, including free/trial product that were installed when the PC was purchased (Norton, McAfee, TM, AVG, OneCare, Malwarebytes (paid), etc). Then, use the cleanup tools here:

List of anti-malware program cleanup/uninstall tools

Verify that you are up-to-date with patches and service packs.

Remove Zone Alarm if you have it and use the Windows Firewall.

Verify that you are not using a proxy server.

Verify that you have “install updates automatically” selected in Windows Update (even though the MSE definition updates are optional).

In IE, reset all security zones to default level.

Through the "MyAccount" site, http://office.microsoft.com/myaccount , Office 365 gives consumers much more direct control over their licenses than in Office 2010 and earlier.

graham mayor say:
As you appear to realise, English versions of Office 2003 included French proofing tools, so you can spell check in English or French.

Regarding the typing of two languages, you need to setup the input languages with their appropriate keyboard mappings in Windows Regional settings, which in Windows XP is accessed from Windows Control Panel > Regional and Language Options > Languages > Details.

Add the French language and keyboard, so that you have your local flavour of English and French only.

From Language Bar check 'show additional language bar icons in the task bar and Show the language bar on the desktop.

You will now see the language icon on the Windows Task Bar when you OK out of the dialog.

From Word 2003 Tools > Language > Set Language and uncheck 'detect language automatically'.

You should now be able to type in either language by selecting the appropriate Windows input language either from the task bar icon or by using the keyboard shortcut, which by default is LeftAlt + Shift.

The only disadvantage is that you cannot see the French keyboard mapping without using the on screen keyboard. This may not be too much of a disadvantage for someone who uses a French keyboard at home. You could however add a French keyboard to the PC should it become a problem.
