Digging around in Windows looking for things that don't belong will always result in failure for someone without the training and knowledge required to understand what does belong.  This is the common problem for something as complex as the Windows operating system, since it long ago grew to a size and complexity that leaves all but a handful of highly technical individuals completely lost.

You're probably better off starting a thread describing the specific items that are occurring which make you believe you have malware and letting someone guide you through the steps required in order to determine if that's really true.  Many things about the operation of Windows may seem suspicious to a typical consumer.

Cookies are small programs that are put on your computer automatically when you visit a website.  They allow improved communication between your computer and the website so you don't have to locate it each time you want to visit.

 

Sometimes that is good, say when you want to interact smoothly with your bank, email provider, etc.  Other times, not so good, say when you are visiting a website you have never been to before and all of a sudden you start getting weird pop-ups.

 

 

You have to determine the level of privacy you are comfortable with.  This is done in the Internet Options, Advanced , Privacy Tab (which you mentioned).  In the Privacy tab, if you move the slider up to the top, you see "Block all cookies.  That means not even your bank will be able to communicate with your PC.  If you move the slider all the way down, you see "Accept all Cookies".  This means you will be getting loads of unwanted things popping up.  Most people use Medium High, which is somewhere in between both extremes.

 

If you click the "Sites" button, another box appears which will allow you to be more specific.  If you trust your bank, for example, you can enter their web address in the box to "allow" them to send cookies to your PC.  If some unknown site is repeatedly sending you trash, you can enter their web address in the box and select "Block"

 

If you simply want to get rid of all cookies (bank, email, trash, etc) and start over: 

Tools

Delete Browsing History.

Check ONLY the box "Cookies and Website Data"

(All other boxes should be unchecked)

Delete

OK

 

NOTE:  This is only temporary.  As soon as you visit a website, another cookie is attached to your computer.  Unfortunately, that's what we have to live with.

Private Declare Sub Sleep Lib "kernel32" (ByVal dwMilliseconds As Long)

Private Type RECT
    Left As Long
    Top As Long
    Right As Long
    Bottom As Long
End Type

Private Const SWP_NOSIZE = &H1
Private Const SWP_DRAWFRAME = &H20

Private m_colhWnd As Collection

Public Sub Main()
    Set m_colhWnd = New Collection
    
    Do
        'Pause to prevent excessive CPU use
        DoEvents
        Call Sleep(1)
        
        Call FixFind
    Loop
    
    Set m_colhWnd = Nothing
End Sub

Public Function FixFind() As Boolean
    Dim lnghWnd As Long, rct As RECT
    
    'Minimum TOP position you'll accept for your 'Find' window
    Const MIN_TOP = 90
    
    'Find the 'Find' window. Exit if not present
    lnghWnd = FindWindow("#32770", "Find")
    If lnghWnd = 0 Then Exit Function
    
    'Check if we moved this 'Find' window before.  If so, exit (don't fight w/ user over positioning)
    If AlreadyProcessed(lnghWnd) Then Exit Function
    
    'Get position info for 'Find' window.  If off screen, move it and record it as processed in our collection.
    Call GetWindowRect(lnghWnd, rct)
    If rct.Top < MIN_TOP Then
        Call SetWindowPos(lnghWnd, 0, rct.Left, MIN_TOP, 0, 0, SWP_NOSIZE Or SWP_DRAWFRAME)
        Call m_colhWnd.Add(lnghWnd, CStr(lnghWnd))
    End If
End Function

Private Function AlreadyProcessed(ByVal hWnd As Long) As Boolean
    Dim lngTemp As Long
    
    On Error GoTo Err_AlreadyProcessed
    
    lngTemp = m_colhWnd(CStr(hWnd))
    AlreadyProcessed = True
    
Err_AlreadyProcessed:
    If Err.Number <> 0 Then Err.Clear
End Function
