graham mayor say:
Almost certainly this is attributable to an orphan lock file relating to the document. Word is a single user application. When you open a Word document Word creates a (normally hidden) lock file in the same folder as the document, which locks that file to the person who has opened the document, thus preventing other users from editing it at the same time. When the document is closed, the lock file is normally cleared. In your case some network issue is preventing (or delaying) the elimination of the lock file. The lock file remains present and thus when you try to open it again you get the other user warning. The other user is in fact you.

When this happens on my network, it is usually a sign that the file server needs rebooting.
