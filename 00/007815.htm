(this link may not work OnClick but will if pasted into address bar)

In Excel is it possible to calculated the arc length?

Thank you

hansv mvp say:

See https://en.wikipedia.org/wiki/Sine#Arc_length for some mathematical background. The arc length of a sine wave is an elliptic integral of the second kind.

The following function can be used to calculate the arc length of y = a*sin(b*x) from x = x1 to x = x2:

Function ArcLength(a As Double, b As Double, x1 As Double, x2 As Double) As Double
    ' Computes approximate arc length of a*sin(b*x) from x=x1 to x=x2
    Const n = 1000 ' we divide the arc into this many pieces
    Dim i As Long
    Dim dx As Double
    Dim dy As Double
    dx = (x2 - x1) / n
    For i = 1 To n
        dy = a * Sin(b * (i - 1) * dx) - a * Sin(b * i * dx)
        ArcLength = ArcLength + Sqr(dx ^ 2 + dy ^ 2)
    Next i
End Function

For example, to find the arc length of the standard sine wave y = sin(x) from x = 0 to x = 2pi, you can use

=ArcLength(1, 1, 0, 2*PI())

This results in 7.640392. The actual length according to the Wikipedia article is 7.640395... so the approximation is quite good.

In your workbook, you can use

=ArcLength(B2, PI()/180, 0, 360)

B2 is the radius, and PI()/180 is the conversion factor from degrees to radians. For a radius of 10 this results in 362.7261.
