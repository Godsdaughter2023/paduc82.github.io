a. user say:

Minor Edit:

A minor edit that removes one or more words or changes the amount of text in a negative direction is not considered to be content removal, since it does not remove information

For example, if the text originally says the following:

There is an the fish in the bowl.

It is not grammatically correct, and should be changed to:

There is a fish in the bowl.

This edit removes one word and shortens another.

Some examples of content removal are:

Single word
Old version: The mixture is made from water, clay, sand, and gravel.
New version: The mixture is made from water, clay, and gravel.

Sentence
Old version: The elephant lives on the prairie. It walks over three miles to get some water. It spends hours drinking. Then it goes home and goes to sleep.
New version: The elephant lives on the prairie. It walks over three miles to get some water. Then it goes home and goes to sleep.

The above examples show how removing just a small amount of text under these circumstances can change the meaning of what is stated. This does not mean it should never be done. It just means that when it is done, it should be done with a good reason, should be explained.

Hi AnujPink,

When you have both a Wi-Fi and a cellular data connection, the default setting is for the phone to always use Wi-Fi instead of cellular.  You do not have to turn off your cellular data connection, the Wi_Fi connection will be used whether the cellular

If your Wi-Fi connection is slower than your cellular data connection and you want to use to use the cellular data connection while still keeping both connections on, then you can change the phone settings.  Go to Settings, then press cellular.  Then under For limited W-Fi connectivity, change don't use cellular data to use cellular data.  That will set the phone to use the fastest connection.  This will use your data plan and may incur extra charges.

    For i = LBound(arrStrSplitPath) To UBound(arrStrSplitPath)

        End If
    Next i

End Function
