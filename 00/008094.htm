In reply to deleted message

This has been a problem for a long time. A couple of thing appear to alleviate the problem but I have not done sufficient testing to guarantee that it is a solution for every instance but it has always worked for me.

Firstly set the zoom (bottom right of worksheet) to 100% and keep it that way.

Secondly set the Picture Property to "Move and Size with cells".

Thirdly set the Picture property to NOT lock the Aspect ratio.

Align the Top, Left, Width and Height to perfectly align with cell borders.

The following VBA code examples can be used to perform the above. (See the comments where editing will be required.)

The first example will align all of the images on the worksheet to align with the cell where the top left of the image starts and the bottom right of the image finishes.

The second example is a combination of 2 subs (between the asterisk lines). It will align individual images to specified cell ranges. You will need to edit the code to specify the image (Picture) name and also the range where the image is required plus add similar code for more than 2 pictures.

Not sure of your expertise so some guidelines to install the code.

  1. Firstly ensure you have a backup of your workbook in case it does perform as required.
  2. Alt and F11 to open the VBA editor.
  3. Select menu item Insert -> Module
  4. Copy the code below and paste into the editor. (You can copy it all in and decide later which code you want to run).
  5. Edit the code as required. (See the comments in green re this)
  6. Close the VBA editor (Cross with red background top right or turns red when cursor hovers over it)
  7. Save the workbook as "Macro enabled workbook"
  8. Ensure Options set to allow macros. See Help for how to do this. (Select the option to "Disable all with notification" should be OK.)
  9. To run the macros select the Developer ribbon. (If not displayed see Help for how to display).
  10. Select Macros icon (towards left of ribbon)
  11. Select the required macro (PositionAllImages or PositionIndivImages) and then click Run.

In the event that you are still having problems keeping the shapes size and alignment then the code can be called from a Workbook Open event or a Worksheet Activate event to automatically resize and postion the shapes. If you require further help with this then let me know.

Sub PositionAllImages()
    'Postions all shapes to align with borders of first and last cells of where the existing image is placed.
    Dim ws As Worksheet
    Dim shpImage As Shape
     
    Set ws = Worksheets("Sheet1")       'Edit "Sheet1" to your worksheet name
    For Each shpImage In ws.Shapes
        With shpImage
            .Placement = xlMoveAndSize
            .LockAspectRatio = msoFalse
            .Left = .TopLeftCell.Left
            .Top = .TopLeftCell.Top
            .Width = .BottomRightCell.Offset(0, 1).Left - .TopLeftCell.Left
            .Height = .BottomRightCell.Offset(1, 0).Top - .TopLeftCell.Top
        End With
    Next shpImage
End Sub
 
 
'*************************************************************************
'The 2 subs between the asterisk lines go together
Sub PositionIndivImages()
    'Positions individual shapes within a specified range of cells.
    Dim ws As Worksheet
    Dim shpImage As Shape
    Dim rngForShape As Range
   
    'Position first image
    Set ws = Worksheets("Sheet1")       'Edit "Sheet1" to your worksheet name
    Set shpImage = ws.Shapes("Picture 1")   'Edit "Picture 1" to your image name
    Set rngForShape = ws.Range("B4:L26")    'Edit "B4:L26" to range for picture
    Call PositionImages(shpImage, rngForShape)
   
    'Position second image
    Set shpImage = ws.Shapes("Picture 2")   'Edit "Picture 2" to your image name
    Set rngForShape = ws.Range("N4:X26")    'Edit "N4:X26" to range for picture
    Call PositionImages(shpImage, rngForShape)
   
    'Repeat code here for additional images
   
End Sub
 
 
Sub PositionImages(shp As Shape, rng As Range)
    With shp
        .Placement = xlMoveAndSize
        .LockAspectRatio = msoFalse
        .Left = rng.Left
        .Top = rng.Top
        .Width = rng.Width
        .Height = rng.Height
    End With

End Sub
'***************************************************************************
