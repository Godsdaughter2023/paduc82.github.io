I would ttroubleshoot this by disabling the 'New' menu entries (except for perhaps Folder and Shortcut). If the menu then functions & doesn't crash Explorer, you can re-enable the items one-by-one until you find the misbehaving one.

A 'New' context menu item appears if the associated file extension has a subkey named 'ShellNew' in the Registry. So open regedit.exe, Ctrl+F (search), type 'ShellNew' and make sure you are searching for keys:

The first one found will most likely be under .bmp. When found, right-click the key, select Rename, and prefix with an 'x' (renaming is sufficient to disable):

Ctrl+F3 will repeat the search, rename all entries except for .lnk and Folder, restart Explorer or log off & back on for the changes to take effect, then test the behavior.

a. user say:

Hi,

This issue might if the video file that you are trying to play is corrupt or missing some codec. Sometimes, outdated display driver could be one of the other reasons causing the issue.

  1. What is the Make and Model of the computer?
  2. Is the issue specific to particular .mpeg file on the computer?

 .MPEG is a supported format in Windows Media Player. However, MPEG-1 system streams do not exclusively use the .mpg and .mpeg extensions. MPEG-2 program streams also frequently use .mpg and .mpeg file extensions, but they contain MPEG-2-encoded video. Because Microsoft Windows operating systems provide only an MPEG-1 video decoder, Windows Media Player cannot play MPEG-2 program streams without an additional MPEG-2 video decoder (also known as a DVD decoder pack) installed. Refer to the article to get all the details:

http://support.microsoft.com/kb/316992/en-us   

Check if the player is able to play another video (of .mpeg file format).

First of all, it’s better to make sure that you update the version of display driver by referring to the manufacturer’s website of the computer and downloading the latest version of driver for Windows 8.1.

Refer to the article to know how to update driver on Windows 8.1:

http://windows.microsoft.com/en-us/windows-8/all-drivers

If the issue still persists and the issue is not specific for a particular .mpeg file, follow the steps to run the troubleshooter:

  • Type Troubleshooting on the start screen.
  • Click on Troubleshooting icon.
  • Click on View all on the left side panel.
  • Double click on Windows Media Player Settings.
  • Follow the onscreen instructions to run the troubleshooter.

Note: Windows Media player troubleshooter resets the settings for the player to default.

For issues related to Video app, you may run the app troubleshooter and check if it fixes the issue.

Refer to the link to run the Modern User interface app troubleshooter to troubleshoot the issue:

http://windows.microsoft.com/en-us/windows-8/what-troubleshoot-problems-app

You may also “Reinstall the app” if the troubleshooter doesn’t help.
