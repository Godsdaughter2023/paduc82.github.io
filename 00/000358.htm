Conditional Formating with dates

Rob

ossiemac say:

Using a formula for the Conditional format then the following formula.

=AND(TODAY()>=EDATE(TODAY(),-1)+1,A2<20)

Explanation of formula

A2 in the formula is the first cell of the "Applies to" range in the Conditional format. (Edit A2 as required)

Edate returns the last day of a month in relation to the month in the function. The month in the function is the month of today's date. The month to return the last day of month is previous month due to the -1 (0 is current month and the month can be any number of months before or after the month referenced by Today).  The +1 adds one day to last day of previous month which makes it first day of current month.

"AND" is simply that both the Today() must be greater or equal to 1st day of month and then value in the cell less than 20.

Right click the calendar, view code and paste this in

Private Sub Calendar1_Click()
    ActiveCell.Value = CDbl(Calendar1.Value)
    ActiveCell.NumberFormat = "mm/dd/yyyy"
    ActiveCell.Select
End Sub

Now paste this code in in the same place, you may need to change the range, I used A1:A10

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Target.Cells.Count > 1 Then Exit Sub
    If Not Application.Intersect(Range("A1:A10"), Target) Is Nothing Then
        Calendar1.Left = Target.Left + Target.Width - Calendar1.Width
        Calendar1.Top = Target.Top + Target.Height
        Calendar1.Visible = True
       
        Calendar1.Value = Date
    ElseIf Calendar1.Visible Then Calendar1.Visible = False
    End If
End Sub

That still doesn't explain all that much. Are you talking about the number of different characters used in a word / sentence / ...? If so

  • what about upper and lowercase? Is 'b' the same as 'B' or not?
  • how do you define accented letters? Are they the contraction of two characters (e.g. o and ¨) or are they just one (e.g. ö)?
  • what about punctuation? Is a ':' a character or not?

Basically provide us with the exact definition (or algorithm) of the word 'unical'. Without it, we can only guess what you are looking for.

A basic macro to get you started (you will have to add every character you want to count) could be:

Sub Unical()

    ' Variables.
    Dim charsToCheck(0 To 200) As String
    Dim i As Integer
    Dim cnt As Integer
    Dim res As String
   
    ' Define all the characters to look for.
    charsToCheck(0) = "a"
    charsToCheck(1) = "b"
    charsToCheck(2) = "c"
    charsToCheck(3) = "d"
    charsToCheck(4) = "e"
    charsToCheck(5) = "f"
   

    cnt = 0
   
    For i = 0 To UBound(charsToCheck)
        If charsToCheck(i) <> "" Then
            Selection.Find.ClearFormatting
            With Selection.Find
                .text = charsToCheck(i)
                .Replacement.text = ""
                .Forward = True
                .Wrap = wdFindContinue
                .Format = False
                ' Set the following line to True if you want to
                ' count 'b' and 'B' as separate characters
                .MatchCase = False
                .MatchWholeWord = False
                .MatchWildcards = False
                .MatchSoundsLike = False
                .MatchAllWordForms = False
            End With
           
            If Selection.Find.Execute = True Then
                ' Found another 'unical' character.
                cnt = cnt + 1
                res = res & " " & charsToCheck(i)
            End If
        End If
    Next

    MsgBox "Found " & cnt & " unical characters:" & vbCrLf & res, vbOKOnly
   
End Sub 

More information on installing macros from listings can be found at http://www.gmayor.com/installing_macro.htm
