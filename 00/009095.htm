How do I transfer a mailing list from Microsoft Publisher to Microsoft - Microsoft Community

carolyn godfrey say:

How do I transfer a mailing list from Microsoft Publisher to Microsoft Word?

How do I transfer a mailing list from Microsoft Publisher 2007 to Microsoft Word 2007

a. user say:

If you created your mailing list in Publisher 2007, at some point you will have been prompted to save it and it should have been saved as an "Office Address List" file, with a .mdb extension. By default it will probably be in 
C:\Users\<yourusername>\Documents\My Data Sources 


In Word 2007, you should be able to re-use that data source directly. Ideally, make sure Publisher is closed, then in the Word Mailings tab, Start Mail Merge group, click Select Recipients, then Use existing list...

Then navigate to the correct folder (by default, I believe you will actually be looking at a "folder" called "All Data Sources". This is actually the result of a Windows file search, and should include all the .mdb type files in My Data Source. You should be able to find your file in there and open it.

Office Address Lists are just Microsoft Access .mdb format files that have to conform to a particular structure. You should be able to share your address list between Publisher and Word, but it's advisable not to try to open and work with it in Access in case you damage that structure.
