I'm sorry you found out the hard way. Microsoft's own support - whether level 1, level 2 or level 100 - consists mostly of hired contractors who are reading from scripts and flowcharts. They are certainly polite, but their actual knowledge is limited.

If you want to sell your computer to someone, you should be doing a factory restore, which restores your computer to exactly how it was when it left the factory - hence the name. In so doing, everything that previously existed on the hard drive is unrecoverable. Thus, you have no worries about leaving your personal information behind.

The procedure for a factory restore is created by the individual PC manufacturers and you can find it on the manufacturer's website. Instructions will be in quite some detail, typically with illustrations. The only thing you need to bear in mind is to use the instructions that apply to your model.

Presuming that no one has monkeyed around with the PC manufacturer's hidden restore partition, this procedure is fast and foolproof.

You can go about this in two ways: You can do the factory restore yourself, which will require you to setup a single Administrators account that your friend can always replace on their own. Or, for maximum effect, you can do the restore with your friend right next to you, and your friend can setup the computer exactly as if it was powered on for the first time.

waseemulla shariff say:
Hi,
 
We recommend you to use Bcdedit.exe to manually create an entry in the BCD Boot.ini file for the earlier version of the Windows operating system.
 
To do this, type the following commands at a command prompt.
1. In Windows 7, click Start, type cmd, right click on cmd and click run as administrator.
2. Type the following command and hit enter: Drive:\Windows\system32\Bcdedit /create {ntldr} /d "Description for earlier Windows version"
Note: In these commands, Drive is the drive where Windows 7 is installed.
Note: In this command, Description for earlier Windows version can be any text that you want. For example, Description for earlier Windows version can be "Windows XP"
3. Type the following command and hit enter: Drive:\Windows\system32\Bcdedit /set {ntldr} device partition=x:
Note: In this command, x: is the drive letter for the active partition.
4. Type the following command and hit enter: Drive:\Windows\system32\Bcdedit /set {ntldr} path \ntldr
5. Type the following command and hit enter: Drive:\Windows\system32\Bcdedit /displayorder {ntldr} /addlast
6. Restart the computer.
 
For more information on this, check this link: http://support.microsoft.com/kb/919529
 
For more information on "Change the default operating system for startup (multiboot)" check this link:
http://windows.microsoft.com/en-us/windows-vista/Change-the-default-operating-system-for-startup-multiboot
 
Hope this helps.
