Private Function TimeLine() As Boolean

'   Description:Update Timeline Slicers from Master Slicer
'   Inputs:     *None       (globals provide input)
'   Outputs:    *None       (Timeline slicers updated)
'   Requisites: *None
'   Example:    *None - This is an event handler found in ThisWorkbook module

'     Date   Ini Modification

    Const cRoutine      As String = "TimeLine"

    Dim dStartDate      As Date             'Start Date
    Dim dEndDate        As Date             'End Date
    Dim bCleared        As Boolean          'Filter Cleared Flag
    Dim bEvents         As Boolean          'Events Enabled Flag

    TimeLine = Failure

'   Initialize Variables
    Set oWkb = Worksheet.Parent
'   Prevent cascading events
    bEvents = Application.EnableEvents
    Application.EnableEvents = False

'   Procedure
    If MasterTimeline <> 0 Then

    '   Get Master Slicer's dates
        Set oSlicer = oWkb.SlicerCaches(MasterTimeline)
        bCleared = oSlicer.FilterCleared
        If Not bCleared Then
            With oSlicer.TimelineState
                dStartDate = .FilterValue1
                dEndDate = .FilterValue2
            End With

                If bCleared Then _

        Next

julianbartel say:

I had the same problem on my SP2. After reading all the threads and many trial and error, I finally got Windows 10 1703 back on my SP2. The final solution was quite simple:

  1. Secure Boot enabled (so no red boot screen).
  2. Clean install of Windows 8.1 from USB stick made with Rufus .
    • the SP2 recovery image did not work with Secure Boot enabled.
  3. Finish the installation of Windows 8.1 (Updates not required)
    • The Windows 10 1703 USB Stick was still unable to boot on my SP2, so I startet the Upgrade from within Windows 8.1.
  4. When it came to the first reboot, I was back in UEFI. Disabled Secure Boot now and detached Windows 10 USB stick.
  5. With the next restart, the Windows 10 Upgrade completed successfully.
  6. Installed all updates
  7. Enabled Secure Boot again.
  8. Surface Pro 2 is back on Windows 10 1703 and Secure Boot working as expected :-)

Hope that helps anyone.
