Reference the Content Control with focused in VBA - Microsoft Community

mario insky say:

Reference the Content Control with focused in VBA

In VBA, what code does reference the active (focused) content control.

I want to select the content of the control when clicked using VBA Document_ContentControlOnEnter event.

Thanks

jay freedman say:

When you use the dropdowns at the top of the code pane to create the event handler (choose Document on the left and then ContentControlOnEnter on the right), VBA automatically supplies the required event handler signature

Private Sub Document_ContentControlOnEnter(ByVal ContentControl As ContentControl)

The argument declares a variable named ContentControl -- confusingly having the data type ContentControl. When the event fires because the cursor entered a content control, that variable contains a pointer to the specific content control involved.

I prefer to change the name of the variable to avoid the confusion, so the code I'll suggest is

Private Sub Document_ContentControlOnEnter(ByVal CC As ContentControl)
  CC.Range.Select
End Sub
