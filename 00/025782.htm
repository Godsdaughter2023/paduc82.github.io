From your description -

#1 - your original machine had the email account configured using POP3 (hence the PST file)

#2 - your new machine has the email account configured using IMAP (hence the OST file)

There' s nothing to print or need to copy to your old PST file, just configure the account using POP3 instead of IMAP. The POP account will download all new emails automatically - just make sure that you configure it to use the existing PST file instead of letting Outlook create a new one

  Dim Arr() As Byte, Count As Integer
  Dim Index() As Long
  Dim Result()
  Dim i As Long, j As Integer, k As Long
 
  ReDim Arr(1 To 75)
  For j = LBound(Arr) To UBound(Arr)
    Arr(j) = j
  Next
  Count = 6
 
  'Setup space for index and result array
  ReDim Index(1 To Count) As Long
  ReDim Result(1 To Count)
  'Setup index for first combination
  For i = 1 To Count
    Index(i) = LBound(Arr) + i - 1
  Next
  Do
    'Build combination
    For i = 1 To Count
      Result(i) = Arr(Index(i))
    Next
    'Save it
    k = k + 1
    Cells(k, 1).Resize(, UBound(Result)) = Result
    DoEvents
    If k = Rows.Count Then
      Sheets.Add
      k = 0
    End If
    'Get next index
    i = Count
    j = 0
    Do
      'Max. position for this index reached?
      If Index(i) = UBound(Arr) - Count + i Then
        j = j + 1
        'Move index before up one step in next loop
        i = i - 1
        If i < 1 Then Exit Sub
      Else
        'Increment position
        Index(i) = Index(i) + 1
        'Setup next indices
        For j = 1 To j
