Getting Data From Outlook E-Mail into Excel - Microsoft Community

jambocaz2 say:

Getting Data From Outlook E-Mail into Excel

Good Day All,

I'm receiving data via Microsoft Outlook E-mail that I want to get into excel on a spreadsheet.  Each E-Mail has the same fields my problem is that I have many many e-mails so I'm wanting to automise this somehow.

Hopefully someone can help, all thoughts appreciated.

Many Thanks

JC

business spreadsheets say:

You can activate the Outlook object in VBA and import the emails from a folder of choice.  To do so, you need to create a macro like the one below.  You also need to create a Reference (under Tools) to the Microsoft Outlook Object Library in the VBA Editor.

VBA Code:

Sub ImportEmails()
    Dim olA As Outlook.Application
    Dim olNS As Outlook.Namespace
    Dim olF As Outlook.MAPIFolder
    Dim olM As Outlook.MailItem
    Dim lrow As Long
    
    Set olA = New Outlook.Application
    Set olNS = olA.GetNamespace("MAPI")
    
    'Select the folder number here where the first one is 0
    Set olF = olNS.Folders.Item(2)
    lrow = 1
    For Each olM In olF.Items
        With ActiveSheet
            .Cells(lrow, 1) = olM.SenderName
            .Cells(lrow, 2) = olM.Subject
            .Cells(lrow, 3) = olM.Body
            lrow = lrow + 1
        End With
    Next
    
    Set olM = Nothing
    Set olF = Nothing
    Set olNS = Nothing
    Set olA = Nothing
End Sub

This will import the email text into each cell.  Further text manipulation can be made on the Body text to extract the data that you need for your analysis.

graham mayor say:
If you create a vba userform to collect your user input then you can validate the form when the fields are filled or when the button to leave the form is clicked.

At its simplest, imagine a userform with two text fields and a command button (all with default names)

The following code will not let the user proceed until there is content in the two fields. The second macro prevents the user from closing the box with the X in the top right corner.

You can make the validation as simple or as sophisticated as you wish.What it cannot do is establish whether the data input is correct. It can only check that it fits certain criteria.


Private Sub CommandButton1_Click()
With Me 'Check if the two fields have content
    If .TextBox1.Value <> "" _
       And .TextBox2.Value <> "" Then
       'if they do complete the processes
        Me.Hide
        'do what you want with the form data here
    Else 'if not show a warning
        MsgBox "Complete all the fields", vbCritical
    End If
End With
End Sub


Private Sub UserForm_QueryClose(Cancel As Integer, CloseMode As Integer)
If CloseMode = vbFormControlMenu Then Cancel = True
End Sub
