  3. A system configuration box will pop up.
  4. Click on Boot Tab.
  5. Select or click the check box Safe mode with networking.
  6. Click on Apply and ok.
  7. You will then see a restart button. Click it to restart your computer. 

Once you completed the steps above follow the below steps on how to go back to normal mode:

  1. Press Windows Key + R.
  2. Run will open, type msconfig and hit the Enter key.
  3. A system configuration box will pop up.
  4. Click on Boot Tab.
  5. If you see safe mode box has a check mark in it, uncheck the safe mode box and hit apply or ok.
  6. You will then see a restart button. Click it to restart your computer.
  7. When your computer comes back up, sign in and you will now be in "normal mode".

Anyway, here's what I did.

First I toally uninstalled iTunes, Quicktime, Apple Software updater, ____-anything with Apple in the name.

I opened regedit using "run as administrator".

I then went to the registry key:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Installer\UserData\S-1-5-18\Components\DA42BC89BF25F5BD0AF18C3B9B1A1EE8\108B43C0CEA676640B35306AE7D24051

Right click & choose permissions
click advanced, and then the owner tab
type "administrators" in current owners field and check th box to replace the owner on subcontainer and volumes

(Note: I did this because I had another key with permission issues after I fixed the original onelisted above)

Apply settings and then click permissions tab. Once again, add "administrators" to the list of permission entries, then click edit and check all permission boxes. Also click the 2 bottom boxes concerning the inheritable permissions.

Click OK

Now back on the main permissions tab, choose administrators in the group or user name window and then check full control and read checkboxes. Click OK.

Back in the main regedit window, you should now see keys in the right pane for each subfolder in:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Installer\UserData\S-1-5-18\Components\DA42BC89BF25F5BD0AF18C3B9B1A1EE8

Regarding changing languages for Office RT, please do the following (note: I cannot speak other languages so bare with me as all instructions will be in English.)
1. Launch Word 2013 from the Start Screen.
2. Tap the 3 dots in the upper right corner.
3. Touch File
4. Select Options
5. Select English
6. Select Set as Default.

If English does not show, follow steps 1-4 then do the following:
1. Select the drop down [Add additional editing languages]
2. Select the language desired (I'll use English for this example)
3. Select Add.
4. Click on Add more Display and Help languages with the language pack from Windows Language preferences.
5. Click on Add a language.
6. Click on Add.
7. Select the new language.
8. Tap Move up until it is at the top of the list
9. Double tap the language to open it
10. Touch Download and install language pack.
11. Once the language is downloaded, click on the X to close the Language window.
12. Under Display Language, select English.
13. Click on Set as Default.
14. Click on OK.
15. Click on OK.
16. Close Word 2013.

When you re-open Word, you will be in the language you chose.

Hi LanxsHH I am sorry to hear this did not help. There are more steps to try with this type of issue.

Method 1: Reset Router.

Click the Start button and navigate to Settings > Network Connections.

Right-click the Wireless Network Connection and select View Available Wireless Networks.

Right-click the name of your wireless connection and click Disable and again enable it.

Method 2: I suggest you to uninstall the network adapter drivers and re-install the same and check if that fixes the issue.

Follow the steps to uninstall the network adapter drivers.

a. Press Windows Key + X and then click Device Manager.
b. Expand network adapters.
c. Right on the network Adapter and select uninstall.

e. Restart the computer.

Once completed, restart your computer. Your network adapter Drivers will automatically install.

You may also logon to the computer manufacturer website and download the latest drivers and get it installed.
