How do I get excel to return the last non empty cell in a row, then - Microsoft Community

pauldonahuetd say:

How do I get excel to return the last non empty cell in a row, then Divide that cell by a different value in the same Row.

 Title pretty much says it all, but to illustrate my question:

I have Row 1, and I want A1 to be the title of the row, so only text in A1. Then B1 will be the first Value in the row.

I want the cell at J1 to divide B1 by the last non empty cell in the row. So say I enter a value at C1, I want the formula to give me B1/C1. However, If i update the sheet and add something into E1, I want excel to give me B1/E1.

I said I wanted the cell at J1 to be the cell where the sum of the formula is shown.

I'm new at excel, so if you give me an answer please add some explanation in the steps so I'll know how to do it myself in the future, like how to adjust the formula to different rows (so if I want to use the same forumla for Row 3), or adjust the ranges (if I want the forulma to be in N1, and to show me B1 divided by the last non empty cell (ranging from C1 to M1).
hansv mvp say:

If you have text in A1, a value in B1 and a formula in J1, J1 would be the last non-empty cell in the row. So I assume that you meant the last non-empty cell to the left of J1.

The value of this cell is LOOKUP(1E+300,C1:I1). In this expression, 1E+300 is simply a huge number larger than anything you're likely to enter in a cell, it is a 1 followed by three hundred 0s. C1:I1 is the range we want to inspect.

So the formula in J1 would be =B1/LOOKUP(1E+300,C1:I1)

This formula returns #N/A if nothing has been entered in C1:I1 yet. To suppress this, we can use the IFERROR function:

 

=IFERROR(B1/LOOKUP(1E+300,C1:I1),"")

 

If B1/LOOKUP(1E+300,C1:I1) results in an error value, the formula returns an empty string "" instead.

 

For row 3, the formula would be

 

=IFERROR(B3/LOOKUP(1E+300,C3:I3),"")

 

And if you want the formula in M7:

 

=IFERROR(B7/LOOKUP(1E+300,C7:L7),"")

mike h.. say:

Hi,


You lost me when your question moved on to macros but I believe this formula does what your require. Assuming your data start in e2 put this in F2.



=IF(E2="","",IF(ISNUMBER(MATCH(E2,SpecialistTable!A:A,0)),"Specialist",IF(ISNUMBER(MATCH(E2,SpecialistTable!B:B,0)),"Specialist 2","Other")))



You can drag it down as far as you want, it will return blanks until something is entered in the corresponding row of Col E.



Note I used full columns in the formula but it's better if instead of that you use A1:a100 or however many rows you need. it will speed things up.

teylyn say:
You can use a Sumif() function:

=Sumif(A1:A1000,"magazine 180ac",D1:D1000)

If there are different spellings of the same magazine, you need to clean up the data first.

Also, you can build a pivot table with just a few clicks that can sum up the data in many different ways. Creating pivot tables is very different between early and current versions of Excel, though. The Excel help files are useful and there are lots of tutorials about pivot tables online. Try this for a start:

http://peltiertech.com/Excel/Pivots/pivotstart.htm

cheers, teylyn
