As for the Header issue - as Daniel asked, it isn't clear what you mean by the term "space line". Empty paragraphs are the most inappropriate means of creating vertical space. for additional space between the header & the first ine of text on each page you can do either of the following:

daniel g. grau say:
The first appears to be well known - Why is it that when selecting UK/British English as my default language word refuses to instigate it and instead keeps US English as the default. I have seen numerous people reporting this bug but nothing appears to have been done!

That's because none of them understand the nature of the beast.

Language is not a setting or preference, but a text attribute, just like bold, font and color. Therefore, you don't choose the working language, you need to apply it to selected text. You can also define it as part of a style, as with bold.
  • If the text in your document is defined as "U.S. English," selecting "U.K. English" in Tools > Language will apply the language change at the cursor's position. Instead, select the text and apply the language, just as if you wanted to make it bold.
  • There is also an additional language attribute: "Do not check spelling or grammar," which is applied in addition to the language. This shows as a checkmark in the Language dialog. Usually, it should remain unchecked.
  • Both of the above attributes (language and check/do not check) can be made permanent by pressing the Default button to store them in the Normal.dotm template. From then on, all your new documents will follow the model.
Verify if the current document's Normal paragraph style (not to be confused with the template of the same name) or any other style has the wrong language attribute. To do this:
  • In the main menu bar, select Format > Style
  • In the list, click on Normal, then on the Modify button
  • Click on the dropdown menu at the bottom left
  • Select Language
  • Define the proper language for the Normal style

It is possible for a Normal template and its Normal paragraph style (and all other styles as well) to have the proper language attributes, yet to have text in another language, because it was manually applied. If this is the case, you'll need to open Normal.dotm and modify it. (Post back for instructions.)

riahorn say:

creating series of invoices from template

jllatham say:

You haven't given us much to go on.  What about the content of the invoices (services, products, quantities, prices, extended prices, etc) and such as that? 

For simply printing blank invoices with basic client information entered to be filled out by hand you could use a small macro that might look something like this.

Sub PrintBlankInvoices()
  'change these Const values to point to the
  'worksheet with the client list/information
  Const listSheetName = "Sheet1"
  Const clientNameCol = "A"
  Const clientStreetCol = "B"
  Const clientCityCol = "C"
  Const clientStateCol = "D"
  Const clientPostalCodeCol = "E"
  Const firstClientEntryRow = 2 ' assumes row 1 has labels
 
  'now do same for your invoice sheet, but need
  'specific cell addresses for the data entries
  Const invoiceSheetName = "InvoiceTemplate"
  Const clientNameCell = "A1"
  Const clientStreetCell = "A2"
  Const clientCityStZipCell = "A3"
 
  'working variables
  Dim listWS As Worksheet
  Dim invoiceWS As Worksheet
  Dim lastRow As Long
  Dim namesList As Range
  Dim anyName As Range
 
  Set listWS = ThisWorkbook.Worksheets(listSheetName)
  lastRow = listWS.Range(clientNameCol & Rows.Count).End(xlUp).Row
  If lastRow < firstClientEntryRow Then
    MsgBox "No client names entered. Quitting", vbOKOnly + vbExclamation, "No Data"
    Set listWS = Nothing
    Exit Sub
  End If
  Set namesList = listWS.Range(clientNameCol & firstClientEntryRow & ":" _
   & clientNameCol & lastRow)
  Set invoiceWS = ThisWorkbook.Worksheets(invoiceSheetName)
  For Each anyName In namesList
    If Not IsEmpty(anyName) Then
      With listWS
        invoiceWS.Range(clientNameCell) = anyName
        invoiceWS.Range(clientStreetCell) = .Range(clientStreetCol & anyName.Row)
        invoiceWS.Range(clientCityStZipCell) = _
         .Range(clientCityCol & anyName.Row) & ", " _
         & .Range(clientStateCol & anyName.Row) & " " _
         & .Range(clientPostalCodeCol & anyName.Row)
      End With
      'printout the invoice sheet with basic info
      invoiceWS.PrintOut copies:=1
    End If
  Next
End Sub
