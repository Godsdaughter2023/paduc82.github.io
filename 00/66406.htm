e. Uncheck Internet Protocol Version 6 (TCP/IPv6).

f. Click on Ok.

a. From the Start button, type CMD, right click on CMD and select Run as administrator.

netsh interface ipv4 reset press Enter.

netsh interface ipv6 reset press Enter.

amrita m say:

Hi FredSmith

 

Welcome to Microsoft Answers!

 

 

There is no direct way to print the content of a drive in Windows, you may try the following steps that prints the contents of a folder.
You may try printing a folder from Dos.

1.      Click on Start

2.      In the Start search bar type cmd and hit enter.

3.      Go to your folder entering CD \(folder name)

4.      Then create a file by DIR >file.txt

5.      EXIT to close the Dos window

6.      Open file.txt in Notepad

7.      Print it as usual (from Notepad)

 

There are other 3 party software’s that you may use to print the drive contents.
