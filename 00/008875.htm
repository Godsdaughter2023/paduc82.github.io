How to show 2 charts on single chart? - Microsoft Community

oemsoft say:

How to show 2 charts on single chart?

I would like to show 2 charts on a single chart,

for example,

Under column A, there is a list of date

Under column B, there is a list of numbers

Under column C, there is a list of numbers

I would like to show the date on x-axis, the numbers on column B on the left and the numbers on column C on the right.

Does anyone have any suggestions on how to do it in excel?

Thanks in advance for any suggestions

rich007a say:

Dificult to tell from your screen shot, but I'll offer the following observations:

1) You are plotting a very large number of data points.  Ploting these as a column chart will result in very small bars, so it actually looks like a line.  You have vertually the same effect by keeping both as series as X-Yscatter type.

2) Check the formula of the two series to make sure they are referencing the correct data.  Click on the second series.  Looking at the formula (if you know what you are looking at), you might find that the x-axis values is starting at row 1, but it should start at row 2.  If you can't decifer the formula, right click on the chart and pick Source Data.  Click on the Series tab.  Check that the x-values and y-values start in row 2  (=Sheet1!$A$2:$A$....), etc.  Problems can arise when you do not present Excel with "good" data, which should not have anything in the cell at the top of the x-axis values (cell A1 in your case).

3) And I think this is the key to your "error on x-axis " issue, because you are now mixing chart types in one chart, the two chart types you have picked (XYscatter and column) have different types of axes.  The XY scatter series is plotted against an xlValue axis type, the column chart is plotted against an xlCategory axis type.  Even though you can't see it, when you plot two series on two different y-axes, and when they are different axis types, they have different settings for min, max and units.  This excellent page explains the difference: http://peltiertech.com/Excel/ChartsHowTo/CatVsValueAxis.html

There are two ways to fix it.  The first is the easiest, but has one important issue.  DO NOT use this if your x-axis data is not evenly spaced.  In your case it is, the x-axis values increase by 1 for each step, but if they increased by unequal amounts your chart would not plot right.  So, because your x data is evenly spaced, you can simply change the XYscatter series to be a Line chart series.  The line chart also plots on a category x axis, like the column chart, so it probably all looks good now.

The second way to fix this is more complicated, and there isn't much point in me typing it in this case, since you have far too many data-points to be able to plot a sensible column chart!  The columns will be so narrow, you wont be able to see that single one column that is has a value of 1.

What are you really trying to do? If you just want to plot a vertical line on the chart to mark a particular event, you should either draw a line on the chart, or do the following:

In cell E1 type 40750   (or whichever x value you want the line to appear)
In cell E2 type 40750

In cell F1 type 0
In cell F2 type 1  (if you want the line to appear on it's own axis, otherwise, type the max value of the left x-axis here.

Select the 4 cells E1:F2, copy.

Go to your chart and go Edit > Paste Special.  In the Paste Special windo that pops up, choose New Series, Columns and check "Categories (X Values) in First Column".

If you put 1 in F2, move that series (you probably wont be able to see it) to the secondary y axis.

Done.

Cheers
Rich

Mail going to wrong folder:

Most likely it's a Mailing List Manager rule that is moving your messages. It's easy to accidentally create a Mailing List Manager rule when marking a message as junk. Delete the Mailing List Manager rule.

RULE BASICS

There are 3 different kind of rules:

Mailing List Manager (MLM)
Junk Mail Filter
Rules (custom rules you create)

They run in the order listed above. The custom rules also run in the order they are listed. (top runs first)

1) POP Rules act on a message ONLY when it first arrives in the Inbox (On My Computer), NOT when it arrives in any other folder. Once it's moved to another folder it won't be seen by rules.

2) IMAP Rules are meant to act on a message ONLY when it first arrives in the IMAP INBOX, not when moved to another IMAP folder.

3) Exchange Rules act on a message ONLY when it first arrives in the Exchange Inbox.

Each type of account has it's own rules. Click on the correct tab to check your rules. POP, IMAP, Exchange, Outgoing
