Hi

The Clipboard viewer (clpbrd.exe) was removed in Windows beginning with Windows Vista. There is no longer a UI (User Interface) for the clipboard.

When you copy something, (Ctrl+C) it is still placed in the Clipboard (Clip.exe) and is removed the next time you copy something. You can still use the Keyboard Shortcuts (Ctrl+C to Copy and Ctrl+V to Paste).

There are a couple of Command Line options. You can see these by opening a Command Prompt and typing  clip.exe /? and pressing Enter.

If you type a very large amount of data to the clipboard, you may see a pop up that there is a large amount of data on the clipboard and would you like to Remove this data.

Microsoft Office has it's own, separate Clipboard.

Copy and paste multiple items by using the Office Clipboard:

https://support.office.com/en-US/Article/Copy-and-paste-multiple-items-by-using-the-Office-Clipboard-714a72af-1ad4-450f-8708-c2931e73ec8a?ui=en-US&rs=en-US&ad=US#bm2

If you wish, you can also create a Shortcut to empty the clipboard.

1. Right Click the Desktop and select New/Shortcut.

2. In the Location box, copy/paste the following command.

cmd /c "echo off | clip"

3. Click Next and name the shortcut Clear Clipboard.

4. Click Finish.

Double Click the shortcut to clear the clipboard.

There are many, free, third party applications that can be used to manage the clipboard.

Let us know if you have more questions.

         set DEVMGR_SHOW_NONPRESENT_DEVICES=1
