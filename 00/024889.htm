It is not the modules that are at issue but the macros themselves. Consider the following. Copy the macros to a standard module

The first macro is run from the rule, and calls the second and/or third. Select a message and run the test macro at the end. You should see the two messages. Create suitable conditions and you will see one or other message.

Option Explicit

Sub RuleScriptMacro(item As Outlook.MailItem)
    'If condition A then
        Macro1Name item
    'End If
    'If condition B then
        Macro2Name item
    'End If
End Sub

Sub Macro1Name(item As Outlook.MailItem)
'Do stuff e.g.
    MsgBox "This is macro1"
End Sub

Sub Macro2Name(item As Outlook.MailItem)
'Do other stuff e.g.
    MsgBox "This is macro2"
End Sub

Dim olMsg As MailItem

    Set olMsg = ActiveExplorer.Selection.item(1)
    RuleScriptMacro olMsg
lbl_Exit:
    Exit Sub
End Sub
