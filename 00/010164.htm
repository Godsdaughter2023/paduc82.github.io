Also, if your computer came from a computer manufacturer in the last few years, it probably has a "restore" partition of about 10 or 12 GB, likely hidden, the function of which is supposed to be to allow you to restore it to the garbage-loaded state that it was when it came from the factory. That image type backup, and the related bootable rescue CD will be your means of returning to Win 7, in case you need to.

Once you have that partition image made and saved, you will need to find a partition manager program to install and use to delete the "restore partition" ... not the "system partition", which is much smaller than the "restore partition".

After the above steps are completed, you should be ready to use the Media Creator Tool, selecting "upgrade this PC now". You will not need to enter any key, it will use your activated Win 7 for that.

monique tang msft say:

Hi Ari-Piroozi,

The following steps are for your reference:

1. Empty Auto Complete list. Navigate to File > Options> Mail> Send messages> click Empty Auto-Complete List> Yes.

2. To identify if the issue is related to Add-ins, please start Outlook in safe mode to see if the issue persists. (Click Start > Run> type outlook.exe /safe> click OK.)

If it works, try disabling add-ins to see if it works:

Click File> Options> Add-ins> Select “COM Add-ins” in the Manage drop-down> click Go> Uncheck any add-ins that you are not using> click OK.

3. Please try recreating the Outlook profiles to see if it works.

4. What’s the detailed version of your Outlook client? Please make sure it’s the latest version on your side by navigating to File> Office Account> Update Options> Update Now.

5. If the issue persists, please try the troubleshooting steps in the following article: http://support.microsoft.com/kb/2682333

Note: It applys to Outlook 2013 too.

Regards,

Monique
