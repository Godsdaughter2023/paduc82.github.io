jimwynne say:

You can't change the content or format of a file just by changing or adding an extension.  If, for example, you have a native PDF file and give it an extension other than .pdf,  it won't open because the target program won't know what to do with it.   So if you have file.pdf, which is an actual PDF file, and you change it to file.doc (using the extension for Word files), Word (the program) won't be able to open it. 

The purpose of the extension is to tell the operating system the type of file being opened, so the OS knows what program to use.  If the extension is .pdf, the file will be opened by the default PDF viewer (often Adobe Acrobat Reader).  If the extension is .doc or .docx,  it will be opened in Microsoft Word if that program is installed.
