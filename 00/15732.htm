hansv mvp say:

You can find the values of symbolic constants as follows:

Press Ctrl+G to activate the Immediate window.

Type (for example)

 

? acFormatXLS

 

and press Enter. You'll find that the acFormat constants are strings, not integers:

 

acFormatXLS = "Microsoft Excel (*.xls)"

 

and

 

acFormatPDF = "PDF Format (*.pdf)"

 

There is no constant acFormatXML but there is a constant

 

acFormatHTML = "HTML (*.html)".

 

To export to XML, use the ExportXML method - see http://msdn.microsoft.com/en-us/library/bb237912(v=office.12).aspx
