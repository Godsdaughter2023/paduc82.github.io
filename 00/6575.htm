Microsoft Office Upload Center now gives you a way to see the state of files you're uploading to a server, in one location. When you upload a file to a web server, Microsoft first saves that file locally to the Office Document Cache before it starts the upload, which means that you can save changes and immediately continue working even when you are offline or have a bad network connection.

If you remove or disable you will not be able to sync it.

The Office Upload Center has great functionality related to synchronizing Office documents in a very granular fashion. It allows multiple people to edit the same document and sync in the various changes.

This is usually used when you have Office files saved on OneDrive and then trying to sync it with locally saved files.

You can sign out of the Office application and then this would not initiate the Office upload center.

Highlight the result of Vlookup through conditional formatting - Microsoft Community

sth_abt_jnd say:

Highlight the result of Vlookup through conditional formatting

How can I highlight the cell which have the data that vlookup search for. For example If cell A1 have the result it be highlighted. 

jacques raubenheimer say:

You don't give a lot of detail about the layout of your worksheet.

That will affect the quality of the responses we give.

For example, there would be no sense in cell A1 actually containing the data returned by the lookup, since a Vlookup searches in one column, and returns a value from another column to the right (and there is nothing left of column A against which we could search). Yes, you could search for something in column A and return column A, but all that does is tell you whether an item is in a list or not.

Nonetheless, assuming your VLOOKUP is in cell C1, you could use this:

Select A1:A10 (I am assuming that your VLookup searches in Column A and returns column A, and that you have a list of 10 items against which you are searching--I have to make assumptions like this since you really did not give us much to work with).

Select A1:A10, making sure that A1 is the active cell (after selction, the name box should display the value A1). The select Conditional Formatting, and select New Rule.

Then select 'use a formula to determine which cells to format'

Use this formula

=A1=$C$1

and add the highlighted formatting of your choice.
