If you're using SUBTOTAL then I'm assuming that you have a filter applied?

If that's the case...

Let's assume this is your data in the range A3:B21 -

A 96
B TEXT
A 30
B #REF!
A 91
B 29
B 23
B #N/A
C 76
B 90
A #REF!
C 13
B 39
C 74
A #NUM!
A 58
C #REF!
A 43
A #VALUE!

You filter on column A and want the average of column B.

Try this array formula** :

=AVERAGE(IF(SUBTOTAL(2,OFFSET(B3:B21,ROW(B3:B21)-ROW(B3),0,1)),B3:B21))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

bill mosca say: You're kinda stuck with it since that is actually a system message. You would have to use VBA to export to a temporary location and then use the FileSystemObject.

Public Sub Move_File()
    Dim strSource As Variant
    Dim strDestination As String
    Dim objFS As Object
    
    strSource = "C:\Temp_C\Test.txt"
    strDestination = "C:\Temp_Test\Test.txt"
    Set objFS = CreateObject("Scripting.FileSystemObject")
    With objFS
        If .FileExists(strDestination) Then
            .DeleteFile (strDestination)
        End If
        .MoveFile strSource, strDestination
    End With
    Set objFS = Nothing
End Sub

neha singh msft say:

Hi KSR,

 

Well, it’s a simple mathematics here. In Excel, the underlying value is always stored in decimal form. So, even if you’ve used number formatting to display something as a percentage (10%), that’s just formatting, or a symbolic representation of the underlying value. Excel always performs calculations on that underlying value, which is a decimal (0.1).

 

  • Format existing values: When you apply percentage formatting to a cell that already has a number in it, Excel multiplies that number by 100 and adds the % sign at the end. Let’s say that you type 10 into cell A2, and you then apply the percentage number format. Because Excel multiples your number by 100 to show it as a percentage (remember that 1% is one part of one hundred), you’ll see 1000% displayed in the cell, not 10%.

 

  • Format empty cells: Excel behaves differently when you pre-format empty cells with percentage formatting and then enter numbers. Numbers equal to and larger than 1 are converted to percentages by default; numbers smaller than 1 that are not preceded with a zero are multiplied by 100 to convert them to percentages. For example, if you type 10 or .1 in a preformatted cell, you’ll see 10%appear in the cell.

 

  • Format as you type: If you type 10% directly in the cell, Excel will automatically apply percentage formatting.
