lz. say:

Bob,

The FREQUENCY function works as expected. The problem comes from the fact you don't use the second argument (Bins_array) appropriatly. Your formula says:

=FREQUENCY(B2:G2, B10:B15)

and you expect the results in B10:B15 as well. If you expect the results in B10:B15 the second argument cannot be B10:B15. It must be another range with a set a values to want to "count" against the values in B2:G2.

Example:

Above I selected B5:B10 and entered:

=FREQUENCY(B2:G2,B2:G2)

and validated itwith Ctrl+Shift+Enter

This counts the frequency each value B2:G2 within B2!G2. Output:

1 time a 2        (in B2)

2 times a 4      (in C2)

0 time a 4       (in D2) - 0 beacuse as it has already been "counted" above

1 time a 3       (in E2)

1 time a 5       (in F2)

1 time a 6       (in G2)

No error there. I can assure you that if you replicate that example on your PC you will get the exact same results - nothing to activate or anything like that.

I would highly recommend you also apply the example from the FREQUENCY function article and carefully read the associated comments. That array function isn't an easy one to understand I must admit

As well as onedrive, another way to get backup versions is through the Windows 10 (and 8) "File History" feature. This feature keeps your files local so it does not use your internet upload limit.

You need an external drive, a USB Thumb drive or SD card or networked HD and you have to turn on the File History feature.

Here is one article on the feature:

http://www.ghacks.net/2016/01/05/how-to-use-file-history-on-windows-10/

You can google for more instructions online.

Couple of tips:

Review the list of folders being backed up by default.  I found many that I did not want "wasting" space on my history drive. You can click on a folder in the list and select to remove it.

You can also add specific folders you want to backup.

When I looked at the folders in the list, I found some "OneDrive" folders in my Windows user profiles. When I looked in those folders they had copies of a bunch of files that I did not want there, so I did some house cleaning.

The default backup interval is 1 hour, you may want to make it less than that.

Depending on how big your backup drive is and how many files you are backing up, you may want to change how long the backup files are kept.

These procedure should take care of the issue with .NET Framework

1) Download and run the Windows Installer Cleanup Utility. Use it to remove

the .NET Framework 1.1 from the machine.

 

Windows Installer Cleanup Utility

http://support.microsoft.com/kb/290301

 

 

2) Download .NET Framework 1.1 from the Microsoft website (don't download

via Windows Update), and save it to your hard disk (file name dotnetfx.exe).

 

.NET Framework 1.1

http://www.microsoft.com/downloads/details.aspx?FamilyId=262D25E3-F589-4842-8157-034D1E7CF3A3&displaylang=en

 

 

3) Download .NET Framework 1.1 Service Pack 1, and save it to your hard disk

(file name NDP1.1sp1-KB867460-X86.exe)

 

.NET Framework 1.1 Service Pack 1

http://www.microsoft.com/downloads/details.aspx?familyid=A8F5654F-088E-40B2-BBDB-A83353618B38&displaylang=en

 

 

4) Run the downloaded dotnetfx.exe to install .NET Framework 1.1.

 

5) Reboot your machine.

 

6) Run NDP1.1sp1-KB867460-X86.exe to install .NET Framework 1.1 SP1.

 

7) Reboot your computer again.

 

8) Now run Windows Update again, to install post SP1 hotfixes.

 

Not all the reboots maybe necessary, but ... better safe than sorry.

_elaine_ say:

Hi, Tom_H_BASES

In addition to what Juan wrote, above, please note that Skype Numbers do not work like conference bridge numbers.  The New Skype application is a work in progress.  Adding callers who call the Skype Number to the conference call once the call has launched, may  or may not work, depending upon which version is used.

Best to have those few participants start new Microsoft accounts with a dedicated email address for use with calls such as these.  Ask them to remember that when they create the new account, the Skype ID will be live:whatever text appears before the @ in the new email address they set up.

This way, you could add these participants to your contact list and simply launch a conference call as a group call.  As far as I am aware, even if people miss the call start and join the call a few minutes after the call launches, they should still be able to click the in-progress call in their Skype client (software) just like in the Skype Classic and earlier versions.

So you can make an informed decision ...
