sheet metal stampers say:

Need a Macro to Encrypt All XLSX files in a folder...

I'd like to achieve this option using a macro. I need to go for a macro because, I have a lot of .xlsx files in a folder that has to be encrypted with a password. I searched all over the web and couldn't find one.

I found a macro which protects/unprotects worksheets in a folder. But, I want the recipient to open the excel file with a password.

Please Help... 

hansv mvp say:

Here you go. The macro will prompt you for the password and for the folder when you run it.

Sub EncryptWorkbooks()
    Dim strPassword As String
    Dim strFolder As String
    Dim strFile As String
    Dim wbk As Workbook
    ' Prompt user to enter password
    strPassword = InputBox("Enter the password to set")
    If strPassword = "" Then
        MsgBox "No password specified!", vbExclamation
        Exit Sub
    End If
    ' Prompt user to select folder
    With Application.FileDialog(4) ' msoFileDialogFolderPicker
        If .Show Then
            strFolder = .SelectedItems(1)
        Else
            MsgBox "No folder selected!", vbExclamation
            Exit Sub
        End If
    End With
    ' Make sure folder path ends in \
    If Right(strFolder, 1) <> "\" Then
        strFolder = strFolder & "\"
    End If
    On Error GoTo ErrHandler
    ' Reduce screen flicker
    Application.ScreenUpdating = False
    ' Get first filename
    strFile = Dir(strFolder & "*.xlsx")
    ' Loop through files
    Do While strFile <> ""
        ' Open workbook
        Set wbk = Workbooks.Open(Filename:=strFolder & strFile)
        ' Set password
        wbk.Password = strPassword
        ' Close and save workbook
        wbk.Close SaveChanges:=True
        ' Get next filename
        strFile = Dir
    Loop
ExitHandler:
    Application.ScreenUpdating = True
    Exit Sub
ErrHandler:
    MsgBox Err.Description, vbExclamation
    Resume ExitHandler
End Sub

Microsoft Lumia 950 - Microsoft Community

a. user say:

Microsoft Lumia 950

Hi everyone,

Does the Lumia 950 still have the restarting issue even after the latest creators update.

flying kiwi say:

No, not at all (that is unless you want to restart it and press the button to shut it down).

My 950 hasn't shut down or restarted other than when I wanted it to ever... and I've had it some time now. I do restart it every few days anyway though so maybe that's important. After the latest update it'd be worth performing a backup and then a hard reset on the device because although that'll only reinstall the same code on the device, sometimes such action will resolve pre-existing issues. I'd suggest testing it out without using the recovery at the end of the hard reset (so you wont get all your old setting etc) but the device will be set back to the most basic configuration possible. If it performs well like this then after some time, you could hard reset it again and on the second part of the reset choose to use the recovery files offered. Then test again and see if that makes any difference - if there is a difference then this should help to point you in the direction of what's causing the problem.
