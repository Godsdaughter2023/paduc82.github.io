In Windows Media Player, you can play audio CDs, data CDs, and data DVDs that contain music or video files. Typically, the disc will start playing automatically. If it doesn't play, or if you want to play a disc that is already inserted, open Windows Media Player, and then, in the Player Library, select the disc name in the navigation pane.

If that doesn’t help, kindly check with the following methods and see if it works for you.

Method 1: Check the DVD/CD driver.

When the player supporting for DVD doesn’t have the compatible driver, you might face such issues. So it’s better to check if the driver is updated with the latest version. To do that,

  • Right click on the Start menu and select Device Manager.

  • Narrow down to DVD/CD-ROM drives and expand it.

  • If your CD/DVD drive shows up with yellow exclamation mark or a red X sign on the driver, we need to reinstall or update the driver.

  • Else, we can right click on the CD driver and click on Update driver to get the latest build.

However, if the issue persists, install the driver in compatibility mode and check if it helps. To do that,

  • Right click on the driver installation file and select Properties. In there, click on the Compatibility tab.

  • Put a check mark on Run this program in compatibility mode.

  • Click the dropdown box and select a previous Operating System and click on OK.

  • Install the driver and check if the issue persists.  

You may also refer to make older programs compatible with this version of Windows for further information.

Method 2: Uninstall and re-install Windows Media Player.

  • Go to Control Panel from the start menu and select All Control Panel Items > Programs and Features.

  • Click on Turn Windows Features On or off and narrow down to Media Features.

  • Uncheck the box Windows Media Player and reboot the machine.

  • Once after the reboot, get back to Control Panel and check the same box we unchecked in the previous step. Reboot the PC and try to play the CD.  

Additional Information: For more information about the version of Windows Media Player, you may refer the article Get Windows Media Player.
