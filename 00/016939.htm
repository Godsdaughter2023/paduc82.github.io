That may document how it is supposed to work, e.g. how it used to work in XP, or perhaps how it works without Protected Mode: On but I would agree with the poster that what I see in W7 or Vista when I use that is less than I expect.

In order to see the desired full list in W7 or Vista users of those OS would first have to understand that most of their browsing is done into a new  "Low" version of the TIF and anything in it (and anything in the new "Low" Cookies directory) is now ignored by the TIF viewer.   Therefore, they may need to enter those directories in a cmd window and do some dir commands there.   Unfortunately that only lists the names of the cached files and not their URLs or any supplemental information about their URLs.  That information is only available in the appropriate index.dat.   Somebody probably should write a PowerShell script which would re-integrate all the missing information with the data that the TIF viewer does manage to show and present it in a window having a similar UI.

Tip:  a quick way to enter the appropriate directory is to use the Explorer window that the documented procedure generates.  E.g. capture the path from that window's Address bar with Alt-d, Ctrl-c; then switch to a cmd window and start typing a chdir command:  cd /d  (plus a space); then right-click Paste and press Enter.   Then you can find all your last accesses in order by entering:  dir/a/ta/s  |  sort  |  more   As I mentioned without their URLs that list has limited usefulness but at least then you can do finds in the all the index.dat  to help you puzzle some out.   BTW the sort stage of that command pipeline is most useful when the user customizes their short date and time format to be  YYYY-MM-DD  HH:MM:SS.

jim gordon mvp say:

Fixy is correct that you have not updated Office to current updates. Fixing permissions is nice, but has nothing at all to do with you not having installed updates.

If you have an Office 365 subscription, after updating you will be able to customize both the Quick Access Toolbar (QAT) and the Ribbon. If you have a stand-alone version you will have the ability to customize only the QAT.

The first thing to do is install the recent updates. Make sure your Mac has a working internet connection, then:

 

Update Mac OS X. Open the App store and click on the Updates button. Install the updates.

 

After, in any Office application go to the Help menu and choose Check for Updates to launch the AutoUpdate app. Let AutoUpdate install updates and then restart your Mac.
