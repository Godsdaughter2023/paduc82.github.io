The basic problem seems to be that the new map loader puts the data in a directory called "cities" on the memory card, whereas previously it was stored in "private". Having the data in both places seems to upset Nokia Maps.

  Sub BackToWhenceYouCame()
    On Error GoTo skip
    CommandBars("Web").Controls("Back").Execute
    skip:
  End Sub

I was able to add MacroButtons calling this macro (Using the Field option on the Insert menu) wherever I wanted the reader to be able to jump back to the previous position.

Private Sub Form_Load()

Dim MyRandomNumber As Integer
'count number of records
recCount = DCount("*", "tblQuotes")

'open form to specific record
DoCmd.GotoRecord acDataForm, "frmMainMenu", acGoTo, MyRandomNumber

End Sub

I don't believe that is quite right.  In this case the upper bound of the range you want is the number of records, or DCount("*", "tblQuotes"), which you have correct, but the lower bound is not 0, but 1 -- you want to pick a record in the range from 1 to DCount("*", "tblQuotes"), and a record number of 0 would not be valid.  Therefore, given recCount = DCount("*", "tblQuotes"), the formula

    Int((upperbound - lowerbound + 1) * Rnd + lowerbound)

becomes

    Int((recCount - 1 + 1) * Rnd + 1)

which resolves to this:

    Int((recCount * Rnd) + 1)
