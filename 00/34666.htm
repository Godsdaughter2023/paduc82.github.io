craiglong say:

This is a copy and paste from a reply I did earlier.  Hopefully it will help.  Let me know how it goes.

First type in the Start menu the word “homegroup”. Go there and make sure your computers are connected to the Homegroup. 

There is a Homegroup password that should be the same in both computers. Go through all those options for file sharing and media streaming and enable most if not all of them. In the “Change Advanced sharing settings” be sure to turn on “Network Discovery” and enable the other sharing options.

Click the Start menu and type in the word “sync”. Go to your PC sync settings and there you can sync all or most of your PC settings.

Next go to the folders that you wish to share, right click, and select “share with Homegroup”. In the same right click context menu go to “Properties” at the very bottom, click and go to the “Sharing” and “Advanced Sharing” tabs and set sharing permissions for the folders.

On your computers go to “Device Manager” to “Network Adapters”. Expand the drop-down menu. Right click the network adapters -> “Properties” -> “Power Management” and check all boxes so that you will be able to wake the computers remotely. When you need to access a remote file Windows might say that the remote PC is asleep and ask for your permission to wake it. The “Movies & TV” app has a “Media servers” tab which will wake the remote computer in order to get to the files it needs. That’s “Media Streaming” from a “Media Server”.

In Windows File Explorer you should now see the word “network” in the menu below the list of files and libraries at the left. Expand “network” and the PCs on your Homegroup should show up with all the files that you have shared. Those files and folders that you see under “Network” are the one’s you can add to your apps such as Movies & TV. Now one computer can serve media and files to the whole house.
