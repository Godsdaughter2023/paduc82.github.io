 THere are apparently some actions which do change the modified date, since it is different than the create date ... but a mystery to me.

 

 What does the "modified date" really mean ?  When is it updated ? 

hansv mvp say:

The last modified date of a table is the last time the design of the table was changed and saved. It has nothing to do with the data in the table. If you want to keep track of when a record in the table was last updated, you have to do this in the form(s) used to enter and edit data in the table. Add a date/time field LastModifiedDate to the table, and create an event procedure for the Before Update event of a form based on the table:

 

Private Sub Form_BeforeUpdate(Cancel As Integer)
    Me.LastModifiedDate = Now
End Sub
