This is a workaround that I have tested succesfully.

Install Chrome Browser (If not already installed)
Make Chrome the default browser in the popup.
Copy iexplorer.exe from C:\Program Files (x86)\Internet Explorer to C:\Program Files (x86)\Google\Chrome\Application
Rename chrome.exe to something else. (Chrome still run from the renamed file if required)
Then rename the copied iexplorer.exe to chrome.exe.
The links from Email are now fooled into opening Internet Explorer instead of Chrome with no popup.
This works even after a reboot.

Hello Everyone,

Using the new Skype for Desktop (Windows, Mac and Linux), the option to delete a conversation is per contact.

When you delete a chat, all the messages in the conversation are deleted, and it will no longer show in your recent chats. The chat will be deleted from that specific device, but will still be visible on other devices unless you delete it from there too. See this article: How do I delete a chat in the new Skype on desktop?

The conversation history (where sent and/or received messages, calls and data are located) are saved in the cloud for a period of time (last 30 days). Any history older than 30 days will be saved in the device's local hard drive depending on your history settings.

 

Your saved conversation from the classic Skype for Windows Desktop will not reflect on the new Skype on Desktop, Mac and Linux (which you are currently using), although you will be able to access those that are saved on the cloud.

 

Moving forward, in your conversations, as well details of any calls you've had on Skype are stored in the cloud for 2 years.
