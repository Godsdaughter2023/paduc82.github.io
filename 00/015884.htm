roady [mvp] say:

Right click on your Calendar and choose Properties and then Permissions.
Other users can then access it via: File-> Open-> Other User's Folder


If you want them to map to your mailbox, then they need to be able to "walk" to your folder. This means that you also must grant them at least "Folder Visible" permissions on the root of your mailbox (not your Inbox folder!). If you don't grant them any other permissions, then they can't read anything else, besides your Calendar.

simon_g---01 say:

the problem is a Windows error, which involves a number of steps to fix

This is what finally resolved the problem:

 

1. Download and install 7-Zip application (or other Zip app)

2. Use 7-Zip to extract the contents of NokiaSoftwareUpdaterSetup_en.exe into a folder on your computer.

3. Inside that folder, locate the file Packages\NokiaSoftwareUpdater\Setup\NokiaSoftwareUpdater.msi

4. Double-click on that file to start the installation.
 If the issue is still present, please try the following steps:

1. Make sure you have administrator rights on the machine
2. Open Registry (Start->Run->type regedit in Open field and click Ok)
3. Take a back up of the registry (Highlight My computer->File->Export)
4. Navigate to HK_CURRENT_USER\Software/PCSuite and delete the complete folder
5. Navigate to HK_CURRENT_USER\Software/Nokia and delete the complete folder
6. Navigate to HKEY_LOCAL_MACHINE/Software/PCSuite and delete the complete folder
7. Navigate to HKEY_LOCAL_MACHINE/Software/Nokia and delete the complete folder
8. Uninstall Nokia PC Suite
9. In Windows navigate to Start > Settings > Control Panel > Add or Remove Programs
10. Uninstall Nokia Connectivity Cable driver
11. Delete the following folders:
• C:\Documents and Settings\%username%\Application Data\PC Suite\ConfServer
• C:\Documents and Settings\%username%\Application DataNokia\ContentCopier
• C:\Documents and Settings\%username%\Application Data\Nokia\PCSync\SynchData
• C:\Documents and Settings\%username%\Application Data\Datalayer
• C:\Documents and Settings\%username%\Phone Browser (hidden folders)
12. Update Windows Installer by downloading the latest version from Microsoft
13. Use Windows Installer CleanUp Utility
14. Reinstall Nokia Ovi suite Suite
