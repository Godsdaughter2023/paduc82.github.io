There is no longer is anything comparable in Excel.

The alternative is to use the Selection Pane which can be displayed by clicking its button in the Arrange group of the contextual [object type] Format tab that appears when an object is selected. The Selection Pane command also can be added to the QAT.

  • Give a meaningful name to each object
