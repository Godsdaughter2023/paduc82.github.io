chris_smith say:

For the knowledge of all others who may run into this issue, this is how I ended up resolving it:

  1. In Device Manager, disable the DVD drive (or whatever device has the letter you need). This frees up (in my case) drive D.
  2. Open File Explorer. Double click your computer. Highlight the drive that you want to change. Click the Open Settings icon at the top. This will open the Computer Management window. Highlight the Disk Management option under Storage (It's all in the left pane). Highlight the drive you want to change, right click, select the 'Change drive letter or path'.
  3. Select your new letter.
  4. Close Computer Management.
  5. Back in Device Manager, re-enable the device you disabled in step 1.
  6. Optionally you can now re-boot.

And you're back.

To decode 0x80070522. The 0x means it's a hexadecimal number, the 8 means error, the first 7 means it a windows error, and the rest of the number, 20, is the actual Windows error.
 
To look up the error we need it in decimal format. Start Calculator (Start - All Programs - Accessories - Calculator) and choose View menu - Scientific, then View menu - Hex. Enter 20. Then View menu -  Decimal. It will say 32.
 
Start a Command Prompt (Start - All Programs - Accessories - Command Prompt) and type
 
net helpmsg 32
 
and it will say
 
The process cannot access the file because it is being used by another process.
 
Use Notepad to search for FATAL and 80070020 in C:\Windows\Windowsupdate.log.
