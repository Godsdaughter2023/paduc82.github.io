Parental Controls and Battery Saver - Microsoft Community

daniel strommen [msft] say:

Parental Controls and Battery Saver

Technical Level : Intermediate

Summary

In Windows 10, when Battery Saver is enabled, you will be unable to change parental controls settings online, including granting time requests via email.  Here's what can happen and the choices you can make to limit usage.


Details

In Windows 10, when Battery Saver is enabled, you will be unable to change parental controls settings online, including granting time requests via email.  Here's what can happen and the choices you can make to limit usage.

As a specific example, if you use Time Limits with your kids, and you go to account.microsoft.com/family to approve more time, or to change their curfew for the day, those changes will not take effect while Battery Saver is enabled.  You can still grant more screen time directly on their device, but if you are trying to put your kid into "Tech Time-Out" through the website, they may be able to use Battery Saver to keep using the device until their regular daily curfew.

We are investigating how we can improve this for a future version of Windows 10, but if you're experiencing this today, there are things you can do.

If you are concerned that your kids are using Battery Saver to circumvent parental controls, because you see them using their device beyond the normal time limits, there is a way to prevent this from happening.

Changing these settings may decrease your child's device's battery life.  We recommend that you do not make these changes unless you are confident that they are using Battery Saver to evade parental controls.

1. Sign in to your child's device as an Administrator.
2. Press the Windows key and type "group policy".
3. Click on "Edit group policy".
4. In the navigation tree, go to Computer Configuration > Administrative Templates > System > Power Management > Energy Saver.
5. Double-click "Energy Saver Battery Threshold (on battery)".
6. Set the policy to "Enabled", and set it the value to 15.
7. Click OK and exit Local Group Policy Editor.

At this point, users on the device (even Administrators like you) will no longer be able to enable Battery Saver whenever they want.  If you want to reset this, you can go back into the Local Group Policy Editor and change "Enabled" to "Not Configured".

This behavior currently happens in all versions of Windows 10.  We will update this article if this behavior is changed in a new version.

Windows 10 Home: Local Group Policy editor is not available on Home edition, so you'll need to run one of these commands from an elevated command prompt.

To restrict Battery Saver to only run when the battery is below 15%:

reg add HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Power\PowerSettings\E69653CA-CF7F-4F05-AA73-CB833FA90AD4 /v DCSettingIndex /t REG_DWORD /d 15

To clear the restriction:

reg delete HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Power\PowerSettings\E69653CA-CF7F-4F05-AA73-CB833FA90AD4 /v DCSettingIndex

Windows 10 S: The only workaround is to plug the device in and ensure that the settings are applied correctly.

doug robbins - mvp office apps & services (word) say:
There is NO system for automatically saving documents in Word.  What there is, is a system for saving "AutoRecover" information, which is intended to allow you to recover a document if the Word application terminates in an abnormal manner.

stefan blom say:

Autorecovered files have the asd filename extension. You can search your hard drive for those files. Unsaved documents, in particular, can be found at File | Info | Manage Versions | Recover Unsaved Documents.

However, it is important to remember that the AutoRecover function was never intended to replace manual saves. You need to make regular backups of your files. For Word files, you can try Graham Mayor's add-in at http://www.gmayor.com/SaveInTwoPlacesAddIn.htm.
