how can i avoid #DIV/0 when i use average formula? - Microsoft Community

constantinos88 say:

how can i avoid #DIV/0 when i use average formula?

i have this cell having scores


9           7            6           empty cell                        =average(A 1: A4)     the result must be 7.3   (9+7+6)/3


5          8         empty        empty                          =average(b1:b4)         the result must  6.5   (5+8)/2

joeu2004 say: Constantinos wrote:
i have this cell having scores
9           7            6           empty cell        =average(A 1: A4)     the result must be 7.3   (9+7+6)/3
5          8         empty        empty              =average(b1:b4)       the result must  6.5   (5+8)/2
must ignore the empty cells!!!!

AVERAGE does ignore empty cells, as well as cells with text such as the null string ("").  To avoid a #DIV/0 error, you must be sure there is at least one number to average.  The following is one way to do that:

=IF(COUNT(A1:A4)>0,AVERAGE(A1:A4),"")

But if you are using XL2007 or later, you can write:

=IFERROR(AVERAGE(A1:A4),"")

That returns the null string if there are no numbers to average.  If you prefer zero, replace "" with 0.

[EDIT] The formula assumes that what appears to be numbers are indeed numeric, not text.  If 9, 7, 6 etc are text, it would be prudent to change them to numeric.  There are several ways to do that.  Post back if you need assistance with the conversion.

jay freedman say:
Don't try to use nested tables. The trouble you've seen so far is only the beginning, so get rid of them

Starting with the regular template table...
  • At the point where the horizontal and vertical rulers would intersect, there's a button that usually contains what looks like a bold letter L. That indicates that clicking in the ruler would insert a left-aligned tab stop. Click that button twice. The first time it changes to an upside-down T (meaning a center-aligned tab stop), and the second time to a backwards L for a right-aligned tab stop. There's more on this at http://www.word.mvps.org/FAQs/Formatting/SettingTabs.htm.
  • Move the mouse pointer above the first table column so it becomes a downward arrow, and then drag across the columns to select them all. (Don't use any of the methods that select the "whole table", as this prevents the next step from working.)
  • Click once on the horizontal ruler near the right-end marker of the first column. If you click and hold for a moment, the backwards-L character will appear where you click and you can drag it left or right to position it exactly. When you release the mouse button, the character will disappear; but when you click in any one table cell in any column, the tab stop will be there.
  • To enter your information, put in the first part; then press Ctrl+Tab (because just Tab takes you to the next cell, Ctrl+Tab is needed to insert a tab character in a table) and put in the right-aligned text on the same line. Press Return to go to the next line, and repeat as needed for that cell.

How to start the Windows Installer service when system services are not loaded - http://support.microsoft.com/kb/929135

If you run a Setup program without starting the Windows Installer service, you may receive the following error message:

The Windows Installer service could not be accessed. Contact your support personnel to verify that the windows Installer service is properly registered.

The Windows Installer service does not start if you clear the Load system services check box in the System Configuration utility. To use the Windows Installer service when system services are not loaded, you must start the service manually. To do this, follow these steps: 

1.Click Start  , right-click Computer, and then click Manage.  If you are prompted for an administrator password or for confirmation, type your password, or click Continue.

2.In the console tree, click Services and Applications, and then click Services.

3.In the details pane, right-click Windows Installer, and then click Start.
