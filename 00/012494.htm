row0:
1, name1, d1 d2 d3, 55, 10, 0.888, 0.567

name1=d1-d3= arbitrary text.
int, text, float fields separate with commas, well, how add in text only fields, double or single quotes????

please tell me for Office 2010 excel, mac 2011 office excel, winVISTA/XP:excel 2002, 2003...
rich007a say:

Hi,

Easiest to do this with a macro.  Press Alt+F11 to open the VBE, then do Insert > Module.

Paste the code below on the right, then close the VBE to go back to Excel.

 

Select the cell containing your data, then do View tab > Macros > Macros..., select "QuotesAroundText" and click Run.

 

Since you wont be able to store a macro in a .csv file, you'll either have to save the file as a Macro Enabled workbook (.xlsm) or you could store the macro in your Personal Macro Workbook so you can re-use it with several different files.  Let me know if you need help doing that.

 

Here's the code:

 

Sub QuotesAroundText()
    Dim c As Range
    For Each c In Selection
        If Not IsNumeric(c.Value) Then
            c.Value = """" & c.Value & """"
        End If
    Next c
End Sub

 

Hope that helps.
