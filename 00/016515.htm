joeu2004 say:

@HJ.... As Charles wrote, it would be helpful if you uploaded an example Excel file that demonstrates your idea to the best of your ability -- at least the first iteration.

I think your idea is:  C2:C11 has formulas of the form =B2, =B3 etc.  You want to freeze those random values in C2:C11 and sort them.  You want to capture 10 such "frozen" sets of sorted random values.  Then you want to average the 10 instances in row 2 (lowest random value), 10 instances in row 3 (next lowest random value), etc.

-----

One way to do this manually is to copy-and-paste-value B2:B11 into C2:C11.  Thus, only column B is recalculated.  Then copy-and-paste-value B2:B11 into D2:D11.  Copy-and-paste-value B2:B11 into E2:E11.  Repeat through column L (10 columns).

[ERRATA] It is not necessary to press f9.  The paste-value operation causes the "volatile" RAND() values in column B to be recalculated.

Then you can sort each of C2:C11, D2:D11 etc separately.  And in M2:M11, enter formulas of the form

=AVERAGE(C2:L2).  Enter that formula into M2 and copy down through M11.

Do you agree that that reflects the idea of what you want to do?

But you want to automate this, without the need to (press f9 and) copy-and-paste-value manually.  Right?

-----

One way to automate this is to use write a VBA subroutine ("macro").

You can do as much or as little of that process in VBA.  IMHO, it would be prudent to simply generate the columns C:L values.  I suspect that you will want to calculate more statistics than just the average.

Suppose C1 contains the number of iterations/columns (10) and D1 contains the number of rows (10) of random values starting in row 2.

Create a new worksheet and enter the values into C1 and D1.  Then right-click the worksheet tab ("Sheet1") and click View Code to open the VBA window.  Click Insert > Module, and copy-and-paste the macro below into the editing pane on the right.  Then press f5 to execute the macro.

(Or download the example Excel file "sorted rand.xls" from https://app.box.com/s/tjvq7drvoqtm2n54sc3ai15q397c5j6d .  Ignore any preview errors; just proceed to download.)

Design note:  We do not need to use =RAND() in column B.  We could use the VBA Rnd function and a VBA array of random values.  However, RAND() is a better implementation.  And your assignment might be specific to RAND().

-----

Option Explicit

Sub doit()
Dim nRows As Long, nIter As Long, i As Long
Dim rRand As Range, rIter As Range

Application.ScreenUpdating = False

' set up column B with =RAND()
nIter = Range("c1")
nRows = Range("d1")
Set rRand = Range("b2").Resize(nRows)
rRand.Formula = "=rand()"

' generate columns of sorted random values
Set rIter = Range("c2").Resize(nRows)
For i = 1 To nIter
    rRand.Copy
    rIter.PasteSpecial Paste:=xlPasteValues
    ActiveSheet.Sort.SortFields.Clear
    rIter.Sort key1:=rIter, order1:=xlAscending
    Set rIter = rIter.Offset(0, 1)
Next
Application.CutCopyMode = False
Range("a1").Select
rRand.Clear

' set up column with =AVERAGE(...)
With rIter.Offset(-1)
    .Value = "Average"
    .HorizontalAlignment = xlRight
End With
rIter.Formula = "=average(" & _

Application.ScreenUpdating = True
MsgBox "done"
End Sub

First, I would change the reference A2 to $A2 in the PERCENTILE.INC formula.  That allows the formula to be copied across multiple columns.

I would copy C2:C11 and copy it into D2:L11, creating the 10 samples.  Initially, all 10 samples are the same, reflecting the current values in B2:B11.

Then I would copy C2:C11 and paste-value back into C2:C11.  Repeat for each of D2:D11, E2:E11, etc.

Since B2:B11 is recalculated after each paste-value operation, each column will have a different set of ordered random values.

Finally, I would enter =AVERAGE(C2:L2) into M2 and copy down through M11.

Since the random values are intended to represent time, I would format C2:M11 as Custom [h]:mm:ss.000 .

I use [h] instead of h just in case, albeit unlikely, that a random value is 0.999999994212963 or more.  That can be interpreted as "23:59:59.999" + 0.0005/86400 or more, which Excel displays as 24:00:00.000, the rounded value.
