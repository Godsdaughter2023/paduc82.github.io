Automatic refresh of data between workbooks - Microsoft Community

roundandround say:

Automatic refresh of data between workbooks

Have two workbooks one has a destination cell and one has a source cell which are linked from one book to the other. 

All is well until a change is made in the source cell. 

The change does not seem to automatically refresh in the other workbook

Looking for the way to refresh the data and change it to the new calculation from the source cell and have it appear in the destination cell. 

Using version Excel 2007.

jllatham say:

With both workbooks open, the formula would look like this (linking to cell B2 on Sheet1 of A.xls)

=[A.xls]Sheet1!$B$2
although it could be written using relative addressing as =[A.xls]Sheet1!B2

If the other workbook's name or worksheet has spaces in either one, you need to enclose the book and sheet name with single quote marks, like:

='[Another Workbook.xls]Sheet1'!$B$2
or
='[Another Workbook.xls]Sheet1'!B2

When the source workbook is closed, Excel will automatically include the full path within the link (or if it was not open when you create the link you have to type it all in) and it would look something like:

=[A.xls]Sheet1!$B$2
or
='Q:\_TestFolder\[Another Workbook.xls]Sheet1'!$B$2

The easiest way to set them up properly is to let Excel do the work for you!  With both workbooks open, simply start a formula in the destination workbook by typing in the = symbol, then
Select the other workbook (from the View tab, Window group or clicking on the proper icon in the system tray), then
Select the proper sheet in the source workbook, and finally
Select the proper cell in the source workbook, and
Press the [Enter] key.

Excel will build the formula for you with proper syntax.

Excel will build the formula with absolute addressing (the $B$2 format for the cell address).  And if you wish to set up links to several cells in the source workbook, say from column B, you could edit the formula to relative addressing (the B2 format) and then fill it down the sheet or across or both and the cell address will update automatically for you.

Another way to copy large groups of cells is to again have both workbooks open and go to the source workbook/worksheet and select the cells you want to have links to in the source workbook and use Edit/Copy with them selected.  Then return to your destination workbook, click in the first (upper most/left most or upper-left most for rectangle area) where you want the links set up and use Edit --> Paste Special and choose the Paste Links option.

I hope this helps get things moving along.

graham mandeno say:

You will have received the message about enabling content because the Trust Center settings are different on the two computers.  You can add the folder containing your ACCDB/ACCDE as a Trusted Location to avoid the message.

It is generally advisable to develop on the lowest Access version on which the database will be running, and create the MDE/ACCDE using the version on the destination computer.  This is because the DBE/ACCDE is "compiled" against the libraries of the version with which it is created, and these are not always completely compatible with another version - even a later version.

BobWald

bobwald say:

I have found the answer to my own question

Note to self - The junction table shows which record in Table1 is related to which record in Table2.  This means that the Junction Table only updates when a record in Table1 is accompanied by a record in Table2.

In the case of an append query Table1 gets a new record in the absence of date entry into Table2.  The Junction Table will update immediately when the One-to-Many, Parent-Child, data entry form is used to add a record to Table2.

I hope this helps others that may have encountered this issue

BobWald

Hi Talor,

Based on your requirements, if you have Stream for Office 365 license (if you don’t have it, you may contact your IT admin to assign),  I suggest you use Stream in Office 365 portal to upload videos and then embed the videos in your team site. Here are the steps:

1. Click Stream in Office 365 portal > upward arrow in the upper right corner > drag files, or select files to upload > set settings (Permissions and so on) > Options > Publish now.
