a. user say:

Finally I was able to replace the files, here's what I did:

  1. Replace the files in the DLL cache.
  2. In the System32 folder, rename shlwapi.dll and uxtheme.dll to shlwapi.bak and uxtheme.bak, respectively.
  3. Move the new files to the System32 folder.
  4. Reboot the computer because the .bak files were still locked.
  5. Delete the .bak files.

For some reason, Windows still gave me the old version information even for the new files after step 3, which mislead me into thinking that it didn't work at first. However, rebooting the computer in step 4 fixed that too.

Method 2:

Remove and reinstall all USB controllers.

  1. Open Device Manager by pressing Windows key + X, and clicking Device Manager from the menu. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
  2. In the list of hardware categories, find and expand Universal Serial Bus controllers.
  3. Right-click every device under the Universal Serial Bus controllers node, and then click Uninstall to remove them one at a time.
  4. Restart the computer, and let the USB controllers get reinstalled.
