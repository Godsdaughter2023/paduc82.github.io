Workaround for Data Loss issues (Fast Ring)?

Hello windows insiders, just wanted to let you know that there is also a work around available to those who are in the fast ring, that I discovered since I first had the data loss issues, you just need to simply create a new document library in the C:/users/yourusername by right clicking and go to new and then choose folder. I would advise calling it docs instead of documents to avoid confusion. Then right click that folder and choose the include in library dropdown and choose create new library.

hope this helps any people who have issues with files being lost when logging back into windows.

please note that this is just a work around and does not fix the issue completely as it doesn't restore any documents lost previous to the updates. more than likely your files will be gone forever as windows will have overwritten your files with the update files needed to run windows 10 insider preview updates.

Update: it appears some of you have misunderstood what this workaround is actually for. Let me clarify that for you. When windows updates files are deleted from your documents folder as if nothing was there, meaning that next time you update you will continue to lose more documents with each update as your new documents will be deleted too. This workaround prevents new documents from being deleted by this bug as the new file library you create is not affected by the bug. Hope this helps you to understand.
