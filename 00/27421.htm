Hello Dannoh:

Thank you for posting to the forum regarding the issues you are having with your Windows Phone.  Issues like this can be related to the phones hardware, firmware, software or apps on the phone.  Below you will find a few things to consider.

First, I would suggest trying a soft reset of the phone.  This process will clear out cached storage and power and tends to lead to better phone performance.  You can do this by going to settings > storage sense > phone > temporary files > delete > Internet Explorer options > delete history.  Once you have done this, press and hold the down volume button + the power button until the phone shuts off and vibrates.  After the phone reboots, please go to settings > emails + accounts > tap and hold the Microsoft account listed and select the sync option.

The next thing I would consider doing is to start removing apps from the phone.  This process may take some time but would be the best way to determine if this behavior is caused by an app on the phone.  If you do not want to remove each one individually you might consider going to the next step.

If the issue continues a factory reset would be the next thing to do.  Before you do this you will want to make sure the back up is turned on in your settings.  Also make sure to go into apps + settings and create a back up now.  Once this is done, go to settings > about > reset your phone.  If you skipped the above step and the reset with your back up restored leads to the same behavior, try a reset and skip the restoring of your back up.

Finally, if nothing we have discussed resolves the issues, try the Lumia recovery tool, http://www.nokia.com/us-en/support/product/lumia925/faq/?action=singleTopic&topic=FA142987&category=softwareupdateanddownload .  This tool will reinstall both the OS and firmware on your phone.

Creating and Saving Cover Page - Microsoft Community

joelrwood74 say:

Creating and Saving Cover Page

This question is in regard to Word 2010.
I need to create a SIMPLE cover page that only has a few lines of text, no fancy formatting or anything.
I need to then be able to save it and insert it into several documents that I already have without messing up the formatting or page numbers of those documents.

(Actually I need to set this up and test it on my machine (W7-Office2010) and then move this solution to another machine (XP-Office2007)).
Thanks in advance.

suzanne s. barnhill say:

Add a Next Page section break at the end of your cover page; that will keep its settings (paper size and orientation, margins, vertical alignment, etc.) to itself. Format the page numbering in the section to start at 0; this will prevent it from affecting the numbering in a document you add it to. Be aware that your inserted cover page will adopt the paragraph style formatting of the target document.

 

You might try saving the page as an AutoText entry or adding it to the Cover Page gallery (not sure how well the latter would work).
