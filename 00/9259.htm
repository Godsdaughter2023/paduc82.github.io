Wow, you're not making much sense there mate.  Let's see if we can break this down.

You have 4 columns of data titled A B C & D, in columns G H L & Q respectively.  I'll refer to the actual column letters (G H L & Q) from now on, since we'll use those in any formulas.

Cell K7 contains a value that you want to find in column G.  When that value is found, you want to return the number on the same row in column H.  In your example, lets say cell G25 contains 65387 (= the value in K7), so H25 contains 20000.

To do this, you can use the VLOOKUP function:

    =VLOOKUP($K$7,$G$19:$Q$72,2,False)

The first argument is what you want to search for (K7)
The 2nd argument is the range to be searched, where the column to be searched must be the left-hand column.
The 3rd arg is the column number to be returned, in our case 2 is the second column in $G$19:$Q$72, i.e. col H.
The 4th arg (False) means it will only return if an exact match is found.

Now VLOOKUP can only return the first match it finds (from the top), so if there are several instances of 65387 in column G, and you want it to return the match from the row containing the minimum value in L you should first select your range G19:Q72 and sort by column L ascending (via Data > Sort).

That just leaves you condition for column Q: "any value of 'D' items, which is equal and less the given number's "D" item 10000".  That just doesn't make any sense.  Please try explaining that one again.

Finally, you started this thread talking about using SUMPRODUCT, but nowhere in your thread do you talk about what you want to multiply by what, and what you want to sum.  All of your conditions appear to return only 1 line from your table, ("the min value for C").  If you want to return the product of columns H L & Q where the match was found between G and K7, you can use the MATCH function to return the position in the table of the match:

   =MATCH($K$7,$G$19:$G$72,0)

and if you've already sorted by col L, it will return the position of the row where col G matches K7 and col L is the minimum of all matches.  If you want to use a value in this position, use the INDEX function.  You can nest the MATCH function in the INDEX function to look up the vertical position of a value in one column and return the value in the same row in another column.  If you want to multiply columns H L & Q you can use:

=INDEX($H$19:$H$72,MATCH($K$7,$G$19:$G$72,0))  *  INDEX($L$19:$L$72,MATCH($K$7,$G$19:$G$72,0))  *  INDEX($Q$19:$Q$72,MATCH($K$7,$G$19:$G$72,0))

•Click on Security Info under Overview

if you see that it is enabled, click on Create a new app password under App passwords and use this as the new password for the Live ID on your phone.
