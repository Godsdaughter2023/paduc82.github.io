There is no simple way to create a table in the back-end from within the front-end.  If you have an existing table in the front-end, you can readily export it to the back-end from the External Data -> Export ribbon group.  But aside from exporting an existing table, you have to use VBA code to create a table in the back-end.  That can done by opening a connection to the back-end database and executing a make-table query via that connection, or using DAO objects to open the back-end database and create a table within it.  But there's no simple, user-friendly way to do it.
