chip pearson say:

> Buttons is not an recognized collection

Actually, it is. Buttons contains all of the Button objects added via the Forms command bar, not the ActiveX command bar. It is a hidden property, but you can see it in the Object Browser. Open the Object Browser, right-click almost anywhere and choose "Show Hidden Members" in the popup menu. You will find both the Button and the Buttons objects in the Excel library.  The control is accessible via both the Buttons collection and the Shapes collection. As a Shape, its type is msoFormControl (= 8). By default, the assigned macro is named Button1_Click, but this procedure is in a regular code module, not the containing sheet's object module.  Both the Button and Shape objects are accessible from within the sheet's object module via the Me reference:

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
    MsgBox "Buttons: " & Me.Buttons("Button 1").Name
    MsgBox "Shapes: " & Me.Shapes("Button 1").Name
    Cancel = True
End Sub

vijay a. verma say:

I propose following VBA code which incorporates OssieMac's suggestion also. I am setting the color to Yellow. You can change to color to anything you like.

If 1904 system is changed to 1900 system, all dates on the worksheet will be impacted except dates before 1-Jan-1904 which will be treated as Text by Excel.

1. Save your file as .xlsm
2. ALT+F11
3. Locate your Workbook name in Project Explorer Window
4. Double click on your worksheet name
5. Copy paste the below code in this

6. Press F5 to run your code

'******** Code for 1904 to 1900 Coloring *************

                 

Sub DateConvFormat()
    Dim Rng As Range
    Dim Cell As Range
    Dim FormulaCells As Range
    Dim ConstCells As Range

   If ActiveWorkbook.Date1904 = True Then
        ActiveWorkbook.Date1904 = False
        Else
        Exit Sub
   End If

   On Error Resume Next

   Set FormulaCells = Me.UsedRange.SpecialCells(xlFormulas, xlNumbers)
   Set ConstCells = Me.UsedRange.SpecialCells(xlConstants, xlNumbers)
   Set Rng = Union(FormulaCells, ConstCells)

   For Each Cell In Rng
        If IsDate(Cell) Then
            Cell.Interior.Color = vbYellow
        End If
   Next Cell

End Sub
