mike h.. say:

here's all three methods, you choose

Round to mearest 10 seconds

=MROUND(A1, 1/8640)

round up to next 10 seconds

=ROUNDUP($A$1*8640,0)/8640

and lastly round down

=ROUNDDOWN($A$1*8640,0)/8640

In case you curoius where 8640 comes from it's like this.

The are 6 ten second intervals in a minute, 60 minutes in an hour and 24 hours in a day. 6*60*24= 8460.

For different rounding multiples you follow a similar logic. For example for 15 minutes use 96  as there are 96 fifteen minute intervals in a day

What is web page filtered? - Microsoft Community

healee say:

What is web page filtered?

Does Microsoft Word let us see the source code or edit when it opens a web page?  I don't seem to be able to change anything of a web page with Microsoft Word.  Is it meant to be read-only mode?  What if we save a new document as a web page with Microsoft Word?  What happens to the source code?  What is web page filtered?

yves dhondt say:

No. Microsoft Word is not intended to be used as a HTML-editor. That was possible in Word 2003 and earlier, but that functionality was removed (luckely).

The fact if you save a document as web page or web page filtered shouldn't matter in your options to edit it. The difference is that with the web page option, a whole bunch of custom Word style elements are stored inside the HTML document. When you choose the filtered option, those elements will be removed. But in both cases you will get access to only a very limited set of the HTML and CSS functionality that exist in a real HTML editor.

If you want to edit HTML documents and have no commercial intend, you might want to check out thehttp://www.microsoft.com/express/Downloads/ . There is a free "Visual Web Developer Express" which is much more suited for HTML design.
