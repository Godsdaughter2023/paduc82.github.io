HDAUDIO means this is a high definition audio device


FUNC_01 means the device is an audio chip or card. (FUNC_02 is a modem.)


VEN_10EC identifies the vendor as RealTek Semiconductor Co. of Taiwan


DEV_0892 means the device is a RealTek high definition audio chip using RealTek's proprietary ALC892 codec


SUBSYS_18496893\&REV_1003\5&1732eb40&0&00 are RealTek's internal identification numbers. REV_1003 means that the documentation is in English.

frederik long say:

Here you go:

  1. Log on with your PIN.
  2. Create a desktop shortcut for cmd.exe.
  3. Right-click this shortcut, then left-click Run as administrator.
  4. Accept the challenge. This will give you a black frame.
  5. Type these commands exactly:
    net  user  "%UserName%"  123456
    (to set your password to 123456)
    net user Spare abcdef /add
    (to create a spare account with a password of abcdef)
    net localgroup administrators Spare /add
    (to give the spare account admin privileges)
  6. Make a record of the spare account details so that you will be able to use it several years from now.

Why create a spare admin account? Well, everyone has a spare house key, for obvious reasons, and everyone needs a spare Windows account, for much the same reasons, as you just found out for yourself!

Do you know when this problem began?  Try a System Restore to a point in timeBEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesn’t work. The recovery disk works a bit different from the above procedures but if you follow the prompts from the System Restore menu option with the above information you should be able to restore with no problems.

Go to Start / Control Panel / Mouse and check all the settings to see if there are any that use mouse functions to control screen size.  If so, disable them and that should resolve the problem.

You may be having troubles with your device drivers, firmware, and/or software. Go to Device Manager by going to start / search box and type device manager and enter and then double-click on the program icon that appears.  Check each device for a red x, yellow ! or white ?.  These identify devices with problems (probably drivers, but also conflicts or something else).  Click on each for further details and troubleshooting tips.  If you need to get drivers (and you shoulddo your video card, monitor, and mouse at least and any you have time to do - they may help in ways beyond just resolving this problem), do so from the computer vendor or device manufacturer (NOT from Microsoft Updates).  In fact, you should turn off automatic driver updates from Windows Update as follows: http://www.addictivetips.com/windows-tips/how-to-disable-automatic-driver-installation-in-windows-vista/. Proceed as follows to get the drivers: http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm.  Once you have the drivers, you can install them through Device Manager as follows:http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html.

I hope this helps.  If not, then we need more information about the problem to identiffy the cause and resolve the problem.  Follow the instructions provided by Gokul.
