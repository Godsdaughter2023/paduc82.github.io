erickchu_msft say:

Hi WOOPWOOP,

Items in the deleted items folder are removed after 30 days by the default retention policy, which can be adjusted. Another policy that lets you recover items in the Recoverable Items folder for up to another 14 days (by default), but it can be customized Rolling Holds. After that, about “After these days the administrator have the option to restore the data for another 30 days?”, there is no official document talking about it. Based on my experience, users need to enable the in-place hold or litigation hold to find emails back.

Note: The time limit on reconnecting a deleted mailbox is 30 days.  This cannot be adjusted.

Question 1: Is it possible to increase the 14 days in stage 1?

If an email is deleted from the Deleted Items folder or it meets the threshold of the retention policy, it goes to the recover deleted items list and stays there for 14 days by default. It can be set to 30 days.

Here are the steps:  

1. Connect to Exchange Online using remote PowerShell.
2. Run the following command:

Set-Mailbox -Identity <User Mailbox> -RetainDeletedItemsFor RetainDays

If anything is unclear, please feel free to let me know.

dhookom say: In a union query, the column/field name is identified in the first select. All other selects in the union get their column names from the first.

If you only want to generate a query then include your original query as another select in the union query.

SELECT [EventID], [Type], [EventDate], CityMain As City
FROM [YourQueryName]
WHERE CityMain Is Not Null
UNION ALL
SELECT [EventID], [Type], [EventDate], CitySub As NewCityMain
FROM [YourQueryName]
WHERE CitySub Is Not Null
UNION ALL
SELECT [EventID], [Type], [EventDate], CitySubSub
FROM [YourQueryName]
WHERE CitySubSub Is Not Null

scottgem say:
You would need to create a query that like this:

SELECT EventID, City
FROM Table

UNION SELECT EventID CitySub AS City
FROM table

UNION SELECT EventID, CitySubSub As City
FROM table;

Then join this query to your main table on EventID and pull City from the Union query.
