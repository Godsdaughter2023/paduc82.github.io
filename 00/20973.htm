Is it possible that one or more of these values are text representations of numbers? Perhaps the minus sign is a non-standard hyphen (from HTML characters) or there are leading or trailing zeroes.

A quick check would be to put =TYPE(<cell with negative number>) into an empty cell to the right and Copy or Fill Down for a number of cells. Each TYPE() should be a 1 (e.g. number). A 2 is text even though it looks like a number.

If you have text values, first set the column's number format to General then try highlighting the column and using the Data tab's, Data Tools group, Text to Columns command. As soon as it opens up, click Finish. This is often enough to convert text representations of numbers to true numbers.

You can also try =VALUE(<cell with negative number>) to attempt a resurecction of the number's true value.
