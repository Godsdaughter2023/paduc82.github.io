considerate_guy say:

What you are trying to open is NOT a movie; it sounds like it is a Movie Maker project file. The project file from Movie Maker actually doesn't contain any media items (video, photos or music). The project file only contains the names of the media items used, their length and order, the names of effects applied, the names of transitions that are between clips, etc.

The project file refers to the media stored on the computer. So if someone has sent you a project file, or if you have copied a project file to another PC, it won't be able to preview the project correctly.

If someone is trying to send you their movie, they should first create the actual movie. There's an option in Movie Maker (File > Save movie) that will follow all the instructions in the project file and will create a single video/audio file with all the changes incorporated. They can then share that with others.


But with just the project file, it's like they gave you a recipe without the ingredients. Trust me, a piece of paper containing the recipe isn't as tasty as the final cake that is baked. So tell your friend to actually bake the cake and send you some. :)
