Outlook tasks created in OneNote 2010 do not show up in Outlook. Synchronization is unreliable. Attempting to modify a task through OneNote returns the error message: "OneNote has encountered a problem and cannot complete the requested action."

 

 

System

  • Windows 7 Professional 64-bit, Office Home and Business 2010, hosted exchange server, Outlook Hotmail Connector, OneNoteVersion: 14.0.4760.1000 (32-bit), notebooks stored and shared on SkyDrive.

 

Symptoms

  • Tasks created in OneNote do not immediately show up in Outlook. Sometimes the delay is so long that it makes it impractical to create tasks in OneNote.  Sometimes it seems as if the sync will not take place at all.
  • Sometimes previously synced tasks will appear to stop syncing, and the links between the two applications will appear to stop working. Clicking on the "Open Task in Outlook" button will return an error message, "OneNote has encountered a problem and cannot complete the requested action" (paraphrased).  Outlook tasks cannot be marked complete or deleted through the OneNote interface.

 

Workaround

  • Create all tasks in Outlook rather than OneNote. It seems that tasks created in Outlook behave differently than tasks created in OneNote. Tasks created in Outlook seem to retain their links more reliably.
  • Use the "Send to One Note" option in Outlook to move the task to the current page in OneNote.

 

Requested Fix

  • Make sure that links between OneNote and Outlook are reliably maintained.
  • Speed up sync between OneNote and Outlook.
  • Add a way to find out the current status of sync between Outlook and OneNote.
  • Add a way to force a fresh reliable sync between OneNote and Outlook (perhaps using the "Sync and status settings" or "Sync All Noteooks Now" interface).

 

Feature Requests

  • In the "Send to OneNote" feature in Outlook, in the dialog box for "Select Location in OneNote", offer a check box for "Always send task notes to the current page." (When sending meetings to OneNote, this option would need to be "Always send meeting notes to the current section.")
  • Synchronization of task names between Outlook and OneNote (optional?)
