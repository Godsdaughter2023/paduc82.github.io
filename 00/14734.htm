john p spencer say:

You could try two query approach.  First Query gets all records and the rank

SELECT DISTINCT [04 Outcomes].[Sent Occ ID]
, ([YJB Outcomes].Rank) AS [Outcome Rank]
FROM (([01 Hearings] RIGHT JOIN
(([03 Offences] RIGHT JOIN
[04 Outcomes] ON [03 Offences].[Offence ID] = [04 Outcomes].[Offence ID]) LEFT JOIN
[02 Hearing Offences] ON [03 Offences].[Offence ID] = [02 Hearing Offences].[Offence ID])
ON [01 Hearings].[Hearing ID] = [02 Hearing Offences].[Hearing ID]) INNER JOIN
[YJB Outcomes] ON [04 Outcomes].[Outcome Type] = [YJB Outcomes].[Outcome Type]) LEFT JOIN
[YJB Remand Types] ON [01 Hearings].[Remand Type] = [YJB Remand Types].[Remand Type]


Second query uses the first query to get the top 2 for each [Sent Occ ID]

SELECT [Sent Occ ID]
, [OutCome Rank]
FROM AboveQuery
WHERE [OutCome Rank]in
  (SELECT Top 2 [OutCome Rank]
   FROM AboveQuery as Temp
   WHERE Temp.[Sent OCC ID] = AboveQuery.[Sent OCC ID]
   ORDER BY Temp.Rank DESC)
