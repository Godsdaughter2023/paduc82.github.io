tanneeru kumar say: Hi,

 

Usually the Scanpst.exe is a hidden file .Follow the below steps to find and run the Scanpst.exe to repair the Outlook PST.

 

1. Enable “show hidden files and folders”

Start >My Computer >Organize >Folder and Search Options > View Tab >Check Show Hidden Folders and Files and uncheck “Hide Extensions for Known File Types”.

 

2. Location for Scanpst.exe:

Vista 32 bit: Start >My Computer >Program Files >Microsoft Office >Office 12 >ScanPST.exe.

Vista 64 bit: Start >My Computer > Program files(x86) >Microsoft Office >Office 12 >ScanPST.exe

 

3. Run the tool, and browse to the following location to choose the Outlook data file which you want to repair.

   “C:\Users\<Account Name>\AppData\Local\Microsoft\Outlook.”

 

4. Click Start, which will start the repair process.

    Note:  The repair might take a long time depending on the size of Data file.

 

5. After the scanning is complete you will be prompted to click repair.

 

Try opening Outlook after the repair is complete to check if the issue is fixed.

Hi Jerry,
 
Open MSE and go to the Settings Tab and make sure the box to scan Remote Drives is checked in Real time protection.
 
Insert the USB Drive.  There are several options but I'll give you two:
 
1.  Open Windows Explorer and right click on the USB Drive and click "Scan with Security Essentials".  MSE will then scan the flash drive.
 
2.  In MSE on the Home Tab, check the radio button for Custom Scan and click scan now.  You will be given choices of what to scan.  Uncheck everything except the USB drive (or only check the USB drive) and run the custom scan.  MSE will then do a custom scan that includes only the flash drive.
