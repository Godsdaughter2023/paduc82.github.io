Remember - everything you see in the Start menu is just a shortcut stored in some folder somehere, so you can create new shortcuts in those folders and when you do, those new shortcuts will show up in the Start menu for All Users, or just a particular user.  It depends on what folder you put them in.

What you see in your Start Menu, Programs is a combination of what XP offers to ALL users by default and then what you might have installed just for a certain user.   Every time you create a new user, they get their own Start Menu, Programs folder under their login so you can add stuff to it if you want.

That is why when you install some programs, it asks you if you want it to be available for All Users or just a particular user (you - the installer).  Depnding on what you choose, the installation program will then put the shortcut to the new program in the correct folder - either the one for all users, or the one just for your login.

For the example above, I created two new shortcuts that will show up for XP All Users in this folder:

C:\Documents and Settings\All Users.WINDOWS\Start Menu\Programs\Accessories\System Tools

That means everybody (all users) will see them.

If I only wanted logged in user Jose to see those shortcuts, I would put the in the Systems Tools folder that belongs to Jose and that would be here:

C:\Documents and Settings\Jose\Start Menu\Programs\Accessories

Then, you can choose 'Run as..'. on the two new shortcuts.

Look around under your Documents and Settings and you will see what it going on and how it works :-)
