Hi,

Creating a wireless Hosted Network on your computer will allow the hotspot connection, thus resolving your concern. Follow the steps below:

  1. Open Command Prompt (Admin), then enter the command below:
    NETSH WLAN set hostednetwork mode=allow ssid=Your_SSID key=Your_Passphrase
    • Note: Replace Your_SSID with your personal username and Your_Passphrase to your own password (e.g.: ssid=Ushmita key=Chakraborty).
  2. Active the Hosted Network by entering the command below:
    NETSH WLAN start hostednetwork

Next, let's share your Internet connection with a Hosted Network.

  1. Type View Network connection in the search bar, then press Enter.
  2. Right-click the network adapter you're connected to, then click Properties.
  3. Go to the Sharing tab.
  4. Check the Allow other network users to connect through this computer's Internet connection option.
  5. From the Home networking connection drop-down menu select the Microsoft Hosted Virtual Adapter.
  6. Click OK.
  7. Try connecting other devices to test.

    lemp say:
    I don't know what these files are, however, any files in the TEMP directory are safe to delete, but you might want to reboot once, just to make certain that there is no "restart to complete this install procedure" pending.

    Files that are shown in blue are compressed.

    If the files are more than 7 days old, you may be able to delete them using Disk Cleanup (Start > Run > cleanmgr.exe > OK).

    Otherwise you can probably delete them by starting in Safe Mode (tap F8 immediately after reboot sequence starts, before the Windows logo appears).

    Or, instead of Safe Mode, the following almost always works:

    • Open a Command Prompt window (Start > Run > cmd > OK).
    • Navigate to the directory where the LAExxx.tmp files are located using the cd command.

                    cd C:\Windows\Temp           {press Enter}

    • Leaving the Command Prompt window open, start Task Manager (CTRL+ALT+DEL)
    • Find explorer.exe in the list of processes, select it and click "End Process"
    • At this point, your display should be blank except for the open Command Prompt window and Task Manager
    • In the Command Prompt window, type the following command

                      del *.*                  (press Enter}

    • Go back to Task Manger, click File > New Task (Run...) and in the "Open" box type

                      explorer.exe

    • Click OK.
    • Your Desktop should be back to normal. Close the Command Prompt and Task Manager windows and the files should be gone.
