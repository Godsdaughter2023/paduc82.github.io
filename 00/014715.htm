rayr001 say:

Windows Phone update 8.1 on HTC 8X

So the Preview for Developers Windows 8.1 update is out.  If you’ve checked out Youtube you can see a list of all the new features with the biggest one being Cortana.

I’ve done the update and all went well with the install for me on my HTC 8X.  But I wanted to just point out 3 small issues I don’t like.

  1. Cortana is great but it still needs work, so don’t expected it to be 100%.  She would not read my emails, or news.  While she can open apps she has yet been able to interact with them as shown on some videos on Youtube but I think this will change as more updates arrive.
  2. I don’t like that I have to hit the search button then the speaker button to access Cortana.  I miss hitting the windows button and being to speak my commands.  While it seems Cortana handles speech better I still find this to be 1 extra step that does not need to be done.
  3. Last on my list after the upgrade I went to change one of my alerts (text to be exact) and the feature to add custom sounds to text and emails is gone and your stuck with what ever ones you added before the upgrade.  You can still change ringtones.
  4. The HTC hub that shows time, weather, stocks ect does not always update

Those are my few issues so far but over all it seems it’s good.  Oh the notifications on the top can be annoying at times also as you never know if it’s a email, an alert from the phone telling you something or a text message until you use the pull down menu to see what the alert is but that is a minor thing.

Just to add my issue 2 I found by hold the search button down you go directly to Cortana but I still miss holding down the middle button guess I just have to get used to that.
