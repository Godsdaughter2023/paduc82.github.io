Hi

Network in essence is a combination of Computers and the OS’ running, Network Hardware, Internal traffic, External Internet traffic, and Users’ level of Computing/Networking knowledge.

That means a lot of exchange, and a lot of dynamic processes  most of it the unseen working in the background.

When troubles occur the “Blame Game” starts some user blame them self other do not like the hardware and there is always the OS and the software to pick on.

The trick is to stay away from the past and the blame game, "it worked before and it is Not working Now" is secondary, what has to be done is finding out why it is Not working Now.

If everything "looks" OK then the fastest solution is to reset redo some of the configuration and go on with Happy computing.

Win7 when configured on peer-to-peer Network has three types of Sharing configurations.

HomeGroup Network = Works only between Win 7 computers. This type of configuration makes it very easy to Entry Level Users to start Network sharing.

 Work Network = Basically similar to the previous methods of sharing that let you control what, how, and to whom folders would be shared with.

 Public Sharing = Public Network (like Internet cafe) to reduce security risks.

 For best results log to each computer's System Screen and set all the computers to be on a Network with the same name while each computer has its own unique name.

http://www.ezlan.net/Win7/net_name.jpg


Make sure that the Software Firewall on each computer allows free local traffic. If you use 3rd party Firewall On, Vista/XP Native Firewall should be Off, and the active Firewall has to adjusted to your Network IP numbers on what is some time called the Trusted Zone (consult your 3rd Party Firewall instructions

General example,  http://www.ezlan.net/faq#trusted

Please Note that some 3rd party Software Firewalls keep blocking aspects of Local Traffic even it they are turned Off (disabled). If possible configure the Firewall correctly, or totally uninstall it to allow clean flow of local network traffic. If the 3rd party software is uninstalled or disables, make sure that Windows native Firewall is On .

If your network is comprised of Win 7 only and you want a simple Network, use this.

http://windows.microsoft.com/en-us/windows7/help/videos/sharing-files-with-homegroup

If the HomeGroup Network keeps given you "Hard time", and you are willing to spend some time on learning and configuring the Network, switch all the computers to the more flexible Work Network.

In the Network Center, clicking on the Network type opens the window to the right.

Choose your type of Network. Notice the check mark at the bottom, and check/uncheck it according to your needs.

http://www.ezlan.net/Win7/net_type.jpg

Win 7 - http://windows.microsoft.com/en-us/windows7/Networking-home-computers-running-different-versions-of-Windows

Win 7  Work Network specific folder sharing - http://www.onecomputerguy.com/windows7/windows7_sharing.htm

Vista File and Printer Sharing- http://technet.microsoft.com/en-us/library/bb727037.aspx

Windows XP File Sharing - http://support.microsoft.com/default.aspx?scid=kb;en-us;304040

Printer Sharing XP - http://www.microsoft.com/windowsxp/using/networking/expert/honeycutt_july2.mspx

Windows Native Firewall setting for Sharing XP - http://support.microsoft.com/kb/875357

Windows XP patch for Sharing with Vista (Not need for XP-SP3) - http://support.microsoft.com/kb/922120

When finished with the setting of the system it is advisable to Reboot all the hardware including the Router and all computers involved.

*Note. Some of the processes described above are done Not for Windows sake but to compensate for different Routers and the way their firmware works and stores the info about the computers that are Networked.

Best Practices For Windows 10 - Microsoft Community

rj0088 say:

Best Practices For Windows 10

Storing User Data & Games or Large Apps.

Don't Lose Your Files Ever Again.

  • Tablets:
     Keep all user data on a Micro SD Card. Never store on your OS drive.

  • Desktop PC:
    Secondary SSD (Solid State Drive) for Games & Large Software Packages like Adobe. A high capacity drive is needed but make it the fastest one you can afford. I use a 500GB SSD for my Games & Data as larger SSD's are still very expensive ($12-1500 for 1TB).
    Windows only needs a stated 20GB so a small drive will do. 128GB is plenty of room and cheap but a 250GB is also reasonably priced. If you want to get into Partitioning. A 250GB or even another 500GB would work. You can use the same drive, 500GB+ for all of it. Just split it up for OS, using 1/4 the drive and use the rest for everything else.

  • Notebooks:
    I try to follow the same process but most laptops can't take more that 1 drive so a 2nd SSD is out of the question & using an SD Card for your User Files is sketchy as they're unreliable. I recently lost everything because Windows suddenly decided the drive was blank after an update an reboot. I think it dumped the Control Files, losing the Master Boot Record (MBR). I tried everything to recover it with no luck as SD Cards are much harder to recover from than HDD's or SSD's. So I will never save anything important on any kind of SD Card ever again. They are not meant to be used as Disk Drives no matter what any operating system developers or card manufacturer say.

  • OneDrive

One last thing: Setup OneDrive on your Windows Device so all your Documents and whatever else you choose. Anything except installed program files. Windows 10 owners get tons of OneDrive space by default and it's very easy and mostly free to be able to get over 1TB in your reserve. Just read the details on the webpage.

A great thing about this is you can sync the files across your devices by just signing in and choosing a location to save the files. All the files you've loaded or just some of them. It's all your choice. Save them all on a Desktop with tons of space and just some on tablets and notebooks.

I like keeping my drivers on there. That way I only need a NIC Driver if needed. Than I get the rest from my OneDrive Drivers Folder. Which holds a Folder for each of my devices.

Hope This Helps

glasssandwich say:

Option 1: Third party anti-malware software. Windows Defender does not always detect malware or adware. Malwarebytes would be my recommendation, since it has a 14 day trial and a free version after that. It's a lot more reliable than Defender, and pretty cheap considering how effective it is. 

Option 2: Back up files you want to keep and reset your computer. While that is a somewhat annoying process, it will probably eliminate the adware. That's really the nuclear option, though. I had to do it once when neither Defender nor Malwarebytes detected a virus that was redirecting all of my searches to sites I'd never heard of nor cared to know of.

Option 2.1: Download Windows 10 ISO to a flash drive, and boot from it. You can do a clean install from there. (If you do this, make sure to write your license key down before doing it.)

Option 3: Direct deletion of all RocketTab files. They should be around C:\Program Files (x86)\Search Extensions. Searching in the 'This PC' section of File Explorer should bring them up as well. 

Option 4: Make sure that nothing named 'BrowserSafeguard with Rocket Tab' is on your computer. It's RocketTab, but with a slightly different name, most likely to confuse you.

Option 5: Place a Linux ISO on a thumb drive and install via BIOS. It will ask if you want to install alongside Windows or if you want to replace it. I would recommend Ubuntu; it's clean and easy to use, and also allows you to boot from the thumb drive in a 'trial mode'. This is also a nuclear option. (Linux will not provide the same user experience as Windows 10.) 

If you logged into Facebook, or Twitter, or your Microsoft account while the adware was active, there's a chance that your accounts have been compromised. After you are absolutely sure that the adware is gone, set new passwords for everything, including a new administrator password. 

If your computer is Lenovo, make sure to read up on SuperFish and the potential security breach in the computer's BIOS. RocketTab and SuperFish seem to be siblings, or cousins at least, so it doesn't hurt to be careful. If RocketTab is related to SuperFish, then Defender might not detect it even in safe mode.
