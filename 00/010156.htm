When you just edit an attachment to a mail message instead of saving the attachment to a separate file before editing it, the file is opened in a temporary folder. When you close the document, that folder may or may not be deleted. Even if the folder is still in existence, it may be hidden.

The best advice is never edit an attachment without first saving it as a regular file.

You might -- might! -- be able to retrieve the file from the temporary folder. See http://www.gmayor.com/outlook_attachments.htm.
