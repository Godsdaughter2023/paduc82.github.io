One of the things I have been able to confirm (which is a lot of work) to fix this and requires additional software.

1) Crop the picture to a perfect square

2) Ensure dimensions are 200 x 200

3) Resolution of the image should be 72 pixels per inch

4) Connect Microsoft Account to Microsoft Office Outlook 2013 client
5) Edit and add picture to contact from there

This has allowed the images to immediately sync with contacts on the iPhone.  

Edit 12/20:

FAIL: When uploading the photo (200x200 resolution of 72px per inch) using People on W10 PC or W10 Mobile, the photo (sometimes) syncs with the cloud (Outlook.com), but does not sync with iOS.

FAIL: When uploading the photo (200x200 resolution of 72px per inch) using Contacts on Mac OSX, the photo stays in the Calendar and Messenges within the MAC but it does not upload to the cloud (Outlook.com) or iOS.

Success: The only time this has proven 90%+ successful has been using Outlook 2013 Client with the image specification of 200x200 with the resolution of 72 pixels per inch.
