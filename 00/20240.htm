ken sheridan say: In a report grouped first by Vendor and TransactionYear you can conditionally aggregate the amounts in the TransactionYear footer with an expression like this:

=Sum([TransactionAmount] * IIf([TransactionAmount]>0,1,0))

If the report is grouped first by TransactionYear and then by Vendor you'd do so in the Vendor footer.

The TransactionYear can be returned with an expression: Year([TransactionDate])

To return the totals independently of a report you can sum the values greater than zero in a query grouped by vendor and year to return the total positive value per vendor per year.  You can see this in the Balances demo in my public databases folder at:

https://onedrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

Note that if you are using an earlier version of Access you might find that the colour of some form objects such as buttons shows incorrectly and you will need to  amend the form design accordingly.  

If you have difficulty opening the link copy its text (NB, not the link location) and paste it into your browser's address bar.

If, in this little demo file, you were to create the following query:

SELECT YEAR(TransactionDate) AS TransactionYear,
CustomerID,
SUM(TransactionAmount) AS TotalCreditAmount
FROM Transactions
WHERE TransactionAmount > 0
GROUP BY YEAR(TransactionDate),CustomerID;

This would return the total credit (positive) amount per year per customer (analogous to vendor in your case).
 
To return the total amounts per customer for a specific year you could restrict the query by a parameter

PARAMETERS [Enter year:] SHORT;
SELECT YEAR(TransactionDate) AS TransactionYear,
CustomerID,
SUM(TransactionAmount) AS TotalCreditAmount
FROM Transactions
WHERE TransactionAmount > 0
AND YEAR(TransactionDate) = [Enter year:]
GROUP BY YEAR(TransactionDate),CustomerID;

To return the total credits for the year for all customers, just omit CustomerID from the query:

PARAMETERS [Enter year:] SHORT;
SELECT YEAR(TransactionDate) AS TransactionYear,
SUM(TransactionAmount) AS TotalCreditAmount
FROM Transactions
WHERE TransactionAmount > 0
AND YEAR(TransactionDate) = [Enter year:]
GROUP BY YEAR(TransactionDate);

    Dim c As Range

    Next c
