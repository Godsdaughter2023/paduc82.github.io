Set up a new IMAP account

¡firedog say:

That's quite a catalogue of errors!

Server Error: 0x800488A6
Server Error: 3219

I think these two are essentially the same: the account has been upgraded to Outlook Mail (Preview) and you can no longer synchronize it using DeltaSync. See how to work around this by using IMAP here:

 Server Error: 3204

This error has affected a number of users who have upgraded to Windows 10. They have discovered that they have to go to their account security page and sign in with the affected account. The account needs verifying, e.g. by submitting a security code sent to the phone number registered on the account or to one of its alternative email addresses. After verification, server error 3204 no longer occurs.

 

It's a good idea when visiting the account security page to ensure that at least two verification channels are registered, e.g. a mobile phone number and an alternative email address, preferably on a different domain like Gmail. This will make it possible to recover the account should you ever lose access to it.

 Server Error: 0x800488FE

This probably means that the account has been temporarily blocked for some reason. The cure is the same as for error 3204: sign in to account.live.com and verify the account.

Dealing with several different accounts online can be problematical because the different sign-ins collide. You might find it easier to visit each account in succession using Internet Explorer's InPrivate mode (press Ctrl-Shift-P after opening IE).

Please tell us how you get on. In particular, I'd like to know whether the account throwing error 0x800488A6 has been upgraded to Outlook Mail (Preview), which you'll see by signing in to the account at mail.live.com.

Please try these options one by one and check after each option.

4. Reset Windows Store through Command Prompt
Type cmd in Windows Search box > Right click on Command Prompt > Run As Administrator > Type WSreset.exe and click Enter > Reboot your computer

5. Re-register All Store apps
Right click on Windows Start > Right click on Windows Powershell (Admin) > Copy the following from below and right click in Powershell to paste > Enter > Restart your computer

Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

6. Uninstall & Reinstall Store
Right click on Windows Start > Right click on Windows Powershell (Admin) > Copy the following from below and right click in Powershell to paste > Enter

Get-AppxPackage -allusers *WindowsStore* | Remove-AppxPackage

Copy the following from below and right click in Powershell to paste > Enter > Reboot your computer

Get-AppxPackage -allusers *WindowsStore* | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}
