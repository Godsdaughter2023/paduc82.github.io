suzanne s. barnhill say:

Perhaps this has changed, but historically, a Word session has always begun with the printer defaults (that is, the default printer is selected, with the settings set as the default in that printer's Properties). If you change printers or printer settings during a Word session, they persist until you change them again. It is possible to create a macro that will change printers or print settings for a specific document and then reset them to the default; I have such a macro that prints from an alternative printer that I rarely use. Publisher operates differently: specific printer settings are saved with the document. This has never been possible in Word.
