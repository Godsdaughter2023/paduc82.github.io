Conditionally format number closest to mean - Microsoft Community

tedmh say:

Conditionally format number closest to mean

I have a dataset with numbers and I need a conditional formatting formula (or other rule) that will trigger formatting on the one value that is closest to the mean of the dataset. Is this possible?
andrea jones - all about resources say:

If your data were in cells A1:A4 you could use this conditional formatting function:

 

=A1=LARGE($A$1:$A$4,COUNTIF($A$2:$A$7,">"&AVERAGE($A$1:$A$4))+1)

 

This will highlight the closest value less than or equal to the average.

joeu2004 say: Ted wrote:
My dataset is multiple columns and rows and as far as I can tell the index/match combination won't work across a two-dimension array.

The Conditional Format formula paradigm that I provided works with data in multiple columns and rows.

For example, if A1:A33 contains 1 through 33, B1:B33 contains 34 through 66, and C1:C34 contains 67 through 100, select A1:C34 and enter the following CF formula:

=ABS(A1-AVERAGE($A$1:$C$34))=MIN(ABS($A$1:$C$34-AVERAGE($A$1:$C$34)))

That correctly highlights B17 and B18, which contain 50 and 51, which are both closest to the average 50.5.
