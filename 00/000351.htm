Adding email account to Outlook in Office 2016 - Microsoft Community

a. user say:

Adding email account to Outlook in Office 2016

Hello. 

I am having difficulty in adding my outlook email account to Outlook in Office 2016.  I'm not sure whether to use IMAP or Pop.

karl timmermans (outlook mvp 2012-2018) say:
>>
I'm not sure whether to use IMAP or Pop.
<<

There's no right or wrong answer as to which is best for your particular scenario. Depends on what you need to do.

Executive Summary

POP3 - Only downloads the <Inbox> emails in a one-way one-time direction. If you do not set "Leave Messages on server for "##" days" then the messages get deleted from the server. Contact/Calendar data is local to the computer only and doesn't sync with the server

IMAP - Email data is sync'd with the server. The Outlook data file for an OST account is a "mirror" of what exists on the server. What you delete on the desktop gets deleted from the server and vice versa. Like POP3, contact/calendar data is local to the desktop only and doesn't sync with the server. You would use this if using multiple devices to connect to your email account and you wanted all to access the same data.

In Office '2013/'2016 - contact/calendar data is stored in folders labeled ith the words "This Computer Only" which means exactly that. The downside is that if you delete the email account then the data file is also deleted resulting in all data in "This Computer Only" folders being lost unless you have manually backed up (Exported) the data in these folders to a PST file.

If you configure your account be only entering your email address/password, Outlook will default to configuring the account using IMAP. The exception to this would be if this is an Outlook.com or hosted Exchange account in which case it would be configured using Exchange where everything (email, contacts, calendar etc) is sync'd with the server.

bilo cape town say:

Hi. I see you have a USB flash drive connected.

Depending on the Boot Order set in the BIOS and which devices are enabled to boot from - this can cause issues.

You have gone to great detail in concentrating on the Device Manager,

But the obvious place to start is in the Disk Management which will show drives which were detected at boot time.. This does not mean that it will find all that are attached.

Right click on the Start Button and select Disk Management.

Then click on 'Action' in the menu and select 'Rescan'. Do this more than once until ALL drives are detected.

If it finds your missing item - it may not have drive letters allocated. It needs a letter for each partition which you want to see in File Manager etc.

Right click on the partitions you want the system facilities to see - and allocate a letter. The default is good enough but you can choose another letter if you wish.

I have a rack for my hard drives that lets me plug in a drive to boot from. Occasionally I get the prompt to say 'No boot device found'. On the second attempt it may find the boot device - otherwise I open the BIOS and make sure that the boot device is at the top of the order. Then it always boots up.

If you installed the preview and it has a storage drive - always make sure that the same storage drive is attached when you boot the preview. Otherwise it will complain about the drive being missing or needing repair.

BTW: I never dual boot OS's because when testing previews - you may get the boot manager confused by drives being disconnected or changed to boot a hard drive for a different system.

All that info and pictures you posted are giving conflicting information about what you are trying to do.

bilo cape town say:

Rule No 1: If it is not broken - don't fix it

If there are newer drivers available than the ones you have that are working - rather leave them alone.

The newer drivers sometimes turn out to be incompatible with your hardware.

The only driver that I have ever had to replace in Windows 7 / Windows 8 / 8.1 / Win10 TP was an audio driver. This was because the correct driver from the manufacturer usually does not have Stereo Mix for playing music or karaoke.

But my pc's that I use to play music in chat rooms - have ASUS motherboards with Realtek and VIA chips onboard.

The Realtek and VIA support sites do have drivers with Stereo Mix and compatibility with Windows 8.1 and Windows 10 TP.

Otherwise - I just let the system get the drivers. There are no generic drivers installed after letting all the updates come in. The drivers for all the hardware come from the manufacturer. (ASUS - VIA and Realtek)
