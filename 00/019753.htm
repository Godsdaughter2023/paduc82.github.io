When you view your email through Outlook.com, the messages are downloaded off the server and onto the mail client that you are using. This option needs to be configured manually on your mail client.

joanne li msft say:

Hi Gentle_man,

It is not feasible to create a lookup column to get information from another subsite using out of box feature in SharePoint Online.

And if we want to add a column from an existing column, this column needs to be created on the top-level site.

Given this situation, you may try creating the list you want to lookup from in the top-level site and creating a site column in lookup type to get information from the list, then you can go to subsite to create a column from this existing column.

Generally the lookup column can only get information that is already on current site. 

Other community members who have experience on this are welcomed to share their insights here.
