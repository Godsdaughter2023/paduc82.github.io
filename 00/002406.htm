If you have the Checkbox Content Controls arranged in the cells of table as shown below with Titles assigned to each of them as indicated by the blue tags

And your bookmarks have the names of OldFemale, OldMale, YoungFemale and YoungMale, then

the following code in the ThisDocument Object will

  1. Cause each pair of checkboxes (Female and Male, Young and Old) to be mutually exclusive.
  2. Cause the selection to go to the related bookmark

Dim cc As ContentControl

Dim strGender as String

Dim strAge as String

Private Sub Document_ContentControlOnEnter(ByVal ContentControl As ContentControl)
With ActiveDocument
    If Selection.Information(wdWithInTable) Then
        For Each cc In Selection.Rows(1).Range.ContentControls
            cc.Checked = False
        Next cc
        For Each cc In Selection.Cells(1).Range.ContentControls
            cc.Checked = True
        Next cc
        For Each cc In Selection.Cells(1).Range.ContentControls
            cc.Checked = True
        Next cc
    End If
End With
End Sub

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
Dim strGender As String
Dim strAge As String
With ActiveDocument
    For Each cc In .ContentControls
        If cc.Type = wdContentControlCheckBox Then
            If cc.Title = "Female" And cc.Checked = True Then
                strGender = "Female"
            End If
            If cc.Title = "Male" And cc.Checked = True Then
                strGender = "Male"
            End If
            If cc.Title = "Young" And cc.Checked = True Then
                strAge = "Young"
            End If
            If cc.Title = "Old" And cc.Checked = True Then
                strAge = "Old"
            End If
        End If
    Next cc
    .Bookmarks(strAge & strGender).Range.Select
End With
End Sub

suzanne s. barnhill say:

When you enabled Different First Page in a document with a single section, you ended up enabling it for the entire document because, as you add new sections, each replicates the layout of the previous section. Fortunately, Different First Page (unlike Different Odd and Even) is not an all-or-nothing setting. You can disable it for any given section. Since you want your watermark only on the cover page, this is the sensible approach. Access the header and skip to Section 2 (if you have the section number displayed on the status bar, it's easier to keep track of your progress). On the Header & Footer Tools | Design tab, clear the check box for Different First Page. Then go to section 3 and repeat the process. Continue for any remaining sections.

If you prefer to just unlink the First Page Header, you can do that, and you can remove the watermark by clicking on it and deleting it, but if you really don't need separate headers on the first pages of subsequent sections, you're better off disabling Different First Page instead, reducing the number of headers you need to deal with.

doug robbins - mvp office apps & services (word) say:

Access the First Page Header in Section 1 and then click on Next in the Navigation section of the Header & Footer Tools>Design tab of the ribbon and then click on the Link to Previous button to unlink that Header from that in the previous section.  You can then delete the text in the First Page Header of the second section, without it affecting the header in the first Section.  

Alternatively, in the Second Section, you can uncheck the box for Different First Page.  You will have to however do that for all of the following Sections.  You will need to use that method if you do want a header that is the same on all pages of the second and subsequent sections.
