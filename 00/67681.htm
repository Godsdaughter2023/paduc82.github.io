mark kevinj say:

As the term explains itself, Write protection means that the disk is protected against write operations. i.e., you can’t write on the disk. You are only allowed to read and copy data from that disk, but unable to write or delete any content of that disk.
This can happen because of any one of these two reasons.

1. Virus attack. Viruses sometimes change the access to be write protected, which may lead to this problem.
 2. Enabling the Lock switch. Some USB device has a lock switch. If you lock that using the switch, then it will make the access restricted from write.

A. Searching for the Lock Switch
If you have locked the switch by mistake then, the only way is to search for the switch and unlock it. There is no other way to remove the write protection from that device.

B. Registry Editing

If plan A fails, then there comes plan B. You have to do a small registry editing to disable Write Protection.

Steps to be followed:

1. Press Windows + R to open the Run Window an then type regedit and press Enter to open the
Registry Editor.
2. Navigate to HKEY_LOCAL_MACHINE -> SYSTEM -> CurrrentControlSet -> Control. Search for StorageDevicePolicies.
 If that doesn’t exist, then add add a new key named StorageDevicePolicies. Now navigate to that and create a new  D-WORD(32 bit), WriteProtect.
3. Change the value of the key WriteProtect to 0.


C. Formatting the USB Drive

You can also format the USB drive to make it write protection free. But here, first you have to Low level Format and then format normally.

1. Now open the Device Manager by right clicking on My Computer icon and selecting Manage.

2. Select Disk Management from left pane. Now right click on the drive and select Format.

D. Another Low Level Formatting tool

If you are still facing the problem, even after following previous three methods. You may use HDD LLF Low Level Format Tool to format your USB Flash Drive. Many users have found this tool very useful. You can find a link to this tool, below in the comment section.

Tired of entering my password

Tired of entering my password everytime I turn on my new Windows 8 touch screen laptop.  How do I lock it in?

a. user say:

Hi,

 

Are you using a Microsoft user account or a local user account?

 

If you are using a local account and would like to log into the Windows 8 computer without entering a password every time, then you can change the computer settings to log in without a password. Removing the password makes your computer accessible to everyone, so I would suggest you to remove the password only if it’s your personal computer and no one accesses it apart from you. You can refer to these steps to remove the log in password:

 

Note: make sure that you perform these steps from an Administrator account.

 

Press Windows key + X, select control panel.

Click on view by option in the top right and change it to larger icon.

Click on user accounts and then click on “manage another account”.

Click on the user account for which you would like to remove the password.

Click on “Change password” and leave the password fields empty.

Click on Ok to save changes.

 

If you are using a Microsoft account, then I would suggest you to perform the following steps to disable the password and login automatically:

 

Press Windows key + F to bring up the search charms bar.

Click on the settings tab below the search box and type “netplwiz” in the search box.

Click on “netplwiz” in the search results.

Click on the user account and uncheck the box against “Users must enter a username and password to use this computer” and follow the instructions.

Click “apply” and “Ok” to save changes.

 

Hope you find this helpful. Feel free to use this forum for any Windows related queries in the future; we’d be happy to help.
