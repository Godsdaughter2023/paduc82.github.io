Hi Dalgleish:

You need to resolve your installation error first: that would, indeed, be causing this error.

What the error message is saying is "Something I expected to be in memory is not where I expected it to be".  Really, it has nothing to do with "converting" as you and I would understand it -- the "Import Filter" used by Word 2004/2011 has not started, and therefore "nothing" is in memory.

Word converts a file into its internal format on open, and re-converts it to .docx on the way out again.  It tried that, and it didn't work.  The most likely cause (once the software is correctly installed and activated) is that you're out of RAM or HDD.

In this case, really it's just a generic "Sorry it didn't work" error.

If the support guys take too long to get back to you, completely remove the prior versions, then remove Office 2011 and re-install it.

1) Quit all Microsoft applications.


2) Track down all instances of pre-2008 Normal template on your computer,

no extension.


3) Find and drag the file Normal.dotm to your desktop. Unless you have

moved it, it should be in

/Users/ ~ /Library/Application Support/Microsoft/Office/User Templates/


4) If the following files exist, Remove or rename them:


~/Library/Preferences/Microsoft/Word Settings (10)


~/Library/Preferences/Microsoft/Word Settings (11)


User/Library/Preferences/com.Microsoft.Word.plist


User/Library/Preferences/Microsoft/Office 2008 (the whole folder!)


Then Repair Permissions with Disk Utility.

graham mayor say:

How about the following, which will move the image until it is the same distance from the right margin as it was from the left. Do you really need 1 second intervals? It's a long time to wait around.

Option Explicit
#If VBA7 Then
    Private Declare PtrSafe Sub Sleep Lib "kernel32" (ByVal dwMilliseconds As Long)
#Else
    Private Declare Sub Sleep Lib "kernel32" (ByVal dwMilliseconds As Long)
#End If

Private Sub UserForm_Activate()
    Call y(Me.Image1.Left)
End Sub

Sub y(iLeft As Long)
    With Me
        Do While .Image1.Left < .Width - (.Image1.Width + iLeft)
            DoEvents
            Sleep 100 'change to 1000 for 1 second
            .Image1.Left = .Image1.Left + 10
        Loop
    End With
End Sub

andreas killer say:
Sub z()

Image1.Left = Image1.Left + 10

x = Now + TimeValue("00:00:01")

Application.OnTime x, "z"

End Sub

The reason is that (AFAIK) there is no way call a sub inside a Userform with Application.Run, resp. Application.Ontime

You can only call public subs within a regular module or within a sheet. But you can call that sub directly from VBA, means you can "Call UserForm1.z" from anywhere.

Means you can move sub Z into a regular module and change it a little:

Sub z()
   UserForm1.Image1.Left = UserForm1.Image1.Left + 10
   x = Now + TimeValue("00:00:01")
   Application.OnTime x, "z"

End Sub

But moving an image this way looks very jerky. So I recommend to use a window timer.

For example:

- Make a new file

- Add a regular module and add this sub:

#If Win64 Then
Sub Move_TimerProc(ByVal hWnd As LongPtr, ByVal nIDEvent As LongPtr, _
    ByVal uElapse As Long, ByVal lpTimerFunc As LongPtr)
#Else
Sub Move_TimerProc(ByVal hWnd As Long, ByVal nIDEvent As Long, _
    ByVal uElapse As Long, ByVal lpTimerFunc As Long)
#End If
  'Call the public sub in our form that moves the image
  UserForm1.Move_Image
End Sub

- Add a Userform
- Add at min. one Image to the form (I suggest to add some text boxes too)

- Copy all code below my name into the code module of the form

- Run it

Andreas.

Option Explicit

#If Win64 Then
Dim hWnd As LongPtr
Private Declare PtrSafe Function SetTimer Lib "user32" ( _
    ByVal hWnd As LongPtr, ByVal nIDEvent As LongPtr, ByVal uElapse As Long, ByVal _
    lpTimerFunc As LongPtr) As LongPtr
Private Declare PtrSafe Function KillTimer Lib "user32" Alias "KillTimer" ( _
    ByVal hwnd As LongPtr, ByVal nIDEvent As LongPtr) As Long
#Else
Dim hWnd As Long
Private Declare Function SetTimer Lib "user32" ( _
    ByVal hWnd As Long, ByVal nIDEvent As Long, ByVal uElapse As Long, _
    ByVal lpTimerFunc As Long) As Long
Private Declare Function KillTimer Lib "user32" ( _
    ByVal hWnd As Long, ByVal nIDEvent As Long) As Long
Private Declare Function GetActiveWindow Lib "user32" () As Long
#End If

Sub Start_Move()
  'Get the handle to our form
  hWnd = GetActiveWindow
  'Create a timer called every 5 milliseconds
  SetTimer hWnd, 0, 5, AddressOf Move_TimerProc
End Sub

Sub Stop_Move()
  KillTimer hWnd, 0
End Sub

Public Sub Move_Image()
  With Me.Image1
    'Move it one point to the right
    .Left = .Left + 1
    'Over the right edge?
    If .Left + .Width >= Me.InsideWidth Then
      'Exact position
      .Left = Me.InsideWidth - .Width
      'Kill the timer
      Stop_Move
    End If
    'Give Excel a time slice to move the image
    DoEvents
  End With
End Sub

Private Sub UserForm_Activate()
  With Me.Image1
    'Move it to the top left corner
    .Top = 0
    .Left = 0
  End With
  Start_Move
End Sub

Private Sub UserForm_Terminate()
  Stop_Move
End Sub
