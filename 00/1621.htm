anita_williams say:

Profile photo restriction on AD

Hi,

Is it possible to restrict profile photos on AD for Office 365?

How to do the same?

a. user say:

Hi Anita,

Could you provide us with more detailed information about your requirement, such as an example, etc.? If you want to restrict users to change the profile pictures synced from the local AD in Office 365, you can create a new Office 365 OWA mailbox policy to achieve it (on-premises policy cannot restrict users to change their profile picture in Office 365).

Here are the steps for your reference:
1. Connect Windows PowerShell to the Service: https://technet.microsoft.com/en-us/library/jj984289(v=exchg.160).aspx
2. Run the following command to create a new OWA mailbox policy:
New-OwaMailboxPolicy -Name “policy name”
3. If you want to restrict all the users to change the pictures in OWA, run the following command:
Set-OwaMailboxPolicy -Identity “policy name”-Default -SetPhotoEnabled $false
4. If you want to restrict some specific users to change the pictures in OWA, run the following commands:
Set-OwaMailboxPolicy -Identity “policy name” -SetPhotoEnabled $false
Set-CASMailbox -Identity “end user” -OwaMailboxPolicy “policy name”

Thanks,
Edward

send a message to my laptop - Microsoft Community

mazen_blue say:

send a message to my laptop

i have received a multimedia message , but i am not able to transfer it to my laptop , i saved the message in the “my folder”s on the mobile , but when i browse my mobile using my laptop i can not find the folder and the message any help??/

patc---01 say: I'm not sure where "my folders" is on the mobile - it's a new one on me and given I'm fairly familiar with the directory structures, I suspect that's your problem.

Can you see this directory using File Manager on the phone? If you can't, then "my folders" isn't part of the memory that is visible to PC Suite, ie: it's not a sub-directory of "phone memory:\data" or on the memory card.

The easiest solution would be to re-save the message in a visible part of memory.

If that's not an option, you could install something like Y-Browser (it's free - google it) which will let you see all of the phone memory, including the parts not supported by PC Suite and File Manager. You could then locate the message file(s) and copy it/them to somewhere in visible memory.

The other possibility is that you are connecting to the laptop in Data Transfer mode (you should be prompted to select the mode on the phone when you connect the USB cable). This only lets you see the contents of the memory card. If "my folders" is on phone memory you won't be able to see it. Reconnect in PC Suite mode to also see sub-directories of "phone memory:\data".
