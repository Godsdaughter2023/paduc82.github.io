If you want to back up your Outlook.com data you must configure your account on a Mail Client.

Follow the steps below to do this:


Download Windows Essentials, it comes with Windows Live Mail, Family Safety, Writer, Outlook Connector pack, Microsoft OneDrive. 

When you download Essentials, you can choose custom install to install only Windows Live Mail depending on your preference.

Windows Essentials

Once Windows Live Mail is installed and configured, just sign in to your account and wait for the program to download your emails.  

When finished downloading you should have an exact copy on your Hard Drive. 

For more info about configuring your account

How do I add an account to Windows Live Mail


After doing this you have a copy of your emails on your hard drive that is synced with the Outlook.com server.

mccbleue say:

An NBU file is created by the Content Copier program - this is only a backup of phone data to restore to the phone after an update or repair or to restore your data to a different phone.

 

To export your messages into a file readable by Excel, you need the Communication Centre program (click the yellow envelope in PC Suite and this opens in message mode, then go to the Edit menu and select "Select All". Then go to the File menu and select Export - enter a file name and press the green tick logo to complete.

 

To open this file in Excel, you will need to make sure Excel is searching for all files or .CSV files, if it is only looking for .xls files it won't find it.

cricfancy say:

Hi interpreter,

 

First of all i dont have 6300 device with me :(  If your phone supports export/import option in Nokia Communication Centre,  As per Nokia PC Suite 7.1.18.0's help file see the following

 

 

Exporting messages to a text file

You can export messages from your device into a single text file on your PC in the following file formats:

  • comma-separated values (.csv)

  • vMessage (.vmg)

  • multimedia message (.mms)

 

Note:

If you want to change the list separator used with .csv files, you can do it in the Regional and Language Options dialog box in the Windows Control Panel.

When exporting multimedia messages, select the .mms file format to include message attachments in the export file.

 

To export messages, do the following:

  1. In the Messages view, select one or more messages for export. If you select multiple messages, export is only possible in the .csv format.

  2. Select File > Export. The Save As dialog opens.

  3. In the dialog, enter a file name and select where the exported file will be stored.

  4. Click Save.

 

The export file is created to the target location.
