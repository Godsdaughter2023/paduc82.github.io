You cannot use a Windows 7 Pro [Multiple Activation Key - MAK] with Windows Anytime Upgrade. Volume License distributions of Windows do not support Anytime Upgrade.

Please note that Anytime Upgrade does not support OEM, MSDN-Academic, Volume License (student faculty licenses). If you are in posession of any of these media, you will have to back up and do a custom clean install. If you wish to do this, please backup your Windows 7 Home Premium or whichever edition you currently have installed. Certain licenses for Windows distributed to students at the tertiary level are not perpetual, meaning you can't continue using the software after completing a course or graduating.

 

You will need to purchase a retail Anytime Upgrade or consider doing a custom clean install.

How to do an Anytime Upgrade.

First, make sure the edition of Windows 7 you are upgrading from is already activated (if its not you will run into complications and you might have start all over by doing a clean install). Click Start, type Anytime Upgrade, click the option to purchase a key. Wait while the Anytime Upgrade page is loaded, click the Windows 7 Professional Upgrade to buy it.

Click Next, fill out your contact information (it is important you do this, since the product key will be emailed to you as a backup just in case you need to reinstall it) enter your credit card information, click Next to begin the upgrade, wait while the key is verified, accept the license agreement, click upgrade, wait while the software upgrades, (it might take 10 minutes or more depending on if updates are needed), your computer will automatically restart, after restarting, Windows 7 will notify you its updating the computer, the computer will restart one more time automatically and the upgrade will be completed, a windows will pop up notifying the upgrade was successful and your computer is ready to use, click close, you should be upgraded to Windows 7 Professional your files, programs and settings preserved, proceed to activate Windows 7 Professional.

amit_sun say:

Hi,

Thank you for writing to Microsoft Community Forums. 

When the latest version of Microsoft Visual C++ is already installed, the installer will detect it and block the installation for the lower version sometimes.

Method 1:  Uninstall the higher version and install the lower version first.

  1. Uninstall any Microsoft Visual C++ higher then 2010, to do that.

  2. Press Windows + R key on your keyboard together.

  3. Type in appwiz.cpl and hit enter.

  4. Search for all the higher version of Microsoft Visual C++ and right click and uninstall it.

  5. Install Microsoft Visual C++ 2010 first and then install the other higher version in ascending order.

If you still face the issue, reply to this post with more information.

Hope it helps.

dcbaker say:

Issue has been resolved.  I was able to turn on "Sync My Settings".  Found the fix in the following thread:

http://answers.microsoft.com/en-us/windows/forum/windows_10-security/unable-to-add-microsoft-account/6fe10660-0b32-45d7-b422-b14dc9a144fa

Since I had logged into the computer with my Windows account first the credentials were already cached (in registry).  I had to delete my user account from registry (see below).

      HKEY_USERS\.DEFAULT\Software\Microsoft\IdentityCRL\StoredIdentities\<Microsoft Account>

Once you delete the account make sure your are logged in as your domain account.  Then add and then add your account back in by going to Start > Settings > Accounts >"Add A Microsoft Account".

Then go to Settings > Accounts > Sync Your Settings and click the "Sync settings" switch.
