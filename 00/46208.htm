lemp say:

Manually deleting files seldom guarantees that you have removed all traces of your personal information from a computer.  There are basically three options for "sanitizing" an old computer prior to disposing of it.  Which one you pick depends on how functional you want the computer to be after it leaves your hands.

1.  Remove and physically destroy the hard drive.  A variant of this is to remove and keep the hard drive to use as a backup drive or extra storage for your current computer.  After this step, the computer is no longer functional until the new owner installs a new hard drive and installs an operating system.

2.  Completely "wipe" the hard drive.  This is best done using a bootable CD (or bootable USB flash drive, if your computer supports that) and a suitable utility.  One of the easiest to use is Darik's Boot and Nuke (DBAN).  To use this method, one would download the DBAN ISO file (LINK -- download button at bottom of page) and use ImgBurn (http://www.imgburn.com/) to create a bootable CD or Rufus (https://rufus.ie/) to create a bootable USB flash drive.  Then put the CD in the CD drive and restart the computer.  Then follow the on-screen directions.  After this step, the new owner must install an operating system of his or her choice.

3.  Use the method provided by the computer's manufacturer to "restore to factory conditions."  After this step, the computer is still functional.  The new user will have to run Windows Update to install all of the (very many) updates released by Microsoft after the date the computer was manufactured.  The new user will also have to remove all of the manufacturer's "bloatware" and free trial software (the free trial periods will have long since expired). 

For directions, see the list of computer manufacturers at the end of Bill Smithers' post here:  https://answers.microsoft.com/en-us/windows/forum/all/forgot-my-windows-password/f2a684b2-4867-44ab-815e-f3a3d81e670a 

A variant of this method is to do a "clean install" of the version of Windows for which you have a valid product key (typically, on the COA sticker on your computer).  Directions for this follow the list of computer manufacturers at Bill Smithers' post linked above.

If you are disposing of a "many years" old laptop which has Windows 7 (or Vista or XP), I recommend option 2.
