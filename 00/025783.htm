Nokia 6700 USB cable connectivity problem -fixed-

Ok, got my 6700 today. Everything worked like a charm, until I tried to connect it to my PC using the supplied USB cable. I downloaded and installed the latest Nokia PC suite which supports this phone (7.1.30.9) and connected the phone. Vista was unable to install a driver, and the phone was listed in the device manager as 'unknown usb device'. The phone display reported 'no response from host' after plugging in the usb cable. 

 

I searched online for a solution, and tried the following:

 

- repaired the install of the Nokia connectivity cable driver: no success.

- deinstalled the nokia pc suite, ran the pc suite cleaner, rebooted and reinstalled the pc suite: no success.

- threw away the unknown usb device in the device manager and plugged the cable back in so that Vista would try to install the driver again: no success.

- tried other USB slots (not behind HUB): no success.

 

At this point I was so frustrated that I decided to take out the supplied 1gb micro sd card and transfer some data to it using a card reader. When I looked in the root of the sd card, I noticed that there was an executable called pc-suite.exe there, among some other files. I decided to delete those files to see if they were causing the problems, I deleted the following files:

 

- autorun.inf

- launcher.exe

- pc-suite.exe

- qf

 

Replaced the micro sd card in the phone, plugged in the usb cable and voila: connection! Everything worked like a charm.

 

During my online search I noticed I wasn't the only one experiencing connectivity problems with the 6700. Maybe this fix will help other people as well, that's why I decided to post it here. ;-)

How to use Camera Function in Microsoft Excel - Microsoft Community

ketul_patel say:

How to use Camera Function in Microsoft Excel

Technical Level : Intermediate

Summary

Article on using Camera Function in Microsoft Excel


Details

Using the Camera Function in Microsoft Excel

The camera tool allows you to take a snapshot of any selected range of data, table, or graph, and paste it as a linked picture.

The pasted snapshot can be formatted and resized using picture tools. They can be copied and pasted into Word and PowerPoint documents as well.

Enabling the camera function in Excel 2010:

In order to use this feature, you need to make the Camera button appear on the Excel Quick Access Toolbar, because it is not present by default. Head over to Quick Access Toolbar and from the small drop down button, click More Commands...

Select Commands Not in the Ribbon

Scroll down the alphabetical list of commands and find Camera, click Add>> button and Click OK to close the Excel Options dialog box.

The camera icon now appears in the Quick Access Toolbar at the top of the screen.

To use the camera, select the portion of the datasheet for which you want to take a snapshot of, and click the Camera button.

To paste the image, left-click the area where you would like the snapshot placed.

Now when you will select this image Picture Tools tab will appear, from where you can apply different styles and designs.

Apply the design and styles which best suites your worksheet image. You can apply shadowing, 3d effects, rotation, colours etc.

Well, that's how you can use Camera Function in Excel. Try it out yourself!!!
