SOLUTION---  Hi Giridhar J.  I've identified the embarrassingly simple fix for the problem.  For me, these steps rectified the problem and cleared ineffective settings that are now allowing my browser to run much faster.  I hope this is a help others that are having similar issues.  Steps two and three below do not appear necessary to correct the primary problem, but I perceived they helped improve browsing performance and decreased, though did not eliminate, the frequency of the issue.

1) Duplicate the error at will by going into Control Panel, Network and Internet, Network Sharing Center, Troubleshoot Problems, and run the "Internet Connections" troubleshooter.  You will receive the error "Windows cannot automatically detect this network's proxy settings."  After taking the steps below, you will repeat this procedure and will not receive this error.

2) Reboot the router by entering the router's IP address in a browser window and identify the "reboot" option.  This appeared to clear conflicting settings that made browsing more effective.  Restart the machine.  This helped browsing performance quite a bit but did not completely eliminate the problem.

3) Reset WiFi settings by going to Control Panel, Network and Sharing Center, select your Wi-Fi connection to the right of the "Connections:" label (write down the name of your network), select "Disable" in the Activity section.  This will disable the wifi setting and you will no longer have network and internet access.  Restart the machine.  Go into Control Panel, Network and Sharing Center, Change Adapter Settings.  Right click your network and select "Enable."  Wait several seconds and the machine will re-connect to the network. 

4) MOST IMPORTANT:  De-select all LAN Settings.  In a browser window select Tools, Internet Options, Connections, LAN Settings, and make sure all boxes are unselected, including "Automatically Detect Settings."  If you have this particular box checked, you are able to duplicate the Proxy Settings error listed in step one.  Unchecking this box eliminates the error.  You can do this same procedure using a different method by going into "Settings" from the charms menu, select Change PC Settings, Network, Proxy, and switch the "Automatically Detect Settings" switch to "off."

5) Repeat the procedure from Step One and you will no longer receive an error.  You should also no longer encounter the "Page Not Found" error while browsing.
