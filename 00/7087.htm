Based on your examples the following will work. Copy the list to a Word document, one item per paragraph and run the following Word macro.

Sub ExtractAccountNumbers()
Dim oPara As Range
Dim oDoc As Document
Dim vAccount As Variant
Dim strAccountNum As String
Dim strText As String
Dim i As Long
    strText = ""
    For i = 1 To ActiveDocument.Paragraphs.Count
        Set oPara = ActiveDocument.Paragraphs(i).Range
        oPara.End = oPara.End - 1
        vAccount = Split(oPara.Text, "\")
        strAccountNum = vAccount(UBound(vAccount))
        strAccountNum = Replace(strAccountNum, "[", "")
        strAccountNum = Replace(strAccountNum, "]", "")
        strAccountNum = Replace(strAccountNum, "-", "")
        vAccount = Split(strAccountNum, "_")
        If InStr(1, strAccountNum, "'") = 0 Then
            strAccountNum = vAccount(0)
        Else
            strAccountNum = vAccount(UBound(vAccount))
            vAccount = Split(strAccountNum, "'")
            strAccountNum = vAccount(UBound(vAccount))
        End If
        strText = strText & strAccountNum & vbCr
    Next i
    Set oDoc = Documents.Add
    oDoc.Range.Text = strText
lbl_Exit:
    Set oPara = Nothing
    Set oDoc = Nothing
    Exit Sub
End Sub

Almost all people are trying to replace the parts of a formula by dismembering and reassembling.

This is often a difficult and complicated way, one can easily make a mistake.

But when we want to replace a part, why not use the REPLACE function from VBA?

This a an easy and safe way.

You've done the first step, show the formula in the immediate window.

The next step is to make the formula compatible with VBA, means you must double any " sign inside the formula, means this formula

  =IF(A1="","a","b")

becomes

  =IF(A1="""",""a"",""b"")

We can combine that with the first step, use a sub and get the VBA code line ready to use:

Sub Formula2VBA()
  'Shows a VBA compatible string of the formula in the immediate window
  Debug.Print "MyFormula = """ & _
    Replace(ActiveCell.Formula, """", """""") & """"
End Sub

The next step is to replace the parts (the compete argument!) in the formula with a placeholder, means your formula above becomes

  =IF(H3=@REF_A,H3,IF(@REF_B=0,"""",@REF_B))

and the result is really simple to read, just a nested IF formula.

Now we can replace @REF_A and @REF_B with the references that we need and the formula works.

And we can use 2 tricks to build the references, see the code below.

Andreas.

Sub Example_ExternalRangeAddress()
  Dim MyFormula As String, Ref As String

  'This is the basic version of the formula with placeholders
  MyFormula = "=IF(H3=@REF_A,H3,IF(@REF_B=0,"""",@REF_B))"

  'Create references in this file with the RANGE object
  Ref = Sheets("Staff Data").Range("A12:A25").Address(External:=True)
  'Replace the placeholder
  MyFormula = Replace(MyFormula, "@REF_A", Ref)

  'Create references to other closed files with a function
  Ref = ExternalRangeAddress("L3", "Sheet 1", _
    "I:\Project Completion Report\Project Completion Report.xlsx")
  'Replace the placeholder
  MyFormula = Replace(MyFormula, "@REF_B", Ref)

  'Note:
  'When you refer to a file that doesn't exist, _
    Excel prompt you for the file when you write the formula into the sheet!
  'When you refer to a file and the sheet doesn't exist, _
    Excel prompt you for the sheet when you write the formula into the sheet!
 
  'Store the formula into the sheet
  Range("L3").FormulaArray = MyFormula

End Sub

Function ExternalRangeAddress(ByVal CellAddress As String, _
    Optional ByVal SheetName As String, Optional ByVal FileName As String) As String
  'Create a formula reference string for external (closed) files
  Dim fso As Object 'Scripting.FileSystemObject
  If FileName <> "" Then
    If SheetName = "" Then
      Err.Raise 5, "ExternalRangeAddress", "Argument 'SheetName' is required!"
    End If
    Set fso = CreateObject("Scripting.FileSystemObject")
    ExternalRangeAddress = "'" & fso.GetParentFolderName(FileName) & _
      "\[" & fso.GetFileName(FileName) & "]" & SheetName & "'!" & CellAddress
  ElseIf SheetName <> "" Then
    ExternalRangeAddress = "'" & SheetName & "'!" & CellAddress
  Else
    ExternalRangeAddress = CellAddress
  End If
End Function

To explain a little further: A .prn file contains the bytes that Word would otherwise send directly to your printer, through the printer driver. That is not readable by anything other than the printer driver. Theoretically, you could send the .prn file to the printer and it would print out the full formatted document, but I'm not sure exactly how you'd do that. Create a group 
Log in
People
Click (+)New V   


New group
Enter group name 
Type / Add appropriate contacts
Save

The steps in adding and deleting members in your Groups are as follows:

Go to your People page.
Locate and click the group name in your contact list.
Click Edit in the command bar on top of the page.
To delete a member of the group, click the "X" button at the bottom right of your contact's picture.
To add a member, type the contact's name in the Add member box and click the desired contact from the drop down option. 
Click Save.

Send group emails 
Log in 
Mail
Click (+)New
In the To box type your name /emailaddress

In the BCC box, type the Group name and click.

Clicking X  will allow you to remove contacts you don't want included.
