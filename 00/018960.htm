A meta refresh automatically reloads an HTML Web page after a specified amount of time, which is a useful tool on websites that publish constantly updated information.  For example, if you are visiting a Web page that has a stock ticker, or watching a live sports event, the information is being updated as the action progresses.  Instead of having to keep hitting the refresh button in the browser, the meta refresh code (META tag)  automatically keeps the page fresh and current, based on the time limit the site developer has written into the HTML code.

Problem is, on pages that don't require this, you get puked out right in the middle of something....... particularly frustrating when you are filling out forms.

You can do this with normal cross-referencing to the Table captions (i.e. not creating new bookmarks) by adding the following to the Field code:

\# 0

With your example do normal XRefs to the caption for Table 1 and Table 20, select the caption for Table 1, right mouse click, Toggle Field Code (or Shft+F9) and you will see the ref code for the table caption, somethine like { REF _Ref225905 \h }, add the above text within the code, like the following:

{ REF _Ref225905 \h \# 0 }

Press F9, to refesh and you should only see the number "1", do the same to Table 20, to diesplay "20".

Then add the "Figures" and "-" text as appropriate.

Figures { REF _Ref225905825\# 0 \h }, { REF _Ref225905827 \h \# 0}, and { REF _Ref225905829 \h \* lower }.

Figures 1, 3, and figure 4.
