teylyn say:
Hello,

you can do two things:

1) if you only show the data values as the labels, format the data in the source table not to show zeros. For example, if your number format is 0.00 change it to

0.00;;;

Then zero values will not show in the source data and also not in the labels.

2) if you want to show the data values and the category label, use a formula to create the labels, for example

=IF(B2=0,"",B2&", "&A2)

where B2 is the data value and A2 is the category label. Then download the free XY Chart Labeler tool and use it to assign the new cells as the data labels. If you don't want to use the tools, create the standard labels, then select each label individually, hit F2, enter a = sign and click the cell with the formula above.

cheers, teylyn
