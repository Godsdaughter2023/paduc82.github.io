Create Gmail labels and make Entourage folders

GMAIL'S RECOMMENDED IMAP CLIENT SETTINGS

Sending:

Do NOT save sent messages on the server. If your client is sending mail
through Gmail's SMTP2 server, your sent messages will be automatically
copied to the [Gmail]/Sent Mail folder.

DO save draft messages on the server. If you want your drafts in your mail
client to sync correctly with Gmail's web interface, set your client to save
drafts to the [Gmail]/Drafts folder.

Deleting:

Do NOT save deleted messages on the server. Messages that are deleted from
an IMAP folder (except for those in [Gmail]/Spam or [Gmail]/Trash) only have
that label removed and still exist in All Mail. Hence, your client doesn't
need to store an extra copy of a deleted message.

Do NOT save deleted messages to your [Gmail]/Trash folder because this will
delete a message in all folders.

Do NOT save deleted messages to your [Gmail]/All Mail folder as some clients
will try to empty this folder and ultimately fail. This can lead to delayed
mail access or excessive battery consumption on a mobile device.

Junk mail and spam:

Do NOT enable your client's junk mail filters. Gmail's spam filters also
work in your IMAP client, and we recommend turning off any additional
anti-spam or junk mail filters within your client. Your client's filter will
attempt to download and classify all of your existing messages, which may
slow down your client until the process is complete.

Thanks to your feedback on Windows 10, we found an issue on systems with a specific make/model of Bluetooth radio that caused loss of mouse or keyboard connectivity and file transfer failures. The problem can be fixed by updating your driver (instructions below).

This update only addresses radios with Hardware Id USB\VID_8086&PID_0189. Instructions on how to find your Hardware Id are below.

How do I find my Hardware Id?

  1. Right-click the Start menu, and select Device Manager.

  2. Expand Bluetooth

  3. Right-click on Intel(R) Centrino(R) Wireless Bluetooth(R) 3.0 + High Speed Adapter and select Properties

  4. Select the Details tab, and select the Hardware Id Property

  5. Verify that one of the values includes VID_8086&PID_0189

How do I update my driver? (if your system has Hardware Id USB\VID_8086&PID_0189)

  1. Right-click the Start menu, and select Device Manager.

  2. Expand Bluetooth

  3. Right-click on Intel(R) Centrino(R) Wireless Bluetooth(R) 3.0 + High Speed Adapter and select Update Driver Software…

  4. Select Search automatically for updated driver software

Issue Details:

The issue is due to an incompatibility issue in a filter driver for the radio interfering with normal L2CAP connectivity.  

 

If you are still experiencing connectivity issues, please let us know, and file feedback in the Feedback Hub with details about the issues you're seeing.

Hi,

try this code

(make a copy before)

each time you run the code

all sheets will be deleted, except from main sheet called:  'Inventory List'

Sub SplitData_NewShts()
'Oct 31, 2014
Const myCol As String = "D" '<<< Type of Inventory, in Column D
Const shName As String = "Inventory List" '<<< Source Sheet name
Dim ws As Worksheet, newWS As Worksheet
Set ws = Sheets(shName)
Dim v As Variant
v = Array("New", "Used", "Trade-in", "Consignment") '<< sht names
Dim r As Long, c As Long, x As Long
Application.ScreenUpdating = False
Application.DisplayAlerts = False
For Each sh In Sheets
If sh.Name <> shName Then sh.Delete
Next
Application.DisplayAlerts = True
r = ws.Cells(Rows.Count, myCol).End(xlUp).Row
c = ws.Cells(1, Columns.Count).End(xlToLeft).Column
ws.AutoFilterMode = False
For x = 0 To UBound(v)
ws.Range(ws.Cells(1, myCol), ws.Cells(r, myCol)).AutoFilter Field:=1, Criteria1:=v(x)
Set newWS = Worksheets.Add(after:=Worksheets(x + 1))
newWS.Name = v(x)
ws.Range("A1").Resize(r, c).SpecialCells(xlCellTypeVisible).Copy Range("A1")
ActiveSheet.UsedRange.EntireColumn.AutoFit
Next x
ws.AutoFilterMode = False
ws.Activate
Range("A1").Select
Application.ScreenUpdating = True
End Sub
