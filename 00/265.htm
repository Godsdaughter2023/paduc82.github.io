divide text into paragraphs - Microsoft Community

patriciawalsh say:

divide text into paragraphs

My task is to divide a body of text into numbered paragraphs.  The first word of each paragraph is noted by being in boldface and underlined therefore my task is to  " Find each single underlined boldfaced word then build a numbered list out of all the paragraphs.  I am thinking she wants me to use the "find" button which I did for the first word but what am I supposed to with it after that?  I would normally just put the cursor in front of each word and hit enter to create a paragraph then number it but apparently that is incorrect.  They only info I can find on paragraphs is for after you have created the paragraph.   Any assistance would be greatly appreciated.

P. 

jay freedman say:
This task can be done in two steps: Separating the paragraphs, and making them numbered.

It's actually easier if you reverse that order. Put the cursor in the single big paragraph, and apply the List Number style to it. That will start the paragraph with the number 1.

Now open the Replace dialog (the Replace button is just below the Find button, or use the shortcut Ctrl+H). Follow these steps exactly:
  • With the cursor in the Find What box, press Ctrl+B and then Ctrl+U. Below the box you should see the notation "Format: Font: Bold, Underline". The box itself will still be empty, meaning that it will look for anything having the bold+underline formatting.
  • Put the cursor in the Replace With box and type in these four characters (the ^ character is Shift+6 on US keyboards):
              ^p^&
              This adds a paragraph mark before each occurrence of bold+underlined text.
  • Click the Replace All button.
You may have to go back to the beginning of the original paragraph and delete an "empty" numbered paragraph mark. Also skim through the rest of the document, and check that the numbering hasn't been applied in places it shouldn't be.

jay freedman say:
There are a couple of possible reasons for this behavior.

The easiest one to verify or rule out is that Office might think it hasn't been activated, and has gone into "reduced functionality mode", which is essentially a viewer. Open Word, click the Help menu, and click Activate Product. If the activation is OK, you'll see a message box saying "This product has already been activated." Otherwise, you'll be asked to activate online or by telephone.

If Office is activated, then the next thing to check is whether the document is locked. Click Tools and click Protect Document, which opens a pane on the right side. If you see a button that says Stop Protection, click it. You might be prompted for a password.

If the protection appears in every new document, then you probably have protection applied to the Normal.dot template. If so -- and especially if it requests a password that you don't know -- then follow the instructions in http://word.mvps.org/FAQs/AppErrors/BlankDocNotBlank.htm to create a new Normal.dot from scratch.
