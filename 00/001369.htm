cbarnhorst say:

One additional thought.  I call the upgrade to get Windows 10 for free the entitlement install.

It is unique, one time, and must precede any other type of installation you will do.  It must be the first and it must be an upgrade because Setup has some business to do with your old OS before it goes away.  

During the entitilement install Setup confirms the qualifying OS (which is why it must be running) and writes the entitlement ticket and puts it in the system root along with the other Windows 10 files.  This gets recorded in the firmware and prepared for transmission to Microsoft's activation servers when Windows 10 connects to the internet later.  Then Setup finishes writing files to the system drive and does a restart.  

This hands off control of the computer to Windows 10 and the installation continues in its second phase.  Windows 10 connects to the internet following this phase and activates.  More precisely it tells Microsoft that it has activated and what edition is now entitled and that gets recorded on the servers.  At that point the computer is registered as a Windows 10 device.  Thereafter you can perform a clean installation of the same Windows 10 edition without needing a product key because the computer is entitled to activate the software you are installing.

jay freedman say:
Files whose names start with the characters ~$ are "owner files", a type of temporary file that Word uses to know that the corresponding document file is open. If another user asks to open the same document, Word sees the owner file and allows the second user to open only a read-only copy. This is explained in detail in the Owner Files section of http://support.microsoft.com/?kbid=211632.

You should not delete an owner file while its document is open. When you close the document, Word is supposed to delete the owner file automatically, but sometimes they get left behind. Then you can clean them up (http://www.gmayor.com/what_to_do_when_word_crashes.htm).

If seeing these temporary files on your desktop bothers you, the best solution is not to store documents on your desktop. Instead, put them in folders, typically in the My Documents folder, where you don't usually look -- out of sight, out of mind. If you want the convenience of opening documents from desktop icons, make shortcuts to the documents (right-click the document file in its folder, and choose Send To | Desktop (as shortcut)).

For i = r To 2 Step -1

End Sub
