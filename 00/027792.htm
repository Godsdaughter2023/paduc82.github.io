    tylerridenour3 say:
    Hi LesleySullivan,

    I'm Tyler, an independent advisor trying to help.

    There are various methods that can get your messages in the Junk folder even though you have marked a Safe Sender. It exists an infobar notification above the message which warns you whether the message has been filtered out by the Outlook Junk email filter.

    If it has been filtered out by Outlook, then the Infobar will contain the following;

    ”This message was marked as spam using the Outlook Junk E-mail filter.”

    When you see this message, you will need to troubleshoot your Junk email settings in Outlook. Make sure you have the latest updates installed and you may want to try turning off the Junk E-mail filter to find out where the messages end up now.     Opera 4 is part of the firmware on the C3. It is an integrated application. Opera 5 is a standalone application that you can install for yourself.

     

    WAP browser is not the only way to install an application. If, for some reason you cannot install applications using the browser, use the OVI suite or copy the application JAD and JAR files to the memory card.

     

    Opera 4 and 5 have different hardware requirements, so some phones support Opera 4, some support Opera 5.  Note that while Opera 5.1 works without problems on the C3, the phone is not even listed on the supported models page for Opera 5.

     

    If Nokia decides to include Opera 5 in a firmware upgrade, Opera Mini 5 will be available as a integrated app.
