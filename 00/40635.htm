4. You will now see a list of your installed Windows Updates. If you click on any one of the "Successfully installed on (date)" links , you will find more details for that update.

Now, Windows updates will try to install the Anniversary update only twice on any device. There won’t be any third time that Windows update will try.

  1. Make sure you have 8GB of free space

  3. Launch MediaCreationTool.exe
  4. Click on Upgrade this PC now
  5. Select Keep personal files and apps
  6. Wait for the installation to complete.
