tim liu msft say:

Hi Jim,

Per my understanding, it's difficult if you want to use some method such as GPO to deploy a word template to both Windows and Mac OS devices. My suggestion is that you can send the template to them, and then they add the template to their Word:

For Mac Users.

Open that template. Click File>Save as template>use the default Templates folder. If he wants to create new document from this template, click File>New from template>select that template.

For Windows Users.

Create a template folder on their local drive, for example D:\Tim Liu\Test File. Open Word>File>Options>Save>enter the path to Default personal templates location.

When they want to use this template, click File>New>Personal tab>find that template.

I think it's also convenient for those users as they don't need to change the default template. They can easily create new document with this template whenever they want to.
