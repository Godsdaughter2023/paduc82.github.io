lennart_52 say:

How do I reference "author (year)" instead of "(author, year)"?

I'm using the build-in citation system of word (APA style). Sometimes I want to reference "author (year)" and other times "(author, year)". But I can't seem to find any way to control that. It always defaults to "(author, year)".

When using "edit this citation" I can add a page number or suppress parts of the citation, like the author, which then leaves me with only "(year)". This kind of works, but is impractical because then I need to type the author's name manually to get to "author (year)" undoing the whole advantage of the reference system in the first place (having a central system to manage and change your references consistently).

Is there any way to fix this? How do I cite "author (year)" in my text, without hassle or manual vulnerabilities if I need to change a reference?

macropod say:

To get the "author (year)" referencing you want, you need to:

• insert the default (author, year) reference;

• copy & paste the 'author' part of the reference so that 'author' precedes the '('; then

• edit the citation to omit both the author and title from it.

That's all Word has ever provided. Pretty lame, but that's it.
