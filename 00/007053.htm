shane devenshire say:

Steps for creating the userform:

1. In the VBE choose Insert, UserForm

2. In the Properties window on the lower left (if you don't see it choose View, Properties Window) change the Caption property to a name you want to display at the top of the dialog box = userform, optionally, change the Name property to something like frmPrint (no spaces in the name property)

3. Click on the userform in the right hand area and the toolbox should be visible. 

4. Click the Button tool  and then click on the user form where you want one of the buttons.

5. With the button selected, change its Name property to btnPrint, change its Caption property to Print.

6. Repeat step 5 for a Print Preview button and a Cancel button

7. Double-click the Print button and add the code the following code to the form's module

Private Sub cmdCancel_Click()
    Unload Me
End Sub

Private Sub cmdPreview_Click()
    Unload Me
    Selection.PrintPreview
End Sub

Private Sub cmdPrint_Click()
    Selection.PrintOut
    Unload Me
End Sub

You will add one line in your regular module to launch the dialog box:

frmPrint.Show

mike h.. say:

Hi,

 

Sumproduct doesn't work across multiple sheets so we need to improvise with; in this case, COUNTIF inside a SUMPRODUCT function.

 

Create a list of the sheet names that you want this to apply to I used M1:M8. Change this to whatever you want in three places in the formula. The sheet names must all be spelt exactly the same as they are on the sheet tab. You can increase the size of this range if 8 worksheet isn't sufficient.

 

=SUMPRODUCT((COUNTIF(INDIRECT("'"&M1:M8&"'!B2:K367"),"*"))*(COUNTIF(INDIRECT("'"&M1:M8&"'!a2:a367"),">=" &Stats!H1))*(COUNTIF(INDIRECT("'"&M1:M8&"'!A2:A367"),"<=" &Stats!H2)))

 

EDIT... I should have added that the list of sheet names must be on the same sheet as the formula.

You can add retangles anywhere you want, but the connectors will only be available within the drawing canvas.

Turn the drawing canvas on or off  
  1. On the Tools menu, click Options.
