jennifer bri say:

Hello,

CHKDSK (Check Disk) creates and displays a status report for a disk based on the file system. It also lists and corrects errors on the disk. Used without parameters, CHKDSK displays the status of the disk in the current drive.

Upon checking, this issue is common after a CHKDSK process but your files were not actually deleted. They seem to be hidden in a found.000 folder. For you to view hidden folders, follow the steps below:

  1. Click on Start then go to Control Panel.
  2. Type folder into the search bar and select Show hidden files and folders.
  3. Click on View and under Advanced Settings locate Hidden files and folders, and then fill in the bubble for "Show hidden files, folders, and drives".
  4. Uncheck the box next to Hide protected operating system files.
  5. You should be able to see the found.000 folder, open it.
  6. You will find many folders labeled as dir0000.chk, where the 0000 is in ascending order. Each of these folders contain groups of files that were recovered.
  7. Drag and drop all these files into their correct folders.

Let us know if the steps above worked for you so we can assist you further, if needed.
