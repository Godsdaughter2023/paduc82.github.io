kfri86 say:

Animations in Sway's Presentation Mode

We have been using Sway to develop internal staff newsletters, which has proven to be a fantastic tool allowing engagement to increase using new technology. It's also been receiving excellent feedback from readers impressed with the 'new look', given our previous newsletters were out of date and often fragile (an in-house system built using HTML).

Some feedback we have received has been less complimentary about the 'animations' that can delay the viewer from going through a newsletter — particularly when the newsletters are lengthy in nature.

I'd suggest to Microsoft that we should have the option to be able to have no animations during Presentation modes, so the end user doesn't have any delay — similar to how owners view it in 'Preview' mode, where each arrow click just opens the new story.

Can this be considered?

jason (office engineer) say:

There are three options in the Layout pane and the third one is optimized for presentations.  Are you referring to the animations that occur only in this layout or are they other animations that happen in all layouts?  If the former, it is possible to speed them up by doing an extra navigation after the first one is done (either with the arrow keys, spacebar, touch swipe, or clicking the navigation arrows).  The first navigation will start the transition and the second navigation, if done before the animations are complete, will jump to the end of the animation.  It is double the number of navigations, but you can get through the Sway much faster.

Another possibility is the button in the lower right next to the navigation buttons that opens a screen allowing you to quickly jump directly to any section in the Sway.

Hope this helps!  And definitely let me know if I misunderstood what you're asking for.

Jason

kfri86 say:

Hi there,

Yes that's the style I'm referring to. It seems to be a bit inaccessible to suggest to users to double click to speed through the pages.

Sway is a fantastic tool, but this 'animation' on this style doesn't seem particularly user-friendly — especially when we want to engage with audiences who will no doubt have a busy calendar and will want information quickly before they need to move on to their next task.

I'd suggest that as an option, that the 'Presentation' mode should have no animations — simply with each click will open up the new card with no animations... would be much more user friendly and keep people wanting to read than getting bored of waiting or having to double click each time a new card comes up.

It should definitely be an option for users to decide if they want animation or not.

Folks,

So far, Microsoft Word has allowed me to elegantly satisfy all requirements for a book that I am creating for a client, except for this one:  The book consists of chapters.  On the first page of each chapter, the header and footer look different than the other pages within the chapter.  How can I set up this one time, automatically, without having to put a break in front of each chapter, and then a break after the first page of every chapter?

Ray.

hansv mvp say:

You only need a section break before the start of each new chapter. Tick the check box "Different first page" in the Options group of the Design tab under Header & Footer Tools; this is only visible when you're editing a header or footer.
