Hi Joshua,
 
After we sync our personal OneDrive for Business library to local drive. We will have a synced folder. The files in this folder is connected with the files in the cloud. The sync tool and the cloud server will scan from background to check if there are some changes for the files. When they find some files have been changed, they will sync the changes to each other (changes in cloud will be synced to the local synced file, changes in a local synced file will be synced to the cloud file).
 
So, normally it won't sync all the files every time. We just need to sign into the OneDrive for Business library and sync it to local drive once. We don't have to click the Sync now button after uploading or editing some files. This option will create a new sync folder and sync all the files again.

mike middleton say:
Any suggestions?

(A) What is half of 25?

(B) The RANDBETWEEN function returns only integer values. If you want uniformly distributed integer percentage values including the range endpoints, you could select 25 cells, format the cells as Percentage, and enter =RANDBETWEEN(10,77)/100 in some and =RANDBETWEEN(80,99)/100 in the others.

(C) If you want continuous-valued percentages instead of discrete integer-valued percentages, use =0.10+0.67*RAND() in some and 0.80+0.19*RAND() in the others.

(D) If you want the 25 results to be randomly ordered, you could select the 25 cells, edit copy, and edit paste special values. Enter =RAND() in 25 cells in an adjacent column. Select the range (two columns by 25 rows). Choose Data > Sort, specifying the RAND column as the "Sort by" column. Then delete the RAND column.

bill manville say:

You would need the HYPERLINK worksheet function.

It needs a text address of the cell to go to, e.g. "'[MyBoox.xlsx]Associate Details'!B22"
This can be produced by the ADDRESS function
ADDRESS(22,2,,,"[MyBoox.xlsx]Associate Details")

To get the row number you would use MATCH so

MATCH(X9,'Associate Details'!B:B,0)where X9 contains the employee number

Putting it together:

=HYPERLINK(ADDRESS(MATCH(X9,'Associate Details'!B:B,0),2,,,"[MyBoox.xlsx]Associate Details"),X9)

Using HYPERLINK does require a separate cell containing the value being looked up.

An alternative to using hyperlinks would be to have an event procedure which e.g. on double-clicking a cell would take you to the details for the employee whose name is in that cell.


In the ThisWorkbook module:


Private Sub Workbook_SheetBeforeDoubleClick(ByVal Sh As Object, ByVal Target As Range, Cancel As Boolean)
  Dim stName As String
  Dim vRow
  stName=Target.Value
  vRow = Application.Match(stName,Sheets("Associate Details").Range("A:A"),0)

  If Not IsError(vRow) Then
    Application.Goto Sheets("Associate Details").Cells(vRow, 2)
    Cancel = True
  End If

End Sub

bill manville say:

You originally said "cells in the org chart" and I assumed you had made an org chart using worksheet cells.

If you mean an org chart using shapes then the story is different of course.

You could attach a macro to each of the people-shapes in the org chart and have that macro extract the name of the person and use it to find the appropriate row in Associate Details and go there.

The macro could be something like this

Sub GoToName()
  Dim O As Object
  Dim stName
  Dim vRow
  Dim iChar As Integer
  Set O = ActiveSheet.DrawingObjects(Application.Caller)
  stName = O.Text
  ' trim down to just the first line
  For iChar = 1 To Len(stName)
    If Asc(Mid(stName, iChar, 1)) < 16 Then
      stName = Left(stName, iChar - 1)
      Exit For
    End If
  Next
  ' find the name in column A of Associate Details
  vRow = Application.Match(stName, ThisWorkbook.Sheets("Associate Details").Columns(1), 0)
  If Not IsError(vRow) Then
    Application.Goto ThisWorkbook.Sheets("Associate Details").Cells(vRow, 1)
  End If
End Sub

This assumes the name is on the first line of text within the object that was clicked.
The macro would go in a standard module (Alt+F11 to VB editor, Insert > Module, paste in the code above)
You would assign the macro to each name box on the org chart, e.g. by right click > Assign Macro > GoToName

You could even write a macro to do the assignment - e.g. if the name boxes are all Rectangle objects and all such objects need the macro assigned:

Sub Assign()
  Dim WS As Worksheet
  Dim R As Rectangle
  For Each WS In ThisWorkbook.Worksheets
    For Each R In WS.Rectangles
      R.OnAction = "GoToName"
    Next
  Next
End Sub
