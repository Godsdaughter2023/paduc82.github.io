"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Hi,

Thank you for writing to Microsoft Community Forums.  

We will need few more details so that we can narrow down to the root cause of the issue and assist you with appropriate troubleshooting steps.

1.    May I know, if you are connected to a domain network?

2.    Are you logged in with administrator account?

3.    Which version of Windows is installed on your computer?

4.    Are you getting this error on any other program of software?

You can try few of the troubleshooting below and check if this works for you.

Method 1: Unblock from Group Policy

1.    Press Windows + R key on your keyboard together.

2.    Type in gpedit.msc and hit enter.

3.    If you get the same error while opening gpedit.

4.    Search for Command Prompt from the start menu, right click and run as Administrator.

5.    Type in gpedit.msc and hit enter, and continue with the below steps

6.    Navigate to Computer Configuration > Windows Settings > Security Settings > Local Policies > Security Options.

7.    Double click on User Account Control: Rull all administrator in Admin Approval Mode.

8.    Under Local Security Settings, check Disabled.

9.    Click on Apply and Ok.

10.  Reboot the computer once and check.

If you are on Windows 10 home and you don’t have group policy, you can try the below method.

Method 2: Create a Registry key.

1.    Open command prompt with the above step 4.

2.    Type in regedit and hit enter.

3.    Navigate to Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System

4.    On the right pane, Right click and create a new Dword(32) key.

5.    Rename it User Account Control: Run all administrators in Admin Approval Mode

6.    Set he value to 0 and save.

7.    Reboot the computer and check.

Changing registry files incorrectly can cause Windows to stop working or make Windows report the wrong information.

I suggest you to back up your registry before making any changes.
