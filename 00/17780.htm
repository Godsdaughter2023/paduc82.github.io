mukherjee ashutosh say:

Try to disable the option to ignore other applications that use DDE.

Click on the Office button > Excel options > Advanced > General > uncheck 'Ignore other applications that use Dynamic Data Exchange (DDE).

deb r_416 say: This box is already unchecked.  Do you have any other ideas.

Take a backup of the data files within the XL Startup folders under program files, and in the user data folder.

Once you have backed up the files, delete them, and then check how Excel works.

Locations for the XL startup folder are as follows:

C:\Program Files\Microsoft Office\Office12\ XLSTART

C:\Documents and Settings\User_name\Application Data\Microsoft\Excel\XLSTART

scott .. b say:

Group Policy Editor (gpedit.msc) is not available in the Basic Windows 8.1 (home edition).  In any case, that would not be the way to go. 

Have you tried opening Tools>Manage Add-ons, looking under RealTech Inc (click "All add-on" at the bottom of the left pane) and seeing if RealPlayer is "enabled"

If you are comfortable with Registry Editor, you can temporarily override the grayed-out setting....

Open the Run box, enter regedit

Navigate to Computer Configuration\Administrative Templates\Windows Components\Internet Explorer\Internet Control Panel\Advanced Page\Allow third-party browser extensions

But, leaving it this way is a security risk, so once you do your download, go back and turn it off.

wondering_00 say:

scott .. b say:

Sorry....gave you the gpedit path.......

 Expand the plus sign next to HKEY_CURRENT_USER
 then Software
 then Microsoft
 then Internet Explorer
 then Main

 Set the value for Enable Browser Extensions. If the value doesn’t exist, you can create it by right-clicking Main and clicking New > String Value. Set it to no if you wish for extensions to be disabled. Set it to yes if you wish for extensions to be enabled.

scott .. b say:

OK........... What happens if you change it to NO, sometimes the terminology is confusing (O or 1) ? Always restart afterwards.

if the Registry is set correctly, then the reason it is grayed out is because your security software is blocking it (e.g., it has turned off Windows Defender app and does not want to allow 3rd party browser extensions because of the risk of allowing malware into the PC).  Couple of things you can try....

1)  Run IE as administrator.......

     right-click on IE icon on taskbar

     right-click (again) on words "Internet Explorer"

     Select "Run as Administrator"

2)  Change your security settings to default......

     Tools

     Internet Options

     Security (tab)

     UNCHECK the box "Enable protectd mode"

     Select "Reset all zones to default" 

     APPLY

     OK

3)  Change your account control settings.....

     Start screen, type account control

     Select "Change User Account Control Settings"

     move the slider all the way down

     Click OK

3)  Allow a program through Windows firewall....

     Start screen, type firewall

     Allow a program through the Windows firewall

     (Only works if your program is on the list)

NOTE:  All of these actions will put your computer at risk, so be certain to put them all back after you try the download.  If none of them work, you will need to contact your AV software provider (e.g., Norton, McAfee, Avast, etc.) and see what needs to be one.  Each has its own security protocol.
