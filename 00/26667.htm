How can I save a word or PDF document as an icon within a word - Microsoft Community

a lau say:

How can I save a word or PDF document as an icon within a word document?

How can I save a word or PDF document as an icon within a word document?  I was able to do that when I used window but now I can't with Mac

[Original title: Word]

a. user say:

In Mac Word 2016, you can do this with Word documents but not, as far as I know, with PDFs.

To do it with a Word document, click in the document where you want to put the icon (let's say it is called mydoc.docx", then

go to the Insert tab, Text dropdown, Object (or Object dropdown then Object...)

If you want to insert a new blank Word document, select Microsoft Word Document, check the "Display as icon" box, then click OK. A new Windows should appear with a blank document titled something like "Document in mydoc.docx"

Edit the document, then click File->Close. You should see an icon tiled "Microsoft Word Document" in mydoc.docx.

If you want to insert an existing Word document, in the Object dialog click the From File... button, select the file, but before you click Insert, click the Options button, and check the Display as Icon box.

If you want to embed the document (i.e. you will make a new, independent copy inside mydoc.docx), ensure the "Link to File" box is unchecked.

If you want to link to the file (i.e., if you double-click on the icon, Word should let you edit the original file), then ensure "Link to File" is checked.

Then click Insert. You should see an icon with the document's file name in mydoc.docx.

If you try this with a PDF in Mac Word 2016, what you will notice is that PDF is not mentioned in the list of object types in the Object dialog, and if you use the From File... button, Word actually inserts an image (probably of page 1 of the PDF) rather than an icon or an icon+link.
