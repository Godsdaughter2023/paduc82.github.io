Is there anyway to reset password??
Yes, there are lots of options built into Windows to help you get out of this predicament:
  • Reboot the machine with your Windows Repair CD, then use System Restore to set Windows back by a few days (works only if you changed the password recently).
  • Log on under your spare admin account, then reset the password to your usual account (works only if you set up a spare admin account).
  • Log on as Administrator in Safe Mode, then reset the password to your usual account (the password is usually blank).
  • Use the password hint (works only if you created a hint and if it is good enough to let you guess the password)
  • Check your paper records (works only if you jotted down the password somewhere)

greg carmack - mvp 2010-2020 say:

This is a good example of why it is important to always boot the installation media to delete all partitions down to Unallocated  Space before install, as illustrated in Clean Install Windows 10 Upgrade .  That is the only true Clean Install, because leaving partitions on the drive otherwise builds up a disk junkyard. 

For example, the boot partition should be the 100mb partition, but it has been cut out after the install with the boot files moved to Recovery as signified by the System Active labels being there now. 

The space used on Recovery also doesn't look like enough to still house Factory Recovery, which is useless after a reinstall anyway unless it's HP which retains the F11 hotlink.  So unless it's being used as the separate Windows 10 Recovery partition (which only needs to be 250mb) housing built-in repair functions (which I doubt) it is wasted space that could have been avoided by correctly deleting all partitions before install, to let Windows 10 decide what service partitions are needed. 

To answer your other question, the stray boot listing cannot be a  for the old install archived in Windows.old or any other way unless it is still intact in its own separate partition on the drive.  It is instead an artifact because the disk wasn't cleared correctly.  You can remove the ghost listing as I already wrote in msconfig>Boot tab or by installing EasyBCD (click Download - no Name or Email required) to highlight and delete it on the Edit OS Menu tab. 

If any further problems arise or if performance isn't perfect then you can always do it over and follow Clean Install Windows 10 Upgrade which assures the best possible install which stays that way as long as you stick with the tools and methods given.
