What you need is a data validation list using INDIRECT, here is and example:

Here are the steps for creating a dynamic data validation list:

1. Create three or more list in the spreadsheet.  For example:
    A                  B             C

Seattle         Toronto     US

Sorry.  Missed that you wanted a macro. You could use it as a macro as is.  But I think the following will run faster than translating the formula.

Please read the notes in the code.

Of course, it won't be dynamic as a formula based solution would be, unless you trigger it with event code.

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.

=============================================

Option Compare Text
Sub CopyData()
    Dim wsAll As Worksheet, wsATT As Worksheet
    Dim vALL As Variant, vATT As Variant
    Dim R As Range
    Dim I As Long, J As Long
    
Set wsAll = Worksheets("All People")
Set wsATT = Worksheets("Attended Event")

'Get ID columns
With wsAll
    vALL = .Range("A1", .Cells(.Rows.Count, "D").End(xlUp))
End With
With wsATT
    vATT = .Range("F1", .Cells(.Rows.Count, "F").End(xlUp)).Resize(columnsize:=4)
End With

'Collect the matching rows, in order
'Note:  Header row assumed
'    If no header row, then start count at 1 vs 2
For I = 2 To UBound(vATT)
    For J = 2 To UBound(vALL)
        If vALL(J, 4) = vATT(I, 1) Then
            vATT(I, 2) = vALL(J, 1)
            vATT(I, 3) = vALL(J, 2)
            vATT(I, 4) = vALL(J, 3)
            Exit For
        End If
    Next J
Next I

'write the results
Set R = wsATT.Range("F1").Resize(UBound(vATT, 1), UBound(vATT, 2))
With R
    .EntireColumn.ClearContents
    .Value = vATT
    .EntireColumn.AutoFit
End With
 
End Sub

Sub MatchID_InTwoShts()

r2 = ws2.Cells(Rows.Count, sCol_2).End(xlUp).Row

Hi,

The syntax of your SUMIF is incorrect.  Try something like this

=SUM(A:Z!AI4)-SUMPRODUCT(SUMIF(INDIRECT($M$4:$M$10&"!D4:D100"),"MTM",INDIRECT($M$4:$M$10&"!E4:E100")))

M4:M10 on the summary sheet has a listing of all worksheets in your workbook.

D4:D100 is the range on each worksheet in which you want to search for MTM

E4:E100 is the range on each worksheet which has the numbers that you want to sum

Adjust the ranges and worksheet names.
