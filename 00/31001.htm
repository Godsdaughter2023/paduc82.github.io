To Disable Login Screen in Windows 10 follow the below steps.

1. Press the Windows Key to bring up the Start Menu. Type “Run.” From the results, right click on the Run icon and select “Run as Administrator.”

2 Enter netplwiz into the Run dialog box and press enter. From the resulting application, select your user account.

3. Locate the Users must enter a username and password to use this computer checkbox and uncheck it.

4. You’ll be presented with a prompt asking for your login credentials. If you used your email ID to sign up, then the complete email address will be your default username. Enter your information and press OK.

5. Reboot your device.

    diana d say:

     

    Hi,

    In order to enable the subtitle feature in media player we need to rename a system file called mfpmp.exe. Follow these simple steps and you will get subtitles in your media player.

    Method 1:

    a.        Click on start.

    b.      In the search bar, type cmd.

    c.       Right Click on it when it is listed and run as administrator.

    d.       In command prompt, type the following and press Enter.

    takeown /f C:\Windows\System32\mfpmp.exe

     

    icacls C:\Windows\System32\mfpmp.exe /grant (user name):F

     

    For example, icacls C:\Windows\System32\mfpmp.exe /grant Pablo:F

     

    e.       You should be able to delete or modify the file now. If you still can't, you may need to reboot and try again. Try renaming the file to mfpmp.old.
