lemp say:
Another one said I should go into Device Manager, uninstall all USBs then restart.

This generally works ... but can be quite dangerous unless you have a PS/2 keyboard.

I also assume that you have tried all of the other USB jacks on the computer; tried a known-good USB device (e.g., a flash drive) in the USB jack where you usually plug in your iPhone; and tried your iPhone in another computer (I know it's rare for a Windows error message to actually mean what it says, but sometimes hardware does fail).

Does the iPhone charge when connected to the USB port (I suspect not)?  This recently happened to me, but a reboot fixed it all up.  I guess that hasn't happened for you.

I take it that you didn't see any error messages in Device Manager or you would have reported them.

You can uninstall all of the USB devices that aren't there as follows:

  • Open a Command Prompt window (Start > Run > cmd > OK)
  • Enter the following command and press Enter

         set devmgr_show_nonpresent_devices=1

  • Start the Device Manager by typing the following into the same Command Prompt window

         devmgmt.msc

  • In Device Manager, click the View menu and select "Show hidden devices"

  • You should see a category right after "Non-Plug and Play Drivers" named "Other devices."  Chances are there will be a big yellow question mark next to this.  When you expand "Other devices" you should see several unidentified "USB devices."  Uninstall all of these.

  • Reboot (with the iPhone NOT connected).

Note: Bidirectional Support is Disabled by the Upgrade to Win 8.1

          This can be found by right clicking on the printer in Devices and Printers,

           Select Printer Properties, and then the PORTS tab... It is near the bottom


1) Printer must be On for it to show up in Devices and Printers after the Upgrade  - Just this one time.

2) Now UNHOOK the Printer


3) REMOVE Lexmark Printer and Lexmark Fax in Devices and Printers.


4) STOP the PRINT SPOOLER.

Printer Preferences, Properties, etc. I unchecked a "Shared" on the first window.


6) After REBOOT, STOP Print Spooler AGAIN.


7) Reinstall Software\Drivers ... Hook Printer back up when Prompted.


8) Unhook Printer again and REBOOT ... This step isn't necessary if Bidirectional Support is already ENABLED.


Skip to step 10 if Enabled.


9) Enable Bidirectional Support and Reboot
