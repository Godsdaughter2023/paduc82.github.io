With any luck, these methods should help find the culprit behind your computer’s unwillingness to boot.

1. Try Windows Safe Mode

Safe Mode is an alternative boot scheme that starts your computer with a minimum of software. The modified boot process can bypass driver and software problems.

2. Check Your Battery

Battery issues could cause boot problems. It’s worth testing an alternate charger cable to see if it solves the problem.

3. Unplug All Your USB Devices

A serious issue with Windows 10 updates is that sometimes your computer won’t boot because of a conflict with a USB device.

4. Turn Off Fast Boot

There’s a setting inside of your BIOS or UEFI called Fast Boot that allows Windows 8 and newer operating systems to boot very rapidly by preloading drivers. Unfortunately, Windows updates can break Fast Boot compatibility. Fortunately, Fast Boot can be toggled on and off through your BIOS.

5. Try a Malware Scan

Malware is a major cause of an unbootable computer. The common method of dealing with malware that causes boot problems is an anti-malware program that can boot from a USB or DVD drive.

6. Boot to the Command Prompt Interface

You’ll need to have Windows 10 on a bootable disk or USB drive to carry out the procedure, so use another computer to set that up before you get started.

7. Use System Restore or Startup Repair

System Restore is a utility that allows you to return to a previous Restore Point when your computer was working normally. It can remedy boot problems that were caused by a change you made, rather than a hardware failure.

The reply below about an extra entry in the device manager was useful.   You don't have to go through all that Singh recommended which is pretty complicated.

Looking into the device manager under the printer/fax, you will probably find a suspicious looking entry you have no need of.   the HP 8610 fax and printer are there and OK .... leave those two alone.

I found an unknown entity device in the printer/fax section of the device manager and took it out.  Remove it.

Then...this is important to do... I reran the Windows Update Troubleshooter, and now, since the device you never needed anyway, was removed from the devices, the troubleshooter will see that device is gone and that the update is not needed and clear up the problem completely at last.

I also ran CC Cleaner on the registry and cleaned out any excess uneeded files afterwards, but I don't think that step was really necessary.

Hope this will help you clear your problem, finally...rather simple to do this.   Get to device manager any way you know how, or via the control panel, if you can find that.

It is sometimes unclear in Windows 10, the best way to bring up the control panel and device managers....I had a shortcut to control panel on the desktop and that lists device manager inside.
