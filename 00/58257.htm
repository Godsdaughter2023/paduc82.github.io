canadian tech say:

Assuming this was done using the built-in Win7 image backup software.....

You will need to boot the system using either your original Win7 install disk or a System Repair disk.

After booting you should be able to find your way to the option to recover your system using an image.  Starting down that trail, should lead you to an option to choose the source of the image.   If that image is still good, you should be able to use it.
