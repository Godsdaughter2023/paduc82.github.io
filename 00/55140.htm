a. user say:

Good question. You should know that, in general, technology companies give words whatever meaning suits their current marketing goals. Which may have nothing to do with what the words mean in standard English. This makes it very hard to understand what a technology term actually means.

Re-installing is simply that: installing again. There are two ways to install again: in-place and clean install.

When you re-install 'in place' the original software remains on your computer and gets over-written. When you upgrade Windows 8 to Windows 10, for example, you are installing 'in place' because Windows 8 remains on your computer and gets over-written by Windows 10.

When you do a 'clean install' you remove the original software and then replace it with new software (or a fresh version of the same software.) When you clean install Windows 10, for example, you remove the previous version of Windows and replace it with Windows 10.

The crucial thing about a clean install of an operating system is that your hard drive is erased - that's how the existing operating system is removed. (Technically, the partition containing the OS is erased.) That means everything you didn't backup is gone. So if you're ever thinking of doing a clean install of Windows, make sure you backup your documents, photos, bookmarks, etc. because everything gets erased.

So far, so good? One more little detail: Microsoft calls a clean install a 'custom install.' Go figure.

A reset is actually a clean install. However, for Microsoft there are two types of reset: a 'remove everything' reset and a 'keep my files' reset.

A 'remove everything' reset is just like a regular clean install: Your hard drive is erased and a fresh copy of Windows is installed. Anything you didn't backup is gone.

A 'keep my files' reset is a clean install with benefits: First, Microsoft copies your Documents, Music, Pictures and Video folders to 'the cloud' (which in this case means a server operated by Microsoft.) Microsoft also copies certain folders and files that are needed to re-create your computer's user accounts. After that, your hard drive is erased just like in any other clean install and a fresh copy of Windows 10 in installed.

After Windows 10 is clean installed your user accounts are re-created from the folders and files previously copied and then Microsoft restores each user account's Documents, Music, Pictures and Video folders. In addition, and for no extra charge, Microsoft will re-install all your Windows Store apps and any data associated with those apps (like your ranking in games.) Microsoft can do this because they keep track of all your Windows Store apps. How nice of them - and nosey.

What does not get restored is any software that wasn't obtained from Windows Store and any of your personal data that wasn't stored in those 4 default folders.

Trivial point: In Windows 8, Microsoft called a 'keep my files' reset by the term refresh.

As Porky Pig would say: That's all, folks!

a. user say:

Okay, now you're talking about something very different. A re-install/reset is for putting Windows back onto the same hard drive. Now you're talking about moving it to a different hard drive.

This will actually be easy, however, because the hard drive manufacturers have you covered. What you need to do is make an image backup of your existing hard drive (more on that later) and then restore the image to your new SSD. If you buy a retail SSD (vs. an OEM SSD) the manufacturer should include imaging software and instructions on how to use it. There will be more explanations on the SSD manufacturer's website.

An image is an exact copy of your hard drive, compressed into a single file. (To be technically correct, blank areas of your hard drive are not copied.) When you restore the image to your new SSD, nothing changes. Your computer will look and work exactly the same as it did before, because it is the same computer, stored on an SSD instead of a HDD.

An image includes the drive's metadata, which describes the drive's configuration and makes it bootable. The metadata is not accessible from Windows, which is why you need the imaging software. You can't simply copy and paste everything from Windows.

If you want, go to the website of your proposed SSD's manufacturer and you'll find the instructions there. And BTW, imaging is also a fabulous way to backup. Imagine never needing to reinstall Windows and your software ever again. If you make an image every day, which is not too often, and your computer gets messed up by somebody's anniversary (just saying) you can always restore an image from the previous day and ta da! you've gone back in time like nothing happened.

For future refference, OE - XP questions should be asked here, not in  Windows 7 forum.

XP - Networking, Mail And Getting Online Forum
http://social.answers.microsoft.com/Forums/en-US/xpnetwork/threads

For your problem:

Error Number: 0x800C0133 = a corrupt Inbox.

Move any mail you wish to save to a local folder you create. Then delete the problem Inbox as follows.

Tools | Options | Maintenance | Store Folder will reveal the location of your Outlook Express files. Write the location down and navigate to it in Windows Explorer or, copy and paste it into Start | Run.

In WinXP, Win2K & Win2K3 the OE user files (DBX and WAB) are by default marked as hidden. To view these files in Windows Explorer, you must enable Show Hidden Files and Folders under Start | Control Panel | Folder Options | View.

With OE closed, find the Inbox.dbx and delete it. A new one will be created automatically when you open OE.

General precautions for Outlook Express:

Do not archive mail in the Inbox or Sent Items. Create your own user defined folders and move the messages you wish to save to them. Empty Deleted Items folder daily. Although dbx files have a theoretical capacity of 2GB, I recommend about a 300MB max for less chance of corruption.

After you are done, follow up by compacting your folders manually while working *offline* and do it often.

Click on Outlook Express at the top of the folder tree so no folders are open. Then: File | Work Offline (or double click Working Online in the Status Bar). File | Folder | Compact all folders. Don't touch anything until the compacting is completed.

Turn off e-mail scanning in your anti-virus program. It is a redundant layer of protection that eats up CPUs and causes a multitude of problems such as time-outs and account setting changes. Your up-to-date A/V program will continue to protect you sufficiently. For more, see:
http://www.oehelp.com/OETips.aspx#3
