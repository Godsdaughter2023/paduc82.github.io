Where Windows update changes a system file, its backup copy in** C:\Windows\WinSxS is also changed.

Where SFC "repairs" a system file, it copies in the backup from** C:\Windows\WinSxS.

This process is only referred to obliquely in SFC documentation [perhaps the clearest reference is in Using System File Checker (SFC) To Fix Issues - Ask the Core Team - Site Home - TechNet Blogs].
