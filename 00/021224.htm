rohn007 say:
Sorry I was under the impression you copied the file to the HD.

The second link does apply to you. It lists the places and types of file (bak, tmp, asd) where you may find an old copy. Take another look at it.

That brings up a separate point. DO NOT EDIT files directly in thumb / flash drives. Due to design issues on Thumb drives windows does not actually write files directly to them. Instead windows writes/saves the files to a cache on the local HD.  Only when a "block" on the flash drive can filled is the information moved to the flash drive. So if the file does not fill a block, the last piece of the file is sitting on the HD. If you forget to shut down the flash drive properly before pulling it out, you will leave part of the file behind on the HD, corrupting the file.

A "point" is a measure of distance.  1 point = 1/72 inch.  A pixel is just a bit of information on your screen.  Some screens have more "pixels per inch" than others.  Therefore, the relationship of the number of pixels per point is highly dependent on the monitor you have and the screen resolution you are using.  You should NOT be tying any of your formatting to pixel density.  Stick to points only for setting row size, column width and the position of objects.  When you print, it won't be dependent on your screen resolution.

suzanne s. barnhill say:

"Exactly" line spacing is always the defined number of points. "At least" line spacing starts with the defined number of points but can increase if the font size demands it. That is, if the "Single" line spacing would be greater than what is defined by "At least," then the line spacing will be greater.

 

So, for example, if you define line spacing of "Exactly 12 points" and use 10-pt Times New Roman, you will get 12-point line spacing. This will stay the same if you change the font size to 12 points. But if you use line spacing of "At least 12 points," it will be 12 points for 10-pt TNR, but if you use 12-pt TNR, it will increase to approximately 14.4 points because the built-in "leading" for that font is approximately 20% greater than the nominal point size.
