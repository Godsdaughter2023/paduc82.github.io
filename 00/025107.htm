Copying the color of a logo - Microsoft Community

a. user say:

Copying the color of a logo

Hi,

I am creating a logo and need to copy an existing color. When I click on the word, it does not bring up any color information. How can I find our what was used to create the color?

Thank you.

jay freedman say:

Word doesn't have the tool you need to copy the color (except for table borders, but that doesn't work on graphics). However, Microsoft Paint does have it, and it's part of Windows.

Copy the logo to the clipboard. Start up Paint (on the Start screen, just type Paint and click the resulting shortcut). Paste the logo into the blank canvas.

On the Home ribbon, click the color picker tool (the eyedropper button in the Tools group) and click the logo. The color will be displayed in the Colors group as Color 1.

Now click the Edit Colors button at the right end of the ribbon. In the dialog that opens, the numeric values of Red, Green, and Blue are shown in the bottom right corner. Write them down.

Back in Word, select the text or graphic where you want to apply the same color. For text, click the down arrow of the Font Color button on the Home ribbon; for a shape from the Insert > Shapes tool, click the arrow of the Shape Fill button on the Drawing Tools ribbon; or for a picture, click Color > More Variations on the Picture Tools ribbon. In any of these menus, click the More Colors choice and enter the Red, Green, and Blue values in the dialog (you may have to go to the Custom tab of the Colors dialog to see those boxes).
