danida_u say:

Hello

Mac devices running on OS X 10.8 and lower are no longer supported. The reason behind this change is to ensure that Skype is providing the best possible application experience and offering the latest technology to its users. In line to this, it is unfortunate but it is also inevitable to retire older versions of Skype and some Skype-enabled devices, to transition Skype from a peer-to-peer video and chat app to a more modern and more diverse mobile friendly cloud architecture. Please visit this help article for more details.

I understand that there will be a number of users who will be affected by this change but please do know that this is also a hard decision for Skype to make. This will benefit a larger group of uses in the long run. Skype users will experience enhanced quality, improved performance and better security.

Affected users can still use Skype for Web to send instant messaging though you need to have Mac OS X 10.9 and up to make calls. I suggest that you update your OS version If possible. Please check this link for the system requirements to run Skype.
