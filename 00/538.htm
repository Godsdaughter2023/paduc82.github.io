bernie deitrick say:

When working with special fonts, you should not rely on specific characters. Instead, you should use character numbers.  To figure out what you want, format an entire column for the font (wingdings) and in the first row, use the formula

=CHAR(ROW())

and copy down for 255 rows.  Then find the character you want, and use the row number like so:

Range("A1").Value = Chr(   insert row number here  )

like

Range("A1").Value = Chr(123)

So, for your three characters:

Sub ShowAll()
    With Range("B1:B3")
        .Font.Name = "Wingdings"
        .Font.Size = 11

        .VerticalAlignment = xlCenter
    End With
    With Range("B1")

    End With

hermannus say:

how to convert text to number

There are plenty of ways to convert numbers to text but only one to convert text to numbers (going one cell at a time). Is there a more rational way to convert sheet full of numbers(written as text) to numbers (that can then be used as such)
j.e. mcgimpsey say: Since you don't give an example of what "numbers(written as text)" means - i.e., do you mean the numeral stored as a text string ('3') or the number written as text ('three') - I can only guess why you think you have to go one cell at a time...

If the latter, as Bob indicated, you'll probably need either a macro or to do a bunch of Find & Replaces.

if it's the former, there are MANY ways to convert them. One:
  • Copy an empty cell
  • Select the 'text' numbers
  • Choose Edit/Paste Special, selecting the Values and Add radio buttons.
Another, if it's a single column:
  • Select the column/cells
  • Choose Data/Text to Columns
  • Click Finish
Another (if your values have the "smart tag" associated with the Number stored as text):
  • Select your values
  • Choose "Convert to Number" from the tag's dropdown.
