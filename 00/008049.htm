steven. s say:

Hello Ann222,

 

This could be a conflict with an add-on that keeps resetting the order of your favorites. You can test this by:

·         Click the Start Orb

·         Click All Programs

·         Click Accessories

·         Click System Tools

·         Click Internet Explorer (No add-ons)

 

Inside of Internet Explorer (No add-ons) change the order of your Favorites, then close and reopen Internet Explorer (No add-ons). If when you open Internet Explorer (No add-ons) the order is correct, then you just have to find which add-on was causing the problem to occur.

·         Close Internet Explorer (No add-ons)

·         Open Internet Explorer

·         Click Tools

·         Click Manage Add-ons

·         Click Toolbars and Extensions

·         Under Show: click All add-ons

·         Click any add-on

·         Press CTRL + A on your keyboard (this should select all add-ons)

·         Click Disable all

·         Click Close

 

Now all add-ons are disabled in Internet Explorer, next step is to enable them one by one.

·         Click Tools

·         Click Manage Add-ons

·         Click the first disabled add-on

·         Click Enable

·         Click Close

·         Surf a few pages

·         Move a few Favorites around (out of alphabetical order)

·         Close Internet Explorer

·         Open Internet Explorer

 

If you Favorites order is still as you left it before you close Internet Explorer, the add-on you enabled is safe to stay. Repeat the Manage Add-on process for the next disabled add-on until you open Internet Explorer and your Favorites are back into alphabetical order. Then the last add-on you enabled will need to be disabled/uninstalled/updated to avoid further problems.

 

Hope that helps, look forward to hearing back from you.

Function GetFileTAG(PathFileName As String, FileTAG As SHELLTAG) As Variant
Dim oShell As Shell32.Shell
Dim oFolder As Shell32.Folder, sPath As String, sFileName As String
Dim oFolderItem As FolderItem2

    sPath = FilePath(PathFileName)
    sFileName = Filename(PathFileName)
    
    If Dir(PathFileName) <> "" Then
        Set oShell = New Shell32.Shell
        Set oFolder = oShell.Namespace(sPath)
        Set oFolderItem = oFolder.ParseName(sFileName)
        
        GetFileTAG = oFolder.GetDetailsOf(oFolderItem, FileTAG)
    End If
    Set oShell = Nothing
    Set oFolder = Nothing
    Set oFolderItem = Nothing

End Function
