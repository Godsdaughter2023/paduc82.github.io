If you can use a VBA function, you can use this function that I wrote some time ago and it is part of my standard library.

Public Function SumOfSheets(RangeAddress As String, _
        ParamArray SheetNames() As Variant) As Variant
''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''
' SumOfSheets
' By Chip Pearson, *** Email address is removed for privacy *** , www.cpearson.com
' This sums the values in the cells specified in RangeAddress
' from the worksheets specified by SheetNames. RangeAddress
' should be a string containing the cell address to sum; e.g.,
' "A1:A10". The SheetNames ParamArray specifies the worksheets
' whose values are to be summed in either of two ways. First,
' it can be a list of worksheet names or index numbers:
'   =SumOfSheets("A1:A10","Sheet1","Sheet2","Sheet5")
'   Or
'   =SumOfSheets("A1:A10",1,2,5)
' In this example, A1:A10 on Sheet1, Sheet2, and Sheet5 will
' be summed.
' Or, it can specify an inclusive range of sheets. In this
' case, SheetNames(0) must be a ':' character, SheetNames(1)
' is the name or index of the first sheet and SheetNames(2)
' is the name or index of the last sheet. All sheets inclusively
' between SheetNames(1) and SheetNames(2) will be summed. E.g.,
'   =SumOfSheets("A1:A10",":","Sheet3","Sheet7")
'   Or
'   =SumOfSheets("A1:A10",":",3,7)
' In this example, range A1:A10 on all sheets between Sheet3
' and Sheet7 inclusive will be summed.
''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''

Dim Total As Double
Dim N As Long
Dim R As Range
Dim WS As Worksheet
Dim FirstWS As Long
Dim LastWS As Long

On Error Resume Next
' ensure SheetNames is present
If IsError(LBound(SheetNames)) Then
    SumOfSheets = CVErr(xlErrValue)
    Exit Function
End If
' ensure RangeAddress is included
If RangeAddress = vbNullString Then
    SumOfSheets = CVErr(xlErrValue)
    Exit Function
End If
' ensure RangeAddress is a valid range
Set R = Worksheets(1).Range(RangeAddress)
If Err.Number <> 0 Then
    SumOfSheets = CVErr(xlErrRef)
    Exit Function
End If

If SheetNames(0) = ":" Then
    ' working with range of sheets. ensure we
    ' have both start and end sheets
    If IsError(SheetNames(1)) Or _
        IsError(SheetNames(2)) Then
            SumOfSheets = CVErr(xlErrRef)
    End If
   
    ' get the first sheet index
    FirstWS = Worksheets(SheetNames(1)).Index
    If Err.Number <> 0 Then
        SumOfSheets = CVErr(xlErrRef)
        Exit Function
    End If
    ' get the last sheet index
    LastWS = Worksheets(SheetNames(2)).Index
    If Err.Number <> 0 Then
        SumOfSheets = CVErr(xlErrRef)
        Exit Function
    End If
   
    ' ensure first sheet is <= last sheet
    If FirstWS > LastWS Then
        SumOfSheets = CVErr(xlErrRef)
        Exit Function
    End If
   
    'loop and sum
    For N = FirstWS To LastWS
        Set R = Worksheets(N).Range(RangeAddress)
        If Err.Number <> 0 Then
            SumOfSheets = CVErr(xlErrRef)
            Exit Function
        End If
        Total = Total + Application.Sum(R)
    Next N
    ' return the result
    SumOfSheets = Total
    Exit Function
Else
    ' we have a list of sheets. loop and sum.
    For N = LBound(SheetNames) To UBound(SheetNames)
        Total = Total + Application.Sum( _
            Worksheets(SheetNames(N)).Range(RangeAddress))
        If Err.Number <> 0 Then
            SumOfSheets = CVErr(xlErrRef)
            Exit Function
        End If
    Next N
    ' return the result
    SumOfSheets = Total
End If
End Function

As parameters, you first pass in a string containing the address of the cells on each worksheet to be summed. For example, "A1:A10" to sum A1:A10 on all the specified worksheets. The remaining parameters indicate which worksheets to sum.  It can be a list of individual worksheets, identified either by Index or Name:

=SumOfSheets("A1:A10","Sheet1","Sheet3","Sheet5")
' or
=SumOfSheets("A1:A10",1,3,5)

This will sum A1:A10 on Sheet1, Sheet3, and Sheet5.

Or, you can specify a start and end sheet, identified by Index or Name, and all the sheets between the start and the end sheet (inclusively) will be summed. To do this, the first element of the SheetNames paramarray must be a colon (:) character, the next element the name or index of the first sheet, and the final element the name or index of the last sheet. For example,

=SumOfSheets("A1:A10",":","Sheet2","Sheet6")
' or
=SumOfSheets("A1:A10",":",2,6)

This will sum A1:A10 on all worksheets between (inclusive) Sheet2 and Sheet6.

Of course, the worksheet need not be named SheetN. Any existing worksheet name if valid.

If successful, the function returns the sum of values. It will return a #VALUE if no paramarray values are passed, or #REF if the RangeAddress or a Sheet specifier is invalid.
