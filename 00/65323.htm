Open Mobile hotspot settings (Settings > Network & Internet > Mobile hotspot)
Turn on mobile hotspot
Under "Related Settings", click Change adapter options
Take note of which adapter is connected to the internet and which adapter is your mobile hotspot (typically named "Local Area Connection* [#]")
Right click the adapter that is connected to the internet and click Properties
Go to the Sharing tab
Uncheck "Allow other network users to connect through this computer's Internet connection". If it is already unchecked, skip this step.
Click OK
Again, open Properties > Sharing for the same adapter.
Check "Allow other network users to connect through this computer's Internet connection"
(If you get a popup that says "Internet Connection Sharing is currently enabled...", click OK.)
Under "Home networking connection:", select the mobile hotspot adapter
Click OK

Press Windows key + X
Click Device Manager
Expand Network adapters
Right click then uninstall all virtual adapters

Open Start then type: regedit
Right click Regedit then click Run as administrator
Navigate to the following registry key:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WlanSvc\Parameters\HostedNetworkSettings\

Delete the following key:
HostedNetworkSettings

Restart then try again.

Run the Network adapter troubleshooter

Open Start > Settings > Update & security > Troubleshoot
Scroll down, select Network adapters
Click Run the troubleshooter

When complete, restart to see if it works.

If not, you can try updating both your Network adapter drivers or uninstall them.

Press Windows key + X
Click Device Manager
Expand network adapters
Right click your adapter, click uninstall

Exit Device Manager then restart

Windows 10 will re-detect them and reinstall them.

OR

Open Start, type: CMD
Right click CMD
Click Run as administrator

Type in at the prompt OR Copy and Paste these one at a time : (Hit enter after each)

netsh wlan set hostednetwork mode=allow ssid=name key=YOURPASSWORD

netsh wlan start hostednetwork

Exit Command Prompt

Open Start > Settings > Network & Internet > Wi-Fi > 'Change adapter options'

Select the new "Local Area Connection #..."

Right click this and select "Properties",

Choose the "Sharing Tab then check 'Allow other network Users to connect through this computer's Internet connection'

Click Apply then OK

a. user say:

Hi Trish,

Thank you for posting your query in Microsoft Community.

Let me answer your query regarding Microsoft account.

Are you referring to the Microsoft account that you used to sign-in to the computer? Do you want to remove the Comcast email account associated with Microsoft account and associate a different email account?

If yes, please be informed that you cannot switch the email account associated to a Microsoft account, i.e., you need to create a new Microsoft account (with different email account) and sign-in.

First, To convert the Microsoft account to local account, you can do the following: 
1.    Sign in to your Microsoft account on Windows 10. 
2.    Click Start and then Settings. 
3.    Click Accounts and Sign in with a local account instead. 
4.    Enter your Microsoft account's password, and click on Next. 
5.    Type in a Username, Password, and Password Hint that you want to set for your Local account and click Next. Click on Sign out and Finish button.

Now you may sign-up for a new Microsoft account (using new email account) by following the below article.

http://windows.microsoft.com/en-za/windows-live/sign-up-create-account-how

Now again switch your account from local user account to new Microsoft account (which you created by associating a new email account) by following the below link.

http://windows.microsoft.com/en-za/windows-10/getstarted-microsoft-account

Hope the information helps, do reply if you need further assistance.

Thank you.

shenan stanley say:

What is it?  It's a temporary area.  Things that need to write someplace as the user do so (usually) in the place you have given - thus the "temp" folder name.

Can you delete them?  Yes.

Should you delete them?  Not really - unless you just need the extra space (although I cannot imagine it being all that great in size.)

Could deleting them have consequences down the line?  Maybe.  Uninstallers sometimes look for those - although less now than in the past.

Need to cleanup the computer in a safe manner - to expose any of the obvious issues that may have developed over years of use?  (Most of the time the system is not actually all that much slower - it just feels that way.  However, the newer programs do generally use more resources than the original versions you were using four years ago - including things like Internet Explorer, Firefox, the various internet plugins like Quicktime, Flash, Shockwave, RealPlayer, etc...  If you started with Office 2002/XP and now have Office 2007/2010 - that's using more resources to do similar work as before - meaning your computer was not made for those applications demand levels.)

Other questions...

1) When was the last time you checked for updated hardware device drivers for the system manually (by visiting the manufacturer(s) web page and browsing their downloads for your model of component?)  In particular this can be useful for video card performance efficiency - by also motherboard chipset and network devices can sometimes benefit from an upgrade of the software (device drivers) that allows them to communicate with the operating system effectively.

2) Do you keep the system patched?  Not just Windows Vista itself - but the applications you have installed?  The other software vendors surely put out patches and updates for their products - have you kept up with those?

3) How do you backup the stuff important to you OFF the machine in question?  If the hard drive was to die right now (electrical surge caught the machine on fire and the hard drive literally melted to slag) - would you be worried about your music, pictures, documents, emails, contacts, databases, spreadsheets, serial numbers/product keys, installation media (for the operating system and other software) and the likes - or do you have that covered?  Could you easily switch to another machine for whatever reason?

4) What antivirus product do you utilize?  Is it a 'all-in-one' suite - or just a pure antivirus product?  Is it kept up-to-date - not just with definitions, but with the version of the product itself?
