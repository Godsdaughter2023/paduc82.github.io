dhookom say: Min/Max: I would have a small lookup table that stores Alpha, Beta,... with a numeric field/column that specifies the order. I would not rely on the text field.

First/Last: I would never use these

Expression: There are times when the aggregate functions only work with Expression. In these instances the Field: row would include the aggregate function

Where/Having: This is the criteria row(s). In a totals query HAVING should only be used in a non-Group By column. It would be appropriate to use the HAVING with columns like: Sum(Quantity) where you want to filter based on the Sum of the quantity.

Group By: Don't get in the habit of assuming the Group By will always provide the Sort By. If you upsize to SQL Server, this will bite you. Think of Group By as buckets. If you group by department, each department has its own buck that you count or sum values into. If you are familiar with Excel Pivot tables, these are like Row Headings.

don varnau say:

Hi,

Your Favorites/bookmarks shouldn't have been deleted. A reset (RIES) doesn't do that. Something else must have happened.

The easiest procedure might work. On the affected computers, go to ["C:\Users\your account name\Favorites"] Right-click Favorites> Properties. Previous versions. Try to restore a previous version.

With the 3rd computer *not* connected to the internet, copy the Favorites folder at

["C:\Users\your account name\Favorites"] to a Flash drive, USB hard drive, etc. and From Internet Explorer> File> Import and export> export Favorites to a file.

(Old article, but the IE8 instructions will work on IE9, 10 or 11)https://support.microsoft.com/kb/211089?wa=wsignin1.0

You can import that HTM file into other computers or copy Favorites to the Favorites folder. Copy Favorites/Internet shortcuts, don't just copy the folder or sub-folders.

Passwords...

Some, but not all passwords will be in Credential Manager. On the unaffected computer, backup your credentials.

http://www.7tutorials.com/how-backup-and-restore-windows-vault-passwords

http://www.7tutorials.com/credential-manager-where-windows-stores-passwords-other-login-details

You can copy that file to the other computers.

For passwords not in Credential Manager...

(This was easier in earlier versions of IE) If it's not installed, install Chrome.

http://www.google.com/chrome/

In Chrome click the 3 horizontal bars in the upper right> Bookmarks> Import bookmarks and settings>  etc.

After importing passwords, favorites, etc. into Chrome go back to 3 horizontal bars in the upper right> Settings> (at the bottom) Show advanced settings> Passwords and forms> Manage passwords.

You won't be able to import or copy those back into IE, but they will be listed so you can login to affected sites.

Don
