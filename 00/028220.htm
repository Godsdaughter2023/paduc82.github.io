suzanne s. barnhill say:

If you are seeing "footer section 15," then you have at least 15 sections, so you have definitely not deleted all the section breaks in the document. "Same as Previous" affects only the content of a header or footer, not the header/footer margin, which can be different in every section. You need to change that in the Position group on the Header & Footer Tools | Design tab.

The message you're getting often results when columns have a problem. The fact that you have at least 14 section breaks suggests that you may have some multi-column sections. With the insertion point in one of the section where you are experiencing difficulty (even if it is a single-column section), go to Page Layout | Page Setup | Columns and click More Columns... to open the Columns dialog. Click on one of the preset pictures (One, Two, Three) to get Word to check the "Equal column width" box. You can then change the number of columns again if needed. When that box is not checked, Word sets column width to an exact amount that doesn't adjust automatically when margins are changed, and this can cause problems especially if, for example, you try to change from Landscape to Portrait orientation, but you can get the error message when making any change in margins.
