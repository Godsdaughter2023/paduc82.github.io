About the only visible indication is if you return to the Home tab you'll see that the Font indicated is the one called for by the Theme you've chosen. Similarly, if you open one of the Color palettes you'll see that the Theme Colors differ from those previously offered.

If the workbook does have content, the Font used & the colors used for Shapes, SmartArt & Word Art, etc. will change according to what's called for by the newly selected Theme but only for those elements to which direct formatting changes have not been made. E.g., If the current Theme calls for Calibri & you choose a different Theme that uses Helvetica, you'll see the cell content change to Helvetica. However, if you first reformat cells to use a font other than what the current Theme specifies, the formatting of those cells will not change when you select a different Theme. The same is true for Theme Colors.

The one difference from prior versions that I've noticed in Excel 2016 is that the Row numbers & Column letters will reflect the font of the currently applied Theme.
