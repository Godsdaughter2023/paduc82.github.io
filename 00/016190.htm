nan02k say:

Got it. Searching through all shapes of the Presentation and filtering out the movies works:

//While iterating through all slides i:
objShapes = objPres.Slides[i].Shapes;
foreach (Microsoft.Office.Interop.PowerPoint.Shape s in objShapes) {
    if(s.Name.Contains(".wmv")){
      s.AnimationSettings.PlaySettings.PlayOnEntry = MsoTriState.msoTrue;
    }
}

steve rindsberg say:

Just a caveat or two: 

Not all movies will necessarily be WMVs (though certainly you may know that in your situation they will be)

In presentations from earlier versions, the shape name won't contain the name of the movie file, so you won't catch these.

Unless the presntation was created in PPT 2010, the movies won't be embedded, they'll be linked; that may open another approach to doing what you're after.  Something like this (you'll have to translate from VBA to .Net):

' for each shape oSh ...

If oSh.Type = msoMedia Then ' it's a media shape of some sort

If osh.MediaType = ppMediaTypeMovie Then ' it's a movie, not a sound or whatever

Debug.Print .LinkFormat.SourceFullName  ' full path to the linked movie file

bernie deitrick say:

You could use this formula in B2:

=IF(ISNUMBER(FIND("-->",A1)),"<font face=""Tahoma"">","") & A2 & IF(A3="","</font>","")

copied down to match your entries in A.  Copy and paste values, then delete A.

OR:

You could also use a macro - the first one removes the number and times, the second does not. Not sure which you wanted:

'First macro

Sub SuffixPrefixRemoveOtherStuff()
    Dim rngC As Range
    Dim lngC As Long
    For lngC = 1 To Cells(Rows.Count, "A").End(xlUp).Row
        If InStr(Cells(lngC, "A").Value, "-->") > 0 Then
            Cells(lngC + 1, "A").Value = "<font face=""Tahoma"">" & Cells(lngC + 1, "A").Value
            Cells(lngC, "A").ClearContents
        End If
        
        If IsNumeric(Cells(lngC, "A").Value) Then Cells(lngC, "A").ClearContents
        
        If Cells(lngC + 1, "A").Value = "" Then
            Cells(lngC, "A").Value = Cells(lngC, "A").Value & "</font>"
        End If
    Next lngC
End Sub

'Second macro

Sub SuffixPrefixLeaveOtherStuff()
    Dim rngC As Range
    Dim lngC As Long
    For lngC = 1 To Cells(Rows.Count, "A").End(xlUp).Row
        If InStr(Cells(lngC, "A").Value, "-->") > 0 Then
            Cells(lngC + 1, "A").Value = "<font face=""Tahoma"">" & Cells(lngC + 1, "A").Value
        End If
        
        If Cells(lngC + 1, "A").Value = "" Then
            Cells(lngC, "A").Value = Cells(lngC, "A").Value & "</font>"
        End If
    Next lngC
End Sub
