jay freedman say:
In all versions of Word on all versions of Windows, the dialog you get when you click the Properties button in the Print dialog is provided by your printer driver, not by Word. It's different for every brand of printer, and sometimes for different models within the same brand.

I think you'll find that the driver supplied by Windows 8 is at least a different version than the one supplied by Windows 7. As an example, I have a Windows 7 computer and a Windows 8 computer both networked to the same printer. The Windows 7 driver's version is 6.1.7601.17514, while the Windows 8 driver for the same printer is version 2.27.2.10 (obviously from a different sequence). That kind of difference could cause the Properties dialog to differ as well.

I'd be very surprised if the driver doesn't include some way to print with less ink, but the wording might have changed from "draft" to "fast" or "economy" or something like that. Visit the website of the printer manufacturer to ask where they put the draft option in the Windows 8 Properties dialog.
