john - project say:

Scott,

No problem, but there are multiple things that need to be set up for it to all work as you need.

First, you need to create a 7 days a week calendar. I don't remember if that is one of the default calendars or not. If not, create it under Project/Properties group/Change Working Time. Create your new calendar based on the standard. Set the work hours as needed.

Second, go to Project/Properties group/Project Information and set the calendar to your 7 days a week calendar. Doing it this way you do NOT need to set the calendar for each task. That will also automatically set the base calendar for each resource as the 7 days a week calendar.

Third, if your 8-5 workday is the standard 8 hours you won't need to change the definition of a day. However, if that workday is actually 9 hours (i.e. no lunch break), then you will want to go to File/Options/Schedule group and change the definition of a day to be 9 hours. If you enter durations in weeks, you should also change the definition of a week to be 56 hours, or 63 hours if you have 9 hour days.

Finally, in order for the Gantt display to show Saturday and Sunday as working, double click on the timescale to bring up the Timescale window. Hit the "non-working time" tab and set the calendar to your 7 days a week calendar.

That should do it. Hope this helps.

John
