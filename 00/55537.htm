Hi Devashish,

To answer your query, I would like to inform that during the process of Clean Installation, when the process fails there are two scenario's that you will incur.

1. If the process fails, you will get an option to return back to the old operating system. The system will indicate where the process failed and that it will roll back.

2. The other is when, you computer will be stuck and frozen while the upgrade. Here you will lose the ability to perform any task. If such is the case, you might lose files and folders which are essential.

However, when you perform Clean Install of the OS and it succeeds, it will erase all you personal files and folders that have been saved in the PC. Hence creating a backup for the same is an essential part of the process.

1. Open a Command Window in Administrator mode:

Close the Command Prompt Window when the job is finished.

One caveat: you need a Windows DVD to enable SFC to make more than minor repairs.
