Method 2:  Rebuild the Media Player library and check if the issue persists.

 

a)   Close Windows Media Player. 

b)   Stop the media sharing service. Go to Start and type services.msc and press Enter.

c)    Right-click on the Windows Media Player Network Sharing Service and select Stop.

d)   Navigate to c:\Users\”your username”\AppData\Local\Microsoft\Media Player.

Note:  You may have to unhide hidden files to do this. Refer: Show hidden files

e)   Now delete the files named “CurrentDatabase_***.wmdb” and “LocalMLS_*.wmdb.”

f)    Start the media sharing service as mentioned in steps b) and c).

g)    Now start Windows Media Player to rebuild the library.

 

Method 3:

a.      Open Windows Media Player 12.

b.      Go to Tools menu.

c.      Select Advanced>>Restore Media Library.

 

You may also try to include the files to the Music Library and check if that helps to retain the music files in Media Player Library. Follow the link given below.

Add items to the Windows Media Player Library

http://windows.microsoft.com/en-US/windows7/Add-items-to-the-Windows-Media-Player-Library

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7
