doug robbins - mvp office apps & services (word) say:

What is the purpose of placing all of the text with the footnotes into Publisher?

If they were endnotes, you could use a macro with the following code to convert them into ordinary text which could then be copied and pasted along with the rest of the document

    Dim aendnote As Endnote
    For Each aendnote In ActiveDocument.Endnotes
        ActiveDocument.Range.InsertAfter vbCr & aendnote.Index & vbTab & aendnote.Range
        aendnote.Reference.InsertBefore "a" & aendnote.Index & "a"
    Next aendnote
    For Each aendnote In ActiveDocument.Endnotes
        aendnote.Reference.Delete
    Next aendnote
    Selection.Find.ClearFormatting
    Selection.Find.Replacement.ClearFormatting
    With Selection.Find.Replacement.Font
        .Superscript = True
    End With
    With Selection.Find
        .Text = "(a)([0-9]{1,})(a)"
        .Replacement.Text = "\2"
        .Forward = True
        .Wrap = wdFindContinue
        .Format = True
        .MatchWildcards = True
    End With
    Selection.Find.Execute Replace:=wdReplaceAll

If you really need to get the document into Publisher, the only way that I could see to do it would be to use Snagit to print the document and then in Snagit save the output as a .jpg file (which will create a separate file for each page in the document) and then insert those pictures into Publisher.

You can get a free trial of Snagit from www.techsmith.com, but it is well worth what it costs for some of things that you can do with it.

That's going to be tedious, but I believe that it is probably the only way to do it.
