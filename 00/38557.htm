jeeped say:
I have already used the "associate a file type with a specific program" tool but every time I choose the program I want, it doesn't change to it, it still just says outlook

That would have been my first approach as well. The important sympton here is the disabled "Always use the selected program to open this kind of file" option in the Open with... Choose default program... dialog. You will need to check and possibly modify one or two registry settings to gain this option. The following are user settings. Look at the bottom for global settings.

Method 1

Click Start or tap the Windows key and type or paste in cmd but do not hit <enter>. Locate cmd from the list above and right click then select Run as administrator. Make sure that Quick Edit Mode is enabled (ALT+<spacebar>, Properties, Options tab, Edit Options, Quick Edit Mode) and use ALT+<spacebar>,E,P to paste this in:

First for the current user,

REG ADD HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer /v NoFileAssociate /t REG_DWORD /d 0

Next to make it system wide,

REG ADD HKLM\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer /v NoFileAssociate /t REG_DWORD /d 0

You may be asked to overwrite the value. Tap Y. You can close the windows once you see The operation completed successfully.

Method 2

Click Start or tap the Windows key and type or paste in regedit but do not hit <enter>. Locate regedit from the list above and right click then select Run as administrator. Navigate to:

First for the current user,

HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\Explorer

Next to make it system wide,

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\Explorer


Figure 1.0 - Registry Policy Explorer

If the Explorer key doesn't exist, create it. Within the Explorer key create a new DWORD value named NoFileAssociate. Give it a value of 0 (zero). If it already exists, just give it a value of 0 (zero). Close RegEdit.

That should restore your own user profile's ability for setting the default programs permannently with Explorer's Open With command.

Dump files are usually found at c:\windows\minidump. You may need to change your settings in Windows to be able to see the files. To show hidden files select the keyboard shortcut Win+S, type File Explorer Options in the Search box and click Folder Options in the drop down menu. Select View, Advanced Settings and verify that the box before "Show hidden files and folders" is checked and "Hide protected operating system files" is unchecked. You may need to scroll down to see the second item. You should also make certain that the box before "Hide extensions for known file types" is not checked.
