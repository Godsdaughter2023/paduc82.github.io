Excel treats numbers as numbers. When it doesn't have a clue how data should be split between X and Y, it guesses. Often the guess doesn't match your expectations.

You can give Excel a clue if you put a label in the cell above the column of Y values, and leave blank the cell above the column of X values. ossiemac say:

The substitute function referenced by the link supplied by Jim Cone is case sensitive. If the characters being searched for must match the case in source data then no problem.

However, if you want to search with case non sensitive then it is necessary to include the Lower or Upper function to the source data to match the case of the character being searched.

Note that with "mutate" and "MUTATE" that the one on row 3 the lower case characters are NOT found.

With "jaunt" and "JAUNT" the characters are found in both rows 4 and 5 because the source data is converted to lower case in the formula.

To return the characters in column B in alphabetical order simply place them in alphabetical order in the formula with the string to be returned when the IF function returns true. See rows 6 and 7 where the "U" is first but the returned characters are in the order of the text to return (ie. "au")

Note that you can replace the LOWER function with the UPPER function if you want to search for uppercase characters but it does not matter if the source is upper or lower.

The formulas can't be copied from the uploaded screen snippet so here are the formulas in column B so you can copy them.

Cell B2   =IF(AND(LEN(A2)-LEN(SUBSTITUTE(A2,"a",""))=1,LEN(A2)-LEN(SUBSTITUTE(A2,"u",""))=1),"au","")

Cell B3   =IF(AND(LEN(A3)-LEN(SUBSTITUTE(A3,"a",""))=1,LEN(A3)-LEN(SUBSTITUTE(A3,"u",""))=1),"au","")

Cell B4   =IF(AND(LEN(LOWER(A4))-LEN(SUBSTITUTE(LOWER(A4),"a",""))=1,LEN(LOWER(A4))-LEN(SUBSTITUTE(LOWER(A4),"u",""))=1),"au","")

Cell B5    =IF(AND(LEN(LOWER(A5))-LEN(SUBSTITUTE(LOWER(A5),"a",""))=1,LEN(LOWER(A5))-LEN(SUBSTITUTE(LOWER(A5),"u",""))=1),"au","")

Cell B6     =IF(AND(LEN(LOWER(A6))-LEN(SUBSTITUTE(LOWER(A6),"a",""))=1,LEN(LOWER(A6))-LEN(SUBSTITUTE(LOWER(A6),"u",""))=1),"au","")

Cell B7      =IF(AND(LEN(LOWER(A7))-LEN(SUBSTITUTE(LOWER(A7),"a",""))=1,LEN(LOWER(A7))-LEN(SUBSTITUTE(LOWER(A7),"u",""))=1),"au","")

myall_blues say:

I see you are using Office 365 but do you have a desktop version of Excel? If so you could use a custom VBA function as per below.

Usage: =LETTERSINWORDS(inword, letter1, [letter2], [...])

inword is the input word - it can be a literal string or a cell reference

letter1, letter2 etc are the letters to search for, letter2 onwards are optional. It will accept a range, a single cell reference, a literal string value or an array of values.

e.g.

=LETTERSINWORDS("mutate","a","u") ->result "au"

=LETTERSINWORDS("magnanimous","a","u") ->result "u". Two a's are ignored

=LETTERSINWORDS(A1,B1:C1)

=LETTERSINWORDS(A1,{"a","u"})

etc.

Steps:

1. Open a blank workbook

2. Press ALT+F11 to open the VBA editor

3. In the project window on the left, right click on the workbook name and choose Insert->Module

4. Paste the code below in the resultant window

5. Save the workbook as an add-in type (.xlam)

6. Load the add-in into Excel (File->Options->Addins)

Function LETTERSINWORDS(InWord As String, ParamArray textn() As Variant) As String
    '
    ' Search for letters in words, with criteria that word contains exactly one of each letter
    '
    Dim i
    Dim rng
    Dim strChr() As String
    Dim iLen As Long, FirstPos As Long, j As Long
    '
    ' Find letters in input word
    '
    For Each rng In textn
        If IsObject(rng) Or IsArray(rng) Then
            For Each i In rng
                FirstPos = InStr(1, InWord, i)
                If FirstPos <> 0 Then
                    If InStr(FirstPos + 1, InWord, i) = 0 Then
                        LETTERSINWORDS = LETTERSINWORDS & i
                    End If
                End If
            Next
        Else
            FirstPos = InStr(1, InWord, rng)
            If FirstPos <> 0 Then
                If InStr(FirstPos + 1, InWord, rng) = 0 Then
                    LETTERSINWORDS = LETTERSINWORDS & rng
                End If
            End If
        End If
    Next
    '
    ' Sort the output in alphabetical order
    '
    iLen = Len(LETTERSINWORDS)
    If iLen > 0 Then
        ReDim strChr(1 To iLen)
        For i = 1 To iLen
            strChr(i) = Left(LETTERSINWORDS, 1)
            LETTERSINWORDS = Right(LETTERSINWORDS, Len(LETTERSINWORDS) - 1)
        Next i
        For i = 1 To iLen - 1
            For j = i + 1 To iLen
                If strChr(j) < strChr(i) Then
                    LETTERSINWORDS = strChr(i)
                    strChr(i) = strChr(j)
                    strChr(j) = LETTERSINWORDS
                End If
            Next j
        Next i
        LETTERSINWORDS = ""
        For i = 1 To iLen
            LETTERSINWORDS = LETTERSINWORDS & strChr(i)
        Next i
    End If
End Function

hansv mvp say:

The most important choice is between Office 365, which is a subscription version, and Office 2019, which is a one-time purchase.

Office 365 Personal provides Word, Excel, PowerPoint, Outlook, Publisher and Access, plus cloud storage for 1 user (but on multiple devices for that user - PC, Mac, tablet, phone) for $60 per year.

Office 365 Home provides the same applications for up to 6 users (on multiple devices) for $80 per year.

The subscription versions are updated every month, with new features from time to time.

Office 2019 Home & Student provides Word, Excel and PowerPoint for 1 user on 1 PC or Mac for a one-time cost of $120. You'll get security updates, but no feature updates.

See https://products.office.com/en/compare-all-microsoft-office-products?tab=1
