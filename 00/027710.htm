Glad it works.  It is useful for returning the last delimited
substring.

You need to analyze from the inside out.

=TRIM(RIGHT(SUBSTITUTE(A1,"/",REPT(" ",99)),99))

SUBSTITUTE(A1,"/",REPT(" ",99)):    (Note the <space> between the
two quote marks).  Replace the delimiter with a large number of spaces
-- some number greater than the longest substring.  99 usually works,
but it is conceivable you might need more.

RIGHT(...,99)    :    Return the rightmost 99 characters.  This will
include the substring as well as a number of leading spaces.

TRIM(...)    :    Get rid of the leading spaces (as well as any
trailing spaces).  So if more than a single consecutive space is
required within the substring; or if trailing or leading spaces are
required, this method will fail.  But that circumstance would be rare.

More robust, with a delimiter that is not a space, might be:

=MID(A1,FIND(CHAR(1),SUBSTITUTE(A1,"/",CHAR(1),LEN(A1)-LEN(SUBSTITUTE(A1,"/",""))))+1,99)

hansv mvp say:

Yep - the scroll area is not saved with the worksheet. You have to set it each time the workbook is opened. You can use VBA code to do this automatically:

  • Press Alt+F11 to activate the Visual Basic Editor.
  • Locate ThisWorkbook in the Project Explorer pane on the left hand side (under Microsoft Excel Objects).
  • Double-click ThisWorkbook.
  • Enter the following code and modify it to suit your needs:


Private Sub Workbook_Open()
    Worksheets("MySheet").ScrollArea = "A1:H25"
End Sub


  • Switch back to Excel.
  • Save the workbook as a macro-enabled workbook (.xlsm, .xlsb or .xls, but not .xlsx)
  • Make sure that you enable macros when you open the workbook. The easiest way to do that is to make the folder containing the workbook a trusted location for Excel - see Create, remove, or change a trusted location for your files.

How to add page numbers, starting on second page - Microsoft Community

lance666 say:

How to add page numbers, starting on second page

I added a section break (continuous) and then, with the cursor definitely in section 2, Insert > Page numbers. If I then click to "Format", I can choose "Start at: 2". But it starts at 1 and in the previous section.

Why is this section break not working? Besides adding the section break, are their some additional steps needed to separate the first section from the second? This isn't a real break. I need a real break (section, that is, not time out).
lance666 say: Thanks for your reply, but i found the solution myself. For anyone else who happens to have this glitch, the fix is like this....

– Click to put the cursor in the first page (first section).
– Insert > Page Numbers.
– Remove the selection mark for “Show number on first page”.
– Click “OK”.

It seems that Word goes backward past the section break and affects the previous section. The solution is to go to that section and change the bad effect.
