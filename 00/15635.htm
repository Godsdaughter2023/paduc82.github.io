amit tandon say:

Enter below formula as an array formula (Ctrl+Shift+Enter):

=MATCH($C$78,INDEX(RBF!$A4:$L$14,MIN(IF((RBF!$A4:$L$14=$C$78)*(ROW(RBF!$A$4:$L$14)-ROW(RBF!$A$4)+1)>0,(RBF!$A4:$L$14=$C$78)*(ROW(RBF!$A$4:$L$14)-ROW(RBF!$A$4)+1))),),0)

This will look for the C78 value in the range A4:L14 and from the first row where it occurs the formula will return the Column position where it first occurs.

Try this.  It selects the sheets for you and then prints.  I didn't crash on me.  Do make sure you've saved the workbook as a "Macro-enabled" version.

Sub Printer()
'
' Printer Macro

Dim MyBook As Workbook
Set MyBook = ActiveWorkbook

With MyBook

 Sheets(Array("Summary", "Sheet2")).PrintOut Copies:=1, Collate:=True, _
        IgnorePrintAreas:=False

    Sheets("Summary").Select
    Columns("A:I").Select
    Selection.ClearContents

    Range("C2:D3").Select
    Selection.ClearContents
    Range("A1").Select

MyBook.Close SaveChanges:=True

End With
