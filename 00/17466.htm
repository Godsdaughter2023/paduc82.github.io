I suggest worksheet event code.   Example code. . . . . .

Private Sub Worksheet_Change(ByVal Target As Excel.Range)
    On Error GoTo enditall
    Application.EnableEvents = False
    If Intersect(Range(Target(1).Address), _
                 Range("B:B, D:D")) _
                 Is Nothing Then GoTo enditall
    With Target
        If .Value <> "" Then
            ActiveSheet.Unprotect Password:="justme"
            .Locked = True
        End If
    End With
    ActiveSheet.Protect Password:="justme"
enditall:
    Application.EnableEvents = True
End Sub

You have to first unlock just columns B and D.

As you fill cells in B and D those cells will be locked for editing.

The code is to be copied into the sheet module.

That really shouldn't be required (removing MSE protection - I download and install all sorts of programs all the time and I have an external drive for backup and from first setup and use to now I have never needed to disable MSE to it or to do anything) - or you can try it anyway and see if works without troubles even if it says it might and do this only if it really does cause issues. If you don't need the internet, disconnect from it entirely as the safest way to go.

 

Do what you need to do and then immediately re-enable real time in MSE by rechecking the box you unchecked and saving it and confirming that the MSE screen goes from a red border to a green border, the Home Tab says MSE is enabled and protecting your system, and the icon in the notication area returns to a green color.
