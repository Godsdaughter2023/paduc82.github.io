Microsoft will still make money from Windows:

- OEMs that preinstall Windows 10 on new PCs still have pay for Windows 10.

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device –at no additional charge.

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device – at no additional charge. No, its not a catch. We are computing in different times. Windows 10 is free so app developers can standardize on one platform and reduce the fragmented market of different versions and editions of Windows that have been an issue in the past. When everybody is on the same platform, it means better quality apps, a healthier ecosystem and more advances in hardware and software innovation.

CMD.exe 0x0000142 error - A SOLUTION! - Microsoft Community

n178 say:

CMD.exe 0x0000142 error - A SOLUTION!

My command prompt was not loading, displaying the above error code: This application cannot open.

To solve this, I used many fixes and passed through several technical support methods to no avail..

The solution I have found - which means CMD.exe and any programs that rely on this work:

#1 - In System32, find the cmd.exe file. Give yourself full file permissions using the advanced settings:

Properties, Security Tab, Advanced button (For special permissions or advanced settings, click Advanced).

Ensure that you make yourself Owner of the file for this, and Enable inheritance. Once you have full control, move the file from System32 (I moved it to my Desktop). Then, move the file back to System32 (I copied Console Window Host, renamed the copy cmd, then proceeded to delete it. Not sure this bit is needed.)

Hence, move cmd.exe back to System32. This seemed to fix the issue!
