Would it be better solution to change date() to now() everywhere? 

    No, because Date() is not the same as Now().  Date() returns the date only, with the time portion of the date-time value set to 0, while Now() returns a date-time value with both the date and the time.  If you have date fields with both date and time in them (when you don't need the time), it makes it more complicated to run queries by date.

    Furthermore, if you have a configuration error of some sort that is causing the Date function to give an error, then it will probably cause other errors to occur also.  Using Now() instead of Date() would be just putting a band-aid on a larger injury.

Here are a few more common steps to fix WU issues:

  • Run Windows Update troubleshooter or wu10.diagcab
  • Make sure date & time are correct
  • Uninstall any 3rd party antivirus program
  • Switch Insider Ring to FAST
  • Set Feedback & Diagnostics to FULL
  • Verify MS account
  • Unplug any USB devices
  • Delete the "Download" folder in the Windows/SoftwareDistribution folder
  • Rename the "SoftwareDistribution" folder by running RenameWU.cmd with admin rights
  • Reset Windows Update (use the auto-script)
  • Remove or disable Bitlocker
  • Disconnect from
  • Use PowerShell (Admin) to run DISM /Online /Cleanup-Image /RestoreHealth and sfc /scannow
  • Delete the WindowsSelfHost key in the registry, you will have to adjust Insider Settings after restart (EXPORT the registry aka backup beforehand just in case): Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsSelfHost

You will be downloading a file similar to his: 18xxx.1000_amd64_en-us_f48384f3_convert.zip

timoteo iv ofr say:

Hello,

There's a possibility that your internet connection is unstable or there's an incorrect settings on your phone. This is why you're unable to see the attachment on your email. Let's work this out by checking if it's stable or not. Then, check your account settings on your phone to see if it's set to Always download full message and internet images.

To check that account settings, you can follow the steps provided below for your device operating system:

For Windows 10 Mobile:

  1. Open Outlook Mail app.
  2. Tap the three dots or the Ellipsis button on the lower right corner and select Manage accounts.
  3. Tap the email account and select Change mailbox sync settings.
  4. Put a check mark on the box that says Always download full message and internet images.
  5. Click Done and Save.
  6. Exit the app and re-open Outlook Mail (Press and hold the back button, then swipe down the app to exit).

For Windows Phone 8:

  1. Open the Outlook app on the device.
  2. Tap the [...] icon in the bottom right.
  3. Select Settings.
  4. Scroll down and enable the option Always download full message and Internet images.
  5. Select the Done option on the bottom to save your changes.
  6. Exit the app and re-open Outlook.

Update us with the outcome.
