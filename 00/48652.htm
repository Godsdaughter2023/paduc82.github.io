bradburykeller say:
Hello sean-1364,

The fan on your surface will run whenever the system temperature hits the point where it must apply additional venting to cool the system down. This may run all the time if there is enough ambient load on the system from an Anti Virus application and or because of other background applications or services.
If you check the Task manager when at idle try looking to see what applications are running and using a large number of resources, especially CPU or GPU.

If you wish to silence the fan you can do so by changing the power settings on your Surface so that the system responses to overheating by slowing down the system rather than by increasing the cooling capacity. This can be done by right-clicking the battery icon and going to Power Options < Change Plan settings (of the active power plan) < Change advanced power settings. From there you can scroll down and expand [code]Processor power management<System Cooling Policy[\code] and change the policy from active to passive for battery mode and plugged in to control when the fan will ramp up instead of the system underclocking to reduce heat.

is my computer going to run ok using windows 10?its older acer - Microsoft Community

noracockrell say:

is my computer going to run ok using windows 10?its older acer computer.it came with windows 7 on it

will my old acer laptop run fine with windows 10.iv had my computer  bout 3 yrs.

a. user say:

Nobody else can give you 100% assurance that your computer will work with Windows 10.  Only you can do this.

The first thing you need to do is to find out whether your hardware is compatible with Windows 10. If you install Windows 10 on incompatible hardware, it's only a matter of time before Windows 10 stops working properly.

To do this, make a list of all the hardware in your computer and attached to your computer, including the brand name and model number. Then go online and find out whether the manufacturer is supporting that model number on Windows 10. If the manufacturer is not supporting that model, check user forums to see if other people are successfully using that model with Windows 10.

The next thing you need to do is to find out whether all the installed software is compatible with Windows 10. Using Windows 10 with incompatible software is asking for trouble. Once again you'll go online and see whether your product and version number is supported on Windows 10, and if it isn't supported, what other users' experience with that version has been.

If your hardware and software are compatible with Windows 10, the next thing is to learn how to install Windows 10 the right way: with a clean install. If instead you choose to upgrade, you're taking a big gamble. A clean install is the best way to install an operating system, and a clean install on supported hardware always works. But a clean install requires a lot of technical knowledge and experience, so you have to prepare accordingly.

If all this sounds like a lot of work, you're right ... it is. But if you want to be assured that your computer will work properly on Windows 10, and you won't have to struggle like so many people here do, you have to do the work before you install Windows 10, so you won't have to unravel a mess after you install Windows 10.

jmh3143 say:

"It's not a true crash, in the sense that the bluescreen was initiated only because the combination of video driver and video hardware was being unresponsive, and not because of any synchronous processing exception".

Since Vista, the "Timeout Detection and Recovery" (TDR) components of the OS video subsystem have been capable of doing some truly impressive things to try to recover from issues which would have caused earlier OS's like XP to crash.

As a last resort, the TDR subsystem sends the video driver a "please restart yourself now!" command and waits a few seconds.


If playing with video driver versions hasn't helped, make sure the box is not overheating.

Try removing a side panel and aiming a big mains fan straight at the motherboard and GPU.

Run it like that for a few hours or days - long enough to ascertain whether cooler temperatures make a difference.

If so, it might be as simple as dust buildup and subsequently inadequate cooling.

I would download cpu-z and gpu-z (both free) and keep an eye on the video temps
