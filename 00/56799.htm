Is Kasperksy Anti-Virus software actually compatible with Windows Vista. Does it conflict with Windows Defenders? Does it cause system crashes? What is the best and most compatible Security Software out there with Vista? Do we really need it with Windows Defender?

Yes, you can run Kaspersky with Vista.  As far as I know, you can also run Defender with it, but note that several anti-virus programs (Norton, McAfee, AVG, and Microsoft Security Essentials) turn off Defender because they do the same thing it does; so it's *possible* that Kaspersky does the same. If it does turn it off, that's good, not bad.

 

Note that Defender is an anti-spyware program, and does *not* substitute for an anti-virus. If you run Defender, you still need an anti-virus.

 

I don't think Kaspersky is the best, though. Here are my recommendations:

 

For an anti-virus program, I recommend eSet NOD32, if you are willing to pay for it. If you want a free anti-virus, I recommend one (do not run more than one) of the following three:
 
Avira AntiVir
Avast
Microsoft Security Essentials
 
You also need anti-spyware software. I recommend that you download and install (freeware) MalwareBytes AntiMalware.
