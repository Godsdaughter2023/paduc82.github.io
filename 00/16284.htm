a. user say:

Hello Hari,

Thank you for posting your query in Microsoft community.

You cannot change the speed of the speech from Word. However, you set it from Control Panel.

To change the Voice and Speed, follow the below steps.

  1. Go to the Start Button.

     2.  Type Speech in the Search programs and files box.

     3.  Choose the Change text to speech settings listed under Control Panel.

     4.  Under Voice selection use the drop down arrow to change the voice. You can only change the voice if you have additional synthesized speeches installed.

     5.  Under Voice speed, use the slider bar to change the rate. Move the slider bar to the left for a slower rate or to the right for a faster rate or halfway for a normal rate of speed.

You can also refer the below article on using speak text-to-speech feature.

https://support.office.com/en-us/article/Using-the-Speak-text-to-speech-feature-459E7704-A76D-4FE2-AB48-189D6B83333C

Hope this helps. If you have any other queries related to office products, feel free to contact us.

Thank you.

phonehacker say:

Factory settings reset. In standby, dial *#7780# and the phone's security code (the nokia's default security code is 12345). This reset restores phone's original settings.

Soft reset. In standby dial *#7370# and the phone's security code. Soft reset deletes ALL your phone's data.

Hard reset (the three key code/reset). Turn off the phone. Press and hold three keys together, the green/talk key, the * key and the number '3'.Then, turn on the phone and don't let the three keys before you see the nokia hands logo/nokia boot up screen.
For 5800 XM hard reset need different keys/buttons. The green/talk key, the red key and the camera key/button. The process is the same.Warning!Hard reset deletes ALL your phone's data/files!
