Hi Jim,

Word has a command called “Create Microsoft Outlook Task” but this command is not visible by default. You can add it to your Quick Access Toolbar (QAT), Ribbon or Toolbar by the following steps:

  1. In Word, navigate to File> Options> Quick Access Toolbar.
  2. In the “Choose command from” list, choose Commands Not in the Ribbon.

  4. Click Add> OK.

To create Outlook task from within Word, save the document first and then click Outlook task button from QAT.

When opening Task in Outlook, you can view the task and double-click to open the document.

Hi,

If you’re trying to create multiple copies of a single sheet, you can still use the “Move or Copy” command, just like what Doug Robbins suggested. Follow these steps:

  1. On the Quick Access toolbar, click the Move or Copy command.
  2. In the Move or Copy dialog box, choose where you want to paste the copy of the sheet from the To book: drop-down. Note: If you want to paste the copy of the sheet to be inside the same workbook, choose the name of the workbook. If you want to place it in a new workbook, choose (new book).

  4. Check the box beside Create a copy.
  5. Click OK.
  6. Repeat the action to create multiple copies of the sheet.

adityarathee96 say:

Download the recent driver from Intel website. If you try to install it, it wont install. So you have to right click on the file and extract it.

Go to the extracted folder and in Graphics folder, you gotta find this file - "igdlh64.inf"

Now open it and find "Windows 8.1 Install". You can do it by pressing 'Ctrl+F'.

There will be "IntelGfx.NTamd64.6.3" written just under it.

You gotta change it to "IntelGfx.NTamd64.6.4".

Now save the document.

No you have to break the digital signature of the driver.

Just delete "igdlh64.cat".

Now go to Settings>Update & Security>Recovery>Advanced Startup (Restart Now).

From there you have to disable the digital signature check. And don't bother to undo this change after you are done. It will undo itself automatically in next boot.

Now press 'Windows+X' and select Device Manager.

In Display adapters, right click on Intel(R) HD Graphics Family and select update driver software.

Select 'Browse my computer' option. From there select 'Let me pick' option. There you have to click on 'Have Disk' button. Go on and provide the location of the driver. And make sure, you provide the exact same file which you edited.

Now this will start installing you drivers. And then Congratulations!!! You have fixed it.

This should work perfectly and your Dedicated Video Memory will be shown and used correctly. To check it, go to Settings>System>Display>Advanced Display Settings>Display Adapter Properties. And make sure you max out your  video memory from BIOS. Although, the System Video Memory shows 0mb still. But it would work flawlessly.

But I call it a temporary fix because in some time (likely after an update to a new build or a graphics update), Windows will change your drivers to the faulty one automatically. Do check the Display Adapter Properties regularly. And if it changes, don't worry. The drivers which we installed are saved.

To install that driver again, go to the Update Driver Software in Device Manager then Browse and the Let me pick. There will be the drivers to select and install. And one of then will be the one which we just installed. You can distinguish it easily as it doesn't have a Digital Signature. Just install it, and you are good to go again.

P.S. - Don't hesitate to ask again, if you find any difficulty in any step. And if it works for you, do inform me.

Lets hope Microsoft fixes it.
