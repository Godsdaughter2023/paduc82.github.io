brian tillman [outlook mvp 2007-2019] say:

Whenever a program (or Windows itself) needs to create a file in order to perform a function, it uses the Temp folder to hold that file.  You should be able to delete those files without any trouble.  Don't delete the folder itself.

 

It is best to delete the files in that folder immediately after logging in before doing anything else.  Open the folder by pressing WinKey+R, entering %temp% in the Open field, then pressing Enter.  Press Ctrl+A to select all the files, then press Delete.

 

You may find that the deletion process fails because a file is locked.  Simply deselect that particular file (it will be the first file shows in Windows Explorer) by holding Ctrl and clicking once on the file.  Press Delete again after that to continue with the deletion.  Once all of the files you can delete are gone, reboot.
