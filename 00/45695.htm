  3. Select View all on the top left corner.
  4. Run Internet Connection.
  5. Follow the on screen instructions.

Method 2: Assign the IP address manually.

 

I would suggest you to follow the steps given below to assign IP address manually and check if it helps.

  1. Right-click the Network icon located at the bottom-right corner of the Desktop screen then click Open Network and Sharing Center.

  7. Press Windows Key + R Key, type comd, open it and type ipconfig. It will show you ip address, gateway address.
  8. Copy the Default Gateway from the Network Connection Details window information.
  9. Press Windows Key + R Key, type comd, open it and type ipconfig /all. Copy the DNS Server info from the Details window.

This issue is easily fixed by running the Deployment Image Servicing and Management (DISM) and System File Checker scan using a command-line in Command Prompt to repair corrupted system files. In your case, Command Prompt is only accessible through the advanced startup and Windows Recovery Environment. Follow these steps:

  1. Insert and boot from a Windows 10 installation media. You can use the media creation tool from this link to create a Windows 10 installation media.
  2. Select a language and click Next.

  3. Select Repair your computer.
  4. Select Troubleshoot > Advanced options > Command Prompt.

    dism /online /cleanup-image /checkhealth
    dism /online /cleanup-image /scanhealth
    dism /online /cleanup-image /restorehealth
    sfc /scannow

  6. Restart your computer.

Should the issue persists or encountered an error, repeat the steps 1 to 4 and execute these commands instead:

bootrec.exe /rebuildbcd

bootrec.exe /fixmbr

bootrec.exe /fixboot
