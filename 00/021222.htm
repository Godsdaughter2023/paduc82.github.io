You've posted in the Office for Mac section. Office Mix doesn't work with Office 2016 for Mac yet. However, if you install Windows using Bootcamp or Parallels, you'll be able to install PowerPoint for Windows 2013 or 2016, and then Mix.

Microsoft wants you to use their cloud service to host Mix presentations. This is pretty easy. geoff_lilley say:

You have a few options on this one:

  1. Hand-type that first cell reference. If the name of the sheet is Sales Summary, and the cell reference is A2, then you type
    ='Sales Summary'!A1
    That way, you can copy and paste down.
  2. Save a little typing. Go to the sheet with the PivotTable, and then find a blank cell. Type =, just like normal, and then type in the cell reference, like A1. Copy it, and then go to the cell in the other sheet where you want the reference to appear. Click the Paste button on the Home tab; toward the bottom, you'll see Other Paste Options; choose Paste Link...
  3. The other option is to use the GETPIVOTDATA function to your advantage. Sometimes, I'm replicating part of the PivotTable's functionality, I do what you do  - I type =, then click a cell in the PivotTable. The GETPIVOTDATA function is a series of name-value pairs; so the first is the name of the field you're calculating, followed by the first cell of the PivotTable. After that, it's the row field's name, the row field's value, the column field's name, and the column field value.

So if my PivotTable has a data field called Item Total, a Row Field called Customer, and a Column Field called Order Year, I can reference the cell in the PivotTable that has the customer Around the Horn's Item Total for 1997 like this:

=GETPIVOTDATA("Item Total",$A$3,"Customer","Around the Horn","Order Year",1997)

With a little fancy footwork, I can change "Around the Horn" to G5, or 1997 to H5.

macropod say:
You can't. You can delete/clear it though, but the method depends on what it is. If it's a textbox or autoshape, simply clicking on any of the lines should be enough to select it, after which you can delete it. If it's a paragraph border, choosing 'no border' from the border dropdown from the Paragraph panel on the Ribbon's Home tab will suffice.
