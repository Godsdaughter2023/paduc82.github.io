thundertiger say:

You should NOT have to change any settings once the 3 access points are configured.  Mine works just fine.  Ignore the packet data section under:

 

Menu - Tools - Settings - Connection - Access Points.

 

If you are trying to use the web browser you need to use the AT&T Internet access point or a WLAN access point.  If you are using wap (who uses that anymore?) Then you would use the AT&T MEdia Net.  The MMS access point should be self explanatory.

 

If you want to reset the AT&T access points then go use the Settings wizard located under:

 

Menu - Tools - Sett. Wizard (located at the bottom)

 

To check if you are using 3G or GSM look here:

 

Menu - Tools - Settings - Phone - Network

 

Check to make sure you are either using UMTS or Dual Mode if you want 3G access.  If you set it to GSM it will only use Edge. When it is using 3G data you will see a small 3G or 3.5G symbol. If it is just using Edge it will show a small E symbol next to the signal indicator bar. If you are using a WLAN connection you will see a small 2x2 box near the battery indicator.

 

If you want to reset your phone settings back to factory use:  *#7780#   and 12345 for the password

 

If you want to reset your phone the way it left the factory (BACKUP ALL DATA FIRST) use: *#7370#  and 12345 for the password.
