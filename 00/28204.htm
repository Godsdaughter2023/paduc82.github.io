doug robbins - mvp office apps & services (word) say:

If any of the fields are blank? or is it sufficient to just check the «DEGREE_CODE1» field?

If the latter, insert the following mergefields

«DEGREE_CODE1»  «DEGREE_CODE1» «DEGREE_YEAR1» UB «SCHOOL1» («MAJOR1»)

then modify it as follows

«DEGREE_CODE1» <> ""  "«DEGREE_CODE1» «DEGREE_YEAR1» UB «SCHOOL1» («MAJOR1»)" ""

hansv mvp say:

You could put code in the Worksheet_Activate event of the invoice sheet:

Private Sub Worksheet_Activate()
  Dim r As Long
  Dim m As Long
  Application.ScreenUpdating = False
  Application.EnableEvents = False
  m = Range("C" & Rows.Count).End(xlUp).Row
  For r = 1 To m
    Range("C" & r).EntireRow.Hidden = (Range("C" & r) = 0)
  Next r
  Application.EnableEvents = True
  Application.ScreenUpdating = True
End Sub

Sub ChangeFont()
'
    With Selection.Find
        .ClearFormatting
        .Replacement.ClearFormatting
        .Text = ""
        .Font.Name = "Calibri"

        .Format = True
        .MatchCase = False

        .MatchWildcards = False
        .MatchSoundsLike = False
        .MatchAllWordForms = False

        .ClearFormatting

    End With
End Sub
