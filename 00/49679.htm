Update: It turns out that there seems to be some kind of conflict between the UAC and the Administrators group. Not sure how i was being affected but i finally ended up Denying the System write permissions to my external drive. Suddenly Builtin Administrator, Builtin users, and NT Authroity showed up, one of them, or some other group, took over the drive, locked it down as Read Only, Denied my access to any files, then took it one step further and removed my ability to Write Attributes or Read them. I was forced to use a back up "Admin that i have on standy with a third one. I reapplied permissions to my main Administrator yet access was still denied. After doing some things with ICACLS I was able to get partial control but I still could not save.

At the end of the road, it turns out that an external hard drive formatted as NTFS is recognized by the computer as a Fixed disk, as if it were part of the computer, and it treated it as such, considering my files System Files by locking them down.

Even with the Hidden Builtin Administrator that I activated, I could not access the files. I was finally forced to move all the filles to the c:\ drive and format the external drive to exFat so that the system didn't think it was part of the computer. Unfortunately, i will never be able to depend on having a System Image Backup because exFat cannot have one written to it.

I have full access, as does anyone and everyone, there are no permissions on exFat, so it's a tradeoff, better than fighting the system for control of my own damn computer.....
