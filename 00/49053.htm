davem121 say:
Hi Rachelle, that command to make the Administrator account active is incorrect and it will not work from inside your user account since you are not being seen as an administrator . . .

Press the Shift key whilst clicking Restart, this will boot your system into the Windows Recovery Environment

Go to Troubleshoot - Advanced Option - Command Prompt

Run this command and hit Enter

net user Administrator /active:yes

Now boot back into Windows, the Administrator account should be active, log into that account and you can make your account an administrator account

Either way you need to turn off Secure Boot and then run Command Prompt to run this command:
