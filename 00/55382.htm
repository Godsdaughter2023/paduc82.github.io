anne ros say:

Hi,

Sticky keys dialog box appears when you have press the Shift key five times continuously. It also lets you use the Shift, Ctrl, ALT, or Windows Logo keys by pressing one key at a time. You can set up Sticky Keys from the Ease of Access Center on the Control Panel. Follow the steps below:

  1. Press the Shift key five times.
  2. Once the dialog box appear, click the Go to the Ease of Access Center to disable keyboard shortcut link.
  3. It will open the Set up Sticky Notes window in the Control Panel.
  4. Uncheck Turn on Sticky Keys when SHIFT is pressed five times option.
  5. You can also set the settings to your desired option.

After doing so, check if the same pop-up will appear while playing the game.

mommyx4 say:

I figured out the problem. 

There was a folder that was minimized (restored down) to a size that the computer found too small. Every folder I opened would flash due to this. 

The fix was to find the actual folder that started the whole mess... the one that was originally minimized to the offensive size per the computer. 

I clicked on the folder today and I knew it was the one because when it popped up it was not flashing like crazy like all the others... it was the same size 2x3 but no crazy flashing occurring. Another big difference was that I was able to use my mouse and increase the size of the folder by dragging the sides which I have been unable to do on all the other folders. 

I dragged the sides until the folder was a larger minimized size and just like that the problem was gone. 

I hope this helps everyone who comes across this problem. Find the original folder that was minimized (restored down) to the "too small" size and then resize that folder, everything will go back to normal once done.

jrista say:

Windows 8.1 Mail App - "Not Junk" does not prevent future mail from ending in Junk

The Windows 8.1 mail app, while handy, is severely lacking in key functionality. Most specifically, the frequently used "Not Junk" button does not actually execute the proper behavior: Marking emails FROM THAT ADDRESS as not junk mail. All the Not Junk button does is move the email out of the Junk folder into the Inbox. When I tell Windows Mail, or Outlook.com for that matter, that an email is not junk...I REALLY MEAN IT.

I believe Microsoft needs to update the Windows 8.1 Mail App to exhibit proper "Not Junk" behavior...when it is clicked, the email address should be added to the list of safe senders, as that is what used to happen in Hotmail (back in the Windows Live and prior days.) As it stands now, email addresses that used to properly go to the Inbox, or any number of other folders, ALWAYS go to my junk email box. These are known senders, people who are friends or acquaintances, companies that I have done business with an ALWAYS want to receive email from, etc. The Mail app offers NO WAY to prevent ALL future emails from a given email address from always going into junk mail...a rather infuriating situation. 

Additionally, Windows 8.1 Mail does not offer any way to set up and configure custom rules. I want to be able to configure my mail just as well through the Mail App as I can on Outlook.com. In that respect, I should be able to set up rules, directing all incoming mail from certain email addresses to certain folders, or with certain keywords in the subject to other folders, etc. This is STANDARD email app behavior, stuff that's been around for DECADES. The constant march towards less functionality and fewer features is probably the most infuriating thing about modern operating systems... It seems to be a fancy trend, a fad, to create the most bare-bones, barely functional applications and call it "good".

There is no reason advanced features need to intrude upon the Mail app's simple interface. The interface simplicity is good...but not at the cost of critical functionality and proper behavior. PLEASE, fix the Not Junk feature, and add some way for users to automatically filter and sort their email according to custom rules. Thanks.
