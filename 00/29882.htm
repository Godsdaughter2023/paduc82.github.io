alex t jr say:

Hello Country Boy,

Sadly i would have to say no.  The reason being is that all Virtual Machines are stored on the system drive.  So you couldnt contain a file that is FAT32 on a drive that is NTFS.  Windows 7 can only install to NTFS. It is impossible to force a FAT32 Virtual Machine on this setup.  There isnt even an option to format to FAT32.
