There is no yellow-highlighted -vs- blue-highlighted code in your post - the entire post is blue.

Possible reasons your code would not run correctly include the presence of paragraphs with first-line indents and/or space before/after settings other than 0. Accordingly, you should check your document's formatting. You can circumvent such issues with code like:

With Doc.ActiveWindow.Panes(1)
  For i = .Pages.Count To 1 Step -1
    Set objRectangle = .Pages(i).Rectangles(1)
    If objRectangle.RectangleType = wdTextRectangle Then

andreas killer say:

Am 04.11.2010 08:16, schrieb Sammy Seal:

I would like to have a dialogue or pop up box appear when a cell is selected that will give users info on what to enter in the selected cell.

eg; if cell b2 was selected then a pop up appears with the text "enter wall length here"

You could use a userform and manage it from the sheet.

For the example below you need a userform named "UserForm1" with a label named "Label1" on it with no code. Put the code below in the code module of your sheet.

If you like to control the position where the form is shown, I recommend you to use the Form Positioner from Chip Pearson:

http://www.cpearson.com/excel/FormPosition.htm

Andreas.

Option Explicit

Private Sub Worksheet_Activate()
  'Show form if we enter the sheet
  UserForm1.Show vbModeless
End Sub

Private Sub Worksheet_Deactivate()
  'Hide form if we leave the sheet
  UserForm1.Hide
End Sub

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
  'Clear message
  UserForm1.Label1 = ""
  'Test if a cell of our interest is selected
  Set Target = Intersect(Target, Range("B2,B3,C5"))
  If Target Is Nothing Then Exit Sub
  'Exit on multiple selection too
  If Target.Count > 1 Then Exit Sub
  'Yes, which cell is it?
  Select Case Target.Address(0, 0)
    Case "B2": UserForm1.Label1 = "enter wall length here"
    Case "B3": UserForm1.Label1 = "enter wall width here"
    Case "C5": UserForm1.Label1 = "enter number here"
  End Select
  'Make it visible if not on screen
  If Not UserForm1.Visible Then UserForm1.Show vbModeless
End Sub
