a. user say:

Hi, DrVenkmen,

Open File Explorer (folder icon in Taskbar)
Click on Drive C if that's where the files are located
In the Search Bar upper Right > Type *.tmp
These are temporary files you never know about
It's safe to delete them.
Click on the first file then press Ctrl+A to Select All
If a file "is in use" message appears, skip it.

Start > Type Run > press Enter
Type cleanmgr /sageset:11
Review the files before selecting to delete by clicking on the listings to see details of what these files relate to.
Click Clean Up System Files when finished.
