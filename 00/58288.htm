1. On the start screen of windows 8 press the ‘Windows key + X’ key on the keyboard and then click on control panel and then Hardware and sound option.

chris.h - support engineer say: Hello Kneed2no,

Windows Fax and Scan is located under the Windows Features.

 - Click Start button
 - Type windows features in the search box
 - Click Turn Windows features on or off
 - Locate Print and Document Services > click + to left to expand
 - Uncheck Windows Fax and Scan
 - Click OK

You will need to restart your computer and then perform  the same steps except check Windows Fax and Scan to reinstall it.
