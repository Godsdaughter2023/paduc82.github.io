The =LINEST(...) function is an array function.  That is, if you set it up correctly, and then press Shift+Control+Enter, it will return an array of results to the spreadsheet.  It looks to me like array functions are rare in Excel, and I have not been able to find any others.  What other array functions exist in Excel (if any)?
