This is the procedure I follow when the messages start popping up:

  1. Go to Outlook and run an Advanced Find. (Click in the search box and a new menu “Search” will open – you will see the button for Search Tools and then Advanced Find).
  2. In the new window “Advanced Find” – on the first line, click browse, and check all the folders (and subfolders) in the email account (You might be able to click the Search Subfolders dialog box, but it didn’t seem to work for me so I checked all the boxes). Hit OK
  3. Go to the second tab “More Choices” and only click the radio box that says “Only items with one or more attachments”. Do NOT click or add anything else.
  4. Click Find Now (on the right).
  5. When all the emails have been displayed, sort by SIZE. You are only interested in the emails that show up with a size of 0 (ZERO!!!!!). Open the emails and you will probably see that these emails have attachments that add up to over 20MB. Delete them – and then delete them again from the trash bin.
  6. Close outlook and reopen it. Run Send/Receive All Folders and it should run without the error message.

jay freedman say:
There isn't any way in the user interface to get rid of it, but it's fairly simple in the macro editor.

First go to Full Screen view, with the toolbar showing. Press Alt+F11 to open the macro editor, and press Ctrl+G to open the Immediate window. Type or paste this line in the Immediate window and press Enter at the end of it:

CommandBars("Full Screen").Visible = False

Close the macro editor. The toolbar will be gone. If you ever want it back for some odd reason, go through the same steps but type True instead of False.

Hello, Dianek77

What website is it?
Have you been able to the website previously?

It is possible the website is having some technical difficulties.

If you use a proxy, check with the administrator for it to see if it is blocked. If not, check to make sure it is disabled by going to Tools, Internet Options, go to the Connections tab and click LAN settings. Uncheck all of the options.

Another thing you can try is clearing the DNS cache. You can do this by going to Start, All Programs, Accessories, right click the Command Prompt and choose Run as Administrator. You may be prompted to continue. In the Command Prompt type the following command and press enter: ipconfig /flushdns

If that doesn’t help run the following command to check the HOSTS file: notepad.exe %windir%\system32\drivers\etc\hosts

In the Notepad window, check for the website in the list. If it is there, remove the line, or put a # in front of it.
