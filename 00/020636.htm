Hi Brian,
 
Thank you for posting your question in Microsoft Community.
 
From your issue description, I understand that you are unable to access some of websites and you are unable to find the option to disable the option “Do Not Track (DNT)”. Please let me know if you are not facing this issue.
 
I can understand the inconvenience caused to you. I will help you resolve the issue!
 
Before we proceed with troubleshooting, I would need little information on the issue.
1. What is the name of websites are you referring to?
2. Do you receive any error message?
3. Are you facing the same issue in different web browser? For Example: Google Chrome, Firefox, etc.
 
Try the general troubleshooting methods and check after each method:
 
Method 1: Try viewing those sites in Compatibility View and see if that helps.
Note: Not all website display problems are caused by browser incompatibility. Interrupted Internet connections, heavy traffic, or problems with the website can also affect how a webpage is displayed.
 
Step 1:
a) Open Internet Explorer.
b) Press ALT+T keys, to launch “Tools” drop down menu.
c) In the “Tools” menu, select “Compatibility View” and then check.
 
Step 2: If the issue is resolved, add the websites to “Compatibility View Settings” list.
a) Open Internet Explorer.
b) Press ALT+T keys, to launch “Tools” drop down menu.
c) In the “Tools” menu, select “Compatibility View Settings” option.
d) In the “Compatibility View” window, add the name of websites (with which you are facing the issue) and then check.
 
Method 2: Add the name of list of websites, to "Trusted list" and then check:
a) Press the ‘Windows + R’ key on the keyboard.
b) In the “Run” Window, type inetcpl.cpl and then click “Ok”.
c) In the “Internet Options” window, click “Security” tab.
d) Click “Trusted Sites” Click “Sites”.
e) Under “Add this website to the zone field”, Type the complete website name and then click Add.
f) Note: If you receive the error message, clear the “Require server verification (https:) for all sites in this zone” check box.
g) Close and reopen the browser and then check.
 
Method 3: Disable “Do not Track (DNT)”:
a) Press the ‘Windows + R’ key on the keyboard.
b) In the “Run” Window, type inetcpl.cpl and then click “Ok”.
c) In the “Internet Options” window, click on “Advanced” tab, scroll down and under “Security” uncheck or remove tick mark for “Always send Do Not Track header”.
d) Click “Apply” and then click “Ok”.
e) Close and reopen the browser and then check.
 
If the issue persists, post your reply with results and questions asked as above. We are happy to assist you further with advanced troubleshooting steps!

 
Hope it helps.

chriscote say:

Display both columns in a 2-column lookup field

I have a database with 2 tables. One table is a list of available technicians who will work on projects with separate fields for first and last name in addition to the AutoNumber ID field. The other table is a list of projects to be assigned.  When I created the project table, I included a lookup field to give the first and last names of the technicians available.  When I enter a project into the table and get to this field, the drop-down appears with both first and last names. However, after I select a technician, it only displays the tech's first name. This is a problem as there are multiple people with the same last name.  Is there a way to see the full name of the technician who is assigned to each job?

Chris Cote

bill mosca say:

Chris

Use a Query as your RowSource. Concatenate the two fields into one. It should look something like this in SQL view. Change the names to match your field names.

SELECT MyAutoNumber, FirstName & " " & LastName As FullName

FROM myTable

scottgem say:

It is not recommended that one use lookup fields on the table level. Lookups should be done on the form level using list controls like a combobox.

A lookup field is essentially a combobox and has pretty much the same properties. One of these properties is that it displays the first non zero width column after a selection is made. So, if you need to display multiple columns after a selection you should use concatenation as Bill describes. The only thing is I would do Last, First:

SELECT MyAutoNumber, LastName & ", " & FirstName As FullName

FROM myTable

ORDER BY Lastname, Firstname;
