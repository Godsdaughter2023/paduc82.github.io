Sub MakeHyperlinks()
'David McRitchie
Dim Cell As Range
On Error GoTo endit

            Selection.SpecialCells(xlConstants, xlTextValues))

Sub SearchForUnderline()
s = ""
s1 = ""
For Each r In ActiveSheet.UsedRange.SpecialCells(xlCellTypeConstants)
For x = 1 To Len(r)
If r.Characters(x, 1).Font.Underline = xlUnderlineStyleSingle Then
If s = "" Then s = r.Address Else: s = s & vbCr & r.Address
GoTo abcd
End If
Next
For x = 1 To Len(r)
If r.Characters(x, 1).Font.Underline = xlUnderlineStyleDouble Then
If s1 = "" Then s1 = r.Address Else: s1 = s1 & vbCr & r.Address

GoTo abcd

End If
Next
abcd:
Next
MsgBox "single underline address:" & vbCr & s _
& vbCr & vbCr & "double underline address:" & vbCr & s1
End Sub

miksu---01 say: Hi,

You may have some conflict with your PC's COM ports. You can also try to change the COM port manually:
1. Connect your phone via CA-42.
2. Go to 'Control Panel' and click 'System'.
3. Select 'Hardware' and press the button 'Device Manager'.
4. Open 'Modems' and double-click 'Nokia CA-42 GSM USB Modem'
5. Select 'Advanced' and press button 'Advanced Port Settings..'.
6. Change the COM port to e.g. COM30 and press OK.
7. Remove cable from your phone and put it back....
If you open the Nokia connection manager, it should show that COM30 is used (click Configure button). If COM port is something else, set it to COM30.
