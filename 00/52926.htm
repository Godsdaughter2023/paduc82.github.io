lemp say:

Most, if not all, laptops have a means to disable the wifi adapter (whether or not the LAN connection is active).  This is probably an easier and better way for you to deal with the issue you're describing.  The actual method differs for different laptops.  For many, it's a combination of the Fn keys and one of the F keys.  Sometimes there's a physical switch located somewhere on the laptop.  You'll have to consult the User Guide for the specific laptops in use.

The feature mentioned in the thread you cited is a function of the driver for your wireless network adapter; not all wireless network adapters have this feature.  According to the comments there, the ability to turn off the wireless adapter when traffic through the Ethernet adapter is sensed is available on some HP and Dell computers but not on Lenovo computers.

Windows itself manages the use of network adapters in two ways.  First, there is a priority sequence for detecting adapters and second there is a setting called the adapter's "metric." 

You can set the priority sequence by following these steps:  https://support.microsoft.com/en-us/help/2526067

Windows also uses a connection's "metric" to determine priority.  The connection with the lowest metric takes priority.  By default, Windows sets the metric automatically based on the estimated speed of the connection.  Because Ethernet is faster than most wifi, the Ethernet connection usually has a lower metric.  You can, however, change these values manually: 

https://support.microsoft.com/en-us/help/299540/an-explanation-of-the-automatic-metric-feature-for-ipv4-routes

https://www.howtogeek.com/howto/27994/how-to-change-the-priority-of-wiredwireless-network-cards-in-windows/

You're happy with the laptop, aside from the problem you've got.  So it doesn't necessarily make sense to replace it, but it is 6 or 7 years old, and it won't last forever.  Keep that in mind with respect to any effort you consider putting into the system.  A quite tolerable laptop can be had for ~$400 USD.

In the event that the disk is found to have problems... A new disk may not be excessively expensive - but it may be a bit difficult finding a "new" disk that works with the system, or there may be a bit of a premium on the cost of the disk as compared with larger, faster disks that use newer technology (that your laptop wouldn't be able to take advantage of).
