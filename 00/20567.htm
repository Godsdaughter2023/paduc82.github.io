greginmich say:

I don't understand how they can do this. My screen locked up so I thought it was legitimate.

No genuine Microsoft alert will ever pop up in a web browser and ask you to call a telephone number for assistance. So there’s really no need to even identify the source – all of these security/virus alerts are fake. Tech Support Scam websites lock up a browser by simply running a script that loads a modal alert dialog – and then reloads that dialog every time you try to close it by clicking on the OK button. The modal alert dialog was originally designed to force a response from the user by locking up the browser until the alert dialog was closed – so simply looping the dialog like this can effectively prevent you from ever regaining access to the browser’s controls.

Microsoft provided Dialog Loop Protection for the Edge browser over a year ago so that users could stop the malicious script responsible for looping the dialog and then close the Tech Support Scam website without having to end the browser process in Task Manager. Microsoft’s belated Dialog Loop Protection followed the precedent that had already been set by other browser developers prior to the release of Microsoft Edge:

https://wilde-it.co.uk/blog/post/locked-in-microsofts-edge-browser

Fortunately for everyone concerned, the third-party browser developers are also leading the effort to first deprecate, and then finally eradicate, the showModalDialog method. For a recent discussion of the Tech Support Scam issue, see this thread:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/defender-ineffective-against-the-windows-warning/dfa1ffd3-56cf-403d-b8a9-96f2a4846909

Since Microsoft still seems to be having a little bit of trouble appreciating the actual scope of this issue, and the severity of its impact; it might be a good thing if more people reported this scam directly:

https://www.microsoft.com/en-us/safety/online-privacy/avoid-phone-scams.aspx#reptech

If that link is broken, here's the list of resources provided in the document:

How to report tech support scams

  • Help Microsoft stop cybercriminals by reporting
  • In the United Kingdom, you can report

    Of course you should be sure to include the URL, telephone number, and business name (if available) in your report, because the scammers already have the ability to spoof the URL, according to this source:

    https://www.bleepingcomputer.com/news/security/microsoft-edge-smartscreen-security-feature-can-be-abused-for-tech-support-scams/
