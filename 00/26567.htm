entegy say:

Google disabled Exchange ActiveSync (EAS) syncing for non-Google Apps accounts. EAS was the protocol that allowed push email, which was the as items arrive.

So now Gmail accounts sync over the IMAP protocol by default. IMAP accounts cannot be set for push email. As well, it seems like Windows Phone cannot search the server on IMAP accounts.

If you have a Google Apps account, you can restore these features by using advanced setup and choosing the Exchange ActiveSync protocol. The server is m.google.com.
