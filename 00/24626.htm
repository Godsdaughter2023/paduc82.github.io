joeu2004 say:

StillActive:

The formula =EXP(LN(3)) returns the value 3 as it should

The formula =POWER(LOG10(3),10) returns the value .00061

EXP(LN(3)) is e^ln(3).

POWER(LOG(3),10) is log(3)^10.

That's your mistake:  you want POWER(10,LOG(3)), which is 10^log(3).

PS....  Both EXP(LN(3)) and POWER(10,LOG(3)) do not return exactly 3.  There is an infinitesimal difference that is hard to see because of a dubious "feature" of Excel arithmetic.  But note that MATCH(3,A1,0) returns #N/A, indicating no match, when A1 is one of those formulas.
