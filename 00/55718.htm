techfreak_ say:

Whether you have an onobard camera or internal camera, the solution is the same.  If you need to disable the camera you can do so by disabling it from device manger under imaging devices.  This option may also be available in the computer's BIOS.  If you actually use that camera then you should disable it to avoid the mic from being initialized due to being part of a mic array that could be triggered in your configuration.  Simply separating inputs and disabling the mic should help avoid that problem.  At least it should be worth a try.

These options are available form my laptops with an internal webcam and mic.  My HP is using a different audio chipset so I am providing you the screenshots from my non-HP units using the Realtek chipset, which I believe you have.

From the Realtek Manger's advanced settings (in Control Panel).  If you do not have the manager, install the latest High Definition drivers from Realtek.  Even that action may help.

One of these recording inputs will be for the onboard mic and the other for any external mic used.  Usually the external mic disables the internal mic, but that is not always the case.  If you are using a USB mic, then it would not be disabled.

Once you have disabled the mics in use, see if that resolves the issue of your webcam light staying on.

The process of installing updates comprises downloading, installing and configuring the operating system to use the new updates. Some updates require the computer to be restarted and when this occurs configuring occurs as the system is shut down and is completed when the computer is restarted. It is important not to interrupt the configuration process as configuration involves making changes to the registry. If interrupted, the result can be a computer that will not boot.

If you have a number of updates to install break the task of updating into two or more tasks. If you have been having problems with Windows Update run Disk CleanUp (or better still cCleaner) and Disk Defragmenter ( or better still Defraggler) before installing any large download.
