shadowedmare say:

Until this crazy problem, I never even knew there were registry keys :) You may want to double-check that you're following the instructions in the link I posted

More specifically, here's what I did:

1) Obviously, my keyboard didn't work. I also couldn't get a USB keyboard to work, so I had to use the Windows 8.1 onscreen keyboard.

2) Once I had the onscreen keyboard up and running, I swiped the mouse to the upper right corner and clicked on "Search" -- I then used the onscreen keyboard and typed regedit to find the Registry Editor program.

3) After opening regedit by double-clicking on it, I was where I needed to be to make the changes.

This is where it got a little tricky -- I really had to make sure I followed the instructions carefully and expand the folders found on the left hand side exactly as written:

For the Keyboard:

HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Class\{4D36E96B-E325-11CE-BFC1-08002BE10318}

For the Mouse:

 HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Class\{4D36E96F-E325-11CE-BFC1-08002BE10318}

Basically, you want to end up at two folders:  4D36E96B-E325-11CE-BFC1-08002BE10318 (for the keyboard), and 4D36E96F-E325-11CE-BFC1-08002BE10318 (for the mouse). The string of numbers is very similar to other folders listed, so you really need to make sure you find the right ones.

4) Once you find the two folders, there is no need to expand them further (if you do, you'll find folders labeled "0000", "0001", "Properties", etc. -- just ignore these).

5) To fix the keyboard, click directly on the 4D36E96B-E325-11CE-BFC1-08002BE10318 folder (don't expand it). Once you click the folder, you'll see registry value files appear on the right hand side (the file icons look like torn pages with a lower case "ab" written in red).

6) The file you're looking for is the last one in the group -- the one named UpperFilters. Double-click on it, and a small box opens titled "Edit Multi-Screen"

7) Once the Edit Multi-Screen box opens, you want to delete any information in the Value data: field (if there is any), and type kbdclass  When finished, click the OK button.

8) Follow the same instructions to fix the mouse -- just be sure to delete any information in the Value data: field and type mouclass before clicking OK

9) When finished, restart your laptop and the keyboard and mouse will again be operational.

Good luck! I really hope it works for you.
