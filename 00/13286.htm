There is no need to buy a third party utility if your profile is not damaged. Try the following steps to export your ost file into .pst file:

1) Open your .ost file in Outlook
2) Click on File menu> Import and export >
3) Select ‘Export to a file’> Choose personal Folder file (.pst) >
4) Select your mailbox and check the box to include sub folders >
5) Browse to the desired location where to save the pst file and click on Finish.
