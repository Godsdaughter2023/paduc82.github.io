what you describe as two X axes is only one X axis, but it has multi-level labels.

I misunderstood your question. I am not aware of any collection of arrays. However, you can build a collection of arrays by assigning the arrays to another array as per the following example.

 

Option Explicit

Sub ArraysWithinArray()
    Dim array_1() As Variant
    Dim array_2() As Variant
    Dim array_3() As Variant
    Dim arrCollection() As Variant
    Dim i As Long
    Dim j As Long
   
   
    array_1() = Array(11, 12, 13, 14, 15, 16, 17)
    array_2() = Array(21, 22, 23, 24, 25, 26, 27, 28)
    array_3() = Array(31, 32, 33, 34, 35, 36, 37, 38, 39)
   
    arrCollection() = Array(array_1(), array_2(), array_3())
   
    For i = LBound(arrCollection) To UBound(arrCollection)
        For j = LBound(arrCollection(i)) To UBound(arrCollection(i))
            MsgBox arrCollection(i)(j)
        Next j
    Next i
   
End Sub

jay freedman say:

Yes, you certainly can do exactly what you want. (I have no idea how Microsoft decides what to put into the building blocks distributed with the program -- they certainly haven't asked anyone who uses Word for a living.)

Be clear about one thing: Page numbers are usually placed at the outside margin, which is the right margin on odd pages and the left margin on even pages. That's the opposite of the arrangement you showed. If you agree, then reverse the directions below.

  • If your current document or template is only one page, go to the end and press Ctrl+Enter to create a temporary second page.
  • Open the header area on page 1. On the Header & Footer Tools tab, check the box for Different Odd & Even Pages.
  • In the page 1 (odd) header, with the cursor at the left margin, press Alt+Shift+P to insert a page number field.
  • Press the Tab key once to move the cursor to the center-aligned tab stop that's part of the Header style, automatically applied to the header.
  • Click Insert > Quick Parts > Field, click Filename in the list of field names, and click OK.
  • Select the entire header and press Alt+F3. In the dialog, enter a name for the building block (include "odd page" in the name), set the gallery to either AutoText or Headers, set the category to General, set the Save In to Building Blocks.dotx, and click OK.
  • On the Header & Footer Tools tab, click the Next Section button to go to the page 2 (even) header.
  • Press the Tab key once to move the cursor to the center-aligned tab stop.
  • Insert a Filename field as before.
  • Press the Tab key again to move the cursor to the right margin.
  • Press Alt+Shift+P to insert the page number field.
  • Select the entire header and press Alt+F3. In the dialog, enter a similar name for the building block (include "even page" in the name), set the gallery to the same value as for the odd header, set the Save In to Building Blocks.dotx, and click OK.
  • The next time you shut down Word, you should be prompted whether to save changes to Building Block.dotx. Click Yes.

The two new building blocks will appear in the Building Block Organizer. If you chose the AutoText value for the gallery, they'll also appear in the Insert > Quick Parts > AutoText list; and if you chose the Header value, they'll appear in the Header & Footer Tools > Header list.
