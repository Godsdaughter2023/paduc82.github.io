There are some suggestions here as to how you might go about this but no difinitive answer. If you wish to use VBA code (macro) then the following code should suffice. However, you will need to create a worksheet and insert the information required for the footers in each page.

Use Col A for the left footer, Col B for the center footer and Col C for the right footer. You must start at row 1 because the code relates the page number to the row number. The code only changes the footers referenced in the code. All other print setup features that have been set in the interactive mode will be preserved.

Don't know how much you know about VBA code so here are instructions for installing the code into your workbook.

If you want this code to always be used when printing the worksheet then copy the code to your workbook as follows:

Open the VBA Editor (Alt/F11)

Double click ThisWorkbook in the project Explorer (Left column)

Copy the code below and paste it into the large white area of the editor.

Close the VBA editor (Cross with red background top right.)

Save As Macro Enabled Workbook.

 

You may if you wish install the code into a standard module and attach it to a button so that it only runs on clicking the button.

Click Excel Options (towards bottom right of dialog box)

Click Trust center (Left column)

Click Trust center settings (Right side of dialog box approx middle)

Click Macro settings

Disable all macros with notification

Private Sub Workbook_BeforePrint(Cancel As Boolean)
 
  On Error GoTo ReEnableEvents
  Application.EnableEvents = False
 
  Cancel = True
  Dim i As Long
 
  With Sheets("Sheet1").PageSetup
    For i = 1 To .Pages.Count
      'Comment out any lines not required - (See RightFooter line below)
      .LeftFooter = Sheets("PageFooters").Range("A" & i)
      .CenterFooter = Sheets("PageFooters").Range("B" & i)
      '.RightFooter = Sheets("PageFooters").Range("C" & i)
      Sheets("Sheet1").PrintOut From:=i, To:=i
    Next i
  End With
 
ReEnableEvents:
  Application.EnableEvents = True
End Sub

The various contexts in Word are represented by a subset of the collection of CommandBar objects. When Show/Hide is set to Hide and the cursor is placed at the beginning of the text in the second paragraph and new text is added, that text inserts itself at the beginning of the hidden text, not where the cursor seems to be located (which insertion causes, at the same time, Show/Hide to actuate and show all hidden text).
