    Just had the same issue after what I believe was an unsuccessful update to Windows 10 version 1709 Fall Creators update. 

    Changing the default Operating System solved the issue.

    First you need to get into the repair options. Easier said then done when your PC won't boot to Windows. 

    Download the Media Creation Tool from https://www.microsoft.com/en-gb/software-download/windows10

    Create the Windows 10 installer onto a hard drive or USB stick which has at least 8 GB free. 

    Once done, boot the PC from the hard drive or USB stick.

    In my case,  It will ask you if you want to perform an update (Yes) or a clean install (No). Click 'No'. This will take you through to Windows setup. If you don't get that message then Windows Setup will launch automatically.

    Windows Setup will start.

    Click 'Next' after you choose your preferred language.

    Click 'Repair This PC' in the bottom left hand corner.

    Click 'Troubleshoot'.

    Choose 'Startup Repair'

    If you have more than one Operating System (In my case I had the failed update and my previous Operating System) then you will see them listed.

    In the bottom left hand corner you will see 'Change the default Operating System' or something similar.

    Change the default to your previous Operating System.

    The PC should reboot to the newly assigned default Operating System.

    Windows and your PC should run as normal.

    Note: When I got into the login finally a message appeared saying my system restore was successful. I did try a few system restores before the above steps. Possibly a combination of a system restore and changing the default Operating System was the ticket to victory so try System Restore first.

    andre da costa say:

    Sounds like you purchased an Anytime Upgrade for WIndows 7. Sorry, you cannot use that to upgrade Windows Vista. An Anytime Upgrade for Windows 7 can only upgrade an existing edition of Windows 7 to a higher edition such as Windows 7 Home Premium to Windows 7 Professional.

    You need to purchase an 'Upgrade version'.

     

    Upgrade pricing:

    http://www.microsoftstore.com/store/msstore/html/pbPage.Windows_Upgrade_Matrix

     

    When making significant changes to your computer such as an upgrade, it is always recommended you backup your system prior to installing a new version of Windows. In your case, you can upgrade to Windows 7 depending on edition and architecture of Windows Vista you have installed. If you are running a 32 bit version of Vista and want to upgrade to a 64 bit version (edition of Windows 7), you will have to a do clean install, this applies vice-versa.

    If you are running consumer editions of Windows Vista such as Home Basic or Premium, you can only upgrade to either Windows 7 Home Premium or Ultimate. Also, you cannot do downgrades, which means, if you are running Windows Vista Ultimate now, you cannot downgrade to Windows 7 Home Premium. You must do logical upgrades:

    Windows Vista Home Basic > Windows 7 Home Basic, Premium or Ultimate

    Windows Vista Home Premium > Windows 7 Home Premium or Ultimate

    Windows Vista Business > Windows 7 Professional, Enterprise or Ultimate

    Windows Vista Enterprise > Windows 7 Enterprise

    Windows Vista Ultimate > Windows 7 Ultimate.
