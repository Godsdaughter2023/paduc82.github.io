kris ver say:

Thanks for the update, Kai.

This issue can occur if a program is interfering with the automatic update process. For example, antivirus programs that use real-time scanning can interfere with the automatic update process.

Issues with installing Windows updates can occur when Windows Update and another program both try to access the same file. Although many programs can cause this issue to occur, the most common are antivirus programs and programs that perform real-time backups of system files.

Let's fix this by closing all programs that are running, and then restart your computer. After the computer has started successfully, try installing the updates again. If the issue persists, start your PC in Safe Mode with Networking, and then install the updates. Safe Mode will start Windows with a minimal set of drivers and services to help troubleshoot issues.

If the issue still persists after these steps, perform a clean boot of the computer and then install the update. It will start Windows by using a minimal set of drivers and startup programs that can eliminate software conflicts that occur when you're opening a program or file.

On the article "How to perform a clean boot in Windows", you will learn the step-by-step process on how to perform a clean boot depending on the Windows version that you're currently using.
