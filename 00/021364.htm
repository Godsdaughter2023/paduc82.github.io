If you click on the Show\Hide (¶) button in Paragraph section of the Home tab of the ribbon, do the indents appear as a number of dots and only on the first line of each paragraph?

If so, if you use CTRL+a to select the whole document and then click on the Center Alignment button and then back on the Left alignment button in the Paragraph section of the Home tab of the ribbon, it will remove such indents.

If that plays too much havoc with the layout of the document, another way to remove them is by pressing CTRL+H to display the Find and Replace dialog and in that dialog click on the More >> button and then check the "Use wildcards" box and then in the Find what control, insert 

^13[ ]{1,}

Note, there is a single space between the [ ], and in the Replace with control, insert

^p

and then click on Replace all.  That will remove the spaces from all, except for the first paragraph in the document.

After you've done what Doug suggests to remove the indents created with spaces, apply a first-line indent to the paragraphs, either by selecting the paragraphs and dragging the First Line Indent marker on the horizontal ruler or, preferably, by modifying the style used for those paragraphs. e.j. gun say:

"Keep source formatting" does not mean keeping the same order.  It means keep the same number formatting, font color, borders, etc.  Use the following macro to take your selection and copy/paste it in the desired order.  The macro retains any cell formatting from the original selection.  It could be modified to copy/paste just the values if desired.

 

HTH,

 

Eric

 

Option Explicit

'

' Assumes the user has already selected the cells that should be copied,

' in the desired order.

'

Sub CopyPasteInOrder()
    Dim i As Long, j As Long, iCount As Long
    Dim nCells As Long
    Dim selRange As Range, destCell As Range
'
' First save the user's selection
'
    Set selRange = Selection
'
' Now ask user to select a destination cell
'

    On Error Resume Next ' If user presses cancel...
    Set destCell = Application.InputBox(prompt:="Select the destination cell for this operation", Title:="Copy/Paste In Order", Type:=8)

    On Error Goto 0
'
    If (destCell Is Nothing) Then Exit Sub
'
' Copy each cell to the new location in the user-selected order
'
    iCount = 0
    nCells = selRange.Cells.Count
    For i = 1 To selRange.Areas.Count
            selRange.Areas(i).Copy Destination:=destCell.Offset(iCount, 0)
            iCount = iCount + selRange.Areas(i).Cells.Count
    Next i
End Sub

They are two different editions of Office suite. Microsoft Office 365 en-us is from Office 365 Home plan. Microsoft Office 365 ProPlus is the Office suite from Office 365 for Business plan. If it's convenient and both these two editions belongs to you, you may remove one of them and keep the other one on your computer.
