"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. to update (change files)  in the game.

installer and go to 'Properties'. Then select the 'Compatibility' tab. At the bottom is an option to

Run as Admin.

executable ( .exe), or short-cut you can also try using the Run as Admin Compatibility option

via one of these files.

- Or, try creating a new folder in C:\   and  installing the game to that location. This will circumvent

any permissions issues that may arise from installing to any of the three 'Programs' Folders.

You should get a prompt at the beginning of installing the game to browse for a 'custom' (or similar term) location. Browse to your new folder and have the game installed there.

    lemp say:

    MalwareBytes should have removed WeWatcher more cleanly.  See --> https://forums.malwarebytes.org/index.php?/topic/170602-removal-instructions-for-wewatcher/

    However, assuming that you have Windows 7:

    1. Click the Start Orb and type "cmd.exe" (without quotes) into the Search box
    2. Right-click on the link for cmd.exe and select "Run as administrator."  Respond as necessary to the UAC prompt.
    3. In the black Command Prompt window, type the following lines and press Enter after each line:
          netsh winsock reset catalog
          netsh int ipv4 reset C:\resetlog.txt
          netsh int ivp6 reset C:\resetlog.txt
    4. Reboot the machine

    That should clean it up.
