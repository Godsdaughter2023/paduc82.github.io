richardholroyd say:

Open Excel

Click on start (big round windows button top left corner)

At the bottom middle of the dialogue box should be a button saying Excel options, click this.

In the Excel Options dialogue box that should have opened, in the left hand Column, click the Save tab.

 At the top of the dialogue box should be Save files in this format and next to it a drop down box with options in it.

click the option you think you want, probably just the first one - Excel Workbook.

click ok to close the dialogue box and that should do it - you may need to shut then reopen the program for the change to take effect.

whilst you are in the dialogue box you can also change where the default file is saved too if you have you own filing system rather than using the windows one (the windows one can make it difficult to find your files again).
