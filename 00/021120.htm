=IIf([Report].[Filter]<>"" And [Report].[FilterOn],Replace("Filtered to show: |","|",[Report].[Filter]))

djsteele, mvp say:

If a filter is being used, that expression returns the phrase Filtered to show: <xxx>, where <xxx> is the actual filter being used. If a filter is not being used, it returns nothing.

Changing that expression won't change what records are selected for your report in any way, shape or mean. You need either to change the query on which the report is based, or pass a Where condition to it when you're opening it.
