It appears that (after much work) I seem to have solved the problem. However, in order to help others who may have a similar (very frustrating) issue, I will include answers to your questions:

    1) Is this the first time, you are facing this issue? If yes, then what are the recent changes that you have made to the computer prior to this issue?

    No…it has been happening for a while. It started as an occasional random event, and built up to totally annoying over the last few months.

    2) Do you have any third party security program installed on the computer?

    Microsoft Security Essentials (I use MalwareBytes and Ad-Aware to scan for viruses, etc. as well). None of these yielded any indication of a virus or other malicious software.

    3) When exactly, do you face this issue? Is it when you leave the computer idle for some time or does this happen even in the middle of your work?

    The problem occurs at what seems to be somewhat random times, but quite often (and frustrating). It is definitely noticeable when in the middle of trying to type something.

Before the issue cropped up, I had downloaded CA Nimsoft Monitoring Software (as I was trying to resolve some network issues). The software did not do what I needed and I simply ignored the program, rather than remove it.

After much frustration scouring the web for an answer, I found a reference to looking at errors using the Windows event viewer. I had known about event viewer (after listening to a scammer from India trying to get me to download a virus – another story), but had never actually used it before.

After seeing the high number of critical errors (i.e. about 800/day) relating to an “Application Error code 1000”, I deduced that this must be the issue and that each time it crashed, it temporarily grabbed the display and then released it after logging the error (or if I clicked the mouse back into the affected window – whichever happened first).

I then examined the error (by double clicking on a couple of the “Error 1000” lines in WEV). From the information, I found the specific program from the faulting application and module directory paths. I then simply uninstalled the application, rebooted the computer and voila, the errors stopped being produced in WEV and the problem seems to have totally disappeared.

I hope this thread may help others understand a bit more how to self diagnose and resolve issues that occur on their computers. Start with the Event Viewer (Start -> Event Viewer) and if you see individual high event log entries, search the web to find out if someone else has seen such issues. Try uninstalling any programs that are contributing to those events to see if such issues disappear. My kudos to Microsoft for providing such a powerful tool.

Hi John,

 

The Xbox  Avatars can be customized easily either from web or from your Xbox console but, with the availability of Xbox games for Windows 8/RT, Microsoft has also provided the option to customize your Xbox  Avatar right from your Windows 8/RT device.

 

Windows 8/RT comes with the Xbox Games app which lets you see all your Xbox account activity like games, game achievements and new releases etc. Xbox Games app also include the options to customize your profile including your Avatar.

 

Follow these steps and check if it helps:

 

a)      open up the Games app by going to Start Screen and typing ‘Games’ and the Games app will show up in results. Press it.

b)      Once the Games app is open, you will automatically be signed-in with your Microsoft account. But, if it doesn’t happen then, you will manually need to sign-in with your Xbox/Microsoft account. Press the arrow button against ‘Network Status‘ on top right corner which will sign-in to your Microsoft account.

c)       Once signed-in, navigate to the start of the app. Here you will see your account details along with the Xbox avatar and other information. Select ‘Customize avatar’.

d)      You would be taken to another page entitled as ‘Customize avatar’. Here you can customize each and everything of your avatar from clothes, body parts to jewelry and other things.

e)      Once you are done customizing your avatar, press the ‘Save’ button to save the changes you have made.

Note: Your customized avatar will sync across all your connected devices like Xbox, Windows Phones and Windows 8/RT PC’s.

 

You may also post your query at Xbox Companion App forum for more information:

http://forums.xbox.com/xbox_forums/xbox_apps/t_-_z/f/2707.aspx

 

Hope this helps. If you need further assistance with Windows, let us know and will be glad to help.

Thank you for posting your query in Microsoft Community.


I understand your concern and will guide you in the right direction.


I suggest you to contact manufacture website to download Smart Label Printer drivers.


Also if windows 10 drivers are not available you can install the drivers in compatibility mode.

a. Right click the driver installation file and select Properties then tap or click the Compatibility tab.

b. Place a check in the Run this program in compatibility mode for: box.

c. Tap or click the drop down box and select a previous operating system, then tap or click OK.

d. Try installing the driver and check.
