jeeped say:

The default setting for Warn if changing between secure and not secure mode is OFF. The recommended setting is ON. The setting itself can be found by tapping the ALT key, then selecting Tools, Internet Options from the menu bar. On the Advanced tab, in the Settings list, scroll all the way to the bottom Security section. The setting should be at or near the second from the bottom of the list.

This setting governs the warning message for both a Web site having a mixture of HTTP and HTTPS links, and being sent from an HTTPS site to a non-secure, HTTP, site.
