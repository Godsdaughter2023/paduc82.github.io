  4. Click Finish.

Tackle the possibility of malware first.  If it turns out to be incorrect i.e. there is a technical system fault that is causing the problem, see the Startup repair notes at the end of this post.

 

1  When you suspect that your PC is infected with malware, run the following checks.  I suggest multiple scanners because some malware can hide itself from some particular scanners some of the time or even disrupt scanner installation but it would be difficult to survive every scan from an array of tools.  There is excellent malware checking & removal advice in the MS Community forum Virus and Malware.  If you find anything during the set of scans recommended below then follow the removal advice that the scanner gives you and then get a second opinion by checking in that forum as well.  It would not be ridiculous to repeat the following scans after cleaning out malware until you get a full set of clear results from them all.

 

2  Update your installed anti-malware utility and do a “full” scan using it [you may well have set it up to do only “quick” scans normally].  Some, such as MS Security Essentials, can also be run in Safe Mode to detect malware that can hide itself during normal operations but not all anti-malware utilities run properly in Safe Mode & you’d have to check the Help file or their website to find out about any particular one.

 

3  Scan with the onboard MS Malicious Software Removal Tool [MRT].  Go to the Start menu search ox, enter mrt then right-click on the shortcut offered to you & select Run as Admin.  This tool is updated as part of Windows Update so your onboard version should already be current.  Run a full scan with it.

 

4  Download the Microsoft Safety Scanner and run a full scan with it.

 

5  Download the free version of Malwarebytes, install it & update it [for both of which you will need Admin approval] and run a full scan using it.  Note that this free version of Malwarebytes can exist alongside your installed anti-malware application so it is always useful to retain it for use as an onboard second opinion. Do not run Malwarebytes in Safe Mode as too few Malwarebytes components load for it to give useful results.  If you have a failed or suspect installation of Malwarebytes then clean it up before trying again - Use mbam-clean.exe to completely remove Malwarebytes Anti-Malware.  If you have difficulty installing Malwarebytes because of the malware infection then  Use Malwarebytes Chameleon to install Malwarebytes Anti-Malware on an already infected system[Malwarebytes Chameleon is a component of Malwarebytes].

 

6  Make a Windows Defender Offline boot disk [the initial download file is a small one that you then use to download the main installation file], update it, boot into it & run a full scan using it.  Windows Defender Offline can be particularly useful for removing a type of malware called a Trojan.  Whilst your system security can prevent a lot of problems, if you have been running the PC whilst logged in to an Admin account then you will have made it easier for Trojans to get into your system - see Why use a standard user account instead of an administrator account?  [you can also see this information in the Help file of the Computer management dialog].

 

7  Download SuperAntiSpyware Portable Scanner Personal Edition, install it [there is no need to update it] and run a “complete” scan.  You can save it to a USB / DVD / CD in order to run it from any PC.  You would need to download a new copy if you wanted to use it again in the future as its definitions were only current when it was downloaded.

 

8  A PC that has been cleaned of malware should then be restored from an image & all user files ought also to be restored.  The malware cleaning should only be regarded as ensuring that no malware can slip into the restored system.  If you have to rescue updated user files after the cleaning then scan them before putting them back onto the PC.

 

9  If you don’t make system images then the drastic tasks of restoring-to-factory or doing a new installation of Windows, reinstalling your applications as well as backing up then restoring user files ought to be undertaken.  If you have to rescue updated user files after the cleaning then scan them before putting them back onto the PC.

 

10  If you are not going to restore the complete system after cleaning all malware then at least do a conventional scan of the hard drive file & folder structures [see 10.1] and run SFC [see 10.2].  Do consider running a Repair Install as well [see 10.3].  If you have to rescue updated user files after the cleaning then scan them before putting them back onto the PC.

 

10.1  Check the drive[s] for errors - Windows explorer, Computer, C:\ drive, right-click, Tools, Error checking - Check now, enter an Admin account name & password.  Clear both checkboxes for an initial scan.  If faults are found then set the checkbox for Automatically fix file system errors and then click on Schedule disk check to run the scan the next time you start Windows.  [If you want the finer control that is possible when running chkdsk from an elevated Command prompt, first check the basic syntax by entering chkdsk /? in a Command prompt and then review the guidance given at Chkdsk - MS TechNet.]

Recycle Bin could have gone corrupt

 

You may have to reset the Recycle Bin store folder to fix problems such as Recycle Bin not showing the files that have been deleted, or you’re unable to completely empty the Recycle Bin. Each drive has a hidden and protected folder named $Recycle.bin, which is where the Recycle Bin stores files and folders that you (and other users in the PC) delete. Windows 7 creates a fresh $Recycle.bin folder automatically after you reset it.

 

Follow the methods listed below to resolve the issue:

 

Method 1: Fixing a corrupt Recycle Bin

 

IMPORTANT: The following procedure permanently deletes all files and folders which are already in the Recycle Bin (from all user accounts in the system) and cannot be recovered.

 

1. Open an elevated Command Prompt window. To open an elevated Command Prompt, click Start, type cmd, right-click Command Prompt, and then click Run as administrator.

 

2. Type the following command and press ENTER:

 

rd /s /q C:\$Recycle.bin

 

(This clears out the $Recycle.bin folder from the C:\ partition. You may have to do this for each hard disk partition in your system depending on the corruption)

 

3. Exit the Command Prompt window.

 

Method 2:

 

If the above steps do not resolve the issue, I’d suggest you to perform a System File Checker (SFC) scan on your computer. SFC tool scans system files and replaces incorrect versions of the system files by using the correct versions.

1.     Click Start, and then type cmd in the Start Search box.

2.     Right-click cmd in the Programs list, and then click Run as administrator.

 

If you are prompted for an administrator password or confirmation, type your password or click Continue

sfc /scannow
