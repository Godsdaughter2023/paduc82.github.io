mark ole say:

Hi FloDN,

Many devices use the 2.4 GHz band because it is better suited to longer ranges and transmission through walls and solid objects. However, because of so many devices use the 2.4 GHz band, this may result to signal interference and can cause slower expected speed or even dropped connections. Unlike the the 5 GHz band which is much less congested and has a shorter effective range, would likely give you more stable connections.

If the issue is not related to signal interference, running the network troubleshooter can help diagnose and fix common connection problems.

To run the troubleshooter, follow the steps below.

  1. In the search box on the taskbar, type Network troubleshooter, and then select Identify and repair network problems from the list of results.
  2. Follow the steps in the troubleshooter and see if that fixes the problem.

If that doesn’t fix your connection problem, try these things:

  • Reset the TCP/IP stack
  • Release the IP address
  • Renew the IP address
  • Flush and reset the DNS client resolver cache

To run these networking commands in a command prompt window

  1. In the search box on the taskbar, type Command prompt, press and hold (or right-click) Command prompt, and then select Run as administrator > Yes.
  2. At the command prompt, run the following commands in the listed order, and then check to see if that fixes your connection problem:

    • Type netsh winsock reset and press Enter.
    • Type netsh int ip reset and press Enter.
    • Type ipconfig /release and press Enter.
    • Type ipconfig /renew and press Enter.
    • Type ipconfig /flushdns and press Enter.

If the steps provided didn’t work, try to run the Surface Diagnostic Toolkit. For more information on how to run the toolkit, click here.

Let us know if you need further assistance.
