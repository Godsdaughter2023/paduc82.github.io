a. user say:

In a successful dual boot setup you have, in effect, two separate computers. Upgrading Windows 8.1 has nothing to do with (and no effect on) Windows 7.

If you want to reinstall Windows 7 in the future, go right ahead. Windows 10 (formerly Windows 8.1) doesn't care.

The procedure for dual booting happens before the operating system loads, so whatever operating system you choose to load is like Sgt. Schultz about the other operating system. ("I see nothing! I hear nothing!)

Ordinarily, each copy of Windows needs to have its own license, but when you upgrade Windows 8.1 to 10 you don't need a license. Instead you have what Microsoft calls a "digital entitlement" to activate Windows 10.

Hi Mike,

The cause for the Windows Installer to stop working is because the Windows Installer files that are on your hard disk are damaged or are missing. Let us repair/re-install windows installer,

Now to re-install Windows Installer, rename the damaged Windows Installer files, and then reinstall the Windows Installer. To do this, follow these steps:
1. Press Windows button + R, type cmd in the Open text box, and then click OK.
2. Type cd %windir%\system32, and then press ENTER.
3. Type attrib -r -s -h dllcache, and then press ENTER.
4. Type ren msi.dll msi.old, and then press ENTER.
5. Type ren msiexec.exe msiexec.old, and then press ENTER.
6. Type ren msihnd.dll msihnd.old, and then press ENTER.
7. Type exit, and then press ENTER.
8. At the command prompt, type exit, and then press ENTER.
9. Click Start, and then click Turn off computer or Shut Down.
10. Select the Restart option, and then click OK, or click Restart.
11. Log on to the computer.
12. Download and install the latest version of the Windows Installer from the link below.
 
Windows Installer 4.5 Redistributable
http://www.microsoft.com/downloads/details.aspx?FamilyID=5a58b56f-60b6-4412-95b9-54d056d6f9f4&displaylang=en

Let us download Windows Installer Cleanup Utility which will clean up any leftover installer files of iTunes that may be causing issues with your current installation. To do this:

Download Windows Installer CleanUp Utility
Download the Windows Installer Cleanup Utility package now.

1. Open the msicuu2.exe file and follow the prompts to install it.
2. Choose Start > All Programs and then click Windows Install Cleanup; the Windows Installer Clean Up Utility window appears, listing software that is currently installed on your computer.
3. Select iTunes from the list and click Remove.
4. Click OK in the confirmation dialog that appears.

This will uninstall your iTunes completely.


Once you’ve the latest version of Windows Installer, try installing iTunes and it should work.
