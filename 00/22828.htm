bob jones aka: cybertaz say:

These 6 characters tripled produce individual styles of Bottom Paragraph Border:

~   #   *   =   _   -

Not to be pedantic, but it's important to realize that these are paragraph borders rather than "horizontal lines"... especially when it comes time to remove one :-) Those as well as other styles also can be applied in the Format> Borders & Shading - Borders dialog.

Horizontal lines are drawing objects which can be selected & deleted in the same way as rectangles or other shapes. They can also be formatted using most of the same formatting tools used  on other shapes.

Some of the most basic paragraph borders can be applied using the Borders button on the Home tab of the Ribbon, as well.

    Dim UserName As String 'The current users notes name

    Dim MailDbName As String 'THe current users notes mail database name

    Dim MailDoc As Object 'The mail document itself

    Dim AttachME As Object 'The attachment richtextfile object

lapstam say: Trust me Mauricio, I feel your pain :)

I would proceed as follows, hopefully it can give you any idea about how to move forward.

Now, you're already familliar with most of the steps and it might be a good idea to first use it as a check list for steps you've done or missed or might wish to try one more time.

To save you from the time it takes to reinstall software, you can of course contact the phone makers support, if this option is available. You want them to confirm that the phone is compatible with Vistas synchronization center (take a look at this control panel and its help section) and if it is, help you to get the phone connected to the pc with the mobile device center. If it is supposed to work but doesn't and they cannot offer any help, you should consider whether it is a hardware fault and use your rights of return and guarantee, if this is options available to you.

1. Throw out all connections in the mobile device center software, then uninstall the application again.
2. In the device manager, uninstall every mobile/portable device, then forget about this settings, when you start all over Vista should take care of all this.
3. Uninstall any other mobile phone synch software you might have on the machine.
4. Disconnect any usb-hub you might have connected to the pc. Then disconnect all other usb units that is not essential at the moment.
5. If you're not using a top virus/spyware/firewall package, uninstall all such stuff for now.
6. Go to start > run and type "msconfig" (without quotes" and press enter. You can check the services tab to make sure all windows services are activated; you can check the autostart tab so that you don't have any non essential software starting automatically, but unless you have a lot of weird stuff on the pc it should be enough to choose "Normal" in the first tab, which will make sure everything that should start will start when you boot the pc.
7. Power off the pc, start it up.
8. Use Windows update and search for new updates.
9. Check your pc-makers site if they have any new usb drivers available for your machine, or docs relating to known issues. You might also want to check the device managers usb section so that there are no warnings about conflicts.
10. Check with your phone maker if they have updated software for the device. You might even want to reinstall the ROM unless you recently did this.
11. Install vistas mobile device center again, the reboot.
12. In the windows mobile connection settings, usb to pc, use the default active synch and the advanced network functions. If this was not what you had, choose it and restart the phone.
13. Connect the usb cable to the pc usb port, preferable one on the back. The pc should give off a sound when it is connected.
14. Conect the mini usb to the phone.
15. Open vistas controlpanel synchronization center.
16. Start vistas mobile device center.

If the drivers do not install or the phone is not connected, use the refresh button in the synchronization center, then the ?help button to follow advice to troubleshoot further.

You can also disconnect the phone, and in step 12, tick off the advanced network functions, then reconnect the phone to the pc.
Make sure also that the pc system and windows mobile is in synch with a time server, so the internal clocks are showing the correct time and date.

Excel adjusts the color slightly, how "slightly" varies. You are being warned so if/when you notice the change you have a clue what happened.

Proceed by saying yes, letting it make the change. When you open the file, take a look at the colors and decide if the change is too much. If it is too much, try a different color and see what excel says.
