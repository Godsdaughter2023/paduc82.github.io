andre da costa say:

Under the terms of the Windows 10 OEM license agreement, no, but you could try phone activation as a work around/loophole. It has worked.

OEM versions of Windows 10 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

1. Press Windows key + X then clickRun, then type: slui.exe 4

2. Next press the 'ENTER' key

3. Select your 'Country' from the list.

4. Choose the 'Phone Activation' option.

5. Stay on the phone (do not select/press any options) and wait for a person to help you with activation.

6. Explain your problem clearly to the support person.
