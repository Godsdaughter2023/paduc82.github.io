If you have a table (called MyTable) and in this table you have 3 fields
WebPageID  =- Autonumber
AddressField = Text
WebPageName = Text

 

So (record example)
WebPageID  = 1
AddressField = www.microsoft.com
WebPageName = Microsoft

 

WebPageID = 2
AddressField = www.google.com
WebPageName = Google

 

On you form add a list box (called MyListBox)

Source =
SELECT MyTable.WebPageID, MyTable.AddressField, MyTable.WebPageName FROM MyTable;

 

Set the column widths for this list box like this
0cm;0cm;3cm
Set the bound column to 1

Put this code on the AfterUpdate event of the list box

Private Sub MyListBox_AfterUpdate()

Application.FollowHyperlink _

"http://" & Me.MyListBox.Column(1)

End Sub

 

Now click a name in the list box and the web page will open

Sub AutoOpen()
    On Error Resume Next
    With ActiveWindow.View
        If Err.Number = 4248 Then Exit Sub
        .Type = wdPrintView
        .TableGridlines = True
        With .Zoom
            .Percentage = 100
            .PageColumns = 2

    End With
