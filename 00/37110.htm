martmcd say:

much as I agree with the suggestion of adding to the thousands that have already sent feedback on this issue, there are ways to take back some control over the updates, in addition there are other things you can do to reduce the amount of data used automatically

have a read of Tryx3's detailed article Manage internet data usage and Windows updates in Windows 10

It is not a good idea to simply turn them of or stop the updates altogether, they include security updates as well as new features. checking only once a month is better than not at all, I suggest you set it to check as often as you can within your data allowance.

assuming you are running windows 10 home (in professional you can use group policy to inform of updates but not download them)

first set active hours to cover the time you are actually using the computer (or as much as you can)

note - the following will stop you being able to check for updates using the button in settings, to do a manual check you will have to run the batch files as an administrator

then create a couple of small batch files and then use task scheduler to run them at the desired time, outside the active hours.

First create a directory to put the batch files, (where you save them is up to you, as long as they are easy to find and accessible)

right click start and select command prompt (admin)
type cd\ and press enter
type md BatchFiles and press enter
close the command prompt.

write a batch file to stop and disable the Windows Update Service.

Open Notepad and type (or copy and paste) the following

@echo off

net stop wuauserv

sc config "wuauserv" start= disabled

quick explanation, @echo off - prevents the process being displayed on the screen, net stop - stops the service, wuauserv - is the windows automatic update service. sc config "wuauserv" Start=disabled - disables the service, stops it restarting.

disabling the service while it is still running does not stop the service

save the file as UpdateOff.bat in C:\BatchFiles. Note - the extension (the bit after the dot) needs to be bat in order for the system to see it as a batch file.

write a batch file to Start the Windows Update Service and check for updates.

Open Notepad and type (or copy and paste) the following

@echo off

sc config "wuauserv" start= auto

powershell -(New-Object -ComObject Microsoft.Update.AutoUpdate).DetectNow()

save the file as ForceUpdate.bat in C:\BatchFiles. Note - the extension (the bit after the dot) needs to be bat in order for the system to see it as a batch file.

Use task scheduler to run the batch files to get windows updates.

Open Task Scheduler and from the right hand pane, select Create Task.

Give it a name (that means something) and description.

Under Security options select a user with administrative permissions, run whether user is logged on or not,  Run with Highest privileges and configure for Windows 10.

Click on the Triggers Tab and select New

this is the 'Start the Service and Force Update' schedule so make sure to select a time that you want it to get Updates. (03:00)

Select Daily, select a time that you are happy to start the Windows Update service and click Ok, in this example I have set it to start shortly after the end of the Active Hours. (You will want 03:00, or just after)

Click on the Actions tab and click New.

type C:\BatchFiles\ForceUpdate.bat into the program/script box..

Click Ok

Click on the Conditions tab and tick the 'wake the computer to run this task' to ensure that sleep does not prevent updates from being installed.

I will let you decide on other Conditions and Settings - they will only affect this task if you change anything.

Click OK

it will ask for User Password - give it the details of an administrator account.

Back in Task Scheduler, from the right hand pane, select Create Task.

Give it a name (that means something) and description.

Under Security options select a user with administrative permissions, run whether user is logged on or not,  Run with Highest privileges and configure for Windows 10.

Click on the Triggers Tab and select New

this is the stop and disable the Service' schedule so make sure to select a time that you want to prevent any Updates. (07:00)

Select Daily, select a time that you are happy to prevent Windows Updates and click Ok, in this example I have set it to stop updates at the start  of the Active Hours. (you will want 07:00, or just before)

Click on the Actions tab and click New.

type C:\BatchFiles\UpdateOff.bat into the program/script box..

Click Ok

Click on the settings tab and tick the 'Run the task as soon as possible if a scheduled start is missed' to ensure that it will run even if it misses the schedule (if the pc is turned off)

I will let you decide on other Conditions and Settings - they will only affect this task if you change anything.

Click OK

it will ask for User Password - give it the details of an administrator account.
