Where do I store CUSTOM/Personal Word 2007 templates and how do I - Microsoft Community

sallyschilling say:

Where do I store CUSTOM/Personal Word 2007 templates and how do I create a new one?

Next if I want to create a new template, I know that I can do a "Save as" and use the template file extention, but where do I store it (see question above)?

jay freedman say:
Yep, Word 2007 got screwed up on this, and it's the same in 2010.  :-P

The first thing to know is that there's a specific location known to Word as the User Templates folder. That's where all your existing and new templates should be stored.

Open the Options dialog, click Advanced, scroll to the bottom, and click File Locations. In the next dialog, click on the User Templates item in the list, and click the Modify button (you don't usually want to modify the location, but this is the only way to see the complete path). By default the Office installer makes this point to the folder
   C:\Users\<your name>\AppData\Roaming\Microsoft\Templates
Because the AppData folder and everything under it is marked as Hidden, you won't normally see this in the file explorer. There's a nice way to get there easily, though:

While this Modify Location dialog is open, move your cursor to the top of the dialog where the  path is shown, and click on the word Microsoft (this takes you up one level). Scroll the left side of the dialog all the way to the top so you can see the Favorites item, and expand it. Scroll the right side down so you can see the Templates folder. Drag the Templates folder from the right side to the left and drop it in the list under Favorites (a horizontal black bar shows where it will drop).
From now on, in the file explorer and in every Open or Save As dialog that looks like the file explorer, that Templates item will be in the Favorites list, and clicking it will take you to that folder.

In the Save As dialog of all older versions of Word, if you set the "Save as type" dropdown to template (.dot), the dialog automatically switched to the Templates folder. Word 2007 and 2010 don't do that -- they just open in the last-used folder, which is usually a place for document storage. When you need to store a template, click the Templates item in the Favorites list.

Here's another tip: To use one of your templates, the recent Word versions want to force you to click the Office button, then New, and then clicking My Templates. That's two clicks too many. Choose to modify the Quick Access Toolbar. Set the category to All Commands, click on the "New Document or Template..." item, click the Add button in the middle of the dialog, and click OK. A single click on the button you just created will go directly to the New dialog where you can choose from the files in the User Templates folder.

lars-Åke aspelin say: For a general probability distribution the expected value E(X) as defined by the average:
  E(X) = sum where k goes from -8 to 8 of P(x=k) * k   (8 should be read as infinity here)
does NOT mean that the probability P1 to get a value below or equal to E(X) which is defined by:
  P1 = sum where k goes from -8 to E(X) of P(x=k)
equals 0.5

This just happen to be the case for the normal distribution but it is not the case for the geometric distribution or any distribution in general. It is however true for all "symmetrical" distributions, i.e. where
P(E(X)-k) = P(E(X)+k) for all values of k.
 
The "quartiles" are defined as intervals to each have 1/4 of the probability.
There is no general relation between standard deviation and the quartiles.
The probability to get a value between E(X)-d and E(X)+d where d is the standard deviation can be anything between 0 and 1. For the normal distribution it is 0.68...
