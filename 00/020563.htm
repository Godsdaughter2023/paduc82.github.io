lars-Åke aspelin say:

I am trying to calculate time in quarter hour increments, but I have run into a problem when the time spans an overnight. I can get the time capture for the overnight hours (11p-7:30a), but my formula won't calculate at 8.5, it gives me hours and minutes. I am using =IF(B2<A2,B2+1,B2)-A2

I can get the following formula to work as long as the time doesn't cover the overnight (11p-7:30a) or a 12-12 shift. 
=TIME(HOUR(SUM(B2-A2)),MROUND(MINUTE(SUM(B2-A2)),15),0)*24

I can't figure out how to nest them without getting an error.  Any ideas on how to make this work? 

Try this formula:

=TIME(HOUR(IF(B2<A2,B2+1,B2)-A2),MROUND(MINUTE(IF(B2<A2,B2+1,B2)-A2),15),0)*24

And don't forget to set the format of the cell where you put this formula to "General" or "Number" if you want the result as 8.5 rather than 08:30.

To change format of a cell, right click on the cell and choose "Format Cells..." from the popup window.

Edit: A shorter formula that will yield the same results is the following:

=ROUND(96*(IF(B2<A2,B2+1,B2)-A2),0)/4

Hope this helps / Lars-Åke

lars-Åke aspelin say:

Thank you so much for your help.  The only shift this does not work on is the 12-12.  It returns a #num error. Any ideas on this?

Check your input in cells A2 and B2. If it is 12:00 in both cells the result should be 0.

Please not that the formula with TIME, HOUR, and MINUTE is only capable of handling time differences less than 24 hours.

The formula with ROUND can handle longer time differences provided you enter the start and end times in A2 and B2 with dates, like

2017-08-16 12:00 in A2 and 2017-08-17 12:00 in B2.

gloria gong msft say:

Hi DeeAble,

Here is an article for your reference: OneDrive retention and deletion

You can delete the manager of the user in SharePoint Admin Center:

1. Sign in Microsoft 365 Admin Center> Admin centers in the left pane> SharePoint.

2. Click user profiles in the left pane> People> Manage User Profiles.

3. Search for the user who is deleted in "Find profiles"> Find.

4. Right click the user> Edit My Profile> In the Manager row, you will see the user's manager> Delete the manager> Scroll down> Save and close.

Next time when you want to delete a user, you can first delete the manager of the user.

In addition, if a manager isn't specified for the user account, but a secondary owner was entered in the SharePoint admin center, the secondary owner will receive an email telling them they have access to the OneDrive, and that the OneDrive will be deleted at the end of the retention period.

You can set the secondary owner by following the steps below:

1. Still in SharePoint Admin Center> User profiles in the left pane> My Site Settings> Setup My Sites.

2. Scroll down and you will see "My Site Cleanup"> Add the secondary owner in the right> Scroll down> OK.

Gloria

Using Track changes cause very fragile document. Its actually recommended on Mac to compare documents instead of Track changes. Each time you make chages or add comments and don't accept changes and save the document gets more and more complex sometimes to the breaking point of the document.

If you can get the document open Go to Track changes accept or reject changes, turn off track changes then use Save As to save  a Replacement document.

 If you need track changes get in the habit of frequently accepting or rejecting changes and saving after doing so (use save As instead of Save).

If Track Changes is not the issue.

  1. Then shut down any office Applications. 
  2. Run Disk Utilities and repair permissions.
  3. Next Run Font book and select all Fonts , the in list of fonts click on 1 font then use ⌘ - A (Select all).
  4. Go to File menu and choose Validate fonts.
  5. If any duplicates are found or any defective fonts are found then Remove them (fon't just turn them off especially if you have Yosemite. It has no tolerance what so ever with even turned off duplicates. No exceptions.
  6. Once you have resolved dupes ad defective fonts. Quit FontBook
  7. Quit all other application and restart Computer then try Office applications.
