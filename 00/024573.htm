Quit Outlook if it's running

Checked and repaired permissions - just in case.

Using the DB Utility, run directly from the Applications>MS Office 2011>Office folder I created a new identity. When I first started out on this task, I was cautious about what to call the Identity. If anyone else is, then don't be. It appears to be entirely arbitrary (but I am happy to be disabused by the more experienced amongst you good people).

I deleted the Documents>MS User Data>Office 2011 Identities>New_Identity_Name>Data Records file called Database.

I copied folders from the old identity one at a time to the new identity. After each copy, I did a rebuild. This was to narrow down the location of the corruption. So Contacts, Events, Mail Accounts, Notes, Signatures were proved clean. I didn't have any Signature Attachments btw, but I would have copied that too if I had.  OL now started at least and had all the content I expected - but was Mail-free.

bob jones aka: cybertaz say:

Please see my reply to your other related post.

The .odp file type is proprietary to the software used to create the file. That file type is not supported by any version of Mac PowerPoint, although later versions of Windows PowerPoint [I believe Office 2010, 2013 & 2016 for Windows) can read it.

The .ppt file type you referred to is an archaic format for which Mac PowerPoint no longer has the necessary filters. You may be able to open the file on a Windows PC equipped with Office 2003 or later, then save the file again in the newer version of the .ppt file type. However, it would be preferable to use Office 2007 or later & save a copy using the current .pptx file type which is the same as used by Mac Office 2008 & later.

jim gordon mvp say:

NeoOffice is one of many "forks" of OpenOffice from Sun Microsystems (now defunct). The default format for saving in NeoOffice is .odp, an open source file format and a popular standard among open source software fans. As has already been mentioned, this file format can be read in some versions of Microsoft Office for Windows but no versions of Office for Mac.

In NeoOffice use File > Save As and choose PowerPoint (.ppt) file format. All versions of PowerPoint on the Mac can open and work with this file format. Files in (.ppt) file format have limited feature support, so PowerPoint users will see "Compatibility Mode" in the title bar when working with these files. PowerPoint features not supported by .ppt format will be disabled in the PowerPoint interface.

If you prefer using open source software and want a more robust feature set, use LibreOffice. LibreOffice can use the current open standard file format (.pptx) for presentations. PowerPoint on the Mac and in Windows work with .pptx files.

More about Mac PowerPoint file formats:

To Reconcile two excel sheet in an excel file - Microsoft Community

polachan say:

To Reconcile two excel sheet in an excel file

Is there any chance to use index.. match or else any other formula to find  the record , to make it faster searching  by avoiding the for loop  in the given macro

Sub tryme()
 Worksheets("Supplier").Activate
 lastrow = Cells(Rows.Count, "A").End(xlUp).Row
 For k = 2 To lastrow
   test = Cells(k, "B")
   Set Rng = Worksheets("Statement").Range("B1:B200")
    If Not IsError(Application.Match(test, Rng, 0)) Then
        rowNum = Application.Match(test, Rng, 0)
        For j = 1 To 3
          Worksheets("Statement").Cells(rowNum, j).Copy
          Worksheets("Supplier").Cells(k, j + 5).PasteSpecial
          Application.CutCopyMode = False
        Next j
        Worksheets("Statement").Rows(rowNum).EntireRow.Delete
     End If
 Next k
End Sub
