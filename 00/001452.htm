Dear All,

How can I establish a network with Windows 8.1 PC & Windows 8.1 Phone using Wifi modem **** router.  Do I need to install an app in phone and PC? or any method to do so?

sven johannsen say:

Don't know of any way to do file transfers from a Windows Phone to a Windows PC over WiFi on the same network. The phone doesn't support the sort of networking features that would require, being a host on the network, (you don't see the phone in network devices in file manager on the PC) and enabling file browsing on the device.

You can transfer file directly using BT, but that is a one at a time effort, and most folks find it difficult to get going. You pair the devices, and then if going from phone to PC, you select the file, share via BT, select the PC, send, then you must accept on the PC. The BT connection does not stay active, it establishes for the transfer then disconnects once complete. That confuses people.

The best way to share files is to use OneDrive. Or do it USB.
