myveryownself say:

Here’s one way to do the first part -- the sorting -- using extra columns. The extra columns are out of the way on a separate sheet named "sort"

Sheet1 contains the original data.

In sheet "sort" put in A2:

  =IF(Sheet1!A2="","",1000000*Sheet1!B2+ROW())

and into B2:

  =IF(A2="","",--RIGHT(SMALL(A:A,ROW()-1),6))

Then copy A2:B2 downward as far as the list can become.

This approach makes sure that if a date is repeated on several rows, each row is dealt with separately.

In Sheet2 put in A2:

  =IF(Sheet1!$A2="","",

      IF(INDEX(Sheet1!$A:$Y,sort!$B2,COLUMN())="","",

          INDEX(Sheet1!$A:$Y,sort!$B2,COLUMN())))

Then copy A2 downward and rightward as long and wide as the list can become, leaving room for the "more data" mentioned in the original post. If $A:$Y in the formula is too narrow, adjust accordingly.

The first IF deals with the case of Sheet1 having fewer rows that the maximum possible.

The second IF deals with a data cell in a valid row being empty.

Note that if changes are made in Sheet1, the "more data" added to Sheet2 might no longer be in the proper row.

julian marsh say: Hi,

I don't know if you are still interested in the answer to this question - but I had exactly the same problem ~ and I found the solution.

So, open the document you want to have the custom table in ~ or predefined table style.

Then click the little button at the bottom right of the Styles tile on the Home tab - to expand the list of styles in a separate window.  Then click on the "Manage styles button" (bottom right) and then select the Set Defaults tab.  There you will see the Font Color being set to automatic.  with + Body selected as the font; change this to be the color you want for your Normal Font (not the table font you want - just the regular font colour you use; probably black or dark grey...) ~ If your header row hasn't already changed color; then you can change it within the table style.

Also - if you want to use a different font to the default - then you will need to set a custom font in change styles and define your Header & Body fonts.  (You will need to do this before I think)
