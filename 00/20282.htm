tom ogilvy say:

Lee CC,

You could use intentional circular references.  In options, you would choose formulas and under calculation options, you would select the checkbox for Enable Iterative Calculations

then set Maximum iterations to 1

then use a formula like

A1:   =if(A1="X","X",if($C$1="X","X","some_other_value"))

Use similar formulas in A2 and A3.

You will not be notified if you unintentionally create a circular reference error as long as you Enable Iterative Calculations. 

An alternative might be to use an event macro to manage cells A1:A3 but a lot would depend on what is causing C1 to change values.
