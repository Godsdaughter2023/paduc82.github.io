Hi k9lover,

You cannot completely customize the Title name in the tab. By design the title of the Internet Explorer (IE) window consists of the page title and the Internet Explorer window title. There's no way that the tab can changed without changing the Uniform Resource Locator(URL).

However, you can change the Internet Explorer Window Title to add or remove the custom Internet Explorer window title by following the steps outlined here: 

Important This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base:

322756 (http://support.microsoft.com/kb/322756/)How to back up and restore the registry in Windows

a.    Click Start and then Run.

b.    Key in regedit in the Run text box.

c.    Navigate to and select the following key in the Registry Editor:

i.    HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main

d.    Double-click the Window Title icon in the right pane.

e.    If the key doesn’t exist, right-click in the pane, then choose New, String Value, type Window Title, and press Enter .

f.     Type what you want to see on IE’s title bar, or type nothing to show only the site name.

g.    Click OK.

h.    Close Registry Editor and any Internet Explorer’s windows.

When you re-launch Internet Explorer, the change will be in effect and you will have your favorite title.
