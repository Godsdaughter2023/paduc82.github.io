Hi Mike,

Thank you for posting in Microsoft Community.

I understand your concern and frustration you have been through. I will be glad to assist you.

1. Is your admin account a Microsoft Account?

If you no longer have any other accounts on the computer, what you can do instead is to enable the in-built

Administrator of your computer, then create a new user with admin privileges instead.

To do this:

1. Press Shift key and restart
2. Click "Repair your Computer" on the bottom left side of the installation window.
3. You will be taken to the Windows Recovery Environment - select  "Troubleshoot".
4. Then click "Advanced Options".
5. From there, click "Command Prompt".
6. The Command Prompt window will open, type the command:
net user Administrator /active:Yes
7. Press Enter, and you'll be told that the command was successfully initiated. Enter the command "Exit" to restart the computer.

If done correctly, you will see a new user "Administrator". This doesn't have any password, so you can easily access it.

You will be able to create a new user from there.

If you are using Microsoft account as administrator account follow the link below.

How to reset your Microsoft account password
http://windows.microsoft.com/en-us/windows-10/how-to-reset-your-microsoft-account-password
