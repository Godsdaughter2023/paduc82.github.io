A POP3 account in local download only and not a synching account such as IMAP, Exchange and Outlook.com. So there is no sync slider for POP3.

Your POP3 emails are kept in a pst-file. Copy over the pst-file from your old computer to your new computer (be careful not to overwrite any existing pst-files) and then reconnect to it in Outlook 2013 via: File-> Open-> Open Outlook Data File...

ossiemac say:

I believe that you can only do this with VBA code. The code example below is based on a very simple chart as per the picture.

To install the code.

  1. Right click the worksheet tab name
  2. Select View code
  3. Copy the code below and paste into the VBA editor.
  4. Edit the range and chart name where I have the comments which display in green.
  5. Close the VBA editor (Cross with red background top right)
  6. Save workbook as VBA enabled workbook.
  7. Ensure options set to allow macros. See help for how to do this. (Option to disable all with notification should be OK.)

Now each time a value is changed in the data value range then the colors will update on the columns.

If your chart is more complex and you cannot sort out the required changes to the code then you will need to upload an example workbook to OneDrive or similar place so I can edit the code to suit.

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim cht As Chart
    Dim chtSeries As Series
    Dim arrSeries As Variant
   
    'Edit Range("B2:K2") in following line to the range of values for chart
    If Not Intersect(Target, Range("B2:K2")) Is Nothing Then
        'Edit "Sheet1" and "Chart 1" in following line to your worksheet name and chart name
        Set cht = ActiveWorkbook.Sheets("Sheet1").ChartObjects("Chart 1").Chart
       
        For Each chtSeries In cht.SeriesCollection
            arrSeries = chtSeries.Values
            If arrSeries(1) < 5 Then
                chtSeries.Interior.Color = RGB(0, 176, 80)  'Green
            Else
                chtSeries.Interior.Color = RGB(255, 0, 0)   'Red
            End If
        Next chtSeries
    End If
   
End Sub
