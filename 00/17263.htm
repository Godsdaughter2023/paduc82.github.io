网游 say:

Hi,

I whent to https://www8.garmin.com/support/download_details.jsp?id=3607

and clicked on the download button on that page and got the same error message:

"Security Alert.........

Your current security settings do not allow this file to be downloaded."

I did some testing and found your solution.

Remove any garmin.com website addresses from your IE Trusted sites list.

... actually you should have no websites in your IE trusted sites list.... the Trusted sites list has a default security level of Medium, the Internet Zone is Medium High. The trusted zone is actually less secure...

Why?....

All public facing websites are designed to work in the IE Internet zone. The default security zone settings have been designed to provide a level of security that helps keep your computer safe.

Many websites have many domains (sub sites). eg. garmin.com has downloads.garmin.com, my.garmin.com etc... google.com has account.google.com.

When you place only a sub-domain of a website in your IE trusted sites list, only that sub-domain is trusted. The other sub-domains are still treated as being in the IE Internet zone.

IE has security settings that prevent domains in different security zones... the error message you are seeing at garmin.com is because you are trying to download a file from downloads.garmin.com (Internet zone) from a web page at my.garmin.com (trusted site)

Instructions:

Tools>Internet Options>Security tab,

1. click the button "Reset all zones to default". this resets all of IE's security zone settings to their factory defaults.

2. click the Trusted sites icon, then the 'Sites' button and remove any website domains. eg https://my.garmin.com

Generally speaking, you should not put websites into your trusted sites list, you may want to do this if the website does not appear to work properly, or you may think that the Trusted sites is more 'secure'.

3. Save changes to Internet Options.

4. Close and restart IE for the changes to take affect.

5. go to the plugin download page

https://www8.garmin.com/support/download_details.jsp?id=3607#Instruct

and click the download button.

you should be able to successfully download the file with your default IE security zone settings.

The code in my previous code would work but it is for a slightly different use. The code to do what you need is simpler.

NOTE this code has been corrected as per Steve's Comment

Try:

Sub exportSlides()
On Error Resume Next
MkDir (Environ("USERPROFILE") & "\Desktop\slides\")
Dim osld As Slide
For Each osld In ActivePresentation.Slides
osld.Export Environ("USERPROFILE") & "\Desktop\slides\" & "mathsslide" & osld.SlideIndex & ".JPG","JPG"
Next osld
End Sub

Windows Insider Preview on compressed OS (WIMBoot) devices - Microsoft Community

a. user say:

Windows Insider Preview on compressed OS (WIMBoot) devices

What is the status of Windows 10 Insider Preview on compressed OS (WIMBoot) devices?

a. user say:

Hi,

We’re no longer blocking the installation of Windows 10 Insider Preview on all compressed OS systems, but there are still a few limitations we’re working on, as of Build 10074:

Free Space Requirements

Windows 10 Insider Preview should work on compressed OS devices that have at least 10 GB of free space available.  The amount of free space you needs depends on the device, but if you have less than 10 GB of free space, we recommend that you wait until a solution for a low disk space upgrade makes it into a future build.

Realtek Network Adapter Support

Windows 10 Insider Preview is not currently working on devices that support the Realtek RTL8723BS Wireless LAN 802.11n SDIO Network Adapter (which is common in compressed OS devices). 

Systems with the current driver for this device will be blocked from upgrading to Windows 10 Insider Preview.  We recommend that you wait until Realtek has an updated driver.  Once the updated driver is released, installing it prior to upgrade will get you past this upgrade block.  You can uninstall the current driver to get past the block, but if you do this, you will not have network connectivity on Insider Preview (unless you have a networking USB peripheral).
