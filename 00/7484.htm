What is the "Catalog" option in Mail Merge? - Microsoft Community

tony goodwin 12942 say:

What is the "Catalog" option in Mail Merge?

What can the "Catalog" option do in Mail Merge and what functions can the "More" placeholders do.  I have no trouble doing labels from an Excel spreadsheet, but think the Catalog option could also allow me to do a simple report listing all or a selected portion of my records.

a. user say:

The Catalog option reproduces the content of the Mail Merge Main Document for each record in the data source, without inserting a page/section break after each copy of the Main Document. The Catalog option always outputs to a new document.

So if for example you have a Main Document containing
<<fieldx>><tab><<fieldy>><tab><end of document (i.e. a single paragraph mark)>

and your sheet contains

fieldx fieldy
one abc
two
three ghi


the output will be a single paragraph like this:

one abc two three ghi

If your Main Document contains an extra paragraph mark at the end, i.e.

<<fieldx>><tab><<fieldy>><tab><pm>
<end of document (i.e. a single paragraph mark)>

that will be included in the output and you will get something that looks more like

one abc
two
three ghi

Or if you use something more like

{ IF "<<fieldx>>" = "" "" "<pm>
fieldx:<tab><<fieldx>>" }{ IF "<<fieldy>>" = "" "" "<pm>
fieldx:<tab><<fieldy>>" }<cr><cr>

then you would get something more like

fieldx: one
fieldy: abc

fieldx: two

fieldx: three
fieldy: ghi

with suppression of tags: value pairs where the value is blank.

So...

<<
 but think the Catalog option could also allow me to do a simple report listing all or a selected portion of my records
>>

...yes, it can. 

<<
what functions can the "More" placeholders do
>>

Well, the above example shows one use of an "IF" field. Most of the other field types available during a merge are described in various places, e.g.

http://office.microsoft.com/en-gb/word-help/field-codes-in-word-HA102110133.aspx?CTT=1

but some of the field types available in Windows Word are not available on Mac, and some of the "switches" available in fields in Windows Word, such as the \b and \f switches for MERGEFIELD fields, are not available on Mac either. (Also, some field types exist on Mac that do not exist on Windows Word, but broadly speaking they are probably best avoided).

Removing manual line breaks from cells in Excel 2016 - Microsoft Community

giannina2 say:

Removing manual line breaks from cells in Excel 2016

Hello, can anyone tell me how to remove manual line breaks from Excel 2016 cells? Alt 01 doesn't work in the 2016 version.

Been looking for ages online and can't find anything.

Thanks

hansv mvp say:

Press Ctrl+H to activate the Replace dialog.

In the 'Find what' box, press Ctrl+J. This is the "code" for a line break.

Leave the 'Replace with' box empty.

Either use Find Next / Replace, or click Replace All if you want to remove all line breaks.
