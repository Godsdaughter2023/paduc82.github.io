liamgerrior say:

Preventing the Windows Touch Keyboard from popping up on the login screen [Resolved]

I use a laptop that has no touch screen,but after buying a small drawing pad whenever I go to log in to my computer the Touch Keyboard pops up. It's incredibly annoying and I don't like it, but as far as I can tell there's no way to disable it. This happens even when the pad is unplugged.


So far I've tried modifying the registry, services, and group policy to no avail, and making sure that no drivers for touch devices were running in the background. Without fail however, the touch keyboard still pops up when I try to log in to my computer.



Windows 10 Education Version 10.0.17763 Build 17763


 EDIT:

Before posting this question I went to the service menu and disabled the service on startup. The problem was that I left the service running, and upon restarting the system reopened what was closed, restarting the service. 

Thanks to those that answered, I determined that the problem was that the service was not disabled before restarting.

So to solve a similar issue:

Search for "Services" in the Windows menu, or run the Services.msc command

Scroll down to find the "Touch Keyboard and Handwriting Panel Service"

Double click, and using the dropdown menu select "Disabled"

Now for me, the options to stop the process were greyed out, so I opened Task Manager, and scrolled down in the Background Processes list.

Find "Service Host: Touch Keyboard and Handwriting Panel Service" and right click, then end the task. You will receive a warning from Windows but it didn't break anything for me.

After a restart the keyboard no longer showed up on the login screen.
