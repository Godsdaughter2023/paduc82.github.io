suzanne s. barnhill say:

Well, you don't. At least not easily. You'll need to create four (borderless) text boxes, one for each side of the page, with the text rotated as required in each, all anchored to the page header.

For some tips on a slightly different application, see http://WordFAQs.mvps.org/BorderArt.htm.

If you really want a sentence to run continuously around the page, you'll need to create a graphic in an illustration program that has the "text on a path" feature, setting up a rectangular path for the text. When you have created a graphic the size needed for a page border, you can then anchor that to the page header (Behind Text).

Hi,

In Outlook or Outlook.com, you can only change the size of the font in the email body. The settings of the font inside the From, To, CC, and Subject fields are all the same. This is the design of Outlook and Outlook.com. If you're using Outlook.com, you can temporarily change the zoom settings of your web browser so that you can easily see the characters inside the From, To, CC, and Subject fields as a workaround
