Error code 0x85ad60af is introduced when your MSN email isn’t able to synchronize properly with the MSN server. Whenever you download or install any software or program, it usually installs the support packages as well. You must ensure each program installed on your system. Uninstall the program you aren’t familiar with. Make sure that the network connection you have is fast and stable. Slow internet connections are often tending to the error code0x85ad60af. MSN error code 0x85ad60af can be resolved via resetting the synchronization settings on your MSN.

when you send a group text, how do you hide everyone's phone - Microsoft Community

smilesmileforu say:

when you send a group text, how do you hide everyone's phone numbers from everyone in the group seeing the phone numbers?

I have a windows 7 phone. I created groups. when I send a text message to that group EVERY one's phone number is displayed to everyone in that group. How to I send a group text, but hide the phone numbers from everyone?
entegy say:
Turn off Group Text in the settings of the Messages app. When this is on, sending a group text will use MMS (also often called picture messaging) in an attempt to keep all replies in one thread. The downside is that MMS reveals all involved in the conversation. When Group Text is off, sending a group text will use SMS. While replies won't be kept in the same thread, SMS are sent to someone as if they are the only recipient and they cannot see the other people's numbers.
