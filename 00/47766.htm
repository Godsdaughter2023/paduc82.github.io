wondim say:

I had the the same problem and I fixed it by enabling Windows Firewall . To enable Windows Firewall follow the steps below.

1.       Click Start, type “services.msc” (without quotes) and press Enter.

2.      Double-click “Windows Firewall. ”

3.       Set the Startup type of“Windows Firewall”to Automatic.

4.      Click Start to start the service and click OK. Exit services.

Now internet connection sharing should work without any problem!
