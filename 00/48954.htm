Hi Ravikanth

Start your PC, just as Windows attempts to load (spinning dots), press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down

Do this twice

On the third start Windows will boot into the Recovery Environment and from there you can access System Repair, Safe Mode, Command Prompt . . . etc.

andre da costa say:

No, Windows XP Mode is not in Windows 8 and Hyper-V works differently from Windows Virtual PC contrary to what ZigZag says. Hyper-V utilizes the Microsoft Management Console (MMC) and requires that you processor supports Secondary Level Address Translation (SLAT), you need to enable some features in the BIOS which means, the setup process is a little bit involved.

To find out if its supported on your computer:

Press Windows Key + X

Click Programs and Features

Click Turn Windows features on or off

Check off Hyper-V

If its greyed out, that means it is not supported.

You can use alternatives such as Virtual Box or VMware Player.
