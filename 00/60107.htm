  1. Select the Start button, enter (type) Device Manager, and select Device Manager from the list of search results.

  4. Select Driver Details and look for a file name that includes "stream.sys". If you find a file with this name, your camera was designed prior to Windows 7 and it won't work with most modern camera apps, such as the Windows Camera app. You'll need to purchase a new webcam to use with modern apps.

  5. If you don't have a "stream.sys" file, you can try rolling back your driver. On the Driver tab, select Roll Back Driver, select Yes, and when the rollback is finished, restart your PC and try using your webcam. Note that not all drivers have a Roll Back Driver option.

When it prompts to choose an option How do you want to search driver, then go with Search automatically for updated driver software. If you have a driver available on your PC, then select Browse my computer for driver software, then select the appropriate location, where the driver located on your PC for updating it. (Ensure that PC is connected to Internet while performing this step)

  7. If rolling back is unavailable or updating didn’t work, try uninstalling the driver. On the Driver tab, select Uninstall, select Delete the driver software for this device, and then select OK.

  8. After the driver is uninstalled, in Device Manager, then reboot your PC. Try using your webcam.

  9. If uninstalling the driver and using the generic driver didn’t work, then go to the camera manufacturer's website to look for an updated driver to install.

  10. If the camera still doesn’t work, use the Windows Feedback tool to report the problem or post to the

If its been more than 10 days since you upgraded, rollback would not be available. You would have to manually downgrade. Here is how you can do that:

Step 1: download Windows 10 1703 from the following link:
https://tb.rg-adguard.net/index.php?sid=74a7cf5...

Choose the following:

- Windows Final
- Windows 10, version 1703
- Windows 10 Pro + Home
- English
- x64 (64 bit) x32 (32 bit)

Step 2: create a bootable copy
https://answers.microsoft.com/en-us/windows/wik...

Step 3: perform a custom install
http://answers.microsoft.com/en-us/insider/wiki...

You should backup your files before downgrading.
Applications will need to be reinstalled.
The custom install will archive your files, so you can recover them from the Windows.old folder - make sure you follow the instructions to the letter.

Software and Hardware that once worked in previous versions, just suddenly stops working in Windows 10 1709 after an upgrade and sometimes loses functionality or stability.
