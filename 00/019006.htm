michael bauer (vboffice.net) say:

You can add a custom field of the type formula. Then try a formula like this:

IIf([From] Like '*vboffice*' Or [From] Like '*bauer*', [To], [From])

where you need to add your name(s), and/or address(es), of course.

The formular means, if your own name or address is shown in the From field, the new column will display the To property else the From property.

One drawback is, you cannot sort by formula fields. If that's an issue for you, you'd need to write some VBA code to store either the sender or the recipient in a custom field.
