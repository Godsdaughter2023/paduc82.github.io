graham mayor say:
The spacing between paragraphs in Word is (or should be) created by paragraph formatting and in particular the space before/space after paragraph formatting parameter. It doesn't matter how big you make this space, it is not supported in plain text format.

You need to create the space in a format that a plain text editor can understand. You can do this with a macro to copy the text and reformat it with paragraph spacing e.g. by using the following macro to copy the text. http://www.gmayor.com/installing_macro.htm  


Sub CopyAsText()
Dim strText As String
Dim dFname As DataObject
Dim fFname As String
    Set dFname = New DataObject
    strText = Selection.Text
    strText = Replace(strText, Chr(13), vbCrLf & vbCrLf)
    fFname = strText
    dFname.SetText fFname
    dFname.PutInClipboard
End Sub
