How do I omit content in brackets from spellcheck? - Microsoft Community

a. user say:

How do I omit content in brackets from spellcheck?

I have compiled an advanced English vocabulary with over 6,000 entries, each with its own pronunciation. Because of the pronunciation symbols, the spellchecker "thinks" they are misspelled words, and my spellchecker can't cope with so many "misspelled" words. My question is: Is there a way to tell spellchecker not to check any "words" contained in brackets. I would think that Word would have a way to ignore certain parameters, but I haven't found it. Thanks.
jay freedman say:

Word can't do quite what you suggest, but there is a way.

The first step is to create a character style whose only "formatting" is the "Do not check spelling or grammar" option. To do this:

  • Select any piece of text in your vocabulary document.
  • Click Review > Language > Set Proofing Language.
  • Check the box for "Do not check spelling or grammar" and click OK.
  • While that text is still selected, click the bottom down arrow on the Styles gallery, and click "Save Selection as a New Quick Style".
  • In the dialog box, change the name (I suggest NoProofing) and click the Modify button. [Important: you must modify the style at this time, because you cannot change it from the default Paragraph style to a Character style later.]
  • In the next dialog, change the Style Type dropdown to Character. At the bottom of the dialog, select the option "New documents based on this template" and click OK.

The second and final step is to use the Replace feature to apply the NoProofing style to all the locations contained in brackets.

  • Open the Replace dialog, and click the More button to expand it.
  • Check the box for "Use wildcards".
  • Paste the following expression into the Find What box:  (\[[!\]]@\])
  • Paste the following expression into the Replace With box:   \1
  • While the cursor is still in the Replace With box, click the Format button at the bottom left, click Style in the menu, select NoProofing from the list of styles, and click OK.
  • Click the Replace All button, click OK in the completion prompt, and close the dialog.

Some notes on the Find What expression:

  • I assumed that by "brackets" you mean square brackets [ ].
    • If you meant curly brackets { } then the expression should be (\{[!\}]@\})
    • If you meant parentheses ( ) then the expression should be  (\([!\)]@\))
  • The meaning of the expression is "match any sequence of characters that starts with a left bracket, ends with a right bracket, and does not contain a right bracket in the middle". For a tutorial on wildcard matching expressions, see http://www.gmayor.com/replace_using_wildcards.htm.
  • The parentheses at the start and end of the expression are not for matching, but tell Word that the matched text will be represented by the \1 in the Replace With expression. That way, the replacement will be the same as the found text (i.e., no change of characters), but the NoProofing style will be added to it.

This array formula (enter it using Ctrl-Shift-Enter) will return the first non-blank  (non "" ) cell in the range

=INDEX(N72:N137,MATCH(1,IF(N72:N137<>0,IF(N72:N137<>"",1)),0))
