4. click test query to confirm the results.

note: it may take about 24 hours for the newly created documents to show in the search results.

anneliese [msft] say:

Let me see if I can clarify. 

You can (or should be able to, at least) report bugs in all apps except Outlook using File > Feedback. You should see three buttons: Send a Smile, Send a Frown, and Send a Suggestion. The first two will open the form that allows you to describe the issue and send (optionally) a screen capture & email address. The last button will open UserVoice; in some countries/regions, it will default back to the Send-A-Smile dialog.

In Outlook (only), things work differently. You can use File > Support > Contact Support to report issues and File > Feedback to suggest features. Alternatively, just post bug reports here in the community. Get in touch with us however you like; all channels feed into the same tool on our backend, where we have dedicated teams reviewing & escalating.

Now that I've said all that, I must confess that File > Support is not yet available to 100% of Outlook Insiders in the Slow ring, but it will be soon. All Fast users should have it. While we wait for it to roll out, feel free to report bugs here. 

- A
