marshall barton say:

First, create a query (named TotalAmt:

SELECT Year(QtnTab.[Date]) AS YrDate, QtnTab.To, Sum(QtnTab.LabourAmt) AS SumOfLabourAmt
FROM QtnTab
GROUP BY Year(QtnTab.[Date]), QtnTab.To

Then create another query to generate the results:

SELECT Q.*
FROM TotalAmt As Q
WHERE Q.To In (SELECT TOP 10 X.TO
                        FROM TotalAmt As X
                        WHERE X.YrDate = Q.YrDate
                        ORDER BY X.SumOfLabourAmt DESC)
ORDER BY Q.YrDate, Q.SumOfLabourAmt DESC

How to (try to) fix it:

  1. Click on the Settings gear in the left menu of Groove Music (next to your name if the menu is expanded).
  2. Click on the link "Choose where we look for music" under "Music on this PC".
  3. Look for any entries that are network folders (starting with "\\").
  4. Remove those entries.
  5. If this still doesn't work, try removing any save locations that are on removable or cloud drives like SD cards or OneDrive folders as well.
