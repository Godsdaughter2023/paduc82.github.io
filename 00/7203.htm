Whilokii is an ad-supported browser add-on or extension that installs toolbars and displays pop-up advertisements. Whilokil generally comes bundled with other free software (often without the knowledge or consent of the user) and can be the source of various issues and problems to inlcude browser hijacking. If you use the Draw Text Box command at the bottom of the Insert > Text Box gallery, the Text Box Tools tab appears on the ribbon, and the Text Direction button is at the left end.

If you use the Insert > Shape button and insert some shape other than a text box, right-click the shape and choose Add Text; then the same Text Box Tools tab will appear.

In either case, clicking the Text Direction once will make the text run top-to-bottom, clicking it a second time makes it run bottom-to-top, and a third click returns it to left-to-right. (There is no provision at all for upside-down right-to-left text.)
