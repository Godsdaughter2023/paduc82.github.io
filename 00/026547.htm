daniel g. grau say:
In the Find slot, I keyed in a particular word that I had capitalized by mistake. In the Replace slot, I keyed in the same word without the capital letter. I hit Replace All and nothing happened.

You would have obtained the same result on a PC. This is because Word makes upper case prevail in the replacement:
  • If you are looking for "something/Something/SOMETHING," word will find "something/Something/SOMETHING" (i.e. any instance of the word, whatever you type) 
  • If you are replacing by "anything," you will respectively get "anything/Anything/ANYTHING/"
  • If you replace by "Anything," you will get "Anything/Anything/ANYTHING"
  • If you replace by "ANYTHING" you will get this word in all three instances
To override the above criteria, you need to disable "Ignore case" in the search tab (which is called up with Command F for the Find field or Command Shift H for the Replace field) or enable "Match Case" in the Advanced version. If you do that, Word will only find the string exactly as written, and will also replace it exactly as written.

You can assign a keyboard shortcut to the Advanced version. To do this:
  • Select Tools > Customize Keyboard
  • Under Categories, click on Edit
  • Under Commands, select EditFindDialog
  • Place the cursor in the "Press new keyboard shortcut" field
  • Press a key combination. For instance, Control F.
  • OK your way out.
