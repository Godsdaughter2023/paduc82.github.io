Problem solved. 

It turns out that Windows updates are NOT installed during shutdown if the "Fast Startup" power button option is turned on.  When turned off, updates are installed during shutdown just like Windows 7.  Too bad this information seems to be very difficult to find.

na-no. say:

1st method) Open the Start Menu and search for "Default software" or "Default programs". Open it, find "Default music player" and change that setting to "Windows Media Player". Apply the settings and exit. (Note: this will affect all the music file types (MP3, MP4, M4A, AAC and so on).)

2nd method) Search any music file, e.g. an MP3 one. Select it by left-clicking once, then right-click it and go to Properties. Go to where it says "Open with:". It will say "iTunes". Click on "Change" and change it to "Windows Media Player". Then click on "Apply" and exit. (Note: This will affect only the specific file type you are editing, e.g. MP3 files only.)
