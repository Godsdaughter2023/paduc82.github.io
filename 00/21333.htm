ronrosenfeld say:

For a problem this complex, I would prefer a User Defined Function (UDF).  It is easier to construct and maintain.

To enter this User Defined Function (UDF), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this User Defined Function (UDF), enter a formula like 

=MatchHeader(row_number_to_search_for_min)

in some cell.

========================================

Function MatchHeader(RW As Long)
    'Could make this an argument instead of hard coding, if you want
    Const X = "Z1:BY1,EB1:GA1,ID1:KC1,MF1:OE1,QH1:SG1,UJ1:WI1,YL1:AAK1,ACN1:AEM1"
   
    Dim V As Variant
    Dim I As Long
    Dim rHeaders As Range, rValues As Range
    Dim C As Range
    Dim dLowest As Double
   
Set rHeaders = Range(X)
Set rValues = rHeaders.Offset(RW - rHeaders.Row)

For Each C In rValues
    If C = dLowest Then
        MatchHeader = C.Offset(rHeaders.Row - RW)
        Exit Function
    End If
Next C

========================================

Hi Akhtarshaikh_fs,

 

Since you only need to migrate 5 mailboxes and don’t want to use third-party tools, we suggest you use the Outlook client to export the mailbox items from the old mailboxes and then import them to the new mailboxes. Please note you need to do this for all the 5 mailboxes. Here are the steps:

1. Configure the old email account on the Outlook client.

2. Export all data to a PST file.

3. Create a new Outlook profile and then configure the new Office 365 account on it via Auto Account Setup.

4. Import the PST file in Step 2 to the new email account.
