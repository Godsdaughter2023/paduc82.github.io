On the Create tab of the ribbon, click Query Design.

Add the table, then close the Show Table dialog.

Click to highlight the Totals button.

Add the category field to the query grid, and leave the Total option for this column set to the default Group By.

In the next column, enter this expression:

WeightedAverage: Sum([Weight]*[Amount])/Sum([Weight])

and set the Total option for this column to Expression.

Jim,

If I may interject. Are you sure you are talking about the Project Summary Task and not just a normal summary task? Note, the Project Summary Task is ID 0. The reason I ask is because you state you can assign any non-budget resource but not a budget resource. Project doesn't allow any resource type OTHER than a budget resource to be assigned to the Project Summary Task. And if you try to assign a budget resource to a normal summary task, it will be greyed out (i.e. you can't do it).

As Trevor noted, once it's all set up the budget cost is assigned through the Task or Resource Usage views. You can make the assignment via the Assign Resource dialog but you can't enter the cost value in that dialogue, the cost value must be entered into the Budget Cost field on either the Task or Resource Usage view.

Hope this helps.
