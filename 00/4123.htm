The possible reasons why e-mails suddenly disappear.
1. Messages were manually deleted either due to accidental deletion or were moved or deleted due to your account filter settings.

2. Messages were deleted due to account compromise issues where a hacker might have gained access to your account and deleted the emails.

3. You are using an e-mail client or a third party spam filtering software to access your account.

4. The account was marked as “Inactive." Accounts become inactive if you don’t sign in via web browser at least once a year.

5. If the feature Conversation Threading feature is turned on.

6. If you use Sweep to schedule a cleanup.

In the case if 1809, it means public release, year 2018, month 09. With preview builds of Windows 10, Microsoft did use code names with a version number such as Redstone 4, Redstone 5, etc. That has now changed to the new format of year plus which half of it they expect to have a release version ready. In the case of 19H1, it means that Microsoft will have a public release version ready for some time within the first half of 2019. In regards to build numbers themselves, that is simply a reference point to let Microsoft and Insiders know what the current build is for testing purposes. When a build has been deemed stable enough to be released to the general public, that will be the one used if it fits within Microsoft's timeframe of having a build ready to release for the general public. A "footer section is outside of a printable area" - Microsoft Community

userfriendly5 say:

A "footer section is outside of a printable area"

A "footer section is outside of printable area" resulting in my page numbers cut off from my newsletter document, how can I fix this? No matter how far away I stay from the borders, it always does this. I am using Word for Mac 2008.
john mcghie say:
Go into Format>Document>Margins.  The settings you need are in there.

First, make sure you have specified the correct paper size (make sure it matches the paper size in the printer!)

Then set your Margins large enough to contain your headers and footers.

Then specify the "From Edge" distance, which specifies where the header or footer sits within the margin.

Be aware that each section of the document has its own margins and headers and footers, so if there is more than one, you will need to make these settings in each section.
