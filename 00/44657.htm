ronnie vernon say:

Hi rdrosche

Check to make sure that System Restore is not turned on for that recovery drive. This is the most common reason for this situation.

  • Go to Start and click Control Panel.
  • On the left side Tasks Menu, select Classic View.
  • Double click System to open.
  • On the Tasks menu, select System Protection.
  • Make sure the System Protection Tab is selected.
  • In the Automatic Restore Points section, remove the check mark from the Recovery Drive letter.
  • Click Apply.
  • Click Turn System Protection Off in the warning dialog.
  • Click OK.
  • Reboot the computer.

This should delete all of the restore points on the Recovery drive, free up a lot of space, and prevent this from happening again.

The Recovery Drive is critically important. In case of a disaster, the files on that drive are the only way you have to reinstall Vista.

Dell has a procedure that allows you to create a Recovery DVD from the files on the Recovery Drive. After you perform this procedure, this will allow you to delete the files on that drive and then use that space for your own files.

Consult the documentation that came with the computer or call Dell Technical Support for this procedure.
