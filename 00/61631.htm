tryx3 say:

Robert,

Restore points - I found them unreliable so I stopped using them. 

  • I rely on system images instead [I partitioned my laptop's single HDD into 2 drives with the OS & apps on one and all my own files on another - this makes system images a breeze as they take less than half an hour to make and the same time to restore].
  • Restore points disappear over time to make room for new ones to be made.
  • Nothing can create a restore point retrospectively - that Restore point you saw after installing the new update agent was probably made at the start of its installation so using that restore point could have set your computer back to the state it was in before you installed the new update agent. 
  • As the restore failed, I don't know what state the computer is now in so I think you need to install the new update agent yet again.
  • I think that Restore points are supposed to do nothing if they cannot complete successfully but I have no idea if that is what happens in practice. 

Installation of new update agent -

I think you need to install it yet again to be certain that all its components remain installed correctly.

The disappearance of the red shield does look like progress [but I am still concerned about what the attempted restore point might or might not have done].

Factory restoration -

Where you see references to restore-to-factory that has nothing to do with Restore points or with WU - "2010 factory state option is still missing"

  • All computer manufacturers who supply computers with preinstalled Windows have to provide a means of reinstalling Windows.  Some achieve this by supplying a reinstallation DVD whilst others do it by creating a restore-to-factory partition on the HDD that is used to reinstall the entire computer from scratch - it reinstalls Windows as it was when you bought the computer [but they can omit some of their own utilities from the restore-to-factory image so this can be disappointing].  
  • A restore-to-factory partition is, in effect, a built-in installation disk.  So it would set you back to 2010 and you would have to reinstall all your applications.  I think you would be worse off than you are now as far as updates go [you would have to reach SP1 state before you could even think about the rollup update].
  • The restore-to-factory partition is almost always on a non-Windows partition [a part of the disk that Windows knows nothing about] so chkdsk would not reveal any useful information about it.
  • If, but I do not recommend it, you wanted to restore-to-factory then you would need to follow the procedure specified in the computer manual & on the computer maker's website.

Corrupt applications -

  • You are likely to have to reinstall these given what you have already written.
  • Since this seems inevitable, installing Windows 10 becomes a practicable alternative - if you could solve the current problems without reinstalling applications then installing Windows 10 would seem like too much extra work.

The Mute light & hardware drivers -

This could be a problem with a hardware driver or a computer maker's utility. 

  • Control of such non-standard things as Mute lights tends to be through drivers installed by the computer maker so reinstalling those might solve this particular problem.
  • I would expect you to find the correct driver on the computer maker's website but, being a non-standard item, I am only guessing when I suggest that it might be the keyboard driver. 
  • On Dell's such a light would probably be a function of Dell's QuickSet utility rather than a keyboard driver so that might also be the case with other computers.
  • If you installed Windows 10 you would then have to do a general review and search for hardware drivers even though most of them will be detected by & installed by either the installation process itself or the initial WU that you run afterwards. 

Best of luck,

Denis
