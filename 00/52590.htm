When a document is copied to the same computer, a different computer on the network, or to a computer on a different network it does not change - it retains whatever information was in the document.  Hyperlinks do not change just because they are in a new location.  The only exception may be the addition of some type of macro to the document so that the hyperlink is actually a macro which alters it based on information about the cjrrent location of the document.  The same applies to a web page, except there it generally requires scripting or more complex programming to accomplish the task.  I don't know how you did this in the past, but I know of no way to make either situation automatically change the hyperlink without programming of some sort.
