scottgem say:
Create an autoexec macro to run the query. This will open up the Access database and automatically run the query. 

williambielfeldt say:

I have never done this. Could you please indicate (very simply) how I would do this.

john w. vinson say:

I'd suggest creating a Shortcut on your desktop to launch Access. Drill down in the Program Files folder and find msaccess.exe; right click it and choose Send To... Desktop (Create Shortcut).

Then right click the shortcut and view its Properties. Set the Target line to something like

"C:\Program Files (x86)\Microsoft Office\Office12\MSACCESS.EXE" "C:\Users\pathtoyourdata\mydata.mdb" /x "RunMyQuery"

This shortcut will open your database and run the Macro named "RunMyQuery". This macro could have just one step, executing the query, or you could follow that step with a Quit line to close Access.

Adjust to your version/filenames/paths and so on of course!
