sba923 say:

Sometimes it's so good to answer your own questions ;-)

I'm hereby sharing what saved my day after struggling with this issue for several days.

http://winaero.com/blog/reset-windows-store-cache-in-windows-10-to-fix-issues-with-universal-apps/ describes how to manually delete the Windows Store cache so that it gets rebuilt from scratch.

Download Windows Updates at a fixed time - Microsoft Community

spigolo say:

Download Windows Updates at a fixed time

Technical Level : Intermediate

Summary
How to download and install Windows Updates at a desired time in Windows 10, using system functions.
Details

If your provider gives you a connection which is free only at a certain hours, probably in the night, you might want to download Windows updates at such hours.

In addition, a gamer might wish to avoid intensive use of bandwidth during a game session.

Standard Windows functions allow to select a metered connection, thus avoiding updates if not explicitly requested:

Settings

Network & Internet

Wi-Fi

Click on your connection

Set Metered connection to On

You can also delay restart after applying updates changing Active Hours in Settings-Update & security, but it is not yet possible to set a specific time to search and download updates.

To reach this result we can use Powershell and Task Scheduler.

First download this Powershell module:
https://gallery.technet.microsoft.com/scriptcenter/2d191bcd-3308-4edd-9de2-88dff796b0bc#content

(courtesy of Michal Gajda MVP)

Unzip and extract PSWindowsUpdate folder in C:\Windows\System32\WindowsPowerShell\v1.0\Modules then start Powershell as administrator and type these two commands:

Set-ExecutionPolicy Unrestricted


Import-Module PSWindowsUpdate

Open Notepad and type the following command:

powershell Get-WUInstall –AcceptAll

Save the file in a folder of your choice and name it “Wuinstall.bat” (double quotes are necessary to avoid adding the extension .txt to the name)

Search with Cortana and execute:
Task Scheduler

On the right panel chose Create Task.

Add an activity as per following images:

At this point, even if your connection is set as metered (in Creators Update you have this setting available for Ethernet connections too), at the fixed time it will start Windows Update search and deploy, even if your pc is off.

By the way, Gajda’s collection allows you other interesting activities.

For example:

Get-WUList: obtains a simple list of waiting updates, showing also the dimension of the download (info which doesn’t yet show in Settings)

Hide-WUUpdate: hides an update you don’t wish to install.

Further information can be found typing in Powershell:

get-help functionname
