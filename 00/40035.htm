Hi Patricia,

 

Do you get this error only when you uninstall Adobe reader?

 

This error can occur if the Windows Installer Engine is corrupted, installed incorrectly, or disabled. This may cause installation issues.

 

Method 1:

Unregister and then reregister Windows Installer service.

To do this, follow these steps:

a. Click “Start”, type “cmd” in the Start search box.

b. Right-click “cmd” and select “Run as administrator”.

c. Type the following commands and hit Enter after each command:

msiexec /unreg

msiexec /regserver

 

Method 2:

Check for the status of Windows Installer service and make sure that it’s set to Automatic.

a. Click “Start”.

b. Type “services.msc” in the Start search box and hit “Enter”.

c. Double-click the “Windows Installer” service.

d. Set the “Startup type” of Windows Installer to “Manual”.

e. Click “Start” to start the service. Note down the error message if any.

f. Click “OK”.

Try uninstalling the program now and see if the error persists.

 

You may also try the steps provided to fix the "Windows Installer issue" from the following article and check.

Troubleshoot problems installing and uninstalling programs on Windows-based computers

willardfarther say:

Have you tried using System Restore?  If you've got a restore point created before you installed the problem program, System Restore can, usually, dial back you system to the way it was before you installed the program that you want to remove.  System Restore won't delete any of your user files, but, if you've installed other software since installing the problem program, that software will disappear, too.  You can always re-install any programs that you want.  System Restore will allow you to choose from a number of restore points, provided that you've got System Protection running and you do have restore points.

You can also restore from a System Image, if you created one before installing the troublesome program.  Windows will replace your current system with the system image, which will remove all changes made since you created the system image, so back up your user files before doing this.

Read the on-screen instructions very carefully, and I hope you succeed in getting rid of whichever program it is that you want to uninstall.
