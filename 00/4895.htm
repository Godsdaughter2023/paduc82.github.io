How to prevent additions of new records to a table - Microsoft Community

dmkg say:

How to prevent additions of new records to a table

Is it possible to create a table which will hold just a single record and prevent users from adding any more records?  I would like to do this so the record would hold information about a company which would in the main remain static and would be used in various queries and reports.  I would like the users would be able to see and amend the information via a form but not be able to create a new record.  Can this be done?
dirk goldgar say:

> prevent users from adding any more records?  I would like to do this so
> the record would hold information about a company which would in the main
> remain static and would be used in various queries and reports.  I would
> like the users would be able to see and amend the information via a form
> but not be able to create a new record.  Can this be done?
 
In addition to setting properties of the form that edits the table, you can
also fix the table to that, at the database engine level, it won't accept
more than one record.  You can do that by giving the record a primary key
field of type Number/Byte (or Integer or Long Integer), with a default value
of 0, and a validation rule that says the field must equal 0.  Thus, no
other records can be accepted, since there can only be one record with a
primary key of 0.

Lumia Icon upgrade to 10080 - Helpful Steps - Microsoft Community

timstone say:

Lumia Icon upgrade to 10080 - Helpful Steps

Like so many Icon users, I was anxious to obtain the Win 10 Preview.  When the 10080 release arrived, I was excited, but also aware that the phone had presented challenges to MSFT in setting up for an upgrade install, so I expected problems.  I was not off track.  The problems at first seemed overwhelming.  However, I was determined, and thus I pushed through, got the best I could ( which was not impressive ) and then stepped back for thoughtful consideration.

I realized that my Icon was fully setup with the apps I liked ( not too many ), accounts, and other stuff that perhaps the update didn't like or know how to handle.  Also, in trying to move forward by using a basic reset, the phone reverted from Denim to Cyan.  Essentially, the install was not very clean, and thus problems were very likely.  In all of my years with computers, how often have I seen this ?  Way too often,  and the solution that always works is a clean slate ( ie. reinstalling Windows and then reinstalling programs ).  With all the garbage cleaned out, the result is much better.

So, I reverted to 8.1 using Microsoft's very easy to use utility.  I made sure the Icon was on 8.1 Denim.  Then, a reboot, install only the Windows Insider program, and I redid the install of Win 10 Preview.  We know there are some hiccups along the way, and especially the "stuck on reboot issue" that is well documented, with a solution, on this forum.   I went through the steps, and got everything installed.  This time I could tell I had a much cleaner display, and I proceeded through the setup.  In some cases, programs needed to be uninstalled and then reinstalled before they would work.  Even those  that appeared by default ( like the Bing Apps ) needed to be removed.  Some ( like News ) never did install / work properly ( and I do like that app ).  The Start Screen, with more tiles turned on, is still showing a partial background.  It's using the sizing for the less tiles setting, and thus the image leaves a black bar down the right third of the screen.  However, at least now the People Hub  is working as it should ( though not with what we all want to see ).  Other apps, once removed and reinstalled from the store, also seem to be working.  In addition, the screen now has its clean high resolution working, and touch is much better.  Yes, there are issues ( some scaling needs to be improved, the Start screen has some issues when resizing and moving tiles, some apps are not ready, and the Store needs a lot of work ), but overall this release is now working a whole lot better.

So, what is my point here ?  If you have an Icon ( or 930 ), and you are not happy with the Win 10 Preview install, might I suggest you do what I did, use the utility to revert the phone back to "Factory New 8.1.1 Denim", and then do the install without setting any of your accounts.  Once you have it fully established as a Win 10 Icon, remove the default apps ( ie. Facebook, Bing apps, etc ) and then reinstall them.   Setup  your accounts, and then add your additional apps.

Also, keep the phone plugged into a power source while doing the update, or use the power pad.  It will help.

For specific info on the setup steps, rebooting, and the stuck on restarting issue, please look at the other thread that discusses it in depth.
