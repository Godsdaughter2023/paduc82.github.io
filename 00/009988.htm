Sub RowSplitter()
k = 1
For i = 1 To 1024
    Range(Cells(1, k), Cells(1, k + 15)).Copy Cells(i + 1, 1)
    k = k + 16
Next
End Sub

Bookmarks and cross references

nathan li msft say:

Hi MCamz,

Please refer to the following steps based on my test:

1. Enter the text SCHEDULE 1;

2. Select the text > click the Insert tab > click Links > Bookmark;

3. Name the bookmark as SCHEDULE1 > click Add;

4. In the spot that you want to insert the cross-reference, click the Insert tab > Links > Cross-Reference;

5. In Reference type, select Bookmark > in Insert reference to, select Bookmark text > select the bookmark that you are going to insert > click Insert.

6. Press Shift+F9 to show the field codes: { REF SCHEDULE1 }

7. Modify the field codes to: { REF SCHEDULE1 \* Lower \* FirstCap } > right-click > Update Field.

8. Press Shift+F9 again, now the cross-reference text is "Schedule 1".

(This is because \* FirstCap can only change an all lowercase text, it cannot change an all capital text. So you need to use \* Lower to change all characters to lowercase before using \* FirstCap. )

Please let us know if you have any further questions.

If your Windows 10 hangs on boot, but not reboot, fix by disabling - Microsoft Community

a. user say:

If your Windows 10 hangs on boot, but not reboot, fix by disabling fast startup.

Fast startup is a hybrid-hibernate mode that changes the shutdown behavior of your computer.  If you have this option selected, your computer does not perform a true shutdown.  Instead, it closes all your applications then hibernates, so that when you start the computer it will pretend to boot fast.  This can cause problems when trying to boot the computer.

You can disable fast startup in control panel/power options/choose what the power button does/change settings that are currently unavailable/uncheck at bottom of the screen.

By disabling this option, I have been able to boot with no problems, but with that option selected (default) my computer hangs and will not complete booting - doesn't even show the spinning dots - just a static blue square.

todd dunn say: Yes, PowerPoint can handle that. You need to use more than a single head for that resolution. Something like a Matrox triple head or a Datapath X4 for example. Most pro companies I work with have either or both as a standard workbox gadget. The E2 will take it from there. troy at tlc say:

Braden,

As Todd said, yes, PowerPoint can definitely be used to develop the content. There are some unique setup and hardware needs to overcome.

1. PowerPoint page size (my recommendation) would be 52"x17.333" (52" is maximum sizing available).

2. Computer will need multiple display out. My preference is to use true graphics card output vs. an interpolated output using the Matrox boxes (connect 1 video from computer and it splits into 2-3-4 outputs, but not powered by actual GFX card processing). For this resolution, which is large even based on my shows (this being said as I am prep'ing a 7040x2560 px show) and will either be a 4 HD head out (1920x4 = 7680) or 2 UHD out (4096x2 = 8192). Computers with this GFX power will be a special build, going to an E2 is perfect.

3. PowerPoint only seeing 1 output. So either need to use NVIDIA PhysX to join outputs so PowerPoint seeing a single monitor (my preference is to use the GFX card to accomplish). Or use the PowerShow software to create a software single monitor (I have a license to the software to use anytime provided hardware cannot accomplish through graphics process).

There are several other factors, but I would work with the production company (supplying the E2) to determine project specifications presentation design and hardware will need to work within. Hope this info is helpful - good luck!

Troy @ TLC
