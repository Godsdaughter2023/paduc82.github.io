I don't know what a clean boot means except in a shoe store, much less your last suggestion. :)

Very good - I love it!

A "clean boot" is shorthand for a "clean boot-up process", that is a boot-up process without the usual stuff that you load at start time. You achieve it like so:
  1. Run msconfig.exe.
  2. Untick every mark under the Startup tab.
  3. Tick the box "Hide Microsoft Services" under the Services tab.
  4. Untick every remaining mark under the Services tab.
  5. Reboot, then try to create a Restore Point.
You could also test a non-GUI method to create a Restore Point. Here is how it works:
  1. Copy & save the code below on your desktop as Restore.vbs. Make sure the extension is .vbs. Windows tends to make it .txt, which will never work.
  2. Double-click this file on your desktop and follow the prompts.
  3. See if a Restore Point was created.
'----------------------------------------------------
'Create a manual System Restore Point
'6 Jan 2012 FNL
'----------------------------------------------------
If WScript.Arguments.Count = 0 Then
  Set oShell = CreateObject("Shell.Application")
  oShell.ShellExecute "wscript.exe", """" & WScript.ScriptFullName & """ Run", , "runas", 1
Else
  sDescription = InputBox("Name of Restore Point", "System Restore", "Manual Restore Point")
  if ltrim(sDescription) = "" then WScript.Quit
  Set oWshShell = WScript.CreateObject("WScript.Shell")
  Set oFSO = CreateObject("Scripting.FileSystemObject")

  Popup True
  sWinmgmts = "winmgmts:\\.\root\default:Systemrestore"
  GetObject(sWinmgmts).CreateRestorePoint sDescription, 0, 100
  Popup False
End If

'-----------------------------
'Create or kill a pop-up panel
'-----------------------------
Sub Popup(bCreate)
  sTempScript = oWshShell.ExpandEnvironmentStrings("%temp%\SRTemp.vbs")
  If bCreate Then
    Set oTempscript = oFSO.CreateTextFile(sTempScript, True)
    oTempscript.WriteLine "MsgBox ""Creating a System Restore Point."" & vbLF & " _
        & """Please wait until the action is complete."", 64, ""System Restore"""
    oTempscript.Close
    oWshShell.Run "wscript.exe """ & sTempScript & """", 0, False
  Else
    Set cProcessList = GetObject("winmgmts:\\.\root\cimv2").ExecQuery _
      ("Select * from Win32_Process Where Name = 'wscript.exe'")
    If cProcessList.Count > 0 Then    'Kill the pop-up dialog box
      For Each oProcess In cProcessList
         If InStr(1, oProcess.CommandLine, sTempScript, 1) > 0 Then oProcess.Terminate
      Next
        oFSO.DeleteFile sTempScript
    End If
    MsgBox "The System Restore Point """ & sDescription & """ was created.", 0, "System Restore"
  End If
End Sub

    Hi Alex,

     

    I see that this is your first post in this forum, welcome to the Microsoft Community.

     

    What happens when you try to format the Windows partition in the computer? Do you get an error?

     

    You won’t be able to delete any partition that is in use, including the Windows partition. If the option to delete the partition in Disk management is greyed out, then probably you have booted from the Windows partition that is on the computer and not the one in the SSD. To resolve this, you need to change the boot order in the BIOS. Refer to these steps:

     

    1. When your computer starts, wait for the manufacturer logo to check the option for boot menu, it will usually be F12.
    2. The boot menu key might differ with each manufacturer. If you are unable to find the boot menu key, you may refer the user manual.
    3. Restart the computer, when the manufacturer's logo is displayed, keep pressing the Boot menu option key to enter the boot menu and change the boot option to SSD.
    4. Once you have booted to Windows from the SSD Windows partition, now press Windows key +X and select Disk management.
    5. Right click on the Windows partition on in the computer and select ‘Delete drive’.

    a. user say:

    Hi Alex,

     

    If you are unable to boot from the SSD Windows partition, it is possible that some boot files are missing or corrupt. I would suggest you to boot using a Windows 8.1 installation disc and then enter the Windows recovery environment to access the command prompt and then try to rebuild the Windows boot files. Refer to these steps:

     

    a)      When your computer starts, wait for the manufacturer logo to check the option for boot menu, it will usually be F12.

    b)      The boot menu key might differ with each manufacturer. If you are unable to find the boot menu key, you may refer the user manual.

    c)       Restart the computer, when the manufacturer's logo is displayed, keep pressing the Boot menu option key to enter the boot menu and change the boot option to CD\DVD ROM.

    d)      After you boot your computer using Windows DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.

    e)      Select the correct time and Keyboard type.

    f)       Click Repair your computer in the lower left corner

    g)      Click on Troubleshooting, advanced option and then Command prompt.

    h)      In the Command prompt type the following commands:

    Type Bootrec.exe and press Enter.

    Now Boot Manager will display a list of options to repair the Boot sector. Type the Command:

    Bootrec.exe/ScanOs and press Enter.

    This option scans all disks for compatible Windows installations and displays entries that are not currently listed in the BCD store. If the SSD partition is listed, then you can use the ‘Rebuild’ command to rebuild the boot sector for this installation. Type the following command:

    Bootrec.exe/RebuildBcd and press Enter.

     

    If you are unable to boot to Windows even after following the above steps, then you can perform a clean installation of Windows 8.1 on the SSD partition and that should resolve the issue.
