Format may not be enough to properly configure your drive for boot.  Remove all partitions from it so that the Windows install can setup the partitions as needed.  The partition layout varies between BIOS and UEFI-mode computers for one thing.  If your computer is UEFI-capable, it's best to set it to UEFI Secure Boot enabled with no BIOS compatibility enabled, unless you also need to run Windows 7 on the same machine.

Hi Robert,

I would suggest you to disable the secure boot and check if it helps.

Basically, Secure Boot is a security standard developed by members of the PC industry to help make sure that your PC boots using only software that is trusted by the PC manufacturer.
When the PC starts, the firmware checks the signature of each piece of boot software, including firmware drivers (option ROMs) and the operating system. If the signatures are alright, the PC boots, and the firmware gives control to the operating system.

Hence, I would suggest you to try to disable it and check at your own risk. For this, you may try to boot to BIOS and then disable the secure boot.

Important: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly.
