Can’t Activate Office 2011
Your Office 2011 product key can not be re-used in any of the following scenarios:

 

  • Erased your startup volume and reinstalled using the 2011 installer
  • Replaced your hard drive or SSD drive
  • Attempt to install on a different Mac

In these situations, you will have to buy a new product key.

 

If you attempted to install with an older installer, you’ll have to remove what you have already installed. Here's how to completely remove Office 2011
https://github.com/pbowden-msft/Remove2011

 

You can download the latest full and final Office 2011 install (which is build 14.7.7) from https://go.microsoft.com/fwlink/?linkid=273962. You will need to enter your brand new never been used before product key (25 characters) to activate it.

 

Alternatives include:

Purchasing an Office 365 Subscription. It seems that every other day Amazon.com offers a $20 discount from the regular $99 price. Check the link price for discounts periodically. Office 2019 requires Mac OS 10.12 or later.

Use the free, limited feature version of Office at Office.com

Use LibreOffice, a free product that is pretty good.

Use Google docs.

Use Apple Keynote, Pages and Numbers apps

 

Regarding the ability to install 2011 anywhere you want, that was accommodated by making exceptions to the license agreement using the special phone number. The phone number is no longer available for this purpose, as all support for Office 2011 ended October 2017. An Office 365 subscription offers the ability for you to manage your own keys, so you can turn them on and off yourself and use them on any devices you want, as long as your subscription is active.

This problem has been covered several times on this forum.

 

The problem is usually caused by poor quality screen protectors and cases that do not have holes in the correct places.  This causes the proximity sensor to think that it's covered which will make the screen stay black when receiving and during calls.

 

The solution is to remove your screen protector and/or case and see if it works correctly.  If it does work you need to buy a better, compatible case or protector.

 

If no case or protector are used then your phone has suffered a hardware fault with it's proximity sensor.  You can get it repaired under warranty at a nokia care point.

Private Sub Worksheet_Change(ByVal Target As Range)
    
    Dim rngC As Range
    Dim rngD As Range
    Dim sh As Shape
    Dim rngS As Range
    
    If Intersect(Target, Range("I7:I111")) Is Nothing Then Exit Sub
    If Target.Cells.Count > 1 Then Exit Sub
    
    Set rngS = Target.MergeArea
    
    'Turn off events to keep out of loops
    Application.EnableEvents = False

    For Each rngD In Target.Dependents
    If rngD.Cells(1).MergeArea.Cells(1).Value = rngD.Cells(1).Value Then
        Call BarMacro(rngD)
    End If
    Next rngD
    
    Application.EnableEvents = True

End Sub
Sub BarMacro(rngC As Range)
    Dim sh As Shape
    Dim rngS As Range
    
    On Error Resume Next
    ActiveSheet.Shapes("sh1" & rngC.Address).Delete
    ActiveSheet.Shapes("sh2" & rngC.Address).Delete
    On Error GoTo 0
    
    Set rngS = rngC.MergeArea
    rngS.Interior.ColorIndex = xlNone
    rngS.Font.ColorIndex = 2

    If rngC.Value > 0 Then
        Set sh = ActiveSheet.Shapes.AddShape(msoShapeRectangle, rngC.Left, rngC.Top + 0.2 * rngS.EntireRow.Height, rngC.Value * rngS.Width, 0.6 * rngS.EntireRow.Height)
        sh.Select
        With Selection.ShapeRange.Fill
            .Visible = msoTrue
            .ForeColor.RGB = RGB(77, 211, 77)
            .Transparency = 0
            .Solid
        End With
        sh.Name = "sh1" & rngC.Address
        Selection.ShapeRange.Line.Visible = msoFalse
    End If
    
    Set sh = ActiveSheet.Shapes.AddShape(msoShapeRectangle, rngC.Left + rngC.Value * rngS.Width, rngC.Top + 0.2 * rngS.EntireRow.Height, (1 - rngC.Value) * rngS.Width, 0.6 * rngS.EntireRow.Height)
    sh.Select
    With Selection.ShapeRange.Fill
        .Visible = msoTrue
        .ForeColor.RGB = RGB(254, 94, 94)
        .Transparency = 0
        .Solid
    End With
    sh.Name = "sh2" & rngC.Address
    Selection.ShapeRange.Line.Visible = msoFalse
    rngC.Select
    
End Sub
