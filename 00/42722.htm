lee hounshell say: I believe I have finally solved this problem!

The chrome computer lockup issue appears to be being caused by Services that are either disabled or did not start.

I changed the "Remote Access Auto Connection Manager" service to Manual

and the following services to "Automatic (Delayed Start)":

 - "Application Layer Gateway Service"
 - "Microsoft .NET Framwork NGEN"
 - "Network Connections"
 - "Network List Service"
 - "Network Location Awareness"
 - "Network Store Interface Service"
 - "Remote Access Connection Manager"
 - "Telephony"

Also, I switched to the "Iron" browser: http://www.srware.net/en/software_srware_iron.php
(The Iron browser is faster to load than Chrome, but it does not fix the lockup.  You need to check your Windows services to fix the lockup problem)

After making these changes to my Windows 7/64 services, and restart
I have not had the problem.

Follow these steps to burn photos to a disc.

1. Insert a blank CD into the disc drive on your computer.

2. If Autoplay is on, a screen will appear on which you'll click "Open folder to view files." This will open the CD folder. If Autoplay is turned on, move on to Step 4.
