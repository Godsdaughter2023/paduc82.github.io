OK I've solved it!

You need to download the following 2 microsoft utilities:

1) Windows XP Video Decoder Checkup Utility: http://www.microsoft.com/downloads/details.aspx?FamilyId=DE1491AC-0AB6-4990-943D-627E6ADE9FCB&displaylang=en

2) Vista MediaCenter Default Decoders: http://mediacenterexpert.blogspot.com/2006/07/vista-media-center-decoder-utility.html

Then reboot.  When you next run media player it will use the ATI MPEG Video Decoder to and you will see the video and hear the audio.

Voila!
