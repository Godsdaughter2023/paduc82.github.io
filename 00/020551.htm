doug robbins - mvp office apps & services (word) say:

You can do what you want IF you assign a name to the range that begins with Row 3 and ends with the row that contains the last record and covers the columns that contain all of the fields.  Then when you attach the data source to the mail merge main document, the name that you assigned to that range will appear in the Select Table dialog that appears along with the names of the Sheets in the workbook.  Select that name and then click on OK and the data in Row 3 will be used as the field names for the mail merge.
