To create OneNote Templates on SharePoint Online, you can follow the steps below:

  1. Navigate to your SharePoint Site -> Site settings.
  2. Click Site content types under Web Designer Galleries.
  3. Create a new content type and select Document Content Types as Parent Content Type.
  4. Go to Advanced settings of this content type.
  5. Select Upload a new document template then upload your template.
  6. Go to the document library that you want to use this template.
  7. Select Library settings in the Library tab.
  8. Select Advanced settings, then change Allow management of content types to Yes.
  9. Click OK. Then select Add from existing site content types to add the content type.
  10. Select the content type. Then users can create new items based on the new template.

Search on Mac works differently from Windows.  On Mac you cannot search both your regular mailboxes AND your online archive simultaneously.  This is because all of the mail in your regular mailboxes is downloaded to your machine first, then indexed for search.  None of the online archives are downloaded to your machine, so it is performing a search on the Exchange server.  So even if you specify "all mailboxes" as you search scope, it will not include both mailboxes and online archive.

Known issue on Windows 10 Technical Preview for phone: Selecting a - Microsoft Community

vinod archak say:

Known issue on Windows 10 Technical Preview for phone: Selecting a photo for the lock screen

Hello,

Users of Windows 10 Technical Preview for phone may experience a crash when selecting a photo to use as their lock screen background.  When the crash occurs, you are suddenly returned to the Start screen.

This issue occurs if you first open the Photos app before going into Settings to change your background or lock screen. 

To work around this issue, do NOT first open the Photos app.  Simply go to Settings, then Personalization, and then select either Background or Lock Screen, and choose Browse.  This then takes you to the Photos app where you can select the photo you want to use.

Note:  If you’ve experienced this issue, you may need to power off the phone first before attempting the workaround steps.

We are aware of this issue, and are currently investigating a fix for a future build.
