Go to Start / Control Panel / Ease of Access / Make the compter easier to see / and uncheck the box to turn on the magnifier.  If that doesn't resolve the problem in that account, go to Start / Search Box and type msconfig.exe and enter and then double-click on the program icon that appears.  Go to the starup tab and see if anything relating to Ease of Access or Magnifier is included - if so, uncheck the box to stop it from running at startup and that should solve the problem.  If you find no such program, go to the services tab and look for the same thing.  If you find it uncheck the box so it won't start at startup and that should solve the problem.  Here's an article on how to disable magnifier in Vista that should work globally and for all accounts: http://www.vistax64.com/tutorials/125037-magnifier-turn-off.html.  I suspect this will do the trick.

If neither of these options work, you can always treat the account as a corrupt user profile and fix it by usinghttp://windows.microsoft.com/en-AU/windows-vista/Fix-a-corrupted-user-profile.

=============================================================================================================================

Vista DOES make automatic restore points just like XP - it just needs to be told to do so. http://www.vistax64.com/tutorials/169523-system-restore-task-change-automatic-restore-point-schedule.html orhttp://www.howtogeek.com/howto/windows-vista/change-how-often-system-restore-creates-restore-points-in-windows-vista/ (this second one is a bit easier to follow).

To find System  Restore, go to Start / Control Panel / Backup and Restore Center / and to the left are two options - one to do a System Restore and one to create a restore point or change the restore point settings.
