shilohbob say:

First, as you are aware RAM memory is the fastest way your computer device PC/Laptop/Tablet/ Mobile Phone., etc. can access data to run the apps on the device.

It is used to store data for your apps that you are currently running, data of apps that you have recently used, it also retains data of apps in anticipation of your future requirements, this could be up to the maximum amount of the RAM you have installed.

If you have your device setup for Windows to manage your Virtual Memory, also remembering that Virtual Memory consist of you RAM plus your Paging File, (Paging File is a section of the HDD that is utilised similar to RAM). Windows will automatically swap app data between RAM & Page File depending on you data needs & data priority.

So to have app data residing in RAM is not a bad think, if you start using an app that requires more memory Windows will discard app data which is not being utilized and replace it with data of the app that you are currently using’

In the example below’ OneDrive’ (which I am an not currently using) has 0.6MD residing in ‘Memory’ and it is not using any other resources ie; CPU, Disk or Network, and if I started using an app like ‘Photoshop’ and it cannot fit in to my RAM, Windows will discard ‘OneDrive’  or any other app data that I am not currently utilizing and install ‘Photoshop’ data instead.

RAM requires power to work, once you turn ‘off’ power everything in RAM is discarded, when you turn ‘off’ your device, if you could see the contents of RAM, it would be empty. This is why when you turn on your device it take a period of time to bring up the work screen, whether it be ‘Start Screen’ or ‘Desktop’. When you put your device to ‘Sleep’ it runs on low power, so therefore data is retained in ‘Memory’

I apologize if my explanation is something which you already aware of, but in general terms there is nothing wrong with your device.
