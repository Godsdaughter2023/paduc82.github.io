bernard liengme say:

The sub below seems to do what you asked.

Right click the worksheet with the data of interest, select View Code and copy and paste sub to the resulting Sheet module

Note the code works only when Open is typed with capital o - is this a problem?

best wishes

Private Sub Worksheet_Change(ByVal Target As Excel.Range)
    If Target.Column = 10 Then
        ThisRow = Target.Row
        If Target.Value = "Open" Then
             ActiveCell.EntireRow.Copy
             Workbooks.Add
             ActiveSheet.Paste
        End If
    End If
End Sub
