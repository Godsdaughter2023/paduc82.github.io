I am setting up a mail merge letter. Normally I would address the letter Dear [Title] [Surname]. In some cases the data source does not have a title. In these cases I want to use the form Dear [firstname].

What is the correct syntax?

andrea jones - all about resources say:

Enter the following IF field using Ctrl+F9 to insert each of the pairs of curly braces:

{ IF { MERGEFIELD Title }= "" {MERGEFIELD First } "{MERGEFIELD Title } {MERGEFIELD Last }" }

when you have done this press F9 to update the field and run your merge.  Note the inverted commas around the fields used to make the Title and Surname and the space between those fields.
