Keyboard Shortcut for Find - Microsoft Community

ross payne say:

Keyboard Shortcut for Find

I know that Shift + F4 is the shortcut key for Find Next. What is the shortcut key for Find Previous? Thanks.
suzanne s. barnhill say:

Actually, Shift+F4 is defined as "Repeat a Find or Go To action." There is no built-in shortcut for Find Previous, and in fact there is actually no such command. But when the Find or GoTo command has been executed, the "browse object" is set to Find/GoTo, so you can use the shortcuts for BrowseNext (Ctrl+Page Down) and BrowsePrev (Ctrl+Page Up).

In Word 2010 and earlier, the browse object can be set manually using the Select Browse Object button between the browse arrows at the bottom of the vertical scroll bar. These were removed from Word 2013 and only partially restored as items in the dropdown Find menu in the Navigation pane.
