cmdrkeene say:

This happens if you have a high DPI screen ("retina display", as Apple calls it), but the software isn't programmed to handle it.  Windows provides some DPI scaling systems to help, but not all programs know how to take advantage of this.

You can usually fix the issue by adjusting the "use one scaling level for all displays" checkbox, but that may not work for all programs.

Adobe is busy working on updates to help their programs work better on high-res screens, but HP is likely to be a laggard in that area.
