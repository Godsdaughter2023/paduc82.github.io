a. user say:

Hi Amir,

 

Raw file system is considered as a file system which is installed on the disk, immediately when it is created in the factory. When we try to convert a raw file system to NTFS which allows us to store our data like pictures, audio, documents and video, we need to format the drive. 

Reasons for causing RAW hard drive are Power surge, format failure, drive would have affected by virus etc.

How to prevent raw drives

  • Using good anti-virus software
  • Handling power surges through voltage guards
  • Using good inverters to avoid abrupt shut down of computers
  • Proper shut down and proper booting of the system

In this case the drive has to be formatted to enable its access for further usage.

Since the data in the hard disk is important I would suggest you to connect the hard disk to a different computer, if your able to access the drive I would suggest you to take the data back up.

Note: Please make sure the data is backed up.Any data present on these hard disk might be lost in the process. The data could be lost.
