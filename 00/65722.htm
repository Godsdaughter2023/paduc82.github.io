rufford155 say:

Difference between Project and Movie in Live Movie Maker

skier-hughes say: I give the analogy based on baking a cake, the project is your recipe, the cake your finished movie.
Until you bake the cake in the oven you can make as many changes as you like, but once baked it's finished.
You can make as many changes to your project as you like, but once you save as a movie file that's it.
The project is like a list of instructions telling WLMM what to do with each file, like the recipe tells you what to do with each ingredient.
You wouldn't use the project file to burn the dvd, just like you wouldn't try to bake the recipe in the oven :)

considerate_guy say:

In order for a DVD to play on a DVD player it must be created with a specific format to match the DVD standard.  As PapaJohn noted, internally that means that the resulting disc will end up with the video encoded as MPEG2 video inside of .vob files.  But you actually don't need to know the specifics of the DVD format to use Windows DVD Maker.  You can just pass it video files and pictures and it will assemble them in the right format to play on a DVD player.

You need your project file (.wlmp) in Windows Live Movie Maker in order to go back to the project later and make more edits.  As skier-hughes noted, the project is like a recipe.  It contains a list of all the ingredients and how you want to combine them.  When it comes time to save your movie, it looks at the list of ingredients (the video, photo and music files on your computer) and puts them together.

If you are saving your movie to your computer, you'll get a .wmv files out of WLMM.  If you are saving your movie to a DVD, internally WLMM tosses the information over to Windows DVD Maker to have it put the files together for the DVD.

To answer your final question about putting .avi files on to a DVD the answer depends on whether you want to have the video play on a DVD player, or if you just want to back up the files onto a DVD disc as a data file.

In the first case, you can just open Windows DVD Maker, pass it your .avi file, photos, etc. and it will do all the "magic" of creating a video DVD that you can play on a DVD player.

In the second case, you would just burn the .avi file onto the disc itself.  This file might be playable in another computer, but it won't be playable in a DVD player.  An .avi file is not part of the DVD standard and thus a disc with an .avi file isn't understood.

      Hi,

      I would suggest you to manually Install Device Drivers in Windows Using INF Files.

      a)      Extract the files if any and place them into a convenient location like a folder on c drive

      b)      Press “Windows Key + R” together or go to Start > Run, on the Run Prompt, type “devmgmt.msc” and press enter to launch device manager.

      c)       On the Device Manager screen, locate the device for which you have downloaded the drivers and right click that device. On right click menu, select Update Driver Software.

      d)      This will show you a Window which will ask you whether you want to install the drivers automatically or from a specific location. Select the Specific location option if you have downloaded the drivers already in step 1.

      e)      Click Browse my computer for driver software and select the driver’s location where you saved and extracted it in step 1. After you locate the folder, click OK and click Next.

      f)       Let the Device manger scan the folder and install the driver. If the driver files are correct for the device, it will be successfully installed and a success message will show up. After this the device manager Window will refresh and show up the installed device properly categorized and without any yellow exclamation marks.
