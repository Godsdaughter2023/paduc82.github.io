myall_blues say: I assume you want the total of the three separate figures that you got?

Unfortunately in SUMIFS the criteria are all cumulative i.e. it is AND, not OR. So in your SUMIFS formula, it is trying to sum the values in H when D:D=U4 AND A:A=TODAY()-7 AND B:B=A24. As you've discovered, there are none that meet that combination of criteria.

to do what you want you need to add the three together e.g.
=SUMIF(Sheet5!B:B,A24,Sheet5!H:H)+SUMIF(Sheet5!A:A,"="&TODAY()-7,Sheet5!H:H)+SUMIF(Sheet5!D:D,$U$4,Sheet5!H:H)

or you can do

=SUM(SUMIF(Sheet5!B:B,A24,Sheet5!H:H),SUMIF(Sheet5!A:A,"="&TODAY()-7,Sheet5!H:H),SUMIF(Sheet5!D:D,$U$4,Sheet5!H:H))

Or this, which won't allow spaces but will allow periods or other punctuation characters if need be:

Private Sub TextBox1_Change()

Dim strText As String
Dim x As Long
Dim sOKChars As String

' Change this as needed to suit your
' local settings
sOKChars = "0123456789."

strText = Me.TextBox1

For x = 1 To Len(strText)
    If Not InStr(sOKChars, Mid$(strText, x, 1)) > 0 Then
        strText = Replace(strText, Mid$(strText, x, 1), "")
    End If
Next

Me.TextBox1 = strText

End Sub
