bob jones aka: cybertaz say:
Contrary to what Phillip suggested, double-clcking a Section Break will not select it -- It opens the Document dialog as you've discovered. Section Breaks cannot be deleted [or inserted] from within that dialog. Only the nature of the current Section can be changed.

However, Section Breaks frequently are used to enable layout changes (e.g., the number/width of columns or the size of margins) from one Section to the next. If you delete the Section Break that layout change cannot be maintained because the Sections preceding & following that Break will become a single Section... The Section preceding the Break will be forced to conform to the same layout specs as the Section that follows the Break. That's what the warning is telling you.

If it is necessary to delete the break you can use Phillip's second option -- Drag across the Section Break line, then press delete. Alternatively, just position the insertion point at the beginning of the Break line then press the del [forward delete] key. On a condensed keyboard use fn+delete.

You didn't indicate why you want to remove it, what type of Section Break it is, nor describe the layout of the Sections involved, so it's impossible to say what the repercussions will be. All I can say is that you will see changes take place in the flow of the document. If those changes aren't acceptable you can use Undo to restore the Section Break.

use autofill to help select an entry from a validation list - combo - Microsoft Community

lily az say:

use autofill to help select an entry from a validation list - combo box

Several years ago I got some great code, (shown below) which allowed me to double click on a cell with a data validation list. As I typed the drop down list would filter and find my selection. The code has been working great, and now I would like to adapt it to another spreadsheet but I can remember how I did it or where I got the help. Is the author out there?  I don't do too much with Excel, but I was able to follow the help I got once beforeso hopefully I can do it again.  Thanks

 

 

Private Sub TempCombo_KeyDown(ByVal _
        KeyCode As MSForms.ReturnInteger, _
        ByVal Shift As Integer)
    'Hide combo box and move to next cell on Enter and Tab
    Select Case KeyCode
        Case 9
            ActiveCell.Offset(0, 1).Activate
        Case 13
            ActiveCell.Offset(1, 0).Activate
        Case Else
            'do nothing
    End Select

End Sub

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
Dim str As String
Dim cboTemp As OLEObject
Dim ws As Worksheet
Dim wsList As Worksheet
Set ws = ActiveSheet
Set wsList = Sheets("ValidationLists")
Cancel = True
Set cboTemp = ws.OLEObjects("TempCombo")
  On Error Resume Next
  With cboTemp
    .ListFillRange = ""
    .LinkedCell = ""
    .Visible = False
  End With
On Error GoTo errHandler
  If Target.Validation.Type = 3 Then
    Application.EnableEvents = False
    str = Target.Validation.Formula1
    str = Right(str, Len(str) - 1)
    With cboTemp
      .Visible = True
      .Left = Target.Left
      .Top = Target.Top
      .Width = Target.Width + 15
      .Height = Target.Height + 5
      .ListFillRange = str
      .LinkedCell = Target.Address
    End With
    cboTemp.Activate
  End If
 
errHandler:
  Application.EnableEvents = True
  Exit Sub

End Sub

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
Dim str As String
Dim cboTemp As OLEObject
Dim ws As Worksheet
Set ws = ActiveSheet
Application.EnableEvents = False
Application.ScreenUpdating = False

Set cboTemp = ws.OLEObjects("TempCombo")
  On Error Resume Next
  With cboTemp
    .Top = 10
    .Left = 10
    .Width = 0
    .ListFillRange = ""
    .LinkedCell = ""
    .Visible = False
    .Value = ""
  End With

errHandler:
  Application.ScreenUpdating = True
  Application.EnableEvents = True
  Exit Sub

End Sub

hansv mvp say:

Activate the sheet where you want to use this.

Activate the Developer tab of the ribbon.

In the Controls group, click Insert.

Under ActiveX Controls, select Combo Box (ActiveX Control).

Click on the sheet to insert the combo box.

With the combo box selected, click in the name box on the left hand side of the formula bar.

You'll see something like ComboBox1 there. Change this to TempCombo (the name used in the code).

In the Controls group of the ribbon, the Design Mode button will be highlighted. Click it to unhighlight it.

 

Right-click the sheet tab and select View Code from the context menu.

Copy the code into the code module that appears.

Switch back to Excel.

 

Since the workbook now contains VBA code, you should not save it as a .xlsx workbook, but as .xlsm, .xlsb or .xls.
