AS I am seeing this is a very common problem, one of which I have now encountered on two of my computers and have now got working on one of them, I am hoping this may be a possible solution for the majority of people out there. I attempted to contact customer support, which the level one tech support staff could not help, unfortunately I could not get through to the level 2 support staff as the hold time killed the battery on both of my cordless phones waiting on an available support agent, so trust me I know everyone's frustrations.

As I was on hold I performed a number of my own troubleshooting ideas, such as running ipconfig in an elevated command prompt that should the connection as a Hybrid without any I.P. Address, I tried to release and renew the connection without any success, then I even tried to create a dynamic I.P. instead of static by specifying a specific unused I.P. Address with no luck as the computer just refused to use an I.P. Address that is needed to get a network connection that will enable Internet capabilities.

The problem that I believe to have figured out is conflicting security software. In my case I believe it was the firewall program I use and Windows Firewall.  As I had been on hold for nearly an hour and a half I decided to uninstall the Internet security suite that I use to verify this assumption.  As the uninstall neared completion, all of a sudden the network started working and I was able to access the Internet again. I rebooted the computer and reinstalled the security suite I use and everything now appears to be working. So my suggestion would be to uninstall any security program (i.e. antivirus, firewall, antimalware, etc.) to see if you get internet access.  If that solves the problem you should be safe to reinstall those apps after you regain network access and everything should be good. Make sure to run the updates those apps as they will be reset to original virus and malware definitions.

I hope this helps, as it was the only thing I could figure out that did work for me.
