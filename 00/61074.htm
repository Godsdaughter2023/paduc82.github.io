tryx3 say:

HDD passwords are normally set in the BIOS and can only be unset using the same utility [not Windows 7 itself].  The arrangements vary between PC brands so you'll need to check in your BIOS, PC manual & PC maker's support site.

In some PCs, such as many Dells, the BIOS "setup password" overrides the HDD password so you can use that to unset the HDD password - but only if your BIOS has this ability. 

In others, the HDD password is completely independent so if there is no advice from the PC maker you have probably lost the HDD concerned & any replacement you put on the same system.  In this case, a specialist data recovery firm might be able to help you at a price.
