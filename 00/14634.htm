Use the following field construction
 
{ IF { PAGE } > 8 { PAGE }  }
 
With the above, the ninth page will be numbered 9
 
If you want the numbering to start at 1 on the ninth page, use
 
{ IF { PAGE } > 8 { = { PAGE } - 1 }  }
 
You must use Ctrl+F9 to insert each pair of field delimiters { } and use Alt+F9 to toggle off their display

4.   If I have something really important, I can upload manually ( same as in step 2) from the phone rather than waiting for the move onto the PC first. 

5. This way I never have to have much phone storage used up by pics. They can be viewed on any of my devices online by opening the OneDrive App.

ossiemac say:

       Edited since initial posting (formula changed)

  • Copy the column of abbreviations to another vacant area of the worksheet (Column D in caption below)
  • Select the NEW copy of abbreviations column (Column D)
  • Select Data ribbon
  • Select Remove duplicates icon (In data tools block)
  • In the dialog check the box against "My data has headers" (Assuming you have column headers)
  • Click OK
  • Now type in the full names against each of the 41 abbreviations.
  • Assuming the new column of unique abbreviations is column D and the adjacent column E has the names then insert the following formula in cell B2.

          =VLOOKUP(A2,$D:$E,2,FALSE)

  • If new columns of unique abbreviations and names is not D:E then edit the Table_array part of the formula ($D:$E,2) to match your columns.
  • Copy the formula down to bottom of data.
  • Note that the table_array is in absolute mode (with the $ signs)
  • If you want to remove the formulas from column B then select the column and then Copy -> Paste Special -> Values over top of itself. (Don't delete the unique list of abbreviations and names unless you have done this.)
