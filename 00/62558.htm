cmdrkeene say:

Alright, let's see what failure is happening that is causing it to restart.

One way to do this is to turn off automatic error reboots.  This way when a failure happens, you'll get a chance to see the error code and then restart yourself.  We can then use that error code to find out what's wrong.

Here's how:

  1. Right-click the Start button and choose Run
  2. Type or paste in SystemPropertiesAdvanced and then click OK
  3. In the "Startup and Recovery" section, click Settings...
  4. Turn off the Automatically Restart option, then click OK
  5. Restart your computer (do not shutdown+turn back on:  do a actual restart)

Now, just wait.  The next time a failure occurs that would have triggered an automatic reboot, you'll see the error message instead.  Note what it says, then restart your computer (with the power button) and come back and let us know what the error was.
