keith- say:

If you trust software that Microsoft Security Essentials has detected, you can stop Microsoft Security Essentials from alerting you about risks that the software might pose to your privacy or your computer. To stop receiving alerts for this software, you must add the software to the Microsoft Security Essentials allowed list. If you decide that you want to monitor the software again later, you can remove it from the Microsoft Security Essentials allowed list at any time.

To add an item to the allowed list

1.   The next time Microsoft Security Essentials alerts you about the software, click theShow details link.

2.   In thePotential threat details dialog box, click the down arrow in theRecommendation column, and then click Allow.

To remove an item from the allowed list and enable Microsoft Security Essentials to monitor it

1.   Click theHistory tab, and then select the Allowed items option.

2.   In Windows Vista or Windows 7, clickView details to see all items. If you're prompted for an administrator password or confirmation, type the password or confirm the action.

3.   In Windows XP, you'll need to log on as an administrator on the computer to see all items.

4.   Select the item that you want to monitor, and then clickRemove. If you're prompted for an administrator password or confirmation, type the password or confirm the action.

Warning:
 

Donot allow software with severe or high alert ratings to run on your computer because it can put your privacy and the security of your computer at risk.

karl timmermans [outlook mvp 2012-2018] say:

There are a variety of elements to your question so first step is understanding what it is you're seeing

In reference to the Outlook AddressBook itself - the following may be of interest

Understanding the Outlook AddressBook
http://www.contactgenie.info/understanding-the-outlook-addressbook


*** how the AB displays the contacts (First/Last or FileAs) is defined via the Settings

In reference to <Suggested Contacts>

Outlook 2010 AutoComplete Cache and Suggested Contacts
http://www.windowsitpro.com/article/outlook/outlook-2010-autocomplete-cache-and-suggested-contacts

In reference to <Frequent Contacts> - Outlook in and of itself doesn't create a folder by that name so it's likely something that got migrated from GroupWise. <Frequent Contacts> in essence are the same as <Suggested Contacts> under a different name. Not entirely sure if this folder can be deleted or not but if you don't want it to appear in the Outlook Address Book as described in the Overview - uncheck the property "Show this folder as an email address book".

To that end, the following may be of value in terms of migrating from GroupWise to Outlook

GroupWise to Outlook Conversion Guide
https://docs.google.com/viewer?a=v&q=cache:Kh0HJo8h3aAJ:www.msmc.la.edu/PDFFiles/OIT/GroupWise%2520to%2520Outlook%2520Conversion%2520Guide.pdf+what+creates+Frequent+Contacts+in+Outlook&hl=en&gl=ca&pid=bl&srcid=ADGEESjieiChnuyYJRldF3XTLAQkxkPMbnlT5m6jf5H9Yx03ilgQtFbSOjW9-siAepuICDXA8l3UwTUISnTux5u2IDdY-O7VHLmQR5AzRmtFCyr1dxFVsskAxR_wtPkgkEXz9oAUQbTg&sig=AHIEtbRF_sIxg4MM1AG3sO_LzNu6OJpxJA&pli=1

Re: All things <Mobile>

Means you have a mobile AddressBook service added - whether or not it should have been is up to you/your organization so would check first before deleting it.

Re: All things prefaced with the word <All> along with the entry Global Address List and Public Folders

Means you are using Microsoft Exchange and there is no way to remove those entries from the Outlook AddressBook list without removing your Exchange account in the Outlook profile. There is a <sync> option in Outlook 2010 between your default contacts and the Exch GAL which may or may not be configured. Some articles in reference to that which may be of interest to you or someone in your organization are:

Disable global address list synchronization for Outlook 2010
http://technet.microsoft.com/en-us/library/ff871432.aspx

Configure Outlook 2010
http://technet.microsoft.com/en-us/library/cc303392.aspx

Determine which features to enable or customize in Outlook 2010
http://technet.microsoft.com/en-us/library/ff631135.aspx


So the real "short" answer to your question of how to "combine the mess" is "there is no concise short answer" - depends on what you need/use in your environment etc. If you want to see the contents of each of the contact folders listed in the AddressBook list - go to the Outlook Navication Pane and click on each of the folders under the <Contacts> group or the <All Folders> group to see what each of those contain vis-a-vis what you may be looking for. You'll need to decide how you want to ultimately organize your contacts from there
