issbrian say:

How do you add another windows bar to the side of the screen or top or bottom.

In XP and Vista, you used to be able to drag a folder to the top, bottom, or sides to create a new bar that has items on it. How do you do that in Win 7 now?
chris.h - support engineer say: Hello issbrian,

Thank you for visiting the Microsoft Answers Community.

In Windows 7, toolbars can only be added to the taskbar, not able to drag and attach to the side of the desktop.

    - Right click the Taskbar > hover Toolbars > select New toolbar
    - Select the folder that you want for your Toolbar > click Select Folder

This will add the toolbar to the right side of the taskbar.

Hope this helps
