  11. Place the Windows 7 install disk in the disk reader and start your computer.
  12. Once the install process is started, choose CUSTOM.  Ignore the check box about drivers, unless you can not proceed further.
  13. When you get to the place where it asks you where to put the installation, click Advanced and delete all partitions.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already own to reactivate if necessary. or use the reset function in Windows 10.

mcnpathfinder say:

I read somewhere that in some cases Disk Cleanup pops up and starts acting like it did for me after installs of Office 2010 that turned out to be faulty. We use a package distribution system called Marimba that said the install had failed (even though Office was working fine). I uninstalled Office using Marimba and then forced a policy update to reinstall. No Disk Cleanup messages anymore. I would assume they got worse when on the network/VPN because our "packages" reside on dedicated servers and Office was attempting to repair itself in the background, or finish out the install and had a problem locating the sources.
