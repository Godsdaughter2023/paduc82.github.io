greg carmack - mvp 2010-2020 say:
Hi Ivan. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

It's important to know what kind of Key you had, or else you could rely on a Keyfinder and use the wrong key since it may only find a default Version key that doesn't activate.

Since you said you bought a key, what kind was it? Full retail, builder's OEM, etc?

If it was a full retail key then you can move it anywhere. If it's still the same install then it should be the Windows Key found when you run ProducKey to scan for keys: http://www.nirsoft.net/utils/produkey-x64.zip

But if it's builder's OEM then it can't migrate to another PC beyond this one.

If it's free Upgrade then there is no key other than the default version key. A Digital License is stored in MS Activation servers so you'd skip the Product Key during install and it will reactivate when you connect to the internet.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
