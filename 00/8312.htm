ossiemac say:

I am assuming that you want to open all files that match the wildcard string. Is this correct. If so, you need to find each one separately and assign the found file to a variable and then use the variable to open the file.

Dir() on its own is like find the next file matching the initial filename with the wild cards.

Sub callLoadingList()

    Dim fileloc As String
    Dim tF As Boolean
    Dim fileName As String
   
    fileloc = Application.ThisWorkbook.Path
   
    fileName = Dir(fileloc & "\*Loading List*.xls")     'find first matching filename and assign to a variable
    
    Do While fileName <> ""
        Workbooks.Open (fileloc & "\" & fileName)     'Open the found file
        fileName = Dir()   'Find the next match and assign to fileName
    Loop

End Sub
