In the end I had to create a temporary .htm file (opening and closing tags surrounding content of Rich Text field), and then Insert it into the Word file I was writing to, along the following lines:

                        'send to temp file then insert
                        Set fso = CreateObject("Scripting.FileSystemObject")  ' New FileSystemObject
                        strTempFile = fso.GetSpecialFolder(2) & "\" & fso.GetTempName & ".htm"
            
                        'write to temporary .htm file
                        Set f = fso.CreateTextFile(strTempFile, True)
                        f.Write "<html>" & [contents of your Rich Text field] & "</html>"
                        f.Close
                        Set f = Nothing
            
                        objWord.Selection.InsertFile strTempFile
                        fso.DeleteFile strTempFile

You'll need a reference set to the Microsoft Scripting Runtime.

You'll find an example of the latter method in Concat.zip in my public databases folder at:

Allen Brown also provides a concatenation function, using the former method, at:

http://allenbrowne.com/func-concat.html

Whichever approach is used, to return a set a set of distinct concatenated values, the function would need to be called in a query which returns a set of distinct AccountID values, and the recordset used within the function would need to return distinct, in combination, AccountID/RecipientsAndDetails values.

Outlook hangs at Loading Profile or Processing - Microsoft Community

diane poremsky mvp (slipstick.com) say:

Outlook hangs at Loading Profile or Processing

If you are using  Office 365 Version 1905 (Build 11629.20196) and higher, please see Outlook won’t open or is stuck at loading profile

There are several things that will cause Outlook to hang on loading the profile.   Wait a few minutes to see if it finished loading, if not, troubleshoot the problem using the following methods:  

  1. Press Alt+Tab. Are there any windows hidden behind other windows, such as a password dialog or macro error dialog? 
  2. Does Outlook load if you start in safe mode? If so, a COM addin could be preventing Outlook from loading. Disable all COM Addins in File > Options > Addins and restart Outlook. Re-enable one addin at a time and restart Outlook to see which is the culprit.  (Safe mode instructions are in the article linked above.) 
  3. Create a new profile. If you use a POP account, let Outlook create a new pst file. If Outlook loads, either the profile or pst is corrupt. 
  4. If you use an Office 365 business email account, use Outlook Diagnostics (Support and Recovery Assistant) to diagnose and repair the problem.

The article at Outlook not responding, stuck at "Processing," stopped working, freezes, or hangs has step-by-step instructions to diagnose the cause.
