john w. vinson say:

I'll agree with Peter. The equipment name field should simply not exist in any table other than the Equipment table. Storing it redundantly in this form's table wastes space, and much more importantly risks data anomalies - what if the description changes? Do you want to look up every single record for that item and change the description? Heck no!

Instead, include the description in the Combo Box's row source, and put a textbox on the form with a control source like

=comboboxname.Column(x)

where x is the zero based index of the item's description (1 if the description is the second column, for example). This will display the description without the need to redundantly store it in your table.

John W. Vinson/MVP

ken sheridan say:

You can even do it without a single line of code.  Base the form on a query which joins the current table to the Equipment (lookup) table on the equipment code fields.  Include all fields from the current table (having removed the redundant equipment name field from the table definition), plus the equipment name field from the Equipment table.  When you select an equipment code the equipment name value will show in a control bound to that field.  You can set its Locked property to True (Yes) and its Enabled property to false (No) to prevent users trying to edit it of you wish (strongly advised).

You should also create a relationship between the two tables on the equipment code columns, and enforce referential integrity, if you have not already done so.

shane devenshire say:

Yes, you can do this.


In 2003:
1. Select the cells you want to format A1:J1, A10:J10
2. Choose Format, Conditional Formatting
3. Choose Formula is from the first drop down
4. In the second box enter the formula:

=A$10>=LARGE($A$10:$J$10,3)

5. Click the Format button
6. Choose a color on the Patterns tab (or any available option)
7. Click OK twice.

In 2007:
1. Highlight all the cells on the rows you want formatted
2. Choose Home, Conditional Formatting, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following formula:

=A$10>=LARGE($A$10:$J$10,3)

5. Click the Format button and choose a format.
6. Click OK twice
