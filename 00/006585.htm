There are two factors in play.

1. It simply takes longer to open text files than other types of files. Text is an inefficient way to save data.

2. From your description, you ran out of RAM and your Mac is using swap space (your startup volume) as additional memory

The most efficient file format you can use is Excel Binary .xlsb file format. You an use File > Save As and change the format to Excel Binary file format in the Save As dialog.

The only solution to running out of RAM is to either get more RAM, or don't have other RAM consuming applications and workbooks open.

suzanne s. barnhill say:

If you have inserted a caption to a wrapped graphic, the caption will be in a text box, which might (but should not) affect the ability to use the style separator. If you use the shortcut Alt+Ctrl+Enter you don't need the style separator on the QAT.

Note that this works for a table of figures only if it is based on the Figure label (or some other caption label). If it is based on the Caption style (or some other style), you will need to change the style of the part you don't want included (that's what the "style separator" is for).

rich michaels - mvp (greatcirclelearning.com) say:

Hi Sue,

Yes, do not leave the style as Normal because, that conflicts with the paragraph style named Normal and the paragraph style named Normal will override what you are attempting to do within the table.

For example, if you want text within table cells to be a certain font or font size, than instead of direct formatting the Normal style that defaults within the table cells, create a paragraph style called MyTableText and assign to it all of the attributes you want... then only use that specific style for text within the table. Make sure all end-of-cell markers are assigned your custom paragraph style... in other words, don't leave the end of cell markers "Normal". The only table markers that cannot be altered are the end-of-row markers. They will stay Normal... but that's okay.

BTW - you can use other custom paragraph styles within the table cells and you can mix them within a cell... the point is... just don't leave them as a Normal style or a style name used in other parts of the document that has different formatting attributes.

Hope this helps

mike h.. say:

Hi,

To do this you need to be running Excel as an Administrator or you will get a RunTime Error 70 'Permission denied'

 

To run as adminsitrator right click the Excel icon - click 'More' and run as administrator. You can then change the system date with:-

Sub ChangeClkDate()
Date = DateSerial(2018, 8, 5)
End Sub

and the time with

Sub ChangeClkTime()
Time = TimeSerial(14, 0, 0)
End Sub

Note. Despite being able to do it, I would need a very compelling reason to do so. Fiddling with the system can have effects that may not have been anticipated.
