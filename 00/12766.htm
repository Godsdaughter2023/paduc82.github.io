Text box from left margin to. right margin. - Microsoft Community

peter a. forkes say:

Text box from left margin to. right margin.

I want to be able to create a default text box that is aligned on the left at the left page margin and aligned at the right on the right margin.

john korchok say:

Charles and Doug both have useful suggestions, I'll give you a third method. You can create a style that includes a frame. The frame portion of the style will take care of the border, light grey background, position and text wrap. Here's how to create it:

  1. Click on Home>Styles Pane to open the list of Styles.
  2. Select a paragraph of text to be highlighted.
  3. On the Styles Pane, click on New Style.
  4. Give it a name like Highlighted Text.
  5. If you want to add this style to the underlying template, so you can use it in future documents, check the Add to template option.
  6. Check the Automatically update option. This is a temporary setting until you get the look just right.
  7. Click on the Format... dropdown and choose Frame...
  8. If your margins vary and you always want the highlighted area to go margin-to-margin, leave the Width dropdown at Auto. If your margins are constant set Width to Exactly and enter the width in the At: field.
  9. In the Horizontal Position section, leave Position at Left for now and change Relative to Margin.
  10. In the Vertical section, set Distance from text to the amount of space you want above and below the highlighted text. 0.13" is a good starting value.
  11. OK Out. The Highlighted Text style is added to the list of styles. With the sample text still highlighted, click on the Highlighted Text style. The sample is popped into a frame, but this frame has no color or border.
  12. Hover your mouse over the frame border until you see a cursor with 4 arrowheads, then right-click and choose Borders and Shading... You'll know if your mouse was at the right position if the Apply to: dropdown is set to Frame.
  13. Set the border properties you want, then click on Box to add the border to all sides. 
  14. Click on the Shading tab and choose a Fill color. OK out.
  15. Adjust the formatting until it looks good to you. When you're done, right click on the style name, choose Modify and uncheck Automatically Update. This will prevent this style from accidental modification.

To use the style, simply select a paragraph of text and click on the style to apply it.

To move the frame relative to the text, hold the Shift key, click on the frame border and drag the frame up and down.

Here is a sample document including this style. Feel free to use Word's Organizer to copy this to your document or template.

Hi JBox117,

As I understand it, you have already created Office 365 cloud user accounts before the cutover migration. If this is true, then the only way that you can continue to finish your migration is to delete those previously created cloud users and let the migration automatically re-create them. This is just how a cutover migration works.

Below are the PowerShell commands you will require to permanently delete those cloud users:
1.    Remove-Msoluser -UserPrincipalName *** Email address is removed for privacy ***
2.    Remove-Msoluser -UserPrincipalName <*** Email address is removed for privacy ***> -RemoveFromRecyclebin –Force

When performing an IMAP migration, it is OK (required, actually) to manually create cloud users before the mailbox data is moved. What is the best way to delete a subset list from a master list in - Microsoft Community

shayla_1227 say:

What is the best way to delete a subset list from a master list in excel?

I have a list of over 3,000 names that were pulled from a current database of employees. I have a separate list of about 400 former employees that need to be removed from the master list. What is the most efficient way to do this in Excel?
hansv mvp say:

I assume that you want to match on both name and title?

Let's say you have sheets Current and Former, and name and title are in columns A and B on both sheets, with the data starting in row 2.

In the first empty column next to the list on Current, enter Former in row 1, and the following array formula (confirmed with Ctrl+Shift+Enter) in row 2;

 

=ISNUMBER(MATCH(A2,IF(Former!$B$2:$B$400=B2,Former!$A$2:$A$400),0))

 

Adjust the ranges as needed. Fill down to the end of the data.

You can now filter for TRUE in the new column, then delete the filtered rows. Finally, remove the filter (and the column with formulas, if you like).
