mel77084 say:

People ask for help, while carefully concealing all of their details of what they have to work with.

Right click on the little speaker icon on the Task Bar, and select "playback devices". There, you can see (and change, if you wish) which device your audio is being sent to, and run the audio test program. If you have audio, but it's being sent to a different jack than you're listening to, just change the default jack.

If you don't find audio getting to any of the output devices, and you have checked through Windows Updates for audio driver updates, and rebooted a couple of times, go into Device Manager, and uninstall the audio device and it's driver, (not disable, but uninstall), and reboot. It will reinstall during the boot if a driver is present.

After the reboot, if there is no audio getting to the output devices, check Windows Update for an updated audio driver. Then, check in Device Manager to see that it shows your audio device as installed and working. If not, check again through Windows Update for a driver update.

In some of the earlier builds, we had to deal differently with getting drivers to install, but that shouldn't be the case for you.

a. user say:

A few facts about System Restore:

1- A restore point is not a backup - it was not designed or intended to be a backup. A restore point is a collection of important system files, system drivers and registry hives that, when restored, will (hopefully) give your computer enough functionality for you to be able to work out of a problem. Only a backup is a backup.

2- You can only restore a restore point that already exists.

3- By design, all restore points are erased when a new feature update, like the October 2018 Update (v.1809,) is installed. If it were possible to restore a restore point from before the update was installed, your computer would be a toxic brew of pre- and post-update files that would be like asking for a crash.

"... how is that supposed to help??"

4- In truth, a restore point is hardly any help at all, which is why System Restore is disabled by default in a clean install of Windows 10. When System Restore was first designed, the type of backup software that exists today wasn't available to consumers, at least not to consumers without a degree in Computer Science.

Today image backup software like Macrium Reflect or Acronis True Image or AOEMI Backupper makes it quick, easy and convenient to backup everyday, which is what we should all be doing anyway. You can make a backup every day, automatically, overnight or during the day while you continue to use your computer.

It takes only minutes to create and to restore an image backup, and when you restore an image your computer goes back to exactly the way it was when the image was made.
