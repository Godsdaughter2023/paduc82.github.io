tammya09 say:

transfer play lists from one computer to another

nsilmike say:
I can't get my play list from my old computer to my new one. I have alot more songs and playlists on the old computer. I went under the account settings and did all I could there. How else can I get all my music and my playlists onto my new computer?
a playlist on the PC is a list of songs, and includes their file location.  If you copy the playlist file, and the music is not where the playlist says it is, you will run into trouble.  But, you can reverse sync them from device to the new PC, after you also get the music there...
I'd suggest you move the music first, then reverse sync playlists.
Copy the music from the old to the new PC by some convenient process. That could be over a home network, by using an external disk drive, burning them to CD and copying to the new PC,etc.  It can even be done by reverse syncing the zune device under certain circumstances...
Zune pass music:  copy or reverse sync to the new PC   Login and try to play a track on the new PC... it may take a few seconds but the DRM (copy protection) will reauthorize the music if you haven't exceeded the 3 PC limit.
Purchased music:  copy MP3 (or any non-DRM'd compatible file) by any convenient means.
Purchased music with DRM (will be wma format.)  If you have any of these, you will have to burn them to an audio (not mp3) CD, and re-rip to the new PC.  Zune went DRM free a couple years back, so you can no longer restore these from the marketplace.
To reverse sync:  connect the device, and in zune software click device.  navigate to the content or playlist and select one or more. drag and drop to the PC icon.  The limitations on this are:
Playlists:  I think autoplaylists will copy back as static or manual playlists only (the 'auto' part is only on the PC.) Static playlists should be fine.
Music:  if you have set the software to transcode music to lower bit rates (smaller files) to fit more on the zune, you can only get that transcoded version back off the zune.

need to know how to do what they say.

  Don't worry about it.  Those tips are obsolete.  The implementation has changed so that in some cases: the module no longer exists, the entry points required to allow the regsvr32 command have been removed, or there are permissions constraints set on the registry keys involved so that even if neither of the first two problems are present the regsvr32 command won't work anyway.  If you want to see a list of modules which you could try doing that to, use DependencyWalker to find which ones still have entry points for DllRegisterServer or DllInstall;  then you could try using the appropriate regsvr32 command syntax accordingly.  In some cases syntax that we might expect to work doesn't and doesn't have any useful effect.  For example,  regsvr32.exe  /i  ieframe.dll  might be expected to work because that module has both entry points in it.   However, only  regsvr32.exe  /n  /i  ieframe.dll  actually does something useful.  Etc.

rohn007 say:
OK, to be completely clear, you had a legal and working copy of Office. Then you installed TRIAL versions of Visio and Project.

Have you bought copies of Visio and Project? 

Yes, Visio and Project 2010 only sold standalone, not as part of any bundle.  BUT, MS insists marketing as part of "Office", what ever that means.  I believe that also means they use the same activation process and system.  When you start "bolting on" stuff (like Visio and Project to "Office"), other stuff can get broken (great technical terminology, right).

Activation is a deep dark black hole that MS just doesn't tell us about.  We see problems like yours here all the time.  "All of a sudden" there is a mysterious problem with activation. Sometimes the cause can be traced back to Windows Updates.

With your system in it's current "fragile" state I agree that simply uninstalling Office is probably not a good idea.

But that is only because of the activation problems you are having. If you currently have a copy of Office purchased from a legal retailer the license terms allow you to re-install it on the same computer even if it is one of the "1-computer" licenses like OEM or PKC.

At this point I see only 2 options. First is to call the MS activation center to see if they can help you with the problem. When you call in, don't answer the initial phone system prompts, wait a long time until you are offered the option to talk to a live service rep.

Other than that the only thing I can suggest is the lame old standard, re-install WINDOWS.  That way you start with a fresh slate.  When you are re-installing the various Office apps, make sure that only one is not activated at a time.

If your Windows re-install is from a refresh partition, then either activate the factory trial of Office, or uninstall it. Reboot the computer, Install your copy of Office, activate it. Then install Visio (or Project) and Activate it. Then finally install the last program and activate it.

rowena cam say:

Hi, PintuNOKIA.

Thank you for posting on Microsoft Community.

We understand that call recording is not showing on settings of your Lumia 950 XL. Please follow the steps below to enable it on your Windows 10 mobile:

How to Enable it on your Windows 10 Mobile:
1. Go to Settings > System > Phone
2. Scroll to the end, and look for option which says “Choose the app you want to use to manage recorded phone calls.”
3. The default is set to none. Tap to select Voice Recorder.

How to use it:
1. Make a voice call, a circular button (Record) will show up.
2. Tap, and it will start the call recording.

Where are the recording saved :
1. Launch File Explorer and look for a folder named as “Recorded Calls”
2. Open, and all the recorded calls will be available against the contact name.

How it the quality of recorded calls :
It is loud, and clear. The files are recorded at 170 kbps.  While the background noise is not completely surpassed, the voices in calls dominate.

Other Features:
You an also take a quick look at recorded call while you are on the dialer. Hit the menu bar > select  Recorded Calls. It lists all the recording chronologically with details on duration of call recording.

1. To view the contact number, and other details,  long press on the recording to get option “Open Contact”.
2. Using the native audio editor (tap on any of the call recording to listen to launch), you can edit, trim, mark important points, and share it over Outlook, bluetooth and other places.

Please let us know if it helps.
