paul domaskis say:

VBA function to extract elements of array

Excel cell formulas allow one to use INDEX(SomeArray....) to extract elements from an array.  That way, you access elements of an array that is generated from evaluating an expression, and you don't need to store the array in a range before accessing it.

 

Is there a similar function in VBA?  I don't want to have to store an array into a variable if it is a temporary result created from evaluating an expression.

chip pearson say:

If you have a function (custom or built in) that returns an array as a result, you can index that result array directly to get an element from the array. For example, the Split function takes a string and splits it into a 0-based array of tokens.

 

Dim S As String
S = Split("a b c d e f", " ")(3)
Debug.Print S

This will display "d" in the Immediate window. There is no need to store the array created by Split into an array variable and then index that variable.

 

I hope this is what you are looking for.
