User Directory "Recent".

The answer was certainly in the Task Bar... Properties... Jump List.

Now I can continue to track all of my activity in the "Recent" directory in File Manager.

Although I don't remember changing this setting, it has been reset and now it works properly.

- Right click on the Task bar and select Properties.
- Click on Jump Lists tab.
- Check the check box against "Store and display recently opened items in the Start menu and the taskbar".
- Set the number (0-60) of recent items and frequent places you want to display up to in Jump Lists and File Explorer, and click on OK.
- Restart the computer and check if the recent option works fine.

This issue can be caused due to incorrect settings or virus. Let’s work together to get this problem resolved.

I would suggest you to try the following methods:

Method 1:

Check the status "Recent Items" and "Frequent Places" in Taskbar Properties.

1. Right click on the Task bar and select Properties.

2. Taskbar properties window opens.

3. Click on Jump Lists tab.

4. Check the check box against "Store and display recently opened items in the Start menu and the taskbar".

5. Set the number (0-60) of recent items and frequent places you want to display up to in Jump Lists and File Explorer, and click on OK.

6. Restart the computer and check if the recent option works fine.

If the issue still persists, then try method 2.

Method 2:

Try to reset the Recent Items folder and check if you can see the recent items:

1. Press Windows key + E key, Windows Explorer window opens.

2. Go to C:\Users\[Username]\AppData\Roaming\Microsoft\Windows folder.

3. Right-click on the Recent folder (if exists) and select Delete. Click Yes when prompted.

4. Press Windows key + R key, type "shell:recent" (without quotation marks) in the open box and press Enter.

Note: It will re-creates the Recent folder, along with a new desktop.ini file and opens the folder.

5. Restart the computer and check if the recent option works fine.

Hope this would help. If issue still persists post back with current status of your computer and result of the proposed suggestion, we will be happy to assist you.

Connect to a computer via USB

Your phone has a mini-USB port so you can connect it to a computer to transfer data, or charge your phone.

On your phone:

  1. Disconnect the USB cable if it is connected to your phone.
  2. Press the Home key.
  3. Open Connectivity.
  4. Open USB Selector.
  5. Connect the cable to your phone and to an available USB port on your computer.
  6. Select a USB personality from the list displayed:
    Modem - Use your phone as a modem for your computer.
    PC Connect - Use your Motorola Original Data Software to transfer data and synchronize contacts, calendar, tasks and email. See the help feature provided with the software for details.
    Mass Storage - Access your memory card data using your computer’s “My Computer” feature.Click Here for more information.

Can I connect my phone to a computer?


You can use your phone’s mini-USB port to connect to a computer.

Note: Motorola Original™ USB data cables and supporting software may be sold separately. Check your computer to determine which cable you need.

Note: Before connecting your phone to your PC download and install the software driver from HERE.

If you use a USB cable connection:

  • Be sure to connect the phone to a high-power USB port on your computer (not a low-power one, such as the USB port on your keyboard or bus-powered USB hub).
    Note: If you connect your phone to a low-power USB port, the computer may not recognise your phone.
  • Make sure both your phone and the computer are turned on.
    Note: When your phone is plugged into a computer and being used for mass storage, it cannot be used for anything else. To restore the phone to normal use, disconnect it from the PC.
