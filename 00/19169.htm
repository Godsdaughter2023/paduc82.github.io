teylyn say:

Hello,

chances are the X Axis label source is not defined correctly.

Open the "Select Data" dialog for the chart and check the range selected in the Horizontal (Category ) Axis Labels setting.  Make sure all desired columns are included.

Or select a data series in the chart and check the formula bar for the series formula

=SERIES(Sheet1!$A$4,Sheet1!$P$1:$S$1,Sheet1!$P$4:$AA$4,3)

The highlighted second parameter contains the range for the X axis labels. See how this one goes from column P to S, whereas the data goes from column P to column AA. Adjust the column S to read AA and the missing labels should appear.

roady [mvp] say:
  1. The AutoSuggest feature doesn't look at your Contacts folder but works from a cache of people that you've emailed before. So when you start with a new account (or add an IMAP account), it is normal that there are no suggestions yet. You can however quickly fill it with your Contacts without send an email. See: Filling the Auto-Complete list with Contact addresses.
     
  2. Don't export to an csv-file when it isn't needed. When these contacts were already stored in Outlook, you can export them to a pst-file and import that (or use the original pst-file and reconnect it). This will maintain more properties, including the Display As field.

That said; You can also address emails by using @mentions. Just type an @ sign into the message body and you'll get a list of names to choose from. This list is looking at your Address Book.
