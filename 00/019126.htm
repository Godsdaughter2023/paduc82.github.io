graham mayor say:

If your printer accepts PCL commands, you may be able to use PRINT fields to force the correct direction.  A PRINT field is simply a type of field that allows you to send instructions directly to the printer.

There are 3 possible data values for duplex setting:  PRINT 27"&l0S" simplex PRINT 27"&l1S" vertical duplex . PRINT 27"&l2S"  horizontal duplex (bookformat)

The codes after PRINT are 27 (escape), double quote, ampersand, lowercase L, 0, 1 or 2, uppercase S and finally terminated with a double quote.

Thus for horizontal duplex you would insert the following field :
{ PRINT 27"&l2S" }

Hi,

 

There are 3 main reasons why you might get a #REF! error.

 

1. You have spelled the workbook name incorrectly or put it in the wrong cell.

2. The workbook doesn't have a worksheet called Sheet1

3. The workbook must be open.

 

One other possibility is that the workbook name isn't exactly as you described it and has a space in the filename, try this alternative

 

=INDIRECT("'[" & A2 &"]Sheet1'!$A$1")
