martarouge say:

Solution found in an other thread of this community.

I report here the solution, it worked with my Mac!

I had the same problem today.  It turned to be a DNS problem.  The server officecdn.microsoft.com was not resolving.  I changed my DNS server entry under TCP/IP settings to point to Google's public servers, 8.8.8.8 and 8.8.4.4 and the download worked right away.

Thank you very much for this answer, which solved the problem for me too. For others, it would be worth explaining that to change the DNS server entry, you click the Apple logo top left, then System Preferences, then Network, then Advanced, then the DNS tab, in which you click the + in order to enter the 8.8.8.8 (which was what I used), and then Apply.  Before doing so, I made a note of the existing entry, which I'm now going to restore.

Best

Marta
