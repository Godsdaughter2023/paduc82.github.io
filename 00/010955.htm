Let's suppose I have a value in A1 of a sheet called MySheetName.

=INDIRECT(C2&"!A1")

Where C2 contains "MySheetName" (without the quotes).

 

So if you're looking up the value of some employee's worksheet, you can just put their name (also the sheet name) in a cell and use indirect.
