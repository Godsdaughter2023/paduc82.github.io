rich michaels - mvp (greatcirclelearning.com) say:

In Mac Word the EditFindDialog function is not by default assigned a keystroke sequence. You need to assign your own using the Customize Keyboard function from the Tools menu.

You can locate the EditFindDialog in the Edit group of commands. On my system, I assigned Cmd + Option + H as the custom keystroke.

Edit of my posting ...

To provide the rationale for not including a default keystroke for opening the EditFindDialog, it is because on a Mac a general ability to Search for characters, words and phrases, is built into the User Interface. It is in the upper right corner of the active window. It performs a "find all" feature. It does not use wildcards but it does have a replace feature. If you need wildcards, the other methods such as the Advanced Find and Replace (accessed from the Edit control on the Menu Bar) is still needed.
