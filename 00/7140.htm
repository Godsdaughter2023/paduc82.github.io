Word calls your "custom purple" pink.

  • Press Alt+F11 to activate the Visual Basic Editor.
  • Click on Normal in the explorer-like pane on the left hand side.
  • Select Insert > Module.
  • Copy the following code into the new module:

Sub FontPink()
    Selection.Font.Color = wdColorPink
End Sub

Sub FontLightBlue()
    Selection.Font.Color = wdColorLightBlue
End Sub

Sub FontRed()
    Selection.Font.Color = wdColorRed
End Sub

  • Switch back to Word.
  • Assign keyboard shortcuts to the macros (File > Options > Customize Ribbon > Customize...)

When you quit Word, it may ask you whether you want to save changes to Normal.dotm. If so, click Yes.

charles kenyon say:
Joe

The thing is, any Add-In has to be automatically loaded. Since you can't save a macro in Normal.dotm and can't put any templates in your Startup folder, you can't have it automatically start through normal processes.

If you can store your template in a macro-trusted location and start word by double-clicking on it, you could have an AutoNew macro load the template as an Add-In.

You can save macros and keyboard shortcuts to a template if it is open for editing (i.e. if you can type in it). You cannot do so if it is only loaded as an Add-In. You can save AutoText to it either way.
