    net user support

    net user support /active:no

Hello

In Skype for Windows 10 (UWP):

  1. Select ...
  2. Under Contact settings, select Manage blocked contacts.
  3. Select the contact you want to unblock.

In the new Skype:

  1. Click your profile and select Application settings.
  2. Under Privacy, select Manage blocked users.

In Skype for Windows desktop (Classic version):

  1. Select Tools, then choose Options...
  2. Under Privacy, select Blocked contacts.
