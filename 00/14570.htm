Manipulating ListTemplates objects often creates additional ListTemplates; this is a known issue, but it shouldn't cause any trouble in newer versions of Word.

You can reduce the risk that you get duplicate ListTemplates if you create a named ListTemplate object and use the name to reference the object:

Sub CreateListStyle()
Dim LT As ListTemplate

Set LT = ActiveDocument.ListTemplates.Add(Name:="MyList")

With LT
'Add commands here for defining numbering
'and assigning paragraph styles to levels

End Sub

End Sub
