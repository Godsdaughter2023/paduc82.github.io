Will Windows 10 expire after one year? - Microsoft Community

andre da costa say:

Will Windows 10 expire after one year?

Is this upgrade for 1 year or forever?

andre da costa say:

Is Windows 10 really free and do I have to pay for it after the first year has passed?

Windows 10 will be free for the first year to all Windows 7, Windows 8.1 and Phone 8.1 users. This means, as long you upgrade within the first 12 months (July 29th 2015 to July 29th 2016), you will not have to pay for Windows 10 at all.

Wow, thats cool, but does that mean, I will have to pay for Windows 10 after the first year?

No. Its a promotion to get as many users to upgrade their existing Windows 7 PCs and Windows 8.1 devices to Windows 10. Once you upgrade within the first year for free, its yours forever at no additional cost. As Microsoft Program Manager for Windows Terry Myerson described it:

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device –at no additional charge.

Source:

I heard Microsoft representatives mention they are moving to a servicing model for Windows 10, does that mean Windows 10 is going to require a subscription?

No, Windows 10 remains a perpetual license, which means, you can upgrade to Windows 10 and use it forever without it expiring or going into any reduced functional mode.

Are you saying there will be no more versions of Windows after Windows 10?

Microsoft is moving Windows to a servicing model with Windows 10. The days of major revisions or what are called big bang releases are no more. As Microsoft Program Manager for Windows Terry Myerson described it:

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device – at no additional charge. With Windows 10, the experience will evolve and get even better over time. We’ll deliver new features when they’re ready, not waiting for the next major release. We think of Windows as a Service – in fact, one could reasonably think of Windows in the next couple of years as one of the largest Internet services on the planet.

And just like any Internet service, the idea of asking "What version are you on?" will cease to make sense – which is great news for our Windows developers.

Source: http://blogs.windows.com/bloggingwindows/2015/01/21/the-next-generation-of-windows-windows-10/

Why is Windows 10 free, is there some catch?

No, its not a catch. We are computing in different times. Windows 10 is free so app developers can standardize on one platform and reduce the fragmented market of different versions and editions of Windows that have been an issue in the past. When everybody is on the same platform, it means better quality apps, a healthier ecosystem and more advances in hardware and software innovation.

Will I have to pay for updates and fixes for Windows 10?
No, updates, fixes, security updates, features all remain free.

nizambakhshi say: Steps to Fix Windows Update Error 0x80070057

Error 0x80070057 indicates that the windows updates have failed. The error is usually followed by a long message which is “There were some problems installing updates, but we’ll try again later. If you keep seeing this and want to search the web or contact support for information, this may help – (0x80070057)” Some users have also reported receiving error 0x80240437 which also has the same fix as below.

In this guide, i will list the steps for you which has worked for 90% of the readers.

Method 1: Restore Windows Update Repository

Download and run Reimage Plus to scan and restore corrupt and missing repositories and then try updating Windows. If it doesn’t work, move to Method 2.

Method 2: Run SFC Scan

Method 3: Run SFCFix

Click here, and run the SFCFix.exe file. Agree to the prompts in the command prompt by pressing any key until the end.

Method 4: Run SURT

If you are running Windows 7, then download SURT and run it. After it has finished installing, try running the windows update again. If it still won’t work, then proceed with the registry steps below.

Method 5: Registry Edit

Hold Windows Key and Press R

Type notepad and Click OK

In notepad, copy and paste the following lines:


Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsUpdate\UX]
“IsConvergedUpdateStackEnabled”=dword:00000000

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsUpdate\UX\Settings]
“UxOption”=dword:00000000

Click File -> Save As

Set file type to All Files and save this file as wufix.reg on your Desktop.

The file must be exactly wufix.reg because the *.reg extension is the trigger to run this file through the registry.

Now run the file, and agree to prompts.

After the program runs check to see if it fixes the issue.

mark st... say:
Hello,

When you have Windows Automatic Repair loop issue please follow these recommendations:

1) Refresh PC

Press-and-hold Power button 3 times to forcefully shut down the machine. The system will enter boot repair page after 2-3 times of reboot, choose Troubleshoot, then go on to Refresh PC.

2) Disable early launch anti-malware protection

You can do it using Command Prompt in Windows Boot Options menu which will appears after your computer restarts several times. Choose Troubleshoot > Advanced Options > Startup Settings. Your computer should now restart and give you a list of options, here choose "Disable early launch anti-malware protection". Restart your PC and check if the issue has been resolved.

3) Delete the problematic file

- In Windows Boot Options menu, choose "Troubleshoot > Advanced Options > Command Prompt".
- In the command line, type: C: cd Windows\System32\LogFiles\Srt. SrtTrail.txt and press Enter button.

Here, if you see something like this: Boot critical file c:\windows\system32\drivers\vsock.sys is corrupt, navigate to the specified location using the Command Prompt and enter Del command to delete the problematic file.

4) Manually repair MBR and rebuild the BCD

In the Windows Boot Options menu, choose Troubleshoot > Advanced Options > Command Prompt. When Command Prompt appears, enter the following lines and press Enter after each line in order to run it:

bootrec.exe /rebuildbcd
bootrec.exe /fixmbr
bootrec.exe /fixboot

5) Perform a system restore or reset

in Windows Boot Options menu, choose Troubleshoot > Advanced Options > System Restore. Follow the Wizard instruction and choose the appropriate restore point. Finally, click Finish to restore the system.

Let me know if you need more help

Why is there no "Recent Items" Jump List in Windows 8? - Microsoft Community

nicolas noakes say:

Why is there no "Recent Items" Jump List in Windows 8?

This thread refers, but it has been locked since it refers to the Consumer Preview of Windows 8: Is there a Windows 8 "Recent Items" Jump List

Dear Microsoft,

Some us us actually use our computers to get stuff done.  You know, like work.  Remember the "I'm a PC, I'm a Mac" days, when Windows could do all sorts of useful things?  Someone in your Windows Team seems to have forgotten this.

Windows vista and Windows 7 had a much-touted feature called "jump lists" - the pop-up menu of relevant documents or actions for an application which could be displayed by right-clicking on the application shortcut.  Or from the little arrow next to application entry on the start menu.  This is why people of upset about the start menu being removed in Windows 8.  Not just because it was a menu (the start screen in Windows 8 is also a menu), but because useful functionality was removed.

I lived with the start screen for a short while on Windows 8, but you can't work with it.  That Excel document you were working on yesterday?  You have three options: (1) Open Excel, click File, click Recent, click the file name, or (2) Open your big folder all recent links (a confusion of programs and documents - my frequently used programs have shortcuts in place already, sirs), and hunt for your file (After setting up this recent documents link as a commonly suggested workaround), or (3) Open windows explorer and navigate for the hundredth time down the folder tree till you get to your file.  In Windows 7, you hover you mouse cursor over the Excel entry in the Start menu, and pick the file of the jump list.  Think of those who work in a corporate environment with the same 5 or 6 files all month long.  That is where your money comes from.  These are the users you need to think about.  Not the pretty-picture-loving blogosphere.

Ever saved a file in the wrong folder and you can't remember where?  Recent documents is by far the quickest way to find it and move it to where it belongs.

I suggest that a right-click (and touch equivalent) option on the start screen should bring up the familiar-to-Windows 8 bar down the right hand side of the screen with a list of recent documents.  Similar to the search bar.  The right-click actions that are currently presented along the bottom of the screen can be then be accessed from a settings "gear" icon at the top/bottom of this recent documents bar.  You know who to give the money to when you implement this.

Thanks for listening.  I trust you are listening.  Remember, it's not the button that people are shouting for, it's not even the menu they think they want, it's the functionality!

In the mean time, I have gladly spent my $4.99 on Start8 so that I can actually get things done.

Regards,
Nicolas
