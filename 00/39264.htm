rscottbrown say:

To turn on, or turn off screen rotation, swipe in from the right, select PC Settings and look for the Screen icon. Tap the Screen icon and you will see a vertical slide control for brightness.  At the top of the brightness control you will see a rectangle, representing a screen, with a curved arrow at the lower right corner. Tap the rectangle once to lock the screen rotation in the current position, portrait or landscape. The curved arrow will change to a lock.  To unlock screen rotation, follow the same procedure, tap the rectangle with the lock and it will change back to the curved arrow. 

No Screen icon when you swipe in from the right?  I had a problem where only the brightness slide control was available.  The Screen icon was missing.  There was no rectangle above the brightness control.  To fix that problem, I went to My Dell on the home screen, then tapped on Drivers and Updates, to see if anything was available.  I found a bios update, installed it, which automatically rebooted my DVP8. and when it restarted, the Screen icon appeared, and when I tapped on it, the rectangle was where it was supposed to be and I could again lock and unlock rotation normally.  

Hope this helps.

Change Wireless Channel

Login Information

Gateway - 192.168.1.254

 Change Wireless Channel

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu on the left select Advanced.
  4. From the new menu on the left select Configuration.
  5. From the sub-menu select LAN.
  6. From that sub-menu select Wireless.
  7. Using the Channel ID dropdown select your desired channel (channels 1,3,9,11)
  8. Select Apply.
