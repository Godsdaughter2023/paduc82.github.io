facetiouszq say: Perhaps not the simplest way of identifying the shapes on any given page, is this macro:
Sub checkShapes()      'This macro MUST be first AND with <Option Explicit> above it
    Dim shp As Shape   'Use this macro in Debug > step into and it will cycle through each shape and give all
                       'details each shape in the Locals window
    For Each shp In ActiveDocument.Pages(1).Shapes    'identifies shapes on Page 1 - Pages(1) change for other pages!
        Debug.Print shp.Name 'The following pieces of info will identify the shapes on page.
        Debug.Print shp.ID
        Debug.Print shp.Top
        Debug.Print shp.Left
    Next shp
End Sub

It can be further refined to the following which will give a printout of all shapes on all pages in a publication:

 Sub checkShapesModified() 'This macro MUST be first AND with <Option Explicit> above it
    Dim pge As Page
    Dim shp As Shape 'Use this macro in Debug > step into and it will cycle through each shape and give all
     'details each shape in the Locals window
    For Each pge In ActiveDocument.Pages
        For Each shp In pge.Shapes
            With shp
                 'The following pieces of info will identify the shapes on page.
                Debug.Print "Page " & pge.PageNumber & ": " & .Name & ", " & .ID & ", " & .Top & ", " & .Left
            End With
        Next shp
    Next pge
End Sub

charleskenyon (ia) say:

Here's the macro: 

Sub FixEndNotes()

' Charles Kenyon

' 7 Feb 2019

' https://answers.microsoft.com/en-us/msoffice/forum/all/endnotes-are-different-font-sizes/de8489c8-3de7-4b1b-ab6c-23a9939a3a91
' Apply style Endnote Text to all Endnotes in a document
'
    Dim iNote As Integer
    Dim rngNote As Range
    For iNote = 1 To ActiveDocument.Endnotes.Count
        Set rngNote = ActiveDocument.Endnotes(iNote).Range
        rngNote.Style = "Endnote Text"
    Next iNote
    Set rngNote = Nothing
End Sub
