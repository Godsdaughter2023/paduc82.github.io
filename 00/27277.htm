john korchok say:

With legacy form fields, the document must be protected for forms before the fields become fillable. A protected form can have within it protected and unprotected sections. The form field go in the protected sections and nothing else is editable there except the fields. In unprotected section, text is editable, but  not all Word formatting commands are available. To edit the form with VBA, one of your first statements must unprotect the form: 

If ActiveDocument.ProtectionType = wdAllowOnlyFormFields Then
  ActiveDocument.Unprotect
End If

  ActiveDocument.Protect Type:=wdAllowOnlyFormFields, noreset:=True
End If

Hi John Wood,

To get the code in your AutoNew and AutoOpen routines to run you have to build in a 1 second delay, this allows time for the Office application on the Mac to fully load. Why?... code execution on a Mac is slower than the PC... at least in the Office applications. Here is what you need:

Sub AutoNew()
    Dim PauseTime, Start
'    Add a pause to allow time for Word to open before continuing
    PauseTime = 1  ' Set duration in seconds
    Start = Timer   ' Set start time.
    Do While Timer < Start + PauseTime
        DoEvents   ' Yield to other processes.
    Loop
    'add your code here
End Sub

jay_answers say:
You can store movies, pictures and music on your SD card. To move stuff from your phone memory to SD card, you will need to connect your phone to your PC and simply cut and paste like you would do with  USB memory stick.

garym123 say:

I have always put the SD card in before I added anything to the phone.

The memory card has to be formated. The phone can do this if the card is seen by the phone.  With the card installed go to settings, phone storage. If the entry for SD card is greyed out you will have to format the card with the PC.

If the entry is not greyed out tap it.  On the nest page will be an option to format the card.

To format it with the PC you need a way to connect it to the PC. Some have a card reader. If not you need a USB card reader. In either event the micro SD card will have to be inserted into a larger adapter to fit in the PC card reader or a USB card reader.

 

Once the card is formated, installed  and the phone turned on you should see it in settings, phone storage. The default settings will be to store pictures and music on the card. You will have a selection to move pictures and music to the SD card.

From then on, all pictures you take and music you transfer will be installed on the memory card.
