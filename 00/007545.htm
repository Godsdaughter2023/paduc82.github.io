donleverton say:

I think I just found a way to get the job done in VBA, but I'm not going to mess with it. ;)

I'll just put it here for anyone else that may stumble upon this thread.


'---------------------------------------------------------------------------------------
' Procedure : cmdAddSheet_Click
' Author    : Don Leverton (with help from Google and others.)
' Date      : 3/22/2015
' Purpose   : This code copies the last sheet in the workbook
'           : and then increments the values from the prior worksheet
'           : this results in a one-click solution to a somewhat tedious task.
'---------------------------------------------------------------------------------------
'
Private Sub cmdAddSheet_Click()

    Dim ws As Worksheet
    Dim lngCnt As Long
    Dim rng1 As Range
    Dim strSht As String
    Dim NewName As String
    Dim varAddress
    Dim OldValue
    Dim varCol
    With ActiveWorkbook
        For lngCnt = .Worksheets.Count To 1 Step -1
            Set rng1 = .Sheets(lngCnt).Cells.Find("*", , xlFormulas)
            If Not rng1 Is Nothing Then
                strSht = .Sheets(lngCnt).Name
                Exit For
            End If
        Next lngCnt
        On Error Resume Next ''Need this to avoid Error 91 on next line down
    ws = ThisWorkbook.Sheets(strSht).Copy(After:=ThisWorkbook.Sheets(ThisWorkbook.Sheets.Count))
    
    If Len(strSht) > 0 Then
       NewName = Left(strSht, 5) & lngCnt 
        .ActiveSheet.Name = NewName
    End If
  
    .ActiveSheet.Range("A2") = NewName
    Dim rng As Range
    Dim row As Range
    Dim cell As Range

    Set rng = ActiveSheet.Range("A5:D44")

For Each row In rng.Rows
    For Each cell In row.Cells
    cell.Activate
    varAddress = ""
    varAddress = cell.Address(RowAbsolute:=False, ColumnAbsolute:=False)

    varCol = Left(varAddress, 1)
    Select Case varCol
        Case Is = "A"
            ActiveCell.Value = ActiveSheet.Previous.Range(varAddress).Value + 40
        Case Is = "B"
            ActiveCell.Value = ActiveSheet.Previous.Range(varAddress).Value + 2000
        Case Is = "D"
            ActiveCell.Value = ActiveSheet.Previous.Range(varAddress).Value + 2000
        Case Else
            'Do Nothing
        End Select

    Next cell
Next row
    
    End With
    
    Set ws = Nothing
    ActiveSheet.Range("A5").Activate 'Returns focus to the top of the sheet.
End Sub

'============================

BTW, this procedure no longer needs to call any function thanks to the following line:

ActiveSheet.Previous.Range(varAddress).Value

joeu2004 say:

You should enter 21m 5s as 0:21:05 or 21:05.0 , formatted as Custom [m]:ss .

The leading "0:" or trailing ".0" causes Excel to correcty interpret that as 21m 5s.

If you enter that into A1, MPH is =1/A1/24 , formatted as General or Number.

-----

If you enter 21:05, Excel interprets that as 21h 5m.

Although that is "bad practice" and it might get you into trouble eventually, if you enter that into A2, MPH is =60/A2/24 , formatted as General or Number.

-----

If A1 contains 1.75 (miles) and B1 contains 0:37:00 (37m 0s), MPH is =A1/B1/24 , formatted as General or Number.

Just to brief about the exact requirement, following are the details:

  • IMEI no. contains multiple parts, out of this one part is the serial no.
  • IMEI no.s are issued by Reporting bodies(like BABT,PTCRB) to the Mobile Device Manufacturers
  • Mobile Device Hardware Manufacturers get each and every handset model registered with Reporting body
  • Generally for one model a range of serial no.s are used

 

Does Nokia provide list of models with IMEI Serial No. Range?

 

So that application developers can create some logic of identify Nokia Handset Model from the IMEI, without going to any 3rd party licensed databases such as NumberingPlans. Generally such databases do not have complete IMEI data also.

Function CountIfsAcross(ByVal Include As String, _
    ByVal Where1 As Range, ByVal Criteria1 As Variant, _
    Optional Where2 As Variant, Optional Criteria2 As Variant, _
    Optional Where3 As Variant, Optional Criteria3 As Variant, _
    Optional Where4 As Variant, Optional Criteria4 As Variant, _
    Optional Where5 As Variant, Optional Criteria5 As Variant) As Variant
  'Same as COUNTIFS but across all sheets
  '  Include is a search mask to match the sheet names, support wildcards (see LIKE operator)
  '  Excludes the caller sheet automatically
  '  Example:
  '    =CountIfsAcross("*",Sheet2!A1:A10,"x",Sheet2!B1:B10,"y")
  '  Note:
  '    This function is volatile and is called on every calculation in the file
  '    But as not all cells are referred on the call the results may not always correct!
  '    You have to recalculate the file manually, see
  '    https://msdn.microsoft.com/de-de/library/office/bb687891.aspx
  Dim OurWs As Worksheet, Ws As Worksheet
  'Include all if no search mask is given
  If Include = "" Then Include = "*"
  With Application
    'Get the sheet we are called form
    Set OurWs = .Caller.Parent
    'This function must be called on every calculation elsewhere!
    .Volatile
  End With
  'Visit each sheet
  For Each Ws In Worksheets
    With Ws
      'Exclude our sheet always
      If (.Name Like Include) And (.Name <> OurWs.Name) Then
        'Call COUTIFS based on the given arguments
        If Not IsMissing(Where5) Then
          CountIfsAcross = CountIfsAcross + _
            WorksheetFunction.CountIfs( _
            .Range(Where1.Address(0, 0)), Criteria1, _
            .Range(Where2.Address(0, 0)), Criteria2, _
            .Range(Where3.Address(0, 0)), Criteria3, _
            .Range(Where4.Address(0, 0)), Criteria4, _
            .Range(Where5.Address(0, 0)), Criteria5)
        ElseIf Not IsMissing(Where4) Then
          CountIfsAcross = CountIfsAcross + _
            WorksheetFunction.CountIfs( _
            .Range(Where1.Address(0, 0)), Criteria1, _
            .Range(Where2.Address(0, 0)), Criteria2, _
            .Range(Where3.Address(0, 0)), Criteria3, _
            .Range(Where4.Address(0, 0)), Criteria4)
        ElseIf Not IsMissing(Where3) Then
          CountIfsAcross = CountIfsAcross + _
            WorksheetFunction.CountIfs( _
            .Range(Where1.Address(0, 0)), Criteria1, _
            .Range(Where2.Address(0, 0)), Criteria2, _
            .Range(Where3.Address(0, 0)), Criteria3)
        ElseIf Not IsMissing(Where2) Then
          CountIfsAcross = CountIfsAcross + _
            WorksheetFunction.CountIfs( _
            .Range(Where1.Address(0, 0)), Criteria1, _
            .Range(Where2.Address(0, 0)), Criteria2)
        Else
          CountIfsAcross = CountIfsAcross + _
            WorksheetFunction.CountIfs( _
            .Range(Where1.Address(0, 0)), Criteria1)
        End If
      End If
    End With
  Next
End Function
