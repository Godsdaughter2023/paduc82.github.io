     - reboot your machine

ken sheridan say:

Try this:

    Dim lngMyNumber As Long
    Dim strMessage As String

    lngMyNumber = [txt_MyD]
    strMessage = "No match in subform for MyID value " & lngMyNumber

    DoCmd.OpenForm "frm2"

    With Forms("frm2").frm2a.Form
        .RecordsetClone.FindFirst "MyID = " & lngMyNumber
        If Not .RecordsetClone.NoMatch Then
            .Bookmark = .RecordsetClone.Bookmark
        Else
            MsgBox strMessage, vbInformation, "Warning"
        End If
    End With

Note that 'frm2a' must be the name of the subform control in the parent form's Controls collection.  It might or might not be the same as the control's source form object.

teylyn say:

If the pictures are held in Excel cells you can use a combination of named ranges and a dynamic image to show the image with an IF condition.

In the following screenshot the images are placed in identical sized cells A1 and B1. 

Next, create the IF formula in cell C5: 

=IF(A3="show yellow","yellow","red")

Now create three named formulas with the name manager

Picture =INDIRECT(Sheet3!$C$5)
red =Sheet3!$B$1
yellow =Sheet3!$A$1

Copy cell A1 as a picture (Copy dropdown > copy as picture> accept the defaults) and paste it somewhere. Select the pasted picture, click the formula bar and enter =Picture

The pasted image will now show what the named formula "Picture" resolved to, i.e. the text showing in cell C5, but turned into a cell reference. If C5 shows "red" then the reference to B1 is returned. If the cell shows "yellow" then the reference to A1 is returned. The image will change to reflect the respective cell.
