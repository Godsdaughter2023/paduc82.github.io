Hello,

try along these lines in B1 and copy down

=IFERROR(INDEX(A:A,SMALL(IF(ISODD(ROW($A$1:$A$15)),ROW($A$1:$A$15)),ROW(A1))),"")

This is an array formula and must be confirmed with CTRL + SHIFT + Enter. Don't use whole column references for the ROW functions. It will slow the formula down too much.

stefan blom say:

You can't use the Paragraph dialog box to change the indentation for a numbered item. Instead, click to select the actual number; then right-click and choose Adjust List Indents. Use the Adjust List Indents dialog box to define the indentation you want. When you are done, click OK and then update the paragraph style to match the selection.

"Number position" determines the position of the number symbol. "Add tab stop at" determines where the first line of text (following the number) starts. "Text indent" defines the indent for the second and subsequent text lines.

All measurements are from the left margin, which means that you should specify the same value for "Text indent" and "Add tab stop at" if you want a hanging indent. (The value needs to be larger than "Number position," of course.)
