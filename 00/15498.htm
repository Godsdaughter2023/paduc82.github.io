Assuming you have the latest version of Pages you can try each of these methods in the order listed until you find one that works;

  1. Export using the .docx format
  2. Export using .doc format
  3. Export using RTF format
  4. Export using Plain Text format
  5. Copy the entire document & paste into a new Word document
  6. Copy the entire document & paste into TextEdit, then Save As using either;
    1. Word 2007 (.docx) format
    2. Word 97 (.doc) format
    3. Rich Text Format (RTF)
    4. Word 2003 (.xml) format
    5. Web Page (.html) format

With many of those options you'll probably have a fair amount of reformatting to do but at least you should be able to recover [most of] the content.
