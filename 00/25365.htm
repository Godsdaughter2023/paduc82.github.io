Excel filter two columns at once - Microsoft Community

servo75 say:

Excel filter two columns at once

Is there a way in Excel to filter two columns with an "OR" condition? For example in the table below, I would like to filter all rows that have "CLG" in either the "AWAY" or "HOME" column.

andreas killer say:
That is only possible with VBA, but very simple.

The Autofilter filters the helper column for > 0, so if the search criteria is changed, we just have to reapply the Autofilter (you can also press that button in the Data tab).

Right-click on the sheet tab
Choose "View Code"
Within the menu click Insert \ Module
Paste this code 

Private Sub Worksheet_Change(ByVal Target As Range)
  Select Case Target.Address(0, 0)
    Case "C1"
      Me.AutoFilter.ApplyFilter
  End Select
End Sub


Customize the address "C1" of the cell that contains your search criteria.

This scenario may occur if there were issues with the original installation or Office 365. In order to resolve this issue, I suggest we will have to completely uninstall Office 365 and reinstall it.

Let us follow these steps to uninstall and reinstall Office 365:

If you are using a wireless connection then you may try to hardwire the connection. Once this is done continue with these steps.

  • We shall remove the remnants of the previous installation from the computer. In order to uninstall Office remnants from your computer, you will have to install and run the Office 365/2013 FIXIT tool.
  • Download, install and run the Office 365/2013 FIXIT tool from the following link: http://support.microsoft.com/kb/2739501
  • Once you have completed running the FIXIT make sure to restart your computer.
  • Now try logging into the My Account page and install Office 365. The URL for the My Account page is http://office.com/myaccount. Sign-in to the My Account page with your Microsoft Account Credentials.

    Check if you were able to install Office 365 successfully. After completing installation check if Office 365 prompts you for activation. If yes, proceed with the steps on screen and activate Office 365 – you may activate by signing in with your Microsoft Account credentials or your Product Key. Restart the computer once again after the above steps are completed. Check if you are able to launch and work with the Office applications normally after every reboot.

    Hope the above steps help. Let us know the result.

    john mcghie say:
    Hi Laurie:

    I would also be looking for hidden tracked changes.

    Have a look at the Word help topics under "Use tracked changes" for information on how to find these.

    If you have deleted eight pages, but not resolved the tracked changes, the deleted pages will still be counted until you resolve the changes (accept or reject).

    Hope this helps

    rich michaels - mvp (greatcirclelearning.com) say:
    It sounds like you might have multiple sections in the document and not all of the sections have page numbering set.

    From the first page of your document open the Footer (I assume that's where your page number is) then using the Formatting Palette pane from the Toolbox, navigate thru each section and make sure there is a page number field in all sections. 

    If you are using the different first page or the different odd and even page setting, then you need to check those footers as well, which might require that you enter manual page breaks into the document section so that you can view all of the footers for odd and even pages.

    If you are very unfamiliar with headers and footers, I suggest that you use Word's Help function and review the instruction sets that cover Headers & Footers.

    Hope this helps
