Try this formula:

a. user say:

How do I set up a Photo folder and Move different files   JPG into that folder.

Seems pretty simple but I have changed from windows 7 to windows 10 and my Photo files are a mess ---need Help   Barry

To create a new folder Open File Explorer by Holding the win logo key down and hit the E key.

You should see something like this. Then you can navigate to where you want to make a new folder and name it what you want by clicking on one of the New folder icons.

Once you have made the Folder you want to move or copy your photos to click on the new folder that you have just named.

Before you continue make sure the File Explorer window is sized so you can have more than one window open at the same time. Then Go ahead and Hold the win Logo key down and hit the E key again to Open another File Explorer window.

You may have to drag the windows around to make it easier to see both windows.

Now you should be able to have the new folder you made and a folder where you want to get photos from open at the same time.

If you are wanting to move the photos you can select the ones you want to move by holding  the Ctrl key down while you click on the photos you want to move.

Then let up on the Ctrl key and move the mouse pointer over one of the photos you have highlighted .

Then hold down on the left mouse button and drag them to the new empty folder you have made earlier and let off the mouse key.

When you move the photos it does just what it sounds like. They will no longer be in the original folder.

If you just want to copy the photos and paste them in the new folder instead of dragging them right click one of the photos you have highlighted and select Copy.

Then move your mouse pointer to the new folder right click and select Paste.
