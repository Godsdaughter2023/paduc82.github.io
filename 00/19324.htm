'Save As Type' Options in MS Word 2007 - Microsoft Community

shopper1 say:

'Save As Type' Options in MS Word 2007

 

Hello.

What  is the difference between the two types? It seems a bit confusing since Word 2007 'opens' both types of files.

graham mayor say:
The internal structure of the two files is entirely different. DOCX is a zip file that contains XML elements that make up the document.  DOC is a Microsoft proprietary format.

There are still a great many people out in the real world who use older versions of Word on older operating systems - such as Word 97 - which cannot accommodate the new format.

If you wished to exchange documents with such users, you would use the older DOC format. While Word versions have backward compatibility for older file formats older versions of Word do not have forward compatibility with formats that had not been invented when the word processor was new to the market.

For your own documents, exchanged with no-one, use the newer DOCX format.

suzanne s. barnhill say:

Everything you have stated is true. The following are also true:

  • If a document is saved in Word 97-2003 (.doc) format, you will not be able to use features introduced in Word 2007.
  • There are some bugs in Word 2007 for which the only workaround is to save in .doc format (these mostly have to do with printing problems).
  • Not an issue in Word 2007, but Word 2010 uses a new type of WordArt that some users don't like as well as the old style; or they may need to replicate WordArt objects created in older documents. The only way to do this is to save the document in Word 97-2003 format, which makes the old-style WordArt available. I think the same is also true of some other drawing/art features in the newer versions.

DoCmd.OpenForm FormName:="fdlgCreateOrderSplit", WindowMode:=acDialog 

There is no need to open it modal.

Amy

Use the OpeningArgs property separating the two values with a pipe.

From the first form:

      Dim strSQL As String
      strSQL= "SELECT SisterShowID, SisterShowName, SplitPercent " _

Private Sub Form_Load()
    Dim aryOpenArgs As Variant
