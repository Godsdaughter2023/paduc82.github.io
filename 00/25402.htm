If you look closely you'll see a thin vertical line that separates the Current Slide pane from the Next Slide pane. The line can be dragged l/r which will change the size of the Current Slide but not that of the Next Slide.

blacklab22 say: As you can see from the current MS announcement at top of the forum pages and the flood of other reports of "lost contacts" there is a obviously still a widespread problem with opening the People page/Contacts list from the Outlook PC site. 

It would be a good idea if everyone still experiencing this problem kept reporting it on MS' Service status page. If enough users do this the issue will be further escalated : https://status.live.com/detail/people

Below are some ways you might tackle the problem until MS get a proper fix in place:

People page won't open... People page just "hangs" forever... can't access my Contacts list... address box does not populate when typing a new e-mail... etc.  This all seems to be the same very common current problem (bug?) with the failure of MS OUTLOOK's contact database to load correctly on the PC site.  Most browsers seem to be pretty equally affected (perhaps Chrome less so - if you have it - try it).

The forum moderators variously suggest clearing your browser's Cache and Cookies, Updating to the latest version and/or Resetting your browser, etc.  All reasonable ideas and if you look through recent forum topics you will find these suggestions laid out in full.  Sadly not that many successful outcomes - but then you might just be lucky?

These are 4 Workarounds being suggested, one or more of them may work for you:

1.  You may be able to at least access your Contacts list by using OUTLOOK's Mobile site login page (https://profile.live.com/home/contacts?bfv=um).  Best to try this when you are fully logged out of OUTLOOK.  Sadly this is of limited use as the Mobile page "footer" link back to the PC site People page will just hang forever as well.  However it will prove your Contacts list still exists and in extremis you could copy/paste or write out e-mail addresses!

2.  Try disabling the Adobe Flash plugin in whatever browser you are using and then just use the reload/refresh button and hopefully the "hung forever" PEOPLE page will then complete loading normally.  It's a pain to have to keep disabling and re-enabling Flash but this Workaround may allow you to use your Contacts list almost normally again.

If not sure how to disable/re-enable the Flash plugin a search engine will quickly find the instructions.

(More information in forum topics: People page hangs on PC site [WORKAROUND FOUND] & Address Book inaccessible in OUTLOOK.COM Hotmail topics.  Thanks to CWH803 for finding this Workaround.)

3.  Try a different browser - Chrome seems to be the least affected, others have had success trying Firefox, and some have succeeded by resetting whatever browser they were currently using.

4.  If you have access to other machines try them... especially if you have an older machine still running XP/SP3 - several users report that their old XP PC is unaffected by this problem.

Hope the above might help - you are not alone - there's a shed load of similar topics, information and other frustrated OUTLOOK users on the forum.  All in all it's a right pigs ear and hardly MS' finest hour?
