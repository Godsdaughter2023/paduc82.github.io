I own an E73, likely with the same Log app, and have noticed the same thing.  To some degree I think it's just buggy.  But I believe the logic works like this:

 

1) If you receive a call from the same person, make an outbound call to the same person, etc. the previous entry is updated with the new date/time stamp.  You will never see the same number listed twice in the same log.

2) Each log is limited to 20 unique entries.  This is backed up by posts like /t5/Eseries-and-Communicators/E90-call-log-limit-to-20-numbers/td-p/806165 and articles like http://www.gsmarena.com/nokia_e72-review-422p4.php .
