You can follow the steps below to get a specific report for users with certain license.

Get-MsolUser -all | Where-Object { $_.Licenses.AccountSKUID -eq "<AccountSkuId name>" } | Export-Csv < filename >

You can also try the following scripts to export specific fields of information for users with certain licenses :

$userlicense = @()
Get-MsolUser -All | ? {$_.Licenses.AccountSKUID -eq "<AccountSkuId name>" } | % {
foreach ($a in $_.Licenses.AccountSKUID) {
$data = New-Object PsObject -Property @{
UPN=$_.UserPrincipalName;
DN=$_.DisplayName;
LicenseName=$a;
}
$userlicense += $data
}
}
$userlicense | Export-csv <filename>
