    Wait until you receive the response confirming the operation has been completed.

  3. Restart Windows.

  4. When Windows restarts, click the Start Button and look in the All Apps section for the green tile with the title Store, and launch it.
    Do not use the grey tile with the title Store (Beta) for this.

If you can put up with using a helper column (which you can hide), and assuming your customer ID is in column B, use a formula in (say) column F along these lines:

=IF(B2=B1,F1,1-F1)

You set F1 to either 1 or zero (which determines which colour goes first in your pattern). The rest of the column will have an alternating pattern of 1s and 0s for each change in customer code

=$F2=1

I usually find it easiest to apply the conditional format to one cell such as A2 first, then change the "applies to" to include the whole row, then the whole data area, checking at each stage that it is working as expected.

shane devenshire say:

The answer to your final question is Yes.

Let assume for this relatively simple solution that the must be a customer ID, no blanks and that they start in B2:

1. Highlight all the cells on the rows you want formatted  A2:H50 or however many rows and columns you want formatted.
2. Choose Home, Conditional Formatting, Manage Rules, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following
formula:

 =MOD(SUM(1/COUNTIF($B$2:$B2,$B$2:$B2)),2)=0

5. Click the Format button and choose a format.
6. Click OK twice
7. New Rule
8. Choose Use a formula to determine which cell to format
9. In the Format values where this formula is true enter the following
formula:

=MOD(SUM(1/COUNTIF($B$2:$B2,$B$2:$B2)),2)

10. Click the Format button and choose a format.
11. Click OK tree times.

Be very carefull with the $ signs!

Try this...

Select the *entire* range A2:D9 starting from cell A2. Cell A2 will be the
active cell. The active cell is the one cell in the selected range that is
not shaded. The formula will be relative to the active cell.

Goto Home tab>Styles>Conditional Formatting>Manage rules>New rule>Use a
formula to determine which cells to format
Enter this formula in the box below:
=MOD(SUMPRODUCT(--($B$1:$B1<>$B$2:$B2)),2)
Click the Format button
Select the Fill tab
Select a shade of green
OK out

Note that this will not work correctly if there are blank cells within the range of column B2:Bn.

The format must not start on row 1. There must be at least one row above where the formatting starts. In your case this will not be an issue since you have a header row with the data rows below. The column header in B1 must not be the same as the entry in cell B2.
