stefan blom say:

If you create a new, blank document (Ctrl+N), is the language correctly set to UK English when you start typing? If so, that tells us that you have successfully set the default proofing language.

To prevent Word from changing the language at the insertion point, turn off the option to "Detect language automatically" in the Language dialog box (Review > Language > Set Proofing Language).

After you have done that, you can deal with existing documents. Often, you will have to (re-)apply the correct language by selecting the whole content (Ctrl+A) and then choose English in the list of languages at Review > Language > Set Proofing Language. (Note that you shall not click the Set As Default option in this case.)

That takes care of the language formatting. However, incorrect quotation marks already inserted in a document won't change. You should be able to make use of Find & Replace to fix them.
