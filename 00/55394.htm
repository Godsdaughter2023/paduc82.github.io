How do I reset Windows Update components?

http://support.microsoft.com/kb/971058

Now install the updates from through Windows Update and check whether you are able to install it on the system.

The error VIDEO_SCHEDULER_INTERNAL_ERROR bug check has a value of 0x00000119. This indicates that the video scheduler has detected a fatal violation. 

Before we proceed, I would like to know if you have made any changes to the PC prior to this issue.

To begin with, I suggest that you place your PC in Safe Mode with Networking and check if the above issue persist when you are in this mode. https://support.microsoft.com/en-us/help/12376/windows-10-start-your-pc-in-safe-mode
