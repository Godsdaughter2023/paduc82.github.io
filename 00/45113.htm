If no new devices are found even after enabling ‘Show disabled devices’ from the recording tab, it is likely no device is being found because it's not detected by Windows.

I suggest that you reinstall the Audio Drivers on your computer, and check if you have the same issue:

Uninstall a Device:

To uninstall a device with Device Manager:

1. Launch Device Manager.
2. In Device Manager's left panel, open the node that represents the type of device that you want to uninstall (Sound, Audio and Game controllers) and then right-click the device entry.

3. On the shortcut menu, click Uninstall.
4. On the Confirm Device Removal dialog box, click OK to start the uninstall process.
5. When the uninstall process is complete, unplug the device.

Note: Make sure that you download the latest drivers from the computer manufacturers’ website before uninstalling the driver.

Also create a System Restore point on your computer, so that you can roll back if this does not work.
