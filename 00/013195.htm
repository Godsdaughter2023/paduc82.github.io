jasles say:

The only way to change case in Excel is by using the built-in functions.

That being said, you could create a macro to run the function on the active cell, and then add a button to a custom tab on the ribbon or add a button to the QAT to run that macro.

gord dibben say:

One method for a single cell is to select the cell then select the formula in the Formula Bar then hit F4 to cycle through the Case options.

Following along with Jason's macro suggestion here are some routines.

Sub Upper()
Dim Cell As Range
Application.ScreenUpdating = False
        For Each Cell In Selection
             Cell.Formula = UCase(Cell.Formula)
    Next
Application.ScreenUpdating = True
End Sub


Sub Lower()
 Dim Cell As Range
 Application.ScreenUpdating = False
        For Each Cell In Selection
             Cell.Formula = LCase(Cell.Formula)
    Next
    Application.ScreenUpdating = True
End Sub


Sub Proper()
Dim Cell As Range
Application.ScreenUpdating = False
        For Each Cell In Selection
             Cell.Formula = Application.Proper(Cell.Formula)
    Next
Application.ScreenUpdating = True
End Sub
