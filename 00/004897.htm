suzanne s. barnhill say:

If you view the horizontal ruler in the relevant TOC style, you'll find that there is a hanging indent and/or a tab stop at the position where your page number appears in the second entry. Because the entry is too short to extend beyond the tab stop, the tab following the entry is stopping there instead of at the tab stop intended for page numbers. john visio mvp say:

There should not be a limit to the number of times a shapes has an operation (Union, Join...) preformed on it.

The basic part of a shape is the Geometry section and when an operation is performed on several shapes, a new shape is created with a new Geomerty section or several Geomerty sections. Whether you can fill the shap depends on whether the Geometry sections of a shape have the NoFill section set to true or false. It is possible tat a shape with several Geomerty sections has different values for their NoFill cell. So for your shape, you will need to open the ShapeSheet for that shape and look at the Geometry sections and the values of the NoFill cell for each Geomerty section.

The NoFill cell can contain a formula and this is how several shapes can have protions of the shape altered when a custom property/shape data is changed. There is a flowchart shape that contains all th various flowchart shapes as different geometries each with a NoFill cell that says, "is it not this flowchart shape?"
