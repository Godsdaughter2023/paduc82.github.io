@Rob

Windows Live Mail will use a Windows function to invoke the default browser to open a link to a web page if no browser is running. You cannot set IE9 64-bit as the default browser, so what you're trying to do can't be done. However, there is no such thing as a '32 bit webpage'.

http://answers.microsoft.com/en-us/ie/forum/ie9-windows_7?tab=all

Get IE working properly first, and you'll find that the problem is solved in Windows Live Mail.
