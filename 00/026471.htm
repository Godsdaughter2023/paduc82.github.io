How to Categorize 2016 Outlook Calendar Items

I've spent HOURS on this tonight and I've found an easy work-around for categorizing calendar appointments. I had tried to add the Categories to the calendar ribbon but it only shows up as grayed-out and inoperative, so here's what I stumbled on instead:

1. From calendar, click New Appointment

3. Add the subject and time etc. to your appt.

3. Save and close. It should appear on your calendar with the category color. You can change it by opening the appt, since you added a category when you originally created it.

Good luck!

2.     Press Ctrl+A to select all the emails.

3.     Drag and drop the email messages in each folders from the new renamed account to preferred folder on your other account (*** Email address is removed for privacy ***).

3. A pop-up message will appear if you want to save or open WLMContacts.csv, select Save. (Note: The WLMContacts.csv file will be saved in your computer Downloads folder.)

2. Click Import from file in People page.

3. In Step 2, choose Microsoft Outlook (CSV).

4. In Step 3, click browse and choose the file you want to import.

5. Click Import contacts.

2. Click Add people.

3. Click Import from file.

4. In Step 2, choose Microsoft Outlook (CSV).

6. Click Import contacts.
