To fix your issue, here's the step-by-step procedure on how to turn on the Pop-up Blocker on Internet Explorer 11.

To turn Pop-up Blocker on:

1. Open Internet Explorer, select the Tools button, and then select Internet options.
2. On the Privacy tab, under Pop-up Blocker, select the Turn on Pop-up Blocker check box, and then select OK.


Block all pop-ups:

1. Open Internet Explorer, select the Tools button , and then select Internet options.
2. On the Privacy tab, under Pop-up Blocker, select Settings.
3. In the Pop-up Blocker settings dialog box, under Blocking level, set the blocking level to High: Block all pop-ups (Ctrl + Alt to override).
4. Select Close, and then select OK.

Turn off notifications when pop-ups are blocked:

1. Open Internet Explorer, select the Tools button , and then select Internet options.
2. On the Privacy tab, under Pop-up Blocker, select Settings.
3. In the Pop-up Blocker settings dialog box, clear the Show Notification bar when a pop-up is blocked check box.
4. Select Close, and then select OK.

You can't move it, but a moderator can.

The error messages you posted are coming from a mail server.  The first one may indicate that one of the recipient addresses you specified was incorrect.  However, it could also mean what the warning message means: that the mail router that is trying to forward your message cannot reach the mail router one step closer to your recipient.  That could be the recipient's mail server or a mail router between you and that person.  Make sure the mail addresses to which you sent your message are all exactly correct, but chances are there is nothing you can do because the error is occurring after the message has left your PC.

Set a word document to a certain number of lines per page - Microsoft Community

pietersurya say:

Set a word document to a certain number of lines per page

I want to set an essay to be exact 45 lines per page. I'm using Word 2016. How to set the essay to be exact 45 lines per page and the number of lines are showing on the side? 

graham mayor say:

Word is not a page layout application. There are no 'pages' (or 'lines' for that matter) in a Word document. The displayed pages are simply a result of text flowed between margins.

The number of 'lines' on the 'page' is determined by the top and bottom margins, the line spacing and the font size. Then there is the issue of paragraph spacing. If you use spacing between paragraphs then the number of 'lines' on the page will be affected by the number of paragraphs on the page.

You could use an empty paragraph between each paragraph, but it is bad practice and makes editing more difficult, but it will overcome the spacing issue.

Line numbering is applied from the Layout tab. Apply some dummy text then move the bottom margin or adjust the font size until you get the required number of displayed lines per page.

suzanne s. barnhill say:
In all versions of Word, page numbers in a TOC are hyperlinked. In Word 2000
and above, the entire TOC entry is hyperlinked by default, though you can
choose not to have the entire entry hyperlinked (the page numbers will still
be hyperlinked).
 
In all versions since (I think) Word 2002, you must press Ctrl while
clicking in order to follow hyperlinks of any kind. So when you mouse over a
TOC entry, you will not see the usual hand icon, but, provided you have
ScreenTips enabled, you should see a popup that says "CTRL + Click to follow
link."
 
If you want to be able to click without pressing Ctrl, you can change the
setting for this at File | Options | Advanced: Editing options. You will
then see the usual hand icon when mousing over a hyperlink, including
hyperlinked TOC entries.
