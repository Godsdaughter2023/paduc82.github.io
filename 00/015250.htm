stevestroud say:

Does anyone know how to open multiple hytper links in excel at once?

I'm specifically trying to open them up in tabs. But I don't specifically know where to ask this question so I decided to ask it here.

james yan msft say:

Hi Steve,

We can achieve this by using VBA:

1. Right click the sheet name > View Code.

2. Paste the following code:

Sub OpenHyperLinks()
    Dim xHyperlink As Hyperlink
    Dim WorkRng As Range
    On Error Resume Next
    xTitleId = "OpenHyperlinksInExcel"
    Set WorkRng = Application.Selection
    Set WorkRng = Application.InputBox("Range", xTitleId, WorkRng.Address, Type:=8)
    For Each xHyperlink In WorkRng.Hyperlinks
        xHyperlink.Follow
    Next
End Sub

3. Click the Save button, save it as .xlsm file.

4. Back to the sheet, select the cells which contain the hyperlinks (you may need to press the Shift or Ctrl key when selecting the cells), right click the sheet name > View Code > press F5, click OK.

It isn't a bug because the commands you've added aren't intended to work in .docx files :-) The ones you added are the ones that appear on the Ribbon if you're working in file a different - but similar - button appears on the Ribbon. j.e. mcgimpsey say: VBA macros (the successor to XL4M) are stored in modules that you can view in the Visual Basic Editor (opt-F11, or on your MBP fn-opt-F11). You can also access the VBE via the developer tab on the ribbon (same tab the record macro button is on).

In VBE, you should see a small "Project" window. There lists all the opened workbooks. If a workbook contains macro, it should have "Modules". Modules are where you can find your macros.
