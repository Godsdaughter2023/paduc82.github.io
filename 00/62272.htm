I'd download the ISO and/or create the latest bootable media by choosing the second option (shown below) from  Media Creation Tool so you have some control over it's download and can observe it, to know that it's still progressing.  Otherwise it will store it in a hidden AppData file where you can't see it.

This is a more stable way to run the upgrade to new version.  You should also use the latest bootable media if you Clean Install Windows 10
