So did the rebooting issue solve the problem or was that a separate issue?

The first thing that I'd try is doing a two-button reset.  With the computer off or on (just with no apps running), press and hold the power button and the volume up button at the same time.  Continue pressing and holding them for at least a full 30 seconds.  The computer should boot to the EFI screen and then eventually shutdown.  This is supposed to reset all of the peripheral devices.  Try it again to see if it made any difference.

If it still doesn't work, go into Device Manager and look for Other Devices and see what's listed there.  They should have yellow exclamation points next to them and if any have something to do with WiFi, you'll need to reinstall drivers for them.

If nothing shows up in device manager, that means that the computer cannot see the WiFi network card and it is likely dead or perhaps even missing from your Surface Book.  You would have to arrange for a repair or exchange in this case.

Just because they are BC with the 360, doesn't mean they will someday become BC with the Xbox One.  The Xbox One BC does Backwards compatibility differently to the way the 360 does it.

The Xbox 360 uses a more traditional emulator, being able to run games directly from the discs.  While the Xbox One requires a dedicated team to reverse engineer each game and repackage it for the Xbox One platform.  Only EA can release the source code to Microsoft to do the BC on these games.  Without the rights, EA won't be able to release the code.

It's not up to MS what game/s become BC, it's the developers/publishers.  Posting a request on here won't get you anywhere, no matter how many replies you get.
