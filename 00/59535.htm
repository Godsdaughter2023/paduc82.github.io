a blue illusion say:

As Guerrero pointed out... it seems like the issue happens "over time" when using instant on. But it doesn't happen for everyone, apparently and it doesn't happen in the same time period for everyone it does happen to.

The thing is, leaving any networked device turned on can eventually cause problems with the connection. Despite the number of people (even some service techs) saying that it doesn't hurt anything to leave it turned on, it can over time. It's just the nature of the beast in many cases. Some devices need to refresh their settings at least on a semi-regular basis... hence why rebooting works, since it forces the device(s) to terminate the connection upon shutdown and refresh it when powering back up.

That being said, I wouldn't give up on using instant on if you really want to. But you may need to settle with the fact that you'll need to either restart your Xbox and/or your modem/router on a regular basis... or be prepared to start doing... a lot... of playing around with network settings, phone calls to your ISP support, etc. to track down what's happening and even then it might be with limited success.

Personally, I found instant on to be more trouble than it's worth. Not because of network issues, just other things about it I didn't like. I don't use it. I also rarely to never have connection issues, slow or laggy UI problems, or a host of other buggy or glitchy type things that some people have mentioned.

a) Press “Windows Logo” + “W” keys from the keyboard.

a) Go to the Start screen.

b) Type “Services.msc” without quotes and press Enter.

Step 2: Check the Windows Updates for any latest driver updates.

a) Press “Windows Logo” + “W” keys from the keyboard.

b) Type “Updates” in the search bar and click on the icon “Windows Update” from the search result.

c) On the “Windows Update” window, click on the link: “Check for updates” from the left hand side of the window to receive a fresh list of updates.

d) If the updates are available, then install all the updates listed there.

davem121 say:
Hi Thirst4Chris4, Trim is by defult turned on in Windows 10 for SSD's


Check if Trim is Enabled:

fsutil behavior query DisableDeleteNotify


Turn Trim On:

fsutil behavior set DisableDeleteNotify 0


Check if Trim is Enabled again:

fsutil behavior query DisableDeleteNotify


If that command returns DisableDeleteNotify = 0 then Trim is turned on (it is counter intuitive, 0 means trim is on and 1 means it is disabled) . . .
