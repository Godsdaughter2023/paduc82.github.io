Auto Sort spreadsheet when new row added

Hello

Column G is a "follow up date".

The sheet is named "Ongoing".

Thanks in advance.

jllatham say:

Try this code.  It is worksheet event code - you put it into the workbook by choosing the 'Ongoing' sheet and right-clicking its name tab and choosing [View Code] paste the code into that module.  Remember that there can only be one _Change() event per sheet.  Be sure to save the workbook as a .xlsm or .xlsb (macro enabled) file after adding the code.

This code checks to see if a change was made to the last row on the sheet below the title row (assumes titles in row 1) and that cells A:I on that sheet have entries in all of them.

Private Sub Worksheet_Change(ByVal Target As Range)
  Const keyCol = "G"
  Const firstCol = "A"
  Const lastCol = "I"
  Const firstDataRow = 2
  Const fullCount = 9 ' number of cells that must be filled
 
  Dim sortRange As Range
  Dim sortKey As Range
  'are we working on the last row on the sheet, and
  'also not in the titles row?
  If Target.Row <> Range(keyCol & Rows.Count).End(xlUp).Row _
   Or Target.Row < firstDataRow Then
    Exit Sub
  End If
  'it's a maybe - it is in the last row on the sheet based
  'on entries in column G
  'are cells A:I filled?
  If Application.WorksheetFunction.CountA(Range(firstCol & Target.Row & _
   ":" & lastCol & Target.Row)) = fullCount Then
    'looks like it needs sorting
    Set sortRange = Range(firstCol & "1:" & lastCol & Target.Row)
    Set sortKey = Range(keyCol & firstDataRow) ' G2
    Application.ScreenUpdating = False
    Application.EnableEvents = False

    'change Order1:= to xlDescending if you need reverse order
    sortRange.Sort Key1:=sortKey, Order1:=xlAscending, Header:=xlYes, _
      OrderCustom:=1, MatchCase:=False, Orientation:=xlTopToBottom, _
      DataOption1:=xlSortNormal
    Application.EnableEvents = True

    Set sortRange = Nothing
    Set sortKey = Nothing
  End If
End Sub
