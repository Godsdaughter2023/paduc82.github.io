jeeped say:
You can use the worksheet function .Index() to pare down your array and Evaluate some rows to return a number set of {1, 2, 3}.

    Dim MyArray() As Double
    ReDim MyArray(5)
    MyArray(0) = 1
    MyArray(1) = 2
    MyArray(2) = 3
    MyArray(3) = 4
    MyArray(4) = 5
    Range("Output") = WorksheetFunction.Average(WorksheetFunction.Index(MyArray, 1, Evaluate("ROW(1:3)")))

In the above case, Range("Output") will receive 2 as (1+2+3)/3 = 2.

charles kenyon say:

If you are simply replicating the result of the first drop-down, why not simply use a REF (cross-reference) field?

The Word equivalent of the Excel formula would be the Word field { IF B7="yes""Yes" "No" }

BTW, IF Field comparisons are case-sensitive. I.e., "Yes" does not equal "yes" nor "YES."

The braces { } cannot be typed but must be inserted by Word. Ctrl+F9 will insert a pair.

Personally, I do not use RowColumn references but bookmarks. If you have a legacy form field, it already has a bookmark name assigned (which you can change).
