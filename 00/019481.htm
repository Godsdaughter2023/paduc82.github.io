Fixed Top 10 filter?

Hello everyone,

I have a list of 700+ items in a pivot table and I need to show ONLY the top 10 values. I am familiar the top 10 filter in excel, however, many of the values in my pivot table are the same. Right now, when I put the top 10 filter on, it is showing 50+ items because they all have the same value. 

I was thinking of instead of doing Top 10 items, do Top 10 Percent instead, but the only issue with this is that I need the pivot table to show only 10 items in the list. Doing it by percent has the potential to throw this off.

ranopano say:

FYI, 

I found a pretty simple solution:

Here is the original pivot table:

Pivot Table Label Pivot Table Value
Label 1 1
Label 2 2
Label 3 3

And then I create a linked table:

New Table Label New Table Value
=A2 =B2
=A3 =B3
=A4 =B4

This way I can manually limit the amount of rows in this new table to however many items I need.

Regardless of whether the pivot table is showing 10 or 500 items in the top 10 filter, the newly made table will only show however many rows I choose to display.

The other cool thing about this is that the slicers still work! The pivot table will get updated by the slicer and because the new table is referencing all of those cells, the new table is updated as well.

It is unicode 2234, and the complimentary because symbol is Unicode 2235
http://en.wikipedia.org/wiki/Therefore_sign

Go to Insert tab > Symbols group > Symbol drop down > More Symbols. In the Symbols tab of the Symbols dialog select "Arial Unicde MS" font, then in the bottom right corner confirm that the "FROM:" field is set to "Unicode (hex)", then in the "Character code" field beside it type 2234.
