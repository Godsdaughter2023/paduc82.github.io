Yes.  And please provide the complete text of the error messages without paraphrasing.

In general, messages at Windows start to the effect that some file is missing are an indication of partially-removed malware.  However, without the file name and error message, that's only a guess.

You research errors like this by doing a Google search for the file name.  Usually they turn up no hits.

When a search results in either no hits or suspicious hits, the answer usually is that the file in question is malware-related and your anti-malware application has deleted or quarantined the file but left behind the Windows registry entry that attempts to start the malware when you start up your computer.

First, download, install, update, and run full scans with both of these free scanning tools:


MalwareBytes AntiMalware
SUPERAntiSpyware

Don't run the scans simultaneously.  They may take a while, so start one up and then go do a non-computer chore for a while.

If the scans don't fix the problem (and they may not, because the actual malware file has already been dealt with), then download and run Autoruns.  Click the "Logon" tab and look in the "Image Path" column for the file named in your error message.  If it isn't shown in the Logon tab, click the "Everything" tab and use the File > Search tool.  An idiosyncrasy of the Autoruns search tool is that you have to move the selection line below the "found" item before you "Find Next."

Once you find the line with the offending file, either "Delete" it or (if you want to be safer) uncheck it.  The latter prevents the line from being executed but you can reactivate it later by removing the checkmark.

Why were SMART Files Removed from OneDrive in Windows 10?

Technical Level : Basic

Summary
This article details why SMART files, also known as placeholder files, were removed from OneDrive in Windows 10.
Details

OneDrive has gone through many changes in Windows 10, most notably the removal of SMART files (placeholder files) from OneDrive. They still exist in Windows 8.1, however they were removed, hopefully temporarily, from OneDrive in Windows 10 because of the fragmented sync engines between OneDrive on Windows 7/Mac, OneDrive on Windows 8.1, and OneDrive for Business. The following article details what changes the OneDrive team made and why they made them:

Taking the Next Step in OneDrive

From a development point of view, it's a much better approach to consolidate their codebase, but from a user point of view, it's very frustrating.

Hotmail is a free Microsoft service that provides email addresses and a website where users can send and receive messages using those addresses. The addresses can take many different forms, depending on how and where the user signs up for one. Here are a few variants: @hotmail.com, @live.com, @msn.com, @windowslive.com, @hotmail.ca (for Canadian users), @live.co.uk (for British ones). They are all treated in exactly the same way. Once you have a Hotmail-type email address, you have a Hotmail account regardless of the form of the address. This account is characterized by the email address and the password you choose when you sign up or subsequently change.

Every Hotmail account is also a Windows Live ID, which means you can use the same email address and password to gain access to any website or service that supports Live ID - like this Answers site.

You can access the website to read your mail and send messages in a variety of ways. Again, there is no difference between them. You can use http://hotmail.com or http://mail.live.com and several others.

You can also use a mail client (a program you install on your computer to manage email accounts). One such program is called Windows Live Mail, but there are many others.
