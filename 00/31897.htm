To do the job from the regular Windows environment, click Start > All Programs > Accessories > Command Prompt and enter these commands:

   cd /d "path "
    dir /b /s  >"file "

where:

   path = full path to the folder
   file = name of file to write

For example, to create a list of the files in My Documents and write it to a file named My Documents.txt :

   cd /d "C:\Documents and Settings\User\My Documents"
    dir /b /s  >"My Documents.txt"

The /s parameter lists files in the selected folder and all subfolders.  Omit it if you just want files in the selected folder.

foxfifth say: You can have an ip address conflict if you are connected to the internet via a wired or wireless router that allows two or more computers or other devices in your house to connect to the internet at the same time typically through a cable or dsl internet provider.  You can have that conflict even if the computers in your house haven't set up a network to share files and printers with each other.  You can go to Control Panel > Network and Sharing Center to see network information.

One common situation that can result in an ip address conflict is that 2 (or more) computers or other devices are connected to the router (via ethernet or wirelessly), both have had ip addresses assigned by the router automatically, one of the computers then goes into sleep mode, and while one is sleeping the router reboots.  When the router comes back online it doesn't know that the sleeping computer is there and that it already has an ip address assigned and so it gives out the ip address to the other computer.  When the sleeping computer wakes the duplicate error will be triggered.  It can be resolved by turning both computers off and back on so that the router will assign them new ip addresses.  If that isn't what is happening in your situation, the following may help:  http://windows.microsoft.com/en-US/windows7/Get-help-with-There-is-an-IP-address-conflict-message
