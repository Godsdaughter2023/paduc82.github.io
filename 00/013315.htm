irish_k say: I found a solution to my problem so I wanted to post back in case anyone else experiences the same issue...

Apparently, the problem is caused by some garbage (maybe cache) files stored in the user profile directory and/or HKCU registry keys.  I was not able to isolate which files or entries but recreating my user account fixed the issue and everything is working fine now.  I used the Windows Easy Transfer utility to save as much of my preferences and data as possible but I made sure not to back up the AppData which contains settings and preferences for Office.  After creating the backup of my profile using Easy Transfer, I logged in as a different user with administrator privileges and then deleted my old account (opting to delete all files).  Then I ran the Easy Transfer utility to restore my account, logged off and logged in to the newly restored account and everything works fine.  Lost a few app preferences, etc. but overall not too painful. 

Hopefully this tidbit will save someone some valuable time and not put them through what I have gone through the last couple of days.

Irish_K
