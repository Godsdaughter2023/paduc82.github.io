mouneshwar r say:

Hi Sunny,

 

Windows keeps a copy of all of your picture, video, and document thumbnails so they can be displayed quickly when you open a folder. The thumbnail preview problems can be resolved by clearing the thumbnail database and let the system rebuild it as and when required. To do so, use these steps:

 

Steps to Clear the Thumbnail Cache using Disk Cleanup

a.    Click Start, type “cleanmgr” (or typedisk cleanup) without quotes and press {ENTER}

b.    Select the system drive letter from the drop-down.

c.    Uncheck all the entries, except Thumbnails

d.    Click OK, and click Delete Files to confirm.

 

Alternatively, you can build thumbnail cache by going to the location below

%LocalAppData%\Microsoft\Windows\Explorer

 

The thumbnail database is stored globally in Windows 7 & Windows Vista, unlike Windows XP, where the thumbnail database is stored on a per-folder basis. In Windows 7 & Windows Vista, the default thumbnail cache location is as mentioned above.

 

Hope this information is helpful and let me know if you need any further assistance.

baldron say:

Page File Windows 10

I would like to increase my page file. I just cant find where this is possible. I go to disk management and it only allows to monitor. Such a easy thing should be easier to find. Wish to go to 4096 currently just over 3k. Thanks in advance.

a. user say:

Hello,

Thank you for posting your concern here in Microsoft Community and welcome to the Windows 10 Family.

From the description provided, I understand that you want to change the virtual memory size for Windows.

Please follow the below steps.

  1. Press Windows Key + Pause Break from the keyboard to open the System Properties. Then choose ‘Advanced System Settings’ in the left pane and then click on the ‘Settings’ button in the ‘Advanced’ option.
  2. In the ‘Performance Options’ window, go to the ‘Advanced’ tab and then click ‘Change’.
  3. Uncheck the box next to ‘Automatically manage paging file size for all drives’.
  4. Now select ‘Custom Size’ and then set the ‘Initial Size’ and ‘Maximum Size’.
  5. Click ‘OK’ and then click ‘OK’ again.
  6. Now, Click ‘OK’ twice to close the ‘Performance Options’ and ‘System Properties’ window.
  7. Click ‘Restart now’ to apply these changes.

Hope the above information helps.

Thank You.

This is my generic how to for proper driver updates : (Check for BIOS and driver

most current. Control Panel - Device Manager - Display Adapter - write down the

write down the version info. Now click UPdate Driver (this may not do anything

will refresh the driver stack.

Repeat that for Network - Network Card (NIC), Wifi, Sound, Mouse and Keyboard
if 3rd party with their own software and drivers and any other major device drivers
you have.

Now go to System Maker's site (Dell, HP, Toshiba as examples) (as rollback)
and then Device Maker's site (Realtek, Intel, Nvidia, ATI as examples) and get
their latest versions. (Look for BIOS, Chipset and software updates at System

Download - SAVE - go to where you put them - Right Click - RUN AD ADMIN -
REBOOT after each installation.

Always check in Device Manager - Drivers tab to be sure the version you are
installing actually shows up. This is because some drivers rollback before the
latest is installed (sound drivers particularly do this) so install a driver - reboot
- check to be sure it is installed and repeat as needed.

Repeat at Device Makers - BTW at Device Makers DO NOT RUN THEIR
SCANNER - check manually by model.

Manually look at manufacturer's sites for drivers - and Device Maker's sites.

canadian tech say:

I'm guessing this PC is more than 5 years old.  If it is, you need to replace the coin cell battery in it.

If it is a desktop PC, it is easy.  Go get yourself a 2032 coin cell.  Commonly available in many places.  Maybe as much as $5.  Unplug your PC and hold the ON switch in for 7 seconds.  Now open the cover on the PC.  You will see the little guy easily.  Take a very small bladed screw driver, like the ones used to tighten eyeglass screws and pry the old battery out.  Now press the new one in its place.  Put everything back now.

When the computer starts, it will need its clock set, but this is the last time you will have to do that for another 5 years.

If it is a notebook PC, it is a very different story.  The battery is probably deep in the innards and will require a competent technician to get to it and he will have to have a special part to replace.
