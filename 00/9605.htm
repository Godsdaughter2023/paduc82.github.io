rich007a say:

You could use some VBA code to do something like this:

    Dim c As Range, rng As Range
    Set rng = Selection.Columns(1)
    For Each c In rng
        If c.Font.Italic = False Then
            c.EntireRow.Hidden = True
        End If
    Next c

 

Simply select the column you want to filter-out (hide) all non-italic cells and run the code.

If you want to hide non-bold cells, use  

    If c.Font.Bold = False Then

instead.
