Sorry, but despite the fact that you get the opposite impression, only 1 image at a time can be inserted into a Word document. When you select & drag multiple images from Finder a circular red indicator will display the number of images selected, but only the last one selected will appear in the document.

Additionally, if you use Insert> Photo you'll be able to select only 1 image at a time, regardless of whether you use Picture from File or Photo Browser. The 1 exception is that if there are several images already in a Word document you can select & Copy those that are on the same 'page', then Paste into a different document.

If you have multiple images to insert you need VBA to automate the procedure. IMHO, if you have so many that inserting them individually is not sufficient, there probably are more than should be in a Word document to begin with :-)
