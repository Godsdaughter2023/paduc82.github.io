c) Now copy and paste the commands mentioned below one by one and hit “Enter” after each command –

 


netsh winsock reset

ipconfig /flushdns
ipconfig /release
ipconfig /renew
netsh int ip stop

e) Now highlight “Internet Protocol Version 4 (TCP/IPV 4)” and select properties.
