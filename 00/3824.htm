Keep leading spaces in Excel from imported text - Microsoft Community

martonj say:

Keep leading spaces in Excel from imported text

In the original text file, open in notepad / textpad, replace all spaces with a character otherwise not in use, such as %. 

Import the amended text into Excel using the Text Import Wizard, using Fixed width and a single column. 

Once imported, write a function in the next column, using SUBSTITUTE to replace the %-s with spaces, i.e. =SUBSTITUTE(A1,"%"," "). 

This method will bring back all the spaces; while using the text import without this would lose the leading spaces.

The RegEdit app can really foul up your Windows installation if used improperly.  If you aren't feeling totally confident and comfortable using it and editing the registry, you shouldn't.  Wait for Microsoft to (Hopefully) release a patch for this instead.   Edit the registry at your own risk, you've been warned.

That said, you can limit the damage by backing up the registry before making any modifications to it.  Here is Microsoft's guide on how to do this, you can save the entire registry and not just specific sections, though it dramatically adds to the file size:  http://windows.microsoft.com/en-us/windows/back-up-registry#1TC=windows-7

Here's a Microsoft article on how to both backup and restore the registry:  https://support.microsoft.com/en-us/kb/322756

You should also create a System Restore point to be extra safe.  You can search for it in Windows 10 by typing "System Restore" and hitting Enter.  There should be an option to create a restore point in the dialog box.  You may also want to create a System Repair Disc which you can use if Windows fails to start.  To do this, have a blank disc ready, search your PC for "Backup and Restore" and on the screen that comes up, choose the "Create a System Repair Disc" to start the process.  Creating a repair disc is a bit overkill in this situation, but you should at least create a restore point.  Here's a link on how to use the repair disc:  http://www.dummies.com/how-to/content/how-to-use-a-system-repair-disc-to-restore-windows.html
