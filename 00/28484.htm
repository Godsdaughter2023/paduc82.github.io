Thanks in advance,

Phil

steve rindsberg say:

This is usually caused when PowerPoint's been displaying a show on a second monitor or projector and later "remembers" to use that same display, even though it's been disconnected.

To move it back onto the main monitor:

Use ALT+TAB to select the slideshow window

Press ALT+SPACE then M

Use the arrow keys to move the slideshow window back onto the main monitor.
