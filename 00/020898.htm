a. user say:

Hello Ganesh,

When you get 'Windows was unable to complete the format' error message, you can try to resolve this issue with Windows Disk Management tool using a Windows computer. All you need to do is to follow this path:

Step 1. Open the Explorer, right-click the computer icon "This PC" and select "Manage". On the pops up window, from the right pane go to "Storage" -> Disk Management.

Step 2. Now find the SD card that shows unable to complete the format error. If there are any partitions on the drive, delete them all, and then you'll get a single volume.

Step 3. Right-click on the drive and hit "Format". Choose NTFS file system if the device is larger than 4GB, or else choose the FAT32 file system.

We're just here if you have further questions.
