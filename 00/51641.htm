I manged to fix it after much reading and testing here is the summary and the solution

  • Supporting files used by Windows for AMD drivers can be corrupted or removed but AMD's driver never warns you about this even on re-installs.
  • The missing files are required for SP1 install
  • If you already tried to install SP1 there may be a further item which can throw you off track in the form of a SP1 specific folder that does not exist but I'll get to that later.
Step 1) Follow this guide on where to put the files back for Windows 7, you can follow his linked guides on extracting the files from the Windows 7 DVD or download them from thefirst post in that thread.

Step 2) Run sfc /scannow via command prompt and see if it logs any errors, if it does then go to Windows\Logs\CBS\CheckSUR.txt and see what files are missing and replace them. Run sfc again until it gives the all clear.

Step 3) Run the System Update Readiness Tool then check the same CheckSUR.txt file to see the results, if it's all okay then you can install SP1 however...

Step 4) If it says "amd64_atiilhag.inf_31bf3856ad364e35_6.1.7601.17514_none_03c46b205be81dfd" is missing this is a SP1 folder, what you do to correct the problem is copy the contents of the winsxs "amd64_atiilhag.inf_31bf3856ad364e35_6.1.7600.16385_none_019357585ef99a63" folder into the winsxs "dfd" folder overriding the files within.

Step 5) Now run the system update tool again and check the txt file if it gives the all clear you can install SP1.

  • Keep a written record of your password.
  • Create a second admin account and keep a written record.
  • Create a password recovery disk.
Each of these would take less than a minute. If you used none of them then you have to use the following extremely painful procedure. You can reset any password by going through the steps outlined below. The process is involved and requires some careful typing. Ask a computer-savvy friend to assist you if you lack experience in this area.
 1. Set your BIOS to use the DVD drive as the primary boot device.
 2. Boot with a Windows Repair CD.
 3. Press a key when the message indicating "Press any key to boot from CD or DVD …" appears.
 4. Select a language, a time and currency, and a keyboard or input method, and then click Next. You are now in Repair Mode.
 5. Select an Operating System, then click Next.
 6. Select the Command Prompt.

    dir C:\Win* {Enter}
    dir D:\Win* {Enter}
    dir E:\Win* {Enter}
    etc.
    until you find the Windows folder.
    I will call that drive "Q:".
 8. Type these commands:
    cd /d Q:\windows\System32 {Enter}
    move  sethc.exe  \  {Enter}
    copy  cmd.exe  sethc.exe  {Enter}
 9. Reboot normally.
10. When challenged for a password, tap the Shift key five times. You will see a small black screen.
11. Type these commands:
    net user  {Enter}
    (To see a list of all accounts that exist on your machine)
    net user Administrator {Enter}
    (To see if the Administrator account is active)
12. Type these commands:
    net user "xxx" ppp  {Enter}
    (Replace xxx with the account for which you wish to set a new password, and ppp with the new password)
    net user Administrator /active:yes {Enter}
    (To activate the Administrator account if desired)
13. Log on under this account, using the new password.
14. Reboot the machine into Repair Mode as per Steps 1..6 above.
15. Type these commands:
    cd /d Q:\windows\System32 {Enter}
    copy  /y  \sethc.exe {Enter}
16. Reboot into Windows.
