Hi Frank,

You can perform a System Restore regardless if the System Protection setting is turned On or Off. The reason why System Protection needs to be turned On is to generate a restore point automatically when you install a new app, driver, or Windows update. For us to better assist you with this, we'd like to ask for the exact error message when you try running performing a System Restore?

Please follow the steps below to turn On/Off System Protection:

  1. Press Windows key + R to open Run dialog.
  2. Type Control Panel, then hit Enter.
  3. Set View by: Small icons, then select System.
  4. In the left pane, click System protection.
  5. Under System Protection tab, you will see Local Disk (C:) (System) is On.
  6. Click Local Disk (C:) (System) to highlight it, then click Configure.
  7. Under Restore Settings, you will see the option to turn On/Off System Protection.
  8. Click Apply.
  9. Click OK.

Note: Under Disk Space Usage, you may adjust the maximum disk space for system protection.
