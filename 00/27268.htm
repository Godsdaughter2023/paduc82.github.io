HyperX 7.1 Audio Devices and Windows 10 Pro (64 bit) versions - Microsoft Community

michaelresch2 say:

HyperX 7.1 Audio Devices and Windows 10 Pro (64 bit) versions 1703 and 1709

A recent Google Search of HyperX 7.1 Audio and Windows 10 revealed that Gamers using USB Audio Headphones with Boom Microphones are having great difficulty getting their Hardware to communicate properly with a PC in a Windows 10 environment.  I have, by persistent experimentation, discovered a plan-of-action to get HyperX 7.1 Audio Hardware to install properly on a PC running Windows 10 Pro versions 1703 and 1709.  The technique is very simple:

  1. Unplug the USB Headphones from the USB Port.
  2. Use the Device Manager to assess devices and drivers in two categories:
  3. "Audio inputs and outputs"
  4. "Sound, video and game controllers"
  5. Uninstall all devices and drivers that are superfluous to what actually needs to be used in those two categories
  6. Perform a System Restart
  7. Use the Device Manager to ascertain which devices and drivers that were previously uninstalled the System has restored
  8. Disable all of the System restored devices and drivers
  9. Perform another System Restart
  10. Login to the PC
  11. Allow the Startup Manager time to completely activate the PC to fully awake
  12. Then Plug the USB Headphones into a USB slot
  13. In around 10 or 15 seconds the OS will notify the user that HyperX 7.1 Audio is installed
  14. Use the Sound app available by Rt+Clicking the Volume Icon on the right side of the Task Bar to configure (Playback Devices & Recording Devices) to the user's needs

nishit@ms say:
Windows Defender Offline system requirements

For both the PC infected with a virus or malware, and the PC that you are creating the bootable media on, you'll need PCs with the following minimum requirements.

Operating system:

  • Windows XP Service Pack 3

  • Windows Vista, Windows Vista with SP1, Windows Vista with SP2, or higher

  • Windows 7, Windows 7 with SP1, or higher

  • Windows Developer Preview, Windows 8 Consumer Preview, Windows 8 Release Preview

Processor:

  • Windows XP: 768 MB RAM or higher

  • Windows Vista, Windows 7, Windows Developer Preview, Windows 8 Consumer Preview, Windows 8 Release Preview: 1 GB RAM or higher

Video resolution: 800 X 600 or higher

Available hard disk space: 500 MB

For the PC infected by a virus or malware

  • The PC must have the same Windows operating system (OS) architecture (32-bit or 64-bit) as Windows Defender Offline. Bootable media created on any version of Windows can be run on any other version of Windows. The only thing that needs to match is the architecture of Windows Defender Offline and the OS of the infected machine.

  • BitLocker must be disabled to use Windows Defender Offline.

For the PC on which you create the bootable media

  • Administrator account: A user account with administrator privileges to create the bootable media.

  • Internet connection: An Internet connection to install and download the latest virus and spyware definitions for WindowsDefender Offline.

  • Internet browser: Windows Internet Explorer 7.0 or higher, or Mozilla Firefox 3.0 or higher.

  • Media: A CD, DVD, or USB flash drive with at least 250 MB of free space.
